'''石岡區'''（[[臺灣客家語|臺灣客家語]][[大埔腔|大埔腔]]：shagˋ gong+ ki+），位於[[臺灣|臺灣]][[臺中市|臺中市]]的一個市轄區，位置在臺中市中部偏北。西與[[豐原區|豐原區]]相鄰，南與[[新社區|新社區]]銜接，東及北與[[東勢區|東勢區]]以[[大甲溪|大甲溪]]為界，設有[[石岡水壩|石岡水壩]]。區內氣候屬[[亞熱帶|亞熱帶]]氣候，熱而濕潤，特產有鶯歌[[桃|桃]]、[[椪柑|椪柑]]、[[葡萄|葡萄]]、[[楊桃|楊桃]]、柿子、[[高接梨|高接梨]]及柳[[松茸|松茸]]等。

== 地理 ==
石岡區位於[[臺中市|臺中市]]中部偏北，地處[[大甲溪|大甲溪]]溪南岸狹長的河谷平原地帶與[[新社河階群|新社河階群]]的交會處。全區幅員不大，東西略寛（6.37公里），南北略窄（4.58公里），輪廓呈東西橫置之樹葉形狀。境內北半部為大甲溪河谷平原，海拔約283公尺（區公所處），為區內精華區；南半部則為不同時期因地盤上升及大甲溪沖積作用而形成的各個河階地，包括金星面河階、仙塘坪河階及南眉河階，都屬於新社河階群的一部分。全區最高點海拔約565公尺，即在金星里西南方之河階台地上<ref>《石岡鄉志》，石岡鄉公所，1989年，第17－18頁</ref>。

[[大甲溪|大甲溪]]是[[中台灣|台灣中部]]的重要河川，屬於[[中央管河川|中央管河川]]，主流上游發源於[[南湖大山東峰|南湖大山東峰]]<ref name="中央管河川表">[http://www.wra.gov.tw/public/PDF/7211919254.pdf 經濟部水利署：臺灣地區之河川─中央管河川] {{webarchive|url=https://web.archive.org/web/20160304132742/http://www.wra.gov.tw/public/PDF/7211919254.pdf |date=2016-03-04 }}</ref>，向西注入[[台灣海峽|台灣海峽]]，流域主要分佈於[[台中市|台中市]]，並包括[[南投縣|南投縣]]、[[宜蘭縣|宜蘭縣]]之一小部分。大甲溪中游自本區東南端石仔倔流入，環繞區境東側及北側，於本區西北端米粉寮附近流出，形成本區與[[東勢區|東勢區]]的天然邊界。

[[食水嵙溪|食水嵙溪]]是大甲溪南岸支流之一，全長大約有15公里，發源地是在[[新社區|新社區]]永源村馬力埔湳堀，其水流經過大南水頭、番社嶺、水尾、[[雙翠水壩|雙翠水壩]]後往新社食水嵙、石岡食水坑，最終於[[石岡水壩|石岡水壩]]下游處匯入大甲溪。食水嵙溪沿岸生態資源豐富，是多種[[鳥類|鳥類]]的棲息地。

石岡區全境屬於[[亞熱帶氣候|亞熱帶氣候]]區內，終年溫和，適於農作。依據石岡水壩觀測資料，年平均氣溫為攝氏21.9度。月平均溫度最低者為一月份之14.2度，最高則為七月之28.3度。全年平均降雨量為2,113.7公厘，而各月降雨量多寡則受[[季風|季風]]所支配。夏半季為雨季，三月至八月平均降雨量佔全年的75.3%；冬半季東北季風因受[[中央山脈|中央山脈]]阻擋影響，降雨量大減，十月至翌年一月平均降雨量僅佔全年的5.7%<ref>《石岡鄉志》，石岡鄉公所，1989年，第25－28頁</ref>。

== 歷史 ==
[[File:Tong_Fong_Bikeway_Broken_Bridge_01.jpg|thumb]]時斷裂的[[東豐自行車道|東豐自行車道]]梅子鐵橋]]

石岡區區治所在聚落舊稱'''石崗仔'''，乃因地處[[大甲溪|大甲溪]]南岸沙洲，在滿佈石磊之坑谷口處建莊而得名。[[台灣日治時期|日治]][[大正|大正]]九年（1920年），區境獨立設行政區'''[[石岡庄|石岡-{庄}-]]'''，並將該聚落易名為'''石岡'''而作為-{庄}-役場所在地。

石岡地區古時為[[平埔族群|平埔族群]][[噶哈巫族|噶哈巫族]]（Kaxabu）樸仔籬群（Varutto）社寮角社社域，早期也是有[[泰雅族|泰雅族]]（Atrayal）出沒的地區。

[[清|清]][[康熙|康熙]]末年石岡開始有[[客籍|客籍]]移民入墾，至[[乾隆|乾隆]]廿六年（1761年），[[漢人|漢人]]墾拓已達現今的土牛里，築有十九個土牛作為漢番之界，並嚴禁漢人越界進墾。

清代，原屬[[彰化縣_(清朝)|彰化縣]]貓霧栜東、西堡，至[[乾隆|乾隆]]中葉改為[[捒東上堡|捒東上]]、[[捒東下堡|下堡]]，同時以本區及[[東勢區|東勢]]、[[豐原區|豐原]]為上堡。[[光緒|光緒]]十三年（1887年）台灣陞建行省，本區隸屬於[[台灣府|台灣府]][[台灣縣|台灣縣]][[捒東上堡|捒東上堡]]。

[[台灣日治時期|日治]]初期，[[明治|明治]]二十八年（1895年）台灣縣改設[[台灣民政支部|台灣民政支部]]，旋改民政支部為[[臺中縣|臺中縣]]，下設辦務署，本區屬於台中縣[[葫蘆墩辦務署|葫蘆墩辦務署]]捒東上堡轄域，遂置'''石崗區'''-{庄}-長，轄有石崗仔-{庄}-、仙塘坪、社寮角、土牛及朴仔口-{庄}-等。明治三十四年（1901年）廢三縣三廳及辦務署制，遂改縣為廳，全台設二十廳，廳下各設支廳管轄。[[大正|大正]]九年（1920年）七月，地方政制改為五州三廳，下設郡街-{庄}-，台中廳改州，支廳改為郡；將原本鄉所屬之樸仔口-{庄}-，以埤頭為界合併葫蘆墩區，而本鄉改稱為[[臺中州|台中州]][[東勢郡|東勢郡]]'''[[石岡庄|石岡-{庄}-]]'''。

民國34年（1945年）[[中華民國政府|中華民國政府]]接收[[大日本帝國|日本]][[台灣總督府|台灣總督府]]轄區，初期改為[[臺中縣|臺中縣]]東勢區'''石岡鄉'''。民國39年（1950年）底，撤廢[[區署|區署]]，改稱臺中縣石岡鄉。

民國88年（1999年）的[[921大地震|921大地震]]對石岡鄉造成嚴重損失，全鄉死亡人數達177人，佔當時居住人口數的1.14%，是該次地震受災鄉鎮市行政區中，地震死亡率最高的<ref>[http://921.gov.tw/rec_doc/doc/g004/921%E9%9C%87%E7%81%BD%E7%9B%B8%E9%97%9C%E9%98%B2%E7%96%AB%E8%B3%87%E6%96%99pdf%202.pdf 韓世寧、陳國東，《九二一集集大地震南投縣台中縣死亡情形調查報告》，行政院衛署，2000年1月25日]</ref>。設置於[[大甲溪|大甲溪]]的[[石岡水壩|石岡水壩]]因位於[[車籠埔斷層|車籠埔斷層]]帶上，右岸壩及第16、17、18號溢洪閘門毀損。

民國99年（2010年）12月25日，臺中縣市合併改制為[[臺中市|臺中市]]'''石岡區'''。

== 經濟 ==
石岡區早期為典型的農業社會，居民多以務農為生，主要耕地分佈於[[大甲溪|大甲溪]]南岸之河谷平原，聚落多為散村型態。近年來隨著工商業的發展，部分居民逐漸遷往交通便利處，形成市街與集村。其中以石岡街區規模最大，具有都市化雛型，人口佔全區三分之一，為區內行政、商業中心<ref>《石岡鄉志》，石岡鄉公所，1989年，第215頁</ref>。

[[農業|農業]]為本區最重要的經濟活動，全區耕地面積約七百餘公頃，其中[[水田|水田]]約佔三分之二，主要作物為[[水稻|水稻]]；[[旱田|旱田]]約佔三分之一，主要作物為[[柑桔|柑桔]]、[[水梨|水梨]]、[[葡萄|葡萄]]、[[香蕉|香蕉]]、[[鳳梨|鳳梨]]等[[水果|水果]]及各種[[蔬菜|蔬菜]]。依據1973年的統計資料，各類作物年產量較多者為稻米2,173公噸、蔬菜1,980公噸、柑桔1,177公噸<ref>《石岡鄉志》，石岡鄉公所，1989年，第218頁</ref>。

畜牧業以禽類為主，年產量較多者為雞45,863隻、鴨14,534隻、鵝4,668隻。漁業多為淡水池塘養殖型態，養殖面積約8公頃，魚種以[[鰱魚|鰱魚]]、[[草魚|草魚]]、[[福壽魚|福壽魚]]、[[鯉魚|鯉魚]]為主<ref>《石岡鄉志》，石岡鄉公所，1989年，第221頁</ref>。

在工商業方面，由於本區河谷平原狹小，北有大甲溪阻隔，南有新社河階地限制腹地規模，再加上東西兩側距離[[東勢區|東勢]]、[[豐原區|豐原]]兩大城鎮皆甚近，工商活動被其吸引，因而自古以來皆難以發展。省道[[台3線|台3線]]擴寬以後原有招來較大規模工商業投資機會，然因受限於「[[石岡水壩|石岡水壩]]特定區計畫」的影響，無法順利展開。依據1976年的統計，境內工廠僅有39家，規模均不大。業別以以鄰近丘陵地生產之竹、木製材業17家最多，其餘較多者為化工業7家、食品工業5家、非金屬製品業5家；商戶僅有111家，家數較多者為零售業79家、個人服務業17家、交通運輸業14家。商業活動主要服務當地居民日常生活所需，專業服務則仰賴東勢、豐原或台中市區等大型城鎮<ref>《石岡鄉志》，石岡鄉公所，1989年，第216頁</ref>。

== 交通 ==

=== 道路 ===
[[省道|省道]][[台3線|台3線]]是石岡區最主的聯外道路，以略呈東南東－西北西走向穿越全鄉精華區所在的[[大甲溪|大甲溪]]南岸河谷平原。經由該道路向西轉西南出區界約1公里，即可連接[[國道四號_(中華民國)|國道4號]]的豐原端，由此可連接國道1號（[[中山高速公路|中山高速公路]]）、國道3號（[[福爾摩沙高速公路|福爾摩沙高速公路]]），快速前往南北各地，每逢上下班尖峰時段車流偏大，因此時常造成石岡區明德路至豐原區豐勢路二段之間大塞車。

若由省道台3線向西南續行，則可通往[[豐原區|豐原]]，再轉向南可經由[[潭子區|潭子]]、[[北屯區|北屯]]進入[[臺中市|臺中市]]中心，亦可續行前往南部各地。若由省道台3線向東出區界經[[東勢大橋|東勢大橋]]過[[大甲溪|大甲溪]]，即可到達[[東勢區|東勢區]]。在東勢大橋東側可沿台3線轉向北通往[[苗栗縣|苗栗縣]]的[[卓蘭鎮|卓蘭]]、[[大湖鄉_(台灣)|大湖]]等地，續行可通往北部山線各地；亦可在東勢大橋東側向南轉接省道[[台8線|台8線]]，前往[[天冷|天冷]]、[[和平區_(臺中市)|和平]]、[[谷關|谷關]]等地，或在天冷轉接省道[[台21線|台21線]]前往[[國姓鄉|國姓]]、[[埔里鎮|埔里]]等地。

[[市道129號|市道129號]]是本區東部的一條聯外道路，起點位於省道台3線東勢大橋西側附近的土牛，向南可通往[[新社區|新社]]、[[大坑_(臺中市)|大坑]]、[[太平區_(臺中市)|太平]]、[[大里區|大里]]等地。

區內另有多條[[台中市鄉道列表|鄉道]]可供區域性連絡，其中屬於連外道路性質的為[[中44-1線|中44-1線]]（石岡向北經[[長庚大橋|長庚大橋]]過大甲溪通往東勢區下灣）、[[中93線|中93線]]（社寮角往南通往新社區十分、崑山、中興嶺、茅埔角等地）及[[中90線|中90線]]（金星面往南經下坑溪金坪橋通往豐原區公老坪、北坑口、豐原等地），而屬於區內連絡道路的僅有[[中91線|中91線]]（石岡經金星面通往仙塘坪）<ref>《臺中縣石岡鄉行政區域圖》，內政部，2005年12月</ref><ref>《臺中縣新社鄉行政區域圖》，內政部，2005年12月</ref><ref>《臺中縣豐原市行政區域圖》，內政部，2005年12月</ref>。

== 教育 ==
=== 國民中學 ===
[[臺中市立石岡國民中學|石岡國中]]位於梅子-{里}-南端與[[新社區|新社區]]中正-{里}-交界處（校園位於新社區境內），創立於民國55年8月，是石岡區唯一的[[國民中學|國民中學]]，也是全區最高學府。
=== 國民小學 ===
[[臺中市石岡區石岡國民小學|石岡國小]]位於石岡-{里}-，學區為石岡區西部各-{里}-。該校前身為「石岡公學校」，創立於日治時期的[[明治|明治]]三十二年（1899年），為區內最古老的學校。[[昭和|昭和]]十六年（1941年）更名為「石岡國民學校」，民國57年（1968年）再更名為「石岡國民小學」。

[[臺中市石岡區土牛國民小學|土牛國小]]位於土牛-{里}-，學區為石岡區東部各-{里}-。該校前身為「石岡公學校土牛分教場」，創立於[[大正|大正]]十年（1921年）。大正十五年（1926年）獨立為「土牛公學校」，昭和十六年（1941年）更名為「土牛國民學校」，民國57年再更名為「土牛國民小學」。

== 宗教 ==
石岡區民的宗教信仰主要為[[道教|道教]]、[[佛教|佛教]]與[[台灣民間信仰|台灣民間信仰]]，而[[基督教|基督教]]亦設有據點。主要的道教及佛道混合宮觀有位於萬興-{里}-的萬興宮、位於萬安-{里}-的萬安宮、位於土牛-{里}-的慈雲宮、位於龍興-{里}-的龍興宮及位於九房-{里}-的廣興宮等，佛教寺院則有位於土牛-{里}-的明山寺，基督教堂有位於九房-{里}-的石岡基督[[長老教會|長老教會]]及位於和盛-{里}-的土牛基督教[[神召會|神召會]]<ref>《石岡鄉志》，石岡鄉公所，1989年，第260－266頁</ref>。

== 文化 ==
[[File:土牛客家文物館.JPG|thumb]]

石岡區圖書館位於石岡街區南側，瀕臨[[食水嵙溪|食水嵙溪]]，樓高四層，館舍面積900餘坪，藏書達8萬餘冊，為區內民眾閱讀研究的最佳處所<ref>[http://www.culture.taichung.gov.tw/LibraryIntroduction.aspx?menuID=442 《石岡區圖書館》，臺中市政府文化局官方網站]</ref>。該館三樓為設有「石岡客家文物館」，館藏品多屬民眾捐獻或清潔隊拾獲，較無系統化蒐集，但也呈現出社區[[博物館|博物館]]的克難精神，也算是一大特色。該館設有3D魔幻劇場，參觀者可以觀賞立體化之[[客家山歌|客家山歌]]演唱，亦可學習[[客語|客語]]及客家詩詞、客家童謠等<ref>[http://www.cam.org.tw/big5/museum03.asp?rowId=238 《石岡客家文物館》，中華民國博物館學會官方網站] {{webarchive|url=https://web.archive.org/web/20130926000835/http://www.cam.org.tw/big5/museum03.asp?rowId=238 |date=2013-09-26 }}</ref>。

[[土牛客家文化館|土牛客家文化館]]位於土牛-{里}-南方，為土牛望族劉家的老伙房改建而成，兩側廂房仍有劉家後人居住，為全台第一座活的[[客家|客家]]文化館<ref>[http://shihkang-travel.taichung.gov.tw/html/travelline-blue.htm 《土牛客家文化館風景線》，石在好玩網站，臺中市石岡區公所] {{webarchive|url=https://web.archive.org/web/20150402150242/http://shihkang-travel.taichung.gov.tw/html/travelline-blue.htm |date=2015-04-02 }}</ref>。其歷史起源於清康熙五十年（1711年），土牛劉家開臺始祖劉元龍自[[廣東|廣東]][[潮州府|潮州府]][[大埔縣|大埔縣]]渡臺，初居在嘉義，劉元龍56歲時返鄉，其子永萬、永順與其孫劉文進（永順之子）留臺並遷往土牛（石岡）地區發展；劉文進號啟成，經營日用雜貨買賣與豆腐生意謀生，因經營得法，漸有積蓄，家業日益興隆，而老夥房即劉家來臺第三代祖先劉文進在清嘉慶末年（約1820年間）時所興建。

== 旅遊 ==
[[File:Shih_Gang_Earthquake_3.JPG|thumb]]]]
石岡區為[[參山國家風景區|參山國家風景區]][[谷關|谷關]]遊憩區的入口城鎮，因地處石岡水壩水源保護區，工業、農牧業發展受限，因此得以保有自然生態環境及傳統文化特色。區內主要的旅遊景點除區公所規劃的石岡水壩、金星石珠、情人木橋、五福臨門、梅子社區、土牛客家文化館及兩圳生態步道等風景線外<ref>[http://shihkang-travel.taichung.gov.tw/html/about.htm 《話說石岡》，石在好玩網站，臺中市石岡區公所] {{webarchive|url=https://web.archive.org/web/20141225053945/http://shihkang-travel.taichung.gov.tw/html/about.htm |date=2014-12-25 }}</ref>，尚有東豐自行車綠廊、石岡農會碾米穀倉、挑物古道等知名景點。

[[石岡水壩|石岡水壩]]位於石岡區西北部，是大甲溪的一個[[攔河堰|攔河堰]]，於1977年完工營運，其上游流域面積1,061[[平方公里|平方公里]]，約佔大甲溪全流域面積之85%，為大甲溪流域中下游水資源運用樞紐；主要功能在於調節大甲溪流域尖峰發電尾水及河川水資源，並提供大台中地區公共給水及灌溉用水。[[921大地震|九二一大地震]]造成右岸壩及第16、17、18號溢洪閘門毀損。震後，石岡水壩部份受損壩體目前被保留起來，曾為[[九二一地震國家紀念地|九二一地震國家紀念地]]，並於第16號溢洪道後方設置[[魚道|魚道]]，維護大甲溪魚類生態。

[[File:東豐自行車綠廊.jpg|thumb]]]]
「[[金星石珠|金星石珠]]」是位於[[金星面庄中福德祠|金星面-{庄}-中福德祠]]門口兩側的大石球，置於木架之上。臨近有張家百忍堂、金星石鎖、石忠宮、石岡圖書館等值得駐足之處<ref>[http://shihkang-travel.taichung.gov.tw/html/travelline-orange.htm 《金星石珠風景線》，石在好玩網站，臺中市石岡區公所] {{webarchive|url=https://web.archive.org/web/20150402140018/http://shihkang-travel.taichung.gov.tw/html/travelline-orange.htm |date=2015-04-02 }}</ref>。「情人木橋」位於萬興-{里}-社寮角聚落，橫越[[食水嵙溪|食水嵙溪]]，為遠東跨距最長的膠合木橋，臨近規劃有客家美食區、情人商店家、石農廣場、食水嵙休閒農業區等觀光據點<ref>[http://shihkang-travel.taichung.gov.tw/html/travelline-lblue.htm 《情人木橋風景線》，石在好玩網站，臺中市石岡區公所] {{webarchive|url=https://web.archive.org/web/20150701182016/http://shihkang-travel.taichung.gov.tw/html/travelline-lblue.htm |date=2015-07-01 }}</ref>。「五福臨門」位於龍興-{里}-火燒坪，原為[[樟樹|樟樹]]、[[相思樹|相思樹]]、[[榕樹|榕樹]]、[[朴樹|朴樹]]、[[楠樹|楠樹]]五棵樹木緊密相鄰，其後朴樹、楠樹相繼枯死，補植[[香楠|香楠]]，加上天然繁衍的樹杞，仍然湊足「五福」之數<ref>[http://shihkang-travel.taichung.gov.tw/html/travelline-brown.htm 《五福臨門風景線》，石在好玩網站，臺中市石岡區公所] {{webarchive|url=https://web.archive.org/web/20150402143112/http://shihkang-travel.taichung.gov.tw/html/travelline-brown.htm |date=2015-04-02 }}</ref>。「梅子社區」位於梅子-{里}-，社區四周遍植[[梅樹|梅樹]]。[[921大地震|921大地震]]災情慘重，居民協力重建，營造許多景點，包括五分車懷舊公園、震臨泉、梅子古道、甕牆等<ref>[http://shihkang-travel.taichung.gov.tw/html/travelline-purple.htm 《梅子社區風景線》，石在好玩網站，臺中市石岡區公所] {{webarchive|url=https://web.archive.org/web/20150402130159/http://shihkang-travel.taichung.gov.tw/html/travelline-purple.htm |date=2015-04-02 }}</ref>。「兩圳生態步道」指[[八寶圳|八寶圳]]及其分流電火圳旁的生態步道，橫跨德興-{里}-、土牛-{里}-、梅子-{里}-、萬興-{里}-等四-{里}-南部，動、植物生態豐富。臨近尚有土牛五分車站紀念公園、福仙祠、南眉101步道、社寮角發電廠等景點，值得一遊<ref>[http://shihkang-travel.taichung.gov.tw/html/travelline-green.htm 《兩圳生態步道風景線》，石在好玩網站，臺中市石岡區公所] {{webarchive|url=https://web.archive.org/web/20140104230628/http://shihkang-travel.taichung.gov.tw/html/travelline-green.htm |date=2014-01-04 }}</ref>。

[[東豐自行車綠廊|東豐自行車綠廊]]是位於[[東勢區|東勢區]]、石岡區及[[豐原區|豐原區]]之間的一條[[自行車道|自行車道]]，全長約12[[公里|公里]]，係由舊[[東勢線|東勢至豐原間的廢棄鐵道]]改建而成，沿線仍保留有多處[[車站|車站]]和[[月台|月台]]建築。其中石岡車站月台附近原為平地，經[[九二一地震|九二一地震]]後出現約十公尺高低落差，為體驗[[地震|地震]][[斷層|斷層]]的極佳地點。沿線設有夜間照明設備，並可以在豐原端與[[后豐鐵馬道|后豐鐵馬道]]相接。然於2006年6月9日因大水衝擊，導致連接[[東勢區|東勢]]及石岡之[[梅子鐵橋|梅子鐵橋]]受損中斷，直到2008年5月31日，該橋修復完成，重新啟用。<ref>{{cite web|title=梅子鐵橋修復 東豐綠廊 6/1 全線通車|url=http://emmm.tw/news_content.php?id=9583|publisher=美美美旅遊聯盟}}</ref>

[[石岡農會碾米穀倉|石岡農會碾米穀倉]]位於萬安-{里}-石岡街區忠孝街，建於日治[[昭和|昭和]]十七年（1942年），為日式杉木結合編竹夾泥牆建築。近年來石岡稻田面積已逐漸減少，該建築實為農業社會時期的歷史見證。其木製結構頗為精密，係日本技師經歷昭和十年（1935年）[[關刀山大地震|關刀山大地震]]後採用的防震結構工法施作成果，在1999年的[[921大地震|921大地震]]中並無重大毀損，獲得建築學界的重視。2002年4月公告為歷史建築，經整修後於2005年7月1日正式對外開放<ref>{{Cite web |url=http://www.tchac.taichung.gov.tw/monuments/Details.aspx?Parser=99,5,28,,,,93,,,,,,,,,,,,,,3,1 |title=《石岡農會碾米穀倉》，臺中市文化資產處官方網站 |accessdate=2012-11-03 |archiveurl=https://web.archive.org/web/20130507055949/http://www.tchac.taichung.gov.tw/monuments/Details.aspx?Parser=99,5,28,,,,93,,,,,,,,,,,,,,3,1 |archivedate=2013-05-07 |deadurl=yes }}</ref><ref>{{Cite web |url=http://www.hakka.gov.tw/ct.asp?xItem=11890&ctNode=1922&mp=1914 |title=《石岡農會日式穀倉》，客家委員會休閒產業網站，客家委員會客家文化發展中心 |access-date=2012-11-03 |archive-url=https://web.archive.org/web/20150924024848/http://www.hakka.gov.tw/ct.asp?xItem=11890&ctNode=1922&mp=1914 |archive-date=2015-09-24 |dead-url=yes }}</ref>。

[[挑物古道|挑物古道]]位於金星-{里}-東側與九房-{里}-、龍興-{里}-交界處，是昔日石崗仔-{庄}-往來仙塘坪-{庄}-（今龍興-{里}-）的一條山徑，2001年底重新整修為登山步道。步道起點位於石忠宮旁，終點位於[[五福臨門神木|五福臨門神木]]，全長約4.7公里。全程最陡峭的路段稱為「瘦身坡」，可考驗登山者的體力<ref>[http://travel.tai-chung.com.tw/index_m.php?ptype=ieb_c&L3_id=35101 《挑物古道》，美美網台中旅遊網]</ref>。
