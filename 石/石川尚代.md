'''石川尚代'''，汉名'''韩晓清'''，日籍华人，现任株式会社日中新闻社社长、曾与《人民日报海外版》有合作关系。人民日报海外版已于2012年8月29日向株式会社日中新闻社发出《终止合作通知》的告知函，终止与日中新闻社以及韩晓清本人的一切合作关系，并告知不得盗用人民日报海外版的名义开展任何活动。但是韩晓清被发现仍旧多次盗用人民日报海外版的名义从事活动<ref>{{cite web|url=http://world.people.com.cn/n1/2019/0216/c1002-30715701.html|title=人民日报海外版严正声明|agency=人民网|access-date=2019年2月16日}}</ref>。

== 生平 ==
石川尚代出生于[[中国大陆|中国大陆]]，后来留学[[日本|日本]]，并且加入日本籍。<ref name=" bull" /> 在其“保钓害国论”受到舆论抨击后，石川尚代自己宣称她仍然是中国国籍<ref>{{cite news|url=http://opinion.huanqiu.com/1152/2012-09/3094310.html|title=日中新闻社韩晓清致函环球网回应“保钓害国论”|agency=环球网|date=2012年09月03日}}</ref>，虽然事实上她已经持有日本国籍<ref name=" bull" />。

2000年8月，石川尚代在日本创办了“日中新闻社”，担任社长。<ref name=" fish" >{{cite web |url=http://news.sohu.com/20080206/n255090702.shtml |title=团结合作襄大业　中华儿女庆新春 之 “日本《日中新闻》代表取缔役　韩晓清”|accessdate= 2008年2月6日|publisher=搜狐网 |work= |language =zh-cn}}</ref>

2007年9月，《人民日报》和“日中新闻社”合作，石川尚代出任《人民日报》日本支社社长。<ref name=" fish" />

2012年8月29日，人民日报海外版已于向株式会社日中新闻社发出《终止合作通知》的告知函，终止与其合作协议<ref>{{cite web|url=http://world.people.com.cn/n1/2019/0216/c1002-30715701.html|title=人民日报海外版严正声明|agency=人民网|access-date=2019年2月16日}}</ref>。但是日本日中新闻社网站和韩晓清（石川尚代）被发现其仍然在盗用人民日报海外版的名义，迫使人民日报海外版分别在2012年9月28日、2016年4月7日、2019年2月16日，数次发表严正声明，人民日报海外版已经终止与日中新闻社以及韩晓清本人的一切合作关系，并告知不得盗用人民日报海外版的名义开展任何活动。韩晓清被发现其仍在盗用人民日报海外版的名义从事活动，对此人民日报海外版宣称将保留依照中国法律、法规和日本当地法律向日中新闻社和韩晓清本人追究法律责任的权利。

== 思想 ==
石川尚代虽然出生在中国大陆有华人血统，但是已经加入日本国籍，言论和行动都代表日本利益。<ref name=" bull" >{{cite web |url= http://opinion.hexun.com/2012-08-30/145293900.html|title=文汇报：韩晓清已认贼作父 保钓害国论不值一驳 |accessdate=2012年8月30日 |publisher=和讯 |work= |language =zh-cn}}</ref><ref>{{cite web|url=http://www.takungpao.com/sy/2012-08/30/content_1015855.htm|title=成报：韩晓清认贼作父 全民痛斥保钓害国论|accessdate=2012-08-30|publisher=大公网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20160304203845/http://www.takungpao.com/sy/2012-08/30/content_1015855.htm|archivedate=2016-03-04}}</ref>

2012年，[[启丰二号|启丰二号]]保钓事件，石川尚代在“日中新闻社”发表文章《保钓登岛破坏国家战略 是害国行为》，提出“保钓害国论”，文章认为中国目前的主要任务是发展经济，并且追忆中国经济现代化是“没有后来的日本的日元借款（[[ODA|ODA]]），也就没有现在的中国现代化”，最后得出结论“笔者认为香港、台湾、大陆所谓的保钓人士所做的登陆钓鱼岛行动，不管他们出于什么动机，都不是爱国行为，而是害国行为。”<ref>{{cite web |url= http://news.qq.com/a/20120828/000388.htm|title=韩晓清：陆港台保钓人士登岛不是爱国而是害国 |accessdate=2012年8月27日 |publisher= 腾讯网|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://news.qq.com/a/20120828/000917.htm |title=日本华媒：保钓登岛破坏国家战略 是害国行为 |accessdate=2012年8月28日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>
