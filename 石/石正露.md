'''石正露'''（{{bd|1963年|2月||}}），[[湖北|湖北]][[大冶|大冶]]人，[[中国人民解放军中将|中国人民解放军中将]]。

曾任[[中国人民解放军第54集团军|陆军第54集团军]]第127师副师长。2008年任装甲第11师师长，次年升任54集团军参谋长。2012年11月，任陆军第54集团军副军长。2014年9月，升任[[济南军区|济南军区]]副参谋长。2014年12月，任54集团军军长<ref>{{cite web |title=石正露履新第54集团军军长 曾任济南军区副参谋长 |url=http://news.ifeng.com/a/20150327/43431216_0.shtml |website=凤凰网 |accessdate=2018-07-16 |date=2015-03-27}}</ref>。2017年1月，升任[[北部战区|北部战区]]副司令员兼参谋长<ref>{{cite web |title=原54军军长石正露任北部战区副司令员兼参谋长 |url=http://news.youth.cn/sz/201708/t20170801_10416288.htm |website=中国青年网|date=2017-08-01 |accessdate=2018-07-16}}</ref>。

2010年7月，晋升少将军衔。2018年7月，晋升中将军衔<ref>{{cite web |title=候补中委、北部战区副司令员兼参谋长石正露晋升中将军衔 |url=https://www.thepaper.cn/newsDetail_forward_2310094 |website=澎湃新闻 |accessdate=2018-08-01}}</ref>。
