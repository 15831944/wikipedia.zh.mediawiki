'''维生素A'''（{{lang-en|Retinol}}）又称为'''视黄醇'''是維生素A的動物形式之一，它是一個[[二萜|二萜]]和[[醇|醇]]的結構，它可以轉換為其他形式的維生素A，並且以醇的衍生物視黃酯充當動物中維生素的儲存形式。

當轉換為視黃醛的形式，維生素A對視力是必不可少的，當轉換成視黃酸對皮膚健康，牙齒礦化和骨骼生長所必需。這些化學化合物統稱為類維生素A，並具有全反式視黃醇作為其結構的一個結構基序的共同特徵。在結構上類維生素A還具有一個β–[[紫羅蘭酮|紫羅蘭酮]]環和不飽和側鏈，與任一醇、醛、羧酸基或酯基。側鏈由四個類[[異戊二烯|異戊二烯]]單元組成，以一系列可能存在於反式或順式構型的共軛雙鍵。

它是人体必需的13种[[维生素|维生素]]之一，是一种脂溶性[[抗氧化剂|抗氧化剂]]，在人体中可以维持[[视力|视力]]并且促进[[骨骼|骨骼]]成长。

維生素A並非單一的一種化合物，而是有許多不同的型態。动物能将[[胡萝卜素|胡萝卜素]]在体内转化为维生素A贮藏在[[肝脏|肝脏]]中；通常是以[[醇|醇]]類的方式存在，稱作[[視黃醇|視黃醇]]，活性也是最高；但也有一些屬於[[醛|醛]]類，稱作[[視黃醛|視黃醛]]；另外還有一些屬於[[酸|酸]]類，稱作[[視黃酸|視黃酸]]。

視黃醇與視黃醛主要掌管桿細胞的視覺循環，而視黃酸主要是掌管人體內上皮組織分化有關，因此有些視黃酸衍伸物（俗稱的A酸）常用於皮膚疾病上的治療，另外有一種稱作視黃酯，其為人體內儲存脂溶性維他命A的主要型式。

== 发现历史 ==
古代[[埃及|埃及]]很早就注意到了维生素A缺乏症的症状，[[中国|中国]]传统[[中医|中医]]也注意并研究了它并且找到了解决办法。早在[[唐朝|唐朝]][[唐太宗|太宗年间]]，[[孙思邈|孙思邈]]就在《[[备急千金要方|备急千金要方]]》中记载了用富含[[胡萝卜素|β-胡萝卜素]]的中草药配合羊肝来治疗[[夜盲症|夜盲症]]的药方。这是较早的关于维生素A的应用的研究，不过那时候的[[医生|医生]]尚不知道究竟是动物肝脏中的什么成分对夜盲症有治疗作用，更不会分析其结构与生化性质。到了17世纪，西方医生也鼓励病人多食用肝脏来治疗夜盲症。

1831年，科学家从植物的黄叶中提取出了[[胡萝卜素|胡萝卜素]]。它可以在体内转化为维生素A，也被称为维生素A原、原维生素A或维生素A前体。

1912年—1914年之间，[[美国|美国]]科学家[[埃尔默·麦考伦|埃尔默·麦考伦]]（Elmer V. McCollum）和玛格丽特·戴维斯（Marguerite Davis）在[[威斯康星州|威斯康星州]]发现动物脂肪或[[鱼肝油|鱼肝油]]的醚提取物可以促进[[老鼠|老鼠]]的生长，在缺少它的10周内，老鼠体重会迅速下降，他们认为这是一种脂溶性维生素。因为之前弗雷德里克·霍普金斯（Frederick Gowland Hopkins）爵士已经发现了谷物中含有可以影响动物和人类生长的维生素，所以他们把这种脂溶性的维生素称为A因子，把谷物中的称为B因子，这也是第一次对维生素进行系统命名。1920年，它被正式命名为维生素A，1933年維生素A的化學性質被確定，1947年，科學家研究出合成維生素A的方法。

[[瑞士|瑞士]][[化学家|化学家]][[保罗·卡勒|保罗·卡勒]]和[[德国|德国]]化学家[[理查德·库恩|理查德·库恩]]因对[[胡萝卜素|类胡萝卜素]]和维生素A的研究而分别于1937年和1938年获得[[诺贝尔化学奖|诺贝尔化学奖]]。1967年，美国生物化学家[[乔治·沃尔德|乔治·沃尔德]]因研究视觉的生物化学过程，其中就包括了对维生素A的各种衍生物的研究，获得了[[诺贝尔生理学或医学奖|诺贝尔生理学或医学奖]]。

== 生产方法 ==
=== 天然产品 ===
天然的维生素A由[[鱼肝油|鱼肝油]]提取。其中海洋鱼类肝脏提取到的是视黄醇（维生素A{{Sub|1}}），淡水鱼类肝脏中提取到的是3-脱氢视黄醇，即维生素A{{Sub|2}}。
=== 人工合成 ===
视黄醇及其衍生物的人工合成路线有很多条，常见的有Isler路线，即{{碳}}{{Sub|13}}-C{{Sub|14}}-C{{Sub|20}}。

[[柠檬醛|柠檬醛]]和[[丙酮|丙酮]]在碱性条件下进行羟醛缩合，得到假紫罗兰酮，在[[硫酸|硫酸]]作用下环合，成为[[紫罗兰酮|紫罗兰酮]]，其中β-紫罗兰酮就是路线中所指的C{{Sub|13}}。C{{Sub|13}}采用Darzens缩合得到缩水甘油酯，再水解、脱羧重排得到C{{Sub|14}}醛。再与C{{Sub|6}}醇制得的格氏试剂缩合，得到C{{Sub|20}}，这就是视黄醇的主链。经过一系列重排，最终得到视黄醇。

== 生化过程 ==
[[File:Chem_157_vitamin_a_synthesis_project.png|thumb]]
从食物中摄取的维生素A是[[前体|前体]]形式，其中，来自动物性来源（如[[鱼肝油|鱼肝油]]、[[牛奶|牛奶]]等）的是视黄醇[[酯类|酯]]的形式，而从植物性来源（如[[胡萝卜|胡萝卜]]、[[菠菜|菠菜]]等）得到的是[[胡萝卜素|β-胡萝卜素]]。

在体内酶的作用下，发生下列生化反应：
:::  β-胡萝卜素 + O<sub>2</sub> = 2分子视黄醇
:::  视黄醇 + NAD<sup>+</sup>（烟酰胺腺嘌呤二核苷酸） = 视黄醛 + NADH +  H<sup>+</sup>
:::  视黄醛 + O<sub>2</sub> + H<sub>2</sub>O = 视黄酸 + H<sub>2</sub>O<sub>2</sub>

其中，视黄醇和视黄醛之间可以互相转化，而氧化为视黄酸的步骤在体内是[[可逆反应|不可逆]]的。

== 生理作用 ==
在人体中的作用：
构成[[视觉|视觉]]细胞内感光物质：全反式视黄醛可以被视黄醛异构酶催化为4-顺式-视黄醛，4-顺-视黄醛可以和视蛋白结合成为视紫红质（rhodopsin）。视紫红质遇光后其中的4-顺-视黄醛变为全反视黄醛，因为构像的变化，引起对视神经的刺激作用，引发视觉。而遇光后的视紫红质不稳定，迅速分解为视蛋白和全反视黄醛，重新开始整个循环过程。

维持上皮结构的完整与健全：视黄醇和视黄酸可以调控基因表达，减弱上皮细胞向鳞片状的分化，增加上皮生长因子受体的数量。

促进生长、发育：这也和视黄醇对基因的调控有关，并且视黄醇具有相当于类固醇激素的作用，可促进糖蛋白的合成。

== 维生素A缺乏 ==
=== 条件 ===
当遇到以下情况的时候，会发生维生素A缺乏：
* 食物缺乏维生素A的来源，如不食用动物内脏或富含胡萝卜素的食品。
* 消化道疾病，造成对维生素A的吸收困难。
* [[肝臟#肝臟疾病|肝脏疾病]]，造成[[肝臟星狀細胞|星狀細胞]]对维生素A的储存困难。
* 需求量突然增加，如儿童生长、孕妇和哺乳期妇女需要。

=== 症狀 ===
維生素A關係到眼睛內[[視紫質|視紫質]]的合成，因此維生素A缺乏時，眼睛方面會出現明顯的症狀。維生素A缺乏會表現以下症狀：
* [[夜盲症|夜盲症]]、[[乾眼症|乾眼症]]、視力下降
* 皮膚乾燥、角質化
* 兒童生長發育緩慢
* 免疫力降低
* 貧血

=== 治療 ===
如果發生维生素A缺乏，主要是透過食物進行補充，或遵照醫囑服用[[魚肝油|魚肝油]]等藥物進行治療。

== 摄入过量 ==
当摄入量超过每日所需后，多余的维生素A会储存在肝脏和脂肪中，在缺乏的时候缓慢释放。而当摄入量大大超过所需，就会引起中毒现象。

=== 急性中毒 ===
当每日摄入量大于100mg的时候，会引发急性中毒，表现为：
* [[颅压|颅压]]升高，头痛，呕吐
* 倦睡，[[烦躁|烦躁]]
* [[脱皮|脱皮]]
* 严重的会导致死亡

=== 慢性中毒 ===
长期大剂量服用维生素A会导致慢性中毒：
* 易激动、易疲劳、头痛、恶心、呕吐、肌肉无力、坐立不安
* 食欲降低、腹泻、腹痛、[[黄疸|黄疸]]、肝脾肿大
* 还有可能引起[[骨质疏松|骨质疏松]]或胎儿[[畸形|畸形]]
