'''凤庆县'''是[[中国|中国]][[云南省|云南省]][[临沧市|临沧市]]辖[[县|县]]，亦称蒲门<ref>{{cite web |url=http://www.ynfq.gov.cn/fqxrmzfmh/5406855026658246656/20061129/87481.html |title=历史沿革 |accessdate=2008-04-23 |date=2004-04-07 |language=zh-hans |archive-url=https://web.archive.org/web/20070906111530/http://www.ynfq.gov.cn/fqxrmzfmh/5406855026658246656/20061129/87481.html |archive-date=2007-09-06 |dead-url=yes }}</ref>，是多民族聚居的贫困县。

== 历史沿革 ==
[[唐朝|唐朝]]，凤庆属[[姚州_(云南)|姚州]]。[[宋朝|宋朝]]时，属永昌。[[元朝|元朝]][[泰定_(元朝)|泰定]]四年（1327年），凤庆内附并设[[顺宁府_(泰定)|顺宁土府]]。[[天顺_(元朝)|天顺]]元年（1328年），设顺宁府宝通州、[[庆甸县|庆甸县]]。[[万历|万历]]二十六年（1598年），改为[[流官|流官]]制，称顺宁府。[[清朝|清朝]][[乾隆|乾隆]]三十五年（1770年），置[[顺宁县|顺宁县]]。[[民国|民国]]2年（1913年）废府留县，先隶[[迤南道|迤南道]]，后属[[迤西道|迤西道]]。民国18年，裁撤道署，直属云南省府。民国31年属[[蒙化第五行政区|蒙化第五行政区]]。民国38年属蒙化第十一行政区。

1950年2月22日，顺宁县人民政府成立。1953年3月属[[大理专区|大理专区]]。1954年8月26日，顺宁县改名为凤庆县。1956年改属[[临沧地区|临沧地区]]，1959年与[[云县|云县]]合并后又分开，分设凤庆建制。

== 地理 ==
凤庆境内[[山脉|山脉]]，属[[怒山|怒山]]、[[云岭|云岭]]两大[[山系|山系]]。澜沧江以南属怒山山系，澜沧江以北属云岭山系。

== 农业 ==
凤庆县农业以[[茶叶|茶叶]]、[[核桃|核桃]]为主<ref>张兴旺.核桃成为凤庆县农林业第二骨干产业[J].中国果业信息，2007，24(12)：33-33.</ref>。

凤庆县具有丰富的茶文化传统，是盛产茶叶的地方，有驰名中外的“滇红茶乡”之称<ref> {{cite web|url=http://www.fq.lcagri.gov.cn/contect.asp?id=277&subject=%C5%A9%D2%B5%B8%C5%BF%F6&name=%B7%EF%C7%EC%B5%C4%C5%A9%D2%B5%B8%C5%BF%F6 |title=凤庆农业概况 |accessdate=2008-05-05 |language=zh-hans }}</ref>。通过普查统计，全县古茶树资源面积56000多亩<ref>许文舟 周武.云南凤庆古茶树记[J].福建茶叶，2006(3)：46-46.</ref>。风庆县境内的香竹箐，有一株茶树高10．6 m，围粗5．82m。据推断，这株茶树已经有超过3000年历史，并被尊称为世界茶王之母<ref>陈开心.茶树起源在临沧[J].蚕桑茶叶通讯，2006(3)：30-31.</ref>。此外，凤庆县还有着独特的传统茶俗 - [[百抖茶|百抖茶]]。百抖茶亦称土法功夫茶<ref>许文舟.云南风庆百抖茶[J].广东茶业，2007(3)：37-37.</ref>，因其制作颇费功夫而得名。

凤庆一直以来都有种植核桃的传统。2004年12月，国家林业局授予凤庆县“中国核桃之乡”称号。

== 旅游 ==
* [[文庙|文庙]] - 位于凤庆县城西文庙街，始建一明万历三十四年（1606 年），清康熙八年（1669年）迁建。
* [[保石洞寺|保石洞寺]] - 位于凤庆县城东南30千米洛党乡箐头村。建于清乾降六十年（1795年），1926年重修。
* [[青龙桥|青龙桥]] - 位于凤庆县城东北约60千米的澜沧江上。始建于清乾隆二十六年（1761年），嘉庆十九年（1814年）、道光二十四年（1844年）两次复修。青龙桥是澜沧江上现顾惟一的古桥。
* [[玉龙池|玉龙池]]和[[黑龙潭|黑龙潭]] - 位于小湾镇与凤山镇交界的山顶上，海拔分别在2400—2860以上，现属澜沧江省级自然保护区。
* [[庆甸遗址|庆甸遗址]] － 位于麦地村明王室（寺）后的老君山峰献山顶，为元朝天顺元年（1328）所置勐氏衙门。
* [[勐氏石城|勐氏石城]] - 为勐氏部落城堡遗址，位于新华乡牛尘山中部。
