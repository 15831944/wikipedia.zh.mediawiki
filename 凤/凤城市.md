'''凤城市'''位于[[中国|中国]][[辽宁省|辽宁省]]东部，是[[丹东市|丹东市]]代管的一个[[县级市|县级市]]，中国第一个由少数民族[[自治县|自治县]]撤县建市，并保留民族自治地方的待遇和各项优惠民族政策的城市。<ref>{{cite journal |author1=金炳镐 |author2=田烨 |title=新世纪中国民族区域自治制度创新的一个亮点——“民族自治市” |journal=西北民族大学学报(哲学社会科学版) |date=2007 |issue=05 |pages=42-54}}</ref>

== 历史 ==
[[明朝|明朝]]后期将定辽右卫从辽阳迁于凤凰堡。[[清朝|清朝]][[光绪|光绪]]二年（1876年）设立凤凰直隶厅，辖岫岩州、安东县、宽甸县。[[民国|民国]]二年（1913年）改凤凰县；因与湖南省凤凰县同名，1914年改为凤城县。1985年撤销凤城县，设立凤城满族自治县；1994年撤销凤城满族自治县，设立凤城市。

== 行政区划 ==
下辖3个[[街道办事处|街道办事处]]、17个[[镇_(中华人民共和国)|镇]]、1个[[民族乡|民族乡]]<ref>{{cite web |url=http://dm.dandong.gov.cn/qhsc.htm |title=存档副本 |accessdate=2009-11-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20091127085859/http://dm.dandong.gov.cn/qhsc.htm |archivedate=2009-11-27 }} 2009年丹东市行政区划统计表</ref>：
*街道：凤凰城街道、凤山街道、草河街道。
*镇：青城子镇、通远堡镇、爱阳镇、赛马镇、鸡冠山镇、边门镇、红旗镇、弟兄山镇、大堡镇、东汤镇、刘家河镇、[[宝山镇_(凤城市)|宝山镇]]、蓝旗镇、白旗镇、四门子镇、石城镇、沙里寨镇、大兴镇。
*民族乡：大堡[[蒙古族|蒙古族]]乡。

== 交通 ==
凤城交通便捷，有[[沈丹客运专线|沈丹客运专线]]、[[沈丹铁路|沈丹铁路]]、[[凤上铁路|凤上铁路]]、[[304国道|304国道]]、[[丹阜高速公路|丹阜高速公路]]贯穿。

== 旅游 ==
凤凰山为国家风景名胜区，有“泰山之雄伟，华山之险峻，庐山之幽静，黄山之奇观，峨眉之秀美”。除外，凤城市境内还有玉龙湖、浦石河原始森林、大梨树花果山、赛马溶洞群、爱河鸽子洞、东汤温泉和古遗址、古城堡等旅游景点。
