==剧情==
该剧讲述三对[[80后|80后]]夫妻（王明轩和蒋新瑶、赵亚彤和文浩、黎昕和张小凡）为家庭奋斗的历程。<ref name=" cat" />

== 花絮 ==
《离婚前规则》、《[[新女婿时代|新女婿时代]]》、《[[岳母的幸福生活|岳母的幸福生活]]》、《[[小麦进城|小麦进城]]》、《[[AA制生活|AA制生活]]》、《[[金太狼的幸福生活|金太狼的幸福生活]]》被誉为2012年六大婚斗剧。<ref>{{cite web |url=http://news.163.com/12/1213/11/8IJQMMU300014JB6.html |title=《离婚前规则》《新女婿时代》盘点2012六大“婚斗”剧 |accessdate= 2012-12-13|publisher=网易 |work= |language =zh-cn}}</ref>

该剧引发80后婚姻“离婚”、“恐婚”、“隐婚”、“[[富二代|富二代]]婚姻”的讨论。<ref>{{cite web |url=http://ent.ifeng.com/tv/news/rjdp/detail_2012_12/14/20174816_0.shtml |title=《离婚前规则》：离婚成了拯救婚姻的最后防线？ |accessdate=2012年12月14日  |publisher=凤凰网 |work=北青报唐书钰 |language =zh-cn}}</ref><ref>{{cite web |url=http://news.hexun.com/2012-12-12/148921918.html |title=《离婚前规则》引发观众“恐婚”争议 |accessdate=2012年12月18日  |publisher= 和讯网|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://ent.qq.com/a/20121127/000018.htm |title=《离婚前规则》浙江独播 六大关键词解读剧集 |accessdate= 2012-11-27 |publisher=腾讯网 |work= |language =zh-cn}}</ref>
