《'''引路菩萨图'''》是一幅于敦煌[[莫高窟|莫高窟]]藏经洞发现的唐代绢画，作者未知，1907年由英国考古学家[[斯坦因|斯坦因]]购入，现藏于英国[[大英博物馆|大英博物馆]]第33号展厅<ref>{{cite web |url=http://britishmuseum.org.cn/exhibition.aspx?id=136 |title=引路菩萨图 |last= |first= |date= |website=大英博物馆中文网 |publisher= |access-date= |quote=}}</ref>。绢画高80.5厘米，宽53.8厘米，画面相当完整。

== 概述 ==
画面以浅褐色为底，右上角有文字题记“引路菩”。菩萨右手持手提型[[香炉|香炉]]，左手持莲花幢幡，唇上有蝌蚪状[[髭鬚|髭鬚]]。左上角有彩云围着的阁楼建筑，表示[[极乐世界|极乐世界]]<ref>{{cite web |url=http://public.dha.ac.cn/zhuanti/hhdg/juanhua.html#p=9 |title=敦煌藏经洞绢纸画临摹复制品 |last= |first= |date=2013-12-28 |website=敦煌研究院 |publisher= |access-date= |quote=}}</ref>。右边为[[往生|往生]]的女信徒，身穿红色唐代贵妇服饰，身量是菩萨的三分之一，但尚不清楚女信徒的身份<ref>{{cite web |url=http://fo.sina.com.cn/culture/painting/2014-01-24/142515950.shtml |title=敦煌度亡图：引路菩萨 |last= |first= |date=2014-01-24 |website=新浪佛学 |publisher= |access-date= |quote=}}</ref>。

引路菩萨是引导亡灵到西方极乐世界的菩萨，起源于[[净土信仰|净土信仰]]。在[[敦煌|敦煌]]发现的引路菩萨像有11幅<ref>{{cite web |url=http://public.dha.ac.cn/content.aspx?id=808245221494 |title=敦煌画中的引路菩萨 |last= |first= |date=2010-03-26 |website=敦煌研究院 |publisher= 王惠民|access-date= |quote=}}</ref>。
