'''夜貓組'''，成立於2015年，成員包含LEO王，春艷，為[[顏社|顏社]]旗下饒舌演唱組合。

== 成立 ==
LEO王原先為搖滾樂隊，「巨大的轟鳴」主唱，春艷於戰犯音樂也發表了一些[[饒舌|饒舌]]作品於網路上，兩位接觸的契機是由於LEO王在網路上看Diss RBL的比賽，一場春艷 vs. [[熊仔|熊仔]]的battle中<ref name=":0">{{Cite news|url=https://www.marieclaire.com.tw/celebrity/story/37293/page_2|title=【金曲新勢力】夜貓組，新世代饒舌奇才|author=Marie Claire|date=2018/06/25|work=Marie Claire|accessdate=}}</ref>，知道了春艷，對於春艷在作品上的態度頗為欣賞，後於Revolver的現場演出時，彼此見面。春艷雖然感覺對方有點古怪<ref name=":0" />，但在後期網路上的音樂交流後，彼此對於對方的作品都很有興趣，因此開啟共同創作之路。其後合作單曲被顏社的老闆迪拉聽到<ref name=":1">{{Cite web|url=http://www.oneday.com.tw/%E8%9B%8B%E5%A0%A1x%E5%A4%9C%E8%B2%93%E7%B5%84/|title=【蛋堡 X 夜貓組】有隊友的嘻哈創作 像進入精神時光屋|accessdate=2018/03|author=駱亭伶|date=2018/03|publisher=《小日子》}}</ref>，認為可以成立饒舌組合，「夜貓組」便正式成立。作品音樂風格多元，橫跨爵士、藍調與雷鬼聲響，並有許多新穎的嘗試<ref>{{Cite web|url=https://www.elle.com/tw/entertainment/voice/a19174007/softlipa-yeemao-kaoinc-music/|title=蛋堡X夜貓組：如果有一件事情讓你只想要好好把它做好，這就是宇宙給你的啟示！|accessdate=2018/03/12|author=ELLE|date=|publisher=ELLE}}</ref><ref>{{Cite web|url=http://www.iwant-radio.com/showinformation.php?sisn=41217|title=夜貓組(Leo王+春艷)『健康歌曲』專輯介紹|accessdate=|author=銀河網路電台|date=|publisher=}}</ref><ref name=":2">{{Cite web|url=https://www.gq.com.tw/entertainment/culture/content-34125.html|title=一張聽了會變ㄎㄧㄤ的專輯 夜貓組的獵奇音樂哲學|accessdate=2017/10/27|author=Eric|date=2017/10/27|publisher=GQ}}</ref>。

== 團名 ==
「夜貓組」來源於諧音[[夜猫子|夜貓族]]，泛指所有晚睡的人，在晚間，可以獨自的做自己喜歡的事，不被日常的瑣事打擾。由於成員於組成前期，白天皆有日班工作，利用晚上時間進行創作，並交換Demo，直到音樂工作有足夠的收入養活自己，因此將夜貓族的概念引申出「夜貓組」，代表可以做自己喜歡的事情，不被其他的外務打擾<ref name=":2" />。

== 成員<ref name=":1" /> ==
'''春艷''' 
本名余承恩，生日：1992.12.26，隸屬戰犯音樂。

個人作品：《大男孩主義》2014-12、《鬧青》2015-3

於2013年參加「Diss RBL」，一場與饒舌歌手熊仔的battle受到矚目，後發表多首個人作品，並涉足影像、雜誌、派對活動等其他領域。
創立YouTube頻道:lasaboyz拉薩男孩

'''Leo王''' 

本名王之佑，生日：1993.10.29，隸屬顏社。

個人作品：《藝術家脾氣》2016-12

臺大社會系肄業，曾為「巨大的轟鳴」主唱，後轉戰饒舌圈，與9m88 合作〈陪妳過假日〉，以輕鬆詼諧並充滿律動的風格，打響知名度<ref>{{Cite web|url=http://dashi.streetvoice.cn/article/20171103/003/|title=夜猫组：我们的怪是天生的|accessdate=2017/11/04|author=Blow吹音樂|date=2017/11/04|publisher=Streetvoice}}</ref>。
