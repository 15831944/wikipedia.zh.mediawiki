《'''夜孔雀'''》（{{lang-fr|'''Le Paon de Nuit'''}}）是[[戴思杰|戴思杰]]导演的一部中国[[爱情片|浪漫剧情片]]。<ref name=Cremin>{{cite web|url= http://www.filmbiz.asia/news/dai-shijie-set-to-wrap-contemporary-drama|title= Dai Shijie set to wrap contemporary drama|author= Stephen Cremin|date= February 9, 2015|accessdate= February 10, 2015|work= [[Film_Business_Asia|Film Business Asia]]|deadurl= yes|archiveurl= https://web.archive.org/web/20150210170518/http://www.filmbiz.asia/news/dai-shijie-set-to-wrap-contemporary-drama|archivedate= 2015年2月10日}}</ref>是一部中法联合制作的电影。[[刘亦菲|刘亦菲]]、[[刘烨_(演员)|刘烨]]、[[余少群|余少群]]和[[黎明|黎明]]主演。中国于2016年5月20日公映。<ref name=飞鸟凉>{{cite web |url= http://news.mtime.com/2016/04/12/1554347.html|title= 《夜孔雀》定档520曝海报预告 刘亦菲谈激情戏：古装剧是小学阶段，我早就长大了|author= 飞鸟凉|date= April 12, 2016|accessdate= April 12, 2016|work= [[news.mtime.com|news.mtime.com]]}}</ref>

== 影片简介 ==
[[法國|法國]]籍華裔[[長笛|長笛]]手埃爾莎（劉亦菲飾）從[[巴黎|巴黎]]到[[成都|成都]]跟三位不同的男人演繹了浪漫的愛情故事。

法国籍华人音乐系留学生埃爾莎（劉亦菲飾）在成都愛上了蜀錦研究員、尺八高手馬榮（黎明飾），两人一起寻找着理想的蠶——夜孔雀。川剧学校学生马林暗恋上了艾尔莎。当马林发现他的父亲与艾尔莎之间的恋爱关系后，就发生了自残行为。后来当父子两人在戏台上表演川剧时，马林拿了一瓶毒药想喝下自杀被马荣阻止，然而马荣却喝下了它。回到巴黎的艾尔莎又认识了紋身師建民（劉燁飾），建民要为她在后背上纹一只夜孔雀，但是又因[[签证|签证]]过期面临要被法国警察驱逐回国。埃爾莎因怀上了马荣的孩子起初想要做[[流产|流产]]手术。最后艾尔莎决定将肚里的孩子生下来，并以要与建民结婚为由将他留了下来。

== 制作 ==
这部影片在法国[[巴黎|巴黎]]和中国成都拍摄。<ref name=Coonan/><ref name=Cremin/>

主题曲《为爱成泥》由[[窦鹏|窦鹏]]作曲，张文和戴思杰作词，余少群演唱。

== 票房 ==
