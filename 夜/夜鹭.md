'''夜鹭'''（[[学名|学名]]：{{lang|la|''Nycticorax nycticorax''}}）又稱'''黑頂夜鷺'''，俗稱'''灰洼子'''、'''夜洼子'''、'''星雁'''、'''暗光鳥仔'''，模樣相近[[綠鷺|綠鷺]]，是[[鹭科|鹭科]][[夜鹭属|夜鹭属]]的一种。

== 生态环境 ==
夜鷺是一種很聰明的鳥類，會用魚餌釣魚 。捕魚時，夜鷺會先把野果扔在水裡，然後在岸上等待。一旦發現獵物，牠就會迅速的衝入水中，飽餐一頓。

== 分布地域 ==
夜鹭是分布非常广泛的一种鸟类，分布遍及[[欧亚大陆|欧亚大陆]]、[[非洲|非洲]]，整个[[美洲|美洲]]大陆及東南亞等地。

*台灣
夜鷺在台灣為常見的留鳥，由於性喜食魚，魚類養殖業者視其為害鳥。

*中國
在[[中国大陸|中国大陸]]全境都能看到它们的身影，在[[中国东北|中国东北地区]]、[[华北|华北]]、[[甘肃|甘肃]]、[[四川|四川]]、[[华中|华中]]、长江流域等省区繁殖，为夏候鸟，在长江以南省份为留鸟。

[[File:Black-crowned_Night_Heron_Baby.jpg|right]]

== 特征 ==
夜鹭雄雌同形同色，'''成鸟'''体长约40厘米至65厘米，頭、枕部略带金屬光泽的深蓝灰色，上体部分和双翅为暗灰色，下体略带乳黄色的白色。頭顶上有兩到三根细長的白色[[羽毛|蓑羽]]。虹膜是血红色，[[喙|喙]]黑色足黄色。'''幼鸟體'''長跟成鳥一樣，上體灰褐色，翅膀上有星星點點分布的白色斑點。

== 亞種 ==
*'''夜鷺指名亞種'''（[[学名|学名]]：{{lang|la|''Nycticorax nycticorax nycticorax''}} <small>（[[Linnaeus|Linnaeus]], 1758）</small> ）：分布最廣的亞種，範圍以[[季風亞洲|季風亞洲]]為東界，經[[中亞|中亞]]、[[兩河流域|兩河流域]]、[[東歐|東歐]]南部、[[中歐|中歐]]內陸，西抵[[伊比利半島|伊比利半島]]，南經[[漠南非洲|漠南非洲]]至[[南非|南非]]與[[馬達加斯加|馬達加斯加]]。

*'''夜鷺美洲亞種'''（[[学名|学名]]：{{lang|la|''Nycticorax nycticorax hoactli''}} <small>Gmelin, 1789</small> ）：分布於[[夏威夷群島|夏威夷群島]]，與北緯55度至南緯25度間的[[美洲|美洲]]地區。

*'''夜鷺南安地斯亞種'''（[[学名|学名]]：{{lang|la|''Nycticorax nycticorax obscurus''}} <small>[[夏爾·呂西安·波拿巴|Bonaparte]], 1855</small> ）：主要分布於[[南美洲|南美洲]]南部。

*'''夜鷺福克蘭亞種'''（[[学名|学名]]：{{lang|la|''Nycticorax nycticorax falklandicus''}} <small>Hartert, 1914</small> ）：僅見於[[福克蘭群島|福克蘭群島]]。

== 食物 ==
夜鹭主要取食[[无尾目|蛙类]]、[[鱼|小鱼]]、虾等水生动物，偶尔吃一些植物性食物；受到惊吓的夜鹭会把吃进去的食物呕吐出来，经消化或未经消化的食物腥臭不堪，以此作为防卫，也可以从中得到研究夜鹭食性的线索。

== 繁殖与保护 ==
[[File:Black-Crowned-Night-Heron.jpg|thumb]]
[[File:Nycticorax_nycticorax_MHNT.jpg|thumb]] 
夜鹭集群繁殖，在树上营巢，常结成数百只鸟的大群霸占一片树林营巢，其间经常会有[[小白鹭|小白鹭]]、[[牛背鹭|牛背鹭]]等混群，夜鹭营巢密度极大，甚至一棵树乃至一个枝桠上就聚集了三四个巢，夜鹭巢系由树枝搭建，内部垫以柔软材料。每巢产卵3－8枚由双亲共同抚育，每年6月－7月间可见幼鸟随亲鸟离巢活动。

在中国大陆，夜鹭为常见鸟类。近年来中国大陆各大城市均有夜鹭数量增加的报告，在一些地方甚至泛滥成灾，因此有些地方政府采取毁巢破卵的方式控制夜鹭的数量。究竟是什么因素导致夜鹭数量的增加，其背后是否隐藏着其他环境因素改变的讯号尚须进一步的研究。

== 軼聞 ==
夜鷺閩南語稱為「[[暗光鳥|暗光鳥]]」，客家語稱為「夜鶴」，這個詞也常被代指徹夜未眠的人（同等夜貓子）。

[[2018年|2018年]]，在[[桃園市|桃園市]][[蘆竹區|蘆竹區]]曾有民眾將[[南崁溪|南崁溪]]棲息的夜鷺誤認為[[企鵝|企鵝]]並向警察局報案<ref>[https://udn.com/news/story/7324/3068594 台灣溪邊有企鵝？民眾不識「指夜鷺為企鵝」]</ref>，由於夜鷺的顏色與站姿的確與企鵝十分神似，這起「指鷺為鵝」烏龍也被網友調侃將夜鷺改名為'''台灣企鵝（南崁企鵝）'''。
