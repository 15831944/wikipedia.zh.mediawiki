'''夜之森車站'''（{{jpn|j=夜ノ森駅|hg=よのもりえき|rm=Yonomori eki|f=y}}），是[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[常磐線|常磐線]]的鐵路車站，位於[[福島縣|福島縣]][[双葉郡|双葉郡]][[富岡町|富岡町]]。2002年入選[[東北車站百選|東北車站百選]]。

2011年發生[[東日本大地震|東日本大地震]]，此站雖未受[[海嘯|海嘯]]侵襲，但[[福島第一核電廠事故|福島第一核電廠事故]]的影響，此站位於該廠方圓20公里警戒區之內，[[游離輻射|游離輻射]]較高的歸還困難區域，因此長期停止營業。2012年8月起，站區及常磐線的西側因放射線劑量降低，已改為居住限制區，不再管制人員進出，但禁止過夜；站區本身及常磐線以東仍屬歸還困難區域而管制進出。

==車站結構==
此站為地面車站，設有[[島式月台|島式月台]]一座，有兩股軌道。月台上設有候車室。車站由鄰近的[[富岡車站_(福島縣)|富岡車站]]管理，屬於簡易委託站。

車站週邊在每年春季是熱門的[[賞櫻|賞櫻]]場所。月台及軌道的兩側種植了六千多株[[杜鵑花|杜鵑花]]，以往每年5月開花期間，[[常陸號列車|超級常陸號列車]]通過此站時會放慢車速以讓乘客觀賞<ref>{{cite web| url =http://sankei.jp.msn.com/life/trend/100507/trd1005071642011-n1.htm| title =ツツジの構内を低速運転　福島・ＪＲ夜ノ森駅| publisher =産経新聞| date =2010-05-07| accessdate =2010年5月7日| deadurl =yes| archiveurl =https://web.archive.org/web/20101119131036/http://sankei.jp.msn.com/life/trend/100507/trd1005071642011-n1.htm| archivedate =2010年11月19日}}{{リンク切れ|date=2011年1月}}</ref>。

== 沿革 ==
* 1921年（[[大正|大正]]10年）3月15日：做為[[鐵道省|鐵道省]]營運的車站新開業<ref name="kanpo19210302">大蔵省印刷局、『官報 1921年03月02日（第二千五百七十一號） 鐵道省告示第十九號』、1921年、34頁</ref>。
* 1949年（昭和24年）6月1日：[[日本國有鐵道|日本國有鐵道]]成立。
* 1952年（[[昭和|昭和]]27年）8月5日：車站管轄移交給水戶鐵道管理局<ref name="tomiokachoushi02">富岡町史編纂委員会、「富岡町史 第一巻 通史」、富岡町、1988年、930頁</ref>。
* 1984年（昭和59年）3月1日：做為國鐵合理化的一環，車站營運業務委外給富岡町<ref name="tomiokachoushi02"></ref>。
* 1987年（昭和62年）4月1日：[[國鐵分割民營化|國鐵分割民營化]]後成為JR東日本的車站。
* 2011年（[[平成|平成]]23年）3月11日：因東日本大震災及福島第一核電廠事故而暫停營業。
* 2016年（平成28年）11月：開始進行復舊工程及站內除污作業，伐除站內種植的杜鵑花。
* 2018年（平成30年）6月：因原本的站房相當老舊，且為降低除污後的放射線劑量，計畫將站房拆除，改建為[[跨站式站房|跨站式站房]]<ref>{{Citenews|title=「夜ノ森駅」解体へ 1世紀にわたり富岡町民らの生活支える|url=http://www.minyu-net.com/news/news/FM20180608-277857.php|newspaper=福島民友 みんゆうNet|date=2018-06-08|accessdate=2018-06-17}}</ref>。
* 2019年（平成31年）1月21日：站房拆除工程開工<ref>{{Cite web|date=2019年01月15日|url=https://www.tomioka-town.jp/oshirase/2299.html|title=JR常磐線夜ノ森駅舎の解体について|publisher=富岡町|accessdate=2019-04-08|archiveurl=https://web.archive.org/web/20190222060222/https://www.tomioka-town.jp/oshirase/2299.html|archivedate=2019-2-22}}</ref><ref>{{Cite news|url=https://mainichi.jp/articles/20190121/k00/00m/040/212000c|title=JR常磐線の夜ノ森駅駅舎の解体工事開始 福島第1原発事故後の不通区間|newspaper=毎日新聞|date=2019-01-21|accessdate=2019-01-21|archiveurl=https://archive.fo/DLc9A|archivedate=2019-01-21}}</ref>。
* 2020年（[[令和|令和]]2年）3月：[[富岡車站|富岡車站]]至[[浪江車站|浪江車站]]間路段恢復行駛，車站恢復營運<ref>{{Cite news|date=2016-06-1|url=http://response.jp/article/2016/06/01/276190.html|title=JR東日本、常磐線小高～原ノ町間は7月12日再開…避難指示解除受け|publisher==[[Response.|レスポンス]]|accessdate=2016-07-31}}</ref><ref>{{Cite web |date=2016-3-10 |url=http://www.jreast.co.jp/press/2015/20160307.pdf |title=常磐線の全線運転再開について |format=PDF |publisher=[[東日本旅客鉄道|東日本旅客鉄道株式会社]] |accessdate=2016-7-31}}</ref><ref name="nhk20180608">{{Cite web|url=https://www3.nhk.or.jp/lnews/fukushima/20180608/6050001361.html|title=夜ノ森駅と地域との通路設置へ|accessdate=2018-06-08|date=2018-06-08|publisher=NHK|archiveurl=https://web.archive.org/web/20180609031119/https://www3.nhk.or.jp/lnews/fukushima/20180608/6050001361.html|archivedate=2018-06-09}}</ref>。東西自由通路及跨站站房啟用<ref name="kahoku20180608">{{Cite web|url=https://www.kahoku.co.jp/tohokunews/201806/20180608_61059.html|title=<富岡・夜ノ森駅>自由通路　来年度末までに整備|accessdate=2018-06-08|date=2018-06-08|publisher=河北新報|archiveurl=https://web.archive.org/web/20180612140831/https://www.kahoku.co.jp/tohokunews/201806/20180608_61059.html|archivedate=2018-06-12}}</ref><ref>{{Cite web |date=2019-2-15 |url=http://www.jrmito.com/press/190215/press_03.pdf |title=常磐線夜ノ森駅東西自由通路及び橋上駅舎新設工事について |format=PDF |publisher=[[東日本旅客鉄道水戸支社|東日本旅客鉄道水戸支社]] |accessdate=2019-2-15}}</ref>。
