'''雁塔区'''是[[陕西省|陕西省]][[西安市|西安市]]的一个区，位于西安市南郊，北以南二环路与[[碑林区|碑林区]]相接，以建工路与[[新城区|新城区]]相接，以昆明路与[[莲湖区|莲湖区]]、[[未央区|未央区]]相接；东以[[浐河|浐河]]与[[灞桥区|灞桥区]]相接；西以[[沣惠渠|沣惠渠]]与[[长安区|长安区]]相接；南以杜陵路与长安区相接。区内包括[[曲江新区|曲江新区]]、[[西安高新技术产业开发区|西安高新技术产业开发区]]。名称由著名的[[大雁塔|大雁塔]]而来。雁塔区是西安市的文化、旅游中心，区内高等院校云集，旅游景点丰富。

== 沿革 ==
* 1954年9月设立雁塔区，以原西安市第九区，及原长安县郭杜区的高堡乡，韦曲区的双桥头、金滹沱2乡和东三爻、杜城乡各一部分，大兆区的三兆乡和春临乡的一部分地区为其的行政区域。
* 1960年5月新城区、莲湖区、碑林区的一部分并入雁塔区。1962年4月恢复。
* 1965年9月撤销雁塔区，与[[未央区|未央区]]、原阿房区、[[灞桥区|灞桥区]]合并设立[[西安市郊区|西安市郊区]]。1980年2月恢复。

== 旅游区 ==
雁塔区所辖[[曲江新区|曲江新区]]是雁塔区的旅游区，区内有[[大唐芙蓉园|大唐芙蓉园]]、[[曲江海洋世界|曲江海洋世界]]、[[大唐不夜城|大唐不夜城]]等景点。

雁塔区著名景点有[[大雁塔|大雁塔]]及其[[大雁塔北广场|北广场]]、[[大兴善寺|大兴善寺]]、[[青龙寺|青龙寺]]、[[秦二世胡亥墓|秦二世胡亥墓]]、[[汉宣帝陵|汉宣帝陵]]、[[寒窑|寒窑]]、[[西安植物园|西安植物园]]、[[陕西历史博物馆|陕西历史博物馆]]、[[大唐芙蓉园|大唐芙蓉园]]等。

雁塔区的酒店有[[陕西宾馆|陕西宾馆]]等。
