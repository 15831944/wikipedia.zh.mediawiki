'''湘南学院'''（{{lang-en|'''Xiangnan University'''}}）是位于[[中华人民共和国|中华人民共和国]][[湖南省|湖南省]][[郴州市|郴州市]]的一所高等院校，由原來的郴州师专、郴州医专、郴州教育学院和郴州师范于2003年合併組建而成。<ref>{{cite web |url=http://www.jyb.cn/high/zjzz/201403/t20140310_573336.html |title=湘南学院着力构建公共课特色教学新模式 |accessdate=2014 |publisher=中国教育新闻网 |work= |language=zh }}</ref><ref>{{cite web |url=http://hn.rednet.cn/c/2013/11/20/3203306.htm |title=嘉禾与湘南学院联袂打造嘉禾民歌研究基地 |accessdate=2013 |publisher=红网 |work= |language=zh }}</ref>现有16个系、4个教研部，31个本科专业。学院有二个校区：王仙岭校区、北湖校区。

== 沿革 ==
2003年，郴州师专、郴州医专、郴州教育学院和郴州师范组建在一起，合成“湘南学院”。

四个学校的历史沿革：

1958年，“郴州师专”建立。1962年，郴州师专停办。1978年，[[中华人民共和国国务院|中华人民共和国国务院]]批准恢复了“郴州师专”，以“湖南师范学院郴州分院”当校址。1989年，它改名作“郴州师范高等专科学校”。

1950年，“郴州地区卫生学校”建立。1994年，郴州地区卫生学校升格为“郴州医学高等专科学校”，开设大学专科。

1979年，“郴州地区教师进修学院”建立。1992年，郴州地区教师进修学院改名作“郴州教育学院”。

1912年，“湖南省立第三女子师范学校”建立。1926年，湖南省立第三女子师范学校改名作“湖南省立第三初级中学”。1928年，湖南省立第三初级中学改名作“湖南省立第六中学”，1934年它又改名作“湖南省立衡阳女子中学”，1941年，它扩建作“湖南省立第三师范学校”。1945年，“湖南省立第三师范学校”、“湖南省立第二师范学校”、“湖南省立第十一师范学校”一起并入成了“湘南联合师范”。1945年，学校又扩建作“湖南省立第三师范学校”。1953年，学校改名作“郴州师范学校”，2003年，[[湖南省人民政府|湖南省人民政府]]批准它并入“郴州师范高等专科学校”。

2013年，学院举行了百年校庆。<ref>{{cite web |url=http://hn.ce.cn/hnqyjj/hncz/cznr2/ztch/201312/02/t20131202_1229856.shtml |title=湘南学院举行“百年办学十年建校”庆典 |accessdate=2013 |publisher=中国经济网 |work= |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20160304141947/http://hn.ce.cn/hnqyjj/hncz/cznr2/ztch/201312/02/t20131202_1229856.shtml |archivedate=2016-03-04 }}</ref>

湘南学院现一共有：16个系、4个教研部、36个本科专业、17个专科专业。
