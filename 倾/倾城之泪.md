{{engname|《倾城之泪》|''The Allure of Tears''}}，2011年12月22日上映于[[中国大陆|中国大陆]]的[[爱情电影|爱情电影]]。由香港知名女导演[[黄真真|黄真真]]执导，由李治廷、周冬雨、任贤齐、梁咏琪、窦骁、陈乔恩主出演，2011年7月21日於[[上海|上海]]开机拍摄，9月8日殺青，<ref>[http://ent.163.com/11/0722/07/79I4SAJS000300B1.html 《倾城之泪》开机 周冬雨梁咏琪陈乔恩演三滴泪]2011-07-22 网易娱乐</ref>影片由三颗泪滴组成，三对恋人分别讲述了爱有时间界限、重拾爱情梦想和当爱遭遇生离死别的三段凄美爱情故事。

== 剧情 ==
该片分为三段。每段以“一滴泪”为一个故事。<ref name="smx " >{{cite web |url=http://enjoy.eastday.com/e/20120202/u1a6339750.html |title=《倾城之泪》上演三滴泪 六大明星倾情出演 |accessdate= 2012年|work=|publisher=东方网 |language = zh-hans}}</ref>

第一滴泪是[[血癌|血癌]]患者少女“给力妹”和[[富二代|富二代]][[脑癌|脑癌]]患者“游乐”之间的爱情故事。<ref name="smx " /><ref name="xmrs " />

第二滴泪是[[小提琴|小提琴]][[演奏家|演奏家]]“杨琳”和“丁大可”二人为母校募款的故事，无关爱情。<ref name="smx " /><ref name="xmrs " />

第三滴泪“陳升”和女友“張彩”之间的爱情故事，此故事涉及古代“[[冥婚|冥婚]]”。<ref name="smx " /><ref name="xmrs " />

== 主题 ==
导演黄真真说该片的主题是讲叙两性之间[[情爱|情爱]]。<ref>{{cite web |url=http://et.21cn.com/gundong/etscroll/2011/12/25/10251429.shtml |title=专访导演黄真真：《倾城之泪》是讲爱的电影 |accessdate=2011年 |work= |publisher=21CN |language=zh-hans |archive-url=https://web.archive.org/web/20150512013555/http://et.21cn.com/gundong/etscroll/2011/12/25/10251429.shtml |archive-date=2015-05-12 |dead-url=yes }}</ref>

== 抄袭门 ==
《倾城之泪》的官方海报被网友们指出抄袭2005年一部[[韩国电影|韩国电影]]《[[悲伤电影|悲伤电影]]》，导演[[黄真真|黄真真]]回应说觉得基本相像，但并非有意抄袭。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2012-02-23/04203562083.shtml |title=国产电影海报屡犯“偷懒病” 缺乏创意高度相似 |accessdate=  2012年2月23日|work=|publisher=新浪网 |language = zh-hans}}</ref>
