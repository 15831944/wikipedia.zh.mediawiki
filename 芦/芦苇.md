'''芦苇'''（[[学名|学名]]：''{{lang|la|Phragmites communis}}''），又稱'''普通蘆葦'''（common reed），是生长于[[沼泽|沼泽]]、河沿、海滩等[[湿地|湿地]]的一种[[禾本科|禾本科]][[植物|植物]]，遍布于全世界[[温带|温带]]和[[热带|热带]]地区，芦苇属的植物大约有10[[种|种]]，有的分类学家认为芦苇是芦苇属的唯一[[种|种]]类。

== 形态 ==

[[多年生草本植物|多年生草本植物]]，地下有匍匐的根茎，可以在适合的地区迅速地铺展繁殖，一年可以平铺延伸5[[米_(單位)|米]]以上，地上[[茎|茎]]高达2-6米，丛生，[[叶|叶]]长达20-50厘米，排列成两行，圆锥[[花序|花序]]，花穗长10-40厘米，每个小穗有4-7朵小[[花|花]]，屬[[风媒花|风媒花]]。[[美洲|美洲]]亚种植株比较矮小。

== 多种用途 ==

蘆葦桿含有[[纖維素|纖維素]]，可以用來造[[纸|纸]]和[[人造纖維|人造纖維]]。[[中國|中國]]從古代就用蘆葦编制“葦席”鋪炕、盖房或搭建臨時建築。古代各國都有用蘆葦的空莖制造的[[樂器|樂器]]——[[蘆笛|蘆笛]]，蘆葦莖内的薄膜做[[笛子|笛子]]的笛膜使用。蘆葦穗可以作[[掃帚|掃帚]]，花絮可以充填[[枕頭|枕頭]]。蘆葦的[[根|根]]是一種[[中藥|中藥]]——蘆根，性寒，味甘，功能是清胃火、除肺热，主治熱病烦渴、胃熱嘔吐、肺痛等症。

在[[臺灣|臺灣]][[西拉雅族|西拉雅族]]的傳統裡，會把蘆葦葉或[[檳榔|檳榔]]葉放進壺裡祭拜，也就是所謂的[[祀壺信仰|祀壺信仰]]。

== 故事传说 ==

[[中国|中国]]古代故事《芦花记》讲[[闵子骞|闵子骞]]的[[后母|后母]]用棉花給自己的孩子做棉襖，虐待子骞，用芦花为他做冬襖，虽厚软但不暖和，闵子骞寒冷無措，父親發現此事，大怒，想要[[休妻|休妻]]，闵子骞居然還為後母求情。
