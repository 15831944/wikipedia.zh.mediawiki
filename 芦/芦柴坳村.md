'''芦柴坳村'''（{{lang-en|'''Luchai'ao Village'''}}）,旧称芦泗坳<ref>{{cite web |url=http://www.hbdsw.org.cn/tbgz/ztx/jnwz/201512/t20151209_80809.shtml| title=抗日战争时期的张体学| accessdate=2015-12-9|publisher=中共湖北省委党史研究室|}}</ref>，是[[中华人民共和国|中华人民共和国]]下辖村级行政区，位于[[湖北省|湖北省]][[麻城市|麻城市]][[夫子河镇|夫子河镇]]，坐落于芦家河山顶之上，地形是一条绵延数十余里的山冲，地势险要，崇山峻岭，林木覆盖。该村由[[邓家山垸|邓家山垸]]与[[芦柴坳垸|芦柴坳垸]]组成，全村共有1508人，耕地面积799亩，其中水田540亩，山林面积6300亩<ref>{{cite web |url=http://jcct.hbncw.cn/Village/Detail/825625| title=芦柴坳村概况| accessdate=2017-1-7|publisher=网上村村通|}}</ref>。
芦柴坳村原隶属于[[芦家河乡|芦家河乡]]，2004年，镇政府驻夫子河，撤区并乡后合并到夫子河镇管辖。<br/>芦柴坳村是[[中国抗日战争|中国抗日战争]]时期鄂东抗日游击挺进队第五大队的诞生地<ref>{{cite web |url=http://info.cnhubei.com/hg/201707/t3857938.shtml| title=一寸山河一寸血千万青年千万兵 盘点从黄冈走出的四支抗日武装| accessdate=2017-01-07|publisher=网上村村通|}}</ref>，1949年中华人民共和国建国后湖北省第一任省长[[张体学|张体学]]曾在此打过游击<ref>{{cite web |url=http://www.macheng.gov.cn/info/1027/32784.htm| title=盐田河旧事六记| accessdate=2017-11-3|publisher=麻城市人民政府网|}}</ref>。湖北省作家协会《[[新作家|新作家]]》编辑部主任[[周胜辉|周胜辉]]曾在抗日战争长篇记实文学中记载过该村抗战事件<ref>{{cite web|url=http://www.cuncunle.com/village-101-545519-article-1011427710024505-1.html|title=革命老区夫子河镇芦柴坳村的调查|accessdate=2012-7-23|publisher=村村乐-中国农村网|5=|archiveurl=https://web.archive.org/web/20180219151243/http://www.cuncunle.com/village-101-545519-article-1011427710024505-1.html|archivedate=2018-02-19|deadurl=yes}}</ref>。
==中国共产党革命根据地==
1938年10月，由[[中国共产党|中国共产党]]领导的鄂东抗日游击挺进队在湖北省[[黄冈县|黄冈县]]（今[[黄冈市|黄冈市]]）贾庙乡孙家冲村张家山成立。同年12月，挺进队被改编为“国民革命军陆军第二十一集团军独立游击第五大队”，下辖四个中队。[[张体学|张体学]]任大队长，[[丁宇宸|丁宇宸]]任副大队长，[[段亚杰|段亚杰]]任政治处主任，改编后，独立游击队将根据地搬迁到麻城夫子河芦柴坳，并在芦柴坳成立了“军政委员会”，[[方毅|方毅]]任书记，[[刘西尧|刘西尧]]任副书记，张体学、段亚杰等任委员。该部队对外称第五大队，是一个团的建制<ref>{{cite web |url=http://www.yg28.cn/html/shuba/xinshu/111.html| title=麻城名胜暨旧事:麻城文化丛书| accessdate=2016-3-22 |publisher=麻城阳光传媒网|}}</ref>。

1939年春节后，第五大队主力迁移至芦家河夏家山，同年9月1日，国民党鄂东游击总指挥[[程汝怀|程汝怀]]连同桂系一七二师师长[[程树芬|程树芬]]主力十八纵队王啸风部，向[[夏家山|夏家山]]、[[芦泗坳|芦泗坳]]等五大队驻地发起全面突袭，张体学的夫人[[戴醒群|戴醒群]]、冈麻罗边中心县委组织部长[[张良卿|张良卿]]、五大队供给处长[[姚渠|姚渠]]等100余名官兵被捕后遭杀害，中共领导的独立游击第五大队损失惨重，这就是震惊中外的“[[夏家山事件|夏家山事件]]”（亦称[[鄂东惨案|鄂东惨案]]）<ref>{{cite web |url=http://www.hbdysh.cn/2017/0218/29527.shtml| title=夏家山事件及其影响 | accessdate=2017-2-18|publisher=湖北党建信息门户网站|}}</ref>。

中华人民共和国成立后，芦柴坳村公路边建有抗日战争纪念馆和纪念碑，纪念馆陈列着[[张体学|张体学]]等五大队成员的珍贵照片，详细地记载了这里发生的抗日事迹。
==奇闻轶事==
1939年9月1日凌晨，国民党鄂东游击总指挥[[程汝怀|程汝怀]]与[[桂系|桂系]]172师师长[[程树芬|程树芬]]主力十八纵队王啸风部围剿国民革命军陆军第二十一集团军独立游击第五大队，企图一举歼灭。据当地村民回忆，那天下着大雨，张体学正在屋内休息，国民党军队将住房围住，住户[[安大脚|安大脚]]急中生智，把张体学拉起来，狠狠打了他几记耳光，大骂”懒种，稻田快被大水冲，还不同去堵决”。张体学趁机开脱，幸免于难。但张体学的夫人、医务室主任[[戴醒群|戴醒群]]、冈麻罗边中心县委组织部长张良卿、五大队供给处长姚渠等100多人被捕后遭杀害。当地的红色革命火种在村民的保护下得以存活<ref>{{cite web |url=http://www.zgxcfx.com/Article/549.html| title=革命老区芦柴坳村的调查| accessdate=2006-11-13|publisher=中国乡村发现网|}}</ref>。
