'''芦秀梅'''（{{bd|1957年||2012年|3月1日|catIdx=Lu卢秀梅}}），毕业于[[中国歌剧舞剧院|中国歌剧舞剧院]]，[[中国大陆|中国大陆]]著名女高音[[歌唱家|歌唱家]]。<ref name=" lxa">{{cite web|url= http://www.yangtse.com/system/2012/03/02/012842788.shtml|title= 女高音歌唱家芦秀梅1日因病去世 享年55岁|accessdate= 2012年|work= |publisher= 扬子晚报网|language= zh-hans|deadurl= yes|archiveurl= https://archive.today/20130428214719/http://news.yzwb.net/guonei/2012-03-02/889303.html|archivedate= 2013-04-28}}</ref><ref name="mea ">{{cite web|url=http://e.gmw.cn/2012-03/02/content_3694970_2.htm|title=女高音歌唱家芦秀梅因病逝世 曾演唱《春天在哪里》|accessdate=2012年|work=|publisher=光明网|language=zh-hans|deadurl=yes|archiveurl=https://archive.today/20130424144952/http://e.gmw.cn/2012-03/02/content_3694970_2.htm|archivedate=2013-04-24}}</ref>丈夫為[[中国中央电视台春节联欢晚会|中国中央电视台春节联欢晚会]]前導演[[趙安_(导演)|趙安]]。<ref name="fhws " >{{cite web |url=http://ent.ifeng.com/idolnews/mainland/detail_2012_03/08/13050390_0.shtml |title=芦秀梅葬礼赵安全程陪伴 宋祖英董文华等好友送别 |accessdate= 2012年|work= |publisher= 凤凰网|language =zh-cn}}</ref>

== 简介 ==
=== 早年生活 ===
1976年，[[文革|文革]]结束，恢复[[高考|高考]]，芦秀梅考入了[[中国歌剧舞剧院|中国歌剧舞剧院]]。<ref name=" lxa" /><ref name="mea " /> 毕业后参与多部[[歌剧|歌剧]]的演出。<ref name=" lxa" /><ref name="mea " />  芦秀梅曾多次参加中央电视台春节联欢晚会。<ref name="mea " /> 

=== 去世 ===
2012年3月1日，芦秀梅因病去世。<ref name=" lxa" /><ref name="mea " /> 

芦秀梅去世后，[[中国歌剧舞剧院|中国歌剧舞剧院]]官方讣告称：{{quote| 中国歌剧舞剧院著名女高音歌唱家、国家一级演员芦秀梅，在与病魔顽强抗争四年之后，在3月1日因病去世，享年55岁。<ref name="mea " /> }}

2012年3月7日，芦秀梅遗体告别仪式举行，安葬于[[北京|北京]][[八宝山革命公墓|八宝山革命公墓]]，文艺界名人[[宋祖英|宋祖英]]、[[杨澜|杨澜]]夫妇、[[朱军|朱军]]夫妇、[[董文华|董文华]]、[[李双江|李双江]]夫妇、[[陈红|陈红]]、[[姜昆|姜昆]]、[[毕福剑|毕福剑]]、[[徐沛东|徐沛东]]、[[黄宏|黄宏]]、[[朱迅|朱迅]]、[[孙悦|孙悦]]、[[林依轮|林依轮]]等都送去了[[花圈|花圈]]并亲临现场告别。<ref name="fhws " /><ref>{{cite web |url= http://ent.ifeng.com/zz/detail_2012_03/07/13030560_0.shtml?_from_ralated|title= 芦秀梅追悼会由丈夫赵安打点 董文华蔡国庆洒泪|accessdate= 2012年|work= |publisher= 凤凰网|language =zh-cn}}</ref>
