'''芦原温泉站'''（{{jpn|j=芦原温泉駅|hg=あわらおんせんえき|rm= Awaraonsen eki|f=y}}）位於日本[[福井縣|福井縣]][[芦原市|芦原市]]春宮一丁目，是[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）和[[日本貨物鐵道|日本貨物鐵道]]（JR貨物）[[北陸本線|北陸本線]]的[[鐵路站|鐵路站]]。

==概要==
本站是舊金津町地區的主要站和代表站。

事務管理代碼是▲541434 <ref>日本国有鐵道旅客局(1984)『鐵道・航路旅客運賃・料金算出表 昭和59年４月20日現行』</ref>。

[[特別急行列車|特急列車]]「[[白鷺號列車|白鷺]]」和「[[雷鳥號列車|雷鳥]]」（除最速達班次外）停站。

是通往縣內北部地區{{tsl|ja|芦原温泉||芦原温泉}}、東尋坊等觀光點的門戶站。
 
作爲主要車站，有定期列車在早上和晚上從本站開往[[福井站_(福井縣)|福井站]]和武生方向。

==歷史==
*[[1897年|1897年]][[9月20日|9月20日]] – 隨[[鐵道省|鐵道院]]的[[福井站_(福井縣)|福井站]] – [[小松站|小松站]]間延伸，以'''金津站'''之名開業。
*[[1909年|1909年]][[10月12日|10月12日]] - 制定線路名称，屬[[北陸本線|北陸本線]]。
*[[1911年|1911年]][[12月15日|12月15日]] - 鐵道院[[三國線|三国線]]到三国站一段開業。隨後的[[1913年|1913年]] [[1月1日|1月1日]]三国線沿海岸延長至三国港站。
*[[1929年|1929年]][[8月14日|8月14日]] -永平寺鐵道的'''金津站'''開業（共用站）。永平寺鐵道線金津 - 新丸岡間開業。同年12月10日延長至永平寺口，永平寺鐵道線全線通車。
*[[1934年|1934年]][[3月1日|3月1日]] 開始運行福井 - 金津 - 三国港間、三国線直通的汽油驅動列車，隨後延長至武生。和蒸氣列車一併運行。
*[[1940年|1940年]][[11月1日|11月1日]] 由於需準備軍備而實施對汽油的消費管制，三国線直通的汽油驅動列車廢止，改由原來的蒸氣列車運行。
*[[1944年|1944年]]
**[[10月11日|10月11日]]  三国線的運行停業<ref>[{{NDLDC|2961821/4}} 「運輸通信省告示第483号」『官報』1944年10月5日]（国立国会圖書館Digital Collection）</ref>。三国線的芦原站停業。三国線的三国 - 三国港間的營業權交由京福電氣鐵道。
**[[12月1日|12月1日]] - 永平寺鐵道併入京福電氣鐵道合併，稱爲永平寺線。
*[[1946年|1946年]][[8月15日|8月15日]] - [[日本国有鐵道|国鐵]]三国線的金津 - 芦原間恢復營業。三国線的列車直通到京福的芦原 - 三国港間。
*[[1969年|1969年]][[9月18日|9月18日]] - 京福電氣鐵道永平寺線的金津 - 東古市間廢止。
*[[1972年|1972年]]
**[[3月1日|3月1日]] - 国鐵三国線廢止。隨着三国線廢止，国鐵芦原站廢止。京福電氣鐵道的芦原站改称'''芦原湯町站'''。爲了取代三国線，国鐵巴士開辦金津三国線。
**[[3月15日|3月15日]] - 隨着三国線芦原站廢止，金津站改称'''芦原温泉站'''。
** 11月 – 改建爲現在的第3代站舍。
*[[1986年|1986年]][[11月1日|11月1日]] - 停止處理行李運送。
*[[1987年|1987年]][[4月1日|4月1日]] - 隨着[[国鐵分割民營化|国鐵分割民營化]]，成爲西日本旅客鐵道（JR西日本）和日本貨物鐵道（JR貨物）的站。
*[[2015年|2015年]][[5月19日|5月19日]] – 在市民請願下建增設升降機<ref name="ele">{{Cite web |url = https://www.facebook.com/awara.turuya/posts/902964589763887 |title = JR芦原温泉站のエレベータ、やっと１機目が完成しました。 |publisher = 芦原市觀光協会官方facebook |accessdate = 2016-03-03}}</ref>。8月，連接站台和站舍的第二台升降機竣工<ref name="ele" />。
*[[2015年|2015年]][[10月|10月]] - 站舍内的設備進行更新。除自動售票機更新至最新型號外，亦增設了售賣手信的商店「[[7-Eleven|7-Eleven]] 土產處 芦原店」。
*[[2018年|2018年]][[9月15日|9月15日]] - [[IC卡|IC卡]]「[[ICOCA|ICOCA]]」開始利用<ref name="fukuishimbun- ICOCA">{{Cite news|url=http://www.fukuishimbun.co.jp/articles/-/344415|title=イコカ福井県内導入は9月15日 福井站と敦賀站は先行で自動改札|newspaper=福井新聞|date=2018-05-31|accessdate=2018-06-04|archiveurl=https://archive.li/lM5VM|archivedate=2018-06-04}}</ref>。

==構造==

[[File:Awara_station_kaisatsu.jpg|剪票口]]

設[[島式月台|島式月台]]2面4線和待避設備的[[地上車站|地上車站]]，站舍爲[[跨站式站房|橋上形式]]。本站是福井地域鐵道部管理的直營站。

建設中的[[北陸新幹線|北陸新幹線]]的車站，當地自治体正推進周邊的建設<ref name="seibi">{{Cite web |url = http://www.city.awara.lg.jp/mokuteki/life/life137/matizukuri_d/fil/awaraonsenkihonkeikaku.pdf |title = 芦原温泉站周辺整備基本計画書 |publisher = 芦原市土木部都市整備課 |format=PDF |accessdate = 2016-03-03 }}</ref>。

站内設[[綠窗口|綠窗口]] 、綠色售票機、觸屏式[[自动售票机|自動售票機]]（可使用交通IC卡）、有人剪票口、交通IC卡專用簡易型[[閘機|自動閘機]] 、發車顯示器、洗手間和商店。

現在的站舍完成当初並沒有任何無障礙設備，2009年10月當地團體就增設升降機遞交請願書<ref>{{Cite web |url = http://www.awara-onsen.org/wakakusa/ele.htm |title = ＪＲ芦原温泉站にエレベーター設置を・・・あわら市長に要望 |publisher = 若草会公式Home Page |accessdate = 2016-03-03}}</ref>。作爲JR北陸本線的整備事業的一環，JR西日本在6年後的2015年，公佈增設升降機<ref>{{Cite web |url = https://www.westjr.co.jp/press/article/2015/01/page_6729.html |title = 特急「サンダーバード」のサービス向上に向けた取り組みについて |publisher = JR西日本 West Japan Railway Company |date =2015年1月28日 |accessdate = 2016-03-03}}</ref>。隨後、安裝了2台升降機並開始使用。

線路上，内側2線爲本線，下行的外側1線爲待避線。運轉指令上則由1號月台開始順序命名爲「下行1號線」「下行本線」「上行本線」。特急列車使用上下本線（2・3號月台），福井至本站折返的列車使用1號月台。

1號月台西側爲三国線月台的遺址。
