'''頂真'''，亦称'''頂針'''、'''連珠'''、'''蟬聯'''，是一種文學[[修辞|修辞]]方法<ref>[https://tw.answers.yahoo.com/question/index?qid=20050212000010KK00207 頂真的用法]</ref>，是指上句的結尾與下句的開頭使用相同的字或詞，用以修飾兩句子的聲韻方法。需要注意的是，使用這個方式時，毋須限制上下句的字數或平仄，但上下句交接點一定要使用相同的字或詞<ref>[http://wapbaike.baidu.com/item/顶真/939291?adapt=1 頂真_百度百科]</ref>。這種修辭技巧，可使文章句子有緊湊含銜的美感以及顯現出上遞下接的趣味。頂真修辭可分為連珠法和連環體這兩種<ref>[http://www.lfes.tc.edu.tw/www2/media_dvd/94CAI/J2_3-修辭寶典/Nchinese_p2_18.htm 修辭寶典----頂真修辭] {{webarchive|url=https://web.archive.org/web/20170112161029/http://www.lfes.tc.edu.tw/www2/media_dvd/94CAI/J2_3-%E4%BF%AE%E8%BE%AD%E5%AF%B6%E5%85%B8/Nchinese_p2_18.htm |date=2017-01-12 }}</ref>。这种方法，也用到影视剧及动漫的[[蒙太奇|蒙太奇]]中，称为[[顶针蒙太奇|顶针蒙太奇]]。

==連珠法==
連珠法的頂真是'''句與句之間'''的頂真。
*例如：
:「歸來見<u>天子</u>，<u>天子</u>坐明堂。」（《[[木蘭辭|木蘭辭]]》）
:「軍書十二<u>卷</u>，<u>卷</u>卷有爺名。」（《[[木蘭辭|木蘭辭]]》）
:「忽聞海上有仙<u>山</u>，<u>山</u>在虛無縹緲間。」（《[[長恨歌|長恨歌]]》）
:「明日復<u>明日</u>，<u>明日</u>何其多。」 （《[[明日歌|明日歌]]》）
:「物格而後知至，知至而後意誠，意誠而後心正，心正而後身修，身修而後家齊，家齊而後國冶，國冶而後天下平。」 （《[[大學_(經傳)|大學]]》）
===中文歌曲===
:「Will you remember me? 若我另有心志，暫別<u>遠去</u>，<u>遠去</u>找那自由再衝刺 。」（[[張國榮|張國榮]]《[[Leslie側面|由零開始]]》）
:「我們不<u>快樂</u>，<u>快樂</u>後不再快樂。」（[[鄭秀文|鄭秀文]]《[[Ladies_First|快樂不快樂]]》）
:「難道我愛<u>你</u>，<u>你</u>愛我都不會成事? 難道你愛<u>我</u>，<u>我</u>愛你要下輩子?」（[[鄭秀文|鄭秀文]]《[[很愛很愛|宿命主義]]》）
:「我的家有個<u>馬桶</u>，<u>馬桶</u>裡面有個<u>窟窿</u>，<u>窟窿</u>上面總有個笑容，笑人間無奈好多。」（[[劉德華|劉德華]]《[[馬桶_(歌曲)|馬桶]]》）
:「熱愛島有<u>熱愛草</u>，<u>熱愛草</u>做<u>熱愛帽</u>，<u>熱愛帽</u>插熱愛花，熾熱愛便碰到。」（[[鄭秀文|鄭秀文]]《[[十誡_(專輯)|熱愛島]]》）
:「世界真細小小<u>小</u>，<u>小</u>得真奇妙妙妙 」（《[[世界真细小|世界真细小]]》）
:「然後你——你最多會笑著迴<u>避</u>，<u>避</u>到底　明明不筋竭都力疲 」（[[楊千嬅|楊千嬅]]《[[Miriam|假如讓我說下去]]》）
:「天空塌<u>下去</u>，<u>下去</u>冰峰也<u>共對</u>，<u>共對</u>海底愛下去，負傷累累。」、「幽谷看<u>內裡</u>，<u>內裡</u>相擁去<u>面對</u>，<u>面對</u>山崩吻下去，互抱頸椎。」、「當充滿<u>俗氣</u>，<u>俗氣</u>中吞吐<u>為你</u>，<u>為你</u>喘息吻著你，絕不避忌。」、「傾覆了<u>瀑布</u>，<u>瀑布</u>風沙太<u>暴躁</u>，<u>暴躁</u>都可跳著舞，踏上荒草。」、「歪風與<u>暴雪</u>，<u>暴雪</u>中跟你<u>滴血</u>，<u>滴血</u>中宣布誓約，亦可蜜月。」、「風光滿<u>毒氣</u>，<u>毒氣</u>充塞我<u>共你</u>，<u>共你</u>一起這樣美，沒對不起。」（[[吳雨霏|吳雨霏]]《狠狠》）

近代連珠法的頂真有以下例子：

:　頭髮在我後面''飛揚　飛揚''也始終在''糾纏　糾纏''的總看''不見　不見''卻永遠不散
:　夕陽在我後面''低沈　低沈''的紅色染''我身　我身''後是我''一生　一生''的紅塵

==連環體==
連環體的頂真是'''段與段之間'''的頂真。
*例如：
:「下武維周，世有哲王。三后在天，''王配於京''。''王配於京''，世德作求。永言配命，''成王之孚''。''成王之孚''，下土之式。永言孝思，孝思維則。」（《[[詩經|詩經]]》）
:「候服於周，天命靡常。殷士膚敏，裸將於京。厥作裸將，常服黼冔。王之藎臣，''無念爾祖''。
:''無念爾祖''，聿修厥德。永言配命，自求多福。殷之未喪師，克配上帝。宜鍳於殷，駿命不易。」（節錄自[[詩經|詩經]]·大雅·[[文王|文王]]）

近代連環體的頂真有以下例子：

:    那榆陰下的一潭，
:    不是清泉，是天上的虹；
:    揉碎在浮藻間，沉澱著彩虹似的夢。
:    尋夢？撐一支長篙，
:    向青草更青處漫溯；
:    滿載一船星輝，
:    在星輝斑斕裡放歌。
:    但我不能放歌，
:    悄悄是別離的笙簫；
:    夏蟲也為我''沉默''，
:    ''沉默''是今晚的康橋。（《[[再別康橋|再別康橋]]》）
