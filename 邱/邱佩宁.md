'''邱佩宁'''，（1958年6月23日－），原名邱沛宁，中国内地女演员，毕业于[[中央民族学院|中央民族学院]]艺术系舞蹈专业。曾担任电视剧《[[红楼梦_(1987年电视剧)|红楼梦]]》的场记，因出演《[[西游记_(1982年电视剧)|西游记]]》中[[嫦娥|嫦娥]]角色而为观众熟知。<ref>{{cite web |url=http://ent.ifeng.com/a/20160331/42598582_0.shtml |title=西游记嫦娥“消失”30年后现身 58岁依旧美艳 |website=凤凰网 |date=2016-03-31 |accessdate=2018-11-11 |archiveurl=http://web.archive.org/web/20160401120515/http://ent.ifeng.com/a/20160331/42598582_0.shtml |archivedate=2016-04-01}}</ref>目前为一家公司的董事长。<ref>{{cite web |title=宝健“摘星计划”背后：多层计酬或涉嫌违规 |url=http://www.cb.com.cn/lingshouyuxiaofeipin/2017_0422/1181805.html |website=中国经营报 |accessdate=2018-11-12 |date=2017-04-22 |archiveurl=http://web.archive.org/web/20181112145303/http://www.cb.com.cn/lingshouyuxiaofeipin/2017_0422/1181805.html |archivedate=2018-11-12}}</ref>
