'''邱顯智'''（{{bd|1976年|4月29日||}}），[[中華民國|中華民國]]政治人物、人權[[律師|律師]]、[[社會運動|社會運動]]者。出生於[[中華民國|中華民國]][[嘉義縣|嘉義縣]][[竹崎鄉|竹崎鄉]]，高中就讀[[國立嘉義高級中學|嘉義高中]]，並於[[臺北大學|臺北大學]]取得[[法學|法學]]學士、碩士、[[德國|德國]][[海德堡大學|海德堡大學]]法學[[碩士|碩士]]、博士候選人<ref>{{cite web|url=https://www.newpowerparty.tw/candidates/293|title=邱顯智|publisher=時代力量|accessdate=2015-09-12}}</ref><ref name=CW>{{cite news|url=http://www.cw.com.tw/article/article.action?id=5060508|title=邱顯智律師：台灣的未來，不再有冤案|publisher=天下雜誌|author=馬岳琳|date=2014-08-20}}</ref>。

2011年邱顯智自德國海德堡返台後與身為新竹人的妻子[[黃琬婷|黃琬婷]]定居於[[新竹|新竹]]，育有兩女，並於新竹、台中兩地律師事務所執業，開始參與[[司法改革|司法改革]]運動，從事如[[鄭性澤殺警案|鄭性澤案]]等冤案救援<ref name="CW" />。2012年年底，邱顯智號召組成義務[[律師|律師]]團，聲援[[全國關廠工人連線抗爭事件|全國關廠工人抗爭事件]]，後續亦以律師身份積極參與如[[洪仲丘事件|洪仲丘案]]、大埔丟鞋案、[[苑裡反瘋車|苑里反瘋車案]]、[[梨山|梨山]]老農案等案件之義務辯護行動。同時邱顯智也是2014年[[太陽花學運|太陽花學運]]義務辯護律師團之一<ref>{{cite news|url=https://musou.tw/focuses/998|title=人物誌》從殯儀館到廟口－邱顯智參選初體驗|author=蕭長展、謝繐吟|publisher=沃草國會無雙|date=2015-08-17}}</ref>。

2015年，邱顯智與其他創黨成員創立政黨[[時代力量|時代力量]]，並宣佈參選2016年第九屆[[新竹市|新竹市]][[立委|立委]]<ref>{{cite news|url=http://www.ettoday.net/news/20150216/468330.htm|title=時代力量挑戰深藍選區　邱顯智、胡博硯參選新竹、中和|author=劉康彥|publisher=ETToday新聞雲|date=2015-02-16}}</ref>，競爭對手包括[[國民黨|國民黨]]籍候選人[[鄭正鈐|鄭正鈐]]、[[民進黨|民進黨]]籍候選人[[柯建銘|柯建銘]]、[[親民黨|親民黨]]籍候選人[[歐崇敬|歐崇敬]]，後落選。

2019年，以最高票之姿當選[[時代力量|時代力量]]決策委員。<ref>{{Cite news|url=https://www.thenewslens.com/article/113206|title=時代力量15位決策委員出爐：黨主席呼聲最高的如此回應「小綠」和柯文哲問題|author=羊正鈺|date=2019-2-2|work=|accessdate=2019-2-13|publisher=關鍵評論網}}</ref>

2019年2月15日以得票率65.3%當選時代力量黨主席。<ref>[https://www.storm.mg/article/949447 接棒黃國昌！邱顯智接任時代力量黨主席]風傳媒 朱冠諭  2019-02-15 15:15</ref>

==早年生活==
邱顯智出生於嘉義縣竹崎鄉奮起湖旁的小農村，父親為[[嘉義農專|嘉義農專]]（現國立嘉義大學）老師，母親為電子廠女工。邱顯智從小在農村環境成長，他的父親經常在颱風天帶著學校所擁有的先進收割機，幫助農民收割作物；在這些經歷下，他有了「擁有專業知識的人，應該要去幫助比較辛苦的人」的想法<ref name=CW/><ref name=YL>{{Cite web|url = http://www.ylib.com/ReadingLifeArticle.aspx?ID=33|title=冤獄平反，追求公義的人權律師：邱顯智|date=2013-09-01|author=楊泰興|publisher=遠流出版}}</ref>。

後來邱顯智就讀嘉義高中，原選擇第三類組，志願攻讀醫學領域，但因為他在求學過程中，對文史科目產生興趣；且因1990年代初期臺灣政治局勢變遷劇烈而開始關注臺灣政治局勢，發現許多臺灣重要政治運動者皆具備法律背景，所以萌生轉換跑道的念頭。在[[1992年中華民國立法委員選舉|1992年立委選舉]]期間，嘉義市中央公園噴水池常有許多黨外人士進行演講，有一次[[李鴻禧|李鴻禧]]到嘉義市中央公園噴水池演講，[[邱顯智|邱顯智]]還特別踩著腳踏車去聽 <ref>[http://www.ylib.com/ReadingLifeArticle.aspx?ID=33　冤獄平反，追求公義的人權律師：邱顯智（德晴法律事務所律師）YLib 遠流博識網]</ref>，更使邱顯智留下深刻印象<ref name=YL/>。

==參與司法改革與社運義務辯護==

=== 從事冤錯案救援 ===
邱顯智自德國留學返台後，受到[[蘇建和案|蘇建和案]]辯護律師羅秉成律師的影響，投身司法改革運動，陸續參與[[臺灣廢除死刑推動聯盟|廢死聯盟]]、冤獄平反協會，曾經擔任過[[鄭性澤|鄭性澤]]、劉炎國、[[沈鴻霖案|沈鴻霖]]等死刑犯之辯護律師，但邱顯智同時也是財團法人犯罪被害人保護協會臺灣臺中分會的義務律師，面對社會上對死刑存廢的爭議，邱顯智認為「犯罪被害人保護」與「[[廢除死刑|廢除死刑]]」被錯誤地對立起來，但其實兩者都是[[人權運動|人權運動]]的一環，精神應是一致的，律師的職責不在於探究真相，而在於阻止國家刑罰權的過度擴張。<ref name="YL" />

=== 參與洪仲丘案委任律師團，推動軍法改革 ===
2003年邱顯智在台南監獄服軍法預官，並在此時期接觸對他日後參與冤獄救援、軍法改革有重大影響的「王瑞豐案」。由於身為現役軍人而遭到軍法審判誤判的王瑞豐是邱顯智接觸的第一起冤錯案，他在擔任監獄官時期曾試圖協助王瑞豐平反，雖然沒有成功，但最終幫助王瑞豐在服刑滿兩年半後獲得假釋。邱顯智在王瑞豐出獄後，更針對此案提請大法官釋憲，此案也成為邱顯智日後參與廢除軍審法改革的動機之一。<ref>{{Cite web|url = http://www.thinkingtaiwan.com/content/19|title = 我的第一次冤獄接觸|accessdate = 2012-08-11|author = |date = |publisher = }}</ref>

2013年7月爆發[[洪仲丘事件|洪仲丘事件]]後，邱顯智成為洪案委任律師團之一，追究此事件中軍方包庇軍官凌虐下屬、隱匿證據等情事，並推動廢除軍審法，爭取軍審回歸司法審判<ref name=":0">{{Cite news|url = http://www.nextmag.com.tw/breaking-news/people/20150210/14643273|title = 【焦點人物】洪仲丘案家屬委任律師邱顯智：比起賺錢，給錢追比較爽|last = |first = |date = |work = |accessdate = }}</ref>。

=== 關廠工人案義務辯護律師 ===
2012年年底，勞動部編列預算向[[全國關廠工人連線抗爭事件|關廠工人]]提起訴訟，追討原本應是國家代位求償的資遣費與退休金，引發一波勞工抗爭，甚至進行臥軌抗議而遭到起訴。邱顯智得知相關訊息後，主動號召五、六十位律師組成義務律師團協助勞工打官司，拜訪專家學者、舉辦公聽會。2014年3月7日台北高等行政法案判關廠工人勝訴，認定當年所謂的「借款」實為基於國家責任的補償，3月10日勞動部全面對工人徹告<ref>{{Cite news|url =http://www.storm.mg/article-page/22638/2|title =專家評析：良知，在司法官僚主義的縫隙中閃耀|last =|first =|date =|work =|accessdate =|deadurl =yes|archiveurl =https://web.archive.org/web/20170409091037/http://www.storm.mg/article-page/22638/2|archivedate =2017-04-09}}</ref> <ref name=":0" /> <ref>{{Cite news|url = http://www.coolloud.org.tw/node/77656|title = 關廠工人案　勞動部撤告、退錢|last = |first = |date = |work = |accessdate = }}</ref>。

=== 大埔丟鞋案、太陽花學運等社運義務辯護律師 ===
2014年2月24日，陳為廷因大埔強拆案導致張森文自殺一事向苗栗縣長劉政鴻丟擲鞋子而被起訴，邱顯智擔任陳為廷等人義務辯護律師，主張此舉乃正當防衛，更多次爭取傳喚劉政鴻出庭<ref>{{Cite news|url = http://www.peoplenews.tw/news/bd0da247-f756-482d-9614-6845a532d403|title = 【專訪】大埔案律師邱顯智感嘆：真的拿劉政鴻沒辦法！|last = |first = |date = |work = |accessdate = }}</ref>。該案二審判決有罪免刑，鞋子發還陳為廷，邱顯智對此表示這在實務上相當罕見，有「為德不足」之感<ref>{{Cite news|url = http://www.thenewslens.com/post/180781/|title = 丟鞋案二審判有罪免刑　陳為廷呼籲關注審理中的大埔案|last = |first = |date = |work = |accessdate = }}</ref>。2014年3月15日，邱顯智接任[[台灣人權促進會|台灣人權促進會]]會長，三天之後即爆發318太陽花運動，邱顯智亦以台權會長身份參與立法院場外的活動、短講。運動退場後，邱顯智擔任[[陳為廷|陳為廷]]、黃燕茹等人的辯護律師<ref>{{Cite news|url = http://newtalk.tw/news/view/2014-12-11/54682|title = 檢翻去年舊案同318一起問　陳為廷：很誇張|last = |first = |date = |work = |accessdate = }}</ref>。此外318太陽花運動參與者魏揚因在運動期間參與割藍委行動稱國民黨籍新竹市立委[[呂學樟|呂學樟]]為「縮頭烏龜」而遭呂學樟控告妨礙名譽，邱顯智亦擔任魏揚的義務辯護律師<ref>{{Cite news|url = http://www.appledaily.com.tw/realtimenews/article/new/20150811/667321/|title = 呂學樟提再議　魏揚：沒格調的政客|last = |first = |date = |work = |accessdate = }}</ref>。

==參選立委==

*國會改革：告別王柯體制
*改革朝野協商：根除密室協商，全部公開透明
*反對黑箱、集權黨團協商，回歸[[委員會|委員會]]專業審查
*修正[[幼兒教育及照顧法|幼兒教育及照顧法]]：遏止幼教商品化、推動社區化幼教、保障[[幼教|幼教]]勞動者權益、全面實施新課綱，以六大幼兒基本能力為教學架構及目標，教材依在地特色自由訂定<ref>{{Cite news|url = https://tw.news.yahoo.com/-063227492.html|title = 時代力量提政策：廢止不合理財團租稅減免}}</ref>邱顯智參選以來，公開發表的政見主要聚焦在[[國會|國會]]改革上，尤其聚焦在[[朝野黨團協商|朝野黨團協商]]。2015年5月，民進黨初步確定提名[[柯建銘|柯建銘]]為立委參選人，面對是否進行在野整合的問題，邱顯智公開表示由於自己與柯建銘在黨團協商上有[[價值|價值]]上的歧見，他主張[[代議政治|代議政治]]應該是政黨在國會辯論後做出決策，而非訴諸於[[黨團協商|黨團協商]]，因此不會與柯建銘整合。<ref>{{Cite news|url = http://newtalk.tw/news/view/2015-05-04/59712|title = 邱顯智：與柯總召價值不合　將參選到底|last = |first = |date = |work = |accessdate = }}</ref>

邱顯智與[[柯建銘|柯建銘]]多次針對黨團協商議題展開交鋒，例如：柯建銘辦公室主任[[何珮珊|何珮珊]]便曾經撰文反駁邱顯智對[[朝野協商|朝野協商]]的批評，指出「朝野協商不是魔鬼交易」<ref>{{Cite news|url = http://www.appledaily.com.tw/realtimenews/article/new/20150504/604289/|title = 朝野協商不是魔鬼交易|last = |first = |date = |work = |accessdate = }}</ref>。2015年6月28日，邱顯智在時代力量政見發表會上提出「告別王柯體制，廢除黨團協商」的主張，批評黨團協商掏空了國會辯論精神，認為過去黨團協商之所以可以擋下一些爭議法案，是高度仰賴立法院長[[王金平|王金平]]的裁量權、而非制度的完善<ref>{{Cite news|url = http://www.storm.mg/article/55039|title = 時代力量提政見：人民決定不分區立委排序、廢除黨團協商|last = |first = |date = |work = |accessdate = }}</ref>。

7月18日，柯建銘公開回應邱顯智對黨團協商的質疑，指出「要參選的人，對立院遊戲規則，應該徹底了解」，表示如果不靠協商，事事依靠表決，馬政府的所有政策都可能被強行通過<ref>{{Cite news|url = http://www.storm.mg/article/57800|title = 柯建銘批邱顯智：要參選應瞭解朝野協商的價值|last = |first = |date = |work = |accessdate = }}</ref>。邱顯智則以「柯總召，這就是你的遊戲規則？」一文回應，以[[會計法第99-1條修正烏龍事件|會計法修法爭議]]為例，質疑朝野兩黨以草率倉促的[[密室協商|密室協商]]替[[顏清標|顏清標]]以公款喝花酒一案開罪，而至今該場協商的會議仍然沒有公佈。

邱顯智並再次抨擊「[[王柯體制|王柯體制]]」，強調「黨團協商能夠保護少數黨」是個[[迷思|迷思]]，它雖然能起到拖延效果，但它的關鍵原因也不在黨團協商本身，而是在於王金平、柯建銘十幾年來「未有協商成果，不強行表決」的默契。這個制度是建立在一個高度[[人治|人治]]的[[國會|國會]][[文化|文化]]上，任何期待改革的進步勢力都不會同意這樣的「遊戲規則」<ref>{{Cite news|url = http://talk.ltn.com.tw/article/paper/899415|title = 柯總召，這就是你的遊戲規則？|last = |first = |date = |work = |accessdate = }}</ref>。

=== 新竹市整合爭議 ===
由於[[新竹市|新竹市]]是[[2016年中華民國立法委員選舉|2016年立委選舉]]中[[時代力量|時代力量]]與[[民進黨|民進黨]]少數未達成整合共識的選區，因此自從邱顯智宣佈參選以來，是否與民進黨在新竹市整合始終是關鍵問題，邱顯智自己也在受訪中表示[[民進黨|民進黨]]動員了許多人脈希望「'''勸退'''」他<ref>{{Cite news|url = http://newtalk.tw/news/view/2015-07-28/62746|title = 像星星點亮闃黑的夜空─素人參選立委系列1(下)|last = |first = |date = |work = |accessdate = }}</ref>。

時代力量與民進黨也進行過多次的黨對黨協商，針對[[淡水|淡水]]選區、新竹選區進行協調，雖然兩黨在淡水區達成比民調的共識，但新竹區則始終未能有共識<ref>{{Cite news|url = http://www.storm.mg/article/48527|title = 民進黨協調時代力量　盼新竹不推人、中和換人選|last = |first = |date = |work = |accessdate = }}</ref>。8月12日，柯建銘在民進黨選舉對策委員會上表達不滿：柯建銘質疑何以民進黨在許多選區禮讓時代力量，卻遲遲無法在新竹市達成整合共識，無法把新竹市「換回來」；柯建銘甚至怒斥，民進黨選對會把他當成「潘仔（凱子）」<ref>{{Cite news|url = http://www.cna.com.tw/news/aipl/201508120400-1.aspx|title = 柯建銘怒斥選對會　批新竹市協調不力|last = |first = |date = |work = |accessdate = }}</ref>。

9月9日民進黨[[中執會|中執會]]中，有多位民進黨中執委對邱顯智不斷批判柯建銘與黨團協商感到不滿；民進黨主席蔡英文除了強調柯建銘很重要、必需讓他當選，更呼籲中執委「有時間多去輔選，大家都去」<ref>{{Cite news|url = http://udn.com/news/story/1/1176745-%E8%94%A1%E8%8B%B1%E6%96%87%E4%B8%8B%E4%BB%A4%E5%8A%9B%E4%BF%9D%E6%9F%AF%E5%BB%BA%E9%8A%98|title = 蔡英文下令力保柯建銘|last = |first = |date = |work = |accessdate = }}</ref> <ref>{{Cite news|url = http://www.cna.com.tw/news/aipl/201509090341-1.aspx|title = 柯建銘立委選情艱困　蔡英文相挺|last = |first = |date = |work = |accessdate = }}</ref>。

面對民進黨要求整合的壓力，邱顯智競選總幹事[[陳為廷|陳為廷]]表示，如果民進黨有誠意來整合，民進黨應該要用[[民調|民調]]來說明，而不是想逼退邱顯智<ref>{{Cite news|url = http://news.ltn.com.tw/news/politics/breakingnews/1432844|title = 民進黨、時代力量新竹整合陳為廷：勿逼退|last = |first = |date = |work = |accessdate = }}</ref><ref>{{Cite news|url = http://udn.com/news/story/1/1163528|title = 邱顯智：新竹市泛綠整合或許進一步討論}}</ref>。2015年9月16日，邱顯智與時代力量黨主席黃國昌等人舉行記者會，會中公開向柯建銘邀請針對價值與政策上的歧異進行辯論，並指出唯有透過辯論釐清雙方差異才可以有討論整合的空間<ref>{{Cite news|url = http://www.storm.mg/article/65637/1432844|title = 「時代力量」邱顯智籲柯建銘公開辯論價值相同才有整合空間|last = |first = |date = |work = |accessdate = }}</ref>。黃國昌指出，邱顯智是新竹市最早表態參選的候選人，以此駁斥「攪局」的說法。面對邱顯智提出公開辯論的要求，柯建銘批評這只是邱顯智炒高自己行情的作法，此外並未多做回應；蔡英文則表示，要不要辯論，尊重各別候選人的意見<ref>{{Cite news|url = http://newtalk.tw/news/view/2015-09-18/64769|title = 邱顯智邀辯論　柯建銘斥：炒高自己的行情|last = |first = |date = |work = |accessdate = }}</ref>。

2016年1月16日，邱顯智落選，得票數為36,309票，得票率16.55%；與柯建銘得票數90,642票，得票率41.33%，兩者差距54,333票。
