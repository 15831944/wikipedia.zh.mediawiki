'''邱毅'''{{BD|1956年|5月8日|catIdx=C邱}}，[[中國國民黨|中國國民黨]]政治人物，[[高雄市|高雄市]][[燕巢區|燕巢區]]人，曾任中華民國第五至第七屆[[立法委員|立法委員]]。

== 生平 ==
===家庭===
*第一任妻子：夏姓女士（1982年－1993年，已離婚）
*第二任妻子：謝京叡（1993年－2005年，已離婚）
*子女：與第二任妻子育有二女一子。

===從政之前===
[[國立臺灣大學|國立臺灣大學]][[經濟學|經濟學]]研究所畢業。

大學時期雖然專攻[[經濟學|經濟學]]，但基於興趣及為了賺取生活費，曾寫過不少[[小說|小說]]。

===補教時期===
邱毅在當兵時邊在下班後去台灣省立屏東農專（今[[國立屏東科技大學|國立屏東科技大學]]）教經濟，被[[高雄|高雄]][[學廬補習班|學廬補習班]]網羅擔任補教老師，開始專教公職考試，從此開啟他的補教人生。後來決定把教學重心由高雄移回台北，並改為專教考研究所及四技、二專，於是邱毅在補習班以「[[陳偉|陳偉]]」化名為名，成立陳偉[[補習班|補習班]]，他和前妻一起上課，並由前妻經營，直到親民黨主席宋楚瑜找他參與政治後，才逐漸淡出。<ref>{{cite news |language = zh-tw |author =聯合晚報 記者程平 |url =http://udn.com/news/story/7314/1485168-%E9%82%B1%E6%AF%85%E7%9A%84%E8%A3%9C%E6%95%99%E4%BA%BA%E7%94%9F-%E5%9B%A0%E6%94%BF%E6%B2%BB%E8%AE%8A%E4%BA%86%E6%A8%A3 |title =邱毅的補教人生 因政治變了樣|publisher =[[聯合新聞網|聯合新聞網]] |date= 2016-02-09 |accessdate=2016-02-09}}</ref>

===從政之後===
===中華社會民主黨時期===
邱毅曾加入過[[中華社會民主黨|中華社會民主黨]]，直到1994年退出。
===新黨時期===
1994年加入[[新黨|新黨]]。隨即替當時加入[[新黨|新黨]]的[[朱高正|朱高正]]助選首屆臺灣省長選舉，並出資成立[[地下電臺|地下電臺]]，參加[[談話節目|談話節目]]累積知名度，進而成為[[政論節目|政論]][[名嘴|名嘴]]。

===親民黨時期===
在[[2000年中華民國總統大選|2000年總統大選]]時，出任[[宋楚瑜|宋楚瑜]]競選總部的發言人，後宋楚瑜組建[[親民黨|親民黨]]，邱毅也隨之加入。於隔年[[2001年中華民國立法委員選舉|立委選舉]]中，以親民黨參選人之身分投入[[高雄市|高雄市]]第二選區參選，在該區應選五席而[[國民黨|國民黨]]提名三席的夾殺下，邱毅仍突圍當選，而國民黨則三席皆落選。[[2002年中華民國直轄市市長暨市議員選舉|2002年直轄市議員選舉]]中，在邱毅的帶領下，親民黨在[[高雄市議員|高雄市議員]]開出提名九人當選七人的佳績。後來爆發[[高雄市議長賄選案|高雄市議長賄選案]]，多位議員喪失資格。2003年進行市議員補選，在邱毅的陪同宣傳下，親民黨候選人都落選。[[2004年中華民國立法委員選舉|2004年立委選舉]]，邱毅再度代表親民黨並當選。

[[2004年中華民國總統選舉|2004年總統大選]]，民進黨的參選人[[陳水扁|陳水扁]]及[[呂秀蓮|呂秀蓮]]以不到三萬票，僅0.228%的差距險勝國民黨和親民黨聯合參選的[[連戰|連戰]]和[[宋楚瑜|宋楚瑜]]，成功連任。由於票數過於接近，加上選舉前日發生了[[319槍擊案|319槍擊案]]，[[泛藍|泛藍]]支持者懷疑民進黨操縱選舉，不滿的群眾開始聚集抗議。高雄市的泛藍支持者在邱毅的帶領下，到[[臺灣高雄地方法院|高雄地方法院]]門前要求驗票，群眾激動的推撞院檢大樓的大門，此時站在宣傳車上的邱毅高喊「同志們，革命的時候到了！」並指揮司機開車衝撞法院，造成多名員警受傷，畫面聳動。之後邱毅在台灣網路上因此事被網友戲稱為「衝車大將軍」<ref>[http://www.nownews.com/2011/03/29/11490-2700493.htm ]邱毅衝撞象腿幫人氣翻紅　網友要「水仙少年」選總統</ref>。

[[臺灣高等法院高雄分院|臺灣高等法院高雄分院]]在2006年依照首謀聚眾妨害公務罪判決邱毅有罪，案件上訴經過[[臺灣最高法院|臺灣最高法院]]審理後，最高法院在2007年3月22日將上訴駁回，邱毅被判處一年又兩個月的[[有期徒刑|有期徒刑]]定讞，但未被[[褫奪公權|褫奪公權]]，且入獄時仍為立法委員之身分。

===國民黨時期===
2007年3月22日，邱毅因涉嫌於[[2004年中華民國總統大選|2004年總統大選]]後率眾包圍及指揮貨車衝撞[[臺灣高雄地方法院|高雄地方法院]]一案，[[臺灣最高法院|最高法院]]維持[[臺灣高等法院高雄分院|二審]]原判，依「首謀聚眾滋擾罪」判處[[有期徒刑|有期徒刑]]一年兩個月定讞，隨後邱毅的照片即遭張貼各港口以防潛逃出境。後因立法院實行特赦政策，連帶減刑為七個月，邱毅在入獄前與妻子離婚。

2007年4月20日，邱毅爆料指出[[319槍擊案|319槍擊案]]之時，總統府有數千萬的不明資金流入並舉出交易明細。總統府則回應邱毅所提明細之「分行別」和「科目別代號」等都是未使用的空號，指稱此為假造的不實文件<ref>{{cite news |language = zh-tw|author = 陳明旺、郭敏政、黃敬平 |url = http://www.wpoforum.com/viewtopic.php?fid=1&tid=17834 |title = 邱毅指扁密存現金1.6億，扁幕僚斥抹黑 強調是競選經費 |publisher=[[蘋果日報_(臺灣)|蘋果日報 (臺灣)]] |date=2006-08-07 |accessdate=2011-01-07}}</ref>。4月26日入獄服刑，獄中編號為「'''3665'''」，服刑滿七個月後出獄。

2007年11月20日，於早上提前出獄，並在2008年1月12日[[2008年中華民國立法委員選舉|第七屆立委選舉]]中，以國民黨身份於全國不分區第五順位當選。後雖參選[[2012年中華民國立法委員選舉|第八屆立委選舉]]高雄市第七選區之區域立委，但據2012年1月14日選舉開票結果，該選區民進黨參選人[[趙天麟_(臺灣)|趙天麟]]獲得最高票，邱毅落選。

2014年，因與台北市議員[[林瑞圖|林瑞圖]]多次在政論節目誹謗[[高雄市市長|高雄市市長]][[陳菊|陳菊]]胞弟陳武進為南部圍標集團首腦，二審判決邱毅有期徒刑6個月、林瑞圖5個月，可以易科罰金，全案定讞不得再上訴。<ref>{{cite news |language = zh-tw|author = |url = https://tw.news.yahoo.com/%E6%9C%80%E6%96%B0-%E6%B6%89%E8%AA%B9%E8%AC%97%E9%99%B3%E8%8F%8A%E8%83%9E%E5%BC%9F-%E9%82%B1%E6%AF%85%E6%9E%97%E7%91%9E%E5%9C%96%E5%88%A4%E5%88%916%E6%9C%885%E6%9C%88-062932997.html|title =涉誹謗陳菊胞弟 邱毅林瑞圖判刑6月5月 |publisher =[東森新聞]] |date =2014-12-24 |accessdate =2014-12-29}}</ref>

2016年立委選舉，以國民黨黨員身分代表新黨參加不分區立委再度落選<ref>[http://news.ltn.com.tw/news/politics/breakingnews/1575094 新黨政黨票未過5％ 邱毅立委夢碎]</ref><ref>[http://www.ettoday.net/news/20160117/632670.htm?t=%E8%BD%89%E6%88%B0%E6%96%B0%E9%BB%A8%E4%B8%8D%E5%88%86%E5%8D%80%E5%A4%B1%E5%88%A9%E3%80%80%E9%82%B1%E6%AF%85%E5%96%8A%EF%BC%9A%E8%88%87%E5%9C%8B%E6%B0%91%E9%BB%A8%E5%85%B1%E7%B5%84%E5%90%88%E4%BD%9C%E8%81%AF%E7%9B%9F 轉戰新黨不分區失利　邱毅喊：與國民黨共組合作聯盟]</ref>。

== 相關事件 ==
===揭露高捷、臺開弊案===
{{Refimprovesect|time=2011-10-17}}
{{see also|高捷弊案|臺開弊案}}
從2005年8月22日[[高雄捷運|高雄捷運]]工地的[[泰國|泰國]][[外籍勞工|外籍勞工]]暴動後，邱毅揭露時任總統府副秘書長的[[陳哲男|陳哲男]]涉嫌非法接受業者招待，在遭到[[中華民國總統府|總統府]]與[[高雄市政府|高雄市政府]]的否認後，邱毅提出多項證據證明高雄捷運工程中確實有弊案存在，包括陳哲男於[[韓國|韓國]][[濟州島|濟州島]][[賭場|賭場]]的關鍵照片，與仲介公司華擎與前[[高雄市長|高雄市長]][[謝長廷|謝長廷]]往來的證據，戳破總統府、高雄市政府與陳哲男的回應。此進而引起社會大眾與新聞媒體的普遍關注，促使檢調開始深入調查此案，導致時任[[行政院院長|行政院院長]]的謝長廷下臺與高雄市代理市長[[陳其邁|陳其邁]]被撤換。總統陳水扁接受[[民視|民視]]訪問時公然表示，高捷弊案是高雄人在高雄的事，他完全不知情，被認為是扁謝關係決裂的原因之一。外勞暴動與高雄捷運弊案重挫民主進步黨縣市長選情，民進黨在縣市長選舉中僅在[[南臺灣|南臺灣]]當選六席縣市首長的因素之一。

2006年5月，邱毅揭露陳水扁的女婿[[趙建銘|趙建銘]]涉及[[臺開弊案|臺開弊案]]，在日式高級料理餐廳「三井」獲取不法內線消息，並進行非法內線交易。趙於同年5月24日遭[[臺北地方法院|臺北地方法院]]裁定[[羈押|收押]]至[[臺北看守所|臺北看守所]]，由於總統府聲明與趙建銘發言被檢調單位的證據拆穿，重挫陳水扁總統聲譽及第一家庭的信用，導致2006年6月以國民黨及親民黨為首的在野黨於立法院推動[[百萬人民倒扁運動|罷免陳水扁議案]]。邱毅兩度於2006年10月3日及10月10日參加反貪腐[[倒扁|倒扁]]集會，要求陳水扁下臺，但因罷免案未過門檻而不了了之。

=== 邱毅、陳致中--世紀大辯論 ===
邱毅於高雄市第7選區參選[[2012年中華民國立法委員選舉區域暨原住民選舉區投票結果列表|2012年立委選舉]]，和第9選區、無黨籍參選人[[陳致中|陳致中]]，選舉期間隔空互批，經歷波折，[[壹電視|壹電視]]促成「邱毅、陳致中世紀大辯論」。2011年12月13日晚間，於[[高雄市議會|高雄市議會]]內展開辯論，由[[李晶玉|李晶玉]]、[[岑永康|岑永康]]主持。會中兩人發言都以[[台語|台語]]為主，，由於兩人以激烈的詞句互罵（[[網路語言|網路語言]]稱為「互婊」）被媒體稱為「毅中各婊」<ref>[http://www.appledaily.com.tw/appledaily/article/headline/20111215/33888291/ 辣蘋果專欄：滋補選情的毅中各婊（余艾苔）]</ref>（與[[馬英九|馬英九]]的[[一中各表|一中各表]]政策諧音）

*邱毅砲火猛烈，直攻法院指稱陳致中曾經「開查某」（[[嫖妓|嫖妓]]），不顧家庭，陳致中說自己高票當選議員，民眾的選票比「國民黨開的法院」還有公信力多了。陳致中反擊，邱毅毫不盡責，做甚麼就不像甚麼。身為[[立法委員|立法委員]]，卻整天跑去《[[2100全民開講|2100全民開講]]》當[[名嘴|名嘴]]，極少出席[[國會|國會]]。在大學任教，卻偷跑到補習班賺錢。在補習班教書，卻「睡自己的學生」（發生[[性關係|性關係]]），而此段[[婚外情|婚外情]]迫使自己[[元配|元配]]離婚；[[離婚|離婚]]後又娶了學生，也就是第二任妻子，第二任妻子為了邱毅，更曾哭得死去活來助選，邱毅在選後又再度離婚，還追討送給第二任妻子的[[不動產|不動產]]，竟還有臉說別人？陳致中加大力道，也創造出一句名言：「邱毅只有假髮是真的！」

*2008年12月15日，前總統[[陳水扁|陳水扁]]在[[陳水扁家庭密帳案|密帳案]]起訴後無保開釋，對此判決結果不滿的邱毅赴[[監察院|監察院]]檢舉審判長[[周占春|周占春]]。但當步出監院大門時，台南市安南區扁友會會長[[黃永田|黃永田]]突然衝出，從邱毅後方用單手將邱毅的[[假髮|假髮]]直接扯下，證實了邱毅真的有禿頭。邱毅形容自己像被脫光衣服，自尊嚴重受損，並堅持對黃永田提出告訴<ref>{{cite news |language = zh-tw |author = 劉志原、王俊忠 |url = http://www.libertytimes.com.tw/2010/new/apr/1/today-south1.htm |title = 扯邱毅假髮賠30萬 黃永田不服 |publisher = [[自由時報|自由時報]] |date = 2010-04-01 |accessdate = 2011-01-07 |deadurl = yes |archiveurl = https://web.archive.org/web/20100406075734/http://www.libertytimes.com.tw/2010/new/apr/1/today-south1.htm |archivedate = 2010-04-06 }}</ref>。網友戲稱此事件為「撕髮迫害」，在網路上流傳事件經過的照片，甚至在[[日本|日本]]的[[綜藝節目|綜藝節目]]也有所提及<ref>{{cite news |language = zh-tw | author =  |url = http://www.chinareviewnews.com/doc/1008/3/1/4/100831451.html?coluid=7&kindid=0&docid=100831451 |title = 邱毅頭皮頭髮分離　網友：撕髮迫害 |publisher =[[中國評論新聞網|中國評論新聞網]] | date =2008-12-16 | accessdate =2011-01-07}}</ref>。家住[[臺南市|臺南市]]的[[黃永田|黃永田]]五十六歲，是當地知名挺扁人士，曾多次北上聲援[[陳水扁|陳水扁]]。警方押黃返回派出所過程中，黃高喊：「你看邱毅要出賣[[臺灣|臺灣]]，臺灣人要勇敢站出來，陳水扁已經倒了，我不能倒！」不過，黃接受警方偵訊時，改口辯稱當時不是故意的。黃妻則喊冤說，邱毅是他們的偶像，因為想上前和他講話，沒想到被警察團團圍住，拉扯中不慎扯落假髮<ref>{{cite news |language = zh-tw |author = |url =http://www.chinareviewnews.com/doc/1008/3/0/9/100830996.html?coluid=7&kindid=0&docid=100830996&mdate=1216141640 |title = 邱毅被扁迷偷扯假髮　驚險一刻 |publisher = [[中國評論新聞|中國評論新聞]] |date =2008-12-16 |accessdate =2011-06-09}}</ref>，絕非故意的。之後[[黃永田|黃永田]]因扯髮案在刑事部份，高等法院已依妨害自由罪判處5月徒刑，可易科罰金15萬定讞，民事損害賠償部份，高等法院則判黃永田須賠償邱毅卅萬元定讞。<ref>{{cite web |url=http://www.libertytimes.com.tw/2010/new/sep/22/today-south7.htm |title=存档副本 |accessdate=2012-01-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20100926101924/http://www.libertytimes.com.tw/2010/new/sep/22/today-south7.htm |archivedate=2010-09-26 }} 扯邱毅髮印名片 黃永田昨敗訴</ref>
