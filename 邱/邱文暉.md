'''邱文暉'''（{{lang-en|'''Khoo Boon Hui'''}}，{{bd|1954年||}}）為[[內政部_(新加坡)|新加坡內政部]]秘書，邱文暉曾經在2008年至2012年期間担任[[国际刑警组织|国际刑警组织]]主席。他亦曾經在1997年7月至2010年1月期間擔任[[警察總監_(新加坡)|新加坡警察總監]] <ref>[https://www.interpol.int/About-INTERPOL/Structure-and-governance/KHOO-Boon-Hui Khoo Boon Hui]</ref>。

== 學歷 ==
邱文暉在英華學校完成中學教育并在1973年获得着名的[[新加坡武裝部隊|新加坡武装部队海外奖学金]]，赴英國留學。

1976年，邱文暉畢業於[[牛津大學聖約翰學院|牛津大学圣约翰学院]]的工程和经济学院。1982年在[[肯尼迪政府学院|哈彿大學肯尼迪政府学院]]獲得了碩士學位。2002年，他在[[沃顿商学院|沃顿商学院]]参加了高级管理課程。

== 职业生涯 ==
1976年，邱文暉從牛津大学畢業後，他被邀请加入[[新加坡警察部队|新加坡警察部队]]。邱文暉在翌年开始他的职业生涯，自此擔任不同崗位，包括1987年主管戰略規劃，1990年擔任警局參謀長，1991年擔任刑事调查部门主管，1995年任副警察专员。 

1997年7月，他被任命为新加坡[[警察總監_(新加坡)|警察總監]]，14年後的2010年2月1日退任，轉任[[內政部_(新加坡)|內政部]]高級副秘書。

== 国际刑警组织 ==
2006年至2009年，邱文暉擔任國際刑警組織執行委員會副主席。 2008年10月9日，他当选为[[国际刑警组织|国际刑警组织]]主席，任期四年，並於2012年離任。

== 奖项和表彰 ==
因著邱文暉在警隊的長期服務和貢獻，他在1992年被授予公共行政奖章(银)，其後在1998年和2003年分別被授予长期服务奖章和[[功绩勋章_(新加坡)|功績勋章]]。

2005年8月，[[马来西亚|马来西亚]]最高元首赋予邱文暉Darjah Kebesaran Panglima Setia Mahkota(Kerhormat) (PSM)<ref>{{Cite web|url=http://www.istiadat.gov.my/index.php/component/semakanlantikanskp/|title=Bahagian Istiadat dan Urusetia Persidangan Antarabangsa|accessdate=2018-11-18|work=www.istiadat.gov.my|language=en-gb}}</ref>。

== 家庭 ==
邱文暉妻子為Puan Sri Betty Au，二人育有两个孩子。
