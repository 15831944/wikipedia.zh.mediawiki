'''宗教年表'''是按[[年代学|<nowiki/>]][[年代学|年代学]]排列的从史前时代到近现代重要的宗教事件纲要。此文广泛涉及史前时代，由于大量的人类宗教经历没有被归入[[信史|信史]]，信史只有将近5000年([[文字|文字]]出现的时间)，缺乏真实的书面记录，大多数对原始宗教的认知来自于[[考古学|考古学发现]]和其他间接来源还有推测，许多原始宗教一直受到持续争论。

== 史前时期 (30万年前至公元前34世纪) ==
通常以宗教信仰的出现和出现刻意制作丧葬标记的习俗为标志。<ref name="asaburial"><cite class="citation web" style="min-height: 144.92px;" contenteditable="false">Morton, Glenn. </cite></ref>

=== 30万年前至公元前51,000年 ===
; 前223,000–前100,000 
: 早期[[人科|原始人类]]的活动迹象，例如[[尼安德特人|尼安德塔人]]<ref name="jewels 1"><cite class="citation web" contenteditable="false">[http://www.gtj.org.uk/en/small/item/GTJ27306/ "Gathering the Jewels"] {{webarchive|url=https://www.webcitation.org/66bOAOxmp?url=http://www.gtj.org.uk/en/small/item/GTJ27306/ |date=2012-04-01 }}. </cite></ref><ref name="british_archaeology_early_burials">[http://www.britarch.ac.uk/BA/ba66/feat1.shtml When Burial Begins]</ref> 还有[[海德堡人|海德堡人]]<ref name="british_archaeology_early_burials">[http://www.britarch.ac.uk/BA/ba66/feat1.shtml When Burial Begins]</ref><ref><cite class="citation book" contenteditable="false">Greenspan, Stanley (2006-02-06). </cite></ref> 常刻意的在墓区排列死者的遗体 。其墓葬遍及[[歐亞大陸|歐亞大陸]] (例如：邦特纽伊德墓穴 (威尔士)、[[阿塔普埃尔卡山|阿塔普埃尔卡山]] (西班牙)、 卡夫泽洞穴、伊斯—斯库尔洞穴、[[克拉皮納-扎戈列縣|克拉皮納]] (克罗地亚)，<ref name="british archaeology_early_burials">[http://www.britarch.ac.uk/BA/ba66/feat1.shtml When Burial Begins]</ref> 可以确信其表示现仪式习俗的出现，虽然有一些关于此的问题还在争论。<ref>Robert Gargett argued that the evidence for purposeful Neanderthal burials is weak, and that they can be explained as a result of accidental deposition. </ref> 尼安德塔人将遗体放置于简单的墓穴和少量或没有在意陪葬品或标记， 无论如何他们的墓穴偶尔出现在石灰岩块内外，可能是古老形式的坟墓标志。<ref name="british_archaeology_early_burials">[http://www.britarch.ac.uk/BA/ba66/feat1.shtml When Burial Begins]</ref>这些习俗可能是由于对部落同伴感同身受的结果，举个例子： 一个婴儿葬在德里耶赫洞穴之前先脱开其关节并特意按标准的解剖学排放其身体部件。<ref name="british_archaeology_early_burials">[http://www.britarch.ac.uk/BA/ba66/feat1.shtml When Burial Begins]</ref>

; 前98,000
: 位于现今的法国和比利时地区, 尼安德特人可能在[[剔肉葬骨|剔肉葬骨]]习俗的一段时期后，不再以此方式安葬死者。<ref name="british_archaeology_early_burials">[http://www.britarch.ac.uk/BA/ba66/feat1.shtml When Burial Begins]</ref>

: 世上已知最古老的动物形雕塑也算是已知最古老的雕像之一 [[奥瑞纳文化|奥瑞纳]]<ref><cite class="citation web" contenteditable="false">[http://www-geology.ucdavis.edu/~cowen/HistoryofLife/CH20images.html "Images for Chapter 20 Hominids"] {{webarchive|url=https://web.archive.org/web/20080519004205/http://www-geology.ucdavis.edu/~cowen/HistoryofLife/CH20images.html |date=2008-05-19 }}. ; 前33,000-前25,000
: [[措迪洛山|措迪洛山]]的一个30,000年前的古代信仰场所被发现于[[波札那|波札那]]西北。<ref><cite class="citation web" contenteditable="false">[http://news.nationalgeographic.com/news/2006/12/061222-python-ritual.html ""Python Cave" Reveals Oldest Human Ritual, Scientists Suggest"]. </cite></ref>全都令人确信证明[[尼安德特人|尼安德特人]]墓葬行为停止了. 大致符合智人引入欧洲和尼安德特人的衰亡期。<ref name="british_archaeology_early_burials">[http://www.britarch.ac.uk/BA/ba66/feat1.shtml When Burial Begins]</ref>开始出现个别头骨或长骨大量沾上红[[赭色|赭]]并单独的埋葬，这个习俗可能是圣遗物的起源。<ref name="british_archaeology_early_burials">[http://www.britarch.ac.uk/BA/ba66/feat1.shtml When Burial Begins]</ref>最古老的发现"沃尔道夫的维纳斯雕像"出现在墓中。 有些被刻意毁坏或被反复刺穿，可能表示谋杀凶手被陪葬<ref name="british_archaeology_early_burials">[http://www.britarch.ac.uk/BA/ba66/feat1.shtml When Burial Begins]</ref> 或其他未知的社会动态。

; 前25,000–前21,000
: 清晰的墓葬样例出现在东欧威尔士[[伊比利亚_(消歧義)|伊比利亚]]。 全部这些都混合大量使用的红赭。此外, 各种各样的物件包含在墓内(如：长春花壳、负重衣、玩偶、鼓槌、猛犸象牙珠、狐狸牙[[墜飾|墜飾]]、象牙制套装、鹿角棒、燧石刀片等等)。<ref name="british_archaeology_early_burials">[http://www.britarch.ac.uk/BA/ba66/feat1.shtml When Burial Begins]</ref>

; 前13,000–前8,000
: 恢复了明显的墓葬活动，先前的殡葬活动形式既不明显而且缺少同辈人保留的这类活动的一些葬礼知识，许多男人女人和孩子埋葬在同一预先10,000年完成的用于墓葬的洞穴 ，全部这些坟墓被描绘在洞穴壁和大型石灰岩块上，墓葬彼此非常相似还有一些共同的特征—赭石、贝壳和猛犸象牙饰品—可以追溯数千年。一些墓葬是成双的, 由成年男性和旁侧的少年男性组成.。他们出现了现代[[墳場|墳場]]所采用的形式，老的墓葬一般被重新挖出并搬迁以让位置给新墓， 较老的遗骨通常集中放在一起。巨石可能作为坟墓的标志。成对的赭色鹿角撑在洞穴内，可类比现代墓前留花的习俗。<ref name="british_archaeology_early_burials">[http://www.britarch.ac.uk/BA/ba66/feat1.shtml When Burial Begins]</ref>

=== 1万年前至公元前34世纪 ===
; 前9831 
: [[新石器革命|新石器革命]]开始导致全世界人口激增。 第一批城市、[[民族国家|民族国家]]、[[王国|王国]]和宗教组织出现。早期民族国家通常是[[神權政治|神權政治]]， 政治权力依据宗教声望。始于[[印度南部|南印度]]第一桑伽姆时期。

; 前9130–前7370 
: [[哥贝克力石阵|哥贝克力石阵]]的存在时间，<ref>"The World's First Temple", Archaeology magazine, Nov/Dec 2008 p 23.</ref> Nevali Çori附近发掘的最古老的人造信仰场所。

; 前8000 
: 4到5个松木柱竖立在位于[[巨石阵|巨石阵]]可能位置的附近。<br>

; 前7500–前5700 
: 加泰土丘定居点发展成[[安那托利亞|小亞細亞]]合适的的信仰中心. 可能公共圣地的崇拜习俗, 其居民遗留许多的粘土雕像和生殖崇拜、女性、狩猎场景的印记.

; 前5500–前4500 
: [[原始印欧人|原始印欧人]]出现，地点可能在東歐大草原(准确原居地仍在争议)之内。原始印欧人发展出专注于牲祭的宗教，并将影响后来印欧语系文化贯穿 欧洲、[[小亞細亞|小亞細亞]]和[[印度次大陸|印度次大陸]]的衍生宗教。

; ~前3750 
: 原始闪米特人出现，原居地通常认为在[[阿拉伯半岛|阿拉伯半岛]]。原始闪米特人将会从[[近東|近東]]迁移至[[美索不达米亚|美索不达米亚]]、 [[埃及|埃及]]、[[埃塞俄比亚|埃塞俄比亚]]和[[地中海|地中海]]东岸。 他们的宗教信仰将影响后世的文化和信仰， 包括[[亞伯拉罕諸教|亞伯拉罕諸教]].
