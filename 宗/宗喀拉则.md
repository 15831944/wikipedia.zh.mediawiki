'''宗喀拉则'''位于[[中华人民共和国|中国]][[青海省|青海省]][[贵德县|贵德县]][[拉脊山|拉脊山]]垭口，[[黄土高原|黄土高原]]和[[青藏高原|青藏高原]]分界线之上，被称为“青藏第一关”，<ref>{{Cite web |url = http://www.chinanews.com/wap/detail/pic/117619.shtml |title = 行走于高原之上，感受与天空更近的距离|website=中国新闻网 |date = 2018-07-24 |language = zh |accessdate = 2019-03-09 }}</ref>总面积6600平方米，其中建筑面积1700平方米，<ref name="2R"/>是得到[[吉尼斯世界纪录|吉尼斯世界纪录]]官方认证的中国最大的“拉则”建筑群。<ref name="1R">{{Cite web |url = http://m.sohu.com/a/199219528_534824 |title = 吉尼斯纪录最大的拉则 |website=搜狐网 |date = 2017-10-20 |language = zh |accessdate = 2019-03-09 }}</ref>于2014年11月建成。<ref name="3R">{{Cite web |url = http://www.chinanews.com/tp/2014/11-21/6803438.shtml |title = 中国安多藏区第一拉则群正式建成 |website=中国新闻网 |date = 2014-11-21 |language = zh |accessdate = 2019-03-09 }}</ref>

==词源==
宗喀拉则因位于[[青海省|青海省]]拉脊山垭口[[宗喀山脉|宗喀山脉]]西北处而得名。<ref name="1R"/>

==地理==
宗喀拉则海拔3820米，位于青海重要的牧区草场拉脊山之上。总面积6600平方米，其中建筑面积1700平方米。<ref name="2R">{{Cite web |url = http://www.qhhngd.gov.cn/html/5494/437034.html |title = 拉脊山--宗喀拉则 |website=贵德县人民政府 |date = 2015-07-23 |language = zh |accessdate = 2019-03-09 }}</ref>

==文化==
在[[藏区|藏区]]人们认为那些对部族有功勋的先祖之灵寄住在山水树木中并保护着子孙后代，<ref name="3R"/>“拉则”是在[[青海省|青海省]]世代居住的民族，以[[藏族|藏族]]为主包括[[蒙古族|蒙古族]]、[[土族|土族]]及[[汉族|汉族]]等各民族人们供奉、祭祀这些[[世俗神灵|世俗神灵]]的而修建的城堡和宫殿。<ref name="2R"/>拉则内部供奉着诸多神灵，有主尊玛沁雪山及创世间九尊神、二郎神、文昌爷，还有藏、蒙、土等民族的山神水神。<ref name="3R"/>
