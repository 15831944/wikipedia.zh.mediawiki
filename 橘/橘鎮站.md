'''橘鎮站'''（{{jpn|j=オレンジタウン駅|hg=オレンジタウンえき|rm=Orenjitaun eki|f=y}}）是一位於[[日本|日本]][[香川縣|香川縣]][[讚岐市|讚岐市]]志度，隸屬於[[四國旅客鐵道|四國旅客鐵道]]（JR四國）的[[鐵路車站|鐵路車站]]。車站編號為'''T18'''，是線內最新設立的車站，亦是JR四國成立以後第3個新設的車站，於1998年啟用。本站主要靠停普通列車，每天朝早及晚上則共提供7班特急列車停靠，若是不計算特急全通過的車站，則是最少特急列車停靠的車站。

本站是JR四國眾車站中唯一以全[[片假名|片假名]]標示的車站，事實上在日本，這類以全片假名標示的亦很少見（較著名的有位於[[北海道|北海道]]的{{ST|新雪谷|1}}（{{lang|ja|ニセコ駅}}）、{{ST|苫鵡|1}}（{{lang|ja|トマム駅}}）和位於[[大阪府|大阪府]]的{{ST|環球影城|1}}（{{lang|ja|ユニバーサルシティ駅}}）等）。車站的命名是因為附近為JR四國屬下專營房地產事業的全資子公司[[四鐵不動產|四鐵不動產]]（{{lang|ja|よんてつ不動産}}）所斥資建成的住宅-{zh-tw:社區;zh-hk:小區;zh-cn:小区;}-「Orange Town」而得來<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/koutoku/ST-ORENGE-T.htm |title=存档副本 |accessdate=2012-12-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20150504225139/http://www.shikoku.org.uk/JR/LINE/koutoku/ST-ORENGE-T.htm |archivedate=2015-05-04 }} 非官方車站介紹。</ref>。本來JR四國以興建車站來吸引居民乘搭，但由於-{zh-tw:社區;zh-hk:小區;zh-cn:小区;}-鄰近國道出入口，加上大部份居民都擁有車輛，因此一直以來使用率均不高。

==車站結構==
車站建有以[[鋼筋混凝土|鋼筋混凝土]]、鐵架及[[玻璃|玻璃]]所建成的單層式站舍一座，位於東邊出入口。<ref>[http://hacchi-no-he.net/line/koutoku/station/0100_orangetown.htm 另一個非官方車站介紹。]</ref>車站可再分成兩部份：右邊為候車室，以半開放式的設計打造，其中超過一半的外牆皆以全落地玻璃幕牆，以加強自然光線透入車站內，左邊的候車座位則用回混凝土磚牆，以配合左側的混凝土建築部份。改札口位於候車室出入口對面，並連接天橋前往月台及西邊出入口。整個候車室設有候車大堂及座位、自動售票機及公眾電話亭。

至於左邊部份則為洗手間及儲物室，外牆均以磚及混凝土為主材料，明顯與右邊的候車大堂區分清楚，頂部則加設了以裝飾為主，由幾何圖案所組成的屋頂。

設有[[島式月台|島式月台]]1個，共提供1面2線的配置。其中2號月台為主軌道，亦是大部份列車停靠的月台；1號月台為避線月台，只在
需要避車時或列車交換時才會駛入。<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/koutoku/ST-ORENGE-T_h2.jpg |title=存档副本 |accessdate=2012-12-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20150505075116/http://www.shikoku.org.uk/JR/LINE/koutoku/ST-ORENGE-T_h2.jpg |archivedate=2015-05-05 }} 一輛普通列車停靠在遠處的1號月台，同時一輛往[[德島站|德島站]]方向的特急「渦潮」正由2號月台通過車站，圖片取自非官方車站介紹。</ref>有效長度為4輛，其中月台末端向[[造田站|造田站]]方向約1輛列車長度為有蓋候車亭，並且連接至有蓋[[行人天橋|行人天橋]]，分別通往車站站舍及西邊出口。<ref>[http://www.shikoku.org.uk/JR/LINE/koutoku/ST-ORENGE-T_h.jpg 圖片取自非官方車站介紹。] {{webarchive|url=https://web.archive.org/web/20150505081702/http://www.shikoku.org.uk/JR/LINE/koutoku/ST-ORENGE-T_h.jpg |date=2015-05-05 }}</ref>兩邊路軌當離站後均設有緊急轉轍器及緊急路軌。

月台原來亦有無障礙通道，在往{{ST|志度|1}}方向的月台末端設有斜道，並橫過2號月台路軌的[[平交道|平交道]]後，沿路前往西面出入口，但目前兩邊通道都已被閘門所封閉，平常時間不能打開使用。
