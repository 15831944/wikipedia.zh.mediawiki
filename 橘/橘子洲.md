'''橘子洲'''位于[[湖南省|湖南省]][[长沙市|长沙市]]城区的[[湘江|湘江]]中，是一个狭长的[[沙洲|沙洲]]<ref>{{cite web |url=http://www.enghunan.gov.cn/wwwHome/200908/t20090809_167688.htm |title=Orange Isle to Hold the Opening Ceremony of Hunan International Tourism Festival |accessdate=2009 |publisher= |work= |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20140312213022/http://www.enghunan.gov.cn/wwwHome/200908/t20090809_167688.htm |archivedate=2014-03-12 }}</ref>，又名'''桔子洲'''、'''桔洲'''、'''橘洲'''、'''水陆洲'''。南北全长约5公里；东西最宽处约140米，最窄约40米。洲上有居民居住，现因为整体规划已迁出许多<ref name="fish">{{cite web |url=http://english.zhangjiajie.gov.cn/?action/viewnews/itemid/1313.html |title=Changsha Orange Isle |accessdate=2012 |publisher= |work= |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20140312224904/http://english.zhangjiajie.gov.cn/?action%2Fviewnews%2Fitemid%2F1313.html |archivedate=2014-03-12 }}</ref>。橘子洲于2001年成为长沙[[岳麓山风景名胜区|岳麓山风景名胜区]]橘子洲景区。每周六晚8:30～8:50放焰火（高考、中考期间和冬季取消），焰火当天从6:00就会禁止进入橘子洲，并且[[长沙地铁2号线|长沙地铁2号线]]会跳过[[橘子洲站|橘子洲站]]而不停站（有时候[[湘江中路站|湘江中路站]]也不停站）。

2016年8月3日，[[国家旅游局|国家旅游局]]宣布撤消橘子洲旅游区5A景区资质，原因是景区安全隐患严重、环境卫生差、旅游服务功能严重退化和管理不规范<ref>{{cite news |title=国家旅游局撤消2家5A级景区 严重警告3家5A级景区 新批准6家5A级景区 |url=http://www.cnta.gov.cn/xxfb/jdxwnew2/201608/t20160803_779658.shtml |accessdate=2016-08-03 |agency=[[中华人民共和国国家旅游局|中华人民共和国国家旅游局]] |date=2016-08-03 |archive-url=https://web.archive.org/web/20160816035633/http://www.cnta.gov.cn/xxfb/jdxwnew2/201608/t20160803_779658.shtml |archive-date=2016-08-16 |dead-url=yes }}</ref>。

2017年12月18日，经过全国旅游资源规划开发质量评定委员会组织专家组复核，橘子洲景区恢复5A景区资质。<ref>{{Cite web|url=http://news.163.com/17/1220/12/D63NREKQ00018AOR.html|title=长沙橘子洲景区重回国家5A级 去年8月曾被摘牌|accessdate=2018-02-24|last=网易|work=news.163.com}}</ref>

== 历史 ==
《[[水经注|水经注]]·湘水》：“[[湘水|湘水]]之北径南津城西，西对橘洲。”《[[荆州记|荆州记]]》：“乃至[[夏水|夏水]]怀山，诸州皆没，橘州独在。”

长沙的名称来历中的一种说法就是因为橘子洲形成之前是江中一道长长的沙滩，因此得名“长沙”。据史料的记载，橘子洲形成于[[晋惠帝|晋惠帝]]永兴二年（公元305年）。[[唐朝|唐朝]]时洲上盛产南橘，在江汉地区盛销。[[杜甫|杜甫]]有诗“桃源人家易制度，橘洲田土仍膏腴”。[[潇湘八景|潇湘八景]]中“江天暮雪”就是描绘橘子洲的。洲上原有水陆寺、拱极楼、江心楼，现已不复存在。据清《长沙县志》记载，橘子洲分上洲、中洲、下洲，即牛头洲、水陆洲、傅家洲。历史上三洲只有在水位极低的情况下才出现首尾相连的情形，所以长沙古谚有-{云}-：“三洲连，出状元。”但近代以来，由于泥沙淤积，现除了傅家洲在水位高时隔断外，三洲基本一体。[[清朝|清朝]]末年1904年长沙对外开放，辟为商埠，自1911年起，洲上开始形成外侨居住区，包括[[英国|英国]]、日本[[领事馆|领事馆]]、海关税务司，以及英、德、美、日商人住宅。[[民国|民国]]时期[[唐生智|唐生智]]也在此有公馆。如今还有少数建筑保存下来。<ref name="bee">{{cite web |url=http://www.hn.xinhuanet.com/2013-12/31/c_118779179.htm |title=140年前的水陆洲，江神庙、拱极楼曾热闹非凡续 |accessdate=2013 |publisher=Xinhua |work= |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20160304030817/http://www.hn.xinhuanet.com/2013-12/31/c_118779179.htm |archivedate=2016-03-04 }}</ref>

橘子洲头位于橘子洲的最南端，现在是'''橘洲公园'''的一部分。[[朱熹|朱熹]]、[[张栻|张栻]]当年往来于湘江之上，就绕过橘子洲头。[[毛泽东|毛泽东]]从年轻时到建国后多次来这里观景、游泳。<ref>{{cite web |url=http://news.ifeng.com/history/zhiqing/ziliao/200912/1225_6858_1487913.shtml |title=伟人伫立橘洲头 毛泽东雕像背后的故事 |accessdate=2009 |publisher=Ifeng |work= |language= zh }}</ref>1925年他在此写下《[[沁园春|沁园春]]·长沙》，发出“问苍茫大地，谁主沉浮”的感慨。建国后他七次到橘子洲头附近的湘江游泳。1974年，81岁的毛泽东最后一次回到湖南长沙，10月15日仍然来到橘子洲头追思。1982年橘洲公园建成，公园里有毛泽东的“橘子洲头”的题字和《沁园春·长沙》的词一体的碑，亭榭，成片的橘林。

2009年12月20日，[[毛泽东青年艺术雕塑|毛泽东青年艺术雕塑]]在橘子洲矗立。<ref>{{cite web |url=http://news.ifeng.com/mainland/200912/1227_17_1489249.shtml |title=32米高青年毛泽东像矗立橘子洲头 |accessdate=2009 |publisher=Ifeng |work= |language= zh }}</ref><ref>{{cite web |url=http://news.rednet.cn/c/2009/12/26/1879072.htm |title=毛泽东诞辰116周年：橘子洲毛泽东雕塑揭幕 |accessdate=2009 |publisher=Red.net |work= |language= zh }}</ref>

从2010年起，橘子洲每年举办“橘子洲国际音乐节”。<ref>{{cite web |url=http://news.xinhuanet.com/english2010/photo/2010-09/23/c_13525524.htm |title=Orange Isle Music Festival kick off in C China |accessdate=2010 |publisher=Xinhua |work= |language=en }}</ref><ref>{{cite web |url=http://www.echinacities.com/news/2013-Orange-Isle-Music-Festival-to-Rock-Changsha-Sept-8-9 |title= 2013 Orange Isle Music Festival to Rock Changsha Sept 8-9 |accessdate=2013 |publisher= |work= |language=en }}</ref><ref>{{cite web |url=http://www.enghunan.gov.cn/SP/Orange_Isle_music_festival/ |title=the 2012 Orange Isle Music Festival |accessdate=2012 |publisher= |work= |language=en }}</ref><ref>{{cite web |url=http://www.enghunan.gov.cn/SP/OrMusic/index.html |title=3rd Orange Isle Music Festival |accessdate= 2013 |publisher= |work= |language=en }}</ref>

2014年4月29日，随着[[长沙地铁2号线|长沙地铁2号线]]一期工程的开通试运营，[[橘子洲站|橘子洲站]]开放运营。
