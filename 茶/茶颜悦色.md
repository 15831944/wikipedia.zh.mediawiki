'''茶颜悦色'''是中国[[湖南|湖南]][[长沙|长沙]]的一个中式古风茶饮品牌，由小葱创立于2013年，经营着多家[[连锁店|连锁]]茶饮店，目前仅在长沙开设分店，不在其他城市开设分店<ref name="声明">{{cite web |title=声明：茶颜悦色从没有股东另创品牌，也并未在澳大利亚开店 |url=https://mp.weixin.qq.com/s/6FaSAdEym7feF1puNBtlyg |publisher=茶颜悦色官方微信公众号 |accessdate=2018-11-30 |date=2018-08-23}}</ref>。「知乎，茶也」则是茶颜悦色旗下的[[文化创意品牌|文化创意品牌]]。茶颜悦色在制作茶饮时强调「中茶西做」，是长沙最受欢迎的本土茶饮店。在形象上，茶颜悦色的[[商标|商标]]图、饮料名称、包装、店铺装修等都体现了传统的中国古典风格<ref name="邓痕痕">{{cite web |author1=邓痕痕 |title=喝一杯长沙特色的「茶颜悦色」，二三线城市茶饮品牌也要崛起了？ {{!}} 早起看早期 |url=http://www.sohu.com/a/134146207_114778 |website=搜狐 |publisher=36氪 |accessdate=2018-08-09 |date=2017-04-15}}</ref>。

== 理念 ==
茶颜悦色强调「中茶西做」，在整体设计上使用了很多中国古典元素，店铺[[招牌|招牌]]配有「越中国，更时尚」的标语。从商标来看，茶颜悦色的商标是一个手持[[纨扇|纨扇]]的中国古装打扮的女子；从茶饮名称来看，茶颜悦色使用了古典的中文名称，如把[[红茶|红茶]]称为[[:wikt:紅顏|红颜]]，把[[绿茶|绿茶]]称为浣纱绿，把增加了[[坚果|坚果]]、[[巧克力|巧克力]]等配料的[[奶盖|奶盖]]茶称为[[豆蔻|豆蔻]]<ref name="邓痕痕" />，也有饮品取名为蔓越阑珊、人间烟火、素颜锡兰等四字词语<ref>{{cite web |author1=美食小情报 |title=中茶西做第一品牌，茶颜悦色单品推荐~ |url=http://wemedia.ifeng.com/74815357/wemedia.shtml |website=大风号 |accessdate=2018-11-30 |date=2018-08-22}}</ref>；从产品包装来看，饮料的包装纸杯外侧面印刷有一些中国古画；从店铺装修来看，茶颜悦色也使用了古代中式装修风格；从其旗下的品牌来看，其文化创意品牌名为「知乎，茶也」，[[化用|化用]]了熟语之乎者也<ref>{{cite web |title=茶颜悦色：古典气质的新中式茶饮 |url=http://hunan.voc.com.cn/xhn/article/201705/201705160911448206.html |website=新湖南 |publisher=大湘菜报 |accessdate=2018-11-21 |date=2017-05-16}}</ref>，其首家概念店名为「方寸间·桃花源」，创意来源于中国[[晋朝|晋朝]]文人[[陶渊明|陶渊明]]的{{单双书号转换|[[桃花源记|桃花源记]]}}<ref>{{cite web |author1=零售老板内参APP |title=茶颜悦色开出首家概念店，表达“桃源不在别处在心间” |url=https://36kr.com/p/5143814.html |website=36氪 |accessdate=2018-11-21 |date=2018-07-18}}</ref>。

== 争议 ==
=== 牌照不全及食品添加剂争议 ===
2017年3月，茶颜悦色开设于[[德思勤城市广场|德思勤城市广场]]的分店因为缺少[[食品经营许可证|食品经营许可证]]而被长沙市[[雨花区|雨花区]][[国家食品药品监督管理局|食品药品监督管理局]]查封，同时，执法人员检查出其在使用[[食品添加剂|食品添加剂]][[氧化亚氮|氧化亚氮]]时没有严格遵守[[食品添加剂管理“五专制度”|食品添加剂管理“五专制度”]]。随后，茶颜悦色的其他分店也宣布关闭店铺进行自查，并在[[微信公众号|微信公众号]]等平台上向消费者致歉。月底，经过整改和执法部门的复查，茶颜悦色各分店恢复营业。对于此次风波，一些消费者表示认可茶颜悦色的整改态度，对茶颜悦色表示信任。<ref>{{cite web |title=茶颜悦色所有门店恢复营业 |url=http://money.163.com/17/0325/08/CGC3HDDH002580S6.html |website=搜狐财经 |publisher=潇湘晨报 |accessdate=2018-11-21 |date=2017-03-25}}</ref><ref>{{cite web |title=长沙茶颜悦色主动关闭5家门店自查 |url=http://news.163.com/17/0311/12/CF8D3FJK000187VG.html |website=网易新闻 |accessdate=2018-11-21 |date=2017-03-11}}</ref><ref>{{cite web |author1=陈露菡 |title=茶颜悦色向消费者道歉 承认部分店铺证照不全 |url=http://hn.people.com.cn/n2/2017/0313/c195194-29842889.html |website=人民网 |publisher=潇湘晨报 |accessdate=2018-08-09 |date=2017-03-13}}</ref>

=== 冒名争议 ===
鉴于茶颜悦色的名气和茶饮质量，有一些茶饮店使用了和茶颜悦色相似甚至相同的店名和商标，使得消费者难以区分，购买到一些不是茶颜悦色的产品。对此，茶颜悦色在官方[[新浪微博|新浪微博]]和微信公众号上发表声明，称其为[[直营店|直营店]]而非[[加盟店|加盟店]]，也未在长沙以外的城市开设分店，并对[[澳大利亚|澳大利亚]][[悉尼|悉尼]]开设分店一事进行辟谣，澄清悉尼的茶颜悦色并非官方所开的分店。<ref name="声明" /><ref>{{cite web |title=长沙“茶颜悦色”陷商标之争 竟是由两家公司经营 |url=http://hunan.sina.com.cn/news/2017-05-28/detail-ifyfqqyh8855516.shtml |website=新浪湖南 |accessdate=2018-08-09 |date=2017-05-28}}</ref>
