'''茶葉蛋事件'''，指的是對2011年的一档[[台湾|台湾]][[綜藝節目|综艺]]和談話性節目對[[中國大陸|中國大陸]][[社會|社會]]現狀的討論，其中包含[[台灣人|台灣人]]對中國大陆[[經濟|經濟]]、[[地域|地域]]、[[風俗|風俗]]習慣和[[消費|消費]]水準等各方面的討論，是对以上种种言论的统称。

== 事件起因 ==
有[[網友|網友]]於2011年在[[中國大陸|中國大陸]]的[[社群網站|社群網站]][[天涯社區|天涯社區]]發帖，內容指出台灣教授[[高志斌_(台灣)|高志斌]]於臺灣[[中視|中視]]2011年8月13日播出的《美食in象勇闖大陸市場發展》上說中國大陸百姓收入低消費不起[[茶葉蛋|茶葉蛋]]，吃不起冷凍[[水餃|水餃]]及在[[深圳火車站|深圳火車站]]吃[[泡麵|泡麵]]而引來五六十人的圍觀等誇張言論<ref>{{Cite web|url=https://www.youtube.com/watch?v=AbdZhIy9A0I|title=美食in象2011-08-13 PT.1/3勇闖大陸市場發展【皇家貴族派】.wmv|accessdate=2018-11-01|publisher=美食映象整合行銷顧問股份有限公司|date=2011-08-16}}</ref>，隨後引來眾多網友嘲諷<ref>[http://news.163.com/14/0328/20/9OETQA5100014JB5.html "茶叶蛋事件"当事人:我没主观恶意 没想贬低大陆] {{Wayback|url=http://news.163.com/14/0328/20/9OETQA5100014JB5.html |date=20140903143319 }} 网易新闻 2014年3月28日</ref><ref>{{cite news|title=台灣教授回應大陸“吃不起茶葉蛋”：特指大半鄉鎮|url=http://big5.huaxia.com/tslj/lasq/2014/03/3816028.html|publisher=華夏經緯網|date=2014-03-28|accessdate=2018-11-01}}</ref><ref>{{Cite book|title=热词新语翻译谭 4|last=[[陈德彰|陈德彰]]编|first=|publisher=[[中译出版社|中译出版社]]|year=2016.01|isbn=978-7-5001-4355-0|location=|pages=226}}</ref><ref>{{Cite book|title=互联网时代的公共关系|last=[[蔡国栋|蔡国栋]]|first=|publisher=北京：[[红旗出版社|红旗出版社]]|year=2016.04|isbn=978-7-5051-3702-8|location=|pages=151}}</ref>。

== 事件高潮 ==
2013年11月2日，又有網友於天涯社區貼文，其題目為《【热】发现一个好玩的贴，原来台湾人是这么看我们的！》<ref>[http://bbs.tianya.cn/post-funinfo-4734307-1.shtml 〖天涯头条〗哈哈哈哈，原来台湾人是这么看我们的！] {{Wayback|url=http://bbs.tianya.cn/post-funinfo-4734307-1.shtml |date=20140806121018 }} 天涯社區 2013年11月2日</ref>，文中貼滿了各種台灣綜藝節目、談話性節目和政論節目等各式各樣的截圖，圖片顯示的内容包括“大陸人消費不起茶葉蛋”、“消費不起冷凍水餃”、“大陸人不敢追台灣女生，因為台灣女生太漂亮怕高攀不起”，“大陸孕婦以前不用坐月子，是台灣把風潮带到大陸的時候才流行起來”等各種言論，這些言論隨即遭到大量中國大陆網友回覆，到11月7日為止，回覆量達到8012次，點閱率達到1276304人次。然而天涯社區的貼文並沒有得到廣泛強烈的回應，事件的轉折點為11月2日名為“八卦_我实在是太CJ了”的中國大陆[[新浪微博|新浪微博]]網友將天涯的帖子和圖片轉載到微博後<ref>[http://www.weibo.com/1742121542/AgSb4b7bZ 微博原文] {{Wayback|url=http://www.weibo.com/1742121542/AgSb4b7bZ |date=20140324094727 }} 新浪微博 2013年11月2日</ref>，立即引來眾多中國大陆網友轉發和評論。<ref>[http://zy.takungpao.com/2014/0325/114303.html "大陆人吃不起茶叶蛋" 遭吐槽 网友纷纷晒"蛋"炫富] {{webarchive|url=https://web.archive.org/web/20141018155922/http://zy.takungpao.com/2014/0325/114303.html |date=2014-10-18 }} 大公网 2014年3月25日</ref><ref>[http://news.sohu.com/20140329/n397408738.shtml “茶叶蛋”当事人：台北也有人吃不起茶叶蛋] {{Wayback|url=http://news.sohu.com/20140329/n397408738.shtml |date=20141017024518 }} 搜狐网 2014年3月29日</ref><ref>[http://news.ifeng.com/gundong/detail_2013_11/05/30973838_0.shtml 网友盘点台湾综艺节目对大陆误读 让人大跌眼镜] {{Wayback|url=http://news.ifeng.com/gundong/detail_2013_11/05/30973838_0.shtml |date=20140903172329 }} 凤凰网 2013年11月05日</ref>

2014年1月，一名广州老板在市区[[珠江新城|珠江新城]]自费购买并免费派送10万个茶叶蛋，以回应高志斌言论<ref>{{Cite news|url=http://cn.rfi.fr/%E4%B8%AD%E5%9B%BD/20140401-%E5%9B%9E%E5%BA%94%E5%8F%B0%E6%B9%BE%E6%95%99%E6%8E%88%E5%A4%A7%E9%99%86%E4%BA%BA%E5%90%83%E4%B8%8D%E8%B5%B7%E8%8C%B6%E5%8F%B6%E8%9B%8B%E8%A8%80%E8%AE%BA-%E5%9C%9F%E8%B1%AA%E5%85%8D%E8%B4%B9%E6%B4%BE%E9%80%8110%E4%B8%87%E4%B8%AA%E8%8C%B6%E5%8F%B6%E8%9B%8B|title=回应台湾教授大陆人吃不起茶叶蛋言论 土豪免费派送10万个茶叶蛋|date=2014-04-01|newspaper=法广|accessdate=2018-10-30|language=zh-CN}}</ref>。

== 事实求证 ==
對於台灣專家對“茶葉蛋”的一則言論，有新聞記者曾經求證當事人高志斌，高志斌在電話解釋為“各地的經濟及風俗民情不同，你不能用[[溫州|溫州]]去跟內地的（相比），或者是中國大陸的幾個都市，[[鄭州|鄭州]]或[[河南|河南]]的[[鄉村|鄉村]]（相比），它的確是有一大半的鄉鎮城市連一顆茶葉蛋都買不起。”但此番言論卻引來中国大陆網友回嗆，說河南無故中槍，並有網友回覆“我河南三線小城市，我能說茶葉蛋大街上確實没有，因為我們都不喜歡吃那玩意兒”<ref name="厦门卫视">[http://v.youku.com/v_show/id_XNjMwOTc5MDQ4.html “大陆人吃不起茶叶蛋”？两岸网友掀论战！-厦门卫视-两岸新新闻] {{Wayback|url=http://v.youku.com/v_show/id_XNjMwOTc5MDQ4.html |date=20131109090147 }}优酷网引用厦门卫视“两岸新新闻”节目。引用于2013年11月7日</ref>。

== 评论 ==
中国大陆网络异议作家[[黄童超|黄童超]]称：2014年中国官方划定的贫困线标准是农村人均纯收入2800元/年，折合7.67元/天，按照汇率换算约等于1.11美元/天，而世界银行的贫困线标准是1.9美元/天。据[[中华人民共和国统计局|中国统计局]]的数据，截至2014年底，中国[[农村|农村]]贫困人口有7017万人，这还不包括城镇贫困人口<ref>{{cite web |title=网易浪潮 | 中国人真吃得起茶叶蛋了？|url=https://chinadigitaltimes.net/chinese/2016/12/%E7%BD%91%E6%98%93%E6%B5%AA%E6%BD%AE-%E4%B8%AD%E5%9B%BD%E4%BA%BA%E7%9C%9F%E5%90%83%E5%BE%97%E8%B5%B7%E8%8C%B6%E5%8F%B6%E8%9B%8B%E4%BA%86%EF%BC%9F/ |website=chinadigitaltimes.net |accessdate=2018-11-12}}</ref>。据中国统计局一份研究显示，收入刚好在贫困线上的中国人，会将收入的53.5%用于购买食品，也就是人均每天食品消费支出约为4.1元。所以据此看来，处在贫困线附近或以下的近一亿中国人，日常不可能消费得起茶叶蛋。（省会郑州茶叶蛋1元，贫困地区茶叶蛋更便宜）
即使农村地区的很多人没有被列为“贫困人口”，但是他们的生活质量也并不高。2014年中国农村居民人均支出为8382元，其中在食品烟酒上的支出为2814元，每天只有约7.7元，也不会每天都能吃上茶叶蛋。在山西、河南、贵州、云南、陕西、甘肃、宁夏等较为不发达的省份，这个数字会更低<ref name=真的吃不起茶叶蛋>{{cite web |title=许多中国人穷得超出你想象，他们到今天也吃不起茶叶蛋 | 浪潮工作室 |url=http://view.163.com/16/1214/00/C8732TAV00018ASD.html |website=view.163.com |accessdate=2018-11-12}}</ref>。
