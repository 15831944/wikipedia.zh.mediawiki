'''近賀由香里'''{{BD|1984年|5月2日||}}是一位[[日本|日本]]女子[[足球|足球]][[運動員|運動員]]，目前效力於[[L聯賽|L聯賽]]的[[INAC神戶雌獅|INAC神戶雌獅]]，位置[[後衛_(足球)|後衛]]及[[中場|中場]]。[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]]出身，湘南學院高等學校、[[日本體育大學|日本體育大學]]體育學科畢業。獲得過第15回L聯賽（2003年）的新人獎，也曾代表[[日本國家女子足球隊|日本]]參加[[2007年女子世界盃足球賽|2007年女子世界盃足球賽]]、[[2011年女子世界盃足球賽|2011年女子世界盃足球賽]]與[[2008年夏季奧林匹克運動會|2008年奧運會]]、[[2012年夏季奧林匹克運動會|2012年奧運會]]，其中2011年女子世界盃日本奪下史上首座冠軍，2012年奧運會則是全部6場賽事踢完全場，最終獲得銀牌。

== 來歷 ==
小學三年級時，因為受到比自己年長4歲的哥哥影響，進入了東汲澤足球俱樂部踢球，並入選男女混合隊伍，開啟了足球生涯。升上橫濱市立戶塚中學校後，經由其他學校教練的介紹，加入女子足球名門「橫須賀海鷗」<ref>{{cite web |url=http://www.e-totsuka.info/s-iku/index-sp03.php |title=存档副本 |accessdate=2013-02-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20120510012127/http://e-totsuka.info/s-iku/index-sp03.php |archivedate=2012-05-10 }}</ref>。當時她的位置是[[前鋒_(足球)|前鋒]]，直到轉入聯賽後才改變。在當時，為球隊連續兩年獲得全國大會冠軍、以及一年季軍的成績作出貢獻。畢業後，與隊中的搭檔[[矢野喬子|矢野喬子]]進入湘南學院高等學校女子足球隊，並在期間球隊替學校奪得高校選手權大會兩座亞軍。在15歲時，她就入選了日本女子U-18，之後也於各年齡別代表隊成為在場上活躍的主力球員。到高校三年級，入選日本女子U-20代表隊，參加在[[印度|印度]][[果阿邦|果阿邦]]舉行的2002年[[亞足聯U19女子青年錦標賽|亞足聯U19女子青年錦標賽]]，並獲得冠軍。同年8月，再次入選代表隊，參與[[U19女子世界盃|U19女子世界盃]]，最終日本止於半準決賽卒業後。同年12月，首次被選入國家隊，參加相關的訓練。

還在[[日本體育大學|日本體育大學]]體育學部在學時，2003年加入[[日視美人|日視美人]]，該年以22場聯賽出賽20場，並攻入6個進球的成績，獲得年度新人獎。2004年，以[[中場|中場]]位置的身分進入[[2004年夏季奧林匹克運動會|2004年奧運會]]的代表隊預備名單，但未入選。隔年2005年3月29日，在[[友誼賽|友誼賽]]對上[[澳洲國家女子足球隊|澳洲]]時首次代表國家隊先發上場。另外，她在大學一、三年級時，也曾參加[[2003年世界大學生運動會|2003年]]與[[2005年世界大學生運動會|2005年]]的[[世界大學生運動會|世界大學生運動會]]，團隊奪得一銀一銅。

2007年，以[[後衛_(足球)|後衛]]位置參加[[2007年女子世界盃足球賽|2007年女子世界盃足球賽]]，在面對[[阿根廷國家女子足球隊|阿根廷]]的比賽時幫助攻入決勝進球，讓[[永里優季|永里優季]]在補時階段絕殺對手，這也是日本在該屆賽會的唯一一場勝利。隔年[[2008年夏季奧林匹克運動會|2008年奧運會]]，她全部比賽完整全場上陣，努力地面對較為陌生的防守方面，並積極從側翼製造機會，在第三戰對上[[挪威國家女子足球隊|挪威]]，在第31分鐘打入生涯首個國家隊進球，並且傳出一助攻，幫助日本5比1逆轉大勝挪威。2010年，在國家隊攻入了兩個進球。

2011年1月，在經過8年之後，與[[澤穗希|澤穗希]]、[[大野忍|大野忍]]、[[南山千明|南山千明]]轉隊至[[INAC神戶雌獅|INAC神戶雌獅]]<ref>[http://www.verdy.co.jp/beleza/belezatop/bnews/b2010/%E9%81%B8%E6%89%8B%E7%A7%BB%E7%B1%8D%E3%81%AE%E3%81%8A%E7%9F%A5%E3%82%89%E3%81%9B.html 選手移籍のお知らせ] {{Wayback|url=http://www.verdy.co.jp/beleza/belezatop/bnews/b2010/%E9%81%B8%E6%89%8B%E7%A7%BB%E7%B1%8D%E3%81%AE%E3%81%8A%E7%9F%A5%E3%82%89%E3%81%9B.html# |date=20121201113708 }} [[日視美人|日視美人]]</ref>。同年7月，以邊後衛的身分在[[2011年女子世界盃足球賽|2011年女子世界盃足球賽]]全部完整出賽，並有一助攻的紀錄，能於球門危急之際及時解圍，並積極參與攻擊與堅實的守備表現，幫助日本獲得冠軍。隔年2012年，於[[2012年夏季奧林匹克運動會|2012年奧運會]]全部比賽完整出賽，最終獲得銀牌。

2012年12月22日，皇后杯準決賽對上[[浦和紅鑽女子隊|浦和紅鑽女子隊]]的比賽中，上半場第25分鐘時因為右膝受傷退場，檢查結果為[[前十字韌帶|前十字韌帶]]斷裂以及[[半月板|半月板]]嚴重損傷，需要6個月時間治療<ref>{{cite web |url=http://sankei.jp.msn.com/sports/news/121224/scr12122419500010-n1.htm |title=存档副本 |accessdate=2013-02-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20130125055635/http://sankei.jp.msn.com/sports/news/121224/scr12122419500010-n1.htm |archivedate=2013-01-25 }}</ref>。
