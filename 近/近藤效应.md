'''近藤效应'''（Kondo effect）是指在高[[电导|电导]]无磁[[金属|金属]]母体中磁性杂质[[原子|原子]]对[[传导电子|传导电子]][[散射几率|散射几率]]的增大。

在某一特征温度TK（所谓[[近藤温度|近藤温度]]）以下，稀磁合金的剩余[[电阻率|电阻率]]随[[温度|温度]]的降低而反常增大<ref>{{cite web |first=Alex C |last=Hewson |coauthors=[[Jun_Kondo|Jun Kondo]] |year=2009 |title=Kondo effect |publisher=[[Scholarpedia|Scholarpedia]] |volume=4 |number=3 |page=7529 |doi=10.4249/scholarpedia.7529 |url=http://www.scholarpedia.org/article/Kondo_effect |accessdate=2009-12-18}}</ref>。1964年，[[近藤淳|近藤淳]]从理论上阐明了该效应的形成机制：由于孤立[[磁性|磁性]][[原子|原子]]与传导[[电子|电子]]之间存在RKKY相互作用，围绕磁性原子的传导电子的[[自旋|自旋]]将反平行[[极化|极化]]，从而屏蔽磁性原子的[[磁矩|磁矩]]，并形成一个多体单态。其他传导电子与这个多体单态之间不再有“自旋相同，空间回避”的限制，结果，磁性杂质对传导电子的散射变得更为有效，[[电阻|电阻]]率增大。
