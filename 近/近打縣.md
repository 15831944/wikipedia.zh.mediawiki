'''近打縣'''（{{lang-ms|Kinta}}）是[[馬來西亞|馬來西亞]][[霹靂州|霹靂州]]的一個縣，位於州的東部、[[蒂蒂旺沙山脈|蒂蒂旺沙山脈]]西麓，東邊和[[彭亨州|彭亨州]]接壤。面積1,038平方公里，{{tsl|en|Kinta River|近打河}}自東北往西南斜貫本縣。全縣最高峰{{tsl|en|Mount Korbu|可布峰}}是西馬第二高峰。

2010年人口749,474人，華裔有318,744人，略多於土著（301,629人）。 <ref name="census2010"> {{cite web|url=http://mykampung.sinchew.com.my/node/171318|title=近打及金寶縣依然最多‧霹華裔人口第二多|work=星洲日報|date=2011年12月24日|accessdate=2012年1月23日|deadurl=yes|archiveurl=https://web.archive.org/web/20141221123842/http://mykampung.sinchew.com.my/node/171318|archivedate=2014年12月21日}}</ref> 

该县县府是[[华都牙也|华都牙也]]，霹雳州首府[[怡保|怡保]]位于此县。

==教育==
近打县的学校主要由'''北近打县教育局'''和'''南近打县教育局'''管辖。

北近打县教育局管辖怡保区共62间国民小学、35间华文小学、14间淡米尔小学、41间国民中学、5间华文中学、1间全寄宿中学、及3间宗教中学。<ref>{{cite web |title=Maklumat Asas Pendidikan - Pejabat Pendidikan Daerah Kinta Utara |url=http://jpnperak.moe.gov.my/ppdkinta/index.php/2011-11-25-07-20-55|work=近打北区教育局}}</ref>

南近打县教育局管辖华都牙也区共15间国民小学、10间华文小学、3间淡米尔小学、5间国民中学、1间华文中学、及1间全寄宿中学。<ref>{{cite web |title=Laman Rasmi PPD Kinta Selatan |url=http://jpnperak.moe.gov.my/ppdks/|work=近打南区教育局}}</ref>

===华文小学===
近打县共有45间[[马来西亚华文小学列表|华文小学]]。<ref>{{cite web |title=霹雳华小通讯录 |url=http://web.jiaozong.org.my/doc/srjk/2018/2018_perak.pdf |publisher=马来西亚华校教师会总会（教总）}}</ref>，其中35间属于北近打县教育局管辖、10间属于南近打县教育局管辖。
