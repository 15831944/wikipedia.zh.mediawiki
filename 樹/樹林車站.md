'''樹林車站'''位於[[台灣|台灣]][[新北市|新北市]][[樹林區|樹林區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(北段)|縱貫線]]的[[鐵路車站|鐵路車站]]，也是[[東部幹線|東部幹線]]高級對號快車的主要起點站和終點站。

== 車站構造 ==
*兩座[[島式月台|島式月台]]、一座[[側式月台|側式月台]]。
*第三月台於2007年5月3日啟用。
*[[跨站式站房_(台鐵)|跨站式站房]]、購票及服務設施設在3樓大廳，與前後站均有連通。
*前站有兩座樓梯，一座進站、一座出站。
*後站僅有一座樓梯，因為原後站出站樓梯，被徵用為第三月台樓梯使用。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#一等站|一等站]]。
*西部幹線主要停靠：區間車，區間快車，莒光號
*東部幹線主要停靠：區間車，區間快車，莒光號，自強號，太魯閣號，普悠瑪號
*為[[東部幹線|東部幹線]]（[[宜蘭車站|宜蘭]]、[[花蓮車站|花蓮]]及[[臺東車站|臺東]]方向路線）對號列車的實際起點站及終點站。
*本站為主要起點站和終點站
*區間車（宜蘭線）：皆停靠
*北迴線自強號 ：皆停靠（本站為起點站，終點站）
*北迴線太魯閣號 ：皆停靠（本站為起點站，終點站）
*北迴線普悠瑪號 ：皆停靠（本站為起點站，終點站）
*[[西部幹線|區間車]]  : 皆停靠
*[[莒光號列車|莒光號]] : 除[[觀光列車|環島觀光列車]]外每班都有停靠
*[[自強號列車|自強號]]停靠順行：272次、170次、140次、138次、144次、152次
*停靠逆行：105次、115次、125次、177/6177次、147次、6473次（週日行駛）。
*[[臺鐵區間快車|區間快車]]  ：皆停靠
*[[普悠瑪列車|普悠瑪號]]（山線）、[[太魯閣列車|太魯閣號]]（海線） ：皆不停靠
*鄰近設有[[樹林調車場|樹林調車場]]並由本站管理，供東部幹線各級列車整備、發車使用。
*通往樹林調車場的軌道為平行於[[縱貫線_(北段)|縱貫線]]的第三線（小運轉線），不影響主線行車。
*如果小運轉線或其前後的[[道岔|道岔]]有問題，列車可以臨時經由西正線進出調車場。

== 歷史 ==
*1901年（[[明治|明治]]34年）8月25日：設站，稱為樹林停車場<ref>臺灣總督府府報，1901年8月19日號外</ref>。
*1940年（[[昭和|昭和]]15年）12月：改稱樹林驛。
*1955年1月1日：升格為二等乙站。
*1966年1月1日：升格為二等甲站。
*1985年7月1日：升格為一等站。
*1994年7月26日：舊站房拆除、重新改建，舊站房遷入臨時站房。
*1997年3月14日：樹林調車場啟用，東部幹線始發及終點從板橋站改為樹林站。
*1997年9月27日：[[跨站式站房_(台鐵)|跨站式站房]]落成啟用。
*2007年5月3日：第三月台啟用。
*2008年6月20日：臺北－樹林站間試辦使用悠遊卡乘車。
*2008年8月1日：[[悠遊卡|悠遊卡]]使用範圍擴展至[[基隆車站|基隆]]－[[中壢車站|中壢]]間的19個站。
*2009年7月：第二月台6至7車處設置販賣便當櫃檯<ref>{{Cite web|url=http://www.youtube.com/watch?v=aokkXLBC2CI|title=山寨版阿福號 進駐台鐵樹林站賣便當|accessdate=2009-07-01|publisher=中時電子報|year=2009|}}</ref>。

== 立體化可行性研究 ==
立體化可行性研究原本規劃的是地下化或高架化，
遷站之議最早於2011年4月16日大紀元報導，當時[[新北市|新北市]]副市長[[李四川|李四川]]率先提出遷站構想。
接著在2013年11月5日的[[新北市議會|新北市議會]]議事錄中出現遷移樹林車站的說法，
[[新北市政府|新北市政府]]隨即進行地方說明會，但遇到絕大多數的反彈，反對新北市府遷站的規劃。
在沉寂一年半後，新北市府再次舉辦地方說明會，在多數市民不知道有此說明會的情況下舉辦，
仍有不少民眾表達反對遷站。

102年11月5日
新北市議會於民國102年11月5日 第1屆第6次定期會議事錄中，[[新北市議員|新北市議員]][[陳世榮|(議員) 陳世榮]]與[[新北市市長|新北市市長]][[朱立倫|朱立倫]]的詢答記錄。

節錄其中一段詢答便可看明瞭為何在可行性研究的方案中會增加改線的研究。

陳議員世榮： 說到樹林的一個發展，還有一個很重要的，就是鐵路要地下化或是立體化的一個問題，這個部分真的是影響到未來整個樹林都市發展，你像現在這個案子不決定，以前有案子樹林要辦站前都更，到現在都停頓了。

朱市長立倫： 非常謝謝議員，我還要拜託議員，因為議員也正好提出這個質詢，樹林過去曾經有提過很多次的鐵路捷運化、鐵路高架化、鐵路地下化，高架化、地下化每次在談的是成本的問題，實際上我們在上個禮拜做了一個整體的檢討，樹林地區一定要請交通局請專業的公司要跟幾位議員還有地方的里長先做一次的了解，第一個，就現有的路線，現有的鐵路路線如果要去做地下化，他的困難點不是錢，每次過去都講錢，真正他們能夠地下化因為卡到包括幾個問題，他能夠地下化的長度有限，有些地方又沒有辦法克服。第二個，議員也很清楚，整個樹林現在如果鐵路跟捷運[[萬大－中和－樹林線|萬大－中和－樹林線]]，他事實上是沒有共站，這也是第二個問題。第三個問題，現有的樹林站他的整體開發的空間是非常有限，有點像板橋過去的板橋車站一樣，所以樹林要不要發展出來一個新樹林站，新樹林站如果可以跟捷運一起連線，就是類似像現在新板橋站一樣，鐵路捷運是在一起的。

2010年6月14日大紀元。板橋到樹林鐵路地下化 有譜。李四川擔任台北縣副縣長時，提出採都市計畫、更新方式辦理，獲得交通部支持，並允諾研擬相關要點配合。<ref>[http://www.epochtimes.com/b5/10/6/15/n2937988.htm 板橋到樹林鐵路地下化 有譜]</ref>

2012年8月21日 交通部制定「鐵路立體化建設及周邊土地開發計畫申請與審查作業要點」，而此作業要點被曲解成可以遷移車站。<ref>[http://motclaw.motc.gov.tw/Law_ShowAll.aspx?LawID=D0125000&Mode=0&PageTitle=%e6%a2%9d%e6%96%87%e5%85%a7%e5%ae%b9 鐵路立體化建設及周邊土地開發計畫申請與審查作業要點]</ref>
