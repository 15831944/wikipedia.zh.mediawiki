'''樹才怪'''（{{lang-ja|'''ウソッキー'''}}；{{ lang-en|'''Sudowoodo'''}}）是[[精靈寶可夢|精靈寶可夢]]系列登場的虛構角色（怪獸）中的其中一種。

== 特徵 ==
樹才怪為岩石屬性的模仿寶可夢。有著樹般的外型，並會偽裝成樹的樣子，身體構造較類似於岩石，非常怕水。

== 遊戲中的樹才怪 ==
*[[神奇寶貝_金·銀|金、銀、水晶版]]中首次登場的寶可夢，全國圖鑑編號為185，城都聯盟編號為106，神奧聯盟編號為93。
*攻擊與防禦皆有不錯的數值，特攻與速度的數值偏低。
*特性為「結實」與「堅硬腦袋」。「結實」的效果為不會被對手一擊必殺的招式影響；「堅硬腦袋」則是在使用會傷害自己的招式時不會減少自身的HP。
*[[神奇寶貝_金·銀|金、銀、水晶版]]和[[神奇寶貝_心金·魂銀|心金、魂銀版]]中，在36號道路對著樹才怪使用「傑尼龜噴壺」後可進入對戰；[[神奇寶貝_紅寶石·藍寶石|綠寶石版]]中，則是在對戰開拓區對著樹才怪使用「吼吼鯨噴壺」後進入對戰，而兩者皆只會出現一隻野生的樹才怪。
*[[神奇寶貝_鑽石·珍珠|珍珠、白金版]]中，野生的樹才怪可在214（白金版沒有）與221號道路遇見，而鑽石版中則無野生種。
*[[神奇寶貝_鑽石·珍珠|鑽石、珍珠、白金版]]中，可以學會格鬥屬性的招式「臂錘」，並能藉由「心之鱗片（{{lang|ja|ハ—トのウロコ}}）」回憶草屬性的強力招式「木槌」。

== 動畫中的樹才怪 ==
*無印（城都聯盟篇）163集中，小智等人為了能夠渡河，而協助渡船老人的孫女得到樹才怪，但樹才怪善於躲藏與使用招式的能力，使得小智等人吃了苦頭，後來渡船老人的孫女完成了研究，小智等人也順利的渡河。
*《[[神奇寶貝超世代|神奇寶貝超世代]]》172集中出現的黃金樹才怪，本為少年欽也持有的寶可夢，因實驗失敗而讓樹才怪變成金色，使得樹才怪因賭氣離開了欽也，後來在與火箭隊的戰鬥中欽也重新找回了樹才怪的信任，最後樹才怪也恢復了原狀。
*《[[神奇寶貝鑽石&珍珠|神奇寶貝鑽石&珍珠]]》14集中，小剛的盆才怪為了逃離火箭隊的綁架，拚命戰鬥而進化成樹才怪，進化後不再愛哭，並且喜歡在出場時敬舉手禮。
*《[[神奇寶貝鑽石&珍珠|神奇寶貝鑽石&珍珠]]》128集中，由於要教訓搶走寶可夢的火箭隊，學會了「臂錘」。

== 漫畫中的樹才怪 ==
*在《[[神奇寶貝特別篇|神奇寶貝特別篇]]》第三章中，主角阿金所擁有的寶可夢，稱呼牠為「胡太郎」。
