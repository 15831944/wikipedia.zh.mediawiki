'''浙江大学人文学院'''是隶属于[[浙江大学人文学部|浙江大学人文学部]]的一所学院，1999年7月成立，现设中国语言文学系、历史学系、哲学系、艺术学系、文物与博物馆学系5个系。

== 历史沿革 ==

1987年浙江大学复设中文系等人文学科。1992年[[杭州大学|杭州大学]]成立人文学院。1995年浙江大学成立人文学院。1998年新浙大成立后，原杭州大学人文学院、原浙江大学人文学院的大部分与原浙江农业大学、浙江医科大学人文学科相关单位组建的新人文学院于1999年7月成立。<ref name="简介">{{cite web |url=http://www.ch.zju.edu.cn/external/news.php?id=14 |language= |title=学院简介 |publisher=浙江大学人文学院 |date= |accessdate=2012年12月 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 现状 ==

学院专业涵盖文、史、哲三大学科，设有汉语言文学（含影视方向）、古典文献、编辑出版、历史学、文物与博物馆学、哲学、艺术设计、美术本科专业。拥有中国语言文学、历史学和哲学3个一级学科[[博士|博士]]点。<ref name= "简介"/> 有古籍研究所、韩国研究所、日本文化研究所等研究所。

现有教职工180人，其中教授60人，副教授49人，[[博士后|博士后]]工作人员、在读博士生、硕士生、外国留学生大约共800名。<ref name= "简介"/>

著名小说家[[金庸|金庸]]先生担任人文学院名誉院长，现任院长为[[楼含松|楼含松]]，学院行政办公地点在[[浙江大学西溪校区|浙江大学西溪校区]]。

== 重点学科 ==
*2个[[教育部|教育部]]基础学科科学研究和人才培养基地：中文、历史
*“[[985工程|985工程]]”二期国家文科创新研究基地：“基督教与跨文化研究中心”和“语言与认知研究中心”
*[[教育部|教育部]]人文社科重点研究基地：汉语史研究中心
*[[浙江省|浙江省]]哲学社会科学重点研究基地：文献集成编纂中心和宋学研究中心
*国家级重点学科：中国古典文献学
*国家级重点培育学科：外国哲学
*浙江省重点学科：中国古代文学、汉语言文字学、中国古典文献学、中国古代史、外国哲学、文艺学
*校级强所：汉语史研究中心、中国古代文学与文献研究中心、中外文化交流史中心、中国古代史研究所<ref name= "简介"/>
