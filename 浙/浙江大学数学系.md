'''浙江大学数学系'''是[[浙江大学|浙江大学]]的一个学系，现属于[[浙江大学理学部|理学部]]。

== 历史 ==
1929年以来，著名数学家[[陈建功|陈建功]]和[[苏步青|苏步青]]教授先后来浙大数学系工作，形成了著名的“陈苏学派”，培养的院士有[[程民德|程民德]]、[[谷超豪|谷超豪]]、[[夏道行|夏道行]]、[[王元|王元]]、[[胡和生|胡和生]]、[[石钟慈|石钟慈]]、[[沈昌祥|沈昌祥]]，以及[[熊全治|熊全治]]、[[杨忠道|杨忠道]]、[[周元燊|周元燊]]等一批知名学者。近年来又培养了[[林芳华|林芳华]]、[[励建书|励建书]]、[[汪徐家|汪徐家]]等多位杰出青年数学家，为中国[[数学|数学]]的发展做出了重要贡献。<ref name="简介">{{cite web |url=http://www.math.zju.edu.cn/news.asp?id=75&TabName=%CF%B5%CB%F9%BD%E9%C9%DC&SubSort= |language= |title=浙江大学数学系介绍 |publisher=浙江大学数学系 |date= |accessdate=2012年12月 }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 现状 ==
数学系有三个本科专业，国家重点学科有[[基础数学|基础数学]]、[[应用数学|应用数学]]。设有[[博士后|博士后]]流动站、数学一级学科博士点和首批国家[[理科|理科]]人才培养基地。[[数学|数学]]是“[[211工程|211工程]]”重点建设学科，也是浙大CAD&CG国家重点实验室的创办单位。<ref name= "简介"/>

国际著名数学家[[丘成桐|丘成桐]]教授于2002年在浙江大学创建了'''数学科学研究中心'''并任主任，推动了数学学科的快速发展。<ref name= "简介"/>

数学系现有在职教师119人，其中博士生导师42人。<ref name= "简介"/>

== 研究机构 ==
浙江大学数学系设有以下研究所：
*高等数学研究所
*科学与工程计算研究所
*计算机图像图形研究所
*运筹与控制科学研究所
*统计研究所
*系统优化技术研究所

自1990年以来，已获得4项[[国家自然科学奖|国家自然科学奖]]（其中1项为第二获奖单位），2项国家级教学成果奖，40多项省部级教学科研成果奖，每年有40余项在研国家级项目。<ref name= "简介"/>
