'''浙江衛視'''（前名：'''浙江电视台新闻综合频道'''<ref>[https://www.youtube.com/watch?v=_31ByfUisEA 浙江卫视开台 (1994年7月)]</ref>，广电总局正式名称是'''浙江电视台综合频道'''），是[[浙江广播电视集团|浙江广播电视集团]]旗下的[[卫星|卫星]]综合频道，于[[1994年|1994年]][[1月1日|1月1日]]上星播出，在[[中国大陆|中国大陆]]因收视率与[[湖南卫视|湖南卫视]]、[[东方卫视|东方卫视]]、[[江苏卫视|江苏卫视]]旗鼓相当且这四个卫视时常位居省级卫视[[收视率|收视率]]的前四名而被民众从整体上习惯性地称为四大卫视<ref>{{cite web |url=http://ent.ifeng.com/a/20161008/42697092_0.shtml |title=盘点央视（中国中央电视台）和四大卫视的综艺当家男主持人 |publisher=凤凰网娱乐 |author=北京青年网 |date=2016-10-08  |accessdate=2016-10-16  }}</ref><ref>{{cite web |url=http://news.xinhuanet.com/zgjx/2016-05/18/c_135368127.htm |title=2016年4月省级卫视全网传播榜 |publisher=中国记协网 |author=中国新闻出版广电报 |date=2016-05-18 |accessdate=2016-10-16 }}</ref>。浙江卫视具有代表性的节目有[[奔跑吧|奔跑吧]]、[[中国好声音|中国好声音]]等。

在2017年“亚洲电视品牌10强”中，浙江卫视母公司浙江广播电视集团跃居亚洲电视品牌第五（[[中国中央电视台|CCTV]]、[[湖南廣播電視台|HBS]]、[[日本廣播協會|NHK]]、[[凤凰卫视|凤凰卫视]]、[[浙江广播电视集团|ZRTG]]）。

==台标==
浙江卫视的台标原意为“水乡”，一个蓝色的圆角[[矩形|矩形]]，其中中间有一个白色字母“Z”，像钱塘江入海的样子，台标整体寓意[[浙江省|浙江]]是水乡。因应台标嵌入图案像个随和手写的“2”字，故网友亦会将其称为“二台”。

1994年1月1日前，浙江卫视台标为旧台标下加ZTV-1，1994年1月1日上星以后，台标才改为现今所用的新台标，最初为右边加ZTV-1<ref name="Opening Ceremony">[https://www.youtube.com/watch?v=_31ByfUisEA China Zhejiang Television (中国浙江电视台) First Station Ident New Year's Day (1 January) 1994]</ref>，2001年1月1日使用没有任何标注的台标。2005年9月5日，频道画面右下角加上“浙江卫视”字样（使用与浙江广播电视集团一样的字体）。2008年1月1日，该台标改为现在至今使用的台标，即台标及右侧加上“浙江卫视”字样。

对于高清版的频道，其画面右上角有“高清”字样，起初使用与浙江卫视字样一样的字体，之后更换了字体，后调整至右下角（为避免《中国好声音》等节目角标遮挡）。之后一段时间高清版频道的“高清”字样放置于画面右下角，使用的字体与浙江卫视字样相同。现在浙江卫视“高清”字样已经恢复右上角显示，但是很淡，不过在《中国新歌声》播出期间，“高清”字样往左边挪动以避免和角标遮挡，此后浙江卫视取消逢正、半点的报时器显示。2018年9月12日，浙江卫视采用标清16:9播出，台标仍维持4：3比例。

从2012年4月13日起，因迎合《中国梦想秀》第三季开播，浙江卫视除新闻时段外，台标呈动态化，每隔几分钟台标右边的文字“浙江卫视”就会和“第一梦想频道”相互循环切换，一直到2013年开始停用，恢复静态台标。

2008年，浙江卫视提出“中国蓝”理念，提出全力打造“最具活力的中国电视娱乐品牌”，这也是国内所有电视媒体中，对自身品牌进行清晰定位与形象区隔的第二家卫视。之后浙江卫视又推出吉祥物“蓝巨星”，还推出了以蓝巨星为形象的动画片《[[蓝巨星和绿豆鲨|蓝巨星和绿豆鲨]]》<ref>{{cite web|url=http://media.people.com.cn/GB/40606/12509130.html|title=朱丹身价狂涨近50倍 梁冬打造中国版《奥兹医生》 |publisher=杭州网 |accessdate=2016年5月18日 }}</ref>。

2016年8月2日，浙江卫视落地暨开播仪式在澳门有线电视总部举行。即日起，澳门观众通过澳门有线159台就可收看浙江卫视的精彩节目。

==节目==
===节目播出时段===
目前，浙江卫视各时段播出的日常节目内容如下（按播出顺序）：
*上午时段：重播《浙江新闻联播》、动画片、《经典剧场》（周一至周五）、重播综艺节目
*下午时段：《百姓剧场》（周一至周五）、重播综艺节目
*晚间时段
**傍晚时段：播出《新闻深一度》（周一至周六）、《浙江新闻联播》、《今日聚焦》和转播[[中国中央电视台|中央电视台]]《[[新闻联播|新闻联播]]》。
**黄金时段：每天19：30播出[[中国蓝剧场|中国蓝剧场]]（周六播出一集、其余时段播出两集）；每周一至四22:00播出中国蓝周播剧场（两集连播）；每周五至日首播综艺节目。
*凌晨时段：重播综艺节目，之后重播《今日聚焦》、社教节目和纪录（专题）片。

此外，浙江卫视也会于每周一至五的上午播出动画片，以方便儿童观众观看，同时符合广电总局发布的上星综合频道必须一周安排一档少儿类栏目时段的要求。每逢[[寒假|寒假]]、[[暑假|暑假]]期间，频道会特别安排，在此期间会增加动画片的播出量。

2017年10月1日，为贯彻落实《[[中华人民共和国国歌法|中华人民共和国国歌法]]》相关规定，浙江卫视亦会在国庆节、国际劳动节等重要的国家法定节日、纪念日期间须在上午10点整播放国歌，国歌片带与央视播放版本一致<ref>{{cite web |url =http://news.ifeng.com/a/20171001/52251501_0.shtml |title = 广电总局：电台电视台应在重要法定节日播放国歌 |website = 凤凰资讯 }}</ref>。

=== 新聞資訊節目 ===
浙江卫视現階段常規的新聞資訊節目包括《浙江新闻联播》、《今日聚焦》、《新闻深一度》、《今日评说》。曾经播出过的节目包括《新闻超视》、《寻找王》、《新闻直通车》（由原《寻找王》改版而来，最初在傍晚时段播出，后改为早间时段播出）、《午间新闻快报》、《今日证券》等。

在突发、重大事件发生期间（例如登陆浙江，影响和破坏力较大的台风、浙江地区发生的灾害性天气，汶川地震发生后也曾直播），浙江卫视会切断原先播出的节目，以特别直播报道时段代替。

此外，浙江卫视也会于每周三晚播出娱乐新闻节目《好看中国蓝》。

===电视剧===
浙江卫视播出的电视剧涵盖各个题材，均为国产剧。此前浙江卫视也曾播出[[韓劇|韓劇]]和[[港劇|港劇]]，现已不再播出。

目前，浙江卫视有四個主要的电视剧播出時段，分別為《经典剧场》（周一至周五上午时段）、《百姓剧场》（周一至周五下午时段）、《[[中国蓝剧场|中国蓝剧场]]》（每天黄金时段）和《中国蓝周播剧场》（每周一至四晚间时段）。其中《经典剧场》时段為近期浙江卫视《中国蓝剧场》播過的电视剧；《百姓剧场》时段為之前在黄金时段播過的电视剧；《[[中国蓝剧场|中国蓝剧场]]》为日播首播剧播出时段，现周日至周五播出两集，周六播出一集；《中国蓝周播剧场》为周播首播剧播出时段，每周一至四播出两集。曾经设置的电视剧播出时段还包括《假日剧场》（周末白天时段），现已取消。

===综艺节目===
自浙江卫视2008年提出“中国蓝”概念后，浙江卫视综艺节目大幅增加，制作出一系列膾炙人口的娛樂與綜藝節目。著名的节目包括《[[奔跑吧|奔跑吧]]》（原为《[[奔跑吧兄弟|奔跑吧兄弟]]》，后因「限韓令」而改名）、《[[中国好声音|中国好声音]]》、《[[我爱记歌词|我爱记歌词]]》（已停播）、《[[我是大评委|我是大评委]]》（已停播）、《[[爱唱才会赢|爱唱才会赢]]》（已停播）、《[[冲关我最棒|冲关我最棒]]》（已停播）、《[[爸爸回来了|爸爸回来了]]》（已停播）、《[[中国梦想秀|中国梦想秀]]》、《高能少年团》、《我就是演员》、《梦想的声音》、《二十四小时》、《王牌對王牌》等。现综艺节目播出时段繁多，其中周五、周六晚间时段播出的均为首播节目，周日至周四晚间时段播出的大部分为重播节目，有时会播出首播节目，此前曾推出“921节目带”，现已取消。在白天时段，浙江卫视也会重播综艺节目。

浙江卫视的综艺节目，大部分都是五个字（不过某些节目除外，例如《来吧冠军》）<ref>[http://epaper.nandu.com/epaper/C/html/2014-08/26/content_3302443.htm 为什么浙江卫视的综艺节目都是5个字？] {{webarchive|url=https://archive.is/20141126131041/http://epaper.nandu.com/epaper/C/html/2014-08/26/content_3302443.htm |date=2014-11-26 }}，南方都市报，2014年08月26日</ref>。为配合广电总局要求，从2016年起，浙江卫视开始增加原创综艺节目的比重。

===社教节目===
目前浙江卫视播出中的社教节目仅有一档，为党建类节目《时代先锋》。此前曾播出的社教节目包括《爱心浙江》、《母爱总动员》等。

===文化节目===
浙江卫视现无首播文化节目，但在深夜时段会安排重播。主要文化节目包括《华少爱读书》。

===纪录片和专题片===
浙江卫视会自制一系列纪录片和专题片，如《西湖》、《江南》系列、《南宋》等，也曾设立专门的播出时段《人文深呼吸》（现已取消）。现纪录（专题）片首播时间通常安排于晚间黄金时段，在深夜时段会安排重播。

此外，浙江卫视也会根据[[广电总局|广电总局]]的要求，播出“[[中国梦|中国梦]]”主题纪录片。

===大型活动===
曾在每年12月31日，浙江卫视会现场直播跨年晚会，但从2016-2017间年底开始，改为12月30日举行“领跑20XX”晚会现场直播；此外，每年10月举行中国（浙江）电视观众节期间，浙江卫视也会播出特别节目。

===王牌节目===

==== 《[[中国好声音|中国好声音]]》 ====
《中国好声音》，是浙江卫视在2012年7月开播的[[音乐|音乐]][[真人秀|真人秀]]节目，源于[[荷兰|荷兰]]节目《[[荷兰之声|荷兰之声]]》，由[[那英|那英]]、[[庾澄慶|庾澄庆]]、[[杨坤|杨坤]]、[[刘欢|刘欢]]、[[汪峰|汪峰]]、[[张惠妹|张惠妹]]、[[齐秦|齐秦]]、[[周杰倫|周杰倫]]担任评委以及导师。《[[The_Voice|The Voice]]》以「真声音、真音乐」为口号，因其独特创新的节目模式迅速风靡全球，选择最具实力和潜质的选手进行后续比拼，延续《The Voice》一贯风格，以正面、励志的态度去选拨最佳歌唱家，所到各国均赢得了最炙手可热的音乐巨星青睐和参与，并获得很高的收视纪录。因版权问题以原创的《[[中国新歌声|中国新歌声]]》取代 ,由[[那英|那英]]、[[庾澄慶|庾澄庆]]、[[汪峰|汪峰]]、[[周杰倫|周杰倫]] 、[[刘欢|刘欢]]、[[陳奕迅|陳奕迅]] 担任评委以及导师。该节目自2018年起改回《中国好声音》 ,由[[庾澄慶|庾澄庆]]、[[周杰倫|周杰倫]]、 [[謝霆鋒|謝霆鋒]]、[[李健|李健]]、[[王力宏|王力宏]]担任评委以及导师。

==== 《[[奔跑吧兄弟|奔跑吧兄弟]]》《[[奔跑吧|奔跑吧]]》 ====
《奔跑吧兄弟》是浙江衛視与[[韩國|韩國]][[SBS_(韩國)|SBS电視台]]联合制作的[[真人秀|真人秀]]节目，原版為SBS电視台的《[[Running_Man|Running Man]]》。由七位班底主持人主持，分別有：[[邓超|鄧超]]、[[李晨_(演员)|李晨]]、[[陳赫_(中國演員)|陳赫]]、[[郑恺|鄭愷]]、[[Angelababy|楊穎]](Angelababy)、[[王祖藍|王祖藍]]、[[鹿晗|鹿晗]]。原本分别参与第一、二、五季的[[王宝强|王寶強]]、[[包贝尔|包貝爾]]与[[迪丽热巴|迪麗熱巴]]与已退出。《奔跑吧兄弟》的游戏设定为成员与嘉宾先前往各个任务地点进行前期任务，由胜出的队伍或个人获得对最终任务有利之提示或权力，最后在最终任务地点完成最终任务。而游戏中最精彩的部分就属紧张刺激的“撕名牌”环节。

本節目因限韓令影響，正式變更為全新節目《[[奔跑吧|奔跑吧]]》（Keep Running）。

==境外播放==
===香港===
由于晚间时段“中国蓝剧场”以及一些周播综艺节目（《[[中国新歌声|中国新歌声]]》及領跑20XX演唱會除外）仅享有在中国大陆地区的首映权（通常在每天19:30-23:00之間），故在香港[[now宽频电视|now宽频电视]]、[[香港有线电视|香港有线电视]]平台会在《[[新闻联播|新闻联播]]》结束后一瞬间立马屏蔽，并使用动画提示覆盖<ref>[http://video.tudou.com/v/XMjk5MDQxNjg4MA==%20 香港now宽频电视555频道（浙江卫视）中国蓝剧场开始屏蔽过程]</ref>。而《[[中国新歌声|中国新歌声]]》由于nowTV购买了香港特区同步转播权，故在收费平台当中的[[now宽频电视|now宽频电视]]555频道、[[香港有线电视|香港有线电视]]35频道传送的浙江卫视的该时段综艺节目则未受影响，而且《'''領跑20XX演唱會'''》（全球直播）则会照常同時直播。

而在《[[中国新歌声|中国新歌声]]》第二季首播期间，香港免费数码广播电视频道[[ViuTVsix|ViuTVsix]]则会期间同步或延迟几分钟的转播（冠军赛改由[[ViuTV|ViuTV]]转播），期间原信号台标、右上角角标和底侧跑马部分均被ViuTV自家角标覆盖（在第三、四季《[[中国好声音|中国好声音]]》由[[now香港台|now香港台]]同步转播，亦是同样处理方式）。在首播期间，ViuTVsix会在右上角台标底侧标注“LIVE FEED”字样（意思是同步直播，实际上在浙江卫视的首播则是录播节目），而最后一场冠军赛的由于是现场直播（实际上和鸟巢现场延时直播90分钟不等），且香港改为ViuTV转播，故右上角台标底侧则标注“现场直播”字样。而所有期数的重播平台在次日24点ViuTV重播的，台标底侧则不标注任何字样。

===新马地区===
浙江卫视在新马地区与[[astro|astro]]和[[星和电信|星和电信]]达成战略合作，目前浙江卫视大多综艺节目都会通过[[astro全佳HD|astro全佳HD]]、[[HUB都会台|HUB都会台]]双平台同步直播和延后录播。另外《[[中国新歌声|中国新歌声]]》还会通过[[八度空间|八度空间]]同步延时播放。

另外，新馬地區也連同台灣、泰國、菲律賓、印尼、緬甸等10個亞洲地區將會由NBC Universal旗下的E!緊貼內地播放，並將會一切計劃中。

==轶闻==
===播出故障===
{{UTC+8}}
于2018年1月13日21时41分，浙江卫视在播出《好看中国蓝之演员十二强》过程中，突然卡顿画面5秒钟，随后切入了浙江风光片，之后突然安排重播上期的《演员的诞生》节目，引起了观众热议。之后，浙江卫视官方微博声明致歉，表示因播出设备故障导致临近结束前1分50秒无法正常收看，之后该节目于次日0时50分和8时55分重新播出<ref>[https://weibo.com/1288369910/FEemjloXF?filter=hot&root_comment_id=4195883425235071&type=comment#_rnd1515866231215 微博@浙江卫视中国蓝 对于今晚设备故障影响致歉声明]</ref>。而在此前浙江电视台广告部的通知当中，浙江卫视当天的晚间编排就是播出《好看中国蓝之演员十二强》和综艺复播（即上期的《演员的诞生》），并于1月20日晚播出《演员的诞生》总决赛。
