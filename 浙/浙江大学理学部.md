'''浙江大学理学部'''是[[浙江大学|浙江大学]]的一个学部，2009年6月成立，拥有[[浙江大学数学系|数学系]]、[[浙江大学物理学系|物理学系]]、[[浙江大学化学系|化学系]]、[[浙江大学地球科学系|地球科学系]]、[[浙江大学心理与行为科学系|心理与行为科学系]]5个系，并设有学术委员会、学位委员会、人力资源委员会、教学委员会4个专门委员会。<ref name="简介">{{cite web|url=http://www.css.zju.edu.cn/chinese/redir.php?catalog_id=17269 | language= | title=学部简介 |publisher=浙江大学理学部 |date= |accessdate=2012年12月 }}</ref>

== 专业设置 ==

学部有12个本科专业。2009年招收硕士研究生335人、博士研究生180人，毕业本科生466人、硕士研究生137人、博士生176人。

2010年在校本科生1473人，研究生1743人（其中博士生741人）。<ref name= "简介"/>

== 重点学科 ==
有数学、化学2个一级学科[[国家重点学科|国家重点学科]]；应用心理学、理论物理、凝聚态物理3个二级学科国家重点学科；计算数学、凝聚态物理、理论物理、光学、有机化学、物理化学、构造地质学7个省级重点学科。<ref name= "简介"/>

学部拥有数学、物理学、化学、地质学、心理学5个[[博士后|博士后]]流动站；数学、物理学、化学、地质学、心理学5个一级学科博士点；27个专业具有博士学位授予权（2个共建），36个硕士学位授予权（2个共建）。<ref name= "简介"/>

== 师资力量 ==
2009年底，学部有教职工574名，其中6名[[中国科学院院士|中国科学院院士]]；149名[[博士生导师|博士生导师]]，143名硕士生导师，171名正高职人员，183名副高职人员。<ref name= "简介"/>

== 研究中心 ==
学部拥有27个研究所、3个校内研究中心，主要如下：<ref name= "简介"/>

*4个国家理科人才培养基地：数学、物理、化学、心理学
*2个国家工科课程基地：化学、物理
*2个国家级教学团队：化学基础课程、数学建模方法与实践
*1个[[国家专业实验室|国家专业实验室]]：工业心理学
*1个[[国家重点实验室|国家重点实验室]]：计算机辅助设计与图形学（CAD/CG）（核心部分）
*2个[[浙江省|浙江省]]重点实验室：应用化学、资源与环境信息系统
*1个[[教育部|教育部]]研究中心：含油气盆地构造研究中心
*1个国家级实验示范中心：化学实验中心
*1个省级实验示范中心：物理实验中心
