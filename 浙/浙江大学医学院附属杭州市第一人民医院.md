'''杭州市第一人民医院'''位于[[杭州市|杭州市]]浣纱路261号，创建于1923年5月29日，1989年在[[浙江省|浙江省]]内首批通过三甲评审，是[[杭州市|杭州市]]属最大的综合性[[三级甲等医院|三级甲等医院]]，现为'''浙江大学医学院附属杭州市第一人民医院'''、'''浙江省医学科学院临床研究院'''。承担市肿瘤、产科、骨科、重症医学、急诊、放射、病理、检验、药事、护理、院内感染等质控中心的建设。<ref name="简介">{{cite web |url=http://www.hz-hospital.com/admin/main/overview/overview.vm |language= |title=杭州市第一人民医院简介 |publisher=杭州市第一人民医院 |date= |accessdate=2018年6月 |deadurl=yes |archiveurl=https://web.archive.org/web/20180612210759/http://www.hz-hospital.com/admin/main/overview/overview.vm |archivedate=2018-06-12 }}</ref>

== 概况 ==
医院现有职工3811人，其中中高级职称卫技人员过半，12人享受国务院和市政府特殊津贴，50名省市各级新世纪培养人才，6名杭州市钱江特聘专家，40名[[南京医科大学|南京医科大学]]教授、副教授，博导和硕导50人。<ref name="简介"/>

院区占地30余亩，核定3000张床位，开放2613张。现有50个一、二级临床医疗和医技科室，其中消化内科、心血管内科、肿瘤学、妇产科、临床诊断学、眼科、神经内科、肾内科、血液内科、临床检验学、耳鼻咽喉科、小儿内科、肿瘤放疗学为省市级医学重点学科。

既是浙江大学医学院附属医院，也是[[杭州师范大学|杭州师范大学]]、[[浙江中医药大学|浙江中医药大学]]、[[温州医学院|温州医学院]]的教学医院，每年承担600余名医学生的临床实习教学工作，设有南京医科大学、浙江中医药大学[[博士|博士]]教学点，教学体系较为完整。<ref name="简介"/>

== 挂靠单位 ==
*杭州市癌症中心
*杭州市心脑血管病诊治中心
*杭州市危重孕产妇抢救中心
*杭州市健康管理中心

== 城北分院 ==
市一医院城北分院项目（市老年病医院），位于[[拱墅区|拱墅区]]沈半路469号，建筑面积97274平米，床位800张。建设工期为2015年至2017年，总投资6.9亿元。

== 杭州国医馆 ==
2016年5月18日，位于学士路3号的杭州国医馆正式开馆，作为杭州市第一人民医院集团旗下的国医馆，它整合了集团6家医院的[[中医|中医]]资源，拥有9间中医诊室和10间[[针灸|针灸]]诊室。<ref >{{cite web|url=http://news.163.com/16/0517/00/BN7RK75000014Q4P.html | language= | title=杭州国医馆明天正式开业 |publisher=青年时报 |date=2016-05-17 |accessdate= }}</ref>
