'''浙江医院'''位于[[杭州市|杭州市]][[灵隐路|灵隐路]]12号，毗邻[[西湖|西湖]]和[[杭州植物园|杭州植物园]]，隶属于[[浙江省卫生厅|浙江省卫生厅]]。始建于1954年，建院初期以[[干部|干部]]医疗保健和涉外医疗为主，1979年全面对外开放，目前是一所集医疗、教学、科研、预防和保健于一体的[[三级甲等|三级甲等]]综合性医院，是杭州市西部急救中心。<ref name="简介">{{cite web |url=http://zjhospital.com.cn/sl/channels/3.html |language= |title=医院介绍 |publisher=浙江医院 |date= |accessdate=2013年1月 |deadurl=yes |archiveurl=https://web.archive.org/web/20130115052343/http://www.zjhospital.com.cn/sl/channels/3.html |archivedate=2013-01-15 }}</ref>

== 概况 ==
医院占地面积6.6万平米，医疗用房面积5.5万平米，开放800张床位，设临床专科35个，病区(包括ICU、CCU)18个，临床诊疗中心8个，医技科室11个。现有职工1000余人，其中183名高级职称，263名中级职称，3人享受国务院特殊津贴。卫技人员占全院职工的80％以上。<ref name="简介"/>
