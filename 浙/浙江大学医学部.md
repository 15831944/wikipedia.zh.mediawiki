'''浙江大学医学部'''是隶属于[[浙江大学|浙江大学]]的医学院校。低年级临床专业本科生在[[紫金港校区|紫金港校区]]就读，高年级临床本科生搬迁往[[华家池校区|华家池校区]]，基础医学系研究生在[[紫金港校区|紫金港校区]]就读，转化医学院研究生在[[华家池校区|华家池校区]]。

==历史==
前身[[浙江医科大学|浙江医科大学]]是[[中国|中国]]最早的医学院校之一，1952年由[[老浙江大学|老浙江大学]]所属的医学院、'''浙江医学专门学校'''合并成立。1960年[[浙江医学院|浙江医学院]]更名为浙江医科大学。

1998年，[[浙江大学|浙江大学]]、[[浙江医科大学|浙江医科大学]]、[[杭州大学|杭州大学]]、浙江农业大学合并组建为新的[[浙江大学|浙江大学]]，浙江医科大学校本部更名为'''浙江大学西校区'''（即[[浙江大学湖滨校区|浙江大学湖滨校区]]），2009年4月29日再次更名为'''浙江大学医学部'''（包含医学院和药学院）。

现任医学部主任是[[中国科学院院士|中国科学院院士]]、[[神经生物学|神经生物学]][[博士|博士]][[段树民|段树民]]。

2007年，浙大医学院成为[[新加坡|新加坡]]卫生部门承认的八所中国大陆医学院之一。<ref>{{Cite web |url=http://www.mdweekly.com.cn/doc/2007/12/5162.shtml |title=新加坡认可中国大陆医学院增至8所 |access-date=2013-03-06 |archive-url=https://archive.is/20130423010836/http://www.mdweekly.com.cn/doc/2007/12/5162.shtml |archive-date=2013-04-23 |dead-url=yes }}</ref>

== 院系设置 ==
[[File:Medical_School_of_ZJU.jpg|缩略图]]
医学部设有[[浙江大学基础医学院|基础医学院]]、[[浙江大学公共卫生学院|公共卫生学院]]、临床医学一系、临床医学二系、临床医学三系、口腔医学院、护理系和[[浙江大学药学院|药学院]]共8个院系，拥有六家附属医院。有教职员工、医护人员1万余人，其中[[中国科学院院士|中国科学院院士]]1人，[[中国工程院院士|中国工程院院士]]3人，[[博士生导师|博士生导师]]212人，硕士生导师663人。现在读4000余医学生，其中800余博士生。<ref name="介绍">{{cite web |url=http://www.cmm.zju.edu.cn/chinese/column.php?id=9 |language= |title=学部介绍 |publisher=浙江大学医学部 |date=2012年11月 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20121012181805/http://www.cmm.zju.edu.cn/chinese/column.php?id=9 |archivedate=2012-10-12 }}</ref>

学部有八年制[[医学博士|医学博士]]、七年制[[医学硕士|医学硕士]]、四年制[[药学|药学]]本科、五年制临床医学本科（国际留学生）等专业，拥有基础医学、临床医学、口腔医学、药学4个[[一级学科|一级学科]]博士点，5个[[博士后|博士后]]流动站。<ref name= "介绍"/>

有4个国家重点学科、1个[[传染病诊治国家重点实验室|传染病诊治国家重点实验室]]和15个省部级重点实验室。获9项[[国家科技进步奖|国家科技进步奖]]和科技发明奖。<ref name= "介绍"/>
