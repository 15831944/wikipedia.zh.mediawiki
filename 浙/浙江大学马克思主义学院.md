'''浙江大学马克思主义学院'''是隶属于[[浙江大学社会科学学部|浙江大学社会科学学部]]的一个教学部门，2015年4月在原浙江大学思想政治教学科研部的基础上成立，为[[浙江大学|浙江大学]]的直属单位。学院的办公地点位于[[浙江大学西溪校区|浙江大学西溪校区]]教学主楼第十层<ref name="简介">{{cite web |url=http://sxzzll.zju.edu.cn/external/column.php?id=17 |language= |title=学院简介 |publisher=浙江大学马克思主义学院 |date= |accessdate=2012年12月 |deadurl=yes |archiveurl=https://web.archive.org/web/20130417084705/http://sxzzll.zju.edu.cn/external/column.php?id=17 |archivedate=2013-04-17 }}</ref>。

== 概况 ==

教研部现有49名教职工，其中44名从事思想政治理论教学、研究和马克思主义理论学科建设的教师，5名行政人员。教师中有6名教授、22名副教授。现有在读生包括32名博士生、160名硕士生。<ref name= "简介"/>

== 重点学科 ==
*一级学科博士点：马克思主义理论
*博士后流动站：马克思主义理论
*6个硕士点：拥有马克思主义基本原理、马克思主义中国化研究、思想政治教育、国际政治、中国近现代史基本问题研究、国际共产主义运动与科学社会主义。<ref name= "简介"/>
