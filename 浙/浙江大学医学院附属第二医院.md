'''浙江大学医学院附属第二医院'''，简称'''浙医二院'''，又名'''浙江省第二医院'''（'''浙二医院'''），是附属于[[浙江大学医学部|浙江大学医学部]]的一所[[三级甲等医院|三级甲等医院]]。位于[[杭州市|杭州市]][[解放路|解放路]]88号。<ref name="简介">{{cite web|url=http://www.z2hospital.com/cms/JS.aspx?NRID=10035&LMID=10 | language= | title=医院简介 |publisher=浙医二院 |date= |accessdate=2012年11月 }}</ref>

== 简介 ==
医院前身为[[英国圣公会|英国圣公会]]于1869年在华设立的'''杭州广济医院'''，1885年到1925年医院附设的'''广济医校'''是中国最早的[[医校|医校]]之一。<ref name="简介" />

医院目前医疗用房16.3万平米，核定床位2000张，有现代化的眼科中心、急诊中心、国际保健中心和门诊科教综合大楼，以及脑科中心（附设急救[[直升机|直升机]]停机坪）。每年门急诊量270余万人次，总手术量近8万台。<ref name="简介" />

现任院长是[[浙江大学医学部|浙江大学医学部]]副主任、心脏中心主任、心血管内科主任、心血管介入中心主任[[王建安_(医学)|王建安]]<ref name=ld>{{cite web|url=http://www.z2hospital.com/cms/LD.aspx?LMID=11 | language= | title=领导班子 |publisher=浙医二院}}</ref>。

== 实力 ==
医院有职工2625人，高级职称专家447人，其中教育部[[长江特聘教授|长江特聘教授]]4人、[[卫生部|卫生部]]有突出贡献中青年专家4人、国家杰出青年3人、获国务院特殊津贴专家18名，[[博士生导师|博士生导师]]61人，有[[临床医学|临床医学]][[一级学科|一级学科]][[博士|博士]]学位授予点和[[博士后|博士后]]流动站。<ref name="简介" />

医院学科齐全，拥有47个临床和医技科室，最著名的有肿瘤、眼科、心血管内科、骨科、神经内外科、急诊医学、普外、呼吸、口腔、耳鼻咽喉科和胸心外科。其次是已获批卫生部临床重点专科的骨科、临床护理、心血管内科、神经外科、眼科、普通外科、烧伤科、急诊医学科、皮肤科和呼吸内科。<ref name= "简介"/>

== 医学技术指导中心 ==

医院重视医疗质量管理，院内附设[[浙江省|浙江省]]多个医学技术指导中心<ref name= "简介"/>：
*浙江省急诊医学质量控制中心
*浙江省临床麻醉质量控制中心
*浙江省医院感染管理质量控制中心
*浙江省病历管理质量控制中心
*浙江省高压氧医疗量控制中心
*浙江省心血管疾病介入诊疗质量控制中心
*浙江省烧伤救治技术指导中心
*浙江省临床营养指导中心
*浙江省骨科技术指导中心
*浙江省超声技术指导中心
*浙江省神经外科技术指导中心 
*浙江省药物滥用监测中心

== 滨江院区 ==
浙二医院滨江院区简称[[滨江医院|滨江医院]]，是2008年浙医二院和杭州市政府、[[滨江区|滨江区]]政府共同出资建立的一所非营利性、国有控股的股份制医院，2013年3月投入使用。浙二医院与滨江院区有B支3路区间公交车直达连接，于2013年12月25日开通。<ref >{{cite web|url=http://zzhz.zjol.com.cn/system/2013/12/24/019774503.shtml | language= | title=公交集团增开B支3路区间 |publisher=住在杭州网 |date=2013年12月24日 |accessdate= }}</ref>

== 荣誉 ==

*1991年，获全国卫生系统先进集体；
*1998年，获全国百佳医院、浙江省示范文明单位；
