'''浙江大学管理学院'''是[[浙江大学|浙江大学]]的一所[[学院|学院]]，属于[[浙江大学社会科学学部|社会科学学部]]。设管理科学与工程、企业管理、会计与财务管理、旅游管理、农业经济与管理5个系。学院位于[[浙江大学紫金港校区|浙江大学紫金港校区]]智能化管理大楼。<ref name="简介">{{cite web |url=http://www.som.zju.edu.cn/cn/about/jianjie.html |language= |title=学院简介 |publisher=浙江大学管理学院 |date= |accessdate=2012年12月 |deadurl=yes |archiveurl=https://web.archive.org/web/20121229234607/http://www.som.zju.edu.cn/cn/about/jianjie.html |archivedate=2012-12-29 }}</ref>

== 概况 ==
目前在校生计3900人，其中280名博士生、540名硕士生、1200名[[MBA|MBA]]专业学位学生、740名[[EMBA|EMBA]]专业学位学生、460名工程硕士学位学生、680名本科生。<ref name= "简介"/>

学院现有教师120余人，其中37名教授、26名[[博士生导师|博士生导师]]、52名副教授，有1名[[中国工程院|中国工程院]]院士，有40名在站[[博士后|博士后]]。<ref name= "简介"/>

学院于1987年创办的《[[管理工程学报|管理工程学报]]》是全国性重要学术期刊。学院已通过“[[211工程|211工程]]”建设和“[[985工程|985工程]]”二期建设。

== 重点学科 ==

*3个[[浙江省|浙江省]]重点学科
*3个[[博士后|博士后]]流动站
*4个博士点
*8个硕士点，工商管理、会计、工程、旅游管理、农业推广5个专业学位点
*9个本科专业

== 研究机构 ==
学院有以下9个校级研究中心与12个研究所：
*农业现代化与农业发展研究中心
*技术创新与科技产业发展研究中心
*人力资源与战略发展研究中心
*创新与发展研究中心
*房地产研究中心
*浙江大学－杭州市服务业发展研究中心
*资本市场与会计研究中心
*企业成长研究中心
*全球创业研究中心

== 国际化 ==
*浙江大学管理学院于2006年在中国大陆率先获得AMBA国际认证；2011年底获得AMBA五年期再认证；2015年3月获得EQUIS五年期再认证；2015年5月9日浙江大学管理学院顺利通过AACSB国际认证。至此，浙江大学管理学院全部通过全球商学院最重要的三大国际认证。
