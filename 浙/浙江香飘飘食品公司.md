'''浙江香飘飘食品有限公司'''是一家位于[[浙江省|浙江省]][[湖州市|湖州市]]的食品生产企业，主要从事[[奶茶|奶茶]]等饮品的生产和销售，比较为人所為人熟知的是其生产的「[[香飘飘奶茶|香飘飘奶茶]]」。该公司最初创办于2005年8月，总投资为2.5亿[[人民币|人民币]]，现拥有职工2000多人。<ref>[http://hzxpp.58food.com/introduce/ 公司简介]</ref>

==发展历程==
*2005年，浙江香飘飘食品有限公司成立，开始了杯装奶的生产与销售。
*2007年11月，成为[[慈溪|慈溪]]首家生产方便食品的企业<ref>[http://www.zj.xinhuanet.com/zjgov/2007-11/28/content_11788511.htm 香飘飘成为慈溪首家生产方便食品企业] {{webarchive|url=https://web.archive.org/web/20140202220410/http://www.zj.xinhuanet.com/zjgov/2007-11/28/content_11788511.htm |date=2014-02-02 }}</ref>
*2008年，该公司持有的“香飘飘”商标被认定为“[[中国驰名商标|中国驰名商标]]”。
*2009年4月，投资3.5亿组建了子公司香飘飘四川食品有限公司，该子公司占地面积约86余亩，厂房面积约3.2万平方米。<ref>[http://eladies.sina.com.cn/nx/2010/0210/0936969509.shtml 奶茶香飘飘：哪里需要我 我就去哪里]</ref>
*2011年，投资兴建了香飘飘天津食品工业有限公司。
*2012年，香飘飘食品有限公司的商标再次被认定为“中国驰名商标”<ref>[http://biz.zjol.com.cn/05biz/system/2012/05/03/018462820.shtml 浙江又有52件商标被认定为中国驰名商标]</ref>。

==食品安全事件==
2008年，一家食品公司的員工爆料，其公司利用香蕉水清除香飘飘奶茶上的生產日期，替换新的生产日期继续销售，但是公司负责人则表示，确有抹掉生产日期这回事，不过抹掉生产日期的产品没有对外销售<ref>[http://news.sina.com.cn/s/2008-11-01/025414661872s.shtml 偷偷抹掉香飘飘奶茶生产日期]</ref>。2010年，又有消費者称，發現其生產的奶茶中出現活昆蟲、毛髮等問題，而公司公關則辯稱產品安全沒有問題，產品中出現昆蟲很正常。<ref>[http://food.china.com.cn/2013-01/15/content_27691310.htm 香飘飘奶茶频现异物 厂家屡屡推卸责任]</ref>2013年，北京市食品安全辦通報了一系列病菌超標的產品名單，其中就包括香飘飘牌的花生類產品，并导致这些产品被迫下架，但是有媒體报道，在中国大陸的[[淘寶網|淘寶網]]上仍然有商家正在銷售这类产品，而香飘飘食品公司则表示与涉案公司毫无关系，亦没有授权或委托该公司生产销售任何产品。<ref>[http://finance.china.com.cn/consume/puguang/20130609/1543019.shtml 香飘飘花生检出致病菌 超市下架淘宝仍热卖]</ref>

==声称获得蘭芳園商标使用权==
*2016年6月，香飘飘和[[蘭芳園|蘭芳園]]签署协议，获得蘭芳園商标在内地的使用权。<ref>{{cite web |url=http://finance.ce.cn/stock/gsgdbd/201808/28/t20180828_30137169.shtml|title=转型液态奶茶遇阻 香飘飘巨量产能如何消化 |website=中国经济网 |publisher=[[时代周报|时代周报]] |access-date=2018-08-30 |date=2018-08-29}}</ref><ref>{{cite web |url=http://www.eeo.com.cn/2017/1202/318122.shtml|title=香飘飘的资本圈：掌门人拆解奶茶历史|publisher=[[经济观察报|经济观察报]] |access-date=2018-08-30 |date=2017-12-02}}</ref>但蘭芳園二代老闆林俊業在接受采访时表示，蘭芳園現時只在香港開設店鋪，除香港以外並無其他分店。同時，他們也親口向客人解釋，或張貼告示提醒，「我們根本在國內是沒有經營茶餐廳，希望人們知道，不要被騙。」<ref>{{cite web |url=https://nextplus.nextmedia.com/news/latest/20180524/588976|title=【山寨蘭芳園遍佈】大陸人無恥認抄襲 老闆無奈：我冇錢同佢打官司|publisher=[[壹週刊_(香港)|壹週刊]]}}</ref><ref>{{cite web |url=https://thestandnews.com/society/%E5%A4%A7%E9%99%B8%E5%86%8D%E7%8F%BE-%E5%B1%B1%E5%AF%A8%E5%BA%97-%E6%B8%AF%E8%98%AD%E8%8A%B3%E5%9C%92-%E5%BE%9E%E6%B2%92%E6%8E%88%E6%AC%8A%E5%85%A7%E5%9C%B0%E9%96%8B%E5%88%86%E5%BA%97/|title=大陸再現「山寨店」　港蘭芳園：從沒授權內地開分店|publisher=[[立場新聞|立場新聞]]}}</ref>
