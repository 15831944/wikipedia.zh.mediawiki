'''浙江大学农业生命环境学部'''是[[浙江大学|浙江大学]]的一个学部，包括[[浙江大学生命科学学院|生命科学学院]]、[[浙江大学生物系统工程与食品科学学院|生物系统工程和食品科学学院]]、[[浙江大学环境与资源学院|环境与资源学院]]、[[浙江大学农业与生物技术学院|农业与生物技术学院]]、[[浙江大学动物科学学院|动物科学学院]]五个学院，以及[[浙江大学农业试验站|农业试验站]]和[[浙江大学农业技术推广中心|农业技术推广中心]]两个直属单位。现任学部主任为[[张国平|张国平]]。<ref name="简介">{{cite web |url=http://www.fale.zju.edu.cn/chinese/redir.php?catalog_id=184 |language= |title=浙江大学农业生命环境学部简介 |publisher=浙江大学农业生命环境学部 |date= |accessdate=2012年12月 |deadurl=yes |archiveurl=https://web.archive.org/web/20100927074802/http://www.fale.zju.edu.cn/chinese/redir.php?catalog_id=184 |archivedate=2010-09-27 }}</ref>

== 重点学科 ==
学部有11个一级学科，其中农业资源利用、园艺学和植物保护学为[[国家重点学科|国家重点学科]]；49个二级学科，其中国家重点学科有植物学、生态学、农业机械化工程、环境工程、作物遗传育种、生物物理学和特种经济动物饲养；有一级学科博士学位授予点9个，二级学科博士学位授予点45个，硕士学位授予点50个，以及9个[[博士后流动站|博士后流动站]]。<ref name= "简介"/>

== 师资力量 ==
学部现有1名[[院士|院士]]，9名国家“千人计划”入选者，5名浙江省特级专家，12名“长江学者奖励计划”特聘教授，3名“973”首席科学家，3名国家级教育名师，20名国家杰出青年基金获得者，14人入选“国家百千万人才工程”，34人入选教育部“跨（新）世纪优秀人才计划”。<ref name= "简介"/>

== 研究中心 ==
主要进行创新研究的机构如下：
*2个[[国家重点实验室|国家重点实验室]]（联合）
*1个[[国家工程实验室|国家工程实验室]]
*3个[[教育部|教育部]]重点实验室
*7个[[农业部|农业部]]重点实验室
*7个[[浙江省|浙江省]]重点实验室
*35个研究所<ref name= "简介"/>
