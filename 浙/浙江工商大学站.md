'''浙江工商大学站'''(工程名为SG8-2标文桥区间风井)是建设中[[杭州地铁8号线|杭州地铁8号线]]的站点，位于[[杭州市|杭州市]][[江干区|江干区]]学府街浙江工商大学正门口。有3个出入口。

== 结构 ==
浙江工商大学站沿学府街东西向布置，为地下三层侧式站台，结构长度181.3米、宽32.4米，深24.72米。有效站台宽度9~11米，长140米，设2组风亭、3个地面出入口，1个消防疏散口。东侧以11.3大口径盾构隧道通过[[钱塘江|钱塘江]]<ref>{{cite web |title=【地铁来了】实探地铁8号线下沙段开工站点！地铁7号线义蓬与靖江之间新增靖义风井？！ - 大江东杂谈 - |url=http://www.hzdajiangdong.com/thread-30467-1-2.html |website=大江东网}}</ref>

==历史==
*2018年4月26日，车站开工。<ref>{{cite web |title=关于地铁8号线文桥风井主体工程施工期间调整交通组织措施的通告 |url=http://www.hzpolice.gov.cn/Html/201804/19/93e9df4b-fa31-4c2a-991f-ca3bea3de343.html |website=杭州公安局}}</ref>
*2018年8月17日，74幅车站地下连续墙工程完工，是8号线进度最快的车站之一。<ref>{{cite web |title=杭州地铁8号线2标文桥区间风井地下连续墙顺利完工 |url=http://www.cr16gd.cn/art/2018/8/22/art_21637_2575870.html |website=中铁十六局集团北京轨道交通工程建设有限公司 公司要闻}}</ref>
