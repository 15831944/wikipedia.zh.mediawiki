'''-{zh-hans:约翰·沙希迪;}-'''（{{lang-en|'''John Shahidi'''}})是一名美国企业家和软件工程师。<ref name = techcrunch/> 他是 [[Shots_Studios|Shots Studios]] 的联合创始人和[[首席执行官|首席执行官]]。<ref name = techcrunch>{{cite web | url=http://techcrunch.com/2013/11/12/justin-bieber-shots-of-me/ | title=Justin Bieber-Backed “Shots Of Me” Launches Selfie Sharing App | publisher=TechCrunch | date=12 November 2013 | accessdate=26 August 2014 | author=Constine, John}}</ref>

== 企业生涯 ==
2010年1月，约翰·沙希迪和他的弟弟山姆建立了一间移动游戏公司名为[[RockLive|RockLive]]。在RockLive建立之前，沙希迪任职为[[弗洛伊德·梅威瑟|弗洛伊德·梅威瑟]]的品牌经理。<ref name = fortune>{{cite web | url=http://fortune.com/2014/06/17/floyd-mayweather-tech/ | title=Money Mayweather's $1 million tech punch | publisher=Fortune | date=17 June 2014 | accessdate=26 August 2014 | author=Roberts, Daniel}}</ref> 梅威瑟后来成为沙希迪移动应用的投资者。<ref name = fortune/> 该公司发行了[[Chad_Johnson|Chad Johnson]], [[迈克·泰森|迈克·泰森]], [[克里斯蒂亚诺·罗纳尔多|克里斯蒂亚诺·罗纳尔多]]和[[尤塞恩·博尔特|尤塞恩·博尔特]]的相关游戏。<ref>{{cite web | url=http://pando.com/2012/07/17/rocklive-and-olympian-usain-bolt-release-addictive-mobile-game-with-the-help-of-celebrity-friends/ | title=RockLive and Olympian Usain Bolt Release Addictive Mobile Game With the Help of Celebrity Friends | publisher=Pando Daily | date=17 July 2012 | accessdate=2 September 2014 | author=Carney, Michael}}</ref> 沙希迪担任公司的主席和[[首席执行官|首席执行官]]。2013年, RockLive更名为Shots.

2013年5月约翰·沙希迪和他的弟弟和[[贾斯汀·比伯|贾斯汀·比伯]]合作开发了一个自拍社交移动应用程序名为[[Shots:_拍照社交应用|Shots]]。<ref>{{cite web | url=http://upstart.bizjournals.com/entrepreneurs/hot-shots/2014/04/11/john-shahidi-teams-with-justin-bieber.html?page=all | title=Friends in high places (including Justin Bieber), this founder keeps it low key | publisher=Upstart Business Journal | date=11 April 2014 | accessdate=2 September 2014 | author=Dalenberg, Alex}}</ref>该应用的目的是要为用户提供一个安全和没有网络欺凌的环境。<ref name = mashable>{{cite web | url = http://mashable.com/2014/04/12/shots-selfie-app/ | title=A-List Investors, 1 Million Downloads: Is Selfie App 'Shots' For Real? | publisher=Mashable | date=12 April 2014 | accessdate=2 September 2014 | author=Wagner, Kurt}}</ref><ref name = techly>{{cite web | url=http://www.techly.com.au/2014/08/20/theres-social-network-selfies-beiber-investor/ | title=There's a Social Network for Selfies, and Bieber is an Investor | publisher=Techly | date=20 August 2014 | accessdate=2 September 2014 | author=Speight, Janek}}</ref> 沙希迪有反对网络欺凌的想法是因为他曾经是被欺凌的受害者。<ref name = mashable/>该应用现在可以在[[安卓|安卓]]和[[iOS|iOS]]下载。<ref name = techcrunch/><ref name = techly/>
