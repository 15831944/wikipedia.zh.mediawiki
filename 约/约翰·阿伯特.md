'''-{zh-hans:约翰·约瑟夫·考德威尔·阿伯特;zh-hk:莊·祖瑟‧高德威·雅博;zh-tw:約翰·約瑟夫·考德威爾·阿伯特;}-爵士''' <small>[[加拿大女皇陛下枢密院|PC]]，[[御用大律师|QC]]，[[圣米迦勒及圣乔治勋章|KCMG]]，[[民法学学士|BCL]]，[[民法学博士|DCL]]</small>（'''{{lang|en|Sir John Joseph Caldwell Abbott}}'''，{{bd|1821年|3月12日|1893年|10月30日|catIdx=A}}）曾任[[加拿大总理|加拿大总理]]，亦是首位在[[加拿大|加拿大]]本土出生的总理。

== 早期生涯 ==
阿伯特於1854年取得[[麦吉尔大学|麦吉尔大学]]的[[民法学|民法学]]学位，其後成为一名出色的律师。他亦是[[蒙特利尔|蒙特利尔]]英语社区的重要支柱，更於1887年至1889年其间出任该市市长。

阿伯特曾经长时间担任[[加拿大国会下议院|国会下议院]]议员。1887年，他获委任为[[加拿大国会上议院|国会上议院]]议员，并加入当时总理[[约翰·亚历山大·麦克唐纳|麦克唐纳]]政府的内阁，出任[[不管部|不管部]]部长。他另於1887年5月12日至1893年10月30日其间出任国会上议院政府领袖一职。凭着其出色的法律及行政才能，阿伯特迅速成为政府的领导人物之一。

== 出任总理 ==
1891年，总理麦克唐纳於任内病逝。面临分裂的[[加拿大保守党|保守党]]希望雅博能够接掌总理一职，雅博只好在不情愿的情况下接受这个请求。在他十八个月的任期内，尽管爆出丑闻，雅博仍然致力复兴政府和保守党，并处理上任政府遗留下来的工作。雅博其中几项政绩包括：改革[[公务员|公务员]]制度、修订刑事法则、跟[[美国|美国]]签订互惠条约。另外，在他任内一共有五十二次补选，其中四十二次均由保守党胜出，令到保守党以十三席之差继续成为多数党。这些胜利无可置疑是加拿大国民对阿伯特作为加拿大总理一职的肯定。

== 最后日子 ==
阿伯特在出任总理一年後尝试将总理一职交予信奉[[罗马天主教|罗马天主教]]的[[约翰·斯帕洛·戴维·汤普森|约翰·斯帕洛·戴维·汤普森]]，但是因为受到有宗教偏见的保守党核心小组反对而未能成事。1892年，罹患初期[[脑癌|脑癌]]的阿伯特因为健康逐渐衰退，被迫退出政坛，而约翰·汤普森最终成为新一任加拿大总理。阿伯特在退休不足一年後病逝，享年七十二岁。他被安葬於魁北克省蒙特利尔皇家山公墓。

== 琐事 ==
* 阿伯特最常作出的政治评论是：「我讨厌政治（I hate politics）」。
* 位于魁北克省[[聖安‧迪‧美景|聖安‧迪‧美景]]（[[:en:Sainte-Anne-de-Bellevue,_Quebec|Sainte-Anne-de-Bellevue]]）的-{zh-hans:约翰·阿伯特;zh-hk:莊‧雅博;zh-tw:約翰·阿伯特;}-学院是以他的名字来命名的。雅博曾经在当地拥有一间渡假屋。
