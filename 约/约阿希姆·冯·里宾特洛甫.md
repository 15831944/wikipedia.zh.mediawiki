'''乌尔里希·弗里德里希·威廉·约阿希姆·冯·里宾特洛甫'''（[[德語|德語]]：'''Ulrich Friedrich Wilhelm Joachim von Ribbentrop'''，{{bd|1893年|4月30日|1946年|10月16日|}}）。[[纳粹德国|德國]]外交部长。[[德國二戰投降|德國二戰投降]]後被[[英军|英军]]拘捕。1946年10月1日，[[纽伦堡审判|纽伦堡国际军事法庭]]判处[[絞刑|絞刑]]，15天后受刑而死。

==生平==
===早年===
里賓特洛甫出生於[[德意志帝國|德意志帝國]][[莱茵兰|萊茵州]]的[[韋塞爾|韋塞爾]]，父親理夏德·乌尔里希·弗里德里希·約阿希姆·馮·里賓特洛甫（Richard Ulrich Friedrich Joachim Ribbentrop）是一名軍官，而母親則是约翰娜·索菲·赫特維希（Johanne Sophie Hertwig）<ref name="bloch 1-2">Bloch, ''Ribbentrop''，第1頁至第2頁</ref>。里賓特洛甫小時候一直接受非正規的教育，直到他十多歲後才到[[德國|德國]]與[[瑞士|瑞士]]的[[私立學校|私立學校]]唸書<ref name="bloch 3-4">Bloch, ''Ribbentrop''，第3頁至第4頁</ref>。里賓特洛甫的父親於1908年從軍中撤職，原因是他曾發表德皇[[威廉二世_(德国)|威廉二世]]是[[同性戀|同性戀]]的詆毀言論，里賓特洛甫家中也因此缺乏了經濟來源。里賓特洛甫藉由一口流利的[[英語|英語]]和[[法語|法語]]，先後在[[法國|法國]][[格勒諾布爾|格勒諾布爾]]和[[英國|英國]][[倫敦|倫敦]]住過一段時間，接著在1910年前往[[加拿大|加拿大]]<ref name="bloch 6">Bloch, ''Ribbentrop''，第6頁</ref>，他先在[[蒙特婁|蒙特婁]]史丹利街（Stanley Street）的莫爾森銀行（Molson）工作，接著換到重建[[魁北克橋|魁北克橋]]的一家工程公司，之後又換到曾修路連接[[蒙克顿|蒙克顿]]和[[温尼伯|温尼伯]]的加拿大[[國家跨洲鐵路公司|國家跨洲鐵路公司]]（National Transcontinental Railway），還曾在[[紐約|紐約]]與[[波士頓|波士頓]]當過記者。之後里賓特洛甫雖然曾因[[結核|結核]]而返德休養，但後又回到加拿大，於[[渥太華|渥太華]]經營一個小公司，進口德國[[酒|酒]]與[[香檳|香檳]]來銷售<ref name="bloch 7">Bloch, ''Ribbentrop''，第7頁</ref>。1914年2月，里賓特洛甫參加舉辦於波士頓的埃利斯紀念盃比賽，與渥太華著名的[[明托滑冰俱乐部|明托]]（Minto）溜冰隊競爭<ref name="bloch 7"/><ref>Lawson, Robert, "Ribbentrop in Canada 1910 to 1914: A Note", ''International History Review'' 第29期，發行於：2007年12月4日，第821頁至第832頁</ref>。因為[[第一次世界大戰|第一次世界大戰]]的爆發，里賓特洛甫離開加拿大返回德國參戰<ref name="bloch 8">Bloch, ''Ribbentrop''，第8頁</ref>，留下了他罹患[[肺結核|肺結核]]的弟弟歐塔·里賓特洛甫於加拿大，之後遭[[英國|英國]]拘捕（1918年去世）。1914年8月15日經由[[霍博肯_(新泽西州)|霍博肯]]開往[[鹿特丹|鹿特丹]]的荷蘭-美國船波茨坦號<ref name="bloch 8"/>回到歐洲，到了1914年秋天，他終於回到祖國，並加入第125輕騎兵團<ref> ''Current Biography 1941''，第707頁至第709頁</ref>。里賓特洛甫後晉升為中尉，並獲得[[鐵十字勳章|一級鐵十字勳章]]。起初他服役於[[東線|東線]]，後到[[西線|西線]]作戰。1918年，里賓特洛甫以參謀身分派駐[[奧斯曼帝國|奧斯曼帝國]]的[[君士坦丁堡|君士坦丁堡]]<ref name="bloch 8-9">Bloch, ''Ribbentrop''，第8頁至第9頁</ref>。里賓特洛甫於一戰外駐[[土耳其|土耳其]]期間，與另一位軍官[[弗朗茨·馮·帕彭|弗朗茨·馮·帕彭]]成為好友<ref name="bloch 9">Bloch, ''Ribbentrop''，第9頁</ref>與認識了[[汉斯·冯·塞克特|汉斯·冯·塞克特]]將軍。德國戰敗後，作為塞克特將軍的副官參加[[巴黎和会_(1919年)|巴黎和会]]，對於當時德國簽下[[凡尔赛条约|凡尔赛条约]]的屈辱場景，印象非常深刻。

===家庭===
1919年，里賓特洛甫與安娜·伊麗莎白·亨克爾（Anna Elisabeth Henkell）相識，朋友都稱她為安内利斯（Annelies）<ref name="bloch 12">Bloch, ''Ribbentrop''，第12頁</ref>。安娜是德國香檳製造廠老闆奧托·亨克爾（Otto Henkell）的千金，里賓特洛甫於1920年7月5日與她在[[威斯巴登|威斯巴登]]結婚，並開始在歐洲銷售他們酒廠的產品<ref name="bloch 12-13">Bloch, ''Ribbentrop''，第12頁至第13頁</ref>。1921年至1940年期間，安娜共生下了5個孩子：
*[[魯道夫·馮·里賓特洛甫|魯道夫]]（Rudolf von Ribbentrop，生於1921年5月11日的[[威斯巴登|威斯巴登]]），1960年與伊爾莎（Ilse-Marie Freiin von Münchhausen，1914-2010）結婚。
*[[贝蒂娜·馮·里賓特洛甫|贝蒂娜]]（Bettina von Ribbentrop，生於1922年7月20日的[[柏林|柏林]]）
*[[烏蘇拉·馮·里賓特洛甫|烏蘇拉]]（Ursula von Ribbentrop，生於1932年12月29日的柏林）
*[[阿道夫·馮·里賓特洛甫|阿道夫]]（Adolf von Ribbentrop，生於1935年9月2日的柏林），育有：
**約阿希姆（Joachim von Ribbentrop，生於1963年7月5日）
**多米尼克（Dominik von Ribbentrop，生於1965年9月25日）
**魯道夫（Rudolf von Ribbentrop，生於1989年6月5日的[[美因河畔法兰克福|美因河畔法兰克福]]）
**弗里德里希（Friedrich von Ribbentrop ，生於1990年6月28日的[[美因河畔法兰克福|美因河畔法兰克福]]）
*[[巴托尔德·亨克爾·馮·里賓特洛甫|巴托尔德·亨克爾]]（Barthold Henkell von Ribbentrop，生於1940年12月19日的柏林），育有：
**塞巴斯蒂安（Sebastian von Ribbentrop，生於1971年2月3日）
安娜·伊麗莎白·亨克爾被認為是一個傲慢、控制欲強的一個女性，並常挨批為[[麦克白|麦克白夫人]]一般主宰自己丈夫的女人<ref name="bloch 12-13"/>。里賓特洛甫之後說服丈夫受封為貴族的姨媽，在他姓中加上了貴族姓「馮」，然而里賓特洛甫從未受到[[貴族|貴族]]接納，也沒有受到過承認。在[[威瑪共和國|威瑪共和國]]的大部分時間裡，里賓特洛甫對政治不怎麼關心，也沒有[[反犹太主义|反猶情節]]<ref name="bloch 16">Bloch, ''Ribbentrop''，第16頁和第20至第21頁</ref>。里賓特洛甫作為他岳父香檳公司的助手處理了許多業務，並和猶太銀行家經常有往來，他成立一個進口公司的資金就是向猶太人貸款<ref name="current 708">''Current Biography 1941''，第708頁</ref>。

===早期政治生涯===
1928年，里賓特洛甫經人介紹給[[希特勒|希特勒]]，希特勒記述他為「能以別人買法國香檳的價格買下德國香檳的男人」，以及為一個能與外國聯繫的商人<ref name="current 708"/>，之後里賓特洛甫在妻子的催促下於1932年5月1日加入了[[納粹黨|納粹黨]]<ref name="bloch 26">Bloch, ''Ribbentrop''，第26頁</ref>。1933年1月，[[弗朗茨·馮·帕彭|弗朗茨·馮·帕彭]]與[[奧斯卡·馮·興登堡|奧斯卡·馮·興登堡]]（Oskar von Hindenburg

，[[保罗·冯·兴登堡|興登堡]]總統之子）一些朋友與希特勒協商，取代掉總理[[库尔特·冯·施莱谢尔|库尔特·冯·施莱谢尔]]<ref>1月22日，奧托·邁斯納和興登堡於里賓特洛甫家中會見希特勒、戈林和威廉·弗利克。經過短暫的閒聊後，希特勒請小興登堡在一間私室里交談，並在那裡講了超過1小時。<br />經過這次會談後，小興登堡改變了原有的想法，轉而說服興登堡總統任用希特勒為總理。</ref>。

這場會談的結果是使得希特勒於1933年1月30日透過任命成為[[德國總理|總理]]，而提供自家作為秘密會談場所、納粹黨黨員和身為帕彭好友的里賓特洛甫就因此受到希特勒的愛戴。因為里賓特洛甫是納粹黨的一名新黨員，並不受到黨內老成員的歡迎，[[戈培爾|戈培爾]]還在日記中將他寫為：「馮·里賓特洛甫的名字是買來的，錢是藉由婚姻娶來的、仕途則是騙來的<ref>Synder, Louis ''Encyclopedia of the Third Reich'', New York: McGraw-Hill，1976年出版，第295頁</ref>」為了彌補在黨中影響力不足的問題，里賓特洛甫開始將[[納粹主義|納粹主義]]也逐漸融入他的生活中，高調的提倡[[反猶主義|反猶主義]]<ref name="bloch 47">Bloch, ''Ribbentrop''，第47頁</ref>，成為一名狂熱的納粹份子。他之後部份憑藉著對於德國外國家的知識而成為希特勒的外交顧問，但大部分是因為對希特勒的諂媚與奉承之詞才有此地位<ref name="craig 420-421">Craig 1953，第420頁至第421頁</ref>。專職的外交部人員會將國外實際情況告訴希特勒，而里賓特洛甫則總是告訴希特勒所想聽到的<ref name="craig 420-425">Craig 1953，第420頁至第425頁</ref>。特別是里賓特洛甫會紀錄下希特勒對一些事物的想法與觀點，再提出與其相似的發言，藉此使希特勒認為自己是一名忠於國家社會主義的外交官<ref name="craig 420">Craig 1953，第420頁</ref>。里賓特洛甫也發現希特勒常以極端、偏激的方式來解決問題，他也因此常將自己的意見轉為此方向。

里賓特洛甫是希特勒的崇拜者之一，他的情緒很大一部份取於希特勒的好惡<ref name="bloch 51-52">Bloch 1992，第51頁至第52頁</ref>。1933年，里賓特洛甫獲得黨衛隊上校的頭銜，一時間裡他還與[[黨衛隊全國領袖|黨衛隊全國領袖]][[海因里希·希姆萊|海因里希·希姆萊]]友好，但最後彼此還是成為了敵人，主要是因為對於黨衛隊堅持自己的外交獨立性，而非倚靠里賓特洛甫<ref name="bloch many">Bloch 1992，第83頁、第208頁至第209頁、第215頁、第230頁、第254頁和第365頁至第367頁</ref>。
===外交官生涯===
1933年11月，里賓特洛甫開始進行他的外交生涯，他以一個非官方的外交官身份拜訪了倫敦，與[[拉姆齊·麥克唐納|拉姆齊·麥克唐納]]首相和[[約翰·西蒙|約翰·西蒙]]外交及聯邦事務大臣會面<ref name="bloch 40-44">Bloch, ''Ribbentrop''，第40頁至第44頁</ref>，但這場會談沒有任何實質上的內容<ref name="bloch 44">Bloch, ''Ribbentrop''，第44頁</ref>。里賓特洛甫在取得外交辦公室的職位之前，大部分時間都花在攻擊現任外交部長[[康斯坦丁·馮·紐賴特|康斯坦丁·馮·紐賴特]]上<ref name="michalka 167">Michalka 1993，第167頁</ref>，積極爭取該職位。起初紐賴特對里賓特洛甫這位對手保持輕視的態度，認為他這種連寫德文都會用字粗俗且[[文法|文法]]出錯的人，更不用說[[英文|英文]]和[[法文|法文]]了，對里賓特洛甫的競爭與攻擊不怎麼擔心<ref name="craig 423-424">Craig 1953，第423頁至第424頁</ref>。

當中國[[抗日戰爭|抗日戰爭]]正式打響，與[[大日本帝國|日本]]友好的[[納粹德國|德國]]起初還在觀望，但在里賓特洛甫主導的外交部隨即下令駐華的德國僑民和武官撤出，進而導致[[日軍|日軍]]變本加厲的侵華。隨後更將預定出售給中國的武器彈藥直接取消訂單，讓[[蔣介石|蔣介石]]、[[李宗仁|李宗仁]]等人驚愕，[[德制師|德制師]]和中央軍因此無法得到充分的彈藥對抗日本。之後中國對抗日本的武器只能向[[蘇聯|蘇聯]]和法國訂製，然而隨著納粹建立[[維琪法國|維琪法國]]、與[[蘇聯|蘇聯]]簽訂[[互不侵犯條約|互不侵犯條約]]並[[瓜分波蘭|瓜分波蘭]]，最後只剩下美國提供援助協同作戰，苦撐到[[美軍|美軍]]正式加入[[第二次世界大戰|第二次世界大戰]]之後，英國和加拿大才提供部分的武器援助[[國民革命軍|國軍]]，直到日本[[日本投降|戰敗投降]]。因此，[[中華民國國民政府|中國政府]][[要員|要員]]如[[蔣介石|蔣介石]]等人痛恨里賓特洛甫，在其[[蔣介石日記|日記]]上對此的記載的文字甚是憤恨。
