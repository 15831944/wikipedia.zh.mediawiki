'''约瑟夫·博胡斯拉夫·费尔斯特'''（{{lang-cs|'''Josef Bohuslav Foerster''' }}，{{bd|1859年|12月30日|1951年|5月29日}})，[[捷克|捷克]]作曲家，音乐评论家。生于一个音乐世家，早年就学于[[布拉格音乐学院|布拉格音乐学院]]，毕业后曾担任乐评家，并成为[[古斯塔夫·馬勒|马勒]]的好友。<ref>{{cite web |url=http://www.classicsonline.com/composerbio/Josef_Bohuslav_Foerster_23232/ |title=存档副本 |accessdate=2009-11-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20110429202534/http://www.classicsonline.com/composerbio/Josef_Bohuslav_Foerster_23232 |archivedate=2011-04-29 }}</ref> 1901年到[[汉堡|汉堡]]任教，1918年回国，晚年受到很高尊崇。费尔斯特寿命很长，是一位多产作曲家，在各种体裁上都有建树。他的作品常带有丰富的个人情感，有时显出神秘倾向。在音乐风格上，很少民族主义色彩，接近[[德奥|德奥]]传统。他的绝大部分作品目前已很少演奏。
