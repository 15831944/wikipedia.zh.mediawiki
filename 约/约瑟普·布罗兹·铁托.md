'''约瑟普·布罗兹·铁托'''（[[塞尔维亚-克罗地亚语|塞尔维亚-克罗地亚语]][[西里爾字母|西里爾字母]]：{{lang|sr|Јосип Броз Тито}}，拉丁字母：{{lang|sr|Josip Broz Tito}}；{{bd|1892年|5月7日|1980年|5月4日|}}），本名'''约瑟普·布罗兹'''，[[南斯拉夫|南斯拉夫]]革命家、[[政治家|政治家]]，[[南斯拉夫共产主义者联盟|南斯拉夫共产主义者联盟]]总书记、主席团主席，[[南斯拉夫社会主义联邦共和国|南斯拉夫社会主义联邦共和国]]执政35年的总统、总理，南斯拉夫元帅。尽管执政时期被人批评为[[威权统治|威权统治]]，但他的个人形象通常被视为南斯拉夫联邦内各民族统一的象征。他还是[[不結盟運動|不結盟運動]]發起人之一，与[[印度總理|印度總理]][[尼赫鲁|尼赫鲁]]、[[埃及總統|埃及總統]][[纳赛尔|纳赛尔]]共事。

== 早期生活 ==
=== 一战前 ===
铁托出生于[[奥匈帝国|奥匈帝国]][[克罗地亚|克罗地亚]]北部的村庄[[库姆罗韦茨|库姆罗韦茨]]，是家里的第七个孩子。其父亲为[[克罗地亚人|克罗地亚人]]，母亲为[[斯洛文尼亚人|斯洛文尼亚人]]。与外公度过童年，1905年小学毕业，1907年离开农村来到[[锡萨克|锡萨克]]當一個[[機械師|機械師]]的学徒，在那里首次见识到了[[五一劳动节|五一劳动节]]并开始关注[[工人运动|工人运动]]。

=== 革命家 ===
1913年，铁托被[[徵兵|徵兵]]加入[[奥匈|奥匈]]军队。1915年因重伤被俄军俘虏，在13个月的康复治疗后被送至[[乌拉尔山|乌拉尔山]][[劳改营|劳改营]]。在那里他被选为狱长。1917年2月，暴动工人冲进监狱将他们释放，铁托随后加入[[布尔什维克|布尔什维克]]组织。之后屡次被捕和逃亡，最终藏身于[[西伯利亚|西伯利亚]]。1918年，他加入了[[南斯拉夫共产党|南斯拉夫共产党]]支部。

== 崛起 ==
铁托在1937年接掌[[南斯拉夫共產黨|南斯拉夫共產黨]]（南共），在1940年10月召开的第五次全國代表大會上被選為南共[[總書記|總書記]]。一開始南共由於受到德蘇協定的影響，並沒有實際參與對德作戰，1940年以前甚至一度加入反戰活動。<ref>許綬南譯，Mihailo Crnobrnja著，1999，《南斯拉夫分裂大戲》。台北：麥田，頁111-112。</ref>

1941年[[軸心國|軸心國]]攻陷南斯拉夫，7月4日，德國進攻蘇聯後不久，南共中央決定發動全面性的武裝戰鬥，同年12月21日，建立第一支正規軍「第一無產階級旅（Prva proleterska brigada）」。 組織解放軍，其正式名稱為「[[南斯拉夫人民解放军和游击队|南斯拉夫人民解放军和游击队]]」，铁托任總司令<ref>黃鴻釗編，1996，《東歐簡史》。台北：書林，頁304-305。</ref>。

當時南斯拉夫除了人民解放軍之外，還有克羅埃西亞的獨立運動組織「[[烏斯塔沙|烏斯塔沙]]」以及塞爾維亞民族主義支持的「[[南斯拉夫祖國軍|南斯拉夫祖國軍]]」。一開始主力同為[[塞爾維亞人|塞爾維亞人]]的民族解放軍與南斯拉夫祖國軍一同作戰，但不久即因理念不同而分道揚鑣。南斯拉夫祖國軍採行等待戰術，等待攻擊[[德軍|德軍]]的適當時機。期間他們經常採取[[民族主義|民族主義]]的恐怖手段，加劇南斯拉夫內部的混亂。烏斯塔沙則是針對塞爾維亞、[[猶太人|猶太人]]，以及許多少數民族進行[[種族清洗|種族清洗]]。

铁托採取雙叉式的策略，一方面為了爭取成為[[反法西斯同盟|反法西斯同盟]]公認成員，對[[軸心國|軸心國]]進行武裝作戰。一方面是要在大戰結束時成為戰勝的政治力量。在1943年後到1945年大戰結束的這段期間，铁托的人民解放軍著手對抗祖國軍以及烏斯塔沙。<ref>許綬南譯，Mihailo Crnobrnja著，1999，《南斯拉夫分裂大戲》。台北：麥田，頁109-113。</ref>

== 重建南斯拉夫 ==
南斯拉夫共產黨在二戰中取得了勝利，並且得以決定國家的形式。1943年11月，[[南斯拉夫民族解放反法西斯議會|南斯拉夫民族解放反法西斯議會]]第二次會議，決定了戰後的南斯拉夫將由五個民族共組[[聯邦|聯邦]]。1945年戰爭結束後，该议会在11月的制憲會議宣布南斯拉夫成立新政府，罷黜[[君主制|君主制]]。重建統一的南斯拉夫國家，凝聚內戰中分裂的各民族，共產黨的領導者铁托扮演了關鍵的角色<ref>江秉彝，2007，〈南斯拉夫疆界與民族衝突(1991-2006)〉，碩士論文，東海大學政治系，頁22-23。</ref>。

由於第一南斯拉夫的經驗顯示，數量最多、分佈最廣的塞裔有壟斷政治的傾向。因而铁托試著降低[[塞爾維亞|塞爾維亞]]的規模，使之處於與其他民族差不多的位置。他重劃戰前的行政區疆界，將某些[[塞爾維亞人|塞爾維亞人]]口撥給[[克羅埃西亞|克羅埃西亞]]、[[波士尼亞|波士尼亞]]，以減少[[塞爾維亞|塞爾維亞]]的規模。[[塞爾維亞|塞爾維亞]]境內的[[匈牙利人|匈牙利人]]以及[[阿爾巴尼亞人|阿爾巴尼亞人]]也給了铁托藉口，建立了[[伏伊伏丁那|伏伊伏丁那自治省]]與[[科索沃|科索沃自治區]]兩個自治地區。1946年1月31日，南斯拉夫戰後第一部憲法規定，由六個共和國以及[[伏伊伏丁那|伏伊伏丁那自治省]]、[[科索沃|科索沃自治區]]組成「[[南斯拉夫聯邦人民共和國|南斯拉夫聯邦人民共和國]]（Federativna Narodna Republika Jugoslavija, FNRJ）」。1963年的憲法，再將之更名為「[[南斯拉夫社會主義聯邦共和國|南斯拉夫社會主義聯邦共和國]]（Socijalistička Federativna Republika Jugoslavija, SFRJ）」，並且將[[科索沃|科索沃]]升格為與[[伏伊伏丁那|伏伊伏丁那]]同等的自治省。<ref>江秉彝，2007，〈南斯拉夫疆界與民族衝突(1991-2006)〉，碩士論文，東海大學政治系，頁22-25。</ref>

== 民族政策 ==
南斯拉夫是一个多民族国家。铁托作为一个[[克罗地亚人|克罗地亚人]]，当政时期对南斯拉夫的主体民族[[塞尔维亚族|塞尔维亚族]]采取了打压政策。1966年起对兰科维奇集团“[[中央集权|中央集权]]主义——[[大塞尔维亚主义|大塞尔维亚主义]]”的打击十分严厉，其影响一直持续到1970年代，有4万塞族干部因此被整肃。在制度上，铁托时代也作了有利于抑制塞族强权的（或用塞民族主义者的话说是“压制塞尔维亚”的）安排。除了从塞族中划出新民族、缩小其版图以外，铁托设立了“[[南斯拉夫人|南斯拉夫族]]”，鼓励人们放弃原有族群认同而去改宗这一新的民族群体。这些政策使得国家凝聚力大为下降，也导致了南斯拉夫日后分解再分解，国家失去了维护主权和领土统一的能力。

== 文明建设 ==
在铁托时代，南斯拉夫党和政府把精神建设作为社会主义事业的一个重要组成部分。为了从整体上推动社会主义精神建设的进程，铁托引导党内和学术界就它所包含的两个方面的内容，即思想建设和文化建设，展开讨论和论证，又在此基础上通过法律程序作出了一系列有关政治思想、教育、科学、文化等方面的决议，并取得实践经验。

== 外交與經濟成就 ==
铁托是戰後首先敢挺身對抗蘇聯的擴張，與[[史達林|史達林]]决裂的共产黨領袖，直到1955年，铁托才与[[赫鲁晓夫|赫鲁晓夫]]改革后的苏联恢复正常外交关系，1956年[[匈牙利事件|匈牙利事件]]后，苏联与南斯拉夫关系一度又有恶化，后来随着[[中苏交恶|中苏交恶]]与决裂，苏南关系又迅速升温。<ref>[http://cpc.people.com.cn/GB/64184/64186/66672/4493667.html 1963年中共三评苏共中央公开信：南斯拉夫是真正的社会主义国家吗？]</ref>1950年代，南斯拉夫開始與亞非等地的第三世界國家有所接觸，先是[[印度|印度]]及[[緬甸|緬甸]]，之後拓展到[[埃及|埃及]]跟[[印尼|印尼]]，1955年與[[中华人民共和国|中华人民共和国]]建交。1956年，铁托與[[印度總理|印度總理]][[尼赫魯|尼赫魯]]及[[埃及總統|埃及總統]][[納瑟|納瑟]]三人在布里奧尼島發表聲明，支持[[不結盟運動|不結盟運動]]，並於1961年在南斯拉夫首都[[貝爾格萊德|貝爾格萊德]]召開首次高峰會。使其不論在國內，或是第三世界國家，都越來越受到歡迎。這項運動的成員國不願涉入冷戰，拒絕與美蘇兩大強國結盟，至2007年已有118成員國，涵蓋了[[聯合國|聯合國]]三分之二的成員國，全世界55%的人口居住在這些國家之中。<ref>江秉彝，2007，〈南斯拉夫疆界與民族衝突(1991-2006)〉，碩士論文，東海大學政治系，頁25。</ref>

1950到1960年代初期，南斯拉夫的經濟成長相當突出。雖然其生活水準始終無法真正趨近於[[西歐|西歐]]的水平，但是顯然超過其他共產黨所領導的國家。人民的滿意度與內部的安定度，不但能夠維持，甚至還進一步提升；隨之而來的驕傲感也不斷增加，使大部分的南斯拉夫人相信，他們的模式是一種有效的辦法。<ref>許綬南譯，Mihailo Crnobrnja著，1999，《南斯拉夫分裂大戲》。台北：麥田，頁119-120。</ref>

苏共在[[斯大林|斯大林]]时代认为[[铁托|铁托]]的南斯拉夫是[[修正主义|修正主义]]的样板，而在1960年代，[[赫鲁晓夫|赫鲁晓夫]]时期的苏联转而认为南斯拉夫是真正的社会主义国家，而[[毛泽东|毛泽东]]领导下的中国坚持继续称[[南斯拉夫|南斯拉夫]]为修正主义国家。<ref>[http://cpc.people.com.cn/GB/64184/64186/66672/4493667.html 1963年中共三评苏共中央公开信：南斯拉夫是真正的社会主义国家吗？]</ref>苏联[[勃列日涅夫|勃列日涅夫]]政府在1968年镇压[[捷克斯洛伐克|捷克斯洛伐克]][[布拉格之春|布拉格之春]]后，苏联与南斯拉夫关系一度又出现紧张，铁托派特使[[爱德华·卡达尔|爱德华·卡达尔]]到北京面见毛泽东，中国与南斯拉夫关系改善。<ref>[http://bbs.tianya.cn/post-free-3091905-1.shtml 南斯拉夫接受现代流行文化比中国早]</ref>为防止中国与南斯拉夫亲密，勃列日涅夫宣布[[勃列日涅夫主义|勃列日涅夫主义]]宣称的苏联可以为了安全武装干预“[[社会主义阵营|社会主义阵营]]”任何一国的范围不包括[[南斯拉夫|南斯拉夫]]，苏南关系才再度改善。<ref>[http://www.cccpism.com/moskow/viewussr/2/3/boke/b36.jpg 1969年勃列日涅夫会见铁托]</ref><ref>{{Cite web |url=http://www.2vi.org/w/waidianpingwenge/1971/09/143.htm |title=南斯拉夫发言人武伊察说：勃列日涅夫访问南斯拉夫“有助于推进南苏友好合作” |access-date=2013-07-07 |archive-url=https://web.archive.org/web/20131002170900/http://www.2vi.org/w/waidianpingwenge/1971/09/143.htm |archive-date=2013-10-02 |dead-url=yes }}</ref>

== 晚年與爭議 ==
1970年代中期以來，[[石油危機|石油危機]]和全球經濟萎縮迫使各地進行改變，也對南斯拉夫產生嚴重的影響。南斯拉夫經濟在1974年[[第一次石油危機|第一次石油危機]]時受到致命的重創，而接下來稱為「契約經濟（contractual economics）」的經濟改革又被評為「一團混亂。運作開銷龐大，生產動機和效率卻幾近於零」。使得南斯拉夫唯有大量向外借款，以維持投資與消費層面的活絡。铁托的晚年消耗在撐過新的政經局面當中，南共及铁托本人的凝聚力漸漸的腐蝕。經濟的崩潰，使得铁托的聲望大幅滑落，不過依舊維持著相當的水平，他更於1974年當選終身總統。

==去世==
铁托于1980年5月4日在[[斯洛文尼亚社会主义共和国|斯洛文尼亚]]首府[[卢布尔雅那|卢布尔雅那]]過世，享年88歲，結束37年南斯拉夫总统职务，但他的政治影響一直到1990年代南斯拉夫解體。<ref>許綬南譯，Mihailo Crnobrnja著，1999，《南斯拉夫分裂大戲》。台北：麥田，頁124。</ref>

== 葬礼 ==
[[File:Grob_Josipa_Broza_Tita.jpg|thumb]]
铁托並非驟然過世，他在醫院躺了四個月，他的病情是每天報紙的頭條新聞。來自世界各地的政要和要人群集铁托的葬禮，場面氣派，勝過其他同類場合。南斯拉夫人為近乎所有國家所表示的友誼和關懷感到驕傲。

铁托下葬在一座樸素的大理石墓中，這處墓地成為大規模的朝聖目標。铁托過世後的最初四年間，共有超過一千一百萬人前來謁靈，這個數字是南斯拉夫人口的一半。<ref>許綬南譯，Mihailo Crnobrnja著，1999，《南斯拉夫分裂大戲》。台北：麥田，頁131-132。</ref>
