'''约瑟夫·张伯伦'''（'''Joseph Chamberlain'''，{{bd|1836年|7月8日|1914年|7月2日}}），英国政治家。与当时其他政治家不同的是，张伯伦是一个没有接受过大学教育、白手兴家的商人。

== 早年 ==
1836年7月8日，約瑟夫·张伯伦生于伦敦，他父親（也叫約瑟夫）是一名成功的鞋匠。張伯倫在學生時代法語和數學成績優異，但由於家庭的原因，16歲時張伯倫主動輟學去自家的工廠上班。

== 工人企業家 ==
18歲時，張伯倫到[[伯明翰|伯明翰]]加入其叔父的螺釘生意，此後事業越做越大，成為著名的螺釘生產商，其產品佔據英格蘭三分之二的市場，並出口至美國、歐洲、印度、日本、加拿大和澳大利亞。

生活中的張伯倫關心時事，思想激進，擅長法語，喜歡辯論，經常參加辯論協會的活動。張伯倫信奉一神教，在英國還沒有建立國民教育體系的年代，他曾在伯明翰的主日學校任教，是一位貼近下層群眾的資本家。

== 從政 ==
張伯倫初入政坛时，是一个[[自由党_(英国)|自由党]]激进派，争取教育改革。四十岁时，他才进入议会，比其他重要政治家较晚。[[威廉·格萊斯頓|威廉·格萊斯頓]]在第二次出任首相时，考虑了张伯伦对自由党基层组织的影响力，邀请他出任[[商業創新及技能大臣|贸易委员会主席]]。这段时间里，他变得广为人知。首先是因为他大加批评[[保守党_(英国)|保守党]]领袖[[羅伯特·蓋斯科因-塞西爾，第三代索爾斯伯利侯爵|梳士巴利勋爵]]，然后是因为他在1885年大选期间，提出一个使农业劳工受惠的“未经批准计划”（Unauthorised Programme）。1886年，他因为不满首相格莱斯顿推动[[爱尔兰自治运动|爱尔兰自治运动]]，辞职下野。自由党分裂后，他加入了大部分党员来自伯明翰及其周边地区的[[自由统一党|自由统一党]]。

自[[1895年英国大选|1895年大选]]起，自由统一党就和梳士巴利勋爵所领导的保守党合作。两党胜出后，梳士巴利勋爵邀请他加入政府，担任[[殖民地大臣|殖民地大臣]]。殖民地大臣任内，[[第二次布尔战争|第二次布尔战争]]爆发，张伯伦负责了战时大部分军务。他是在[[1900年英国大选|1900年的“卡其大选”]]中左右选情的人物之一。1903年，他辞去殖民地大臣一职，进行关税改革运动。在这个议题上，虽然大部分自由统一党议员都支持张伯伦，但是仍然有少部分人与他意见不合，自此与他分道扬镳。结果，自由统一党在1906年大选中完全失败。几个月后，刚满七十岁的他患上了中风，从此退出政坛。

== 后人评价 ==
虽然张伯伦未能成为首相，但是他仍然被视为19世纪、20世纪之交最重要的英国政治家之一。他既是一位出色的演说家，又是一个有趣的人物，曾使两个主要政党分裂。[[温斯顿·丘吉尔|温斯顿·丘吉尔]]称他为一个“呼风唤雨”的人。张伯伦是[[奥斯丁·张伯伦|奥斯丁·张伯伦爵士]]和首相[[内维尔·张伯伦|内维尔·张伯伦]]的父亲。
