'''约瑟夫·穆斯卡特'''（'''Joseph Muscat'''，{{bd|1976年|1月22日||}}），[[马耳他|马耳他]]政治家，2013年起担任[[马耳他总理|马耳他总理]]。2008年6月6日穆斯卡特成为工党领袖，在2008年10月1日至2013年3月10日任反对党领袖。此前2004年至2008年他是[[欧洲议会|欧洲议会]]议员。

== 簡介 ==
穆斯卡特当过工党广播电台的记者，1996年在电视台任新闻主管助理。2001年和2004年之间他也是工党在线报纸的编辑。

穆斯卡特在圣阿洛伊修斯学院读完中学后，进入[[马耳他大学|马耳他大学]]和[[布里斯托大学|布里斯托大学]]，毕业获得商业管理和公共政策学士（1995年马耳他大学）、公共政策与荣誉文学士（1996年马耳他大学）学位、欧洲研究文学硕士（马耳他大学，1997年）和管理研究哲学博士（布里斯托尔大学，2007年）。<ref>{{cite web|url=http://www.scribd.com/doc/127035781/Joseph-Muscat-PhD-Thesis|title=Joseph Muscat PhD Thesis|work=Scribd}}</ref>

工党在2013年马耳他大选中获胜后，穆斯卡特在3月11日出任总理。穆斯卡特上任後，進行[[自由主義|自由主義]]改革，包括容許[[民事結合|民事結合]]。

2016年，[[巴拿马文件|巴拿马文件]]公布后。马耳他记者[[達芙妮·卡魯阿娜·加里齊亞|達芙妮·卡魯阿娜·加里齊亞]]据此指控总理约瑟夫·穆斯卡特和妻子收受外国资金。

== 健康 ==
2014年4月7日，穆斯卡特因紫外线辐射烧伤他的眼角膜而暂时失明。有60人在参加政治集会的前一天与他一样遇到相同问题。<ref>{{cite web|url=http://www.maltatoday.com.mt/news/national/37784/prime_minister_ordered_to_stay_at_home_as_his_vision_slowly_returns#.U0xj5ccjxl8|title=Updated - Minister, parliamentary secretaries return to work|work=MaltaToday.com.mt}}</ref>
