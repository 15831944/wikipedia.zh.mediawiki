《'''夹边沟'''》是中国大陆导演[[王兵_(导演)|王兵]]的第一部故事片，于2010年9月6日上映。影片根据[[楊顯惠|楊顯惠]]的小说《[[告别夹边沟|告别夹边沟]]》中的片段改编，以中国甘肃右派劳改农场为背景，描绘了右派分子被强制劳动改造、饥饿而死的悲惨景象<ref>{{cite journal |url=http://biweekly.hrichina.org/article/787 |publisher= 余世存 |title=历史没有音乐——关于电影《夹边沟》}}</ref>。该片曾在法国arte台播放<ref>{{cite web |url = http://videos.arte.tv/fr/videos/_le_fosse_de_wang_bing-3412586.html |title = "Le fossé" de Wang Bing - videos.arte.tv| date =2010-09-07| language=fr |publisher=arte }}</ref>。

== 评论 ==
王兵：“我不认为这是一部揭露性、抗议性的电影，我认为对中国是一部建设性的电影。作为今天的中国人，我们能够回顾、正视自己的历史、同时要尊重历史，不是为了抗议任何东西，只是为了让未来我们这个社会更加合理、更加公正、更加自由。”<ref>{{cite web|url= http://www.rfa.org/mandarin/yataibaodao/you-09072010145220.html|title= 反右影片震撼 《夹边沟》竞逐金狮奖（组图）|accessdate= 2010-09-07 |publisher= 自由亚洲电台|language=zh-hans}}</ref>
