'''贺子珍'''（{{bd|1910年|9月20日|1984年|4月19日|catIdx=H}}），原名'''贺桂圆'''，[[江西省|江西省]][[永新县|永新县]]人，[[毛泽东|毛泽东]]的第三任[[妻子|妻子]]。

== 简介 ==
[[File:_Mao_and_third_wife_He_Jijen.jpg|thumb]]

[[清|清]][[宣統|宣統]]二年（1910年）8月17日<ref> 孔东梅《生死两茫茫 听贺子珍讲述与毛泽东那过去的事情》：“1979年9月3日，也就是妈妈带我回京上学的第三天，中央派专机把外婆接到了北京。这个日子离9月9日外公逝世三周年纪念日还有四天，离10月7日外婆70岁生日还有一个月。 ”；“1976年是闰年，9月8日是农历八月十五，9月9日是八月十六。十五的月亮十六圆，两天都可以说是中秋节。八月中秋，正是外婆的生日。桂子飘香，花好月圆，所以得名桂圆。”。1979年10月7日是农历八月十七，八月十五、十六都是月圆之夜，两天都可以说是中秋节，八月十七也是月圆之夜中秋节期间。</ref>，贺子珍出生在[[江西|江西]][[永新县|永新县]]，同兄[[贺敏学|贺敏学]]、妹[[贺怡|贺怡]]一起被称作“永新三贺”。<ref>{{cite web |url=http://book.163.com/09/1110/11/5NOM5HFV00923K7T.html |title=“对毛泽东最好的女人”贺子珍的沉寂晚年 |accessdate=2009-11-10 |publisher=网易 |work= |language =zh-cn}}</ref>

贺子珍年輕時就读于[[芬兰|芬兰]][[内地会|内地会]]女传教士所办的永新女子学校。1927年5月参加[[中国共产党|中国共产党]]，并担任永新县委委员、妇女部长兼青年团委书记。“[[四·一二事件|四·一二事件]]”之后随其哥哥[[贺敏学|贺敏学]]上[[井冈山|井冈山]]，在[[袁文才|袁文才]]與[[王佐|王佐]]的山寨躲避[[国民党|国民党]]追捕。毛泽东上井冈山后，被袁文才派到毛身边工作。1928年5月与[[毛泽东|毛泽东]]結婚{{efn|当时毛泽东与杨开慧并未离婚，杨开慧于1930年被[[国民党|国民党]]将领[[何键|何键]]逮捕并处决。}}。<ref>http://v.ifeng.com/v/ykhmzd0423/#9816c9b6-e023-4c65-a7cf-cef57dae6737</ref><ref>{{cite web |url=http://news.xinhuanet.com/theory/2011-06/28/c_121594458.htm |title=存档副本 |accessdate=2014-11-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20151107103252/http://news.xinhuanet.com/theory/2011-06/28/c_121594458.htm |archivedate=2015-11-07 }}</ref>

1931年担任[[中華蘇維埃共和國|中央苏区政府]]机要科科长。1934年参加红军[[长征|长征]]。在路上，为掩护[[宋任穷|宋任穷]]的夫人，头、背、肺部被炸入弹片。

1937年初，中国共产党的党中央和中央军委机关迁到[[延安|延安]]，贺子珍跟随毛泽东抵达延安，随后贺子珍加入[[中国人民抗日军事政治大学|中国人民抗日军事政治大学]]成为一名学生，由于生活紧张、营养不良、气候不适和[[贫血症|贫血症]]，一日，贺子珍晕倒在学校的厕所中，旋即停学养病。<ref name="duck " >{{cite web |url= http://history.people.com.cn/GB/205396/15723279.html|title=贺子珍离开毛泽东是江青“插足”造成的吗？ |accessdate=2011年9月22日 |publisher=人民网 |work= |language =zh-cn}}</ref> 贺子珍养病期间，毛泽东经常与[[美国|美国]]女记者[[艾格尼丝·史沫特莱|艾格尼丝·史沫特莱]]以及女翻译[[吴莉莉|吴莉莉]]跳舞、聊天，引发贺子珍的误会，两人感情出现裂缝，不久之后，贺子珍怀孕了，由于她生孩子过多不想再生，打算去[[上海|上海]]做人工流产，1937年10月，贺子珍离延安远赴[[西安|西安]]。<ref name="duck " /> 当贺子珍抵达西安，上海被日军占领，她打算去上海市的计划破产，11月份，[[王明|王明]]、[[康生|康生]]、[[陈云|陈云]]从苏联莫斯科坐飞机经[[新疆|新疆]]、[[兰州|兰州]]、西安去延安，贺子珍受到启发，于是坐汽车从西安到了兰州，中共驻兰州办事处主任[[谢觉哉|谢觉哉]]曾劝说其返回延安，但贺子珍还是执意去了莫斯科。<ref name="duck " /> 

1938年，贺子珍去[[苏联|苏联]]治伤，但经苏联医生证实，弹片已无法取出。后入[[莫斯科中山大學|莫斯科东方大学]]学习，毕业后留在国际儿童院东方部工作，因女儿病重和领导发生争执，时任[[第三国际|第三国际]]东方部长的[[王明|王明]]指示强制將其送入精神病院拘押。

1947年经[[王稼祥|王稼祥]]和[[罗荣桓|罗荣桓]]向第三国际交涉，被釋放回国，但毛泽东已经和[[江青|江青]]同居。贺子珍留在[[中國東北地區|东北]]任东北财经委员会党支部书记。

1950年被定为三级甲等残疾，但终生没有领取残疾军人抚恤金。

[[中华人民共和国|中华人民共和国]]成立后，由于[[江青|江青]]阻挠，無法回[[北京|北京]]，於是南下任[[浙江省|浙江省]][[杭州市|杭州市]]妇联主任，后赴[[上海|上海]]养病。1959年7月或8月，在[[庐山|庐山]]会见毛泽东。1979年6月被[[中国人民政治协商会议|中国人民政治协商会议]]第五届二次会议增补为全国政协委员。

1984年4月19日在上海去世。

== 个人生活 ==

贺子珍的初恋是[[欧阳洛|欧阳洛]]，为中共湖北省委书记，1930年被国民党抓捕杀害。<ref name=" sheep" /> 1928年，贺子珍和毛泽东恋爱并且结婚，1938年贺子珍负气出走苏联，两人分手。<ref name=" sheep" /> 1947年，贺子珍回国住在[[哈尔滨|哈尔滨]]，后来住到[[上海|上海]]，两人的女儿[[李敏_(陕西)|李敏]]变成为两人情感沟通的特殊使者。<ref name=" sheep" /> 1959年7月9日，毛泽东和贺子珍在[[庐山|庐山]]的“[[美庐|美庐]]”相会，这是他们时隔22年后再次相见，这也是他们最后一次生前相见。<ref name=" sheep" /> 直到1976年毛泽东逝世后，贺子珍才在北京见到毛泽东的遗体。<ref name=" sheep" >{{cite web |url=http://www.china.com.cn/city/txt/2007-04/16/content_8122709.htm |title=贺子珍的初恋不是毛泽东 |accessdate=2007-04-16  |publisher=中国网 |work= |language =zh-cn}}</ref>

贺子珍和毛泽东共怀有10胎<ref>{{cite web|url=http://vip.book.sina.com.cn/book/chapter_38913_22674.html|title=情牵延安 |accessdate=2010-04-22 |first=孔东梅 |date=2005-04-04 |work=新浪网文化读书频道 |language=zh-hans}}</ref>，生了6个孩子，但只存活4个，其中三个孩子因红军转移被留在苏区不知音信，只有[[李敏_(陕西)|李敏]]一人在苏联陪同贺子珍，李敏以后在北京工作。她的其他子女分別如下：
*長女，[[楊月花|楊月花]]（1929年3月），原名“[[毛金花|毛金花]]”，生於[[福建|福建]]，失散至1973年才被找到，但未能和贺子珍以及毛泽东相见。<ref>[http://news.xinhuanet.com/school/2005-12/05/content_3879846.htm 毛泽东寻访长女 福建农妇是“红色公主”] {{webarchive|url=https://web.archive.org/web/20061115023526/http://news.xinhuanet.com/school/2005-12/05/content_3879846.htm |date=2006-11-15 }}</ref>
*四子<ref>以毛泽东所有子女排序，贺子珍所生第二个孩子、第一个男孩。</ref>（1930年），生於[[江西|江西]]，出生后夭折。
*五子，[[毛岸紅|毛岸紅]]（1932年11月），乳名“毛毛”，生於[[福建|福建]]，長征開始後由[[毛澤覃|毛澤覃]]撫養，毛澤覃死後下落不明。 
*次女，[[熊化芝|熊化芝]]（1935年2月），長征時生於[[貴州|貴州]]，後來被一戶苗族人家收養，被稱為“毛妹”，2003年被找到。
*三女，[[李敏_(陕西)|李敏]]（1936年），原名“[[李敏_(陕西)|毛姣姣]]”，生於[[陝北|陝北]]。 
*六子（1938年6月），有一个俄文名，生於[[莫斯科|莫斯科]]，十个月后夭折。 

贺子珍兄妹三人都是[[井冈山|井冈山]]时期的红军，贺子珍的哥哥[[贺敏学|贺敏学]]后来任[[福建省|福建省]]副省长，省政协主席；贺子珍的妹妹[[贺怡|贺怡]]（1911年—1949年，毛泽东之弟[[毛泽覃|毛泽覃]]之妻）曾任[[中国共产党|中共]][[吉安市|吉安]]地委组织部长，因为去寻找贺子珍留在苏区的儿子未果，在[[江西|赣]]南因车祸身亡，其子女被贺子珍抚养长大。
