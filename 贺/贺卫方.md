'''贺卫方'''（{{bd|1960年|7月17日|||H}}），[[山东|山东]][[牟平|牟平]]人，中共党员及[[北京大学法学院|北京大学法学院]][[教授|教授]]、[[博士生导师|博士生导师]]，著名[[法学家|法学家]]、[[公共知识分子|公共知识分子]]，《[[中外法学|中外法学]]》前主编。网名'''守门老鹤'''。他一直倡导中国要走宪政民主之路，2001年第一期《[[中国青年|中国青年]]》把他列为“将影响21世纪中国的100个青年人”之一，2011年度美国《外交政策》“全球百大思想家”之一。

==生平==
*1978年─1982年，[[西南政法学院|西南政法学院]]（后改名西南政法大学）学生；
*1982年─1985年，[[北京政法学院|北京政法学院]]（后中国政法大学）硕士研究生；
*1985年─1995年，[[中国政法大学|中国政法大学]]任教；
*1995年─2008年，[[北京大学|北京大学]]任教；
*2007年底，获聘[[浙江大学|浙江大学]][[光华法学院|光华法学院]]教授，宣布离开北大并办理离职手续，后因浙大受到某些不可抗力而毁约以致其未能成行；
*2008年底，居北京家中；
*2009年3月，重返北大法学院，被派往新疆[[石河子大学|石河子大学]]支教。
*2011年1月，回到[[北京大学|北京大学]]继续任教。

== 事件 ==
*1998年在《[[南方周末|南方周末]]》发表《复转军人进法院》一文，引起轩然大波，并引发大家对中国法官制度的思考<ref>{{Cite web |url= http://news.163.com/13/0828/22/97D9DJQF0001124J.html |title= 官方资料显示:内地31名高院院长7人无法学学历 |author=  |date= 2013-08-28 |publisher= 网易新闻 |language= zh-hans |accessdate= 2013年8月28日 |quote=  }}</ref>。
*2003年[[孙志刚事件|孙志刚事件]]中与另外几位教授上书，建议启动宪法规定的“特别调查委员会”。
*2005年6月23日公开声明暂停招收硕士生，被认为是对现行招生体制的质疑。
*2005年12月25日就[[武汉大学|武汉大学]]学术委员会副主任[[周叶中|周叶中]]剽窃事件发表文章“周叶中教授事件及其他”，严厉声讨学术腐败。
*2007年2月其讲述[[章诒和|章诒和]]著作的貼子（发表在新浪博客上）被无故删去，贺与三位人士发表抗议声明。
*2007年底，[[浙江大学|浙江大学]][[光华法学院|光华法学院]]院长[[孙笑侠|孙笑侠]]邀请时为[[北京大学法学院|北京大学法学院]]教授的贺卫方加入光华法学院，以期共同实现教授治校的理想。2008年7月，《南方周末》、《[[中国青年报|中国青年报]]》等媒体报道了贺卫方将辞去北大教职南下浙大。11月，[[华东政法大学|华东政法大学]]杨师群教授在自己的博客上称，辞去北大教职的贺被浙大拒收，“正处于无单位生存的尴尬境地”<ref>[http://shiqun2007.blog.sohu.com/104102455.html 《重庆会议遇名教授贺卫方》]</ref>。几日后，贺卫方在博客里委婉向外界表示自己没有去浙大，仍然居住在北京<ref>[http://blog.sina.com.cn/s/blog_488663200100bs04.html 《关于邮寄地址之启事》]</ref>。2008年12月16日，[[北京大学法学院|北京大学法学院]]院长[[朱苏力|朱苏力]]表示贺卫方已回北大任职。
*2008年12月10日，賀衛方和303位中國各界知識份子共同簽署《[[零八憲章|零八憲章]]》，呼籲中國實行民主憲政<ref>{{cite news|title=北大教授或因签署零八宪章遭报复|url=http://www.voachinese.com/a/a-21-w2009-03-13-voa33-61254917/1018621.html|agency=美國之音|date=2009-03-13}}</ref>。
*2009年3月贺卫方赴[[新疆|新疆]][[石河子大学|石河子大学]]支教。
*2011年1月回归[[北京大学|北京大学]]。獲美國「外交政策」（Foreign policy）雜誌選為「全球百大思想家」之一<ref>{{cite journal|title=The FP Top 100 Global Thinkers|journal=Foreign Policy|date=2011-11-28|url=http://foreignpolicy.com/2011/11/28/the-fp-top-100-global-thinkers-4/}}</ref>。
*2013年6月4日，贺卫方接受采访新加坡[[联合早报|联合早报]]采访时，表示希望中国共产党带领中国社会转向[[北欧模式|北欧模式]]的社会格局，并把自己改造成为一个[[社会民主党|社会民主党]]，然后承认一种竞争性的政治的合理性<ref>{{cite web |url=http://www.zaobao.com.sg/special/zbo/others/story20130604-212263/page/0/2|title=专稿：对话贺卫方谈中国宪政争议 |accessdate= 2013年6月4日 |publisher=联合早报|language=zh-cn}}</ref>。并称这就是他的[[中国梦|中国梦]]。
* 2013年，贺卫方在“新民说”文化沙龙上称，中国没有形成西式法治秩序，是因为2000年前就走错了路。他还宣称，西方对中国近代以来的欺凌，是清政府欺负西方人的结果<ref>{{cite web |url=http://news.ifeng.com/mainland/detail_2013_09/15/29623978_0.shtml|title=北大教授贺卫方：中国没形成法治秩序 2000年前走错路 |accessdate= 2013年9月15日 |publisher=凤凰网|language=zh-cn}}</ref>。
*2017年5月下旬，贺卫方表示，由于中共当局对[[言论自由|言论自由]]的打压达到几十年来空前严厉的程度，他个人多个社交媒体账号屡屡被封，因此未来无法再在社媒上发声，也不会在广大中国读者无法接触到的海外社媒平台上撰文<ref>{{cite web|url=http://www.voachinese.com/a/voanews-chinese-law-professor-20170527/3873607.html | language= | title=中国法学家贺卫方言论封锁下被噤声 |publisher=美国之音 |date=2017年5月27日 |accessdate= }}</ref>。
