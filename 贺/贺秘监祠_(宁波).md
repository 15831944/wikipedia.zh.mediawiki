'''贺秘监祠'''，位于[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[海曙区|海曙区]]柳汀街98号。贺秘监祠始建于[[明朝|明]][[洪武|洪武]]，后多次搬迁重建，现存建筑为[[清朝|清]][[同治|同治]]四年修复。1981年12月，贺秘监祠被公布为海曙区文物保护单位。
==历史==
宋绍兴十四年，时任郡守[[莫将|莫将]]在[[贺知章|贺知章]]读书故地取李白称呼贺知章的“四明逸老”建立“逸老堂”来祭祀[[贺知章|贺知章]]和[[李白|李白]]。<ref>{{cite web |url= http://daily.cnnb.com.cn/nbwb/html/2012-04/17/content_456419.htm|title=旧影“穿越”之旅：从月湖到日湖廿条桥|date=2012年4月17日 |publisher=宁波晚报|quote=到了南宋，柳汀之上又开始了新一轮的营建，先是绍兴十四年（１１４４）太守莫将在寻访盛唐诗人贺知章位于柳汀的读书故地之后，于众乐亭之南建造了“逸老堂”，其制坐南面北，而“逸老”之名则来源于李白对贺知章的称呼。}}</ref>[[元朝|元]][[至正|至正]]十九年，浙东农民起义军刘仁本、邱楠重建贺秘监祠堂，作祠专祀。明洪武间，贺秘监祠迁至现存之处。1981年12月，贺秘监祠被公布为海曙区文物保护单位。<ref>{{cite web |url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=917&type=2 |title=贺秘监祠 |date= 2006-02-09|publisher=宁波市文物保护管理所|quote=  贺秘监祠位于宁波海曙区柳汀街98号，为纪念唐代诗人贺知章而建，1981年12月被公布为区级文物保护单位，现存建筑为清同治四年（1865）修复。宋绍兴十四年（1144），郡守莫将在贺读书故地辟“逸老堂”以祀知章和李白。以后屡经兴废、更添，至元至正十九年（1359）始作祠专祀之。明洪武间（1368－1398）迁祠于今地。}}</ref>2012年7月，海曙区文物管理所投入资金400万对贺秘监祠进行修缮，工程于2013年2月验收完成。<ref>{{cite web |url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=22039 |title=贺秘监祠修缮工程顺利通过竣工验收|date=2013-02-04 |publisher=宁波市文物保护管理所|quote=2月1日，海曙区贺秘监祠修缮工程顺利通过竣工验收，来自市区两级文物部门及设计、施工、监理等单位的人员参加了验收会议。该修缮工程自2012年7月动工，总投资400余万元，由常熟古建园林建设集团有限公司负责施工。}}</ref>
==建筑结构==
贺秘监祠坐北朝南，分为均为五开间的三进，其中正殿题有“唐秘书监贺公祠”字样。<ref>{{cite web|url=http://www.haishu.gov.cn/zwdt/NewsInfo2.aspx?Id=173700|title=月湖夜景|date=2006-10-19|publisher=宁波市海曙区人民政府|quote=现存建筑为清同治四年(公元1865年)重修的，坐北朝南。该建筑共有三进，均为五开间。正殿门额题有“唐秘书监贺公祠”}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
