'''贺建奎'''（{{bd|1984年|<ref>{{cite news |author1=俞任飞 王湛 |title=钱报记者深圳对话贺建奎合伙人 |url=http://qjwb.zjol.com.cn/html/2018-11/28/content_3718431.htm?div=-1 |accessdate=2018-11-28 |work=钱江晚报 |date=2018-11-28}}</ref>|||He, Jiankui}}）是[[中华人民共和国|中国]]生物物理学者，曾為[[南方科技大学|南方科技大学]]副教授。其团队进行[[基因编辑|基因编辑]]的一对婴儿[[基因编辑婴儿事件|在2018年诞生]]，引发学术界和社会伦理的诸多争议与指责。<ref name="首例">{{cite news|url=https://finance.sina.com.cn/chanjing/cyxw/2018-11-26/doc-ihpevhck7262981.shtml|title=世界首例免疫艾滋病的基因编辑婴儿在中国诞生|author1=吕绍刚|author2=陈育柱|editor=万露|publisher=[[人民网|人民网]]|via=[[新浪|新浪]]|date=2018-11-26|access-date=2018-11-26|archive-url=https://web.archive.org/web/20181126073825/https://finance.sina.com.cn/chanjing/cyxw/2018-11-26/doc-ihpevhck7262981.shtml|archive-date=2018-11-26|dead-url=no}}</ref><ref>{{Cite web |url=https://news.sina.com.cn/c/2018-11-26/doc-ihmutuec3827836.shtml |title=南科大：贺建奎2月已停薪留职 项目违背学术伦 |access-date=2018-11-26 |archive-url=https://web.archive.org/web/20181126125952/https://news.sina.com.cn/c/2018-11-26/doc-ihmutuec3827836.shtml |archive-date=2018-11-26 |dead-url=no }}</ref>2018年，贺建奎登上英国科学杂志《自然》。《自然》稱他是“基因編輯流氓”（CRISPR rogue）。<ref>{{Cite web|url=https://finance.sina.cn/usstock/mggd/2018-12-19/detail-ihqhqcir8176299.d.html?from=wap|title=英国《自然》杂志年度十大科技人物：贺建奎上榜|accessdate=2018-12-19|author=于健|date=2018-12-19|publisher=新浪财经综合}}</ref>

2018年11月26日[[人民网|人民网]]记者吕绍刚、陈育柱以《世界首例免疫艾滋病的基因编辑婴儿在中国诞生》为标题披露贺建奎宣布他的团队将基因编辑技术用于人类[[受精卵|受精卵]]并植入母亲子宫，“一对名为露露和娜娜的基因编辑婴儿于11月在中国健康诞生。这对双胞胎的一个基因经过修改，使她们出生后即能天然抵抗艾滋病。”<ref name="首例"/>

消息公布后该实验所涉及的伦理等问题立刻引发国内外科学界的质疑和谴责，如112位中国科学家针对此实验的联合声明中，称实验存在严重的[[伦理学|生命伦理]]问题:“这项所谓研究的生物医学伦理审查形同虚设。直接进行人体实验，只能用疯狂来形容。”“此项技术（[[CRISPR|CRISPR基因编辑技术]]）早就可以做，但是全球的生物医学科学家们不去做、不敢做，就是因为脱靶的不确定其他巨大风险以及更重要的伦理。”“程序不正义和将来继续执行带来的对人类群体的潜在风险和危害是不可估量的。”<ref name="舆论场">{{cite web |title=舆论场：以科学之名 商人贺建奎的疯狂实验 |url=http://news.dwnews.com/china/news/2018-11-27/60101148_all.html |publisher=多维新闻网 |location=美国 纽约 |language=zh |date=2018-11-27}}</ref>  以致此事被媒体广泛称之为"[[基因编辑婴儿事件|基因编辑婴儿事件]]"，贺建奎本人也因此“一夜成名”。<ref name="一夜成名">{{cite web |author1=新京报记者 林子 杨砺 |title=“一夜成名”贺建奎：出身农门 被富豪导师改变人生 |url=https://tech.sina.com.cn/d/f/2018-11-28/doc-ihpevhcm1692947.shtml |website=新浪科技 |publisher=新浪网 |location=北京 |date=2018年11月28日 |access-date=2019-01-11 |archive-url=https://web.archive.org/web/20181128144957/https://tech.sina.com.cn/d/f/2018-11-28/doc-ihpevhcm1692947.shtml |archive-date=2018-11-28 |dead-url=no }}</ref> 人民网将此前上述正面角度的报道删除，人民日报之后的社评则对贺建奎进行了严厉批评。<ref name="許祺安">{{cite web |author1=許祺安 |title=賀建奎基因改造嬰兒或釀人類災難　誰該負責？ |url=https://www.hk01.com/%E8%AD%B0%E4%BA%8B%E5%BB%B3/265093/%E8%A7%80%E5%AF%9F-%E8%B3%80%E5%BB%BA%E5%A5%8E%E5%9F%BA%E5%9B%A0%E6%94%B9%E9%80%A0%E5%AC%B0%E5%85%92%E6%88%96%E9%87%80%E4%BA%BA%E9%A1%9E%E7%81%BD%E9%9B%A3-%E8%AA%B0%E8%A9%B2%E8%B2%A0%E8%B2%AC |website=香港01新闻 |publisher=香港01有限公司 |location=香港 |date=2018年11月29日}}</ref>

2018年11月28日，贺建奎在香港第二届基因编辑国际峰会上称，对“基因编辑婴儿”这项医学上的突破感到自豪。在峰会之后，[[南方科技大学|南科大]]的校长陈十一与其会谈并询问了实验的详情并随后一同回到深圳，[[蘋果日報_(香港)|苹果日报]]、[[纽约时报|纽约时报]]声称贺建奎现已被校方“软禁”<ref>{{Cite web|url=https://hk.news.appledaily.com/china/realtime/article/20181201/58981489|title=【基因編輯嬰兒】內幕消息：賀建奎被「軟禁」  實驗團隊有哈佛教授|accessdate=2018-12-02|work=Apple Daily 蘋果日報|archive-url=https://web.archive.org/web/20181202202641/https://hk.news.appledaily.com/china/realtime/article/20181201/58981489|archive-date=2018-12-02|dead-url=no}}</ref><ref>{{cite web | title=基因编辑婴儿事件后，贺建奎疑被隔离看守 | website=纽约时报中文网 | date=2019-01-02 | url=https://cn.nytimes.com/china/20190102/he-jiankui-china-scientist-gene-editing/ | language=zh | access-date=2019-01-11}}</ref>。不過，{{fact|南方科技大學隨後向《南華早報》否認帶走賀建奎。}}
