'''賀蘭山'''（[[蒙古語|蒙古語]]：[[File:Alasa_aghula.svg|17px]]；Alaša aɣula）位於今日[[中國|中國]][[寧夏回族自治區|寧夏回族自治區]]與[[內蒙古自治區|內蒙古自治區]]的交界，為南北走向，長度約200公里，主峰敖包疙瘩海拔高3556米。贺兰山以西是[[腾格里沙漠|腾格里沙漠]]，以東是宁夏平原，是[[地理|地理]]和[[气候|气候]]的重要分界。

贺兰山地区自古就有人类活动，有[[旧石器时代|旧石器时代]]的[[贺兰山岩画|贺兰山岩画]]，[[西夏|西夏]]时期的[[拜寺口双塔|拜寺口双塔]]等遗迹。在[[蒙语|蒙语]]中，贺兰是奔腾的骏马的意思，贺兰山的名字也是由此而来(此说有误，唐代韦蟾已经有诗:“贺兰山下果园成”）。

== 地质地貌 ==
[[File:贺兰山_大水沟.JPG|200px]]
贺兰山为南北向地垒式山地，南北长约180公里，东西最宽处可达50公里。[[File:The_helan_mountain_peaks.JPG|200px]]，海拔3555米]]<ref>{{Cite web |url=http://www.57uu.com/ningxia/lxjj/13_38_48_610.htm |title=宁夏回族自治区地理概况 |access-date=2009-04-22 |archive-url=https://web.archive.org/web/20080913095503/http://www.57uu.com/ningxia/lxjj/13_38_48_610.htm |archive-date=2008-09-13 |dead-url=yes }}</ref>贺兰山与南部的[[牛首山|牛首山]]褶断带、[[清水河|清水河]]—六盘山褶断带、罗山—云雾山隆起带构成了一系列的背向斜的断层。地貌上看，贺兰山西侧平缓，而东侧陡峭险峻，有大量露出地表的断层，东侧与银川平原垂直落差可达2000米。贺兰山北部以[[花岗岩|花岗岩]]为主，由于接近乌兰布和沙漠干旱少雨，所以物理风化强烈，贺兰山主体在贺兰山中部，山势陡峭，山体庞大，海拔较高，一般在2000-3000米之间，主峰[[敖包疙瘩|敖包疙瘩]]就在贺兰山中部，海拔3555米，贺兰山中部东西宽度可达50公里。贺兰山南部山势相对和缓。有汝其沟、大水沟、小水沟、贺兰沟、插旗沟、苏峪口沟、三关口沟等50多条沟谷<ref>[http://gfjy.jiangxi.gov.cn/HTMNew/2059.htm 贺兰山]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，沟道成V型，下部较为宽阔，沟底砾石遍布，沟口一般是碎石遍布的[[冲积扇|洪积扇]]。<ref>[http://www.cfern.org/wldt/wldtDisplay.asp?Id=836 宁夏贺兰山森林生态系统定位研究站建设项目启动] {{webarchive|url=https://web.archive.org/web/20150602055707/http://www.cfern.org/wldt/wldtDisplay.asp?Id=836 |date=2015-06-02 }}</ref>

== 气候 ==
贺兰山是中国[[季风气候|季风气候]]和非季风气候的分界线，贺兰山被[[毛乌苏|毛乌苏]]，[[乌兰布和|乌兰布和]]，[[腾格里|腾格里]]三大沙漠包围，湿润的海洋气流很难对贺兰山产生大的影响，而来自西伯利亚的冷空气在贺兰山地区活动频繁，因此，贺兰山区终年干旱少雨，四季分明，日超强烈，无霜期158天，年平均气温8.5℃，最高气温35℃左右，最低气温-25℃左右<ref>[http://yunnan.stis.cn/html/cyzn/yndlxx/lytj/20070821/299816.html 宁夏贺兰山]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。降水主要集中在7-9月，年降水量200-400毫米，年蒸发量在1000毫米以上，由于蒸发量与降水量的差值巨大，所以空气极其干燥。

== 生态 ==
[[File:贺兰山_山丹丹.JPG|200px]]]]

贺兰山地处温带草原区与荒漠草原的分界线，植被垂直分布非常明显，植被类型非常复杂，贺兰山植物群落有11个植被型55个群系。有[[草原|草原]]，[[荒漠|荒漠]]，[[针叶林|针叶林]]，疏林草原，还有各种灌丛、草甸和少量[[落叶阔叶林|落叶阔叶林]]。<ref>{{cite web |url=http://www.cfern.org/wldt/wldtDisplay.asp?Id=836 |title=存档副本 |accessdate=2009-04-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20150602055707/http://www.cfern.org/wldt/wldtDisplay.asp?Id=836 |archivedate=2015-06-02 }}</ref>

山地草原分为典型草原和荒漠草原两个亚类。疏林草原分布于贺兰山低山半干旱地带，海拔1600－2000米间的广阔范围中，介于山地草原带与山地针叶林带之间，形成一个完整的山地疏林草原带。由于贺兰山低山区干旱，蒸发量大，中山区与亚高山温度又太低，以及[[水热条件|水热条件]]不适等因素，导致贺兰山区落叶阔叶林面积很小，并且分布零散。 

贺兰山区植被垂直分非常明显，主要分为4个植被垂直带，海拔1600米以下山前荒漠与[[荒漠草原|荒漠草原]]带，1600～1900米出现覆盖度较大，草本植物生长较好的山麓与[[低山草原|低山草原]]带、1900～3100米分布有大量以青海云杉为主的中山和[[亚高山针叶林|亚高山针叶林]]带，其中分布于海拔2400~3100米的阴坡的青海云杉纯林带郁闭度大，更新良好，是贺兰山区最重要的林带；<ref>[http://www.nxnet.cn/zhuanti/2008zt/wenhuacheng/hlswh/200902/t20090212_444237.htm 贺兰山山脉简介] {{webarchive|url=https://web.archive.org/web/20111227135020/http://www.nxnet.cn/zhuanti/2008zt/wenhuacheng/hlswh/200902/t20090212_444237.htm |date=2011-12-27 }}</ref>  3100米以上主要是高山与亚[[高山灌丛|高山灌丛]][[草甸|草甸]]带，是较好的夏季牧场。

贺兰山区阴阳坡之间的植被分布差异很大，在低山带, 阳坡多为草原植物群落，而阴坡则主要是灌丛；在中山带，阴坡以[[青海云杉|青海云杉]]林为主，阳坡以灰榆、杜松疏林和其它中生灌丛为主；3100米以上的[[高山草甸|高山草甸]]阴阳坡分异不明显。由于靠近沙漠，贺兰山南段和北段[[荒漠化|荒漠化]]程度较高，森林面积很小。贺兰山东坡陡峭险峻，保水能力差，相对于更加平缓的西坡，森林覆盖率，植被更加单一。

贺兰山区植物群落多样性丰富，森林类型主要有青海云杉林，油松林，杜松林，山杨林，灰榆林，油松林、青海云杉林，青海云杉、山杨林，油松、山杨林，杜松、灰榆林。保护区有野生维管植物80科324属690种，其中野大豆、四合木、沙冬青、蒙古扁桃、贺兰山丁香是国家和自治区重点保护植物。<ref>[http://www.nxforestry.gov.cn/slbh.asp?ClassID=178&SpecialID=206]</ref> 

贺兰山区野生动物种类繁多，主要有有[[加拿大马鹿|马鹿]]、獐、[[盘羊|盘羊]]、[[金钱豹|金钱豹]]、青羊、石貂、蓝马鸡等180余种。有野生动物177种，隶属于20目44科，其中金雕、黑鹳、大鸨、胡兀鹫、白尾海雕为国家一级保护动物，有[[岩羊|岩羊]]、马鹿、蓝马鸡、[[马麝|马麝]]等国家二级保护动物13种。<ref>[http://www.nxforestry.gov.cn/slbh.asp?ClassID=178&SpecialID=206]</ref>

1988年，賀蘭山被列為[[國家級自然保護區|國家級自然保護區]]。

== 資源 ==
山区有石嘴山等10座大型矿区，富含优质[[煤炭|煤炭]]。另外有[[磷灰岩|磷灰岩]]、石英砂岩、粘土岩、灰岩等矿产，其中小滚钟口生产的粘板岩质地莹柔，用它雕成的贺兰石[[砚|砚]]是[[宁夏五宝|宁夏五宝]]之一。

[[漢能控股|漢能控股]]2002年起籌辦賀蘭山[[風力發電|風力發電]]廠持續裝機，2013年共計31台機組。

== 文化与历史 ==
自古以來，各民族在這裡繁衍，留下了不少史前文化、遊牧文化、農耕文化遺產。西夏在寧夏立國,有大批[[西夏|西夏]]的皇陵。除此之外，有[[賀蘭山岩畫|賀蘭山岩畫]]、[[滾鐘口寺|滾鐘口寺]]廟建筑群等文化遺跡。

賀蘭山也是北方[[遊牧民族|遊牧民族]]的標誌。著名的[[宋詞|宋詞]]《[[滿江紅|滿江紅]]》(疑為後人偽托[[岳飛|岳飛]]所作)有「駕長車踏破賀蘭山缺」的詞句。
