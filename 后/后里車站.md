== 歷史 ==
[[File:后里車站增建東站2017年11月工程.jpg|thumb]]
*1905年5月15日：設「-{後里庄}-停車場」。
*1908年4月20日：正式啟用。
*1920年10月1日：改為「-{後里}-」。
*1955年10月10日：改為「后里」。
*現今站房於1997年落成。
*2008年5月15日：正式停靠[[自強號列車|自強號]]。
*2015年6月30日：啟用多卡通刷卡機。
*因應[[2018年臺中世界花卉博覽會|2018年臺中世界花卉博覽會]]，新建東站，整建西站，所有設施於11月3日啟用<ref>[http://money.udn.com/money/story/5641/941173-2018%E8%8A%B1%E5%8D%9A%E5%9C%A8%E5%8F%B0%E4%B8%AD-%E5%8F%B0%E9%90%B5%E5%B0%87%E8%A8%AD%E5%90%8E%E9%87%8C%E6%9D%B1%E7%AB%99 2018花博在台中，台鐵將設后里東站] {{webarchive|url=https://web.archive.org/web/20150603190411/http://money.udn.com/money/story/5641/941173-2018%E8%8A%B1%E5%8D%9A%E5%9C%A8%E5%8F%B0%E4%B8%AD-%E5%8F%B0%E9%90%B5%E5%B0%87%E8%A8%AD%E5%90%8E%E9%87%8C%E6%9D%B1%E7%AB%99 |date=2015-06-03 }}，聯合報，2015-06-02</ref>。

== 評估計畫 ==
*[[2010年中華民國直轄市市長暨市議員選舉|2010年五都選舉]]中數位台中市長候選人提出海岸線臺中段雙軌高架捷運化的構想，同時也規劃一條連結臺中市大甲區、外埔區、后里區的大台中高架環狀捷運鐵路。臺中市政府預計將於2013年開始進行「大臺中地區山海線鐵路雙軌高架化建置計畫」可行性評估的的工作，在高架計畫中預計將后里車站予以高架捷運化。{{R|future_plan-1}}{{R|future_plan-2}}{{R|future_plan-3}}{{R|future_plan-4}}

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]，停靠車種以[[臺鐵區間車|區間車]]為主，少數[[自強號列車|自強號]]也有停靠。
**[[普悠瑪列車|普悠瑪號]]：全部班次皆不停靠。
**[[自強號列車|自強號]]：僅停靠北上(順行)102次、104次、274次、170次，南下(逆行)177次(週日停駛)、6177次(週日行駛)、155次、6473次(週日行駛)。
**[[觀光列車|環島觀光列車]]：不停靠。
**[[台鐵區間快車|區間快車]]：停靠北上(順行)2042次，南下(逆行)2011次。
*后里車站是中南部[[臺鐵區間車|區間車]]主要折返點之一。(后里-[[潮州車站|潮州]]/([[嘉義車站|嘉義]]))
*車站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳章留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。
