'''后里區'''是[[臺中市|臺中市]]的市轄區，位於臺中市北端，[[大甲溪|大甲溪]]與[[大安溪|大安溪]]之間，原名「內埔」，後因與[[臺灣省|臺灣省]][[屏東縣|屏東縣]][[內埔鄉|內埔鄉]]同名，改以轄內最大聚落「后里」為名。氣候屬[[亞熱帶|亞熱帶]]氣候，夏季多雨、冬季乾旱。特產有[[紅龍果|紅龍果]]、巨峰[[葡萄|葡萄]]、高接[[梨|梨]]、馬鈴薯等；此區亦是台灣製造[[薩克斯風|薩克斯風]]的產業聚集地（如[[張連昌|張連昌]]、[[薩得爾企業有限公司|薩得爾公司]]等），使后里成為臺灣西洋樂器製造地。[[2018年臺中世界花卉博覽會|2018年臺中世界花卉博覽會]]主辦場。

== 歷史 ==
后里區原屬[[諸羅縣|諸羅縣]]，[[清|清]][[雍正|雍正]]九年（1731年）改隸[[淡水廳|淡水廳]]，清[[光緒|光緒]]十三年（1887年）劃歸[[苗栗縣|苗栗縣]]。

[[台灣日治時期|日本統治]]台灣後，1898年改隸屬[[台中縣|台中縣]][[大甲區|大甲]]辦務署[[苗栗三堡|苗栗三堡]]，1901年改為[[苗栗廳|苗栗廳]]苗栗三堡內埔區，1909年改為[[臺中廳|台中廳]]苗栗三堡'''內埔區'''，1920年改制[[臺中州|台中州]][[豐原郡_(台灣)|豐原郡]]'''[[內埔庄_(台中州)|內埔-{庄}-]]'''。1935年的[[1935年新竹台中地震|新竹台中地震]]造成內埔-{庄}-962人死亡、1,694戶住家毀損。<ref>{{cite book|title=昭和十年　臺灣震災誌|publisher= 臺灣總督府|date=1936-03-25}}</ref>

[[臺灣戰後時期|二戰後]]為臺中縣豐原區'''內埔鄉'''，後因與[[屏東縣|屏東縣]][[內埔鄉|內埔鄉]]同名，復於1955年改以鄉內另一大聚落'''后里'''為名，成為'''后里鄉'''，同地亦有[[后里車站|后里車站]]；唯后里本名'''-{後}-里'''，在[[台語|台語]]仍讀舊名（[[白話字|白話字]]：{{unicode|Aū-lí}}）。2002年2月7日，樂業村併入眉山村（今眉山-{里}-）、義里村（今義-{里}--{里}-）西側分割出義德村（今義德-{里}-）。2010年12月25日隨著臺中縣市合併，改為'''后里區'''。

== 自然水文 ==
;地形
*后里區位於台中市北翼，北隔[[大安溪|大安溪]]，和[[臺灣省|臺灣省]][[苗栗縣|苗栗縣]]相接；南隔[[大甲溪|大甲溪]]，與[[豐原區|豐原區]]、[[神岡區|神岡區]]相臨；西連[[月眉山_(臺中市)|月眉山]]與[[外埔區|外埔區]]為界；東以[[埤頭山|埤頭山]]與[[東勢區|東勢區]]為鄰，地理環境成自然風格，氣候宜人、資源豐富，是個擁有好山好水，且地靈人傑的好地方。
*后里區五分之四的面積屬於台地地形，另外五分之一為丘陵地形；[[后里台地|后里台地]]位於大安溪、大甲溪兩溪之間，[[大甲扇狀平原|大甲扇狀平原]]以東，[[東勢丘陵|東勢丘陵]]以西，地形走勢從東北側的海拔300公尺，逐漸向西南緩降成為120公尺。
;氣候 
屬於亞熱帶季風氣候，冬季乾旱、夏季高溫多雨，年平均雨量1300~1500公釐，年平均溫度22.1度，適宜農作生產及其他經濟活動。

== 行政 ==
[[File:內埔庄役場.jpg|thumb]]

== 經濟發展 ==
{{Expand-section |time=2011-04-26}}
;工商產業
早期政府在設有[[月眉糖廠|月眉糖廠]]、后里馬場，屬農牧工業類的產業。近年政府再利用后里農場、七星農場土地成立[[中部科學工業園區|中部科學工業園區]]后里園區(后里基地、七星基地)，使工業類型轉型朝科技產業發展。后里糖廠、后里馬場、[[泰安車站_(舊山線)|泰安舊站]]與[[后豐鐵馬道|后豐鐵馬道]]的轉型、[[麗寶樂園|麗寶樂園]]的設置。[[2018年臺中國際花卉博覽會|2018年國際花卉博覽會]]的主辦城市，於2018年11月3日至2019年4月28日止，在后里區的后里馬場、后里軍營區、豐原葫蘆墩公園、外埔永豐段等地舉行，將奠定了后里觀光產業發展的基礎<ref>[http://www.nownews.com/2012/09/12/91-2853578.htm 2018花博在臺中！胡志強：帶動5百億元效益，盼超越臺北]</ref>。


;農畜業
區內政府設有月眉農場、舊社農場、外埔農場等土地。主要的農特區有[[鬱金香|鬱金香]]、[[文心蘭|文心蘭]]、[[劍蘭|劍蘭]]、[[火鶴花|火鶴花]]、[[百合|百合]]、[[彩色海芋|彩色海芋]]、[[蕃茄|牛奶蕃茄]]、[[草莓|草莓]]、[[紅柿|紅柿]]、[[馬鈴薯|馬鈴薯]]、[[高接梨|高接梨]]、[[巨峰葡萄|巨峰葡萄]]、[[椪柑|椪柑]]、[[紅龍果|紅龍果]]。

*[[后里馬場|后里馬場]]：創建於1937年，是[[台灣總督府|台灣總督府]]的產馬牧場，戰後1952年5月設立[[聯勤|聯勤]]種馬牧場，替軍隊提供軍需馬匹。後來部隊機械化，馬匹的需求不再這麼重要，1997年移交鄉公所經營。由於曾經設置過很多電影的佈景，所以早期台灣的電影、電視劇取景都曾在-{后里}-馬場內拍攝，後來這些設施都陸續撤除。近年，在縣政府發展觀光的政策下，后里馬場會不定時的舉辦藝文活動，例如前一陣子的兩馬音樂季，吸引了大批民眾的參與，也使得本來平平無奇的馬場，變成后里重要的觀光勝地<ref>[http://taiwan.net.tw/pda/m1.aspx?sNo=0001112&id=9407 交通部觀光局 后里馬場] {{webarchive|url=https://web.archive.org/web/20120119044306/http://taiwan.net.tw/pda/m1.aspx?sNo=0001112&id=9407 |date=2012-01-19 }}</ref>。
*[[毘盧禪寺|毘盧禪寺]]：為台灣負有盛名的寺院，1930年落成，位於后里區八仙山支脈太平山山麓，特色為酷似西洋建築的建築外型還有清幽的環境，其中「毘廬曉鐘」更被列為原[[臺中縣|臺中縣]]八景，更值得一提的，進入禪寺之前，外門牌坊的正上方豎立著「毘廬禪寺」四大字，[[吳佩孚|吳佩孚]]所題，可見毘廬禪寺所歷經年代之久遠。
