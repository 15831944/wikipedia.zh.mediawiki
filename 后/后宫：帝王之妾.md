《'''后宫：帝王之妾'''》（{{lang-ko|'''후궁: 제왕의 첩'''}}），2012年6月6日上映于[[韩国|韩国]]的古装情欲片，导演{{link-ko|金大承|김대승}}。<ref>{{cite web |url=http://ent.sina.com.cn/m/2012-05-25/12093639938.shtml |title=韩片《后宫》曝大尺度海外版海报 |accessdate=  2012年5月25日|publisher=新浪网 |work= |language =zh-cn}}</ref>

== 剧情 ==
电影以[[朝鲜王朝|古朝鲜王国]]为背景，讲述宫廷斗争<ref name="rat"/>，敘述朝鲜国王、内侍和华研三人在朝鲜宫廷中的命运<ref>{{cite web |url= http://ent.163.com/12/0528/08/82J1CREG000300B1.html|title= 《后宫：帝王之妾》曝国际版海报 大尺度引关注|accessdate= 2012-05-28 |publisher=网易 |work= |language =zh-cn}}</ref>。
