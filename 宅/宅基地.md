'''宅基地'''是指现行[[中华人民共和国户籍制度|中华人民共和国户籍制度]]和[[中华人民共和国土地制度|土地制度]]下，[[农业人口|农业人口]]获得的用以建造[[住宅|住宅]]的集体所有权的农村土地。

== 问题 ==
[[农业人口|农业人口]]可以购买城镇住房，而作为[[非农业人口|非农业人口]]的城镇居民则不得购买农村宅基地。但在经济杠杆下，[[小产权房|小产权房]]屡禁不绝。[[空心房|空心房]]亦是地方政府难以消灭的社会问题。由于[[性别歧视|性别歧视]]，女性申请宅基地较为困难<ref>{{Cite web |url=http://mt.sohu.com/20160413/n444096084.shtml |title=女性申请宅基地真的那么难吗？ |accessdate=2017-04-05 |author= |date=2016-04-13 |work= |publisher=搜狐网，来源：社员网 |language=Zh-hans |deadurl=no |archiveurl=https://web.archive.org/web/20170406111442/http://mt.sohu.com/20160413/n444096084.shtml |archivedate=2017-04-06 }}</ref>，同时女性失去土地权益的情况在增加<ref>{{Cite web |url=http://wemedia.ifeng.com/282574491713450/wemedia.shtml |title=又一例！浙江金华妇女无权分宅基地：一直都是儿子才分地 |accessdate=2017-04-05 |author= |date=2016-09-24 |work= |publisher=凤凰新闻 |language=Zh-hans |deadurl=no |archiveurl=https://web.archive.org/web/20170406110807/http://wemedia.ifeng.com/282574491713450/wemedia.shtml |archivedate=2017-04-06 }}</ref>。

2010年代，中国各地推行宅基地制度改革试点，目标主要有三个：（1）让农民“居有其屋”，（2）实现农民的宅基地财产权，（3）腾退多余宅基地。其中让农民“居有其屋”是宅基地制度改革的根本前提，这是中华人民共和国农村基本制度安排和社会保障。虽然大量农民在城市化过程中进城务工经商，乃至在城市居住，但进城的主要是青壮年劳动力，且在城市定居有困难，所以[[中华人民共和国农村土地承包法|农村土地]]和宅基地仍是农民根本的社会保障。实现农民的宅基地财产权一般是通过抵押获得贷款的权利，以及通过经营获得收入的权利；但前者的困难是中央政策明确要求不允许农民失去农村土地承包权和宅基地使用权，所以抵押担保不可能实现；而后者的问题是只有城中村、城乡结合部及少数沿海发达地区农村的宅基地才有通过经营获得收入的可能，而且在宅基地制度改革之前这些地方的农民已通过宅基地经营获得收入。政府动员农民腾退多余宅基地的目的是希望在建设用地增减挂钩制度中，通过腾退多余宅基地，将农村宅基地转为耕地，腾出的指标用于增加城镇建设用地。但事实上中国并不急需扩大耕地面积，农村多余宅基地事实上有不少也被农民当作耕地使用，另一方面中小城镇并没有快速建设扩张的可能，大中城市又不缺建设用地指标，政府投入巨资腾退多余宅基地，却很难收回成本<ref>{{Cite web |url = http://www.guancha.cn/HeXueFeng/2017_05_18_408910_1.shtml |title =贺雪峰：农村宅基地改革试点的若干问题  |author =  |date =2017-05-18  |publisher = 观察者网|language =  |accessdate =  |quote =  }}</ref>。
