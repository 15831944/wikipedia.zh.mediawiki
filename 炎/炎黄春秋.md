《'''炎黄春秋'''》是一本综合中文月刊，1991年創刊，中国艺术研究院主办、[[中国炎黄文化研究会|中国炎黄文化研究会]]主编，编辑部在[[北京市|北京市]][[西城區|西城區]][[月坛南街|月坛南街]]69号。現任社長為贾磊磊（法定代表人），总编辑陈剑澜。

該刊及網站在過去一段時間在中国带有政治[[自由派|自由派]]色彩，代表建制内自由主义势力，与政治[[极左|极左]]雜誌及网站《[[乌有之乡|乌有之乡]]》立場及意見相左<ref name="秦晉">{{cite book |author = 秦晉 |title = 求索與守望--中國民運江湖回望錄 |url = http://books.google.com/books?id=Oiu-RLPiq5AC&pg=PT151 |accessdate = 2012-12-24 |publisher = 秀威資訊 |ISBN = 978-986-5915-24-7 |pages = 151–152 }}</ref>。該刊及網站主要刊載古今中外的歷史記述及評論文章，著者以[[中國共產黨|中共]][[中共八大元老|元老]]、作家與學者為主，具有一定權威性<ref>[http://cpc.people.com.cn/GB/69112/137694/137697/8341286.html 萧克将军与《炎黄春秋》]，《[[人民日报|人民日报]]海外版》，2001-09-11</ref>；因其刊物多刊登有關[[中共黨史|中共黨史]]敏感事件的評論與時評文章，而文章時與官方的歷史結論有出入，或者主張黨內各項改革，故刊物內容受到關注<ref>[http://www.voachinese.com/content/china-control-media-20130104/1577529.html 炎黄春秋被关闭 南方周末遭插手]，[[美国之音|美国之音]]，2013年1月7日</ref>，2015年《國防參考》刊發[[中國社會科學院|中國社會科學院]][[馬克思主義|馬克思主義]]研究院研究員龔雲撰文文章《「起底」〈炎黃春秋〉》；文章稱，《炎黃春秋》打着一些合法的旗幟，假借客觀公正之名，「對普通民眾特別是離退休幹部具有很大迷惑性和欺騙性」。文章還稱，《炎黃春秋》「抹黑毛澤東，抹黑英烈，虛無歷史，實際上是把新中國的歷史顛倒過去，為把中國拉回資本主義做輿論準備」；此篇文章在同年6月3日曾被[[解放軍報|解放軍報]]微博轉載並因此引發討論<ref>[http://www.zaobao.com/special/zbo/commentary/story20150604-487879 军报炮轰《炎黄春秋》掀波澜]，[[聯合早報|聯合早報]]2015年6月4日</ref><ref>[http://ww3.sinaimg.cn/large/87e90f81jw1esqqt29orqj20c83kkb29.jpg 解放軍報轉載全文]</ref>。

因管理层遭撤换，原制作班底于2016年7月17日宣布《炎黄春秋》停刊，但新的管理层仍继续出刊<ref>[http://www.voachinese.com/a/io-20160718-ynahuang-chunqiu-ending/3422715.html 时事大家谈: 《炎黄春秋》被迫停刊，从此历史皆党音]</ref>。而2016年以前在官方网站上的历史备案也全部遭到删除，仅保存有导读和目录。

==歷史==
=== 2008退休事件 ===
2008年11月，北京知情消息人士传出消息，由于《炎黄春秋》多次刊登涉及前中共中央总书记[[赵紫阳|赵紫阳]]的文章，[[江泽民|江泽民]]担心自己的影响会因此被削弱，甚至被否定，所以引起江泽民不满，要求时任中共中央政治局常委[[李长春|李长春]]严肃处置。故而文化部以杂志相关人员年龄过大劝其退休。但在时任中共中央总书记[[胡锦涛|胡锦涛]]的干预下，当时的社长[[杜导正|杜导正]]继续任职<ref>{{cite web |url = http://china.dwnews.com/news/2012-12-06/59004491.html |title = 维基解密曝江胡因处理《炎黄春秋》叫板 |publisher = 多维新闻网 |date=2012-12-06 |language =zh-hans }}</ref>。该事件引起海内外的广泛的关注，杂志的命运被华人社会担忧。本杂志在2008年刊登的敏感文章有《文革后期我与四川省委书记的交往》、《我看党与国家和人民的关系》、《中国政治体制改革论稿》等。

中国知名作家[[邵燕祥|邵燕祥]]随后写诗表示支持《炎黄春秋》。

=== 网站被封事件 ===
2009年5月22日起，炎黄春秋网站（[http://www.yhcqw.com/ yhcqw.com]）开始无法访问，据推测可能与[[六四事件|六四事件]]20周年临近有关<ref>[http://news.boxun.com/news/gb/china/2009/05/200905242105.shtml 炎黄春秋网站被屏蔽 《炎黄春秋》杂志遭整肃]，博讯网，2009年5月24日</ref>，至5月25日下午网站恢复正常。不久（截止5月30日14:00）该网站的炎黄论坛显示“设置数据不存在，调用已终止”而无法访问。5月31日论坛恢复正常。

2010年11月，[[李普_(记者)|李普]]去世后，炎黄春秋论坛亦不能访问，其介绍说因为技术原因只能周一至周五早9点至晚5点开放<ref>多维历史：[http://history.dwnews.com/news/2013-01-05/59061334-2.html 《炎黄春秋》几度遭禁封]. 多维新闻</ref>。

2013年1月4日9时左右，炎黄春秋网站被关闭。炎黄春秋编辑部在[[新浪微博|新浪微博]]中称，2012年12月31日分别收到署名为“[[工业和信息化部网站备案系统|工业和信息化部网站备案系统]]”的短信和邮件：“您备案信息中的网站《炎黄春秋网》已被注销，该网站的备案号‘京[[ICP|ICP]]备08100492号-1’已被收回。特此通知！”但通知中没有注明网站被突然注销的原因<ref>{{cite web |url = http://www.bbc.co.uk/zhongwen/simp/chinese_news/2013/01/130104_yanhuang_website.shtml |title = 《炎黄春秋》杂志网站突遭关闭 |accessdate = 2013年1月4日 |publisher = BBC中文网 |archiveurl = http://www.bbc.co.uk/zhongwen/simp/chinese_news/2013/01/130104_yanhuang_website.shtml |archivedate = 2013年1月5日 |work = |language = zh-cn}}</ref><ref>{{cite web |url = http://www.chinese.rfi.fr/中国/20130104-党史杂志《炎黄春秋》网站新年伊始遭封禁 |title = 党史杂志《炎黄春秋》网站新年伊始遭封禁 |accessdate = 2013年1月4日  |publisher = 法广中文网|archiveurl = http://www.chinese.rfi.fr/中国/20130104-党史杂志《炎黄春秋》网站新年伊始遭封禁 |archivedate = 2013年1月5日 |work = |language = zh-cn}}</ref>。1月5日，炎黄春秋网站使用一个临时网站暂时恢复上线<ref>{{Cite web |url = http://www.chinese.rfi.fr/中国/20130105-炎黄春秋临时恢复上线 |title= 炎黄春秋临时恢复上线  |work = 法广 |publisher = |date = |accessdate = 2013-01-10 }}</ref>。

=== 更换人事和“停刊声明” ===
2016年7月14日，[[中国艺术研究院|中国艺术研究院]]发布一份《炎黄春秋》人事任免通知，根据通知，《炎黄春秋》杂志重要职务全部为官方派来的人所取代，原社长、总编辑[[杜导正|杜导正]]、副社长[[胡德华_(胡耀邦三子)|胡德华]]（前中共中央总书记[[胡耀邦|胡耀邦]]之子）和总编辑徐庆全悉數被撤换。《炎黄春秋》杂志社随后发表声明：“按照《中国艺术研究院与炎黄春秋杂志社协议书》明文约定，我社有人事任命权、财务自主权和发稿自主权，双方盖章，具有法律效力。我方不同意单方终止协议书，我社已委托律师对该院提起诉讼”。杂志表示拥护以[[习近平|习近平]]为总书记的党中央“依法治国”的方针。<ref>{{cite web |url=http://www.voachinese.com/a/publisher-sacked-yanhuang-chunqiu-20160714/3418643.html |language=中文 |title=改革派杂志遭整肃 从此炎黄无春秋 |publisher=美国之音 |date=2016年7月14日 |accessdate=2016-07-15}}</ref>

随后的7月17日，杂志社执行主编吴伟发出了杂志社社长、法人代表杜导正签名的的“停刊声明”。声明中确认，7月15日，中国艺术研究院派员强行进入杂志社，窃取和修改了《炎黄春秋》官方网站的密码，导致“我刊丧失了基本的编辑出版条件”。声明中同时提到，停刊决定经过炎黄春秋杂志社社委会讨论并一致决定，自即日起任何人以《炎黄春秋》名义发行的出版物，均与“本社”无关。<ref>{{Cite web|url=http://www.voachinese.com/a/news-outspoken-chinese-political-magazine-20160718/3423011.html|title=炎黄春秋停刊玉碎 对簿公堂以求生机|accessdate=2016-07-19}}</ref><ref>{{Cite web|url=http://cn.rfi.fr/中国/20160718-《炎黄春秋》被接管后“玉碎”-社委会宣布停刊|title=《炎黄春秋》被接管后“玉碎” 社委会宣布停刊 - 中国 - RFI|accessdate=2016-07-19|date=2016-07-18|work=法广}}</ref>

《炎黄春秋》委托北京[[莫少平|莫少平]]律师事务所，向北京[[朝阳区_(北京市)|朝阳区]]法院提出[[民事诉讼|民事诉讼]]，朝阳区人民法院于7月28日发出一份裁定书，决定对中国艺术研究院违反合同裁撤管理人员一案不予受理。莫少平律师表示将会在十日内向北京市第三中级人民法院提起[[上诉|上诉]]。<ref >{{cite web|url=http://www.voachinese.com/a/news-chinese-court-rejected-reformist-magazine-law-suit-20160728/3438476.html | language= | title=朝阳法院不予受理 炎黄春秋上诉中院 |publisher=美国之音 |date=2016-07-29 |accessdate= }}</ref>2016年10月，一篇名为《炎黄春秋杂志社被某单位 “鸠占鹊巢” 社会各界纷纷表态》的文章在一些网站上被转载，被认为是国内一些媒体对原来的《炎黄春秋》的声援。<ref>{{cite web |url=http://www.fzyshcn.com/shyf/2016-10-10/17642.html |language= |title=炎黄春秋杂志社被某单位 “鸠占鹊巢” 社会各界纷纷表态 |publisher=法制与社会网 |date=2016-10-10 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20161013160221/http://www.fzyshcn.com/shyf/2016-10-10/17642.html |archivedate=2016-10-13 }}</ref><ref >{{cite web|url=http://www.boxun.com/news/gb/china/2016/10/201610130223.shtml#.V_8ATPlJLGI | language= | title=《炎黄春秋》事件继续发酵 |publisher=博讯网 |date=2016年10月13日 |accessdate= }}</ref>

== 過去受關注之文章 ==
* 2004年第8期所载中國共產黨元老[[任仲夷|任仲夷]]的「任仲夷谈邓小平与广东的改革开放」访问记里，披露了任仲夷对[[邓小平|邓小平]]不足的看法。
* 2005年第1期所刊前[[毛澤東|毛澤東]]秘書李锐的短文《乘風破浪又一年》中道：「……1949年後，卻遺憾地走過了一段大彎路；那時面對一系列嚴重明顯的錯誤，無人敢於站出來反對……但同舊習慣有關的政治、文化等體制的改革，未能與經濟同步，以致[[貪污腐敗|貪污腐敗]]、實利[[犬儒|犬儒]]之風盛行，使人感到我們還沒有從過去歷史的種種迷誤中徹底清醒過來、還未能對歷史進行全面反思。」
* 2005年11月，正值中共在[[六四事件|六四事件]]後首度公開紀念[[胡耀邦|胡耀邦]]誕辰90周年，配合官方與胡之女[[李恒|李恒]]（滿妹）的著作出版，該刊刊載了「我们心中的胡耀邦」專題，請到[[田紀雲|田紀雲]]、李锐、胡之秘書[[曹治雄|曹治雄]]等多位黨內元老與人士對胡之評價。該刊與周刊《[[冰点_(中国青年报)|冰點]]》刊登「我心中之耀邦」一文同被[[中共中央宣傳部|中宣部]]批評。該期並短暫被封禁，後始復售。
* 2007年第2期，前[[中國人民大學|中國人民大學]]副校長[[谢韬|谢韬]]的《民主社会主义模式与中国前途》。文章认为马克思主义的最高成果是民主社会主义，民主社会主义寄托着人类的希望，只有民主社会主义才能救中国。
* 2007年第7期，[[田纪云|田纪云]]《國務院大院的記憶》，對因六四事件而被免职的已故前[[中共中央總書記|中共中央總書記]][[趙紫陽|趙紫陽]]作了正面描述，稱讚趙紫陽在擔任[[國務院總理|國務院總理]]時倡導節儉、反對鋪張浪費的精神。当局禁止《炎黄春秋》在网上发表这篇文章，并不准其他媒体转载。<ref>[http://news.bbc.co.uk/chinese/simp/hi/newsid_6890000/newsid_6898500/6898515.stm 田纪云公开撰文赞扬赵紫阳]，BBC中文网，2007年7月13日</ref>
* 2009年第4期，李锐的《向胡耀邦学习——〈胡耀邦传〉序言》。
* 2009年第5期，彭迪和钱行合写的《“[[五四運動|五四]]”精神　[[普世价值|普世价值]]》，不点名批评[[全国人大常委会委员长|全国人大常委会委员长]][[吴邦国|吴邦国]]的“绝不照搬西方那一套，绝不搞[[政黨輪替|多党轮流执政]]”的说法<ref>彭迪、钱行：[http://www.yhcqw.com/html/qsp/2009/59/ACGC_2.html 《五四精神　普世价值》]，炎黄春秋网</ref>。
* 2009年第12期，周瑞金的《发展起来以后的问题——国庆60周年感言》，总结了当前改革遇到的种种问题，提出了解决这些问题的一些想法。
* 2011年第2期，第3期值前[[中共中央主席|中共中央主席]][[华国锋|华国锋]]诞辰90周年之际，发表了由韩钢撰写的《关于华国锋的若干史实》，厘清了一部分关于华的误解。
* 2011年第7期，因涉及中共建党九十周年敏感内容，在一部分高校[[图书馆|图书馆]]无法借阅。
* 2013年第一期，发表新年献词《宪法是政治体制改革的共识》，呼吁当局将虚置的《[[中华人民共和国宪法|中华人民共和国宪法]]》变成现实的制度体系，将现行一切[[违宪|违宪]]的制度、法令、政策改变过来，使其与宪法一致。并称政治体制改革实质是一场“维宪行动”<ref>[http://newpaper.dahe.cn/hnsb/html/2013-01/04/content_834378.htm 《炎黄春秋》：宪法是政治体制改革的共识]，[[河南商报|河南商报]]，2013年1月4日</ref>。
