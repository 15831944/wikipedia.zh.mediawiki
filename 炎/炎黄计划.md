'''炎黄计划'''是一个对百位[[黄种人|黄种人]]进行[[人类基因组|基因组]]测序的计划，由[[华大基因|深圳华大基因研究院]]、生物信息系统国家工程研究中心与[[中国科学院北京基因组研究所|中国科学院北京基因组研究所]]合作研究。该计划完成了全球首份亚洲人基因组图谱的绘制。

该项目的负责人是中国科学院院士、华大基因研究中心主任[[杨焕明|杨焕明]]。

== 炎黄一号 ==
炎黄计划于2006年启动，计划的第一步“炎黄一号”是对一位中国人进行基因组测序。这是自[[人类基因组计划|人类基因组计划]]1%任务、[[国际人类基因组单体型图计划|国际人类基因组单体型图计划]]10%任务后，中国科学家独自进行的人类基因组测序计划。<ref>{{cite web|url=http://www.hb.xinhuanet.com/health/2010-03/12/content_19235388.htm|title=揭秘"炎黄计划"|publisher=新华网|date=2010-03-12|deadurl=yes|archiveurl=https://web.archive.org/web/20141018202410/http://www.hb.xinhuanet.com/health/2010-03/12/content_19235388.htm|archivedate=2014-10-18}}</ref>

2007年，华大基因研究院宣布绘制完成了首份黄种人基因组图谱，并于2008年在《[[自然_(期刊)|自然]]》杂志上发表了相关研究成果。这一测序数据总量为1177亿[[碱基|碱基]]对，平均测序深度为36倍，有效覆盖率达99.97%，变异检测精度则达99.9%以上。这也是自DNA之父[[詹姆斯·沃森|詹姆斯·沃森]]与基因组研究先驱[[克雷格·文特尔|克雷格·文特尔]]之后进行的全球第三例人类个人基因组测序。<ref>{{cite web|url =http://news.sina.com.cn/c/2008-11-08/024914696719s.shtm|title =“炎黄一号”在《自然》杂志发表|publisher =新浪新闻|date =2008-11-08}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url = http://www.genomics.cn/navigation/show_navigation?nid=1608 |title = 炎黄计划 |publisher = [[华大基因|华大基因]]}}</ref>

== 炎黄99 ==
“炎黄99”是炎黄计划的第二步，旨在于“炎黄一号”后再对99位黄种人进行基因组测序，并构建一个东亚人种的高分辨率遗传多态性图谱，用以促进对东亚人种的医学研究。该计划于2008年启动。<ref>{{cite web |url = http://tech.sina.com.cn/d/2008-01-06/16461955952.shtml |title = 深圳华大基因研究所征战“炎黄99”计划 |publisher=新浪科技 |date=2008-01-06}}</ref>
