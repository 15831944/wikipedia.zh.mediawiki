《'''炎拳'''》（{{lang-ja|ファイアパンチ}}）是一部[[日本|日本]][[少年漫画|少年漫画]]，作者是[[日本漫畫家|日本漫画家]]{{Tsl|ja|藤本タツキ|藤本樹}}，于2016年4月18日在[[集英社|集英社]]的[[網絡漫畫|网络漫画]]发布网站《[[週刊少年Jump的增刊號|少年Jump+]]》上正式开始连载<ref>{{Cite news|url=https://shonenjumpplus.com/episode/10833497643049550135|title=[1話]ファイアパンチ {{!}} 少年ジャンプ＋|newspaper=少年ジャンプ＋|accessdate=2017-02-11|language=ja}}</ref>，每周一更新。

== 概要 ==
藤本樹凭借2013年11月的短篇漫画《戀愛令人盲目》（{{lang-ja|恋は盲目}}）出道，这是他的首部连载漫画作品。

[[宝岛社|宝岛社]]举办的[[這本漫畫真厲害！|这本漫画真厉害！]]2016年9月期男生篇中以222分排名第一<ref>{{Cite news|url=http://konomanga.jp/special/74679-2|title=【9月の「このマンガがすごい！」ランキング オトコ編】第1話がとにかく大反響を呼んだ、あのWEBマンガが単行本化！ オトコ編第1位は……|newspaper=このマンガがすごい！WEB|accessdate=2017-02-11|language=ja}}</ref>。

== 故事简介 ==
自诞生便能运用奇迹的人们被称之为'''祝福者。'''正是因为一位名为冰之魔女的祝福者，这个世界被冰雪、饥饿和疯狂覆盖。冻僵的人们祈求着火焰。

== 登场人物 ==
; {{nihongo|阿格尼|アグニ}}
:男主角，再生能力者，原本與妹妹生活再一普通村莊，利用自己的再生能力砍下手臂給缺乏糧食的村民吃，在貝西姆德魯克軍隊來「借」糧食時，被當時的指揮官德瑪用不滅的火焰焚燒全身，卻因為過於強大的再生能力而保持的烈火焚身的痛楚，完本想放棄再生的他因妹妹斷氣前鼓勵他活下去，之後便為了復仇而生。

; {{nihongo|露哪|ルナ}}
:主角的妹妹，再生能力者，與哥哥生活再一普通村莊，非常愛慕著哥哥，甚至有了與哥哥「造小孩」的想法，在德瑪焚燒村莊時因再生能力較弱而成為焦炭死亡。

; {{nihongo|桑|サン}}
:意做太陽，主角在成為火人後的第八年碰巧救的一個男孩，因長相極似女生所以常常被誤認，性格單純、樂觀，稱呼阿格尼為神大人，周圍的人也會被其個性影響而擁有希望。

; {{nihongo|奈奈特|ネネト}}
:桑被軍隊捉住時一起被關進房間的少女，黑髮，頗機靈，之後在火車上被利賀田搭救，代價是當攝影小妹

; {{nihongo|利賀田|トガタ}}
:活了300歲的超強再生能力者，唯一一個看過冰河時期前的人類社會的人，雖然身體是女性卻非常嚮往男人，曾做過變性手術，卻因再生能力而失敗，夢想是拍出一部好電影，徒手戰鬥能力極強。
