《'''炎炎消防隊'''》（{{lang-ja|炎炎ノ消防隊}}），是[[日本|日本]][[漫畫家|漫畫家]][[大久保篤|大久保篤]]繪畫的[[少年漫畫|少年漫畫]]作品，於《[[週刊少年Magazine|週刊少年Magazine]]》2015年43號開始連載<ref>{{Cite web|url=http://natalie.mu/comic/news/160814|script-title=ja:「ソウルイーター」の大久保篤、マガジンで新連載「炎炎ノ消防隊」始動|accessdate=2015-09-23|publisher=[[Natalie_(網站)|Comic Natalie]]|language=ja|date=2015-09-23}}</ref>。

== 故事簡介 ==
描述在一個有人體自燃現象（在此稱為焰人）的世界，原國樣貌大變，進而將國名改為東京皇國。因為焰人的出現會攻擊人類，因此成立了特殊消防隊，（至今共八大隊）由聖陽教教會、灰島重工及國軍協辦。然而，在這樣的世界，「惡魔的足跡」的誕生，將掀起人類的戰爭，究竟特種消防隊、反派的灰焰騎士團誰會勝出呢？森羅又如何解開人體自燃之謎，當拯救世界的真正英雄？人體自燃之謎又是什麼？一部讓人熱血沸騰的英雄故事，就此展開。

== 登場角色 ==
本作中的所有角色，通常是名字在前，姓氏在後。

=== 灰焰騎士團 ===
; {{nihongo|傳導者|伝導者（でんどうしゃ）}}

; {{nihongo|象 日下部|象 日下部（ショウ・クサカベ）}}
: 灰焰騎士團團長、第三柱。第三世代能力者，受「傳導者」的加持下能干涉所有事物。森羅的弟弟，在十二年前以為他(一歲)跟母親死於火災當中，但在消防員新人大會中被神秘男子告知他還活著。在於地下基地對上森羅，取回記憶與情感，但又受蒿梅雅影響。

; {{nihongo|烈火 星宮|烈火 星宮(レッカ・ホシミヤ)|聲優：[[関智一|関智一]]<ref>{{Cite news|publisher=[[アニメイト|アニメイト]]|newspaper=[[アニメイトタイムズ|アニメイトタイムズ]]|url=https://www.animatetimes.com/news/details.php?id=1552380339|title=TVアニメ『炎炎ノ消防隊』第1特殊消防隊の中隊長 烈火星宮役が関智一さんに決定|date=2019-03-12|accessdate=2019-03-12}}</ref>}}
: 第一大隊中隊長·神父。瞳孔特徵為星星，個性熱血。以拳發出火焰的第三世代能力者。實際為灰焰騎士團一員，並將「蟲」植入正常人變成焰人，尋找能承受「蟲」不變為焰人的「適合者」。在森羅打敗並打算燒光時，被卡力姆的能力活體冰凍，但是被突如的一道光貫穿胸膛逝去。

; {{nihongo|亞羅|アロー}}
: 第三世代能力者，視力很好的弓箭手。瞳孔特徵箭頭。曾被火繩打敗。象的守護者。搶奪第六柱時，暫時為蒿梅雅的守護者保護她。

; {{nihongo|約拿|ヨナ}}
: 能將人的外貌做改變。瞳孔特徵超粗圓體Ａ字，有著鋒利的鼻子，連紙都能割開。自己的臉似乎也加工過。似乎不是人類。在250年前偽裝聖陽教教祖拉夫魯斯一世。

; {{nihongo|哈蘭|ハラン}}
: 跟著亞羅、約拿行動，當被亞瑟逼到絕境，食用蟲變成「鬼」。被紅丸解決。

; {{nihongo|福萊魯|フレイル}}
: 偽裝第三特殊消防隊隊員，Dr.喬凡尼指示殺了伏爾甘。被茉希打敗。

; {{nihongo|米拉奇|ミラージュ}}
: 第三世代能力者。能力「陽炎」，發出炎熱的霧氣，甚至能使出幻覺。

; {{nihongo|菲拉|フィーラー}}
: 假名{{nihongo|麗莎|リサ}}。曾是無能力者，但受到蟲而覺醒。能力「漁之火」，像章魚腳的火焰。
: 由焰人引起的火災失去父母，差點進修道院。被Dr.喬凡尼扶養、教導控制火焰、接近伏爾甘。在伏爾甘的廢料場休息時，被他收留。儘管很想到伏爾甘身邊，但受到Dr.喬凡尼的洗腦無法輕易離開。目前被第八收留，但精神衰落嚴重。

; {{nihongo|阿薩魯特|アサルト}}
: 第三世代能力者，能變化外觀。對上環，被她的體質影響難以攻擊時，先被愛麗絲打後又被環與愛麗絲打暈。

; {{nihongo|蒿梅雅|ハウメア}}
: 擁有亞多拉爆裂之火的第二柱。第三世代能力者，操控電漿。間接引起日下部家火災，讓變成鬼的母親把嬰兒的象帶走。

; {{nihongo|卡龍|カロン}}
: 第二世代能力者，將受到攻擊的運動能量蓄積於體內轉化熱能量再釋放攻擊。蒿梅雅的守護者，從蒿梅雅還小就一起行動。每次都會頻繁回問確認，讓蒿梅雅生氣。在哪吒失控後，保護了差點被攻擊的身為人柱的森羅。

; {{nihongo|因果 春日谷|因果 春日谷（インカ　カスガタニ）}}
: 能看見火焰的流向，氣味化為的線條。並且揮動指頭的地方就會有火焰。由第一柱預測的第五人柱，被消防隊與騎士團盯上，但是選擇了能給予自己危險刺激的騎士團。等著森羅主動來找她。

; {{nihongo|律|}}
: 能將死人變為焰人操控。因果的守護者。

; {{nihongo|第一柱|一柱目(ひとはしらめ)}}
: 亞多拉連結人柱之一。似乎為『天照』的人柱。長得像愛麗絲的女人，討厭人類，利用連結操控森羅。

=== 灰島重工 ===
; {{nihongo|葛雷歐 灰島||}}
: 灰島重工社長，李希特的上司。

; {{nihongo|哪吒 孫||}}
: 由烈火所植入「蟲」而覺醒的孩子，為灰島重工的實驗。並且覺醒亞多拉連結，灰焰騎士團的因果與卡龍準備帶走的第六柱。

; {{nihongo|黑野||}}
: 目前為哪吒的訓練對象，自稱死神叔叔。被稱為「最狂」能力者，整個右手患有灰病而碳化。喜歡虐待小孩、欺負弱者。在森羅小時候對過他。

== 用語解說 ==
; {{nihongo|[[人體自燃現象|人體自燃現象]]|人体発火現象(じんたいはっかげんしょう)}}
: 人體不明原因突然燃燒起來。

; {{nihongo|特殊消防隊|とくしゅしょうぼうたい|FIRE FORCE}}
: 由聖陽教會、東京軍隊、消防廳招募人員組成。

; {{nihongo|焰人／第一世代能力者|焔ビト(ほむらびと) / 第一世代能力者(だいいちせだいのうりょくしゃ)}}
: 會起火並失去自我直到生命盡頭都保持狂暴狀態的被害者為「第一世代」，通常稱為「焰人」。有意識的「焰人」為少數。

; {{nihongo|第二世代能力者|だいにせだいのうりょくしゃ|}}
: 覺醒並會控制火焰，但自身不會起火可以控制火焰、防禦火焰。

; {{nihongo|第三世代能力者|だいさんせだいのうりょくしゃ|}}
: 覺醒並會控制火焰，不但能自身起火還能自由控制火焰。

; {{nihongo|特殊裝甲消防車||}}
: 通稱「火柴盒」。

; {{nihongo|聖陽教|せいようきょう|}}
: 世界的大宗教。由宗教的修女為「焰人」進行祈禱鎮魂。

; {{nihongo|119|ワンワンニャイン|}}
: 消防廳的吉祥物。由長得像狗的大叔‧小守(まもる君)、狗‧雷斯(レス君)以及貓‧小Q(Qちゃん)組合。原本是兩狗一貓，但消防廳高層抱怨：「隊員全部都由動物來表示這樣好嗎」，於是其中一隻就變成長得像狗的大叔。在女學生間很有人氣，環覺得小守很可愛。

; {{nihongo|蟲|むし|}}
: 引起「人造焰人」的謎之蟲。

; {{nihongo|起火極限|オーバーヒート/発火限界|}}
: 能力使用過頭，用盡起火源頭(體內氧氣)而會頭暈。

; {{nihongo|久遠式火力發電「天照」|久遠式火力発電｢天照｣(くおんしきかりょくはつでん アマテラス)}}
: 隨著太陽曆的開始由聖陽教與灰島重工所建造的大型發電所。以亞多拉爆裂為動力供給全東京皇國全部能量。但據伏爾甘所說那是他們一族所製造的。而在中華半島上也有類似的天照，但比較小。稱『御神體』，由一位穿著斗篷的女性修建。

; {{nihongo|鬼|}}
: 有角的焰人。

; {{nihongo|亞多拉連結|アドラバースト|}}
: 在第三世代中能力者非常罕見，被認為是只有太陽神的後人才擁有的指引人類的聖火，不摻雜任何物質的純粹火焰。被稱為太陽神在創造時的火種，即「原始火焰」。但也有稱「亡滅之炎」。

; {{nihongo|亞多拉爆裂之火|}}
: 異世界火焰，能干涉一切事物的火焰。覺醒後稱為「第四世代」。
: 共有八位人柱：第一柱(1)、蒿梅雅(2)、象(3)、森羅(4)、因果(5)、哪吒(6)

; {{nihongo|亞多拉|}}
: 象所言為異世界，地獄。
