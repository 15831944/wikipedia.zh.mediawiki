'''葛瑞姆·波爾'''（{{lang-en|Graham Poll}}，{{bd|1963年|7月29日}}）是已經退役的[[英格蘭足球超級聯賽|英格蘭足球超級聯賽]][[足球|足球]][[裁判_(足球)|裁判]]，普遍被認為是[[國際足球歷史和統計聯合會|國際足球歷史和統計聯合會]]所維護的裁判名錄中，近二十五年來最偉大的英格蘭籍裁判<ref>{{cite web |title=IFFHS 波爾曾執法[[2004–05年歐洲足協盃|2005年歐洲足協盃]]決賽，也是兩次世界盃和[[2000年歐洲國家盃|2000年歐洲國家盃]]的英格蘭籍裁判代表。

[[2006年世界盃|2006年世界盃]]，波爾負責執法[[2006年世界盃足球賽決賽週_-_F組#克羅埃西亞對澳洲|克羅埃西亞對澳洲]]的分組賽，在給了[[約瑟普·希穆尼奇|約瑟普·希穆尼奇]]第二張黃牌後卻未將希穆尼奇驅逐出場，後來發現後才再補了希穆尼奇一張黃牌，成為足壇史上著名的「三張黃牌」大烏龍<ref>{{cite news |url=https://www.theguardian.com/football/2006/jun/24/worldcup2006.sport4 |work=The Guardian |author=Biggs, Alan/Kelso, Paul |title=Poll's career on the line after Stuttgart debacle |date=2006-06-24 |accessdate=2008-07-23 |location=London |language=en |archiveurl=https://web.archive.org/web/20140112220355/http://www.theguardian.com/football/2006/jun/24/worldcup2006.sport4 |archivedate=2014-01-12 |deadurl=no }}</ref>。波爾在這個重大的執法失誤發生後不久便宣布從國際賽決賽退役。他後來繼續在英超聯賽、英冠聯賽和國際賽中執法，但表示他不會再接受於任何國際錦標賽決賽中被提名代表英超聯賽，因為他認為自己已經有過機會<ref name="retirement">{{cite news | title = Poll retires from international game | url = http://football.guardian.co.uk/worldcup2006/story/0,,1809120,00.html | publisher = [[The_Guardian|The Guardian]] | date = 2006-06-29 | accessdate = 2006-06-29 | location = London | first = Tom | last = Lutz | language = en | archiveurl = https://web.archive.org/web/20071119020103/http://football.guardian.co.uk/worldcup2006/story/0,,1809120,00.html | archivedate = 2007-11-19 | deadurl = no }}</ref>。

== 執法生涯 ==
波爾在1980年從[[英格蘭足球依斯米安聯賽|依斯米安聯賽]]開始他的足球執法生涯，後於1986年在[[英格蘭足球聯賽|英格蘭足球聯賽]]擔任[[助理裁判|助理裁判]]。在累積了五年吹判經驗後，波爾晉升為英格蘭足球聯賽的裁判，並隨後在1993年進入[[英格蘭足球超級聯賽|英格蘭足球超級聯賽]]執法。<ref>{{cite web |url = http://www.football-league.premiumtv.co.uk/page/RefereeProfilesDetail/0,,10794~628215,00.html |title = Profile |publisher = [[the_Football_League|the Football League]] official website |accessdate = 2008-02-04 |archiveurl = https://web.archive.org/web/20080417072957/http://www.football-league.premiumtv.co.uk/page/RefereeProfilesDetail/0,,10794~628215,00.html |archivedate = 2008-04-17 |deadurl = yes }}</ref>

波爾在1996年被授予[[國際足總|國際足總]]徽章，其後開始執法一些歐洲的賽事，其中最重要的賽事是2005年[[欧足联欧洲联赛|歐聯]]決賽，由[[莫斯科中央陸軍足球俱樂部|莫斯科中央陸軍]]出戰[[葡萄牙體育俱樂部|里斯本體育]]。他也執法過許多國際賽事，包括國際足總和[[歐洲足球總會聯盟|歐洲足聯]]的頂級賽事，如[[2000年歐洲國家盃|2000年歐洲國家盃]]分組賽，[[法國國家足球隊|法國]]對[[捷克國家足球隊|捷克]]和[[挪威國家足球隊|挪威]]對[[斯洛維尼亞國家足球隊|斯洛維尼亞]]的比賽。波爾於2007年5月13日執法了生涯最後一場英超賽事，由[[樸茨茅斯足球俱樂部|樸茨茅斯]]出戰[[兵工廠足球俱樂部|兵工廠]]，他在比賽中做出了一個備受爭議、但正確的判決，將[[尼高·卡蘭積卡|尼高·卡蘭積卡]]的進球沒收，改判為[[越位|越位]]。

波爾的裁判生涯的最後一場執法賽事原定是2007年6月6日的[[2008年歐洲足球錦標賽外圍賽|2008年歐洲足球錦標賽外圍賽]]<ref>{{cite web |url = http://www.uefa.com/competitions/euro/fixturesresults/round=2241/match=83778/report=lu.html |title = Finland 2 - 0 Belgium |language = en |accessdate = 2007-06-06 |publisher = [[UEFA|UEFA.com]] |archiveurl = https://web.archive.org/web/20080126123945/http://www.uefa.com/competitions/euro/fixturesresults/round%3D2241/match%3D83778/report%3Dlu.html |archivedate = 2008-01-26 |deadurl = no }}</ref>。但後來波爾被發現他接受了一場訪談，在其中指責當裁判被球隊經理和教練批評時，聯賽高層沒有為裁判挺身而出<ref>{{cite web |url = http://news.bbc.co.uk/sport2/hi/football/6703221.stm |title = Interview given in May 2007 |publisher = [[BBC.co.uk|BBC.co.uk]] |date = 2007-05-30 |accessdate = 2007-05-31 |archiveurl = https://web.archive.org/web/20161028084837/http://news.bbc.co.uk/sport2/hi/football/6703221.stm |archivedate = 2016-10-28 |deadurl = no }}</ref>。這場比賽後來改由{{link-en|麥克·萊里|Mike Riley (referee)}}吹判。2007年8月，波爾出版了他的自傳《見紅》（{{lang-en|Seeing Red}}），逐漸將重心轉移到[[大眾媒體|媒體]]工作，在[[BBC體育|BBC體育]]的足球單元擔任[[時事評論員|時事評論員]]，並在《[[每日郵報|每日郵報]]》擔任[[專欄作家|專欄作家]]<ref>{{cite web |url = https://archive.is/20120723112143/http://www.dailymail.co.uk/pages/columnists/archive.html?in_page_id=1951&in_author_id=428&in_article_id=488621 |title = "The Official Line" |publisher = [[Daily_Mail|Daily Mail]] |accessdate = 2008-02-03 |language=en}}</ref>。
