'''葛庭燧'''（{{bd|1913年|5月3日|2000年|4月29日|catIdx=G葛}}），出生于[[山东省|山东省]][[蓬莱县|蓬莱县]]的一个农民家庭<ref>{{cite web |url = http://www.jsxx.org/93/html/mzykx/democracy99_5/index.htm |title = 科学无国界　科学家祖国 |publisher = 九三学社马鞍山市委员会网站，来源：民主与科学，双月刊（1999年第5期，总第60期） |author = 葛庭燧 |date = 1999年10月15日（民主与科学出版） |accessdate = 2012-06-19 |language = zh-hans |deadurl = yes |archiveurl = https://web.archive.org/web/20061110184012/http://www.jsxx.org/93/html/mzykx/democracy99_5/index.htm |archivedate = 2006年11月10日 }}</ref>，[[金属物理学|金属物理学]]家，[[中国科学院院士|中国科学院院士]]。[[九三学社|九三学社]]中央委员会顾问。

1937年获[[清华大学|清华大学]]理学士学位。1943年获[[美国|美国]][[加州大学伯克利分校|加州大学伯克利分校]]物理学博士学位。

== 贡献 ==
首创“[[葛氏扭摆|葛氏扭摆]]”，首先发现晶界内耗峰（[[葛峰|葛峰]]）。
