'''葛瑞格·布萊克'''（{{lang-en|'''Greg Bryk'''}}，本名'''Gregory Michael Bryk'''，{{bd|1972年|}}）是一名[[加拿大|加拿大]]電影和電視劇演員。他曾在多部電影中演出，而他最出名的演出是在三部不同的電視劇——《''{{link-en|ReGenesis|ReGenesis}}''》、《[[殺手十三|殺手十三]]》和《[[狼女_(電視劇)|狼女]]》。他也出名在2018年電子遊戲《[[極地戰嚎5|極地戰嚎5]]》中聲演主要反派Joseph Seed<ref>{{Cite web |url = https://www.dualshockers.com/far-cry-5-father-edition-greg-bryk/ |title = Unbox Far Cry 5: Father Edition with the Actual Father, Greg Bryk |author = Emily Hobbs |publisher = DualShockers |language = en}}</ref>，還有在續集《[[極地戰嚎：破曉|極地戰嚎：破曉]]》中重複扮演Joseph Seed。
