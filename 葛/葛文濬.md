'''葛文濬'''（{{bd|？||？|}}）号'''慕川'''，[[浙江省|浙江省]][[秀水县|秀水县]]（今[[嘉兴市|嘉兴市]]）人<ref name=yiyun/>，[[清朝|清朝]]及[[中华民国|中华民国]]教育家，[[清室善后委员会|清室善后委员会]]委员。

==生平==
[[清朝|清朝]]戊戌年（1898年）春，葛文濬在嘉兴创办了一所名为“毓秀”的学堂，收亲友子弟入学，课程有中文、英文、数学等。同年，因[[戊戌政变|戊戌政变]]终止[[戊戌变法|戊戌变法]]而停办。[[侯官|侯官]][[方雨亭|方雨亭]]任嘉兴县知县后，与葛文濬商议筹设秀水学堂，此为近代嘉兴县第一所公立学校<ref name=yiyun>{{Cite web |url = http://dushu.qq.com/read.html?bid=928502&cid=10 |title =亦云回忆  |author = 沈亦云 |date =  |publisher = 腾讯 |language =  |accessdate =2018-08-15  |quote = 第10章 我的父亲 }}</ref>。[[光绪|光绪]]三十四年（1908年），嘉兴商会会长[[褚辅成|褚辅成]]，适应[[清末新政|清末新政]]需要，召集葛文濬（葛慕川）、[[张右企|张右企]]、[[沈稚岩|沈稚岩]]、[[祝心梅|祝心梅]]、[[朱仙槎|朱仙槎]]，选出董事16人，成立[[育婴堂|育婴堂]]董事会，当选者全是地方公正士绅，并定每月26日开常会，讨论救婴养老事宜，历任总理的是葛文濬（葛慕川）、沈稚岩、[[沈子兰|沈子兰]]、[[高叔荃|高叔荃]]、[[陈米山|陈米山]]，直到[[南京国民政府|南京国民政府]]成立前<ref>{{Cite web |url =  http://txcs.tx.gov.cn/Article/ShowArticle.asp?ArticleID=108|title =嘉兴历史上的慈善事业  |author =何释如  |date =2010-7-19 |publisher =桐乡市慈善总会  |language =  |accessdate =  |quote =  }}</ref>。1924年，[[清室善后委员会|清室善后委员会]]成立，葛文濬任委员<ref name=tuandui>{{Cite web |url = http://www.dpm.org.cn/about/history_leader.html |title =  历任团队|author =  |date =  |publisher =故宫博物院  |language =  |accessdate = 2018-08-15 |quote =  }}</ref>。

==家庭==
