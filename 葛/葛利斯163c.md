'''葛利斯163c'''（'''Gliese 163 c''' 或 '''Gl 163 c'''，{{IPAc-en|icon|ˈ|ɡ|l|iː|z|ə}}）是一顆可能[[行星適居性|適合居住]]的[[太陽系外行星|太陽系外行星]]<ref name="PHL-20120829">{{cite web |last=Méndez |first=Abel |title=A Hot Potential Habitable Exoplanet around Gliese 163 |url=http://phl.upr.edu/press-releases/ahotpotentialhabitableexoplanetaroundgliese163 |date=August 29, 2012 |publisher=[[阿雷西博波多黎各大學|University of Puerto Rico at Arecibo]] (Planetary Habitability Laboratory) |accessdate=September 20, 2012 }}</ref><ref name="Space-20120920">{{cite web |author=Nola Taylor Redd |title=Newfound Alien Planet a Top Contender to Host Life |url=http://www.space.com/17684-alien-planet-gliese-163c-extraterrestrial-life.html |date=September 20, 2012 |publisher=[[Space.com|Space.com]] |accessdate=September 20, 2012 }}</ref>，母恆星是[[紅矮星|紅矮星]][[葛利斯163|葛利斯163]]<ref name="Simbad-20120920">{{cite web |authors=Staff |title=LHS 188 -- High proper-motion Star |url=http://simbad.u-strasbg.fr/simbad/sim-id?Ident=HIP+19394 |date=September 20, 2012 |publisher=[[Centre_de_données_astronomiques_de_Strasbourg|Centre de données astronomiques de Strasbourg]] (Strasbourg astronomical Data Center) |accessdate=September 20, 2012 }}</ref>。母恆星與地球的距離大約是49光年或15秒差距，位於[[劍魚座|劍魚座]]。葛利斯163c是行星系統三顆行星的其中一顆，質量至少是地球的6.9倍<ref name="PHL-20120829" /><ref name="Space-20120920" />，因此被分類為[[超級地球|超級地球]]（2.5到10倍地球質量類地行星）<ref name="PHL-20120829" /><ref name="Space-20120920" />。
