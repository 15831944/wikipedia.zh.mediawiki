'''葛琴'''（{{bd|1907年|12月24日|1995年|1月3日||catIdx=G葛}}），女，出生于[[江苏省|江苏省]][[宜兴|宜兴]][[丁蜀镇|丁蜀镇]]，作家，[[左联|左联]]成员，先后写有50万字的作品。父亲葛沐春。<ref>{{Cite web |url=http://www.jnwb.net/wlwx/2013/0801/65953.shtml |title= 从战火中走出的无锡女作家 |author=  |date=2013年8月1日 |publisher= 江南晚报 |language= zh-cn |accessdate= 2014年5月8日 }}</ref>

==生平==
* 1921年，考取[[苏州乐益女子中学|苏州乐益女子中学]]，毕业后升入[[上海景贤女子中学|上海景贤女子中学]]，由在该校任教[[张闻天|张闻天]]的介绍，她进入上海大学，并在工人夜校兼职教课。
* 1926年11月，葛琴由王茀仑、郑普秋介绍，参加了中国共产党。<ref>{{Cite web |url=http://www.azcolabs.com/jnhy_hg_qlgq.html |title=一段尘封的往事 |author= 葛小鹰 |date=2013年4月7日 |publisher= |language= zh-cn |accessdate= 2014年5月8日 }}</ref>
* 1927年，参加了上海工人第三次武装起义，失败后以小学教员作为掩护，担任党中央宣传部的机要秘书，常与[[刘少奇|刘少奇]]等中央领导人进行联系。
* 1927年5月，葛琴与[[华岗|华岗]]同居，婚后育有一女(1928年3月)，一子(1931年9月)，名叫华贻芳。<ref>{{Cite web |url=http://xmwb.xinmin.cn/html/2012-07/19/content_40_1.htm |title=革命“三剑客”之一葛琴 |author=丁言昭 |date=2012年7月19日 |publisher=上海滩 |language= zh-cn |accessdate= 2014年5月8日 }}</ref>
* 1932年，在[[丁玲|丁玲]]主编《[[北斗_(雜誌)|北斗]]》上发表处女作——小说《总退却》，此后，又写了《一天》《蓝牛》《犯》《路》《罗警长》《枇杷》等短篇小说，都反映了社会底层人民的苦难和他们的抗争。
* 1932年，华岗被捕入狱。
* 1937年3月，出版了短篇小说集《总退却》，由[[鲁迅|鲁迅]]修改并作序。
* 1938年，葛琴与[[邵荃麟|邵荃麟]]结婚。
* 1943年，生子葛小鹰。
* “文革”前，葛琴是[[北京电影制片厂|北京电影制片厂]]的艺术副厂长。
* 1968年3月，葛琴被隔离审查，中央专案组“一办”立案，罪名之一是“叛徒”，之二是写小说《贵宾》攻击污蔑[[江青|江青]]，后长期遭受迫害导致偏瘫。
* 1995年1月3日，葛琴在[[北京友谊医院|北京友谊医院]]病逝。
