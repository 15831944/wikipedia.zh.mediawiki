'''葛雷格·克羅哈曼'''（{{lang-en|Greg Kroah-Hartman}}，姓名縮寫為GKH）， [[Linux核心|Linux核心]]開發者，{{As of|2013|04|alt=目前}} 為 Linux 核心中穩定分支（<code>-stable</code>）的維護者<ref name="MAINTAINERS">{{cite web|url=http://git.kernel.org/?p=linux/kernel/git/torvalds/linux-2.6.git;a=blob;f=MAINTAINERS|title=Linux kernel Maintainers file|publisher=git.kernel.org|date=|accessdate=2013-04-14}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，他也是staging 子系統<ref name="MAINTAINERS" />、[[Universal_Serial_Bus|USB]]<ref name="MAINTAINERS" />driver core、[[debugfs_(Linux_kernel)|debugfs]]、[[kref|kref]]、[[kobject|kobject]]、[[sysfs|sysfs]] kernel 子系統<ref name="MAINTAINERS" />、 [[tty_(Unix)|TTY]] layer <ref name="MAINTAINERS" />、[[linux-hotplug|linux-hotplug]]、[[User_space|Userspace]] [[Input/output|I/O]]（與 Hans J. Koch 共同維護）等專案的維護者<ref name="MAINTAINERS" />，也創立了[[udev|udev]]專案。除此之外，他亦協助維護[[Gentoo_Linux|Gentoo Linux]]中上述程式及 kernel 的套件。

==生平==
{{as of|2012|02|01|alt=在 2012 年 2 月 1 日以前}}，他在 [[Novell|Novell]] 的 [[SUSE_Linux_distributions|SUSE]] Labs工作，而後效力於[[Linux基金會|Linux基金會]]<ref name="LiFo" /><ref>{{cite web | url=http://www.kroah.com/log/linux/what_greg_does.html | title=What Greg Does | last=KH | first=Greg | date=2012-02-20 | accessdate=2013-04-14}}</ref>，全心致力於[[Linux驅動程式專案|Linux驅動程式專案]]上<ref>{{cite web|author=Greg Kroah-Hartman |url=http://www.kroah.com/log/linux/linux_driver_project_kickoff.html |title=Linux Driver Project Kickoff |publisher=Kroah.com |date=2007-09-27 |accessdate=2009-06-20}}</ref>。

他是《[[Linux_驅動程式|Linux 驅動程式]]》<ref>{{cite web|url=http://www.oreilly.com/catalog/linuxdrive3 |title=O'reilly: Linux Device Drivers, Third Edition |publisher=Oreilly.com |date= |accessdate=2009-06-20}}</ref> 的共同作者，與Linux Kernel in a Nutshell<ref>{{cite web|url=http://www.oreilly.com/catalog/9780596100797/ |title=O'reilly: Linux Kernel in a Nutshell |publisher=Oreilly.com |date= |accessdate=2009-06-20}}</ref> 的作者。他是《Linux Journal》雜誌的編輯之一，在電腦新聞網站[[LWN.net|LWN.net]]上也發表了許多文章。

克羅哈曼經常透過發表演說的方式，為核心與驅動程式開發的細節留下文件<ref>{{cite web|author=Greg Kroah-Hartman |url=http://www.kroah.com/log/linux/ols_2006_keynote.html |title=Linux Symposium: Keynote by Greg Kroah-Hartman, myths, lies, and truths about Linux kernel development |publisher=Kroah.com |date=2006-07-23 |accessdate=2009-06-20}}</ref><ref>{{cite web|url=http://conferences.oreillynet.com/cs/os2006/view/e_sess/8387 |title=O'reilly Net: Current State of the Linux Kernel |publisher=Conferences.oreillynet.com |date= |accessdate=2009-06-20}}</ref>與教學<ref>{{cite web|url=http://conferences.oreillynet.com/cs/os2006/view/e_sess/8388 |title=O'Reilly Net: Write A Real, Working Linux Driver |publisher=Conferences.oreillynet.com |date= |accessdate=2009-06-20}}</ref><ref>[http://elinux.org/images/4/46/Celf_2006_tutorial.pdf Linux Symposium: Write a real, working Linux driver tutorial]</ref>。在 2006 年，他發表了一份[[CD_映像檔|CD 映像檔]]，內容為開發 Linux 設備驅動程式的入門教材。<ref>{{cite web|url=http://ftp.be.debian.org/pub/linux/kernel/people/gregkh/ddk/ |title=Linux DDK |publisher=Debian.org |date=2006-05-24 |accessdate=2012-04-05}}</ref>
==事迹==
2010年2月3日，Linux内核开发者Greg Kroah-Hartman将Android的驱动程序从Linux内核“状态树”（“staging tree”）上除去。
