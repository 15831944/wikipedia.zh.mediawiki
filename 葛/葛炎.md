'''葛炎'''（{{bd|1997年|8月23日}}，綽號'''小孩游神'''），中国大陆《[[英雄联盟|英雄联盟]]》IG战队前adc选手。<ref>{{cite web |url=http://www.52pk.com/gl/6446710.shtml |title=《英雄联盟LOL》IG.kid小孩游神介绍 |last= |first= |date=2015-06-15 |website= |publisher=52PK原创 |access-date= |quote=}}</ref> 現已宣佈退役。<ref>{{Cite web|url=https://www.weibo.com/ttarticle/p/show?id=2309404215412584776581|title=iGKid断开连接|author=小孩Kiddddd|date=2018-03-08|work=|publisher=選手本人微博文章}}</ref>

==名称由来==

葛炎的绰号是“小孩游神”，这个名称来自于中国大陆服务器电信一区（艾欧尼亚区）的ID“小孩游神”。因此被玩家们称为“小孩游神”。<ref>{{Cite web |url=http://lol.mingrent.com/star/1539.html |title=小孩游神-葛炎 |accessdate=2016-06-24 |archiveurl=https://web.archive.org/web/20160701181915/http://lol.mingrent.com/star/1539.html |archivedate=2016-07-01 |deadurl=yes }}</ref>

==早年经历==

葛炎出生于1997年8月23日，毕业于[[河南省|河南省]][[焦作市|焦作市]]18中，经常在万方亭附近的网吧上网。在英雄联盟[[中国大陆|中国大陆]]地区内测时，他加入了《[[英雄联盟|英雄联盟]]》，<ref>{{Cite web |url = http://lol.sgamer.com/news/201403/166492.html |title = 大神专访：国服三大腿之一 IG小孩游神 |author = 萦绕千千岁@DW |date = 2014年3月29日 |publisher = 超级玩家 |language = zh-cn }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>一开始在汉宫战队担任ADC位置，在2012TGA上，他击败了WE中单选手[[若风|若风]]，自此成名。<ref>{{Cite web |url = http://www.qushiw.com/lolziliao/20154.html |title = IG小孩介绍 小孩游神 葛炎个人资料 |author = 命。 |date = 2014年12月24日 |publisher = 趣视网 |language = zh-cn }}</ref>后来又被[[Invictus_Gaming|IG]]管理联系，从而加入IG。

==争议==
2015年6月13日，LPL职业联赛IG对阵OMG的比赛中，葛炎所使用的薇恩只打了443点输出，创下职业比赛ADC史上新低输出纪录。<ref>[http://lol.kuai8.com/news/95907.html LOL小孩443事件原因]</ref>

2015年10月，葛炎在直播中称S5比赛失利的原因与韩援打野Kakao和中单Rookie有关并声称李炫君（圣枪哥）也不行，结果与网友产生语音冲突。<ref>[http://news.candou.com/616511.shtml IG小孩甩锅Kakao和Rookie惹众怒 直播狡辩LOLS5失利]</ref>

==趣闻==

在IG的一次赛后讨论最后，葛炎说了一句“别说了，先点菜吧”，因此产生了“IG点菜梗”这种说法。<ref>{{Cite web |url = http://www.91danji.com/news/87639.html |title = 英雄联盟扫盲那些传说中的梗之小孩 别说了先点菜吧 |author = 贝伦卡斯泰露 |date = 2014年10月23日 |publisher = 91单机游戏网 |language = zh-cn }}</ref>

在443事件之前，IG的一位名叫“射可可”的辅助也在一次比赛中使用锤石创造了辅助单场输出最低的556输出数据，而这个数据加上葛炎的443相加就是999，因此他和射可可这对组合被玩家戏称为“999皮炎平”。<ref>{{Cite web |url = http://www.52pk.com/gl/6446468.shtml |title = 《英雄联盟LOL》小孩和射可可999皮炎平是什么梗 |author = FT |date = 2016年6月15日 |publisher = 52PK游戏网 |language = zh-cn }}</ref>
