'''芘'''（{{lang-en|pyrene}}，读音bǐ，[[Unicode|Unicode]]代码8298）是一種四環[[多環芳香烃|多環芳香烃]]類，分子式为C<sub>16</sub>H<sub>10</sub>，分子量202.26。

== 物理性质 ==
外观为淡黄色棱状[[晶体|晶体]]。熔点151.2[[摄氏度|°C]]，沸点404°C，密度1.271g/cm<sup>3</sup>(23°C)。不溶于[[水|水]]，溶于[[乙醇|乙醇]]和[[乙醚|乙醚]]。

== 化学性质 ==
可以发生[[卤化|卤化]]、[[硝化|硝化]]、[[磺化|磺化]]等[[取代反应|取代反应]]，也可以发生[[氧化反应|氧化反应]]。

== 制备 ==
由[[煤焦油|煤焦油]]得[[蒽|蒽]]油[[馏分|馏分]]中分离而得。

== 用途 ==
用于制造[[合成树脂|合成树脂]]、[[染料|染料]]等。可形成具有[[荧光|荧光]]性的[[激发二聚体|激发二聚体]]。
