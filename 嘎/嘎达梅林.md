'''嘎达梅林'''（{{mong|m=ᠭᠠᠳᠠ ᠮᠡᠢᠷᠡᠨ|k=Гаадаа мэйрэн|Ppp=γada meyiren}}；{{bd|1892年||1931年|4月5日|}}），姓名'''莫勒特图·那达木德'''（{{mong|Ppp=Mültütü Nadmid}}），又名'''业喜'''，汉名'''孟青山'''，[[蒙古族|蒙古族]]，[[内蒙古|内蒙古]][[哲里木盟|哲里木盟]]（今[[通辽市|通辽市]]）[[达尔罕旗|达尔罕旗]]（今[[科尔沁左翼中旗|科尔沁左翼中旗]]）[[腰茫哈苏木|腰茫哈苏木]]（今[[舍伯吐镇|舍伯吐镇]]）[[塔木扎兰屯|塔木扎兰屯]]（今[[塔本扎兰嘎查|塔本扎兰嘎查]]）人。“嘎达”（同[[东北话|东北话]]“老嘎达”）[[蒙古语|蒙古语]]中意为家中最小的兄弟（老么），“[[梅林|梅林]]”是其官职，即[[札萨克|札萨克]][[答剌罕|答剌罕]]亲王[[那木济勒色楞|那木济勒色楞]]的[[总兵|总兵]]。

==生平==
嘎达梅林读过几年[[私塾|私塾]]，通[[蒙文|蒙]][[汉语|汉]]文。历任旗卫队[[章京|章京]]、[[昆都|昆都]]、[[扎兰|扎兰]]、[[梅林|梅林]]等职。

从[[清朝|清朝]]末期开始，为巩固边疆外防[[俄罗斯帝国|俄罗斯帝国]]进一步侵略，中央政府开始逐渐在蒙旗开垦土地，[[汉族|汉族]]民众被迁入成为[[佃农|佃农]]。[[中华民国|中华民国]]成立后，1914年2月，[[中华民国北京政府|中华民国政府]][[中华民国内务部|内务]]、[[中华民国农商部|农商]]、[[中华民国财政部|财政]]等部及[[中華民國蒙藏委員會|蒙藏事务局]]联合制定《禁止私放蒙荒通则》和《垦辟蒙荒奖励办法》，准备大兴蒙垦。从1916年，[[奉系军阀|奉系军阀]][[张作霖|张作霖]]开始大量放垦。放垦戍边增强了边防，促进当地的经济发展，但也损害了[[蒙古族|蒙古族]][[牧民|牧民]]的利益。从1904年的[[白音大赉|白音大赉]]起义开始，抗垦起义此起彼伏。

张作霖与常住[[奉天|奉天]]的[[那木济勒色楞|那木济勒色楞]]王爷商定开垦该旗的大片草原为农耕地，到1928年[[达尔罕旗|达尔罕旗]]四分之三的土地被放垦，牧场缩小，牧民被迫背井离乡，引起当地牧民的不满。嘎达梅林多次到[[垦务局|垦务局]]反对开垦，被免职。

1929年初，“[[东北易帜|东北易帜]]”后不久，[[张学良|张学良]]继续开垦蒙旗土地的计划。嘎达梅林等人发起“[[独贵龙运动|独贵龙运动]]”，即所有请愿的人在纸上围着一个圆圈签名以隐藏领头人，去沈阳向那木济勒色楞请愿。7月26日请愿代表[[色仁尼玛|色仁尼玛]]、[[赵舍旺|赵舍旺]]、[[僧格嘎如布|僧格嘎如布]]和嘎达梅林被捕，被押回本旗投入监牢。11月13日夜，嘎达梅林的妻子[[牡丹其其格|牡丹其其格]]伙同一些人劫牢反狱，将嘎达梅林救出<ref>{{cite web|url=http://www.cqvip.com/QK/97737X/200801/27232937.html|title=嘎达梅林之妻——牡丹}}</ref>。于是嘎达梅林组织[[起义|起义]]，领导了一支700多人的抗垦军队，提出了“打倒测量局，不许抢掠民财”的口号，袭击垦务局和垦荒军，驱逐测量队，转战于[[昭乌达盟|昭乌达盟]]（今[[赤峰市|赤峰市]]）、[[哲里木盟|哲里木盟]]一带。

张学良命[[张海鹏|张海鹏]]部一个骑兵团和[[汤玉麟|汤玉麟]]属下东北骑兵第17旅[[李守信|李守信]]团出兵围剿。1931年4月5日，抗垦队伍在今[[通辽|通辽]]北[[舍伯勒图|舍伯勒图]]附近[[新开河|新开河]]（今[[乌力吉木仁河|乌力吉木仁河]]）畔的[[红格尔敖包屯|红格尔敖包屯]]渡口，准备渡河南去时，被包围歼灭，嘎达梅林战死。[[李守信|李守信]]割下了嘎达梅林的头，回[[开鲁|开鲁]]送给[[崔兴武|崔兴武]]旅长，崔兴武用木匣包装送[[汤玉麟|汤玉麟]]。汤玉麟又交给了达尔罕亲王，并在旗内悬挂示众。后来在[[乌力吉木伦河|乌力吉木伦河]]边民众修建了[[梅林庙|梅林庙]]。

五个月后，[[九一八事变|九一八事变]]爆发，张学良的[[东北军|东北军]]撤入关内，牧垦草原的计划没有得以继续实施。

== 后果 ==
嘎达梅林的起义虽然失败，但是张学良的牧垦草原的计划被延迟。

嘎达梅林的起义是为了保护蒙古牧民的利益，而放垦对今天最大的危害是对环境的破坏。由于短视和无知，人们对草原的放垦并未因嘎达梅林的起义而停止。光[[科尔沁草原|科尔沁草原]]就“出荒”十一次。今天大部分草原都已沙化，成为“[[科尔沁沙地|科尔沁沙地]]”，是[[西辽河沙丘平原|西辽河沙丘平原]]的一部分，属正在发展的[[沙漠化|沙漠化]]土地，以风蚀沙地半固定状态为主。目前科尔沁沙地正以每年1.9%速度在发展，总面积8000万亩，是中国最大的沙地。嘎达梅林牺牲的[[乌力吉木仁河|乌力吉木仁河]]，如今已然变成一条沙沟。河早已断流，四周也是一片片沙化的农田和一座座沙包、沙坨子。有关当局在努力使沙漠化逆转，但当前尚无明显成效。

另据[[新华社|新华社]]报道，[[中国国家林业局|中国国家林业局]]最新监测，科尔沁沙地每年绿化面积大于沙化面积约75万亩。<ref>{{cite web |url=http://www.gog.com.cn/jqzt/zt0201/ca485612.htm |title=存档副本 |accessdate=2005-10-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20050212060116/http://www.gog.com.cn/jqzt/zt0201/ca485612.htm |archivedate=2005-02-12 }}</ref>

== 文艺作品 ==
[[蒙古族|蒙古族]]民歌《嘎达梅林》是一首流传很广的著名歌曲，其节奏舒展从容、稳健有力，旋律宽广豪迈、庄严肃穆。原歌歌词有五百多段，人们经常演唱的四段歌词是由[[作曲家|作曲家]][[安波|安波]]整理译配的。
:南方飞来的小鸿雁呀，不落[[长江|长江]]不呀不起飞。
:要说起义的嘎达梅林，是为了蒙古人民的土地。
:北方飞来的小鸿雁呀，不落[[长江|长江]]不呀不起飞。
:要说起义的嘎达梅林，是为了蒙古人民的土地。
:天上的鸿雁从南往北飞，是为了追求[[太阳|太阳]]的温暖呦。
:反抗王爷的嘎达梅林，是为了蒙古人民的利益。
:天上的鸿雁从北往南飞，是为了躲避[[贝加尔湖|北海]]的寒冷呦。
:造反起义的嘎达梅林，是为了蒙古人民的利益。 

1960年代，女作曲家[[辛沪光|辛沪光]]根据这首民歌创作了[[交响诗|交响诗]]《[[嘎达梅林_(交响诗)|嘎达梅林]]》。

[[File:gadameilin.jpg|thumb]]

2002年[[冯小宁|冯小宁]]编剧、导演了电影《[[嘎达梅林_(电影)|嘎达梅林]]》，[[鄂布斯|鄂布斯]]主演嘎达梅林。电影配乐是女作曲家[[辛沪光|辛沪光]]三子[[三宝_(作曲家)|三宝]]作曲，主题歌演唱[[腾格尔|腾格尔]]。

另外还有根据民歌《嘎达梅林》改编的[[琵琶|琵琶]]独奏、[[马头琴|马头琴]]曲等多种音乐作品。

這些文藝作品中，均不提是張學良的東北軍閥殺害了嘎达梅林，而是[[蒙古王公|蒙古王公]]勾結當時的[[國民政府|國民政府]]所為<ref>{{cite web |url=http://fengxuerong.blog.sohu.com/304156728.html |title=《冯学荣：是谁杀了嘎达梅林》-冯学荣读史-搜狐博客 |accessdate=2018-05-04 |deadurl=yes |archivedate=2014-07-08 }}</ref>，而發動劫獄的嘎达梅林妻子牡丹其其格，後來改嫁給東北軍將士胡寶山一事，也完全沒有在這些文藝作品中呈現<ref>[http://mongoliinmedee1.blogspot.tw/2015/09/blog-post_14.html 蒙古新闻: 嘎达梅林妻子牡丹的后半生]</ref><ref>[http://www.nmgwsg.org.cn/a/lingjinsuiyu/2014/0701/392.html 内蒙古自治区文史研究馆]{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
