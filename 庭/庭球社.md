《'''庭球社'''》<ref name=cn1/><ref name=cn2/>（{{lang|ja|てーきゅう}}，另译《网球并不可笑嘛》、《網球少女》<ref name=tw1/>）是由{{link-ja|Roots (漫画家)|ルーツ (漫画家)|Roots}}原作，由Piyo作画的一部[[日本|日本]][[漫画|漫画]]作品。于[[EARTH_STAR_Entertainment|地球之星娛樂]]发行的月刊漫画杂志《[[Comic_Earth_Star|Comic Earth Star]]》连载中。描写[[网球|网球]]部所属的四名女高中生的日常故事。作品名称《{{lang|ja|てーきゅう}}》来源于日语中网球的汉字写法“庭球”<ref>{{Cite comic| Writer = Roots| Artist = Piyo| Story = 先輩と一期一会| Title = てーきゅう| Volume = 1| Issue = 1| Date = 2012年8月11日| Publisher = EARTH STAR Entertainment| Page = p.7| }}</ref>。

==概要==
故事以虛構的龜井戶高校為舞台，雖然題材是網球，卻不著重在描寫網球部部員的練習和比賽。漫畫和電視動畫雙方的官方網站宣傳語均為“幾乎沒有網球要素的網球漫畫”（{{lang|ja|テニスをほとんどしないテニス漫畫}}），每話均為獨立故事。通常以每月2話的形式在《Comic Earth Star》上連載。每話標題均是“第x面”（x為話數）和“{{lang|ja|先輩と}}（电影標題）”<ref name="kanae">{{Cite comic| Writer = Roots| Artist = Piyo| Story = 入部屆け（新莊かなえ）| Title = てーきゅう| Volume = 1| Date = 2012年8月11日| Publisher = EARTH STAR Entertainment| Page = p.32}}</ref>。

在單行本第1-{卷}-發售的同時公布[[電視動畫|電視動畫]]化的企畫，以“連載開始才5個月就動畫化”成為話題，成為史上最快動畫化的漫畫<ref name="anime">{{Cite web |date=2012-08-11|url=http://news.mynavi.jp/news/2012/08/10/056/|title=TVアニメ『てーきゅう』、2012年10月より放送開始! 監督は板垣伸氏|accessdate=2013-01-01}}<br />{{Cite web |date=2012-08-10|url=http://mantan-web.jp/2012/08/10/20120809dog00m200055000c.html|title=てーきゅう ：連載わずか5カ月でテレビアニメ化決定|publisher=[[毎日新聞|毎日新聞]]|accessdate=2013-01-01}}<br />{{Cite web |date=2012-08-10|url=http://natalie.mu/comic/news/74452|title=ルーツ原作、テニス部女子の日常劇「てーきゅう」アニメ化|publisher=[[娜塔莉_(網站)|娜塔莉]]|accessdate=2013-01-01}}<br />{{Cite web |date=2012-08-10|url=http://www.ota-suke.jp/news/77492|title=TVアニメ「てーきゅう」キャストに鳴海杏子、三森すずこ、渡部優衣|accessdate=2013-01-01}}<br />{{Cite web |date=2012-08-10|url=http://www.animate.tv/news/details.php?id=1344558473|title=コミック アース・スターで連載中の『てーきゅう』テレビアニメ化決定!2012年10月7日(日)より放送開始。キャストは三森すずこさんなど。|publisher=[[Frontier_Works|Frontier Works]]）|accessdate=2013-01-01}}<br />{{Cite web |date=2012-08-10|url=http://otalab.net/news/detail.php?news_id=2738|title=『てーきゅう』TV アニメ化決定　主演聲優に三森すずこ|accessdate=2013-01-01}}<br />{{Cite web |date=2012-08-11|url=http://animeanime.jp/article/2012/08/11/11089.html|title=連載開始5カ月でアニメ化決定「てーきゅう」とは?|accessdate=2013-01-01}}</ref>。

《我是高宮茄乃》是以本傳登場人物高宮茄乃為主角的外傳漫畫，在Comic Earth Star 2014年5月號開始連載，描寫的是初中時代的高宮茄乃與她朋友們之間的日常故事。
