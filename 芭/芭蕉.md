'''芭蕉'''（[[学名|学名]]：{{lang|la|''Musa basjoo''}}）是[[芭蕉科|芭蕉科]][[芭蕉属|芭蕉属]]的植物。是中國傳統的[[庭園|庭園]]植物之一。<ref>{{Cite web |url = http://songshuhui.net/archives/64762 |title = 黄了香蕉，绿了芭蕉 |author = 钟蜀黍 |date = 2012年2月21日 |publisher = 科學松鼠會 |language = zh-hans |accessdate = 2014年3月2日}}</ref>

==形态==
多年生[[草本|草本]]，具匍匐茎。假茎绿或黄绿，略被白粉；叶片基部圆形，不对称，叶柄长30厘米，叶翼开张；穗状花序下垂，苞片红褐或紫；黄色肉质果实，有多数种子。

==分布==
分布于[[日本|日本]]、[[琉球|琉球]]、[[台灣|台灣]]以及[[中国|中国]]的[[上海|上海]]、[[江苏|江苏]]、[[浙江|浙江]]、[[湖南|湖南]]、[[湖北|湖北]]、[[四川|四川]]、[[陕西|陕西]]、[[贵州|贵州]]、[[云南|云南]]、[[广西|广西]]等地，生长于海拔500米至800米的地区，常生长在河谷、村边及山坡林缘，目前尚未由人工引种栽培。

== 别名 ==
甘蕉（植物名实图考），天苴、板蕉、牙蕉（福建），大叶芭蕉（江西），大头芭蕉、芭蕉头（四川），芭苴（古称），紅姬芭蕉。
