《'''持續狩獵史萊姆三百年，不知不覺就練到LV MAX'''》是[[森田季節|森田季節]]所作的[[日本|日本]][[輕小說|輕小說]]作品，插画由[[紅緒|紅緒]]担当。该书首在《[[成為小說家吧|成為小說家吧]]》上连载，随后发布单行本。此后发布了同名漫画并在《[[GANGAN_ONLINE|GANGAN ONLINE]]》上进行连载，作画由{{lang|ja|シバユウスケ}}擔任。

==簡介==
前世過勞死的OL相澤梓在異世界轉生成不老不死的魔女，作為日課每日狩獵史萊姆持續了三百年，不知不覺到了等級99後，開始有很多聽到傳聞的人來訪。

== 登場角色 ==
; {{nihongo|相澤梓|相沢梓（あいざわ あずさ）/アズサ・アイザワ|[[配音員|聲]]：[[悠木碧|悠木碧]]}}
:外号「高原的魔女」，外貌是17岁的少女，不老不死。
:原本是日本的社畜，每天都在辛劳工作，苦命的在二十七岁时过劳死亡，被女神可怜而以现在姿态转生到异世界。
:平常住在普拉达村旁的高原上，成天过着耕田做草药杀史莱姆耍废的悠闲生活，就这样过了三百年，因时常帮村民治病并阻止了数次瘟疫，很受村民尊敬，被称「高原的魔女」。三百年間持續狩獵史萊姆，不知不覺就練到LV.99的滿級。
:因為祟尚过着悠闲生活，不太願意卷入別人的麻煩當中，但又會因為怜悯對方而出手相助。
:把法露法、夏露夏和桑朵拉認作女兒，對打算(即便只是威脅)傷害女兒和弟子們的人會毫不留情全力擊倒。

; {{nihongo|萊卡|ライカ|聲：[[本渡楓|本渡楓]]}}
:红龙族，可以化身作人型，外貌是国中少女。三百歲。有一個姐姐蕾拉，已婚。很擅長木工，可以把弄毀了的梓的房子修好，還擴建了。
:原本是楠堤尔州、红龙族中最强，听到「高原的魔女」的消息后赌上最强名号前往挑战，然后输掉了。因戰鬥中誤毀了相澤梓的房子，在賠償道歉後成为了梓的第一個弟子，很尊敬打敗了自己的梓。
:因最强之龍成為了梓的弟子的傳聞，偶然令打算挑戰梓的人變少了。
:很妒忌佩克菈和弗拉德魯特對梓過於親密的行為，甚至要和弗拉德魯特進行決鬥。

; {{nihongo|法露法|ファルファ/ファルファ・アイザワ|聲：[[千本木彩花|千本木彩花]]}}
; {{nihongo|夏露夏|シャルシャ/シャルシャ・アイザワ|聲：[[田中美海|田中美海]]}}
:史莱姆妖精姐妹，外貌是十岁小女生，已有五十歲。法露法是姐姐，夏露夏是妹妹。
:300年间被梓干掉的史莱姆的幽灵聚合而生，因為梓而誕生，從而視梓为「媽媽」。
:法露法活潑好動，擅長數學。夏露夏沉默寡言，擅長歴史和神學。二人都很聰明，曾就自己擅長的主题發表過論文，在人類學者之間受到很高的評價。
:法露法因不想「媽媽」被殺死向梓告知妹妹的行動。
:夏露夏開頭因史莱姆的怨恨而打算殺死梓，苦練出『破邪「高原的魔女」』的魔法，但只對梓有效，會被梓以外的其他人輕易打敗。之後因魔力耗盡，暫時無法使用「破邪魔法」，再加上跟梓和姐姐的勸解，而暫時和解，把梓認作「媽媽」。變得親密後，很後悔當初打算殺死梓。

; {{nihongo|哈爾卡拉|ハルカラ|聲：[[原田彩楓|原田彩楓]]}}
:金发巨乳的森精灵，天然呆兼有點冒失，經常都會闖禍，但很有經營頭腦，也很熟悉植物和蘑菇的知識(但不能活用，因此常常誤食毒蘑菇，梓也曾受害)。
:自稱因製作「營養酒」而當中的成份傷害了别西卜而被别西卜追跡和通緝，族人為防止受牽連而把她趕出村，走投無路之際聽到「高原的魔女」的傳聞而來向梓求助。梓無奈地收為弟子以作庇護。
:在梓與別西卜交手和對談後，發現只是誤會，之後哈爾卡拉也繼續作為梓的弟子住下來。
:之後建立工廠，重新開始製作「營養酒」和其他酒類。
:在覲見魔王時，因一連串冒失行為而弄傷了魔王，被誤當成刺客，在梓一番周旋下才獲救。

; {{nihongo|[[巴力西卜|別西卜]]|ベルゼブブ|聲：[[沼倉愛美|沼倉愛美]]}}
:上级恶魔，白色长发有角的御姐，魔界现任农业大臣，別號「蒼蠅王」，能夠化為蒼蠅。因為很可靠和幾次幫助了梓，被梓私下當成姐姐。
:開頭被傳因哈爾卡拉製作出了「營養酒」，當中的成份使自己受害而追跡和通緝她，一度化為蒼蠅跟蹤在哈爾卡拉身邊，不現身而只偷喝哈爾卡拉的「營養酒」直到被梓發現。之後道出通緝哈爾卡拉的目的只是誤會，實際上只是因工作過勞而導致身體抱恙，而別西卜追跡和通緝哈爾卡的真正目的只是因為太喜歡哈爾卡拉製作的「營養酒」，而哈爾卡拉突然停製，打算追跡她，讓她重新開始製作。
:現出真身後，挑戰梓以令她交出哈爾卡拉，因梓忘記了自己設置的對魔族結界，而偶然令別西卜撞上結界落敗，之後被梓的治癒魔法恢復。之後二人成了朋友，別西卜也常常來作客，也教她召喚恶魔的魔法，必要時可以召喚自己。
:很喜歡小孩子，因此很喜歡妖精姐妹，常常來作客時都會帶禮物給她們，甚至常常請求梓讓她把姐妹們給她當養女。在梓因誤食毒蘑菇變小時，也進人了她的守備範圍。

; {{nihongo|芙拉德露忒|フラットルテ}}
:藍龙族，也可以化身作人型，外貌是高中少女。藍龙族與红龙族為仇敵(只是藍龙族單方面敵視、對红龙族找碴，而红龙族則只當藍龙族是一群肌肉笨蛋)。不太喜歡穿衣服,經常在狩獵時嫌衣服阻礙而脱光。
:一直很渴望能和心儀的龍結婚，已追求好幾個红龙族的龍，但因她是藍龙族和常常找红龙族的碴，而好幾次被拒絕。對萊卡的姐姐蕾拉結婚很妒忌，因而率領一部分藍龙族到婚禮會場和红龙族的領地的洛克火山搗亂。
:因攻擊危及女兒的安全以及襲擊的手法令梓感到不快，而被梓出手打倒。之後在梓和別西卜的威脅下，被迫簽下藍龙族不再與红龙族争鬥的條約，以及被萊卡要求了一大筆賠償費，在還清前要滯留在红龙族的領地。經常被梓唸錯名字。
:之後為防違反條約，被魔王佩克菈強制喚來讓梓對她進行支配的儀式(讓梓觸摸她的龍角)，因為被支配了，而必須來到梓的家。一度因為藍龙族的规矩而只在梓給指示時才行動，其他時侯則只站着。在梓給出「可以自由的生活」的指示後，才開始自由活動，因為梓的溫柔把她當成媽媽，起初常常與萊卡爭鋒吃醋，後來因喜好相近，漸漸關係變好。

; {{nihongo|羅莎莉|ロザリー}}
:女幽靈(地縛靈)。生前為某商人的千金，被父親欺騙以為要嫁給別的貴族，實為把她賣給了娼寮，發現被騙後自殺，化為幽靈。盤踞在哈爾卡拉選作工廠的大屋里，常常作祟嚇跑住客和屋主。
:被梓請來的別西卜抓住後，被梓提議來她那裡住，感激之餘，尊敬的稱呼梓為「大姐頭」。
:因為是地縛靈，所以必須附身在其他人(哈爾卡拉)身上，才能轉移到梓的家。可能因為相性太好，附身在哈爾卡拉身上後一度不能分離，在多番嘗試後，梓她們把羅莎莉弄醉，再丟到水里，才成功使二人分離。

; {{nihongo|法特拉|ファートラ}}
; {{nihongo|瓦妮婭|ヴァーニア}}
:[[利維坦|利維坦]]姐妹，別西卜的手下，接送貴賓時，二人輪流分別負責接送和招待，其中一位會變回原來的巨獸模樣作飛行接送，另一人化身人型，招待貴賓在飛行時的飲食。法特拉是姐姐，瓦妮婭是妹妹。
:法特拉是別西卜的秘書，為人認真，以完美完成任何工作為目標，如果出現了一點失誤受挫，會很失落。對妹妹工作時的不認真很不滿，經常對此有微言。
:瓦妮婭工作能力不比姐姐差，但常常不認真，以及工作時去偷懶而被姐姐或上司的別西卜抓包。

; {{nihongo|佩克菈|プロヴァト・ペコラ・アリエース/ペコラ}}
:現任魔王，全名「普朗瓦德．佩克菈．愛麗絲」。喜歡捉弄人，想找到比自己強的人來撒嬌。
:在迎見梓等人時，因哈爾卡拉的冒失行為而被弄傷，一度昏迷。
:在醒來後，因想看看梓的實力，故意以「釋放哈爾卡拉」為條件挑戰梓，因偶然說了威脅哈爾卡拉生命的言論，而使梓認真將其擊敗。落敗後，將梓當做姐姐撒嬌，要求她對自己的命令要帶点強硬，不然不會接受。

; {{nihongo|愛諾|エノ}}
:外号「洞窟的魔女」，與梓同樣是不老不死的魔女，活了一百五十年。
:擅長製造草藥的藥片，但缺乏自信及害怕出現在人面前，一直都沒有名氣，藥片也就沒有銷量。
:冒充為有名的「高原的魔女」，希望有人會因此購買自己的藥片以及得到人氣，雖然藥片的銷量的確是有了，但名義卻到了「高原的魔女」那裡。
:梓聽到了有自己的冒充者，為免對方誤作壞事，使自己的名譽受損，與萊卡多番追查，終於找到化身了的愛諾。萊卡因不滿有人冒充自己的師傅，一下子就捅破了愛諾的謊言。
:為了得到一點建議，愛諾求助於梓。萊卡對愛諾缺乏自信以及與人隔絕這些跟她想要有名氣的理想相违的行為看不過眼，對此出言指責，梓也告訴她，要有名氣，首先就要敢於出現在人前，於是愛諾就半推半就的到人的城鎮裡開店。
:因為藥片的功效極佳(能解除誤食毒蘑菇時產生的中毒反應，也因此助了梓幾次)，自己和藥片也有了名氣，得到了「洞窟的魔女」的外号。

; {{nihongo|優芙芙|ユフフ}}
:水滴妖精，充滿母性的氣質，對眾多妖精來說像姐姐或媽媽的存在。
:透過風之妖精知道「高原的魔女」和新誕生的妖精姐妹的事，邀請她們來参加妖精的派對，並想見一下作為法露法和夏露夏媽媽的梓。
:看到梓、和她交談後，看出她在這個世界沒有媽媽的存在，自願作為她的媽媽，要她把自己叫作「優芙芙媽媽」。因為她的母性氣質，梓也欣然接受了。
:經常很偶然的出現在梓附近，很喜歡把梓抱到自己的懷裡。因為作為梓的媽媽，也宣稱是法露法、夏露夏和桑朵拉她們的外婆。

; {{nihongo|桑朵拉|サンドラ}}
:活了三百年的[[曼德拉草|曼德拉草]]，能自主移動和說話，有點傲嬌的性格，毒舌。外貌是幼女。原本生活在別西卜的庭園裏。
:因為魔女經常會抓曼德拉草當藥材，本能上很畏懼魔女，最初看到梓時立刻就嚇的逃跑了。也很厭惡會採摘植物的森精灵。
:在被愛諾和眾多魔女尋找時，被梓保護了起來，因此跟上了梓，被梓命名為「桑朵拉」。被梓當成自己的第三個女兒。平時由法露法、夏露夏教授讀寫。
:用作為植物的知識，幫梓改善了她的菜園。

; {{nihongo|女子女子神|メガーメガ神}}
:世界的女神，最初可怜梓，把她轉生到這個世界。
:本來負責管理多個异世界(包括梓原本的世界和這個异世界)的高位女神，但因為過於偏袒女性，被其他的神降格，現在只負責管理這個世界。
