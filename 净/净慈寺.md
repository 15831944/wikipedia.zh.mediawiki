'''净慈寺'''，又称'''净慈报恩光孝禅寺'''，位于[[浙江省|浙江]][[杭州市|杭州]][[杭州西湖|西湖]]湖畔南屏山，[[汉族地区佛教全国重点寺院|汉族地区佛教全国重点寺院]]之一。

== 历史沿革 ==
净慈寺初为“'''慧日永明院'''”，为[[五代十国|五代十国]][[后周|后周]][[显德|显德]]元年（954年）[[吳越國|吳越]][[國王|國王]][[錢弘俶|錢弘俶]]为[[永明延壽|永明延壽]]禅师所建。[[北宋|北宋]]改为'''寿宁禅院'''，[[南宋|南宋]]又为'''净慈报恩光孝禅寺'''，简称'''净慈寺'''或'''净慈禅寺'''。被列为[[禅宗|禅宗]]五山之一。[[明|明]]、[[清|清]]以后屡毁屡建<ref name="A">{{Cite web |url=http://www.hangzhou.com.cn/20050801/ca1177211.htm |title=净慈寺 |access-date=2009-09-03 |archive-url=https://web.archive.org/web/20100510081241/http://www.hangzhou.com.cn/20050801/ca1177211.htm |archive-date=2010-05-10 |dead-url=yes }}</ref>。

== 建筑特点 ==

净慈寺为典型[[汉传佛教|汉传佛教]]寺庙风格。大雄宝殿西例有济祖殿，殿前有运木古井一口，殿内供奉[[济公|济公]]禅师像<ref name="A"/>。在净慈寺门右边有“[[南屏晚钟|南屏晚钟]]”碑亭，为杭州著名的[[西湖十景|西湖十景]]之一。

[[明太祖|明太祖]][[洪武|洪武]]年间，铸一座巨鐘，每日傍晚，夕阳西下，暝色苍茫，钟声在群山碧空中回荡，响彻云霄。由于南屏山空穴怪石较多，钟声经石穴回荡互激，钟声传播到十多里外<ref name="B">[http://www.aoohz.com/mingsheng/sl/020.htm 净慈寺介绍] {{webarchive|url=https://web.archive.org/web/20121207152236/http://www.aoohz.com/mingsheng/sl/020.htm |date=2012-12-07 }}</ref>。康熙皇帝也在净慈寺寺门外建碑亭，上刻“南屏晚钟”四字<ref>《西湖志》卷九：在南屏山净慈寺前，旧称南屏晓钟，康熙三十八年[[清圣祖|清圣祖]]仁皇帝御题十景，改南屏晓钟，建亭其处。</ref><ref>《湖山便览》卷七：在寺门，面临万工池。宋陈清波、张择端，明戴进俱有南屏晚钟图，圣祖御书四字，为十景之一，恭摹勒石建亭于此。</ref>。清朝末年，铜钟在战乱中消失<ref name="A" />。1984年，净慈寺在日本[[曹洞宗|曹洞宗]]大本山永平寺相助下，重建大梵钟。1986年11月21日，中日佛教界人士数百人欢聚净慈寺，举行了隆重的大梵钟落成法会<ref name="B" />。

在净慈寺后院，还保存着宋代高僧如净禅师墓塔<ref name="B" />。
