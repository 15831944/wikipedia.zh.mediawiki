'''亲亲我'''（英文名：'''Give me a kiss'''）2011年5月30日上映于[[中国大陆|中国大陆]]的[[喜剧电影|喜剧电影]]。<ref name=" fhw" >{{cite web |url=http://ent.ifeng.com/zz/detail_2011_05/10/6283115_0.shtml |title=《亲亲我》即将上映 老演员呼吁关注国产儿童片 |accessdate= 2011年|work=|publisher=凤凰网 |language = zh-cn}}</ref>

== 剧情 ==
该片讲述“[[亲情|亲情]]”为主题的故事。<ref>{{cite web |url=http://roll.sohu.com/20110513/n307483876.shtml |title=《亲亲我》将全国上映 小观众呼唤自己的儿童片 |accessdate= 2011年|work=|publisher=搜狐网 |language = zh-cn}}</ref>

孩童虫虫只有五岁，天真烂漫，他的爸爸是一个在城市里流浪工作的男子，因为贫穷和坏习惯太多，他的妈妈离开了这个家庭，他爸爸从此在爷爷眼中是“逆子”。怀着失落和困顿，他的爸爸带着他回到了江南老家，但是爷爷却对他们的到来表示冷漠和冷淡。天资聪慧的虫虫用各种小手段化解整个家庭里种种不开心不愉快的事情和矛盾。他爸爸一次用身体挡住了刺向爷爷的利刃，爷爷从此对他们刮目相看，再也不认为是“逆子”了，从而走入了正道，开始兢兢业业地做起事情来，后来慢慢地发家致富，那个因为贫穷而离家的妈妈也回到了这个家庭中。<ref>{{cite web |url=http://baby.sina.com.cn/news/2011-05-09/180048014.shtml |title=儿童情感电影《亲亲我》剧情梗概 |accessdate= 2011年|work=|publisher= 新浪网|language = zh-cn}}</ref><ref>{{cite web |url=http://datalib.ent.qq.com/movie/5085/movieDetails.shtml |title=亲亲我 |accessdate= 2011年|work=|publisher=腾讯网 |language = zh-cn}}</ref>

== 幕后 ==
演员[[高明_(演员)|高明]]呼吁市场上增加儿童片的数量，他说：“现在的电影市场太功利了，赚钱是许多人拍电影唯一的目的，我26年前就拍过儿童片，现在市场上的儿童电影比那个时候还少，这实在值得人深思。”<ref name=" fhw" />
