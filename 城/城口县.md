'''城口县'''地处[[大巴山|大巴山]]南麓，位于[[重庆市|重庆市]]最北端，也是重庆经济最不发达的一個县，距离重庆主城区334公里，公路距离404公里。

其东邻[[陕西|陕西]][[镇坪|镇坪]]、[[平利|平利]]，西与[[四川|四川]][[宣汉|宣汉]]、[[万源|万源]]相交，南连重庆[[开州区|开州区]]、[[巫溪|巫溪]]，北接陕西[[岚皋|岚皋]]、[[紫阳|紫阳]]。因居三省之门户名“城”，扼四方咽喉称“口”而得其名。城口縣在明、清時期曾先後被劃歸[[陝西|陝西]]行省和[[四川|四川]]行省。清朝[[道光|道光]]年间置[[城口廳|城口廳]]。民国初年，改为城口县。1950年起，历属[[万县专区|万县专区]]、[[万县地区|万县地区]]、[[万县市|万县市]]。1997年，撤销万县市，随之劃入重慶市。

城口幅员面积3292.4平方公里。全县辖24个乡镇、258个行政村、戶籍22.26万人，實際居住人口為19.3萬人，是一個人口凈流出縣。县政府驻葛城街道。

== 旅游资源 ==
城口境内有大巴山国家级自然保护区，动植物资源丰富。珍稀树种有水杉、光叶[[珙桐|珙桐]]、[[银杏|银杏]]，珍稀动物有华南虎、豹、白唇鹿、川金丝猴、梅花鹿，金雕。主要的风景有：鸡鸣寺、八台云海、亢河奇景、河鱼两扇门、青龙峡，扁桶峡风光、川陕界梁、崩溪河风景以及诸葛寨、[[黄安坝草场|黄安坝草场]]、九重山风景区等名胜古迹。

== 老区文化 ==
1932年中国工农红军第四方面军入川，点燃革命烽火。次年10月，[[李先念|李先念]]、[[徐向前|徐向前]]、[[王维舟|王维舟]]等率领红军克服千难万险，风餐露宿，跋山涉水，以秋风扫落叶之势东进，一举解放了城口县城。红军在城口打土豪、分田地，建立了中共城口县委和村、乡区、县各级苏维埃政权，城口成为川陕革命根据地的重要组成部分。

== 人口概况 ==
根据2010年人口普查数据，全县常住人口为19.30万人，全县常住人口中，男性人口为9.84万人，占50.99%；女性人口为9.46万人，占49.01%。总人口性别比（以女性为100，男性对女性的比例）为104.03。全县常住人口中，汉族人口为19.28万人，占99.93%；共有少数民族136人，其中，土家族89人，苗族15人，彝族8人，回族7人，布依族5人，藏族4人，蒙古族和侗族各2人，壮族、瑶族、佤族和仡佬族各1人。<ref name="人口概况">[http://wenku.baidu.com/view/171dec1055270722192ef725.html 重庆市城口县2010年第六次全国人口普查主要数据公报]，百度文库。</ref>

== 农林特产 ==
城口县境内林特资源十分丰富。盛产生漆、茶叶、核桃、[[板栗|板栗]]、薯芋、猕猴桃、薇菜、竹笋等农副土特产品，以及杜仲、天麻、黄连、党参等中药材。<ref>{{cite news|title=巴山深处聚宝盆|url=http://www.cqck.gov.cn/yszy/52/39.shtml|accessdate=2014-11-18|publisher=重庆市城口县人民政府}}</ref>
