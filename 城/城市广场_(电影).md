《'''城市广场'''》（{{lang-en|'''Agora'''}}）2009年10月9日上映于[[欧洲|欧洲]]的史诗片。

== 剧情简介 ==
公元4世纪，横跨亚欧非大陆的[[罗马帝国|罗马帝国]]日渐衰落。在埃及的[[亚历山大港|亚历山大港]]，年轻的女学者[[希帕提娅|希帕提娅]]开馆授徒，她是[[多神教|多神教]]的教徒，精通[[数学|数学]]、[[天文学|天文学]]。

412年，[[亚历山大的区利罗|西里尔]]出任[[基督教|基督教]]的[[大主教|大主教]]，他极富野心和才干，在亚历山大港大力推行基督教的传播，与当地多神教的传统发生了不可调和的矛盾。随着基督徒三番五次地掀起动乱，罗马执政官下令多神教徒撤出[[亚历山大图书馆|亚历山大图书馆]]，基督教在亚历山大港取得了压倒性优势，开始对异教徒进行大肆迫害。希帕提娅最终被狂热的教徒残忍地杀害。

== 花絮 ==
电影参加了[[第62届戛纳电影节|第62届戛纳电影节]]。<ref>{{cite web|url=http://www.m1905.com/news/20100426/344101.shtml|title=《城市广场》登陆英国 债务缠身“邦德”新片停拍|accessdate=2010-04-26|publisher=电影网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20100428185829/http://www.m1905.com/news/20100426/344101.shtml|archivedate=2010-04-28}}</ref>

电影围绕女学者希帕提娅的生平展开故事情节，展现了[[基督教|基督教]]早期传播的血腥和暴力，同时展现了基督教早期对[[科学|科学]][[技术|技术]]的压制和诋毁。

电影的取景地之一是[[马耳他|马耳他]]的[[里卡所里堡|里卡所里堡]]。<ref>{{cite web |url=http://movie.douban.com/subject/3007809/ |title=城市广场 |accessdate= |publisher=豆瓣网 |work= |language =zh-cn}}</ref>
