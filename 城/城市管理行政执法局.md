'''城市管理行政执法局'''或'''城市管理综合行政执法局'''，简称“'''城管局'''”，是[[中华人民共和国城市建制|中华人民共和国城市]]管理中负责综合行政执法的部门。2017年根据《城市管理执法办法》成立，城市管理执法人员属于行政执法类公务员，通过公务员考试并接受正规训练后，按照局、队的执法人员编制而调配。

城管执法监察制度的探索经历了从城建监察大队，到城市管理监察大队，到城市管理监察办公室，到城市管理综合行政执法局，再到城市管理综合行政执法监察局的发展。主要经过了以下三个阶段的发展：第一阶段，城建管理监察制度的诞生（1978—1995）；第二阶段，城管监察综合执法制度的探索（1996—2012）；第三阶段，城管执法监察的探索与发展（2012至2019）

== 職能 ==
城管执法，事实上是将过去中國城市各政府机构所拥有的各自范畴内的城市执法职能集中行使，包括市容[[环境|环境]][[卫生|卫生]]、[[城市规划|城市规划]]管理（无证违法建设处罚）、道路[[交通|交通]]秩序（主要是违法占路、封路处罚等）、[[工商|工商]]行政管理（无照经营处罚）、市政管理、[[公用事业|公用事业]]管理、城市供水管理、停车分配管理、[[园林|园林]]绿化管理、[[环境保护|环境保护]]管理（垃圾不分撿的处罚）、施工现场管理（含拆迁工地管理）、城市河湖管理、黑车、黑的、黑导游等各方面非刑事，需要城管以出动执法的事宜。

目前官方定义的城管主要职责如下：
# 贯彻实施国家及本市有关城市管理方面的法律、法规及规章，治理和维护城市管理秩序。
# 组织起草本市有关城市管理综合行政执法方面的地方性法规、规章草案，研究提出完善本市城市管理综合行政执法体制的意见和措施。
# 负责本市城管监察行政执法的指导、统筹协调和组织调度工作。
# 负责本市城管监察行政执法队伍的监督和考核工作。
# 负责本市市政设施、城市公用、城市节水和停车场管理中的专业性行政执法工作；负责城管监察队伍行政执法中跨区域和领导交办的重大案件的查处工作。
# 负责本市城管监察行政执法系统的组织建设、作风建设、队伍建设以及廉政勤政建设工作。
# 承办市政府及市市政管委交办的其它事项。

雖然城管有权对违规当事人进行教育或罚款，对违规物品在出具暂扣证明后进行暂扣，当事人可凭暂扣证明去城管办公室接受教育或罚款后，保证以后不会再犯后可以领回被扣物品，但'''理论上'''城管没有没收物品的权力。

== 概念 ==
{{原創研究}}
城市管理综合行政执法，或城管执法因经常被称为“城管”而被等同于城市管理工作而与“[[城市管理|城市管理]]”的概念混淆。而实际上，城市管理与城管执法是两个不同的概念。城市管理是时间、逻辑和知识三个维度的系统管理<ref>[http://www.mgov.cn/complexity/complexity3.htm 复杂性科学视野下的城市管理三维结构]</ref>。

# 从知识维看，理想中城市管理应是政府机构受权于市民，遵循自然科学、社会科学、人文科学等多种知识，对市政基础设施、公用事业、城市交通、环境卫生、市容景观、环境保护等城市生活众多领域作综合管理。城市管理行政执法也就是官方设立专门部门统一全权行使执法针对城市管理各领域，而期望减低多头执法扰民的现象并改善综合行政执法的整体效率。
# 从时间维看，城市管理是政府机构受权于市民，对城市规划、城市建设、城市运行全过程作管理，包括规划管理、建设管理和运行管理，这三个阶段的管理都包括执法和监督的部分，也就是城管执法的工作，包括规划执法、建设执法和市容、节水、绿化等城市运行管理的执法。
# 从逻辑维看，城管执法则是官方在城市管理中设置的决策、执行、监督逻辑（施政）流程的末端环节。

|官方定义城市管理行政执法局的设置不仅仅是为加强各专业执法之间的协调和整合，加强执行效率，同时也为将城市管理各专业领域的决策与执行一定程度的分离以加强相互的监督。

== 现实施行 ==
现实的问题是城管面对社会的底层群体，很多时候并没有足够有效的措施去管理、处罚违法人。且违法人经常因为自身的处境博得社会舆论的关注，从而使城管的工作更加难以进行。此外，城市政府部门缺乏有力监督制衡，部门间不断扯皮，经常是各部门将管不好、不好管的、面对社会低层群体的事务交予城管，如街头游商（有店面的归工商管理部门）、违章建筑（有执照的归规划部门）等等，而较易规范的则留在本部门。在许多地方城管也成了各类社会矛盾的焦点。
