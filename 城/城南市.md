'''城南市'''（{{kor|諺=성남시|漢=城南市|羅=Seongnam si|f=y}}）是[[大韓民國|韓國]][[京畿道|京畿道]]中央的一個市，位於韓國首都[[首爾|首爾]]的[[瑞草區|瑞草區]]、[[江南區|江南區]]以及[[松坡區|松坡區]]以南，比鄰首爾，是[[韓國首都圈|韓國首都圈]]的其中一個高速發展的新城市。城南市面積141.82[[平方公里|平方公里]]，人口977,166人(2004年終統計)。由於現時城南市的人口接近100萬人，所以最近有要求升格為[[廣域市|廣域市]]的請求。

== 地理 ==
城南市距離首爾的市中心約26[[公里|公里]]，北面是首爾市的三區，瑞草區、江南區及松坡區，東面是[[廣州市_(京畿道)|廣州市]]及[[河南市|河南市]]，西面是[[義王市|義王市]]、[[果川市|果川市]]，南面是[[龍仁市|龍仁市]]。

由於城南市距離首爾的[[江南區|江南區]]、[[瑞草區|瑞草區]]等[[副都心|副都心]]地帶非常近，它是不少打工一族的理想置業地點，加上國鐵[[盆唐線|盆唐線]]的開通，使當地人口急速上升。

== 歷史 ==
城南市取名於[[南漢山城|南漢山城]]之南側。公元前18年，[[百濟|百濟]]始祖[[溫祚王|溫祚王]]在城南市建立都邑。城南市原來是京畿道[[廣州郡|廣州郡]]的一部份，大韓民國建國初期開始開發。1973年7月1日，根據在同年3月12日制訂的法律第2597號，從原來廣州郡的[[大旺面|大旺面]] ({{lang|ko|대왕면}})、[[樂生面|樂生面]]({{lang|ko|낙생면}}，事務所位於[[金谷里|金谷里]])·突馬面和中部面的丹垈里·上大院里·炭里·壽進里·福井里·倉谷里組成新的城南市<ref>시설치와군의폐치분합에관한법률 (제정 1973年3月12日 법률 2597호)</ref>

== 行政区划 ==

全市分為三個區，分別是：西北部的[[壽井區|壽井區]]({{lang|ko|수정구}})、東北部的[[中院區|中院區]]({{lang|ko|중원구}})及南部的[[盆唐區|盆唐區]]({{lang|ko|분당구}})。

== 旅遊 ==

城南市的[[南漢山城|南漢山城]]是城南市的名稱由來，因為城南市位於南漢山城的南門以南。南漢山城是[[朝鮮王朝|朝鮮王朝]]時期興建的山城，位於[[首爾地鐵|首爾地鐵]][[首爾地鐵8號線|8號線]]“[[山城站|山城站]]”(822)及“[[南漢山城入口站|南漢山城入口站]]”(823)之間。1636年[[丙子胡亂|丙子胡亂]]，新立國的[[滿清|滿清]]由[[可汗|可汗]][[皇太極|皇太極]]親自領兵，入侵當時的朝鮮，時任朝鮮國王的[[朝鮮仁祖|朝鮮仁祖]]曾經遷都往南漢山城達45日之久（欲逃[[江华岛|江华岛]]不及，改入南汉山城），並與清軍苦戰。

該市旅遊景點尚有[[盆唐中央公園|盆唐中央公園]]和[[栗洞公園|栗洞公園]]。
