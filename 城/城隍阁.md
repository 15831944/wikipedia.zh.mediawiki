'''城隍阁'''，位于[[中国|中国]][[浙江省|浙江省]][[杭州市|杭州市]]新西湖十景吴山天风景区，为[[元朝|元朝]]和[[明朝|明朝]]建筑，为历史文化古迹，总面积1000亩。<ref>{{cite web|url=http://www.53chg.com/history.asp?id=4|title=城隍阁景区|accessdate=2012年10月1日|publisher=城隍阁官方网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20121204024343/http://www.53chg.com/history.asp?id=4|archivedate=2012年12月4日}}</ref>

== 历史 ==
城隍阁建立在[[吴山|吴山]]之上，一共七层，今存元朝和明朝建筑。

== 建筑 ==
城隍阁的现存建筑为元朝和明朝建筑以及后来的仿古建筑，七层楼，高41.6米，建筑面积3789平方米，基址是用石头堆砌而成。

== 文化 ==
城隍阁的每层楼都分别布有图画、木雕、古代建筑等。

*一层楼：《[[斗茶|斗茶]]图》、[[东阳|东阳]][[木雕|木雕]]、《[[西湖|西湖]][[龙舟|龙舟]]竞渡》、《[[南宋|南宋]]杭城风情图》、《南宋宫廷大傩图》、《货郎出街》<ref>{{cite web|url=http://www.53chg.com/history.asp?id=8|title=城隍阁景区一楼介绍|accessdate=2012年10月1日|publisher=城隍阁官方网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20121203191633/http://www.53chg.com/history.asp?id=8|archivedate=2012年12月3日}}</ref>
*二层楼：《[[孙权|孙权]]收[[台湾|宝岛]]》、《大江风采》、《[[宋孝宗|宋孝宗]]砸匾》、《[[乾隆帝|乾隆]]除恶霸》、《一词识[[柳永|柳永]]》、《兵围[[韩侂胄|韩王府]]》、《[[温日观|温日观]]骂贼》、《怒毁[[魏忠贤|魏阉]]祠》、《吴山清韵》、《[[胡雪岩|胡雪岩]]助银》<ref>{{cite web|url=http://www.53chg.com/history.asp?id=9|title=城隍阁二楼|accessdate=2012年10月1日|publisher=城隍阁官方网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20121204024426/http://www.53chg.com/history.asp?id=9|archivedate=2012年12月4日}}</ref>
