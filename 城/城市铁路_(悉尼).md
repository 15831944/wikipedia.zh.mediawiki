'''城市鐵路'''，中文简称'''城铁'''<ref>[http://www.sydneytrains.info/languages/chinese_simplified 雪梨城市鐵路简体中文網站]; [http://www.sydneytrains.info/languages/chinese_traditional 雪梨城市鐵路繁體中文網站]</ref>（{{lang-en|Sydney Trains}}），又稱'''悉尼列車'''，是澳洲[[新南威爾士|新南威爾士]]州政府的鐵路營運機構，由政府全資擁有及營運所有在[[悉尼|雪梨]][[客運|客運]]列車服務。

雪梨城市鐵路糅合[[地鐵|地下鐵路]]、[[市域铁路|市區鐵路]]、[[通勤铁路|近郊鐵路]]的特色，提供市區及市郊的快速公共運輸服務。繁忙時間主要路線班次約三分鐘一班車，非繁忙時間班次約10-15分鐘，週末列車班次約15-30分鐘，而某些路線的聯合班次則較頻密<ref>[http://www.sydneytrains.info/timetables/#landingPoint 時間表]Sydeny Trains</ref>。

[[新南威爾士州|新南威爾士州]]的鐵路系統由兩個機構負責：其中一個就是城市鐵路，負責營運市區及近郊線（簡單的說，只是在雪梨的範圍內營運鐵路）；另外一個機構則是[[新州鐵路|新州鐵路]]，負責營運遠郊線及城際線（簡單的說，是在雪梨的範圍以外營運鐵路，包括去[[堪培拉|坎培拉]]、[[墨尔本|墨爾本]]與[[布里斯班|布里斯班]]的火車）。

現時雪梨城市鐵路和新州鐵路遠郊線採用[[澳寶卡|澳寶卡]]（Opal Card）無接觸式的自動售票系統。

==歷史==
*[[File:Sydney_railway_map.gif|thumb]]2012年5月，交通部長宣佈對[[新南威爾士州鐵路局|新南威爾士州鐵路局]]（RailCorp）進行重組<ref>[http://www.smh.com.au/nsw/railcorp-job-cuts-first-of-many-unions-20120515-1yo93.html#ixzz1uwKcuUpH RailCorp job cuts first of many: unions] [[悉尼晨鋒報|悉尼晨鋒報]]</ref>。原由[[悉尼城市鐵路_(1988-2013)|城鐵]]運營的雪梨市區及近郊路線（寶路華站、鴯鶓地站、麥嘉特站及[[瀑布火車站|瀑布]]站以内）在2013年7月1日由雪梨城鐵轉由新的'''Sydney Trains'''铁路运营公司运营（但中文名称未改，仍称“城市鐵路”）。而遠郊及城際路線則由新組成的[[新州鐵路|新州鐵路]] （NSW TrainLink）營運。<ref>[http://www.yourfuture.transport.nsw.gov.au/About-the-Reform About the reform ] {{webarchive|url=https://web.archive.org/web/20130220191939/http://www.yourfuture.transport.nsw.gov.au/About-the-Reform |date=2013-02-20 }} Sydney Trains</ref>
*2013年4月，城市鐵路新標誌正式亮相<ref>[http://www.transport.nsw.gov.au/media-releases/fixing-trains-new-approach-customer-service Fixing the Trains: New approach to customer service] {{webarchive|url=https://web.archive.org/web/20130422135428/http://www.transport.nsw.gov.au/media-releases/fixing-trains-new-approach-customer-service |date=2013-04-22 }} Transport for NSW 18 April 2013</ref>。 
*2013年7月1日，新的城市鐵路成立並開始營運雪梨的市區線，由前[[倫敦地下鐵|倫敦地下鐵]]行政總裁霍士林（Howard Collins OBE）出任雪梨城市鐵路的行政總裁。
*2013年10月20日，城市鐵路重組為七條線。
*2015年2月8日，雪梨的西南鐵路線(South West Rail Link)開通，兩個新的車站開始啟用。T2線和T5線開始使用新線來往利坪頓。
*2016年8月1日，雪梨城市鐵路停售所有紙車票，但當日淩晨過後，部分售票機仍有售賣紙車票。
*2017年11月26日，城市鐵路再次重組為八條線，舊的T2線分成兩條新線，成為新的T2線與T8線。T2線和T5線在南區不再來往金寶鎮，只來往利坪頓。

=== 深宵巴士 ===
深宵巴士（{{Lang|en|NightRide}}）是一項雪梨城市鐵路與雪梨私營巴士營辦商在1989年成立的服務，共有13條路線和120個巴士站，服務時間為午夜12時至清晨5時，乘客可以憑有效雪梨列車車票使用服務，而巴士路線號碼以「N」為開首。

=== 城市鐵路車輛 ===
截至2017年，雪梨城市鐵路擁有六款近郊型列車。所有列車均為雙層電動組合列車，以直流電1,500伏特的[[高架電纜|高架電纜]]系統提供電力，並採用1,435毫米的[[軌距|標準軌距]] 。

==== 近郊型列車 ====
* A 系：Waratah - VVVF控制，有冷暖空調，現代化車身和自動報站，8輛編組
* B 系：Waratah series 2 - VVVF控制，A系的增購版本（共24列），除顯示系統不同外與A系沒有太大差異
* M 系：Millennium - VVVF控制，有空調，現代化車身和自動報站
* T 系：Tangara - 斬波控制，有空調，現代化車身
* C 系：Silver set - 斬波控制，有空調
* K 系：Silver set - 有空調
* S 系：Silver set - 無空調

=== 管轄區域 ===
城市鐵路有三個管轄區域，並由五間車廠負責日常維修。
* '''第一區'''：麼谷車廠 ：城東線，伊拉華線
* '''第二區'''：菲林明頓車廠：機場與東山線、城南線、彬士鎮線、金巴倫線、奧運線、卡寧福線
* '''第三區'''：康士比車廠：北岸線、城北線、西區線
* 奧本車廠
* 艾菲利車廠

== 票務系統 ==
雪梨城市鐵路自2014年4月起採用[[澳寶卡|澳寶卡]]（Opal Card）無接觸式的自動票務系統<ref>{{cite web |url=https://www.opal.com.au/en/about-opal/what-services-can-i-use-it-on/Trains/ |title=存档副本 |accessdate=2014-03-28 |0=2014-03-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20140303140325/https://www.opal.com.au/en/about-opal/what-services-can-i-use-it-on/Trains/ |archivedate=2014-03-03 }} </ref>。在2016年1月起，雪梨鐵路正式停用在1992年所推出的[[磁條|磁帶式車票]]，並轉用智能卡車票。<ref>{{Cite web|url=https://www.opal.com.au/en/opal-fares/no_more_paper_tickets/|title=No more paper tickets {{!}} NSW Government {{!}} Opal|website=www.opal.com.au|access-date=2016-07-20}}</ref> 。澳寶卡亦適用於巴士、渡輪及輕鐵服務但要留意有不同收費。 <ref>{{cite web|title=Fares and benefits|url=https://www.opal.com.au/en/opal-fares/|website=opal.com.au|publisher=Transport for NSW|accessdate=1 August 2016}}</ref>

與其他澳洲城市的自動售票系統不同，雪梨城市鐵路的票價系統是按距離計算的。
