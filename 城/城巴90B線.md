'''城巴90B線'''是[[香港島|香港島]]一條來往[[海怡半島|海怡半島]]和[[金鐘站|金鐘（東）]]的路線，主要是海怡半島及香港仔居民來往[[薄扶林|薄扶林]]及中上環的路線；而往金鐘的乘客，現大多選擇乘搭港鐵南港島綫。

==歷史==
*1994年12月19日，本線開辦，以全空調巴士服務。
*1995年2月17日，總站遷往第四期下（以前總站設於現海韻閣站）。
*1995年11月22日，沿-{[[干諾道西|干諾道西]]}-直入干諾道中。
*1997年5月5日，往海怡半島方向改經香港仔巴士總站，不停[[石排灣道|石排灣道]]。
*1997年7月7日，往海怡半島方向，恢復於石排灣道停站。
*2000年12月26日，改為循環運作。
*2010年6月26日，取消循環運作，以海怡半島及金鐘（東）作兩邊總站，以解決循環線太長導致班次失準問題。

==收費==
全程：$6.2
*瑪麗醫院往金鐘（東）：$5.2
*中山紀念公園往金鐘（東）：$3.7
*瑪麗醫院往海怡半島：$5.2
*石排灣道近華富道往海怡半島：$3.2

===八達通轉乘優惠===
乘客登上本線後指定時間內以同一張八達通卡轉乘以下路線，或從以下路線登車後指定時間內以同一張八達通卡轉乘本線，次程可獲車資折扣優惠：

;90B←→4、7、37A/37B/37X、40、40M、40P、43X、71/71P、91，轉乘時限為90分鐘，必須於瑪麗醫院轉乘
*'''90B往金鐘''' → 4、7、71P、91往中環、37B/37X往金鐘、40P往羅便臣道、71往上環，次程車資全免
*'''90B往金鐘''' → 40往灣仔碼頭或40M往政府總部，次程可獲$4.2折扣優惠
*4、7、71P、91往中環、37B/37X往金鐘、40往灣仔碼頭、40M往政府總部或71往上環 → '''90B往金鐘'''，次程車資全免
*40P往羅便臣道 → '''90B往金鐘'''，次程可獲$4.2折扣優惠
*'''90B往海怡半島''' → 7往石排灣、37A往置富、40/40M往華富、43X往華貴或71往黃竹坑，次程車資全免
*7往石排灣或71往黃竹坑 → '''90B往海怡半島'''，次程可獲$3.8折扣優惠
*37A往置富或40/40M往華富 → '''90B往海怡半島'''，次程可獲$3.2折扣優惠

;90B←→97A
*'''90B往海怡半島''' → 97A往鴨脷洲大街，次程車資全免，轉乘時限為90分鐘
*97A往深灣 → '''90B往金鐘'''，回贈首程車資，轉乘時限為60分鐘

;90B←→A11、A12
*'''90B往金鐘''' → A11或A12往機場，回贈首程車資，轉乘時限為90分鐘
*A11往北角碼頭 → '''90B往海怡半島'''，次程車資全免，轉乘時限為120分鐘
