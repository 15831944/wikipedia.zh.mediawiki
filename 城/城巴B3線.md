'''城巴B3線'''為一條來往[[屯門碼頭|屯門碼頭]]和[[深圳灣口岸|深圳灣口岸]]的跨境巴士路線，由城巴營運，而'''B3A線'''和'''B3X線'''則為由B3線分拆的支線，同樣由城巴營運，分別由[[山景邨|山景邨]]和[[屯門市中心|屯門市中心]]往返深圳灣口岸。至於'''B3M線'''則為B3X線的循環支線，循環來往深圳灣口岸[[屯門站_(西鐵綫)|屯門站]]。

== 簡介及歷史 ==
{{main|深圳灣口岸}}

[[深圳灣口岸|深圳灣口岸]]於2007年7月1日正式開通，政府公開招標三條由[[屯門區|屯門區]]和[[元朗區|元朗區]]開往[[深圳灣口岸|深圳灣口岸]]的巴士線和一條由元朗區（[[天水圍北|天水圍北]]）開往深圳灣口岸的專線小巴線，而[[城巴|城巴]]成功投得該線。該線是城巴唯一的過境專利巴士路線。在招標時，原本計劃屯門區總站為屯門碼頭，並繞經[[山景邨|山景]]、[[大興邨|大興]]、[[建生邨|建生]]及[[兆康苑|兆康]]，但被屯門區議員指路線過份繞路，結果由B3分拆出一條'''B3X'''，提供[[屯門市中心|屯門市中心]]不停站（2008年8月16日起加停中途站）特快往來深圳灣口岸的服務。該兩條路線與[[新大嶼山巴士B2線|新大嶼山巴士B2線]]合稱為「深西快線」。該線配合口岸開通，於2007年7月1日18:00正式開始行走。

不過，並非屯門區所有地區均有直達深圳灣口岸的服務，其中屯門西北區的[[良景邨|良景邨]]、[[寶田邨|寶田邨]]及屯門東北區的[[富泰邨|富泰邨]]，欲要前往深圳灣口岸，必須轉車才可前往，再加上B3過份迂迴，在多名屯門區議員爭取下，運輸署終於批准城巴B3及B3X作出以下重整，並於2008年8月16日起生效<ref>[http://www.td.gov.hk/traffic_notices/index_tc_inmode_3_trafficid_13144.htm 運輸署通告－城巴巴士服務第B3號及B3X號服務調整及開辦城巴巴士服務第B3A號] {{webarchive|url=https://web.archive.org/web/20080814010727/http://www.td.gov.hk/traffic_notices/index_tc_inmode_3_trafficid_13144.htm |date=2008-08-14 }}</ref>：

* B3號線改為來回程取道屯門公路、杯渡路直接往來元朗公路，不再經[[山景邨|山景]]、[[大興邨|大興]]、[[建生邨|建生]]及[[兆康苑|兆康]]。
* B3X號線加設中途巴士站，往深圳灣方向加停屯門公路[[屯門新墟|新墟]]、[[紅橋_(香港)|紅橋]]巴士站，而往屯門市中心方向，加停屯門公路紅橋及屯興路[[屯門官立中學|屯門官立中學]]巴士站。
* 增闢一條新線'''B3A'''，往來山景至[[深圳灣口岸|深圳灣口岸]]，途經大興、[[良景邨|良景]]、[[寶田邨|寶田]]、[[建生邨|建生]]、[[兆康苑|兆康]]及[[富泰邨|富泰]]，取代原有B3號線由兆康苑至山景一段的服務，主要方便屯門西北區及東北區居民往來深圳灣過境。<!-- 由於位於[[石排頭路|石排頭路]]的山景巴士總站正在擴建，因此開辦初期以[[鳴琴路|鳴琴路]]作臨時總站。 -->本線是城巴首條服務屯門良景邨及富泰邨一帶的路線。

B3A開辦時的頭車由山景於0645開出，但由於B3分拆前，由屯門碼頭開出的頭車為0600，約0620抵達山景，被居民投訴變相延遲頭車時間，因此開辦後一星期，便提早山景及深圳灣口岸之頭車開出時間，成為少數開辦後不久便作出改動的專利巴士路線之一。

至於B3M為新開設，來往西鐵綫屯門站，以「M」為路線後綴。

==批評==
屯門區內大型商場為求增加內地客人流，購物滿一定金額便送出B3X路線[[城巴|城巴]]免費車票吸引內地客購物，令[[深圳灣|深圳灣]]居民蜂擁跨境購物及水貨客問題更趨嚴重<ref>{{cite news |language = 中文  |url = http://hk.apple.nextmedia.com/news/art/20150126/19016596 |title = 內地客逼爆屯門　萬二人搶搭城巴 |publisher = 蘋果日報 |date = 2015年1月26日 |accessdate = 2015年5月1日}}</ref><ref>{{cite news |language = 中文  |url = http://hk.apple.nextmedia.com/news/art/20150124/19014324 |title = 水客變招　逼爆巴士運貨 屯門元朗重災　恃人多懶理車長阻止 |publisher = 蘋果日報 |date = 2015年1月24日 |accessdate = 2015年5月1日}}</ref>。有關注團體在假日點算，一度有數百人排隊輪候巴士，人龍長逾100米。估計3小時內有逾12,000人滿載而歸，乘車返回深圳灣。當中逾1,800人違規攜帶大件及超重行李乘車，涉嫌違反《公共巴士服務規例》，批評城巴職員未有阻止違例水貨客上車，有市民歸咎於附近商場贈送乘車券給旅客，生活很受影響。城巴職員要將有行李及沒行李的乘客分開處理。<ref>{{cite news |language = 中文  |url = http://hk.apple.nextmedia.com/news/art/20150126/19016596|title = 水貨客疑用城巴運貨 團體點算3小時近2,000人行李違規 |publisher = 蘋果日報 |date = 2015年1月25日 |accessdate = 2015年5月1日}}</ref>

巴士公司職員指，為疏導乘客，B3X線班次已加密至兩、三分鐘一班，但「都清唔晒」。城巴回應指車長雖有權拒絕攜帶過大行李的乘客上車，但同時會據實際情況來酌情處理。<ref>{{cite news|url=https://hk.news.yahoo.com/%E5%B1%AF%E9%96%80b3x%E6%B7%AA%E6%B0%B4%E8%B2%A8%E5%AE%A2%E5%B0%88%E5%B7%B4-221514449.html|title=屯門B3X淪水貨客專巴|date=2015-01-26|publisher=[[太陽報|太陽報]]|accessdate=2015-01-26|language=zh-hk|deadurl=yes|archiveurl=https://web.archive.org/web/20150127135315/https://hk.news.yahoo.com/%E5%B1%AF%E9%96%80b3x%E6%B7%AA%E6%B0%B4%E8%B2%A8%E5%AE%A2%E5%B0%88%E5%B7%B4-221514449.html|archivedate=2015-01-27}}</ref>

城巴表示2014年只有 7% 的乘客是使用乘車券乘搭這條線，認為加密班次及限制行李數目足以疏導人流，未有打算停售乘車券予商場。信和旗下的[[屯門市廣場|屯門市廣場]]表示，會繼續向訪港旅客提供換領過境車票優惠。而新鴻基旗下的[[V_City|V City]]未有回應傳媒的查詢。<ref>{{cite news|url=http://cablenews.i-cable.com/webapps/news_video/index.php?news_id=451116|title=屯門內地客增加居民指影響生活|date=2015-01-31|publisher=[[有線新聞|有線新聞]]|accessdate=2015-01-31|language=zh-hk}}</ref>被傳媒廣泛報道後，本線向屯門大型購物商場派發乘車券的安排在2015年1月30日取消<ref>{{cite news |language = 中文  |url = http://hk.on.cc/hk/bkn/cnt/news/20150131/bkn-20150131133437639-0131_00822_001.html |title = 水貨客逼爆　城巴突煞停B3路線換領券計劃 |publisher = 東方日報 |date = 2015年1月31日 |accessdate = 2015年5月1日}}</ref>。

在取消向屯門大型購物商場派發乘車券的安排後，B3系列路線依然有大量水貨客以它們運送水貨，《[[人口政策關注組|人口政策關注組]]》指城巴未有有效制止攜帶超重行李的乘客上車，如果城巴繼續拒絕執行《公共巴士服務規例》的行李法例，運輸署應考慮控告城巴<ref>{{cite news |language = 中文  |url = http://hk.apple.nextmedia.com/news/art/20150218/19046808 |title = 團體促運輸署檢控城巴 |publisher = 蘋果日報 |date = 2015年2月18日 |accessdate = 2015年5月1日}}</ref>。而水貨客的問題在「[[一周一行|一周一行]]」實施後略有改善。
