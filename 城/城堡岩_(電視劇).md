《'''城堡岩'''》（{{lang-en|''Castle Rock''}}）是一部[[心理學恐懼|心理學恐懼]]類型的[[網絡劇|網絡電視劇集]]。故事改編自[[史蒂芬·金|史蒂芬·金]]創作的一系列小說，通過虛構的小鎮城堡岩將人物和情節串聯起來。本劇由製片人[[J·J·亞伯拉罕|J·J·亞伯拉罕]]和編劇斯蒂芬·金攜手打造，於2018年7月25日在[[Hulu|Hulu]]首播。2018年8月14日，Hulu確定預訂本劇第二季。

== 故事背景 ==
《城堡岩》以[[史蒂芬·金|史蒂芬·金]]的一系列小說中虛構世界和人物為背景，在[[緬因州|緬因州]]方圓數[[英里|英里]]的森林小鎮上編織出一段關於黑暗和光明的史詩傳奇<ref name="AVC">{{cite news|last=Chavez|first=Danette|url=http://www.avclub.com/article/hulu-gives-mysterious-castle-rock-10-episode-order-250754|title=Hulu gives J.J. Abrams’ mysterious Castle Rock a 10-episode order|date=2017-02-21|work=[[影音俱樂部|影音俱樂部]]|accessdate=2018-06-26|language=en-US}}</ref>。

== 演員與角色 ==
=== 主要角色 ===
==== 第一季（2018年） ====
*; [[安德烈·荷蘭|安德烈·荷蘭]] 飾演 亨利·迪佛（{{lang|en|Henry Deaver}}）
*; {{le|卡萊爾·哈里斯|Caleel Harris}} 飾演 幼年亨利·迪佛
: 一名死刑律師。因為是父親被害案件的唯一嫌犯——雖然他對此沒有任何記憶——致使他無法回到家鄉小鎮<ref name="Characters">{{cite web|last1=Li|first1=Shirley|title=Welcome to 'Castle Rock': On the set of Stephen King's most terrorized town|url=http://ew.com/tv/2018/05/24/castle-rock-preview-set-visit-stephen-king-jj-abrams/|website=Entertainment Weekly|accessdate=2018-06-19|date=2018-05-24|language=en-US}}</ref>。其失蹤11天的真相是他意外進入平行世界，在平行世界的亨利與莫莉的幫助下回到原本的世界並被潘格彭尋獲。在對「少年」不信任之下把他重新關入鯊堡監獄，一年後與溫德爾定居於城堡岩。
*; [[比爾·史柯斯嘉|比爾·史柯斯嘉]] 飾演 鯊堡監獄囚犯／「少年」／亨利·迪佛（{{lang|en|Shawshank Prisoner / "The Kid" / Henry Deaver}}）
: 在30年前被戴爾·萊西典獄長秘密關入[[麗塔海華絲與蕭山克監獄的救贖|鯊堡州立監獄]]，2018年被尋獲並安置在嚴加看管區域，沈默寡言，似乎遺忘了自己的姓名，外表與27年前相比毫無變化。指定亨利為他提供法律上的協助<ref name="Characters"/>。具有讓觸摸他或是他所在的環境發生邪惡事件的能力，因此造成獄友全身被癌細胞侵蝕、城堡岩火山爆發、札萊夫斯基的監獄屠殺、精神病院大火以及潘格彭的意外身亡。真實身份為平行世界的另一位亨利·迪佛，為露絲與馬修的親生骨肉，為一名頗具盛名的阿茲海默症研究者，因馬修的自殺回到城堡岩，並在自家找到了另一個世界的幼年亨利，在幫助幼年亨利回到自己的世界時一同被捲入其時空，在樹林遊蕩幾天後被萊西典獄長尋獲並關入鯊堡監獄。在告知莫莉真相後企圖說服亨利，但被不信任他的亨利重新關進鯊堡監獄中。
: <small>[[比爾·史柯斯嘉|比爾·史柯斯嘉]]在《[[牠_(電影)|牠]]》以及《[[牠：第二章|牠：第二章]]》飾演{{le|牠／跳舞小丑潘尼懷斯|It (character)}}。而《[[末日逼近|末日逼近]]》中有一同名角色「The Kid」，未知是否為同一人。</small>
*; [[梅兰妮·林斯|梅兰妮·林斯]] 飾演 莫莉·史川德（{{lang|en|Molly Strand}}）
*; 卡西蒂·麥克林西 飾演 幼年莫莉·史川德
: 莫莉·史川德房地產公司的老闆，為亨利兒時的鄰居。由於具有[[心靈感應|心靈感應]]以及[[共情|共情]]等[[鬼店|閃靈能力]]，必須仰賴非法物質過活<ref name="Characters"/>。一年後仍繼續做不動產經紀人的工作，並去探望祖母
: 平行世界的莫莉為一名市議會成員，為幫助穿越過來幼年亨利意外被札萊夫斯基的子彈打死。
: <small>「莫莉（Molly）」是[[史蒂芬·金|史蒂芬·金]]愛犬的名字。年輕的莫莉是[[雷蒙斯合唱團|雷蒙斯合唱團]]的歌迷，該樂團為《[[禁入墳場|禁入墳場]]》提供了主題曲。此外，莫莉提到她住在一名連環殺手自殺身亡的房子裡，該連環殺手為《[[死亡禁地|死亡禁地]]》的法蘭克·多德。</small>
*; [[西西·史派克|西西·史派克]] 飾演 露絲·迪佛（{{lang|en|Ruth Deaver}}）
*; {{le|莎勒·費斯克|Schuyler Fisk}} 飾演 中年露絲·迪佛
: 退休大學教授，馬修的妻子，亨利·迪佛患有[[癡呆症|癡呆症]]的養母。過著亨利無法認同的生活，就算被恐怖籠罩也不願離開城堡岩<ref name="Characters"/>。由於她的症狀，她不斷經歷人生中的各個重要時期，因此會在所到之處擺放西洋棋來確定她處於現在。一年後因病逝世，與潘格彭合葬。
: 平行世界的露絲在亨利（「少年」）幼年時離開馬修，並與潘格彭在[[薩拉索塔_(佛羅里達州)|薩拉索塔]]同居。
: <small>[[西西·史派克|西西·史派克]]在《[[魔女嘉莉_(1976年電影)|魔女嘉莉]]》飾演{{le|嘉莉·懷特|Carrie White}}。而「露絲（Ruth）」是[[史蒂芬·金|史蒂芬·金]]母親的名字。</small>
*; [[珍·莉薇|珍·莉薇]] 飾演 黛安·「潔姬」·托倫斯（{{lang|en|Diane “Jackie” Torrance}}）
: 一名有志的作家，計程車司機，同時也是莫莉·史川德房地產公司的員工。知道城堡岩歷史有關的大量知識。為[[鬼店|傑克·托倫斯]]的姪女，刻意把名字「黛安」改成與叔叔相似的「潔姬」以報復家人。一年後正在寫作托倫斯家族的家族史，並打算在接下來的半個月前往[[鬼店|全景飯店]]完成這本書。
: <small>其姓名與《[[鬼店|鬼店]]》主角傑克·托倫斯相似，本名「黛安」(Diane)可能命名自《[[闪灵_(电影)|鬼店]]》與[[史丹利·庫柏力克|史丹利·庫柏力克]]的共同編劇黛安·強森(Diane Johnson)。而根據潔姬描述傑克使用的凶器是斧頭而非大鐵鎚，可推論本劇與[[史丹利·庫柏力克|史丹利·庫柏力克]]所拍攝的電影版《[[闪灵_(电影)|鬼店]]》發生於同一時空。無獨有偶，她在從高登手上救出亨利前使用的凶器就是斧頭。</small>
