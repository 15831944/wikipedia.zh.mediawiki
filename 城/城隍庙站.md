'''城隍庙站'''，建设时曾用名'''柳汀街站'''，是[[浙江省|浙江省]][[宁波市|宁波市]]一座地下轨道交通车站，得名于东侧的[[宁波府城隍庙|宁波府城隍庙]]。车站于2015年9月26日启用<ref name="main">{{cite web|url=http://news.cnnb.com.cn/system/2010/12/23/006790463.shtml|title=2号线车站设计人性化而有宁波味 效果图抢先看|author=陈锦源|publisher=中国宁波网|date=2010-12-23|accessdate=2011-01-21}}</ref>。

== 位置 ==
城隍庙站位于[[海曙区|海曙区]][[柳汀街|柳汀街]]、解放南路路口南侧<ref name="main" />。

== 车站形制 ==
城隍庙站长149米，标准段宽18.32米，深17.09米，设4个出口，2组风亭。地下一层为站厅，二层为[[岛式车站|岛式车站]]，宽11米，与改造后的玲珑宾馆形成地下二层，地上四层屬商业开发。<ref name="main" />

== 车站装饰 ==
[[File:Chenghuangmiao_Station,_2015-10-01_02.JPG|thumb]]
城隍庙站为2号线装修重点站之一。车站装修主题为市井文化，采用窗格造型的镂空通透吊顶进行装饰<ref name="decoration-1">{{cite web|url=http://nb.zjol.com.cn/system/2015/09/07/020821603.shtml|title=洋溢甬上味道 宁波地铁二号线开通在即|publisher=宁波在线|date=2015-09-07|accessdate=2015-09-28}}</ref>。车站文化墙主题为“百味城隍”。文化墙由三层构成，背景为一幅反映老宁波城隍庙风貌的素描，第二层为传统建筑装饰，第三层则为宁波特色小吃形象，以期表现宁波老城隍庙的风俗和文化<ref>{{cite web|url=http://www.nbmetro.com/news_dt.php?info/961|title=2号线一期6大车站文化墙亮相|publisher=宁波轨道交通集团|date=2015-09-23|accessdate=2015-09-27}}</ref>。

== 交通接驳 ==
城隍庙站接驳公交第一医院站、广济街站和开明街站<ref name="query" />。
