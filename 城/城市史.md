'''城-{}-市史'''又称'''都-{}-市史'''，论述[[城市|城市]]发展的历史。城市与村庄都是人类的定居点，但是城市与村庄不同的是，居住在村庄的大部分人口从事农业生产，而居住在城市的大部分人口则从事非农业生产，如工业、商业、政治管理、和其他服务行业。从古到今，城市越来越多，可以说人类文明发展的历史，也是一部人类[[城市化|城市化]]的历史，研究城市发展进程的规律，有助于理解人类文明发展的历史，同时也有利于现代城市的建设与完善。

== 城市的起源 ==
城市起源于人类定居的需要。人类早期逐水草而生，居无定所，进入[[农业文明|农业文明]]时代之后，以耕种土地为生，从而定居。而定居人群较为集中的地方，就是城市。目前考古发现人类最早的城市是[[两河流域|两河流域]]中下游公元前3500年时代的[[苏美尔|苏美尔文化]]时期，有[[尼普尔|尼普尔]]城，[[巴比伦|巴比伦]]城。

中国有考古发掘证明的最早城市在河南[[洛阳|洛阳]][[偃师|偃师]][[二里头|二里头]]，年代约为公元前2100～前1700年<ref>邸永君：[http://iea.cass.cn/ygtd/html/mzs_2007083114160316308.htm 城市的起源和发展]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 古代城市 ==
古希腊于地中海沿岸建立了[[特洛伊|特洛伊]]、[[叙拉古|叙拉古]]、[[斯巴达|斯巴达]]、[[雅典|雅典]]等城市。[[罗马帝国|罗马帝国]]时期，出现一批围绕着军事要塞形成的政治城市。[[基督教|基督教]]合法化之后，传教士逐步建立了一些传教据点，其中一些据点发展成为[[教区|主教区]]中心。

中国古代扬州城的建立。[[春秋时期|春秋时期]]，公元前486年，吴灭邗，筑邗城，开[[邗沟|邗沟]]，连接长江淮河。以后越灭吴，楚灭越。公元前319年，楚在邗城旧址上建城，名[[广陵|广陵]]。秦统一中国后，设广陵县，属九江郡。汉代称广陵、江都，属十三刺史部中的徐州。吴王[[刘濞|刘濞]]在此“即山铸钱、煮海为盐”，开盐河，盛极一时，成为以商业为主的城市。

== 中世纪城市 ==
罗马帝国瓦解之后，欧洲进入了封建割据的[[中世纪|中世纪]]。一批封建领主或者邦国国王分别建立起各自的政治统治中心，天主教主教们也以主教区教堂为核心，建立起教区中心。一批商人和工匠，依附于领主城堡或者主教教堂，建立起集贸市场，逐步形成政治工商城市，例如[[伦敦|伦敦]]、[[汉堡|汉堡]]。而有些集贸市场索性建立在交通便利的小村庄上，例如[[威尼斯|威尼斯]]和[[阿姆斯特丹|阿姆斯特丹]]，成为自发的商业城市。当时欧洲中世纪的乡村农民人身依附于领主，处于农奴或者半农奴地位。而城市市民则是自由民。当时主要的商业城市有[[阿马尔菲|阿马尔菲]]、[[威尼斯|威尼斯]]、[[比萨|比萨]]、[[米兰|米兰]]、[[佛罗伦萨|佛罗伦萨]]、[[马斯特里赫特|马斯特里赫特]]、[[布鲁日|布鲁日]]、[[根特|根特]]、[[鲁昂|鲁昂]]等。

=== 城邦 ===
{{main|威尼斯共和国}}
中世纪最先富裕起来的城市是意大利北部地区的城市。这个地区处于欧洲与中东[[列凡特|列凡特]]地区之间，当时欧洲最主要的进口商品——食品[[香料|香料]]，是通过意大利的商船来运销的。而[[威尼斯|威尼斯]]、[[佛罗伦萨|佛罗伦萨]]、[[热那亚|热那亚]]等城市，也发展起玻璃制造、丝绸纺织、毛纺织、造船、兵器等制造业，其产品销往东西方各地。威尼斯、佛罗伦萨、热那亚都发展成为城邦国家。

== 近代商业城市 ==
乔尔·科特金认为[[阿姆斯特丹|阿姆斯特丹]]是第一个成功的现代商业城市。该城市原来是一个小渔村，在[[荷兰独立战争|荷兰独立战争]]中，逃离西班牙专制统治的新教徒、犹太人、银行家、工匠等等，汇集到阿姆斯特丹，使之成为欧洲第一个摆脱贵族和僧侣控制，由唯利是图的商人和工匠自治的城市。新教[[加尔文教派|加尔文教派]]的牧师删除了旧天主教中关于禁止高利贷的教义。世界历史上第一个股票交易所也于1609年在阿姆斯特丹诞生。
