== 創作 ==
《城市獵人》在正式連載之前北条司先行發表《城市獵人 -XYZ-》以及《城市猎人 -双刃剑-》（{{lang|ja|シティーハンター -ダブルエッジ-}}）兩部[[平行世界|平行世界]]作品，主角冴羽獠的原型來自北条司之前漫畫作品《[[貓眼三姐妹|貓眼三姐妹]]》的神谷真人。<ref>{{cite web |title=5分で振り返る「シティーハンター」30年史 |url=https://woman.infoseek.co.jp/news/entertainment/dmenueiga_1258493 |website=楽天WOMAN |language=ja |date=2019/02/05}}</ref>因受到高度好評終於在1985年至1991年期間正式於[[集英社|集英社]]《[[週刊少年Jump|週刊少年Jump]]》長期連載。作者另一部漫畫《[[天使心_(北条司)|天使心]]》則是城市獵人的平行世界作品，只延用人物設定而非其續集<ref>《天使心》單行本第一期封面內頁作者聲名</ref>。

== 角色 ==
; 冴羽獠
: 配音：[[神谷明|神谷明]]（日本）；[[魏伯勤|魏伯勤]](中視)／[[梁興昌|梁興昌]](Animax)（臺灣）；[[張炳強|張炳強]]（香港）
{{main|冴羽獠}}
: 男主角，以[[新宿|新宿]]為根據地活動的城市獵人。
: 中國大陸曾有译名「寒羽良」，[[港台盜版時期|港台盜版時期]]譯名「孟波」<ref>由於早期台灣出版商未授权就擅自出版，再加上當時[[蔣經國政府|蔣經國政府]][[反日|反日]]意識強烈，出版商為了避嫌而用了譯者“孟波”的名字來代替原名</ref>

; 槙村香
: 配音：[[伊倉一惠|伊倉一惠]]（日本）；[[馮友薇|馮友薇]](中視)／[[傅其慧|傅其慧]](Animax)（臺灣）；[[袁淑珍|袁淑珍]]（香港）
: 槇村秀幸的妹妹，1965年3月31日出生。在兄長槇村秀幸被殺後，替上其兄長本來為獠拍擋的位置。她的長相和打扮較為中性，經常被當作是男性或[[變性人|變性人]]，是唯一不會令獠勃起的女性。同時亦是令獠無法防禦攻擊的人，善用鐵槌。
: 後來得到海坊主的傳授其設置陷阱的功夫，利用其鐵槌及陷阱設置，多次阻止好色的的獠與委託者中的女性發生關係。对獠怀有恋愛的感情，但常因他的好色行为而气愤的使用铁锤。在高二時因跟蹤兄長而曾與獠見面，一直以為獠沒有認出她是當年的那位小朋友，但是在最後一卷中終於知道其實獠從一開始就知道了。
: 香繼承了槙村的[[史密斯威森M36左輪手槍|史密斯威森M36左輪手槍]]小型左輪手槍，裝彈量5發。但因獠不想香成為沾滿血腥的殺手，故把槍改成怎樣也打不中目標，因此香的槍法甚差。
: 香亦十分著緊獠的生意，如沒有生意時，很多時亦會替獠找尋生意並做宣傳，有些委託人不合獠的心意並不接會接受，但香亦都會代替獠保護委託人，可是香的技術並不及獠並經常因此遭受險境，但獠往往亦會在香的背後暗中保護並且解決她所遇上困難。
: 其實香並非槙村的親生妹妹，只是槙村的刑警父親在追捕疑犯時帶回來的孩子。真正名字是久石香，有一個親生姐姐立木小百合（因父母離婚後用母親姓氏），父親是意外身亡的殺人疑犯久石純一。兩姐妹手上都帶著母親遺留的戒指。小百合本想和香相認，並帶她到美國；但最終因不想破壞香和獠的生活而放棄。
: 中国大陸曾译做「牧村香」，[[港台盜版時期|港台盜版時期]]譯名「林惠香」。角色名「香」取自原作者北条司的責任編輯[[堀江信彥|堀江信彥]]當時剛出生的女兒之名。

; 槙村秀幸
: 配音：[[田中秀幸|田中秀幸]]（日本）；[[李景唐|李景唐]](Animax)（臺灣）；[[林國雄|林國雄]]（香港）
: 香的兄長。獠的好友和拍檔。原與野上冴子是[[警視廳|警視廳]]的拍檔。因在一次破獲人蛇集團的行動中手下女警十多人全體被殺而引咎辭職，從此和冴羽獠搭檔城市獵人；其工作除確認委託人的委託之外，還有情報的收集。在最後因拒絕幫派「樂生會」拉攏而慘遭殺害，臨死前將香託付給冴羽獠。
: [[港台盜版時期|港台盜版時期]]譯名「林仲村」。原作漫畫中本來並未設定槙村的名字，後因動畫版配音員田中秀幸的名字而採用之。 

; 野上冴子
: 配音：[[麻上洋子|麻上洋子]]（日本）；[[黃麗芳|黃麗芳]]（香港）
: [[警視廳|警視廳]]的刑警，性感妖艷的美女，聰明能幹身手矯健，也能善用自己的美色破案，故得外號「警視廳的狐狸精」，暗戀冴羽獠。個性看似是個獨立女強人，其實是隱藏渴望被愛溫柔小女人的偽裝，出勤時大腿內側一定暗藏數把飛刀供危難時急用。與獠熟識，為了偵查刑事重案，時常私底下以一夜情作為條件誘惑獠協助破案，平常沒事也願獻身供獠洩慾或毛手毛腳，但都會被阿香阻止。父親是警視廳總監，為了冴子總是不滿意他安排的相親對象而煩惱不已，冴子還有兩個妹妹：野上麗香、野上唯香。
: [[港台盜版時期|港台盜版時期]]譯名「汪美君」。

; 海坊主
: 配音：[[玄田哲章|玄田哲章]]（日本）；[[馮永和|馮永和]]／[[陳永信|陳永信]]（香港）
: 本名是伊集院隼人，是在實力與獠不相伯仲的同行，因曾是傭兵游擊隊，因此善於佈置陷阱及游擊戰術，在戰爭中被服用天使塵的獠打傷視覺神經，因此在漫畫後期失去視覺，並轉行成為貓眼咖啡店的店主，但是身手依舊俐落。雖被獠奪走視覺，但對獠並沒有恨意，認為這是戰爭給他的傷痕，並非獠給他的傷痕。而「海怪」此名字亦是獠為他而設。海坊主面對女性是會滿面通紅，並不能說話，是極為害羞的。精於各方面技巧的海坊主，只有害怕貓這一個弱點，幼貓尤其嚴重（包括貓的聲音）。
: [[港台盜版時期|港台盜版時期]]譯名「海怪」。

; 美樹
: 配音：[[小山茉美|小山茉美]]（日本）；[[孫明貞|孫明貞]]／[[雷碧娜|雷碧娜]]（香港）
: 海坊主的女朋友。在小時候因地方戰亂成為孤兒，被海坊主收養後成為職業傭兵；繼而單戀海坊主。因為海坊主害羞，所以蹉跎而分離。為了追求海坊主，美樹追到日本東京並繼續要求結婚。海坊主因希望美樹能遠離槍彈並過著普通女人的生活，故提出美樹襲擊冴羽獠成功才能認可美樹的戰爭技藝並結婚的條件。在過程中海坊主為其真情感動而公開接納美樹，雖還是害羞的沒有答應結婚的要求但認可美樹成為其夥伴。事件結束後，美樹開設貓眼咖啡店。海坊主隨後立即協助美樹主持店內吧台服務。隨著兩人感情日漸穩定，加上海坊主視力漸漸惡化，美樹與海坊主兩人的生活重心在後期多集中在貓眼咖啡廳。貓眼咖啡廳也因此成為漫畫系列後期的重要場景。在最後一卷中，於眾人見証下與海坊主結婚。

; 米克·安吉爾（MICK ANGEL）
: 配音：[[成田劍|成田劍]]（日本）；[[馮錦堂|馮錦堂]]（香港）
: 獠在美國時的好友兼搭擋，各方面與獠不相伯仲，但缺點是和獠一樣的好色。漫畫劇情中在第32卷登場，受海原神委託來殺獠，最後因愛上香而在決鬥中認輸。回美國途中被海原神炸毀飛機擒獲，並被注射天使毒品成為對付獠的傀儡士兵，最後因對香的愛而恢復神智，但卻雙手殘廢不能再拿槍。最終與和江成為一對戀人。動畫版則為暗殺費力丹王國公主而故意接近獠；但任務失敗最後與獠決鬥中身亡。

; 海原神/堅尼
: 配音：動畫未登場（日本）；[[梁耀昌|梁耀昌]]（香港）
: 樂生會的長老，亦是獠的養父。雖然初期與獠關係良好，但到後期因戰爭的持續，使其良知逐漸地被抹滅，開始主張以天使塵來做出無敵的軍隊，並以獠為實驗對象，因而被游擊隊驅逐，其後成為樂生會的一員，之後威脅米克必須殺死獠。米克任務失敗後，海原將米克乘坐的飛機炸毀，並親自來到獠的家警告獠。最後與獠在船上決戰，被獠一槍射穿心臟身亡。他是《城市獵人》中獠最後的大敵。

== 漫畫 ==
《城市獵人》漫畫单行本全35卷由集英社出版，但完整收錄全系列（含連載版原型的初期短篇）的「完全版」則改由[[德間書店|德間書店]]發行。台灣中文版單行本由集英社授權[[時報文化|時報文化]]率先出版，後來改由[[東立出版社|東立出版社]]出版共35卷。東立版第1～3卷是[[Coamix|Coamix]]授權發行，第4～35卷是[[North_Stars_Pictures|North Stars Pictures]]授權發行。

=== 電影 ===
;《[[城市獵人_(香港電影)|城市獵人]]》

;《[[孟波_(香港電影)|孟波]]》（Mr.Mumble）
:由於《城市獵人》在香港的受歡迎程度，香港藝人都希望開拍這套漫畫的電影版。當時集英社指定讓[[成龍|成龍]]開拍電影版，但被觀眾指成龍太肥，身高跟書裡的孟波有很大距離，以及未能演出主角的精髓，并且与原著内容差异太大。反而另一套由[[周文健|周文健]]開拍的《[[孟波|孟波]]》，雖因沒買版權而不能使用漫畫原名，但大多數觀眾都認為他和飾演海坊主(海怪)的[[吳康寧|吳康寧]]更能演活這個漫畫角色。

;《[[九一神鵰俠侶|九一神鵰俠侶]]》
:劉德華開設的[[天幕製作|天幕製作]]旗下首部作品，架構結合了《城市獵人》、《[[神鵰俠侶|神鵰俠侶]]》兩部作品中的多個元素。不過除了人物關係與設定部分相仿外，同原作在劇情上并沒有多大關聯。<ref>當時劉德華曾cosplay孟波為《城市獵人》動畫作宣傳，并為動畫演唱了主題曲。</ref>
