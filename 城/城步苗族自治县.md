'''城步苗族自治县'''是[[湖南省|湖南省]][[邵阳市|邵阳市]]下辖的[[自治县|自治县]]，中国五个苗族自治县之一，位于湖南西南边缘、邵阳市西南部，毗邻[[广西|广西]]。全县面积2620平方公里；2009年总人口263336人，常住人口239515人。<ref name="2009年统计公报"/> 城步经济欠发达，是[[国家级贫困县|国家级贫困县]]<ref name="扶贫开发">{{cite web |title=邵阳城步县加速推进扶贫开发工作 |publisher=西部网 |url=http://news.cnwest.com/content/2011-02/17/content_4140183.htm}}</ref>；城步的旅游资源丰富，2010年获“中国最具影响力文化旅游百强县”荣誉称号<ref>{{cite web |title=城步成中国文化旅游百强县 |publisher=红网 |url=http://hn.rednet.cn/c/2011/01/29/2171110.htm}}</ref>。

== 历史 ==
[[隋|隋]]末，[[萧铣|萧铣]]据邵阳置[[建州|建州]]，设[[武攸县|武攸县]]，治今城步县城儒林镇，为城步置县之始。

[[唐|唐]][[武德|武德]]四年（621年），唐平萧铣，改武攸为[[武冈县|武冈]]，隶[[南梁州|南梁州]]。[[贞观|贞观]]十年，改南梁州为[[邵州|邵州]]。

[[北宋|北宋]]初年，武冈县移治于今[[武冈市|武冈市]]城关镇，[[熙宁|熙宁]]八年（1075年）原县治改置城步寨，“城步”之名始于此。

[[崇宁|崇宁]]二年（1103年）析武冈、[[绥宁|绥宁]]地置'''城步县'''，治城步寨，隶[[邵州|邵州]]。

[[崇宁|崇宁]]五年（1106年）改邵州武冈县设置武冈军。城步县隶[[武冈军|武冈军]]。

[[清|清]][[乾隆|乾隆]]三年至六年（1738年～1741年）城步一度改隶[[靖州|靖州]]。

民国二年（1913年）隶[[湘江道|湘江道]]，民国十一年直隶于省，民国二十六年属第六行政督察区。

1949年10月14日，城步县为[[中国人民解放军|中国人民解放军]]占据，隶属邵阳区督察专员公署，后隶属邵阳专员公署、邵阳地区行政公署。

1956年11月30日，撤销城步县建制，建立城步苗族自治县，系全国第二个[[苗族|苗族]][[自治县|自治县]]。

1986年1月27日，邵阳撤地设市，城步苗族自治县隶属邵阳市。

== 地理 ==
城步位于[[雪峰山脉|雪峰山脉]]与[[南岭山脉|南岭山脉]]交汇之处，[[沅水|沅江]]支流[[巫水|巫水]]上游，地处湖南、[[广西|广西]]二[[省|省]][[自治区|区]]交界地。相邻[[县|县级政区]]有，西临[[通道侗族自治县|通道侗族自治县]]，北连[[绥宁县|绥宁县]][[武冈市|武冈市]]，东接[[新宁县|新宁县]]，南和[[广西|广西]][[龙胜各族自治县|龙胜各族自治县]]、[[资源县|资源县]]接壤。<ref name="湖南省地图册">{{Cite book |title=湖南省地图册 |publisher=星球地图出版社 |year=2008  |isbn=978-7-80212-365-6}}</ref>

最高山峰为[[二宝鼎|二宝鼎]]，海拔2021米；其它较高的山峰有[[南山顶|南山顶]]，海拔1940米；[[牛坡头|牛坡头]]，海拔1913米；[[金紫山|金紫山]]，海拔1883.1米；[[黔峰山|黔峰山]]，海拔1678米。<ref name="湖南省地图册"/>

城步有三大水系：[[巫水|巫水]]、[[资水|资水]]、[[浔江|浔江]]。最大的湖泊为人工湖——[[白云水库|白云水库]]。<ref name="湖南省地图册"/>

==人口和语言==
2009年总人口263336人，常住人口239515人。超过半数为[[苗族|苗族]]人，约40%为[[汉族|汉族]]人，其余还散居有[[侗族|侗族]]等少数民族。城步的主要语言有[[湘语|湘语]]（娄邵片）和[[苗语|苗语]]（[[猫家话|猫家话]]）。

== 教育 ==
2009年末有各级各类学校187所，在校学生27288人，其中[[普通中学|普通中学]]25所，在校学生9053人；[[小学|小学]]159所，在校学生17516人。<ref name="2009年统计公报"/>

== 卫生 ==
