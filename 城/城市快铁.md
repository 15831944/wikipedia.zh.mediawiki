'''城市快铁'''（{{lang-de|S-Bahn}}）是一种连接[[市中心|城市中心]]及[[近郊|近郊]]地区的快捷[[铁路|铁路]]系统，主要应用于[[德国|德国]]、[[奥地利|奥地利]]、[[瑞士|瑞士]]、[[丹麦|丹麦]]等歐洲國家。

== 释义及历史 ==
[[File:S-Bahn_train_leaving_Friedrichstrasse_station.jpg|225px]]）]]
[[File:Karte_S-Bahnnetze_in_Deutschland.svg|225px]]路網圖]]
S-Bahn这个词是[[德语|德语]]「城市快捷铁路」（{{lang|de|Stadtschnellbahn}}）的缩写，并于1930年12月首先在[[柏林|柏林]]引入。当时，作为近郊[[通勤列车|通勤列车]]线路的{{link-en|柏林城市鐵路|Berlin Stadtbahn}}與（{{lang|de|Berlin Stadtbahn}}）{{link-en|柏林環狀鐵路|Berliner Ringbahn|環狀鐵路}}（{{lang|de|Berliner Ringbahn}}）分别在1928和1929年完成电气化（[[第三轨|第三轨]]供电，750伏），S-Bahn便成为这类电气化线路的通称。此后，其他通往近郊的放射状铁路也陆续电气化，并被改造成快速交通工具：发车间隔不多于20分钟，同时许多线路在市中心共用同一条主线。该铁路系统在1936年[[1936年夏季奥林匹克运动会|柏林奥运]]期间达到顶峰时，发车间隔少于2分钟。

这种将重轨火车应用于都市快速交通在当时并不仅限于柏林。[[汉堡|汉堡]]在1908年就已经完成了[[汉堡火车总站|火车总站]]和[[阿尔托纳区|阿尔托纳]]之间的铁路电气化（采用的[[接触网|接触网]]供电，之后才改为第三轨供电）。1934年，汉堡的这套系统开始采用与柏林同样的名称S-Bahn。同年，[[哥本哈根|哥本哈根]]的「S-tog」开通了第一条线路。[[维也纳|维也纳]]的城铁主线也于1908年完成电气化，自1954年以來称为「快速铁路」（Schnellbahn），但是正式改名为S-Bahn要到2005年。在1930年代，德国开通了许多S-Bahn，但通常只是将电气化的区域铁路改了一下名称而已。

== 定义==
S-Bahn是一种服务于城市中心以及近郊和附近乡镇交通的铁路系统。它的标准因城市而异，但是一个共通的特征是高效和同调的发车时间（通常每条线路的发车间隔不少于20分钟，但是汇合到主线的部分只有2、3分钟）。在大城市中，这意味着对各条轨道的充分利用。在城市中心部分，一般都采用地下或者高架轨道。

== 与地铁的比较 ==
S-Bahn的意思為「城市快捷鐵路」，也就是中文所稱的[[通勤铁路|通勤铁路]]或[[区域铁路|区域铁路]]。中國一些城市的通勤铁路，和德奧諸國一樣，以S開頭加數碼標識路綫。截至2014年8月，已開通了[[南京|南京]]的[[宁高城际轨道交通|S1]]號綫、[[宁天城际|S8]]號綫和[[北京|北京]]的[[北京市郊铁路S2线|S2]]號綫。
德國城市快鐵叫S-Bahn，地鐵叫U-Bahn。很多大城市如[[科隆|科隆]]、[[杜塞爾多夫|杜塞爾多夫]]同時有城市快鐵S-Bahn和地鐵U-Bahn。
