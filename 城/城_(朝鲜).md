[[File:남한산성.jpg|250px]]的朝鲜城之一]]
'''朝鲜式城'''是朝鲜文化中传统的城堡形式{{r|KOCIS}}。

朝鲜的城分为数类：山城、邑城、平地城、关城、[[千里长城|长城]]、栅城{{r|高句丽史|page1=309|民族21}}。朝鲜式的城大量使用岩石修筑（石城），并且以利用山丘地形修建的山城为多，体现了一种以岩石为建材的传统{{r|车勇杰|page1=36|UNESCO}}，受到汉文化影响之后，才在高句丽中后期出现了采用泥土作为建筑材料的土城{{r|孙玉良|page1=130,241}}。

目前，在朝鲜半岛有约2,400座山城，其中[[忠清北道|忠清北道]]有212座{{r|KOCIS|UNESCO}}。

== 历史 ==
朝鲜式的城起源于[[高句丽|高句丽]]，传播到[[百济|百济]]和[[新罗|新罗]]{{r|车勇杰|page1=33}}，此后的[[高丽王朝|高丽王朝]]和[[朝鲜王朝|朝鲜王朝]]继承了这一传统{{r|UNESCO}}。朝鲜，尤其是历史上的高句丽{{r|民族21|Jeong|梁时恩}}，常常被称为“山城之国”{{r|KOCIS|UNESCO|文化财厅|page3=65|KF}}。

高句丽古城目前已发现约170座，分布区域包括朝鲜半岛、中国{{r|Jeong}}和[[蒙古国|蒙古]]{{r|김운회|成宇济|경향신문}}境内，而中国境内有99座{{r|魏存成|page1=70-91}}。其中在战争史上较著名的是位于今[[辽宁省|辽宁省]][[鞍山市|鞍山市]]和[[海城市|海城市]]的[[安市城|安市城]]，在[[唐太宗征讨高句丽|唐朝侵略高句丽的战争]]中，高句丽军队曾在此击退了[[唐朝|唐朝]]的进攻{{r|Kim Li-na|page1=100|Kim Jinwung|page2=50}}。

== 影响 ==
7世纪至8世纪，日本[[大和朝廷|大和朝廷]]的[[古代山城_(日本)|古代山城]]是由百济人及后裔建造或在其指导下建造的{{r|UNESCO|Turnbull|page2=26}}。

== 现状与保护 ==
{{Main article|华城|南汉山城|高句丽王城、王陵及贵族墓葬|百济历史遗迹地区}}
位于韩国的[[华城|华城]]{{r|华城}}和[[南汉山城|南汉山城]]{{r|南汉山城}}被[[联合国教科文组织|联合国教科文组织]]认定为[[世界遗产|世界遗产]]。位于中国的[[丸都山城|丸都山城]]与[[国内城|国内城]]、[[五女山城|五女山城]]作为[[高句丽王城、王陵及贵族墓葬|高句丽王城、王陵及贵族墓葬]]的一部分被认定为世界遗产。位于韩国的[[公山城|公山城]]、[[扶苏山城|扶苏山城]]、[[扶余罗城|扶余罗城]]作为[[百济历史遗迹地区|百济历史遗迹地区]]的一部分被认定为世界遗产。

[[凤凰山山城|凤凰山山城]]被指定为[[中国全国重点文物保护单位|中国全国重点文物保护单位]]{{r|文保}}。
