'''城桐央'''（{{lang|ja|しろきり ちか}}）是[[日本|日本]]的自由游戏剧本作家。

== 概要 ==
在2007年7月27日发售的[[Key_(公司)|Key社]]作品[[Little_Busters!|Little Busters!]]中，城桐央首次创作游戏剧本，担任了两条角色路线的剧本作家。<ref name="LB">{{Cite web|url=http://key.visualarts.gr.jp/product/little/products.htm |script-title=ja:リトルバスターズ！製品内容 |author=[[Key_(公司)|Key]] |date= |work=Little Busters!官方网站 |publisher=[[Visual_Art's|Visual Art's]] |language=ja |accessdate=2012年4月29日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20110518025600/http://key.visualarts.gr.jp/product/little/products.htm |archivedate=2011年5月18日 }}</ref>Key社于2008年7月25日发售Little Busters!的成人向化作品[[Little_Busters!|Little Busters! EX]]，其中城桐央除了前作的两条角色路线外，还担任了一条新增加的角色路线的剧本作家。由于上述两作中城桐央负责的能美·库特莉亚芙卡路线很受玩家欢迎，Key社于2010年6月25日发售了库特线的后传作品[[Kud_Wafter|Kud Wafter]]，剧本由城桐央创作。<ref name = "KW">{{Cite web |url= http://key.visualarts.gr.jp/kudo/spec.html |script-title= ja:クドわふたー｜Key Official HomePage |author= [[Key_(公司)|Key]]|date= |work= Kud Wafter官方网站|publisher= [[Visual_Art's|Visual Art's]]|language= ja|accessdate= 2012年4月29日|quote=  }}</ref>

城桐央还曾经担任两首Little Busters!的角色音乐<ref name = "LB Music">{{Cite web |url= http://key.soundslabel.com/ksl_html/ksl0049/ksl0049.html |script-title= ja:『リトルバスターズキャラクターソング"raison / ピクルスをおいしくするつくりかた"』 |author= [[Key_Sounds_Label|Key Sounds Label]]|date= |work= Key Sounds Label HomePage|publisher= [[Visual_Art's|Visual Art's]]|language= ja|accessdate= 2012年4月29日|quote=  }}</ref>和两首Kud Wafter的乐曲<ref name = "KW Music">{{Cite web |url= http://key.soundslabel.com/ksl_html/ksl0077/ksl0077.html |script-title= ja:『クドわふたー Original SoundTrack』|author= [[Key_Sounds_Label|Key Sounds Label]]|date= |work= Key Sounds Label HomePage|publisher= [[Visual_Art's|Visual Art's]]|language= ja|accessdate= 2012年4月29日|quote=  }}</ref>的作词。

城桐央除了创作自己参与创作的游戏的官方小说和外传小说外，还热衷于[[同人|同人]][[小说|小说]]的创作，先后在网络上连载了数部与Little Busters!相关的小说<ref name = "WN"/>，他的一篇同人小说收录在了[[Comic_Market|C80]]发售的一本同人小说集里<ref name ="SS"/>。
