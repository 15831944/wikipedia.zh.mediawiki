'''城大動物醫療中心'''({{lang-en|CityU Veterinary Medical Centre}})位於[[深水埗|深水埗]][[茘枝角道|茘枝角道]]339號，是一所亞洲最具規模的[[動物醫學|動物醫學]][[教學醫院|教學醫院]]、[[公共衞生|公共衞生]]及動物醫療機構。

== 歷史==
中心前身為1984年成立的「太平道寵物診所 」({{lang-en|Peace Avenue Veterinary Clinic}})（原為麥子健寵物診所)，是香港首間由本地人開設的獸醫診所。
2016年9月1日，中心由[[香港城市大學|香港城市大學]]全權擁有以配合教研發展，2019年遷往深水埗現址並改名為「城大動物醫療中心」<ref>{{cite web|url=http://www.peaceavevet.com.hk/tc/about/us|title=城大太平道動物診所}}</ref><ref>{{cite web|url=https://newscentre.cityu.edu.hk/zh-hant/media/news/2016/08/31/cityu-acquires-prestigious-veterinary-clinic|title=城大購入知名獸醫診所}}</ref>。

== 設施 ==
中心佔地三層，面積達3.3萬平方呎，設有全港首個動物深切治療部、心臟專科中心、22間診症室、9間手術室，及由註冊急症科專科獸醫管理、提供全年無休的24小時急症服務和專業的全科獸醫診治。

== 專科 ==
中專科獸醫團隊涵蓋八大專科-急症、[[外科|外科]]、[[麻醉|麻醉]]科、[[神經內科|神經內科]]、[[皮膚科|皮膚科]]、[[心臟科|心臟科]]、[[眼科|眼科]]、[[內科|內科]]，[[深切治療|深切治療]]科及[[腫瘤|腫瘤]]專科，另外更提供[[針灸|針灸]]治療。

== 動物醫學教研 ==
中心為香港城市大學「賽馬會動物醫學及生命科學院」提供案例及相關[[公共衛生|公共衛生]]研究<ref>{{cite web|url=https://newscentre.cityu.edu.hk/zh-hant/media/news/2019/03/28/cityu-veterinary-medical-centre-opens|title=城大動物醫療中心投入運作　配合動物醫學及生命科學院發展}}</ref>，亦為各地[[獸醫|獸醫]]學生提供實習機會。
