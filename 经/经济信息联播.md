'''经济信息联播'''是[[中国中央电视台|中国中央电视台]]的一档财经新闻节目，于1992年8月31日开播，1996年间停播，2002年7月30日复播。该节目现每天晚上20:30在[[中国中央电视台财经频道|中国中央电视台财经频道]]播出，每集大约45分钟<ref>{{Cite web |url=http://finance.sina.com.cn/roll/20021101/1507274019.html |title=《经济信息联播》栏目简介 |date=2002-11-01 |publisher=新浪财经 |accessdate=2015-01-05}}</ref><ref>{{Cite web |url=https://baike.baidu.com/item/经济信息联播|title=《经济信息联播》|date=|publisher=百度百科 |accessdate=2019-02-22}}</ref>。

== 播出时间 ==
*首播：中国中央电视台财经频道：每天20:30-21:15
*重播（2015年5月1日前）：中国中央电视台财经频道：周一至周四：23:42-次日00:33；周六：00:12-00:58；周日：00:10-00:56、23:39-次日00:30
*重播（2015年6月1日至2017年末）：中国中央电视台财经频道：周一至周日凌晨
*2018年起，本节目取消重播时段。
