'''经验主义'''（{{lang-en|Empiricism}}）又作'''經驗論'''，通常指相信现代[[科学方法|科学方法]]，相信证据，着重认为[[理论|理论]]应建立于对于事物的观察，而不是[[直觉|直觉]]或[[迷信|迷信]]。意即通过实验研究而后进行理论归纳优于单纯的逻辑推理。它最重要的科学方法包括：所有理论和假设都必须被实验来检验，而不是单纯且唯一地依赖于先验推理，直觉和启示。所有[[經驗主義|經驗主義]]最早的發源地是在[[歐陸|歐陸]]([[歐洲大陸|歐洲大陸]])，主要是發源於[[歐洲大陸|歐洲大陸]]的[[古希臘|古希臘]]時代的[[城邦國家|城邦國家]]的[[醫學|醫學]]專業知識及專業的[[醫療|醫療]]技術，[[經驗主義|經驗主義]]是當時[[古希臘|古希臘]][[城邦|城邦]][[國家|國家]]的醫生看診治療病人的方法。经验主义的代表人物有[[希波克拉底|希波克拉底]]、[[托马斯·阿奎纳|托马斯·阿奎纳]]、[[弗兰西斯·培根|弗兰西斯·培根]]、[[托马斯·霍布斯|托马斯·霍布斯]]、[[约翰·洛克|约翰·洛克]]、[[乔治·贝克莱|乔治·贝克莱]]和[[大卫·休谟|大卫·休谟]]。

经验主义，经常被自然科学家提及说：「知识是建立在实验之上的。」认知论原理之一就是感官体验创造了知识。经验主义的研究，包括实验和经过验证的测量工具。正是这些领导着[[科学方法|科学方法]]。

与经验主义相对的是[[欧洲|欧洲]][[理性主义|理性主义]]，认为哲学应经由思考和[[演绎推理|演绎推理]]而得出结论，代表人物有[[笛卡尔|笛卡尔]]、[[斯宾诺莎|斯宾诺莎]]、[[莱布尼茨|莱布尼茨]]、[[康德|康德]]。

“经验主义”这个词有许多不同的用法，但有两种用法最为突出。首先，它用来表示研究现实的一般方法，即只有通过经验和感官获得的知识才是可接受的。换句话说，这一立场意味着思想在被认为是知识之前必须经受严格的测试。这个词的第二个意思与此相关，指的是一种信念，即“事实”的积累本身就是一个合理的目标。第二种含义有时被称为“朴素经验主义”。（Alan Bryman 社会研究方法第四版）

== 与科学的关系 ==
经验主义是[[逻辑实证主义|逻辑实证主义]]的前身。直到今天，经验主义的方法还在影响[[自然科学|自然科学]]，是[[自然科学|自然科学]]研究方法的基础。而[[自然科学|自然科学]]方法则是传统观念的发展。然而近几十年来，一些新的理论学说，例如[[量子力学|量子力学]]，[[构成主义|构成主义]]，[[托马斯·库恩|托马斯·库恩]]的《科学革命的结构》对经验主义在科学研究的方法上独一无二的地位产生了轻微的冲击。

== 哲學上的關係 ==
[[經驗主義|經驗主義]]一詞原本意指[[古希臘|古希臘]][[醫生|醫生]]的經驗，拒絕一味接受當代的宗教教條，而是依照所觀察到的現象為分析依據。[[經驗主義|經驗主義]]首先在十七世紀由[[英國人|英國人]][[洛克|洛克]]系統性的闡述。[[洛克|洛克]]主張人的心智原本是一块[[白板_(哲学)|白板]]（{{lang-la|tabula rasa}}），而是經驗註記其上。[[經驗主義|經驗主義]]否定了人擁有與生俱來的知識的觀點或不用藉由經驗就可以獲得的知識。值得注意的是，[[經驗主義|經驗主義]]並不主張人們可以從實務中自動地取得知識。根據[[經驗主義|經驗主義]]者的觀點，經由感受到的經驗，必須經過適當[[歸納|歸納]]或[[演繹|演繹]]，才能鑄成知識。在哲學發展上，[[經驗主義|經驗主義]]一直和[[理性主義|理性主義]]作為對比。[[理性主義|理性主義]]認為大部分的知識是來自感覺上的獨立思考。無論如何，這種對比已被視為過於簡單化，因為近代的[[歐陸|歐陸]]和[[不列顛群島|不列顛群島]]([[英國|英國]])的理性學者也倡导利用[[科學方法|科學方法]]去取得實際經驗；而以[[洛克|洛克]]為首的[[英國|英國]]學者和許多[[歐陸|歐陸]]學者也認為超自然的知識（如宗教神學）必須單獨藉由直覺或推理才能取得。

== 历史 ==

=== 背景 ===
主题：[[觀察研究|觀察研究]]

在科学和科学方法中最重要的中心理念就是：必须''实证地''基于感官证据。[[自然科学|自然科学]]和[[社会科学|社会科学]]都是通过[[观察|观察]]和[[实验|实验]]来检验[[假说|假说]]的。而''半实证的''时常被用来描述使用[[公理|公理]]，已被建立的科学理论，或者是之前实验的结果的理论方法。以此来进行理性的模型建构和理论探究。

哲学实证家认为没有知识是可以被推导的——除非它是从一个人感官经验所总结而来。这种观点普遍被认为是与[[理性主义|理性主义]]——认为知识是可以从理智中得来，而并不只是从感官中获得——相对立的。{{啟蒙時代}}
