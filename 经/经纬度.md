'''經緯度'''是'''[[經度|經度]]'''與'''[[緯度|緯度]]'''的合稱組成一個[[座標系統|座標系統]]。又称为'''地理座標系統'''，它是一種利用三度空間的球面來定義[[地球|地球]]上的空間的[[球面坐標系統|球面坐標系統]]，能夠標示地球上的任何一個位置。

== 經度 ==
{{Main|經度}}

[[經度|經度]]是[[地球|地球]]上一個地點離一根被稱為[[本初子午線|本初子午線]]的南北方向走線以東或以西的度數。[[本初子午線|本初子午線]]的[[經度|經度]]是0°，[[地球|地球]]上其它地點的[[經度|經度]]是向東到180°或向西到180°。不像[[緯度|緯度]]有[[赤道|赤道]]作為自然的起點，[[經度|經度]]沒有自然的起點而使用經過[[倫敦|倫敦]][[格林尼治天文臺|格林尼治天文臺]]旧址的[[子午線|子午線]]作為起點。東經180°即西經180°，約等同於[[國際日期變更線|國際日期變更線]]，國際日期變更線的兩邊，日期相差一日。

== 緯度 ==
{{Main|緯度}}

[[緯度|緯度]]是指某點與[[地球|地球]]球心的連線和[[地球|地球]][[赤道|赤道]]面所成的線面角，其數值在0至90度之間。位於[[赤道|赤道]]以北的點的[[緯度|緯度]]叫北緯，記為N，位於[[赤道|赤道]]以南的點的[[緯度|緯度]]稱南緯，記為S。

== 海拔 ==
{{Main|海拔}}

[[海拔|海拔]]是指某地點與海平面的高度差，是現時量度一個地方的高度標準。

== 經緯度表示及轉換 ==
經緯度以度數表示，一般可直接以小數點表示，但亦可把度數的小數點分為[[角分|角分]]（1角分等於六十分之一度），和秒（一秒等於六十分之一分）

# 度分秒表示（度:分:秒）－49°30'00"-123d30m00s
# 度分表示（度:分）－49°30.0'-123d30.0m
# 度數表示－49.5000°-123.5000d（一般會有四位小數）。

不少軟件可把不同的經緯度表示方式轉換。

== 大地测量与经纬度 ==
由于地面高低的不同和地球形状的不正规，天体测量所得的信息（如[[卫星导航系统|卫星导航系统]]）不足用于明确地计算地理位置。以下是更详细的经纬度系统：
* [[WGS84|WGS84]]，被美国[[全球定位系统|全球定位系统]]使用
* [[GRS80|GRS80]]
