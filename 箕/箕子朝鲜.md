'''箕子朝鲜'''（{{lang-ko|기자조선}}），根据《[[史记|史记]]》、《[[尚书大传|尚书大传]]》、《[[汉书|汉书]]》等史书的说法，是在公元前11世紀到前2世紀期間由商朝宗室[[箕子|箕子]]及中原地区移民在朝鲜半岛与当地原住民[[东夷人|东夷人]]一起建立的一个国家。但是也有学者认为该国原为箕氏侯国，在辽西的[[喀左县|喀左县]]一带，与[[孤竹国|孤竹国]]相邻；在[[战国时代|战国时代]]为[[燕国|燕国]]将领[[秦开|秦开]]所败，其国君被迫率百姓遁入朝鲜半岛。<ref>[http://lywsw.cn/article/lsxp/200811/20081121144044.Html 邱华东] {{webarchive|url=https://web.archive.org/web/20101209032651/http://www.lywsw.cn/article/lsxp/200811/20081121144044.Html |date=2010-12-09 }}</ref>

== 史料记述 ==
[[File:Han_Dynasty.jpg|thumb]]
“朝鲜”这一名字最早出现于《[[尚书大传|尚书大传]]》之中。根据《尚书大传·卷二》记载：“武王释箕子之囚，箕子不忍周之释，走之朝鲜。武王闻之，因以朝鲜封之箕子。”《尚书大传》还记载：“西方者何也？鲜方也。”

周武王灭商之后，把被纣王囚禁的大臣[[箕子|箕子]]（又名'''胥餘'''，[[子姓|子姓]]<ref name=G155g>見[[首爾大學|首爾大]][[奎章閣|奎章閣]]電子圖書館的[http://e-kyujanggak.snu.ac.kr/GDS/GDS_VIEW.jsp?ptype=list&subtype=wy&bookid=GR36071_00IB&volnum=1&articleid=003&dtype=2 文本紀錄]</ref>）从监狱中释放。箕子不愿臣服于周朝，率商朝遗民出走东方的“朝鲜”。周武王听说之后，便以“朝鲜”之地封之，成为朝鲜侯。

在之后的《[[史记|史记]]》、《[[漢書|漢書]]》中也記載了箕子出走朝鲜的事件。箕子教朝鮮人民礼儀、[[耕织|耕织]]农耕及养蚕，还带入了大量青铜器。受[[商朝|商朝]]文化的教化下，還定下“相杀以当时偿杀；相伤以穀偿；相盗者男没入为其家奴，女子为婢，欲自赎者，人五十万。”的法律条文（出自《乐浪-{朝}-鲜民犯禁八条》），大大地改善了-{朝}-鮮人民的生活。


根据《汉书·地理志》记载，箕子的“八条之教”的全部内容如下：

#“相杀，以当时偿杀”。
#“相伤，以穀偿”。
#“相盗者，男没入为其家奴，女子为婢，欲自赎者，人五十万”。
#“妇人贞信”。
#“重山川，山川各有部界，不得妄相干涉”。
#“邑落有相侵犯者，辄相罚，责生口、牛、马，名之为‘责祸’”。
#“同姓不婚”。
#“多所忌讳，疾病死亡，辄捐弃旧宅，更造新居”。


中国史料中还记载着[[朝鲜|朝鲜]]后来称王，被[[燕国|燕国]]将领[[秦开|秦开]]侵略而丧失了两千多里土地，从此一蹶不振的事件。《[[三国志|三国志]]·魏志·东夷传》注引《[[魏略|魏略]]》中记载：“昔箕子之后朝鲜侯，自称为王，后子孙稍骄虐，燕乃遣将秦开，攻其西方，取地两千余里，至满番汗为界，朝鲜遂弱”。

《[[史记|史记]]·朝鲜列传》以及《[[后汉书|后汉书]]》记载了[[箕子朝鲜|箕子朝鲜]]被[[衛滿朝鮮|衛滿朝鮮]]篡夺的历史事件：[[西漢|西漢]]初年，[[燕國|燕]]王[[盧綰|盧綰]]背叛[[漢朝|漢朝]]，逃往[[匈奴|匈奴]]。其臣[[衛滿|衛滿]]亦一同出走，並帶同千餘人進入朝鮮。朝鮮王[[準王|箕準]]信任[[衛滿|衛滿]]，让他守卫西部边疆。但[[卫满|卫满]]却发兵篡位，占据了[[箕子朝鲜|箕子朝鲜]]都城[[王險城|王險城]]，箕準率領殘部数千人泛海逃往马韩，破之，自立为韩王。

《[[史记|史记]]》卷一百一十五{{〈}}朝鲜列传{{〉}}第五十五“集解”引张晏-{云}-：“朝鲜有湿水、洌水、汕水，三水合为洌水，疑乐浪朝鲜取名于此也。”索隐-{云}-：“朝音潮，直骄反，鲜音仙。以有汕水，故名也。汕一音讪。”中国古籍《[[山海经|山海经]]·海内经》曰：“东海之内，北海之隅，有国名朝鲜”。

== 对箕子朝鲜的考证 ==
昔日东亚文化圈曾长期认为箕子朝鲜在朝鲜半岛境内。《[[舊唐書|舊唐書]]》卷199上便记载后世的[[高句丽|高句丽]]“頗有箕子之遺風”，“其俗多淫祀，事靈星神、日神、可汗神、箕子神”。<ref>[[:s:舊唐書/卷199上|《舊唐書》-{卷}-一百九十九上]]</ref>显然是把高句丽当作了箕子朝鲜的传承国。

韩国古代学者们普遍认为箕子将先进文明从中国传入朝鲜半岛，“我国教化礼仪自箕子始”，对他推崇备至。<ref>《[[高丽史|高丽史]]》肃宗七年（1102年）十月壬子朔：“礼部奏：我国教化礼仪，自箕子始，而不载祀典。乞求其坟茔，立祠以祭。”</ref>然而，根据《[[旧唐书|旧唐书]]·东夷传》的记载，辽东之地，[[周朝|周朝]]时期为箕子之国，[[汉朝|汉朝]]时期为[[玄菟郡|玄菟郡]]。而《隋书·裴矩传》则记载，高丽之地原为[[孤竹国|孤竹国]]，周朝以其地封箕子，汉朝分为三个郡，[[晋朝|晋朝]]亦统治辽东。<ref name="隋書卷67"/>这几段箕子朝鲜在辽东的历史记载曾让[[朝鲜王朝|朝鲜王朝]]时期的学者产生很大的困惑。<ref name="tan121"/>

一部分考古发现表明被认为是箕子封地的朝鲜（今辽东、北朝鲜）琵琶型的铜剑在外形和金属成分上与同期中国其他地区的类似武器有所不同，但[[礼器|礼器]]、生活器具则大体相当。<ref>[http://www.artron.net/artronlist/bowuguan/liaolin/ll1.php 辽宁省博物馆] {{webarchive|url=https://web.archive.org/web/20090423130613/http://www.artron.net/artronlist/bowuguan/liaolin/ll1.php |date=2009-04-23 }}</ref><ref>{{cite web |url=http://www.artron.net/artronlist/bowuguan/liaolin/ll1.php |title=存档副本 |accessdate=2009-04-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20090423130613/http://www.artron.net/artronlist/bowuguan/liaolin/ll1.php |archivedate=2009-04-23 }}</ref>

有一种观点认为箕子朝鲜的疆界是随着时间推移而变动的，起初与[[孤竹国|孤竹国]]一同处于辽西地区，有辽西出土的殷晚期青铜器中，方鼎内底中心有铭文“亚侯”为佐证。后来由于燕国的强大而退居辽东和北朝鲜。<ref>[http://lywsw.cn/article/lsxp/200811/20081121144044.Html 邱华东] {{webarchive|url=https://web.archive.org/web/20101209032651/http://www.lywsw.cn/article/lsxp/200811/20081121144044.Html |date=2010-12-09 }}</ref> 箕子朝鲜后被[[卫满朝鲜|卫满朝-{}-鲜]]所灭。[[汉武帝|汉武帝]]后又灭了卫满朝鲜。

== 箕子朝鲜存在的真实性争议 ==
在封建时代，朝鲜半岛统治者一直十分推崇箕子，认为箕子将文明带入了朝鲜。不过进入20世纪初期，箕子朝鲜存在的真实性受到朝鲜学者的广泛质疑。而箕子朝鲜的存在始终得不到任何考古学的证明，因此国际学术界普遍否定箕子朝鲜的存在。<ref name="tan121">《韩国檀君神话研究》，张哲俊著，北京大学出版社2013年出版，121~124页（ISBN 978-7-301-23118-0）</ref>

《[[論語|論語]]》微子第十八中记载箕子与[[微子|微子]]、[[比干|比干]]合称「殷有三仁」，《[[今本竹書紀年|今本竹書紀年]]》殷紀中记载[[纣王|纣王]]五十一年“冬十一月戊子，周師渡孟津而還。王囚箕子，殺王子比干，微子出奔。”但记载箕子的这两本古书并没有提及箕子从中国迁徙到朝-{}-鲜或者箕子建立朝鲜的描述。

1908年，[[申采浩|申采浩]]发表《[[读史新论|读史新论]]》一文，可谓开此说之先驱。<ref name="Shim 278">Shim, 2002, p.278.</ref>申采浩是一位[[朝鲜纯血主义|朝鲜纯血主义]]者，他认为朝鲜民族是[[檀君|檀君]]的后裔，与箕子没有任何关系。<ref name="schmid33">Schmid, 1997, p.33-34.</ref>箕子逃到朝鲜后，被[[夫餘国|夫餘国]]王降为附庸，仅仅给予他一小块领地以生存。<ref name="schmid33"/>此后，[[崔南善|崔南善]]和[[李丙燾|李丙燾]]二人也发现，先秦史料与之后中韩史料中的矛盾。<ref name="shim278">Shim, 2002, pp. 278–79.</ref>

1945年，[[日本投降|日本投降]]以后，朝鲜半岛分为北方的[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]和南方的[[大韩民国|大韩民国]]。两韩的官方观点不约而同地认为箕子朝鲜是[[伪史|伪史]]，是[[西汉|西汉]]为了合理[[汉四郡|吞并朝鲜半岛]]而凭空捏造出来的故事。<ref>Shim, 2002, pp. 279–80.</ref>朝鲜官方的历史书籍都没有将箕子朝鲜列在朝鲜历史年表之中。<ref>{{ko icon}}{{zh-hans icon}}[http://naenara.com.kp/ch/history/period.php {{lang|ko|조선력사 시대구분표}} - 朝鲜历史时代划分表] - [[Naenara|Naenara]]</ref>[[金日成|金日成]]也认为箕子入朝鲜是西汉“为了借口侵略[[古朝鲜|古朝鲜]]”而“凭空捏造”出来的事件，“把[[朝鲜民族|朝鲜民族]]看成箕子的后裔是对具有五千年悠久历史的我们民族的侮辱。”金日成更是在1959年下令拆毁了平壤的[[箕子陵|箕子陵]]，而根据朝鲜官方的说法，在箕子陵中没有尸骨，也没有任何文物。1973年，韩国考古学家金貞培宣称，在朝鲜半岛境内没有发现过任何先秦时期的青铜器。<ref name="shim278"/>{{fact|不过，韩国认为箕子朝鲜与[[檀君朝鲜|檀君朝鲜]]并存，箕子朝鲜建国于朝鲜半岛极西端的一角。}}

韩国历史学家[[尹乃铉|尹乃铉]]在其所著的《韩国古代史新论》中承認箕子朝鲜的存在，并认为箕子来源于中国中原<ref>[http://books.google.com/books?id=yW9Lyom56T4C&pg=RA1-PA27&lpg=RA1-PA27&dq=%E6%B8%8A%E5%87%80%E5%9C%9F&source=bl&ots=BQE8_irzZm&sig=T3XG3kcutvUew8SO6Se3CL-Ee-E&hl=en&ei=AaEpSuydHZ7aswOI6LSnCw&sa=X&oi=book_result&ct=result&resnum=5#PRA1-PA8,M1 韓國歷史與現代韓國 By Jiang-zuo Jian]</ref>。

对于箕子、卫满迁居并占据朝鲜，以及[[準王|準王]]（箕準）占据[[马韩|马韩]]的记载，在日韩国史学家[[金两基|金两基]]认为：
:三韩的领域和国境后世记载非常模糊和粗略，国境并非互不可干涉的界线，而是一个缓冲地带。根据《三国志·魏书东夷传·辰韩条》中便记载昔日不堪秦朝徭役的大量中国人逃到[[辰韩|辰韩]]，辰韩王割让东方国境地带的土地给他们居住，说明了这一点。此流民居住之地自然产生了有实力者。具有王和君长的经历，所以参加了权力斗争，自然成为了支配者。因此卫满篡夺準王的地位没有什么奇怪的。<ref>金両基《物語　韓国史》中公新書　47頁〜48頁</ref>

与朝鲜半岛学者相对的是，有许多中国学者认为箕子朝鲜是实际存在的，更有甚者，認為檀君朝鮮是韓民族主義为了“强调国史”而「杜撰」的。<ref>[http://phtv.ifeng.com/phoenixtv/72999905567703040/20061211/907524.shtml 中国边疆史学争议频发]</ref>

根据《[[旧唐书|旧唐书]]·东夷传》的记载，辽东之地，[[周朝|周朝]]时期为箕子之国，[[汉朝|汉朝]]时期为[[玄菟郡|玄菟郡]]。<ref>《[[:s:舊唐書/卷199上|旧唐书·列传第一百四十九·东夷·高丽]]》：“侍中裴矩、中书侍郎温彦博曰：‘辽东之地，周为箕子之国，汉家玄菟郡耳。’”</ref>而《隋书·裴矩传》则记载，高丽之地原为[[孤竹国|孤竹国]]，周朝以其地封箕子，汉朝分为三个郡，[[晋朝|晋朝]]亦统治辽东。<ref name="隋書卷67">《[[:s:隋書/卷67|隋书·卷六十七列传·第三十二·裴矩]]》：“矩因奏状曰：‘高丽之地，本孤竹国也。周代以之封于箕子，汉世分为三郡，晋氏亦统辽东。今乃不臣，别为外域。’”</ref>这几段箕子朝鲜在辽东的历史记载曾让[[朝鲜王朝|朝鲜王朝]]时期的学者产生很大的困惑。<ref name="tan121"/>

1955年，[[辽西|辽西]]地区[[凌源县|凌源县]]海岛营子村村民唐永兴、张怀仁等发掘出16件商周时期的[[青铜器|青铜器]]，现藏于[[辽宁博物馆|辽宁博物馆]]。这批青铜器的特征与同时期中原的青铜器类似，而此时的东北地区没有达到铸造此类青铜器的水平，说明了商朝人曾经移居到辽西一带。1973年春，考古学家在辽宁省[[喀喇沁左翼蒙古族自治县|喀喇沁左翼蒙古族自治县]]北洞村发掘出刻有“{{僻字|㠱|上己下其}}侯”名字的青铜器方鼎，“{{僻字|㠱|上己下其}}”即为“箕”，商朝[[甲骨文|甲骨文]][[卜辞|卜辞]]中有“{{僻字|㠱|上己下其}}侯、王其”，而该方鼎与[[安阳|安阳]]出土的“父己”方鼎形制相近，证实是商末周初的方鼎。<ref name="tan121"/>同年，又在同一个县发现了六件[[中国青铜器|青铜器]]，其中一件铜[[罍|罍]]（[[酒器|酒器]]）上有“[[父丁|父丁]]、孤竹、[[亚微|亚微]]”的铭文，表明该铜罍是孤竹国国君亚微为父丁铸造的。箕侯和孤竹国君的青铜器同时期出现在辽西地区，印证了“箕子朝鲜原为孤竹国”的说法。

但中国学者[[张哲俊|张哲俊]]则认为，箕子及其后人一直居住在辽西的说法恐怕也不一定正确，因为箕子朝鲜的最后一代国君[[準王|準王]]定都于[[王险城|王险城]]（可能即是今天的[[平壤|平壤]]）。可能箕子朝鲜受到中原诸侯国的侵略，被迫向东迁徙，来到朝鲜半岛，后来又被[[卫满朝鲜|卫满朝鲜]]驱逐到辽東半岛的最南端。<ref name="tan121"/>
