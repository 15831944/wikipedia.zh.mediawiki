'''鰻魚'''是'''鰻鱺目'''（ 学名：Anguilliformes）分類下的[[物種|物種]]總稱，包含20[[科_(生物)|科]]111[[屬|屬]]總共大約800[[物種|種]]。部分鰻魚和[[鮭魚|鮭魚]]一樣具有-{[[洄游|洄游]]}-特性。

另外有些魚類如[[八目鰻|八目鰻]]、[[電鰻|電鰻]]、[[黃鱔|黃鱔]]等，長相類似鰻魚而容易被誤認，但在分類上屬於完全獨立於鰻魚外的[[演化支|演化支]]。

==特征==
[[File:Conger_conger_Gervais.jpg|thumb]]為最重的鰻魚。]]
鰻魚的外型類似長條[[蛇|蛇]]狀的棍棒形，長度最小為{{convert|5|cm|abbr=on}}（[[單頜鰻屬|單頜鰻屬]]），最長則可至{{convert|4|m|abbr=on}}（[[長尾彎牙海鱔|長尾彎牙海鱔]]）。<ref name=EoF>{{cite book |editor1=Paxton, J.R. |editor2=Eschmeyer, W.N. |author= McCosker, John F. |year=1998|title=Encyclopedia of Fishes|publisher= Academic Press|location=San Diego|pages= 86–90|isbn = 0-12-547665-5}}</ref>成年體重範圍大約在{{convert|30|g|abbr=on}}至超過{{convert|110|kg|abbr=on}}。[[腹鰭|腹鰭]]退化，部分種類缺乏[[胸鰭|胸鰭]]，[[胸鰭|胸鰭]]、[[臀鰭|臀鰭]]與[[尾鰭|尾鰭]]則完全相連。<ref name=FB>{{FishBase order|order= Anguilliformes |year=2009|month=January}}</ref>

大部分的鰻魚居住在[[淺海|淺海]]，潛伏在沙中、泥中或是礁石當中，多為[[夜行性|夜行性]]。也有部分鰻魚生活在[[深海|深海]]中，深度可達{{convert|4000|m|abbr=on}}。[[鰻鱺科|鰻鱺科]]為迴游性魚類，生活於淡水但會游回海中產卵。<ref name=EoG>{{cite book |author= Prosek, James |year=2010|title=Eels: An Exploration|publisher= HarperCollins |location=New York|isbn = 978-0-06-056611-1}}</ref>

最重的鰻魚為[[歐洲康吉鰻|歐洲康吉鰻]]，最高體重{{convert|110|kg|abbr=on}}，體長則可達{{convert|3|m|abbr=on}}。<ref>[http://www.fishbase.org/summary/speciessummary.php?id=301 Conger conger, European conger: fisheries, gamefish, aquarium]. Fishbase.org</ref>其他鰻魚體重較輕，但擁有更長的身長，例如[[長尾彎牙海鱔|長尾彎牙海鱔]]最大體型就可達{{convert|4|m|abbr=on}}。<ref>[http://fishbase.mnhn.fr/Summary/speciesSummary.php?ID=8595&genusname=Strophidon&speciesname=sathete&lang=French FishBase] {{webarchive|url=https://web.archive.org/web/20131110144455/http://fishbase.mnhn.fr/Summary/speciesSummary.php?ID=8595&genusname=Strophidon&speciesname=sathete&lang=French |date=10 November 2013 }}. FishBase (2011-11-15).</ref>

== 分佈 ==
全世界的鰻魚主要生長於熱帶及溫帶地區水域，除了[[歐洲鰻|歐洲鰻]]（{{lang|en|''Anguilla anguilla''}}）及[[美洲鰻|美洲鰻]]（{{lang|en|''Anguilla rostrata''}}）分布在[[大西洋|大西洋]]外，其餘均分布在[[印度洋|印度洋]]及[[太平洋|太平洋]]區域。

== 生活史 ==
鰻魚在陸地的河川中生長，成熟後洄游到海洋中產卵地產卵，一生只產一次卵，產卵後就死亡。這種生活模式，與[[鮭魚|鮭魚]]的[[溯河洄游|溯河洄游]]（{{lang|en|anadromous}}）相反，稱為[[降河洄游|降河洄游]]（{{lang|en|catadromous}}）。在溯河的過程中，鰻苗必須穿過許多障礙物，例如[[堰|堰]]、[[堤壩|堤壩]]或是天然[[瀑布|瀑布]]。現今有許多河道設有[[魚道|魚道]]，可以幫助這些魚在不耗盡體力的前提下溯河。

其生活史分為6個不同的發育階段，為了適應不同環境，不同階段的體型及體色都有很大的改變：
* 卵期（egg-stage）：位於深海產卵地。
* 柳葉鰻（{{lang|en|leptocephalus}}）：在大洋隨[[洋流|洋流]]長距離漂游，此時身體扁平透明，薄如柳葉，便於隨波逐流。以[[海洋雪|海洋雪]]為主食。
* 玻璃鰻（glass eel）：在接近沿岸水域時，身體轉變成流線型，減少阻力，以脫離強勁洋流。
* 鰻線（elvers）：進入河口水域時，開始出現黑色素，卻也形成養殖業鰻苗的捕捉來源。
* 黃鰻（yellow eel）：在河川的成長期間，魚腹部呈現黃色。
* 銀鰻（silver eel）：在成熟時，魚身轉變成類似深似深海魚的銀白色，同時眼睛變大，胸鰭加寬，以適應迴游至深海產卵。

== 菜餚 ==
[[File:Tokyo_Chikuyotei_Unadon01s2100.jpg|240px]]，又稱鰻丼]]
鳗鱼口感軟嫩，富含不饱和脂肪酸，对于软化血管降低血脂极其有利。

[[日本菜|日本菜]]經常使用淡水鰻（-{うなぎ}-; ''unagi''）及海水鰻（[[康吉鰻|康吉鰻]]；{{lang|en|Conger eel}}, ''anago''）。但一般鰻魚都不會作刺身及生食，這是由於鰻魚血液中含有毒性蛋白，必須經過加熱烹煮才能分解。如著名的日本握壽司「[[星鰻|星鰻]]壽司」即在蒸熟之後以噴燈炙(或鐵網)烤數秒鐘之後方能上盤，其他鰻魚多半以[[蒲燒|蒲燒]]形式烹調，搭配[[米飯|米飯]]製成[[鰻魚飯|鰻魚飯]]。而日本人有於土用丑日吃鰻魚的習俗，始於江戶時代安永年間。根據傳說，某生意不太好的鰻魚店店主，因銷情未如理想，於是想出於店頭張貼「本日丑之日」，說吃了很保健康，結果銷情大好。此後，其他鰻魚店也依樣模仿，也逐漸成為一種習慣<ref>[https://bkb.mpweekly.com/cu0003/20180720-78583 你不能不懂的食物市場學 關於鰻魚的土用丑日]</ref>。

[[粵菜|粵菜]]及[[上海菜|上海菜]]也常使用鰻魚。

[[韓國|韓國]]使用鰻魚膽汁加燒酒，{{fact|對男人有強身健體的功效，對女人有美顏瘦身的功效。|date=April 2018}}

[[臺灣|臺灣]]會將[[鯙科|鯙魚]]去皮後[[油炸|油炸]]、[[蒸|清蒸]]或[[紅燒|紅燒]]食用。

在歐美及其他地方則食用[[歐洲鰻|歐洲鰻]]及其他淡水鰻。一道傳統的[[倫敦|倫敦]]菜即為[[鱔魚凍|鱔魚凍]]（{{lang|en|jellied eels}}）。

== 類似魚種 ==
因為分子生物學的進步，始魚的先祖分化關係變得明朗，也就造成分類的不同。
* [[背棘魚科|背棘魚科]]（{{lang|en|Notacanthiformes}}）中的[[刺鰻|刺鰻]]（{{lang|en|spiny eel}}）及（[[海蜥魚科|海蜥魚科]]）也非常像鰻魚。這些深海魚過去認為屬於[[北梭魚目|北梭魚目]]（{{lang|en|Albuliformes}}）的亞目。
* [[電鰻|電鰻]]（{{lang|en|electric eel}}）並非鰻魚，而與[[鯰魚|鯰魚]]（{{lang|en|catfish}}）較為相近。
* [[黃鱔|黃鱔]]與[[泰國鱔|泰國鱔]]像鰻魚，卻是[[合鰓目|合鰓目]]淡水底棲魚類。
* [[八目鰻|八目鰻]]（或稱七鰓鰻）與[[盲鰻|盲鰻]]也很像鰻魚，但牠們都是[[無頜綱|無頜綱]]的魚類。

== 其他 ==
在[[歐洲|歐洲]]衹有[[英國|英國]]可以用合法的手擲網方式捕鰻魚。數千年來手擲網是在英國[[柏瑞河|柏瑞河]]（{{lang|en|River Parrett}}）與[[塞文河|塞文河]]（{{lang|en|River Severn}}）實行的捕鰻魚方法。
