'''劍魚'''或稱'''劍旗魚'''，是一種大型的掠食性魚類，為[[輻鰭魚綱|輻鰭魚綱]][[鱸形目|鱸形目]][[鯖亞目|鯖亞目]]'''劍旗魚科'''[[劍旗魚屬|劍旗魚屬]]的唯一一個物種<ref>[http://ci58.lib.ntu.edu.tw/cpedia/Content.asp?ID=19428&Query=3 劍魚]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }} - [[中國大百科智慧藏|中國大百科智慧藏]]</ref>。

== 分布 ==
此魚分佈於全球的[[熱帶|熱帶]]和[[溫帶|溫帶]]海域。

== 深度 ==
水深0-650公尺。

== 特徵 ==
本魚體似旗魚，但吻部為平扁的劍狀突出，且顯著延長。齒僅見於幼魚，成長後完全消失。成魚無鱗，僅胸部以下面有粗糙之感。側線不發達，尾柄兩側各有1個堅強的突起稜脊。背鰭2枚，第一背鰭基底甚短，呈三角形，佔體軀前部1/3，第二背鰭短小，偏魚體之後端。臀鰭2枚，以第一臀鰭較發達。尾鰭深分叉。全長可超過5公尺，體重可達500公斤。<ref>[http://www.soa.gov.cn/kepu/yu/xc39.htm 劍魚] {{webarchive|url=https://web.archive.org/web/20070704163356/http://www.soa.gov.cn/kepu/yu/xc39.htm |date=2007-07-04 }}</ref>。

== 生態 ==
劍魚以[[烏賊|烏賊]]和[[魚類|魚類]]為食。雖然體型龐大，但其游速可達每小時100公里，是海中游速最高的魚類之一，每年8-9月產卵，性情兇猛，常以其吻攻擊大型水生生物，如鯨魚。

== 經濟利用 ==
為食用魚，肉白略帶淡紅色，肉質柔軟脂肪多，是做[[生魚片|生魚片]]的好材料，亦可做成旗魚羹、魚丸等物品，也是著名的遊釣魚，不過要小心體內汞的累積問題，不宜長期食用。
