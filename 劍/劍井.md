'''劍井'''又稱'''[[國姓井|國姓井]]'''，是一個位於[[台中市|台中市]][[大甲區|大甲區]][[鐵砧山|鐵砧山]]上的古井的名稱。在劍井的夜景「鐵砧晚霞」為[[全臺十二勝景之一|全臺十二勝景之一]]。

井的直徑約三尺，水深僅一尺左右，井水清澈見底，甘美清冽。

== {{or|傳說}} ==
{{or|早在[[漢朝|漢朝]]就有這類的[[劍泉|劍泉]]傳說。

[[明朝|明朝]][[永曆|永曆]]十六年{{可疑}}農曆[[五月初四|五月初四]]{{可疑}}、近[[中華端午節|端午]]節日，[[延平郡王|延平郡王]][[鄭成功|鄭成功]]的部將率兵越海北上，由大安港南側登陸，被道卡斯平埔族包圍，水源斷絕，只好向天祝禱：「天若助我鄭軍部隊，平安歸去，此寶劍插地，必得甘泉」。<ref> {{cite web |url=http://design92-1.town-all.org.tw/view88/html/five1.html |title=存档副本 |accessdate=2007-05-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20070320062046/http://design92-1.town-all.org.tw/view88/html/five1.html |archivedate=2007-03-20 }}</ref>，然後拔劍插地，泉水竟然就在劍峰兩邊不斷地湧出來，飢渴得以解決。

端午節遊客到此汲取[[午時水|午時水]]，相傳可治百病，且在午時紛紛在此[[立蛋|立蛋]]。

在該井旁有石碑，上鐫刻碑記如下：「台北府新竹縣大甲鐵砧山國姓井，相傳鄭成功駐兵處被圍困於山頂缺乏水源，部將以寶劍插地得甘泉。之後大旱不涸，年年清明前，有群鷹自鳳山來聚哭，不至疲憊不止。或云：兵魂固結而成，後山麓田螺斷尾能活。謂當時螺殼棄置者均著奇異，僕曾經其地思古蹟不可磨滅爰集同人建廟立碑為記，以誌久遠-{云}-。」
<ref>[http://www.ttvs.cy.edu.tw/kcc/9593ta/sw1.htm]</ref>

清[[光緒|光緒]]18年，立碑稱此古井為「國姓井」，相傳部將把[[國姓爺|國姓爺]]的寶劍插入地後，此劍每年端午才會浮出。民國42年(1953年)，[[于右任|于右任]]所題「劍井」兩字的石碑，[[賈景德|賈景德]]為劍井作記。從此處上登階可抵「[[開台聖王祠|開台聖王祠]]」，也可往上步行至「[[台中軍人忠靈祠|台中軍人忠靈祠]]」。}}
