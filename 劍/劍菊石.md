'''劍菊石'''（[[學名|學名]]：{{lang|la|'''''Xipheroceras'''''}}）是生存在早[[侏羅紀|侏羅紀]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]。其化石分布於[[英國|英國]]等地。<ref name="Fossilworks">{{Cite web |url = http://fossilworks.org/bridge.pl?a=taxonInfo&taxon_no=15815 |title = Fossilworks: Xipheroceras |language = en |accessdate = 2018-08-13 }}</ref>

== 特徵 ==
劍菊石的外觀和[[原微菊石|原微菊石]]十分相似。其單一不分岔的肋條呈[[翼|翼]]狀，近腹側有一尖銳的突起，是兩者的不同之處。<ref name="Xipheroceras1">{{Cite web |url = http://liusir.pixnet.net/blog/post/248637560-xipheroceras-%E5%8A%8D%E8%8F%8A%E7%9F%B3-%28%E8%8B%B1%E5%9C%8B1%29 |title = Xipheroceras 劍菊石 (英國1) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-13 }}</ref><ref name="Xipheroceras2">{{Cite web |url = http://liusir.pixnet.net/blog/post/248636837-xipheroceras-%E5%8A%8D%E8%8F%8A%E7%9F%B3-%28%E8%8B%B1%E5%9C%8B%EF%BC%8C%E5%BA%A7%E7%8B%80%E6%AF%8D%E5%B2%A9%29 |title = Xipheroceras 劍菊石 (英國，座狀母岩) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-13 }}</ref>
