'''頌安（Chung On）'''是香港[[沙田區議會|沙田區議會]]下轄的[[選區|選區]]，代號R24，於2003年設立，現任區議員為[[工黨_(香港)|工黨]][[葉榮|葉榮]]。

== 範圍 ==
選區範圍為[[頌安邨|頌安邨]]、[[聽濤雅苑|聽濤雅苑]]、[[觀瀾雅軒|觀瀾雅軒]]及[[天宇海|天宇海]]<ref>{{cite web|url=http://www.eac.gov.hk/pdf/distco/maps/dc2015r.pdf |title=區議會選區分界圖—沙田區 |publisher=[[選舉事務處|選舉事務處]] |date=2015-11-12 |accessdate=2015-12-13}}</ref>，與其相鄰的選區有[[錦濤|錦濤]]、[[耀安|耀安]]、[[恆安_(選區)|恆安]]和[[鞍泰|鞍泰]]。

==沿革==
由於[[錦泰苑|錦泰苑]]2000年落成，加上[[錦濤_(選區)|錦豐]]、[[鞍泰|恒濤]]選區發展成熟，人口增加，選舉管理委員會2002年改劃原有[[錦濤_(選區)|錦豐]]、[[鞍泰|恒濤]]選區為'''頌安'''、[[錦濤|錦濤]]及[[鞍泰|鞍泰]]三個選區。頌安選區包括原錦豐選區內的[[頌安邨|頌安邨]]、恒濤選區內的[[聽濤雅苑|聽濤雅苑]]及[[觀瀾雅軒|觀瀾雅軒]]。當中以頌安邨人口較多，選區名稱亦由此而來。

[[2003年區議會選舉|2003年區議會選舉]]，頌安選區由[[民主黨_(香港)|民主黨]][[方鎮邦|方鎮邦]]和時任[[錦濤_(選區)|錦豐]]區議員[[民建聯|民建聯]][[陳克勤|陳克勤]]爭奪，最終方鎮邦勝出<ref>{{cite web|url=http://www.legco.gov.hk/yr03-04/chinese/panels/ca/papers/ca0419cb2-2010-c.pdf |title=二零零三年區議會選舉報告書 |publisher=選舉管理委員會 |date= |accessdate=2015-12-13}}</ref>。方鎮邦在2006年因桃色事件被迫退出民主黨，轉投[[自由黨_(香港)|自由黨]]，並缺席多次會議<ref>{{cite news|url=http://the-sun.on.cc/channels/news/20060515/20060515023541_0000.html|title=	
前民主黨區議員 上月起缺席會議 方鎮邦涉桃色失蹤|newspaper=[[太陽報_(香港)|太陽報]]|date=2006-05-15|accessdate=2015-12-13}}</ref>。2007年11月26日方鎮邦騙取區議會營運開支津貼罪成，被判囚四個月<ref>{{cite news|url=http://archive.news.gov.hk/isd/ebulletin/tc/category/lawandorder/071126/html/071126tc08003.htm|title=區議員方鎮邦騙津貼囚四月|date=2007-11-26|accessdate=2015-12-13|publisher=香港政府新聞網}}</ref>。根據《區議會條例》，方的議席自同日起被褫奪，但因[[2007年區議會選舉|正式選舉]]已於11月18日舉行，故沒有舉行補選。

[[2007年區議會選舉|2007年區議會選舉]]中，當年估計人口為20,653人<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/summaryr.html |title=區議會選舉2007 - 選區分界建議概要(沙田) |publisher=[[選舉事務處|選舉事務處]] |date=2012-06-19 |accessdate=2015-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20151222151848/http://www.elections.gov.hk/dc2007/chi/summaryr.html |archivedate=2015-12-22 }}</ref>，選民人數9,387人<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/num_electors_sha_tin.html |title=區議會選舉2007 - 選舉簡介 |publisher=[[選舉事務處|選舉事務處]] |date=2012-06-19 |accessdate=2015-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20151222085702/http://www.elections.gov.hk/dc2007/chi/num_electors_sha_tin.html |archivedate=2015-12-22 }}</ref>。民建聯[[葛珮帆|葛珮帆]]出選頌安選區，其對手為民主黨[[勞潤明|勞潤明]]，結果葛珮帆以2,379票當選。2011年區議會選舉中，當年估計人口為21,288人<ref>{{cite web|url=http://www.elections.gov.hk/dc2011/chi/summaryr.html |title=2011年區議會選舉 - 選區分界概要 |publisher=[[選舉事務處|選舉事務處]] |date=2011-09-05 |accessdate=2015-12-13}}</ref>，選民人數為11,067人<ref>{{cite web|url=http://www.elections.gov.hk/dc2011/chi/ebriefsr.html |title=2011年區議會選舉 - 區議會選舉簡介 |publisher=[[選舉事務處|選舉事務處]] |date=2011-09-20 |accessdate=2015-12-13}}</ref>。葛珮帆競逐連任，對手為[[民主動力|民主動力]][[劉家儀|劉家儀]]，結果葛珮帆以2,455票連任。

[[2015年區議會選舉|2015年區議會選舉]]中，尋求再度連任的葛珮帆與[[殘疾人士|殘疾人士]]、[[工黨_(香港)|工黨]]的[[葉榮|葉榮]]爭奪議席。最後葉榮以2,506票擊敗取得2,376票的葛珮帆當選。葉榮本身為[[香港立法會|立法會]]議員[[張超雄|張超雄]]的助理，後來轉到[[馬鞍山_(香港市鎮)|馬鞍山]]地區辦事處工作<ref>{{cite news|url=http://www.inmediahk.net/1025572c|title=坐輪椅唔係弱勢！——專訪葉榮|publisher=[[香港獨立媒體|香港獨立媒體]]|accessdate=2015-12-14|date=2014-08-26|deadurl=yes|archiveurl=https://web.archive.org/web/20141225205554/http://www.inmediahk.net/1025572c|archivedate=2014年12月25日|df=}}</ref>，成為香港首位「輪椅社區幹事」<ref>{{cite news|url=http://www.inmediahk.net/node/10331244|title=【地區政治系列】工黨葉榮：黃絲帶唔易做|publisher=[[香港獨立媒體|香港獨立媒體]]|accessdate=2015-12-14|date=2015-04-07}}</ref>。他在2015年10月3日報名參選<ref>{{cite web|url=http://www.elections.gov.hk/dc2015/pdf/nomination/ShaTin_20151016_c.pdf|title=2015年區議會一般選舉提名—沙田區|publisher=選舉事務處|accessdate=2015-12-14}}</ref>，公佈選舉結果之前葉榮表示一直抱著「必輸」的心態出選<ref>{{cite news|url=https://hk.mobi.yahoo.com/finance/%E6%93%8A%E6%95%97%E6%B0%91%E5%BB%BA%E8%81%AF%E8%91%9B%E7%8F%AE%E5%B8%86%E5%B7%A5%E9%BB%A8%E8%91%89%E6%A6%AE%E5%85%8B%E6%9C%8D%E6%AE%98%E9%9A%9C%E6%86%91%E8%89%AF%E5%BF%83%E6%9C%8D%E5%8B%99%E8%A1%97%E5%9D%8A-010736313.html|title=擊敗民建聯葛珮帆工黨葉榮克服殘障憑良心服務街坊|work=香港電台|publisher=雅虎香港|date=2015-11-26|accessdate=2015-12-14}}</ref>。當選之後葉榮表示身為一個殘疾人士，選舉結果意義重大，是締造奇蹟。他亦呼籲其他殘疾人士站出來，為自己和香港人發聲<ref>{{cite news|url=http://www.inmediahk.net/node/10390825|title=工黨3人當選區議會 葉榮：每人皆可站出來發聲|publisher=香港獨立媒體|date=2015-11-23|accessdate=2015-12-14}}</ref>。他亦期望他的當選能改變社會對殘疾人士觀感<ref>{{cite news|url=http://news.tvb.com/local/5655d1af6db28c456f000007|title=葉榮當選區議員 盼改社會對殘疾人士觀感|date=2015-11-25|accessdate=2015-12-14|publisher=無綫新聞}}</ref>。

有網民在葛珮帆落選後在[[Facebook|Facebook]]成立「香港各界慶祝EQ撻Q委員會」專頁慶祝。11月24日約100人應網上號召，在其位處頌安邨辦事處舉行慶祝活動，期間參加者帶同香檳、花生<ref name=od>{{cite news|url=http://hk.on.cc/hk/bkn/cnt/news/20151124/bkn-20151124204016910-1124_00822_001.html|title=葛珮帆區選落敗百人贈興　圍觀市民中蛋|newspaper=東方日報|date=2015-11-24|accessdate=2015-12-14}}</ref>及[[格林威治大學_(諾福克島)|夏威夷格林威治大學]]仿製畢業證書<ref>{{cite news|url=http://news.memehk.com/posts/12249|title=區選落敗怪政府 葛珮帆：插針樓是致命傷|publisher=謎米香港|date=2015-11-25|accessdate=2015-12-14}}</ref>等物品，並唱出[[徐小鳳|徐小鳳]]歌曲《喜氣洋洋》以示慶祝。期間曾有參加者被雞蛋擲中但並無大礙<ref name=od />。
