'''劳尔·冈萨雷斯·布兰科'''（{{lang-es|'''Raúl González Blanco'''}}，{{bd|1977年|6月27日|catidx=Raul}}），簡稱'''劳尔'''，西班牙著名[[足球員|足球員]]。西班牙足球傳奇巨星之一。2004年，他位列球王[[比利|比利]]評選的[[FIFA_100|125位最出色的在世球星（FIFA 100）中]]。

劳尔1992年加盟皇马青年队，1994年起为皇马一队出战，夺得6次[[西甲联赛|西甲联赛]]冠军、3次[[欧洲冠军联赛|欧洲冠军联赛]]冠军、2次[[洲际杯足球赛|洲际杯]]冠军等荣誉，2010年7月劳尔离开皇马加盟德甲[[沙尔克04足球俱乐部|沙尔克04足球俱乐部]]。

== 早年经历 ==
劳尔·冈萨雷斯·布兰科在San Cristóbal de los Ángeles的Alevín隊開始足球生涯。1990年，他加入了[[马德里竞技|马德里竞技]]成為青訓球員，1992年當時會長基爾下令解散青訓，魯爾隨後加盟[[皇家馬德里|皇家馬德里]]青年隊。

== 皇家馬德里 ==
[[File:Raul_Gonzalez_10mar2007.jpg|thumb]]
[[File:Raul_Gonzalez_Supercopa_2008.jpg|thumb]]
1994-95，魯爾被當時的教練[[豪尔赫·巴尔达诺|豪尔赫·巴尔达诺]]提升至一隊比賽，成為當時[[西班牙足球甲级联赛|西甲联赛]]歷史上最年輕的出場球員（17歲04個月）。天生擁有足球天份的他立即成為隊中重要一員，第一季上陣28場共射入9球，協助球隊取得1994-95的西甲錦標。翌年他開始發揮可怕的射門能力，40場聯賽射入19球令他成為神射手第五名，該屆聯賽冠軍正是魯爾的母會馬德里體育會。1997年他與[[克羅地亞|克羅地亞]]前鋒[[达沃·苏克|达沃·苏克]]合作，協助球隊壓倒宿敵[[巴塞罗那足球俱乐部|巴塞罗那]]成為聯賽盟主。1998年他協助球隊重奪多年的[[欧洲冠军联赛|欧洲冠军联赛]]冠軍，其後他的入球為球隊多次奪冠，包括3次欧洲冠军杯冠軍，5次[[西班牙足球甲级联赛|西甲联赛]]冠軍及3次[[西班牙超級盃|西班牙超級盃]]冠軍，其它奖项还包括[[歐洲超級盃|歐洲超級盃]]和[[洲际杯足球赛|洲际杯]]等。皮里斯成為新任會長，並每年都招攬世界級球員加入球隊，[[費高|費高]]、[[施丹|施丹]]、[[朗拿度|朗拿度]]及[[贝克汉姆|贝克汉姆]]等一些球员的加盟使皇家马德里进入“银河舰队”时代。2003年，魯爾接替離隊的[[费尔南多·耶罗|费尔南多·耶罗]]成為隊長。

2005年9月28日，他在[[欧洲冠军联赛|欧洲冠军联赛]]對[[奥林匹亚克斯足球俱乐部|奥林匹亚克斯]]時頂入一球，打破了球會名將[[阿尔弗雷多·迪·斯蒂法诺|迪·斯蒂法诺]]41年前於欧冠联赛射入49球的紀錄，同時成為第一位在欧冠联赛射入50球的球員（共上陣97場）。

2009年2月15日，勞爾在對[[希洪競技|希洪競技]]的賽事中以正選上陣，而在比賽取得2個入球，超越了由名宿[[阿尔弗雷多·迪·斯蒂法诺|迪·斯蒂法诺]]所保持的307球，以309球成為皇馬史上入球最多的球員。

2009年2月21日，勞爾在對[[皇家貝蒂斯|皇家貝蒂斯]]的賽事中，又用2個進球創造了皇馬另一頁歷史。在本場比賽之後，勞爾共為皇馬攻入了218個西甲進球，超越了之前與其並列的[[阿尔弗雷多·迪·斯蒂法诺|迪·史提芬奴]]所保持的216球，獨自領跑皇馬歷史聯賽射手榜。然后2009年底，随着劳尔年龄渐增，西班牙有评论员文章，建议劳尔回马竞，并说马德里竞技才是他真正的家。<ref>http://www.as.com/futbol/articulo/raul-tiene-hecho-schalke/dasftb/20100704dasdaiftb_33/Tes</ref>

2010年7月26日，劳尔正式在马德里宣布离开西班牙转投德甲[[沙尔克04|沙尔克04]]，从此告别这家他个人之前唯一效力的球会，16年间劳尔在皇家马德里共取得了16个冠军的辉煌荣誉。而据称劳尔将会同沙尔克04签署为期2年的合约。<ref>[http://www.goal.com/cn/news/38/%E8%A5%BF%E7%8F%AD%E7%89%99/2010/07/26/2042787/%E5%8A%B3%E5%B0%94%E6%B7%B1%E6%83%85%E5%91%8A%E5%88%AB%E4%BC%AF%E7%BA%B3%E4%B9%8C-%E5%8D%81%E4%BA%94%E8%BD%BD%E4%BC%A0%E5%A5%877%E5%8F%B7%E4%BB%8A%E8%B0%A2%E5%B9%95 劳尔深情告别伯纳乌 十五载传奇7号今谢幕]</ref>

== 沙尔克04 ==
劳尔在沙尔克04的第一个赛季，联赛中出场34次攻入15球，2010年11月20日对阵[[云达不莱梅|云达不莱梅]]德甲比赛中，劳尔上演了在[[沙尔克04|沙尔克04]]的第一个帽子戏法，帮助球队4:0击败对手。该季[[欧洲冠军联赛|欧洲冠军联赛]]1/4决赛对阵[[国际米兰|国际米兰]]的比赛，劳尔在两回合各入一球。同时球队也以7-3总比分血洗国际米兰，历史性地闯入欧冠四强。同时劳尔带领沙尔克04夺得了该赛季[[德国杯|德国杯]]的冠军。

2011年11月19日，劳尔取代了受伤的[[贝尼迪克特·赫韦德斯|赫韦德斯]]成为沙尔克04的新队长。

2012年4月，魯爾向外界宣布，賽季結束後會離開[[史浩克04|史浩克04]]，告別[[歐洲|歐洲]]球壇。<ref>{{cite news|author = 球迷世界|date = 2012-04-24|language = zh-hk|publisher = Fanpiece|title = 魯爾宣布告別歐洲球壇！一齊回顧金童18年足球生涯|url = http://www.fanpiece.com/football-fans-world/c1056508.html|deadurl = yes|archiveurl = https://web.archive.org/web/20120426234219/http://www.fanpiece.com/football-fans-world/c1056508.html|archivedate = 2012-04-26}}</ref>之后不久，沙尔克04俱乐部宣布将劳尔效力期间所穿的7号球衣永久封存。但2013年俱乐部重新启用了这一号码。

== 國家隊生涯 ==

1994年時劳尔·冈萨雷斯·布兰科在皇馬的突出表現，令他入選西班牙18歲以下國家隊，1996年曾隨國家隊出戰[[1996年夏季奥林匹克运动会|亚特兰大奥运会]]。同年10月9日，魯爾披上10號球衣首次代表國家隊出戰[[捷克國家足球隊|捷克]]，其後一直作为國家隊核心球員。1998年，他協助球會奪得欧洲冠军杯冠軍，随后跟隨國家隊首次參加國際足球大赛——[[1998年世界盃足球賽|1998年世界杯]]。西班牙的首场比赛中，劳尔凭借他的妙射帮助球队一度领先[[尼日利亞國家足球隊|尼日利亞]]，可惜最終對手連入兩球，以3-2逆转西班牙获胜。西班牙随后战平[[巴拉圭国家足球队|巴拉圭]]，又大胜[[保加利亚国家足球队|保加利亞]]，但可惜小组排名第三而遭到淘汰。

[[2000年欧洲足球锦标赛|2000年欧锦赛]]，劳尔再次带领球队出战，在1/4决赛遭遇[[法国国家足球队|法国]]。上半场，法国队由[[齐达内|齐达内]]打进一个直接任意球，随后[[盖兹卡·门迭塔|盖兹卡·门迭塔]]為西班牙追平，但德约卡夫的进球使法国队再度领先。比赛接近尾声时，西班牙获判点球，但由劳尔操刀射失，令西班牙再次黯然出局。[[2002年世界杯足球赛|2002年世界杯]]，魯爾在賽事中共射入3球，但西班牙因一个具争议性的入球被主辦國[[南韓國家足球隊|南韓]]淘汰出局。世界盃後魯爾接替退役的[[费尔南多·耶罗|费尔南多·耶罗]]成為西班牙國家隊新任隊長。

2005年10月12日，他為國家隊在[[2006年世界盃外圍賽|世界盃外圍賽]]對[[圣马力诺国家足球队|圣马力诺]]时出场，打破了前國家隊後衛[[费尔南多·耶罗|费尔南多·耶罗]]的非門將（最高為上陣126場的[[安东尼·苏比萨雷塔|安东尼·苏比萨雷塔]]）最高出场次数纪录，至[[2006年世界杯|2006年世界杯]]時，西班牙被法國淘汰出局。此后劳尔再也没有入选过西班牙国家队，他的队长职位被西班牙门将[[卡西利亚斯|卡西利亚斯]]取代。
