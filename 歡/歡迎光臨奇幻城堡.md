{{作品名稱|name=''The Florida Project''|hk=歡迎光臨夢幻樂園|tw=歡迎光臨奇幻城堡|cn=佛罗里达乐园}}是一部2017年美國劇情片，由[[西恩·貝克|西恩·貝克]]執導並與{{le|克里斯·貝戈赫|Chris Bergoch}}共同撰寫劇本<ref>{{cite web|title=VIDEO: First trailer for "The Florida Project" film highlights life at budget motel near Walt Disney World {{!}} Inside the Magic|url=http://www.insidethemagic.net/2017/08/video-first-trailer-florida-project-film-highlights-life-budget-motel-near-walt-disney-world/|website=Inside the Magic|date=2017-08-14}}</ref>。電影主演包括布魯克琳·普林斯、{{le|布莉亞·文奈特|Bria Vinaite}}、[[威廉·達佛|威廉·達佛]]、娃萊莉亞·庫托、克里斯托弗·里维拉與{{le|卡賴伯·蘭里·瓊斯|Caleb Landry Jones}}。該片最先於2017年5月22日在[[第70屆坎城影展|第70屆坎城影展]]的[[導演雙週|導演雙週]]上首映，並於2017年10月6日在美國上映。

== 劇情 ==
故事圍繞在一間名為「奇幻城堡」的廉價摩鐵中，主要描述經理巴比和以此為家的夢妮與母親海莉之故事。

== 發行 ==
《歡迎光臨奇幻城堡》於2017年5月22日在[[第70屆坎城影展|第70屆坎城影展]][[導演雙週|導演雙週]]公開全球首映<ref name="Quinzaine">{{cite web |url=http://www.quinzaine-realisateurs.com/en/fortnight-2017/ |title=Fortnight 2017: The 49th Directors' Fortnight Selection |accessdate=2017-04-20 |work=Quinzaine des Réalisateurs }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref>{{cite web |url=https://variety.com/2017/film/news/directors-fortnight-lineup-juliette-binoche-gerard-depardieu-1202390797/ |title=Cannes: Juliette Binoche-Gerard Depardieu Drama to Kick Off Directors Fortnight |author=Elsa Keslassy |date=2016-04-19 |accessdate=2017-04-20 |work=Variety}}</ref>。

== 評價 ==
《歡迎光臨奇幻城堡》獲得了普遍好評。在[[爛番茄|爛番茄]]上，根據229條評論而持有96%的新鮮度，平均得分8.8 / 10，觀眾平均分數為78%<ref>{{cite web|url=https://www.rottentomatoes.com/m/the_florida_project|title=THE FLORIDA PROJECT (2017)|work=[[Rotten_Tomatoes|Rotten Tomatoes]] |publisher= |accessdate=2018-1-25}}</ref>。在[[Metacritic|Metacritic]]則獲得了92分<ref>{{cite web|url=http://www.metacritic.com/movie/the-florida-project|title=The Florida Project reviews |website=[[Metacritic|Metacritic]]|accessdate=2017-12-19}}</ref>。
