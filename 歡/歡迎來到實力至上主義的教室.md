{{日本作品|link=no|歡迎來到實力至上主義的教室|ようこそ実力至上主義の教室へ}}，是[[衣笠彰梧|衣笠彰梧]]所作的[[日本|日本]][[輕小說|輕小說]]作品，[[插圖|插圖]]由[[知世俊作|知世俊作]]繪畫，是由撰寫戀愛冒險遊戲《[[曉之護衛|曉之護衛]]》、《REMINISCENCE》劇本的衣笠彰梧負責著作，並由同樣擔任《曉之護衛》、《REMINISCENCE》原畫的繪師[[知世俊作|知世俊作]]負責小說的插畫繪製。

以「真正的實力、真正的平等到底是什麼？」為主題的本作品，描寫全新的學園默示錄。2017年5月宣布電視動畫化。日本地区於2017年7月12日起由[[AT-X|AT-X]]、[[東京都會電視台|東京都會電視台]]等開始播出<ref>{{cite web|url=http://www.animenewsnetwork.com/news/2017-04-30/yokoso-jitsuryoku-shijo-shugi-no-kyoshitsu-e-light-novel-series-gets-tv-anime-in-july/.115515|title=Yōkoso Jitsuryoku Shijō Shugi no Kyōshitsu e Light Novel Series Gets TV Anime in July|publisher=[[动画新闻网|动画新闻网]]|accessdate=2017-07-20|date=2017-05-01}}</ref><ref name="Premiere">{{cite web|title=Yōkoso Jitsuryoku Shijō Shugi no Kyōshitsu e Anime Reveals Ending Theme Artist, July 12 Debut|url=http://www.animenewsnetwork.com/news/2017-06-12/yokoso-jitsuryoku-shijo-shugi-no-kyoshitsu-e-anime-reveals-ending-theme-artist-july-12-debut/.117334|website=动画新闻网|date=2017-06-12|accessdate=2017-07-20}}</ref>。其它地區於2017年7月12日起由[[bilibili|bilibili]]、[[巴哈姆特動畫瘋|巴哈姆特動畫瘋]]、[[LiTV_線上影視|LiTV 線上影視]]同日同步播出。

== 劇情大綱 ==
因某個理由而在入學考試偷工減料的主人公 綾小路清隆，被分配到了被揶揄為次品聚集地的 D 班。與同一個班級、成績優秀但性格超麻煩的美少女 堀北鈴音，還有由牽掛與善良構成的天使般的少女 櫛田桔梗 的相遇，使得狀況開始發生改變。

== 設定用語 ==
; {{Nihongo|高度育成高等學校|高度育成高等学校}}
: 99.9% 能夠就職于期望的工作，擁有最新的設備。每個月學校發放貨幣，髮型和私有物的攜帶等等極其自由的校風。學校指南的小冊子裡是個由夢幻般文字書寫而成的學校。
: 然而其真實情況則是一切皆由自身實力來衡量，完全的實力主義教育學校。入學測試的綜合分數（包括筆試、面試、國中時的評價）按照高到低，從A班開始分配。一開始能力就很高且無不良紀錄的人必然被分到A班。而因爲能力低或擁有不良紀錄的人則會被分配到D班，因此入學後的點數必然會被越拉越開。優秀的人會被作爲優秀的存在而被鄭重其事地對待。因爲比起筆試，面試，也就是個人的外在占的比重更大，所以D班裡必然聚集了一堆問題人物（不良品）。

; {{Nihongo|S系統|Sシステム}}
: 入學後班級人員的分配，以及班級點數增減原因，都根據此系統決定，是一個潛藏在高度育成高等學校未明朗化的系統，學生只能根據自己的手上的資訊，來推測此系統的正體，整體狀況不明。

; {{Nihongo|S點數|Sポイント}}
: 又稱為-{「}-班級點數-{」}-，簡稱-{「}-Cl-{」}-，動畫又簡稱-{「}-Pt-{」}-。每個月底班級點數匯入各班並轉換成個人點數，-{「}-1 Cl-{」}-可轉換成-{「}-100 Pt-{」}-。

:; {{Nihongo|個人點數|個人的なポイント}}
:: 簡稱-{「}-Pt-{」}-。-{「}-1 Pt-{」}-相當於-{「}-日圓1元-{」}-，在這所學校中，只要有點數，沒有買不到的東西（除了違反校規之外）。
:: 個人持有兩千萬個人點數可以單獨轉去想去的班級，是除了依靠提升班級點數超越別班之外，另一個可以往A班的方法，只是創校以來，沒有任何一個學生成功達成此功績。
:: 就連三年前依靠違反校規的手段收集個人點數，達成創校以來收集最多個人點數紀錄的3年B班學生，直到畢業前夕最高也只有收集到一千兩百萬，而且後來此學生還被學校退學。
:: 龍園打算在這三年內收集八億個人點數，讓C班全員轉入A班。

; {{Nihongo|特別考試|特別試験}}
: 在這所學校除了其他正常學校有的考試之外，還有獨特的考試，其次數甚至超過正常的考試，不如說這所學校的考試主軸就是特別考試。
: 根據幸村的說法和實際體驗痛悟，在這所學校只會學習是不行的，只會運動也是不行的，即使兩者兼備，也無法只依靠這些通過這所學校全部的特別考試，直覺、靈感、感覺，這所學校依次考驗在人類社會中立足所必不可缺的東西。單打獨鬥已經是沒有出路的了，將會需要的是能把那些要素全部填補的隊伍與團結。

; {{Nihongo|White Room|ホワイトルーム}}
: 綾小路清隆成長並受教育的地方，故事開始一年前，綾小路清隆都在這裡度過<ref group="註" name="第7卷">第7卷 《再會與別離的通知》</ref>。由綾小路清隆的父親所掌管，在'''White Room''' 裡，綾小路清隆的父親的命令是絕對的<ref group="註" name="第7卷" />。
: 根據綾小路清隆的說法，'''White Room''' 不是一個很有人道的地方，甚至見不得人，還有人權等諸多問題，但是這裡可以受到最好的教育，因此綾小路清隆累積了遠超越一般人一生所能學習到的知識<ref group="註" name="第7卷" />。
: 這裡的教育不止有單純的能在普通學校能學習到的，還有各種各樣的東西，[[學問|學問]]、[[學術|學術]]、[[武術|武術]]、[[防身術|防身術]]、[[處世術|處世術]]……等等，綾小路清隆的父親認為必要的部分都是'''White Room''' 必須學習的<ref group="註" name="第7卷" />。
: 反之綾小路清隆的父親認為不必要的部分，如[[遊戲|遊戲]]、[[動漫|動漫]]……這些世俗東西之類的，不在'''White Room'''的學習範圍內<ref group="註" name="第7卷" />，導致綾小路清隆對於世俗方面的東西不是很了解<ref group="註">向堀北鈴音詢問泡麵通常的價錢；向山內春樹表示對於遊戲裡面，女性角色可以穿著沉重的鎧甲活動抱有疑問；外村秀雄向他說起有關動漫的知識，綾小路完全沒有半點概念、向須藤健詢問正常高中生的握力平均值……等等。</ref>。
: '''White Room''' 將綾小路清隆培養成一個全才，並稱作最高傑作品<ref group="註" name="第7卷" />。
: 因為某些原因，'''White Room''' 暫停了一年，之後再度重啟，不過都沒有出現有如綾小路清隆的人才，導致綾小路清隆的父親想用各種手段把綾小路清隆從學校帶回去<ref group="註" name="第7卷" />。
: 坂柳有-{栖}-對'''White Room'''的想法是，爲了讓『沒有天分的人』成爲天才而准備的場所，作爲天才出生的人不需要那個場所<ref group="註" name="第7.5卷">第7.5卷 《伊吹澪災難的一天》</ref>。
