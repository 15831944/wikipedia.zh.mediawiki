'''愚者自将'''（Fool's Mate）又称两回合将杀，是[[国际象棋|国际象棋]]中最快的[[将死|将杀]]方法——黑方第2手用王后将杀白方。因该局面只有当白方棋手水平极低时才会走出来，因此谓之“愚者自将”。初学者如果没练习过该局面，也可能被如此将杀。

此外白1. f3起手，或白1. g4 黑1... e5等走法亦殊途同归。

== 类似陷阱 ==

两步将杀有许多类似的模式。如1. e4 g5 2. d4 f6?? 3. Qh5#和愚人自将同理，黑进f兵和g兵暴露斜线，被白后直接将杀。据传1959年梅斯菲尔德对特林卡中出现愚人自将，该局以1. e4 g5 2. Nc3 f5?? 3. Qh5#三回合快速结束<ref>{{cite book |author=Mike Fox and Richard James |title=The Even More Complete Chess Addict |publisher=Faber and Faber |year=1993 |pages=177}}</ref><ref>{{cite book |author=Winter, Edward |authorlink=Edward G. Winter |title=Chess Facts and Fables |publisher=McFarland & Co. |year=2005 |isbn=978-0-7864-2310-1 |pages=253–254}}</ref><ref>{{cite web |url=http://www.chesshistory.com/winter/winter25.html#4491._Rousseau_and_chess_C.N.s_4401_ |title=Chess Notes 4493. Short game |author=[[Edward_G._Winter|Edward G. Winter]] |date=August 2006}}</ref><ref>{{cite web |url=http://www.chesshistory.com/winter/winter25.html#4503._N_for_knight |title=Chess Notes 4506. Short game (C.N. 4493) |author=[[Edward_G._Winter|Edward G. Qh4#。

1896年弗兰克·梅尔维尔·蒂德和尤金·德尔马一场[[荷兰防御|荷兰防御]]开局中，出现的知名陷阱<ref>[http://www.chessgames.com/perl/chessgame?gid=1242990 Teed vs. Delmar]</ref>：

; 1. d4 f5 2. Bg5 h6 3. Bf4 g5 4. Bg3 f4
: 黑看似赚得一象，但接下来……
; 5. e3
: 准备Qh5#愚人自将
; 5... h5 6. Bd3?!
: 走6.Be2更好，但这招伏有陷阱
; 6... Rh6??
: 防白Bg6#，然而……
; 7. Qxh5+!
: 白方弃后引离黑车，解除黑对g6的控制
; 7... Rxh5 8. Bg6#

[[吉克诺·格雷科|吉克诺·格雷科]]1625年公布的一局棋中出现了类似陷阱：

; 1. e4 b6 2. d4 Bb7 3. Bd3 f5? 4. exf5 Bxg2? 5. Qh5+ g6 6. fxg6 Nf6??
: 若黑改走6... Bg7将f7留出供王躲避，则尚有机会和白扳扳手腕。虽然白后续有7.Qf5! Nf6 8.Bh6 Bxh6 9.gxh7 Bxh1 10.Qg6+ Kf8 11.Qxh6+ Kf7 12.Nh3取得优势，但游戏时间已大大延长<ref>{{cite book |author=Lev Alburt |title=Chess Openings for White, Explained |publisher=Chess Information Research Center |year=2011 |page=509}}</ref>。此外黑可走9... e6给王留出另一空位，则白10.Qg6+将军时有10... Ke7等后续手段。
; 7. gxh7+! Nxh5 8. Bg6#
