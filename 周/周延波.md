'''周延波'''（{{bd|1962年|2月25日||}}），籍贯[[湖北省|湖北]][[光化县|光化]]，中国[[企业家|企业家]]、教育工作者，西安思源学院创始人兼现任董事长，北京邮电大学世纪学院、[[北京人文大学|北京人文大学]]现任董事长。除此之外，他曾担任过政协委员。<ref name=C>{{cite web |title=民教人物-周延波 |url=http://www.cnhsi.com.cn/RW/Show?id=93035cef-3f37-42c9-8e51-57c9eed52356 |website=中国民办教育信息网 |accessdate=2019-01-29 |archive-url=https://web.archive.org/web/20180816001257/http://www.cnhsi.com.cn/RW/Show?id=93035cef-3f37-42c9-8e51-57c9eed52356 |archive-date=2018-08-16 |dead-url=no }}</ref><ref>{{cite web |title=周延波 |url=http://edu.163.com/11/1102/17/7HSE48KG00294KOA_mobile.html |website=网易 |accessdate=2019-01-29}}</ref>

== 人物经历 ==
1984年7月，周延波毕业于[[西安交通大学|西安交通大学]]机械工程系，并留在学校从事教育工作。起初是一名学生辅导员，后升任团工委书记、系办公室主任等领导职务。其间，在1992年的某一天，周延波赴河南一中学招生，发现该校有多名高考落榜生正在校外的出租屋复读。有些学生因复读多年导致竞技状态欠佳，上大学希望渺茫。<ref name=D>{{cite news |title=西安思源学院董事长周延波：教育，让“白鹿”常驻此原 |url=http://edu.nen.com.cn/system/2017/04/25/019817013.shtml |accessdate=2019-01-29 |work=东北新闻网 |date=2017-04-25}}</ref>

为了给更多落榜生接受[[高等教育|高等教育]]的机会，周延波于1994年创立了西安交大机械学院培训中心，以举办高等教育自学考试本、专科班和计算机短期培训班为主。但随着培训中心规模的扩大，校内的办学条件无法顺应办学发展要求，引发众多学生的不满。对此，他多次赴教育厅、民办学校寻求解决方案。直至1998年7月1日，在陕西省教育厅批准之下，周延波的培训中心与西安交通大学产业集团于[[白鹿原_(地名)|白鹿原]]共同创办了西安思源科技培训学院（今西安思源学院）。<ref name=D/>

自西安思源科技培训学院成立以来，周延波致力于发展学校教育、提高学校教学质量。除了与加拿大院校合作开展“3+0”项目、为学校编写必修课教材《创新人才培养》，还与著名作家[[陈忠实|陈忠实]]合办了白鹿书院，推动学校升格为本科高等院校。<ref name=D/><ref>{{cite web |title=西安思源学院探索民办高等教育应用型人才特色发展之路 |url=http://jyt.shaanxi.gov.cn/jynews/rdjj/201810/01/83264.html |website=陕西省教育厅 |accessdate=2019-01-29}}</ref>

== 社会兼职 ==
在从事教育工作的同时，周延波亦有参与政治。2008年1月，他担任[[宁夏回族自治区|宁夏回族自治区]]第九届政协委员；2013年1月，转任[[陕西省|陕西省]]第十二届人大代表。2018年2月，改任陕西省第十二届政协委员。除此之外，他还担任过[[西安市|西安市]]第十三届、十四届人大代表以及西安市人大内务及司法委员会委员。<ref name=C/><ref>{{cite web |title=现任领导 |url=http://www.bjrwdx.com/about/culture.html |website=北京人文大学 |accessdate=2019-01-30 |archive-url=https://web.archive.org/web/20170716132046/http://bjrwdx.com//about/culture.html |archive-date=2017-07-16 |dead-url=no }}</ref>

== 争议事件 ==
2019年1月24日，周延波的前妻冯捷东在网上爆料称，周在担任政协委员期间违规获得加拿大[[枫叶卡|枫叶卡]]；与此同时，周延波与冯捷东的“加拿大永久居民身份批准证明”{{NoteTag|此份证明于2011年获得，当时周延波与冯捷东仍为夫妻关系；}}也在网上传开。“周延波成为加拿大永久居民”一事顿时引发热议。次日，周延波并未正面回应此事，只是称自己仍为中国公民，不会加入外国国籍；其律师李青山也发表声明称，周没有加拿大国籍和枫叶卡。<ref name=A>{{cite news |title=陕西一政协委员被指移民加拿大？ 当地相关部门已介入 |url=http://news.ynet.com/2019/01/25/1643800t70.html |accessdate=2019-01-29 |work=北青网 |date=2019-01-25}}</ref>

1月28日，冯捷东在接受《[[新京报|新京报]]》采访时指出，自己早在2009年就已经和丈夫、女儿一起申请加拿大投资移民；2011年，他们一家三口携“登陆纸”入境加拿大，在海关办理相关手续后成为[[加拿大永久居民|加拿大永久居民]]，并获得加拿大枫叶卡及社保卡{{NoteTag|在谈及此事时，冯捷东还向记者提供了周延波的加拿大[[枫叶卡|枫叶卡]]（复印件）、社保卡等。}}。对于周延波及其律师的声明，冯捷东则予以反驳，称其不具备说服力。随后，《新京报》致电周延波的律师李青山，不过这一次他仅说周没有加拿大国籍。周延波本人也在当晚接到《新京报》记者的来电，但在听到采访要求后，他拒绝接受采访并挂断电话。<ref>{{cite news |title=陕西政协委员周延波陷“移民”争议 目前仍正常履职 |url=http://www.bjnews.com.cn/news/2019/01/28/543685.html |accessdate=2019-01-29 |work=新京报 |date=2019-01-28 |archive-url=https://web.archive.org/web/20190128214846/http://www.bjnews.com.cn/news/2019/01/28/543685.html |archive-date=2019-01-28 |dead-url=no }}</ref>

3月19日，陕西省委教育工委公布事件调查结果，表示已给予周延波党内严重警告处分，并责令其辞去政协委员职务。<ref name=B>{{cite web |title=关于周延波有关问题调查处理情况的通报 |url=http://jyt.shaanxi.gov.cn/jynews/rdjj/201903/19/88523.html |website=陕西省教育厅 |accessdate=2019-03-19 |date=2019-03-19}}</ref>
