'''周邊元件互連特別興趣小組'''（{{lang-en|'''Peripheral Component Interconnect Special Interest Group'''}}，[[縮寫|縮寫]]：{{lang|en|'''PCI-SIG'''}}），是一個負責訂定[[PCI|PCI]]、[[PCI-X|PCI-X]]和[[PCI_Express|PCI Express]] (PCIe)電腦匯流排規格的電子工業聯盟。PCI-SIG不同於有著相似名稱的'''{{tsl|en|PICMG|PICMG}}（{{tsl|en|PICMG|PCI工業電腦製造小組}}）。

PCI特別興趣小組成立於1992年，初期是以相容性計劃的形式幫助製造商實現[[Intel|Intel]]所制定的規格，<ref name="Inc.1994">{{cite book|author=InfoWorld Media Group, Inc.|title=InfoWorld|url=http://books.google.com/books?id=aDgEAAAAMBAJ&pg=PA36|date=3 October 1994|publisher=InfoWorld Media Group, Inc.|pages=36|id={{ISSN|01996649}}}}</ref>於2000年時轉變為非營利組織，並正式命名為'''“PCI-SIG”'''。加入PCI-SIG的會員公司超過800家，各自遵循PCI-SIG所訂定的規格，研發功能有異但可相互操作溝通的產品。PCI-SIG已經訂定的規格有PCI、PCI-X與PCI Express。

PCI-SIG現任董事會董事成員是來自[[Intel|Intel]]、[[Dell|Dell]]、[[AMD|AMD]]、[[Qualcomm|Qualcomm]]、[[HP|HP]]、[[Synopsys|Synopsys]]、[[Agilent_Technologies|Agilent Technologies]]與[[NVIDIA|NVIDIA]]的公司代表。

PCI-SIG的主席與總裁是來自IBM公司的傑出工程師：Al Yanes。執行董事是[http://www.vtmgroup.com VTM Group]的總裁：Reen Presnell。

PCI-SIG免費提供組織會員下載其所訂定的規格。<ref>{{cite web |url=http://www.pcisig.com/specifications/ |title=Specifications |publisher=PCI-SIG |accessdate=24 June 2010}}</ref>非會員可以購買紙本規格書，價格介於美金1000至3500元間，或是以美金5000元來取得內含該組織訂定的所有規格書CD。<ref>{{cite web |url=http://www.pcisig.com/specifications/order_form/ |title=Specification order form |publisher=PCI-SIG |accessdate=24 June 2010}}</ref>
