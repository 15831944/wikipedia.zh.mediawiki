'''周龙'''（{{bd|1953年}}），美籍华裔作曲家，出生于[[北京市|北京市]]鼓楼的一个四合院里。

== 经历 ==
周龙的父亲是画家，母亲是歌唱家，都在艺术高校任教<ref>{{Cite web |url=http://www.sinbam.com/class/201312/120111593.html |title=存档副本 |access-date=2013-12-30 |archive-url=https://web.archive.org/web/20140103012323/http://www.sinbam.com/class/201312/120111593.html# |archive-date=2014-01-03 |dead-url=yes }}</ref>。1983年毕业于[[中央音乐学院|中央音乐学院]]，曾被聘为中国国家广播交响乐团驻团作曲家。现任美国密苏里州立大学堪萨斯城音乐舞蹈学院全职作曲教授<ref>{{cite web |url=http://www.tjcm.edu.cn/showwords.asp?airid=2292 |title=存档副本 |accessdate=2013-12-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20131231002850/http://www.tjcm.edu.cn/showwords.asp?airid=2292 |archivedate=2013-12-31 }}</ref>。2011年凭借《白蛇传》获得[[普利策音乐奖|普利策音乐奖]]，是亚裔在普利策奖历史上首次获得该奖项。<ref>http://news.xinhuanet.com/2011-05/24/c_121449137.htm</ref>。
