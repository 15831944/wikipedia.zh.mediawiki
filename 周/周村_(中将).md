'''周村'''（{{bd|1926年|||}}）原名'''周祖华'''，今[[河南省|河南省]][[信阳市|信阳市]][[浉河区|浉河区]][[谭家河乡|谭家河乡]]人，[[中国人民解放军中将|中国人民解放军中将]]。

==生平==
1939年参加[[中国共产党|中国共产党]]领导的革命，1942年加入中国共产党。[[抗日战争|抗日战争]]时期，历任[[新四军|新四军]]第五师二分军区第二十五团连政治指导员、独立第五团副教导员，新四军第五师十五旅四十五团营教导员、营长。[[第二次国共内战|第二次国共内战]]时期，历任（前）[[中原军区|中原军区]]第二纵队第十五旅第四十五团营长，[[江汉军区|江汉军区]]警卫团营长、教导大队教导员。[[中华人民共和国|中华人民共和国]]成立后，历任国家核试验委员会委员，防化部部长，防化技术学校校长，[[中国人民解放军总参谋部|中国人民解放军总参谋部]]防化部副部长，[[国家地震局|国家地震局]]政委，[[中国人民解放军总参谋部动员部|中国人民解放军总参谋部动员部]]副部长。1986年调任[[南京|南京]][[南京陆军指挥学院|陆军指挥学院]]副院长，后升任院长。1988年获授中将军衔<ref>{{Cite web|url = http://www.shqqw.cn/a/lishiyange/36.html|title = 红色浉河|author = |date = 2013-10-09|publisher = 浉河区情网|language = |accessdate = |quote = }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url = http://www.shihe.gov.cn/content/?2529.html |title = 周村 |author =  |date = 2009-08-18 |publisher =信阳市浉河区人民政府  |language =  |accessdate =  |quote =  }}</ref>。
