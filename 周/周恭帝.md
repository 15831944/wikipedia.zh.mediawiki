'''周恭帝柴宗训'''（{{bd|953年|9月14日|973年|4月6日|catIdx=C柴宗訓}}），[[五代|五代]]时期[[后周|后周]][[皇帝|皇帝]]，[[柴荣|周世宗]]第四子。显德六年（959年）封为'''梁王'''。世宗于同年六月病死，他于同月甲午日继位，沿用[[郭威|周太祖]][[年号|年号]]“显德”。

== 即位 ==
柴宗训即位时，年仅七岁，由[[符太后|符太后]]垂帘听政，[[范质|范质]]、[[王溥_(宋朝)|王溥]]等主持军国大事。柴宗训在位期间，特别重用其父親任命的[[赵匡胤|赵匡胤]]負責軍事事務。

== 後周滅亡 ==
{{main|陳橋兵變}}

显德七年（960年）正月元旦，群臣正在朝贺柴宗训时，镇（今[[河北省|河北省]][[正定县|正定县]]）、定（今[[河北省|河北省]][[定县|定县]]）两州遣人来报，[[辽国|辽国]]和[[北汉|北汉]]合兵南侵。范质命令殿前都點檢赵匡胤率领禁军北上抵御。禁军到达开封东北部的[[陈桥驿|陈桥驿]]后，突然发动[[陈桥兵变|兵变]]，拥赵匡胤为帝，黃袍加身。赵匡胤回师[[开封|开封]]，朝中大臣[[范质|范质]]等人被挟迫拜见“新天子”，显德七年，后周恭帝柴宗训禅让帝位于赵匡胤，降封郑王，符太后改称周太后，郭威柴荣的宗族被迁往房州（今[[湖北省|湖北省]][[房县|房县]]）居住。

同年赵匡胤建立[[宋朝|宋朝]]，改元“建隆”。柴宗训在位前后仅六个月。[[后周|后周]]亡。

== 逝世 ==
宋太祖頒布[[聖旨|聖旨]]優待帝母子，賜柴氏“[[丹書鐵券|丹書鐵券]]”（[[免死金牌|免死金牌]]），保證柴氏子孫永享富貴，即使犯罪也不得[[刑罰|加刑]]。

建隆三年（962年）柴宗训被迁往房陵（今[[湖北省|湖北省]]房县）居住，开宝六年（973年）三月卒于当地，终年仅21岁（一说宋太祖将其殺害，免除后患）。[[赵匡胤|赵匡胤]]“闻之震恸”，「素服發哀，輟朝十日」，谥曰“恭皇帝”，归葬于世宗庆陵之侧。
