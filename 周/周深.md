'''周深'''{{BD|1992年|9月29日}}，[[中國大陸|中國大陸]]男歌手，[[代表作|代表作]]《大魚》。迄今為止，先後為二十多部[[電影|電影]]、[[電視劇|電視劇]]、[[國產|國產]][[漫畫|漫畫]]、[[遊戲|遊戲]]、[[舞臺劇|舞臺劇]]獻唱[[原声音乐|原聲]]歌曲，連續榮獲[[東方風雲榜|東方風雲榜]]、[[全球華語榜中榜|全球華語榜中榜]]等二十多個獎項。2017年11月發行個人首張原創[[專輯|專輯]]《深的深》，並憑借此張專輯獲得2018年全球華人歌曲排行榜年度最佳新人獎，主打歌《藍色降落傘》獲得年度金曲獎。2018年3月獻唱[[奧斯卡最佳影片獎|奧斯卡最佳影片]]《[[水形物語|水形物語]](又譯：水底情深)》(The Shape of Water)同名中文推廣曲<ref>{{cite web |title=從《大魚》到《水形物語》，周深的水樣情結 |url=http://www.artsbj.com/show-197-573266-1.html |publisher=北京文藝網 |accessdate=2018年3月29日}}</ref>。2018年5月為[[中國中央電視臺|中國中央電視臺]]欄目《[[等著我|等著我]]》演唱同名[[主題曲|主題曲]]。2018年5月啟動“深空間”個人首輪[[巡迴演唱會|巡迴演唱會]]。

== 個人簡介 ==

周深祖籍[[湖南|湖南]]，小二隨父母到[[貴陽|貴陽]]定居。從小就被老師發現“唱歌不會跑調”從而加入合唱團。但是在[[變聲|變聲]]期，他的聲線並沒有發生變化，依然保持了介於童聲和女聲之間的音色，以至在初中三年時間內都不敢在同學面前唱歌。但升上高中不久的一場校園歌唱大賽周深奪得冠軍，使得他在校園裏漸漸有了名氣，更因此有學長主動邀請他演唱自己的原創歌曲《雨後你不見了》，讓他重新找回唱歌的自信。

高中畢業後周深遠赴[[烏克蘭|烏克蘭]]求學，原先研修醫科，在入學一年後轉至[[烏克蘭|烏克蘭]]{{link-en|利沃夫國立音樂學院|Lviv_Conservatory}}學習[[聲樂|聲樂]]專業[[美聲唱法|美聲唱法]]，並於2016年6月畢業。

== 歌唱生涯 ==

=== 出道前 ===
2010年周深進入網路語音平臺YY唱歌，化名“卡布叻”。當時的網路語音平臺上，眾人只能聽到他的聲音，看不到他的樣子。除了唱歌之外，他還善於模仿各種各樣的聲音，從而在網上收獲了自己的第一批粉絲。這個時期的粉絲稱自己為“布丁”。

=== 2014年參加[[中國好聲音_(第三季)|好聲音]] ===
由於聲音和外形條件不符合大眾的普遍認知，周深在很長一段時間裏都只敢在網絡上躲在屏幕後面唱歌，而不太敢公開露面唱歌。在2014年初，有一位好聲音導演連續找了周深三個月，才終於說服了他參加好聲音。2014年7月周深參加[[浙江衛視|浙江衛視]]《中國好聲音 (第三季)》，盲選時以[[齊豫|齊豫]]的《歡顏》獲得三位導師轉身，轉身後的導師們看見他的樣子都表示很吃驚，也都對他的演唱大加贊賞<ref>{{cite web|url=http://news.163.com/14/0726/06/A22DQSNC00014AED.html|title=周深：擔心個性聲音被觀眾罵|publisher=網易新聞|date=2014年7月26日}}{{zh-cn}}</ref>。周深加入了[[那英|那英]]組，與李維合唱的《貝加爾湖畔》獲得[[中國好聲音_(第三季)|中國好聲音 (第三季)]]年度金曲。同年底到隔年初周深參與了包括[[北京|北京]][[工人體育館|工體]]、[[香港|香港]][[香港體育館|紅磡]]、[[上海|上海]][[上海大舞臺|大舞臺]]、[[深圳|深圳]][[深圳體育場|體育場]]、[[臺北小巨蛋|臺北小巨蛋]]等多場好聲音巡迴演唱會。

在電視上露面之後，周深失去了一些粉絲，然而又收獲了更多能接受他聲音和外形的粉絲，這之後的粉絲給自己取名“生米”，並和周深一起確定應援色是[[藍色|藍色]]。

=== 2015年首支個人原唱[[單曲|單曲]] ===
2015年，周深除了繼續參加好聲音巡回演唱會之外，還在2月參加了[[北京衛視|北京衛視]]春節聯歡晚會獻唱多國語言版的《[[隨它吧|Let It Go]]》，支持北京申辦[[冬季奧林匹克運動會|冬奧會]]。在2月、3月[[春節|春節]]期間他还參加「文化中國·四海同春」藝術團的慰僑活動，赴[[澳大利亞|澳大利亞]]和[[新西蘭|新西蘭]]五座城市巡回演出。5月，周深代表《中國好聲音》前往[[英國|英國]][[倫敦|倫敦]]參加歐洲區《中國好聲音》參賽選手的總決選，並擔任評審及表演嘉賓<ref>{{cite web |title=The Voice of China: behind the scenes of the European auditions |url=https://www.telegraph.co.uk/culture/tvandradio/the-voice/11586462/The-Voice-of-China-behind-the-scenes-of-the-European-auditions.html |website=The Telegraph |accessdate=2015年5月30日}}{{en}}</ref>。2015年7月，周深與李維共同推出首張翻唱專輯《回味》。

著名音樂人[[高曉松|高曉松]]聽到周深在好聲音演唱的歌曲之後，說“這樣的聲音不拿來唱唱我的歌，這不虧了嗎”<ref>{{cite web |title=對音樂最有發言權的人說，如今中國音樂人很有希望。 |url=http://ent.163.com/17/1113/10/D348ARKM000396AR_mobile.html |publisher=网易新闻 |accessdate=2017年11月13日}}</ref>，於是通過那英找到了周深。2015年8月8日，周深首支個人[[錄音室|錄音室]]原唱[[單曲|單曲]]《玫瑰與小鹿》發行。這首歌曲由高曉松作曲，尹約作詞，周深演唱。《玫瑰與小鹿》为周深赢得了第23屆[[東方風雲榜|東方風雲榜]]最佳新人和2016年亞洲新歌榜年度十大金曲。

=== 2016年[[代表作|代表作]]《大魚》 ===
2016年2月春節期間，周深再次參加「文化中國·四海同春」藝術團的慰僑活動， 這一次是[[美國|美國]]和[[加拿大|加拿大]]六座城市的巡演。2016年5月周深為[[動畫電影|動畫電影]]《[[大魚海棠|大魚海棠]]》獻唱印象曲《大魚》，在6月[[第16屆中國電影華表獎|第16屆中國電影華表獎]]典禮時現場演唱。《大魚》成為周深的代表作，共為他獲得八個獎項，其中包括第17屆[[音樂風雲榜|音樂風雲榜]]年度最佳電影歌曲獎、第24屆[[東方風雲榜|東方風雲榜]]十大金曲等。2016年9月周深又飛到了[[歐洲|歐洲]]，在[[法國|法國]][[波爾多|波爾多]]參加《情系金秋·2016法國[[中秋|中秋]]盛典暨[[浙江衛視|浙江衛視]]國際頻道開播十周年慶典晚會》。

2016年10月，周深參加[[江蘇衛視|江蘇衛視]]《[[蒙面唱將猜猜猜_(第一季)|蒙面唱將猜猜猜 (第一季)]]》，隱藏外形，在蒙面的情況下演唱了四首歌曲，其中包括與[[側田|側田]]合唱的《我真的受傷了》，揭面之後又在年度盛典與[[李琦_(歌手)|李琦]]合唱了一首《你要的愛》。在蒙面熱度的帶領下，周深的全新單曲《你》也在10月25日發行，上線只一天就登上了[[酷狗音樂|酷狗]]飆升榜第一、酷狗華語新歌榜第二的位置<ref>{{cite web |title=《蒙面唱將猜猜猜》“我可不是什麼幺蛾子”周深惹網友嫉妒 新歌領跑音樂飆升榜 |url=http://fj.people.com.cn/n2/2016/1028/c363308-29220348.html |publisher=人民網 |accessdate=2016年10月28日}}</ref>。

=== 2017年七支單曲及首張個人原唱[[專輯|專輯]]《深的深》 ===
由於為動畫電影《大魚海棠》獻唱的印象曲大獲成功，2017年周深成了OST歌曲的新寵，這一年他一共發布了《何處是天涯》、《濃情淡如你》等七首錄音室原唱單曲，全部是OST歌曲。2017年1月周深應[[加拿大|加拿大]][[中國學生學者聯合會|中國學生學者聯合會]]的邀請，再赴加拿大[[多倫多|多倫多]]，參加在索尼中心舉行的雞年春晚。3月和4月分別獲得了第24屆東方風雲榜全民選擇男歌手和第21屆[[全球華語榜中榜|全球華語榜中榜]]網絡票選人氣新聲稱號。2017年9月，周深作為幫唱嘉賓參加《[[中國新歌聲|中國新歌聲]]》，與郭沁合唱《大魚》，現場版的大魚驚艷了所有在場的評委，幫助郭沁奪得了當季新歌聲那英組冠軍。

2017年11月周深發行了個人首張原唱國語專輯《深的深》，該專輯由著名音樂人高曉松監製、[[尹約_(作詞人)|尹約]]擔任制作人、並與曲作錢雷共同創作，歷經三年的精心打磨，在美國頂級錄音室錄制，最後才制作完成<ref>{{cite web |title=撫慰人心的藍色降落傘 - 周深&高曉松專訪 |url=https://www.youtube.com/watch?v=5hnBrxFwBMg |website=youtube |publisher=STAR!調查團 |accessdate=2017年11月23日}}</ref><ref>{{cite web |title=高曉松領銜“高家班”集體亮相 為周深新專輯站臺 |url=http://www.xinhuanet.com/ent/2017-11/07/c_1121918076.htm |publisher=新華網 |accessdate=2017年11月07日}}</ref>。此專輯又為周深贏得了三個獎項，其中包括第25屆東方風雲榜年度飛躍歌手獎。另外作曲錢雷和作詞高曉松及尹約也因此專輯中的歌曲共獲得了三個獎項。

=== 2018年二十支單曲及首次個人[[巡回演唱會|巡回演唱會]] ===
2018年初，祁辛邀請周深為奧斯卡最佳電影《水形物語》獻唱中文推廣曲，說有天放假在家看電視，一個臺周深在唱大魚，換個臺周深還在唱大魚，看了他的演唱，覺得他特別適合這部電影。這次合作讓周深在OST領域更上了一個臺階。2018年周深共發布了20首錄音室原唱單曲，其中16首是OST歌曲，包括《[[梅香如故|梅香如故]]》、《獨白》、《緣起》等。2018年1月，周深因在貴州貧困地區支教的經歷，在輕松籌公益盛典獲年度公益星勢力獎。5月，周深發布了“深空間”巡迴演唱會的宣傳曲《花開》，之後分別在[[上海市|上海]]、[[武漢|武漢]]及[[成都|成都]]完成“深空間”首輪個人巡回演唱會。《花開》這首歌在2019年3月獲得了第26屆東方風雲榜十大金曲獎。

作為美聲專業畢業的流行歌手，周深一直在嘗試把美聲唱法結合到流行音樂裏面來，這在《大魚》和《淺淺》裏面都有體現。2018年11月，周深參加了湖南衛視的一檔美聲節目《[[聲入人心|聲入人心]]》，用美聲和流行兩種風格相結合的方式演唱了一首[[意大利|意大利]][[歌劇流行音樂|歌劇流行音樂]]的名曲《[[告別的時刻|Time To Say Goodbye]]》。同月周深還再次參加了江蘇衛視音樂綜藝節目《[[蒙面唱將猜猜猜_(第三季)|蒙面唱將猜猜猜 (第三季)]]》，以「宮廷背包客」身份出場，用[[美聲|美聲]]方式演唱迷惑猜評團和觀眾，以美聲和[[流行|流行]]方式演唱了《雪落下的聲音》等三首歌曲<ref>{{cite web |title=周深再登《蒙面唱將猜猜猜》舞臺 專業美聲征服人心 |url=http://k.sina.com.cn/article_5991141691_16519913b00100c3d3.html |publisher=新浪看點 |accessdate=2018年12月03日}}</ref>。截至2019年4月20日，《雪落下的聲音》在[[YouTube|YouTube]]的播放量合計已經超過了五百萬。隨後，周深又重新回到聲入人心舞臺，用[[英語|英語]]和[[漢語|漢語]]演唱了七首歌曲，其中一首《Memory》的[[中文|中文]]部分是周深自己[[填詞|填詞]]<ref>{{cite web |title=網易娛樂獨家專訪 #聲入人心# 周深 Zhou Shen |url=https://m.weibo.cn/status/4324424713233596? |website=微博網易娛樂頻道 |publisher=娛樂新鮮派 |accessdate=2019年1月3日}}</ref>。這次美聲和流行唱法相結合的演唱歷程，為周深在2019年3月贏得了第26屆東方風雲榜年度最具突破藝人稱號。

=== 2019年多個巡回演唱會 ===
2019年周深已經發布了四首錄音室原唱單曲，分別是懸疑[[電視劇|電視劇]]《[[古董局中局|古董局中局]]》的[[插曲|插曲]]《隨風》、《[[倚天屠龍記_(2019年電視劇)|倚天屠龍記 (2019年電視劇)]]》插曲《此生惟你》、《[[燕陽春|燕陽春]]》電視劇插曲《落花》和[[動畫|動畫]]《[[鬥破蒼穹|鬥破蒼穹]]特別篇2沙之瀾歌》的[[主題曲|主題曲]]《蛻》。2019年第14期[[QQ音樂|QQ音樂]]巔峰人氣榜，周深有四首歌曲《隨風》、《落花》、《好的晚安》和《月彎彎》同時上榜。《此生惟你》獲得[[全球中文音樂榜上榜|全球中文音樂榜上榜]]4月第二周周榜冠軍。2019年4月12日《蛻》發布的當天即空降QQ音樂巔峰人氣榜榜首，人氣值突破100萬，獲得“人氣黃金單曲”認證。4月20日《蛻》和《隨風》人氣值均突破300萬，雙雙獲得QQ音樂巔峰人氣榜“巔峰黃金單曲”認證，同時解鎖“巔峰人氣公益季”公益項目進度。同一時期內，《蛻》還獲得了全球中文音樂榜上榜4月第三周周榜冠軍。

同時，第二輪“深空間”個人[[巡回演唱會|巡回演唱會]]已經於1月20日在北京舉行。3月28日舉行了in-Music“深時深刻”音樂分享會。4月參與了“聲入人心”北京和天津的演唱會。4月還將參加“不說再見”上海場的演唱會，以及珠海場“聲入人心”演唱會。5月18日將在上海舉行“深齡其境”周深[[黃齡|黃齡]]演唱會。

== 藝術風格 ==
周深是男生，因為變聲期不完全，他的嗓音唱高音比較像女聲，又保留了童聲的特質，可以理解成女聲童聲男聲三種嗓音混合後的嗓音，因此他在低中高三頻段的聲音上，會有很多微妙的變化，低音區偏男聲，高音區偏女聲。周深音域比一般男性高，擅長用弱混和氣聲唱歌，加上天賦音色純凈，而且混聲唱歌會有空靈感，所以很多人覺得他音色空靈婉轉。一些歌壇前輩也都對周深公開表達過高度贊揚。
*[[那英|那英]]對周深高喊“你比女生唱的還美”，後來又說“有一種女人的那種細膩”。（《中國好聲音》第三季第二期，2014年7月25日）
*[[楊坤|楊坤]]說“你的聲音是我從事音樂這麼多年，少見的一種聲音，是跨越年齡，跨越性別，你有太多的可能性了”。（《中國好聲音》第三季第二期，2014年7月25日）
*[[華少|華少]]在介紹周深的時候說“事實上我們都說，他的聲音跨越了年紀，跨越了性別”。（《中國好聲音》第三季《重返巔峰》，2014年9月1日）
*[[王治平|王治平]]大力稱贊周深說“如果我是一個制片我下一步要拍一個歷史古裝劇，我第一個人選一定找你，你來唱的話我的票房一定多加一個億”，“如獲至寶的聲音”，“我覺得你是目前我這樣聽到，我覺得唯一我認為可以走出中國的聲音”。（《中國好聲音》第三季《重返巔峰》，2014年9月1日）
*[[李健|李健]]評價周深說“他是難得的天賦...他這首歌賦予了貝加爾湖更多的色彩”。（中國正在聽2014-10-20期 李健專訪）
*[[高曉松|高曉松]]盛贊周深的嗓音是“介乎童聲與女聲之間的魔音，一個人的唱詩班”<ref>{{cite web |title=周深《玫瑰與小鹿》成績好 高曉松贊其天籟魔音 |url=http://www.xinhuanet.com/ent/2015-08/12/c_128119943.htm |publisher=新華網 |accessdate=2015年8月12日}}</ref>（2015年8月12日）。
*[[巫啟賢|巫啟賢]]說“她（他）擁有所有唱歌該有的技巧和能力...聽到她（他）的聲音，我覺得會有小鹿亂撞”。（《蒙面唱將猜猜猜》第一季第六期揭面之前，2016年10月23日）
*那英說“周深聲音是有那種特別的淒美，是那種特高級的妖勁兒”。（《中國新歌聲》第二季第十期，2017年9月15日）
*高曉松說“其實我基本就是兩個標準，一個就是聲音有很好的特色，一個就是有很好的完成度，周深是不世出的那種歌手，就是不用你去猶豫...周深是你聽一耳朵，哎，說這樣的聲音不拿來唱唱我的歌，這不虧了嗎，是不是，你當然喜歡你聽聽說，哦，用這樣的聲音來唱唱我寫的東西是什麽樣子的，那當然是很幸福的了”,“所有音樂人終生追求的其實就是六個字，叫“知輕重、大而整”。知輕重可能會耗掉一個音樂人大半輩子的時間，有人到老都不知輕重...對於科班出身的，利沃夫音樂學院的，首先要解決的就是怎麽知輕重...我覺得周深在這張專輯裏解決了知輕重的問題，就是讓我已經非常欣慰。”<ref>{{cite web |title=近些年樂壇前輩們對周深的評價 |url=https://www.youtube.com/watch?v=kBQ0X3hEI40 |website=youtube |accessdate=2019年2月12日}}</ref><ref>{{cite web |title=對音樂最有發言權的人說：如今中國音樂人很有希望 |url=http://ent.163.com/17/1113/10/D348ARKM000396AR.html |website=網易娛樂 |accessdate=2017年11月13日}}</ref>。（在周深2017新專輯《深的深》音樂品鑒沙龍接受網易采訪，2017年11月13日）
*[[大張偉|大張偉]]說“背包老師（周深）已經唱到準到已經覺得這才是人肉自動修音，就是她（他）每個音都是齊準無比的”。（《蒙面唱將猜猜猜》第三季第六期蒙面的時候，2018年11月25日）
*[[林誌炫|林誌炫]]說“我覺得更重視的是她（他）聲音的集中度，還有那個寬度，它其實已經脫離了我們對於流行音樂的認知”。（《蒙面唱將猜猜猜》第三季第六期蒙面的時候，2018年11月25日）
*巫啟賢說“我覺得我們不能再用一個流行歌手的眼光去看待周深，我覺得以他的唱腔，他的唱功，他表現出來的那種藝術的層次，我覺得你已經是一個藝術家”。（《蒙面唱將猜猜猜》第三季第七期揭面的時候，2018年12月2日）
*[[吳宗憲|吳宗憲]]对周深说“你的聲音太美好了...”（《蒙面唱將猜猜猜》第三季第七期揭面的時候，2018年12月2日）
*吳宗憲評價周深說“像他這樣的歌手，他是有機會往國際級的路線去走的”。（《蒙面唱將猜猜猜》第三季采訪花絮，2018年12月2日）<ref>{{cite web |title=《蒙面》周深揭面,巫啟賢稱他是藝術家,吳宗憲感嘆超越極限 |url=http://k.sina.com.cn/article_6776114864_193e34eb000100fej0.html |website=新浪網 |accessdate=2018年12月3日}}</ref>

周深的歌曲主要屬於[[流行_(音樂類型)|流行]]音樂，帶有一些美聲唱法的因素。周深所演唱的絕大多數都是抒情歌曲。在蒙面第一季揭面的時候，周深說“我每一次站在舞臺上好好唱歌的時候，就希望大家能夠感受到我是真的在好好唱歌，而不是在用獵奇的眼光在說，哦，你是在反串。我真的每一首歌我都很用心在唱，所以希望大家多多聽我的歌聲就好，謝謝”。
