'''周杰'''（{{bd|1927年||2005年|7月10日}}）[[江苏|江苏]][[阜宁|阜宁]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中共中央办公厅|中共中央办公厅]]副主任，[[第七届全国人大常委会|第七届全国人大常委会]]副秘书长<ref name=shishi/>。

==生平==
1944年参加革命工作，同年加入[[中国共产党|中国共产党]]，历任[[阜宁县|阜宁县]]第三区游击队员、公兴区区委副书记兼组织委员等职。[[第二次国共内战|第二次国共内战]]时期，历任阜宁县三区区委书记、华中五分区总工会常委、阜宁县委工作组组长等职。1948年8月参加[[河北省|河北省]][[平山县|平山县]][[中央团校|中央团校]]第一期学习，后来留校工作。[[中华人民共和国|中华人民共和国]]成立后，在中央团校历任教育科科员、政治经济学教研室主任兼校党委委员、团校副教育长等职。[[文革|文革]]中受冲击。1972年春，调到国家出版局[[人民出版社|人民出版社]]从事编辑工作。[[中共十一届三中全会|中共十一届三中全会]]后，任人民出版社副社长、党委副书记。1979年春，调到[[中共中央办公厅|中共中央办公厅]]工作，历任[[中共中央办公厅秘书局|中共中央办公厅秘书局]]秘书、副局长，[[中共中央政策研究室|中央书记处研究室]]室务委员等职。1983年2月任中共中央办公厅副主任。1987年12月任[[中共中央政策研究室|中央政治体制改革研究室]]副主任。1988年3月当选第七届[[全国人大法律委员会|全国人大法律委员会]]委员，同年9月任[[第七届全国人大常委会|第七届全国人大常委会]]副秘书长。1993年6月离休。离休后享受部长级医疗待遇<ref name=shishi/><ref>{{Cite web |url = http://news.sina.com.cn/c/2005-07-20/18486485605s.shtml |title =中共中央办公厅原副主任周杰同志逝世  |author =  |date =2005-07-20  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

2005年7月10日，周杰在北京病逝，享年78岁<ref name=shishi>{{Cite web |url =https://news.qq.com/a/20050720/001658.htm  |title = 中共中央办公厅原副主任周杰同志在北京逝世 |author =  |date =2005-07-20  |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。
