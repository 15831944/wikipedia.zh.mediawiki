'''周穎南'''（{{bd|1929年||2014年|2月27日|}}），原名'''周國輝'''，新加坡公民，生於[[福建|福建]][[仙遊縣|仙遊]]（[[中國|中國]][[福建省|福建省]][[莆田市|莆田市]]西南部），畢業於福建省立[[莆田學院|仙遊師範學校]] <ref name="周穎南文庫">{{cite news |language = zh |author =高占祥| url = http://book.people.com.cn/GB/69365/4968156.html |title =《周穎南文庫》：情系文化 根植中華|publisher = 人民網|date = 2006-10-27 |accessdate = 2018-12-27}}</ref>。周穎南是新加坡同樂飲食業集團及海洋紡織有限公司創辦人、武漢新民眾樂園董事長。經商之餘投入文學創作，著述豐碩，是一位華文作家、文藝理論家、鑒賞家，被[[新加坡|新加坡]]作家協會推舉為永久名譽理事長，是一位文壇與企業經營都取得成功的儒商。

曾任中國社會福利教科文中心國畫研究會副會長<ref>{{cite news |language = zh|author =秋寶| title =中國社會福利教科文中心國畫研究會在京成立|publisher =文匯報|pages =第二版|date = 1987-09-10|accessdate = 2018-12-28}}</ref>、世界中國烹飪聯合會、世界華文文學家協會顧問，受多所知名大學聘為名譽校長、教授<ref name="懷念周穎南">{{Cite web |url=http://www.ptwhw.com/?post=8626 |title =懷念周穎南先生|author =方紀龍|date = 2014-7-15 |publisher =莆田文化網|language = zh |accessdate = 2018-12-28}}</ref>。畢生著述200多萬字，被譽為「南洋一枝筆」<ref>{{cite web|url=http://www.chinaqw.com/node2/node2796/node3475/node3496/userobject6ai180971.html|title=文坛商海双丰收的周颖南|date=|website=www.chinaqw.com|accessdate=2019-01-05}}</ref>、「商海奇文人」<ref>[http://88088.com/wzpp/txwzr/2007/0116/154523.shtml 張執任和他的《世界華人名人錄》]中國溫州商會網.</ref>，[[北京師範大學|北京師範大學]]將他的著作輯錄成《周穎南文庫》<ref>{{Cite web |url = http://www.chinawriter.com.cn/bk/2006-10-28/26074.html|title = 赤誠于文化  根植于中華——《周穎南文庫》研討會暨捐贈儀式 |author = 江湖|date = 2006-10-28|publisher = 中國作家網|language =zh |accessdate = 2018-12-29}}</ref>，由中國現代文學館專門收藏。

== 生平 ==
周穎南生於閩中大蜚山下仙遊縣鯉魚鎮一戶書香門第，為家中獨子，父親周子溪是一位商人、教育家，出國經商多年後，返鄉於仙遊創辦新式學校振文小學，有仙遊[[陶行知|陶行知]]之名<ref> {{Cite book | author = 伍豪| title = 可怕的福建人 | publisher = 右灰文化傳播有限公司| date = 2011| ISBN = 9787560267265| accessdate =2018-12-31 | url = https://books.google.com/books?id=jCu-DQAAQBAJ&pg=PT177&lpg=PT177&dq=%E5%91%A8%E5%AD%90%E6%BA%AA&source=bl&ots=oTCkUQH4g2&sig=iJPM0_-zQIpXQ5UX9COa7qklwhg&hl=zh-TW&sa=X&ved=2ahUKEwiUi9jBgcvfAhUS7J8KHe20AU8Q6AEwCXoECAkQAQ#v=onepage&q=%E5%91%A8%E5%AD%90%E6%BA%AA&f=false| language =zh}} </ref>。他編輯了一本古文小冊《文陣雄師》<ref name="周穎南文庫"/>，讓周穎南抄寫背誦，使周打下古文基礎，奠定新儒家觀念，影響他日後經商處事之道。

周穎南自幼勤學苦讀，遊心於學，中學就讀仙游縣慕范陶德聯合中學，後考入福建省立仙遊師範學校，師範學校畢業後，出任當地小學教師<ref name="周穎南文庫"/>。1950年，周穎南21歲，為了家庭生計，隨父親遠渡印尼開創事業。他先在[[泗水_(印尼)|蘇臘巴亞]]市一家工廠擔任會計和生產管理，學習印尼語。1955年時，與朋友合夥開辦「同豐貿易公司」，從事歐美汽車零配件的貿易。見汽車市場日漸飽和，便前往首都[[雅加達|雅加達]]，由貿易經營轉向保險金融業<ref name="周穎南文庫"/>。1970年，周穎南全家遷居新加坡。此時，百業待舉，便與友人合資創辦紗廠，又橫跨經營植物油提煉廠、針織廠、染整廠、成衣廠、塑膠廠等產業，經營進出口業務<ref name="懷念周穎南"/>。

1980年開始，新加坡經濟成長，躍升為世界最大的中轉港（後被[[上海|上海]]超過）及國際商業金融中心。周穎南開設了一家中式高檔餐廳「湘園酒樓」，主打[[湖南菜|湖南菜]]，原本是為了經商閒暇之餘與文壇友人有一個聚會賦詩的場所，因酒樓設計富有濃厚中華文化風采與精緻菜色而取得成功。之後，周穎南共開設了八家不同風格的餐廳，進一步組成「同樂飲食集團」，於2001年3月在新加坡掛牌上市<ref name="界面新闻">{{cite web|url=https://www.jiemian.com/article/1785833.html|title=南洋一支笔！新加坡同乐周氏家族的文化积淀-界面新闻 · JMedia|author=|date=|website=www.jiemian.com|accessdate=2019-01-05|archive-url=https://web.archive.org/web/20190102094653/https://www.jiemian.com/article/1785833.html|archive-date=2019-01-02|dead-url=no}}</ref>。

周穎南自中學開始，於家鄉地方報社投稿發表小說及散文，到了印尼創業初期仍不忘筆耕，於《新报》等数家印尼華文報刊發表作品，在著名國學大師梁披雲創辦的華文《火炬報》擔任義務特約記者。1965年，在印尼雅加達舉行的亞非會議周年紀念大會上，採訪過時任中國總理的[[周恩來|周恩來]]先生<ref>{{cite web |author1=张敏 王宝芬 |title=“周颖南八十华诞暨华文著作研讨会”举行 |url=http://culture.people.com.cn/GB/87423/8664097.html|date = 2009-01-13|publisher=人民网-《人民日报海外版》 |accessdate=2019-01-07}}</ref>。

周穎南著作豐碩，自1977年出版《迎春夜話》散文處女作後，陸續出版：《海粟老人近作》、《穎南選集》、《南國聲華》、《周穎南與中國文化》、《映華樓隨筆》、《六十年風雨歷程》等文學著作，使他被新加坡作家協會推選為「永久名譽理事長」。周穎南經常返回中國，與[[俞平伯|俞平伯]]、[[葉聖陶|葉聖陶]]、[[巴金|巴金]]、[[冰心|冰心]]、[[劉海粟|劉海粟]]、[[趙樸初|趙樸初]]、[[豐子愷|豐子愷]]、[[吳作人|吳作人]]、潘受、錢君匋、謝稚柳、[[啟功|啟功]]、梁披雲等中國當代知名作家、學者與書畫家多有交流<ref name="懷念周穎南"/>。

1993年，北京舉辦了「周穎南從事文學創作四十周年作品研討會」；2002年，匈牙利世界華文出版社《世界華人名人錄》出版《周穎南》專書。2003年6月，[[中國管理科學研究院|中國管理科學研究院]]人文科學研究所授予周穎南「全國人文科學優秀專家學者」稱號；同年9月，於第5屆國際儒商大會上，榮獲「國際儒商貢獻獎」。[[中國國務院|中國國務院]]授予「海外華人形象大使」榮銜<ref>{{cite web|url=http://news.sina.com.cn/o/2004-07-02/05182964754s.shtml|title=海外侨界人物荣誉展(组图)|date=|website=news.sina.com.cn|accessdate=2019-01-05|archive-url=https://web.archive.org/web/20190102143121/http://news.sina.com.cn/o/2004-07-02/05182964754s.shtml|archive-date=2019-01-02|dead-url=no}}</ref>。

周穎南於2014年2月27日因病逝世，享年86歲<ref>{{cite web|url=https://www.zaobao.com.sg/realtime/singapore/story20140227-315008|title=本地知名儒商周颖南逝世 - 早报|author=|date=|website=www.zaobao.com.sg|accessdate=2019-01-05|archive-url=https://web.archive.org/web/20190102095155/https://www.zaobao.com.sg/realtime/singapore/story20140227-315008|archive-date=2019-01-02|dead-url=no}}</ref>，他的傳記載錄於《龍的傳人在海外——30位傑出華人的成功之路》、《世界華人億萬富翁傳》及《世界名人錄》等專著<ref name="懷念周穎南"/>。
