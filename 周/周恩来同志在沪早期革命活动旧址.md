'''周恩来同志在沪早期革命活动旧址'''是中国上海的一处近现代重要史迹，位于[[虹口区|虹口区]][[四川北路街道|四川北路街道]]1953弄[[永安里|永安里]]44号。这是一幢建于1925年的三层石库门新式里弄住宅，是周恩来的堂伯母家的住所。1927年到1931年间，这里曾是周恩来夫妇为危急时期的一处秘密藏身地点。1931年[[顾顺章|顾顺章]]被捕后，周恩来夫妇秘密藏身于此，并从这里逃脱，到达苏区。
 
2009年，永安里44号周恩来同志在沪早期革命活动旧址被列为[[虹口区文物保护单位|虹口区文物保护单位]]。2014年4月4日，周恩来同志在沪早期革命活动旧址被列为第八批[[上海市文物保护单位|上海市文物保护单位]]<ref>{{cite web |language = |publisher =上海市文化广播影视管理局|title =上海市市级文物保护单位一览表|url = http://wgj.sh.gov.cn/node2/n1716/n1718/n1755/n1776/n1785/u1a81911.html|author =市文物局|date =2012-10-01|accessdate =|deadurl =yes|archiveurl = https://web.archive.org/web/20140714210255/http://wgj.sh.gov.cn/node2/n1716/n1718/n1755/n1776/n1785/u1a81911.html|archivedate =2014-07-14}}</ref> <ref>{{cite web |language =  |publisher = 上海市地方志办公室 |title = 上海市市级文物保护单位 |url = http://www.shtong.gov.cn/newsite/node2/node4429/node70351/node70359/index.html |author = 上海市地方志办公室 |date = |accessdate = }}</ref>。
