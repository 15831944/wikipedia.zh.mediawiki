'''周俊偉'''（'''Lawrence Chou Tsun Wai'''，{{bd|1979年|4月19日}}），是香港，加拿大籍[[演員|演員]]及[[歌手|歌手]]。

== 出道 ==
周俊偉是香港聲樂男高音[[林祥園|林祥園]]和香港知名歌手[[華娃|華娃]]之學生。周俊偉首次踏上舞台，是在1994年參加[[温哥華|温哥華]]的一次歌唱大賽，以13歲之齡贏得冠軍。 其後加入舞蹈團體影舞者擔任舞蹈員。14至16歲其間曾參加多次歌唱大賽，屢奪冠軍。17歲的周俊偉回流香港，經[[黄霑|黄霑]]的幫助下到香港各大唱片公司試音，被[[劉德華|劉德華]]前經理人[[李潤華|李潤華]]賞識而納入旗下，其後再簽約台灣[[BMG|BMG]]成為歌手。1997年在台灣BMG十週年演唱會及[[彭佳慧|彭佳慧]]校園巡迴演唱會初試啼聲，得到公司欣賞力棒為1998年度重點新人。1998年推出單曲EP與[[劉德華|劉德華]]合唱一曲（你別傻了），迅速成為傳媒焦點，更榮獲香港IFPI單曲銷量冠軍。及後正式推出個人專輯大碟《我愛你》，以《我愛你》一曲榮獲馬來西亞「動感三二一」金曲獎，亦成為當年電台情人節必播、校園中甚具名氣的經典金曲，更令他首次奪取Channel【V】華語榜最佳男歌手新人獎的榮譽。

2013年周俊偉參與了香港電視拍攝的兩部重頭劇集，在《[[警界線|警界線]]》和《[[驚異世紀|驚異世紀]]》擔當男主角；更參與頭炮綜藝節目《[[挑戰|挑戰]]》，到[[越南|越南]][[韓松洞|韓松洞]]（全世界最大洞穴）探險拍攝，憑著堅毅意志完成艱辛旅程；亦與香港電台的電視部合作無間，經常演出不同的寫實劇集；熱愛演出的周俊偉，曾演出香港經典舞台劇《潮性辦公室》，一連演出超過二十場，場場爆滿，觀眾爆笑連場。

周俊偉亦經常參與[[無綫電視|無綫電視]]的綜藝節目（[[獎門人系列|獎門人系列]]較多），但有趣的是周俊偉未正式與無綫電視合作過。

== 音樂創作 ==
2001年起周俊偉開始投身幕後音樂製作的道路，與范曉萱自資開設個人錄音室, 學習電腦錄音程式、作曲、編曲、填詞、混音, 嘗試一手包辦整個唱片製作程序。曾參與製作出不少獨特的流行音樂，如：[[ASOS|ASOS]]的『變態少女』、[[范曉萱|范曉萱]]『絕世名伶』等等，均得到極大的迴響。2003年周俊偉更積極推動獨立音樂的崛起，與[[范曉萱|范曉萱]]、[[金木義則|金木義則]]合組成團體[[福祿壽|福祿壽]]，自資推出首張音樂大碟[[福祿壽|福祿壽]]之『序』，更榮獲中國內地《第四屆華語音樂傳媒大獎》之「最佳電子音樂大獎」。

== 電影 ==
2001年周俊偉開始參與電影工作，從[[林海峰|林海峰]]介紹下主演了由[[卓韻芝|芝See菇Bi]]執導的電影「戀愛起義 - 不得了」，夥拍當年的新人[[蔡卓妍|蔡卓妍]]一同參與演出, 頗受大眾接受與認同，也迅速得到電影內行人的欣賞，令周俊偉得到更多在電影上的演出機會，成為電影新貴。及後再由電影監製[[曾志偉|曾志偉]]的帶領下，夥拍[[楊丞琳|楊丞琳]]主演了「初戀嗱喳麵」及導演[[彭氏兄弟|彭氏兄弟]]的「見鬼」，周俊偉的演出得到了大眾的歡迎與肯定，更憑著電影《初戀嗱喳麵》被提名角逐為第二十一屆[[香港電影金像獎|香港電影金像獎]]「最佳新演員」，展開了他在電影業的銀色旅途。其後他在電影作品多以不同型像的破格演出，更每每令觀眾眼前一亮，經典作品包括：戀愛起義、初戀嗱喳麵、見鬼、六樓后座、AV、生日快樂、指甲刀人魔、東風破……等
