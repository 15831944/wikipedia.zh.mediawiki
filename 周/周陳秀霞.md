'''周陳秀霞'''（{{bd|1958年|1月14日|||}}），[[親民黨|親民黨]][[臺南市|臺南市]]政治人物，前[[臺南縣議會|臺南縣議會]]議長[[周五六|周五六]]之妻，現任[[立法委員|立法委員]]。

==選舉==
===[[2016年中華民國立法委員選舉|2016年中華民國立法委員選舉]]===
原本[[親民黨|親民黨]]要推派[[周五六|周五六]]，但[[周五六|周五六]]不想再從政<ref>{{cite web|url=https://tw.news.yahoo.com/%E5%91%A8%E4%BA%94%E5%85%AD%E5%A4…|title=周五六太座陳秀霞 列親民黨不分區}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，但因台南鄉親反彈很大<ref>{{cite web |url=http://news.ltn.com.tw/news/politics/breakingnews/1519981|title=列親民黨不分區 陳秀霞坦言周五六不想再從政}}</ref>，最後推派她出來代表[[親民黨|親民黨]]不分區<ref>{{cite web |url=http://www.chinatimes.com/newspapers/20151126000652-260107|title=列橘營不分區第三 陳秀霞將為弱勢發聲}}</ref>，順利當選<ref>{{cite web |url=http://www.chinatimes.com/vote2016/legislator/pfp/03|title=周陳秀霞不分區立委候選人- 2016總統、立委大選}}</ref>。
