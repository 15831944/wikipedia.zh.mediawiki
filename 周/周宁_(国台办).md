'''周宁'''（{{bd|1960年|4月||}}）[[北京市|北京市]]人，[[中华人民共和国|中华人民共和国]]官员。

==生平==
1984年1月周宁加入[[中国共产党|中国共产党]]，1984年8月参加工作，[[中国政法大学|中国政法大学]]本科毕业，法学学士。1984年8月至1987年12月，任[[北京市|北京市]][[海淀区|海淀区]]司法局干部。1987年12月至1988年9月，任[[中央台办|中央台办]]干部。1988年9月至1992年1月，任中央台办四组主任科员。1992年1月至1994年6月，任中央台办综合局法规处副处长。1994年6月至2000年6月，任中央台办、[[国台办|国台办]]综合局法规处处长。2000年6月至2000年9月，任中央台办、国台办研究局四处处长。2000年9月至2003年9月，任[[中央人民政府驻香港特别行政区联络办公室|中央人民政府驻香港特别行政区联络办公室]]台务部四处处长。2003年9月至2005年11月，任中央台办、国台办港澳局副局长。2005年11月至2006年8月，任中央台办、国台办法规局副局长。2006年8月至2013年11月，任中央台办、国台办法规局局长。2013年11月至2014年7月，任中央台办、国台办研究局局长。2014年7月起，任中央台办、国台办主任助理、研究局局长<ref>{{Cite web |url = http://www.gwytb.gov.cn/gtb/zhouning/ |title = 周宁 |author =  |date =  |publisher = 中共中央台办、国务院台办  |language =  |accessdate = 2017-10-11 |quote =  }}</ref>。

周宁曾多次参加[[海峡两岸|海峡两岸]]事务性磋商。其中最有名的是产生“[[九二共识|九二共识]]”的“[[九二香港會談|香港谈判]]”。当时，周宁以[[海协会|海协会]]咨询部副主任的名义，与[[台湾|台湾]][[海基会|海基会]]法律服务处处长[[許惠祐|許惠祐]]作为双方的主谈代表，在香港开展第二次两岸公证书使用查证协商<ref>{{Cite web |url = http://news.sina.com.cn/o/2006-02-08/13278154348s.shtml |title = 评论：反“独”促统 法律先行 |author =  |date = 2006-02-08 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。
