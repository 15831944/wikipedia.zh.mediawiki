'''周水子站'''是位于[[中国|中国]][[辽宁省|辽宁省]][[大连市|大连市]][[甘井子区|甘井子区]]的一座火车站，建于1907年，现为[[沈阳铁路局|沈阳铁路局]]管辖的二等站。未来的[[大连快轨四号线|大连快轨四号线]]将要经过本站。

== 车站结构 ==
島式月台1面2线地上车站。

== 车站周边 ==
车站周边主要是居民住宅，西侧是[[大连周水子国际机场|大连周水子国际机场]]。

== 历史 ==
* 1907年 车站启用。
