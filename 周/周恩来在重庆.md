《'''周恩来在重庆'''》，2008年上映于[[中国大陆|中国大陆]][[中国中央电视台综合频道|中国中央电视台综合频道]]的战争剧，导演[[董亚春|董亚春]]，主演刘劲、黄薇、马晓伟、于小慧、韩月乔、唐国强。

==剧情==
1937年7月7日，[[卢沟桥事变|卢沟桥事变]]爆发，[[大日本帝国|大日本帝国]]开始了全面侵华战争，为了组成[[抗日民族统一战线|抗日民族统一战线]]，周恩来接受[[中共中央|中共中央]]的指示，去[[重庆市|重庆市]]联络[[中国国民党|中国国民党]]，两党在重庆市举行了重庆谈判，结成了抗日统一战线，为打败侵华日军而埋下伏笔。

==花絮==
该剧是“纪念周恩来诞辰110周年”的献礼剧。<ref name=" cat" >{{cite web |url=http://cul.sohu.com/20080303/n255478504.shtml |title=《周恩来在重庆》前晚开播 |accessdate=2008-03-03 |publisher=搜狐网 |work= |language =zh-cn}}</ref><ref name=" bull" >{{cite web |url=http://culture.people.com.cn/GB/40473/70988/6923417.html |title= 纪念周恩来诞辰110周年《周恩来在重庆》登陆央视|accessdate=2008 |publisher=人民网 |work= |language =zh-cn}}</ref>

该剧的制作方是中共重庆市委宣传部、重庆广播电视集团、江北区区委区政府和[[八一电影制片厂|八一电影制片厂]]。<ref>{{cite web |url=http://ent.ifeng.com/movie/news/mainland/detail_2009_02/16/69239_0.shtml |title=《周恩来在重庆》通过广电总局审看 2月底隆重推出  |accessdate=2009年02日16日  |publisher=凤凰网 |work= |language =zh-cn}}</ref>

该剧的主要内容是汪精卫与大日本帝国合作、[[重庆大轰炸|重庆大轰炸]]、[[皖南事变|皖南事变]]、[[重庆谈判|重庆谈判]]、[[双十协定|双十协定]]。<ref name=" bull" />

该剧是刘劲第34次出演周恩来。<ref name=" cat" /><ref name="fish " >{{cite web |url=http://cpc.people.com.cn/GB/n/2012/1008/c87228-19183308.html |title=《周恩来在重庆》：表现周恩来的大情大爱 |accessdate=2012年10月8日 |publisher=中国共产党新闻网 |work= |language =zh-cn}}</ref>

编剧王朝柱写该剧几度落泪。<ref name="fish " />

该剧擅自使用[[陶鹏|陶鹏]]《[[川江船夫号子|川江船夫号子]]》作为配乐而被告上法庭。<ref>{{cite web|url=http://www.cq.xinhuanet.com/news/2009-03/05/content_15869595.htm|title=《周恩来在重庆》擅自用《川江船夫号子》作配乐被判侵权|accessdate=2009年3月5日|publisher=新华网|work=|language=zh-cn}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
