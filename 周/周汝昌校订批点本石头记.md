'''周汝昌校订批点本石头记'''，2010年1月1日最早由[[漓江出版社|漓江出版社]]出版发行。作者为[[中国|中国]]著名[[红学家|红学家]][[周汝昌|周汝昌]]。

== 简介 ==
该书为[[周汝昌|周汝昌]]六十年来对各个《[[红楼梦|红楼梦]]》版本进行考证、对比、总结而成，集正文、[[脂评|脂评]]与周按于一体的《红楼梦》。<ref name="xlwgs " >{{cite web |url=http://sh.sina.com.cn/book/new/2010-04-07/080127.html |title=周汝昌校订批点本石头记 |accessdate=2010年4月7日 |work=|publisher= 新浪网|language=zh-hans}}</ref>

== 特色 ==
该书一共分为三大部分，其一：对十余种《[[红楼梦|红楼梦]]》古抄本的汇校；其二：梳理和核对[[脂批|脂批]]；其三：列出周汝昌自己的批注，称为“周按”。<ref name="xlwgs " />

== 创作过程 ==
该书倾注了周汝昌六十年的心血，于其九十岁高龄才得以完成和面世。<ref name="xlwgs " />

== 作者简介 ==
周汝昌，本字禹言，号敏庵，后改字玉言，籍贯天津咸水沽。笔名：念述、苍禹、雪羲、顾研、玉工、石武、玉青、师言、茶客等。周汝昌是[[中华人民共和国|中华人民共和国]]研究《红楼梦》第一人，享誉海内外的[[考证派|考证派]]主力和集大成者。<ref>{{cite web |url=http://culture.people.com.cn/GB/42223/140260/|title=《红学泰斗——周汝昌》|accessdate=2008|work=|publisher=人民网|language= zh-hans}}</ref>
