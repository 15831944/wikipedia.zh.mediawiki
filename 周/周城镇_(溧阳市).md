'''周城镇'''，是{{PRC admin/fullname|32/04/81/117/000|3}}下辖的一个已撤销[[乡镇|乡镇]]级行政单位，位于溧阳市西南部。2007年撤销。

==历史==
北宋设從山鄉，南宋增设周城步，明朝從山鄉改为從山區。中华民国时期重建县下区划，设置周城镇，一度作为[[县辖区_(中华民国)|县辖区]]驻地，1949年拆分乡镇后改为周城乡，隶属於县辖社渚区，1957年后直属於[[溧阳县|溧阳县]]，1958年后改称周城[[人民公社|公社]]，1960年至1961年划归[[社渚镇|社渚公社]]，1983年后复称周城乡。

1988年周城撤乡建镇，2007年撤销周城镇，原辖地大部划归[[社渚镇|社渚镇]]，东部少部划归[[天目湖镇|天目湖镇]]<ref>{{cite web |title=省政府关于同意调整溧阳市部分镇行政区划的批复 |url=http://www.jiangsu.gov.cn/art/2007/3/17/art_46143_2543681.html |website=江苏省政府网 |publisher=江苏省人民政府 |accessdate=2018-08-10 |date=2007-03-17}}</ref>。

== 行政区划 ==
撤销前周城镇下辖以下地区：

周城镇社区、梅山村、金庄村、周城村、汤山村、金峰村、金山村、欣丁村、洙漕村和濮家村。
