'''周二补丁日'''（{{lang-en|'''Patch Tuesday'''}}）也称'''补丁星期二'''、'''星期二更新'''（'''Update Tuesday'''<ref>{{cite web|url=http://blogs.windows.com/bloggingwindows/2014/08/05/august-updates-for-windows-8-1-and-windows-server-2012-r2/|title=August updates for Windows 8.1 and Windows Server 2012 R2|work=Windows Experience Blog|accessdate=25 November 2015}}</ref>）等，它是指[[微軟|微軟]]于每个月的第二個星期二定期發佈软件更新补丁的非正式术语，这在行业内被广泛使用<ref>{{cite web | url = http://news.cnet.com/8301-10805_3-20118106-75/microsoft-patch-tuesday-to-target-windows-ie/ | title = Microsoft Patch Tuesday to target Windows, IE | publisher = CNet | date = October 10, 2011  | accessdate = November 9, 2011}}</ref><ref>{{cite web|url = http://blogs.technet.com/b/blairn/archive/2006/03/28/netfx1164annc.aspx|title = .NET Framework 1.1 Servicing Releases on Windows Update for 64-bit Systems|publisher = Microsoft|date = March 28, 2006|accessdate = November 8, 2011|deadurl = yes|archiveurl = https://web.archive.org/web/20120327232948/http://blogs.technet.com/b/blairn/archive/2006/03/28/netfx1164annc.aspx|archivedate = 2012年3月27日}}</ref><ref>{{ cite web | url = http://windows.microsoft.com/en-us/windows/understanding-windows-automatic-updating | title = Understanding Windows automatic updating | publisher = Microsoft — Understanding Windows — Get Help | accessdate = July 3, 2014 }}</ref>。微软于2003年10月正式开始了此模式<ref>{{cite web|last1=Budd|first1=Christopher|title=Ten Years of Patch Tuesdays: Why It’s Time to Move On|url=http://www.geekwire.com/2013/ten-years-patch-tuesdays-time-move/|website=GeekWire|accessdate=28 July 2015}}</ref>。除对[[Microsoft_Windows|Microsoft Windows]]的补丁外，還有[[Microsoft_Office|Microsoft Office]]、[[Visual_Studio|Visual Studio]]和[[SQL_Server|SQL Server]]等產品的补丁。用戶可从[[Windows_Update|Windows Update]]下載和安装补丁。

==傳統==
此傳統从[[Windows_98|Windows 98]]開始，[[UTC|UTC]]时间每个月的第二個星期二17:00至18:00之間发布补丁。
==例外==
*每天更新：[[Windows_Defender|Windows Defender]]的更新。
*一般緊急补丁：在定期补丁後14天釋出。
*非常緊急补丁：不定期釋出。
