'''周士第'''（{{bd|1900年|9月9日|1979年|6月30日|catIdx=Zhou Shidi}}），又名'''周力行'''、'''周平'''、'''周士悌'''、'''周士弟'''，[[廣東|廣東]]樂會（今[[海南|海南]][[瓊海|瓊海]]）人，[[中国人民解放军上将|中国人民解放军开国上将]]。

周士第曾任[[孙中山|孙中山]]陆海军大元帅府铁甲车队副队长、队长，1924年從[[黃埔軍校|黃埔軍校]]第一期畢業，同年加入[[中國共產黨|中國共產黨]]，參加過[[北伐|北伐]]和[[南昌起義|南昌起義]]。1928年脱党，1934年重回[[中央苏区|中央苏区]]，任军事教员，參與[[長征|長征]]。1935年重新入党，任[[红十五军团|红十五军团]]参谋长，後任[[红二方面军|红二方面军]]参谋长。[[抗戰|抗戰]]初任[[八路軍|八路軍]][[120師|120師]]參謀長，[[國共戰爭|國共戰爭]]時期任第十八兵團司令員，[[中華人民共和國|中華人民共和國]]成立后，任川西军区司令员兼成都市市长，西南军区副司令员，中国人民解放军防空部队司令员，[[训练总监部|训练总监部]]副部长兼外军训练部部长，[[总参谋部|总参谋部]]顾问。曾任第一至第三届[[國防委員會|國防委員會]]委員、第三、第四届[[全国政协|全国政协]]常委、第一、第四、第五届[[全國人民代表大會|全國人民代表大會]]代表、第五屆[[全國人民代表大會常務委員會|全國人民代表大會常務委員會]]委員，[[中國共產黨|中國共產黨]]第七、八次全國代表大會代表，第八届中央监察委员会委员。1955年被授予上將军銜，获一级[[八一勋章_(1955年)|八一勋章]]、一级[[独立自由勋章|独立自由勋章]]、一级[[解放勋章|解放勋章]]。

== 生平 ==
=== 早年 ===
周士第早年曾参与[[五四运动|五四运动]]，被选为琼崖学生联合会干事。1920年起，为乐会第二区高等小学教员，1923年6月参加中共外围组织“新琼崖评论社”，出版《新琼崖评论》。1924年5月考入黄埔军校第一期学习，11月任孙中山的建国陆海军大元帅府铁甲车队见习官。12月加入[[中国共产党|中国共产党]]。1925年2月任铁甲车队副队长，6月任队长<ref>{{cite web|url=http://www.hq.xinhuanet.com/news/2011-08/01/content_23363950_7.htm|title=两任队长都是海南人 神秘之师铁甲车队|work=海南日报|publisher=新华网海南频道|5=|date=2011-08-01|accessdate=2014-11-07|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20141107235357/http://www.hq.xinhuanet.com/news/2011-08/01/content_23363950_7.htm|archivedate=2014-11-07}}</ref>。同年4月兼任广州革命政府航空局“飞机掩护队”队长，参与了[[廣州商團叛亂|平定广州商团叛乱]]、支援[[省港大罢工|省港大罢工]]的斗争。

[[File:Zhou_Shidi_Huangpu.jpg|thumb]]
1925年底，铁甲车队改编为[[叶挺独立团|国民革命军第四军独立团]]，[[叶挺|叶挺]]为团长，周士第任一营营长。1926年5月任独立团参谋长，10月任代理团长，参与北伐战争，在[[汀泗桥战役|汀泗桥]]、[[贺胜桥战斗|贺胜桥]]、[[武昌|武昌]]等战役中立有战功。1926年底，任[[国民革命军第四军|国民革命军第四军]]第二十五师第七十三团团长。

=== 第一次国共内战时期 ===
1927年8月1日，周士第参与中共的[[南昌起义|南昌起义]]，任起义军第二十五师师长。后南昌起义军南下，一路遭受重大损失。10月，周士第与[[李硕勋|李硕勋]]前往香港寻找党组织。1927年年底，周士第到达香港，向[[李立三|李立三]]做了汇报。后周士第染上疟疾，但当时党组织的经费十分紧缺，其同乡[[陈超鹏|陈超鹏]]劝说周士第赴南洋治病。1928年1月，周士第听从了陈超鹏的意见，不告而别，与中共脱离组织关系。

1928年冬天，周士第回国，参加[[中国国民党临时行动委员会|中国国民党临时行动委员会]]黄埔革命同学会<ref>{{cite web |url =http://www.people.com.cn/GB/198221/198458/12399705.html|title=周士第加入过第三党 曾谋划刺杀蒋介石|work=|publisher=人民网||date=2010-08-10|accessdate=2014-11-07|language= zh-hans}}</ref>。1930年春，周士第奉[[邓演达|邓演达]]之命去陕西西安，建立中国国民党临时行动委员会陕西省省委，改名周平。1931年夏被捕，又先后被送押南京[[国民党中央党部|国民党中央党部]]、军法司拘留所关押，在[[宋庆龄|宋庆龄]]的要求下获释<ref>{{cite web |url =http://news.sohu.com/20100920/n275160772.shtml|title=周士第反省出走南洋之举 被捕后宋庆龄两度营救|work=|publisher=搜狐网||date=2010-09-20|accessdate=2014-11-07|language= zh-hans}}</ref>。1933年夏，周士第被派到福建支援[[闽变|闽变]]，到第四十九师任团长，此时与中共恢复联系。闽变失败后，[[十九路军|十九路军]]残部被[[陈济棠|陈济棠]]缴械遣散，周士第迅速逃离<ref>{{cite book |author=严如平、郑则民 |title=蒋介石传 上|year=2013|publisher=中华书局 |location=北京|pages=255|isbn=978-7-1010-9135-9|language =zh-hans}}</ref>，于1934年2月进入[[中央苏区|中央苏区]]，任红军大学军事教员。

1934年10月参加[[长征|长征]]，任干部团上干队指挥科科长。1935年7月，接替[[萧劲光|萧劲光]]任上干队队长。1935年10月，周士第在到达陕北前夕重新加入中国共产党。1935年11月，周士第任[[红十五军团|红十五军团]]参谋长，协助[[徐海东|徐海东]]指挥[[直罗镇战役|直罗镇战役]]和[[东征战役|东征战役]]。1936年1月兼任红军东征河口渡河司令员，同年8月至1937年1月任保安[[红军大学|红军大学]]第二科科长。1936年12月，任[[红二方面军|红二方面军]]参谋长兼[[红二军团|红二军团]]参谋长。<ref name="欧阳青">{{cite book |author=欧阳青 |title=大授衔：1955共和国将帅授衔档案|year=2011|publisher=长城出版社 |location=北京|pages=420-423|isbn=978-7-5483-0058-8 |language =zh-hans}}</ref>
[[File:HeGuanZhou.jpg|thumb]]（中）、[[关向应|关向应]]（右）在冀西北前线。]]
=== 抗日战争时期 ===
抗日战争爆发后，红二方面军改编为[[八路军|八路军]]第[[120师|120师]]，周士第任师参谋长，协助[[贺龙|贺龙]]指挥[[雁门关伏击战|雁门关伏击战]]，参与创建晋西北抗日根据地。1939年2月率部挺进冀中，兼任冀中区总指挥部参谋长、冀中军政委员会委员，参与指挥[[黑马张庄伏击战|黑马张庄伏击战]]、[[齐会战斗|齐会战斗]]、[[陈庄战斗|陈庄战斗]]<ref>{{cite web|url=http://hebei.ifeng.com/news/detail_2014_07/21/2627723_0.shtml|title=
陈庄亲历者说：张宗逊将军回忆陈庄歼灭战|date=2014-07-21|publisher=凤凰网|accessdate=2014-11-07}}</ref>、[[黄土岭战斗|黄土岭战斗]]等<ref name="欧阳青"/>。

1940年10月，任晋西北军政委员会委员，参与[[百团大战|百团大战]]。同年11月任八路军第120师兼[[晋西北军区|晋西北军区]]参谋长、军区交通司令员。1941年7月起，兼任[[中国人民抗日军政大学|中国人民抗日军政大学]]第七分校校长。1942年1月，参与指挥晋绥春季反“扫荡”，并取得[[田家会战斗|田家会战斗]]的胜利<ref>[http://www.balujun.org/blsh/dxzl/636.html 田家会战斗]</ref>。1942年5月，任中共晋绥分局委员，10月任八路军第120师兼[[晋绥军区|晋绥军区]]参谋长。1943年10月，任晋绥军区副司令员。1944年2月兼任晋绥分局对敌斗争委员会副书记、财政经济委员会委员。1944年6月因病回延安休养，1945年出席了[[中共七大|中共七大]]。

=== 第二次国共内战时期 ===
[[File:Xxq73-2.jpg|thumb]]、周士第、[[王世英|王世英]]、[[滕代远|滕代远]]]]
[[第二次国共内战|第二次国共内战]]时期，任晋绥军区副司令员。1946年6月，兼任[[晋北野战军|晋北野战军]]司令员和政治委员，指挥[[晋北战役|晋北战役]]，歼灭[[阎锡山|阎锡山]]部国军8,100余人，攻占五座城市。其中攻克[[崞县|崞县]]的战斗为[[毛泽东|毛泽东]]所欣赏，毛专门电示贺龙：“攻崞经验甚好”<ref name="欧阳青"/>。1947年9月，兼任中共中央晋绥分局城工部副部长。11月，任晋绥军区党委书记。1948年3月，兼任[[陕甘宁晋绥联防军区|陕甘宁晋绥联防军区]]军政干部学校副校长、党委书记。

1948年5月，周士第调任[[华北军区|华北军区]]第一兵团副司令员兼副政治委员，协助[[徐向前|徐向前]]指挥[[晋中战役|晋中战役]]。后又协助徐向前和[[彭德怀|彭德怀]]指挥[[太原战役|太原战役]]<ref>{{cite book |author=徐向前|title=《历史的回顾》|year=1987年 |publisher=解放军出版社 |location=北京市西城区地安门西大街40号|pages=761-762页 |isbn=7506501260|language = zh-cn}}</ref>，任太原前线司令部副司令员兼副政治委员，攻占整个山西。1949年2月，周士第任[[中国人民解放军|中国人民解放军]]第18兵团副司令员兼副政治委员，5月接替徐向前任司令员兼政治委员，随后率部加入[[第一野战军|第一野战军]]序列，进军西北战场，参与[[扶眉战役|扶眉战役]]和[[兰州战役|兰州战役]]<ref>{{cite web|url=http://military.china.com/history4/62/20140819/18720218_all.html|title=解放军鏖战大西北：与西北“四马”骑兵的血腥较量|date=2014-08-19|publisher=中华网|accessdate=2014-11-07}}</ref>。1949年8月，任[[中共中央西北局|中共中央西北局]]委员、第一野战军党委常务委员。

=== 中华人民共和国成立后 ===
1949年11月，周士第任[[中共中央西南局|中共中央西南局]]委员。12月，周士第在[[贺龙|贺龙]]指挥下进军西南，由陕入川参与[[秦岭战役|秦岭战役]]和[[成都战役|成都战役]]<ref name="欧阳青"/>，攻占[[汉中|汉中]]、[[广元|广元]]、[[剑阁|剑阁]]、[[江油|江油]]、[[绵阳|绵阳]]等40多座城市。1949年12月。任中国人民解放军[[成都市|成都市]]军事管制委员会副主任兼成都市市长。1950年1月，任中共川西区委委员、统战部部长。同年2月，任[[西南军区|西南军区]]副司令员、[[川西军区|川西军区]]司令员兼政治委员<ref>{{cite book |author=张明金、刘立勤 |title=《中国人民解放军历史上的200个军区》|year=2010年 |publisher=解放军文艺出版社 |location=北京市西城区地安门西大街40号 |pages=534、546页 |isbn=9787503322525 |language = zh-cn}}</ref>。同年6月至1953年3月任[[西南军政委员会|西南军政委员会]]委员。

1950年10月，任[[中国人民解放军防空军|中国人民解放军防空军]]司令员，曾亲自到东北地区和朝鲜战场进行实地勘察，参与指挥[[朝鲜战争|朝鲜战争]]防空作战。1953年4月任解放军[[训练总监部|训练总监部]]副部长，1955年5月，兼军外训练部部长，9月被授予上将军衔，荣获一级[[八一勋章_(1955年)|八一勋章]]、一级[[独立自由勋章|独立自由勋章]]、一级[[解放勋章|解放勋章]]。

1959年，周士第离职休养。1962年9月，被选举为[[中国共产党中央监察委员会|中国共产党中央监察委员会]]委员。[[文化大革命|文化大革命]]时期未遭迫害<ref>{{cite web|url=http://news.ifeng.com/history/zhongguoxiandaishi/detail_2011_05/10/6279877_0.shtml|title=曾脱党七年的开国上将如何躲过“文革”浩劫？|date=2011-05-10|publisher=凤凰网|accessdate=2014-11-07}}</ref>，1978年2月，任解放军[[总参谋部|总参谋部]]顾问。1979年6月30日在[[北京|北京]]逝世。<ref>{{cite web |url =http://www.gov.cn/gjjg/2009-02/02/content_1219215.htm|title=周士第|work=|publisher=中央人民政府门户网站||date=2009-02-02|accessdate=2014-11-07|language= zh-hans}}</ref>
