'''周倪安'''{{BD|1972年|10月7日||catIdx=Zhou周}}[[臺灣|臺灣]][[政治人物|政治人物]]，臺北市人，畢業於[[中山女中|中山女中]]、[[國立臺灣大學|國立臺灣大學]]等校。前任[[臺灣團結聯盟|臺灣團結聯盟]]全國不分區[[立法委員|立法委員]]，以及該黨中執委、政策部副主任、立法院黨團幹事長。

== 2008年第七屆立法委員選舉 ==
雖於2007年才進入[[台灣團結聯盟|台灣團結聯盟]]後，即於翌年便代表台聯參加首次[[單一選區兩票制|單一選區兩票制]]的第七屆立委選舉，成為當年台聯十三位區域候選人之一。該次選舉在台北縣第九選區（[[永和區|永和]]及[[中和區|中和]]十七里）[[民進黨|民進黨]]並未推出候選人的情況下，成為該選區唯一舉綠旗的參選人。面對分裂的泛藍陣營，選舉結果僅得到15,669票（得票率11.05％）敗選，未得到該選區傳統藍綠7比3的三成票數。<ref>[http://www.bas.ntpc.gov.tw/web66/_file/1528/upload/statistic/electionpeopl10009.doc　臺北縣選民結構分析] {{webarchive|url=https://web.archive.org/web/20140312213913/http://www.bas.ntpc.gov.tw/web66/_file/1528/upload/statistic/electionpeopl10009.doc |date=2014-03-12 }}新北市政府主計處</ref>該選區[[泛藍|泛藍]]參選人，[[國民黨|國民黨]]籍連任立委[[林德福|林德福]]得98,634票（得票率69.61％）；[[無黨籍|無黨籍]]（原[[國民黨|國民黨]]籍永和市長，脫黨參選）參選人洪一平得27,044票（得票率19.09％），該次選舉亦是[[永和區|永和]]地區選舉史上，[[泛綠|泛綠]]首次僅獲得一成選票。次屆立法委員選舉，[[民進黨|民進黨]]參選人[[許又銘|許又銘]]再次面對分裂的泛藍陣營（[[國民黨|國民黨]][[林德福|林德福]]、[[雷倩|雷倩]]）獲得27.55%選票，終於逐漸回歸該區三成[[泛綠|泛綠]]選票。

台聯在2008年立法委員選舉中，13個區域立委候選人得票數共93,840票；政黨票共得344,887票，得票率3.53%，在[[單一選區兩票制|單一選區兩票制]]下沒跨過5%門檻，未能取得任何立委席次。

== 2010年台北市議員選舉 ==
2010年，任職台聯中央黨部組織部副主任，離開短暫經營的台北縣，改投入[[台北市|台北市]]第六選區（[[大安區_(臺北市)|大安區]]、[[文山區|文山區]]）議員選舉，在21位候選人中，以4,092票（得票率1.36％）排名18，再次敗選。成為該次選舉中，該選區有[[政黨|政黨]]提名之候選人中，得票數最低的候選人。<ref>[http://db.cec.gov.tw/histQuery.jsp?voteCode=20101101T1B2&qryType=ctks&prvCode=01&cityCode=000&areaCode=06　99年直轄市議員選舉（區域） 候選人得票數] 中選會資料庫</ref>

== 2012年第八屆立法委員選舉 ==
在台聯的政策制度下，採取立委輪流當的政治首例，第一輪立法委員[[許忠信|許忠信]]、[[黃文玲|黃文玲]]兩年任期屆滿，於2014年2月1日起請辭，依序由周倪安與[[賴振昌|賴振昌]]遞補不分區立委；2014年2月7日中選會頒發當選證書，三日後於立法院宣誓正式成為中華民國第八屆立法委員。

== 2014年太陽花學運期間遭警毆打事件 ==
2014年3月18日因[[海峽兩岸服務貿易協議存查事件|服貿爭議]]爆發[[太陽花學運|太陽花學運]]，反服貿學生於3月24日轉攻[[行政院|行政院]]，遭到大批警力鎮壓。周倪安到場聲援學生，也遭到警方用警棍敲頭，並被踹倒在地，<ref>{{cite news|url=http://news.ltn.com.tw/news/focus/paper/764988|title=台聯立委周倪安 被警棍敲暈|publisher=自由時報|date=2014-03-25|language=zh-tw}}</ref>造成她眼骨碎裂、肋骨[[骨折|骨折]]、大腿挫傷，以及[[腦震盪|腦震盪]]等，被送往[[台大醫院|台大醫院]]並住院－周。<ref>{{cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20140324/365997/|title=【挺學生】台聯立委周倪安骨折、腦震盪住院|publisher=《蘋果日報》|date=2014-03-24|language=zh-tw}}</ref>事後台聯主席[[黃昆輝|黃昆輝]]表示要協助學生對警方提告，<ref>{{cite news|url=http://www.chinatimes.com/realtimenews/20140325003097-260407|title=立委周倪安受傷 台聯揚言提告|publisher=《中國時報》|date=2014-03-25|language=zh-tw}}</ref>民進黨立委[[吳宜臻|吳宜臻]]則要求檢方還原真相。<ref>{{cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20140403/372323/|title=台聯周倪安被打　立委要求檢方快找兇手|publisher=《蘋果日報》|date=2014-04-03|language=zh-tw}}</ref>3月31日警政署長[[王卓鈞|王卓鈞]]前往台大醫院病房探視，經民進黨立委要求，向周道歉，但調查仍未有進展。4月17日周倪安在[[立法院|立法院]]會質詢，控訴[[行政院長|行政院長]][[江宜樺|江宜樺]]殺人未遂，並表示自己仍有頭痛的後遺症。<ref>{{cite news|url=http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS1/8621573.shtml|title=周倪安：江宜樺殺人未遂|publisher=《聯合報》|date=2014-04-18|language=zh-tw}}</ref>6月27日，周倪安召開記者會表示，已經以「殺人未遂」罪名，對行政院長江宜樺等人提出刑事自訴，並要求江宜樺親自出庭，否則江應辭職下台以示負責。<ref>{{cite news|url=http://news.ltn.com.tw/news/politics/breakingnews/1041741|title=324學運驅離遭毆打 周倪安控江宜樺殺人未遂|publisher=《自由時報》|date=2014-06-27|language=zh-tw}}</ref>

== 2016年轉為平民 ==
投入2016年立法委員選舉（新北市第九選區），由於該選區為[[泛綠|泛綠]]陣營所評估之艱困選區，透過民進黨選舉對策委員會召集人[[蘇嘉全|蘇嘉全]]協調，與同選區獨立參選人[[李幸長|李幸長]]進行民調整合，最終由[[李幸長|李幸長]]勝出。周倪安因而宣布將退出2016立委選舉「轉為平民」<ref>{{cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20151111/730526/|title=民調整合輸李幸長　周倪安宣布「轉為平民」|publisher=《蘋果即時》|date=2015-11-11|language=zh-tw}}</ref>，不過依舊名列台聯不分區立委參選名單第九名，最後因台聯之政黨票僅獲2.5%，連任失敗。
