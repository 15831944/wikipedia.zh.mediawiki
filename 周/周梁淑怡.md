'''周梁淑怡'''{{GBSHK}}{{OBE|C}}{{JP}}（{{lang-en|'''Selina Chow Liang Shuk-yee'''}}，{{bd|1945年|1月25日|catIdx=Zhou周}}），生於[[香港|香港]]，籍貫[[廣東|廣東]][[番禺|番禺]]，[[中國人民政治協商會議|全国政协]]委员，是香港著名[[電視|電視]]界人物，[[自由黨_(香港)|自由黨]]榮譽主席，前[[香港行政會議|行政會議]]成員，前[[香港立法會|立法會]][[新界西選區|新界西選區]][[議員|議員]]、前[[旺角區議會|旺角區議會]]議員。

周梁淑怡是[[香港電視廣播|香港电视]]发展史的重要人物，其曾任香港三大免费电视台（[[無綫電視|無綫電視]]、已結束免費電視廣播的[[亚洲电视|亚洲电视]]、已結業的[[佳艺电视|佳艺电视]]）的管理层，为香港电视界、电影界、乐坛培育了台前幕后的多位殿堂级人才。周梁淑怡以“唯才是用”、“人尽其才”等方法，提携多位各具特点的创作、制作人才，真正是[[香港演藝圈|香港演藝圈]]的金牌伯乐。其提拔的徒子徒孙，如[[王晶_(導演)|王晶]]、[[甘国亮|甘国亮]]、[[谭家明|谭家明]]、[[许鞍华|许鞍华]]、[[杜琪峰|杜琪峰]]、[[徐克|徐克]]等均称为香港影视界的中流砥柱。周梁淑怡主张以“创作主导制作”、“创作主导营运”的创新精神经营电视台<ref>[http://v.youku.com/v_show/id_XNDI2OTgxMjEy.html 《亚视百人》101-周梁淑怡] 2012-06-20</ref>。

==教育==
她曾就讀[[聖保羅男女小學|聖保羅男女小學]]及[[聖保羅男女中學|聖保羅男女中學]]、[[香港大學|香港大學]]。由於她的父母喜愛[[京劇|京劇]]、[[粵曲|粵曲]]和[[紹興戲|紹興戲]]，而且她父親[[梁基浩|梁基浩]]經營電影發行及戲院生意，她便對[[戲劇|戲劇]]及[[電影|電影]]產生了興趣。<ref name="文匯">{{cite news |language = zh-hk |author = |coauthors = |url = http://paper.wenweipo.com/2007/01/24/MR0701240001.htm|title = 周梁淑怡 從政無野心 做官當為民 |work = |publisher = 文匯報 |pages = A30 |date = 2007-01-24 |accessdate = 2008-09-07}}</ref>後來她到[[英國|英國]][[羅斯布魯佛朗誦及戲劇學院研究院|羅斯布魯佛朗誦及戲劇學院研究院]]作研究，並且在[[英國皇家音樂學院|英國皇家音樂學院]]戲劇院擔任教師及演員。

==電視生涯==
周梁淑怡是香港第一代資深電視人，和[[電視廣播有限公司|無綫電視]]（TVB）一起陪著很多香港人一起成長，与[[蔡和平|蔡和平]]一样20出头已经是TVB的领军人物。為香港培養了無數幕前幕後的電視人才。她製作過不少膾炙人口的電視劇如《[[狂潮|狂潮]]》、《[[家變_(1977年電視劇)|家變]]》等，至今仍為人津津樂道。

1974年出任節目發展主任；一年後升任節目及製作經理；1976年—1977年期間出任助理總經理。

1978年初，[[佳藝電視|佳藝電視]]以高薪羅致周梁淑怡出任總經理，但因「[[佳視#七月攻勢|七月攻勢]]」計畫失敗與「[[佳視六君子|佳視六君子]]」引咎辭職。

1980年代初，出任[[富才製作有限公司|富才製作有限公司]]主席及行政總裁。

1987年，[[林百欣|林百欣]]邀請周梁淑怡出任[[亞洲電視|亞洲電視]]行政總裁，到1991年任董事及顧問，1993年離任。

2009年9月，她再次踏入電視圈，每週主持[[有線電視財經資訊台|有線電視財經資訊台]]節目《[[周梁有請|周梁有請]]》。

==從政時期==
在1981年被委任為[[立法局|立法局]]議員，後來出選[[功能組別|功能組別]]。自1981年，她便歷任立法局議員、[[臨時立法會|臨時立法會]]議員、[[香港立法會|立法會]]議員，是香港有史以來擔任香港立法機關時間最長的議員。

作為[[自由黨_(香港)|自由黨]]在[[行政會議|行政會議]]的代表，她表示會致力將市民的意見帶入[[行政會議|行政會議]]，促進行政會議和[[香港立法會|立法會]]的互相了解。

2003年，自由黨因民情轉而反對[[二十三條立法|二十三條立法]]，她的民望非常高，故在[[2004年香港立法會選舉|2004年香港立法會選舉]]，第一次代表[[自由黨_(香港)|自由黨]]於[[新界西|新界西]]參與地區直選，首次成為[[地區直選|地區直選]][[香港立法會|立法會]]議員。

[[2008年香港立法會選舉|2008年香港立法會選舉]]，周梁淑怡在新界西地區直選中落敗，決定辭去自由黨副主席，亦向時任行政長官[[曾蔭權|曾蔭權]]辭去行政會議成員。曾蔭權表示會因應政治形勢，對請辭作出慎重考慮。<ref name="東方">{{cite news |language = zh-hk |author = |coauthors = |url = http://orientaldaily.on.cc/archive/20080909/new/new_a03cnt.html |title = 田北俊周梁淑怡齊辭職 |work = 東方日報 |publisher = |pages = A02 |date = 2008-09-09 |accessdate = 2008-09-09 }}</ref>最終曾蔭權於9月19日接納其請辭。<ref>[http://www.mpinews.com/htm/INews/20080919/gb51125w.htm 特首接納周梁辭任行會成員] {{webarchive|url=https://web.archive.org/web/20080923233737/http://www.mpinews.com/htm/INews/20080919/gb51125w.htm |date=2008-09-23 }}</ref>

2012年12月，周梁淑怡在沒有競爭的情況下，自動當選自由黨主席，接替因落選立法會議員而在9月辭任主席的[[劉健儀|劉健儀]]，而立法會議員[[方剛|方剛]]及[[鍾國斌|鍾國斌]]則擔任副主席。其後重返立法會的[[田北俊|田北俊]]成為首任自由黨黨魁，自由黨正式實施「黨政分家」。

==旅遊發展局==
周梁淑怡在2000年被委任為[[香港旅遊協會|香港旅遊協會]]主席；2001年旅遊協會改組為[[香港旅遊發展局|香港旅遊發展局]]後，她亦出任主席。任內，她致力加強旅遊發展局與業界的聯繫，聯手推廣[[香港旅遊業|香港旅遊]]。在[[嚴重急性呼吸系統綜合症|SARS]]疫症期間，香港旅遊業陷入低谷，經過旅發局與業界的努力，旅遊業發展才逐步回復。
