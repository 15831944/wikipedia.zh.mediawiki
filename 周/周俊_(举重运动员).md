'''周俊'''（{{bd|1995年|2月27日||}}）是一位[[中国|中国]]女子[[举重|举重]]运动员，出生于[[湖北省|湖北省]][[秭归县|秭归县]]。2004年6月入选秭归县体校，2005年由县体校选送到[[宜昌市|宜昌市]]体校，2009年入选湖北省举重队。

2010年4月8日，她代表中国国家队参加在[[吉尔吉斯斯坦|吉尔吉斯斯坦]]举行的亚洲青年举重锦标赛中夺得女子组53公斤级[[抓举|抓举]]、[[挺举|挺举]]、总成绩三块金牌。2012年4月26日至30日，周俊代表中国参加了在[[韩国|韩国]]举行的亚洲女子举重锦标赛女子58公斤级比赛，以抓举95公斤、挺举125公斤、总成绩220公斤分别获得抓、挺、总三项第三名。<ref>{{cite web | language = zh-cn | publisher = 荆楚网| title =周俊曾在亚洲青年举重锦标赛中夺三块金牌| url = http://sports.cnhubei.com/2012/hb/mj/201207/t20120717_10589.shtml| date = 2012-07-17 | accessdate = 2012-07-30 }}</ref>

2012年7月，周俊代表中国出战在[[英国|英国]][[伦敦|伦敦]]举行的[[2012年夏季奥林匹克运动会|奥林匹克运动会]][[2012年夏季奥林匹克运动会举重比赛|举重比賽]][[2012年夏季奥林匹克运动会举重比赛－女子53公斤级|女子53公斤级]]賽事，这被媒体称为最大冷门之一，是有史以来秭归籍运动员第一次参加奥运会比赛项目，周俊也成为秭归人参加奥运会比赛第一人。<ref>{{cite web | language = zh-cn | publisher = 荆楚网| title =探访《让子弹飞》拍摄地| url = http://www.cnhubei.com/sport/2012/hb/201207/t20120712_9953.shtml| date = 2012-07-12 | accessdate = 2012-07-30 }}</ref>7月29日，周俊在53公斤级抓举比赛中连续3次失败，提前退赛，成为[[中国国家举重队|中国女子举重队]]在奥运会历史上首位未获得任何成绩的运动员。<ref>{{cite web | language = zh-cn | publisher = 人民网| title =党报：周俊“白卷”反映中国举重深层次问题| url = http://www.people.com.cn/h/2012/0730/c25408-1-2165995354.html| date =2012-07-30| accessdate = }}</ref>
