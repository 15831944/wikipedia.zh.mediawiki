'''周鸿祎'''（{{bd|1970年|10月4日|||Z周}}），籍贯[[湖北省|湖北省]][[黄冈|黄冈]][[蕲春|蕲春]]<ref name="天天向上20130201期：IT大佬周鸿祎舌战天天兄弟（下）">{{cite web| url =https://www.mgtv.com/b/10082/3191703.html| title =天天向上20130201期：IT大佬周鸿祎舌战天天兄弟（下）| publisher =芒果台| accessdate =2013-02-01}}</ref> ，成长于河南郑州，毕业于[[西安交通大学|西安交通大学]]<ref>{{cite web | language=zh-cn | publisher=中国经济网 | title=周鸿祎：我打架就没怕过谁 | url=http://book.ce.cn/xw/rwzj/201307/30/t20130730_1180827.shtml | author= | date= | accessdate=2013-07-30 | archive-url=https://web.archive.org/web/20140719042601/http://book.ce.cn/xw/rwzj/201307/30/t20130730_1180827.shtml | archive-date=2014-07-19 | dead-url=yes }}</ref>。曾就职于[[方正集团|方正集团]]，2004年任[[雅虎中国|雅虎中国]]总裁，现任[[奇虎360|奇虎360]]公司董事长。

==生平及經歷==
*1970年10月4日出生于湖北黄冈。<ref name="天天向上20130201期：IT大佬周鸿祎舌战天天兄弟（下）" />
*1988年保送[[西安交通大学|西安交通大学]]教改班，计算机专业。<ref name=":0" />
*1992年，毕业于西安交通大学电信学院计算机系，因学习成绩优异，同年被保送西安交大读管理学院系统工程系研究生，并于1995年研究生毕业，获硕士学位<ref>{{Cite web|url = http://www.aliyun.com/zixun/aggregation/1478.html|title = 周鸿祎|accessdate = 2015-09-03|author = |date = |publisher = 阿里云资讯网|language = zh-cn}}</ref>。
*1995年，周鸿祎就职于方正集团<ref>{{Cite web|url = http://money.163.com/special/002540GR/zhouhy.html|title = 周鸿祎 奇虎360董事长|accessdate = 2015-09-03|author = |date = |publisher = 网易财经|language = zh-cn}}</ref>。
*1998年，辞去方正集团事业部总经理的职务<ref name=":0" />，创立因特国风软件有限公司<ref name=":0" />，推出对当时中国大陆網路影响较大的软件“[[雅虎助手|3721上網助手]]”<ref>{{cite web | language =zh-cn | publisher=腾讯网 | title=周鸿祎_科技人物库_腾讯科技频道 | url=http://datalib.tech.qq.com/people/89/brief.shtml | author= | date= | accessdate=2008-11-01 }}</ref>，并因此得到了“中国流氓软件之父”的恶名<ref>{{Cite web|url = http://it.sohu.com/20101022/n276230338.shtml|title = 周鸿祎：改“邪”归正的“流氓软件”之父|accessdate = 2015-09-03|author = |date = 2010年10月22日|publisher = 搜狐网|language = zh-cn}}</ref>。
*1999年，登上《IT经理世界》杂志封面。<ref name=":0" />
*2001年5月，决定3721开始收费。<ref name=":0" />
*2001年，开始与CNNIC对峙。<ref name=":0" />
*2003年9月，10月，3721两次被百度以不正当竞争为由告上法庭，周鸿祎代表3721出庭。<ref name=":0" />
*2003年11月，周鸿祎把3721以1.2亿美金卖给了[[雅虎|雅虎]]。<ref name=":0" />
*2004年1月，在[[雅虎中国|雅虎中国]]任职中国区总裁。又因合作问题与杨致远爆发冲突，周鸿祎并于2005年8月31日正式辞去[[雅虎中国|雅虎中国]]区总裁职务。<ref>{{cite news | url=http://it.sohu.com/s2005/zhouhongyi.shtml | title=周鸿祎辞去雅虎中国总裁职务 | publisher=搜狐IT | accessdate=2016-02-14 }}</ref>
*2006年3月，重新回到他自己以及众多风投公司共同投资的奇虎公司。<ref name=":0" />
*2006年8月，成立[[奇虎360|奇虎360]]并担任董事长，随后开发了[[360安全卫士|360安全卫士]]等安全软件，其中[[360杀毒|360杀毒]]开创了中国大陆免费杀软之先河<ref>{{Cite web|url = http://tech.sina.com.cn/s/2010-12-20/08315003872.shtml|title = 免费午餐 盘点2010年国内免费杀毒软件|accessdate = 2015-09-03|author = |date = 2010年12月20日|publisher = 新浪科技|language = zh-cn}}</ref><ref>{{cite web | language =zh-cn | publisher=奇虎网 | title=创始人_关于奇虎360 | url=http://www.360.cn/about/founder.html | author=360安全中心 | date= | accessdate=2012-08-23 }}</ref>。
*2006年8月，与当时雅虎执行总裁田健公开争执。<ref name=":0" />
*2008年7月17日，召开发布会并宣布正式推出360杀毒软件，并宣布永久免费。<ref name=":0" />
*2010年5月25日下午，周鸿祎在[[新浪微博|新浪微博]]和[[腾讯微博|腾讯微博]]密集发布微博揭露[[360安全卫士|360安全卫士]]和[[金山毒霸|金山毒霸]]的恩怨<ref>[http://tech.qq.com/a/20100526/000543.htm 金山安全CEO回应周鸿祎指责 发微博再谈恩怨]</ref><ref>[http://news.xinhuanet.com/it/2010-05/26/c_12142251.htm 周鸿祎发布40条微博:金山让我想起岳不群]</ref>。而在360和腾讯冲突全面升级前后，周鸿祎都在自己的微博上公开发表批评甚至是鄙视腾讯的言论。在360推出隐私保护器之后，其腾讯微博曾暂停更新，直到[[3Q大战|3Q大战]]之后更新。
*2011年3月30日，带领奇虎360在纽交所上市。<ref name=":0" />
*2012年，因360特供機和[[小米手機|小米手機]]之爭，周鴻禕和[[雷軍|雷軍]]被網友戲稱為周冷禪和雷不群<ref>[http://www.cnbeta.com/articles/195536.htm 雷军：周鸿祎做手机是欲将卡巴故事重演] {{webarchive|url=https://web.archive.org/web/20120808010603/http://www.cnbeta.com/articles/195536.htm |date=2012-08-08 }}</ref>。
*2012年8月，[[南京大学|南京大学]]一名学生通过某节目电话采访周鸿祎的视频中泄露的[[双音多频|双音多频]]（DTMF）拨号音成功破译了周鸿祎的手机号码并将过程发到网上，虽隐藏了号码中的几位数字但仍引致了大量网友拨打电话“验证”形成骚扰电话<ref>[http://tech.qq.com/a/20120901/000028.htm 大学生破译周鸿祎手机号 李开复放“橄榄枝”]</ref>。
*2014年12月，360投资4.095亿美元与[[酷派集團|酷派]]成立合资公司[[奇酷科技|奇酷科技]]，周鸿祎担任CEO<ref>{{Cite web|url = http://tech.ifeng.com/a/20150212/40981348_0.shtml#_zbs_baidu_bk|title = 周鸿祎将任酷派360合资公司CEO 郭德英任董事长|accessdate = 2015-09--03|author = |date = 2015年2月12日|publisher = 凤凰科技|language = zh-cn}}</ref>。
*2015年，榮獲“2014中国互联网年度人物”<ref>{{Cite web|url=http://politics.people.com.cn/n/2015/0213/c1001-26563026.html|title=2014中国互联网年度人物名单揭晓|accessdate=2019-01-11|author=|date=|work=politics.people.com.cn|publisher=}}</ref>。
*2018年，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>，同年榮獲“2017十大经济年度人物”<ref>{{Cite web|url=http://finance.sina.com.cn/china/gncj/2018-01-30/doc-ifyqyesy3993919.shtml|title=人民日报：2017十大经济年度人物揭晓|accessdate=2019-01-11|date=2018-01-30|last=人民日报|work=finance.sina.com.cn}}</ref>。
*2018年5月29日，周鸿祎針對[[区块链|区块链]]平台[https://eos.io/ EOS]的一系列高危安全漏洞，經直播宣布了 360 的企业级区块链安全解决方案，因而在区块链社区中取得了不少影響力。<ref>{{Cite web|url=http://www.360.cn/newslist/zxzx/360fxqklssjldkwqkzxnhbjy.html|title=360发现区块链史诗级漏洞 可完全控制虚拟货币交易|accessdate=2019-01-11|work=www.360.cn}}</ref>

== 个人生活 ==
周鸿祎已婚，1996年在北大方正任職計算機軟件工程師時相識相戀同事[[胡歡|胡歡]]，婚後育有2兒1女。爱好[[槍械|射击]]运动、阅读[[书籍|书籍]]和看[[电影|电影]]。<ref>[http://www.360kan.com/special/xiaoshandaai3.html  《小善大爱》公益真人秀]</ref>

== 诉讼 ==
*2013年7月22日，北京市二中院确认周鸿祎涉案7条微博言论构成对北京[[金山软件|金山软件]]有限公司的名誉侵权。判决要求周停止侵权，并删除其在微博中的相关文章，并在各微博首页发表致歉声明，向金山公司公开赔礼道歉、消除影响；另赔偿金山公司经济损失5万元。<ref>{{cite web| language=zh-cn| publisher=法制日报| title=“微博第一案”终审:周鸿祎败诉| url=http://epaper.legaldaily.com.cn/fzrb/content/20130730/Articel08002GN.htm| author=余瀛波| date=2013-07-30| accessdate=2013-11-04| deadurl=yes| archiveurl=https://web.archive.org/web/20140204000114/http://epaper.legaldaily.com.cn/fzrb/content/20130730/Articel08002GN.htm| archivedate=2014-02-04}}</ref>
*2013年10月18日，朝阳法院公开审理[[方舟子|方舟子]]诉周鸿祎侵害名誉权一案。因周在其微博上转发“网友爆料方舟子被百度重金收买和利用”等言论，被方以名誉侵权为由将周诉至法院，索赔精神损失20万元。<ref>{{cite web | language =zh-cn | publisher=腾讯科技 | title=方舟子诉周鸿祎名誉侵权案今日开庭 | url=http://tech.qq.com/a/20131018/014267.htm | author=marsrxdou | date=2013-10-18 | accessdate=2013-11-04 }}</ref>最终，法院判定北京奇虎公司与周鸿祎侵犯百度名誉权事实成立，连带赔偿百度公司经济损失五万元，驳回百度公司的其他诉讼请求。<ref>{{cite web | language =zh-cn | publisher=网易科技 | title="百度收买方舟子"案宣判 360被判侵权赔偿5万元 | url=http://tech.163.com/14/0325/09/9O61O0FE000915BF.html | author=顾晓波 | date=2014-03-25 | accessdate=2014-03-25 }}</ref>
