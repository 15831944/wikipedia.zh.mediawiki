== 公益活動 ==
*在2009年11月24日，周柏豪出席[[電影|電影]]《[[聽說|聽說]]》首映禮<ref>{{cite web |language=zh-cn |url=http://ent.sina.com.cn/m/p/2009-11-25/10242784296.shtml | title=《聽說》首映-周柏豪面帶微笑 |publisher=[[新浪網|新浪網]] |date=2009-11-25 |accessdate=2011-04-17}}</ref><ref>{{cite web |language=zh-hk |url=http://orientaldaily.on.cc/cnt/entertainment/20091126/00282_018.html | title=周柏豪爆遭鄭融狂追 |publisher=[[東方日報|東方日報]] |date=2009-11-26 |accessdate=2011-04-17}}</ref>，為慈善機構「[[龍耳|龍耳]]」宣傳「[[聾|聾]]健共融」的信息，讓社會加深對聾人的了解。
*2015年與[[林欣彤|林欣彤]]及[[梁文禮|梁文禮]]為香港[[世界宣明會|世界宣明會]][[飢餓三十人道救援行動#.E7.AC.AC32.E5.B1.86_.282015.E5.B9.B4.29|饑饉三十]]擔任「饑饉之星」，於3月3日至6日前往菲律賓探訪當地兒童。
