'''周志鑫'''（{{bd|1965年|8月||}}），[[安徽省|安徽省]][[太湖县|太湖县]]人，[[中华人民共和国|中华人民共和国]]空间遥感应用专家，[[中国人民解放军少将|中国人民解放军少将]]，[[中国科学院|中国科学院]]信息技术科学部院士<ref name=zhou>{{Cite web |url = http://www.casad.cas.cn/aca/371/xxjskxb-201512-t20151224_4502666.html |title = 周志鑫 |author =  |date =2015-12-24  |publisher = 中国科学院 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1982年毕业于[[安徽省太湖中学|安徽省太湖中学]]。1982年至1986年在[[中国人民解放军电子工程学院|中国人民解放军电子工程学院]]学习，获学士学位。1986年至1989年，在[[哈尔滨工业大学|哈尔滨工业大学]]航天学院学习，获硕士学位。1989年至1993年，在[[中国人民解放军空军第一飞行学院|中国人民解放军空军第一飞行学院]]仿真中心任工程师。1993年至1997年，在哈尔滨工业大学航天学院学习，获博士学位。1997年至1999年，在[[中国科学院|中国科学院]]自动化所模式识别国家重点实验室当[[博士后|博士后]]。1999年起，任北京市遥感信息研究所研究员，国家重点项目技术负责人<ref name=thx/>。2015年12月7日当选[[中国科学院|中国科学院]]信息技术科学部院士<ref name=thx>{{Cite web |url =  http://www.thx.gov.cn/content/detail/5672999aaf88bc98134dae35.html  |title = 周志鑫当选为中国科学院院士 |author =  |date = 2015-12-17 |publisher =太湖县人民政府  |language =  |accessdate =  |quote =  }}</ref>。后任北京市遥感信息研究所所长<ref name=zhou/>。曾在[[中国人民解放军总参谋部情报部|总参二部]]航侦局任职。[[中国人民解放军战略支援部队|中国人民解放军战略支援部队]]成立后，周志鑫出任中国人民解放军战略支援部队某局局长。2017年，调任新组建的[[中国人民解放军战略支援部队航天工程大学|中国人民解放军战略支援部队航天工程大学]]校长<ref name=liangxiang>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1849027 |title =航天工程大学军政主官亮相：周志鑫、纪多分任校长、政委  |author =  |date = 2017-11-03 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

周志鑫长期从事空间遥感理论与应用技术研究，在海上移动目标高精度探测定位技术、典型地物特征检测识别等方面取得多项成果。系统研究并发展出天基海洋探测定位以及海上移动目标自动检测方法，提出并且建立了空间遥感数据存储、处理、应用架构，解决了卫星海量数据高效高精度自动处理及应用的技术问题，研究建立了典型地物目标特征库。曾获得国家科技进步特等奖1项、一等奖1项，[[中国科协|中国科协]]“求是杰出青年实用工程奖”，[[何梁何利基金|何梁何利基金]]奖“科学与技术创新奖”<ref name=zhou/>。
