'''周小川'''（{{bd|1948年|1月29日||catIdx=Z周}}），[[江苏|江苏]][[宜兴市|宜兴市]]人；[[北京化工大学|原北京化工学院]]毕业，[[清华大学|清华大学]][[工学|工学]]博士。1986年加入[[中国共产党|中国共产党]]，是中共第十六、十七届中央委员。现任货币政策委员会主席和博鱉亞洲論壇副理事長。曾任[[中国人民政治协商会议全国委员会副主席|全国政协副主席]]、[[中国人民银行|中国人民银行]]行长、[[中国建设银行|中国建设银行]]行长、[[中国证券监督管理委员会|中国证券监督管理委员会]]主席。 

在任央行行长期间，媒体曾将他同前[[中华人民共和国国家发展和改革委员会|国家发改委]]主任[[马凯|马凯]]、前[[中华人民共和国财政部|财政部]]部长[[金人庆|金人庆]]以及前[[中华人民共和国商务部|商务部]]部长[[吕福源|吕福源]]并列为[[温家宝|温家宝]]“[[温家宝政府|内阁]]”的“四大财经官员”。2009年3月间发表系列论文，提出去主权化“国际货币体系改革的理想目标”，即“创造一种与主权国家脱钩、并能保持币值长期稳定的国际储备货币”<ref>{{cite web | last = 周小川 | title = 《关于改革国际货币体系的思考》 |  publisher = 转载自中国人民银行网站 www.pbc.gov.cn/detail.asp?col=4200&ID=279 原文被删除 | date = 2009年3月23日 | url = http://www.hongfan.org.cn/file/upload/2009/08/25/1251279219.pdf}}</ref>；阐述金融危机中，中国宏观调控政策的作用<ref>[http://news.xinhuanet.com/fortune/2009-03/26/content_11079284.htm 周小川：关于改变宏观微观顺周期性的进一步探讨]</ref>。2010年末，面对[[热钱|热钱]]流入加速的形势时曾提出《池子理论》，引起了广泛的关注。2011年9月被《[[欧洲货币|欧洲货币]]》杂志评选为“年度全球最佳央行行长”<ref>[http://finance.ifeng.com/news/hqcj/20110927/4689537.shtml 周小川当选最佳行长]</ref>。

== 早年经历 ==
周小川出身中共[[高级干部|高级干部]]家庭。父亲[[周建南|周建南]]曾于1980年代出任[[中华人民共和国机械工业部|机械工业部]]部长<ref>[http://news.163.com/13/0316/11/8Q37SC1T0001124J.html 周小川行长身世曝光 父亲曾任一机部副部长]</ref>，母亲杨维哲曾是[[中华人民共和国化学工业部|化工部]]离休老干部。杨维哲曾任国家科委委员。她筹备成立了中国第一家高科技风险投资公司，并担任第一任董事长<ref>[http://www.cnfatong.com.cn/main/home/ns_detail.php?id=6928&nowmenuid=47 周建南]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1966年，周小川从北京八中毕业，同年，文革开始了。据两名知情人透露，周小川曾是一个红卫兵组织的领袖。<ref>{{cite web|title=周小川：中国的改革派央行行长 - 头版 - FT中文网|url=http://www.ftchinese.com/story/001063537?full=y}}</ref>中学毕业后，周响应[[知识青年|知识青年]]“[[上山下乡|上山下乡]]”的号召，赴[[黑龙江生产建设兵团|黑龙江生产建设兵团]][[上山下鄉|插队]]劳动；1972年，受推荐进入[[北京化工学院|北京化工学院]]学习，成为“[[工农兵学员|工农兵学员]]”。毕业后，被分配到化工部北京自动化技术研究所系统研究室工作，曾参与“川汉输气工程”、“30万吨乙烯成套设备自主设计制造工程”等项目。

1978年，在中国大陆恢复[[研究生|研究生]]招生后，周小川考入[[机械工业部|机械工业部]]部属研究机构——[[机械研究院|机械研究院]][[系统工程|系统工程]]应用工业的研究生，师从[[严筱均|严筱均]]教授。1982年8月，进入[[清华大学|清华大学]]成为系统工程系首批招收的博士生。在此期间，周一直从事经济体制改革的政策分析及许多经济课题的研究工作。1986年9月，参加[[莫干山会议|莫干山会议]]，后出任[[国务院体改方案领导小组|国务院体改方案领导小组]]成员，并兼任[[中国经济体制改革研究所|中国经济体制改革研究所]]副所长，为国务院高层担任智囊；后又兼任[[国家经济体制改革委员会|国家经济体制改革委员会]]委员。

==仕途之路==
1986年12月，周小川被任命为[[中華人民共和國对外经济贸易部|对外经济贸易部]]部长助理，开始从政之路，时年38岁。此后，历任[[中国银行|中国银行]]常务董事、副行长；[[国家外汇管理局|国家外汇管理局]]局长；[[中国人民银行|中国人民银行]]副行长、[[中国人民银行货币政策委员会|货币政策委员会]]第一届委员。1998年2月出任[[中国建设银行|中国建设银行]]行长；2000年2月，转任[[中国证监会|中国证监会]]主席；2002年12月，根据[[朱镕基|朱镕基]]总理提名，接替[[戴相龙|戴相龙]]出任[[中国人民银行|中国人民银行]]行长。其后先后在[[温家宝政府|温家宝政府]]和[[李克强政府|李克强政府]]中连任，历经三位[[国务院总理|国务院总理]]，是任期最长的行长。2013年3月当选[[全国政协副主席|全国政协副主席]]，成为[[党和国家领导人|党和国家领导人]]。

==思考言论==
关于货币体系演变的思考和言论。
*2009年3月23日，国际金融危机爆发不久，在央行官方网站发表《关于改革国际货币体系的思考》一文，提出了与去主权化、总量恒定的[[比特币|比特币]]体系相似的概念，即“创造一种与主权国家脱钩、并能保持币值长期稳定的国际储备货币”。
*2014年4月11日，比特币“与集邮者收集的邮票一样”，“是一种能够交易的资产而非支付货币”。<ref>{{cite web| last =王自宸、郭信峰| title =周小川：比特币非支付货币 不存在取缔一说| publisher =新华网海南频道| date =2014年4月11日| url =http://www.hq.xinhuanet.com/news/2014-04/11/c_1110209773.htm| deadurl =yes| archiveurl =https://web.archive.org/web/20140716085052/http://www.hq.xinhuanet.com/news/2014-04/11/c_1110209773.htm| archivedate =2014年7月16日}}</ref>
*2016年2月13日，“比特币是不要中央银行的”数字货币。<ref>{{cite web | last = 王烁 张继伟 霍侃 | title = 【专访周小川之五】数字人民币还有多远？| publisher = 财新周刊| date = 2016年2月13日| url = http://economy.caixin.com/2016-02-13/100908683.html }}</ref>
*在2017年3月10日，“数字货币、区块链等技术会产生不容易预测到的影响。”<ref>{{cite web | last = 李志杰 李一丁 李付雷| title = 数字货币碰上区块链，可带来可靠的货币体系
