'''周志怀'''（{{bd|1956年|6月||}}），[[安徽|安徽]]人，[[中华人民共和国|中华人民共和国]][[台湾问题|台湾问题]]专家，立場相比強硬派而言較中間派。<ref>[http://news.163.com/17/0614/14/CMT7KQHQ000187V5.html 網易-論戰略定力]</ref>

==生平==
1982年，毕业於[[吉林大學|吉林大學]]中文系。后任[[中国社会科学院台湾研究所|中国社会科学院台湾研究所]]副所长、[[全国台湾研究会|全国台湾研究会]]执行副会长兼秘书长<ref>{{Cite web |url = http://world.huanqiu.com/hot/2016-12/9810594.html |title =周志怀 ：中国社科院台研所所长  |author =  |date = 2016-12-14 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref>。

2013年在[[余克禮|余克禮]]退休後，接任中国社会科学院台湾研究所所長<ref>{{cite news|title=接掌台研所 周志懷：貼近民意|url=http://news.singtao.ca/toronto/2013-12-10/taiwan1386664669d4829640.html|accessdate=2016-08-23|agency=星島日報|date=2013-12-10}}</ref><ref>{{cite news|author1=邱國強|title=陸涉台智庫掌門周志懷 屆退休年齡|url=https://tw.news.yahoo.com/%E9%99%B8%E6%B6%89%E5%8F%B0%E6%99%BA%E5%BA%AB%E6%8E%8C%E9%96%80%E5%91%A8%E5%BF%97%E6%87%B7-%E5%B1%86%E9%80%80%E4%BC%91%E5%B9%B4%E9%BD%A1-114313078.html|accessdate=2016-08-23|agency=中央社|date=2016-08-22}}</ref>。2017年2月卸任所长职务，由[[中国现代国际关系研究院|中国现代国际关系研究院]]副院长[[杨明杰|杨明杰]]接任<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1622218|title =杨明杰出任中国社科院台湾研究所所长，接替周志怀   |author =  |date = 2017-02-19  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

==立場==
周志懷認為隨著中國大陸整體實力增長，[[鄧小平|鄧小平]]當年提出的一國兩制提議已經沒有可能，因為其實那是一種寬鬆條件。<ref name=ct1>[https://www.chinatimes.com/newspapers/20181017000084-260301 議題難再 陸學者示警 台失談判籌碼]</ref>

當年鄧版一國兩制的一種提議是「祖國統一後台灣特別行政區可有一定獨立性，可實行同大陸不同的制度。司法獨立，終審權不須到北京。台灣可有自己軍隊，只是不能構成對大陸的威脅。大陸不派人駐台，不僅軍隊不去行政人員也不去。台灣的黨政軍系統，都由台灣自己管。中央政府還要給台灣留出名額。」

但這是建立在兩個前提條件上，第一是大陸軍經實力發展離美國遙遠，但25年後鄧小平模式取得的成就與速度連大陸人民自己都感到驚訝，再過10年到2020年代末整體實力已經不再需要和台灣或任何國家[[談判|談判]]任何條件，也沒有國家敢在這一議題上挑戰中華人民共和國的利益與主張。<ref name=ct1 />第二前提是台灣主動靠向統一包含當年還存在[[國統綱領|國統綱領]]，但台獨思想造成台灣社會取得大共識進入和平統一談判的可能性消失，許多台籍檯面上人物也嚴重低估了大陸政府和人民統一的意志與決心。當這兩條件都不存在時這一種「寬鬆條件」版本的統一自然也消失，最終台灣只能自己面對嚴酷版本的走向。
