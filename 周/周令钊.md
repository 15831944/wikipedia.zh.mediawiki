'''周令钊'''（{{bd|1919年||}}）[[湖南省|湖南省]][[平江县|平江县]]人，[[中华人民共和国|中华人民共和国]]画家、设计师<ref name=yysh>{{Cite web |url = http://www.yueyang.gov.cn/daj/6658/6666/content_699435.html |title =周令钊  |author =  |date =2015-04-15  |publisher = 岳阳市档案局 |language =  |accessdate =  |quote =  }}</ref><ref name=ym>{{Cite web |url = http://www.cafa.edu.cn/aboutcafa/?c=303&N=1975 |title = 周令钊 |author =  |date =  |publisher =中央美术学院  |language =  |accessdate = 2018-02-13 |quote =  }}</ref>。

==生平==
周令钊生于平江县，自幼受当国画教员的母亲影响，喜爱美术。1929年，因军阀混战，周令钊随家人迁居[[长沙|长沙]]。1931年[[九一八事变|九一八事变]]后，正在长沙中学附小就读的周令钊随老师用美术字书写抗日标语，画宣传画。高小毕业时，跳过初中，考入华中美专，后来进入[[武昌藝術專科學校|武昌艺专]]学习。1935年毕业于武昌艺专师范科。毕业后到[[上海|上海]]学习印刷制版，同时半工半读，坚持学画<ref name=yysh/><ref name=ym/>。

1937年[[八一三事变|八一三事变]]，[[日军|日军]]进攻上海，周令钊返回长沙，参加了湖南省抗敌画会，每周赶出一个画展，到街头和乡镇巡回展出。1937年抵达[[广州|广州]]，参加“八·一三”歌咏队，在“八·一三”火炬大游行中画了两幅大画《挺起胸膛，按住疮伤，前进》、《起来，不愿做奴隶的人们》。1938年抵达[[武汉|武汉]]，进入[[国民政府军事委员会|国民政府军事委员会]]政治部宣传厅艺术处音乐美术科，参加了在武汉举办的“抗日宣传周”，参与创作武汉[[黄鹤楼|黄鹤楼]]《全民抗日》巨幅壁画。随后自武汉撤退到长沙，又被分配至[[国民政府军事委员会委员长桂林行营|国民政府军事委员会委员长桂林行营]]政治部宣传组。先后在该行营宣传组绘制抗日宣传画，在艺术馆画《国家至上》演出海报，在国艺社为《明末遗恨》、《[[梁红玉|梁红玉]]》设计舞台布景。“新中国剧社”成立后，周令钊参加该剧社，画话剧《大雷雨》海报<ref name=yysh/>。

当时“前方吃紧，后方紧吃”，大后方供应困难，[[田汉|田汉]]劝青年去抗战前方服务，1942年周令钊乃到[[柳州|柳州]]参加了抗敌演剧五队。其间，他参与演出反[[法西斯|法西斯]]话剧《人兽之间》，设计了舞台布景，画海报，参加演唱《[[黄河大合唱|黄河大合唱]]》等[[中共|中共]]抗日根据地歌曲。后随五队经[[昆明|昆明]]、[[大理|大理]]、[[保山|保山]]、[[芒市|芒市]]过宛町桥到[[缅甸|缅甸]]，为[[中国远征军|中国远征军]]慰问演出。途中他演过《风波亭》、《丁大夫》等。他设计布景，饰演配角，并沿途写生<ref name=yysh/><ref name=ym/>。

1945年[[日本投降|日本投降]]，周令钊离开缅甸经昆明抵达广州，在电影院画广告，又到[[香港|香港]]画电影布景。1946年经长沙到武汉，在演剧六队为《夜店》以及[[田汉|田汉]]的《丽人行》设计布景。1947年到上海看望田汉，田汉让他先到上海育才学校美术组任教。任教期间，他和同学到上海农村写生，创作农村题材的绘画，并用组画、连环画画《[[武训|武训]]传》，还在上海、[[杭州|杭州]]参加[[反饥饿、反内战、反迫害运动|反饥饿反内战]]宣传<ref name=yysh/>。

1948年，周令钊经人介绍给[[徐悲鸿|徐悲鸿]]，被徐悲鸿聘至[[国立北平艺术专科学校|北平艺专]]实用美术系任教。1948年底，[[中国人民解放军|中国人民解放军]]包围[[北平|北平]]，在北平艺专的中共地下党组织下，他参与创作了迎接解放的木刻宣传画，同北平艺专师生迎接解放军。1949年[[北平和平解放|北平和平解放]]后，同年11月国立北平艺专与其他学校合并，1950年1月定名[[中央美术学院|中央美术学院]]，徐悲鸿任院长。周令钊历任中央美术学院副教授、教授。1949年7月1日，徐悲鸿、[[廖静文|廖静文]]夫妇为周令钊和陈若菊在府上主持婚礼。国共[[北平和谈|北平和谈]]在[[六国饭店_(北京)|六国饭店]]举行，由[[周恩来|周恩来]]主持，周令钊奉命带学生设计布置会场，周令钊画了戴[[中国工农红军|红军]]帽栩栩如生的[[毛泽东|毛泽东]]像<ref name=yysh/><ref name=ym/>。

1949年10月1日前夕，周令钊奉命为[[中华人民共和国开国大典|中华人民共和国开国大典]]画悬挂在[[天安门|天安门]]的毛泽东像。[[中华人民共和国|中华人民共和国]]成立初期，他先后参与设计了[[中国新民主主义青年团|中国新民主主义青年团]]团旗（1950年）、[[中国少年先锋队|中国少年儿童队]]队旗（1950年[[刘易晏|刘易晏]]设计，周令钊美化），并参加了[[中华人民共和国国徽|中华人民共和国国徽]]（1950年）的设计<ref name=yysh/>。

1950年起，每年“[[五一国际劳动节|五一]]”、“[[中华人民共和国国庆节|十一]]”周令钊担任天安门前游行队伍中文艺队伍的美术设计。他参与设计并执笔绘制了“[[八一勋章_(1955年)|八一勋章]]”、“[[独立自由勋章|独立自由勋章]]”和“[[解放勋章|解放勋章]]”。1950年起，担任第二、三、四套[[人民币|人民币]]的美术设计<ref name=yysh/><ref>{{Cite web |url =http://www.chinanews.com/cul/2013/09-30/5341827.shtml  |title = "国家形象"设计者周令钊：参与设计三套人民币 |author =  |date = 2013-09-30 |publisher =中国新闻网  |language =  |accessdate =  |quote =  }}</ref>。

自中华人民共和国初期到[[文革|文革]]前，周令钊任中国邮票局艺术设计顾问，并负责具体的美术指导工作。文革后又任全国邮票设计评审委员。他参加了《[[中国共产党第八次全国代表大会|中国共产党第八次全国代表大会]]》（1956年）等邮票的设计，自己设计了《胜利超额完成[[第一个五年计划_(中国)|第一个五年计划]]》纪念邮票（1959年）、《[[中国文联|中国文学艺术工作者第三次代表大会]]》邮票（1960年）、《[[新兴力量运动会|第一届新兴力量运动会]]》邮票（1963年）、《[[狗年|壬戌年]]》[[生肖|生肖]]邮票（1982年）、《[[四大名楼|中国历代名楼]]》邮票（1987年）等众多[[中国人民邮政|中国人民邮政]]邮票。他设计的《中华人民共和国成立十周年》（第四组）（1959年）获最佳邮票设计奖。1997年设计了“[[炎帝陵|炎帝陵]]”电话卡<ref name=yysh/>。

1951年，周令钊为[[中国国家博物馆|中国革命历史博物馆]]创作了油画《[[五四运动|五四运动]]》。1959年参加[[民族文化宫|民族文化宫]]建筑的美术设计。1959年为[[人民大会堂|人民大会堂]]湖南厅设计了[[韶山冲毛泽东同志故居|毛泽东故居]]《[[韶山|韶山]]》[[湘绣|湘绣]][[屏风|屏风]]画稿。1963年为[[中华人民共和国主席|国家主席]][[刘少奇|刘少奇]]访问[[印度尼西亚|印度尼西亚]]赠给[[苏加诺|苏加诺]]总统的国礼“友谊彩虹”[[苏绣|苏绣]]绘制画稿。他为历届[[中华人民共和国全国运动会|中华人民共和国全国运动会]]的开幕式、闭幕式设计了大型[[团体操|团体操]]背景画，并以第三届全国运动会的团体操背景画设计获国家颁发金质奖章。1980年代，为中华人民共和国最早的[[主题公园|主题公园]]——[[深圳华侨城控股股份有限公司|深圳华侨城]]的“[[锦绣中华|锦绣中华]]”和“民俗村”做总体艺术设计。另外，他还设计了“储蓄会徽”、中国少数民族文化艺术基金会会徽、[[中日友好医院|中日友好医院]]院徽、[[中央财经大学|中央财经大学]]校徽等。他创作了大量书刊封面、插图、招贴，例如《[[李季_(诗人)|王贵与李香香]]》和《[[刘三姐|刘三姐]]》等的插图，[[程砚秋|程砚秋]]《荒山泪》、[[梅兰芳|梅兰芳]]《洛神赋》、[[世界乒乓球锦标赛|第26届世界乒乓球锦标赛]]等招贴画<ref name=yysh/><ref>{{Cite web |url = http://news.ifeng.com/gundong/detail_2013_09/30/30002487_0.shtml |title = “国家形象”设计者周令钊 |author =  |date =2013-09-30  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

1959年，周令钊为中国革命历史博物馆中央大厅创作了[[金丝楠木|金丝楠木]]立粉勾金重彩壁画《世界各国人民大团结》（与[[黄永玉|黄永玉]]的《中国各族人民大团结》壁画分别位于大厅两侧墙上，大厅正面墙上是[[张充仁|张充仁]]创作的[[马克思|马]][[恩格斯|恩]][[列宁|列]][[斯大林|斯]]浮雕像<ref>黄永玉，《中国各族人民大团结》，中国国家博物馆馆刊2011年05期</ref>）。另外，周令钊为新建的黄鹤楼设计制作了大型彩色陶板壁画《白云黄鹤》，为[[人民日报|人民日报社]]办公楼创作了[[丙烯|丙烯]]壁画《漓江迎宾》，为渤海石油设计院设计了陶瓷壁画，为深圳华侨村文化中心的防火幕设计了金属喷绘画，为[[浙江|浙江]][[南浔|南浔]]绘制了《南浔水乡》长卷。2001年，周令钊为[[北京饭店|北京饭店]]创作了刻漆壁画《古都集翠》，为家乡湖南创作7幅画，为平江县创作壁画《深山邮情》、《春洒黄金堰》、《连云茶香》，为[[常德|常德]]诗墙设计《[[沅江|沅江]]水暖》、《[[澧水|澧水]]花繁》两幅石刻长卷。2002年，周令钊为[[岳阳市|岳阳市]]南湖宾馆绘制丙烯壁画《春风又绿盘石洲》、《天下忧乐情满楼》。2003年，为[[人民大会堂|人民大会堂]]设计[[大理石|大理石]]浮雕《松竹梅》。2005年创作《[[丽江|丽江]]古镇》长卷<ref name=yysh/>。2013年，94岁周令钊为[[北京地铁6号线|北京地铁6号线]][[朝阳门站|朝阳门站]]陶瓷浮雕壁画《京东粮道》学用釉上彩开脸<ref>{{Cite web |url = http://www.sohu.com/a/212693835_99906798 |title =周令钊：为中国造型 为民族设计  |author =  |date =2017-12-25   |publisher =搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.bjghw.gov.cn/web/dtggys/dtggys_detail6/detail6-22.html  |title = 六号线公共艺术设计方案 |author =  |date = 2012-08-30 |publisher = 北京市规划和国土资源管理委员会 |language =  |accessdate =  |quote =  }}</ref>。2018年[[中国邮政|中国邮政]]《戊戌年》邮票由首轮狗票设计者周令钊设计<ref>{{Cite web |url =  http://www.sohu.com/a/201737301_748615|title = 2018-1《戊戌年》狗票来了…… |author =  |date =2017-11-01  |publisher =  搜狐|language =  |accessdate =  |quote =  }}</ref>。
