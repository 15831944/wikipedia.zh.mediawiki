'''周星'''{{BD|1957年|8月}}，[[中華人民共和國|中國]][[陝西|陝西]][[丹凤县|丹鳳]]人，[[文化人類學|文化人類學]]及[[民俗學|民俗學]][[博士|博士]]，[[中國共產黨|中國共產黨]]黨員，現為[[日本|日本]][[愛知大學|愛知大學]]國際交流學部[[教授|教授]]、[[中國民俗學會|中國民俗學會]]顧問<ref>{{cite news |author1=北京科技大學文法学院 |title=百年实践下熠熠生辉 开放包容中源远流长——日本爱知大学国际中国学研究中心博士课程指导教授周星做客文法学院精品社科讲座 |url=http://news.ustb.edu.cn/xueyuandongtai/wenfa/2018-03-28/68003.html |agency=北京科技大学党委宣传部、新闻中心 |date=2018-03-28}}</ref><ref>{{cite web |title=周星教授发表演讲：高度的日常及其危机——日本的垃圾分类及回收体系 |url=http://igea.muc.edu.cn/info/1018/1064.htm |website=中央民族大学世界民族学人类学研究中心 |date=2014-07-27}}</ref>。

==經歷==
周星於1981年從[[西北大學_(中國)|西北大學]]歷史系考古專業本科畢業，1984年從[[中國社會科學院|中國社會科學院]]歷史研究所取得[[碩士|碩士]]學位，1989年於同校民族研究所[[博士|博士]]畢業。他在1989至2000年間先後在[[北京大學|北京大學]]社會學人類學研究所從事博士後研究，任職副教授、教授、副所長及博士生導師<ref>{{cite web |title=周星個人簡介 |url=http://www.cuaes.org/scholardetail-453.htm |website=中国人类学民族学研究会}}</ref>。

周星從2000年至今擔任日本[[愛知大學|愛知大學]]國際交流學部、國際中國學研究中心教授、博士課程指導教授。同時兼任中國民俗學會副理事長、中國藝術人類學學會副會長、[[中山大學|中山大學]]中國非物質文化遺產研究中心學術委員以及[[沖繩國際大學|沖繩國際大學]]南島文化研究所研究員等<ref>{{cite web |title=民俗学者周星 |url=http://www.chinesefolklore.org.cn/expert/index.php?ExpertID=104 |website=中国民俗学会}}</ref>。
