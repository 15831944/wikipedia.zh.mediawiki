'''周厉王'''（{{bd|前890年||前828年|catIdx=J}}），[[姓|姓]]'''[[姬姓|姬]]'''，名'''胡'''，在[[西周|西周]]青铜器铭文中作“{{僻字|㝬|⿰害夫，左害右夫}}”<ref>[[王辉|王辉]]：《商周金文》，[[文物出版社|文物出版社]]2006年，第206页，“{{僻字|㝬|⿰害夫，左害右夫}}簋”。</ref>。为[[周夷王|周夷王]]之子。《[[史記|史記]]》作'''厉王'''，西周[[青铜器|青铜器]]铭文作'''剌王'''<ref>[[江林昌|江林昌]]《由新出遂公盨、来氏铜器论夏商周世系及虞代问题》。</ref>，在文献中又作'''汾王'''。<ref>據《中華通史》第一冊，[[章嶔|章嶔]]著</ref>[[国人暴动|国人暴动]]之后，他居住在今[[山西省|山西省]][[汾河|汾河]]流域，因此传世典籍和[[青铜器|青铜器]]铭文又称他为'''汾王'''<ref>《[[诗经|诗经]]·大雅·荡之什·韩奕》“韩侯取妻，汾王之甥。”[[郑玄|郑玄]]笺:“厉王流于彘，彘在汾水之上，故时人因以号之。”又[[王国维|王国维]]《今本竹书纪年疏证》：“厉王名胡”。 王国维注：“居彘，有汾水焉，故又曰汾王”。</ref> 。厉王为[[西周|西周]]第十位君王，在位約49年（[[夏商周断代工程|夏商周断代工程]]，为約前877年－前841年，在位37年），连同共和的十四年。《[[太平御览|太平御览]]》八十四引《史記》載：“周孝王七年，厉王生。”

== 在位及逃亡 ==
[[File:克鼎08485.jpg|thumb]]厲王在位期间，重用賢臣[[荣夷公|荣夷公]]，不听奸佞[[周定公|周定公]]、[[召穆公|召穆公]]等人為保一己之私的計策，实行有利於富國強兵的“专利”政策，雖說因為之過甚而有奴役百姓之嫌，不让他们有言论自由，“防民之口，甚於防川”，<ref>《[[國語|國語]]•周語上》</ref>以至于行人来往，只能透過眼神来示意。但是使[[周朝|周朝]]国势更加強盛，朝政更加清明。但因人民不理解王的作為，又被那些奸佞貴族迷惑，于是在842年聚众[[國人暴動|國人暴動]]，冲进王宫，试图杀掉厉王，史称“国人暴动”。厉王只好逃出[[镐京|镐京]]，越过[[黄河|黄河]]，逃到[[汾水流域|汾水流域]]的[[彘|彘]]（今[[山西|山西]][[霍州市|霍州市]]），由[[共伯和|共伯和]]執政（一說是周定公及召公代為執政），史稱「[[共和行政|共和行政]]」。

周共和14年（前828年），厲王死，周定公及召公立厲王之子[[周宣王|靜]]，是為宣王。

== 在位年份的爭議 ==
按照《[[史記|史記]]》當中數篇世家的內容，厲王出逃之年與共和元年（前841年）相差了一年，[[夏商周断代工程|夏商周断代工程]]則認為厲王37年與共和元年是同一年。據[[白川靜|白川靜]]的西周斷代銘器分類結果，厲王在位時間有13年以上。

===《宗周鐘》===
《宗周鐘》又名「{{僻字|㝬|⿰害夫，左害右夫}}鐘」或「胡鐘」，記載厲王親自征服南方諸小國的故事，乃厲王存世最重要的西周天子作器，高65.6公分，重34.9公斤，計有[[銘文|銘文]]123字，是[[商朝|商]]、[[周朝|周]]兩朝[[甬鐘|甬鐘]]單件銘文之最，現藏於[[臺北|臺北]][[國立故宮博物院|國立故宮博物院]]。<ref>[http://www.npm.gov.tw/zh-TW/Article.aspx?sNo=04001042 國立故宮博物院]</ref>
