'''周家蔚'''（{{lang|en|'''Janet Chow'''}}，{{bd|1983年|8月25日|catIdx=Z周}}），[[香港|香港]][[商人|商人]]，前[[香港|香港]][[無線電視|無線電視]]演員。2006年度[[香港小姐競選|香港小姐競選]]亞軍及最上鏡小姐。現已婚，丈夫是同為電視演員[[洪天明|洪天明]]。

==簡介==
周家蔚在[[香港|香港]]出生，7歲時移民到[[加拿大|加拿大]]<ref>{{cite web|title=名人親子﹕專訪周家蔚：百厭星變乖媽媽|url=http://hk.news.yahoo.com/%E5%90%8D%E4%BA%BA%E8%A6%AA%E5%AD%90-%E5%B0%88%E8%A8%AA%E5%91%A8%E5%AE%B6%E8%94%9A-%E7%99%BE%E5%8E%AD%E6%98%9F%E8%AE%8A%E4%B9%96%E5%AA%BD%E5%AA%BD-213550638.html|deadurl=yes|archiveurl=https://archive.is/20130428163003/http://hk.news.yahoo.com/%E5%90%8D%E4%BA%BA%E8%A6%AA%E5%AD%90-%E5%B0%88%E8%A8%AA%E5%91%A8%E5%AE%B6%E8%94%9A-%E7%99%BE%E5%8E%AD%E6%98%9F%E8%AE%8A%E4%B9%96%E5%AA%BD%E5%AA%BD-213550638.html|archivedate=2013-04-28}}</ref>，畢業於[[马克维尔高级中学|Markville Secondary School]]及[[多伦多大学|多伦多大学]]经济系。2006年，參加[[香港小姐競選|香港小姐競選]]獲得亞軍及最上鏡小姐獎，其後，代替香港小姐冠軍[[陳茵媺|陳茵媺]]出戰[[波蘭|波蘭]]參選《[[世界小姐|世界小姐]]選舉》。

周家蔚於婚後已淡出娛樂圈，2013至2018年期間更一直停職留薪。2014年，她在[[銅鑼灣|銅鑼灣]][[皇室堡|皇室堡]]開設童裝及兒童用品店，半年後約滿賣盤。後來為了發展內地化妝品微商生意，與藝人[[曹永廉|曹永廉]]、[[唐文龍|唐文龍]]及夫婿之朋友劉永健合資。2018年10月與無綫電視約滿，全面投身商界。

== 个人生活 ==

2012年3月21日，周家蔚和演员[[洪天明|洪天明]]举办了婚礼，婚礼仪式按照中华传统习俗“接新娘、玩新郎、踢轿门、背新娘、敬茶”一步步完成，出席婚礼的人士包括了香港演艺界的名人。<ref>{{cite web |url=http://ent.ifeng.com/photo/hot/detail_2012_03/22/13369523_0.shtml |title=洪金宝儿子与港姐大婚 香港群星道贺 |accessdate= 2012年|work= |publisher=凤凰网 |language =zh-cn}}</ref><ref>{{cite web |url=http://yule.sohu.com/20120322/n338511711.shtml |title=洪金宝娶媳妇群星道贺 洪天明与周家蔚玩神秘 |accessdate= 2012年|work= |publisher= 搜狐网|language =zh-cn}}</ref><ref>{{cite web |url=http://ent.ifeng.com/idolnews/hk/detail_2012_03/22/13367230_0.shtml |title=洪天明周家蔚昨日结婚 获好友赠送专属唯一App |accessdate= 2012年|work= |publisher=凤凰网 |language =zh-cn}}</ref>。現育有兩子，分別在2013年及2015年出生。
