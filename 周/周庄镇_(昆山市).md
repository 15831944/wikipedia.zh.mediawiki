'''周庄'''，是{{PRC admin/fullname|32/05/83/101/000|3}}下辖的一个[[乡镇|乡镇]]级行政单位， 周围湖荡环列，镇内河港纵横，是典型的江南水乡，被誉为“中国第一水乡”。目前已经成为著名风景旅游区。

== 地理 ==
周庄位于[[中华人民共和国|中华人民共和国]][[江苏省|江苏省]][[昆山市|昆山市]]西南隅，东经120°53′，北纬31°14′处。东南与[[上海市|上海市]][[青浦县|青浦县]]的商塌为邻，西南与[[吴江市|吴江市]]的莘塔接壤，东北与[[锦溪镇|锦溪镇]]相衔，西北与[[吴县|吴县]]的淞南相望。为昆山、吴江、上海三市交界处。

周庄地处府县边隅，四周群湖环抱，河港纵横。“咫尺往来，皆须舟楫”{{Fact|time=2012-08-13T12:40:29+00:00}}。镇区北依白蚬湖、急水港，为[[吴淞江|吴淞江]]漕粮北运要道；南衔张矢鱼湖（今南湖），古为文人志士隐逸之地。元代居户聚集于[[市河|市河]]东岸，今北市街一带，镇西仍荒僻。明代中期，镇址向镇西后港及港西方面延伸，而后太平桥、世德桥、福洪桥相继建成，道路沟通，遂由城隍埭继向普庆桥（今中市街）一带发展。至清初，规模日盛，整个市镇以富安桥为中心，有南北市街和中市街两条[[丁字型|丁字型]]大街，形成闹市。清至民初，因地处三县之中，商业繁盛<ref>《周庄镇志》.1991 第一篇（建置区划）第三章（村镇概况）中载“八条长街商贾列肆，货物充盈，井字形的水道上，舟船衔接，来往穿梭。成为苏州[[葑门|葑门]]外一巨镇。”</ref>。

据史书记载，北宋元佑年间（1086年），周[[宣教郎|迪功郎]]信奉佛教，将庄田 200亩（13公顷多）捐赠给全福寺作为庙产，百姓感其恩德，将这片田地命名为“周庄”。1127年，金二十相公跟随宋高宗南渡迁居于此，人烟逐渐稠密。

周庄在元代中期时属苏州府长洲县。明清中期属[[松江|松江]]府华亭县，清初复归长洲县。清雍正三年(公元1725年)，周庄镇因元和县一分为二，约五分之四属元和县(今吴县市)，五分之一属吴江县(今吴江市)。乾隆二十六年(公元1761年)，江苏巡抚陈文恭将原驻吴县[[甪直|甪直]]镇的巡检司署移驻周庄，管辖[[澄湖|澄湖]]、[[黄天荡_(苏州)|黄天荡]]、[[独墅湖|独墅湖]]、[[尹山湖|尹山湖]]和[[白蚬湖|白蚬湖]]地区，几乎有半个县的范围。

清末至民国年间，战乱频仍。周庄以其独特的地理条件，成为[[南社|南社]]发起人陈去病、柳亚子等人聚会酬唱之地。20年代，流品渐杂，成为方圆附近吴江、青浦乡里间的一大集镇。

1949年5月8日，周庄解放，归属吴江甪直区。1950年把镇西原属吴县部分划归吴江，结束了两县分治的状况。1952年以后，周庄镇归昆山县(现为昆山市)管辖。 

建国后，商业渠道随[[政区|政区]]而变化，趋向单一的[[农业经济|农业经济]]。[[改革开放|改革开放]]后，乡镇工业发展，对外交往增多。1986年昆（山）陈（墓）周（庄）公路建成通车，镇内以其保存完好的明清古建筑群，以及古朴的石拱桥、石河埠等吸引了大批游客。1988年周庄成立周庄旅游服务公司，旅游设施不断完善。来此考察和观光的各地游客、中外官员、学者、专家日益增多，并且成为写生作画、摄制影视片的热门场所。

== 历史 ==
远古时期，已有人类在镇北太史淀<ref>太史淀原作太史田，在急水港北。1976年冬围垦工程中，在湖东北部（今属王东村、秀南村）七处出土石器，3处黑衣陶和印纹陶，6处木井</ref>一带耕牧繁衍。

春秋至汉代时期，吴王少子摇和汉越摇君在周庄地域册封，称“摇城”。
         
南宋元祐元年（1086年），周迪功郎在周庄镇收获设庄，把自己的宅子改造为全福寺，周庄就是这么来的。

南宋建炎四年（1127年），[[韩世忠|韩世忠]]在周庄境内加强防守<ref>周庄有自然村名曰寒贞，前称韩镇，相传便是韩世忠驻守之所。</ref>。

元朝至顺元年（1330年），沈万三的父亲沈祐由湖南南浔沈家漾搬至周庄东垞，垦殖经营。人渐渐多了起来，便成了市镇。

明洪武六年（1373年），明太祖[[朱元璋|朱元璋]]建筑南京城垣，[[沈万三|沈万三]]助建三分之一，并请犒军，为皇帝猜忌，被发配到云南。

洪武七年（1374年），沈万三案株连的事情很多，有把周庄屠城之说。镇人徐民望到京城告御状，才没有做。

清顺治五年（1648年），总戎[[杨承祖|杨承祖]]迎海涌峰（今[[虎丘|虎丘]]）世尊大佛于全福寺。乾隆七年（1742年），沈本仁建敬业堂，为沈厅前身。乾隆十八年（1753年），[[章腾龙|章腾龙]]《贞丰拟乘》脱稿，嘉庆十三年（1808年）[[陈勰|陈勰]]增辑付梓，为周庄第一部方志。道光二十三年（1843年），建永安桥和世德桥（即[[双桥|双桥]]），两桥纵横相连，俗称钥匙桥。光绪八年（1882年），[[里人|里人]][[陶煦|陶煦]]编《周庄镇志》六册，刊印成书。

民国7年（1918年），[[柳亚子|柳亚子]]来周庄同[[叶楚怆|叶楚怆]]、[[王大觉|王大觉]]以文会友，作《南湖草堂夜集》。民国8年（1919年），[[五四运动|五四运动]]爆发，周庄乡立高等小学学校等校师生上街游行演讲。民国9年（1920）十二月，[[柳亚子|柳亚子]]抵周庄，居七日，唱酬于[[迷楼|迷楼]]，赋诗甚多，后编为《迷楼集》。民国12年（1923年）5月，叶楚怆为新[[南社|南社]]起草《发起宣言》，周庄[[陶惟坻|陶惟坻]]，[[朱翊新|朱翊新]]等10余人加入新南社。是年，[[柳会成|柳会成]]在周庄研制成第一台直流收音机，其容积有两只[[八仙桌|八仙桌]]大。民国35年（1946年），抗日战争胜利后，周庄改名楚怆镇。两年后复名周庄镇，是年，[[朱润苍|朱润苍]]《贞丰八年血泪录》问世。

1949年5月8日，周庄解放。

== 闻名 ==
1979年画家[[陈逸飞|陈逸飞]]至周庄[[写生|写生]]。1980年陈逸飞飞赴[[美国|美国]]，四年后获亨特学院艺术硕士学位。在纽约期间，陈逸飞的画作风格受到了哈默画廊的主人、美国西方石油公司董事长[[阿曼德·哈默|阿曼德·哈默]]博士的赏识<ref>{{Cite web |url= http://zh.wikipedia.org/wiki/%E9%99%88%E9%80%B8%E9%A3%9E |title= 维基百科“陈逸飞”词条 |author= 维基百科 |date=2012.7.28 16：00 最后修改 | }}</ref>，他以周庄中心双桥为题材的作品《故乡的回忆》被其以重金购下，并在1984年作为访问[[中国|中国]]送给[[邓小平|邓小平]]的礼物。大大促进了周庄及周围江南水乡的旅游业，由此周庄开始闻名。

在[[同济大学|同济大学]]教授[[阮仪三|阮仪三]]的帮助下<ref>{{Cite journal|title=阮仪三|url=https://zh.wikipedia.org/w/index.php?title=%E9%98%AE%E4%BB%AA%E4%B8%89&oldid=49418956|date=2018-05-04|journal=维基百科，自由的百科全书|language=zh}}</ref>，周庄决定保护自身的古建筑，维护原有的水乡风貌，开辟成旅游区，现在周庄所有的景点已经形成古镇旅游一票制。

周庄于2003年被评为[[中国历史文化名镇|中国历史文化名镇]]。

== 乡村 ==
周庄实行镇管村制度，下设20个行政村，分别为：东浜、全旺、高勇、王东、秀南、云南、对金、复兴、西泽、澄尚、龙亭、蟠龙、祁浜、建新、双庙、西田、龙凤、协义、蚬北、南湖、市镇。自然村共61个。
