'''周炜良'''（{{bd|1911年|10月1日|1995年|8月10日|catIdx=Z周}}<ref name="丘成桐谈数学" /><ref name="陈省身所写介绍" />），华裔[[数学|数学]]家，[[代数几何|代数几何]]研究领域的主要代表人物之一<ref name="结识吴文俊" />。由他引入的[[周形式|周形式]]、[[周环|周环]]及[[周坐标|周坐标]]在圈内知名度很高<ref name="结识吴文俊" />。<ref name="教育在线" /><ref name="陈省身所写介绍" /><ref name="介绍谢瓦莱">{{cite web |url= http://www.cssn.cn/zhx/201608/t20160823_3172864_8.shtml |format= |title=克劳德·谢瓦莱：布尔巴基巨匠、数学结构代言人 |author1=阎晨光 |author2=邓明立 |journal=自然辩证法通讯 |pages= |language=zh-cn |date=2016年8月23日 |access-date=2018年8月27日}}</ref>

周炜良20世纪30年代在德国学习，后回国在大学任职<ref name="丘成桐谈数学" />。他之后因为家庭原因离开学界，去管理家族企业<ref name="丘成桐谈数学" />。但因个人财富在[[第二次世界大战|第二次世界大战]]期间大量流失，他战后重新回归数学研究领域，并取得重要成就<ref name="丘成桐谈数学" />。他是世界级的华人数学家之一<ref name="家中老三" />，但他在公众中的知名度远低于其学术影响力，他取得的荣誉也不能充分匹配他的学术成就<ref name="陈省身所写介绍" /><ref>见{{harvnb|张奠宙|2003}}。</ref>。

== 生平 ==
1911年<ref name="结识吴文俊" /><ref name="丘成桐谈数学" />10月1日<ref name="陈省身所写介绍" />，周炜良出生于[[上海|上海]]的一个富裕家庭<ref>见{{harvnb|张奠宙|2003}}。</ref>，原籍[[安徽|安徽]]至德（今[[东至|东至]]）。他的父亲叫[[周美权|周美权]]（也就是周达<ref name="家中老三"/>）。周炜良在家中排行第3<ref name="家中老三">{{cite web |url= http://news.sciencenet.cn/sbhtmlnews/2010/2/228760.html?id=228760 |format= |title=丘成桐：从明治维新到二战前后中日数学人才培养之比较 |author=[[丘成桐|丘成桐]] |editor=卢小兵 |journal=科学时报 |pages="A3 观察”版面 |language=zh-cn |date=2010年2月3日 |access-date=2018年8月27日}}</ref>，是家中最小的孩子。

1924年，周炜良前往[[美国|美国]]读书。1929年，周炜良进入[[芝加哥大学|芝加哥大学]]学习[[经济学|经济学]]。在芝加哥大读书时，他看了代数专家{{link-en|巴特尔·范德瓦尔登|Bartel Leendert van der Waerden|B·L·范德瓦尔登}}写的名著《代数学》(''Algebra'')，并由此产生对数学的强烈兴趣<ref>见{{harvnb|张奠宙|2003}}。</ref>。1931年，他获得学士学位<ref name="结识吴文俊" />。1932年10月，周炜良前往当时的世界数学中心德国[[哥廷根大学|哥廷根大学]]，但不巧赶上了哥廷根大学走向衰落的时期<ref>见{{harvnb|张奠宙|2003}}。</ref>。于是他又于1933年夏天入读德国[[莱比锡大学|莱比锡大学]]，师从自己仰慕已久的范德瓦尔登研究[[代数几何|代数几何]]<ref name="结识吴文俊" />。1934年，他在汉堡大学度假期间认识了后来的妻子玛尔葛特·维克托(Margot Victor)<ref>见{{harvnb|张奠宙|2003}}。</ref>，还随数学家[[埃米尔·阿廷|埃米尔·阿廷]]听课<ref>见{{harvnb|张奠宙|2003}}。</ref>。1936年初，周炜良回到莱比锡，并获得莱比锡大学数学博士学位<ref name="结识吴文俊" />，并在[[德国|德国]]和维克托结婚<ref>见{{harvnb|张奠宙|2003}}。</ref>。数学家[[陈省身|陈省身]]是他婚礼上邀请的唯一中国宾客<ref>见{{harvnb|张奠宙|2003}}。</ref>。同年回国后，他担任[[南京|南京]][[國立中央大學|中央大学]]数学系教授。

1937年，[[抗日战争|抗日战争]]爆发后，周炜良因家庭原因，不得已回到上海<ref name="结识吴文俊" />。周的岳父也因德国的动乱和种族迫害而丢了工作，几乎身无分文地回到上海<ref>见{{harvnb|张奠宙|2003}}。</ref>。在之后的一段时间里，周炜良一直滞留于上海<ref name="结识吴文俊" />。

抗战胜利后，周炜良经[[陈省身|陈省身]]劝说，于1947年3月赴美<ref name="结识吴文俊" />。同年，他来到美国[[普林斯顿大学|普林斯顿大学]]，重返离开了10年的数学界。周炜良在普林斯顿向著名数学家[[所罗门·莱夫谢茨|所罗门·莱夫谢茨]]学习。在此期间里，他做出了许多开创性的工作<ref name="丘成桐谈数学" />。他于1949年起执教于美国[[约翰斯·霍普金斯大学|约翰斯·霍普金斯大学]]<ref name="结识吴文俊" />，{{fact|1955年}}起，他担任霍普金斯大学数学系主任，共11年。周炜良曾邀请日本数学家[[小平邦彦|小平邦彦]]去那里当一年客座副教授。<ref name="小平邦彦">{{cite web |url= http://www.nihaowang.com/news/news-78245.html |format= |title=游里工夫独造微：小平邦彦 |author=颜一清 |journal=数学传播 |volume=25 |issue=1 |pages= |language=zh-cn |date=2001 |access-date=2018年8月27日}}</ref>1952年，小平邦彦与周炜良证明出有两个[[代数独立|代数独立]](meromorphically independent)的[[有理函数|有理函数]]的曲面必为[[代数曲面|代数曲面]]<ref name="小平邦彦" />。1959年，他入选[[中華民國|中華民國]][[中央研究院|中央研究院]]第3屆(數理科學組)院士<ref>{{cite web |url= https://academicians.sinica.edu.tw/index.php?func=1-D |title=周煒良 逝世院士一覽表 |publisher=台湾[[中央研究院|中央研究院]] |language=zh-tw |date= |access-date=}}</ref>。 

1977年，周炜良退休。1995年<ref name="丘成桐谈数学" />8月10日，他在美国去世<ref name="陈省身所写介绍" />。<ref name="结识吴文俊" />

== 研究成果 ==
* [[周環|周環]]（也叫周氏[[环_(代数)|环]]<ref name="介绍谢瓦莱" />、周炜良环<ref name="介绍谢瓦莱" />）
* [[周簇|周簇]]
* [[周坐標|周坐標]]
* [[周定理|周定理]]

这些以周炜良名字命名的数学名词常见于各种代数几何学的专著、教材及数学辞典。其中仅日本《岩波数学词典》中就收录达7个之多。<ref name="陈省身所写介绍" />

== 个人生活 ==
周炜良淡泊名利，不喜欢社交<ref name="陈省身所写介绍" /><ref>见{{harvnb|张奠宙|2003}}。</ref>。虽然身为中华民国[[中央研究院|中央研究院]]院士，但是长期不参加会议活动<ref>见{{harvnb|张奠宙|2003}}。</ref>。

他与父亲周达都有[[集邮|集邮]]的爱好<ref>见{{harvnb|张奠宙|2003}}。</ref>。

=== 逸闻 ===
周炜良曾经赵孟养的介绍，认识了中国数学家[[吴文俊|吴文俊]]<ref name="结识吴文俊" />。周炜良在研究风格上曾给吴文俊提出过中肯的意见，指出吴的做法不好、杀鸡不应用牛刀，这使吴对这一评价非常重视。<ref>{{cite web |url= http://www.amss.cas.cn/kxcb1/kpzt/wwj/xsly/201705/P020170508438793962415.pdf |format=pdf |title=读吴文俊口述自传——《走自己的路》，缅怀数学大师 |author=林开亮 |website=中国科学院数学与系统科学研究院 |pages=2 |language=zh-cn |date=2017年5月8日 |access-date=2018年8月27日}}</ref>
