'''周刊邮报'''（{{Jpn|j=週刊ポスト|kn=しゅうかんポスト|f=h}}）是日本[[小學館|小學館]]出版社发行的综合周刊[[杂志|杂志]]。

==概要==
《周刊邮报》于1969年8月22日創刊<ref>{{cite web |title=週刊ポスト　創刊号　昭和44年8月22日号 |url=https://www.kosho.or.jp/products/detail.php?product_id=135980046 |publisher=東京都古書籍商業協同組合 |accessdate=2019-02-15}}</ref>。前身是面向初高中学生的杂志《{{tsl|ja|ボーイズライフ|少年生活}}》。纸张规格为B5（182×257mm），通常周一发行<ref>{{cite web |title=週刊ポスト 発売日 判型 |url=https://www.shogakukan.co.jp/magazines/series/083000 |publisher=小学館 |accessdate=2019-02-15}}</ref>。杂志的[[商标|商标]]权归[[獅王|獅王]]公司所有<ref>{{cite web |title=商標番号照会 （週刊ポスト商標登録番号第942495号）|url=https://www.j-platpat.inpit.go.jp/web/shouhyou/sybs/SYBS_GM101_Top.action|publisher=JPO & INPIT |accessdate=2019-02-15}}</ref>。

《周刊邮报》的实际贩卖量在1988年至1991年的4年间，以及1994年至2003年的10年间，位列日本国内综合周刊杂志的首位<ref>{{cite web|url=http://homepage2.nifty.com/htnk/200212.htm|title=1988年下期～2002年の週刊誌の実売部数の推移|publisher=nifty}}{{Dead link|date=2019年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。然而在2004年，它的实际贩卖量下滑到约80万份，输给了《[[周刊文春|周刊文春]]》。从此之后，贩卖量一直呈下滑趋势，2008年度上半年实际贩卖量30万6010份<ref>{{cite web|url=http://web.archive.org/web/20090308001932/http://sankei.jp.msn.com/culture/books/081115/bks0811150806003-n1.htm |title=【花田紀凱の週刊誌ウォッチング】（１８４）|publisher=[[產經新聞|MSN産経ニュース]]|date=2009-03-08}}</ref>，2008年度下半年29万7120份<ref>{{cite web|url=http://web.archive.org/web/20090516004024/http://sankei.jp.msn.com/culture/books/090509/bks0905091301005-n1.htm|title=【花田紀凱の週刊誌ウォッチング】（２０８）|publisher=MSN産経ニュース|date=2009-05-16}}</ref>。2018年度上半年降至21万1336份<ref name=PRE>{{cite web |title=文春砲も苦戦「売れない週刊誌」の断末魔 |url=https://president.jp/articles/-/26864 |publisher=PRESIDENT Online |accessdate=2019-02-15 |date=2018-12-01|archiveurl=https://web.archive.org/web/20181204130112/https://president.jp/articles/-/26864|archivedate=2018-12-04}}</ref>。

目前主要的读者层是中老年男性<ref>{{cite web |title=石原さとみが強すぎ！老若男女問わず「好きな顔」だと判明 |url=https://news.biglobe.ne.jp/entertainment/0716/mes_180716_3996058639.html |publisher=BIGLOBE |archiveurl=https://web.archive.org/web/20190215143131/https://news.biglobe.ne.jp/entertainment/0716/mes_180716_3996058639.html |archivedate=2019-02-15 |date=2018-07-16}}</ref>。自创刊以来，《周刊邮报》几乎都是以一位女演员或女性电视艺人的照片作为封面图片。然而从2016年1月开始，采用文章大标题形式作为封面，不再出现封面图片。其封面内容也更关注[[老人|老人]]健康<ref name=PRE/>，多刊载与[[退休金|退休金]]和医疗等有关的信息<ref>{{cite web |author=篠田博之 |title=平成のスクープ誌『週刊文春』でも部数減が止まらない理由 |url=https://ironna.jp/article/11583 |publisher=iRONNA（[[產經新聞|産経新聞]]） |archiveurl=https://web.archive.org/web/20190215142620/https://ironna.jp/article/11583 |archivedate=2019-02-15 |date=2019-01-10}}</ref>。
