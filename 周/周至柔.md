'''周至柔'''（{{bd|1899年|11月30日|1986年|8月29日|catIdx=Zhou周}}），原名'''百福'''，字'''至柔'''，[[以字行|以字行]]<ref name="李國鼎">李國鼎：我的台灣經驗－李國鼎談台灣財經決策的制定與思考</ref>{{rp|620}}，[[中華民國陸軍|中華民國陸軍]]、[[中華民國空軍|空軍]][[一級上將|一級上將]]。[[浙江|浙江]][[台州|台州]][[臨海縣|臨海縣]]人<ref name="百年人物全書，第1602頁">《中國國民黨百年人物全書》下，劉國銘主編，北京：團結出版社，2005年12月，第1602頁，ISBN 7-80214-039-0</ref>。1951年，晉陆军[[一級上將|一級上將]]銜<ref name="百年人物全書，第1603頁">《中國國民黨百年人物全書》下，劉國銘主編，北京：團結出版社，2005年12月，第1603頁，ISBN 7-80214-039-0</ref>。
[[File:周至柔2.jpg|thumb]]

== 早年經歷 ==
生於1899年11月30日（舊曆10月28日）。<ref name="百年人物全書，第1602頁"/>1919年，考取[[保定軍官學校|保定軍官學校]]第八期步科，在校與[[陳誠|陳誠]]、[[羅卓英|羅卓英]]等結為[[換帖|換帖]]兄弟。<ref name="百年人物全書，第1602頁"/>1922年畢業，在[[中國國民黨|中國國民黨]]軍事派系中，屬於陳誠[[土木系_(國民革命軍)|土木系]]。後任[[黃埔軍校|黃埔軍校]]教官，參與[[國民黨北伐|國民黨北伐]]。曾出國考察[[空軍|空軍]]教育，返國後被任命為[[中央航空學校|中央航空學校]]校長。

== 中年經歷 ==
1936年，調[[航空委員會|航空委員會]]主任。[[抗戰|抗戰]]中任中國空軍的總指揮。1946年，航空委員會改制為[[國防部空軍司令部|空軍總司令部]]，中國空軍正式建軍。周至柔擔任第一任空軍總司令，但其军種仍为陆军，并未改為空军。

== 晚年經歷 ==
1949年4月10日到溪口<ref name="寧靜">[[蔣經國|蔣經國]]著：〈危急存亡之秋〉，載《風雨中的寧靜》，台北：[[正中書局|正中書局]]，1988年</ref>{{rp|181}}。同年隨國軍到[[台灣|台灣]]。遷台初期曾指揮空軍[[上海空戰|轟炸上海]]。

1950年3月，蔣介石以周至柔為[[參謀總長|參謀總長]]，[[孫立人|孫立人]]為陸軍總司令<ref name="年表">[[陳布雷|陳-{布}-雷]]等編著：《蔣介石先生年表》，台北：[[傳記文學|傳記文學]]出版社，1978-06-01</ref>{{rp|64}}。3月17日，以陸軍中將軍銜接替顧祝同擔任參謀總長，並兼任空軍總司令。4月25日，晉升陸軍二級上將<ref>[http://gaz.ncl.edu.tw/detail.jsp?sysid=E0606221]《總統令 (1950年3月17日)》，臺北，中華民國總統府公報，1950-03-31，第3版</ref><ref>[http://gaz.ncl.edu.tw/detail.jsp?sysid=E0606268]《總統令(1950年4月25日)》，臺北，中華民國總統府公報，1950年4月29日，第2版</ref>。在國防部任內，建立了軍官退休制度和輪調制度。後被捲入[[毛邦初|毛邦初]]事件，但經查無辜。

1954年2月25日，樂信‧瓦旦原以「高山匪諜案」的叛亂罪名被判處15年，後來被時任參謀總長周至柔改判死刑，蔣介石核示「如擬」，1954年2月25日判決死刑確定。他們分別是：阿里山鄒族的[[湯守仁|雅巴斯勇‧優路拿納]]（[[湯守仁|湯守仁]]）、吾雍‧雅達烏猶卡那（[[高一生|高一生]]，吳鳳鄉鄉長)、[[汪清山|汪清山]]（嘉義縣警察局巡官，歌手[[高慧君|高慧君]]的外祖父)、方義仲(達邦村村長)、角板山泰雅族的[[樂信·瓦旦|樂信‧瓦旦]]（[[林瑞昌|林瑞昌]]：臺灣省議員)、高澤照（桃園縣警察局巡官）。1954年4月17日下午2時30分，在被驗明正身之後，六位原住民社會的領導人被發交台北憲兵隊，綁赴馬場町刑場執行槍決。

1954年7月，蔣介石以周至柔為國防會議秘書長<ref name="年表" />{{rp|78}}<ref>澤生：〈五十年代台灣國防部見聞〉，《[[明報月刊|明報月刊]]》，[[香港|香港]]：[[明報|明報]]雜誌有限公司，1991年1月號</ref>{{rp|62}}。1957年，任第6任[[台灣省政府|台灣省政府]]主席，任內主持[[新竹海埔新生地|新竹海埔新生地]]工程。

1962年，任[[總統府|總統府]]參軍長。1967年，任[[國家建設計劃委員會|國家建設計劃委員會]]主任委員。1986年8月29日，任內病逝於[[臺北|臺北]][[三軍總醫院|三軍總醫院]]，享年88歲。骨灰罈安置於[[美國|美國]][[紐約|紐約]][[海德公園|海德公園]]附近[[莊嚴寺|莊嚴寺]]內<ref>澤生：〈台灣海軍的轉折與發展見聞錄〉，《明報月刊》，香港：明報雜誌有限公司，1993年2月號</ref>{{rp|116}}。
