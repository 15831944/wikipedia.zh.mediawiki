'''周滨'''（{{bd|1972年|1月|||}}），[[江苏|江苏]][[无锡|无锡]]人，商人，毕业于[[西南石油大学|西南石油大学]]，前[[中共中央政治局常委|中共中央政治局常委]]兼[[中央政法委|中央政法委]][[书记|书记]][[周永康|周永康]]之子。[[周永康案|周永康案]]重要關係人。'''周滨'''与四川涉黑商人[[刘汉_(四川商人)|刘汉]]是生意合作伙伴<ref name="lzgx">{{cite web |url = http://www.rfa.org/mandarin/yataibaodao/renquanfazhi/cyl-02212014102657.html|title=刘汉的保护伞是谁？|publisher = RFA |date = 2014-02-21}}</ref>。2013年9月，传出其因涉嫌经济犯罪被[[軟禁|軟禁]]的消息<ref name="法广2013">{{cite web |url = http://www.chinese.rfi.fr/%E4%B8%AD%E5%9B%BD/20130921-%E6%8D%AE%E4%BC%A0%E5%91%A8%E6%B0%B8%E5%BA%B7%E4%B9%8B%E5%AD%90%E5%91%A8%E6%BB%A8%E8%A2%AB%E6%8E%A7%E5%88%B6 |title = 据传周永康之子周滨被控制 |publisher = 法广 |date = 2013年9月21日}}</ref>。周濱涉嫌參與四川省與石油工業的非法交易，將面臨起訴，北京獨立學者[[高瑜|高瑜]]表示，因其涉及石油業非法交易的報導可信度高，周濱若被起訴意謂着中國政府正式將司法清算重點移向周永康家族。<ref name="世界日報2014">{{cite news|url=http://worldjournal.com/bookmark/24374590--|newspaper=世界日報|title=周永康子涉貪被訴最高可處死|publisher=世界日報|date=2014年1月11日}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
周滨已经因涉嫌非法经营罪，被[[湖北省|湖北省]][[宜昌市|宜昌市]]检察院批准逮捕。<ref name="财经网">{{cite news |url = http://politics.caijing.com.cn/20140729/3636187.shtml |newspaper=财经网 |title = 周永康之子周滨因涉嫌非法经营罪被批捕|publisher=财经网 |date = 2014年7月29日}}</ref>2016年6月15日，湖北省宜昌市中级人民法院一审判处周滨有期徒刑18年，并处罚金3.502亿，周滨服从不上诉。<ref>[http://news.sohu.com/20160615/n454581610.shtml 周永康之子周滨一审获刑18年 并处罚金3.502亿]</ref>

== 履历 ==
周滨自[[西南石油大学|西南石油大学]]本科毕业后，在[[德克萨斯州|美国德州]][[得克萨斯大学|得克萨斯大学]][[达拉斯|达拉斯]]分校攻读石油相关专业的研究生学位，在美国期间，周滨同[[黄渝生|黄渝生]]的女儿[[黄婉|黄婉]]已有接触，2000年后，周滨和黄婉的住址都变为黄婉家人所居住的[[新泽西州|新泽西州]]。当时，周滨与黄婉已经将重心转移到中国，经常居住在北京。周滨回到中国后曾在全球知名的油田服务公司斯伦贝谢供职，2003年前后就拥有自己创办的公司，2009年12月成为北京中旭阳光能源科技股份有限公司的股东。中旭阳光只是周滨夫妇投资的公司之一，周滨也经常辗转于香港等地，料理其他生意。<ref>{{cite web|url=http://finance.sina.com.cn/china/20140226/121618337650.shtml|title=周滨其人：身材高大眼睛细长 白手套替其打理生意|publisher=新浪网|date=2014年2月26日}}</ref>

== 腐败问题 ==
{{see|周永康案}}
2013年9月25日，中国[[财新网|财新网]]首次披露周滨及岳父全家在[[2013年中石油腐败案|中石油系统的腐败]]行为，报道发出不久即遭删除，但相关稿件于11月再度复活<ref>{{cite web |url = http://www.dw.de/%E8%B4%A2%E6%96%B0%E7%BD%91%E7%82%B9%E5%90%8D%E5%91%A8%E6%BB%A8%E5%91%A8%E6%B0%B8%E5%BA%B7%E9%9A%BE%E5%AE%89/a-17253066|title=财新网点名周滨，周永康难安？|publisher = 德国之声|date = 2013-11-25 }}</ref>。11月22日，财新网再发特稿，披露中共自2013年8月底在中石油系统掀起的反腐风暴，包括中海油也难逃反腐清查。文中直接点名周滨涉嫌以不正当手段获取高额利益。之前，已经至少有8名与周滨父親[[周永康|周永康]]关系密切的中石油高层和心腹被捕<ref>{{cite web |url = http://www.bbc.co.uk/zhongwen/simp/china/2013/11/131122_china_cnpc_zhoubin.shtml |title = 中国石油反腐风暴持续　周滨旧友“被带走”|publisher = BBC |date = 2013年11月22日}}</ref>。
