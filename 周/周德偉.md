'''周德偉'''（{{bd|1902年|10月16日|1986年|catIdx=Zhou周}}），[[湖南|湖南]][[長沙|長沙]]人，[[中華民國|中華民國]][[自由主義|自由主義]][[經濟學|經濟學者]]。[[湖南大學|湖南大學]]經濟系[[教授|教授]]兼系主任，第二屆[[國民參政員|國民參政員]]。到[[台灣|台灣]]後曾任[[中華民國財政部|中華民國財政部]][[財政部關務署|關務署]][[署長|署長]]、外匯貿易改革委員會副主任委員，並在[[台灣大學|台灣大學]]、[[政治大學|政治大學]]兼任教授<ref>
{{cite web
|url=http://www.ylib.com/hotsale/wistaria/index.asp
|title=關於周德偉
}}</ref>。

==早年==
1916年考入[[長郡中學|湖南第一聯合縣立中學]]，1919年四年級肄業。1920年入[[國立北京大學|國立北京大學]]預科，從[[哲學系|哲學系]]轉入經濟系。在北大期間，周德偉師從[[陶孟和|陶孟和]]、[[顧孟餘|顧孟餘]]等北大教授。當時北大的[[馬克斯|馬克斯]]學說研究會，曾吸收他做會員，但很快地就退出了。是少數非[[左派|左派]]的學生。1925年春，[[汪精衛|汪精衛]]在北方四處演講，宣揚[[中山主義|中山主義]]，感動了周德偉。周德偉見當時的形勢，[[北洋政府|北洋政府]]已屆末日，只有[[國民黨|國民黨]]與[[中國共產黨|共產黨]]夠格打倒北洋政府。他不相信[[共產主義|共產主義]]及[[唯物史觀|唯物史觀]]，故加入國民黨。1928年，周德偉在[[天津|天津]]創辦《[[天津雙週刊|天津雙週刊]]》，鼓吹實行憲政，反對軍人主政，反對打內戰，主張南方好好發展經濟、厚植實力，軍閥勢力自然會被歷史淘汰。但到了1929年「目睹當時國民黨之所謂訓政，無非新軍閥、新財閥攘奪權力，距離群眾益遠。新軍閥以[[蔣介石|蔣介石]]為領袖，附者盡闒庸奴才，財閥亦蔣氏一手培植……」「北洋舊財閥多以向蔣氏輸誠，滲入國民黨內分享杯羹。而才智之士，則排斥殆盡，歷朝開國規模之隘，無逾於此者……」 「當時萬念叢生，真不知如何著手，曾思獨組一自由黨闡明吾說，只使政府職權協助人民自由發展，而當時謀生不暇，資歷聲望又所不孚，不足以資號召，只有構想而已，終身無成愧恨而已。」1929年周德偉加入了以實施[[憲政|憲政]]為號召，由[[汪精衛|汪精衛]]、[[顧孟餘|顧孟餘]]領導以青年知識份子為主的國民黨「改組派」秘密組織。

==留學==
1932年汪蔣復合，黨內及政界眼看前途不錯，他卻自認學識不足。他相信要救中國，必須要在[[經濟學|經濟學]]上下功夫。在別人積極從事政黨活動時，他卻用功讀書，在1933年拿到鐵道部獎學金，留學[[英國|英國]]，進入[[倫敦政治經濟學院|倫敦政治經濟學院]]，師從著名[[自由主義|自由主義]][[經濟學|經濟學家]][[弗里德里克·哈耶克|弗里德里克·哈耶克]]，廣讀奧國學派與北歐學派的論著。後又轉入[[柏林大學|柏林大學]]，用[[德文|德文]]完成《中立貨幣論》論文<ref name="不容青史盡成灰">
{{cite web
|url=http://www.taaze.tw/sing.html?pid=11100566176
|title=落筆驚風雨：我的一生與國民黨的點滴 他序 不容青史盡成灰
|author=周渝
}}</ref>。

==返國後的經歷==
1937年[[抗日戰爭|抗日戰爭]]爆發，周德偉返國任教[[湖南大學|湖南大學]]經濟系兼系主任。與時任湖大文學院長前北大老同學[[李壽雍|李壽雍]]先生，合作創辦《中國之路》半月刊，闡揚[[自由主義|自由主義]]、[[民主|民主]]、[[法治|法治]]、[[人權|人權]]與[[市場經濟|市場經濟]]等學理與主張，並對當時的[[共產主義|共產主義]]或左翼社會主義、計畫經濟嚴加批評<ref name="豈有文章覺天下">
{{cite web
|url=http://www.ylib.com/hotsale/wistaria/index.asp
|title=豈有文章覺天下，忍將功業苦蒼生
|author=周渝
}}
</ref>。後來在[[重慶|重慶]]任第二屆[[國民參政員|國民參政員]]。周德偉並在[[重慶大學|重慶大學]]、[[中央大學|中央大學]]兼任教授。

==在台灣的經歷== 
1949年到台灣。1950年擔任財政部關務署署長，並在[[台灣大學|台灣大學]]、[[政治大學|政治大學]]兼任教授。在財政部時致力於外匯貿易改革方案，此方案於1958年[[尹仲容|尹仲容]]任外匯貿易改革委員會主任委員、他擔任副主任委員時完成。1969年自關務署署長一職退休。

50年代關務署署長宿舍[[紫藤廬|紫藤廬]]成為以台大為中心的[[自由主義|自由主義]]學者的聚會場所，[[張佛泉|張佛泉]]、[[殷海光|殷海光]]、[[夏道平|夏道平]]、[[徐道鄰|徐道鄰]]等十餘位學術界人士每隔兩週在此聚會討論，有系統地介紹哈耶克的思想。周德偉譯有八十萬字哈耶克巨著《自由憲章》，[[殷海光|殷海光]]先生並在其鼓勵之下翻譯《[[通往奴役之路|到奴役之路]]》。當時在台大就讀的[[李敖|李敖]]、[[陳鼓應|陳鼓應]]、[[林毓生|林毓生]]、[[張灝|張灝]]等亦常來紫藤廬與前輩交流<ref name="不容青史盡成灰"/>。在當時[[蔣介石|蔣介石]]政府[[台灣白色恐怖時期|白色恐怖]]的政策下，紫藤廬常被[[特務|特務]]監視，[[電話|電話]]更是被監聽。在當時有些年輕人被抓走，家人朋友來請託周德偉，希望以他在黨國中的資深地位與人脈，能保出被抓的人。周德偉為他們打電話，求託朋友，尋求關係，但終難有效果<ref name="豈有文章覺天下"/>。

==旅居美國==
1975年偕妻旅居[[美國|美國]][[加州|加州]][[洛杉磯|洛杉磯]]。周德偉並且親筆寫下自傳《落筆驚風雨：我的一生與國民黨的點滴》，可惜僅書至其任教湖南大學而已。書中基於其第一手回憶與官方民國史、國民黨黨史有非常不同的觀點。他說：「我不寫國民黨史，國民黨將永無真史！」

==子女==
女兒[[周芷|周芷]]是大學聯考三類組榜首，獲選為[[美國國家科學院|美國國家科學院]]外籍院士。小兒子周渝是[[紫藤廬|紫藤廬]]茶館創辦人。
