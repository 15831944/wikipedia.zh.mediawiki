'''周村区'''是山东省[[淄博市|淄博市]]下属的市辖区，面积263平方千米，2012年，周村区总面积307平方公里，辖3个镇、5个街道。区政府驻青年路街道。

周村区地势南高北低。孝妇河、范阳河、淦河、米沟河、水浊河流贯境内。周村区属[[温带季风气候|温带季风气候]]，年均气温13.1℃，年均降水量646.3毫米。矿藏有铝矾土、焦宝石、石英、耐火材料等。是山东省重要的纺织工业基地。农业主产小麦、玉米、大豆。胶济铁路，309、308国道过境。主要旅游景点有周村古商城、蒲松龄书馆等。周村区有“全国科普示范区”、“天下第一村”等荣誉。

== 历史 ==

周村区辖区曾分属[[淄川县|淄川县]]、[[长山县_(隋朝)|长山县]]。周村之名可追溯至明朝[[嘉靖|嘉靖]]年间的“周村店”。清代周村因商业贸易而繁荣。清中期，各地民变四起。[[咸丰_(年号)|咸丰]]十一年（1861年）二月十一日和[[同治|同治]]元年（1862年），[[捻军|捻军]]两度攻克周村。在1861年，开始修筑[[周村城墙|周村城墙]]，三年后筑成<ref name="a1"/>。

1937年12月中旬，[[侵华日军|侵华日军]]迫近周村。24日，驻守周村的国民革命军第22师师长[[谷良民|谷良民]]率部南逃。25日，日军侵占周村西北一带，26日攻占周村。1945年8月，[[日本投降|日本投降]]后，[[国民政府|国民政府]]和[[中共政权|中共政权]]对此地争夺激烈。至[[第二次国共内战|第二次国共内战]]期间，周村在国共双方之间四度易手。最终，中共取得[[周村战役|周村战役]]的胜利，[[中国人民解放军华东野战军|中国人民解放军华东野战军]][[山东兵团|山东兵团]]第九纵队在1948年3月12日占领周村。[[李荆和|李荆和]]获中共渤海区任命，为中共周村市委书记兼市长<ref name="a1">{{Cite web |url=http://www.zhcqq.gov.cn/dashiji/200408/6.html |title=大事记 1840年～1948年|accessdate=2016-05-02 |author= |date=2004-08-11|work= |publisher=周村区情网，来源：山东省淄博市周村区史志办公室|language=Zh-hans }}</ref>。

1950年3月，[[山东省人民政府|山东省人民政府]]撤消周村市建制，并入[[长山县_(隋朝)|长山县]]。11月28日，山东省人民政府依政务院批复，设立[[淄博市|淄博市]]（县级市）、[[张周市|张周市]]（县级市），属[[淄博专区|淄博专区]]。周村区和[[张店区|张店区]]为张周市下辖区，周村区为张周市机关驻地。1955年3月，[[山东省人民委员会|山东省人民委员会]]撤消张周市建制，设立县级周村区<ref>{{Cite web |url=http://www.zhcqq.gov.cn/dashiji/200408/7.html |title=大事记 1949年～1978年|accessdate=2016-05-02 |author= |date=2004-08-11|work= |publisher=周村区情网，来源：山东省淄博市周村区史志办公室|language=Zh-hans }}</ref>

==行政区划==
周村区辖5个街道、4个镇：
* 街道：丝绸路街道、大街街道、青年路街道、永安街道、城北街道。
* 镇：北郊镇、南郊镇、王村镇、商家镇。
