'''周振基教授'''， 金紫荆星章、银紫荆星章、铜紫荆星章、太平绅士，{{BD|1954年|||}}，香港出生，祖籍广东汕头峡山，香港大學博士。

==生平==
周振基，企业家，出生于香港。中学毕业后赴美国金门大学继续深造，获商学院学士及工程管理硕士学位，随后在香港大学攻读经济及工商管理课程，获得博士学位。<ref>{{cite web |url=http://stda.shantou.gov.cn/article/view.asp?cid=160&sid=0&vid=807 |title=周振基简介 |access-date=2014-01-06 |archive-url=https://web.archive.org/web/20140106062356/http://stda.shantou.gov.cn/article/view.asp?cid=160&sid=0&vid=807 |archive-date=2014-01-06 |dead-url=yes }}</ref>

作为粤剧的爱好者，周氏拜香港粤剧名伶[[林家声|林家声]]为师，2012年及2016年先後被香港艺术学院授予荣誉博士及榮譽教授。<ref>{{cite web |url=http://www.takungpao.com/hkol/content/2012-06/29/content_590812.htm |title=周振基获颁香港艺术学院荣誉博士学位 |access-date=2014-01-06 |archive-url=https://web.archive.org/web/20140106055445/http://www.takungpao.com/hkol/content/2012-06/29/content_590812.htm |archive-date=2014-01-06 |dead-url=yes }}</ref>此外，周氏亦曾担任国内外多间大学及教育机构的校监或校董。

1981年，周氏在香港创立振基电子集团，主要供应超声波铝线和一系列半导体及电子工业物料。2006年，周氏在广东省汕头市创立骏码凯撒有限公司，主要从事高科技半导体及微电子材料产品的自主研发和生产。<ref>{{cite web |url=http://www.dahuawang.com/strb/html/2011-02/01/content_180487.htm |title=香港潮州商会副会长周振基教授：“幸福汕头”让人感到家的温暖 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
