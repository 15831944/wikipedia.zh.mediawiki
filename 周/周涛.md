'''周涛'''（{{bd|1968年|3月23日|catIdx=Z周}}），[[安徽|安徽]][[淮南|淮南]]人。[[女|女]]，[[汉族|汉族]]。[[中国|中国]]著名节目[[主持人|主持人]]，曾任央视大型节目中心副主任，现任北京演艺集团首席演出官。多次于中国中央电视台春节联欢晚会中担任主持。1995年担任[[中央电视台|中央电视台]]《[[综艺大观|综艺大观]]》（现已停播）节目主持人后成为家喻户晓的人物。

== 简介 ==
周涛于1993-1995年进入[[北京电视台|北京电视台]]，为《北京新闻》主播人之一。1995-2000年就职于中央电视台，担任《综艺大观》主持人，此后，随着综艺大观的成功，周涛成为家喻户晓的主持人之一，另外，她开始多次担任中国中央电视台春节联欢晚会的主持人。2001年开办以环保为主题的综艺节目《真情无限》，任制片人、主持人。2008年主持北京奥运会开幕式。

2016年11月，离开央视，调入[[北京演艺集团|北京演艺集团]]，担任首席演出官<ref>{{cite news |title = 央视主持人周涛正式调入北京演艺集团，担任首席演出官|url = http://www.thepaper.cn/newsDetail_forward_1564356 |accessdate=2016-11-19 |publisher = 澎湃新闻}}</ref>。

== 家庭 ==
周涛的前夫叫[[姚科|姚科]]，是中国[[中央人民广播电台|中央人民广播电台]]《千里共良宵》的主持人，也是比周涛高三届的学长，两人最后不欢而散<ref name="cob " >{{cite web |url=http://ent.ifeng.com/idolnews/mainland/detail_2011_09/22/9388387_0.shtml |title=央视主播周涛前夫姚科曝光 央广王牌栏目主持人 |accessdate=2011-09-22 |work= |publisher=凤凰网 |language =zh-cn}}</ref>。周涛现在的丈夫叫[[路云|路云]]，两人1995年相识，路云是一个从事文化产业的商人<ref name="cob " /><ref name=" shark" >{{cite web |url= http://ent.163.com/special/00032H0C/cctv1229.html|title=央视主持人的另一半 |accessdate=2011 |work= |publisher=网易 |language =zh-cn}}</ref>。
