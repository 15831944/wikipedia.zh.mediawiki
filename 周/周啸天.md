'''周啸天'''（{{bd|1948年|5月||}}），号'''欣托'''，[[四川|四川]][[渠县|渠县]]人，曾任[[四川大学|四川大学]]文学与新闻学院教授。[[第六届鲁迅文学奖|第六届鲁迅文学奖]]诗歌奖得主。<ref>{{cite web|url =http://lj.scu.edu.cn/cdwxxy1.2/index.php?app=default&act=teacher_view&id=2987481181195C7C69F08C18F949CC80&is_dimission=1|title =离退休教职工-周啸天|publisher =四川大学文学与新闻学院}}</ref>

== 生平 ==
1977年1月，毕业于[[西华师范大学|西华师范大学]]数学系。1981年12月，获[[安徽师范大学|安徽师范大学]]文学硕士学位。1982年，在[[四川师范大学|四川师范大学]]中文系任教。1992年，任四川大学文学与新闻学院教授。现任中华诗词学会副会长。<ref>{{cite web |url = http://cd.wenming.cn/top/ddjt2014/jsk/yjz/201502/t20150225_1601149.shtml |title = 周啸天 |publisher = 成都文明网 |date=2015-02-25}}</ref><ref>{{cite web |url = http://www.artsweb.cn/usercenter.php?act=tapersonalintroduction&u=2870&cssid=zy_tit1 |title = 周啸天_介绍 |publisher = 华夏艺术网}}</ref>

== 所获奖誉 ==
