'''周明權'''博士{{OBE|C}}{{JP}}（{{lang-en|'''Dr. Joseph Ming-kuen Chow'''}}，{{bd|1941年|7月19日|2018年|10月13日|Chow, Joseph}}，享年77歲），周明權工程顧問有限公司創辦人，[[香港考試及評核局|香港考試及評核局]]前主席，[[香港工程師學會|香港工程師學會]]前會長。

== 生平 ==
周明權為專業土木及結構工程師，早年就讀[[聖若瑟書院|聖若瑟書院]]，1964年畢業於[[香港大學|香港大學]]，取得[[工學士|工學士]]學位，1992年創立周明權工程顧問有限公司，從事本地和海外大型基建和建築項目的設計、建造和管理工作<ref>[http://www4.hku.hk/honfellows/chit/honorary-university-fellows/ir-dr-joseph-ming-kuen-chow/ 2013 名譽大學院士 周明權博士][[香港大學|香港大學]]</ref>。曾經參與[[紅磡|紅磡]][[海底隧道_(香港)|海底隧道]]和[[香港地鐵|地下鐵路]]等大型基建工程，並在工程界擔任多份公職，包括香港工程師註冊局及香港建造業工人註冊管理局主席<ref name="hku_press_release">{{Cite web |url=https://www4.hku.hk/honfellows/chit/press-releases/20131007 |title=香港大學今年頒授名譽大學院士銜予8位傑出人士 |accessdate=2018-10-14 |date=2013-10-07 |publisher=香港大學 新聞稿}}</ref><ref name="mingpao1014"/>。1992年，獲委任為[[太平紳士|太平紳士]]，1997年獲頒英帝國[[官佐勳章|官佐勳章]]。1999年獲[[香港理工大學|香港理工大學]]頒授榮譽工程學[[博士|博士]]學位。<ref name="hku_press_release"/>

但在2000年因受到[[圓洲角短樁案|居屋短樁事件]]影響，周明權當時[[香港房屋委員會|香港房屋委員會]]成員，也是周明權工程顧問的主要顧問，受到角色衝突問題而退任<ref>[http://std.stheadline.com/archive/fullstory.asp?andor=or&year1=2000&month1=2&day1=3&year2=2000&month2=2&day2=3&category=all&id=20000203a01&keyword1=%20&keyword2 周明權又任短樁公司顧問]2000年2月3日，[[星島日報|星島日報]]</ref>。

1996年至2003年任[[香港考試及評核局|香港考試及評核局]]主席<ref>[http://hk.apple.nextmedia.com/news/art/20030822/3489609 周明權玩名出神入化][[蘋果日報_(香港)|蘋果日報]]，2003年8月22日</ref>。2001年中，當選為[[香港工程師學會|香港工程師學會]]會長。

2013年10月，獲香港大學頒授名譽大學院士銜<ref name="hku_press_release"/>。

面對香港土地供應緊拙，2018年初社會有人提議在[[葵涌貨櫃碼頭|葵涌貨櫃碼頭]]上建造房屋，周明權與[[麥齊光|麥齊光]]表態支持，周指於該處上蓋建屋在建造時間、成本和技術方面都有優勢，「與在平地上建屋分別不大」，在總面積283公頃的貨櫃碼頭上建屋相當於9個[[太古城|太古城]]或16個[[美孚新邨|美孚新邨]]，足以容納幾十萬居民，他預計需時6至7年發展。<ref>{{Cite web |url=https://www.am730.com.hk/news/%E6%96%B0%E8%81%9E/%E5%91%A8%E6%98%8E%E6%AC%8A%E5%80%A1%E8%91%B5%E6%B6%8C%E8%B2%A8%E6%AB%83%E7%A2%BC%E9%A0%AD%E4%B8%8A%E8%93%8B%E5%BB%BA%E5%B1%8B-%E9%BA%A5%E9%BD%8A%E5%85%89%EF%BC%9A%E5%80%BC%E5%BE%97%E7%A0%94%E7%A9%B6-112622 |title=周明權倡葵涌貨櫃碼頭上蓋建屋 麥齊光：值得研究 |date=2018-01-22 |publisher=am730}}</ref><ref>{{Cite web |url=https://hk.finance.appledaily.com/finance/daily/article/20180118/20278018 |title=麥齊光倡貨櫃碼頭起樓 |date=2018-01-18 |publisher=蘋果日報}}</ref><ref>{{Cite web |url=https://hk.news.appledaily.com/local/daily/article/20180405/20352644 |title=嚴迅奇提貨櫃碼頭「天空之城」方案 |date=2018-04-05 |publisher=蘋果日報}}</ref>他還在[[facebook|facebook]]開設「碼頭上蓋的天空之城」專頁，認為方案是切實可行的。<ref>{{Cite web |url=https://www.facebook.com/castleintheskyhk/ |title=碼頭上蓋的天空之城facebook專頁 |accessdate=2018-10-14}}</ref><ref>{{Cite news |url=https://www.hk01.com/article/207228/ |title=土地大辯論 挺貨櫃碼頭上蓋建屋可行　周明權：填海「封盤」 |date=2018-07-05 |publisher=香港01}}</ref>

逝世前為[[海港企業有限公司|海港企業有限公司]]、[[利基控股有限公司|利基控股有限公司]]、[[其士國際集團有限公司|其士國際集團有限公司]]和[[路勁基建有限公司|路勁基建有限公司]]獨立非執行董事。

2018年10月13日，周明權逝世，享年77歲<ref name="mingpao1014">{{cite news |title=周梁淑怡丈夫周明權逝世 終年77歲 曾倡貨櫃碼頭上蓋建屋 |url=https://news.mingpao.com/ins/instantnews/web_tc/article/20181014/s00001/1539486080033 |accessdate=2018-10-14 |agency=明報}}</ref>。

== 家庭 ==
周明權配偶是前[[自由黨_(香港)|自由黨]]主席[[周梁淑怡|周梁淑怡]]，育有兩名女兒[[周智美|周智美]]及[[周智琪|周智琪]]，皆已婚。<ref name="明報新聞網">{{cite news |language = zh-hk |author = |coauthors = |url = http://ol.mingpao.com/cfm/star5.cfm?File=20070511/saa01/mba1.txt|title = 周梁淑怡 母女嗌交情更深 |work = 明報新聞網海外版(溫哥華) |publisher = |pages = C03 |date = 2007-05-11 |accessdate = 2012-12-04 }}</ref>
