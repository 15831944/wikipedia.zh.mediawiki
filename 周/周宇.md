'''周宇'''（{{bd|？|||}}）女，[[湖北省|湖北省]][[武汉市|武汉市]]人，[[中华人民共和国外交部|中华人民共和国外交部]]官员，[[中共十九大|中共十九大]]代表。

==生平==
周宇1995年自[[武汉外国语学校|武汉外国语学校]]毕业，保送到[[北京外国语大学|北京外国语大学]][[英文|英文]]专业学习。1999年毕业，同年通过中华人民共和国外交部考试，成为中华人民共和国外交部译员。2002年左右，[[APEC|APEC]]议题火热，外交部和[[南开大学|南开大学]]合作开办了在职“世界经济研究生班”，周宇报名参加，取得了南开大学经济学硕士学位。后来她公派[[美国|美国]][[约翰霍普金斯大学|约翰霍普金斯大学]]攻读硕士学位，选择了国际金融方向课程，毕业时获评为全美优秀毕业生。2007年周宇学成归国。[[2008年金融危机|2008年国际金融危机]]爆发，[[G20|G20]]升级为峰会，中美间应对金融危机的合作交流不断增多，周宇成为中国许多经济金融外交活动优先选择的翻译人选。后来，周宇报考[[中国社会科学院|中国社会科学院]]金融学专业的博士，以专业全国第一的总分获得录取，研究方向是全球经济治理。读博期间，她升任外交部翻译司英文处处长。2015年，周宇毕业取得金融学博士学位。她是外交部翻译司英文处参赞兼处长<ref name=zggw>{{Cite web |url =http://www.zgg.org.cn/zhtbd_5658/dbfcl/201710/t20171019_669494.html  |title = 外交翻译的飒爽英姿——记外交部翻译司英文处参赞兼处长周宇 |author =  |date = 2017-10-19 |publisher =紫光阁网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url = http://news.jxnews.com.cn/system/2015/10/04/014333401.shtml|title = 习近平访美翻译被认出：中学成绩一直班上前三|author = |date = 2015-10-04|publisher = 中国江西网|language = |accessdate = |quote = }}{{Dead link|date=2019年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[中共十八大|中共十八大]]至[[中共十九大|中共十九大]]之间，周宇完成了[[习近平|习近平]]访问[[美国|美国]]、[[英国|英国]]、[[联合国|联合国]]总部、[[欧盟|欧盟]]总部等将近40个国家和国际组织，出席[[G20峰会|G20峰会]]、[[亚太经合组织峰会|亚太经合组织领导人非正式会议]]等20多起国际会议的翻译任务。2013年至2017年，她承担了习近平接待来访的各国元首和重要外宾160余场会谈会见的翻译工作。周宇率领外交部翻译司英文处与翻译司其他人员共同完成了2014年[[上海|上海]][[亚信峰会|亚信峰会]]、[[北京|北京]]亚太经合组织领导人非正式会议、2015年[[博鳌亚洲论坛|博鳌亚洲论坛]]年会、2016年[[G20杭州峰会|G20杭州峰会]]、2017年[[厦门|厦门]][[金砖国家|金砖国家]]领导人会晤等全部中国重要主场外交活动的翻译工作，同时她还率领外交部翻译司英文处完成了应对[[南海仲裁案|南海仲裁案]]、[[联合国安理会|联合国安理会]]改革等热点问题的翻译工作<ref name=zggw/>。

2017年她当选[[中共十九大|中共十九大]]代表，2017年10月24日在[[人民大会堂|人民大会堂]]一楼中央大厅北侧举办的中共十九大“党代表通道”第三场采访活动中作为中共十九大代表回答了中外记者提问<ref>{{Cite web |url = http://cpc.people.com.cn/19th/n1/2017/1024/c414536-29606375.html |title = 周宇代表讲述习主席外交故事：中国特色大国外交走进新时代 |author =  |date = 2017-10-24 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。2017年4月她担任习近平与[[美国总统|美国总统]][[唐纳德·特朗普|特朗普]][[海湖庄园|海湖庄园]]会晤时的翻译。2017年11月她担任特朗普总统来北京进行国事访问同习近平会晤时的翻译<ref>{{Cite web |url =  http://news.ifeng.com/a/20171109/53112886_0.shtml|title =这位十九大代表，是习特会的翻译  |author =  |date =2017-11-09  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。
