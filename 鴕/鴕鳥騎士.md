'''《鴕鳥騎士》'''（英語：''Joust''）是由[[威廉斯電子|威廉斯電子]]開發並發行於1982年的街機遊戲。雖然不是第一款具有雙人[[合作游戏模式|合作遊戲]]的遊戲，但《鴕鳥騎士》比其前版遊戲更為成功，並推廣了這一概念。玩家使用按鈕和[[操纵杆|操縱桿]]來控制騎著鴕鳥的騎士。目標是通過擊敗騎乘[[禿鷲|禿鷹]]的敵方騎士團隊來進行各級攻擊。

約翰·紐康姆領導的開發團隊的目標是創造一個合作雙人遊戲的飛行遊戲，但是想要避免當時很受歡迎的太空主題，於是製作了這遊戲。

這個遊戲在商場和評論界廣受好評，他們稱讚遊戲玩法，其中的機制影響了其他開發人員的頭銜。《鴕鳥騎士》在四年後續集續作，並被移植到眾多家庭和便攜式平台上。<br>街機續集：[[鸵鸟骑士2：适者生存|鸵鸟骑士2：适者生存]]於1986年發布，它具有與垂直屏幕上新元素類似的遊戲玩法。<ref>{{cite video game| title= {{tsl|en|Midway Arcade Treasures|Midway Arcade Treasures}}| developer= {{tsl|en|Backbone Entertainment|Backbone Entertainment}}| publisher= [[Midway_Games|Midway Games]]| date= 2003-11-18| platform= [[PlayStation_2|PlayStation 2]] | level= 內幕的故事'''鴕鳥騎士'''}}</ref>

== 遊戲性 ==
從第三人角度來看，玩家控制著騎著[[鸵鸟|鴕鳥]]或[[鹳属|鸛]]的黃色[[骑士|騎士]]。使用雙向定向操縱桿和拍動鴕鳥翅膀的按鈕，玩家在浮動的岩石平台和[[熔岩湖|熔岩池]]之上飛行，當操縱屏幕到任何一邊時，玩家將繼續從另一側重新出現路徑。玩家反复按下按鈕的速度會導致鴕鳥向上飛，盤旋或緩慢下降。

玩家導航騎士與敵人相撞。如果主角的騎槍長於敵人，那麼反派就會被擊敗，反之亦然。高度相等的碰撞導致兩名騎士相互反彈。被擊敗的敵人會變成一個落在屏幕底部的蛋，玩家可以通過這個蛋獲得積分。坐在平台上足夠長的雞蛋會孵化成新的騎士; 如果玩家不接他，他將獲得新的坐騎，必須再次擊敗。

其目標是打敗騎乘蝙蝠的敵人騎士團隊，這些蝙蝠在每個級別都被稱為“波”。一旦完成一波，隨後，更具挑戰性的波開始。該遊戲包含三種類型的敵人騎士 - 壞蛋（Bounder），獵人（Hunter）和暗黒騎士（Shadow Lord），擊敗他們各自能獲得不同的點數。一隻幾乎堅不可摧的[[翼龍目|翼手龍]]在預定的時間框架後出現，以追捕英雄，作為完成等級的動機。

控制鸛上的藍色騎士的第二個玩家可以加入遊戲。這兩名玩家既可以[[合作|合作]]完成每波，也可以在擊敗敵人的同時互相攻擊。<ref>{{cite journal| journal = [[电子类游戏|电子类游戏]]| publisher = Reese Communications| title = 投幣式街機教室：一些要記住的騎士| pages = 114–115| volume = 1| issue = 17|date=1983年7月}}</ref><ref name="Fever">{{cite book| title = 街機熱潮(街機書籍): 電視遊戲黃金時代的粉絲指南| pages = 94–95| publisher = {{tsl|en|Running Press|Running Press}}  |date=2001年8月| isbn = 0-7624-0937-1}}</ref>

== 開發 ==
''鴕鳥騎士'' 由[[WMS_Industries|威廉斯电子]]開發, 约翰·纽康姆擔任首席設計師。 程序員Bill Pfutzenrueter和藝術家Janice Woldenberg-Miller和Python Anghelo協助他。Tim Murphy和John Kotlarik負責音樂設計。<ref name="RG-63">{{cite journal| journal = {{tsl|en|Retro Gamer|Retro Gamer}}| publisher = {{tsl|en|Imagine Publishing|Imagine Publishing}}| title = 一起來[[马上长矛比武|長槍比武大賽]]| issue = 63| pages = 36–41}}</ref> 遊戲在19英寸彩色[[陰極射線管|CRT]]顯示器上放大了[[單聲道|單聲道]]聲音和[[位图|光柵圖形]]。<ref name="Arcade-Museum">{{cite web| url = http://www.arcade-museum.com/game_detail.php?game_id=8243| title = 鴕鳥騎士 – 威廉姆斯的電子遊戲| publisher = 國際街機博物館 | date = 2010-07-13}}</ref> 和其他[[WMS_Industries|威廉斯]]街機遊戲一樣，''鴕鳥騎士''也是用[[組合語言|組合語言]]編程的。<ref name="Halcyon">{{cite book| title = {{tsl|en|Halcyon Days (book)|Halcyon Days}}: 與經典電腦遊戲和遊戲程式設計師訪談| editor = James Hague| year = 1997| chapter = Eugene Jarvis| publisher = Dadgum Games| chapterurl = http://www.dadgum.com/halcyon/BOOK/JARVIS.HTM}}</ref> 一包三節[[AA電池|AA電池]]提供電力來保存遊戲設置和高分數，當機器從電源插座拔下時。<ref name="PriceGuide-1">{{cite book| title = 經典遊戲官方價格指南| pages = 337–338| chapter = 街機經典| publisher = [[Random_House|Random House]]| isbn = 0-375-72038-3| year = 2004}}</ref><ref name="PriceGuide-2">{{cite book| title = 經典遊戲官方價格指南 2| pages = 433–437| chapter =經典街機遊戲設置，恢復和修復| publisher = [[Random_House|Random House]]| isbn = 0-375-72038-3| year = 2004}}</ref> Anghelo的櫥櫃藝術品被刻在木框上。<ref name="RG-63"/><ref name="PriceGuide-1"/> Anghelo是為宣傳設計了藝術作品;<ref name="RG-63"/> 一個這樣的[[遊戲選單|遊戲選單]]以[[近代英语|近代英語]]為特色，這也被納入了遊戲的屏幕指示和遊戲結束消息。<ref name="Fever"/>

== 評價與影響 ==
{{Video game reviews
| Allgame = {{Rating|5|5}}<ref name="allgamereview">{{cite web |url = http://www.allgame.com/game.php?id=420&tab=review|title = 鴕鳥騎士評論 |publisher = [[AllGame|AllGame]] |date = 2017年8月6日 |archiveurl=https://web.archive.org/web/20141114221702/http://www.allgame.com/game.php?id=420&tab=review|archivedate=2014年11月14日}}</ref>

因為遊戲控制與前版不同，威廉姆斯擔心遊戲不成功。 雖然商場猶豫以同樣的理由購買遊戲，但鴕鳥騎士賣得很好。威廉斯最終賣出26,000個，和在1983年描述為“非常受歡迎”的[[電子遊戲|電子遊戲]]。<ref name="eg198306">{{cite news | url=https://archive.org/stream/electronic-games-magazine-1983-06/Electronic_Games_Issue_16_Vol_02_04_1983_Jun#page/n45/mode/2up | title= 夢幻遊戲的玩家指南 | work=電子遊戲 | date=1983年6月 | page=47}}</ref> 後來發布了一個雞尾酒桌版，由Leo Ludzia設計。它不同於其他雞尾酒遊戲，因為它具有並排座位而不是對立的兩側。 這種設置使Williams能夠使用與直立式機櫃相同的[[ROM|ROM]]芯片。<ref name="RG-63"/> 雞尾酒版是一款難得的，受歡迎的遊戲。<ref name="PriceGuide-1"/><ref name="PriceGuide-3">{{cite book| title = 經典遊戲官方價格指南 3 | page = 385| chapter = 街機經典| publisher = [[Random_House|Random House]]| isbn = 0-375-72038-3| year = 2004}}</ref> 製造了250至500個。<ref name="PriceGuide-1"/>

1996年，{{tsl|en|Next Generation (magazine)|下一代}}（雜誌）在其「百年有史以來的100佳遊戲」中將街機版本列為83號，稱其為「三種成分經常做出經典的完美例子：原創概念，古怪設計，以及 - 以上所有可玩性。只有三種控制（左，右和襟翼），鴕鳥騎士創造了一個優雅戰鬥的整個世界。」<ref>{{cite magazine|title=有史以來最好的100個遊戲 |magazine={{tsl|en|Next Generation (magazine)|下一代}}|issue=第21期 |publisher={{tsl|en|Imagine Media|Imagine Media}} |date=1996年9月|page=43}}</ref> 作者史蒂夫肯特認為鴕鳥騎士是當時比較難忘的遊戲之一。<ref name="Ultimate">{{cite book| title = 遊戲的終極歷史| page = 177| chapter = 黃金時代（第2部分：1981 - 1983年）| publisher = {{tsl|en|Three Rivers Press|Three Rivers Press}}| isbn = 0-7615-3643-4| year = 2001}}</ref> 作者大衛埃利斯同意，並表示遊戲直到今天仍然是很好玩的。<ref name="PriceGuide-1"/> 2008年，[[吉尼斯世界纪录大全|吉尼斯世界紀錄]]將其列為第六十九屆街機遊戲，其技術，創意和文化影響。<ref name="GWR08">{{cite book| editor= Craig Glenday| title= 200年8遊戲版拿吉尼斯世界紀錄| series= [[吉尼斯世界纪录大全|吉尼斯世界纪录大全]]| date= 2008-03-11| publisher= 吉尼斯| isbn= 978-1-904994-21-3| page= 231| chapter= 前100名街機遊戲：前100-51}}</ref> 作者稱鴕鳥騎士的「遊戲插圖」為異國風情，並稱讚這部動畫栩栩如生。<ref>{{cite journal| journal = 鴕鳥騎士遊戲插圖| publisher = Ion International| title = 留意: 在遊戲[[马上长矛比武|長槍比武大賽]]| author = 遊戲插圖員工| page = 61| issue = 3|date=1982年12月}}</ref>Antic也叫Atari 8位版本被稱為「獨特，令人上癮的街機遊戲」，與原版「幾乎相同」。該雜誌的結論是， 鴕鳥騎士是「自從Atari星際攻略以來最好的」。<ref name="antic198403">{{Cite magazine  |date=1984年3月 |title=產品評論 |url=http://www.atarimagazines.com/v2n12/productreviews.html |magazine=Antic}}</ref>

凱文·鮑恩在[[GameSpy|GameSpy]]的經典遊戲寫道，儘管他稱之為「令人難以置信的愚蠢」的概念，但鴕鳥騎士是一款吸引人的遊戲，擁有良好的控制和競技遊戲。凱文·鮑恩進一步評論說，多人遊戲方面在當時將游戲與其他人區分開來。他形容它是「第一個真正有趣的多人遊戲」，也是視頻遊戲[[死亡竞赛|死亡競賽]]的先驅。<ref name="ClassicGaming">{{cite web|url=http://classicgaming.gamespy.com/View.php?view=GameMuseum.Detail&id=117 |title=本週熱門遊戲 : 鴕鳥騎士 |publisher=Classic Gaming |date=2010-07-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20090821104444/http://classicgaming.gamespy.com/View.php?view=GameMuseum.Detail&id=117 |archivedate=2009-08-21 |df= }}</ref>

復古遊戲玩家作家邁克·貝凡對遊戲中的[[物理引擎|物理引擎]]稱讚，稱他們「 精彩地體會到」，並將鴕鳥騎士描述為威廉姆斯最出色和最受歡迎的遊戲之一。<ref name="RG-63"/> [[电脑与电子游戏|電腦與電子遊戲]]稱為遊戲「神秘和奇妙」。<ref>{{cite journal| 	
journal = [[電腦遊戲|電腦遊戲]]| publisher = 以前稱為{{tsl|en|Ascential|EMAP}}| date = 1983年2月| title = 街機行動| page = 30}}</ref> 作者約翰塞勒斯讚揚了雙人遊戲的競爭力，並將游戲的吸引力歸功於雙人合作系統。<ref name="Fever"/> 2004年，埃利斯將鴕鳥騎士描述為當時電子遊戲行業缺乏創新風險的一個例子。<ref name="PriceGuide-1"/>

回想起來，约翰·纽康姆讚揚威廉姆斯的管理層對他和比賽冒險。<ref name="RG-63"/> 這款遊戲也獲得了行業專業人士的讚譽。GearWorks Games的Jeff Peters稱讚了遊戲性，並將其描述為獨特而直觀。融合學習系統公司的Jeff Johannigman對雙人合作系統表示讚賞，[[微软|微軟]]的 Kim Pallister 喜歡多人遊戲。<ref name="Coin-OpFavs">{{cite web| url = http://www.gamasutra.com/features/20050805/hong_01.shtml| title = 本週問題回复：投幣式遊戲？ | publisher = [[Gamasutra|Gamasutra]]| date = 2005-08-05| accessdate = 2009-05-12}}</ref>
