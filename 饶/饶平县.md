'''饶平县'''，得名于“'''饶永不瘠，平永不乱'''”，位于[[中国|中国]][[广东省|广东省]]最东端，与[[福建省|福建省]]接壤。地理范围约在[[东经|东经]]116°35′～117°11′、[[北纬|北纬]]23°28′～24°14′之间。[[北回归线|北回归线]]在县境南部穿过。饶平县是[[潮州市|潮州市]]下辖的一个县，县城为[[黄冈镇|黄冈镇]]。饶平西南与[[湘桥区|湘桥区]]、[[汕头市|汕头市]][[澄海区|澄海区]]接壤、西邻[[潮安区|潮安区]]、北临[[梅州市|梅州市]][[大埔县|大埔县]]、东北一隅与[[福建省|福建省]][[平和县|平和县]]相接，东邻福建省[[诏安县|诏安县]]，南临[[南海|南海]]，与[[汕头市|汕头市]][[南澳县|南澳县]]隔海相望。是广东重要的陶瓷生产基地，也是岭头单丛茶和铁皮石斛的重要产地。素有“广东东大门”之称。

== 历史 ==
*[[商朝|商]]末[[西周|西周]]时期，饶平境内已有[[部落|部落]]聚居，创造“浮滨型文化”。
*[[春秋时期|春秋]][[战国|战国]]时期，饶平为[[百越|百越]]地。
*[[中国朝代|秦]][[汉朝|汉]]时期属[[中国|南海郡]]揭阳县。
*[[晋朝|晋]]时属[[义安郡|义安郡]][[海阳县_(东晋)|海阳县]]。
*[[明朝|明]]成化十二年十月甲申（1476年10月31日）从海阳县析光德的弦歌（清为避康熙帝名讳改称元歌）、灤州、清远、太平乡的宣化、信宁及怀德乡的隆眼城、苏湾、秋溪等三乡八都置饶平县，人口92921人，隶属[[潮州府|潮州府]]。嘉靖五年，割灤州、清远二都置[[大埔县|大埔县]]；四十四年析苏湾都置[[澄海县|澄海县]]，割秋溪都归还海阳县。万历四年以信宁都海岛置南澳镇，与漳州诏安县共管，深隆二澳属饶平。
*1949年10月25日，县属隆都区（今澄海[[隆都镇|隆都镇]][[莲华镇|莲华镇]]大部）划归澄海县管辖。1951年8月县人民政府在黄冈镇设立办事处，1953年1月饶平县治由三饶镇迁此，黄冈镇成为了饶平县县城。1958年11月，县属凤凰区（今潮安凤凰镇）划归潮安县管辖。县境管辖范围保持至今。
*饶平明清时期均属潮州府，民国时期属潮汕地区行政公署管辖。
*建国后至1991年隶属[[汕头市|汕头市]]管辖。
*1991年12月至今归[[潮州市|潮州市]]管辖。

===事件===
*[[黄冈起义|黄冈起义]]：[[清|清]][[光绪|光绪]]三十三年(1907年)四月十一日，[[孙中山|孙中山]]率领的黄冈丁未革命爆发。由于起义军寡不敌众，粮械缺乏，战败，惨毙多命。此次起义是[[孙中山|孙中山]]领导的十次起义中之第三次，该年为农历[[丁未|丁未]]年，因而称之为“黄冈丁未革命”或'''[[黄冈起义|黄冈起义]]'''。

== 地理 ==
饶平县位于[[潮汕|潮汕]]平原边端，是广东山区县之一。其地形依山傍海，地势北高南低，有岛屿13个，其中海山岛最大（现已与陆地相连，实为半岛），面积46.9平方公里；有海拔超1000米的山峰7座，其中西岩山最高，海拔1256米。[[黄冈河|黄冈河]]是饶平县内最大的河流，全长87.2公里，途经12个乡镇，注入[[南海|南海]]。饶平县有柘林、高沙、大埕三个较大海湾，其中柘林湾最大，面积68平方公里。饶平[[海岸线|海岸线]]长136公里，最大的港口是三百门[[港口|海港]]。

===气候===
饶平县[[气候|气候]]属于海洋副热带[[季风|季风]]气候区，年[[降雨量|降雨量]]充沛，为1475．9毫米，年平均[[气温|气温]]21．4°C，农作物一年三熟。


== 资源 ==
饶平县有较为丰富的海产河山地资源。其中浅海滩共57万余亩，盛产各类海[[鱼|鱼]]、[[对虾|对虾]]、[[牡蛎|牡蛎]]等。山地[[野生动物|野生动物]]则有[[眼镜蛇|眼镜蛇]]、[[金环蛇|金环蛇]]、[[蝮蛇|腹蛇]]、[[穿山甲|穿山甲]]、[[石蛤|石蛤]]等各种珍禽。饶平县盛产[[陶土|陶土]]。矿产资源有[[花冈岩|花冈岩]]、[[钾长石|钾长石]]、[[高岭土|高岭土]]、[[耐火土|耐火土]]、[[铜|铜]]、[[铁|铁]]、[[钨|钨]]、[[锡|锡]]筹。饶平县也是凤凰系列产茶叶的产地。

*林场： [[韩江林场|韩江林场]]。

== 交通 ==

=== 公路 ===
*{{中国高速公路名|G15}}汕汾高速公路（[[沈海高速|沈海高速]]广东汕头——汾水关段）从县境南部联饶镇、黄冈镇、高堂镇、钱东镇经过，全长24.96公里，在联饶镇山门境内设有粤闽收费站和黄冈收费站，在钱东镇境内设有钱东收费站。
*{{中国高速公路名|G15W3}}（又称宁莞高速）潮漳高速公路（甬莞高速广东潮州——福建漳州段）自东向西依次经过县境东山镇、浮山镇、浮滨镇、樟溪镇，在东山镇境内设有粤闽饶平主线收费站和东山收费站，在浮山镇境内设有浮山收费站；在樟溪镇境内设有樟溪服务区。
*{{中国国道名|324}}（又称汕汾公路）从东部诏安县汾水关进入县境，全长25.3公里。经过黄冈镇、钱东镇，向西南在九溪桥出饶平界，进入[[澄海区|澄海区]]盐鸿镇。1996-2004年实施改线工程，新建一条东起黄冈镇碧春村，向南经过东风埭、林厝埭、青山埭至钱东镇灰寨塔护村接原国道的公路，使国道324线绕过县城。而原线则编入县级公路网（即{{中国县道名|087}}）。道路在2015-2017进行大修，全线改为双向四车道一级公路，采用沥青路面，并沿途设置5个交通信号灯路口，改善了路段通行条件和秩序。
*{{中国省道名|222|P=广东}}坑海公路。从北部韩江林场进入县境，向南经三饶镇（三饶城区段部分与{{中国省道名|334|P=广东}}共线）、汤溪镇、浮滨镇、浮山镇、新圩镇、联饶镇、黄冈镇、汫洲镇、海山镇，终于海山镇隆西居委会。饶平境内长度为85.5公里，是连接饶平南北的重要通道。道路在2014-2015年大修，从浮山镇荔林村至黄冈镇上林村约23公里的路段被改造成双向4车道的二级公路，方便了饶平南北交通。
*{{中国省道名|334|P=广东}}柏丰公路。起自北部上饶镇与[[福建省|福建省]][[平和县|平和县]]交界的柏嵩关，经过上饶镇、饶洋镇、新丰镇、三饶镇、新塘镇。于新塘镇外宫村进入潮安区境内。2011年对上饶镇区段进行改线，2013年完工，与G324改线类同，该工程使S334线绕开上饶镇城区。
[[File:饶平县交通概况图.png|thumb]]
*{{中国县道名|081}}碧铁公路。起点位于饶平县黄冈镇碧岗村接{{中国国道名|324}}，过仙春村约1.5公里后与进港大道共线约2.8公里，在所城镇北山村脱离进港公路转向东南，经过所城镇、大埕镇，在闽粤界铁炉港与[[诏安县|诏安]]{{中国县道名|553}}相接。
*{{中国县道名|086}}铁汫公路。1996年建设，是饶平县城连接潮州市区的重要通道。起点位于汫洲镇潮州港区，与{{中国省道名|222|P=广东}}交汇后继续往西北延伸，经过汫洲镇、钱东镇后向西出饶平界，在[[湘桥区|湘桥区]]铁铺镇与{{中国省道名|335|P=广东}}相接。该公路原本计划是为了连接潮州市区与位于汫洲小红山的潮州港，作为疏港通道，但由于潮州港工程尚未完工，XO86未能实现预期的疏港货运功能，但其仍在发挥着重要的交通作用。道路于2013-2015年大修。
*{{中国县道名|084}}钱坪公路。是连接钱东与浮滨的重要公路。起于钱东镇区与{{中国国道名|324}}相接，向北穿过樟溪镇，于浮滨镇区与{{中国省道名|222|P=广东}}交汇后转向西南，经过坪溪后在饶平界与[[潮安县|潮安]]{{中国县道名|074}}相连。
*{{中国县道名|087}}白塔公路。即原{{中国国道名|324}}饶平段的一部分，全长16.22公里，1996年建设新国道后，原线就改为{{中国县道名|087}}。起自黄冈镇碧春村，终于钱东镇灰寨塔护村。
*潮州港进港大道。又称进港公路，2011年建设。起于黄冈碧岗村接{{中国国道名|324}}相接，向南经过所城镇，终于柘林镇潮州港开发区。公路全段按照高速公路标准建设，是一条“准高速公路”。
*饶平站进站大道。又称进站大道，2013年建设。起于钱东紫云村接{{中国县道名|086}}相接，向北经过高堂镇，终于[[饶平站|饶平站]]站前广场。公路全长3.63公里，拥有双向4车道和水泥混凝土路面，是目前进入饶平火车站唯一的公路。未来计划将道路往北延伸约5公里接驳{{中国省道名|222|P=广东}}，以方便交通。
其它公路还有{{中国省道名|221|P=广东}}、{{中国县道名|001}}、{{中国县道名|079}}、{{中国县道名|080}}、、{{中国县道名|082}}、{{中国县道名|083}}、{{中国县道名|085}}等公路。

=== 铁路 ===
[[厦深铁路|厦深铁路]]从东部诏安县入县境，经过联饶镇、高堂镇、钱东镇，向南入潮安区。于高堂镇设[[饶平站|饶平站]]，是饶平第一个火车站。

未来计划建设潮州港疏港铁路。

=== 水运 ===
[[潮州港|潮州港]]是饶平县最大的港口，分为三部分，即海山镇三百门码头，汫洲镇红山码头，柘林镇柘林港。前两个都不是货运码头，而是渔港，货运码头的计划现在仍在规划中。唯一的的货运码头位于柘林镇，称为亚太码头，由亚太集团主营。但现在仍在建设中。

==文化==

===方言===
饶平县的大部分地区通行[[潮州话|潮州话]]，不过各地口音有一定差别，县城与其他乡镇的口音就有较大差别。新丰镇溁溪村以北的地区通行[[客家语|客家语]][[饶平客家语|饶平话]]。

===小吃===
[[烧卖|肖米]]是饶平的传统特色小吃，在过去常被作为配桌点心。饶平的肖米起源于北方的“[[烧卖|烧卖]]”，但与烧卖并非完全相同。沿袭烧卖的外形，主要的馅料为萝卜（街上出卖的主要原料是豆干构成）、瘦肉，并在开口处放置一形状为方型的卤肉。吃的时候配以沉醋，味道十分鲜美可口。

== 旅游 ==
*'''三饶镇“道韵楼”'''
[[File:道韵楼全景.jpg|thumb]]
 
道韵楼位于饶平县三饶镇南联村，[[明朝|明]]万历十五年(1587年)建，是全国最大的八角形[[客家围楼|客家围楼]]。土楼鼎盛时可容纳600多[[客家|客家]]人居住，现尚有100多人居住在土楼中。土楼墙厚1.6米、墙高超过11米。土楼的特色之处在于整个土楼处处与八卦相关，所有建筑的数目都是八的倍数，如水井32眼，天窗16个，房72间，梯112架。土楼中厅有左右两口水井，若[[八卦|八卦]]两仪。土楼具有八防的功能：防[[水|水]]、防[[火|火]]、防兽、防震、防贼、防旱等。土楼的瓦砖是经土布包好，再经火烧而制成，现尚可分辨出瓦棱上的布纹。土楼全部使用竹钉，历经多年，这些竹钉也无腐朽的迹象。据说在清朝顺治年间，土楼曾经被官府收编的土匪包围三个月而攻不破，土楼里的数百居民利用储粮和井水以供自己饮食，利用土楼上面的内沟灌水以守住外面官兵的火攻，因而逃过劫难。土楼前后有三层环形房屋相套，最外环有三层楼高。土楼历经数百年仍然十分坚固。在1918年的大[[地震|地震]]中，附近的房屋大多倒塌，而土楼里只有几间房屋左右倾斜，反而支撑了整个土楼不倒。文革期间，该土楼曾受到部分拆除，这就是造成今天该土楼外围楼层高低不匀的原因。据居住在土楼的当地老人介绍，土楼修建成八卦形是机缘巧合，当初设计的是圆形的围楼，不料三建三倒，后来才建成全国少见的大型八卦形围楼。 

*'''海山镇“辟龙”'''
“辟龙”位于饶平县海山镇，属[[贝壳冲积岩|贝壳冲积岩]]，东西长4500米，宽1150米，后2米至10米，总面积为2875公顷。因裸露在地面部分横卧于海滩，形似长龙，俗称“辟龙”。经专家考证，该[[海滩岩田|海滩岩田]]是由[[文石|文石]]、[[高镁方解石|高镁方解石]]、[[有孔介虫|有孔介虫]]、海生[[贝壳|贝壳]]、微体[[古生物|古生物]]和细砂粒在特定的地址条件下长期堆积、[[胶结|胶结]]而成，约有五千年历史。现为[[广东省|广东省]]第三个[[地质遗迹保护区|地质遗迹保护区]]。　 

*'''柘林镇“镇风塔”'''
“镇风塔”位于饶平县柘林镇，地处风吹岭峡谷口。镇风塔始建于元代，至今已有600多年历史。塔高22米，基围16米，七层八面。塔尖为制作精美的千斤石葫芦，塔身周围是石板围成的环形面屏。该塔现为广东省重点文物保护单位。

*'''黄冈镇“石壁庵”'''
“石壁庵”位于离饶平县黄冈镇约二公里处的栖云山，因此山多为石头山而得名。山上有“涑玉泉”、“古雷音寺”、“纳海楼”、“慈悲亭”等景观。其中涑玉泉成于山上天然岩洞，如卧龙吐水，长年清泉不竭，为县级文物保护单位<ref>{{cite web|url=http://www.czmc.gov.cn/wwkb/xjww_list.asp?itemid=349&id=262|title=《潮州市(县)级文物保护单位：涑玉泉》|accessdate=2009-10-20|work=|publisher=潮州市名城办}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

*'''樟溪镇“青岚冰臼”'''
饶平县樟溪镇青岚溪谷中分布有两千多个的形态各异的[[冰臼|冰臼]]，是距今约二三百万年前[[冰川运动|冰川运动]]时留下的遗迹。冰臼群中最大的一个圆形冰臼口宽六七米，深二三米。冰臼群处于[[北纬|北纬]]23"12'，海拔不到100米的热带亚热带地区，这对[[广东|广东]]地区可能在数百万年前曾是冰天雪地的严寒地带说法有重要意义。

*'''樟溪镇“马山湖”'''
马山湖湿地位于樟溪镇径北村南部马山湖水库旁，原本是黄冈河支流樟溪流入马山湖水库处的一个冲积平原，因水库水位下降导致湿地变成草原，后被网友发现而出名，被称是继架桥潭、石庵、草岚武之后的又一天然风景区。不过近来因游客增加导致马山湖生态环境略趋恶化，不少网友呼吁保护马山湖。<ref>[http://cz.southcn.com/c/2014-12/11/content_114107016.htm 马山湖网络蹿红引大批游人 景区环境堪忧]</ref>。

*'''柘林镇“白鹭天堂”''' 
饶平县柘林镇西澳岛上因白鹭成群，而被称为“白鹭天堂”。西澳的[[白鹭|白鹭]]为[[黄嘴白鹭|黄嘴白鹭]]，是[[国家二级保护动物|国家二级保护动物]]。

*'''高堂镇“金光寺”''' 
　　金光寺始建于明代，创寺长老法名"立禅"大师。金光寺建筑规模宏大，殿宇巍峨 富丽，明清两代极盛时有大小庙宇五、六十间，完整的藏经一套共八千余部（册），僧众二百多人，朝山礼佛的人络绎不绝。而且，寺内绝大多数塑像和佛具都是用 纯铜铸造的。《永昌府志》载："寺有明代铜佛二尊，殿宇皇壮，邑中名胜，推为首指"。明末著名的地理学家徐霞客曾游此寺，他在《滇游日记》中写道："寺规模宏敞，正殿亦南向，八角层甍，高十余丈，地盘数亩"。 金光寺以其幽雅的环境，雄伟的建筑，繁芜的佛典，完善的佛礼吸引了千千万万游客僧众。但由于年久失修，金光寺日渐衰落，七十年代前，仅存留大殿一间，偏殿一间两层，楼上藏经。另有厢房十三间，只有原建筑的五分之二。存经书四十二部，五百二十卷；而且寺内大佛像完全被损坏。直到八十年代后，才正式修复，现主要建筑均已修复，且新建一座雕梁画栋的门楼，上书"滇西名胜"四个鎏金大字。现有担当和尚近代名人李根源等题书。大殿里供奉的佛像，
造形精湛。殿外，人工栽培的木莲花、茶花、含笑、白木瓜等各种花卉竞放于高大的青松翠柏之间，把寺院装点得更加明丽清亮。 

金光寺的兴盛，还取决于它得天独厚的地理位置。金光寺距离古西南丝绸之路一博南古道不远，便利的交通，不仅带来了繁荣，也促进了文化的发展。

== 古迹 ==
[[File:PICT0220(1).JPG|thumb]]

*'''浮滨塔仔金山文物'''
1974年，在饶平县浮滨塔仔金山发现了古穴墓群，墓中发现了大量的[[石器|石器]]与[[陶器|陶器]]的陪葬品。其中有戈、矛、斧、凿、砾石[[石器|石器]]，尊、壶、豆、杯、盆、盂、罐等陶器。部分陶器有酱色釉。经考古坚定，浮滨发现的[[文物|文物]]时间为[[商朝|商]]代中晚期至[[西周|西周]]前期。该发现对[[中原文化|中原文化]]与当地区别于诸越文化的土著文化有重要影响。
