'''饶毅'''（{{bd|1962年||||CatIdx=R饶}}），[[江西|江西]]人，著名[[生物学家|生物学家]]。[[北京大学|北京大学]]终身讲席教授、北京大学校务委员会副主任。他於1991年在加州大學舊金山分校（UCSF）獲得神經科學博士學位，之後在哈佛大學完成博士後課程。後來他加入密蘇里州華盛頓大學當教師，在那裡工作了10年，並成功申請了美國公民身分{{来源请求}}。

2007年，饒毅回到中國，在北大就職，他還放棄了自己的美國國籍{{来源请求}}。

饒毅是中华人民共和国「[[千人計劃|千人計劃]]」中的一員，为北京大学首位全球公开招聘的院长<ref>[http://www.biotech.org.cn/news/news/show.php?id=55764 饶毅任职北大生命科学学院] {{webarchive|url=https://web.archive.org/web/20141112133645/http://www.biotech.org.cn/news/news/show.php?id=55764 |date=2014-11-12 }}</ref>。美国[[加州大学旧金山分校|加州大学旧金山分校]][[神经科学|神经科学]]博士。曾任美国[[圣路易斯华盛顿大学|圣路易斯华盛顿大学]]、[[西北大学|西北大学]]教授。

现任北京大学IDG/麦戈文脑研究所所长、[[北大清华生命科学联合中心|北大清华生命科学联合中心]]主任、[[未来论坛|未来论坛]]科学委员会成员。

== 教育背景 ==
* 1978-1983 [[江西医学院|江西医学院]]（现[[南昌大學醫學院|南昌大學醫學院]]）学士 
* 1983-1985 [[上海第一医学院|上海第一医学院]]（现[[复旦大学|复旦大学]]上海医学院）硕士
* 1985-1991 [[加州大学旧金山分校|加州大学旧金山分校]]神经科学博士
* 1991-1994 [[哈佛大学|哈佛大学]]生物化学与分子生物学系博士后
* 1994-2004 [[圣路易斯华盛顿大学|圣路易斯华盛顿大学]]教授
* 2004-2007 [[美国西北大学|美国西北大学]]讲席教授

== 学术工作 ==
1997年，饶毅证明脊椎动物的双眼来源于同一前体区域（眼形态发生场）。1999年，饶毅发现神经纤维导向的排斥性导向分子Slit，并发现Slit可以作用于包括白细胞在内的多种细胞。解析了排斥性和吸引性导向蛋白发生作用的细胞内信号转导机理。2004年后饶毅研究行为的分子和细胞机理，

* 1999年，[[蒲慕明|蒲慕明]]、[[吴建屏|吴建屏]]、[[鲁白|鲁白]]、[[梅林|梅林]]、饶毅建立[[中国科学院上海神经科学研究所|中国科学院上海神经科学研究所]]。
* 2000年，饶毅和[[吴家睿|吴家睿]]创立[[BIO2000|BIO2000]]研究生课程。
* 2002年，德国科学家[[施瓦茨|施瓦茨]]和饶毅创建[[中国科学院上海交叉学科研究中心|中国科学院上海交叉学科研究中心]]。
* 2004年，[[王晓东|王晓东]]、[[邓兴旺|邓兴旺]]和饶毅建立[[北京生命科学研究所|北京生命科学研究所]]。
* 2007年，饶毅任[[北京大学生命科学学院|北京大学生命科学学院]]院长。
* 2013年，饶毅卸任北大生科院院长<ref>{{cite web |url = http://news.sciencenet.cn/htmlnews/2013/9/282066.shtm |title = 饶毅卸任北大生科院院长 |publisher = 科学网 |language = zh-hans |date=2013年9月3日}}</ref>。
* 2013年，饶毅任北京大学IDG/麦戈文脑研究所所长。
* 2016年，饶毅任北京大学理学院院长。

== 言论和争议 ==
=== 批判中国科研体制 ===
饶毅公开建言改革中国科技体制，2004年与[[鲁白|鲁白]]、[[邹承鲁|邹承鲁]]联名在《自然》增刊发表文章，建议[[中华人民共和国科技部|中华人民共和国科技部]]只管政策，不管经费<ref>{{cite journal en |title = 中国科技需要的根本转变∶从传统人治到竞争优胜体制——中长期规划将留下优秀遗产、还是错失良机 |author=饶毅、鲁白、邹承鲁 |date=2004年11月18日 |volume=432 |pages=A12-A17|journal=中国之声——Nature杂志增刊 |url = http://zimp.zju.edu.cn/~tcmp/refpdf/RY_RuleMerit.pdf | language=zh-hans}}</ref>。2010年，饶毅与好友[[施一公|施一公]]（[[清华大学生命科学学院|清华大学生命科学学院]]院长）在《[[科学_(期刊)|科学]]》杂志上联合撰文<ref>{{cite journal en |author = Shi, Yigong; Rao, Yi|year=2010|title=China's Research Culture|journal=Science|volume=329|issue=5996|pages=1128|doi=10.1126/science.1196916 |id= PMID 20813923|language=en}}</ref>，批判中国的科研经费分配体制和科研文化问题。他们指出，尽管中国研究经费近年来持续以20%的比例增长，但这种增长没有对中国科学研究应有的强大起到促进作用，现行的科研经费分配体制在某种程度上阻碍了中国创新能力的发展。他们认为，中国必须打破研究经费管理中的各种潜规则，建立健康的研究文化，一个简单但非常重要的起点是所有新的研究经费必须基于学术优劣分配，而不能依赖私人关系。<ref>{{cite web|url=http://edu.gmw.cn/2013-05/02/content_7489777.htm|title=施一公当选美国两院院士 曾与饶毅批中国科研体制|publisher=光明网|date=2013-05-02|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20130506125739/http://edu.gmw.cn/2013-05/02/content_7489777.htm|archivedate=2013-05-06}}</ref>

=== 預測諾貝爾獎得主 ===
2002年，饶毅评论了20项值得[[诺贝尔奖|诺贝尔奖]]的工作，其中9项已经获奖{{来源请求}}{{何時}}。

2008年10月6日，饒毅在個人的[[科學網|科學網]][[博客|博客]]發表〈美妙的生物荧光分子与好奇的生物化学家〉一文，介紹[[下村脩|下村脩]]、[[錢永健|錢永健]]等人的成就。饒毅主張下村脩應該獲得[[諾貝爾獎|諾貝爾獎]]，但很可能遭到諾獎委員會忽略<ref>[http://www.guokr.com/post/452457/ 【搬运】饶毅：美妙的生物荧光分子与好奇的生物化学家 | 小组 | 果壳网 科技有意思]</ref>。兩天後，文章介紹的下村脩、錢永健獲得了[[諾貝爾化學獎|諾貝爾化學獎]]。

=== 落选2011年中科院院士引发争议 ===
在2011年[[中國科學院院士|中國科學院院士]]评选中，[[施一公|施一公]]和饶毅两个学界明星双双落选，引发了舆论对中科院的质疑。2011年12月9日，在新增院士名单出炉后数小时，饶毅便发表了一篇标题为《解剖“逆淘汰”社会现象的一只麻雀》的4000余字博文<ref>{{cite web |title=解剖“逆淘汰”社会现象的一只麻雀（修定）|author=饶毅 |publisher=科学网 |url = http://blog.sciencenet.cn/blog-2237-516724.html |date=2011年12月9日 |language=zh-hans}}</ref>，为施一公的落选鸣不平。饶毅认为，中科院在院士评选中，重视的不是学术水平、年龄、学术年资、在国内科学贡献大小，而是“在有些人面前低头排队时间的长短”。饶毅当时宣布不再参选院士<ref>{{cite web |url = http://epaper.jinghua.cn/html/2013-05/02/content_1986460.htm |title = 施一公当选美双院外籍院士 |publisher = 京华时报 |date = 2013-05-02 |language = zh-hans |deadurl = yes |archiveurl = https://web.archive.org/web/20130506093258/http://epaper.jinghua.cn/html/2013-05/02/content_1986460.htm |archivedate = 2013-05-06 }}</ref>。而后，施一公在2013年院士评选中当选中科院院士<ref>{{cite web |url = http://www.cas.cn/ggfw/tzgg_1/201312/t20131220_4002310.shtml |title = 关于公布2013年中国科学院院士增选和外籍院士选举结果的公告 |publisher=中国科学院 |date=2013-12-19 |language = zh-hans }}</ref><ref>{{cite web |url = http://news.sina.com.cn/c/2013-12-20/020029031820.shtml |title=话题教授施一公当选中科院院士 两院新增104人 |publisher=北京晨报 |date=2013-12-20 |language = zh-hans }}</ref>。
