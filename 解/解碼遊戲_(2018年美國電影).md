{{作品名稱|name=''Under the Silver Lake''|hk= |tw=解碼遊戲|cn=银湖之底}}是一部2018年美國奇幻冒險懸疑喜劇愛情劇情片，由[[大卫·罗伯特·米切尔|大卫·罗伯特·米切尔]]執導兼編劇。該片於2018年5月入圍[[第71屆坎城影展|第71屆坎城影展]]官方競賽單元。

== 劇情簡介 ==
故事發生於現代洛杉磯。山姆被富豪謀殺疑雲吸引著迷，令他相信世界充滿著陰謀論。直至有一天他看見了富豪的情婦莎拉——那個令他鍾情的女人失蹤後，他決定踏上拯救莎拉的冒險之旅。但似乎這一切的浮雲的背後，他隐藏了一個不想看到的真相⋯⋯

== 發行 ==
2016年5月，[[A24_(公司)|A24]]取得美國的電影發行權<ref>{{cite web|url=http://www.thewrap.com/andrew-garfield-still-attached-as-a24-lands-david-robert-mitchells-under-the-silver-lake/|title=Andrew Garfield’s ‘Under the Silver Lake’ Picked Up by A24|website=[[The_Wrap|The Wrap]]|first=Meriah|last=Doty|date=2016-05-12|accessdate=2016-11-02}}</ref>。2018年該片被選在[[坎城影展|坎城影展]]世界首映<ref name="select">{{citeweb|url=http://www.festival-cannes.com/en/infos-communiques/communique/articles/the-2018-official-selection|title=The 2018 Official Selection|work=Cannes|accessdate=2018-04-12}}</ref><ref name="Variety">{{citeweb|url=https://variety.com/2018/film/news/cannes-lineup-includes-new-films-from-spike-lee-jean-luc-godard-1202751300/|title=Cannes Lineup Includes New Films From Spike Lee, Jean-Luc Godard|work=Variety|accessdate=2018-04-12}}</ref>，並於同年6月22日在美國小規模上映。<ref>{{cite web|url=http://www.indiewire.com/2018/03/under-the-silver-lake-release-synopsis-david-robert-mitchell-andrew-garfield-1201934032/|title=‘Under the Silver Lake’ First Details: ‘It Follows’ Director Returns This Summer With Andrew Garfield and Neo-Noir Crime|website=[[Indiewire.com|Indiewire.com]]|first=Zack|last=Sharf|date=2018-03-01|accessdate=2018-03-01}}</ref>另有報導稱臺灣檔期為7月20日，但電影至今仍未在台上映<ref>{{Cite web |url = http://www.chinatimes.com/realtimenews/20180412004391-260404 |title = 「前蜘蛛人」前進坎城 《解碼遊戲》入圍競賽片 |author = 潘鈺楨 |date = 2018-04-12 |publisher = 中國時報 |language = zh-hant |accessdate = 2018-05-19 }}</ref>。A24於2018年6月4日宣布美國延期至同年12月7日上映。<ref>{{cite web|url=http://bloody-disgusting.com/movie/3501899/follows-directors-silver-lake-gets-bumped-back-six-months/|title=MOVIES‘It Follows’ Director’s ‘Under the Silver Lake’ Gets Bumped Back Six Months|website=[[Bloody-Disgusting.com|Bloody-Disgusting.com]]|first=John|last=Squires|date=2018-06-01|accessdate=2018-06-01}}</ref>之後改為2019年4月19日在美國上映。
