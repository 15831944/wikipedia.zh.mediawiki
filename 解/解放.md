「'''解放'''」（{{lang-en|Emancipation}}），指摆脱束缚、压迫的一种状态，包括[[精神|精神]]与[[物质|物质]]两方面。如[[奴隶解放|奴隶解放运动]]，使人获得[[自由|自由]]。

== 精神方面 ==
“精神解放”指摆脱精神的桎梏，如[[政治|政治]]、[[宗教|宗教]]给人的枷锁。

== 物质方面 ==
物质解放，通过改善人民物质生活条件，使人获得更多的自由，常与生产力水平的提高相适应。如通过[[工业革命|工业革命]]之后，随着[[机械化|机械化]]水平的提高，使人摆脱繁重的体力劳动，同时使人有更多时间享受物质文明的成果，促进[[社会|社会]][[文化|文化]]的发展。

== 詞彙爭議 ==
「解放」亦是中國大陸常用術語，常見[[中國共產黨|中國共產黨]]用於形容某些事件，例如[[中国人民解放军|中国人民解放军]]對[[中華民國|中華民國]][[中華民國政府|政府]]控制地區的佔領，被中共稱為「解放」，[[第二次国共内战|第二次国共内战]]被称为“[[解放战争|解放战争]]”。但是香港學者[[陳雲_(學者)|陳雲根]]認為：[[中共中央宣傳部|中共中央宣傳部]]善於生產這類詞彙，以擄略人心。當義軍瓦解了暴政或敵外政權，古文稱為「重光、[[光復|光復]]、收復失地、還我河山」，但是「解放」就絕對稱不上———因為在[[國共內戰|國共內戰]]，受到[[蘇聯|蘇聯]]資助和支配的中共在1949年推翻的[[國民政府|國民政府]]並非敵外政權。又如[[二次大戰|二次大戰]]期間，盟軍攻陷[[納粹|納粹]][[德國|德國]]統治之地，及攻破關鎖[[猶太人|猶太人]]之[[集中營|集中營]]，猶太人重獲自由，才能稱之「解放」。而[[同盟國_(第二次世界大戰)|盟軍]]「解放」了納粹德國的佔領區，人民是紛紛回國定居，而不是像中國人般往外逃難的。<ref>[https://hk.news.yahoo.com/blogs/sandwich/%E8%A7%A3%E6%94%BE%E8%88%87%E5%BB%BA%E6%94%BF.html 解放與建政]，雅虎香港，陳雲 三文治，2009年11月11日</ref>
在[[中国大陆|中国大陆]]，以“解放”作为地名或路名的地方有成千上百处，例如[[解放路|解放路]]、[[解放街道|解放街道]]等。[[广州|广州]]的中华路在[[中華人民共和國|中華人民共和國]]建國後就被改成了“[[解放路_(广州市)|解放路]]”。
