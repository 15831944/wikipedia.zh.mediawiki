== 劇情 ==
在虛擬網路中一名暱稱為海盜船長的極客（[[韓庚|韓庚]]飾），因實踐自己的平民英雄主義，而捲入密謀案<ref>{{Cite web |url = https://news.tvbs.com.tw/life/930906 |title = 韓庚、鳳小岳、山下智久　《解碼遊戲》男神齊聚 |author = 林芷卉 |date = 2018-06-02 |publisher = TVBS |language = zh-hant |accessdate = 2018-06-25 }}</ref>。
