'''输元河'''是[[河北|河北]][[邯郸市|邯郸市]]主城区西北地区以及[[邯郸县|邯郸县]]一带季节性泄洪河流，为[[滏阳河|滏阳河]]四条支流之一{{注|其他三条是[[牤牛河|牤牛河]]、[[沁河_(邯郸市)|沁河]]、[[渚河|渚河]]。}}。河流总长约20公里，流域面积达82平方公里{{参|name=参考1|{{cite web|author1=吴文锋|title=河流|url=http://www.hd-water.gov.cn/web/pinfo.aspx?id=3846|website=邯郸水利信息网|accessdate=2017-05-11}}}}。

== 源头 ==
[[File:Shuyuan_River_1.jpg|220px]]
输元河源头在邯郸市西部地区的[[紫山|紫山]]东麓以及[[邯郸县|邯郸县]]周边的[[周窑村|周窑村]]、[[姜窑村|姜窑村]]附近，姜窑村西北的[[罗敷潭|罗敷潭]]，便是输元河的发源地{{参|{{cite web|author1=苑清民|title=姜窑本是罗敷家|url=http://www.handannews.com.cn/new_epaper/hdwb/html/2017-04/19/content_169625.htm|website=邯郸新闻网|publisher=邯郸日报|accessdate=2017-05-12}}}}。由此地往南流到[[薛庄|薛庄]]，然后向东北方向流去，流经京广铁路之后在北十五里铺东北方向流入滏阳河。[[1957年|1957年]]输元河改道，在[[苏里村|苏里村]]西北方向汇入滏阳河。输元河下游现在已成为化工区废污水的排放河{{参|name=参考1}}。

== 开发 ==
鉴于输元河流经区域大，邯郸市政府在输元河中下游加强了退耕还林，关闭排污口，河道净化清淤治理的工作，特别是在输元河与滏阳河交汇处修建了[[北湖公园(邯郸)|北湖公园]]，成为主城区北部地域唯一的一片湿地公园，秋季来临时，北湖地区时有南迁的[[候鸟|候鸟]]在此觅食和休息。

== 污染 ==
2016年，当地居民反映，输元河彩虹桥附近有人往河里倾倒垃圾，希望有关部门调查{{参|{{cite web|author1=崔志梅|title=有人向输元河倾倒垃圾|url=http://www.handannews.com.cn/new_epaper/hdwb/html/2016-04/28/content_120746.htm|website=邯郸新闻网|publisher=邯郸日报|accessdate=2017-05-12}}}}。
