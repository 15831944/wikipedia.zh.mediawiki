《'''廉石传奇'''》，中国大陆的古装剧，导演阚卫平，主演于荣光、罗嘉良、杜淳、赵文瑄，2010年5月杀青，[[央视|央视]]2011年首播两集后停播。<ref>{{cite web |url=http://et.21cn.com/tv/roll/2012/08/13/12652483.shtml |title=资料：电视剧《廉石传奇》主创介绍 |accessdate=2012 |publisher=21CN |work= |language=zh-hans |archive-url=https://web.archive.org/web/20141006152959/http://et.21cn.com/tv/roll/2012/08/13/12652483.shtml |archive-date=2014-10-06 |dead-url=yes }}</ref>该剧2011年[[中国中央电视台|中国中央电视台]]曾经播出了2集，媒体称其是“开年大戏”、“反腐倡廉引发中纪委关注”、“贪官比内鬼隐藏得还要深（隐射[[薄熙来|薄熙来]]、[[周永康|周永康]]篡权夺位）”之类，该剧借助三国时期的腐败辛辣讽刺了[[胡温时期|胡温时期]]已经病入膏肓亡党亡国的腐败，与当时[[胡锦涛|胡]][[温家宝|温]]高喊的“[[和谐社会|和谐社会]]”相冲，结果[[中国共产党|中国共产党]][[言论自由|控制新闻媒体及电视台]]的原因而中断播出，之后再也没有播出。<ref>{{cite web |url= http://ent.qianlong.com/4543/2011/08/15/3462@7263865.htm|title=《廉石传奇》群星荟萃 版权协议一波三折终履约 |accessdate=2011 |publisher=千龙网|quote = 30集古装“清官剧”《廉石传奇》，曾因入围央视2011年开年大剧而倍受观众期待。尚有媒体报道说，该剧因肃贪拒贿反腐倡廉的思想主题而引起了中纪委的关注，这无疑更增加了它的“中标”指数。事实上，不仅《廉石传奇》被CCTV“屏蔽” |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/v/m/2010-03-04/10182887652.shtml |title= 三国“反腐剧”《廉石传奇》引来纪委关注|accessdate=2010 |publisher=新浪网|quote =“贪官”比“内鬼”隐藏得还深 |work= |language=zh-hans}}</ref>

==剧情==
该剧分为：惩腐篇、倡廉篇、警世醒世篇。讲述了[[三国|三国]]时期[[东吴|东吴]]“陆绩”惩治腐败、造福一方的人生故事。<ref>{{cite web |url=http://ent.sina.com.cn/v/2012-08-13/08233709672.shtml |title= 资料：电视剧《廉石传奇》故事梗概|accessdate= 2012|publisher= 新浪网|work= |language=zh-hans}}</ref>
