'''斤'''，是[[中国|中国]]傳統[[中国度量衡|度量衡]]中的[[質量|質量]]單位，起源於[[中國|中國]]，再傳到[[日本|日本]]、[[朝鮮半島|朝鮮半島]]、[[越南|越南]]、[[臺灣|臺灣]]、[[馬來半島|馬來半島]]等地。又稱'''司马斤'''。現時多為傳統市場慣用。

==起源==
中国古代有个官职叫「[[司马|司马]]」，司馬主要掌管軍事，其中因為糧秣管理需要秤重，於是「司馬」就和重量單位扯上關係。中国自周代开始有重量单位：斤、[[两|两]]、[[錢_(單位)|钱]]、分，也稱为「司马斤」、「[[司马两|司马两]]」等，计量的工具叫做「司马[[秤|秤]]」，这一标准也叫「司马平制标识」。一司马斤等于十六司马两，[[成語|成語]]「半斤八两」就是这么来的。

==質量==
不同時代、地區的斤實際質量有所變動，但與其他尺貫法單位的質量比例維持不變。一斤等於十六兩，一斤等於百分一[[擔|擔]]，亦即十六[[兩|兩]]或一百二十分之一[[石_(质量单位)|石]]。[[晚清|晚清]]時開始[[英文|英文]]通常寫成'''catty'''或'''kan'''<ref name='tong'>{{cite book|title=唐字音英語|author=莫文暢|date=20世紀初}}</ref>。

==現代的斤==
現代的「斤」按照各地使用習慣，與公制有如下換算：

===中國大陆===
1斤等于500克，是由1929年2月制定、1930年1月1日施行的《[[市制#中華民國|中华民国度量衡法]]》第五条所规定：
:“市用制長度以公尺三分之一為市尺（簡作尺），'''重量以公斤二分之一為市斤（簡作斤）'''，容量以公升為市升（簡作升），'''一斤分為十六兩'''，一千五百尺定為一里，六千平方尺定為一畝，其餘均以十進。”
该法第六条规定：
:“兩：等於斤十六分之一即十錢（0.0625斤）。 ”
所以从1930年到1959年，中国法定的“两”为500克的16分之一，即31.25克。此前，中国各地、各行业都有各自的斤、两重量标准。这次度衡量改制后，这一历史时期的文献大多采用公制与市用制并行。例如，1944年9月19日[[晋绥边区|晋绥边区]]行政公署颁发的供给标准： “口粮(每人每天)：军队系统小米1斤8两(750克)”；1945年1月[[新四军|新四军]][[浙东游击纵队|浙东游击纵队]]司令部规定的供给标准：每人每天“油5钱(15.625克)”，其中的斤、两的定义都符合上述法规。

1959年6月25日中华人民共和国国务院发布《关于统一计量制度的命令》，保留市制，“市制原定十六两为一斤，因为折算麻烦，应当一律改为十两为一斤。”中药计量仍袭旧制不变。1市斤=0.5千克，1市斤=10两。<ref>[http://news.xinhuanet.com/ziliao/2005-01/10/content_2440001.htm 《国务院关于统一我国计量制度的命令》]</ref>

===台灣、澎湖===
[[File:Lin_Hua_Tai_tea_pricelist.jpg|thumb]]

[[台灣|台灣]]使用之-{[[台制|台制]]}-，受日本[[日本度量衡|尺貫法]]影響，1 台斤為1 日斤，故1 台斤 = 600 [[公克|公克]] = 0.6 [[公斤|公斤]]，1台兩 = 37.5 公克。

===金門、馬祖===
[[金門|金門]]與[[馬祖|馬祖]]一直使用[[市制#重量|市斤]]，1市斤 = 500 公克，1 市兩 = 31.25 公克。

===香港及澳門===
[[File:Hong_kong_weight_standards_2.jpg|thumb]]

[[香港|香港]]及[[澳門|澳門]]：1斤約等於604.79克。

原[[香港法例|香港法例]]，1884年第廿二條，一斤為1<sup>1</sup>⁄<sub>3</sub>[[常衡|常衡]][[磅|磅]]（即3斤等如4[[磅|磅]]）。
現時[[香港度量衡制度|香港法律]]规定一斤等於一百分之一[[担|担]]或者十六兩，即0.60478982 [[公斤|公斤]]<ref name = "hk_law">{{cite web|title = 度量衡條例 |work = 電子版香港法例 | url = https://www.elegislation.gov.hk/hk/cap68!zh-Hant-hk }}</ref>。

===日本===
[[日本|日本]]斤：1斤等於600克，现已基本不用。

現時用在[[吐司|吐司]]切片前的大块面包（{{lang|ja|{{link-ja|食パン}}}}）計價的斤，据日本面包{{link-ja|公平交易协议会|公正取引協議会}}（{{lang|ja|日本パン公正取引協議会}}）的{{link-ja|公正競争規約}}，只需在340克以上。510克即可称为1.5斤。<ref>{{cite web |script-title=ja:日本パン公正取引協議会：包装食パンの斤表示の義務化 |url=http://www.pan-koutorikyo.jp/display.html |website=www.pan-koutorikyo.jp}}</ref>

===新加坡===
[[新加坡|新加坡]]斤：1斤約等於604.8克

新加坡法律規定一斤為1<sup>1</sup>⁄<sub>3</sub>[[常衡|常衡]][[磅|磅]]，等於十六兩，即0.6048 [[公斤|公斤]]<ref name = "sg_law">{{cite web|title = Weights and Measures Act (CHAPTER 349) Third Schedule |work = Singapore Statues | url = http://statutes.agc.gov.sg/non_version/cgi-bin/cgi_getdata.pl?actno=1976-REVED-349&doctitle=WEIGHTS%20AND%20MEASURES%20ACT%0A&date=latest&method=part&sl=1&segid=888373245-001666 }}</ref>

=== 馬來西亞 ===
[[馬來西亞|馬來西亞]]斤：1斤約等於604.8克

=== 越南 ===
[[越南|越南]]目前使用'''斤'''稱之為『'''cân ta'''』：1'''司馬斤'''（'''cân ta'''） = 0.6046[[千克|千克]] = 604.6 [[克|克]]；
