'''放射性同位素'''（{{lang-en|radionuclide}}，{{lang|en|radioactive nuclide}} 或 {{lang|en|radioactive isotope}}），一種具有[[放射性|放射性]]的[[核素|核素]]。是一種[[原子核|原子核]]不穩定的[[原子|原子]]，每個[[原子|原子]]也有很多[[同位素|同位素]]，每組同位素的[[原子序|原子序]]雖然是相同，但是卻有著不同的[[原子量|原子量]]，如果這原子是有放射性的話，它會被稱為'''物理放射性核種'''或'''放射性同位素'''。放射性同位素會進行[[放射性衰變|放射性衰變]]，從而放射出[[伽瑪射線|伽瑪射線]]，和[[次原子粒子|次原子粒子]]。

[[化學家|化學家]]和[[生物學家|生物學家]]都把放射性同位素的技術應用在我們的食品、水和身體健康等事項上。不過他們也察覺到危險性，因而制訂使用的安全守則。有些放射性同位素是天然存在的，有些則是人工製造的，稱為'''人造放射性同位素'''。

== 起源 ==
天然存在的放射性同位素主要是[[恆星|恆星]]的內部。例如[[鈾|鈾]]，在星體內直接被形成。而當今仍然存在，是因為它們的[[半衰期|半衰期]]的時間很長，而它們還沒有完全[[衰變|衰變]]。放射產生的同位素，如[[碳-14|碳-14]]，是由長期衰變得出的。(這個也是全部目前[[氦|氦]]形成方法： 雖然它沒有放射性，但是它容易逃離地球，因此氦被從地下倉庫等地方獲得。)

[[核反應堆|核反應堆]]和放射性同位素[[發電機|發電機]]的[[粒子加速器|粒子加速器]]可人工製造放射性同位素。

放射性同位素生產核反應堆中的高流和大量的[[中子|中子]]。 在核反應堆中，中子用來生激活放射性元素。 來自一個核回應堆的一種典型的生成物是[[鉈|鉈]]（Tl）-201。 

== 使用 ==
放射性同位素的用途主要有两个：利用它们的化学特性以及作为辐射的来源。
放射性同位素已廣泛使用在[[核醫學|核醫學]]、，農業<ref>{{cite web|url=http://nuclearconnect.org/know-nuclear/applications/agriculture| title=Agricultural Applications|accessdate=2018-04-30}}</ref>、考古<ref>{{cite web|url=http://nuclearconnect.org/know-nuclear/applications/art-and-science| title=Art and Science Applications|accessdate=2018-04-30}}</ref>、商業<ref>{{cite web|url=http://nuclearconnect.org/know-nuclear/applications/commercial-applications| title=Commercial Applications |accessdate=2018-04-30}}</ref>、工業<ref>{{cite web|url=http://nuclearconnect.org/know-nuclear/applications/industrial-applications| title=Industrial Applications |accessdate=2018-04-30}}</ref>和太空探測<ref>{{cite web|url=http://nuclearconnect.org/know-nuclear/applications/space| title=Using Nuclear Technology to Explore Space |accessdate=2018-04-30}}</ref>。
在食物的保存上，辐射可以阻止[[根块植物|根块植物]]在收成之后发芽，杀除一些攀在植物上的寄生虫与害虫，还可以用来控制水果以及蔬菜的成熟度。

== 危險 ==
如果放射性同位素被釋放到環境，透過一些事故、不良的處理或者其他方法，他們能潛在並引起有害的效應，[[放射性污染|放射性污染]]。如果過分地使用在醫療或生活應用上，也能導致危險，為[[放射性中毒|放射性中毒]]。放射性同位素也能導致電器故障。
