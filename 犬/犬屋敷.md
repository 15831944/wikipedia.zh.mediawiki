== 故事簡介 ==
既不是帥哥，也不是美女，在家裏不被人信賴，也沒有威信，在公司裏也沒有出人頭地——主角就是這樣一個可憐至極的老頭上班族。但在某天，醫生突然宣告他患了晚期癌症，時日無多，他從此開始自暴自棄。

當晚，被突然飛來的UFO的墜落所捲入，轉變成機械身體的他，獲得了遠超人類的力量。另一方面，同樣遭遇事故的高中生——獅子神皓，開始隨心所欲地使用他所獲得的力量。不斷傷害不合己意之人的獅子神，與不斷拯救被獅子神所傷害之人的犬-{}-屋敷。

兩個不同年代的人，兩段不同的人生軌跡，兩人的命運走向了截然不同的道路。

== 登場角色 ==
=== 主角 ===
; {{nihongo|犬-{}-屋敷壹郎|犬屋敷 壱郎（いぬやしき いちろう）|聲：[[小日向文世|小日向文世]]|演：[[木梨憲武|木梨憲武]]}}
: 58歲上班族，非常普通、隨處可见的大叔。既没有希望也没有人望，又不被家人所重视的平凡男人。常常覺得自己活著没有目标、為了家人奉献心力却又得不到回报。 即使有时会想从这种困境中挣脱而出，却又缺少了强壮的身体去面对现实中的考验。
: 但在某一天，却突然得知自己将不久于世，感到绝望的犬-{}-屋敷在公园里抱著爱犬花子痛哭。此时被失控的外星人飛船撞到，被改造成机器人而获得力量。大叔自此走上帮助别人的道路。
: 最後犧牲自己啟動了自爆系統與將要撞擊地球的隕石一同炸成了碎片和灰燼。
; {{nihongo|獅子神皓|獅子神 皓（ししがみ ひろ）|聲：[[村上虹郎|村上虹郎]]、[[藤田奈央|藤田奈央]]（小學生）|演：[[佐藤健|佐藤健]]}}
: 同样被外星人改造成机器人，同样是全新的力量，少年所带来的却是残酷的无差别杀戮。随机挑选想要歼灭的家族，不管是老弱妇孺都不放过。因为唯有人类在死前的挣扎，才能够让变成机械的他感受到自己真正活著。
: 最後犧牲自己啟動了自爆系統與將要撞擊地球的隕石一同炸毀，但隕石沒有這樣就消失，最後由犬-{}-屋敷壹郎再補上一次自爆，隕石才消失不見。

=== 犬-{}-屋敷的家人 ===
; {{nihongo|犬-{}-屋敷麻里|犬屋敷 麻里（いぬやしき まり）|聲：[[上坂堇|上坂堇]]|演：[[三吉彩花|三吉彩花]]}}
: 犬-{}-屋敷壹郎的女儿。高中生，是与父母相貌完全不同，拥有黑色长发的美少女。有点爱慕虚荣，对爸爸的物质供给能力略有不满。是狮子神和安堂的同学，但彼此间没有什么接触。
: 夢想是成為漫畫家，且對安堂父子有濃厚的競爭意識。在家裡公開表明希望能在jump上連載時，得到父親以好好完成高中學業的條件並完全支持。
: 某次跟蹤父親與安堂時意外發現父親在各家醫院挽救已經無法醫治的病人，並發現父親的身體已經變成機械。
: 曾經在新宿的觀景台上被獅子神操控的飛機波及受困於大樓，並失去生命跡象。後在父親的努力下救回，其後與父親恢復以往的親密。
: 在父親前往要墜落於地球的小行星前與其做約定，一定會回到家人身邊，但最終約定未能實現。
: 故事尾聲，麻里投稿的漫畫成功在jump上連載。
; {{nihongo|犬-{}-屋敷剛史|犬屋敷 剛史（いぬやしき こうし）|聲：[[朝井彩加|朝井彩加]]|演：[[福崎那由他|福崎那由他]]}}
: 犬-{}-屋敷壹郎的儿子。初中生，外表有点像他父亲。曾和同学一起看到父亲孤独的一个人吃着饭，有点受到打击。
; {{nihongo|犬-{}-屋敷萬理江|犬屋敷 万理江（いぬやしき まりえ）|聲：[[西宏子|西宏子]]|演：[[濱田麻里|濱田麻里]]}}
: 犬-{}-屋敷壹郎的妻子。
; {{nihongo|花子|はな子}}
: 犬-{}-屋敷領養的愛犬。

; {{nihongo|渡邊紫音|渡辺 しおん（わたなべ しおん）|聲：[[諸星堇|諸星堇]]|演：[[二階堂富美|二階堂富美]]}}
: 与狮子神同班的女同学。父母因癌症雙亡，與奶奶（聲：[[谷育子|谷育子]]）相依为命。喜欢皓，曾當面向他表白过。在皓被警察追捕的时候，收留了皓，并被皓治好了家族遗传倾向的癌症。
; {{nihongo|獅子神優子|獅子神 優子（ししがみ ゆうこ）|聲：[[加藤有生子|加藤有生子]]|演：[[齊藤由貴|齊藤由貴]]}}
