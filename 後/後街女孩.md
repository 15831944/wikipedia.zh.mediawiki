《'''後街女孩'''》（Back Street Girls）是Jasmine Gyuh創作的一部[[日本漫画|日本漫画]]，自《[[週刊Young_Magazine|週刊Young Magazine]]》（[[講談社|講談社]]）2015年16号起開始連載。该作品是一部描述[[黑道|黑道]]3人組因為犯錯而為了不要被老大殺掉，因此被迫接受[[變性手術|變性手術]]與全身[[整形|整形]]手術，進而成為地下偶像「後街女孩」出道的[[搞笑|搞笑]]漫畫。

2018年改編為電視動畫，該年7月至10月間播出10話。2019年2月8日，改編為電影上映。

== 登場人物 ==
=== 後街女孩 ===
; {{Nihongo|山本愛理／山本健太郎|山本アイリ／山本健太郎（やまもと アイリ／やまもと けんたろう）|声：[[贯井柚佳|贯井柚佳]]（愛理）／[[小野大辅|小野大辅]]（健太郎）}}
: 本名山本健太郎，通稱大哥。28岁，留着茶色的中分头发，後街女孩的隊長。當黑道時留著平頭，臉上有一個刀疤。为防止长胡子而注射[[女性荷尔蒙|女性荷尔蒙]]。小时候被家人抛弃，由祖父抚养长大。

; {{Nihongo|立花真理／立花隆|立花マリ／立花リョウ（たちばな マリ／たちばな リョウ） |声：[[前田佳织里|前田佳织里]]（真理）／[[日野聪|日野聪]]（隆）}}
: 本名立花隆，通稱阿隆。25岁。美麗的姐姐型角色。當黑道時留著一頭後梳頭。被稱作是犬金組中最強的打手。父親也是名黑道，但久違的再會後，才發現父親也跟著變性。
: 切下的[[生殖器|生殖器]]找到後，被電視評論員評為：「細細長長的，應該是中學生的尺寸」。

; {{Nihongo|杉原千佳／杉原和彦|杉原チカ／杉原和彦（すぎはら チカ／すぎはら かずひこ）|声：[[赤尾光|赤尾光]]（千佳）／[[兴津和幸|兴津和幸]]（和彦）}}
: 本名杉原和彥，通稱阿和。23歳。可愛的妹型角色。當黑道時留著一頭電棒頭，還有小鬍子。高中時曾打過[[拳擊|拳擊]]。

; {{Nihongo|莉娜|リナ|声：[[高橋未奈美|高橋未奈美]]（莉娜）／[[川田紳司|川田紳司]]（喬治）}}
: 本名喬治。後街女孩練習生第1期生。25岁。美國人，出身自[[紐約|紐約]]貧民街，是[[義大利|義大利]]黑幫GOD MONEY的成員。但由於表現失態，為了獲得如何經營偶像團體的知識而被送往犬金組，並因此遭到變性。

; {{Nihongo|中村結衣|中村結衣（なかむら ゆい）|声：[[真野步|真野步]]}}
: 後街女孩練習生第2期生。17岁。[[青森縣|青森縣]]人，夢想加入後街女孩，因此直接和犬金組長談判後以練習生身分加入。是後街女孩中唯一的女性，也和黑道沒有任何關係。曾被後街女孩被害人協會設立的HG製作公司挖角未果。

=== 後街女孩的相關人物 ===
; {{Nihongo|犬金鬼萬次郎|犬金鬼万次郎（いぬがねきまんじろう）|声：[[藤原啓治|藤原啓治]]}}
: 犬金組組長。犬金企画的社長兼製作人兼作詞家。前大儲組組員。性格腹黑而且鬼畜，对“后街女孩”采取暴力严惩的法规，甚至让其他犯错或不听话的组员进行变性手术。要求组员称自己为“老师”。 最后为“后街女孩”的解体危机，而自愿切掉男性生殖器。
; {{Nihongo|曼德林木下|マンダリン木下（マンダリン きのした）|聲：[[諏訪部順一|諏訪部順一]]}}
: 後街女孩的經紀人兼製作人。捧紅過許多偶像團體，是經紀人界的傳奇人物。在他的手上沒有不會紅的偶像。但由於只是名與犬金組沒有任何關係的一般人，所以並不知道後街女孩的真面目，對於他們作為偶像卻有著脫序的言行感到不解，並受到很大的困擾。
: 透過數年來對於少女心理学的研究，獲得日本第一位少女学博士（講談大学大学院基礎文化研究科少女心理学）博士（心理学）。
; {{Nihongo|木村|木村（きむら）|聲：[[花江夏樹|花江夏樹]]}}
: 後街女孩的助手。犬金組的若眾。是三人組在當黑道時的小弟。現在是千佳的粉絲。

=== 後街女孩被害人協會 ===
; {{Nihongo|岡本鐵平|岡本鉄平（おかもと てっぺい）|声：[[斧篤志|斧篤志]]}}
: 後街女孩被害人協会会長，家里开着蛋拌飯專門店，也是元祖蛋拌飯第2代店主。67岁。由於後街女孩演出的美食節目使得他已經營業84年的店倒閉，讓他也因此組成後街女孩被害人協會。

; {{Nihongo|村上裕子|村上裕子（むらかみ ゆうこ）|声：[[恒松步|恒松步]]}}
: 後街女孩被害人協會副会長。62岁。雖然有夫有子，但由於父子都成為真理的粉絲的事情曝光，讓她的家庭因此崩壞。

; {{Nihongo|竹村真治|竹村真治（たけむら しんじ）|声：[[福島潤|福島潤]]}}
: 前犬金組的金庫守衛。32岁。因為侵占款項的事情曝光而被迫變性，但因為臉孔讓人感到噁心而被組織驅逐。

; {{Nihongo|岩村太郎|岩村太郎（いわむら たろう）|声：[[江川央生|江川央生]]}}
: 犬金組中的第五號人物。46岁。因為弄傷後街女孩的臉，被組長強迫施以豐胸手術。

; {{Nihongo|田中夏子|田中なつ子（たなか なつこ）|声：{{tsl|ja|斉藤貴美子|齊藤貴美子}}}}
: 大牌[[演歌|演歌]]歌手。犬金組組長之妻。57岁。為人毒舌，是名新人殺手。討厭偶像，誤以為老公和後街女孩有外遇。

=== 其他 ===
; {{Nihongo|藤井岳美}}
: 32歲，後街女孩被害人協會副會長之子，迷戀後街女孩，偷走真理當年被切下來的生殖器，成立「後街女孩教」，自命教祖。
