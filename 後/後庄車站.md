'''後庄車站'''位於[[高雄市|高雄市]][[大寮區|大寮區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[屏東線|屏東線]]的[[鐵路車站|鐵路車站]]。於1908年時設立的後庄車站，其站名由來源自所在地[[大寮區|大寮區]]三大聚落「前-{庄}-」、「中-{庄}-」與「後-{庄}-」中的後-{庄}-之地名。

== 車站構造 ==
後庄是一個設置有兩座[[島式月台|島式月台]]的地面車站，但在1992年時降等為簡易站後，就只使用第二月台。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#簡易站|簡易站]]。
*停靠狀態：[[臺鐵區間車|區間車]]皆停靠。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[一卡通_(台灣)|一卡通]]、[[悠遊卡|悠遊卡]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1908年6月25日：設「後庄驛」，由打狗驛（今[[高雄港車站|高雄港車站]]）派員售票。
*1935年11月7日：由打狗驛分離，正式升為一獨立運作之車站。
*1992年12月12日：由三等站降為簡易站，並指定由[[鳳山車站_(臺鐵)|鳳山站]]管理。
*2013年9月30日：啟用多卡通刷卡機。
