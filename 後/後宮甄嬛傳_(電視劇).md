《'''後宮甄嬛傳'''》，是根據[[流瀲紫|流瀲紫]]同名網絡言情小說《[[后宫甄嬛传|后宫甄嬛传]]》改編而成的一部[[後宮|後宮]]爭鬥題材電視劇，講述主角[[孝聖憲皇后|甄嬛]]從不諳世事的善良女子，成長為善於謀權的深宮婦人故事。[[電視劇|電視劇]]由[[郑晓龙|郑晓龙]]导演，為延续其现实主义风格，他將小說原先的[[架空歷史小說|架空歷史]]，改編成[[清朝|清朝]][[雍正|雍正]]的後宮之中，並將原作情感的部分简单处理。

[[北京电视艺术中心|北京电视艺术中心]]購進電視版權，於2010年9月18日在[[北京|北京]]開機拍攝。[[孙俪|孙俪]]、[[蔡少芬|蔡少芬]]、[[蒋欣|蒋欣]]分別飾演[[孝聖憲皇后|甄嬛]]、[[皇后|皇后]]和[[敦肅皇貴妃|華妃]]一角。此外，沈眉莊與安陵容分別由[[斓曦|斓曦]]與[[陶昕然|陶昕然]]扮演。[[皇帝|皇帝]]、温实初與[[允禮|果郡王]]則分別由[[陈建斌|陈建斌]]、[[张晓龙|张晓龙]]與[[李东学|李东学]]飾演。2011年1月30日，全劇在[[圓明園|圓明園]]殺青。

2011年11月起，《後宮甄嬛傳》陸續在[[中國大陆|中國大陆]]各地方台播出。[[臺灣|臺灣]]於2012年3月20日起在[[華視主頻|華視]]首播。3月26日起在[[安徽广播电视台卫星频道|安徽衛視]]、[[东方卫视|东方卫视]]同步首播。[[香港|香港]]於9月起在[[香港有線電視|有線電視]]首播。電視劇網路版權則由[https://www.litv.tv/vod/drama/content.do?brc_id=%20&content_id=VOD00044842 LiTV 線上影視]、[[乐视网|乐视网]]購得。該劇在2012年多次重播後引起網民不少反響，在大陆年度收視率排名14位。

== 背景與發行 ==
《後宮甄嬛傳》改編自女作家[[吴雪岚|吴雪岚]]的同名网络小说。2010年，电视剧版权被[[北京电视艺术中心|北京电视艺术中心]]购进。同年9月18日，電視劇在北京开机拍攝。<ref>{{cite web|url=http://www.hinews.cn/news/system/2011/12/14/013810726.shtml|title=湖州籍80後作家吳雪嵐創作電視劇熱播|publisher=南海網|author=|date=2011-12-14|accessdate=2012-01-31 |language=Zh-hans}}</ref>電視劇由[[郑晓龙|郑晓龙]]导演，这是郑晓龙首次执导的古装戏。<ref name="搜狐2">{{cite web|title=《后宫甄嬛传》压轴岁末 陈建斌蔡少芬助阵开播|url=http://yule.sohu.com/20111129/n327259255.shtml|website=搜狐娱乐|accessdate=2018-01-10}}</ref>郑晓龙表示作品延续自己的现实主义风格，不同於原作設定在架空的大周朝，郑晓龙将故事安排在[[雍正|雍正]]年间。對此，他表示：“落地是为了接地气，安排在一个朝代，是为了让故事真实”。<ref name="人民1">{{cite web|title=《后宫甄嬛传》引热议 导演批当下后宫戏弱智|url=http://culture.people.com.cn/GB/22219/16582914.html|website=人民网|accessdate=2018-01-10}}</ref>為求得真實，郑晓龙曾三次造訪故宫，停留一整天。<ref name="东方早报">{{cite web|title=《甄嬛传》制作方保守估计一集收益400万|url=http://ent.sina.com.cn/v/m/2012-05-03/10283620927.shtml|website=新浪网|publisher=东方早报|accessdate=2018-01-10}}</ref>此外，他認為“过度渲染帝王如何至情至性并不真实”，因而將原作情感的部分简单处理，希望拍一个有历史厚重感的戏，体现出封建制度对人性的摧残，对封建社会进行理性批判。<ref name="人民1"/>

2011年1月30日，《后宫甄嬛传》在北京圆明园遗址公园的冰面上杀青。<ref>{{cite web|title=《后宫甄嬛传》杀青 孙俪李东学演绎宫廷绝恋|url=http://ent.qq.com/a/20110131/000286.htm|website=腾讯网|accessdate=2018-01-10}}</ref>電視劇粗剪后約有90余集，為了让故事更加紧凑，郑晓龙將影片剪到76集。<ref name="人民1"/>2011年11月，《後宮甄嬛传》登陸中國大陸8家地面频道。<ref name="网易娱乐">{{cite web|title=孙俪《后宫甄嬛传》20日首播 将掀新宫斗热点|url=http://ent.163.com/11/1114/16/7IR787C000031GVS.html|website=网易娱乐}}</ref>2012年3月26日起，電視劇在天津、江西、上海、安徽四家卫视同步上星。<ref>{{cite web|title=《后宫甄嬛传》卫视开播 李天柱角色显忠诚(图)|url=http://ent.sina.com.cn/v/m/2012-03-26/17183590498.shtml|website=新浪网|accessdate=2018-01-10}}</ref><ref name="搜狐">{{cite web|title=《甄嬛传》东方卫视开播 38天速读职场教科书|url=http://yule.sohu.com/20120326/n338905343.shtml|website=搜狐娱乐|accessdate=2018-01-10}}</ref>其中在東方衛視的播出，创下该台电视剧有史以来在本地和全国的收视最高值。<ref name="东方早报"/>台灣方面，《後宮甄嬛傳》於2012年3月在[[中華電視公司|華視]]首播<ref>{{cite web|title=《甄嬛》重播5次 打趴國內節目|url=https://tw.appledaily.com/headline/daily/20130125/34793087|website=蘋果日報|accessdate=2018-01-10|language=zh-TW}}</ref>，最後一集以2.97收視率，創下華視8點檔多年來新紀錄，而緯來電視台隨後跟進播出，完結篇收視亦達3.34。<ref>{{cite web|title=華視墮落玩重播 1年搞3次甄嬛-助長中國劣質劇 扼殺台灣電視劇發展|url=http://ent.ltn.com.tw/news/paper/634741|website=自由娛樂|accessdate=2018-01-10}}</ref>香港方面，2012年9月，《後宮甄嬛傳》在[[香港有線電視|香港有線電視]]播出，但觀眾反应平淡。<ref>{{cite web|title=《甄嬛传》香港爆冷 观众对宫斗审美疲劳|url=http://ent.qq.com/a/20121206/000155.htm|website=腾讯网|accessdate=2018-01-10|language=zh-CN}}</ref>網路方面，[[乐视网|乐视网]]以2000万元购得了《后宫甄嬛传》的独家网络版权。電視劇使網站点击率创造历史的新高，给整个网站带来了20亿的流量。<ref>{{cite web|title=乐视网独播《甄嬛传》收益5000万|url=http://finance.ifeng.com/stock/ssgs/20120503/6407816.shtml|website=凤凰网|accessdate=2018-01-10}}</ref>

== 選角 ==

== 劇情与特色 ==
全劇講述一位出身官宦人家的漢軍旗女子甄嬛，她聰明伶俐、美貌與智慧並存。她自小的夢想是「願得一心人，白首不相離」，並沒有入宮為妃的打算。不想入宮的她，卻意外留在宮中。初入宮就吸引皇帝的目光，但受到幾乎全後宮的嫉妒和暗算。在幾次受寵與失寵之際，令她看透了後宮的真面目。從不諳世事的善良女子，成長為善於謀權的深宮婦人，藉由才智自無數次失勢中翻身，終究登峰造極，成為至高無上的皇太后，同時體驗到深宮中的冰冷孤寂。

《後宮甄嬛傳》是一部宮廷情感大戲，更寫實描述後宮嬪妃的真實情感。不同於過去以後宮鬥爭為主的古裝電視劇，劇中絕大多數的場景與故事都集中在後宮（除了甄嬛落飾出家期間有部分野外與甘露寺、凌雲峰及清涼台等場景），雖然後宮鬥爭也牽扯著朝政與軍事，但相關場景極少，就連皇帝上朝的場景也不多，這是《後宮甄嬛傳》的特點。劇情中前朝與後宮的關係，如劇中的台詞：「一榮俱榮，一損俱損。」嬪妃的盛寵興衰，關係到自家父親的宦途甚至全族的命運。

[[南京|南京]]语言文字专家[[朱启丰|朱启丰]]在看完电视剧全集之后，透过新闻媒体指出了电视剧中演员诸多的读音错误。第一处，台词“[[江南|江南]]有二乔，[[河北|河北]][[甄宓|甄宓]]俏。”{{noteTag|1=“甄宓”是[[三国|三国]][[袁绍|袁绍]]的儿媳妇，后来被[[曹丕|曹丕]]夺取。见[[曹植|曹植]]《[[洛神赋|洛神赋]]》。 }} 剧中“宓”读作“mì密”，正确读音为“fú福”；第二处，台词“你弹的是《淇奥》{{noteTag|1= 《淇奥》即《[[诗经|诗经]]·[[卫风|卫风]]·淇奥》}} ”的“奥”读作“ào澳”，正确读音为“yù玉”；第三处，台词“嬛嬛一袅楚宫腰”{{noteTag|1=“嬛嬛一袅楚宫腰”典出[[宋朝|宋朝]][[蔡伸|蔡伸]]的《一剪梅·堆枕乌云堕翠翘》：“堆枕乌云堕翠翘。午梦惊回，满眼春娇。嬛嬛一袅楚宫腰。那更春来，玉减香消。柳下朱门傍小桥。几度红窗，误认鸣镳。断肠风月可怜宵。忍使恹恹，两处无聊。” }} 读作“huán环”正确读音为“xuān宣”，原小说作者流瀲紫原本讀“xuān宣”音，但後來由於普羅大眾皆讀之為“嬛（huán）”，故而順應時勢，改讀“嬛（huán）”音。<ref> {{cite web |url= http://culture.ifeng.com/8/detail_2012_05/04/14322069_0.shtml|title= 《甄嬛传》中皇上念错字 甄宓的宓应念“福”|publisher=凤凰网 |accessdate=2012年5月4日  |work= |language =zh-cn}}</ref>

而《[[咬文嚼字|咬文嚼字]]》编辑部在2012年底公布的“年度十大语文差错”中，关于“嬛”的读音问题没有收入年度语文十大差错。<ref>[http://ent.sina.com.cn/v/m/2012-12-27/09363821802.shtml 咬文嚼字公布十错 甄嬛被指读错未入围]</ref>

===甄嬛週邊===
本剧热播後市場掀起一股甄嬛熱，除了小說再版狂銷外。各式各樣與研究裡面人物故事的週邊書也應聲而出，如導演鄭曉龍與總製片人曹平推出的《你沒看懂的后宮甄嬛傳》劇評專書，羅霖（Rosy）在東森新聞雲、Nownews連載超過四十萬字，獲媒體評為「兩岸第一甄嬛迷」。

香港在鳳凰衛視專門播出[[紀連海|紀連海]]的評論後出版《紀連海說甄嬛》，解說雍正歷史和妃嬪的細節，圖文詳細，全彩印刷，系列節目也很受歡迎，兩岸三地均有販售。

此外，臺灣還出版許多講述職場的《甄嬛教會我的36則職場生存術：職場中「周全自己」，也「周全所有人」的厚黑攻略》、《後宮甄嬛傳教我的80件事》、《職場即後宮：《甄嬛傳》教會我的辦公室賤招》、《從菜鳥到終極CEO：甄嬛傳裡的職場啟示錄》、《華妃早知道該多好的道理》、《後宮心機美容：太醫如何讓嬪妃們青春永駐》、《見人說人話之超效溝通指南》、《人人心理都有鬼：實現自我價值的幽默心理學》等各類相關書籍。

===甄嬛体===
本剧一经热播，其中台词优雅而略带复古的语言风格也受到关注。不少[[网民|网友]]在[[微博|微博]]、[[网络论坛|论坛]]上广泛使用甄嬛体的语言。<ref>{{cite web |url= http://www.zaobao.com/special/china/zaodian/pages3/zaodian_chongqing120510.shtml|title= 戏看甄嬛|publisher=联合早报 |accessdate=2012年5月10日  |work=顾功垒 |language =zh-cn}}</ref>

===清宫剧更名事件及清宫剧串联===
本剧热播后，[[黑龙江卫视|黑龙江卫视]]播出的[[清宫剧|清宫剧]]《[[宫_(2011年电视剧)|宫]]》更名为甄嬛前传，但这只是一种宣传。此外，网友还将《[[步步惊心_(电视剧)|步步惊心]]》更名为《甄嬛前传2》，《[[还珠格格|还珠格格]]》更名为《甄嬛后传》<ref>{{Cite web |url=http://news.hexun.com/2012-05-10/141245687.html |title=你以为你换个名字 观众就不认得了吗 |accessdate=2018-04-29 |author= |date=2012-05-10 |work= |publisher=和讯网，来源：温州商报 |language=Zh-hans }}</ref>，更有加入《[[宫锁珠帘|宫锁珠帘]]》的大串联——281集《甄嬛演义》。而五部电视剧的先后顺序依次是《甄嬛前传（宫）》、《戏说甄嬛（步步惊心）》、《甄嬛秘史（宫锁珠帘）》、《后宫甄嬛传》、《甄嬛后传（还珠格格）》，戏称是“中国电视剧史上最长宫廷剧”<ref>{{Cite web |url=http://yule.sohu.com/20120508/n342635921.shtml |title=《甄嬛传》引发清宫剧大串联 纯元皇后是若曦？ |accessdate=2018-04-29 |author= |date=2012-05-08 |work= |publisher=搜狐网，来源：北国网-半岛晨报 |language=Zh-hans }}</ref>。

===植入广告争议===
本剧中“[[东阿阿胶|东阿阿胶]]”出现的频率越来越高，不但剧中的后妃、宫女、[[太醫|太医]]都在念叨，主要人物都吃过，甚至连[[皇帝|皇帝]]的赏赐中也出现有“[[东阿阿胶|东阿阿胶]]”商标的盒子。有网友戏称，《甄嬛传》要成“[[阿胶|阿胶]]传”。<ref>[http://gz.ifeng.com/lvyoujingdian/detail_2012_04/18/184798_0.shtml 深陷“造假门” 东阿阿胶欲借《甄嬛传》“还魂”] {{webarchive|url=https://web.archive.org/web/20130115103854/http://gz.ifeng.com/lvyoujingdian/detail_2012_04/18/184798_0.shtml |date=2013-01-15 }}</ref>

===重播問題===
本剧於2012年在全中國各地不停重播，[[黑龙江卫视|黑龙江卫视]]等電視台更空出一個頻道，半年內24小時除《[[新闻联播|新闻联播]]》及当地新闻外滾動播出，被網友戲稱為「甄嬛台」。<ref>[http://hk.apple.nextmedia.com/realtime/entertainment/20121223/51195911 蔡少芬《甄嬛傳》全國無限loop 網民哀嚎﹕放過我媽]《蘋果日報》2012年12月23日</ref><ref>[http://www.cnr.cn/gundong/201212/t20121221_511614293.shtml 黑龙江卫视频繁重播同一剧遭调侃是“甄嬛台”] 中国广播网 2012-12-21</ref>

===价值观争议===
2013年9月，《[[人民日报|人民日报]]》刊文称韩剧《[[大長今_(電視劇)|大长今]]》“只有坚持正义才能最终战胜邪恶”的[[价值观|价值观]]比《甄嬛傳》“好人斗不过坏人，好人只有变坏、变得比坏人更坏才能战胜坏人”更正确。并指出评价历史题材作品时，最重要的标准还不仅仅是真实性标准，而是价值观标准。不正确的价值观会导致观众把不正确的生存理念带入现实生活<ref>[http://hb.people.com.cn/GB/n/2013/0919/c192237-19554534.html 《大长今》比《甄嬛传》价值观更正确] {{webarchive|url=https://web.archive.org/web/20130921054751/http://hb.people.com.cn/GB/n/2013/0919/c192237-19554534.html |date=2013-09-21 }} 人民网-人民日报 2013年09月19日 </ref>。

2014年1月，《[[求是|求是]]》提出了与《人民日报》相反的观点，为《甄嬛传》正名，认为《甄嬛传》“不是歌颂阴谋、欣赏斗争，而是借一个个青春女性理想和生命的惨烈毁灭，揭示出封建社会的腐朽本质。”文章强调人民大众的审美鉴赏能力是值得信任的，“文艺批评要充分尊重艺术规律，充分尊重作家艺术家的创造性劳动，最大限度地贴近社会大众的审美心理”。<ref>[http://news.xinhuanet.com/local/2014-01/03/c_125953600.htm 求是为《甄嬛传》正名：有思想穿透力 不是歌颂阴谋] 新华网-求是 2014年01月03日 </ref>

===海外播出===
2014年，《甄嬛传》制片人曹平就透露，76集的《甄嬛传》由美国团队操刀，缩减成6集的电视电影将于年底登陆[[HBO|美国HBO电视网]]，成为首部在美国主流电视台播出的中国电视剧。这个消息好让国人激动一番，但前期宣传片都放出来了，却没任何要在美国播出的准确消息。雪上加霜的是，HBO亚洲新业务高级副总裁范贝贝公开表示这部电视剧并没有卖到HBO，是片方的一场炒作。据她透露，“连英剧都难卖到美国主流媒体，何况中文的宮鬥剧？”遭遇如此“响亮的耳光”，《甄嬛传》制片人曹平随后表示不太清楚《甄嬛传》在美国的播出情况，还矢口否认说过会在HBO播出，“哪个平台播没有定，没有详细过问。只是说会在美国播，具体在哪个电视台播我们没说过。过段时间就有眉目了。”

2015年正式在美国[[netflix|netflix]]的TVshows播出，排名第三，在开播遇冷后，短短十几天在5分制的netflix评分为3·8，直逼《卧虎藏龙》的3·9分。在美国imdb评分已到8·6分。<ref>{{cite news|title=HBO否认播美版《甄嬛传》|url=http://news.mtime.com/2015/02/13/1539528.html|accessdate=2014-12-23|newspaper=时光网|date=2015-01-01}}</ref>。
