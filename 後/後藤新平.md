'''後藤新平'''{{BD|1857年|7月24日|1929年|4月13日|catIdx=Gotou, Shinpei}}為[[大日本帝國|大日本帝國]]時期的[[政治家|政治家]]、[[醫師|醫師]]，[[陸奥|陸奥]]塩釜（今[[岩手縣|岩手縣]][[奥州市|奥州市]]）人，曾任[[東京都知事列表|東京市]]第七屆[[市長|市長]]、[[東京放送局|東京放送局]]（今[[日本放送協會|日本放送協會]]）第一任總裁、[[拓殖大學|拓殖大學]]第三屆校長、[[臺灣總督府|臺灣總督府]][[民政長官|民政長官]]、[[南滿洲鐵道|南滿鐵道會社]]首任社長、[[內務大臣|內務大臣]]和[[外務大臣|外務大臣]]、[[帝都復興院|帝都復興院]]總裁，1906年封[[男爵|男爵]]，1922年升為[[子爵|子爵]]，1928年晉封[[伯爵|伯爵]]。後藤熱衷於[[童軍|童軍]]活動，也是[[日本童軍總會|日本童軍總會]]的首任會長，是童軍運動推廣者。

後藤是極為稱職的[[技術官僚|技術官僚]]，在[[臺灣|臺灣]][[民政長官|民政長官]]（1898年－1906年）任內，因[[臺灣總督|臺灣總督]][[兒玉源太郎|兒玉源太郎]]軍務繁忙，後藤實際掌握[[臺灣政治|臺灣政治]]，後出任[[南滿洲鐵道|南滿鐵道會社]]首任社長，在[[長春|長春]]、[[大連|大連]]等地致力建設，奠定了日本在[[滿洲|滿洲]]的殖民基礎，並在[[關東大地震|關東大地震]]後出任[[帝都復興院|帝都復興院]]總裁，重建了[[帝都|帝都]][[東京|東京]]。後藤在各地的治績，至今皆為人所稱道。

== 早年 ==
1857年7月24日，後藤新平出生於[[日本|日本]][[陸奥國|陸奥國]][[胆澤郡|胆澤郡]]塩釜村（今[[岩手縣|岩手縣]][[奥州市|奥州市]][[水澤區|水澤區]]）一武士之家，父親十右衛門，在[[仙台藩|仙台藩]]水澤館主[[伊達將監|伊達將監]]門下當[[侍衛長|侍衛長]]，明治維新後歸家務農。後藤新平年幼時給[[膽澤|膽澤]][[大參事|大參事]][[安場保和|安場保和]]當[[書僮|書僮]]，工作之餘，勤學不倦。

1871年，後藤到東京去求學，不得志而歸。1874年，[[安場保和|安場保和]]任[[福島縣|福島縣]]縣長時，進入福島縣[[須賀川醫學校|須賀川醫學校]]學習後取得醫師資格，行醫數年後留學德國得到[[醫學博士|醫學博士]]學位。

== 青年時期 ==
1876年，適逢[[安場保和|安場保和]]任[[愛知縣|愛知縣]]縣長，邀請他去名古屋一學校當教師，隨後到愛知縣醫院出任三等醫師。[[西南戰爭|西南戰爭]]時任陸軍臨時醫院醫師，到大阪去護理傷病兵員。1882年，25歲的後藤出任愛知醫學校校長及病院院長。

根據有關資料描述，當時西醫雖然是奇缺，但後藤的提升速度也著實地讓人吃驚。其原因有兩點：一是他醫術高明精湛，有讓人敬佩之處；二是他處世狡黠，善於投機，左右逢源<ref> 王勝利等主編:《大連近百年史人物》，遼寧人民出版社，1999年。</ref>  。比如，1882年4月，自由党總理[[板垣退助|板垣退助]]在岐阜縣遇刺，根據電邀，他想前去診治，當和愛知縣府方面商榷時，縣府方面不同意，於是他決心犧牲自己的前途，立即前往岐阜醫治，[[板垣退助|板垣退助]]看他言語行動與眾不同，對旁人說：“新平是個非凡人物，不當政治家實屬可惜。”正是基於這一點，1883年1月，[[安場保和|安場保和]]將女兒嫁給了後藤。

後藤鑒於當時社會一般衛生知識思想幼稚，感到有設立衛生局的必要，在1889年發表了《國家衛生原理》，主題思想除了談論衛生工作，但字裡行間中已顯露出治國的韜略。此後不久[[內務省|內務省]]設立衛生局時，受局長[[長興専齋|長興専齋]]推薦任內務部技師。1890年被派往德國學習兩年。

1892年11月，任內務部衛生局局長，他就在其躊躇滿志之時卻因“相馬家內訌”案（{{link-ja|相馬事件}}）的牽連，不僅被迫丟掉了局長的位子，而且還被關進了監獄 ，半年後，真相大白而出獄。

後藤新平並未因此次挫折和打擊而沉淪。1894年中日[[甲午戰爭|甲午戰爭]]爆發，軍醫總監黑石忠憨推薦他到明治天皇所在的廣島大本營工作，直到戰後簽訂和約，後藤建議檢疫是當前的緊急任務，於1895年4月任陸軍檢疫事務官長。時任陸軍次官的[[兒玉源太郎|兒玉源太郎]]看到他非常勤奮努力工作，把他記在心中，同年9月重任衛生局長，再任2年半，致力於內務行政及衛生設備的革新改善，與[[北里柴三郎|北里柴三郎]]共同創立了預防傳染病研究所。

== 台灣民政長官 ==
1898年2月26日，首相[[伊藤博文|伊藤博文]]任命[[兒玉源太郎|兒玉源太郎]]為第4任台灣總督，並給兒玉配備了一位能幹的助手，就是衛生局來的後藤新平，後藤為民政長官，兩人於該日正式走馬上任。

兒玉源太郎重用後藤新平，後藤新平也對兒玉源太郎畢恭畢敬，所以兩人合作非常融洽。兒玉源太郎有好幾次因為要擔任其他職位，要推薦後藤新平擔任台灣總督，但是因為按照日本當時的內規，台灣總督必須是[[將領|將領]]，兒玉源太郎甚至想要建議日本政府取消這個規定，可是都被後藤新平婉辭了。

今天的縱貫鐵路，絕大部分就是在後藤新平的任內所完成的。在台灣林業史上曾經扮演重要角色的阿里山森林鐵路，也是後藤新平主導興建的。到了1904年，台灣財政已可完全自主，無須日本中央在行補貼，甚至一些公共建設還比日本內地還要先進，例如當時的台北地下水道覆蓋率是亞洲第一。

=== 台灣舊習慣調查會 ===
後藤新平上任後，檢討以前日本的統治政策，發現有一些缺失。首先後藤研究出臺灣一直被[[清朝|清朝]]視為化外之地，[[臺灣人|臺灣人]]的地方[[仕紳|仕紳]]於是各自為政，自治非常發達。這種自治是一種[[習慣法|習慣法]]，然而日本統治者卻輕易破壞這個自治制度，想要引進新的文明、新的法令，[[臺灣人|臺灣人]]根本聽不下去，難措手足，這就是失敗的根源。今後的方針，應該恢復舊習慣，總督府只站在監督者的立場，只有發現弊害時，才逐漸改善，這樣的行政就事半功倍。當時大多數[[日本人|日本人]]認為日本文明而台灣野蠻，後藤新平這樣肯尊重台灣舊習慣的想法實在難能可貴。

後藤新平為了徹底尊重台灣舊習慣，首先他展開大規模調查。1900年，後藤新平聘請[[京都帝國大學|京都帝國大學]]教授[[岡松參太郎|岡松參太郎]]，於1901年成立臨時台灣舊習慣調查會，後藤新平自己擔任會長。1903年，京都帝國大學教授[[織田萬|織田萬]]加入調查會。途中，織田萬對調查的對象太龐大、複雜，而有些氣餒，後藤新平勉勵他說：「這是為了世界文化，而且只有日本人能做。」

=== 廢除「三段警備法」===
後藤新平面臨的最大問題還是抗日[[游擊隊|游擊隊]]。在[[乃木希典|乃木希典]]時代，對抗日游擊隊採取的策略是「[[三段警備法|三段警備法]]」，在抗日游擊隊出沒的地區，由軍隊與憲兵管轄。在平地與市街地，由警察管轄。介於中間的地區，則由警察與憲兵協力管轄。

這個「三段警備法」表面上看起來很合理，其實非常不合理。因為軍隊與警察，不但指揮系統不同，想法也南轅北轍，甚至經常感情不合。對滲透到住民之中的游擊隊，正規軍根本無用武之地。軍隊一聽到游擊隊出現，趕過去時，游擊隊早已消失。於是軍隊開始懷疑當地住民，而難免施行暴力，反而遭來民眾恨意。

後藤新平與兒玉源太郎看出其中缺點，決定廢除「三段警備法」，他們決定讓警察負起治安全責，後藤新平對軍隊演講說：「你們的職務是在治理台灣，不是在征討台灣！」

從此，後藤新平的策略是讓警察深入民眾，孤立游擊隊，並且對游擊隊招降。後藤鑑於台灣義民抗日活動頻繁，便把乃木希典總督試行的「土匪招降策」在全台各地推行，但卻在招降抗日分子後進行槍決，最著名的是1902年（明治三十五）4月25日，在斗六、林圯埔、崁頭厝、西螺、他里霧、內林等「歸順式場」中，將被誘降而來的「歸順匪徒」全部當場屠殺。在儀式會場上「臨機處置」或其他方式「斷然處置」的台灣人，據後藤新平在所著的《日本殖民政策一斑》中自承，屠殺的「叛徒多達一萬一千九百五十人」。到了1902年，幾乎已經看不到抗日游擊隊的蹤影。

=== 鴉片問題 ===
{{see|臺灣日治時期鴉片政策}}
{{Cquote|世人有認台灣已經成功，這只是單獨看到台灣財政的獨立，而忘了其他。台灣財政的獨立，實因當初關於帝國殖民地統治的輿論甚為險惡，為了對付這一情勢而取的應急辦法；至其必然因此而生的弊害，不但不能讓外國知道，亦不能讓台灣新附人民知道。}}
除了抗日游擊隊的問題之外，鴉片也是一大問題。日本國內很多人主張應該嚴禁，可是後藤新平認為應該逐漸禁止。原因在於：
# 認為日本人不易染上鴉片，只會危害台灣人，不會危害日本人。
# 當時沒有低痛苦的戒癮療法，怕成癮者受不了。
# 鴉片專賣收入很高，幫助台灣財政獨立。或投入到軍隊花費，減少抗日人士及匪徒的實力。

他的鴉片漸禁政策是：<br> 
# 鴉片由政府專賣，政府在各地指定專門藥店，只有在這裡才能買到藥用鴉片。<br> 
# 由醫生診斷確定為鴉片中毒者，發給証明，持此証明可以買到鴉片。<br> 
# 鴉片課以高稅。<br><br>
或有論者以為，在後藤新平的政策下，1900年台灣的鴉片中毒者為16萬9千人，17年後，減少到6萬2千人，1928年，減少到2萬6千人，是後藤新平的漸禁策略獲得成功。實際上，後藤新平的鴉片漸禁政策以獲得鴉片暴利為優先考量，用簡單可行的方式管理吸食鴉片的人口為次，幾十年後鴉片吸食人口的大量減少，和鴉片漸禁政策其實關係不大。鴉片人口的減少原因如下：<br>
# [[杜聰明|杜聰明]]等人發明新的戒癮方法，戒癮時低痛苦；在此之前，鴉片成癮者的自然死亡是使用率下降的主因。<br>
# 日本人教育成功，讓臺灣人知道鴉片之害，且臺灣人現代意識覺醒（殖民初期，日方一直強調會允許以成癮者繼續使用，以減少抗日情緒；但也在學校向臺灣人宣傳鴉片危害，因而受到國際肯定)。<br>
# 台灣上流社會及知識份子態度丕變：本來吸鴉片是台灣高級社交場所的日常景象，反而是日本警察經常在這種社交場所找麻煩、阻止未成癮者進入這種場所；但在1929年左右，台灣知識份子及上流社會突然轉向、要求總督府加強戒癮輔導。<br>
# 醫學進步，鴉片可以提煉藥用嗎啡，而且日本在二戰時需要大量藥用嗎啡。<br>
# 鴉片收入佔總督府收入銳減，停售鴉片也不會造成財政問題。<br>
綜上所述，後藤新平在鴉片政策上的成功乃是對於台灣財政獨立的扶持之功，並以之支持日軍鎮壓抗日分子的費用，如何幫助鴉片成癮的台灣人戒除毒品並非後藤新平考量的要點。

=== 土地調查 ===
在解決完抗日游擊隊和鴉片兩個問題後，後藤新平接著進行土地調查。台灣的土地所有權非常複雜，有大租戶（墾首）、小租戶、（墾戶）、佃農（佃戶），而且有許多隱田。1885年，清朝新任巡撫劉銘傳曾經想要測量全島農田，卻遭至全台灣住民的反抗，甚至引起暴亂，結果失敗。

1898年，後藤新平設立臨時台灣土地調查局，自己擔任局長。後藤新平對手下再三交代，進行土地測量時要尊重當地人的感情，要尊重他們的祖墳。1904年，土地調查告一段落，台灣總督府以公債向大租戶買農田。而且經過這番土地調查，發現許多隱田，結果實際農田是以往的1.7倍。

1904年，基督徒[[竹越與三郎|竹越與三郎]]（1865－1950）目睹後藤新平的詳細土地調查，讚嘆說：「與這個相比，日本在明治七年實施的地租改革簡直是兒戲！」

=== 道路、鐵路、港灣 ===
接下來，後藤新平著力的是道路、鐵路、港灣等交通建設。日本接收台灣的時候，幾乎沒有像樣的道路。村落與村落之間，村落與市街之間，只有30公分寬的小路。在後藤新平主持下，台灣建築了1.8公尺寬的道路5千5百公里，超過1.8公尺寬的道路2千9百公里，超過5.4公尺寬的道路8百公里，超過7.2公尺寬的道路80公里。

接著是鐵路。在清朝統治期間，從1887年花了7年鋪設從基隆到新竹約100公里的鐵路。但是品質不佳，橋是木造，稍為有些轉彎的地方，旅客必須下車一起推火車。日本接收台灣後，民間成立台灣鐵路公司，計劃興建台灣縱貫鐵路。後藤新平上任後，主張鐵路國有，並且認為台灣鐵路的1千萬日圓興建計畫不夠，應該追加到3千萬日圓。後藤新平成立台灣鐵路部，自己擔任部長。1909年，台灣縱貫鐵路完成，比當初預定的還要提早兩年。

接著，後藤新平實施的是建設基隆港與充實衛生設備。尤其是後者，是後藤新平非常重視的項目。台灣當時有許多傳染病，甚至曾經因鼠疫而死了4500人。充實衛生設備最大的重點是建設上下水道。日本剛開始統治台灣時，軍用水道只有基隆與淡水兩處，後來在後藤新平積極推動下，台灣的上下水道比日本國內還要早完成。[[竹越與三郎|竹越與三郎]]於1904年第一次訪問台灣時，驚訝地說：「我所看到的台灣首府，純然是歐洲風的清潔宏闊市街，道路好像東京的道路，甚至比東京的道路還漂亮，步道與車道之間，鋪有水泥，下面有下水道，讓雨水和汙水可以排放。」

=== 製糖產業 ===
後藤新平並且從美國聘請[[新渡戶稻造|新渡戶稻造]]擔任台灣的殖產局長。新渡戶稻造一上任，就前往[[爪哇|爪哇]]考察砂糖產業，回來之後，他對後藤新平說，先讓他了解一下台灣的現況再寫報告書。後藤新平說：「不必先了解一下台灣的現況，在你還未了解台灣的現況之前，寫報告書。要是等你了解台灣的現況，說不定你就寫不出來。不如就你在爪哇所看到的寫心得就好。」不久，新渡戶稻造提出「糖業改良意見書」，根據這份意見書，後藤新平開始改良甘蔗品種、改善耕作方法（特別是獎勵施肥）。在一般傳統社會，要農民採取新方法不是一件容易的事，後藤新平便用補助金方式來「利誘」。新品種和新耕作方法不久就普及開來，甘蔗的生產量大增。接著是製糖工廠的現代化與大規模化。1900年砂糖產量為3萬噸，到兒玉、後藤時代終期為6萬噸，到1937年為100萬噸，二次大戰期間最多為160萬噸，台灣已經不是日本的負擔，而是日本的金雞母。

== 治理臺灣 ==
後藤新平在臺灣總督府[[民政長官|民政長官]]任內，進行土地調查、[[戶口普查|戶口普查]]及風俗習慣調查，推展[[生物學|生物學]]的殖民政策，促進[[科學|科學]]、[[農業|農業]]、[[工業|工業]]、[[衛生|衛生]]、[[教育|教育]]、[[交通|交通]]與治安[[警察|警察]]等[[發展|發展]]，並招撫[[抗日|抗日]]份子，攏絡臺灣的[[門閥|門閥]][[士紳|士紳]]，奠定日本在台灣往後的統治基礎。

後藤對臺灣的建設始於[[醫療|醫療]]和[[衛生|衛生]]的改善。1895年，他受聘擔任臺灣總督府衛生顧問時，著手規劃關於[[鴉片|鴉片]]政策、衛生、[[自來水|自來水]]和[[下水道|下水道]]等建設。接任民政長官之後，後藤秉持「生物學原則」，開始大規模地進行的調查與建設。後藤曾說：「[[殖民地|殖民地]]行政計畫，在目前科學進步之下，必須根據生物學的原則，也就是要發展農業、工業、衛生、教育、交通、警察。如果以上各項能夠完成，我們就可以在生存競爭中獲得保全及勝利。」

舊慣調查為臺灣的文化、風俗、民情和律法留下重要的紀錄；而土地及戶口調查的全面與精確，不但成為日本殖民統治與建設的重要基礎，其影響甚至延續到戰後[[國民政府|國民政府]]在臺的施政。

[[File:Shinpei_Goto_in_the_Aoyama_Cemetery.JPG|thumb]]青山公墓的後藤新平墓]]
[[File:Goto_Shinpei_Bronze_Monument.JPG|thumb]]
在產業上，後藤新平選定了在臺灣原本就有基礎的[[糖|糖]]業，引進新式製糖技術、經營模式及大量資本，促使臺灣糖業蓬勃發展。並大力推動[[鐵路|鐵路]]、[[港口|海港]]和[[公路|公路]]等交通建設，現今的[[縱貫鐵路|縱貫鐵路]]絕大部分就是在其任內所完成，在臺灣林業史上曾經扮演重要角色的[[阿里山森林鐵路|阿里山森林鐵路]]亦是由後藤新平主導興建。到了1904年（明治37年），臺灣財政已可完全自主，無須日本中央補貼，一些公共建設甚至比[[日本內地|日本內地]]還要先進，例如當時[[臺北市|臺北市]][[下水道|下水道]]覆蓋率已是[[亞洲|亞洲]]第一高。

[[國立臺灣博物館|國立臺灣博物館]]入口大廳右邊一龕位原置放其銅雕像，目前已移除改放於頂樓展覽廳與[[兒玉源太郎|兒玉源太郎]]銅像一起展示<ref>{{cite web |language=zh-tw |url=http://www.ntm.gov.tw/tw/public/public.aspx?no=105 |title=建築瑰寶 |publisher=國立臺灣博物館 |date= |accessdate=2011-09-27 |archive-url=https://web.archive.org/web/20110713130239/http://www.ntm.gov.tw/tw/public/public.aspx?no=105 |archive-date=2011-07-13 |dead-url=yes }}</ref>。



== 評論 ==
=== 與孫中山之關係 ===
1900年，[[中國|中國]]革命家[[孫中山|孫中山]]計劃在[[廣東|廣東]][[惠州|惠州]][[起事|起事]]反[[清|清]]，時[[臺灣總督|臺灣總督]][[兒玉源太郎|兒玉源太郎]]贊同孫中山的計畫，於是兒玉總督命時任民政長官的後藤新平協助孫中山起事。後因[[日本內閣|日本內閣]]改組，[[第4次伊藤內閣|第4次伊藤內閣]]反對介入中國革命，導致惠州起義失敗。1900年11月10日，孫中山[[化名|化名]]「吳仲」，與後藤新平同行乘「橫濱丸」自[[基隆|基隆]]啟航返回日本。


=== 與共濟會的關係 ===
根據[[文化人類學|文化人類學]]學者[[綾部恒雄|綾部恒雄]]所言，後藤新平應該是共濟會會員<ref>綾部恒雄《秘密結社》（講談社、2010年）193頁。</ref>。後藤新平重用的[[新渡戶稻造|新渡戶稻造]]為共濟會會員的可能性也相當高。<ref>吉田祐二「日本初・国際“超”高級官僚としての新渡戸稲造」（副島＋SNSI副島国家戦略研究所『フリーメイソン＝ユニテリアン教会が明治日本を動かした』）231頁。</ref>

=== 黃旺成評論 ===
1927年2月20日，臺灣社會運動者[[黃旺成|黃旺成]]（菊仙）在《臺灣民報》上認為，後藤新平提出了「治台三策」：第一，臺灣人貪財愛錢，可用利益誘惑；其次，臺灣人貪生怕死，得用高壓手段威脅；第三，臺灣人非常愛面子，可用虛名攏絡。<ref>菊仙（黃旺成），[http://59.125.121.246/TM/tm/tm_show.php?id=A2012040002304 〈後藤新平氏的「治台三策」〉]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}，《[[台灣民報|台灣民報]]》第145號（1927年2月20日）第14版。菊仙在該文中說：「後藤新平氏在臺灣做民政長官的時候，從臺灣人的性質上發見了三條的弱點，因為要利用這弱點，所以定了治臺的三策：一、臺灣人怕死－－要用高壓的手段威嚇的。二、臺灣人愛錢－－可以用小利誘惑的。三、臺灣人重面子－－可以用虛名籠絡的。」但實際上未有記錄顯示後藤新平曾提出過這三點。</ref>後來這種說法衍生為評斷台灣人民性的知名論點，即：「貪財、怕死、愛作官」（又稱「愛錢、怕死、愛面子」）。此論點認為，後藤新平採用了「[[糖飴與鞭|糖飴與鞭]]」的兩手策略，一方面以嚴刑峻法對待異議人士，另一方面積極致力於建設臺灣經濟。
