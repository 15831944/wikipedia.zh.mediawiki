'''後營里'''位於[[臺南市|臺南市]][[西港區|西港區]]東北部，因里內聚落「後營」而得名{{r|縣|page=589}}{{r|西港常民文化|page=69}}{{r|地名}}。而由於後營聚落人口眾多，所以在行政上分成東邊的後營與西邊的營西兩里{{r|縣|page=589}}{{r|西港常民文化|page=69}}{{r|地名}}。

== 沿革 ==
該里在日治時期屬於[[西港庄|西港庄]][[大字|大字]]後營，二次大戰後因後營人口眾多，以庄廟普護宮前道路為界，將東邊劃為「後營村」{{r|縣|page=589}}{{r|西港常民文化|page=69}}{{r|地名}}。2010年12月25日，因臺南縣市合併升格，改為後營里。

== 聚落與地名 ==
後營里境內只有後營一個聚落{{r|縣|page=589}}{{r|西港常民文化|page=69}}{{r|地名}}。後營地名的由來，據說與明鄭時期的屯墾制度有關{{r|縣|page=589}}{{r|西港常民文化|page=69}}{{r|地名}}。據說當時屯墾的北路軍隊之後援部隊於此地屯墾，故稱後營{{r|縣|page=589}}{{r|西港常民文化|page=69}}。另外又有福建省泉州府同安縣人（今金門縣）蔡應科{{NoteTag|蔡應科，字士京，後人稱「士京公」，出生於明崇禎十一年（1638年）{{r|西港常民文化|page=71}}。據蔡家後裔的說法，士京公因未能和名字「應科」一樣中舉感到遺憾，而兒輩忙於農務，所以將希望寄託在孫輩上{{r|西港常民文化|page=71}}。而後來一門15孫有9人中舉，士京公因此被尊稱為「旗竿祖」{{r|西港常民文化|page=71}}。}}隨鄭軍到此開墾，傳有五房，其中以四房人口最多{{r|縣|page=589}}{{r|西港常民文化|page=70、71}}{{r|地名}}。這五房各成一個角頭，再加上新厝、水溝（仔）、竹圍（仔）、巷仔、社尾（仔）五個角頭，便是普護宮的十角頭{{r|縣|page=589}}{{r|西港常民文化|page=70、71}}{{r|地名}}。

== 宗教信仰 ==
該里以後營普護宮為信仰中心，過去麻豆公學校後營分校（今後營國小）創立之初曾借用普護宮為校舍{{r|縣|page=589、590}}{{r|西港常民文化|page=71}}{{r|地名}}。
