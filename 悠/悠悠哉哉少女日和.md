{{日本作品|悠悠哉哉少女日和|のんのんびより}}，是日本漫畫家{{link-ja|atto|あっと}}的漫畫作品，在[[Media_Factory|Media Factory]]發行的《[[月刊Comic_Alive|月刊Comic Alive]]》連載中，已出版12冊[[單行本|單行本]]。為描寫某個鄉下地方學校「旭丘分校」的日常喜劇漫畫，並以作者的單篇作品《{{lang|ja|とことこ}}》為基礎再編成<ref>{{cite news|url=http://mantan-web.jp/2013/12/23/20131222dog00m200040000c.html|script-title=ja:マンガ質問状：「のんのんびより」 “にゃんぱすー”誕生は作者のノリから|publisher=[[每日新聞|每日新聞Digital]]|date=2013-12-23|accessdate=2014-01-24|language=ja}}</ref>。作者過去作品的登場角色亦會於此作登場。

2013年2月宣布動畫化<ref>{{cite news|url=http://www.animenewsnetwork.com/news/2013-02-21/non-non-biyori-countryside-school-comedy-manga-gets-anime|title=Non Non Biyori Countryside School Comedy Manga Gets Anime|publisher=[[Anime_News_Network|Anime News Network]]|date=2013-02-21|accessdate=2013-09-14|language=en}}</ref>，並於2013年10月7日播出<ref>{{cite web|url=http://www.tvdrama-db.com/drama_info/p/id-56000|script-title=ja:のんのんびより（のんのんびより(1)）|publisher=TV Drama Database|language=ja|accessdate=2015年7月8日}}</ref>。2014年4月6日宣布决定制作電視動畫第2期<ref>{{cite web|script-title=ja:「のんのんびより」第2期製作決定！|url=http://nonnontv.com/news/#news104|work=TV動畫「悠悠哉哉少女日和」官方網站|publisher=[[SILVER_LINK.|SILVER LINK.]]|language=ja|date=2014-04-06|accessdate=2014-04-09}}</ref>。2014年12月27日發表第2期標題為《悠悠哉哉少女日和 Repeat》<ref>{{cite web|script-title=ja:「TVアニメ『のんのんびより』第2期のタイトルが『のんのんびより りぴーと』に決定。キービジュアルも公開|url=http://moca-news.net/article/20141227/201412271038a/01/|publisher=moca|language=ja|date=2014-12-27|accessdate=2015-02-26}}</ref>。2015年3月20日，制作方于宣傳影片中公布，第2期於2015年7月6日開始播放<ref>{{YouTube|5acy-c1uHPA|【PV】「のんのんびより　りぴ～と」！プロモーション映像第1弾！|language=ja}}</ref>。2017年5月23日，宣布发售的单行本第11卷發表新作动画制作决定；剧场版动画于2018年8月25日上映。<ref>{{Cite web|url=http://nonnontv.com/news/#post-3510|title=TVアニメ『のんのんびより りぴーと』公式サイト|accessdate=2017-06-02|work=TVアニメ『のんのんびより りぴーと』公式サイト|language=ja}}</ref>

== 概要 ==
作品舞台是一個掛著「注意牛隻穿越」的看板，公車不等上2個小時<ref>漫画中5个小时</ref>不會來的鄉下農村。主角一條螢因父親工作調職的關係從東京遷居此地，故事即是描寫螢與越谷姐妹及宮內姐妹渡過的日常光景。

== 登場角色 ==
; {{nihongo|宮內蓮華|宮内 れんげ（みやうち れんげ）|聲：[[小岩井小鳥|小岩井小鳥]]<ref name="character">{{cite web|url=http://nonnontv.com/character/|script-title=ja:きゃらくた|work=TV動畫「悠悠哉哉少女日和」官方網站|accessdate=2016年12月6日|language=ja}}</ref>}}
: 旭丘分校小學1年級生，生日12月3日。會在語尾加上「{{lang|ja|のん}}」這個詞，臉上表情變化較少為特點，有著獨特的感性與奇特的思路，其言行常常讓周圍的人無法理解，小腦袋有著未知的驚人潛力。
: 名字源自[[紫雲英|紫雲英]]（日語同音），暱稱為「{{nihongo|小蓮|れんちょん}}」。有著「{{nihongo|喵安|にゃんぱすー|extra=常用音译为「喵帕斯」}}」這句獨特的問候語，飼養一隻名為「具」的狸貓，擁有非常獨特的特技。

; {{nihongo|一條螢|一条 蛍（いちじょう ほたる）|聲：[[村川梨衣|村川梨衣]]<ref name="character"/>}}
: 旭丘分校小學5年級生，生日5月28日。因父親工作調職的關係而從東京搬家。身體成長較早熟，使得外表給人的感覺不像小學生，往往被誤認成大人。崇拜著學姐小鞠。飼養一隻狗「佩琪」<ref>單行本第3集第24話。</ref>。
: 擅於裁縫，房間內擺著大量手工小鞠布偶。曾與小鞠兩人相約出遊時因用心打扮，成熟的穿著及戴著眼鏡的外型，反讓小鞠誤認為他人。
: 總是給人較年長的氣質。某次與蓮華被關在兔子小屋時曾經哭泣並大聲求救，流露出孩子氣的一面。
: 暱稱為「{{nihongo|螢螢|ほたるん}}」。

; {{nihongo|越谷夏海|越谷 夏海（こしがや なつみ）|聲：[[佐倉綾音|佐倉綾音]]<ref name="character"/>}}
: 旭丘分校中學1年級生，生日1月24日。小鞠的妹妹。性格開朗的熱鬧氣氛製造者。喜歡捉弄別人，尤以小鞠居多。
: 小時候就經常惡作劇，連母親也都覺得傷腦筋，暱稱為「{{nihongo|夏夏|なっつん}}」。
: 年幼時曾有「我要成為哥哥的新娘」這類[[戀兄情結|戀兄情結]]發言。
: 雖然不擅長學習，卻在某些方面有著淵博的知識。

; {{nihongo|越谷小鞠|越谷 小鞠（こしがや こまり）|聲：[[阿澄佳奈|阿澄佳奈]]<ref name="character"/>}}
: 旭丘分校中學2年級生，生日9月14日。夏海的姐姐。外貌嬌小晚熟，或許發育較遲，很介意自己不到140cm的身高，對成為大人有著憧憬，因此總是裝成大人的模樣。
: 實際有著害怕恐怖東西的弱小一面，暱稱為「{{nihongo|小小鞠|コマちゃん}}」（由來並非名字的暱稱而是身高）。某次跟大家去海邊玩的時候，曾被當成迷路的孩子帶到服務中心<ref>單行本第2集第54頁。</ref>。

; {{nihongo|宮內一穗|宮内 一穂（みやうち かずほ）|聲：[[名塚佳織|名塚佳織]]<ref name="character"/>}}
: 旭丘分校教師，24歲。蓮華的姐姐。由於旭丘分校學生總數只有5人，上課時基本上是自習，並會在講桌上睡覺。
: 家中有著農田，曾指導螢她們插秧的方法。
: 有時會表現出沒有責任感的一面。

; {{nihongo|越谷卓|越谷 卓（こしがや すぐる）}}
: 旭丘分校中學3年級生。夏海、小鞠的哥哥，生日4月11日。全校唯一的男學生，不愛說話且存在感低，有著[[御宅族|御宅族]]的氣息，能用黏土做出精巧的[[手辦模型|人形]]。

; {{nihongo|越谷雪子|越谷 雪子（こしがや ゆきこ）|聲：[[平松晶子|平松晶子]]}}
: 卓、夏海、小鞠的母親。會邀請蓮華和螢來家裡吃早餐，是位溫柔的母親，但也有責罵夏海不理想成績的嚴格一面。

; {{nihongo|石川穗乃香|石川 ほのか（いしかわ ほのか）|聲：[[高垣彩陽|高垣彩陽]]}}
: 小學1年級生。暑假跟著父親回老家時遇到蓮華，而與蓮華成為朋友。

; {{nihongo|宮內日影|宮内 ひかげ（みやうち ひかげ）|聲：[[福圓美里|福圓美里]]<ref name="character"/>}}
: 高中1年級生。蓮華的姐姐，一穗的妹妹。在東京唸書。
: 喜歡炫耀在東京的見聞，比如坐新幹線。
: 也有在作者的另一部作品《小惡魔蛋白甜餅》中登場<ref>單行本第2集收錄《小惡魔蛋白甜餅》外傳。</ref>。

; {{nihongo|加賀山楓|加賀山 楓（かがやま かえで）|聲：[[佐藤利奈|佐藤利奈]]<ref name="character"/>}}
: 旭丘分校的畢業生，人稱賣零食的，稱一穗為學姐，20歲。
: 經營零食店，販售和租賃各種日常用品、被子，也有代購物品服務，並包辦滑雪的租賃工作。
: 因為在中學生時曾照顧過蓮華，雖然自稱對小孩子沒轍，但私底下對蓮華會展現出溫柔的態度有著類似姐姐或母親般的感情。

; {{nihongo|富士宮木實|富士宮 このみ（ふじみや このみ）|聲：[[新谷良子|新谷良子]]}}
: 旭丘分校的畢業生，就讀當地高中的3年級生。
: 很坦率，有戲弄人的喜好。
: 住在越谷家隔壁，常常到越谷家玩。

; {{nihongo|新里葵|新里 あおい（にいざと あおい）|聲：[[下地紫野|下地紫野]]}}<ref>{{Cite web|url=http://nonnontv.com/movie/|title=劇場版《悠悠哉哉少女日和 Vacation》|accessdate=2018|work=劇場版アニメ『のんのんびより ばけーしょん』|language=ja}}</ref>
: 劇場版原創角色，[[民宿|民宿]]「宿家·新里」（{{lang|ja|やど家 にいざと}}）經營者的女兒，負責招待前往住宿的主角一行人。
: 性格比起年紀相仿的越谷夏海來得成熟，兩人同為中學一年級。
: 喜愛打[[羽毛球|羽毛球]]，被夏海撞見練習的樣子，才開始有了交集，後來成為了好朋友。
