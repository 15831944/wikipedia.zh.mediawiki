《'''再會吧、那些愛憐的悲傷啊'''》（{{lang-ja|サラバ、愛しき悲しみたちよ}}）是日本女子組合[[桃色幸運草Z|桃色幸運草Z]]的第9張[[單曲|單曲]]，於2012年11月21日由[[STARCHILD|STARCHILD]]發售。同名A面曲《再會吧、那些愛憐的悲傷啊》是日本電視劇《[[我的學生是惡夢|我的學生是惡夢]]》的主題曲。

== 發行及宣傳 ==
此單曲分為附送[[DVD|DVD]]的「首批限量版」（{{lang|ja|初回限定盤}}）和只有CD的「普通版」（{{lang|ja|通常盤}}），共兩個版本發售。DVD收錄了A面曲《再會吧、那些愛憐的悲傷啊》的[[音樂影片|音樂影片]]。<ref>{{cite web|url=http://www.momoclo.net/discography/single14.html |language=ja |title=<span lang="ja">DISCOGRAPHY｜週末ヒロイン ももいろクローバーZ オフィシャルサイト</span> |publisher=STARDUST PROMOTION |date=2012 |accessdate=2013-03-02}}</ref>

2012年10月24日，單曲封面和B面曲的資料公布。<ref name="official" />10月26日，《再會吧、那些愛憐的悲傷啊》的MV上載至[[YouTube|YouTube]]，觀看次數於上載兩個半月後超過1000萬次。<ref>{{cite web|url=http://www.rbbtoday.com/article/2013/01/16/101229.html |language=ja |title=<span lang="ja">ももクロ＆布袋寅泰のコラボ曲、YouTube再生回数が1000万回突破……紅白で披露し話題に</span> |publisher=RBBTODAY |date=2013-01-13 |accessdate=2013-03-05}}</ref>11月7日，《再會吧、那些愛憐的悲傷啊》於music.jp率先開放下載。<ref name="official" />一星期後，即11月14日，B面曲《Wee-Tee-Wee-Tee》亦於[[iTunes_Store|iTunes Store]]開放下載。<ref name="official">{{cite web|url=http://www.momoclo.net/pub/pc/information/?id=72 |language=ja |title=<span lang="ja">11/21（水）9thシングル「サラバ、愛しき悲しみたちよ」発売決定！</span> |publisher=STARDUST PROMOTION |date=2012 |accessdate=2013-03-02}}</ref>

同年11月5日，桃色幸運草Z於[[日本武道館|日本武道館]]舉辦、只限男性參與的「桃色秋天兩大祭典 ～男祭2012～」演唱會（{{lang|ja|ももクロ秋の2大祭り ～男祭り2012～}}）上首次演唱《再會吧、那些愛憐的悲傷啊》。<ref>{{cite web|url=http://natalie.mu/music/news/79465 |language=ja |title=<span lang="ja">ももクロ武道館で汗まみれの男祭り、新曲「サラバ」初披露</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2012-11-06 |accessdate=2013-03-02}}</ref>她們亦曾於《[[MUSIC_JAPAN|MUSIC JAPAN]]》（同年11月18日）和《[[MUSIC_STATION|MUSIC STATION]]》（同年11月23日）等音樂節目宣傳單曲。<ref>{{cite web|url=http://natalie.mu/music/news/79965 |language=ja |title=<span lang="ja">週末MJでももクロ、ファンモン、近藤真彦ら新曲披露</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2012-11-15 |accessdate=2013-03-02}}</ref><ref>{{cite web |url=http://www.tv-asahi.co.jp/music/contents/m_lineup/1128/index2.html |language=ja |title=<span lang="ja">ミュージックステーション 2012/11/23</span> |publisher=[[朝日電視台|朝日電視台]] |date=2012 |accessdate=2013-03-02 |archive-url=https://web.archive.org/web/20150501231948/http://www.tv-asahi.co.jp/music/contents/m_lineup/1128/index2.html |archive-date=2015-05-01 |dead-url=yes }}</ref>

== 歌曲 ==
A面曲《再會吧、那些愛憐的悲傷啊》是由[[北川景子|北川景子]]主演、[[日本電視台|日本電視台]]系連續劇《[[我的學生是惡夢|我的學生是惡夢]]》（2012年10月13日至12月22日播映）的主題曲，為桃色幸運草Z的第一首電視劇主題曲。<ref name="oricon1002" />

這首歌由著名音樂人[[布袋寅泰|布袋寅泰]]作曲及編曲。<ref name="oricon1002">{{cite web|url=http://www.oricon.co.jp/news/movie/2017319/full/ |language=ja |title=<span lang="ja">ももクロ、布袋書き下ろしで初ドラマ主題歌「夢にも思ってなかった」</span> |publisher=[[Oricon|Oricon]] |date=2012-10-02 |accessdate=2013-03-02}}</ref>桃色幸運草Z製作一方邀請布袋寅泰作曲時，曾在他本人得到消息之前被其經理人拒絕。之後布袋寅泰跟[[Christopher_Peppler|Christopher Peppler]]（{{lang|ja|クリス・ペプラー}}）一起用膳時，Christopher建議布袋接受邀請，指桃色幸運草Z「並非只是普通的[[日本偶像|偶像]]。她們很搖滾」，才讓布袋想嘗試跟她們合作。<ref>{{cite web|url=http://www.oricon.co.jp/news/music/2021197/full/ |language=ja |title=<span lang="ja">布袋寅泰、ももクロのオファー一度断っていた　クリス・ペプラー説得で実現</span> |publisher=[[Oricon|Oricon]] |date=2013-02-01 |accessdate=2013-03-02}}</ref>此外，布袋寅泰曾指他「留意到桃色幸運草Z在其最大魅力所在——演唱會中，能讓會場全部人合為一體」，而這首歌亦有像「[[天使|天使]]與[[魔鬼|魔鬼]]合為一體，互相追趕躲藏」般的印象。<ref>{{cite web|url=http://natalie.mu/music/news/77552 |language=ja |title=<span lang="ja">ももクロ、新曲は布袋寅泰書き下ろし「悪夢ちゃん」主題歌</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2012-10-02 |accessdate=2013-03-02}}</ref>

B面曲《Wee-Tee-Wee-Tee》是[[Takara_Tomy|Takara Tomy]]電子寵物「[[菲比精靈|菲比精靈]]」（Furby）的廣告歌曲。「Wee-Tee」一詞在「菲比語」（Furbish）中的意思是「歌唱」，日本報導故將歌名翻譯成「歌唱吧」（{{lang|ja|歌おう}}）和「歌唱，歌唱」（{{lang|ja|歌う、歌う}}）等。這首歌的部份歌詞使用了菲比語。<ref>{{cite web|url=http://www.oricon.co.jp/news/music/2017625/full/ |language=ja |title=<span lang="ja">ももクロ、ファービーCMでノリノリ</span> |publisher=[[Oricon|Oricon]] |date=2012-10-11 |accessdate=2013-03-02}}</ref><ref>{{cite web|url=http://news.mynavi.jp/news/2012/10/11/135/index.html |language=ja |title=<span lang="ja">ももいろクローバーZが新曲も衣装も"ファービー化"、新CMで全力応援!</span> |publisher=Mynavi |date=2012-10-11 |accessdate=2013-03-02}}</ref><ref>{{cite web|url=http://www.furby.com/en_CA/furbish-dictionary |language=en |title=Furbish Dictionary |publisher=Hasbro |date= |accessdate=2013-03-02}}</ref>

== 封面和音樂影片 ==
單曲封面以「善與惡」為主題，以白色和黑色為基調，表現歌曲中兩面的世界觀。桃色幸運草Z亦在MV中扮演[[天使|天使]]與[[魔鬼|魔鬼]]。<ref name="asahi">{{cite web|url=http://www.asahi.com/showbiz/music/TKY201211150337.html |language=ja |title=<span lang="ja">ももいろクローバーＺが新曲「サラバ、愛しき悲しみたちよ」をリリース</span> |publisher=[[朝日新聞|朝日新聞]] |date=2012-11-15 |accessdate=2013-03-02}}</ref>

《再會吧、那些愛憐的悲傷啊》的MV由曾為[[SMAP|SMAP]]、[[南方之星|南方之星]]製作MV的[[黑田秀樹|黑田秀樹]]執導。<ref name="asahi" />MV中有部份舞蹈鏡頭使用以「無線[[電致發光|EL]]線控制系統」來製作的發光服裝，當中使用的系統由[[學校法人片柳學園|學校法人片柳學園]]（{{lang|ja|学校法人片柳学園}}）的藤本實助教連同他的學生一起開發。<ref>{{cite web |url=http://www.news2u.net/releases/105844 |language=ja |title=<span lang="ja">藤本実メディア学部助手とメディア学部3年生がももいろクローバーZのミュージックビデオにシステム提供[東京工科大学]</span> |publisher=News2u |date=2012-11-22 |accessdate=2013-03-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20121220014533/http://www.news2u.net/releases/105844 |archivedate=2012-12-20 }}</ref>

== 反應 ==
不少相關報導和評論都指出《再會吧、那些愛憐的悲傷啊》帶有[[搖滾|搖滾]]風格。<ref name="oricon1002" /><ref name="asahi" /><ref name="cdjournal">{{cite web|url=http://artist.cdjournal.com/d/-/4112100154 |language=ja |title=<span lang="ja">ももいろクローバーZ/サラバ、愛しき悲しみたちよ</span> |publisher=CDJournal.com |date=2012 |accessdate=2013-03-02}}</ref><ref name="Billboard Japan">{{cite web |url=http://www.billboard-japan.com/hs_discs/detail/525 |language=ja |title=<span lang="ja">ロックシーンを代表する御大は、ももクロに超大作を用意した</span> |publisher=Billboard Japan |date=2012 |accessdate=2013-03-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20130314175442/http://www.billboard-japan.com/hs_discs/detail/525 |archivedate=2013年3月14日 |df= }}</ref>音樂網站CDJournal的試聽評論指出，這首歌的低音部充份利用了重低音，結他的音色能激發情緒，且充滿奔馳的感覺。<ref name="cdjournal" />另一音樂網站Billboard Japan的杉岡祐樹讚賞布袋寅泰寫的曲子，指他完全沒有依賴特效來創作音效或使用簡單的編曲，而是在歌裏加入「帶苦味的結他重複段（riff）」、「搶眼的副歌」等元素。<ref name="Billboard Japan" />

=== 銷售成績 ===
此單曲發行首周的銷量約為7.4萬張，於2012年12月3日的[[Oricon公信榜|Oricon公信榜]]單曲周榜取得第2名，屈居於[[渡邊麻友|渡邊麻友]]《[[發光體_(渡邊麻友單曲)|發光體]]》之下。<ref name="oricon" />這是桃色幸運草Z截至當時為止的最高首周銷量，也是於該排行榜取得的最高名次。<ref>{{cite web|url=http://news.mynavi.jp/news/2012/12/03/001/ |language=ja |title=<span lang="ja">ももいろクローバーZ、バレンタインイベントを開催! 「チョコを作りたい!」</span> |publisher=Mynavi |date=2012-12-03 |accessdate=2013-03-02}}</ref>此外，此單曲在[[Japan_Hot_100|Billboard JAPAN Hot 100]]排行榜取得首名。<ref name="hot100" />

下載銷量方面，《再會吧、那些愛憐的悲傷啊》一曲取得[[RecoChoku|RecoChoku]]每日排行榜首名，為桃色幸運草Z初次於該榜奪冠。這首歌的MV亦同時在RekoChoku影片日榜上取得首名。<ref>{{cite web|url=http://www.billboard-japan.com/d_news/detail/8512 |language=ja |title=<span lang="ja">ももクロ 宇多田ヒカルら強豪との上位争い制し、初の1位獲得</span> |publisher=Billboard Japan |date=2012-11-22 |accessdate=2013-03-02}}</ref>

桃色幸運草Z於2012年年底的[[NHK紅白歌合戰|NHK紅白歌合戰]]演唱《再會吧、那些愛憐的悲傷啊》一曲後，受到「紅白效應」影響，於2012年12月31日至2013年1月6日的Oricon公信榜單曲周榜上，由第10名上升至第6名。當時的累積銷量約為11.5萬張。<ref>{{cite web|url=http://www.oricon.co.jp/news/rankmusic/2020376/full/ |language=ja |title=<span lang="ja">【オリコン】ももクロ、斉藤和義ら紅白初登場組が急上昇</span> |publisher=[[Oricon|Oricon]] |date=2013-01-08 |accessdate=2013-03-02}}</ref>

== 現場表演 ==
2012年12月31日，桃色幸運草Z初次登上NHK紅白歌合戰的舞台，演唱以「桃色紅白Z」（{{lang|ja|ももいろ紅白だZ!!}}）為題的組曲。該組曲由《再會吧、那些愛憐的悲傷啊》和其主流出道單曲《[[行動吧！怪盜少女|行動吧！怪盜少女]]》兩首歌曲組成。她們先穿上黑色或白色的長裙演唱《再會吧、那些愛憐的悲傷啊》，然後脫去身上的黑白長裙，露出裏面以五名成員各自的代表顏色（紅、黃、粉紅、綠、紫）為主色的裙子繼續演唱下一首歌曲。<ref>{{cite web|url=http://www.oricon.co.jp/news/movie/2020272/full/ |language=ja |title=<span lang="ja">【紅白】ももクロ、4年越しの夢舞台で号泣　「あかりんに届いたと思う」</span> |publisher=[[Oricon|Oricon]] |date=2012-12-31 |accessdate=2013-03-02}}</ref>
