== 劇情簡介 ==
敘述不擅長與男性交際的女大學生小南靜歌與[[偽娘|穿女裝]]的男子霧島涼相識相戀的故事<ref>{{Cite web |url= http://w-comics.jp/introduction/?pid=1382|title= 作品紹介 スカートの中はケダモノでした。|publisher= Wcomics（ダブリューコミックス）|accessdate=2017-06-23}}</ref>。
== 登場角色 ==
; {{nihongo|小南靜歌|小南 静歌（こみなみ しずか）|聲：花影螢}}
: 性格老實，思想保守的女大學生。不擅長跟男性相處。
; {{nihongo|霧島涼|霧島 涼（きりしま りょう）|聲：皐月栞}}
: 跟靜歌在聯誼的地方相識的大學生。外表看起來是颯爽美人，其實是女裝男子。
; {{nihongo|倉谷奏介|倉谷奏介（くらたに そうすけ）|聲：金部影人}}
: 跟靜歌同期的大學生。性格沉穩，喜歡靜歌，總是很關心靜歌。
; {{nihongo|櫻井堇|桜井 菫（さくらい すみれ）|聲：原舞香}}
: 跟涼同期的大學生。是選美優勝者，性格親切又溫柔。
