'''儒格保安6型'''（{{Lang-en|Ruger Security-Six}}）及其衍生型'''現役6型'''（{{Lang-en|Service-Six}}）、'''快速6型'''（{{Lang-en|Speed-Six}}）是由[[斯特姆，儒格|斯特姆，儒格]]公司在1972年推出、直到1988年間生產的[[扳機#雙動操作|雙動操作]]式[[左轮手枪|左轮手枪]]產品系列。這些左輪手槍被推向警方執法、軍隊任務和民用自衛三個市場方向。發射，發射[[.357麥格農|.357 S&W馬格南]]或火力相對較弱的[[.38_Special|.38 S&W特種彈]]、{{Link-en|.38 S&W|.38 S&W}}、[[9×19mm魯格彈|9×19毫米「帕拉贝鲁姆」]]等多種[[手枪|手枪]][[子彈|子彈]]。<ref name="wg">{{cite web |url=http://world.guns.ru/handguns/hg210-e.htm |title=Ruger Security Six, Police Service Six, Speed Six and GS32-N revolvers (USA) |work=Modern Firearms & Ammunition |deadurl=yes |archiveurl=https://web.archive.org/web/20100701154653/http://modernfirearms.net/handguns/hg210-e.htm |archivedate=2010-07-01 |accessdate=2008-12-14 }}</ref>

== 研發和歷史 ==
儒格保安6型及其衍生型的推出標誌著斯特姆，儒格首次嘗試進入雙動左輪手槍市場。該公司的早期設計是[[柯爾特單動式陸軍轉輪手槍|柯爾特「和平捍衛者」]]樣式的單動左輪手槍。為了降低{{Link-en|生產成本價值理論|Cost-of-production theory of value|生產成本}}，儒格採用了{{Link-en|熔模鑄造|Investment casting}}工藝以製成其武器以上的大多數零件。<ref name="ch">Tong, David. [http://www.chuckhawks.com/ruger_security_six.htm “Ruger Security-Six .357 Magnum Revolver”] Chuck Hawks Web site. Accessed December 14, 2008.</ref>與儒格的所有槍械一樣的是，保安6型左輪手槍的設計堅固，採用了大型重型部件，以提高耐用性，並可以該公司擅長的熔模鑄造生產。<ref name="ch"/><ref name="rrr">Crumpston, Mike. [http://findarticles.com/p/articles/mi_m0BTT/is_149_24/ai_65910639 "Revisiting Ruger's Revolvers"] BNET Web site. Accessed December 14, 2008.</ref>“6型系列”系列因其基本功能、堅固的結構和有競爭力的價格而獲得了銷售上的成功。<ref name="ch"/>

各種型號獲多間美國政府機構所採用，比如前[[美国移民及归化局|移民及歸化局]]、[[美國郵政署|郵政署]]、{{Link-en|美國邊境巡邏隊|United States Border Patrol|邊境巡邏隊}}和眾多警察機構。保安6型及其直系衍生型亦成為了眾多警察部門的標準公發制式武器，此外還有許多該槍出口到海外。<ref name="wg"/>雖然儒格保安6型系列自1988年以來已經停產，但生產期間總共生產了150多萬把左輪手槍，它們仍然受到人們所愛好和崇敬，並且在二手市場上備受追捧。<ref name="wg"/><ref name="rrr"/>

GP100取代了儒格產品系列中的保安6型。

== 特徵 ==
保安6型在基本設計上或多或少與其衍生型有相同之處，只是在瞄準具（固定型或可調節）和[[機匣|底把]] （圓形或方形握把）方面存在著細微的差別。儘管尺寸而言是為中等底把，但是保安6型的強度比[[史密斯威森M19左輪手槍|史密斯威森M19左輪手槍]]等競爭對手較大；這得益於儒格的底把較厚，並無側板切口，更堅固的槍管支撐可防止6英寸型號的左輪手槍引發毀滅性的槍管膛炸，內部零件較大亦較強固，並且在[[直径|直径]]有所加大的[[弹巢|弹巢]]以上裝有偏移式[[槍栓|槍栓]]閉鎖槽口。最初，該新型左輪手槍採用{{Link-en|烤藍|Bluing (steel)}}[[碳鋼|碳鋼]]所製成；其後1975年，所有型號的[[不鏽鋼|不鏽鋼]]版本都獲加入到產品陣容當中。<ref name="wg"/>採用6發[[弹巢|弹巢]]的保安6型系列是首批現代左輪手槍設計之一，採用了螺旋狀[[弹簧|弹簧]]為動力、經傳遞桿擊發系統所推動的[[擊錘|擊錘]]，並可裝填各種[[中央式底火|中央式底火]]彈藥，包括[[.357麥格農|.357 S&W馬格南]]和[[.38_Special|.38 S&W特種彈]]，以及{{Link-en|.38 S&W|.38 S&W}}和[[9×19mm魯格彈|9×19毫米「帕拉贝鲁姆」]]（又稱：9×19毫米「魯格」）。<ref name="wg"/><ref name="rrr"/>所有保安6型系列左輪手槍都配備了[[代工生产|代工生产]]商（OEM）所提供的制式[[木材|木製]]{{Link-en|手槍握把|Pistol grip|握把}}。<ref name="wg"/>全部木製握把是由現已解散的承包商、位於[[新罕布什尔州|新罕布什尔州]]的W.F.勒克生產公司為儒格所生產。大多數的這些木製握把都採用了壓滾花的菱形格子防滑表面側板，可也有一部份紀念型號與裝上了光滑而未被壓出方格的[[核桃|核桃]]木握把側板一併出貨。原廠亦有提供超大型核桃木射靶／戰鬥握把作為客戶選擇。自1980年代起，一部份快速6型和保安6型也裝有[[橡膠|橡膠]]製帕茨米亞（{{Lang-en|Pachmayr}}）握把，握把以內裝有銀色的儒格標誌。

保安6型的另一個特徵是可直接拆卸，除了握把[[螺絲|螺絲]]需要利用的平頭螺絲刀、硬幣或彈殼底緣以拆卸下來以外，無需任何的工具。

保安6型的拆卸步驟如下：
# 確保槍枝已經退彈。
# 擰出握把螺絲。
# 取下握把。
# 壓縮主彈簧。
# 插入銷釘並使主彈簧減壓。
# 拆下主彈簧。
# 拆下擊錘銷。
# 拆下擊錘。
# 鬆開並且拆下扳機組件。
# 鬆開並且拆下[[弹巢|弹巢]]。

[[File:Ruger357.jpg|thumb]]和[[橡膠|橡膠]]握把的儒格保安6型。]]

=== 保安6型 ===
於1972年推出的保安6型是當時的新系列的原初型號。<ref>[http://www.ruger.com/service/productHistory/RE-SPSSix.html Ruger Security Six, Single Six, Service Six Serial Number History]</ref>大部分的這款槍型都製有可調節的瞄準具，儘管有一部份早期型號配以固定式瞄準具出售。保安6型還可以藉由訂購裝上制式或“{{Link-en|全國步槍協會精密手槍|NRA Precision Pistol|射靶}}”（戰鬥）樣式方形尾部握把。幾乎所有的保安6型都配備了可發射[[.357麥格農|.357 S&W馬格南]]子彈的[[弹巢|弹巢]]，這也可讓它裝填彈長較短的[[.38_Special|.38 S&W特種彈]]。儒格還可因應一些警察的訂單，通過裝配不同的弹巢以只能容納.38 S&W特種彈彈藥，使該槍就只能發射.38 S&W特種彈。保安6型可選用的槍管長度包括69.85毫米（2{{frac|3|4}}英吋）、101.6毫米（4英吋）和152.4毫米（6英吋）。<ref name="wg"/><ref name="rrr"/>

.357 S&W馬格南搭配4英吋槍管的型號是直到前[[美国移民及归化局|美国移民及归化局]]的制服人員<ref name="Former I&NS Special Agent">前美国移民及归化局[[特務|特務]]。</ref>以及{{Link-en|美國邊境巡邏隊|United States Border Patrol}}的巡邏探員採用[[.40_S&W|.40口徑]][[半自動手槍|半自動手槍]]以前，這兩間機構都採用的標準配備。<ref name="Former Border Patrol Agent">前邊境巡邏隊探員。</ref>

=== 現役6型 ===
生產了數個月以後，儒格改稱保安6型的固定式瞄準具版本為現役6型，或是“警用現役6型”。這主要的原因是營銷決策，並試圖利用當時執法機關的現役左輪手槍市場獲得豐厚的利潤。現役6型配備了可發射[[.357麥格農|.357 S&W馬格南]]子彈的[[弹巢|弹巢]]，但亦可因應一些警察的訂單，製成[[.38_Special|.38 S&W特種彈]]和[[9×19mm魯格彈|9×19毫米魯格彈]]（帕拉贝鲁姆彈）的弹巢版本。<ref name="ch"/>[[美军|美國軍方]]簽訂了配備固定式瞄準具的.38特種彈型號，為握把尾部新增了掛繩環，並且將其命名為M108。它倆都發配給空軍機員和[[宪兵|宪兵]]用以取代日漸老化的[[史密斯威森軍警型左輪手槍|史密斯威森軍警十型]]左輪手槍。9毫米的型號配備了圓筒[[膛室|膛室]]，並在弹巢底部鑿出空間以將子彈掛在彈殼口而非底緣以上，利用獲得[[专利|专利]]的彈簧式{{Link-en|月型夾|Moon clip}}，一個具有多個金屬凸緣與凹槽的夾子以勾住子彈的勾槽，以便抽出發射過後的彈殼。這些改動和配件使得「無緣式」（Rimless）彈殼的9×19毫米子彈可裝填於左輪手槍的設計以上。現役6型可選用的槍管長度包括69.85毫米（2{{frac|3|4}}英吋）和101.6毫米（4英吋）。<ref name="ch"/>9×19毫米的型號亦以M109之名在市面銷售。

=== 快速6型 ===
[[File:Rugerspeedsix.JPG|thumb]][[口徑|口徑]]快速6型衍生型，利用{{Link-en|月型夾|Moon clip|半月夾}}以裝填無緣式子彈。]]
快速6型採用固定式瞄準具與圓形尾部握把，有[[.357麥格農|.357 S&W馬格南]]、[[.38_Special|.38 S&W特種彈]]、{{Link-en|.38 S&W|.38 S&W}}（.38-200）和[[9×19mm魯格彈|9×19毫米「魯格」]]的的弹巢版本，旨在供便衣探員和其他想要使用能更隱蔽的手槍的人士使用。這款型號的標準[[槍管|槍管]]長度與現役6型的相同，但在某些執法機關的合同訂單當中——例如[[美國郵政署|美國郵政署]]——亦包括3英寸長度版本（型號為GS33-PS）。.357 S&W馬格南搭配3英吋槍管的型號是直到前[[美国移民及归化局|美国移民及归化局]]的特別探員<ref name="Former I&NS Special Agent"/>以及執行便衣任務的{{Link-en|美國邊境巡邏隊|United States Border Patrol}}的巡邏探員採用[[.40_S&W|.40口徑]][[半自動手槍|半自動手槍]]以前，這兩間機構都採用的標準配備。<ref name="Former Border Patrol Agent">前邊境巡邏隊探員。</ref>{{Link-en|.38 S&W|.38 S&W}}衍生型（在英國又被稱為.380英國或.38-200）還配備了軍用式掛繩，並出售給在印度的執法機構。
