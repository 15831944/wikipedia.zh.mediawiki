'''儒利安·伯勒斯貝克'''（{{lang-de|'''Julian Pollersbeck'''}}，{{bd|1994年|8月16日}}）是[[德國|德國]]的一位足球運動員，在場上的位置是守門員。他現在效力於[[德國足球甲級聯賽|德國足球甲級聯賽]]球隊[[漢堡足球俱樂部|漢堡]]<ref>{{cite web |url=http://www.worldfootball.net/player_summary/julian-pollersbeck/ |title=Julian Pollersbeck |website=worldfootball.net |publisher=HEIM:SPIEL |access-date=2016-10-02}}</ref>。

== 職業生涯 ==
=== 俱樂部 ===
伯勒斯貝克出身於[[上巴伐利亞行政區|上巴伐利亞行政區]]，他最初在出身地的球隊SV DJK 愛莫廷接受青訓。在16歲時，伯勒斯貝克轉會至[[德國足球丙級聯賽|德國足球丙級聯賽]]球隊[[瓦克爾布格豪森|瓦克爾布格豪森]]，該球隊也是伯勒斯貝克的出身地[[阿爾特廷|阿爾特廷]]最強的球隊。在2011/12賽季，伯勒斯貝克代表瓦克爾布格豪森的U19代表隊參加巴伐利亞州的A級少年聯賽，此外他還代表瓦克爾布格豪森B隊參加[[巴伐利亞足球州級聯賽|巴伐利亞州南部聯賽]]。2012/13賽季，伯勒斯貝克代表瓦克爾布格豪森B隊七次出場巴伐利亞州南部聯賽。他在這個賽季還三次被選入瓦克爾布格豪森一線隊名單，參加了德國足球丙級聯賽，但只是替補。這一賽季中期，伯勒斯貝克轉會[[凯泽斯劳滕足球俱乐部|凯泽斯劳滕]]。在2014/15賽季，伯勒斯貝克升入凯泽斯劳滕一線隊，並在2014年11月獲得了一份職業合同<ref>{{cite web |url=https://www.der-betze-brennt.de/news/7740-julian-pollersbeck-wird-fck-profi.php |title=Julian Pollersbeck wird FCK-Profi |website=der betze brennt |publisher=der betze brennt |access-date=2017-07-18}}</ref>。2016年9月11日，他首次在德國足球乙級聯賽的賽事中出場<ref>{{cite web |url=http://www.kicker.de/news/fussball/2bundesliga/spieltag/2-bundesliga/2016-17/4/3317583/spielanalyse_sv-sandhausen-1604_1-fc-kaiserslautern-37.html |title=Rollentausch: Pollersbeck bleibt gesätzt |website=kicker.de |publisher=kicker |access-date=2017-07-18}}</ref>。伯勒斯貝克此後和凯泽斯劳滕簽訂了一份至2020年的合同<ref>{{cite web |url=http://www.kicker.de/news/fussball/2bundesliga/vereine/661184/artikel_fck-bindet-torwarttalent-pollersbeck.html |title=FCK bindet Torwarttalent Pollersbeck |website=kicker.de |publisher=kicker |access-date=2017-07-18}}</ref>。

在2017/18賽季，伯勒斯貝克轉會[[德國足球甲級聯賽|德國足球甲級聯賽]]球隊[[漢堡體育俱樂部|漢堡]]，並簽訂了一份至2021年6月30日的為期四年的合同<ref>{{cite web |url=https://www.hsv.de/news/meldungen/allgemein/juni-2017/hsv-verpflichtet-pollersbeck/ |title=HSV verpflichtet Julian Pollersbeck |website=HSV |publisher=HSV |access-date=2017-07-18 |archive-url=https://web.archive.org/web/20170817163231/https://www.hsv.de/news/meldungen/allgemein/juni-2017/hsv-verpflichtet-pollersbeck/ |archive-date=2017-08-17 |dead-url=yes }}</ref>。

=== 國家隊 ===
伯勒斯貝克是[[德國21歲以下國家足球隊|德國U21國家隊]]的一員。他首次代表德國國家隊出場是於2016年11月10日在[[柏林|柏林]]舉辦的一場德國對土耳其的比賽，這場比賽德國隊以1:0獲勝。[[斯特凡·昆茨|斯特凡·昆茨]]將伯勒斯貝克選入了[[2017年欧洲U-21足球锦标赛|2017年欧洲U-21足球锦标赛]]德國隊大名單並任用他為德國隊的主力守門員。在6月27日德國對英格蘭的半決賽比賽中，伯勒斯貝克在點球大戰中扑出了英格蘭的兩個點球，是德國隊進入決賽的功臣<ref>{{cite web |url=http://www.kicker.de/news/fussball/u21emspielplan/u-21-europameisterschaft/2017/4/3796216/spielbericht_england-3708_deutschland-3096.html |title=Deutsche U 21 dank Elfmeterheld Pollersbeck im Finale |website=kicker.de |publisher=kicker |access-date=2017-07-18}}</ref>。在德國隊於決賽戰勝西班牙隊奪冠之後，伯勒斯貝克被歐洲足總（UEFA）評為本屆賽事最佳門將<ref>{{cite web |url=http://www.uefa.com/under21/news/newsid=2483058.html |title=Official Under-21 Team of the Tournament |website=UEFA.com |publisher=UEFA |access-date=2017-07-18}}</ref>。
