'''硬膠'''是一個常用於[[香港|香港]][[網上社區|網上社區]]的詞語，原本是指「[[聚苯乙烯|聚苯乙烯]]」，其後有人用此詞取其[[諧音|諧音]]代替「戇{{Link-yue|㞗}}」（意謂「傻屌」）這個[[髒話|粗口]]，形容'''形象差劣'''的[[事物|事物]]。

== 出處 ==
「硬膠」早於[[周星驰|周星驰]]電影《[[整蠱專家|整蠱專家]]》已出現雛型。戲中一台詞「超級戇膠膠」可見，'''「膠（gaau<sup>1</sup>）」'''字是用作借代[[粵語髒話|粵語粗口]]的'''「{{Link-yue|㞗}}」'''字（音 /gau<sup>1</sup>/）。

現時 「硬膠」一詞的使用已遍布香港網絡社區。由於大部分討論區禁止網民使用粗言穢語，有些網民便把「戇㞗」諧音成「硬膠」，但也有可能被檢舉。網民並把發言古怪、發帖毫無根據、有違道理的會員定性為「硬膠」始祖。某程度上硬膠文化與「[[無厘頭文化|無厘頭文化]]」近似。

== 發展 ==
硬膠風氣始於[[高登討論區|高登討論區]]。當時的高登[[網站管理員|管理員]]仍為「[[wikia:evchk:DrJim|DrJim]]」，但在管理權易手至「[[wikia:evchk:Fevaworks|Fevaworks]]」（科域盈創有限公司）後，硬膠文化反更根深柢固。如高登的網頁標題曾為{{fact|「香港高登 - 全港至硬嘎討論區」，其中「硬」是指「硬件」抑或「硬膠」，只有Fevaworks才知道。}}

許多高登用語都受「硬膠文化」影響，如「[[wikia:evchk:硬膠之星|硬膠之星]]」；甚至把硬膠圖像化，以小丑圖示表示，此後「[[小丑神|小丑神]]」更成為「硬膠」的同義詞。

== 影響 ==
=== 網絡媒體方面 ===
在[[香港高登討論區|香港高登討論區]]內，「硬膠」2字的使用次數非常頻密，{{fact|「硬膠文化」已成為「[[高登文化|高登文化]]」之核心}}。[[高登|高登]]內很多網民往往在自己的[[網名|網名]]上加入“膠”一字。{{fact|由於出現「[[wikia:evchk:高登化|高登化]]」現象，這個情況更在香港其他討論區出現。但在一些管理極為嚴格的論壇，[[版主|版主]]往往把膠字定性為[[粗口|粗口]]，予以封殺，並把使用「膠」字的版友作[[wikia:evchk:停版|停版]]處理。}}

===傳統媒體方面===
由於不少有寫[[網誌|網誌]]習慣的報章專欄作者如[[林忌|林忌]]、[[黃世澤|黃世澤]]等常在其文章使用「硬膠」2字，因此「硬膠」除影響部分網民的用字外，其影響力亦逐漸延伸至傳統媒體。例如在[[香港|香港]]《[[蘋果日報|蘋果日報]]》財經版撰寫專欄的作者[[孫柏文|孫柏文]]在其專欄文章《金手指》不時用到「膠」或「硬膠」等字眼來批判某些人或事的光怪陸離作風<ref>[http://appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20071121&sec_id=15307&subsec_id=15320&art_id=10444835&cat_id=5885349&coln_id=5673918 孫柏文, 「金手指: 第壹膠黨『[[公民黨|公民黨]]』」, 《蘋果日報》, 11/21/2007] {{webarchive|url=https://web.archive.org/web/20071122085151/http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20071121&sec_id=15307&subsec_id=15320&art_id=10444835&cat_id=5885349&coln_id=5673918 |date=2007-11-22 }}</ref>。此外，《蘋果日報》亦不時出現「[[wikia:evchk:膠系術語|膠系術語]]」如「[[wikia:evchk:派膠|派膠]]」等<ref>[http://appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20071121&sec_id=15307&subsec_id=15320&art_id=10444837&cat_id=6271926&coln_id=6339611 尹思哲, 「案內人隨筆: 環保成本」, 《蘋果日報》, 11/21/2007] {{webarchive|url=https://web.archive.org/web/20071123104905/http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20071121&sec_id=15307&subsec_id=15320&art_id=10444837&cat_id=6271926&coln_id=6339611 |date=2007-11-23 }}</ref>。

而曾為長期高登會員的[[林忌|林忌]]， 於2007年發佈改編歌曲《[[wikia:evchk:福佳始終有你|福佳始終有你]]》時，把歌詞中的「膠」字定位為「荒謬」之異體，後來在他出版的著作《[[wikia:evchk:潮童不宜|潮童不宜]]》中，內容更是「膠」字橫行，甚似高登政治風氣盛行時之討論。
