'''硬件加速'''是指在[[计算机|计算机]]中通过把计算量非常大的工作分配给专门的[[硬件|硬件]]来处理以减轻[[中央处理器|中央处理器]]的工作量之技术。尤其是在图像处理中这个技术经常被使用。

[[雷神之锤3|雷神之锤3]]是第一个必须要求硬件加速的[[三维|3D]]游戏。[[Google_Chrome|Google Chrome]]浏览器也设置了“硬件加速”选项，用户可根据需求开启或关闭此功能。

== 技术 ==
中央处理器的结构使得它能够在短时间内完成各种各样不同的指令。它能够处理什么指令主要由[[软件|软件]]限制。但是由于中央处理器的结构有些重复任务无法非常有效和迅速地被处理。由于软件的原因处理器优化的可能性有限。

通过使用专门为这样的重复任务设计的特殊硬件元件（芯片或者处理器）可以解决这个问题。这些特殊硬件元件不必像中央处理器那样灵活，因此它们的硬件设计就已经顾及了优化处理这些特殊问题的需要，这样一来中央处理器有时间去处理其它任务。

有些任务能够通过把它们分解为上千小任务非常有效地被解决。比如对一定的频率带做[[傅里叶变换|傅里叶变换]]或者[[渲染|渲染]]一小块图像。这些小任务可以互相之间不相关地平行计算。通过大量平行计算，即适用大量平行运行的小处理器来处理这些特殊任务总的计算速度可以大大提高。在许多情况下计算速度随平行处理器的数量线性提高。比如在{{lang|en|[[GeForce_200|GeForce 200]]}}图像卡上192个[[流处理器|流处理器]]平行运行<ref>[http://www.nvidia.cn/object/geforce_gtx_260_cn.html Nvidia官方网站上对GTX260的介绍]</ref>。

从有效利用能源的角度出发这样的平行计算也有意义。能源使用随平行处理器的数量线性提高，而随处理器频率成平方比提高。因此通过平行运算处理器的频率不必过高，使用的能量也比较少。

== 显示卡 ==
在计算机技术起步的时候屏幕显示是中央处理器的任务。后来一块专门的芯片被用来显示荧光屏上的字符。再后来图像芯片还开始管理自己的图像内存和显示颜色。随着[[图形用户界面|图形用户界面]]的出现[[显示卡|显示卡]]开始处理简单的、一再重复出现的任务，比如画方块、直线和圆。操作系统把这些图像指令传给特殊的显示卡驱动程序，再由这些驱动程序交给显示卡处理。中央处理器雖然也可以在软件状态下处理这些任务，但是这样的话计算机的速度就要慢得多。

1996年出现了所谓的三维显示卡。一开始它们的任务是非常快地把三角形（由多面体组成的三维物体的基本结构）写到图像内存中去并为它们覆盖上相应的表面结构（比如{{lang|en|3dfx}}的{{lang|en|Voodoo}}卡<ref>[http://www.thedodgegarage.com/3dfx/] 与二维图像卡平行运行的3D Voodoo显示卡</ref>）。2000年初计算三维模型的任务也转到了显示卡上去了（所谓的{{lang|en|[[T&L|T&L]]}}元件）。从约2006年开始显示卡也越来越多地开始通过所谓的[[物理處理器|物理處理器]]来计算三维模型的物理运动<ref>[http://www.nvidia.cn/object/nvidia_physx_cn.html] NVIDIA的物理处理器PhysX</ref>。

== 其它应用 ==
[[File:80386with387.JPG|thumb]]
计算机工业不断推出新的逻辑元件来减轻主处理器的工作，解决特殊任务<ref>[http://focus.ti.com/docs/solution/folders/print/342.html] 一个典型笔记本中的特殊TI线路</ref>。其中包括处理{{lang|en|[[MP3|MP3]]}}数据、处理录像数据<ref>{{cite web |url=http://ati.amd.com/technology/avivo/features.html |title=存档副本 |accessdate=2009-04-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20090428171300/http://ati.amd.com/technology/Avivo/features.html |archivedate=2009-04-28 }} ATI专门处理高清晰度电视信息的Avivo</ref>、产生声频信号、[[数字信号处理器|数字信号处理器]]和[[人工智能加速器|人工智能加速]]等。1980、90年代里使用的数学[[辅助处理器|辅助处理器]]也可以算在里面。这些数学辅助处理器协助中央处理器计算[[浮点数|浮点数]]。它们可以被插在[[主板|主板]]上的专门插座里。今天它们和主处理器被集成到同一[[集成电路|芯片]]上去了。
