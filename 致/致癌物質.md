'''致癌物質'''<ref>[http://www.toa365.com.tw/cancer_news4_06.htm 致癌物質的介紹] {{webarchive|url=https://web.archive.org/web/20111117193306/http://www.toa365.com.tw/cancer_news4_06.htm |date=2011-11-17 }}</ref>（{{lang-en|'''Carcinogen'''}}）是指任何會直接導致生物體產生[[癌症|癌症]]的[[物質|物質]]，包括化学物质、病毒、[[放射性核素|放射性核素]]等。這些物質进入机体后會直接或间接使机体[[細胞|細胞]]受到損害，导致[[生物大分子|生物大分子]]异常、细胞代偿性增生、[[原癌基因|原癌基因]]活化、[[抑癌基因|抑癌基因]]失活、[[DNA|DNA]]损伤修复缺陷及[[表观遗传|表观遗传]]改变等，從而诱发癌症。

[[世界卫生组织|世界卫生组织]]轄下的[[國際癌症研究機構|國際癌症研究機構]]将致癌物质按照危險程度分为4類：

* [[1類致癌物|1類致癌物]]：对人体有明确致癌性的物质或混合物，如[[大气污染|大气污染]]、[[日晒床|日晒床]]、[[黃麴毒素|黃麴毒素]]、[[砒霜|砒霜]]、[[石棉|石棉]]、[[六价铬|六价铬]]、[[二噁英|二噁英]]、[[甲醛|甲醛]]、[[白酒|酒精饮料]]、[[烟草|烟草]]及[[槟榔|槟榔]]等。
* [[2A類致癌物|2A類致癌物]]：对人体致癌的可能性较高的物质或混合物，在[[动物实验|动物实验]]中发现充分的致癌性证据。对人体虽有理论上的致癌性，而实验性的证据有限。如[[丙烯酰胺|丙烯酰胺]]、无机[[铅|铅]]化合物及[[氯霉素|氯霉素]]等。
* [[2B類致癌物|2B類致癌物]]：对人体致癌的可能性较低的物质或混合物，在动物实验中发现的致癌性证据尚不充分，对人体的致癌性的证据有限。用以归类相比二类A致癌可能性较低的物质。比如[[咖啡|咖啡]]、[[泡菜|泡菜]]<ref name="WHOIARCVol51">{{cite web |url=http://monographs.iarc.fr/ENG/Monographs/vol51/volume51.pdf |title=存档副本 |accessdate=2006-09-19 |0=2006-09-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20060927172427/http://monographs.iarc.fr/ENG/Monographs/vol51/volume51.pdf |archivedate=2006-09-27 }}</ref>、手机辐射<ref>http://www.iarc.fr/en/media-centre/pr/2011/pdfs/pr208_E.pdf </ref>、[[氯仿|氯仿]]、[[滴滴涕|滴滴涕]]、[[敌敌畏|敌敌畏]]、[[萘|萘]][[卫生球|卫生球]]、[[镍|镍]]金属、[[硝基苯|硝基苯]]、[[柴油|柴油]]燃料及[[汽油|汽油]]等。
* [[3類致癌物|3類致癌物]]：对人体致癌性尚未归类的物质或混合物，对人体致癌性的证据不充分，在动物实验中致癌性证据不充分或有限。或者有充分的实验性证据和充分的理论机理表明其对其他某些动物有致癌性，但对人体没有同样的致癌性。如[[茶|茶]]<ref name="WHOIARCVol51"/> [[苯胺|苯胺]]、[[苏丹红|苏丹红]]、[[咖啡因|咖啡因]]、[[二甲苯|二甲苯]]、[[糖精|糖精]]及其盐、[[地西泮|安定]]、[[氧化铁|氧化铁]]、有机铅化合物、[[静电|静电]][[磁场|磁场]]、[[三聚氰胺|三聚氰胺]]及[[汞|汞]]与其无机化合物等。
* [[4類致癌物|4類致癌物]]：对人体可能没有致癌性的物质，缺乏充足证据支持其具有致癌性的物质。如[[己内酰胺|己内酰胺]]。

[[联合国|联合国]]的[[全球化学品统一分类和标签制度|全球化学品统一分类和标签制度]]將致癌物質分為兩類：
* 1組：被確認是或者相信對人類有致癌潛在可能
** 1A組：測驗基於人類證據
** 1B組：測驗基於動物證據
* 2組：懷疑對人類的致癌物質

[[美國衛生及公共服務部|美國衛生及公共服務部]]將致癌物質分為2組：
*被確認是對人類有致癌潛在可能
*有理由地懷疑對人類的致癌物質

== 致癌物質種類 ==
===[[化學物質|化學物質]]===
====按作用方式分类====
按照作用方式可以将化学致癌物质分为直接致癌物质、间接致癌物质与促癌物{{NoteTag|本身无致癌作用，但可以与致癌物协同作用，诱发突变细胞克隆扩增或加速癌细胞发展，进而促进癌症发生的物质。}}三种。<ref name="D">{{Cite book | author = 楼宜嘉 | title = 药物毒理学（第四版）| location = 北京 | publisher = 人民卫生出版社 | date = 2016-2 | ISBN = 978-7-117-22371-3 | accessdate = 2017-9 | language = zh }}</ref>
*直接致癌物：[[烷化剂|烷化剂]]、一些[[金属|金属]]及其[[盐_(化学)|盐]]、[[放射性物質|放射性物質]]等。
*间接致癌物：[[多环芳香烃|多环芳香烃]]、[[杂环化合物|杂环芳香烃]]、[[芳香胺|芳香胺]]类、含一个或多个[[硝基|硝基]]取代的[[芳香族化合物|芳香族化合物]]、烷基[[肼|肼]]类、[[氨基甲酸酯|氨基甲酸酯]]类、[[黃麴毒素|黃麴毒素]]、[[免疫抑制剂|免疫抑制剂]]、[[烟碱|烟碱]]、{{le|檳榔鹼|Arecoline}}、[[石棉|石棉]]等。
*促癌物：[[佛波酯|佛波酯]]、[[巴豆|巴豆油]]、[[煤焦油|煤焦油]]中的[[酚|酚]]类、[[激素|激素]]、[[卤代烃|卤代烃]]、[[有機氯殺蟲劑|有機氯殺蟲劑]]、[[多氯联苯|多氯联苯]]等。
====按致癌作用机制分类====
化学致癌物进入机体后，有些直接与机体[[遗传物质|遗传物质]]结合导致[[体细胞|体细胞]]癌变；有些则不作用于遗传物质，而是通过其他方式使体细胞发生癌变。因此，根据致癌作用机制的不同可将化学致癌物质分为遗传毒性致癌物与非遗传毒性致癌物。
*遗传毒性致癌物：包括绝大多数的直接致癌物与间接致癌物（如黃麴毒素、芳香族化合物、烟碱、甲醛等）。此类化学物质经[[代谢|代谢]]进入细胞后大多能与细胞[[DNA|DNA]]共价结合。
*非遗传毒性致癌物：包括促癌物、免疫抑制剂、一些特殊的固态物质（如石棉、铀矿或[[赤铁矿|赤铁矿]]粉尘）、[[过氧化物酶体|过氧化物酶体]]增生剂（如{{le|氯贝丁酯|Clofibrate}}、[[邻苯二甲酸二(2-乙基己)酯|邻苯二甲酸二(2-乙基己)酯]]等）、细胞毒物（如{{le|氮川三乙酸|Nitrilotriacetic acid}}）<ref name = "D"/>

===[[微生物|微生物]]===
*[[乙肝病毒|乙肝病毒]]和{{le|丙肝病毒|Hepatitis C virus}}与肝癌的发生密切相关。<ref name = "p"/>
*[[HTLV-1|人類嗜T細胞病毒]]可以導致成人T細胞[[白血病|白血病]]。
*[[人類乳突病毒|人類乳突病毒]]中的高危型（主要是16型与18型）可以導致[[子宮頸癌|子宮頸癌]]；57b型与[[鼻咽癌|鼻咽癌]]的发生密切相关；12型与32型等与[[口腔癌|口腔癌]]有联系；6型与11型导致的[[尖锐湿疣|尖锐湿疣]]则可诱发[[陰莖癌|陰莖癌]]。<ref name = "p"/>
*[[皰疹病毒|皰疹病毒]]科中，2型[[单纯疱疹病毒|单纯疱疹病毒]]与[[子宮頸癌|子宮頸癌]]有一定联系；{{le|卡波西肉瘤相關皰疹病毒|Kaposi's sarcoma-associated herpesvirus}}（HHV-8）能夠導致[[卡波西氏肉瘤|卡波西氏肉瘤]]；[[EB病毒|EB病毒]]可以導致{{le|伯奇氏淋巴瘤|Burkitt's lymphoma}}（即非洲儿童恶性淋巴瘤）、[[何杰金氏病|何杰金氏病]]与{{le|淋巴组织增生性疾病|Lymphoproliferative disorders}}，同时与[[鼻咽癌|鼻咽癌]]有着较为紧密的联系。<ref name = "p">{{Cite book | author1 = 刘晶星|author2 = 李凡 | title = 医学微生物学（第八版）| location = 北京 | publisher = [[人民卫生出版社|人民卫生出版社]] | date = 2013-3 |ISBN = 978-7-117-17152-6 | accessdate = 2017-10 | language =zh }}</ref>
*[[幽門螺桿菌|幽門螺桿菌]]能引发胃癌。

==接触途径==
*[[吸煙|吸煙]]：[[香烟|香烟]]主要成份有[[尼古丁|尼古丁]]和少量[[丙酮|丙酮]]。
*[[饮酒|饮酒]]：酒类主要成分为[[乙醇|乙醇]]，大量食用會損害[[肝臟|肝臟]]。乙醇先轉為[[乙醛|乙醛]]，乙醛有[[麻醉|麻醉]][[神經|神經]]作用。
*长期使用某些药物：如长期使用[[保泰松|保泰松]]可抑制[[骨髓|骨髓]]造血功能进而导致[[白血病|白血病]]、长期服用[[非那西汀|非那西汀]]或[[阿司匹林|阿司匹林]]等会增加[[膀胱癌|膀胱癌]]的风险、肾脏病变患者长期滥用[[非甾体抗炎药|非甾体抗炎药]]可增加泌尿系统肿瘤的风险、[[蛋白同化激素|蛋白同化激素]]如[[苯丙酸诺龙|苯丙酸诺龙]]等长期使用易诱发[[肝癌|肝癌]]、[[氯米芬|氯米芬]]则有造成[[卵巢癌|卵巢癌]]的风险、一些细胞毒性[[抗肿瘤药|抗肿瘤药]]则会诱发继发性的癌症等。<ref name="D"/>
*[[饮食|饮食]]：[[硝酸鹽|硝酸鹽]]來自[[土壤|土壤]]，經過[[蔬菜|蔬菜]]的吸收、亞硝酸鹽來自貯存的[[肉類|肉類]]，例如[[火腿|火腿]]、[[香腸|香腸]]、[[熱狗|熱狗]]及[[臘肉|臘肉]]等等，[[胺類|胺類]]化合物則存於[[海產|海產]]，例如[[秋刀魚|秋刀魚]]及[[魷魚|魷魚]]等中。硝酸鹽與胺類共同食用，會形成亞硝酸胺致癌物。亞硝酸胺與[[口腔|口腔]]、[[食道|食道]]及[[胃癌|胃癌]]有關。另外，食物處理不當，包括高溫[[油炸|油炸]]、[[燒烤|燒烤]]及[[煎|煎]][[烤|烤]]等都會使到[[食物|食物]]產生[[多環芳香碳氫化合物|多環芳香碳氫化合物]]。此物質與[[呼吸道|呼吸道]]及腸胃道的癌症促成關係密切。煙燻類食物的致癌物質，包括於燃燒的材料，例如[[甘蔗|甘蔗]]及[[稻穀|稻穀]]等等，當中含有多環芳香碳氫化合物及芳香胺類致癌物質。[[醃製|醃製]]類、[[發酵|發酵]]類及乾製[[海產|海產]]食物也可能致癌。
*[[環境污染|環境污染]]
*微生物感染
*接触[[輻射|輻射]]或强[[磁場|磁場]]
