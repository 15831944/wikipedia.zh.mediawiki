《'''致青春'''》[[辛夷坞|辛夷坞]]的同名小说改编的一部[[电视剧|电视剧]]。于2016年7月11日起于[[东方卫视|东方卫视]]、[[安徽卫视|安徽卫视]]、[[北京卫视|北京卫视]]开始播出。<ref>{{cite web |title=剧版《致青春》定档711 杨玏陈瑶组微笑CP |url=https://ent.sina.cn/tv/tv/2016-06-21/detail-ifxtfrrf0748611.d.html?from=wap |website=新浪娱乐}}</ref>

== 剧情梗概 ==
女主角郑微深爱男生林静，考入大学后发现林静已出国留学。她与同舍女生之间结下了友谊，之后又意外爱上了学长陈孝正。而后者亦选择了出国留学。
几年后，林静和陈孝正先后进入郑微的生活，好友阮莞意外离世，促使郑微重新思考自己的人生道路。
