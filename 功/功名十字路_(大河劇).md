'''功名十字路'''（{{lang|ja|功名が辻～山内一豊の妻～}}）是[[日本放送協會|NHK]]於2006年製播的第45部[[NHK大河劇|大河劇]]，也是自1994年的[[花之亂|花之亂]]以後，再次以女性歷史人物當第一主角的大河劇。而本劇是繼1998年[[德川慶喜_(大河劇)|德川慶喜]]以後再次改編[[司馬遼太郎|司馬遼太郎]]的作品，而女主角[[仲間由紀惠|仲間由紀惠]]則是自2003年的[[武藏MUSASHI|武藏MUSASHI]]後再次參與大河劇演出，當時她是主演大河劇的最年輕女性演員（26歲2個月），此紀錄後來在2008年被[[宮崎葵|宮崎葵]]主演的[[篤姬_(電視劇)|篤姬]]（22歲1個月）改寫。

== 工作人員 ==
*原作：[[司馬遼太郎|司馬遼太郎]]
*劇本：[[大石靜|大石靜]]
*音樂：[[小六禮次郎|小六禮次郎]]

==演員==
===山内家===
====山内家一门====
*[[山內千代|千代]]（ちよ）
: 演员：[[仲間由紀惠|仲間由紀惠]]
: 本作女主角，山內一豐的正室，拥有过人的智慧，在幕后支持丈夫一丰。因为父母都死于战争，所以对战争很反感。
*[[山內一豐|山內一豐]]（やまうち かずとよ）
: 演员：[[上川隆也|上川隆也]]
: 本作男主角，性格愚直的“猪武者”，在妻子支持下不断取得功名，经历信长，秀吉，家康三个主君，最后成为土佐24万石领主。
*[[法秀尼|法秀尼]]（ほうしゅうに）
: 演员：[[佐久間良子|佐久間良子]]
: 一丰的母亲，千代的理解者，曾经收留幼年的千代。
*[[山内康豐|山内康豐]]（やまうち やすとよ）
: 演员：[[玉木宏|玉木宏]]
: 一丰的弟弟，曾经与出仕信长的哥哥反目，后来和解，并成为哥哥的理解者和助手。流浪中曾经和被幽闭的[[细川玉|细川玉]]相会，最后儿子国松成为一丰的养子继承山内家。
*[[若宮友興|若宮友興]]：[[宅麻伸|宅麻伸]]
*若宮友興之妻：[[木村多江|木村多江]]
