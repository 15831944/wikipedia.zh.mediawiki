《'''功夫侠'''》（{{lang-en|'''Kung Fu Man'''}}）2013年上映于中国大陆的功夫片，主演是[[陈虎_(演员)|陈虎]]、[[蒋梦婕|蒋梦婕]]、[[范妮莎·布兰奇|范妮莎·布兰奇]]等。<ref>{{cite web |url=http://ent.163.com/13/0719/09/944SNI0H00032DGD.html |title=陈虎亮相"功夫侠"首映 坦言：下次还跟基努合作 |accessdate=2013 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.163.com/13/0721/08/949SU0P700032DGD.html |title=《功夫侠》上演“小鬼当家”（图） |accessdate=2013 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/c/2013-07-19/11183967986.shtml |title= 《功夫侠》公映 演绎中国版“小鬼当家”  |accessdate=2013 |publisher=新浪网 |work= |language=zh-hans}}</ref>

==剧情==
该片讲述一个[[中国功夫|武术青年]]“陈平”营救遭到国际绑架组织绑架的孩童的故事。
