'''秀山土家族苗族自治县'''，简称'''秀山县'''，是[[重庆市|重庆市]]的一个[[自治县|自治县]]，位于重庆市东南部，距离重庆502公里，地处[[武陵山|武陵山]]脉中段，[[四川盆地|四川盆地]]东南缘的外侧，位于渝[[湖北省|鄂]][[湖南省|湘]][[贵州省|黔]]四省市交界地带，有“渝东南大门”之称，是重庆乃至[[成渝经济区|成渝经济区]]连接[[珠江三角洲地区|珠三角城市群]]的重要关节点。<ref>{{cite web |title=重庆市秀山自治县人民政府公众信息网-秀山概况 |url=http://xs.cq.gov.cn/zjxs/262/ |accessdate=2018-07-28}}</ref>

== 建置沿革 ==
自[[三国|三国]]至[[南北朝|南朝]]初，今秀山地属[[酉阳县|酉阳县]]，[[隋朝|隋]][[唐朝|唐]]属[[务川县|务川县]]，[[宋朝|宋]]起为[[羁縻|羁縻]][[思州|思州]]辖地，[[明朝|明朝]]时为[[四川布政使司|四川布政使司]]辖区内，设有平茶洞长官司、邑梅洞长官司、石耶洞长官司、地坝洞副长官司。[[清朝|清]][[乾隆|乾隆]]元年（1736年），为进行[[改土归流|改土归流]]，割[[酉阳县|酉阳县]]东南境石堤、宋农、晚森等[[土司|土司]]据地置秀山县，初隶[[四川省_(清)|四川省]][[黔彭直隶厅|黔彭直隶厅]]，后改隶[[川东道|川东道]][[酉阳直隶州|酉阳直隶州]]。<ref name=":0">{{Cite web|url=http://xs.cq.gov.cn/zjxs/264/|title=重庆市秀山自治县人民政府公众信息网-建制沿革|accessdate=2018-07-29|author=|date=|publisher=}}</ref><ref>{{Cite book|title=中国地方志集成·四川府县志辑48-光绪秀山县志|last=|first=|publisher=巴蜀书社|year=1992年|isbn=7-80523-478-7|location=成都|pages=}}</ref>

曾为湘鄂川黔革命根据地的一部分。民国24年（1935年）隶[[四川省_(中華民國)|四川省]]第八行政督察区。民国28年（1939年）曾确立为待成立的“青江省”省会，后计划撤销。民國35年（1946年），秀山縣之九江鄉劃歸貴州省[[松桃縣|松桃縣]]管轄（1952年重新确认），貴州省松桃縣[[訝駕鄉|訝駕鄉]]所屬的洪安（亦稱貴州塘）劃歸秀山縣管轄。民国38年（1949年）4月划归[[綏靖公署#川湘鄂邊區綏靖公署|川湘鄂边区绥靖公署]]<ref name=":0" /><ref>《內政部公函：關於勘劃川黔省界經呈奉國府令準備案函請查照辦理見復由》（中華民國三十五年五月二十日）， 《黔川兩省關於省界問題的調整等報告》（1942-1949 年），貴州省檔案館M8-1-3032</ref><ref>1952年6月11日川東人民行政公署民政廳【民政字第0159號】批覆，1952年11月10日貴州省民政廳【民民發字第273號】批覆</ref>。

1949年11月7日，[[中国人民解放军|中国人民解放军]][[中国人民解放军第二野战军|第二野战军]]三兵团攻破[[宋希濂|宋希濂]]部[[川东|川东]]防线，攻克秀山，并以秀山为跳板进入川东地区，拉开[[西南战役|西南战役]]的序幕。同年11月20日，秀山县人民政府成立，隶[[川东行政区|川东行署区]][[涪陵专区|涪陵专区]]，1950年1月划入新成立的[[酉阳专区|酉阳专区]]，1952年川东行署区与酉阳专区撤销，划归[[四川省|四川省]]涪陵专区。1983年4月，撤销秀山县，成立秀山土家族苗族自治县。1988年5月，改隶新成立的[[黔江地区|黔江地区]]。1997年，因[[重庆市|重庆]]直辖改隶直辖重庆市，由[[黔江开发区|黔江开发区]]代管，2001年撤销黔江开发区，由重庆市直管。<ref name=":0" /><ref>{{Cite book|title=中华人民共和国行政区划沿革地图集|last=|first=|publisher=[[中国地图出版社|中国地图出版社]]|year=2003年|isbn=978-7-5031-3184-4|location=北京|pages=}}</ref>

== 自然地理 ==
秀山土家族苗族自治县位于[[武陵山脉|武陵山脉]]二级隆起带南段，[[四川盆地|四川盆地]]东南缘的外侧，地处川东南褶皱带。县内河流切割强烈，地表起伏大，主要地形为山地丘陵，中部有盆地平坝。<ref name=":1">{{Cite web|url=http://xs.cq.gov.cn/zjxs/263/|title=重庆市秀山自治县人民政府公众信息网-自然地理|accessdate=2018-07-29|author=|date=|publisher=}}</ref>

秀山属亚热带湿润积分气候，年平均气温为16℃，年均降水量为1341.1毫米，年均日照时长为1213.7小时。<ref name=":1" />

县内资源丰富，主要矿产资源有[[锰|锰]]、[[汞|汞]]、[[煤矿|煤]]等，同时拥有丰富的水资源，年水力发电量达1.3亿千瓦时。县内森林覆盖面积达49.2%，以[[油桐|油桐]]、[[油茶|油茶]]最突出，特产的[[桐油|桐油]]被称为“秀油”，同时拥有丰富的草资源、野生动物资源、中药材资源。<ref>{{Cite web|url=http://xs.cq.gov.cn/zjxs/266/|title=重庆市秀山自治县人民政府公众信息网-自然资源|accessdate=2018-07-29|author=|date=|publisher=}}</ref>[[秀山花蛛|秀山花蛛]]以秀山地名命名<ref>{{Cite web|url=http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=14550|title=秀山花蛛|accessdate=2018-07-28|author=中国科学院动物研究所|date=|work=中国动物物种数据库|publisher=中国科学院微生物研究所|language=zh-CN}}</ref>。

== 交通 ==
秀山土家族苗族自治县地处重庆、湖北、湖南、贵州四省市交界地带，地理位置重要，[[326国道|326国道]]、[[319国道|319国道]]、[[包茂高速公路|包茂高速公路]]、[[秀松高速公路|秀松高速公路]]、[[渝怀铁路|渝怀铁路]]贯穿全县。<ref>{{Cite news|url=http://news.gog.cn/system/2016/12/22/015301725.shtml|title=重庆秀山至松桃高速23日通车 贵州再增一条出省通道|author=|date=2016-12-23|work=贵州都市报|accessdate=2018-07-29}}</ref>

== 经济 ==
秀山土家族苗族自治县传统经济以农业为主，近年来开始由传统工业、传统商业向新兴工业和现代服务业转变。2017年，秀山实现[[地区生产总值|地区生产总值]]167亿元，同比增长9%。工业增加值实现58.8亿元，增长9%。固定资产投资完成166.8亿元，增长16%。社会消费品零售总额实现75.7亿元，增长14%。贫困发生率降低至1.2%。城乡常住居民人均可支配收入分别实现30094元、10199元，增长9.5%、10.1%。<ref>{{Cite web|url=http://xs.cq.gov.cn/zjxs/267/|title=重庆市秀山自治县人民政府公众信息网-经济概况|accessdate=2018-07-29|author=|date=|publisher=}}</ref><ref>{{Cite web|url=http://xs.cq.gov.cn/zfxx/news/2018-4/8_25193.shtml|title=秀山土家族苗族自治县人民政府工作报告(2018)|accessdate=2018-07-29|author=秀山县政府办|date=|publisher=}}</ref>
