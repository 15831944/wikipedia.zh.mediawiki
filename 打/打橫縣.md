'''打橫縣'''（{{lang-id|Kabupaten Tasikmalaya}}，{{lang-su|ᮊᮘᮥᮕᮒᮦᮔ᮪ ᮒᮞᮤᮊ᮪ᮙᮜᮚ}}）是[[印度尼西亞|印度尼西亞]]（印尼）[[西爪哇省|西爪哇省]]屬下的一個{{tsl|en|Regency (Indonesia)|县 (印度尼西亚)|縣}}，縣治[[打横|打横]]是全縣最大的城市。

打橫縣面積{{convert|2712.52|km2}}，除去縣治[[打橫|打橫市]]後約為{{convert|2552.19|km2}}<ref name="geogjabar" />，東西距離約{{convert|56.25|km}}，南北{{convert|75|km}}<ref name="loca">{{cite web |url=http://www.jabarprov.go.id/index.php/subMenu/99 |title=Peta Jawa Barat |author= |authorlink=http://www.jabarprov.go.id |publisher=West Java Government |language=id |accessdate=2011-01-06 |archive-url=https://web.archive.org/web/20110105145520/http://www.jabarprov.go.id/index.php/subMenu/99 |archive-date=2011-01-05 |dead-url=yes }}</ref>。該縣2010年人口共1675675人，2015年人口1736000人，其中大部分人口為農民<ref name="pend2jabar">{{cite web |url=http://www.jabarprov.go.id/root/dalamangka/DDA2009penduduk3.pdf |title= Population and Labour force – West Java |author= |authorlink=http://www.jabarprov.go.id |year=2008 |work= |publisher=West Java Government |pages=88 |format=PDF |accessdate=2011-01-06}}</ref>。該縣共有39個區，下轄351個村<ref name="pemejabar">{{cite web |url=http://www.jabarprov.go.id/root/dalamangka/DDA2009pemerintahan2.pdf |title=Government – West Java |author= |authorlink=http://www.jabarprov.go.id |year=2008 |work= |publisher=West Java Government |pages=39 |format=PDF |accessdate=2011-01-06}}</ref>。
