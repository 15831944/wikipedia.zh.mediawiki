'''打首狱门同好会'''（{{Jpn|j=打首獄門同好会|kn=うちくびごくもんどうこうかい|rm=uchikubi gokumon tōkōkai}}）是[[日本|日本]]的三人[[摇滚乐队|摇滚乐队]]。2004年结成。

==成员==
===现任成员===
*大澤敦史（{{jpn|j=大澤 敦史|kn=おおさわ あつし|rm=Ohsawa Atsushi}}），吉他手与主唱。在乐队内担任作词、作曲、录音、混音、网站运营、总体设计、演唱会交涉、电缆制作等。被称为“会长”。使用7弦吉他。2018年被任命为静冈县滨松市“滨松试一试大使（{{lang|ja|浜松やらまいか大使}}）”。
*河本明日香（{{jpn|j=河本あす香|kn=かわもと あすか|rm=Kawamoto Asuka}}），鼓手与主唱。在乐队内担任[[mixi|mixi]]平台管理、周边商品管理、安排练习时程等。
*junko，贝斯手与主唱。2006年11月加入。使用5弦贝斯。虽然弹贝斯时用右手，但平时吃饭都是用左手。
**2018年12月20日举办的演唱会中，进行了“junko生日会”环节，宣布了junko已到[[还历|还历]]之年（60岁）<ref>{{cite web |title=【ライブレポート】打首獄門同好会のjunko、実は昨日で60歳に（動画あり） |url=https://natalie.mu/music/news/313303 |website=音楽ナタリー |accessdate=2019-03-09 |language=ja}}</ref>，该消息的引发的冲击极大，以致一度占据了Twitter趋势的头名。

===支援===
*SAKAMUKE（{{lang-ja|サカムケ}}），[[影像骑师|VJ]]兼经纪人。2018年10月宣布结束活动<ref>{{cite web |title=打首獄門同好会マネージャー兼VJのサカムケ☆レシピが卒業、10月にお別れライブ（コメントあり） |url=https://natalie.mu/music/news/297310 |website=音楽ナタリー |accessdate=2019-03-09 |language=ja}}</ref>。
*风乃海（{{lang-ja|風乃海}}），VJ。SAKAMUKE退出后现一人负责原来两人的工作。

===前成员===
*高山明，贝斯手与主唱。2006年3月退出。

==简介==
当初决定乐队名时，河本提议“{{lang|ja|チョンマゲトリオ}}”（Chonmage三重唱），而大泽与高山为了不用这个名字而说“‘打首獄門同好会’或者‘切腹愛護团体’或者‘終生遠島協同組合’哪个更好？”，因此决定了乐队名。决定后开始的一段时间曾用同读音的“{{lang|ja|打首獄門瞳孔開}}”名义活动，其他成员觉得给人印象过于恐怖，而改为现在的同好会。

==经历==
2004年9月乐队结成。2005年2月发布第一张Demo《{{lang|ja|デモ暮らしーVol.1}}》。

2006年3月贝斯手高山退出，11月junko加入。

2009年开始正式出道，发行全国流通专辑《{{lang|ja庶民派爆弾さん}}》。

2014年结成10周年纪念，在[[YouTube|YouTube]]开始“{{lang|ja|10獄放送局}}”节目，播放一些策划短片<ref>{{cite web |title=打首獄門同好会、10周年ベストに「水どう」関連曲も - 音楽ナタリー |url=https://natalie.mu/music/news/118279 |website=音楽ナタリー |accessdate=2019-03-09 |language=ja}}</ref>。

2017年开始“四季”企划，连续四个季节发布新作品。同年的[[RISING_SUN_ROCK_FESTIVAL|RISING SUN ROCK FESTIVAL]] 2017音乐节中，与当地合作的饭店“10獄食堂”开张<ref>{{cite web |title=平成29年 石狩観光トピックス - 北海道石狩市公式ホームページ |url=http://www.city.ishikari.hokkaido.jp/soshiki/syoukour/29962.html |website=北海道石狩市公式ホームページ|accessdate=2019-03-09}}</ref>。12月发布的《冬盘》中收录歌曲《不想从被窝里出来》的MV由人气漫画形象“{{lang|ja|コウペンちゃん}}”担任主角<ref>{{cite web |title=コラボできたの？すごーい！打首獄門同好会MVにコウペンちゃん出演（動画あり） |url=https://natalie.mu/music/news/262371 |website=音楽ナタリー |accessdate=2019-03-09 |language=ja}}</ref>。

2018年3月在[[日本武道馆|日本武道馆]]开办专场演唱会<ref>{{cite web |title=打首獄門同好会、日本武道館ワンマンは2018年3月開催 - 音楽ナタリー |url=https://natalie.mu/music/news/231828 |website=音楽ナタリー |accessdate=2019-03-09 |language=ja}}</ref>，同日四季企划的最后一张《春盘》也将发售<ref>{{cite web |title=打首獄門同好会「春盤」は武道館で販売、全貌は当日明らかに - 音楽ナタリー |url=https://natalie.mu/music/news/270454 |website=音楽ナタリー |accessdate=2019-03-09 |language=ja}}</ref>。此外与[[安桥_(公司)|安桥]]合作推出限量定制款头戴式耳机<ref>{{cite web |title=打首獄門同好会×オンキヨーのヘッドフォン登場、ハウジングに「獄」 - 音楽ナタリー |url=https://natalie.mu/music/news/272921 |website=音楽ナタリー |accessdate=2019-03-09}}</ref>。

2019年3月结成15周年，预定举行23场巡回演唱会<ref>{{cite web|title=打首獄門同好会「獄至十五ツアー」対バンにアルカラ、ヤバTら13組 |url=https://natalie.mu/music/news/319143 |website=音楽ナタリー |accessdate=2019-03-09 |language=ja}}</ref>。
