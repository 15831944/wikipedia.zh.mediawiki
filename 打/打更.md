[[File:Posto_do_Guarda-Nocturno_no_Patane_沙梨頭更館.jpg|缩略图]]的一間前身為更館的[[博物館|博物館]]——[[沙梨頭更館|沙梨頭更館]]，更館是居民消息的集散地，居民遇到事情會到更館找更夫求助，然後由更夫負責發放消息和作出相應的協助。]]
[[File:Canton_night_watchman._Peking_city_guard._John_Thomson_Wellcome_L0056447.jpg|缩略图]]

'''打更'''是[[中國|中國]]古代夜間的一種報時制度，兼具提醒居民防火防盜，以及協助救火和捉賊等作用，以此為業的人士稱為更夫（或稱更練和打更匠等），這種行業習俗隨著[[城市|城市]]的[[現代化|現代化]]以及[[鐘表|鐘表]]的普及而逐漸消失，目前[[中國大陸|中國大陸]][[四川|四川]][[黃龍溪鎮|黃龍溪]]和[[江西省|江西省]][[婺源縣|婺源縣]]的一些地區依然保留著打更的習俗，而位於[[澳門|澳門]]的[[沙梨頭更館|沙梨頭更館]][[博物館|博物館]]是[[省港澳|省港澳]]地區碩果僅存的更館建築<ref>[https://www.icm.gov.mo/cn/NightWatchHouse 沙梨頭更館簡介] {{webarchive|url=https://web.archive.org/web/20170515182704/http://www.icm.gov.mo/cn/NightWatchHouse |date=2017-05-15 }}，[[澳門文化局|澳門文化局]]</ref><ref>[http://aamacau.com/2015/12/18/%E6%B2%99%E6%A2%A8%E9%A0%AD%E6%9B%B4%E9%A4%A8%E9%96%8B%E5%B9%95-%E5%B1%95%E7%8F%BE%E6%9B%B4%E7%B7%B4%E8%A1%8C%E6%A5%AD%E6%AD%B7%E5%8F%B2/ 沙梨頭更館開幕 展現更練行業歷史] {{webarchive|url=https://web.archive.org/web/20170816084005/http://aamacau.com/2015/12/18/%E6%B2%99%E6%A2%A8%E9%A0%AD%E6%9B%B4%E9%A4%A8%E9%96%8B%E5%B9%95-%E5%B1%95%E7%8F%BE%E6%9B%B4%E7%B7%B4%E8%A1%8C%E6%A5%AD%E6%AD%B7%E5%8F%B2/ |date=2017-08-16 }}，[[論盡媒體|論盡媒體]]，2015-12-18</ref>。

==制度由來==
[[File:沙梨頭更館_Posto_do_Guarda-Nocturno_no_Patane_(4).jpg|缩略图]]
古時候，由於平民百姓家裡沒有精確的[[計時工具的歷史|計時儀器]]，為了在晚上掌握時間，因此出現了打更制度，負責報時的更夫根據官府標準的刻漏時間以擊打更鼓或其他器具的方式宣告時間<ref name = "譚01">譚龍曼《中國傳統文化經典：傳統文化常識》，遠流出版，2012年，第13-14頁</ref>。

==職責==

[[File:沙梨頭更館_Posto_do_Guarda-Nocturno_no_Patane_(3).jpg|缩略图]]
[[File:打更工具.jpg|缩略图]]和木梆。]]
打更的工作可分為日更與夜更，更夫在日間也需要處理一些和居民有關的事務，如巡邏、維持治安、通報等。夜更的工作則須要巡更、通宵熬夜，守着[[水鐘|滴漏]]或燃香用來計算和掌握時間等，更夫在晚上巡更時會在沿途敲打報更工具如用竹筒製成的梆子、鑼梆、木更、[[鑼_(東亞)|銅鑼]]、木鼓等，走街串巷地報知時間。從晚上七點開始一夜分五更，每隔兩個小時報一次更，直至淩晨三點。除了喊出更次，更夫還會叫警告語，提醒居民防火防盜，注意安全。另外，一些地區的更夫沒有固定工錢，而是由商舖和居民出資僱用<ref name="MM1">[http://www.macauzine.net/?action-viewnews-itemid-951 澳門碩果僅存的 沙梨頭更館] {{webarchive|url=https://web.archive.org/web/20161024022526/http://www.macauzine.net/?action-viewnews-itemid-951 |date=2016-10-24 }}, 澳門雜誌, 2016-03-11</ref><ref name ="B1"/>。

古時候的夜晚分為五個時段，謂[[五更|五更]]，又稱五鼓，成語「三更半夜」便是由此而來<ref name = "譚01" />。以下為常見的打更時間、警告語和打更聲<ref name = "MM1"/>：

==各地的打更文化==
[[File:沙梨頭更館_Posto_do_Guarda-Nocturno_no_Patane_(1).jpg|缩略图]]

===澳門===
由於[[澳門|澳門]]是沿海城市，當[[海水|海水]][[漲潮|漲潮]]時，更夫便要通知居民，以便他們做好準備以及到安全地方暫避。根據昔日的《更練服務組織條例》，更夫必須是深獲居民信任和富有德望的人，且須要遵守不得擅離職守、不收賄賂、保持儀容整潔、每月須定期巡邏及佩戴由澳門治安警察廳（現為[[澳門治安警察局|澳門治安警察局]]）發出的更夫識別證等規定<ref name = "MM1"/>。

===香港===
[[香港警隊|香港警隊]]在十九世紀中成立，由於當時警隊人數不足，且主力維持外籍人士居住的地方的治安，當地華商便集資組建更練團，自行維護華人社區的治安，後來[[港英政府|港英政府]]在最多華人居住的地區[[上環|上環]]、[[中環|中環]]、[[下環|下環]]和[[西環|西環]]設置「四環更館」，又訓練華人更練。更練團並不屬於警隊，而是由「華民政務司」管理，但亦會執行類似警員的工作如捉賊等，商民用戶則每月支持三元作為保安費和管理費。1950年代開始，警隊的人手增加，「四環更館」結束，1956年，香港政府取消了更練條例，更練一職被保安員和護衛員等新興行業取代<ref>《「更練」－昔日的警民合作》, [[正報|正報]], 第四版, 2016-01-18</ref>。

===四川===
在古時候的[[四川|四川]][[成都|成都]]，當地更夫在日間負責鳴鑼傳達[[官府|官府]]的通知、當地[[鄉長|鄉長]]和鎮長臨時決定的政務，如交[[稅|稅]]、[[捐納|捐納]]、徵收壯丁款的期限或召集百姓開會的時間地點等，是古成都的一種信息專遞方式。至今在[[雙流縣|雙流縣]]黃龍溪等地還保留著在古鎮晚上打更的習俗
<ref name="B1">[http://history.people.com.cn/BIG5/n/2014/0819/c372330-25493754.html 打更：古代報時手段] {{webarchive|url=https://web.archive.org/web/20180428161447/http://history.people.com.cn/BIG5/n/2014/0819/c372330-25493754.html |date=2018-04-28 }}, [[華西都市報|華西都市報]], 2014-08-19</ref>。

===江西===
2017年，[[江西省|江西省]][[婺源縣|婺源縣]]文化部門的一名工作人員表示，他們正在準備給縣裡「最後的打更人」「余加久打更」申報[[非物質文化遺產|非物質文化遺產]]。時年76歲的余加久在接受訪問時說他已有30多年的打更歷史，[[理坑村|理坑村]]村民都習慣聽著他的打更聲進入夢鄉，這是他感到驕傲和自豪之處。觀看余加久打更是遊客到當地觀光的原因之一<ref>邹晓华, [http://jiangxi.jxnews.com.cn/system/2017/03/20/015895480.shtml 最后的打更人已76岁 谁来接班？ 老人已坚持30多年] {{webarchive|url=https://web.archive.org/web/20180428161448/http://jiangxi.jxnews.com.cn/system/2017/03/20/015895480.shtml |date=2018-04-28 }}, 中国江西网, 2017-03-21</ref>。
