《'''打噴嚏'''》（{{lang-en|'''''A choo'''''}}），[[中國大陸|中國大陸]]上映片名為《'''不倒俠'''》，改編自臺灣知名作家[[九把刀|九把刀]]小說《[[九把刀#小說|打噴嚏]]》。此片原訂2014年10月31日上映，但因後製尚未完成，另因[[房祖名柯震東吸毒事件|吸毒事件]]影響，以致上映延期<ref>[https://www.facebook.com/AChoomovie/photos/a.228760707329361.1073741828.200380276834071/311854452353319/ 打噴嚏官方臉書公佈上映日期後延]</ref>。歷經3年延期後，官方於2017年10月宣布同年12月15日[[中國大陸|中國大陸]]上映，但最終還是未能上映，持續延期。<ref>{{cite web |title=《不倒侠》再遭封杀，“劣迹艺人”柯震东难翻身 |url=https://www.jiemian.com/article/1820007.html |accessdate=2018-12-11}}</ref><ref>{{cite web |title=新片被撤、持續被封殺？柯震東回應了 |url=https://stars.udn.com/star/story/10090/2863325 |accessdate=2018-12-11}}</ref>

== 劇情簡介 ==
{{Cquote|為了妳 我要 最勇敢|電影《打噴嚏》<small>宣傳口號</small>}}
這是一個存在超級英雄的城市。

在孤兒院，義智（柯震東飾演）遇見心心姐姐（林依晨飾演）。只要義智想她，她就會打噴嚏。長大之後的義智，為了愛，他要變成最勇敢的男人！他意外成為拳擊場上的「不倒俠」，意外出現超人情敵，音波俠（張曉龍飾演）；當心心陷入危急，他終於明白自己存在的意義。他永遠不能成為超級英雄，但他永遠是心心姐姐的專屬大超人。
