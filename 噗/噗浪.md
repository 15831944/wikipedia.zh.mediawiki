'''噗浪'''（{{lang-en|'''Plurk'''}}）是一个[[微网志|微网志]][[社交網路服務|社群網站]]，由印度裔加拿大人 Kanwardeep Singh Johar、波士尼亞裔丹麥人 Amir Salihefendic、華裔馬來西亞人 [[雲惟彬|Alvin Woon]] 共同於2008年5月12日創辦網站<ref name="amix-2008"/>。服務雖然类似[[Twitter|Twitter]]<ref>{{cite web|date=2008-06-23|accessdate=2008-07-23|url=http://www.pcmag.com/article2/0,2817,2321480,00.asp|title=Plurk.com - Full Review - Reviews by PC Magazine|author=Michael Muchmore|publisher=Ziff Davis Publishing Holdings Inc.|language=en}}</ref><ref>{{cite web|date=2008-06-03|accessdate=2008-07-23|url=http://www.techleader.co.za/stii/2008/06/03/plurk-the-new-twitter/|title=Plurk, the new Twitter?|author=Stii Pretorius|publisher=Mail & Guardian Online|language=en|deadurl=yes|archiveurl=https://web.archive.org/web/20090912035353/http://www.techleader.co.za/stii/2008/06/03/plurk-the-new-twitter|archivedate=2009-09-12}}</ref>，但其最大的特色就是在一条「[[时间轴|时间轴]]」上显示自己与好友的所有消息。同时，和[[Twitter|Twitter]]的「<nowiki>@</nowiki>」回覆不同的是，在Plurk中主題之下的回覆，都是属于該主題，而不是独立的。

噗浪目前已经有包含中文在内的30餘种[[语言|语言]]版本可用<ref name="pctp-plurk-2010"/>，除[[英文|英文]]外的所有語言版本都是由[[志愿者|志愿者]]贡献的。噗浪網頁上提供多个[[时间轴|时间轴]]、[[背景|背景]]及控制板的主题，同时，用户也可以使用[[CSS|CSS]]來自定义自己的页面。

噗浪限制發文字数由「140個字」變更为「360个字」，<ref>[http://www.plurk.com/p/hgsu2o 噗浪=360個字]，by [[雲惟彬|雲惟彬]]（alvin），2012-10-14</ref><ref>[http://zh.blog.plurk.com/archives/1688 讓圖片排列更美觀 限制發文字数为140个字 改为210个字] {{webarchive|url=https://web.archive.org/web/20130529090634/http://zh.blog.plurk.com/archives/1688 |date=2013-05-29 }} - 噗浪通訊社 - 噗浪官方部落格(華文)，2012-12-18</ref>但[[汉字|汉字]]和英文字母都计为1个字，因此通常中文用户可以发布更多内容。2009年12月4日，Plurk推出官方的[[API|API]]<ref>[http://www.plurk.com/API Plurk API]</ref>，但是此前已经有用户开发出非官方的[http://code.google.com/p/rlplurkapi/ RLPlurkAPI]<ref> {{cite web | url = http://code.google.com/p/rlplurkapi/ | title = rlplurkapi The unofficial Plurk API |author = lwh.ryan | publisher = Google Inc.}}</ref>、[http://code.google.com/p/jplurk/ jPlurk]<ref> {{cite web | url = http://code.google.com/p/jplurk/ | title = jPlurk - Plurk Official API Binding in Java | author = Askeing, ChingYiChan.tw | publisher = Google Inc}}</ref>。

== 命名 ==
噗浪共同創辦人暨介面設計師[[雲惟彬|雲惟彬]]（Alvin Woon）接受[http://chinese.engadget.com/ 《Engadget中文版》]訪問時說：「其實這個字（Plurk），沒有任何意思。當初我們在想名字的時候，我們的想法就是要用一個簡單發音的字，想來想去，就決定用Plurk了。後來當然我們有幫這五個單字，各想了一個衍伸意義。但事實上，Plurk這個字並沒有任何意思。」<ref>[http://chinese.engadget.com/2009/02/18/interview-with-plurk-founder-alvin-woon/ 特派J：噗浪創辦人專訪！]</ref>

目前公司對Plurk這名字的解釋有：

* 'people'（人）及'lurk'（埋伏）的[[簡寫|簡寫]]
* 'play'（玩樂）及'work'（工作）的[[混成詞|混成詞]]
* '''P'''eace（和平相處）、'''L'''ove（愛心）、'''U'''nity（團體）、'''R'''espect（尊重）跟'''K'''arma（命運）的[[首字母縮略字|首字母縮略字]]

== 特色 ==
Plurk最主要的特色是可以在一條時間軸上顯示自己和好友的所有消息，每一位[[會員|會員]]都可以單獨評論別人的Plurk內容。除此之外，Plurk亦有一個熱度系統，其名稱叫作Karma，以令會員更加投入[[網站|網站]]上的活動。另外，Plurk亦有各種[[語言|語言]]版本以供選擇。

使用者可以上傳48*48尺寸以下之圖片作為自訂表符，團隊也有研發各類執骰表符供用戶使用，截至2018年6月為止已有以下幾種：普通六面骰(dice)、眼睛六面骰(dice2)、四面骰(dice4)、八面骰(dice8)、十面骰(dice10)、十二面骰(dice12)、二十面骰(dice20)、求籤(draw)、擲筊(bobei)、內褲骰 (panties)、拉霸(slot)、擲硬幣 (coin)。

== Karma值 ==
[[Karma_(消歧義)|Karma值]]在Plurk中是一个热度值。拥有更高的Karma可以使用到更多的服务，如额外的表情符號、时间轴右上角的生物等。发表适量的plurk、保持活动都可以增长Karma；反过来，长期不活动、每天发表超过30条plurk、交友申請被拒絕等，也会以降低Karma的形式作出相应的惩罚。

== 偷偷說 ==
在發表新主題時，將發語詞選擇為「偷偷說」，即可以暱名的方式發表。以此方式發表示主題，發表者的名字會以「ಠ_ಠ」顯示，而其他回應者的名字由一英文名詞加上一串隨機數字組成。

== 噗幣 ==
透過網站或行動裝置應用程式付費，以取得進階搜尋、表情符號擴充及移除廣告等功能。採用月費制。<ref>[http://www.bnext.com.tw/article/view/id/39774 噗浪商業模式大轉彎！推出付費服務，不再被廣告綁架]</ref>

;趨勢

== 過去功能（現在網站已移除功能） ==
* 噗浪的隨機聊天功能（有段時間叫噗浪樂透）在 2010年12月10日取消。
* 噗浪趋势是在2009年5月18日推出的新功能，能生成消息中的關鍵字所產生的趨勢時間圖表。这个图表能概估該關鍵字在所有plurkers (即plurk使用者)發出的消息中所出现的頻率次數，在首頁出來之後就不見了。

== 過去發展 ==
2018年，Plurk成為小眾的SNS服務，在多數地區被[[Facebook|Facebook]]、[[Twitter|Twitter]]、[[新浪微博|新浪微博]]等服務取代。
=== 北美 ===
雖然噗浪發源於加拿大，但使用者多以當地華裔居多，特別是來自台灣的華人。而美國最主流的微網誌服務仍以[[推特|推特]]拔得頭籌。

=== 台灣 ===
由於噗浪很快就有[[繁體中文|繁體中文]]版頁面，加上許多原本在[[無名小站|無名小站]]經營部落格的部落客集體遷到噗浪，因此噗浪在全球總瀏覽數中佔最多數的主要為台灣使用者。<ref>[http://www.epochtimes.com/b5/10/3/7/n2837444.htm 噗浪創辦人雲惟彬 最愛台灣夜市美食] - 大紀元，2010年3月7日</ref>

;政黨
台灣的[[中國國民黨|中國國民黨]]、[[民主進步黨|民主進步黨]]皆有官方噗浪帳號，[[2012年中華民國總統選舉|2012年總統大選]]主要候選人[[馬英九|馬英九]]及[[蔡英文|蔡英文]]也都有噗浪帳號。許多其他政治人物包括[[謝長廷|謝長廷]]、[[蘇貞昌|蘇貞昌]]也都利用噗浪與網友互動，[[謝長廷|謝長廷]]甚至舉辦數次「噗友啪」。

;媒體
台灣主要媒體網站皆提供簡易的按鈕供閱覽者將資訊轉貼至噗浪；並在2009年[[八八水災|八八水災]]時，噗浪成了災民與網友表達需求物資的管道。許多歌手、公司、書店亦有官方噗浪帳號提供第一手資訊。現在噗浪也有動漫、小說等的愛好者為虛構角色甚至是古人創立帳號進行角色扮演，稱作"角色噗"，例如[[哈利波特|哈利波特]]系列小說，[[新世紀福爾摩斯|新世紀福爾摩斯]]裡的人物，皆有角色噗。

;商業行銷
許多公司行號也嘗試在噗浪上進行行銷推廣，其中位於[[台北市|台北市]]的餐飲業者最早運用噗浪Karma值進行行銷活動，如「卡馬也能當飯」、「卡馬%OFF」等，並號召「台灣噗友蓋史上最高樓」，最後蓋出10346層，網址代號pi5wg，為台灣著名噗浪行銷案例。

;其他
台灣發售的[[智慧型手機|智慧型手機]]中，除了常見的[[臉書|臉書]]、[[推特|推特]]、[[Google+|Google+]]、[[新浪微博|新浪微博]]等主流社群網站服務外，還有以台灣使用者為主要用戶的噗浪。

== 軼事 ==
*在Plurk中，有一個常見的[[彩蛋_(視覺)|彩蛋]]，而且它使用的是[[科樂美秘技|科樂美秘技]]（↑↑↓↓←→←→BA）。
**在任一個人的Plurk頁面中打上這個秘技，頁面上方的最近訊息就會開始繞著一個圓圈旋轉起來。再按一次「↑」即可復原。
**2017年，旋轉特效被改為所有的訊息掉到河道底部。<ref>[https://www.plurk.com/p/m6ugch 噗浪美術頻道：ㄐㄐ特製ㄉ掉落噗浪彩蛋，他說都沒人發現。]</ref>
**另外，如果鍵入另一個秘技（↑↑↓↓←←→→AB）則會得到整個噗浪頁面色彩反轉的效果。
