'''陈冬'''{{BD|1978年|12月|||CatIdx=C陈}}，中國航天員，籍贯[[河南|河南]][[郑州|郑州]]，生于[[洛阳|洛阳]]<ref >{{cite web|url=http://www.chinanews.com/gn/2016/10-16/8032993.shtml | language= | title=陈冬同志简历 |publisher=新华社 |date=2016年10月16日 |accessdate= }}</ref>，从小在洛阳长大。1997年考入[[空军长春飞行学院|空军长春飞行学院]]<ref name="了解">{{cite web|url=http://news.dahe.cn/2016/10-16/107613154.html | language= | title=神十一航天员曝光！和景海鹏搭档的河南帅爸陈冬 你了解多少  |publisher=大河客户端 |date=2016年10月16日 |accessdate= }}</ref>，1997年8月入伍，1999年4月加入中国共产党。[[中国人民解放军航天员大队|中国人民解放军航天员大队]]三级航天员，[[上校|上校]]军衔。<ref>{{cite web |title=航天员张晓光晋升少将，至此，我国已有7名航天员成为将军 |url=https://xw.qq.com/cmsid/20180315A0FN1D00 |website=腾讯网 |accessdate=2018-07-17}}</ref>

== 简历 ==
曾任空军某师某团飞行大队大队长，安全飞行1500小时，被评为空军一级飞行员。2010年5月，正式成为中国第二批航天员，从初选到定选，他在同批航天员中成绩都是第一<ref name="了解"/>。2016年6月，入选[[天宫二号|天宫二号]]与[[神舟十一号|神舟十一号]]载人飞行任务航天员乘组。2016年10月17日7时30分，神舟十一号飞船发射<ref>{{cite web |url = http://www.thepaper.cn/newsDetail_forward_1544116 |title = 神舟十一号明日7时30分发射，飞行乘组由景海鹏和陈冬组成 |publisher = 澎湃新闻 |date = 2016年10月16日 }}</ref>。2016年11月18日14時03分，神舟十一号载人飞船返回舱在内蒙古中部预定区域成功着陆，执行飞行任务的航天员景海鹏、陈冬身体状态良好。<ref>{{Cite web|url=http://slide.news.sina.com.cn/c/slide_1_2841_105305.html#p=1|title=神舟十一号飞船返回地球|accessdate=2016-11-18|work=slide.news.sina.com.cn}}</ref>

== 家庭 ==
陈冬父亲陈树林、母亲黄焱，父母育有2个儿子，陈冬的哥哥陈波是名[[医生|医生]]，比陈冬大4岁，现在[[广东|广东]][[东莞|东莞]]工作，母亲住在东莞帮忙照看陈波的孩子。<ref >{{cite web|url=http://news.dahe.cn/2016/10-18/107621105.html | language= | title=6年未见陈冬 哥哥希望他回来后能拍一张全家福 |publisher=大河网-河南商报 |date=2016年10月18日 |accessdate= }}</ref>

陈冬已婚，妻子是[[浙江|浙江]][[嘉兴|嘉兴]]人（陈冬曾在嘉兴空军飞行大队服役），夫妇俩育有一对[[双胞胎|双胞胎]]儿子，父亲陈树林一年的大部分时间都在[[北京|北京]]照看陈冬的2个孩子。<ref name="了解"/><ref >{{cite web|url=http://zjnews.zjol.com.cn/zjnews/zjxw/201610/t20161018_1982451.shtml | language= | title=景海鹏将度过"太空生日"，"嘉兴女婿"陈冬这样告诉双胞胎儿子 |publisher=钱江晚报 |date=2016-10-18 |accessdate= }}</ref>
