'''陈道明'''（{{bd|1955年|4月26日|catIdx=C陈}}），出生於[[中國|中國]][[天津|天津]]，祖籍[[浙江|浙江]][[绍兴|绍兴]]，现居[[北京|北京]]。[[中國大陸|中國大陸]]著名[[男演員|男演員]]，畢業於[[中央戏剧学院|中央戏剧学院]]。<ref name="坚持">{{cite web |url = http://ent.163.com/ent_2003/editor/030706/030706_177822.html |language = |title = 陈道明:在沉淀中坚持 |publisher = 网易娱乐 |date = 2003年 |accessdate = }}</ref>

==生平==
1989年因《[[末代皇帝_(电视剧)|末代皇帝]]》中的[[溥仪|溥仪]]一角获得第7届“[[金鹰奖|金鹰奖]]”最佳男演员奖和第9届全国电视“[[飞天奖|飞天奖]]”优秀男主角奖。陈道明在1990年改编自[[围城_(小说)|同名小说]]的10集电视连续剧《[[围城_(电视剧)|围城]]》中饰演的方鸿渐一角为其经典角色和主要代表作，赢得第11届中国电视剧[[飞天奖|飞天奖]]优秀男主角奖、第7届《大众电影》“金鹰奖”最佳男演员奖、中国电影表演艺术学会第3届学会奖和全国制片厂第2届优秀电视剧评选最佳男主角奖。1999年电影《我的1919》饰演的顾维钧赢得第20届[[金鸡奖|金鸡奖]]最佳男主角奖。2001年凭借电视剧《[[康熙王朝|康熙王朝]]》中的[[康熙帝|康熙帝]]一角获得第20届全国电视[[金鹰奖|金鹰奖]]最佳男演员奖。
