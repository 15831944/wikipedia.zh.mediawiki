'''陈乐宗'''{{BD|1956年||||catIdx= C陈}}，祖籍[[广东省|广东省]][[顺德市|顺德市]]，曾是[[哈佛大学医学院|哈佛大学医学院]][[病理学|病理学]]研究员，现任香港晨兴创投主席。<ref name="hexun" >{{cite web |url=http://renwu.hexun.com/figure_1589.shtml |title=陈乐宗-人物频道 |accessdate= 2014|publisher=和讯网 |work= |language=zh-hans}}</ref>

==生平==
早年，陈乐宗曾担任哈佛大学医学院病理学研究员，并且修读[[加州大学洛杉矶分校|加州大学洛杉矶分校]]的科学辐射生物学博士学位。

1986年，陈乐宗建立“香港晨兴创投”。

2006年，陈乐宗兄弟捐款1亿港元在[[香港中文大学|香港中文大学]]成立“[[晨兴书院|晨兴书院]]”。<ref name="fish" >{{cite web |url=http://news.qq.com/a/20140917/016030.htm |title=香港富豪3.5亿美元捐哈佛 下一笔捐南加州大学 |accessdate= 2014|publisher=腾讯网 |work= |language=zh-hans}}</ref>

2014年9月，陈氏兄弟以其先父[[陈曾熙|陈曾熙]]的名义向哈佛大学捐款3.5亿美元，之后对[[南加州大学|南加州大学]]进行捐款。<ref name="fish" /><ref name="bull" >{{cite web |url=http://news.ifeng.com/a/20140924/42069778_0.shtml |title=香港富豪捐款美国大学称冠全球 占境外捐款额17%
 |accessdate=2014 |publisher=凤凰网 |work= |language=zh-hans}}</ref>

==家庭==
陈乐宗的父亲陈曾熙在1960年创建“恒隆集团”，主营地产开发，他还有个哥哥[[陈启宗|陈启宗]]。<ref name="fish" />
