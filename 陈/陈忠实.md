'''陈忠实'''（{{bd|1942年|8月3日|2016年|4月29日|catIdx=C陈}}），[[陕西省|陕西省]][[西安市|西安市]][[灞桥区|灞桥区]]西蒋村人。中国大陸当代[[作家|作家]]，曾任[[陕西省作家协会|陕西省作家协会]]主席、[[中国作家协会|中国作家协会]]副主席<ref>{{cite web |url=http://culture.people.com.cn/n/2013/1030/c172318-23374581.html |title=作家陈忠实：老友相继凋零 倍感人生匆促(图) |accessdate=2013 |publisher=人民网 |work= |language=zh-hans}}</ref>。1965年开始创作[[散文|散文]]，1993年以《[[白鹿原_(小说)|白鹿原]]》一书一举成名，后任陕西省[[作家协会|作家协会]]主席。出版有30余种作品。<br />

2016年4月29日，陳忠實因病逝世，享年七十三歲<ref>[http://news.eastday.com/c/20160429/u1ai9331282.html 《白鹿原》作者陈忠实因病在西安去世 享年73岁]</ref><ref>{{cite web |title=74岁作家陈忠实去世 |url=http://culture.people.com.cn/n1/2016/0429/c87423-28314423.html |website=人民网 |accessdate=2018-10-08 |language=中文}}</ref>。

== 履历 ==
于1965年初发表处女作，文体为散文，创作生涯由此开始。

曾任过中国作家协会副主席，陕西省作家协会名誉主席、党组成员。也是西北五省自从建国以来继[[柯仲平|柯仲平]]之后第二位担任中国作家协会副主席这一重要职位。


== 生平 ==

1962年高中毕业，1965年开始发表作品。1979年加入中国作家协会。<br />
著有短篇小说集《乡村》、《到老白杨树背后去》，中篇小说集《初夏》、《四妹子》，《陈忠实小说自选集》（3卷），《陈忠实文集》（5卷），散文集《告别白鸽》等。<br />
短篇小说《信任》获1979年全国优秀作品奖、《立身篇》获1980年《飞天》文学奖，中篇小说《康家小院》获上海首届《小说界》文学奖、《初夏》获1984年《当代》文学奖、《十八岁的哥哥》获1985年《长城》文学奖，[[报告文学|报告文学]]《渭北高原，关于一个人的记忆》获全国1990年－1991年报告文学奖，长篇小说《白鹿原》获1993年陕西双五文学奖、1996年[[人民文学出版社|人民文学出版社]]炎黄杯文学奖、第四届[[茅盾文学奖|茅盾文学奖]]。<br />
陈忠实喜爱足球，曾经创作有关足球散文15篇。2002年世界杯，陈忠实为新华社体育部撰写专栏稿件。整个世界杯期间，陈忠实一共写了四篇专栏文章：《细腻了的英国人》、《收获反省》、《惨烈的场面与蒸红苕的技巧》和《桑巴与桑巴之外的魅力》<ref>[http://sports.sohu.com/20170429/n491191440.shtml 陈忠实与足球的故事：先是个球迷，其次才算个作家]</ref>。

==病逝==
據媒體報道，从陈忠实家人处获悉，2016年4月29日晨7：40左右，陈忠实因病在[[西安|西安]][[西京医院|西京医院]]去世，终年73岁。2015年12月，陈忠实因为胸部[[腫瘤|腫瘤]]，曾入院接受手术。2016年4月28日上午，陈忠实因[[舌癌|舌癌]]恶化，癌细胞扩散，突吐血不止，被送往医院急救，因搶救無效病逝<ref>[http://cn.epochtimes.com/gb/16/4/29/n7786873.htm 《白鹿原》作者去世，曾与王岐山密谈4小时] [[大紀元|大紀元]] 2016-04-30 </ref>。

隨後，[[陝西省|陝西省]]作家協會設置靈堂悼念陳忠實，現場有[[習近平|習近平]]、[[劉雲山|劉雲山]]、[[王岐山|王岐山]]等[[中共中央政治局常委|中共中央政治局常委]]，及[[胡錦濤|胡錦濤]]、[[曾慶紅|曾慶紅]]等前[[黨和國家領導人|黨和國家領導人]]送來的花圈，但獨欠中共前總書記[[江澤民|江澤民]]。 [[北京|北京]]《[[新京報|新京報]]》的微信，近日統計中央領導人向去世官員、專家送花籃的情況，從4月14日至20日共有6名官員、專家去世，江澤民是唯一一個全數缺席、未有表達悼念的前領導<ref>{{cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20160501/850886/ |title = 中國官員專家去世 前領導送花圈獨缺江澤民 |publisher =  蘋果日報（台灣） |date=2016-05-01}}</ref>。
