'''陈独秀旧居'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[北池子大街|北池子大街]][[箭杆胡同|箭杆胡同]]20号，是[[陈独秀|陈独秀]]在北京的旧居，也是《[[新青年|新青年]]》编辑部旧址，[[北京市文物保护单位|北京市文物保护单位]]。

==历史==
1915年9月15日，陈独秀在[[上海|上海]]创办《青年杂志》（后更名《新青年》）出版。1917年初，陈独秀迁居今箭杆胡同20号（原箭杆胡同9号），同期由他创办的《新青年》杂志编辑部也从上海迁来此处。1918年初，《新青年》从陈独秀主撰变为同仁刊物，由[[北京大学|北京大学]]教授[[胡适|胡适]]、[[钱玄同|钱玄同]]、[[刘半农|刘半农]]、[[鲁迅|鲁迅]]、[[李大钊|李大钊]]、[[高一涵|高一涵]]等轮流编辑。在该院编辑的《新青年》杂志近30期。[[李孤帆|李孤帆]]、[[吴君毅|吴君毅]]、[[高语罕|高语罕]]、[[刘叔雅|刘叔雅]]夫妇等其他[[新文化运动|新文化运动]]及[[五四运动|五四运动]]期间的文化名人、[[中国共产党|中国共产党]]早期建党人士等都曾到访这里<ref name=zxw/>。陈独秀在此从1917年一直居住到1919年<ref name=bjwb>{{Cite web |url =  http://www.bjww.gov.cn/wbsj/bjwbdwDetail.htm|title =陈独秀旧居  |author =  |date =  |publisher = 北京文博 |language =  |accessdate = 2018-02-20 |quote =  }}</ref>。

2013年初，北京市东城区将陈独秀旧居腾退修缮纳入东城区名城保护重点项目，列入区政府“折子工程”。腾退工作结束后，陈独秀旧居的修缮工作是与[[北京新文化运动纪念馆|北京新文化运动纪念馆]]合作进行的。2015年9月15日是《新青年》杂志创刊一百周年纪念日。此前一天即2015年9月14日，箭杆胡同20号的陈独秀旧居及《新青年》编辑部完成腾退修缮，北房门口挂上“陈独秀旧居”和“《新青年》编辑部旧址”的牌子。修缮遵循修旧如旧的原则，原房屋的房梁只要能继续使用便保留，地面上的砖也有50余块是旧居原先使用的<ref name=zxw>{{Cite web |url = http://www.chinanews.com/cul/2015/09-14/7522747.shtml |title = 陈独秀旧居及《新青年》编辑部完成腾退修缮 |author =  |date =2015-09-14  |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.xinhuanet.com/politics/2016-04/12/c_128884899.htm |title =陈独秀旧居年内对外开放  |author =  |date = 2016-04-12 |publisher =  新华网|language =  |accessdate =  |quote =  }}</ref>。

在2015年陈独秀旧居修缮完工前后，东城区[[东华门街道|东华门街道]]办事处与北京新文化运动纪念馆签订合作协议，计划推出相关主题展览陈列<ref name=zxw/>。2017年6月15日，中共北京市东城区委常委、宣传部部长周家雷到陈独秀旧居调研展陈项目的设计布置情况<ref>{{Cite web |url = http://beijing.qianlong.com/2017/0619/1782961.shtml |title = 东城区委常委周家雷到东华门街道陈独秀旧居调研 |author =  |date = 2017-06-19 |publisher = 千龙网 |language =  |accessdate =  |quote =  }}</ref>。

==建筑==
该院坐北朝南，如意大门，正房三间，南房三间，硬山合瓦过垅脊<ref name=bjwb/>。2015年修缮完成后，院落保持了原有格局，整体占地面积约250平方米，有房屋12间<ref name=zxw/>。
