'''陈思思'''{{BD|1976年|12月28日|||catIdx=C陈}}出生在湖南省[[常德市|常德市]]，籍贯[[江苏省|江苏省]][[南京市|南京市]]，中国歌唱家，师承[[金铁霖|金铁霖]]、[[李双江|李双江]]，[[中国共产党|中国共产党]]党员，[[中国人民解放军|中国人民解放军]][[第二炮兵|第二炮兵]]政治部文工团副团长、[[大校|大校]]军衔，曾担任[[全国人大代表|第十届全国人大代表]]、[[全国青联常委|全国青联常委]]。陈思思数度登上[[中国中央电视台春节联欢晚会|中国中央电视台春节联欢晚会]]，对[[慈善|慈善]]事业有贡献。<ref>{{cite web |url=http://news.163.com/photoview/00AQ0001/12022.html |title= 歌唱家陈思思出任二炮文工团最年轻副团长|accessdate=2010 |publisher=网易|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/y/2010-11-09/09403140126.shtml |title=陈思思升任二炮文工团副团长 已捐四所希望小学 |accessdate=2010年11月9日 |publisher=新浪网 |work= |language =zh-cn}}</ref><ref name="fox ">{{cite web|url=http://www.china.com.cn/culture/txt/2009-11/02/content_18808901.htm|title=陈思思在台办大陆女星首个个唱 演绎邓丽君经典|accessdate=2009-11-02|publisher=中国网|archiveurl=https://web.archive.org/web/20140203104356/http://www.china.com.cn/culture/txt/2009-11/02/content_18808901.htm|archivedate=2014年2月3日|work=|language=zh-cn|deadurl=yes}}</ref>

==生平==
=== 早年生涯 ===
1976年，陈思思出生在[[湖南省|湖南省]][[常德市|常德市]]，曾就读于常德市七中，1990年，陈思思考入[[怀化学院|怀化学院]]，1995年，陈思思考入[[湖南师范大学|湖南师范大学]]音乐系本科，后又在[[解放军艺术学院|解放军艺术学院]]、[[中国音乐学院|中国音乐学院]]声歌系读书。1999年，陈思思进入[[中国人民解放军第二炮兵政治部文工团|中国人民解放军第二炮兵政治部文工团]]，后升职为解放军第二炮兵政治部文工团副团长。<ref>{{cite web|url=http://news.stnn.cc/c6/2012/09/28/1348810088542.html|title=娱乐圈九大美女党员|accessdate=2012|publisher=星岛环球网|work=|language=zh-cn}}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>

=== 职业生涯 ===
1995年，陈思思演唱新民歌《情哥哥去南方》，在中国大陆歌坛“[[民族音乐|民族音乐]]”新秀崛起。<ref name=" fish" /> 后来，陈思思陆续出版专辑《情哥哥去南方》、《中国女孩》、《刘三姐》、《你让我感动》、《共度好时光》、《锦绣年代》。<ref name=" fish" >{{cite web |url=http://ent.sina.com.cn/s/2007-03-14/14331478562.html |title=资料：著名歌手陈思思简介|archiveurl=http://ent.sina.com.cn/s/2007-03-14/14331478562.html |archivedate=  2012年11月3日|accessdate= 2007|publisher=新浪网 |work= |language =zh-cn}}</ref>

2009年，陈思思赴[[台湾|台湾]]举办个人演唱会，是首位中国大陆赴台湾举办音乐会的歌手，由于陈思思是中国共产党党员和[[中国人民解放军|中国人民解放军]]中人物，[[內政部入出国及移民署|內政部入出国及移民署]]召开联审会，并且经过[[中華民國國軍|军方]]点头后放行，陈思思在台湾有一个姑姑是书法家[[詹秀容|詹秀容]]。<ref name="fox " /><ref>{{cite web |url=http://news.sohu.com/20091028/n267811068.shtml |title=大陆歌唱家陈思思赴台 台媒关注解放军身份 |accessdate=2009 |publisher=搜狐网|archiveurl=http://news.sohu.com/20091028/n267811068.shtml |archivedate= 2012年11月3日 |work= |language =zh-cn}}</ref>
