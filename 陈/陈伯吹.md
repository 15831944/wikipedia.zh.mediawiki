'''陈伯吹'''（{{bd|1906年||1997年|catIdx=C陳伯吹}}），原名'''陈汝壎'''，曾用笔名'''夏雷'''，江苏宝山（今[[上海市|上海市]][[宝山区_(上海市)|宝山区]]）人，[[陈佳洱|陈佳洱]]之父。中国著名的现代儿童文学作家、翻译家、出版家、教育家，是中国儿童文学的一代宗师。小学毕业后辍学，当过学徒。16岁起在乡村小学教书多年，1927年在[[商务印书馆|商务印书馆]]出版了他的第一部儿童小说《学校生活记》，从此开始了业余文学创作活动。曾任[[中国作家协会|中国作家协会]]理事、[[上海少年儿童出版社|上海少年儿童出版社]]社长等。

== 职业生涯 ==
1930年主编《小学生》半月刊，并编辑《小朋友丛书》。1934年起，任[[儿童书局|儿童书局]]编辑部主任，并在[[上海幼儿师范学校|上海幼儿师范学校]]任教职。1942年去重庆在编译馆工作，1946年在上海发起组织[[上海儿童文学工作者联谊会|上海儿童文学工作者联谊会]]，开展[[儿童文学|儿童文学]]的创作和理论研究。1949年以后，历任[[华东师范大学|华东师范大学]]教授、[[北京师范大学|北京师范大学]]教授、[[上海少年儿童出版社|上海少年儿童出版社]]社长等职。

== 主要作品 ==
他的作品塑造了众多的各具特色的少年儿童形象，语言流畅，故事生动曲折，富于浪漫色彩。主要作品有童话《一只想飞的猫》、童话集《幻想张着彩色的翅膀》，小说集《飞虎队和野猪队》等。此外，尚有散文集《三门峡工地两少年》及《儿童文学简论》、《作家和儿童文学》、《儿童文学研究》等理论著作。

== 社会贡献 ==
1981年捐献稿费五万五千元创立儿童文学园丁奖，每年评判一次，几乎每年都会出版一本获奖作品集。1988年更名为[[陈伯吹儿童文学奖|陈伯吹儿童文学奖]]。“陈伯吹儿童文学奖”是中国连续运行时间最长的文学奖项之一，对鼓励和促进儿童文学的创作、培养儿童文学作家起到了极大作用。“陈伯吹儿童文学奖”自1981年举办第1届评奖到2007年已经举办了27届。
