'''陈笠笠'''<!-- {{BD|||||catIdx=C}} -->，出生于[[湖南省|湖南省]][[长沙市|长沙市]]，毕业于[[中国音乐学院|中国音乐学院]]，为中国大陆女歌手。

== 生平 ==
8岁起，陈笠笠师从长沙陈莉老师学习音乐。1996年，陈笠笠考入湖北省艺术职业学院，师从肖淑云和余辉。1999年，陈笠笠考入[[中国音乐学院|中国音乐学院]]，师从[[金铁霖|金铁霖]]。<ref>{{cite web |url=http://news.hexun.com/2013-06-12/155069131.html |title=陈笠笠民歌玩“跨界” |accessdate=2013年6月12日 |publisher=和讯新闻 |work= |language=Simplified Chinese}}</ref><ref>{{cite web |url=http://yue.ifeng.com/news/detail_2013_03/19/23274855_0.shtml |title=陈笠笠参加金铁霖音乐会 红色礼服彰显大气(图) |accessdate= 2013年3月19日|publisher=凤凰网 |work= |language=Simplified Chinese}}</ref>2004年，陈笠笠分配到北京歌舞剧院工作。2007年，陈笠笠加入[[中国人民解放军海政文工团|中国人民解放军海政文工团]]。
