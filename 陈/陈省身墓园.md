'''陈省身墓园'''，是美国籍华人数学家，微分几何之父——[[陈省身|陈省身]]及其夫人[[郑士宁|郑士宁]]的陵墓，落成于2011年6月，位于[[天津市|天津市]][[南开大学|南开大学]]校园内省身楼与南门之间。
[[File:南开大学陈省身墓园.jpg|thumb]]

== 陈省身 ==
陳省身，号辛生，美国籍华人[[数学家|数学家]]，[[微分几何|微分几何]]学家。[[美国国家科学院|美国国家科学院]]院士、[[中央研究院|中央研究院]]院士，同时是[[法国科学院|法国科学院]]、[[意大利国家科学院|意大利国家科学院]]、[[英国皇家学会|英国皇家学会]]和[[中国科学院|中国科学院]]的外籍院士。1926年考入[[南开大学|南开大学]]数学系，1930年毕业并获学士学位。1934年获清华大学理学硕士学位。1936年获德国汉堡大学理学博士学位。1938年为西南联合大学教授。1943年为美国普林斯顿高级研究院研究员。1946年为南京中央研究院数学研究所代所长。1949年为美国芝加哥大学教授。1960年至 1979年为美国[[伯克利加州大学|伯克利加州大学]]教授。1961年加入美国籍。1981年至1984年任[[美国伯克利数学研究所|美国伯克利数学研究所]]首任所长。1985年，在南开大学建立[[南开数学研究所|南开数学研究所]]，并担任首任所长，同年南开大学授予他名誉博士学位。后久居南开大学校园内，2004年陈省身逝世，2011年葬于南开大学内。<ref>{{cite web | url=http://www.nim.nankai.edu.cn/sschern/shengping.htm | title=陈省身数学研究所网站-生平介绍 | accessdate=September 3, 2011}}</ref>

== 概述 ==
[[File:陈省身墓碑.jpg|thumb]]
陈省身墓园位于南开大学校园内津河北岸，树木与草地之间用黑白相间的石条铺成了一个呈不规则菱形的广场，设有一块黑板状的墓碑和23个矮凳，可供休息和思考。郑士宁女士去世后，陈省身表示将来要将自己和夫人的骨灰安葬在母校[[南开大学|南开大学]]的校园内，并希望在墓碑前有黑板、有座位，以便大家讨论学术问题。

== 纪念碑 ==
2011年6月，南开大学在纪念陈省身数学研究所理论物理研究是成立25年之际，举行了陈省身先生夫妇纪念碑揭幕仪式，[[杨振宁|杨振宁]]为纪念碑揭幕并表示“纪念碑上的手稿是陈先生的杰作，他开启了20世纪后半叶数学新的方向。”<ref>{{cite web | url=http://news.sciencenet.cn/sbhtmlnews/2011/6/245611.html | title=院士专家齐聚南开 大师精神感召学子 ——南开大学理论物理前沿讨论会侧记 | accessdate=August 25, 2011}}</ref><ref>{{cite web | url=http://www.chinadaily.com.cn/dfpd/tj/2011-06/25/content_12776721.htm | title=杨振宁到南开大学为陈省身纪念碑揭幕 | accessdate=September 2, 2011}}</ref>陈省身的生前好友[[范曾|范曾]]等和一些前来参加研讨会的中科院院士及南开大学师生出席揭幕仪式。

陈省身与其夫人[[郑士宁|郑士宁]]的墓碑（纪念碑）由两块石头组成，一块是弧形汉白玉，另一块是平板型的黑色花岗岩。墓碑整体横截面呈曲边三角形，象征数学史上著名的“高斯—博内—陈”公式。墓碑以数学公式的手稿作为墓志铭，上面刻写的是陈省身在美国任教时讲义中的“高斯—博内—陈”公式的手迹。墓碑下方，刻写着郑世宁和陈省身的姓名及生卒年月。<ref>{{cite web| url=http://www.022net.com/2011/6-18/553872282767340.html| title=世上独一无二的数学大师墓碑今在天津南开园揭幕| accessdate=September 2, 2011| deadurl=yes| archiveurl=https://web.archive.org/web/20110830181707/http://www.022net.com/2011/6-18/553872282767340.html| archivedate=2011年8月30日}}</ref>
