'''陈道祥'''（{{bd|1962年|9月|catIdx=T}}），汉族，[[安徽省|安徽省]][[宣城|宣城]]人，[[中国人民解放军少将|中国人民解放军少将]]。

== 簡歷 ==
历任战士、排长、参谋、股长，[[南京军区|南京军区]]司令部作战部综合处副处长、处长、副部长，[[中国人民解放军陆军第七十三集团军|陆军第31集团军]]第86师师长。2010年4月，任[[中国人民解放军陆军第七十二集团军|陆军第1集团军]]参谋长。2011年7月，晋升少将军衔。是第十三届全国人大代表<ref>{{cite news |title=中华人民共和国第十三届全国人民代表大会代表名单 |url=http://www.xinhuanet.com/2018-02/24/c_1122448968_6.htm |publisher=新华网 |access-date=2019-04-09}}</ref>。
2014年7月，升任[[广州军区|广州军区]]副参谋长<ref>{{cite web |title=陆军第1集团军原参谋长陈道祥调任广州军区副参谋长 |url=http://news.sohu.com/20141002/n404824152.shtml |website=搜狐 |access-date=2019-04-09}}</ref>。2016年1月，任[[南部战区|南部战区]]副参谋长<ref>{{cite web |title=原广州军区副参谋长陈道祥少将已任南部战区副参谋长 |url=http://news.eastday.com/c/20171112/u1a13417216.html |website=东方网 |access-date=2019-04-09}}</ref>。2019年4月8日，中央军委主席[[习近平|习近平]]签署命令，任命陈道祥少将接替[[谭本宏|谭本宏]]中将担任[[中國人民解放軍駐香港部隊|中国人民解放军驻香港部队]]司令员<ref>{{cite web |title=陈道祥少将任解放军驻香港部队司令员 |url=http://www.xinhuanet.com/2019-04/08/c_1210102941.htm |website=新华网 |access-date=2019-04-09}}</ref>。
