'''陈赛娟'''（{{bd|1951年|5月21日|||catIdx=Chen陈赛娟}}），女，[[浙江|浙江]][[鄞县|鄞县]]人，[[中国|中国]][[血液病学|血液病学]]家和[[分子遗传学|分子遗传学]]家，[[中国工程院院士|中国工程院院士]]。其丈夫为[[中国科学院院士|中国科学院院士]]、原[[中华人民共和国卫生部|卫生部]]部长[[陈竺|陈竺]]。

==生平==
陈赛娟1989年获[[法国|法国]][[巴黎第七大学|巴黎第七大学]][[博士|博士]]学位，现任[[上海交通大学医学院|上海交通大学医学院]][[医学基因组学国家重点实验室|医学基因组学国家重点实验室]]主任，[[上海血液学研究所|上海血液学研究所]]所长。

== 学术成就 ==
其学术成就主要在[[白血病|白血病]]和[[分子遗传学|分子遗传学]]研究方面：
* 率先[[克隆|克隆]]了Ph[[染色体|染色体]]阳性急性白血病中BCR[[基因|基因]]第一[[内含子|内含子]]的断裂点丛集区，并提出BCR-ABL基因重排理论模型。
* 首先发现急性早幼粒细胞白血病（APL）变异型染色体易位t（11;17）并克隆了受累的PLZF基因。
* 成功克隆[[核孔|核孔]][[蛋白|蛋白]]NUP98和HRX等多种相关白血病的致病基因。
* 在APL等多种白血病基因产物靶向治疗方面获得突破，使急性早幼粒细胞白血病治愈成为可能，也为其他类型白血病和[[肿瘤|肿瘤]]的治疗研究提供了借鉴。

2003年，当选为[[中国工程院|中国工程院]][[院士|院士]]。曾获国家自然科学二等奖等多项国家级重要奖励。是国家杰出青年科学基金获得者。

2011年，陈赛娟获选[[法国国家医学科学院|法国国家医学科学院]]外籍院士。<ref name="陈赛娟法国院士">{{cite web |url = http://news.sciencenet.cn/htmlnews/2011/12/257391.shtm |title=陈赛娟当选法国国家医学科学院外籍院士 |publisher=[[科学网|科学网]] |date = 发布时间：2011-12-22 17:36:59  |accessdate=2011年12月25日 |language=zh-hans}}</ref>2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。
