'''陈同海'''({{bd|1948年|9月18日|catIdx=C陈}})，[[江苏|江苏]][[响水|响水]]人，出生于[[山东省|山东省]][[惠民县|惠民县]]；大庆石油学院采油工程专业毕业，教授级高级经济师。1973年7月，加入[[中国共产党|中国共产党]]。曾任[[浙江省|浙江省]][[宁波市|宁波市]][[市长|市长]]，[[国家计委|国家计委]]副主任；[[中国石油化工集团公司|中国石油化工集团公司]][[总经理|总经理]]，[[中国石化|中国石化]][[董事长|董事长]]等职。2007年6月22日，因涉嫌受贿，被“[[双规|双规]]”；同时辞去了在[[中国石化|中国石化]]的所有职务。2009年7月15日，[[中华人民共和国法院和法庭列表|北京市第二中级人民法院]]一审认定犯[[受贿罪|受贿罪]]成立，贪污受贿的款项逾人民币1.9亿，被判处[[死刑缓期执行|死刑（缓期两年执行）]]，并[[剥夺政治权利|剥夺政治权利]]终身。

== 早年经历 ==
陈同海出身中共高级干部家庭。父亲[[陈伟达|陈伟达]]，曾是[[中共|中共]][[上海|上海]]地区的领导人之一，领导过上海学生运动；建国后，曾先后在[[浙江|浙江]]、[[天津|天津]]等地，担任主要领导职务；特别在天津期间，[[胡启立|胡启立]]、[[李瑞环|李瑞环]]都曾先后出任过他的副手。

1976年，28岁的陈同海从[[大庆石油学院|东北石油学院]]采油工程专业毕业，被分配至[[大庆油田|大庆油田]]工作，任大庆研究院开发一室地质员；不久，被调往[[浙江|浙江]]。此时，他的父亲[[陈伟达|陈伟达]]正担任中共浙江省委副书记（1972年至1978年）。<ref>[http://past.tianjindaily.com.cn/xwrw/31/11313125.htm 陈伟达同志生平]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }} 原载《天津日报》</ref><ref>[http://chinaps.cass.cn/renda/shengweishuji/zejiangsheng.htm 中共浙江省委员会历任负责人]</ref>

== 步入政界 ==
在浙江省科委机关工作了一段时间后，1983年3月，陈同海被任命为[[浙江|浙江]][[镇海|镇海]]石油石化总厂的负责人，成为政坛的起步。此后，他历任[[浙江省|浙江省]][[宁波市|宁波市]][[市长|常务副市长]]，省计经委副主任、党组副书记，[[中共|中共]][[宁波|宁波]][[市委书记|市委副书记]]，[[宁波市|宁波市]]代市长、市长等职。1994年1月，陈同海被提拔为原[[国家计划委员会|国家计划委员会]]副主任，成为副部级官员，时年45岁。

== 执掌国企 ==
1998年3月，[[中华人民共和国国务院|国务院]]机构改革，原[[国家计委|国家计委]]撤销，陈同海被调任[[中国石化集团|中国石化集团]]副总经理。2000年2月，又出任[[中国石化|中国石化]]首届董事会董事、副董事长。2003年4月，接替[[李毅中|李毅中]]出任[[中国石化集团|中国石化集团]]总经理，同时兼任[[中国石油化工股份有限公司|中国石油化工股份有限公司]][[董事长|董事长]]。

== 受贿下台 ==
2007年6月22日，陈同海以“个人原因”突然宣布辞职。当年10月15日，[[国资委|国资委]]主任[[李荣融|李荣融]]向外界证实，陈已经被“[[双规|双规]]”，但没有解释原因。<ref>[http://www.caijing.com.cn/2007-10-15/100033680.html 中石化原董事长陈同海被确认“双规”]</ref> 后来才公布，陈同海的辞职是因为经[[中共中央|中共中央]]批准后，[[中纪委|中纪委]]、[[监察部|监察部]]已对陈的严重违纪问题开始进行立案检查。理由是，“利用职务便利，为他人谋取利益，收受钱款数额巨大；利用职权为情妇谋取巨额不正当利益；生活腐化”<ref>[http://news.hexun.com/2008-01-26/103282040.html 中石化原总经理陈同海被双开 移送司法机关处理]</ref>。

2009年7月15日，[[中华人民共和国法院和法庭列表|北京市第二中级人民法院]]一审认定陈同海犯[[受贿罪|受贿罪]]成立，被判处[[死刑缓期执行|死刑（缓期两年执行）]]，[[剥夺政治权利|剥夺政治权利]]终身；并处没收个人全部财产。<ref>[http://www.chinadaily.com.cn/zgzx/2009-07/15/content_8431955.htm 中石化原老总陈同海受贿2亿被判死缓 没收全部财产]</ref>
