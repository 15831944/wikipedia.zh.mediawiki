'''陈云纪念馆'''，位于[[上海市|上海市]][[青浦区|青浦区]][[练塘镇|练塘镇]]老朱枫公路3516号，是[[中国共产党|中国共产党]]领导人[[陈云|陈云]]的纪念馆。

== 简介 ==
陈云纪念馆是经[[中共中央|中共中央]]批准建立的中国唯一系统展示陈云生平业绩的纪念馆，设在陈云的家乡上海
市青浦区练塘镇。该馆在[[中共上海市委|中共上海市委]]直接领导下，在中央有关部门指导下，在“陈云故居”和“青浦革命历史陈列馆”原址基础上，于2000年6月6日陈云诞辰95周年之际建成开馆，时任中共中央总书记[[江泽民|江泽民]]题写馆名“陈云故居暨青浦革命历史纪念馆”<ref name=jj/><ref>{{cite web  | url = http://www.sh-aiguo.gov.cn/node2/node4/node6/u1a50.html |title=陈云故居暨青浦革命历史纪念馆  |publisher = 中共上海市委宣传部 |date = |accessdate = 2012-05-14 }}</ref>。2005年6月8日陈云诞辰100周年之际，在纪念馆落成陈云铜像，江泽民为铜像题词。2008年3月10日起，作为上海4家爱国主义教育示范基地之一，免费对公众开放。2013年5月26日，经中央批准，“陈云故居暨青浦革命历史纪念馆”更名为“陈云纪念馆”，保留“青浦革命历史纪念馆”的牌子。江泽民再次题写了“陈云纪念馆”馆名。2015年6月2日陈云诞辰110周年之际，陈云生平业绩展暨陈云文物馆开展仪式在陈云纪念馆举行<ref name=jj>{{Cite web |url = http://www.cyjng.net/Default.aspx?tabid=70&language=zh-CN |title =纪念馆简介  |author =  |date =  |publisher = 陈云纪念馆 |language =  |accessdate =  2017-05-29|quote =  }}</ref>。

2017年，陈云纪念馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。

陈云纪念馆由铜像广场、主馆、陈云文物馆、陈云故居、陈云手迹碑廊、文化创意街组成<ref name=jj/>。
