'''陈垣'''（{{bd|1880年|11月12日|1971年|6月21日}}），[[表字|字]]'''援庵'''，[[广东|广东]][[江門|江門]][[蓬江|蓬江]]棠下石头人。[[中国|中国]][[宗教史|宗教史]]研究巨匠，歷任[[輔仁大學|輔仁大學]]、[[北京師範大學|北京師範大學]]校長，[[燕京大學|燕京大學]]「哈佛－燕京學社」首任社長。與[[錢穆|錢穆]]、[[呂思勉|呂思勉]]、[[陳寅恪|陳寅恪]]並稱為[[嚴耕望|嚴耕望]]所評選的「現代四大史學家」。

== 生平 ==
陳垣十二歲時在学馆閱讀[[张之洞|张之洞]]《[[书目答问|书目答问]]》，並攻讀《[[四库全书总目提要|四库全书总目提要]]》，應[[科舉|科舉]]不第，1905年在[[廣州|廣州]]創辦《[[時事畫報|時事畫報]]》，议论时政，鼓吹革命思想。1907年考取[[博濟醫學堂|博濟醫學堂]]。[[辛亥革命|辛亥革命]]成功之後於1913年當選[[中华民国初年国会|国会]]眾議員，1919年在北京[[中华基督教会缸瓦市教堂|缸瓦市教堂]]受洗加入[[基督新教|基督新教]]，1921年出任[[中華民國教育部|中華民國教育部]]次長。後覺政治腐敗，留居[[北京|北京]]，從事歷史研究與教育工作。
 
陳垣以《元也里可温考》一文成名，在宗教史、[[校勘學|校勘學]]、[[考古學|考古學]]方面均有相當成就。

陳垣於1926年任[[輔仁大學|輔仁大學]]校長，1939年初，在故宫古物陈列所发现《[[嘉兴藏|嘉兴藏]]》，與《嘉兴藏目录》對校，发现[[木忞|木忞]]的《北游集》，自此重視和尚語錄，撰《语录与顺治宫廷》、《[[顺治皇帝|顺治皇帝]]出家》。陳垣發現《[[旧五代史|旧五代史]]》輯本传写脱误，因撰《旧五代史辑本发覆》，曰：“一百五十年来，学者承诵引据，以为薛史真本如此，信奉不疑，而孰料其改窜至于如此。今特著其忌改之例，以发其覆。”<ref>《陈垣学术论文集》第二集，第149页。</ref> 陳垣还曾与[[胡适|胡适]]论[[杨守敬|杨守敬]]《水经注疏》的成书年代，胡适于1948年8月14日写成《跋杨守敬论水经注案的手札两封》，陳垣則認定[[光绪|光绪]]二十三年丁酉（1897年）四月，当时还没有《水经注疏》。

1950年12月8日陈垣在《[[人民日报|人民日报]]》发表文章《美国从来就是我们的敌人》响应[[三视教育运动|三视教育运动]]。文章指出：“社会上还有些人有‘崇美’‘恐美’心理”，“凡是‘崇美’的，是因为没有民族的自尊心。凡是‘恐美’的，是没有民族的自信心。”<ref>{{cite web |author1=陈垣 |authorlink1=陈垣 |title=美国从来就是我们的敌人 ——四十五年前的回忆 |url=http://www.hprc.org.cn/wxzl/bksl/kmyzzt/kmyzlszn_2763/kmyzlszn_2764/201007/t20100723_56039.html |website=中华人民共和国国史网 |accessdate=2018-10-28 |language=中文 |date=1950-12-08}}</ref>1951年11月，毛澤東在懷仁堂舉行國宴時，向別人介紹說：“這是陳垣，讀書很多，是我們國家的國寶。”

1952年[[院系调整|院系调整]]中[[天主教辅仁大学|天主教辅仁大学]]大部并入[[北京師範大學|北京師範大學]]後，他續任[[北京師範大學|北京師範大學]]校長。
[[File:1962-02_1962年_历史学家陈垣.jpg|thumb]]
1959年他加入[[中國共產黨|中國共產黨]]，后来於[[文化大革命|文化大革命]]，陳垣幾次被抄家，差點因為和劉少奇的合影而被定罪，並且被逼寫了幾次檢討悔過書張貼於北師大，而後遭到軟禁。<ref>{{cite news|url=http://www.epochtimes.com/b5/17/1/13/n8699834.htm|title=留在大陸的中華民國院士之大結局（四）|author=張東園|publisher=大紀元|date=2017-01-13}}</ref>

1971年卒於家中。著作有《南宋初河北新道教考》、《通鑑胡注表微》等。

陈垣的學生有[[姚从吾|姚从吾]]、[[黄现璠|黄现璠]]、[[郑天挺|郑天挺]]、[[方国瑜|方国瑜]]、[[蔡尚思|蔡尚思]]、[[邓广铭|邓广铭]]、[[单士元|单士元]]、[[柴德赓|柴德赓]]、[[白寿彝|白寿彝]]、[[牟润孙|牟润孙]]、[[赵光贤|赵光贤]]、[[陈述|陈述]]、[[启功|启功]]、[[史念海|史念海]]、[[周祖谟|周祖谟]]、[[刘乃和|刘乃和]]、[[史树青|史树青]]等。2002年9月8日，陈垣全身[[铜像|铜像]]在北京师范大学落成。

== 家庭 ==
陈垣祖籍廣東省新會縣石头鄉，先祖為中原南逃難民；祖父名海学（1810年-1878年），是药材商人，在广州开设陈信义药材行；祖母锺氏、黄氏（1833年—1921年）。父親名维启（1854年—1909年），也是药材商人；陈垣的母親為周氏（1858年—1934年）；过继父名维举（1853年—1886年），实为三伯，过继母為李氏（1857年-1947年）。

陈垣一生有三位夫人：[[光緒|光緒]]二十二年（1896年），年僅十七歲的陈垣与邻乡篁边的[[邓泽如|邓泽如]]之堂侄女[[邓珍安|邓珍安]]（1882年—1966年5月）成婚。光緒二十九年（1903年）在家乡娶[[吴淑媛|吴淑媛]]（1884年-1912年）为平妻。1913年，陈垣与[[徐蕙龄|徐蕙龄]]（1892年—1966年5月）结婚。两人一起生活时间最长，直到1948年两人感情不和而分居。

陈垣共有有十一名子女：其中桂辛、博、约之為邓氏所生，仲益、利、让、潜為吴氏所生，善、容、慈、冬為徐氏所生。

*陈仲益（1904年—1970年），次子，死于[[肺癆|肺癆]]。
*陈利（1908年—1932年），次女，死于肺癆。

*陈潜（1912年—1947年），三女，死于肺癆。
