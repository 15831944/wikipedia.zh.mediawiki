'''陈巍'''（{{Lang-en|'''Nathan Chen'''}}，{{bd|1999年|5月5日|||catIdx=Chen, Nathan}}）[[华裔美国人|美国华裔]][[花样滑冰|花样滑冰]][[花样滑冰#.E5.96.AE.E4.BA.BA.E6.BA.9C.E5.86.B0|男子单人滑]]运动员。

蝉联[[2018年世界花式滑冰錦標賽|2018]]和[[2019年世界花式滑冰錦標賽|2019年世锦赛]]两届冠军，蝉联[[2017–18国际滑冰联盟花样滑冰大奖赛总决赛|2017]]及[[2018–19国际滑冰联盟花样滑冰大奖赛总决赛|2018年大奖赛总决赛]]两届冠军，2017年度[[四大洲花样滑冰锦标赛|四大洲锦标赛]]冠军，2017–2019年三届[[美國花式溜冰錦標賽|美国锦标赛]]冠军。 [[2018年冬季奥林匹克运动会|2018平昌冬奥]]团体赛铜牌。被誉为 “四周跳之王” 的陈巍是目前世界唯一能做五种四周跳（[[勾手跳|勾手跳]]4Lz，[[后内点冰跳|后内点冰跳]]4F，[[后外跳|后外跳]]4Lo，[[后内跳|后内跳]]4S，[[后外点冰跳|后外点冰跳]]4T）的男子选手。

陈巍是第一位在国际花滑联盟正式比赛上，于长节目完成五个四周跳的选手，被中国冰迷称为“五四青年”。<ref>{{Cite web |url = http://api.5club.cctv.cn/cctvsports/cctv5/cctvsportsshare/article/article.html?id=ARTI1521969797093766 |title = 陈巍：“五四青年”的进化史 | author = 闫海莹 |date = 3/25/18 |publisher =  央视体育}}</ref>2018年在平昌创下奥运纪录，成为首位在单一节目中挑战六个高难度四周跳的选手，<ref>{{Cite web |url = http://www.sohu.com/a/223037552_99985320 |title = 陈巍六次完成四周跳创冬奥历史 | date = 2/17/18 |publisher = 搜狐综合体育}}</ref>改写史上最高技术分记录。<ref>{{Cite web |url = http://www.icenetwork.com/news/2018/02/17/266642592/chen-earns-redemption-in-historic-free-skate |title = Chen earns redemption with historic free skate |author = Philip Hersh |date = 2/17/18 |publisher = icenetwork |access-date = 2018-02-18 |archive-url = https://web.archive.org/web/20180219090143/http://www.icenetwork.com/news/2018/02/17/266642592/chen-earns-redemption-in-historic-free-skate# |archive-date = 2018-02-19 |dead-url = yes }}</ref>同年获世锦赛金牌，是继[[叶甫根尼·普鲁申科|普鲁申科]]之后最年轻的世界冠军。<ref>{{Cite web |url = http://olympics.nbcsports.com/2018/03/24/nathan-chen-wins-figure-skating-world-championship/ |title = Nathan Chen wins world title by nearly 50 points after everyone falls | author = Nick Zaccardi | date = 3/24/18 |publisher =  NBC Sports}}</ref>2019年继[[阿列克谢·亚古丁|亚古丁]]之后，在十多岁之龄卫冕世界冠军，为21世纪以来第一人。

== 个人简介 ==
陈巍出生于[[美国|美国]][[犹他州|犹他州]][[盐湖城|盐湖城]]，<ref name=ISU-1617/>  父亲陈志东（{{lang|en|Zidong Chen}}）祖籍广西来宾，于1988年留学美国，是一名医学博士。母亲王禾（{{lang|en|Hetty Wang}}）出身北京，在医院担任翻译。 <ref>{{Cite news | url=http://hk.crntt.com/crn-webapp/touch/detail.jsp?coluid=7&kindid=0&docid=104941639 | title=華裔陳巍的奧運冠軍路：靠實力不靠運氣|date=2018-01-15|publisher=中評社}}</ref>陈巍是家庭中最小的孩子，四个哥哥姐姐分别是：陈菲（{{lang|en|Alice}}）、陈莎（{{lang|en|Janice}}）、陈晓（{{lang|en|Tony}}）和陈茜（{{lang|en|Colin}}）。<ref name=USFS-NC/><ref>{{Cite news | url=http://m.sohu.com/n/322470216/ | title=20多年后重游故地 来宾籍留美博士期待贡献|date=2011-10-17|publisher=广西新闻网}}</ref> 四个兄姐如今分别在科技、金融与航空航天等领域工作。<ref>{{Cite news | url=https://new.qq.com/omn/20190330/20190330A06Y8G.html | title=击败羽生结弦的少年|author = 夏桑 |date=2019-03-30|publisher=腾讯网}}</ref>

在2013年8月以高二学生身份就读在线高中连接学院（Connections Academy）前，他分别在犹他州盐湖城的{{Link-en|西部高中 (犹他州)|West High School (Utah)|西部高中}}和[[加利福尼亚州|加利福尼亚州]]{{Link-en|箭头湖 (加利福尼亚州)|Lake Arrowhead, California|箭头湖}}的{{Link-en|世界边缘高中|Rim of the World High School}}就读。<ref name=school/> 为奥运停学一年的他，以美国高考SAT数学满分800分的成绩申请大学，于2018年秋天正式入读美国常青藤名校[[耶鲁大学|耶鲁大学]]。<ref>{{Cite web |url = https://xw.qq.com/cmsid/CCE2018042402963400 |title = 陈巍进耶鲁修双学位为未来铺路| date = 24/3/18 |publisher = 腾讯体育}}</ref>陈巍在[[耶鲁大学|耶鲁大学]]主修统计学，计划辅修生物工程或者医学预科。耶大提供学校曲棍球队的冰场Ingalls Rink供他使用。他尝试兼顾滑冰和学业，开始身兼全时学生和世界级运动员的双重身份生活。每天日程表包括早上九点开始上课，下午自己训练，晚上再回学校学习。他认为过着不以滑冰为全部的生活，让眼界变宽。<ref>{{Cite news | url=https://www.nytimes.com/2019/03/17/sports/nathan-chen-yale.html | title=Nathan Chen’s Yale Juggling Act|author = Karen Crouse |date=2019-03-17|publisher=紐約時報}}</ref>

陈巍与花滑的缘分始于2002年的盐湖城冬奥会，他的姐姐参加了开幕式节目《光之子》的演出。由于盐湖城建了许多冰场，冰上活动相当受欢迎。三岁的陈巍原希望追随哥哥打冰球，当守门员，但因年纪太小，便穿上姐姐的冰鞋先练花滑，从而展开花样滑冰生涯。

作为花滑的延伸训练，陈巍七岁起在{{Link-en|西部芭蕾学院|Ballet West}}接受六年正规芭蕾课程，每周六堂课，八岁能做芭蕾空转两圈，早早展露天分，他曾参加西部芭蕾舞团"[[天鹅湖|天鹅湖]]""[[睡美人|睡美人]]""[[胡桃夹子|胡桃夹子]]"等芭蕾舞剧演出。<ref name=ballet/> <ref>{{Cite web |url = http://www.dancemagazine.com/nathan-chen-ballet-2531928874.html |title = This Former Ballet Kid Is Heading To The Olympics—And Is Expected To Win | author = Jennifer Stahl | date = 2/6/18 |publisher = 美国舞蹈杂志}}</ref>参加区域及全国的高级体操赛事长达七年。他还加入冰球队，师事世界级钢琴家，并赢得过犹他州古典钢琴比赛冠军。这些训练使他具备流畅的舞蹈表现能力，音乐敏锐度，体操控制身体的能力，冰球的强悍，紧凑的行程课表也培养出有效运用时间的纪律。<ref>{{Cite news | url=http://www.espn.com/espn/feature/story/_/id/22257148/figure-skater-nathan-chen-take-sport-new-heights-2018-winter-olympics | title=One Giant Leap | author = Bonnie D. Ford|date=2018-2-8|publisher=ESPN}}</ref>

由于花滑训练花费昂贵，家境小康的陈巍仍曾陷买不起冰鞋，比赛付不起教练交通食宿费用的窘境。他学习花滑的历程仰赖许多人士协助，支付冰鞋和训练的费用，其中{{Link-en|迈克尔·韦斯|Michael Weiss (figure skater)}}基金会曾于十年内赞助陈巍奖学金约七万五千美元，<ref>{{Cite web |url = https://www.usatoday.com/story/sports/winter-olympics-2018/2018/02/02/nathan-chen-elevates-usa-hopes-olympic-skating-gold/1089579001/ |title = Nathan Chen elevates U.S. hopes for figure skating gold at Winter Olympics | author = Christine Brennan | date = 2/2/18 |publisher = USA TODAY SPORTS}}</ref>他的教练也曾提供财务支援。<ref>{{Cite web |url = https://rsport.ria.ru/interview/20171209/1129822500.html |title = Натан Чен: "Своей победой в Москве я расстроил кучу людей" |  date = 10/12/18 |publisher = Россия сегодня}}</ref>

== 花滑生涯 ==
=== 早年经历 ===
陈巍从三岁起就开始学习[[花样滑冰|花样滑冰]]，并在一年后（2003年）首度参加花滑赛事。从2007年到2009年，他获得了参加美国少年花样滑冰锦标赛少年组和中级组的资格。在2007年和2008年的全美花滑青少年锦标赛中，陈巍分别拿下了少年组第十名和第三名的成绩；而在2009年，他拿下了中级组的银牌。

在2009-10赛季里，陈巍晋级到新手组。在2010年于[[华盛顿州|华盛顿州]][[斯波坎縣|斯波坎縣]]举办的[[美國花式溜冰錦標賽|美国花样滑冰锦标赛]]上，陈巍拿下了新手组冠军，并以10岁的年纪成为美国花样滑冰锦标赛史上最年轻的新手组冠军。<ref name=rsp/> 由于陈巍的年纪偏小，使得他在2010-11赛季里依旧被划归在新手组，从而他蝉联了2011年于[[北卡罗来纳州|北卡罗来纳州]][[格林斯伯勒_(北卡罗来纳州)|格林斯伯勒]]举办的美国花样滑冰锦标赛新手组冠军。<ref name=IN110124/><ref name=GS110124/>

在2011-12赛季里，陈巍在全国级别赛事中晋升到少年组级别。此时他师从{{Link-en|耶夫根·车尔尼索娃|Evgenia Chernyshova}}，并前往[[加利福尼亚州|加利福尼亚州]]的{{Link-en|箭头湖 (加利福尼亚州)|Lake Arrowhead, California|箭头湖}}与跳跃专家{{Link-en|拉斐尔·阿鲁秋尼杨|Rafael Arutyunyan}}合作。<ref name=IFS110703/> 2011年12月中旬，拉斐尔·阿鲁秋尼杨成为了陈巍的主教练。<ref name=IN120123/> 陈巍在2012年1月24日于加利福尼亚州[[圣何塞_(美国加州)|圣何塞]]举行的美国花样滑冰锦标赛上摘下少年组冠军。<ref name=GS120125/> 作为陈巍的第一次国际亮相，他摘取了2012年在[[意大利|意大利]][[塞尔瓦迪瓦尔加尔德纳|塞尔瓦迪瓦尔加尔德纳]]举办的{{Link-en|加迪纳杯春季赛|Gardena Spring Trophy}}的新手组冠军。<ref name=IN120411/><ref name=IN120414/>

=== 2012-2013赛季 ===
在2012-13赛季里，陈巍依旧在全国范围内保持着少年组的参赛水平。并且他在这个赛季达到了[[国际滑冰联盟青少年花样滑冰大奖赛|国际滑冰联盟青少年花样滑冰大奖赛]]（JGP）系列赛的参赛门槛年龄（13周岁）。<ref name=GS120125/> 陈巍的第一个JGP系列赛便是2012年9月12日到16日在[[奥地利|奥地利]][[林茨|林茨]]举办的{{Link-en|国际滑冰联盟青少年花样滑冰大奖赛奥地利站|ISU Junior Grand Prix in Austria|奥地利国际青少年大奖赛}}。<ref name=USFSA-1314comp/> 他以迄今JGP赛事最高分（222.00分）获得了金牌。<ref name=IN120915/> 2012年10月初，由于陈巍在{{Link-en|国际滑冰联盟青少年滑冰大奖赛克罗地亚站|ISU Junior Grand Prix in Croatia|克罗地亚青少年大奖赛}}[[短节目|短节目]]中小腿受伤而退赛，进而导致他无缘本赛季[[国际滑冰联盟青少年花样滑冰大奖赛|国际滑冰联盟青少年花样滑冰大奖赛总决赛]]。<ref name=IN121006/> 2013年1月22日，陈巍在[[内布拉斯加州|内布拉斯加州]][[奥马哈_(内布拉斯加州)|奥马哈]]举行的美国花样滑冰锦标赛上，拿下少年组铜牌。<ref name=IN130123/>

=== 2013-2014赛季 ===

=== 2016-2017赛季 ===
迎接第一个国际成人组赛季，赴密西根坎顿市委托[[玛丽娜·祖耶娃|玛丽娜·祖耶娃]]编舞，为提升节目内容分PCS，留训接受[[玛丽娜·祖耶娃|玛丽娜·祖耶娃]]和Oleg Epstein指导。为准备第一个成人组国际赛季，他在髋骨受伤六个月后，开始练习勾手四周跳（4Lz）和后内点冰四周跳（4F）。<ref>{{Cite web |url = http://web.icenetwork.com/news/2016/10/29/207519314/ |title = Chan, Chen in Canton with different goals in mind |author = Lynn Rutherford |date = 10/29/16 |publisher = icenetwork }}</ref>

[[File:2016–2017_Grand_Prix_of_Figure_Skating_Final_Nathan_Chen_IMG_4000_03.jpg|thumb]]]]
具备四种四周跳－后外点冰跳（4T），后内跳（4S），后内点冰跳（4F），勾手跳（4Lz）－陈巍于2016年芬兰杯取得本赛季首面金牌。其后的大奖赛事中，陈巍在2016年法国站的短节目和长节目，都成功完成组合跳跃勾手四周跳接外点冰三周跳（4Lz+3T），以及后内点冰四周跳。大奖赛法国站取得第四名成绩后，陈巍回归加州{{Link-en|拉斐尔·阿鲁秋尼杨|Rafael Arutyunyan}}的训练团队。

获2016年NHK杯银牌后，首度晋级成人组大奖赛总决赛。在法国马赛举办的总决赛中，他于长节目完成四个四周跳，最终取得银牌，是继[[叶甫根尼·普鲁申科|普鲁申科]]（1999年16岁之龄）之后，于大奖赛总决赛得奖的最年轻的选手（17岁）。<ref>{{Cite web |url = http://web.icenetwork.com/news/2017/01/18/213823372/ |title =Chen revamps programs in preparation for KC |author = Lynn Rutherford |date = 1/18/17 |publisher = icenetwork }}</ref>

2017年美国锦标赛中，陈巍在短节目完成两个四周跳（4Lz+3T，4F），长节目完成五个四周跳（4Lz+3T，4F，4T+2T+2Lo，4T，4S），以美国滑冰史上最高成绩（短节目106.39，长节目 212.08分，总分318.47）及最高分差（55.44）赢得金牌。

二月赢得四大洲锦标赛金牌，成为国际滑冰联盟史上第三位短节目超过100分的选手，并取得国际赛事上总分超过300分的佳绩。

四月世界锦标赛于赫尔辛基举办，他虽然总共挑战了八个四周跳（短节目两个，长节目六个），由于冰鞋耗损导致失误，最终获得第六名，仍与获第七名的队友杰森·布朗，联手为美国取得三个奥运参赛名额。<ref>{{Cite web |url = http://www.usfigureskating.org/story?id=90301&type=media |title =U.S. FIGURE SKATING ANNOUNCES WORLD, FOUR CONTINENTS AND WORLD JUNIOR TEAMS |date = 1/22/17 |publisher = U.S. Figure Skating }}</ref>陈巍的第一个成人组赛季，年度世界排名第三。<ref>{{Cite web |url = http://www.isuresults.com/ws/wr2016-17/wrmen.htm |title =

大学新鲜人陈巍趁着秋假参加大奖赛美国站，以总分280.57分，分差41.06分的优势取得胜利。这是大奖赛美国站自现行评分制度实施16年来，所有项目的最大分差。<ref>{{Cite news | url=https://olympics.nbcsports.com/2018/10/20/nathan-chen-wins-skate-america/ | title=Nathan Chen wins Skate America by largest margin in history |author = Nick Zaccardi |date=2018-10-20 |publisher=NBCSC}}</ref>自由滑189.99分以及总分，为平昌奥运后新规则实施以来，当时的的最高纪录。<ref>{{Cite news | url=https://www.olympicchannel.com/en/stories/news/detail/nathan-chen-retains-skate-america/?utm_source=Twitter&utm_medium=tw-post&utm_campaign=social-owned&utm_content=OCContent_OC%20Content&sf200471203=1 | title=NATHAN CHEN RETAINS SKATE AMERICA TITLE IN STYLE |date=2018-10-21 |publisher=Olympic Channel}}</ref>接连赢得大奖赛法国站冠军，为近十年来首位在大奖赛取得五连胜的男单选手，本赛季双胜的成绩获得总决赛参赛资格。<ref>{{Cite news |url=https://olympics.nbcsports.com/2018/11/24/nathan-chen-wins-grand-prix-france/ | title=Nathan Chen rallies, wins GP France, sets possible Yuzuru Hanyu matchup |author = Nick Zaccardi|date=2018-11-24 |publisher=NBC Sports}}</ref>12月于加拿大温哥华举办的大奖总决赛蝉联冠军，为1995年大奖总决赛举办以来，继[[叶甫根尼·普鲁申科|普鲁申科]]，[[陈伟群|陈伟群]]，[[羽生结弦|羽生结弦]]之后，第四位连霸的男单选手。<ref>{{Cite news |url=https://www.teamusa.org/News/2018/December/08/Nathan-Chen-Repeats-As-Grand-Prix-Final-Champion-Keeping-Perfect-Season-Alive | title=Yale Freshman Nathan Chen Repeats As Grand Prix Final Champion, Keeping Perfect Season Alive |author = Nick Zaccardi |date=2018-12-8 |publisher=美国滑冰联盟}}</ref>也是继[[塔拉·利平斯基|塔拉·利平斯基]]之后第二位在大奖总决赛连胜的美国单人花滑运动员。陈巍此赛自由滑技术分101.79改写本赛季新规则施行后新纪录。

这一年美国锦标赛在底特律举行，配置两个四周跳的短节目113.42分，四个四周跳的自由滑228.80分，总分342.22为花滑史上最高分數，雖不列為国际花滑联盟正式紀錄，零失误的惊天表现，为他连续赢得第三个美国冠军头衔，以绝对强势再度问鼎世锦赛王座。<ref>{{Cite news | url=https://www.usatoday.com/story/sports/columnist/brennan/2019/01/27/us-mens-national-figure-skating-nathan-chen-third-title/2697110002/ | title=Nathan Chen balances Yale studies with third U.S. figure skating championship |author = Christine Brennan |date=2019-1-27 |publisher=USA TODAY}}</ref>

陈巍利用学校春假出征于日本埼玉举行的世锦赛，他与日本名将[[羽生结弦|羽生结弦]]的对决堪称经典。这是自平昌冬奥会之后，两人在世界顶级赛场的首次交锋，陈巍继美锦赛后再度上演两套零失误的节目。滴水不漏的短节目107.40分，以领先近13分的优势位列第一。自由滑比赛中，在两届冬奥会冠军[[羽生结弦|羽生结弦]]完美发挥、自由滑和总成绩打破世界纪录的情况下，陈巍展示强大心理素质和超强实力，行云流水般完美的表演，滑出令人叹为观止的216.02分，总分323.42，一举打破羽生刚创造的自由滑和总成绩世界纪录，毫无悬念地成功卫冕。<ref>{{Cite news | url=http://newsxmwb.xinmin.cn/wentihui/wtrw/2019/03/24/31505902.html | title=芭蕾范儿 花滑腕儿 华裔小将陈巍蝉联花滑世锦赛冠军 | author = 厉苒苒 |date=2019-1-27 |publisher=新民晚报 }}</ref>成为继亚古丁之后，在十多岁之龄卫冕的世界冠军，为21世纪以来第一人。也是继[[斯科特·汉密尔顿|斯科特·汉密尔顿]]之后，第一个卫冕世界冠军的美国男单选手。<ref>{{Cite news | url=https://olympics.nbcsports.com/2019/03/25/by-any-measure-nathan-chens-performance-at-worlds-matches-standard-for-transcendent-greatness/| title=By any measure, Nathan Chen’s performance at Worlds matches standard for transcendent greatness |author = Philip Hersh |date=2019-3-25 |publisher=NBC SPORTS }}</ref>这场陈巍与[[羽生结弦|羽生结弦]]的经典之战，双双突破了110分技术分与300分总分，为双雄竞争的局面立下里程碑。<ref>{{Cite news | url=https://www.theguardian.com/sport/2019/mar/29/nathan-chen-yuzuru-hanyu-figure-skating| title=Nathan Chen v Yuzuru Hanyu: the rivalry that has taken over figure skating |author = Nick McCarvel |date=2019-3-29 |publisher=英国卫报 }}</ref>陈巍近两个赛季12战11胜的傲人战绩，让他蝉联年度世界排名第一，世界总排名（三年合计）第一。<ref>{{Cite web |url = http://www.isuresults.com/ws/wr2018-19/wrmen.htm |title = 2018-2019 ISU Season's World Ranking| date = 3/24/18 |publisher = 国际滑冰联盟}}</ref>

四月代表美国参加世界团体锦标赛，个人成绩排名第一，团体成绩第一。

== 赛事记录 ==
[[File:How_Nathan_Chen's_Quad_Record_Is_A_Game_Changer_For_The_2018_Olympics.webm|thumb]]
*现行评分规则的总分最高纪录323.42分（2019世锦赛）。

*21世纪以来最年轻的男单世界冠军（18岁）。
*1966年以来最年轻的男单全美冠军（17岁）。

*赢得大奖总决赛的最年轻的美国男单花滑运动员（18岁）。
*第一位数度赢得大奖总决赛冠军的美国男单花滑运动员。
*[[2018年世界花式滑冰錦標賽|2018年世锦赛]]成为第一位在赛事中完成八次四周跳（短节目4Lz3T，4F。长节目4Lz，4F2T，4F，4T，4T3T，4S）的选手，大幅领先47.63分优胜为的史上最高分差。<ref>{{Cite web |url = https://www.theguardian.com/sport/2018/mar/24/nathan-chen-wins-figure-skating-worlds |title = Nathan Chen completes six quads to capture world figure skating title | date = 24/3/18 |publisher = 英国卫报 }}</ref>
* [[2018年冬季奥林匹克运动会|2018年平昌冬奥]]成为第一位在一套节目中完成六次四周跳（五次完美落冰，一次手扶冰）的选手。技术分127.64分史上最高，长节目总分215.08为奥运史上最高。 <ref>{{Cite web |url = http://www.isuresults.com/results/season1718/owg2018/OWG2018_MenSingleSkating_FS_Scores.pdf |title = 2018平昌冬奥男子花滑长曲计分表| date = 17/2/18 |publisher = 国际滑冰联盟 }}</ref>
* 第一位在一套节目中成功完成两次勾手四周跳的选手（2017年大奖赛美国站）。
* 第一个在比赛里完成五种四周跳的选手（2017年美国秋季国际经典赛）。
* 第一个在比赛里完成七次无失误四周跳的选手（2017年[[美國花式溜冰錦標賽|美国花样滑冰锦标赛]]）。
* 第一个在比赛里完成四种不同动作四周跳（后外点冰四周跳、后内四周跳、勾手四周跳、后内点冰四周跳）的选手（2017年美国花样滑冰锦标赛、2017年[[四大洲花样滑冰锦标赛|四大洲花样滑冰锦标赛]]）。
* 第一个在比赛里自由滑项目中完成五个四周跳的选手（2017年美国花样滑冰锦标赛）。
* 后外点冰四周跳（13.76分，2017年美国花样滑冰锦标赛）、后内点冰四周跳（14.73分，2017年四大洲花样滑冰锦标赛）、后内点冰四周跳接后外点冰三周跳（18.03分，2016年NHK杯国际花滑大奖赛）、勾手四周跳接后外点冰三周跳（20.33分，同时也是目前花滑单一元素最高得分，2017年四大洲花样滑冰锦标赛）四项动作最高分保持者。

* 第一个在ISU比赛里完成菲利普四周跳接后外点冰三周跳的选手（2016年日本NHK大奖赛）。
* 第一个在ISU比赛里完成勾手四周跳接后外点冰三周跳的选手（2016年法国大奖赛SP）。<ref>{{Cite web |url = https://sportiva.shueisha.co.jp/clm/othersports/figure/2017/04/17/17/index_2.php |title = 羽生結弦の最強ライバルに。17歳ネイサン・チェンが驚速で成長中！| date = 17/4/17| author = 折山淑美 |publisher = STORTIVA }}</ref>

== 个人荣誉 ==
* 盐湖郡订5月16日为“陈巍日”。<ref>{{Cite web |url = https://www.sltrib.com/sports/2018/05/16/utahs-nathan-chen-awarded-proclamation-making-may-16-nathan-chen-day-in-salt-lake/ |title = Utah’s Nathan Chen awarded proclamation making May 16 ‘Nathan Chen Day‘ in Salt Lake | author = Christopher Kamrani |date = 16/5/18 |publisher =盐城报}}</ref>

* 获选可口可乐，普利司通，联合航空，耐克，家乐氏等品牌 [[2018年冬季奥林匹克运动会|2018平昌冬奥]]代言人。

* 入选美国奥林匹克委员会授予的2016年12月<ref name=USOC011017/>，2017年1月<ref name=USOC070217/>，10月，12月，2018年8月美国队成最佳男子运动员。2016年度全美最佳16名运动员。<ref name=SM121916/>

* 获滑冰杂志2016-17读者之选奖（关颖珊盃）。<ref>{{Cite web |url = http://www.usfigureskatingfanzone.com/uncategorized/nathan-chen-wins-readers-choice-award// |title = Nathan Chen Wins Readers’ Choice Award |date = 8/2/17 |publisher =美国花样滑冰协会}}</ref>

*  2017年5月获邀参加"Stars on Ice"巡回演出。<ref>{{Cite web |url = http://www.starsonice.com/skater-detail.html?sid=99/ |title = Stars on Ice |publisher = IMC}}</ref>

* 2017年获选时代杂志“下个时代领导人物”<ref>{{Cite web |url = http://time.com/collection-post/4684920/nathan-chen-next-generation-leaders/ |title = NEXT GENERATION LEADERS - The Ice Prince | author = Alice Park | date = 3/2/17 |publisher = 美国时代杂志}}</ref>

* 荣膺美国职业花滑联合会授予的2016年美国花样滑冰锦标赛最佳男子表演奖。<ref name=PSA030816/>

* 荣获2009年、2010年、2011年、2012年、2013年、2014年和2015年{{Link-en|迈克尔·韦斯|Michael Weiss (figure skater)}}金会奖学金。<ref name=MichaelWeiss/>

* 荣膺2012年“荣恩与盖尔赫什博格奖”。<ref name=rgha/>

* 荣膺2011年“运动员校友大使奖”。<ref name=IN110516/>

* 荣获2011年、2010年“塞西莉亚学院纪念基金奖”。<ref name=IN110202/>

* 荣获2015年、2014年、2013年、2012年、2011年、2010年、2009年和2008年美国花滑纪念基金。<ref name=USFSA-memorialfund/>

* 入选2016–17年度、2015–16年度、2014–15年度、2013–14年度、2012–13年度、2011–12年度、2010-11年度美国花样滑冰国家队。<ref name=USFSA-team/>

* 2010年2月19日获选[[美国广播公司|美国广播公司]]《世界新闻》“本周人物”。<ref name=ABC100219/>

* 2010年5月1日至5日，获邀参加[[中国|中国]][[浙江省|浙江省]][[杭州|杭州]]举办的“{{lang|en|WiWa}}冰上秀”。<ref name=IFSM1007/>

* 自2010年开始获邀参加[[爱达荷州|爱达荷州]]{{Link-en|森瓦利 (爱达荷州)|Sun Valley, Idaho|森瓦利}}的“森瓦利夏日冰上秀”。<ref name=IME100630/><ref name=SunValley/>

* 获邀成为亚洲滑冰邀请赛（{{lang|en|Skate Asia}}）嘉宾在2010年（中国杭州）、2011年（[[泰国|泰国]][[曼谷|曼谷]]）、<ref name=ifssa11/> 2012年（[[马来西亚|马来西亚]][[吉隆坡|吉隆坡]]）<ref name=skateasia12/> 和2013年（泰国曼谷）表演。
