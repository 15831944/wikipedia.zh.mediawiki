'''陈剑飞'''（{{bd|1962年|5月||}}），[[江苏|江苏]][[仪征|仪征]]人，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
陈剑飞曾任[[中央军委装备发展部政治工作局|中国人民解放军总装备部政治部]]组织部部长。2012年，任[[中国人民解放军陆军工程大学|中国人民解放军军械工程学院]]政治委员。中国人民解放军军械工程学院位于河北省。根据2017年1月6日《[[河北日报|河北日报]]》刊登的《河北省第十二届人民代表大会常务委员会公告》（第105号），2016年9月河北省第十二届人大常委会第二十三次会议后，代表出缺19人，其中由驻河北解放军选举的河北省第十二届人大代表陈剑飞等11人因工作原因调离河北省行政区域，2017年1月辞去河北省人大代表职务。2016年，陈剑飞任[[中央军委机关事务管理总局|中央军委机关事务管理总局]]政治委员<ref>{{Cite web|url=http://news.ifeng.com/a/20170919/52076020_0.shtml|title=陈剑飞履新军委机关事务管理总局政委 |publisher=凤凰网|date=2017-09-19}}</ref><ref>{{Cite web |url = http://china.caixin.com/2017-09-19/101146953.html |title =陈剑飞少将任军委机关事务管理总局政委  |author =  |date =  2017-09-19|publisher =  财新网|language =  |accessdate =  |quote =  }}</ref>。
