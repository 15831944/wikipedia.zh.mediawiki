'''陈嘉堡'''，（{{bd|1969年|7月15日|}}）[[台湾|台湾]][[心理学家|心理学家]]，中国二级[[心理咨询师|心理咨询师]]，APEI国际注册心理咨询师，现担任[[济南|济南]]心理卫生协会专家顾问，量子转念技术创办人<ref name=b/>。著有《还记得我是谁吗?》《我是自己生命的治疗师》、《量子转念的效应》<ref name=c/>等[[书籍|书籍]]<ref name=a>{{cite web |title=陳嘉堡老师公益讲座《揭开命运的神秘面纱》 |url=http://www.cctv-gy.cn/show_12_4598_1.html?from=timeline&isappinstalled=0 |website=www.cctv-gy.cn |accessdate=2019-04-01}}</ref>。

==人物简历==
*陈嘉堡于1969年7月生于台湾，毕业于[[中华民国|中华民国]][[陆军军官学校|陆军军官学校]]（[[黄埔军校|黄埔军校]]），退伍后，开始自主[[创业|创业]]<ref>{{cite web |title=陈嘉堡老师量子转念引导技术10月香港开课 |url=http://wemedia.ifeng.com/81893698/wemedia.shtml |website=wemedia.ifeng.com |accessdate=2019-04-01}}</ref>。
*2014年11月15日，陈嘉堡和其夫人林雨昙女士先后被聘请为济南心理卫生协会专家[[顾问|顾问]]<ref name=b>{{cite web |title=济南心理卫生协会 |url=http://www.jnxlwsxh.com/Intro.asp?m=-1 |website=www.jnxlwsxh.com |accessdate=2019-04-01}}</ref>。
*2019年4月4日，陈嘉堡受新加坡意识无限工作室邀请<ref>{{cite web |title=对话陈嘉堡《量子转念——改变人生的力量》 |url=https://ishare.ifeng.com/c/s/7m03xmCW45j?from=timeline&isappinstalled=0 |website=ishare.ifeng.com |accessdate=2019-04-19}}</ref>，前往[[新加坡|新加坡]]，出席[[量子|量子]]转念公开活动<ref>{{cite web |title=2019年量子转念新加坡大型发表会 - 活动 - CCTV《公益之声》丨传播中国公益好声音 |url=http://www.cctv-gy.cn/show_12_7421_1.html?from=timeline&isappinstalled=0 |website=www.cctv-gy.cn}}</ref>。本次他为新加坡民众带来的[[演讲|演讲]]主题是：《看见不一样的自己》<ref>{{cite web |title=2019年4月4日陈嘉堡老师新加坡站大型公开讲座 |url=http://news.cnhan.com/html/guonei/20190121/997366.htm |website=news.cnhan.com |accessdate=2019-04-01}}</ref>。
