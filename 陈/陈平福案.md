'''陈平福涉嫌煽动颠覆国家政权案'''是指2012年发生在[[中华人民共和国|中华人民共和国]][[甘肃省|甘肃省]][[兰州市|兰州市]][[皋兰县|皋兰县]]的一起已撤诉刑事案件。皋兰县退休教师陈平福因在网络发表和转发多篇文章，被皋兰县[[公安局|公安局]]以涉嫌[[煽动颠覆国家政权罪|煽动颠覆国家政权罪]][[监视居住|监视居住]]，因陈平福患有严重[[心脏病|心脏病]]而未收押入狱，2012年9月4日，[[甘肃省兰州市中级人民法院|甘肃省兰州市中级人民法院]]审查该案件，2012年12月14日，兰州市中级人民法院准许检察机关撤回对陈平福的起诉，该案终结<ref name="fish">{{cite web |url=http://news.qq.com/a/20121218/000050.htm |title=兰州失业教师陈平福发帖被控颠覆国家政权案撤诉 |accessdate=2012-12-18 |publisher= 腾讯网|archiveurl=http://news.qq.com/a/20121218/000050.htm |archivedate=2012年12月18日 |work=新京报作者宋识径 |language=zh-cn}}</ref>。

== 当事人 ==
陈平福，1957年出生于甘肃省平凉市崆峒区，[[大学|大学]][[本科|本科]]生，曾在[[国有企业|国有企业]]子弟学校执教，是[[数学|数学]]教师，因企业破产而[[下岗|下岗]]，以在街头拉[[小提琴|小提琴]][[卖艺|卖艺]]为生，后遭到[[保安|保安]]、[[公安|公安]]、[[城管|城管]]、救助站的驱逐，以“影响市容”的原因禁止其在街头卖艺，2011年，陈平福到[[云南省|云南省]]的一个朋友那谋职，因发表和转载文章被皋兰县人民政府[[跨省遣返|跨省遣返]]<ref name="fish" /><ref name="voa" />。陈平福曾去信访办上访，但是被工作人员轰走<ref name="fish" /><ref name="voa" >{{cite web |url = vhttp://www.voachinese.com/content/citizen-activist-visited-chen-pingfu-20120913/1507246.html |title = 作家陈平福因言获罪 300多人联署反对文字狱 |accessdate = 2012年12月18日 |publisher = 美国之音 |archiveurl = http://www.voachinese.com/content/citizen-activist-visited-chen-pingfu-20120913/1507246.html |archivedate = 2012年12月18日 |work = |language = zh-cn}}</ref>。

== 事件 ==
2007年起，陈平福注册了[[网易|网易]]、[[百度|百度]]、[[搜狐|搜狐]]、[[新浪|新浪]]、[[天涯|天涯]]等大型网站的[[博客|博客]]和[[微博|微博]]，用儿子的电脑在网络发表和转载文章，批评当局，一直到2012年3月<ref name="fish" /><ref name="bull">{{cite web |url = http://news.qq.com/a/20120906/001718.htm |title = 兰州失业教师因发帖被控煽动颠覆国家政权罪 |accessdate=2012年9月6日 |publisher=腾讯网|archiveurl=http://news.qq.com/a/20120906/001718.htm |archivedate=2012年12月18日 |work= 中国甘肃网 |language=zh-cn}}</ref><ref name="BBCnews" />。

2012年6月27日，陈平福被皋兰县公安局以涉嫌[[煽动颠覆国家政权罪|煽动颠覆国家政权罪]][[监视居住|监视居住]]。<ref name="fish" />

8月14日，[[兰州市人民检察院|兰州市人民检察院]]向[[甘肃省兰州市中级人民法院|甘肃省兰州市中级人民法院]]提起公诉，指控陈平福煽动颠覆国家政权，罪证是他发表和转载的34篇文章<ref name="fish" /><ref name="BBCnews" />。

9月4日，兰州市中级人民法院开庭审理，陈平福的辩护律师是甘肃荣庆律师事务所[[何辉新|何辉新]]律师，由于罪名重大等原因而延期，获得[[甘肃省高级人民法院|甘肃省高级人民法院]]的准许<ref name="nan" /><ref name="fish" /><ref name="bull" />。

9月12日，网络作家[[李化平|李化平]]探视陈平福，陈平福对李化平说：“没有网友的关注，他的处境会更糟。”<ref name="voa" />

9月28日，海内外各界人士[[艾晓明|艾晓明]]、[[王书瑶|王书瑶]]、[[朱毅|朱毅]]、[[王荔蕻|王荔蕻]]、[[葛洵|葛洵]]等发起联署签名，要求当局释放陈平福。<ref name="voa" /><ref>{{cite web |url=http://www.rfa.org/mandarin/yataibaodao/ck-09282012092324.html |title=海内外维权人士发起签名活动 要求当局释放陈平福 |accessdate=2012-09-28 |publisher=自由亚洲电台 |archiveurl=http://www.rfa.org/mandarin/yataibaodao/ck-09282012092324.html |archivedate= 2012年12月18日|work= |language=zh-cn}}</ref>

10月24日和11月22日，兰州市检察院两人申请延期审判该案，12月13日，兰州市检察院取消起诉书，该案终结。<ref name=" fish" /><ref>{{cite web |url = http://www.chinese.rfi.fr/%E4%B8%AD%E5%9B%BD/20121216-%E7%BD%91%E4%BC%A0%E5%85%B0%E5%B7%9E%E6%A3%80%E5%AF%9F%E9%99%A2%E6%92%A4%E5%9B%9E%E5%AF%B9%E9%99%88%E5%B9%B3%E7%A6%8F%E7%9A%84%E8%B5%B7%E8%AF%89%E5%BC%95%E5%8F%91%E7%BD%91%E6%B0%91%E6%AC%A2%E5%91%BC |title = 网传兰州检察院撤回对陈平福的起诉引发网民欢呼 |accessdate = 2012年12月16日|publisher=法广中文网 |archiveurl = http://www.chinese.rfi.fr/%E4%B8%AD%E5%9B%BD/20121216-%E7%BD%91%E4%BC%A0%E5%85%B0%E5%B7%9E%E6%A3%80%E5%AF%9F%E9%99%A2%E6%92%A4%E5%9B%9E%E5%AF%B9%E9%99%88%E5%B9%B3%E7%A6%8F%E7%9A%84%E8%B5%B7%E8%AF%89%E5%BC%95%E5%8F%91%E7%BD%91%E6%B0%91%E6%AC%A2%E5%91%BC|archivedate= 2012年12月18日 |work=周西 |language=zh-cn}}</ref>
