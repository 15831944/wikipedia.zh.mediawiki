'''陳靖'''，1951年出生於[[福建省|福建省]][[廈門市|廈門市]]。現定居於[[香港|香港]]。2008年底被中國民間武術家聯誼會評選為「中華武術百傑」和「中華傳統武術名師」。

== 生平 ==
陳靖自幼喜好中華拳學，參學萬里，足遍大江南北，遍訪名師有數百之多，涉獵諸多流派的拳學知識。先後得遇[[陳江霖|陳江霖]]、[[張長信|張長信]]、[[周永福|周永福]]、[[李延璽|李延璽]]、[[孫德明|孫德明]]等明師的言傳、身授、心教。其後更得到楊家太極傳人[[汪永泉|汪永泉]]宗師的兒子[[汪仲明|汪仲明]]老師的悉心栽培，傾囊相授。

陳靖明中華拳道之源來，曉拳學大道之畦徑，能辨析左道之迷惑，善深入淺出之教授。

陳靖精通[[形意拳|形意拳]]、[[六合八法拳|六合八法拳]]、[[太極拳|太極拳]][[老六路|老六路]]等[[內家拳|內家拳]]。對中華拳學中的「[[內功|內功]]」、「[[內勁|內勁]]」的功夫有深刻的體會和論述。著有拳道學說：〈1〉《拳道要畧》〈2〉《拳道中等學 : 太極拳鍊與悟》〈3〉《鑿琢順軌 太極拳論評說 1, 王宗岳太極拳論》。

在陳靖的論著中，拳道境界的高下，不在於拳種的名稱或招式動作，而是決定於功夫所運用能量的性質。也就是「力」與「勁」的分別。

中華拳道的初等學，是「[[力|力]]」的表現。無論是哪種拳，只要是以「力」為能量，使招式產生威力的，都是「力」的功夫。用「力」的功夫，需要[[支點|支點]]。在攻擊的一刹那，全身一緊，達到「整」的效果，發出爆炸力。力的大小，則視乎筋的柔韌、骨的堅硬、肌肉鬆緊，配合速度、距離、時間等因素的配合。<ref>《拳道中等學 : 太極拳鍊與悟》，2004年，頁89</ref>這是明勁的階段。<ref>《拳道要略》，2005年，頁21至24</ref>

中國拳術的高級階段，則是「勁」的表現。「勁」是神、意、氣的化合。也稱為「[[內勁|內勁]]」、「[[內功|內功]]」所謂「以心行氣，以氣運身」，「用意不用力」。無論是哪種拳，只要是以「勁」為能量，使招式產生威力，都是「勁」的功夫。用「勁」的功夫，不需要支點。只要與對手有一接觸點，對手全身任何一點均在內勁的攻擊範圍之內。在攻擊的一刹那，全身一鬆，達到「整」的效果，發出比「力」更強的能量。<ref>《鑿琢順軌太極拳論評說. 1, 王宗岳太極拳論》，2002年，頁38，79至82</ref>這是暗勁至化勁的階段。<ref>《拳道要略》，2005年，頁25至38</ref>

用勁與用力不同的地方，是它以鬆為原則。身體越鬆越不用力，勁就越大。所謂「一分鬆，一分功」。反之，身體哪個部位緊了，勁就在那個地方斷。因為緊張的肌肉就像淤塞的水管阻礙水的流通。用力越多，肌肉就越緊張，產生的內力就越少。這鬆的原則，使「用力的」決不可能冒充「用勁的」。<ref>《拳道中等學 : 太極拳鍊與悟》，2004年，頁43至54，頁89</ref><ref>《鑿琢順軌太極拳論評說. 1, 王宗岳太極拳論》，2002年，頁50至56</ref>

「力」的功夫是有限的，有止境的。一旦年紀老邁，筋骨退化時，一身的功夫也隨而消逝。然而，「勁」的功夫卻没有止境。儘管年紀老邁，筋骨退化，但內勁卻越練越強<ref>陳靖。《拳道要略》。香港：陳靖，修訂版：2005年12月。頁22，33至34</ref>。[[王薌齋|王薌齋]]、[[吳翼翬|吳翼翬]]、[[張長信|張長信]]等先輩，以及現在北京楊家太極[[汪永泉|汪永泉]]一脈的名家[[孫德明|孫德明]]、[[李延璽|李延璽]]等老，就是著名的實例<ref>張長信。《華嶽心意六合八法拳》。香港：陳靖，1990。頁4-9</ref><ref>陳靖。《拳道要略》。香港：陳靖，修訂版：2005年12月。頁133-134</ref><ref>陳靖。《拳道中等學 : 太極拳鍊與悟》。香港：陳靖，第三版：2004年。頁104-105</ref>。

陳靖在論著中，還詳細論述了內勁的培養及壯大之法。著者以親身的實踐體驗和理解，來闡述自己多年的學拳心血和經驗結晶。<ref>《拳道中等學 : 太極拳鍊與悟》，2004年，頁14至15</ref>

每個人都有與生俱來的「內氣」，只是有強弱不同的差異而已。武術家可以通過有效的鍛鍊，培養及壯大本身的「內氣」。成為支援招式攻擊、防守的能量。<ref>《拳道中等學 : 太極拳鍊與悟》，2004年，頁38至41</ref>

培養及壯大「內氣」的關鍵，在於能「鬆」。「鬆者，蓬鬆也；寛而不緊也；輕鬆也；放開也；輕鬆暢快也；不堅凝也。含有小孔以容其他物質之特性也」<ref>吳圖南。《太極拳之研究：吳圖南太極功/ 吳圖南講授 ; 馬有清編著》。香港：商務印書館，2004。頁12</ref>。具體內容包括骨骼鬆、肌肉鬆、臟腑鬆、氣鬆和精神鬆<ref>陳靖。《拳道中等學 : 太極拳鍊與悟》。香港：陳靖，第三版：2004年。頁43至54</ref>。這是在提起精神狀態之下的「鬆」。是令氣機啟動、內氣鼓盪的積極的「鬆」。而求鬆的法則是「先梢後根」。也就是「先手腕，腳脖鬆開，靈活自如。再繼續鍛鍊膝和肘的放鬆。而後至肩、胯等等的放鬆」<ref>陳靖。《拳道中等學 : 太極拳鍊與悟》。香港：陳靖，第三版：2004年。頁102</ref>。能「鬆」，才有能量。不能「鬆」，就没有能量。這是「勁」的能量。所以，[[太極拳|太極拳]]大師[[楊澄甫|楊澄甫]]每日授徒時，必重複十餘次提醒道：「要鬆，要鬆，要鬆凈。要全身鬆開。」「不鬆，不鬆，不鬆就是挨打的架子。」<ref>鄭曼青。《鄭子太極拳十三篇》。台北：蘭溪圖書，1984年。頁45 </ref>。
