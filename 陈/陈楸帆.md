'''陈楸帆'''（Stanley Chan,{{bd|1981年|11月30日|||C陈}}），[[中国大陆|中国大陆]][[科幻|科幻]][[作家|作家]]，出生于[[广东省|广东省]][[汕头|汕头]]，為[[中国|中国]]更新代代表科幻作家之一，以[[现实主义|现实主义]]和[[新浪潮|新浪潮]]风格而著称，被视为“中国的威廉‧吉布森”。<ref>{{cite web |url=http://bestsf.net/chen-qiufan-the-flower-of-shazui-interzone-243-novdec-2012/|title="Chen Qiufan. The Flower of Shazui. (Interzone #243, Nov/Dec 2012)"|accessdate=2012年9月29日|work=MARK WATSON|publisher=http://bestsf.net/|language= en}}</ref>陈楸帆曾在[[百度|百度]]（2007-2008年、2013-2015年）、[[Google|Google]]（2008-2013年）工作，从事市场营销、商务合作、品牌推广等工作，目前为[[诺亦腾科技有限公司|诺亦腾科技有限公司]]品牌副总裁，专注于动作捕捉技术及虚拟现实领域。他同时还身为影视编剧、剧本顾问、专栏作家，并频繁受邀出席各类公众活动，发表于科技、科幻、媒体、影视相关领域的见解观点。

==影响==
=== 译作输出 ===
与[[星云奖|星云奖]]、[[雨果奖|雨果奖]]双料作家[[刘宇昆|刘宇昆]]建立深厚友情，包括《丽江的鱼儿》、《沙嘴之花》、《鼠年》等多作由其翻译成英文版，并发表在《[[Clarkesworld|Clarkesworld]]》、《[[F&SF|F&SF]]》、《[[Lightspeed|Lightspeed]]》、《[[Interzone|Interzone]]》等欧美主流科幻奇幻杂志并获得广泛赞誉，在其中许多媒体都是作为首位中国作家登台亮相，从而扩大了个人和中国科幻文学的国际影响力，其中《丽江的鱼儿》更是获得了第二届科幻奇幻翻译奖最佳短篇奖（Science Fiction & Fantasy Translation Award / Short Form）。<ref>{{cite web |url=http://clarkesworldmagazine.com/chen_08_11/|title="The Fish of Lijiang"|accessdate=2012年8月11日|work=CHEN QIUFAN / KEN LIU|publisher=Clarkesworld Magazine|language= en}}</ref>另一方面，英国作家、编剧[[乔纳森·克莱门茨|乔纳森·克莱门茨]]，也对陈的作品在英语界作了推广之力。<ref>{{cite web |url=http://www.sf-encyclopedia.com/entry/chen_qiufan|title="Chen Qiufan"|accessdate=2012年9月15日|work=Jonathan Clements|publisher=sf-encyclopedia|language= en}}</ref>

瑞典汉学家、翻译家[[陈安娜|陈安娜]]作为[[莫言|莫言]]、[[余华|余华]]、[[苏童|苏童]]、[[韩少功|韩少功]]的瑞典文译者，也对[[陈楸帆|陈楸帆]]作品高度关注并将其作品翻译成瑞典文。<ref>{{cite web |url=http://www.infzm.com/content/81989|title="【2012年诺贝尔奖】1000本，终于卖光——莫言在瑞典的三部小说"|accessdate=2012年10月18日|work=朱晓佳|publisher=南方周末|language= 中文}}</ref>

=== 科幻传播 ===
陈多次在《周末画报》、《LIME》等杂志上与韩松、刘慈欣、刘宇昆等人进行对谈，并将他们以自己的视角推广向潜在的读者群。<ref>{{cite web |url=http://site.douban.com/106829/widget/articles/12199558/|title="Non Ficton"|accessdate=2012年12月1日|work=陈楸帆|publisher=豆瓣作家小站|language= 中文}}</ref>同时他也是极为活跃的科幻活动者及演说者，经常受邀出席科技、媒体、电影及创作相关的公众活动，向大众传播展示科幻的魅力及可能性。
