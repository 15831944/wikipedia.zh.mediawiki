'''陈永贵'''（{{bd|1915年|2月14日|1986年|3月26日|catIdx=C陈}}），[[中國|中國]][[山西|山西]][[昔阳|昔阳]][[大寨镇_(昔阳县)|大寨]]人，為[[农民|农民]]及[[政治人物|政治人物]]，曾[[挂职|同时担任]][[中共中央政治局委员|中共中央政治局委员]]、[[中华人民共和国国务院副总理|中华人民共和国国务院副总理]]兼[[山西省革命委员会|山西省革命委员会]]副主任、中共[[晋中地区|晋中]]地委书记兼[[昔阳县|昔阳]][[县委书记|县委书记]]兼[[大寨|大寨]][[生产大队|大队]][[党支部|党支部]]委员。

== 早年 ==
据映泉著、[[长江文艺出版社|长江文艺出版社]]出版《陈永贵传》第4页记载：“不少人说他生于1914年。早几个月也罢，迟几个月也罢，原本与我们探究他的一生并无多大影响。问题在于，他于自己的出生年月也从没有弄清楚过，不知是出生在头一年冬，还是第二年春。……按照农历，他给自己定为正月初一，那是一个好日子。这是苦人图的吉利。”1915年的正月初一，就是阳历的2月14日。

陈永贵出身[[贫农|贫农]]，6岁时随父迁居大寨；不久，陈父[[自缢|自缢]]身亡，陈寄居在一[[寡妇|寡妇]]家中靠当[[长工|长工]]为生。

陈永贵从未受过正规教育。早在[[中國抗日戰爭|抗日战争]]期间，共产党队伍和[[侵华日军|侵华日军]]在[[昔阳|昔阳]]拉锯。1942年，日本殖民当局强迫村民建立维持会以压制共产党[[游击队|游击队]]活动，他被推选为大寨村日伪组织“维持会”代表，后加入日伪组织“[[兴亚会|兴亚会]]”。1943年，陈永贵被关进“留置场”一年，几乎丢了性命；出来后便辞了职。

[[日本投降|日本投降]]之后，昔阳全县掀起了反奸复仇、清算血债的[[群众运动|群众运动]]。陈永贵受到共产党方面的拘留，最后定为“一般历史问题”。

==政治生涯==
[[File:1963-11_1963_陈永贵.jpg|thumb]]
在[[土改运动|土改运动]]中，陈永贵以“受压迫者”身份带领大寨的贫雇农分掉了地主的土地。1948年，陈永贵加入了[[中国共产党|中国共产党]]。

1952年，陈永贵接替“主动让贤”的原书记[[贾进才|贾进才]]出任中国共产党[[山西省|山西省]]昔阳县[[大寨镇_(昔阳县)|大寨]]大队支部书记，在环境非常恶劣的、被[[石山|石山]]包围的大寨村，他带领农民制定了改造大自然的规划。他们凭着扁担、箩筐、锄头、铁镐，在土石山上开沟造地，平整田地，蓄水保粮，抗旱防涝，用秸杆还田，增施农家肥，改良了土壤，使粮食产量逐年上升。1952年亩产237斤，1962年增至亩产774斤。1963年8月，大寨遭受特大暴雨，冲垮100条大石坝，颗粒无收的耕地180亩，冲塌了113孔窑洞，倒塌房屋77间。灾情十分严重，但大寨人没有气馁，他们不要国家救济款、救济粮、救济物质，自力更生，艰苦奋斗，仅用一年时间就医治了这场毁灭性的灾害。大寨大队正确处理国家、集体和个人的关系，在逐渐改善社员生活的同时，向国家交售粮食175万斤（每户平均2000斤），这在当时困难重重、粮食产量普遍不高的情况下，他们的奉献精神是难能可贵的。他的事迹被中央政府肯定，[[毛泽东|毛泽东]]号召全国要“工业学大庆、[[农业学大寨|农业学大寨]]”，因此成为全国的榜样<ref>{{cite web |url=http://www.syzy.com.cn/baicao/nhf/rdzt/dzdhx/dzc/200611/20061120194054.html |title=存档副本 |accessdate=2010-11-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20101209234305/http://www.syzy.com.cn/baicao/nhf/rdzt/dzdhx/dzc/200611/20061120194054.html |archivedate=2010-12-09 }}《陈永贵墓》，百草园网，2006年11月20日.</ref>。

[[File:1966-11_1966年红卫兵与陈永贵.jpg|thumb]]与陈永贵]]
[[File:1968-03_1968年_毛泽东与陈永贵.jpg|thumb]]
1967年，[[文化大革命|文化大革命]]开始后，山西的[[造反派|造反派]]向[[中共山西省委|中共山西省委]]、[[山西省人民委员会|山西省人民委员会]]夺权，[[周恩来|周恩来]]接见陈永贵时授意“你们也可以组织自己的[[红卫兵|红卫兵]]嘛”。于是陈永贵同已经被打倒的原[[中共山西省委|中共山西省委]]书记[[刘格平|刘格平]]等人组织自己的造反组织，名为“总站派”，为夺取山西政权同另一派“兵团派”展开大规模武斗。1969年，陈永贵被选为[[中国共产党第九次全国代表大会|中国共产党第九次全国代表大会]]代表、[[中共中央委员会|中共中央委员会]]委员，开始进入中央政治领导层。

1972年，时任山西一把手的军队代表[[谢振华|谢振华]]，利用周恩来主导的反对“左”倾的运动，要求将核算权从[[生产大队|生产大队]]降到[[生产队|生产队]]；陈永贵坚决反对，坚持在昔阳县反右倾，得到毛泽东的支持，谢振华被调离山西。另外，谢振华一直想以陈永贵的“[[兴亚会|兴亚会]]”历史问题为借口，把陈永贵整下去；谢被调离山西，他整治陈的打算也就无从实施。1973年，在[[中国共产党第十次全国代表大会|中国共产党第十次全国代表大会]]（中共十大）上，为了弥补[[林彪|林彪]]等人遗留的空缺，陈永贵等人被选为[[中共中央政治局|中共中央政治局]]委员，成为了[[副国级|副国级]]“[[党和国家领导人|党和国家领导人]]”。

陈永贵的上升，是毛泽东要实现中华人民共和国成为名副其实的[[工农联盟|工农联盟]]国家的“[[乌托邦|乌托邦]]”思想结果。与陈永贵同时期在政治上上升的还有工人[[吴桂贤|吴桂贤]]、[[孙健|孙健]]。

== 國務院副總理 ==
1975年1月，在中国共产党中央委员会主席[[毛泽东|毛泽东]]的主持下，陈永贵被选为'''[[中共中央政治局|中共中央政治局]]委员、[[国务院副总理|国务院副总理]]'''，并[[史无前例|史无前例]]地从上到下[[挂职|同时担任]]'''[[山西省革命委员会|山西省革命委员会]]副主任、中共[[晋中地区|晋中]]地委书记兼[[昔阳县|昔阳]][[县委书记|县委书记]]兼[[大寨|大寨]][[生产大队|大队]][[党支部|党支部]]委员'''。当时副总理的工资为每月400多元，并且无论在哪里视察，都得按照规定自缴饭费和[[粮票|粮票]]。陈永贵不领工资，只挣在大寨的工分，无法承担伙食费，所以政府另行补助他每月120元“生活补贴”。

他请求在工作期间以1/3时间到全国调查、1/3时间回大寨务农、剩余1/3时间留在中央工作，受到毛泽东的批准和肯定。

他负责[[中国农业|中国农业]]，但无明显建树。他在调查期间，被认为仅以自己的经验为准。如到[[甘肃|甘肃]]看到到处是黄土，就批评甘肃的农业（甘肃和缺土的大寨不同，[[黄土高原|黄土高原]]有的是土但缺乏水）。他又实行“左”的路线，他在全国实行“革资本主义尾巴”：取消[[自留地|自留地]]，生产核算权放在大队一级等。在[[邓小平|邓小平]]主导实行[[改革开放|改革开放]]后，陈的路线都被取消。但他对新政策抵触，拒不在大寨下放自留地，也没有奉命执行“[[实践是检验真理的唯一标准|实践是检验真理的唯一标准]]”的讨论。陈于1980年9月被迫辞职。之后在[[北京|北京]]东郊农场担任顾问。他极力反对邓小平的[[改革开放|改革开放]]政策，也批评邓为[[四人帮|四人帮]]辩护，有消息称其下台后思想有所转变。<ref>{{cite web |url=http://www.leadfellows.cn/classics/6700540e4e006b21770b671b96486c388d35/ |title=存档副本 |accessdate=2009-02-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20160305002915/http://www.leadfellows.cn/classics/6700540e4e006b21770b671b96486c388d35/ |archivedate=2016-03-05 }}陈大斌：《最后一次看望陈永贵》，载《炎黄春秋》2005年第6期。</ref>

==离世==
[[File:Dazhai.jpg|right]]]]

陈永贵于1986年3月26日在[[北京|北京]]因[[肺癌|肺癌]]去世。按其遗嘱，骨灰安放回大寨。

==陵园==
[[File:ChenYonggui.jpg|left]]

位于山西省昔阳县大寨虎头山。陈永贵5米多高雕像的前面，是大寨展览馆和大寨虎头山故居。陈永贵雕像的后面，是陈永贵墓。陈永贵墓共4层台地，228级青石台级，前有墓碑，上书“陈永贵之墓”，碑阴镌刻陈永贵传略。碑后建墓冢，圆形平顶，高1．5米，上饰一桃形石墩。

陵园仿照南京[[中山陵|中山陵]]的风格，台阶分三组：一组三十八阶寓意陈永贵的党龄，一组七十二阶寓意陈永贵年龄，一组八阶寓意陈永贵在[[中共中央|中共中央]]工作年限。墓地两旁梯田式的花坛内，松柏亭亭玉立，百花争芳斗艳。

== 争议 ==
[[王震|王震]]曾经表示，是陈永贵提出「割资本主义尾巴」，而当时毛泽东表示反对陈的说法<ref>{{cite web |title = 王震晚年怒批《河殇》 |url = http://www.guancha.cn/ZhongHuaWang/2013_11_13_185329.shtml |website=中华网 |accessdate=2017-05-24 }}</ref>。

==家族==
元配李虎妮（1926－1965），生有两子。遗孀宋玉林（1927－2018），与丈夫均为北京户籍，生有一子一女。

* 长子陈明珠，昔阳县委宣传部部长，已退休。
* 二子陈明善，昔阳县电业局工作。
* 女儿陈明花，昔阳县公安局警官。
* 幼子陈明亮，澳大利亚成功商务集团董事和广东东莞新中食品有限公司的集团主席。1998年移居[[澳大利亚|澳大利亚]]。
