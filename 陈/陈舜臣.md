'''陈舜臣'''（{{bd|1924年|2月18日|2015年|1月21日|Chin Shunshin}}），[[亞洲|亞洲]]知名作家，[[臺灣|臺灣]][[臺北|臺北]][[新莊|新莊]]人，[[祖籍|祖籍]][[福建省|福建省]][[泉州府|泉州府]]，生于[[日本|日本]][[兵庫縣|兵庫縣]][[神户市|神户市]]元町，[[歷史小說|歷史小說]]作家。1990年取得日本國籍，擁有中華民國與日本雙國籍<ref>1971年後由於日本不承認中華民國，因此1990年入籍時並未要求其放棄中華民國國籍。</ref>。1998年受章勳三等瑞宝章。[[日本艺术院|日本艺术院]]会员。日本[[中国歷史|中国歷史]]小说的先行者。代表著作{{《}}鸦片战争{{》}}、{{《}}太平天国{{》}}、{{《}}秘本三国志{{》}}、{{《}}小说十八史略{{》}}等。琉球史小說{{《}}琉球之風{{》}}及{{《}}憤怒的菩薩{{》}}還被改編成電視劇。翻译了{{《}}[[鲁拜集|鲁拜集]]{{》}}。死後[[日本政府|日本政府]][[追贈|追贈]][[從四位|從四位]][[品秩|品秩]]。

== 生平 ==
陳舜臣之父[[陳通|陳通]]是一位台灣貿易商，在[[台灣日治時期|台灣日治時期]]往來台灣與日本之間做生意，為經商之便，於1919年與其家庭一同移居日本。陳舜臣在家中十名子女中排行第二，在日本[[神户市|神户市]]元町出生、長大。

幼時受其祖父[[陳恭和|陳恭和]]所影響，大量吸收漢學知識，奠定其歷史素養<ref name="端文一">{{cite web |title=在台灣、日本和中國之間流轉，陳舜臣的身份和文學 |url=https://theinitium.com/article/20160510-taiwan-Chen-Shun-Chen/}}</ref>。高中就讀第一神港商业学校（即后来的神户市立神港高等学校）。1941年，陳舜臣进入大阪外国语学校（现[[大阪大学|大阪大学]]外国语学部）印度语学科，专攻[[印地语|印地语]]和[[波斯语|波斯语]]。陳舜臣和[[司马辽太郎|司马辽太郎]]（蒙古语科）、俳人[[赤尾兜子|赤尾兜子]]是同学。1943年毕业，陳舜臣在同校的西南亚西亚语研究所当助手，做[[印度语|印度语]]词典的编纂工作。

在[[第二次世界大戰|第二次世界大戰]]結束後，因為其[[在日台灣人|在日台灣人]]的身份，陳舜臣被劃歸[[中華民國國民|中華民國國民]]，失去了日本国籍和工作。1946年3月，陳舜臣返回台湾，在台北县[[新北市立新莊國民中學|新莊初級中學]]当英语教师<ref>{{cite web |title=陳舜臣與新莊初級中學銜牌合影 |url=https://scontent-tpe1-1.xx.fbcdn.net/v/t1.0-9/12920527_1701150206832665_752828406855501333_n.jpg?_nc_cat=0&oh=a7e88d0b6f897702766d59e5e11a9e18&oe=5BD12880}}</ref>；三年半（1946年3月－1949年9月）後回到神户，回歸家族事業，持[[中華民國護照|中華民國護照]]往來臺灣與日本<ref name="蕭雄淋">蕭雄淋, [http://www.lawtw.com/article.php?template=article_content&area=%20&parent_path=,1,561,195,&job_id=26606&article_category_id=385&article_id=12061 諸葛孔明一書的出版爭議], [[自立晚報|自立晚報]], 1992年4月20日十八版（蕭雄淋律師著《著作權法漫談（二）》）</ref>，從事贸易为生，在閒暇時進入东洋史学者[[宮崎市定|宮崎市定]]門下學習。

1961年，陳舜臣在神户出版了长篇推理小说《枯草之根》获得[[江户川乱步奖|江户川乱步奖]]，从此之后，他成為作家。后来陳舜臣陆续在《宝石》、《讲谈俱乐部》、《小说中央公论》等雜誌發表作品。初期的作品获得了江户川乱步奖、[[直木奖|直木奖]]、[[日本推理作家协会奖|日本推理作家协会奖]]。

1967年，陳舜臣出版第一部历史题材小说《[[鸦片战争|鸦片战争]]》，获得了众多读者的支持。后来的题材有：日本与中国的交流、中国的[[佛教|佛教]]、[[伊斯兰教|伊斯兰教]]、[[基督教|基督教]]等。从此之后，陆续出现了[[宫城谷昌光|宫城谷昌光]]、[[酒见贤一|酒见贤一]]、[[伴野朗|伴野朗]]、[[塚本青史|塚本青史]]等中国历史小说作家。后来更是出现了[[田中芳树|田中芳树]]代表的陈舜臣山派。他的琉球史小说《琉球之风》在1993年改編成NHK[[大河剧|大河剧]]。后来，陳舜臣出版了《中国任侠传》、《唐朝传奇》等中国古典风格的历史小说。

陳舜臣因寫作歷史需入出[[中華人民共和國|中華人民共和國]]找資料，而引起中華民國駐日本大使館的注意，在他赴其他國家開會時於[[簽證|簽證]]上予以刁難。1973年，陳舜臣申請[[中華人民共和國護照|中華人民共和國護照]]。1989年6月4日，[[北京|北京]]發生[[六四天安門事件|六四天安門事件]]，陳舜臣召開記者會，揮淚聲明放棄[[中華人民共和國國籍|中華人民共和國國籍]]，改入日本籍。但陳舜臣從未向[[中華民國政府|中華民國政府]]申請喪失[[國籍|國籍]]，並仍保有在[[臺灣|臺灣]][[臺北市|臺北市]][[大同區_(臺北市)|大同區]]之[[戶籍|戶籍]]（陳舜臣在臺灣的親人曾出具1991年12月由大同區戶政事務所核發之陳舜臣的[[戶籍謄本|戶籍謄本]]），因而仍保有[[中華民國國籍|中華民國國籍]]，從而得於1992年享有《[[著作權法|著作權法]]》對[[國民|國民]]著作之保護，其著作據以向[[中華民國內政部|內政部]]申請註冊擁有「台內著字號」著作權執照。<ref name="蕭雄淋"/>

陳舜臣生前任{{tsl|ja|和辻哲郎文化賞|和辻哲郎文化賞}}（一般部门）的选考委员。他是首位连获江户川乱步奖、直木奖、日本推理作家协会奖这三大奖的小說家。

陳舜臣亞洲文藝館於2014年在神戶市[[中央區_(神戶市)|中央區]]開館<ref name="n1">{{Cite news |title= 陳舜臣アジア文藝館」プレオープン　神戸市中央区 |newspaper= |date=2014-04-16 |author= |url=http://www2.kobe-np.co.jp/odekake/event/%E3%80%8C%E9%99%B3%E8%88%9C%E8%87%A3%E6%96%87%E8%97%9D%E9%A4%A8%E3%80%8D%E3%81%8C5%E6%9C%886%E6%97%A5%E3%83%97%E3%83%AC%E3%82%AA%E3%83%BC%E3%83%97%E3%83%B3%E3%80%80%E7%A5%9E%E6%88%B8%E5%B8%82%E4%B8%AD|accessdate=2014-12-19}}</ref>。

2015年1月21日，陳舜臣在神戶醫院逝世，享壽90岁<ref>[http://tchina.kyodonews.jp/news/2015/01/90467.html 著名日籍華裔作家陳舜臣去世 享年90歲]</ref><ref>{{Cite news |title=作家の陳舜臣さんが死去 90歳、日中の文化的架け橋 |newspaper=朝日新聞 |date=2015-01-21 |author= |url=http://www.asahi.com/articles/ASH1P4711H1PPTFC008.html?iref=comtop_6_01|accessdate=2015-01-21}}</ref><ref>[http://www.jiji.com/jc/c?g=soc_30&k=2015012100456 作家の陳舜臣さん死去＝「阿片戦争」、90歳] 時事通信 2015年1月21日閲覧</ref>。日本政府於陳舜臣逝世後追贈[[從四位|從四位]]<ref>『官報』第6479号 9頁 平成27年2月25日付</ref>。
