'''陈俊愉'''（{{bd|1917年|9月21日|2012年|6月8日|catIdx=Chen陳}}），[[安徽|安徽]][[安庆|安庆]]人，生于[[天津市|天津市]]日租界。曾祖父[[陈尧斋|陈尧斋]]清末曾任新疆布政司，祖父[[陈超衡|陈超衡]]。

1940年毕业于[[金陵大学|金陵大学]]园艺系，并留校任[[助教|助教]]，师从[[汪菊渊|汪菊渊]]教授。1947年任[[复旦大学|复旦大学]]农学院副教授，不久赴丹麦留学，攻读[[园艺学|园艺学]]。1950年回国，执教于[[武汉大学|武汉大学]]。1957年调入[[北京林业大学|北京林业大学]]。

现任[[中国园艺学会|中国园艺学会]]副理事长、[[中国花卉协会|中国花卉协会]]梅花腊梅分会会长、[[北京林业大学|北京林业大学]]教授。1997年被评为[[中国工程院|中国工程院]]院士，1998年6月转为资深院士。1998年8月，被[[国际园艺学会|国际园艺学会]]任命为“国际梅品种登录权威”，被誉为“梅花院士”。<ref name="人民网">{{cite web |url=http://scitech.people.com.cn/GB/1057/4052613.html |title=“梅花院士”——陈俊愉 |accessdate=2009-11-16  |author=李炜娜 |publisher=人民网 |language=zh-hans }}</ref><ref name="中国经济网">{{cite web |url=http://sci.ce.cn/yjjj/rw/rwxw/200904/24/t20090424_18904017.shtml |title=国际梅品种登录权威陈俊愉:梅花人生溢芬芳(图) |accessdate=2009-11-16  |author=王信川 |publisher=中国经济网 |language=zh-hans }}</ref>
