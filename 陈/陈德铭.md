'''陈德铭'''（{{bd|1949年|3月|catIdx=C陈}}），[[上海市|上海市]]人；[[南京大学|南京大学]]国际商学院毕业，数量经济学硕士、[[工商管理学|工商管理学]]博士。1974年9月加入[[中国共产党|中国共产党]]，是[[中国共产党第十七届中央委员会候补委员列表|中共第十七届中央候补委员]]。曾任[[中共陕西省委|中共陕西省委]]副书记、[[陕西省省长|陕西省省长]]，[[国家发展和改革委员会|国家发展和改革委员会]]副主任，[[中华人民共和国商务部|中华人民共和国商务部]]部长，以及[[海峡两岸关系协会|海峡两岸关系协会]]会长等职。屬[[江澤民派系|江澤民派系]]，2012年中共十八大时担任商务部部长的他落選中央委員。現為[[南京大學|南京大學]]特聘教授。

==早年经历==

陈德铭出身于知识分子家庭，早年毕业于上海市西中学。从小痴迷[[无线电|无线电]]，还曾经担任过上海少年宫的无线电兴趣小组组长<ref>[http://www.yn.xinhuanet.com/people/2006-10/08/content_8199719.htm 陈德铭四年三大步：从市委书记到发改委副主任] {{webarchive|url=https://web.archive.org/web/20071121064316/http://www.yn.xinhuanet.com/people/2006-10/08/content_8199719.htm |date=2007-11-21 }} 新华网，原载《南方人物周刊》陈磊</ref>。受[[文革|文革]]影响，陈德铭没有能够在[[无线电|无线电]]方面继续发展；1969年4月，他和大多数同龄人一样，到农村[[插队落户|插队]]劳动，在[[江西|江西]][[瑞金|瑞金]]的谢坊镇落户。后来，受推荐进入[[江西农业大学|原江西共产主义劳动大学]]农机系学习。毕业后，被分配到[[江西省|江西省]]农机局工作。1980年，因为组织上照顾解决夫妻两地分居状况，陈德铭调[[江苏省|江苏省]]食品公司工作。

== 进入政坛 ==

此后一直在[[江苏省|江苏省]]任职，先后出任[[泰州|泰州]]肉联厂副厂长，省商业厅办公室副主任、省食品公司经理、商业厅副厅长，省政府副秘书长、兼省政府办公厅主任，[[苏州|苏州]]市委副书记、市长，[[市委书记|市委书记]]，[[苏州工业园区|苏州工业园区]]党工委书记等职。期间，还于1988年考入[[南京大学|南京大学]]国际商学院，攻读数量经济[[硕士|硕士]]学位；1996年，获得[[南京大学|南京大学]]管理学[[博士|博士]]学位，成为当时还是为数不多的博士官员。2001年9月，陈德铭升任[[中共江苏省委|中共江苏省委]]常委、[[苏州|苏州]][[市委书记|市委书记]]、继续兼任苏州工业园区党工委书记。

2002年5月，陈德铭调到[[陕西|陕西]]工作，先是出任省委常委、副省长兼省政府党组副书记；两年后，任代理省长；2005年2月，正式当选[[陕西省|陕西省]][[省长|省长]]。在陈德铭赴陕西工作后，曾用问候[[苏州|苏州]]的方式，以《来自黄土高坡的思念》为题在《[[苏州日报|苏州日报]]》上发表文章，表达了对苏州的深深眷恋和赴任陕西后感受到的重大责任感。文章发表后，在[[苏州|苏州]]各界引起很大的反响。

然而，在仅担任了一年零七个月的[[省长|省长]]后，中央又一纸调令将陈德铭任命为[[国家发改委|国家发改委]]副主任、兼党组副书记（正部长级待遇）<ref>[http://news.sina.com.cn/c/2006-06-14/18289204161s.shtml 国务院任命陈德铭张茅为国家发改委副主任] 新浪网 2006年06月14日</ref>。

== 从商务部到海协会 ==

2007年10月，在[[中共十七大|中共十七大]]上，陈德铭在[[中国共产党中央委员会|中国共产党中央委员会]]委员选举中出人意料地未能当选[[中央委员|中央委员]]，因而未能顺利接任国家发展改革委主任。但他随后当选为[[中央候补委员|中央候补委员]]<ref>[http://www2.fhy.net/cgi-bin/anyboard.cgi/collections?cmd=get&cG=1383238343&zu=3138323834&v=2&gV=0&p=%20 陈德铭将接掌中国商业部]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }} 枫华园，原载多维新闻网</ref>；此后调任商务部工作。2007年12月2日，[[新华社|新华社]]发布新闻稿称，陈德铭以[[中华人民共和国商务部|商务部]]副部长的身份首次参加对外活动<ref>[http://news.xinhuanet.com/fortune/2007-12/02/content_7185186.htm 陈德铭以商务部副部长身份亮相中日经济高层对话] {{webarchive|url=https://web.archive.org/web/20071206160123/http://news.xinhuanet.com/fortune/2007-12/02/content_7185186.htm |date=2007-12-06 }} 中国新闻网 2007年12月2日</ref>；12月29日，[[第十届全国人大常委会|第十届全国人大常委会]]正式通过任命，陈德铭出任商务部部长。这也是罕见的中央候补委员出任正职内阁部长的情况。

在[[商务部|商务部]]部长任期内，陈德铭努力推动[[ECFA|ECFA]]谈判取得进展。但陈德铭未能推动[[世贸组织|世贸组织]][[多哈回合谈判|多哈回合谈判]]取得实质性进展，也未推动中国加入[[TPP|TPP]]和[[TISA|TISA]]。

2013年4月26日，在[[海协会|海协会]]第三届理事会第一次会议上，被推选为第三任海峡两岸关系协会会长。接替[[陳雲林|陳雲林]]<ref>[http://3g.163.com/touch/article.html?docid=8TCNVF7300014JB5 陈德铭当选海协会新任会长]</ref>。2018年4月27日，由前國台辦主任[[張志軍|張志軍]]接任海協會會長<ref>[http://www.cna.com.tw/news/firstnews/201804270212-1.aspx 前中國國台辦主任張志軍 出任海協會會長]</ref>。
