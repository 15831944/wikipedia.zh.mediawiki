'''陈家祠'''（'''陈氏书院'''）位於[[中国|中国]][[广东省|广东省]][[广州市|广州市]][[中山路_(广州)|中山七路]]恩龙里34号，始建于[[清朝|清朝]][[光绪|光绪]]十四年（1888年），是当时[[广东省|广东省]]七十二县陈姓宗亲合资兴建的合族[[宗祠|宗祠]]。

== 歷史 ==
[[File:Chan_Clan_Ancestral_Hall_Balustrade_Canton.jpg|thumb]]
[[File:Chan_Clan_Ancestral_Hall_Courtyard_Canton.jpg|thumb]]
[[File:Chan_Clan_Ancestral_Hall_1930s.jpg|thumb]]
[[File:Chan_Clan_Ancestral_Hall_Canton.jpg|thumb]]
陳氏書院建于[[清朝|清朝]][[光绪|光绪]]十四年至二十年（1888年至1894年）间，由[[黎巨林|黎巨林]]設計。作為[[書院|書院]]，始建時用作[[廣東|廣東]]各縣[[陳氏|陳氏]]子弟來[[省城|省城]]應[[科舉|科舉]]時學習及住宿場所，也是祭祀祖宗的宗祠，為私人物業。

[[中華民國|中華民國]]年間，陳家祠內供奉的[[牌位|牌位]]，據統計有上萬塊之多。日軍佔領廣州期间，陳家祠並未遭遇嚴重損毀。

[[中华人民共和国|中华人民共和国]]成立后，1957年，進行了全面維修。1959年，陈氏书院被政府辟为'''广东民间工艺博物馆'''，從此停止祭祖功能。1964年對頭門和聚賢堂進行維修。

1966年[[文化大革命|文化大革命]]期間，萬塊牌位盡被燒毀，儘餘兩塊倖存。兩棟堅厚的木大門上的兩個門神公，被人用鑿子挖成大坑洼。大廳兩面的壁畫被鑿毀。1981年再次重修，1983年重新開放。政府重修陳家祠時，将木大門、牆壁用塗料填充平整，請門神畫原作者陳略用[[中國畫|中國畫]]傳統工筆按原貌繪製。

1988年，[[中华人民共和国国务院|中华人民共和国国务院]]公布陈氏书院为[[第三批全国重点文物保护单位|第三批全国重点文物保护单位]]。1996年，陳氏書院被評為“廣州十大旅遊美景”之首。2001年進行了局部修葺，2002年對全祠灰塑進行維修。

陈家祠被列入2002年和2011年兩個版本的[[羊城八景|羊城八景]]，名为“'''古祠留芳'''”<ref>{{cite web|url=http://www.ycwb.com/epaper/xkb/html/2010-11/16/content_971298.htm|title=广州巨变移步是景 再评新景正当其时|publisher=新快报|date=2010年11月16日|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20130921054016/http://www.ycwb.com/epaper/xkb/html/2010-11/16/content_971298.htm|archivedate=2013年9月21日}}</ref>或“'''古祠流芳'''”<ref>{{cite web|url=http://www.ycwb.com/ePaper/xkb/html/2011-05/19/content_1115178.htm|title=羊城新八景劲揭芳名|publisher=新快报|date=2011年5月19日|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20110526083950/http://www.ycwb.com/ePaper/xkb/html/2011-05/19/content_1115178.htm|archivedate=2011年5月26日}}</ref><ref>{{cite web|url=http://www.ycwb.com/ePaper/ycwb/html/2011-03/18/content_1064708.htm|title=“留芳”还是“流芳”？|publisher=羊城晚报|date=2011年3月18日|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20130921054022/http://www.ycwb.com/ePaper/ycwb/html/2011-03/18/content_1064708.htm|archivedate=2013年9月21日}}</ref>。

== 建筑 ==
[[File:刘庆伏狼驹3.jpg|left]]
[[File:Guangzhou_Chenjia_Citang_2012.11.17_14-13-31.jpg|thumb]]
[[File:Guangzhou_Chenjia_Citang_2012.11.17_13-51-02.jpg|thumb]]
陈家祠是集[[岭南|岭南]]历代建筑艺术之大成的典型代表，包括前院、西院、东院及后院，占地15,000多平方米。其建筑深三进，广五间，由9座廳堂、6個院落、10座廂房和長廊巷組成，建筑中心是高达14.5米的中进主殿“聚贤堂”。整个建筑根据[[中国古建筑|中国古建筑]]形式美的原则，把众多大小不同的建筑物巧妙地组合布局在平面方形的建筑空间里，前后左右，严谨对称，虚实相间，极富层次。长廊、青云巷使整个建筑四通八达，庭院园林点缀其中，形成各自独立而又相互联系的整体。

陈氏书院的主體建築正門兩邊的外牆上有《梁山聚義》、《梧桐杏柳鳳凰群》等6幅大型磚雕；全院的門、窗、屏、牆、欄、樑架、屋脊等處装饰中广泛采用[[木雕|木雕]]、[[石雕|石雕]]、[[砖雕|砖雕]]、陶塑、灰塑、[[壁画|壁画]]和铜铁铸等不同风格的工艺，雕刻技法既有简练粗放、又有精雕细琢，上下呼应、相得益彰。

尤具特色的還有第二進後側長廊上的柚木屏門雙面鏤雕，分別雕有歷代歷史故事和民間傳說“[[三顧茅廬|三顧茅廬]]”、“[[赤壁之戰|赤壁之戰]]”等20幅木雕，被讚譽為“木刻鋼刀雕就的中國歷史故事長廊”。 

中路一、二進間的院子內有石欄杆鑲嵌的鐵鑄雙通花欄板“[[金玉滿堂|金玉滿堂]]”、“三羊啟泰” 。

[[郭沫若|郭沫若]]曾於1959年造訪陳家祠並賦詩讚美其建築藝術：

{{Cquote|天工人可代，人工天不如。果然造世界，勝讀十年書。<ref>{{cite web|url=http://travel.sina.com.cn/china/2013-08-07/0817206421.shtml|title=陈家祠：岭南第一“大宅门”|publisher=新浪旅游}}</ref>}}

'''广东民间工艺博物馆'''，位于[[广东省|广东省]][[广州市|广州市]][[荔湾区|荔湾区]]中山七路恩龙里34号的陈家祠，是收藏、研究、展览以广东地区为主，兼及中国各地民间工艺品的艺术类博物馆，是广州市文化广电新闻出版局直属[[事业单位|事业单位]]<ref name=yange>{{Cite web|url = http://www.gzchenjiaci.com/museum/history.aspx?MenuID=02010101|title = 博物馆沿革|author = |date = |publisher = 广东民间工艺博物馆|language = |accessdate = 2017-07-07|quote = }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

中华人民共和国成立后，广州市人民政府接收了陈家祠，交由广州市行政干部学校使用。1957年12月31日，广州市行政干部学校将陈家祠移交广州市文化局。1958年，广州市文管会负责陈家祠的维修，决定用陈家祠作为馆址筹建'''广东民间工艺馆'''。1959年10月1日，经过全面修缮，以陈家祠作为馆址成立的广东民间工艺馆正式对外开放。[[文革|文革]]爆发后，1966年9月广州市新华印刷厂迁入陈家祠，广东民间工艺馆停止对外开放。1980年底，广州市新华印刷厂迁出，陈家祠经维修后，于1983年2月恢复对外开放。1994年6月23日，经广州市编制委员会同意，广东民间工艺馆更名'''广东民间工艺博物馆'''<ref name=yange/>。2017年，广东民间工艺博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。

广东民间工艺博物馆的藏品分为彩瓷、青瓷、黑瓷白瓷、青花、广州彩瓷、枫溪瓷器、珐琅、广州刺绣、潮州刺绣、墨、漆、广东剪纸、佛山木版年画、潮州麦杆贴画、[[黎族|黎族]]服饰、石湾陶器、广州套色蚀花玻璃、铜镜、金漆木雕、广东木雕、广东石雕、广州象牙雕刻、玉雕、竹雕、椰榄雕等等<ref>{{Cite web|url = http://www.gzchenjiaci.com/museum/gcjpindex.aspx?MenuID=020102|title = 馆藏精品|author = |date = |publisher = 广东民间工艺博物馆|language = |accessdate = 2017-07-07|quote = |archive-url = https://web.archive.org/web/20160402093942/http://gzchenjiaci.com/museum/gcjpindex.aspx?MenuID=020102|archive-date = 2016-04-02|dead-url = yes}}</ref>。

== 绿化广场 ==
陈家祠绿化广场位于陈家祠的东侧，于1999年9月落成，前身是广州电筒厂和低矮的居民楼宇。廣場面积17.155万平方米，绿化面积15.7万平方米，分前广场、下沉广场和小游园三部分，中心下沉广场作为人流集散中心，并与[[康王路|康王路]]地下商业街、[[陈家祠站|地铁站]]衔接，总投资1.44亿元人民幣，是[[广州市人民政府|广州市人民政府]]“一年一小变”的重点工程之一。

2009年，荔湾区人民政府将包括[[广州市第四中学|广州市第四中学]]初中部（原广州市第三十二中学）在内的校舍及商铺拆除，将广场扩建至原面积的两倍。

== 英文名稱 ==
{{further|广州地名}}
陳家祠的英文名稱在1949年前曾有非標準音譯「{{lang|en|Chun-Ka-Chie}}」。「陳」的粵式[[郵政式拼音|郵政式拼音]]原本應為「{{lang|en|Chan}}」，現時已被[[漢語拼音|漢語拼音]]「{{lang|pinyin|Chen}}」替代。現時的英文名稱，博物館方面用「{{lang|en|Chen Clan Ancestral Hall}}」，地鐵和巴士站名用「{{lang|en|Chen Clan Academy}}」，交通指示牌用「{{lang|en|Ancestral Temple of the Chen Family}}」，非常混亂。

在2009年5月實施的《公共標誌英文譯法規範》（下稱《規範》）中，關於陳家祠的譯法被政府單方面定為「{{lang|en|Chen Clan Academy}}」。廣東民間工藝博物館的負責人表示，「{{lang|en|Chen Clan Academy}}」和「{{lang|en|The Chen Clan Academy}}」中的「{{lang|en|Academy}}」意為「學院」，是根據中文「陳氏書院」直接翻譯而來。由於陳家祠雖又稱作「陳氏書院」，但在實際落成之後就從來沒有用作書院的功能，譯成「{{lang|en|Chen Clan Academy}}」或「{{lang|en|The Chen Clan Academy}}」會讓外國人認為這裡是陳氏族人接受教育的地方。因此從功能上反映，陳家祠就是陳氏合族祠，用「{{lang|en|Chen's Lineage Hall}}」更為傳神。而「{{lang|en|Ancestral Temple of the Chen Family}}」的英文譯名，因「{{lang|en|Temple}}」有廟宇之意，同樣誤讀了陳家祠的功能。從2008年起，景區內已逐步將陳家祠的英文翻譯統一為「{{lang|en|Chen's Lineage Hall}}」，其後再於2014年改為「{{lang|en|Chen Clan Ancestral Hall}}」。與此同時，廣東民間工藝博物館作為陳家祠管理機構，希望能與社會上其他的製作英文標誌的部門溝通，將全廣州關於陳家祠的譯名統一起來。<ref>{{cite web|url=http://news.dayoo.com/guangzhou/200911/18/73437_11319084.htm|title=一个陈家祠竟有四个英文译名|date=2009-11-18|accessdate=2014-08-15|publisher=廣州日報|deadurl=yes|archiveurl=https://web.archive.org/web/20140819084624/http://news.dayoo.com/guangzhou/200911/18/73437_11319084.htm|archivedate=2014-08-19}}</ref>
