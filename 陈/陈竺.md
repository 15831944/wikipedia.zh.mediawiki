'''陈竺'''（{{bd|1953年|8月17日|catIdx=C}}），[[男性|男]]，[[汉族|汉族]]，[[祖籍|祖籍]][[江苏省|江苏]][[镇江市|镇江]]，出生于[[上海|上海]]。[[中华人民共和国|中华人民共和国]][[血液学|血液学]]家、[[分子生物学|分子生物学]]家。[[上海第二医学院|上海第二医学院]]医疗系一部血液病学硕士，法国[[巴黎第七大学|巴黎第七大学]][[血液学|血液学]]研究所[[肿瘤|肿瘤]]发病基础专业[[博士|博士]]，[[教授|教授]]、[[研究员_(中华人民共和国职称)|研究员]]（正高级[[职称|职称]]），[[中国科学院|中国科学院]][[中国科学院院士|院士]]，[[美国国家科学院|美国国家科学院]]外籍院士、[[法国科学院|法国科学院]]外籍院士，[[上海交通大学医学院|上海交通大学医学院]]教授、[[上海交通大学医学院附属瑞金医院|上海交通大学医学院附属瑞金医院]]终身教授。

末任[[中华人民共和国卫生部|中华人民共和国卫生部]]部长。现任[[全国人民代表大会常务委员会|全国人民代表大会常务委员会]][[全国人民代表大会常务委员会副委员长|副委员长]]，[[中国农工民主党|中国农工民主党]][[中央委员会|中央委员会]]主席以及[[欧美同学会|欧美同学会]]·中国留学人员联谊会会长、[[中国红十字会|中国红十字会]]会长、[[红十字会与红新月会国际联合会|红十字会与红新月会国际联合会]]副会长。

== 成长背景 ==
陈竺的父亲[[陈家伦|陈家伦]]和母亲[[许曼音|许曼音]]都是著名[[内分泌|内分泌]]专家<ref>{{Cite web|url=http://www.ebiotrade.com/newsf/2005-6/2005627114819.htm|title=我和丈夫陈竺的爱情|accessdate=2017-09-05|author=陈赛娟|date=2005-06-27|work=|publisher=中国生命科学网}}</ref>，所以从小他就有了学医的理想。

中学毕业后，陈竺作为[[知青|知青]]，于1970年4月下放到[[江西省|江西省]][[信丰县|信丰县]][[小江镇_(信丰县)|小江]][[人民公社|人民公社]]山香[[生产大队|生产大队]]的老圳头[[生产队|生产队]]。1973年11月，陈竺离开信丰，来到江西省[[横峰县|横峰县]]凭着自学成为一名“[[赤脚医生|赤脚医生]]”。1975年，他被推荐到江西省[[上饶专区|上饶地区]]卫生学校医士专业学习。1977年11月至1978年9月留校任内科教研组教师<ref>{{Cite web|url=http://news.sohu.com/20080317/n255757573.shtml|title=部长档案：“三国院士”陈竺(图)|accessdate=2017-09-05|author=|date=2008-03-17|work=|publisher=搜狐}}</ref>。

1978年，陈竺被推荐到[[上海交通大学医学院|上海第二医学院]]附属[[上海交通大学医学院附属瑞金医院|瑞金医院]][[内科|内科]]进修。当年10月，高校恢复研究生招生考试，陈竺以总分第二、血液学专业第一的好成绩被录取，师从著名的[[血液病学|血液病学]]专家[[王振义_(医学家)|王振义]]；并于1981年获[[血液病学|血液病学]]硕士学位。

毕业后，陈竺成为瑞金医院血液病研究室的一名内科[[住院医师|住院医师]]。1984年，获机会前往[[法国|法国]]求学；在[[巴黎第七大学|巴黎第七大学]]圣路易医院血液中心实验室任外籍住院医师，并攻读博士学位，主修[[分子生物学|分子生物学]]。后做博士后研究。1986年1月，陈竺的夫人、[[研究生|研究生]]同学[[陈赛娟|陈赛娟]]来到陈竺所在研究所攻读细胞遗传学博士学位，两人合作发表了6篇[[论文|论文]]，在白血病分子生物学研究领域取得了重要成果。

== 崭露头角 ==
1989年7月，陈竺夫妇回国。此后，他牵头承担了一大批国家重点科研项目，在[[血液学|血液学]]、[[分子生物学|分子生物学]]等领域取得了很多突破性成果。历任瑞金医院内科[[主治医师|主治医师]]，上海血液学研究所分子生物学中心实验室主任、[[研究员_(中华人民共和国职称)|研究员]]，上海血液学研究所副所长、所长。1995年11月3日，陈竺当选[[中国科学院|中国科学院]]生物学部[[中国科学院院士|院士]]，是当时中国医学界最年轻的院士。1997年出任上海市科协副主席；1998年，出任中国[[国家人类基因组南方研究中心|国家人类基因组南方研究中心]]主任，成为[[973计划|973计划]]首席科学家；2000年10月，出任中国科学院副院长。

== 从政经历 ==
2007年6月29日，被[[第十届全国人民代表大会常务委员会|第十届全国人大常委会]]第二十八次会议决定为[[中华人民共和国卫生部|中华人民共和国卫生部]]部长人选，接替[[高强|高强]]；成为[[中华人民共和国|中华人民共和国]]继[[傅作义|傅作义]]1974年卸任水利部部长后数十年，第二位非中共人士担任正职部长职务（第一位是[[中国致公党|致公党]]中央主席[[万钢|万钢]]担任科技部部长）。

2012年9月，陈竺加入[[中国农工民主党|中国农工民主党]]；同年12月，接替[[桑国卫|桑国卫]]，成为新一届农工民主党中央主席<ref>{{cite web|url=http://news.ifeng.com/mainland/detail_2012_12/11/20030421_0.shtml|title=卫生部长陈竺当选农工党中央主席 今年9月低调入党|accessdate=2015-05-06|author=|date=2012-12-11|publisher=凤凰网}}</ref>。2013年3月，当选[[第十二届全国人民代表大会常务委员会|第十二届全国人民代表大会常务委员会]][[全国人民代表大会常务委员会副委员长|副委员长]]，跻身[[党和国家领导人|党和国家领导人]]行列。2018年3月连任<ref>{{cite web|title=十三届全国人大一次会议选举产生全国人大常委会副委员长、秘书长|url=http://www.xinhuanet.com/2018-03/17/c_1122550486.htm|website=新华网|accessdate=2018-03-17}}</ref>。

== 红会经历 ==
2015年5月6日，在[[中国红十字会|中国红十字会]]第十次全国会员代表大会会议上，陈竺当选为中国红十字会会长<ref>{{cite web|url=http://www.redcross.org.cn/hhzh/zh/newscenter/gzdt/201505/t20150506_25973.html|title=中国红十字会第十次全国会员代表大会闭幕|accessdate=2015-05-06|author=|date=2015-05-06|publisher=中国红十字会}}</ref>。2017年11月8日，在[[红十字会与红新月会国际联合会|红十字会与红新月会国际联合会]]第21次会议上当选副会长。<ref>{{cite web|url=http://news.163.com/17/1110/08/D2SBA4F10001899N.html|title=副委员长陈竺再获新职 两年前已当选红十字会会长|accessdate=2017-11-10 |author=|date=2017-11-10 |publisher=网易新闻}}</ref>

== 学术研究 ==
陈竺长期从事人类[[白血病|白血病]]和[[基因|基因]]的研究工作。他提出的白血病“[[协同靶向治疗|协同靶向治疗]]”观点，为[[肿瘤|肿瘤]]细胞的选择性分化、凋亡治疗开辟了新路。自1994年，陈竺就开始参与中国[[人类基因组计划|人类基因组计划]]的运筹、组织和管理工作，组建了中国第一个国家级[[基因組|基因組]]研究中心——[[国家人类基因组南方研究中心|国家人类基因组南方研究中心]]以及[[上海系统生物医学研究中心|上海系统生物医学研究中心]]。
