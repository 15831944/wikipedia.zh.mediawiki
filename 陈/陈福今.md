'''陈福今'''（{{bd|1941年|5月}}），[[云南|云南]][[石屏|石屏]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员。

==生平==
1965年7月加入[[中国共产党|中国共产党]]，1965年9月参加工作，[[北京大学|北京大学]]哲学系哲学专业毕业，大学学历，编审。1982年12月到1986年3月，任[[中国煤炭报|中国煤炭报]]社总编辑（正局级）。1986年3月到1990年12月，任[[中共中央办公厅调研室|中共中央办公厅调研室]]副主任（正局级）。1990年12月到1993年5月，任[[中共中央办公厅人事局|中共中央办公厅人事局]]局长、机关党委副书记兼机关纪委书记。1993年5月到1999年12月，任[[中共中央办公厅|中共中央办公厅]]副主任、机关党委书记，[[中央机构编制委员会办公室|中央机构编制委员会办公室]]副主任。1999年12月到2006年7月，任[[国家行政学院|国家行政学院]]党委书记、副院长（正部长级）。他是中共第十六届中央委员，中共第十五届中央纪律检查委员会委员<ref name=jl>{{Cite web |url =http://news.sina.com.cn/o/2006-07-13/16569451679s.shtml  |title = 国务院任命姜异康为国家行政学院副院长 |author =  |date = 2006-07-13 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref>{{cite news|title=陈福今建议修订文物保护法 出台博物馆条例|url=http://www.sach.gov.cn/tabid/1368/InfoID/32240/Default.aspx|accessdate=2012-04-19|newspaper=国家文物局|deadurl=yes|archiveurl=https://web.archive.org/web/20120501211528/http://www.sach.gov.cn/tabid/1368/InfoID/32240/Default.aspx|archivedate=2012-05-01}}</ref>。

==著作==
陈福今长期研究党的建设和政府管理理论，先后在《[[人民日报|人民日报]]》、《[[求是|求是]]》、《[[光明日报|光明日报]]》等报刊发表数十篇文章，主编《领导科学概论》、《公共行政概论》、《公共危机管理》等书稿<ref name=jl/>。
