'''陳姓'''是[[華人|華人]]、[[韓國人|韓國人]]、[[越南人|越南人]]中相當常見的姓氏之一，在《[[百家姓|百家姓]]》中排行第10位。其為華人主要姓氏，在[[香港|香港]]、[[澳門|澳門]]、[[臺灣|臺灣]]、[[新加坡|新加坡]]都是第一大姓；但在[[中國大陸|中國大陸]]是屬第五大姓，其中在[[廣東省|廣東]]、[[福建省|福建]]、[[浙江省|浙江]]、[[江蘇省|江蘇]]、[[海南省|海南]]等省為第一大姓。

根據中華人民共和國户籍管理部門的“全國公民身份信息系统”（NCIIS），陳姓按[[人口|人口]]計算是[[百家姓#當代百家姓|現代中國大陸第五大的姓]]<ref>[http://news.sina.com.cn/c/2007-04-24/161812855907.shtml 中國姓氏排行2007年04月24日16:18 新華網]</ref>，根據1977年中國史學家李棟明，在《東方雜誌》發表的一篇有關「姓」的論文上指出，陳姓是全球華人十大姓之一。<ref>根據1977年中國史學家[[李棟明|李棟明]]，在《東方雜志》發表的一篇有關「姓」的論文上指出，陳姓是全球華人十大姓之一。</ref>中國陳姓人口約7000萬，位居第五，全球約8000萬。

由於在[[現代標準漢語|標準漢語]]及多種[[漢語方言|漢語方言]]中，陳的發音近似其它姓，為了區別清楚，陳姓多以「耳東陳」作為強調，其中“耳”指[[阜部|阜]]字旁“阝”。

== 羅馬字譯寫 ==
*以Chen（[[漢語拼音|漢語拼音]]）與Ch'en（[[威妥瑪拼音|威妥瑪拼音]]，一般多省略送氣音標記）最為常見。（注：有一些姓氏如“曾”也是写成“Chen”的）。[[香港|香港]]地區多使用Chan（[[香港政府粵語拼音|香港政府粵語拼音]]）。在[[新加坡|星]][[馬來西亞|馬]]地區，以[[閩南語|閩南語]]（泉漳話和潮汕話）為母語的華人則多拼寫成Tan。

*在[[东南亚|东南亚]]和海外的华侨群体许多国家的部分陈氏都写成Ching或Chin（较接近[[四邑話|四邑話]]和[[客家话|客家话]]“陈”的读音）。

*还有几个和“陈”的读音相近的姓氏都读“Ching”，如[[程姓|程姓]]、[[岑姓|岑姓]]等

== 起源 ==
根据《[[通志|通志]]·氏族略》所总结，陈姓来源可以归纳为：以国为姓、改姓为陈、外族改姓为陈。
* 陳姓最早出自[[媯姓|媯姓]]，是[[舜|舜帝]]的後裔。據《[[通志|通志]]·氏族略》記載：[[武王克殷|武王克殷]]，建立[[周朝|周朝]]以後，找到舜的後人[[胡公滿|胡公滿]]。封他在陳（今[[河南省|河南省]][[淮陽縣|淮陽縣]]）這個地方，建立了「[[陳國|陳國]]」。陳胡公傳至10世孫[[陈完|陈完]]，陳國內亂，陈完怕株連自己，出奔到[[齊國|齊國]]，其子孫改為[[田氏|田氏]]。或有以故國為氏，稱陳氏。 
* 劉氏改姓，《[[通志|通志]]·氏族略》記載：「廣陵（今[[江蘇|江蘇]][[江都縣|江都縣]]東北）陳氏，實為劉氏，因[[魯國|魯]][[國相|相]]無子，以外孫[[陳矯|劉矯]]為嗣。」指的是漢末[[三國|三國]]時代的[[陳矯|陳矯]]、[[陳騫|陳騫]]一脈。 
* 白氏改姓，《河南官氏志》記載：隴右[[白永貴|白永貴]]，於[[隋|隋]]初改為陳氏，此為萬年（今[[陝西|陝西]][[西安市|西安市]]）之陳氏。白姓原為[[龜茲|龜茲]]王帛氏，因入仕內地，怕胡人之姓遭受歧視，而改為陳氏，史稱陳永貴。他有孫叫令英，史稱陳令英。 
* 侯莫陳氏改姓，據《[[魏書|魏書]]·官氏志》記載：[[北魏|北魏]]時期[[鮮卑|鮮卑]][[貴族|貴族]]侯莫陳氏隨[[北魏孝文帝|北魏孝文帝]][[遷都|遷都]][[洛陽|洛陽]]後，實行[[孝文漢化|孝文漢化]]改革，易胡姓為[[漢姓|漢姓]]，於496年把侯莫陳氏改為單姓陳氏。
* [[女真族|女真族]]陳氏，據陳述《[[漢姓女真姓的對照和說明|漢姓女真姓的對照和說明]]》可知女真皇族完顏氏中，有的在金末已改為陳氏。
* [[蒙古族|蒙古族]]陳氏，據《[[續通志|續通志]]‧氏族略四》載，陳氏，明太祖朱元璋賜給降明的蒙古貴族的姓。如恰恰賜名[[陳守忠|陳守忠]]，哈哈賜名[[陈元_(明朝)|陈元]]等。
* [[滿族|滿族]]陳氏，[[滿族|滿族]]陳氏有兩種情況：一是後金[[努爾哈赤|努爾哈赤]]統治時居住在東北的漢族陳氏被強迫改為滿族陳氏，二是本系滿族姓氏改為陳氏，仍屬滿族；據《[[清朝通志|清朝通志]]‧氏族略八》滿洲旗分內高麗姓載：「陳氏，世居平壤地方，巴顏鑲藍旗包衣人。天聰時來歸，其曾孫二達色任護軍。」其後多由高麗（人其祖上多為漢人）融入[[滿族|滿族]]。劉慶華《[[滿族姓氏錄|滿族姓氏錄]]》載，世居東北張當地方的陳佳氏，後改為陳氏。其實陳佳氏因明代居於遼東，後為金國努爾哈赤所統治。被逼改為雙宇，滿族姓陳佳，即陳家的同音宇。

== [[郡望|郡望]](發源地) ==
* [[潁川郡|潁川郡]]：秦王政十七年（公元前230年）置。相當於今天河南省[[許昌|許昌]]一帶地區。
* [[汝南郡|汝南郡]]：漢高帝四年（公元前203年）置。相當於今天河南省[[上蔡縣|上蔡縣]]至[[安徽省|安徽省]][[淮河|淮河]]以北地區。
* [[下邳郡|下邳郡]]：東漢永平十五年（公元72年）將臨淮郡改為下邳國，南朝宋改為郡。在今天[[江蘇省|江蘇省]]西北部。
* [[廣陵郡|廣陵郡]]：東漢（公元一世紀）置。相當於今天江蘇省[[揚州市|揚州市]]一帶。
* [[東海郡|東海郡]]：漢代的東海郡在今天山東省 城一帶，隋唐的東海郡在今江蘇省[[東海縣|東海縣]]以東，淮河以北。
* [[河南郡|河南郡]]：漢高帝二年（公元前205年）置。相當於今天河南省[[洛陽市|洛陽市]]一帶。
* [[武當郡|武當郡]]：北魏置郡，治所在湖北省均縣西北。這一支出自陳寔之後。
* [[廬江郡|廬江郡]]：漢名舒，屬廬江郡，隋初皇置廬江縣。治所在舒（今安徽省廬江）。轄境相當於今天安徽省長江以北廬江一帶。
* [[馮翊郡|馮翊郡]]：漢武帝置郡，治所在臨晉（今陝西省大荔）。此支陳氏，出自陳宣帝之子沅陵王陳叔興之後。
* [[京兆郡|京兆郡]]：漢時置郡，治所在長安（今陝西省西安）。此支陳氏，出自唐代遷居京兆的陳寔後裔陳忠之後。
