== 演艺经历 ==
2014年9月11日，当时还是建筑设计师的“陈一发儿”在[[斗鱼网|斗鱼网]]开始了自己的首场直播。其直播内容以聊天、唱歌和游戏为主<ref name="陈一发儿甲"/>，并迅速爆红，成为斗鱼一线女主播<ref name="陈一发儿丙">{{cite web|author1=新浪游戏编辑|title=路透社报道斗鱼女主播直播纽约漫展背后的偶然和必然|url=http://games.sina.cn/cyfw/cyhw/2016-10-14/detail-ifxwvpaq1295741.d.html?from=wap|website=新浪网游戏|accessdate=2017-08-29|location=北京|language=zh-cn}}</ref>，与当时的[[周二珂|周二珂]]、[[大表姐_(主播)|大表姐]]、[[冯提莫|冯提莫]]并称“斗鱼F4”。

2016年6月，参加江苏卫视《星厨驾到》节目<ref>{{cite web |title=姜敏赫吴俊余和网络人气主播玩转《星厨驾到》 |url=http://ent.163.com/16/0809/07/BU0TF4HE0003501E.html |website=网易 |accessdate=2018-10-20}}</ref>；9月，发行首支单曲《童话镇》。

2017年4月，担任《网易云音乐斗鱼主播翻唱大赛》评委<ref>{{cite web |title=谁是斗鱼歌王？主播翻唱大赛今日决战开启 |url=http://yue.ifeng.com/a/20170411/39782571_0.shtml |website=凤凰网 |accessdate=2018-10-20}}</ref>；5月20日，发行个人单曲《阿婆说》；5月29日，参加《2017斗鱼嘉年华》。<ref>{{cite web |title=2017斗鱼嘉年华陈一发演唱会完整版视频：阿婆说+童话镇 |url=http://www.sjzcmw.com/news/shipin/201705/27825.html |website=石家庄传媒网 |accessdate=2018-10-20}}</ref>

2018年2月發表單曲《弦上有春秋》<ref name="陈一发儿甲"/><ref name="陈一发儿甲"/><ref name="陈一发儿乙">{{cite web|author1=网易娱乐编辑|title=陈一发：请叫我电竞长泽雅美 最燃血的事是炒股|url=http://gg.163.com/16/1219/16/C8LN5QVG00314QVB.html|website=网易网科技|accessdate=2017-08-29|location=北京|language=zh-cn}}</ref>。

== 人物争议 ==
2018年7月31日，陈一发儿2016年间一段直播秀片段突然间在全网散播，在此视频片段中，陈一发儿使用“南京大屠杀呀、日本刀好快呀、[[东三省|东三省]]不保啦、快请冯将军抗日。”等不当类比形容某一同平台歌唱比赛中选手的日语歌唱得好，并面带笑容。另陈一发儿还曾在游戏中，将游戏人物动作戏称为“参拜[[靖国神社|靖国神社]]”<ref name="ycwb">{{Cite web|url=http://news.ycwb.com/2018-08/02/content_30057751.htm|title=愤怒！网红女主播公然调侃南京大屠杀惹众怒！网友：底线何在？ |accessdate=|author=|date=2018-08-02|publisher=金羊网}}</ref>。“江苏网警”微博接到举报后对此事进行了转发。在“江苏网警”发表相关微博10分钟后，陈一发儿在新浪微博就早年的一些有关南京大屠杀、东三省沦陷的言论发表道歉<ref>{{Cite web|url=http://tech.163.com/18/0731/14/DO2377OV00097U82.html|title=斗鱼主播陈一发被爆曾拿南京大屠杀调侃 发微博道歉|accessdate=|author=观察网|date=2018-07-31|publisher=网易科技}}</ref><ref>{{Cite web|url=http://china.huanqiu.com/article/2018-07/12616619.html|title=“日本刀好快呀！”网络女主播调侃民族惨痛记忆 遭曝光后公开道歉|accessdate=|author=环球网|date=2018-07-31|publisher=}}</ref>。而后中共[[紫光阁_(杂志)|紫光阁杂志]]、[[新华网|新华网]]等官方微博对此事进行了转发，其中新华网微博发表评论称“让挑战底线者彻底凉凉”。斗鱼在先后两次发表声明后宣布封禁陈一发儿直播间，同时启动对所有主播的[[爱国主义|爱国主义]]教育行动<ref>{{Cite web|url=http://www.gamersky.com/news/201807/1080271.shtml|title=斗鱼宣布封禁主播陈一发 将对所有主播展开爱国教育|accessdate=2018-07-31|author=夜雨陈灯|date=2018-07-31|publisher=[[游民星空|游民星空]]}}</ref>。在斗鱼宣布封禁陈一发儿的直播间后，中国大陆各大音乐网站也将陈一发儿的歌曲全数下架，同时中国大陆的视频网站将涉及到陈一发儿的片段全部进行删减（或下线）处理。有个别律师认为，陈一发儿的过去相关言论已经违反了于2018年通过的《[[中华人民共和国英雄烈士保护法|中华人民共和国英雄烈士保护法]]》<ref>{{Cite web|url=http://games.sina.com.cn/g/g/2018-08-08/hhkuskt8901900.shtml|title=一首凉凉！陈一发遭主流媒体点名后 作品被全平台封杀|accessdate=2018-09-07|author=|date=2018-08-08|publisher=新浪游戏 }}</ref>。

2018年9月11日（即陈一发儿于斗鱼直播四周年纪念日），有粉丝在陈一发儿被封禁的直播间赠送火箭等大额礼物、发送祝福弹幕，当日直播间热度一度达到324.5万。对此斗鱼表示，陈一发儿直播间已不能发送弹幕，但用户仍然可以正常进入直播间并使用打赏功能，惟主播不能进行正常的礼物结算<ref>{{Cite web|url=http://news.sina.com.cn/o/2018-09-12/doc-ihiycyfx7205770.shtml|title=陈一发被封后 粉丝为庆祝其直播四周年刷22万礼物|accessdate=2018-09-13|author=|date=2018-09-12|publisher=澎湃新闻 }}</ref>。9月14日，斗鱼将陈一发儿直播间礼物系统关闭，用户赠礼时会提示“由于主播多次违规，积分过低，礼物系统不可用”。但用户仍可送免费道具礼物“鱼丸”来增加主播的热度值。另外斗鱼平台已将“陈一发儿”彻底封禁，用户输入“陈一发儿”或其直播房间号“67373”搜索均无相关内容<ref>{{Cite web|url=https://www.thepaper.cn/newsDetail_forward_2439863|title=陈一发儿礼物系统被关闭，但斗鱼上被封禁还能获打赏的不止她|accessdate=2018-09-15|author=|date=2018-09-14|publisher=澎湃新闻 }}</ref>。此次事件也疑似导致斗鱼客户端于10月[[中华人民共和国国庆节|国庆假期]]期间在各大应用市场下架<ref name=sina>{{cite news|title=下架8天后斗鱼发声明：永久关闭陈一发直播间|url=https://news.sina.com.cn/s/2018-10-13/doc-ihmhafir3110727.shtml|publisher=环球时报 |author=|date=2018-10-13 |accessdate=2018-10-14}}</ref>。

2018年10月12日，斗鱼直播发布声明称，由于平台管理失误，导致陈一发儿直播间仍然出现打赏、弹幕运行事件。声明指出，斗鱼在永久关闭陈一发直播账号的基础上，彻底取消其直播间，并针对相关责任人给予严肃处理<ref name=sina/>。此外，陈一发儿的新浪微博也被封禁。

2013年3月18日，陈一发在公交车上偷拍其他女性裙底，并发布到自己的微博称发福利<ref>{{Cite web|title=鬥魚封禁陳一發，最不願提及的黑歷史曝光，偷拍女生裙底！|url=https://itw01.com/Q9H7CEQ.html|work=ITW01|accessdate=2019-04-16|language=zh-TW}}</ref><ref>{{Cite web|title=陈一发辱华被封杀，然而她做过的事情不止这些，样子都是装出来的-北京时间|url=https://item.btime.com/m_99a1a27532a24e16c|work=item.btime.com|accessdate=2019-04-16}}</ref>。
