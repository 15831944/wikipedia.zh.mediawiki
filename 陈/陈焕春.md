'''陈焕春'''（{{bd|1953年|3月20日|||catIdx=C}}），[[中华人民共和国|中国]]家畜传染病学及动物[[病毒学|病毒学]]专家。

== 履历 ==
1953年3月20日，陈焕春出生于[[湖北省|湖北省]][[恩施市|恩施市]]。1975年7月，陈焕春从[[华中农学院|华中农学院]]畜牧兽医系兽医专业毕业。1984-1988年，陈焕春前往[[德国|德国]][[慕尼黑大学|慕尼黑大学]]攻读研究生，获兽医学博士学位。1988年2月到1989年1月，陈焕春在德国[[柏林自由大学|柏林自由大学]]国际畜牧业生产动物传染病防制研讨班学习。1989年起，陈焕春在华中农大任讲师，副教授。1993-1994年，陈焕春前往德国慕尼黑大学合作科研。1994年12月，陈焕春评为教授。1995年，陈焕春评为博士生导师。同年，担任畜牧兽医学院院长。2000年1-4月，陈焕春到德国[[巴伐利亚州|巴伐利亚州]]动物健康服务中心合作科研。从德国留学归来后，陈焕春一直在华中农业大学主持[[预防兽药学|预防兽药学]]研究室工作。

== 主要贡献 ==
陈焕春率先确定[[猪伪狂犬病|猪伪狂犬病]]在中国的流行，分离鉴定了猪伪狂犬病毒并制定诊断国家标准（GB/T 18641—2002 伪狂犬病诊断技术<ref>{{cite web|url=http://www.dyxmj.com/xm/bz/%CE%B1%BF%F1%C8%AE%BC%BC%CA%F5.pdf |title=GB/T 18641--2002 伪狂犬病诊断技术 |accessdate=2008-06-06 |first=陈焕春 |coauthors=何启盖,李晓成,吴斌,方六荣,金梅林,邱德新,吴美州 |format=PDF |language=zh-hans }}</ref>）。同时，还研制了相关疫苗。研制的油乳剂灭活疫苗获农业部一类新药证书，而研究成果《伪狂犬病鄂A株分离鉴定及分子生物学与综合防治研究》于2001年获得了国家科技进步二等奖<ref>{{cite web |url=http://www.caas.net.cn/caas/achievement/AgriAchievementText.asp?id=2167 |title=伪狂犬病鄂A株分离鉴定及分子生物学与综合防治研究 |accessdate=2008-06-06 |language=zh-hans }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

陈焕春还对[[猪细小病毒|猪细小病毒]]和[[猪圆环病毒|猪圆环病毒]]两个全[[基因组序列|基因组序列]]及多种病原的50多个[[功能基因|功能基因]]进行了克隆[[测序|测序]]，由他分离鉴定的22个基因被[[:en:Genbank|Genbank]]所收录。他共研制了10种基因工程疫苗和10种分子诊断试剂盒，获得3项鉴定成果和14项专家验收认定成果。同时，他还制定了猪细小病毒检疫规程农业行业标准，修定了猪血痢检疫规程检疫检验行业标准。

陈焕春共培养博士[[研究生|研究生]]45名和64名硕士研究生，发表论文142篇，会议论文73篇，专著1部，参编2部，编写论文集6本。

2003年当选为[[中国工程院院士|中国工程院院士]]，成为中国第三位兽医院士<ref>{{cite web |url=http://www.biotech.org.cn/news/news/show.php?id=11199 |title=陈焕春：中国第三位兽医院士 |accessdate=2008-06-05 |date=2004-01-06 |language=zh-hans }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
