'''陈玉田'''（{{bd|1944年|9月||}}）[[河北省|河北省]][[易县|易县]]人，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
[[北京理工大学|北京理工大学]]管理系管理工程专业毕业，在职大学学历。1961年8月参加[[中国人民解放军|中国人民解放军]]。1964年1月加入[[中国共产党|中国共产党]]。历任中国人民解放军某师某团班长、排长、司令部参谋，中国人民解放军某军司令部炮兵室参谋，中国人民解放军某师炮兵团副参谋长。1974年，任中国人民解放军某师炮兵团参谋长。1978年，任中国人民解放军某师炮兵团团长。1981年1月，任中国人民解放军某师参谋长。1983年4月，任中国人民解放军某师副师长。1985年7月，任[[中国人民解放军陆军第六十五集团军|中国人民解放军陆军第六十五集团军]]参谋长。1987年9月到1989年7月，在[[中国人民解放军国防大学|中国人民解放军国防大学]]离职学习。1992年9月，任[[青海省军区|青海省军区]]副司令员。1993年1月，任[[山西省军区|山西省军区]]副司令员。1998年8月，任[[河北省军区|河北省军区]]司令员。1998年10月后，任[[中共河北省委|中共河北省委]]常委、河北省军区司令员<ref>{{Cite web |url = http://news.sohu.com/62/87/news147388762.shtml?term=qyd15 |title = 中共河北省六届委员会常委、书记、副书记简历 |author =  |date =2001-12-09 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://edu.sina.com.cn/gaokao/2010-09-24/1609268633.shtml |title = 北京理工大学杰出校友：陈玉田 |author =  |date = 2010-09-24 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。
