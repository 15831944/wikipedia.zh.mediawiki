'''陈全国'''（{{bd|1955年|11月|catIdx=C陈}}），[[中華人民共和國|中國]][[河南省|河南]][[平舆县|平舆]]人。1973年12月参加工作，1976年2月加入[[中国共产党|中国共产党]]。高考恢復後1981年12月本科畢業於[[郑州大学|郑州大学]]经济系政治经济学专业；1997年1月在職畢業於原[[武汉理工大学|武汉汽车工业大学]][[经济学|经济学]][[硕士|硕士]]；2004年获[[武汉理工大学|武汉理工大学]][[管理学|管理学]][[博士|博士]]学位。[[中国共产党第十七届中央委员会候补委员列表|中共第十七届候补中央委员]]，[[中国共产党第十八届中央委员会委员列表|第十八届中央委员]]，[[中国共产党第十九届中央委员会|第十九届中央委员]]、[[中央政治局委员|中央政治局委员]]。曾任[[中国共产党河南省委员会|中共河南省委]]副书记、[[河北省人民政府|河北省人民政府]]省长、[[中国共产党西藏自治区委员会|中共西藏自治区党委]]书记。2016年8月至今任[[中国共产党新疆维吾尔自治区委员会|中共新疆维吾尔自治区党委]]书记。

== 生平 ==
=== 早年经历 ===
陈全国中学毕业后，18岁入[[中国人民解放军|中国人民解放军]]服役；1977年转业地方，被分配到[[驻马店市|驻马店]]汽车配件厂工作。第二年，[[普通高等学校招生全国统一考试|高考]]制度恢复后，考入[[郑州大学|郑州大学]]经济系（1978年2月-1981年12月）。大学毕业后，分配到[[河南省|河南省]][[平舆县|平舆县]]辛店公社工作（1981年12月-1983年9月）；由中共政坛的最底层，开始了从政之路<ref>[http://news.enorth.com.cn/system/2006/10/28/001446424.shtml 北方网：陈全国简历]</ref>。

=== 政坛轨迹 ===
陈全国进入政界后，曾长期在河南基层工作。历任中共[[驻马店市|驻马店]]地委办公室秘书（1983年9月-1985年4月）、地委副秘书长兼任政策研究室主任（1985年4月-1988年9月），[[遂平县|遂平县]]委书记（1988年9月-1992年9月），[[驻马店市|驻马店]]地委委员（1992年9月-1994年2月），[[平顶山市|平顶山市]]委常委、组织部长（1994年2月-1996年3月），[[漯河市|漯河市]]委副书记、市长等职（1996年3月-1998年1月；期間1995年3月-1997年7月武汉理工大学工商管理学院经济学专业在职攻读硕士学位并获工商管理硕士学位）。1998年1月，任[[河南省人民政府|河南省人民政府]]副省长，成为时任省长[[李克强|李克强]]的副手，在其手下工作五年，被外界视为李的亲信。2000年11月，任[[中国共产党河南省委员会|中共河南省委]]常委、组织部部长。2003年4月，任中共河南省委副书记。2005年11月，任中共河南省委副书记、省委党校校长、河南行政学院院长。

2009年11月，任[[中国共产党河北省委员会|中共河北省委]]副書記、代理[[河北行政长官列表|河北省省长]]<ref>[http://news.xinhuanet.com/politics/2009-12/01/content_12567745.htm 陈全国同志任河北省委副书记] {{webarchive|url=https://web.archive.org/web/20091204151130/http://news.xinhuanet.com/politics/2009-12/01/content_12567745.htm |date=2009-12-04 }} 原载《河北日报》</ref>。2010年1月，在河北省十一届人大三次会议上，正式出任河北省省长。

=== 边疆治理 ===
2011年8月25日，[[中国共产党中央委员会|中共中央]]宣布，任命陈全国为[[中国共产党西藏自治区委员会|中共西藏自治区党委]]書記<ref>[http://www.xizang.xinhua.org/gdbb/2011-08/25/content_23550986.htm 陈全国任西藏自治区党委书记] {{webarchive|url=https://web.archive.org/web/20120415200943/http://www.xizang.xinhua.org/gdbb/2011-08/25/content_23550986.htm |date=2012-04-15 }}</ref>。2012年5月8日，陳全國當選為[[中国人民解放军西藏军区|西藏軍區]]黨委[[第一书记_(中国)|第一書記]]。陈全国在西藏主政期间，虽然外界对陈全国在西藏推行的一些政策褒贬不一，但[[西藏經濟|仅就经济而言]]，西藏确实取得了可观的发展。有数据显示，2012年西藏GDP增速为11.8%；2013年增速12.1%；2014年增速12%，全国排名第一；2015年增速为11%，全区生产总值首次突破千亿元；在2016年上半年，西藏以10.6%增速领跑中国其他省市<ref name="治疆策">{{cite web |author1=李止戈 |title=【治疆策】“特殊”陈全国：闷声做事的边疆派 |url=http://news.dwnews.com/china/news/2019-02-19/60119033.html |website=多维新闻 |accessdate=2019-02-28 |date=2019-02-19}}</ref>。2016年8月28日，不再担任西藏党委书记<ref >{{cite web|url=http://china.caixin.com/2016-08-28/100982627.html | language= | title=吴英杰接棒西藏书记 陈全国已任两省份主官将再履新 |publisher=财新网 |date=2016年8月28日|accessdate= }}</ref>。2016年8月29日，任[[中国共产党新疆维吾尔自治区委员会|中共新疆维吾尔自治区党委]]书记<ref >{{cite web|url=http://news.sohu.com/20160829/n466581623.shtml | language= | title=陈全国任新疆自治区书记 张春贤另有任用 |publisher=新华社 |date=2016-08-29 |accessdate= }}</ref>，同時兼任[[新疆生产建设兵团|新疆生产建设兵团]]黨委第一書記、第一政委。在2017年10月25日召开的[[中国共产党第十九届中央委员会第一次全体会议|中共十九届一中全会]]上，陈全国当选[[中国共产党中央政治局|中央政治局]]委员<ref>{{Cite web|url=http://news.xinhuanet.com/politics/19cpcnc/2017-10/25/c_1121853954.htm|title=中国共产党第十九届中央委员会第一次全体会议公报|accessdate=2017-10-26|work=|date=2017-10-25|publisher=新华网}}</ref>。

陈全国到任新疆前，其前任[[张春贤|张春贤]]采取“柔性治疆”的策略未能有效阻止新疆暴恐事件发生和治安恶化。陈全国到任后，向中共中央总书记[[习近平|习近平]]立下军令状<ref>{{cite web|url=https://www.zaobao.com/special/report/politic/cnpol/story20190228-935597|title=两年无暴恐事件 陈全国铁腕治疆反恐模式将成中国各省模范|work=联合早报|date=2019年2月28日|quote=陈全国三年前由西藏调掌新疆后，曾向中共总书记习近平立下军令状，称将用三年时间彻底改变新疆治安形势。}}</ref>，提出“在新疆，没有稳定一切皆为零”的口号<ref>{{cite web |author1=孟亚旭 |author2=施秀丽 |title=西藏到新疆，“维稳”与“暗访”始终与陈全国相伴 |url=http://news.ifeng.com/a/20170223/50722521_0.shtml |website=凤凰网 |accessdate=2019-02-28 |date=2017-02-23}}</ref>。此后，新疆采取大规模清查表面为政府工作但实际上被指支持疆独的“[[两面人#范围延伸|两面人]]”并调整人事、设立职业技能教育培训中心、社区联保等措施，大幅改善了治安环境。另一方面，陈全国继续推出发展经济、保障就业、安居工程、改善基建、保护生态等政策，让新疆维吾尔自治区生产总值由2012年的7529亿元增加到2017年的10920亿元，年均增长9%。基础设施投资累计完成1.94万亿元，年均增长27.5%，林覆盖率由4.24%提高到4.87%，绿洲森林覆盖率由23%提高到28%<ref name="治疆策"/>。陈全国到任后，自2017年起，新疆没有发生恐怖袭击事件。据《[[明报|明报]]》报道，陈全国在新疆的维稳措施受到中共高層肯定<ref>{{cite news |title=鐵腕治疆 陳全國表現獲中央肯定 反恐模式或「輸出」他省鄰國 |url=https://news.mingpao.com/pns/%E4%B8%AD%E5%9C%8B/article/20190227/s00013/1551205408771/%E9%90%B5%E8%85%95%E6%B2%BB%E7%96%86-%E9%99%B3%E5%85%A8%E5%9C%8B%E8%A1%A8%E7%8F%BE%E7%8D%B2%E4%B8%AD%E5%A4%AE%E8%82%AF%E5%AE%9A-%E5%8F%8D%E6%81%90%E6%A8%A1%E5%BC%8F%E6%88%96%E3%80%8C%E8%BC%B8%E5%87%BA%E3%80%8D%E4%BB%96%E7%9C%81%E9%84%B0%E5%9C%8B |accessdate=2019-02-28 |work=明報 |date=2019年2月27日 |archiveurl=https://web.archive.org/web/20190228065952/https://news.mingpao.com/pns/%E4%B8%AD%E5%9C%8B/article/20190227/s00013/1551205408771/%E9%90%B5%E8%85%95%E6%B2%BB%E7%96%86-%E9%99%B3%E5%85%A8%E5%9C%8B%E8%A1%A8%E7%8F%BE%E7%8D%B2%E4%B8%AD%E5%A4%AE%E8%82%AF%E5%AE%9A-%E5%8F%8D%E6%81%90%E6%A8%A1%E5%BC%8F%E6%88%96%E3%80%8C%E8%BC%B8%E5%87%BA%E3%80%8D%E4%BB%96%E7%9C%81%E9%84%B0%E5%9C%8B |archivedate=2019-02-28}}</ref>。

== 争议 ==
陈全国在新疆采取设立职业技能教育培训中心（外媒称“再教育营”）等一些维稳措施引起了批评。2018年8月29日，美国两党国会中国执行委员会共和党共同主席，共和党参议员[[马可·卢比奥|卢比奥]](Marco Rubio)和共和党众议员史密斯(Chris Smith)联名写信给美国国务卿[[迈克·蓬佩奥|蓬佩奥]]和财政部长[[史蒂芬·姆努钦|姆努钦]]，声称“中国西部自治区新疆的穆斯林在[[新疆再教育营|新疆再教育营]]里受到任意拘留、酷刑，以及对宗教活动和文化的高度限制。他们的日常生活都受到数位监控系统监视”，呼吁[[美国国务院|美国国务院]]和[[美国财政部|财政部]]根据《[[全球马格尼茨基法案|全球马格尼茨基法案]]》，制裁中国政府高官和实行监督政策的共产党官员，其中包括时任新疆党委书记的陈全国。除了卢比奥和史密斯之外，还有7名共和党议员、7名民主党议员和1名独立人士总共17名美国参议员和代表参与联署<ref>{{cite web|url=https://www.dw.com/zh/%E6%96%B0%E7%96%86%E9%A2%91%E4%BC%A0%E4%BA%BA%E6%9D%83%E4%BE%B5%E5%AE%B3-%E8%81%94%E5%90%88%E5%9B%BD%E4%BF%83%E4%B8%AD%E5%9B%BD%E6%94%BE%E4%BA%BA/a-45285697?&zhongwen=simp | language= | title=新疆频传人权侵害 联合国促中国放人 |publisher=德国之声 |date=2018年8月31日|accessdate= }}</ref>。对此，中国外交部发言人[[华春莹|华春莹]]于同年8月30日回应称，“中国少数民族政策和实际情况比美国好很多”，“相关美国议员没有资格就少数民族问题对中国进行横加指责”<ref>{{cite web|url=http://www.gov.cn/xinwen/2018-08/30/content_5317852.htm | language= | title=外交部就我国成立最高人民法院国际商事专家委员会等答问 |publisher=中华人民共和国中央人民政府官网 |date=2018年8月30日|accessdate= }}</ref>。

陈全国于2004年获得[[武汉理工大学|武汉理工大学]]管理学[[博士|博士]]学位，当时他曾撰写论文《中部地区人力资本积累与经济发展相关性研究》。网民指出，这篇论文涉嫌大面积剽窃他人多篇论文，重复内容非常多<ref>{{cite web |title=新疆黨委書記陳全國爆醜聞！博士論文涉大面積剽竊|url=https://news.ltn.com.tw/news/world/breakingnews/2707470 |website=自由時報 |accessdate=2019-03-02 |date=2019-02-23}}</ref><ref>{{cite web |title=中共迫害維吾爾人番外篇 新疆領導人陳全國博士論文被踢爆「大面積抄襲」！ |url=https://www.storm.mg/article/984368 |website=風傳媒 |accessdate=2019-03-02 |date=2019-02-23}}</ref>。
