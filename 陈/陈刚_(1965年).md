'''陈刚'''（{{bd|1965年|4月||catIdx=C陈}}），[[江苏省|江苏]][[高邮市|高邮]]人。男，汉族。1986年12月加入[[中国共产党|中国共产党]]，1990年8月参加工作。[[扬州大学|扬州师范学院]]化学系毕业，[[哈尔滨工业大学|哈尔滨工业大学]]应用化学系高分子材料专业硕士研究生，[[北京大学|北京大学]]无机化学专业理学博士。现任[[中国共产党河北省委员会|中共河北省委]]常委，[[雄安新区|雄安新区]]党工委书记、管委会主任，[[河北省人民政府|河北省人民政府]]副省长。[[中国共产党第十九届中央委员会候补委员列表|中共第十九届中央候补委员]]。

== 早年经历 ==
1980年9月至1984年9月，在[[扬州大学|扬州师范学院]]（今[[扬州大学|扬州大学]]）化学系学习。1984年9月至1987年9月，在[[哈尔滨工业大学|哈尔滨工业大学]]应用化学系高分子材料专业攻读硕士研究生。1987年9月至1990年8月，在[[北京大学|北京大学]]化学系无机化学专业攻读博士研究生。

1990年8月，任[[北京玻璃研究所|北京玻璃研究所]]晶体室工程师、副主任。1994年5月，任北京玻璃研究所副所长。1994年7月，任北京一轻集团有限责任公司副总经理兼北京玻璃研究院副院长。

== 北京政坛 ==
2000年7月，任[[北京市外经贸委|北京市外经贸委]]副主任。2002年3月，任[[中国国际贸易促进委员会|中国贸促会]]北京市分会党组书记、主任。2003年1月，任中共[[朝阳区_(北京市)|朝阳区]]委副书记、副区长、代区长（2004年1月当选区长）。2006年10月，任中共朝阳区委书记<ref>{{cite web|url=http://district.ce.cn/newarea/sddy/201208/01/t20120801_23544304.shtml|title=程连元接替陈刚任北京朝阳区委书记 吴桂英提名区长|accessdate=2016-06-08|date=2012-08-01|publisher=中国经济网}}</ref>。2012年7月3日，任[[中国共产党北京市委员会|中共北京市委]]常委，晋升副部级<ref>{{cite web|url=http://district.ce.cn/newarea/sddy/201207/03/t20120703_23458350.shtml|title=两位陈刚新晋北京市委常委|accessdate=2016-06-08|date=2012-07-03|publisher=中国经济网}}</ref>。

== 贵州政坛 ==
2013年6月，陈刚离开工作23年的北京，任[[中国共产党贵州省委员会|中共贵州省委]]常委<ref>{{cite web|url=http://district.ce.cn/newarea/sddy/201307/04/t20130704_24540757.shtml|title=苟仲文任北京市委常委 陈刚不再担任|accessdate=2016-06-08|date=2013-07-04|publisher=中国经济网}}</ref>。7月5日，任中共贵州省委常委、[[贵阳市|贵阳市]]委书记<ref>{{cite web|url=http://district.ce.cn/newarea/sddy/201307/06/t20130706_24547818.shtml|title=陈刚任贵州省委常委 接替李军任贵阳市委书记|accessdate=2016-06-08|date=2013-07-06|publisher=中国经济网}}</ref>，在贵州工作四年。

== 河北政坛 ==
2017年5月，任[[中国共产党河北省委员会|中共河北省委]]常委<ref>{{Cite web|url=http://heb.hebei.com.cn/system/2017/05/31/018256977.shtml|title=陈刚同志任河北省委常委|accessdate=2017-05-31|date=2017-05-31|publisher=长城网}}</ref>、[[雄安新区|雄安新区]]临时党委书记<ref>{{Cite web|url=http://china.caixin.com/2017-06-03/101096077.html|title=成立两个月 雄安新区临时党委书记换将|accessdate=2017-06-02|date=2017-06-02|publisher=财新网}}</ref>。2017年6月，任中共河北省委常委，雄安新区党工委书记、管委会主任。7月28日，陈刚被任命为[[河北省人民政府|河北省人民政府]]副省长。
