'''陈景元'''（{{bd|1962年|6月||}}）[[辽宁|辽宁]][[宽甸|宽甸]]人，[[中国人民解放军少将|中国人民解放军少将]]，[[中华人民共和国|中华人民共和国]]医学家<ref name=lvxin>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1775492 |title = 解放军总医院医务部主任陈景元履新军委后勤保障部卫生局局长 |author =  |date =  2017-08-26|publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref name=jysh>{{Cite web |url = http://doeh.fmmu.edu.cn/nr.jsp?urltype=news.NewsContentUrl&wbnewsid=89857&wbtreeid=19453 |title =陈景元教授  |author =  |date = 2010-11-03 |publisher = 第四军医大学军事预防医学系劳动与环境卫生学教研室 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1985年7月，自[[中国人民解放军第二军医大学|中国人民解放军第二军医大学]]临床医学专业，获医学学士学位。1991年7月，自[[中国人民解放军第四军医大学|中国人民解放军第四军医大学]]军队卫生学专业，获[[医学硕士|医学硕士]]学位。1996年7月，自中国人民解放军第四军医大学生理学专业，获[[医学博士|医学博士]]学位。2000年到2002年，在[[美国|美国]][[哥伦比亚大学|哥伦比亚大学]]公共卫生学院从事[[博士后|博士后]]研究<ref name=jysh/>。

曾任中国人民解放军第四军医大学军事预防医学系主任、[[长江学者|长江学者]]特聘教授、[[博士生导师|博士生导师]]，同时任国家重点学科（军事预防医学）、[[中华人民共和国教育部|教育部]]“[[211工程|211工程]]”重点建设学科（军事特殊环境学）带头人<ref name=jysh/>。后任中国人民解放军第四军医大学训练部部长，教育部长江学者特聘教授，国家[[973计划|973项目]]首席科学家，[[中国人民解放军总后勤部|中国人民解放军总后勤部]]科技银星，同时担任国家重点学科（军事预防医学）、教育部“211工程”重点建设学科（军事特殊环境医学）、教育部长江学者创新团队带头人，教育部“特殊作业环境健康危害与防护”重点实验室主任，全军寒区军事医学重点实验室主任<ref name=kongjun>{{Cite web |url =  https://www.fmmu.edu.cn/info/1061/1023.htm|title = 陈景元 |author =  |date =  |publisher =  中国人民解放军空军军医大学|language =  |accessdate = 2017-08-26 |quote =  }}</ref>。此后历任中国人民解放军第四军医大学医务部主任，[[中国人民解放军总医院|中国人民解放军总医院]]医务部主任。2017年，任[[中央军委后勤保障部卫生局|中央军委后勤保障部卫生局]]局长<ref name=lvxin/>。

他的主要学术兼职先后有：国际神经毒理学会（INA）常务理事，中国毒理学会理事、副理事长，中国毒理学会神经毒理专业委员会主任委员，[[中华预防医学会|中华预防医学会]]理事，中华预防医学会环境卫生分会常委，全军医学计量科学技术委员会常委，全军军队卫生学专业委员会副主任委员，全军医学科学技术委员会委员，陕西省预防医学会环境卫生专业委员会主任委员，陕西省生理科学会副理事长等职。他是2011年[[西安|西安]]国际神经毒理学大会执行主席<ref name=jysh/><ref name=kongjun/>。

他长期从事特殊环境医学与环境神经毒理学的科研、教学、管理工作。先后主持了国家“973”首席项目、国家重大科技专项、国家“[[十一五规划|十一五]]”科技支撑计划重点课题、[[国家自然科学基金|国家自然科学基金]]重点项目等课题十多项，先后获军队科技进步一等奖、二等奖等奖项。主编多部专著、教材，发表[[SCI|SCI]]收录[[英文|英文]]论文数十篇。2007年被评为中国人民解放军总后勤部优秀教师，2008年获得中华公共卫生与预防医学发展贡献奖，2010年被评为全军医学科技先进个人，2011年享受[[政府特殊津贴|政府特殊津贴]]，2012年被评为全国优秀科技工作者<ref name=kongjun/>。
