'''幼儿师范站'''，为中国[[安徽省|安徽省]][[合肥市|合肥市]]的一座地铁车站，工程站名'''学林路站'''，位于[[文忠路|文忠路]]与[[学林路|学林路]]交叉口，为[[合肥轨道交通3号线|合肥轨道交通3号线]]站点。<ref>{{cite web |title="图书馆"易误解 合肥市民为地铁3号线站名“挑刺” |url=http://www.ahwang.cn/hefei/20190125/1848031.shtml |publisher=安徽网 |accessdate=2019-01-25}}</ref>

== 车站结构 ==
幼儿师范站位于文忠路与学林路交叉口北侧，沿文忠路南北向布置，为地上三层双柱三跨鱼腹岛式站，共设2个出入口。<ref>{{cite web |title=合肥地铁3号线沿线站点公布！开通运营时间定在 |url=http://wemedia.ifeng.com/44596356/wemedia.shtml |publisher=凤凰网安徽 |accessdate=2019-02-01}}</ref>
