'''幼年特發性關節炎'''（JIA, Juvenile idiopathic arthritis），前稱幼年類風濕關節炎，在兒童中，是最常見的持續性關節炎。

==與其他關節炎之區分==
JIA常被誤稱為[[幼年型慢性關節炎|幼年型慢性關節炎]]（JCA），這是不準確的，JIA並不包括所有形式的兒童慢性關節炎。[[關節炎|關節炎]]是關節[[滑膜|滑膜]]（滑膜組織）發炎。幼年特發性關節炎不等同於普遍發生於兒童中之關節炎。

JIA是兒童關節炎中一個子群，可能是慢性、暫時性和有局限性。它明顯不同於常見於成人的關節炎（[[骨性關節炎|骨性關節炎]]、[[類風濕性關節炎|類風濕性關節炎]]），和其他的兒童慢性關節炎，如[[銀屑病關節炎|銀屑病關節炎]]和[[強直性脊柱炎|強直性脊柱炎]]。

== 症狀 ==
JIA最初的症狀並不特異，包括[[嗜睡|嗜睡]]，[[疲倦|疲倦]]，胃口差（可能由藥物引起）。特別在年幼兒童最大的表徵可能是拐步跛行。兒童可能有嚴重[[流感|流感]]樣的病態。主要的臨床病狀是在受影響的關節有持續性腫脹，常發生於膝關節、踝關節、腕關節和手腳的小關節。難以臨床檢測的部份，如脊椎關節、骶髂關節、[[肩|肩]]、[[髖|髖]]和[[下巴|下巴]]，可以用[[超聲波|超聲波]]或[[MRI|MRI]]掃瞄成像技術檢測。

患有JIA的兒童都不同程度的獨特症狀，[[疼痛|疼痛]]是JIA一個重要共通特色，但年幼兒童可能在溝通上難以描述此症狀。後期關節炎的影響，包括關節攣縮（僵硬，關節彎曲）和損害等。

== 病因 ==
JIA的成因一如其名「特發性」仍然未明，是目前一個積極研究的項目。根據目前的了解，JIA的出現，可能是因環境因素令到基因受到影響。 

== JIA的類型 ==
3個主要類型的JIA
=== Oligoarticular JIA===
‘Oligo’解作少數。Oligoarticular  JIA （少數關節JIA）在病發的首6個月會影響五個或更少的關節。前稱為pauciarticular JIA。

=== Polyarticular JIA ===
在疾病首6個月，影響五個或以上的關節。這個亞型的影響，包括[[頸部|頸部]]和[[下顎|下顎]]，以及小關節都會受影響。這種類型多見於小女孩。

=== 系統性JIA（斯蒂爾病） ===
[[斯蒂爾病|斯蒂爾病]]（Still's disease）特點是關節炎、出現發燒和桃紅色皮疹。系統性JIA比較容易診斷，因為出現不定時的發燒及出[[疹|疹]]的（過敏）病徵。不像其他兩種亞型的JIA，它同時平均地影響男性和女性。

系統性JIA可能涉及內部器官，並導致[[漿膜炎|漿膜炎]]、[[心包炎|心包炎]]。SoJIA：System onset JIA

== 發病 ==
JIA在男女雙方都會發生，年齡由學前到十幾歲的兒童。依賴亞型JI的分析其類型及徵狀。

== 流行程度 ==
幼年特發性關節炎的影響，估計在美國有30.0萬兒童。這些兒童，50%的有少數關節JIA、40%的有polyarticularjia JIA和10%的有系統性JIA。
