'''恒生中國企業指數'''（簡稱'''國企指數'''或'''H股指數'''）由[[恒生銀行|恒生銀行]]旗下[[恒生指數有限公司|恒生指數有限公司]]負責計算。國企指數反映了在[[香港交易所|香港交易所]]上市的H股中較大型股的表現。目前，國企指數成分股數目現為50隻，包含H股、民企股及[[紅籌股|紅籌股]]。

[[H股|H股]]是指在[[中國內地|中國內地]]註册成立，在[[香港|香港]]上市的企業，H代表「Hong Kong」。H股企業的主要持股人為[[中國政府|中國政府]]及地方政府，所以亦稱為[[國企股|國企股]]{{Fact|time=2018-07-16T12:13:16+00:00}}。

H股指數期貨於2003年12月8日推出，期權於2004年6月8日推出。恒指及國指期貨是香港最受歡迎的兩個[[股票指數期貨|股票指數期貨]]。

2016年2月底，在香港上市的H股共有229隻（主板206隻<ref>[http://www.hkex.com.hk/chi/stat/smstat/chidimen/cd_hmb_c.htm 中國企業H股公司名單 (主板)]{{zh-hant}}{{zh-hans}}{{en}}</ref>+創業板23隻<ref>[http://www.hkex.com.hk/chi/stat/smstat/chidimen/cd_hg_c.htm 中國企業H股公司名單 (創業板)]{{zh-hant}}{{zh-hans}}{{en}}</ref>）。市值最大的三間公司依次是[[建設銀行|建設銀行]]、[[中國工商銀行|工商銀行]]和[[平安保險|平安保險]]。

== 歷史 ==
國企指數於1994年8月8日推出，但基準日後來更改至2000年1月3日，基準日指數為2000點。[[恒指服務有限公司|恒指服務有限公司]]會每半年檢討一次成份股。

2006年12月27日，受惠於內地[[A股|A股]]在[[聖誕節|聖誕節]]假期急升，恒生中國企業指數開市時已突破10000點，當日收市報10,255.31點，上升665.24點或6.9%，為恒生中國企業指數設立以來首次突破10,000點。當日[[恒生中國H股金融行業指數|恒生中國H股金融行業指數]]更急升1,249.96點或10.6%。

2007年9月19日，為恒生中國企業指數設立以來首次突破15,200點，報15255.49點。

2007年9月20日，恒生中國企業指數設立以來創下新高，收市報15,552.23點，升296.74（1.95%）

2010年3月8日，恒生中國企業指數將成份股固定為40隻。

2017年8月17日，[[恒指服務有限公司|恒指服務有限公司]]公布優化恒生中國企業指數的詳情，共10隻紅籌股和民營企業將加入恒生中國企業指數，指數內的H股數目將維持40隻，因此國指成分股數目將增加至50隻。成分股名單將於2018年2月與定期指數檢討結果一併公布。為減低可能對市場的潛在影響，新增的10隻股份將分5個階段加入國指，為期12個月，首階段於2018年3月進行<ref>{{Cite web |url=http://www.hsi.com.hk/HSI-Net/static/revamp/contents/zh_hk/dl_centre/other_materials/HSCEI20170817c.pdf |title=優化恒生中國企業指數之詳情 |access-date=2017-08-17 |archive-url=https://web.archive.org/web/20170817211201/http://www.hsi.com.hk/HSI-Net/static/revamp/contents/zh_hk/dl_centre/other_materials/HSCEI20170817c.pdf |archive-date=2017-08-17 |dead-url=yes }}</ref><ref>{{Cite web |url=http://www.hsi.com.hk/HSI-Net/static/revamp/contents/zh_hk/news/pressRelease/20170817c.pdf |title=恒生指數公佈優化恒生中國企業指數詳情 |access-date=2017-08-17 |archive-url=https://web.archive.org/web/20170817205941/http://www.hsi.com.hk/HSI-Net/static/revamp/contents/zh_hk/news/pressRelease/20170817c.pdf |archive-date=2017-08-17 |dead-url=yes }}</ref>。

== 各項紀錄 ==
歷史最高為2007年10月30日，恒生中國企業指數曾突破20,600點，高見20,609.10點。

歷史最高收市為10月29日，恒生中國企業指數突破20,400點，收20,400.07點。

歷史最長升浪於2017年12月28日至2018年1月24日出現，連升19日。

== 計算理念 ==
由2006年3月6日開始，國企指數的計算方法改為以流通市值調整計算。經[[流通系數|流通系數]]調整後，每隻成份股的比重上限為15%。此舉主要是為了避免指數受個別大市值成份股（如[[建設銀行|建設銀行]]）所左右而失去代表性。

2010年則更進一步，將每隻成份股的比重上限設定為10%。

== 流通系數調整 ==
[[流通量調整系數|流通量調整系數]]（Freefloat adjusted factor，簡稱[[流通系數|流通系數]]）是流通股份佔總發行股之百份比。各成份股之發行量將經流通系數調整後才用以編算指數，策略性持有的股權將不納入指數編算中。

== 計算公式 ==
今日國企指數的計算公式：

<math> \textrm{Current~Index}= \frac{\sum\textrm{P(t)}\times\textrm{IS}\times\textrm{FAF}\times\textrm{CF}}{\sum\textrm{P(t-1)}\times\textrm{IS}\times\textrm{FAF}\times\textrm{CF}} \times \textrm{Yesterday's~Closing~Index}</math>

* P（t）：Current Price at Day t  現時股價
* P（t-1）：Closing Price at Day (t-1)  上日收市股價
* IS：Issued Shares  已發行股票數量
* FAF：Freefloat-adjusted Factor流通系數
* CF：Cap Factor比重上限系數

基準日是2000年1月3日，基準日指數是2000點。
