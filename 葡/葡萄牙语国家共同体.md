'''葡萄牙语国家共同体'''（{{lang-pt|Comunidade dos Países de Língua Portuguesa}}。[[国际音标|国际音标]]：{{ipa|/kumuni'dad(ɨ) duʃ pɐ'izɨʒ dɨ 'lĩŋgwɐ puɾtu'gezɐ/}}（[[葡萄牙葡语|葡萄牙葡语]]）；{{ipa|/komuni'dadʒi dus pa'izis dʒi 'lĩŋgwa poɾtu'gezɐ/}}（[[巴西葡语|巴西葡语]]）。[[首字母縮略字|首字母縮略字]]：CPLP，简称：'''葡语国家共同体'''）是一个以葡萄牙语为[[官方语言|官方语言]]的国家组成的友好论坛。有超过2.23亿人生活在说葡语的[[主权国家|主权国家]]，遍布全球。

== 组成与成员国 ==
葡语国家共同体于1996年成立的时候有七个国家：[[葡萄牙|葡萄牙]]，葡萄牙在[[南美|南美]]的前[[殖民地|殖民地]][[巴西|巴西]]，五个在[[非洲|非洲]]的前殖民地——[[安哥拉|安哥拉]]、[[佛得角|佛得角]]、[[几内亚比绍|几内亚比绍]]、[[莫桑比克|莫桑比克]]、[[圣多美和普林西比|圣多美和普林西比]]。[[东帝汶|东帝汶]]从[[印度尼西亚|印度尼西亚]]独立出来后于2002年加入共同体。

葡语国家共同体是一个正在建设的集团，八个国家的社会间仅有少量的互相了解。葡语国家共同体的特点之一就是成员国由一种[[通用语|通用语]]及[[文化|文化]]特色所维系，这为这些位于不同[[大陆|大陆]]相去甚远的国家之间铺设了桥梁。

2005年，在[[罗安达|罗安达]]的会议期间，八国文化部长宣布5月5日为葡萄牙语文化日（葡萄牙语：{{lang|pt|Dia da Cultura Lusófona}}）。

2006年7月在[[比绍|比绍]]的最高级会谈中，[[赤道几内亚|赤道几内亚]]和[[毛里求斯|毛里求斯]]获得观察员身份<ref>CPLP Associate Observers in [http://www.cplp.org/quemsomos_obsA.asp CPLP]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，同时有17个国际联盟及组织获得[[顾问|顾问]]身份。

葡语国家共同体成立时，赤道几内亚申请观察员资格。赤道几内亚作为葡萄牙的前殖民地直到18世纪，其[[领土|领土]]上有以葡萄牙语演化的[[克里奥尔语|克里奥尔语]]使用者，文化也与圣多美和普林西比及葡萄牙有关联。同时，赤道几内亚最近与非洲的葡语国家及巴西在教育层面上有合作。在2004年7月在圣多美和普林西比的最高级会谈中，共同体国家成员同意改变观察员资格，接纳观察员身份。毛里求斯，这个除了葡萄牙人航行至此外对多数[[欧洲人|欧洲人]]来说都很陌生的国家，也于2006年获得观察员身份。[[摩洛哥|摩洛哥]]也对观察员资格表示了兴趣。

== 葡语国家共同体的重要性 ==
葡语国家有超过2.23亿的人口遍布全球，有相似的文化及共同的历史。葡语国家共同体的国家总面积超过1074.2万平方公里，比加拿大还大。

自成立起，共同体就帮助圣多美和普林西比、几内亚比绍解决[[政变|政变]]引起的问题，从而帮助两个国家进行[[经济|经济]]重建（如圣多美）及[[民主|民主]]重建（如几内亚比绍）。

共同体的领导人们相信安哥拉和莫桑比克以及东帝汶的独立有助于共同体的长远发展及[[多边合作|多边合作]]的加强。

在葡语非洲及东帝汶，有许多孩子不能上学，这些地区的教育机构向葡萄牙及巴西寻求帮助加强人们讲流利葡语的[[教育|教育]]（如在许多城市及乡镇建立[[贾梅士学院|贾梅士学院]]语言中心及分支），葡语正在成为南部非洲的主要语言之一，如[[纳米比亚|纳米比亚]]和[[南非|南非]]都教授葡语。

在讲葡语的发展中国家中，葡语作为政府及经济用语意味着讲葡语的非洲国家人民可以与世界上的其他地方的人工作与交流，特别是经济实力更强的葡萄牙及巴西。许多非洲讲葡语的国家的领导人担心语言标准未达流利要求，并将葡语列为[[学校|学校]][[必修课|必修课]]以使年轻非洲人可以说一门世界性的语言并在今后的生活中获益。

== 葡语国家共同体体系 ==
共同体的执行秘书处负责设计并执行项目及计划。它位于葡萄牙[[里斯本|里斯本]]。执行秘书有两年的任期，只能被重选一次。

共同体的方针及重点由两年一次的国家首脑会议确定，共同体的行动计划由每年的外长理事会议批准。

此外还有常设指导委员会每月召开会议来跟进特别的计划及项目。

共同体由八个成员国负担经费。

在东帝汶成为成员国后，现在的葡语国家共同体旗帜上有八个花瓣，而不是七个。

== 主要计划 ==
* 葡语国家共同体[[艾滋病|艾滋病]]计划——旨在帮助五个非洲成员国
* 企业家技能发展中心——在安哥拉罗安达建立
* [[公共行政|公共行政]]发展中心——在莫桑比克马普托建立
* 东帝汶官方语言中心
* 防治[[疟疾|疟疾]]会议——在圣多美和普林西比举行
* 葡萄牙语语言调查
* [[数字化|数字化]]学校和大学
* 几内亚比绍[[选举|选举]]（东帝汶[[总统|总统]]，[[诺贝尔奖|诺贝尔奖]]得主[[若泽·拉莫斯·奥尔塔|若泽·拉莫斯·奥尔塔]]（{{lang|pt|José Ramos-Horta}}）是葡语国家共同体在选举过程中的代表）
* 支持几内亚比绍重建[[公共机构|公共机构]]的紧急计划
* 重建东帝汶司法及公共行政
* 对抗[[贫穷|贫穷]]和[[饥饿|饥饿]]
* “Felino”演习——葡语国家共同体军事力量年度联合[[演习|演习]]
