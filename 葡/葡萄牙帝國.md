{{葡萄牙历史}}
'''葡萄牙帝國'''（{{lang-pt|'''Império Português'''}}）為[[歷史|歷史]]上第一個全球性[[殖民地|殖民]][[帝國|帝國]]，以及[[歐洲|歐洲]]最早建立和最長久的殖民帝國（1415年8月21日─1999年12月19日）。

[[葡萄牙|葡萄牙]]在15世紀進行的[[大西洋|大西洋]]探險和1415年對[[休达|休达]]的征服標誌著葡萄牙成為海上強權的開始。其後，葡萄牙開始大力發展航海學校，以及加強[[地图学|地图学]]的知識。在一個世紀後，葡萄牙終於成為歐洲首個打通往印度航線的國家。葡萄牙航海家如[[迪亞士|迪亞士]]，[[瓦斯科·達·伽馬|瓦斯科·達·伽馬]]，以及为[[西班牙|西班牙]]国王服务的[[斐迪南·麥哲倫|斐迪南·麥哲倫]]創下了不朽的航海歷史。

葡萄牙帝國國力在16世紀達至頂峰。但其後因為一度被鄰國[[西班牙帝國|西班牙]][[联合统治|联合统治]]，以及其他新興殖民帝國如[[英國|英國]]、[[荷蘭|荷蘭]]和[[法國|法國]]的競爭下，葡萄牙帝國開始走向衰落。18世紀後，葡萄牙便集中其在[[巴西|巴西]]及[[非洲|非洲]]殖民地的統治。而巴西殖民者為葡萄牙帶來的[[黃金|黃金]]重新振興了這個帝國。但無奈的是，1755年災難性的[[里斯本大地震|大地震]]卻嚴重打擊了葡萄牙首都[[里斯本|里斯本]]的國際地位，加上1822年巴西的獨立和1890年英國打擊了其在非洲的擴張企圖，都使得這個殖民帝國趨向衰亡。

[[第二次世界大戰|第二次世界大戰]]後，葡萄牙曾經嘗試抵抗殖民地發起的[[非殖民地化|非殖民地化]]浪潮，並因此爆發了[[葡萄牙殖民地戰爭|葡萄牙殖民地戰爭]]（1961年-1974年）。[[印度|印度]]亦在1961年出兵并收复了葡萄牙位於印度的殖民地[[果阿|果阿]]和[[達曼-第烏|達曼-第烏]]兩地，這使得葡萄牙在[[亞洲|亞洲]]只剩下[[澳門|澳門]]及[[东帝汶|东帝汶]]兩個殖民地。而經歷1974年[[康乃馨革命|康乃馨革命]]後，葡萄牙政府放棄其殖民地，直至葡萄牙政府在1999年12月20日，[[澳門回歸|澳門主權移交中國]]後，葡萄牙帝國才正式宣告結束。

== 建立（1415年─1580年） ==
[[File:Lisbon_monument.jpg|thumb]]
在1415年，葡萄牙士兵佔領了北非港口城市休達，並在1418年擊敗了企圖奪回休達的[[摩尔人|摩尔人]]。而一年後，葡萄牙王子[[航海家恩里克|航海家恩里克]]的兩名船長在風暴下駛進[[馬德拉|馬德拉]]，而其他葡萄牙航海家更在1427年發現了[[亚速尔群岛|亚速尔群岛]]。葡萄牙帝國就因而開始發跡。葡萄牙的船隊相繼於1434年和1445年到達非洲的[[博哈多爾角|博哈多爾角]]、[[塞內加爾|塞內加爾]]和[[佛得角|佛得角]]。1446年，葡萄牙航海家[[安东尼奥·费尔南德斯|安東尼奥·費爾南德斯]]更發現了[[塞拉利昂|塞拉利昂]]，這些地理上的大發現都為葡萄牙帝國的崛起提供了有利條件。

其時，葡萄牙開始對[[亞速爾群島|亞速爾群島]]和馬德拉進行殖民政策，以攫取其在蔗糖和酒類生產上的利益，尤其是來自[[几内亚|几內亚]]的黃金更大大刺激了葡萄牙的商業能力。從科學及宗教的角度出發，遠洋航行無疑是有利可圖的。在[[阿方索五世_(葡萄牙)|阿方索五世]](1443年-1481年)領導下，葡萄牙的海洋探險已遠至[[几內亚湾|几內亚湾]]一帶。而他更在1458年和1471年分別從摩爾人手上佔據了[[摩洛哥|摩洛哥]]一部份和[[丹吉爾|丹吉爾]]等地。而其繼承人約翰二世亦在[[西非|西非]][[加纳|加纳]]建立保護商貿用的要塞，其後這個要塞更在1637年起成為葡萄牙在西非的總部。此外，葡萄牙在1482年發現了[[剛果|剛果]]，並在1488年派遣[[迪亞士|迪亞士]]繞過[[好望角|好望角]]。

最後，葡萄牙籍航海家[[瓦斯科·達伽馬|瓦斯科·達伽馬]]更率先到達[[印度|印度]]，開闢了第一條由歐洲通往印度的航線。通過這些發現，葡萄牙提高了本國的經濟，並為它帶來龐大的貿易利益。這個征服東印度香料群島的行動，由「葡萄牙戰神」[[阿方索·德·阿爾布克爾克|阿方索·德·阿爾布克爾克]]一手完成，確定16世紀葡萄牙大致壟斷歐洲香料市場的結果（雖然主要是壟斷便宜香料的市場，昂貴香料仍在東南亞以走私方式，通過土耳其─威尼斯輸入歐洲），讓葡萄牙暴富崛起，一度躋升為歐洲的一流[[大國|大國]]。

當時，人們都知道在[[印度洋|印度洋]]向東出發就可以到達印度，但[[哥倫布|哥倫布]]認為向西航行也能到達印度，但這個想法卻被當時的君主[[若昂二世|約翰二世]]堅決地反對了。其後，[[卡斯蒂利亚|卡斯蒂利亚]]的[[伊莎貝拉女王|伊莎貝拉女王]]卻大力支持哥倫布的想法，並遣之實行。結果，哥倫布成功地發現[[美洲|美洲]][[西印度群島|西印度群島]]，此地其後更成為西班牙在美洲的殖民地。

而在[[東非|東非]]，由[[穆斯林|穆斯林]]統治的一帶東非海岸如[[莫桑比克|莫桑比克]]、[[蒙巴萨|蒙巴萨]]等地也落入葡萄牙人手中。其後在1490年，葡萄牙航海家已經到達東非的[[埃塞俄比亚|埃塞俄比亚]]海岸。此外，葡萄牙人的勢力也伸展至阿拉伯海一帶，他們在1506年侵吞了[[索科特拉岛|索科特拉岛]]，同年更到達印度洋上的[[錫蘭|錫蘭]]。而葡萄牙人更一直在印度洋上進行探險，如1506年發現的[[馬達加斯加|馬達加斯加]]和1507年發現的[[毛里求斯|毛里求斯]]等，葡萄牙亦在1509年發生的[[第乌战役_(1509年)|第烏戰役]]擊敗了[[奥斯曼帝国|奧斯曼土耳其]]的[[苏丹_(称谓)|蘇丹]]以及[[威尼斯共和国|威尼斯共和国]]，成為印度洋上的霸主。至於葡萄牙人也大力開始在美洲的擴張，如在1500年登陸[[巴西|巴西]]的[[塞古羅港|塞古羅港]]，並以之作為[[巴西紅木|巴西紅木]]的貿易站。

而葡萄牙和西班牙兩國更在1494年簽訂《[[托尔德西里亚斯条约|托尔德西里亚斯条约]]》。該條約列明西葡兩國共同壟斷歐洲以外的世界，並以[[維德角|維德角]]以西300[[里格|里格]](大約1770[[公里|公里]])作為兩國分界線，以東為葡萄牙勢力範圍，而以西即為西班牙所擁有。自此，葡萄牙殖民地帝國如日中天。

== 哈布斯堡（1580－1640年） ==
在1580至1640年這段期間，掌有西班牙王冠的哈布斯堡王朝亦取得葡萄牙王冠，從而葡萄牙王國與其海外殖民地皆由[[哈布斯堡王朝|哈布斯堡王朝]]掌理，兩國間形成[[君合國|君合國]]關係。西葡兩國的聯合艦隊，攻佔了抵抗哈布斯堡王朝統治的葡萄牙領地亞速爾群島，並清除從亞速爾群島來進行貿易的[[法國|法國]]人。

此後，掌有葡萄牙海外殖民地的哈布斯堡王朝繼續向海外擴張，並在1616年發動了一次軍事擴張，驅逐了法國在南美洲的殖民者。此外，更在1625年率領艦隊佔據了由荷蘭人控制的[[薩爾瓦多|薩爾瓦多]]城市。但好景不常，自1627年卡斯蒂里亞經歷經濟大崩潰後，荷蘭人便乘機組織海軍，並破壞了西班牙的海上貿易。自此，英荷兩國便開始頻繁攻擊葡萄牙的海岸線，以掠奪哈布斯堡王朝的財富。哈布斯堡王朝人完全不能應付海軍的威脅，因此爆發了[[荷西戰爭|荷西戰爭]]。

1638年至1640年間，荷蘭逐漸控制了巴西東北部的地區，並以[[累西腓|累西腓]]為首府。1640年，[[葡萄牙王政復古戰爭|葡萄牙王國脫離哈布斯堡王朝]]，由本地貴族創建[[布拉干薩王朝|布拉干薩王朝]]，1648年西班牙在對荷蘭戰爭的勝出遏止了荷蘭在巴西持續擴張的勢頭，其後，荷蘭當局交還其巴西的佔領地予葡萄牙。雖然荷蘭在巴西的殖民政策失敗了，但其後荷蘭卻搶奪了[[葡屬錫蘭|葡屬錫蘭]]、[[葡屬好望角|葡屬好望角]]和[[葡屬東印度群島|葡屬東印度群島]]一帶的土地，更壟斷了在[[日本|日本]][[长崎市|长崎市]]的貿易。自此，葡萄牙在[[遠東|遠東]]的勢力減少至澳門和東帝汶兩地。

== 巴西帝國（1640－1822年） ==
[[File:Flag_Portugal_(1707).svg|thumb]]

1661年，葡萄牙給予英國[[孟買|孟買]]和[[丹吉爾|丹吉爾]]兩地作為兩國聯姻的嫁妝。而其後一百年，英國乘印度[[莫臥儿帝国|莫臥儿帝国]]的崩潰而逐漸控制整個印度及其貿易。但葡萄牙手中仍然掌握西印度的果阿等地。[[1755年里斯本大地震|1755年里斯本大地震]]動搖了葡萄牙的殖民野心。因此，葡萄牙帝國在18世紀開始走下坡。

儘管巴西在起初並不太重要，但它後來卻成為了葡萄牙殖民地的中心，尤其是它積存了來自葡萄牙大量的黃金、[[寶石|寶石]]、[[甘蔗|甘蔗]]、[[咖啡|咖啡]]和[[穀物|穀物]]等。葡萄牙更在此進行了黑人奴隸的貿易，因此巴西的人口開始大增，並開始走上發展之路。

而[[葡屬巴西|葡屬巴西]]的獨立運動也在此時開始萌生。在1789年曾經發生過一次起義，但最後失敗，領導者亦被絞死。而1808年，[[法国君主列表|法國皇帝]][[拿破仑一世|拿破崙]]聲稱葡萄牙暗中援助英國，因此對葡萄牙發動入侵。而葡萄牙攝政王若奧親王為免受到波及，便把王室遷往巴西。而若奧六世在1812年趕回葡萄牙以解決國內的問題，並留下兒子[[佩德罗一世_(巴西)|佩德羅]]作巴西攝政王。其後在人民的強烈呼聲下，佩德羅便領導人們進行巴西的獨立。最後，他在1822年正式加冕為[[巴西君主列表|巴西皇帝]]，建立[[巴西帝國|巴西帝國]]，巴西也在此時獨立於葡萄牙。

== 非洲帝國（1822－1945年） ==
在[[殖民主義|殖民主義]]高漲的19世紀中，葡萄牙已經失去了其在[[美洲|美洲]]和[[亞洲|亞洲]]的領土。為了挽回劣勢，葡萄牙便把殖民政策投放在非洲身上，並把[[葡屬維德角|葡屬佛得角]]、[[葡屬圣多美普林西比|葡屬圣多美普林西比]]、[[葡屬幾內亞|葡屬幾內亞比索]]、[[葡屬西非|葡屬安哥拉]]和[[葡屬東非|葡屬莫三比克]]置於控制之下。

== 帝国瓦解和终结 ==
[[第二次世界大戰|第二次世界大戰]]後，很多[[歐洲|歐洲]]國家也紛紛自願及被迫放棄其殖民地。葡萄牙卻選擇拒絕放棄其殖民地，因此它仍然維持著龐大的殖民帝國。1961年，[[印度收復果阿|印度出兵果阿]]，葡萄牙丧失在南亚次大陆的所有殖民地。一直到1974年葡萄牙國內發生[[康乃馨革命|康乃馨革命]]，葡萄牙政府才放棄海外殖民地，葡萄牙所屬的非洲殖民地1975年獨立，成為[[莫三比克|莫三比克]]、[[安哥拉|安哥拉]]、[[幾內亞比紹|幾內亞比紹]]、[[聖多美與普林西比|聖多美與普林西比]]等國；而在亞洲的兩處殖民地東帝汶與澳門中，葡萄牙放棄[[葡屬帝汶|葡屬帝汶]]，使得[[葡屬澳門|葡屬澳門]]成為葡萄牙唯一也是最後的海外殖民地，葡萄牙的殖民帝國已名存实亡。

1975年11月28日，[[东帝汶|东帝汶]]獨立，但在9天之後馬上就[[东帝汶印度尼西亚占领时期|被印尼入侵并占领]]，1976年7月成为印尼的一个省——[[东帝汶|东帝汶]]（Timor Timur）。接下来的20年-{zh-hans:里;zh-hant:裡;}-，不断发生抗争运动，并造成了10－25万人的死亡。1999年8月30日，在[[联合国|联合国]]监督下举行了[[全民公投|全民公投]]，绝大多数的东帝汶人投票同意从印尼独立。在公投到1999年9月下旬多国维和部队进驻期间，由印尼军方组建和支持的反对独立的东帝汶民兵展开了[[1999年东帝汶危机|大规模的焦土运动]]进行报复。民兵杀害了大约1,400名东帝汶人，并强行驱离了30万人到[[西帝汶|西帝汶]]成为[[难民|难民]]。国家大部分的基础设施（包括住宅、灌溉系统、供水系统、学校、电网等）几乎100％被摧毁。1999年9月20日，以[[澳大利亚|澳大利亚]]为首的国际部队进入以后，才结束了这些暴力行动。 經[[聯合國|聯合國]]託管约兩年半后，于2002年5月20日零时正式恢復獨立，國名為東帝汶民主共和國，成为国际公认的一个独立国家。東帝汶自2002年5月20日正式獨立後僅僅四個月，於同年9月27日加入聯合國，成為該國際組織第191個會員國。

而葡萄牙的最後一個殖民地──[[澳門|澳門]]則在1999年12月20日正式[[澳門回歸|主權移交中華人民共和國]]。至此，存在时间长达五个世纪的葡萄牙帝国自此走入历史，也标志着[[殖民主义|殖民主义]]在亚洲的终结。

== 其他 ==
16世紀時教皇[[亞歷山大六世|亞歷山大六世]]曾做仲裁：以经度46度以东为[[葡萄牙帝國|葡萄牙帝國]]所有，即[[非洲大陆|非洲大陆]]、[[亚洲|亚洲]]和今[[巴西|巴西]]；而南美洲西部以西则为[[西班牙帝国|西班牙帝国]]所有。
