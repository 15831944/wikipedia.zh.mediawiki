'''葡佔屯門'''（{{lang-pt|Tamão}}）是指[[葡萄牙帝國|葡萄牙]]在1514–1521年間在[[東莞縣|東莞縣]]（今[[香港|香港]]）[[屯門島|屯門島]]、[[屯門海澳|屯門海澳]]及[[葵涌海澳|葵涌海澳]]及附近地區的[[軍事佔領|軍事佔領]]地。<ref>蕭國健：《香港古代史（修訂版）》，香港：中華書局，2006年，第47頁。</ref><ref>蕭國健、林天蔚：《香港前代史論集》，台北：商務印書局，1985年。</ref>

== 背景 ==
15世紀[[地理大發現|地理大發現]]時期，[[葡萄牙人|葡萄牙人]]從[[歐洲|歐洲]]往東方發展貿易，在[[非洲|非洲]][[安哥拉|安哥拉]]、[[印度|印度]][[果亞|果亞]]、[[東南亞|東南亞]][[馬六甲|馬六甲]]等地先後建立起軍事及貿易據點。一般認為1493年[[葡萄牙帝國|葡萄牙和阿爾加爾维王國]]（[[阿維斯王朝|阿維斯王朝]]）軍隊抵達[[明朝|大明帝國]][[東莞|東莞]]守禦千所的領地（今[[深圳|深圳]]及[[香港|香港]]沿海），與東莞守禦千所千戶[[袁光|袁光]]於[[岑子澳|岑子澳]]激戰，因戰敗而撤退。<ref>深圳博物館編、張一兵著:《深圳古代簡史》，文物出版社，1997年,第148-155頁。</ref>這也是現代西方社會首度與中國接觸。

== 建立 ==
1514年，[[歐維士|歐維士]]（Jorge Álvares）率領一支葡萄牙船隊到達[[珠江三角洲|珠江三角洲]]，發出登陸貿易的要求，明朝政府不予接納，因而只能在水面上與華商貿易。同年，{{link-ms|塞克拉|Diego Lopez de Sequiera}}（Diogo Lopes de Sequeira）再率領葡萄牙船隊佔領「屯門海澳」，並在其上刻立佔領石碑，修築工事，設刑場、製火器。<ref>深圳博物館編、張一兵著:《深圳古代簡史》，文物出版社，1997年,第148-155頁。</ref>

== 擴張 ==
1515－1517年間，葡萄牙人因貿易問題與明帝國起糾紛。1517年6月17日，葡萄牙使者[[皮莱资|皮莱资]]（Tomé Perez）領[[費爾南·佩雷茲·德·安德拉德|費爾南·佩雷茲·德·安德拉德]]（Fernão Perez de Andrade）等人所統領的八船船隊前往[[珠江|珠江]]一帶，並於8月15日抵達「屯門島」。「屯門島」應是指今[[香港|香港]]的[[大嶼山|大嶼山]]，<ref>[http://www.historychina.net/qsyj/ztyj/gat/2007-06-11/27638.shtml 施存龙：<葡人入居澳门前侵占我国“南头”考实>]，載《中国边疆史地研究：澳门专号》1999年第2期。「南頭島」和「大濠島」是大嶼山別稱。參見呂烈：《大嶼山》，香港：三聯書店，第14頁。</ref>也有人認為是指[[内伶仃島|内伶仃島]]或[[大鏟島|大鏟島]]。<ref>深圳說以張一兵為首，以駁斥主流的大嶼山說。其論據為大嶼山島太大，不便葡萄牙人管理。然此說已被施存龙加以駁斥，見前注。深圳博物館編、張一兵著:《深圳古代簡史》，文物出版社，1997年,第148-155頁。</ref>

1518年，費爾南之弟{{link-pt|西眇|Simão_de_Andrade}}（Simão D. Andrade）率領葡船一艘、帆船三艘抵達屯門，並在鄰近島嶼建設堡壘壕障，設置刑場，並肆意掠奪，拒絕明朝課稅。<ref>蕭國健：《香港古代史（修訂版）》，香港：中華書局，2006年，第47、184頁。</ref>葡萄牙人企圖進一步攻佔[[南山半島|南山半島]](今[[深圳市|深圳市]][[南山區|南山區]][[蛇口街道|蛇口]])，但由於明帝國駐軍太多而失敗，撒退至「屯門島」安營扎寨，作下一步軍事準備。此後，葡人又在「屯門海澳」及「葵涌海澳」（今香港[[青衣島|青衣島]]、[[葵涌|葵涌]]一帶）探查據點，製火器，立石碑，並與當地居民衝突。<ref>深圳博物館編、張一兵著:《深圳古代簡史》，文物出版社，1997年,第148-155頁。</ref>

== 葡軍戰敗及撤退 ==
1521年，明帝國[[嘉靖帝|嘉靖帝]]派遣[[廣東海道副使|廣東海道副使]][[汪鋐|汪{{僻字]][[屯門海戰|進攻葡佔屯門]]。葡佔屯門由歐維士帶領，得到[[卡爾佛|卡爾佛]]（Diogo Calvo）、<!----{{link-pt|科埃略|Duarte_Coelho}}--->[[科埃略|科埃略]]（Duarte Coello）及[[雷戈|雷戈]]（Ambrocio do Rego）的支援，予以還擊，展開為時近一個月的葡佔屯門攻防戰。

戰爭初期，明軍因葡佔屯門軍火砲猛烈而不敵。汪{{僻字|鋐|左金右宏}}轉而策動[[間諜|間諜]][[間諜戰|戰]]，向葡萄牙船上的水手查問葡軍[[蜈蚣船|蜈蚣船]]和[[佛朗機炮|佛朗機炮]]的構造，並策動下一輪攻勢。

9月，明軍準備了很多滿載柴草及油料的小船，待一天刮強南風，汪鋐率軍士約4000人，乘50多艘船隻再次攻打葡軍船隊。明軍先將填有膏油草料的船隻點燃，讓火船高速駛向葡軍艦隊，由於葡軍艦隊巨大而緩慢，無法躲開火船進攻，很快燃燒了起來，葡軍大亂。汪汪{{僻字|鋐|左金右宏}}又趁機派人潛入水中，將未起火的葡軍船隻鑿漏，不少葡軍將士只得跳海逃命。其後，明軍登上敵船與葡軍短兵交接。葡軍不敵，率餘下三艘大船，在9月7日乘夜黑之際開往附近島嶼匿藏。乘第二天清晨風向逆轉之機，葡軍艦隊借北風避過明軍追擊，往[[滿剌加|滿剌加]]撤退。<ref>《汪公遺愛祠記》：「於正德辛巳出師，嘉靖壬午凱旋」</ref>

1522年4月，葡軍再派出由六艘軍艦組成的艦隊企圖重新佔領[[屯門|屯門]]，8月兩軍在大奚山（即今大嶼山）[[茜草灣_(大嶼山)|茜草灣]]附近對峙，是為第四次明葡戰爭（[[茜草灣之戰|茜草灣之戰]]）。此戰從茜草灣海面一直打到了[[哨州|哨州]]<ref>哨州位置約在大嶼山以西，[[萬山群島|萬山群島]]以東</ref>海面，最後明軍獲勝。至此，明帝國接管了「屯門島」、「屯門海澳」及「葵涌海澳」等葡佔屯門全境。葡萄牙人自此再也未能重奪在香港的失地。<ref>《殊域周咨錄》：「有東莞縣白沙巡檢何儒，前因委抽分，曾到佛朗機船，見有中國人楊三、戴明等，年久住在彼國，備知造船、鑄銃及製火藥之法。鋐令何儒密遣人到彼，以賣酒米為由，潛與楊三等通話，諭令向化，重加賞齎，彼遂樂從。約定其夜，何儒密駕小船，接引到岸，研審是實，遂令如式製造。鋐舉兵驅逐，亦用此銃取捷。奪獲伊銃大小二十餘管。」</ref><ref>《重建汪公生祠記》：「公（汪鋐）以儒發身戎務，若非素習，壹旦挺身行陣，摧數百年未睹之強寇，豈偶然者哉！公生平忠義自許，剔歷中外，始終壹節。」</ref><ref>蕭國健、林天蔚：《香港前代史論集》，台北：商務印書局，1985年。</ref><ref>呂烈：《大嶼山》，香港：三聯書店，第13頁。</ref>

== 後續 ==
葡萄牙失去香港一帶的屯門島、屯門海澳及葵涌海澳後，輾轉逃至[[上川島|上川島]]、{{link-zh-yue|浪白滘|浪白竈}}，及後再轉而向[[澳門|澳門]]埋首。葡人先是轉往上川島一帶落腳<ref name=rthk5000>香港電台廣播節目《[[中華五千年|中華五千年]]第四百九十八集·澳門開埠》</ref>，至1542年，葡人又佔領浪白滘。1545年，明帝國封鎖上川島。

1553年，葡萄牙人賄賂海道副使[[汪柏|汪柏]]，終取得停泊船隻於[[澳門|澳門]]的權利和澳門居留權，獲得更多在中國的利益，並繼續拓展葡萄牙在[[遠東|遠東]]的版圖<ref>深圳博物館編、張一兵著:《深圳古代簡史》，文物出版社，1997年,第148-155頁。</ref><ref>Jonathan Porter. Macau, the Imaginary City: Culture and Society, 1557 to the Present. [S.l.]: Westview Press, 1996. ISBN 0813328365</ref><ref>Richard L. Edmonds. China and Europe Since 1978: A European Perspective. [S.l.]: Cambridge University Press, 2002. ISBN 0521524032</ref>，而澳門亦落入葡萄牙手中成為殖民地<ref name=rthk5000 /><ref>{{cite book|author=爾東|editor=周海燕|title=《香港歷史之謎》|date=2007年4月|publisher=明報出版社有限公司|location=香港|isbn=978-962-973-634-7|pages=頁12}}</ref>。

== 遺址 ==
今香港大嶼山[[大澳|大澳]]仍保存著葡佔屯門的據點番鬼塘。「番鬼」在粵語中指「西洋人」。<ref>[https://www.youtube.com/watch?v=gAlXQwQSpkE 蕭國健：《歷史考察工作坊：香港漁民與大澳棚屋》（視聽資料），聯合出版集團，2013年，於2015年7月7日查看。]</ref>
