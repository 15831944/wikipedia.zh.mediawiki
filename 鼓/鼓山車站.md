'''鼓山車站'''位於[[高雄市|高雄市]][[鼓山區|鼓山區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]之[[鐵路車站|鐵路貨運車站]]，此站是1929年配合淺野水泥株式會社（{{lang|ja|浅野セメント株式会社}}）運輸其於[[壽山|壽山]]所採原料所建，當時稱為「'''田町驛'''」；配合[[高雄市區鐵路地下化計畫|鐵路地下化]]工程暫時裁離，地下化站體已於2018年10月14日通車。

== 簡介 ==
縱貫線自本站以南原為西南向至打狗驛（1920年改稱高雄）；1940年代配合高雄驛遷建[[三民區|三民區]]今址，而自此站南邊另建縱貫新線－以東向大拐彎連結[[屏東線|屏東線]][[三塊厝車站|三塊厝驛]]（並未設有月臺），再入今[[高雄車站|高雄車站]]。原高雄驛同時改稱[[高雄港車站|高雄港]]以為區別。

由於分歧新舊縱貫線之故，且往高雄車站方向改為三線，高雄市區鐵路地下化工程開始前，鼓山車站股道及場站配置頗為複雜。

未來[[環狀輕軌|環狀輕軌]]第二階段會在台鐵出入口旁興建月台，定位為「特色站」。

== 車站構造 ==
===裁撤前===
* 站房一棟。
* 與[[花蓮港車站|花蓮港]]等貨運專用車站類似，站名燈箱設在軌道側。
* 無月臺，站房緊鄰軌道（但目前站內軌道已拆除），為花草植栽所隔。
===復設後===
* 站體長288公尺，寬24.4公尺，為一地下二層二股道岸壁式月臺車站，採用[[明挖覆蓋|明挖覆蓋]]順打工法施工之{{Link-en|車站箱體|station box|箱型結構}}。

== 利用狀況 ==
*裁撤前為三等站，只辦理貨運。
*配合[[高雄市區鐵路地下化計畫|高雄市區鐵路地下化計畫]]，於2009年5月13日暫時裁撤。工程完工後，另行啟用新建車站並辦理客運。

== 歷史 ==
* 1929年10月1日：由淺野水泥株式會社（{{lang|ja|浅野セメント株式会社}}）所建「田町驛」。
* 1950年4月15日：改稱今名。
* 1993年：第一代站房停用便拆除，第二代站房啟用。
* 2009年5月13日：配合[[高雄市區鐵路地下化計畫|高雄市區鐵路地下化計畫]]暫時裁撤（第二代站房停用）。
*2018年10月14日：隨著[[高雄鐵路地下化|高雄鐵路地下化]]工程完工，啟用新建車站（第三代站房啟用）。
