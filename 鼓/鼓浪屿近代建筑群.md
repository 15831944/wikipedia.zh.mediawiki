'''鼓浪屿近代建筑群'''位于中国[[福建省|福建省]][[厦门市|厦门市]][[思明区|思明区]][[鼓浪屿|鼓浪屿]]。2006年，美国领事馆旧址等十处十三座建筑被联合列为[[第六批全国重点文物保护单位|第六批全国重点文物保护单位]]。<ref>{{Cite book |author=国家文物局 编 |title=全国重点文物保护单位（第六批）·第Ⅴ卷 |publisher=文物出版社 |year=2008 |isbn=978-7-5010-2446-9 |pages=409-413 }}</ref>

2013年，厦门海关税务司公馆旧址、海关理船厅公署旧址、海关验货员公寓旧址、会审公堂旧址、[[故宫鼓浪屿外国文物馆|救世医院旧址]]、博爱医院旧址、毓德女学堂旧址、“万国俱乐部”旧址、海天堂构、黄荣远堂10处共17幢建筑在[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]公布时被列为“鼓浪屿近代建筑群”的扩展合并项目，归入[[第六批全国重点文物保护单位|第六批全国重点文物保护单位]]“鼓浪屿近代建筑群”内<ref>{{Cite web |url = http://www.taihainet.com/news/xmnews/szjj/2013-05-28/1075338_2.html |title =鼓浪屿新增17幢"国保"建筑 均为申遗核心要素点  |author =  |date =  2013-05-28|publisher = 台海网 |language =  |accessdate =  |quote =  }}</ref>。

2017年7月8日，在波兰克拉科夫举行的世界遗产大会上，鼓浪屿获准列入[[世界文化遗产|世界文化遗产]]名录<ref>{{Cite web |url = http://news.sina.com.cn/c/nd/2017-07-08/doc-ifyhwehx5383584.shtml |title =鼓浪屿申遗成功 我国世界遗产总数达52处  |author =  |date =  2017-07-08|publisher = 新浪网 |language =  |accessdate =  |quote =  }}</ref>。
