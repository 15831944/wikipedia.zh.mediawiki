'''果壳网'''是[[中国大陆|中国大陆]]的一家泛科技兴趣社区网站，致力于向公众倡导科技理念，提供负责任、有智趣的[[科学普及|科学普及]]类内容。果壳网在2010年由[[姬十三|姬十三]]创立<ref>{{Cite web |url=http://www.guokr.com/blog/16969/|title=征途——果壳史前史揭秘 |accessdate=2012-01-27 |publisher=果壳网 |date=2011-03-23 }}</ref>，与其之前创办的[[非盈利组织|非盈利组织]][[科学松鼠会|科学松鼠会]]在运营上完全独立。果壳网现有科学人、小组、问答、MOOC学院等板块，由专业科技团队负责编辑，网站主编为[[拇姬|拇姬]]。果壳传媒另有“果壳阅读”这一阅读品牌，负责科普类图书的编辑。

== 词源 ==
[[哈姆雷特|哈姆雷特]]曾言“即使我身处果壳之中，仍自以为是无限[[宇宙|宇宙]]之王”，[[霍金|霍金]]以此为题著书《[[果壳中的宇宙|果壳中的宇宙]]》，暗示自己身处[[轮椅|轮椅]]之上的那些时光。[[姬十三|姬十三]]是在一次洗澡中想到了这个典故，并决定用果壳来命名这个[[网站|网站]]：网络是我们的果壳，然而这里所谈论的，大至[[宇宙|宇宙]]小至[[原子|原子]]，没有什么能束缚我们的好奇<ref>{{Cite web|url=http://www.nbweekly.com/news/business/201206/30108.aspx|title=用科学玩转商业|accessdate=2012-07-20|publisher=南都周刊|date=2012-06-01|deadurl=yes|archiveurl=https://web.archive.org/web/20120918190512/http://www.nbweekly.com/news/business/201206/30108.aspx|archivedate=2012-09-18}}</ref>。

== 网站构成 ==
*科学人：由果壳编辑负责的网站频道。谣言粉碎机、死理性派、性情等十六个主题站负责发布与社会热点和公众生活相关的科技文章，用户可以依据兴趣来关注以便更精准地阅读。
*问答：果壳的社区产品。偏重于解除科技方面的困惑，也提倡以科学的视角来解读生活。
*小组：果壳的社区产品。网友之间交流讨论兴趣话题与互相分享科技新知的地方，2018年8月下线。
*[[MOOC学院|MOOC学院]]：在线课程学习社区。收录各MOOC站点的课程链接和简介（部分由网友自愿提供中文翻译），并供网友分享MOOC课程笔记或心得，以及交流和讨论相关问题。另有“观察”栏目，不定期发布和MOOC学习相关的博客文章，2018年8月下线。

== 团队 ==
*[[姬十三|姬十三]]：果壳网创始人及CEO。神经生物学博士，[[科学松鼠会|科学松鼠会]]创始人。
*[[拇姬|拇姬]]：果壳网主编。作家，原《[[新京报|新京报]]》新知周刊统筹编辑。

== 与科学松鼠会的关系 ==
科学松鼠会是民政注册[[非营利机构|非营利机构]]哈赛中心旗下的一个公益科学传播项目。与哈赛中心旗下的未来光锥（原名果壳时间）、科学支教和科学互助等项目平行<ref>{{cite web|language =zh-hans|url=http://news.ifeng.com/gundong/detail_2011_03/21/5275812_0.shtml|title=哈赛中心让科学流行起来|publisher=凤凰网资讯|date=2011-03-21|accessdate=2012-07-20 }}</ref>。果壳网与之最大的区别是果壳网为商业化媒体。
