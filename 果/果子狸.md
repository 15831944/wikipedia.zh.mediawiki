'''果子狸'''（[[學名|學名]]：''Paguma larvata''）又名'''花面狸'''、'''白鼻心'''、'''果子-{猫}-'''（{{lang-nan-tw|kué-tsí-bâ}}）<ref>[http://twblg.dict.edu.tw/holodict_new/result_detail.jsp?n_no=4213&curpage=1&sample=%E6%9E%9C%E5%AD%90&radiobutton=1&querytarget=1&limit=20&pagenum=1&rowcount=4 果子-{猫}-－臺灣閩南語常用詞辭典]</ref>，一种[[灵猫科|灵猫科]]的[[哺乳动物|哺乳动物]]。

== 特征 ==
果子狸躯体瘦长，成年个体长45-65厘米，重4.5-8千克。果子狸外形似[[家猫|家猫]]，颜面狭长，鼻吻前突，颈短面粗，眼睛大而圆，两耳较小，上层背部两侧有20根黑白相间、富有弹性的胡须，头部被毛有7块大小不同黑、白相间的斑块，故称花面狸。

果子狸四肢粗短，足具5趾，趾端有爪，非常锋利。趾掌无毛，掌面皮肉增厚，富有弹性。尾巴粗壮有力，长30—40厘米，约占体长的2／3，在跳跃、攀登时能起平衡作用。[[肛門|肛門]]附近有臭腺，遭遇敵人時會釋出異味藉此嚇跑敵人。

== 分布 ==
果子狸主要生活在[[亚洲|亚洲]]的[[热带|热带]]或[[亚热带|亚热带]]海拔200~2200公尺的山林中，在[[中国|中国大陆]]的[[华南|华南]]地区数量较多<ref name=herperstesurva>記者蔡惠萍／台北報導,"台二線 有野生動物專用道"[http://udn.com/NEWS/LIFE/LIF8/7612627.shtml],聯合報,2013.01.06 02:51 am.</ref>。

== 习性 ==
果子狸食性独特，主要吃植物，尤其喜食[[水果|水果]]，与别的[[食肉目|食肉目]]动物不同。[[动物|动物]]性食物是次要食物。[[鼠|鼠]]类是果子狸最喜食的动物性食物，也吃[[蛇|蛇]]、[[鸟|鸟]]、[[蛙|蛙]]、[[蚯蚓|蚯蚓]]、小[[昆虫|昆虫]]等。

果子狸善攀缘，属[[夜行性动物|夜行性动物]]，具有昼伏夜出的习性。人工饲养的果子狸这种习性并不明显。

== 分類 ==
果子狸有17个[[亚种|亚种]]，包括：
* 海南亚种 - ''P. l. hainana''
* 西南亚种 - ''P. l. intrudens''
* 藏南亚种 - ''P. l. lanigerus''
* 指名亚种 - ''P. l. larvata''
* 印缅亚种 - ''P. l. neglecta''
* 察隅亚种 - ''P. l. nigriceps''
* 秦巴亚种 - ''P. l. reevesi''
* 台湾亚种 - ''P. l. taivana''

== 果子狸与严重急性呼吸道综合征（SARS） ==
在[[中華人民共和國|中国]]南方，特别是在[[云贵高原|云贵高原]]及[[两广|两广]]地区，常被食用，因此医学界许多人士认为人类通过食用果子狸而感染[[严重急性呼吸道综合征|严重急性呼吸道综合征]]。广东防治SARS科技攻关组的专家们在“广东省传染性非典型肺炎（SARS）防治研究”科技成果的鉴定会上认定：果子狸是严重急性呼吸道综合征冠状病毒的主要载体。不过果子狸只是严重急性呼吸道综合征病毒的中间宿主，天然宿主是[[中华菊头蝠|中华菊头蝠]]，它亦是被传染的<ref>新聞速報/中央社,"專家：SARS罪魁不是果子狸"{{cite web |url=http://news.chinatimes.com/focus/130501/132013040100856.html |title=存档副本 |accessdate=2013-04-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20130404004348/http://news.chinatimes.com/focus/130501/132013040100856.html |archivedate=2013-04-04 }},中國時報,2013-04-01/15:39.</ref>。

== 果子狸的养殖 ==
果子狸的人工养殖大约是从1980年代开始的。在果子狸的驯养中，经常会遇见的疾病主要是[[腹泻|腹泻]]、[[食欲不振|食欲不振]]、[[便秘|便秘]]、[[病毒性肠炎|病毒性肠炎]]。

== 流行文化 ==
[[金庸|金庸]]在《[[神鵰俠侶|神鵰俠侶]]》裡寫到：“那百粵之地毒蛇作羹，老貓燉盅，斑魚似鼠，巨蝦稱龍，肥蠔炒響螺，龍蝨蒸禾蟲，烤小豬而皮脆，煨果狸則肉紅，洪七公如登天界，其樂無窮。”
