'''果敢族'''（英文：'''Kokang'''）是緬甸東北邊境的世居民族，具有[[緬甸|緬甸]]官方少數民族的地位。在緬甸，他們被认為從[[中國|中國]][[雲南省|雲南省]]而來，居住在[[果敢县|果敢县]]的中國[[漢族|漢人]]<ref>王士录，(2005年)，{{〈}}[http://scholar.ilib.cn/Abstract.aspx?A=sjmz200505010 缅甸的"果敢族":族称、来历、状况及跨国互动] {{webarchive|url=https://web.archive.org/web/20070927212858/http://scholar.ilib.cn/Abstract.aspx?A=sjmz200505010 |date=2007-09-27 }}{{〉}}，《世界民族》第2005年05期。</ref>；而在中國，有人认为他們是[[明朝|明朝]]在緬甸的[[遺民|遺民]]。在緬甸，這些果敢族人還成立了[[果敢邦|果敢邦]]（即今日的[[掸邦第一特区|掸邦第一特区]]），是緬甸境內唯一由[[漢族|漢族]]為主體成立的地區組織。<ref name="tsa" >{{cite web|url=http://news.qq.com/a/20090827/001870.htm |title=果敢族：流落缅甸的明朝汉人遗民 |accessdate= 2009年|work= |publisher=腾讯网 |language =zh-cn}}</ref>

果敢邦是由一個[[楊姓|楊姓]]的[[土司|土司]]家族所成立，他們聲稱是[[明朝|明朝]][[永曆帝|永曆帝]]的遺民，在17世紀明亡時遷移至[[雲南省|雲南省]]，然後再移居到中緬邊境的狹小土地上。雖然[[大英帝國|大英帝國]]在1885年併吞了[[上緬甸|上緬甸]]，使果敢族的主權轉移到[[英國|英國]]，楊氏家族依然得以維持在區內的權力。緬甸獨立後，上、下緬甸統一，使果敢族失去權力，而原來的統治家族亦因此而流亡海外。近年來，果敢漢人一直受到緬甸政府的壓迫，不時發生緬甸軍隊攻擊果敢族的事件。

== 历史 ==
{{main|果敢历史}}
=== 起源 ===
[[明朝|明朝]]末年，[[南明|南明]]因为[[战争|战争]]迁徙而来到[[云南|云南]]与[[缅甸|缅甸]]交界处，居住于[[科干山|科干山]]，[[果敢土司县|果敢土司县]]。<ref name=" flsa" >{{cite web |url=http://news.ifeng.com/history/zhongguogudaishi/detail_2009_08/31/329283_0.shtml |title=缅甸果敢史上多战乱 果敢族是明朝汉人遗民 |accessdate= 2010年|work= |publisher=凤凰网 |language =zh-cn}}</ref><ref name="tsa" />
=== 形成 ===
[[明朝|明朝]][[永历帝|永历帝]][[朱由榔|朱由榔]]定都[[肇庆|肇庆]]，后来迁都[[昆明|昆明]]，号称“滇都”。<ref name=" flsa" /><ref name="tsa" />

[[清朝|清朝]][[顺治|顺治]]十六年（1659年）正月初三，[[吴三桂|吴三桂]]带兵驱逐他们到[[明朝属国|明朝属国]][[缅甸|缅甸]]的边境[[瓦城|瓦城]]（即[[曼德勒|曼德勒]]近郊的古都阿瓦），缅甸国王挽留他们住在那里。<ref name=" flsa" /><ref name="tsa" />

1661年，缅京老王[[宾德勒|宾德勒]]（莽达）去世，新王[[比耶王|比耶王]]（莽白）一上台就囚禁了[[明朝|明朝]][[永历帝|永历帝]]的旧臣和下属官员，并把永历帝献给了[[吴三桂|吴三桂]]。<ref name=" flsa" /><ref name="tsa" />

[[康熙|康熙]]元年（1662年），[[永历帝|永历帝]]在[[昆明|昆明]][[金禅寺|金禅寺]]被[[吴三桂|吴三桂]]用弓弦勒死，[[南明|南明]]政权灭亡。<ref name=" flsa" /><ref name="tsa" />

[[清政府|清政府]]设立“[[木邦宣慰司|木邦宣慰司]]”管理该地区，命令一个杨姓的将领为“[[土司王|土司王]]”，实行了统而不治。<ref name=" flsa" /><ref name="tsa" />

1852年，[[大英帝国|大英帝国]][[殖民者|殖民者]]侵占[[缅甸|缅甸]]。<ref name=" flsa" /><ref name="tsa" />

1897年，英国人和[[清王朝|清王朝]]订立《[[中英续议缅甸条约|中英续议缅甸条约]]》，标志着[[大英帝国|大英帝国]]侵占了该地区，该地区并入[[英属缅甸|英属缅甸]]，该地区成为英属缅甸的领土范围，并开始种植[[罂粟|罂粟]]，并且迫使[[汉族|汉族]]改族名为“果敢族”，[[汉语|汉语]]改名为“[[果语|果语]]”，[[中文|中文]]改名为“[[果文|果文]]”。<ref name=" flsa" /><ref name="tsa" />

1947年，缅甸立国的“班弄”会议，[[土司|土司]][[杨文炳|杨文炳]]作为“果敢族”的代表出席了民族加盟[[缅甸联邦政府|缅甸联邦政府]]的签字仪式，“缅甸政府正式承认300年来不被接纳的果敢族为其境内合法少数民族”。<ref name=" flsa" /><ref name="tsa" />

二十世纪五十年代，[[中华民国|中华民国]][[国军|国军]]93师部在该地开办“[[雲南省反共抗俄大學|雲南省反共抗俄大學]]”，“[[果敢王|果敢王]]”[[彭家声|彭家声]]就是首批22个学员之一，[[罗星汉|罗星汉]]、[[坤沙|坤沙]]也是学校的著名学员。<ref name=" flsa" /><ref name="tsa" />

1959年，缅甸废除[[土司制度|土司制度]]，于是造成了该地几十年的动乱，1989年停战，并成立了“缅甸掸邦第一特区—[[果敢|果敢]]”。<ref name=" flsa" /><ref name="tsa" />

1992年，第一特區政府内讧，[[彭家声|彭家声]]離開果敢。<ref name=" flsa" /><ref name="tsa" />

1994年，[[杨茂贤|杨茂贤]]潜入[[中国大陆|中国大陆]][[云南省|云南省]]被当地警察抓获并按照法律贩毒罪处死，[[彭家声|彭家声]]重新带领缅甸政府军占据该地。<ref name=" flsa" /><ref name="tsa" />

1995年12月20日，缅政府、彭家声、果敢三方成立了“[[果敢临时政府|果敢临时政府]]”。<ref name=" flsa" /><ref name="tsa" />

2003年，果敢特区政府承诺在2003年全境成为无毒区，开始全面查毒禁毒。<ref name=" flsa" /><ref name="tsa" />

== 地域分布 ==
果敢族辖制着缅北[[萨尔温江|萨尔温江]]东岸，领土面积5200平方千米。<ref name=" flsa" /><ref name="tsa" />

== 人口 ==
族人20多万，95%以上是[[汉族|汉族]]血统，都是[[明朝|明朝]][[永历帝|永历帝]][[朱由榔|朱由榔]]的部属后代。<ref name="flsa" />

== 文化 ==
果敢族属于[[汉文化圈|汉文化圈]]：果敢族使用日常以[[汉语|汉语]][[西南官話|西南官話]]交流，间杂一點當地其他民族語言以及缅甸语；果敢族現在使用[[簡體汉字|簡體汉字]]作为正式书写语言，间杂些許缅甸文。<ref name=" flsa" /><ref name="tsa" />

“果敢”是[[掸族|掸族]]语，“果”是九的意思，而“敢”是人家的意思，合起来就是九户人家的意思。果敢族的英文名称为“Kokang”，在雅虎网站上解释为 “居住在缅甸金三角的一支来自中国的少数民族”。果敢人对自己的解释却是：“果”等于果断，“敢”等于勇敢。“果断而勇敢”就是今天果敢人对自己族名的宣示<ref>{{cite web|url=http://www.taihainet.com/news/txnews/sz/2014-12-30/1352126.html|title=“果敢族”的由来|date=2014-12-30|publisher=台海网|deadurl=yes|archiveurl=https://web.archive.org/web/20160920132247/http://www.taihainet.com/news/txnews/sz/2014-12-30/1352126.html|archivedate=2016年9月20日|df=}}</ref>。
=== 姓氏 ===
果敢族主要姓氏是杨氏、彭氏、罗氏。<ref name=" flsa" /><ref name="tsa" />

== 政治制度 ==
果敢族所在地区是[[自治区|自治区]]，享受民族[[高度自治|高度自治]]权。前任特区主席是[[彭家声|彭家声]]，现任是[[白所成|白所成]]。<ref name=" flsa" /><ref name="tsa" />
