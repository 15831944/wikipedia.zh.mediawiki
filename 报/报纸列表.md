== 美国 ==
* [[纽约时报|纽约时报]]：是美國嚴肅報刊的代表，長期以來擁有良好的公信力和權威性。由於風格古典嚴肅，它有時也被戲稱為「灰色女士」（The Gray Lady）。
* [[华盛顿邮报|华盛顿邮报]]：是美國華盛頓哥倫比亞特區最大、最老的報紙。1970年代初通過揭露水門事件和迫使理察·尼克森總統退職，《華盛頓郵報》獲得了國際威望。許多人認為它是繼《紐約時報》後美國最有聲望的報紙。由於它位於美國首都，它尤其擅長於報導美國國內政治動態。
* [[华尔街日报|华尔街日报]]：是側重金融、商業領域報導的日報，創辦於1889年。該日報是美國發行量最大的報紙。
* [[洛杉矶时报|洛杉矶时报]]：美國發行量最大的報紙之一，創刊於1881年12月4日。
* [[芝加哥论坛报|芝加哥论坛报]]：美國銷量第5高的報章。它以美國伊利諾州芝加哥為基地，是芝加哥地區和美國中西部的主要日報。
* [[今日美国|今日美国]]：是美國唯一的彩色版全國性對開日報，1982年9月15日創刊。

== 英国 ==
* [[泰晤士报|泰晤士报]]：是一張對全球政治、經濟、文化發揮巨大影響力的報紙。昔譯《太晤士報》。
* [[金融时报|金融时报]]：是培生集團旗下一份國際性大報，一般大眾對其評價頗高，公信力享譽全球。
* [[卫报|卫报]]：是英國的全國性綜合內容日報。創刊於1821年。
* [[每日镜报|每日镜报]]
* [[独立报|独立报]]：是比較性質中立的報紙。
* [[太陽報_(英國)|太阳报]]：是富商默多克擁有的新聞集團（News Corporation）旗下的一份小開型日報。是全英國銷量最高的報章。
* [[每日电讯报|每日电讯报]]：英國大開型報章，成立於1855年6月29日，是英國銷量最高的報紙之一。
