{{日本作品|想要傳達給你的愛戀|恋×シンアイ彼女}}是Us:track在2015年10月30日<ref>{{Cite web|url=http://moedigi.com/game/2015/05/ustrack-1st-project-yoyaku-start/|title=『恋×シンアイ彼女』10月発売で予約開始　―AMUSE CRAFT新ブランドUs:trackのデビュー作|publisher=moedigi|date=2015-05-29|accessdate=2018-05-14|language=ja}}</ref>發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]。

== 故事簡介 ==
國見洸太郎將來的願望是想要成為小說家，但他因為小時候初戀的痛苦而寫不出戀愛小說。升上高二的春天，洸太郎就讀的御影之丘學園與其他兩個學園合併，在充滿期待與不安的氛圍中，洸太郎與昔日的初戀姬野星奏和好朋友新堂彩音再次相遇<ref>{{Cite web|url=http://www.tgsmart.jp/article.aspx?a=5586|title=【TG SCOOP】新ブランドUs:track処女作『恋×シンアイ彼女』は少年少女たちの等身大恋愛劇がテーマの青春学園ADV!!|publisher=moedigi|date=2015-05-22|accessdate=2018-05-14|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|國見洸太郎|國見 洸太郎（くにみ こうたろう）|聲：原田友貴（成人）}}
: 作品男主角，御影之丘學園二年級生，加入的社團是文藝社，未來希望成為小說家。
; {{nihongo|姬野星奏|姫野 星奏（ひめの せな）|聲：阿部朔<ref name="Character">{{cite web|url=http://ustrack.amusecraft.jp/koikake/character.html|title=Character|publisher=Us:track|accessdate=2018年6月16日|language=ja}}</ref>}}
: 御影之丘學園二年級生，洸太郎的小學同學，也是洸太郎的初戀，在小學畢業後便搬到北海道，升上高二的春天再次回到御影之丘町。
; {{nihongo|新堂彩音|新堂 彩音（しんどう あやね）|聲：[[遙空|遙空]]<ref name="Character"/>}}
: 御影之丘學園二年級女學生，洸太郎的國中同學，國中畢業後原本就讀有服飾專門課程的櫻代學園，三校合併後就讀御影之丘學園的普通科。
: 國中時跟洸太郎交情很好，在高中與洸太郎再次碰面後卻與他針鋒相對。
; {{nihongo|小鞠結衣|小鞠 ゆい（こまり ゆい）|聲：{{lang|ja|[[遠野そよぎ|遠野そよぎ]]}}<ref name="Character"/>}}
: 御影濱學園的國三生，園藝社員，經常會來照顧御影之丘學園的花朵，跟洸太郎的妹妹菜子是好朋友。
; {{nihongo|四條凜香|四條 凜香（しじょう りんか）|聲：日傘世玲那<ref name="Character"/>}}
: 御影之丘學園三年級生，學生會長。喜歡看到他人困擾的表情。

=== 其他角色 ===
; {{nihongo|國見菜子|國見 菜子（くにみ なこ）|聲：秋野花<ref name="Character"/>}}
: 御影濱學園的國三生，洸太郎的妹妹，負責打理家中的一切事情。和結衣是好友。
; {{nihongo|貴志涼介|貴志 涼介（きし りょうすけ）|聲：古河徹人<ref name="Character"/>}}
: 御影之丘學園二年級男學生，加入的社團是電影研究社，跟洸太郎是從國中就認識的好友。
; {{nihongo|櫻田志乃|桜田 志乃（さくらだ しの）|聲：{{lang|ja|萌花ちょこ}}<ref name="Character"/>}}
: 御影之丘學園二年級女學生，運動少女，壘球社成員。
; {{nihongo|愛美|愛美さん（あゆみさん）|聲：[[川島梨乃|川島梨乃]]<ref name="Character"/>}}
: 咖啡廳「monetto」的店主，以前曾是御影之丘學園文藝社的社員。
; {{nihongo|如月奈津子|如月 奈津子（きさらぎ なつこ）|聲：{{lang|ja|椎名ヨオコ}}<ref name="Character"/>}}
: 櫻代學園的原學生會長與文藝社社長。拒絕與御影之丘的文藝社合併。

== 評價 ==
《想要傳達給你的愛戀》在Getchu.com舉辦的美少女遊戲大賞2015中獲得綜合部門第19名<ref>{{Cite web|publisher=Getchu.com|url=http://www.getchu.com/pc/2015_g_ranking/|title=美少女ゲーム大賞2015 －総合部門投票・結果発表－|accessdate=2018-05-14|language=ja}}</ref>、音樂部門第7名<ref>{{Cite web|publisher=Getchu.com|url=http://www.getchu.com/pc/2015_g_ranking/music.html|title=美少女ゲーム大賞2015 －ミュージック部門投票・結果発表－|accessdate=2018-05-14|language=ja}}</ref>。於「[[萌系遊戲大賞|萌系遊戲大賞]]」2015年10月的月間賞獲得第5名<ref>{{cite web|url=http://moe-gameaward.com/ranking/index.html?year_id=2015&month_id=10|title=2015年10月発売タイトル月間賞投票|work=萌系遊戲大賞|accessdate=2018-05-14|language=ja}}</ref>，並於年間排名獲得第29名<ref>{{cite web|url=http://moe-gameaward.com/annual/2015.html|title=萌えゲーアワード2015年間ランキング確定|work=萌系遊戲大賞|accessdate=2018-05-14|language=ja}}</ref>。
