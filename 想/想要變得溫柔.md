《'''想要變得溫柔'''》（{{lang-ja|やさしくなりたい}}），是[[日本|日本]]男歌手[[齊藤和義|齊藤和義]]（{{lang|ja|斉藤和義}}）的第39張[[單曲|單曲]]，於2011年11月2日由[[SPEEDSTAR_RECORDS|SPEEDSTAR RECORDS]]發行。同名A面曲《想要變得溫柔》是2011年收視率最高的日本電視劇《[[家政婦女王|家政婦女王]]》的主題曲。

== 發行及宣傳 ==
此作品與上一張單曲《[[一直都喜歡|一直都喜歡]]》（{{lang|ja|ずっと好きだった}}）發行相距約1年7個月，為齊藤和義於2011年發行的第一張單曲。

齊藤和義曾於《[[MUSIC_STATION|MUSIC STATION]]》（同年10月21日）<ref>{{cite web|url=http://natalie.mu/music/news/58250 |language=ja |title=<span lang="ja">MステにPerfume、斉藤和義、Superfly、テゴマス、Fairies</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2011-10-18 |accessdate=2013-02-11}}</ref>、《[[Coming_Soon!!|Coming Soon!!]]》（同年11月7日）<ref>{{cite web|url=http://natalie.mu/music/news/59129 |language=ja |title=<span lang="ja">カミスン！に斉藤和義、Perfume、キンキ、西野カナ、SKE</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2011-11-04 |accessdate=2013-02-11}}</ref>等音樂節目宣傳單曲。

== 歌曲 ==
《想要變得溫柔》為[[日本電視台|日本電視台]]週三晚間[[連續劇|連續劇]]《[[家政婦三田|家政婦三田]]》的主題曲。這是齊藤和義出道十九年來首次為民放電視劇提供主題曲。<ref>{{cite web|url=http://www.oricon.co.jp/news/movie/2002229/full/ |language=ja |title=<span lang="ja">斉藤和義、2年ぶりにドラマ主題歌～デビュー19年目で民放初</span> |publisher=[[Oricon|Oricon]] |date=2011-09-29 |accessdate=2013-02-11}}</ref>B面目《{{lang|ja|あいされたいやつらのひとりごと}}》則是[[讀賣電視台|讀賣電視台]]、日本電視台系綜藝節目《{{lang|ja|にけつッ!!}}》的主題曲。<ref>{{cite web|url=http://news.mynavi.jp/news/2011/10/09/011/index.html |language=ja |title=<span lang="ja">斉藤和義が『にけつッ!!』のテーマ曲を制作 - ケンコバ「めっちゃ嬉しい!」</span> |publisher=mynavi |date=2011-10-09 |accessdate=2013-02-11}}</ref>第三首歌曲《{{lang|ja|ウサギとカメ}}》原是收錄於其專輯《[[45_STONES|45 STONES]]》的歌曲，此單曲收錄的是齊藤和義於2011年8月31日在[[Ustream|Ustream]]演唱的版本。<ref>{{cite web|url=http://www.jvcmusic.co.jp/saito/yasashikunaritai/ |language=ja |title=<span lang="ja">やさしくなりたい｜斉藤和義</span> |publisher=SPEEDSTAR RECORDS |date=2011 |accessdate=2013-02-11}}</ref>

根據齊藤和義所說，他在聽說關於《家政婦三田》的內容時，得知該劇以失去不同事物的人們重生為主題，故以此主題寫了《想要變得溫柔》。他又指因為該劇有很多認真的部份，所以希望盡可能在自己的歌曲中加入救助的元素。<ref>{{cite web|url=http://natalie.mu/music/news/57244 |language=ja |title=<span lang="ja">斉藤和義、松嶋菜々子主演の家政婦ドラマに主題歌提供</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2011-09-29 |accessdate=2013-02-11}}</ref>

2012年12月31日，齊藤和義初次於[[第63回NHK紅白歌合戰|NHK紅白歌合戰]]演出，演唱歌曲正是《想要變得溫柔》。<ref>{{cite web|url=http://www.oricon.co.jp/music/special/page/340/2/ |language=ja |title=<span lang="ja">『第63回 NHK紅白歌合戦』出演アーティスト＆曲順 一覧</span> |publisher=[[Oricon|Oricon]] |date=2012 |accessdate=2013-02-11}}</ref>

== 音樂影片 ==
《想要變得溫柔》的[[音樂影片|音樂影片]]（MV）跟上一張單曲《一直都喜歡》一樣，都以[[披頭四樂隊|披頭四樂隊]]（{{lang|ja|ザ・ビートルズ}}）的演唱會為主題。這次模仿的是披頭四樂隊於1966年在[[東京|東京]][[日本武道館|日本武道館]]舉行的演唱會。<ref name="oricon20111101">{{cite web|url=http://www.oricon.co.jp/news/music/2003300/full/ |language=ja |title=<span lang="ja">斉藤和義、新曲MVでビートルズ完コピ　45年前の日本武道館公演再現</span> |publisher=[[Oricon|Oricon]] |date=2011-11-01 |accessdate=2013-02-11}}</ref>齊藤和義將MV中的樂隊名為「THE BEACHIKS」（{{lang|ja|ザ・ビーチクス}}）。雖然參與MV的成員跟上次相同，但飾演的角色有變。齊藤和義本人飾演[[約翰·連儂|約翰·連儂]]，[[中川雅也|中川雅也]]飾演[[保羅·麥卡尼|保羅·麥卡尼]]，[[2丁拳銃|2丁拳銃]]的[[小堀裕之|小堀裕之]]飾演[[佐治·夏里遜|佐治·夏里遜]]，[[濱田岳|濱田岳]]飾演[[靈高·史達|靈高·史達]]。<ref name="oricon20111101" />司儀方面，由當時為披頭四演唱會擔任司儀的[[岡田泰美|E·H·Eric]]的甥兒[[岡田真善|岡田真善]]飾演。<ref name="oricon20111101" />除了VOX公司製作的[[放大器|放大器]]外，所有舞台設置和衣着等等都完全複製自披頭四樂隊的武道館演唱會。<ref name="oricon20111101" />

== 排行榜成績 ==
此單曲得到2011年11月14日的[[Oricon公信榜|Oricon公信榜]]單曲周榜第6名。<ref name="oricon" />其後排名慢慢下跌，於三星期後（12月5日）跌至第34名；然後排名回升，於兩星期後（12月19日）升至第13名。<ref>{{cite web|url=http://www.hotexpress.co.jp/news/111213_kazuyoshisaito/ |language=ja |title=<span lang="ja">斉藤和義 キミを笑わせたい…「家政婦のミタ」主題歌がロングヒット</span> |publisher=hotexpress |date=2011-12-13 |accessdate=2013-02-11}}</ref>《[[家政婦三田|家政婦三田]]》於同年12月21日播映大結局，獲得極優異的收視率（平均40.0%，最高42.8%）{{noteTag|1=在該劇播映前，日本在1977年9月26日後播映的一般電視劇的大結局平均收視率首名是1983年《推倒積木・親子的200日戰爭》（{{lang|ja|積木くずし・親と子の200日戦争}}）的45.3%，第二名是2000年《[[美麗人生_(日本電視劇)|美麗人生]]》的41.3%，第三名是1979年《[[熱中時代|熱中時代]]》的40.0%。<ref>{{cite web|url=http://www.oricon.co.jp/news/movie/2005012/full/ |language=ja |title=<span lang="ja">『家政婦のミタ』最終回、驚愕の視聴率40％で幕！ 一般劇歴代視聴率3位に</span> |publisher=[[Oricon|Oricon]] |date=2011-12-22 |accessdate=2013-02-11}}</ref>該劇也是2011年最高收視的日本電視劇。<ref>{{cite web|url=http://www.oricon.co.jp/news/movie/2019001/full/ |language=ja |title=<span lang="ja">『家政婦のミタ』、12月に再放送　1年ぶりに4姉弟集結で新撮も</span> |publisher=[[Oricon|Oricon]] |date=2012-11-23 |accessdate=2013-02-11}}</ref>}}，此單曲亦得以在Oricon公信榜由第13名升至第8名，相隔七星期再次取得該榜頭十名。<ref>{{cite web|url=http://www.oricon.co.jp/news/rankmusic/2005141/full/ |language=ja |title=<span lang="ja">【オリコン】『家政婦のミタ』主題歌が７週ぶりTOP10</span> |publisher=[[Oricon|Oricon]] |date=2011-12-27 |accessdate=2013-02-11}}</ref>隨着大結局播映，《想要變得溫柔》一曲的累積下載次數亦超過了一百萬次。<ref>{{cite web|url=http://www.hotexpress.co.jp/news/111222_kazuyoshisaito/ |language=ja |title=<span lang="ja">斉藤和義 視聴率40％の「家政婦のミタ」、主題歌も大台突破</span> |publisher=hotexpress |date=2011-12-22 |accessdate=2013-02-11}}</ref>

在2012年[[第63回NHK紅白歌合戰|第63回NHK紅白歌合戰]]後，由於「紅白效應」，此單曲於Oricon公信榜上由200名以外急升至第28名，相隔48星期再次進到該榜頭三十名；單曲當時的累積銷量約為11.9萬張。<ref>{{cite web|url=http://www.oricon.co.jp/news/rankmusic/2020376/full/ |language=ja |title=<span lang="ja">【オリコン】ももクロ、斉藤和義ら紅白初登場組が急上昇</span> |publisher=[[Oricon|Oricon]] |date=2013-01-08 |accessdate=2013-02-11}}</ref>

== 反應 ==
音樂網站「hotexpress」的山本純評論指「來自『歌手』的訊息雖然非常簡單，卻悄悄地將自身心中睡着的重要事物確實地喚醒過來」。<ref>{{cite web|url=http://www.hotexpress.co.jp/review/3298/ |language=ja |title=<span lang="ja">“愛なき時代に生まれたわけじゃない”斉藤和義の普遍的な想い</span> |publisher=hotexpress |date= |accessdate=2013-02-11}}</ref>另一音樂網站「Listen Japan」的近藤陽則評論這首歌很有擅於描寫心中渴望的齊藤和義的風格，歌中「想要變得堅強 想要變得溫柔」{{noteTag|1=歌詞原文：{{lang|ja|強くなりたい　やさしくなりたい}}}}一句讓人留有深刻印象。<ref>{{cite web |url=http://listen.jp/store/diskreview_35021.htm |language=ja |title=<span lang="ja">斉藤和義／やさしくなりたい</span> |publisher=Listen Japan |date= |accessdate=2013-02-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20120223071157/http://listen.jp/store/diskreview_35021.htm |archivedate=2012-02-23 }}</ref>

《想要變得溫柔》獲得第71屆[[日劇學院賞|日劇學院賞]]的主題曲獎，該屆的第2名和第3名分別是[[嵐|嵐]]的《[[迷宮戀曲|迷宮戀曲]]》和[[中島美雪|中島美雪]]的《{{lang|ja|荒野より}}》。<ref>{{cite web|url=http://blog.television.co.jp/drama/academy/2012/02/post_775.html |language=ja |title=<span lang="ja">“ミタ”の斉藤和義が初受賞！</span> |publisher=<span lang="ja">webザテレビジョン</span> |date=212-02-08 |accessdate=2013-02-11}}</ref>此歌曲亦於2012年獲得第54屆[[日本唱片大獎|日本唱片大獎]]的優秀作品獎<ref>{{cite web|url=http://www.oricon.co.jp/news/movie/2018968/full/ |language=ja |title=<span lang="ja">第54回『レコ大』各賞発表　AKB、斉藤和義らが大賞候補</span> |publisher=[[Oricon|Oricon]] |date=2012-11-22 |accessdate=2013-02-11}}</ref>，以及於2013年獲得第27屆[[日本金唱片大獎|日本金唱片大獎]]的五大下載歌曲獎。<ref>{{cite web|url=http://www.golddisc.jp/award/27/Prize_2.html#gd27_23 |language=ja |title=<span lang="ja">第27回 ゴールドディスク大賞</span> |publisher=[[日本唱片協會|RIAJ]] |date=2013 |accessdate=2013-02-27}}</ref>
