'''效用'''（{{lang-en|'''Utility'''}}），是[[微观经济学|微观经济学]]中最常用的概念之一。一般而言，效用是指对于[[消费者|消费者]]通过消费或者享受闲暇等使自己的需求、欲望等得到的满足的一个度量。

== 历史 ==
效用的概念是[[丹尼尔·伯努利|丹尼尔·伯努利]]在解释[[圣彼得堡悖论|圣彼得堡悖论]]（丹尼尔的表兄[[尼古拉一世·伯努利|尼古拉一世·伯努利]]设计出来的一个悖论）时提出的，目的是挑战以金额期望值([[expected_monetary_value|expected monetary value]], EMV)作为决策的标准。

丹尼尔·伯努利对这个悖论的解答在1738年的论文裡，主要包括两条原理：
# [[边际效用递减|边际效用递减]]原理：一个人对于财富的占有多多益善，即效用函数一阶[[导数|导数]]大于零；随着财富的增加，满足程度的增加速度不断下降，效用函数二阶导数小于零。
# 最大效用原理：在风险和不确定条件下，个人的决策行为准则是为了获得最大期望效用值而非最大期望金额值。

== 效用的基数性和序数性 ==
[[经济学家|经济学家]]对于效用的理解是有一个过程的。19世纪的[[杰文斯|杰文斯]]、[[瓦尔拉斯|瓦尔拉斯]]和[[阿尔弗莱德·马歇尔|马歇尔]]等早期经济学家认为效用如同人们的身高和体重一样是可以测量的，而[[约翰·希克斯|希克斯]]（John Hicks，1946）则尝试了只在序数性效用的假定下，也取得了很多的研究成果。[[约翰·希克斯|希克斯]]认为，效用的数值表现只是为了表达偏好的顺序，并非效用的绝对数值。[[保罗·萨缪尔森|保罗·萨缪尔森]]也认为效用反映着人的欲望，而欲望是无法具体度量的，只能通过人们满足自己的愿望付出了多少价格来判断<ref>{{cite book |first=Alfred |last=Marshall |year=1920 |title=Principles of Economics. An introductory volume |edition=8th |location=London |publisher=Macmillan }}</ref>{{RP|78}}。从教科书等内容判断，现在比较通用的应该是后者的序数性效用。

== 边际效用 ==
{{main|边际效用}}
假设一个经济中存在<math>n</math>种商品，消费者从消费商品向量：

<math>\mathbf{x}=(x_1,x_2\cdots,x_n),\forall i\in \{1,2,\cdots,n\}, x_i\geq 0</math>

来得到效用。为了集中讨论效用问题，我们这里不考虑消费约束，也就是说消费者不用担心自己是否能够承担这些消费。假设消费者的效用函数存在（效用函数的存在性证明需要较大篇幅，这里不予给出），则可以表示成：

<math>U=u(\mathbf{x})</math>。

边际效用可以定义为，在其他情况不变的情况下，消费者通过增加一个单位的商品<math>i</math>的消费导致的效用的增加量。形式上，可以用下面的表达式来表示：

<math>MU=\frac{\partial u(\mathbf{x})}{\partial x_i}=u_i(\mathbf{x})</math>。
