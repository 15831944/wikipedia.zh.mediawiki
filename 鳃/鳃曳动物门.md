'''鳃曳动物门'''（[[學名|學名]]：{{lang|la|Priapulida}}）是[[动物界|动物界]]的一个[[门_(生物)|门]]，是一小类[[海洋|海洋]][[底栖动物|底栖动物]]，多分布在靠近[[两极地区|两极地区]]的冷海中，在泥沙中、淺海或[[棲管|管居]]生活。身体虽然有体环，但是不分节。目前仅报道过20多种。它们的近亲是[[动吻动物门|动吻动物门]]和[[鎧甲动物门|鎧甲动物门]]，此三門統稱為[[有棘動物|有棘動物]](Scalidophora)。本门的拉丁文名来自希腊神话中的生殖之神[[普里阿普斯|普里阿普斯]]，因本门的一些种在外观上类似雄性[[智人|智人]]的[[阴茎|阴茎]]。

== 物種 ==
「[[奧托蟲|奧托蟲]]」（Ottoia）又被稱為「丁丁蟲（penis worm）」（因外型像[[陽具|陽具]]而得名），是最著名的鰓曳動物。體長從8[[公分|公分]]到40公分不等，是有棘動物中比較大的類群。今日除了還有幾種奧托蟲仍存活在世外，其餘的都被保留在[[伯吉斯頁岩|伯吉斯頁岩]]（Burgess Shale Formation）中。<ref>{{Cite web |url = http://chuansong.me/n/1765017752915 |title = 曳鰓、鎧甲、動吻：有棘動物類群 |author = 小白 |date = 2017年4月12日 |language = zh-hans }}</ref>
