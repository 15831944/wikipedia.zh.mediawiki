'''消化系統（消化道）氣體'''或'''胃腸氣'''、胃腸氣脹，俗稱'''屁'''（flatulence），亦稱'''矢氣'''<ref>[http://m.sohu.com/n/483917617/] 搜狐健康：中醫將放屁叫做「矢氣」，總是放屁是怎麼回事？ 2018年02月09日</ref>，指[[動物|動物]][[消化道|消化道]][[肛門|肛門]]排出的[[氣體|氣體]]。有研究指出，[[牛|牛]][[羊|羊]]等[[反芻動物|反芻動物]]放屁所產生的[[甲烷|甲烷]]可能導致[[全球暖化|全球暖化]]<ref>{{cite news|title=牛屁害了地球？　聯合國報告：牛的廢氣污染比汽車嚴重 |url=http://e-info.org.tw/node/17555 |language=中文 |publisher=東森新聞 |date=2006年12月12日 }}</ref>。

== 来源 ==
屁的主要气体来源有二：
# 咀嚼时吞入[[消化道|消化道]]的空气；
# [[大肠|大肠]]和[[直肠|直肠]]内的[[细菌|细菌]]，特别是[[大肠杆菌|大肠杆菌]]，[[发酵|发酵]][[食物|食物]]残渣而产生的气体。

== 屁与食物 ==
屁的气味和产量与食物有重大的关系。

当食物[[蛋白质|蛋白质]]含量较高（如肉类），屁中[[硫化氢|硫化氢]]、[[吲哚|吲哚]]和[[粪臭素|粪臭素]]的含量升高，此时屁有明显的臭味；当食物[[淀粉|淀粉]]含量较高（如薯类），屁中[[二氧化碳|二氧化碳]]含量升高，此时屁量明显增多。因为[[二氧化碳|二氧化碳]]的增加量明显大于[[硫化氢|硫化氢]]、[[吲哚|吲哚]]和[[粪臭素|粪臭素]]，而且气压增大时，放屁的气流速度也增大，放屁的[[声音|声音]]也就更响，但是[[二氧化碳|二氧化碳]]没有气味，所以有“响屁不臭，臭屁不响”和“响屁不臭闷屁臭”的说法。

某些食物因其内含有的特殊物质，可以使屁的产量提高或者气味改变，比如：
# [[红薯|红薯]]：含有气化[[酶|酶]]，在[[胃酸|胃酸]]的作用下容易产生大量的[[二氧化碳|二氧化碳]]气体。
# [[白萝卜|白萝卜]]：其成分[[淀粉酶|淀粉酶]]和[[木质素|木质素]]{{fact|会疏理肠气}}，所以生吃会放屁，而且有特殊臭气，俗称“萝卜屁”。
# [[洋葱|洋葱]]、[[韭菜|韭菜]]、[[大蒜|大蒜]]：含有刺激性物质，过量食用会使屁有洋葱、韭菜、大蒜气味。
# [[大豆|大豆]]等豆类：含有大量[[寡糖|寡糖]]和[[蛋白质|蛋白质]]，可以产生大量的气体和恶臭物质，故屁多而臭。

== 屁与健康 ==
正常人每天都會放屁5至10次，约排出500毫升左右的气体。屁的气味、多少都能指示[[消化道|消化道]]的[[健康|健康]]状况。

若屁多但不臭，可能有[[胃炎|胃炎]]、[[消化性溃疡|消化性溃疡]]等[[胃|胃]]部疾病及[[肝|肝]]、[[胆|胆]]、[[胰|胰]]疾病等。

若屁多且臭，可能有[[消化不良|消化不良]]。

若屁奇臭难闻，可能有[[消化道出血|消化道出血]]、[[菌痢|菌痢]]、[[阿米巴痢疾|阿米巴痢疾]]、[[溃疡性结肠炎|溃疡性结肠炎]]、[[出血性小肠炎|出血性小肠炎]]等炎症。此外，恶性[[肿瘤|肿瘤]]晚期，因癌肿组织糜烂，蛋白质腐败，由于细菌的作用，放出的屁也会很臭。

若无屁或少屁，并有如[[腹痛|腹痛]]、[[腹胀|腹胀]]、[[便秘|便秘]]、肠鸣音亢进或消失、气过水声等症状，则可能是[[肠梗阻|肠梗阻]]。此外，腹部手术者，其肠蠕动会出现反射性抑制，胃肠内气体和液体滞积，也会出现无屁的情况

== 俚语 ==
中文中'''放屁'''可以指：

* 从[[肛门|肛门]]排出'''臭气'''（屁）的行为或动作。
* [[脏话|骂人的话]]，用以比喻说话没有根据或不合情理。近义的同源用语还有：“屁！”、“狗屁！”、“放狗屁！”等。
