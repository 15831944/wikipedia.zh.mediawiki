'''郭振山'''（{{bd|1962年|11月||}}），男，[[河北省|河北省]][[东光县|东光县]]人，现任[[天津美术学院|天津美术学院]]副院长，教授、硕士生导师，《中国水粉画》杂志主编。郭振山是中国当代知名水粉画家，为[[中国美术家协会|中国美术家协会]]会员及天津美术家协会理事、中国包装联合会设计委员会副主任。<ref>{{cite web | url=http://news.163.com/10/1227/10/6OTEK0BB00014AED.html | title=天津美院院长郭振山来青城亲授画技 | publisher=网易 | accessdate=2012-12-08}}</ref>著有《现代中国水粉画家郭振山水粉静物精选》。

郭振山跨界设计了诸多题材的邮品，并垄断了天津市大型运动赛事的会徽和吉祥物的评审、设计工作，尤其以2013年东亚运动会、2017年全运会吉祥物具有争议性并引发关注<ref name="郭振山全运会吉祥物" />。

== 简历 ==
郭振山出生于[[河北省|河北省]][[沧州市|沧州市]]东光县，1981年9月参加工作。后考入[[天津美术学院|天津美术学院]]装潢艺术设计系。1986年6月，毕业前一年，郭振山加入了中国共产党<ref>{{cite news|title=天津市市管干部提任前公示|url=http://www.tj.gov.cn/zwgk/gsgg/szfgg/201403/t20140327_233602.htm|accessdate=2016-08-20|agency=天津日报|date=2014-03-27|deadurl=yes|archiveurl=https://web.archive.org/web/20160921194518/http://www.tj.gov.cn/zwgk/gsgg/szfgg/201403/t20140327_233602.htm|archivedate=2016-09-21}}</ref>。1987年6月，郭振山毕业后留校任教。郭振山曾先后担任天津美术学院视觉传达设计系主任、天津美术学院副院长、设计学院院长等职务。<ref name="品牌中国网">{{cite web | url=http://www.brandcn.com/ppj/oneppj/cyr/200707/94757.html | title=天津美术学院设计艺术学院副院长郭振山成为品牌节倡议人 | publisher=品牌中国网 | accessdate=2012-12-08}}</ref>

2004年2月，天津市文联举办的天津市第三届中青年“德艺双馨”文艺工作者评选中，郭振山被评为天津市第三届中青年“德艺双馨”文艺工作者<ref name='中青年"德艺双馨"文艺工作者评选揭晓'>{{cite news|author1=尚志明|title=天津：中青年"德艺双馨"文艺工作者评选揭晓|url=http://www.tj.xinhuanet.com/news/2004-02/18/content_1645663.htm|accessdate=2016-08-20|agency=新华社|publisher=新华网|date=2004-02-18|deadurl=yes|archiveurl=https://web.archive.org/web/20170329013052/http://www.tj.xinhuanet.com/news/2004-02/18/content_1645663.htm|archivedate=2017-03-29}}</ref>。2009年，时任天津美术学院设计学院副院长的郭振山入选天津市宣传文化“五个一批”人才的文艺类别人才<ref name="郭振山教授入选天津市宣传文化“五个一批”人才">{{cite web|author1=郭振山|title=郭振山教授入选天津市宣传文化“五个一批”人才|url=http://www.tjculture.com/read.php?id=161839|website=天津文化艺术网|publisher=天津文化艺术网|accessdate=2016-08-20}}</ref>。

2014年7月，郭振山担任当选天津市包装技术协会设计委员会主任职务<ref name="郭振山当选天津市包装技术协会设计委员会主任职务">{{cite news|title=郭振山当选天津市包装技术协会设计委员会主任职务|url=http://www.022meishu.com/Design/2014/07-17/9458_0.html|accessdate=2016-07-11|agency=天津美术网}}</ref>。2016年6月，天津市美术家协会水彩画专业委员会换届，郭振山当选新一届副会长<ref name="美协水彩画专委会换届">{{cite news|author1=周凡恺|title=美协水彩画专委会换届|url=http://epaper.tianjinwe.com/tjrb/tjrb/2016-06/03/content_7460947.htm|accessdate=2016-08-20|agency=天津日报|publisher=天津日报|date=2016-06-03|archive-url=https://web.archive.org/web/20160826003725/http://epaper.tianjinwe.com/tjrb/tjrb/2016-06/03/content_7460947.htm|archive-date=2016-08-26|dead-url=yes}}</ref>。

== 水粉画创作与教学 ==
郭振山专长是水粉静物画，最显著的特点是画面干净、明快，色彩鲜明，质感强烈，笔法讲究，尤其擅长以水果为素材的水粉画，著有《现代中国水粉画家郭振山水粉静物精选》，该书为水粉画专业的畅销教材<ref>{{cite journal|author1=白路|title=水色风范，粉彩雅趣——青年水粉画家郭振山先生述评|journal=美术之友}}</ref>。他的作品水粉画作品《妞妞》入选第三届全国水彩粉画展、《青》入选首届中国水彩画展、《跳动的旋律》入选第第九届全国美展。<ref name="品牌中国网" />

== 邮品设计 ==
郭振山曾参与《社会发展共创未来》《中华人民共和国第八届运动会》《国际老年人年》《亚太经合组织2001会议中国》《二滩水电站》《2002年世界杯足球赛》《沙漠植物》《2012年全国大学生运动会》《联合国成立七十周年》等重大题材的邮票、纪念封的设计。<ref name="品牌中国网" /><ref>{{cite news|title=郭振山设计《联合国成立七十周年》纪念邮票发行|url=http://www.022meishu.com/Design/2015/10-08/23568_0.html|accessdate=2016-08-20|agency=天津美术网|publisher=天津美术网|date=2015-10-08}}</ref>

== 会徽、吉祥物设计 ==
郭振山专长是水粉画创作，但却跨界参与了多项重大体育赛事、活动和组织的会徽、吉祥物设计。自1990年代以来，郭振山垄断了天津市举办、承办的大型赛事的会徽、吉祥物的设计工作，在设计活动中同时兼任设计评审和设计参选者，如1999年天津世界体操锦标赛会徽和[[2012年全国大学生运动会|2012年全国大学生运动会]]、[[2013年天津东亚运动会|2013年天津东亚运动会]]和[[天津泰达足球俱乐部|天津泰达足球俱乐部]]的吉祥物等<ref>{{cite web | url=http://sports.enorth.com.cn/system/2011/01/26/005595563.shtml | title=郭振山参与泰达吉祥物设计 曾设计东亚运吉祥物 | publisher=北方网 | accessdate=2012-12-08}}</ref><ref>{{cite web | url=http://news.163.com/10/0222/10/604AR1U8000120GR.html | title=“津津东东”设计人：天使分别象征友谊和谐 | accessdate=2012-12-08}}</ref>
。其中，天津泰达足球俱乐部在公开征集吉祥物后，郭振山曾主动登门拜访要求为其设计吉祥物<ref name="郭振山参与泰达吉祥物设计">{{cite news|author1=刘欣|title=郭振山参与泰达吉祥物设计 曾设计东亚运吉祥物|url=http://sports.enorth.com.cn/system/2011/01/26/005595563.shtml|accessdate=2016-08-20|agency=天津北方网|publisher=天津北方网|date=2011-01-26}}</ref>。

郭振山在吉祥物设计方面，倡导“在外形和内容上更多的融入浓郁的地方特色，将民族文化和地方文化要素融为一体”，因此，郭振山在设计吉祥物是偏好采用[[杨柳青年画|杨柳青年画]]的元素<ref name="大型综合性运动会吉祥物的设计原则和要求">{{cite news|title=郭振山：大型综合性运动会吉祥物的设计原则和要求|url=http://www.022meishu.com/Design/2013/07-25/2452_0.html|accessdate=2016-08-20|agency=天津美术网|publisher=天津美术网|date=2013-07-25}}</ref>。特别是设计的2013年天津东亚运动会吉祥物“津津”和“东东”，这对吉祥物被外界认为十分简陋，具有浓重的乡土气息，引发恶评如潮<ref>{{cite web | title = 2013年第六届东亚运动会吉祥物——津津东东 |url=http://sports.enorth.com.cn/system/2009/12/02/004300927.shtml | accessdate = 2010年1月7日 | date = 2009年12月2日| publisher=北方网}}</ref>。2016年7月3日，[[中华人民共和国第十三届运动会|2017年天津全运会]]会徽和吉祥物对外公布，郭振山设计的“津娃”因形象与现代审美观有激烈的冲突，再度招致恶评并引发争议，郭振山在回应中称“无论从发型、眼睛、动作等等各个方面综合考量，都是一个二十一世纪初创造的现代卡通形象”<ref name="郭振山全运会吉祥物">{{cite news|author1=谷菲|title=“津娃”创作者：具有极高识别度 最能代表天津|url=http://sports.enorth.com.cn/system/2016/07/06/031051910.shtml|accessdate=2016-08-20|agency=北方网|date=2016-08-20}}</ref>。

在第十三届全运会开幕前夕，郭振山设计的吉祥物津娃开始大规模亮相，遭到网民的批评，甚至有有运动员表示“看的自己都不想参加全运会”<ref>{{cite news|title=全运会吉祥物"津娃"草雕亮相 运动员吐槽看了不想参加比赛|url=http://news.ifeng.com/a/20170728/51523312_0.shtml|agency=凤凰网|date=2017-07-28}}</ref>。2017年7月底，由于舆论对津娃草雕造型评价尤其负面，津娃草雕在进场放置后被组委会连夜撤掉，不知所踪<ref>{{cite news|author1=郭城|title=草雕“津娃”再刷网友审美观，官方闻讯连夜撤后说了啥？|agency=央视网|date=2017-07-29}}</ref>。
