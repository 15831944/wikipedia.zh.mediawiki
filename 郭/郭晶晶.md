'''郭晶晶'''{{BD|1981年|10月15日||}}是[[河北|河北]][[保定|保定]]人，[[中国|中国]]著名[[跳水|跳水]]运动员。分别在[[2004年夏季奥林匹克运动会|2004年雅典奥运会]]和[[2008年北京奥运会|2008年北京奥运会]]上，获得3米跳板跳水冠军；并且和[[吳敏霞|吳敏霞]]合作，同时蝉联了这两届奥运会双人3米跳板的冠军。她总共获得4枚奥运金牌2枚银牌，是继[[高敏|高敏]]、[[伏明霞|伏明霞]]后，[[中国女子跳水队|中国女子跳水队]]的“领军人物”。郭晶晶於2011年初宣布退役；她在役的14年間，為中國跳水隊贏得31个世界冠軍，成為擁有最多世界冠軍銜頭的中國運動員<ref name="燕赵晚报110124">[http://sports.sina.com.cn/o/2011-01-24/11025426670.shtml 郭晶晶三度萌生退役想法 元旦后正式提出退役申请]《燕赵晚报》2011-01-24</ref>。2013年8月27日，郭晶晶在香港产下一子，体重3,300公克。<ref>[http://www.apdnews.com/news/32987.html 霍启刚微博报喜郭晶晶产子] {{webarchive|url=https://web.archive.org/web/20131225160303/http://www.apdnews.com/news/32987.html |date=2013-12-25 }}，亚太日报，2013年8月27日</ref>

== 运动经历 ==
1988年在河北保定训练基地开始跳水训练，教练是李芳；1993年进入河北省跳水队，教练是李芳；1992年入选国家跳水队，教练是[[于芬|于芬]]。<ref>[http://sports.sina.com.cn/star/guo_jingjing/ 郭晶晶运动经历]《新浪体育》</ref> 1998年转由[[钟少珍|钟少珍]]教练手下。

[[1996年夏季奥林匹克运动会|1996年亚特兰大奥运会]]，年仅15岁的郭晶晶第一次登上了奥运会的舞台，那时的实力和名气都无法与对手抗衡，最终获得了十米台项目第五名。[[2000年夏季奥林匹克运动会|2000年悉尼奥运会]]，郭晶晶在3米跳板项目中无法撼动当时的师姐[[伏明霞|伏明霞]]在该项目的统治地位而取得一枚银牌，而在和[[伏明霞|伏明霞]]合作的奥运会新项目双人3米跳板中却最终不敌[[俄罗斯|俄罗斯]]组合[[尤里娅·帕卡琳娜|尤里娅·帕卡琳娜]]/[[维拉·伊莲娜|维拉·伊莲娜]]屈居第二，使得自己的第二次奥运之旅与金牌擦肩而过。[[2004年夏季奥林匹克运动会|2004雅典奥运会]]以中国女子跳水队头号选手身份出征的郭晶晶终于在3米跳板个人赛和双人赛中折桂。[[2008年夏季奥林匹克运动会|2008北京奥运会]]上郭晶晶与[[吴敏霞|吴敏霞]]搭档，再次夺得了3米跳板双人赛金牌，同时还蝉联了个人赛冠军。至此，郭晶晶收获了自己在奥运会历史上的第四枚金牌，和[[邓亚萍|邓亚萍]]、[[伏明霞|伏明霞]]、[[王楠|王楠]]并列成为中国获得奥运会金牌最多的选手。同时，她也以4金2银的奥运奖牌数超越[[伏明霞|伏明霞]]（4金1银）。自1912年夏季奥运会设立女子跳水项目以来，郭晶晶是第一位在该项目上获得了6枚奖牌的女子运动员。<ref>[http://www.beijing2008.cn/news/sports/headlines/diving/n214556310.shtml 女子3米跳板郭晶晶完美折桂 4金2银奥运战绩创历史] {{webarchive|url=https://web.archive.org/web/20080906233602/http://www.beijing2008.cn/news/sports/headlines/diving/n214556310.shtml |date=2008-09-06 }} beijing2008.cn 第29届奥林匹克运动会官方网站</ref>

从2001年至2009年之间的五届世锦赛中郭晶晶囊获了10枚金牌。<ref>[http://hk.news.yahoo.com/article/080827/3/7xa2.html 郭晶晶无意退役或入港队]《[[星岛日报|星岛日报]]》2008-08-28]</ref>

===退役===
郭晶晶在[[2009年東亞運動會|2009年東亞運動會]]後，向國家隊提出休息一年<ref name="燕赵晚报110124"/>。期間，她在[[英國|英國]]進修[[英語|英語]]，雖然學習時間並不長，但有不錯的效果<ref name="新闻晨报110124">[http://sports.sina.com.cn/o/2011-01-24/12405426865.shtml 郭晶晶正在英国读书 曾透露结束学业后想做公务员]《[[新闻晨报|新闻晨报]]》2011-01-24</ref>。

在淡出405天之後，中國跳水隊領隊[[周繼紅|周繼紅]]終在2011年1月22日確認，郭晶晶已遞交了退役申請，並盛讚郭「創造了一個屬於自己的郭晶晶時代」<ref name="新闻晨报110124"/>。兩日後，[[河北省|河北省]]體育局也證實，已收到及接納郭晶晶的退役申請；並表示郭在退役後的身份，仍會是河北省體育局游泳跳水運動管理中心的副主任兼河北省跳水隊教練<ref>[http://sports.sina.com.cn/o/2011-01-24/20015427389.shtml 河北体育局接受郭晶晶退役申请 跳水女皇告别泳池]《新华网》2011-01-24</ref>。

郭晶晶退役後連續三年為健康快車主辦的慈善跑步行擔任活動大使。於2017年9月，她與奧運體操金牌得主李小鵬，一起於記者會上示範運動前簡單熱身動作，於訪問中表示兒子霍中曦性格活躍喜歡運動，之前為他安排教練上堂學，可是他不喜歡游水，不會強迫他，怕有反效果<ref>[http://bka.mpweekly.com/focus/local/20170921-82203 郭晶晶拒逼囝囝學游水]</ref>。

== 恋情 ==
郭晶晶早期曾与本国跳水选手[[田亮|田亮]]的名字连在一起，合称“亮晶晶”；香港传媒时常描述他们关系亲密。在2004年，香港传媒开始描写郭晶晶与前[[全国政协副主席|全国政协副主席]][[霍英东|霍英东]]长孙[[霍启刚|霍启刚]]交往，之后感情渐趋稳定。在2005年11月，香港传媒更称郭晶晶与霍启刚在北京崇文区购入豪宅，事件未经证实。

2012年11月8日，郭晶晶和[[霍启刚|霍启刚]]在香港正式註冊結婚。2013年8月27日，郭晶晶為霍啟剛誕下一名男嬰<ref>[http://www.apdnews.com/news/17861.html 郭晶晶与霍启刚在香港註册结婚] {{webarchive|url=https://web.archive.org/web/20130120084247/http://www.apdnews.com/news/17861.html |date=2013-01-20 }}，亚太日报，2013年8月20日</ref><ref>{{Cite news|url=http://sports.sina.com.cn/o/2012-11-08/15166288821.shtml|title=郭晶晶霍启刚正式注册结婚 跳水女皇成霍太太(图)|publisher=[[新浪网|新浪网]]|date=2012-11-08|language=zh-hans}}</ref>霍中曦。2017年4月21日，郭晶晶再為霍啟剛誕下一名女嬰霍中妍<ref>{{Cite news|url=http://hk.apple.nextmedia.com/enews/realtime/20170426/56614874|title=【外公離世】朱玲玲一個月內經歷紅白事 霍啟剛：多謝關心|publisher=[[蘋果日報|蘋果日報]]|date=2017-04-26|language=zh-hant}}</ref>。

== 健康問題 ==
郭晶晶在跳水時，因長期受高壓訓練，除膝蓋突出，因視網膜受損而嚴重近視，也患上[[風濕性關節炎|風濕關節炎]]、第三[[椎间盘脱出症|椎間盤突出]]和[[胸膜炎|胸膜炎]]。<ref>{{Citation|last=纵观古今---圈内叫兽|title=隱瞞17年，郭晶晶的病終於瞞不住了，難怪霍啟剛對她死心塌地|url=https://www.youtube.com/watch?v=mUXLOZLysd8|accessdate=2019-01-09}}</ref>
