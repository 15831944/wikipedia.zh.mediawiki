原'''內湖郭氏古宅'''，又稱「內湖紅樓」<ref>{{cite book|author=林宜君撰文|title=市定古蹟內湖庄役場會議室暨郭氏古宅調查研究計畫|year=2000|publisher=臺北市內湖區公所|location=臺北市|isbn=9570113847|pages=9}}</ref>，位於[[臺灣|臺灣]][[臺北市|臺北市]][[內湖區|內湖區]]，1917年興建{{noteTag|1=郭氏後人與臺北市政府文化局登載資料皆為1917年，然根據林宜君撰文《市定古蹟內湖庄役場會議室暨郭氏古宅調查研究計畫》指出宅第興建時期有下列幾種說法：陳金讚《內湖傳家寶》記載為1917年；李乾朗《臺北市古蹟簡介》記載為1919年；《市定古蹟內湖庄役場會議室暨郭氏古宅調查研究計畫》解讀郭氏古宅土地登記簿史料，研判應為1924年興建。}}，有「內湖地區最精緻的磚樓」雅譽。1999年6月29日指定為直轄市定古蹟<ref>{{cite web|author=|title=內湖郭氏古宅|url=https://nchdb.boch.gov.tw/assets/advanceSearch/monument/19990629000006|work=文化部文化資產局|accessdate=}}</ref>，現時名為「郭子儀紀念堂」，亦是世界郭氏宗親總會所在地。

== 歷史 ==
[[雍正|雍正]]5年（1727年）來臺，築居於[[淡水廳|淡水廳]][[芝蘭堡|芝蘭堡]]內湖州仔，租園拓墾，首開內湖郭氏事業之先。郭崇飽育五子，開枝散葉，至[[臺灣日治時期|臺灣日治時期]]末，已有四代香火。此時郭家漸在地方嶄露頭角，第四代子孫始進入[[公職|公職]]體系，從事地方政經活動。

四世[[郭文鴻|郭文鴻]]飽讀經史，曾開設「文昌堂」[[私塾|私塾]]，來學就教者眾。五世[[郭章龍|郭章龍]]為郭文鴻二子，原為[[光緒|光緒]]年間[[秀才|秀才]]，於日治時期掌內湖區長一職（1908年）。1920年7月30日，[[殖民|殖民]]政府頒令地方新制，此時當選首任[[內湖庄|內湖庄]]長之人即是郭氏六世子孫[[郭華讓|郭華讓]]。

興建內湖紅樓的[[郭華溪|郭華溪]]與郭華讓互為堂親，歷任內湖[[保正|保正]]、庄協議會員、內湖[[公學校|公學校]]保護者會長等職，並捐助多項公共設施。

日治時期[[大正|大正]]初年，郭華溪繼承父親[[郭章安|郭章安]]之祖產，經營[[貸地業|貸地業]]，成為大地主。郭華溪成家後，遷出位於今[[臺北市立內湖國民中學|內湖國中]]校園內的祖厝，大正13年（1924年），擇定內湖[[獅頭山|獅頭山]]山腰現址，購地興建「內湖紅樓」。

== 特色 ==
[[File:內湖郭氏古宅對聯.jpg|thumb]]。]]
[[File:內湖郭氏古宅蓮花池.jpg|thumb]]池。]]
「內湖紅樓」[[建材|建材]]使用[[福州杉|福州杉]]、[[烏心石|烏心石]]、[[檜木|檜木]]、[[肖楠|肖楠]]等，為[[磚|磚]]造兩層樓[[建築|建築]]，坐[[北|北]]朝[[南|南]]，可遠眺[[公館山|公館山]]及[[南港山_(臺北市)|南港山]]脈，平面略呈「凸」字形，正身左右各突出兩層樓房間，前後端各設通廊。為「三間式正身」（即正廳、左耳房、右耳房布局）、「一正後側兩廂」布局（正廳、左耳房、右耳房在前，後夾左右偏房）。立面[[山牆|山牆]]對稱，除有中式傳統吉祥圖案、西方花草紋樣，並有洗石子、泥塑及進口彩色瓷磚裝飾。二樓前端規劃突出弧狀[[陽臺|陽臺]]，增加建築物主體縱線之視覺變化。圓拱[[窗|窗]]、尖拱窗亦遍布周身，富有節奏韻律。紅樓屋頂則為板瓦屋坡，正身山牆為火形[[馬背|馬背]]。有一說為紅樓採取寶座造型，陽臺如椅、[[女牆|女牆]]短柱為扶手、尖形山頭為椅背，如此依山穩坐眺望，是出於[[風水|風水]]考量之配置。紅樓前庭設置[[花園|花園]]，後院密植果林，顯示當時鄉紳對於居家風格與生活品味的理想，亦可從建築式樣得知其遠瞻及創新的意念。內湖紅樓中各種不同[[文化|文化]]的圖像與建材融合密切，類似[[迪化街|迪化街]]洋樓，是為大正年間典型的[[豪宅|豪宅]]。

== 功能特色 ==
[[File:內湖郭氏古宅近景.jpg|thumb]]
[[File:內湖郭氏古宅山牆.jpg|thumb]]。]]
=== 堂號汾陽 ===
內湖紅樓自1924年起即為郭華溪自宅，山牆題有「[[汾陽|汾陽]]」二字，以示承襲郭氏宗族[[堂號|堂號]]。正門亦可見以「華」、「溪」二字起首之[[對聯|門聯]]（現已不存）。臺灣日治時期，總督府欲規劃臺北城為可容納60萬人的[[城市|城市]]，並於[[昭和|昭和]]7年（1932年）擇定城外東北隅興建[[機場|機場]]（即今「[[臺北松山機場|臺北松山機場]]」）。昭和11年（1936年）機場完工，此時正值[[太平洋戰爭|太平洋戰爭]]多變局勢，機場成為[[盟軍|盟軍]]轟炸目標，當局勸導周遭居民疏散，而內湖紅樓恰位於航道，郭華溪為顧及家人安全，遂將家產轉售時任內湖庄庄長郭華讓等宗親，隨即舉家遷居[[桃園區|桃園]]。昭和15年（1940年），郭華讓購得此地，將紅樓做為[[別墅|別墅]]，又於民國39年（1950年）贈予七世郭國荃。後因宗親族人相繼赴海外發展，紅樓則租借[[佃農|佃農]]使用。

=== 籌建碧奉宮 ===
汾陽堂額曾有一段時間被更換為剪黏，上書「內湖庄六大角／碧奉宮／歲次甲午穀旦重建」等字，蓋因內湖庄六大角頭（頂番陂、梘頭、港墘、新陂尾、下番陂、山腳等）祀神並無固定廟宇安座，六大角神明群（[[天上聖母|天上聖母]]、[[福德正神|福德正神]]、[[五穀先帝|五穀先帝]]、[[三官大帝|三官大帝]]、[[開漳聖王|開漳聖王]]）「有神無廟」，每年由輪值[[爐主|爐主]]祭奉，由來已久；1980年代，紅樓一度欲為供奉內湖庄[[六大角|六大角]][[天上聖母|天上聖母]]宮址，由於紅樓前埕深度不足以及遭受鄰近居民反對而未實現，致使古宅荒廢經年，蔓草叢生。

=== 郭子儀紀念堂 ===
為緬懷郭氏先祖[[郭子儀|郭子儀]]，現古宅主人郭詩詠、郭明晉昆仲無條件提供郭氏古宅產權，申請修建為「郭子儀紀念堂」之用，郭氏宗親總會得[[臺北市政府|臺北市政府]]及全球郭氏宗親贊助，負責古宅修復及嗣後維護。[[古蹟|古蹟]]建築修復設計由林錦鴻建築師設計監造，古蹟匠師郭清波承攬修復工程執行，室內空間再利用展示軟體由陳順惠建築師規劃設計。2010年6月1日，修建工程啟動，2012年1月5日完工。古宅修復後更名為「郭子儀紀念堂」，奉祀汾陽王郭子儀，現時亦為世界郭氏宗親總會所在地。

== 地理位置 ==
郭氏古宅（郭子儀紀念堂）位於臺灣臺北市內湖區文德路267巷19號，造訪時可由文德路267巷巷口旁（[[內湖高中|內湖高中]]斜對面），依導覽指示沿坡道直行，或由文湖線捷運[[文德站_(台灣)|文德站]]1號出口處專闢登山步道拾階而上即可抵達。
