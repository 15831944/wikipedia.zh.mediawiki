'''郭尊先'''（{{bd|1958年|1月||}}）[[河北省|河北省]][[霸州市|霸州市]][[王庄子乡|王庄子乡]]王圪垯村人。[[中国人民解放军少将|中国人民解放军少将]]、[[中华人民共和国|中华人民共和国]]书法家<ref name=zgjw>{{Cite web |url = http://www.81.cn/2015aaaaa/2015-01/12/content_6305258.htm |title = 郭尊先简历 |author =  |date = 2015-01-12 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref><ref name=xhw>{{Cite web |url =  http://news.xinhuanet.com/mil/2016-08/12/c_129225466.htm |title = 一身正气写春秋 两袖清风铸军魂——记国防大学联合指挥与参谋学院原政委郭尊先 |author =  |date = 2016-08-12 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1978年3月入伍，到[[河北省|河北省]][[易县|易县]][[中国人民解放军总参谋部|中国人民解放军总参谋部]]某通信团服役。历任汽车驾驶员、团政治处报道员。1981年12月加入[[中国共产党|中国共产党]]。1981年12月由战士提干，历任排长、团宣传干事、连副指导员、团组织干事。1985年3月，考入[[中国人民解放军武汉通信指挥学院|中国人民解放军武汉通信指挥学院]]政工专业学习。1987年2月，毕业回总参谋部某通信团，历任连指导员、团组织股股长。1990年4月，调任总参谋部某通信总站组织科副营职干事（师党委秘书）。1991年3月，任营教导员。1993年8月，任团政治处副主任<ref name=zgjw/>。

1994年1月，调入[[中国人民解放军国防大学|中国人民解放军国防大学]]。历任国防大学政治部组织部干事、政治部组织部副部长。2006年2月，调任国防大学校务部政治处主任。2008年初，经组织批准，进入国防大学基本系联合战役指挥员班学习1年。同年底，出任国防大学校务部副部长。2011年5月，任国防大学进修系主任。2012年7月，任[[中国人民解放军国防大学进修学院|国防大学进修学院]]院长。2013年2月，任[[中国人民解放军国防大学联合指挥与参谋学院|国防大学联合指挥与参谋学院]]政治委员（副军职）。他自[[中共中央党校|中共中央党校]]哲学专业毕业，取得研究生学历<ref name=zgjw/>。2016年退役<ref name=xhw/>。

郭尊先自幼学习[[书法|书法]]。2005年初，拜书法家[[苏季群|苏季群]]为师<ref name=xhw/>。在国防大学任职期间，还兼任[[中国文联|中国文联]]书画艺术中心一级创作员，[[中国书法家协会|中国书法家协会]]会员，中国楹联协会会员，北京书法家协会会员。截至2015年，他曾在《[[人民日报|人民日报]]》、《[[光明日报|光明日报]]》、《[[解放军报|解放军报]]》等报刊发表20余篇文章并获奖，还曾立三等功2次。2004年，他被[[中国人民解放军总政治部|中国人民解放军总政治部]]评选为全军优秀纪检干部，2006年受到[[中央纪委|中央纪委]]、[[中华人民共和国监察部|监察部]]嘉奖为全国纪检系统先进个人<ref name=zgjw/>。
