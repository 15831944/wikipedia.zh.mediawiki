'''郭奕芯'''（{{lang-en|Ashina Kwok}}，{{bd|1995年|11月22日}}），原名'''郭嘉銘'''，[[香港|香港]]電影電視劇演員，出道代表作《[[同班同學_(2015年電影)|同班同學]]》、《[[鬼網|鬼網]]》、《[[拳道|拳道]]》、《[[非分熟女|非分熟女]]》等。

==簡歷==
郭奕芯於內地出生後來港升學，13歲時就讀於[[匯知中學|匯知中學]]時簽約入行做模特兒，2009年在比賽中獲全港甜美校花，2012年獲邀加入北京AG娛樂華人女團Angel girl，全國發佈會當日首唱更獲[[周杰倫|周杰倫]]力挺，[[方文山|方文山]]首度招收「開門弟子」，2014年同期間出過三本寫真集<ref>{{Cite web|url=http://jet.my-magazine.me/article/detail/interview/8195|title=JET - Interview - Interview - 二十歲之後 郭奕芯|website=Jet|language=zh-HK|access-date=2018-07-23}}</ref><ref>{{cite web |title=18歲嫩模郭奕芯捧33C青筋乳 跪求周星馳「給我機會」|url=https://star.ettoday.net/news/400898 |website=ETToday星光雲 |accessdate=2019-04-18 |language=zh-TW |date=2014-09-13}}</ref><ref>{{cite web |title=同班同學大起底 噴血肉照任睇 |url=https://eastweek.my-magazine.me/main/47251 |website=東周網 |accessdate=2019-04-18 |language=zh-HK |date=2015-11-03}}</ref>。

2015年，郭奕芯簽約Royal Management並參與電影拍攝<ref>{{cite web |title=彭浩翔新女鬥膽大 |url=https://hk.lifestyle.appledaily.com/nextplus/magazine/article/20151015/2_17642749/%E5%BD%AD%E6%B5%A9%E7%BF%94%E6%96%B0%E5%A5%B3%E9%AC%A5%E8%86%BD%E5%A4%A7 |website=香港蘋果日報 |accessdate=2019-04-18 |language=zh-HK |date=2015-10-12}}</ref>，19歲已經擔正做香港據影響力導演[[彭浩翔|彭浩翔]]監製電影《[[同班同學_(2015年電影)|同班同學]]》女主角<ref>{{Cite news|url=http://hk.on.cc/hk/bkn/cnt/entertainment/20151027/mobile/bkn-20151027221754176-1027_00862_001.html|title=郭奕芯模仿自己？溫碧霞：個Feel都似|work=on.cc東網|access-date=2018-07-23|language=zh-hk}}</ref>。該電影成為[[東京國際電影節|東京國際電影節]]開幕電影及亞洲未來組別競賽作品<ref>{{Cite news|url=http://hk.on.cc/hk/bkn/cnt/entertainment/20151022/mobile/bkn-20151022170850423-1022_00862_001_cn.html|title=东京国际电影节　郭奕芯“胸”赢苍井空|work=東網|access-date=2018-07-23|language=zh-hk}}</ref>、[[台北金馬影展|台北金馬影展]]奈派克（NETPAC）單元競賽作品、[[斯德哥爾摩電影節|斯德哥爾摩電影節]]競賽作品、波蘭華沙五味影展新亞洲電影單元競賽作品、香港亞洲電影投資會HAF獎，香港亞洲電影投資會[[愛奇藝|愛奇藝]]選擇獎。不過幾個月之後公司都未曾給她做任何工作。2016年簽約獨立經理公司三雅士製作繼續發展，2018年合約完滿後在2019年加盟[[英皇娛樂|英皇娛樂]]集團。

2017年，郭奕芯被[[夏雨|夏雨]]相中，擔任驚悚片《[[鬼網_(電影)|鬼網]]》女主角<ref>{{cite web |title=夏雨欽點 《同班》郭奕芯做女一|url=https://nextplus.nextmedia.com/article/2_395921_0 |website=壹週刊 |accessdate=2019-04-18 |language=zh-HK |date=2016-05-22}}</ref><ref>{{Cite web|url=https://hk.entertainment.appledaily.com/enews/realtime/article/20171019/57351768|title=【失聲&畫面全黑】郭奕芯拍鬼片遇靈異事件 成組人當冇事發生|website=Apple Daily 蘋果日報|access-date=2018-07-23}}</ref>，同年電影《[[同囚|同囚]]》上映，當中飾演香港演員[[關楚耀|關楚耀]]的老婆<ref>{{cite web |title=新片爆粗打人兼有床戲 關楚耀自爆險溺死台灣 - 香港文匯報 |url=http://paper.wenweipo.com/2017/05/04/EN1705040028.htm |website=文匯報 |accessdate=2019-04-18 |language=zh-HK |date=2017-05-04}}</ref>。同年在網絡劇《[[反黑|反黑]]》飾演[[柯受良|柯受良]]兒仔[[柯有倫|柯有倫]]女朋友，戲份頗多<ref>{{cite web |title=【反黑．多圖】伶伶簽Model公司墜騙案 郭奕芯當年都有個明星夢 |url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/123848/%E5%8F%8D%E9%BB%91-%E5%A4%9A%E5%9C%96-%E4%BC%B6%E4%BC%B6%E7%B0%BDmodel%E5%85%AC%E5%8F%B8%E5%A2%9C%E9%A8%99%E6%A1%88-%E9%83%AD%E5%A5%95%E8%8A%AF%E7%95%B6%E5%B9%B4%E9%83%BD%E6%9C%89%E5%80%8B%E6%98%8E%E6%98%9F%E5%A4%A2 |website=香港01 |accessdate=2019-04-18 |language=zh-HK |date=2017-10-05}}</ref>。2017年客串《[[春嬌救志明|春嬌救志明]]》，2018年初客串彭浩翔監製電影《[[是日公映|是日公映]]》，同拍攝《[[拳道|拳道]]》做女主角，拍檔係香港動作演員[[陳國坤|陳國坤]]<ref>{{Cite web|url=http://paper.wenweipo.com/2018/03/07/EN1803070022.htm|title=新片咀郭奕芯  陳國坤爆妻拒探班 - 香港文匯報|website=paper.wenweipo.com|language=zh-HK|access-date=2018-07-23}}</ref>。

2018年拍攝英皇電影《非分熟女》，同台灣影帝及視帝[[吳慷仁|吳慷仁]]有對手戲<ref>{{cite web|url=https://hk.mobi.yahoo.com/celebrity/%E9%9D%9E%E5%88%86%E7%86%9F%E5%A5%B3-%E9%83%AD%E5%A5%95%E8%8A%AF%E6%96%B0%E6%88%B2%E5%85%A8%E7%B4%A0%E9%A1%8F%E4%B8%8A%E9%99%A3-%E6%80%95%E5%9A%87%E5%A3%9E%E5%8F%B0%E7%81%A3%E8%A6%96%E5%B8%9D%E5%90%B3%E6%85%B7%E4%BB%81-090000481.html|title=【非分熟女】郭奕芯新戲全素顏上陣 怕嚇壞台灣視帝吳慷仁|website=hk.mobi.yahoo.com|language=zh-HK|access-date=2018-07-23}}</ref>。同年參與彭浩翔導演及《[[春嬌與志明|春嬌與志明]]》班底的2019年賀歲電影《[[恭喜八婆|恭喜八婆]]》<ref>{{cite web |title=【恭喜八婆】19歲全裸拍《同班同學》 郭奕芯獲邵音音大讚有潛質 |url=https://www.hk01.com/%E9%9B%BB%E5%BD%B1/291911/%E6%81%AD%E5%96%9C%E5%85%AB%E5%A9%86-19%E6%AD%B2%E5%85%A8%E8%A3%B8%E6%8B%8D-%E5%90%8C%E7%8F%AD%E5%90%8C%E5%AD%B8-%E9%83%AD%E5%A5%95%E8%8A%AF%E7%8D%B2%E9%82%B5%E9%9F%B3%E9%9F%B3%E5%A4%A7%E8%AE%9A%E6%9C%89%E6%BD%9B%E8%B3%AA |website=香港01 |accessdate=2019-04-18 |language=zh-HK |date=2019-02-05}}</ref>，之後在[[ViuTV|ViuTV]]電視劇《[[好人好姐|好人好姐]]》飾演JoJo。
