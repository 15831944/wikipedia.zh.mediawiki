'''郭志達'''（{{bd|1974年}}），台灣電影、廣告、音樂錄影帶美術指導<ref>{{Cite web |url = http://blog.swiio.com/post/54 |title =郭志達 全方位的視覺魔法師|date = 2017-07-24 }}</ref>。

==簡介==
畢業於台北[[復興商工|復興商工]]美工科，在就學期間專注於油畫創作。畢業後輾轉開始接觸音樂錄影帶的美術場景設計工作，從2000年開始與[[鄺盛|鄺盛]]導演合作一連串[[周杰倫|周杰倫]]、[[蔡依林|蔡依林]]、[[張惠妹|張惠妹]]、[[羅志祥|羅志祥]]<ref>{{Cite web |url = http://m.mtv.com.tw/#!/news/5243 |title =『狮子吼』首波主打舞曲《爱投罗网》MV邀来JEFF导演执导|date = 2013-10-24 }}</ref>、[[蕭亞軒|蕭亞軒]]、[[五月天|五月天]]等華語歌手的音樂錄影帶美術製作，奠定出鮮明的個人美術風格，並同時將觸角伸往廣告與電影製作,2007年首度參與電影代表作《[[不能說的秘密|不能說的秘密]]》。在商業空間設計領域也取得不錯的成績，其設計的 二十輪旅店大安館（SWIIO HOTEL DAAN）入選 金點設計獎 GOLDEN PIN DESIGN AWARD 的肯定.<ref>{{Cite web |url = http://stn.eslite.com/Article.aspx?id=284 |title =他們的小事，卻是我們重要的大事|date = 2009-09-16 }}</ref>。
