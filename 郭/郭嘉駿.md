'''郭嘉駿'''（{{lang-en|Denis Kwok}}；{{bd|1990年|1月14日|||吕}}），為《[[Good_Night_Show_全民造星|Good Night Show 全民造星]]》四十強。現為香港男子組合[[ERROR_(香港男子組合)|ERROR]]成員。

== 簡介 ==

* 郭嘉駿參加《[[Good_Night_Show_全民造星|Good Night Show 全民造星]]》前為一名時裝採購，參賽編號為26<ref>{{cite news |language = cn |author = Viutv|url= https://www.facebook.com/ViuTV.hk/photos/26-193%E9%83%AD%E5%98%89%E9%A7%BF%E5%B9%B4%E9%BD%A1-28%E8%81%B7%E6%A5%AD-%E6%99%82%E8%A3%9D%E6%8E%A1%E8%B3%BC%E6%93%85%E9%95%B7-%E5%94%B1%E6%AD%8C%E5%B0%8F%E7%B5%90%E4%BB%96/1994399927482085/ |title = 【《Goodnight Show全民造星》26號——193郭嘉駿】 |publisher = Viu TV|date = 2018-5-24 |accessdate = 2018-12-10}}</ref>。
* 《前傳》中首次亮相，因身高達193cm而受矚目。節目監製黃慧君（花姐）為他改了暱稱為193。後來他澄清身高為約192至193cm
* 於《[[Good_Night_Show_全民造星|Good Night Show 全民造星]]》40強止步、後加入組合[[ERROR_(香港男子組合)|ERROR]]正式出道

== 人物 ==
* 參加全民造星前曾報名無綫電視藝員訓練班。<ref>{{cite news |language = cn |author = |url= http://std.stheadline.com/daily/article/detail/1937845-%E5%A8%9B%E6%A8%82-ERROR%E8%B5%B0%E9%A8%8E%E5%91%A2%E8%B7%AF%E7%B7%9A%E7%8E%A9%E5%89%B5%E6%96%B0 |title = 【ERROR走騎呢路線玩創新】 |publisher = 《頭條日報》|date = 2019-1-1 |accessdate = 2019-1-1}}</ref>全民造星比賽前曾接受賽前特訓，學習唱歌
* 《全民造星》第30集組內對決階段，綵排時觸傷舊患、右手甩骹仍繼續負傷演出，因而衍生出[[ERROR_(香港男子組合)|ERROR]]歌曲《404》歌詞中「193 怎斷手」一段<ref>{{cite news |language = cn |author = |url= https://www.youtube.com/watch?v=O8_JLNATDXs |title = 【《ERROR 內戰！193 肥仔鬥掰一字馬！ 】 |publisher = U Magazine |date = 2018-12-20 |accessdate = 2018-12-20}}</ref>
* 大學主修心理學，自稱觀人於微，擅長模仿
* 害怕動物，因而被節目《花姐ERROR遊》安排到日本奈良餵野生鹿群，大受驚嚇<ref>{{cite news |language = cn |author = |url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/296789/%E8%8A%B1%E5%A7%90error%E9%81%8A-%E8%8A%B1%E5%A7%90%E7%95%B6%E7%9C%BE%E5%90%8C%E5%B0%8E%E6%BC%94%E9%98%BFmike%E7%82%92%E5%A4%A7%E9%91%8A-%E5%94%94%E6%95%A2%E6%94%BE%E6%89%8B%E7%95%80%E4%BD%A2%E5%89%B5%E4%BD%9C |title = 【花姐ERROR遊】花姐當眾同導演阿Mike炒大鑊：唔敢放手畀佢創作 |publisher = HK01 |date = 2019-02-19 |accessdate = 2019-02-19}}</ref><ref>{{cite news |language = cn |author = |url=https://www.youtube.com/watch?v=Az5STaeHi94&list=PLzIjX2ookwAiMer5VtyY6Xlw10tX9SxAc&index=32&t=0s |title = 【193奈良獻身餵鹿】 |publisher = Denis Kwok 193 Club
