'''郭锡章'''（{{bd|1933年|4月||catIdx=}}），[[湖北|湖北]][[武穴|武穴]]人，[[中国人民解放军中将|中国人民解放军中将]]。 

==生平==
郭锡章是武穴花桥镇刘常村西湖口垸人，1933年4月出生。早年曾在如今的武穴市梅川高级中学学习。后毕业于[[中共中央党校|中共中央党校]]。1951年7月参加[[中国人民解放军|中国人民解放军]]。1953年9月加入[[中国共产党|中国共产党]]。历任团司令部作训股参谋、股长，团副参谋长、参谋长、团长。1979年，出任副师长。1980年，任援外军事专家组长。1982年，任陆军第十二军三十六师师长。1983年，任陆军第十二军军长。1985年8月，任[[中国人民解放军陆军第十二集团军|陆军第十二集团军]]军长。1990年4月到1996年7月，任[[南京军区|南京军区]]副司令员。1988年10月被授予少将军衔。1993年7月晋升为中将军衔。他是第八届全国人大代表，第九届全国政协委员。[[中国共产党第十三次全国代表大会|中国共产党第十三次全国代表大会]]代表<ref>{{Cite web |url =http://www.hbwuxue.com/xwzx/rdxw/201611/t20161104_54418.shtml  |title =  将军郭锡章情系家乡|author =  |date = 2016-07-20 |publisher =武穴新闻网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url = http://www.wxmcgz.com/userlist/wxmcgz/newshow-5151.html|title = 南京军区中将副司令员郭锡章1998年4月视察母校|author = |date = 2013-05-22|publisher = 武穴市梅川高级中学|language = |accessdate = |quote = }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name=zj>{{Cite web |url = http://www.wxmcgz.com/userlist/wxmcgz/newshow-4552.html |title = 中将郭锡章 |author =  |date = 2012-06-26 |publisher = 武穴市梅川高级中学 |language =  |accessdate =  |quote =  |archive-url = https://web.archive.org/web/20161027035538/http://www.wxmcgz.com/userlist/wxmcgz/newshow-4552.html |archive-date = 2016-10-27 |dead-url = yes }}</ref>。
