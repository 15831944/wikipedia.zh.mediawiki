'''郭晓东'''（{{bd|1960年|11月||}}），[[内蒙古|内蒙古]][[赤峰|赤峰]]人，[[蒙古族|蒙古族]]，[[中国人民解放军少将|中国人民解放军少将]]，[[第十三届全国人民代表大会代表|第十三届全国人民代表大会代表]]<ref>{{cite news|title=全国人民代表大会|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=200813294162|accessdate=2018-05-29}}</ref>。

==生平==
郭晓东曾任[[中国人民解放军北京军区|中国人民解放军北京军区]]干部部副部长、[[中国人民解放军陆军第六十五集团军|中国人民解放军陆军第六十五集团军]]某师政治委员。2009年3月任中国人民解放军陆军第六十五集团军副政治委员。2015年7月接替到龄退役的[[李光聚|李光聚]]少将，升任[[中国人民解放军河北省军区|中国人民解放军河北省军区]]政治委员。2015年9月转任[[中国人民解放军济南军区|中国人民解放军济南军区]][[中国人民解放军陆军第二十六集团军|陆军第二十六集团军]]政治委员<ref>{{Cite web |url = http://china.caixin.com/2015-10-01/100860205.html |title = 郭晓东转任济南军区第26集团军政委 |author =  |date = 2015-10-01 |publisher =  财新网|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://mil.sohu.com/20151102/n424895859.shtml |title = 郭晓东已任26军政委 搭档最年轻军长张岩 |author =  |date =2015-11-02  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1354699  |title = 郭晓东少将接任河北省军区政委，60岁李光聚少将去职 |author =  |date =2015-07-21  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。2017年，调任新调整组建的[[中国人民解放军北部战区陆军|中国人民解放军北部战区陆军]][[中国人民解放军陆军第七十八集团军|第七十八集团军]]政治委员<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1734325 |title =  郭晓东少将已调任陆军第78集团军政委|author =  |date =2017-07-16  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。2018年11月，转任[[南部战区|南部战区]]政治工作部副主任<ref>{{cite web |author1=王俊 |title=第78集团军原政委郭晓东少将转任南部战区政治工作部副主任 |url=https://www.thepaper.cn/newsDetail_forward_2707712 |website=澎湃新闻 |accessdate=2019-02-02 |date=2018-12-05}}</ref>。
