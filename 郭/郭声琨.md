'''郭聲琨'''{{BD|1954年|10月|||catIdx=G郭}}，[[江西省|江西]][[兴国县|兴国]][[客家民系|客家人]]<ref>{{cite web |url = http://www.chinanews.com/tw/2010/07-05/2380015.shtml |title = 郭声琨彰化用客家话与农民拉家常 吃到幼时小吃 |accessdate=2017-09-30 |date=2010-07-05 |website=中新网 }}</ref>；[[江西理工大学|原江西冶金學院]][[采矿业|礦業]]系畢業，[[中南工业大学|中南工业大学]][[管理工程|管理工程]]系[[硕士|硕士]]，[[北京科技大学|北京科技大学]][[管理学|管理学]][[博士|博士]]，[[教授|教授]]級高級[[工程師|工程師]]（正高级[[职称|职称]]）。现任[[中共中央政治局|中共中央政治局]]委员、[[中国共产党中央书记处|书记处]]书记、[[中共中央政法委員會書記|中央政法委书记]]。

1974年12月加入[[中国共产党|中国共产党]]，是中共第十六、十七届[[中国共产党中央委员会候补委员|中央候补委员]]，第十八届、十九届中央委员，第十九届[[中央政治局|中央政治局]]委员、[[中国共产党中央书记处|中央书记处]]书记。曾参与创办[[中國鋁業|中國鋁業公司]]，任首任[[总经理|总经理]]；后任[[中国共产党广西壮族自治区委员会|中共廣西壯族自治區黨委]]書記、[[广西壮族自治区人大常委会|广西壮族自治区人大常委会]]主任<ref>{{cite news |url = http://www.nownews.com/2012/04/28/91-2808986.htm |title = 深化桂台經貿合作　郭聲琨、馬飆南寧會見王志剛 |date = 2012-04-28 |website = [[今日新聞網|今日新聞網]] |accessdate = 2012-06-16 |language = zh-Hant }}</ref>。

==早年经历==
1973年8月，18岁的郭声琨自[[兴国县平川中学|兴国县平川中学]]毕业后，响应[[中国共产党中央委员会|中共中央]]号召，加入[[知识青年|知識青年]][[上山下乡运动|上山下乡]]的运动中，在兴国县[[五里亭街道|五里亭公社]]插队劳动。1974年12月，加入中国共产党。1977年3月，获推荐进入[[江西理工大学|江西冶金学院]](今[[江西理工大学|江西理工大学]]）矿业系选矿专业学习，成为[[工农兵|“工农兵”学员]]。三年学满后，1979年12月被分到位于兴国县境内、[[中华人民共和国冶金工业部|冶金工业部]]下属的[[画眉坳钨矿|画眉坳钨矿]]工作，历任机选车间技术员、工段长、党支部书记，车间生产副主任，车间行政副主任、车间主任。1985年8月，出任[[中国有色金属工业总公司|中国有色金属工业总公司]]画眉坳钨矿矿长。

1990年5月起，郭声琨参与筹建[[贵溪市|贵溪]]银矿{{NoteTag|[[贵溪冷水坑银矿|贵溪冷水坑银矿]]位于江西省鹰潭市[[贵溪市|贵溪县]][[冷水镇_(贵溪市)|冷水镇]]银炉岭，矿区南北长7km，东西宽5km，面积35km<sup>2</sup>，该矿区的发现与勘探经历了从“脉带型”铅锌矿、斑岩型铅锌矿、斑岩型银矿、层控叠生型银矿和金矿等几个相互交织的阶段。经过20多年的工作，终于探明了该大型银矿。1990年兴建了冷水坑矿区的第一座矿山——银路岭银矿，1992年正式投产，日处理矿石500t,年产白银22.96t,价值上亿元。保有银储量3336t，含银品位为146.21g/t。}}，兼任贵溪银矿建设指挥部总指挥、党委书记。1992年11月，出任中国有色金属工业总公司贵溪银矿矿长、党委书记。1993年5月，出任[[中国有色金属工业总公司|中国有色金属工业总公司]]南昌公司经理、党组书记。期间，入选1995年9月开学的[[中共中央党校|中共中央党校]]一年制中青年幹部培訓班。1994年9月到1996年5月，在[[中南工业大学|中南工业大学]]管理工程系管理工程专业研究生班课程在职学习，获得[[硕士|硕士]]学位。

==执掌国企==
在基层冶金、及有色金属行业工作了二十年的郭声琨，于1997年6月调到[[北京市|北京]]工作，出任[[中国有色金属工业总公司|中国有色金属工业总公司]]副总经理、党组成员。1998年3月，公司改组为由[[中华人民共和国国家经济贸易委员会|国家经贸委]]管理的[[国家有色金属工业局|国家有色金属工业局]]，郭声琨任局党组成员。1999年2月，任国家有色金属工业局副局长、党组成员。

2000年6月，46岁的郭声琨任[[国务院国有重点大型企业监事会|国务院国有重点大型企业监事会]]主席，晋升副部级。2000年8月，任国务院国有重点大型企业监事会主席、[[中國鋁業|中国铝业公司]]筹备组组长。2001年2月，出任中国铝业公司首任总经理、党组书记。2001年9月起，任中国铝业公司总经理、党组书记兼[[中國鋁業|中国铝业股份有限公司]]董事长、总裁。其间，于2002年3月至2002年5月在[[中央党校省部级干部进修班|中央党校省部级干部进修班]]学习。2002年11月，郭声琨当选[[中国共产党第十六届中央委员会|中共第十六届]][[中国共产党第十六届中央委员会候补委员列表|中央候补委员]]。

==从政之路==
2004年4月，拥有大型国企管理经验的郭声琨，调任[[中国共产党广西壮族自治区委员会|中共广西壮族自治区党委]]副书记、[[广西壮族自治区人民政府|自治区人民政府]]副主席、自治区人民政府党组副书记。期间，于2003年9月至2007年7月在[[北京科技大学|北京科技大学]]经管学院管理科学与工程专业博士研究生在职学习，并获[[管理学|管理学]][[博士|博士]]学位。

2007年11月30日，53岁的郭声琨升任广西自治区党委书记。2008年1月26日起，兼任广西壮族自治区人大常委会主任、党组书记<ref>{{Cite web|url=http://politics.people.com.cn/GB/41223/6826193.html|title=郭声琨当选广西人大常委会主任 马飚当选广西主席|accessdate=2017-09-30|author=|date=2008-01-26|work=|publisher=人民网}}</ref>。

2012年12月19日，郭声琨离开广西，任[[中华人民共和国公安部|公安部]]党委书记，在12月28日召开的[[第十一届全国人民代表大会常务委员会|第十一届全国人大常委会]]第三十次会议上接任公安部部长，同时兼任[[中国人民武装警察部队|武警部队]]第一政委。2013年3月16日，在[[第十二届全国人民代表大会|第十二届全国人大第一次会议]]上当选[[国务委员|国务委员]]<ref>{{Cite web|url=http://news.xinhuanet.com/2013lh/2013-03/16/c_124466958.htm|title=杨晶、常万全、杨洁篪、郭声琨、王勇为国务委员|accessdate=2017-09-30|author=|date=2013-03-16|work=|publisher=新华网}}</ref>。同年4月，出任[[中共中央政法委员会|中共中央政法委]]副书记。

2017年10月，63岁的郭声琨在[[中国共产党第十九次全国代表大会|中共十九大]]上当选为[[中国共产党第十九届中央委员会委员列表|十九届中央委员]]，并且在10月25日召开的[[中国共产党第十九届中央委员会第一次全体会议|十九届一中全会]]上当选为[[中国共产党中央政治局|中央政治局委员]]、[[中国共产党中央书记处|中央书记处书记]]，数日后出任[[中共中央政法委员会|中共中央政法委]][[中共中央政法委員會書記|书记]]<ref>{{Cite web|url=http://news.xinhuanet.com/2017-10/31/c_1121885950.htm|title=中央政法委书记郭声琨：以习近平新时代中国特色社会主义思想为指导 奋力开创新时代政法事业新局面|accessdate=2017-10-31|author=|date=2017-10-31|work=|publisher=新华网}}</ref>。2017年11月，不再担任公安部部长、党委书记职务<ref>{{Cite web|url=http://www.thepaper.cn/newsDetail_forward_1850179|title=全国人大常委会决定任命赵克志为公安部部长|accessdate=2017-11-04|author=|date=2017-11-04|work=|publisher=澎湃新闻}}</ref><ref>{{Cite web|url=http://news.163.com/17/1101/08/D254CQ5F0001899O.html|title=赵克志已出任公安部党委书记|accessdate=2017-11-04|author=|date=2017-11-01|work=|publisher=网易}}</ref>。2018年，兼任新成立的[[中央全面依法治国委员会|中央全面依法治国委员会]]办公室主任<ref>{{cite web |title=中央全面依法治国委员会办公室组成人员公开 郭声琨任主任 |url=https://news.sina.com.cn/c/2018-09-09/doc-ihivtsyk4235263.shtml |website=新浪 |accessdate=2018-09-09}}</ref>。
