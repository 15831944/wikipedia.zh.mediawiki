'''郭子豪'''（{{lang|en|'''Arnold Kwok Tsz Ho'''}}，{{bd|1991年|7月9日}}），[[香港|香港]]男演員及模特兒，現為[[無綫電視藝員列表|無綫電視經理人合約]]藝員<ref>{{cite web|url=http://artiste.tvb.com/main/artistedetail/arnoldkwok |language= |title=郭子豪 Arnold Kwok - TVB藝人資料 |publisher=tvb.com |date= |accessdate=2015-09-02}}</ref>。

== 簡介 ==
郭子豪生於富有家庭，父親經營時裝貿易生意<ref name=AK>{{cite web|url=https://www.msn.com/zh-hk/entertainment/news/%E3%80%90%E8%B7%B3%E8%BA%8D%E7%94%9F%E5%91%BD%E7%B7%9A%E3%80%91%E9%83%AD%E5%AD%90%E8%B1%AApackage%E9%9D%9A%E5%88%B0%E5%86%87%E6%9C%8B%E5%8F%8B%EF%BC%81%E5%A4%A7%E5%8F%B015%E4%BD%8D%E5%AF%8C%E8%B2%B4%E5%85%B5%E5%9C%98%E9%80%90%E5%80%8B%E6%8D%89/ar-BBP6HUW |language= |title=【跳躍生命線】郭子豪 package 靚到冇朋友！大台15位富貴兵團逐個捉 |publisher=[[東方新地|東方新地]] |date=2018-10-30 |accessdate=2018-11-07}}</ref>。他曾就讀[[喇沙書院|喇沙書院]]，並往[[英國|英國]][[布萊頓公學|布萊頓公學]]繼續唸書，畢業於[[倫敦帝國學院|倫敦帝國學院]][[機械工程|機械工程]]系，2012年回港到[[保時捷|保時捷]]當實習生時獲美國服裝品牌「[[Abercrombie_&_Fitch|Abercrombie & Fitch]]（A&F）」相中，擔任香港旗艦店與全球店鋪開幕禮的模特兒<ref name=AK/>，是首位及唯一一位華人從事此工作<ref>{{cite web|url=http://orientaldaily.on.cc/cnt/entertainment/20121230/00282_064.html |language= |title=郭子豪裸出野心升首席華模 |publisher=[[東方日報_(香港)|東方日報]] |date=2012-12-30 |accessdate=2015-09-02}}</ref>。他其後成立獨立製作公司，任微電影、廣告、音樂錄像導演和編劇<ref name=AK/>；2014年底選擇加入無綫電視，翌年則修讀[[無綫電視藝員訓練班|第28期無綫電視藝員訓練班]]，現時主力拍攝劇集、經營自家服裝品牌<ref>{{cite web|url=http://www.bastillepost.com/hongkong/4-%E5%A8%9B%E5%9C%88%E4%BA%8B/1829887-%E9%83%AD%E5%AD%90%E8%B1%AA%E6%9C%88%E5%84%B22-5%E8%90%AC-%E6%9C%9B30%E6%AD%B2%E8%B2%B7%E6%A8%93%E7%B5%90%E5%A9%9A |language= |title=郭子豪月儲2.5萬 望30歲買樓結婚 |publisher=[[巴士的報|巴士的報]] |date=2017-02-23 |accessdate=2017-02-23}}</ref>，以及任職 [[MegaBox|MegaBox]] [[Goji_Studios|Goji Studios]] 分店的私人健身教練。

2017年，他於該台劇集《[[親親我好媽|親親我好媽]]》裡飾演籃球王子「連國佳（Hugo）」一角而為人認識<ref>{{cite web|url=http://www.nmplus.hk/enterainment/%e9%83%ad%e5%ad%90%e8%b1%aa-%e4%b8%81%e5%ad%90%e6%9c%97/?utm_source=rss&utm_medium=rss&utm_campaign=%25e9%2583%25ad%25e5%25ad%2590%25e8%25b1%25aa-%25e4%25b8%2581%25e5%25ad%2590%25e6%259c%2597 |language= |title=TVB新晉 小鮮肉 ！郭子豪、丁子朗逐個看 |publisher=[[新Monday|新Monday]] |date=2017-04-05 |accessdate=2017-04-05}}</ref><ref>{{cite web|url=https://www.hk01.com/%E5%A8%9B%E6%A8%82/70848/-%E8%A6%AA%E8%A6%AA%E6%88%91%E5%A5%BD%E5%AA%BD-%E8%82%8C%E7%94%B7%E9%83%AD%E5%AD%90%E8%B1%AA%E4%BF%82A-F%E5%BE%A1%E7%94%A8%E7%94%B7%E6%A8%A1-%E8%90%AC%E4%BA%BA%E8%BF%B7%E6%A3%84%E5%B7%A5%E7%A8%8B%E5%B8%AB%E7%82%BA%E6%8B%8D%E5%8A%87 |language= |title=【親親我好媽】肌男郭子豪係A&F御用男模　萬人迷棄工程師為拍劇 |publisher=[[香港01|香港01]] |date=2017-02-15 |accessdate=2017-02-15}}</ref>。2018年，他參加《2018 [[PCA_PRO_及香港健美及運動體適能錦標賽|PCA PRO 及香港健美及運動體適能錦標賽]]》，且奪得「第一屆華南地區男子運動模特組別冠軍」<ref>{{cite web|url=http://hk.on.cc/hk/bkn/cnt/entertainment/20180825/bkn-20180825220344544-0825_00862_001.html |language= |title=健美比賽奪冠　郭子豪操肌日飲16公升水 |publisher=[[ON.CC|東網]] |date=2018-08-25 |accessdate=2018-08-26}}</ref>，亦成為首個在健美比賽中獲取冠軍的香港藝人<ref>{{cite web|url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/227386/%E9%A6%96%E4%BD%8D%E9%A6%99%E6%B8%AF%E8%97%9D%E4%BA%BA%E5%A5%AA%E5%81%A5%E7%BE%8E%E5%86%A0%E8%BB%8D-%E9%83%AD%E5%AD%90%E8%B1%AA%E6%84%9F%E8%AC%9D%E6%95%99%E7%B7%B4-%E5%A5%B3%E5%8F%8B%E6%9B%BE%E6%87%B7%E7%96%91%E6%88%91%E5%93%8B%E5%98%85%E9%97%9C%E4%BF%82 |language= |title=首位香港藝人奪健美冠軍　郭子豪感謝教練：女友曾懷疑我哋嘅關係 |publisher=香港01 |date=2018-08-26 |accessdate=2018-08-27}}</ref>。同年，他又於劇集《[[跳躍生命線|跳躍生命線]]》內飾演救護員「陸昊然（六號）」，演技獲不少網民讚賞<ref>{{cite web|url=https://holiday.presslogic.com/2018/11/06/article/94521 |language= |title=《跳躍生命線》郭子豪獲網民大讚演技有進步！執雀巢一幕超催淚 網友：睇到心翳 |publisher=[[假期日常|假期日常]] |date=2018-11-06 |accessdate=2018-11-07}}</ref>。

2019年3月27日，郭子豪在[[尖沙咀|尖沙咀]][[麼地道|麼地道]]近[[梳士巴利道|梳士巴利道]]交界駕駛私家車時遇到警察路障，被截停要求進行酒精呼氣測試時錄得約50度，較法定上限大概超標一倍，故被警方以涉嫌[[酒後駕駛|酒後駕駛]]罪名拘捕，期間也報稱不適，需由救護車送往[[伊利沙伯醫院|伊利沙伯醫院]]<ref>{{cite web |title=無綫藝員郭子豪 尖沙嘴遇警路障揭涉嫌酒駕被捕 |url=https://hk.news.appledaily.com/breaking/realtime/article/20190327/59416429 |publisher=[[蘋果日報_(香港)|蘋果日報]] |date=2019-03-27 |accessdate=2019-03-27}}</ref><ref>{{cite web |title=TVB 男藝員郭子豪尖沙嘴遇路障 涉醉駕被捕 |url=http://hd.stheadline.com/news/realtime/hk/1466067/ |publisher=[[頭條日報|頭條日報]] |date=2019-03-27 |accessdate=2019-03-27}}</ref>。但他事後曾向傳媒否認相關指控和自稱當時身在家中，整晚沒有駕駛出外<ref>{{cite web |title=郭子豪涉酒駕被捕　無綫：聯絡唔到佢！ |url=https://hk.on.cc/hk/bkn/cnt/entertainment/20190327/bkn-20190327032042485-0327_00862_001.html |publisher=東網 |date=2019-03-27 |accessdate=2019-03-27}}</ref>，至同日下午才承認責任且公開致歉<ref>{{cite web |title=【承諾注意安全駕駛】涉醉駕被捕 郭子豪道歉：承擔責任 |url=https://ol.mingpao.com/ldy/showbiz/latest/20190327/1553675356149/%E3%80%90%E6%89%BF%E8%AB%BE%E6%B3%A8%E6%84%8F%E5%AE%89%E5%85%A8%E9%A7%95%E9%A7%9B%E3%80%91%E6%B6%89%E9%86%89%E9%A7%95%E8%A2%AB%E6%8D%95-%E9%83%AD%E5%AD%90%E8%B1%AA%E9%81%93%E6%AD%89-%E6%89%BF%E6%93%94%E8%B2%AC%E4%BB%BB |publisher=[[明報|明報 OL 網]] |date=2019-03-27 |accessdate=2019-03-27}}</ref>。
