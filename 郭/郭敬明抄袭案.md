'''郭敬明抄袭事件'''，是指2003年至2006年的[[庄羽|庄羽]]控诉[[郭敬明|郭敬明]]抄袭案，北京法院判决郭敬明小说《梦里花落知多少》抄袭[[庄羽|庄羽]]小说《圈里圈外》成立。郭敬明抄袭案入选2006年北京十大知识产权典型案例<ref>{{Cite news|url=http://news.163.com/06/1207/02/31N3QHB000011229.html|title=郭敬明抄袭案入选北京十大知识产权典型案例|last=|first=|date=2006-12-07|work=竞报|accessdate=}}</ref>，是中国最著名的知识产权官司之一。

== 事件背景 ==
[[郭敬明|郭敬明]]是新概念作文大赛一等奖得主，因2003年出版长篇小说《幻城》走红<ref>{{Cite news|url=http://www.hb.xinhuanet.com/newscenter/2004-04/08/content_1937511.htm|title=17岁写出畅销书《幻城》郭敬明将做客金报|last=|first=|date=|work=|accessdate=2004年04-08|deadurl=yes|archiveurl=https://web.archive.org/web/20040502043313/http://www.hb.xinhuanet.com/newscenter/2004-04/08/content_1937511.htm|archivedate=2004-05-02}}</ref>。[[郭敬明|郭敬明]]和[[韩寒|韩寒]]两人被认为是少年作家中最受瞩目的人物，深受青少年欢迎<ref>{{Cite news|url=http://news.ifeng.com/a/20160927/50033378_0.shtml|title=新概念作文大赛18年：除了韩寒郭敬明 其他人都在哪|last=|first=|date=|work=|accessdate=2016年9月27日}}</ref>。

== 事件经过 ==
2003年起，[[郭敬明|郭敬明]]在[[萌芽|《萌芽》]]杂志上开始连载长篇小说《梦里花落知多少》，受到读者的欢迎<ref>{{Cite news|url=http://www.china.com.cn/chinese/RS/435038.htm|title=郭敬明新作《梦里花落知多少》将上市|last=|first=|date=|work=|accessdate=2003年11月4日}}</ref>。当年11月，《梦里花落知多少》小说由春风文艺出版社出版，不到一月时间已销售60万册<ref>{{Cite news|url=http://www.southcn.com/edu/newbook/shuping/200312110113.htm|title=郭敬明为新作《梦里花落知多少》 签名售书|last=|first=|date=2003-12-11|work=|accessdate=}}</ref>。

但部分读者发现，[[郭敬明|郭敬明]]的这部《梦里花落知多少》，在人物性格和语言上，与女作家[[庄羽|庄羽]]在2003年1月出版的小说《圈里圈外》相当雷同，质疑郭敬明抄袭庄羽。庄羽得知这一情况后，开始请律师维权，但郭敬明和春风文艺出版社则否认抄袭一事<ref>{{Cite news|url=http://news.sina.com.cn/c/2003-12-11/05341310753s.shtml|title=少年作家面临诉讼 《梦里花落知多少》被指剽窃|last=|first=|date=|work=|accessdate=2003年12月11日}}</ref>。

2003年12月[[庄羽|庄羽]]将郭敬明、出版发行单位春风文艺出版社及销售商北京图书大厦诉上法庭，请求法院判决三被告立即停止侵害，赔礼道歉，并索赔经济损失50万元等。

2004年12月，北京市第一中级人民法院作出一审判决，认定郭敬明所著《梦里花落知多少》侵犯了庄羽的著作权，判令被告郭敬明、春风文艺出版社立即停止《梦里花落知多少》一书的出版发行，共同赔偿原告庄羽经济损失20万元，被告北京图书大厦有限责任公司停止销售《梦里花落知多少》一书。但郭敬明方不服判决，提出上诉<ref>{{Cite web|url=http://news.163.com/41207/6/170UFELI0001124J.html|title=少年作家郭敬明被判剽窃另一少年作家作品|accessdate=2004-12-07|author=|date=2004-12-07|publisher=}}</ref>。

2006年5月，北京市高级人民法院作出终审判决，维持北京市第一中级人民法院的一审判决，认定郭敬明所著《梦里花落知多少》对庄羽的《圈里圈外》整体上构成抄袭，判决郭敬明与春风文艺出版社赔偿庄羽经济损失20万元，春风文艺出版社与北京图书大厦停止《梦里花落知多少》的出版、销售行为。法院还认定郭敬明的抄袭与出版行为给庄羽造成了严重的精神损失，判决其赔偿庄羽精神抚慰金1万元。此外，还要求郭敬明与出版社在15日内在《中国青年报》上公开道歉<ref>{{Cite news|url=http://old.chinacourt.org/public/detail.php?id=208706|title=郭敬明终审被判抄袭 判赔经济损失21万元|last=|first=|date=|work=|accessdate=2006-05-23|archive-url=https://web.archive.org/web/20161220164955/http://old.chinacourt.org/public/detail.php?id=208706|archive-date=2016-12-20|dead-url=yes}}</ref>。

事后，郭敬明通过经纪人将赔款支付给庄羽，但拒绝道歉<ref>{{Cite news|url=http://ent.news.cn/2006-12/14/content_5485197.htm|title=与庄羽“纠葛”难断 郭敬明重申“拒不道歉”|last=|first=|date=|work=|accessdate=2006年12月14日|deadurl=yes|archiveurl=https://web.archive.org/web/20161220104720/http://ent.news.cn/2006-12/14/content_5485197.htm|archivedate=2016年12月20日}}</ref>。庄羽曾考虑申请法院强制执行<ref>{{Cite news|url=http://ent.sina.com.cn/s/m/2006-06-07/10371113718.html|title=郭敬明只赔钱不认错 庄羽考虑申请法院强制执行|last=徐|first=颖|date=2006年6月7日|work=新闻晨报|accessdate=}}</ref>，但郭敬明至今没有道歉。

== 事件后续 ==
郭敬明2003年出版的长篇小说《幻城》就曾被指抄袭日本著名漫画《圣传》<ref>{{Cite news|url=http://yule.sohu.com/2003/12/01/62/article216346239.shtml|title=郭敬明小说《幻城》被指抄袭日本漫画《圣传》|last=|first=|date=2003-12-01|work=东方网|accessdate=}}</ref>，受庄羽案的影响，《幻城》抄袭争议再次受到关注。

后来郭敬明出的几部长篇小说，无一不受到抄袭质疑。《夏至未至》被指抄袭日本漫画《NANA》<ref>{{Cite news|url=http://ent.163.com/06/0622/15/2K7VOFQ900031H2L.html|title=郭敬明《夏至未至》被指抄袭日本漫画《NANA》|last=|first=|date=2006-06-22|work=中新网|accessdate=}}</ref>，《悲伤逆流成河》被指抄袭落落的小说《年华是无效信》<ref>{{Cite news|url=http://www.china.com.cn/book/txt/2007-03/27/content_8016536.htm|title=郭敬明《悲伤逆流成河》还未出又被指抄袭|last=|first=|date=2007-03-27|work=青年报|accessdate=}}</ref>，《爵迹》被指抄袭日本漫画《Fate》<ref>{{Cite news|url=http://xian.qq.com/a/20100903/000025.htm|title=出版社否认郭敬明新书《爵迹》涉嫌抄袭日本|last=肖|first=琳|date=2010年9月3日|work=阳光报|accessdate=}}</ref>，《小时代》被抄袭美国电影《[[穿著Prada的惡魔_(電影)|穿Prada的女王]]》<ref>{{Cite news|url=http://news.sohu.com/20080928/n259797452.shtml|title=《小时代》被疑抄袭 郭敬明新书再遭信用危机|last=|first=|date=2008年9月28日|work=中国网|accessdate=}}</ref>。因为抄袭事件的影响，2007年郭敬明进入中国作家协会时，被作协部分成员和公众激烈反对<ref>{{Cite news|url=http://view.news.qq.com/a/20070927/000038.htm|title=郭敬明入作协惹口水 反对者：是对作协的侮辱|last=谢|first=正宜|date=2007年9月26日|work=四川在线－天府早报|accessdate=}}</ref>。

郭敬明的部分粉丝因此对[[庄羽|庄羽]]产生仇视情绪，在社交媒体上多次攻击庄羽<ref>{{Cite news|url=http://www.ce.cn/books/main/author/200607/03/t20060703_7591406.shtml|title=粉丝出言不逊庄羽希望读者辨别是非|last=赵|first=明宇|date=2006年7月3日|work=北京娱乐信报|accessdate=|deadurl=yes|archiveurl=https://web.archive.org/web/20161220144224/http://www.ce.cn/books/main/author/200607/03/t20060703_7591406.shtml|archivedate=2016年12月20日}}</ref><ref>{{Cite news|url=http://news.sohu.com/20060623/n243896343.shtml|title=庄羽 赢了官司多了困惑|last=张|first=静|date=2006年6月23日|work=西安晚报|accessdate=}}</ref>。韩寒在2006年在博客以“狗护主”批判郭敬明粉丝的行为<ref>{{Cite news|url=http://ent.newssc.org/system/2006/07/18/010058876.shtml|title=韩寒博客大骂郭敬明粉丝是“狗护主”|last=|first=|date=2006-07-18|work=四川新闻网|accessdate=}}</ref>。庄羽在2010年在博客上发表《写给郭敬明粉丝的一些话》回应郭敬明粉丝的攻击<ref>{{Cite news|url=http://blog.sina.com.cn/s/blog_4701faca0100l7xr.html|title=写给郭敬明粉丝的一些话|last=庄|first=羽|date=2010-08-21|work=新浪博客|accessdate=}}</ref>。

2014年琼瑶诉于正抄袭案胜利后，[[庄羽|庄羽]]通过微博发布博文《由琼瑶女士胜诉想到的<ref>{{Cite web|url=http://weibo.com/p/1001603793093643088073?from=page_103505_profile&wvr=6&mod=wenzhangmod|title=由琼瑶女士胜诉想到的|accessdate=|author=庄羽|date=2014年12月29日|publisher=}}</ref>》，回顾自己诉郭敬明官司的细节。
