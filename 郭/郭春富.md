'''郭春富'''（{{bd|1958年|6月||}}），[[山东|山东]][[广饶|广饶]]人，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
郭春富长期在[[中国人民解放军|中国人民解放军]]后勤系统工作，曾任[[中央军委后勤保障部财务局|中国人民解放军总后勤部财务部]]事业财务局局长等职，2011年前后，调入当时挂靠在[[中国人民解放军总后勤部|中国人民解放军总后勤部]]的[[中央军委审计署|中国人民解放军审计署]]任副审计长。2014年底，中国人民解放军审计署由总后勤部划归中央军委建制。2015年4月14日前，郭春富调任[[中国人民解放军国防大学联合勤务学院|中国人民解放军后勤学院]]院长。2015年11月，[[中央军委后勤保障部办公厅|中国人民解放军总后勤部司令部]]副参谋长[[谢维宽|谢维宽]]少将接替郭春富任[[中国人民解放军国防大学联合勤务学院|中国人民解放军后勤学院]]院长。2015年12月，郭春富履新中国人民解放军审计署审计长。2016年1月，中央军委机关改为十五个职能部门，中国人民解放军审计署撤销，组建[[中央军委审计署|中央军委审计署]]，郭春富继续任中央军委审计署审计长。同时郭春富兼任[[中央军委纪律检查委员会|中央军委纪律检查委员会]]委员<ref>{{Cite web |url = http://politics.people.com.cn/n1/2016/0112/c70731-28039781.html |title =中央军委机关改为十五个职能部门  |author =  |date =2016-01-12  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://finance.ifeng.com/a/20170216/15201682_0.shtml  |title = 中央军委审计署审计长郭春富少将兼任军委纪委委员 |author =  |date = 2017-02-16  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://news.ifeng.com/a/20151230/46893701_0.shtml  |title = 解放军后勤学院院长郭春富任解放军审计署审计长 |author =  |date =2015-12-30  |publisher = 凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://news.qq.com/a/20151230/045549.htm  |title = 郭春富任解放军审计署审计长 年内两度履新 |author =  |date = 2015-12-30 |publisher =  腾讯|language =  |accessdate =  |quote =  }}</ref>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。
