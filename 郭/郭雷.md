'''郭雷'''（{{bd|1961年|11月1日||}}）生于[[山东|山东]][[淄博|淄博]]，[[中华人民共和国|中华人民共和国]]控制科学家<ref name=amss>{{Cite web |url = http://lsc.amss.ac.cn/~lguo/index-c.htm |title = 郭雷 |author =  |date =  |publisher =中国科学院系统控制重点实验室  |language =  |accessdate = 2017-03-05 |quote =  }}</ref>。

==生平==
1982年，自[[山东大学|山东大学]]数学系毕业，获得学士学位。1984年，自[[中国科学院|中国科学院]]系统科学研究所毕业，获得硕士学位。1987年，自中国科学院系统科学研究所毕业，获得博士学位<ref name=amss/>。

1987年6月至1989年5月，在[[澳大利亚国立大学|澳大利亚国立大学]]做博士后。1990年2月至1992年1月，任中国科学院系统科学研究所副研究员。1992年2月起，任中国科学院系统科学研究所研究员。1994年11月至1998年10月，任中国科学院系统科学研究所系统与控制重点实验室副主任。1995年1月至1998年11月，任中国科学院系统科学研究所学术委员会副主任。1998年11月至2002年12月，任中国科学院系统与控制重点实验室主任。1998年12月至2002年12月，任中国科学院系统科学研究所所长。2002年12月至2012年4月，任中国科学院数学与系统科学研究院院长。2010年12月起，任中国科学院国家数学与交叉科学中心主任<ref name=amss/>。2001年当选为中国科学院院士<ref>{{Cite web|url =http://sourcedb.cas.cn/sourcedb_ad_cas/zw2/ysxx/xxjskxb/|title =中国科学院学部与院士·院士信息·信息技术科学部·郭雷|author =|date =|publisher =|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20120425223835/http://sourcedb.cas.cn/sourcedb_ad_cas/zw2/ysxx/xxjskxb/|archivedate =2012-04-25}}</ref>。

他是[[全国青联|全国青联]]第八届、九届常委，2005年出任第十届副主席。1998年至2007年，任第九届、十届全国政协委员。2008年，任第十一届全国人大常委会委员、[[全国人大外事委员会|全国人大外事委员会]]副主任委员<ref>{{cite news|title=全国人大代表信息-郭雷|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=110449|accessdate=2013-04-05|newspaper=全国人大网}}</ref>。2013年，任第十二届全国人大常委会委员、全国人大外事委员会副主任委员。2014年2月27日，第十二届全国人民代表大会常务委员会第七次会议通过，决定任命郭雷为第十二届全国人民代表大会常务委员会副秘书长<ref name=zhangguo>{{Cite web |url = http://hn.ifeng.com/hnzhuanti/hnztc/zuixin/detail_2014_02/28/1915798_0.shtml |title =郭雷任全国人大常委会副秘书长  |author =  |date =2014-02-28  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。
