'''郭宜品'''{{BD|1962年|9月|||catIdx=G}}[[河南人|河南人]]，毕业于[[河南农业大学|河南农业大学]]农经系农业经济管理专业，现任[[河南省|河南省]][[洛阳市|洛阳市人民政府副市长]]，2014年9月，郭宜品神秘失踪，同时失去联系的还有当地房地产开发商，而郭宜品的子女早就移民外国，妻子也不见踪影。郭宜品在[[伊川县|伊川县]]主政期间，大搞[[房地产|房地产]]开发，调任到洛阳市之后，继续勾结房地产商牟取暴利，已经曝出的资金数额是500万人民币，而妻子和子女移民外国，是中共典型的[[裸官|裸官]]之一。<ref name="bull" >{{cite web |url=http://news.163.com/14/0922/02/A6NBKSGD00014AED.html |title= 洛阳失联副市长孩子出国妻子失踪|accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref><ref name="dog" >{{cite web |url=http://www.bjnews.com.cn/news/2014/09/19/334513.html |title=涉嫌受贿500万 失联洛阳副市长郭宜品被网上追逃 |accessdate=2014 |publisher=新京报 |work= |language=zh-hans}}</ref><ref name=" bee" >{{cite web |url=http://news.qq.com/a/20140916/040396.htm |title=官方确认洛阳市副市长郭宜品失联 公安介入排查 |accessdate= 2014|publisher=腾讯网 |work= |language=zh-hans}}</ref>

== 生平 ==
1962年，郭宜品出生在[[河南省|河南省]][[洛宁县|洛宁县]]。

1982年9月，郭宜品参加政治工作，1983年5月，他加入了[[中国共产党|中国共产党]]。

1981年9月到1998年3月，郭宜品一直在洛宁县工作，担任过团委干部、党委书记等职务。

1998年3月到2003年3月，郭宜品升职为河南省[[栾川县|栾川县]]副县长。

2003年3月到2005年7月，郭宜品担任栾川县委常委、组织部长。

2005年7月到2006年5月，郭宜品升职为栾川县委副书记、县纪委书记。2006年5月，郭宜品连任河南省栾川县委副书记，2007年11月，他第三次出任栾川县委副书记。

2007年12月到2009年3月，郭宜品调到河南省[[伊川县|伊川县]]，担任县委副书记、县长，2009年3月，郭宜品被擢升为县委书记。

2014年2月，郭宜品再度被擢升为[[洛阳市|洛阳市人民政府副市长]]，同时连任伊川县委书记。4月，郭宜品担任洛阳市副市长。9月15日，[[网易|网易新闻]]半夜发布新闻稿件称郭宜品神秘失踪，之后各大新闻媒体转载，当地官员和群众则称郭宜品在8月份就已经失去踪影，官员称郭宜品向市长[[李柳身|李柳身]]请假，理由是带母亲去[[北京市|北京市]]看病，其子女早已经移民外国，妻子也办理了[[身份证|假身份证]]，不见踪影。<ref name="fish" >{{cite web |url=http://news.163.com/14/0916/11/A68TODS100014JB6.html#f=dlist |title= 官方确认洛阳市副市长郭宜品失联 公安介入排查|accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref><ref name="cat" >{{cite web |url=http://news.ifeng.com/a/20140916/41991830_0.shtml |title= 洛阳副市长失联前曾向市长请假 称要带母亲去北京看病|accessdate=2014 |publisher=凤凰网 |work= |language=zh-hans}}</ref>
