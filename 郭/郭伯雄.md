'''郭伯雄'''（{{bd|1942年|7月|catIdx=G郭}}），[[陕西省|陕西省]][[礼泉县|礼泉县]]人，[[中国人民解放军|中国人民解放军]]前主要领导人之一，绰号“'''西北狼'''”。1961年8月入[[中国人民解放军|解放军]]服役，三年后被提升为军官，成为职业军人。1963年3月，在军中加入[[中国共产党|中国共产党]]，是中国共产党第十六、十七届[[中国共产党中央政治局|中央政治局]]委员、[[中國共產黨中央軍事委員會副主席|中央军事委员会副主席]]。

郭伯雄于2012年11月卸任[[中國共产党中央軍事委員會副主席|中國共产党中央軍事委員會副主席]]职务，2013年3月卸任[[中華人民共和國中央軍事委員會副主席|中華人民共和國中央軍事委員會副主席]]职务，正式退休。另一位前军委副主席[[徐才厚|徐才厚]]在2014年7月被开除党籍、军籍之后，郭伯雄也出现涉腐传闻，并被中国大陆坊间广泛认为是军队[[政治腐敗|反腐]]的下一个主要目标，但当时[[中国共产党中央委员会|中共中央]]对此没有回应。

2015年4月9日，中共中央依照中国共产党的纪律条例，决定对郭伯雄进行组织调查，但此事对外并没有公开。7月30日，[[中国共产党中央纪律检查委员会|中纪委]]网站正式宣布，郭伯雄涉嫌[[賄賂|受贿]]犯罪，中共中央政治局会议已决定将其开除中共党籍，移交[[中华人民共和国最高人民检察院|最高人民检察院]]授权军事检察机关依法处理<ref name="双开" />。郭伯雄虽与徐才厚同为军委副主席，但他在解放军中的资历比徐才厚深（郭伯雄和徐才厚于1999年9月分别以常务（第一）副总参谋长和总政治部常务（第一）副主任身份进入中央军委；郭伯雄于2002年11月起担任中央军委副主席，而徐才厚则是2004年9月起担任中央军委副主席），因此他也是迄今为止在中国人民解放军中被调查的职务最高、资历最深的一位军官。

2016年4月5日，[[中国人民解放军军事检察院|中国人民解放军军事检察院]]对郭伯雄涉嫌受贿犯罪案侦查终结，移送审查起诉<ref name="审查起诉">{{cite web |url = http://www.ccdi.gov.cn/yw/201604/t20160405_77123.html |title = 郭伯雄涉嫌受贿犯罪案已移送审查起诉 |accessdate = 2016年4月5日 |website = [[中央纪委监察部网站|中央纪委监察部网站]] }}</ref>。2016年7月25日，郭伯雄因受贿罪一审被判处[[无期徒刑|无期徒刑]]，褫夺[[中国人民解放军上将|上将]]军衔，郭伯雄本人当庭表示不上诉<ref>{{Cite web |url = http://news.ifeng.com/a/20160725/49643154_0.shtml |title = 郭伯雄一审被判处无期徒刑 剥夺上将军衔 |accessdate=2017-09-13|author=|date=2016-07-25 |website = 凤凰网 }}</ref><ref>{{Cite web |url = http://news.xinhuanet.com/legal/2016-07/25/c_1119277741.htm |title=郭伯雄案件一审宣判后军事法院负责人答记者问 |accessdate=2017-09-13 |author=|date=2016-07-25 |website=新华网 }}</ref>。目前郭伯雄在位于北京市昌平区的[[秦城监狱|秦城监狱]]服刑。

== 生平 ==
=== 早年经历 ===
1942年7月，郭伯雄出生于陕西省礼泉县[[新时乡|新时乡]]张则村，小名'''锤锤'''<ref>{{Cite web|url=http://www.zjol.com.cn/05zjol/system/2015/03/25/020570307.shtml|title=锤锤、元根、六百帝！官员“小名”知多少？|accessdate=2017-09-13|author=|date=2015-03-25|work=|publisher=浙江在线}}</ref>。1958年8月，刚年滿16岁的郭伯雄，在[[初级中学|初中]]毕业后参加当地工厂招工，被分配到陕西省[[兴平市|兴平市]]一家编号“四零八”的[[兵工廠|兵工廠]]当工人。在三年[[學徒|學徒]]期满后，郭伯雄应征入伍，被分配到[[中国人民解放军陆军|中国人民解放军陆军]]第十九军55师164团。入伍后两年，升为副班長；1963年，在加入中共后，荣升班长。1964年，服役期满后，部队决定将郭伯雄留在军中，并由士兵提升为干部（简称“提干”），担任164团的一个排长。

=== 提升===
郭伯雄在排长任上只干了三个月，就被调入164团机关任职，在政治处宣传股当干事；八个月后，转到164团司令部任参谋。也就是说，在提升为干部后刚一年的时间，郭伯雄已经升到了正连职军官。而在其后的20多年间，郭伯雄一直都在军中的作战训练部门任职，雖長期主管軍隊作戰訓練，但并無實戰經驗。1962年[[中印边境战争|中印边境战争]]，當時19軍55師曾全師開進[[西藏自治区|西藏]]，不過郭伯雄所在的164團僅擔當預備隊。<ref name="zyzz">{{cite web|url=http://news.singtao.ca/toronto/2007-10-18/1192732774d570977.html|title=黃土出奇人 虎符統三軍 (郭伯雄之一)|publisher=星岛日报|date=2007-09-28| language =zh-tw }}</ref>

1970年，郭伯雄被提拔为164团作训股股长（副营级军官）；九个月后，进入第十九军军部，任作训处参谋（正营级军官）。此后十年，他被逐步提升，相继出任第十九军作训处副处长（正团级军官）、处长（副师级军官）。1981年，回到他刚入伍时待过的第十九军55师，任师参谋长；1983年，升任第十九军参谋长。1985年，中国人民解放军进行整编工作，陆军第十九军的编制被取消；郭伯雄很幸运地进入[[中国人民解放军兰州军区|兰州军区]]，出任军区副参谋长。

1990年4月，[[傅全有|傅全有]]调任兰州军区司令员；两个月后，郭伯雄获重用，被安排出任兰州军区下辖的野战纵队[[中国人民解放军陆军第四十七集团军|第47集团军]]的军长，时年48岁。1992年10月，在傅全有升任[[中国人民解放军总后勤部|解放军总后勤部]]部长、并在[[中国共产党第十四次全国代表大会|中共十四大]]上进入[[中央军事委员会|中央军事委员会]]后，曾在其手下工作过的一些兰州军区的将领随之受到重用；其中，郭伯雄就在1993年调升[[中国人民解放军北京军区|北京军区]]副司令员，开始担任副大军区级职务，时年51岁。1997年，在[[中共十五大|中共十五大]]召开前，郭伯雄被调回兰州军区，并晋升为军区司令员，担任正大军区级职务，时年55岁。香港《[[星島日報|星島日報]]》引述解放军軍中人士称，[[江澤民|江澤民]]担任[[中国共产党中央委员会总书记|中共中央總書記]]兼[[中央军委主席|中央军委主席]]期间，[[李登輝|李登輝]]曾抛出[[台灣獨立運動|台獨]]言論，中共中央遍詢各大軍頭纷纷主戰，惟有郭伯雄反對，深合江泽民心意，成為郭伯雄再获升迁的契機。<ref name="zyzz"/>

两年后的1999年9月，在中共十五届四中全会上，除了[[胡锦涛|胡锦涛]]被增补为[[中央军事委员会|中共中央军委]]副主席外，郭伯雄亦被增补为[[中央军事委员会|中共中央军委]]委员。就在当月，郭伯雄又晋升[[中国人民解放军上将|中国人民解放军上将]]军衔，并出任解放军常务副总参谋长，时年57岁。

2016年4月6日，郭伯雄被移送审查起诉之后，中国国内媒体披露，“西北狼”郭伯雄的崛起竟是源于一次站岗，报导大领导到兰州军区视察47军时，47军[[军长|军长]]郭伯雄亲自带枪到大领导住所为其站岗，此举很受大领导赏识，从此郭官运亨通<ref>{{Cite web|url=http://www.xilu.com/jstj/20160406/1000010000938341_3.html|title=郭伯雄案通报 解密西北狼陨落背后的10个故事郭伯雄案通报|accessdate=2017-09-13|author=|date=2016-04-06|work=|publisher=西陆网}}</ref>。关于郭伯雄的发迹，另据文学城2015年3月报导，[[江泽民|江泽民]]带[[宋祖英|宋祖英]]往西北采风时，郭曾以[[司令员|司令员]]身份为江通宵站岗值守，因[[宋祖英|宋祖英]]的感谢而使郭得到江的赏识，郭遂之升迁，终于再度回到北京。<ref>{{cite web|url=http://www.wenxuecity.com/news/2015/03/02/4070725.html|title=郭伯雄荒谬发迹史：与江泽民如出一辙|publisher=[[文学城|文学城]]|date=2016-03-02}}</ref>

=== 军方领袖 ===
[[File:Gates_meets_guo.jpg|thumb]]会面]]
2002年11月[[中国共产党第十六届中央委员会|中共十六届一中全会]]上，中央军委副主席[[胡锦涛|胡锦涛]]当选[[中国共产党中央委员会总书记|中国共产党中央委员会总书记]]，同时已经从戎四十载，刚刚步入花甲之年的郭伯雄，当选为第十六届[[中国共产党中央政治局|中共中央政治局]]委员、[[中國共產黨中央軍事委員會副主席|中共中央军委副主席]]；并在次年3月当选[[中華人民共和國中央軍事委員會副主席|中華人民共和國中央軍事委員會副主席]]；成为[[中国人民解放军|中国人民解放军]]的最高领导人之一。2002年[[中国共产党第十六次全国代表大会|中共十六大]]召開前，郭伯雄力挺[[江泽民|江泽民]]卸任[[中国共产党中央委员会总书记|中共中央總書記]]後續任[[中国共产党中央军事委员会主席|中共中央軍委主席]]<ref name="zyzz"/>。2007年10月，在[[中国共产党第十七届中央委员会第一次全体会议|中共十七届一中全会]]上，郭伯雄继续当选为中央政治局委员以及中央军委副主席。

===四川大地震===

2015年5月2日，《中国企业家》杂志披露[[温家宝|温家宝]]总理救援2008年[[汶川大地震|汶川大地震]]时，军队不听调遣，温被气得摔电话。时任军委副主席郭伯雄在四川私自成立军队抗震指挥部，拒绝听抗灾总指挥[[温家宝|温家宝]]的命令，而只听从已经退休的“军委首长”[[江泽民|江泽民]]的命令，致使救灾延误。<ref name="wenxuecity">{{cite web|url=http://www.wenxuecity.com/news/2015/05/04/4241008.html|title=汶川地震时郭伯雄气得温家宝摔电话 内幕曝光(图)|accessdate=2016年4月6日|author=|date=2015年5月4日|publisher=[[文学城|文学城]]}}</ref><ref name="nanzao">{{cite web |url = http://www.nanzao.com/sc/national/14ef0f40da9d5f3/guo-bo-xiong-shi-jian-bao-lu-bei-jing-yin-you |title = 汶川地震时郭伯雄气得温家宝摔电话 内幕曝光(图) |accessdate = 2016年4月6日 |publisher = [[南華早報|南華早報]]}}</ref>

=== 卸任 ===
2012年11月，郭伯雄在[[中共十八届一中全会|中共十八届一中全会]]后卸下中共中央政治局委员、中央军委副主席的职务退休。

== 腐败被查 ==
中共于2015年7月30日正式公布对郭伯雄的审查一事，并宣布开除其党籍，移交司法机关处理。此前郭伯雄被查之事曾被中国大陆坊间广泛传闻，并被认为是在[[徐才厚|徐才厚]]之后中共反腐的下一个主要目标，被境外媒体多次报导。
=== 被公布调查前的相关传闻 ===
郭伯雄在2012年秋[[中国共产党第十八次全国代表大会|中共十八大]]之后卸任中央政治局委员，有较长一段时间没有公开现身。2013年3月5日，郭伯雄出席了在北京召开的[[第十二届全国人民代表大会|第十二届全国人大第一次会议]]开幕式。2014年1月20日，郭伯雄和[[徐才厚|徐才厚]]同时陪同新任中共中央总书记、中央军委主席[[习近平|习近平]]出席中央军委慰问驻京部队老干部迎新春文艺演出。

据中国大陆境外媒体观察，原[[中国人民解放军福州军区|福州军区]]副政治委员[[王直_(少将)|王直]]遗体[[告別式|告別式]]2014年4月13日在福州举行，在官方媒体的报道中，[[范长龙|范长龙]]、[[许其亮|许其亮]]、[[常万全|常万全]]、[[张万年|张万年]]等一众军方现役和退役将领的名字入列军方吊唁名单，唯缺郭伯雄和徐才厚两位前中央军委副主席。<ref>{{cite web|title=王直同志遗体告别仪式在榕举行|url=http://fjrb.fjsen.com/fjrb/html/2014-04/14/content_728868.htm?div=-1|date=2014-04-14|publisher=福建日報|deadurl=yes|archiveurl=https://web.archive.org/web/20160304071519/http://fjrb.fjsen.com/fjrb/html/2014-04/14/content_728868.htm?div=-1|archivedate=2016-03-04}}</ref>

兰州军区于2014年7月7日进行人事调整，格外引起外界关注；因为71岁的徐才厚落马之前，他曾长期服役的[[沈阳军区|沈阳军区]]于2014年1月底进行了高层人事调整。

2014年7月10日，香港《[[明报|明报]]》报道称，在徐才厚被开除党籍之际，郭伯雄也傳出涉及貪腐傳聞。该报称郭伯雄的兒子[[郭正钢|郭正钢]]夫婦因涉及相關案件確被[[中央军委纪律检查委员会|中央軍委紀委]]帶走協助調查。<ref>{{cite web|url=http://news.mingpao.com/20140710/cab1.htm|title=軍委前副主席郭伯雄之子 涉貪助查|publisher=明报|date=2014年7月10日 | language =zh-hans }}</ref>2015年1月14日，郭正钢晋升[[中国人民解放军少将|少将]]<ref>{{Cite news|url=http://china.caixin.com/2015-01-16/100775001.html|title=郭正钢任浙江省军区副政委并晋升少将|author=|date=2015-01-16|work=财新网|accessdate=2017-09-13}}</ref>。
