'''郭家明'''<small>{{BBS}}{{MBE}}{{JP}}<ref>{{cite web |url=http://www.info.gov.hk/cml/eng/miscell/index3g.htm|title=JP Lists|accessdate=2008-04-13|work=香港特區政府網頁}}</ref></small>（{{lang-en|Kwok Ka Ming}}，{{bd|1949年|10月30日}}）前香港著名[[足球|足球]][[運動員|運動員]]，退役後成為足球教練及導師，持有[[國際足協|國際足協]]及[[亞洲足協|亞洲足協]]A級教練牌照，先後擔任[[香港足總|香港足總]]技術顧問、[[國際足協|國際足協]]教練導師、[[亞洲足協|亞洲足協]]教練導師、[[國際足聯|國際足聯]]技術委員<small><ref>[http://news.xinhuanet.com/sports/2006-06/26/content_4750025.htm  2006年6月26日 南方日報]</ref></small>，是香港家喻戶曉的足球名宿，現任[[國際足協|國際足協]]、[[亞洲足協|亞洲足協]]及[[香港足總|香港足總]]的技術顧問，近年受邀擔任[[澳門足總|澳門足總]]的技術顧問，並曾擔任[[中國足協|中國足協]]技術主任。

== 生平 ==
;早年生活
郭家明有二姊一兄，早年於[[新界|新界]][[北區_(香港)|北區]][[粉嶺|粉嶺]][[聯和墟|聯和墟]]一帶居住，父母經營大牌檔。郭家明曾就讀[[沙頭角官立小學|沙頭角官立小學]]（後改為沙頭角官立中學）和[[大角咀|大角咀]][[聖芳濟書院|聖芳濟書院]]。他自幼喜歡運動，小時候已與友人組織足球隊「七射」，到北區、九龍及香港島不同地方與其他自組球隊踢足球。當時已接觸[[梁能仁|梁能仁]]等未出身的足球員。{{noteTag|關於郭家明家庭背景、早年生活與就讀學校的資料出自香港電台第二台廣播節目《守下留情》於2017年2月13日播出的郭家明訪問。}}於中學時期，他成為校際籃球及乒乓球選手，協助校隊奪得丙組及乙組冠軍。及後在1965年夏季參加[[香港足球總會|香港足球總會]]舉辦的青少年訓練班，受教於[[何應芬|何應芬]]及[[鄧森|鄧森]]，同屆學員還有[[梁維光|梁維光]]及[[羅灶石|羅灶石]]等。

同年，由於有三支[[香港甲組足球聯賽|香港甲組足球聯賽]]球隊沒有組織預備組，[[香港足球總會|香港足球總會]]遂選派香港（代替[[光華足球會|光華]]）、九龍（代替[[東華體育會|東華]]）及新界（代替[[傑志體育會|傑志]]）青年軍補上參加[[香港預備組聯賽|預備組聯賽]]<small><ref>{{zh-hk}} 《大公報》，1965年9月25日，第2張第8版，「青年軍參加預備組 分別填補空缺出賽」，於2009年1月13日查閱</ref></small>，郭家明入選新界青年軍，從而展開足球事業。一年後加入[[元朗區體育會|元朗]]預備組，其後由於應付[[中學會考|中學會考]]而暫停參賽<small><ref name="十八歲簽約流浪">{{zh-hk}} 《大公報》，1978年10月30日，第3張第12版，「其人其事－十八歲簽約流浪」，於2009年1月8日查閱</ref></small>。

;流浪
1968年初，在梁維光遊說下加入[[香港流浪足球會|流浪]]預備組，更獲選代表香港參加[[亞洲青年足球錦標賽|亞青盃]]；亞青盃後剛滿18歲的郭家明中五畢業，簽約加盟[[香港流浪足球會|流浪]]，成為職業球員<small><ref>[http://specials.mingpao.com/cfm/News.cfm?SpecialsID=79&Page=1&News=88df5897a17f0746cafe411b325e752aa6da09efba74542c888a5573aa76765888c6&ChineseName=%A7%EB%B8%EA%B5%A6%B2%A4%AD%B1%AD%B1%C6%5B 足總顧問郭家明 星島日報 2006年6月26日]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref></small>。

1971年遠赴[[英國|英國]]參加教練訓練班，翌年取得[[英格蘭|英格蘭]]和[[蘇格蘭|蘇格蘭]]初級教練文憑。1973年年僅二十四歲，便開始在流浪擔任球員兼教練。

;加山
1977年，獲[[加山|加山]]女班主[[陳瑤琴|陳瑤琴]]招攬加盟，同樣擔任球員兼教練，並獲選為[[香港十大傑出青年|香港十大傑出青年]]及獲頒授[[MBE|MBE]]勛銜。

1979年郭家明與加山兩年合約屆滿，由於球市不景，加山來年改組「青春班」參賽，並聘請前[[廣東|廣東]]隊教練[[陳頌揚|陳頌揚]]執教，願意讓郭家明無條件轉會<small><ref>{{zh-hk}} 《大公報》，1979年6月22日，第3張第12版，「嘉山組青春班」，於2009年1月14日查閱</ref></small>，已屆30歲的郭家明決定掛靴。

;銀禧體育中心
[[香港足球總會|足總]]有意聘請郭家明擔任[[香港足球代表隊|港隊]]教練<small><ref>{{zh-hk}} 《大公報》，1979年6月26日，第4張第16版，「足總銀禧擬聘郭家明任教」，於2009年1月14日查閱</ref></small>，但同時獲[[銀禧體育中心|銀禧體育中心]]主任[[祈赋福|祈赋福]]賞識，終受聘為銀禧中心足球教練，負責學校及青少年足球訓練工作<small><ref>{{zh-hk}} 《大公報》，1979年6月27日，第3張第12版，「郭家明今簽約銀禧 擔任足球教練職位」，於2009年1月14日查閱</ref></small>，簽約三年，於7月1日起生效，足總主席[[許晉奎|許晉奎]]亦有出席簽約儀式，銀禧中心與足總協議，如有需要，可以借出郭家明給足總<small><ref>{{zh-hk}} 《大公報》，1979年6月28日，第3張第12版，「郭家明簽約銀禧 訓練青少年球員」，於2009年1月14日查閱</ref></small>。

== 國際賽 ==
;擔任隊長
1973年，郭家明首次出任[[香港足球代表隊|香港隊]]隊長，在[[賀歲盃足球賽|賀歲盃足球賽]] 0–11 慘負予[[葡萄牙|葡萄牙]]勁旅[[賓菲加|賓菲加]]<small><ref name="輸得最慘一仗">{{zh-hk}} 《大公報》，1978年10月31日，第3張第12版，「其人其事－輸得最慘一仗」，於2009年1月8日查閱</ref></small>。同年，隊長身份帶領香港代表隊，參加在[[漢城|漢城]]舉行的世界盃外圍賽，最終不敵南韓未能出線。

;非全職教練
1978年接替[[包勤|包勤]]，擔任[[香港足球總會|香港足球總會]]非全職教練及主持教練訓練班<small><ref>{{zh-hk}} 《大公報》，1978年9月20日，第3張第12版，「郭家明兼足總教練」，於2009年1月8日查閱</ref></small>，首仗於10月29日出戰[[秘魯|秘魯]]球隊[[士砵亭水晶|水晶隊]]（Sporting Cristal），[[香港足球代表隊|港隊]]淨吞三隻光蛋<small><ref>{{zh-hk}} 《大公報》，1978年10月30日，第3張第12版，「港隊淨吞三蛋」，於2009年1月8日查閱</ref></small>。

;告別港腳生涯
1979年5月，郭家明隨[[香港足球代表隊|香港隊]]到[[泰國|泰國]][[曼谷|曼谷]]參加[[亞洲盃足球賽|亞洲盃]]第三區外圍賽，是他代表香港參加最後一項大賽，香港有[[戴歷·居理|戴歷·居理]]及[[安德遜|安德遜]]兩名歸化外援首次入選，於分組賽以次名出線，5月11日對[[馬來西亞國家足球隊|馬來西亞]]爭奪決賽權，戰至[[加時賽|加時]]後，雙方仍然 0–0 平手，需[[互射十二碼|互射十二碼]]決勝，擔任副隊長的郭家明負責首射卻告「宴客」，其餘各人均能射入，香港以十二碼 4–5 飲恨而回<small><ref>{{zh-hk}} 《大公報》，1979年5月12日，第3張第12版，「港隊昨不敵馬來西亞 喪師曼谷絕緣亞洲杯」，於2009年1月13日查閱</ref></small>。

;五一九戰役
{{main|五一九事件}}
1985年5月19日，時任[[香港足球代表隊|香港隊]]主教練的郭家明率領港隊做客[[北京|北京]][[北京工人體育場|工人體育場]]對[[中國國家足球隊|中國隊]]，參加[[1986年世界盃|1986年世界盃]]外圍賽。港隊憑[[張志德|張志德]]和[[顧錦輝|顧錦輝]]的入球，以 2–1 擊敗中國隊，令中國隊失去晉級參加[[世界盃|世界盃]]的機會，被中國球迷稱為「五一九」事件<small><ref>{{Cite web |url=http://sports.21cn.com/zhuanti/056/duishou/2004/11/15/1853805.shtml |title=对话郭家明：两队差距从来只有一两球 |access-date=2008-06-03 |archive-url=https://web.archive.org/web/20041208165717/http://sports.21cn.com/zhuanti/056/duishou/2004/11/15/1853805.shtml |archive-date=2004-12-08 |dead-url=yes }}</ref></small>，郭家明亦被中國傳媒譽為「智多星」。

== 教練導師 ==
1990年，郭家明成為[[亞洲足協|亞洲足協]]足球教練導師。1993年為澳洲[[世青盃|世青盃]]擔任技術研究工作。1996年，他成為[[國際足協|國際足協]]足球教練導師，亦為[[1996年夏季奧林匹克運動會|阿特蘭大奧運]]足球賽擔任技術研究。1998年更成為[[亞洲足協|亞洲足協]]的顧問<small><ref name="the-sun">[http://the-sun.on.cc/channels/news/20060603/20060603030330_0000.html 首位獲殊榮華人 任世盃研究員郭家明揚威太陽報 2006年6月3日]</ref></small>。

[[2006年世界盃|2006年世界盃]]郭家明獲[[國際足協|國際足協]]委任為十二位[[2006年世界杯足球赛#技術研究小組|技術研究員]]之一，與[[喀麥隆|喀麥隆]]的[[羅傑·米拉|羅傑·米拉]]和英格蘭的[[罗伊·霍奇森|鶴臣]]等為賽事撰寫技術報告<small><ref name="the-sun" /></small><small><ref>[http://2006.sina.com.cn/arg/2006-06-11/070626205.shtml 国际足联技术研究小组郭家明：阿根廷胜在临门一脚 新华网 2006年06月11日]</ref></small>，成為歷來首位擔任技術研究員的華人，也是繼[[球證|國際球證]][[陳譚新|陳譚新]]參與[[1982年世界盃|1982年世界盃]]後，第二位參與世界盃決賽週的香港人。

2010年12月3日，[[中國足協|中國足協]]宣布聘請郭家明擔任技術主任，任期至少四年<small><ref>[http://orientaldaily.on.cc/cnt/sport/20101204/00286_011.html 郭家明北上拯救國足]</ref></small>。

2014年6月，郭家明連續3屆世界盃獲委任為國際足協技術研究小組（TSG）成員。全亞洲只有他和前日本國家隊隊長[[宮本恆靖|宮本恆靖]]獲得此職銜。他會在6月7日啟程前往[[巴西|巴西]]，不斷穿州過省觀賞比賽，直至決賽結束後7月中才返回香港。<ref>{{cite news|url=http://football.on.cc/cnt/news/newc/20140606/fbnewc0116x0.html|title=郭家明入球大好友|date=2014年6月6日|newspaper=''[[東方日報_(香港)|東方日報]]''}}</ref>
