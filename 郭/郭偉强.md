'''郭偉强'''{{JP}}（{{lang-en|Kwok Wai-keung}}，{{bd|1978年4月15日<ref>{{cite web|title=中評論壇：香港青年從政意願及顧慮|url=http://mag.chinareviewnews.com/crn-webapp/mag/docDetail.jsp?coluid=0&docid=102146830&page=12|publisher=中國評論月刊網絡版|accessdate=2015-12-16|date=2012-06-21}}</ref>|}}），[[香港|香港]]政治人物，[[工聯會|工聯會]]會員。現任[[香港立法會|香港立法會]][[香港島選區|香港島選區]]議員，前勞工界議員（[[自動當選|自動當選]]）， [[東區區議會|東區區議會]]議員及註冊[[社工|社工]]， 房委會成員。

== 簡歷 ==
[[2003年香港區議會選舉|2003年香港區議會選舉]]，參選[[黃大仙區議會|黃大仙區議會]][[正安_(選區)|正安選區]]落敗。

[[2007年香港區議會選舉|2007年香港區議會選舉]]，以無黨派候選人身份參選並當選為東區區議員（[[和富|和富]]），及後加入[[工聯會|工聯會]]，於[[2011年香港區議會選舉|2011年]]及[[2015年香港區議會選舉|2015年]]，兩度自動當選連任。

[[2008年香港立法會選舉|2008年香港立法會選舉]]，在香港島隨[[民建聯|民建聯]][[曾鈺成|曾鈺成]]名單參選，但沒有取得議席。

[[2012年香港立法會選舉|2012年香港立法會選舉]]，代表[[工聯會|工聯會]]參選功能組別勞工界，自動當選為[[香港立法會|香港立法會]]議員。

[[2016年香港立法會選舉|2016年香港立法會選舉]]，代表[[工聯會|工聯會]]循香港島地方選區當選。

== 工作 ==
=== 政策倡議 ===

2017年8月8日，其聯同工聯會四位立法會議員，與勞福局局長羅致光會面後提出一系列相關政策建議，主要包括：

# 取消對沖安排 全面檢討強積金計劃。
# 完善法定產假及侍產假的安排。
# 檢討標準僱傭合約。
# 劃一法定假日及公眾假期日數。
# 優化低收入在職家庭津貼。
# 「福建計劃」及福利可攜措施。

=== 立法會事務 ===

==== 2012年 ====

* 9月當選為勞工界[[香港立法會|香港立法會]]議員。
* 10月17日就「立法規管工時」提修正案。
* 11月28日就「關注青年人面對的學業、就業、住屋置業和創業問題」提修正案。

==== 2013年 ====

* 1月23日就「幫助中產」提修正案。
* 6月5日就「積極推動家庭友善政策」提修正案。
* 6月26日就「提升本港專上教育質量」提修正案。
* 7月10日就「推動廢物循環再造業，創造就業機會」提原[[議案|議案]]，並獲得通過。
* 12月北角五洲大廈致命火災後，發現有牌賓館在違反大廈公契情況下，依然可以領牌營業，而大廈內有無牌賓館及影子賓館。其隨即要求民政事務局解決違反大廈公契事宜，並解決無牌賓館及影子賓館肆虐情況，促成政府檢討《旅館業條例》。政府於2014年7月發表諮詢文件，並在2015年3月公布修例方向，計劃在2016/17立法年度，進行立法修訂。

==== 2014年 ====
10月下旬在立法會大會作出口頭質詢，要求政府跟進世衛10月份建議，研究立法規管電子煙。他說最近發現不少電子煙產品在網上及商場格仔舖銷售。不少電子煙包裝標榜不含尼古丁，但研究卻顯示有可能釋出致癌物質，吸食電子煙更可能導致青少年最終成為煙民。最初，政府只回應會留意世衛建議及其他地區實施情況，其後表示會積極考慮。2015年吸煙及健康委員會公布調查，建議香港全面禁止電子煙銷售、推銷及宣傳。食物及衛生局局長高永文建議循全面禁止電子煙，作立法準備。2016年10月政府表示會在這一個立法年度，提交規管電子煙的立法建議。

==== 2016年 ====
7月13日在[[香港立法會|香港立法會]][[香港施政報告|施政報告]]答問大會就能否立法制定[[標準工時|標準工時]]，向[[梁振英|梁振英]]提出多次質詢。他於公務員及資助機構員工事務委員會就聘用非公務員合約僱員的情況向公務員事務局局長提質詢及意見。並於財政預算案立法會財委會特別會議就培訓基金向財政司司長向[[曾俊華|曾俊華]]先生提質詢及意見。

==== 2017年 ====
1月11日提出「對下任行政長官的期望」的議案辯論：
{{Cquote|香港回歸以來因政治爭拗而持續內耗，社會難以聚焦民生及經濟議題，導致發展停滯不前；傳統支柱產業優勢面對重大挑戰，以及新興產業發展緩慢，均令本地產業結構變得單一空洞；高地價政策促成地產業獨大及物業炒賣成風，令租金過高，窒礙經濟多元化發展；政府欠缺改善勞工權益的決心，無法扭轉『強資本、弱勞工』的局面；就此，本會期望下任行政長官廣納建言以凝聚社會共識、維護國家主權、貫徹落實『一國兩制』，以及推動政制改革，以帶領香港走出政治爭拗的困局；下任行政長官亦應集中精力發展經濟、改善民生，並著力改善勞工權益，令香港重回繁榮與安定的軌道。}}

=== 個案跟進 ===

==== 埃及熱氣球爆炸事故 ====
2013年2月底發生的香港旅客在[[埃及|埃及]]不幸遇上熱氣球爆炸，多人遇難。在3月上旬郭偉强協助受難家屬，作多次會晤，希望盡快召開死因庭。死因研訊已於2016年6月7日審結，死因裁判官裁定9名遇難港人死於意外，裁判官形容事件是史上最嚴重的熱氣球災難，並批評涉事旅行社勝景遊或香港旅遊業議會，事前均沒有為熱氣球活動進行評估，不能洞察當中風險；事發至今3年，沒有為喪失9個性命而有所警惕，沒作任何檢討，反映香港旅遊業界對旅客福祉的敏感度太低。死因庭對旅遊業議會作了6項建議，對勝景遊作了5項建議，要求兩機構在3個月內回覆。郭偉强2016年11月16日與旅遊業議會主席黃進達及總幹事董耀中會晤，跟進建議的落實與推行。

== 爭議 ==
=== 學歷 ===
郭偉强於2009年入讀[[香港理工大學|香港理工大學]]社會政策及行政(榮譽)文學士兼讀制學位課程，他向外界解釋因兼顧議員工作而延遲至2014年才畢業。

2014年，有人發現郭偉強在立法會網頁申報之學歷為「社會政策及公共行政學士」，他稱是助手填錯資料，但由於課程內容包括公共行政的科目，堅持自己讀公共行政的說法並無問題。更甚的是，當被問到會否認為理大要為學科改名時，郭竟回應稱「都應該喎」。他的言論引起部分該課程的學生及畢業生的強烈不滿，並發起網上聯署要求他道歉。事後郭偉强在立法會網頁更正其學歷為「社會政策及行政(榮譽)文學士」。
