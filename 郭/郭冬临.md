'''郭冬临'''（{{bd|1966年|7月20日|catIdx=Guo,郭}}<ref>一說10月18日，但根據[http://ent.cntv.cn/mingxing/guodonglin/index.shtml 央視旗下中國網絡電視臺綜藝台的明星庫]，正確生日應爲7月20日。</ref>），中国男演员。

== 生平 ==

出生于[[山东|山东]][[烟台|烟台]]，毕业于[[上海戏剧学院|上海戏剧学院]]表演系，中国著名[[话剧|话剧]]、[[小品|小品]]、[[相声|相声]]、[[影视|影视]][[演员|演员]]，多次在[[中国中央电视台春节联欢晚会|中国中央电视台春节联欢晚会]]中登台亮相，也屡次在中国中央电视台中串客。原为[[天津快板|天津快板]]演员。在登上春节联欢晚会后，连续多次继续春节联欢晚会生涯。所表演的小品有：《[[路口|路口]]》、《[[有事您说话|有事您说话]]》、《[[球迷|球迷]]》、《[[相亲|相亲]]》 、《[[旧曲新唱|旧曲新唱]]》等。

== 评价 ==
2012年，[[央视|央视]]龙年春晚，郭冬临、[[魏积安|魏积安]]等春晚小品《超市面试》在播出后，网友在[[微博|微博]]随即爆出其全面抄袭[[日本|日本]]搞笑组合[[UNJASH|UNJASH]]小品《打工面试》，几乎是整体上翻译过来的。<ref>{{cite web |url=http://ent.ifeng.com/tv/special/chunwan/dujiabaodao/detail_2012_01/23/12126978_0.shtml|title=《郭冬临小品被指抄袭日本UNJASH小品《打工面试》》|accessdate=2012年1月23日 |work=|publisher=凤凰网|language= zh-hans}}</ref>

2005年以後，郭冬臨的小品以描述夫妻關係為主，但是很多網友指出這些小品風格套路化，且經常體現一種“男卑女尊”的現象。
