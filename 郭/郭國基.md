'''郭國基'''（{{lang-nan|Koeh Kok-ki}}；{{bd|1900年|4月10日|1970年|5月28日|CatIdx=G郭}}）是[[台灣|台灣]]的政治人物。出生於東港（今[[屏東縣|屏東縣]][[東港鎮_(台灣)|東港鎮]]）。

擔任台灣省議員期間，郭國基問政強悍，與[[吳三連|吳三連]]、[[李萬居|李萬居]]、[[郭雨新|郭雨新]]、[[李源棧|李源棧]]、[[許世賢|許世賢]]並稱為省議會「五龍一鳳」。他曾經說過「賜我光榮死於議壇」，結果一語成讖，當選[[立法委員|立法委員]]第一年的開議初期因[[癌症|癌症]]而病故。

郭國基在台灣省參議會最著名的事蹟是，在1950年於省參議會中質詢[[國立台灣大學|台大]]校長[[傅斯年|傅斯年]]後，傅因校務勞累下台後[[腦溢血|腦溢血]]病逝，代理議長[[李萬居|李萬居]]接受《[[台灣新生報|台灣新生報]]》訪問時表示傅斯年「棄世」時遭記者誤傳為「氣死」，導致台大學生遊行至省參議會要求交出郭國基的事件。

== 早年生平 ==
郭國基於1900年出生於今屏東縣東港鎮，8歲時入[[東港公學校|東港公學校]]就讀，公學校畢業後，考進[[台南長老教會學校|台南長老教會學校]]（今[[長榮中學|長榮中學]]），在中學時受到日本人歧視，使郭國基產生民族意識。{{r|郭|page=6}}

[[大正|大正]]六年（1917年），郭國基在18歲時前往[[日本|日本]]，先後就讀[[東京|東京]]青山學院（今[[青山學院大學|青山學院大學]]）中學部、[[明治大學|明治大學]]預科（先修班），後至[[明治大學|明治大學]]政治系就學。大正九年（1920年），臺灣留學生於東京組織「[[新民會|新民會]]」，當時就讀明治大學的郭國基與同校同學[[彭華英|彭華英]]等16人參與其中，同年6月14日，郭國基帶頭發動留日學生拒絕臺灣總督行之有年的「始政宴」，以抗議日本殖民政府高壓統治。{{r|康|page=38}}並投身[[社會運動|社會運動]]。翌年，加入[[台灣文化協會|台灣文化協會]]。1922年，再加入「[[台灣議會期成同盟會|台灣議會期成同盟會]]」。更在1923年，組織「留學生文化演講團」，到台灣各地巡迴演講，由於言詞犀利，便有了「郭大炮」這類的稱號。

大正十四年（1925年），郭國基從明治大學畢業。同年夏天，郭國基經朝鮮、遼東前往中華民國北京市，與總統[[黎元洪|黎元洪]]見面，上萬言書報告台灣事情，並獲得黎元洪總統玉照。{{r|郭|page=7}}1926年，留日華僑舉行[[郭松齡|郭松齡]]夫婦追悼大會，郭國基於會中巧遇就讀[[日本陸軍大學|日本陸軍大學]]的[[盛世才|盛世才]]，與之為友。{{r|郭|page=8}}並經盛世才介紹，秘密加入[[中國國民黨|中國國民黨]]東京支部為黨員。{{r|郭|page=8}}

[[昭和|昭和]]十四年（1929年），郭國基在其妹介紹下與出身[[新潟縣|新潟縣]]、東洋女子齒科醫學專業學校畢業的鈴木久代結婚，婚後相繼回臺。回台灣後定居高雄，任教長榮等女校；後任高雄州廳民事調停官。

昭和十七年（1942年），日本殖民當局在前高雄州特別高等警察課課長[[仲井清一|仲井清一]]主導下，羅織[[東港事件|東港事件]]等政治事件，大批搜捕有反日思想者。郭國基妹婿[[張明色|張明色]]遭控副主謀（主謀[[歐清石|歐清石]]），而郭國基為營救張明色奔走，也遭指控通謀美軍轟炸高雄港、炸毀高雄軍事設施，並在監獄中遭「廿四名無情漢，用卅六種刑具」刑求，屈打成招之下承認接受[[蔣介石|蔣介石]]資金反叛日本。{{r|郭|page=10}}郭國基在一審被判無期徒刑，複審改判十年有期徒刑。在[[日本投降|日本投降]]前夕[[保外就醫|保外就醫]]。

郭國基其妻鈴木久代在此段期間常遭到日本右翼份子羞辱，說她是日本人竟嫁給殖民地人民為妻，而且居然還敢謀反，令鈴木久代以淚洗面。

== 踏入政壇 ==
1945年，日本戰敗投降。郭國基結合[[東港事件|東港事件]]被害者，組織高等事件復仇會，決定「血祭」全臺特務。將主導東港事件的仲井清一私刑處決。

當時台灣社會充滿對中國的幻想，彌漫在歡迎「祖國」的氣氛。然而去過中國的郭國基卻未如此樂觀，他專程自高雄出發，一路走過被美軍轟炸的鐵公路，前往臺中阿罩霧（霧峰）遊說[[林獻堂|林獻堂]]，力勸他不要以為臺灣回歸「祖國」就一切都好了，必須把[[台灣文化協會|文化協會]]演講、議會設置請願等運動維持下去。然而林獻堂並沒有聽下去。那年雙十節還在臺北中山堂前的慶祝大會發表演說，歡迎「祖國」到來。{{r|康|page=40}}

1946年，當選高雄市參議會議員；又被推為臺灣省參議員，擔任中國國民黨高雄市黨部第一任指導員。

1947年2月底發生[[二二八事件|二二八事件]]，郭國基遭政敵挾怨報復，被指煽動暴動於5月6日被捕，入獄210天，最後無罪釋放。而在獄中體認到國民黨省黨部的袖手旁觀，使郭國基心灰意冷，在出獄後退出國民黨。{{r|郭|page=15}}

===質詢傅斯年事件===
1950年12月20日下午六時，郭國基於省參議會中質詢[[國立台灣大學|國立台灣大學]]校長[[傅斯年|傅斯年]]，內容為詢問台大的「秀才教育」及[[聯合國|聯合國]]捐贈台大器材遺失的問題，據在場緊鄰傅斯年的教育廳長[[陳雪屏|陳雪屏]]回憶，問答者雙方均無參雜火藥氣味<ref>陳雪屏，「北大與台大的兩段往事」，《傳記文學》第28卷第1期，頁16；引自《戰後台灣政治議會史之研究》，[[鄭梓|鄭梓]]，華世出版社，台北，1988年，頁230</ref>。然而因傅斯年長年健康不佳、校務勞累，導致下台後[[腦溢血|腦溢血]]病逝。時任代理議長的議長[[李萬居|李萬居]]在接受《[[台灣新生報|台灣新生報]]》記者訪問時用[[國語|國語]]發音表示傅斯年「棄世」，遭記者誤傳為「氣死」，導致傳出「郭大砲氣死傅斯年」傳言。甚至隔日（12月21日）四百名台大學生遊行至[[台北市|台北市]][[南海路|南海路]]省參議會要求交出郭國基出來，隨後李萬居在省參議會第十次大會第五次會議上上以代理主席表示「昨日郭參議員詢問各點，均屬普通問題，態度亦即溫和，彼回復時亦及心平氣和，並無憤怒之事。今日報載『棄世』誤為『氣死』，實為誤聽。」<ref>《台灣省參議會第十次大會特輯》，頁15；引自《戰後台灣政治議會史之研究》，鄭梓，同上，頁228</ref>

=== 省議員時期 ===
1957年，臺灣省臨時省議會第三屆，轉移至台北競選，獲第二高票當選，競選期間，使他成為臺灣選舉史上成功的街頭演說家之一。

1960年，於臺北競選第二屆省議員，高票當選，但轉而控訴「選舉無效」，要求查驗投票所選票數，革除選舉各種弊端，被判駁回。

1963年，於臺北競選第三屆省議員，然而在選前[[高玉樹|高玉樹]]大力支持無黨籍的楊玉城。最後郭國基以些微票數落選。

1968年，郭國基回到高雄競選[[1968年臺灣省議員選舉|第四屆省議員選舉]]，在同時舉行的[[1968年中華民國縣市長選舉#高雄市|市長選舉]]中表態支持以往有宿怨的[[楊金虎|楊金虎]]爭取黨外票源，最後郭國基與楊金虎雙雙當選。

郭議員在省議會期間問政犀利，咄咄逼人，常為選民所愛戴。其政治主張不涉個人好惡，而力主民主法治之推行，以及民生生活水平之改善，其問政常受知識分子執著之影響，而能一針見血，擊中要害，為省府首長最難應付之人物之一。

== 競選立委及晚年 ==
1969年，郭國基因[[直腸癌|直腸癌]]赴[[臺大醫院|臺大醫院]]手術。一度赴日本接受[[鈷六十|鈷六十]]放射治療。

同年，舉行[[1969年中華民國立法委員選舉|增選立法委員選舉]]，為國民黨政府遷台來首次立委增選補選。本次立委為終身職。郭國基以「助我進軍立法院」為號召，投入[[台灣省|台灣省]]第二選區參選，成功當選立法委員。{{r|打|page=58}}在立法委員就職典禮上，當記者問他到國會後第一件事要爭取什麼。郭國基不假思索回答「成立[[反對黨|反對黨]]：沒有反對黨，說是民主政治，是欺世盜名。」，只是不久後他就辭世。{{r|打|page=59}}

1970年5月28日零時七分，因直腸癌併發尿毒症，郭國基於立法委員任內病逝臺大醫院。{{r|郭|page=11}}

== 相關事蹟 ==
* 郭國基在競選第二、三屆省議員時曾在傳單自言，在日本中學時代作答日本歷史考卷時，寫下[[應神天皇|應神天皇]]是[[神功皇后|神功皇后]]在征伐朝鮮時與[[武內宿禰|武內宿禰]]通姦所生，如[[呂不韋|呂不韋]]與[[秦始皇|秦始皇]]母子，以張顯日本皇統非萬世一系。曾一度險被開除學籍，後來以「臺灣學生不識日本皇室尊嚴、神聖不可侵犯」，「姑暫寬恕」。{{r|郭|page=162}}{{r|郭|page=226}}（據《[[日本書紀|日本書紀]]》載，[[仲哀天皇|仲哀天皇]]未隨神功皇后征韓就急死。神功皇后征韓後產下應神天皇）
* 郭國基在競選時，最具特色的是播放[[日軍|日軍]][[軍歌|軍歌]]《[[軍艦行進曲|軍艦行進曲]]》，搭配大砲造型。以形容自己為「郭大砲」的形象。
* 1957年郭國基在當時還是省轄市的臺北市競選省議員，有一次在廣州街警察學校政見發表會上，有許多聽眾叫鬧着不許他用[[臺語|臺語]]演講，郭國基激昂陳詞：「我生為台灣人，是食台灣水活的，是食台灣米大的，日本政府還不敢禁止咱講台灣話。台灣人在台灣，不准講台灣話？你們少數作風如同『[[國民黨|乞丐]]趕廟公』、「[[關老爺|關老爺]]借[[荊州|荊州]]、霸荊州」、『豆油借你醞，碟子都搶去』…」這兩句「乞丐趕廟公」、「豆油借你醞，碟子都搶去」後來成為社會上許多人朗朗上口的俗語。<ref>[http://www.peoplenews.tw/news/8edfa9d1-ade5-4f58-928d-acd77f434022 【專文】郭國基的4個故事]，李筱峰，《民報》，2014年9月5日</ref>
* 郭國基對於黨國不分的「[[國民黨黨歌|國民黨黨歌]]」兼「[[中華民國國歌|國歌]]」深不以為然。他曾在省議會質詢主張將「吾黨所宗」改為「吾國所宗」，在省議會裡引起風潮，但無結果。每逢省議會開議唱「國歌」時，他絕不起立，他說那不是國歌，而是國民黨黨歌（此事實至今猶然）。有一次有外賓來省議會參觀，引起議長[[黃朝琴|黃朝琴]]緊張，拜託郭國基在唱「國歌」時勿再坐著，否則外賓問起，實在不知如何解釋。郭國基給了黃朝琴面子，答應可以不坐著，但絕不肅立，只站一隻腳。果然，唱「國歌」時，記者們的眼睛都在注意郭國基的腳，郭果然只站一隻腳，喧騰一時。<ref>[http://www.peoplenews.tw/news/8edfa9d1-ade5-4f58-928d-acd77f434022 【專文】郭國基的4個故事]，李筱峰，《民報》，2014年9月5日</ref>

== 家族 ==
* 郭國基與鈴木久代育有兩男一女，長子郭拔山，次子郭蓋世（取楚霸王名言「力拔山兮氣蓋世」）。女兒郭美娜。鈴木久代在高雄[[鹽埕區|鹽埕區]]新興街開設久代齒科醫院。郭蓋世與[[康寧祥|康寧祥]]在高中時同班。

== 評價 ==
由於郭國基同時具有[[大中華主義|大中華主義]]思想、反對[[國民黨|國民黨]]一黨獨裁及反對國民黨政府的打壓[[臺語|臺語]]政策等政治立場，因此郭國基曾受到統獨兩派人物的稱許。
* 立場親統派的作家[[李敖|李敖]]评价郭国基说：“郭国基为什么了不起？因为他有一个奇怪的观念，他被[[国民党|国民党]]政府抓住以后，给他带了脚镣，六斤半重的脚镣，人都不大能走路了，说他涉案228事件，事实上跟他无关，当时他被嫌疑，就把他关起来。有一次他做省议员的时候，就跟[[陈诚|陈诚]]见面，他跟陈诚说，你们这些外省人，太小看我们台湾人了，你们以为我们要搞台湾独立，我们那么笨吗？台湾这麼小，大陆这么大，大陆这些土地资源财产，是你们的祖先跟我们的祖先共同经营来创造的，为什么我只要台湾，我不要大陆？这是我祖先留给我的财产，大陆也是我们的，也是我们台湾人的，为什么你们老以为我们台湾人想独立，你太小看我们了……大家见过这么有气魄的，这么有眼光的，这么有胸怀大志的台湾人吗？这就是郭国基先生。”<ref>中国当代，李敖，《李敖语妙天下》，“郭国基：只载祖国放弃台湾，未闻台湾放弃祖国”，新加坡亚盛卫视</ref>
* 立場親獨派的歷史學者[[李筱峰|李筱峰]]則曾稱讚郭國基抗議國民黨政府黨國不分以黨歌當[[國歌|國歌]]、禁止臺灣話的政策，以及積極與[[雷震|雷震]]推動成立[[中國民主黨_(1960年)|中國民主黨]]的事蹟。<ref>[http://www.peoplenews.tw/news/8edfa9d1-ade5-4f58-928d-acd77f434022 【專文】郭國基的4個故事]，李筱峰，《民報》，2014年9月5日</ref>
