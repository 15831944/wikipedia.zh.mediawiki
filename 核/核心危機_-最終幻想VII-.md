{{vgname|核心危机 -最终幻想VII-|ja=クライシス コア -ファイナルファンタジーVII-|en=Crisis Core: Final Fantasy VII}}是史克威尔开发并发行於[[PlayStation_Portable|PlayStation Portable]]平台上的[[动作角色扮演游戏|动作角色扮演游戏]]。游戏于2007年首发，是1997年PS版《[[最终幻想VII|最终幻想VII]]》的[[前传|前传]]，亦是相关系列“[[最终幻想VII补完计划|最终幻想VII补完计划]]”的作品之一。

游戏主要讲述[[扎克斯·菲爾|扎克斯·菲尔]]的故事，他是神羅公司旗下[[准军事部队|准军事部队]]的年轻战士，受命寻找失踪的高级战士[[傑尼西斯·拉普索道斯|杰尼西斯·拉普索道斯]]。在寻找杰尼西斯的過程中，扎克斯发现了杰尼西斯的出身——计划G，以及他同另两名高级战士[[萨菲罗斯|萨菲罗斯]]和[[安吉爾·休雷|安吉尔·休雷]]的关系。游戏故事从巨型公司神罗与五台的战争起，經尼布尔海姆事件，最後於《最终幻想VII》开始之前的時間點結束。

游戏制作由田畑端指挥，[[野村哲也|野村哲也]]设计角色。在开发前，最初计划為[[手机游戏|手机游戏]]《[[危机之前_-最终幻想VII-|危机之前 -最终幻想VII-]]》的PlayStation Portable移植版，但在讨论后轉為决定制作一款新游戏。剧情依照了编剧[[野岛一成|野岛一成]]在创作《最终幻想VII》时的构思。《核心危机》在全球售出319万套<ref>http://www.vgchartz.com/db/game.php?id=7385</ref>。

== 玩法 ==

《核心危机》是一款动作角色扮演游戏，玩家将操作角色扎克斯·菲尔。玩家控制扎克斯在开放区域间移动、和[[非玩家角色|非玩家角色]]对话、与环境互动，并和怪物交战。在存档点，玩家可从选择执行一个解锁支线任务，之后扎克斯就会被送到特殊区域完成任务，任务通常会要求击败一只或数只怪物，若玩家成功完成任务就会获得有用的道具奖励，往往另一新任务也随之解锁。无论玩家支线任务完成成功与否，一当任务结束，扎克斯就会回到主线游戏的存档点处。

《核心危机》使用实时战斗系统，玩家可以移动扎克斯四处跑动、直接攻击、发动特殊能力或魔法、使用道具、抵挡或闪避攻击<ref name="battle">{{cite web | author=Gantayat, Anoop| year=2006 | title=Hands On: Final Fantasy VII Crisis Core | url=http://www.ign.com/articles/2006/12/17/hands-on-final-fantasy-vii-crisis-core| publisher=IGN | accessdate=2007-03-09}}</ref><ref name="battle2">{{cite web| author=Suzaku| year=2007| title=New Crisis Core details| url=http://forums.adventchildren.net/showthread.php?p=2795715| archiveurl=https://web.archive.org/web/20070603042528/http://forums.adventchildren.net/showthread.php?p=2795715| archivedate=2007年6月3日| publisher=Crisis-Core| accessdate=2007-03-09| deadurl=yes}}</ref>。扎克斯战斗中的能力由此时装备的魔石决定<ref name="battle3">{{cite web | author=Suzaku | year=2007 | title=New Crisis Core Scans from Famitsu | url=http://forums.adventchildren.net/showthread.php?p=2939910 | archiveurl=https://web.archive.org/web/20070723092310/http://forums.adventchildren.net/showthread.php?p=2939910 | archivedate=2007年7月23日 | publisher=Crisis-Core.net | accessdate=2007-06-12 | deadurl=yes }}</ref>。玩家至多能同时装备6个魔石，它们可以发动特殊攻击、发动[[魔力值|魔法]]、提升[[生命值|生命值]]等能力值，或是显示当前战斗中敌人的能力值。魔石还能通过合成提升力量；比如将攻击魔石和元素魔法魔石合成，就可获得能同时造成魔法和物理伤害的新攻击技能魔石。游戏中收集的特殊道具也能在魔石合成中进一步提升魔石力量。

《核心危机》使用“数码情感波动”（DMW）系统来影响战斗，该系统类似[[角子机|角子机]]，为两组三轮盘；一组为游戏中和扎克斯关系密切角色的头像，另一组为1到7的数字。在战斗开始时，屏幕左上角会出现DMW轮盘，当扎克斯“战士点数”不少于10时，转盘就会自动随机转动，战士点数可通过击败敌人获得。头像转盘依次按左、右、中的顺序停止，当头像轮盘转毕后，数字转盘就会停止。若DMW轮盘左右转轮的头像相同，屏幕就会切换为大转轮；此时若DMW停止时三张图片相同，扎克斯就会发动此角色的极限技，包括大幅伤害敌人，或是大幅自我治愈。在大转轮下，如果出现两个或以上相同数字，对应装备栏位魔石的力量就会强化；如果转出数字为“777”，扎克斯就会升级，生命值、战士点数和能力值都会随之提升。小转轮转出的数字则会短暂提供附加效果，如无敌或无消耗发动技能与能力。Limit等级影响头像匹配的几率。在获得特殊游戏道具后，DMW图片就可能随机变化为召唤兽，其破坏力更强且能提升DWM匹配的成功率。

在游戏完成后，玩家将可以选择[[二周目|二周目]]选项<ref>{{cite web|date=2007-09-05|title=Game Grep - ''Crisis Core'' Review|url=http://www.gamegrep.com/news/5050-ffvii_crisis_core_reviewed_given_875_out_of_10/| publisher=Gamegrep|accessdate=2007-01-06}}</ref>。北美和欧洲版《核心危机》还加入了困难模式，该模式敌人的力量和生命值都会提升。

== 剧情 ==

=== 角色 ===

{{main|最终幻想VII系列角色列表}}

《核心危机》开始于《最终幻想VII》事件的七年前，《最终幻想VII》及其他相关作品中的许多角色都有登场。然而，游戏的主要角色是战士（神罗雇佣兵）和他们的特工精英组织[[最终幻想VII系列角色列表#塔克斯|塔克斯]]。

游戏的[[主人公|主人公]]是年轻而友善的战士扎克斯·菲尔。他的导师兼朋友安吉尔·休雷是一等战士<ref>{{cite web | url=http://www.ign.com/articles/2008/02/08/crisis-core-character-profiles|title=Crisis Core: Character Profiles|publisher=IGN |author=Clements, Ryan|page=1 |accessdate=2010-07-24}}</ref>，安吉尔和同僚战士萨菲罗斯与杰尼西斯是好友，后者最终变成游戏主要敌人<ref>{{cite web | url=http://www.ign.com/articles/2008/02/08/crisis-core-character-profiles?page=2|title=Crisis Core: Character Profiles|publisher=IGN |author=Clements, Ryan|page=2 |accessdate=2010-07-24}}</ref><ref>{{cite web | url=http://www.ign.com/articles/2008/02/08/crisis-core-character-profiles?page=3|title=Crisis Core: Character Profiles|publisher=IGN |author=Clements, Ryan|page=3 |accessdate=2010-07-24}}</ref>。战士由主任拉扎德负责操作，拉扎德是神罗总裁的私生子。扎克斯还和塔克斯关系友好，其中特别是其首领曾，以及女塔克斯希斯内。在游戏进程中，扎克斯遇到在米德加贫民区一间破损教堂照看花的女子[[最终幻想VII系列角色列表#艾莉絲·蓋恩斯巴勒|艾丽丝·盖恩斯巴勒]]，二人关系逐渐亲近。他还和神罗士兵[[克勞德·史特萊夫|克劳德]]成为好友<ref>{{cite web | url=http://www.ign.com/articles/2008/03/07/crisis-core-character-profiles-revisited|title=Crisis Core: Character Profiles Revisited|publisher=IGN |author=Clements, Ryan|page=1 |accessdate=2010-07-24}}</ref><ref>{{cite web | url=http://www.ign.com/articles/2008/03/07/crisis-core-character-profiles-revisited?page=2|title=Crisis Core: Character Profiles Revisited|publisher=IGN |author=Clements, Ryan|page=2|accessdate=2010-07-24}}</ref>。

== 开发 ==

《核心危機》的構思源自田畑端被选为一款待出品PlayStation Portable作品的总监时。在同野村哲也和[[北濑佳范|北濑佳范]]讨论后，田畑决定游戏应该是另一款最终幻想VII补完计划作品，他认为给畅销系列制作游戏的压力能同时激发职员和他本人的积极性<ref name="oneup" />。他们最初设想的是制作一款PlayStation Portable移植版《危机之前 -最终幻想VII-》，因为它已经為移动电话再发行，而且工作人員想要拓展游戏。然而在决定扎克斯担当新游戏的主人公后，便放棄了《危机之前》移植的想法<ref name="ukign" />。

北瀨想確保遊戲超出愛好者的期望，因此他決定不將遊戲製作為《最終幻想VII》的「致敬物」，而和其他作品都是最終幻想VII補完計畫的重要一環<ref name="ukign" />。類似的，对于制作贯穿的更改“《最终幻想VII》标志人物的爱好者认识”，田畑称他和工作人员都非常谨慎。比如《最终幻想VII》主要背景事件之一的尼布尔海姆毁灭在[[原创动画录影带|OVA]]《[[最终命令_-最终幻想VII-|最终命令 -最终幻想VII-]]》中做了一定变动，因爱好者对此变动反应消极，游戏职员决定《核心危机》中不做大的改变<ref name="ukign">{{cite web | url=http://www.ign.com/articles/2008/04/28/crisis-core-final-fantasy-vii-uk-interview |title=Crisis Core: Final Fantasy VII UK Interview |work=IGN |first=Dave |last=McCarthy |accessdate=2009-03-08 |page=1}}</ref>。然而他们在涉及这些角色“年轻时候”时，还是动了大手术，因此他们得以在不大幅改变角色的情况下加入新元素<ref name="oneup" />。

选扎克斯担任主人公的主因之一是他的命运已经预先决定了<ref name="oneup">{{cite web | author=Mielke, James | date=2008-03-19| title=Final Fantasy VII: Crisis Core (PSP) Preview| url=http://www.1up.com/previews/crisis-core | publisher=[[1UP.com|1UP.com]]| accessdate=2010-07-22}}</ref>。北濑称扎克斯的故事“已经筹划10年了”，虽然他在《最终幻想VII》中只是次要角色，但野村已经制作了概念图，野岛一成也编写了故事。游戏最初打算用更多场景描绘扎克斯和克劳德在逃往米德加的路上，以便详述他们的友情和未完成的计划，但是这些场景因[[UMD|UMD]]的限制而移除，因此工作人员更加注重于扎克斯身为一名强大战士的背景<ref name="triviadev" />。在制作人今泉英树和杰尼西斯日版配音演员[[Gackt|Gackt]]讨论后，增补了杰尼西斯的性格，今泉对角色在《[[地獄犬的輓歌_-最終幻想VII-|地狱犬的挽歌 -最终幻想VII-]]》秘密结局中的简短登场印象深刻，并认为扩展角色的余地非常大<ref name="ignpage2" />。萨菲罗斯角色特别被编写的让他展示“更有人性的一面”<ref name="ignpage2">{{cite web | url=http://www.ign.com/articles/2008/04/28/crisis-core-final-fantasy-vii-uk-interview?page=2 |title=Crisis Core: Final Fantasy VII UK Interview |work=IGN |first=Dave |last=McCarthy |accessdate=2009-03-08 |page=2}}</ref>。游戏标识代表各个主角：蓝天象征扎克斯，白羽毛象征安吉尔，水象征爱丽丝<ref>{{cite book |title=Crisis Core: Final Fantasy VII 10th Ultimania |year=2007|isbn=978-4-7575-2126-1 |language=ja | publisher=Square-Enix |pages=}}</ref>。

最初设想的《核心危机》是动作游戏，但因几乎所有工作职员都较有设计[[电子角色扮演游戏|角色扮演游戏]]的经验，他们最终决定改变，制作和标准角色扮演游戏更相似的战斗系统<ref name="triviadev">{{cite web | author=Gantayat, Anoop| date=2010-10-14| title=The 3rd Birthday Has Skippable Event Scenes| url=http://andriasang.com/comq6w/3rd_birthday_and_crisis_core/ | publisher=Amdriasang | accessdate=2010-10-14}}</ref>。但他们为战斗加入了更多动作导向的元素，使游戏较传统RPG更多趋向动作角色扮演游戏。魔石系统的设计使玩家可以选择“朝角色扮演游戏方向强化”和“朝动作方向强化”，并帮助游戏的平衡性。此外，数码情感波动系统为游戏系统加入运气元素，同时冲淡了战斗重复的感觉<ref name="oneup" />。野村和北濑想将此加入游戏，因为他们酷爱[[彈珠機|弹珠机]]<ref name="ukign" />。

| publisher=[[GameSpot|GameSpot]] | accessdate=2008-05-22}}</ref>，以仅次于《[[战神：奥林匹斯之链|战神：奥林匹斯之链]]》34.05万的销量成为三月第二畅销PlayStation Portable游戏，并成为全平台第6畅销游戏<ref name="NPDmarch" />。史克威尔艾尼克斯宣布，到2009年3月31日时《核心危机》全球售出310万套，其中日本83万<ref>{{cite web | title=Results Briefing: Fiscal Year ended May 31, 2009 | date=2009-05-19 | url=http://www.hd.square-enix.com/eng/pdf/news/20090525_01en.pdf#8 | work=Square-Enix.com | accessdate=2009-07-21}}</ref>。游戏在史克威尔艾尼克斯2009财年售出84万，其中欧洲55万<ref>{{cite web | url = http://www.siliconera.com/2009/05/21/what-square-enix-games-sold-well-last-year/ | title = What Square Enix Games Sold Well Last Year? | publisher = Siliconera | author = Spenser | date = 2009-05-21 | accessdate = 2009-05-21 }}</ref>。作为对游戏销量的回应，史克威尔艾尼克斯授予《核心危机》他们当年最佳PSP游戏称号<ref>{{cite web | url = http://www.mcvuk.com/news/read/crisis-core-biggest-psp-release-of-2008| title = Crisis Core 'biggest PSP release of 2008'| publisher = [[Market_for_Home_Computing_and_Video_Games|Market for Home Computing and Video Games]] | author = Ingh".am, Tim | date = 2008-06-13 | accessdate = 2010-08-25 }}</ref>，称其为“不可思议的成功”。史克威尔英国销售总监道格·博恩称其为“2008年必备PSP游戏”<ref>{{cite web | url = http://www.mcvuk.com/news/read/square-enixs-crisis-core-reigns-supreme-over-psp-chart | title = Square Enix's Crisis Core reigns supreme over PSP chart| publisher = [[Market_for_Home_Computing_and_Video_Games|Market for Home Computing and Video Games]] | author = Harker, Jonathon | date = 2008-08-29 | accessdate = 2010-08-25 }}</ref>。

《核心危机》普遍获得积极评价。游戏在[[GameRankings|GameRankings]]的评论综合得分为82%<ref name="gr" />，而在[[Metacritic|Metacritic]]的得分为83<ref name="meta" />。日本游戏杂志《[[Fami通|Fami通]]》上的游戏总分为35/40，四个独立评分为9/9/8/9<ref name="maxconsole2007" />。[[GameSpot|GameSpot]]授予游戏“编辑推荐”称号，称赞其故事、战斗系统及其呈现，并称“《核心危机》是一次激动且令人沉痛的旅行，所有角色扮演游戏爱好者都应该展开这旅程”<ref name="gspot" />。[[IGN|IGN]]的莱安·克莱门茨因游戏聚焦于扎克斯的成长及他同战士们的友情，而称赞游戏有别于其他角色扮演游戏。他称“《核心危机》是一款很棒的游戏，除了各处的一些小问题，它不会令人失望”。游戏还进入PlayStation Portable平台推荐游戏的“编辑精选”栏<ref name="ignreview" />。[[GameSpy|GameSpy]]也给游戏好评，称其“胜任致敬部分，并将传统设计的精华和全新的技术独到结合，协调汇集了将一些旧物和一些新品”。GameSpy也称一些玩家可能不喜欢扎克斯，“你将对角色不跟着你的冒险成长而相当厌倦”<ref name="Gspy" />。
[[VideoGamer.com|VideoGamer.com]]给了游戏9分的总评，认为其战斗系统“使人上瘾”，并提出CGI场景动画和《最终幻想VII -降临之子-》的质量相似<ref name="videogamereview" />。《[[GamePro|GamePro]]》也给游戏相似的积极评价，称其为“视效最好的PSP作品”，并称赞其同补完计划的元素结成一体，并同时创造了新方面<ref name="gamepro" />。[[GamesRadar|GamesRadar]]的AJ·格拉瑟称有了DMW系统，游戏将会简单，让“半熟玩家”也尝试以困难模式玩游戏。格拉瑟还认为游戏足够讨人喜爱，甚至让不喜欢最终幻想VII系列的玩家也开始喜欢它<ref name="gradar" />。《[[Play_(英国杂志)|Play]]》的塞缪尔·罗伯茨称游戏为2008年最佳游戏<ref>{{cite web |url=http://www.play-mag.co.uk/opinion/crisis-core-final-fantasy-vii-%e2%80%93-my-game-of-the-year/ |title=Crisis Core: Final Fantasy VII – my game of the year |work=[[Play_(UK_magazine)|Play]]|accessdate=2010-09-11|author=Roberts, Samuel}}</ref>。

许多评论还提到了游戏和《最终幻想VII》的关系。[[1UP.com|1UP.com]]称其为史上最佳前传之一，提出游戏“更好的将玩家放入《FFVII》的世界，甚至超越了原作所为”<ref name="oneupreview" />。而[[GameTrailers|GameTrailers]]称《核心危机》“并非省事与复制资源”，他们认为游戏和《地狱犬的挽歌》不同，更能吸引玩家<ref name="Gt" />。《[[电脑与电子游戏|电脑与电子游戏]]》称虽然游戏主线情节只有12小时，但支线任务帮游戏加长了时间。他们还称赞战斗系统“永不无聊”，同时《最终幻想VII》漫长游荡和过长中心的缺点在本作不是问题<ref name="cvg" />。

然而游戏获得一些负面评价。虽然[[Eurogamer|Eurogamer]]称其为最佳《最终幻想VII》衍生作品，但也批评称“对于其针对的26、28、30岁者，游戏除了完美的伤感外，鲜有演出其它内容”<ref name="eurogamereview" />。虽然[[PALGN|PALGN]]将DWM系统称为“简便的工具”，但同时认为这是游戏的最大弱点<ref name="palgn" />。[[X-Play|X-Play]]给游戏打出2/5，对无法跳过的过场、欠佳的对话、反复的游戏性以及薄弱的剧情表示不满<ref name="xplay">{{cite web | author=D.F.
