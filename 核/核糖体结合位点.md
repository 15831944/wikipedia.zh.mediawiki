'''核糖体结合位点'''（{{lang-en|Ribosome-binding site}}），簡稱RBS，是位於[[信使核糖核酸|信使核糖核酸]]（mRNA）[[起始密碼子|起始密碼子]]上游，在[[轉譯|轉譯]]起始時可結合[[核糖體|核糖體]]的序列。

==原核生物==
[[原核生物|原核生物]]的核糖體結合位點為[[夏因-达尔加诺序列|夏因-达尔加诺序列]]（SD序列），其序列為5'-AGGAGG-3'<ref name=":6">{{Cite journal|title = Determinant of cistron specificity in bacterial ribosomes|url = http://www.nature.com/nature/journal/v254/n5495/abs/254034a0.html|journal = Nature|date = 1975-03-06|pages = 34–38|volume = 254|issue = 5495|doi = 10.1038/254034a0|first = J.|last = Shine|first2 = L.|last2 = Dalgarno|pmid=803646}}</ref>，[[核糖體|核糖體]]小次單元中的[[16S_核糖体RNA|16S 核糖体RNA]]（16s rRNA）則有與之互補的[[反夏因-達爾加諾序列|反夏因-達爾加諾序列]]（5'-CCUCCU-3'），[[原核轉譯|轉譯]]起始時，這兩個序列可透過[[核鹼基|鹼基]]間的[[氫鍵|氫鍵]]相互結合<ref name=":0">{{Cite web|title = Ribosomal Binding Site Sequence Requirements|url = https://www.thermofisher.com/ca/en/home/references/ambion-tech-support/translation-systems/general-articles/ribosomal-binding-site-sequence-requirements.html|website = www.thermofisher.com|accessdate = 2015-10-16}}</ref><ref name=":1">{{Cite web|title = Help:Ribosome Binding Site - parts.igem.org|url = http://parts.igem.org/Help:Ribosome_Binding_Site|website = parts.igem.org|accessdate = 2015-10-16}}</ref>。

==真核生物==
核糖体自组装发生之后，在Kozak序列内部的起始密码子处翻译起始。因为[[Kozak序列|Kozak序列]]并未参与[[核糖体自组装|核糖体自组装]]，它通常不被视为核糖体结合位点。<ref >{{Cite web|title = Ribosomal Binding Site Sequence Requirements|url = https://www.thermofisher.com/ca/en/home/references/ambion-tech-support/translation-systems/general-articles/ribosomal-binding-site-sequence-requirements.html|website = www.thermofisher.com|accessdate = 2015-10-16}}</ref>
