'''核足球'''（{{lang-en|Nuclear football}}）是一種特殊的通信工具，外表是個黑色手提皮包，內藏的工具允許[[美國總統|美國總統]]即使在遠離指揮中心（如[[白宮戰情室|白宮戰情室]]）的情況下亦能授權發動核攻擊<ref name="cn.nytimes@20180104">{{Cite web |url = https://cn.nytimes.com/asia-pacific/20180104/nuclear-button-trump-north-korea/zh-hant/ |title = 揭秘「核按鈕」：首先，並沒有什麼按鈕 |author = Russell Goldman |date = 2018-01-04 |publisher = 紐約時報中文網 |accessdate = 2018-03-31 |deadurl = no |archiveurl = https://web.archive.org/web/20180328093050/https://cn.nytimes.com/asia-pacific/20180104/nuclear-button-trump-north-korea/zh-hant/ |archivedate = 2018-03-28 }}</ref>。作為美國戰略防禦系統的移動樞紐，它由五名[[侍從官|副官]]24小時輪流保管，而他們正是代表[[美軍|美國武裝部隊]]的五個組成部分<ref name="cn.nytimes@20180104"/><ref>{{cite web |url = http://www.js.xinhuanet.com/xin_wen_zhong_xin/2010-03/29/content_19368676.htm |title = 美国总统常忘带“核按钮手提箱” |publisher = 新华网 |date = 2010年3月29日 |accessdate = 2013-03-30 |deadurl = yes |archiveurl = https://web.archive.org/web/20140124072413/http://www.js.xinhuanet.com/xin_wen_zhong_xin/2010-03/29/content_19368676.htm |archivedate = 2014年1月24日 }}</ref>。

正常情況下，美國總統會在白宮戰情室直接下達核攻擊命令，只有在外出時才有機會用到「核足球」，例如乘坐[[空軍一號|空軍一號]]、[[海軍陸戰隊一號|海軍陸戰隊一號]]或總統車隊的時候。

== 歷史 ==
「核足球」的歷史可以追溯到[[艾森豪威爾|艾森豪威爾]]時期，但其使用方式是在[[古巴導彈危機|古巴導彈危機]]後才出現，主要因為時任總統[[約翰·甘迺迪|約翰·甘迺迪]]擔心處於古巴的蘇聯指揮官會繞過莫斯科擅自發射導彈<ref name="The Football"/>，甚至美軍可能於未經由其批准的情况下擅自使用核武器攻擊蘇聯<ref>{{cite web |author=Michael Evans |url=http://www2.gwu.edu/~nsarchiv/nsa/cuba_mis_cri/index.htm |title=The Cuban Missile Crisis, 1962: A Political Perspective After 40 Years |publisher=www.gwu.edu |date= |accessdate=2018-04-01 |deadurl=no |archiveurl=https://web.archive.org/web/20150226135433/http://www2.gwu.edu/~nsarchiv/nsa/cuba_mis_cri/index.htm |archivedate=2015-02-26 }}</ref>。肯尼迪曾經詢問了幾個有關發射美國核武器的問題，包括<ref>{{Cite book | author = Marc Trachtenberg | title = A Constructed Peace: The Making of the European Settlement, 1945–1963 | location = Princeton | publisher = Princeton University Press | date = 1999 | accessdate = 2018-04-01 | language = en}}</ref>：
* 假設我收到來自受到嚴密保護的消息來源的信息，令我得出美國應立即對[[東方集團|東方集團]]發動核打擊的結論。參謀長聯席會議的緊急行動檔案會否允許我發動此類攻擊，而無需事先諮詢國防部長和/或參謀長聯席會議？
* 我知道我的電話座機上的紅色按鈕能與白宮軍隊信號局（WHASA）的總機連接，而WHASA總機能立即將我接駁至聯合作戰室。如果我在沒有提前通知下打電話至聯合作戰室，我會向誰講話？
* 我會如何向聯合作戰室表示直接發動直接核打擊？
* 接受我的指示的人如何去驗證它們？

美聯社的一篇文章指出，「足球」這個綽號是源於代號「Dropkick」的攻擊計劃<ref name="USA Today@20050505"/>。

在[[吉米·卡特|吉米·卡特]]及[[羅納德·雷根|羅納德·雷根]]擔任總統期間，他們都傾向於將發射代碼保存在他們的夾克口袋內<ref>{{Cite book | author = Ronald Reagan | title = An American Life | location = New York | publisher = Simon and Schuster | date = 1990 | pages = 257 | ISBN = 0743400259 | accessdate = 2018-04-01 | language = en}}</ref>。前任美國眾議院議員{{link-en|約翰·克林因|John Kline (politician)}}在美國海軍陸戰隊擔任上校期間，曾經成為卡特及列根的副官並攜帶「核足球」<ref>{{Cite web |url = http://www.washingtonpost.com/wp-dyn/content/article/2009/07/12/AR2009071202298.html |title = Key Republican Ready to Roll Back Testing Mandates of "No Child Left Behind" |author = Nick Anderson |date = 2009-07-13 |publisher = The Washington Post |language = en |accessdate = 2018-04-01 |deadurl = no |archiveurl = https://web.archive.org/web/20171203162539/http://www.washingtonpost.com/wp-dyn/content/article/2009/07/12/AR2009071202298.html |archivedate = 2017-12-03 }}</ref>。

在[[列根遇刺案|1981年的企圖暗殺]]發生後，這張代碼卡曾與列根分開过<ref name="BBC@19990426">{{Cite web |url = http://news.bbc.co.uk/2/hi/americas/328442.stm |title = Clinton drops nuclear football |date = 1999-04-26 |publisher = BBC News |language = en |accessdate = 2018-04-01 |deadurl = no |archiveurl = https://web.archive.org/web/20180107062249/http://news.bbc.co.uk/2/hi/americas/328442.stm |archivedate = 2018-01-07 }}</ref>。当时創傷小組在急救室剪開列根的衣服並進行檢查時，卡片便掉落至他其中一隻在急救室地板上的鞋內，後來才被人發現。由於当时列根昏迷不醒，已經無法行使總統的職責，所以攜帶「核足球」的副官按規定立即返回白宮待命<ref>{{Cite web |url = http://www.people.com.cn/GB/guoji/20010323/424020.html |title = 美國差點啟動核提箱 |date = 2001-03-23 |publisher = 人民網 |accessdate = 2018-04-01 |deadurl = no |archiveurl = https://web.archive.org/web/20050329020151/http://www.people.com.cn/GB/guoji/20010323/424020.html |archivedate = 2005-03-29 }}</ref>。另外在1973年6月，時任總統尼克遜在[[大衛營|大衛營]]向到訪的蘇聯領導人[[列昂尼德·伊里奇·勃列日涅夫|列昂尼德·伊里奇·勃列日涅夫]]送上一部「林肯大陸」房車，其後布里茲尼夫突然駕駛這部房車並與尼克遜一同前往鄰近的高速公路，負責保護尼克遜的特勤局人員則被留於大衛營，而尼克遜與負責攜帶「核足球」的副官分開接近30分鐘<ref>{{Cite book | author = Peter Radetsky | title = The Soviet Image: A Hundred Years of Photographs from Inside the TASS Archives | publisher = Chronicle Books | date = 2007 | pages = 188 | ISBN = 0811857980 | accessdate = 2018-04-01 | language = en}}</ref>。此外，時任總統的[[杰拉尔德·福特|杰拉尔德·福特]]、占美·卡特、[[乔治·赫伯特·沃克·布什|乔治·赫伯特·沃克·布什]]<ref>{{Cite web |url = http://legacy.sandiegouniontribune.com/news/world/20040604-0731-pope-bush-football.html |title = Bush's nuclear 'football' in Vatican hallowed halls |author = Philip Pullella |date = 2004-06-04 |language = en |accessdate = 2018-04-01 |deadurl = no |archiveurl = https://web.archive.org/web/20170913073610/http://legacy.sandiegouniontribune.com/news/world/20040604-0731-pope-bush-football.html |archivedate = 2017-09-13 }}</ref>及[[比爾·克林頓|比爾·克林頓]]<ref name="BBC@19990426"/>都曾經與「核足球」分開一段時間。

== 設計 ==
根據[[華盛頓郵報|華盛頓郵報]]的一篇文章指出，總統總是由一名副官陪同，而這名副官則攜帶著附有核武器發射代碼的「足球」<ref>{{Cite web |url = http://www.washingtonpost.com/wp-dyn/content/article/2008/12/21/AR2008122100869.html |title = Cheney, Biden Disagree on VP's Power in Separate TV Interviews |author = Dan Eggen |date = 2008-12-22 |publisher = 華盛頓郵報 |language = en |accessdate = 2018-03-31 |deadurl = no |archiveurl = https://web.archive.org/web/20170901132941/http://www.washingtonpost.com/wp-dyn/content/article/2008/12/21/AR2008122100869.html |archivedate = 2017-09-01 }}</ref>。「足球」是由[[零·哈里伯頓|零·哈里伯頓]]專門製造，外表為一個黑色柔軟皮包，皮革底下縫有由鈦金屬<ref >{{cite news |url = http://www.sltrib.com/business/ci_2779895 |title = Security: Sleek, sexy and oh, so safe |author = Glen Warchol |newspaper = The Salt Lake Tribune |date = 2005-06-05 |accessdate = 2018-03-31 |archiveurl = https://web.archive.org/web/20150924103049/http://www.sltrib.com/business/ci_2779895 |archivedate = 2015-09-24}}</ref>。皮包的總重量約45磅（20公斤），安裝有密碼鎖<ref name="USA Today@20050505">{{Cite web |url = https://usatoday30.usatoday.com/news/washington/2005-05-05-nuclear-football_x.htm |title = Military aides still carry the president's nuclear 'football' |author = USA Today |date = 2005-05-05 |language = en |accessdate = 2018-04-01}}</ref>。另外，在皮包近提把位置有一支小型天線突出<ref name="USA Today@20050505"/>。

白宮軍事辦公室前主任{{link-en|華倫·比爾·格里|Warren Gulley}}在其《Breaking Cover》一書中寫道<ref name="USA Today@20050505"/>：
{{quote|「足球」內藏四份文件，包括：一份被稱為「黑色手冊」，羅列了四種核武器攻擊程度不同的行動方案，記錄了美國的核彈數目、部署地點及發射預定路綫等內容的文件；一份秘密基地名單，羅列了在緊急情況下可供美國總統使用的秘密基地的名單；一份有關緊急廣播程序、裝有八至十頁紙的文件夾，當中記錄了進行核武器攻擊後，美國總統在進行全國廣播的程序；一張3×5英吋的認證卡，記載有美國總統的認證號碼，此卡未被確認時，以美國總統名義下達的任何命令均為無效。}}

== 使用 ==
[[File:VOA_How_a_Nuclear_Strike_Is_Authorized_in_the_United_States.webm|thumb]]
在發生緊急情況時，當總統決定下令使用核武器，副官會把總統帶到一邊並打開手提箱，其後向[[美國參謀長聯席會議|參謀長聯席會議]]發出一個指揮信號或「警戒」警報。然後，總統將與副官一同審查攻擊預案並從中選擇合適的方案，預案範圍包括發射一枚巡航導彈，甚至是發射全部洲際導彈。這些預先製定的戰爭計劃是根據OPLAN 8010（以前稱為{{link-en|單一綜合作戰計劃|Single Integrated Operational Plan}}）而開發<ref>{{Cite web |url = http://www.thedrive.com/the-war-zone/9056/heres-americas-plan-for-nuking-its-enemies-including-north-korea |title = Here's America's Plan for Nuking its Enemies, Including North Korea |author = Joseph Trevithick |date = 2017-04-07 |language = en |accessdate = 2018-04-01 |deadurl = no |archiveurl = https://web.archive.org/web/20180101202443/http://www.thedrive.com/the-war-zone/9056/heres-americas-plan-for-nuking-its-enemies-including-north-korea |archivedate = 2018-01-01 }}</ref>。在經過雙人驗證程序後，代碼會被輸入至{{link-en|啟動連接裝置|Permissive Action Link}}。

在軍方處理此命令前，總統需要使用手提箱內的認證卡以確認身份，而這張認證卡的綽號為「餅乾」<ref name="Jason-Fritz@2009">{{cite journal |author = Jason Fritz |date = 2009 |title = Hacking Nuclear Command and Control |url = http://www.icnnd.org/Documents/Jason_Fritz_Hacking_NC2.pdf |journal = International Commission on Nuclear Non-proliferation and Disarmament |access-date = 2018-04-01 |deadurl = no |archiveurl = https://web.archive.org/web/20180401195228/http://www.icnnd.org/Documents/Jason_Fritz_Hacking_NC2.pdf |archivedate = 2018-04-01 }}</ref><ref>{{Cite web |url = https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%9C%8B%E9%9A%9B/134137/%E7%BE%8E%E5%9C%8B%E6%9C%83%E8%A8%8E%E8%AB%96%E9%99%90%E5%88%B6%E7%B8%BD%E7%B5%B1%E6%8C%89%E4%B8%8B%E6%A0%B8%E6%AD%A6%E6%8C%89%E9%88%95-%E7%89%B9%E6%9C%97%E6%99%AE%E7%99%BC%E5%8B%95%E6%A0%B8%E6%88%B0%E6%A9%9F%E6%9C%83%E6%88%96%E6%B8%9B%E4%BD%8E |title = 美國會討論限制總統按下核武按鈕　特朗普發動核戰機會或減低 |author = 黃家欣 |date = 2017-11-16 |publisher = 香港01 |language = zh |accessdate = 2018-04-01}}</ref>。基於美國的核發射設施採用雙人管理制度，只有總統能下令發射核武器，但此命令必須由[[美國國防部長|國防部長]]核實是總統本人發出的確實命令（如果總統不幸在襲擊中身亡，則按[[美國總統繼任順序|繼任順序]]依序遞補）。此驗證過程僅驗證命令是否由總統本人發出，國防部長並沒有否決權，而且必須遵守總統的命令<ref name="Jason-Fritz@2009"/>。當成功驗證所有代碼，軍方便會向適當單位發佈攻擊命令。但有意見認為，總統幾乎擁有一次啟動核武攻擊的權力，而國防部長只是需要核實其命令的真確性，卻無法在法律上否決命令<ref>{{Cite web |url = https://www.vox.com/platform/amp/2016/8/3/12367996/donald-trump-nuclear-codes |title = If President Trump decided to use nukes, he could do it easily |author = Zack Beauchamp |date = 2016-08-03 |publisher = Vox |language = en |accessdate = 2018-04-01 |deadurl = no |archiveurl = https://web.archive.org/web/20180211094541/https://www.vox.com/platform/amp/2016/8/3/12367996/donald-trump-nuclear-codes |archivedate = 2018-02-11 }}</ref><ref>{{Cite web |url = https://www.politico.com/magazine/story/2016/06/2016-donald-trump-nuclear-weapons-missiles-nukes-button-launch-foreign-policy-213955 |title = What Exactly Would It Mean to Have Trump’s Finger on the Nuclear Button? |author = Bruce Blair |date = 2016-06-11 |publisher = Politico Magazine |language = en |accessdate = 2018-04-01 |deadurl = no |archiveurl = https://web.archive.org/web/20180222174322/https://www.politico.com/magazine/story/2016/06/2016-donald-trump-nuclear-weapons-missiles-nukes-button-launch-foreign-policy-213955 |archivedate = 2018-02-22 }}</ref><ref>{{Cite web |url = https://www.nytimes.com/2016/08/05/science/donald-trump-nuclear-codes.html?_r=1 |title = Debate Over Trump’s Fitness Raises Issue of Checks on Nuclear Power |author = William J. 「核足球」由五名副官24小時輪流保管，而他們皆來自五大軍種。副官是美國軍方的軍官，工資級別為O-4或以上，並通過了全國最嚴格的背景調查（Yankee White{{noteTag|1=「Yankee White」是一個行政綽號，指一些通過了背景調查而且與美國總統及副總統一起共事的美國國防部人士和承包商僱員。}}）<ref name="The Football">{{Cite web |url = https://www.globalsecurity.org/wmd/systems/nuclear-football.htm |title = The Football |language = en |accessdate = 2018-04-01 |deadurl = no |archiveurl = https://web.archive.org/web/20171109000858/https://www.globalsecurity.org/wmd/systems/nuclear-football.htm |archivedate = 2017-11-09 }}</ref><ref name="Stephen P. 記者{{link-en|羅恩·羅森巴姆|Ron Rosenbaum}}指出，指揮系統能驗證指揮官的身份及命令的真實性從而展開核攻擊的作戰計劃，但卻沒有任何保障措施來驗證發佈命令的人實際上是否理智<ref name="Ron Rosenbaum@20110228">{{Cite web |url = http://www.slate.com/articles/life/the_spectator/2011/02/an_unsung_hero_of_the_nuclear_age.single.html |title = An Unsung Hero of the Nuclear Age |author = Ron Rosenbaum |date = 2011-02-28 |publisher = Slate |language = en |accessdate = 2018-04-01 |deadurl = no |archiveurl = https://web.archive.org/web/20111014232809/http://www.slate.com/articles/life/the_spectator/2011/02/an_unsung_hero_of_the_nuclear_age.single.html |archivedate = 2011-10-14 }}</ref>。值得注意的是，少校{{link-en|哈羅德·赫林|Harold Hering}}因在1973年提問：「我如何知道我收到的核導彈發射命令是來自一個理智的總統？」而被空軍解僱，而當時擔任總統的是[[理查德·尼克松|理查德·尼克松]]<ref name="Ron Rosenbaum@20110228"/>。
