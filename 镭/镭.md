'''镭'''（舊譯作'''鈤'''{{NoteTag|1903年10月10日：[[魯迅|魯迅]]以筆名“自樹”在[[東京|東京]]出版的《[[浙江潮|浙江潮]]》月刊第八期上首次發表介紹鐳的文章《說鈤》。但“鈤”亦為化學元素[[鍺|鍺]]的舊譯，因造成混淆而採用新譯。}}、'''銧'''）是一种化学[[元素|元素]]，它的[[化学符号|化学符号]]是'''Ra'''，它的[[原子序数|原子序数]]是88，是一种银白色的[[碱土金属|碱土金属]]，带有[[放射性|放射性]]，而且十分贵重，每克约100美金。

镭在1898年由[[玛丽亚·居里|居里夫人]]及她丈夫[[皮埃尔·居里|皮埃尔·居里]]在[[捷克|捷克]]北[[波希米亚|波希米亚]]发现。他们发现[[铀|铀]]在衰变后，衰变物仍带放射性。镭的拼音名称''Radium''即是放射性的意思。

镭-226為鐳的最穩定同位素，[[半衰期|半衰期]]為1600年，进行α-蜕变，放出[[α射线|α射线]]和[[γ射线|γ射线]]。它衰变时会放出[[氡|氡]]气到大气中。氡仍有放射性，且可被生物吸入，危害生命。

鐳和鈣屬於同一主族，有著相似的性質。身體在攝取到鐳以後，無法區分鐳和鈣，誤將鐳當作鈣，合成了骨細胞。這些含有鐳的骨細胞中的鐳，緩慢地發生衰變，而衰變過程中發射出的中子，就像砲彈一樣，殺傷了細胞。而恰巧，人類的下頜骨是骨細胞中更新最快的，所以也更容易富集被鐳元素替換過的骨細胞。所以，下頜骨也是受放射損傷最嚴重的部分。於是就有了鐳毒頜顏。

镭能够致癌，但是它也能够治療癌症。<ref>{{Cite web|url = http://nehrc.nhri.org.tw/toxic/toxfaq_detail.php?id=156|title = 国家卫生研究院国家环境毒物中心-镭|accessdate = 2015-09-04|author = |date = |publisher = }}</ref>


== 相關化學反應 ==
鐳在空氣中相當不穩定，很容易與[[氮|氮]]气和[[氧|氧]]氣發生反應，生成[[氮化鐳|氮化鐳]]和[[氧化鐳|氧化鐳]]：

3 Ra + N<sub>2</sub> → Ra<sub>3</sub>N<sub>2</sub>

2 Ra + O<sub>2</sub> → 2RaO

鐳也可以和[[水|水]]發生反應，生成[[氫氧化鐳|氫氧化鐳]]：

Ra + 2 H<sub>2</sub>O → Ra(OH)<sub>2</sub> + H<sub>2</sub> ↑
