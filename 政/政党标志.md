'''政党标志'''包括党旗、党徽、党歌、代表色、吉祥物以及[[象征物|象征物]]等不同类型的[[标志|标志]]。除官方标志外，亦可能是约定俗成而产生，例如做为美国[[共和黨_(美國)|共和党]]、[[民主党_(美国)|民主党]]两党标志的[[驴象之争|驴象]]即源于美国政治漫画家[[托马斯·纳斯特|托马斯·纳斯特]]的创作<ref>{{Cite web |url=http://epaper.gmw.cn/zhdsb/html/2016-12/07/nw.D110000zhdsb_20161207_2-18.htm |title=漫话漫画“驴象之争”之父 |accessdate=2017-10-31 |author=李亚航 |date=2016-12-07 |work= |publisher=光明网，来源：《中华读书报》（2016年12月07日18版）|language=Zh-hans }}</ref>。

== 党旗、党徽 ==

党旗和党徽是常见的政党标志，但不是所有政党都会使用这类标志。[[中華人民共和國政黨|在中国大陆境内，合法政党中]]，除[[中国共产党|中国共产党]]拥有[[中國共產黨黨旗黨徽|党旗、党徽]]外，其余[[八大民主党派|八大民主党派]]均未使用党旗、党徽<ref>{{Cite web |url=http://news.ifeng.com/opinion/detail_2006_08/17/1318676_0.shtml |title=中国民主党派的出路与未来 |accessdate=2017-10-31 |author=记者：欧阳斌 |date=2006-08-17 |work= |publisher=凤凰网|language=Zh-hans }}</ref>。

在[[一党专政|一党专政]]的国家中，尤其是[[黨國體制|黨國體制]]下，[[执政党|执政党]]的政党标志与[[国家标志|国家标志]]之间有着高度相关性。两者同时出现时，又有着敏感性。2017年[[庆祝中国人民解放军建军90周年阅兵|庆祝中国人民解放军建军90周年阅兵]]中，因[[中国共产党党旗|中国共产党党旗]]领先于[[中华人民共和国国旗|中华人民共和国国旗]]，备受争议。

== 代表色 ==

选择何种代表色，通常由自身历史或党派倾向决定。譬如，传统左翼政党使用[[红色|红色]]作为代表色。

== 吉祥物  ==

与党旗、党徽相比，使用吉祥物，主要是为在民众中塑造亲切、友善的政党形象。至于所起的作用，各方有不同的观点<ref>{{Cite web |url=http://www.zaobao.com.sg/special/report/singapore/ge/story20101028-136092 |title=两政党推出 吉祥物贴近选民 政治观察家：能引起注意但不能赢选票 |accessdate=2017-10-31 |author=游润恬 |date=2010-10-28 |work= |publisher=联合早报网|language=Zh-hans }}</ref>。
