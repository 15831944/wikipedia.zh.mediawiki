在大多數[[共產黨|共產黨]][[一黨執政|一黨執政]]的國家，'''政治正確'''是一個重要且具影響力的概念。它的基本涵義是：「做任何事（包括政治層面和非政治層面），都要保證[[意識形態|意識形態]]本質符合執政黨與政府的規定（所謂的「路線、方針、政策」）。」

然而，究竟什麼是「符合規定」，如何做到「符合規定」，一般沒有明確的定義，不同時期的規定也不盡相同，甚至完全相反。

在某些[[極權社會|極權社會]]時期，如[[中華人民共和國|中華人民共和國]]的[[文革|文革]]時期，以至現時由[[金正恩|金正恩]]家族統治的[[朝鲜|朝鲜]]，「政治正確」幾乎是每位國民都需要時時注意的教條。

在一般時期，「政治正確」對以下人員是特別重要的事情：宣傳機構的從事人員、[[媒體|媒體]]的審查者、某些政府官員（公務員）和共產黨官員等。政治正確甚至影響學術界領域，如要求文章語氣必需在馬克思主義的思維框架中論述，不可公然對顛撲不破的真理逾矩，類似古代要求文章內容須合於先王聖道之言，而非不受規範。

另外，由政府強力推銷特定的意識形態（如[[愛國主義|愛國主義]]、[[雅各賓俱樂部|雅各賓派]]、[[纳粹党|納粹黨]]、[[法西斯主义|法西斯主義]]）也被視為是一種政治正確的行為表現，跟共產黨的政治正確有異曲同工之处。

== 中华人民共和国 ==
[[中华人民共和国|中华人民共和国]]的“正确的政治观”大多数时候指的是党的“路线、方针、政策”，如[[四项基本原则|四项基本原则]]。

在[[中蘇交惡|中蘇交惡]]時期，中國[[毛澤東|毛澤東]]政權曾經將前蘇聯[[赫魯晓夫|赫魯晓夫]]和[[勃列日涅夫|勃列日涅夫]]政權稱做「蘇修（[[社會帝國主義|社會帝國主義]]）」政權。至於其他社會主義政權，如[[古巴|古巴]]、[[東德|東德]]和[[朝鲜|朝鲜]]，出自於對歐美等西方列強的敵意，官方媒體都會将西方國家冠以「[[法西斯主義|法西斯主義]]（東德當局曾將[[柏林圍牆|柏林圍牆]]稱做「反法西斯保護牆」）」、「美帝」、「[[帝國主義|帝國主義]]集團」或「侵略者」等稱謂。

== 越南 ==
[[越南|越南]]曾在[[法越戰爭|法越戰爭]]、[[越南戰爭|越南戰爭]]和[[中越戰爭|中越戰爭]]等時期，先後同樣將法國、美國和中國等越南觀點之「侵略者」視為「帝國主義集團」；在中越戰爭以後，越南開始與中國為敵並與美國靠攏，而官方媒體大部分的敵意稱謂則多半針對中國。{{fact}}

== 傳媒用法 ==
{{Wikisourceshas|关于正确使用涉台宣传用语的意见|中共黨政軍機關企業學術機構團體旗歌及人員職銜統一稱謂實施要點}}
* 中国大陆的媒體（如[[新華社|新華社]]、[[央視|央視]]、[[人民日報|人民日報]]）與部分港澳地區的[[親中媒體|親中媒體]]（如[[大公報|大公報]]、[[文匯報|文匯報]]、[[香港商報|香港商報]]、[[澳門日報|澳門日報]]、[[澳廣視|澳廣視]]、[[亞洲電視|亞視]]、[[電視廣播有限公司|TVB]]等），凡是報導兩岸新聞時，都將中华民国各級官署和官職加引号或改变称法，如「總統府」、「立委」、「行政院長」、「內政部長」、「國防部長」分别称为“领导人办公室”、“民代”、“行政机构负责人”、“内政部门负责人”、“防务部门负责人”，以表示不承认中华民国政府的合法性。
* [[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]的官方媒體（如[[勞動新聞|勞動新聞]]、[[朝鮮中央電視台|朝鮮中央電視台]]）凡是提到[[大韓民國|大韓民國]]各級官署和官職時，一律將南韓方面稱之為「叛亂集團」或「南朝鲜[[傀儡政權|傀儡政權]]」。
