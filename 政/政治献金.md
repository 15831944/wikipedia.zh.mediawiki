'''政治獻金'''係指對從事競選活動或其他政治相關活動之個人或團體，無償提供之動產或不動產、不相當對價之給付、債務之免除或其他經濟利益。但黨費、會費或義工之服務不包括在內<ref>{{cite web |url=http://homepage.vghtpe.gov.tw/~ged/top2/top2_e13.htm |title=存档副本 |accessdate=2013-04-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20130402121725/http://homepage.vghtpe.gov.tw/~ged/top2/top2_e13.htm |archivedate=2013-04-02 }}</ref>。政治獻金是政治現今美式制度民主的現象，一方面認為是民主社會的[[政治參與|政治參與]]的[[人權|人權]]精神，確保政治活動公平及公正。但另一方面是由於選舉技術所造成的嚴重弊端來源。<ref>http://www.moi.gov.tw/dca/02people_003.aspx</ref>，清廉的政治人物也必須直接或間接利用政治獻金來選舉；許多國家有法令來判斷政治獻金是否為賄賂，大體而言，沒有[[對價|對價]]關係叫政治獻金、有對價關係但公開並合乎相關法令的叫[[遊說|遊說]]（許多民主國家有遊說法）、有對價關係且違法的叫[[賄賂|賄賂]]、更容易造成[[利益輸送|利益輸送]]及公共利益損失的問題，尤其是[[財團|財團]]給的政治獻金更需公平且明確的規範，但是2014年4月2日，美國最高法院判決限制選舉政治獻金的合計金額違反了[[美國憲法第一修正案|美國憲法第一修正案]]，也就是日後美國的選舉政治獻金金額將無上限。

==用途==
===用于选举===
'''選舉捐款'''只是政治獻金的一種形式。收取獻金者是[[候選人|候選人]]的團隊，目前多數選舉制度已被各國主要政黨塑造成必須投入鉅額資金，才能參與候選，參選的重要門檻就是有一定的財力（雖然不一定是錢最多的就能當選），就算沒有[[賄選|賄選]]等違法行為，買[[媒體|媒體]][[廣告|廣告]]、寄[[傳單|傳單]]、插[[旗|旗]]海、[[造勢大會|造勢大會]]、[[選舉日|選舉日]]出[[號外|號外]][[告急|告急]]，[[開支|開支]]都很大。[[參選者|參選者]]急需大量[[選舉經費|選舉經費]]，在此選舉制度下，政治獻金是必然的[[資金|資金]]來源。也造成政黨執政後，往往藉由權力，籌措下一次選舉資金。

==政治獻金與賄賂的分界==
在多數制度下，政治人物收的錢必須符合相當條件才能稱為政治獻金，尤其是商人給的錢的控制更為嚴格：例如收受外國捐款經常更嚴格甚至禁止，單一[[贊助|贊助]]不可以多於總捐款的一定比例，政治獻金的附加條件有嚴格限制甚至禁止（沒有附加條件為政治獻金、有附加條件但合法的稱為[[利益團體|利益團體]]對政治人物的[[遊說|遊說]]、有附加條件且不合理者為賄賂），例如菸商給政治人物的錢，若附加條件為「放寬菸害防制」則一律是為賄賂而非遊說（雖然不一定有相關法案）公務員及政治人物不得利用職務獲得政治獻金、也不得阻止他人獲得合法的政治獻金，當局規定賬目要公開示人等。

===實質影響力===
以往司法機關往往以有無直接決策權來定義是政治獻金或行賄，就發生很多民代收受不當利益後，藉質詢或審查預算的權力來向政府機關施壓，造成很大的不公平現象。近年來司法見解漸趨於實質影響力，亦即是否因為收受利益後向相關機關關說，導致相關機關改變原有模式，造成交付利益的行為人獲利(包括流程加快、非法變合法等)，就構成行賄，以杜絕這種隔層施壓的現象。例如<ref>{{Cite web |url=http://www.appledaily.com.tw/appledaily/article/headline/20130612/35078504/?utm_source=FB&utm_medium=MWeb_Share&utm_campaign=http%3A%2F%2Fwww.appledaily.com.tw%2Fappledaily%2Farticle%2Fheadline%2F20130612%2F35078504%2F |title=高志鵬收賄50萬 重判7.5年}}</ref>

由於政治人物接受政治獻金的管道很多，會聘請秘書等專人處理，當政治獻金來源不當時，政客可以將責任推給這些人，此現象在日本以及台灣都非常常見。<ref>{{Cite web |url=http://www.epochtimes.com/b5/11/1/14/n3142633.htm |title=違反政治獻金法 前立委被訴}}</ref>

===攻擊手段===
在民主社會，很多政客為了打擊敵對勢力，往往聲稱競爭對手進行違法勾當，這件事情在台灣特別常見。像是：[[周玉蔻#三億男事件|三億男事件]]<ref>{{Cite news |url=http://news.ltn.com.tw/news/politics/paper/847906 |title=違政治獻金上限 立委促查}}</ref><ref>{{Cite news |url=http://news.ltn.com.tw/news/society/breakingnews/1299676 |title=三億男是誰？柯建銘請假明天不作證了}}</ref>、[[頂新獻金案|馬英九政治獻金案]]<ref>{{Cite news |url=http://www.peoplenews.tw/news/0738af2c-87a1-4fcb-97a9-2017a2e5dba1 |title=國民黨與馬英九說不清的政治獻金}}</ref>等等。
