'''蔡武'''（{{bd|1949年|10月||catIdx=Cai蔡}}），出生于[[甘肃省|甘肃省]][[兰州市|兰州市]]，祖籍[[甘肃|甘肃]][[武都|武都]]；北京大学国际政治系毕业。1973年1月加入中国共产党，是中共第十七、十八届中央委员。曾任[[中共中央宣传部|中共中央宣传部]]副部长、[[中华人民共和国文化部|中华人民共和国文化部]]部长，[[国务院新闻办公室|国务院新闻办公室]]主任等职。他还是[[中国人民大学|中国人民大学]]国际关系学院教授、博士生导师，[[北京大学|北京大学]]国际关系学院学术委员会高级顾问。

== 早年经历 ==
“[[文化大革命|文化大革命]]”时期，中学毕业的蔡武在甘肃北部的[[张掖市|张掖市]][[高台县|高台县]][[插队落户|插队劳动]]；1971年，在附近的[[山丹县|山丹煤矿]]当工人；期间，曾兼任《[[甘肃日报|甘肃日报]]》业余通讯员。文革结束后，1976年10月，调入甘肃省燃化局、煤炭局工作，任政治部干事。[[高考|高考]]制度恢复后，于1978年考入[[北京大学|北京大学]]国际政治专业学习；毕业后，留校任教。

== 仕途 ==
1983年7月，蔡武调[[共青团中央|共青团中央]]工作，任团中央国际联络部部长；后升任团中央常委、[[全国青联|全国青联]]负责人；1995年，调[[中共中央對外联络部|中共中央對外联络部]]任研究室主任；于同年升任副秘书长；1997年，晋升中联部副部长；2005年8月，升任[[中共中央宣传部|中共中央宣传部]]副部长、[[中共中央对外宣传办公室|中共中央对外宣传办公室]]主任、[[国务院新闻办公室|国务院新闻办公室]]主任。

2008年3月，在[[第十一届全国人大|第十一届全国人大]]第一次会议上，被任命为[[中华人民共和国文化部|中华人民共和国文化部]]部长。2014年12月，不再担任。<ref>{{Cite web |url=http://news.ycwb.com/2014-12/28/content_8492267.htm |title=雒树刚任文化部部长 蔡武不再担任 |access-date=2014-12-28 |archive-url=https://web.archive.org/web/20141228145934/http://news.ycwb.com/2014-12/28/content_8492267.htm |archive-date=2014-12-28 |dead-url=yes }}</ref>

== 重要事件 ==
2010年9月，时任[[中华人民共和国文化部|文化部部长]]的蔡武以“中华文化联谊会”名誉会长的身份，率团访问[[台湾|台湾]]，是继1998年前[[中华人民共和国科学技术部|科技部部长]][[朱丽兰|朱丽兰]]之后，首位访台的大陆在任部长级官员<ref>[http://news.ifeng.com/taiwan/special/caiwufangtai/content-0/detail_2010_09/02/2415642_0.shtml 文化部长蔡武今起访台 两岸文化首长将首度会面]</ref>。
