'''蔡愛智'''（{{bd|1911年||1998年|8月6日}}）是專長於[[基督教神學|基督教神學]]的首位在[[美國|美國]][[台灣人|台灣人]]牧師，基督宗教哲學[[喀爾文主義|喀爾文主義]]的臺灣[[哲學家|哲學家]]。<ref name = "存在交涉：日治時期的臺灣哲學_2016"> {{ cite book | title = 存在交涉：日治時期的臺灣哲學 | url = https://books.google.com.tw/books?id=wsoMDAAAQBAJ&pg=PA35&lpg=PA35&dq=%E8%94%A1%E6%84%9B%E6%99%BA&source=bl&ots=N5FqQVrOR5&sig=ACfU3U0VQrEO6KPMp56WTTGksYYpZyheaw&hl=zh-TW&sa=X&ved=2ahUKEwjvu-b1jNbhAhUR9bwKHQsADU04ChDoATAJegQICRAB#v=onepage&q=%E8%94%A1%E6%84%9B%E6%99%BA&f=false | publisher = 聯經出版事業公司, 2016 | author = 作者：洪子偉 | isbn = 9789860472905 | page = 35 | language = zh-tw | date = 2016 }} </ref>

== 經歷 ==
在[[二次大戰|二次大戰]]時期是少數在美國的台灣人，因為了解[[日本|日本]]與[[臺灣日治時期|臺灣日治時期]]，並且精通[[台語|台語]]、[[日語|日語]]及[[英語|英語]]接受徵招服務於美軍文職。
* 1944年擔任[[美國海軍學院|美國海軍學院]]研究分析師、教師與翻譯
* 1945年擔任[[美國情報體系|美國情報體系]]研究分析師
* 1946年到1947年加入[[聯合國善後救濟總署|聯合國善後救濟總署]]赴日本、[[上海|上海]]與[[台灣|台灣]]等地調查戰後[[美援|美援]]物資在台灣分配情形，協助戰後重建<ref name = "悼念西雅圖第一位台灣人蔡愛智牧師 ◎陳顯貴 /太平洋時報 1988-08-20_2017"> {{ cite web | title = 悼念西雅圖第一位台灣人蔡愛智牧師 ◎陳顯貴 /太平洋時報 1988-08-20 | url = https://www.tahistory.org/%E6%82%BC%E5%BF%B5%E8%A5%BF%E9%9B%85%E5%9C%96%E7%AC%AC%E4%B8%80%E4%BD%8D%E5%8F%B0%E7%81%A3%E4%BA%BA%E8%94%A1%E6%84%9B%E6%99%BA%E7%89%A7%E5%B8%AB-%E2%97%8E%E9%99%B3%E9%A1%AF%E8%B2%B4-%E5%A4%AA/ | publisher = 台美人歷史協會 | language = zh-tw | date = 2017-04-20 }} </ref><ref name = "星島日報_2017"> {{ cite web | title = 星島日報 | url = https://www.singtaousa.com/la/453-%E5%8D%97%E5%8A%A0%E6%96%B0%E8%81%9E/118163-%E5%82%B3%E7%B5%B1%E5%91%A8%E5%8F%B0%E7%BE%8E%E4%BA%BA%E6%AD%B7%E5%8F%B2%E4%BA%BA%E7%89%A9%E5%B1%95++%E8%AC%9B%E8%BF%B0%E4%BA%8C%E6%88%B0%E9%96%93%E7%BE%8E%E8%BB%8D%E5%8F%B0%E7%BE%8E%E4%BA%BA%E6%AD%B7%E5%8F%B2/ | publisher = 星島日報 | language = zh-tw | date = 2017-05-17 }} </ref>
* 1948年到1978年擔任美國西雅圖日本教會牧師
* 1969年協助籌建在美國[[西雅圖|西雅圖]]的台灣人教會<ref name = "蔡愛智牧師生平_2017"> {{ cite web | title = 蔡愛智牧師生平 | url = http://www.tahistory.org/%E8%94%A1%E6%84%9B%E6%99%BA%E7%89%A7%E5%B8%AB%E7%94%9F%E5%B9%B3/ | publisher = 台美人歷史協會 | language = zh-tw | date = 2017-05-10 }} </ref><ref name = "傳統週歷史文物展 傳誦臺美人事跡_2017"> {{ cite web | title = 傳統週歷史文物展 傳誦臺美人事跡 | url = http://www.epochtimes.com/b5/17/5/19/n9162981.htm | publisher = 大紀元時報 | author = 袁玫柔 責任編輯：張頓 | language = zh-tw | date = 2017-05-20 }} </ref>
* 1970年西雅圖地區台灣同鄉會首任會長<ref name = "6. Brief History of Taiwanese Association of Greater Seattle (TAGS) 大西雅圖區台灣同鄉會簡介_2017"> {{ cite web | title = 6.
