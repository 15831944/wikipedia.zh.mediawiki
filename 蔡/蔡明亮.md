'''蔡明亮'''（{{bd|1957年|10月27日||}}）是一位出生於[[馬來西亞|馬來西亞]]的[[臺灣|臺灣]][[台灣電影|電影]][[導演|導演]]，是在國際間享有高知名度的[[作者論|電影作者]]<ref>{{Cite web |url=http://filmmedia.berkeley.edu/course-information/2396838-auteur-theory-taiwan-film-auteurs |title=Auteur Theory: Taiwan Film Auteurs |access-date=2017-10-15 |archive-url=https://web.archive.org/web/20170820035550/http://filmmedia.berkeley.edu/course-information/2396838-auteur-theory-taiwan-film-auteurs |archive-date=2017-08-20 |dead-url=no }}</ref>，曾在歐洲影展獲得多個重要獎項，包括以《[[愛情萬歲_(台灣電影)|愛情萬歲]]》(1994)與《[[郊遊_(電影)|郊遊]]》(2013)於[[威尼斯影展|威尼斯影展]]分別獲得[[金獅獎|金獅獎]]與[[威尼斯影展評審團大獎|評審團大獎]]，以及以《[[河流_(电影)|河流]]》(1997)獲得[[評審團大獎銀熊獎|柏林影展評審團特別銀熊獎]]。

== 生平 ==
=== 早年生活 ===
蔡明亮於1957年出生在[[砂拉越州|砂拉越州]][[古晋市|古晋市]]的華人家庭，祖籍為[[廣東|廣東]][[揭阳|揭阳]]。他高中畢業後前往[[台湾|台湾]]念書，1981年畢業於[[中国文化大学|中国文化大学]]戲劇系影剧組。在學生時代，蔡明亮便開始撰寫[[舞台劇|舞台劇]][[劇本|劇本]]，並親自執導了三部作品；蔡明亮筆下的作品時以幽默手法，表現著現代社會、寂寞、都會生活形態的狂亂等主題，例如《速食酢醬麵》(1981)與《黑暗裡打不開的一扇門》(1982)；1983年，他自編、自導、自演一人作品《房間裡的衣櫃》，嘗試探討都市居民有意識的自我防衛，現代人疏離風格也是他多數電影作品反覆出現的母題。

1984年，蔡明亮幫電影《小逃犯》撰寫劇本，以「三一律」劇本原則刻劃一天之內在一個公寓房子發生的情節。往後的幾年他投身於電視劇本寫作，同時也從事劇場的戲劇課程教學；在此時期，他完成許多電視劇本於執導的工作，他以單元劇《麗香的感情線》(1990)與《給我一個家》(1991)兩度獲得金鐘獎最佳導播獎。

=== 1992-1998 ===
當蔡明亮拍攝《小孩》(1991)時，偶然地在[[西門町|西門町]]一個青少年聚集的[[電動|電動]][[遊樂場|遊樂場]]中發現了從來沒有受過任何表演訓練的[[李康生|李康生]]，1992年蔡明亮完成了以李康生為主角首部電影長片《[[青少年哪吒|青少年哪吒]]》，在國際上獲得許多獎項，包括獲得[[東京影展|東京影展]]銅獎、法國[[南特影展|南特影展]]處女作最佳影片獎、義大利都靈影展最佳影片獎，在[[金馬獎|金馬獎]]也入圍最佳導演與最佳原著劇本獎。從這部片開始，蔡明亮開始創作以李康生飾演「小康」這個角色的電影。

1994年，蔡明亮以《愛情萬歲》獲得[[威尼斯影展|威尼斯影展]]金獅獎、費比西獎、[[金馬獎|金馬獎]]最佳導演獎等獎項；1997年以《[[河流_(电影)|河流]]》獲得[[評審團大獎銀熊獎|柏林影展評審團特別銀熊獎]]；1998年完成受法國電視台「2000年見聞」計畫之託的《[[洞_(电影)|洞]]》，首次進入[[第51屆坎城影展|第51屆坎城影展]]正式競賽單元，並獲得[[坎城影展費比西國際影評人獎|費比西國際影評人獎]]。這段時期，蔡明亮接連以三部電影在威尼斯、柏林與坎城影展獲獎，奠定其在國際影壇上的藝術地位。

=== 1999-2009 ===
2000年，蔡明亮聯合李康生與其他獨立電影工作者成立了「汯呄霖電影」，開始自行製作與行銷電影。2001年，《[[你那邊幾點|你那邊幾點]]》獲選為[[第54屆坎城影展|第54屆坎城影展]]正式競賽片，在這部片裡他首度將部份場景設在[[巴黎|巴黎]]，並與法國演員[[讓-皮埃爾·里奧|尚-皮耶·李奧]]合作。2002年以紀錄短片《與神對話》描繪台灣廟會文化，獲邀參加法國馬賽紀錄片影展。

2002年，蔡明亮租下即將關門的永和福和戲院，隔年他完成在戲院裡面拍攝的《[[不散|不散]]》<ref>{{Cite web |url=http://www.epochtimes.com/b5/3/3/31/n294084.htm |title=不見不散福和戲院開鏡 石雋苗天成了老影迷 |access-date=2017-10-15 |archive-url=https://web.archive.org/web/20171015150707/http://www.epochtimes.com/b5/3/3/31/n294084.htm |archive-date=2017-10-15 |dead-url=no }}</ref>，進入[[第60屆威尼斯影展|第60屆威尼斯影展]]正式競賽元。2004年，他獲法國文化部頒發文藝騎士榮譽勳章。
[[File:The_Wayward_Cloud.jpg|left]]》法國版海報]]
2005年，《[[天邊一朵雲|天邊一朵雲]]》在柏林影展正式競賽單元中首映，雖然獲得[[傑出藝術貢獻銀熊獎|傑出藝術成就銀熊獎]]、[[亞佛雷德鮑爾獎|亞佛雷德鮑爾獎]]與[[柏林影展費比西國際影評人獎|費比西國際影評人獎]]，但因電影內的裸露與性愛場景，在台灣引起爭議。2006年，他回到馬來西亞拍攝的《[[黑眼圈_(電影)|黑眼圈]]》再度進入[[威尼斯影展|威尼斯影展]]正式競賽單元，但在[[金馬獎|金馬獎]]評審團批評他的電影「太過沉浸個人風格」，「電影應該要感動人，還要考慮到票房與觀眾的接受度」後，蔡明亮憤而召開記者會宣布退出金馬獎。<ref>{{Cite web |url=http://www.epochtimes.com/b5/6/11/4/n1509661.htm |title=不願作品被蹂躝 蔡明亮宣布退出金馬獎 |access-date=2017-10-15 |archive-url=https://web.archive.org/web/20171015150709/http://www.epochtimes.com/b5/6/11/4/n1509661.htm |archive-date=2017-10-15 |dead-url=no }}</ref>

[[羅浮宮|羅浮宮]]在2006年自200多位導演候選名單中，選出蔡明亮拍攝典藏於羅浮宮藝術作品為主題的電影；2009年，蔡明亮以[[達文西|達文西]]名畫「聖施洗約翰」及背後的故事「[[莎樂美|莎樂美]]」為主題完成電影《[[臉_(電影)|臉]]》，由臺灣、法國、[[荷蘭|荷蘭]]、[[比利時|比利時]]四國合製，第三度進入坎城影展正式競賽單元，此為羅浮宮典藏的第一部電影作品。

=== 2010-現今 ===
[[File:Asian_Filmmaker_of_the_Year,_2010.jpg|right]]2010 獲頌「亞洲年度電影工作者」。]]
2013年，蔡明亮以《[[郊遊_(電影)|郊遊]]》第四度進入[[第70屆威尼斯國際電影節|第70屆威尼斯影展]]正式競賽單元<ref>[http://www.apdnews.com/news/33923.html 蔡明亮谈《郊游》—“慢”很美] {{webarchive|url=https://web.archive.org/web/20130909030743/http://www.apdnews.com/news/33923.html |date=2013-09-09 }}，亚太日报，2013年9月6日</ref>，最終獲得[[威尼斯影展評審團大獎|評審團大獎]]<ref>{{cite news|author=張哲鳴、蔡敦仰╱威尼斯報導、攝影|title=蔡明亮奪威尼斯評審團大獎 不屑中國謝台灣|url=http://ent.appledaily.com.tw/enews/article/entertainment/20130909/35281151/|accessdate=2013-09-09|newspaper=台灣蘋果日報|date=2013-09-09|archive-url=https://web.archive.org/web/20130912043845/http://ent.appledaily.com.tw/enews/article/entertainment/20130909/35281151|archive-date=2013-09-12|dead-url=no}}</ref>，同年在[[第50屆金馬獎|第50屆金馬獎]]還獲得第二座最佳導演獎<ref>{{Cite web |url=http://www.cna.com.tw/news/firstnews/201311230051-1.aspx |title=蔡明亮最佳導演 劉德華爭戲約，中央社，2013年11月24日 |access-date=2013-11-24 |archive-url=https://web.archive.org/web/20131126134900/http://www.cna.com.tw/news/firstnews/201311230051-1.aspx |archive-date=2013-11-26 |dead-url=no }}</ref>。蔡明亮在威尼斯受訪時表示《郊遊》可能是他的最後一部長片作品，未來將專注於拍攝「更像藝術品」的電影或短片<ref>{{cite news|author=張哲鳴╱台北報導|title=蔡明亮特寫電鍋20分鐘 打趣歡迎贊助|url=http://ent.appledaily.com.tw/enews/article/entertainment/20120827/34467159|accessdate=2013-09-09|newspaper=台灣蘋果日報|date=2013-08-27|archive-url=https://web.archive.org/web/20131005021141/http://ent.appledaily.com.tw/enews/article/entertainment/20120827/34467159|archive-date=2013-10-05|dead-url=no}}</ref> ；他同時也不認同台灣戲院制度，因此《郊游》最後並未在台灣上院線，而是在北師美術館舉辦特展放映<ref>{{Cite web |url=http://www.funscreen.com.tw/feature.asp?FE_No=1548 |title=流浪，慢走，美術館——蔡明亮談進入美術館 |access-date=2017-10-15 |archive-url=https://web.archive.org/web/20171015150425/http://www.funscreen.com.tw/feature.asp?FE_No=1548 |archive-date=2017-10-15 |dead-url=no }}</ref>。

2014年，他獲頒法國藝術與文學軍官勳位<ref>{{cite news|title=法國二度頒勳 蔡明亮升「軍官」|url=http://news.ltn.com.tw/news/entertainment/paper/804333|newspaper=自由時報|author=鄒念祖|date=2014-08-14|access-date=2014-08-14|archive-url=https://web.archive.org/web/20140814075422/http://news.ltn.com.tw/news/entertainment/paper/804333|archive-date=2014-08-14|dead-url=no}}</ref>。

2015，他赴日拍攝的短片《[[無無眠|無無眠]]》在[[台北電影獎|台北電影獎]]獲得[[台北電影獎最佳導演|最佳導演獎]]。2017年，他在[[hTC|hTC]]邀請下首次嘗試使用VR技術，最終完成短片《家在蘭若寺》，在[[第74屆威尼斯影展|第74屆威尼斯影展]]首次舉辦的VR電影競賽單元中首映<ref>{{Cite web |url=http://www.chinatimes.com/realtimenews/20170727005544-260404 |title=蔡明亮執導VR電影《家在蘭若寺》 入圍威尼斯影展 |access-date=2017-10-15 |archive-url=https://web.archive.org/web/20171015150354/http://www.chinatimes.com/realtimenews/20170727005544-260404 |archive-date=2017-10-15 |dead-url=no }}</ref>；他在受訪時表示預計2018年要拍攝長片作品。

=== 私人生活 ===
蔡明亮於2014年發行的《郊遊》電影書中首次公開承認，自己是一名[[男同性戀|男同性戀]]者<ref>{{Cite web |url=http://ent.sina.com.cn/s/m/2014-07-04/13514169797.shtml |title=蔡明亮公开出柜：接受真正的自己 |access-date=2014-07-05 |archive-url=https://web.archive.org/web/20140707065124/http://ent.sina.com.cn/s/m/2014-07-04/13514169797.shtml |archive-date=2014-07-07 |dead-url=no }}</ref>。

2016年，他針對中國網站許多侵權行為展開行動，包括公開呼籲與密集寫信給各個影音網站<ref>{{Cite web |url=http://www.nextmag.com.tw/realtimenews/news/297516 |title=蔡明亮「念經式攻擊」　中國網站崩潰自宮盜版了 |access-date=2017-10-15 |archive-url=https://web.archive.org/web/20171015201952/http://www.nextmag.com.tw/realtimenews/news/297516 |archive-date=2017-10-15 |dead-url=no }}</ref><ref>{{Cite web |url=http://www.mnw.cn/movie/huayu/1382587.html |title=蔡明亮发表反盗感言 点名各视频网站及网盘洗脚上岸 |access-date=2016-10-01 |archive-url=https://web.archive.org/web/20161008170714/http://www.mnw.cn/movie/huayu/1382587.html |archive-date=2016-10-08 |dead-url=no }}</ref>，包括[[騰訊|騰訊]]與[[Bilibili|Bilibili]]皆撤下蔡明亮作品，並公開道歉<ref>{{Cite web |url=http://ent.sina.com.cn/m/c/2016-08-21/doc-ifxvcsrn8828073.shtml |title=蓝影网向蔡明亮道歉:永不再分享盗版资源 |access-date=2016-10-01 |archive-url=https://web.archive.org/web/20160822185911/http://ent.sina.com.cn/m/c/2016-08-21/doc-ifxvcsrn8828073.shtml |archive-date=2016-08-22 |dead-url=no }}</ref><ref>{{Cite web |url=http://ent.163.com/16/0825/12/BVAKC7L0000300B1.html |title=B站为盗版致歉蔡明亮 后者:不在官网道歉诚意何在 |access-date=2016-10-01 |archive-url=https://web.archive.org/web/20160828134231/http://ent.163.com/16/0825/12/BVAKC7L0000300B1.html |archive-date=2016-08-28 |dead-url=no }}</ref><ref>{{Cite web |url=http://ent.163.com/16/0909/08/C0GPFVLR000380D0.html |title=蔡明亮导演不满腾讯道歉再发公开信 |access-date=2016-10-01 |archive-url=https://web.archive.org/web/20171015151759/http://ent.163.com/16/0909/08/C0GPFVLR000380D0.html |archive-date=2017-10-15 |dead-url=no }}</ref>。

== 風格 ==
蔡明亮電影風格鮮明，他電影中的人物時常寡言，常以長鏡頭拍攝人物，並以實際時間的流逝來展現角色內在的情緒反應，被許多學者和影評認為是{{link-en|緩慢電影|Slow cinema}}的代表導演之一。水的意象反覆出現在他的作品中；他也曾多次在電影中使用歌舞元素，包括《[[洞_(电影)|洞]]》(1998)、《[[天邊一朵雲|天邊一朵雲]]》(2005)與《[[臉_(電影)|臉]]》(2009)。自《[[河流_(电影)|河流]]》(1997)開始，他的每部電影皆有他手寫的簽名。
