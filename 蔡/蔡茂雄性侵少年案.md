'''蔡茂雄性侵少年案'''，是臺灣在2005年6月時爆發的[[性醜聞|性醜聞]]案件。時任[[民主進步黨|民主進步黨]]臺北縣汐止市黨部主任委員的蔡茂雄，被揭發曾[[性侵害|性侵害]]或猥褻3名12歲至15歲不等的男孩，引發社會輿論譁然。由於蔡茂雄是民進黨地方黨部幹部，犯案地點甚至就在該黨汐止市黨部內，這起事件也成為政治醜聞，影響民進黨聲譽。蔡茂雄被以《[[中華民國刑法|中華民國刑法]]》強制性交罪、趁機性交罪等罪名起訴。歷經多年訴訟，[[中華民國最高法院|中華民國最高法院]]於2011年6月15日駁回蔡茂雄上訴，蔡茂雄有罪定讞，合併應執行13年[[有期徒刑|有期徒刑]]。<ref name="最高法院判決">{{Cite book|author=洪文章、王居財、郭毓洲、沈揚仁、林立華|title=〈[[:s:最高法院100年度台上字第3173號刑事判決|最高法院刑事判決一○○年度台上字第三一七三號]]〉|publisher=[[中華民國最高法院|中華民國最高法院]]|location=臺北市|date=2011-06-15|language=zh-tw}}</ref>

== 案情 ==
蔡茂雄時任民主進步黨臺北縣（今[[新北市|新北市]]）汐止市黨部（現已裁撤）主任委員，負責統籌民進黨在汐止地區的組織發展，包括行政、選舉事務等，在當地頗有政治影響力。蔡茂雄已婚，案發時53歲，育有2個已成年的女兒。<ref>{{Cite news|author=黃敦硯|title={{單雙書號轉換|民進黨汐止主委 性侵2少男}}|url=http://news.ltn.com.tw/news/society/paper/22870|work=《[[自由時報|自由時報]]》|publisher=自由時報|location=臺北|date=2005-06-24|accessdate=2018-05-26|language=zh-tw|archiveurl=https://web.archive.org/web/20180527061955/http://news.ltn.com.tw/news/society/paper/22870|archivedate=2018-05-27}}</ref>蔡茂雄過去就曾因強暴男童被起訴，只是當時《[[中華民國刑法|刑法]]》強制性交罪還不是[[非告訴乃論|公訴罪]]；蔡茂雄與受害男童家長私下和解後，受害男童家長撤回告訴，法院也判他無罪。<ref>{{Cite news|author=莊榮宏、胡守得、陳恩惠|title={{單雙書號轉換|性侵男童 蔡茂雄有前科}}|url=http://news.ltn.com.tw/news/society/paper/22873|work=《自由時報》|publisher=自由時報|location=臺北|date=2005-06-24|accessdate=2018-05-26|language=zh-tw|archiveurl=https://web.archive.org/web/20180527062244/http://news.ltn.com.tw/news/society/paper/22873|archivedate=2018-05-27}}</ref>

本案被害者A、B、C都是逃家少年，就讀於臺北縣某[[國民中學|國民中學]]。<ref name="陳一雄">{{Cite news|author=陳一雄、劉峻谷|title={{單雙書號轉換|性侵三少年　民進黨主委蔡茂雄被捕}}|work=《[[聯合報|聯合報]]》|publisher=[[聯合報系|聯合報系]]|location=臺北|date=2005-06-24|language=zh-tw}}</ref>根據《[[聯合報|聯合報]]》報導，受害最深的A少年於2004年7月剛從[[國民小學|國民小學]]畢業時，在汐止某[[便利商店|便利商店]]被蔡茂雄[[搭訕|搭訕]]。<ref name="陳一雄"/>隔日，蔡茂雄開車載A少年到金山鄉（今[[金山區_(新北市)|金山區]]）泡溫泉，並在旅館對A少年[[肛交|肛交]]。A少年出身於[[單親家庭|單親家庭]]，因為不滿父親的管教方式而逃家；蔡茂雄便將他留宿於民進黨汐止市黨部2樓，每隔2、3天便對A少年肛交一次，有時更將[[跳蛋|跳蛋]]塞入A少年肛門取樂，並採摘汐止市黨部所種植的[[蘆薈|蘆薈]]作為[[潤滑劑|潤滑劑]]使用。<ref name="陳一雄"/><ref>{{Cite news|author=林君宜|title={{單雙書號轉換|蔡茂雄 吸安且性逞獸慾 灌酒餵毒玩3P}}|work=《[[中央日報|中央日報]]》|publisher=中央日報社|location=臺北|date=2005-06-25|language=zh-tw}}</ref>

其後，蔡茂雄經A少年介紹，認識與A少年同齡（12歲）的B少年及年紀較長的C少年（15歲）。<ref name="士院判決">{{Cite book|author=蔡明宏、莊明達、周明鴻|title=〈[[:s:臺灣士林地方法院94年度矚訴字第1號刑事判決|臺灣士林地方法院刑事判決94年度矚訴字第1號]]〉|publisher=[[臺灣士林地方法院|臺灣士林地方法院]]|location=臺北市|date=2007-02-14|language=zh-tw}}</ref>2004年9月某日，蔡茂雄讓A、B少年一同留宿民進黨汐止市黨部，灌醉B少年後，脫掉B少年和自己的衣物，用自己的[[陰莖|陰莖]]摩蹭B少年臀部猥褻得逞。接著，蔡茂雄邀請A、B、C三名少年留宿民進黨汐止市黨部，趁C少年熟睡的時候脫掉他的衣物，用猥褻B少年的方式猥褻C少年得逞。<ref name="士院判決"/>無論A、B、C少年，都是在明確拒絕、未表示意願或無法表示意願的情況下，被蔡茂雄性侵害或猥褻得逞。<ref name="士院判決"/>

除了強暴、猥褻少年外，蔡茂雄還誘拐A少年吸食[[安非他命|安非他命]]，這種藥物在臺灣是第二級毒品，無論持有或吸食都有罪；而為了讓A少年每次性交時不會反抗，蔡茂雄還會以肢體暴力強迫A少年服從。<ref name="士院判決"/>之後，A少年把蔡茂雄提供毒品、施加肢體暴力的部分告訴朋友蘇姓少女（15歲）；蘇姓少女便找蔡茂雄理論，卻遭蔡茂雄打傷臉部。<ref name="士院判決"/>此外，在強暴、猥褻、毆打3名少年的過程中，蔡茂雄都會拍攝他們的[[裸照|裸照]]，存放在民進黨提供給他的辦公室電腦裡，成為日後[[檢察官|檢察官]]起訴的證據。<ref name="士院判決"/>

== 輿論反應 ==
2005年6月24日，臺灣主要報紙《[[自由時報|自由時報]]》、《聯合報》、《[[聯合晚報|聯合晚報]]》、《[[中國時報|中國時報]]》、《[[中央日報|中央日報]]》和《[[中華日報|中華日報]]》以頭版報導方式揭露蔡茂雄性侵少年案，引起輿論譁然。案件之所以爆發，是因為蔡茂雄固定給予A少年每日[[新臺幣|新臺幣]]300元至500元不等的零用錢，作為A少年的生活費和封口費；而A少年的父親偶然發現兒子購物時出手闊綽，追問之下，才發現蔡茂雄的行徑，並向[[臺灣士林地方法院檢察署|臺灣士林地方法院檢察署]]告發。<ref name="蕭承訓">{{Cite news|author=蕭承訓、陳佳鑫|title={{單雙書號轉換|民進黨黨工 孌三少 迫吸毒性侵 蔡茂雄爆醜聞 收押}}|work=《[[中國時報|中國時報]]》|publisher=中國時報社|location=臺北|date=2005-06-24|language=zh-tw}}</ref>檢察官於是指揮[[內政部警政署刑事警察局|內政部警政署刑事警察局]]偵辦調查，並傳喚B、C少年訊問案情，確認A少年所述無誤且B、C少年也曾受害後，於同年6月23日在蔡茂雄的住處將他拘提到案。<ref name="蕭承訓"/>

因蔡茂雄為地方重要政治人物，身分敏感，新聞媒體立刻在隔日（6月24日）大篇幅報導本案，並採訪各界人士發表評論。《中國時報》援引匿名地方人士評論指出，蔡茂雄的[[孌童癖|孌童癖]]在當地早有傳聞；<ref name="孫廷龍">{{Cite news|author=孫廷龍、許俊偉|title={{單雙書號轉換|蔡怪癖 地方早有耳聞}}|work=《中國時報》|publisher=中國時報社|location=臺北|date=2005-06-24|language=zh-tw}}</ref>該名地方人士曾在某個晚上到民進黨汐止市黨部找蔡茂雄，卻撞見蔡茂雄與一名男孩只穿內褲坐在辦公室裡。<ref name="孫廷龍"/>《聯合晚報》則報導，蔡茂雄的鄰居也指控蔡茂雄猥褻他的孫子，並企圖用錢誘拐男孩。<ref name="陳珮琦">{{Cite news|author=陳珮琦|title={{單雙書號轉換|戀童癖？ 鄰居控蔡茂雄曾猥褻孫子}}|work=《[[聯合晚報|聯合晚報]]》|publisher=聯合報系|location=臺北|date=2005-06-25|language=zh-tw}}</ref>《中央日報》則指出，蔡茂雄曾多次恐嚇A少年與其父親，說他「在地方上黑白兩道都很熟」，能夠大事化小、小事化無；還找人跟蹤A少年上學，讓A少年心生恐懼。<ref name="王鵬捷">{{Cite news|author=王鵬捷、林君宜、李効儒、周子琦|title={{單雙書號轉換|民進黨汐止主委蔡茂雄性侵案擴大 受害少年滾雪球}}|work=《中央日報》|publisher=中央日報社|location=臺北|date=2005-06-25|language=zh-tw}}</ref>

除此之外，《[[蘋果日報_(台灣)|蘋果日報]]》也引述地方人士言論指控，蔡茂雄1991年時曾在[[五堵|五堵]]地區開設[[電動玩具|電動玩具]]店，吸引許多少年、男童前往；蔡茂雄便將安非他命研磨成粉末，放在[[空調|空調]]通風口處，待室內少年、男童意識不清後，再帶到其他房間性侵害。<ref name="黃泊川">{{Cite news|author=黃泊川、廖炳棋|title={{單雙書號轉換|蔡茂雄再爆另姦五少男}}|work=《[[蘋果日報_(台灣)|蘋果日報]]》|url=https://tw.appledaily.com/headline/daily/20050711/21904165|publisher=[[壹傳媒|壹傳媒]]|location=臺北|date=2005-07-11|accessdate=2018-05-26|language=zh-tw|archiveurl=https://web.archive.org/web/20180527062415/https://tw.appledaily.com/headline/daily/20050711/21904165|archivedate=2018-05-27}}</ref>針對上述指控，蔡茂雄全盤否認，其委任律師更要求檢察官對A、B、C少年測謊。<ref name="黃泊川"/>

== 影響 ==
由於蔡茂雄利用其政治身分、在民進黨地方黨部辦公室內性侵未成年少年，輿論矛頭均指向民進黨，要求該黨出面負責。當時為執政黨的民進黨先是在案件爆發當日（6月24日）表示，若蔡茂雄被移送法辦，就會立刻開除他的黨籍<ref>{{Cite news|author=林晨柏、孫廷龍、許俊偉|title={{單雙書號轉換|黨中央：送辦即開除黨籍}}|work=《中國時報》|publisher=中國時報社|location=臺北|date=2005-06-24|language=zh-tw}}</ref>；隔日（6月25日）再表示，臺北縣黨部已召開評議委員會，全體無異議通過開除蔡茂雄黨籍。<ref name="王鵬捷"/>時任[[民進黨秘書長|民進黨秘書長]]的[[李逸洋|李逸洋]]則向媒體承諾，中央黨部將修改黨務人員選舉辦法，制定「排黑條款」，有重大刑事前科者一律不得擔任重要黨職幹部。<ref>{{Cite news|author=林君宜|title={{單雙書號轉換|斷尾停損 蔡茂雄除黨籍 亡羊補牢 民進黨研擬排黑}}|work=《中央日報》|publisher=中央日報社|location=臺北|date=2005-06-25|language=zh-tw}}</ref>就連時任[[總統府秘書長|總統府秘書長]]兼[[民進黨黨主席|民進黨黨主席]]的[[蘇貞昌|蘇貞昌]]也批評，蔡茂雄是「害群之馬」，嚴重影響該黨聲譽，衝擊[[2005年中華民國縣市長暨縣市議員選舉#臺北縣|2005年臺北縣縣長選舉]]選情。<ref name="黃雅詩">{{Cite news|author=黃雅詩|title={{單雙書號轉換|性侵案 蘇：恐衝擊北縣長選情}}|work=《聯合報》|publisher=聯合報系|location=臺北|date=2005-06-26|language=zh-tw}}</ref>

2005年，臺灣士林地方檢察署檢察官以《刑法》加重強制性交罪、趁機性交罪、對未成年人恐嚇罪、引誘少年吸食第二級毒品罪及傷害罪等罪名，對蔡茂雄提起公訴；2007年2月14日，[[臺灣士林地方法院|臺灣士林地方法院]]作成有罪判決，蔡茂雄應執行16年有期徒刑，入監前還得做強制治療及毒品勒戒。<ref name="士院判決"/>針對刑責最重的加重強制性交罪與趁機性交罪，蔡茂雄提起上訴，主張A少年說謊、證人證詞不可信等；2011年6月15日，中華民國最高法院駁回蔡茂雄上訴，蔡茂雄有罪定讞，合併應執行13年有期徒刑。<ref name="最高法院判決"/>
