'''蔡厝港公共圖書館''' （{{lang-en|Choa Chu Kang Public Library}}）是一所位于[[蔡厝港|蔡厝港]]规划区{{link-en|新加坡圖書館理局|National Library Board}}旗下的[[公共圖書館|公共圖書館]]。此图书馆位于{{le|Lot One购物中心|Lot_One}}里的四楼。1997年图书馆正式开幕，2008年经翻新后重新开幕。

==历史==
1997年2月蔡厝港公共圖書館正式于{{le|Lot One购物中心|Lot_One}}的三楼开幕，时任新加坡卫生部长姚照东主持了开幕典礼。<ref name=nlb>{{cite web|url=http://eresources.nlb.gov.sg/infopedia/articles/SIP_673_2005-01-24.html|title=Choa Chu Kang Public Library - Infopedia|first=National Library Board,|last=Singapore|website=eresources.nlb.gov.sg|language=en|deadurl=no|archiveurl=https://web.archive.org/web/20171211191344/http://eresources.nlb.gov.sg/infopedia/articles/SIP_673_2005-01-24.html|archivedate=2017-12-11|df=}}</ref>

2008年末圖書館进行为期一个月的翻新，并且搬迁至{{le|Lot One购物中心|Lot One}}的四楼。翻新后的图书馆以「大自然」的设计主题重新开放。<ref name=nlb/>新图书馆的面积比翻新前大约40八仙。<ref name=newspaper>[http://www.straitstimes.com/Breaking+News/Singapore/Story/STIStory_305111.html Choa Chu Kang library opens] {{webarchive|url=https://web.archive.org/web/20090122233737/http://www.straitstimes.com/Breaking%2BNews/Singapore/Story/STIStory_305111.html |date=2009-01-22 }}</ref>  图书馆内有个特色展区，展览了古代金典中华文学作品如 《[[论语|论语]]》，《[[千字文|千字文]]》 等。<ref>{{Cite web|url=http://m.sohu.com/n/443994048/?pvid=000115_3w|title=读书全指南——新加坡即将推行全国读书运动-新闻频道-手机搜狐|accessdate=2018-06-09|work=m.sohu.com|deadurl=no|archiveurl=https://web.archive.org/web/20180712043503/http://m.sohu.com/n/443994048/?pvid=000115_3w|archivedate=2018-07-12|df=}}</ref>
