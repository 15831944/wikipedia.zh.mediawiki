'''蔡秋敏'''（{{bd|1961年||catIdx=Cai蔡}}），中華民國政治人物，現為[[民主進步黨|民主進步黨]]籍，[[雲林縣議會|雲林縣議會]]第3選區議員。

== 政治生涯 ==
* 1998年第一次參選公職，以第三高票當選雲林縣第3選區縣議員（當年該選區名額8人）。
* 2002年連任雲林縣議員。
* 2005年參選雲林縣[[虎尾鎮|虎尾鎮]]鎮長，以不到700票之差落敗。
* 2009年以第一高票當選雲林縣議員。
* [[2014年九合一選舉|2014年九合一選舉]]中繼續參選雲林縣第3選區縣議員。

== 政治理念 ==
* 問政重點：農業問題、婦女議題、弱勢福利、地方發展、老人關照、產業結構等議題。
* 至今雖僅在地方參選，仍有國家層級的觀點。堅持臺灣意識，堅持[[臺灣|臺灣]]是一[[主權|主權]][[獨立國家|獨立國家]]，與[[中華人民共和國|中華人民共和國]]是[[臺灣主體性|一邊一國]]

== 政績 ==
* 1998年當選縣議員後，曾有議長參選人開價千萬向她買票，被她拒絕。<ref name="50com">http://www.50.com.tw/?p=3619</ref>
* 1998年當選縣議員後，呼籲保存[[虎尾郡守官邸|虎尾郡守官邸]]（今日的｢[http://www.ylstoryhouse.org.tw/ 雲林故事館]｣）和[[虎尾鐵橋|虎尾糖廠鐵橋]]，這兩個今日都是虎尾鎮最熱門的景點。<ref name="50com"></ref>
* 從縣議會開始呼籲中央政府提高稻米與蒜頭收購價格，被接納。<ref name="50com"></ref>

== 其他經歷 ==
謝長廷競選高雄市長顧問、蘇貞昌競選台北縣長顧問、1997雲林女青商會會長、國立土庫商工校友會會長、中正國小愛心媽媽創會會長、虎尾婦女防火宣導隊創會分隊長、家扶中心莪馨媽媽、第14.15屆雲林縣議員、身心障礙者保護委員、雲林地方法院少年法庭少年志工。
