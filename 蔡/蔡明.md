'''蔡明'''{{bD|1961年|10月21日||}}，[[女|女]]，[[北京|北京]]人，[[回族|回族]]。[[中国民主促进会|中国民主促进会]]会员<ref>[http://ent.163.com/16/0509/11/BMKBOTT800031H2L.html#p=9HDPTG6O00AJ0003 相声演员于谦已低调加入中国国民党革命委员会]</ref>。[[中国|中国]]著名[[演员|演员]]。<ref>{{cite web|url=http://www.sd.xinhuanet.com/news/2014-02/18/c_119380250.htm|title=《出彩中国人》第二期萌娃一秒落泪获赞 蔡明下跪|accessdate=2014|publisher=Xinhua|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20160304045514/http://www.sd.xinhuanet.com/news/2014-02/18/c_119380250.htm|archivedate=2016-03-04}}</ref><ref>{{cite web |url=http://news.163.com/14/0217/02/9L8ICL2900014Q4P.html |title= 蔡明：模仿也不容易 |accessdate=2014 |publisher=Wangyi |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://www.chinanews.com/yl/2014/02-15/5841184.shtml |title=元宵晚会蒋大为唱《套马杆》 蔡明演绎《卷珠帘》  |accessdate=2014 |publisher=Chinanews |work= |language=zh-hans}}</ref>小学时扮演电影《[[海霞_(电影)|海霞]]》中的小海霞而成名，之后蔡明多次在[[春节联欢晚会|春节联欢晚会]]等节目中亮相，成为了中国大陆家喻户晓的影视、小品演员。<ref>{{cite web |url=http://news.163.com/14/0217/02/9L8JVEEN00014Q4P.html |title=四岁长春孩儿“身经百战” 获蔡明“膜拜” |accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref>

== 关于 ==
蔡明在小学时担任了电影《海霞》中的主人公小海霞而一炮成名，另外她还主演过《[[离婚大战|离婚大战]]》、《[[捣蛋部队|捣蛋部队]]》、《[[临时家庭|临时家庭]]》、《[[心理诊所|心理诊所]]》以及喜剧《[[闲人马大姐|闲人马大姐]]》等[[连续剧|连续剧]]、[[喜剧|喜剧]]。另外，蔡明还出了一系列歌曲翻唱的唱片，被人们称为“百变蔡明”。
