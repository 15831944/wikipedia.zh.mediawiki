'''蔡玉玲'''（1955年8月22日-），台灣女性法律界及政治界人物。曾任[[中華民國|中華民國]][[行政院政務委員|行政院政務委員]]兼[[蒙藏委員會|蒙藏委員會]]委員長。民國66年自[[國立臺灣大學|國立臺灣大學]]法律學系畢業，曾任[[宜蘭地方法院|宜蘭地方法院]][[法官|法官]]、[[彰化地方法院|彰化地方法院]]法官、[[桃園地方法院|桃園地方法院]]法官、[[臺北士林地方法院|臺北士林地方法院]]法官、[[IBM|IBM]]大中華區法務長。她與丈夫李宗德共創理慈國際科技法律事務所，並任事務所主持律師。2013年11月出任行政院政務委員兼[[蒙藏委員會|蒙藏委員會]]委員長。

== 事件 ==
* vTaiwan虛擬世界法規調適計畫政策討論平台
推動虛擬世界法規調適計畫，為第一位政府官員至g0v零時政府黑客松提案建立「vTaiwan虛擬世界法規調適計畫」政策討論平台，由g0v志工建立，採用線上公開徵詢、討論、交流意見，邀集相關學者專家、業者、民眾參與實體會議，網路全程直播，並由各部會負責回覆討論區流言，更廣泛蒐集多元的意見，提昇政策形成及法規修訂的品質，法令推動更快速。
* 電子商務
致力推動電子商務相關法規及產業發展等政務。審查電子支付機構管理條例(第三方支付)草案，於104年1月16日立院三讀通過，同年5月3日施行，相關授權法規命令共15項，金管會亦於同年5月3日完成訂定。
* 研擬「公司法」修正草案
增加「閉鎖性股份有限公司」專節，104年6月15日經立院三讀通過，同年9月日施行。納入特別股複數表決權、無面額股等規範，提供新創公司股權安排彈性。
* 推動開辦「創業家簽證」
參考國外經驗並推動開辦「創業家簽證」，提高國外創業家來台創業誘因，使國際創意與本土資源結合，活絡國內創業活動及經濟成長動能。
* 推動開放民間業者辦理股權式群眾募資
金管會修正「證券商設置標準」、「證券商管理規則」及「證券商負責人與業務人員管理規則」；發布豁免申報函令；授權櫃買中心訂定「證券商經營股權性質群眾募資管理辦法」，相關法規已於104年4月30日公布。
* 推動放寬外籍白領受聘來台
推動放寬創新新創事業聘僱外籍專技人士之工作經驗、學歷、資本額與營業額的限制及僱用外籍主管之適用範圍及人數限制。
* 推動vMaker行動計劃
  1.推動以「Fab Truck」巡迴全台共497所高中職推廣Maker概念開始，2年時間拜訪全台各高中職，每一所停留2~3天，向當地學生推廣創客文化與介紹基本工具。
  2. 推動舉辦「逗卡車」等創客競賽，從提升產業、普及教育、人才養成三方面推動深耕；除與民間Fab Lab團體協力發展數位製造概念，更將國際發展知識與各國案例帶進校園，鼓勵校園Fab Lab設置參與國際組織，並希望以鼓勵創新和創意的方式創造自造者基地，藉自造者運動（Maker Movement）之推廣與匯聚，奠定台灣創客基地之定位。
* 研擬虛擬實境（Virtual Reality）產業政策
參訪台灣及國外虛擬實境相關產業及組織，相互交流，研擬並規劃台灣虛擬實境產業相關政策。
* 推動金點設計獎國際化
開放國際徵件，擴大業界參與，組成籌備委員會，連結跨世代參與創新，結合產業行銷通路，與太平洋百貨合作辦理「設計金點年代-聖誕嘉年華」並協助設計作品產業化，推薦得獎廠商申請登錄創櫃板作業。
* 匯流五法法案審查
國家通訊傳播委員會於民國104年底擬具「電子通訊傳播法」、「電信基礎設施與資源管理法」、「電信事業法」、「有線多頻道平臺服務管理條例」與「無線廣播電視事業與頻道服務提供事業管理條例」等五法草案，召開近20場跨部會審查會議及相關業者(含OTT業者)溝通會議，經行政院第3498次院會(民國105年5月5日)討論通過，送請立法院審議。
* 菸酒管理法修正草案審查
配合網路科技的發展，審查財政部擬具菸酒管理法部分條文修正草案，開放在網路上開放販賣酒品，並訂定配套管控措施。 
* 推動並草擬「企業資產擔保法草案」
自民國104年底開始召集十多次學者會議草擬草案條文，並於民國105年5月擬定。


* 適任問題
蔡玉玲首度於[[立法院|立法院]]備詢時，因回答不出來或回答不完整立委們提出與蒙藏有關的的基本問題，被質疑無法勝任委員長一職。並因蔡玉玲創辦的理慈律師事務所專長為科技法律，恰好與其兼任政務委員督導業務相同，亦被質疑有利益衝突。對此，蔡玉玲委屈表示：「利益迴避是看個人，不是在位置，到了年齡不是為了名利，希望在能在公部門盡點心力。」蔡玉玲也忍不住哽咽落淚的說：「不要嚇壞企業界的人不敢進來，不要擋掉企業界想進來的人。」<ref>{{Cite news|url = http://www.chinatimes.com/newspapers/20131119000436-260102|title = 立委炮轟 蔡玉玲哽咽落淚|last = 顧佳欣|first = |date = 2013-11-19|work = 中時電子報|accessdate = }}</ref><ref>{{Cite news|url = http://www.cna.com.tw/news/aipl/201311180251-1.aspx|title = 蒙藏業務不熟 蔡玉玲：速了解|last = 蘇龍麒|first = |date = 2013-11-18|work = 中央社|accessdate = }}</ref>

* 兩岸問題
蔡玉玲於[[立法院內政委員會|立法院內政委員會]]審查蒙藏委員會明年度預算時，對於[[民主進步黨|民進黨]]立委[[陳其邁|陳其邁]]質詢兩岸是否同屬中國問題，表示要交給[[行政院大陸委員會|陸委會]]回答。立法院內政委員會會議主席、[[中國國民黨|國民黨]]立委[[吳育昇|吳育昇]]表示：「這樣的政務官是至為不當的作法，沒有辦法捍衛憲法政策」，並對蔡玉玲說：「妳連國家政策都不敢講，要我們怎麼支持妳！」因此他隨即宣布會議散會，預算擇期再審。<ref>{{Cite news|url = http://www.appledaily.com.tw/realtimenews/article/new/20141023/493174/|title = 「這樣的政務官不當」　吳育昇痛批蔡玉玲|last = 黃揚明|first = |date = 2014-10-23|work = 蘋果日報|accessdate = }}</ref>
