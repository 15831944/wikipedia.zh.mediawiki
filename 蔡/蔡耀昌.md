'''蔡耀昌'''（'''Tsoi Yiu Cheong Richard'''，{{bd|1967年9月11日||||CatIdx=C蔡}}），[[香港|香港]][[民間監管公共事業聯委會|民間監管公共事業聯委會]]發言人、[[支聯會|支聯會]]副主席、[[真普選聯盟|真普選聯盟]]成員團體代表、[[民主動力|民主動力]]執委、[[民主黨_(香港)|民主黨]]中委，曾任[[沙田區|沙田區]]區議員。祖籍[[廣東|廣東]][[深圳|深圳]]，中學時期開始參與社運，大學時期支援[[八九民運|八九民運]]並參與「[[六四事件|六四]]」鎮壓後的營救工作。1993年8月被沒收[[港澳居民來往內地通行證|回鄉證]]，不能踏足內地直至2016年。政界暱稱「阿狗」或「狗哥」<ref>[http://hk.apple.nextmedia.com/news/art/20100911/14439484] 隔牆有耳：華人與狗禁入西環，蘋果日報，2010年9月11日</ref>。

==早年生活==
蔡耀昌於[[慈雲山|慈雲山]]的[[香港公共屋邨|公共屋邨]]成長，其父生前為「[[苦力|苦力]]」，其母為家庭主婦。1975年，蔡父因病逝世。蔡於中學時期經常參與校內外的社會服務活動，並曾於暑假期間在[[茶樓|茶樓]]售賣[[點心|點心]]。由於其數學科成績一直排名全級第一，同學戲稱其為「數王」。蔡於1984年[[香港中學會考|香港中學會考]]考獲3A、2B、2C，翌年成為當時首屆「暫取生」，入讀[[香港中文大學|香港中文大學]]，主修數學系，副修社會學系<ref>[http://life.mingpao.com/cfm/dailynews3b.cfm?File=20081128/nclvx001/vx001b.txt] 原初，《門》很「核突」，明報通識網，2008年11月28日LIFE</ref>，入讀中文大學6年後(1991年)退學，未能畢業。<ref>[http://www.alumni.cuhk.edu.hk/magazine/nov98/html/p12.htm 蔡耀昌：年期只是空洞圖騰]</ref>

==參與學運的「叛逆歲月」==
{{see|香港中文大學學生會|香港專上學生聯會}}

蔡耀昌參與社會事務始於大學年代，屬學運出身的政治人物。

1985年，蔡耀昌入讀香港中文大學，先後出任[[中大學生會|中大學生會]]福利幹事、副會長（其時會長為學者[[蔡子強|蔡子強]]），其後出任[[中大學生會|中大學生會]]代表會主席及[[學聯|學聯]]代表會署理主席，並投入爭取「[[八八直選|八八直選]]」、爭取中大學生代表出任教務會正式成員，又帶領同學到訪中國大陸認識內地學生的社會實踐行動、反對「中大四改三」及於1989年協助學聯參與籌組支援[[八九民運|八九民運]]。因熱心學運而忽略學業，蔡最終不能完成首個學位課程<ref>[http://www.alumni.cuhk.edu.hk/magazine/nov98/html/p12.htm] 蔡耀昌：年期只是空洞圖騰，《中大校友》，CUHK Alumni Affairs Office，1998年11月</ref>，1994年入讀香港大學法律系並畢業。。

1990年，出任[[香港專上學生聯會|香港專上學生聯會]]秘書長。1998年，蔡耀昌與[[蔡子強|蔡子強]]、[[黃昕然|黃昕然]]、[[莊耀洸|莊耀洸]]等學運中人合作編撰學運文獻選輯《叛逆歲月：香港學運文獻選輯》及評論《同途殊歸：前途談判以來的香港學運》，成為研究香港學運發展的重要文獻。

==出任政治幕僚及參選經歷==
六四事件後，蔡耀昌在[[民主派|民主派]]政治陣營中擔任重要角色，1991年協助[[劉千石|劉千石]]及[[林鉅成|林鉅成]]當選首屆直選[[香港立法局|立法局]]議員（出任助選團秘書長），同年出任劉千石議員助理，專職議會政策研究，曾閱讀立法局所有文件，獲同業公認為質素高的議員助理。1995年出任民主派「[[新九組連線|新九組連線]]」助選團團長，協助李卓人及[[梁耀忠|梁耀忠]]等人當選立法局議員。同年出任劉千石、[[李卓人|李卓人]]、梁耀忠等人的聯合議員辦事處政策研究員。1995至97年出任立法局議員助理協會會長。

1996年，蔡與[[劉慧卿|劉慧卿]]等人籌組政治組織「[[前綫_(1996年)|前綫]]」（The Frontier)（後部分成員併入[[民主黨_(香港)|民主黨]]）。1997年， 民主派議員集體「落車」，蔡耀昌於1998年初轉任[[香港城市大學|香港城市大學]]當代中國研究中心研究員，期間與蔡子強合編《香港立法局重要投票紀錄匯編》。1998年中，蔡再度出任劉千石議員辦事處政策研究員。

[[2000年香港立法會選舉|2000年香港立法會選舉]]，與劉慧卿合組名單參選立法會（[[新界東|新界東]]選區），劉慧卿當選。同年，蔡組織學生運動「老鬼」聲援[[張韻琪|張韻琪]]等學運份子，反對政府打算檢控他們違反《[[公安條例_(香港)|公安條例]]》，政府最終不提告。

[[2000年香港立法會香港島地方選區補選|2000年香港立法會香港島地方選區補選]]，[[余若薇|余若薇]]當選立法會議員，蔡出任助選團核心小組成員。2001年，蔡代表[[香港人權聯委會|香港人權聯委會]]赴[[日內瓦|日內瓦]][[聯合國經濟社會文化權利委員會|聯合國經濟社會文化權利委員會]]游說，同行包括[[何喜華|何喜華]]、劉慧卿、[[涂謹申|涂謹申]]、李卓人、[[胡紅玉|胡紅玉]]、[[羅沃啟|羅沃啟]]等，成功推動委員會要求特區政府就禁止[[種族歧視|種族歧視]]進行立法。同年，蔡參與成立「反董連任大聯盟」，反對[[董建華|董建華]]連任行政長官。

2002年，出任新成立的[[民主動力|民主動力]]執委兼司庫（時召集人為[[鄭宇碩|鄭宇碩]]、副召集人劉慧卿及[[張超雄|張超雄]]，秘書為[[何俊仁|何俊仁]]）。

2003年統籌組織「[[七一大遊行|七一大遊行]]」號召五十萬人上街，其後特區政府宣布無限期擱置「[[香港特別行政區基本法第二十三條|廿三條]]」立法。2004年始擔任[[香港市民支援愛國民主運動聯合會|香港市民支援愛國民主運動聯合會]]（支聯會）常委，2008年起任副主席及維權部負責人至今；2005至06年間先後受[[美國政府|美國政府]]、[[歐洲議會|歐洲議會]]及歐洲委員會邀請，走訪[[美國|美國]]多個城市、[[歐盟|歐盟]]總部及歐洲議會。‬

[[2003年香港區議會選舉|2003年香港區議會選舉]]中，蔡耀昌出選[[沙田區|沙田區]][[火炭_(選區)|火炭]]選區當選，成為[[沙田區議會|沙田區議會]]議員，惟於[[2007年香港區議會選舉|2007年香港區議會選舉]]中以184票之差敗於[[親建制派|親建制派]]、[[新世紀論壇|新世紀論壇]]的[[龐愛蘭|龐愛蘭]]，連任失敗。

[[2012年香港立法會選舉|2012年香港立法會選舉]]中，蔡耀昌代表民主黨參選[[新界東|新界東]]選區，以10,028票排第14名落選，被沒收選舉按金。

==籌辦2003年「七一大遊行」==
{{see|民間人權陣線|七一大遊行}}

2003年7月1日，[[香港主權移交|香港主權移交]]中國六週年，當年的[[七一大遊行|七一大遊行]]，是[[八九民運|八九民運]]以來最大規模的遊行示威，受到國際輿論重視。由於《[[香港基本法第二十三條|香港基本法第二十三條]]》的立法程序，加上[[非典型肺炎|SARS]]事件導致香港社會不景氣，大量香港市民不滿。遊行主題為「反對廿三，還政於民」，大會呼籲市民穿黑色衣服參與遊行，以表達對政府的不滿。雖然當日香港天氣炎熱，但有大量人士參與遊行。主辦團體[[民間人權陣線|民間人權陣線]]（民陣）估計遊行人數超過50萬，而警方則公佈當日截至下午六時，由起點到終點之間共有35萬人。[[司徒華|司徒華]]在回憶錄中提到，他收到消息，警方內部獲悉的遊行人數為67萬5千人。遊行所引發的七一效應，使政府停止對二十三條的立法程序，並使[[親建制派|親建制派]]在同年的[[2003年香港區議會選舉|2003年香港區議會選舉]]大敗。

[[路透社|路透社]]、[[美聯社|美聯社]]、[[法新社|法新社]]、[[美國有線新聞網|美國有線新聞網]]、[[英國廣播公司|英國廣播公司]]和[[德新社|德新社]]等西方[[通訊社|通訊社]]，圖文並茂報道香港的大遊行。美聯社形容，示威人士是「憤怒和憂慮的香港人」；路透社稱：「示威者來自社會各階層，包括商人、退休人士，也有年輕夫婦推嬰兒車，與著名的民主派人士一起遊行」。

當年的蔡耀昌並非[[泛民主派|泛民]]重量級人物，擔任民陣的二十三條工作組召集人，事後回想，「嗰時覺得好大鑊，又唔知（[[SARS|沙士]]）幾時完，宣傳又難，加上23條去到法案審議階段，好技術性，冇人會理，但點都要堅持」。原來民怨已在沙士期間不斷升溫：「沙士前年輕人唔理政治，但沙士嗰時返唔到學，中學生、大學生因為冇嘢做，又開始有網上討論區，就開始留意時事。」 回看遊行，蔡耀昌認為當年50萬人上街，令特區政府撤回23條立法，向新一代年輕人證明，只要市民團結，和平遊行一樣可以成功：「7.1就係大家一齊做到一樣嘢，改變到政局。」<ref>[http://hk.apple.nextmedia.com/news/art/20130619/18303721]蔡耀昌信遊行可改變政局，《蘋果日報》，2013年6月19日。</ref>

== 協助新移民申領綜援爭議 ==
2013年12月17日，蔡耀昌以[[香港社區組織協會|香港社區組織協會]]幹事身份，協助一名新移民就申領綜援的居港七年期限提出司法覆核，獲終審法院裁定勝訴<ref>{{cite web |url=http://m.rthk.hk/news/20131217/971131.htm |title=存档副本 |accessdate=2013-12-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20131219191232/http://m.rthk.hk/news/20131217/971131.htm |archivedate=2013-12-19 }}，終院：綜援居港7年限制抵觸人口政策，香港電台，2013年12月17日</ref>，事件引發爭議，被指顛覆新移民福利制度，更成為中國內地人士假結婚來港的誘因。<ref>[http://eastweek.my-magazine.me/?aid=30765 綜援期限違憲 假結婚勢湧現 舔盡香港]，東周網</ref>他之後受到不少網民的攻擊和辱罵，其社交網頁帳戶，在短短一個月內已收到來自近一百人的近一千個惡意留言，包括罵他做「賣港賊」，要求他「落地獄」、「絕子絕孫」，甚至揚言會對其妻子及女兒施以性暴力、「揸把牛肉刀喺街斬死佢哋」等，令其妻子感到憤怒及困擾，社協本身亦收到不少惡意及滋擾性的電話及電郵。蔡隨後與何喜華一同到紅磡警署報案。不少評論指蔡經此一役，在政治上受到打擊，此後的選舉中都會被對手以此議題作為攻擊的目標。<ref>[http://the-sun.on.cc/cnt/news/20140118/00407_081.html 受網絡恐嚇 蔡耀昌報警] 2014年01月18日 太陽報</ref>

== 獲發回鄉證 ==
蔡耀昌在1993年8月陪同內地工運領袖[[韓東方|韓東方]]回內地期間，在廣州遭沒收回鄉證及遣返回港，此後便一直不能踏足中國內地，也未能成功重新申證，直至2016年12月17日，蔡被中共當局發還回鄉證，這是因為中共早前透過[[幫港出聲|幫港出聲]]的[[周融|周融]]發佈消息，會向香港反對派人士發還回鄉證，被沒收或註銷回鄉證的[[民主派|民主派]]人士可獲「解凍」重新申請領證。他拿證後即日在[[羅湖|羅湖]]過關再乘[[和諧號|和諧號]]動車上廣州，又拍照留影，這是他23年以來再次踏足內地。他坦言經過[[羅湖橋|羅湖橋]]時「有點感觸」，堅稱日後「該做的事都會繼續」，如與內地[[維權人士|維權人士]]見面等。在內地關口，他感覺比其他人需時較長，過程約2分鐘，關員在電腦輸入一些資料，不過過程中關員沒有向他提問。<ref>{{cite news|title=23年來首次 支聯會副主席蔡耀昌獲發回鄉證 成功過關到廣州|url=http://www.hk01.com/sns/article/60828|agency=香港01|date=2016-12-18}}</ref><ref>{{cite news|title=【解禁回鄉證】支聯會蔡耀昌獲「綠燈」 23年後穿六四Tee再踏足內地|url=http://hk.apple.nextmedia.com/realtime/news/20161218/56064043|publisher=蘋果日報|date=2016-12-18}}</ref>
