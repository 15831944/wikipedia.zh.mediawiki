'''蔡加讚'''{{BBSHK}}（'''Karson Choi'''，{{bd|1986年||catIdx=Choi蔡}}），生于[[香港殖民地時期|英属香港]]，籍貫[[廣東|廣東]][[揭陽|揭陽]]，[[香港|香港]][[企業家|企業家]]，「玩具大王」[[蔡志明|蔡志明]]之独子，全球最大規模的玩具生產商<ref name="toyindustry">{{cite web|url=https://www.industryhk.org/tc/news_detail.php?id=1825931|title=香港工業總會|author=|date=|website=www.industryhk.org}}</ref>旭日國際集團現任[[副董事長|董事會副主席]]及香港高級鐘錶零售商譽一鐘錶集團[[董事長|董事會主席]]，同時亦是極速汽車集團董事會主席。

蔡加讚是[[中國人民政治協商會議全國委員會香港地區委員|港區全國政協委員]]、廣西政協常委兼召集人、香港廣西社團總會會長、香港各界扶貧促進會會長、九龍社團聯會首席會長及香港政協青年聯會創會會長。蔡加讚亦是香港義工聯盟常務副主席、[[懲教署|懲教署]]更生先鋒領袖執行總監及安基司國際教育集團校董。

蔡加讚在[[美國南加州大學|美國南加州大學]]政治經濟學系畢業，其後加入父親經營的旭日國際集團，並於2010年起收購及接管喜運佳鐘錶集團業務並開設譽一鐘錶<ref name ="captalCEO2012">{{cite web|url=http://www.capitalceo.com/common_emag/view_content.php?issue_no=101&pub_id=16&top_id=115&art_id=158952|title=喜運佳主席蔡加讚創譽一品牌 掀尊貴購錶風尚|author=|date=|website=Capital}}</ref>。

==事業==
2007年，蔡加讚出任旭日國際集團地產部行政總監，將[[九龍城廣場|九龍城廣場]]形象翻新，更名「KCP商場」，顧客流量於6個月內大增3倍<ref name="polyu">{{cite web |url=http://www.polyu.edu.hk/aado/mt33/ch5/choi-kt.pdf |title=蔡加讚 做實力派接班人}}</ref>。

2009年，蔡加讚開始投入玩具業務，成為旭日國際集團有限公司執行董事，管理科技部及人力資源部門<ref name="polyu"/>。旭日國際集團乃全球最大規模的玩具生產商<ref name="toyindustry" />，集團旗下有數十萬名員工，部門達40多個 <ref name="Early Light">{{cite web|url=http://www.earlylight.com.hk/en/index.html|title=Early Light International (Holdings) Ltd.|author=|date=|website=www.earlylight.com.hk}}</ref>。集團其他業務範圍包括醫療保健、零售，以及物業投資與管理 <ref name="polyu"/>。

2010年，旭日國際集團收購本地錶行喜運佳鐘錶集團，蔡加讚出任喜運佳鐘錶集團董事會主席，並建立譽一鐘錶品牌，當時分店數目為11間<ref name="Forbes">{{cite web|url=https://www.forbes.com/sites/forbesasia/2012/10/24/his-time-will-come/|title=Next Tycoons: Karson Choi's Time Will Come|first=Karl|last=Shmavonian|date=|website=Forbes}}</ref>。

2012年4月，在[[銅鑼灣|銅鑼灣]]開設譽一鐘錶旗艦店 <ref name="SCMP">{{cite web|url=https://www.scmp.com/article/1000682/karson-chois-unique-venture-flashy-if-somewhat-nervous-start|title=Karson Choi's Unique venture off to a flashy, if somewhat nervous, start|author=|date=2018-12-22|website=South China Morning Post}}</ref>，將業務擴展至澳門及中國大陸，中港澳三地分店數目增加一倍至20間以上。

2015年，KCP Auto Plaza（九龍城汽車廣場）正式開幕 <ref name="ODNKCP">{{cite web|url=http://www.orientaldaily.on.cc/cnt/lifestyle/20150123/00302_003.html|title=九龍城汽車廣場揭幕 世界名車匯集 - 東方日報|author=|date=|website=www.orientaldaily.on.cc}}</ref>，是香港首個精品式汽車廣場，佔地約60,000平方呎。譽一鐘錶佐敦旗艦店開業，雲集14個世界級名錶品牌，同年業務擴展至[[澳門銀河|澳門銀河]]時尚匯 <ref name="HWwebsitegalaxy">{{cite web |url=http://www.halewinner.com/tchi/news_halewinner_main.php?&page=2 |title=譽一鐘錶佐敦旗艦店開業及進駐澳門銀河時尚匯 |accessdate=2016-02-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20160417050130/http://www.halewinner.com/tchi/news_halewinner_main.php?&page=2 |archivedate=2016-04-17 }}</ref>。

2016年，譽一鐘錶於[[澳門巴黎人|澳門巴黎人]]擴展8間分店，並成為勞力士及帝舵在香港及[[澳門|澳門]]的零售商<ref name="MACAU">{{cite web|url=http://hd.stheadline.com/news/daily/hk/696928/|title=神奇小子 蔡加讚澳賭場展拳腳 心臟地帶開19鐘錶店|author=|date=|website=頭條日報 Headline Daily}}</ref><ref name="ZERO">{{cite web|url=https://eastweek.my-magazine.me/main/80812|title=由零開始 店舖數目冠絕全澳 獨家專訪 鐘錶界神奇小子 - 時事 - 新聞專題|author=|date=|website=東周網 東周刊官方網站 Eastweek.com.hk}}</ref> 。<br>蔡加讚與旗下投資的WowWee Group合作研發的人工智能小狗CHiP獲[[香港工業總會|香港工業總會]]頒發消費品設計大獎。<ref name="ELECTRIC">{{cite web|url=http://std.stheadline.com/daily/news-content-nocache.php?id=1520136&target=2|title=蔡加讚研發機械狗奪四連霸|author=|date=|website=std.stheadline.com}}</ref>

2017年，譽一鐘錶贊助農曆大年初二「[[香港賀歲煙花匯演|譽滿香江煙花大匯演]]」，慶祝[[香港特別行政區成立二十周年紀念|香港特區成立二十周年]]<ref name="FIRE">{{cite web|url=http://www1.hkej.com/dailynews/investment/article/1473517/%E5%96%9C%E9%81%8B%E4%BD%B3%E2%80%A2%E8%AD%BD%E4%B8%80%E9%90%98%E9%8C%B6+%E8%AD%BD%E6%BB%BF%E9%A6%99%E6%B1%9F%E7%85%99%E8%8A%B1%E5%A4%A7%E5%8C%AF%E6%BC%94|title=今日信報-- 喜運佳•譽一鐘錶 譽滿香江煙花大匯演|author=|date=|website=今日信報}}</ref>。同年，喜運佳鐘錶集團正式易名為「譽一鐘錶集團」<ref name="ABOUTUS">{{cite web |url=https://www.utpieces.com/zh/about-us/|title=譽一鐘錶 - 關於我們|author=|date=|website=}}</ref>。 

2018年，譽一鐘錶集團與瑞士高級腕錶品牌愛彼，推出皇家橡樹萬年曆腕錶，全球限量50枚<ref name="LIMIT">{{cite web|url=https://paper.hket.com/article/2137795|title=愛彼推出限量腕錶 慶祝與譽一鐘錶多年夥伴關係 |newspaper=香港經濟日報|author=|date=}}</ref>。

==事件==

;違規經營汽車展銷場
2011年，蔡加讚被揭在父親旗下物業[[九龍城廣場|九龍城廣場]]，將一層貨車停車場改作大型汽車展銷場分租，但展銷場未[[補地價|補地價]]，被質疑違規營業<ref>{{Cite web|title=汽車展銷場被揭違規 蔡加讚衰咗！|url=http://hk.appledaily.com/news/art/20110905/15584980|work=Apple Daily 蘋果日報|accessdate=2019-02-09}}</ref>。

==家庭==
蔡加讚与妻子汪圓圓育有一对双胞胎儿子和一女。他是家中孻子，三个姐姐分别是[[蔡加敏|蔡加敏]]、蔡加颖（已故，烧炭自杀，终年26岁）和[[蔡加怡|蔡加怡]]，父親是享有「玩具大王」美誉的旭日國際集團董事會主席[[蔡志明|蔡志明]]博士GBS[[太平紳士|太平紳士]]，母親为蔡太李惠莉。

==個人生活==
蔡加讚除了是年青企業家，同時亦是名車及鐘錶收藏家，其名表珍藏逾40枚<ref name="etnet">{{cite web |url=http://lifestyle.etnet.com.hk/column/index.php/management/executive/10362 |title=26歲CEO 孭起逾億鐘錶生意 |author= |date=2012-05-10 |agency=信報 生活副刊}}</ref>。此外，蔡加讚亦擁有全球限量20部的[[林寶堅尼|林寶堅尼]]Reventon<ref name="headline">{{cite web |url=http://www.hkheadline.com/culture/culture_content.asp?contid=53172&srctype=g |title=1500萬林寶堅尼夢想成真 蔡加讚打江山起革命 拒做二世祖 |author= |date=2012-05-10 |agency=頭條日報 |archiveurl=https://archive.is/hOjbJ |archivedate=2013-06-29}}</ref>及全球限量350部的Murcielago LP670-4 SuperVeloce<ref name="the sun">{{cite web|url=http://the-sun.on.cc/cnt/news/20100314/00408_004.html|title=政情：蔡加讚用辛苦錢買名車 - 太陽報|author=|date=2010-03-14|website=the-sun.on.cc}}</ref>。

另外，蔡加讚亦積極參與社會公益事務，協助青少年及弱勢社群，推動社會發展。如成立「蔡加讚童軍基金」，持續贊助香港童軍總會新界地域各類型交流活動及興建設施配套，包括：「香港童軍100周年-關愛長者樂滿心」活動、童軍總會新界地域大露營及即將於屯門落成的鑽禧館等<ref name="tm">{{cite web |url=http://www.scout-ntr.org.hk/mainweb/big5/about/history.php |title=香港童軍總會新界地域}}</ref>。他亦有透過[[香港義工聯盟|香港義工聯盟]]推動跨界別協作扶助弱勢社群，親身帶領年青人走訪各院舍探望長者<ref name="elderly">{{cite web |url=https://hk.celebrity.yahoo.com/news/%E8%88%87500%E9%95%B7%E8%80%85%E5%90%8C%E6%AD%A1-%E5%B1%A5%E8%A1%8C%E6%B8%AF%E4%BA%BA%E8%B2%AC%E4%BB%BB-%E8%94%A1%E5%8A%A0%E8%AE%9A%E5%B9%B4%E6%8D%90%E9%80%BE%E5%84%84%E8%A1%8C%E5%96%84-222110368.html |title=與500長者同歡 履行港人責任 蔡加讚年捐逾億行善}}</ref>，並為基層家庭及學生提供食物及升學資助<ref name="sponsor1">{{cite web |url=http://std.stheadline.com/yesterday/loc/0609ao05.html |title=3,000基層學生獲團體派免費奶}}</ref><ref name="sponsor2">{{cite web |url=http://www2.sjsgia.org.hk/index_topic.php?did=169640&didpath=/111071/218416/169640 |title=心連心行動慈善基金送贈 愛心書包及精美文具}}</ref>，以改善生活質素及跨代貧窮。他亦有持續捐獻慈善團體，包括[[保良局|保良局]]、[[香港公益金|香港公益金]]及[[東華三院|東華三院]]等 <ref name="donate">{{cite web |url=http://www.tungwah.org.hk/press-release/%E5%96%9C%E9%81%8B%E4%BD%B3%E2%80%A7%E8%AD%BD%E4%B8%80%E9%90%98%E9%8C%B6%E5%91%88%E7%8D%BB%EF%BC%9A%E6%9D%B1%E8%8F%AF%E4%B8%89%E9%99%A2145%E5%91%A8%E5%B9%B4%E6%85%88%E5%96%84%E6%99%9A%E5%AE%B4/ |title=喜運佳‧譽一鐘錶呈獻：東華三院145周年慈善晚宴「時光倒流 1870」善長齊心為東華三院社企籌款 福澤社群}}</ref>。
蔡加讚曾在《[[東周刊|東周刊]]》一篇題為《成功需苦幹 蔡加讚》的專訪中談及成功之道，旋即成為網絡討論區上被談論的對象。而[[廣東話|廣東話]]裡「苦」與「父」僅[[聲調|聲調]]不同，被部分[[網民|網民]]評為「成功需[[父幹|父幹]]」<ref name="prizes">{{cite web |url=http://app3.rthk.hk/special/pau/article.php?aid=442 |title=內地頂級富二代 金句被網民惡搞}}</ref>。2018年，蔡加讚在《[[東周刊|東周刊]]》回應「成功需父幹」一事，形容現時已為人父，真的需要「苦幹」兼「父幹」<ref name="prize">{{cite web |url=https://eastweek.my-magazine.me/main/80812 |title=由零開始 店舖數目冠絕全澳 獨家專訪 鐘錶界神奇小子}}</ref>。

2016年獲選為世界傑出華人青年<ref name="young">{{cite web |url=http://std.stheadline.com/daily/article/detail/984090-%E5%A8%9B%E6%A8%82-%E8%94%A1%E5%8A%A0%E8%AE%9A%E6%A6%AE%E7%8D%B2%E4%B8%96%E7%95%8C%E5%82%91%E5%87%BA%E9%9D%92%E5%B9%B4%E5%A4%A7%E7%8D%8E|title=蔡加讚榮獲世界傑出青年大獎}}</ref>。

2017年，蔡加讚獲特區政府授予[[銅紫荊星章|銅紫荊星章]]，嘉許語稱讚他熱心參與社會服務，表現傑出，尤其致力於鄉眾、青年義工及慈善方面的工作<ref name="work">{{cite web |url=http://paper.wenweipo.com/2017/07/06/CH1707060038.htm|title=【銅紫荊星章】蔡加讚：續獻力社會和諧}}</ref>。

2018年1月，當選第十三屆全國政協委員<ref name="china">{{cite web |url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html |title=中國人民政治協商會議第十三屆全國委員會委員名單}}</ref>。

此外，他亦是[[香港賽馬會|香港賽馬會]]的會員、馬主，名下馬匹包括「車神」、「宇宙車神」、「地上車神」，並分別交由[[約翰摩亞|約翰摩亞]]及[[蘇偉賢|蘇偉賢]]訓練。
