'''蔡尚精'''（遊戲代號：'''GreenTea'''，'''綠茶'''，{{bd|1991年|4月24日}}），出生於[[臺灣|臺灣]]，曾經為[[英雄联盟|英雄联盟]][[電子競技|電子競技]]職業選手。曾效力過[[Ahq电子竞技俱乐部|Ahq电子竞技俱乐部]]、[[MAD_Team|MAD Team]]，目前為[[Royal_Never_Give_Up|Royal Never Give Up]]戰隊英雄練盟項目戰術教練。

== 簡介 ==
主要參與《[[英雄聯盟|英雄聯盟]]》賽事，在該遊戲台港澳伺服器[[天梯|天梯]]榜上，屬前50名列為「頂尖召喚師」的[[菁英|菁英]]階級玩家，曾首次在同一時間內以兩支帳號包辦天梯榜第一、二名。曾隸屬於[[鈦宇數位|鈦宇數位]]旗下的[[ahq_e-Sports_Club|ahq e-Sports Club]]職業戰隊，擔任選手及戰隊教練。同時也是名知名[[戲實況|戲實況]]玩家，其擁有的[[Twitch|Twitch]]實況頻道累計觀看人次達六百萬，並擁有七萬兩千名追隨者。


== 經歷 ==
*2013年6月13日，與電子競技用品品牌EG（EpicGear藝極科技）簽約成為其產品年度代言人，並在網路上公開直播簽約記者會過程。

*2017年底離開[[ahq_e-Sports_Club|ahq e-Sports Club]]並加入[[MAD_Team|MAD Team]]擔任教練。<ref>{{Cite web |url =https://www.facebook.com/MadTeam.club/photos/a.295026821007389/294768397699898/?type=3&theater|title =瘋狂電競有限公司 (MAD CREW ESPORT LTD.) 新聞稿|author =|date =2017年11月29日 |publisher =|language =zh-tw }}</ref>

*2018年10月帶領[[MAD_Team|MAD Team]]以[[LMS職業聯賽|LMS職業聯賽]]第二種子身份進入[[英雄联盟2018赛季全球总决赛|英雄联盟2018赛季全球总决赛]]，最後止步16強。<ref>{{Cite web |url =https://www.4gamers.com.tw/news/detail/36739/2018-lol-worlds-group-stage-day6-group-c-mad|title =《英雄聯盟》MAD不敵EDG，今年小組賽確定遭淘汰|author =|date =2018年10月16日 |publisher =[[4gamers|4gamers]] |language =zh-tw }}</ref>

*2018年12月16日與[[MAD_Team|MAD Team]]協商確定加入[[Royal_Never_Give_Up|Royal Never Give Up]]成為戰術教練，並將暫替原[[Royal_Never_Give_Up|Royal Never Give Up]]孫大永教練，為隊伍完成 B/P 工作。
