'''蔡廷锴'''{{BD|1892年|4月15日|1968年|4月25日|catIdx=Cai蔡}}，[[广东|广东]][[罗定|罗定]]人，字贤初。民国将军。是[[區壽年|區壽年]]舅舅。

== 早年經歷 ==
北伐時任國民革命粵軍第四軍二十八團團長。

== 中年經歷 ==
[[File:蔡將軍香煙.jpg|thumb]]後，市場出現一款蔡廷鍇牌香煙，此為當時廣告。|替代=]]
1927年率第十師参加[[南昌起義|南昌起義]]。而據張國燾《我的回憶》所述：八月…，“從李家渡到撫州的途中，周恩來等候在路旁，與我會商一件重要的事情，他懊喪的說：據一位逃出來的同志報告，行進到進賢的第十師蔡廷鍇，業已叛變，拖著隊伍向東跑了。蔡廷鍇在進賢藉集合幹部訓話為名，將三十團團長范孟聲及其他中共同志三十餘人，一律扣留殺害，其餘同志數十人，也下落不明。…”。1930年，升任十九军军长。1932年6月，任十九路军总指挥。1930年参与了进攻中共[[苏区|苏区]]的战斗。1932年[[一二八事變|一二八事變]]，率领[[十九路军|十九路军]]在[[上海|上海]]英勇對日抗战。

对于[[蔣介石|蔣介石]]的反共政策抱有疑虑。随后被[[蔣介石|蔣介石]]调往[[福建省_(中华民国)|福建]]和[[中国工农红军|红军]]作战，1934年率领[[十九路軍|十九路軍]]参加[[閩變|閩變]]，失败后逃亡海外。1935年回国后，担任[[国民革命军第16集团军|国民革命军第16集团军]]总司令，参加[[抗日战争|抗日战争]]。1939年与罗西欧结婚。抗战结束后，积极倡导和平。

1946年在[[广州|广州]]与[[李济深|李济深]]等共组[[中国国民党|中国国民党]][[中國民主促進會|民主促進會]]。1947年1月6日，國民政府令：經核准退役之陸軍中將蔡廷鍇任為陸軍上將<ref name="民國史大事記">{{cite book |editor = 李新 總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室 編，韓信夫、姜克夫 主編 |title=《中華民國史大事記》 |location=[[北京|北京]] |publisher=[[中華書局|中華書局]] |date=2011年7月}}</ref>{{rp|8258}}。蔡廷鍇赴[[香港|香港]]组织了行[[中国国民党|中国国民党]][[國民黨左派|民主派]]第一次代表会议。1948年1月，蔡廷锴与李济深等人在[[香港|香港]]发起组织[[中国国民党革命委员会|中国国民党革命委员会]]，任中央常务委员兼财政部部长。5月5日，蔡廷锴代表[[中國民主促進會|民促]]，与其他[[民主黨派|民主黨派]]领导人和[[無黨派|無黨派]]民主人士，联名致电[[中共中央主席|中共中央主席]][[毛泽东|毛泽东]]，响应中共“[[五一口號|五一口號]]”，拥护召开[[中國人民政治協商會議|新政协会议]]。1948年7月1日的[[豫東戰役|豫東戰役]][[龍王店戰鬥|龍王店戰鬥]]中，負責鎮守[[開封|開封]]的外甥[[區壽年|區壽年]]（第七兵團中將司令），被[[中國人民解放軍|中國人民解放軍]]所俘。

== 晚年經歷 ==
1949年后，先后当选为[[中央人民政府委员|中央人民政府委员]]、[[全国政协|全国政协]]常务委员、[[全国人大常委会|全国人大常委会]]委员，后又任[[全国政协副主席|全国政协副主席]]，[[中华人民共和国国防委员会|中华人民共和国国防委员会]]副主席，[[国家体委|国家体委]]副主任，[[中国国民党革命委员会|中国国民党革命委员会]]中央委员会副主席及[[世界和平理事会|世界和平理事会]]理事。

朝鲜战争爆发后，担任抗美援朝保家卫国委员会常务委员。1951年，蔡廷锴被推为慰问团总团的副团长赴朝鲜。

“文革”期间作为周恩来运作保护的民主人士之一，躲过一劫<ref>{{cite book|title=上海滩|year=2004|publisher=《上海滩》杂志社}}</ref><ref name="刘武生2006">{{cite book|author=刘武生|title=周恩来的晚年岁月|year=2006|publisher=人民出版社|isbn=978-7-01-005338-7}}</ref>。

1968年4月25日，蔡廷锴在[[北京|北京]]逝世。

== 纪念 ==
*[[廷鍇紀念中學|廷鍇紀念中學]]，位於廣東省羅定市，是該市人民政府為紀念羅定籍蔡廷鍇將軍，於1989年12月18日創辦廷鍇紀念中學，於1990年9月正式招生辦學，學校現已發展成為一所五千多學生的重點中學。
