'''蔡徐坤'''（{{bd|1998年|8月2日||}}），[[中国大陆|中国大陆]][[男歌手|男歌手]]，[[演员|演员]]。出生于[[浙江省|浙江]]，祖籍[[湖南省|湖南省]][[怀化市|怀化市]]<ref>{{Cite web|url=http://www.huaihuashizx.com/hhxw/33341.html|title=偶像练习生冠军蔡徐坤居然是怀化人|accessdate=2018-12-05|publisher=怀化新闻}}</ref>。2012年4月，蔡徐坤在综艺节目《[[向上吧！少年|向上吧！少年]]》中入围全国200强，借此进入娱乐圈<ref>{{cite web|url=http://tv.sohu.com/20120604/n344749552.shtml|title=向上吧！少年 20120604 小正太蔡徐坤萌翻观众|date=2012-06-04}}</ref>；同年8月，参演偶像剧《[[童话二分之一|童话二分之一]]》。2014年3月，参演电影《[[完美假妻168|完美假妻168]]》；2015年7月，蔡徐坤参加真人秀节目《[[星動亞洲|星動亞洲]]》，并成功进入全国前十五强；2016年10月，蔡徐坤通过10人男子组合[[SWIN|SWIN]]正式出道；2018年1月，参加[[爱奇艺|爱奇艺]]《[[偶像練習生|偶像練習生]]》，在初次評級獲得全場首個A等級<ref>{{cite web|url=http://www.iqiyi.com/v_19rrfrjcuc.html|title=【偶像练习生】蔡徐坤：评级揭晓不负众望获全场首A|date=2018-01-19}}</ref>，最終以47640887票，獲得該節目第一名，並以[[NINE_PERCENT|NINE PERCENT]]之隊長及C位出道<ref>{{Cite web|url=https://m.weibo.cn/6382798471/4225938050279118|title=微博|accessdate=2018-08-14|work=m.weibo.cn|language=zh}}</ref>。

== 早年经历 ==
蔡徐坤出生于湖南省怀化市，父亲是[[浙江省|浙江]][[温州市|温州]]人，母亲是湖南怀化人，是一名演员。他的童年是在湖南与父母和祖父母一起度过的，他曾在湖南担任校长助理和学生会主席。

在上小学时，蔡徐坤被选为[[TFBOYS|TFBOYS]]学员之一。因他的父母希望他能专心学习，所以他放弃了这一机会。在深圳[[南山区|南山]]第二实验中学毕业后，选择了出国留学。他在美国[[加利福尼亚州|加利福尼亚州]]的格雷斯兄弟高中（{{lang|en|Grace Brethren High School}}）学习，在那里他加入了学校的篮球队。

== 演藝經歷 ==
2018年4月12日，蔡徐坤在優酷獨播影片《[[我才不會被女孩子欺負呢|我才不會被女孩子欺負呢]]》中飾演男主角——葉麟。這也是他的首部主演網劇。

== 争议 ==
2019年1月18日，NBA官方在新浪微博宣布蔡徐坤为首位NBA新春贺岁形象大使，并与三位球员拍摄宣传片，此举引发质疑。球迷翻出蔡徐坤以往节目中打球的片段，并质疑其球技。同时，打球片段也被网友再创作以进行调侃或者恶搞。<ref>{{Cite web|title=蔡徐坤给NBA当形象大使，凭什么？|url=https://news.pedata.cn/508885.html|work=news.pedata.cn|accessdate=2019-04-21}}</ref>

2019年4月12日，蔡徐坤工作室转发律师告知函，称[[Bilibili|哔哩哔哩视频弹幕网]]（B站）上存在大量严重侵犯蔡徐坤权益的打篮球[[鬼畜视频|鬼畜视频]]，内容涉嫌故意诽谤、滥用肖像，还使用了诸多侮辱性词汇，且对蔡徐坤的表演视频素材进行了恶意剪辑。<ref>{{Cite web|url=http://www.mnw.cn/news/ent/2150833.html|title=蔡徐坤工作室向B站发律师函：存在大量严重侵犯蔡徐坤的内容|accessdate=|author=闽南网|date=2019-04-12|publisher=}}</ref>同日，B站回应蔡徐坤方律师声明，相信法律自有公断。<ref>{{Cite web|url=http://ent.sina.com.cn/s/m/2019-04-12/doc-ihvhiqax2180538.shtml|title=B站回应蔡徐坤方律师声明：相信法律自有公断|accessdate=|author=新浪娱乐|date=2019-04-12|publisher=}}</ref>
