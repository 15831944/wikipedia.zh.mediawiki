'''蔡楚生纪念铜像及碑记'''，是[[潮阳区文物保护单位|潮阳区文物保护单位]]之一，位于中国[[广东省|广东省]][[汕头市|汕头市]][[潮阳区|潮阳区]][[文光街道|文光街道]]兴归居委，在[[潮陽文光塔|潮陽文光塔]]下方。此铜像是中国电影导演[[蔡楚生|蔡楚生]]的{{link-en|半身像|Bust (sculpture)|半身铜像}}，为纪念其逝世20周年而设，铜像后为纪念碑。

== 历史 ==
[[蔡楚生|蔡楚生]]（1906年－1968年）是一名中國[[電影導演|電影導演]]，对推动[[中国电影|中国电影]]前进有所贡献，在[[文化大革命|文化大革命]]中受迫害，后获恢复名誉<ref name="潮南区文学艺术界联合会"/>。

1989年，为纪念蔡楚生逝世20周年，[[中国电影家协会|中国电影家协会]]广东分会、[[中国电影技术研究中心|中国电影技术研究中心]]、[[中国共产党|中国共产党]]汕头市委员会、中国共产党潮阳县委员会（当时为[[县级行政区|县]]）、[[珠江电影制片公司|珠江电影制片公司]]共同组织纪念活动，并决定在蔡楚生家乡潮阳的潮陽文光塔下，建造其半身铜像和纪念碑<ref name="潮南区文学艺术界联合会"/><ref name="文集"/>。铜像在2月21日落成揭幕，由中国共产党潮阳县委员会书记[[卓胜泰|卓胜泰]]主持揭幕仪式，[[广播电影电视部|广播电影电视部]]副部长[[陈吴苏|陈吴苏]]、[[中国人民政治协商会议广东省委员会|中国人民政治协商会议广东省委员会]]副主席[[吴南生|吴南生]]负责揭幕，广播电影电视部及下属電影管理局、中国电影家协会及广东分会、珠江电影制片公司分别向铜像献花篮<ref name="年鉴"/><ref name="中央文献出版社"/>。

== 结构与设计 ==
蔡楚生紀念銅像为{{link-en|半身像|Bust (sculpture)}}，由广州雕塑家[[尹积昌|尹积昌]]制作，銅像描绘的蔡楚生戴上眼镜，头发中分、额头饱满、鼻梁挺直<ref name="文集"/>。半身像下为基座，高度约是一人多高，正面刻有[[周恩來|周恩來]]夫人[[邓颖超|邓颖超]]的“蔡楚生先生是中国进步电影的先驱者！”题词<ref name="潮南区文学艺术界联合会"/><ref name="环球潮人网"/>。铜像后方建有一面纪念碑墙，名为“蔡楚生同志之碑”，由剧作家[[夏衍|夏衍]]撰文、艺术家[[黄苗子|黄苗子]]代为起草、吴南生手书碑上<ref name="环球潮人网"/><ref name="南方都市报"/>。

== 保护 ==
1992年4月7日，蔡楚生紀念銅像及碑記被潮阳县人民政府列为第三批县级文物保护单位之一（现为区级）<ref name="潮阳市志"/>。1996年，蔡楚生紀念銅像及碑記被中國共產黨潮陽市委員會、潮阳市人民政府（当时潮陽为[[县级市|县级市]]）列为[[爱国主义教育基地|爱国主义教育基地]]<ref name="文集"/>。
