'''蔡振华'''（{{bd|1961年|9月3日|catIdx=C}}），[[江苏|江苏]][[无锡|无锡]]人，[[中华人民共和国|中国]][[乒乓球|乒乓球]]运动员、教练员。1997年加入[[中国共产党|中国共产党]]，是[[中国共产党第十七届中央委员会候补委员|中共第十七届]]、[[中国共产党第十八届中央委员会候补委员|十八届中央候补委员]]。曾任[[国家体育总局|国家体育总局]]副局长，现任[[中华全国总工会|中华全国总工会]]副主席、书记处书记，并担任[[中国足球协会|中国足球协会]]、[[中国乒乓球协会|中国乒乓球协会]]和[[中国羽毛球协会|中国羽毛球协会]]主席。

== 球员生涯 ==
1972年，蔡振华进入无锡业余体校接受乒乓球训练；次年进入[[江苏省|江苏省]]乒乓球队；1978年进入[[中国|中国]]国家队，次年成为国家队主力。1981年，蔡振华和队员获[[世界乒乓球锦标赛|世界乒乓球锦标赛]]男子团体、男子双打冠军（与[[李振恃|李振恃]]），并获得男子单打的亚军。1983年，蔡振华成为中国男子乒乓球队的队长，并率队于1983年再度夺取世锦赛团体冠军，1985年蔡同[[曹燕华|曹燕华]]合作获得世锦赛混合双打冠军。

蔡振华原为[[削球|削球]]手，后改为左手横拍两面攻结合[[弧圈球|弧圈球]]打法。其球拍两面颜色一样但性能不同，因此蔡练就一手发球时倒球拍的怪招。如果对手猜错他使用哪一面发球，就会造成接发球质量不高，甚至直接失分。在1981年世锦赛男子团体决赛中，[[匈牙利|匈牙利]]球星[[盖尔盖伊|盖尔盖伊]]在决胜局15：15平时，连续猜错五次，直接导致匈牙利队陷入被动。为了限制这种情况，[[国际乒联|国际乒联]]修改规则，规定运动员球拍两面胶皮必须采用不同颜色。因此，也促使蔡振华决定退役。

== 教练生涯 ==
退役后，蔡振华曾被公派前往[[意大利|意大利]]执教4年。1991年，中国国家男队在世锦赛中仅取得第七名的成绩，主教练[[郗恩庭|郗恩庭]]引咎辞职；在其力邀下，蔡振华回国接任男队主教练的工作。

蔡接手主教练一职后，以男子双打为突破口，培养出[[王涛_(乒乓球运动员)|王涛]]和[[吕林|吕林]]组合，并夺得[[1992年夏季奥林匹克运动会|1992年夏季奥林匹克运动会]]的金牌。在单打方面，蔡振华启用了一批全新打法的球手，如第一个熟练掌握[[直拍反打|直拍反打]]技术的[[刘国梁|刘国梁]]和攻击力超强的削球手[[丁松|丁松]]。1995年，蔡振华率队一举夺回世锦赛男团冠军，从而重新让中国男子乒乓球队独步天下。

因此，蔡振华也于1997年升任中国乒乓球队的总教练。其后，他率领中国乒乓球队先后率队包揽了[[悉尼奥运会|悉尼奧運會]]和2001年世乒赛的全部冠军。

== 管理生涯 ==
2002年11月，蔡振华升任体育总局乒羽管理中心副主任（副局级）。2004年11月22日，升任乒羽管理中心主任兼党委副书记，2005年，被任命为[[国家体育总局|国家体育总局]]局长助理。2006年，蔡振华当选成为[[中国羽毛球协会|中国羽毛球协会]]副主席。2007年4月，蔡振华晋升为[[国家体育总局|国家体育总局]]的副局长，并在2010年8月分管[[中国足协|中国足协]]工作。2009年2月18日，他被选举为中国羽毛球协会主席<ref>[http://sports.sohu.com/20090218/n262318118.shtml 中国羽协换届选举 蔡振华当选主席副主席共24人]</ref>。同年2月27日，他接替[[徐寅生|徐寅生]]，出任[[中国乒乓球协会|中国乒乓球协会]]主席，成为中国体育史上首个兼任乒羽两大协会主席的人物<ref>[http://sports.qq.com/a/20090227/000505.htm 蔡振华任中国乒协主席 18副主席辅佐欲续辉煌]</ref>。2014年1月17日，蔡振华连任中国乒乓球协会主席<ref>[http://tabletennis.sport.org.cn/home/xhxw/gnxw/2014-01-20/427876.html 中国乒协换届大会落幕 蔡振华连任主席] {{webarchive|url=https://web.archive.org/web/20140202210043/http://tabletennis.sport.org.cn/home/xhxw/gnxw/2014-01-20/427876.html |date=2014-02-02 }}</ref>。1月21日，他当选为新一任中国足球协会主席<ref name="Cai to CFA">[http://www.apdnews.com/photo/63346.html 蔡振华当选新任中国足协主席] {{webarchive|url=https://web.archive.org/web/20140121172703/http://www.apdnews.com/photo/63346.html |date=2014-01-21 }}，亚太日报，2014年1月21日</ref>。1月24日，他连任中国羽毛球协会主席成功，8天内连续当选三个球类协会的主席<ref>[http://sports.china.com.cn/live/2014-01/24/content_24530020.htm 一人兼三项！蔡振华8天内连续当选乒足羽协会主席] {{webarchive|url=https://web.archive.org/web/20140203022411/http://sports.china.com.cn/live/2014-01/24/content_24530020.htm |date=2014-02-03 }}</ref>。蔡振华在担任足协主席期间因由于[[中国国家足球队|中国国家足球队]]战绩持续低迷而屡遭诟病，其对足球的冷漠态度更被球迷质疑<ref>{{cite news|url=http://sports.163.com/api/photoview/0B6P0005/134546.html#p=B8VHBQ200B6P0005|title=天河集结!现场球迷标语爆粗口怒骂蔡振华|publisher=网易体育|date=2015-11-21|accessdate=2015-11-24}}</ref><ref>{{cite news|url=http://www.taihainet.com/news/media/hot/2015-11-22/1593872.html|title=三张图告诉你 为何全国盼蔡振华必须下课|publisher=台海网|date=2015-11-22|accessdate=2015-11-24}}</ref>。

2016年11月[[苟仲文|苟仲文]]接替[[刘鹏|刘鹏]]出任国家体育总局局长。2017年6月爆发的[[国乒兵变|国乒兵变]]，使得苟蔡两人不合、政治斗争的坊间传闻进入公众视线<ref>{{Cite web |url=http://www.sohu.com/a/153127183_488304|title=关于刘国梁事件，你所不知道的事实... |accessdate=2017-07-06 |author= |date=2017-06-29|work= |publisher=搜狐网|language=Zh-hans }}</ref>。2017年7月，蔡振华未当选中共十九大代表。

2018年9月29日，不再担任国家体育总局副局长<ref>{{Cite web |url=http://sports.people.com.cn/n1/2018/0929/c22155-30321511.html|title=国务院任免国家工作人员 免去蔡振华体育总局副局长一职|accessdate=2018-09-30 |author=欧兴荣 |date=2018-09-29|work= |publisher=人民网-体育频道|language=Zh-hans }}</ref>。

== 进入全总 ==
2018年9月，蔡振华进入[[中华全国总工会|中华全国总工会]]，任党组成员，并于9月20日当选全国总工会副主席、书记处书记<ref>{{Cite web |url=http://sports.sina.com.cn/others/pingpang/2018-09-21/doc-ihkhfqnt4903218.shtml|title=蔡振华当选全国总工会副主席 目前仍是足协主席|accessdate=2018-09-23 |author= |date=2018-09-21|work= |publisher=澎湃新闻|language=Zh-hans }}</ref>。
