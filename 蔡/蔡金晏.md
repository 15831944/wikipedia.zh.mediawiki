'''蔡金晏'''（{{bd|1978年|3月25日||}}），政治人物，出生於[[高雄市|高雄市]]，攻讀完[[國立成功大學|國立成功大學]]水利及海洋工程工學博士學位後，參與[[2010年中華民國縣市改制直轄市|高雄縣市合併]]後第一屆[[高雄市議會|高雄市議會]]第6區（旗津、鼓山、鹽埕區）議員選舉<ref>{{Cite web |url = http://www.twtimes.com.tw/index.php?page=news&nid=304364 |title = 名人開講 蔡松雄 海派媽註 鐵漢柔情 |author = 吳榮偉、陳萬強、呂佩琍 |date = 2013年1月20日 |publisher = 臺灣時報 |language = zh-hant |accessdate = 2015年10月14日 |quote = 問：交棒給下一代後，副議長對長子蔡金晏議員有何期望？其他子女呢？答：金晏當選市議員後，盡心盡力的作好民意代表份內的工作，辦事比我還要認真，令我感到十分欣慰，當然年輕人也有自己的想法和看法，我都會予以尊重與支持。至於明年有機會是否讓他更上一層樓？順其自然就好。}}</ref>，續任第二屆[[高雄市議會|高雄市議會]]議員以後，正式加入參選[[2016年中華民國立法委員選舉|2016年中華民國立法委員選舉]]鹽埕、鼓山、旗津、三民西區立委候選人<ref>{{Cite web |url = http://www.kmtks.org.tw/blog/post/5195 |title = 第九屆高雄市立法委員選舉提名名單 |author = |date = 2015年8月6日 |publisher = 中國國民黨高雄市委員會 |language = zh-hant |accessdate = 2015年10月14日 |quote = 高雄市第5選區鼓山區、鹽埕區、旗津區、三民西區41里：蔡金晏}}</ref>。

== 服務政見 ==
===第九屆立法委員選舉政見===
*一、海洋及水利政策
#爭取預算投入再生水相關設施建設，積極開發伏流水等多元水資源以解決缺水困境。
#建構韌性城市，推廣高雄都會區滯洪經驗，加速山林區水土保育，以因應未來極端氣候可能災害。
#倡議海岸永續環境，監督海岸法施行細則之訂定與實行，確保國土復育及永續管理。
*二、推動高雄亞洲新灣區再生與開發，由中央協調各單位產權土地，規劃為新興產業園區，提升高雄城市競爭力。
*三、促進高雄新創平台，協助媒合創投企業及資源募集，提升技術並增加能見度，鼓勵在地青年創業，吸引高雄子弟回鄉。
*四、持續推動旗鼓鹽文創觀光特區，發展在地巷弄文旅經濟，重視文化史蹟保存，讓文創觀光成為高雄經貿發展新動能。
*五、加速旗津港埠用地解編與開發，同時協調中央積極改善旗津交通問題。
*六、廣邀市民參與高雄鐵路地下化後地面之開發，整合政府與民間共識，尋求城市發展最大利益。
*七、監督三民中都地區重劃開發，完善社區整體發展與維護計畫，打造高雄新都心。
*八、立法獎勵產業穩定人力資源政策，減少低工資、長工時等非常態工作，勞資雙方共享政策獎勵，並落實勞資對等協商機制，創造更優質尊嚴的勞動環境。
*九、落實教育資源均質化，技職、常態、頂尖教育分立並重，增設實驗性特色中小學，並爭取預算投入特教資源。
*十、監督政府落實長照制度，除妥善照顧失能及需長期照護者，為失能者家屬建立喘息服務，給予家屬心理支持，提升失能家庭生活品質。
*十一、督促中央修改食品安全衛生管理法，從嚴規範並加強查緝，徹底落實源頭管理。
*十二、推廣新住民母語教學，鼓勵新住民第二代學習媽媽母語，培育東南亞第二外語人才。

#爭取開闢哈瑪星鐵道文化園區花海。
#完成鼓山路全線重新鋪設柏油路，重視機車族行的安全。
#推動設立美術館小學，堅持保障當地學童優先入學權益。
#改善柴山邊坡滑動、路面傾斜狀況。

#推動鼓山台泥滯洪池設置。
