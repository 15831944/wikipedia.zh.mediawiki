== 生平 ==
蔡荣名，出生于[[浙江|浙江]][[台州|台州]][[黄岩|黄岩]]，出身书香世家，曾考中[[廪生|廪生]]，好读古诗文，主张诗文[[复古主义|复古]]，诗文见解接近[[前七子|前七子]]、[[后七子|后七子]]，反对当时文坛盛行的[[程朱理学|程朱理学]]、[[八股文|八股文]]和[[台阁体|台阁体]]。<ref name=" fish" />蔡荣名后来在[[南京|南京]]当了[[王世贞|王世贞]]的门客，中晚年家道衰落。<ref>{{cite web |url=http://news.qq.com/a/20100320/000743.htm |title=《金瓶梅》作者蔡荣名学说”立论依据 |accessdate=2010年3月20日 |publisher=腾讯网 |work= |language=zh-cn}}</ref>

2009年，国际金瓶梅研究会执行会长陈明达提出《金瓶梅》是蔡荣名的作品，列举了证据《[[四库全书|四库全书]]·弇洲续稿》、《芙蓉亭诗钞》的文字内容，和《金瓶梅》中的黄岩方言、黄岩食品、黄岩风俗习惯等都符合。<ref>{{cite web |url=http://news.xinhuanet.com/book/2011-01/28/c_121036313.htm |title= 首届国际金瓶梅研讨会在杭州召开 学者为名著翻案|accessdate=2011年1月28日 |publisher=新华网 |work= |language=zh-cn}}</ref>
