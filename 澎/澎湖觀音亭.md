'''澎湖觀音亭'''，位於[[馬公市|馬公市]]中興里（媽宮北甲），古稱'''觀音廟'''，[[光緒|光緒]]年後才泛稱觀音亭<ref>{{Cite book|title=《走過從前：澎湖懷舊照片專輯》|last=洪瑞全、呂信樺|first=|publisher=澎湖縣文化局|year=1995|isbn=9789570052619|location=澎湖縣|pages=31}}</ref>，[[闔澎公廟|闔澎公廟]]之一。

當地人今稱「觀音亭」有兩層意思，廣義泛指涵蓋觀音亭佛寺與整片海景的海水浴場，狹義則單指佛剎觀音亭。

== 建築 ==
狹義的觀音亭為清康熙三十五年（1696年）[[澎湖水師協|澎湖水師協]]右營[[游擊_(官名)|游擊]]'''薛奎'''倡建，主祀觀音大士，寺埕隔水近山、煙波浩香，可遠眺餘暉、漁翁島（[[西嶼鄉|西嶼鄉]]）美景不可勝收。由於建寺歷時悠久，陸續在乾隆、[[嘉慶|嘉慶]]年間修建。<ref name=":0">{{Cite book|title=《澎湖廳志．規制》|last=林|first=豪|publisher=成文|year=1983|isbn=|location=台北市|pages=58}}</ref><ref>{{Cite book|title=《澎湖廳志．職官》|last=林|first=豪|publisher=成文|year=1983|isbn=|location=台北市|pages=200}}</ref>

現今澎湖觀音亭外貌乃奠基於[[台灣日治時期|日治]][[大正|大正]]14年（1925年），大木匠師謝江、[[謝自南|謝自南]]父子的設計；格局為五開間兩進落，俯瞰廟身呈現正方形，穿廊兼具護龍與迴廊的功能，且左右護龍後方各置鐘鼓樓的格局，當時放諸全臺灣係較為罕見。<ref>{{Cite book|title=《2010澎湖縣文化資產手冊》|last=陳|first=英俊|last2=高|first2=啟進|last3=林|first3=文鎮|last4=郭|first4=金龍|publisher=澎湖縣政府文化局|year=2010|isbn=9789860262797|location=澎湖縣|pages=|language=zh-tw}}</ref>{{Rp|47}}（鐘鼓樓不對外開放）

龍王廟（又作龍神廟）乃[[道光|道光]]六年（1826年）[[澎湖通判|澎湖通判]][[蔣鏞|蔣鏞]]開基，日治時期被併祀澎湖觀音亭內，今安奉於東廂房中。<ref>{{Cite web|url=https://www.taiwangods.com/html/cultural/3_0011.aspx?i=184|title=馬公觀音亭|accessdate=2019-01-26|author=|date=|publisher=臺灣宗教百景|language=zh-tw}}</ref>

== 文物 ==
[[File:澎湖觀音亭楹聯_劉燦瑩_題(1886年).jpg|缩略图]]
光緒十一年（1885年）曾遭[[清法戰爭|清法戰爭]][[澎湖之役_(1885年)|澎湖之役]]遭法軍戰火延燒，雖造成佛寺建築部分毀壞，但寺內羅漢、神像卻被方丈以便宜的價格變賣給法國士兵<ref>{{Cite book|title=《孤拔元帥的小水手》|last=鄭順德（翻譯）|first=|publisher=中央研究院臺灣史研究所|year=|isbn=9789860217629|location=台北市|pages=129}}</ref>，中文文獻婉言為「鐘鼓等物盡攜去」。<ref name=":0" />光緒十二年重建，同年落成，澎湖[[通判|通判]]程邦基乃題「大慈悲匾」致贈。

參與澎湖之役的德義營管帶劉璨瑩，1886年亦在觀音亭楹柱，題字「寶珞耀雲端萬朵祥光震旦曇花隨法現，慈航橫海表幾聲梵響乾陀魚鼓逐潮來。東官，劉璨瑩謹獻並記。」附記：「乙酉花朝，法夷犯順，余率師扼溝力禦，宵深忽見廟畔霰光照耀，若內典所傳珠瓔寶珞者，用是於砲彈煙雨之中，血戰兩晝夜，幸免傷痍，書於榜楹以誌神貺。」

== 祭祀 ==
觀世音菩薩是佛教神祇，佛教不碰葷腥，供品切勿以葷食或五辛類食物參拜，奉齋菜、蔬果、花卉即可。

* 農曆二月十九（觀音菩薩聖誕之日）
* 農曆六月十九（觀音菩薩成道之日）
* 農曆九月十九（觀音菩薩出家之日）

澎湖地區農曆七月舉辦[[普渡|普渡]]時，往例由[[媽宮城隍廟|媽宮城隍廟]]開辦，最後在觀音亭結尾，所以當地有「城隍廟放，觀音亭收」之俗諺。<ref>{{Cite web|url=https://penghu.info/OB9E861EA865DB4109E7|title=城隍廟|accessdate=2019-01-26|author=|date=|publisher=Penghu.Info｜澎湖知識服務平台|language=zh-tw}}</ref>

副祀的龍王廟則奉[[龍王|龍王]]，主庇佑海象安穩、航行平安。

此外，根據民間[[五行|五行學說]]，龍王乃水界之神，調節凡間降雨，[[八字命学|八字]]命理中缺水者，適宜向龍王祈求用神，補命中水之不足；白衣觀音大士則五行為金，八字中喜用神為金者，亦同理可向白衣觀音祈求平安<ref>{{Cite web|url=https://jason168chiu.pixnet.net/blog/post/52126644-%E5%85%AB%E5%AD%97%E5%96%9C%E7%94%A8%E4%BA%94%E8%A1%8C%E7%9A%84%E9%96%8B%E9%81%8B%E6%B3%95|title=八字喜用五行的開運法|accessdate=2019-01-27|author=太極易研舘|date=2012-04-17|publisher=|language=zh-tw}}</ref>。

== 周邊景觀 ==
[[File:澎湖-觀音亭.JPG|缩略图]]
觀音亭下面的海水浴場是平沙細浪，是戲水游泳的好地方，更是澎湖帆船選手訓練基地。這裡有一逆[[玄武岩|玄武岩]]形成的海堤，長約500公尺，由[[國立澎湖高級海事水產職業學校|國立澎湖高級海事水產職業學校]]西邊海岸，向西南海中突出，形成一道天然的[[防波堤|防波堤]]。因為堤防的阻隔，使得中央的海水變得格外平靜，成了天然[[海水浴場|海水浴場]]。

2004年，澎湖縣政府斥資五千萬，在觀音寺埕前造堤搭建「西瀛虹橋」，作為每年花火節發放地點，入夜後寺前之虹橋點起七彩燈光，吸引遊人如織<ref>{{Cite web|url=https://nchdb.boch.gov.tw/assets/advanceSearch/monument/19851127000093|title=馬公觀音亭|accessdate=|author=|date=|publisher=文化部文化資產局}}</ref>，卻也造成防波堤內泥沙淤積嚴重，生態改變的負面影響。
