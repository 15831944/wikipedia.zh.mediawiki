'''澎湖菜宅'''又稱「宅內」，是一種遮蔽與擋風的石牆，用以保護田地內的[[農作物|農作物]]。在[[農業|農業]]上，靠近住家以[[硓𥑮石|硓{{僻字]]堆砌而成稱為「圍仔」；遠離[[聚落|聚落]]，不一定使用硓{{僻字|𥑮|左「石」右「古」}}石做防風牆的稱為「園仔」。

== 地理背景 ==
[[澎湖|澎湖]]全縣沒有屏障的高[[山|山]]，位於臺灣本島與中國大陸之中，形成風力資源豐沛的群島。尤秋冬東北季風盛行，平均風速每秒5至6[[公尺|公尺]]，最大陣風可達每秒13至14公尺，農作物生長不易，為了阻檔[[東北季風|東北季風]]與其挾帶的鹹[[雨|雨]]，「菜宅」乃因應而生。

== 作用 ==
菜宅為保護田地內的農作物，使其不受強勁[[東北季風|東北季風]]摧毀。

== 建材 ==
菜宅通常就地取材砌築，有岩石的地方就開鑿[[火山|火山]]熔岩凝結的[[玄武岩|玄武岩]]；有[[珊瑚|珊瑚]]淺硼的地方使用硓{{僻字|𥑮|左「石」右「古」}}石；水源豐富的便種植[[蘆竹|蘆竹]]。

== 建造方式 ==
取玄武岩、硓{{僻字|𥑮|左「石」右「古」}}石，硓{{僻字|𥑮|左「石」右「古」}}石需經暴曬，去雜質[[鹽|鹽]]份數日。建築時，先打[[地基|地基]]挖約深三十公分土溝，以大石頭做基礎慢慢往上愈小塊[[石頭|石頭]]堆疊上去，大石塊間以小石填以空隙，利用石頭間的稜角相互契合加以穩固。

== 作物 ==
菜宅內的作物遇[[春|春]]、[[夏|夏]]兩季以[[南瓜|南瓜]]、澎湖[[菜瓜|菜瓜]]、澎湖[[菜豆|菜豆]]、[[越瓜|越瓜]]為主要。[[秋|秋]]、[[冬|冬]]季則多種植[[高麗菜|高麗菜]]、[[蕃茄|蕃茄]]、[[大頭菜|大頭菜]]、[[茼蒿|茼蒿]]等蔬果。
菜宅除了種植時令的蔬果，最主要就是栽植冬作[[蕃薯|蕃薯]]，稱為「栽母」。

== 風貌 ==
* 菜宅堆砌四面的稱為「宅內」，[[北|北]]牆最高，約1.6公尺至2.5公尺；[[東|東]]、[[西|西]]牆次之，高約1.5公尺至1.8公尺；[[南|南]]牆最矮，大多不超過1公尺。
* 由玄武岩所堆砌的宅內，因不易砌得牢靠，隙縫多，石塊契合度低，通常高度為1.5公尺以下。
