'''澎湖機場'''（{{Airport codes|MZG|RCQC}}），位於[[臺灣|臺灣]][[澎湖縣|澎湖縣]][[湖西鄉|湖西鄉]]的機場，為[[澎湖縣|澎湖縣]]主要聯外機場。
澎湖機場為軍民合用機場，民用部分由[[交通部民用航空局|交通部民用航空局]]'''馬公航空站'''（機關名稱—民航局'''馬公航空站'''，未來則配合交通部組織改造時，再改制為民航局'''澎湖航空站'''。）管理及營運；軍用部分為'''空軍馬公基地'''。

== 歷史 ==
*1957年：由民航空運公司以[[C-46運輸機|C-46型運輸機]]開始營運。
*1966年：始興建航站大廈，成立馬公候機室，隸屬於高雄航空站。
*1977年8月1日：成立馬公航空站（丙種航空站）。
*1984年1月30日：春節期間為疏運旅客國內線加班，民航機不敷調配，協調華航向空軍租用[[B727|B727]]機兩架本日起擔任北高、高馬航線客貨運輸任務。<ref name=seventyone>[https://www.mkport.gov.tw/info/MemorabiliaDetail.aspx?menuid=94020609D487A03C&first=75D9DC3536073A53 機場大事紀-民國71至民國75年]</ref>
*1984年7月14日：本日出入旅客創歷年最高紀錄達4961人，班機飛航達64架次。<ref name=seventyone />
*1988年12月1日：空軍馬公基地實施跑道面整修，施工期限60個工作天，班機改用滑行道起降。<ref>[https://www.mkport.gov.tw/info/MemorabiliaDetail.aspx?menuid=94020609D487A03C&first=1E51FF758084574E 機場大事紀-民國76至民國80年]</ref>
*1991年5月11日：接管[[七美機場|七美]]、[[望安機場|望安]]兩離島機場，成立七美、望安輔助站。
*1991年7月15日：[[大華航空|大華航空]]開航台中至馬公班機。
*1993年1月18日：行政院核定本站升格為乙種航空站。<ref>[https://www.mkport.gov.tw/info/MemorabiliaDetail.aspx?menuid=94020609D487A03C&first=9C0FB94DBB005BC5 機場大事紀-民國81至民國85年]</ref>
*2010年4月2日：免稅商店正式營業，並於同月8日舉辦開幕活動。<ref>[https://www.mkport.gov.tw/info/MemorabiliaDetail.aspx?menuid=94020609D487A03C&first=5D9A06BB852BB996 機場大事紀-民國99年]</ref>
*2015年12月25日：開始拆除舊航廈。<ref>[https://www.mkport.gov.tw/info/MemorabiliaDetail.aspx?menuid=94020609D487A03C&first=09F85499C4DE4877 機場大事紀-民國104年]</ref>
*2018年3月7日：[[澎湖縣政府|澎湖縣政府]]宣佈馬公航空站將更名為[[澎湖機場|澎湖機場]]，並於其後半年內完成相關的作業。<ref>{{Cite news|title=馬公機場正名澎湖機場！縣府晚間突宣布 菊島各界譁然|newspaper=[[自由時報|自由時報]]|date=2018-03-07|url=http://news.ltn.com.tw/news/life/breakingnews/2358863|accessdate=2018-03-08|archiveurl=https://web.archive.org/web/20180307163725/http://news.ltn.com.tw/news/life/breakingnews/2358863|archivedate=2018-03-08}}</ref>
*2018年4月24日~5月8日：因應2018[[澎湖|澎湖]]花火節，[[華信航空|華信航空]]營運澎湖至[[香港國際機場|香港]]的包機，每週二、五出發。
*2018年8月9日：正式更名為[[澎湖機場|澎湖機場]]<ref>[https://fnc.ebc.net.tw/FncNews/Content/48171?from=webpush 馬公機場沿用一甲子走入歷史「新名字」上路拚國際行銷]</ref>。

== 航廈設計現況 ==
*樓層數 地下一層地上三層建築物
*總面積 44,048 <big>㎡</big>
*建物高度 18.45公尺
*年旅客容量 440萬人次
*尖峰小時旅客容量 1760人次/小時
*跑道長度 長3000公尺，寬45公尺
*停機坪 設置可提供A321、B737與B757以下大、中、小型飛機使用之停機坪九機位，直昇機停機坪一機位
*空橋 三座，對應之登機門為5、6及9號，對應之停機坪為3、4及5號
*停車場 小客車停車場可停放小客車480輛，機車停車場可停放392輛
*助航設施 目視滑降燈、進場燈、末端燈、閃光燈、跑道燈、邊燈、氣象自動測報系統。
*新航廈啟用日期 2002年9月

== 意外事件 ==
*1995年3月27日：台灣航空公司671班次由澎湖七美飛往澎湖馬公，於馬公機場降落後，滑入機坪時因主輪剎車故障導致外胎燃燒，經地勤人員撲滅並無人員傷亡，隨即修復離場。<ref>[https://www.mkport.gov.tw/info/MemorabiliaDetail.aspx?menuid=94020609D487A03C&first=9C0FB94DBB005BC5 機場大事紀-民國81至民國85年]</ref>
*1998年8月5日：國華航空由澎湖七美飛往澎湖馬公7016號班機 (註冊編號B-12253) 於落地時6500呎處衝出跑道，馬公機場於15:36時至16:31時期間關閉。<ref>[https://www.mkport.gov.tw/info/MemorabiliaDetail.aspx?menuid=94020609D487A03C&first=1F52637C647E5D07 機場大事紀-民國86至民國90年]</ref>
*2012年8月17日：華信航空[[ERJ-190|ERJ-190]]型客機 (註冊編號B-16825) ，由[[台北松山機場|台北松山機場]]起飛前往[[馬公機場|馬公機場]]，在馬公機場跑道著陸時，飛機偏離跑道到左側草地，導致鼻輪起落架折斷、機身蒙皮受損，所幸全機110名乘客及機組員都平安。
* 2014年7月23日：[[復興航空222號班機空難|復興航空222號班機]]、機身編號B-22810的[[ATR_72|ATR 72-500型]]客機，由[[高雄國際機場|高雄國際機場]]飛往澎湖馬公機場航線時，疑因[[颱風麥德姆_(2014年)|颱風麥德姆]]風雨過大造成飛機降落不順利，重飛失敗，於澎湖縣湖西鄉西溪村墜落，起火燃燒，造成機上人員48人死亡，10人重傷。另外波及11棟民宅，5人輕傷。
