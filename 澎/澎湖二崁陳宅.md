'''二崁陳宅'''，又名'''二崁陳家大厝'''，位於[[臺灣|臺灣]][[澎湖縣|澎湖縣]][[西嶼鄉|西嶼鄉]][[二崁村|二崁村]]，由[[陳嶺|陳嶺]]（1872年－1948年）、[[陳邦|陳邦]]（1875年－1959年）昆仲所建。建築形制為三間三進宅院，屬[[閩洋折衷|閩洋折衷]]式樣建築。於1988年公告為縣定古蹟<ref>{{cite web|title=澎湖二崁陳宅|url=https://nchdb.boch.gov.tw/assets/advanceSearch/monument/19880425000002|work=文化部文化資產局|accessdate=}}</ref>。

== 歷史 ==
澎湖人氏陳嶺、陳邦兄弟於1887年渡海赴臺，初期從事[[酒廠|酒廠]]工作，之後轉業，至金協源漢藥鋪學習藥理。學成後，即於[[臺南|臺南]]經營元益藥鋪取得成功。陳氏兄弟乃返鄉興築陳家大厝，由池東匠師[[陳媽挺|陳媽挺]]主持施工。建築大厝時，先建二、三進，最後再興建第一進，1912年竣工。

== 空間配置 ==
二崁陳宅位於村落中央，坐東南朝西北，融合閩南式建築與澎湖地方建築特色，並反映[[台灣日治時期|日治時期]]西洋歷史式樣建築之影響，成為閩洋折衷式樣建築。依建築中軸線之安排，依次為門廳、正廳、後堂等三個主要空間，每一進配置院落，寬敞明亮。

== 造型 ==
建築主要以澎湖[[咾咕石|咾咕石]]與[[玄武岩|玄武岩]]組成，外觀厚實。大門除楹聯與門額外，尚有造型浮雕、吉祥花磚等裝飾。第一進以洋式建築風格為主，二、三進則使用傳統閩南建築手法。<ref>{{cite book|author=傅朝卿|title=台灣建築的式樣脈絡|year=2013|publisher=五南|location=臺北市|isbn=9789571170862|pages=123}}</ref>正立面[[巴洛克式|巴洛克式]]的半圓形門樓上，以[[老鷹|老鷹]]意寓「雄鷹飛天」；燈甕[[臺語|臺語]]為「丁」，[[時鐘|時鐘]]拆字可得「金童」二字，都有保祐家丁興旺的意思。

== 現況 ==
目前為私人擁有，並開放收費參觀。
