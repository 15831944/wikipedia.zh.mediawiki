'''盛港公共图书馆'''（{{lang-en|Sengkang Public Library}}）是一所位于[[盛港|盛港]]规划区{{link-en|新加坡圖書館管理局|National Library Board}}旗下的[[公共圖書館|公共圖書館]]，并于2002年正式开幕。此图书馆位于[[勘宝坊|勘宝坊]]的三楼与四楼。

==历史==
此图书馆的原名是'''盛港民众图书馆'''（{{lang-en|Sengkang Community Library}}），于2002年11月30日由担任副总理兼内政部长的[[张志贤|张志贤]]宣布正式落成并向公众开放。<ref name=":Sengkang Public Library">{{cite web|url=https://www.nlb.gov.sg/Portals/0/library/documents/Sengkang%20Public%20Library%20-%20Fact%20Sheet.pdf|title=Sengkang Public Library - Fact Sheet|publisher=National Library Board|format=PDF|access-date=2018-03-13|language=en|deadurl=no|df=|archive-url=https://web.archive.org/web/20150319133646/http://www.nlb.gov.sg/Portals/0/library/documents/Sengkang%20Public%20Library%20-%20Fact%20Sheet.pdf|archive-date=2015-03-19}}</ref> 2008年改名为盛港公共图书馆。

2015年10月此图书馆进行为期一年半的翻新，并于2017年3月18日重新开放。[[人民行动党|人民行动党]]副主席[[雅国|雅国]]主持了重新开馆仪式。<ref name=":Channel Newsasia">{{Cite news|url=http://www.channelnewsasia.com/news/singapore/revamped-sengkang-public-library-opens-doors-to-public-8574114|title=Revamped Sengkang Public Library opens doors to public|work=Channel NewsAsia|access-date=2019-03-13|language=en-US|archive-url=https://web.archive.org/web/20171011182348/http://www.channelnewsasia.com/news/singapore/revamped-sengkang-public-library-opens-doors-to-public-8574114|archive-date=2017-10-11|dead-url=no}}</ref><ref name=":National Library Board">{{cite web | url = https://www.nlb.gov.sg/VisitUs/BranchDetails/tabid/140/bid/341/Default.aspx?branch=Sengkang+Public+Library | title = National Library Board - Sengkang Public Library | format =  | work = National Library Board | access-date = 2019-03-13 | archive-url = https://web.archive.org/web/20190107074757/https://www.nlb.gov.sg/VisitUs/BranchDetails/tabid/140/bid/341/Default.aspx?branch=Sengkang+Public+Library | archive-date = 2019-01-07 | dead-url = no }}</ref> 此图书馆目前占据勘宝坊的三楼与四楼，比之前大百分之十八。<ref name=":Lianhe Zaobao">{{cite web | url = https://www.zaobao.com.sg/znews/singapore/story20170319-737614 | title = 盛港图书馆设专属区 添购适龄儿童少年读物 | format =  | work = 联合早报 | access-date = 2019-03-14 | dead-url = no | archive-url = https://web.archive.org/web/20170322043147/http://www.zaobao.com.sg/znews/singapore/story20170319-737614 | archive-date = 2017-03-22 }}</ref>
