'''盛氏花厅'''又名'''停舻'''，位于[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[海曙区|海曙区]]郁家巷1号、5号。盛氏花厅初为清[[林廷鳌|林廷鳌]]所建的藏书楼，后经转手辟为[[盛炳纬|盛炳纬]]退休回乡后的书房。1981年12月，盛氏花厅被公布为海曙区文物保护单位。
== 历史 ==
盛氏花厅建于清[[同治|同治]]、[[光绪|光绪]]年间，是[[林廷鳌|林廷鳌]]居宅中作藏书之用的近性楼，因远看建筑如同巨大的船舶，故被称作停舻。<ref>{{cite web |url= http://www.nbwb.net/pd_wwbh/info.aspx?Id=915&type=2|title=盛氏花厅 |date=2005-11-25|publisher=宁波市文物保护管理所|quote=盛氏花厅的原主人为清同治、光绪年间的诸生林廷鳌，花厅系居宅中的书阁――近性楼，因该阁飞檐翼角，远看如一巨舰，故又名“停舻”。}}</ref>后因林家家境逐渐衰落变卖资产，而退休回乡的时任浙江学政[[盛炳纬|盛炳纬]]将其买下，变为书房，因此改称盛氏花厅。<ref>{{cite web |url=http://news.163.com/17/0322/00/CG3GDJ1T000187VI.html |title=闹市中觅静，花厅接书廊|date=2017-03-22|publisher=网易新闻|quote=到了清朝末年，原本家境颇为殷实的林家，开始家道中落，家里的积蓄无法维持庞大的开销，林宅附近的近性楼就成了需要变卖的固定资产。幸运的是，当时辞官回家的盛炳纬看中了这里，并出资将其买下，这才有了后来的盛氏花厅。}}</ref>
其中风格小巧精致的庭园建筑在宁波范围内独一无二。<ref>{{cite web |url=http://www.nbwb.net/info.aspx?Id=10438 |title=郁家巷意醉神迷的蜕变 |author=楼小娴|date=2009-03-08 |publisher=宁波市文物保护管理所|quote=这类小巧精致的庭园建筑，在宁波市区仅此一家。}}</ref>1981年12月，盛氏花厅被公布为海曙区文物保护单位。2008-2009年，盛宅进行了最大规模的维修，盛氏花厅前檐柱因开裂过多而维修。<ref>{{Cite journal|title=宁波郁家巷盛宅保护维修研究|author=郑雨|url=http://www.ixueshu.com/document/92314614180fd6ab.html|journal=《农业科技与信息:现代园林》|issue=2010年01期|doi=|others=|year=|volume=|page=|pmid=}}</ref>

== 结构 ==
盛氏花厅坐西朝东，占地约600平方米，为重檐歇山顶、明次间抬梁式建筑，由堂、过厅、两厢，以及居室的附属建筑物组成。建筑内栋梁、楼道、窗棂为木结构、阶沿、柱础、卯榫为石结构，都具有较为精美的雕刻，其中后进拥有一块少见的1平方米采用透雕手法的隔墙砖雕。<ref>{{cite web |url=http://www.nbwb.net/info.aspx?Id=10438 |title=郁家巷意醉神迷的蜕变 |author=楼小娴|date=2009-03-08 |publisher=宁波市文物保护管理所|quote=从郁家巷１号墙门可以进入当年的盛氏花厅。百年古宅的明堂、过厅、两厢，以及居室的一应附属建筑物，风貌完整依旧。随意浏览，可见木结构的栋梁、楼道、窗棂和石结构的阶沿、柱础、卯榫，都雕刻得极为精致；后进的一块隔墙砖雕，约１平方米，采用透雕手法，图案之美，工艺之精，实属罕见。}}</ref>

== 现状 ==
盛氏花厅现为[[月湖盛园|月湖盛园]](郁家巷历史街区)的组成部分，设立保艺美术馆。<ref>{{cite web |url=http://gtoc.ningbo.gov.cn/art/2016/7/14/art_207_418882.html |title=月湖盛园|date= 2016-07-14|publisher=宁波市人民政府办公厅}}</ref><ref>{{cite web|url=http://www.nbtv.cn/life/news/2014/10/24/201410241228112.html|title=曾藏于雷峰塔砖中的2米经卷现身甬盛氏花厅|date=2014-10-24|publisher=宁波纽米地传媒科技有限公司|quote=昨天（10月23日），当记者看到它时，这里已经进驻了一家民间投资、由民政局审核通过的民间博物馆——保艺美术馆。|access-date=2017-07-05|archive-url=https://web.archive.org/web/20150621035955/http://www.nbtv.cn/life/news/2014/10/24/201410241228112.html|archive-date=2015-06-21|dead-url=yes}}</ref>
