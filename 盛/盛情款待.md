'''盛情款待'''（{{lang-en|Omotenashi}}）{{r|mirrormedia20180412insight003}}{{r|funscreen687}}是[[2018年|2018年]]的台日合作電影，由曾獲[[金馬獎|金馬獎]]最佳短片《小偷》的新銳導演[[陳鈺杰|陳鈺杰]]執導的[[電影|電影]]，並擔任[[製片|製片]]、[[編劇|編劇]]、[[攝影|攝影]]及[[剪接|剪接]]等職務。本片是[[日本|日本]][[松竹映畫|松竹攝影所]]首次的跨國製作。

本片為第42屆[[香港國際電影節|香港國際電影節]]開幕電影<ref>[http://www.hkiff.org.hk/film/getdetail?fid=228 香港國際電影節官方網站資訊]</ref>，於2018年3月19日在[[香港會議展覽中心|香港會議展覽中心]]舉行香港首映。亦分別於3月3日及4月13日在日本和台灣正式公映。

== 劇情提要 ==
明月館，一間位於京都近郊的日式旅館，具有美麗的湖景和傳統的日式風情，但由於建築殘舊，且位處偏僻，客人稀少。...

== 電影配樂 ==
大橋好規（Yoshinori Ohashi）負責電影音樂，是一位擅以鋼琴、吉他、貝斯、爵士鼓、歌唱表演才華的音樂家<ref>[https://star.ettoday.net/news/960814 ETtoday新光雲]上新聞</ref>。

== 跨國製作 ==
本片集結了台、日與美等各地的影視人員，如片大部分的拍攝場景在[[日本|日本]][[近畿地方|關西區域]]，包含了[[大阪|大阪]]、[[京都|京都]]與[[滋賀|滋賀]]等地。而主場景明月館雖位在[[滋賀|滋賀]]山區，但導演為了可以完整體現琵琶湖的明媚風光與故事完整性，堅持在湖畔旁搭景複製了明月館，包含一樓內景，配以藍幕，使用大量特效製作。<ref>[https://www.beautimode.com/article/content/84868/ BeautiMode創意生活風格網]資料</ref>。並由六間特效公司（傳翼、光和影、崛影、像數設計、嵐天企業、炎殺黑龍波）及美國3D動畫與合成師前後共花費了將近九個月完成。
