'''盛光祖'''（{{bd|1949年|4月||catIdx=}}），[[中國|中國]][[江苏省|江苏]][[南京市|南京]]人。1968年8月参加工作。退休[[中华人民共和国|中华人民共和国]][[省部级正职|正部长级]]官员，退休前长期在[[中華人民共和國鐵路運輸|中国铁路系统]]工作，曾任[[中华人民共和国海关总署|海关总署]]署长，末任[[中华人民共和国铁道部|铁道部]]部长，首任[[中国铁路总公司|中国铁路总公司]]总经理、党组书记<ref>{{cite news |title = 铁道部末任部长、中铁总首任总经理，67岁盛光祖退休了 |url = http://www.thepaper.cn/newsDetail_forward_1540453 |accessdate = 2016-10-09 |publisher = 澎湃新闻}}</ref>。前任[[全国人大财政经济委员会|全国人大财政经济委员会]]副主任委员。

== 经历 ==
=== 早年 ===
据公开资料显示，盛光祖是[[江苏|江苏]][[南京|南京]]人，其妻曾是商场的一名普通职工，岳父也是普通退休干部；至于盛光祖的家世就没有很多资料<ref>[http://business.sohu.com/s2013/jrzj69/ “末代铁道部长”盛光祖]</ref>。1968年8月，19岁的盛光祖参加工作，在江苏省南京市[[江宁县|江宁县]][[东山公社|东山公社]]前进大队[[插队|插队]]。1970年12月，他离开前进大队，进入[[同济大学|同济大学]]工程系就读。不久他转入[[上海铁道学院|上海铁道学院]]工程系铁道工程专业，1974年5月毕业，获大专学历<ref>{{cite book|editor=朱静縺|title=拼搏者的足迹 第2集|date=1990-08|publisher=知识出版社|location=北京|isbn=7-5015-0455-5|pages=510}}</ref>。毕业后，他长期在[[中华人民共和国铁路运输|铁路系统]]工作。1974年5月至1977年7月，他在[[上海铁路局南京铁路分局|上海铁路局南京铁路分局]]南京工务段技术室担任干部。1977年7月，他调任上海铁路局南京铁路分局革委会、政治部办公室秘书。1982年6月起，他历任上海铁路局南京铁路分局镇江工务段副段长、代段长、党委副书记，段长、党委副书记。1983年11月至1985年4月间，他先后出任上海铁路局工务处副处长、人事处副处长<ref name=nj>{{cite journal|title=领导人物|journal=中国铁道年鉴|volume=2012|pages=431|publisher=《中国铁道年鉴》编辑部|issn=1009-6957}}</ref>。

=== 铁路系统领导 ===
1985年4月，任上海铁路局南京铁路分局党委书记。1986年9月，任上海铁路局南京铁路分局党委书记兼政治部主任。1988年4月，任[[上海铁路局杭州铁路分局|上海铁路局杭州铁路分局]]分局长、党委书记。1989年5月，任上海铁路局南京铁路分局分局长、党委书记。1993年4月，任[[济南铁路局|济南铁路局]]局长、党委副书记<ref name=nj/>。

1994年11月，任[[中华人民共和国铁道部|铁道部]]党组成员、总经济师。1996年1月，任铁道部政治部主任、党组成员兼直属机关党委书记。1999年1月，任铁道部副部长、党组成员<ref name=nj/>。

=== 海关总署领导 ===
2000年4月，任[[海关总署|海关总署]]副署长、党组成员。2002年2月，任海关总署副署长、党组副书记，海关副总监。2007年7月，任海关总署党组书记、副署长（正部长级）、海关副总监；并在[[中共十七大|中共十七大]]上，当选中央委员会委员。2008年3月，任海关总署署长、党组书记、海关总监<ref name=nj/>。

=== 铁道部部长 ===
2011年2月，前铁道部部长[[刘志军|刘志军]]因贪腐问题下台，盛光祖接任铁道部党组书记、部长<ref>{{cite web |language = zh-hans |publisher = 凤凰网 |title = 铁道部部长刘志军被免职 |url = http://finance.ifeng.com/news/people/20110213/3392629.shtml |author = 新华 |date = 2011-02-13 |accessdate = 2011-02-13}}</ref><ref>{{cite web |url = http://www.gov.cn/rsrm/2011-02/25/content_1810797.htm |title = 全国人大常委会表决决定任命盛光祖为铁道部部长 |publisher = 中国中央人民政府网站 |accessdate = 2011-02-25}}</ref>。上任数月后，发生了[[2011年杭深线动车组列车追尾事故|2011年杭深线动车组列车追尾事故]]。盛光祖被认为对事故处置不力，2011年12月28日，向[[国务院|国务院]]作出深刻检查<ref>{{cite news |title = 温家宝主持召开国务院常务会议 |url = http://www.gov.cn/ldhd/2011-12/28/content_2032218.htm |accessdate = 2011-12-28 |newspaper = 国务院办公厅}}</ref>。

=== 铁路总公司总经理 ===
2013年3月，按照[[第十二届全国人民代表大会第一次会议|第十二届全国人民代表大会第一次会议]]通过的《[[国务院机构改革方案#2013|国务院机构改革和转变职能方案]]》，铁道部被分拆，成立新的[[国家铁路局|国家铁路局]]和[[中国铁路总公司|中国铁路总公司]]，盛光祖成为最后一任[[铁道部|铁道部]]部长<ref name="国务院机构改革方案公布：减少4个正部级机构">{{cite web |url = http://news.ifeng.com/mainland/special/2013lianghui/content-3/detail_2013_03/10/22931367_0.shtml |title = 国务院机构改革方案公布：减少4个正部级机构 |accessdate = 2013-03-10}}</ref>；3月14日，中国铁路总公司注册成立，盛光祖被任命为[[总经理|总经理]]和公司[[法定代表人|法定代表人]]<ref>{{cite web |url = http://news.xinhuanet.com/fortune/2013-03/15/c_124461395.htm |title = 中国铁路总公司成立 盛光祖任总经理 |accessdate = 31 05, 2013}}</ref>。2016年10月9日，到龄退休<ref>{{cite web |language = zh-hans |publisher = 网易财经 |title = 
陆东福任中铁总总经理 盛光祖退休 |url = http://money.163.com/16/1009/15/C2UOSGAI00258105.html |author = 网易 |date = 2016-09-10 |accessdate = 2016-09-10}}</ref><ref>{{cite news |title = 陆东福接替盛光祖任中铁总总经理 |url = http://www.ce.cn/cysc/newmain/pplm/qyrw/201610/09/t20161009_16574168.shtml |accessdate = 2016-10-09 |publisher = 中国经济网}}</ref>。

===全国人大财政经济委员会副主任===
2016年11月7日，[[第十二届全国人大常委会|第十二届全国人大常委会]]第二十四次会议经表决，任命盛光祖为[[全国人大财政经济委员会|全国人大财政经济委员会]]副主任委员<ref>{{cite news |title = 盛光祖任全国人大财经委副主任委员 |url = http://news.qq.com/a/20161107/028763.htm|accessdate = 2016-11-07 |publisher = 新华社}}</ref>。
== 评价 ==
===自我评价===
在[[中国铁路总公司|中国铁路总公司]]2016年10月9日召开的干部大会中，代表[[中国共产党中央委员会|中共中央]]宣布中国铁路总公司主要领导调整的决定的[[中共中央组织部|中共中央组织部]]副部长[[邓声明|邓声明]]对盛光祖的任内工作进行了高度评价，对2012年以来其在铁路工作所取得的成绩给予充分肯定。盛光祖在这次会议中总结自己工作时表示，其任内“完成[[中国共产党中央委员会|党中央]]、[[中华人民共和国国务院|国务院]]交给我们的光荣任务，顺利实施了铁路管理体制重大改革，保持了铁路安全基本稳定，建设持续保持在高位运行，客货运输组织改革不断深化，技术创新取得新的突破，“走出去”工作迈出重要步伐，铁路职工收入有了一定增长，铁路事业取得了新发展新进步。”<ref>{{cite news|title=中国铁路总公司召开干部大会|url=http://www.peoplerail.com/rail/show-456-294892-1.html|date=2016-10-21|author=曹丹|publisher=人民铁道网}}</ref>
