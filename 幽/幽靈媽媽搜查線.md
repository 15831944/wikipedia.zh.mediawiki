《'''幽靈媽媽搜查線'''》（{{lang-ja|'''ゴーストママ捜査線'''}}，{{lang-en|'''My Mommy is a Ghost!'''}}），是[[日本|日本]][[漫畫家|漫畫家]][[佐藤智一|佐藤智一]]的一部[[日本漫畫|漫畫]]，2000年於《[[Big_Comic_Original|Big Comic Original]]》開始連載，台灣中文版由[[東立出版社|東立出版社]]代理，單行本全3卷已發售。

== 劇情內容 ==

== 主要角色 ==
=== 原平家 ===
; 原平蜻蜓
: 7歲。愛哭鬼男孩子。壽第一小學1年級。蝶子死後有了近視，因為那個契機透過蝶子的眼鏡可以看見幽靈。
; 原平蝶子
: 蜻蜓的母親。
; 原平蟬彥
: 蜻蜓與蜜子的父親。蝶子的丈夫。
; 原平蜜子
: 蜻蜓的姊姊。壽第三中學二年級。
; 小八
: 原平家飼養的小獵犬。

=== 景灣警察署 ===
; 桑形
: 景灣警察署景灣派出所執勤的警察。

=== 景灣公立壽第一小學 ===
; 三葉
: 蜻蜓班上的級任老師。
; 向手
: 蜻蜓學校的體育老師。
; 校長
: 從小時候就住在景灣市，對鎮上非常熟悉。

=== 其他角色 ===
; 小金
: 作靈感占卜的老奶奶。
; 鈴木
: 蝶子的幽靈夥伴的情報員。
; 謙本桐三郎
: 蜻蜓所住的景灣市的市長。

=== 播映時間 ===
[[香港|香港]][[無綫劇集台|無綫劇集台]]於2013年3月起播出，譯名為「'''我的幽靈媽媽'''」。[[台灣|台灣]][[緯來日本台|緯來日本台]]於2013年8月起播出，譯名為「'''第六感搜查線'''」。

=== 劇情內容 ===
上原蝶子（[[仲間由紀惠|仲間由紀惠]] 飾）樂於助人，在生活安全課擔當警察官，和丈夫航平（[[澤村一樹|澤村一樹]] 飾）、其夫前妻所生的女兒葵（[[志田未來|志田未來]] 飾）、還有她的兒子蜻蜓（[[君野夢真|君野夢真]] 飾）過著幸福生活。可惜，某天她當值時為了拯救一個在[[火災|火災]]現場的小女孩，被濃煙嗆死，但因為掛念仍在讀小學一年級的兒子，她不願成佛而變成幽靈，留駐於人世間，其子可以透過她所遺留的眼鏡見到母親──蝶子的幽靈。就這樣，蝶子和蜻蜓開始了他們不可思議的生活…

=== 登場人物 ===
==== 上原家 ====
* 上原蝶子（終年32）  - [[仲間由紀惠|仲間由紀惠]]（童年時期：[[酒井彩音|酒井彩音]]）（粵語配音：[[梁少霞|梁少霞]]）
: 本姓志村。生活安全科的警察，開朗且樂於助人，無法對有困難的人置之不理的性格，想到的話就不計後果的去行動。和深愛著的丈夫航平、女兒葵、兒子蜻蜓過著幸福的生活。某天卻突然因為縱火案殉職，但是因為擔心軟弱的兒子蜻蜓，而變成幽靈留在人間，出現在蜻蜓面前。
* 上原航平（40） - [[澤村一樹|澤村一樹]]（粵語配音：[[陳廷軒|陳廷軒]]）
: 蝶子的丈夫，在下町商店街經營著上原照相館。個性正經深情，先前曾有一段婚姻，蝶子是他續絃的妻子。
* 上原葵（17） - [[志田未來|志田未來]]（童年時期：[[小高實優|小高實優]]）（粵語配音：[[何璐怡|何璐怡]]）

* 上原蜻蜓（6） - [[君野夢真|君野夢真]]（粵語配音：[[張頌欣|張頌欣]]）
: 蝶子的兒子，小學1年級學生，和母親正好相反，說的好聽點是溫和、說的難聽點就是軟弱的性格，連石橋也不敢走的類型。母親過世後，每天都悶悶不樂。某天戴上蝶子留下的眼鏡後，居然看到了蝶子的幽靈。從此之後，和幽靈媽媽蝶子一起經歷各式各樣的事件中，自己也因為這些歷練而有所成長。

* 三船義光（終年50） - [[生瀨勝久|生瀨勝久]]（粵語配音：[[李錦綸_(配音員)|李錦綸]]）
