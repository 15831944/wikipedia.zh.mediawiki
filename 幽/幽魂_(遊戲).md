《'''幽魂'''》（{{lang-en|''Phantasmagoria''}}）是美國遊戲公司[[雪樂山|雪樂山]]在1995發行的[[PC|PC]]電子[[冒險遊戲|冒險遊戲]]，在1997年有推出適用於遊樂器[[Sega_Saturn|Sega Saturn]]的版本并稱改《'''Phantasm'''》。

作品的特色是由透過真人演員所演出的[[互動式電影|互動式電影]]風格遊戲內容，並包含著有恐怖、血腥[[暴力|暴力]]、[[性愛|性愛]]場景的橋段，因此被歸類在非18歲以上成人不適合遊玩的限制級遊戲。

因《幽魂》充滿大量影音聲光效果的資料，而以多達7片遊戲光碟包裝發行，在該年代的遊戲產業界中為罕見的份量。之後在Sega Saturn上發行的版本內容更高達8片光碟。<ref>{{Cite web |url=http://www.segagagadomain.com/saturn16/phantasm.htm |title=Saturn - Phantasm |trans_title=Sega Saturn上的《幽魂》 |work=segagagadomain |accessdate=2012-10-05 }}</ref>

其遊戲系列的續作《[[幽魂2|幽魂2]]》，則在1996年推出。

== 遊戲劇情 ==

女作家愛德琳（Adrienne）、與他從事攝影師的丈夫唐諾德（Donald）購買了一間叫做卡諾維治（Carnovasch）的莊園作為新家。原先愛德琳夫婦相當滿意這間具有古色風格的大宅，卻殊不知這間大屋過去存在著一段黑暗的歷史。

某一日，愛德琳意外開啟屋內一個神秘的盒子、將盒內所封印的邪惡力量給釋放出來，而該邪惡力量則在神不知鬼不覺的情況下附著在唐諾德身上。

之後愛德琳在面對逐漸喪失心智、充滿暴行的丈夫威脅下，為了尋回先前正常的生活，而開始設法找出卡諾維治莊園先前鮮為人知的可怕秘密。

== 遊戲系統 ==

遊戲為一般電子冒險遊戲常採用的點選模式，即在畫面中使用滑鼠游標點選玩家認為可疑的物品、或是需要移向探查的場景，之後依遊戲角色的反應來猜想、解開前往下一遊戲章節的方式。

而為了增加遊戲的刺激程度，在最後一個章節會限制遊戲存檔的功能。<ref>{{cite journal zh |title=幽魂攻略 |journal=[[電腦玩家|電腦玩家]] |volume=52 |pages=179頁 |publisher=電腦玩家 |location=台灣 |accessdate=2012-10-05 |date=1995年11月}}</ref>

== 遊戲製作 ==

《幽魂》由製作過雪樂山遊戲《[[謎之屋|謎之屋]]》及《[[國王密使|國王密使]]》系列的[[蘿貝塔·威廉斯|蘿貝塔·威廉斯]]女士負責主要設計，<ref>{{cite journal zh |title=幽魂 Phantasmagoria |journal=[[軟體世界|軟體世界]] |volume=73 |pages=6頁 |publisher=[[智冠科技|智冠科技]] |location=台灣 |accessdate=2012-10-05 |date=1995年4月}}</ref>其製作團隊人數多達200人以上，並聘請了約135名專員為遊戲配樂合唱。<ref>{{Cite web |url=http://www.csoon.com/issue8/phantasm.htm |title=Phantasmagoria by Sierra On-Line |trans_title=由雪樂山公司製作 |work=Coming Soon Magazine! |accessdate=2012-10-05}}</ref>

遊戲畫面為透過真人演員在[[色鍵|色鍵]]佈景前演出後，合成在由遊戲美工人員所繪製的背景畫面裡。<ref name="interview">{{Cite web |url=http://www.adventureclassicgaming.com/index.php/site/interviews/127 |title=Roberta Williams |trans_title=1999年蘿貝塔·威廉斯的訪談 |work=adventureclassicgaming.com |author=Andy Bellatti |date=1999-10-25 |accessdate=2012-10-05}}</ref><ref>{{cite journal zh |title=幽魂 Phantasmagoria |journal=[[軟體世界|軟體世界]] |volume=73 |pages=7頁 |publisher=[[智冠科技|智冠科技]] |location=台灣 |accessdate=2012-10-05 |date=1995年4月}}</ref>

而《幽魂》的劇情劇本內容多達約550頁，並使用超過100張的[[分鏡圖|分鏡圖]]及花了4個月時間拍攝，之後完成的成品包含超過800個遊戲場景。<ref name="interview" />

後續2006年蘿貝塔·威廉斯在網站adventureclassicgaming.com上的訪談時，表示著《幽魂》是她遊戲設計職業生涯中的代表作品。<ref>{{Cite web |url=http://www.adventureclassicgaming.com/index.php/site/interviews/198/ |title=Roberta Williams |trans_title=2006年蘿貝塔·威廉斯的訪談 |work=adventureclassicgaming.com |author=Philip Jong |date=2006-07-16 |accessdate=2012-10-05}}</ref>

=== 重製版本 ===

1999年雪樂山公司在PC上推出了《Phantasmagoria Stagefright》，其為將《幽魂》與《幽魂2》兩份作品合裝一套的版本。

後續在2010年遊戲採購網站[[Good_Old_Games|Good Old Games]]將《幽魂》以[[電子軟體分發|電子軟體分發]]方式販售，讓使用者能透過付費後下載遊玩。<ref>{{Cite web |url=http://www.gog.com/en/gamecard/phantasmagoria |title=Phantasmagoria for download |trans_title=付費下載版的《幽魂》 |work=gog.com |accessdate=2012-10-05}}</ref>

== 反應 ==

=== 銷售 ===

因《幽魂》包含大量不宜未成年的內容，當時在[[澳洲|澳洲]]被禁止販售，部份如[[CompUSA|CompUSA]]的零售商也未將《幽魂》上架販售。

儘管具備多項爭議情節，《幽魂》仍為1995年銷售最熱門的PC電子遊戲之一。<ref>{{Cite web |url=http://articles.orlandosentinel.com/1995-12-16/lifestyle/9512140575_1_ms-dos-cd-rom-gt-interactive |title=Best-selling Titles: Games |trans_title=1995年電子遊戲販售成績 |work=orlandosentinel.com |accessdate=2012-10-05}}</ref>
