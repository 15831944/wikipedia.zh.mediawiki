'''幽靈漢字'''（{{Jpn|j=幽霊文字|hg=ゆうれいもじ}}）是[[JIS_X_0208|JIS基本漢字]]所收錄的出處不明的[[漢字|漢字]]的總称。

== 概述 ==
JIS C 6226（後來的[[JIS_X_0208|JIS X 0208]]）中規定的JIS第1、第2水準漢字在下文中統稱「JIS基本漢字」，它由當時的[[通商產業省|通商產業省]]於1978年制定。

但制定JIS基本漢字時，由於規格未要求明示各文字的典據，JIS基本漢字中出現了不少音、義、歷史、用例皆不明確的文字，即所謂的「幽靈文字」，大約有60字，一說超過100字。

最具代表性的有「[[wikt:妛|妛]]」、「[[wikt:彁|彁]]」等字。「妛」是JIS X 0208第1次規格（JIS C 6226-1978）中選定收錄的文字時，疑是因殘損處被誤認為一橫而產生的<ref>{{Cite book| author = JIS X 0208:1997| title = 7ビット及び8ビットの2バイト情報交換用符号化漢字集合| year = 1997| publisher = 日本規格協会| pages = pp.289f| language=ja}}</ref><ref>{{cite book|author = 笹原宏之| title = 日本の漢字| year = 2006| publisher = 岩波書店| series = 岩波新書| isbn = 4-00-430991-3| pages = p82| language=ja}}</ref>（原字應為「{{JIS2004フォント|𡚴}}」，收錄於{{Link-ja|JIS X 0213|JIS X 0213}}）。而「彁」則來源、用例均無法判定（同定不能）<ref>{{Cite book| author = JIS X 0208:1997| title = 7ビット及び8ビットの2バイト情報交換用符号化漢字集合| year = 1997| publisher = 日本規格協会| pages = pp.291f| language=ja}}</ref>。

在日本規格JIS X 0208:1997（JIS基本漢字）中，有若干來源不明的漢字，這些字稱作「幽靈文字」。1978年制定JIS標準時，因為沒有註明每個字的出處，所以混進了不少音義不詳、來源不明的漢字。據說本來有100多個漢字被定義為「幽靈文字」，而經過仔細研究，已經找出大部分字的出處（多用於人名、地名），而現在被定義為「幽靈文字」的漢字減少到12個字<ref>http://bbs.zdic.net/forum.php?mod=viewthread&tid=244595&highlight=%E5%B9%BD</ref>。

== 殘留的原因 ==
[[笹原宏之|笹原宏之]]等人對幽靈文字的調査是1997年的JIS漢字改正工作中的一環。JIS漢字改正曾於1983年的「83JIS改正」中，對字形、文字編碼作出大幅變更，引起了嚴重的混亂，而本次改正則不希望重蹈覆轍。另外，由於JIS漢字已經收錄於[[Unicode|Unicode]]，此時對JIS的變更已經不僅是日本國內的問題。結果，幽靈文字不僅在JIS基本漢字中被保留，也可以-{zh-cn:通过; zh-tw:透過;}-[[電腦字型|-{zh-cn:计算机字体; zh-tw:電腦字型;}-]]於全世界的電腦中使用。

== 用例 ==
原本來源不明的文字，即使弄清了其正字、字義，使用頻率依然不高。實際上，被判定為錯別字的漢字有時會用來代用本字，而至今仍來源不明的字有時會用來代用字形相似的文字，作為異字、代替字、俗字而存在。

另外，有人利用幽靈文字「來源不明」「讀法不明」的特性，將其作為暗号文書，即所謂「忍者文字」。
==幽灵汉字在汉语中==
“幽灵汉字”是日语中的说法，不少幽灵汉字在汉语中有意义{{來源請求|reason=未给出详细来源|date=2019年3月17日}}，如：

*妛：古同“媸”，见《汉语大字典》。
*墸：同“躇”，见《汉语大字典》。
*壥：古同“廛”，见《汉语大字典》。
*挧：姓。见《中华字海》。
*槞：古同“櫳”，见《汉语大字典》。 
*閠：古同“闰”，见《汉语大字典》。 
*椦：古同“棬”，见《汉语大字典》。

以下汉字仍无出处：
*彁
*暃
*蟐
*駲
