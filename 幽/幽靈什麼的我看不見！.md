《'''幽靈什麼的我看不見！'''》（{{lang-ja|'''ゆうれいなんか見えない!'''}}）是[[日本|日本]][[作家|作家]][[紫雪夜|紫雪夜]]（{{lang|ja|むらさきゆきや}}）所創作的[[輕小說|輕小說]]，[[插圖|插圖]]由Munyu（{{lang|ja|むにゅう}}）以及Sugar Picola（{{lang|ja|しゅがーピコラ}}）擔任{{noteTag|1=第一冊由Munyu負責封面以及內頁的插圖，第二冊開始由Sugar Picola擔任內頁插圖，而Munyu負責封面的插圖。}}。在2009年3月31日以《幽靈什麼的我看不見》（{{lang|ja|幽霊なんか見えない}}）的標題入圍GA文庫大獎，於同年4月30日獲頒第一屆GA文庫大獎中的獎勵獎{{noteTag|1='''GA文庫大獎'''（{{lang-ja|[[:ja:GA文庫大賞|GA文庫大賞]]}}）是由日本[[Softbank|SOFTBANK Creative]]所評選頒發出來的獎賞。}}<ref>{{cite web|url=http://ga.sbcr.jp/novel/taisyo/01/4th.html | script-title=ja:GA文庫大賞：途中経過＆結果発表」|publisher={{lang|ja|GA文庫}}|date=2009-03-31|accessdate=2012-04-18 |language=ja}}</ref>。該作品於2010年4月開始在[[SoftBank|SOFTBANK Creative]]所出版的小說類別[[GA文庫|GA文庫]]上連載，後來[[單行本|單行本]]出版後改為現今的標題<ref>{{cite web|url=http://ga.sbcr.jp/novel/taisyo/01/5th.html | script-title=ja:GA文庫大賞：途中経過＆結果発表」|publisher={{lang|ja|GA文庫}}|date=2009-04-30|accessdate=2012-04-18 |language=ja}}</ref>。在2011年5月15日發售第四冊限定版的同時附贈[[廣播劇|廣播劇]]CD<ref name="vol4">{{cite web|url=http://ga.sbcr.jp/mgabunko/017076/ | script-title=ja:「ゆうれいなんか見えない！ 4」に二つの限定版が登場！|publisher={{lang|ja|GA文庫}}|date=2011-04-27|accessdate=2012-04-18 |language=ja}}</ref><ref name="vol4-1">{{cite web|url=http://www.koepota.jp/news/2011/03/16/0801.html | script-title=ja:「ゆうれい」「ニャル子」「かんなぎ家」「俺修羅」がドラマCDとして2ヵ月連続で発売|publisher={{lang|ja|声優 ニュース イベント サイト}}|date=2011-03-16|accessdate=2014-04-07 |language=ja}}</ref>。番外篇在2010年8月10日發行的《GA Magazine Vol.4》（{{lang|ja|GAマガジン}}）刊載短篇〈神社之怪〉（{{lang|ja|神社の怪}}）<ref>{{cite web|url=http://ga.sbcr.jp/mgabunko/015370/ | script-title=ja:コーティ抱き枕カバー＆ニャル子が表紙のＧＡマガジン第４号発売！|publisher={{lang|ja|GA文庫}}|date=2010-08-02|accessdate=2012-04-18 |language=ja}}</ref>。

== 登場人物 ==
=== 主要人物 ===
; {{nihongo|調 敦志|調 敦志（しらべ あつし）}}
:本作品的男主角，就讀水森學園高等部一年C班。中學時期看得到[[幽靈|幽靈]]的緣故使得周遭的人覺得他是個古怪的人。為了與中學時代的同班同學分隔，並且過著平靜的校園生活而就讀離家遠的高中。因此自己一人住在套房公寓裡。他是鬼之眼的持有者，因為某種理由被封印。
; {{nihongo|鞍馬 依|鞍馬 依（くらま より）||聲優：[[田村由香里|田村由香里]]}}
:本作品的女主角，就讀水森學園小學部三年二班。擁有細緻的白髮跟琥珀色的眼睛，喜歡與開玩笑跟甜食的驅魔師，書包是她的標準裝備。
; {{nihongo|姬倉 香利奈|姫倉 香利奈（ひめくら かりな）}}
:就讀水森學園高等部，擔任學生會長，喜歡花朵。
; {{nihongo|御堂 美穗|御堂 美穂（みどう みほ）}}
:敦志的同班同學，與藤岡智久從小是青梅竹馬，單戀著敦志。
; {{nihongo|藤岡 智久|藤岡 智久（ふじおか ともひさ）}}
:敦志的同班同學，與御堂美穗從小是青梅竹馬，不會出手打女孩子。
; {{nihongo|鞍馬 雹一郎|鞍馬 雹一郎（くらま ひょういちろう）}}
:鞍馬家當今的當家，同時也是依的未婚夫。
; {{nihongo|山田 費歐蕾緹娜|山田 フィオレンティーナ（やまだ ふぃおれんてぃーな）}}
:臉上有傷痕的女孩子，簡稱費歐娜。

=== 敵對人物 ===
; {{nihongo|眠蛇|眠れる蛇（ねむれるへび）}}
:被當作神明供奉的災難之神。
; {{nihongo|牛鬼|牛鬼（うしおに）}}
:擁有牛的頭蜘蛛的身體的妖。
; {{nihongo|蜜柑|蜜柑（みかん）}}
:眠蛇的信奉者，是一個[[貓又|貓又]]但是她討厭別人叫她貓又，經常與鞍馬蒼月一起行動。
; {{nihongo|鞍馬 蒼月|鞍馬 蒼月（くらま　そうげつ）}}
:鞍馬依的養父，試圖引發災難並瞄準擁有鬼之眼的持有者。總是穿著法衣的老人。

== 用語 ==
;{{nihongo|水森學園|水森学園（みずもりがくえん）}}
:在同一塊校地內有小學、中等，高中的綜合巨大學園。
;{{nihongo|鬼之眼|鬼ノ眼（おにのめ）}}
:可以看見萬物的眼睛，只有強大靈力的人所擁有的能力。
;{{nihongo|眠蛇的信奉者|眠れる蛇の信奉者（ねむれるへびのしんぼうしゃ）}}
:為了讓被斬首的眠蛇覺醒的信徒。
;{{nihongo|妖|妖（およずれ）}}
:對世間很強烈怨恨的死者靈魂，長期漂浮時與動物的靈魂混合的人，原本在人世時的記憶消失。
