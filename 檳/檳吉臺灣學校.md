'''馬來西亞檳吉臺灣學校'''，簡稱'''檳吉臺校'''，英文校名縮寫為'''CTSP'''（Chinese Taipei School Penang），前身為'''檳城臺灣僑校'''（Penang Tai Chiao School），由「[[英業達集團|英業達集團]]」創辦人[[溫世仁|溫世仁]]先生設立於1991年，為海外第一所臺灣學校。

學校成立之初尚無校舍，因此租借[[檳城|檳城]][[韓江中學|韓江中學]]（今[[韓江學院|韓江學院]]）部分教室進行教學；1996年，遷校至檳城中路「閱書報社」；2004年，「閱書報社」業主欲收回建物成立「[[檳城孫中山紀念館|檳城孫中山紀念館]]」，於是馬來西亞臺灣商會總會長李芳信博士，籌畫以[[檳城|檳城州]]臺灣商會及[[吉打|吉打州]]臺灣商會共同接辦臺灣學校。2005年，檳城臺校在[[中華民國教育部|教育部]]輔導之下再次搬遷，保留原有的教師、學生及設備，於[[檳城州|檳城州]][[北赖|北赖]]成立「檳吉臺灣學校」，校名「檳吉」的由來，即取自「[[檳城|檳城州]]」、「[[吉打|吉打州]]」兩州州名的首字結合而成。
　　

== 學校沿革 ==
*1991年：2月25日，「'''檳城臺灣僑校'''」成立，設有小學部，租借韓江中學校舍。
*1992年：8月，成立初中部。
*1993年：8月，成立高中部。
*1996年：3月，遷校至檳城「閱書報社」（今「孫中山紀念館」）。
*2004年：臺灣商會總會長李芳信博士發起籌辦「檳吉臺灣學校」。
*2005年：8月，於北海才能園成立「'''檳吉臺灣學校'''」。
*2009年：6月，檳吉球場落成啟用。
*2019年：6月30日，惜別感恩活動與校史展覽。

== 學制與課程 ==
* 採[[臺灣教育|臺灣學制]]，每學年分上、下學期，每學期開學、結業式、寒暑假日期與臺灣大致相同。
* 學部分為小學部、國中部、高中部，每年級各一個班級。
* 教科書採用臺灣書商版本，教材與臺灣同步銜接；小學部英文科採分級授課，共分為六級。
* 授課科目以[[中華民國國語|華語]]進行教學（英文科除外）；教材採用[[繁體中文|正體中文]]及[[注音符號|注音符號]]。
* 小學部三至六年級加授[[馬來語|馬來文]]課程。
　
== 特色活動 ==
* 檳吉盃語文競賽：分為「小學說故事比賽(華語/英語)」、「中學演講比賽(華語/英語)」。
* 教師教學輔導講座活動：每學年上學期舉辦，從臺灣聘請高中、國中、國小教師蒞校指導，為期一週。
* 親師生輔導研習活動：每學年上學期舉辦，從臺灣聘請大學教授、中小學輔導教師蒞校教學，為期一週。
* 登山健行活動：每學年上學期舉辦。
* 元旦升旗活動：每年元旦舉辦，邀請駐馬代表處、北馬地區臺商、僑胞與會。
* 文化教學活動：每學年下學期舉辦，由[[中華民國僑務委員會|僑委會]]協助、從臺灣聘請民俗技藝或舞蹈等專家蒞校教學，為期三至四週。
* 臺商臺校聯合運動會：每學年下學期舉辦，由檳吉臺灣學校、檳城州臺灣商會及吉打州臺灣商會共同承辦。
　
== 刊物出版 ==
