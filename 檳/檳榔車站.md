'''檳榔車站'''位於[[台灣|台灣]][[臺東縣|臺東縣]][[卑南鄉|卑南鄉]]，曾為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]（現已廢止）的[[鐵路車站|鐵路車站]]。

== 概況 ==
檳榔站設於1922年，在1982年[[臺東線#臺東線拓寬計畫|東拓工程]]完成前，臺東線由卑南溪畔經賓朗、[[東成車站|東成]]、[[初鹿車站|初鹿]]、[[嘉豐車站|嘉豐]]及[[中興車站|中興]]才到達今日的[[鹿野車站|鹿野站]]。其中檳榔站及嘉豐站由於坡度高達千分之21，設計成[[折返式鐵路|折返式車站]]。

== 車站構造 ==
*折返式車站

== 利用狀況 ==
* 東拓工程進行改線，完工後，現已廢止。

== 車站週邊 ==
*賓朗國小

== 歷史 ==
*1922年4月20日：設立「'''日奈敷乘降場'''」（{{jpn|j=日奈敷乗降場|kn=ひなしきじょうこうじょう|rm=Hinashiki Jōkōjō}}），取自當地部落名「檳榔樹格」（''pinaski''）。<ref>{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>
*1943年：改稱「'''日奈敷停車場'''」（{{jpn|j=日奈敷停車場|kn=ひなしきていしゃじょう|rm=Hinashiki Teijajō|link=no}}）
*戰後（時間不詳）：改為「檳榔」，仍為原住民語音譯。
*1982年6月27日：隨著東擴線通車後廢止。
