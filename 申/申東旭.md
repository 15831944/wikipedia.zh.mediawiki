'''申東旭'''（{{lang-ko|'''신동욱'''}}，{{bd|1982年|9月14日|catIdx=S申}}），[[韓國|韓國]][[男演員|男演員]]。2003年[[KBS|KBS]]第20期公開招募合格演員。2010年入伍服役期間，罹患罕見疾病「複雜性局部疼痛症候群」（CRPS），因持續治療及復健，長達7年時間未能拍戲。<ref>[http://kpopn.niusnews.com/=K2uujfs9 演員申東旭罹患罕見疾病]kpopn</ref>

== 經歷 ==
2003年KBS第20期公開招募合格演員，同期演員包括[[鄭敬淏|鄭敬淏]]、[[智鉉寓|智鉉寓]]等。由於申東旭身材修長，外貌清秀，當時倍受期待。

2004年起陸續擔任多部電視劇配角，至2006年開始在[[MBC|MBC]]《靈魂伴侶》擔任主角。
2007年與[[朴新陽|朴新陽]]共同演出[[SBS|SBS]]水木連續劇《[[錢的戰爭|錢的戰爭]]》。

2010年入伍服役期間，被診斷出罹患罕見疾病「複雜性局部疼痛症候群」（Complex regional pain syndrome, CRPS），於2011年除役<ref>{{Cite web |url=https://www.kpopn.com/2011/07/27/申東旭因疾病判定除役/ |title =申東旭因疾病判定除役}}</ref>。因CRPS之故，申東旭有超過六年的時間無法演戲，直到2017年，終於以[[MBC|MBC]]《[[守望者_(電視劇)|守望者]]》中神父一角回歸<ref>{{Cite web |url=http://tenasia.hankyung.com/archives/1170605#_enliple  |title =신동욱 측 “‘파수꾼’ 출연 확정… 건강 상태 호전” (공식)}}</ref>。

2018年在[[tvN|tvN]]週末劇《[[Live_(韓國電視劇)|Live]]》中飾演崔明浩警長，正直、能幹、重義氣、性格好的形象，加上帥氣的外貌及演技力，受到觀眾相當的注目與好評<ref>{{Cite web |url=http://www.kookje.co.kr/mobile/view.asp?gbn=v&code=0500&key=20180320.99099008467#cb |title =‘라이브’ 성동일-장현성-신동욱-이시언,‘선배 라인’ 배우 군단}}</ref>。

在連續擔任兩次配角後，申東旭終於在2018年下半年製作播出的[[MBC|MBC]]週四綜藝電視劇《[[大長今在看著|大長今在看著]]》中重新回歸男主角的位置<ref>{{Cite web |url=https://hk.mobi.yahoo.com/celebrity/大長今在看著-舉行發布會-申東旭權俞利等出席-103028064.html}}</ref>。

2019年1月被爆與爺爺簽訂「孝道合約」獲贈房屋及土地，但未有履行照顧對方的承諾，並將房屋送給女友，要求爺爺在兩個月內搬走。爺爺將其告上法庭。申東旭則反駁指是爺爺主動贈送土地，對爺爺的說法感到荒唐，並想要將房屋及土地歸還爺爺。<ref>{{Cite web|title=稱對方自願送屋同地 申東旭否認曾呃爺爺|url=https://hk.on.cc/hk/bkn/cnt/entertainment/20190105/bkn-20190105015328750-0105_00862_001.html|work=on.cc東網|accessdate=2019-03-05|language=zh-hk}}</ref>事件曝光後數日，透過經理人公司表示爲了不影響劇組其他人員，主動退出正在拍攝中的[[TVN|tvN]]電視劇《[[觸及真心|觸及真心]]》，角色其後由[[李尚禹|李尚禹]]接替演出。<ref>{{Cite web|url=http://www.bntnews.cn/app/news.php?nid=53720|title=申东旭从《触及真心》下车 tvN﹕不影响拍摄进度|accessdate=2019-03-05|author=|date=2019-01-07|publisher=bnt新闻}}</ref><ref>{{Cite web|title=tvN《觸及真心》李尚禹接替申東旭出演！新預告再公開…劉寅娜誓言要把李棟旭「撕碎」？|url=https://www.koreastardaily.com/tc/news/113059|work=KSD 韓星網|accessdate=2019-03-05|language=zh-Hant}}</ref>
2019年2月7日申東旭的祖父出面道歉，表示自己對孫子提出的孝道詐欺訴訟是嚴重的誤會，對於自己單方面的主張對申東旭造成巨大的傷害和損失感到抱歉。<ref>{{Cite web |url=http://m.chosun.com/svc/article.html?sname=news&contid=2019020702003&Dep0=www.google.com&utm_source=www.google.com&utm_medium=unknown&utm_campaign=news#Redyho |title ='효도사기' 소송 낸 배우 신동욱 조부 "큰 오해" 손자에게 사과}}</ref>

== 社會公益活動 ==
2017年1月至2018年1月25日，申東旭透過DAUM網站story funding平臺，以「培養宇宙狂粉」為主題進行公開募款。<ref>{{Cite web |url=https://storyfunding.daum.net/project/12844?#tabMenu/ |title =배우 신동욱의 우주덕후 키우기 /스토리펀딩}}</ref>

2018年4月，申東旭利用前述活動募得之款項，與出版社茶山BOOKS共同捐贈480冊科學圖書給韓國白血病兒童癌症協會。<ref>{{Cite web |url=http://m.nowtimes.co.kr/news/articleView.html?idxno=3212 |title =다산북스와 배우 신동욱, 한국백혈병소아암협회에 과학 도서 기증}}</ref>

2018年5月，申東旭接受韓國白血病兒童癌症協會聘任為宣傳大使，參與兒童癌症認知宣導、治療基金籌募等相關活動。<ref>{{Cite web |url=http://www.soaam.or.kr/board/view.php?code=nanumTalk&cat=&sq=6456&PHPSESSID=6c1fc424073f49010d58c10cb1d55fcb&ckattempt=1 |title =배우 신동욱, (사)한국백혈병소아암협회 홍보대사 위촉}}</ref>
