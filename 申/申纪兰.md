'''申纪兰'''（{{bd|1929年|12月|catIdx=S}}），女，[[汉族|汉族]]，[[山西省|山西省]][[长治市|长治市]][[平顺县|平顺县]]人。[[中华人民共和国|中华人民共和国]][[全国劳动模范|全国劳动模范]]，[[厅局级正职|正厅级]]干部<!-- 注：行政级别是终身的-->，從1954年當選第1屆中华人民共和国[[全国人民代表大会代表|全国人民代表大会代表]]的代表開始，连续13届当选中國人大代表<ref>{{Cite web |url = http://news.sina.com.cn/c/2013-03-10/045926483738.shtml |title = 人民日报撰文掌声送给申纪兰 称选民对其信任 |author = 冀业 |date = 2013年3月10日 |publisher = 《[[人民日报|人民日报]]》 |language = zh-cn |accessdate = 2013年3月10日 |quote =  }}</ref>。

==生平==
===早期===
1947年，18岁的申纪兰嫁到[[山西省|山西省]][[平顺县|平顺县]][[西沟乡_(平顺县)|西沟乡]]西沟村，一直居住至今。<ref name="fbfz" />此前，1943年2月6日，[[李顺达|李顺达]]在西沟村响应[[中共中央|中共中央]]、[[毛泽东|毛泽东]]“组织起来”的号召，联络了其他六户农民，在中国较早建起了一个农业劳动互助组。他们采取劳武结合（即农业劳动同对敌武装斗争相结合）的方法，既克服困难发展生产，度过灾荒，又不耽误参军、参战、支前。此后，李顺达及其互助组连续多年受到[[中国共产党|中国共产党]]领导下的地方政府的表彰。<ref>[http://book.sohu.com/20050503/n225429922.shtml 李顺达——新中国第一代著名劳模，搜狐，2005年05月03日] {{webarchive|url=https://web.archive.org/web/20150518091349/http://book.sohu.com/20050503/n225429922.shtml |date=2015年5月18日 }}</ref>

===中华人民共和国成立初期===
[[File:Shen_Jilan_in_1953.jpg|缩略图]]
1950年，申纪兰联系了十多位要好的姐妹加入[[互助组|互助组]]<ref name="fbfz" />。1951年3月9日，《人民日报》第一版刊登了[[新华社|新华社]]播发的《保证今年农作物丰收！李顺达互助组向全国挑战——[[中央人民政府农业部|中央农业部]]号召奋起应战 形成爱国竞赛热潮》的报道<ref>保证今年农作物丰收！李顺达互助组向全国挑战——中央农业部号召奋起应战 形成爱国竞赛热潮，人民日报1951年3月9日，第1版</ref><ref>[http://medianet.qianlong.com/7692/2004/06/21/33@2120411.htm 新华社山西分社原社长:穆青同志激励着我们! ，千龙网，2004-06-21] {{Wayback|url=http://medianet.qianlong.com/7692/2004/06/21/33@2120411.htm |date=20150518082028 }}</ref>。由此在全中国掀起了“爱国主义生产竞赛”<ref>爱国主义生产竞赛澎湃展开，中国农业科学1951年04期</ref>。

1952年，上级批准李顺达组织26户农民办起了初级农业生产合作社，名为“西沟农林牧生产合作社”。李顺达被社员推选为社长，申纪兰被推选为副社长，该合作社民主制定了合作社章程。这是中华人民共和国第一个初级农业生产合作社。为了挖掘劳动潜力，李顺达支持申纪兰率领妇女同男人展开劳动竞赛，争取妇女同工同酬。办社当年，恰逢春旱，申纪兰组织了一个妇女小组，积极耢耙保墒，担水点种，担水浇苗，同男人一样干活。由此，妇女们通过自身的努力争取到了同工同酬。结果该年获得大面积丰产，秋后的粮食产量比互助组时期每亩增产30多斤，由此获得[[中央人民政府农业部|中央人民政府农业部]]颁发的金星奖章。自此，李顺达把社名改为“西沟金星农林牧生产合作社”<ref name="czjf">[http://www.czzx.changzhi.gov.cn/czzxcms/wenxianshiliao/2011-08-24/2272.html 李顺达与金星农林牧生产合作社，载 长治市政协文史处编，长治文史资料 第十一辑 解放篇 纪念长治解放50周年] {{webarchive|url=https://web.archive.org/web/20121031103053/http://www.czzx.changzhi.gov.cn/czzxcms/wenxianshiliao/2011-08-24/2272.html |date=2012-10-31 }}</ref>。

西沟金星农林牧生产合作社贯彻自力更生、艰苦奋斗、勤俭办社原则，使该社自成立后发展较为兴旺。比如1955年春前，副社长申纪兰所提的“发动妇女上山拾羊粪”的建议被采纳之后，所积羊粪约值1000元[[人民币|人民币]]以上。1955年[[毛泽东|毛泽东]]主持编辑《[[中国农村的社会主义高潮|中国农村的社会主义高潮]]》一书（1956年出版）时，收入了中共平顺县委书记李琳、[[新华社|新华社]]驻山西分社记者马明撰写的介绍该社事迹的《勤俭办社，建设山区》一文，毛泽东为此撰写按语称<ref>《中国农村的社会主义高潮》按语（一九五五年九月、十二月），新华网，于2013-03-02查阅</ref><ref name="czjf" />：
{{引用|这里说的是李顺达领导的金星农林牧生产合作社。这个合作社办了三年，变成了一个包括二百八十三户的大社。这个社所在的地方是那样一个太行山上的穷地方，由于大家的努力，三年工夫，已经开始改变了面貌。劳动力的利用率，比抗日以前的个体劳动时期提高了百分之一百一十点六，比建社以前的互助组时期也提高了百分之七十四。合作社的公共积累已经由第一年的一百二十元，增加到了一万一千多元。一九五五年，社员每人平均收入粮食八百八十四斤，比抗日以前增加了百分之七十七，比建社以前增加了百分之二十五点一。这个社已经做了一个五年计划，实行三年的结果，生产总值已经达到五年计划的百分之一百零点六。这个合作社的经验告诉我们，如果自然条件较差的地方能够大量增产，为什么自然条件较好的地方不能够更加大量地增产呢？ }}

西沟村也名扬全中国。[[中华人民共和国|中华人民共和国]]成立后很长时间内，西沟村是全国地图上唯一标出的行政村<ref name="fbfz">[http://news.ifeng.com/mainland/special/2013lianghui/yanlun/detail_2013_03/02/22662614_0.shtml 申纪兰：我觉得共产党还是好的 腐败分子都是混进来的，凤凰网，2013年03月02日]</ref>。

=== 全国人民代表大会代表任期 ===
[[File:Hu_Wenxiu,_Guo_Lanying,_Li_Hui_and_Shen_Jilan_in_1954.jpg|缩略图]]的四位山西女代表，右一为申纪兰]]
[[File:Shen_Jilan_at_Tiananmen_in_1954_National_Day.jpg|缩略图]]，在[[天安门|天安门]]观礼台的全国人大代表，前排左三为申纪兰]]
1953年，申纪兰加入[[中国共产党|中国共产党]]。1954年，申纪兰当选[[第一届全国人大代表|第一届全国人大代表]]，骑着毛驴到北京开会，投票选举[[毛泽东|毛泽东]]为[[中华人民共和国主席|中华人民共和国主席]]。此后她成为唯一一位连任13届全国人大代表的人<ref>{{cite news |url = http://www.infzm.com/content/57507 |title=申纪兰 连任55年的全国人大代表 |publisher=南方人物周刊|date=2011-04-02 |language = zh-hans}}</ref><ref name="fbfz" />。她说，令她印象最深的还有1975年国务院总理[[周恩来|周恩来]]在[[第四届全国人民代表大会第一次会议|第四届全国人民代表大会第一次会议]]上最后一次作[[政府工作报告|政府工作报告]]的场景。她回忆称，“我听说周总理当时生着病，看着他那么瘦，还站在台上作报告，我一边听一边掉眼泪，我旁边的人也在掉眼泪，我们一边哭一边鼓掌。”<ref name="fbfz" />

后来，她担任过中共[[平顺县|平顺县]]委副书记。1973年，出任山西省妇联主任。在出任妇联主任初期，她曾赴[[太原|太原]]就职，但发现实在无事可做。有位[[新华社|新华社]]记者为她出主意称，主任继续当，但只在开会时来太原，平时则继续在西沟村劳动。申纪兰同他人商量后，采用了该办法，仅到妇联“坐会”，开完会后便回西沟村。<ref name="fbfz" />

此后，她长期担任山西省[[长治市|长治市]]人大常委会副主任，西沟村党总支副书记，西沟“金星经济合作社”社长。李顺达于1983年逝世后，申纪兰成为西沟村的精神象征。申纪兰在西沟乡很有威望，人称“申主任”。西沟乡建于1960年代的“西沟展览馆”中，展出有李顺达、申纪兰等人的事迹。<ref name="fbfz" />

媒体曾公开报道，申纪兰在2008年5月将“一生的积蓄”1万元捐赠[[汶川大地震|汶川大地震]]灾区。申纪兰是一家注册资金五千万元的贸易公司『山西申纪兰贸易有限公司』的董事长，还有一家房地产公司『申纪兰房地产开发公司』 。申纪兰的回应是：曾办过公司，钱是由村集体出的，后来转让了，她未从中赚取钱。<ref>{{cite web |url = http://news.dl.soufun.com/2012-03-05/7188673_3.html |title = 申纪兰：我没搞房地产 |publisher = 搜房网（来源：中国网） |author= |date=2012-03-05 |accessdate=2012-03-06 |language = zh-hans}}</ref>

2013年3月，在[[第十二届全国人民代表大会第一次会议|第十二届全国人民代表大会第一次会议]]山西代表团首次全体会议上，[[中共山西省委|中共山西省委]]书记[[袁纯清|袁纯清]]表示：“作为正厅级干部，她保持了人民本色，谁能做得到？这是我们的骄傲和自豪。这就是我们理直气壮的理由。”<ref>{{Cite web |url = http://news.ifeng.com/mainland/special/2013lianghui/yanlun/detail_2013_03/04/22687878_0.shtml |title =袁纯清：正厅级申纪兰保持人民本色 值得理直气壮  |author =  |date =2013-03-04  |publisher =  凤凰网|language =  |accessdate =  |quote =  }}</ref>之后，申纪兰当选全国人大代表时，[[中国共产主义青年团|共青团]]山西省委[[微博|微博]]批评道：“[[反右运动|反右]]你代表，[[文革|文革]]你代表，任何时候你都代表，可你一次也没有代表过人民......试问你是怎样被选举的？你究竟代表谁？”这条微博很快就被删除，省委之后也在该帐号上澄清，言论不代表团省委意见，纯系维护人员操作失误<ref>{{Cite web |url = http://news.sina.cn/?sa=t124d7682831v71&from=wap |title =共青团山西省委微博斥申纪兰|date =2013-02-01  |publisher =  新浪网|language =  |accessdate =  2018-01-31|quote =  }}</ref>。

晚年，申纪兰仍然定居西沟村，坚持“不转户口、不定级别、不要工资、不配专车、不脱离群众、不离开西沟”，耳聪目明，腿快腰直，经常下地劳动，每日收看[[新闻联播|新闻联播]]。作为人大代表，申纪兰关心扶贫攻坚、污染防治、农村教育医疗卫生、农村基础设施建设等方面工作<ref>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1980719|title = 89岁全国人大代表申纪兰：经常下地劳动，每天收看新闻联播 |author =  |date =2018-02-02   |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

2018年12月18日，在庆祝改革开放40周年大会上，申纪兰获得改革先锋称号，被誉为“初心不改的农村的先进模范代表”。<ref>{{Cite web|url=http://stock.jrj.com.cn/2018/12/18110926758636.shtml|title=大会宣读获得改革先锋称号、中国改革友谊奖章人员名单|accessdate=2018-12-18|author=|date=2018-12-18|work=|publisher=证券时报·e公司}}</ref>

== 言论 ==
2010年，申纪兰表示：“我非常拥护共产党。当代表就是要听党的话，我从来没有投过反对票。”<ref>{{cite news |title=“最老”人大代表申纪兰:55年从没投过反对票 |publisher=|date= |url = http://news.qq.com/a/20090311/003017.htm|format=|language= |accessdate= |quote= }}</ref>这一言论引发了广泛的社会争论<ref>{{cite news|title=申纪兰老人的话令人遗憾|publisher=法制晚报|date=|url=http://fzwb.ynet.com/article.jsp?oid=49429853|language=|accessdate=|quote=}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。而申纪兰则在2011年回应称，作为[[人民代表大会制度|人民代表大会制度]]的见证人，内心拥护的事，她就赞成，不拥护的，她就不投票<ref>{{cite news |title = 申纪兰回应“从未投过反对票”：不拥护的就不投票 |website = 凤凰网 |date= |url = http://news.ifeng.com/mainland/special/2011lianghui/detail_2011_03/10/5065256_0.shtml |language= |accessdate= |quote = }}</ref>。媒體引述，其被民間戲稱為「舉手機器」、「人大活化石」<ref name="apple0303"/>。

《[[南方都市报|南方都市报]]》称，申纪兰不上网，也不明白[[互联网|互联网]]是怎么回事，但对互联网有些意见。她认为有些人在网上看了不正当的东西，就毒害青年了。
{{cquote|我有个想法，网也应该有人管，不是谁想弄就能弄，就跟人民日报一样，外国那些人那是瞎弄的，咱不能这样，咱要按照原则去弄，不要好的弄成坏的了，想说什么就说什么，咱是共产党领导下的社会主义国家。……这个网，你谁想上就能上？还是要组织批准呢？}}
