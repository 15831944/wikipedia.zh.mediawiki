《'''申根协议'''》（{{lang-de|Schengener Abkommen}}；{{lang-fr|Convention de Schengen}}；{{lang-nl|Verdrag van Schengen}}），是一項[[歐洲大陸|歐洲大陸]]國家間的條約協定，其簽約目的是取消相互之間的邊境檢查點，並協調對申根區之外的邊境控制。即在成員國相互之間取消邊境管制，持有任一成員國有效[[身份證|身份證]]或[[申根簽證|申根簽證]]的人可以在所有成員國境內自由流動。根據該協定，旅遊者如果持有其中一國的旅遊簽證即可合法地到所有其他申根國家。

《申根公約》的成員國亦稱“申根國家”或者“申根公約國”，成員國的整體又稱“[[申根區|申根區]]”。申根区目前包含26个国家，其中有22个属于[[欧盟|欧盟]]成员。四个非欧盟成员国中，[[冰岛|冰岛]]和[[挪威|挪威]]以[[北欧护照联盟|北欧护照联盟]]成员国的身份加入申根区，官方分类属于与欧盟申根区活动相关的国家。不屬於歐洲大陸的[[愛爾蘭|愛爾蘭]]及[[英國|英國]]沒有加入。

== 歷史 ==
《申根公約》於1985年6月14日，由5個[[歐洲共同體|歐洲共同體]]國家（[[西德|西德]]、[[法國|法國]]、[[荷蘭|荷蘭]]、[[比利時|比利時]]、[[盧森堡|盧森堡]]）在[[盧森堡|盧森堡]]的一個小城市[[申根|申根]]簽署。由於[[歐盟|歐盟]]成立的影響，進入1990年代初，又有有5個歐洲國家陸續加入申根，分別為[[義大利|義大利]]、[[西班牙|西班牙]]、[[葡萄牙|葡萄牙]]、[[希臘|希臘]]、[[奧地利|奧地利]]。

1995年3月26日，申根公約正式生效，首先簽約的5個國家，取消邊境檢查。

北歐5國於1996年12月19日加入申根，分別為[[丹麥|丹麥]]、[[冰島|冰島]]、[[瑞典|瑞典]]、[[挪威|挪威]]、[[芬蘭|芬蘭]]。

2004年5月1日加入[[歐盟|歐盟]]的10個國家，同時加入《申根協定》，分別是[[愛沙尼亞|愛沙尼亞]]、[[拉脫維亞|拉脫維亞]]、[[立陶宛|立陶宛]]、[[波蘭|波蘭]]、[[捷克|捷克]]、[[匈牙利|匈牙利]]、[[斯洛伐克|斯洛伐克]]、[[斯洛文尼亞|斯洛文尼亞]]、[[马耳他|马耳他]]、[[賽普勒斯|賽普勒斯]]。根據各國達成的安排，歐盟將分步驟吸收10個新成員國加入《申根協定》。這些國家與15個現協定國之間的陸路和海路邊卡於2007年12月撤除，而機場邊境檢查也在2008年3月30日取消，只有賽普勒斯和馬爾他因為是島國的關係，要求保留部分邊境管制措施。

2004年10月16日，[[瑞士联邦议会|瑞士联邦议会]]通过加入《申根协定》的协议，2005年6月5日，瑞士[[全民公投|全民公投]]以微弱多数通过了《申根协定》<ref>[http://news.sina.com.cn/w/2005-06-07/02386097458s.shtml 瑞士“天堂之门”向欧洲洞开(组图)]</ref>。2008年12月12日瑞士正式加入申根区<ref>[http://news.sina.com.cn/o/2008-12-12/140714871164s.shtml 国际纵横:瑞士正式加入申根区]</ref>。

2007年後，[[羅馬尼亞|羅馬尼亞]]、[[賽普勒斯|賽普勒斯]]、[[保加利亞|保加利亞]]、[[列支敦士登|列支敦士登]]分別簽訂加入申根公約，原預計分別均會在2011年內正式實施。除[[列支敦士登|列支敦士登]]外，其餘三國均於2011年宣布延後實施，至今沒有加入。

2013年7月1日起，[[克羅地亞|克羅地亞]]成為[[歐盟|歐盟]]成員國，目前預計2016年申請加入申根公約。

== 被延遲加入申根區的國家 ==
[[意大利|意大利]]、[[奧地利|奧地利]]、[[希臘|希臘]]三國雖於1990年代初加入申根區，但因安全或是邊界海防等因素均被推遲加入申根區，前兩國於1997年才獲准加入申根區，[[希臘|希臘]]則是直到2000年才允許加入申根區。

[[羅馬尼亞|羅馬尼亞]]、[[保加利亞|保加利亞]]、[[塞浦路斯|塞浦路斯]]三國虽已簽訂加入申根公約，但[[塞浦路斯|塞浦路斯]]因北賽問題被推遲加入申根區，[[羅馬尼亞|羅馬尼亞]]和[[保加利亞|保加利亞]]因國內治安與情勢仍较其他歐盟國家不穩，目前被部份歐盟國家反對而被推遲加入申根區。

== 爭議 ==
人口自由流动也带来治安问题，一些国家的治安，像[[瑞士|瑞士]]，在加入条约后治安有恶化趋势。

同时[[欧洲难民危机|欧洲难民危机]]也给人口自由移动带来新的挑战，[[德国|德国]]、[[法国|法国]]、[[比利时|比利时]]等均有暂停申根的举动。
