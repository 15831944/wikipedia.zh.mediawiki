'''柴俊畴'''（{{bd|？||？|}}）[[天津|天津]]人，[[清朝|清朝]]及[[中华民国|中华民国]]铁路工程师。

==生平==
柴俊畴是[[天津|天津]][[北洋武备学堂|北洋武备学堂]]铁路工程班1893年首届毕业生。毕业后先任[[关内外铁路|关内外铁路]]工程司，后来参与[[京张铁路|京张铁路]]建设。1907年7月18日（[[清朝|清朝]][[光绪|光绪]]卅三年六月九日）陪同[[詹天佑|詹天佑]]选线。1907年5月率员勘测设计[[京门铁路|京门支线]]。1909年10月5日（清朝[[宣统|宣统]]元年九月二日）被授予州同归部铨选<ref name=xuyong/>。

[[京张铁路|京张铁路]]开始筹建后，多名当年天津北洋武备学堂铁路工程班毕业生被派去参与京张铁路勘测和修筑，其中有[[陈西林|陈西林]]、[[俞人凤|俞人凤]]、[[翟兆麟_(民国人物)|翟兆麟]]和柴俊畴等。詹天佑在《京张铁路工程纪略》中写道：“本路工程始终出力各员为：正工程司[[颜德庆|颜君德庆]]、陈君西林、俞君人凤、翟君兆麟，工程司柴君俊畴、[[张鸿诰|张君鸿诰]]、[[苏以昭|苏君以昭]]、[[张俊波|张君俊波]]，余繁不及备载。”<ref name=bilichi>{{Cite web |url = http://www.peoplerail.com/rail/show-474-213159-1.html |title =  翟兆麟和他的比例尺|author =  |date =2015-01-28 |publisher =人民铁道网  |language =  |accessdate =  |quote =  }}</ref>

在京张铁路勘测阶段，詹天佑、陈西林、张鸿诰、[[徐文炯|徐文炯]]先详细定线，将整条线路划分成3段，分别派工程司勘测。[[邝孙谋|邝孙谋]]、俞人凤和柴俊畴主要负责第一段柳村至[[南口地区|南口]]的勘测。颜德庆、陈西林、张鸿诰和苏以昭负责第二段南口至[[岔道城|岔道城]]的勘测，该段是京张铁路最难修的地段，关沟段就在其中。翟兆麟和[[山海关北洋铁路官学堂|山海关北洋铁路官学堂]]毕业生[[刘锜|刘锜]]、[[李鸿年|李鸿年]]和[[耿瑞芝|耿瑞芝]]负责第三段岔道城至[[张家口|张家口]]的勘测，该段中的鹞儿梁山峭坡陡，修筑难度不下于关沟段，詹天佑曾亲自参与此处的选线比对。京张铁路先后共修4条支线，京门支线和鸡鸣驿煤矿支线均是为煤炭运输而建<ref name=bilichi/>。

1914年6月至9月，柴俊畴参与[[北京环城铁路|北京环城铁路]]初测。[[中华民国|中华民国]]时期，柴俊畴历任京绥铁路局副总工程司兼京丰总段长（1919年）等职<ref name=xuyong>许勇，天津武备学堂与中国铁路精英，世纪桥2011年21期</ref>。

1916年4月1日，[[中华工程师学会|中华工程师学会]]会长詹天佑向柴俊畴颁发会员证，编号第二十六号。1962年，柴俊畴之子柴寿縥将该会员证捐赠给青龙桥詹天佑陈列室，1987年拨交[[詹天佑纪念馆|詹天佑纪念馆]]，在[[中国铁道博物馆|中国铁道博物馆]]第二批文物定级时被定为国家二级文物<ref name=zheng>{{Cite web |url = http://www.peoplerail.com/rail/show-435-204505-1.html |title =B4 百年铁路见证：中华工程师学会会员证书  |author =  |date = 2014-11-21 |publisher = 人民铁道网 |language =  |accessdate =  |quote =  }}</ref>。
