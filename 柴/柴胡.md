'''柴胡'''（拉丁文：Bupleuri Radix），又稱作'''北柴胡'''、'''南柴胡'''、'''芽柴胡'''，《'''[[神农本草经|神农本草经]]'''》中称为'''茨胡'''，为[[伞形科|伞形科]]多年生[[草本植物|草本植物]][[北柴胡|北柴胡]]（Bupleurum chinense DC.）和[[狭叶柴胡|狭叶柴胡]]（南柴胡 Bupleurum scorzonerifolium Willd.）的根或全草。同属植物如[[银州柴胡|银州柴胡]]、[[兴安柴胡|兴安柴胡]]、[[长白柴胡|长白柴胡]]等亦可入药。

《[[本经|'''神农本草经''']]》言：柴胡“主心腹肠胃结气，饮食积聚，寒热邪气，推陈致新。”现代用以疏散退热，疏肝解郁，升阳举陷。

== 分佈 ==
*北柴胡：[[河北|河北]]、[[河南|河南]]、[[陝西|陝西]]。
*南柴胡：[[江蘇|江蘇]]、[[四川|四川]]、[[湖北|湖北]]。
== 性味归经 ==
味苦，性微酸，無毒。归[[手厥阴心包经|心包经]]、[[肝_(臟腑)|肝]]、[[三焦|三焦]]、[[膽_(臟腑)|胆]]经。

== 功效 ==
解表退熱，疏肝解鬱，升舉陽氣。 


和解表里，疏肝，升阳。治寒热往来，胸满胁痛，口苦耳聋，头痛目眩，疟疾，下利脱肛，月经不调，子宫下垂。
《本经》：主心腹肠胃中结气，饮食积聚，寒热邪气，推陈致新。
（《中药大辞典）1183页）

*肝鬱型的慢性肝炎本品有較好療效
== 禁忌 ==
陰虛潮熱肝火上升忌用
