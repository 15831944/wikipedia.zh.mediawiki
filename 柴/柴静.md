'''柴静'''（{{bd|1976年|1月1日|}}），生于中國[[山西省|山西]][[临汾市|临汾]]，[[中国|中国]]女[[记者|记者]]及[[主持人|主持人]]，以深入新闻前线及具批判性的调查而闻名。

== 經歷 ==
19岁时（約在1995年），柴静就在湖南文艺广播电台主持《[[夜色温柔|夜色温柔]]》节目。

22岁时（約在1998年），她在[[湖南卫视|湖南卫视]]主持《[[新青年|新青年]]》节目。

2001年起，供职于[[中国中央电视台|中国中央电视台]][[中国中央电视台新闻频道|新闻频道]]，先后在《[[时空连线|时空连线]]》、《[[新闻调查|新闻调查]]》、《[[24小时_(节目)|24小时]]》、《[[面对面|面对面]]》、《看见》等栏目担任主持人与记者 <ref name="mzs ">{{cite web|url=http://cul.book.qq.com/zt2012/kanjian/index.htm|title=柴静《看见》官网|accessdate=2012年12月15日|work=|publisher=腾讯文化|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20121217063957/http://cul.book.qq.com/zt2012/kanjian/index.htm|archivedate=2012年12月17日}}</ref>。

2003年，曾深入抗击[[非典型肺炎|非典型肺炎]]第一线，被《[[南方都市报|南方都市报]]》评选为年度风云记者。

2006年起，每年作为[[两会|两会]]（[[全国人民代表大会|全国人民代表大会]]和[[中国人民政治协商会议|中国人民政治协商会议]]）观察员，报道会议召开期间的热点议题。

2012年12月，出版讲述十年央视历程的自传性作品《[[看见|看见]]》，该书由[[广西师范大学出版社|广西师范大学出版社]]出版，首印高达50万册<ref name="ww" >{{cite web |url= http://book.people.com.cn/n/2012/1213/c69360-19889018.html|title=柴静新书《看见》首发50万册( |accessdate= 2012年12月13日|work= |publisher=人民网 |language =zh-cn}}</ref><ref>{{cite web |url=http://culture.ifeng.com/event/detail_2012_12/26/20537363_0.shtml |title= 好友谈柴静：她不啰嗦不爱玩 敢跟众老男人拼白酒|accessdate=2012年12月26日 |publisher=凤凰网 |archiveurl= http://culture.ifeng.com/event/detail_2012_12/26/20537363_0.shtml|archivedate=2012年12月26日 |work=晶报	作者：邓妍 邵登 |language=zh-cn}}</ref>。

2013年7月，《看见》节目暂停播出，据同事透露这与柴静准备赴美生子有关。

2013年10月，柴靜在美國產下1女，取名柴知然，引發輿論「假愛國」的爭議<ref>{{cite web|url=http://qjwb.zjol.com.cn/html/2014-02/18/content_2536734.htm?div=-1|title=钱江晚报|work=zjol.com.cn}}</ref><ref>{{cite web|url=http://hk.apple.nextmedia.com/international/art/20140218/18628893|title=「女神」赴美誕女 被轟假愛國 - 蘋果日報 - 兩岸國際 - 20140218|work=Apple Daily 蘋果日報}}</ref><ref>[http://www.appledaily.com.tw/appledaily/article/international/20140218/35647919/ 央視主播赴美產女「假愛國」  蘋果日報]</ref>。

2014年初，柴静从央视离职<ref>[http://ent.sina.com.cn/v/m/2014-10-13/09554223028.shtml 独家：柴静年初已离职央视 在家带孩子].新浪娱乐.2014年10月13日.[2014年10月14日].</ref>。

2014年2月，《[[南都娛樂週刊|南都娛樂週刊]]》在[[春節|春節]]前後拍到久未露面的柴靜，出現在[[北京首都国际机场|北京機場]]，手裡抱著1名女嬰<ref>{{cite web|url=http://www.sd.xinhuanet.com/news/2014-02/17/c_119374027.htm|title=柴靜美國産女後身材發福 老公趙嘉為知名攝影師|work=xinhuanet.com|deadurl=yes|archiveurl=https://web.archive.org/web/20170114024838/http://www.sd.xinhuanet.com/news/2014-02/17/c_119374027.htm|archivedate=2017-01-14}}</ref>。

2015年2月28日，自費拍攝與推出空气污染调查纪录片《[[穹顶之下_(纪录片)|穹顶之下]]》，影片以女兒生时患瘤為引入點<ref>{{cite web|url=http://i.ifeng.com/news/news?aid=96368831&vt=5&ch=rj_bd_me&mid=1LhPxe|title=柴静离职央视后花百万调查雾霾：女儿出生时患瘤|work=ifeng.com}}</ref>。《穹顶之下》在網上播出不久後，48小时后，该视频的总播放次数已超2亿，但同年3月2日，[[中共中央宣传部|中共中央宣传部]]作出指示，要求新闻媒体不要发布有关《穹顶之下》的报道，人民网和新华网等官媒开始撤下视频，并删除了几乎所有相关文章。<ref>http://www.bbc.com/zhongwen/simp/china/2015/03/150306_view_chaijing_politics</ref><ref>{{cite news|title=柴靜《穹頂之下》在爭議聲中被禁播|url=http://cn.nytimes.com/china/20150308/c08dome/zh-hant/|publisher=紐約時報|date=2015-03-08|accessdate=2015-03-10}}</ref>

2015年4月，入选《[[时代周刊|时代周刊]]》2015年“百位最具影响力人物”。<ref >{{cite web|url=http://www.bbc.co.uk/zhongwen/simp/china/2015/04/150416_time_100_list | language= | title=《时代》杂志“百人榜”：习近平及柴静均上榜 |publisher=BBC中文网 |date=2015年4月16日 |accessdate= }}</ref>
