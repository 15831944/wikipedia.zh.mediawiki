'''圆融无碍禅寺'''坐落于[[河南省|河南省]][[焦作市|焦作市]][[修武县|修武县]]西村乡当阳峪村，当地称'''圆融寺，'''始建公元351年，国家4A级景区、焦作市[[佛教|佛教]]协会驻地、[[韩国|韩国]]圆融宗的祖庭，是中国唯一一处体现佛教圆融思想的佛寺，中原唯一坐西向东的千年古刹，有“太行小灵山”的美誉。寺内有不同朝代的珍宝，至今未收取过门票。<ref>{{Cite web|url=http://www.zynews.com/lvyou/20181111104511.html|title=河南必去4座千年古刹！极灵验，都没门票！游客：这才是真寺庙_中原网|accessdate=2019-01-10|work=www.zynews.com}}</ref><ref>{{Cite web|url=http://www.sohu.com/a/280257650_381004|title=河南18个地市的千年古寺 神秘的地方 悠久的历史 源远流长_中国|accessdate=2019-01-10|work=www.sohu.com}}</ref>

== 历史 ==
[[File:圆融寺一角.jpg|缩略图]]
圆融寺历经各朝代战火兴废，包括近代二战（[[华北治安军|皇协军]]），[[文化大革命|文化大革命]]期间更是遭受惨重毁坏。2002年焦作市启动了圆融寺复建工程。<ref>{{Cite web|url=http://news.sina.com.cn/o/2009-07-16/070215961062s.shtml|title=“大哉问与小哉问——圆融禅寺”系列之二　成住坏空圆融寺|accessdate=2019-01-10|author=大河报|date=|work=news.sina.com.cn|publisher=新浪新闻}}</ref>

== 文化 ==
所谓“圆融”，即“破除偏执，圆满融通”；“无碍”，乃“心无挂碍，行无所碍”。

== 景点 ==
现今圆融寺为十一进计十九殿、两塔院、七堂、二院落、一馆及十七处泉，低处为山门，最高处的天然[[溶洞|溶洞]]“古佛洞”暂时未开放，寺庙方面称会根据复建进度开放。洞内原有文革幸存的“白玉佛”，现存[[修武县|修武县]]博物馆，为[[国家一级文物|国家一级文物]]。<ref>{{Cite web|url=http://www.huaxia.com/ytsc/zywh/hzhn/2009/07/1500643.html|title=“大哉问与小哉问——圆融禅寺”系列之三　新古建古造像古寺“圆融”|accessdate=2019-01-10|author=|date=|work=华夏经纬网|publisher=}}</ref>寺内可见历朝历代建寺文物，包括高250厘米的[[隋朝|隋朝]]沙石卢舍那坐佛、[[明朝|明朝]]白玉辟支坐佛、[[唐代|唐代]]汉七级白玉佛塔及唐代狮虎兽、石虎，明朝白玉马、供桌、文殊吼，[[清朝|清朝]]的戏水兽等。<ref>{{Cite web|url=http://zx.hnjzta.gov.cn/yrs.jhtml|title=圆融寺 - 焦作旅游|accessdate=2019-01-10|work=zx.hnjzta.gov.cn}}</ref>

旅游区内罗汉寺是新开发建设的一处景点，是全国唯一的窑洞式罗汉洞。

另外，寺庙所处的村落还有当阳峪窑遗址，其绞胎瓷烧制技艺为已列入国家非遗。<ref>{{Cite web|url=https://news.artron.net/20130929/n515864.html|title=绞胎瓷的前世今生_非遗_雅昌新闻|accessdate=2019-01-10|work=news.artron.net}}</ref>
