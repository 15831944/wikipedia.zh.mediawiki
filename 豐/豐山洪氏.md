'''豐山洪氏'''（{{lang-ko|풍산 홍씨}}）是朝鮮民族的[[本貫|本貫]]，位於[[朝鮮半島|朝鮮半島]][[慶尚北道|慶尚北道]]東北部[[安东市|安東郡]]的[[豐山邑|豐山面]]<ref name=surname_loc>{{cite web |url=http://www.surname.info/local/pung_san.html |publisher=성씨정보 ([[姓氏情報|姓氏情報]])|title=풍산 (豊山) |accessdate=2013-02-17 |language=ko }}</ref>，始祖是[[高麗王朝|高麗]][[高麗高宗|高宗]]29年文科狀元<ref name=surname_hong>{{cite web |url=http://www.surname.info/hong/pung_san.html |publisher=성씨정보 ([[姓氏情報|姓氏情報]])|title=풍산홍씨 (豊山洪氏) - 시조(始祖) : 홍지경(洪之慶) |accessdate=2013-02-17 |language=ko }}</ref>、[[國學直學|國學直學]][[洪之慶|洪之慶]]。當中的知名人物有[[朝鮮正祖|朝鮮正祖]]的生母[[惠慶宮洪氏|惠慶宮洪氏]]和[[洪國榮|洪國榮]]，還有近代的[[兴親王妃|兴親王妃]]（[[朝鮮高宗|朝鮮高宗]]的嫂嫂）、[[洪震|洪震]]和[[洪良浩|洪良浩]]。
