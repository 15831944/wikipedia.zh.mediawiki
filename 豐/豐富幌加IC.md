'''豐富幌加IC'''（{{lang|ja|豊富幌加インターチェンジ}}）是一個位於[[北海道|北海道]][[天鹽郡|天鹽郡]][[豐富町|豐富町]]幌加的[[交流道|交流道]]，屬於[[豐富繞道|豐富繞道]]的一部分。

== 道路 ==
* [[豐富繞道|豐富繞道]]

== 收費站 ==
國道繞道不設收費站。

== 接續道路 ==
* [[北海道道923號目梨別豐富線|北海道道923號目梨別豐富線]]
* [[國道40號|國道40號]]（間接接續）

== 歷史 ==
* 2004年11月6日：[[豐富繞道|豐富繞道]]的[[豐富佐呂別IC|豐富佐呂別IC]]至[[豐富北IC|豐富北IC]]路段通車，該交流道正式啟用。
* 2010年3月1日：伴隨著[[幌富繞道|幌富繞道]]的[[幌延IC|幌延IC]]至豐富佐呂別IC路段通車，而變更IC編號。
