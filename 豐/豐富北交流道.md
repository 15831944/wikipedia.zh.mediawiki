'''豐富北IC'''（{{lang|ja|豊富北インターチェンジ}}）是一個位於[[北海道|北海道]][[天鹽郡|天鹽郡]][[豐富町|豐富町]]開源的[[交流道|交流道]]，屬於[[豐富繞道|豐富繞道]]的一部分。

至2010年為止，豐富北IC為日本最北端的交流道。

== 道路 ==
* [[豐富繞道|豐富繞道]]

== 收費站 ==
國道繞道不設收費站。

== 接續道路 ==
* [[國道40號|國道40號]]
* [[北海道道138號豐富猿拂線|北海道道138號豐富猿拂線]]

== 歷史 ==
* 2004年11月6日：[[豐富繞道|豐富繞道]]的[[豐富佐呂別IC|豐富佐呂別IC]]至豐富北IC路段通車。
* 2010年3月1日：伴隨著[[幌富繞道|幌富繞道]]的[[幌延IC|幌延IC]]至豐富佐呂別IC路段通車，而變更IC編號。
