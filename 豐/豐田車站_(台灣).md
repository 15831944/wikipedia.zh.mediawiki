'''豐田車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[壽豐鄉|壽豐鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。本站配合「花東新車站運動」計畫進行改建，改建工程已正式完工。

== 車站構造 ==
*[[岸式月台|岸式月台]]兩座(第二月台原為[[島式月台|島式月台]]，因側線拆除而成為[[岸式月台|岸式月台]])。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#簡易站|簡易站]]。
*為極少數停靠對號列車之簡易站。
== 停靠狀態 ==
*[[莒光號列車|莒光號]]：停靠
順行602次、640次(逢週六行駛)、642次(逢週六、週日停駛)、6642次(逢週日行駛)。 
逆行653(逢週日停駛)、6653次(逢週日行駛)、611次。
*[[臺鐵區間車|區間車]]：皆停靠。
順行4552次 逢週六、日及逢例假日停駛。 
順行4550次 逢週六、日及逢例假日行駛。
*對號列車停靠率：14.0%；所有車種停靠率：30.6%。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1913年5月1日：設立'''豐田乘降場'''（招呼站）。
*1916年9月10日：站房等設施完工，同日起改設'''豐田驛'''。
*東拓前的舊站房係重建於民國42年（1953年）5月，直到東拓時才被拆除，而東線拓寬時站場路線及月台及路線也一樣被加高，而今日所見之站係完成於民國70年（1981年）10月。
*2015年10月1日：由於壽豐=南平間雙軌化啟用，不再辦理列車交會、待避等行車業務，本站由[[三等站|三等站]]降為[[簡易站|簡易站]]，並指定由[[壽豐車站|壽豐站]]管理。
*2016年6月28日：啟用多卡通刷卡機。
