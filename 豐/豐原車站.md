'''豐原車站'''位於[[臺灣|臺灣]][[臺中市|臺中市]][[豐原區|豐原區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺中線|臺中線]]的[[鐵路車站|鐵路車站]]。過去曾為臺灣鐵路管理局[[東勢線|東勢線]]（現已廢止）的端點站。2016年10月16日在[[台中鐵路高架化|台中鐵路高架化]]第一階段工程完工後，啟用新的高架車站。

== 車站構造 ==
*目前為高架車站，配置兩座島式月台四股道，並預留擴建至3島式月台6股道之機制。係配合[[臺鐵捷運化|臺鐵捷運化]]政策所興建。
*原先使用中的臨時車站，原為[[島式月台|島式月台]]二座，後因臺中地區鐵路高架計畫，改為[[島式月台|島式月台]]、[[側式月台|側式月台]]各一座。
*地面時期的豐原車站，站內2樓設有超商，高架化後2017年5月中旬於車站一樓設置便利超商，於2017年4月上旬設置飲料自動販賣機。

===車站出口===
*東側出口：豐勢路
*西側出口（臨時）：中正路，經臨時通道連結原前站出口

== 歷史 ==
*1905年5月15日：增設「葫蘆墩驛」。
*1920年：改名為「豐原驛」。
*戰後改名為「豐原車站」。
*1959年1月12日：東勢線正式通車啟用。
*1960年11月：因客貨業務驟增，原有站房不敷使用而改建為鋼筋水泥建築站房。
*1991年9月1日：東勢線廢線。
*2008年2月25日：開始停靠[[太魯閣列車|太魯閣號]]。
*2015年6月30日：啟用多卡通刷卡機。
*2016年10月16日：高架站體第一階段完工並先行通車，各級列車改行駛於高架軌道。
*2016年11月間：豐原車站舊站房拆除
*2017年3月5日：西側出口行人通道調整，原舊站臨時通道封閉。
*2018年9月13日：豐原西站正式開放<ref>車站公告</ref>

== 轉乘運輸 ==
=== 臺中市公車 ===
*[[臺中市公車|臺中市公車]]站名之「豐原」站停靠豐原車站西側的[[豐原客運豐原站|豐原客運豐原站]]，而「豐原東站」停靠於豐原車站東側的廣五停車場內，為豐原重要的市公車[[轉運站|轉運站]]。
**豐原站位於臺中市豐原區三民路46號，提供往東勢、卓蘭、谷關、-{后}-里、大甲、臺中等地的市公車路線，亦提供國道路線（6606）。
**豐原東站位於豐原車站東側廣五停車場內，提供往社口、神岡、大雅、沙鹿、大肚、清水、梧棲等地的市公車路線。
**另[[統聯客運|統聯客運]]也在豐原車站西側設站，提供國道客運的另一種選擇（1617）。
*公車資訊更新時間為2018年4月17日。
