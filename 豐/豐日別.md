'''豐日別'''（{{lang-ja|トヨヒワケ或トヨビワケ}}）乃《[[古事記|古事記]]》之記述，祂是[[日本神話|日本神話]]裡的[[神祇|神祇]]，為[[豐國|豐國]]（今[[大分縣|大分縣]]與福岡縣東部）之[[國魂|國魂]]。

== 概要 ==
=== 古事記之記載 ===
依照《[[古事記|古事記]]》上卷記述[[伊奘諾尊|伊邪那岐命]]與[[伊奘冉尊|伊邪那美命]]產國，先生下淡道之穗之狹別島，次生伊豫之二名島（今[[四國|四國]]）。該島有一身四面，面面皆有其名：[[伊予國|伊豫國]]謂「愛媛」（今[[愛媛縣|愛媛縣]]）、[[讚岐國|讚岐國]]謂「飯依彥」（今[[香川縣|香川縣]]）、[[阿波國|粟國]]名「[[大宜都比賣|大宜都比賣]]」（今[[德島縣|德島縣]]）、[[土佐國|土左國]]稱「建依別」（今[[高知縣|高知縣]]）。接著生下[[隱岐群島|隱伎之三子島]]（又名天之忍許呂別）、筑紫島，該島有一身四面，面面皆有其名：[[筑紫國|筑紫國]]謂白日別（今[[福岡縣|福岡縣]]）、[[豐國|豐國]]謂'''豐日別'''（今[[大分縣|大分縣]]與福岡縣東部）、[[火國|肥國]]謂建日向日豐久士比泥別（今[[長崎縣|長崎縣]]、[[佐賀縣|佐賀縣]]、[[熊本縣|熊本縣]]一帶）、[[熊襲|熊曾國]]謂建日別（今[[鹿兒島縣|鹿兒島縣]]）。隨後再生[[壹岐島|伊伎島]]（又名天比登都柱）、[[对马岛|津島]]（又名天之狹手依比賣）、[[佐渡岛|佐度島]]、[[本州|大倭豐秋津島]]（又名天御虛空豐秋津根別）等，此謂大八島國。

== 解說 ==
神名裡的「豐」代表[[豐國|豐國]]（含[[豐前國|豐前國]]、[[豐後國|豐後國]]），既是豐國自身之神明，即為其[[國魂|國魂]]。

位於[[福岡縣|福岡縣]][[行橋市|行橋市]]的豐日別宮，社傳記載著[[欽明天皇|欽明天皇]]2年[[筑紫國|筑紫]]日別大神的神官大伴連牟彌奈里被一位自稱是[[猿田彥|猿田彥]]神託夢，表示猿田彥神是[[天照大神|天照大神]]的分身。既然豐日別大神坐鎮本宮，則猿田彥坐鎮於別宮。自此以後，豐日別和猿田彥即被視為同一神格。

== 信仰 ==
在日本供奉豐日別神的[[神社|神社]]計有下述：
* 豐日別宮（[[福岡縣|福岡縣]][[行橋市|行橋市]]）：別名草場神社。
* 闇無濱神社（[[大分縣|大分縣]][[中津市|中津市]]）：稱作「豐日別國魂神」。
* 溫泉神社（[[長崎縣|長崎縣]][[雲仙市|雲仙市]]小濱町）
