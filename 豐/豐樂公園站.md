'''豐樂公園站'''位於[[臺中市|臺中市]][[南屯區|南屯區]]，為[[臺中捷運|臺中捷運]][[臺中捷運綠線|綠線]]（[[烏日文心北屯線|烏日文心北屯線]]）之[[捷運車站|捷運車站]]。站址所在地之古地名為'''麻糍埔'''<ref>{{Cite web |url=https://www.taichung.gov.tw/9962/1030128/post |title=捷運綠線G12豐樂公園站坐擁商場與綠地 中市府打造植生牆相呼應 |publisher=[[臺中市政府|臺中市政府]] |work=[[臺中市政府交通局|臺中市政府交通局]] |date=2018-06-27 |archiveurl=https://archive.today/20180628014315/https://www.taichung.gov.tw/9962/1030128/post |archivedate=2018-06-28 |access-date=2018-08-23 |dead-url=no }}</ref>。

== 車站概要 ==
車站位於[[文心路|文心南路]]與文心南五路一段路口附近（文心南路上，[[南苑公園|南苑公園]]西側），所在地古地名為[[麻糍埔|麻糍埔]]，現為南屯區豐樂里；車站代碼為'''G12'''<ref>[http://www.dorts.gov.taipei/ct.asp?xItem=660998&CtNode=20285&mp=115001 臺北市政府捷運工程局 >捷運興建中路網 >臺中捷運烏日文心北屯線 >路線說明]，臺北市政府捷運工程局</ref>。

== 車站構造 ==
高架車站，兩座[[側式月台|側式月台]]，一個出入口。

=== 車站出口 ===
於南苑公園內設有單側處入口。另預留未來增設對側出入口之銜接口。
