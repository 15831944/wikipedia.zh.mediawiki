'''豐樂工業區'''位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[臺東市|臺東市]]，為一座由中華民國[[經濟部工業局|經濟部工業局]]所管轄的[[工業園區|工業園區]]，年產值約有[[新臺幣|新臺幣]]35億6730萬元。園區內主管機關為'''經濟部工業局豐樂工業區服務中心'''<ref name="經濟部工業局"/>。豐樂工業區是臺東縣境內唯二兩座較大型工業匯集的區域，另一座則是[[臺東縣政府|臺東縣政府]]規劃的利嘉工業區。<ref name="臺東市公所"/>

== 沿革 ==
豐樂工業區最初是[[中華民國經濟部|經濟部]]為配合政府推動之[[臺灣東部|東部]]區域計畫，以及加速東部各項交通建設以促進各項產業由[[臺灣西部|西部]]東移，因此由旗下工業局進行工業區之規劃與設計工作。<ref name="經濟部工業局"/><ref name="豐樂工業區服務中心"/>

後續實際園區施工委由[[中華工程公司|中華工程公司]]施作，並於1982年完成園區建設，隔年一月開始對外出售園區土地，當時工業局計畫將引進鐵材加工、[[機械|機械]]、[[電子|電子]]、[[電機|電機]]等製造修配業及[[食品|食品]]業等[[污染|污染]]性較小之工業。<ref name="經濟部工業局"/><ref name="豐樂工業區服務中心"/>

== 園區概況 ==
[[File:Fongle_Industrial_Park_02.jpg|thumb]]
豐樂工業區全區總開發面積為18.8948公頃，除了公共設施占用的4.3001公頃之外，其餘供廠商投資建廠的面積約有13.1648公頃。除此之外，豐樂工業區內另有畫設1.4299公頃的住宅社區用地。<ref name="豐樂工業區服務中心"/>

* 道路及綠地：32,000平方公尺。
* 路燈：全區共設有77盞[[水銀|水銀]]路燈。
* 排水溝：19,000平方公尺。
* 自來水工程：由[[台灣自來水公司|台灣自來水公司]]供應工業及民生用水，並且供水路線沿園區內道路設置，共分為100、150、200mm等三種管徑之輸水管線。
* 電力來源：由[[台灣電力公司|台灣電力公司]]馬蘭一次配電[[變電所|變電所]]進行供電，可供給161KV與11.4KV兩種電壓之電力，並且園區內輸電線路皆已完成地下化<ref name="台灣電力公司"/>。
* 電信系統：園區內電信線路已完成地下化。
* 環境綠化：園區內道路旁皆設有人行道，並規劃種植包含[[垂榕|垂榕]]、[[菩提|菩提]]等樹種<ref name="豐樂工業區服務中心"/>。

== 進駐概況 ==
目前豐樂工業區進駐廠商中，生產中的廠商有87家，新興建廠中的4家，處於停工狀態的廠商有3家，已歇業的廠商有5家。園區內主要的進駐廠商為[[汽車|汽車]]及其零件製造業、金屬製品製造業，以及基本[[金屬|金屬]]製造業。<ref name="經濟部工業局"/>
