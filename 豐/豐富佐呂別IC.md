'''豐富佐呂別IC'''（{{lang|ja|豊富サロベツインターチェンジ}}）是一個位於[[北海道|北海道]][[天鹽郡|天鹽郡]][[豐富町|豐富町]]字東豐富的[[交流道|交流道]]，屬於[[豐富繞道|豐富繞道]]和[[幌富繞道|幌富繞道]]的一部分。

== 道路 ==
* [[豐富繞道|豐富繞道]]
* [[幌富繞道|幌富繞道]]

== 收費站 ==
國道繞道不設收費站。

== 接續道路 ==
* [[北海道道84號豐富濱頓別線|北海道道84號豐富濱頓別線]]

== 歷史 ==
* 2004年11月6日：[[豐富繞道|豐富繞道]]的豐富佐呂別IC至[[豐富北IC|豐富北IC]]路段通車。
* 2010年3月1日：伴隨著[[幌富繞道|幌富繞道]]的[[幌延IC|幌延IC]]至豐富佐呂別IC路段通車，而變更IC編號。
* 2010年3月14日：[[幌富繞道|幌富繞道]]的[[幌延IC|幌延IC]]至豐富佐呂別IC路段通車。
