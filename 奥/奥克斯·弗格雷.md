'''奥克斯·弗格雷'''（'''Oakes Fegley'''，{{bd|2004年|11月11日||}}）是一名美國童星演員。他最出名是在電影《''{{link-en|Fort Bliss (film)|Fort Bliss (film)|Fort Bliss}}''》中飾演配角Paul Swann、電視劇《[[疑犯追蹤|疑犯追蹤]]》中飾演Gabriel，以及在2016年迪士尼電影《[[尋龍傳說|尋龍傳說]]》中飾演Pete<ref>{{Cite web |url = http://www.mcall.com/entertainment/movies/mc-petes-dragon-oakes-fegley-allentown-20160807-story.html |title = How Allentown's Oakes Fegley became a Disney star |author = Kathy Lauer-Williams |date = 2016-08-07 |publisher = The Morning Call |language = en}}</ref>。
