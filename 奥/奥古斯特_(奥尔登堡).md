'''保尔·弗里德里希·奥古斯特'''（'''Paul Friedrich August'''，{{bd|1783年|7月13日|1853年|2月27日|catIdx=A}}），[[奥尔登堡|奥尔登堡]]第一任大公，1829年至1853年在位。
== 家族与早年生活 ==
奥古斯特是奥尔登堡公爵彼得一世与符腾堡的弗里德里克·伊丽莎白·阿玛丽埃的长子，1783年生于拉施泰德。1803年到1805年间曾在[[莱比锡大学|莱比锡大学]]求学。法国兼并奥尔登堡期间，他跟随父亲流亡[[俄罗斯|俄罗斯]]，并参与了反抗[[拿破仑|拿破仑]]的战争。

1811年到1816年，奥古斯特任俄属[[爱沙尼亚|爱沙尼亚]]总督，期间废除了[[农奴制|农奴制]]。在1812年到14年参与战争后，他返回俄罗斯，后于1816年回到奥尔登堡。

奥古斯特的父亲于1829年3月21日去世，奥古斯特继位成为奥尔登堡公爵。
== 在位期间 ==
一周后的3月29日，奥古斯特改称大公，成为了奥尔登堡的第一位大公，尽管早在1815年奥尔登堡就成为了大公国。为了表示对父亲的敬意，奥古斯特颁发了奥尔登堡家族和彼得·弗里德里希·路德维希公爵的贡献勋章（''Oldenburgischer Haus- und Verdienstorden des Herzogs Peter Friedrich Ludwig''）。

1831年底，一部关于邦[[议会|议会]]起草[[宪法|宪法]]的法规出台。民众对于这部宪法的渴望导致了1848年的事件。在民众的反抗和建议下，1849年2月18日，邦议会通过，奥古斯特大公颁布了宪法。并在1852年进行了修订。

1853年，奥古斯特大公在奥尔登堡市去世，终年69岁。

=== 奥古斯特时期的奥尔登堡首相 ===
* 卡尔·路德维希·弗里德里希·约瑟夫·冯·勃兰登施泰因男爵（Karl Ludwig Friedrich Joseph Freiherr von Brandenstein），任期：1814年10月12日—1842年6月30日
* 京特尔·亨利·冯·贝格男爵（Günther Heinrich Freiherr von Berg），任期：1842年7月1日—1843年9月9日
* 威廉·恩斯特·冯·博留-马尔孔耐（Wilhelm Ernst Freiherr von Beaulieu-Marconnay），任期：1843年9月9日—1848年8月1日
* 约翰·亨利·雅各布·施洛伊费（Johann Heinrich Jakob Schloifer），任期：1848年8月1日—1849年8月13日
* 克里斯蒂安·狄德里希·冯·布特尔（Christian Diedrich von Buttel），任期：1849年8月13日—1851年5月1日
* 彼得·弗里德里希·路德维希·冯·罗辛男爵（Peter Friedrich Ludwig Freiherr von Rössing），任期：1851年5月1日—1874年6月23日

== 婚姻与子女 ==
[[File:Queen_Amalia_of_Greece.jpg|thumb]]
奥古斯特于1817年7月24日在[[绍姆堡|绍姆堡]]与[[安哈尔特统治者列表|安哈尔特-贝恩堡-绍姆堡-霍伊姆亲王]][[维克多·卡尔·弗里德里希_(安哈尔特-贝恩堡-绍姆堡-霍伊姆)|维克多·卡尔·弗里德里希]]的次女阿德莱德（Adelheid，1800年2月23日—1820年9月13日）结婚，阿德莱德是奥古斯特大公的远房表妹（二人都是英国国王乔治一世的后裔），育有二女：
*长女[[阿玛利亚王后_(希腊)|玛丽·弗里德里克·'''阿美莉埃''']]（'''Marie Friederike ''Amelie'''''，1818年12月21日—1875年5月20日），1836年与[[希腊君主列表|希腊国王]][[奥托一世_(希腊)|奥托一世]]结婚，成为希腊王后。
*次女伊丽莎白·玛丽·'''弗里德里克'''（'''Elisabeth Marie ''Friederike'''''，1820年6月8日—1891年3月20日），1855年嫁给马克西米利安·冯·华盛顿男爵。
大公夫人阿德莱德1820年便去世了。1825年6月25日，奥古斯特娶阿德莱德的四妹伊达（Ida，1804年3月10日—1828年3月21日）为第二任妻子，二人育有一子：
*长子[[彼得二世_(奥尔登堡)|尼库劳斯·弗里德里希·'''彼得''']]（'''Nikolaus Friedrich PETER'''，1827年8月7日—1900年6月13日），1853年继位为奥尔登堡大公。
1828年，大公夫人伊达去世。
[[File:Cäcilie_von_Schweden_Großherzogin_von_Oldenburg_by_Theodor_Hamacher.jpg|thumb]]
1831年5月5日，奥古斯特与堂侄女[[瑞典君主列表|瑞典国王]][[古斯塔夫四世·阿道夫|古斯塔夫四世·阿道夫]]的第三女塞西莉娅（Cäcilie，1807年6月22日—1844年1月27日）结婚，有三子：
*次子'''亚历山大'''·弗里德里希·古斯塔夫（'''''Alexander'' Friedrich Gustav'''，1834年—1835年），早夭。
*三子尼库劳斯·弗里德里希·'''奥古斯特'''（'''Nikolaus Friedrich ''August'''''，1836年—1837年），早夭。
*四子安东·京特尔·弗里德里希·'''埃里玛'''（'''Anton Günther Friedrich ''Elimar'''''，1844年1月23日—1895年10月17日），1876年与地位较低的贵族小姐娜塔莉（Natalie Frn Vogel von Friesenhof）结婚。娜塔莉于1896年被封为韦尔斯堡女伯爵（Gfn von Welsburg）。这次婚姻被认为是贵庶通婚，他们的一子一女不被认为是奥尔登堡家族的成员。
