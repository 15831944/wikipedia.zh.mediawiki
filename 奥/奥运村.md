'''奧運村'''（別稱'''奧林匹克村'''或'''選手村'''），是於每屆[[夏季奧林匹克運動會|夏季奧運]]與[[冬季奧林匹克運動會|冬季奧運]]中，主辦國為各國運動員、官員、裁判、志願者和工作人員等提供的[[宿舍|宿舍]]。
== 起源 ==
由[[1896年夏季奧林匹克運動會|1896年夏季奧運]]至[[1920年夏季奧林匹克運動會|1920年]]的奧運中，各國的運動員、官員都分散於不同的[[賓館|賓館]]內。此舉被[[國際奧委會|國際奧委會]]認為並「不利於奧運組織工作及各國運動員之間的交流」。於是並在《[[奧林匹克憲章|奧林匹克憲章]]》中規定：奧運會組委會應提供「至少一座在奧林匹克運動會開幕式前兩周至閉幕式後3天期間可入住的奧林匹克村」。<ref name="espnsex">{{cite web

== 開始 ==
到了1924年舉行的[[1924年夏季奧林匹克運動會|第8屆巴黎奧運]]中，[[法國|法國]]首次把運動員等刻意的集中於指定的房屋中，後來，奧運村漸漸的成為了每屆奧運的指定宿舍。當中的宿舍一定要在主體育場、練習場附近，並且必須提供膳食和[[交通|交通]]服務，同時要有如[[醫院|醫院]]、[[商店|商店]]等其他配套設施。

隨著奧運愈來愈廣泛，愈多國家和運動員參與，奧運村的面積亦越大和越多配套設施，如於[[2004年夏季奧林匹克運動會|2004年雅典奧運]]中，整個奧運村佔地1.24[[平方公里|平方公里]]，更包括了一所醫院、興奮劑檢測中心、幾個宗教場所、[[餐廳|餐廳]]、[[游泳池|游泳池]]、[[跑道_(體育)|跑道]]和兩個室內訓練館。除了奧運以外，其他大規模的體育賽事也會提供選手村。

奧運結束後，有些奧運村會拆除，另外有些則會改裝後做為[[商品房|商品房]]、[[公營住宅|公營住宅]]或[[社會住宅|社會住宅]]出租、出售，或轉交[[大學|大學]]使用作寄宿用途。
