-{zh-cn:'''奥陌陌'''; zh-hk:'''首使星'''; zh-tw:'''斥候星'''，台語諧音暱稱'''黑麻麻'''<ref name="tammib"/><ref>{{Cite web |url = http://pansci.asia/archives/135560 |title = 真．星際過客：首次發現太陽系外的星際天體「黑麻麻」ʻOumuamua！ |author = 歐柏昇 |date = 2018-02-22 |publisher = [[PanSci_泛科學|PanSci 泛科學]] |language = zh-tw |access-date = 2018-06-04 |archive-url = https://web.archive.org/web/20180504074412/http://pansci.asia/archives/135560 |archive-date = 2018-05-04 |dead-url = no }}</ref>}-（''''Oumuamua'''<ref group="n">以[[夏威夷语|夏威夷語]]命名，意為「第一位來自遠方的信使」。</ref><ref>{{Cite web |url = http://www.storm.mg/article/362130 |author = 楊舒晴 |date = 2017年11月22日 |title = 天外飛來一根紅色雪茄？人類首度觀測到「星際訪客」，它來自織女星！ |publisher = [[風傳媒|風傳媒]] |access-date = 2017-11-22 |archive-url = https://web.archive.org/web/20171122011250/http://www.storm.mg/article/362130 |archive-date = 2017-11-22 |dead-url = no }}</ref>、'''1I/2017 U1'''<ref group="n">1I/2017 U1为其学名，“1I”中的字母I代表了星际天体，1是此类天体中的第一个。</ref> ，舊稱'''A/2017 U1'''、'''C/2017 U1 (PANSTARRS)'''）<ref name="tammib">[http://tamweb.tam.gov.tw/V3/mobile/content.asp?mtype=c2&idx=1740 2017-11-08 瓶中信──誰人寄來的小行星 1I/2017 U1 （'Oumuamua） - 網路天文館] {{Wayback|url=http://tamweb.tam.gov.tw/V3/mobile/content.asp?mtype=c2&idx=1740 |date=20171201032253 }}，[[臺北市立天文科學教育館|臺北市立天文科學教育館]]</ref> （陸譯「-{奥陌陌}-」，臺譯「-{斥候星}-」，港譯「-{首使星}-」） 是已知的第一颗經過[[太陽系|太陽系]]的[[星際彗星|星際天體]]。它於2017年10月18日（UT）在距離地球約{{convert|0.2|AU|km mi|abbr=on|lk=on}}處被[[泛星計畫|泛星1號]]望遠鏡發現，並在極端[[雙曲線|雙曲線]]的軌道上運行。起初它被認為屬於一顆彗星，但經過長時間觀測研究后後，發現其不具有[[彗髮|彗髮]]構造，也不是[[小行星|小行星]]<ref>{{cite journal |last1=Rafikov |first1=Roman R. |title=Spin Evolution and Cometary Interpretation of the Interstellar Minor Object 1I/2017 'Oumuamua |journal=arXiv |date=2018-09-17 |url=https://arxiv.org/abs/1809.06389 |accessdate=2018-10-11 |author= |archive-url=https://web.archive.org/web/20181011043810/https://arxiv.org/abs/1809.06389 |archive-date=2018-10-11 |dead-url=no }}</ref>。

依據最初二個星期的觀測，2017 U1的軌道[[離心率|離心率]]是1.1922 ± 0.00268，是在已知在太陽系內天體最高的<ref name="pseudoMPEC"/>；之前的紀錄是{{link-en|C/1980 E1|C/1980 E1}}的1.057<ref name="JPL-ecc"/>。以如此高的離心率進入，顯示它必然是一顆星際天體，會不受太陽引力的約束，以高離心率離開太陽系。

== 觀測 ==
這可能是[[星際彗星|星際天體]]的第一個例子，相對於太陽以每秒26公里的[[雙曲線軌道|雙曲線過剩速度]]出現在[[天琴座|天琴座]] <ref group="n">相較之下，彗星{{link-en|C/1980 E1|C/1980 E1}}只在500天文單位的距離上以4.2Km/s的速度移動。</ref>。這個方向接近[[太陽向點|太陽向點]]，是太陽系外天體最有可能出現的方位<ref name=skyandtelescope/>。如果天體測量沒有嚴重的問題，唯一可能的軌道就是明確的雙曲線<ref name="Discovery"/>。

假設它的[[反照率|反照率]]是10%，有可能是顆直徑約160米的岩石<ref name=skyandtelescope/>。4.2米的[[威廉·赫歇爾望遠鏡|威廉·赫歇爾望遠鏡]]在10月25日觀察了它的光譜，呈現出[[海王星外天體#Colors|紅色]]，像是古柏帶的天體<ref name=WHT/>。

回推它的軌道，計算出這顆小行星在2017年9月9日通過了[[近日點|近日點]]，並在2017年10月14日以{{ convert|0.16|AU|km mi|abbr=on|lk=on}}的距離掠過地球。<ref name=jpldata/>。這顆天體微小且黯淡，光度始終在[[視星等|視星等]]始終在21等之下<ref name="precoveryobs"/>。通過近日點後，朝[[飛馬座|飛馬座]]方向離去。

噴射推進實驗室的計算機顯示，該天體在100年前距離太陽大約為558天文單位（830億公里）。

在2017年10月26日，從[[卡特林那巡天系統|卡特林那巡天系統]]在10月14日和10月17的資料中找到兩次的[[回溯發現|回溯發現]] <ref name="precoveryobs"/>。

== 小行星的本質 ==
在2017年10月25日，[[甚大望遠鏡|甚大望遠鏡]]（VLT）以非常深刻的疊加影像發現該天體沒有任何顯示有[[彗髮|彗髮]]的跡象。因此，該天體被重新命名為A/2017 U1，成為第一顆從彗星永遠[[彗星#命名規則|轉命名為小行星]]的天體<ref name="A2017U1"/>。

缺乏彗髮顯示，它必然是起源在任何一顆恆星的[[凍結線_(天文物理)|凍結線]]外側的天體，或是在該恆星系統外的區域足夠長的時間，所有接近表面的物質都凍結成冰了。

據多個望遠鏡對 ʻOumuamua 軌道測量，發現它受到來自太陽、行星與氣體和塵埃釋出導致的作用力，因此屬於彗星成分。<ref>{{Cite web|url=https://www.solidot.org/story?sid=57021|title=星际天体 'Oumuamua 是一颗彗星|accessdate=2018-07-23|author=|date=|work=www.solidot.org|publisher=|archive-url=https://web.archive.org/web/20180723064243/https://www.solidot.org/story?sid=57021|archive-date=2018-07-23|dead-url=no}}</ref><ref>{{Cite news|url=https://www.nytimes.com/2018/06/27/science/oumuamua-comet-asteroid.html|title=An Alien Visitor Turns Out to Be a Comet|accessdate=2018-07-23|language=en|archive-url=https://web.archive.org/web/20180722063833/https://www.nytimes.com/2018/06/27/science/oumuamua-comet-asteroid.html|archive-date=2018-07-22|dead-url=no}}</ref>

也有科学家相信，ʻOumuamua是利用[[光帆|光帆]]推进的外星飞船<ref>{{cite journal |title=Could Solar Radiation Pressure Explain 'Oumuamua's Peculiar Acceleration? |date=2018-10-26 |url=https://arxiv.org/abs/1810.11490 |language=en |journal= |author= |access-date=2019-02-01 |archive-url=https://web.archive.org/web/20190129123038/https://arxiv.org/abs/1810.11490 |archive-date=2019-01-29 |dead-url=no }}</ref>。
