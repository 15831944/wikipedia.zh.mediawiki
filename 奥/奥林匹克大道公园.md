'''奥林匹克大道公园'''，位于[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[北戴河区|北戴河区]]滨海大道，是中国第二个以[[奥林匹克|奥林匹克]]为主题的公园。

==简介==
秦皇岛是2008年[[第二十九届夏季奥林匹克运动会|第二十九届奥林匹克运动会]]协办城市之一。2002年4月，经[[国家体育总局|国家体育总局]]批准，并且征得第二十九届奥林匹克运动会组织委员会同意，建设北戴河奥林匹克大道公园。2005年5月1日建成并对社会免费开放。2005年被国家体育总局评为“全国优秀体育公园”，这是河北省唯一获此荣誉的体育公园<ref>{{Cite web |url =  http://news.sina.com.cn/c/2005-10-11/00167133676s.shtml|title = 奥林匹克大道公园成"国优" |author =  |date =2005-10-11  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

奥林匹克大道公园由秦皇岛市北戴河区人民政府在直通第二十九届奥林匹克运动会[[足球|足球]]比赛秦皇岛赛区场馆[[秦皇岛奥林匹克体育中心|秦皇岛奥林匹克体育中心]]的滨海大道上投资兴建，成为北京[[奥林匹克公园_(北京)|奥林匹克公园]]之外中国第二个以奥林匹克为主题的公园。整个公园占地面积17.33公顷，以“科技奥运、人文奥运、绿色奥运”作为设计理念。是完全开放式的免费公园<ref name=cri/>。主要景观及设施有：
[[File:秦皇岛奥林匹克公园_-_panoramio.jpg|thumb]]
*音乐喷泉广场：位于公园主入口。广场中央立有一座巨型[[和平鸽|和平鸽]]主体雕塑，高25米，[[不锈钢|不锈钢]]材质<ref name=cri/>。
*单体雕塑：公园内散落有63件奥运项目的单体雕塑，分别表现[[冲浪|冲浪]]、[[跳水|跳水]]、[[游泳|游泳]]、[[赛艇|赛艇]]、[[艺术体操|艺术体操]]、[[冰舞|冰舞]]、[[滑雪|滑雪]]、[[摔跤|摔跤]]、[[武术|武术]]、[[拳击|拳击]]、[[举重|举重]]、[[田径|田径]]、[[柔道|柔道]]、[[跆拳道|跆拳道]]、[[乒乓球|乒乓球]]、[[羽毛球|羽毛球]]、[[篮球|篮球]]、[[足球|足球]]、[[曲棍球|曲棍球]]等体育运动项目。这些雕塑均是向中国雕塑艺术家征集创作<ref name=cri/>。
*中国奥运冠军手足印和签名纪念柱：公园内有条甬道，甬道两侧立有许多铜质雕塑纪念柱，造型为翻开的书页，上面印有45位中国奥运冠军的手印、足印和签名<ref name=cri/>。
*奥运冠军林：离这条甬道不远，截至2008年已有12位来过北戴河的中国奥运冠军在此种下12株[[银杏|银杏]]。
*奥林匹克浮雕墙：长312米，由254块[[花岗岩|花岗岩]]雕刻而成。分为三部分：古代奥运、现代奥运、中国奥运。全墙由浮雕、圆雕、线刻、石刻文字以及一组铜雕五环组成<ref name=cri/>。
*高标准[[轮滑|轮滑]]体育运动场<ref name=cri/>
*体育健身设施<ref name=cri>{{Cite web |url = http://news.cri.cn/gb/23204/2008/07/25/3105s2162611.htm |title = 感受奥林匹克文化：北戴河奥林匹克大道公园 |author =  |date = 2008-07-25  |publisher = 国际在线 |language =  |accessdate =  |quote =  }}</ref>
