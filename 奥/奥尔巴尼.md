'''奥尔巴尼'''（{{lang-en|'''Albany'''}}）是[[美国|美国]][[纽约州|纽约州]]首府，也是[[纽约州|纽约州]][[奥尔巴尼县_(纽约州)|奥尔巴尼县]]县治所在地。1686年取得特許狀，1797年成為州府。奥尔巴尼位於[[哈德遜河|哈德遜河]]畔，距[[纽约|纽约]]、[[波士顿|波士顿]]的路程几乎成一个等边三角形，乘巴士旅行的时间约2個多小时左右。

== 历史 ==
早於1614年該市已經有歐洲移民聚居之記載，是美國立國前[[十三殖民地|十三個殖民地]]之一；至1686年該市正式成立；1797年成為[[紐約州|紐約州]]的首府。

最先在阿伯尼聚居的是荷蘭商人，主要是從事皮毛貿易。1664年英國人接管了此城，正式稱作「Albany」。

阿伯尼是全美國最早有公共水管和下水道的城市之一。同時它也是世界最早建設天然氣管和電力輸送網的城市。這些種種建設皆為阿伯尼帶來了大量的工業。當時阿伯尼的出口商品主要是[[啤酒|啤酒]]、[[木材|木材]]、[[印刷|印刷]]品和[[鋼鐵|鋼鐵]]等等。1908年又建設成世界第一個民用[[機場|機場]]（{{link-en|奧巴尼國際機場|Albany International Airport}}）。

在1960年代，市政府乃大規模地進行資助，推動大型建設之進行。其中的代表是{{link-en|帝國廣場 (奧巴尼)|Empire State Plaza|帝國廣場}}（Empire State Plaza），此一容納數個紐約州政府機關的複合設施，總投資達20億[[美元|美元]]，結果其建設所引起的政府債務，直到2004年才還清。

阿伯尼分別於1991年及2009年獲得全美城市獎（[[All-America_City_Award|All-America City Award]]）殊榮。

==地理==
阿伯尼是位處[[哈得遜河|哈得遜河]]上游河畔，屬山谷地區。城市最高位置為海拔99米，但最低則只為0.61米。

市內有多個湖泊，同時有一大片的[[松樹|松樹]]群。

== 人口 ==
自建市以來阿伯尼的居民都是以白人為主。由於歷史原因，這些[[白人|白人]]都是以[[荷蘭|荷蘭]]和[[英國|英國]]人居多。自二次大戰後阿伯尼的黑人所佔比例正不斷增大，由1950年的3%升至1960年的6%；1970年有12%，而2010年則為30.8%。

按2010年全美人口普查，市中心人口為97,856人，阿伯尼都會區人口為1,168,485人，為紐約第四大都會區、全美第52大都會區，僅次於[[紐約市|紐約市]](全美第一大都會區)、[[水牛城|水牛城]](全美第50大都會區)及[[羅徹斯特|羅徹斯特]]都會區(全美第51大都會區)。居民以白色人種為主（佔57％）；約有30.8%為黑人或非洲裔，而0.3%是美洲原住民或[[阿拉斯加|阿拉斯加]]人；至於亞裔人則為5%。

而根據2000年人口普查資料，阿伯尼人口中，有18.1%的人祖先是[[愛爾蘭|愛爾蘭]]人，12.4%為意大利人後裔，德裔則為10.4%，至於英國則僅為5.2%。

== 经济 ==
阿伯尼在二次大戰前是一個工業城市。當地的啤酒釀造、印刷、鋼鐵和木材均曾享譽一時。但自從二次大戰後，製造業衰微，這些產業都業已遷離至其他地方。

由於擁有州首府的地位，阿伯尼的就業崗位一般與政府職位有關。這些工作崗位包括了醫療和教育。整個城市的人口，超過25%是公務員。
