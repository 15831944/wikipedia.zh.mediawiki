'''奥莉加·亚历山德罗芙娜·拉德任斯卡娅'''（{{lang-ru|'''Óльга Алекса́ндровна Лады́женская'''|link=no|p=ˈolʲɡə ɐlʲɪˈksandrəvnə ɫɐˈdɨʐɨnskəɪ̯ə|a=Ru-Olga Aleksandrovna Ladyzhenskaya.wav}}，{{bd|1922年|3月7日|2004年|1月12日}}），俄罗斯数学家。她主要对于[[偏微分方程|偏微分方程]]（特别是[[希尔伯特第十九问题|希尔伯特第十九问题]]）与[[流体力学|流体力学]]有着重大贡献<ref name="Bolibruch">{{cite book |author1=Bolibruch, A. A. |author2=Osipov, Yu. S. |author3=Sinai, Ya. G. |title=Mathematical Events of the Twentieth Century |publisher=Springer-Verlag |location=Beilin |isbn=978-3-540-23235-3 |pages=VIII+545|language=en}}</ref>。奥莉加给出了[[纳维-斯托克斯方程|NS方程]][[有限差分法|有限差分法]]收敛的严格证明，也证明了二维NS方程的整体适定性<ref>{{Citation| last = Ladyzhenskaya| first = O.A.| title = The Mathematical Theory of Viscous Incompressible Flow| place = New York; London; Paris; Montreux; Tokyo; Melbourne| publisher = Gordon and Breach| series = Mathematics and Its Applications| volume = 2| origyear = 1963| year = 1969| edition = Revised Second| pages = xviii+224| mr = 0254401| zbl = 0184.52603}}</ref>。她是{{link-en|伊万·彼得罗夫斯基|Ivan Petrovsky}}的学生<ref name="agnesscott">{{cite web |title=Olga Alexandrovna Ladyzhenskaya |url=https://www.agnesscott.edu/lriddle/women/ladyzhen.htm|language=en}}</ref>，也是{{link-en|冯·诺伊曼讲座奖|John von Neumann Lecture}}1998年的获奖者和[[罗蒙诺索夫金质奖章|罗蒙诺索夫金质奖章]]2002年的获奖者。奥莉加一生留下超过250篇论文，其中有6篇专论。

== 生平 ==
奥莉加在[[科洛格里夫|科洛格里夫]]出生并成长，是一位数学教师的女儿，从小对数学耳濡目染。1937年，她的父亲被苏联的[[内务人民委员部|内务人民委员部]]加以“人民的敌人”的罪名而逮捕、处决。1939年，奥莉加高中毕业，但由于其父亲的罪名，无法通过政治审查进入[[列宁格勒国立大学|列宁格勒国立大学]]（今[[圣彼得堡大学|圣彼得堡大学]]）。<ref name="encyclopedia.com">{{cite web|url=https://www.encyclopedia.com/science/dictionaries-thesauruses-pictures-and-press-releases/ladyzhenskaya-olga-alexandrovna|title=Ladyzhenskaya, Olga Alexandrovna|website=www.encyclopedia.com|access-date=7 March 2019|language=en}}</ref>

1941年6月[[苏德战争|德国入侵苏联]]后，奥莉加进入了一所教育机构，在科洛格里夫任教。1943年，奥莉加进入[[莫斯科大学|莫斯科大学]]得以继续学习，并于1947年毕业。大学毕业后，她移居[[列宁格勒|列宁格勒]]，并迅速与她的学生安德烈·基谢廖夫结婚。1950年，她开始于[[列宁格勒国立大学|列宁格勒国立大学]]任教。1953年，她发表了博士学位论文。1954年，她加入[[斯捷克洛夫数学研究所|斯捷克洛夫数学研究所]]数理实验室并于1961年成为该实验室的负责人。<ref name="encyclopedia.com" />1959年，奥莉加加入圣彼得堡数学协会，并最终于1991年成为该协会的会长。<ref name="google">{{cite web |author1=Google |title=奥尔加·拉德任斯卡娅诞辰 97 周年 |url=https://www.google.com/doodles/olga-ladyzhenskayas-97th-birthday |website=https://www.google.com/ |accessdate=2019-03-08|language=en}}</ref>

在发表了她的理论之后，奥莉加继续她的教学与研究工作。苏联解体后，她选择留在俄罗斯。2004年1月24日，奥莉加去世，留下超过250篇论文。<ref name="standard">{{cite web|url=https://www.standard.co.uk/lifestyle/google-doodle-olga-ladyzhenskaya-russian-mathematician-a4084616.html|title=Google Doodle: Who was Russian mathematician Olga Ladyzhenskay?|date=6 March 2019|website=Evening Standard|language=en|access-date=2019-03-08}}</ref>

== 学术成就 ==
奥莉加对于[[偏微分方程|偏微分方程]]（特别是[[希尔伯特第十九问题|希尔伯特第十九问题]]）与[[流体力学|流体力学]]有着重大贡献<ref name="Bolibruch" />。她给出了[[纳维-斯托克斯方程|纳维-斯托克斯方程]][[有限差分法|有限差分法]]收敛的严格证明。她是{{link-en|伊万·彼得罗夫斯基|Ivan Petrovsky}}的学生<ref name="agnesscott" />，也是{{link-en|冯·诺伊曼讲座奖|John von Neumann Lecture}}1998年的获奖者和[[罗蒙诺索夫金质奖章|罗蒙诺索夫金质奖章]]2002年的获奖者。

奥莉加曾被提名1958年的[[菲尔兹奖|菲尔兹奖]]<ref>{{cite journal|last1=Barany|first1=Michael|title=The Fields Medal should return to its roots|journal=Nature|date=2018|volume=553|pages=271–273|doi=10.1038/d41586-018-00513-8|language=en}}</ref>，但最终没有获奖。

== 社会认可 ==
奥莉加是[[苏联科学院|苏联科学院]]院士（1990年）、[[俄罗斯科学院|俄罗斯科学院]]院士（1991年）<ref>{{cite web |author1=俄罗斯科学院 |title=Ладыженская О.А. - Общая информация |url=http://www.ras.ru/win/db/show_per.asp?P=id-210.oi-1.vi-.fi-.uk-10 |website=http://www.ras.ru/ |accessdate=2019-03-09 |language=ru}}</ref>。

在2019年3月7日，奥莉加的97周年诞辰，搜索引擎[[谷歌|谷歌]]推出了[[Google涂鸦|Google涂鸦]]纪念奥莉加。谷歌的评论写道：“今天的涂鸦纪念俄罗斯数学家奥莉加·拉德任斯卡娅，她克服了个人的悲剧和障碍，成为了她那个时代最有影响力的思想家之一”。<ref name="google" /><ref name="standard" />
