在美國政府公佈[[海豹部隊|海豹部隊]]擊斃了[[本·拉登|本·拉登]]的消息後，部分個人和媒體卻認爲'''奥萨马·本·拉登之死是個陰謀'''<ref name=WNN>{{cite web |url = http://news.qq.com/a/20110510/000895.htm |title=拉登之死引爆阴谋论 专家称源于美方信息不透明 |accessdate=2014-08-12 |language=zh-hans |publisher=世界新聞報}}</ref>。2013年4月2日，根據一家專門分析政治動向的美國機構 — [[公共政策民調基金會|公共政策民調基金會]]所做的調查顯示，有6%的美國民眾投票認為，奧薩瑪·賓·拉登還活著<ref>{{cite web|language = en|url=http://www.publicpolicypolling.com/main/2013/04/conspiracy-theory-poll-results-.html |title=Conspiracy Theory Poll Results|publisher=[[公共政策民調基金會|公共政策民調基金會]] |date=2013-04-02|accessdate=2017-04-03}}</ref>。

==背景==
{{Main|奥萨马·本·拉登之死}}

美國東部時間2011年5月1日，美國的海豹部隊在一次代號爲「海神之矛」的軍事行動中將隱居在巴基斯坦境內的[[本·拉登|本·拉登]]擊斃，引發了各界的強烈反響。其中，美國的部分做法（如不公佈本拉登被擊斃的照片等）引發了部分人士的質疑，他們認爲奥萨马·本·拉登的死是美國一手策劃的陰謀。<ref name=DW>{{cite web |url=http://global.dwnews.com/big5/news/2011-05-05/57688097.html |title=白宮改口拉登未反抗 未发尸照受質疑 |publisher=多維新聞 |accessdate=2014-08-13 |archive-url=https://web.archive.org/web/20140814074356/http://global.dwnews.com/big5/news/2011-05-05/57688097.html |archive-date=2014-08-14 |dead-url=yes }}</ref>

==論點及論據==
本拉登死後，白宮以「（公佈照片）對消除懷疑沒有什麼效果，而且照片內容『過於觸目驚心』」的理由未公佈本拉登被擊斃的照片，部分人士對此提出了質疑。比如，一名塔利班的指揮官稱「如果美國真殺死了本拉登，爲什麼不向我們出示證據？」<ref name=DW/><ref name=WNN>{{cite web |url = http://news.qq.com/a/20110510/000895.htm |title=拉登之死引爆阴谋论 专家称源于美方信息不透明 |accessdate=2014-08-12 |language=zh-hans |publisher=世界新聞報}}</ref>。另外，美方匆匆將本拉登的屍體[[海葬|海葬]]這一行爲，也成爲了懷疑的對象。也有人對美國直升機突然進入[[巴基斯坦|巴基斯坦]]領空而巴方卻無任何反應的合理性提出了質疑<ref name=AD/>。還有人懷疑說本拉登的死是爲了轉移人們的視線，使得人們不再關注北約的飛機炸死[[卡扎菲|卡扎菲]]的兒子賽義德和另外他的另外三名孫子這件事<ref>{{Cite web|url=http://www.21ccom.net/articles/qqsw/qqgc/article_2011050234651.html|title=姚新勇：拉登之死与美国阴谋|publisher=[[共識網|共識網]]|date=2011-05-03|accessdate=2014-08-12|language=zh-hans|archiveurl=https://web.archive.org/web/20140812204347/http://www.21ccom.net/articles/qqsw/qqgc/article_2011050234651.html|archivedate=2014-08-12|deadurl=yes}}</ref>。

==美方的回應==
針對「海豹部隊的攻擊行動是捏造的」的這一質疑，美方公佈了行動的錄音帶<ref name=AD>{{cite web|publisher=蘋果日報 |url = http://www.appledaily.com.tw/appledaily/article/headline/20110504/33362944/|title=海葬不見屍 賓拉登 生死遭疑 |accessdate=2014-08-13}}</ref>
。而時任[[美國總統|美國總統]][[奧巴馬|奧巴馬]]對於不公佈照片的解釋則是「（照片）可能引發暴力或者被用作宣傳工具。」，並說已經通過DNA確定了死者確爲本·拉登。時任[[白宮發言人|白宮發言人]][[卡尼|卡尼]]說<ref name=DW/>：「基地組織的成員們也都相信本拉登死了。」對海葬的解釋則是「這是爲了尊重穆斯林要在死後24小時下葬的習俗」。另外，美方還曾已對海葬全過程進行了錄象。而關於巴方並未干涉美方直升機的突然入境一事，美方則稱是巴方未來得及反應所致。<ref name=AD/>。
