'''奥塞梯人'''主要居住在[[北高加索|北高加索]]的[[奥塞梯|奥塞梯]]地区，该地区被分为[[俄罗斯|俄罗斯]]的[[北奥塞梯-阿兰共和国|北奥塞梯-阿兰共和国]]、[[格鲁吉亚|格鲁吉亚]]和已事實獨立的[[南奧塞梯|南奧塞梯]]共和國。奥塞梯人使用[[奥塞梯语|奥塞梯语]]（属于[[印欧语系|印欧语系]][[印度-伊朗语族|印度-伊朗语族]]），多数人信仰[[东正教|东正教]]，少部分人信仰[[伊斯兰教|伊斯兰教]]。

== 分布 ==
今天，大多数奥塞梯人居住在大高加索的中心部分的两个奥塞梯。很多人散居到[[土耳其|土耳其]]，一些奥塞梯人移居到[[俄罗斯|俄罗斯]]、[[法国|法国]]、[[瑞典|瑞典]]、[[叙利亚|叙利亚]]、[[美国|美国]]、[[加拿大|加拿大]]以及世界的其他国家。

== 族源和歷史 ==
奥塞梯人来源于[[伊朗|伊朗]]部落（[[奄蔡|奄蔡]]），被称为[[阿兰人|阿兰人]]、[[斯基泰人|斯基泰人]]和[[萨尔马提亚人|萨尔马提亚人]]。[[中世纪|中世纪]]时期，在[[格鲁吉亚|格鲁吉亚]]的影响下，皈依[[基督教|基督教]]。元朝時奧塞梯人屬於[[色目人|色目人]]，當時被稱為[[阿蘇特|阿蘇特]]。在[[金帐汗国|金帐汗国]]统治时期，奥塞梯人被迫离开他们原来的家园（今[[俄罗斯|俄罗斯]]的[[顿河|顿河]]南部），他们越过[[高加索山脉|高加索山脉]]迁移到北高加索，在这里，他们形成了三个具有独特风格的地区：
* 西部的迪戈尔受到了相邻的，信仰[[伊斯兰教|伊斯兰教]]的[[卡巴爾達人|卡巴爾達人]]的影响。
* 南部的Tualläg，位于[[格鲁吉亚|格鲁吉亚]]中部区域的[[什达－卡尔特里|什达－卡尔特里]]。1924年成为南奥塞梯。
* 北部的伊隆于1767年为[[沙皇|沙皇]]统治，成为现在的[[北奥塞梯-阿兰共和国|北奥塞梯-阿兰共和国]]。

20世纪，格鲁吉亚的奥塞梯人在[[苏联|苏联]]统治下和1990年代的[[格鲁吉亚奥塞梯冲突|格鲁吉亚奥塞梯冲突]]中遭受痛苦。

== 宗教信仰 ==
大多数来自北奥塞梯和南奥塞梯的奥塞梯人，信仰[[俄罗斯东正教|俄罗斯东正教]]。少数人信仰[[伊斯兰教|伊斯兰教]]（主要是[[逊尼派|逊尼派]]，[[苏菲派|苏菲派]]也很多）。
