'''奥古斯特三世'''，有时被以绰号称为'''撒克逊'''或'''臃肿者'''（{{lang-de|August III. von Polen}}; {{lang-pl|August III. Sas, August III. Gruby}}），在1733年到1763年间，为[[萨克森选帝侯列表|萨克森选王侯]]，称为'''弗里德里希·奥古斯特二世''' ({{lang-de|Kurfürst Friedrich August II.}})，在1734年到1763年间，为[[波兰君主列表|波兰国王]]和[[立陶宛大公国|立陶宛大公]]，綽號「肥胖王」。

==传记==
[[Image:King_Augustus_III_of_Poland.jpg|thumb]]
奥古斯特是[[奥古斯特二世_(波兰)|奥古斯特二世]]唯一的婚生子，为萨克森帝国[[选帝侯|选帝侯]]和[[波兰立陶宛联邦|波兰立陶宛联邦]]君主。他被父亲成功培养为一位波兰国王和立陶宛大公，并于1717年转信[[天主教|天主教]]。1719年，他在剛建好的[[茨溫格宮|茨溫格宮]]中，與[[哈布斯堡|哈布斯堡]]皇帝的公主瑪利亞·約瑟法舉行婚禮，當天也是雄偉的[[茨溫格宮|茨溫格宮]]之揭幕典禮。

在他父亲死后，继承了萨克森并[[波蘭選王制|当选]][[波兰君主列表|波兰国王]]，并在[[俄罗斯帝国|俄罗斯]]和[[奥地利|奥地利]]的支持下，发动了[[波兰王位继承战争|波兰王位继承战争]]（1733年－1738年）。

作为国王，奥古斯特三世对波兰立陶宛联邦的事务不感兴趣，而对爱好例如打猎，歌剧和集邮更感兴趣（参见[[老主人图片走廊|老主人图片走廊]]）。在他30年的统治期间，他在波兰上只花了不到3年，当[[恰尔托雷斯基王室|恰尔托雷斯基王室]]和[[波托茨基家族|波托茨基家族]]在冲突中瘫痪了[[波兰共和国众议院|众议院]]（[[自由否决权|自由否决权]]）时， 促使国内政治局面进一步混乱，并削弱了[[波兰立陶宛联邦|波兰立陶宛联邦]]的国力。奥古斯特三世将他的大部分权利交给了变为波兰准[[独裁者|独裁者]]的[[海因里希·冯·布鲁尔|海因里希·冯·布鲁尔]]。 

在奥古斯特三世统治的30年里，波兰经历了[[七年战争|七年战争]]（1754年及1756年到1763年），邻国[[普鲁士|普鲁士]]，[[奥地利|奥地利]]和[[俄罗斯帝国|俄罗斯帝国]]提出了[[瓜分波兰|瓜分波兰]]的计划的其中之一。

他的大儿子，[[弗里德里希·克里斯蒂安_(萨克森选侯)|弗里德里希·克里斯蒂安]]最终成功继承了[[萨克森统治者列表|萨克森选侯]]，但不是波兰国王。新任波兰国王为[[斯坦尼斯瓦夫·奥古斯特·波尼亚托夫斯基|斯坦尼斯瓦夫·奥古斯特·波尼亚托夫斯基]]，他于1764年9月7日由[[恰尔托雷斯基家族|恰尔托雷斯基家族]]发动的一场获俄罗斯支持的[[政变|政变]]成功后当选[[波兰立陶宛联邦|波兰立陶宛联邦]]国王。

==婚姻和子女==

在1719年于[[德累斯顿|德累斯顿]]，奥古斯特与皇女，[[约瑟夫一世_(神圣罗马帝国)|神圣罗马帝国皇帝约瑟夫一世]] 之女[[玛利亚·约瑟法_(奥地利女大公)|玛利亚·约瑟法]]结婚，他们有15个子女：

# 弗雷德里克·奥古斯特·弗兰兹·泽维尔（1720年11月18日于德累斯顿-1721年1月22日于德累斯顿）。
# 约瑟夫·奥古斯特·威廉·弗雷德里克·弗兰兹·泽维尔·约翰·内波穆克（1721年10月24日于皮尔尼兹-1728年3月14日）。
# [[弗里德里希·克里斯蒂安_(萨克森选侯)|弗雷德里克·克里斯蒂安·利奥波德·约翰·乔治·弗兰兹·泽维尔]]（1722年9月5日于德累斯顿-1763年12月17日），继任萨克森选帝侯。
# 死产的女儿（1723年6月23日于德累斯顿-1723年6月23日于德累斯顿）。
# [[玛利亚·阿玛利亚_(萨克森)|玛利亚·阿玛利亚·克里斯蒂娜·弗兰西斯卡·弗洛拉·瓦尔布加]]（1724年11月24日于德累斯顿-1760年9月27日于布恩雷蒂罗）；在1738年6月19日与那不勒斯国王卡洛四世，后来的[[卡洛斯三世|西班牙国王卡洛斯三世]]結婚。
# 玛利亚·玛格丽特·弗兰西斯卡·泽维利亚（1727年9月13日于德累斯顿-1734年2月1日于德累斯顿）。
# [[玛利亚·安娜·索菲亚_(萨克森)|玛利亚·安娜·索菲亚·萨宾娜·安吉拉·弗兰西斯卡·泽维利亚]]（1728年8月29日于德累斯顿-1797年2月17日于慕尼黑），于1747年8月9日与[[马克西米连三世·约瑟夫_(巴伐利亚选帝侯)|马克西米连三世·约瑟夫]]结婚。
# [[弗兰兹·泽维尔_(萨克森)|弗兰兹·泽维尔·阿尔伯特·奥古斯特·路德维希·本诺]]（1730年8月25日于德累斯顿-1806年6月21日于德累斯顿），萨克森摄政者（1763年－1768年）。
# [[玛利亚·约瑟法_(萨克森)_(1731年-1767年)|玛利亚·约瑟法·卡洛琳那·埃莱奥诺蕾·弗兰西斯卡·泽维利亚]]（1731年11月4日于德累斯顿-1767年3月13日于凡尔赛）；于1747年2月9日与[[路易十五|路易十五]]之子[[路易斯_(法国王子)|法国王子路易斯]]结婚（她也是[[路易十六|路易十六]]之母）。
