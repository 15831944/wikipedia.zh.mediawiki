'''奥列格·伊万诺维奇·斯克里波齐卡'''（{{lang-ru|Оле́г Ива́нович Скри́почка}}，{{bd|1969年|12月24日|}}）是[[俄罗斯|俄罗斯]][[宇航员|宇航员]]。

==早年==
出生于军人家庭，幼年在[[涅溫諾梅斯克|涅溫諾梅斯克]]和[[堪察加彼得巴甫洛夫斯克|堪察加彼得巴甫洛夫斯克]]接受教育，后在[[扎波罗热|扎波罗热]]完成中学学业，是扎波罗热青年宇航员小队成员<ref name="fakty_2010">{{cite book|author=Левченко И.|title=«Олег «заболел» космосом, когда школьником записался в единственный в Советском Союзе экспериментальный отряд космонавтов при городской станции юных техников»|url=http://fakty.ua/120204-oleg-zabolel-kosmosom-v-zaporozhe-kogda-shkolnikom-zapisalsya-v-edinstvennyj-v-sovetskom-soyuze-eksperimentalnyj-otryad-kosmonavtov-pri-gorodskoj-stancii-yunyh-tehnikov-|date=2010-10-07}}</ref>，1993年毕业于[[鲍曼莫斯科国立技术大学|鲍曼莫斯科国立技术大学]]航空学院<ref>{{cite web|title=Космос открывается смелым и настойчивым|url=http://www.bmstu.ru/abitur/news/news.html?newsid=4039|date=2017-03-17}}</ref><ref>{{cite web|author=Лапин А. В.|title=Список выпускников и преподавателей ИМТУ, МММИ, МВТУ, МГТУ с 1865 до 2012 года|url=http://people.bmstu.ru/abcdef/sk.htm}}</ref>。1987年至1991年在[[科罗廖夫能源火箭航天集团|科罗廖夫能源火箭航天集团]]担任测试技师，1990年至1993担任专业技术人员，1993年8月至1997年8月在科罗廖夫航天集团项目局担任技术工程师，负责地面运输车辆的开发<ref name="NASA Bio">{{Cite web |url=http://www.jsc.nasa.gov/Bios/htmlbios/skripochka-oi.html |title=Cosmonaut Bio: Oleg Ivanovich Skripochka |author=[[NASA|NASA]] |date=2010 |accessdate=2010-10-17}}</ref>。
==宇航员生涯==
体检合格后于1997年10月14日成为宇航员候选人<ref name="NASA Bio"/>。
===远征25/26===
[[莫斯科时间|莫斯科时间]]2010年15日17时55分（北京时间22时55分），他和[[费奥多尔·尤尔奇欣|费奥多尔·尤尔奇欣]]打开国际空间站对接舱舱门，开始了约6个小时的[[舱外活动|舱外活动]]。他的主要任务是在[[星辰号服务舱|星辰号服务舱]]外侧安装一个多功能工作站<ref>{{cite web |author=刘恺 |title=国际空间站俄罗斯宇航员出舱太空行走|url=http://news.sciencenet.cn/htmlnews/2010/11/240235.shtm|publisher=科学时报 |accessdate=2019-04-10 |language=zh |date=2010-11-16}}</ref>。

莫斯科时间2011年1月21日17时29分（[[北京时间|北京时间]]22时29分），他与同事[[德米特里·孔德拉季耶夫|德米特里·孔德拉季耶夫]]打开[[国际空间站|国际空间站]]的[[码头号对接舱|码头号对接舱]]舱门，开始了2011年的首次舱外作业。他们将一个高速通讯系统的壳体安装到[[星辰号服务舱|星辰号服务舱]]表面，同时也拆卸了服务舱上的一件仪器。他们还在[[曙光號功能貨艙|曙光號功能貨艙]]上安装了一个[[摄像头|摄像头]]<ref>{{cite web |author=耿锐斌 |title=国际空间站两名俄罗斯宇航员完成太空作业 |url=http://news.sciencenet.cn/sbhtmlnews/2011/1/241047.html?id=241047 |publisher=科学时报 |accessdate=2019-04-10 |language=zh |date=2011-01-24}}</ref>。

莫斯科时间2011年2月16日，他与同事孔德拉季耶夫一起完成了自己的第三次[[舱外活动|舱外活动]]任务<ref>{{cite web |url=http://forum.nasaspaceflight.com/index.php?topic=61.1080 |title=Schedule of ISS flight events |publisher=NASAspaceflight.com |date=2010-07-17 |accessdate=2010-07-24}}</ref>。
