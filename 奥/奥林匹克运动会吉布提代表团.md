'''[[吉布提|吉布提]]'''迄今已參與八屆[[夏季奥林匹克运动会|夏季奥林匹克运动会]]賽事，且未曾參加[[冬季奥林匹克运动会|冬季奥林匹克运动会]]。該國於[[1984年夏季奥林匹克运动会|1984年洛杉矶奧運]]首次參賽，並派出了三名運動員，但並未取得任何獎牌<ref name="Sports Reference - Countries - Djibouti">{{cite web | url=http://www.sports-reference.com/olympics/countries/DJI/ | title=Sports Reference – Countries – Djibouti | publisher={{tsl|en|Sports Reference||Sports Reference}} | accessdate=2017-01-02}}</ref><ref name="Sports Reference - Los Angeles 1984 - Djibouti">{{cite web | url=http://www.sports-reference.com/olympics/countries/DJI/summer/1984/ | title=Sports Reference – Los Angeles 1984 – Djibouti | publisher=Sports Reference | accessdate=2017-01-02}}</ref>。參與選手數量最多的一次為[[1992年夏季奥林匹克运动会|1992年]][[巴塞隆納|巴塞隆納]]奧運，當屆共派出了八個選手<ref name="Sports Reference - Countries - Djibouti" /><ref name="Sports Reference - Barcelona 1992 - Djibouti">{{cite web | url=http://www.sports-reference.com/olympics/countries/DJI/summer/1992/ | title=Sports Reference – Barcelona 1992 – Djibouti | publisher=Sports Reference | accessdate=2017-01-02}}</ref>。1988年，該國選手{{tsl|en|Hussein Ahmed Salah|Hussein Ahmed Salah|侯賽因·阿罕默德·薩拉赫}}於奪得{{tsl|en|Athletics at the 1988 Summer Olympics – Men's marathon|1988年夏季奧林匹克運動會馬拉松比賽|馬拉松}}銅牌，為該國迄今的唯一一面獎牌<ref name="Sports Reference - Seoul 1998 - Djibouti - Athletics">{{cite web | url=http://www.sports-reference.com/olympics/countries/DJI/summer/1988/ATH/ | title=Sports Reference – Seoul 1998 – Djibouti – Athletics | publisher=Sports Reference | accessdate=2017-01-02}}</ref>。
