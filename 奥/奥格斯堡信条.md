《'''奥格斯堡信条'''》，（英文：'''Augsburg Confession'''， 拉丁文：'''Augustan Confession''' 或 Confessio '''Augustana'''），是[[信義宗|信義宗]]（[[信義宗|路德教会]]） 主要承认的信仰教條。和[[宗教改革|宗教改革]]最重要的文件之一。 《奥斯堡信条》使用德文和[[拉丁语|拉丁]]文並列，并由一些[[神圣罗马帝国|德国]]统治階層和自由市民，於1530年6月25日在[[奥格斯堡|奥格斯堡]]議會提出。

[[神聖羅馬皇帝|神圣罗马帝国皇帝]]曾要求在王子[[查理五世_(神圣罗马帝国)|查尔斯]]五世在德国的自由领土解释他们的宗教信念，以试图恢复对[[神圣罗马帝国|神圣罗马帝国]]宗教和政治的团结和支持，以對抗[[维也纳之围|土耳其入侵]]的。《奥斯堡信条》被收錄於[[信義宗|信義宗]]重要教典《[[協同書|協同書]]》的第四号文件。

== 背景 ==

[[墨蘭頓|菲利浦·墨蘭頓]]（1497-1560），[[馬丁路德|馬丁路德]]和猶圖斯·約拿士在1529年的《施瓦巴信條》中已經起草了對他們神學觀點的陳述，於1530年1月21日，皇帝[[查理五世|查爾斯五世]]從[[博洛尼亞|博洛尼亞]]發出信件，邀請帝國議會於4月8日在奧格斯堡會面，以討論和決定各種重要問題。 雖然邀請函是用非常和平的語言表達的，但一些[[新教|新教]]徒卻懷疑它的目的。 黑森的領主菲利普一世對參加議會猶豫不決，但是薩克森的選舉人約翰於3月11日收到了令狀後，3月14日便與馬丁·路德，猶圖斯·約拿士，約翰內斯·布根哈根和菲利普·莫蘭頓在[[托尔高|托爾高]]會面，他曾在那里的議會於[[神聖羅馬帝國|神聖羅馬帝國]]皇帝面前介紹[[路德派|路德派]]信仰的大綱。{{sfn|Kolde|1914}}

第一次官方出版（Editio princeps）由[[馬丁路德大學|維滕貝格大學]]（[[信義宗|信義宗]]慣用譯名：威登堡大學）教授，[[馬丁路德|馬丁路德]]的親密同事亦即朋友[[墨蘭頓|菲利浦·墨蘭頓]]編輯。

== 内容 ==

=== 28条信條 ===
奧斯堡信條由路德宗王子和奧斯堡議會中的“自由城市”代表提交的28篇文章組成，這些文章提出了路德教會在積極（論文）和否定（反對）陳述中所認為、教導和承認的內容。 這些論文是21條主要的信仰條款，描述了路德教會所持有的基督教信仰的規範性原則；加上7個對立陳述，描述了他們所看到的對羅馬教會中存在的基督教信仰的濫用。

==== 主要信仰條文(论文) ====

!标题
!描述
|-
|1
|[[三位一體|上帝]]
|基督徒相信的三位一体上帝和拒绝关于上帝本性的[[非三位一體派|其他解释]]的。
|-
|2
|[[原罪|原罪]]
|路德会认为，人的本质是罪恶的，描述为不敬畏上帝、不信靠上帝、以及充滿慾望的。 此罪需通过[[信義宗|洗礼]]和聖靈方可得赦。
|-
|3
|上帝之子
|路德会相信道成肉身，也就是說，耶穌同時是完全的人與完全的神。由於此神聖的聯合的特性，只有耶穌基督才能實現人與上帝的和解。
|-
|4
|[[因信稱義|因信稱義]]
|人不能通過自己的能力在上帝面前稱義; 我們完全依賴耶穌基督與上帝和好。 （這通常被描述為“路德教會屹立不倒”的一篇文章。）<br />
|-
|5
|教會聖職
|路德宗相信，為了確保耶穌基督的福音在全世界傳揚，基督已經建立了聖事的職權。
|-
|6
|新順服
|路德宗相信基督徒的善行是信仰和救恩的結果，而不是為他們付出的代價（例如善功、奉獻）。<br />
|-
|7
|教会
|路德宗相信「一聖基督教會」（參[[使徒信經|使徒信經]]），無論在哪里傳講[[福音|福音]]之真理和聖潔，[[聖禮|聖禮]]都按照福音來管理。<br />
|-
|8
|何謂教会
|儘管教會（以及人之間）可能存在虛偽，但聖經和聖禮總是有效的，因為它們是由基督設立的，無論管理他們的人是否犯罪。<br />
|-
|9
|[[洗禮|洗礼]]
|路德教會相信洗禮是必要的，並且通過洗禮可以領受上帝的恩典。 兒童洗禮亦可使他們領受上帝的恩典。<br />
|-
|10
|聖餐
|路德宗相信基督的聖體和寶血是「真實臨在」於聖禮的餅和酒中，並且譴責那些否定此教導的人。<br />
|-
|11
|認罪
|路德教會相信「私赦」應保留在教會中，儘管信徒不需要列舉他所有的罪行，因為人不可能列舉他應該被寬恕的所有罪行。
|-
|12
|悔改
|悔改分為兩部分：一是懺悔根據律法犯下的罪，二是通過福音提供的信仰。 信徒永遠不能擺脫罪惡，也不能生活在上帝的恩典之外。<br />
|-
|13
|圣礼
|聖禮（洗禮和聖餐）是上帝話語的身體表現和他對我們的承諾。 聖禮不僅僅是物質元素，而是上帝的話語和承諾與他們聯繫在一起。<br />
|-
|14
|教會職分
|路德派只允許那些被“正式呼召”（Rite vocatus）的人公開宣講或管理聖禮。
|-
|15
|教会禮儀
|路德教會認為教堂假日、日曆和節日對於宗教儀式有助益，但刻意遵守日曆和儀式對於救贖並非必需。 人類所設立之傳統（例如紀念、齋戒、葷食等規範的制定）被教導成一種「行善功以『配得』恩惠的方式」，這其實是福音相對立的。<br />
|-
|16
|公民參與
|世俗政府和職業被認為是上帝自然秩序的一部分; 基督徒可以自由地在政府和軍隊服務，並從事世界的商業和職業。 除非這些命令會導致觸犯基督教誡命裡所規範的罪，否則應遵循政府律法。<br />
|-
|17
|[[耶稣再临|基督復臨]]行審判
|路德宗相信基督會回來復活死者並審判世界; 敬虔的人將獲得永恆的喜樂，不敬虔的人將受到“無盡的折磨”。 此條文棄絕其他教派「在亡者前將復活先有一個千禧王國」這類的宣告。<br />
|-
|18
|自由意志
|路德教會相信我們在“屬世正義”（或“理性的事物”）領域擁有自由意志，但我們在“屬靈正義”中沒有自由意志。 換句話說，我們可以自由地選擇和行動，但屬靈事物除外。 屬靈信心不是憑藉人的工作，而是聖靈的工作。<br />
|-
|19
|罪的成因
|路德宗相信罪不是由上帝引起的，而是由“惡人的意志”引起的，背離了上帝。
|-
|20
|善功
|路德宗神學重點為「因信稱義」，但並非是反對善行; 信仰使人自發去行善功，作為我們稱義（或救恩）的標誌，善功並非是救恩的要求條件。有善功而無信心依然不能稱義。<br />
|-
|21
|圣徒崇拜
|路德派認為可以保留聖徒名分，但不是作為代表上帝的救世主或代禱人，而僅是作為我們自己的信仰和生活的榜樣和靈感。<br />
|}

==== 譴責羅馬天主教滥用職權之行为 ====
{| border="1" class="wikitable" 
!信條
!主旨
!簡述
|-
|22
|聖餐兩種原質
|使信徒領受餅和酒兩者，而不僅僅是餅，這是恰當的，符合基督教導和早期教會的實踐。<br />
|-
|23
|牧者的婚姻
|路德教允許他們的神職人員進入婚姻制度，因為早期教會主教可以結婚。上帝保佑婚姻是一種創造秩序，並且因為婚姻和生育是人類性慾的自然出路。
|-
|24
|彌撒
|路德教會保留了彌撒的實踐，但僅作為社區禮拜和領受聖餐的公共聚會。 路德教會拒絕將彌撒作為救贖和世俗（金錢）收益的“工作”的做法。<br />
|-
|25
|認罪
|路德派堅持認罪和赦免的必要性，但拒絕承認懺悔應該引起基督徒內疚或焦慮的觀念。 赦免是為所有的罪而提供的，而不只是在懺悔中可以敘述的罪，因為一個人不可能細述他所有的過犯。<br />
|-
|26
|食物之區別
|人類靠傳統禁食和特殊遵守飲食限制作為獲得上帝恩惠的手段，是與福音相反的。 雖然禁食和其他做法是有用的精神實踐，但它們並不能使人成義，也不能提供救贖。<br />
|-
|27
|修道誓願
|人無法在社群中成為聖潔，也無法與世界其他地方隔離，任何誓言或單獨行動都無法達到完美。
|-
|28
|教會權柄
|賦予祭司或主教的唯一權力是通過聖經提供，傳授，教導和管理聖禮的權力。 在政府或軍隊問題上賦予神職人員的權力只能通過民事手段給予和尊重; 他們不是神聖權利的政府和軍隊的民事統治者。<br />
|}

=== 對音乐的影響 ===
[[孟德爾頌|孟德爾頌]]的第五交響曲（實際上是他的第二交響樂作曲）是為慶祝奧斯堡信條300週年而成，因此獲得了宗教改革交響曲的稱號。 然而，由於作曲家的猶太人起源、或交響樂本身之不適當，而未被採用於慶祝活動。 相反地，愛德華·格雷爾（Eduard Grell）為四男聲量身打造了無伴奏合唱（a cappella）。
