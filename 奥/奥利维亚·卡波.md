'''奥利维亚·卡波'''（{{lang-en|'''Olivia Culpo'''}}，{{bd|1992年|5月8日|||catIdx= K卡波}}）出生于[[美国|美国]][[罗德岛州|罗德岛州]][[克兰斯顿_(罗得岛州)|克兰斯顿]]，为美国女大学生，2012年年度“[[美国小姐|美国小姐]]”称号得主。<ref name="dailynews">[http://www.nydailynews.com/entertainment/usa-2012-rhode-island-olivia-culpo-named-usa-2012-gallery-1.1089406 Miss USA 2012: Miss Rhode Island Olivia Culpo named Miss USA 2012]</ref><ref>[http://www.metro.us/boston/comment/article/1144662--five-things-to-know-about-boston-university-student-olivia-culpo-the-new-miss-usa Five things to know about Boston University student Olivia Culpo, the new Miss USA] {{webarchive|url=https://web.archive.org/web/20120606185056/http://www.metro.us/boston/comment/article/1144662--five-things-to-know-about-boston-university-student-olivia-culpo-the-new-miss-usa |date=2012-06-06 }}</ref>

== 生平 ==
1992年，卡波出生在[[美国|美国]][[罗德岛州|罗德岛州]][[克兰斯顿_(罗得岛州)|克兰斯顿]]，她是彼得和苏珊·卡波的女儿，父母都是[[音乐家|音乐家]]。<ref name="missribio"/>

卡波是罗德岛爱乐青年管弦乐团的成员，她擅长[[大提琴|大提琴]]。<ref name="missribio">{{Cite web |url=http://www.missrhodeislandusa.com/riusa_bio.html |title=About Olivia |access-date=2012-06-05 |archive-url=https://web.archive.org/web/20120130081925/http://www.missrhodeislandusa.com/riusa_bio.html |archive-date=2012-01-30 |dead-url=yes }}</ref>

目前，卡波在就读[[波士顿大学|波士顿大学]]二年级读书。<ref>{{cite web|url= http://pic.news.sohu.com/911196/911317/group-351565.shtml?pvid=tc_news&a=&b=http%3A%2F%2Fi2.itc.cn%2F20120605%2F2bf8_517f05e3_e3ac_406b_5e8d_096eae44e594_1.jpg#0|title= 2012美国小姐出炉|accessdate= 2012年6月5日|publisher= 搜狐网|work= |language= zh-cn|archive-url= https://web.archive.org/web/20120608074804/http://pic.news.sohu.com/911196/911317/group-351565.shtml?pvid=tc_news&a=&b=http%3A%2F%2Fi2.itc.cn%2F20120605%2F2bf8_517f05e3_e3ac_406b_5e8d_096eae44e594_1.jpg|archive-date= 2012年6月8日|dead-url= yes}}</ref>

== 荣誉 ==
2012年，卡波荣获“2012年美国小姐”，马里兰州小姐娜娜·梅里韦瑟（Nana Meriwhether）是亚军，第三名是俄亥俄小姐奥黛丽·博尔特（Audrey Bolte）。<ref>{{cite web |url=http://news.163.com/12/0604/16/835TMG6C00014JB6.html |title=罗德岛佳丽获美国小姐桂冠 将参选环球小姐 |accessdate=2012-06-04 |publisher=网易 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://ent.qq.com/a/20120605/000419.htm |title=2012年美国小姐出炉 20岁大学生奥利维亚摘桂冠 |accessdate=2012年6月5日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

== 選美記錄 ==
=== 羅德島州小姐 ===
卡波於2011年9月18日成為美國羅德島州小姐。
=== 美國小姐 ===
卡波於2012年6月3日代表羅德島州參加第61屆美國小姐，代表羅德島州取得冠軍，成為第61屆美國小姐。
=== 環球小姐 ===
卡波於2012年12月19日代表美國參加第61屆環球小姐比賽，榮獲第１名。<ref>http://www.appledaily.com.tw/realtimenews/article/new/20121220/157672 2012環球小姐　美國佳麗奪冠</ref>
