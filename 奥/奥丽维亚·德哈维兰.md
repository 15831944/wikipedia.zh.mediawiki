'''奥丽维亚·玛丽·德哈维兰'''女爵士{{DBE}}（{{lang-en|Dame '''Olivia Mary de Havilland'''{{DBE}}}}，{{bd|1916年|7月1日||catIdx=De Havilland}}），生于[[日本|日本]][[东京|东京]]，[[英国|英国]][[电影演员|电影演员]]，《[[乱世佳人|乱世佳人]]》的主演之一，两度获得[[奧斯卡最佳女主角奖|奧斯卡最佳女主角奖]]。另一位奥斯卡影后的[[瓊·芳登|瓊·芳登]]是她的亲生妹妹，由此她们是全世界唯一一对都获得过奥斯卡奖表演类奖项的亲生手足，而且双方都非常长寿至少都活了96岁；但令人错愕的是，这份手足情感几十年如一日的无限期冷战僵持，即使95岁之后也依旧几乎完全毫无往来，简直好像永不和解。

== 早期生活 ==
[[File:Olivia_de_Havilland_Publicity_Photo_1938.jpg|thumb]]
德哈维兰是[[英国|英国]]专利[[律师|律师]]沃特·德哈维兰和演员莉莲·方丹的大女儿。沃特在日本东京工作，1914年与莉莲结婚。她妹妹同样是奥斯卡影后[[琼·芳登|琼·芳登]]，但姐妹俩几十年如一日感情失和。

她父亲同父异母兄弟查尔斯·德哈维兰的儿子[[杰弗里·德哈维兰|杰弗里·德哈维兰]]是航空先驱之一。

在德哈维兰两岁的时候，她的家庭从日本东京搬到了美国[[加利福尼亚州|加利福尼亚州]]，她在那里度过了自己的中学时代。

== 演艺生涯 ==
1935年，德哈维兰出演了处女作电影"Alibi Ike"，和[[乔伊·布朗|乔伊·布朗]]搭档。她第一个舞台剧的角色是[[威廉·莎士比亞|威廉·莎士比亞]]名剧《[[仲夏夜之梦|仲夏夜之梦]]》中的赫米娅，该剧在[[好莱坞露天剧场|好莱坞露天剧场]]演出。之后，这部舞台剧被搬上了银幕。[[华纳兄弟|华纳兄弟]]与她签了一份长达七年的合同。[[埃罗尔·弗林|埃罗尔·弗林]]是她银幕上的黄金搭档，两人联袂主演了《[[铁血船长|铁血船长]]》、《[[英烈传|英烈传]]》和《[[侠盗罗宾汉_(1938年电影)|罗宾汉历险记]]》等八部影片。

1939年，德哈维兰在影片《[[乱世佳人_(电影)|乱世佳人]]》中饰演梅兰尼·威尔克斯，这个角色为她获得了[[奥斯卡最佳女配角奖|奥斯卡最佳女配角奖]]提名。影片共有四个主要角色，梅兰尼是唯一在片中死去的角色。然而，在现实生活中，她活得比其他三位演员都长（[[克拉克·盖博|克拉克·盖博]]、[[费雯丽|费雯丽]]和[[莱斯利·霍华德|莱斯利·霍华德]]）。

1941年，奥丽维亚加入[[美国|美国]]国籍，也因此得到了更多的机会。她觉得自己的演技有所提高，可以胜任不同类型的角色，而不是一再演出端庄娴静的少女之类的角色，因此她开始拒绝出演雷同的角色。而她每拒绝一个角色，她的合同也就会相应地延长，这也是当时电影公司对付不合作的演员的惯用方式。

大部分演员出于无奈，接受电影公司不公平的合同，其中有一些演员试图反抗。1930年代，[[贝蒂·戴维斯|贝蒂·戴维斯]]被[[华纳兄弟|华纳兄弟]]告上法庭，最终败诉。1940年代，德哈维兰也收到了诉讼，但是她打赢了官司。从此，电影公司的权利被削减，而演员的选择角色的自由度大大增加。这个事件对[[好莱坞|好莱坞]]的法律健全有深远的影响。德哈维兰的勇气也使她得到其他演员的尊敬。

德哈维兰开始有机会饰演不同的角色。1940年代后期，她因在《[[风流种子|风流种子]]》和《千金小姐》({{tsl|en|The_Heiress|The Heiress}})中的出色表演而获得了两座奥斯卡最佳女主角奖杯。

1950年代以后，德哈维兰出演的电影不多，她只是偶尔出现在银幕上，直到1970年代后期。1980年代，德哈维兰开始她的电视生涯，并因《[[安娜斯塔西娅：安娜疑案|安娜斯塔西娅：安娜疑案]]》（Anastasia: The Mystery of Anna）获得金球奖最佳连续短剧女配角奖。

== 个人生活 ==
尽管奥丽维亚和[[艾羅爾·弗林|艾羅爾·弗林]]被认为好莱坞最般配的银幕搭档之一，并一起合作了8部电影，但是他们在银幕下却从来没有罗曼史。

1940年代早期，德哈维兰和[[约翰·休斯顿|约翰·休斯顿]]、[[詹姆斯·斯图尔特|詹姆斯·斯图尔特]]和[[霍华德·休斯|霍华德·休斯]]传出过戀情。1946年，她嫁给了作家马库斯·古德里奇，但是于1953年离婚。两人育有一子本杰明，1991年因[[霍奇金淋巴瘤|霍奇金淋巴瘤]]并发症去世。

1955年，德哈维兰和法国杂志主持人皮埃尔·加朗特结婚，并于次年产下女儿吉赛尔。1979年，德哈维兰和加朗特离婚，两人仍保持很好的关系。1998年，为了照顾病重的加朗特，德哈维兰缺席了奥斯卡70周年庆礼。

德哈维兰与同时代的明星[[贝蒂·戴维斯|贝蒂·戴维斯]]和[[格罗丽亚·斯图尔特|格罗丽亚·斯图尔特]]是密友。

== 息影後生活 ==
1950年代，德哈维兰开始在[[巴黎|巴黎]]定居。息影后她很少在公共场合出现。2003年她出席了第75届奥斯卡颁奖晚会。2006年6月，她相继出席了美国影艺学院和[[洛杉矶县立艺术博物馆|洛杉矶县立艺术博物馆]]为她举办了90岁生日活动。

2003年[[凯瑟琳·赫本|凯瑟琳·赫本]]逝世后，奥丽维亚·德哈维兰和妹妹琼·芳登被认为是好莱坞1930年代和1940年代最后的两位顶尖女演员。
