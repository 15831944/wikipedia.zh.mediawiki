'''奥赞·亚那'''（{{lang-tr|Ozan Yanar}}，1987年5月3日生於[[土耳其|土耳其]][[伊斯坦布爾|伊斯坦布爾]]）是[[芬蘭|芬蘭]]的政治家和[[芬蘭國會|芬蘭國會]]議員，代表[[綠色聯盟|綠色聯盟]]。2015年以來他是綠色青年和學生聯合會的主席。<ref>{{cite web |url=http://www.vaalikone.fi/eduskunta2015/tulos/0-01/ehdokas/89 |title= Vihreiden nuoret valitsivat puheenjohtajat |publisher= Yle |date= 2014-11-29 |accessdate=2015-04-22}}</ref>他在2015年的選舉中以4 196票當選為國會議員。<ref>{{cite web |url=http://tulospalvelu.vaalit.fi/E-2015/fi/val01.html |title=Valitut ehdokkaat Helsingin vaalipiiri |publisher=芬蘭法務部 |accessdate=2015-04-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20150427143010/http://tulospalvelu.vaalit.fi/E-2015/fi/val01.html |archivedate=2015-04-27 }}</ref><ref>[http://finland.fi/Public/default.aspx?contentid=325282&nodeid=41805&culture=zh-CN 芬兰外交部新闻司, 芬兰推广委员会: 这就是芬兰: 2015年芬兰议会大选: 芬兰大选结束，组阁仍不明朗]</ref>

== 个人生活 ==
亚那出生于[[土耳其|土耳其]]，小时候曾在[[英国|英国]]和[[北塞浦路斯|北塞浦路斯]]居住过。他在14岁时随父亲搬至芬兰。他同时具有土耳其和芬兰国籍。<ref name="HS_millainen"/>
