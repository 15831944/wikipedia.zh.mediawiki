《'''奧林匹克聖歌'''》（又稱'''奧林匹克頌詞'''；{{lang-gr|'''Ολυμπιακός Ύμνος'''}}，{{lang-fr|'''Hymne Olympique'''}}，{{lang-en|'''Olympic Hymn'''}}）是'''奧林匹克會歌'''（{{lang-en|'''Olympic Anthem'''}}），是一首[[合唱|合唱]]的[[清唱劇|清唱劇]]；奧林匹克聖歌於1958年的第54次國際奧委會全體會議上定為奧林匹克會歌。奧林匹克會歌由[[國際奧林匹克委員會|國際奧林匹克委員會]]的第一任[[主席|主席]][[澤麥特里烏斯·維凱拉斯|澤麥特里烏斯·維凱拉斯]]親自選擇的；由[[希臘|希臘]][[歌劇|歌劇]][[作曲家|作曲家]][[斯皮羅·薩馬拉斯|斯皮羅·薩馬拉斯]]作曲、希臘作家兼[[詩人|詩人]][[科斯蒂斯·帕拉馬斯|科斯蒂斯·帕拉馬斯]]作詞。

== 歷史 ==
《奧林匹克聖歌》於1896年的[[1896年夏季奧林匹克運動會|雅典第一屆奧運會]]上首次被奏起<ref name="IOC268">{{en}} {{cite web|url=http://multimedia.olympic.org/pdf/en_report_268.pdf |archiveurl=https://web.archive.org/web/20020221030912/http://multimedia.olympic.org/pdf/en_report_268.pdf |title=THE OLYMPIC ANTHEM |publisher=[[國際奧委會|國際奧委會]] |date=2002年1月31日 |accessdate=2010年7月31日 |archivedate=2002年2月21日 |deadurl=bot: unknown }}</ref>；該歌曲由[[希臘|希臘]][[歌劇|歌劇]][[作曲家|作曲家]][[斯皮羅·薩馬拉斯|斯皮羅·薩馬拉斯]]作曲<ref>{{cite book |author = [[國際奧委會|國際奧委會]] |title=[[奧林匹克憲章|奧林匹克憲章]] |year=2010 |page = 21 |location=[[瑞士|瑞士]][[洛桑|洛桑]] }}</ref>、[[希臘|希臘]]作家兼[[詩人|詩人]][[科斯蒂斯·帕拉馬斯|科斯蒂斯·帕拉馬斯]]作詞<ref name="CIL">{{en}}{{cite web |title = Olympic Anthem and Olympic Flag |publisher = Compare Infobase Limited |accessdate = 2010年7月31日 |url = http://www.mapsofworld.com/olympic-trivia/olympic-anthem-and-flag.html |deadurl = yes |archiveurl = https://web.archive.org/web/20100529023924/http://www.mapsofworld.com/olympic-trivia/olympic-anthem-and-flag.html |archivedate = 2010年5月29日 }}</ref>，但當時仍未被確定為奧林匹克會歌；及後至1956年的[[奧運會|奧運會]]，奧運會的主辦國均委託當地的[[作曲家|作曲家]]為奧林匹克聖歌作詞<ref>{{en}}{{cite web |title = Olympics |publisher = www.nationalanthems.info |accessdate = 2010年7月31日 |url = http://www.nationalanthems.info/oly.htm }}</ref>。

1954年，國際奧委會為奧林匹克會歌進行全球徵集；國際奧委會在是次徵集中收到來自40個不同國家合共392份參賽作品。[[波蘭|波蘭]][[作曲家|作曲家]]{{link-pl|米哈爾·什皮沙克|Michał Spisak}}的作品除了被第50次國際奧委會全體會議上挑選出來外，也獲得身兼[[國際奧委會委員|國際奧委會委員]]的[[摩納哥|摩納哥]][[皮埃爾·德·波利尼亞克|皮埃爾親王]]為奧林匹克會歌徵集而設的1000[[美元|美元]]獎金；獲挑選的作品原被安排於1956年舉辦的[[1956年夏季奧林匹克運動會|墨爾本奧運會]]奏起，惟該作曲家徵求大量版權費而被棄用<ref name="IOC268" />。

1958年，國際奧委會採納了身兼[[國際奧委會委員|國際奧委會委員]]的[[丹麥|丹麥]][[亞斯里王子|亞斯里王子]]把希臘作曲家斯皮罗·萨马拉斯作曲的《奧林匹克聖歌》作為奧林匹克會歌的建議；並於當年在[[日本|日本]][[東京|東京]]舉行的第54次國際奧委會全體會議上批准了這個建議方案<ref name="IOC268" />。奧林匹克聖歌分別在1960年2月及8月舉行的[[1960年冬季奧林匹克運動會|斯闊谷冬季奧運]]及[[1960年夏季奧林匹克運動會|羅馬奧運會]]正式以奧林匹克會歌奏起；而奧林匹克聖歌更是首次在冬季奧運會上奏起<ref>{{en}}{{cite book |author= [[國際奧委會|國際奧委會]] |title = Factsheet, Opening ceremony of the Olympic Winter Games |year=2009 |publisher=國際奧委會 |location=[[瑞士|瑞士]][[洛桑|洛桑]] }}</ref>。現在，奧林匹克聖歌除[[希臘語|希臘語]]外已使用過[[日语|日语]]、[[西班牙语|西班牙语]]、[[韓语|韓语]]、[[俄语|俄语]]、[[英语|英语]]、[[法语|法语]]、[[義大利語|義大利語]]等語言演唱並翻譯成[[日语|日语]]、[[西班牙语|西班牙语]]、[[韓语|韓语]]、[[俄语|俄语]]、[[汉语|汉语]]等多達12種不同語言。
<ref>{{en}}{{cite web |title=Olympic Hymn |publisher=News around the Globe |date=2010年2月13日 |accessdate=2010年7月31日 |url=http://www.makli.com/olympic-hymn-0025247/olympic-hymn/ }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>其樂譜也被存放位於[[瑞士|瑞士]][[洛桑|洛桑]]的[[國際奧委會|國際奧委會]]總部<ref>{{zh-cn}}{{en}}{{fr}} {{cite web|url=http://www.beijing2008.cn/49/71/article211997149.shtml |archiveurl=https://web.archive.org/web/20070119054759/http://www.beijing2008.cn/49/71/article211997149.shtml |title=奥林匹克会歌（Olympic Anthem）（Hymne Olympique ） |publisher=[[北京奧運|第29届奥林匹克运动会组织委员会]] |date=2005年12月31日 |accessdate=2010年7月31日 |archivedate=2007年1月19日 |deadurl=bot: unknown }}</ref>。

==使用時機==
*每屆夏、冬季奧運會開閉幕式升降會旗時。
*每年國際奧會年會開幕式。
*國際奧會年會宣布下屆夏、冬季奧運會主辦城市時。
*國際奧會年會選舉新任主席宣布當選人時。
== 歌詞 ==
國際奧委會規定奧林匹克會歌必須用[[希臘语|希臘语]]演唱，但可配合主辦國以該國語言演唱或混合英語演唱，如在2016年的里約熱內盧奧運時，奧林匹克會歌以[[英语|英语]]及[[葡萄牙语|葡萄牙语]]交叉式演唱。
