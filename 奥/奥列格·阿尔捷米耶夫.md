'''奥列格·格尔马诺维奇·阿尔捷米耶夫'''（{{lang-ru|Оле́г Ге́рманович Арте́мьев}}，{{bd|1970年|12月28日|}}）是[[俄罗斯|俄罗斯]][[宇航员|宇航员]]。

==早年==
1990年毕业于[[塔林理工大学|塔林理工大学]]。1998年毕业于[[鮑曼莫斯科國立技術大學|鮑曼莫斯科國立技術大學]]，获得[[制冷|制冷]]技术学位。毕业后在[[科罗廖夫能源火箭航天集团|科罗廖夫能源火箭航天集团]]工作<ref name=bio/>。

==宇航员生涯==
2003年5月29日成为宇航员人选，6月16日至2005年6月28日期间接受基础太空训练，并以优异成绩通过考验，2005年7月5日获得实验宇航员资格<ref name=bio>{{cite web |title=ARTEMYEV Oleg Germanovich |url=https://www.energia.ru/en/iss/iss39/artemyev.html |website=energia.ru |accessdate=2019-04-14 |language=en}}</ref>。

===远征39/40===
[[莫斯科时间|莫斯科时间]]2014年3月26日凌晨1时17分（[[北京时间|北京时间]]5时17分），[[联盟-FG|联盟-FG]]运载火箭在拜科努尔航天发射场发射成功。9分钟后，载有阿尔捷米耶夫、[[亚历山大·斯克沃尔佐夫|亚历山大·斯克沃尔佐夫]]和[[史蒂文·斯旺森|史蒂文·斯旺森]]的[[联盟TMA-12M|联盟TMA-12M]]号宇宙飞船与运载火箭分离，并进入预定轨道。飞船绕地球运行了4圈后前往国际空间站<ref>{{cite web |author=郭思文 |title=俄罗斯“联盟TMA-12M”载人飞船今晨成功发射 |url=http://world.huanqiu.com/exclusive/2014-03/4931608.html |publisher=[[环球网|环球网]] |archiveurl=https://web.archive.org/web/20190414072036/http://world.huanqiu.com/exclusive/2014-03/4931608.html |archivedate=2019-04-14 |date=2014-03-26}}</ref>。

[[格林尼治时间|格林尼治时间]]8月18日14时（北京时间22时）左右，他和斯克沃尔佐夫走出空间站，开始了5个多小时的太空行走。出舱行走大约22分钟后，他用戴着手套的右手将一颗来自[[秘鲁|秘鲁]]的人造迷你[[卫星|卫星]]送入太空<ref>{{cite web |author=徐超 |title=俄宇航员太空放飞迷你卫星 |url=http://news.sina.com.cn/o/2014-08-20/152930716602.shtml |publisher=[[新浪|新浪]]新闻 |archiveurl=https://web.archive.org/web/20190414073613/http://news.sina.com.cn/o/2014-08-20/152930716602.shtml |archivedate=2019-04-14 |date=2014-08-20}}</ref>。

2014年9月11日，搭载远征40任务成员的联盟TMA-12M号飞船在哈萨克斯坦预定区域着陆。飞船着陆时的安全由约200名官兵、14架[[Mi-8直升機|米-8直升机]]、3架[[安托諾夫An-12|安-12]]和[[安托諾夫An-26|安-26]]飞机以及6台机动车辆保障<ref>{{cite web |author1=许宏瑞 |title=俄联盟号飞船搭载国际空间站3名宇航员成功着陆 |url=http://world.huanqiu.com/exclusive/2014-09/5135244.html |archiveurl=https://web.archive.org/web/20190414072538/http://world.huanqiu.com/exclusive/2014-09/5135244.html |archivedate=2019-04-14 |date=2014-09-11}}</ref>。

===远征55/56===

莫斯科时间2018年3月21日20点44分（北京时间22日凌晨1点44分），他和美国宇航员[[安德鲁·福伊斯特尔|安德鲁·福伊斯特尔]]和[[理查德·阿诺德|理查德·阿诺德]]乘坐[[联盟MS-08|联盟MS-08]]载人飞船从[[哈萨克斯坦|哈萨克斯坦]][[拜科努尔发射场|拜科努尔发射场]]发射升空。他们此行携带了一个[[足球|足球]]<ref>{{cite web |title=俄火箭将俄美３名宇航员送往空间站 外带一足球 |url=http://www.xinhuanet.com/science/2018-03/23/c_137058711.htm |publisher=[[新华社|新华社]] |accessdate=2019-04-14 |archiveurl=https://web.archive.org/web/20180325134653/http://www.xinhuanet.com/science/2018-03/23/c_137058711.htm |archivedate=2018-03-25 |location=[[莫斯科|莫斯科]] |date=2018-03-23}}</ref>。这个足球被称作“[[电视之星_18|电视之星 18]]”，是6月[[2018年國際足協世界盃|俄罗斯世界杯]]的比赛用球<ref>{{cite web |title=宇航员返回地球 把世界杯揭幕战足球从太空带回了！ |url=http://www.xinhuanet.com/2018-06/04/c_1122932093.htm |publisher=中国新闻网 |archiveurl=https://web.archive.org/web/20190414070142/http://www.xinhuanet.com/2018-06/04/c_1122932093.htm |archivedate=2019-04-14 |date=2018-06-04}}</ref>。

4月2日，阿尔捷米耶夫在[[Twitter|Twitter]]上传一张在[[微重力|微重力]]环境下发芽的太空[[洋葱|洋葱]]照片。这是他第二次将洋葱带上太空<ref>{{cite web |title=有新鲜菜吃了！俄罗斯宇航员往国际空间站带洋葱 |url=http://www.chinanews.com/gj/2018/04-04/8483123.shtml |publisher=[[中国新闻网|中国新闻网]] |accessdate=2019-04-14 |archiveurl=https://web.archive.org/web/20180923162317/http://www.chinanews.com/gj/2018/04-04/8483123.shtml |archivedate=2018-09-23 |date=2018-04-04}}</ref>。

4月，他和什卡普列罗夫在太空参加了2018年的“世界俄语听写大赛”，他也曾在2014年太空任务中参加该年度大赛<ref>{{cite web |author1=华迪 |title=俄现役宇航员将在太空参加世界俄语听写大赛 |url=http://ru.people.com.cn/n1/2018/0412/c408039-29922916.html |publisher=人民网 |archiveurl=https://web.archive.org/web/20190414073128/http://ru.people.com.cn/n1/2018/0412/c408039-29922916.html |archivedate=2019-04-14 |date=2018-04-12}}</ref>。

[[6月12日|6月12日]]，他在[[国际空间站|国际空间站]]录制视频，向俄公民祝贺“[[俄罗斯国庆日|俄罗斯国庆日]]”<ref>{{cite web |title=太空祝福！俄宇航员从国际空间站祝贺“俄罗斯日” |url=http://news.cctv.com/2018/06/13/ARTIEaFTFyN1ejQL3CMzs1gs180613.shtml |publisher=[[中国中央电视台|CCTV]] |archiveurl=https://web.archive.org/web/20190414065413/http://news.cctv.com/2018/06/13/ARTIEaFTFyN1ejQL3CMzs1gs180613.shtml |archivedate=2019-04-14 |date=2018-06-13}}</ref>。

莫斯科时间8月30日凌晨时分，俄罗斯飞行控制中心值班员发现国际空间站舱内有微量漏气现象。后来经过检查，阿尔捷米耶夫等队员在联盟号飞船轨道舱壁内发现了一个直径约1.5[[毫米|毫米]]的穿孔。他们用胶水和胶带将穿孔填补封好。阿尔捷米耶夫在事后称，同事[[谢尔盖·普罗科皮耶夫|谢尔盖·普罗科皮耶夫]]的头发因此白了40多根。普罗科皮耶夫则称，白发是飞船减压事故后才有的<ref>{{cite web |title=国际空间站发生漏气事故 俄宇航员“愁白头” |url=http://www.xinhuanet.com/world/2018-09/21/c_129958391.htm |publisher=中国新闻网 |accessdate=2019-04-14 |archiveurl=https://web.archive.org/web/20181127170315/http://www.xinhuanet.com/world/2018-09/21/c_129958391.htm |archivedate=2018-11-27 |date=2018-09-21}}</ref>。

哈萨克斯坦时间2018年10月4日17时44分许（北京时间19时44分许），[[联盟MS-08|联盟MS-08]]载人飞船在哈萨克斯坦[[杰兹卡兹甘|杰兹卡兹甘]]附近一处空旷地区成功着陆<ref>{{cite web |title=俄美3宇航员从国际空间站安全返回地球 |url=http://news.sciencenet.cn/htmlnews/2018/10/418322.shtm |publisher=新华社 |archiveurl=https://web.archive.org/web/20190414070431/http://news.sciencenet.cn/htmlnews/2018/10/418322.shtm |archivedate=2019-04-14 |date=2018-10-05}}</ref>。
