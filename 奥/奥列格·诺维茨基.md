'''奥列格·维克托罗维奇·诺维茨基'''（{{lang-ru|Оле́г Ви́кторович Нови́цкий}}，{{lang-be|Алег Віктаравіч Навіцкі}}，{{bd|1971年|10月12日|}}）是[[白俄罗斯|白俄罗斯]]出身的[[俄罗斯|俄罗斯]][[宇航员|宇航员]]。

==早年==
1971年10月12日出生于[[苏联|苏维埃社会主义共和国联盟]][[白俄罗斯苏维埃社会主义共和国|白俄罗斯苏维埃社会主义共和国]][[明斯克州|明斯克州]][[切爾文|切爾文]]。1988年毕业于切尔文市第二中学，毕业后在以[[瓦列里·契卡洛夫|契卡洛夫]]命名的[[鲍里索格列布斯克|鲍里索格列布斯克]]军事飞行员学校学习，后在[[叶伊斯克高等军事航空学校|叶伊斯克高等军事航空学校]]学习，1994年毕业。1994年10月至2004年9月在空军支队担任飞行员和指挥官等职务。2004年至2006年在[[加加林空军学院|加加林空军学院]]学习空军军队调度管理<ref name=bio>{{cite web |title=Biographical Data - Oleg Viktorovich Novitskiy |url=https://web.archive.org/web/20171103182012/https://www.jsc.nasa.gov/Bios/htmlbios/novitskiy.html |publisher=[[NASA|NASA]] |accessdate=2019-04-16}}</ref>。
==宇航员生涯==
2006年10月，他通过了[[俄罗斯科学院生物医学问题研究所|俄罗斯科学院生物医学问题研究所]]的体检，成为[[加加林宇航员培训中心|加加林宇航员培训中心]]的宇航员候选人，后开始接受基础培训。2008年6月22日至28日，他和[[叶莲娜·谢罗娃|叶莲娜·谢罗娃]]和[[谢尔盖·列温|谢尔盖·列温]]在[[乌克兰|乌克兰]][[塞瓦斯托波尔|塞瓦斯托波尔]]参与了特殊训练。2009年6月2日，他以优异成绩通过了最后考验，6月9日获得实验宇航员资格。2010年3月至2012年5月接受了[[国际空间站|国际空间站]]远征31/32备用机组成员飞行工程师的专门培训<ref>{{cite web |title=NOVITSKIY Oleg Victorovich |url=https://www.energia.ru/en/iss/iss48/novitskiy.html |publisher=[[科罗廖夫能源火箭航天集团|Korolev RSC "Energia"]] |accessdate=2019-04-16 |date=2016}}</ref>。
===远征33/34===
[[协调世界时|协调世界时]]2012年10月23日10时51分，[[联盟TMA-06M|联盟TMA-06M]]号载人飞船连同“联盟-FG”号运载火箭搭载诺维茨基和[[叶夫根尼·塔列尔金|叶夫根尼·塔列尔金]]、美国宇航员[[凯文·福特|凯文·福特]]，从哈萨克斯坦的拜科努尔航天发射场发射升空，前往[[国际空间站|国际空间站]]<ref>{{cite web |title=俄“联盟”号飞船搭载3名宇航员发射升空 |url=http://cn.chinadaily.com.cn/2012-10/23/content_15840368.htm |publisher=[[中国日报|中国日报]] |archiveurl=https://web.archive.org/web/20190416145022/http://cn.chinadaily.com.cn/2012-10/23/content_15840368.htm |archivedate=2019-04-16 |date=2012-10-23}}</ref>。

2013年3月16日，他和[[叶夫根尼·塔列尔金|塔列尔金]]、福特成功降落在[[哈萨克斯坦|哈萨克斯坦]]境内。他们原计划15日返航，但由于降落地点天气不佳而推迟到16日<ref>{{cite web |title=三名宇航员从国际空间站返回地球 |url=https://www.voachinese.com/a/space-station-20130316/1622672.html |publisher=[[美国之音|美国之音]] |archiveurl=https://web.archive.org/web/20190416142519/https://www.voachinese.com/a/space-station-20130316/1622672.html |archivedate=2019-04-16 |date=2013-03-16}}</ref>。
===远征50/51===
北京时间2016年11月18日凌晨4时20分，载有他和[[佩姬·惠特森|佩姬·惠特森]]、[[托马·佩斯凯|托马·佩斯凯]]的联盟MS-03飞船在哈萨克斯坦的[[拜科努尔发射场|拜科努尔发射场]]成功发射升空，奔赴国际空间站<ref>{{cite web |title=俄罗斯“联盟”飞船今载3名国际宇航员发射升空 |url=http://m.news.cctv.com/2016/11/18/ARTI8rUvHio8UmOYxVypQCTD161118.shtml |publisher=[[CCTV|CCTV]] |archiveurl=https://web.archive.org/web/20190416143134/http://m.news.cctv.com/2016/11/18/ARTI8rUvHio8UmOYxVypQCTD161118.shtml |archivedate=2019-04-16 |date=2016-11-18}}</ref>。

任务期间，他和[[法国|法国]]宇航员佩斯凯相互交流[[学习|学习]]了各自的[[语言|语言]]（[[法语|法语]]和[[俄语|俄语]]）<ref>{{cite web |title=法国宇航员在外太空执行考察任务期间学会俄语 |url=http://www.chinanews.com/gj/2017/08-09/8300256.shtml |publisher=[[中国新闻网|中国新闻网]] |accessdate=2019-04-16 |archiveurl=https://web.archive.org/web/20170809180723/http://www.chinanews.com/gj/2017/08-09/8300256.shtml |archivedate=2017-08-09}}</ref>。

莫斯科时间2017年6月2日13时47分（北京时间18时47分），他和佩斯凯乘坐[[联盟MS-03|联盟MS-03]]飞船脱离国际空间站<ref>{{cite web |author=魏忠杰 |title=俄法宇航员从国际空间站平安返回地面 |url=http://www.xinhuanet.com//mil/2017-06/03/c_129624368.htm |publisher=[[新华社|新华社]] |archiveurl=https://web.archive.org/web/20190416141321/http://www.xinhuanet.com//mil/2017-06/03/c_129624368.htm |archivedate=2019-04-16 |date=2017-06-03}}</ref>，并于17时10分（北京时间22时10分）成功降落在[[哈萨克斯坦|哈萨克斯坦]]境内的预定区域。计划原拟于5月16日返地，由于相关参数发生了变化，最终时间推迟了约半个月<ref>{{cite web |title=“联盟号”飞船载俄法宇航员在哈成功着陆 |url=http://www.xinhuanet.com//world/2017-06/04/c_129624489.htm |publisher=[[中国新闻网|中国新闻网]] |archiveurl=https://web.archive.org/web/20190416141718/http://www.xinhuanet.com//world/2017-06/04/c_129624489.htm |archivedate=2019-04-16 |date=2017-06-04}}</ref>。
