'''梁河公交'''是[[云南省|云南省]][[德宏傣族景颇族自治州|德宏傣族景颇族自治州]][[梁河县|梁河县]]的[[公共汽车|城市公共汽车]]系统，由梁河县城市公交有限公司（隶属[[德宏交通集团|德宏交通集团]]）运营。梁河公交于2018年1月10日正式开通，运营有两条线路，首期投入使用公交车4辆，随后将增加到6辆。营运时间为早上7点至晚上8点，循环发车，实行全程一票制，采取上车自动投币，票价2元。公交系统的主要换乘站为九保桥头，2路分为A、B两线，是环线。<ref name="线路">{{cite web |url=https://mp.weixin.qq.com/s?__biz=MjM5NDI0ODY4NQ==&mid=2651303079&idx=2&sn=1b2c7b4c1ccfe2c3f1d99dc03cfb932a&chksm=bd79ae258a0e273320a218653a3a4e94a3fe7cb866c21a9600006a9057088f01f3646cf36345&mpshare=1&scene=23&srcid=01117zjOfrR9JnI6cMGCfdbq#rd |title=喜讯！梁河开通城市公交 有这2条线！ |author=何成江 |date=2018-01-10 |website= |publisher=德宏团结报 |access-date=2018-01-10 |quote=}}</ref><ref>{{cite web |url=http://www.dhlh.gov.cn/Web/Detail.aspx?id=13346 |title=举行《梁河县城市公交运营方案》（草案）运营线路及运营价格听证会的公告 |author= |date=2017-09-13 |website=www.dhlh.gov.cn |publisher=梁河县人民政府 |access-date=2018-01-10 |quote= }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref>{{cite web |url=http://www.dhlh.gov.cn/Web/Detail.aspx?id=14353 |title=《梁河县城市公交运营方案》（草案）运营线路及运营价格听证会报告的公告 （第3号） |author= |date=2017-09-30 |website=www.dhlh.gov.cn |publisher=梁河县人民政府 |access-date=2018-01-10 |quote= }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>
