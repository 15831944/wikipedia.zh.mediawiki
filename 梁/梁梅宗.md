'''梁梅宗'''，[[臺灣|臺灣]][[體操|體操]][[教練|教練]]。

== 個人生活 ==
梁梅宗與雙胞胎哥哥[[梁梅松|梁梅松]]是[[高雄市立五福國民中學|高雄市立五福國民中學]]的體操選手，並一同就讀[[國立臺灣體育學院|國立臺灣體育學院]]<ref name = M00062651>{{cite news|title=孿生體操好手|publisher=民生報|url=http://catalog.digitalarchives.tw/pdf/00/00/a3/d6.pdf|date=1985-05-06|author=楊權寶|accessdate=2018-03-14|language=zh-hant}}</ref>。

== 生平 ==
1985年梁梅宗在[[臺灣區運動會|臺灣區運動會]]中，獲得[[雙槓|雙槓]]金牌及[[跳馬|跳馬]]銀牌<ref name = M00062651/>。同年舉辦的[[全國大專院校運動會|全國大專院校運動會]]上，梁梅宗獲得[[自由體操|地板]]和跳馬的雙面金牌，並於男子個人全能比賽排名冠軍<ref>{{cite news|title=大專運動會體操賽 黃保協鞍馬、吊環、雙槓連中三元 梁梅宗地板、跳馬奪魁|publisher=民生報|url=http://catalog.digitalarchives.tw/pdf/00/00/a2/8f.pdf|date=1985-05-08|author=|accessdate=2018-03-14|language=zh-hant}}</ref><ref>{{cite news|title=大運會體操甲組競技 北體雙料冠軍|publisher=民生報|url=http://catalog.digitalarchives.tw/pdf/00/00/a0/67.pdf|date=1985-05-06|author=|accessdate=2018-03-14|language=zh-hant}}</ref>。

1991年3月梁梅宗經[[中華民國|中華民國]][[教育部體育署|教育部體育署]]補助經費招考的國小體操教練，派駐於[[高雄市前鎮區光華國民小學|高雄市前鎮區光華國民小學]]<ref>{{cite news|title=體操金牌教練被控性侵|publisher=自由時報|url=http://news.ltn.com.tw/news/focus/paper/1182928|date=2018-03-11|author=方志賢、黃建華|accessdate=2018-03-13|language=zh-hant}}</ref><ref name = T2003689135>{{cite news|title=Gym coach questioned in underage rape probe|publisher=Taipei Times|url=http://www.taipeitimes.com/News/taiwan/archives/2018/03/12/2003689135|date=2018-03-12|author=Jason Pan|accessdate=2018-03-13|language=en}}</ref><ref>{{cite news|title=體操／台版#MeToo案一事 體操協會今發聲明|publisher=聯合報|url=https://udn.com/news/story/7005/3027078|date=2018-03-12|author=毛琬婷|accessdate=2018-03-13|language=zh-hant}}</ref><ref>{{cite news|title=操協：性侵案屬實將撤銷資格|publisher=臺灣時報|url=http://www.twtimes.com.tw/?page=news&nid=704574|date=2018-03-13|author=張添福|accessdate=2018-03-13|language=zh-hant}}</ref><ref name = T3382577>{{cite news|title=Taiwan’s #MeToo Moment: schools to raise awareness about rape via education|publisher=Taiwan News|url=https://www.taiwannews.com.tw/en/news/3382577|date=2018-03-13|author=Sophia Yang|accessdate=2018-03-13|language=en}}</ref>。

2009年梁梅宗帶領著[[林恩霆|林恩霆]]、[[郭芳瑋|郭芳瑋]]、[[康智凱|康智凱]]及[[林恩澔|林恩澔]]四位體操選手參加全國體操錦標賽，在12月19日分別以個人和團體共獲得6面金牌<ref>{{cite news|title=翻滾吧男孩 血淚交織4人奪6金|publisher=中央社|url=|date=2009-12-29|author=王淑芬|accessdate=2018-03-13|language=zh-hant}}</ref>。
