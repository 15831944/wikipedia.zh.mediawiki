'''梁家騮'''（'''Leung Ka-lau'''，{{bd|1962年}}），出生於[[英屬香港|英屬香港]]，籍貫[[廣東|廣東]][[肇慶|肇慶]]<ref>{{cite web|url=http://news.sina.com.hk/cgi-bin/nw/show.cgi/3/1/1/1100636/1.html|title=梁美芬梁家騮係同鄉|author=|date=|website=香港新浪|access-date=2009-04-14|archive-url=https://web.archive.org/web/20090422222312/http://news.sina.com.hk/cgi-bin/nw/show.cgi/3/1/1/1100636/1.html|archive-date=2009-04-22|dead-url=yes}}</ref>，现任香港京剧院名誉院长，前[[香港立法會|香港立法會]][[功能界別|功能界別]]（[[醫學界功能界別|醫學界]]），[[沙田|沙田]][[威爾斯親王醫院|威爾斯親王醫院]][[外科|外科]][[醫生|醫生]]，他是首名從[[香港公立醫院|香港公立醫院]]醫生出身的立法會議員。

== 簡介 ==
梁早年就讀[[皇仁書院|皇仁書院]]，1986年畢業於[[香港中文大學醫學院|香港中文大學醫學系]]，成為首屆畢業生，2005年取得香港中文大學[[醫學博士|醫學博士]]銜頭。

他曾自費跟[[醫院管理局|醫院管理局]]打官司<ref>[[香港醫生工時運動|香港醫生工時運動]]</ref>，成功為公共醫生爭取假日工作補償，為年輕醫生爭取改善待遇<ref>{{cite web|url=http://www.mingpaonews.com/20080909/gaj3.htm|title=梁家騮險勝打破私醫壟斷|author=|date=|website=明報新聞網|access-date=2008-09-10|archive-url=https://web.archive.org/web/20080911221943/http://www.mingpaonews.com/20080909/gaj3.htm|archive-date=2008-09-11|dead-url=yes}}</ref>。

在[[2008年香港立法會選舉|2008年香港立法會選舉]]擊敗角逐連任、[[泛民主派|泛民主派]]的[[郭家麒|郭家麒]]，當選立法會議員<ref>{{cite web|url=http://www.rthk.org.hk/rthk/news/expressnews/20080908/news_20080908_55_521002.htm|title=醫學界梁家騮擊敗郭家麒當選|author=|date=|website=rthk.org.hk}}</ref>。

[[2012年香港立法會選舉|2012年香港立法會選舉]]成功連任，是[[香港回歸|香港回歸]]後首位成功連任的醫學界立法會議員。

2011年9月，和任職公關顧問的關天燕（Cannas）結婚<ref>{{cite web|url=http://hk.apple.nextmedia.com/news/art/20110914/15612212|title=隔牆有耳：怪醫梁家騮做老襯|author=|date=|website=Apple Daily 蘋果日報}}</ref>。

现任香港京剧院名誉院长。

== 政治立場 ==
梁家騮在2015年投票反對政改方案前，一直被歸類為[[建制派|建制派]]一員。因他並非[[泛民主派|泛民主派]]成員，故過去習慣上他仍被視為建制派的一員。<ref>{{cite web|url=http://hk.apple.nextmedia.com/legco2016/infographics/detail/04415235-8054-4477-926a-3181560557b2|title=Apple Daily HK: LegCo 2016|author=|date=|website=Apple Daily 蘋果日報|access-date=2016-06-29|archive-url=https://web.archive.org/web/20160701233825/http://hk.apple.nextmedia.com/legco2016/infographics/detail/04415235-8054-4477-926a-3181560557b2|archive-date=2016-07-01|dead-url=yes}}</ref>

雖然未就其對[[普選|普選]]的立場作出明確表態，但是梁家騮亦於[[2012年香港立法會選舉|2012年香港立法會選舉]]的[[商業電台|商業電台]]選舉論壇中表示有五成以上醫學界選民支持取消功能界別，並且表示取消程序應為一次過取消，即是不應只取消部分功能界別，個人亦贊成。

梁家騮在2008年12月17日動議回購[[領匯|領匯]]股份議案，但結果被否決。<ref>{{cite web|url=http://www.legco.gov.hk/yr08-09/chinese/counmtg/voting/v200812175.htm|title=立法會會議 (投票結果) 2008年12月17日 「回購領匯股份」議案|work=立法會}}</ref>

[[六四事件|六四事件]]立場方面，他於2009年5月27日，泛民主派提出平反六四動議，梁支持[[李卓人|李卓人]]首先提出的[[動議|動議]]，但於泛民其他人提出修正案的時候則離場。2010和2013年，他繼續投票支持平反六四。<ref>{{cite web|url=http://www.legco.gov.hk/yr09-10/chinese/counmtg/voting/v201006021.htm|title=立法會會議 (投票結果) 2010年6月2日 「六四事件」議案|work=立法會}}</ref><ref>{{cite web|url=http://www.legco.gov.hk/yr12-13/chinese/counmtg/voting/v20130529.pdf|title=立法會會議 (投票結果) 2013年5月29日 「六四事件」議案|work=立法會}}</ref>

2010年1月16日，在669億元[[廣深港高速鐵路#.E7.88.AD.E8.AD.B0|廣深港高速鐵路撥款爭議]]中，梁在沒有了解業界意見的情況下，支持高鐵工程，令部分醫學界選民對其十分不滿和失望。其後梁在其[[個人網頁|個人網頁]]上貼出一份《[[明報|明報]]》報導，標題為「騮哥為高鐵投票做poll」，聲稱有為高鐵撥款爭議進行過[[民意調查|民意調查]]了解民意。然而其民調只是一個在撥款通過前一天才開始的網上投票，而且梁未有正式公佈或通知過醫學界選民有此民調。最終梁根據該進行了一天之民調結果，對高鐵撥款議案投下支持一票。<ref>[http://news.sina.com.hk/news/1125/3/1/1399680/1.html 騮哥為高鐵投票做poll]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web|url=http://life.mingpao.com/cfm/dailynews3b.cfm?File=20100119/nalgz/gzb4.txt|title=騮哥為高鐵投票做poll|author=|date=|website=life.mingpao.com|access-date=2017-07-07|archive-url=https://web.archive.org/web/20180805052103/https://life.mingpao.com/cfm/dailynews3b.cfm?File=20100119%2Fnalgz%2Fgzb4.txt|archive-date=2018-08-05|dead-url=yes}}</ref>

2010年6月23日，在表決[[2012年香港政治制度改革|2012年香港政治制度改革]]方案當中，他投下贊成票。<ref>{{cite web|url=http://www.legco.gov.hk/yr09-10/chinese/counmtg/voting/v201006232.htm|title=立法會會議 (投票結果) 2010年6月23日 就修改行政長官產生辦法提出的議案|author=|date=|website=www.legco.gov.hk}}</ref>

2013年11月，表決引用立法會權力及特權條例調查[[香港免費電視牌照爭議|香港免費電視發牌事件]]議案，他投下贊成票。<ref>{{cite web|url=http://www.legco.gov.hk/yr13-14/chinese/counmtg/voting/v20131106.pdf|title=郭榮鏗議員對莫乃光議員根據《立法會（權力及特權）條例》動議的議案作出的修正案|website=www.legco.gov.hk|date=2013年11月6日}}</ref>

2014年，[[雨傘革命|雨傘革命]]期間，他並無參與其他41名建制派議員反對佔領行動及支持[[香港警察|香港警察]]的聯署。

== 投票否決政改方案 ==
2015年6月18日，立法會表決2017年行政長官產生辦法的政改方案，他稱跟隨業界意願，投下反對票，包括26名泛民主派議員，連同[[本土派|本土派]]的[[黃毓民|黃毓民]]及他自己，政改方案最終被否決。其後他接受訪問時，表明從未自視為建制派，認為自己立場中立、獨立，不會分門分派。他說不論泛民、建制，如果邀請他討論政事，他向來也會出席。<ref>{{cite web|url=http://www.881903.com/Page/ZH-TW/newsdetail.aspx?csid=261_341&itemid=805849|title=881903.com 商業電台 - 梁家騮說沒自視為建制派|author=|date=|website=881903.com}}</ref>

== 就《2016年醫生註冊(修訂)條例草案》進行拉布抗爭 ==
2016年6月及7月，因為醫學界就《2016年醫生註冊(修訂)條例草案》有爭議，醫學界強烈反對增加業外成員加入醫務委員會，因為這會導致政府加強操控醫委會，並且不會加快個案進度。所以梁家騮代表業界利益，就此條例草案進行拉布抗爭。當中6月29日的會議更因為梁家騮多次提出點算法定人數而流會收場。其後，梁家騮先後提出中止待續、交予專責委員會和休會辯論，結果在7月15日（第五屆立法會最後一天）均未能就條例三讀表決，在休會辯論階段中止會議。<ref>{{cite web|url=http://hk.on.cc/hk/bkn/cnt/news/20160706/bkn-20160706133903169-0706_00822_001.html|title=梁家騮狂點23次人數　各方轟拉布促放過香港|author=|date=|website=on.cc}}</ref>

== 立法會表現 ==
由2008年10月15日起，截至2009年7月9日為止，立法會一共舉行了39次周三例會。梁家騮缺席了5次，不投票率為51.5%，排名僅次於[[李國寶|李國寶]]和[[謝偉俊|謝偉俊]]。不投票率即議員無缺席會議，但投票時不在場的次數。梁發言次數為10次，提出質詢次數為3次。以30位功能界別議員平均來計算，發言和提出質詢次數分別為21.1和8.3次。

2009年7月10日，《[[東方日報|東方日報]]》引述了梁家騮對以上統計數字之回應。梁議員說，他仍要半職工作，每周在醫院上班兩天，因此，當某些委員會要在他的上班日開會，若議程重要，他會請假出席，若議程不重要，他就不會出席了。遇上大會投票，他也是看議題的性質，若對他的界別不重要，他會寧願留在議員辦公室處理其他事務，看電視直播，而不介意在他的投票紀錄上出現「不在現場」的紀錄。
