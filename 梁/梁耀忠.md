'''梁耀忠'''（{{lang-en|Leung Yiu-chung}}；{{bd|1953年|5月19日|catIdx=L梁}}），生於[[香港|香港]]，籍貫[[廣東|廣東]][[中山市|中山]]，[[香港立法會|香港立法會]][[區議會（第二）功能界別|區議會（第二）功能界別]]議員，現任[[葵青區|葵青區]][[葵芳邨|葵芳選區]]議員，[[街坊工友服務處|街坊工友服務處]]執行監督，香港[[泛民主派|泛民主派]]成員，他曾五次當選[[新界西選區|新界西選區]][[地區直選|地區直選]]立法會議員，曾任[[香港仔工業學校|香港仔工業學校]]教師。因不喜歡主持會議，自出任立法會議員以來，鮮有擔任委員會主席<ref name="rejectChair">{{cite news|url=https://topick.hket.com/article/2101729/%E6%A2%81%E8%80%80%E5%BF%A0%E6%93%94%E4%BB%BB%E8%AD%B0%E5%93%A123%E5%B9%B4%E5%A0%85%E6%8B%92%E4%B8%BB%E6%8C%81%E6%9C%83%E8%AD%B0%E4%B9%8B%E8%AC%8E%E3%80%80|title=梁耀忠擔任議員23年堅拒主持會議之謎|publisher=香港經濟日報 - TOPick|date=2018-06-25}}</ref>，包括在2016年香港立法會主席選舉中，因無法主持會議，導致[[梁君彥|梁君彥]]得以在混亂的情況下成為立法會主席。

2016年，他代表[[街工|街工]]及[[泛民主派|泛民主派]]，參加[[2016年香港立法會選舉|2016年香港立法會選舉]]的[[區議會（第二）功能界別|區議會（第二）功能界別]]，最終成功當選。

==參政經歷==
===新青學社===
[[民運人士|民運人士]][[劉山青|劉山青]]於1975年成立工人夜校[[新青學社|新青學社]]，梁耀忠在1978年從[[英國|英國]]回港後加入成為當中的一名義工。其後，劉山青於1981年在中國大陸被捕，判以[[反革命|反革命]]罪名，梁耀忠繼承新青學社，並改組為「[[街坊工友服務處|街坊工友服務處]]」。

===葵青七子===
適逢香港[[26座問題公屋醜聞|26座問題公屋醜聞]]，梁耀忠帶領受災最重的[[葵芳|葵芳]]廉租屋村居民調查，揭發承建商偷工減料，導致葵芳村第五座需要即時拆卸，而葵芳村其餘十幢大廈均需於樓齡未足廿年之下重建。梁因而在1985年當選葵青區議會葵芳及[[葵興|葵興]]選區議員。與當時同期當選的[[李永達|李永達]]、[[單仲偕|單仲偕]]、梁廣昌等並稱「葵青七子」。梁是[[葵青區|葵青區]]和[[新界西|新界西]]資深的社區領袖，自1985年起當選[[區議員|區議員]]後，在葵芳選區連任，從無間斷；[[2003年香港區議會選舉|2003年香港區議會選舉]]，更以全港最高票連任，成為票王。

===參選立法局===
1991年首度參選[[香港立法局|香港立法局]]落敗，1995年通過立法局[[新九組|新九組]]的紡織及製衣界當選。為突顯代表基層的形象，他甚少穿西裝出席會議。

===成為首名被逐離場的議員===
1996年，在立法局會議中以一句「臭罌出臭草」諷刺回歸後的行政長官選舉不民主及小圈子，被時任立法局主席[[黃宏發|黃宏發]]驅逐揚名，成為立法機關歷史上，首名被驅逐出場的立法局議員。

===重返立法會===
1997年6月30日，他隨其他民主派議員一同「落車」。

[[1998年香港立法會選舉|1998年香港立法會選舉]]，成功當選重返[[香港立法會|香港立法會]]，並在2000年、2004年、2008年、2012年的立法會選舉中，連任[[新界西|新界西]][[地區直選|地區直選]]立法會議員。

[[2003年香港區議會選舉|2003年香港區議會選舉]]，街坊工友服務處只派出四人在葵青區的[[石蔭|石蔭]]、[[安蔭|安蔭]]、[[葵興|葵興]]及[[葵芳|葵芳]]參選，均告勝出，其中同時為民主派前綫執委，在安蔭選區大勝的梁永權，當選葵青區議會副主席。梁並在[[2004年香港立法會選舉|2004年香港立法會選舉]]中，夥拍葵青區議會[[石蔭|石蔭]]選區區議員[[尹兆堅|尹兆堅]]參選立法會，但只有梁當選。數年後，尹兆堅退出街工，加入民主黨。

2010年11月，傳媒報導退出了公民黨的[[張超雄|張超雄]]，有意和他、[[何秀蘭|何秀蘭]]及[[李卓人|李卓人]]合組[[工黨_(香港)|前進聯盟]]<ref>{{cite web |url=http://www.hkdailynews.com.hk/news.php?id=130345 |title=存档副本 |accessdate=2010-11-03 |deadurl=yes |archiveurl=https://archive.is/20120903203213/http://www.hkdailynews.com.hk/news.php?id=130345 |archivedate=2012-09-03 }}</ref>。其後，在街工會員大會上，大比數否決街工加入工黨，而梁耀忠也被會員否決以個人身份加入。梁耀忠會後對結果坦言「愕然」，擱置加入工黨的計劃。

[[2016年香港立法會選舉|2016年香港立法會選舉]]，作為少數兼任區議員的泛民主派立法會議員，他代表泛民主派參加俗稱「超級區議會」的區議會 (第二) 功能界別。他代表泛民主派中偏向基層及勞工權益的聲音，最終成功當選。

== 家庭 ==
前妻[[黎紹珍|黎紹珍]]亦是前社會運動中人，自中學時代已是活躍份子，曾是激進[[托洛茨基主义|托洛茨基主义]]團體[[革命马克思主义者同盟|革命马克思主义者同盟]]成員。黎亦曾是立法會議員[[何秀蘭|何秀蘭]]的助理、泛民主派政黨前綫的秘書，現已淡出民主及社會運動。梁與黎育有一子<ref>{{Cite news|url=https://hk.on.cc/hk/bkn/cnt/news/20160523/bkn-20160523161106157-0523_00822_001.html|title=政Whats噏：父子情淡　梁耀忠嘆工時長拖累|publisher=東網|date=2016-05-23}}</ref><ref>《香港特區名人錄》，第255頁，ISBN 62-8017-128</ref>。

== 爭議 ==
===「五區公投」立場===
2010年梁耀忠多次與「[[五區公投|五區公投]]」民主運動中和社民連開火<ref>[http://www.youtube.com/watch?v=Aq3plwD74BY 五區公投 長毛火拚梁耀忠]</ref><ref>[http://paper.wenweipo.com/2009/12/04/HK0912040025.htm 長毛梁耀忠為總辭互罵]</ref>，並曾建議兩黨取消五區辭職行動，受到網民極大反響<ref>[http://www.hkreporter.com/talks/viewthread.php?tid=868090 梁耀忠建議兩黨取消五區辭職行動]</ref>。梁耀忠最終在2010年政改方案中投下反對票。

=== 常缺席投票 ===
以民生為政綱的梁耀忠，在立法會房屋事務委員會投票以權力及特權條例調查鉛水事件時，與[[涂謹申|涂謹申]]及[[陳偉業|陳偉業]]一同缺席投票，被人質疑是他們導致動議被否決。然而，房屋事務委員會的投票並無法定約束力，即使投票通過亦要再交由立法會大會審議。<ref>http://news.memehk.com/posts/8980</ref>其後，民主派在立法會大會中提出引用特權法，成立獨立專責調查委員會調查公屋食水含鉛超標事件，梁耀忠投下贊成票，但議案在建制派反對下被否決。

=== 2016年立法會主席選舉風波 ===
在2016年10月12日立法會首次會議，按照議事規則，由立法會最資深的議員負責主持立法會主席的選舉。由於最資深的[[涂謹申|涂謹申]]獲[[非建制派|非建制派]]推舉出選立法會主席，因此由第二資深的梁耀忠負責主持會議，梁耀忠在過去擔任議員的21年間，因不喜歡擔任委員會主席，2016年前從未擔任過任何委員會的主席及主持會議的經驗。<ref name="rejectChair"/>

不過，獲建制派推舉的[[梁君彥|梁君彥]]因被揭擁有[[英國國籍|英國國籍]]，參選[[立法會主席|立法會主席]]的資格成疑，但梁君彥在選舉主席當日已展示其已放棄英國國籍的官方文件。首次會議當日，三位非建制派議員[[游蕙禎|游蕙禎]]、[[梁頌恆|梁頌恆]]、[[姚松炎|姚松炎]]宣誓無效，而被立法會秘書處剝奪參與投票的權利（2012年[[黃毓民|黃毓民]]同樣無進行有效宣誓，但獲容許參與投票。但兩者情況有所不同，立法會秘書長當時並無即時表示黃毓民宣誓無效，而是事後由立法會主席[[曾鈺成|曾鈺成]]要求黃毓民重新宣誓。相反，今次事件中，監誓的秘書長即時表示無法為三位議員監誓）。當時梁耀忠不滿有關決定，但按議事規則，他必須在議員宣誓後隨即進行投票。因此他決定離席抗議，不再主持主席投票。席上第三資深的建制派[[石禮謙|石禮謙]]按議事規則代他主持會議。石禮謙其後無視非建制派議員的不滿，決定啟動投票，[[梁君彥|梁君彥]]在[[非建制派|非建制派]]離場及撕毀選票抗議的情況下，以38票當選主席
<ref>[http://www.hk01.com/%E6%B8%AF%E8%81%9E/48014/-%E7%AB%8B%E6%9C%83%E9%96%8B%E9%91%BC-%E7%9F%B3%E7%A6%AE%E8%AC%99%E5%BC%B7%E8%A1%8C%E5%95%9F%E5%8B%95%E6%8A%95%E7%A5%A8-%E6%A2%81%E5%90%9B%E5%BD%A538%E7%A5%A8%E7%95%B6%E9%81%B8%E4%B8%BB%E5%B8%AD-%E5%A4%9A%E5%9C%96- 石禮謙強行啟動投票　梁君彥38票當選主席] 2016-10-12 [[香港01|香港01]]</ref>。

根據立法會議事規則，當時主持會議的梁耀忠作為代主席事實上是擁有主席的權力。事後，大批網民在其Facebook專頁留言，批評他缺乏處事能力，並認為背負逾30萬選票的他無盡議員的責任，呼籲他應立即辭去立法會議員職務。梁耀忠事後兩度致歉，但指立法會秘書處誤導他指當時主持會議的他並沒有主席的權力。<ref>[http://hk.apple.nextmedia.com/news/art/20161014/19800521 主持選立會主席突劈炮　受猛烈批評 梁耀忠道歉　稱遭秘書處出賣] 2016-10-14 [[蘋果日報_(香港)|蘋果日報 (香港)]]</ref>對此，立法會前主席[[曾鈺成|曾鈺成]]表示，主持星期三大會會議的梁耀忠和石禮謙權力等同主席，但是否有權判斷未完成宣誓的議員能否參與會議，議事規則並未清楚界定，新一屆議事規則委員會需要釐清。<ref>[https://hk.news.yahoo.com/%E6%9B%BE%E9%88%BA%E6%88%90-%E4%B8%BB%E5%B8%AD%E9%81%B8%E8%88%89%E4%B8%BB%E6%8C%81%E6%AC%8A%E5%8A%9B%E7%AD%89%E5%90%8C%E4%B8%BB%E5%B8%AD-112626204.html 曾鈺成：主席選舉主持權力等同主席] 2016-10-13 now新聞</ref>
