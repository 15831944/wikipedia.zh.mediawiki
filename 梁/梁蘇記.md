'''梁蘇記遮廠'''由梁智華於1885年在[[清朝|中國]][[廣州|廣州]]惠愛西路（今[[中山路_(廣州)|中山六路]]）創辦，並以純正鋼骨及永久包修為口號及賣點，是[[香港品牌|香港品牌]][[老字號|老字號]]。

== 歷史 ==
梁智華曾以[[收買佬|收買佬]]為業，別字蘇，[[粵語|粵語]]稱「傘」為「遮」，「梁蘇記遮廠」因而為名。

梁智華1885至1925年在廣州惠愛西路開設遮廠，期間在1920年於[[高第街|高第街]]67號開設第一支店，黑長遮此時開始在中國流行。

20世紀，梁蘇記遮廠業務曾經一分為三（四房長子負責當時[[中國大陸|中國大陸]]及[[澳門|澳門]]的業務，六房則在[[香港島|香港島]]經營，五房則在[[九龍|九龍]]及[[新界|新界]]經營。）直至1986年香港島及大陸所有業務結束。1994年持有九龍及新界業務的第五房後人於[[深水埗|深水埗]][[西九龍中心|西九龍中心]]開業。

1941年，梁蘇記於[[港島|港島]][[德輔道中|德輔道中]]122號開業，當時的縮骨遮只為高尚人士所擁有。1944年，在[[九龍|九龍]][[上海街|上海街]]321號開設香港第二間分店，亦為現時五房所擁有。1960至70年代，梁蘇記遮廠開遍[[旺角|旺角]]、[[青山道|青山道]]、[[新蒲崗|新蒲崗]]、[[沙田|沙田]]，到現時只剩下[[深水埗|深水埗]]店。

== 出品 ==
梁蘇記遮廠所製造的[[雨傘|雨傘]]現在依然由[[人手|人手]]生產。

== 相關作品 ==
梁蘇記遮廠的事述於1995年被[[杜國威|杜國威]]改編成以《人間有情》為名的[[電影|電影]]及[[舞台劇|舞台劇]]。

== 趣聞 ==
#二十世紀梁蘇記曾經容許製造[[豬|豬]]鬃毛[[牙刷|牙刷]]的「梁新記」（[[佛山|佛山]]創立，廣州發展）把牙刷放於廣州店內寄賣，但後來以訛傳訛地變成梁蘇記。「梁蘇記牙刷—一毛不拔」是現在仍然是佛山、廣州和香港廣泛流傳的[[歇後語|歇後語]]。
#梁蘇記出品的雨傘素被[[國術|國術]]界當作[[中国武术器械列表#奇门兵器|奇門]][[冷兵器|兵器]]來使用，相關的套路有「[[洪拳|洪佛]]追魂傘」（又稱[[洪拳|洪家]]飛鴻傘（南傘）），電影《[[黃飛鴻_(1991年電影)|黃飛鴻]]》中，[[李連杰|李連杰]]用來作[[武器|武器]]使用的雨傘，正是梁蘇記的出品。
