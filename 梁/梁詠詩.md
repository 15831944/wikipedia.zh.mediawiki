'''梁詠詩'''（{{lang-en|Jenny Leung Wing-sze}}），是一名一直居於[[筲箕灣|筲箕灣]]的中學[[教師|教師]]，有逾十年教學經驗，畢業於[[香港中文大學|香港中文大學]]中文系<ref>{{cite news |language = 中文 |author = 懷鈺 |url = https://hk.entertainment.appledaily.com/enews/realtime/article/20160903/55591007 |title = 【老師的吶喊】80後Miss倡人心教育 |publisher = 蘋果日報 |date = 2016-09-03}}</ref>。

==從政經歷==
梁詠詩於[[2015年區議會選舉|2015年區議會選舉]](筲箕灣區)首次參選。<ref>{{cite news |language = 中文 |author = 陳凱晴 |url = http://www.inmediahk.net/node/1038477 |title = 「筲箕灣人」梁詠詩拒絕建制壟斷 盼將聲音帶入議會 |publisher = 獨立媒體 |date = 2015-10-27}}</ref>當時人口估算約13,250人，其中有6,413位是選民。這屆選情非常激烈，[[民建聯|民建聯]][[勞鍱珍|勞鍱珍]]宣布本屆不參選。由徒弟[[林心亷|林心亷]]參選。而[[筲箕灣民生關注組|筲箕灣民生關注組]]的梁詠詩及「疑似假傘兵」[[新青年聯盟|新青年聯盟]]的[[潘詠妍|潘詠妍]]於同日報名參選。筲箕灣民生關注組梁詠詩得票1,429票，民建聯林心亷以1,444票，梁詠詩以15票之差僅敗<ref>{{cite news |language = 中文 |url = https://news.mingpao.com/pns/dailynews/web_tc/article/20151124/s00002/1448301874939 |title = 最心悒落選者只輸3票 |publisher = 明報 |date = 2015-10-24}}</ref>，而「疑似假傘兵」新青年聯盟的潘詠妍就有107票。

敗選後，雖然不少傘兵已經退出政壇，梁詠詩依然從事社區工作<ref>{{cite news |language = 中文 |author = 袁慧妍 | url = http://nextplus.nextmedia.com/news/spot/20151203/308007 |title = 【我只輸幾票】 |publisher =  壹周刊 |date = 2015-12-03}}</ref>，如繼續「漂書」<ref>{{cite news |language = 中文 |url = https://www.youtube.com/watch?v=1wDFa0kLbqg |title = 筲箕灣 (C05) 梁詠詩老師：漂書、社區、教育 |publisher =  香港電台 |date = 2015-04-04}}</ref>及關注區內事務<ref>{{cite news |language = 中文 |author = 鄭啟源 | url = https://hk.news.appledaily.com/local/daily/article/20170215/19928670 |title = 區議員成功爭取綠燈多1秒 |publisher =  蘋果日報 |date = 2017-02-15}}</ref>，定期出版及發放社區資訊「貓紙」。[[2016年香港立法會選舉|2016年立法會選舉]]前，曾被[[熱血公民|熱血公民]]支持者在網上攻擊，因梁詠詩在網上公開批評熱血公民候選人[[鄭錦滿|鄭錦滿]]團隊的街站「遲到早退」<ref>{{cite news |language = 中文 |author = 甘瑜 |url = http://paper.wenweipo.com/2016/05/23/HK1605230043.htm |title = 網議政事：網批鄭錦滿無落區 梁詠詩辣慶「熱狗粉」 |publisher = 文匯報 |date = 2016-05-23}}</ref>。

==與電車的關係==

梁詠詩曾提及童年住[[筲箕灣電車總站|筲箕灣電車總站]]附近<ref>{{Cite web|url=https://www.facebook.com/skwjenny/photos/a.822901111116467.1073741827.822748974465014/977949978944912/?type=3&theater|title=是夜與筲箕貓看叮叮聽叮叮。|accessdate=2015-11-14|author=|date=|publisher=}}</ref>，亦曾開街站及網上發起「救救[[電車|電車]] 保護叮叮」，曾拍片堅決反對刪除中環至金鐘路段的電車軌，接近7000人次瀏覽量，近70個分享，組織街坊向城規會表達意見。期後獲電車公司邀請，與筲箕灣街坊參觀電車車廠<ref>{{Cite web|url=https://www.skwjenny.com/single-post/tramdepot|title=與筲箕貓同遊電車廠之旅|accessdate=2017-10-21|author=|date=|publisher=}}</ref>。
