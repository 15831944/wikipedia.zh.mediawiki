'''梁平区'''位于重庆市东北部，目前是[[重庆市|重庆市]]下辖的一个[[市辖区|市辖区]]，原名[[梁山縣（四川省）|梁山縣]]、梁平县，是国家可持续发展实验区、国家生态文明先行示范区、国家循环经济示范区、全国农村改革试验区<ref>{{Cite web|url=http://lp.cq.gov.cn/html/zjlp/|title=走进梁平|accessdate=2018-07-15|work=lp.cq.gov.cn}}</ref>。[[渝万高速公路|渝万高速公路]]与[[318国道|318国道]]、[[达万铁路|达万铁路]]通过本区。幅员面积1892平方公里，总人口92万人。

== 历史沿革 ==
梁平区原名梁山县，以境内高粱山为名，[[西魏|西魏]][[元钦|元欽]]二年（公元553年）置，先后隶[[南州_(北周)|南州]]、[[巴东郡|巴东郡]]、[[信州_(南梁)|信州]]、南浦州、[[夔州|夔州]]、[[浦州|浦州]]、[[万州_(重庆)|万州]]。[[北宋|北宋]][[开宝|开宝]]三年（公元970年）设[[梁山军|梁山军]]（元时升为[[梁山州|梁山州]]，明初废），领梁山县，隶[[夔州路|夔州路]]。清时改隶[[忠州直隸州|忠州直隶州]]。<ref name=":0">{{Cite web|url=http://lp.cq.gov.cn/html/zjlp/lsyg/|title=历史沿革|accessdate=2018-07-15|author=|date=|publisher=梁平区政府网}}</ref>

[[中华人民共和国|中华人民共和国]]成立后，梁山县隶[[川东行政区|川东行政公署]][[大竹專區|大竹专区]]。1952年12月3日因與[[山東省|山東省]][[梁山县|梁山縣]]（1949年9月改[[平原省|平原省]][[昆山县_(山东)|昆山縣]]為梁山縣）縣名重複，以縣境高梁山麓有平壩，更名為梁平縣。1953年划归[[万县专区|万县专区]]至重庆直辖。<ref name=":0" />2016年11月，经国务院批准，撤销梁平县，设立重庆市梁平区<ref>{{cite news|url=http://cq.cqnews.net/html/2016-12/02/content_39705189.htm|title=重庆调整部分行政区划 梁平、武隆撤县设区|date=2016年12月2日|newspaper=华龙网|deadurl=yes|archiveurl=https://web.archive.org/web/20161202103320/http://cq.cqnews.net/html/2016-12/02/content_39705189.htm|archivedate=2016年12月2日}}</ref>。

== 地方特色 ==
=== 文物景点 ===
*川东名刹[[双桂堂|双桂堂]]，为[[禅宗|禅宗]]寺庙，被誉为“蜀中丛林之首”、“西南禅宗祖庭”，入选[[汉族地区佛教全国重点寺院|汉族地区佛教全国重点寺院]]、[[第七批全国重点文物保护单位|第七批]][[全国重点文物保护单位|全国重点文物保护单位]]
=== 民间艺术 ===
梁平传统有梁平“三绝”之称，即竹帘、年画、灯戏。<ref>{{cite web |title=梁平木板年画 |url=http://lp.cq.gov.cn/html/zjlp/fyxm/Index_168.shtml |publisher=梁平区政府网 |accessdate=2018-07-20}}</ref>

*[[梁平癞子锣鼓|梁平癞子锣鼓]]，一种汉族民间器乐，入选第一批[[中国国家级非物质文化遗产列表|国家级非物质文化遗产名录]]
*[[灯戏|梁山灯戏]]，入选第一批国家级非物质文化遗产名录
*[[梁平木版年画|梁平木版年画]]，入选第一批国家级非物质文化遗产名录
*[[梁平竹帘|梁平竹帘]]，入选第一批国家级非物质文化遗产扩展项目名录
*[[梁平抬儿调|梁平抬儿调]]，又称抬工号子，入选第二批国家级非物质文化遗产名录

=== 特产 ===
*梁平柚：即梁平平顶柚，与[[广西|广西]]的[[沙田柚|沙田柚]]、[[福建|福建]]的[[文旦柚|文旦柚]]齐名，系全国三大名柚之一
*袁驿豆干
*梁平张鸭子：一种卤烤制品，以干、香、瘦特色闻名

== 名人 ==
*[[孟浩然_(四川省梁山縣國大代表)|孟浩然]]，[[川军|川軍]]軍長，四川省[[梁山縣（四川省）|梁山縣]]選出之[[第一屆國民大會代表|第一屆國民大會代表]]。

*范纪曼（1906—1990），[[中国共产党|中国共产党]]，曾经创建、担任梁山县特支、县委第一任书记，长期从事地下工作，曾任国民党国防部二厅任代理少将专员，专搞国际情报。 中华人民共和国成立后，担任[[上海戏剧学院|上海戏剧学院]]教授，1955年因受[[潘汉年|潘汉年]]、[[杨帆|杨帆]]冤案株连而被捕入狱，中共十一届三中全会后平反昭雪，1990年12月在上海病逝。

*[[谢世杰|谢世杰]]（1934年－），中华人民共和国政治人物，曾任[[中共四川省委|中共四川省委]]常委、[[四川省|四川省]]副省长、四川省委副书记、书记、[[四川省人民代表大会常务委员会|四川省人大常委会]]主任。
