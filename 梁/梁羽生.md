'''梁羽生'''（本名'''陳文統'''，{{bd|1924年|4月5日|2009年|1月22日||catIdx=Liang梁羽生}}），香港著名武俠小說家，著有[[武俠小說|武俠小說]]三十五部，另有散文、联话若干。1987年移居[[澳洲|澳洲]][[悉尼|悉尼]]。2009年1月22日病逝悉尼。梁羽生被誉为「新派武俠小說」的開山鼻祖。<ref name="multiplesh45 " /><ref name="multiple561 " >{{cite web |url=http://news.sohu.com/20090126/n261953652.shtml |title=“武侠小说大师”梁羽生1月22日悉尼去世 |accessdate=2009年1月26日 |work=|publisher=搜狐 |language=zh-hans}}</ref><ref name="multiple123 " />

==生平==
陳文統1924年生于[[广西省|广西]][[蒙山县|蒙山县]]文圩镇屯治村，出身於書香[[門第|門第]]，熟讀[[古文|古文]]、擅於[[對聯|對聯]]，8歲能背誦《[[唐詩三百首|唐詩三百首]]》。在[[广西|广西]][[桂林中学|桂林中学]]读完高中，在中学时就喜欢写词。因日军侵扰返乡，适逢数位粤籍学者避难[[蒙山|蒙山]]，遂依礼拜[[简又文|简又文]]为师。而[[太平天国|太平天国]]史专家[[简又文|简又文]]和以[[敦煌学|敦煌学]]及诗书画著名的[[饶宗颐|饶宗颐]]都在他家里住过，梁羽生曾向他们学习历史和文学。后随师返穗，考入[[嶺南大學_(廣州)|嶺南大學]]經濟系，專修國際經濟。

1949年定居[[香港|香港]]，经校长介绍，于《[[大公报|大公报]]》任副刊助理[[编辑|编辑]]，迅即提正，并成为社评委员会之成员。次年底，调入附属《大公报》的《[[新晚报|新晚报]]》工作。<ref name="multiple561 " />

1954年1月17日，香港白鶴派掌門人[[陳克夫|陳克夫]]與太極派掌門人[[吳公儀|吳公儀]]，因互說己派武功最好，遂在澳門（因香港嚴禁私鬥）擂台比武<ref>{{cite web |url=https://www.youtube.com/watch?v=xqvfuCdW9nk |title=1954 陳吳大戰: (太極拳吳公儀) vs 白鶴拳(陳克夫) 足版本 |publisher=YouTube }}</ref>。這新聞在當時相當轟動，《新晚報》總編輯[[羅孚_(作家)|羅孚]]遂要求陳文統於翌日開始寫武俠小說於《新晚報》連載。陳文統未寫過小說，但愛看武俠小說，尤喜[[白羽|白羽]]，唯有以白羽為師，取筆名「梁羽生」，以吳陳的打擂台故事及太極門派為骨架，寫出處女作《[[龍虎鬥京華|龍虎鬥京華]]》，成為首部「新派武俠小說」。<ref>{{ cite news |url=http://paper.wenweipo.com/2015/05/19/OT1505190004.htm |title=琴台客聚：梁羽生的處女作 |date=2015-05-19 |publisher=文匯報 |author=黃仲鳴  }}</ref>

陳文統此後以「梁羽生」筆名笔耕不缀，至1983年《[[武当一剑|武当一剑]]》止，共连载武侠小说33部，總字數達一千餘萬。又《[[絕塞傳烽錄|絕塞傳烽錄]]》与《[[幻剑灵旗|幻剑灵旗]]》出版时均被一分而二（分別為《彈指驚雷》與《絕塞傳烽錄》，以及《劍網塵絲》與《幻剑灵旗》） ，《[[武林三绝|武林三绝]]》仍在修订之故，今传者34部。封笔前后并曾推出部分小说的修订本，但均未得以出版。

他的作品多次被改編成[[電影|電影]]或[[電視劇|電視劇]]，自認《[[萍蹤俠影錄|萍蹤俠影錄]]》、《[[女帝奇英傳|女帝奇英傳]]》及《[[雲海玉弓緣|雲海玉弓緣]]》三書是代表作。

1987年移居澳大利亚。他是中国作家协会会员，也曾受聘为深圳市楹联学会的名誉会长。<ref name="multiple561 " /> 1994年70歲時在澳大利亚改信基督教。<ref>[http://www.christiantimes.cn/news/12192/%E7%BB%86%E6%95%B0%E4%B8%89%E4%BD%8D%E6%88%90%E4%B8%BA%E5%9F%BA%E7%9D%A3%E5%BE%92%E7%9A%84%E4%BC%A0%E5%A5%87%E7%94%B7%E4%BD%9C%E5%AE%B6%EF%BC%9A%E6%9F%8F%E6%9D%A8%E3%80%81%E5%80%AA%E5%8C%A1%E3%80%81%E6%A2%81%E7%BE%BD%E7%94%9F 细数三位成为基督徒的传奇男作家：柏杨、倪匡、梁羽生]</ref>
2004年11月30日，陳文統獲[[香港嶺南大學|香港嶺南大學]]頒授榮譽文學博士學位。<ref>{{cite web |url=https://www.ln.edu.hk/news/20040817/01 |title=嶺南大學頒授榮譽博士學位予四位傑出人士 |date=2004-08-17 |publisher=香港嶺南大學 }}</ref> 2006年12月回[[香港|香港]]參加[[天地圖書|天地圖書]]出版公司30周年活動時就曾突然中風，手腳活動不便。<ref name="multiple561 " /> 及後一直在陳秉達療養院中療養<ref name="A1688">{{cite web |url=http://www.1688.com.au/news/200801/hotnet16590.shtml |title=存档副本 |accessdate=2009-01-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20090129181549/http://1688.com.au/news/200801/hotnet16590.shtml |archivedate=2009-01-29 }}</ref>，2009年1月22日在[[悉尼|悉尼]]去世，享年85岁<ref name="A1688" />。

===筆名===
陳文統以「梁羽生」為筆名，筆名以「梁」為姓，因為[[魏晉南北朝|魏晉南北朝]]時期的南朝為「[[南朝宋|宋]]、[[南朝齊|齊]]、[[南朝梁|梁]]、[[南朝陳|陳]]」，「梁」在「陳」前，是以「繼承梁陳文統」自勉，而「羽生」則是「羽客」的意思。<ref>{{ cite book |url= |title=《他還活著: 澳華文壇掠影. 第一集》 |date=2010 |publisher=秀威資訊 |author=何與懷 |location=台灣| pages=頁160  |ISBN=9789862215845 }} </ref>

==武侠小说==

2009年，梁羽生在[[悉尼|悉尼]]荣获[[澳大利亚|澳大利亚]][[华人|华人]][[文化|文化]]团体联合会颁发的“[[澳华文化界终生成就奖|澳华文化界终生成就奖]]”。<ref name="multiplesh45 " />

2018年，[[南方报业传媒集团|南都報系]]、梁羽生故鄉廣西[[蒙山县|蒙山县]]人民政府創辦[[梁羽生文學獎|梁羽生文學獎]]，獎勵各種不同類型的大眾小說。<ref>[http://www.xinhuanet.com/book/2018-04/28/c_129861512.htm 首届“梁羽生文学奖”启动，网络文学出版物也可以参选]</ref>

== 评价 ==
梁羽生被誉为“新式武侠小说开山鼻祖”。<ref name="multiplesh45 " >{{cite web |url=http://news.qq.com/a/20090126/000355.htm |title=梁羽生荣获澳大利亚澳华文化界终生成就奖 |accessdate= 2009年1月26日 |work=|publisher=腾讯网 |language=zh-hans}}</ref> 六七十年代的读者将梁羽生和[[金庸|金庸]]并称武侠两座高峰：“金梁并称，一时瑜亮”。<ref name="multiple123 " >{{cite web |url= http://www.infzm.com/content/23229|title=梁羽生葬礼举行 金庸送挽联“自愧不如” |accessdate=2009-02-01 |work=|publisher=南方周末 |language=zh-hans}}</ref> 梁羽生所創作的《[[龍虎鬥京華|龍虎鬥京華]]》被公認為是新武俠小說之始，開創了武俠小說的一代新風，後以《[[萍蹤俠影錄|萍蹤俠影錄]]》等名著為新派武俠綻放了文學情思上的異彩。梁羽生曾經這樣評價自己的武俠創作地位：「開風氣也，梁羽生，發揚光大者，金庸。」 <ref name="multiple123 " />

梁羽生擅长于在不被认为是正统的武侠小说中，渗入更多的中国传统文化艺术。他喜在小说中穿插中国传统诗文，融和文學、歷史、詩詞、聯語於一爐，迎合了中国传统文化的审美觀。除武侠小说之外，梁羽生亦擅写随笔杂谈，[[对联|联语]][[棋话|棋话]]皆广受好评。<ref name="multiple123 " />

武侠小说大师[[金庸|金庸]]在梁羽生的挽联中写道<ref name="multiple123 " />：
{{cquote 3|
悼梁羽生兄逝世：<br />
同行同事同年大先辈；亦狂亦侠亦文好朋友<br />
自愧不如者：同年弟金庸敬挽
}}

著名导演[[张纪中|张纪中]]：“梁羽生文学最大的特点就是历史感很厚重，应该说他笔下武侠世界中的人都生活在真实的历史背景和社会中，相比之下，[[金庸|金庸]]的作品往往表现的是生活在武侠世界里的侠客，他们过着与世隔绝的生活。”<ref name="multiple123 " /><br />
