'''梁毅苗'''，蒙古名'''纳迪亚'''，出生于[[内蒙古自治区|内蒙古自治区]]，毕业于[[中国传媒大学|中国传媒大学]]，为[[中国中央电视台体育频道|中国中央电视台体育频道]]主持人。<ref name="fish " >{{cite web |url= http://sports.sina.com.cn/o/2010-07-18/13525110716.shtml|title=央视体育女主播梁毅苗成长之路 12年感恩之心 |publisher= 新浪网|accessdate= 2010年7月18日 |work= |language =}}</ref>

== 简介 ==
梁毅苗出生于传媒世家，父母、弟弟都是新闻工作者，她的名字是她外婆取的，意思是坚强、茁壮成长。<ref name="fish " /> 1994年，梁毅苗从[[银川第一中学|银川第一中学]]毕业，同年考入[[中国传媒大学|中国传媒大学]]。<ref name="bird " /> 1997年，梁毅苗到[[中国中央电视台体育频道|中国中央电视台体育频道]]实习，1998年正式上岗。<ref name="bird " /> 现在主持《[[体坛快讯|体坛快讯]]》节目。<ref name="bird " >{{cite web |url= http://www.zhuchijie.com/show_93_z7_t1.html|title= 梁毅苗|publisher=主持界 |accessdate=2011年 |work= |language =}}</ref>

== 个人生活 ==
梁毅苗喜欢[[羽毛球|羽毛球]]运动，是明星羽毛球队的队员之一。<ref name="sheep " >{{cite web |url= http://sports.sohu.com/20100721/n273648917.shtml|title=走近央视美女主播梁毅苗 邻家大姐姐惹人爱 |publisher=搜狐网 |accessdate=2010-07-21 |work= |language =}}</ref>
