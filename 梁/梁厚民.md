'''梁厚民'''（{{bd|1940年||2017年|9月7日}}）[[河北|河北]][[玉田|玉田]]人，[[中华人民共和国|中华人民共和国]]曲艺演员，主要擅长[[快板书|快板书]]<ref name=shishi>{{Cite web |url =http://news.ifeng.com/a/20170909/51929795_0.shtml  |title = 快板名家梁厚民逝世 |author =  |date =2017-09-09  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref name=qushi>{{Cite web |url = http://news.163.com/17/0909/01/CTRTPUNE000187VI.html |title =快板《奇袭白虎团》作者梁厚民去世  |author =  |date =2017-09-09  |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
梁厚民自幼喜爱文艺，曾从事[[笛子|笛子]]、[[箫|箫]]独奏，以及民族舞蹈、[[京剧|京剧]]、[[评剧|评剧]]、[[话剧|话剧]]、[[山东快书|山东快书]]、[[相声|相声]]、[[快板|快板]]、快板书等的表演。1960年，考入[[北京矿业学院|北京矿业学院]]地质系。毕业后不顾家人反对，投身艺术界，学习快板书。梁厚民多次骑[[自行车|自行车]]从北京去[[天津|天津]]，向快板书大师[[李润杰|李润杰]]学习。他先后在北京农村文化工作队、北京文工团担任演员。1969年，在[[北京曲艺团|北京曲艺团]]任快板书演员。1985年，拜[[高凤山|高凤山]]为师。1963年起，以快板书为主要擅长曲种。他是一级演员、[[中国曲艺家协会|中国曲艺家协会]]会员、北京曲艺家协会理事、[[北京大学中文系|北京大学中文系]]兼职教师、北京对外友协理事，曾任[[崇文区|崇文区]]政协常委<ref name=shishi/><ref name=qushi/>。

梁厚民曾获全国优秀短篇曲艺创作奖、中青年优秀表演奖、北京市文艺调演荣誉奖、北京市新编曲艺调演获奖并获[[中共北京市委|中共北京市委]]表彰名册。2009年，获北京曲艺家协会授“杰出成就艺术家”称号<ref name=shishi/><ref name=qushi/>。

梁厚民有相声演员[[李菁|李菁]]等众多弟子。2013年，54岁的演员[[梁天_(北京)|梁天]]拜师梁厚民，成为梁厚民的关门弟子<ref name=shishi/><ref name=qushi/>。

2017年9月7日，梁厚民在北京因[[心脏病|心脏病]]突发逝世，享年76岁<ref name=shishi/><ref name=qushi/>。

==作品==
梁厚民最知名的快板书节目是他自己创作的《[[奇袭白虎团|奇袭白虎团]]》，这是梁厚民根据[[山东省京剧团|山东省京剧团]]同名京剧剧本改编，他在其中加入了不少京剧动作如旋子、小翻儿等等，该作品是梁厚民的成名作。梁厚民一直坚持创作，由他主演主创的节目主要有《奇袭白虎团》、《大闹药王庙》、《炒电车》、《犟姑娘》、《争夺》、《百年大计》、《[[西安事变|西安事变]]》、《幸福属于谁》、《倔闺女》、《人间真情》等快板书60多段，都在广播电台录音、在电视台录像，并录制为唱盘<ref name=shishi/><ref name=qushi/>。
