'''梁相宜翻白眼事件'''发生于2018年3月13日，[[第十三届全国人民代表大会第一次会议|第十三届全国人民代表大会第一次会议]]“部长通道”记者采访时。[[上海广播电视台|上海广播电视台]][[第一财经|第一财经]]蓝衣女记者梁相宜在红衣女记者、“外媒”“[[美国全美电视台|美国全美电视台]]”执行台长[[张慧君|张慧君]]提问时翻[[白眼|白眼]]，这一瞬间被[[中国中央电视台|中国中央电视台]]直播而[[网络爆红|网络爆红]]，被美国《[[纽约时报|纽约时报]]》评价为“[[史诗|史诗]]级白眼”<ref name ="NYT">{{cite news |author=孟宝勒 |title=史诗级白眼：两会记者夸张表情引爆中国互联网 |url = https://cn.nytimes.com/china/20180314/china-eye-roll-liang-xiangyi/ |website = 纽约时报中文网 }}</ref>。

==相关人物==
'''梁相宜'''（1986年8月28日－），青岛市人，中华人民共和国资深政经、财经条线记者。梁相宜2009年毕业于[[中国传媒大学|中国传媒大学]]，现任[[第一财经电视|第一财经电视]]记者，曾服务于[[北京晨报|北京晨报]]、[[第一财经日报|第一财经日报]]。作为一名资深记者，梁相宜曾多次参与[[2016年二十国集团杭州峰会|2016年二十国集团杭州峰会]]、[[两会|全国两会]]等重大活动采访及报道并出镜<ref>{{cite news|title=【反白眼爆紅】梁相宜被翻出「嫩口」舊照 自小表演慾強|url=https://hk.news.appledaily.com/china/realtime/article/20180313/57940828|agency=蘋果日報|date=2018-03-13}}</ref><ref>{{cite news|title=美女财经记者峰会前忙碌的一天，很美很杭州|url=http://lavozchina.com/portal.php?mod=view&aid=142736|agency=西班牙侨声报|date=2016-9-4}}</ref>。

'''张慧君'''（1981年9月24日－），媒体记者，毕业于[[河北大学|河北大学]]，曾在中央电视台实习。在第十一届全国政协第四次会议上（2011年），她的身份是“[[香港有線電視|香港有线]]中国旅游与经济电视台执行台长”；在第十二届全国人大第一次会议（2013年）上，她的身份是“世界知识杂志记者”；在第十三届全国人大第一次会议上（2018年），她的身份是“[[美国全美电视台|美国全美电视台]]执行台长”<ref>{{cite web |title=【麻辣总局】红衣女记者的成长之路|url = https://chinadigitaltimes.net/chinese/2018/03/%E3%80%90%E9%BA%BB%E8%BE%A3%E6%80%BB%E5%B1%80%E3%80%91%E7%BA%A2%E8%A1%A3%E5%A5%B3%E8%AE%B0%E8%80%85%E7%9A%84%E6%88%90%E9%95%BF%E4%B9%8B%E8%B7%AF/|website=中国数字时代}}</ref>。

== 事件 ==
2018年3月13日，梁相宜参与采访全国两会。在13日的部长通道环节中，因在邻座的美国全美电视台记者兼执行台长张慧君采访[[国务院国有资产监督管理委员会|国务院国资委]]主任[[肖亚庆|肖亚庆]]期间翻白眼而走红网络<ref>{{cite news |url = http://ent.163.com/18/0313/13/DCPIRKCN00038FO9.html |title = 笑翻！"戏精"女记者开会现场对同行翻白眼走红 |author=|date=2018-3-13 |accessdate= |website=网易娱乐}}</ref>。

该事件发生于当日11点，梁相宜在全国[[两会|两会]]的记者提问环节与全美电视台台长张慧君同时举手提问，着红衣的张慧君获得了提问的机会。但是由于她问题描述冗长不清，梁相宜忍不住接连翻了多个[[白眼|白眼]]，表示自己对张慧君的不满<ref>{{Cite web |url = https://www.douban.com/group/topic/113761190/ |title=蓝衣记者可能是把很多在场记者的内心白眼翻出来了 |accessdate=2018-03-13 |website = www.douban.com |language=zh-Hans }}</ref>。全过程被[[中国中央电视台|中国中央电视台]]直播，12点该消息传遍中国互联网。社交媒体流传的[[微信|微信]]聊天记录截图显示，梁相宜在得知自己翻白眼被直播后，对朋友说：“你愁（瞅）她那德行，问问题比回答的还多。”<ref>{{cite news |url = http://www.bbc.com/zhongwen/simp/chinese-news-43385477 |title=中国两会：女记者翻白眼被直播引爆社交网络 |author=周卫 |date=2018-03-13 |website = BBC中文网 |accessdate=2018-03-13 |language=en-GB }}</ref>又二人当时衣着恰好一红一蓝，对比因此成为了网络热点，有关于两人的文章在各种网媒，尤其是微信公众号上层出不穷。《[[环球时报|环球时报]]》总编辑[[胡锡进|胡锡进]]发微博说：“第一财经女记者梁相宜火了。她对旁边提冗长问题的记者翻白眼，被电视直播了出来，笑翻网民。希望这样的花絮能在网上留的住，别被删掉。老百姓就是喜欢这种东西，人性的这种DNA改不了。党员干部要随时讲政治，但普通人不必。普通人看政治的视角也喜欢花絮化，但这无伤大雅。”<ref>{{cite web |url = https://weibo.com/huxijin?refer_flag=1001030101_&is_hot=1 |title=胡锡进的微博 |accessdate=2018-03-13 |author= [[胡锡进|胡锡进]] |date= |website=weibo.com }}</ref>然而言論之後急速收緊，16时起，“蓝衣记者”已经成为[[新浪微博|新浪微博]]的[[敏感词|敏感词]]，[[中华人民共和国网络审查|不可进行搜索]]，胡锡进所发微博也遭[[中华人民共和国言论审查|审查人员]]删除<ref>{{Cite news |url = https://chinadigitaltimes.net/chinese/2018/03/%E3%80%90%E6%95%8F%E6%84%9F%E8%AF%8D%E5%BA%93%E3%80%91%E4%B8%A4%E4%BC%9A%E7%BF%BB%E7%99%BD%E7%9C%BC%E9%81%AD%E5%B1%8F%E8%94%BD-%E8%93%9D%E8%A1%A3%E8%AE%B0%E8%80%85%E6%88%90%E6%95%8F%E6%84%9F%E8%AF%8D/ |title=【敏感词库】两会翻白眼遭屏蔽 蓝衣记者成敏感词 - 中国数字时代 |date=2018-03-13 |website = 中国数字时代 |accessdate=2018-03-13 |language=zh-CN }}</ref>。

由於事件意外引起人们高度讨论，梁相宜被曝可能工作不保，被定性为影响会议严肃性。此事也逐漸傳遍全球，据纽约时报和BBC分析指出，起初事件擴大之際，部分的網民將「藍」、「紅」之爭，引申為兩會本身實質為[[橡皮圖章_(政治)|橡皮圖章]]表演行為的諷刺，伴隨著[[人肉搜索|人肉搜索]]引發的部分官場腐敗傳言不斷出現，因此遭到了官方封殺，中国中央电视台随后重播也将此部分剪去<ref name="NYT" /><ref>{{cite web |author=周衛 BBC中文記者 |url = http://www.bbc.com/zhongwen/trad/chinese-news-43385477 |title=中國兩會：女記者翻白眼被直播引爆社交網絡 - BBC 中文网 |publisher=Bbc.com |date= |accessdate=2018-03-20 }}</ref>。
