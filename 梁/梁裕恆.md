'''梁裕恆'''（{{Lang-en|Flow Leung Yu Hang}}，{{bd|1985年|8月24日|catIdx=L梁裕恆}}），香港男演員及健身教練，曾參加《[[2016年度香港先生選舉|2016年度香港先生選舉]]》，現為[[無綫電視藝員列表|無綫電視基本藝人合約]]藝員<ref>[http://artiste.tvb.com/main/artistedetail/leungyuhang 梁裕恆 Leung Yu Hang - TVB藝人資料]</ref>。

== 簡歷 ==
梁裕恆從小在[[石硤尾|石硤尾]]長大<ref>{{Cite web |url=https://archive.fo/J5Zfu/494871160176c8db00ad2f310dc2882575c5d1f4.jpg |title=呢個細路好樣衰但幾得意 |date=2016-10-28 |publisher=Facebook（私人健身教練Flow Leung）}}</ref>，曾就讀[[中聖書院|中聖書院]]<ref>{{Cite web |url=https://archive.fo/pj3qU/9b482000eee739ccdcee3e3b419271fc7d298190.jpg |title=中學朋友 做義工又撞番 好朋友既女朋友 又係 TVB 既同事 真有緣！ |date=2016-12-07 |publisher=Facebook（私人健身教練Flow Leung）}}</ref>，涉足演藝圈前曾任職[[港島區|港島區]]消防員（2010 - 2016年<ref name=FL>{{Cite web |url=https://www.instagram.com/p/Bidfjv6hXAR/ |title=比賽相 |date=2018-05-07 |publisher=Instagram（flow_leung）}}</ref>），2016年報名參加《[[香港先生選舉|香港先生選舉]]》且成功晉級最後十強<ref>{{Cite web |url=http://hk.apple.nextmedia.com/enews/realtime/20160826/55559349 |title=【上次彈跳東平洲頁岩】半裸香港先生今日入南生圍餵蚊 |date=2016-08-26 |publisher=[[香港蘋果日報|香港蘋果日報]]}}</ref><ref>{{Cite web |url=http://www.bastillepost.com/hongkong/4-%E5%A8%9B%E5%9C%88%E4%BA%8B/1370697-%E5%8D%81%E4%BD%8D%E5%80%99%E9%81%B8%E6%B8%AF%E7%94%B7%E5%A4%A7%E6%99%92%E5%86%B7 |title=十位候選港男大晒冷 |date=2016-09-04 |publisher=[[巴士的報|巴士的報]]}}</ref>，落敗後便加入[[無綫電視|無綫電視]]，入讀為期3個月的[[無綫電視藝員訓練班|2016 - 17年度演藝進修班]]，2017年3月3日正式簽約成為旗下合約藝員。2018年，他在該台劇集《[[飛虎之潛行極戰|飛虎之潛行極戰]]》內因展示了健碩身形而受到觀眾注目<ref>{{Cite web |url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/188270/ |title=【飛虎之潛行極戰】首集前港男爆肌入浴　黃宗澤鋼條身型唔輸蝕 |date=2018-05-15 |publisher=[[香港01|香港01]]}}</ref><ref>{{Cite web |url=https://www.cosmopolitan.com.hk/cosmogirl/Flying-Tiger-flow-man |title=搶鏡過黃宗澤？《飛虎之潛行極戰》大隻仔梁裕恆以前係廢青 |date=2018-05-18 |publisher=[[COSMOPOLITAN|COSMOPOLITAN]]（香港）}}</ref>；翌年4月透過其[[面書|面書]]發文支持[[黃心穎|黃心穎]]，並認為她才是「[[安心偷食事件|安心偷食事件]]」裡的受害者，結果引起網民不滿和被指故意突顯自己<ref>{{Cite web |url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/320120/%E5%8A%9B%E6%92%90%E9%BB%83%E5%BF%83%E7%A9%8E-%E8%90%BD%E9%81%B8%E6%B8%AF%E7%94%B7%E6%A2%81%E8%A3%95%E6%81%86%E5%A4%A7%E8%B5%B7%E5%BA%95-%E6%9B%BE%E6%8F%9A%E8%A8%80-%E6%88%91%E6%83%B3%E8%A6%81%E7%9F%A5%E5%90%8D%E5%BA%A6 |title=【力撐黃心穎】落選港男梁裕恆大起底　曾揚言：我想要知名度！|date=2019-04-20 |publisher=香港01}}</ref><ref>{{Cite web |url=http://hd.stheadline.com/life/ent/realtime/1483566/ |title=【安心出軌】力撐黃心穎 港男梁裕恆爆「內情」：男方曾承諾會離婚 |date=2019-04-20 |publisher=[[頭條日報|頭條日報]]}}</ref><ref>{{Cite web |url=https://ol.mingpao.com/ldy/showbiz/latest/20190420/1555732383192/%E3%80%90%E7%88%86%E5%85%A7%E6%83%85%E3%80%91%E8%90%BD%E9%81%B8%E6%B8%AF%E7%94%B7%E6%A2%81%E8%A3%95%E6%81%86%E7%82%BA%E9%BB%83%E5%BF%83%E7%A9%8E%E7%99%BC%E8%81%B2-%E4%BD%A2%E4%BF%82%E5%8F%97%E5%AE%B3%E8%80%85 |title=【爆內情】落選港男梁裕恆為黃心穎發聲：佢係受害者 |date=2019-04-20 |publisher=[[明報|明報 OL 網]]}}</ref>。

梁裕恆除了當演員，亦有兼職泰拳及健身教練<ref name=FL0/>，曾以「Wolf」名義參與2016年《雷霆猛男亞洲區選拔賽》，贏取「網上我最喜愛至 Fit 猛男」獎項<ref>{{Cite web |url=https://www.bastillepost.com/hongkong/article/1229973-%E7%88%86%E8%82%8C%E6%A2%81%E8%A3%95%E6%81%86%E6%84%9B%E5%BF%83%E6%BB%BF%E7%80%89%E4%BA%BA%E6%B0%A3%E6%97%BA |title=爆肌梁裕恆愛心滿瀉人氣旺 |date=2016-07-04 |publisher=巴士的報}}</ref>；又於《2018環球國際邀請賽（香港站）》中奪得男子健體組別冠軍<ref name=FL/>。
