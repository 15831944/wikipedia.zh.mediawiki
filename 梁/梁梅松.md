'''梁梅松'''，[[臺灣|臺灣]][[體操|體操]][[教練|教練]]。

== 個人生活 ==
梁梅松與雙胞胎弟弟[[梁梅宗|梁梅宗]]是[[高雄市立五福國民中學|高雄市立五福國民中學]]的體操選手，並一同就讀[[國立臺灣體育學院|國立臺灣體育學院]]<ref name = M00062651>{{cite news|title=孿生體操好手|publisher=民生報|url=http://catalog.digitalarchives.tw/pdf/00/00/a3/d6.pdf|date=1985-05-06|author=楊權寶|accessdate=2018-03-14|language=zh-hant}}</ref>。

== 生平 ==
梁梅松曾參加過全國中正盃體操錦標賽，在[[自由體操|地板]]和[[單槓_(體操)|單槓]]的競賽項目中，獲得兩面金牌<ref name = M00062651/>。並於1985年[[全國大專院校運動會|全國大專院校運動會]]上，梁梅松與[[黃瑞榮|黃瑞榮]]並列男子個人全能比賽第三名<ref>{{cite news|title=大運會體操甲組競技 北體雙料冠軍|publisher=民生報|url=http://catalog.digitalarchives.tw/pdf/00/00/a0/67.pdf|date=1985-05-06|author=|accessdate=2018-03-14|language=zh-hant}}</ref>。

2005年梁梅松擔任[[高雄市私立明誠高級中學|高雄市私立明誠高級中學]]體操教練，帶領學生[[盧幸宜|盧幸宜]]和[[蔡慶瑜|蔡慶瑜]]奪下[[全國中等學校運動會|全國中等學校運動會]]個人全能金牌和銅牌<ref>{{cite news|title=盧幸宜 全能首度摘金|publisher=蘋果日報|url=https://tw.appledaily.com/entertainment/daily/20050420/21723477|date=2005-04-20|author=詹健全|accessdate=2018-06-12|language=zh-hant}}</ref>。同年舉辦的[[中華民國全國運動會|中華民國全國運動會]]中，梁梅松所組隊的學生盧幸宜，獲得地板和[[跳馬|跳馬]]兩面銀牌<ref>{{cite news|title=跳馬王子黃怡學獲1金2銅|publisher=蘋果日報|url=https://tw.sports.appledaily.com/daily/20051021/2144427/|date=2005-10-21|author=黃建仁|accessdate=2018-06-12|language=zh-hant}}</ref>。
