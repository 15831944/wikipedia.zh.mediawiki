'''梁家輝'''（'''Tony Leung Ka-fai'''，{{bd|1958年|2月1日|catIdx=L梁}}），亞洲影壇多項演技大獎得主。以演技精湛見稱， 深沉又具爆發力，曾多次奪金像獎影帝殊榮，亦是香港電影金像獎歷史上最年輕的影帝。 包括1983年初露頭角的垂簾聽政， 其他經典演出包括92黑玫瑰對黑玫瑰， 愛在他鄉的季節及黑社會等。

== 早年生活 ==
梁家輝於[[香港島|香港島]][[北角邨|北角邨]]中座長大。父親是文員，母親則在[[銅鑼灣|銅鑼灣]]豪華戲院做售票員，阿姨則在戲院大堂賣[[爆谷|爆谷]]。<ref name="早年家庭背景1">{{cite web|url=http://www.lifeweek.com.cn/2012/1011/38796.shtml | title="梁家辉：“人生最不该有风景占尽的念头” |publisher=三联生活周刊|author=李东然 |date=2012-10-11 }}</ref><ref name="早年家庭背景2">{{cite web|url=http://www.bjnews.com.cn/ent/2010/10/26/78300.html | title="梁家辉：被封杀不是最艰难的 |publisher=新京报 |date=2010-10-26 }}</ref>梁家輝從小因此常有機會免費看電影以消磨時光，梁接受訪問時曾表示包括瑪麗蓮·夢露等作品都令我印象深刻，尤其是《[[星光伴我心|星光伴我心]]》令梁家輝覺得片中的小童星很像他自己因而他從頭哭到尾。<ref name="早年家庭背景2"/>

梁家輝曾經是香港短跑紀錄冠軍，<ref name="鲁豫有约"/>求學時期因[[短跑|短跑]]專長曾想成為職業運動員，後來從教練那得知自己的身材並不適合短跑才放棄。<ref>{{cite web|url=http://ent.qq.com/a/20080821/000016.htm | title="组图：梁家辉与妻子相拥看奥运 双胞胎爱女曝光 |publisher=腾讯娱乐 |date=2008-08-21 }}</ref>梁家輝畢業於[[玫瑰崗學校|玫瑰崗學校]]（中學部）之後就讀於[[香港理工大學|香港理工学院]]平面設計，<ref name="南方人物周刊">{{cite journal|title=梁家輝 一半退休，永不退役 |journal=南方人物周刊 |volume = 2010002 |date= 2010-01-20  |url=http://magazine.sina.com/bg/southernpeopleweekly/2010002/2010-01-19/ba82648.html |accessdate=2014-08-03 }}</ref>其後梁陪同朋友報名[[無綫電視藝員訓練班|無綫電視藝員訓練班]]，<ref name="南方人物周刊"/>因此在1980年考入[[電視廣播有限公司|香港無線電視台]]的[[無綫電視藝員訓練班|無綫電視藝員訓練班]]第10期藝員訓練班<ref name="訓練班">{{cite web|url=http://yule.sohu.com/20121014/n354810238.shtml |title="TVB艺员训练班教学大公开 揭秘不为人知的校规 |publisher=搜狐网 |date=2012-10-14 }}</ref>著名的同期同學有[[劉德華|劉德華]]、[[吳家麗|吳家麗]]、[[戚美珍|戚美珍]]、[[張之亮|張之亮]]、[[徐錦江|徐錦江]]等。在訓練班期間，梁家輝因為勤奮跟劉德華是當時露臉最多的兩個龍套，<ref name="南方人物周刊"/>其中為人所知的一次是與[[劉德華|劉德華]]一起被選中，為電視劇《[[千王群英會|千王群英會]]》配戲，扮演當時[[周潤發|周潤發]]的打手。<ref name="訓練班"/><ref>[http://star.ettoday.net/news/677935 劉德華、梁家輝出道始末曝光！周潤發：他們是我的臨演]</ref>

後來梁家輝念了不到一年訓練班就因為做事不守規矩被趕出。<ref name="早年家庭背景2"/>離開無線訓練班的梁家輝因此創辦了一本雜誌，其中採訪、攝影、編輯、賣廣告、督印都是他自己獨自完成。<ref name="早年家庭背景1"/>

== 演藝經歷 ==
梁家輝在創辦雜誌期間，曾找導演[[李翰祥|李翰祥]]的女兒李殿朗做雜誌的封面女郎並和她拍拖，<ref>{{cite web |url=http://mpaper.org/Story.aspx?ID=416211 |title="李翰祥改變梁家輝一生 |publisher=澳門日報　 |date=2013-04-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20160202045916/http://mpaper.org/Story.aspx?ID=416211 |archivedate=2016-02-02 }}</ref>因而認識當時正在籌拍《[[火燒圓明園|火燒圓明園]]》的[[李翰祥|李翰祥]]，並被李翰祥邀請到[[北京|北京]]，剃頭扮演[[咸豐帝|咸豐帝]]，並以此角色接連拍攝《[[火燒圓明園|火燒圓明園]]》和《[[垂簾聽政_(電影)|垂簾聽政]]》兩片。當時的梁家輝完全不會說[[普通話|普通話]]，令他與當時扮演[[慈禧|慈禧]]的[[劉曉慶|劉曉慶]]無法交流對戲，<ref name="鲁豫有约"/>引致[[劉曉慶|劉曉慶]]要教梁做一些小動作以提示劉講對白。<ref>{{cite web |url=http://et.21cn.com/gundong/etscroll/a/2013/0929/08/24278015.shtml |title="刘晓庆自曝拍《火烧圆明园》时偷梁家辉饭票(图) |publisher=21cn　 |date=2013-09-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20140811163727/http://et.21cn.com/gundong/etscroll/a/2013/0929/08/24278015.shtml |archivedate=2014年8月11日 |df= }}</ref>1984年，26歲的梁家輝其後更憑藉《[[垂簾聽政_(電影)|垂簾聽政]]》在[[1984年香港電影金像獎|第三屆香港電影金像獎]]获得最佳新人奖提名，並成為歷來[[香港電影金像獎最佳男主角|金像獎最佳男主角]]中最年輕得主。雖然梁家輝第一部電影《[[垂簾聽政_(電影)|垂簾聽政]]》便替他拿到最佳男主角，但是梁家輝後來卻說那時候的自己並不懂得表演，只是按照導演的示範來完成角色。<ref name="鲁豫有约">{{cite web|url=https://www.youtube.com/watch?v=AaQooT0vnRY | title="鲁豫有约 梁家辉郭富城 影帝对决2012-11-08 |publisher=凤凰卫视官方频道 |date=2013-06-09 }}</ref>

之後梁家輝因《[[垂簾聽政_(電影)|垂簾聽政]]》获得[[香港電影金像獎最佳男主角|香港電影金像獎最佳男主角]]被[[中华民国行政院新闻局|中华民国行政院新闻局]]全面封殺，所有作品无法登台公映，致使香港电影公司不敢用其出任角色。為了營生，而在[[石澳|石澳]]一帶作無牌小販在街頭販賣皮飾等飾品，後因生意慘淡而轉到[[銅鑼灣|銅鑼灣]]一帶繼續販賣。<ref name="鲁豫有约"/>期间在[[罗维影业|罗维影业]]、[[邵氏影业|邵氏影业]]以及[[李翰祥|李翰祥]]的[[新昆仑|新昆仑影业]]出演过一些角色。

1987年[[解嚴|台湾解严]]之後，对梁家輝的禁令自然取消，得以繼續專攻電影，從影31年間拍過逾百部作品。其電影一般以文藝片為主，同時也拍過不少商業性質的戲，包括[[周潤發|周潤發]]的3部電影：《[[監獄風雲|監獄風雲]]》、《[[英雄本色3之夕陽之歌|英雄本色3之夕陽之歌]]》、《[[賭神2|賭神2]]》；在《[[火燒島|火燒島]]》一片則與動作片巨星[[成龍|成龍]]合作；另外[[麥當雄|麥當雄]]之《[[三狼奇案|三狼奇案]]》和《[[黑金|黑金]]》梁家輝也有參演。梁家輝以熟練的演技見稱，如在《[[黑金|黑金]]》飾演一個[[台灣|台灣]][[黑社會|黑社會]][[領導人|領導人]]，只用一個眼神就成功演繹出黑道頭目之威勢。1990年，與[[張曼玉|張曼玉]]合作的《[[爱在別乡的季节|爱在別乡的季节]]》使其获第27届台湾电影[[金马奖|金马奖]][[第27屆金馬獎#最佳男主角|最佳男主角奖]]。他於1992年为其演艺事业创出另一高峰，不但获颁[[香港艺术家年奖|香港艺术家年奖]]演员奖，更成功打入国际影坛─以法國影片《[[情人_(1992年電影)|情人]]》打破[[法国|法国]]电影两年来的票房纪录，同年以《[[92黑玫瑰对黑玫瑰|92黑玫瑰对黑玫瑰]]》再次成为[[香港電影金像獎|香港電影金像獎]]影帝。

雖然自1990年代末[[港產片|港產片]]市道持續低迷，梁家輝仍然有一定數量之電影演出。2005年，参加[[八一电影制片厂|八一电影制片厂]]拍摄的抗日题材影片《[[太行山上|太行山上]]》演出，在片中扮演一位独臂的[[八路军|八路军]]团长[[贺炳炎|贺炳炎]]，单刀力劈[[日军|日军]]，虽然只有几分钟镜头，但被公认为本片最震撼的部分。2006年，以《[[黑社會_(電影)|黑社會]]》一片，第3度贏得[[香港电影金像獎|香港电影金像獎]]影帝，是繼[[黃秋生|黃秋生]]、[[梁朝偉|梁朝偉]]後，第3位憑[[三級片|三級片]]成為影帝者，距离上次得奖相隔14年。2012年，以電影《[[寒戰|寒戰]]》入圍[[亞太影展|亞太影展]]最佳男主角，更成功再第4度奪得[[香港電影金像獎|香港電影金像獎]][[影帝|影帝]]。目前梁家輝每年電影產量穩定，為華人電影界不可或缺的實力派演員之一。

梁家輝曾兩次因醉酒駕駛而留下刑事紀錄。2002年11月9日，他駕車與一輛巴士及一輛私家車相撞，其後他打傷巴士司機，並被警方揭發他體內酒精含量超標。2003年1月15日，他於區域法院承認普通襲擊及醉酒駕駛罪，被判罰款港幣12,000元及停牌18個月。<ref>[http://hk.apple.nextmedia.com/news/art/20030116/3065791 梁家輝醉駕傷人　停牌留案底]</ref> 2005年10月29日，他再涉酒後駕駛。2006年1月6日，他雖於東區裁判法院承認醉酒駕駛罪，但裁判官認為因其再犯及作為公眾人物應樹立榜樣，判處他監禁2個月，緩刑3年，罰款港幣10,000元及停牌3年。<ref>[http://hk.apple.nextmedia.com/news/art/20060107/5552288 官：國際明星應立榜樣 三年兩醉駕　梁家輝判囚緩刑]</ref>

==個人生活==
梁家輝80年代曾與[[黎芷珊|黎芷珊]]拍拖，不過年紀的差距導致兩人分手。後來，在梁家輝最不順遂時，遇上[[電台|電台]]製作人[[江嘉年|江嘉年]]，兩人其後結緍並生下一對[[雙胞胎|雙胞胎]]。梁家輝婚後甚少傳出緋聞，是娛樂圈中少見的模範夫妻。
