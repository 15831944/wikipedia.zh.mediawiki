'''梁益誌'''（{{bd|1982年|2月28日|catIdx=Liang梁}}）是一名生於[[臺灣|臺灣]][[高雄市_(1979年-2010年)|高雄市]]的[[臺灣政治|政治]]、[[社運|社運]]人士，現任[[臺灣綠黨|臺灣綠黨]]中央執行委員，並曾擔任高雄同志遊行聯盟發起人、第二屆高雄同志遊行聯盟總召集人<ref>{{cite news|publisher = 公民新聞|title=高雄同志遊行三千人上街　訴求走出冷漠挺同志 |url=http://www.peopo.org/news/84666}}</ref><ref>{{cite news|publisher = 立報|title=5大事件 透露南台灣同志人權問題 |url=http://www.lihpao.com/?action-viewnews-itemid-111009}}</ref>、2013年[[309廢核大遊行|309廢核大遊行]]南臺灣主持人，同時也在[[高雄市|高雄市]]經營[[性別友善|性別友善]]的南人窩咖啡廳和南人獨立工作室<ref>[http://woooo59.pixnet.net/blog/post/11364210 理念與夢想]，南人窩Blog，2010年</ref>。

梁益誌長期為[[同志|同志]]和[[臺灣反核運動|反核議題]]發聲，特別在高雄地區經營，舉辦各項遊行與活動；並在[[2014年中華民國地方公職人員選舉|2014年九合一選舉]]代表綠黨參選高雄市議員，主打環保、食安、勞工、青年、平權等議題，是台灣選舉史上少有的已[[出櫃|出櫃]]同性戀候選人。

== 生平經歷 ==
台大畢業後於台灣[[同志諮詢熱線協會|同志諮詢熱線協會]]擔任義工，開始學習以性別角度關注各種社會議題。在經歷多場遊行抗爭後，深深感悟理念傳達無法單靠知識的傳播，年輕人必須要身體力行，參與更多社會運動，於是創立高雄同志遊行聯盟，舉辦各種高雄在地的性別運動。也透過本身的參與，接觸到更多社會議題，像是居住、土地、環境、勞工等。<ref>[http://green.democracy.tw/star/bio 台灣綠黨候選人介紹網站 - 公民進擊！幸福升級！梁益誌] {{webarchive|url=https://web.archive.org/web/20141106205405/http://green.democracy.tw/star/bio |date=2014-11-06 }}，台灣綠黨，2014年</ref>

== 參選經歷 ==
在歷經10年運動後，因為所關注的社會議題遇到瓶頸，特別是性別教育在學校的落實面上沒有太大的成長，決定參選市議員，進入體制內，解決政策執行上的問題。<ref>{{cite news|publisher = 環境資訊中心|title=【2014選舉 × 環境】從性別運動出發 梁益誌關注食安醫療、為弱勢發聲 |url=http://e-info.org.tw/node/102045}}</ref>
*2014年2月，正式宣佈參選高雄市三民區市議員。
*2014年6月，舉辦「公民參與」政見審議座談會。
*2014年7月，舉辦「多元平等」政見審議座談會。
*2014年8月，親手打造太陽能宣傳腳踏車隊，環保不是口號，力求自行實踐，使用低碳環保的方式選舉。<ref>{{cite news|publisher = 民報|title=梁益誌親手打造太陽能宣傳腳踏車隊「應徵市議員」 |url=http://www.peoplenews.tw/news/30512d0e-176a-4d99-b864-0d8659176039}}</ref>
*2014年9月，帶領6對同志伴侶登記結婚，依法無法登記，籲馬總統正視「同志婚姻需求」。<ref>{{cite news|publisher = 民報|title=梁益誌：馬總統 同志是有婚姻需求的！ |url=http://www.peoplenews.tw/news/3b4f4077-e008-4693-ac9b-0236fbbe6cab}}</ref>
