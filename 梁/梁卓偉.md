'''梁卓偉'''教授{{GBSHK}}（Professor '''Gabriel Matthew Leung'''，{{bd|1972年|5月28日|catIdx=梁}}），前[[香港行政長官辦公室主任|香港行政長官辦公室主任]]，曾經擔任首任[[食物及衞生局|食物及衞生局]]副局長<ref name="fhb1">{{cite web | language = zh-hk |publisher = |title = 食物及衛生局副局長簡歷 |url = http://www.fhb.gov.hk/cn/aboutus/biography/usfh.htm |author = |date = 2008-09-18 |accessdate = 2009-05-17 }}</ref>，當時直屬上司為局長[[周一嶽|周一嶽]]<ref name="fhb2">{{cite web |language = zh-hk |publisher = |title = 食物及衞生局:組織及職能 |url = http://www.fhb.gov.hk/cn/aboutus/org/ |author = |date = 2008-09-18 |accessdate = 2009-05-17 }}</ref>。現任[[香港大學李嘉誠醫學院|香港大學李嘉誠醫學院]]院長、公共衞生學院講座教授。<ref name=":0">{{Cite news|url=http://sph.hku.hk/tc/about-us/faculty-and-staff/academic-staff/leung,-gabriel-m|title=梁卓偉|accessdate=2018-02-07|language=zh-tw}}</ref>

==資歷==
梁卓偉曾經於[[香港華仁書院|香港華仁書院]]讀書及於[[Stonyhurst_College|Stonyhurst College]]留學<ref name="政情：梁卓偉做指揮有睇頭">[http://the-sun.on.cc/cnt/news/20100711/00408_002.html] 政情：梁卓偉做指揮有睇頭</ref>，其後畢業於加拿大Crescent School中學<ref name="Crescent School">[http://www.crescentschool.org/] Crescent School</ref> 及於[[加拿大|加拿大]][[西安大略大學|西安大略大學]]醫學院畢業，在[[多倫多大學|多倫多大學]]完成家庭醫學專科訓練<ref name="fhb1" />；並取得[[美國|美國]][[哈佛大學|哈佛大學]]公共衛生碩士學位及[[香港大學|香港大學]]醫學博士學位，同時擁有多項專業資歷<ref name="sphhku">{{cite web |language = en |publisher = The University of Hong Kong |title = School of Public Health Li Ka Shing Faculty of Medicine The University of Hong Kong |url = http://sph.hku.hk/faculty_and_staff_detail.php?id=13 |author =  |date =  |accessdate = 2009-05-12 |deadurl = yes |archiveurl = https://web.archive.org/web/20090829092901/http://sph.hku.hk/faculty_and_staff_detail.php?id=13 |archivedate = 2009-08-29 }}</ref><ref name="mhrnhku">{{cite web |language = en |publisher = Faculty of Medicine The University of Hong Kong |title = Medical and Health Research Network - Membership - Leung Gabriel Matthew |url = http://www.hku.hk/facmed/mhrn/membership/gmleung.htm |author =  |date =  |accessdate = 2009-05-12 |deadurl = yes |archiveurl = https://web.archive.org/web/20090927105522/http://www.hku.hk/facmed/mhrn/membership/gmleung.htm |archivedate = 2009-09-27 }}</ref>。

梁卓偉1999年加入[[香港大學李嘉誠醫學院|香港大學醫學院]]，2006年晉升為公共衛生及社會醫學教授，於2008年5月成為食物及衞生局副局長，乃首批獲行政長官[[曾蔭權|曾蔭權]]委任的[[香港政治委任制度|副局長]]之一<ref name="isdpr1">{{cite web |language = zh-hk |publisher = 政府新聞處 |title = 行政長官委任副局長(附圖) |url = http://www.info.gov.hk/gia/general/200805/20/P200805200124.htm |author = |date = 2008-05-20 |accessdate = 2009-05-21 }}</ref>，並在2011年9月30日接替[[譚志源|譚志源]]出任第五任特首辦主任。

他曾擔任[[世界衛生組織|世界衛生組織]]、[[世界銀行|世界銀行]]及多個國際組織顧問，亦為[[香港小交響樂團|香港小交響樂團]]監察委員會成員<ref name="isdpr1" />。

在2010到2014年間，出任亞太衞生體系與政策觀察組織首任主席。此組織由多國政府、發展機構和科研界的多方合作組成。

2012年5月，梁卓偉被問及個人去留問題，以英國大文豪[[莎士比亞|莎士比亞]]名劇《[[亨利五世_(戏剧)|亨利五世]]》的著名台詞「Turn him to any cause of policy，the gordian knot of it he will unloose」回應。有議員指他所引的台詞，原意是指要用不常規的方法解決問題，但亦有「快刀斬亂麻，早走早好」之意<ref>{{cite news|title=梁卓偉引莎翁名句 暗示走人|url=http://www.worldjournal.com/view/full_news/18535656/article-%E6%A2%81%E5%8D%93%E5%81%89%E5%BC%95%E8%8E%8E%E7%BF%81%E5%90%8D%E5%8F%A5-%E6%9A%97%E7%A4%BA%E8%B5%B0%E4%BA%BA?instance=hk_bull|newspaper=世界新聞網|date=2012-05-10}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 。同月月底，審計報告質疑特首外訪的安排未有適當使用公帑。梁卓偉代表特首辦接納意見，並指作為特首辦主任，雖未必百分百知道所有細節，但自己亦要負上全責<ref>{{cite news|title=梁卓偉承認政府有不足　自己願負全責|url=http://hk.news.yahoo.com/%E6%A2%81%E5%8D%93%E5%81%89%E6%89%BF%E8%AA%8D%E6%94%BF%E5%BA%9C%E6%9C%89%E4%B8%8D%E8%B6%B3-%E8%87%AA%E5%B7%B1%E9%A1%98%E8%B2%A0%E5%85%A8%E8%B2%AC-095700701.html|newspaper=香港電台|date=2012-05-31}}</ref> 。

2013年5月，接任香港大學社會醫學系系主任一職一年後，出任醫學院院長，並在八月一日履新。

2018年2月，港大校董會批准梁卓偉續任港大醫學院長一職，任期至2023年。

== 研究領域 ==
梁卓偉的研究領域為公共衞生醫學，主要圍繞：<ref name=":0" />
# 對本地人口健康有重大影響；
# 香港作為中國內地可靠且獨特的流行病哨點；及
# 香港具備得天獨厚的條件解決當前的基礎科學問題
