'''梁恩碩'''（{{bd|2000年|10月2日}}），出生於[[高雄市|高雄市]]<ref name="australian_open">{{cite news |author = 詹健全 |url = https://tw.appledaily.com/headline/daily/20180128/37916903 |title = 史上第一人 台灣小鋼砲 澳網捧雙冠 |publisher = [[蘋果日報|蘋果日報]] |date = 2018-01-28 |accessdate = 2018-07-25 |archive-url = https://web.archive.org/web/20180727064010/https://tw.appledaily.com/headline/daily/20180128/37916903 |archive-date = 2018-07-27 |dead-url = no }}</ref>，[[臺灣|臺灣]]女子青少年[[網球|網球]]選手。她因為球質渾重，被譽為「'''臺灣小鋼砲'''」<ref name="australian_open" />。她的[[女子網球聯合會|WTA]]單打排名在2019年2月4日達到生涯新高的第233名，雙打排名紀錄則是2019年2月25日的第520名。她擁有各一座單、雙打[[國際網球總會|ITF]]冠軍。

2018年1月，梁恩碩一人包攬[[2018年澳洲網球公開賽|2018年澳洲網球公開賽]]青少年組女子單、雙打冠軍，成為臺灣首位在大滿貫青少年組女子單打奪冠的球員，同時也是繼2004年[[詹詠然|詹詠然]]之後，臺灣網壇第二位在澳網青少年女雙封后的球員<ref name="cna">{{cite news |author = 李晉緯 |url = http://www.cna.com.tw/news/firstnews/201801270045-1.aspx |title = 澳網青少年女單 梁恩碩奪冠台灣第一人 |publisher = [[中央社|中央社]] |date = 2018-01-27 |accessdate = 2018-07-25 |archive-url = https://web.archive.org/web/20180725214336/http://www.cna.com.tw/news/firstnews/201801270045-1.aspx |archive-date = 2018-07-25 |dead-url = no }}</ref>；同年8月參與[[2018年亞洲運動會|雅加達巨港亞運]]拿下[[2018年亞洲運動會網球比賽|女子單打]]銅牌，為臺灣繼1958年[[劉香谷|劉香谷]]睽違60年再度贏得[[亞洲運動會|亞運]]網球女單獎牌<ref name="asiangames18">{{cite news |author = 詹健全 |url = https://tw.sports.appledaily.com/daily/20180824/38106405 |title = 60年第1姝 梁恩碩女網單打奪銅 |publisher = [[蘋果日報|蘋果日報]] |date = 2018-08-24 |accessdate = 2018-08-24 |archive-url = https://web.archive.org/web/20180824081911/https://tw.sports.appledaily.com/daily/20180824/38106405/ |archive-date = 2018-08-24 |dead-url = no }}</ref>。

梁恩碩現就讀於[[台南市|台南市]][[善化高中|善化高中]]<ref name="cna" />。她曾接受同為網球選手的[[詹謹瑋|詹謹瑋]]和其父詹佛成執導<ref name="australian_open" />，目前則與旅居廣州的臺灣籍教練馬偉開及其團隊合作<ref name="asiangames18" />。

== 早年 ==
梁恩碩於2000年出生於[[高雄市|高雄市]]。她的父親梁世銘是工程師，與她的母親夏碧霙原本皆在中國工作。為了照顧女兒練球，梁恩碩的媽媽辭掉工作，專職帶著女兒出國參賽<ref>{{cite news |author = 詹健全 |url = https://tw.appledaily.com/headline/daily/20180128/37916913 |title = 青少網賽沒獎金 梁爸貸300萬燒光 |publisher = 蘋果日報 |date = 2018-01-28 |accessdate = 2018-07-25 |archive-url = https://web.archive.org/web/20180727064010/https://tw.appledaily.com/headline/daily/20180128/37916913 |archive-date = 2018-07-27 |dead-url = no }}</ref><ref>{{cite news |author = 楊勝凱 |url = https://www.ltsports.com.tw/tennis/local/122813-2017-04-26-17-45-39 |title = 全中運》網球單打許育俢、梁恩碩封王　網球家庭心酸多 |publisher = [[麗台運動報|麗台運動報]] |date = 2017-04-26 |accessdate = 2018-07-25 |archive-url = https://web.archive.org/web/20180725214437/https://www.ltsports.com.tw/tennis/local/122813-2017-04-26-17-45-39 |archive-date = 2018-07-25 |dead-url = no }}</ref>。梁恩碩四歲時就接觸音樂、習琴，曾獲得台北市文化盃幼童組鋼琴比賽冠軍<ref name="tennis">{{cite news |author = 詹健全 |url = https://tw.appledaily.com/sports/daily/20180128/37916787 |title = 梁恩碩 減肥遇真愛 音樂神童迷上網球 |publisher = 蘋果日報 |date = 2018-01-28 |accessdate = 2018-07-25 |archive-url = https://web.archive.org/web/20180727064010/https://tw.appledaily.com/sports/daily/20180128/37916787 |archive-date = 2018-07-27 |dead-url = no }}</ref>。由於練鋼琴需要久坐，梁媽媽為了讓身形微胖的梁恩碩瘦身，於是送她去學網球，意外開啟了梁恩碩的網球生涯。2012年，她球齡不到一年時，即在四維盃學童網球錦標賽拿下二年級女生組冠軍，嶄露頭角<ref name="tennis" /><ref name="udn" />。2016年，她在臺灣[[全中運|全中運]]摘下國中組女子單打金牌<ref name="udn" />。

2018年7月，梁恩碩於中國南昌公開賽32籤單打會內賽對戰地主選手[[鄭嫵雙|鄭嫵雙]]，以兩個6：3勝出，首度挑戰WTA巡迴賽便拿下勝利<ref name="WTA首勝">{{cite news |author = 梁偉銘 |url = http://sports.ltn.com.tw/news/paper/1218839 |title = 南昌女網賽》梁恩碩拍落地主 WTA會內首勝開張 |publisher = 自由時報 |date = 2018-07-24 |accessdate = 2018-07-27 |archive-url = https://web.archive.org/web/20180724110704/http://sports.ltn.com.tw/news/paper/1218839 |archive-date = 2018-07-24 |dead-url = no }}</ref>。她以17歲10個月之齡，成為繼[[謝淑薇|謝淑薇]]（15歲8個月）、[[詹詠然|詹詠然]]（17歲2個月）後，第三年輕便拿到WTA會內賽首勝的臺灣女子網球選手<ref name="WTA首勝" />。她翌日再於十六強賽與泰國選手普莉普耶珂（P. Plipuech）交手時以 6：1、6：2 成功出線，挺進八強<ref name="udn">{{cite news |author = 謝進盛 |url = https://udn.com/news/story/7005/3274923 |title = 網球／新星梁恩碩 謝淑薇後台灣女網新希望 |publisher = 聯合新聞網 |date = 2018-07-26 |accessdate = 2018-07-27 |archive-url = https://web.archive.org/web/20180726211825/https://udn.com/news/story/7005/3274923 |archive-date = 2018-07-26 |dead-url = no }}</ref>。梁恩碩隨後敗給大會第三種子、波蘭選手[[瑪格達·蕾妮特|瑪格達·蕾妮特]]，止步八強<ref>{{cite news |url = http://www.thenews.pl/1/5/Artykul/375075,Tennis-Poland%E2%80%99s-Linette-into-semis-at-Nanchang |language = en |title = Tennis: Poland's Linette into semis at Nanchang |publisher = Radio Poland |date = 2018-07-72 |accessdate = 2018-08-01 }}</ref>。

梁恩碩先後自高雄市[[高雄市苓雅區中正國民小學|中正國小]]、臺北市[[臺北市立新興國民中學|新興國中]]畢業後，現就讀於台南市善化高中<ref name="cna" />。
