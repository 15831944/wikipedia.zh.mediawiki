'''硼'''（{{lang|en|Boron}}）是一种化学[[元素|元素]]，[[化学符号|化学符号]]为'''B'''，[[原子序数|原子序数]]为5，是一种[[類金属|類金属]]。由於硼的產生完全來自于[[宇宙射線散裂|宇宙射線散裂]]而非[[恆星核合成|恆星核合成]]反應，硼在[[太陽系|太陽系]]與[[地殼|地殼]]的含量相當稀少。天然的硼主要存在于[[硼砂|硼砂]]（{{chem|Na|2|B|4|O|7|·10H|2|O}}）矿中。

== 发现及命名==
硼的元素名稱，來自於[[硼砂|硼砂]]（borax）。天然含硼的化合物硼砂早为古代医药学家所知悉。中国[[西藏|西藏]]是世界上盛产硼砂的地方之一。

1702年法国医生{{le|霍姆贝格|Wilhelm Homberg}}首先用硼砂与硫酸反应制得硼酸，称之为镇静盐（Sal Sedativum ）。1741年法国化学家帕特指出，硼砂与硫酸作用除生成硼酸外，还得到硫酸钠。1789年拉瓦锡把硼酸基列入元素表。1808年英国化学家[[戴维|戴维]]和法国化学家[[盖·吕萨克|盖·吕萨克]]、{{le|泰纳|Louis_Jacques_Thénard}}各自获得单质硼。

硼的[[拉丁文|拉丁文]]名称为 boracium，來自[[阿拉伯文|阿拉伯文]]buraq（بورق）或[[波斯文|波斯文]]burah（بوره）；兩者皆為[[硼砂|硼砂]]之意。[[日文|日文]]與[[韓文|韓文]]便因此將之意譯為「硼素」。

== 单质 ==
{{see also|硼的同素異形體}}

硼有两种[[同素异形体|同素异形体]]：结晶形硼是有光泽的灰色晶体，硬度与[[金刚石|金刚石]]相近，但极脆；另一種硼則是暗棕色的粉末。

== 分布 ==
硼绝少单独存在，通常以[[硼砂|硼砂]]的化合形式出现。硼砂可用作水软化剂和清洁剂。

== 同位素 ==
{{Main|硼的同位素}}

已发现的硼的[[同位素|同位素]]共有9种，包括硼8，硼9，硼10，硼11，硼12，硼13，硼14，硼15，硼17，其中只有硼10，硼11是稳定的，其他同位素都带有[[放射性|放射性]]

== 用途 ==
* 硼在[[半导体|半导体]]工业大量用作半导体掺杂物，同时硼的化合物在[[杀虫剂|杀虫剂]]、[[防腐剂|防腐剂]]的制造中也有重要的地位。硼还是植物的必需营养元素之一。
* [[硼烷|硼烷]]可以作为火箭的[[高能燃料|高能燃料]]。
* [[硼酸|硼酸]]（{{chem|H|3|BO|3}}）是有防腐性能的[[弱酸|弱酸]]，同時也是[[中子|中子]]吸收劑，在[[核反應爐|核反應爐]]內控制[[核分裂|核分裂]]速率。
* [[硼化鎂|硼化鎂]]在一定的低溫下會變成[[超導體|超導體]]。
* [[氮化硼|氮化硼]]可以作为工业上的固体润滑剂，被称为“白色石墨”。氮化硼和[[石墨|石墨]]类似，具有一种坚硬的同素异形体，其硬度仅次于[[钻石|钻石]]。
* [[氧化硼|氧化硼]]或五水硼砂，可以用來製作耐熱[[玻璃|玻璃]]（硼矽玻璃）。
* [[碳化硼|碳化硼]]作為組成核反應[[控制棒|控制棒]]的材料
