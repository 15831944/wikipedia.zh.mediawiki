'''扁核木'''（[[学名|学名]]：{{lang|la|''Prinsepia utilis''}}）是[[蔷薇科|蔷薇科]][[扁核木属|扁核木属]]的植物。分布于[[巴基斯坦|巴基斯坦]]、[[印度|印度]]、[[尼泊尔|尼泊尔]]、[[不丹|不丹]]以及[[中国大陆|中国大陆]]的[[西藏|西藏]]、[[四川|四川]]、[[云南|云南]]、[[贵州|贵州]]等地，生长于海拔1,000米至2,560米的地区，见于山坡、山谷、荒地及路旁。

== 别名 ==
*青刺尖（滇南本草）
*枪刺果（曲靖）
*打油果、鸡蛋果（云南）
*阿那斯（纳西语）

== 特徵 ==
扁核木是一種[[灌木|灌木]]，最高可達3.5米高，3米寬，每年春季約2月開花，果實於6月成熟<ref>{{cite web |url=http://www.pfaf.org/user/Plant.aspx?LatinName=Prinsepia+utilis |title=Prinsepia utilis |publisher=Plant for a Future植物數據庫 |accessdate=2011-10-12 | language=en}}</ref>。
果實可生吃或煮熟來吃，其果仁可搾取油，即「青刺果油」，是一種天然营养植物油，含豐富的[[不飽和脂肪酸|不飽和脂肪酸]]。
