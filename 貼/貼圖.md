'''貼圖'''（{{lang-en|Sticker}}，{{lang-ja|スタンプ}}），是指使用[[即時通訊軟體|即時通訊軟體]]聊天時所搭配的內嵌式圖片，一般比較常見為可愛[[卡通圖片|卡通圖片]]或[[明星|明星]]表情圖片。

== 歷史 ==
2011年3月，日本發生了[[2011年日本東北地方太平洋近海地震|東北地區太平洋近海地震]]，韓國[[Naver_(公司)|Naver]]公司的創辦人李海珍因無法即時聯絡家人，而有了在日本推出[[LINE_(應用程式)|LINE軟體]]的想法，並於當年的6月在日本推出了LINE即時通訊軟體。當時日本的東北地區因地震導致電訊網路阻塞，而LINE的在線語言、聊天機制效果很好，這讓LINE迅速打開了日本的市場。

LINE在2012年大幅增加了市場份額和貼圖的種類，並出現了一些爆紅的貼圖[[卡通明星|卡通明星]]角色。<ref>{{cite web|last=Russell |first=Jon |url=http://thenextweb.com/asia/2013/07/12/stickers/ |title=Stickers: From Japanese Craze to Global Messaging Phenomenon |publisher=Thenextweb.com |date=2013-07-12 |accessdate=2013-08-16}}</ref>

== 競爭 ==
[[連我|LINE]]、[[臉書|臉書]]、[[微信|微信]]、[[Viber|Viber]]、[[Telegram|Telegram]]、[[Google_Allo|Google Allo]]、[[KakaoTalk|KakaoTalk]]等各種軟體都相繼推出了貼圖功能，但截至目前最為成功的是LINE軟體。

== 盈利 ==
LINE於2013年4月底推出了貼圖商店，用戶可以在裡面付費購買各種心儀的貼圖。據[[NHN|NHN]]透露，2013年8月份貼圖的銷售為[[LINE公司|LINE公司]]帶來了3億日元的營收。
