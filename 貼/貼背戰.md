{{作品名稱|name=''Tag''|hk=捉依因同學會|tw=貼背戰|cn=抓人游戏}}是一部2018年[[美國|美國]][[喜劇片|喜劇片]]，由傑夫·托姆希奇（{{lang|en|Jeff Tomsic}}）執導，{{le|羅伯·麥基裡克|Rob McKittrick}}與馬克·史泰倫（{{lang|en|Mark Steilen}}）共同撰寫劇本。故事描述一群好友每年5月都會玩起捉人遊戲而引發一連串的鬧劇；其主演包括[[艾德·赫姆斯|艾德·赫姆斯]]、[[傑克·強森_(演員)|傑克·強森]]、[[安娜貝爾·瓦莉絲|安娜貝爾·瓦莉絲]]、[[漢尼拔·布瑞斯|漢尼拔·布瑞斯]]、[[艾拉·費雪|艾拉·費雪]]、{{le|拉西達·瓊絲|Rashida Jones}}、{{le|蕾絲莉·畢柏|Leslie Bibb}}、[[喬·漢姆|喬·漢姆]]和[[傑瑞米·雷納|傑瑞米·雷納]]。該片的故事靈感取自《[[華爾街日報|華爾街日報]]》上的一篇文章。《貼背戰》定於2018年6月15日在美國上映。

== 劇情 ==
故事敘述五名好朋友在每年5月都會玩起鬼抓人的遊戲，以維持彼此的友誼。而當他們當中最厲害的傑瑞（{{lang|en|Jerry}}）將要結婚時，另外四人本以為這是能打敗他的方法，但未料，對方卻早已做好萬全的準備<ref>{{cite news|url=https://star.ettoday.net/news/1141971|title=鷹眼《復仇者3》消失去哪？新預告洩蹤跡網友全笑瘋|work=ETtoday星光雲|date=2018-03-31|accessdate=2018-03-31}}</ref>。

== 製作 ==
《貼背戰》的故事靈感取自2013年1月《[[華爾街日報|華爾街日報]]》上的一篇文章，是關於捉人遊戲中當鬼需要的準則；一班高中認識的朋友每年二月都會玩這項遊戲<ref name="slashtag">{{cite web|last1=Lussier|first1=Germain|title=Real Life 23-Year-Long Game of Tag To Be Turned into a Movie|url=http://www.slashfilm.com/real-life-23-year-long-game-of-tag-to-be-turned-into-a-movie/|website=Slashfilm|accessdate=2018-03-23}}</ref>。這篇文章引起了製片人{{le|陶德·嘉納|Todd Garner}}和馬克·史泰倫的注意，並在經過會談後，決定將這個故事拍成電影。隔月，他們將故事版權賣給了嘉納和史泰倫<ref name="slashtag" />。最初，該片鎖定[[威爾·法洛|威爾·法洛]]與[[傑克·布萊克|傑克·布萊克]]主演<ref>{{Cite news|url=http://deadline.com/2013/05/medieval-times-headed-for-movie-screen-497797/|title=Medieval Times Headed For Movie Screen|last=Fleming Jr.|first=Mike|date=2013-05-13|work=Deadline|access-date=2017-08-12}}</ref>，但兩人在後來都退出了該項目，而改由[[艾德·赫姆斯|艾德·赫姆斯]]、[[傑克·強森_(演員)|傑克·強森]]和[[傑瑞米·雷納|傑瑞米·雷納]]等人主演。在拍攝期間，雷納曾弄斷了他的右肘和左手腕<ref>{{cite web|url=https://variety.com/2017/film/spotlight/jeremy-renner-broke-his-arms-stunt-avengers-infinity-war-1202489347/|title=Jeremy Renner Fractured Both Arms During Stunt Gone Wrong|first=Leo|last=Barraclough|date=2017-07-07|website=Variety.com|accessdate=2017-10-04}}</ref>。

== 發行 ==
《貼背戰》在美國於2018年6月15日上映；臺灣原定於2018年6月29日上映<ref name="Trailer">{{cite web|url=https://www.youtube.com/watch?v=DwViVZ7COck|title=【貼背戰】中文官方預告，6月29日(週五) 抓交替|publisher=華納兄弟台灣粉絲俱樂部|work=Youtube|date=2018-03-30|accessdate=2018-03-31}}</ref>，後提前至6月22日上映。
=== 評價 ===
《貼背戰》整體獲得了好壞參半的評價。[[爛番茄|爛番茄]]根據169條評論，此片持有55%的新鮮度，平均得分為5.4／10；該網站共識：「對於尋求一些高概念但要求不高的動作喜劇的觀眾來說，《貼背戰》可能足夠接近它。」<ref>{{Cite web |title=Tag (2018) |url=https://www.rottentomatoes.com/m/tag_2018|website=[[Rotten_Tomatoes|Rotten Tomatoes]]|publisher=Fandango Media|accessdate=2018-06-26}}</ref>該片在[[Metacritic|Metacritic]]上則獲得了57分，這代表了褒貶不一的評價<ref>{{Cite web |title=Tag Reviews |url=http://www.metacritic.com/movie/tag|website=[[Metacritic|Metacritic]]|publisher=[[CBS_Interactive|CBS Interactive]]|accessdate=2018-06-19}}</ref>。
