'''熱帶神經衰弱'''-{zh-tw:，又稱'''熱帶狂亂'''、'''南洋呆'''，;zh-hk:，;zh-hans:，}-最早出現於[[1902年|1902年]]，作為[[神經衰弱|神經衰弱]]的延伸，主要用以表示[[白人|白人]]前往[[熱帶|熱帶]][[殖民地|殖民地]]後，一系列身體不適的症狀，產生神經系統的機械性衰弱。<ref>{{cite web |title = 台電綠網-熱帶神經衰弱| url = http://greennet.taipower.com.tw/Page_Knowledge/113}}</ref><ref>{{cite book|author=李尚仁 |title = 《帝國與現代醫學》| url = https://books.google.com.tw/books?id=RQN2CgAAQBAJ&printsec=frontcover&dq=%E5%B8%9D%E5%9C%8B%E8%88%87%E7%8F%BE%E4%BB%A3%E9%86%AB%E5%AD%B8&hl=zh-TW&sa=X&ved=0ahUKEwisxbKk0pHaAhXImpQKHX4DAJUQ6AEILzAB#v=onepage&q&f=false}}</ref>

1930年代，[[日本南方醫學研究會|日本南方醫學研究會]]亦做了一次500人的調查，發現在[[台灣|台]]居住一段時間的日本工作者，有相當比例得到了與熱帶精神衰弱相似的症狀。<ref>{{cite web |title = 【熱帶神經衰弱】日治時期，在台日人效率與道德流失的惡夢 | url = https://www.ettoday.net/news/20180314/1130196.htm}}</ref><ref>{{cite web |title = 外派到台灣就變懶散、遲鈍？讓日本殖民政府超苦惱的「南洋呆」，究竟是什麼離奇病 | url = http://www.storm.mg/lifestyle/380578}}</ref>

而這個疾病，也成為1930年代[[大日本帝國|大日本帝國]]政府[[南進|南進]]派與[[中國東北|中國東北]]派之間的重要議題之一。

==症狀==

身體[[疲倦|疲倦]]、怕冷、[[嗜睡|嗜睡]]、[[食慾不振|食慾不振]]等。

精神上則是[[憂鬱|憂鬱]]、[[注意力渙散|注意力渙散]]，[[思考|思考]][[邏輯|邏輯]]能力下降等。

==歷史==

[[神經衰弱|神經衰弱]]為19世紀出現的用語，被認為是因為[[文明|文明]][[壓力|壓力]]的關係造成的精神疾病。

而[[熱帶神經衰弱|熱帶神經衰弱]]此一名詞最早出現於[[1902年|1902年]]，主因是許多[[殖民者|殖民者]]進入熱帶殖民地一段時間後，常出現的一系列身體不適，因症狀與神經衰弱相似，故加上熱帶以形容此新的症狀。

最初被認為起因為熱帶地區的濕熱[[氣候|氣候]]，高溫與潮濕的狀態下，使得身體產生的能量下降，導致[[神經|神經]]的衰弱。

也因為這樣子的關係，前往殖民地的白人常被賦予勇猛，為[[國家|國家]]犧牲奉獻的形象，成為一種光榮的象徵。


然而到了[[1920年代|1920年代]]，[[精神分析|精神分析]]成為白人地區的主流理論。此一轉變使得熱帶神經衰弱的[[致病因|致病因]]由氣候轉變為[[壓抑|壓抑]][[慾望|慾望]]。

因為殖民地區缺乏舒適以及調適身心的[[娛樂|娛樂]]，長期導致身體的衰退。


[[1915年|1915年]]，美國地理學家[[Ellsworth_Huntington|Ellsworth Huntington]]在自己的著作中，將世界各地的氣候做劃分，1到100分，越高分越適合文明發展，歐美地區多獲得80以上的分數，而熱帶地區則常只獲得相當低的分數。

結合此論點，加上當時[[德國|德國]]的[[退化論|退化論]]，在不適合文明發展的地區居住並傳宗接代，身體會逐漸的退化，成為低等[[人種|人種]]，攜帶著劣質[[基因|基因]]。


此外，德國精神醫生[[Eugen_Paul_Bleuler|Eugen Paul Bleuler]]則認為，精神疾病是源於[[逃入疾病|逃入疾病]]的心理機制，藉由逃避現實取得個人利益。

因為此理論，此疾病被認為可能是因為殖民地[[開拓者|開拓者]]因無法適應，因此藉病想返回家鄉。


也因為這些理論，熱帶神經衰弱逐漸由正面轉成負面的疾病。

也導致了1920年後，殖民地[[開拓者|開拓者]]被歧視的現象。


而精神疾病教授[[中脩三|中脩三]]則用[[心因性|心因性]]，並引用[[森田正馬|森田正馬]]的理論去做解釋。

他認為此疾病是源於[[神經質|神經質]]，因為過度靈敏導致身體對於小缺陷過度放大，產生的過度反應。

並且他認為，對自我要求強烈是[[日本人|日本人]]引以為傲的民族性，因此在殖民地不適的環境，正是因為其強烈的民族性，產生的疾病。

==相關紀錄==

*[[林芙美子|林芙美子]]在其著作《[[浮雲|浮雲]]》寫到：「植物若不是生長在它們的土地上，一定無法生長得很好。」以及「我們失去了在內地時所擁有的旺盛靈魂……像被移植的日本杉，一點一點地枯萎……無意間，我們都成了南洋呆的犧牲品。」

*[[大喜多孝|大喜多孝]]則形容日本的櫻花在台灣「紅得像有毒一般，笨拙地開著」。
