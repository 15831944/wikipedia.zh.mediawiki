'''熱河戰役'''，又叫'''热河事变'''，發生於1933年2月至3月，位於[[中華民國|中華民國]][[熱河省_(中華民國)|熱河省]]。由於[[滿洲國|滿洲國]]成立時，《[[建國宣言|建國宣言]]》中即曾表達過熱河為滿洲一部份。於是[[日本|日本]]即方面根據《[[日滿議定書|日滿議定書]]》，積極侵略熱河。熱河戰役首先於1933年1月爆發的是位於[[榆關|榆關]]的[[山海關戰鬥|大型戰鬥]]。[[張學良|張學良]]以保存實力為由，退居[[山海關|山海關]]之內，積極謀求抵抗。日本軍隊駐守長城外，佔領熱河直取[[北平|北平]]態勢相當明顯，不過就整體而言，日本是想藉由攻擊北平行動，換取[[中國|中國]][[國民政府|國民政府]]對[[滿洲國|滿洲國]]的承認。

因為中國國內輿論普遍不願意承認滿洲國，2月11日，國民政府[[行政院長|行政院長]][[宋子文|宋子文]]至北平，與包括張學良等27名將領一起發表「保衛」熱河通電。2月21日，熱河戰役爆發。裝備不良，士氣低落的東北軍節節敗退，3月4日[[承德|承德]]失守，熱河抗戰結束。熱河最後遭日軍佔領。

== 背景 ==
熱河本身南有[[長城|長城]]可為屏障，西有[[蘇克斜魯山脈|蘇克斜魯山脈]]；東南與[[遼寧|遼寧]]相接。[[1932年|1932年]][[滿洲國|滿洲國]]成立時，《[[建國宣言|建國宣言]]》中即曾表達過熱河為[[滿洲|滿洲]]一部份。在[[九一八事變|九一八事變]]侵占東北後，按照計劃繼續侵略熱河，以達到蠶食中國之目的。1932年7月3日，日本[[參謀本部|參謀本部]]次長[[真崎甚三郎|真崎甚三郎]]到錦州與[[關東軍|關東軍]]將領策劃進攻熱河。<ref name="民國史第八卷，第140頁">李新總主編，中國社會科學院近代史研究所中華民國史研究室編，周天度、鄭則民、齊福霖、李義彬等著，《中華民國史》第八卷，北京：中華書局，2011年7月，第140頁</ref>7月11日下午7時，關東軍特派員[[石本權四郎|石本權四郎]]被捕。7月17日，日軍藉口軍事聯絡員石本權四郎等3人在[[朝陽|朝陽]][[北票|北票]]之間失蹤，以武裝搜索為名，派日軍300餘人攻佔熱河邊境之朝陽寺。<ref name="民國史第八卷，第140頁"/>比為第二個[[中村事件|中村事件]]。7月23日，數架日軍飛機飛至熱河[[平泉|平泉]]、[[凌源|凌源]]、[[承德|承德]]一帶偵察，並散發傳單。<ref name="民國史第八卷，第140頁"/>。[[熱河省|熱河省]]政府主席[[湯玉麟|湯玉麟]]允諾速救出石本。7月29日，石本被覓得交與日軍。同時日本[[陸軍大臣|陸軍大臣]][[荒木貞夫|荒木貞夫]]則表示「熱河省歸屬滿洲，為國際調查團所熟知」。國民政府命令設置[[臨永|臨永]]警備司令部，任命[[東北軍|東北軍]]獨立第九旅旅長何柱國兼任司令，管轄[[臨榆|臨榆]]、[[撫寧|撫寧]]、[[昌黎|昌黎]]、[[盧龍|盧龍]]、[[遷安|遷安]]5縣和在[[都山|都山]]設治局，統率[[步兵|步兵]]約2萬人，防守榆關地區。<ref name="民國史第八卷，第140頁"/>

8月19日，日軍一度佔領[[南嶺|南嶺]][[車站|車站]]，炸毁南嶺鐵橋，並不斷派飛機到熱河上空挑衅。<ref name="民國史第八卷，第140頁"/>

12月25日，在[[張學良|張學良]]調遣[[萬福麟|萬福麟]]率5個[[旅|旅]]陸續進入熱河後，[[蔣介石|蔣介石]]再電張學良，決定密備中央軍6個師隨時北運增援，並且表示：「[[倭寇|倭寇]]北犯侵熱，其期不遠」，「今日之事，惟有決戰可以挽救民心，雖敗猶可圖存，否則必為民族千古之罪人」<ref>〈蔣介石致張學良有電〉，刊[[秦孝儀|秦孝儀]]主編：《中華民國重要史料初編——對日坑戰時期》緖編﹙一﹚，第563頁</ref>

== 榆關戰鬥 ==
1933年1月爆發[[山海關戰鬥|榆關戰鬥]]。[[張學良|張學良]]退至[[山海關|山海關]]之內，積極謀求抵抗，並且於1月23日爆發榆關戰鬥。1月11日，[[日本|日本]][[陸軍省|陸軍省]]發表聲明：「熱河省係舊『[[東北四省|東北四省]]』之一，與其他三省有不可分之關係」。1月21日，外相[[內田康哉|內田康哉]]在[[貴族院|貴族院]]作外交方針演說時稱：「滿蒙與中國系以[[長城|長城]]為境界者，由[[歷史|歷史]]而言，亦無議論之餘地。尤以[[熱河省|熱河省]]之屬於[[滿洲國|滿洲國]]之一部者，徵諸該國建國之[[經緯|經緯]]，當可明了。……《[[日滿議定書|日滿議定書]]》，規定滿洲國領土由兩國共同負責維持治安。故所謂『熱河問題』，為滿洲國內部問題，同時日本因有[[條約|條約]]上的義務，對此抱重大關心。」<ref>《國聞周報》，[[天津|天津]]，第十卷第五期</ref>此些言論一出即已表達其欲攻打熱河之意。

== 戰役過程 ==
[[File:Japanese_Army_entering_Chaoyang.jpg|thumb]]
[[File:Japanese_Army_in_Chengde,_1.jpg|thumb]]
[[File:Japanese_Army_in_Chengde,_3.jpg|thumb]]
1933年2月21日，熱河戰役的前哨戰[[南嶺戰鬥|南嶺戰鬥]]開打。2月21日，熱河戰役爆發。日本人分三線配置兵力：第一線左翼為[[滿洲國軍|滿洲國軍]][[張海鵬_(軍事將領)|張海鵬]]五隊；第二線新編[[满洲国军|滿軍]]李、程等五軍。第三線日本皇軍錦朝線第7、第8師團23,000餘人及飛機四大隊、綏凌線第14、第7師團25,000餘人及飛機二大隊。至於中國方面，由於湯玉麟拒絕其他軍隊入境，因此分為三線：一為開魯、魯北、綏東，由[[崔興武|崔興武]]率4,500人，[[孫殿英|孫殿英]]增援：二為[[朝陽|朝陽]]，湯玉麟、董福亭負責；三為[[凌源|凌源]]，[[萬福麟|萬福麟]]部組成。2月22日下午五時，日軍200人佔劉龍台。[[中國|中國]][[北票|北票]][[董福亭|董福亭]]軍退至[[朝陽|朝陽]]；[[開魯|開魯]][[崔興武|崔興武]]軍暗中投降，日軍與[[孫殿英|孫殿英]]部激戰，孫退守[[圍場|圍場]]。2月23日，日本使館秘書[[上村伸一|上村伸一]]向[[中國|中國]]交節略要求撤出熱河軍隊遭拒。2月24日，日軍受到北票商民往迎，隨即[[朝陽戰鬥_(抗日戰爭)|朝陽戰鬥]]開打。2月26日，日軍進佔[[朝陽|朝陽]]，中方不戰而退，爆發[[凌南戰鬥|凌南戰鬥]]。3月1日，駐[[平泉|平泉]]萬扶麟至[[凌源|凌源]]視察，聞前方戰況不佳，即返平泉。同時[[凌源|凌源]]各機關退至[[平泉|平泉]]，爆發[[赤峰戰鬥|赤峰戰鬥]]。3月2日日軍隨即進入[[凌源|凌源]]。3月3日，熱河省主席[[湯玉麟|湯玉麟]]以為日軍已陷平泉，遂率1,000餘人逃走。至[[承德|承德]]西20里的[[廣仁嶺|廣仁嶺]]得知[[赤峰戰鬥|赤峰戰鬥]]勝利，凌源未失才返回。然日軍裝甲車已經逼近[[承德|承德]]僅90里的六溝。3月4日早上七時，湯玉麟再度棄城逃走。八時日機飛[[承德|承德]]，轟炸湯玉麟軍，十時日軍達距承德30里的紅石嵐。下午一時[[日本皇軍|日本皇軍]]裝甲車開進承德，而日軍竟只128名骑兵即陷[[承德|承德]]。[[熱河省|熱河省]]十九萬[[平方公里|平方公里]]的領土遂併入[[滿洲國|滿洲國]]國土。

== 敗戰原因 ==
湯玉麟主政熱河八年，以其子[[湯佐輔|湯佐輔]]之言是從。賣官種煙、苛捐雜稅，甚至摧殘教育、破壞司法。不戰而逃被認為是敗戰的原因。其軍隊四師，號稱三萬，實則二萬，欠餉10月。以致於民怨沸騰，歡迎日滿聯軍。加上[[張學良|張學良]]軍事指揮失敗，其時據守[[熱河|熱河]]的軍閥[[湯玉麟|湯玉麟]]是張學良父執輩，擁兵自重，根本指揮不動，也不允許張學良部進入熱河，甚至有降日傾向。蔣建議張迅速解決湯，但計劃洩露，為避免大戰之前先內訌，爭取湯積極抵抗，張只能暫時將所部佈置在熱河外圍以穩住湯，並派去[[張作相|張作相]]督戰。都是敗戰原因。

在那時一期《[[獨立評論|獨立評論]]》曾刊登熟悉熱河和華北戰略地理的[[丁文江|丁文江]]教授所攥寫的《假如我是張學良》一文，明白指出：
{{Quote|
<p>“一旦熱河有了軍事行動，北京天津是萬萬守不了的。”</p>
<p>“我們真正的防禦、長期的戰爭，不在平津，而在熱河。假如我是張學良，要預備積極抵抗，第一步先把司令部移到張家口，同時把重要的軍實和北甯路車輛逐次的運到居庸關以北；只留一部分軍隊在山海關、秦皇島、天津等處；在這些幾處經過相當的抵抗以後，也預備從冷口、喜峰口、古北口分別退至口外。現在駐在熱河邊界的軍隊應該從速進到朝陽、並且積極籌備朝陽、淩原，平原、承德各地間的運輸。熱河東南兩部完全是山地，不但日本人的坦克重炮都不能使用，就是飛機也有許多危險。喜峰、古北、河南口三處都是天險，每處有一兩萬人防守，日本非有一倍以上的軍力不能進攻。”</p>
<p>“只要守得住熱河，放棄了平津是不足惜的。只要當局有必死的決心，充分的計畫，熱河是一定守得住的。”</p>
}}

一個多月以後，熱河佈防失當，丁文江又撰寫一文：

{{quote|<p>“熱河部隊只有四步軍旅、六騎兵旅，合計不過二萬支槍。朝陽、北原的守兵一共不及四千多支槍。日本如在錦州、義縣進兵，該地防軍就沒有抵抗的能力！”</p>
<p>“我們現在將二十旅兵力全放在冀察兩省，而將熱河交給湯玉麟去防守，這是什麼戰略？我不懂！”</p>
}}
