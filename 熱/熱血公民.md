'''熱血公民'''（{{lang-en|Civic Passion}}）是一個[[本土派_(香港)#城邦派|城邦派]][[政黨|政黨]]，於2012年2月29日成立，以「文化抗共」的旗幟號召香港「文化建國」，核心人物有[[鄭松泰|鄭松泰]]及[[鄭錦滿|鄭錦滿]]<ref>{{cite web| url=https://www.youtube.com/watch?v=qDakTdXIzkk |title=黃洋達、陳秀慧同熱血讀者拜年!}}</ref>。其主張為「[[五區公投|五區公投]]，[[香港人|全民]][[制憲議會|制憲]]」及「永續基本法」。

== 背景 ==
[[黃洋達|黃洋達]]、陳秀慧為競選[[2012年香港立法會選舉|2012年香港立法會選舉]][[九龍東選區|九龍東選區]]議席，於2月29日成立熱血公民組織為兩人在3月1日至9月9日選舉期間助選。黃洋達為熱血公民的首領，選舉後熱血公民營運《[[熱血時報|熱血時報]]》網站、網台、出版刊物，以及在香港各處主辦及協辦多項本土意識活動。由於在過往行動或活動中常發生言語及肢體衝突，熱血公民成員常被敵對組織謔稱為「[[熱狗|熱狗]]」<ref>[http://the-sun.on.cc/cnt/news/20141116/00410_020.html SUN潮語：【熱狗】@太陽報]</ref><ref>[https://www.youtube.com/watch?v=FTEEO7MnsAE 金鐘糾察長郭紹傑指罵「我知你係熱狗，你做乜影我」]</ref>。

在[[2016年香港立法會選舉|2016年香港立法會選舉]]中，[[鄭松泰|鄭松泰]]取得[[新界西選區|新界西選區]]議席，在九龍東選區落選的黃洋達將組織交由鄭松泰帶領，並於同年9月11日完成熱血公民政黨化改組，鄭松泰成為該黨主席，同年9月14日[[鄭錦滿|鄭錦滿]]擔任副主席一職。2017年初，熱血公民與黃洋達的《熱血時報》分家。

==社區參與==

===熱血貓民===
「熱血貓民」是熱血公民其中一個支隊，主要任務是成為持牌的貓隻領域護理計劃義工，為各區貓隻進行一連兩日的「捕捉、絕育、放回」工作，減少貓隻過量繁殖。他們會於晚上設置捕獸器，待翌日早上再帶被捉的貓隻往[[愛護動物協會|愛護動物協會]]進行[[絕育|絕育]]手術，到晚上再將牠們原區放回。另外，「熱血貓民」成員不時會在[[熱血時報|熱血時報]]網頁上發表有關[[保護動物|保護動物]]的文章<ref>{{cite web| url=http://www.passiontimes.hk/author/%E7%86%B1%E8%A1%80%E8%B2%93%E6%B0%91| title=熱血貓民|work= 熱血時報|accessdate=2016-03-18}}</ref>，亦會舉辦及協辦貓狗領養日、出席有關爭取寵物福利的活動及公聽會，同時亦組織清潔狗場或到學校向學生推廣愛護動物等義務工作。

===鄭松泰立法會議員辦事處===
在元朗、青衣及其他新界西地區舉辦不同關懷老幼的活動，例如親子工作坊、義務為安老院老人剪髮及贈湯活動等等。

===熱血鳳凰計劃===
建立一個平台，作為受政治檢控的釋囚在出獄後重返社會的起點，並提供持續性的服務及幫助。

===熱血登山隊===
主張「愛護山林，港人有責」，每月定期舉辦登山活動強身健體，並沿途提醒及教育山友盡公民責任愛惜山林動物及設施，強調「自己垃圾自己執」。

===好人經濟計劃===
透過應用程式鼓勵市民參與不同項目以拓展社區服務，當完成項目後參加者會得到積分獎勵，積存指定的積分可換用品或服務。

==政治參與==
===常規活動===
《熱血社區》第一期於2018年6月4日正式出版，其後每雙月出版一期免費在新界區派發。

《熱血時報》試刊號於2012年10月1日正式出版，其後每月第三或第四個星期日免費在各區派發，在2016年11月8日派發後暫停出版。在2017年7月19日開始的[[香港書展|香港書展]]派發復刊號，之後在各地區辦事處及街站派發。

===六四集會===
2013年在維園參加支聯會活動，主張「不要平反六四，我要中共倒台」。2014年在[[香港文化中心|香港文化中心]]自由戰士廣場主辦《香港人的六四集會》活動，出席人數約七千人。

2015年與[[普羅政治學苑|普羅政治學苑]]、[[本土民主前線|本土民主前線]]及[[香港本土力量_(2015年)|香港本土力量]]等[[本土派_(香港)|本土派]]團體聯辦「遍地開花六四集會」，於港九新界18個地點設立六四街站及巴士巡遊，並於晚上在尖沙咀鐘樓舉行集會。<ref>[http://www.passiontimes.hk/article/06-01-2015/23368 遍地開花64集會　扣連政改堅拒假普選] </ref> <ref>[http://www.metroradio.com.hk/News/live.aspx?NewsId=20150604210253 熱血公民在尖沙咀舉行晚會悼六四] </ref>

2016年與[[普羅政治學苑|普羅政治學苑]]及[[香港復興會|香港復興會]]主辦《香港人六四集會》活動，口號是「本土、民主、反共、建國」，時間為晚上8至10時，分別在港九5個地方舉行集會，包括﹕[[西灣河|西灣河]]、[[觀塘|觀塘]]、[[尖沙咀|尖沙咀]]、[[沙田|沙田]]及[[屯門|屯門]]。<ref>[https://www.facebook.com/passiontimes/photos/a.422169814512858.102977.420361564693683/1122506661145833/?type=3&theater 香港人六四集會@熱血時報] </ref>

2017年在港九新界十多個地方派發《香港變了，但我們不能失去希望》的宣傳單張。<ref>[http://www.passiontimes.hk/article/06-04-2017/38299 熱血公民六四深入社區　勉勵港人不要失去希望] 熱血時報 2017-04-06</ref>

2018年在港島及新界八個地方派發第一期《熱血社區》刊物。<ref>[https://www.facebook.com/chengchungtai/posts/954862141354810 《熱血社區》報首發．「六四」街站] </ref>

===七一街站===
熱血公民開始在2012年參與[[七一遊行|七一遊行]]。<ref>{{youtube|49LMBdSii4M| 『熱血公民』7.1 大遊行}}</ref>2013年晚上八時至十一時於灣仔循道衞理堂外行人路主辦「全民制憲起動集會」，約有1,000人參與。部份參加者輪流焚燒《基本法》，同時[[熱血時報|熱血時報]]在網頁上開放「全民網上制憲」系統，讓市民對《基本法》條文上寫上自己的意見。<ref>[http://www.passiontimes.hk/?view=cwiki 熱血政治 全民制憲]</ref>

2014年熱血公民與[[普羅政治學苑|普羅政治學苑]]於下午8時在灣仔循道衛理教會外行人路舉行「全民制憲大會」，向市民交代過去一年推動全民制憲的工作及未來策略。部份支持者焚燒[[立法會綜合大樓|立法會綜合大樓]]及[[政府總部_(添馬艦)|政府總部大樓]]模型，模型著火後迅即被警方撲熄及清理。另外，熱血公民於下午舉行的[[七一遊行|七一遊行]]路線沿途設立街站，進行傳訊工作。

2015年熱血公民在[[灣仔|灣仔]][[循道衛理香港堂|循道衛理香港堂]]外行人路設置街站派發熱血時報及招收成員。2016年熱血公民與[[普羅政治學苑|普羅政治學苑]]於銅鑼灣及灣仔設置8個街站，宣傳全民制憲運動。當晚熱血公民在灣仔循道衛理教會外行人路舉行「香港國旗升旗儀式」，該旗是來自早前《熱血時報》所舉辦的「香港國旗設計比賽」的勝出作品。<ref>[http://www.passiontimes.hk/article/07-01-2016/31392 熱血公民七一辦升旗禮　展示香港國旗設計比賽勝出作品]</ref>

===[[雨傘革命|雨傘革命]]===
熱血公民在2014年9月27日晚上在[[金鐘|金鐘]][[海富中心|海富中心]]外圍設置街站，聲援由[[香港專上學生聯會|香港專上學生聯會]]和[[學民思潮|學民思潮]]在9月26日所發動[[重奪「公民廣場」行動|重奪「公民廣場」行動]]。

2014年9月28日[[雨傘革命|雨傘革命]]發生，事件為期79天直至12月15日。熱血公民積極投入事件，在金鐘、銅鑼灣、旺角及尖沙咀4個佔領區皆有其成員參與其中，而在金鐘及旺角兩處更設立了物資站，收集損贈者物資及免費派發給參與者。另外與其直接聯繫之[[熱血時報|熱血時報]]，亦派遣記者在事件期間提供新聞資訊。

====2015年====
2015年1月25日熱血公民當天在各處派發最新一期《熱血時報》，下午亦在旺角西洋菜南街設置街站。成員在觀塘派發報紙時被人滋擾，雙方推撞有人受傷，6名熱血公民成員被警方拘捕，事後當晚黃洋達與熱血公民成員約100人在觀塘警署門口示威，6名熱血公民成員於深夜四時獲得保釋外出。

====2016年====
成員陳柏洋涉及[[2016年農曆新年旺角騷亂|2016年農曆新年旺角騷亂]]事件，因襲警及抗拒警務人員罪成，同年10月6日被判監九個月，不准保釋，即時入獄。其後在2017年1月26日因上訴獲准保釋。之後在2017年3月29日被高等法院駁回，即時再度收監。<ref>[http://news.now.com/home/local/player?newsId=195859 旺角騷亂陳柏洋案　黃毓民批判刑過重] </ref><ref>[https://www.881903.com/Page/ZH-TW/newsdetail.aspx?ItemId=931810&csid=261_341 法官駁回陳柏洋上訴指法庭不姑息或縱容襲警行為] </ref>

成員[[鄭錦滿|鄭錦滿]]在其Facebook專頁，發表題為「香港圖書館戰爭 自發將『殘體書』下架抗洗腦」的影片，片段中鄭錦滿指自己在旺角的圖書館，將簡體字書丟入垃圾桶、又將部分簡體字書放在書架與牆之間的罅隙，又收藏於滅火筒櫃中。警方網絡安全及科技罪案組人員在4月6日12時半，在筲箕灣港鐵站附近的一個街站拘捕鄭錦滿，涉嫌觸犯不誠實使用電腦和盜竊罪，案件押後至2017年9月19日再訊，鄭錦滿被控一項盜竊罪，在九龍城法院被裁定罪名成立，罰款3,000元。<ref>[http://hd.stheadline.com/news/realtime/hk/1016579/  鄭錦滿扔圖書館9本簡體字書 盜竊罪成]</ref>

8月23日[[有線電視|有線電視]]在九龍城「[[香港兆基創意書院|香港兆基創意書院]]」舉行立法會選舉論壇，代表「[[本土力量|本土力量]]」的九龍西選區候選人[[何志光|何志光]]，在論壇上狙擊候選人[[黃毓民|黃毓民]]，論壇完結後，「熱血公民」及「本土力量」兩派人馬在場外衝突，其間「熱血公民」成員李政熙及朱寶生疑因拉扯警員被控以襲警罪，兩人事後退出「熱血公民」。朱寶生在2017年7月11日在九龍城法院承認控罪，在2017年8月8日被判入獄4星期，李政熙否認襲警，在2017年12月12日被九龍城法院裁定兩項襲警罪成立，在2017年12月27日被判入獄7星期。<ref>[http://hk.on.cc/hk/bkn/cnt/news/20170711/bkn-20170711124902511-0711_00822_001.html 前熱血公民成員襲警囚4周　官斥趁混亂犯事</ref><ref>[https://news.mingpao.com/ins/instantnews/web_tc/article/20171228/s00001/1514438609750 熱血公民前成員襲警罪成囚7周　官：控罪嚴重須阻嚇 </ref>

大學學生周韋樂9月1日立法會選舉期間在觀塘翠屏邨替黃洋達助選時，因與中年女街坊政見不合執起啤酒樽扑落對方頭部，幸玻璃樽未有破裂，周韋樂事後被捕並認罪，被裁判官判處24個月感化令。<ref>[http://hk.apple.nextmedia.com/realtime/news/20160919/55664527 身穿熱普城上衣助選 嶺大學生一言不合扑傷人]</ref>

====2017年====
副主席[[鄭錦滿|鄭錦滿]]在2014年佔領行動期間，阻礙法庭執達吏執行在旺角禁制令，被控藐視法庭，鄭錦滿早前承認控罪，在2017年3月30日被高等法院判即時入獄三個月，不得保釋。<ref>[https://www.881903.com/Page/ZH-TW/newsdetail.aspx?ItemId=931810&csid=261_341 違佔旺禁制令 熱血公民鄭錦滿判囚三月]</ref>

主席[[鄭松泰|鄭松泰]]在2016年10月19日立法會第二次會議時，在點算人數期間，[[民建聯|民建聯]]議員全部走出議事廳之時，鄭松泰走到民建聯議員坐席位置，並將議員自行帶來插在枱上的五星國旗和香港區旗上下倒轉。2017年4月10日鄭松泰稱接獲中區警署通知，警方將會正式落案起訴鄭松泰「侮辱國旗罪」及「侮辱區旗罪」，翌日晚上十時鄭松泰在深水埗街頭被便衣警員帶往中區警署落案之後可離開警署。裁判官在2017年9月29日裁定鄭松泰兩項控罪均成立，每項控罪各判罰款2,500港元，合共罰款5,000港元。<ref>{{cite web|title=議事廳上倒插國旗區旗 鄭松泰罪成判罰5,000元|url=http://hk.apple.nextmedia.com/realtime/news/20170929/57269669|publisher=蘋果日報|accessdate=2017-09-29|date=2017-09-29}}</ref>

===五區公投，全民制憲===
2016年9月﹕[[香港復興會|香港復興會]]、熱血公民及[[普羅政治學苑|普羅政治學苑]]聯合參與立法會地區直選。三個組織派人於五區參選，以「五區公投，全民制憲」為共同綱領。[[黃毓民|黃毓民]]、[[陳雲_(學者)|陳雲]]、[[黃洋達|黃洋達]]、[[鄭松泰|鄭松泰]]及[[鄭錦滿|鄭錦滿]]向傳媒公佈參選計劃，並以「爭取民意授權，推動全民制憲」為目標。<ref>[http://www.passiontimes.hk/article/02-29-2016/29063 熱血普羅復興會宣佈參選立會 冀爭五區授權 推動全民制憲]</ref>但最終只有鄭松泰當選。

==選舉==
===[[2012年香港立法會選舉|2012年香港立法會選舉]]===
黃洋達與妻子陳秀慧於2012年香港立法會選舉中，以同一張名單競選[[九龍東|九龍東]]議席，熱血公民遂成立為兩人助選，成員在[[觀塘區|觀塘區]]及[[黃大仙區|黃大仙區]]等設立街站宣傳，最後黃洋達以1,938票之差敗給[[建制派|建制派]]的[[謝偉俊|謝偉俊]]。<ref>{{cite web| url=https://www.facebook.com/MPNews2012/posts/423281627728725 |title=九東謝偉俊險勝黃洋達| publisher=[[Facebook|Facebook]]}}</ref>

===[[2015年香港區議會選舉|2015年香港區議會選舉]]===
熱血公民派出六名成員參選，分別為[[鄭松泰|鄭松泰]]（[[屯門|屯門]]樂翠選區）、[[李政熙|李政熙]]（[[北區|北區]]天平西選區）、[[方智龍|方智龍]]（[[深水埗|深水埗]]元州及蘇屋選區）、[[梁𧙗敬|梁𧙗敬]]（[[黃大仙|黃大仙]]慈雲西選區）、[[黃潤基|黃潤基]]（[[南區|南區]]利東一選區）及[[蔡文龍|蔡文龍]]（南區利東二選區）。<ref>{{cite web| url= http://www.passiontimes.hk/article/11-10-2015/26212|title=熱血公民出選2015區議會選舉 鄭松泰等6人「為香港人身份參選」|publisher=熱血時報|date=2015-11-10}}</ref>，結果六人全數落選，熱血公民首領黃洋達承認失敗。<ref>{{cite web| url=http://www.passiontimes.hk/article/11-23-2015/27042|title=黃洋達承認區選失敗向支持者致歉 強調熱血公民會堅持抗爭路線 |publisher=熱血時報|date=2015-11-23}}</ref>

===[[2016年香港立法會選舉|2016年香港立法會選舉]]===
熱血公民聯同[[普羅政治學苑|普羅政治學苑]]及[[香港復興會|香港復興會]]組成[[熱普城|熱普城]]聯盟，參與2016年香港立法會選舉[[地區直選|地區直選]]，以「五區公投，全民制憲」為共同綱領。

熱血公民首領'''[[黃洋達|黃洋達]]'''出戰[[九龍東選區|九龍東選區]]；成員'''[[鄭松泰|鄭松泰]]'''及[[張耀心|張耀心]]合組一張名單出戰[[新界西選區|新界西選區]]；成員'''[[鄭錦滿|鄭錦滿]]'''及[[鍾琬媛|鍾琬媛]]合組一張名單出戰[[香港島選區|香港島選區]]；成員[[李珏熙|李珏熙]]排在[[香港復興會|香港復興會]]主席'''[[陳云根|陳云根]]'''名單第二位在[[新界東選區|新界東選區]]出選；[[普羅政治學苑|普羅政治學苑]]'''[[黃毓民|黃毓民]]'''及[[馬愉生|馬愉生]]在[[九龍西選區|九龍西選區]]出戰。

結果，熱普城聯盟只有鄭松泰成功獲選。選舉翌日黃洋達宣佈選舉失利。黃洋達辭去領導職位及離開熱血公民交棒給鄭松泰，其後增設副手職位給鄭錦滿。
