'''《熱愛人生》'''（{{lang-sr|'''Љубав живи'''}} [[轉寫|轉寫]]：'''Ljubav živi'''）是[[塞爾維亞|塞爾維亞]]著名流行女歌手[[戚𡤕|Ceca]]的第十四張專輯，發布日期爲2011年6月17日，曲風融合了[[獨立搖滾|獨立搖滾]]和[[流行-民謠|流行-民謠]]。<ref>{{cite web |url=http://ceca.rs/muzika/diskografija/ljubav-zivi-2011-album/|title=Ljubav živi (2011) album|language=sr |work=Ceca|}}</ref>  該專輯祇推出了一隻[[雷射唱片|雷射唱片]]版本 <ref>{{cite web|url=https://www.discogs.com/Ceca-Ljubav-%C5%BDivi/release/2990550|title=Ceca – Poziv|publisher=discogs.com|work=Discogs}}</ref>

== 關於專輯 ==
久違樂壇五年，戚𡤕終於推出名爲「熱愛人生 (Љубав живи)」的新專輯。這張專輯錄製於2010年和2011年，是一張[[概念專輯|概念專輯]]。今次該專輯的風格較爲特別，戚𡤕在歌曲裏，注入了一些歐美[[獨立搖滾|獨立搖滾]]的元素，然後再把[[流行-民謠|流行-民謠]]融合在一起。 此外，雖然該專輯沒有作任何宣傳，也沒有排行唱片銷售榜榜首，但它的銷量於[[塞爾維亞|塞爾維亞]]也超過十五萬張。<ref>{{cite web |url=http://www.vbs.rs/scripts/cobiss?ukaz=DISP&id=1551272655645145&rec=6&sid=3|title= Ljubav živi |language=sr |work=Ceca}}</ref>而所有歌曲的歌詞影片，均在2011年6月17日的晚上，上傳到[[米鴷鴐毫克音樂|米鴷鴐毫克音樂]]的YouTube頻道。那些歌詞影片，錄得超過一億點擊率。 最後，歌曲「紛紜雜沓 (Расуло) 」十分流行 ，其音樂錄影帶於2012年5月3日晚上，上傳到YouTube，點擊率不俗。
