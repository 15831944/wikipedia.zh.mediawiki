'''熱帶風暴奧麥斯'''（[[英語|英語]]：{{lang|en|Tropical Storm Omais}}，菲律賓大氣地理天文部門：'''Agaton'''，國際編號：'''1001'''，[[JTWC|JTWC]]：'''02W'''）為[[2010年太平洋颱風季|2010年太平洋颱風季]]的第一個被日本氣象廳編號的[[熱帶氣旋|熱帶氣旋]]。

==發展過程及路徑==
{{NoteUTC+8}}

=== 橫過西北太平洋 ===
一熱帶擾動於[[3月17日|3月17日]]在北緯4度，東經149度附近形成，<ref>{{Cite web|url = http://www.nrlmry.navy.mil/tc-bin/tc_display.cgi?STATUS=inactive&ACTION=Latest_Photos&ACTIVES=None&MOSAIC_SCALE=20&PHOT=yes&ATCF_BASIN=wp&CGI=SATPRODUCTS&MO=MAR&AGE=Prev&SIZE=Thumb&NAV=tc&ATCF_YR=2010&ATCF_FILE=/SATPRODUCTS/kauai_data/www/atcf_web/public_html/image_archives/2010/wp022010.10032812.gif&CURRENT=20100317.2057.gms6.x.vis1km_high.98WINVEST.15kts-1010mb-40N-1487E.100pc.jpg&CURRENT_ATCF=wp022010.10032812.gif&ATCF_NAME=wp022010&ATCF_DIR=/SATPRODUCTS/kauai_data/www/atcf_web/public_html/image_archives/2010&STYLE=frames&YEAR=2010&YR=10&BASIN=WPAC&STORM_NAME=02W.OMAIS&ARCHIVE=all&PRODUCT=vis&DIR=/SATPRODUCTS/TC/tc10/WPAC/02W.OMAIS/vis/geo/1km_zoom&TYPE=vis&PROD=vis&SUB_PROD=geo&SUB_SUB_PROD=1km_zoom|title = NRL Tropical Cyclone Page|accessdate = 2015-05-15|author = 聯合颱風警報中心|date = 2010-03-17|publisher = |language = en}}</ref>其北面的副熱帶高壓脊帶領著它向西北偏西移動。該熱帶擾動在大致良好的環境下於[[3月22日|3月22日]]增強為一[[熱帶低氣壓|熱帶低氣壓]]，初時其對流旺盛并集中于其中心北部。<ref>参考图片：[http://www.nrlmry.navy.mil/htdocs_dyn_apache/tc_pages/thumbnails/thumbs/tc10/WPAC/02W.OMAIS/vis/geo/1km_zoom/thumb/20100321.2130.gms6.x.vis1km_high.02WTWO.25kts-1004mb-66N-1434E.100pc.jpg]</ref>該熱帶低氣壓于[[3月24日|3月24日]]晚間增強為[[熱帶風暴|熱帶風暴]]，並被命名為奧麥斯<ref name=":0" />。此時，奧麥斯亦已達到其強度顛峰，其中心附近最高持續風速達到每小時65公里<ref>{{Cite news|url = http://www.weather.com.cn/static/html/article/20100325/207376.shtml|title = 热带风暴“奥麦斯”将向北偏西方向移动|last = |first = |date = 2010-03-25|work = |accessdate = 2015-05-15|publisher = 中国天气网|language = zh-cn}}</ref>。

=== 減弱消散 ===
奧麥斯北面的副熱帶高壓脊於[[3月25日|3月25日]]減弱東退，令奧麥斯轉向東北移動，而奧麥斯亦受北面強風切及低水溫的海域影響減弱。它於[[3月26日|3月26日]]減弱為熱帶低氣壓，隨後漸漸消散<ref name=":0">{{Cite news|url = http://www.weather.com.cn/static/html/article/20100327/208065.shtml|title = 第1号热带风暴“奥麦斯”停止编号|last = |first = |date = 2010-03-27|work = |accessdate = 2015-05-15|publisher = 中国天气网|language = zh-cn}}</ref>。

==影響==
{{NoteUTC+8}}

=== 雅蒲島 ===
奧麥斯曾以熱帶低氣壓的強度在[[雅蒲島|雅蒲島]]以北掠過。由於奧麥斯強度偏弱，未有為當地帶來重大影響。雅蒲島風勢於[[3月23日|3月23日]]增強，但未錄得[[強風|強風]]，而陣風一度達烈風程度<ref>{{Cite web|url = http://www.wunderground.com/history/airport/PTYA/2010/3/23/DailyHistory.html|title = Hourly Weather History & Observations|accessdate = 2015-05-15|author = The Weather Channel, LLC|date = 2010-03-23|publisher = }}</ref>。
