'''熱帶性低氣壓WP012019'''（{{lang-en|Tropical Depression WP012019}}；臺灣[[中央氣象局|中央氣象局]]：'''02'''；[[菲律賓大氣地球物理和天文管理局|菲律賓大氣地球物理和天文管理局]]：Amang）是[[2019年太平洋颱風季|2019年太平洋颱風季]]首個被菲律賓大氣地球物理和天文管理局給予當地名稱的[[熱帶氣旋|熱帶氣旋]]。

該系統自1月2日於[[吉爾伯特群島|吉爾伯特群島]]東面形成後，未有明顯發展並一直西移至菲律賓附近，在1月中下旬為菲律賓中部帶來暴雨，造成山體滑動使7名礦工被活埋，截至25日3時已有5名確認已罹難。

==氣象歷史==
一個低壓區於1月2日在[[吉爾伯特群島|吉爾伯特群島]]東面海域形成，[[美國海軍研究實驗室|美國海軍研究實驗室]]給予[[熱帶擾動|熱帶擾動]]編號90W<ref>{{cite web|title=90W.INVEST Track file|url=https://www.nrlmry.navy.mil/tcdat/tc19/WPAC/90W.INVEST/trackfile.txt|date=2018年1月3日|author=美國海軍研究實驗室|archiveurl=http://webcache.googleusercontent.com/search?q=cache:jPBifh4qypMJ:gwydir.demon.co.uk/advisories/90WINVEST_201901030000.htm|archivedate=2018年1月5日|deadurl=yes}}{{en}}</ref>。

1月4日，[[聯合颱風警報中心|聯合颱風警報中心]]對此系統發出[[熱帶氣旋形成警告|熱帶氣旋形成警告]]<ref>{{Cite web|url=http://www.webcitation.org/75d5mATGu|title=wp9019熱帶氣旋形成警報(一月)|accessdate=2019-01-23|author=|date=2019-01-04|publisher=聯合颱風警報中心}}{{en}}</ref>，並在翌日將其升格為[[熱帶低氣壓|熱帶低氣壓]]，給予編號WP012019<ref>{{cite web |title=熱帶性低氣壓01W警告 Nr 001 |url=http://tgftp.nws.noaa.gov/data/raw/wt/wtpn32.pgtw..txt |publisher=聯合颱風警報中心 |accessdate=2019-01-08 |archiveurl=https://www.webcitation.org/75GICQ55V?url=http://gwydir.demon.co.uk/advisories/WTPN32-PGTW_201901042100.htm |archivedate=2019-01-08 |date=2019-01-04 |dead-url=no }}{{en}}</ref>，隨後此系統沒有預期的增強，反而低層環流中心變得不明顯，因此1月7日下午，聯合颱風警報中心將其降格為低壓區，並在晚上11時對其發出最後報告<ref>{{cite web |title=熱帶性低氣壓01W警告 Nr 008 |url=http://tgftp.nws.noaa.gov/data/raw/wt/wtpn32.pgtw..txt |publisher=聯合颱風警報中心 |accessdate=2019-01-08 |archiveurl=https://www.webcitation.org/75GIMubDm?url=http://gwydir.demon.co.uk/advisories/WTPN32-PGTW_201901061500.htm |archivedate=2019-01-08 |date=2019-01-06 |dead-url=no }}{{en}}</ref>，並在之後於1月8日發布的最佳路徑中，認為該系統只有擾動強度<ref name="聯合">{{cite web|title=bwp012019.dat |url=http://ftp.emc.ncep.noaa.gov/wd20vxt/hwrf-init/decks/bwp012019.dat |publisher=聯合颱風警報中心 |accessdate=2019-01-08 |date=2019-01-08}}{{en}}</ref>，之後穩定的向西移動。

1月19日下午2時，[[日本氣象廳|日本氣象廳]]將其升格為熱帶低氣壓<ref>{{cite web |title=WWJP25 RJTD 190600 |url=http://tgftp.nws.noaa.gov/data/raw/ww/wwjp25.rjtd..txt |publisher=日本氣象廳|accessdate=2019-01-19 |archiveurl=https://web.archive.org/web/20190119085114/http://tgftp.nws.noaa.gov/data/raw/ww/wwjp25.rjtd..txt |archivedate=2019-01-19 |date=2019-01-19}}{{en}}</ref>，[[菲律賓大氣地球物理和天文管理局|菲律賓大氣地球物理和天文管理局]]於晚上11時跟隨，給予當地名稱「Amang」<ref name="S1"/>。該部門隨後表示，系統在20日晚上8時登陸於[[锡亚高岛|锡亚高岛]]<ref>{{Cite news|url=https://www.rappler.com/nation/special-coverage/weather-alert/221467-tropical-depression-amang-pagasa-forecast-january-20-2019-11pm|title=熱帶低氣壓Amang在Siargao登陸|author=|date=2019-01-20|work=Rappler|accessdate=2018-01-23|archive-url=https://web.archive.org/web/20190123192913/https://www.rappler.com/nation/special-coverage/weather-alert/221467-tropical-depression-amang-pagasa-forecast-january-20-2019-11pm|archive-date=2019-01-23|dead-url=no}}{{en}}</ref>，不過日本氣象廳和聯合颱風警報中心的定位顯示，該系統僅在[[菲律賓|菲律賓]]以東海域掠過<ref name="jam"/><ref name="聯合"/>。由於系統所在海域海水炎熱、[[比熱容量|比熱容量]]高，同時[[垂直風切變|垂直風切變]]微弱，[[外流_(氣象學)|高空輻散]]良好，該系統稍為增強，有對流爆發，系統上空流出呈放射狀<ref>{{cite web|url=http://tgftp.nws.noaa.gov/data/raw/wt/wtpq30.rjtd..txt|title=RSMC熱帶氣旋預測推理第1報，TD位於12.8N 126.0E|author=日本氣象廳|date=2019年1月21日|deadurl=yes|archiveurl=https://www.webcitation.org/720afE1zw?url=http://gwydir.demon.co.uk/advisories/WTPQ30-RJTD_201808271800.htm|archivedate=2018-08-28|access-date=2019-01-25}}{{en}}</ref>。[[臺灣|臺灣]][[中央氣象局|中央氣象局]]在21日晚上8時將此系統升格為熱帶性低氣壓，給予編號02<ref>{{Cite news|url=https://tw.news.yahoo.com/%E8%8F%B2%E6%B5%B7%E5%9F%9F%E7%86%B1%E5%B8%B6%E4%BD%8E%E5%A3%93-%E5%BD%A2%E6%88%90%E8%9D%B4%E8%9D%B6%E9%A2%B1%E9%A2%A8%E6%A9%9F%E7%8E%87%E4%BD%8E-021544422.html|title=菲海域熱帶低壓 形成蝴蝶颱風機率低|author=中廣新聞網|date=2019-01-21|work=Yahoo奇摩新聞|accessdate=2019-01-25}}{{中文}}</ref>，日本氣象廳緊接在晚上9時05分發佈烈風警報<ref name="jam">{{Cite news|url=https://www.ettoday.net/news/20190122/1362362.htm|title=最凍9度今晚到！周六強冷空氣接力冷到下周2號颱風「蝴蝶」恐生成|author=|date=2019-01-22|work=ETtoday|accessdate=2019-01-23}}{{en}}</ref>，表示該系統在隨後1日內有機會增強為[[熱帶風暴|熱帶風暴]]。

此系統在菲律賓東方近海先轉向緩慢向北移動，之後受冷空氣影響，轉向南並逐漸減弱，日本氣象廳於1月22日下午取消烈風警報<ref>{{Cite web|url=http://www.webcitation.org/75d7s2rB6|title=2019年第二號熱帶性低氣壓烈風警告|accessdate=2019-01-23|author=|date=2019-01-22|publisher=日本氣象廳}}{{ja}}</ref>。

==防災措施及影響==
此系統1月19日在菲律賓{{Link-en|希納圖安|Hinatuan,_Surigao_del_Sur}}東南東方580公里，菲律賓大氣地球物理和天文管理局將其升格為熱帶性低氣壓的同時，發出[[熱帶氣旋警告|一號風暴信號]]<ref name="S1">{{Cite web|url=http://www.webcitation.org/75dMulE8V|title=2019熱帶氣旋Amang＃1|accessdate=2019-01-23|author=|date=2019-01-19|publisher=菲律賓共和國大氣地球物理和天文管理局}}{{en}}</ref>；到了1月22日此系統位於{{Link-en|卡塔曼_(北薩馬省)|Catarman,_Northern_Samar|卡塔曼}}東方85公里，被該局降格為低壓區，同時撤下一號風暴信號<ref>{{Cite news|url=https://www.philstar.com/nation/2019/01/23/1887306/tropical-depression-amang-weakens|title=熱帶低壓Amang減弱|author=|date=2018-01-22|work=Philippine Star|accessdate=2018-01-23}}{{en}}</ref>。

此系統在菲律賓造成暴雨導致山洪暴發，在[[北阿古桑省|北阿古桑省]]一個礦場有8名礦工正在開採，不幸遇到山體滑動，有1名礦工受傷<ref>{{cite news|url=http://cnnphilippines.com/regional/2019/01/24/Agusan-del-Norte-landslide-tropical-depression-Amang.html|title=4 dead, 3 missing in Agusan del Norte landslide|publisher=CNN Philippines|date=2019-01-24|accessdate=2019-01-24|language=en}}</ref>，而其餘7名礦工都被活埋，並確認已罹難<ref>{{Cite news|url=https://www.gmanetwork.com/news/news/regions/682645/7-miners-killed-in-landslide-in-agusan-del-norte/story/?amp|title=7 miners killed in landslide in Agusan del Norte|date=2019-01-24|publisher=GMA News|accessdate=2019-01-24|language=en}}</ref>；上個月受[[熱帶低氣壓WP352018|熱帶低氣壓35W]]重創的[[東米沙鄢|東米沙鄢]]，許多地方在1月21日停課<ref>{{Cite news|url=https://newsinfo.inquirer.net/1075446/classes-suspended-in-eastern-visayas-towns-cities-due-to-amang|title=由於'Amang'，東米沙鄢的城市停課|author=|date=2019-01-21|work=INQUIRER.net|accessdate=2019-01-23|archive-url=https://web.archive.org/web/20190123220613/https://newsinfo.inquirer.net/1075446/classes-suspended-in-eastern-visayas-towns-cities-due-to-amang|archive-date=2019-01-23|dead-url=no}}{{en}}</ref>，有些航班受到影響，此系統在[[比科爾區|比科爾]]附近的海域吹起強風，導致數個船班停航，導致有1,620人滯留在港口<ref>{{Cite news|url=http://www.pna.gov.ph/articles/1059550|title=由於'Amang'，有1,620名乘客滯留在比科爾港口|author=|date=2019-01-21|work=pna.gov.ph|accessdate=2019-01-25}}</ref>，而{{Link-en|國家災害應變管理委員會|National_Disaster_Risk_Reduction_and_Management_Council}}提撥24.467億預算作為災害準備<ref>{{Cite report|url=http://www.ndrrmc.gov.ph/attachments/article/3593/Sitrep_no_3_re_Flashflood_and_Landslide_Incident_in_Davao_Oriental_Province_as_of_26_January_2019_6AM.pdf|title=NDRRMC更新：Sitrep No.
