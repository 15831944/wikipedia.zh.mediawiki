'''静海区'''是[[中华人民共和国|中国]][[天津市|天津市]]西南部下辖的一个[[市辖区|市辖区]]。面积1476平方千米，人口51万，区人民政府驻静海镇。<ref>[http://www.tjjh.gov.cn/ 静海政务公众信息网]</ref>

== 历史 ==
金[[明昌|明昌]]四年（1193年）置靖海县，治所涡口寨。元[[至元|至元]]二年（1265年）并入会川县（即今青县）；嗣后，再置靖海县。明[[洪武|洪武]]初年，改称静海县，相沿至今。1973年由[[河北省|河北省]]划归天津市。2015年7月撤县设区。<ref>{{cite web |url=http://202.108.98.30/description?dcpid=2015|title=中华人民共和国二〇一五年县级以上行政区划变更情况}}</ref>

== 行政区划 ==
静海区现下辖16个镇、2个乡：
* 镇：静海镇、独流镇、台头镇、王口镇、梁头镇、子牙镇、沿庄镇、双塘镇、陈官屯镇、唐官屯镇、大丰堆镇、西翟庄镇、大邱庄镇、团泊镇、蔡公庄镇、中旺镇。
* 乡：良王庄乡、杨成庄乡。
