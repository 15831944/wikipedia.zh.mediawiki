'''独立电视公司'''（{{Lang-en|ITV plc}}）是一家总部位于[[英格兰|英格兰]][[伦敦|伦敦]]的[[英国|英国]][[大眾傳媒|媒体业]][[公众有限公司|公众有限公司]]。独立电视公司拥有13到15家区域性电视台，并以授权[[联播|联播]]的形式组建成面向全国播出的[[独立电视网|独立电视网]]。独立电视网是英国最资深和最大的[[商業廣播|商业]][[地面電視|地面]][[电视网|电视网]]。独立电视网拥有同品牌的电视频道——[[独立电视台|独立电视台]]，自1950年代开始就与[[BBC第一台|BBC第一台]]争夺全英国最受欢迎的电视台地位。

独立电视公司是由[[格拉納達股份|格拉纳达股份有限公司]]（[[ITV格拉納達|格拉纳达电视台]]所有者）对[[卡尔顿通信股份|卡尔顿通信股份有限公司]]进行[[购并|购并]]之后组建而成；格拉纳达股份有限公司拥有新公司68%的股份并获得[[控股|控股权]]，卡尔顿通信股份有限公司则拥有剩下的32%的股份。<ref>{{cite web |title=ITV shares on London stock market |url=http://news.bbc.co.uk/2/hi/business/3450397.stm |website=[[BBC在线|BBC在线]] |publisher=[[BBC新闻|BBC新闻]] |accessdate=2019-02-04}}</ref> 独立电视公司于2004年2月2日正式上市交易，这是由原两家公司合并成独立电视网之后其区域转播权的最新更新日期。2009年，独立电视公司从[[华特迪士尼公司|华特迪士尼公司]]手中并购了{{Link-en|早安电视台|GMTV}}；早安电视台持有独立电视网[[晨间新闻|晨间新闻]]节目{{Link-en|ITV晨间|ITV Breakfast}}25%的转播权。<ref>{{cite web |author=James Robinson |title=ITV takes full control of breakfast TV broadcaster GMTV |url=https://www.theguardian.com/media/2009/nov/26/itv-buys-remaining-gmtv-stake |publisher=[[卫报|卫报]] |accessdate=2019-02-04}}</ref> 独立电视公司在2011年从{{Link-en|逸腾敦集团|Yattendon Group}}处并购了[[ITV海峽電視|海峡电视台]]；<ref>{{cite web |title=Channel Television To Be Sold To ITV Plc |url=http://www.channelonline.tv/channel-television-sold-itv-plc/ |publisher=海峡电视台 |accessdate=2019-02-04}}</ref> 并在2015年花费1亿[[英镑|英镑]]购下[[UTV電視台|阿尔斯特电视台]]。<ref>{{cite web |author1=John Campbell |title=UTV Media agrees sale of TV stations to ITV for £100m |url=https://www.bbc.co.uk/news/uk-northern-ireland-34508529 |website=[[BBC在线|BBC在线]] |publisher=[[BBC新闻|BBC新闻]] |accessdate=2019-02-04}}</ref> 2016年2月29日，独立电视公司将上述频道的所有权转移给了独立电视网。

独立电视公司目前于[[伦敦证券交易所|伦敦证券交易所]]上柜交易，是[[富时100指数|富时100指数]]的成分股之一。

== 公司运营 ==
=== 公司架构 ===
独立电视公司划分为两大事业部门：<ref>{{cite web |title=What we do |url=https://www.itvplc.com/about/what-we-do |publisher=独立电视公司 |accessdate=2019-02-04}}</ref>

* 广播与网络部门：负责运营[[独立电视网|独立电视网]]及所属台站，并包括[[ITV新闻|ITV新闻]]（这是通过区域授权播出的形式进行全国播出）。

* {{Link-en|ITV工作室|ITV Studios}}：包括制作面向英国本土和国际市场的电视节目；管理独立电视公司转播权事务和通过其下属发行公司环球娱乐（{{lang|en|Global Entertainment}}）进行节目版权开发。

=== 所属台站 ===
通过ITV广播有限公司（{{lang|en|ITV Broadcasting Ltd.}}），独立电视公司持有独立电视网所拥有的15个联播许可证中的13个，这些覆盖了英国本土的绝大部分独立电视网播出区域；同时独立电视公司持有[[英格兰|英格兰]]与[[威尔士|威尔士]]的所有牌照，以及[[海峡群岛|海峡群岛]]和[[北爱尔兰|北爱尔兰]]的单一牌照。<ref>{{cite web |title=Review of ITV Networking Arrangements |url=https://www.ofcom.org.uk/__data/assets/pdf_file/0016/53800/itv-networking.pdf |publisher=[[通讯管理局|通讯管理局]] |accessdate=2019-02-04}}</ref>
