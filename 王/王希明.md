'''王希明'''（{{bd|1962年|||}}）[[天津市|天津市]][[宝坻区|宝坻区]]人，[[中国人民解放军少将|中国人民解放军少将]]<ref name=bfw/>。

==生平==
王希明生于天津市宝坻区一个普通农家。[[高考|高考]]填报志愿时，王希明经其身为军人的二爷劝说，又考虑到家庭经济条件不高，而军校不但津贴高而且包食宿及服装，便填报了[[中国人民解放军石家庄陆军指挥学院|中国人民解放军石家庄陆军指挥学院]]。后来，王希明先后在[[中国人民解放军天津警备区|中国人民解放军天津警备区]]和[[中国人民解放军北京军区|中国人民解放军北京军区]]任职。他全程参与了[[中国人民解放军驻香港部队|中国人民解放军驻香港部队]]的组建，并且自1999年10月起连续3年担任中国人民解放军驻香港部队保卫处处长。后任[[中国人民解放军国防大学|中国人民解放军国防大学]]政治部保卫部部长<ref name=bfw>{{Cite web |url =  http://news.enorth.com.cn/system/2007/07/30/001794914.shtml|title = 纪念建军80周年：原驻港部队保卫处处长王希明 |author =  |date =2007-07-30  |publisher =北方网  |language =  |accessdate =  |quote =  }}</ref>。2011年任[[中国人民解放军国防大学防务学院|中国人民解放军国防大学防务学院]]政治委员，跻身副军职<ref>{{Cite web |url =http://www.chinanews.com/gn/2011/09-15/3329925.shtml  |title =解放军副总长章沁生与孟加拉国陆军参谋长会谈  |author =  |date = 2011-09-15 |publisher =中新网 |language =  |accessdate =  |quote =  }}</ref>。后来任中国人民解放军国防大学政治部副主任、党委政法委副书记等职务。2017年，出任新调整组建的中国人民解放军国防大学政治工作部主任<ref name=wangzheng>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1760627 |title =王希明、郑云华分任国防大学政治工作部主任、教育训练部部长  |author =  |date = 2017-08-12 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
