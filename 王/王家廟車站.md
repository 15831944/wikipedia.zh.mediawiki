'''王家廟車站'''位於現今[[臺北市|台北市]][[北投區|北投區]]公館路東華街口北側，天母玫瑰社區大廈旁，曾為[[台灣鐵路管理局|台灣鐵路管理局]][[臺鐵淡水線|淡水線]]的[[鐵路車站|鐵路車站]]。鄰近[[台北捷運淡水線|台北捷運淡水線]][[唭哩岸站|唭哩岸站]]、[[奇岩站|奇岩站]]。因鄰近[[王爺廟|王爺廟]]，但因[[誤植|誤植]]而命名為王家廟車站。

== 車站構造 ==
*[[側式月台|側式月台]]一座

== 利用狀況 ==
*廢站前為[[招呼站|招呼站]]，以[[石牌站_(臺北市)|石牌車站]]為管理站。

== 歷史 ==
*1958年6月1日：為便利[[大同公司|大同北投廠]]員工通勤而設立。除上下班時間外，列車均直接通過本站。
*1988年7月16日：廢站。

== 車站週邊 ==
*大同北投廠
*王爺廟
