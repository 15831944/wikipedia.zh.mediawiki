'''王洛勇'''{{BD|1958年|12月24日|||catIdx= W王}}，出生于[[河南省|河南省]][[洛阳市|洛阳市]]，[[中国大陆|中国大陆]]知名[[演员|演员]]。<ref name=" " >{{cite web |url=http://men.sohu.com/20110516/n307455111.shtml |title=王洛勇：我的巅峰从下一刻开始 |accessdate= 2011年|work=|publisher=搜狐网 |language = zh-hans}}</ref>

== 简介 ==

王洛勇的父母因参与洛阳重点工程的建设来到洛阳，他因此出生在此地<ref>{{Cite web |url=http://news.lyd.com.cn/system/2011/06/26/001013709.shtml |title=专访焦裕禄扮演者王洛勇：焦裕禄“是属于世界的” |accessdate=2013-09-18 |author=记者：赵志伟  |date= 2011-06-26 |work= |publisher=洛阳网，来源：洛阳日报 |language=Zh-hans }}</ref>。艺术启蒙老师是他的[[舅舅|舅舅]]，他教会了王洛勇吹[[笛子|笛子]]，至今王洛勇仍记得舅舅教会他的第一首曲目为《[[抬头望见北斗星|抬头望见北斗星]]》。<ref name=" " />

1971年，王洛勇被迫参加“[[上山下乡运动|上山下乡运动]]”，在此期间，他考入了[[湖北|湖北]][[十堰|十堰]][[京剧团|京剧团]]。<ref name=" " /> 在剧团中因对《[[智取威虎山|智取威虎山]]》中[[圆号|圆号]]的声音颇为喜好，向剧团请求学吹圆号，于是他被安排到[[湖北歌舞团|湖北歌舞团]]学习。<ref name=" " /> 在学习期间，他又考上了[[武汉音乐学院|武汉音乐学院]]，学会了吹圆号、作曲和其他一些技艺。<ref name=" " /> 学成之后继续回到京剧团。他一边表演一边学习，并考上[[上海戏剧学院|上海戏剧学院]]<ref>{{Cite book|title=音乐剧之旅|last=周小川等著|first=|publisher=北京：新世界出版社|year=1998.11|isbn=7-80005-443-8|location=|pages=532}}</ref>。

1986年，王洛勇在[[上海戏剧学院|上海戏剧学院]]毕业后，他去了[[美国|美国]]留学。<ref name=" " /> 当时他就读的[[路易斯安那大学|路易斯安那大学]]戏剧学院给了他6000美元奖学金，不过因其[[英语|英语]]水平太差而被取消，为了生活费，他不得不打零工挣钱：刷盘子，送外卖，除草等<ref>{{Cite book|title=百位影星非常档案|last=徐林正著|first=|publisher=杭州：浙江文艺出版社|year=2006.04|isbn=7-5339-2299-9|location=|pages=195}}</ref>。

一段时间后，他有考上了[[美国|美国]][[波士顿大学|波士顿大学]]戏剧表演系，而且得到了学校的奖学金。<ref name=" " /> 1989年，王洛勇获得了[[波士顿大学|波士顿大学]]戏剧系的硕士学位，并去了[[威斯康辛州|威斯康辛州]]的[[米尔沃基大学|米尔沃基大学]]担任表演教师。后来，他因主演《[[西贡小姐|西贡小姐]]》，6年连演2478场，而一举走红全[[美国|美国]]。<ref name=" " /><ref name="mxk " >{{cite web |url=http://people.mtime.com/918716/details.html |title=王洛勇 |accessdate= 2000年|work=|publisher=时光网 |language = zh-hans}}</ref>

2001年，王洛勇回到[[中国|中国]]，扩展事业。<ref name="mxk " />

2017年12月27日，王洛勇在微博上发布了其朗诵英语版《[[出师表|出师表]]》的视频<ref name="出师表" >{{cite web |url=https://weibo.com/1738734804/FBAfAlZ59?ref=collection&type=comment |title=王洛勇的微博动态 |accessdate= 2017年12月27日|work=|publisher=微博 |language = zh-hans}}</ref>，引起中国网民热议和称赞<ref name="新浪娱乐" >{{cite web |url=http://ent.sina.com.cn/2017-12-28/doc-ifypyuve0295537.shtml |title=百老汇中国第一人王洛勇 英文版《出师表》超厉害 |accessdate= 2017年12月28日|work=|publisher=新浪娱乐 |language = zh-hans}}</ref><ref name="中国青年报" >{{cite web |url=http://epaper.ynet.com/html/2017-12/29/content_274423.htm |title=“英语8级”的诸葛亮 折服众网友 |accessdate= 2017年12月29日|work=|publisher=[[中国青年报|中国青年报]] |language = zh-hans}}</ref>。
