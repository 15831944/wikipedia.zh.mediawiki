'''王健'''（{{bd|1961年|12月15日|2018年|7月3日}}），[[天津|天津]]人，[[海航集团|海航集团]]有限公司联合创始人、海航集团有限公司董事局[[董事长|董事长]]。2018年7月3日在法国离世。

== 生平 ==
1983年，王健毕业于[[中国民用航空学院|中国民用航空学院]]（现[[中国民航大学|中国民航大学]]）经营管理专业，获经济管理系航空经营管理专业学士学位。

1983-1990年，在[[中国民用航空总局|中国民用航空总局]]计划司工作。1990年，参与创建海南省航空公司<ref>{{Cite book|title=顶级品牌|last=欧阳斌著|first=|publisher=北京：中国经济出版社|year=2014.10|isbn=978-7-5136-3441-0|location=|pages=136-137}}</ref>。2001年1月1日，在王健、[[陈峰|陈峰]]等人领导下，海航集团组建完成，并在一年之内完成[[并购|并购]][[海口美兰国际机场|海口美兰国际机场]]、[[长安航空|长安航空]]、[[新华航空|新华航空]]和[[山西航空|山西航空]]，并先后出任[[海航集团|海航集团]]有限公司[[CEO|CEO]]、董事局副董事长、董事长<ref>{{Cite book|title=影响中国企业未来的十大管理实践|last=侯贵松，王璐璐编|first=|publisher=北京：中国纺织出版社|year=2004.05|isbn=7-5064-2911-X|location=|pages=185-186}}</ref><ref>{{cite web |author1=新浪财经 |title=海航集团有限公司联合创始人、董事长王健意外离世 |url=http://tech.sina.com.cn/i/2018-07-04/doc-ihevauxk3257312.shtml |website=新浪科技 |accessdate=2018-07-04 |date=2018-07-04}}</ref>。

2018年7月3日，王健在[[法国|法国]]公务考察期间，于[[博尼约|博尼约]]游玩时意外跌落导致重伤，经抢救无效离世，享年57岁<ref>{{Cite web|url=http://money.163.com/18/0704/18/DLT076R000259B6N.html|title=王健去世 海航所有股东承诺在离世时捐赠所有股份|accessdate=2018-07-04|last=网易|work=money.163.com}}</ref><ref>{{Cite news|url=https://www.nytimes.com/2018/07/04/business/hna-chairman-wang-jian-death.html|title=Wang Jian, Co-Founder of Chinese Giant HNA, Dies in France|accessdate=2018-07-05|language=en}}</ref>。法國警方表示，王健和家人照相時站在牆的邊緣，因失足急速從15米高牆上跌下<ref>{{cite web |title=海航董事長王健跌死　法國警方指無可疑 |url=https://thestandnews.com/finance/%E6%B5%B7%E8%88%AA%E8%91%A3%E4%BA%8B%E9%95%B7%E7%8E%8B%E5%81%A5%E8%B7%8C%E6%AD%BB-%E6%B3%95%E5%9C%8B%E8%AD%A6%E6%96%B9%E6%8C%87%E7%84%A1%E5%8F%AF%E7%96%91/ |website=立場新聞 |accessdate=2018-07-24 |date=2018-07-05}}</ref>。2018年7月4日，海航集团王健董事长治丧委员会发布讣告<ref>{{Cite web|url=http://news.sina.com.cn/o/2018-07-04/doc-ihevauxk3989322.shtml|title=海航集团联合创始人、董事长王健在法摔伤去世|accessdate=2018-07-04|work=news.sina.com.cn}}</ref>。

而法國《[[世界报_(法国)|世界報]]》華裔記者張竹林在为[[香港蘋果日報|香港蘋果日報]]撰写的独家報導中称，事發地附近奔牛村教堂工作的奧狄伯特（Hubert Audibert）神父对王健的身故原因表示质疑，他对記者張竹林说“您怎麼能想像到一位57歲的男人這樣跳來跳去？這種完全是瞎編的”<ref>{{cite web |title=壹週刊 【有片】海航董事長跌死　法記者：疑點重重 |url=https://tw.appledaily.com/new/realtime/20180719/1394218/ |website=蘋果日報 (台灣) |accessdate=2018-07-24 |date=2018-07-19}}</ref><ref>{{cite web |title=海航王健跌死疑點重重 法國神父不信官方說法 |url=https://udn.com/news/story/7332/3258781 |website=聯合新聞網 |accessdate=2018-07-24 |date=2018-07-18}}</ref>。该报道还对其他的事件细节有关报道进行了调查，但没有得出结论。

2019年1月末，法国《[[解放报|解放报]]》发布长达3页的[[调查报导|调查报导]]，认为王健是[[自杀|自杀]]身亡。其后，参与调查的[[法国宪兵|法国宪兵]]与[[检察院|检察院]]方面发表声明，重申王健为意外身亡。<ref>{{Cite web|title=海航前董事长王健死于意外还是谋杀？|url=http://cn.rfi.fr/%E4%B8%AD%E5%9B%BD/20190131-%E6%B5%B7%E8%88%AA%E5%89%8D%E8%91%A3%E4%BA%8B%E9%95%BF%E7%8E%8B%E5%81%A5%E6%AD%BB%E4%BA%8E%E6%84%8F%E5%A4%96%E8%BF%98%E6%98%AF%E8%B0%8B%E6%9D%80|work=法广|date=2019-01-31|accessdate=2019-03-31|language=zh-Hans}}</ref>
