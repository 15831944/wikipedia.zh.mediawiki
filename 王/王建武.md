'''王建武'''（{{bd|1958年|8月||}}），[[河南|河南]][[洛宁|洛宁]]人。[[中国人民解放军中将|中国人民解放军中将]]。中国共产党第十九届中央委员。

长期在原[[中国人民解放军济南军区|济南军区]]服役，曾任[[中国人民解放军陆军第八十三集团军|第54集团军]]政治部副主任、主任，2010年7月晋升[[中国人民解放军少将|少将]]军衔，至迟在2013年1月接替张建设，出任原[[中国人民解放军济南军区|济南军区]]联勤部政委，成为正军级将领；2016年8月升任[[中国人民解放军西藏军区|西藏军区]]政委，跻身副大军区级<ref>{{cite news|url=http://china.caixin.com/2016-08-29/100982731.html|title=王建武少将任西藏军区政委 跻身副大军区级|author=|date=2016-08-29|work=|newspaper=财新网|accessdate=2017-09-17}}</ref>。2017年7月31日，晋升为[[中国人民解放军中将|中将]]军衔。2017年12月，出任[[中央军委政治工作部|中央军委政治工作部]]副主任<ref>{{cite web|title=西藏军区原政委王建武中将已调任中央军委政治工作部副主任|url=https://www.thepaper.cn/newsDetail_forward_2142922|website=澎湃新闻|accessdate=2018-05-25}}</ref>。随后兼任[[中国人民解放军选举委员会|中国人民解放军选举委员会]]委员、[[国务院扶贫开发领导小组|国务院扶贫开发领导小组]]副组长等职务<ref>{{cite web |title=全国人民代表大会常务委员会批准任命的名单（2月24日） |url=http://www.gov.cn/xinwen/2018-02/24/content_5268524.htm |website=新华网 |accessdate=2019-01-04 |date=2018-02-24}}</ref><ref>{{cite web |title=国务院办公厅关于调整国务院扶贫开发领导小组组成人员的通知 |url=http://www.gov.cn/zhengce/content/2018-05/21/content_5292430.htm |website=中国政府网 |accessdate=2019-01-04 |date=2018-05-21}}</ref>。2018年12月，升任[[中国人民解放军南部战区|南部战区]]政治委员，晋升正战区级<ref>{{cite web |author1=林韵诗 |title=中央军委政治工作部副主任王建武中将出任南部战区政委 |url=http://china.caixin.com/2019-01-05/101366964.html |website=财新网 |accessdate=2019-01-06 |date=2019-01-05}}</ref>。
