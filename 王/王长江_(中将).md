'''王长江'''（{{bd|1959年|2月||}}），河北滦县人，[[中国人民解放军海军中将|中国人民解放军海军中将]]，[[第十三届全国人民代表大会代表|第十三届全国人民代表大会代表]]<ref>{{cite news|title=全国人民代表大会|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=200813272547|accessdate=2018-05-29}}</ref>。

==生平==
王长江毕业于[[中国人民解放军空军第四航空学校|中国人民解放军空军第四航空学校]]，长期从事飞行训练工作，是[[海空雄鹰团|海空雄鹰团]]第22任团长。历任[[中国人民解放军海军装备部|中国人民解放军海军装备部]]副部长、[[中国人民解放军东部战区海军航空兵|中国人民解放军海军东海舰队航空兵]]副司令员、[[中国人民解放军海军|中国人民解放军海军]]副参谋长等职务。至迟在2012年初已任[[中国人民解放军海军南海舰队|中国人民解放军海军南海舰队]]副司令员兼[[中国人民解放军南部战区海军航空兵|中国人民解放军海军南海舰队航空兵]]司令员。2016年，王长江海军少将任[[中国人民解放军北部战区|中国人民解放军北部战区]]副司令员<ref>{{Cite web |url =http://china.caixin.com/2016-09-19/100989320.html  |title = 王长江少将任北部战区副司令员 |author =  |date =  2016-09-19|publisher = 财新网 |language =  |accessdate =  |quote =  }}</ref>。

2008年晋升海军少将军衔。2017年晋升海军中将军衔<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1749352 |title =北部战区副司令员王长江已晋升中将军衔  |author =  |date = 2017-08-01 |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。
