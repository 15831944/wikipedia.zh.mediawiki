'''王枫'''（{{bd|1931年|||}}），[[辽宁省|辽宁省]][[营口市|营口市]]人。[[中华人民共和国广播电影电视部|中华人民共和国广播电影电视部]]副部长、[[中国中央电视台|中国中央电视台]]台长。中共党员，中共十三大代表。

==生平==
1946年，王枫参加[[中国共产党|中国共产党]]领导的革命。1953年，王枫和9位青年技术人员被中国选派到[[捷克斯洛伐克|捷克斯洛伐克]]，学习无线电大功率发射机制造技术两年，成为中国首批接触电视技术者。回国后，王枫一直在广播系统担任技术员或技术官员、研究员。1972年初，经国务院总理[[周恩来|周恩来]]、副总理[[李先念|李先念]]批准，中央广播事业局军管小组技术业务办公室副组长王枫率团到[[欧洲|欧洲]]五国（[[法国|法国]]、[[联邦德国|联邦德国]]、[[荷兰|荷兰]]、[[瑞士|瑞士]]、[[英国|英国]]）考察彩色电视。考察归国后，1973年5月王枫调入北京电视台（今中央电视台）主管技术工作。1976年担任北京电视台副台长。1978年5月1日，“北京电视台”更名为“中央电视台”，英文呼号CCTV。1982年11月16日，王枫被任命为中央电视台台长。中央电视台台长自王枫开始，定级为司局级。1983年任彩电中心工程总工艺师。1987年7月16日任中华人民共和国广播电影电视部副部长兼中央电视台台长，1988年辞去中央电视台台长职务<ref name=liren>{{Cite web |url =http://www.guancha.cn/wenzhengming/2015_04_11_315504_s.shtml  |title =  从开创春晚到反腐风暴 盘点历任央视台长背后的故事|author =  |date =  2015-04-11 |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref><ref name=deng>邓艳玲，那些掌管CCTV的“一哥”们，滇池天下2011年第23期</ref><ref name=licheng>{{Cite web |url = http://www.dvbcn.com/2009/09/09-38149.html |title = 1936年-2009年中国广播电视发展历程 |author =  |date = 2017-03-06 |publisher = 众视 |language =  |accessdate =  |quote =  }}</ref>。

王枫是中国彩色电视的开创者。1983年，王枫开创了[[中国中央电视台春节联欢晚会|中国中央电视台春节联欢晚会]]。另外，王枫延续了[[戴临风|戴临风]]将中国古典文学名著改编为电视剧的想法，担任大型电视连续剧《[[西游记_(1982年电视剧)|西游记]]》的总监制，《[[三国演义_(电视剧)|三国演义]]》领导小组组长。王枫还参与组织策划了大型电视连续剧《[[唐明皇_(电视剧)|唐明皇]]》及特别节目《人与人》、《[[话说长江|话说长江]]》等。[[第十一届亚运会|第十一届亚运会]]期间，王枫任亚运会组委会广播电视委员会主任，圆满完成本届亚运会的广播电视转播任务，被[[国际奥委会|国际奥委会]]授予“奥林匹克贡献奖”<ref name=liren/>。

== 家庭 ==
妻子：林如，[[中央人民广播电台|中央人民广播电台]]第一代播音员。

女儿：[[王雪纯|王雪纯]]，[[中央电视台|中央电视台]]著名主持人。
