'''王淵'''，媒體形容為「雨衣大盜」，共犯下[[台北市|台北市]]三起運鈔車搶案，造成兩名保全受傷，一名保全死亡，共得手4086萬元。<ref>[http://tw.news.yahoo.com/%E7%8A%AF3%E8%B5%B7%E6%90%B6%E6%A1%88-%E9%9B%A8%E8%A1%A3%E5%A4%A7%E7%9B%9C%E9%81%AD%E6%B1%82%E9%87%8D%E5%88%91-065708358.html 犯3起搶案 雨衣大盜遭求重刑]</ref>。2013年12月最高法院依強盜殺人等罪判他無期徒刑，褫奪公權終身，併科罰金110萬元確定。

本案件於2012年被改編成[[廉政英雄|廉政英雄]]戲劇系列劇情。

== 涉及案件 ==
一開始警方認為「雨衣大盜」在台北犯下以下七件運鈔車搶案

# 2002年10月21號：中信證券南京分公司，涉案金額：206萬9376元
# 2004年8月11號：台新銀行復興分行，涉案金額：476萬5966元
# 2005年12月8號：台灣企銀南京東路分行，涉案金額：1157萬元
# 2007年4月20號：合作金庫中山分行，涉案金額：1679萬元，2名保全中彈，1死1重傷
# 2008年4月21號：台北萬華青年郵局，涉案金額：430萬元
# 2009年3月6號：國泰世華銀行278萬4177元，涉案金額：278萬4177元
# 2012年3月26號：台企銀南京東路分行，涉案金額：1250萬元，1名保全員雙腿中彈 

王淵本人坦承和警方的DNA和彈道證據顯示他犯下其中三起搶案，2005和2012年台企銀搶案、2007年合作金庫搶案。造成兩名保全受傷，一名保全死亡，總共得手4086萬元。<ref>{{cite web |title=犯3案 機場落網 雨衣大盜是媒體人 曾任職中天電視 |url=https://tw.appledaily.com/headline/daily/20120525/34253738/ |publisher=蘋果日報}}</ref>
其他四案，目前尚未破案。
