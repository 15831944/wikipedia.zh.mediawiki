'''王攀元'''（{{lang-en|Wang Pan-yuan}}<ref name=ncafroc_award>{{cite web | title = 國家文化藝術基金會 國家文藝獎 / 第五屆 | url = http://www.ncafroc.org.tw/award-artist.aspx?id=1211 | publisher = 國家文化藝術基金會 | language = zh-tw | accessdate = 2018-07-02}}</ref>；1909年<ref>[http://news.ltn.com.tw/news/life/paper/1162485 國寶畫家王攀元辭世 嵩壽109歲]</ref><ref>[https://news.housefun.com.tw/news/article/162015185170.html 110歲國寶畫家王攀元辭世　黃春明送別老師]</ref>12月16日－2017年12月22日），臺灣畫家。生於江蘇省北部的徐家洪望族，排行老二，因父母早逝，求學過程艱辛，畢業於[[上海美術專科學校|上海美術專科學校]]，戰後攜眷來台，輾轉定居宜蘭，終老於此。

==生平==
三歲時父親英年早逝，七歲上[[私塾|私塾]]受教，母親告誡要勤奮向學，對他影響深遠。母親在十三歲時不幸辭世、關係較好的親戚一個個遠嫁，深刻感受人情冷暖，在紛亂的大家族裡逐漸養成隱忍的個性。小學畢業考上[[江蘇|江蘇]]省立淮安中學，家族卻對他求學之事未積極資助，休學一年後經同情親友的協助，終於正式入學。某次開學逾一周，家裡始終毫無動靜，最後由燒飯的傭人朱媽發動佃農協助才暫時緩解問題。

1933年（民國二十二年）考上[[復旦大學|復旦大學]]法律系，因志趣不合又重考進上海美專西畫科。王攀元就讀上海美專的時候，曾任北大校長的蔡元培是校董，時任校長的劉海粟校風頗為開放，所以王攀元在美專就學期間，就已將國畫的意境融入西畫裡。

他美專時期的老師，對王攀元有深遠影響的，首先是留法的張弦為他奠定了紮實的基礎，除此之外，陳人浩也教他水彩，同時，他還隨諸聞韻和[[潘天壽|潘天壽]]學國畫。週末也繼續到研究所聽取[[潘玉良|潘玉良]]和[[王濟遠|王濟遠]]的美學課程。
事實上，在他的學院初期，「中體西學」的啟蒙甚早，諸如[[劉海粟|劉海粟]]、[[潘天壽|潘天壽]]、[[潘玉良|潘玉良]]等人，都是西風東進的一時之選。

1949年來台，人地生疏，生活艱困，幾經波折，終於1952年應[[羅東|羅東]]中學之聘，擔任美術老師，從此定居[[宜蘭|宜蘭]]。

長期駐居蘭陽平原期間，大時代的動盪與環境交迭，促使他經年累月將獨立不凡且熱愛生命的性情投注於繪畫。也因為自我內在修持的慣性，導致他長期與孤獨對話，特別是他的作品，因為蘭陽、因為太陽、因為太平洋，交互輝映出一種「永不日落」的紅透半邊天的藝象，一本可閱讀的作品，一如他與大自然交換詩情的篇章，在在都成就了他個人特有的「畫布的詩人」風格。
 
值得一提的是，1992年3月，蘇富比公司首次在臺北市舉行拍賣會，當時王攀元的油畫「風景」以一百萬元；水彩「拜石圖」也以六十萬元成交，成為次日報紙藝文版面的頭條，且被譽為是「八大明星畫家之一」。同在1992這一年，臺北誠品畫廊個展時出版畫冊。

1993年[[臺灣|臺灣]]省立美術館個展、1994年臺北形而上畫廊「孤獨原來不寂寞」個展並出版《老天爺的劇本》、1996年臺北形而上畫廊「不只是孤獨」個展並二版《老天爺的劇本》、1997年國際藝術博覽會特展（形而上畫廊）、2001年再度獲邀於[[歷史博物館|歷史博物館]]舉辦「攀圓追日：王攀元九十自選展」並再出版《攀圓追日：王攀元自選集》；2001臺北形而上畫廊舉辦「九個太陽：王攀元精選展」並改版《攀圓追日：老天爺的劇本》。
 
2003年，王攀元更榮獲國家文藝獎的肯定。

2017年12月22日百餘歲高齡逝世，生前預定於2018年5月在國立歷史博物館舉行的畫展，也已經順利展出完美落幕。<ref>{{cite news|title=享壽106歲！國內最年長國寶級畫家王攀元走了 |url=https://udn.com/news/story/7328/2889137?from=udn-catelistnews_ch2 |language=zh-tw |author=戴永華 |publisher=聯合報 |date=2017-12-22|accessdate=2017-12-23 }}</ref>

以孤寂寫心，以境言志，是傳統文人文化，王攀元將個人坎坷經歷，以象徵手法入畫，作品用色大塊，結構簡約，畫中人物極度壓縮。<ref>{{cite news | title = 這顆紅日何日再現？ 史博館最後一瞥王攀元 | url = http://news.ltn.com.tw/news/culture/paper/1212995 | publisher = 自由時報 | author = 藍祖蔚 | language = zh-tw | date = 2018-07-01}}</ref>

== 展出經歷 ==
* 1959年，在宜蘭羅東農會開第一次畫展，賣出所有展出的三十幅水墨及水彩。<ref name = ncafroc_bio>{{cite web | title = 藝術家介紹 王攀元 / Wang Pan-yuan 記事 | url = http://www.ncafroc.org.tw/award-artist.aspx?id=1211 | 
 publisher = 國家文化藝術基金會 | language = zh-tw | accessdate = 2018-07-02}}</ref>
* 1966年(民國五十五年)，在台北的畫友李德協助下，在亞洲國際畫廊舉辦首次個人畫展。<ref name = ncafroc_bio />
* 1979年(民國六十八年)，在老友李德的勸說下，再次在台北春之藝廊展出畫作。
* 1983年，獲得中華民國畫學會金爵獎，受邀在台北市立美術館名人畫廊舉辦水彩畫展。<ref name = ncafroc_bio />
* 1985年，參加[[韓國|韓國]]第一屆亞洲國際美展，之後持續參加十六屆。<ref name = ncafroc_bio />
* 1986年(民國七十五年)，首次水墨畫展於雄獅畫廊。
* 1987年(民國七十六年)，在台北皇冠藝文中心舉辦個展，應邀在[[國立歷史博物館|國立歷史博物館]]國家畫廊舉行個展。<ref name = ncafroc_bio />
* 1988年，應邀參加全國美展、亞洲國際美展、國家畫廊當代藝術展聯展。<ref name = ncafroc_bio />
* 1989年，參加全國美展，以及國立歷史博物館國家畫廊主辦之「當代藝術發展展」。<ref name = ncafroc_bio />
* 1990年，台北「亞洲藝術中心」個展。<ref name = ncafroc_bio />
* 1991年，台灣省立美術館二度典藏作品。<ref name = ncafroc_bio />
* 1992年，台北「誠品畫廊」個展、台中省立美術館個展。<ref name = ncafroc_bio />
* 1992年3月(民國八十一年三月)，[[蘇富比|蘇富比]]公司首次在[[台北市|台北市]]舉行繪畫拍賣會，油畫「風景」以一百萬元、水彩「拜石圖」六十萬元成交，次日報上譽為「八大明星畫家之一」。
* 1994年，台北「形而上畫廊」舉辦「孤獨原來不寂寞」個展。<ref name = ncafroc_bio />
* 1995年，在台北「形而上畫廊」舉辦「不只是孤獨」個展。高雄市立美術館典藏油畫與水彩作品。<ref name = ncafroc_bio />
* 1997年，台北國際藝術博覽會特展。<ref name = ncafroc_bio />
* 2001年5月-6月於[[國立歷史博物館|國立歷史博物館]]舉行王攀元老師九十自選展。<ref>{{cite news | title = 可憐身是畫中人 ──畫壇隱士王攀元 | url = https://www.taiwan-panorama.com/Articles/Details?Guid=90a675d0-65a2-4a4a-bcdc-3e8bab9c382a&CatId=2 | publisher = 台灣光華雜誌 | author = 蔡文婷 | language = zh-tw,ja,en | date = 2001.06.01}}</ref>
* 2018年5月4日[[國立歷史博物館|國立歷史博物館]]主辦「過盡千帆——王攀元繪畫藝術」展出王攀元老師近150畫作。作品來源包括國立歷史博物館館藏、[[台北市立美術館|台北市立美術館]]館藏，私人收藏家及其家屬收藏；畫作媒材分為油彩、水彩與水墨三大類；依主題則分為山、水、船、孤獨人影、人群、包括黃山、北大荒與東北雪景、[[中國|中國]]山水回憶與[[蘭陽|蘭陽]]雨、[[太平山|太平山]]、[[蘇澳港|蘇澳港]]與[[龜山島|龜山島]]等[[台灣|台灣]]景物之美；首度亮相的70幅水墨畫作包括中國傳統[[宋朝|宋朝]]、[[元朝|元朝]]山水畫與若[[八大山人|八大山人]]筆下的孤獨鳥；開幕展上有國立歷史博物館館長廖新田、[[中華民國文化部|中華民國文化部]][[鄭麗君|鄭麗君]]、資深藝術家謝里法、資深藝術家廖修平與編舞家姚淑芬。<ref>{{cite web | title = 千呼萬喚始出來．王攀元 過盡千帆的繪畫藝術 | url = https://artouch.com/artouch2/content.aspx?aid=2018050422713&catid=03 | publisher = 典藏藝術網 | author = 陳意華 | language = zh-tw | date = 2018-05-04}}</ref><ref>{{cite news | title = 史博館閉館前最後大展 過盡千帆王攀元 | url = http://www.cna.com.tw/news/acul/201805040323-1.aspx | publisher = 中央通訊社 | author = 汪宜儒 | language = zh-tw | date = 2018-05-04}}</ref><ref>{{cite web | title = 過盡千帆——王攀元繪畫藝術 開幕典禮暨記者會 | url = https://www.moc.gov.tw/information_250_83644.html | publisher = 文化部 | language = zh-tw | date = 2018-05-04 | access-date = 2018-07-01 | archive-url = https://web.archive.org/web/20180701194325/https://www.moc.gov.tw/information_250_83644.html | archive-date = 2018-07-01 | dead-url = yes }}</ref><ref>{{cite news | title = 王攀元回顧展 日久他鄉是故鄉 | url = http://www.chinatimes.com/newspapers/20180510001566-260115 | publisher = 中時電子報 | author = 李怡芸 | language = zh-tw | date = 2018-05-10}}</ref><ref name = udn_award_info>{{cite news | title = 孤獨傳奇 錯過難再 王攀元最後畫展明截止 | url = https://udn.com/news/story/7270/3226784 | publisher = 聯合報 | author = 何定照 | language = zh-tw | date = 2018-06-30}}</ref>
