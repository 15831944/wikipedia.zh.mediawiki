'''王国强'''（{{bd|1960年|3月|catIdx=W}}），前[[中国共产党|中国共产党]][[辽宁省|辽宁省]][[凤城市|凤城市]][[市委书记|市委书记]]，2012年4月24日携妻子[[丹东海关|丹东海关]]主任科员谭某从[[沈阳桃仙国际机场|沈阳桃仙国际机场]]私自[[出境|出境]]前往[[美国|美国]]。王国强存在收受贿赂、违规持有[[中华人民共和国护照|因私护照]]、非法擅自出境、违规入股投资从事营利性活动等违纪违法行为和生活作风问题。<ref name="辽宁凤城市委书记逃美过程披露">{{cite web | url=http://news.sina.com.cn/c/2012-08-29/075525057877.shtml | title=辽宁凤城市委书记逃美过程披露 | publisher=新浪网 | accessdate=2012-08-29 | language=Zh-hans}}</ref>

== 简历 ==
王国强1983年8月参加工作，1984年12月加入[[中国共产党|中国共产党]]，[[在职研究生|在职研究生]]学历，学士学位。2008年起任辽宁省[[丹东市|丹东市]]下辖的凤城市市委书记。2012年3月，王国强在“海德供热事件”调查后，被调往[[丹东市|丹东市]]一项目建设指挥部任职。

== 海德供热事件 ==
2004年转制为“丹东凤城海德热电有限公司”的凤城市热电厂，负责对王国强任市委书记的[[凤城市|凤城市]]实施[[集中供热|冬季集中供热]]，供热面积达254万平方米，占凤城市城区的80%；供暖用户达1.9万户。在2011年至2012年冬季，海德供热频繁出现设备故障的情况，造成凤城市大面积供暖中断，引发居民不满。特别是2012年春节期间，仍出现停止供热的情况。在供热用户投诉后，海德热电未予回应。随后，中共辽宁省纪委监察厅对海德热电进行调查，王国强被调离中共凤城市委书记一职。<ref>{{cite web | url=http://news.hexun.com/2012-08-29/145242038.html | title=辽宁凤城原市委书记擅自离境被双开 疑卷款2亿 | publisher=和讯网 | accessdate=2012-08-29 | language=Zh-hans}}</ref>

== 私自离境 ==
2010年7月，王国强以赴美国参加女儿[[畢業|毕业典礼]]的理由提出申请并办理了因私护照，但最终未能成行。2012年3月，王国强在[[美国驻沈阳总领事馆|美国驻沈阳总领事馆]]办理了[[签证|赴美签证]]。<ref name="辽宁凤城市委书记逃美过程披露" />
2012年4月24日，王国强与在丹东海关任主任科员的妻子谭某从[[沈阳桃仙国际机场|沈阳桃仙国际机场]]离境前往美国。4月28日，中共辽宁省[[纪委|纪委]]对王国强立案调查；同月，王国强被免去凤城市市委书记职务。5月2日，[[丹东电视台|丹东电视台]]报道了王国强私自离境一事。<ref>{{cite web | url=http://dianzibao.cb.com.cn/html/2012-09/03/content_20449.htm | title=辽宁凤城弊案：王国强“事发”海德热电并购 | author=李慧敏 | publisher=[[中国经营报|中国经营报]] | date=2012-09-03 | accessdate=2019-04-02 | language=Zh-hans }}</ref><ref name="辽宁凤城市委书记逃美过程披露" />5月28日，王国强所涉案件移交检察机关。8月15日，经[[中共辽宁省委|中共辽宁省委]]批准，辽宁省纪委决定开除王国强的[[中国共产党|中共党籍]]和政府公职。<ref>{{cite web | url=http://www.jmnews.com.cn/c/2012/08/28/20/c_1268689.shtml | title=辽宁凤城原市委书记王国强擅自出境被双开 | publisher=东北新闻网 | date=2012-08-28 | accessdate=2012-08-28 | language=Zh-hans }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web | url=http://news.hexun.com/2012-08-29/145243297.html?from=rss | title=凤城原市委书记王国强被双开 | publisher=和讯网 | accessdate=2012-08-29 | language=Zh-hans}}</ref>

2012年8月，有传言称作为[[裸官|裸官]]的王国强在4月至5月间携带[[人民币|人民币]]2亿元巨款私自离境，引起舆论关注。<ref name="人民网">{{cite web | url=http://politics.people.com.cn/n/2012/0827/c1001-18846016.html | title=辽宁凤城市委书记“缺席”官网介绍 当地工作人员称其出事已久 | accessdate=2012-08-28|language=Zh-hans}}</ref><ref>{{cite web | url=http://www.takungpao.com/news/content/2012-08/27/content_995704.htm | title=传辽宁凤城市委书记王国强卷2亿逃美 | publisher=大公网 | work=转自香港商报 | accessdate=2012-08-28 | language=Zh-hans | deadurl=yes | archiveurl=https://web.archive.org/web/20120830035558/http://www.takungpao.com/news/content/2012-08/27/content_995704.htm | archivedate=2012-08-30 }}</ref>中共丹东市委宣传部表示，王国强系辽宁省省管干部，丹东市无权管辖。<ref>{{cite web | url=http://opinion.people.com.cn/n/2012/0829/c159301-18858516.html# | title=市委书记王国强的“擅自离境”太蹊跷 | accessdate=2012-08-29 | language=Zh-hans | deadurl=yes | archiveurl=https://web.archive.org/web/20120831005131/http://opinion.people.com.cn/n/2012/0829/c159301-18858516.html | archivedate=2012-08-31 }}</ref>

王国强逃亡到美国后，为了防止被中国相关部门发现和被美国相关部门抓获，致使夫妻不敢用护照，有病不敢就医，与国内亲人不敢联络，与美国的同学和朋友不敢联系。在乘坐交通工具方面，他也只能乘坐可以不使用护照的[[灰狗巴士|灰狗巴士]]。而他所居住的合租屋条件相对差劣<ref>{{cite web | url=http://www.xinhuanet.com/legal/2016-06/14/c_129058167.htm | title=外逃贪官生活素描：以为是天堂 原来是地狱| publisher= 新华网  | accessdate=2016-06-14| language=Zh-hans}}</ref>。

==自首与审判==
2014年12月22日，王国强回国自首。<ref name="news_辽宁凤城">{{Cite web| url = http://news.xinhuanet.com/2014-12/22/c_1113736984.htm| title = 辽宁凤城市委原书记王国强从美国回国投案自首| work = 新华网 | language=Zh-hans}}</ref>

2017年1月23日，[[辽宁省沈阳市中级人民法院|辽宁省沈阳市中级人民法院]]一审以受贿罪、巨额财产来源不明罪判处王国强有期徒刑8年，并处罚金人民币90万元；扣押在案的受贿所得赃款以及巨额财产来源不明的非法所得予以没收，上缴国库<ref>{{cite web | url=http://china.cnr.cn/gdgg/20170123/t20170123_523522116.shtml | title=辽宁凤城原市委书记王国强外逃后回国自首 一审获刑8年| publisher=央广网 | accessdate=2017-01-23| language=Zh-hans}}</ref>。
