'''王夢麟'''{{BD|1954年|4月15日}}，1980年代初[[台灣|台灣]][[校園民歌|校園民歌]][[歌手|歌手]]。原為[[計程車|計程車]]司機，於1978年通過[[新格唱片|新格唱片]]發表第一首詞曲創作〈雨中即景〉一舉成名，與民歌手[[黃大城|黃大城]]、[[趙樹海|趙樹海]]三人時常搭檔演唱。1997年，王夢麟、趙樹海與黃大城合組[[MIB三重唱|MIB三重唱]]，MIB三重唱總共發行兩張唱片專輯《Music Is Back》（1997年發行）、《母親我愛您》（1999年發行）。

王夢麟原唱的著名歌曲有〈雨中即景〉、〈木棉道〉、〈阿美阿美〉、〈小草〉、〈廟會〉等。

== 早年 ==
王夢麟從小在眷村（台北市大安區成功國宅前身）長大，因家境清寒，經常在公園看大哥哥們彈吉他並邊看邊學，沒錢買吉他，只好開口向鄰居借了一把吉他，慢慢把吉他學會。王夢麟初中沒畢業，又因為小時候混過[[太保|太保]]，有違規事跡，家人於是想送他去[[海軍新兵訓練中心|海軍士官學校]]磨練一番；但他連考士校三期都沒中，想到台北鐵工廠工作，一樣沒有人願意收留他，最後只好隨爸爸到酒店打雜。

於是，王夢麟參加歌唱比賽，得到第十名，奠定了他進入到演藝圈的信心與意志；當時的主持人是[[包國良|包國良]]。

== 青年 ==
王夢麟當兵退伍後，便立即結婚。由於當時沒有一技之長，王夢麟便買了一把吉他，在[[西餐廳|西餐廳]]唱日本流行歌曲，每天唱一個鐘頭，在家裡有空就寫寫歌，想當個創作及駐唱歌手。

從事[[計程車|計程車]]司機的生意期間，由於聽到[[邰肇玫|邰肇玫]]及[[楊耀東|楊耀東]]（[[金韻獎|金韻獎]]第一屆歌手）的創作及歌聲，更加興起自己寫歌的想法。也因英雄惜英雄而認識了趙樹海，又因為趙樹海認識邰肇玫。於是趙樹海建議王夢麟，把歌曲直接寄給[[新格唱片|新格唱片]]。1978年，王夢麟通過新格唱片發表第一首詞曲創作〈雨中即景〉一舉成名，與趙樹海及[[黃大城|黃大城]]時常搭檔演唱。當時，正好碰到[[中華民國|中華民國]]退出[[聯合國|聯合國]]的時代背景，沒有太多快樂的歌曲抒發；〈雨中即景〉歌曲表現詼諧與寫實，馬上成為大家朗朗上口的曲子！王夢麟是少數沒有參加金韻獎比賽，就成為唱片公司力捧的民歌手。

1979年，王夢麟有一首[[光棍|光棍]]想交女友的應景情歌〈阿美阿美〉，這首歌因緣際會成為追老婆的好調子，許多人的老婆都叫「阿美」；當時許多人靠這首歌也追到另一半，還讓王夢麟民歌生涯達到最高峰！

王夢麟原唱的著名歌曲有〈雨中即景〉、〈木棉道〉、〈阿美阿美〉、〈小草〉、〈廟會〉等。

== 中年 ==
1997年，王夢麟與趙樹海與黃大城共3人合組[[MIB三重唱|MIB三重唱]]，MIB三重唱總共發行兩張唱片專輯《Music Is Back》（1997年發行）、《母親我愛您》（1999年發行）。

2000年，王夢麟因犯過失殺人罪被判處[[有期徒刑|有期徒刑]]兩年，出獄後重回歌壇<ref>{{Cite web|url=http://news.tvbs.com.tw/old-news.html?nid=455525|author=[[拉娃谷幸|拉娃谷幸]]、孫銘遙|language=zh-tw|title=當紅、殺人、坐牢！ 王夢麟大起大落|publisher=[[TVBS|TVBS]]新聞|date=2005-02-16|accessdate=2015-12-04}}</ref>。

2015年7月16日，王夢麟否認他曾參加金韻獎，他從來沒有參加民歌比賽；〈雨中即景〉是他在家拿2台[[錄音機|錄音機]]錄好之後，請趙樹海託邰肇玫轉交新格唱片，受到新格唱片賞識才一舉成名<ref>{{Cite web|url=http://www.chinatimes.com/realtimenews/20150717001795-260404|author=張怡文、陳昀|language=zh-tw|title=王趙合體的創作小故事|publisher=[[中時電子報|中時電子報]]|date=2015-07-17|accessdate=2015-07-18}}</ref>。
