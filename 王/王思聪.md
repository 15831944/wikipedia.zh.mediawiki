'''王思聪'''（{{bd|1988年|1月3日|catIdx=W}}），生于中国辽宁省大连市，籍贯四川省广元市，原华人世界首富、中国首富、[[万达集团|万达集团]]董事长[[王健林|王健林]]的獨生子，现为网红，兼任[[大连万达集团|万达集团]]董事，[[北京普思投资有限公司|北京普思投资有限公司]]董事长，[[熊猫TV|熊猫TV]]创始人兼CEO。

王思聪自小被送往国外读书，回国后在家族企業万达集团任职。2011年，王思聪成立IG电子竞技俱乐部，此后，王一直对[[电子竞技|电子竞技]]行业有所投资<ref>{{Cite web|url=http://games.sina.com.cn/ol/n/2015-12-07/fxmihae9159613.shtml |title=王思聪：强势进入电竞圈是为了增加选手的收入 |accessdate=2016-01-25 |author= |date=2015-12-07 |publisher=新浪游戏 |archiveurl=https://web.archive.org/web/20160125051216/http://games.sina.com.cn/ol/n/2015-12-07/fxmihae9159613.shtml |archivedate=2016-01-25 |deadurl=yes }}</ref>。2015年，王思聪创立视频直播网站熊猫TV并担任CEO。

== 早年生活与教育 ==
王思聪于1988年1月3日出生于[[辽宁省|辽宁省]][[大连市|大连市]]，祖籍为[[四川省|四川省]][[广元市|广元市]][[苍溪县|苍溪县]][[元坝镇_(苍溪县)|元坝镇]]。他自小就被送到国外，小学就读于新加坡瑞士小屋（Swiss Cottage），中学与大学则就读于英国[[温切斯特公学|温切斯特公学]]与[[伦敦大学学院|伦敦大学学院]]哲学系。<ref>{{Cite web|url=http://edu.sina.com.cn/a/2015-03-04/0848258026.shtml |title=起底“国民老公”王思聪留学履历 |accessdate=2016-01-25 |author= |date=2015-03-04 |publisher=[[新浪|新浪]]教育 |archiveurl=https://web.archive.org/web/20150718191005/http://edu.sina.com.cn/a/2015-03-04/0848258026.shtml |archivedate=2015-07-18 |deadurl=yes }}</ref>

== 事业 ==
王思聪从英国归来即担任万达集团董事，但并不负责任何事务<ref>{{Cite web|url = http://www.yahui.cc/economy/China/741714-1.htm|title = 王思聪个人经历|accessdate = 2015-07-21|author = |date = 2014-10-15|publisher = }}</ref>。2009年，王健林拿出5亿元人民币让王思聪成立北京普思投资公司练手<ref>{{Cite web|url = http://ent.163.com/14/0716/09/A192105400031H2L.html|title = 王健林甩5亿让王思聪开公司练手:有本事就接班|accessdate = 2015-08-02|author = |date = 2014-07-16|publisher = 网易娱乐}}</ref>，不过2016年7月王思聪表示其父并未提供那么多，“一次性也就500万、1000万的支持，鼓励创业”<ref>{{Cite web|url=http://www.ithome.com/html/it/239903.htm|title=王思聪否认父亲提供5亿元创业款：一次仅500到1000万|accessdate=2016-07-10|author=仲平|date=2016-07-10|publisher=IT之家}}</ref>。至2010年，王思聪持有万达院线约1000万股<ref>{{Cite web|url = http://finance.sina.com.cn/money/cfgs/20141202/144320976500.shtml|title = 上市公司子承父业渐入高峰 少东家掌印几家兴盛几家衰|accessdate = 2015-07-31|author = |date = 2014年12月2日|publisher = 新浪财经}}</ref>。

2011年8月，王思聪收购CCM电竞俱乐部，成立[[IG电子竞技俱乐部|iG电竞俱乐部]]，他表示要“强势进入，整合[[电竞|电竞]]。”<ref>{{Cite web|url = http://games.ifeng.com/lanjie/yejiexiaoxi/detail_2011_08/03/8141446_0.shtml|title = 中国首富之子强势进入 立誓整合电竞圈|accessdate = 2015-07-31|author = |date = 2011年8月3日|publisher = 凤凰网游戏}}</ref>

2015年8月，王思聪成立经纪公司“[[香蕉計劃|香蕉计划文化发展有限公司]]”，同時簽約韓國女團[[T-ara|T-ara]]<ref>{{Cite web|url = http://money.163.com/api/15/0819/11/B1CJ0D0000254U80.html|title = 王思聪正式进军娱乐圈 被曝已签下韩国女子组合|accessdate = 2016-01-24|author = |date = 2015-08-19 |publisher = 网易}}</ref>。9月，王思聪在微博上宣布担任即将上线的视频直播平台[[熊猫TV|熊猫TV]]的CEO，并启动[[借贷|融资]]计划<ref>{{Cite web|url = http://tech.sina.com.cn/zl/post/detail/it/2015-09-08/pid_8489929.htm|title = 王思聪携熊猫TV来挖角，斗鱼缘何遭“围斗”？|accessdate = 2015-09-09|author = |date = |publisher = 新浪专栏}}</ref><ref>{{Cite web|url = http://tech.sina.com.cn/it/2015-09-07/doc-ifxhqhui4909824.shtml|title = 熊猫TV启动融资计划 王思聪在朋友圈向业界融资|accessdate = 2015-09-09|author = |date = |publisher = 新浪科技}}</ref>。多家外媒对此事予以报道，并称此举意图对抗[[亚马逊公司|亚马逊]]的[[Twitch|Twitch]]<ref>{{Cite web|url = http://www.mcvpacific.com/news/read/asia-panda-tv-to-launch-in-china-hopes-to-rival-twitch/0155567|title = ASIA: Panda TV to launch in China, hopes to rival Twitch|accessdate = 2015-09-23|author = |date = 2015-09-09|publisher = |language = en}}</ref><ref>{{Cite web|url = http://www.ubergizmo.com/2015/09/twitch-competition-china-panda-tv/|title = Twitch Could Face Competition From China’s Own Panda TV|accessdate = 2015-09-23|author = |date = 2015-09-08|publisher = ubergizmo|language = en}}</ref><ref>{{Cite web|url = http://games.qq.com/a/20150909/028280.htm|title = 外媒热议王思聪成立熊猫TV：中国首富之子开的|accessdate = 2015-09-23|author = |date = 2015-09-09|publisher = 腾讯游戏}}</ref>。10月24日，王思聪担任[[中国移动电竞联盟|中国移动电竞联盟]]第一届轮值主席<ref>{{Cite web|url = http://tech.qq.com/a/20151024/038291.htm|title = 中国移动电竞联盟成立 王思聪任主席|accessdate = 2015-10-27|author = |date = 2015-10-24|publisher = 腾讯科技}}</ref><ref>{{cite web|author1=莫苑|title=王健林给的5个亿去了哪 国民老公咋玩营销|url=http://edu.sina.com.cn/bschool/2015-02-05/1719457172.shtml|website=edu.sina.com.cn|publisher=新浪教育|accessdate=2017-08-04|ref=2015年02月05日17:19}}</ref>。

2016年12月30日，网友爆料称，王思聪与好友、青年演员[[林更新|林更新]]均是一家名为“上海水晶荔枝娱乐文化有限公司”的企业的自然人股东，注册资金1000万。网友表示，这家新公司与之前的“[[香蕉计划|香蕉计划]]”有“异曲同工之妙”——都是水果<ref>{{Cite web|url=http://news.21cn.com/caiji/roll1/a/2016/1230/15/31830702.shtml|title=曝王思聪林更新合伙开公司并取名“水晶荔枝”|accessdate=2016-12-30|author=|date=2016-12-30|publisher=21CN.COM|archiveurl=https://web.archive.org/web/20161230230404/http://news.21cn.com/caiji/roll1/a/2016/1230/15/31830702.shtml|archivedate=2016-12-30|deadurl=yes}}</ref>。

2018年8月17日，IG战队副总裁、IG官方俱乐部以及王思聪本人发布声明，王思聪正式注册成为电竞选手，加入IG电竞俱乐部，游戏ID：iG.WXZ<ref>{{cite news |title=史上最大牌的职业选手！王思聪注册成为IG队员征战LPL？ |url=http://ent.ifeng.com/a/20180817/43090339_0.shtml |work=凤凰网娱乐}}</ref>。9月19日，王思聪正式退役。<ref>{{cite news |title=王思聪放弃英雄联盟S赛名额正式退役 职业生涯31天百分百胜率 |url=http://sports.qq.com/a/20180919/013677.htm |work=腾讯体育}}</ref><ref>{{cite news |author1=刘姝君 |title=王思聪放弃S赛名额，注册一月多便退役 |url=http://www.bjnews.com.cn/sport/2018/09/19/506423.html |work=新京报网}}</ref>

== 个人生活 ==
2010年4月，企业家[[张兰_(企业家)|张兰]]声称[[徐熙媛|大S]]与[[汪小菲|汪小菲]]在[[海南省|海南]][[三亚市|三亚]]结婚的婚礼酒店由[[万达集团|万达]]免费提供，王思聪在微博上予以否认，称张兰是在造谣，并斥责张兰炒作。张兰对此回应称王思聪为“血气方刚的孩子”，后王思聪再度放话，称“张兰就造谣这事不值得尊重”<ref>{{Cite web|url = http://ent.qq.com/a/20110406/000316.htm|title = 王思聪义愤难平 再度炮轰张兰造谣不值得尊重|accessdate = 2015-07-31|author = |date = 2011-04-06|publisher = 腾讯娱乐}}</ref>。

2014年9月，王思聪认为[[搜狐|搜狐]]娱乐刊登的报道《富二代王思聪掷千万投票网友：[[SNH48|SNH48]]完爆[[AKB48|AKB48]]》一文捏造虚假事实，影响其名誉，将搜狐网等三被告起诉到法院，要求删除该报道，公开道歉并赔偿经济损失20万元<ref>{{Cite web|url = http://china.cnr.cn/ygxw/20150205/t20150205_517654455.shtml|title = 王思聪诉搜狐网名誉权案一审开庭 认为报道捕风捉影|accessdate = 2015-07-31|author = |date = 2015-02-05|publisher = 央广新闻}}</ref>。

2015年2月，王思聪在接受采访时调侃道自己的择偶标准是“胸大”，遭到中国官媒[[新华社|新华社]]痛批，后者称其“传播三俗”“不以为耻，反以为个性”<ref>{{Cite web|url = http://fashion.ifeng.com/a/20150305/40087173_0.shtml|title = 王思聪称择偶标准要胸大 官媒批其“三俗”|accessdate = 2015-09-23|author = |date = 2015-03-05|publisher = 凤凰网}}</ref>。

2015年5月，王思聪在微博上发布其爱犬王可可佩戴两块苹果手表的照片引发网友热议。此事亦被美国[[YAHOO|YAHOO]]财经频道报道<ref>{{Cite web|url = http://finance.yahoo.com/news/chinese-heir-buys-apple-watches-for-dog-163153148.html#|title = Chinese heir buys Apple Watches for dog|accessdate = 2015-08-02|author = |date = 2015-05-26|publisher = }}</ref><ref>{{cite web|title=王思聪和爱犬王可可合照曝光 网友:时刻秀恩爱啊!_千寻生活|url=http://www.orz520.com/a/fun/2017/1030/6648464.html?from=haosou_news|website=www.orz520.com|publisher=千寻生活|accessdate=2017-11-04|ref=http://news.so.com/ns?q=%E7%8E%8B%E5%8F%AF%E5%8F%AF|pages=1|language=zh-hans|format=html|date=2017-10-30<!-- 13:43-->}}</ref>。

2014年，王思聪与艺人[[张予曦|张予曦]]相识并交往。2015年1月21日，张予曦在新浪微博发文承认与王思聪分手<ref>{{Cite web|url=http://weibo.com/1662262590/C0AvNanD8|title=王思聪与张予曦分手|accessdate=2015-07-31|author=张予曦|date=2015-01-21|publisher=}}</ref>。
