'''王娜娜事件'''是一起冒名顶替上大学事件。

2002年，[[周口市|周口市]][[沈丘县|沈丘县]]的高考生'''王娜娜'''落榜。2003年，再次参加[[高考|高考]]，未收到录取通知书，她以为再次落榜。便外出打工，继而结婚生子<ref name = "huanqiu">{{Cite web |url = http://china.huanqiu.com/hot/2016-07/9163505.html |title = "王娜娜事件"再调查：尚未收到信息公开答复 |author =  |date = 2016-7-13 |publisher = 環球網 |language = 中文 |accessdate = 2018-8-3 |quote =  }}</ref>。

2014年，王娜娜办理银行业务时被拒。次年申请[[借贷|贷款]]时亦被拒，原因是银行发现其为[[專科學校|大专]]学历，而其却填的是高中学历。由此，王娜娜发现自己被人顶替上了大学<ref name = "huanqiu" />。顶替者叫'''张莹莹'''（2003年高考落榜），其父张合停托人找到了王娜娜的录取通知书，而后在一家打印店制作了一系列假证件。随后张莹莹改名王娜娜，顶替完成学业，最后成为一名教师<ref>{{Cite web|url=http://edu.people.com.cn/n1/2016/0429/c367001-28314492.html|title=周口再次公布"王娜娜事件"调查结果 三人移送司法|accessdate=2018-8-3|author=|date=|last=|work=|publisher=人民网}}</ref>。

之后，王娜娜联系到张莹莹（假王娜娜）希望对方注销学籍，但对方声称即使告到联合国也没用<ref>{{Cite web|url=https://www.guancha.cn/Education/2016_02_27_352287_s.shtml|title=校方证实“女生被顶替上大学” 顶替者曾称闹到联合国也不怕|accessdate=2018-08-02|author=|date=|last=|first=|work=|publisher=|language=zh-hans}}</ref>。由于对方嚣张的气焰，引起网友愤慨，微博热议。媒体也争相报道<ref>{{Cite web|url=http://news.163.com/16/0229/02/BGV8CG4A00014Q4P.html|title=“假王娜娜”学历已被注销顶替者曾扬言告到联合国也没用_网易新闻|accessdate=2018-08-02|last=网易|work=news.163.com}}</ref><ref>{{Cite web|url=http://www.xinhuanet.com/local/2016-05/03/c_128951181.htm|title=“王娜娜事件”调查报告公布 距发现被顶替正好一年-新华网|accessdate=2018-08-02|work=www.xinhuanet.com}}</ref>。张莹莹父亲希望通过8万元私了，被拒绝。假王娜娜的学历被注销，并被学校解聘。但是学校拒绝为王娜娜恢复学籍<ref>{{Cite web|url=http://edu.qq.com/a/20161110/014314.htm|title=王娜娜申请恢复学籍被拒准备高考 官方回应_教育_腾讯网|accessdate=2018-08-02|work=edu.qq.com|language=zh-CN}}</ref><ref>{{Cite web|url=http://epaper.oeeee.com/epaper/A/html/2016-05/02/content_33030.htm|title=[个论]熊丙奇专栏:王娜娜事件的最终真相是什么-南方都市报·奥一网|accessdate=2018-08-02|work=epaper.oeeee.com|language=en}}</ref><ref>{{Cite web|url=http://news.ifeng.com/a/20161111/50239139_0.shtml|title=拒绝王娜娜重新入学，不合情也不合理|accessdate=2018-08-02|work=news.ifeng.com}}</ref>。

2017年，王娜娜再次参加高考<ref>{{Cite web|url=http://opinion.people.com.cn/n1/2017/0607/c1003-29322545.html|title=钱江晚报：王娜娜再高考，不向被篡改的命运屈服--观点--人民网|accessdate=2018-08-02|author=|date=|last=|work=opinion.people.com.cn|publisher=}}</ref><ref>{{Cite web|url=http://news.chinaxiaokang.com/dujiazhuangao/2017/0627/213197.html|title=王娜娜刚过专科线 回顾王娜娜的14年高考路 王娜娜事件张莹莹现状如何？_中国小康网|accessdate=2018-08-02|work=news.chinaxiaokang.com}}</ref>。
