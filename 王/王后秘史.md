《'''王后秘史'''》（{{lang-en|The King Maker}}）2005年10月20日上映于泰国的历史动作片，讲述了泰国历史上的[[暹罗|暹罗]]时期[[大城王国|大城王国]]王后篡位的历史故事。

== 剧情 ==
电影主要讲述1457年，暹罗时期[[大城王国|大城王国]]王后篡位，后来被诛杀的历史故事。<ref>{{cite web |url=http://news.sohu.com/20070309/n248621057.shtml |title= 好莱坞大片讲泰国宫廷故事|publisher=搜狐网 |accessdate=2007-03-09 |work= |language =}}</ref>

菲尔南多·德伽玛是[[西班牙|西班牙]]人，他的父亲和塔先生是好友，但是后来在一个风雨交加的晚上，塔先生纵火将其杀害，那时候，德伽玛才8岁，亲眼目睹了这场变故，对杀父仇人终生不忘，经常做噩梦梦见那晚的那一幕。

多年以后，德伽玛长成了大个子，打算出海寻宝，在泰国海域不幸被打败，成为泰国市场上的[[奴隶|奴隶]]，当[[奴隶主|奴隶主]]抓着他到奴隶市场交易时，他悄悄地逃走了，奴隶主开始追捕他，一身好武艺的他屡次逃过他们的追捕，最后还是奴隶主人多势众，伤了他，正好倒在葡萄牙女人玛利亚面前，看到是欧洲人，玛利亚出钱买下了他，并且给他换上了新衣服。

玛利亚的父亲是泰国国王的工程师，精通城堡的设计，也就是德伽玛的杀父仇人，他已经来到泰国8年了。

泰国国王因为战事而忽略了和王后沟通感情，对王后的冷漠，没有满足其权力欲望和[[性需求|性需求]]，身居深宫寂寞难耐的王后选择了[[外遇|偷情]]，背叛了国王，她经常和一个美男子偷食禁果，并且怀了孕。

一次趁着国王出征，王后找到塔先生请他帮忙除掉国王，而国王在出征中看到德伽玛武艺非凡，于是命令他做了随从。当国王出征回国，塔先生雇用的[[日本|日本]][[武士|武士]]在夜晚前去刺杀就寝了的国王，但是被士兵和德伽玛击毙了，随后国王下令全国通缉犯人同伙，最终查出凶手为塔先生，并且将他绳之以法。

刺杀计划失败后，王后找到女巫，女巫给王后炼制了一种毒药，王后用毒药毒死了国王，随后亲手杀掉了自己的儿子，故作痛苦，然后把情人扶持到了国王的位置，随后对全国不服从者依法处斩，将德伽玛和玛利亚都抓捕了起来，并且不慎将一些葡萄牙人杀害。

王后的倒行逆施引发了[[葡萄牙|葡萄牙]]人的不满，在最后，被葡萄牙王子带领的军队杀掉。

== 花絮 ==
该片和1963年的《[[埃及艳后_(电影)|埃及艳后]]》都被誉为[[史诗电影|史诗电影]]。<ref name="lion " >{{cite web |url= http://ent.sina.com.cn/m/f/2007-05-22/12171564095.html|title=好莱坞重金打造泰国艳后 《王后秘史》华丽登场 |publisher=新浪网 |accessdate=2007年5月22日 |work= |language =}}</ref>[[好莱坞|好莱坞]]打造该片继承了《[[角斗士|角斗士]]》、《[[亚瑟王|亚瑟王]]》的血性和暴力场面。<ref>{{cite web |url=http://ent.sina.com.cn/m/f/2007-04-12/07261515981.html |title=《王后秘史》17日重庆上映 模仿黄金甲吸引眼球 |publisher= 新浪网|accessdate=2007-04-12 |work= |language =}}</ref>

该片诸多元素灵感来自华语电影《[[满城尽带黄金甲|满城尽带黄金甲]]》、《[[卧虎藏龙|卧虎藏龙]]》和[[成龙|成龙]]的[[功夫片|功夫片]]。<ref name="lion " /><ref>{{cite web |url=http://ent.sina.com.cn/m/f/2007-03-20/16581486131.html |title=古装宫廷大片《王后秘史》很中国 打斗场面出色 |publisher= 新浪网|accessdate=2007-03-20 |work= |language =}}</ref>

该片主演加里·斯加奇是[[英国|英国]]拳坛的世界冠军。<ref>{{cite web |url=http://yule.sohu.com/20070310/n248634197.shtml |title=《王后秘史》将公映 以西方视角展现东方魅力 |publisher=搜狐网 |accessdate=2007-03-10 |work= |language =}}</ref>
