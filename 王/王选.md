'''王选'''（{{bd|1937年|2月5日|2006年|2月13日|catIdx=W王}}），生于[[上海|上海]]，[[江苏|江苏]][[无锡|无锡]]人，[[中国科学院院士|中国科学院院士]]、[[中国工程院院士|中国工程院院士]]，[[北京大学|北京大学]]教授，[[九三学社|九三学社]]中央副主席、[[全国政协副主席|全国政协副主席]]。他是[[汉字激光照排系统|汉字激光照排系统]]的创始人和技术负责人，在1987年和另外6人共同获得了首届毕昇印刷技术奖<ref>[http://www.chinaprint.org/rdzt/bisheng/ 中国印刷行业网-历届毕昇印刷技术奖获奖名单]</ref>。此外，凭借照排技术，王选创立了[[北大方正|北大方正]]<ref>[http://www.founder.com/about/intro.html 北大方正介绍]</ref>。

== 简历 ==
1941年至1954年间曾就读于上海市私立南洋模范中小学（小学部为今[[上海市天平路第一小学|上海市天平路第一小学]]；初中部今[[上海市七宝中学|上海市七宝中学]]；高中部为今[[上海市南洋模范中学|上海市南洋模范中学]]）长达13年<ref>{{cite web |url = http://scitech.people.com.cn/GB/25509/55787/58263/58264/4104129.html|title=王选回忆在南洋模范的日子|publisher=人民网 |accessdate=2010-05-03}}</ref>，1958年毕业于[[北京大学|北京大学]]数学力学系。1975年开始主持中国计算机汉字激光照排系统与电子出版系统并取得成功。1992年研制成功世界首套中文彩色照排系统。他所领导的科研集体研制出的汉字激光照排系统为新闻、出版全过程的计算机化奠定了基础，被誉为“汉字印刷术的第二次发明”。

2001年[[江泽民|江泽民]]总书记向他颁发[[国家最高科学技术奖|国家最高科学技术奖]]，他是第三个获得此奖的科学家。2003年當選為[[中國人民政治協商會議|中國人民政治協商會議]]第十屆全國委員會副主席。2006年2月13日11时3分因病在[[北京|北京]]去世。为表示纪念，小行星4913号被命名为“王选星”。

== 家庭 ==
王选出生于无锡望族，曾祖父[[王縡|王縡]]为清[[同治|同治]]年间[[进士|进士]]。父亲王守其毕业于[[南洋公学|南洋公学]]，是一名[[会计师|会计师]]。母亲周邈清也来自书香门第，她的父亲[[周道章|周道章]]是中国近代将数学算式从竖排改为横排的第一人<ref>{{cite web |url = http://www.founder.com/cn/paperhtml/201002/show-67-20100224111737.htm |title = 汉字作证——记“当代毕昇”王选院士 |publisher = 方正集团 |date = 2010-02-24}}</ref>。王选有兄弟姊妹五人，其中二哥[[王迅_(物理学家)|王迅]]为[[复旦大学物理系|复旦大学物理系]]教授，亦为[[中国科学院院士|中国科学院院士]]<ref>{{cite web|url=http://www.physics.fudan.edu.cn/tps/people/xwang/info.htm |title = 王迅教授 |publisher = 复旦大学物理系}}</ref>。

王选与妻子[[陈堃銶|陈堃銶]]在北京大学读书时相识，并于1967年结婚。不过他们夫妇二人并没留下子女。
