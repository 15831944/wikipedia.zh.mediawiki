《'''王稼祥'''》，又名:厚爱，中国大陆电视剧，导演[[王一岩|王一岩]]，主演王洛勇、王霙、卢奇、刘劲、秦丽，2014年5月取得发行许可证。

== 剧情 ==
该剧讲述了“王稼祥”1930年从[[苏联|苏联]][[莫斯科中山大学|莫斯科中山大学]]学成归国到1951年卸任中华人民共和国第一任驻苏联大使恢弘壮阔的人生故事。<ref name="man " /><ref>{{cite web |url=http://www.chinanews.com/yl/2012/04-20/3836038.shtml |title=领袖明星云集电视剧《王稼祥》  |accessdate=2012年4月20日 |publisher= 中新网|work= |language =zh-cn}}</ref>。2011年4月在北京开机<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2011-04-29/03083295359.shtml |title=电视剧《王稼祥》开机 王洛勇饰演王稼祥(图)  |accessdate=2011年4月29日 |publisher= 新浪娱乐|work= |language =zh-cn}}</ref>。

== 花絮 ==
该剧由[[中国电视艺术家协会|中国电视艺术家协会]]、[[中央新闻纪录电影制片厂|中央新闻纪录电影制片厂]]、[[中国文学艺术基金会|中国文学艺术基金会]]、安徽京之盾文化传播有限公司一同制作。<ref name=" fish" >{{cite web |url=http://www.chinanews.com/cul/2011/12-22/3552664.shtml |title=王洛勇出演电视剧《王稼祥》:他的人生经历打动人 |accessdate=2011年12月22日 |publisher= 中新网|work= |language =zh-cn}}</ref>

该剧曾在[[上海|上海]]、[[江西|江西]][[瑞金|瑞金]]、[[安徽|安徽]]、[[延安|延安]]、[[西柏坡|西柏坡]]、[[中国东北|中国东北]]、[[莫斯科|莫斯科]]等地方取景。<ref name=" fish" />曾取景的影视基地是八一电影制片厂摄影棚、[[八一王佐影视基地|八一王佐影视基地]]、[[怀柔中影基地|怀柔中影基地]]、[[横店影视城|横店影视城]]。<ref name="man " >{{cite web |url= http://www.cndfilm.com/20120508/103292.shtml|title= 电视连续剧《王稼祥》横店顺利杀青|accessdate=2012年5月8日  |publisher=中央新影集团 |work= |language =zh-cn}}</ref>
