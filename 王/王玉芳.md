'''王玉芳'''（{{bd|1942年|12月|||}}），女，[[彝族|彝族]][[撒尼人|撒尼人]]，出生于[[云南省|云南省]][[昆明市|昆明市]][[石林彝族自治县|石林彝族自治县]][[长湖镇_(石林县)|长湖镇]]宜政村，被称为“活在民间的阿诗玛”<ref>*{{cite journal |author1=巴胜超 |author2=罗雨 |date=2018 |title=活在民间的“阿诗玛”——云南叙事长诗《阿诗玛》国家级“非遗”传承人王玉芳访谈录 |journal=《文化遗产》 |issue=2 |pages=129-135 }}</ref>。2007年被[[中华人民共和国文化部|中华人民共和国文化部]]确定为《[[阿诗玛|阿诗玛]]》的[[国家级非物质文化遗产|国家级非物质文化遗产]]项目传承人<ref>{{cite web |url=http://www.yizuren.com/tradition/yzwhccybh/36312.html |title=阿诗玛的故乡石林，新增三位国家级非物质文化遗产项目传承人 |author1=许华 |author2=黄平山 |author3=孔庆云 |date=2018-01-10 |website=www.yizuren.com |publisher=彝族人网 |access-date=2018-04-19 |quote=}}</ref>，随着2013年另一位传承人[[毕华玉|毕华玉]]故去，王玉芳成为唯一一位《阿诗玛》的国家级非遗传承人。与毕华玉重在彝族传统文化的研究和整理不同的是，王玉芳主要口传和演唱《阿诗玛》<ref>{{cite web |url=http://cn.globalgeopark.org/gyzt/one/shilin/xw/4965.htm |title=石林阿诗玛传习馆7月亮相 |author=贾薇 |date=2008-06-17 |website=cn.globalgeopark.org |publisher=[[中国地质科学院|中国地质科学院]] |access-date=2018-04-19 |quote=}}</ref>。王玉芳能演唱“喜调”、“老人调”、“悲调”、“哭调”、“骂调”等多种版本的《阿诗玛》，是目前石林唱《阿诗玛》最全、技艺最高的艺人<ref>{{cite web |url=http://www.chinastoneforest.com/showarticle.aspx?cid=131&aid=4235 |title=寻访流淌在血液里的阿诗玛 她让石林更美丽 |author=邓建华 |date=2010-11-10 |website=www.chinastoneforest.com |publisher=云南信息港 |access-date=2018-04-19 |quote=}}</ref>。除《阿诗玛》外，王玉芳也掌握《牧羊姑娘》、《圭山彩虹》、《竹叶长青》、《库吼调》、《地名调》等经典撒尼民歌的演唱<ref>{{cite web |url=http://travel.yunnan.cn/html/2017/jinritoutiao_0417/3839_2.html |title=石林长湖《阿诗玛》传习馆建成使用 |author=王丹 |date=2017-04-17 |website=travel.yunnan.cn |publisher=[[云南网|云南网]] |access-date=2018-04-19 |quote=}}</ref>。
