'''王端端'''（{{bd|1978年|3月16日|catIdx=W王}}），[[中國中央電視臺|中國中央電視臺]][[中国中央电视台中文国际频道|中文國際頻道]]《[[中國新聞|中國新聞]]》[[主持人|主持人]]。已婚，[[中國共產黨|中國共產黨]]黨員。2006年CCTV十佳主持人之一，2007年CCTV[[月历|月历]]人物。

== 主要經歷 ==
王端端大学毕业于[[沈阳理工大学|沈阳理工大学]]国际金融系。<ref name="has " /> 大学毕业后，王端端在[[辽宁电视台|辽宁电视台]]任职。<ref name="has " /> 2003年至今，王端端擔任[[中國中央電視臺|中國中央電視臺]][[中国中央电视台中文国际频道|中文國際頻道]]《中國新聞》系列新聞類節目主要主持人之一。<ref name="has " >{{cite web |url=http://money.591hx.com/article/2010-12-07/0000135913s_2.shtml |title=从地方台跳槽央视后大红大紫的女主播 |accessdate= 2010年|work= |publisher=华讯 |language =zh-cn}}</ref>

== 活动 ==
*2005年11月6日 [[北京|北京]] 2005年中国品牌经济城市峰会闭幕式暨授牌仪式
*2007年1月23日 [[北京|北京]] 中国·2006-2007建设[[和諧社會|社会主义新农村]]论坛
*2007年，王端端在《[[現代快報|現代快報]]》上不定期有專欄。

== 奖项 ==
*1999年，获[[朵而|朵而]]女性新主持人大赛东北赛区总冠军，全国优胜奖。
*2000年，获[[中央电视台|中央电视台]][[荣事达|荣事达]]杯全国主持人大赛优秀奖。
