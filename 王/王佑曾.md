'''王佑曾'''（{{lang-en|'''Eugene Wong'''}}，{{bd|1934年|12月24日|}}），[[中央研究院院士|中央研究院院士]]<ref>{{Cite web |url=https://db1n.sinica.edu.tw/textdb/academicians/index.php?func=10 |title=院士基本資料查詢 |accessdate=2012-08-12 |work= |publisher=[[中央研究院|中央研究院]] |date= }}</ref>。

==生平==
1959年取得美國普林斯頓大學電機博士學位。柏克萊大學名譽教授，<ref>{{Cite web |url=http://www.eecs.berkeley.edu/Faculty/Homepages/wong.html |title=Eugene Wong EECS at UC Berkeley |accessdate=2012-08-12 |work= |publisher=UC Berkeley |date=2012-02-01}}</ref>中央研究院資訊科學研究所特聘講座研究員。1992年當選19屆中央研究院院士。2012年以中華民國[[國家科學委員會|國科會]]科學技術發展諮議會委員身份，參與「第一次科技諮議會」。<ref>{{Cite web |url=http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS6/7274857.shtml |title=國科會第一次諮議會6日展開為期兩天會議 |accessdate=2012-08-12 |work= |publisher=聯合新聞網 |date=2012-08-06}}</ref>
