'''王小洪'''（{{bd|1957年|7月||catIdx=W王}}），[[福建省|福建]][[福州市|福州]]人。男，汉族。1974年7月参加工作，1982年12月加入[[中国共产党|中国共产党]]。[[中国人民公安大学|中国人民公安大学]]公安管理专业毕业，[[中共中央党校|中共中央党校]]函授学院经济学本科。曾任[[河南省人民政府|河南省人民政府]]副省长兼省公安厅厅长、[[北京市人民政府|北京市人民政府]]副市长市公安局局长。现任[[中华人民共和国公安部|中华人民共和国公安部]]常务副部长（[[正部长级|正部长级]]）、党委副书记兼[[北京市公安局|北京市公安局]]局长、[[党委书记|党委书记]]，[[副总警监|副总警监]][[中华人民共和国人民警察警衔|警衔]]。[[中国共产党第十九届中央委员会委员列表|中国共产党第十九届中央委员会委员]]。

== 早年经历 ==
1974年7月，响应[[中国共产党中央委员会|中共中央]]对于[[知识青年|知识青年]][[上山下乡运动|上山下乡]]的号召，赴[[福建省|福建省]][[闽侯县|闽侯县]][[插队|插队]]，后进入“闽侯县风机厂”当工人。

== 仕途 ==
*1979年12月，任闽侯县公安局干部、团支部书记。
*1984年5月，任闽侯县公安局副局长（1985年9月至1987年7月，在[[中国人民公安大学|中国人民公安大学]]公安管理专业学习）。
*1989年1月，任闽侯县公安局局长，[[福州市|福州市]]公安局郊区分局局长。
*1993年8月，任福州市公安局副局长、党委副书记（1994年10月确定为正处级，1994年8月至1996年12月，在[[中共中央党校|中共中央党校]]函授学院本科班经济管理专业学习毕业）。
*1998年2月，任福州市公安局局长、党委书记、市委政法委副书记，[[漳州市|漳州市]]公安局局长、党委书记、市委政法委副书记、市委委员（副厅级）。
*2002年5月，任[[福建省公安厅|福建省公安厅]]副厅长、兼福建省公安边防总队第一政委、党委第一书记。
*2011年9月，任[[厦门市|厦门市]]副市长、市委政法委副书记兼市公安局局长，授[[一级警监|一级警监]][[警衔|警衔]]。

*2013年8月，任[[河南省人民政府|河南省人民政府]]省长助理兼[[河南省公安厅|河南省公安厅]]厅长<ref>{{cite web|url=http://henan.sina.com.cn/news/z/2013-08-01/2106-84646.html|title=王小洪任河南省公安厅厅长(简历)|accessdate=2017-10-24|date=2013-08-01|publisher=厦门政府网}}</ref><ref>{{cite web|url=http://www.mps.gov.cn/n16/n1282/n1720292/3860962.html|title=王小洪被任命为河南省公安厅厅长|accessdate=2017-10-24|date=2013-08-02|publisher=中华人民共和国公安部|deadurl=yes|archiveurl=https://web.archive.org/web/20160304100252/http://www.mps.gov.cn/n16/n1282/n1720292/3860962.html|archivedate=2016-03-04}}</ref>。2014年12月，任河南省副省长兼省公安厅厅长（2015年1月，被授予副总警监警衔）<ref>{{Cite web|url=http://news.163.com/14/1208/01/ACTIGI980001124J.html|title=近期多省公安厅长被任命为副省长 均由异地调任|accessdate=2017-10-24|date=2012-08-01|publisher=网易}}</ref>。

*2015年3月，任[[北京市人民政府|北京市人民政府]]副市长兼[[北京市公安局|北京市公安局]]局长<ref>{{cite web|url=http://gov.163.com/15/0326/17/ALLAQQLD00234IJL.html|title=王小洪被任命为北京市副市长、市公安局长|accessdate=2017-10-24|author=|date=2015-03-26|publisher=新京报|language=}}</ref><ref>{{cite web|url=http://www.bbc.co.uk/zhongwen/simp/china/2015/03/150326_new_beijing_police_chief|title=习近平旧部王小洪调任北京公安局长|accessdate=2017-10-24|date=2015-03-26|publisher=BBC中文网}}</ref>。

*2016年5月，任[[中华人民共和国公安部|中华人民共和国公安部]]副部长、[[北京市人民政府|北京市人民政府]]副市长兼[[北京市公安局|北京市公安局]]局长<ref>{{cite web|url=http://news.qq.com/a/20160520/026021.htm|title=王小洪任公安部副部长 北京两任公安局长获重用|accessdate=2016-05-20|date=2016-05-20|publisher=长安街知事}}</ref>。2017年5月，被明确为正部长级<ref>{{cite news|url=http://district.ce.cn/newarea/sddy/201705/11/t20170511_22710554.shtml|title=公安部领导排名调整 王小洪排第四|date=2017-05-11|newspaper=中国经济网|accessdate=2017-05-22}}</ref><ref>{{cite news|url=http://www.thepaper.cn/newsDetail_forward_1690295|title=官方媒体解读干部人事：公安部的正部级副部长都有谁？|date=2017-05-21|newspaper=澎湃新闻|accessdate=2017-05-22}}</ref>。

*2018年1月30日，北京市舉行十五屆人大一次會議，王小洪不再擔任北京市副市長職務。
*2018年3月，任公安部党委副书记、分管日常工作的副部长（正部长级）<ref>{{Cite web|url=http://www.mps.gov.cn/n2254314/n2254315/n2254317/n4894885/index.html|title=王小洪|accessdate=2018-03-25|publisher=中华人民共和国公安部}}</ref>。

== 任内成就 ==
王小洪自担任河南省公安厅厅长后，对涉黄涉赌的娱乐场所进行了有效的查处。2013年8月，王小洪在河南公安系统开展了“无声风暴”扫黄打赌专项行动，指派[[新乡市|新乡市]]公安局的民警对郑州[[郑东新区|郑东新区]][[皇家一号|皇家一号]]会所进行了查封，此事件受到社会与舆论的一致好评。在2015年3月调任北京市公安局局长后，北京警方于5月对6家涉黄涉赌娱乐场所进行责令停业整顿<ref>{{cite web|url=http://news.sina.com.cn/o/2015-05-26/161931878174.shtml|title=王小洪转战北京扫黄|accessdate=2017-10-24|date=2015年5月26日|publisher=新安晚报}}</ref>。
