'''王春法'''（{{bd|1963年|11月||}}）[[山东|山东]][[乳山|乳山]]人，[[中国共产党|中国共产党]]党员，[[中華人民共和國政府|中华人民共和国政府]]官员，现任[[中国国家博物馆|中国国家博物馆]]馆长<ref name=gb>{{Cite web|url =http://www.chnmuseum.cn/tabid/301/Default.aspx|title =关于国博|author =|date =|publisher =中国国家博物馆|language =|accessdate =2018-01-04|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170329173501/http://chnmuseum.cn/tabid/301/Default.aspx|archivedate =2017-03-29}}</ref>。

==生平==
王春法1981年考入兰州大学历史学系。<ref name="sme.bit">{{cite web|title=中国科学技术协会王春法研究员受聘我校兼职教授|url=http://sme.bit.edu.cn/zw/xwzx/xyyw/7393.htm|publisher=北京理工大学管理与经济学院科研与学术交流中心|accessdate=2018-05-10}}</ref>本科毕业后，他在兰州大学经济学系继续读书，1988年获经济学硕士学位。<ref>{{cite web|title=前进中的兰州大学经济学院|url=http://msite.eol.cn/1868/91999.shtml|publisher=中国教育在线|accessdate=2018-05-10}}</ref><ref>{{cite news|title=中国科协书记处书记王春法简历|url=http://edu.people.com.cn/n/2013/0515/c1053-21491704.html|accessdate=2018-05-10|work=人民网|date=2013-05-15}}</ref>1988年6月加入中国共产党，1988年6月参加工作。先后在[[中国地质大学|中国地质大学]]、[[中国社会科学院|中国社会科学院]]从事教学及学术研究。参加工作后，王春法于1998年获中国社会科学院研究生院经济学博士。曾在[[美国|美国]][[乔治·华盛顿大学|乔治·华盛顿大学]]当访问学者。历任中国社会科学院政策研究室学术秘书（副局级），[[全国人大常委会办公厅|全国人大常委会办公厅]]研究室国际室主任，[[中国科协|中国科协]]调研宣传部副部长（主持工作）、部长、机关党委书记。2011年5月，出任中国科协常委、书记处书记、党组成员。其间，2016年2月至6月任[[中央巡视组|中央第七巡视组]]副组长。2017年12月起任[[中国国家博物馆|中国国家博物馆]]馆长（副部长级）<ref name=gb/><ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1935720  |title = 王春法任国家博物馆馆长 |author =  |date =2018-01-04  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

王春法长期从事科技政策尤其是技术创新理论和政策研究，多次参与中央重要文件的研究起草，多篇调研报告曾获[[党和国家领导人|党和国家领导人]]批示。曾任国家中长期科学和技术发展规划纲要起草组成员、战略研究专家，中国文字著作权协会副会长，中国科学学与科技政策研究会副理事长，是《科学学研究》、《中国科技论坛》、《科学学与科技管理》等杂志编委。发表学术论文100多篇，主持翻译了《决策科学化译丛》、《科学文化译丛》等学术著作<ref name=gb/>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。
