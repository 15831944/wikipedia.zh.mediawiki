'''王方庆'''（{{bd|？||702年|catIdx=W}}），名'''綝'''，字'''方庆'''，[[以字行|以字行]]<ref name=NBT72>《新唐书》卷七十二</ref>，封'''石泉县公'''，谥'''贞'''，[[唐朝|唐朝]]、[[武周|武周]]官员，[[武则天|武则天]]年间为[[宰相|宰相]]。

== 家世 ==
王方庆出自[[琅邪王氏|琅邪王氏]]，属于[[王导|王导]]第三子[[王洽_(东晋)|王洽]]一支，[[北周|北周]]军于[[承圣|承圣]]三年（554年）攻占南梁都城[[江陵|江陵]]时，王方庆的曾祖[[王褒_(南北朝)|王褒]]被北周俘虏。王褒及其后人随后做了北周及相继的[[隋朝|隋朝]]、唐朝官员，从丹徒迁徙到咸阳。王方庆父[[王弘直|王弘直]]历任唐朝开国皇帝[[唐高祖|唐高祖]]之子汉王[[李元昌|李元昌]]和荆王[[李元景|李元景]]之友。<ref name=BT89>[[:s:旧唐书/卷89|《旧唐书》卷八十九]]</ref><ref name=NBT116>[[:s:新唐书/卷116|《新唐书》卷一百一十六]]</ref>

王方庆本人十六岁入仕，为[[唐太宗|唐太宗]]子越王[[李贞_(唐朝)|李贞]]府[[参军|参军]]，向李贞记室[[任希古|任希古]]学习《[[史记|史记]]》《[[汉书|汉书]]》，后来任希古迁[[太子舍人|太子舍人]]{{noteTag|因史籍没进一步记载任希古何时升迁，故此处所指太子是谁不详。}}，王方庆也跟随他继续学习。[[唐高宗|唐高宗]][[永淳|永淳]]年间（682年-683年），王方庆升迁为太仆少卿。永淳元年二月，高宗破例立孙[[李重照|李重照]]为[[皇太孙|皇太孙]]，想为他开府置僚属，咨询吏部侍郎[[裴敬彝|裴敬彝]]和时任吏部郎中的王方庆，王方庆认为皇太子尚在而立皇太孙史无前例，但高宗可开此先例。高宗很高兴，但最终未能为皇太孙设立府僚。<ref>[[:s:旧唐书/卷5|《旧唐书》卷五]]</ref><ref>[[:s:新唐书/卷81|《新唐书》卷八十一]]</ref><ref>[[:s:资治通鉴/卷203|《资治通鉴》卷二百零三]]</ref>永淳二年（683年），高宗崩，他有权势的妻子武皇后（后世称武则天）先后作为他们的儿子[[唐中宗|唐中宗]]（[[嗣圣|嗣圣]]元年（684年）废）、[[唐睿宗|唐睿宗]]的太后和摄政。她摄政期间，王方庆累迁[[广州|广州]]都督。当时广州作为国际贸易港口日益重要，王方庆的前任[[路元睿|路元睿]]因冒取马来商人的货物，于[[文明_(唐朝)|文明]]元年（684年）被马来商人所杀。王方庆继任后，诚实廉洁不受贿。他的辖区内长期有受到特殊照顾并贿赂官员的贪腐部落首领，王方庆禁止官员和他们交通，并拘捕了最贪腐的部落首领们。当时议者认为唐朝建立以来，再无人管治广州能超过王方庆。武太后也下诏褒奖他，赐瑞锦、杂彩，<ref name=BT89/>褒其善政。<ref name=NBT116/><ref>[[:s:褒广州都督王方庆制|:s:褒广州都督王方庆制]]</ref>

[[天授_(武周)|天授]]元年（690年），时年十三岁的[[张九龄|张九龄]]上书王方庆，王方庆大为嗟赏，说：“此子必能致远。”<ref>[[:s:旧唐书/卷99|《旧唐书》卷九十九]]</ref><ref>[[:s:新唐书/卷126|《新唐书》卷一百二十六]]</ref>

== 武则天年间 ==
武则天从睿宗手中夺取帝位称帝，建立周朝，中断唐朝，[[证圣|证圣]]元年（695年），召回王方庆为[[洛州|洛州]]长史，不久加[[银青光禄大夫|银青光禄大夫]]，封'''石泉县男'''。<ref>《新唐书》本传作'''县子'''。</ref>[[万岁通天|万岁通天]]元年（696年），转[[并州|并州]]长史，封'''琅邪县男'''，尚未就职，九月又迁鸾台侍郎，授同凤阁鸾台平章事，为实质宰相。<ref name=NBT4>[[:s:新唐书/卷4|《新唐书》卷四]]</ref><ref>[[:s:资治通鉴/卷205|《资治通鉴》卷二百零五]]</ref>不久转凤阁侍郎，仍居相位。<ref name=BT89/><ref name=NBT116/><ref>[[:s:旧唐书/卷6|《旧唐书》卷六]]</ref>麟台少监[[李峤|李峤]]为王方庆上表辞让，未果。<ref>[[:s:为王方庆让凤阁侍郎表|李峤《为王方庆让凤阁侍郎表》]]</ref>

[[神功|神功]]元年（697年），武则天伯父[[武士让|武士让]]之孙清边道大总管建安王[[武攸宜|武攸宜]]破[[契丹|契丹]]回朝献俘，按传统应由军乐队奏乐。因武攸宜回朝时，正值高宗去世当月，内史[[王及善|王及善]]请求备鼓吹但不奏乐。王方庆引用晋朝先例，称乐队只有当武攸宜回朝正值高宗忌日时才不奏乐，建议乐队仍然奏乐，武则天同意了。<ref>[[:s:宋史/卷123|《宋史》卷一百二十三]]</ref>大约同时，一次，武则天去万安山的玉泉祠，上山的道路很险峻，想乘腰舆上山。王方庆反对，称作为君主这样做太危险了，她同意了。同年，她升他为'''石泉县子'''。大约同时，还有每月一日于明堂行告朔之礼的仪式是否废止的争论。王方庆倡议维持，武则天同意了。<ref>[[:s:旧唐书/卷22|《旧唐书》卷二十二]]</ref><ref name=NBT199>[[:s:新唐书/卷199|《新唐书》卷一百九十九]]</ref><ref>[[:s:明堂告朔议|吴扬吾《明堂告朔议》]]</ref><ref>[[:s:读时令议|杜佑《读时令议》]]</ref>王方庆重视礼法，当时朝臣在有大功的大臣去世之后没有按礼法致哀，他于是上书反映此现象，武则天接纳其说并责备了不遵守礼法的官员。<ref name=BT89/>经过这些事件，在朝的官员们都很敬畏他。<ref name=NBT116/>

武则天曾向王方庆求取[[王羲之|王羲之]]的书法，王方庆称王家早已将王羲之的四十余份书法呈给[[唐太宗|唐太宗]]，只剩一轴了，但他愿意呈上十一世祖[[王导|王导]]、十世祖[[王洽_(东晋)|王洽]]、九世祖[[王珣|王珣]]、八世祖[[王昙首|王昙首]]、七世祖[[王僧绰|王僧绰]]、六世祖[[王俭_(南齐)|王俭]]、五世祖[[王骞|王骞]]、高祖[[王规|王规]]、曾祖王褒及九世从祖[[王献之|王献之]]等其它28人的书法。献上书法时，又由李峤作表。<ref>[[:s:为凤阁侍郎王方庆进书法表|李峤《为凤阁侍郎王方庆进书法表》]]</ref><ref>[[:s:为凤阁侍郎王方庆进南齐临轩图表|李峤《为凤阁侍郎王方庆进南齐临轩图表》]]</ref>武则天在武成殿将书法展示给大臣们，令中书舍人[[崔融_(初唐)|崔融]]作序，号为《宝章集》并重新赐给王方庆，<ref name=NBT57>[[:s:新唐书/卷57|《新唐书》卷五十七]]</ref>士人以为荣宠。<ref name=BT89/><ref name=NBT116/>

[[圣历|圣历]]元年（698年）二月，宰相[[狄仁杰|狄仁杰]]倡议召回流放中的庐陵王[[唐中宗|李哲]]（原唐中宗），王及善和王方庆也赞成。武则天近臣右肃政中丞[[吉顼|吉顼]]和男宠[[张易之|张易之]]、[[张昌宗|张昌宗]]也都倡议。而武则天又有意立侄梁王[[武三思|武三思]]为太子。王方庆有子为[[眉州|眉州]]司士参军，武则天说：“君在相位，为何儿子的任所那么远？”王方庆对答：“庐陵王是陛下爱子，尚在远处，臣之子怎敢相近？”以比[[战国|战国]]时舍人[[仓唐|仓唐]]说服[[魏文侯|魏文侯]]召还远在[[中山国|中山国]]的[[魏武侯|太子击]]之事。后来武则天又召狄仁杰说：“朕数次梦见下[[双陆棋|双陆棋]]不胜，为什么？”狄仁杰与王方庆异口同声说：“双陆不胜，是‘无子’。这是天意在警示陛下啊！且太子，天下之本，本一摇，天下危险了。[[李世民|文皇帝]]身蹈锋镝，勤劳而有天下，传之子孙。先帝寝疾，诏陛下[[监国|监国]]。陛下掩神器而取之十多年，又欲以武三思为继承人。且姑侄与母子哪个更亲？陛下立庐陵王，则千秋万岁后常享宗庙；如果武三思得立，宗庙不祔其姑母。”武则天最终召回李哲，不久立为太子。<ref>[[:s:新唐书/卷115|《新唐书》卷一百一十五]]</ref>八月，时为银青光禄大夫、行凤阁侍郎、同平章事、[[上柱国|上柱国]]、石泉县开国子的王方庆因身体健康恶化，请求闲职，武则天任他为麟台监，修国史，不再任相。<ref name=NBT4/><ref>[[:s:资治通鉴/卷206|《资治通鉴》卷二百零六]]</ref><ref>[[:s:授王方庆麟台监修国史制|李峤《授王方庆麟台监修国史制》]]</ref>武则天令侍御医[[张文仲|张文仲]]集当时名医共撰治疗风气的药方，令王方庆监其修撰。<ref>[[:s:旧唐书/卷191|《旧唐书》卷一百九十一]]</ref><ref>[[:s:新唐书/卷204|《新唐书》卷二百零四]]</ref>李哲被立为太子时，已改名[[李显|李显]]，武则天任王方庆为行太子左庶子。<ref>[[:s:授王方庆左庶子制|李峤《授王方庆左庶子制》]]</ref>二年（699年），武则天想在季冬讲武，有司办理不及，遂决定明年孟春再讲武。王方庆说：“按《[[月令|月令]]》：‘冬季，天子命将帅习武，练习射箭、驾战车，角力较量’。这是三个季度务农，一个季度练武，平安之日不忘战乱之时。春天不可以举兵。兵，当五行之金；金胜木。春天，正是木旺之时，举金会伤德，扼杀生气。孟春季节而干冬季干的事，将会发生水涝，雪霜也会使人不安，种子不能入土。如今孟春时练兵，以阴政触犯阳气，危害了生物的生机。臣担心涝水将会冲毁东西，霜雪会打坏庄稼，以致夏麦不熟。恳请陛下不要违背时令，还是提前到冬季练兵，以顺应自然规律。”武则天手诏褒允之。<ref>[[:s:答王方庆谏孟春讲武手制|:s:答王方庆谏孟春讲武手制]]</ref>同年武则天正授他为太子左庶子，进封石泉县公，俸禄仍视同三品宰相，命他兼侍太子读书。王方庆上奏：“人臣对于天子，没有指斥太子名字不当的。以前晋朝[[山涛|山涛]]《启事》称皇太子时不称其名；[[孝敬皇帝|孝敬皇帝]]为太子时，天下‘弘’字改为‘崇’字；[[章怀太子|沛王]]为太子时，天下‘贤’字改为‘文’字。现在东宫的门、殿名称很多涉嫌触讳，恳请一一改去，使能遵符旧典。”武则天下诏称可。武则天在三阳宫时，王方庆与御史大夫[[杨再思|杨再思]]为东都留守，引太学生[[徐坚|徐坚]]为判官，专委以表奏。王方庆善《三礼》之学，每有疑难，常问徐坚，徐坚总能引用旧说详明训释，<ref name=NBT199/><ref>[[:s:答王方庆问服制书|徐坚《答王方庆问服制书》]]</ref>王方庆深以为善，又赏识其文章典实，常称赞他说：“掌纶诰之选也。”<ref>[[:s:旧唐书/卷102|《旧唐书》卷一百零二]]</ref>王方庆卒于[[长安_(武周)|长安]]二年（702年）五月，赠[[衮州|衮州]]都督。<ref>《新唐书》本传作[[兖州|兖州]]都督。</ref>中宗复位后，因王方庆是旧臣，赠吏部尚书。<ref name=BT89/><ref name=NBT116/>

王方庆博学多才，熟悉朝章，著书二百多篇，特别精通《周礼》、《校礼》、《礼记》。学礼者向他咨询疑难，他一定回答得十分深刻精到，他的门生因而把他的答问辑录成《杂礼答问》一书。他家藏书极多，与皇家秘府不相上下，图画也都是精本。但诸子不能守业，他死后，藏书渐渐散佚。<ref name=BT89/><ref name=NBT116/>

[[唐德宗|唐德宗]]年间，王方庆得以续图[[凌烟阁|凌烟阁]]。<ref>[[:s:新唐书/卷191|《新唐书》卷一百九十一]]</ref>

== 评价 ==
*《旧唐书》史臣曰：王方庆干城南海，羽冀东宫，台阁枢机，无不功济，所谓君子不器者也。苟非文学，斯焉取斯。
*《旧唐书》赞曰：方庆之才，周旋特立。<ref name=BT89/>
*《新唐书》赞曰：[[李德裕|李德裕]]著书称：“方庆为相时，子为眉州司士参军。武后曰：‘君在相位，何子之远？’对曰：‘卢陵是陛下爱子，今尚在远，臣之子庸敢相近？’以比仓唐悟文侯事。”嗟乎，君子哉！虽造次不忘悟君於善。及建言不斥太子名，以动群臣，示中兴之渐，所谓人难言者，於方庆难乎哉！德裕之称，为不诬矣。<ref name=NBT116/>
