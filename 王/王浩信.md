'''王浩信'''（{{lang|en|'''Vincent Wong Ho Shun'''}}，{{bd|1983年|7月7日|}}），[[香港|香港]]男演員及歌手，以歌手身分出道，現為[[無綫電視藝員列表|無綫電視經理人合約]]藝員兼當紅小生之一，2017年憑《[[踩過界|踩過界]]》於《[[萬千星輝頒獎典禮|萬千星輝頒獎典禮]]》奪得「最佳男主角」及於 《[[星和無綫電視大獎2017|星和無綫電視大獎2017]]》奪得「我最愛TVB男主角」。

== 簡歷 ==
王浩信生於[[中國|中國]][[新疆維吾爾自治區|新疆維吾爾自治區]]，父親為[[漢族|漢族]]，母親為[[維吾爾族|維吾爾族]]，因此為漢維混血兒，<ref>{{cite web |author1=洛洛 |title=从兜巴信到视帝，他是TVB最能打的大长腿！ |url=http://www.sohu.com/a/239793482_228074 |website=搜狐}}</ref>童年時代隨父母遷往香港。<ref>{{cite web |author1=皑皑说娱乐 |title=TVB的当家小生王浩信，竟是地地道道的新疆人？ |url=https://baijiahao.baidu.com/s?id=1600779910837970193&wfr=spider&for=pc}}</ref>

王浩信加入[[娛樂圈|娛樂圈]]前為平面設計師，因一次唱[[卡啦OK|卡啦OK]]時給經理人發掘、經試音後而正式成為歌手。2005年簽約[[Neway|Neway]]集團旗下唱片公司[[Neway_Star|Neway Star]]，於2006年推出其首張個人專輯，約滿後加入[[電視廣播有限公司|無綫電視]]，主要向電視劇方面發展。2010年，在台慶劇《[[刑警_(無綫電視劇)|刑警]]》中憑其出色表現而人氣急升，遂榮獲《[[MY_AOD我的最愛頒獎典禮2011|MY AOD我的最愛頒獎典禮2011]]》頒發“我的最愛最具潛質男藝人名”獎項。2013年，他在《[[萬千星輝頒獎典禮2013|萬千星輝頒獎典禮2013]]》獲得「飛躍進步男藝員」。

現時為無綫電視力捧新晉小生之一。2015年，王浩信在《[[樓奴_(電視劇)|樓奴]]》首度擔任第一男主角，因劇中搞笑演出而令人氣再次上升，更首度被提名“最佳男主角”。
2016年，在超時空警匪劇《[[EU超時任務|EU超時任務]]》中飾演表面古惑內心正義的圍村村長「渠頭」獲得正評<ref>{{cite web|url=http://ol.mingpao.com/php/showbiz3.php?nodeid=1460448387242&subcate=latest&issue=20160412 |title=王浩信開心到喊 鍾意《EU超時任務》主題曲「無需有萬次只需一次愛」 |publisher=[[明報|明報]]OL網 |date=2016年4月12日}}</ref><ref>{{cite web|url=http://hk.on.cc/hk/bkn/cnt/entertainment/20160412/bkn-20160412164624999-0412_00862_001.html |title=網民激推做視帝　王浩信感動到喊 |publisher=[[ON.CC|東網]] |date=2016年4月12日}}</ref><ref>{{cite web|url=http://www.orientalsunday.hk/%e6%9c%80%e6%96%b0%e5%a8%9b%e8%81%9e/%e7%8e%8b%e6%b5%a9%e4%bf%a1-%e8%a2%81%e5%81%89%e8%b1%aa-%e3%80%8aeu%e3%80%8b%e6%99%92%e8%82%8c%e4%bb%bb%e5%8b%99/ |title=王浩信 袁偉豪 《EU》晒肌任務 |publisher=[[東方新地|東方新地]] |date=2016年3月26日}}</ref>，更被網民力捧為《[[萬千星輝頒獎典禮2016|萬千星輝頒獎典禮2016]]》「最佳男主角」和「最受歡迎電視男角色」得主。2016年2月，拍攝《[[致命復活|致命復活]]》時受傷<ref>{{cite web|url=http://hk.apple.nextmedia.com/entertainment/art/20160302/19512135 |title=左眼矇額頭腫　王浩信再入院 |publisher=[[蘋果日報_(香港)|蘋果日報]] |date=2016年3月2日}}</ref><ref>{{cite web|url=http://www.orientaldaily.on.cc/cnt/entertainment/20160312/00282_037.html |title=王浩信留疤：幾型呀 |publisher=[[東方日報_(香港)|東方日報]] |date=2016年3月12日}}</ref><ref>{{cite web|url=http://orientaldaily.on.cc/cnt/entertainment/20160408/00282_013.html |title=王浩信傷病交煎激鬥頂硬上|publisher=東方日報 |date=2016年4月8日}}</ref>。

雖然王浩信是以歌手身分出道，轉型之後仍經常在公開場合演唱，但自加入無綫之後卻未曾為無綫劇集唱過主題曲、片尾曲及插曲。直到2016年王浩信才為台慶劇《[[致命復活|致命復活]]》獻唱主題曲。2017年在《[[踩過界|踩過界]]》中作出突破，首次挑戰盲人角色，飾演盲人大律師文申俠（盲俠、Hope Man），同時繼《[[致命復活|致命復活]]》後再次唱劇集主題曲，並憑此劇首次獲得《[[星和無綫電視大獎2017|星和無綫電視大獎2017]]》「我最愛TVB男主角」，成為新加坡視帝。同時，王浩信亦獲得《[[萬千星輝頒獎典禮2017|萬千星輝頒獎典禮2017]]》「最佳男主角」，首次成為香港視帝。同年10月，開拍林志華新劇《[[兄弟_(無綫電視劇)|兄弟]]》飾演網絡教主張非凡。

2018年，拍攝新劇《[[解決師|解決師]]》，向武打動作挑戰，自言每天工作就是與人打架，亦慶幸拍攝以來沒有受傷太多<ref>[https://bka.mpweekly.com/focus/20180815-127280 【日日打架】 王浩信周圍搵人弱點]</ref>。

他近年成為無綫電視監製[[林志華|林志華]]的常用演員。

== 家庭生活 ==
王浩信與女友[[陳自瑤|陳自瑤]]於2011年11月11日上午11時舉行婚禮。2012年4月27日，妻子陳自瑤誕下一女<ref>{{cite web|url=http://ol.mingpao.com/php/showbiz3.php?nodeid=1458496876384&subcate=news&issue=20160321|title=王浩信抱恙與妻女隔離|publisher=[[明報|明報]]OL網|date=2016年3月21日}}</ref>。

2017年9月，王浩信接受訪問時透露遺傳了自己設計天分給5歲的女兒靖喬，小朋友喜歡繪畫，父女之間透過爸爸繪畫、囡囡油顏色作親子溝通。王浩信同時表示女兒遺傳了他很多，包括長腿，擔心她因此而多人追求<ref>[https://news.mingpao.com/pns/dailynews/web_tc/article/20170901/s00016/1504201940135 5歲囡囡靖喬遺傳長腿 王浩信擔心太靚多人追]，[[明報|明報]]，2017年9月1日</ref>。

== 趣聞 == 
王浩信近年參演的劇集，每逢他演出的[[古裝劇|古裝劇]]、時裝劇，他都被安排有脫衣服鏡頭，有這些鏡頭的劇集包括《[[換樂無窮|換樂無窮]]》第7集 ，第10集，第15集、《[[飛虎_(無綫電視劇)|飛虎]]》第2集、《[[好心作怪|好心作怪]]》、《[[法外風雲|法外風雲]]》第15集、《[[廉政行動2014|廉政行動2014]]》、《[[東坡家事|東坡家事]]》、《[[EU超時任務|EU超時任務]]》、《[[致命復活|致命復活]]》、《[[不懂撒嬌的女人|不懂撒嬌的女人]]》、《[[踩過界|踩過界]]》和《[[兄弟_(無綫電視劇)|兄弟]]》，堪稱「除衫王」<ref>[https://www.hk01.com/%E5%A8%9B%E6%A8%82/99760/%E7%8E%8B%E6%B5%A9%E4%BF%A1-%E8%B8%A9%E9%81%8E%E7%95%8C-%E5%8F%88%E8%A6%81%E5%89%9D%E8%A1%AB-%E8%82%8C%E7%94%B7%E9%83%BD%E9%99%A4%E5%88%B0%E7%81%B0%E6%99%92-%E5%8A%87%E6%9C%AC%E5%92%81%E5%AF%AB%E5%8F%AA%E4%BF%82%E8%B7%9F%E4%BD%8F%E5%81%9A 王浩信《踩過界》又要剝衫　肌男都除到灰晒：劇本咁寫只係跟住做]</ref>。
