'''王偉雄'''（[[英文|英文]]：'''Wai-hung Wong'''），香港[[分析哲學|分析哲學]]家，[[斯多葛主義|斯多葛主義]]者<ref name = "STOICISM">{{cite web |author1=王偉雄 |title=斯多葛提示|url =https://thestandnews.com/philosophy/%E6%96%AF%E5%A4%9A%E8%91%9B%E6%8F%90%E7%A4%BA/|agency=[[立場新聞|立場新聞]]|date=2018-05-02|accessdate=2019-02-24|}}</ref>，現《[[立場新聞|立場新聞]]》哲學版主編<ref>{{cite web |author1=立場科哲 |title=About|url =https://www.facebook.com/sn.thirdculturelab/posts/616684665418102|agency=[[Facebook|Facebook]]|date=2019-02-01|accessdate=2019-02-24|}}</ref>，[[加州州立大學奇科分校|加州州立大學奇科分校]]哲學教授<ref>{{cite web |author1=王偉雄 |title=王偉雄|url =https://thestandnews.com/author/%E7%8E%8B%E5%81%89%E9%9B%84/|agency=[[立場新聞|立場新聞]]|accessdate=2019-02-24|}}</ref><ref name = "fb">{{cite web |author1=王偉雄 |title=About|url =https://www.facebook.com/waihung.wong.666/about?lst=100000701305331%3A1048695481%3A1550982086|agency=[[Facebook|Facebook]]|accessdate=2019-02-24|}}</ref>，曾任職於[[維珍尼亞聯邦大學|維珍尼亞聯邦大學]]<ref name = "fb"/>。

==生平==
王偉雄在香港出生和成長，大學本科主修中國語文及文學，於香港大學讀碩士時，方轉讀哲學，後於美國[[柏克萊加州大學|柏克萊加州大學]]完成哲學博士學位。<ref>{{cite web |author1=王偉雄|title=克服亨利 · 詹姆斯|url =https://thestandnews.com/culture/%E5%85%8B%E6%9C%8D%E4%BA%A8%E5%88%A9-%E8%A9%B9%E5%A7%86%E6%96%AF/|agency=[[立場新聞|立場新聞]]|date=2019-02-21|accessdate=2019-02-24|}}</ref>至今在美國生活了二十多年。<ref name = "Hoeng Gong"/>

王偉雄廣泛介入香港議題，與眾多香港名筆有過論戰，包括[[陳雲|陳雲]]、[[陶傑|陶傑]]、[[屈穎妍|屈穎妍]]、[[古德明|古德明]]等等。
==思想==
===倫理學===
王偉雄厭惡[[康德|康德]]之[[義務倫理學|義務論]]<ref>{{cite web |author1=王偉雄|title=真的很討厭康德的道德哲學|url =https://www.facebook.com/waihung.wong.666/posts/10215590557650880|agency=[[Facebook|Facebook]]|date=2018-10-05|accessdate=2019-02-24|}}</ref>，認為義務論無人情味又過份強調[[理性|理性]]<ref>{{cite web |author1=王偉雄|title=康德道德哲學|url =https://www.facebook.com/waihung.wong.666/posts/10213280619663874|agency=[[Facebook|Facebook]]|date=2017-12-01|accessdate=2019-02-24|}}</ref>。王偉雄是[[斯多葛主義|斯多葛主義]]者，他列出數條命題：「認清事情，判別哪些方面是自己能控制的、哪些方面在自己控制之外，不多想後者」、「沒有任何事物可以天長地久，一切皆如過眼雲煙」、「成事不盡在人，順逆無跡可尋」、「不順意甚至不幸的事隨時會發生」、「在情緒即將翻騰時立刻停止思想和行動，深深呼吸三下，然後反覆思考上述四個命題」、「睡前反省當天的思想和言行」，主張任何事物，如果令我們心境不平靜，那不是由於事物本身，而是由於我們對事物的看法，又主張嫉妒別人的成就或所得，以致心有不甘，這心理痛苦之源頭盡在自己，可說是自討苦吃，並且認為斯多葛式思想習慣的練習方法特別能對治嫉妒之苦。<ref name = "STOICISM"/>王偉雄認為，斯多葛主義屬於[[德行倫理學|德行倫理學]]。<ref>{{cite web |author1=王偉雄|title=一切都不在乎|url =https://hk.lifestyle.appledaily.com/lifestyle/columnist/%E7%8E%8B%E5%81%89%E9%9B%84/daily/article/20180210/20300600|agency=[[蘋果日報_(香港)|蘋果日報]]|date=2018-02-10|accessdate=2019-02-24|}}</ref>
===知識論===
王偉雄，批評有若干人是「天真的理性主義者」，粗疏瞭解理性，高估自己的理性程度。王偉雄以歸納法為例子證明，理性思考裏有些成份是沒有充份理據支持的：「歸納法到目前為止都很可靠，因此往後也可靠」，所以支持歸納法的理由只是使用歸納法來支持，而沒有充份理據支持歸納法到歸納法。王偉雄繼而諷刺「天真的理性主義者」：「如果你認為理性思考不應該有任何成份，沒有充份理據支持，你除了應該停止使用歸納法，你的所有理據都應該有理據支持，而這些理據的理據也應該有理據支持，如此類推。」、「問題是，你真的能夠對所有理據提出理據嗎？如果做不到，就是不理性嗎？」<ref>{{cite web |author1=王偉雄|title=天真的理性主義者|url =https://thestandnews.com/cosmos/%E5%A4%A9%E7%9C%9F%E7%9A%84%E7%90%86%E6%80%A7%E4%B8%BB%E7%BE%A9%E8%80%85/|agency=[[立場新聞|立場新聞]]|date=2015-05-16|accessdate=2019-02-24|}}</ref>

===尼采===
王偉雄認為，[[尼采|尼采]]《[[查拉圖斯特拉如是說|查拉圖斯特拉如是說]]》能讓人全新或更深刻領會人生和世事：

<blockquote>「自有人類以來，人們自尋歡樂的事太少；單單這一點，我的兄弟們，那是我們的原罪！如果我們學會了更好地自尋歡樂，就最能使我們忘掉使他人受苦和想出折磨他人的詭計。」([[尼采|尼采]]《[[查拉圖斯特拉如是說|查拉圖斯特拉如是說]]》．第二部：〈同情者〉)</blockquote>

王偉雄認為，尼采是認為自得其樂和折磨他人有對立關係：越是要「使他人受苦和想出折磨他人的詭計」，就越難自得其樂。王偉雄從中得到一個觀點：我們可以「遺忘使他人受苦和想出折磨他人的詭計」，而享受自得其樂的時刻，例如：當打算報復某人在Facebook的惡意留言，不如即刻去沖一杯香濃咖啡，邊飲邊細心閱讀尼采《查拉圖斯特拉如是說》。<ref>{{cite web |author1=王偉雄|title=自得其樂和折磨他人|url =https://thestandnews.com/personal/%E8%87%AA%E5%BE%97%E5%85%B6%E6%A8%82%E5%92%8C%E6%8A%98%E7%A3%A8%E4%BB%96%E4%BA%BA/|agency=[[立場新聞|立場新聞]]|date=2016-02-06|accessdate=2019-02-24|}}</ref>

===文學===
關於如何設定特定讀者，並且驅趕特定讀者以外之人，王偉雄以[[尼采|尼采]]為例子說明。王偉雄認為，尼采一早設定了自己特定之讀者，而不關心特定讀者以外之人，怎樣看待他的著作，是否讀得懂，會不會誤解等。王偉雄指出，尼采之寫作風格可以說是故意為難「特定讀者以外之人」，要令他們望而生厭或知難而退，尼采著作裏有很多極長的段落令不少人讀得辛苦，就是一個例子。王偉雄認為，尼采是寫作天才，十二、十三歲時寫的文章已很成熟得體，又認為尼采是注重風格和有強烈自我意識。<ref>{{cite web |author1=王偉雄|title=故意為難讀者？|url =https://thestandnews.com/culture/%E6%95%85%E6%84%8F%E7%82%BA%E9%9B%A3%E8%AE%80%E8%80%85/|agency=[[立場新聞|立場新聞]]|date=2016-06-26|accessdate=2019-02-24|}}</ref>
