'''王银成'''（{{bd|1960年||}}），[[山西|山西]][[临猗|临猗]]人，高级[[会计师|会计师]]。中华人民共和国政治人物。

== 生平 ==
王银成取得[[中南财经政法大学|中南财经政法大学]]经济学博士学位。1982年进入[[中国人民保险公司|中国人民保险公司]]，历任公司计财部副总经理、[[深圳市|深圳市]]分公司总经理、公司总经理助理、公司副总裁、总裁兼副董事长。2017年2月23日涉嫌严重违纪，正接受组织审查。<ref >{{cite web|url=http://news.xinhuanet.com/2017-02/23/c_1120518998.htm | language= | title=中国人保总裁王银成涉嫌严重违纪接受组织审查 |publisher=新华社 |date=2017-02-23 |accessdate=2017-02-23 }}</ref><ref >{{cite web|url=http://www.ce.cn/xwzx/gnsz/gdxw/201702/23/t20170223_20457829.shtml | language= | title=中国人保集团总裁王银成被审查(简历) |publisher= 中国经济网|date=2017年2月23日 |accessdate=2017-02-23 }}</ref>2017年7月5日被[[双开|双开]]，移送司法机关<ref>{{cite web |url = http://www.ccdi.gov.cn/jlsc/zggb/djcf_zggb/201707/t20170705_102226.html |title = 中国人民保险集团股份有限公司原党委副书记、副董事长、总裁王银成严重违纪被开除党籍和公职 |accessdate = 2017年7月5日 |publisher = 中央纪委监察部网站}}</ref>。

2018年5月24日，[[福建省福州市中级人民法院|福建省福州市中级人民法院]]一审宣判，对被告人王银成以受贿罪判处有期徒刑11年，并处罚金人民币100万元；对王银成受贿所得财物及其孳息予以追缴，上缴国库<ref>{{cite web |url = http://finance.sina.com.cn/money/insurance/bxyx/2018-05-24/doc-ihaysvix7852953.shtml|title = 人保原总裁王银成受贿870余万 一审被判11年罚100万 |accessdate = 2018年5月24日 |publisher = 新浪财经}}</ref>。
