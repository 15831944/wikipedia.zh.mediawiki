'''王下七武海'''是[[日本漫畫|日本漫畫]]《[[ONE_PIECE|ONE PIECE]]》裡虛構的稱號，簡稱「'''七武海'''」，是被[[ONE_PIECE角色列表#世界政府|世界政府]]公認並具有特權的七大海賊。

== 組織簡介 ==
「王下七武海」是《[[ONE_PIECE|ONE PIECE]]》世界的偉大航道「三大勢力」之一，是七名實力超凡的海賊，但這七人並不屬於同一個組織；他們在世界政府認許下可自由搶奪未開發的土地及其他海賊<ref name="ONEPIECE漫畫_069">《ONE PIECE》漫畫第69話。</ref>。在一般海賊眼中是「世界政府的走狗」，政府會憑藉他們「實力超凡」和「知名度」這兩項條件對這世上的所有海賊形成威慑；他們亦須響應政府的徵召，為其參與戰爭，雙方擁有足以震撼世界的合作關係，彼此之間必須互利共生。

「王下七武海」的名號和角色是由約瑟夫在「惡龍篇」首次提及。

=== 特權 ===
在海賊成為「王下七武海」的同時，他們原本被懸賞的[[通緝令|通緝令]]會被停止，而世界政府亦不會再追究他們之前所犯的一切罪行。此外，他們的[[私掠行為|私掠行為]]亦獲世界政府默許，但卻要把一定數量的財物收成交給世界政府<ref name="ONEPIECE漫畫_069"/>。
此外，世界政府為保持「王下七武海」的「合作關係」，亦會答允他們的「個別要求」，例如允許[[波雅·漢考克|波雅·漢考克]]不讓海軍船隻駛近[[亞馬遜百合_(ONE_PIECE)|亞馬遜百合]]範圍、答應[[吉貝爾|吉貝爾]]恩赦「魚人海賊團」所有船員之前犯下的罪行等<ref name="ONEPIECE漫畫_624">《ONE PIECE》漫畫第624話。</ref>、[[唐吉訶德·多佛朗明哥|唐吉訶德·多佛朗明哥]]則藉著七武海的權力拓展旗下事業甚至是進行非法交易。

=== 選拔和離位措施 ===
世界政府會依據該名海賊的作為、實力、影響力，徵招對方並賦予王下七武海的職位，例如世界政府曾為了促進種族和諧，邀請吉貝爾成為七武海成員；[[馬歇爾·D·汀奇|馬歇爾·D·汀奇]]藉著擊敗「火拳」[[波特卡斯·D·艾斯|波特卡斯·D·艾斯]]並送入[[推進城|推進城]]，以換取遞補七武海的資格；[[托拉法爾加·羅|托拉法爾加·羅]]向世界政府獻上100顆海賊的心臟，而被授予七武海的稱號。

倘若七武海成員自行放棄名號（如「海俠」[[吉貝爾|吉貝爾]]放棄名號後，懸賞金額超過四億）、犯下的罪行過重、和其他海賊團結盟、或被世界政府認定無留任價值而被褫奪職位，將恢復追緝該名海賊，更嚴重甚至是下令抹殺。如[[克洛克達爾|克洛克達爾]]（已逃獄）和[[唐吉訶德·多佛朗明哥|唐吉訶德·多佛朗明哥]]涉及諸多爭議而在其勢力垮臺後被送入推進城、[[月光·摩利亞|月光·摩利亞]]被政府高層判定實力太弱而下令抹殺、[[托拉法爾加·D·瓦特爾·羅|托拉法爾加·D·瓦特爾·羅]]因為和[[魯夫|魯夫]]等人結成海賊同盟曝光遭到除名並由海軍元帥盃下令要求大將藤虎進一步追緝。

== 角色介紹 ==
=== 現任成員 ===
; {{nihongo|[[喬拉可爾·密佛格|喬拉可爾·密佛格]]|ジュラキュール・ミホーク|}}
: 聲優：[[青野武|青野武]]／[[掛川裕彥|掛川裕彥]]（日本）；[[孫中台|孫中台]]／[[符爽|符爽]]（台灣）；[[黎家希|黎家希]]／[[蔡忠衛|蔡忠衛]]（香港）
: 外號「鷹眼」、「世界第一劍豪」，原懸賞金：不明，擁有世界最強黑刀的「夜」。

; {{nihongo|[[革命軍_(ONE_PIECE)#巴索羅繆·大熊|巴索羅繆·大熊]]|バーソロミュー・くま|}}
: 聲優：[[堀秀行|堀秀行]]（日本）；[[孫中台|孫中台]]（台灣）；[[何偉誠|何偉誠]]（香港）
: 外號「暴君」，原懸賞金：2億9,600萬貝里。擁有多重身分，表面上是加盟國國王及海賊，私底下是[[革命軍_(ONE_PIECE)|革命軍]]的幹部。曾經幫助過[[草帽海賊團|草帽海賊團]]，在頂點戰爭前自願被Dr.貝卡帕庫完全改造而失去自我意識。

; {{nihongo|[[波雅·漢考克|波雅·漢考克]]|ボア・ハンコック|}}
: 聲優：[[三石琴乃|三石琴乃]]（日本）；[[錢欣郁|錢欣郁]]（台灣）；[[鄭家蕙|鄭家蕙]]（香港）
: 外號「海賊女帝」、「蛇姬」，原懸賞金：8,000萬貝里。是亞馬遜百合的皇帝和九蛇海賊團的船長，每次遠征會搶奪大量財寶及石化大量海賊。

; {{nihongo|[[巴其_(ONE_PIECE)|巴其]]|バギー|}}
: 聲優：[[千葉繁|千葉繁]]（日本）；[[符爽|符爽]]／[[孫中台|孫中台]]（台灣）；[[馮志輝|馮志輝]]／[[蔡忠衛|蔡忠衛]]（香港）
: 外號「千兩小丑」，原懸賞金：1,500萬貝里。在頂點戰爭結束不久，因世界政府對其在推進城收了大量推進城囚犯作手下有所忌憚，因而邀請他遞補七武海，之後藉職務之便成為海賊派遣公司的團長。

; {{Anchor|艾德華·衛伯|衛伯}}{{nihongo|艾德華·衛伯|エドワード·ウィーブル|}}
: 聲優：[[鹽屋浩三|鹽屋浩三]]（日本）；[[符爽|符爽]]（台灣）
: 自稱「白鬍子二世」，使用和[[白鬍子海賊團#艾德華·紐蓋特|白鬍子]]一樣的大刀，原懸賞金額：4億8,000萬貝里。生日12月10日。兩年後已擊敗16個白鬍子旗下海賊團的船長，被黃猿形容單論力量就像是年經時的白鬍子一樣。
: 疑似是九年前重創「黑腕」捷風及其學生的海賊，一年前成為七武海，導致捷風另立新海軍。<ref>《ONE PIECE》第千卷。</ref>
: 約30、40年前其母親[[ONE_PIECE角色列表#Miss.芭晴|芭晴女士]]在當海賊時曾與白鬍子同船，頂點之戰後的兩年間自稱「白鬍子愛人」，由於認為白鬍子留有巨大的遺產，因此帶著其子盯上了白鬍子海賊團餘黨。<ref>《ONE PIECE》第802話。</ref>

=== 離任成員 ===
; {{nihongo|[[克洛克達爾|克洛克達爾]]|クロコダイル|}}
: 聲優：[[大友龍三郎|大友龍三郎]]（日本）；[[宋克軍|宋克軍]]（台灣）；[[蔡忠衛|蔡忠衛]]／[[李家輝|李家輝]]（香港）
: 外號「沙」、「Mr.0」，原懸賞金額：8,100萬貝里，現懸賞金額：不明。在阿拉巴斯坦事件被魯夫打敗，後被世界政府授權海軍褫奪其稱號並逮捕至推進城，後由黑鬍子遞補其空缺。

; {{nihongo|[[月光·摩利亞|月光·摩利亞]]|ゲッコー・モリア|}}
: 聲優：[[寶龜克壽|寶龜克壽]]（日本）；[[孫中台|孫中台]]（台灣）；[[蔡忠衛|蔡忠衛]]（香港）
: 現懸賞金額：3億2,000萬貝里。於恐怖三桅帆船篇中被魯夫打敗，在頂點戰爭後因世界政府的高層認定其實力不足以擔當七武海，因此褫奪其稱號並命令多佛朗明哥將他抹殺，但被其部下阿布薩羅姆所救。

; {{nihongo|[[唐吉訶德·多佛朗明哥|唐吉訶德·多佛朗明哥]]|ドンキホーテ・ドフラミンゴ|}}
: 聲優：[[田中秀幸|田中秀幸]]（日本）；[[宋克軍|宋克軍]]／[[孫中台|孫中台]]（台灣）；[[李家傑_(配音員)|李家傑]]（香港）
: 外號「天夜叉」、「JOKER」，原懸賞金額：3億4,000萬貝里。前任[[多雷斯羅薩|多雷斯羅薩]]王國國王，於多雷斯羅薩事件中被魯夫打敗，後由上將藤虎為首的海軍褫奪其稱號並逮捕至推進城。

; {{nihongo|[[吉貝爾|吉貝爾]]|ジンベエ|}}
: 聲優：[[鄉里大輔|鄉里大輔]]／[[寶龜克壽|寶龜克壽]]（日本）；[[宋克軍|宋克軍]]（台灣）；[[譚漢華|譚漢華]]（香港）
: 外號「海俠」原懸賞金額：2億5000萬貝里，現懸賞金額：4億3,800萬貝里。在頂點戰爭前夕因不想與白鬍子為敵，拒絕世界政府的徵召而遭逮捕至推進城，後於頂點戰爭期間自行放棄其稱號。

; {{nihongo|[[馬歇爾·D·汀奇|馬歇爾·D·汀奇]]|マーシャル・D・ティーチ|}}
: 聲優：[[大塚明夫|大塚明夫]]（日本）；[[孫中台|孫中台]]／[[符爽|符爽]]（台灣）；[[譚漢華|譚漢華]]（香港）
: 外號「黑鬍子」，原懸賞金額：0，現懸賞金額：22億4,760萬貝里。藉由生擒艾斯的功績自薦成為七武海，後於頂點戰爭期間自行放棄其稱號，戰後取代[[艾德華·紐蓋特|艾德華·紐蓋特]]成為新任[[四皇|四皇]]。

; {{nihongo|[[托拉法爾加·羅|托拉法爾加·D·瓦特爾·羅]]|トラファルガー·D·ワーテル·ロー|}}
: 聲優：[[神谷浩史|神谷浩史]]（日本）；[[宋克軍|宋克軍]]（台灣）；[[蔡忠衛|蔡忠衛]]（香港）
: 外號「死亡外科醫生」，原懸賞金額：4億4000萬貝里，現懸賞金額：5億貝里。在頂點戰爭後的兩年間送給世界政府100顆海賊的心臟而被授予此稱號，在與草帽海賊團結盟的消息走漏風聲後，由上將藤虎宣布褫奪其稱號。
