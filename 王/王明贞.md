'''王明贞'''（{{bd|1906年|11月18日|2010年|8月28日}}），生于[[江苏|江苏]][[苏州|苏州]]，[[中国|中国]][[物理学家|物理学家]]，[[清华大学|清华大学]]第一位女性教授<ref>{{cite web |title=清华师生深切缅怀第一位女教授王明贞 |url=http://news.tsinghua.edu.cn/publish/thunews/10303/2011/20110225232533796749789/20110225232533796749789_.html |website=news.tsinghua.edu.cn |accessdate=2019-02-23}}</ref>。是[[乔治·乌伦贝克|乔治·乌伦贝克]]的学生。毕业于[[燕京大学|燕京大学]]、[[金陵女子大学|金陵女子大学]]、[[密歇根大学|密歇根大学]]，毕业后曾在[[麻省理工学院|麻省理工学院]]工作，此后在[[云南大学|云南大学]]和[[圣母大学|圣母大学]]任教。1955年回国并成为清华大学教授。文革时曾关押于[[秦城监狱|秦城监狱]]。1976年退休<ref>{{cite web |title=“中国居里夫人”的传奇人生 物理学家王明贞(组图) |url=http://edu.163.com/10/1105/15/6KO3Q6RK00293L7F.html |website=edu.163.com |accessdate=2019-02-23}}</ref>。2010年于北京病逝，享年104岁<ref>{{cite web |title=世纪人生路 清华不了情——追记清华大学女教授王明贞 |url=http://news.tsinghua.edu.cn/publish/thunews/10303/2011/20110225232549812220424/20110225232549812220424_.html |website=news.tsinghua.edu.cn |accessdate=2019-02-23}}</ref>。

==家族==
王明贞是[[王颂蔚|王颂蔚]]的孙子，[[王季同|王季同]]的儿子。丈夫[[俞启忠|俞启忠]]是[[俞明震|俞明震]]的孙子，[[俞启威|俞启威]]（又名黄敬）的弟弟。
