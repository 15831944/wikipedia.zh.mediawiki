'''王贵德'''（{{bd|1914年|6月|2017年|4月7日}}），[[福建省|福建省]][[上杭县|上杭县]][[太拔乡|太拔乡]]大地村人，[[中国人民解放军开国少将|中国人民解放军开国少将]]<ref name=shishi>{{Cite web |url =http://news.xinhuanet.com/2017-05/11/c_1120955213.htm  |title =王贵德同志逝世  |author =  |date = 2017-05-11 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。

== 生平 ==
1930年加入[[中国共产主义青年团|中国共产主义青年团]]，1931年参加[[中国工农红军|中国工农红军]]，1932年加入[[中国共产党|中国共产党]]。[[第一次国共内战|第一次国共内战]]时期，历任上杭县独立团一连战士，[[红十二军|红十二军]]第三十四师第一〇〇团班长、排长，1931年底进入福建军区随营学校学习，任政治连排长，同年8月任[[长汀县|长汀县]]独立团一连指导员。1933年初任红十二军第三十四师第一〇〇团九连政委、团政治处党总支部书记。1934年进入[[红军大学|红军大学]]政治队学习，毕业后任[[红八军团|红八军团]]第二十一师第六十二团政委。后任[[红五军团|红五军团]]第十三师三十九团政治处主任。参加了[[中央苏区|中央苏区]]第二至五次反“围剿”、[[湘江战役|湘江战役]]、会宁战斗、田水铺战斗、[[山城堡战役|山城堡战役]]等战役战斗，参加了[[二万五千里长征|二万五千里长征]]。1935年7月[[红一方面军|红一方面军]]、[[红四方面军|红四方面军]]会合后，调任红四方面军[[红三十一军|红三十一军]]第九十一师第二七三团政委<ref name=shishi/>。

[[抗日战争|抗日战争]]时期，1937年8月任[[八路军|八路军]][[129师|一二九师]]第三八六旅第七七一团政治处教育股长，1938年1月任该团政治处主任，1938年8月任该团政委，后任[[冀南军区|冀南军区]]新编第九旅政治部主任，冀南军区第二军分区政委，冀南军区第二、第四、第三军分区副政委等职，参加了[[广阳战斗|广阳战斗]]、[[神头岭战斗|神头岭战斗]]、[[响堂铺战斗|响堂铺战斗]]、反“九路围攻”、[[百团大战|百团大战]]、[[南乐战役|南乐战役]]、占磁、[[邯郸战役|邯郸战役]]等战役战斗<ref>{{cite news|author=乔天富撰文 本报记者　乔天富　摄影|title=老红军——王贵德|url=http://chn.chinamil.com.cn/2012yinxiang85/2012-05/17/content_4864487.htm|accessdate=2012-05-17|newspaper=解放军报社|deadurl=yes|archiveurl=https://web.archive.org/web/20130517033810/http://chn.chinamil.com.cn/2012yinxiang85/2012-05/17/content_4864487.htm|archivedate=2013-05-17}}</ref><ref name=shishi/>。

[[第二次国共内战|第二次国共内战]]时期，历任冀南军区第二军分区副政治委员、[[华北野战军|华北野战军]]第十三纵队三十九旅副政治委员、三十八旅政治委员，1948年8月任[[中国人民解放军第一野战军|中国人民解放军第一野战军]]第十八兵团第六十一军第一八二师政治委员。参加了[[临汾战役|临汾战役]]、[[晋中战役|晋中战役]]、[[太原战役|太原战役]]等战役战斗<ref name=shishi/><ref name=minxi/>。

[[中华人民共和国|中华人民共和国]]成立后，历任[[中国人民解放军第六十一军|中国人民解放军第六十一军]]政治部主任、[[川北军区|川北军区]]副政治委员、[[中国人民解放军陆军第十四集团军|中国人民解放军第十四军]]副政治委员，[[贵州省军区|贵州军区]]副政治委员、第一副政治委员、政治委员<ref name=shishi/><ref name=minxi>{{cite news|title=王贵德|url=http://www.mxrb.cn/hxkjw/content/2013-12/16/content_1286190.htm|date=2013-12-16|newspaper=闽西新闻网}}</ref>。1964年任[[中国人民解放军铁道兵|中国人民解放军铁道兵]]党委常委、副政治委员兼政治部主任，参与了[[成昆铁路|成昆铁路]]、[[京原铁路|京原铁路]]、[[青藏铁路|青藏铁路]]、[[南疆铁路|南疆铁路]]、[[北京地铁|北京地铁]]等国家重点工程建设，后任铁道兵顾问<ref name=shishi/><ref>{{cite news|title=王贵德|url=http://www.fjlyta.com/mxrw/jjp/201002/t20100205_129758.htm|date=2010-02-05|newspaper=福建省龙岩市旅游局|deadurl=yes|archiveurl=https://web.archive.org/web/20140223203033/http://www.fjlyta.com/mxrw/jjp/201002/t20100205_129758.htm|archivedate=2014-02-23}}</ref>。

王贵德是[[中国共产党第八次全国代表大会|中国共产党第八次全国代表大会]]代表、[[第五届全国政协委员|第五届全国政协委员]]。1955年被授予[[中国人民解放军少将|少将]]军衔<ref name=shishi/><ref>{{cite news|title=揭秘：中共首批开国将军的级别是怎样划分的？ |url=http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |accessdate=2012-06-15 |newspaper=凤凰网（转自摘自《党史博览》2010年第5期） |date=2010-06-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20120620091504/http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |archivedate=2012-06-20 }}</ref>。曾获二级[[八一勋章_(1955年)|八一勋章]]、一级[[独立自由勋章|独立自由勋章]]、一级[[解放勋章|解放勋章]]和一级[[红星功勋荣誉章|红星功勋荣誉章]]<ref name=shishi/>。

2017年4月7日，王贵德在[[北京|北京]]病逝，享年103岁<ref name=shishi/>。
