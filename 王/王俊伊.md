'''王俊伊'''（英语：'''Joyee Wong'''，{{bd|1991年|9月18日||}}），毕业于武汉科技大学2010级会计系，中国内地影视男演员、歌手。<ref>{{Cite web|url=http://people.mtime.com/2164636/details.html|title=王俊伊 个人档案 – Mtime时光网|accessdate=2016-10-10|work=people.mtime.com}}</ref>2013年，在香港旅游时被星探发掘从而进入香港演艺圈。2014年，主演首部电影《巴黎之情海星空》正式出道，并凭借此片获得第33届香港电影金像奖最佳新人奖提名。 2015年，在古装神话剧《无双谱》中饰演小皇子而受到观众熟知。<ref>{{Cite web|url=http://hn.china.com/news/today/11145527/20151021/20601838.html|title=《无双谱》__中华网河南频道|accessdate=2016-10-12|last=ck16917|first=F7 13456,|work=hn.china.com}}</ref> 同年，参演电影《澳门风云3》因合约问题戏份被删。音乐方面，王俊伊是全球首位在香港红馆开个唱的90后男歌手。2016年4月，王俊伊在香港举办人生第一场个人演唱会A.I.N.Y。2017年，王俊伊返回中国内地发展。2017年4月，王俊伊于北京成立影视工作室独立打理个人演艺事务。2017年8月，主演新版《神雕侠侣》饰杨过。2018年，搭档郑爽主演偶像剧《樱花季节遇见你》。同年，主演宫廷剧《杨贵妃传》饰唐玄宗李隆基。时尚方面，王俊伊是迪奥男装大中华区形象代言人。

== 演艺经历 ==
2013年，王俊伊主演处女作电影《巴黎之情海星空》<ref>{{Cite web|url=http://movie.mtime.com/230686/fullcredits.html|title=巴黎之情海星空 演职员表 – Mtime时光网|accessdate=2016-10-10|work=movie.mtime.com}}</ref>

2014年，王俊伊参演TVB古装神话剧《[[無雙譜_(2015年電視劇)|无双谱]]》。<ref>{{Cite web|url=http://fun.youth.cn/2015/1202/3006803.shtml|title=港剧无双谱电视剧全集剧情介绍1-20集大结局演员表_娱乐24小时_娱乐频道__中国青年网|accessdate=2016-10-10|last=青网娱乐合作张辉|work=fun.youth.cn|archive-url=https://web.archive.org/web/20161009174813/http://fun.youth.cn/2015/1202/3006803.shtml|archive-date=2016-10-09|dead-url=yes}}</ref>

2015年8月，王俊伊出演王晶刘伟强联合执导的贺岁片电影《[[澳门风云3|澳门风云3]]》后因合约问题戏份被删。<ref>{{Cite web|url=http://star.pclady.com.cn/147/1477855.html|title=澳门风云3到哪天下映播到3月【图】_综艺库_明星_太平洋时尚网|accessdate=2016-10-18|last=chenchufei_gz|first=wuzhiping_gz|work=star.pclady.com.cn}}</ref>

2016年7月，王俊伊被爆出演钟汉良执导的处女作电影《沙漏》。<ref>{{Cite web|url=http://bj.people.com.cn/n2/2016/0706/c233082-28620439-4.html|title=《微微一笑很倾城》后郑爽杨洋不分开 新作《沙漏》导演竟然是他--北京频道--人民网|accessdate=2016-10-10|last=L_104278|work=bj.people.com.cn}}</ref>

2016年4月—10月，王俊伊举办个人首场A.I.N.Y世界巡回演唱会。

== 演唱会 ==
2016年4月，王俊伊演唱会团队启动A.I.N.Y世界巡回演唱会。演唱会有香港站、澳门站、台北站、马来西亚站、新加坡站、温哥华站、多伦多站、旧金山站、首尔站、东京站。

== 感情生活 ==
2015年12月2日，有狗仔在香港国际机场拍到王俊伊与一女子十指相扣一同出机场闸口，行为举止相当甜蜜，地下恋情曝光。据悉，该女子是独生女，籍贯武汉，家境相当富裕，平时兼职模特。

== 公益活动 ==
2015年12月5日，王俊伊出席由香港东华三院和香港电视广播有限公司(TVB)联合主办的大型慈善晚会“欢乐满东华2015”，并参与募捐。晚会所筹善款将用于东华三院的免费医疗服务和扩展医疗、教育及社会福利服务。<ref>{{Cite web|url=http://finance.chinanews.com/ga/2015/12-08/7661308.shtml|title=“欢乐满东华2015”在港举行 香港各界助力慈善发展-中新网|accessdate=2016-10-10|last=chinanews|work=finance.chinanews.com}}</ref>

2016年4月10日，携手范冰冰李小双出席首届武汉马拉松明星助跑公益活动。<ref>{{Cite web|url=http://hb.qq.com/a/20160410/007568.htm|title=武汉马拉松鸣枪起跑 李小双等明星选手领衔_大楚网_腾讯网|accessdate=2016-10-18|work=hb.qq.com}}</ref>
