'''王乙康''' （{{lang-en|Ong Ye Kung}}，{{bd|1969年|11月15日||}}<ref>{{cite web|title=關於王乙康|url=https://www.facebook.com/ongyekung/info?tab=page_info|website=facebook|accessdate=2018-06-22}}</ref>），[[新加坡華人|新加坡華人]]、政治人物，人民行動黨籍，他是現任[[新加坡國會|新加坡國會]]議員、[[新加坡教育部|新加坡教育部]]部長。他在[[2015年新加坡大選|2015年新加坡大選]]時代表人民行動黨當選[[國會議員|國會議員]]<ref name=參選>{{cite web |language=en|url=https://www.channelnewsasia.com/news/singapore/ong-ye-kung-amrin-amin-join-pap-s-sembawang-grc-team-8239876|title= Ong Ye Kung, Amrin Amin join PAP's Sembawang GRC team|publisher=[[亞洲新聞台|亞洲新聞台]]|date=2015-08-14|accessdate=2018-06-22}}</ref><ref name=勝選>{{cite web|language=en|url=https://www.channelnewsasia.com/news/singapore/-good-result-for-pap-but-an-excellent-result-for-singapore--pm-l-8236652|title=
‘Good result for PAP but an excellent result for Singapore’: PM Lee|publisher=[[亞洲新聞台|亞洲新聞台]]|date=2015-09-12|accessdate=2018-06-22}}</ref>。2016年他曾任新加坡教育部雙部長之一（高等教育及技能）兼[[新加坡國防部|新加坡國防部]]第二部長<ref name=2016教育部長>{{cite web |language=zh-hans|url=https://www.zaobao.com.sg/realtime/singapore/story20161028-683453|title=黄志明和王乙康升任正部长|publisher=[[聯合早報|聯合早報]] |date=2016-10-28|accessdate=2018-06-22}}</ref>。2018年內閣改組後，卸下國防部第二部長的職務，僅擔任合二為一的教育部部長<ref name="2018內閣">{{cite web |language=zh-hans|url=https://www.zaobao.com.sg/realtime/singapore/story20180424-853440|title=【内阁改组】陈振声出掌贸工部 王瑞杰和王乙康留任财政部和教育部|publisher=[[聯合早報|聯合早報]] |date=2018-04-24|accessdate=2018-06-22}}</ref>。

==政治生涯==
[[2015年新加坡大選|2015年新加坡大選]]時他代表人民行動黨在[[三巴旺集選區|三巴旺集選區]]當選國會議員。大選後他與[[黃志明_(新加坡)|黃志明]]做為內閣新面孔擔任教育部長，由他專責高等教育及技能相關事務並兼職國防部高級政務部長<ref name=參選/><ref name=勝選/><ref>{{cite web |language=zh-sg|url=https://www.channel8news.sg/news8/singapore/20150928-sg-new-cabinet/2156394.html|title=黄志明和王乙康：将深入了解教育课题 积极培育优秀人才|publisher=[[8頻道新聞|8頻道新聞]] |date=2015-09-28|accessdate=2018-06-22}}</ref><ref>{{cite web |language=zh-hans|url=http://www.kwongwah.com.my/?p=22132|title=出现多项“第一次” 李显龙宣布新内阁|publisher=光華網 |date=2015-09-29|accessdate=2018-06-22}}</ref>。

2016年他與黃志明一同從教育部代部長升任為教育部長並負責高等教育及技能相關事務<ref name=2016教育部長/>，8月29日他受[[新加坡金融管理局|新加坡金融管理局]]委任為董事<ref>{{cite web |language=zh-sg|url=https://www.channel8news.sg/news8/singapore/20160829-sg-mas/3082022.html|title=王乙康即日起担任金管局董事|publisher=8頻道新聞 |date=2016-08-29|accessdate=2018-06-24}}</ref>。

2018年4月24日，[[新加坡總理|新加坡總理]][[李顯龍|李顯龍]]宣布[[新加坡內閣|新加坡內閣]]改組，改組後他仍然擔任教育部長，不過從原先與黃志明共同掌管教育部改為只由他一人擔任唯一的教育部長，此外他也被視為總理接班人的熱門人選之一<ref name="2018內閣"></ref><ref>{{cite web |language=zh-tw|url=http://news.ltn.com.tw/news/world/breakingnews/2405480|title=李顯龍改組內閣 潛在接班人掌管重量級部會|publisher=自由時報 |date=2018-04-24|accessdate=2018-09-06}}</ref>。
