「'''王維林'''」是一名於1989年6月5日，[[六四事件|六四事件]]的翌日，在[[北京|北京]][[長安街|長安街]]上隻身阻擋[[中国人民解放军|中国人民解放军]]坦克車隊前進的男子<ref>{{Cite web |url= http://www.lexpress.fr/actualite/monde/asie/l-homme-de-la-place-tiananmen_765077.html |title= L'homme de la place Tiananmen |author= Marc Epstein |date= 2009-06-04 |publisher= {{Link-en|L'Express|L'Express (France)}} |language= fr |accessdate= 2014-01-17 |quote=  }}</ref>，實際本名不詳。其照片成為近代歷史上最為著名的照片之一<ref name = "Patrick Witty 1">{{Cite web |url= http://lens.blogs.nytimes.com/2009/06/03/behind-the-scenes-tank-man-of-tiananmen/ |title= Behind the Scenes: Tank Man of Tiananmen |author= Patrick Witty |date= 2009-06-03 |publisher= 《[[紐約時報|紐約時報]]》 |language= en |accessdate= 2014-01-17 |quote=  }}</ref>。該名男子嘗試阻擋約18輛[[59式戰車|59式戰車]]車隊行進。儘管領頭的坦克駕駛一度試圖轉向繞過，但是仍然遭到攔阻而不得前進。該名男子還爬上坦克[[砲塔|砲塔]]，並試圖和車長溝通。他似乎被說服而離開，坦克隊伍則繼續往前行進，但由於該名男子又跑向前，駕駛員再度緊急煞停，對話一陣子後，在僵持之際，一名騎單車的男子前來勸說。最後他才被幾名身穿藍色衣服人士帶離現場。男子獨自阻擋坦克隊伍行進的過程，被{{Link-en|傑夫·懷登|Jeff Widener}}（Jeff Widener）等攝影師及國外媒體記錄下來。很快地，相關相片和影像隨即在國際間各大報紙頭版與新聞媒體中廣為流傳和引起討论<ref name = "Patrick Witty 1"/>。

在[[東歐劇變|東歐劇變]]發生後，該名男子阻擋坦克的影像紀錄影響許多東歐國家的民眾。而在這之後，許多關於[[六四事件|六四事件]]的紀錄片和相關展覽上，都會提及男子阻擋坦克這一事件<ref name = "辛菲">{{Cite web |url= http://www.epochtimes.com/b5/6/6/1/n1336133.htm |title= 港教授披露六四英雄王維林仍在世 |author= 辛菲 |date= 2006-06-01 |publisher= 《[[大紀元時報|大紀元時報]]》 |language= zh-hant |accessdate= 2014-01-17 |quote=  }}</ref><ref>{{Cite web |url= http://tw.aboluowang.com/2012/1211/272982.html |title= 人民網緊急刪除的一張照片 |author= 于飛 |date= 2012-12-11 |publisher= 阿波罗综合新闻网站 |language= zh-hant |accessdate= 2014-01-17 |quote=  }}</ref>，並將阻擋坦克的男子和「王維林」這個名字視為標誌性的象徵<ref name = "于飞">{{Cite web |url= http://www.aboluowang.com/2008/0610/90739.html |title= 魏京生爆六四最新内幕：王维林生死和屠杀人数 |author= 于飞 |date= 2008-06-10 |publisher= 阿波罗综合新闻网站 |language= zh-hans |accessdate= 2014-01-17 |quote=  }}</ref><ref>{{Cite web |url= http://www.epochweekly.com/b5/128/6531.htm |title= 坦克人與天安門 |author= 徐沛 |date= 2009-06-18 |publisher= 《[[新紀元周刊|新紀元周刊]]》 |language= zh-hant |accessdate= 2014-01-17 |quote=  }}</ref>。不過實際上，今日對於該名人物的真實姓名、身分及下落等，仍然沒有獲得確認<ref name = "皮科·艾爾">{{Cite web |url= http://content.time.com/time/magazine/article/0,9171,988169,00.html |title= The Unknown Rebel |author= {{Link-en|皮科·艾爾|Pico Iyer}} |date= 1998-04-13 |publisher= 《[[時代_(雜誌)|時代]]》 |language= en |accessdate= 2014-01-17 |quote=  }}</ref>，[[公共广播电视公司|公共電視網]]節目《{{Link-en|前線 (美國電視節目)|Frontline (U.S. TV series)|前線}}》曾在2006年為此拍攝紀錄片<ref name = "Patrick Witty 1"/>。「王維林」是當前絕大部份中文媒體對於男子的稱呼，不過該名字最早是以英文「Wang Weilin」出現在[[英國|英國]]《[[每日快報|星期日快報]]》上<ref name = "洛杉磯時報">{{Cite web |url= http://articles.latimes.com/1990-06-03/news/mn-819_1_man-tank-pictures |title= Man Who Defied Tanks May Be Dead |author= [[美聯社|美聯社]] |date= 1990-06-03 |publisher= 《[[洛杉磯時報|洛杉磯時報]]》 |language= en |accessdate= 2014-01-17 |quote=  }}</ref>，另外不同新聞媒體對於該名阻擋坦克者也有不同稱呼。

[[八一電影製片廠|八一電影製片廠]]出品的新聞紀錄片《[[共和國衛士|共和國衛士]]》將該名男子稱為「螳臂擋車的歹徒」<ref name="八九天安門事件解放軍縂政治部資料片">{{Cite web |url= https://www.youtube.com/watch?v=qdJPhjMgrQA |title= 八九天安門事件解放軍縂政治部資料片 |author=  |date= 2013-06-05 |publisher=  |language= zh-hant |accessdate= 2018-05-30 |quote=  }}</ref>。儘管相關影像紀錄被視為標誌性的象徵，中國政府禁止網際網路上流傳相關圖像，這導致大部分中國年輕民眾對此並不了解<ref name = "Patrick Witty 1"/>。西方社會則將該名男子稱作「坦克人」（Tank Man），或者是「無名的抗議者」（Unknown Protester）。1998年4月，美國《[[時代_(雜誌)|時代]]》雜誌將該名男子以「無名的反抗者」（The Unknown Rebel）為名，評選為20世紀最具影響力的[[時代100人：本世紀最重要的人物|100個人物]]之一。而在2003年時，由《[[生活_(雜誌)|生活]]》雜誌發行的《{{Link-en|改變世界的100幅照片|100 Photographs that Changed the World}}》中，同樣也收錄了阻擋坦克車隊的照片。

== 紀錄 ==
{{See Also|六四事件}}
[[File:Tank_Man_Long_Shot_by_Stuart_Franklin.jpg|thumb]]

1989年6月5日，中國政府實施[[六四清場|六四清場]]作業已經過了2天<ref>{{Cite web |url= http://www.foreignaffairs.com/articles/56670/andrew-j-nathan/the-tiananmen-papers |title= The Tiananmen Papers |author= [[黎安友|黎安友]] |date= 2001年2月 |publisher= 《[[外交政策_(雜誌)|外交政策]]》 |language= en |accessdate= 2013-01-18 |quote=  }}</ref>。自北京市[[天安門|天安門]]出發的中國製[[59式戰車|59式戰車]]車隊正在[[故宮|故宮]]以南的東[[長安街|長安街]]行進，之後一名雙手持[[購物袋|購物袋]]、穿著白色襯衫與黑色褲子的男子，主動擋在坦克隊伍的行進路線上<ref>{{Cite book | author = Andrew Langley | title = Tiananmen Square: Massacre Crushes China's Democracy Movement | location = 美國曼凱托（Mankato） | publisher = {{Link-en|凱普斯出版社|Capstone Publishers}} | date = 2009-01-01 | pages = 第45頁 | ISBN = 978-0756541019 | accessdate = 2013-12-28 | url = http://books.google.com.tw/books?id=1tU90qvym94C&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false | language = en | quote =  }}</ref>。當下坦克隊伍停止前進，該名男子則揮手示意要坦克後退。領頭的坦克之後試圖繞開該名男子，但男子仍然跟隨不斷左右移動、繼續[[非暴力抵抗|擋在坦克前方]]<ref name = "皮科·艾爾" />，這時後頭至少有18輛坦克也因此受到阻擋<ref>{{Cite web |url= http://www.chine-informations.com/guide/manifestations-de-la-place-tiananmen_758.html |title= Manifestations de la place Tiananmen |author=  |date=  |publisher= Chine Informations |language= fr |accessdate= 2014-01-17 |quote=  }}</ref>。在經過多次嘗試後，第一輛坦克放棄移動並關閉引擎，後方坦克隊伍也陸續關閉引擎<ref name = "黃明珍">{{Cite web |url= http://www.pbs.org/wgbh/pages/frontline/tankman/interviews/wong.html |title= Interviews - Jan Wong |author= [[公共電視網|公共電視網]] |date= 2006-04-11 |publisher= 《{{Link-en|前線 (美國電視節目)|Frontline (U.S. TV series)|前線}}》 |language= en |accessdate= 2013-12-28 |quote=  }}</ref>。在迫使整個坦克隊伍停下後，該名男子爬上領頭的坦克上；他先是短暫地站在駕駛員艙門上，之後開始探查坦克[[砲塔|砲塔]]上每個能夠打開的艙門。

最後爬上坦克砲塔上的男子，似乎在砲手艙門打開後彼此交談一陣子。結束談話後，該名男子從坦克側邊離去<ref>{{Cite web |url= http://www.epochtimes.com/b5/13/5/28/n3880725.htm |title= 六四擋坦克王維林下落? 最完整錄像曝光 |author= 肖笙 |date= 2013-05-28 |publisher= 《[[大紀元時報|大紀元時報]]》 |language= zh-hant |accessdate= 2014-01-17 |quote=  }}</ref>。同時原本在坦克裡的車長從自己艙門口出現，並向後頭的坦克隊伍揮手、表示停止前進。之後第一輛坦克突然發動引擎並準備向前行進，在坦克側邊1公尺至2公尺的男子看見坦克加速前進後，急忙跑上去並再度擋在坦克前方。雙方對峙一段時間後，先是有一名騎著[[腳踏車|-{zh-cn:自行车;zh-hk:單車;zh-tw:腳踏車;zh-sg:脚车;}-]]的男子騎向坦克前方，並與阻擋坦克的男子交談幾句<ref name = "詹姆斯·巴朗">{{Cite web |url= http://www.nytimes.com/1989/06/06/world/crackdown-beijing-one-man-can-make-difference-this-one-jousted-briefly-with.html |title= CRACKDOWN IN BEIJING; One Man Can Make a Difference: This One Jousted Briefly With Goliath |author= {{Link-en|詹姆斯·巴朗|James Barron (journalist)|詹姆斯·巴朗}} |date= 1989-06-06 |publisher= 《[[紐約時報|紐約時報]]》 |language= en |accessdate= 2014-01-17 |quote=  }}</ref>。爾後2名身穿藍色上衣、並高舉雙手的男子也前往現場，接著便帶著阻擋坦克的男子快步離開街道混入人群中，坦克隊伍隨後則繼續向前行進<ref name = "皮科·艾爾" />。

介紹整起事件最常使用的[[相片|相片]]為隸屬於[[美國|美國]]最大通訊社[[美聯社|美聯社]]的[[記者|記者]]{{Link-en|傑夫·懷登|Jeff Widener}}在[[北京飯店|北京飯店]]6樓陽臺拍攝的版本，當時距離事發地點大約有800公尺。懷登原本於6月3日晚間在長安街上遭到民眾投擲的石塊丟到頭部，最後是因為所使用的[[尼康_F3|尼康 F3]]相機吸收了衝擊力道而保住性命<ref name = "Kallie Szczepanski"/><ref name = "Patrick Witty 1"/>。受傷並且罹患流感的他後來通過公安人員的檢查進入北京飯店後，之後帶著攝影器材和底片的他就使用{{Link-en|尼康 FE2|Nikon FE2}}相機搭配400mm 5.6 ED IF[[遠攝鏡頭|遠攝鏡頭]]以及[[尼康F接環增距鏡|TC-301增距鏡]]拍攝天安門廣場的情況<ref>{{Cite web |url= http://www.cbsnews.com/news/tank-man-the-picture-that-almost-wasnt/ |title= "Tank Man": The Picture That Almost Wasn't |author= Sean Alfano |date= 2009-06-02 |publisher= [[哥倫比亞廣播公司|哥倫比亞廣播公司]] |language= en |accessdate= 2014-01-17 |quote=  }}</ref>。後來他發現底片不夠後則拜託科克·馬特森（Kirk Martsen）尋覓底片，在馬特森找到了富士100 ASA彩色負片單捲並且將其裝上相機後，懷登便拍下了阻擋坦克的相片<ref name = "Patrick Witty 1"/>。儘管傑夫·懷登一度擔心相片的品質可能不如預期，但是他仍拜託馬特森將底片交給美聯社辦公室，馬特森之後則拜託[[美国驻华大使馆|美國駐華大使館]]將底片交給美聯社<ref>{{Cite web |url= http://tw.aboluowang.com/news/2012/0606/248982.html |title= 王維林擋坦克震撼照 驚險拍攝細節曝光 成就姻緣 |author= Patrick Witty |date= 2012-06-05 |publisher= 阿波羅綜合新聞網站 |language= zh-hant |accessdate= 2014-02-14 |quote=  }}</ref><ref>{{Cite web |url= http://lightbox.time.com/2012/06/05/tiananmen/#1 |title= Tank Man Revisited: More Details Emerge About the Iconic Image |author= Patrick Witty |date= 2012-06-05 |publisher= 《[[時代_(雜誌)|時代]]》 |language= en |accessdate= 2014-01-17 |quote=  }}</ref>。隨後這張相片便被世界各地的報紙大量採納而廣泛被視為六四事件的象徵，甚至所有歐洲報紙的頭版都是使用這個版本的相片<ref name = "Patrick Witty 1"/>。

同樣注意著天安門廣場狀況的國際攝影經紀公司[[馬格蘭攝影通訊社|馬格蘭攝影通訊社]]的攝影師{{Link-en|斯圖爾特·弗蘭克林|Stuart Franklin}}也在北京飯店5樓拍下了另一個版本的相片，其中這一系列相片的視野比傑夫·懷登還要來得遠並且拍到更多後方的坦克。之後底片由一名法國學生藏在茶葉罐裡面走私出去<ref name = "Patrick Witty 1"/>，其中有些相片被2003年由《[[生活_(雜誌)|生活]]》雜誌發行的《{{Link-en|改變世界的100幅照片|100 Photographs that Changed the World}}》所收錄<ref name="生活">{{Cite web|url=http://digitaljournalist.org/issue0309/lm25.html |title=Tiananmen Square 1989 |author=《[[生活_(雜誌)|生活]]》 |date= |publisher=《{{Link-en|改變世界的100幅照片|100 Photographs that Changed the World}}》 |language=en |accessdate=2014-01-17 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20151207073824/http://digitaljournalist.org/issue0309/lm25.html |archivedate=2015-12-07 }}</ref>。而和[[公共電視網|公共電視網]]攝影團隊在同一陽臺拍攝的美國[[紐約|紐約]]《[[新聞週刊|新聞週刊]]》工作人員[[查理·柯爾_(攝影師)|查理·柯爾]]在拍完阻擋坦克的相片後，隨即將底片藏在北京飯店的馬桶水箱，接著[[公安机关|公安機關]]便前來搜查他的房間，並且將所找到尚未使用的底片以及拍攝受傷示威群眾的底片加以破壞，之後要求柯爾簽署一份懺悔書<ref name = "Patrick Witty 1"/>。在搜索結束後，柯爾成功找回沒有被沖走的底片，並且交由《新聞週刊》發表<ref name = "Monika Rębała">

=== 其他推測 ===
在視訊末段畫面顯示2名身穿藍色衣服的男子突然跑到現場，並且將阻擋坦克的男子快速帶離現場，但是許多人對於將阻擋坦克者帶離的2人真實身分抱持著懷疑。其中拍攝現場畫面的《新聞週刊》工作人員查理·柯爾在2005年10月時便公開表示，認為身穿藍色衣裝將男子拉走者可能就是便衣[[公安機關|公安機關]]成員，並且認為在這之後該名阻擋坦克的男子可能因而遭到強行拘捕<ref name = "Picture power"/>。但是另外一方面，《[[環球郵報|環球郵報]]》記者黃明珍則從兩人帶走阻擋坦克的男子動作來看，認為拉走男子的3名青年並非穿著便衣的[[中國人民武裝警察部隊|中國人民武裝警察部隊]]或者是中國人民解放軍人員，而可能只是剛好看見此事而好心幫助的當地民眾<ref name = "黃明珍"/>。而[[吳仁華|吳仁華]]則在2010年6月時發表研究，認為該名男子阻擋坦克之後便遭到便衣公安人員帶走並且遇害；並且表示從旁邊的坦克部署來看，實際上阻擋坦克的地點應該是已經禁止民眾進入的[[南池子大街|南池子大街]]<ref name = "吳仁華"/>。同時吳仁華也認同另一名美國[[柏克萊加州大學|-{zh-cn:加州大学伯克利分校;zh-tw:加州大學柏克萊分校;}-]]人類學教授教授所指稱，針對3名青年和阻擋坦克男子的肢體動作仔細研究後，認為3名青年是受過專業訓練的便衣人員，因此阻擋坦克的男子可能當場就遭到逮捕，並且至今下落不明<ref>{{Cite web |url= http://www.rfa.org/mandarin/yataibaodao/wang-06042010143022.html |title= 吴仁华：只身挡坦克的王维林已遇害 |author= CK |date= 2010-06-04 |publisher= [[自由亞洲電台|自由亞洲電台]] |language= zh-hans |accessdate= 2014-01-17 |quote=  }}</ref>。

在2008年6月時，流亡美國的[[中華人民共和國民主運動|中華人民共和國民主運動]]人士[[魏京生|魏京生]]在劍橋大學的《勿忘六四：歷史，現實與未來》討論會上則提到另一個版本。他表示坐在遭攔截的第一輛坦克車裡的軍人是自己以前的同學，他的同學看該名男子試圖阻擋坦克後，立刻告訴軍隊偵查連要求把男子帶走以保護他，避免後者因為這項舉動而遭到坦克輾過。該名男子隨後被帶到[[中國人民革命軍事博物館|中國人民革命軍事博物館]]安置，但之後又趁亂逃離並且因為再度阻擋坦克行進而喪生<ref name = "于飞"/>。同時也有報導指稱，該名命令坦克停止前進的車隊指揮官已經因為這次事件而被迫降級和申誡處分<ref name = "聯合報"/>。另外關於該名男子爬上砲塔之後與車內的對話內容也有許多推測版本<ref name = "黃明珍"/>，一些報導認為該名男子可能試圖說服坦克內的軍人回頭並且停止殺害民眾的舉動<ref name = "洛杉磯時報"/><ref name = "詹姆斯·巴朗"/>，另外也有一些說法則明確指稱可能講過「滾出去！」、「離開我的城市、這是我的地方！<ref name="汪宜儒">{{Cite web |url= http://news.chinatimes.com/Chinatimes/Philology/Philology-artnews/0,3409,112009060300168+110513+20090603+news,00.html |title= 念六四 網路串連 Tank Man Tango 全球一起來跳 |author= 汪宜儒 |date= 2009-06-03 |publisher= [[中時電子報|中時電子報]] |language= zh-hant |accessdate= 2014-11-17 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20140201205538/http://news.chinatimes.com/Chinatimes/Philology/Philology-artnews/0,3409,112009060300168+110513+20090603+news,00.html |archivedate= 2014-02-01 }}</ref>」、「回去、轉過頭去、不要再殺害同胞。<ref name = "聯合報"/>」或者是「你們為什麼在這裡？你們只會帶來痛苦。」等話語<ref name = "part 3">{{Cite web |url= http://asianhistory.about.com/od/china/a/TiananmenSquare_3.htm |title= Tiananmen Square Massacre, part 3 |author= Kallie Szczepanski |date=  |publisher= [[About.com|About.com]] |language= en |accessdate= 2014-01-17 |quote=  }}</ref>。

2017年7月，《[[蘋果日報_(香港)|苹果日报]]》报道，“王维林”本名可能叫张为民，北京[[石景山区|石景山]]人，1989年时24岁。判决书指其以砖头砸坦克，最初被判[[无期徒刑|无期徒刑]]，后减至20年，并在关押地[[北京市延庆监狱|北京市延庆监狱]]得奖后，10年前获[[假释|假释]]离开监狱。但出狱后无亲无房，染上赌博，几年后又收监至[[北京市监狱管理局清河分局|垦华监狱]]（地处天津，北京管理），加刑2年。爆料者自称在延庆监狱时期与张为民是同一监室，张为民现在在垦华监狱11分监区，预计即将出狱。<ref>{{Cite news|url=http://hk.apple.nextmedia.com/realtime/china/20170719/56976286|title=【六四坦克人】未死？不叫王維林名叫張為民|author=苹果日报|date=2017-07-19|work=|accessdate=}}</ref>

== 散佈 ==
=== 廣為流傳 ===
這次阻擋坦克事件中共有4名攝影師將其拍成相片、並且於第一時間發佈出去<ref name = "Patrick Witty 1"/>，此外有一名攝影師一直到2009年6月4日時才發表了他從地面上所拍攝的事件相片<ref name = "Patrick Witty 2"/>。到了6月5日時全球數百個各大報紙和雜誌便陸陸續續提到這次事件，同時主要新聞媒體頻道也將其列為頭條新聞，而且使用了從靠近[[天安門廣場|天安門廣場]]的[[北京飯店|北京飯店]]所拍攝的[[視訊|視訊]]畫面或者相片<ref name = "Patrick Witty 1"/>。透過世界各地媒體的播出，很快地國際上便將阻擋坦克的畫面視為六四事件的重要形象<ref name = "Patrick Witty 2"/><ref name = "Jane Macartney"/>，並且廣泛認為這張圖像是20世紀標誌性的畫面<ref name = "南希·裴洛西"/><ref name = "Kallie Szczepanski"/>。其中在2006年3月時，有線電視新聞網的新聞主播{{Link-en|凱拉·菲利普斯|Kyra Phillips}}在評論{{Link-en|2006年法國青年抗議|2006 youth protests in France}}將其與天安門廣場上的抗議活動相連結，並表示：「這把人帶回到過去天安門廣場的回憶，當年我們看見這些積極份子站在坦克面前。」不過由於後來確認在抗議活動中並沒有人因而死亡，使得其這番言論引起了一部分人士的批評<ref>{{Cite web |url= http://www.fin24.com/International/French-protests-Tiananmen-20060328 |title= French protests 'Tiananmen' |author= [[法新社|法新社]] |date= 2006-03-28 |publisher= Fin24 |language= en |accessdate= 2014-11-17 |quote=  }}</ref>。之後[[國際社會|國際社會]]多將阻擋坦克的男子視為成為中國民主運動的象徵、或者是一名青年願意冒著生命危險反抗[[極權主義|極權主義]]的共產政權壓迫之形象<ref>{{Cite web |url= http://www.epochtimes.com/b5/6/6/3/n1338160.htm |title= 六四祭：以共產邪惡為恥，以拒絕暴力為榮 |author= 萬生 |date= 2006-06-03 |publisher= 《[[大紀元時報|大紀元時報]]》 |language= zh-hant |accessdate= 2014-01-17 |quote=  }}</ref>，又或者是公民對於武裝鎮壓所進行的[[非暴力|非暴力]][[公民不服從|不服從行動]]等<ref>{{Cite web |url= http://www.rfa.org/mandarin/pinglun/tbq-20050214.html |title= 中国非暴力抗争的先行者----徐勤先 |author= [[唐柏橋|唐柏橋]] |date= 2005-02-14 |publisher= [[自由亞洲電台|自由亞洲電台]] |language= zh-hans |accessdate= 2014-01-17 |quote=  }}</ref><ref>{{Cite web|url=http://www.libertytimes.com.tw/2014/new/jan/8/today-t3.htm|title=阻台電施工判無罪 法官：如六四擋坦克|author=顏宏駿|date=2014-01-08|publisher=《[[自由時報|自由時報]]》|language=zh-hant|accessdate=2014-01-17|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20140109200132/http://www.libertytimes.com.tw/2014/new/jan/8/today-t3.htm|archivedate=2014-01-09}}</ref>。

當阻擋坦克的影像傳至東歐地區後，其被視為爭取[[自由民主制|自由民主]]權利的無畏[[英雄|英雄]]，並且促使許多當地民眾發起遊行活動<ref name = "part 3"/>。在事件消息傳開後，位於美國的[[中國自由民主陣線|中國自由民主陣線]]和加拿大的[[中國民主促進會_(加拿大)|中國民主促進會]]共同表示，在找到該名阻擋坦克的男子後願意提供1萬加拿大元<ref name = "民運人士資料搜尋">{{Cite web |url= http://prison.alliance.org.hk/?p=1624 |title= 王維林 |author= [[香港市民支援愛國民主運動聯合會|香港市民支援愛國民主運動聯合會]] |date=  |publisher= 民運人士資料搜尋 |language= zh-hant |accessdate= 2014-01-17 |quote=  }}</ref>；而[[中国钱币学会|中國錢幣學會]]也同樣設計出以阻擋坦克為主題的紀念銀幣，並且交由[[美國鑄幣局|美國鑄幣局]]鑄造<ref>{{Cite web |url= http://hk.apple.nextmedia.com/news/art/20090529/12814915 |title= 英美曾鑄銀幣紀念八九民運 |author=  |date= 2009-05-29 |publisher= 《[[蘋果日報_(香港)|蘋果日報]]》 |language= zh-hant |accessdate= 2014-11-17 |quote=  }}</ref>。而1998年4月時，美國《[[時代_(雜誌)|時代]]》雜誌以「無名的反抗者」（The Unknown Rebel）作為名稱評選為20世紀最具影響力的[[時代100人：本世紀最重要的人物|100個人物]]之一，而這也是除了[[中國共產黨中央委員會主席|中國共產黨中央委員會主席]][[毛澤東|毛澤東]]以外唯一獲選的中國人<ref name = "民運人士資料搜尋"/>。在2003年時，由《[[生活_(雜誌)|生活]]》雜誌發行的《{{Link-en|改變世界的100幅照片|100 Photographs that Changed the World}}》同樣選擇收錄了由查理·柯爾所拍攝的阻擋坦克照片，並且評論表示「這張照片顯示了十億多名中國人仍然是有希望的」<ref name = "生活"/>。而2006年4月時，[[公共電視網|公共電視網]]系列電視節目《{{Link-en|前線 (美國電視節目)|Frontline (U.S. TV series)|前線}}》在製作關於六四事件的紀錄片時也將影片命名為「坦克人」（The Tank Man）；其內容除了重新檢視該名男子在六四事件中所扮演的角色外，還報導1989年以後中國在經濟和政治上的變化。

== 影響 ==
=== 照片評價 ===
今日美國和歐洲地區在討論六四事件時往往會提到阻擋坦克事件，並且視阻擋坦克者為20世紀最偉大的英雄之一<ref>{{Cite web |url= http://www.chinagaze.com/2013/06/04/what-happened-to-the-june-4-1989-tank-man/ |title= What Happened to the June 4, 1989 ‘Tank Man’? |author= Su Mingyu |date= 2013-06-04 |publisher= China Gaze |language= en |accessdate= 2014-01-17 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20140220025542/http://www.chinagaze.com/2013/06/04/what-happened-to-the-june-4-1989-tank-man/ |archivedate= 2014-02-20 }}</ref>。在1995年10月，[[公共電視網|公共電視網]]拍攝的紀錄片《[[天安門_(紀錄片)|天安門]]》則評論阻擋坦克的男子為「這是人類良知與勇氣在向無情的國家機器挑戰」。而《時代》雜誌將阻擋坦克者評為20世紀最具影響力的100個人物之一，便於相關文章中提到「他的一個舉動為這個世界重塑了勇氣的象徵」，並且引用中國民主運動人士的話表示「坦克照片的英雄有兩個：未知的身影冒著生命危險站在在重型坦克面前，以及冒著道德挑戰的駕駛员拒絕輾斃他的同胞」<ref name = "皮科·艾爾"/>。而在2009年[[六四事件二十週年|六四事件二十週年]]紀念場合上，美國眾議院議長[[南希·裴洛西|南希·裴洛西]]對於這次事件的照片則形容為「20世紀最持久的形象將再度永遠烙印在世界的良知中」<ref name = "南希·裴洛西"/>。

而在六四事件中擔任學生領導人的[[周锋锁|周鋒鎖]]表示：「王維林擋坦克的形象，作為89民運的一個代表、一個象徵，已經深入人心，這可以說是中國人民為世界人道與自由事業，做出的最大貢獻。<ref name = "楊樺">{{Cite web |url= http://big5.soundofhope.org/node/366922 |title= 王維林入美西海岸最大群雕「記住他們」 |author= 楊樺 |date= 2013-06-06 |publisher= [[希望之聲國際廣播電台|希望之聲國際廣播電台]] |language= zh-hant |accessdate= 2014-11-17 |quote=  }}</ref>」而另外一名學生領導人[[封從德|封從德]]則評論說：「王維林擋坦克的照片，是20世紀最重要的一幅畫面，一個象徵。就是這個畫面，象徵了肆虐半個地球、整個20世紀的共產政權，走向衰亡。它滅亡的起點，就是『六四』。而『六四』真正的象徵，就是王維林。<ref>{{Cite web |url= http://www.epochtimes.com/b5/8/6/7/n2146146.htm |title= 原89民運學生領袖封從德談六四象徵王維林 |author= CK |date= 2008-06-07 |publisher= 《[[大紀元時報|大紀元時報]]》 |language= zh-hant |accessdate= 2014-01-17 |quote=  }}</ref>」而歷史學者[[唐德剛|唐德剛]]則在自己著作中提到：「王維林不是王丹，更不是宋教仁。他只是膽大包天的一個小神經而已。一個偉大的共和國，連一個神經兮兮的街坊小青年，都不能放過，那還是什麼個國家呢？<ref>{{Cite book | author = [[唐德剛|唐德剛]] | title = 《[[袁氏當國|袁氏當國]]》 | location = 臺灣臺北 | publisher = [[遠流出版公司|遠流出版公司]] | date = 2002年 | pages = 第108頁 | ISBN = 978-9573246800 | accessdate = 2013-12-28 | url =  | language = zh-hant | quote =  }}</ref>」

但是[[王军涛|王軍濤]]則認為「王維林」的影像影響了後來人們對於六四事件的觀感，他表示：「王維林擋坦克那個鏡頭雖然是一個經典鏡頭，在西方最廣為人知，但是按照[[方政|方政]]先生的説法，這個鏡頭又曲解了天安門廣場上坦克和人之間的真實關係。就是説，實際上坦克是軋了人的。<ref>{{Cite web |url= http://www.voachinese.com/content/chen-weiming/1146364.html |title= 紐約紀念64 王維林擋坦克時報廣場上演 |author= 葉兵 |date= 2012-06-02 |publisher= [[美國之音|美國之音]] |language= zh-hant |accessdate= 2014-01-17 |quote=  }}</ref>」另外一方面，中國政府則將阻擋坦克者視為一個不負責任的機會主義者，或者是聽信西方世界宣傳而作出這一舉動的人士<ref>{{Cite web|url=http://china.dwnews.com/news/2014-04-03/59462855.html|title=坦克谣言轮番登场 “六四”余毒积重难返|author=泉野|date=2014-04-03|publisher=[[多維新聞|多維新聞]]|language=zh-hans|accessdate=2014-01-17|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20140606215357/http://china.dwnews.com/news/2014-04-03/59462855.html|archivedate=2014-06-06}}</ref>。其中中國共產黨中央委員會總書記[[江澤民|江澤民]]在2000年接受[[哥倫比亞廣播公司|哥倫比亞廣播公司]]主持人[[邁克·華萊士|邁克·華萊士]]採訪時，他對於華萊士所拿出阻擋坦克的照片表示：「在1989年動亂中我們完全理解學生要求更大民主自由時的激情。事實上，我們總是一直在改進我們的民主制度。但是，我們不可能允許懷有不良動機的人利用學生，以民主和自由為借口推翻政府。……我知道你在推進啥，但我要強調，我們完全尊重每一位公民自由表達個人希望和願望的權利，但我不贊成在[[緊急狀態|緊急狀態]]時任何對政府行為的公然反對。坦克停住了，沒有壓過他。<ref>{{Cite web |url= http://www.cbsnews.com/news/chinas-leader-talks-to-60-minutes/ |title= China's Leader Talks To 60 Minutes |author= [[CBS公司|CBS公司]] |date= 2000-08-31 |publisher= 《[[60分鐘|60分鐘]]》 |language= en |accessdate= 2014-01-17 |quote=  }}</ref>」
