'''王霙'''{{noteTag|1=“霙”读音“Yīng”，音同“英”。}}{{BD|1957年|11月|||catIdx=W}}，[[浙江省|浙江省]][[杭州市|杭州市]][[古荡湾|古荡湾]]人，毕业于[[中央实验话剧院|中央实验话剧院]]，为中国大陆影视演员，是飾演[[毛澤東|毛澤東]]的特型演員，為[[国家一级演员|国家一级演员]]。<ref>{{cite web |url= http://news.fznews.com.cn/zt/2010/hsyl/zf/2010-2-5/201025upbAoduGml163359.shtml|title= 王霙：一辈子我就演好一个人|accessdate=2010 |publisher= 福州新闻网|archiveurl=http://news.fznews.com.cn/zt/2010/hsyl/zf/2010-2-5/201025upbAoduGml163359.shtml |archivedate= 2013年1月4日|work= |language=zh-cn}}</ref>

== 生平 ==
王霙出生于[[浙江省|浙江省]][[杭州市|杭州市]][[古荡湾|古荡湾]]，曾在浙江省艺术学校舞蹈班、[[中央红旗越剧团|中央红旗越剧团]]、[[中央实验话剧院|中央实验话剧院]]学习[[舞蹈|舞蹈]]、[[越剧|越剧]]、[[话剧|话剧]]，后从事影视职业。

== 感情生活 ==
1996年，王霙在[[云南省|云南省]]拍摄《彝海结盟》，认识了化妆师“贺亚琳”，后两人相恋而结婚。
