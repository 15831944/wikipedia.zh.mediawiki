'''王凱傑'''（本名：Ersan Yamur，原名：Yamur Kursat Ersagun，{{bd|1986年|9月12日|}}），男，[[土耳其|土耳其]]籍與[[加拿大|加拿大]]籍，身高178厘米，2007年來台灣，[[淡江大學|淡江大學]]學生，曾參加庾澄慶、ELLA主持的交友節目《王子的約會》。

== 個人簡介 ==

土耳其男子王凱傑（本名：Ersan Yamur）來台就學，2015年判處拘役20日。畢業後離台在中國大陸廣東省深圳市就業。

== 事情經過 ==
王凱傑被控在2013年4月間，在網絡認識一名女大學生，兩人相約在捷運站見面，之後王以要回家拿東西，女大生意圖陷害

== 法院判決 ==

最高法院於2015年12月27日認定他不構成強制性交未遂罪，判王男無罪，並駁回上訴，全案定讞。
2015年1月30日被臺灣士林地方法院依傷害罪判處拘役20日，可易科罰金2萬元。
