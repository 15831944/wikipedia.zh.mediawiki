'''王宁'''{{BD|1966年|7月|||catIdx=W}}，[[山西人|山西人]]，[[中国共产党|中国共产党]]党员，[[中共中央党校|中共山西省委党校]]在职研究生，在2014年11月[[习近平反腐|习近平反腐]]中落马之前，担任中共山西省[[吕梁市|吕梁市]][[柳林县|柳林县]]县委书记。<ref>{{cite web |url=http://news.sina.com.cn/c/2014-11-12/183031133816.shtml |title=山西吕梁柳林县委书记王宁接受组织调查 |accessdate= 2014|publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url= http://news.163.com/14/1112/19/AASG5VC600014SEH.html|title= 山西吕梁县委书记被查 曾任职省委办公厅秘书处|accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref>

==生平==
1966年7月，王宁出生在[[山西省|山西省]][[临县|临县]]。

1984年10月，王宁参加工作，1986年3月，加入[[中国共产党|中国共产党]]。

1984年，王宁进入[[山西省军区|山西省军区]]担任战士，之后担任正排职干事、副科长，一直到1992年。1992年12月到1999年1月，王宁在山西省委办公厅工作。

1999年7月，王宁升职为中共[[孝义市|孝义市]]委副书记。

2006年6月，王宁调任中共[[交口县|交口县]]委副书记、交口县人民政府代县长。

2009年11月，王宁调任中共[[柳林县|柳林县]]委副书记、柳林县人民政府代县长，2011年1月，王宁转正县委书记，兼任县长。2011年6月，王宁担任中共柳林县委书记。

2014年11月12日，王宁接受[[中纪委|中纪委]]调查。<ref>{{cite web |url=http://www.ccdi.gov.cn/ajcc/201411/t20141112_30413.html |title=山西省吕梁市柳林县县委书记王宁接受组织调查  |accessdate= 2014|publisher=中纪委 |work= |language=zh-hans}}</ref>
