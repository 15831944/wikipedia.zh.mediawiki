'''王青'''（'''Wong Ching'''，{{bd|1949年|6月24日}}），本姓姜，綽號「大口青」，於[[上海|上海]]出生，香港演員，也是前[[邵氏|邵氏]]演員，曾擔任[[龍虎武師|龍虎武師]]及動作指導。八十年代曾為電影界一線演員，當主角的電影以[[李修賢|李修賢]]執導的《[[Friend過打Band|Friend過打Band]]》為代表。王青曾是[[香港|香港]][[無綫電視|無綫電視]]和[[香港電視網絡|香港電視]]演員<ref>{{cite web |title=好久不見／縱橫戲劇多年 「大口青」王青老當益壯 |url=https://tw.news.yahoo.com/%E5%A5%BD%E4%B9%85%E4%B8%8D%E8%A6%8B%EF%BC%8F%E7%B8%B1%E6%A9%AB%E6%88%B2%E5%8A%87%E5%A4%9A%E5%B9%B4-%E3%80%8C%E5%A4%A7%E5%8F%A3%E9%9D%92%E3%80%8D%E7%8E%8B%E9%9D%92%E8%80%81%E7%95%B6%E7%9B%8A%E5%A3%AF-120155183.html |website=[[Yahoo奇摩新聞|Yahoo奇摩新聞]] |date=2017-02-21|accessdate=2018-08-27}}</ref><ref>{{cite web |title=轉行做警察？ 網民疑似捕獲「大口青」 |url=http://nextplus.nextmedia.com/news/spot/20160226/360747 |publisher=[[壹週刊|壹週刊]]|date=2016-02-26 |accessdate=2018-08-27}}</ref>。

王青的哥哥是寫實[[警匪片|警匪片]][[導演|導演]][[王鍾_(香港)|王鍾]]
，而王青的堂哥是著名演員[[秦沛|秦沛]]以及[[姜大衛|姜大衛]]。
