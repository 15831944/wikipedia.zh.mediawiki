'''王瑞杰'''（Heng Swee Keat，1961年11月1日-），[[新加坡華裔|新加坡華裔]]政治家。曾于1997年至2000年间担任[[李光耀|李光耀]]的秘书，2005年至2011年间任[[新加坡金融管理局|新加坡金融管理局]]局长<ref name="zbsg20110330">{{cite news |language = zh|url = https://www.zaobao.com.sg/special/report/singapore/ge/story20110330-135734 |title = 成为行动党候选人？ 金管局局长王瑞杰传将辞职 |publisher = 联合早报|date = 2011年3月30日}}</ref>，2011年至2015年任新加坡教育部长，2015年至今任新加坡财政部长<ref name="zbsg20150929">{{Cite news|language=zh| url=https://www.zaobao.com.sg/zpolitics/news/story20150929-531715| title=王瑞杰：财政部将盘点已实施计划成效及未来方向|publisher = 联合早报 | date= 2015年9月29日。}}</ref>。

== 教育背景 ==
在新加坡警察部队海外奖学金的资助下，王瑞杰前往[[剑桥大学|剑桥大学]]学习，取得经济学学士学位。后又于美国[[哈佛大学|哈佛大学]]取得[[公共行政硕士|公共行政硕士]]学位<ref name="zbsg20110330"></ref>。

== 政治生涯 ==
2015年9月，王瑞杰由原先的教育部長一職轉任為[[財政部_(新加坡)|財政部長]]<ref>{{cite web |language=zh-my|url=http://news.seehua.com/?p=96554|title=新加坡新内阁名单出炉　过半部门换部长|publisher=詩華日報|date=2015-09-27|accessdate=2018-12-13}}</ref>。

2018年11月，王瑞杰於[[人民行動黨|人民行動黨]]35屆幹部大會後，接替時任新加坡副總理[[張志賢|張志賢]]擔任人民行動黨第一助理秘書長一職，也因此被視為新加坡總理熱門人選。<ref>{{cite web |language=zh-sg|url=https://www.channel8news.sg/news8/latestnews/20181123-sg-pap-new-cec/4187394.html|title=人民行动党宣布 新一届中央执委名单|publisher=8頻道新聞 |date=2018-11-23|accessdate=2018-11-27}}</ref>。
