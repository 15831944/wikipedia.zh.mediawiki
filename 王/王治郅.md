'''王治郅'''{{BD|1977年|7月8日||}}，生于[[北京|北京]]，[[中国|中国]]职业篮球运动员，身高2.14米，司职[[中鋒|中鋒]]或[[大前鋒|大前鋒]]，外號「大郅」；他是第一位效力于[[美国|美国]][[國家籃球協會|NBA]]联盟的非美籍的亚洲球员。王治郅成名于1996年亚特兰大奥运会并在1996年到2012年之间四次参加奥运会（1996,2000,2008,2012）并两次打入八强，成为男篮奥运历史上第一人。

== CBA时期 ==
王治郅在1991年12月进入八一青年男子篮球队。1993年初入选中国少年特殊身材篮球队，并于同年入选中国青年男子篮球队，后加入八一男子篮球队。1994年底入选国家男子篮球集训队。1995年入选国家青年队。1996年入选[[宫鲁鸣|宫鲁鸣]]执教的国家队。

他是[[中国男子篮球职业联赛|中国男子篮球职业联赛]] (CBA)[[八一火箭队|八一火箭队]]的主力[[中锋|中锋]]，在1996年至2001年賽季间，八一队垄断了期间所有的联赛冠军。 王治郅虽然有着2米14的身高，但動作仍十分協調靈巧，並擁有靈活的進攻腳步，再加上慣用手為左手，令防守者對他特別難以捉摸，這使得早期的他成為[[亞洲|亞洲]]頂尖的中鋒球員。

1995年7月，年仅18岁初次亮相国际舞台的王治郅在[[希腊|希腊]]举行的世界青年男子篮球锦标赛上，作为主力中锋带领中国青年队取得第9名的好成绩。整个系列赛王治郅以总得分排名第二，篮板总数排名第二，火鍋次数排名第一的骄人成绩评为本届比赛的最佳中锋，同时荣膺火鍋王美誉，并入选了最佳阵容。

1996年，在[[1996年夏季奥林匹克运动会|亚特兰大奥运会]]男篮赛事中，王治郅表現出色，在和美国[[梦之队|梦之队]]的比赛中防守「海军上将」[[大卫·罗宾逊|大卫·罗宾逊]]时还送给对方一记大火锅。在亚特兰大奥运期间，曾有六所大学球队邀请其加盟。96年，中国队首次进入奥运会男篮八强。王治郅场均11.1分、5.6个篮板和2.0次盖帽。罚球线上32罚29中，命中率90.6%，列所有球员第二位。

此后王治郅坐稳了中国男篮主力中锋，幫助中國男籃隊几乎囊获了[[亞洲運動會|亚运会]]男籃項目、[[亞洲籃球錦標賽|亞錦賽]]、[[东亚运动会|东亚运动会]]男籃項目等亚洲赛事的所有冠军。

== 加盟NBA ==
王治郅在1999年[[NBA选秀|NBA选秀]]中被[[达拉斯小牛队|达拉斯小牛队]]于第2轮第36位选中，他是第一位进军NBA的亚洲球员（儘管在50年代NBA已有美籍亚裔球员[[三阪亙|三阪亙]]，但並非亞洲人）；但在2001年三月正式加盟达拉斯小牛队，擔任替補中鋒。後來王治郅陸續转会到[[洛杉矶快船队|洛杉矶快船队]]、[[迈阿密热火队|迈阿密热火队]]等球隊。職業生涯後期，王治郅一的主要進攻手段轉變成三分線外的投籃，並較常扮演替補以及功能性（投籃得分）的角色。

== 与中国篮协的误会 ==
2002年违反[[中国篮协|中国篮协]]规定留美参加夏季联赛并私自决定在美国与中国国家队会合参加同年在美国举行的[[世界盃籃球賽|世界锦标赛]]，而被中国国家队除名，此后王治郅有四年時間无法回国参加国家队的比赛，直到再次回国前，一直遭到大陆媒体封杀；经过各方面的协调，王治郅終於在2006年4月回到中国。並再度入选[[中国国家男子篮球队|中国国家男子篮球队]]，参加了[[日本|日本]]举行的[[2006年世界男子篮球锦标赛|2006年世界男子篮球锦标赛]]。在小组赛对阵[[斯洛文尼亚|斯洛文尼亚]]的关系到中国小组能否出线的关键比赛中，王治郅在第四節接連命中中投和三分，幫助中国队穩住戰局最终险胜对手，成功完成了晉級十六強的基本目標。

== 重返CBA ==
2006-07赛季王治郅带领八一队以4-1击败了[[易建联|易建联]]领衔的夺冠大热门[[广东华南虎篮球俱乐部|广东华南虎]]获得CBA总冠军。这也是迄今为止军旅最后一次获得CBA总冠军。

== 重返国际赛场 ==
王治郅从1996年成名于奥运会，在他参加的每届奥运会都有上佳表现。

在重返国家队后，投入[[日本世锦赛|日本世锦赛]]和[[北京奥运会|北京奥运会]]的比赛中。在世锦赛上帮助中国队拿下斯洛文尼亚，小组出线。在北京奥运会上和[[姚明|姚明]]，[[李楠_(篮球运动员)|李楠]]等资深运动员合力确保中国队进入八强。在与西班牙的小组比赛中，王治郅发挥出色，几乎绝杀世界冠军，但中国队还是在加时赛中实力不济落败。

2009年[[男篮亚锦赛|男篮亚锦赛]]的決賽中表現出色，但中国队大比分輸給[[伊朗|伊朗]]未夺冠。
2010年[[男篮世锦赛|男篮世锦赛]]的比賽中表現出色，但中国队小组出线后輸給[[立陶宛|立陶宛]]未能进入八强。

2010年11月24日，在[[广州亚运会|广州亚运会]]中国於準決賽对阵[[伊朗国家男子篮球队|伊朗国家男子篮球队]]的最后时刻，'''王治郅'''和[[王仕鹏|王仕鹏]]投中三分，狙杀对手，使得中国队最终逆轉成功。兩日後，王治郅在中國對南韓的冠軍賽中獲得全場最高分，並在關鍵時刻投進中國隊最後一個非罰球進球，幫助中國隊險勝南韓隊奪冠。赛后，国家队老兵王治郅受到中国男篮全体队员的顶礼膜拜，中国男篮队员纷纷把自己的金牌一一挂在王治郅的脖子上。

2011年[[男篮亚锦赛|男篮亚锦赛]]中表現出色，在武汉体育中心进行的决赛争夺中，东道主中国男篮以70-69险胜约旦男篮，从而以9战全胜的不败战绩赢得本届亚锦赛冠军并获得唯一1张直通2012年伦敦奥运会的入场券。该场比赛CCTV-5的解说嘉宾为[[姚明|姚明]]、[[王仕鹏|王仕鹏]]，[[易建联|易建联]]获得该届男篮MVP。


在2012年奥运会比赛中出任中国队场上队长，但中国队在小组赛中先后输给[[西班牙国家男子篮球队|西班牙]]、[[俄罗斯国家男子篮球队|俄罗斯]]、[[澳大利亚国家男子篮球队|澳大利亚]]、[[巴西國家男子籃球隊|巴西]]、[[英国國家男子籃球隊|英国]]等，小组赛未能出线。

2013年，王治郅，[[朱芳雨|朱芳雨]]，[[王仕鹏|王仕鹏]]等老将再次入选中国国家队征战男篮亚洲锦标赛。但这次比赛最终并未进入四强成为中国男篮一队半个世纪以来最差的亚锦赛成绩，中国男篮国家队也历史上第二次缺席男篮世界杯。

== 退役 ==
在经历了2012年伦敦奥运会和2013年亚洲竞赛赛的失利后，36岁的王治郅于2014年选择退役。

== 主教練 ==
因[[阿的江|阿的江]]身体不好离任，由王治郅接任。
