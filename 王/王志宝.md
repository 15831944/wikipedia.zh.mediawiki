'''王志宝'''（{{bd|1939年|6月||}}）[[山东|山东]][[招远|招远]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[国家林业局|国家林业局]]局长、党组书记<ref name=lyw/><ref name=kf>{{Cite web |url = http://www.china.com.cn/economic/txt/2009-09/29/content_18623901.htm |title = 2000年 西部大开发：我向朱总理建议西部退耕还林 |author =  |date = 2009-09-29 |publisher = 中国网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
王志宝是高级工程师。1965年5月加入[[中国共产党|中国共产党]]。1963年7月自[[吉林电力学院|吉林电力学院]]动力系电厂热能动力工程专业毕业。同年8月参加工作。曾任[[内蒙古|内蒙古]][[牙克石市|牙克石]]林管局基建处技术员、机电处副处长，绰尔林业局党委副书记、代局长，牙克石林管局副总工程师兼计划处处长、副局长。1986年6月，任[[中华人民共和国林业部|林业部]]计划司副司长。1989年4月，任林业部综合计划司司长。1992年9月，任林业部副部长、党组成员。1998年3月，任[[国家林业局|国家林业局]]局长、党组书记，[[全国绿化委员会|全国绿化委员会]]副主任<ref name=lyw>{{Cite web|url = http://www.forestry.gov.cn/LdzqProtalAction.do?dispatch=more&colid=5077|title = 王志宝|author = |date = |publisher = 中国林业网|language = |accessdate = 2017-11-23|quote = |archive-url = https://web.archive.org/web/20170722232959/http://www.forestry.gov.cn/LdzqProtalAction.do?dispatch=more&colid=5077|archive-date = 2017-07-22|dead-url = yes}}</ref><ref>{{Cite web |url = http://www.forestry.gov.cn/main/5077/content-839553.html |title =  王志宝|author =  |date = 2016-01-26 |publisher =国家林业局  |language =  |accessdate =  |quote =  }}</ref>。任至2000年。1999年在[[延安|延安]]考察时，王志宝向国务院总理[[朱镕基|朱镕基]]建议西部地区[[退耕还林|退耕还林]]<ref name=kf/>。在担任国家林业局局长期间，曾参与[[天然林保护工程|天然林保护工程]]、[[退耕还林|退耕还林]]工程、[[京津沙源治理|京津沙源治理]]工程等国家重大生态建设工程的决策，并负责主持上述工程的规划、实施工作<ref name=jl>{{Cite web |url = http://news.sohu.com/20070306/n248525999.shtml |title = 全国政协委员王志宝简历 |author =  |date =2007-03-06  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。2000年任[[国务院西部地区开发领导小组|国务院西部地区开发领导小组]]办公室副主任<ref name=kf/><ref>{{Cite web |url = http://www.china.com.cn/economic/txt/2003-03/08/content_5289407.htm |title =政协五委员就扶贫和西部开发答记者问  |author =  |date = 2003-03-08  |publisher =中国网  |language =  |accessdate =  |quote =  }}</ref>。任职期间参与了[[退牧还草|退牧还草]]工程、[[生态移民工程|生态移民工程]]、[[农村能源工程|农村能源工程]]等国家重点工程的决策<ref name=jl/>。2007年任[[中国绿化基金会|中国绿化基金会]]主席<ref name=kf/>。

王志宝是第九、十届[[全国政协|全国政协]]委员，第十届全国政协经济委员会委员。2005年后，参加了全国政协组织的[[社会主义新农村|社会主义新农村]]建设中的关于农村发展和基础建设、农村专业合作组织、农村金融问题、西部开发农村发展问题等专题调研活动。2006年全国[[两会|两会]]期间，在[[中共中央总书记|中共中央总书记]][[胡锦涛|胡锦涛]]参加农业联组会听取社会主义新农村建设意见时，王志宝就利用耕地以外的可利用的国土资源，如林地资源、沙地资源、湿地资源和草原资源，以《拓宽思路，走出粮食增产、农民增收双赢之路的建议》作专题发言，并通过提案将意见转交有关部门<ref name=jl/>。
