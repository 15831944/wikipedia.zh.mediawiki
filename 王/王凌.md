'''王凌'''（{{bd|172年||251年|6月15日|catIdx=W王}} <ref>《三國志·魏書·齊王紀》：（嘉平三年）四月丙午（6月7日），聞太尉王凌謀廢帝，立楚王彪，太傅司馬宣王東征凌。五月甲寅（6月15日），凌自殺。 </ref> <ref>《魏略》：凌行到项，夜呼掾属与决曰：“行年八十，身名并灭耶！”遂自杀。</ref>），字'''彦云'''，[[太原郡|太原郡]][[祁縣|祁縣]]人，[[三国|三国]]时[[曹魏|曹魏]]重要将领，[[东汉|东汉]]末年谋诛[[董卓|董卓]]的[[司徒|司徒]][[王允|王允]]之侄。

== 生平 ==
早年與同郡[[王昶_(三國)|王昶]]齊名，因年長而被王昶以兄事之<ref>《王昶傳》：王昶字文舒，太原晋阳人也。'''少与同郡王凌俱知名。凌年长，昶兄事之'''。</ref>。後王允一家被[[李傕|李傕]]等殺害後，王凌與其兄[[王晨_(三國)|王晨]]逃出長安，跑回太原。後來舉孝廉，被并州刺史[[梁習|梁習]]舉薦<ref>《三國志·常林傳》：后刺史梁习荐州界名士[[常林|林]]及[[楊俊_(三國)|杨俊]]、'''王凌'''、[[王象|王象]]、[[荀纬|荀纬]]，太祖皆以为县长。</ref>，先後任[[發干|發干]]縣長及[[中山郡|中山]]太守，有治績，後來得到受[[曹操|曹操]]赏识，當丞相掾屬。

[[曹丕|曹丕]]稱帝後，任散騎常侍，出任[[兗州|兗州]]刺史，曾參與222年的[[洞口之戰|洞口之戰]]，因功封為'''宜城亭侯'''加建武將軍。來任[[青州|青州]]刺史時又整理法度，重建當地戰亂後的秩序以助平定利城[[唐咨|唐咨]]的叛亂<ref>《三國志·魏文帝紀》：（帝）遣屯騎校尉任福、步兵校尉段昭與'''青州刺史'''討平之</ref>，得到百姓稱頌；期間錄用[[王基_(三國)|王基]]為州別駕<ref>《王基傳》：黄初中，察孝廉，除郎中。是时青土初定，'''刺史王凌'''特表请基为别驾，后召为秘书郎，凌复请还。</ref>。後來在[[夾石之戰|夾石之戰]]率兵勇戰，令[[曹休|曹休]]得以撤退。後先後任[[揚州|揚州]]和[[豫州|豫州]]刺史，甚得軍民歡心。

太和五年（231年），由於對[[滿寵|滿寵]]有意見加上被吳國詐降計坑了，就反將滿寵受誣回朝。<ref>《三國志·滿寵傳》全文（附裴注）：其明年，吴将孙布遣人诣扬州求降，辞云：“道远不能自致，乞兵见迎。”'''刺史王凌'''腾布书，请兵马迎之。宠以为必诈，不与兵，而为凌作报书曰：“知识邪正，欲避祸就顺，去暴归道，甚相嘉尚。今欲遣兵相迎，然计兵少则不足相卫，多则事必远闻。且先密计以成本志，临时节度其宜。”宠会被书当入朝，敕留府长史：“若凌欲往迎，勿与兵也。”凌於后索兵不得，乃单遣一督将步骑七百人往迎之。布夜掩击，督将迸走，死伤过半。初，'''宠与凌共事不平，凌支党毁宠疲老悖谬'''，故明帝召之。既至，体气康强，见而遣还。〈世语曰：王凌表宠年过耽酒，不可居方任。帝将召宠，给事中郭谋曰：“宠为汝南太守、豫州刺史二十餘年，有勋方岳。及镇淮南，吴人惮之。若不如所表，将为所闚。可令还朝，问以方事以察之。”帝从之。宠既至，进见，饮酒至一石不乱。帝慰劳之，遣还。〉</ref>

[[正始_(曹魏)|正始]]元年（240年），任征東將軍，假節都督揚州諸軍事。次年（241年），在[[芍陂|芍陂]]大戰擊敗吳將[[全琮|全琮]]，全琮撤退。進封'''南鄉侯'''，遷車騎將軍，儀同三司。<ref>《三國志·魏書·齊王紀》：（二年）六月己卯（7月2日），以征東將軍王凌為車騎將軍。</ref>又於正始九年（248年）升任[[司空|司空]]。

[[嘉平_(曹魏)|嘉平]]元年（249年），[[司馬懿|司馬懿]]發動[[高平陵之變|高平陵之變]]，大將軍[[曹爽|曹爽]]被[[處死|處死]]，司马氏掌握魏国大权。王凌則升任[[太尉|太尉]]<ref>《三國志·魏書·齊王紀》：冬十二月辛卯（250年1月28日），以司空王凌為太尉。</ref>，[[节钺|假節鉞]]。

當時[[兗州|兗州]]刺史[[令狐愚|令狐愚]]是王凌的外甥，二人都掌兵。二人合謀要推舉楚王[[曹彪|曹彪]]為帝，推翻司马氏，令狐愚更派部將[[张式_(三国)|張式]]聯絡曹彪，但同年令狐愚就病死。次年夜觀星象，認為有祥瑞的兆頭，決定待機會實行計劃。

嘉平三年（251年），趁著吳軍在涂水有軍事行動，王凌決定藉此機會行動，上表要求討伐[[孙吴|東吳]]，但未有回音，王凌於是派將軍[[杨弘_(曹魏)|楊弘]]將計劃通知兗州刺史[[黄华_(东汉)|黃華]]，但他們卻報告給司馬懿。司馬懿於是討伐王凌，但先下令赦免王凌罪行。王凌自知不敵，於是投降，司馬懿納降，並派兵送王凌回[[洛邑|洛邑]]。王凌害怕事情不如就此完結，向司馬懿取棺材釘以知其心意，但司馬懿還是命人送他一顆；王凌知道自己必定會被殺，到[[項縣|項縣]]時飲毒藥自殺。後來司馬懿到[[壽春|壽春]]，張式將王凌的計劃供出，相關的人包括王凌及令狐愚等都被誅滅三族，王凌和令狐愚更被開棺暴屍三日。王凌有一妹为[[郭淮|郭淮]]妻，因郭淮求情，终得免罪。

== 逸聞 ==
*王凌自杀前项经过[[賈逵_(三國)|贾逵]]庙，大呼：「贾梁道！只有你才知道王凌是大魏忠臣啊！」<ref>《[[晋书|晋书]]·宣帝纪》</ref>
*[[南朝宋|南朝宋]]將領[[王玄謨|王玄謨]]被伯父評有王淩之風。<ref>《[[宋書|宋書]]·王玄謨傳》：王玄謨，字彥德，太原祁人也。六世祖宏，河東太守，綿竹侯，以從叔司徒允之難，棄官北居新興，仍為新興、雁門太守，其自敘雲爾。……玄謨幼而不群，世父蕤有知人鑒，常笑曰：「此兒氣概高亮，有'''太尉彥雲（王淩表字）之風'''。」</ref>

== 子女 ==
* [[王廣_(三國)|王廣]]，字公渊，王凌之子，《[[漢晉春秋|漢晉春秋]]》載其知道王凌要廢曹芳立曹彪後，曾勸王凌不要輕舉妄動，但王凌不聽，然裴松之質疑此記載的真實性。後與王凌家族一同被誅殺，死時四十多歲。
* [[王飞枭|王飞枭]]，王广弟。
* [[王金虎|王金虎]]，王广弟。
* [[王明山|王凌幼子]]，字明山，知名而擅長書法，后因王凌事被捕。
