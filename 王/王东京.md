'''王东京'''（{{bd|1959年|11月||}}）[[湖南|湖南]][[安乡|安乡]]人，[[中国共产党|中国共产党]]官员，[[中共中央党校|中共中央党校]]副校长<ref name=dbxx>{{Cite web |url =http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=131360|title = 代表信息：王东京 |author =  |date =  |publisher =中国人大网  |language =  |accessdate =2018-03-13  |quote =  }}</ref>。

==生平==
1991年自[[中国人民大学|中国人民大学]]经济系毕业，获博士学位。历任[[中共中央党校|中共中央党校]]教务部主任（原经济学部主任）、《中国经济观察》主编、中央党校教授、[[博士生导师|博士生导师]]、享受[[中华人民共和国国务院|国务院]][[政府特殊津贴|政府特殊津贴]]专家。2014年3月任中央党校教育长。根据2015年2月5日[[中共中央|中共中央]]文件（2015）89号通知精神，中共中央批准王东京任中央党校副校长，免去[[张伯里|张伯里]]的中央党校副校长职务<ref name=rmw>{{Cite web |url = http://hn.people.com.cn/n/2015/0227/c195194-24004997.html |title = 王东京任中央党校副校长 系湖南安乡人 |author =  |date =2015-02-27  |publisher =  人民网|language =  |accessdate =  |quote =  }}</ref>。2018年当选[[第十三届全国人大|第十三届全国人大]]河南代表<ref name=dbxx/>。2018年3月，任[[全国人大财政经济委员会|全国人大财政经济委员会]]委员<ref name=caijing>{{Cite web |url = http://www.npc.gov.cn/npc/xinwen/2018-03/13/content_2048059.htm|title = 第十三届全国人民代表大会财政经济委员会主任委员、副主任委员、委员名单 |author =  |date = 2018-03-13 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>。
