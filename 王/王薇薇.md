'''王薇薇'''（{{lang|en|'''Vera Wang'''}}，{{bd|1949年|6月27日|catIdx=Wang, Vera}}），[[美國|美國]][[紐約州|紐約州]][[時裝設計師|時裝設計師]]，她以設計結婚禮服而著稱。王薇薇的禮服打破了舊式婚紗的繁複和華麗，以簡約、時尚、現代的設計在美國時尚界擁有了自己的地位。

雖然王薇薇的禮服價格不菲，但是由於她的設計永遠不會讓明星在典禮上成為“最差著裝”，因而她的設計在[[好萊塢|好萊塢]]可謂趨之若鶩。<ref>[http://www.haibao.cn/brand/1259/ 時尚品牌庫：王薇薇]</ref>

而在[[香港|香港]]暢銷女[[作家|作家]][[亦舒|亦舒]]的筆下，女主角結婚都是“摇一个电话到纽约王薇薇处订婚纱”。<ref>{{Cite web |url=http://female.jxnews.com.cn/system/2009/06/04/011127703.shtml |title=十位叱咤时尚界的华裔设计师 |access-date=2009-06-04 |archive-url=https://web.archive.org/web/20130303031341/http://female.jxnews.com.cn/system/2009/06/04/011127703.shtml |archive-date=2013-03-03 |dead-url=yes }}</ref>

== 早年經歷 ==
王薇薇出生於[[紐約|紐約]]一個家境[[殷實|殷實]]的[[華裔|華裔]]家庭，她的父母都是中国上海人，外祖父是民國時代奉系軍閥[[吳俊陞|吳俊陞]]。她在進入[[巴黎大學|巴黎大學]]之前就讀于[[曼哈頓沙潘學校|曼哈頓沙潘學校]]。她最後取得了[[莎拉劳伦斯学院|莎拉劳伦斯学院]]的藝術史[[學位|學位]]。

王薇薇的母親喜歡并經常帶王薇薇去[[巴黎|巴黎]]參加時尚展，而她的父親則擁有一個[[化工|化工]]工廠。

幼年，王薇薇曾經被培訓成一個[[花樣滑冰|花樣滑冰]]選手。她參加了1968年的全美花樣滑冰錦標賽，并獲選[[運動畫刊|運動畫刊]]1968年1月8日“非一般的面孔”（{{lang|en|[[:en:Faces_in_the_Crowd|Faces in the Crowd]]}}）系列。<ref>《[[運動畫刊|運動畫刊]]》（2006年12月15日 號）第66頁</ref> 在落選美國奧運隊資格之後，她進入了時裝行業。

== 職業生涯 ==
[[File:Vera_Wang.jpg|thumb]][[中央公園|中央公園]]]]

王薇薇曾在《{{lang|en|[[Vogue雜誌|VOGUE]]}}》（美國版）擔任了16年的高級時尚編輯。1985年，王薇薇因申請總編輯職務被拒離開了《{{lang|en|VOGUE}}》，而這個職位最後由[[安娜·溫特|安娜·溫特]]（{{lang|en|[[:en:Anna_Wintour|Anna Wintour]]}}）獲得。離開《{{lang|en|VOGUE}}》后，王薇薇在[[Polo_Ralph_Lauren|拉爾夫·勞倫]]品牌下擔任了兩年的設計總監。

1990年，王薇薇創建了屬於自己的[[品牌|品牌]]，並且在[[紐約市|紐約市]]的卡利斯尔酒店展出她的服裝設計。除了晚裝之外，王薇薇也在設計[[花樣滑冰|花樣滑冰]]比賽服裝，她的體育服裝顧客包括[[关颖珊|关颖珊]]，[[南茜·克里根|南茜·克里根]]跟[[陈巍|陈巍]]。1994年[[1994年冬季奧林匹克運動會|冬奧會]]銀牌得主[[南茜·克里根|南茜·克里根]]，2010年冬奥会金牌得主伊万·莱萨切克正是穿著王薇薇設計的服裝參加比賽的。她還為[[美國|美國]]橄欖球隊[[費城老鷹|費城老鷹]]的啦啦隊設計了兩套服裝。近年来，王薇薇开始在影视中展示自己设计的作品，例如在《[[慾望城市_(電影)|慾望城市]]》（電影版）中[[凱莉·布雷蕭|凱莉·布雷蕭]]的多套婚紗套裝；《[[绯闻女孩_(电视剧)|緋聞女孩]]》第一季最後一集中主角的婚紗。

目前王薇薇將她的品牌從服裝延伸到[[香水|香水]]、[[珠寶|珠寶]]、[[眼鏡|眼鏡]]、[[鞋|鞋]]、[[傢具|傢具]]設計、[[瓷器|瓷器]]和[[水晶|水晶]]上。

2001年10月23日，王薇薇出版了她的第一本書《王薇薇的婚禮計畫》。

2005年6月，王薇薇獲得了{{lang|en|CFDA}}（全美時裝設計師協會）年度最佳女裝設計師。

2006年5月27日，王薇薇獲得了由[[美國薩凡納藝術設計學院|美國薩凡納藝術設計學院]]頒發的安德鲁·里昂·塔利（{{lang|en|[[:en:André_Leon_Talley|André Leon Talley]]}}）终生成就奖。

2010年，为美国前总统比尔·克林顿和现任国务卿希拉里·克林顿的女儿切尔西·克林顿的婚礼设计婚纱。

2018年，於Vera Wang SS19中展示了裸色婚紗系列，附合現時審美觀的剪裁及用色引起極大迴響<ref>[https://www.mings-fashion.com/vera-wang-%E5%A9%9A%E7%B4%97-%E6%98%A5%E5%A4%8F-4415/ 當婚紗好睇過Ready-to-wear〡Vera Wang Spring 2019 的裸色婚紗]MING'S</ref>。

== 個人生活 ==
王薇薇目前居住在[[美國|美國]][[紐約市|紐約市]]，她在1989年嫁給了亞瑟·貝克。丈夫亞瑟·貝克是[[IT|IT]]諮詢服務公司{{lang|en|[[:en:NaviSite|NaviSite]]}}的[[CEO|CEO]]。

她目前有兩個女兒：茜茜尼婭（1990年出生）和約瑟芬（1993年出生），就讀于王薇薇的母校[[曼哈頓|曼哈頓]][[:en:Chapin_School_(Manhattan)|沙潘學校]]。

2012年7月11日，王薇薇與其丈夫亞瑟·貝克結束了他們長達23年的婚姻。<ref>{{cite web|author=Nardine Saad|title=Designer Vera Wang splits from hubby Arthur Becker after 23 years|url=http://www.latimes.com/entertainment/gossip/la-et-mg-vera-wang-split-arthur-becker-separated,0,7510363.story|publisher=洛杉磯時報|accessdate=2012-07-12}}</ref>
