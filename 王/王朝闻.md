'''王朝闻'''（{{bd|1909年|4月18日|2004年|11月11日}}），原名'''王昭文'''，笔名'''汶石'''、'''[[廖化|廖化]]'''、'''席斯珂'''，生于[[四川省|四川省]][[合江县|合江县]]，[[中华人民共和国|中华人民共和国]]文艺理论家、美学家、雕塑家，艺术教育家，中华人民共和国[[马克思主义|马克思主义]]文艺理论和美学的开拓者与奠基人之一<ref name=fuxing>福星，美学大家王朝闻，美术观察1997(1):52-53</ref><ref name=geng>{{Cite web |url =https://www.sohu.com/a/224092486_250532  |title = 著名摄影家 耿兴余《王朝闻与我谈摄影艺术》 |author =  |date =2018-02-26  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref><ref name=yinian>{{Cite web |url = http://legal.scol.com.cn/2012/03/07/2012030723405408.834.htm |title =忆念美学家王朝闻  |author =王应槐  |date = 2012-03-07 |publisher = 四川法制报 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
王朝闻原名王昭文，后取《[[论语|论语]]·里仁》“朝闻道，夕死可矣”句，更名“王朝闻”。他有一枚闲章“夕不甘死”，其义也是从此衍生<ref name=fuxing/>。王朝闻自幼喜爱绘画<ref name=geng/>。为学美术，王朝闻不顾贫困，两次放弃家人期望的钱庄学徒和银行练习生的“铁饭碗”职业。他曾在合江县县立高等小学学习，曾考入泸县中学。1926年在成都艺专等校学习美术<ref name=yinian/>。

1932年，王朝闻离开四川，考入[[国立杭州艺术专科学校|国立杭州艺术专科学校]]（今[[中国美术学院|中国美术学院]]），当时[[林风眠|林风眠]]任校长，[[刘开渠|刘开渠]]任雕塑教员。王朝闻靠勤工俭学读书，在校期间常去图书馆摹绘西方雕塑和绘画的图片<ref name=fuxing/>。其木刻《三等车厢》曾受到[[鲁迅|鲁迅]]关注<ref name=yinian/>。[[抗日战争|抗日战争]]全面爆发后，王朝闻中断学业，在浙江投身抗日救亡，绘制过不少抗日宣传画、连环画和漫画，还当过临时演员<ref name=fuxing/>。1937年在浙江从事抗日文艺宣传活动，同年加入[[中国共产党|中国共产党]]。1939年在成都私立南虹艺专等校任教，任成都民众教育馆美术部主任。1940年12月到[[延安|延安]]后，曾在[[鲁迅艺术学院|鲁迅艺术文学院]]美术系任教。1941年，他为延安[[中共中央党校|中央党校]]大礼堂创作的大型[[毛泽东|毛泽东]]浮雕像，被视为[[解放区|解放区]]美术作品的代表作<ref name=yinian/>。他还创作了鲁迅浮雕像等，发表了《再艺术些》等短文，反对艺术作品公式化和概念化。后来他在[[华北联合大学|华北联合大学]]艺术学院美术系任教期间，系统阅读了中国古代画论、词论、文论和毛泽东哲学、军事著作<ref name=geng/><ref name=fuxing/>。

[[中华人民共和国|中华人民共和国]]成立后，他曾在[[中共中央宣传部|中共中央宣传部]]文艺处等部门工作。曾任[[中央美术学院|中央美术学院]]教授、副教务长，讲授文艺理论和雕塑创作，创作了《毛泽东浮雕像》和《[[刘胡兰|刘胡兰]]》。他创作的《[[毛泽东选集|毛泽东选集]]》封面浮雕等作品产生了广泛影响。1953年，他受文化部委托筹组了民族美术研究所（今[[中国艺术研究院|中国艺术研究院]]美术研究所），还曾任《人民美术》、《美术》月刊和文科教材《美学概论》的主编。“[[文革|文革]]”结束后，王朝闻曾任[[中国艺术研究院|中国艺术研究院]]副院长，[[博士生导师|博士生导师]]，主持国家重点科研项目《中国美术史》的编撰。他还曾任《美术》月刊顾问，[[中华人民共和国文化部|文化部]]文艺研究所顾问，[[中国美术家协会|中国美术家协会]]常务理事、副主席、顾问，[[中国作家协会|中国作家协会]]理事、顾问，中华美学学会会长、名誉会长，[[国务院学位委员会|国务院学位委员会]]第一届学科评议组成员，全国政协第三、四、五、六届委员，[[中国文联|中国文联]]第一、二、三、四届全国委员，文化部艺委会委员等职<ref name=geng/><ref name=yinian/><ref name=shishi/><ref name=fuxing/>。

2004年11月11日，王朝闻在北京病逝，享年96岁<ref name=geng/><ref name=yinian/><ref name=shishi>{{Cite web |url =http://www.people.com.cn/GB/wenhua/22219/3003735.html  |title = 卓越艺术家王朝闻同志逝世 |author =  |date = 2004-11-22 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。
