'''王太后'''，是[[王|王]]、[[親王|親王]]、[[國王|國王]]之母的封號。最早在[[中國|中國]][[戰國|戰國時期]]就有王太后出現，是世界上最早的[[太后|太后]]。

== 東亞 ==
=== 中國 ===
中國最早在戰國時期，就已經有王太后存在，如[[趙國|趙國]]的-{[[趙威后|趙威后]]}-、[[秦國|秦國]]的[[宣太后|宣太后]]、[[夏姬|夏太后]]等等，都是王的母親，因而得稱王太后。

[[漢朝|漢朝]]延續了戰國時期王母稱王太后的制度，後宮[[妃嬪|妃嬪]]若生子、且其子有封國，可在[[皇帝|皇帝]]崩後出宮前往其子的封國，成為該國的王太后，又稱'''王國太后'''<ref>[http://zh.wikisource.org/wiki/%E8%AB%8B%E5%AE%9A%E5%85%AC%E4%B8%BB%E6%AF%8D%E7%A8%B1%E8%99%9F%E7%8B%80  請定公主母稱號狀]：「伏尋漢制，諸王母稱'''王國太后'''；晉宋以降，則曰王國太妃。國朝酌前代典故，從晉宋之儀，王母命為太妃」</ref>，如代太后[[薄姬|薄姬]]、中山太后[[馮媛|馮媛]]等等。[[汉武帝|汉武帝]]时，[[齊郡|齐王]][[劉閎|刘闳]]之母[[王夫人_(汉武帝)|王夫人]]深受宠爱。王夫人死后，武帝赐她为“齐王太后”<ref>《史记 卷六十 三王世家》</ref>。

東漢以後，王母降稱[[太妃|王太妃]]，或稱'''王國太妃'''<ref>[http://zh.wikisource.org/wiki/%E8%AB%8B%E5%AE%9A%E5%85%AC%E4%B8%BB%E6%AF%8D%E7%A8%B1%E8%99%9F%E7%8B%80  請定公主母稱號狀]：「伏尋漢制，諸王母稱王國太后；晉宋以降，則曰王國太妃。國朝酌前代典故，從晉宋之儀，王母命為'''太妃'''」</ref>，如唐朝[[杨贵妃_(唐太宗)|趙國太妃楊氏]]，而王太后一號在中國自漢朝以後，僅於部分君主只稱王的政權所使用，如[[五胡十六國|五胡十六國]]的[[前涼|前涼]]，[[張重華|張重華]]登基以後，生母[[馬太后_(前涼)|馬氏]]、嫡母[[严王后|嚴氏]]並尊為王太后，同時為有所區別，生母馬氏稱王太后、嫡母嚴氏稱'''大王太后'''。

=== 日本 ===
日本在[[推古天皇|推古天皇]]以前，使用「大王」、「大君」、「皇尊」等作為君主稱號，因此在推古天皇以前的諸[[天皇|天皇]]，嚴格而言是國王，而非皇帝。雖然[[日本書紀|日本書紀]]記載，開國天皇[[神武天皇|神武天皇]][[皇后|皇后]][[媛蹈鞴五十鈴媛命|媛蹈鞴五十鈴媛命]]，在[[綏靖天皇|綏靖天皇]]登基後被尊為皇太后，是日本正史中最早的[[太后|太后]]，但實際上而言，一直到推古天皇以前的太后，都是王太后，而非皇太后。

=== 朝鮮半岛 ===
[[高麗王朝|高麗王朝]]時期，君主稱國王，其母稱王太后，如[[高麗穆宗|穆宗]]母[[獻哀王后|千秋王太后]]。自第25任君王[[忠烈王|忠烈王]]開始，由於高麗王朝成為[[元朝|元朝]][[附屬國|附屬國]]，使王母降稱[[太妃|王太妃]]。[[李氏朝鮮|李氏朝鮮]]在1894年從中國獨立時，重新將王太妃（[[王大妃|王大妃]]）改稱王太后，而[[明憲太后|明憲太后]]洪氏是唯一使用過這稱銜的女性，1897年[[大韓帝國|大韓帝國]]成立，洪氏就改稱[[皇太后|皇太后]]。

===越南===
[[阮朝|阮朝]]嘉隆二年（1803年），尚未称帝的[[阮福映|阮福映]]将母亲'''国母王太妃'''[[阮氏環_(阮興祖)|阮氏環]]尊为'''王太后'''。嘉隆五年（1806年），阮福映称帝，晋尊母親為皇太后。

== 其他地區 ==
現時西方國家前任國王之王后稱為Queen dowager，翻譯為王太后。如王太后是現任君主之母，又被稱為[[母后太后|母后太后]]（Queen mother）。由于东亚鲜见太上女皇或太上女王，荷兰女王[[贝娅特丽克丝_(荷兰)|贝娅特丽克丝女王]]的母亲[[朱丽安娜女王|朱丽安娜女王]]退位后被中文媒体称为王太后<ref>{{cite web |url = http://gb.cri.cn/321/2004/03/21/601@103785.htm |title=荷兰王太后朱丽安娜患肺炎逝世 享年94岁 |publisher=国际在线 |author=编辑：朱小梅|date=2004-03-21 |accessdate=2012-01-10 |language=Zh-hans}}</ref><ref>{{cite web |url = http://news.sina.com.cn/o/2004-03-21/14482103379s.shtml |title=荷兰王太后逝世 |publisher=新浪网 来源：信息时报 |author=方倩华 |date=2004年3月21日 |accessdate=2012-01-10 |language=Zh-hans}}</ref>，实际上她使用[[Princess|Princess]]头衔，应当被称为公主殿下或者女亲王殿下。
