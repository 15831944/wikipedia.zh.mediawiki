'''王慕道'''，[[臺灣人|台湾]]数学家，[[哥伦比亚大学|哥伦比亚大学]]数学教授。<ref name="profile">{{Cite web|url=http://www.math.columbia.edu/~mtwang/|title=Mu-Tao Wang Faculty Profile|accessdate=31 January 2015}}</ref>

== 教育经历 ==
1984年，王慕道进入[[國立臺灣大學|国立台湾大学]]学习，最初专业是国际商务，一年后转到数学专业。1988年，他获得了国立台湾大学学士学位。4年后，他获得了国立台湾大学硕士学位。1998年，他获得了[[哈佛大学|哈佛大学]]数学博士学位，论文题为“广义谐映射和离散群的表示”，导师是[[菲尔兹奖|菲尔兹奖得主]]、[[微分几何|微分几何学家]][[丘成桐|丘成桐]]。

== 职业生涯 ==
2001年，王慕道成为了[[哥伦比亚大学|哥伦比亚大学]]的助理教授，2009年成为教授。而在加入哥伦比亚大学之前，他是[[史丹佛大學|斯坦福大学]]的Szego助理教授。<ref name="lu">{{Cite news|url=http://news.sciencenet.cn/htmlnews/2010/12/242081.shtm|title=晨兴数学金奖得主王慕道：做数学，很开心 (Morningside Medalist Mu-Tao Wang: Happy To Be Doing Mathematics)|last=Lu|first=Qi|work=科学时报|accessdate=31 January 2015|language=zh}}</ref>

他是[[美國數學學會|美国数学学会]]会士。
