'''王行'''，字止仲，号半轩，更号楮园，自称淡如居士，是位有影响的[[苏州市|苏州市]][[文学家|文学家]]，他跟[[高启|高启]]等人一起被称为“[[北郭十友|北郭十友]]”，又称“[[明初十才子|明初十才子]]”。<ref>{{Cite web|url = http://www.subaonet.com/cul/2012/0722/964631.shtml|title = 撰写者王行被沈万三聘为家庭教师|accessdate = 2012-07-22-UTC4:00}}</ref><ref name=":0">{{Cite book|title = 明史·卷285|last = [[张廷玉|张廷玉]]等|first = |publisher = |year = |isbn = |location = |pages = |url = https://zh.wikisource.org/wiki/明史/卷285}}</ref>與明朝開國功臣[[藍玉|藍玉]]為莫逆之交，大将军礼遇甚厚。<ref>[[王锜|王锜]]《寓圃杂记》載:“偶舍于蓝都督之傍。蓝有家人子肄业于行，每归，蓝取其学课以观，重加称赏。一日，召行见，行以为奇遇，所谈皆韬略，无一语及文业。 蓝曰：‘先生文武才也，何见之晚也？’因徙置府中，以师事之。 不久蓝以谋反见执，或劝行曰：‘可行矣。’行曰：‘临难无苟免。’亦被执。 刑官无以入其罪，命刑供状。 行曰：‘行本一介书生，蒙大将军礼遇甚厚，今将举事，焉敢不从。’遂以同谋被诛。”</ref>洪武年间，被[[沈万三|沈万三]]聘为教师，<ref>{{Cite book|title = 明史·卷140|last = [[张廷玉|张廷玉]]等|first = |publisher = |year = |isbn = |location = |pages = |url = https://zh.wikisource.org/zh-hans/%E6%98%8E%E5%8F%B2/%E5%8D%B7140}}</ref>每写成一篇文章，沈万三就用一镒（20两，一说24两）银子进行酬谢。之后王行在[[石湖|石湖]]隐居。<ref name=":0" />后来，王行的两个儿子去了[[南京|南京]]，洪武二十六年[[藍玉|蓝玉]]案发，王行被殺，二子亦被杀。<ref>[[谈迁|谈迁]]《国榷》称：“乙酉太子太傅凉国公蓝玉谋大逆，族诛……坐党论死者，可二万人。 知名之士，若南海孙蕡、长洲王行，皆在诛中。”</ref>
