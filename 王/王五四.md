'''王五四'''，本名王永智<ref name="王五四，喜欢调侃现实的荒诞">{{cite web |url=http://paper.people.com.cn/hqrw/html/2016-04/16/content_1673013.htm |title=王五四，喜欢调侃现实的荒诞 |accessdate=2016-04-16 |publisher=人民网 }}</ref><ref name="王五四- China Digital Space">{{cite web |url=http://chinadigitaltimes.net/space/%E7%8E%8B%E4%BA%94%E5%9B%9B |title=王五四- China Digital Space |publisher=中国数字时代 }}</ref>，中国大陆网络大V、知名[[专栏作家|专栏作家]]，1982年生于[[山东|山东]]烟台，2005年毕业于[[山东大学|山东大学]]。因微信公众号“王五四”上的辛辣犀利的时评文章被中国大陆网友熟知<ref name="王五四，喜欢调侃现实的荒诞"/><ref name="生鲜电商如何“轻装上阵”？王五四的花样菜场选择从农贸市场入手">{{cite web |url=http://www.jiemian.com/article/822706.html |title=生鲜电商如何“轻装上阵”？王五四的花样菜场选择从农贸市场入手 |accessdate=2016-08-29 |publisher=界面新闻 }}</ref><ref name="再见，2016——中国传媒业大事评点">{{cite web |url=http://www.ftchinese.com/story/001070817?page=rest |title=再见，2016——中国传媒业大事评点 |accessdate=2017-01-13 |publisher=腾讯娱乐 }}</ref><ref name="张晓舟：互联网大IP如何捆绑鲍勃·迪伦与宋冬野">{{cite web |url=http://ent.qq.com/a/20161018/031917.htm |title=张晓舟：互联网大IP如何捆绑鲍勃·迪伦与宋冬野 |accessdate=2016-10-18 |publisher=腾讯娱乐 }}</ref>，女权研究者[[曾金燕|曾金燕]]将王五四比作微信时代的[[韩寒|韩寒]]<ref name="韩寒，“口炮党”和“送饭党”之争">{{cite web |url=https://cn.nytimes.com/opinion/20160929/letter-zengjinyan-hanhan-response/ |title=韩寒，“口炮党”和“送饭党”之争 |accessdate=2016-09-29 |publisher=纽约时报 }}</ref><ref name="China: Widow of Nobel Laureate Feared ‘Disappeared’">{{cite web |url=https://www.hrw.org/news/2017/07/20/china-widow-nobel-laureate-feared-disappeared |script-title=en:China: Widow of Nobel Laureate Feared ‘Disappeared’ |accessdate=2017-07-20 |publisher=Human Rights Watch }}</ref>。

==人物评价==

“他（王五四）把对政治人物和一些公共政策的批评隐藏在文章中，直指社会的强势的作恶，及对这些强势群体的调侃；他文字的活泼，受众阅读的快感，传播的速度，打造出一个特殊的传播案例”<ref name="时事大家谈：从妖孽四大妈看微信时代话语权之变化">{{cite web |url=https://www.voachinese.com/a/io-20150721-weixin-si-dama-right-speech/2871894.html |title=时事大家谈：从妖孽四大妈看微信时代话语权之变化 |accessdate=2016-08-29 |publisher=美国之音 }}</ref>。
——温云超<br/>
“相对于前社交媒体时代网络大V，王五四发出的是另外一种标新立异的声音，他的话语完全颠覆了以前那些大V们启蒙性的话语，也解构了体制的话语霸权”<ref name="时事大家谈：从妖孽四大妈看微信时代话语权之变化"/>。
——野渡<br/>
“从媒介、民间舆论需求的关系角度来讲，在2009年开始的新社交媒体时代，韩寒其实就是今天动辄微信公号文章阅读量过10万但随时被封号的王五四”<ref name="韩寒，“口炮党”和“送饭党”之争"/>。
——曾金燕
