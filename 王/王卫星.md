==生平==
曾在[[中国人民解放军|中国人民解放军]]的团、师、军、军区机关和部队工作。1979年参加“[[对越自卫反击战|自卫还击、保卫边疆]]”作战。1996年，调入[[中国人民解放军军事科学院|中国人民解放军军事科学院]]。曾任军事科学院台海军事研究中心主任，军事科学院世界军事研究部四室主任，军事科学院外国军事研究部部长、研究员。还曾兼任中国军事科学学会国际军事分会会长，军事科学院国家高端智库学术委员会副主任委员，[[海峡两岸关系协会|海峡两岸关系协会]]理事、全国台湾研究会常务理事、海峡两岸关系研究中心学术顾问。个人研究成果曾获国家“[[五个一工程奖|五个一工程奖]]”，首届全军政治理论研究优秀成果一等奖，全国图书“金钥匙奖”等奖项<ref name=youyi/><ref name=sanming>{{Cite web |url =http://china.caixin.com/2017-09-07/101140523.html  |title = 军事科学院领导层集中调整 三名将领履新 |author =  |date = 2017-09-07 |publisher = 财新网 |language =  |accessdate =  |quote =  }}</ref>。2017年，任新调整组建的[[中国人民解放军国防大学|中国人民解放军国防大学]]副政治委员<ref name=sanming/>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。
