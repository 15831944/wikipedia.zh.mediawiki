'''王树'''（{{bd|1958年|5月||}}），[[河北省|河北省]][[冀州市|冀县]]人<ref name=daibiao/>，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
王树是[[中国共产党|中国共产党]]党员。毕业于[[中国人民解放军南京政治学院|中国人民解放军南京政治学院]][[马克思主义|马克思主义]]理论专业，大学本科学历<ref name=daibiao/><ref name=caixin/>。曾任[[中国人民解放军总政治部宣传部|中国人民解放军总政治部宣传部]]部队教育局副局长、[[中国人民解放军总政治部|中国人民解放军总政治部]][[中国人民解放军三一一基地|三一一基地]]主任。2014年12月，接替到龄退役的[[蒋乾麟|蒋乾麟]]，任[[中国人民解放军南京政治学院|中国人民解放军南京政治学院]]院长<ref>{{cite news|title=“军中北大”南京政治学院院长易人，王树接棒蒋乾麟|url=http://www.thepaper.cn/newsDetail_forward_1307825|date=2015-03-04|newspaper=澎湃新闻}}</ref><ref name=caixin>{{Cite web |url = http://china.caixin.com/2017-10-09/101143539.html |title =曾被称“石黑手” 石忠武少将任国防大学副校长  |author =  |date = 2017-10-09 |publisher = 财新网 |language =  |accessdate =  |quote =  }}</ref>。2017年，出任新调整组建的[[中国人民解放军国防大学|中国人民解放军国防大学]]副政委<ref name=caixin/>。

2013年，当选[[第十二届全国人民代表大会|第十二届全国人民代表大会]]解放军代表<ref name=daibiao>{{cite news|title=全国人大代表信息-王树|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=122744|accessdate=2013-04-05|newspaper=全国人大网}}</ref>。
