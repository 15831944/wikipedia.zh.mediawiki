'''王怡人'''（{{lang-ko|왕이런}}，{{bd|2000年|12月29日|}}），出生於[[中华人民共和国|中国]][[浙江省|浙江省]][[杭州市|杭州市]]，[[乐华娛樂|乐华娛樂]]旗下韓國[[女子音乐组合|女子组合]][[EVERGLOW|EVERGLOW]]成员。<ref>{{cite web |title=乐华新女团EVERGLOW即将出道 |url=https://cn.yna.co.kr/view/ACK20190218001900881 |website=韩联社}}</ref>

== 经历 ==

=== 出道前 ===
王怡人是浙江杭州人。小学就读于[[杭州市文三街小学|杭州市文三街小学]]，初中就读于[[杭州市第十三中学|杭州市第十三中学]]（与同为乐华娱乐旗下艺人的[[李汶翰|李汶翰]]同校），后于初中二年級转学至[[浙江艺术职业学院|浙江艺术职业学院]]。

王怡人从小喜欢舞蹈。小学时，她参加了不少的舞蹈比赛，获得了不少成绩。上初中后，王怡人成为十三中舞蹈社团的主力，參與大大小小的舞蹈活动。其后赴韩成為乐华娱乐的练习生。

2018年6月15日，王怡人以乐华娱乐练习生身份参与由[[Mnet媒體|Mnet]]和日本AKB48集团主办的韩日合作选秀节目《[[PRODUCE_48|PRODUCE 48]]》。在《Produce 48》中，王怡人就被所有练习生评选为颜值第一名。<ref>{{cite web |title=杭州00后姑娘惊艳出道 节目里被韩国人选为颜值第一 |url=http://zj.sina.com.cn/news/s/2019-02-28/detail-ihsxncvf8476510.shtml |website=新浪}}</ref>
