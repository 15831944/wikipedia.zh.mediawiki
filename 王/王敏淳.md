'''王敏淳'''（{{bd|2001年|1月10日}}），台灣女藝人，藝名'''香奈兒'''，2017年出道，是藝人[[檢場|檢場]]、歌手[[李翊君|李翊君]]的女兒，擅長演奏樂器為[[長笛|長笛]]；目前就讀高二，本就讀[[臺北市私立立人國際國民中小學|臺北市私立立人國際國民中小學]]，後轉學就讀[[新北市|新北市]]新店[[康橋雙語學校|康橋雙語學校]]秀岡校區。於2018年9月參加跳級考試轉學至美國[[伯克利音樂學院|伯克利音樂學院]]，目前就讀中。

== 關於藝名 ==
王敏淳表示，香奈兒這個藝名是外婆取的，剛出生時在想中文名字時，本來要讓外婆想的，結果竟然出現「鳳珠」之類的名字，媽媽一聽覺得不太可行，委婉拒絕了外婆想的名字。由於外婆看起來很失望，為了滿足外婆取名的願望，「在想英文名字的時候，就拜託外婆來想。」沒想到外婆認識的英文單字不多，就脫口而出乾脆叫「香奈兒」，讓爸媽聽了啼笑皆非。<ref>[http://ctweekly.chinatimes.com/20170911004222-261902 檢場女兒闖演藝圈 阿嬤賜名香奈兒 2017年09月11日 16:32  蕭雅玲、實習記者林宛樺 中時電子報]</ref>

== 簡歷 ==
2014年7月3日，王敏淳與父母檢場、李翊君上[[中天綜合台|中天綜合台]]、[[中天娛樂台|中天娛樂台]]節目《[[SS小燕之夜|SS小燕之夜]]》。李翊君在節目中笑說，女兒反而不聽她的歌，對於音樂她有自己的獨特見解，擅長長笛、[[聲樂|聲樂]]，音樂架勢完全不輸天后媽媽。<ref>[http://enjoy.eastday.com/e/20140703/u1a8194344.html 台天后李翊君女兒照片曝光被稱酷似周冬雨  2014-7-3 15:42:38 来源:网易娱乐 选稿:王静瑶 东方网 >> 娱乐频道 >> 滚动新闻 >> 正文]</ref>

2017年3月7日，王敏淳在[[Instagram|Instagram]]曬出與[[歐陽娜娜|歐陽娜娜]]擁抱[[親吻|親吻]]照片，「女女吻」照掀起不少人熱議。對此李翊君表示2人是好閨蜜，小女生鬧著玩而已。[[台北市議員|台北市議員]][[歐陽龍|歐陽龍]]接受訪問時也回應：「每個人都是獨立個體，尊重她們的決定。」<ref>[http://www.setn.com/News.aspx?NewsID=230993 爆蕾絲情！歐陽娜娜擁吻香奈兒　星爸媽：喜歡男女都會尊重 2017/03/07 08:56:00 娛樂中心／綜合報導 三立新聞網]</ref>

2017年8月4日，王敏淳正式出道。她在6月接下台視、東森新戲《[[我和我的四個男人|我和我的四個男人]]》，當天出席首映會，將在戲中捉弄李毓芬，單戀嚴爵。製作人[[王佩華|王佩華]]笑說，她的聲音很清亮，就是想要的那種天真無害的感覺。不過，講到試鏡過程，父母互動也笑翻她。王敏淳表示，謝謝劇組大家都很照顧她，也因從小耳濡目染，對於演藝圈一切感到很新鮮。<ref>[https://star.ettoday.net/news/981983 檢場16歲女兒香奈兒出道！看歐陽娜娜「輕度肥胖」說  ETtoday新聞雲  › ETtoday星光雲  › 電視 2017年08月04日 17:42  星光雲  記者林思妤／台北報導]</ref>
