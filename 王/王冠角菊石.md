'''王冠角菊石'''（[[學名|學名]]：{{lang|la|'''''Corongoceras'''''}}）是生存在早[[白堊紀|白堊紀]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]，其[[化石|化石]]被發現於[[南美|南美]]、[[南歐|南歐]]、[[尼泊爾|尼泊爾]]和[[古巴|古巴]]等地。<ref name="Fossilworks" />

== 特徵 ==
王冠角菊石和[[布蘭佛德菊石|布蘭佛德菊石]]類似，殼體兩側[[Y|Y]]字形分叉的肋條在腹側並不相連，而形成一道縱向的溝，可見兩者的親緣關係很近，但是王冠角菊石的殼體略寬，且殼側約一半的位置（即肋條分叉處）有一個明顯的結節，有些標本上還能見到保存良好的棘刺，是本屬和布蘭佛德菊石屬最大的不同之處。<ref name="Berriasella">{{Cite web |url = http://liusir.pixnet.net/blog/post/248633579-corongoceras-%E7%8E%8B%E5%86%A0%E8%A7%92%E8%8F%8A%E7%9F%B3-%28%E5%B0%BC%E6%B3%8A%E7%88%BE%29 |title = Corongoceras 王冠角菊石 (尼泊爾) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-07-21 }}</ref>
