'''王重盈'''（{{bd|9世纪||895年|2月12日|catIdx=W}}<ref name=ZZTJ260>[[:s:資治通鑑/卷260|《资治通鉴》卷二百六十]]</ref>），[[晚唐|晚唐]][[军阀|军阀]]，历任[[陕虢节度使|陕虢节度使]]和[[护国节度使|护国节度使]]。

== 家世 ==
王重盈是太原府人。<ref>《旧唐书》作河中府人。此从《新唐书》、司空图《故盐州防御使王纵追述碑》。</ref>父王纵，官至盐州防御使。因父有功，王重盈与弟[[王重荣|王重荣]]得其荫补，历任军职。兄弟俩都以骁雄毅武闻名军中。<ref name=BT182>[[:s:舊唐書/卷182|《旧唐书》卷一百八十二]]</ref><ref name=NBT187>[[:s:新唐書/卷187|《新唐书》卷一百八十七]]</ref>王重盈、王重荣分别是王纵的第三和第四子，上有兄长[[王重章|王重章]]、[[王重简|王重简]]，下有幼弟[[王重益|王重益]]。<ref>《故盐州防御使王纵追述碑》、司空图《蒲帅燕国太夫人石氏墓志铭》</ref>

== 早期仕途 ==
王重盈约[[乾符|乾符]]年间曾任[[汾州|汾州]][[刺史|刺史]]。<ref>[[郁贤皓|郁贤皓]]《[[唐刺史考全编|唐刺史考全编]]》</ref>[[广明_(唐朝)|广明]]元年（880年）四月，[[李琢|李琢]]受任充蔚朔等州诸道行营都招讨使，讨伐[[沙陀|沙陀]]叛军，王重盈兵马归李琢统属。<ref name=BT19-2>[[:s:旧唐书/卷019下|《旧唐书》卷一十九下]]</ref>二年（881年），民变首领[[黄巢|黄巢]]的军队渡过淮河之际，僖宗提拔王重盈为[[陕虢观察使|陕虢观察使]]。<ref name=NBT187/>后来京城[[长安|长安]]落入黄巢之手，迫使[[唐僖宗|唐僖宗]]逃到[[成都|成都]]。黄巢称帝，建国号齐。时王重荣任[[河中节度使|河中节度使]]。[[中和_(唐朝)|中和]]二年（882年）正月，僖宗任[[宰相|宰相]][[王铎_(唐朝)|王铎]]充诸道行营都都统，作为讨伐黄巢的总帅，任王重盈为王铎的东面都供军使。<ref>[[:s:資治通鑑/卷254|《资治通鉴》卷二百五十四]]</ref>这年，王重盈奏授[[卢质|卢质]]为芮城令。<ref>[[:s:旧五代史/卷093|《旧五代史》卷九十三]]</ref>次年，唐军败齐军，迫黄巢弃长安后，陕虢军被升为节度使，王重盈被任为节度使。<ref>[[:s:資治通鑑/卷255|《资治通鉴》卷二百五十五]]</ref>[[光启|光启]]元年（885年），僖宗加王重盈宰相衔[[同中书门下平章事|同中书门下平章事]]。<ref name=ZZTJ256>[[:s:資治通鑑/卷256|《资治通鉴》卷二百五十六]]</ref>王重盈曾在陕虢节度使任上加检校左仆射。<ref>《故盐州防御使王纵追述碑》</ref>

== 就任护国军 ==
后来河中节度使改名护国节度使。<ref name=ZZTJ256/>王重荣在护国军期间，为政日益严酷。光启三年（887年），他处罚了牙将常行儒，六月，常行儒兵变杀王重荣，<ref name=BT182/><ref name=NBT187/>推立王重盈为兵马留后，<ref name=BT19-2/><ref>[[:s:新唐書/卷009|《新唐书》卷九]]</ref><ref name=NHFD42>[[:s:新五代史/卷42|《新五代史》卷四十二]]</ref><ref>《旧五代史·王珂传》《新五代史·王珂传》《[[册府元龟|册府元龟]]》误作王重盈为王重荣弟。</ref>[[河东节度使|河东节度使]][[李克用|李克用]]也上表请求以王重盈继任。<ref>[[:s:旧五代史/卷025|《旧五代史》卷二十五]]</ref>僖宗下诏任命王重盈继任，<ref name=HFD14>[[:s:旧五代史/卷014|《旧五代史》卷一十四]]</ref>又任王重盈子[[王珙|王珙]]权知陕虢留后。王重盈就任护国军后，捕杀常行儒及其党羽，安抚军民。<ref name=ZZTJ257>[[:s:資治通鑑/卷257|《资治通鉴》卷二百五十七]]</ref>

[[河阳节度使|河阳节度使]][[李罕之|李罕之]]持续劫掠地方，[[文德_(唐朝)|文德]]元年（888年）二月，全力攻[[绛州|绛州]]，绛州刺史王友遇以城降，李罕之又攻击[[晋州|晋州]]，王重盈求援于[[宣武军节度使|宣武军节度使]][[朱全忠|朱全忠]]，李罕之解围而退。王重盈使反间计使得李罕之盟友河南尹[[张全义|张居言]]背叛李罕之。当年春，李罕之全力攻晋州，张居言夜袭河阳，俘李罕之一家，迫李罕之出奔，自己占领了河阳。<ref name=ZZTJ257/><ref>[[:s:旧五代史/卷015|《旧五代史》卷一十五]]</ref>[[大顺_(唐朝)|大顺]]元年（890年）十二月，[[唐昭宗|唐昭宗]]讨伐李克用兵败，晋州、绛州都被河东军攻取，河东军劫掠河中，赤地千里。昭宗寻求和解，[[左仆射|左仆射]][[韦昭度|韦昭度]]上表赞成，并建议也对王重盈等进行谕旨慰勉，获准。<ref name=BT20-1>[[:s:旧唐书/卷020上|《旧唐书》卷二十上]]</ref>二年（891年）八月，唐昭宗加王重盈加检校[[太傅|太傅]]及更大的宰相衔兼[[中书令|中书令]]。<ref name=ZZTJ258>[[:s:資治通鑑/卷258|《资治通鉴》卷二百五十八]]</ref>[[景福_(唐朝)|景福]]元年（892年）正月，护国军治下五郡联属请求为王重盈建生祠。五月，都押衙录事参军又奏请立碑纪功。于是[[司空图|司空图]]奉令作《太尉琅琊王公河中生祠碑》。<ref>司空图《太尉琅琊王公河中生祠碑》</ref>

[[乾宁|乾宁]]二年（895年）正月，王重盈卒，其时官爵为河中节度使、检校[[太师|太师]]、中书令、河中尹、[[上柱国|上柱国]]、'''琅邪郡王'''。其兄王重简子[[王珂_(唐朝)|王珂]]为王重荣养子，时任护国军行军司马，护国军士兵拥戴他为知留后事。<ref>[[:s:新唐書/卷010|《新唐书》卷十]]</ref>当时陕虢军已改名保义军，时任保义军节度使的王珙也想领护国军，从而引发了王珂、王珙的继承之争。<ref name=HFD14/><ref name=BT20-1/><ref>[[:s:旧唐书/卷179|《旧唐书》卷一百七十九]]</ref>因王珂准岳父李克用支持王珂，昭宗任王珂为护国军节度使。此举导致支持王珙继任的[[凤翔节度使|凤翔节度使]][[李茂贞|李茂贞]]、[[静难节度使|静难节度使]][[王行瑜|王行瑜]]和[[镇国军节度使|镇国军节度使]][[韩建|韩建]]不满，导致了当年五月的三帅逼宫事件。<ref name=ZZTJ260/><ref name=NHFD42/><ref>[[:s:旧五代史/卷026|《旧五代史》卷二十六]]</ref><ref>[[:s:新五代史/卷04|《新五代史》卷四]]</ref><ref>[[:s:新五代史/卷40|《新五代史》卷四十]]</ref>
