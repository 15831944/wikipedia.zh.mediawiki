'''王亚平'''{{BD|1980年|1月|||CatIdx=W王}}<ref>[http://www.apdnews.com/news/28020.html 神十将于11日17时发射 王亚平成第二位女航天员] {{webarchive|url=https://web.archive.org/web/20130613055245/http://www.apdnews.com/news/28020.html |date=2013-06-13 }}，亚太日报，2013年6月13日</ref>，女，[[汉族|汉族]]，[[中华人民共和国|中国]][[山东省|山东]][[烟台市|烟台]][[福山区|福山区]][[张格庄|张格庄]]人，中国第二位执行航天飞行任务的女[[航天员|航天员]]。

1986年9月，入张格庄中心完全小学就学{{NoteTag|山东小学当时五年制，2000年代初各地逐渐转六年制。}}，1991年9月，入烟台二十三中读初中，1994年，入烟台福山一中读高中，1997年高中毕业<ref>{{Cite news |url = http://www.wenxuecity.com/news/2013/06/12/2451340.html |title = 神舟十号女航天员王亚平大量旧照曝光 |publisher=[[文学城|文学城]] |date=2013-06-12}}</ref>，考入[[中国人民解放军空军长春飞行学院|中国人民解放军空军长春飞行学院]]，8月入伍。王亚平是中国第七批女飞行员，空军[[上校|上校]]军衔，能飞四种机型。为[[武汉|武汉]]空军[[运输机|运输机]]飞行员，曾驾机参加过多次战备演习、[[汶川大地震|汶川抗震救灾]]、[[北京奥运会|北京奥运会]]消云减雨等重大任务<ref>{{Cite news |url = http://www.chinanews.com/gn/2013/06-10/4918988.shtml |title = 新闻背景：神舟十号唯一女航天员王亚平 |publisher=[[中新网|中新网]] |date=2013-06-10}}</ref><ref>{{Cite news |url = http://www.cnr.cn/2012zt/sj/news/201206/t20120613_509892104.html |title = 探访神九女航天员刘洋王亚平 揭秘少为人知的成长故事 |newspaper=《[[解放日报|解放日报]]》 |publisher=[[中国广播网|中国广播网]]|date=2012-06-13}}</ref>。安全飞行1600小时，被评为空军二级飞行员。2010年5月，正式成为中国第二批航天员<ref name="cv">{{Cite news |url = http://news.xinhuanet.com/mil/2013-06/10/c_116111448.htm |title=王亚平同志简历 |newspaper= |publisher=[[新华网|新华网]] |date=2013-06-10}}</ref>。

2012年6月16日，[[神舟九号|神舟九号]]飞船将搭载首位女航天员发射升空，王亚平就是候选人之一，编入第二梯队，但最终第一梯队的[[刘洋_(航天员)|刘洋]]成为中国首位女航天员。2013年6月11日发射的[[神舟十号|神舟十号]]飞船搭载王亚平升空，使她成为中国第二位女航天员，被誉为“神女二号”，同时也是中国首位[[80后|80后]]飞天宇航员<ref>{{cite news |url = http://www.chinanews.com/gn/2013/06-15/4930408.shtml |title=王亚平领衔中国航天“80后” 太空授课将引关注 |publisher=中国新闻网 |date=2013-06-15}}</ref>。

== 年龄疑问 ==
[[新华网|新华网]]最新的官方简历显示，王亚平的出生年月为1980年1月<ref name="cv" />。[[中国中央电视台|央视]]在神十发射后的报道中，也多次强调王亚平是80后<ref>{{cite web |title = “神舟十号”发射倒计时 王亚平：中国“80后”女航天员 |url = http://news.cntv.cn/2013/06/10/VIDE1370877359137973.shtml |publisher=央视网 |accessdate=2013-06-13}}</ref><ref>{{cite web |title = 女航天员王亚平：第一个飞向太空的80后 |url = http://yueyu.cntv.cn/20130611/102190.shtml |publisher = 央视网 |accessdate = 2013-06-13 |deadurl = yes |archiveurl = https://web.archive.org/web/20141026071003/http://yueyu.cntv.cn/20130611/102190.shtml |archivedate = 2014-10-26 }}</ref>。

但[[中国军网|中国军网]]、[[央视网|央视网]]、[[网易|网易]]新闻等媒体曾经报道，她的出生年月为1978年4月。不过在这些媒体的资料内容中，她仍然是1997年[[高考|高考]]后进入空军长春飞行学院<ref name="chinamil_1978birth">{{cite news |url = http://www.chinamil.com.cn/site1/ztpd/2005-02/04/content_133201.htm|title=中国空军第七批女飞行员写真 |publisher=[[中国军网|中国军网]] |date=2005-02-04|archiveurl=https://web.archive.org/web/20060831011530/http://www.chinamil.com.cn/site1/ztpd/2005-02/04/content_133201.htm|archivedate=2006-08-31}}</ref><ref>{{cite news |url = http://news.cntv.cn/2013/06/10/ARTI1370848370322292.shtml |title = 神舟十号飞行乘组由聂海胜、张晓光和王亚平组成 |publisher=[[中国网络电视台|中国网络电视台]] |date=2013-06-10}}</ref><ref>{{cite news |url = http://news.163.com/12/0611/09/83N5T34300014JB5.html |title = 资料:神舟九号女宇航员候选人王亚平(图/简历) |publisher=[[网易|网易]] |date=2013-06-11}}</ref>。

2013年6月26日，航天员科研训练中心主任邓一兵回应此事，表示王亚平出生年月确系1980年1月，此前信息为媒体报道失实<ref>{{cite news|title=官方回应对王亚平出生年月质疑:确系1980年1月|url=http://news.sina.com.cn/c/2013-06-26/114527502723.shtml|accessdate=2016-10-23|agency=中央电视台|publisher=新浪新闻|date=2013-06-26}}</ref>。
