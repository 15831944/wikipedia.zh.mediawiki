'''王辉青'''（{{bd|1960年|||}}）<ref name=wenti>王辉青，关于军事系统研究的方法论问题，军事运筹与系统工程2013,27(2):10-15</ref>，[[山东省|山东省]][[烟台市|烟台市]][[牟平区|牟平区]]人，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
王辉青是研究员、博士、[[博士生导师|博士生导师]]，主要研究方向是军事战略、作战实验<ref name=wenti/>。曾任[[中国人民解放军军事科学院|中国人民解放军军事科学院]]军事运筹分析研究所所长<ref>{{Cite web |url = http://news.cnwest.com/content/2011-11/20/content_5565590.htm |title = 军科院转变科研方式 5年完成650多项重大研究成果 |author =  |date =2011-11-20  |publisher =西部网  |language =  |accessdate =  |quote =  }}</ref>。2012年任中国人民解放军军事科学院战争理论和战略研究部副部长<ref>{{Cite web |url = http://news.163.com/12/0913/09/8B98HB0F00014JB6.html |title = 第四届海峡两岸（滨州）孙子文化论坛开幕 |author =  |date = 2012-09-13 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。同时兼任军事系统工程专业委员会副主任委员<ref name=wenti/>。后任[[中国人民解放军战略规划部|中国人民解放军战略规划部]]部长。2016年，任新组建的[[中央军委战略规划办公室|中央军委战略规划办公室]]主任<ref>{{Cite web |url =http://www.sohu.com/a/112484858_260616  |title = 长安街知事:各就各位!军委机关15部门主官全部亮相 |author =  |date = 2016-08-28 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。
