'''王姓'''是一个[[汉族|汉族]]姓氏，在[[中國|中國]]《[[百家姓|百家姓]]》中排第8位。根据[[中华人民共和国国家统计局|中华人民共和国国家统计局]]2010年[[第六次全国人口普查|第六次全国人口普查]]数据分析显示：王姓为中国大陆第一大姓，有大约9468万人，占中国大陆人口总数的7.10%<ref>{{cite journal|author=武洁、杨建春|title=张、王、李、赵谁最多——2010年人口普查姓氏结构和分布特点|journal=中国统计|volume=|issue=总390期|pages=21-22|publisher=中国统计出版社|url=http://mp.weixin.qq.com/s?__biz=MjM5Njg5MjAwMg==&mid=200354543&idx=1&sn=29de013ce1896a8e78f9c0d774cd528a&scene=3#rd|accessdate=2016-12-29|date=2014年6月}}</ref>。根據2010年7月1日[[中華民國|中華民國]][[中華民國內政部|內政部]]戶政司戶籍資料統計，王姓在台灣排名第六大姓。根據1977年中國史學家李棟明，在《東方雜誌》發表的一篇有關「姓」的論文上指出，王姓是全球華人十大姓之一。<ref>根據1977年中國史學家[[李棟明|李棟明]]，在《東方雜志》發表的一篇有關「姓」的論文上指出，王姓是全球[[華人|華人]]十大姓之一。</ref>另外，王姓亦是[[日本|日本]]单姓之一，但极为罕见。

由于在多種[[汉语方言|汉语方言]]中，“王”与“[[黄姓|黄]]”的发音相近甚至相同。为区别两姓起見，中文多以“三畫（横）王”或“三横一竖王”，黃姓則叫「草頭黃」

== 来源和演变 ==
===漢族王姓===
王姓的来源有多个，但构成现代[[漢族|漢族]]王姓的主要来源有四个：[[姬姓|姬姓]]、[[子姓|子姓]]、[[妫姓|妫姓]]和[[少數民族|少數民族]]改姓。
==== 姬姓王氏 ====
最多的王姓来源于[[周朝|周朝]]的[[王族|王族]]（不过这一说法并无可靠消息佐证），周朝的王族本姓[[姬姓|姬]]，但从这个家族（以及周的分封国中姬姓的王族中）中不断有一些人由于失势或亡国而分离出来，因为他们过去是属于王家的因此以王为姓。这一支王姓以「[[王子乔|王子乔]]」为始祖。

根据史书记载，[[周武王|周武王]]灭掉[[商朝|商朝]]以后，定都于[[镐京|镐]]，历史上称为[[西周|西周]]。传二十一世至[[周灵王|周灵王]]（前571年-前545年在位），国都在[[成周|成周]]，即今河南[[洛阳|洛阳]]。[[周灵王|周灵王]]的儿子[[王子乔|太子晋]]（也被称为王子晋或王子侨），因为直谏被废为平民。其子[[宗敬|宗敬]]仍在朝中任[[司徒|司徒]]之职，当时的人们因其是王族的后代，便称其为“王家”。从此，这支族人就以“王”为姓。传到第八代孙[[王错|王错]]拜了[[魏国_(战国)|魏国]]将军，姬姓王氏又重新贵显。[[先秦|先秦]]时期，这支王姓一直活跃于河南[[洛阳|洛阳]]一带。秦末[[汉朝|漢]]初，[[秦国|秦]]名將[[王翦|王翦]]的後代，武城侯[[王离|王离]]之子[[王元_(秦朝)|王元]]和[[王威|王威]]，为了躲避战乱，分别迁徙到[[山东|山东]][[琅琊|琅琊]]和[[山西|山西]][[太原|太原]]，最终发展成了最为著名的[[琅琊王氏|琅琊]]和[[太原王氏|太原]]两大王姓望族。
*[[琅琊王氏|琅琊王氏]]和[[太原王氏|太原王氏]]，从三国开始至唐末在中国历史大放异彩。
*[[太原王氏|太原王氏]]又分为[[晋阳王氏|晋阳王氏]]和[[祁县王氏|祁县王氏]]，在三国时代[[王允|王允]]、[[王凌|王凌]]的[[祁县王氏|祁县王氏]]，和[[王昶_(三國)|王昶]]的[[晋阳王氏|晋阳王氏]]已经有区别。
*[[祁县王氏|祁县王氏]]的王神念，王僧辩这一支和王思政在唐朝都有女儿做到皇后，嫔妃。
*[[晋阳王氏|晋阳王氏]]的[[王慧龙|王慧龙]]在唐朝更发展成为[[七姓十家|七姓十家]]，在朝廷被禁止互相婚嫁。
*[[开闽王氏|开闽王氏]]的始祖王潮等三兄弟来自[[琅琊王氏|琅琊王氏]]。

==== 子姓王氏 ====
{{or|最古老的王姓出自[[子姓|子姓]]。传说在[[商朝|商朝]]末年，[[商纣王|商纣王]]的叔父[[比干|比干]]与[[箕子|箕子]]、[[微子|微子]]一起被称为商末“三仁”。纣王荒淫无道，比干多次犯颜上谏，反遭到纣王杀害，其子孙因为是王族的后裔，所以就以“王”为姓，被称为“子姓王氏”。子姓王氏的历史至今已经有约3100年了。经历了从[[秦朝|秦朝]]一直到[[唐朝|唐朝]]，再到今天。期间子姓王氏一直居住在[[河南|河南]]地区，形成著名的[[汲郡|汲郡]]王姓望族，后来散播到了[[甘肃|甘肃]]、[[山东|山东]]、[[河北|河北]]和[[山西|山西]]等地。}}

==== 妫姓王氏 ====
奉[[虞|虞]][[舜|舜]]为祖先的妫姓王氏是王姓中重要的一支。[[战国七雄|战国七雄]]之一的[[田齐|齐国]]被秦国所灭之后，[[项羽|项羽]]封齐王[[田建|田建]]的长孙[[田安|田安]]为济北王，随后项羽被[[刘邦|刘邦]]所灭，田安也随之失去了王位，子孙遂改姓王。这支王姓以[[北海郡|北海郡]]为郡望。
*新朝王莽就是属于这一支。
*五胡十六国，苻坚的丞相王猛也属于这一支。

==== 外族改姓 ====
現代漢族融入了古代[[游牧民族|游牧民族]]，还有许多后来改姓或赐姓王的，比较著名的有：

[[兩漢|兩漢]]时的[[匈奴人|匈奴人]]、[[西羌|西羌]][[钳耳|钳耳]]氏、[[南北朝|南北朝]]时候的[[高句丽|高句丽]][[拓王|拓王]]氏、[[鲜卑人|鲜卑人]]的[[乌桓人|乌桓人]]、[[隋唐|隋唐]]时的[[西域|西域]][[月氏国|月氏国]]的[[胡人|胡人]]、[[唐代|唐代]][[回鹘|回鹘]][[阿布思|阿布思]]氏、[[契丹人|契丹人]]、11世紀—12世紀初葉时[[女真人|女真人]]中的[[完颜|完颜]]氏、[[夹谷|夹谷]]氏、[[北宋|北宋]][[西夏|西夏]][[党项人|党项人]]。

这些外族王姓随着时间的推延和经过[[通婚|通婚]]等，多数同化为[[中国|中国]][[中国北方|北方]]的[[汉族|汉族]]王姓。

王姓在[[朝鲜语|朝鲜语]]中写作“'''왕'''”。据统计，[[大韩民国|韩国]]在2000年时，有王姓人口23447人。[https://web.archive.org/web/20060511021345/http://kosis.nso.go.kr/cgi-bin/sws_999.cgi?ID=DT_1INOOSB&IDTYPE=3&A_LANG=1&FPUB=3&ITEM=%20&CLASS1=S.000]

“王”曾经是[[朝鲜半岛|朝鲜半岛]]上最大的一个姓氏。《[[后汉书|后汉书]]》记载，朝鲜半岛的王姓始祖是[[公元前2世纪|公元前2世纪]]从中国山东（当时的[[田齐|齐国]]）迁到朝鲜的[[王仲|王仲]]与后来的[[王仁|王仁]]。

虽然“王”是朝鲜半岛历史上[[高麗_(918年－1392年)|高丽]]王室的姓，但今天已经不多见了。据说，当[[高丽王朝|高丽王朝]]灭亡之后，那些姓王的人们为了避免[[朝鮮王朝|朝鮮王朝]]的严重迫害而改姓其他的姓了，各自改為[[玉姓|玉姓]]、[[金姓|金姓]]、[[全姓|全姓]]、[[琴姓|琴姓]]、[[田姓|田姓]]、[[申姓|申姓]]、[[車姓|車姓]]、[[姜姓|姜姓]]、[[馬姓|馬姓]]、[[龍姓|龍姓]]、[[柳姓|柳姓]]、[[乃姓|乃姓]]。在[[开城|开城]]王氏的家谱中至今仍可以找到高丽统治者的部分痕迹。

== 迁徙和分布演变 ==
===漢族===
王姓在[[先秦|先秦]]、[[兩汉|兩汉]]、[[魏晋|魏晋]]时期一直以[[华北|华北]]地区为主要的活动地区，发展十分迅速。[[隋|隋]][[唐|唐]]时期王姓各派除了向内地迁徙之外，最主要的事件是[[河南|河南]][[固始|固始]]人[[王审知|王审知]]南迁到[[福建|福建]]，建立了[[闽国|闽国]]，成为[[五代十国|五代十国]]之一，是福建王姓的总开山鼻祖，历史上被称为开闽王氏。

==== 宋辽金元 ====
[[宋代|宋代]]时，王姓人口平均约590万，约占当时全宋朝人口的7.7%（是时[[南宋|南宋]]国境内人口约为8060万），为宋代第一大姓。王姓的第一大省是[[山东省|山东省]]，约占全国王姓人口的14.7%，占山东省总人口的12.7%。在宋朝的分布主要集中在山东、[[河北|河北]]、[[四川|四川]]、[[河南|河南]]、[[山西|山西]]，这五省的王姓约占宋朝王姓总人口的55%。其次分布于[[江西|江西]]、[[陕西|陕西]]、[[浙江|浙江]]，这三省的王姓又集中了25.5%。宋朝形成了[[陕|陕]][[晋|晋]][[冀|冀]][[豫|豫]][[鲁|鲁]]、[[赣|赣]]、[[浙|浙]]以及[[蜀|蜀]]三大块王姓人口的聚集地区。

==== 明 ====
[[明代|明代]]时，王姓人口平均约620万，约占明朝人口的6.7%，仍然为第一大姓（明代的人口由6500多万至1.4億不等）。[[宋朝|宋]]—[[明|明]]600年间，人口的净增长率为20%，王姓人口的增长比明朝人口的增长速度要缓慢得多，600年间王姓人口总增长仅为5%，净增加30万。

王姓在明朝的分布主要集中于长江中下游地区和[[山东半岛|山东半岛]]。[[江苏|苏]][[浙江|浙]][[山东|鲁]]三省王姓大约占王姓总人口的40%。其次分布于[[江西|江西]]和[[山西|山西]]，这两省又集中了17%。[[浙江|浙江]]一跃成为王姓第一大省。[[中国南方|中国南方]]王姓在这600年间有了发展，[[广东|广东]]、[[广西|广西]]、[[福建|福建]]的王姓人口已经大大超过了宋朝。而[[靖康之禍|靖康之禍]]后的王姓大省[[河北|河北]]、[[河南|河南]]和[[四川|四川]]，在当时的比率已经下降了近一半。宋元明时期，王姓的分布总格局变化比较大，其人口主要向东南方向迁移，而四川地区王姓人口主要向[[湖北|湖北]]、[[湖南|湖南]]、[[广东|广东]]、[[广西|广西]]等方向迁移。明朝范围内又重新形成了赣浙苏皖和鲁豫晋冀两大块王姓人口聚集的地区。

==== 当代 ====
在[[中國|中國]]，王姓人口已经达到了9288.1万，为中国大陆第一大姓，约占全部人口的7.25％（其他三大姓氏依次为[[李姓|李]][[张姓|张]][[刘姓|刘]]）。从明至今的600年间，王姓人口由620万穩定增長到近9288.1万，接近14倍。明代的平均人口接近9300万，当代中國人口如果按12亿计算的话，增长了近13倍。王姓人口的增长速度略高于中國人口的增加速度。近1000年中，王姓人口增长率曲线呈现出“V”字型态势。

目前，王姓人口在中国大陆的分布主要集中于[[河南|河南]]、[[山东|山东]]、[[四川|四川]]、[[河北|河北]]四省，大约占王姓总人口的36%。其次分布于[[辽宁|辽宁]]、[[安徽|安徽]]、[[江苏|江苏]]、[[黑龙江|黑龙江]]、[[浙江|浙江]]、[[陕西|陕西]]和[[湖北|湖北]]，这七省的王姓又集中了33%。河南居住了王姓人口的11.5%，为王姓第一大省，占省总人口的10%。以[[长江|长江]]为界，形成了高比率北方王姓区和低比率南方王姓区，而北方区又以[[太行山|太行山]]为界，东部为高密度王姓区，西部为低密度王姓区。在近600年间，中國的王姓人口流动的程度和方向与[[宋朝|宋]][[金朝|金]][[元朝|元]][[明|明]]期间有很大区别，由东部向[[华中|华中]]、华北的回迁成为主流。

在台灣，王姓為第六大姓，僅次於[[陈姓|陳]]（250萬人）、[[林姓|林]]（188萬人）、[[黄姓|黃]]、[[张姓|張]]、[[李姓|李]]等姓。然而，台灣基於歷史上的漢化改姓政策，王姓不見得都具有漢族血統。

=== 分布特点 ===
==== 分布密度 ====
在[[中國|中國]]，王姓人口密度最高的地区在[[黑龙江|黑龙江]]、[[吉林省|吉林]]、[[辽宁|辽宁]]、[[内蒙古|内蒙古]]东部、[[河北|河北]]东部、[[山东|山东]]、[[江苏|江苏]]北部、[[安徽|安徽]]东北部。这些地区每平方公里的王姓人口达21人以上，中心地区达30人以上。每平方公里21以上的地区占中國国土总面积的12.3%，王姓人口约3007万。每平方公里12.6-21人的地区占中國国土总面积的19.9%，王姓人口约2904万。每平方公里4.2-12.6人的地区占中國国土总面积的33.8%，王姓人口约2660万。每平方公里4.2人以下的地区占中國国土总面积的34%，王姓人口约321万。

==== 分布频率 ====
王姓人口在[[中國|中國]]的分布很广，但不均衡。

[[黑龙江|黑龙江]]、[[吉林省|吉林]]、[[辽宁|辽宁]]、[[内蒙古|内蒙古]]东部和中部、[[河北|河北]]东部、[[山东|山东]]、[[河南|河南]]北部、[[新疆|新疆]][[喀什|喀什地区]]、[[海南|海南]]是王姓最常见的地区，其覆盖面积只有中國国土总面积的19.8%，王姓占当地人口的比例在9.2%以上，有的高达16.2%。

[[长江|长江]]以北和[[青海|青海]][[西宁|西宁]]以东的其他地区，也是王姓的常见区域，王姓在人群中的比例在6.9%-9.2%之间，覆盖面积是中國国土总面积的19%。

[[长江|长江]]以南及[[新疆|新疆]]东北部和西部，是王姓人口相对较少的地区，王姓在人群中的频率在2.3%-6.9%之间，覆盖面积是中國国土总面积的34.6%。

[[青海湖|青海湖]]以西的其他地区、[[广东|广东]]西部、[[广西|广西]]东部、是王姓分布稀少的地区，王姓在人群中的频率不足2.3%，覆盖面积是中國国土总面积的26.6%。

== 其他民族中的王姓 ==
=== 台灣原住民王姓 ===
{{see also|台灣原住民命名文化}}
臺灣王姓部分不見得具有漢族血統：例如[[台灣原住民|原住民]]時因[[國民政府|國民政府]]的漢化政策改姓。這使在臺灣其姓氏亦在非漢族群體中很常見。

=== 苗族王姓 ===
在[[苗语|苗文]]中“王”写作“'''Vang'''”、或“'''Vaj'''”或“'''Vaaj'''”。它和汉语中的王姓来自于同一个词源，而且是一个在[[苗族|苗族]]中很常见的姓。王姓氏族是苗族里最大的十八个氏族之一。

== 王姓的羅馬化 ==
汉语的“'''王'''”在翻译成[[英语|英语]]时，按照普通话汉语拼音、广东话音译分别被译为“'''Wang'''”或“'''Wong'''”。

== 名人 ==
王姓是著名的大姓之一，从古到今涌现出不少名人。《[[中国人名大辞典|中国人名大辞典]]》中收入了历代王姓名人共计2360名，占名人总数的5.19%，排在中國名人姓氏第一位；王姓的著名[[文学家|文学家]]占中国历代文学家总数的6.86%，排在第一位；王姓的著名[[医学家|医学家]]占中国历代医学家总数的5.37%，排名第一位。
