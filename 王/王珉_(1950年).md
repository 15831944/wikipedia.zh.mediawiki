'''王珉'''（{{bd|1950年|3月30日|catIdx=W王}}），[[安徽省|安徽]][[淮南市|淮南]]人。1968年9月参加工作，1985年7月加入[[中国共产党|中国共产党]]。[[南京航空航天大学|南京航空学院]]（今[[南京航空航天大学|南京航空航天大学]]）机械制造专业博士研究生毕业；曾长期从事专业领域教育工作，被普遍认为是少数具有真正高学历的高级官员。[[中国共产党第十八届中央委员会委员列表|中共第十七届]]、[[中国共产党第十八届中央委员会委员列表|十八届中央委员]]。曾任[[吉林省|吉林省]]省长，[[中国共产党吉林省委员会|中共吉林省委]]书记，[[中国共产党辽宁省委员会|中共辽宁省委]]书记，[[第十二届全国人民代表大会|第十二届全国人大]][[全国人民代表大会教育科学文化卫生委员会|教育科学文化卫生委员会]]副主任委员。

== 早年经历 ==
1950年3月，王珉出生于安徽淮南九龙岗矿路公司医院。1968年[[文化大革命|文化大革命]]期间，高中毕业的王珉下乡到[[舒城县|舒城县]]石岗公社插队劳动；受“家庭出身”影响，直到1972年，他才被招工到淮南化工机械厂当上一名钣金工开刨床。1975年，王珉被推荐到淮南煤炭学院（现[[安徽理工大学|安徽理工大学]]）机电系矿机专业学习，后来又留校在机制工艺教研室当教师。1979年，恢复[[普通高等学校招生全国统一考试|高考]]后第一届研究生招生，他大胆报名，被[[北京航空航天大学|北京航空学院]]录取。1981年，王珉研究生毕业后又回到淮南煤炭学院当教师，2年后他又到[[南京航空航天大学|南京航空学院]]机械工程系机械制造专业攻读博士研究生。1985年，博士研究生毕业后留校任教，历任机械制造系讲师、实验室主任，机械工程系副主任、副教授，系主任、教授，南京航空学院秘书长，南京航空学院副院长、博士生导师，1994年升为常务副校长<ref>{{cite web|url=http://news.sina.com.cn/c/zg/2016-03-04/doc-ifxqaffy3617806.shtml|title=王珉落马，回马枪显威？|accessdate=2016-03-04|author=|date=2016-03-04|publisher=新浪网}}</ref>。

== 从政履历 ==
1994年7月，王珉由[[南京航空航天大学|南京航空航天大学]]常务副校长转到地方工作，出任[[江苏行政长官列表|江苏省省长]]助理，开始从政之路。尽管起步晚，但王珉一路升迁；先后出任江苏省副省长，中共[[苏州市|苏州市]]委书记；并于2002年8月当选[[中国共产党江苏省委员会|中共江苏省委]]常委；两年后，调升[[中国共产党吉林省委员会|中共吉林省委]]副书记；数月后，又接任吉林省代省长；2005年1月29日，[[吉林省|吉林省]]十届人大三次会议正式选举王珉为[[吉林行政长官列表|吉林省省长]]；不到两年，他又接任[[中共吉林省委|中共吉林省委]]书记职务。

2009年11月，[[中国共产党中央委员会|中共中央]]決定，任命王珉为[[中国共产党辽宁省委员会|中共辽宁省委]]书记<ref>{{Cite web|url=http://news.hexun.com/2009-11-30/121861622_3.html|title=河南等5省区党委主要负责同志职务调整|accessdate=2017-09-21|author=|date=2009-11-30|work=|publisher=和讯网}}</ref>。2013年1月王珉当选辽宁省人大常委会主任<ref>{{cite web|url=http://district.ce.cn/newarea/sddy/201301/31/t20130131_24082885.shtml|title=王珉当选辽宁省人大主任 陈政高当选省长|accessdate=2014-06-03|author=|date=2013-01-31|publisher=[[中国经济网|中国经济网]]}}</ref>。2015年5月，不再担任辽宁省委书记<ref>{{cite web|url=http://news.sina.com.cn/c/2015-05-04/160631791563.shtml|title=李希任辽宁省委书记 王珉不再担任|accessdate=2015-05-04|author=|date=2015-05-04|publisher=新浪网}}</ref>、省人大常委会主任<ref>{{cite web|url=http://district.ce.cn/newarea/sddy/201505/30/t20150530_5505593.shtml|title=王珉辞任辽宁省人大常委会主任|accessdate=2015-05-30|author=|date=2015-05-30|publisher=中国经济网}}</ref>。

2015年7月1日，[[全国人民代表大会常务委员会|全国人大常委会]]召开会议，会议任命王珉为第十二届[[全国人民代表大会教育科学文化卫生委员会|全国人民代表大会教育科学文化卫生委员会]]副主任委员<ref>[http://www.chinanews.com/gn/2015/07-01/7377903.shtml 王珉任全国人大教科文卫委员会副主任委员]</ref>。

== 落马 ==
{{See also|辽宁拉票贿选案}}

2016年3月4日上午十点，[[中央纪委监察部网站|中央纪委监察部网站]]宣布，第十二届[[全国人大教育科学文化卫生委员会|全国人大教育科学文化卫生委员会]]副主任委员王珉涉嫌严重违纪，接受组织调查<ref>{{cite web |url = http://www.ccdi.gov.cn/xwtt/201603/t20160304_75426.html |title=十二届全国人大教育科学文化卫生委员会副主任委员王珉涉嫌严重违纪接受组织调查 |accessdate=2016-03-04 |date=2016-03-04 |website =[[中央纪委监察部网站|中央纪委监察部网站]] }}</ref>。王珉被调查当天其妻子张静亦在沈阳被控制并送往北京<ref>{{cite web |url = http://news.163.com/16/0409/17/BK7Q1S1C0001124J.html |title=王珉被调查当天其妻子被控制 并送往北京 |accessdate=2016-04-09|author=|date=2016-04-09|website=网易}}</ref>。2016年4月，辞去第十二届全国人民代表大会代表职务，其第十二届全国人民代表大会教育科学文化卫生委员会副主任委员职务相应终止<ref name="d18h">{{cite news |url = http://www.npc.gov.cn/npc/xinwen/2016-04/29/content_1988760.htm |title=全国人民代表大会常务委员会公告〔十二届〕第十八号|author=|date=2016-04-29|website=中国人大网|accessdate=2016-07-03}}</ref>。

2016年8月，经[[中国共产党中央委员会|中共中央]]批准，[[中国共产党中央纪律检查委员会|中共中央纪委]]对王珉严重违纪问题进行了立案审查。

经查，王珉严重违反政治纪律和政治规矩，身为省委书记没有履行管党治党政治责任，未按照中央要求履行换届工作第一责任人的职责，对辽宁省有关[[辽宁拉票贿选案|选举发生拉票贿选问题]]负有主要领导责任和直接责任；履行全面从严治党主体责任不力，公开妄议并违反中央八项规定精神，公款大吃大喝、顶风违纪，对抗组织审查；违反组织纪律，利用职权和职务影响，在干部选拔任用等方面为他人提供帮助并收受财物；违反廉洁纪律，在企业经营等方面为他人谋取利益，其亲属从利益输送中获得经济利益；利用职权和职务影响为他人谋取利益并收受财物，涉嫌受贿犯罪。

王珉身为中央委员，理想信念动摇，纪律意识丧失，严重违反党的纪律，且在党的[[十八大|十八大]]后仍不收敛、不收手，造成恶劣政治影响，应予严肃处理。依据《中国共产党纪律处分条例》等有关规定，经中央纪委常委会议研究并报[[中国共产党中央政治局|中共中央政治局]]会议审议，决定给予王珉开除党籍、开除公职处分；收缴其违纪所得；将其涉嫌犯罪问题、线索及所涉款物移送司法机关依法处理。

2016年8月11日，最高人民检察院宣布，依法决定对王珉立案侦查<ref>{{Cite news|url=http://www.spp.gov.cn/qwfb/201608/t20160811_162205.shtml|title=最高人民检察院依法决定对王珉立案侦查|last=|first=|date=2016-08-11|work=中华人民共和国最高人民检察院|accessdate=2016-09-26}}</ref>。10月27日报道<ref>{{cite web|url=http://xw.qq.com/news/20161027038226/NEW2016102703822605|title=赵宪庚、咸辉递补为中央委员 武警副司令牛志忠等4人被开除党籍|publisher=腾讯网，来源：新华社|accessdate=2017-06-13|date=2016-10-27| language = zh-Hans}}</ref>，[[中国共产党第十八届中央委员会第六次全体会议|中共十八届六中全会]]确认对王珉等四人开除党籍的处分。

2017年8月4日，[[河南省|河南省]][[洛阳市|洛阳市]]中级人民法院一审宣判，对被告人王珉以受贿罪、贪污罪、玩忽职守罪数罪并罚，决定执行无期徒刑，剥夺政治权利终身，并处没收个人全部财产<ref>{{cite web|url=http://www.ccdi.gov.cn/yw/201708/t20170804_104233.html|title=全国人大教育科学文化卫生委员会原副主任委员王珉受贿、贪污、玩忽职守案一审宣判|accessdate=2017年8月4日|author=|date=2017年8月4日|publisher=中央纪委监察部网站}}</ref>。
