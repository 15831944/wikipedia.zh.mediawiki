'''王世忠'''<ref name=辽227>{{harvnb|李澍田|1986|p=227}}；引自《辽夷略》</ref>，原名'''克把库'''<ref name=辽227/>、又作'''革把库'''<ref>{{harvnb|李澍田|1986|p=286}}；引自《开原图说》</ref>，[[哈达那拉氏|哈达那拉氏]]，[[哈达_(海西女真)|哈达]]国主[[孟格布禄|孟格布禄]]次子，哈达灭亡后，入明成长为官，谋求恢复故地，并对[[后金|后金]]国内的海西女真旧部实行策反，[[清兵入关|清兵入关]]后投降清廷。

==生平==
克把库为哈达国主孟格布禄次子，年幼时逢建州首领[[努尔哈赤|努尔哈赤]]攻克哈达城，与其父孟格布禄、兄[[吴尔古代|吴尔古代]]等一同被俘，后努尔哈赤迫于明朝压力又将其释还。克把库遂被家人带入明朝，时年八九岁，改名王世忠，由[[明神宗|明神宗]]抚养于宫中，因此衣冠举止颇为汉化<ref name=南明史1878>{{harvnb|钱海岳|2006|p=1878}}</ref>。[[叶赫|叶赫]]灭亡时，明朝因其为金台石妻侄，授广宁卫指挥加游击衔<ref name=南明史1878/><ref>{{harvnb|赵尔巽等|1998|p=9145}}</ref>，后再由[[王象乾|王象乾]]推荐为中军都督府佥书抚夷总兵<ref name=南明史1878/><ref>{{harvnb|谢贵安|2003|p=180}}；引自《明实录》天启七年十月壬寅条</ref><ref>{{harvnb|中国人民大学清史研究所|1988|pp=40，45}}</ref>。因王世忠之妹嫁[[察哈尔部|察哈尔]][[林丹汗|林丹汗]]，所以[[王之臣_(潼关)|王之臣]]利用王世忠与林丹汗的姻亲关系，将其调往宣云，用以结交察哈尔部，取得了积极的成果，此后跟随[[大学士|大学士]][[孙承宗|孙承宗]]前往辽东，晋封为左都督<ref name=南明史1878/>。后金于撤守广宁后，王世忠统兵进驻，依托察哈尔部，暗中策反后金内部原[[海西女真|海西四部]]部众，以期起到瓦解后金的作用<ref name=清史研究>{{cite journal en|quotes = |author = 陈永祥|date =  2014|title =《乌尔古岱案补释》|journal = 清史研究|issue = 2014年第04期|pages = 104-107|publisher = 中国人民大学清史研究所|location = 北京市|issn = |url = |language = Zh-hans}}</ref>。明朝曾希望以其奉祀哈达，收复故地以号令女真，在此期间，王世忠还对其兄吴尔古代使用离间计，被认为是导致吴尔古代在后金汗廷失势的原因之一<ref name=清史研究/>。

[[崇祯|崇祯]]年间，王世忠因贪污丢官，于是投靠[[左良玉|左良玉]]，左良玉与其结为亲家，令其子[[左梦庚|左梦庚]]娶王世忠之女。因王世忠[[满语|满语]]娴熟，左梦庚还曾对此向其进行学习<ref name=南明史1878/>。顺治元年（1644年）<ref>{{harvnb|中央研究院近代史研究所|2008|p=155}}</ref>，王世忠降清，在朝廷中历任显职，后为清军招降左梦庚出力甚多<ref name=南明史1878/>。
