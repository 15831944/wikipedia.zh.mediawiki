'''王光宇'''（{{bd|1919年|11月|2017年|8月23日|catIdx=}}），曾用名'''王元乾'''，[[安徽省|安徽省]][[霍邱县|霍邱县]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员<ref name=hefei>{{Cite web |url =http://news.wehefei.com/system/2017/08/25/011093056.shtml  |title = 安徽省原省人大常委会主任王光宇在合肥逝世 享年99岁 |author =  |date =2017-08-25  |publisher = 合肥网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1919年11月，王光宇生于安徽省霍邱县临水镇。1936年7月到1937年9月，先后在[[颍上县|颍上县]]、[[上海市|上海市]]参加[[陶行知|陶行知]]领导的生活教育社、山海工学团，学习并开展抗日救亡活动。1937年10月到1938年5月，先后在颍上县、霍邱县任教<ref name=hefei/>。

1938年6月，参加[[中国共产党|中国共产党]]领导的革命工作。1938年10月加入中国共产党。1938年6月到1944年8月，历任霍邱县动委会干事、中共霍邱县委宣传部部长、中共皖东北直属区委书记、县委组织部长，中共[[泗东县|泗东县]]委书记、中共[[泗南县|泗南县]]委副书记、中共[[泗宿县|泗宿县]]委书记<ref name=hefei/><ref>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1773916|title = 安徽省人大常委会原主任王光宇逝世，享年99岁 |author =  |date = 2017-08-24 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

1944年9月到1949年2月，历任中共淮北路西[[永城县|永城县]]委书记，[[新四军|新四军]]淮北路西八分区独立二团政委，[[中共中央华中分局|中共中央华中分局]]八地委民运部部长、八分区肖永砀工委书记，中共豫皖苏三地委副书记，[[淮海战役|淮海战役]]北线支前指挥部政委<ref name=hefei/>。

1949年3月到1952年，任中共[[阜阳|阜阳]]地委书记。1952年5月至1954年10月，任[[中共安徽省委|中共安徽省委]]秘书长。1954年10月至1956年4月，任中共安徽省委农村工作部部长。1954年12月起，任中共安徽省委常委。1956年5月到[[文革|文革]]初期，任[[安徽省人民委员会|安徽省人民委员会]]副省长，负责农口。1958年7月至1967年1月，任中共安徽省委书记处书记，分管农业工作。其间，1955年4月到1957年4月兼任安徽省人民委员会第五办公室主任，1956年5月到1961年2月兼任[[安徽农学院|安徽农学院]]院长，1963年2月到1964年10月兼任中共安徽省委农村工作部部长，1963年7月起兼任安徽省农林水办公室主任<ref name=hefei/>。

文革期间受到冲击。1969年起，任[[安徽省革命委员会|安徽省革命委员会]]生产指挥组副组长。1971年5月起，升任中共安徽省委常委。1973年5月到1983年3月，任中共安徽省委书记（当时设有第一书记）；1983年3月到1984年12月，任中共安徽省委副书记。1973年5月到1979年12月，任安徽省革命委员会副主任。1985年3月到1993年2月，任[[安徽省人大常委会|安徽省人大常委会]]主任、[[党组|党组]]书记。1994年6月离休<ref>{{cite news|title=王光宇|url=http://www.ah.xinhuanet.com/jg60nahp/2009-08/25/content_17502319.htm|date=2009-08-25|newspaper=新华网|deadurl=yes|archiveurl=https://web.archive.org/web/20100625161849/http://www.ah.xinhuanet.com/jg60nahp/2009-08/25/content_17502319.htm|archivedate=2010-06-25}}</ref><ref name=hefei/><ref name=jianzheng>王光宇，见证安徽治淮60年，党史纵览2009(9):23-27</ref><ref name=shishi>{{Cite web |url = http://news.ifeng.com/a/20170824/51747853_0.shtml |title = 安徽省人大原主任王光宇逝世，生前曾与万里共事 |author =  |date = 2017-08-24 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

1977年6月，[[万里|万里]]出任中共安徽省委第一书记后，指定安徽省长期负责农业工作的王光宇同他一起搞农村调研。1977年11月28日，发布《中共安徽省委关于目前农村经济政策几个问题的规定（试行草案）》（俗称“省委六条”）。1978年安徽省遭受大旱，全省共6000余万亩农田受灾，4000余万人口的地区缺乏生活用水，分管农业的王光宇向万里提出借鉴1954年安徽水灾中采用过的“借地度荒”的办法，将耕地借给农民个人耕种。该建议获得采纳，使安徽全省最终超额完成秋种计划，并为在农村广泛推行[[包产到户|包产到户]]、[[包干到户|包干到户]]提供了契机和条件<ref name=shishi/>。

王光宇是[[中共八大|中共八大]]、[[中共十一大|中共十一大]]、[[中共十二大|中共十二大]]、[[中共十五大|中共十五大]]代表，中共第十一届、十二届中央委员会委员，第四届、第六届、第七届全国人大代表<ref name=hefei/><ref name=shishi/>。

2017年8月23日，王光宇在[[合肥|合肥]]病逝，享年99岁<ref name=hefei/>。
