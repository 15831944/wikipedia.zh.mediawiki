'''王金河'''（{{bd|1916年|4月23日|2014年|3月13日|}}），生於[[台灣|台灣]][[台南縣|台南縣]][[北門區|北門鄉]]（今[[台南市|台南市]]北門區），[[台灣|台灣]]著名醫生。為[[烏腳病|烏腳病]]貢獻一生，有「[[台灣|台灣]][[烏腳病|烏腳病]]之父」之稱。

2011年，王金河獲得母校[[日本|日本]][[明治學院大學|明治學院大學]]榮譽博士學位。王金河曾獲終身醫療奉獻獎和台南市卓越市民。

== 生平 ==
王金河於1916年（[[大正|大正]]五年）4月23日生於[[臺南廳|臺南廳]]北門嶼支廳。曾就讀北門公學校、[[長榮中學|長榮中學]]。

1934年（[[昭和|昭和]]九年）到[[日本|日本]]內地就讀就讀[[明治學院大學|明治學院大學]]中學部，進入東京醫專（今[[東京醫科大學|東京醫科大學]]）學醫。1941年（昭和十六年），王金河自日本東京醫專畢業。曾在太久保醫院外科服務，因母親病危，因此於1942年（昭和十七年）自日本返台，1944年至故鄉北門開業，，開設「憐憫之門」診療所。

[[第二次世界大戰|第二次世界大戰]]結束後，王金河曾被推選為第一屆[[北門區|北門鄉]]鄉長，並曾當選兩屆台南縣議員。

1950年代末期，於台灣西南沿海的北門、布袋、學甲和義竹等鄉鎮發生[[烏腳病|烏腳病]]，當時治療烏腳病的唯一方法為截肢。1954年，王金河，與[[孫理蓮|孫理蓮]]牧師、[[謝緯|謝緯]]醫師合作，免費為患病鄉民治病。王金河同時開始研究烏腳病。

2014年3月13日晚間8點15分，王金河因[[腦溢血|腦溢血]]併發[[心臟衰竭|心臟衰竭]]，病逝於[[臺大醫院|臺大醫院]]，享壽97歲<ref>[http://www.cna.com.tw/news/asoc/201403140332-1.aspx 烏腳病之父病逝 地方不捨]，中央社，2014年3月14日</ref><ref>[http://www.appledaily.com.tw/realtimenews/article/life/20140314/360352/%E5%8F%B0%E7%81%A3%E7%83%8F%E8%85%B3%E7%97%85%E4%B9%8B%E7%88%B6%E7%8E%8B%E9%87%91%E6%B2%B3%E9%80%9D%E4%B8%96%E3%80%80%E5%90%84%E7%95%8C%E7%B7%AC%E6%87%B7 台灣烏腳病之父王金河逝世]</ref>。

== 烏腳病 ==
王金河回到北門後，只因當時當地蔓延著未知而可怕的疾病，也就是飲用深水井所導致的過量[[砷中毒|砷中毒]]的末梢血管病變。他在極度年輕的生命裡就立下為家鄉人服務的壯志，除了用畢生積蓄蓋了專門治療烏腳病的診所、自費顧用當時年僅15歲李秀稠擔任護士一職；兩人用免費無償的方式每日去看診在當時無藥可治且被所有人們排擠的患者，甚至幫他們動手術切除已壞死的雙腳，在他們受不了因惡疾纏身的苦痛與煎熬而選擇上吊自殺離開人世時，還會親自動手扛著往生者去安葬。就這樣過了十二年的歲月裡，隨著當地逐漸建設自來水管路，以自來水取代井水，砷中毒才逐漸根除。

== 家族 ==
* 王金河的妻子毛碧梅為[[六甲庄|六甲庄]]（今台南市[[六甲區|六甲區]]）人，王金河與毛碧梅於1942年（昭和十七年）結婚，毛碧梅於1922年（大正十一年）生，1995年逝世。父親毛維麟曾任六甲庄長，毛碧梅大姊毛雪芬為[[鹽分地帶|鹽分地帶]]文學家[[吳新榮|吳新榮]]原配。
