'''王梦恕'''（{{bd|1938年|12月24日|2018年|9月20日}}<ref>{{cite news |title=媒体：“高铁院士”王梦恕去世，享年80岁_中国政库_澎湃新闻-The Paper |url=https://www.thepaper.cn/newsDetail_forward_2460345 |work=www.thepaper.cn}}</ref>），[[河南|河南]][[焦作|焦作]][[温县|温县]]人，中国[[隧道工程|隧道工程]]专家、[[中国工程院|中国工程院]][[中国工程院院士|院士]]。毕业于[[唐山铁道学院|唐山铁道学院]]（今[[西南交通大学|西南交通大学]]），生前担任[[北京交通大学|北京交通大学]]土木建筑工程学院教授<ref>{{Cite web |url = http://jgrsc.njtu.edu.cn/personnel/teacher_view.asp?id=3 |title = 王梦恕：中国工程院院士，隧道及地质工程专家 |publisher = 北京交通大学 |accessdate = 2013-01-12}}</ref>，[[全國人大代表|全國人大代表]]。其关于[[中国高速铁路|中国高速铁路]]、[[2011年甬台温铁路列车追尾事故|2011年甬台温铁路列车追尾事故]]等相关社会事件的评论在社会上引起广泛关注和争论<ref>{{Cite news |url = http://www.dooland.com/magazine/article_176368.html |title = 漩涡中的王梦恕 |publisher = 新民周刊 |year = 2011 |accessdate = 2013-01-12}}</ref>。他曾被提名为“2011年中国科学年度人物”<ref>{{Cite web |url = http://2011people.sciencenet.cn/content.aspx?id=388 |title = 2011中国科学年度人物评选 |publisher = 科学网 |accessdate = 2013-01-12 |deadurl = yes |archiveurl = https://web.archive.org/web/20120309204606/http://2011people.sciencenet.cn/content.aspx?id=388 |archivedate = 2012-03-09 }}</ref>。他也是[[渤海海峡跨海通道|渤海海峡跨海通道]]（即[[烟大海底隧道|烟大海底隧道]]）战略规划研究项目组组长<ref >{{cite web|url=http://news.163.com/14/0830/15/A4TGRET70001124J.html | language= | title=烟大海底隧道方案将报国务院:耗资堪比三峡 |publisher= 人民网 |date=2014-08-30 |accessdate= }}</ref>。

===广深港高铁一地两检===
王夢恕也曾對[[廣深港高速鐵路|廣深港高速鐵路]]發表意見，他認為香港政府沒有把高鐵設計權交給[[中共中央|中央]]，導致設計耽誤多時，所需費用較中國內地貴一倍以上，又認為香港負責高鐵的班子很落後、很慢。他稱中央要接通香港，且不會做登記，形容香港都是中國的土地，坐高鐵應該要「不檢，不簽證」。他更直言內地對香港太嬌慣，「內地要接通，要通過就通過」，認為香港是[[中华人民共和国行政区划|中國的一個省]]，「誰不聽話就滾出去」<ref>{{cite news|title=高鐵耽誤 工程院院士指中央太嬌慣香港|url=http://hk.on.cc/hk/bkn/cnt/news/20170305/bkn-20170305124843769-0305_00822_001.html|agency=東網|date=2017-03-05}}</ref>。

但王此番言論不被部分人如[[香港立法會|香港立法會]]議員[[田北辰|田北辰]]認同，認為王夢恕根本不理解《[[香港特別行政區基本法|基本法]]》，「根本不知道香港是一個特區」<ref name="apple170305">{{cite news|title=內地專家指香港僅一個省高鐵不用檢 田北辰：不理解一國兩制|url=http://hk.apple.nextmedia.com/realtime/news/20170305/56387664|agency=蘋果日報 (香港)|date=2017-03-05}}</ref>，是間接否定[[鄧小平|鄧小平]]的[[一國兩制|一國兩制]]。他稱王夢恕誤解一國兩制是過渡性安排，以為《基本法》實施50年後便會失效，但只要一國兩制不走樣、不變形、不引致危害國家安全的事發生，《基本法》的法律價值便不會成疑，也沒有終止時效<ref>{{cite news|title=田北辰指王夢恕不理解基本法 言論不可能代表中央意念|url=http://news.rthk.hk/rthk/ch/component/k2/1317301-20170305.htm|agency=香港電台|date=2017-03-05}}</ref>。「中央政府應該協助王夢恕認識一國兩制」<ref name="apple170305"/>。他認為其言論不可能代表中央意念<ref name="apple170305"/>。時事評論員[[劉銳紹|劉銳紹]]認為王言論明顯損害了一國兩制，但是側面反映了北京當局的看法，是北京對一國兩制已失去了耐性的表現<ref>{{cite news|title=全國兩會 高鐵專家談一地兩檢：香港係中國一個省 誰不聽就滾|url=http://hk.apple.nextmedia.com/realtime/china/20170305/56387030|agency=蘋果日報 (香港)|date=2017-03-05|language=zh-yue}}</ref>。
