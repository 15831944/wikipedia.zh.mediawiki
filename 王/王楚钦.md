'''王楚钦'''（{{bd|2000年|5月11日||}}）出生于[[吉林省|吉林省]]，是一名中国男子乒乓球运动员。<ref>{{cite web |author1=ITTF |title=Player profile-Personal Data |url=http://results.ittf.link/index.php?option=com_fabrik&view=details&formid=99&Itemid=266&rowid=121558&resetfilters=1 |website=Player profile |accessdate=2018-10-14}}</ref>2018年10月10日，在[[2018年夏季青年奥林匹克运动会|2018年夏季青年奥林匹克运动会]]乒乓球男子单打决赛中，王楚钦以4比1战胜日本选手[[张本智和|张本智和]]，获得冠军。<ref>{{cite web |author1=新浪体育综合 |title=10-0！青奥王楚钦打蒙张本智和 夺冠后脱衣庆祝 |url=http://sports.sina.com.cn/others/pingpang/2018-10-11/doc-ifxeuwws3036658.shtml |accessdate=2018-10-14}}</ref>
