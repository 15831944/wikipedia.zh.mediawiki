'''王夕越'''（{{bd|1980年|12月31日|||catIdx=王}}）是一名[[美籍华人|美籍华人]]在读[[博士|博士]]，就读于[[普林斯顿大学|普林斯顿大学]]历史系。2016年8月8日，[[伊朗政府|伊朗政府]]以王夕越涉嫌在[[伊朗|伊朗]]境内从事[[间谍|间谍]]行为而将其逮捕；<ref name="twp">{{cite web |author1=Erin Cunningham |author2=Carol Morello |title=Iran sentences Princeton graduate student to 10 years for espionage, report says |url=https://www.washingtonpost.com/world/iran-sentences-princeton-graduate-student-to-10-years-for-espionage-report-says/2017/07/16/664320c4-6a46-11e7-abbc-a53480672286_story.html |publisher=[[华盛顿邮报|华盛顿邮报]] |accessdate=2019-04-05 |archive-url=https://web.archive.org/web/20190317092701/https://www.washingtonpost.com/world/iran-sentences-princeton-graduate-student-to-10-years-for-espionage-report-says/2017/07/16/664320c4-6a46-11e7-abbc-a53480672286_story.html |archive-date=2019-03-17 |dead-url=no }}</ref>2017年7月，他被伊朗政府判处10年[[有期徒刑|有期徒刑]]。<ref name="twp" /><ref>{{cite web |author1=Elizabeth Redden |title=Iran Jails Princeton Ph.D. Student as Spy |url=https://www.insidehighered.com/quicktakes/2017/07/17/iran-jails-princeton-phd-student-spy |publisher=Inside Higher Ed |access-date=2019-04-05 |archive-url=https://web.archive.org/web/20181214094746/https://www.insidehighered.com/quicktakes/2017/07/17/iran-jails-princeton-phd-student-spy |archive-date=2018-12-14 |dead-url=no }}</ref><ref name="theguardian">{{cite web |author1=Saeed Kamali Dehghan |title=Iran sentences Chinese-born American to 10 years in jail on spying charges |url=https://www.theguardian.com/world/2017/jul/16/iran-american-10-years-jail-xiyue-wang-princeton |publisher=[[卫报|卫报]] |accessdate=2019-04-05 |archive-url=https://web.archive.org/web/20190208081921/https://www.theguardian.com/world/2017/jul/16/iran-american-10-years-jail-xiyue-wang-princeton |archive-date=2019-02-08 |dead-url=no }}</ref>

== 个人简历 ==
王夕越于1980年12月31日出生于[[中国|中国]][[北京|北京]]，<ref name="rfa0719" /><ref name="voa2018">{{cite web |title=第三个在牢狱中的生日 普林斯顿再呼吁释放王夕越 |url=https://www.voachinese.com/a/wang-xiyue-student-iran-20181231/4723136.html |website=美国之音中文网 |publisher=[[美国之音|美国之音]] |accessdate=2019-04-05 |archive-url=https://web.archive.org/web/20190101014625/https://www.voachinese.com/a/wang-xiyue-student-iran-20181231/4723136.html |archive-date=2019-01-01 |dead-url=no }}</ref>1999年前往美国求学，先后就读于[[华盛顿大学|华盛顿大学]]和[[哈佛大学|哈佛大学]]。<ref name="rfa0719" />2000年，王夕越移民美国。<ref name="rfi2017" />2009年，当王夕越在香港工作时结识了曲桦；并于2012年结婚。<ref name="nytimes" />2013年，王夕越被普林斯顿大学录取，成为该大学的历史系博士在读生；同年3月，他和曲桦诞下他们的第一个儿子。<ref name="nytimes" />

== 案件缘由 ==
2016年1月下旬，研究19世纪亚洲内陆地区政府间的相似之处的王夕越为毕业论文搜集资料踏上了前往[[伊朗|伊朗]]的旅程。<ref name="nytimes" />期间，他曾两次进出伊朗，但在当年8月准备离开伊朗时遭到[[伊朗政府|伊朗政府]]的逮捕。<ref name="voa2018" />伊朗政府宣称王夕越在伊朗逗留期间从事间谍行为。<ref name="twp" /><ref name="theguardian" />伊朗政府指控王夕越在搜集[[卡扎尔王朝|卡扎尔王朝]]资料时，支付了大笔美金用以制作数字文件；同时试图访问多家图书馆的保密区域。<ref>{{cite web |author1=JON GAMBRELL |title=Iran TV videos target 2 detainees accused of espionage |url=https://apnews.com/28b272d75f0f4c8cae332e6be1bd7210 |publisher=[[美联社|美联社]] |accessdate=2019-04-05 |archive-url=https://web.archive.org/web/20180615161119/https://www.apnews.com/28b272d75f0f4c8cae332e6be1bd7210 |archive-date=2018-06-15 |dead-url=no }}</ref><ref>{{cite web |author1=Adam Schreck |title=SCHOLAR DEFENDS CHINESE-AMERICAN RESEARCHER JAILED IN IRAN |url=http://macaudailytimes.com.mo/scholar-defends-chinese-american-researcher-jailed-iran.html |website=澳门每日时报 |publisher=[[美联社|美联社]] |accessdate=2019-04-05 |archive-url=https://web.archive.org/web/20170917124533/http://macaudailytimes.com.mo/scholar-defends-chinese-american-researcher-jailed-iran.html |archive-date=2017-09-17 |dead-url=no }}</ref>随后，伊朗当局在2017年7月宣判王夕越[[有期徒刑|有期徒刑]]10年。<ref name="theguardian" />

== 营救行动 ==
针对伊朗当局对王夕越的指控，[[美国国务院|美国国务院]]和[[普林斯顿大学|普林斯顿大学]]都表示了否认，普林斯顿大学更是强调王夕越的行为属于学术范围。<ref name="twp" /><ref name="theguardian" /><ref name="voa2018" />[[联合国|联合国]]任意拘留问题工作组于2019年表态，认为伊朗政府对王夕越的指控没有法律依据，王夕越遭遇到了不公拘押；而[[美洲大學協會|美国大学联盟]]和[[欧洲大学协会|欧洲大学协会]]均呼吁释放王夕越。<ref name="voa2018" />

王夕越被捕后，其妻曲桦一直在和普林斯顿校方几美国国务院沟通，希望促成其释放。普林斯顿最初奉行沉默政策，希望伊朗可以以抓错了为由放人。由于美国与伊朗断交，并对伊朗进行制裁，双方没有常设外交渠道。伊朗方面在同时期共逮捕了四名美国人。在特朗普上台后，特朗普政府不认为与伊朗改善关系可以解决人质问题的长远困境。并宣布退出《聯合全面行動計劃》（伊朗核协议） 。其妻子则一直在为其发声，希望政府能促成其释放。<ref name="nytimes" />

王夕越出生于中国，其妻曲桦仍为中华人民共和国[[公民|公民]]，因此曲桦希望中国政府能出面营救王夕越。<ref name="nytimes" />[[中国外交部|中国外交部]]对此在2017年7月17日的例行记者会上向记者回应，王夕越拥有美国国籍，不具有中国国籍。<ref name="rfa0719" />2019年4月，[[美国之音|美国之音]]报道称，有负责此案的不具名官员告诉曲桦，北京方面提出愿意从[[人道主义|人道主义]]出发，协助美方援救王夕越。不过截至发稿时，尚无中方人员与曲桦取得联系，她无从知晓北京方面打算如何介入。<ref>{{cite web |author1=萧雨 |title=北京转变立场，德黑兰囚禁千日的美籍华裔学者有望回家？ |url=https://www.voachinese.com/a/china-may-step-in-to-rescue-chinese-american-scholar-detained-in-iran-20190404/4862525.html |website=美国之音中文网 |publisher=[[美国之音|美国之音]] |accessdate=2019-04-05}}</ref>
