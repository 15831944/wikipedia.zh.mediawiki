'''王仲才'''（{{bd|1963年|11月||}}），[[河北省|河北省]][[保定市|保定市]][[蠡县|蠡县]]人，[[中国人民武装警察部队海警总队|中国人民武装警察部队海警总队]]司令员，[[武警少将|武警少将]]警衔。

==生平==
===早年===
王仲才入伍参军后，于1980年进入预备航校，毕业后在[[中国人民解放军海军飞行学院|飞行学院]]第二训练团一大队担任航校领航教员，为正连级飞行员，飞行时数超700小时。1987年，王仲才考入[[中国人民解放军海军广州舰艇学院|中国人民解放军海军广州舰艇学院]][[飞行员舰长班|飞行员舰长班]]，历时3年半学习后，1991年1月毕业，取得舰艇指挥专业本科学历和工学学士学位。王仲才毕业后业分配至[[北海舰队|北海舰队]]驱逐舰第一支队，历任[[053H1Q型导弹护卫舰|四平号导弹护卫舰]]见习副舰长，[[哈尔滨号导弹驱逐舰|哈尔滨号导弹驱逐舰]]副舰长，[[青岛号导弹驱逐舰|青岛号导弹驱逐舰]]副舰长。2000年后，王仲才担任[[杭州号导弹驱逐舰|杭州号导弹驱逐舰]]副舰长、舰长<ref>{{Cite web |url = http://news.sohu.com/20050413/n225156563.shtml |title =  18年后的回眸与展望—中国航空母舰舰长班揭秘|author =  |date = 2005-04-13 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。此后，王仲才长期在东海舰队服役。2011年9月16日至12月29日，王仲才担任任务副指挥员，乘[[和平方舟号|和平方舟号]]医院船从浙江舟山某军港启航，横跨太平洋，穿越巴拿马运河，赴古巴、牙买加、特立尼达和多巴哥、哥斯达黎加等拉美四国，执行代号为“和谐使命—2011” 出访暨医疗服务任务<ref>{{Cite web |url = http://military.people.com.cn/n/2014/1017/c389726-25858265.html|title =  和谐使命2011：中国海军“和平方舟”拉美四国医疗服务纪实|author =  |date = 2014年10月17日18:49  |publisher = 人民网|language =  |accessdate =  |quote =  }}</ref>。2013年，王仲才出任[[东海舰队|东海舰队]][[中国人民解放军海军厦门水警区|厦门水警区]]司令员<ref name=autogenerated1>{{Cite web |url = http://news.youth.cn/gn/201710/t20171008_10834091.htm|title = 率中国军舰首访伦敦 这名少将啥来头？|author =  |date =2017-10-08|publisher=中国青年网 |language =  |accessdate =  |quote =  }}</ref>。2016年6月2日，王仲才大校已经出任海军东海舰队副参谋长，跻身副军级军官<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1478071 |title = 王仲才任东海舰队副参谋长，系我军首届“飞行员舰长班”学员|author = 岳怀让 |date =2016-06-02 |publisher =澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

===索马里护航及对外访问===
2017年4月1日上午，王仲才出任编队指挥员，率领[[黄冈号导弹护卫舰|黄冈号导弹护卫舰]]、[[扬州号导弹护卫舰|扬州号导弹护卫舰]]及[[高邮湖号远洋综合补给舰|高邮湖号远洋综合补给舰]]组成的中国海军护航编队从舟山某军港起航，前往[[亚丁湾|亚丁湾]]、[[索马里|索马里]]海域接替第二十五批护航编队，执行[[中国人民解放军海军索马里护航行动|中国人民解放军海军索马里护航行动]]第二十六批护航任务<ref>{{Cite web |url = http://www.81.cn/jmywyl/2017-04/01/content_7548501.htm|title =中国海军第二十六批护航编队起航 |author =刘亚迅 林健|date =  2017年09月15日|publisher =中国军网 |language =  |accessdate =  |quote =  }}</ref>。2017年4月27日开始独立执行护航任务，8月28日完成护航任务，先后完成42批64艘船舶护航、向[[索马里|索马里]]政府移交3名疑似海盗、护送[[世界粮食计划署|世界粮食计划署]]船舶至肯尼亚[[蒙巴萨港|蒙巴萨港]]、为[[向阳红系列海洋调查船|向阳红10号科考船]]进行19天随船护<ref>{{Cite web |url =http://world.people.com.cn/n1/2017/0915/c1002-29539062.html|title =中国海军舰艇编队首次访问比利时|author = |date =  2017年09月15日|publisher =人民网-国际频道  |language =  |accessdate =  |quote =  }}</ref>。护航期间的2017年5月4日，王仲才率护航编队携手驻吉布提使馆慰问吉布提首都吉布提市巴拉巴拉区第三小学，向该学校师生赠送办公设备和学习用品<ref>{{Cite web |url =http://www.fmprc.gov.cn/web/zwbd_673032/gzhd_673042/t1459009.shtml |title =我海军第二十六批护航编队携手驻吉布提使馆慰问吉巴拉巴拉区第三小学|author = |date = 2017-05-04|publisher =中华人民共和国外交部 |language =  |accessdate =  |quote =  }}</ref>。

护航结束后，王仲才率领编队先后访问了比利时安特卫普、丹麦哥本哈根、英国伦敦、法国土伦和马赛，在丹麦与丹麦海军[[伊瓦尔·休特菲尔德号护卫舰|伊瓦尔·休特菲尔德号护卫舰]]进行了海上联合演练<ref>{{Cite web |url =http://world.people.com.cn/n1/2017/0930/c1002-29570583-3.html|title = 海军第二十六批护航编队结束对丹麦的友好访问|author = |date =2017年09月30日|publisher =人民网-国际频道  |language =  |accessdate =  |quote =  }}</ref>，在伦敦期间，英国北部华人企业家协会向王仲才赠送清朝晚期在英国纽卡斯尔制造的致远号、靖远号、超勇号和杨威号四艘战舰图片辑<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1816631 |title =中国军舰首访伦敦，指挥长王仲才遇30年前海军某学院同学|author = 张平 |date =2017-10-06 |publisher =澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.guancha.cn/military-affairs/2017_10_06_429966.shtml |title =中国舰队首访伦敦 英国华人赠送北洋水师购买英舰的老照片|author =  |date =2017-10-06 |publisher =观察者网 |language =  |accessdate =  |quote =  }}</ref>。护航编队在2017年12月1日返回舟山某军港<ref>{{Cite web |url = http://navy.81.cn/content/2017-12/01/content_7852094.htm |title = 第26批护航编队凯旋，创下海军护航多个首次|author = 代宗锋 林健等 |date =2017-12-01 |publisher =澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
