{{vgname|国家的崛起|en=Rise of Nations|cn=国家的崛起|tw=王國的興起}}，是一款[[即时战略|即时战略]](RTS)游戏。游戏由[[Big_Huge_Games|Big Huge Games]]公司开发，并由[[微软|微软]]公司在2003年5月20日发行。2004年4月28日又发行了资料片'''{{link-en|王國的興起：政權保衛戰|Rise of Nations: Thrones and Patriots}}'''（Rise of Nations: Thrones and Patriots）。在2006年5月9日<ref>http://www.xiaohx001.com/264335</ref>推出的'''[[王國的興起：傳說再現|-{zh-cn:传奇延续; zh-tw:傳說再現;}-]]'''（Rise of Nations: Rise of Legends）雖被視為是續作，但其遊戲背景則有重大改變。

《王国的兴起：扩展版》是本作的扩展版，于2014年6月在[[Steam|Steam]]发行，于2017年9月在Windows 10商店发行。

== 遊戲特色 ==

王國的興起同其他的即時戰略遊戲相比有這些不同之處:

# 遊戲中可以通過修建城市和發展科技及興建城堡來擴展領土。
# 部隊和建築會隨著擁有的數量而增長生產價格。
# 當一個國家研發了損耗科技後，任何與其敵對的部隊都會在它的領域內受到緩慢而持續的傷害，補給車可以防止損耗。另外，間諜和斥候永遠不受損耗傷害。
# 建造奇觀會給國家帶來額外的加分。
# 遊戲中如果末日時鐘倒數到0，則遊戲自動結束，所有的玩家都輸了。
# 世界末日: 當核子飛彈爆炸時，便會開始倒數，此後，任何核子武器的爆炸都會使倒數再減一，當倒數為零時，所有玩家都輸了。請謹慎的使用核子武器，不要貪圖勝利而葬送了人類的未來。

王國的興起使用2D引擎顯示建築物和地形，使用3D引擎來顯示移動單位。同時支持單人和多人在線遊戲。

== 科技 ==

想在王國的興起中獲勝，就必須研發科技。

=== 圖書館科技 ===

[[圖書館|圖書館]]科技是遊戲中最重要的科技，圖書館科技包含了時代、軍事、民事、商業和科學的五大類科技。

; 時代演進
: 時代演進意味著民族在歷史上的發展，時代的演進讓民族可以有新的升級。
:* I-[[遠古時代|遠古時代]]
:* II-[[古典時代|古典時代]]
:* III-[[中古時代|中古時代]]
:* IV-[[火藥|火藥]]時代
:* V-[[啟蒙時代|啟蒙時代]]
:* VI-[[工業時代|工業時代]]
:* VII-[[現代|現代]]
:* VIII-[[資訊時代|資訊時代]]
:* 研发完资讯时代的所有科技之后，可以研发四类新的科技项目，将极大提升玩家的所控制的国家实力。
; 軍事科技
: 軍事科技可以增加人口上限，並允許新增或升級軍事單位，以及新建军事建筑。
; 民事科技
: 民事科技可以增加城市上限並擴張領域，这里的城市上限仅仅限于玩家新建的城市，如果攻占其他玩家的城市则不会受到此限制。
; 商業科技
: 商業科技可以提高商業限制和商隊數量限制。
; 科學研究
: 科學等級的提升可以降低其他科技研究的成本以及研發時間，並增加斥候及瞭望台的視野。

== 政權保衛戰：政府 ==

政府體制能讓國家軍隊及非軍隊額外的加乘效果。首先當國家晉升為「古典時代」後，就能夠要建立一個參議院 （Senate），並且研發適當的科技。每一種政府體制也能夠建立一個「愛國者 （Patriot）」，愛國者是一種與眾不同的領袖，能夠提升國家的軍事或是經濟力量。參議院能提供數種政府體制：

* 《[[專制政治|專制政治]]》
**政府是由一位專治的統治者所領導並擁有絕對的權利。
#軍事研發以及軍營單位便宜 25%。
#獲得「專制君主（Despot Patriot）」，能夠提供供給、視線以及掠奪優勢的將軍。

* 《[[共和政體|共和政體]]》
**政府是由選出能夠（或是不能）代表人民意願的眾議員所掌管。
#商業上限增加 50。
#獲得「議員（The Senator Patriot）」，能夠提供治癒、收買以及建築防禦優勢的將軍。

* 《[[君主制|君主政治]]》
**由單一世代世襲的統治者所掌管的政府。
#馬廄單位便宜 25%，而且建造速度更快。
#獲得「君主（The King Patriot）」，能夠提供供給、圍攻以及騎兵戰鬥優勢的將軍。

* 《[[民主主義|民主主義]]》
**政府的領導者是由所有公民所選出來的，其也代表所有人民的意願。
#非圖書館科技便宜 20%。
#獲得「總統（The President Patriot）」，能夠提供生產、建造以及治癒優勢的將軍。

* 《[[社會主義|社會主義]]》
**在生產以及物品分配上比市場原則更加公平的政府。
#工廠、空軍基地以及碼頭便宜 20%，而且建造速度更快。
#獲得「主席同志（The Comrade Patriot）」，能夠提供供給、戰鬥以及同化優勢的將軍。

**在生產以及物品分配上擁有自由競爭市場的政府。

== 征服世界戰役 ==

在单人模式的征服世界战役中，玩家必須選擇一支民族並開始遊戲，某些场景的战斗时间也许会超过2个小时。扩展版增加了四个经典的历史战役：[[亚历山大大帝|亚历山大大帝]]的远征、美洲殖民、[[拿破仑|拿破仑]]和[[冷战|冷战]]。

== 世界奇觀 ==
建造每个奇观可以增加你的奇观点数，这在某些游戏中是胜利条件之一。

征服世界战役的所有场景中都不允许建奇观。
* 《[[凡爾賽宮|凡爾賽宮]]》
** 增加所有建築物建造速度；減少除了知識外的所有資源的研究成本；並且加快圍攻部隊、大砲部隊、以及後勤支援部隊的移動力，使得所建造国家的补给车具有治疗受伤部队的能力。
* 《[[兵馬俑|兵馬俑]]》
**每 30 秒就可以獲得一隻免費的步行單位。（如果你控制了一个从兵马俑出来的单位，那么这个时间会增加0.5秒）
* 《[[蒂卡爾|提卡爾神殿]]》
**國家邊界、攻擊能力、以及攻擊範圍等全部都提升。此外，玩家收集「木材」資源的速度會增加，而木材的商業限制範圍也會提高。
* 《[[泰姬瑪哈陵|泰姬瑪哈陵]]》
**建築的防禦能力會增加；經由商隊貿易所得財富收入也會增加，商業財富限制提高。
* 《[[大型強子對撞機|超級對撞機]]》
**在市場上進行物品買賣時只需花費 100 單位的財富；可以馬上進行新技術研究。
* 《[[自由女神像|自由女神像]]》
**免費升級地面部隊和空軍部隊、減少軍隊在敵人領土的損耗，轟炸機和防空建築的成本降低。
* 《[[吳哥窟|吳哥窟]]》
**加快採集金屬速度，金屬貿易限制提高，建造軍營、馬廄以及碼頭的成本降低。
* 《[[太空計劃|太空計劃]]》
**觀測地圖上所有的單位、建築物。建造飛機、飛彈成本及需耗費的時間降低。
* 《[[金字塔|金字塔]]》
**加快資源累積的速度，和「食物」和「木材」的收成；增加城市的農田數 (一般有五座) 2 個單位，可建造比一般都市範圍更大的城市，增加1個城市上限。
* 《[[大報恩寺塔|大報恩寺塔]]》
**提升建造船隻的速度，掌握領土上所有稀少資源，玩家的資源和市場的收入提高。
* 《[[克里姆林宮|克里姆林宮]]》
**提升食物、木材、金屬的商業等級 (+200)；增加伐木和採礦的勞動人口數；讓入侵的敵軍受到加倍的損耗；擁有附加的「間諜」優勢：如果間諜被傷害，還可以自動恢復而不需任何花費。
* 《[[艾菲爾鐵塔|艾菲爾鐵塔]]》
**石油收益加倍，增加石油貿易上限，擴大國家的疆界。
* 《[[羅得島太陽神銅像|羅德斯島巨像]]》
**加快蒐集財富的速度，能夠生產更多商隊來進行貿易以聚集財富，允許玩家有更多的人口數量。
* 《[[紫禁城|紫禁城]]》
**可當作城市般使用。
* 《[[大競技場|大競技場]]》
**擴大國家的疆界；讓入侵的敵軍增加50%損耗；堡壘和塔樓成本降低。
* 《[[巴比倫空中花園|巴比倫空中花園]]》
**粮仓、木材厂、冶金厂产量升级费用便宜66%。
* 《[[紅堡|紅堡]]》
**免費升級堡壘，而且所有堡壘額外獲得33%生命點數；駐守在堡壘內的單位，治癒速度比一般速率+500%。

* 《[[巴比倫空中花園|巴比倫空中花園]]》、《[[紫禁城|紫禁城]]》和《[[紅堡|紅堡]]》為在政權保衛戰新增之奇觀。
