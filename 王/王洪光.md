'''王洪光'''（{{bd|1949年|8月}}），[[山东省|山东省]][[新泰市|新泰市]]人，[[中国人民解放军中将|中国人民解放军中将]]，原[[南京军区|南京军区]]副司令员。其父为开国少将[[王建青|王建青]]，岳父则为原空军政委、开国少将[[高厚良|高厚良]]。<ref>{{cite web|url=http://politics.dwnews.com/news/2007-08-09/3154363-2.html|title=多维独家报导：胡锦涛调兵遣将万无一失防兵变（5）|publisher=[[多维新闻网|多维新闻网]]|date=2007|deadurl=yes|archiveurl=https://web.archive.org/web/20140419013431/http://politics.dwnews.com/news/2007-08-09/3154363-2.html|archivedate=2014-04-19}}</ref>

1968年入伍。曾在[[总参谋部|总参谋部]]任职，担任过总参兵种部装甲兵局局长、总参兵种部副部长等。1996年晋升为少将。1998年2月出任[[装甲兵工程学院|装甲兵工程学院]]院长。<ref>{{cite web |url = http://www.gmw.cn/01gmrb/1998-11/18/GB/17880%5EGM5-1812.HTM |title=迎接世界军事发展的挑战——访中国人民解放军装甲兵工程学院院长王洪光少将 |publisher=光明日报 |date=1998-11-18}}</ref>2002年9月起任[[中国人民解放军总装备部通用装备保障部|中国人民解放军总装备部通用装备保障部]]部长。2005年12月起任[[南京军区|南京军区]]副司令员。2007年7月晋升为中将。2012年12月退役。

== 言论 ==
2014年12月，王洪光曾在《环球时报》撰文，指出[[朝鲜|朝鲜]]的诸多行径损坏中国的利益，绝非中国的“战略屏障”，认为“朝鲜早就放弃了以[[马克思列宁主义|马克思列宁主义]]作为建党的指导思想，在[[意识形态|意识形态]]上与中国没有任何相同之处”。同时主张朝鲜如果一意孤行，中国没必要为其负责，“朝鲜真要崩溃，中国也救不了它”。<ref>{{Cite web|url=http://news.qq.com/a/20141201/010070.htm|title=中将：朝鲜若崩溃中国救不了 中国人不必为朝打仗_|accessdate=2017-03-23|author=|date=|work=|publisher=腾讯网|language=zh-CN}}</ref>

2015年1月，王洪光為《[[環球時報|環球時報]]》撰文，称[[張靈甫|張靈甫]]除去「[[中國抗日戰爭|抗日]]」，算不上什麼名將，只能說是[[解放軍|解放軍]]的「手下敗將」，並宣稱[[國軍|國軍]]「消極抗戰」無法粉飾，中共敵後戰場才是抗日主戰場，引发争议。

2017年3月，在[[中国人民政治协商会议第十二届全国委员会|第十二届全国政协]]第五次会议开幕后，王洪光接受记者采访时表示：“[[美国|美国]]在台湾部署[[萨德飞弹|萨德]]之日，就是我们解放台湾之日！”<ref>{{cite web |url = http://news.ifeng.com/a/20170303/50751680_0.shtml |title=中将：美在台部署萨德之日 就是解放台湾之日 |publisher=环球网|date=2017-03-03}}</ref>该言论随即成为媒体舆论焦点。随后，解放军官方网站中国军网发表评论称：“对王洪光委员的讲话，不必大惊小怪，也不必过度解读，並非再出狂言，他不是军队的新闻发言人，可以理解是他个人的观点。”<ref>{{cite web |url = http://www.81.cn/jwgz/2017-03/04/content_7513216.htm|title=别过度解读王洪光委员的“硬话” |publisher=中国军网|date=2017-03-04}}</ref>

2018年3月，[[美國國會|美國國會]]通过《[[台湾旅行法|台湾旅行法]]》王洪光为此在[[环球网|环球网]]发表《“六战一体”武统台湾》，中國是抗戰主力，中國不放棄武統台灣，解放軍將在這一戰全滅台獨，令台獨心驚膽戰，台獨求饒也沒用，武統台灣沒難度。并打算撰写《攻下台灣只要三天》《“台独”哪里逃》。<ref>{{cite news|title=“武统”台湾到底怎么打？解放军中将霸氣放話：六种战法，三天拿下！|url=http://taiwan.huanqiu.com/article/2018-03/11697635.html|accessdate=2018-03-27|work=环球网|date=2018-03-27}}</ref>

== 家庭 ==
*父親~[[王健青|王健青]]（1911年3月21日 - 1991年9月6日）：中国人民解放军将领、中国人民解放军开国少将。
*母親~[[郭立前|郭立前]]（1918-1986）：山东新泰人；與王建青育有1女5子；
*前妻~[[高燕燕|高燕燕]]，开国少将[[高厚良|高厚良]]之女，畢業於西安第四軍醫大學；獲美國華盛頓的美利堅大學 (American University)管理學碩士學位。曾任黑石顧問業務部(Blackstone Advisory Partners)董事总经理；2015年加入Comprador擔任董事總經理；<ref>[http://www.compradorltd.com/b5_leadership2.php Comprador高燕燕女士]</ref>高燕燕夫婦育有1子，2005年王洪光、高燕燕夫婦離婚。王洪光後來再婚。
