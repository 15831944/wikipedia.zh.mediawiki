'''榛果俪'''（{{lang-it|Frangelico}}）是一款[[意大利|意大利]][[皮埃蒙特|皮埃蒙特大区]][[库内奥省|库内奥省]]Canale市出产的榛子香草风味力娇酒，酒精浓度20%<ref>{{cite web |url=http://www.frangelico.com/info/faqs.asp |title=存档副本 |accessdate=2011-05-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20111002193559/http://www.frangelico.com/info/faqs.asp |archivedate=2011-10-02 }}</ref>。20世纪80年代最初发布的时候，榛果俪的酒精浓度为24%。因为其出众的外观设计，榛果俪一经发布就吸引了众多注意力：榛果俪的酒瓶被设计为一个修道士的样子，瓶子腰部缠着一根打了结的白线。

根据制造商的说法，Frangelico这个名字是源自于一个名叫“Fra Angelico”的隐士，正式此人发明了这一独一无二的力娇酒配方。然而榛果俪的酒瓶更贴近[[方济各会|方济各会]]修道士的形象，因此人们更多的认为其名来自于[[意大利|意大利]][[文艺复兴|文艺复兴]]时期著名的画家，[[道明会|道明会]]修士[[:en:Fra_Angelico|Fra Angelico]]，只不过他通常身穿白色长袍而且不系腰带。

榛果俪品牌最初由[[:en:C&C_Group_plc|C&C集团]]持有，[[格兰父子|格兰父子]]收购[[:en:C&C_Group_plc|C&C集团]]后拥有了这个品牌。随后2010年[[格兰父子|格兰父子]]将榛果俪卖给了[[:en:Campari_Group|金巴利集团]]，后者目前持有该品牌<ref>http://www.irishtimes.com/newspaper/breaking/2010/0916/breaking15.html</ref>。

== 生产 ==
榛果俪的制作方法和其他果仁力娇酒类似：将坚果碾碎，与可可、香草、浆果和其他天然香料混合，然后浸泡在烈酒中。当酒液吸收了配料的香味后过滤掉残渣，加糖，最后瓶装。

== 饮用 ==
榛果俪可以用来调制多种鸡尾酒，如“榛子马丁尼”，“飘香榛果俪”以及“榛果俪蔓越莓”等，榛果俪混合[[伏特加|伏特加]]可以调出“巧克力蛋糕子弹特饮”。或者可以简单的就冰块、[[苏打水|苏打水]]或者咖啡饮用。
