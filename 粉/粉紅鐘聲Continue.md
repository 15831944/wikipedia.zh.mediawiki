{{日本作品|粉紅鐘聲Continue|ぱすてるチャイムContinue}}是由[[ALICESOFT|ALICESOFT]]於2005年6月17日發售的[[角色扮演遊戲|角色扮演]]類型[[日本成人遊戲|成人遊戲]]，<ref>{{cite web |title=ぱすてるチャイムContinue |url=http://www.getchu.com/soft.phtml?id=132741 |website=Getchu.com |accessdate=2019-03-26 |language=ja}}</ref>粉紅鐘聲系列第二作。2005年6月17日發售[[Fandisc|Fandisc]]《粉紅鐘聲C++》（{{lang|ja|ぱすちゃC++}}）<ref>{{cite web |title=ぱすちゃC＋＋ |url=http://www.getchu.com/soft.phtml?id=202781 |website=Getchu.com |accessdate=2019-03-26 |language=ja}}</ref>，2010年12月9日由[[5pb.|5pb.]]發售[[PlayStation_Portable|PlayStation Portable]]版<ref>{{cite web |title=5pb.出品《粉紅鐘聲 Continue》劍與魔法的戀愛養成RPG |url=https://gnn.gamer.com.tw/4/48974.html |website=巴哈姆特電玩資訊站 |language=zh-tw |date=2010-11-30}}</ref>。

==故事==

==角色==
聲優順序為PC版／PSP版
;{{nihongo|薙原勇氣|薙原 ユウキ}}
:本作的男主角，從Farnese冒險者學校轉入光綾學園的青年。
;{{nihongo|龍膽莉娜|竜胆 リナ|聲優：白波遥／[[喜多村英梨|喜多村英梨]]}}
:光綾學園的學生，勇輝的青梅竹馬。
;{{nihongo|斎香 S. Farnese|斎香・S・ファルネーゼ|聲優：[[北都南|北都南]]／[[佐藤利奈|佐藤利奈]]}}
:武器製造商「Farnese」的獨生女。在Farnese冒險者學校退學後又來到光綾學園復學。
;{{nihongo|Lucy Minseard|ルーシー・ミンシアード|聲優：みる／[[早見沙織|早見沙織]]}}
:光綾學園的妖精族學生。
;{{nihongo|Phil Ehart|フィル・イハート|聲優：[[一色ヒカル|一色ヒカル]]／新名彩乃}}
:光綾學園的妖精族學生，自稱是勇輝的未婚妻。

==評價==
粉紅鐘聲Continue在Getchu.com舉辦的「2005年美少女遊戲排名」中獲得系統部門第7名。<ref>{{cite web |title=2005年美少女ゲームユーザーランキング システム部門 |url=http://www.getchu.com/pc/ranking2005/system.html |website=Getchu.com |language=ja}}</ref>另外在《[[ファミ通|ファミ通]]》1148號的クロスレビュー對PlayStation Portable版給予27/40評分。<ref>{{cite web |title=ぱすてるチャイムContinue |url=https://www.famitsu.com/cominy/?m=pc&a=page_h_title&title_id=15234&redirect=no |website=ファミ通.com |accessdate=2019-03-26 |language=ja}}</ref>
