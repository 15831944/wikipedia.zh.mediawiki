== 花絮 ==
在导演选择男演员时，阿雅喊道“我要选[[李晨|李晨]]”，令张馨予尴尬不已。<ref>{{cite web |url= http://news.sina.com.cn/o/2013-08-24/104728036464.shtml |title=《粉红女郎》里演“男人婆”张馨予拒绝再“搭”李晨 |accessdate=2013 |publisher=新浪网 |work= |language=zh-hans}}</ref>

为了更加逼真，导演要求阿雅到飞机场向男人们抛媚眼、摆性感姿势，导致男人们纷纷逃离。<ref>{{cite web |url=http://gb.cri.cn/27564/2013/08/13/3465s4215799.htm |title=《粉红女郎》曝制作特辑 阿雅"放电"吓坏路人 |accessdate=2013 |publisher=国际在线 |work= |language=zh-hans}}</ref>

张馨予的角色则是一个女汉子形象。<ref>{{cite web |url=http://ent.163.com/13/0828/11/97C4LKEE00034C67.html |title=《粉红女郎》热映 张馨予演绎新一代女汉子 |accessdate=2013 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.163.com/13/0823/11/96V6R9OO000300B1.html |title=《粉红女郎》公映 张馨予本色出演“男人婆” |accessdate=2013 |publisher= 网易|work= |language=zh-hans}}</ref>
