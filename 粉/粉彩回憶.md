== 故事簡介 ==
20XX年，被稱為御宅族聖地的秋葉原所盛行的御宅文化開始衰退。在秋葉原經營御宅商店「兔小屋總店」的店長在神秘兔子「扭曲兔子」的引導下，知道了御宅文化衰退的原因是被一種名為病毒的怪物所影響，而扭曲兔子能給予在店裡打工的女孩變身的能力，希望她們消滅病毒並找回消失的御宅文化。

== 登場角色 ==
=== 兔小屋總店 ===
; 店長
: 兔小屋總店的店長，從祖父那裡繼承了店面。本身對病毒免疫並且記得大多數的作品，因此成為女孩們進入作品世界的媒介。
; {{nihongo|淺木泉水|浅木 泉水（あさぎ いずみ）|聲：[[新田日和|新田日和]]<ref name="character">{{cite web|url=https://pasumemotv.com/staffcast/|title=聲優|website=電視動畫「粉彩回憶」官方網站|language=ja}}</ref>}}
: 皋月橋高中二年級生。跟店長從小是青梅竹馬。
; {{nihongo|榊亞矢香|榊 亜矢香（さかき あやか）|聲：[[鳥部萬里子|鳥部萬里子]]<ref name="character"/>}}
: 皋月橋高中二年級生，泉水的好友。有著蒐集[[BL_(和製英語)|BL]]書籍的興趣。
: 少年漫畫忠實粉絲，但只要有旁人對少年漫畫相關設定有意見，突然會進入“令人畏懼”的認真模式並說著少年漫畫精髓&美妙之處，受害者呈現彷彿被洗腦而神智不清。
; {{nihongo|伊莉娜·列斯科娃|イリーナ・レスコヴァ|聲：[[內山夕實|內山夕實]]<ref name="character"/>}}
: 皋月橋高中三年級生，從俄羅斯來到日本留學。個性認真，熱愛日本文化。

: 優良木之坂高中二年級生，二條院財閥的大小姐。是位完美主義者，既使中途遇到困境，不論如何總能想出突破困境的方法。
; {{nihongo|町屋結衣奈|町屋 結衣奈（まちや ゆいな）|聲：[[村川梨衣|村川梨衣]]<ref name="character"/>}}
: 優良木之坂高中三年級生。個性自由奔放且感情豐富，有著如同貓咪般的個性。
; {{nihongo|江戶川橋美智|江戸川橋 美智（えどがわばし みち）|聲：[[大空直美|大空直美]]<ref name="character"/>}}
: 優良木之坂高中一年級生。個性沉默寡言且散發神秘氣質。
; {{nihongo|千住南海|千住 南海（せんじゅ みなみ）|聲：{{link-ja|山本亞衣|山本亜衣 (声優)}}<ref name="character"/>}}
: 鶯里高中三年級生。小時候曾住在京都因此會說[[京都方言|京都方言]]。擅長料理與家事，因而擔任店裡的經理。
; {{nihongo|舞子智真理|舞子 ちまり（まいこ ちまり）|聲：[[小倉唯|小倉唯]]<ref name="character"/>}}
: 鶯里高中一年級生。個性認真有禮貌，但性格膽小，受驚嚇過度時會馬上昏倒。也會沿用日本戰國武將的名句來表達。
; {{nihongo|目白渚央|目白 渚央（めじろ なお）|聲：[[金元壽子|金元壽子]]→[[花守由美里|花守由美里]]<ref name="character"/>}}
: 鶯里高中一年級生。打扮較為男孩子氣，擅長運動。
: 但其實不擅長料理，所做出的菜色十分不理想。只要有旁人稱讚就突然性格變得很少女化，連料理水準也跟著變好，但只要冷靜下來就會變回原本性格。
; {{nihongo|來島怜|来島 怜（くるしま れい）|聲：[[久保百合花|久保百合花]]<ref name="character"/>}}
: 綾巴學園高中二年生。喜歡可愛的東西，個性不坦率。
; {{nihongo|里中小町|里中 小町（さとなか こまち）|聲：[[戶田惠|戶田惠]]<ref name="character"/>}}
: 綾巴學園高中二年生。總是攜帶著相機。
; {{nihongo|六鄉沙織|六郷 沙織（ろくごう さおり）|聲：[[小岩井小鳥|小岩井小鳥]]<ref name="character"/>}}
: 綾巴學園高中一年生。擅長發明物品並已取得多項專利。
: 有隱性人格，每個月某天會突然變得非常消極甚至不那麼主動，過完一天後就能變回原性格，但不記得自己在消極時發生甚麼事。
; {{nihongo|扭曲兔子|ねじれウサギ|聲：[[森永千才|森永千才]]<ref name="character"/>}}
: 兔小屋總店的吉祥物，能給予女孩們進入各種作品中擊敗病毒的能力。

== 用語 ==
; 作品世界
: 與現實對應作品而誕生的世界，也存在多數作品的世界。

; Memories變身
: 因了解這部作品要素而產生此能力，喊Memories變身後會變成符合該作品風格的角色。

; 黑色菁英
: 破壞世界的夢想的邪惡組織，並將母病毒寄送作品世界進行侵略。
