'''粉嶺戲院'''位於[[香港|香港]][[新界|新界]][[粉嶺|粉嶺]][[聯和墟|聯和墟]]聯發街1號，於1959年9月11日開業，2010年1月6日結業，正式結束其半世紀的歷史使命。粉嶺戲院曾是全港最舊、最後一間的鄉村[[戲院|戲院]]，亦為[[上水|上水]]、[[粉嶺|粉嶺]]及[[大埔區|大埔區]]唯一的戲院。粉嶺戲院由粉嶺戲院有限公司經營，內設有兩間迷你戲院，合共721座位<ref>{{cite web |url=http://www.fdc.gov.hk/filmzone/tc/cinema_info/cinema_info_45.htm |title=存档副本 |accessdate=2008-10-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20081102185011/http://www.fdc.gov.hk/filmzone/tc/cinema_info/cinema_info_45.htm |archivedate=2008-11-02 }} 香港電影發展局 粉嶺戲院資料</ref>，主要放映首輪西片。目前戲院原址已出租予私人公司。

== 歷史 ==
粉嶺戲院前身是一間露天戲院，於1949年開始營業，為新界[[北區_(香港)|北區]]第一間電影院。當時設備簡陋，只有四面牆壁作隔音，座椅全為木製<ref>http://life.mingpao.com/cfm/probe3.cfm?File=20080708/probe02/gfp1.txt 《明報》校際專題報道比賽傑出獎：聯和墟</ref>。現時戲院的建築物於1959年改建而成，開幕當天由[[新界|新界]]副行政署長[[黎敦義|黎敦義]]主持揭幕，他致詞時說：「（[[聯和墟|聯和墟]]）居民靠工農事業維持生活，貴院今日之開幕，將使本區居民獲到娛樂享受，即亦可藉娛樂事業而引起農村之繁榮。」<ref>http://ol.mingpao.com/cfm/style5.cfm?File=20080418/sta02/wia3.txt 新界粉嶺戲院：北區戲院不倒翁</ref>

踏入廿一世紀，當北區各戲院相繼結業，粉嶺戲院成為最後一間碩果僅存位於新界北區的戲院，其間更轉型為專播放[[三級片|三級片]]的另類戲院。可惜，粉嶺戲院最後亦不敵時代的變遷，於2010年1月10日悄悄地低調結業，無聲告別聯和墟的眾街坊，原因之一是跨或留原區看電影成為粉嶺市民的慣性，例如使用港鐵東鐵線直往沙田市區、九龍塘又一城、原區逸峰MCL、旺角新世紀廣場、大埔娛藝、西洋菜街百老匯、原區及黃埔花園嘉禾，乘坐大老山隧道巴士路線277X前往九龍灣星影匯、MCL和觀塘Palace APM，乘坐獅子山隧道巴士路線270A前往尖沙咀和油麻地百老匯。

2015年12月，[[洲立影藝|洲立影藝]]有限公司向[[東方日報_(香港)|東方日報]]證實，MCL院線正積極籌備於粉嶺[[逸峯|逸峯廣場]]開設新戲院，於2017年1月21日開幕，並設3個影院。<ref>{{cite news|url=http://hk.on.cc/hk/bkn/cnt/news/20151228/bkn-20151228154518871-1228_00822_001.html|title=闊別6年戲院重臨北區　粉嶺MCL明秋開幕|date=2015年12月28日|newspaper=''[[東方日報_(香港)|東方日報 (香港)]]''}}</ref>

== 特色 ==
粉嶺戲院保留了舊式戲院的單幢建築，樓高兩層，為現時香港碩果僅存的單棟戲院建築。建築物頂部為三角尖頂，上面鋪有瓦片，極具傳統風味。裝修後戲院內部仍保留舊式戲院的暗斜地台、乳膠墊座椅、銀幕四周的木條裝飾，而人手劃位亦為該院的特色。因此成為不少劇集取景的地方，[[無線|無線]]劇集[[胭脂水粉|胭脂水粉]]亦曾到粉嶺戲院取景。2016年9月，[[古物古蹟辦事處|古物古蹟辦事處]]將戲院評定為[[三級歷史建築|三級歷史建築]]。
