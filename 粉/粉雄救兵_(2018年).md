'''《粉雄救兵》'''（{{Lang-en|Queer Eye}}）是[[美国|美国]][[流媒体|流媒体]]平台[[网飞|网飞]]的一档[[真人秀|真人秀]]节目，于2018年2月7日首播。该节目是[[美国精彩电视台|美国精彩电视台]][[粉雄救兵|同名真人秀]]的重启，由全新的闪耀五人组（{{lang|en|Fab 5}}）主持。全新的闪耀五人组包括：美酒与佳肴专家{{Link-en|安东尼·波罗夫斯基|Antoni Porowski}}、时尚专家{{Link-en|谭·法兰斯|Tan France}}、文化专家{{Link-en|卡拉莫·布朗|Karamo Brown}}、室内设计专家{{Link-en|鲍比·伯克|Bobby Berk}}和发型专家{{Link-en|乔纳森·凡·奈斯|Jonathan Van Ness}}。2018年7月，《粉雄救兵》续约了第三季，并在2019年3月15日发布。

== 反响评论 ==
在[[汇总媒体|评论聚合]]网站[[烂番茄|烂番茄]]、[[Metacritic|Metacritic]]、[[豆瓣|豆瓣网]]上，《粉雄救兵》都收获了较好的评价。

第一季，《粉雄救兵》在烂番茄上的新鲜度为97%，专业剧评人给出的平均分为8.23分（满分10分）；而普通观众给出的爆米花指数为88%，平均得分为4.4分（满分5分）。<ref>{{cite web |title=QUEER EYE: SEASON 1 |url=https://www.rottentomatoes.com/tv/queer_eye_2018/s01 |publisher=[[烂番茄|烂番茄]] |accessdate=2019-03-17}}</ref> 而在Metacritic上的得分为73分（满分100分）。<ref>{{cite web |title=QUEER EYE (2018) : SEASON 1 |url=https://www.metacritic.com/tv/queer-eye-2018 |publisher=[[Metacritic|Metacritic]] |accessdate=2019-03-17}}</ref> 豆瓣网给出的评分为9.2分（满分10分）。<ref>{{cite web |title=粉雄救兵 第一季 |url=https://movie.douban.com/subject/26964443/ |publisher=[[豆瓣|豆瓣]] |accessdate=2019-03-17}}</ref>

第二季，《粉雄救兵》在烂番茄上的新鲜度为85%，专业剧评人给出的平均分为8.36分（满分10分）；而普通观众给出的爆米花指数为83%，平均得分为4.1分（满分5分）。<ref>{{cite web |title=QUEER EYE: SEASON 2 |url=https://www.rottentomatoes.com/tv/queer_eye_2018/s02 |publisher=[[烂番茄|烂番茄]] |accessdate=2019-03-17}}</ref> 而在Metacritic上的得分为79分（满分100分）。<ref>{{cite web |title=QUEER EYE (2018) : SEASON 2 |url=https://www.metacritic.com/tv/queer-eye-2018/season-2 |publisher=[[Metacritic|Metacritic]] |accessdate=2019-03-17}}</ref> 豆瓣网给出的评分为9.4分（满分10分）。<ref>{{cite web |title=粉雄救兵 第二季 |url=https://movie.douban.com/subject/30180156/ |publisher=[[豆瓣|豆瓣]] |accessdate=2019-03-17}}</ref>
