'''粉碎四人帮'''，当时称为“一举粉碎四人帮”，又称'''懷仁堂事變'''，指1976年10月6日，时任[[中國共產黨中央委員會副主席|中共中央第一副主席]]、[[中华人民共和国国务院总理|国务院总理]][[華國鋒|華國鋒]]联合[[中國共產黨中央委員會副主席|中共中央副主席]][[葉劍英|葉劍英]]和[[中共中央办公厅|中共中央办公厅]]主任[[汪東興|汪東興]]等人将时任[[中共中央政治局委员|中共中央政治局委员]][[王洪文|王洪文]]、[[張春橋|張春橋]]、[[江青|江青]]、[[姚文元|姚文元]]、毛远新五人于北京市[[中南海|中南海]]以“隔离审查”名义抓捕并拘禁的事件。事後，華國鋒接任[[中國共產黨中央委員會主席|中國共產黨中央委員會主席]]，結束了[[毛泽东|毛泽东]]所发动的“[[文化大革命|无产阶级文化大革命]]”。後來，[[四人幫|四人幫]]及其主要追随者均遭到[[揭批查运动|政治清洗]]，有的被[[林彪、江青反革命集团案|刑事追究]]。

== 背景 ==
十年[[文化大革命|文革]]使得國家陷入一片混亂，民眾支持鄧小平[[1975年整頓|整頓秩序、發展經濟]]的政策，而對掌握大權的[[四人幫|四人幫]]及其激進路線不滿。[[毛澤東之死|毛澤東去世]]後，黨內出現權力真空，通過文革造反起家的四人幫與文革中受到打擊的老幹部在路線上存在巨大分歧，內部鬥爭異常激烈。<ref>{{cite web|url=https://www.youtube.com/watch?v=ECAlOxdv2mc|title=搜狐视频大视野：四人帮覆灭记第1集：主席身后 纷争暗起}}</ref>

== 策劃 ==
1976年9月10日、即毛泽东逝世的第二天，中共中央副主席[[王洪文|王洪文]]未经[[中国共产党中央委员会|中共中央]]授权，指示其工作人员[[米士奇|米士奇]]<ref>{{cite book |author=[[葉永烈|葉永烈]]
 |title=《“四人幫”興亡》 |url=http://cul.china.com.cn/lishi/2012-04/13/content_4937353.htm |publisher=人民日報出版社 |year=2009 }}</ref>以[[中共中央办公厅|中共中央办公厅]]名义通知各省、[[直辖市|直辖市]]、[[自治区|自治区]]：如发生重大问题，要及时报告；重要问题不好解决的，要及时请示；凡报告和请示，均与米士奇直接联系。

9月11日，王洪文背着[[中国共产党中央政治局|中共中央政治局]]，在中南海另設「中央辦公廳值班室」，並用中共中央辦公廳名義通知全國各地，重大問題要及時向他們指定值班人員請示，企圖切斷中共中央、華國鋒同各省、市、自治區之聯繫，由他們發號施令指揮全國<ref name="共和国史稿第三卷" >{{cite book |author=[[当代中国研究所|当代中国研究所]] |title=《[[中华人民共和国史稿|中华人民共和国史稿]]》第三卷 |location=北京 |publisher=人民出版社 |date=2012-09-01 }}</ref>{{rp|302}}。华国锋从[[中共湖南省委|中共湖南省委]]书记[[张平化|张平化]]给他电话中得知此事，同[[叶剑英|叶剑英]]商议，决定以中央政治局名义通知王洪文，立即关闭擅自设立之中央办公厅值班室，并以中共中央名义通知各地：发生重大问题，应向华国锋请示<ref name="共和国史稿第三卷" />{{rp|298}}。在事前电话约定后，华国锋以到医院检查身体为由，到西皇城根9号[[李先念|李先念]]临时住处，谈话不到十分钟。华国锋谈到：“同四人幫斗争不可避免，是時候解決了”<ref name="共和国史稿第三卷" />{{rp|299}}。华国锋请李先念代表他去見叶剑英，就解決“四人幫”问題请求叶剑英意見，又同他研究此事<ref name="共和国史稿第三卷" />{{rp|299}}。当天华国锋又请求[[汪东兴|汪东兴]]给予协助，汪东兴同意了。9月14日，李先念借去香山植物园的名义外出，转往[[玉泉山|玉泉山]]9号楼叶剑英住处。李先念传达华国锋的意見，並同他研究此事<ref name="共和国史稿第三卷" />{{rp|2989}}；当天，李先念即向华国锋汇报谈话内容和叶剑英的态度。

9月16日，华国锋召集李先念、[[吴德|吴德]]、[[陈锡联|陈锡联]]、[[纪登奎|纪登奎]]、[[陈永贵|陈永贵]]等人在国务院会议厅开会。其间华国锋提问：“毛主席提出的四人帮的问题，怎么解决？”纪登奎说：“对这些人，恐怕还是要区别对待。”华国锋以此将中立的纪登奎排除在行动以外。

9月17日下午，[[中国共产党中央政治局常务委员会|中共中央政治局常委会]]扩大会议在[[人民大会堂|人民大会堂]]新疆厅召开，主要议题为毛泽东手稿如何处理，以及[[毛远新|毛远新]]是否需要回到辽宁。四人帮与华国锋及叶剑英发生争执。之后，华国锋又获得了[[陈锡联|陈锡联]]與[[苏振华|苏振华]]的支持。

9月21日，叶剑英来到华国锋住处商讨初步方案。9月26日晚，[[国务院小礼堂|国务院小礼堂]]电影放完后，华国锋留下李先念和吴德商量方案。吴德提出逮捕和投票否决的形式，经商议确定采用逮捕{{NoteTag|李先念反对开会投票解决。}}。华国锋再次委托李先念将密商结果通报叶剑英。 

9月29日深夜23时，华国锋主持中共中央政治局会议在人民大会堂东大厅召开，主要讨论[[中华人民共和国国庆节|国庆节]]的各项活动安排。期间江青要求散会，留下江青、张春桥、王洪文、姚文元、华国锋、汪东兴六人。江青要求，毛泽东的文件、档案应交毛远新清理、保管，中央办公厅办理的群众来信来访工作和材料交由他们在[[北京大學|北京大學]]组织的人去处理；但华国锋没有回应<ref name="实施过程" >{{cite web |url=http://www.tianyabook.com/junshi/sirenbang.htm |title=粉碎“四人帮”的策划、实施过程 }}</ref>。

9月30日晚，中共中央在[[天安门|天安门城楼]]上举行首都各界人士国庆座谈会。会后，华国锋、李先念、吴德等人到国务院小礼堂看了一场电影；看完电影，三人又在小礼堂旁边的小会议室再次商谈了解决四人帮的时间和可能发生的问题。

政治局势变得异常复杂。10月2日，毛遠新調派[[中国人民解放军沈阳军区|瀋陽軍區]]1個坦克師開往北京，同時要求上海民兵、北京大學民兵和[[清華大學|清華大學]]民兵做好準備<ref name="寒山碧" >{{cite book |authors=[[寒山碧|寒山碧]]原著，[[伊藤潔|伊藤潔]]縮譯，唐建宇、[[李明|李明]]譯 |title=鄧小平傳|location=香港 |publisher=東西文化事業公司 |year=1993 }}</ref>{{rp|170}}。瀋陽軍區[[司令员|司令員]][[李德生|李德生]]，向葉劍英報告毛遠新出動部隊，葉立即下令軍隊退回原地<ref name="寒山碧" />{{rp|170}}。四人幫住宅戒備森嚴，每個住宅有二重以上警備<ref name="寒山碧" />{{rp|171}}。同日15时许，叶剑英来到汪东兴在中南海南楼的办公室交换意见；当晚，汪东兴、华国锋、吴德三人在[[东交民巷|东交民巷]]的住地交谈。华国锋之后秘密召见了[[中国人民解放军北京卫戍区|北京卫戍区]]司令员[[吴忠_(少将)|吴忠]]，获得了支持。10月4日下午，华国锋等三人最终敲定了方案<ref>{{cite web |url=http://mil.eastday.com/m/20070416/u1a2769027.html |title=惊心动魄的1976:中共中央抓捕"四人帮"决策纪实 }}</ref>。10月4日傍晚，葉劍英趕到東交民巷華國鋒住所，商量對策<ref name="共和国史稿第三卷" />{{rp|301}}。原定部署國慶節後準備10天視情況再動手；葉劍英提出改為提前採取行動，「先發制人，以快打慢」，下決心「一破一立除四害」<ref name="共和国史稿第三卷" />{{rp|301}}；華國鋒決定「至遲後天動手」，請葉劍英同汪東興落實行動計劃<ref name="共和国史稿第三卷" />{{rp|301}}。葉劍英即到汪東興處，詳細商決一切，使汪東興提出之行動方案周全可行<ref name="共和国史稿第三卷" />{{rp|301}}。

== 行動 ==
10月6日下午3时，中共中央办公厅通知中央政治局常委到中南海[[怀仁堂|怀仁堂]]开会，内容如下：一是研究《[[毛泽东选集第五卷|毛泽东选集第五卷]]》的出版问题；二是研究[[毛主席纪念堂|毛主席纪念堂]]的选址问题。下午，汪东兴对行动人员召开短会。下午5时，中南海东八所小会议室也有小型会议分配任务。汪东兴、叶剑英和华国锋分别于晚上6時30分、7时20分和7时40分到达怀仁堂。晚上8时，是同“四人帮”鬥爭胜负之关键时刻<ref name="共和国史稿第三卷" />{{rp|301}}。华国锋、叶剑英在中南海怀仁堂召集中央政治局常委会议，通知姚文元列席<ref name="共和国史稿第三卷" />{{rp|301}}。在张春桥、王洪文、姚文元先後到达会议室时，分別宣布对他们实作隔离审查<ref name="共和国史稿第三卷" />{{rp|301}}；同时，派人到江青住处宣布执行同样决定<ref name="共和国史稿第三卷" />{{rp|301}}。张春桥先到场，立即被捕；王洪文企图反抗，被控制；姚文元只说了声“走吧”，就跟着士兵离开。江青则在中南海春藕斋西侧201号楼被宣布隔离审查；江青在宣读后問“能不能再念一遍”，然后写了一封信并密封交给工作人员，随即被运送至秘密地点[[软禁|软禁]]{{NoteTag|一说关押在中南海某处地下室。}}近8个月，再转至[[秦城监狱|秦城监狱]]<ref>马晓先：《我给江青当护士》</ref>。同时[[迟群|迟群]]、[[谢静宜|谢静宜]]、[[毛远新|毛远新]]等也被捕。晚上10时，[[耿飚|耿飚]]到达[[中央人民广播电台|中央人民广播电台]]，把华国锋的手令交给了[[中华人民共和国广播电影电视部|中央广播事业局]]局长[[邓岗|邓岗]]，然后派人迅速控制了[[中央三台|中央三台]]。

== 后续 ==
四人帮被抓后，华国锋在[[玉泉山|玉泉山]]9号楼叶剑英住处连夜召开中央政治局紧急会议<ref>[http://space.tv.cctv.com/act/article.jsp?articleId=ARTI1209107669566495&nowpage=3 叶剑英决定“摊牌” 抓捕四人帮全程记录] {{webarchive|url=https://web.archive.org/web/20090523144946/http://space.tv.cctv.com/act/article.jsp?articleId=ARTI1209107669566495&nowpage=3 |date=2009-05-23 }}</ref>。晚上10时许，出席会议者有中央政治局成员叶剑英、李先念、汪东兴、陈锡联、苏振华、纪登奎、吳德、倪志福、陈永贵、吳桂贤，共11人<ref name="共和国史稿第三卷" />{{rp|301-302}}。叶剑英提议，中央政治局一致通过，华国锋任[[中国共产党中央委员会主席|中国共产党中央委员会主席]]、[[中国共产党中央军事委员会主席|中国共产党中央军事委员会主席]]，将来提请中央全会追认<ref name="共和国史稿第三卷" />{{rp|302}}。华国锋正式成为继毛泽东以后的[[党和国家最高领导人|最高领导人]]<ref>[http://news.xinhuanet.com/politics/2008-10/07/content_9728801.htm 1976年 中共中央政治局一致通过华国锋为中央委员会主席(10月7日)] {{webarchive|url=https://web.archive.org/web/20140817055831/http://news.xinhuanet.com/politics/2008-10/07/content_9728801.htm |date=2014-08-17 }}</ref>。会后，即下发关于华国锋任职决议，通知「立即在党內传达」<ref name="共和国史稿第三卷" />{{rp|302}}。会议开到10月7日凌晨4点多才结束<ref name="共和国史稿第三卷" />{{rp|302}}。李德生根據華國鋒、葉劍英命令逮捕毛遠新等人<ref name="寒山碧" />{{rp|172}}。中共中央用電話通知[[中国共产党上海市委员会|上海市委]]書記[[马天水|马天水]]（1977年初被停职审查，1988年死于上海精神病院）和[[中国人民解放军上海警备区|上海警備區]]司令員[[周纯麟|周纯麟]]到北京召開緊急會議，在北京機場被逮捕<ref name="寒山碧" />{{rp|173}}。

10月18日，中共中央印发《关于王洪文、张春桥、江青、姚文元反党集团事件的通知》，要求向全体党员传达毛泽东批评四人帮一系列指示，说明党中央同四人帮斗争之经過、以及对四人帮实行隔离审查的原因，指导开展“揭发批判四人帮运动”<ref name="共和国史稿第三卷" />{{rp|302}}。10月21日，中共中央通过广播和报纸，把一举粉碎“四人帮”公之于世<ref name="共和国史稿第三卷" />{{rp|302}}。时任[[第四届全国人民代表大会常务委员会|第四届全国人大常委会]][[全国人民代表大会常务委员会副委员长|副委员长]][[郭沫若|郭沫若]]在当天还即兴填[[词_(文学)|词]]一首《[[水调歌头|水调歌头]]·粉碎四人帮》，当时流传甚广：

{{quote|
大快人心事，揪出四人帮。政治流氓文痞，狗头军师张。还有精生白骨，自比则天武后，铁帚扫而光。篡党夺权者，一枕梦黄梁。

野心大，阴谋毒，诡计狂。真是罪该万死，迫害红太阳。接班人是俊杰，遗志继承果断，功绩何辉煌。拥护华主席，拥护党中央！
}}

10月24日，北京百万军民在[[天安门广场|天安门广场]]隆重举行庆祝大会，庆祝粉碎“四人帮”<ref name="共和国史稿第三卷" />{{rp|303}}。

此后全国范围内开展了揭发批判林彪、“四人帮”的罪行、清查文革中与林彪“四人帮”有牵连的人和事的[[揭批查运动|揭批查运动]]。

1980年公开审判[[林彪、江青反革命集团案|林彪、江青反革命集团案]]，并判处相应刑罚。

== 影响和评价 ==
中共官方以“粉碎‘四人帮’[[反革命|反革命]]集团的胜利，标志着历时10年的‘[[文化大革命|文化大革命]]’动乱从此结束”来评价此事<ref>[http://dangshi.people.com.cn/GB/221024/221026/221623/15018744.html 粉碎“四人帮”]</ref>。
