'''濱川崎站'''（{{jpn|j=浜川崎駅|hg=はまかわさきえき|rm=Hama-Kawasaki eki|f=y}}）是一個位於[[神奈川縣|神奈川縣]][[川崎市|川崎市]][[川崎區|川崎區]]，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）、[[日本貨物鐵道|日本貨物鐵道]]（JR貨物）的[[鐵路車站|鐵路車站]]。

== 停站路線 ==
停站路線有[[東海道本線|東海道本線]]（貨物支線）、[[南武線|南武線]]（濱川崎支線、旅客導覽是「南武支線」）、[[鶴見線|鶴見線]]3條路線，當中東海道本線是此站的[[日本鐵路車站#所屬線|所屬線]]。但是旅客營業只有南武線與鶴見線，是兩條路線的轉乘站。此站位於川崎市內，但是根據[[特定都區市內|特定都區市內]]制度被視為「橫濱市內」的車站。南武支線的[[車站編號|車站編號]]是'''JN 54'''、鶴見線是'''JI 08'''。

通往{{link-ja|東京貨物總站|東京貨物ターミナル駅}}方向的東海道本線支線供[[貨物列車|貨物列車]]專用で。此東海道本線支線與南武線濱川崎支線結合成{{link-ja|東海道貨物線|東海道貨物線}}，2條路線之間可以在此站互相停靠，並且可以直通運行的構造。

== 旅客站 ==
南武線與鶴見線的旅客站因道路夾着而分開了站舍。

兩條路線都是[[島式月台|島式月台]]1面2線的[[地面車站|地面車站]]，南武線側是1、2號月台，鶴見線側是3、4號月台。

== 貨物站 ==
JR貨物的貨物站位於旅客站的東側，設有JR貨物的站員駐在（委托{{link-ja|神奈川臨海鐵道|神奈川臨海鉄道}}濱川崎運輸所）。
