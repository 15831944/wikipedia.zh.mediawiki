'''濱松町站'''（{{jpn|j=浜松町駅|hg=はままつちょうえき|rm=Hamamatsuchō eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[港區_(東京都)|港區]]，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）、[[東京單軌電車|東京單軌電車]]的[[鐵路車站|鐵路車站]]。但是東京單軌電車正式站名是「單軌電車濱松町站」<ref>{{cite web|url=http://www.tokyo-monorail.co.jp/tc/guidance/tennouzuairu/index.html|title=東京單軌電車>單軌電車使用指南>單軌電車濱松町站>站內資訊|publisher=東京單軌電車|accessdate=2019-03-07}}</ref>。

JR東日本站區的所在地是[[海岸_(東京都港區)|海岸]]一丁目，東京單軌電車站區是[[濱松町|濱松町]]二丁目。

== 停站路線 ==
此站有JR東日本的各線（後述）與東京單軌電車的[[東京單軌電車羽田機場線|東京單軌電車羽田機場線]]停靠。另外，JR東日本車站的{{link-ja|3字母簡寫|3レターコード}}為「{{車站編號s|black|#ffffff|HMC}}」。
* JR東日本：各線（後述）
* 東京單軌電車：[[File:Tokyo_Monorail_Line_symbol.svg|15px]] 東京單軌電車羽田機場線 - [[車站編號|車站編號]]「'''MO 01'''」
JR東日本車站停靠的路線在軌道名稱上稱為[[東海道本線|東海道本線]]（詳情參見路線條目與{{link-ja|鐵道路線的名稱|鉄道路線の名称}}），運轉系統則分為行駛[[電車線與列車線|電車線]]的[[京濱東北線|京濱東北線]]列車與[[山手線|山手線]]列車，因此旅客資訊不使用「東海道（本）線」。另外，此站是往{{link-ja|東京貨物總站|東京貨物ターミナル駅}}方向的東海道本線支線（{{link-ja|東海道貨物線|東海道貨物線}}）的起點站，但是現在此站－東京貨物總站之間停駛。
* [[File:JR_JK_line_symbol.svg|15px]] [[京濱東北線|京濱東北線]]：行走電車線的東海道本線、[[東北本線|東北本線]]近距離列車。經[[橫濱站|橫濱站]]直通運行至根岸線。 - 車站編號「'''JK 23'''」
* [[File:JR_JY_line_symbol.svg|15px]] [[山手線|山手線]]：行走電車線的[[環狀線|環狀路線]] - 車站編號「'''JY 28'''」
依據[[特定都區市內|特定都區市內]]制度，本站屬於「[[特定都區市內#設定區域一覧|東京都區內]]」與「[[東京山手線內|東京山手線內]]」。

== 历史 ==
本站一帶是自日本鐵路通車時就有列車運行，然而當時未在本地設站。之後，[[鐵道省|國有鐵道]]東海道本線[[品川站|品川]] - [[新橋站|烏森]]（之後的新橋站）開通的1909年（明治42年）12月16日開設本站。鐵道通車當時的新橋站是之後的貨物站汐留站，而擁有東京往北鐵道網的[[日本鐵道|日本鐵道]]，其總站[[上野站|上野站]]與此地之間尚未連通。對此，本站附近開始建設從舊線分岐的高架橋。當時的舊新橋站是東海道本線的起點，本站是東海道本線支線上的車站。本站開業時僅停靠山手線電車，月台採島式月台1面2線構造。車站東側是竹芝與芝浦等[[東京港|東京港]]棧橋，因此成為往{{link-ja|伊豆七島|伊豆七島}}航線的入口站<ref name = "駅の歴史_149">[[#eki|駅の歴史]]、p.149</ref><ref>{{Cite book|author = 小野田滋 |title = 高架鉄道と東京駅［下］ |edition = 第1刷 |date = 2012-02-15 |publisher = 交通新聞社 |isbn = 978-4-330-26812-5 |pages=pp.90-91 }}</ref>。

1914年（大正3年）12月20日，[[東京站|東京站]]開業，以東京站為起點的市區高架線命名為東海道本線。本站也成為東海道本線的車站。東海道本線開始行駛長距離列車時，由於本站沒有長距離列車使用的月台，所以只能停靠近距離電車。配合東京站開業，京濱線電車（之後的京濱東北線）開始運轉，並停靠本站<ref name = "駅の歴史_149" />。

山手線與京濱東北線在田町 - 田端之間共用同一線路，然而隨著旅客的增加，開始進行線路增設與分離工程，並於1956年（昭和31年）11月19日正式啟用。從此，本站JR月台改成現在的島式月台2面4線構造<ref>[[#eki|駅の歴史]]、p.127</ref>。

1964年（昭和39年）9月17日，東京單軌電車在此設站，成為機場接駁總站<ref name="mono50th-9">[[#50th|50年史]]、p.9。</ref><ref name="asahi-np-2015-2-14">上沢博之(2015年2月14日). “東京五輪物語 浜松町-羽田モノレール開通 空港まで15分、突貫工事”. [[朝日新聞|朝日新聞]](朝日新聞社)</ref>。2000年（平成12年）12月12日，附近的都營地下鐵大江戶線大門站開業<ref name = "駅の歴史_149" />。

過去[[東海道新幹線|東海道新幹線]]靠海側有前往{{link-ja|東京貨物總站|東京貨物ターミナル駅}}的貨物線（通稱大汐線），但因應[[都營地下鐵大江戶線|都營地下鐵大江戶線]]工程而停駛。另外，1990年至1998年的旅遊季，貨物線上會開設特設月台發出往九州與北海道方向的[[人車同行|人車同行列車]]。

=== 年表 ===
* 1909年（[[明治|明治]]42年）12月16日：[[鐵道省|鐵道院]]東海道本線[[品川站|品川]]－[[新橋站|烏森]]間開通，濱松町站開業。
* 1914年（[[大正|大正]]3年）12月20日：京濱線電車（現在京濱東北線）開始運轉。
* 1949年（昭和24年）6月1日：[[日本國有鐵道|日本國有鐵道]]成立。
* 1956年（[[昭和|昭和]]31年）11月19日：京濱東北線與山手線分離運行。
* 1964年（昭和39年）
** 9月17日：單軌電車濱松町站開業<ref name="mono50th-9"/><ref name="asahi-np-2015-2-14" />。
** 10月1日：都營地下鐵淺草線大門站開業，開始轉乘業務。
* 1965年（昭和40年）11月：東京單軌電車獨自出資建設的國鐵南口剪票口與國鐵單軌連絡天橋啟用<ref>[[#50th|50年史]]、p.249。</ref>。
* 1968年（昭和43年）10月1日：國鐵車站停止貨運業務。
[[File:Hamamatsucho_Station_Tokyo_Japan01s3.jpg|thumb]]
* 1974年（昭和49年）9月：單軌電車月台有效長度延長至現行的90公尺<ref>[[#50th|50年史]]、p.15。</ref>。
* 1984年（昭和59年）
** 3月：國鐵濱松町站的跨站式站房完工<ref>[[#50th|50年史]]、p.25。</ref>。
** 4月21日：單軌電車濱松町站改建工程完工，新設電扶梯與階梯<ref>{{Cite journal|author = 石川尹巳 |title = モノレール浜松町駅改修工事完成 |journal = 鉄道ピクトリアル |issue = 435 |year = 1984 |month = 8 |pages = 91}}</ref>。
* 1985年（昭和60年）前年竣工的濱松町站內跨線天橋獲得{{link-ja|土木學會田中賞|土木学会田中賞}}。
* 1987年（昭和62年）4月1日：[[國鐵分割民營化|國鐵分割民營化]]，東海道本線（山手線、[[京濱東北線|京濱東北線]]）由JR東日本繼承。
* 1988年（昭和63年）3月13日：京濱東北線快速班次開始運行，該線午間時段不停本站。
* 1989年（[[平成|平成]]元年）7月：單軌電車濱松町站在月台設置冷氣<ref>[[#50th|50年史]]、p.59。</ref>。
* 1990年（平成2年）5月21日：啟用營業里程，{{link-ja|東海道貨物線|東海道貨物線}}將本站劃入路線內。
* 1996年（平成8年）7月18日：單軌電車濱松町站開設南口<ref>[[#50th|50年史]]、p.60。</ref>。
* 1998年（平成10年）
** 1月30日：因應都營地下鐵大江戶線工程，東海道貨物線濱松町－東京貨物總站間（通稱「大汐線」）停駛。
** 11月15日：單軌電車濱松町站3樓開設'''MCAT'''（單軌電車城市航空總站）<ref>[[#50th|50年史]]、p.62。</ref>。
* 2000年（平成12年）12月12日：都營地下鐵大江戶線大門站開業。
* 2001年（平成13年）11月18日：JR東日本啟用[[IC卡|IC卡]]「[[Suica|Suica]]」。
* 2002年（平成14年）
** 4月21日：東京單軌電車啟用IC卡「Suica」<ref name="mono50th-74">[[#50th|50年史]]、pp.74-75。</ref>。
** 7月14日：改點，本站成為京濱東北線快速電車停靠站<ref name="mono50th-74"/><ref>{{Cite journal|title=鉄道記録帳 |journal = RAIL FAN |date = 2002年10月号 |issue = 10 |volume = 49 |publisher = 鉄道友の会 |page = 22 }}</ref>。改善車站設備。
* 2003年（平成15年）7月19日：JR與東京單軌電車新設聯絡通道<ref>{{Cite journal|title = 鉄道記録帳2003年7月 |journal = RAIL FAN |date = 2003-10-01 |issue = 10 |volume = 50 |publisher = 鉄道友の会 |page = 22}}</ref>。
* 2008年（平成20年）11月27日：MCAT閉店<ref name="mono50th-63">[[#50th|50年史]]、p.63。</ref>。
* 2012年（平成24年）10月31日：{{link-ja|VIEW PLAZA|びゅうプラザ}}閉店。

本站為[[島式月台|島式月台]]2面4線地面車站。

出入口分為[[新橋站|新橋]]側地面的北口、月台中段階梯、電扶梯、電梯往上的中央口、經由同付費區往[[東芝|東芝]]大廈與[[東京港#日出碼頭（日之出碼頭）|日之出棧橋]]方向與金杉橋方向的南口，共三處。中央口與南口是在月台上層的[[跨站式站房|跨站式站房]]，從檢票口可經[[世界貿易中心大廈_(東京)|世界貿易中心大廈]]連絡通道與其北側通道通往東京單軌電車車站。

2002年，JR東日本收購東京單軌電車，進行付費區改建工程、增設電扶梯、新設電梯、新設與東京單軌電車車站直通的連絡通道與出口專用檢票口，使兩方的轉乘更為便利。

山手線外回、京濱東北線南向月台（3、4號線月台）的田町側有座「[[尿尿小童|尿尿小童]]」銅像，會隨著季節更換衣裝。

位於世界貿易中心大廈旁，月台為2面1線的[[港灣式月台|港灣式月台]]設計。2面月台分別為上車用月台與下車用月台。但因為在停靠的列車未開出前，其他列車不能駛進，故一直未能增加班次。

東京單軌電車的正式車站名稱為「單軌電車濱松町」，但車站與列車內的自動廣播、目的地表示、列車資訊顯示器等都省略為「濱松町」，其他旅客資訊也可見到「<small>單軌電車</small>濱松町」與「濱松町」混用的情形。

出入口有開業時設置的中央口，與1990年代後半在天王洲島側設置的南口。中央檢票口設有定期券售票處。

電梯分別連通於下車月台 - 中央檢票口間、下車月台 - 南檢票口間、東京單軌電車本社1樓 - 3樓間、中央檢票口與1樓間。下車月台 - 中央檢票口間為單行道。原先站內已有許多電扶梯，在JR東日本收購東京單軌電車後於3樓與JR付費區間再新設兩座電扶梯，配合前述的電梯與京濱東北線快速班次，可順利的消化[[羽田機場|羽田機場]]乘客。過去本站曾設有'''MCAT'''（單軌電車城市航空總站），但已於2010年9月30日結束營業<ref name="mono50th-63"/>。原址之後被改建為中央檢票口 - 乘車月台間的電梯與[[花旗銀行|花旗銀行]]出張所（2012年12月28日結束營業）。花旗銀行出張所用地在2013年6月7日開設保險代理店{{link-ja|保險窗口集團|ほけんの窓口グループ|保險窗口}}。
2009年6月，東京單軌電車向[[國土交通省|國土交通省]]提出計畫，將本站擴建為兩面月台兩條軌道（[[複線|複線]]）。。東日本旅客鐵道（JR東日本）也在定期記者會中表示，將進行濱松町站西口周邊開發，與JR、單軌電車車站整體改建。單軌電車延伸東京站的構想則因工期、成本問題，目前難以實行。

== 車站周邊 ==
[[File:world_trade_center_tokyo.JPG|thumb]]
車站周邊是大企業與中小企業辦公室聚集的都內知名辦公商區。車站西側為世界貿易中心大廈，東側為[[舊芝離宮恩賜庭園|舊芝離宮恩賜庭園]]。北口是[[汐留|汐留SIO SITE]]南邊的玄關口，在大型辦公大樓如{{link-ja|汐留芝離宮大廈|汐留芝離宮ビルディング}}、{{link-ja|汐留大廈|汐留ビルディング}}完成後，使用人數逐漸增加。

由於鄰近東京都交通局（[[都營地下鐵|都營地下鐵]]）的大門站，大江戶線車站的副站名為「濱松町」（建設時的臨時站名也是「濱松町」）。

JR線只有大江戶線的轉乘資訊。轉乘淺草線時在鄰站新橋轉乘較為便利。

另外，本站步行還可至往[[伊豆諸島|伊豆諸島]]與[[小笠原諸島|小笠原諸島]]的定期船碼頭[[東京港#竹芝碼頭|竹芝棧橋]]、水上巴士與海上巴士碼頭[[東京港#日出碼頭（日之出碼頭）|日之出棧橋]]、[[東京鐵塔|東京鐵塔]]、[[芝公園|芝公園]]、[[增上寺|增上寺]]、{{link-ja|港區役所 (東京都)|港区役所 (東京都)|港區役所}}等。

== 整備計畫 ==
濱松町站周邊被定為都市再生緊急整備地域、[[國土交通省|國土交通省]]、東京都、港區、JR東日本、東京單軌鐵路5者合作在濱松町周辺地區進行交通施設整備計劃。

2008年完工的汐留再開發令乘降客增加。
2012年10月，發表本站周邊的交通設施整備計畫概要，除了大幅拓寬付費區，2014年度將進行世界貿易中心大廈整修，2024年度完工<ref>[http://www.47news.jp/CN/201210/CN2012101801001453.html 世界貿易センタービルを建て替え　老朽化で、２４年度完成] 2012/10/18 19:02 共同通信</ref>。2013年2月的JR東日本定期記者會，宣布將進行JR、單軌電車車站整修工程<ref>[http://www.kensetsunews.com/?p=6798　浜松町駅改築を検討/西口周辺開発にあわせ/JR東日本] 2013-02-06　建設通信新聞</ref>。
