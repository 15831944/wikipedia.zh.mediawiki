'''濱田山站'''（{{jpn|j=浜田山駅|hg=はまだやまえき|rm=Hamadayama eki|f=y}}）是一個位於[[東京都|東京都]][[杉並區|杉並區]][[濱田山|濱田山]]三丁目，屬於[[京王電鐵|京王電鐵]][[井之頭線|井之頭線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''IN11'''。

== 歷史 ==
* 1933年（[[昭和|昭和]]8年）8月1日 - 隸屬於帝都電鐵的車站開始營運。
* 1940年（昭和15年）5月1日 - 與小田原急行鐵道合併，成為該公司帝都線的車站。
* 1942年（昭和17年）5月1日 - [[小田急電鐵|小田急電鐵]]與[[東京急行電鐵|東京急行電鐵]]（大東急）合併。
* 1948年（昭和23年）6月1日 - 從東急分離，成立京王帝都電鐵，成為該公司井之頭線的車站。
* 1996年（[[平成|平成]]8年） - 月台延伸、站房地下化。

== 車站構造 ==
[[File:濱田山站旁平交道.jpg|240px]]
具有1面2線[[島式月台|島式月台]]的[[地面車站|地面車站]]。月台的[[澀谷站|澀谷]]端及[[吉祥寺站|吉祥寺]]端皆設有歩行者専用[[平交道|平交道]]。

1995年井之頭線導入新型列車{{link-ja|京王1000系電車 (2代)|京王1000系電車 (2代)|1000系}}，新列車相較於舊型列車要來得長，為20m車5輛編成設計，因此進行了月台的延伸工程，也同時將車站站房地下化。在工程完成以前，設有站內平交道作為月台間的連絡道。

車站閘口設於地下，而車站出入口設於鐵道北側，需經由平交道才能往返車站南側的地區。
