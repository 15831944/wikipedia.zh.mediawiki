'''丹阳市'''为[[中国|中国]][[江苏省|江苏省]]下辖[[县级市|县级市]]，由地级[[镇江市|镇江市]]代管。丹阳为经济较发达、具备较强实力的[[县级行政区|县市]]，其综合竞争力[[中国百强县#县域经济论坛评价|2010年列江苏省县（市）的第八位，全国县（市）第十六位]]。

== 历史 ==
早在[[新石器时代|新石器时代]]，丹阳境内就有人类活动。丹阳建置始于[[战国|战国]]时期，初为[[云阳邑|云阳邑]]。公元前221年，[[秦始皇|秦始皇]]一统天下，实行郡县制，改云阳邑置[[云阳县_(秦朝)|云阳县]]。不久又更名为[[曲阿县|曲阿县]]。

[[新朝|新]]莽[[始建国|始建国]]元年(9年)改曲阿县为[[凤美县|凤美县]]。[[东汉|东汉]]初年复名曲阿县。[[東漢|東漢]]末年[[孫策|孫策]]和[[劉繇|劉繇]]在此地發生戰爭。

[[孫吳_(三國)|东吴]][[嘉禾_(年号)|嘉禾]]三年(234年)改曲阿县为云阳县。[[西晋|西晋]][[太康_(西晋)|太康]]二年(281年)复名曲阿县。

[[唐朝|唐]][[天宝_(唐朝)|天宝]]元年(742年)改[[润州_(隋朝)|润州]]为[[丹阳郡_(唐朝)|丹阳郡]]，曲阿县为[[丹杨县|丹杨县]]，属丹阳郡。这也是今天的丹阳市第一次出现丹阳之名，唐以前此地从未属于[[丹阳郡|丹阳郡]]，而是历属[[会稽郡|会稽郡]]、[[吴郡|吴郡]]、[[毗陵郡|毗陵郡]]、[[江都郡|江都郡]]等。[[乾元_(唐朝)|乾元]]元年(758年)改丹阳郡为润州，丹阳县仍属润州，丹阳之名逐渐专指此地。

民国元年(1912年)丹阳县直属[[江蘇省_(中華民國)|江苏省]]。1949年4月下旬，[[渡江战役|渡江战役]]开始，4月23日，[[中国人民解放军|中国人民解放军]]占领丹阳县全境<ref>{{Cite web|url=http://jsjjb.xhby.net/html/2009-06/11/content_23936.htm|title=总前委在丹阳|accessdate=2014-04-13|author=|date=2009年6月11日|work=|publisher=江苏经济报|language=Zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20141021064318/http://jsjjb.xhby.net/html/2009-06/11/content_23936.htm|archivedate=2014年10月21日}}</ref>。同年建[[镇江专区|镇江专区]]，丹阳县隶之。1958年9月改镇江专区为[[常州专区|常州专区]]，丹阳县属常州专区。1959年9月又改常州专区为镇江专区。1983年实行市管县后，丹阳县属镇江市。1987年12月，丹阳撤县设市，为丹阳市，仍属镇江市。

== 地理 ==
=== 区位 ===
丹阳的地理位置优越，地处[[长江|长江]]下游南岸，[[江苏省|江苏省]]南部，东距[[上海|上海]]200公里，西距[[南京|南京]]68公里。所辖区域位于东经119°24′～119°54′，北纬30°45′～32°10′，其南北长44公里，东西宽32.5公里。东邻[[常州市|常州市]][[新北区|新北区]]及[[武进|武进]]区，南接[[常州市|常州市]][[金坛区|金坛区]]，西、北部毗邻[[镇江市|镇江市]][[丹徒区|丹徒区]]，东北隔长江夹江与[[扬中市|扬中市]]相望。

全市总面积1047平方公里，其中陆地面积850.2平方公里，占总面积的81.2%；水域面积196.8平方公里，占18.8%。在全市陆地面积中，平原6.12万公顷，占陆地的72% ；低山丘陵岗地2.38万公顷，占28%。2002年底，全市耕地面积5.69万公顷，其中农田保护区5.38万公顷。

=== 地貌 ===
丹阳地势由西北向东南逐渐降低。[[九曲河|九曲河]]以北、[[京杭大运河|京杭大运河]]以东是[[宁镇丘陵|宁镇丘陵]]余脉的低山丘陵岗地，多低山丘陵，低山群中最高峰为水晶山，主峰海拔166米；西南接近[[茅山|茅山]]山麓。其余地区以平原为主，地形低平，一般在海拔7米左右，属[[太湖平原|太湖平原]]最北端。

=== 气候 ===
典型的[[亚热带季风气候|亚热带季风气候]]。

== 交通 ==
丹阳全市已形成了铁路、公路、水路、航空、港口综合交通运输体系。
* 铁路：[[京沪铁路|京沪铁路]]横贯市区，设丹阳站（南广场），[[沪宁城际铁路|沪宁城际铁路]]设丹阳站北广场，南北广场将修隧道链接，已预留条件。[[京沪高速铁路|京沪高速铁路]]在丹阳境内设丹阳北站，[[连淮扬镇铁路|连淮扬镇铁路]]即将开工建设，由丹阳站接入沪宁城际铁路通往上海方向，另有规划中的沿江[[镇南铁路|镇南铁路]]，未来丹阳将成铁路枢纽，铁路成网状密布；
* 公路：[[沪宁高速公路|沪宁高速公路]]、312国道、122省道、338省道等是通过公路出入丹阳的主要干道，规划建设中的[[五峰山过江通道|五峰山过江通道]]以及[[泰州长江大桥|泰州长江大桥]]将使得位于丹阳东北的工业重镇成为交通枢纽；
* 水运：[[京杭大运河|京杭大运河]]纵贯丹阳，九曲河是向北通往长江的又一出口，丹金溧漕河曾是主要的漕运河道之一；对外开放的[[长江|长江]]第三大港──[[大港|大港]]港口离市区18公里。
* 航空：西距[[南京禄口国际机场|南京禄口国际机场]]80公里，东距[[上海虹桥机场|上海虹桥机场]]、[[浦东国际机场|浦东国际机场]]2小时车程，班机直航世界各地；

== 自然资源 ==
=== 水资源 ===
* 地表水：全市共有河道96条，计长464公里，其中以京杭运河、九曲河、丹溧漕河、鹤溪河等为主脉，沟通全市水系，形成全市水系网络，为水资源的蓄、引、提、调、排发挥巨大作用。另有大小水库9座，库容688万立方米；大小塘泊近万个，蓄水1500万立方米。
* 过境水：水资源丰富，北可引长江水，西和西南有句容、丹徒的客水。可利用过境水主要是长江水，沿江有主要引排涵闸10余座，引排流量500多立方米/秒，每年可引水5亿立方米左右。
* 地下水：一般在200米～300米深处，部分地下浅层在30米～100米处。地下水储藏量约4亿立方米，可供开采量约1.2亿立方米/年。目前因地表水丰富，地下水极少利用。

=== 矿产资源 ===
主要集中在北部低山丘陵区。金属矿种主要有铁、锰、方铅，但质低量少。非金属矿种主要有石英、陶土、泥炭、煤炭、砾石、斧劈、奇石、白云质灰岩和石灰岩，其中斧劈和奇石质量优良，蕴藏量达亿万吨。斧劈石产于中国较多地区，犹以江苏丹阳、武进的斧劈石在盆景界最为有名。

== 文物古迹 ==
丹阳是[[南朝齐|南朝齐]]高帝[[萧道成|萧道成]]、[[梁|梁]]武帝[[萧衍|萧衍]]两代开国皇帝的故里，历史文化底蕴丰厚，名胜古迹众多，古有云阳八景、延陵八景、经山崇教寺八景、练湖二十四景、七峰山房等。享有“江南文物之邦”盛誉。

今遗存的南朝齐梁帝王陵墓共有12座，三国吴帝王陵墓1座。其中南朝齐梁帝王陵墓的[[永安陵石刻|永安陵石刻]]、[[景安陵石刻|景安陵石刻]]、[[修安陵石刻|修安陵石刻]]、[[兴安陵石刻|兴安陵石刻]]、[[金王陈佚名陵石刻|金王陈佚名陵石刻]]、[[丹阳南朝陵墓石刻|郁林王陵石刻]]、[[丹阳南朝陵墓石刻|海陵王陵石刻]]、[[建陵石刻|建陵石刻]]、[[修陵石刻|修陵石刻]]、[[庄陵石刻|庄陵石刻]]、[[陵口石刻|陵口石刻]]11处、26件，均被列为[[全国重点文物保护单位|全国重点文物保护单位]]。

被列为[[江苏省文物保护单位|江苏省文物保护单位]]的有江南第一梵钟——[[唐中和铜钟|唐中和铜钟]]、[[唐朝延陵季子墓碑及碑亭|唐朝延陵季子墓碑及碑亭]]、[[总前委旧址|总前委旧址]]、[[华东局与华东军区旧址|华东局与华东军区旧址]]4处(件)。

被列为镇江市文物保护单位的有[[三国吴孙坚墓高陵|三国吴孙坚墓高陵]]、[[新四军江南指挥部与中共京(宁)沪线路北特委旧址|新四军江南指挥部与中共京(宁)沪线路北特委旧址]]、[[近月轩惨案旧址|近月轩惨案旧址]]、[[贺甲战斗旧址|贺甲战斗旧址]]、[[新丰战斗旧址|新丰战斗旧址]]5处。

被列为丹阳市文物保护单位的有[[陀罗尼经幢|陀罗尼经幢]]、[[石曼卿神道碑|石曼卿神道碑]]、[[开泰桥|开泰桥]]、[[通泰桥|通泰桥]]、宋朝[[玉乳泉井|玉乳泉井]]、明朝[[万善塔|万善塔]]等55处。

其他还有古文化遗址多处，并从中发掘大量历史文物，其中有[[良渚文化|良渚文化]]遗物——[[黑陶弧腹罐|黑陶弧腹罐]]、新石器时代晚期遗物——[[黑陶宽把盖杯|黑陶宽把盖杯]]、[[湖熟文化|湖熟文化]]遗物——[[鬲|鬲]]、[[马家浜文化|马家浜文化]]遗物——腰沿釜、西周青铜凤纹尊等。其文物古迹级别之高，数量之多，为全省乃至全国县邑所罕见。

== 语言 ==
丹阳处于[[吳語|吳語]]和[[官話|官話]]两大方言区的交界地，历来有“吴头楚尾”之称。就全市而言，本地人称“四门十八腔”，东乡人也往往听不懂西乡人的话，其方言之奇特在全中国亦不多见。

== 历代名人 ==
春秋时吴[[季札|季札]]，为吴王[[寿梦|寿梦]]少子。不受君位，封于延陵，号延陵季子，史称"季子"。后人称颂其高风亮节季子；尤备受孔子推崇。

丹阳是南朝[[齐|齐]]、[[梁|梁]]皇帝的故里。自三国以来，丹阳曾出过宰相27人，皇太后、皇后5人；唐至清代出状元2人，进士231人。宋朝抬棺进谏的大学士[[陈东_(宋朝)|陈东]]，力主抗金，被陷害后故人厚葬之于丹阳东北的桐村。

近现代，丹阳孕育了著名教育家、上海[[复旦大学|复旦大学]]创始人[[马相伯|马相伯]]，著名社会科学家、教育家[[戴伯韬|戴伯韬]]，著名书画家、美术教育家[[吕凤子|吕凤子]]，语言学泰斗[[吕叔湘|吕叔湘]]、教育家[[匡亚明|匡亚明]]。

[[台湾|台湾]]作家[[眭澔平|眭澔平]]的原籍也是丹阳的。

== 经济 ==
丹阳经济较为发达，经济综合实力居[[江苏省|江苏省]]十强县（市）第8位，经济基本竞争力一直位居全国[[百强县|百强县]]（市）前50以内。产业特色鲜明，形成了以眼镜、五金工具、汽车零部件、木业、医疗器械、电子元器件等为主业的一系列规模企业，其中有天工，丹化，鱼跃，大亚，沃德，恒宝、利华等7家境内外上市企业。

== 教育 ==
丹阳市民重视教育，因此成就了良好的教育氛围和较高的教育水平。

主要学校：[[丹阳师范学校|丹阳师范学校]]、[[江苏省丹阳高级中学|江苏省丹阳高级中学]]、[[吕叔湘中学|吕叔湘中学]]、[[丹阳市第五中学|丹阳市第五中学]]、[[丹阳市第六中学|丹阳市第六中学]]、[[丹阳市第八中学|丹阳市第八中学]]、[[丹阳市实验中学|丹阳市实验中学]]、[[丹阳市实验小学|丹阳市实验小学]]等。
