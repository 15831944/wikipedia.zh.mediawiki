'''丹顶鹤'''（[[学名|学名]]：''{{lang|la|Grus japonensis}}''），也叫仙鹤、日本鶴、白鹤（古代名稱，現代語言中[[白鶴|白鶴]]是另一種[[鶴屬|鶴屬]]鳥類）、鴜鷜，[[中国|中国]]古籍文献中对丹顶鹤有许多称谓，如《尔雅翼》中称其为仙禽，《[[本草纲目|本草纲目]]》中称其为胎禽。丹顶鹤是[[鹤|鹤类]]中的一种，因头顶有红肉冠而得名。它是[[东亚|东亚]]地区所特有的[[鸟|鸟]]种，因体态优雅、颜色分明，在这一地区的文化中具有吉祥、忠贞、长寿的寓意。

== 特征 ==
丹頂鶴身長約120公分至150公分，翅膀打開約200公分，丹顶鹤具备鹤类的特征，即三长——嘴长、颈长、腿长。成鸟除颈部和[[羽毛|飞羽]]后端为黑色外，全身洁白，头顶皮肤裸露，呈鲜红色。其次级飞羽和三级飞羽形长弯曲成弓状，两翼折叠时，覆于整个短尾上面，被人误认为是尾羽。平均壽命在50-60歲之間，在鳥類中屬少見長壽故古代也作為長壽象徵。<ref>[http://www.xinhuanet.com/science/2017-09/21/c_136625979.htm 丹顶鹤为何被称为“仙鹤”]</ref>

传说中的剧毒[[鹤顶红_(毒药)|鹤顶红]]（也有稱鹤顶血）正是此处，但纯属谣传，鹤血是没有毒的，古人所说的“鹤顶红”其实是[[砒霜|砒霜]]，即不纯的[[三氧化二砷|三氧化二砷]]，鹤顶红是古时候对砒霜隐晦的说法。<ref>[http://tlpsr.chinawuxi.gov.cn/wzzt/kxzz/2006nd6q/618450.shtml 解读中国古代九大毒药]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://www.epochtimes.com/b5/4/11/25/n728668.htm 丹頂鶴頭上紅色丹頂 傳說是劇毒?]</ref>丹顶鹤的[[尾脂腺|尾脂腺]]被[[羽毛#粉䎃|粉䎃]]。幼鸟体羽棕黄，[[喙|喙]]黄色。亚成体羽色黯淡，2岁后头顶裸区红色越发鲜艳。

== 分布 ==
丹顶鹤繁殖地在[[中国|中国]][[黑龙江|黑龙江]]的[[三江平原|三江平原]]和[[嫩江|嫩江]]中下游地区、[[俄罗斯|俄罗斯]]的远东和[[日本|日本]]等地。它在中国东南沿海各地及[[长江|长江]]下游、[[朝鲜|朝鲜]]海湾、日本等地[[越冬|越冬]]。

历史上丹顶鹤的分布区比现在要大得多，越冬地更为往南，可至中國[[福建|福建]]、[[海南島|海南島]]、[[台灣|台灣]]等地。由于这种鸟在文化中的特殊地位，[[地方志|地方志]]书中一直有着详细的记载，为研究它的古代分布提供了詳实的资料。

== 习性 ==
[[File:Grus_japonensis_-Hokkaido,_Japan_-several-8_(1).jpg|thumb]]
丹顶鹤每年要在繁殖地和越冬地之间[[鸟类迁徙|迁徙]]，只在日本[[北海道|北海道]]是当地的留鸟，不进行迁徙，这可能与冬季当地人有组织地投喂食物，食物来源充足有关。

丹顶鹤的栖息地是[[沼泽|沼泽]]和沼泽化的[[草甸|草甸]]，食物主要是浅水的鱼虾、软体动物和某些植物根茎，以季节不同而有所变化。

丹顶鹤成鸟每年[[换羽|换羽]]两次，[[春季|春季]]换成夏羽，[[秋季|秋季]]换成冬羽，属于完全换羽，会暂时失去飞行能力。

丹顶鹤的鸣声非常嘹亮，作为明确领地的信号，也是发情期交流的重要方式。

丹顶鹤属于[[单配制|单配制]]鸟，若无特殊情况可维持一生。每年的繁殖期从3月开始，持续6个月，到9月结束。它们在浅水处或有水湿地上营巢，巢材多是芦苇等[[禾本科|禾本科]]植物。丹顶鹤每年产一窝卵，产卵一般2~4枚。孵卵由雌雄鸟轮流进行，孵化期31~32天。雏鸟属[[早成雏|早成雏]]。

有人认为北海道的丹顶鹤种群在鸣叫和迁徙上与大陆种群有差异，可划分为两个[[亚种|亚种]]，但基因分析表明二者没有显著形差异，因此这种说法至今没有得到认同。

== 文化 ==
[[File:宋徽宗_-_瑞鶴圖.jpg|thumb]]》]]
[[File:100_views_edo_102.jpg|230px]]「[[名所江戶百景|名所江戶百景]]」描繪的丹頂鶴。]]
东亚地区的居民，用丹顶鹤象征幸福、吉祥、长寿和忠贞。<ref>《琅环记》载：“鹤一名仙子，一名沈尚书，一名蓬莱羽士。鹤，仙禽也，于物为多寿，感于阳，故鸣于子，雄则声闻数里，雌则声下二不扬，江陵鹤泽中多鹤，因以名郡。”</ref>在各国的文学和美术作品中屡有出现，[[殷商|殷商]]时代的墓葬中，就有鹤的形象出现在雕塑中。[[春秋战国|春秋战国]]时期的青铜器钟，鹤体造型的礼器就已出现。汉人[[刘安|刘安]]在《[[淮南子|淮南子]]·说林训》中，出现“鹤寿千岁”的说法。[[陆玑|陆玑]]在《毛诗草木鸟兽虫鱼疏》描述：“大如鹅，长脚，青翼，高三尺余，赤顶，赤目，喙长四寸余，多纯白”。[[杜牧|杜牧]]有诗：“清音迎晓月，愁思立寒蒲。丹顶西施颊，霜毛四皓须。”[[王象晋|王象晋]]的《群芳谱》中评价丹顶鹤：“体尚洁，故色白；声闻天，故头赤。”[[道教|道教]]中丹顶鹤飘逸的形象已成为长寿、成仙的象征。甚至誤认为鹤是[[胎生|胎生]]。<ref>《墨客挥犀录》记载：“刘渊材迂阔好怪，尝畜两鹤。客至，夸曰：‘此仙禽也。凡禽卵生，此禽胎生。’语未卒，园丁报曰：‘鹤夜半生一卵。’渊材呵曰：‘敢谤鹤耶！’未几，鹤延颈伏地，复诞一卵。渊材叹曰：‘鹤亦败道，吾乃为刘禹锡话所误。’”</ref>

在日本，8世紀的皇族[[長屋王|長屋王]]的邸宅跡地，出土繪有丹頂鶴的土器，此為繪有丹頂鶴的最古老的作品。1964年，指定丹頂鶴為北海道的道鳥。1984年至2004年發行的[[一千日圓紙幣|一千日圓紙幣]]，背面採用丹頂鶴的圖案。[[日本航空|日本航空]]的「鶴丸」標誌，也是以丹頂鶴為設計的原型。明代朝服用丹顶鶴[[補子|補子]]为文官最高象征。

== 保护状况 ==
丹顶鹤是大型[[涉禽|涉禽]]，在[[湿地|湿地]]环境中属于食物链的上层，是湿地生物多样性的关键种。日本北海道的[[阿伊努人|阿伊努人]]把生活在钏路湿地的丹顶鹤称为“湿地之神”。目前它们面临的威胁主要有：
* 栖息地的破坏。在中国东北和远东地区人类活动对湿地的破坏在1960年代以后急剧加重，对湿地的围垦不仅侵占了原有的栖息地，还使原本连通的水系阻断，再加上近些年远东地区气候干旱化趋势明显，水域面积缩小严重。人类活动引入的污染也威胁着丹顶鹤的生存，此外如烧荒等开垦方法，对丹顶鹤的巢材和掩蔽处毁坏严重，致使其分布更为狭窄。
* 偷猎：由于自古东亚地区对丹顶鹤就有着对其羽毛和器官的需求，猎杀就难以避免。虽然近些年随着保护法规的建立，直接的猎杀很少发生，但是投毒用来猎捕其他水禽的方法，已成为丹顶鹤的重要死因。

目前，丹顶鹤是中國[[中国国家一级重点保护野生动物名录|国家一级保护动物]]，在[[世界自然保護聯盟|世界自然保護聯盟]]（IUCN）的红皮书中是濒危物种，在[[瀕危野生動植物種國際貿易公約|瀕危野生動植物種國際貿易公約]]（CITES）中列入附录一。

日本在1935年指定丹頂鶴及其繁殖地為[[天然紀念物|天然紀念物]]，1952年指定「釧路的丹頂鶴」為特別天然紀念物，1967年指定全國各地域的丹頂鶴為特別天然紀念物，1993年指定丹頂鶴為稀少野生动植物种。

== 历史 ==
===研究历史===
丹顶鹤由于体形大、颜色分明，很容易辨认。人们对丹顶鹤的知识很早就有了一定的积累。中国的地方志书对其有连续的记录，丹顶鹤很早就被人们所饲养，[[唐|唐]][[宋|宋]]年间尤为盛行。现在许多地方都有饲养的丹顶鹤供观赏之用。

1980年代后，对丹顶鹤的专项研究展开，至今已对其分布、繁殖地和越冬地的生态和行为、迁徙等掌握的一定的信息。目前丹顶鹤的人工繁殖和人工授精技术已经成熟。1990年代以后，根据环境研究和人造卫星跟踪技术，丹顶鹤的迁徙路线已经明确。

===學名緣由===
有趣的是，丹顶鹤雖主要分佈在中國大陸，但卻擁有一個與中國完全扯不上邊的拉丁文學名 ─ ''Grus japonensis''（意即日本鶴）

於清末時期，清廷奉行鎖國政策，學者無從進入中國研究丹頂鶴這種當時尚未“發現”的雀鳥。正好當時日本有向這些學者提供丹頂鶴的樣本，因此當學者命名這種新雀鳥時就依規則以樣本的來源地（即日本）來進行命名。<ref>NOW新聞台新聞報道（2008年9月5日）</ref>

===北方的稀客===

鳥友驚喜的在新竹市[[香山濕地|香山濕地]](2004年),記錄丹頂鶴丹丹在新竹沿海的生活史，中央、地方政府及民間團體，為了一隻鳥全面總動員，在台灣保育史上前所未見，積極的作法也贏得國際肯定。

===贈送歷史===
*[[臺灣|臺灣]]：日本首度將丹頂鶴贈送台灣。2011年，Big、Kika（貴華），來自北海道[[钏路市|钏路市]]动物园，現於[[台北市立動物園|台北市立動物園]]<ref>NOWnews今日新聞 (2011年4月22日) [http://www.nownews.com/n/2011/04/22/475815 北海道釧路動物園送2隻丹頂鶴　9月台灣迎嬌客]</ref>。
