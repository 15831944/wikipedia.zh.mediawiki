'''丹尼斯·強森'''（{{lang-en|'''Dennis Johnson'''}}，{{bd|1954年|9月18日|2007年|2月22日|catIdx=L}}），美國職業籃球運動員和教練。

== 球員生涯 ==
丹尼斯·強森在大學時期就有好表現，他有著平均15.7分5.8個籃板和3.3次助攻，以及強硬的防守。他只讀了1年的大學，就去參加了1976年的選秀，並在第2輪被[[西雅圖超音速|西雅圖超音速]]選上。他是個大器晚成的球員，他克服了先天的困境，作為一名後衛，他首個賽季就有9.2分和1.5助攻，之後表現亮眼，於1979年隨隊奪冠，並當選MVP。不過隔年被交易到[[鳳凰城太陽|鳳凰城太陽]]，在太陽隊的3年，強森靠著它強硬的防守為球隊建功。1983年，他加入了[[波士頓塞爾提克|波士頓塞爾提克]]，有著[[賴瑞·柏德|賴瑞·柏德]]、[[罗伯特·帕里什|罗伯特·帕里什]]、[[凱文·麥克海爾|凱文·麥克海爾]]等球星的塞爾提克，再加上強森，讓它在80年代再奪2座冠軍。1990年，他宣布退休。

== 教練生涯 ==
他曾執教過CBA的[[La_Crosse_Bobcats|La Crosse Bobcats]]，並在2003年短暫帶過[[洛杉磯快艇|洛杉磯快艇]]，之後的3年內他也當過2支[[美國國家籃球發展聯盟|NBDL]]的教練，並有1次最佳教練的殊榮。

== 死亡 ==
2007年2月22日，他因為心臟病而死亡。
