'''丹尼爾·干沙利斯·古爾沙'''（{{lang-es|'''Daniel González Güiza'''}}，{{bd|1980年|8月17日|catidx=Guiza, Dani}}）是一名[[西班牙|西班牙]]的職業足球員，擔任[[前鋒_(足球)|前鋒]]，現在效力[[土耳其足球超級聯賽|土超]]球會[[費倫巴治|費倫巴治]]，效力[[西甲|西甲]]球隊[[皇家馬略卡|皇家馬略卡]]時，在[[2007年至2008年西班牙足球甲級聯賽|2007－2008年]]的賽季中更力壓各名牌射手奪得[[西甲|西甲]]神射手的寶座，亦得到了歐洲銀靴獎。

== 球會生涯 ==
古爾沙青年時代在[[赫雷斯足球俱樂部|薩雷斯]]開始其職業足球生涯，其後古爾沙被[[西班牙足球甲級聯賽|西甲]]球隊[[皇家馬略卡|皇家馬略卡]]發掘。不過，古爾沙首個馬略卡球季被外借到[[西班牙足球丙級聯賽|西丙]]的[[杜斯靴曼拿斯足球俱樂部|杜斯靴曼拿斯]]（[Dos Hermanas），平均每兩場就入球，得以返回皇家馬略卡B隊。<ref name="CUE">[http://www.fifa.com/worldfootball/clubfootball/news/newsid=775689.html 鑽石丹尼·古爾沙足球生涯的線索]</ref>

在皇家馬略卡的三個球季，古爾沙未能在一隊站住腳，一直都在一隊和後備隊中浮浮沈沈。他在皇家馬略卡A隊中只上陣七場射入一球，在皇家馬略卡B隊卻上陣70場射入28球。<ref name="CUE"/>

最終，古爾沙再次被外借，[[維爾瓦足球會|維爾瓦]]和[[巴塞隆納足球會|巴塞隆納]]的B隊先後成為他的東家。<ref name="CUE"/> 古爾沙在這兩支球隊中表現未見特出，為了尋找轉機，他轉會至[[西班牙乙組足球聯賽|西乙]]的[[穆爾西亞城足球會|穆爾西亞城]]（Ciudad de Murcia），它是[[皇家穆爾西亞足球俱樂部|皇家穆爾西亞]]的衛星隊。古爾沙正正是在穆爾西亞城找到在他的職業生涯的一個轉折點，兩個賽季就進了36個進球。<ref name="CUE"/>

古爾沙及後一季藉著西乙的優秀表現重返西甲，加盟已升班一年但未站穩住腳的[[赫塔費足球俱樂部|赫塔費]]。古爾沙在赫塔費表現不錯，[[2006年至2007年西班牙足球甲級聯賽|2006－07年賽季]]進了11球。

2007年7月，古爾沙回到了[[皇家馬略卡|皇家馬略卡]]，並成為[[2007年至2008年西班牙足球甲級聯賽|2007－2008年]]的賽季的西甲神射手，總共攻入了27球，而且沒有一球是[[點球|點球]]。這令到各大球會都想羅致古爾沙如[[羅馬體育會|羅馬]]、[[巴塞羅那足球俱樂部|巴塞羅那]]、[[皇家馬德里|皇家馬德里]]、[[費倫巴治|費倫巴治]]<ref>[http://www.ultimahora.es/segunda.dba?-1+9+437216 Fenerbahçe goes for Güiza] {{webarchive|url=https://web.archive.org/web/20080714074327/http://www.ultimahora.es/segunda.dba?-1+9+437216 |date=2008-07-14 }} {{es icon}}</ref>、[[華倫西亞足球會|華倫西亞]]及[[阿仙奴|阿仙奴]]等<ref>[http://goal.com/en/Articolo.aspx?ContenutoId=718344 Arsenal set sights on Güiza]</ref>。

經過掙扎後，古爾沙於2008年7月轉投[[土耳其足球超級聯賽|土超]]班霸[[費倫巴治|費倫巴治]]。首個在費倫巴治的賽季，古爾沙射入十一球，但費倫巴治只能得到聯賽第四名，無緣[[歐洲冠軍聯賽|歐聯]]。

2012年9月，古爾沙外借加盟[[馬來西亞足球超級聯賽|馬來西亞足球超級聯賽]]勁旅[[柔佛DT足球俱樂部|柔佛]]， 為期半年。他為[[柔佛DT足球俱樂部|柔佛]]上陣11場進7球。

2013年9月，古爾沙轉投[[巴拉圭|巴拉圭]]球隊[[波特諾山丘|波特諾山丘]]，為該隊上陣46場進15球。

2015年3月，古爾沙被[[巴拉圭|巴拉圭]]球隊[[波特諾山丘|波特諾山丘]]解約，成為[[自由球員|自由球員]]，8月和[[西班牙足球丙級聯賽|西班牙足球丙級聯賽]][[加的斯足球俱樂部|卡的斯]]簽約。

== 國家隊生涯 ==
2007年11月8日，古爾沙第一次被徵召入[[西班牙國家足球隊|西班牙隊]]。這次徵召是為了應付[[2008年歐洲國家盃|2008年歐洲國家盃]]外圍賽對陣[[北愛爾蘭足球代表隊|北愛爾蘭隊]]的比賽。直到[[2008年歐洲國家盃|2008年歐洲國家盃]]前，古爾沙代表西班牙上陣四場，沒有任何入球。
[[File:Spain_Euro_08_celebration_1.jpg|thumb]]、古爾沙和[[法布雷加斯|法布雷加斯]]慶祝奪得2008年歐洲國家盃]]
後來，古爾沙入選了[[2008年歐洲國家盃|2008年歐洲國家盃]]西班牙隊參賽名單中。古爾沙在6月18日分組賽對陣[[希臘國家足球隊|希臘隊]]的比賽中射入第一次國際賽入球，為西班牙隊鎖定勝局<ref>[http://news.bbc.co.uk/sport2/hi/football/euro_2008/7363473.stm Russia 2-0 Sweden & Greece 1-2 Spain]，[[BBC|BBC]]，2008年6月18日</ref>。6月26日舉行的準決賽，西班牙隊對戰[[俄羅斯國家足球隊|俄羅斯隊]]，古爾沙於70分鐘進球，拉開比數使俄羅斯隊失去扳平的機會<ref>[http://news.bbc.co.uk/sport2/hi/football/euro_2008/7363541.stm Russia 0-3 Spain]，[[BBC|BBC]]，2008年6月26日</ref>。古爾沙協助西班牙隊羸得[[2008年歐洲國家盃|2008年歐洲國家盃]]冠軍。

自2008年歐國盃後，雖然西班牙隊有著[[大卫·比利亚|大卫·比利亚]]及[[费尔南多·托雷斯|费尔南多·托雷斯]]等前線好手，但古爾沙仍能成為西班牙隊的常客。作客[[亞塞拜然國家足球隊|亞塞拜然隊]]的友誼賽中，古爾沙取得一個入球。

一年後的[[2009年洲際國家盃|2009年洲際國家盃]]，古爾沙同樣被挑選入伍，但古爾沙在分組賽并沒有大多上陣機會，只有大約30分鐘的上陣時間。<ref>[http://www.fifa.com/confederationscup/statistics/players/player=297328/mainstatistics.html# 古爾沙，數據]，2009年洲際國家盃，[[FIFA|FIFA]]</ref> 在2009年6月28日的季軍戰中，古爾沙在西班牙隊落後[[南非國家足球隊|南非隊]]一球時被派上陣。古爾沙神奇地在一分鐘內連入兩球，令西班牙隊反超前南非隊。最後西班牙隊在[[加時賽|加時賽]]以3比2勝出，成為2009年洲際國家盃季軍。

== 技術風格 ==
因為古爾沙帶球[[速度|速度]]十分高，所以他精於快速反擊。由於他成長時效力的球會實力都不強，使身為[[前鋒_(足球)|前鋒]]的他處境困難，因而善於動腦走位，爭取入球<ref>[http://www.fifa.com/confederationscup/players/player=297328/profile.html 古爾沙，描述] {{webarchive|url=https://web.archive.org/web/20100513003715/http://www.fifa.com/confederationscup/players/player%3D297328/profile.html |date=2010-05-13 }}，2009年洲際國家盃，[[FIFA|FIFA]]</ref>。另外，古爾沙的球風非常獨特，并不像[[西班牙|西班牙]]其他前鋒，當球賽呈膠著狀態時，古爾沙就能大派用場，為球隊增加進攻板斧。此特性令他即使效力水平比[[歐洲五大聯賽|四大聯賽]]差的土耳其聯賽，仍能經常入選西班牙隊。<ref name="cctv">[[中央電視台|中央電視台]]2009年洲際國家盃季軍戰於2009年6月28日的直播。</ref>

在2009年洲際國家盃季軍戰中，正正展現了上述三點：球賽呈膠著狀態時入替[[费尔南多·托雷斯|费尔南多·托雷斯]]，增加進攻變化，并一分鐘內連入兩球，一球利用美妙的走位，另一球是快速反擊。<ref name="cctv"/>
