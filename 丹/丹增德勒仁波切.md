'''丹增德勒仁波切'''（{{bo|z=བསྟན་འཛིན་བདེ་ལེགས་|w=Bstan 'dzin bde legs}}，{{bd|1950年|9月22日<ref name="luan"/>|2015年|7月12日|catIdx=D}}），俗名'''阿安扎西'''，法名'''丹增德勒圖登曲吉尼瑪'''，簡稱'''丹增德勒'''<ref name="仁">{{cite book |author1=[[唯色|唯色]] |title=《[[仁波切之殤|仁波切之殤]]》 |date=2015/11/05 |publisher=雪域出版社 |location=台灣 |isbn=978-986-89868-7-9}}</ref>，1950年出生于[[西康省|西康省]][[理塘|理化县]]，[[四川省|四川省]][[甘孜藏族自治州|甘孜藏族自治州]][[雅江县|雅江县]][[红龙乡|红龙乡]][[乌托寺|乌托寺]]僧侣<ref name="luan">{{cite web |author1=栾春晖 |title=煽动分裂国家和实施爆炸 阿安扎西和洛让邓珠被依法严惩 |url=http://news.southcn.com/china/zgkx/200301290146.htm |website=南方网 |date=2003-01-29}}</ref>，[[藏傳佛教|藏傳佛教]]上師[[仁波切|仁波切]]。

2002年4月7日，阿安扎西因为被指控涉入包括2002年4月3日发生在四川省[[成都市|成都市]]中心[[天府廣場|天府廣場]]的[[成都天府广场爆炸案|爆炸案]]在内的多起爆炸案，在雅江縣被捕。2002年12月2日，甘孜州中级人民法院以[[爆炸罪|爆炸罪]]、[[煽动分裂国家罪|煽动分裂国家罪]]判处阿安扎西死刑，[[死缓|缓期]]2年执行，[[剥夺政治权利|剥夺政治权利]]终身<ref>{{cite web |title=分裂分子制造成都天府广场爆炸案一审被判死刑 |url=http://news.sohu.com/80/24/news204732480.shtml |website=搜狐-华西都市报}}</ref>，此后一直在狱中服刑<ref>{{cite web |title=服刑罪犯病亡 曾制造成都爆炸案而入狱 |url=http://www.xichu.net/news/folder1988/2015/07/2015-07-17372253.html |website=西楚网 |date=2015-07-17}}</ref>。2005年1月26日，四川省高级人民法院裁定将阿安扎西刑罚减为[[无期徒刑|无期徒刑]]，剥夺政治权利终身<ref>{{cite web |title=制造成都天府广场爆炸的分裂分子被减刑为无期 |url=http://www.china.com.cn/chinese/law/769243.htm |website=中国网-新华网}}</ref>。2015年7月12日，他在成都川東監獄中去世，獄方稱阿安扎西有受到人道對待。{{r|易}}

==生平==
=== 早年生涯 ===
1950年出生于[[西康省|西康省]][[理塘|理化县]]牧場的貧窮牧戶。幼年3歲即父親去世，後來就由外婆撫養長大。1956年於[[理塘寺|理塘寺]]出家為僧，不久後因改革浪潮而被遣送回家，與外婆相依為命，生活相當艱苦。1961年至1963年中國各地發生[[三年大饑荒|大饑荒]]，1979年在[[第十四世达赖喇嘛|尊者達賴嘛喇]]派遣第一批5人代表團行經理塘時，因向代表團投訴當地飢荒真相，而被當局盯上。
1982年9月徒步流亡至印度，10月10日在達蘭薩拉尊者達賴喇嘛宮殿由尊者授予[[比丘戒|比丘戒]]。之後，在印度南部的[[哲蚌寺|哲蚌寺]][[果莽札倉|果莽札倉]]刻苦學習，成績優異。1983年，獲[[第十四世达赖喇嘛|尊者達賴嘛喇]]認證為[[康區|康區]]南部吾托地方帳篷寺院的[[格西阿登彭措|格西阿登彭措]]的轉世仁波切。
1987年丹增德勒仁波切從印度返回入理塘寺，因反對供奉「[[凶天|凶天]]」與主要喇嘛產生矛盾，更因為在印度求學背景被其他活佛忌憚排斥。
1988年起，在以前待過的帳篷寺院所在地建寺院，後用三年時間，在當地民眾支持下，得[[十世班禅|十世班禪喇嘛]]同意，在今[[甘孜州|甘孜州]]雅江縣紅龍鄉（又稱塔子壩）建立主要寺院，全名「康那爛陀特青襄丘林貢巴」（音譯），又稱[[崇新寺|崇新寺]]。
自1994年起籌辦學校，在當局暫時默許下於1998年5月建立格西溝和平小學，收容[[理塘縣|理塘縣]]、[[雅江縣|雅江縣]]、[[康定縣|康定縣]]、[[瀘定縣|瀘定縣]]等地貧苦兒童與孤兒，所有經費包含教職員工資與學生食衣開支平均每月13000元人民幣皆由他負擔，後來學校於2000年被當局強制命令解散。
在弘法期間深受百姓擁戴與信仰，與創辦[[喇榮五明佛學院|喇榮五明佛學院]]的[[晋美彭措|晉美彭措]]法王並稱「康北的晉美彭措，康南的阿安扎西」。但因為向當局直言投訴惹惱當地官員，1997年[[甘孜州|甘孜州]]宗教局下令不予承認活佛資格。當局於1997年和2000年展開調查，使得丹增德勒仁波切不得不外逃避難，康區南部農牧民聯名按手印上書達數萬份，選派代表懇求政府寬恕，當局終於首肯保證不予處置，終於讓丹增德勒重返寺院，當地數萬民眾夾道歡迎。{{r|仁}}<ref name="西">{{cite web |title=時政評論：丹增德勒仁波切生平業績、冤判入獄、突然身亡詳情 |url=https://www.vot.org/cn/%E6%97%B6%E6%94%BF%E8%AF%84%E8%AE%BA%EF%BC%9A%E4%B8%B9%E5%A2%9E%E5%BE%B7%E5%8B%92%E4%BB%81%E6%B3%A2%E5%88%87%E7%94%9F%E5%B9%B3%E4%B8%9A%E7%BB%A9%E3%80%81%E5%86%A4%E5%88%A4%E5%85%A5%E7%8B%B1%E3%80%81/ |website=2015年7月19日 |publisher=西藏之聲 |accessdate=2018-09-01}}</ref>

=== 被捕及逝世 ===
2000年4月3日於[[成都|成都]][[天府廣場|天府廣場]]發生一起爆炸案，警方逮捕一位雅江縣農民洛讓鄧珠（Lobsang Dhondup，也稱洛桑頓珠），隨後宣稱審問過程洛讓鄧珠指認受到丹增德勒仁波切指使。
於是丹增德勒仁波切在2002年4月7日和其他至少三名僧侶一同於寺院被捕。{{誰|同時宣稱}}之前在甘孜州的四起爆炸案（一度說是七起爆炸案）皆為2人所為。
中國當局指控他涉入爆炸案，甘孜藏族自治州中級人民法院於2002年11月29日開庭審判後，在12月2日以「製造系列爆炸、煽動分裂國家」罪名判處[[死刑|死刑]]，緩期執行兩年。洛讓鄧珠也在2002年12月因同案被判處死刑，並於2003年1月被處決。國際人權團體及聯合國人權專家對本案審判不公提出抗議，指他們被剥奪獲得公正審判的權利，且在拘押時遭到酷刑虐待。<ref>[https://www.amnesty.org/en/documents/asa17/029/2003/en/ Miscarriage of Justice? The trial of Tenzin Deleg Rinpoche and related arrests] [[國際特赦組織|國際特赦組織]], 2009-09-30.</ref><ref>[https://www.hrw.org/report/2004/02/08/trials-tibetan-monk/case-tenzin-delek Trials of a Tibetan Monk: The Case of Tenzin Delek] [[人權觀察|人權觀察]]，2004-02-08。</ref>後來丹增德勒發表聲明：「因為我是西藏人，我將永遠真誠的為西藏人民的利益及福祉奉獻我自己。這是中國人不喜歡我，並陷害我的真正原因。」<ref name="ITSN-PoliticalPrisoners2">{{cite web|last = Staff|first = |authorlink = |coauthors = |title = Political Prisoners|publisher = ''[[International_Tibet_Support_Network|International Tibet Support Network]]''|url = http://www.tibetnetwork.org/politicalprisoners|accessdate = 2010-02-03|deadurl = yes|archiveurl = https://web.archive.org/web/20091213143300/http://www.tibetnetwork.org/politicalprisoners|archivedate = 2009-12-13}}</ref><ref name="ITSN-PoliticalPrisoners-Transcript">Transcript of recording of Tenzin Delek Rinpoche obtained from the detention center in Dartsedo, the capital of Kardze Tibet Autonomous Prefecture, Sichuan province, on January 20, 2003. </ref>

他在2005年1月26日緩刑期滿後被減刑為無期徒刑。許多海外丹增德勒的聲援者持續要求重新審判。<ref name="郵">{{cite web |author1=國際西藏郵報 |title=籲英國首相卡梅倫向習近平提高西藏與人權問題 |url=https://www.tibet.org.tw/news_detail.php?news_id=7963 |publisher=達賴喇嘛西藏宗教基金會 |accessdate=2018-09-01}}</ref>
2009年11月，丹增德勒仁波切的故鄉有四萬多名藏人聯署簽名要求本案重審，並進行數日[[絕食|絕食]][[抗議|抗議]]，70多位藏人因此遭到逮捕。<ref name="ITSN-PoliticalPrisoners"/>
在服刑過程中獄方稱阿安扎西的身體狀況並不好，且患有多項疾病且為其定期體檢與予以治療。<ref name="易">{{cite web |author1=中國新聞網 |title=煽動分裂國家罪入獄 阿安扎西受到人道主義待遇 |url=http://news.163.com/41230/5/18S2UMDO0001124S.html |publisher=網易 |accessdate=2018-09-04}}</ref>
2015年7月12日，中國成都川東監獄當局通知家屬，丹增德勒已在獄中過世。在他去世之前，藏人及國際人權團體曾一再要求讓他保外就醫。在他逝世後，家人、支持者及人權團體要求交還遺體，以檢驗死因並為他舉行宗教葬儀。<ref>[https://www.hrw.org/news/2015/07/15/china-release-tibetan-monks-body China: Release Tibetan Monk’s Body] [[人權觀察|人權觀察]]，2004-02-08。</ref>但中國當局卻未經驗屍即火化遺體。<ref>{{cite news|title = High-profile Tibetan monk dies in prison in China|url = http://cn.nytimes.com/china/20150717/c17tibet/zh-hant/|agency = 紐約時報中文網|date = 2015-07-17|accessdate = 2015-08-18}}</ref><ref>{{cite news|last1 = Watt|first1 = Louise|title = Tibetan lama Tenzin Delek Rinpoche dies in Chinese prison|url = http://news.yahoo.com/tibetan-lama-tenzin-delek-rinpoche-dies-chinese-prison-071515369.html|accessdate = 2015-07-14|agency = AP|date = 2015-07-14 }}</ref>

在台藏人團體在台藏人福利協會、西藏青年會台灣分會及台灣社運團體包括[[台灣人權促進會|台灣人權促進會]]、台灣圖博之友會、[[華人民主書院|華人民主書院]]、華人民主文化協會、[[民間司法改革基金會|民間司法改革基金會]]、廢除死刑推動聯盟、台灣漢藏友好協會、人權公約施行監督聯盟等團體於7月18日在[[台北市|台北市]][[中正紀念堂|自由廣場]]舉辦悼念丹增德勒仁波切晚會。<ref>{{cite web |title=【新聞稿】丹增德勒仁波切 圓寂頭七燭光晚會 |url=https://www.tahr.org.tw/news/1627 |publisher=[[台灣人權促進會|台灣人權促進會]] |accessdate=2018-09-01}}</ref>

== 评价 ==
国际西藏网（International Tibet Network）称，丹增德勒對於藏東牧民在社會、醫療、教育及宗教建設上有很大的貢獻。同時，也是抗議濫伐及採礦計畫的環保人士，更是藏人與漢人間的協調者。<ref name="ITSN-PoliticalPrisoners">[http://freetibetanheroes.org/portfolio-items/tenzin-delek-rinpoche/ Free Tibetan Heroes: Tenzin Delek Rinpoche] [[International_Tibet_Network|International Tibet Network]].</ref>{{r|西}}

他是西藏流亡精神领袖[[达赖喇嘛|达赖喇嘛]]的支持者，被国际社会认为是中国最重要的[[政治犯|政治犯]]之一。<ref>{{cite web|url=http://www.bbc.com/zhongwen/simp/china/2015/07/150719_tibet_monk_tenzin | language= zh-cn | title=中国官媒：丹增德勒因心脏病而去世 |publisher=BBC中文网 |date=2015-07-20 |accessdate= 2018-09-01}}</ref>
