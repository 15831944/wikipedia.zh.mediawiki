'''日本軍用手票'''（{{lang-ja|'''軍用手票'''}}，在日本簡稱為'''軍票'''）是[[第二次世界大戰|第二次世界大戰]]中流通的一種[[貨幣|貨幣]]。

== 類別 ==
[[日本貨幣商協同組合|日本貨幣商協同組合]]發行的《[[日本貨幣分類2007|日本貨幣分類2007]]》把軍票分為：
* 日清戰爭（[[中日甲午戰爭|中日甲午戰爭]]）軍票：單位為[[兩|兩]]。
* 日俄戰爭軍票：單位為[[銀圓|銀圓]]。
* 青島出兵軍票：單位為銀圓。
* [[西伯利亞干涉|西伯利亞出兵]]軍票：單位為[[金圓|金圓]]。
* 支那事變（[[抗日戰爭|抗日戰爭]]）軍票：單位為銀圓。
* 大東亞戰爭（[[太平洋戰爭|太平洋戰爭]]）軍票：單位為各被佔領國貨幣。

== 香港的軍票 ==
以第二次世界大戰的[[香港|香港]]為例，1941年聖誕日[[香港淪陷|香港淪陷]]後，日軍即以軍票作為貨幣，規定[[港幣|港幣]]和軍票的兌換率為1兌2。其後於1942年7月，日軍又將軍票面值貶至1兌4。日軍更於1943年6月起硬性規定軍票為香港唯一合法的交易媒介，港幣和其他貨幣一概禁止流通，市民私藏港幣屬違法行為。

日軍對[[香港經濟|香港經濟]]的影响程度十分嚴重，到1945年8月日本投降為止，香港共有19多億圓的軍票在香港流通。

1945年8月15日，日本投降，全部軍票頓變廢紙，同時港幣恢復成為香港[[法定貨幣|法定貨幣]]。

== 賠償問題 ==
戰後军票使占领国很多人一夜之間失去全部財產，而[[日本政府|日本政府]]多年來亦未有任何實際行動向曾被佔領國的人民換回軍票。一些手持軍票的市民向日本政府要求賠償，但多年來都不成功。1993年8月13日，香港軍票索賠委員會向[[東京地方法院|東京地方法院]]提起訴訟，要求日本政府償還欠下香港百姓的軍票債務。1999年6月17日，東京地方法院承認了[[香港人|香港人]]在軍票問題上的痛苦遭遇，但因日本沒有相關賠償法而未有作出賠償判決，日本政府亦以[[舊金山和約|舊金山和約]]自辯。至今被佔領國人民仍無法取回所失财物。

香港現今有數個民間團體向日本政府索償，當中最為人熟悉的是[[香港索償協會|香港索償協會]]。另外，日本亦有專家認為日本應換兌軍票，按現時香港人擁有之軍票貨幣之值額計算，總賠償金額達最少5000億港元（約8兆[[日元|日元]]）。
