'''炳草岗大桥'''有时亦简称为'''炳大桥'''或'''炳桥'''，是[[中华人民共和国|中华人民共和国]][[四川省|四川省]][[攀枝花市|攀枝花市]][[东区_(攀枝花市)|东区]]的一座公路桥，跨越[[金沙江|金沙江]]，连接北岸弄弄坪片区[[长寿路街道_(攀枝花市)|长寿路街道]]和南岸[[炳草岗街道|炳草岗片区]]。由於炳草岗大桥北岸邻近[[枣子坪街道|枣子坪街道]]，在规划设计阶段被称為'''炳枣大桥'''，至今部分市民仍然使用此名称。

== 历史 ==
攀枝花市是地处[[横断山脉|横断山脉]]的金沙江沿岸城市，被山脉和江河隔离成多个片区聚落，交通非常依赖桥梁和隧道。至20世纪末全市共建成11座跨金沙江的桥梁，是金沙江上桥梁最密集的地区。然而这其中只有7座公路桥，而且大部分都出现了不同程度的病害。炳草岗大桥建成以前，弄弄坪片区与市中心[[炳草岗街道|炳草岗片区]]之间没有桥梁直接相连，往来此两地必须绕行[[渡口大桥|渡口大桥]]。而渡口大桥及周边地带作为攀枝花市三个[[市辖区|市辖区]]之间唯一的交通要冲，道路已经拥挤不堪，城市路网结构亟待改善。炳草岗大桥于1999年1月开工、2001年12月31日建成通车<ref>{{cite web|url=http://www.chinahighway.com/news/2011/585512.php|title=瞧不尽攀枝花的桥|publisher=中国公路网|author=张世渝|date=2011-08-25|}}</ref>，是攀枝花市在21世纪建成的第一座跨江大桥。桥梁建设过程中，斜拉桥主塔下部曾因建筑质量不合格而炸毁返工。

== 详述 ==
[[File:Bingcaogang_Bridge_-_view_from_Panzhihua_Park.jpg|thumb]]
[[File:Panzhihua,_a_famous_bridge.jpg|thumb]]
炳草岗大桥全长516.3米，桥面全宽23.9米，双向四车道。桥梁主体部分由一座单塔双索面斜拉桥和一座T形刚构桥串联而成，由北向南跨径布置为149米+200米+51米（不含桥南立交匝道），中间的200米主跨由斜拉桥和刚构桥共同构成。设计荷载为汽车-超20级，挂车-120级<ref>{{cite web|url=http://www.lvyoutu.com/maps/2009820/m492314618.html|title=跨越金沙江的桥梁|publisher=lvyoutu.com|date=|5=}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。桥梁工程总投资约2亿元，由四川省交通厅公路规划勘察设计院设计，中铁二局五处和攀枝花桥梁工程公司承建。斜拉桥主塔位于北岸，塔基部位只在雨季涨水时才被淹没。斜拉桥共有23对拉索，双索面共计92根。主塔高180.8米，桥面距离江面约有百米，下方不仅跨越了金沙江，还跨越了多条道路，包括南岸的金沙江大道中段(原滨江大道)和人民街，以及北岸的钢城大道中段(原江北二路)、木棉路末段(原健康路)和[[成昆铁路|成昆铁路]]渡口支线渡口站。

炳草岗大桥承载的弄弄坪东路为城市[[主干道|主干道]]，在大桥南北两岸通过[[立交桥|立交桥]]与其它道路衔接。南岸立交桥名为“炳草岗大桥南立交”，由大桥直接引出，接入攀枝花大道(原江南三路)，并可经由攀枝花大道主线高架桥连接至人民街，远期还将增建直连人民街的[[匝道|匝道]]。北岸立交桥连通木棉路(原冶金东街)，立交桥距离炳草岗大桥较远，通过400米长的一段弄弄坪东路主线与大桥衔接，目前此立交桥没有正式名称。

炳草岗大桥所在处的金沙江实际为由南向北流，桥梁呈东西走向。但因金沙江在攀枝花市城区内总体为由西向东流，所以人们习惯称两岸为江北和江南、桥北和桥南，公交车站亦以南北称之。

== 观光 ==
炳草岗大桥180.8米的塔高目前排在[[世界最高桥梁列表/按结构高度排列|世界桥梁结构高度百强]]之内，2001年末刚建成时在中国内地仅次于[[徐浦大桥|徐浦大桥]]、[[杨浦大桥|杨浦大桥]]、[[大佛寺长江大桥|大佛寺长江大桥]]、[[南京长江二桥|南京长江二桥]]等极个别桥梁的索塔高度。虽然此后快速涌现了大量的高桥，但炳草岗大桥在金沙江峡谷和山城风貌的陪衬下仍然拥有举世无双的观赏性。另外由于炳桥是由斜拉桥和T型刚构组成的非对称式组合体，也使其成为一座独特的桥梁。大桥从2012年12月起加强了夜间景观照明效果<ref>{{cite web|url=http://pzhrb.newssc.org/html/2012-12/22/content_1759008.htm|title=绚烂灯光装点炳草岗大桥|publisher=攀枝花日报|date=2012-12-22|5=|deadurl=yes|archiveurl=https://web.archive.org/web/20140914134435/http://pzhrb.newssc.org/html/2012-12/22/content_1759008.htm|archivedate=2014年9月14日|df=}}</ref><ref>{{cite web|url=http://region.scdaily.cn/jrsz/content/2012-12/24/content_4501299.htm?node=3538|title=攀枝花炳草岗大桥“靓”了|publisher=四川日报网|date=2012-12-24|}}</ref>。桥上机动车道两侧有人行道，可以步行过桥，乘坐公交车可以抵达桥南或桥北。由于山城不易骑行非机动车，炳草岗大桥与攀枝花市其它桥梁和道路一样未设置非机动车道。

== 车辆通行费 ==
包括炳草岗大桥在内的整个弄清公路（即弄弄坪路）整体为一个收费还贷公路项目，初期计划设2个收费站，分别在东区炳草岗大桥北和西区凉风坳<ref>{{cite web|url=http://www.110.com/fagui/law_242016.html|title=四川省交通厅、四川省财政厅、四川省物价局关于炳草岗金沙江大桥和弄清公路（含凉风坳隧道）试行收取车辆通行费的批复|publisher=四川省财政厅、四川省物价局、四川省交通厅|date=2002-12-18|}}</ref>。此举引起广泛争议，后因两收费站距离不足等原因，收费未及实施即被叫停。炳草岗大桥北收费站建成后又被拆除，凉风坳收费站则尚未建设。2004年，弄清公路被纳入《攀枝花市路桥车辆通行费年费加次费征收管理办法》，从2005年1月1日起与攀枝花市另外3条公路合并收取年费加次费<ref>{{cite web|url=http://www.panzhihua.gov.cn/zfxxgk/fggw/gfxwj/27327.shtml|title=攀枝花市路桥车辆通行费年费加次费征收管理办法|publisher=攀枝花市公众信息网|date=2006-10-31|5=}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。2009年后随着全国燃油税改革的推进，各地逐渐取消二级及以下公路收费，取消路桥车辆通行费的呼声日高。四川省最终在2013年1月1日0时起取消政府还贷二级公路收费<ref>{{cite web|url=http://scnews.newssc.org/system/2012/12/25/013695511.shtml|title=1月1日起 四川省取消政府还贷二级公路收费|publisher=四川新闻网|date=2012-12-25|}}</ref>，包含炳草岗大桥在内的攀枝花市路桥车辆通行费正式取消<ref>{{cite web|url=http://www.sc.gov.cn/10462/10464/10465/10595/2013/1/9/10243798.shtml|title=攀提前取消政府还贷二级公路“过路费”|publisher=四川省人民政府|date=2013-01-09|}}</ref>。

== 意义 ==
炳草岗大桥与先期建成的弄清公路（即弄弄坪路）和[[凉风坳隧道_(攀枝花)|凉风坳隧道]]一起，将炳草岗、弄弄坪和[[西区_(攀枝花市)|西区]][[清香坪街道|清香坪]]串联起来，形成第二条贯穿东西的主干道，改善了城市路网结构。攀枝花市城区依山势而建，各片区聚落高低不一，此前只有经由较低处的道路才能跨越金沙江，炳草岗大桥是首个高空跨江通道，促进了城市交通立体化。
