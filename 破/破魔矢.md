'''破魔箭'''或'''破魔矢'''（{{lang|ja|はまや}}）是一種日本傳統祈福用品，同一般的箭相比，它沒有箭頭，而箭翎處則飾有[[鈴鐺|鈴鐺]]、[[繪馬|繪馬]]等有吉祥意義的物品。

常常跟破魔矢一同出現的是'''破魔弓'''（{{lang|ja|はまゆみ}}），作爲附屬的儀式福物，起着類似的作用。

== 由來 ==
[[File:Templos_de_Nikko-Japon33.jpg|缩略图]]破魔矢的“破魔”二字被認爲是取自[[佛教|佛教]]用語。關於破魔矢的起源，有着不同的說法。

=== 源賴義起源說 ===
公元11世紀後半葉，[[源賴義|源賴義]]接受了京都[[石清水八幡宮|石清水八幡宫]]賜予的弓箭，前往[[陸奧國|陸奧國]]，由此終結了[[前九年之役|前九年之役]]。據傳該箭現在被供奉於[[鶴岡市|鹤冈]]的八幡宫，該八幡宫是[[源赖朝|源賴朝]]從由比浜遷移過來的。

源賴義年輕時勇武過人，尤擅弓箭。人們希望可以以供奉弓箭的形式來尋求他的庇護，後來便有了破魔矢的說法。

=== 烏摩勒伽起源說 ===
根據另外一說，日本佛教中的[[青面金剛|烏摩勒伽（うまろきゃ）]]持有金弓，他的箭具有殺死惡魔的功效。供奉烏摩勒伽的'''[[輪王寺|輪王寺]]'''以其名義售賣弓箭“'''龍神破魔矢'''”，羣衆由此以期平安，後來便有了破魔矢的說法。<ref>{{Cite web|url=http://rinnoji.or.jp/precincts/taiyuin|title=日光山輪王寺大猷院・「龍神破魔矢」解説詞|accessdate=|author=|date=|publisher=|archive-url=https://web.archive.org/web/20180831060742/http://rinnoji.or.jp/precincts/taiyuin|archive-date=2018-08-31|dead-url=no}}</ref>

因爲破魔矢的意義在於用箭上的淨化魔力攻擊邪魔所散發出的邪氣、邪意、邪道、邪心等不潔的事物，而非攻擊人或動物，因此破魔矢沒有鋒利的箭尖。

=== 農業占卜說 ===

還有一說，認爲破魔矢源於日本古代的農業占卜儀式。<ref>{{Cite web|url=https://www.thenewslens.com/article/54975|title=日式守護符的使用方法|accessdate=|author=|date=|publisher=|archive-url=https://web.archive.org/web/20181104010142/https://www.thenewslens.com/article/54975|archive-date=2018-11-04|dead-url=no}}</ref>

在正月時，各地區會選出自己的代表進行射箭比賽，人們認爲獲勝的地區將會在來年迎來大豐收。在該儀式中使用的弓箭就被稱爲「'''破魔弓'''」和「'''破魔矢'''」。而將破魔矢供奉在家中用於祈福的習俗，則是到了江戶時代（西元1603-1867年，約相當於中國明末到清末）才開始形成。

起初人們使用整套的「破魔弓」和「破魔矢」，後來漸漸簡化爲只供奉單支的「破魔矢」，而破魔弓則成爲了附屬的祈福物。

== 飾物的意義 ==
[[File:Iwamizawa-Shrine_Hamaya.JPG|替代=]]除了繪馬，破魔矢還裝飾有鈴鐺，繪有不同色彩。破魔矢上的繪馬通常繪有當年的生肖，或是祈禱元氣、財富、事業等運勢的語句。鈴鐺發出的鈴聲，被認爲有着吸引神靈寄宿和保佑，驅除邪惡的作用<ref name=":0">{{Cite web|url=https://wonder-solutions.info/1373.html|title=日本網站的破魔矢科普|accessdate=|author=|date=|publisher=|archive-url=https://web.archive.org/web/20181103215005/https://wonder-solutions.info/1373.html|archive-date=2018-11-03|dead-url=no}}</ref>。破魔矢还有着不同的顏色，但是這並沒有特殊的意義<ref name=":0" />。

== 獲得途徑 ==
在[[日本新年|日本新年]]期間，人們在去[[寺庙|寺廟]]、[[神社|神社]]“[[初詣|初詣]]”（はつもうで）時，往往會請奉一隻破魔矢，以此祈求新年的好運與吉祥。
除此之外，舊時在建造房屋時，人們會在“[[上梁|上棟式]]”儀式中在房樑上朝“[[鬼門|鬼門]]”——也就是東北方向放上一隻破魔矢，以此表達驅邪避害的美好願景。在當代，人們會在喬遷新居時贈送破魔矢來替代這一傳統習俗。<ref>{{Cite web|url=http://jyoutousiki.mayuha.com/ryakusiki-17|title=上棟式過程簡述|accessdate=|author=|date=|publisher=|archive-url=https://web.archive.org/web/20170520091910/http://jyoutousiki.mayuha.com/ryakusiki-17|archive-date=2017-05-20|dead-url=no}}</ref>

另外，在男孩出生後過第一個正月或[[男孩節|男孩節]]時，親友也會贈送破魔矢和破魔弓來表示驅災消難。這項儀式後來也普及到了女孩身上。

== 擺放 ==

因爲破魔矢具有特殊的祈福意義，所以需要恭敬供奉，不能隨意擺放。一般來說，破魔矢會被安奉在家中的[[龕|神龕]]或[[凹間|凹間]]等清靜之處，或是其他不會顯得不敬的地方。

以豎直方式擺放時，箭羽須朝上。比較講究者則會讓箭頭指向神社指示的「兇位」。

== 破魔矢的現狀 ==
破魔矢作爲日本傳統的祈福道具，在現今日本人的生活中仍處於重要地位。

但是現在，伴隨着商業化，破魔矢已衍生出了新的形態。比如有一種產品是，生肖不再畫在繪馬上，而是做成立體動物口中銜着或手持迷你破魔矢。這些產品多是作爲紀念品向外國遊客出售，價格比一般的破魔矢要高很多。

在許多影視及遊戲作品中，都出現了破魔矢的影子，比如《[[犬夜叉|犬夜叉]]》和[[阴阳师_(游戏)|陰陽師]]。破魔矢作爲日本傳統文化的代表物之一，正在被世界更多的人民接受和喜愛。

== 其他注意事項 ==
同[[御守|御守]]一樣，破魔矢也是有“保存期限”的。經過一年之後，破魔矢需要被送回原來的神社，通過神社進行的“[[左義長祭|左義長祭]]”等儀式焚化，以此除掉一年以來的壞運氣。
