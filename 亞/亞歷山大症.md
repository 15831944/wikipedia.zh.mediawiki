'''亞歷山大症'''（{{lang-en|'''Alexander disease'''}}）是一種緩慢惡化但致命的[[神經退行性疾病|神經退行性疾病]]，
亞歷山大症是一種非常罕見的疾病，起因於[[基因突變|基因突變]]，主要影響嬰幼兒和兒童，造成[[發育遲緩|發育遲緩]]和身體特徵的變化<ref>[http://www.highbeam.com/doc/1P1-38896449.html "MUTATION KEY TO ALEXANDER DISEASE" - United Press International]{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="Gene">[http://www.ncbi.nlm.nih.gov/bookshelf/br.fcgi?book=gene&part=alexander "Alexander Disease" - Gene Reviews]</ref><ref name="NINDS" />。

== 臨床特點 ==
部份生理、心理及行為的發展出現遲緩，可能有[[大頭症|大頭症]]（macrocephaly，亦作「[[巨头症|巨头症]]」，一種頭部逐步脹大的病變）<ref>{{cite web |url = http://www.mmh.org.tw/earlyintervention/old/%E5%85%92%E7%AB%A5%E7%99%BC%E5%B1%95%E9%81%B2%E7%B7%A9%E7%9A%84%E5%8E%9F%E5%9B%A0.htm |title=兒童發展遲緩的原因 |publisher=[[馬偕紀念醫院|馬偕紀念醫院]]小兒科 |author=邱南昌醫師 |date=2008-05-06 |accessdate=2012-05-25 |language = zh-Hant }}</ref>、[[腦癇症|腦癇]]、[[痙攣|痙攣]]，在某些情況​​下還可能會出現[[腦積水|腦積水]]或[[失智|失智]]（[[癡呆|癡呆]]）<ref name="Gene" />。

== 病理學 ==
亞歷山大病是一種遺傳性疾病，影響[[中樞神經系統|中樞神經系統]]（[[中腦|中腦]]和[[小腦|小腦]]）。
它是一種與[[膠質纖維酸性蛋白|膠質纖維酸性蛋白]]（[[GFAP|GFAP]]）相對應的[[染色體17|染色體17]]q21段的基因突變<ref>{{cite web |url = http://gene.bhp.doh.gov.tw/index.php?mo=DiseasePaper&action=paper1_show&cate=Set1&csn=12&sn=202 |publisher=[[中華民國|中華民國]][[行政院衛生署|行政院衛生署]][[國民健康局|國民健康局]]遺傳疾病諮詢服務窗口 |title=Alexander disease |accessdate=2012-05-25 |language=zh-hant |deadurl=yes |archiveurl = https://archive.is/20120708151908/http://gene.bhp.doh.gov.tw/index.php?mo=DiseasePaper&action=paper1_show&cate=Set1&csn=12&sn=202 |archivedate=2012-07-08 }}</ref>。這種病是一種[[隐性遗传|隐性遗传]][[代谢性疾病|代谢性疾病]]<ref name=9med />，所以若孩子的父母帶有這個基因突變，他們的子女會有一半機會遺傳有這種病<ref name="Gene" />。

亞歷山大症屬於{{le|腦白質失養症|Leukodystrophy}}（Leukodystrophy）的一種類型<ref>{{cite web |url = http://gene.bhp.doh.gov.tw/index.php?mo=DiseasePaper&action=paper1_show&cate=Set1&csn=71&sn=313 |publisher=[[中華民國|中華民國]][[行政院衛生署|行政院衛生署]][[國民健康局|國民健康局]]遺傳疾病諮詢服務窗口 |title = Metachromatic leukodystrophy；MLD |accessdate=2012-05-25 |language=zh-Hant |deadurl=yes |archiveurl = 根據亞歷山大症患者的[[計算機斷層掃描|計算機斷層掃描]]（CT）顯示：
* 患者的[[腦白質|腦白質]]密度較低；
* 腦[[額葉|額葉]]佔主導性；
* +/- [[側腦室|側腦室]]擴張

== 治療 ==
亞歷山大症目前還沒有根治的方法，或標準治療過程<ref name="Gene" /><ref name="NINDS"  /><ref name="Alexander" />。[[同種異體移植|同種異體]][[骨髓移植|骨髓移植]]曾在一名七個月大的患病女嬰身上嘗試，但對她的病情未有改善<ref name="Alexander" /><ref name="Nature">{{cite journal |url = http://www.nature.com/bmt/journal/v20/n3/abs/1700871a.html |title="Allogeneic bone marrow transplantation for Alexander's disease" |journal=[[自然_(雜誌)|Nature]] |language=en |volume=20 |issue=3 |pages=247-249 |authors=M-J Staba1, S Goldman1, F L Johnson2 and P R Huttenlocher3 |date=August 1997 }}</ref>。

== 預後 ==
預後一般較差。對於發病時間較早的病例，病人通常在病徵出現之後10年內死亡。通常情況下，發病時間越晚，病者的病發時期亦越慢<ref name="Gene" /><ref name="NINDS"  />。
