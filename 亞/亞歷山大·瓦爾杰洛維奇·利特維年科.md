'''亞歷山大·瓦爾杰洛維奇·利特維年科'''（[[俄語|俄語]]：'''{{lang|ru|Александр Вальтерович Литвиненко}}'''；[[英語|英語]]：'''{{lang|en|Alexander Valterovich Litvinenko}}'''，{{bd|1962年|8月30日<ref name="telegraph">{{cite web|url=http://www.telegraph.co.uk/news/main.jhtml?xml=/news/2006/11/25/db2502.xml|title=利氏出生日期|date=2006年11月25日|publisher=The Daily Telegraph| accessdate = 2006年11月25日}}</ref><ref name="independent">{{cite web|url=http://news.independent.co.uk/people/obituaries/article2013283.ece|title=Alexander Litvinenko obituary|date=2006年11月25日|publisher=The Independent|accessdate=2007年1月19日|deadurl=yes|archiveurl=https://web.archive.org/web/20071001083243/http://news.independent.co.uk/people/obituaries/article2013283.ece|archivedate=2007年10月1日}}</ref>|2006年|11月23日|catIdx=Litvinenko}}），前[[蘇聯|蘇聯]][[克格勃|克格勃]]上校，原[[俄羅斯聯邦安全局|俄羅斯聯邦安全局]]（以下簡稱FSB）[[中校|中校]]。曾批評部門高層及受罰，2000年离开[[俄羅斯|俄羅斯]]前往[[英國|英國]]，其後出版兩本书批評[[普京|普京]]政府，至2006年10月歸化英籍，后中毒，於同年11月23日不治身亡，体内被验出高浓度剧毒金屬[[釙|釙]]，疑因普京派遣特工到英國将其毒杀，事件引起國際注意，並為全球多家媒體所報導。

== 個人背景 ==
利特維年科生於俄羅斯城市[[沃羅涅日|沃羅涅日]]，1980年完成當地中學課程，其後被徵召加入蘇軍，服役期間因表現良好獲得嘉許，並晉升為[[中校|中校]]。1988年，利特維年科加入[[克格勃|克格勃]]，負責[[反間諜|反間諜]]等工作。1991年加入俄聯邦安全局，進行[[反恐|反恐]]及有組織罪案滲透工作，期間曾參與莫斯科罪案調查局（MUR）的滅罪行動，獲得MUR的嘉許。至1997年，利特維年科被委派往俄聯邦安全局的最秘密部門，從事有組織罪案分析工作。

== 變節 ==
利特維年科專長打擊[[有組織罪案|有組織罪案]]，在1998年，他聲稱[[鮑里斯·別列佐夫斯基|鮑里斯·別列佐夫斯基]]被俄罗斯[[联邦安全局|联邦安全局]]列入暗殺名單之中，別列佐夫斯基與當時的總統[[葉利欽|葉利欽]]私交甚篤。利特維年科也聲稱當時已遭FSB開除，其後曾兩次被捕。

至2000年，利特維年科因提供假證而再度被捕，他在入獄前以偽造護照逃到[[土耳其|土耳其]]與妻子及兒子會晤，同年11月偕同妻兒以申請[[政治庇護|政治庇護]]為由移居英國，2006年10月成為英國公民。其間就[[車臣|車臣]]等問題批評[[俄罗斯总统|總統]][[弗拉基米尔·普京|普京]]政權，因而被列入[[莫斯科|莫斯科]]當局的通緝名單中。

利特維年科在2005年7月接受[[波蘭|波蘭]]報章《共和报》（''{{lang|pl|Rzeczpospolita}}''）的訪問當中，指[[蓋達|蓋達]]組織的[[艾曼·扎瓦希里|艾曼·扎瓦希里]]連同其他組織人物，曾於1998年在[[達吉斯坦|達吉斯坦]]（一毗鄰車臣的[[俄羅斯自治共和國|俄羅斯自治共和國]]）接受FSB的特訓。

== 中毒身亡 ==
2006年11月1日，利特維年科突感不適送院。在接受訪問期間，他指入院前一天曾與兩名前克格勃人員會面，當日下午曾與意大利朋友斯卡拉梅拉（Mario Scaramella）在[[倫敦|倫敦]]一家名為[[:en:Itsu|Itsu]]的西式日本料理快餐店用膳 ，這名[[意大利|意大利]]人聲稱知悉2006年10月於莫斯科寓所遇害的48歲女記者[[安娜·波里科夫斯卡婭|安娜·波里科夫斯卡婭]]案件內情。疑食物遭下毒，他其後更出現[[嘔吐|嘔吐]]及[[休克|休克]]等嚴重病況，更被立即送院治理。

有不少報導稱，下毒主謀為俄羅斯當局。經過多日醫治後，他最終於11月23日宣告不治，終年44歲。

== 调查 ==
2006年11月24日，[[英国卫生防护局|英国卫生防护局]]宣布，利特维年科的尿液內发现了釙-210，且含量极高。这表明利特维年科被人下毒。英国警方也宣布，已在利特维年科的家中和他发病当天光顾的两个地方找到了钋-210的残留物。<ref>{{cite news|url=http://news.xinhuanet.com/world/2006-11/25/content_5373597.htm|title=英国称在俄前特工体内检测出放射性元素|date=2006年11月25日|accessdate=2006-10-26|publisher=新华网}}</ref>

11月29日，[[英国航空公司|英国航空公司]]3架[[波音767|波音767]]客机发现钋-210而停飞接受调查。有两架停在伦敦[[希思罗国际机场|希思罗国际机场]]，另一架则停在莫斯科[[多莫杰多沃国际机场|多莫杰多沃国际机场]]。和利特维年科死亡事件有关的人士搭乘了这3架飞机。英航与10月底以来曾搭乘过这3架客机的乘客联络，英航工作人员也接受检查，确定是否受到污染。<ref>{{cite news|url=http://news.xinhuanet.com/world/2006-11/30/content_5414427.htm|title=英航客机因发现放射性物质接受调查|date=2006年11月30日|accessdate=2007-10-26|publisher=新华网}}</ref>

2007年5月底，英国检查机关指控俄罗斯商人[[安德烈·卢戈沃伊|安德烈·卢戈沃伊]]涉嫌毒杀利特维年科，并要求俄方将其[[引渡|引渡]]到英国。7月5日，[[俄罗斯总检察院|俄罗斯总检察院]]发表公告，正式拒绝此引渡要求。<ref>{{cite news|author=刘洋|url=http://news.xinhuanet.com/world/2007-07/06/content_6334824.htm|title=俄罗斯拒绝向英国引渡特工中毒案嫌疑人|publisher=新华网|date=2007年7月6日|accessdate=2007-10-26}}</ref>7月16日，[[英国外交大臣|英国外交大臣]]说，由于引渡被拒而驱逐4名俄外交官。<ref>{{cite news|author=杨川|url=http://news.xinhuanet.com/world/2007-07/17/content_6386660.htm|title=英国宣布将驱逐4名俄罗斯外交官|date=2007年7月17日|publisher=新华网|accessdate=2007-10-26}}</ref>7月19日，[[俄罗斯外交部|俄罗斯外交部]]宣布4名英国外交官为不受欢迎的人，并限定他们在10日内离开俄罗斯。<ref>{{cite news|author=王丹蒂|url=http://news.xinhuanet.com/world/2007-07/19/content_6402425.htm|title=详讯：俄罗斯宣布驱逐4名英国外交官|publisher=新华网|date=2007年7月19日|accessdate=2007-10-26}}</ref>
