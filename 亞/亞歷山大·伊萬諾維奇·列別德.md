'''亞歷山大·伊萬諾維奇·列別德'''（{{Lang-ru|Алекса́ндр Ива́нович Ле́бедь}}；{{bd|1950年|4月20日|2002年|4月28日|catIdx=L}}）是[[俄罗斯|俄罗斯]]軍人及從政者。<ref name=guardian2002>{{Cite web |url=https://www.theguardian.com/news/2002/apr/29/guardianobituaries.russia |title=General Alexander Lebed |date=2002-04-29 |publisher=The Guardian |author=Ian Traynor}}</ref>

列別德曾參選[[1996年俄羅斯總統選舉|1996年俄羅斯總統選舉]]，在6月16日首輪投票得票居第三位，得票率為14.7%。首輪投票過後，總統[[鲍里斯·尼古拉耶维奇·叶利钦|葉利欽]]於6月18日委任列別德出任[[俄羅斯聯邦安全會議|俄羅斯聯邦安全會議]]秘書長。同年，列別德與[[车臣共和国|車臣]]叛軍首領[[阿斯兰·马斯哈多夫|阿斯兰·马斯哈多夫]]進行談判，雙方達成協議，俄軍撤出車臣，[[第一次車臣戰爭|第一次車臣戰爭]]結束。1996年10月17日，葉利欽解除列別德的聯邦安全會議秘書長職務。<ref>{{Cite web |url=http://people.com.cn/GB/historic/1017/3461.html |title=1996年10月17日 总统签署命令　列别德被解职 |website=人民網}}</ref>

1998年，列別德當選[[克拉斯諾亞爾斯克邊疆區|克拉斯諾亞爾斯克邊疆區]]行政長官。2002年4月28日，列別德乘坐直升機前往出席活動時，直升機在大霧中撞上輸電線後在[[阿巴坎|阿巴坎]]附近墜毀，列別德與其他7人遇難身亡。列別德的遺體葬於[[莫斯科|莫斯科]]的[[新圣女公墓|新圣女公墓]]。<ref name=bbc01>{{Cite news |url=http://news.bbc.co.uk/2/hi/europe/1958939.stm |title=Russia's Alexander Lebed laid to rest |date=2002-04-30 |work=BBC News}}</ref>

列別德有兩子一女。<ref name=guardian2002/>
