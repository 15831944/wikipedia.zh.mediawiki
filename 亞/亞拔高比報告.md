《'''亞拔高比報告'''》（{{lang-en|Abercrombie Report 1948}}），即《'''初步規劃報告書'''》（{{lang-en|Hong Kong Preliminary Planning Report}}），是一份由一位[[英国|英國]]著名的建築師和[[城市规划|城市規劃]]專家阿拔高比爵士於1948年所構建的香港發展城市藍圖。

這報告對於由房屋到交通，市區到鄉郊，經濟發展以至政府決策的方面都有詳盡分析和建議。這份報告亦提出香港必須發展[[香港新市鎮|新市鎮]]的建議<ref>{{Cite web|url=http://www.heritagemuseum.gov.hk/documents/2199315/2199704/tk_tunnel7.pdf|title=時光隧道（七）舊貌展新顏|accessdate=2018-07-23|author=香港文化博物館|date=|publisher=}}</ref>。

== 歷史 ==

香港戰前的城市發展缺乏全面規劃，許多工程都只為解決燃眉之急，而沒有長遠周全的計劃。

[[第二次世界大战|第二次世界大战]]後，英國派出負責策劃[[伦敦|伦敦]]近郊新市鎮的規劃師{{le|亞拔高比|Patrick Abercrombie}}（Patrick Abercrombie）來港，研究[[香港市區|香港市區]]和[[維多利亞港|海港]]的未來發展方向，定出長遠的規劃原則<ref>{{Cite web|url=http://commons.ln.edu.hk/cgi/viewcontent.cgi?article=1003&context=jchkhlp_exhibitions|title=規劃我城|accessdate=2018-07-23|author=嶺南大學|date=|publisher=賽馬會香港歷史學習計劃}}</ref>。

== 報告內容 ==
報告把香港對外定位為東南亞的貿易港，而內部亦要解決人口擠逼的問題，從而提出三大重點：

# 改善城市內部及對外的交通配套
# 開拓市郊，擴充城市空間以解決人口增長引發的問題
# 完善城市規劃的法例和機制

=== 維港兩岸的交通配套 ===
亞拔高比參考1941年大衛．奧雲 (David Owen) 的海港規劃，包括[[維多利亞港|維港]]兩岸的碼頭及配套。其中較前瞻性的是興建海底隧道，不過當時方案是從[[尖沙咀|尖沙咀]]來往[[中環|中環]]，跟1972年落成的[[海底隧道_(香港)|紅磡海底隧道]] ([[九龍|九龍]][[紅磡灣|紅磡灣]]來往[[香港島|港島]][[銅鑼灣|銅鑼灣]])不同。另外亦建議鐵路系統考慮九龍半島以西經[[油麻地|油麻地]]一帶，取代九龍塘至尖沙咀一段。碼頭則主要連繫報告提到會發展的工商業區。
