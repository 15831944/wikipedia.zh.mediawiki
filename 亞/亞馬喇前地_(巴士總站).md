'''亞馬喇前地'''（{{lang-pt|'''PRAÇA FERRIRA AMARAL'''}}、{{lang-en|'''Ferrira Amaral Square'''}}），俗稱：'''亞馬喇'''、'''葡京轉乘站'''、'''亞馬喇轉乘站'''，正式名稱為“'''亞馬喇前地公共巴士轉乘區'''”。亞馬喇前地總站於[[澳門半島|澳門半島]]南部，是[[亞馬喇前地|亞馬喇前地]]的一部份，毗鄰[[嘉樂庇總督大橋|嘉樂庇總督大橋]]，是澳門首個也是最大的公共巴士轉乘區。轉乘區鄰近多個大型娛樂場酒店及商業中心，包括：[[中國銀行澳門分行大廈|中銀總行]]、[[新葡京|新葡京]]、[[葡京酒店|葡京酒店]]、[[永利澳門|永利澳門]]、[[澳門財富中心|澳門財富中心]]及[[英皇娛樂酒店_(澳門)|英皇酒店]]等。

== 沿革 ==
為落實[[澳門特區政府|澳門特區政府]]推行「公交優先」計劃，[[土地工務運輸局|土地工務運輸局]]在亞馬喇前地推出澳門首個巴士轉乘區，目的希望利用較大的空間，以集中巴士線，讓居民於同一地點轉乘巴士，達到“零距離轉乘”效果。提高巴士運作效率。直至2015年1月31日，共有41條、55個方向的巴士路線途經亞馬喇前地巴士轉乘區，為澳門最多巴士使用的巴士站點。<ref> [http://www.dssopt.gov.mo/tc/news/index.php?id=520  土地工務運輸局新聞稿]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 歷史 ==
* 2008年2月5日起，亞馬喇前地公共巴士轉乘區首次有巴士路線進駐，最初進入該站的線路為原總站設於[[羅保博士街|羅保博士街]]的[[澳門巴士MT1、MT2路線|澳氹快線MT1、MT2線]]及夜間巴士路線[[澳門巴士N1線|N1]]路線。
* 2008年3月19日上午10時開始，位於新葡京門口之「殷皇子馬路／葡京」正式取消。所有停靠該站的路線改為停靠本站，[[新福利9路線|9]]、[[新福利9A路線|9A]]路線除外。但由於同日起，土地工務運輸局進行路面重鋪工程影響，只有七條離島線，包括[[澳巴11路線|11]]、[[澳巴21A路線|21A]]、[[澳巴22路線|22]]、[[澳巴25路線|25]]、[[新福利26A路線|26A]]、[[新福利28A路線|28A]]、及[[新福利33路線|33]]路線首先停靠亞馬喇前地公共巴士轉乘區。
* 2008年3月28日起，亞馬喇前地公共巴士轉乘區正式全面投入運作，除七條離島線之外，兩家巴士公司其餘包括共[[新福利3路線|3]]、[[新福利3A路線|3A]]、[[新福利8路線|8]]、[[澳巴10路線|10]]、[[澳巴10A路線|10A]]、[[澳巴10B路線|10B]]、[[澳巴12路線|12]]、[[澳巴23路線|23]]、[[新福利28BX路線|28BX]]及[[新福利32路線|32]]路線延伸本站。
* 2008年12月7日起，[[交通事務局|交通事務局]]實施第一階段[[巴士路線調整|巴士路線調整]]，新增之[[澳巴25X路線|25X]]路線為途經該站路線之一。
* 2009年4月26日起，[[交通事務局|交通事務局]]實施第二階段[[巴士路線調整|巴士路線調整]]。亞馬喇前地公共巴士轉乘區由四條通道增至五條通道，同時，路線調整中新增[[新福利2A路線|2A]]、[[新福利3X路線|3X]]、[[新福利5X路線|5X]]、[[新福利7A路線|7A]]及[[澳巴10X路線|10X]]路線以此站為終點站，另外，[[澳門巴士H1線|H1]]山頂醫院線亦延伸至本站。
* 2013年10月26日起，[[交通事務局|交通事務局]]對亞馬喇前地公共巴士轉乘區作出調整，增設F車道，使轉乘區的通道增至六條，同時把原停靠C車道的[[澳巴22路線|22]]、[[澳門巴士25路線|25]]、[[澳門巴士25X路線|25X]]及[[澳門巴士28A路線|28A]]路線會改為停靠F車道，[[新福利28C路線|28C]]則會改為停靠D車道，C車道只保留[[澳門巴士22F路線|22F]],另外，新增之[[澳門巴士73路線|MT3U]]路線停靠新增的F車道。
* 2014年10月28日起，[[交通事務局|交通事務局]]對亞馬喇前地公共巴士轉乘區作出調整，50及50X路線分別改為停靠B及A車道。<ref>[http://www.dsat.gov.mo/bus/site/bus_news_detail.aspx?a_id=2608 優化亞馬喇轉乘站乘車秩序 50、50X路線調整靠站位置]</ref>
* 2016年12月12日起，本站D車道及E車道路線對調。<ref>[http://www.dsat.gov.mo/bus/site/bus_news_detail.aspx?a_id=3651 亞馬喇前地巴士轉乘站周一二部分路線調整停靠區域]</ref>
*2017年8月19日，[[澳巴11路線|11]]、[[澳門巴士MT2路線|MT2]]路線改為停靠E車道。<ref>{{cite news |title=四路線周六起取消停靠“亞馬喇前地”站 |url=http://www.dsat.gov.mo/dsat/news_detail.aspx?data_category=1&a_id=4039 |publisher=[[交通事務局|交通事務局]] |date=2017-08-17}}</ref>

== 設施 ==
* 六條候車通道，每條通道有兩個候車亭；
* 設有四個候車區域，提供六個不同方向路線選擇；
* 在A、B及C、D兩個候車區域設有轉乘站指引圖；
* 兩家業者站長室各一；
* 公廁：設於地下停車場內；
* 在A通道出口提供給公車使用的交通燈一組。

== 澳門通轉乘優惠 ==
{{main|澳門巴士轉乘服務}}
'''轉乘服務'''是澳門兩間巴士公司提供予乘客之轉車優惠，讓乘客透過轉乘模式，用一程車費，搭乘最多三程的路線，免除點對點路線服務的不足。

亞馬喇前地公共巴士轉乘區的設置，主要就是為了推廣'''轉乘服務'''。

== 嚴重事故 ==

== 爭議 ==
* 候車亭太小，根本不能滿足大量乘客轉乘。
* 環境太空曠，遮蔭的設備不足。當太陽猛烈和下雨時，不方便候車者。
