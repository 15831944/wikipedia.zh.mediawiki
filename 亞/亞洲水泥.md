'''亞洲水泥'''，簡稱'''亞泥'''，為[[遠東集團|遠東集團]]旗下[[水泥|水泥]]企業，1957年3月21日成立，董事長為[[徐旭東|徐旭東]]<ref>[http://www.acc.com.tw/about-us/company-profile-history 官網公司簡介]</ref>。

==歷史沿革==
亞洲水泥股份有限公司(以下簡稱亞泥)，由[[徐有庠|徐有庠]]於1957年所創立,生產洋房牌水泥，於[[台灣|台灣]][[新竹縣|新竹縣]][[橫山鄉|橫山鄉]]設立首座水泥廠，後來在1973年於[[花蓮縣|花蓮縣]][[新城鄉|新城鄉]]設立第二座廠 ，目前新竹與花蓮兩廠的廠能已達500萬公噸，此外公司在中國大陸也有轉投資水泥事業-[[亞洲水泥（中國）|亞洲水泥（中國）]]，其在2008年於[[香港聯交所|香港聯交所]]上市。

==2017年亞泥採礦權展延爭議==
在2017年亞泥採礦權展延爭議中，多位[[經濟部礦務局|經濟部礦務局]]現職與退休高官兼任[[中華民國鑛業協進會|中華民國鑛業協進會]]理監事，疑似加快放行礦權展延案。<ref>{{Cite web |url = http://www.epochtimes.com.tw/n215190/%E7%90%83%E5%93%A1%E7%95%B6%E8%A3%81%E5%88%A4-%E7%A4%A6%E5%8B%99%E5%B1%807%E9%AB%98%E5%AE%98%E5%85%BC%E6%B0%91%E9%96%93%E7%A4%A6%E6%A5%AD%E5%8D%94%E6%9C%83.html |title = 球員當裁判 礦務局7高官兼民間礦業協會  |date = 2017-06-16  }}</ref>

6月25日，不滿經濟部在《礦業法》修法通過前再次讓亞泥取得20年礦權展延，20多個民間團體發起《看見亞泥搶救太魯閣》遊行活動，三千人上街頭，訴求主張撤銷礦權展延。亞泥對此回應已將在太魯閣國家公園內之礦區去除，一切合法，亞泥董事長[[徐旭東|徐旭東]]並講出名言「挖深是對還是不對呢?我說將來養魚也是對的」，引發網友批評。<ref>{{Cite web |url = https://udn.com/news/story/7321/2545729 |title = 反亞泥礦權展延 3千民眾上街怒吼  |date = 2017-06-25  }}</ref><ref>{{Cite web |url = https://money.udn.com/money/story/5648/2546194 |title = 亞泥發聲明 捍衛礦權 |date = 2017-06-26 }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url = https://theme.udn.com/theme/story/10926/2546657 |title = 亞泥董座徐旭東：挖深是對的 養魚也沒有錯  |date =2017-06-27  }}</ref>

對於亞泥礦權展延案是否有官商勾結情事，[[台北地檢署|台北地檢署]]已主動簽分[[他字案|他字案]]偵辦。<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20170628/1149858/ |title = 亞泥展延有黑幕？　北檢分案查有無官商勾結  |date = 2017-06-28 }}</ref>

7月11日[[立法院|立法院]]已有半數以上[[立法委員|立法委員]]要求將《礦業法》修正案排入第二次臨時會審查。[[民主進步黨|民主進步黨]]立法委員[[林淑芬|林淑芬]]表示不該以法案應公告兩個月為由拖延修法時程，並批評新上任礦務局局長所公告之法案居然與已下台局長的雷同，這難道還是「新局長」嗎？<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/2127896 |title = 亞泥礦權展延惹議 近半立委要求二次臨會排審  |date = 2017-07-11  }}</ref>

7月17日[[監察院|監察院]]副院長[[孫大川|孫大川]]、[[監察委員|監察委員]][[林雅鋒|林雅鋒]]率領調查官到亞泥新城山礦場履勘，林雅鋒表示，5月起著手調查亞泥礦權展延案，若涉及不法，將依法處理<ref>{{Cite web |url = https://udn.com/news/story/6656/2588800 |title = 亞泥展延案惹議 監察院履勘：若涉不法將依法處理  |date = 2017-07-17 }}</ref>。

8月1日[[行政院|行政院]]政務顧問[[伊凡·諾幹|伊凡·諾幹]]承認，這一年來，[[蔡英文政府|蔡英文政府]]不但未積極處理《原住民族土地或部落範圍土地劃設辦法》爭議與亞泥採礦權展延爭議，甚至也沒有將[[臺灣原住民族|臺灣原住民族]][[轉型正義|轉型正義]]的議題納入《促進轉型正義條例》<ref>{{Cite web|url=https://www.civilmedia.tw/archives/66705|author=林庭葦|title=總統向原住民族道歉滿周年 原團批原轉會淪為公關部門|publisher=公民行動影音紀錄資料庫|date=2017-08-02|accessdate=2019-02-08|language=zh-tw}}</ref>。

8月17日亞泥花蓮廠員工在公聽會場外陳情，希望《礦業法》修法能夠合理、並保障員工生存權與工作權<ref>{{Cite web |url = https://udn.com/news/story/7314/2647899 |title = 礦業法修法公聽會 亞泥員工捍衛工作權  |date = 2017-08-17 }}</ref>。

9月4日[[行政院訴願委員會|行政院訴願委員會]]做出駁回居民訴願的決定<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20170907/1199275/ |title = 亞泥展延合法　環團將提行政訴訟   |date = 2017-09-07   }}</ref>。
