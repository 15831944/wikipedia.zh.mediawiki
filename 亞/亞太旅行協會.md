'''亞太旅行協會'''（Pacific Asia Travel Association, PATA）是於1951年於[[美國|美國]][[夏威夷|夏威夷]]創立的一個非營利組織，其宗旨為促進在亞太地區的觀光事業發展，為亞太地區最重要的國際觀光組織之一，會員為政府、航空公司及觀光產業三類的團體會員<REF name="千華數位文化商鼎數位出版2014"/>。

亞太旅行協會成立目標在於協助各分會會員及地區對外及對內之觀光發展，並推廣旅遊產業對國際社群應有之責任<ref name = "亞太旅行協會中華台北分會"/>。該會推展来亚太地区及区域内關於旅行与旅游业的可持续增长、价值和品质的提升<ref name = "关于PATA"/>。

==成立歷程==
亞太旅行協會的前身是「太平洋临时旅游协会」（Pacific Interim Travel Association），由1952年首届太平洋地区旅行大会（Pacific Area Travel Conference）的91名与会代表開始準備成立，名於太平洋地区旅游协会（Pacific Area Travel Association）正式成立於第二届太平洋地区旅行大会。1986年在[[科伦坡|科伦坡]]，经表决协会正式更名为亚太旅游协会<ref name = "关于PATA"/><ref name="GeeLurie2001"/>。

==影響力==
根據《中国旅游年鑑》，中國旅遊業為中國城市爭取「[[世界旅游组织|世界旅游组织]]、[[亚太旅游协会|亚太旅游协会]]等国际旅游机构重大会展和论坛的举办地」<REF name="中国旅游年鑑2009"/>。

==部門機構==
亞太旅行協會有公關部、會員發展部、展覽部與資訊服務部共4個部門機構<ref name = "亞太旅行協會中華台北分會"/>。

協會會員有80多個國家及地區的政府單位、數百家旅遊及旅館業、40多個郵輪及航空公司、以及隸屬世界各地PATA分會之數千名旅遊專業人士<ref name = "亞太旅行協會中華台北分會"/>。

==服務內容==
關於服務內容，亞太旅行協會促進亞太地區間、區域內相關旅遊業持續性增長和品質的提升；為個人與團體提供專業諮詢服務和指導；對旅遊業相關數據分析（包括對亞太地區出、入境旅遊的統計、預測和分析）做出旅遊市場戰略報告；增加各會員國旅遊商機以各種旅遊活動；支持環境保護、文化遺產的維護、及教育。
