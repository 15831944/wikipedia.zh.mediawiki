'''阿拉姆語'''（{{lang-arc|ܐܪܡܝܐ}}；亦汉譯為'''亚拉姆語'''、'''亞蘭語'''、'''阿拉米語'''、'''阿拉美語'''或'''阿辣米语'''<ref>[http://www.jewishjournal.com/you_tai_ren/item/two_names_of_the_twelve_prophets_20100502/ 这个“阿辣米语”是什么语言？]</ref>，下稱阿拉姆語）是[[闪米特语族|闪米特语族]]（[[閃族|閃族]]）的一种语言，与[[希伯来语|希伯来语]]和[[阿拉伯语|阿拉伯语]]相近。

==歷史==
[[阿拉姆語|阿拉姆語]]有3000年的历史，是世界上少数存活了上千年的古老语言之一。它是[[旧约圣经|旧约圣经]]中的[[以斯拉记|以斯拉记]]及[[但以理书|但以理书]]书写时所用的语言，并被认为是[[耶稣基督|耶稣基督]]时代的[[犹太人|犹太人]]日常時的用语。一些學者更認為[[耶稣基督|耶稣基督]]是以這種语言傳道，被以通行於[[東|東]][[地中海|地中海]]的普通[[希臘文|希臘文]]記錄下來。

现在，还有许多居住在[[伊拉克|伊拉克]]、[[叙利亚|叙利亚]]、[[黎巴嫩|黎巴嫩]]、[[亚美尼亚|亚美尼亚]]、[[瑞典|瑞典]]、[[美国|美国]]、[[西欧|西欧]]、[[拉丁美洲|拉丁美洲]]等国家和地区的[[亚述人|亚述人]]及部分[[阿拉姆人|阿拉姆人]]（即[[阿拉美亚人|阿拉美亚人]]）后裔使用阿拉姆語。现代'''阿拉姆語'''又被称为'''新亚述语'''或新阿拉姆語。使用[[阿拉姆語|阿拉姆語]]的人多信仰[[叙利亚正教|叙利亚正教]]、[[东方亚述教会|东方亚述教会]]、迦勒底[[天主教|天主教]]、叙利亚天主教或[[曼达教派|曼达教派]]。[[美国|美国]]有阿拉姆語电视台，而澳大利亚[[悉尼|悉尼]]的民族[[电视台|电视台]]每週日都有两小时的阿拉姆語节目。而[[微軟|微軟]][[Windows|Windows]]也由[[Windows_XP|Windows XP]]开始支持[[阿拉姆語|阿拉姆語]]，并提供相关的字型。

== 地理分布 ==
在[[公元前12世纪|公元前12世纪]]，以阿拉姆語为[[母语|母语]]的[[阿拉姆人|阿拉姆人]]，开始大量迁入现在的[[叙利亚|叙利亚]]、[[伊拉克|伊拉克]]和东[[安纳托利亚|安纳托利亚]]地区。阿拉姆語因此变得越来越通行，它开始在整个[[地中海|地中海]]东部的沿海地区传播开，一直传播到[[底格里斯河|底格里斯河]]的东部。随着[[犹太人|犹太人]][[移民|移民]]的进入，阿拉姆語也被带进了[[北非|北非]]和[[欧洲|欧洲]]，基督传教士把它带进[[萨珊王朝|波斯]]、[[笈多王朝|印度]]，甚至[[唐朝|大唐]]。但从7世纪中叶起，[[阿拉姆語|阿拉姆語]]被[[阿拉伯语|阿拉伯语]]所取代成为[[中东|中东]]的通用语言。然而，在[[犹太人|犹太人]]、[[曼达教|曼达教]]信徒和[[基督教徒|基督教徒]]中阿拉姆语作为一种[[文学|文学]]和[[礼拜|礼拜]]仪式的语言保存了下来<ref>[http://www.jewishjournal.com/you_tai_ren/item/_20100422/ 犹太人的“卡迪士”基本上都是阿拉姆语]</ref>。今天，在原本受阿拉姆語影響的範圍內還有少數個別且分散的群體在使用這一個語言。最近兩個世紀以來的社會動盪促使講[[阿拉姆語|阿拉姆語]]的人被分散到世界各地去，這當中包括以它為母語的人及使用它作為文學和禮拜儀式的人。阿拉姆語被现代[[亚述人|亚述人]]当做唯一的民族语言。

阿拉姆語实际上是一组相關的语言，而不是单一的一种语言。[[阿拉姆語|阿拉姆語]]悠長的歷史，其廣泛的文學作品及為不同宗教社群所使用，都是這語言分化的原因。阿拉姆語的方言只有部份能溝通，其他的都不可以。阿拉姆語有些分支現時以其他語言的名稱存在，例如：[[敘利亞語|敘利亞語]]。大多數阿拉姆語的方言都可以歸納為東部方言或西部方言，以[[幼發拉底河|幼發拉底河]]或河的西岸為分界線。阿拉姆語是一個包含很廣泛的語言家族，所以現時世代通行的阿拉姆語又被稱為「[[新阿拉姆語|新阿拉姆語]]」，以資識別。至於已不再通行的阿拉姆語成員，我們可以按其通行年代劃分，可劃分為「現代」、「中世代」及「古代」。這個同時從地域及時代的劃分，基本上可以讓我們把這個言族的語言加以分類。

== 书写体系 ==
[[File:Syriac_Sertâ_book_script.jpg|thumb]]書寫的書]]

阿拉姆語曾經用過兩種文字來書寫，而這兩種文字都有共同的源頭。現時我們所描述的[[阿拉姆字母|阿拉姆字母]]其實源自[[腓尼基字母|腓尼基字母]]。阿拉姆字母的獨特「方角」亦影響了[[希伯來字母|希伯來字母]]的發展，使它從最初與腓尼基字母差不多的式樣發展至今日的樣貌；而另一方面，受了阿拉姆字母影響的希伯來字母亦成為了[[猶太人|猶太人]]用來書寫阿拉姆語的工具，也就是[[新約聖經|新約聖經]]時代猶太人所通用的語言。

而基督徒社區亦從原有的阿拉姆字母發展出另一套草書體的字母，是為今日的[[敘利亞字母|敘利亞字母]]。另外還有一種從阿拉姆字母大幅修改過的字母，用於[[米底安語|米底安語]]，是為[[米底安字母|米底安字母]]。

== 古代阿拉姆語 ==
古代阿拉姆語是一门经历了13个世纪发展历程的语言。经历如此漫长的时间间隔是因为它包括所有如今已经灭绝的阿拉姆語。古代阿拉姆語的主要转折点是在公元前500年，当古阿拉姆語（阿拉姆人用的语言）成为法定阿拉姆語（强大帝国用的语言）的时候。当[[希腊语|希腊语]]代替阿拉姆語成为当地的法定语言时，阿拉姆語已经发展成为多种多样的交谈方言。

=== 古阿拉姆語 ===
古阿拉姆語是指从其起源到它成为[[新月沃地|新月沃地]]的正式“[[通用语|通用语]]”这一时期[[阿拉姆人|阿拉姆人]]的阿拉姆語。它是[[大马士革|大马士革]]、[[哈马斯|哈马斯]]和[[Arpad(Syria)|阿尔帕德]]这几个城邦的语言。

==== 早期古阿拉姆語 ====
从公元前10世纪起就有许多阿拉姆語的碑铭。这些碑铭是古代阿拉姆城邦的政治文书。这一时期的阿拉姆語正寫法似乎是基于腓尼基字母的，并且在书面语言中有统一的迹象。在东部阿拉姆地区，这一时期已经有了统一的正寫法、统一语言的趋势。特别的，Tiglath-Pileser III统治下的亚述帝国在公元前8世纪征服阿拉姆地区，导致阿拉姆語作为官方语言的建立。

[[File:Bar-rakib.jpg|left]])]]

==== 晚期古阿拉姆語 ====
公元前7世纪起，该语言开始向四方广泛传播，但同时失去了自身的同质性。在[[美索不達米亞|美索不達米亞]]、[[巴比伦|巴比伦]]、[[黎凡特|黎凡特]]、[[埃及|埃及]]出现了不同的方言。不过，阿卡德语开始影响亚述的阿拉姆語，只有有影响了巴比伦等地区的语言。在圣经[[列王纪下|列王纪下]]第十八章26節中，犹大王[[希西家|希西家]]用阿拉姆語和亚述使臣交流，使得老百姓无法理解他们在说什么。大约公元前600年，[[迦南王|迦南王]][[安东|安东]]曾用阿拉姆語给埃及法老写信。

迦勒底阿拉姆語是[[迦勒底人|迦勒底人]]的[[新巴比倫王國|新巴比倫王國]]的通用阿拉姆語，[[聖經|聖經]]部分章節是用這種語文寫成的。另外有現代語言的新迦勒底亞蘭語，須注意不要混淆。

=== 帝国阿拉姆語 ===
公元前5世紀，[[阿契美尼德帝國|阿契美尼德帝國]]的[[大流士一世|大流士一世]]把阿拉姆語訂立為帝國西部的官方語言。雖則在前巴比倫帝國管治的地方裡，當地官員早已在日常工作裡使用東阿拉姆語在各地的各種方言，但大流士的詔令使阿拉姆語處於一個堅固及統一的基礎上。

==== 後阿契美尼德阿拉姆語 ====
[[亞歷山大大帝|亞歷山大大帝]]的征服未有使阿拉姆語言及文學立即被摧毀：直到公元後2世紀，在當地仍然可找到以阿拉姆語的文字紀錄，而且與公元前5世紀時所通行的沒有太大分別。
