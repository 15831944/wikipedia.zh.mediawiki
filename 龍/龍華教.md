'''龍華教'''為[[台灣民間信仰|台灣民間信仰]]的一支，又稱為'''龍華齋教'''，因持齋的教徒互相尊稱為「老官」，又稱為'''老官齋教'''，部份教徒也自稱'''在家佛教'''。在[[台灣|台灣]][[台灣日治時期|日治時期]]，為了管理方便，[[總督府|總督府]]將當時流行於台灣的民間宗教'''龍華教'''、[[先天道|先天道]]、[[金幢教|金幢教]]一同列入[[齋教|齋教]]管理，故後世台灣學者也有稱成為'''齋教龍華派'''。龍華教教徒和佛教一樣同守三皈五戒，吃長齋，但可嫁娶<ref>[http://ccbs.ntu.edu.tw/FULLTEXT/JR-BJ010/bj97036.htm#2 從龍華教到佛教---光復前台南德化堂的歷史]</ref>。龍華教起源於[[中國|中國]]，在[[台灣清朝統治時期|清治]][[乾隆|乾隆]]年間傳入[[台灣|台灣]]<ref>[林美容。2008。台灣的齋堂與巖仔-民間佛教的視角。初版，台灣齋堂的發展源流與現況，台灣地區的齋堂。台北：台灣書局。]</ref>，現雖式微，但在台灣宗教之中仍有一定的影響力。

== 歷史 ==
龍華教是[[中國|中國]][[明代|明代]]時的[[中國民間宗教|民間信仰]]，由[[羅思孚|羅思孚]]、[[殷繼南|殷繼南]]、[[姚文宇|姚文宇]]三祖逐漸建立，但從歷史的考證來說，龍華教並不是由羅思孚直接創立，是由二祖[[殷繼南|殷繼南]]在整理羅思孚的思想後，再加上[[儒教|儒教]]、[[道教|道教]]的教義所整合，而三祖姚文宇並未在教義上有所變動，而是將其制度化及儀式化，並鼓勵齋友興建齋堂以傳教。

'''[[羅思孚|羅思孚]]'''，[[山东省|山东省]][[莱州府|莱州府]][[即墨縣|即墨縣]]人，從軍於[[北直|北直]][[密雲|密雲]]，退伍後於各大[[寺廟|寺廟]][[打坐|打坐]]修行，認為人都應該要虛靜無為，以便回歸「真空家鄉」的「[[無生老母|無生父母]]」身邊，于[[正德|正德]]四年（1509年）在密雲衛靈霧峯開創[[羅教|羅教]]。

'''[[殷繼南|殷繼南]]'''，一說名'''應繼南'''，[[浙江省|浙江省]][[處州府|處州府]][[縉雲縣|縉雲縣]]人，本是羅教丁予的弟子，法號「普能」，因為善於演說，自稱是羅思孚[[轉世|轉世]]，所以得到許多[[檀越|檀越]]的信仰。他將羅思孚的思想整理後，歸納出一套系統，[[萬曆|萬曆]]年間出行傳教，集結信徒達三千七百人。因而萬曆四年被官府追捕，囚禁六年，但一出獄又吸引大批信眾召開法會，官府立刻以「妖言惑眾罪」收斬，辭世詩曰：「將頭臨白刃，猶似斬春風，智不對愚訴，如刀劈水中。」（或作「如刀劈水聲」。）死後由弟子「普福」繼承掌門。

'''[[姚文宇|姚文宇]]'''，浙江省處州府[[慶元縣|慶元縣]]人，自稱為[[殷繼南|殷繼南]][[應化身|應化身]]。姚創了「禮、義、廉、恥、孝、悌、忠、信」等八字輩，後又加上了「和」輩，他自稱「教主姚相公」，後其義子金佛壽向[[南明|南明]]將領[[楊鼎卿|楊鼎卿]]進讒言，楊鼎卿率兵包圍姚家，將姚文宇殺死。姚妻周夫人攜姚鐸逃出，姚鐸子姚軾、姚轍，軾、轍兩人復興了'''龍華教'''。故在中國江浙和江南地區，姚文宇派下仍奉其子[[姚鐸|姚鐸]]為「四祖」。<ref>[http://thomrun.com/ming/page10/styled-7/index.html 龍華齋門課程] {{Wayback|url=http://thomrun.com/ming/page10/styled-7/index.html |date=20160304091426 }}</ref>

龍華教在[[台灣清治時期|清治時期]]傳入台灣，[[台灣清治時期|清治]][[乾隆|乾隆]]三十年（1765年）在[[臺南|臺南]][[安平|安平]]設立第一座齋堂「善化堂」，此後宗教勢力漸擴大。[[台灣日治時期|日治時期]][[大正|大正]]九年（1920年）在台灣更成立「[[臺灣佛教龍華會|臺灣佛教龍華會]]」，廖炭任首屆會長，教勢達到頂峰。[[二次大戰|二次大戰]]戰後以來，因[[國民黨|國民黨]]的政治管制有式微趨勢，[[解嚴|解嚴]]後稍有復興跡象。現有龍華聯誼會，作為齋堂間的聯誼中心。

== 源流 ==
根據龍華教本身的記載，龍華教本身的法派來源於「先天初祖」<ref>[林美容。2008。台灣的齋堂與巖仔-民間佛教的視角。初版，朝天堂龍華派之法脈源流，在家佛教-台灣彰化朝天堂所傳的龍華派齋教。台北：台灣書局。]</ref>，之後其法脈傳至[[伏羲氏|伏羲氏]]、[[龍吉先生|龍吉先生]]、[[神農氏|神農氏]]、[[廣成子|廣成子]]、[[黃帝|黃帝]]、[[許由|許由]]、[[唐堯|唐堯]]、[[巢父|巢父]]、[[虞舜|虞舜]]至[[燃燈佛|燃燈佛]]，傳[[過去七佛|過去六如來]]，直到[[釋迦佛|釋迦佛]]，釋迦佛傳法給[[摩訶迦葉|摩訶迦葉]][[羅漢|尊者]]，摩訶迦葉傳予[[阿難|阿難]]尊者，歷傳[[西天二十八祖|西天二十八祖]]而至[[達摩祖師|達摩祖師]]，達摩傳[[神光|神光]]、[[鑒智|鑒智]]、[[道信|道信]]、[[弘忍|黃梅弘忍]]、[[慧能|曹溪惠能]]、[[南岳懷讓|南岳懷讓]]，至二十五祖為[[李頭陀|李頭陀]]，[[法脈|法脈]]改傳俗家，即為[[羅教|羅教]]初祖羅祖[[羅思孚|羅思孚]]。

== 教義 ==
'''龍華教'''起源於[[中國|中國]][[明朝|明朝]][[羅教|羅教]]，又多混合[[佛教|佛教]][[禪宗|禪宗]][[臨濟宗|臨濟宗]]、[[淨土宗|淨土宗]]、[[儒教|儒教]]、[[道教|道教]]甚至是[[摩尼教|摩尼教]]等信仰，也吸納了[[儒家|儒家]][[孝順|孝親]][[仁|愛人]]的說法。其所悟之道稱為「無為法」。有《[https://zh.wikisource.org/zh-hant/%E4%BA%94%E9%83%A8%E5%85%AD%E5%86%8C 五部六冊]》之重要著作，目的在救眾出離生死苦海，永超凡世。

龍華教教義有五大核心：
*佛教常唸誦之經卷、禪宗頓悟之功夫
*道教之經咒歌訣、養生之道，
*儒家之倫理思想、天道觀念，
*[[羅祖|羅祖]]的無為教系統，即人如何開悟到世界本體以返回[[真空家鄉|真空家鄉]]和[[無生老母|無生老母]]。
*[[殷繼南|殷繼南]]所傳授的「三乘教法」，其內涵為[[無生老母|無生老母]]、[[彌勒佛|彌勒佛]]、三期末劫信仰，是受到當時民間教派的影響。

== 齋堂法器與擺設 ==
*龍華派齋堂，傳統正殿主奉'''[[觀世音菩薩|觀世音菩薩]]'''，觀世音菩薩像對面供奉'''[[韦陀|韋陀尊天菩薩]]'''，但亦會供奉其餘佛道教神祇，如:[[燃燈佛|燃燈古佛]]、[[釋迦如來|釋迦如來]]、[[藥師如來|藥師如來]]、[[彌陀如來|彌陀如來]]、[[彌勒如來|彌勒如來]]、[[文殊菩薩|文殊菩薩]]、[[地藏菩薩|地藏菩薩]]、[[普賢菩薩|普賢菩薩]]、[[勢至菩薩|勢至菩薩]]、[[善財|善財]]、[[龍女|龍女]]、[[十八羅漢|十八羅漢]]、[[三官大帝|三官大帝]](三界公)、[[呂仙祖|呂仙祖]]、[[關聖帝君|關聖帝君]]、[[天上聖母|天上聖母]]、[[福德正神|福德正神]]、[[城隍|城隍老爺]]等。

*佛堂正殿神桌上，會擺置三張小木椅，俗稱'''三公椅'''象徵[[羅清|羅普仁]]、[[殷繼南|殷普能]]、[[姚普善|姚普善]]三位祖師。

*'''五部六冊寶卷'''（羅祖撰）、'''龍華科儀寶卷'''（湯普霄編）、'''金剛科儀寶卷'''（又稱金剛對卷，大乘佛教經懺典籍，'''宗鏡禪師'''編輯，收錄於[[大正藏|大正藏經]]，多用於[[作功德|作功德]]。）

== 節慶與儀式 ==
*龍華派的儀式可以分為例行祭儀、朔望供佛（農曆初一、十五日）、佛誕祭儀、「開光場」法會和作功德。
*'''光場法會'''，為期七天，除一般佛事外，亦會誦讀[[梁皇寶懺|梁皇寶懺]]、[[金剛經|金剛寶懺]]、[[八十八佛洪名寶懺|八十八佛洪名寶懺]]、[[藥師經|藥師寶懺]]、[[阿彌陀經|彌陀寶懺]]等佛教經懺經典，以及'''五部六冊'''。亦會配合進行[[蒙山施食|蒙山施食]]及瑜珈[[焰口施食|焰口施食]]佛事。
*龍華派之神佛慶日，除了齋教各派都紀念的[[羅清|羅祖大師]]三大節日 （羅祖生日：十二月初一日；得道日：十月十八日；升天日：正月廿九日）之外，尚有：[[彌勒菩薩|彌勒古佛佛誕]] （正月初一日）、[[玉皇誕|玉皇上帝聖誕]] （農曆正月初九）、[[天官大帝|天官大帝聖誕]] （正月十五；[[上元節|上元節]]）、[[釋迦文佛|釋迦文佛出家]] （二月初八，佛出家）、[[觀世音菩薩|觀音大士誕]] （二月十九；觀音聖壽）、[[佛誕|釋迦文佛佛誕]] （四月初八，俗稱如來佛祖誕、釋迦老祖誕、佛祖誕）、觀音大士得道日 （六月十九；觀音歸空）、[[地官大帝|地官大帝聖誕]] （農曆七月十五；[[中元節|中元節]])、[[地藏王菩薩|地藏王菩薩佛誕]] （農曆七月廿九或三十）、觀世音菩薩出家 （農曆九月十九）、[[水官大帝|水官大帝聖誕]] （農曆十月十五；[[下元節|下元節]]）、[[釋迦牟尼|釋迦文佛成道]] （十二月初八），依各齋堂祭祀神祇不同，亦有其餘慶典。
