{{engname|《龍騰四海》|'' Gun n' Rose ''}}是1992年出品的一部香港電影，由[[霍耀良|霍耀良]]執導，以警匪故事為電影題材，[[黎明|黎明]]、[[鄧光榮|鄧光榮]]、[[劉德華|劉德華]]、[[任達華|任達華]]、[[林保怡|林保怡]]、[[陳法蓉|陳法蓉]]、[[吳家麗|吳家麗]]、[[李麗珍|李麗珍]]等演出<ref>{{Cite web |url= http://hkmdb.com/db/movies/view.mhtml?id=7610&display_set=big5|title =龍騰四海（1992）| publisher =香港影庫|language = 中文}}</ref><ref>{{Cite web |url= http://www.hkbo.com.hk/search-details-new.php?filmid=9797&run=1 |title =電影資料庫| publisher = 香港票房有限公司|language = 中文}}</ref>，1992年6月5日至6月18日期間在香港上映，是1992年全年最高開畫日票房的五大電影之一<ref>{{Cite web |url= http://www.hkbo.com.hk/top5.html?year=1992|title =1992年全年最高五大電影開畫日票房| publisher = 香港票房有限公司|language = 中文}}</ref>。

==情節介紹==
電影內容以黑幫的恩怨情仇為題材，龍日夫是台灣黑幫的老大，故事講述他遭到襲擊，由義子龍一拼命相救。龍日夫有意讓龍一成為他的繼承人，卻引起其親生兒子龍二及龍三的不滿。龍三企圖殺害龍一，龍日夫命龍二殺掉龍三，龍二的心腹小林到處追殺龍一。眾人展開了一系列的矛盾與衝突，最後決定淡出江湖，做個平凡人<ref>{{Cite book|author =陳善之、翟浩然|title =善緣滿佈 - 陳善之回首前塵四十載|publisher =明窗出版社|ISBN = 9789888338689|date =2016年7月11日| url =https://books.google.com.hk/books?id=xbtgDQAAQBAJ&vq=%E9%BB%8E%E6%98%8E&hl=zh-TW&source=gbs_navlinks_s=false|language =中文|}}</ref><ref>{{Cite web |url= https://moviecool.asia/film/mGudATuGBEF|title =龍騰四海| publisher =影史庫|language = 中文}}</ref>。

==主要角色==
* [[黎明|黎明]]－小林，台灣黑幫成員，龍二的心腹，是個殺手。
* [[鄧光榮|鄧光榮]]－龍一，台灣黑幫成員，龍日夫的義子。
* [[陳法蓉|陳法蓉]]－沈雪，龍一的妻子，是個善良的護士。
* [[任達華|任達華]]－龍二，台灣黑幫成員，龍日夫的兒子。
* [[林保怡|林保怡]]－龍三，台灣黑幫成員，龍日夫的兒子。
* [[劉德華|劉德華]]－大飛，香港的黑幫成員，是個游手好閒的小混混。
* [[吳家麗|吳家麗]]－麗姐，大飛的姊姊，是個排檔的老闆娘。
