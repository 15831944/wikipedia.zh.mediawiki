'''龍舌蘭属'''（[[学名|学名]]：''{{lang|la|Agave}}''），又名龍舌掌、番麻、萬年蘭、百年草，一般直接稱為'''龍舌蘭'''，屬於[[天門冬科|天門冬科]][[龍舌蘭科|龍舌蘭亞科]]龍舌蘭屬的[[多年生|多年生]][[常綠|常綠]][[灌木|灌木]]，原產於[[北美洲|北美洲]]，其中在[[墨西哥|墨西哥]]有最多的種類，適於溫暖氣候，莖非常短，它的葉子又寬又長，是龍舌蘭最引人注意的部分；成長約十年成熟後開[[花|花]]，外觀類似[[蘆薈|蘆薈]]但兩者並不相近（兩者皆是[[天門冬目|天門冬目]]，然而後者分類為[[蘆薈科|蘆薈科]]）。

== 特征 ==
龍舌蘭有目前地表植物最長的[[花序|花序]]，目前量測最高為8公尺<ref>由底算起的花莖則高達12公尺</ref>，[[花莖|花莖]]高直從葉叢內生出，會在花莖旁長出數株直徑約6至8公分的漏斗狀淡白或淡黃色花朵，[[雄蕊|雄蕊]]6枚，[[子房|子房]]3室，[[柱頭|柱頭]]3裂；龍舌蘭單株一生中只會開一次花，花謝後母株會逐漸枯死，若授粉成功後可結成長橢圓形[[蒴果|蒴果]]；[[葉片|葉片]]長約1至1.5公尺，寬15至25公分，呈針形由短莖長出向上成放射狀，邊緣略白並帶有細銳刺，葉中的汁液含多種[[甾體皂甙|甾體皂甙]]帶有[[毒性|毒性]]，會刺激[[皮膚|皮膚]]產生灼熱感、發癢、出紅疹，甚至產生水泡，若長期誤食則會產生厭食、呆滯、四肢麻痺症狀，並造成胃黏膜充血及[[肝臟|肝臟]]傷害，嚴重可致死，對人體黏膜部分（比如[[眼睛|眼睛]]）亦會造成傷害。

== 種植 ==
龍舌蘭需要充足[[日光|日光]]及排水良好的[[砂|砂]]質[[土壤|土壤]]，適合溫度為[[攝氏|攝氏]]15度至25度，[[濕度|濕度]]約40%，最低耐受溫度約攝氏5度，生長期適當水分即可生長，日照不足溫度下降時進入[[冬眠|冬眠]]期則避免大量水分以免[[根|根]]部腐壞，對於環境適應力佳不需時常施肥，過度施肥易造成肥害；可取下側芽或利用分枝進行繁殖。雖然一株龍舌蘭只會開一次花結一次果，但在需要大量繁殖或是園藝配種時，播種法還是有人使用。

== 用途 ==
# 葉部纖維處理後可製成繩索及織布。
# 葉片堅挺不易枯萎且不懼烈日，適合戶外園藝擺設做為[[觀葉植物|觀葉植物]]。
# 可種植成排作為圍籬、海岸防風林、 定砂植物。
# 可利用其鋸齒葉、葉尖利硬及汁液毒性，做為防禦用。
# [[藍色龍舌蘭|藍色龍舌蘭]]和[[黃邊龍舌蘭|黃邊龍舌蘭]]的鱗根中包含的豐富[[多糖|多糖]]可以分解成[[果糖|果糖]]製成{{link-en|龍舌蘭糖漿|Agave nectar}}。
# 藍色龍舌蘭的鱗根提煉出來的糖漿再發酵蒸餾可製成[[龍舌蘭酒|龍舌蘭酒]]（Tequila）。
# Pulque酒：與Tequila類似但未經蒸餾，於宗教上有相關用途。
# Mezcal酒：泛指所有龍舌蘭製成的酒類，規定較為寬鬆。
# 鱗根含豐富[[澱粉|澱粉]]，可煮熟後食用。
# [[中醫|中醫]]上可用做解毒、止血、潤肺、化痰、止咳、治虛勞咳嗽、吐血、哮喘。
# 臨床實驗嘗試運用龍舌蘭口服液治療慢性支氣管炎。
