'''[[龍運巴士|龍運巴士]]A41P線'''是[[香港|香港]]一條來往[[烏溪沙站|烏溪沙站]]及[[香港國際機場|機場（地面運輸中心）]]的巴士路線。

'''龍運巴士NA40線'''則是香港一條機場巴士路線，為A41P線的通宵版本，行車路線與A41P線大致相同，惟來回程繞經[[源禾路|源禾路]]、[[沙田市中心|沙田市中心]]、[[國泰城|國泰城]]。

==歷史==
*1999年6月30日：A41P線投入服務，班次每45-60分鐘一班
*2001年9月29日：A41P線繞經[[亞公角街|亞公角街]]、[[富安花園|富安花園]]及[[錦泰苑|錦泰苑]]，不再經[[源禾路|源禾路]]及[[沙田市中心|沙田市中心]]
*2002年3月16日：A41P線改經[[青衣北岸公路|青衣北岸公路]]
*2007年12月22日：A41P線往機場（地面運輸中心）方向，繞經[[香港國際機場二號客運大樓|二號客運大樓（翔天廊）]]
*2009年7月5日：A41P線班次加密至每30分鐘一班
*2013年8月25日：A41P線由馬鞍山（耀安）延長至烏溪沙，並更改[[馬鞍山_(香港市鎮)|馬鞍山]]區內之行車路線。同日起，削減大部分班次至每45分鐘一班<ref>[http://www.lwb.hk/lib/A41P.pdf 龍運巴士A41P線改以烏溪沙鐵路站為總站]，lwb.com.hk，2013年8月21日</ref><ref>[http://www.lwb.hk/lib/A41P%20pax%20notice.pdf 龍運巴士A41P線改以烏溪沙鐵路站為總站]，lwb.com.hk，2013年8月21日</ref>
*2014年11月15日，A41P線加強班次至20-35分鐘一班。<ref> {{cite web |url=http://lwb.hk/tc/news/press/archives/news2014111255.html |title=存档副本 |accessdate=2014-11-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20141116135459/http://www.lwb.hk/tc/news/press/archives/news2014111255.html |archivedate=2014-11-16 }}</ref>
*2016年12月17日，NA40線投入服務；行車路線與A41P線大致相同，惟來回程繞經[[源禾路|源禾路]]、[[沙田市中心|沙田市中心]]、[[國泰城|國泰城]]，方便在機場工作的馬鞍山及沙田市中心居民。
*2017年9月1日，NA40線加開一班車，於00:20由機場開出。
*2018年10月24日：NA40線總站延長至[[港珠澳大橋|港珠澳大橋]]香港口岸，往港珠澳大橋方向班次增至三班，開出時間分別為03:40、04:10及04:40。<ref>[http://search.kmb.hk/KMBWebSite/AnnouncementPicture.ashx?url=1539962610_9858_0.pdf 深宵 “NA” 線直達港珠澳大橋香港口岸]</ref>

===八達通轉乘優惠===
乘客登上本線後指定時間內以同一張八達通卡轉乘以下路線，或從以下路線登車後指定時間分鐘內以同一張八達通卡轉乘本線，次程可獲車資折扣優惠：

;A41P←→九巴獨營路線
*由任何一條九巴獨營路線（不論方向，聯營路線、假日旅遊路線、非常規特別路線、馬場路線及港鐵接駁巴士除外）於上車後兩小時內轉乘任何一條龍運A線往機場，或由往新界方向的任何一條龍運A線轉乘任何一條九巴獨營路線（不論方向），第二程可獲 $6.00 轉乘優惠或減免接駁龍運A線的一程九巴車資，以較低者為準。

;A41P←→龍運巴士E線
*'''A41P往機場''' → E31、E33/E33P、E34A/E34B/E34P或E42往東涌／機場，次程車資全免，於青馬收費廣場轉乘，轉乘時限為150分鐘
*E31、E33/E33P、E34A/E34B/E34P或E42往市區 → '''A41P往烏溪沙站'''，回贈首程車資，於青馬收費廣場轉乘，轉乘時限為90分鐘
*'''A41P往機場''' → E32或E41往機場博覽館，次程車資全免，於青馬收費廣場或機場（一號停車場）轉乘，轉乘時限為150分鐘
*E32或E41往往市區 → '''A41P往烏溪沙站'''，回贈首程車資，於機場（地面運輸中心）或青馬收費廣場轉乘，轉乘時限為90分鐘

;A41P←→R8，於青馬收費廣場轉乘，次程可獲$3折扣優惠
*'''A41P任何方向''' → R8往迪士尼樂園，轉乘時限為120分鐘
*R8往青馬收費廣場 → '''A41P任何方向'''，轉乘時限為60分鐘

;NA40←→九巴N73、N271
*'''NA40往烏溪沙鐵路站'''→ 九巴N73或N271往大埔／落馬洲，次程車資全免，於沙田市中心轉乘，轉乘時限為150分鐘
*九巴N73或N271往沙田／九龍 → '''NA40往機場'''，回贈首程車資，於沙田市中心轉乘，轉乘時限為150分鐘
