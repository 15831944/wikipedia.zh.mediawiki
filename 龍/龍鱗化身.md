{{Vgname|龍鱗化身|en=Scalebound}}是一部由[[白金工作室|白金工作室]]开发、[[微软工作室|微软工作室]]发行的动作冒险游戏。游戏运行于[[Microsoft_Windows|Microsoft Windows]]以及[[Xbox_One|Xbox One]]平台。遊戲首次於2014年E3展上發表，但在2017年，[[微軟|微軟]]正式宣布遊戲開發終止<ref name="Scaleboundcanceled">{{cite web|title=Microsoft confirms: Xbox One exclusive Scalebound has been canceled|url=https://www.windowscentral.com/xbox-exclusive-scalebound-has-been-canceled|date=2017-01-09|language=en|accessdate=2018-06-22}}</ref>。

== 遊戲玩法 ==
遊戲中玩家將操作能駕馭強大飛龍的主角，一人一龍聯手對抗敵對勢力。主角可以使用刀劍與弓箭等遠近距離的武器來克敵，飛龍則是能以灼熱的火焰與巨大的軀體掃蕩敵人。當面對威力巨大的巨獸時，主角可以運用來自飛龍的力量變身為更強大的型態，牽制巨獸的行動，讓飛龍以威力強大的烈焰摧毀巨獸<ref name="GC15">{{cite web|title=【GC 15】《龍鱗化身》首度曝光實機遊玩示範 人龍一體對抗邪惡勢|url=https://gnn.gamer.com.tw/8/118878.html|publisher=巴哈姆特 |date=2015-08-05|language=zh|accessdate=2018-06-22}}</ref>。

== 開發 ==
微軟宣布遊戲將推出中文化<ref name="RennieAddabbo">{{cite web|title=【E3 15】微軟亞太行銷總經理透露新作《ReCore》《黑暗靈魂 3》等將中文化|url=https://gnn.gamer.com.tw/9/116549.html|publisher=巴哈姆特 |date=2015-06-17|language=zh|accessdate=2018-06-22}}</ref>。

== 漫畫 ==
