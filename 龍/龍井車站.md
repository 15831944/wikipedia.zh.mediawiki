'''龍井車站'''位於[[臺灣|臺灣]][[臺中市|臺中市]][[龍井區|龍井區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[海岸線_(臺鐵)|海岸線]]的[[鐵路車站|鐵路車站]]。

本站的站名源自當地地名，而「龍井」這地名又起源於當地的古蹟[[龍目井|龍目井]]，在1920年時取龍目井的頭尾兩字將原本的地名「茄苳-{庄}-」改為龍井，因此在同年底設置的本站也隨之命名為龍井車站{{R|Official}}。

== 車站構造 ==
龍井車站內配置有一座[[島式月台|島式月台]]，在過去由於月台與站房之間並無設置任何的行人立體穿越設施，因此欲進出月台需在站務人員的指揮下直接跨越鐵軌。2010年時配合車站站房與站前廣場的整修翻新，在月台與車站入口之間設立一座[[人行天橋|人行天橋]]，而改善了旅客的搭乘安全。

== 營運狀況 ==
龍井車站是臺鐵下轄的[[臺灣鐵路管理局車站等級#三等站|三等站]]之一，但與一般海線三等站多會停靠較高等級列車的狀況不同，在客運業務上龍井車站是以停靠[[臺鐵區間車|區間車]]為主，整體的業務量並不高。
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]，僅停靠[[臺鐵區間車|區間車]]。
過去，在龍井車站內設有一條往車站南側、配合[[台中發電廠|台中火力發電廠]]而設置的龍井儲煤場之貨運側線，每日皆有數班以運煤[[斗車|斗車]]為主的貨運列車班次在本站進出，[[煤|燃煤]]運量達每日平均38,000噸，甚至負擔整個台中運務段大部分的貨運收益，而成為臺鐵旗下排名第一的貨運車站。自龍井運出的燃煤主要還供應台灣北部[[林口發電廠|林口]](2012年12月底起鐵路停運)、[[深澳發電廠|深澳]](2007年停運)等[[火力發電廠|火力發電廠]]使用{{R|Official}}。

*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[icash|icash 2.0]]及[[HappyCash|HappyCash]]付費。

== 歷史 ==
*1920年12月15日：設站。是配合海線大甲溪南側路段通車而一同設置的新車站之一{{R|Official}}。
*1973年8月15日：因客運業務冷清，降等為簡易站，由[[沙鹿車站|沙鹿車站]]負責管理{{R|Official}}。
*1981年5月1日：配合龍井儲煤場的設置，車站升等，恢復成三等站{{R|Official}}。
*2014年7月7日：一列北上[[基隆車站|基隆]]的第112車次[[自強號列車|自強號]]，在行經龍井車站南側時不慎撞上正在軌道維修的道班工與台鐵職員，當場造成兩人殉職，列車停擺{{R|Incident_20140707}}。
*2015年6月30日：啟用多卡通刷卡機。
