'''龍志權'''（原名 王雍泰 Nicky Wong，{{bd|1992年|7月12日}}）香港男藝人。現為[[毛記電視|毛記電視]]「偽員」<ref>{{cite web |url=http://www4.tvmost.com.hk/201611032256_artist_dragon |title=龍志權 - 毛記電視 }}</ref>，其藝名惡搞自韓國組合[[BIGBANG|BIGBANG]]成員[[權志龍|權志龍]]。 <ref>{{cite news|title=娛樂人物：龍志權自我感覺唔係樣衰就得 |url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20180924/20506514 |newspaper=Apple Daily 蘋果日報 |date=2018-9-24 }} </ref>。

== 簡介 ==
龍志權中學就讀[[喇沙書院|喇沙書院]]，及後於[[香港專業教育學院|香港專業教育學院]]主修工商管理<ref>{{Cite web|url=https://hk.entertainment.appledaily.com/enews/realtime/article/20180923/58712843|title=【毛記主播入十強】被網民指唔樣衰但有X味《全民造星》龍志權：已經係讚美|date=2018-9-23|work=Apple Daily 蘋果日報}}</ref>，畢業後曾於網上媒體 GagMode工作。

龍志權於2016年曾製作二次創作短片《地鐵超人》，自行設計對白及配音，並改編廣東歌《塞車》歌詞，自唱主題曲，以Youtube帳戶名稱Nickyky Wong (王力奇)上載YouTube，獲得約24萬次點擊<ref>{{cite AV media |people= |date=2016-3-29 |year= |title=《地鐵超人》Youtube |url=https://www.youtube.com/watch?v=Cw9MIqddfJo}}</ref>，短片於 Facebook 的播放量更達82萬次<ref>{{cite web |title=《地鐵超人》Facebook |url=https://www.facebook.com/nickykywong/videos/1104483486239064/}}</ref> 。

2016 年加入《100毛》，擔任創作助理及「偽員」，取藝名「龍志權」。2016年5月5日，龍志權首次於「[[毛記電視|毛記電視]]」《勁曲金曲挑戰站》中亮相<ref>{{cite web |title=Pricerite x 100毛【林雪@勁曲金曲挑戰站】 |url=https://www.youtube.com/watch?v=LY4nis2rON0 |publisher=100毛 |accessdate=2016-5-6}}</ref>，及後多次在《[[六點半左右新聞報道|六點半左右新聞報道]]》、《[[星期三港案|星期三港案]]》等節目中擔任主播、客串演員及旁述。由於經常在節目中模仿韓國人以廣東話口音作自我介紹，並以韓文「莎啦嘿喲」加上韓式心心手勢作為招牌動作，使其「韓流主播」形象為網民所熟悉。

2018年5月，龍志權參加了ViuTV 選秀節目《[[Good_Night_Show_全民造星|Good Night Show 全民造星]]》，參賽編號11號。首輪比賽憑藉在由ViuTV官方facebook主辦、網民公開投票的「《全民造星》Eye Catching Idol選舉」中，取得第二高票數，直入50強<ref>{{cite news |title=【《全民造星》變造票？】參賽者被質疑買票入50強 當事人解畫：我有好多朋友 |url=https://hk.entertainment.appledaily.com/enews/realtime/article/20180723/58473934 |work=Apple Daily 蘋果日報 |date=2018-7-23}}</ref>。

於參賽過程中，龍志權以加入電台DJ獨白方式演唱《戀愛預告》、一人分飾男女兩角演唱K房版《會過去的》（YouTube點擊達13萬次<ref>{{cite web |title=《Goodnight Show全民造星》龍仔勁呀！男女聲都唱到？ 龍志權《會過去的》K房版！ |url=https://www.youtube.com/watch?v=nx4PmKefOac#t=01m11s |publisher=ViuTv}}</ref>）以及改編歌曲《廿六歲》<ref>{{cite web |title=【十強粉絲見面會】龍志權 － 《17歲》 |url=https://www.youtube.com/watch?v=bymWljl5p6Q |publisher=ViuTv}}</ref>等表演，展示創作與表演才華，最後成功入圍十強<ref>{{cite news |title=《全民造星》10強誕生 投票一起選出最後冠軍 |url=https://news.now.com/home/entertainment/player?newsId=320587&refer= |work=NOW新聞 |date=2018-9-17}}</ref>。

在總決賽中，除了於第一輪表演中演唱《你是我的傳奇》，更在第二輪以獨白形式表演「講嘢」—— 讀出一封給fans阿嵐的信<ref>{{cite web |title=龍志權 全民造星決賽表演講嘢 |url=https://www.youtube.com/watch?v=VeapD9-Mxj4}}</ref>，讓人印象難忘。

龍志權於2018年11月參與演出毛台劇《我殺死了尊貴客戶》，在劇中飾演「肥龍」一角，演技得到著名導演及演員[[杜汶澤|杜汶澤]]的肯定<ref>{{cite web |title=杜汶澤Facebook貼文 |url=https://www.facebook.com/photo.php?fbid=1411674138966496&set=a.111122362355020&type=3&theater |quote=你唔使驚人哋笑你唔識做戲，因為我覺得你識}}</ref>。
