'''龍姓'''在新百家姓中排列第81。

== 來源 ==
龍姓的初源，[[歷史|歷史]]亦有記載，可追溯的源頭有下列數項：
# 出自[[黃帝|黃帝]]臣子「龍行」。據《[[姓氏錄源|姓氏錄源]]》及《[[竹書紀年|竹書紀年]]》所載，黃帝臣有龍行，黃帝居有熊（今[[河南|河南]][[新鄭|新鄭]]）。是為[[新鄭|新鄭]]龍氏。
# 出自御龍氏之後《[[姓氏考略|姓氏考略]]》所載，龍姓出自[[御龍氏|御龍氏]]，望出[[武陵|武陵]]、[[天水|天水]]。如[[夏朝|夏朝]]御龍氏[[劉累|劉累]]（堯之後，因有馴化龍的本領，被夏帝孔甲賜為御龍氏）的後裔中，就有以龍為氏的。劉累的故城在今河南偃師縣南，是為河南龍氏。
# 出自[[豢龍氏|豢龍氏]]。據《[[通志氏族略|通志氏族略]]》及《[[名賢氏族言行類稿|名賢氏族言行類稿]]》所載，相傳董父，已姓，以畜養龍而被舜賜姓豢龍氏。其後有以龍為氏的，是為[[湖北|湖北]]龍氏，[[項羽|項羽]]四大名將之一就叫龍且。
# [[舜帝|舜帝]]時的[[諫官|諫官]]，賜姓龍。
# [[粵|粵]]地姓氏，相傳悅城龍母拾珠產子，冠以龍姓。
# 據《[[華陽國志|華陽國志]]》所載，[[西漢|西漢]]時的牂牁大姓中有龍氏。[[牂牁郡|牂牁郡]]，治今[[貴州省|貴州省]][[凱裏縣|凱裏縣]]西北。
# [[東漢|東漢]]時[[西域|西域]]強國[[焉耆|焉耆]]及[[且彌|且彌]]的國王姓龍。唐朝时[[焉耆国|焉耆国]]王姓龙，名突骑支，常役于[[西突厥|西突厥]]。俗有鱼鳖之利。[[贞观|贞观]]十八年，[[郭孝恪|郭孝恪]]平之，由是臣属[[唐朝|唐朝]]。
# [[滇|滇]]地姓氏，[[苗族|苗]]及[[彝|彝]]人多以龍為其姓。
# 古代有[[勾龍|勾龍]]氏，後人改姓龍。如[[宋代|宋代]]有[[勾龍爽|勾龍爽]]、[[勾龍如淵|勾龍如淵]]等。

== 歷史名人 ==
* [[龍叔|龍叔]]：又名龍子，孟子曾因其賢而推薦給滕文公。
* [[龍且|龍且]]：與鍾離昧、英布、彭越合稱項羽四大名將，在山東被韓信所殺。
* [[龍會|龍會]]：焉耆國王，消滅龜茲，統一蔥嶺以西，其子龍熙後來投降西涼呂光。
* [[龍汝元|龍汝元]]（？-1859年）：清直隸宛平（一作大興，今均屬北京市）人，字春舫。行伍出身，曾從[[河南|河南]]巡撫英桂鎮壓太平軍，歷任遊擊、參將。咸豐八年（1858年）升大沽協副將，助僧格林沁加強天津海防。次年英法聯軍挑起第二次大沽之戰，他堅守北岸前炮臺，手燃巨炮，重創敵艦，卒中炮陣亡。
* [[龍啟瑞|龍啟瑞]]（1814-1858）：清廣西臨桂（今桂林）人，字輯五，號翰臣，道光狀元，授翰林院修撰。太平天國起義後，在籍開辦團練。後歷任江西學政、江西布政使，著有《經德堂詩文集》、《小學高注補正》等。
* [[龍澤厚|龍澤厚]]：清末廣西臨桂人，字積之，光緒優貢。以知縣引見，在廣州從康有為學，為萬木草堂學長。曾參加桂林聖學會、上海強學會、上海不纏會、上海中國國會和自立軍起義。光緒二十九年（1903），因為參加 《蘇報》工作，被清政府勾結上海公共租界巡捕房逮捕。曾主持三藩市《文興報》。後為上海天遊學院教務長，編有《南海先生上書記》。
* [[龍湛霖|龍湛霖]]（1837年－1905年），字芝生，湖南攸縣（今湖南省攸縣）人，清朝政治人物、進士出身, 光緒十四年，任江西學政, 光緒十七年，任詹事、內閣學士。光緒十九年，任福建鄉試正考官、刑部右侍郎,
* [[龍榆生|龍榆生]]（1902－1966）：名沐勋，字榆生，中国20世纪词学名家。
* [[龍濟光|龍濟光]]：中華民國廣東軍閥。
* [[龍雲|龍雲]]：中華民國雲南軍閥，有「雲南王」之稱。
* [[龍成邦|龍成邦]]：潛逃台灣的香港退休貪污探長。
