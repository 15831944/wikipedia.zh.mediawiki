'''[[龍運巴士|龍運巴士]]A37線'''是[[香港|香港]]一條機場特快巴士路線，來往[[洪水橋|洪水橋]]（[[洪元路|洪元路]]）及[[香港國際機場|機場（地面運輸中心）]]。

'''龍運巴士NA37線'''是香港一條由天水圍市中心單向往港珠澳大橋香港口岸的巴士路線，途經路線大致上和NA34線相同，惟改經[[屏欣苑|屏欣苑]]及[[洪水橋|洪水橋]]不經元朗市。

==歷史==
因應洪水橋[[洪福邨|洪福邨]]入伙，[[龍運巴士A36線|龍運巴士A36線]]在2015年開辦，為洪水橋及元朗居民於[[繁忙時間|繁忙時間]]提供特快往返機場服務，同時疏導[[龍運巴士E34B線|E34B線]]的龐大客量。此外，由於天水圍往返機場的需求日益增加，而[[龍運巴士E34A線|E34A]]線需繞經東涌及機場後勤區，令車程頗為費時。因此運輸署和龍運巴士於《2016-2017年度巴士路線計劃》建議開辦本線，全日特快來往洪水橋及機場並[[繞經|繞經]]天水圍。該計劃建議本線於開辦初期每小時開一班，並視乎客量變化再加密至半小時一班；同時建議A36線提升至全日雙向服務，但縮短至朗屏站及不經[[屏山|屏山]]，改為服務[[朗屏邨|朗屏邨]]及[[元朗市中心|元朗市中心]]一帶乘客。<ref>[http://www.districtcouncils.gov.hk/yl/doc/2016_2019/tc/working_groups_doc/WGBS/10806/wgbs_2016_003.pdf 《2016-2017年度元朗區巴士路線計劃》]</ref>

龍運巴士和運輸署在2016年6月提交修訂方案，建議於開線時實施半小時一班之班次<ref>[http://www.hkitalk.net/HKiTalk2/forum.php?mod=viewthread&tid=971292 2016-17 元朗區路線發展計劃：結果]，hkitalk.net</ref>，隨後再宣佈本線於同年7月25日正式投入服務<ref name="LWB">[http://www.lwb.hk/chi/pdf/2_20160725%20A37_promo.pdf 龍運巴士開辦A37宣傳單張]，2016年7月22日。</ref><ref name="LWB1">龍運巴士有限公司，〈[http://www.lwb.hk/tc/news/press/archives/news20160722139.html 龍運開辦新機場巴士路線A37]〉［新聞稿］，2016年7月22日。</ref><ref name="TD">香港特別行政區政府運輸署，〈[http://www.td.gov.hk/filemanager/tc/content_13/20160722%20a37%20introduction%20notification%20(chi)..pdf 開辦龍運巴士第A37號線]〉［交通通告］，2016年7月22日。</ref>，不過原計劃與本線綑綁實施的A36之改動則延至同年8月4日才實施。換言之，本線開辦後有十天與A36共用相同的起訖點。

*2016年7月25日：本線投入服務<ref name="LWB"/><ref name="LWB1"/><ref name="TD"/>。
*2018年3月12日：往洪水橋（洪元路）方向增停[[洪天路|洪天路]]「[[石埗村|石埗村]]」分站。

===八達通轉乘優惠===
乘客登上本線後指定時間內以同一張八達通卡轉乘以下路線，或從以下路線登車後指定時間內以同一張八達通卡轉乘本線，次程可獲車資折扣優惠：

;A37←→九巴獨營路線
*由任何一條九巴獨營路線（不論方向，聯營路線、假日旅遊路線、非常規特別路線、馬場路線及港鐵接駁巴士除外）於上車後兩小時內轉乘任何一條龍運A線往機場，或由往新界方向的任何一條龍運A線轉乘任何一條九巴獨營路線（不論方向），第二程可獲 $6.00 轉乘優惠或減免接駁龍運A線的一程九巴車資，以較低者為準。

;A37←→龍運E線
*'''A37往機場''' → E31、E33/E33P、E34A/E34B/E34P/E34X或E42往機場，次程車資全免，於青馬收費廣場轉乘，轉乘時限為150分鐘
*E31、E33/E33P、E34A/E34B/E34P或E42往市區 → '''A37往洪水橋'''，回贈首程車資，於青馬收費廣場轉乘，轉乘時限為90分鐘
*'''A37往機場''' → E32或E41往機場博覽館，次程車資全免，於青馬收費廣場或機場轉乘，轉乘時限為150分鐘
*E32往葵芳站或E41往大埔頭 → '''A37往洪水橋'''，回贈首程車資，於機場（地面運輸中心）或青馬收費廣場轉乘，轉乘時限為90分鐘

;A37←→R8，次程可獲$3折扣優惠，於青馬收費廣場轉乘
*'''A37任何方向''' → R8往迪士尼樂園，轉乘時限為120分鐘
*R8往青馬收費廣場 → '''A37任何方向'''，轉乘時限為60分鐘

==使用狀況==
由於本線能為[[洪水橋|洪水橋]]一帶的居民提供直接來往機場的服務，而且在[[天水圍市中心|天水圍市中心]]一帶相對能提供較[[龍運巴士E34A線|E34A線]]快大約30-40分鐘的車程，加上E34A線在繁忙時間經常出現頂閘，因此吸引了不少[[洪水橋|洪水橋]]及[[天水圍市中心|天水圍市中心]]的居民使用此線往返機場上班或外遊，繁忙時間的客量仍不俗。惟在非繁忙時間，由於此線的收費較E34A線昂貴及後者在該時段仍有剩餘的載客空間，因此客量僅能維持在偏低水平，部分班次或會出現吉車遊街。

為了與輕鐵751線競爭，本線設有天慈邨往洪水橋$7.7單向分段收費，因此本線往洪水橋方向在天慈後登車的乘客比由機場出發的全費乘客多，這是在往來[[青嶼幹線|青嶼幹線]]的巴士路線中少見的（另2條為城巴A12線及E21A線）。

== 未來發展 ==
為配合[[洪水橋新市鎮|洪水橋新市鎮]]發展，[[天影路|天影路]]P1路段（屏廈路至天華路一段）將會拆除，並由[[屏廈路|屏廈路]]取代原有功能。屆時本線將會改道行駛。

由於{{LWB|E34A}}線在天水圍南一帶經常出現爆滿的情況，因此運輸署於2017年7月向區議會遞交文件，建議此路線改經[[天湖路|天湖路]]及[[天耀路|天耀路]]，藉此紓緩E34A線於繁忙時間的壓力及為天水圍南一帶的居民提供更快捷的服務往返機場<ref>[http://www.districtcouncils.gov.hk/yl/doc/2016_2019/tc/committee_meetings_doc/ttc/12740/ttc_2017_063.pdf 龍運巴士服務建議 — 更改第A37號線之行車路線]，元朗區議會交通及運輸委員會文件第63/2017號。</ref>，有關計劃最終因區議會反對而擱置。<ref>[http://www.inmediahk.net/node/1052247 發佈調查報告批評運輸署「亂搞」A37巴士線]，香港獨立媒體網，2017年9月22日</ref><ref>[http://www.districtcouncils.gov.hk/yl/doc/2016_2019/tc/working_groups_doc/WGBS/12129/wgbs_2017_044.pdf 陳美蓮議員動議，杜嘉倫議員和議，「本會強烈要求A37巴士保留途經天慈邨，確保原有慈祐居民不受影響。」]</ref>

而[[工聯會|工聯會]]則要求本路線加經天水圍北。<ref>[http://www.districtcouncils.gov.hk/yl/doc/2016_2019/tc/working_groups_doc/WGBS/12129/wgbs_2017_043.pdf 李月民議員及鄧焯謙議員動議，姚國威議員、陸頌雄議員、劉桂容議員及郭慶平議員和議，「本工作小組強烈要求增加A37巴士班次及修改A37巴士路線，由洪水橋出發沿天影路到天恒邨後轉入天瑞路，到頌富商場後再轉入天華路行經天水圍市中心後轉回天華路，再直往多一個路口至天葵路、天龍路，再返回天城路必經天慈邨直出機場，額外令天水圍北、美湖居居民及剛開張的悅品天秀酒店旅客受惠。」]</ref>
