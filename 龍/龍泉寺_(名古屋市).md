'''龍泉寺'''（日語：りゅうせんじ），是位於日本[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[守山區|守山區]]竜泉寺的[[天台宗|天台宗]][[佛寺|寺院]]。山號為「'''松洞山大行院'''」，另外也記為「'''竜泉寺'''」。是鎮守[[名古屋城|名古屋城]]的[[尾張四觀音|尾張四觀音]]之一；屬於[[尾张三十三观音灵场|尾張三十三觀音]]靈場第二十五號禮所。祭祀的本尊為[[馬頭明王|馬頭觀音]]。

== 歷史 ==
根據〈沙石集〉，尾張龍泉寺是由[[龙王|龍王]]一夜間所建立的寺院，據說可看到[[馬頭明王|馬頭観音]]現出池水的遺跡。把此當為原型的〈龍泉寺記〉（[[寶曆_(日本)|寶曆]]5年、1755年記），在[[延曆|延曆]]年間傳教大師[[最澄|最澄]]在[[熱田神宮|熱田神宮]]參籠中得到龍神啟示，在羅池畔唱誦經文的話，從池中會有龍升天，同時會有馬頭觀音出現，以此被認為是本尊祭祀而開基<ref>{{cite book |title=大日本地名辭書, 第 2 卷 |date=1907 |publisher=冨山房 |page=1288}}</ref>。

另一方面，弘法大師[[空海|空海]]也將熱田神宮參籠中的熱田八劍中之三劍，埋納在此龍泉寺，因此龍泉寺成為熱田神功的奥之院<ref>{{cite book |author1=大島建彦 |title=日本の神仏の辞典 |date=2001 |publisher=大修館書店 |page=1364}}</ref>。

龍泉寺是位於能眺望庄内川的山上，是由庄内川、崖、濕地所圍繞且能一眺[[濃尾平原|濃尾平野]]的要害之地，自古以來在戰爭時被認為有軍事價值而常在此設陣。一般認為在[[弘治_(日本年號)|弘治]]2年（[[1556年|1556年]]）[[織田信行|織田信行]]築城，是[[永祿|永祿]]3年（[[1560年|1560年]]）[[桶狹間之戰|桶狹間之戰]]前，為在[[清洲城|清須城]]東北方建立防衛線，由[[織田信長|織田信長]]派往龍泉寺的一員<ref>{{cite book |editor=愛知県姓氏家系大辞典編纂委員会 |title=愛知県姓氏歴史人物大辞典 |date=1991 |publisher=角川書店 |page=1056}}</ref>。[[天正_(日本)|天正]]12年（[[1584年|1584年]]）[[小牧、长久手之战|小牧・長久手之戰]]之際時，為討伐進入小幡城的[[德川家康|德川家康]]，[[丰臣秀吉|羽柴秀吉]]在此布陣（不過之後誕生了「一夜堀」的傳說）、秀吉在退兵時放火將所有建築物都燒毀了<ref>{{cite book |author1=吉永昭 |title=愛知: 史蹟鄉土史 |date=1982 |publisher=講談社 |page=66}}</ref>。在[[慶長|慶長]]3年（[[1598年|1598年]]），由鄰近寺院密藏院[[僧伽|僧人]]秀純再興，[[明治|明治]]39年（[[1906年|1906年]]）2月再度遭遇火災，除了[[多寶塔|多寶塔]]、[[金剛力士|仁王門]]、鐘樓外，全部燒毀。從火災遺跡中，挖掘出藏有慶長大判切2枚、慶長小判98枚的小判容器，有以此為基金來重建本堂的小故事<ref>{{cite book |title=日本地理風俗大系〈第5巻〉中部地方 |date=1959 |publisher=誠文堂新光社 |page=158}}</ref>。

多寶塔是在明治28年（[[1895年|1895年]]）所建<ref>[http://www.geocities.jp/kawai5510/aiti-ryuusenji.htm 愛知県の塔（龍泉寺）]</ref>。

== 龍泉寺城 ==
在上述有關尾張的歷史中，龍泉寺好幾次被作為[[军事基地|軍事施設]]所使用，因此也被認為是「龍泉寺城」。現在境内周圍一部分殘有防空壕狀的痕跡，在境内內側[[日本庭園|日本庭園]]旁邊建造起三層[[天守|模擬天守]]，藏有[[重要文化財|重要文化財]]木造[[地藏菩萨|地藏菩薩]]像和[[圆空|円空]]佛等等，兼做為寶物館（每週日開館）。
