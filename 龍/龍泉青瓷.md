'''龍泉青瓷'''是以[[中國|中國]][[浙江省|浙江省]][[龍泉市|龍泉市]]古代窯址為核心、分支廣佈週邊、影響遍及[[華東|華東]]、[[中南|中南]]及[[西南|西南]]的瓷窯系統。一般認為龍泉窯的鼎盛時期在[[南宋|南宋]]。由於融合南北技術優點，提高燒製溫度，成品瓷化程度高，在技術層面達到了現代學術認定的瓷器要求；同時改進釉料配方，創制出[[粉青|粉青]]、[[梅子青|梅子青]]等前所未見的品種，被歷來藏家激賞，列入宋代名窯。此後龍泉窯幾經興衰，不絕如縷，延續至今。其產品以[[青瓷|青瓷]]為主，歷代依據當時的資源條件和市場需求，發揮青瓷多變的特性，衍生出眾多品種，相互之間差別很大，是中國持續時間最長、影響最大的青瓷產地。現代龍泉青瓷主流為厚釉、無開片、翠青色產品，胎體澄淨，紫骨鐵足，包括日常器皿和仿古陳設器。

== 历史 ==

[[南宋|南宋]]初[[处州府|处州府]][[龙泉县|龙泉县]]琉田市建立的瓷窑，其中[[章生一|章生一]]在龙泉琉田创建的瓷窑叫[[哥窑|哥窑]]，这种胎质细密洁白，釉色以粉青、梅子青为代表，花纹装饰上出现了堆塑和贴花；一般称龙泉窑皆指此。章生一的弟弟[[章生二|章生二]]在龙泉琉田创建的瓷窑叫'''[[弟窑|弟窑]]'''，也称为'''章龙泉窑'''、'''章窑'''。

[[元代|元代]]龍泉青瓷大量运销海外，瓷窑范围扩大至丽水、永嘉等地；[[明代|明代]]仍然继续烧造，[[清代|清代]]中期以后衰落。

龙泉窑古瓷窑1960年曾加发掘，现已发现300多个窑址，依龙泉山而建，其中以大窑地区最多，[[大窑龙泉窑古瓷窑址|大窑龙泉窑古瓷窑址]]于1988年列入第三批[[全国重点文物保护单位|全国重点文物保护单位]]。2001年3月，龙泉窑遗址被评选为[[中国20世纪100项考古大发现|20世纪中国百大考古重大发现]]。

2009年9月，龙泉青瓷传统烧制技艺入选[[人类非物质文化遗产代表作名录|人类非物质文化遗产代表作名录]]。

2012年9月，龙泉窑遗址与[[上林湖越窑遗址|越窑遗址]]一同作为青瓷窑遗址列入由国家文物局公布的《中国世界文化遗产预备名单》（共45项）<ref>{{cite web |url=http://www.sach.gov.cn/tabid/312/InfoID/36881/Default.aspx |language= |title=关于印发更新的《中国世界文化遗产预备名单》的通知 |publisher=国家文物局 |date=2012-11-17 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20130531051409/http://www.sach.gov.cn/tabid/312/InfoID/36881/Default.aspx |archivedate=2013-05-31 }}</ref>。

2013年5月，[[庆元县|庆元县]]的[[上垟窑址|上垟窑址]]做为[[第七批全国重点文物保护单位中的古遗址|第七批全国重点文物保护单位中的古遗址]]并入[[大窑龙泉窑古瓷窑址|大窑龙泉窑古瓷窑址]]。6月9日，浙江省文物局公布其为第一批[[浙江省省级考古遗址公园|省级考古遗址公园]]<ref>{{cite web|url=http://zfxxgk.zj.gov.cn/xxgk/jcms_files/jcms1/web36/site/art/2013/6/13/art_2427_701080.html | language= | title=浙江省文物局关于第一批省级考古遗址公园名单的公示 |publisher=浙江省文物局 |date=2013年6月9日 |accessdate= }}</ref>。

== 工藝特點 ==
通過對於龍泉窯瓷器的胎釉的化學組成、釉層厚度、燒成溫度、氣氛性質以及顯微結構等檢測與研究，得到結論如下<ref name="中國陶瓷">馮先銘:《中國陶瓷》上海古籍出版社：2001, ISBN 7-5325-3001-9, 第413頁</ref>
# 製作胎體的瓷土有黑白兩種：黑胎用紫金土，一般含鐵3～5％，最高可以達到15％，用於燒製釉色較深的器物；白胎用經過粉碎的瓷石，與景德鎮的瓷土構成成份類似。
# 無論黑胎還是白胎，都採用相同的燒製工藝，在同一窯中同時燒成。
# 釉料可以分成[[石灰釉|石灰釉]]和[[石灰鹼釉|石灰鹼釉]]兩種。[[北宋|北宋]]用石灰釉，燒製時容易流釉，所以不能外罩厚釉；[[南宋|南宋]]用石灰鹼釉，高溫下比較黏稠，因此可以多次上釉，釉層很厚，燒成品玻璃化的程度很高。粉青釉即為石灰鹼釉，上佳者如同青玉；梅子青釉燒成溫度和玻化程度更高。
# 青釉本身要求胎體不能過於潔白，通常會在瓷土中摻入紫金土來加深素胎的顏色。
# 燒成溫度在1180℃～1230℃之間。梅子青要求溫度最高，在1250℃～1280℃。
