'''龍崎區'''位於[[中華民國|中華民國]][[臺南市|臺南市]]東南端，北臨[[新化區|新化區]]，東臨[[左鎮區|左鎮區]]、[[高雄市|高雄市]][[內門區|內門區]]，西鄰[[關廟區|關廟區]]，南接[[高雄市|高雄市]][[田寮區|田寮區]]。因地形土壤關係，且距臺南市中心不遠，人口多已外移，目前為臺南市人口最少的行政區。

== 地理 ==
本區屬[[阿里山山脈|阿里山山脈]]尾稜，為海拔約80~350公尺左右的[[丘陵|丘陵]]地，地勢東陡西緩，由於地質特殊加上河流侵蝕沖刷，形成不少的惡地地形，氣候上屬[[熱帶季風氣候|熱帶季風氣候]]。產業則因全區有九成地區屬山坡地保育區，開發嚴峻，故仍以[[農業|農業]]為主、[[製造業|製造業]]僅有3家廠商，亦有人口外流及老化問題，

龍崎區公所位於崎頂里，其山下即為關廟區，距臺南市中心亦僅10公里餘，許多居民移居以求較好之生活品質。近來則有牛埔泥岩水土保持教學園區、308高地等新興景點吸引遊客探訪。

== 歷史 ==
龍崎區舊稱「番社」，係因早期[[平埔族|平埔族]][[西拉雅族|西拉雅族]][[新港社|新港社]]人遷居本區大坪、牛埔一帶而得名。[[臺灣清治時期|清領時期]]屬[[臺灣縣_(1684年-1887年)|臺灣縣]]新豐里，[[道光|道光]]年間改隸「內新豐里」。[[台灣日治時期|日治時期]]前期'''內新豐里'''為[[臺南廳|臺南廳]]關帝廟支聽內新豐區管理，直到1920年臺灣地方改制，取較大聚落龍船及崎頂之字，設置[[龍崎庄|龍崎-{庄}-]]，隸屬[[臺南州|台南州]][[新豐郡|新豐郡]]。龍崎-{庄}-役場位於番社(佇大坪村)，1926年5月8日遷至崎頂。戰後改設臺南縣龍崎鄉，2010年12月25日改稱龍崎區。

== 文化 ==
龍崎區因人口外移嚴重及老化造成許多傳統文化相繼失傳，留下來的這些傳統陣頭文化皆以老傳幼的方式傳承，據老一輩的說法早期每個村莊都有組成陣頭來保衛村莊的安全達到守望相助的功能。

◎龍崎區文武陣頭
*石梯代天府-宋江陣(崎頂里)
*烏山頭清泉寺-天子門生柒響陣(牛埔里)
*土崎代天府-南龍吾獅陣(土崎里)
*龍湖宮-苦苓湖宋江陣(龍船里)
*龍安寺-牛犁陣(石嘈里)
