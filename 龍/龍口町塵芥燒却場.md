'''龍口町塵芥燒却場'''，[[臺北市|台北市]][[中正區_(臺北市)|中正區]]的歷史建築，是台北市於1932年至1933年興建的三處大型[[垃圾焚烧|垃圾焚化廠]]中，目前僅存之一處<ref name=":1">{{Cite web|url=https://nchdb.boch.gov.tw/assets/overview/historicalBuilding/20140822000001|title=龍口町「塵芥燒却場」|accessdate=2019-04-08|author=|date=|publisher=文化部文化資產局-國家文化資產網}}</ref>。其曾一度被違建所包覆，在2010年也曾發生大火<ref>{{Cite web|url=https://www.coolloud.org.tw/node/79128|title=第58次文資會通過王寵惠墓園古蹟指定，登錄龍口町塵芥燒却場歷史建築|accessdate=2019-04-08|author=台北市政府文化局|date=2014-06-20|publisher=}}</ref>。

== 沿革 ==
當時台北市每日垃圾量可多達三萬七千貫，而台北市內原有的七處垃圾焚化廠，其垃圾處理量合計僅有六千三百貫，其餘的垃圾便運送至低地的掩埋場，因而造成臭氣產生及蚊蟲聚集等衛生問題<ref>{{Cite web|url=http://ds3.th.gov.tw/ds3/app000/list3.php?ID1=00010585001|title=台北市塵芥燒卻爐新設費資金借入|accessdate=2019-04-08|author=|date=1932-05-01|publisher=臺灣總督府公文類纂}}</ref>。後來於是新設三處大垃圾焚化廠，分別位於[[大龍峒町|大龍峒町]]、[[龍口町|龍口町]]、[[大安_(臺北市)|大安]]。新的垃圾焚化廠處理量合計一日三萬貫，之後便停用原本的垃圾焚化廠。

== 建築特色 ==
焚化爐採用的「岩本式燒却爐」，為岩本嘉平創立的岩本工業株式會社所發明<ref>{{Cite web|url=https://www.jstage.jst.go.jp/article/jsmcwm/26/0/26_135/_pdf/-char/ja|title=台北市で発見された岩本式焼却炉について|accessdate=2019-04-08|author=水谷聰、顏嘉真|date=2015-09-02|publisher=第26回廃棄物資源循環学会研究発表会}}</ref>，共有四爐，以耐火磚砌成拱形，後方連結共同煙道與煙囪。爐前設有鋼筋混凝土造垃圾傾倒台，連接斜坡車道，垃圾可直接載運至上方傾倒口。<ref name=":1" />

== 垃圾處理流程 ==
台北市內的垃圾掃除分成七個轄區，分別為城內、東門、南門、萬華、西門、北、大成。1939年共聘有120位清潔人員，負責將垃圾送至塵芥燒却場，或雜廢物捨場（堆置灰燼、土砂、瓦礫、枝葉等）。掃除工作在人口密集區每天一次，其他區域二至三日一次；排洩物則是一週一次，送至屎尿殺菌池，台北市內共四處。<ref name=":0">{{Cite web|url=http://stfb.ntl.edu.tw/cgi-bin/gs32/gsweb.cgi?o=dbook&s=id=%22jpli2008-bk-sxt_0767_65_1939%22.&searchmode=basic|title=臺北市衞生施設要覽|accessdate=2019-04-08|author=|date=1939-05-12|publisher=臺北市衞生課}}</ref>
