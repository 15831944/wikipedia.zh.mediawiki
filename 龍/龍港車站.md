'''龍港車站'''位於[[台灣|台灣]][[苗栗縣|苗栗縣]][[後龍鎮_(台灣)|後龍鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[海岸線_(臺鐵)|海岸線]]的[[鐵路車站|鐵路車站]]。因興建新站房，原有的木造站房已拆除。由於其周邊居住人口不多，該站為西部幹線平均進出人數最少車站之一。<ref>{{Cite web|url=http://www.shawcat.com/2014/05/longgang.html|title=縱貫線上被遺忘的小站--龍港車站|accessdate=2016-12-22|work=www.shawcat.com}}</ref>

== 車站構造 ==
*[[側式月台|側式月台]]二座。
*跨線[[天橋|天橋]]。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]，僅停靠[[臺鐵區間車|區間車]]。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1922年10月11日：設「公司寮驛」。
*1952年12月1日：改為「龍港」。
*1986年6月1日：由三等站降為甲種簡易站，並指定由[[後龍車站|後龍站]]管理。
*1991年6月22日：由甲種簡易站降為招呼站。
*2015年6月30日：啟用多卡通刷卡機。
