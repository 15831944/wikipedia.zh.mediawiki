'''龍泉車站'''位於[[台灣|台灣]][[南投縣|南投縣]][[集集鎮|集集鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[集集線|集集線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[側式月台|側式月台]]兩座。
(兩月台間僅有一股軌，為[[西班牙式月台佈局|西班牙式月台佈局]]，為全台灣唯一列車會雙門同開的車站)

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]。
*行駛車輛為DR1000型[[柴油|柴油]]客車，等級為[[臺鐵區間車|區間車]]。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1922年1月14日：開通，初名'''隘寮'''。
*1962年4月1日：與[[源泉車站|源泉]]、[[大山車站|大山]]、[[山佳車站|山佳]]、[[鎮安車站|鎮安]]、[[猴硐車站|侯硐]]、[[平和車站|平和]]等車站同時改稱今名。
*1979年5月1日：由簡易站降為招呼站，並指定由[[二水車站|二水站]]管理。
*2015年6月30日：啟用多卡通刷卡機。
*2016年：增設的南側月台啟用。
