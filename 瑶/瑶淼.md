== 简介 ==
1978年，瑶淼出生于[[北京市|北京市]]。<ref name="fish " /> 1991年，瑶淼在[[北京|北京]][[西城区|西城区]]外国语学校读书。<ref name="fish " /> 1995年，瑶淼考入[[中国传媒大学|中国传媒大学]]。<ref name="fish " /> 瑶淼曾长时间在[[海南电视台|海南电视台]][[旅游卫视|旅游卫视]]担任主持人，后来到[[中国中央电视台|中国中央电视台]]担任主持人。<ref name="fish " /> 

== 个人生活 ==
瑶淼的丈夫是陈雕龙，女儿2009年出生，昵称橙子。<ref>{{cite web |url=http://ent.163.com/08/1007/17/4NLTRKTO00031H2L.html |title=央视女主播瑶淼传喜讯 明年情人节生仔当妈 |publisher=网易 |accessdate=2008年 |work= |language =}}</ref>
