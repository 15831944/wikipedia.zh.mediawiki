== 墨西哥第一帝国 ==

===伊图比德王朝===
1822年，[[西班牙|西班牙]]贵族后裔，土生白人'''阿古斯汀·德·伊图尔维德'''自称[[墨西哥|墨西哥]][[皇帝|皇帝]]，次年被推翻，逃亡[[欧洲|欧洲]]。第一帝国终结。1824年返回[[墨西哥|墨西哥]]企图复辟[[君主制|君主制]]，不久被捕，处以枪决。

*1822年-1823年    [[阿古斯汀一世_(墨西哥)|阿古斯汀一世]]（''Agustín I.''）

== 墨西哥第二帝国 ==

===[[哈布斯堡王朝|哈布斯堡王朝]]===
1862年，[[法国|法国]]入侵[[墨西哥|墨西哥]]，攻克首都[[墨西哥城|墨西哥城]]后，扶植[[奥地利|奥地利]]大公斐迪南·马克西米连为[[墨西哥|墨西哥]][[皇帝|皇帝]]，称'''马西米连诺一世'''。1866年[[法国|法国]]撤军，前总统[[贝尼托·华雷斯|贝尼托·华雷斯]]重新取得政权，次年将马西米连诺逮捕并枪决。

*1864年-1867年    [[马西米连诺一世_(墨西哥)|马西米连诺一世]] （''Maximiliano I.''）

== 墨西哥皇位继承人 ==
===伊图尔维德王朝===
*1824年-1864年：[[阿古斯汀二世_(墨西哥)|阿古斯汀·赫罗尼莫皇储]] （''Agustin Jerõnimo''，称阿古斯汀二世）

===哈布斯堡-伊图尔维德王朝===
[[马西米连诺一世_(墨西哥)|马西米连诺一世]]收养[[阿古斯汀二世_(墨西哥)|阿古斯汀·赫罗尼莫皇储]]的侄子[[阿古斯汀三世_(墨西哥)|小阿古斯汀]]和[[萨尔瓦多亲王_(墨西哥)|萨尔瓦多]]为养子，封为伊图尔维德亲王。[[马西米连诺一世_(墨西哥)|马西米连诺一世]]被枪决后，[[阿古斯汀三世_(墨西哥)|小阿古斯汀]]成为[[墨西哥|墨西哥]]皇位继承人。

*1867年-1925年：[[阿古斯汀三世_(墨西哥)|阿古斯汀亲王]] （阿古斯汀三世）
*1925年-1949年：[[玛利亚·何塞法_(墨西哥)|玛利亚·何塞法]] （''María Josefa''，[[萨尔瓦多亲王_(墨西哥)|萨尔瓦多亲王]]长女，称玛利亚一世）

===高岑-伊图尔维德王朝===
阿古斯汀亲王没有子嗣，萨尔瓦多亲王只有三个女儿。玛利亚·何塞法死后伊图尔维德王朝灭绝。她与夫君[[匈牙利|匈牙利]]贵族亚诺什·敦克尔男爵有两个女儿：玛利亚·安娜和玛利亚·吉塞拉。玛利亚·安娜（称玛利亚二世）没有子嗣；玛利亚·吉塞拉（称玛利亚三世）嫁给了[[摩拉维亚|摩拉维亚]]贵族古斯塔夫·阿道夫·冯·高岑[[伯爵|伯爵]]，他们的后代改姓“高岑-伊图尔维德”。玛利亚·何塞法去世前指定外孙，马西米连诺·德·高岑-伊图尔维德为她的继承人。

*1949年至今：[[马西米连诺二世_(墨西哥)|马西米连诺二世]]  （全名''Maximiliano Gustav Richard Albrecht Agustin Götzen-Yturbide''，[[玛利亚·吉塞拉_(墨西哥)|玛利亚·吉塞拉]]的独子。）
