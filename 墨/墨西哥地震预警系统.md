'''墨西哥地震预警系统'''（{{lang-es|Sistema de Alerta Sísmica Mexicano}}, SASMEX），是[[墨西哥|墨西哥]]依托潜在震源地附近的地震台网，在震后数秒内快速估算地震影响范围和程度，在破坏性的[[S波|S波]]和[[面波|面波]]到达监测区域时发布警报的[[地震预警系统|地震预警系统]]<ref name="论文2"/>。该系统的前身“墨西哥城地震预警系统”（{{lang-es|Sistema de Alerta Sísmica para la Ciudad de México, SAS}}）于1991年9月开始试运行，并于1993年8月正式向墨西哥城的民众提供预警信息<ref name="论文2"/>，是世界上第一个正式为公众提供预警的地震预警系统<ref name="论文2"/>。2005年，[[墨西哥城|墨西哥城]]和[[瓦哈卡市|瓦哈卡市]]对地震预警系统进行了整合，形成了墨西哥地震预警系统<ref name="论文E1"/>。

墨西哥地震预警系统由非营利性民间组织“墨西哥地震仪表与记录中心”（{{lang-es|Centro deInstrumentacióny RegistroSísmico, A.C}}或CIRES）管理，并受到{{link-es|墨西哥国家地震局|Servicio Sismológico Nacional (México)}}的支持<ref name="论文E1"/>。

== 历史 ==
在[[1985年墨西哥城大地震|1985年墨西哥城大地震]]中，距离震中有一定距离的[[墨西哥城|墨西哥城]]等地遭受到重创<ref name="论文M1"/>。重大的人员伤亡和财产损失促使墨西哥政府自1987年开始资助加速度计的部署<ref name="论文E1"/>，以便及时获得相应地区的地震信息。在地方政府的支持下，墨西哥地震仪表与记录中心开始了地震预警系统的研究和开发工作<ref name="论文E1"/>。并在1991年9月正式对“墨西哥城地震预警系统”开展试运行工作<ref name="论文M2"/>，最初仅部署了12个台站<ref name="论文M2"/>，对一些小学开展了广播早期警报试验<ref name="论文8"/>。1992年，在墨西哥城政府的组织下，就地震预警系统进行了6次公众评议，并吸收了来自公共和私营组织的建议<ref name="论文8"/>。1993年8月，该系统开始向墨西哥城的公众发布地震预警信息<ref name="论文2"/>。

1995年9月14日，在{{link-en|1995年格雷罗地震|1995 Guerrero earthquake|格雷罗地震}}中，该系统在地震波到达前72秒发布了地震预警，为紧急避险赢得了大量时间<ref name="论文5"/>，这是世界上首次对公众发布地震警报<ref name="论文M2"/>。而在{{link-en|1999年瓦哈卡地震|1999 Oaxaca earthquake}}后，墨西哥地震仪表与记录中心为[[瓦哈卡州|瓦哈卡州]]开发了类似系统<ref name="论文E1"/>。2003年，“瓦哈卡市地震预警系统”（{{lang-es|Sistema de Alerta Sísmica para la Ciudad de Oaxaca, SASO}}）投入使用<ref name="论文3"/>。

2005年，这两个系统得到整合，合并为“墨西哥地震预警系统”<ref name="论文E1"/>。2010年，墨西哥城政府投资更新了地震预警系统。通过这项投资，新增了64个地震监测台站，提高了监测密度，使地震预警系统覆盖了[[哈利斯科州|哈利斯科州]]、[[科利马州|科利马州]]、[[米却肯州|米却肯州]]和[[普埃布拉州|普埃布拉州]]，并增加了[[格雷罗州|格雷罗州]]的监测设备<ref name="论文E3"/>{{NoteTag|截至2016年，格雷罗州有监测台站33个，瓦哈卡州有39个，米却肯州有9个，哈利斯科州有6个，科利马州和普埃布拉州各有台站5个<ref name="论文M2"/>。}}。

== 原理 ==
[[File:Earthquake_Early_Warning_(Japan).svg|thumb]]系统的原理类似，当地震发生后，震源向外传播P波、S波时，地震监测台站接收到到达监测区域的地震波后，可将观测资料传到监测中心，经分析计算后，抢在地震波到达前对设防区域发布地震预警。]]
地震预警系统是指在地震发生後、而未及造成严重破坏前发出警告。在[[震中|震中]]发生地震後，具破坏性的[[S波|S波]]（每秒3.2-4千米）有一段時間尚未傳到各地，而由於[[電磁波|電磁波]]比地震波快，系统可以把烈度仪监测到的[[P波|P波]]（每秒5.5-7千米）信息转为電磁波，在地震波到达前的數秒至幾十秒發出预警<ref name="论文3"/>。

常规的地震预警系统可分为异地预警模式、现地（原地）预警模式与混合预警模式<ref name="论文2"/>。墨西哥地震预警系统采用异地预警模式<ref name="论文3"/>。异地预警模式是在地震发生后，利用震中附近的监测台站所观测到的地震波，快速计算出[[地震规模|地震规模]]、震中位置与[[震源深度|震源深度]]，并以此预估各地区的[[烈度|烈度]]与地震波到达的时间，再利用通讯技术通知各地区<ref name="论文4"/>。异地预警模式需要部署较多监测台站，且运算时间较长，但精确度较高，适用于距离震中较远的地区<ref name="论文2"/><ref name="论文4"/>。

在实际运用中，由于墨西哥城所处的特殊盆地环境，导致传播至此的地震动会被放大100-500倍，较其他地区通常情况下高出许多，因此这一系统对墨西哥首都圈地区具有良好的预警效果<ref name="论文1"/>。

== 运作 ==
[[File:Tectonic_setting_western_Mexico.png|thumb]]（Pacific Plate）和[[科科斯板块|科科斯板块]]（Cocos Plate）的[[隐没带|隐没带]]上。位于该国太平洋沿岸西部的{{tsl|en|Middle America Trench|中美洲海沟}}是该国主要的地震带之一，墨西哥地震预警系统将这一地区作为其监测区域，并部署了大量监测台站。]]
由于墨西哥的国土面积广大，加上墨西哥各级政府对地震预警系统的重视不够，因此该系统仅在墨西哥部分地区部署监测台站<ref name="论文2"/>。当前，该系统包含97个监测站，主要分布在[[巴亚尔塔港|巴亚尔塔港]]沿岸南部、[[格雷罗州|格雷罗州]]山区、[[普埃布拉|普埃布拉]]北部和[[瓦哈卡州|瓦哈卡州]]大部地区。目前，墨西哥有关当局正计划新增28个监测站，将提高[[瓦哈卡州|瓦哈卡]]地峡、[[韦拉克鲁斯|韦拉克鲁斯]]内陆地区和[[恰帕斯|恰帕斯]]西部三分之二地区的监测预警能力<ref name="论文E3"/>。

在地震监测区内，由墨西哥地震仪表与记录中心负责建设和维护任务，处理本区域及周边地区的台站观测数据，并负责地震预警信息发布<ref name="论文E1"/>。而作为政府机构的墨西哥国家地震服务中心则监督地震预警的发布，并根据检知的地震信息组织防震救灾应急响应工作<ref name="论文E1"/>。

墨西哥地震预警系统由地震检测系统、通讯系统、中央控制系统和警报发布系统4个子系统组成<ref name="论文6"/>。地震发生后，首先将加速度计数据传到墨西哥地震仪表与记录中心下属的分析处理中心，分析处理中心利用海岸附近的加速度计获得的地动[[加速度|加速度]]（gal）进行地震的观测，对台站的记录进行积分（从P波的触发至2倍的S-P波的时间长度内）以获得能量参数，同时快速计算能量增长速率，根据过往经验判断地震级别。在地震规模达到“中强震”级（Mw 5-6级）时，会由墨西哥地震仪表与记录中心向特定机构播发“预防性预警信息”<ref name="论文2"/>。在地震规模达到“强震”级（Mw 6级以上）时，则会向公众播发“公开性预警信息”<ref name="论文2"/>。

墨西哥地震仪表与记录中心的预警传输方式类似于[[美国国家海洋和大气管理局|美国国家海洋和大气管理局]]的[[NOAA气象广播|NOAA气象广播]]。通过[[甚高频|甚高频]]波段广播地震预警，并使用特定区域信息编码，所需时间更短，以便在两秒钟以内发出及时的预警<ref name="论文E1"/>。

作为历史最悠久的地震预警系统之一，墨西哥地震预警系统相较于之后建成的地震预警系统而言，技术手段仍较为原始，是在具破坏性的S波和面波到达监测区域后才发出警报<ref name="论文2"/>。但由于设防区域距离监测台站较远<ref name="论文3"/>，因此若在监测区域发生地震，仍能够进行有效的预警<ref name="论文E1"/>。

== 播送 ==
[[File:Receptor_del_Sistema_de_Alerta_Sísmica_para_la_Ciudad_de_México.JPG|缩略图]]
墨西哥地震预警系统在墨西哥主要地区得到了广泛的运用<ref name="论文2"/>。一般而言，该系统能够向墨西哥城提供60秒左右的预警时间，并向瓦哈卡市提供超过30秒的预警时间<ref name="论文E1"/>。在{{link-en|2012年格雷罗 - 瓦哈卡地震|2012 Guerrero–Oaxaca earthquake}}期间，该系统为瓦哈卡市提供了25秒的预警时间，而奇尔潘辛戈和阿卡普尔科则有45秒预警时间，墨西哥城的预警时间更达到了75秒<ref name="论文E6"/>。在[[2017年恰帕斯州地震|2017年恰帕斯州地震]]中，该系统也发挥了作用<ref name="报道E7"/>。而在[[2017年普埃布拉州地震|2017年9月19日的地震]]中，该系统为墨西哥城赢得了20秒的预警时间<ref name="报道E8"/><ref name="报道M1"/>。尽管预警时间较往常更短<ref name="报道2"/>，但[[英国广播公司|英国广播公司]]认为，这足以使人们采取必要的避险措施<ref name="报道E9"/>。

=== 广播电视 ===
预警系统在发布地震预警时，会即时通知墨西哥城的58个广播电台和6个电视台<ref name="论文2"/>，托卢卡市的3个广播电台和1个电视台<ref name="论文2"/>，以及[[阿卡普尔科|阿卡普尔科]]、[[奇尔潘辛戈|奇尔潘辛戈]]等地的广播电视机构向公众发布警报信息<ref name="论文2"/><ref name="论文E5"/>。

墨西哥城的商业广播电台自1995年9月起开始在[[调频|调频]]、[[调幅|调幅]]波段中发布“公开性预警信息”<ref name="论文8"/>。预警发布时，各电台会中断正常节目，播放预先录制好的60秒音频，内容包括播音员用西班牙语播送的语音：“地震警报，地震警报。<ref name="论文8"/>”在预警系统建立的初期，音频需要通过人工插入磁带才能播放。但之后已能够通过专用音频控制系统自动广播<ref name="论文8"/>。

=== 信息网络 ===
在发布方面，墨西哥地震仪表与记录中心通过名为“AlertaSísmicaSASMEX”的[[推特|推特]]帐号定期提供各种信息，包括各种震级报告和警报试广演习信息<ref name="论文M3"/>。在该帐号发布的信息中，所有地震相关信息使用#TenemosSismo（我们有地震）的[[主题标签|主题标签]]，而“公开性预警信息”则使用#AlertaSismica（地震警报）的主题标签<ref name="论文M3"/>。

=== 设施场所 ===
墨西哥地震仪表与记录中心发布“公开性预警信息”时，会通过架设在墨西哥城等地的城市警报广播系统播发地震预警<ref name="论文E1"/>。截至2015年，墨西哥城共配备了超过8,200个警报扬声器<ref name="论文E6"/>。在平时，该系统也会每周进行警报试验广播以测试系统运行情况，确保接收机连入预警网络<ref name="论文E4"/>。此外，墨西哥城附近的公立学校几乎都配备了警报接收机<ref name="论文E4"/>，服务学校师生的地震逃生避险<ref name="论文E4"/>。地震仪表与记录中心还会向[[墨西哥城地铁|墨西哥城地铁]]发布定制化紧急地震信息，服务地铁运输的地震紧急处置<ref name="论文E1"/>。当收到“预防性预警信息”时，地铁控制部门可以使运行中的列车紧急停车或扣停已到站的列车<ref name="论文E1"/>。同时，墨西哥城的90,000多名用户也能在地震时及时接收地震预警<ref name="论文E4"/>。另外，墨西哥国家灾害防御中心、民防局、墨西哥公共事务部、墨西哥城公共电力局、墨西哥城警察局等地也装配了警报接收机<ref name="论文8"/>。

== 局限 ==
=== 准确性 ===
有研究者认为，该系统虽然能有效捡拾到破坏性地震的发生并及时发布预警信息，但其估计震级的准确性有待进一步提高，以便更可靠地发布“公开性预警信息”或“预防性预警信息”<ref name="论文1"/>。还有研究者认为，墨西哥地震预警发布比较粗放，尤其是震级的估算不精确<ref name="论文7"/>。根据墨西哥国家地震局的标准，只有发布预警而没有发生地震才算误报<ref name="论文7"/>。统计表明，在墨西哥地震预警系统发布的66个“公开性预警信息”中，仅有大约30%的震级估算是准确的<ref name="论文7"/>。

对于地震预警系统而言，漏报和误报无法避免<ref name="论文2"/>。在1991年10月至2009年3月间，墨西哥城的地震预警系统共发布了13次“公开性预警信息”<ref name="论文2"/>和53次“预防性预警信息”<ref name="论文7"/>，并有2次漏报和1次误报<ref name="论文2"/>。而在2003年至2013年间，瓦哈卡市的地震预警系统共发布了3次“公开性预警信息”和5次“预防性预警信息”，并有2次漏报<ref name="论文2"/>。1993年10月24日，该系统漏发了一次6.7级地震的预警信息<ref name="论文8"/>。1993年11月16日，墨西哥城地震预警系统发布了一次“公开性预警信息”，事后证明并没有地震发生，不过这没有给墨西哥城居民造成恐慌和伤亡<ref name="论文7"/>。2017年9月19日，墨西哥地震预警系统在进行警报试广时，错误地触动了警报装置，导致警报扬声器发出了地震警报音频<ref name="报道1"/><ref name="报道E9"/>。但在误报3小时后发生的7.1级地震中<ref name="报道E9"/>，该系统则由于地震震源浅、地震波传播快而没有及时发布预警信息<ref name="报道2"/>。墨西哥国家灾害防御中心主任卡洛斯表示，“至少需要三个传感器才能使警报器报警，技术上来说预警系统不可能覆盖整个国家。现在的系统主要监控国家沿海区域（离莫雷洛斯州很远），莫雷洛斯州的震源深度浅，导致地震传播速度太快，传感器监测到地震与地震发生几乎是同一时刻。<ref name="报道2"/>”

=== 覆盖率 ===
此外，该系统的覆盖范围亦有很大不足<ref name="论文1"/>。在墨西哥城，2,000万人口中仅有约440万人可以及时接收到预警信息<ref name="论文3"/>。而在瓦哈卡市，由于当地主管部门对地震预警系统的重视程度不够，造成预警系统长期缺乏资金维持，系统也未能有效维护，许多地方仍未安装预警接收机<ref name="论文1"/>。截至2013年，瓦哈卡市的5,500所学校中只有76所配备了预警接收机<ref name="论文1"/>。

同时，对地震预警的教育和培训工作也很不到位<ref name="论文1"/>。2009年，一项针对墨西哥地震预警系统接收效果的调查显示，约91%的受访者知道墨西哥地震预警系统<ref name="论文9"/>。但在216名受访者中，只有135人能够在地震来临时对预警作出有效识别，占比62.5%<ref name="论文9"/>。
