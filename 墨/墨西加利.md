'''墨西卡利'''（[[西班牙語|西班牙語]]：'''{{lang|es|Mexicali}}'''）是[[墨西哥|墨西哥]][[下加利福尼亞州|下加利福尼亞州]]的[[首府|首府]]，北面接壤[[美國|美國]][[加利福尼亞州|加利福尼亞州]][[加利西哥|加利西哥]]及東北面[[亞利桑那州|亞利桑那州]]。根據2005年普查人口佔下加州的75%；人口來自不同種族，並有一個[[唐人街|唐人街]]，約有華人5000人<ref>[http://gocn.southcn.com/qwxw/200506280062.htm 餐饮业――墨西卡利华人的主营行业]，广东侨网，2005年6月28日</ref>。

墨西卡利的名字取自'''墨西'''哥和美国'''加利'''福尼亚州的字首<ref>[http://www.xzqh.org/suoyin/sjdm/152.htm 中国行政区划·世界地名索引] {{webarchive|url=https://web.archive.org/web/20070808051925/http://www.xzqh.org/suoyin/sjdm/152.htm |date=2007-08-08 }}</ref>，所以亦有譯作'''墨西加利'''。

2006年，[[美國籃球協會_(21世紀)|美國籃球協會]]新組成的球隊[[墨西卡利哨兵隊|墨西卡利哨兵隊]]以此為基地。

== 氣候 ==
為了避開[[科羅拉多河|科羅拉多河]]氾濫，墨西卡利地處[[沙漠|沙漠]]中心的[[谷地|谷地]]，[[氣候|氣候]]頗為極端，[[冬季|冬季]]低至[[攝氏|攝氏]]4度（[[華氏|華氏]]40度），[[夏季|夏季]]則可達攝氏40至50度（華氏100至120度），年降水量不足250毫米。

== 歷史 ==
早在[[哥倫布|哥倫布]]發現[[新大陸|新大陸]]之前，[[科羅拉多河|科羅拉多河]][[三角洲|三角洲]]的[[美洲原住民|美洲原住民]]已經在此定居。當[[西班牙|西班牙]]人後來沿河抵達此地，人們已經在此種植[[南瓜|南瓜]]、[[甜瓜|甜瓜]]、[[豌豆|豌豆]]以及黃、藍、白、紅及藍白五色的[[玉米|玉米]]；亦已懂得使用[[草藥|草藥]]、及[[龍舌蘭|龍舌蘭]]作多種用途。

在其後一段時間，西班牙人及墨西哥人都無暇參與此地的發展。在[[美國南北戰爭|美國南北戰爭]]期間，一位[[耶魯大學|耶魯大學]][[地理學家|地理學家]]沿科羅拉多河到達此地探勘[[南太平洋鐵路|南太平洋鐵路]]路線，發現三角洲沉積物厚達2.5公里，土地肥沃可作良田；但其實當地原住民早已知悉此事，只欠水源。進入20世紀，美國加州土地公司得到政府批准，從科羅拉多河興建[[運河|運河]]，把水源引到墨西卡利谷地。
由於大片土地橫跨美墨邊境，在美國的一邊以「加州」和「墨西哥」的字首名為卡萊克西科，墨西哥境內則名為墨西加利。

1903年3月14日，墨西卡利建鎮，首任鎮長是Manuel Vizcarra，吸引到首500位[[農夫|農夫]]遷入。
到1904年底，已有10,000人在此定居，在100,000英畝（405平方公里）灌溉農地種有[[棉花|棉花]]、蔬果等農作物。
1915年1月，Francisco L. Montejano就任首任市長，到同年11月20日，墨西卡利成為了下加州北部的首府。
1920年代，華人開始移居當地。由於當地僱有不少華工，所以當時的墨西卡利又有「小廣州」的外號<ref>{{cite episode |title=闖拉美#2 |episodeurl=http://programme.rthk.org.hk/rthk/tv/programme.php?name=tv/rootsoldandnew2016&d=2016-01-23&p=7122&e=340576&m=episode 
|series=[[華人移民史_(紀錄片)|華人移民史 (紀錄片)]]
|serieslink=http://programme.rthk.org.hk/rthk/tv/programme.php?name=tv/rootsoldandnew2016 
|network=[[港台電視|港台電視]] |station=[[港台電視31|港台電視31]] |date=2016-01-23 |season=3 |seriesno=2 |language=zh-yue }}</ref>。
在[[經濟大蕭條|經濟大蕭條]]期間，墨西哥發生「排華」活動，很多華人連同其家人被強行遣返，到1954年才結束。有不少不願意離開墨西哥的華人，就來到華人人口眾多的墨西卡利繼續生活。
1936年至1937年間，科羅拉多河土地公司讓農夫們購回自己的農地。
到1940年，墨西哥華人人口從大蕭條之前的數萬人，只餘下不足一萬人。

== 矽谷邊境 ==
美國加州晶片業者為減低成本，在墨西卡利建立晶片製造基地，獲得墨西哥聯邦及下加州政府投資2百萬美元，總統並提供投資超過10億美元的廠商享有10年免稅優惠。加州州長[[阿諾·史瓦辛格|阿諾·史瓦辛格]]一再在電台廣播中呼籲支持。配套工程包括興建新公路及過境設施等。預計該項目將會為墨西哥帶來10萬個工作崗位。

有報導指，矽谷邊境是為抗衡台灣及中國的晶片產業。藉著墨西卡利工業用水、天然氣能源、低廉勞動力以及1994年[[北美自由貿易協議|北美自由貿易協議]]的優勢，加州矽谷業者希望重奪市場佔有率。但亦有業者以[[阿聯酋|阿聯酋]][[迪拜|迪拜]]的「矽谷綠洲」失敗的經驗指，墨西哥較低的員工文化程度以及能源勞動力成本是不明朗因素<ref>[http://gb.cri.cn/7383/2005/02/21/153@455264.htm  美國瞄準墨西卡利 欲打造矽谷新“殖民地”]，國際線上，2005年2月21日</ref>。

而台灣廠商亦有在墨西卡利設廠<ref>[http://www.benq.com.tw/page/?pageId=5 BenQ 台灣 - 明基大事紀]</ref>。

== 旅遊業 ==
作為邊境城市的墨西卡利與[[蒂華納|蒂華納]]相似，在假日吸引不少美國人前來消費。
