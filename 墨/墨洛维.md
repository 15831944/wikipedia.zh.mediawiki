'''墨洛维'''（{{lang-fr|Mérovée}}，{{bd|412年||457年||catIdx=M}}），半神話化的[[法蘭克人|法蘭克]]王國[[墨洛温王朝|墨洛温王朝]]國王（450年—457年在位）。

== 生平与传说 ==

相傳是法蘭克[[萨利昂法兰克|萨利昂部族]]的墨洛温王朝的始創人（應是其父[[克洛迪奧|克洛迪奧]]建立的）、墨洛溫王朝建立者[[克洛維一世|克洛維]]的祖父，墨洛溫王朝是以墨洛維的名字命名。歷史上關於墨洛維的記載大多已經神化半神化，根據弗萊德加編年史記載，墨洛維是克洛迪奧的妻子的兒子，父親是一位海神。一些學者指出，法蘭克人的酋長墨洛維，可能是法蘭克人在皈依基督教前崇拜的某個神或半神半人。

== 死亡 ==

墨洛维死于457年，活了45岁。

== 配偶与子女 ==

他的妻子也是未知，只知道有一子[[希尔德里克一世|希尔德里克一世]]。
