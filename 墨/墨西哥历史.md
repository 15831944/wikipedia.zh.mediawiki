{{墨西哥历史}}
[[墨西哥|墨西哥]]位於[[中美洲|中美洲]]，是世界上最大的[[西班牙语国家和地区列表|西班牙語國家]]。在古代的人類足跡被發現後，人類在[[墨西哥|墨西哥]]的活動已可追溯到距今約四萬年前。
[[File:Flag_of_Mexico.svg|right]]

== 西元前 ==
{{main|{{le|前哥伦布时期的墨西哥|Pre-Columbian Mexico}}}}
大約九千年前，古代墨西哥人開始使用[[玉米|玉米]]作為糧食作物，推動農業革命，從而導致形成了許多文明。這些文明造就出了[[城市|城市]]、[[建築|建築]]、[[天文|天文]]研究、[[數學|數學]]和[[軍隊|軍隊]]。著名的文明有[[奧爾梅克|奧爾梅克]]、[[提奧提華坎|提奧提華坎]]、[[阿茲特克|阿茲特克]]和[[馬雅|馬雅]]文明。

=== 奧爾梅克 ===
[[File:Olmeca_head_in_Villahermosa.jpg|左]]
{{main|奧爾梅克}}奧爾梅克文明是已知的最古老的美洲文明。它約存在和於公元前1200年到公元前400年，位在於現在的墨西哥中南部的熱帶雨林中，以大型的頭部雕像聞名。

=== 玛雅文明 ===
{{main|玛雅文明}}
[[File:Maya_Chac_Mool_by_Luis_Alberto_Melograna.jpg|thumb]]出土的[[玛雅文明|玛雅]]恰克穆尔，现陈列于[[国立人类学博物馆|墨西哥国家人类学博物馆]]]]

=== 提奧提華坎 ===
{{main|提奧提華坎}}
提奧提華坎文明起始於西元前200年左右，大約在現在墨西哥的中部，是個在奧爾梅克文明滅亡之後才誕生，約與馬雅文明同期的古印地安文明，提奧提華坎人在西元元年到150年間，建造了一個人口約為5萬的城市，堪稱是整個美洲地區最早存在的城市等級之聚落，其建造許多金字塔。

== 西班牙統治 ==
{{main|西班牙征服阿兹特克帝国}}
1519年，[[西班牙|西班牙]]入侵[[墨西哥|墨西哥]]。1521年，[[阿茲特克|阿茲特克]]首都[[特諾奇蒂特蘭|特諾奇蒂特蘭]]被[[西班牙|西班牙]][[科薩特|科薩特]]所率領的軍隊征服，但是[[西班牙|西班牙]]並沒有完全征服[[阿茲特克|阿茲特克]]，而是在兩世紀後才完成，從過程中可歸納出兩個主要原因使西班牙人勝利：
#[[阿茲特克人|阿茲特克人]]因為相信要有恆定的活人犧牲，宇宙才會持續運作，所以每一年都犧牲了許多人以舉辦獻祭，這一些人大多來自戰爭的戰俘，但在和平的時候，阿茲特克人依然為了獲得戰俘而常常不由分說的開啟戰端，因此鄰近不少原住民部落才會心甘情願加入[[西班牙|西班牙]]對付[[阿茲特克人|阿茲特克人]]的的部隊 。
#另一個重要因素，[[西班牙人|西班牙人]]帶入各種瘟疫和傳染病帶到[[美洲|美洲]]，例如[[天花|天花]]、[[流感|流感]]、[[鼠疫|鼠疫]]、[[麻疹|麻疹]]，數以十萬計的土人受到感染，這些流行病可能造成大約八百萬當地人死亡。

== 獨立 ==
[[File:Mexico.DF.Coyoacan.MiguelHidalgo.Statue.01.jpg|100px]]
受到法国和美国革命影响，1808年[[墨西哥|墨西哥]]起义者得到一个历史背景下所產生的机遇。此时[[马德里|马德里]]的国王已被[[拿破侖|拿破侖]]废除，[[西班牙|西班牙]]持續受到[[拿破侖战争|拿破侖战争]]和被占领的影响。

墨西哥的獨立戰爭由民族英雄[[米格尔·伊达尔戈·伊·科斯蒂利亚|米格尔·伊达尔戈·伊·科斯蒂利亚]]在1810年發起，推翻西班牙統治。原定在10月1日起義，但是由於叛徒告密洩露，所以提早在9月16日的多洛雷斯小鎮[[多洛雷斯呼聲|發動]]，一直到1821年解放軍才進入[[墨西哥市|墨西哥市]]，建立[[墨西哥第一帝國|墨西哥第一帝國]]。1823年軍隊推翻帝制，建立[[墨西哥合众国|墨西哥共和國]]，[[西班牙总统|瓜達盧佩維多利亞]]為其首任總統。墨西哥第一帝國減亡之際，[[中美洲联邦共和国|中美洲联邦共和国]]（今日的[[哥斯达黎加|哥斯达黎加]]、[[萨尔瓦多|萨尔瓦多]]、[[危地马拉|危地马拉]]、[[洪都拉斯|洪都拉斯]]和[[尼加拉瓜|尼加拉瓜]]）亦趁機獨立。

==第一次法國入侵==
{{main|糕点战争}}
1838年至1839年間，[[法國|法國]]第一次入侵墨西哥，名為[[糕点战争|糕点战争]]，以[[法國|法國]]勝利告終。

== 美國開戰 ==
[[File:Mapa_Mexico_1845.PNG|thumb]]
[[美墨戰爭|美墨戰爭]]由1846年一直打到1848年，戰爭原因是因為[[德克薩斯|德克薩斯]]（現[[德州|德州]]）獨立和[[美國|美國]]擴張主義的關係，美國以利益吸收[[德克薩斯共和國|德克薩斯共和國]]為第28個州，並且對墨西哥遙遠的[[加利福尼亞|加利福尼亞]]和[[新墨西哥|新墨西哥]]進攻，戰後墨西哥喪失了其原國土三分之一的面積，導致墨西哥人現在還是會仇恨美國人，也創立了新一批的政治家，他們擺脫了獨栽者[[安东尼奥·洛佩斯·德·桑塔·安纳|桑塔·安納]]對墨西哥的統治，並於1857年建立了一個自由共和國。

== 第二次法國入侵 ==
{{main|法國武裝干涉墨西哥}}
在1861年[[改革战争|改革战争]]內戰剛結束後，墨西哥總統[[贝尼托·胡亚雷斯|胡亞雷斯]]停止向外國支付[[債券|債券]]的利息，結果激怒了最大的借款人[[法國|法國]]、[[西班牙|西班牙]]及[[英國|英國]]，以三國聯軍武力追債並一度佔領首都，扶殖了[[傀儡|傀儡]]政權[[墨西哥第二帝國|墨西哥第二帝國]]及其皇帝[[马西米连诺一世|马西米连诺一世]]。[[贝尼托·胡亚雷斯|胡亞雷斯]]一度被推翻並逃亡至[[華雷斯城|華雷斯城]]和[[奇瓦瓦市|奇瓦瓦市]]，後來終於成攻擊退三國聯軍並於1867年處決了[[马西米连诺一世|马西米连诺一世]]，恢復民主制度並且競選總統連任成功，成為了民族英雄。在1871年，他再度當選，他的反對黨自由黨，他們認為選舉過程不民主。胡亚雷斯去世一年後，米格爾德特哈達繼任（Miguel Lerdo de Tejada）。

== 迪亞斯獨裁 ==
在1876年Lerdo再度當選，擊敗[[波费里奥·迪亚斯|迪亞斯]]。1876年，迪亞斯反抗政府並宣布計劃推翻 Lerdo 自立為總統。迪亞斯推翻原政府後，Lerdo逃離該國，迪亞斯成為新的總統。從而開始了超過三十年（1876年到1911年）的獨裁統治，這個時期的相對繁榮與和平，該國的基礎設施大為改善，這有賴外國投資。

==当代==
在经过[[波菲里奥·迪亚斯|波菲里奥·迪亚斯]]長時期的獨裁統治後，[[墨西哥革命|墨西哥革命]]終於在1910年爆发。革命武装击败了联邦军队，但却又发生内鬥，让墨西哥在长达二十年的时间裡陷入[[墨西哥內戰|内战]]。革命结束后，墨西哥[[革命制度党|革命制度党]]获得了政权，并且统治着墨西哥，直到20世纪末，而於2000年實現了首次的[[政黨輪替|政黨輪替]]。

2006年7月2日，墨西哥总统和议会选举开始投票。<ref>[http://news.xinhuanet.com/newscenter/2006-07/03/content_4786062.htm]</ref>9月5日，墨西哥联邦选举法院正式宣布，[[墨西哥国家行动党|墨西哥国家行动党]]总统候选人[[费利佩·卡尔德龙·伊诺霍萨|费利佩·卡尔德龙·伊诺霍萨]]在总统选举中获胜，当选墨西哥新总统。<ref>{{cite web |url=http://news3.xinhuanet.com/world/2006-09/06/content_5055285.htm |title=存档副本 |accessdate=2006-09-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20140714230820/http://news3.xinhuanet.com/world/2006-09/06/content_5055285.htm |archivedate=2014-07-14 }}</ref>
