《'''墨西哥往事'''》（{{lang-en|'''Once Upon a Time in Mexico'''}}）2003年上映于[[美国|美国]]的[[动作片|动作片]]，由[[罗伯特·罗德里格兹|罗伯特·罗德里格兹]]担任导演、配乐师、剪辑师、摄影师，安东尼奥·班德拉斯、莎玛·希恩、強尼·戴普、米基·洛克、伊娃·门德斯、丹尼·特乔等担任主演，讲述了[[墨西哥|墨西哥]][[沙漠|沙漠]]上的民间英雄和正义战士“埃尔·马拉奇”传奇人生。<ref>{{cite web |url= http://yule.sohu.com/90/55/article214375590.shtml|title=外语影片《墨西哥往事》主要演员名单 |accessdate=2003-10-13 |publisher=搜狐网 |work= |language =zh-cn}}</ref>

== 剧情 ==
埃尔·马拉奇是一个[[吉他手|吉他手]]，同时也是一个神枪手，他在墨西哥沙漠上与毒枭、黑社会、帮派组织为敌，经常破坏他们的“好事”，正是因为他喜好“惹是生非”，导致自己家庭破碎，妻子被谋杀，失去家庭和爱情的马拉奇从此浪迹天涯、神龙见首不见尾。

大毒枭巴罗瑞在墨西哥声名日响，他的帮派四处攻打其他弱小组织，确立了他在墨西哥毒品市场的王者地位，看到自己俨然成为老大之后，巴罗瑞并不满足，他决心踏足政界，成为政治明星，于是就策划谋杀[[墨西哥总统|墨西哥总统]]，推翻现政府，建立自己的新政府。美国[[中央情报局|中央情报局]]（CIA）探员桑德斯一直监视巴罗瑞的一举一动，桑德斯得知这个惊天信息之后，打算摧毁其阴谋，桑德斯便开始寻找马拉奇，并且说服马拉奇重出江湖，主持正义。<ref>{{cite web |url= http://yule.sohu.com/11/56/article214375611.shtml|title=外语影片《墨西哥往事》剧情简介 |accessdate=2003-10-13 |publisher= 搜狐网|work= |language =zh-cn}}</ref>
