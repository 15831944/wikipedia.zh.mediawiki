'''墨子'''是[[戰國|戰國]]著名[[思想家|思想家]]、[[政治家|政治家]]、[[科学家|科学家]]、[[军事家|军事家]]。春秋末战国初期[[宋国|宋国]]（今[[河南|河南]][[商丘|商丘]]）人，一说[[鲁国|鲁国]]（今[[山东|山东]][[滕州|滕州]][[木石鎮|木石鎮]]）人，約為前479年—前390年以内{{efn|1=有说墨子生卒约公元前468年—公元前376年或公元前480年—前400年。[[司马迁|司马迁]]在《史记·孟子荀卿传》末附载云：“或曰并孔子时，或曰在其后”。[[孫詒讓|孫詒讓]]考定，墨子生年约为公元前468年，卒年则为前376年。}}。他提出了“[[兼爱|兼爱]]”、“非攻”、“尚贤”、“尚同”、“天志”、“明鬼”、“非命”、“非儒”、“非樂”、“節葬”、“節用”、“交相利”等观点，创立'''[[墨家|墨家]]'''学说，并有《[[墨子_(书)|墨子]]》一书传世。[[墨家|墨家]]在当时影响很大，《[[孟子_(著作)|孟子]]·[[滕文公|滕文公]]》篇云：「[[楊朱|楊朱]]、墨翟之言盈天下，天下之言，不歸於楊，即歸墨。」可知[[戰國|戰國]]之世，墨家屬顯學。《墨子》一书中體現的墨子的思想在后世仍具有一定影响，广为流传的《[[千字文|千字文]]》中便記載了出自《墨子·所染》的「墨悲絲染」的故事。到了[[清朝|清朝]]，随着诸子学的兴起，学者对墨子思想的研究又提升到新的高度。

== 生平 ==
墨子曾学习儒术，因不满“礼”之烦琐，另立新说，聚徒讲学。据说楚王曾计划攻[[宋国|宋]]，墨子前往劝说楚王，并在与[[公输般|公输般]]（魯班）的模拟攻防中取得胜利，楚王只得退兵。墨子的思想与主张见[[墨家|墨家]]。

清朝學者[[汪中|汪中]]治[[墨學|墨學]]，他将墨子与[[孔子|孔子]]平视，认为“孔子鲁之大夫也，而墨子宋之大夫也，其位相埒”。然墨子與制禮作樂的[[孔子|孔子]]有相當程度上的差異。孔子主張久喪厚葬；子曰：“食不厭精，膾不厭細。”<ref>《論語·鄉黨篇》</ref>墨子一生過著簡樸的生活，“量腹而食，度身而衣”<ref>《墨子·魯問篇》</ref>，主張“節用，節葬，非樂”，他的弟子也是“短褐之衣，藜藿之羹，朝得之，則夕弗得”，“摩顶放踵，利天下，为之”。<ref>《孟子·尽心上》</ref>

=== 墨子姓名考 ===
[[历史|历史]]学界至今无法确定墨子的真实[[姓名|姓名]]，有以下各种说法：

==== 墨氏，名翟 ====
最传统的说法是，墨子，墨氏，名翟，《[[吕氏春秋|吕氏春秋]]》、《[[淮南子|淮南子]]》、《[[史记|史记]]·孟子荀卿列传》内都如是称。[[司馬遷|司馬遷]]沒有給墨子立傳，僅在[[孟子|孟子]]、[[荀卿|荀卿]]傳後，有簡略記載：“蓋墨翟宋之大夫，善守禦，為節用，或曰並孔子時，或曰在其後”。《[[元和姓纂|元和姓纂]]》称，“墨氏，[[孤竹君|孤竹君]]之後，本墨台<ref>讀若「怡」</ref>氏，後改為墨氏，[[戰國|戰國]]時[[宋国|宋]]人。墨翟著書號墨子。”而《[[新唐书|新唐书]]·艺文志》也沿用这种说法。[[童书业|童书业]]根据《[[广韵|广韵]]·六脂》以及《[[姓氏急救篇|姓氏急救篇]]》所引的《[[世本|世本]]》文字“[[宋襄公|宋襄公]]子墨夷须为大司马，其后有墨夷皋。”及《[[通志|通志]]·氏族略》记载的“墨台氏，子姓，宋成公子墨台之后。”考证得出，[[宋襄公|宋襄公]]之兄[[子鱼|公子目夷（子鱼）]]，也就是《[[左传|左传]]·僖公二十二年》所记载的著名的[[子鱼|子鱼]]论战故事的主人公，其名'''目夷'''又写作'''墨夷'''或'''墨台'''，其后裔以墨夷为氏。童书业据此认为墨子为[[宋国|宋国]]公室子姓墨夷氏之后<ref>童书业，春秋左传研究（校订本），北京：中华书局，2006年8月，第222页。</ref>，因此墨子继承了贵族身份为宋国大夫。

==== 翟姓，名乌 ====
[[南齐|南齐]][[孔稚圭|孔稚圭]]所著的《[[北山移文|北山移文]]》则称墨翟为“翟子”，[[元朝|元朝]][[伊世珍|伊世珍]]所著的《[[瑯環記|瑯環記]]》{{efn|1=《瑯環記》原文：“墨子姓翟名烏。其母夢，日中赤鳥飛入室中。炎輝照耀，目不能正警覺。生烏，遂名之”。}}也附和此说，并认为墨子姓[[翟姓|翟]]名乌。[[清朝|清朝]][[周亮工|周亮工]]所著的《[[固树屋书影|固树屋书影]]》更具体地提出：“以墨为道，今以姓为名。”认为他姓翟，并将姓转成名；而“墨”是一种学派。晚清学者[[江瑔|江瑔]]所著的《[[读子卮言|读子卮言]]》承袭周亮工的说法，并进一步说明，以为古代确实有“翟”这姓氏，但无“墨”姓，而且战国诸子中儒、道、名、法、阴阳、纵横、杂、农、小说等，都没以姓作为学派名，因此墨应该是学派的名称。

==== 墨取自「刑罚」或「膚色」 ====
近代学者[[钱穆|钱穆]]从[[墨刑|墨刑]]是古代刑名之一的角度展开研究，认为古人犯轻刑，则罚作奴隶苦工，故名墨为刑徒，实为奴役，而墨家生活菲薄，其道以自苦为极。墨子和弟子们都“手足胼胝，面目黎黑，役身给使，不敢问欲”，人人皆可使“赴火蹈刃，死不旋踵”。这样，就被称为墨了<ref>钱穆《墨子传略》</ref>。而历史上[[西汉|西汉]]淮南王[[英布|英布]]遭[[黥刑|黥刑]]后称为黥布，也是一证。而翟确实是名，因《墨子》和《吕氏春秋》记载墨翟，往往称“翟”。

也有人認為[[墨家|墨家]]「手足胼胝，面目黧黑」，「墨」即是「黧黑」。這有可能是對墨子刻苦自勵的一種形容詞。或者墨子根本是來自於[[印度|印度]]的外族人<ref>李隆獻《墨子概說》.台灣大學</ref>。

=== 墨子里籍考 ===
墨子生于何处，现有书籍并无明确记载，历史上也有不同的说法，曾有宋人说，楚人说，鲁人说，甚至有[[印度人|印度人]]说等。其中鲁人说已经为[[墨学|墨学]]界绝大部分人所认可。

====宋人说 ====
* 《[[史记|史记]]·[[孟子|孟子]][[荀卿|荀卿]]列传》：“盖墨翟，宋之大夫，善守御，为节用。或曰并孔子时，或曰在其后。”据此有学者称其为宋人。此说一直流行到[[清代|清代]]。

==== 楚人说 ====
* [[清代|清代]][[毕沅|毕沅]]注《墨子》时，据《[[吕氏春秋|吕氏春秋]]》[[高诱|高诱]]之注，并参照墨子诸多与[[鲁阳文君|鲁阳文君]]的对话，推定其为[[鲁阳|鲁阳]]人，即楚人。

==== 鲁人说 ====
===== 对宋人说与楚人说的批判 =====
* 宋人说和楚人说遭到[[梁启超|梁启超]]的批驳。
* 在《[[墨子学案|墨子学案]]》中，据《墨子·公输》“归而过宋”，力证非宋人。
* 据《墨子·贵义》“墨子南游于楚。”若墨子是[[楚国|楚国]]鲁阳人，则当为“游郢”。
* [[方授楚|方授楚]]《[[墨学源流|墨学源流]]》，[[张纯一|张纯一]]《[[墨子鲁人说|墨子鲁人说]]》都对墨子宋人说、楚鲁阳人说进行了批判。
* [[姜宝昌|姜宝昌]]《从<吕氏春秋>高诱注论到墨子为东鲁人而非西鲁人》：毕沅误读了高诱注。<ref>姜宝昌《从<吕氏春秋>高诱注论到墨子为东鲁人而非西鲁人》《经济参考报》2010年11月26日</ref>

===== 鲁人说的提出 =====

* 清代[[孙诒让|孙诒让]]作《[[墨子间诂|墨子间诂]]》，在附文《墨子传略》中，第一次提出墨子为鲁国人。其主要依据为： 
* 《墨子·贵义》“墨子自鲁即齐”。
* 《墨子·鲁问》“以迎墨子于鲁”。
* 《吕氏春秋·爱类》“公输般为云梯，欲以攻宋，墨子闻之，自鲁往”。

===== 滕州说的提出 =====
* 著名墨学研究学者[[张知寒|张知寒]]在《墨子原为[[滕州|滕州]]人》、《墨子里籍新探》等论文中进一步考证，墨子出生地应为古代[[邾国|邾国]]的“滥邑”（现[[山东|山东]][[滕州|滕州]]境内），滥邑后来归属鲁国<ref>张如寒等，济南：山东人民出版社 1996年8月《墨子里籍考论》</ref>。其主要依据有：
* 墨子是[[宋国|宋国]]贵族[[目夷|目夷]]之后，史载滕州东南有目夷亭。
* 墨子学说继承了[[邾娄文化|邾娄文化]]的传统，而邾娄号称“百工之乡”，而墨子生活习惯和科技成就与此密切相关。
* 该学说现得到绝大多数墨学研究者（[[匡亚明|匡亚明]]、[[任继愈|任继愈]]、[[杨向奎|杨向奎]]、[[张岱年|张岱年]]、[[季羡林|季羡林]]）的认同<ref>姜宝昌《从<吕氏春秋>高诱注论到墨子为东鲁人而非西鲁人》《经济参考报》2010年11月26日</ref>[http://finance.ifeng.com/roll/20101126/2954862.shtml]。

==== 其他学说 ====
* [[胡怀琛|胡怀琛]]撰《墨翟为印度人辨》、《墨子学辨》等论著，首次提出墨子为印度人。他認為墨並非姓，翟也不是姓，更不是名，而是「貊狄」或「蠻狄」之音轉；而且墨子長得黑，主張兼愛、非攻，顯示墨子應該是[[婆羅門|婆羅門]]。金祖同、陳盛良則說墨子可能是[[穆罕默德|穆罕默德]]以前的[[近東|近東]]人士，衛聚賢考證墨子應為[[印度人|印度人]]或[[阿拉伯人|阿拉伯人]]。

== 著作 ==
{{Main|墨子 (书)|l1=《墨子》}}
现存《[[墨子_(书)|墨子]]》五十三篇，由墨子的各代门徒逐渐增补而成，是研究墨子和[[墨家|墨家]]学说的基本材料。近代有大量學者如[[钱临照|钱临照]]鑽研《墨經》，發現《墨經》幾乎涵蓋了哲學、邏輯學、心理學、政治學、倫理學、教育學、自然科學等多個學科內容。

==思想==
{{Main|墨家}}
墨子主張尚儉節用，及兼愛非攻。墨子特別著重「利」和「功」，尤其是「國家百姓人民之利」，是評估一切價值的標準，合乎國家百姓人民之利的，才有價值；國家百姓人民之利，即是人民的「富庶」，凡能使人民富庶的事物，都是有用。其主張與[[效益主義|效益主義]]相近。對百姓沒有直接用處或有害的事物都要將其節約，反對奢侈<ref name="馮">{{cite book |author=馮友蘭|title=《中國哲學史》上冊|publisher=三聯書店（香港）有限公司|location=香港|year=1992|isbn=9620410149|language=zh-hans}}</ref>{{rp|89-92}}，主張節葬短喪，反對音樂；國家人民的最大問題，是國家人民的互相鬥爭，而互相鬥爭的起因，在於人民的互不相愛<ref name="馮"/>{{rp|93-95}}。墨子因而提出兼愛之說，兼愛能利人利己，為了阻止鬥爭，墨子提出「非攻」，反對戰爭，更付诸行动，研究防御戰術，曾与[[魯班|魯班]]於攻防戰中较量。人沒有相愛的本能，為了促使人民兼愛，墨子提出種種的制裁，主張有神明在上，賞賜兼愛者與懲罰不兼愛的人，鬼神同樣賞善罰惡，人亦須自求多福，不應垂手而祈求神靈保祐。賞罰都是個人自招的，並非命定<ref name="馮"/>{{rp|98-102}}。

墨子也著重政治的制裁，要使世界和平，人民安樂，不但需要有一帝在天上，亦需要有一帝於人間。應以天子的號令為是非的絕對標準，除此之外不應再有其他標準，在下者一律服從，而天子又以「兼相愛交相利」為令，那麼天下人都會「兼相愛交相利」。天子代天發號施令，人民只可服從天子，天子可說兼君主與教皇於一身<ref name="馮"/>{{rp|103-106}}。

《墨子．魯問》：「凡入國，必擇務而從事焉。國家昏亂，則語之尚賢、尚同；國家貧，則語之節葬、節用；國家熹音湛湎，則語之非樂、非命；國家淫僻無禮，則語之尊天、事鬼；國家務奪侵凌，則語之兼愛、非攻，故曰：擇務而從事焉。」

尚同：推舉賢人統治，由賢明的統治者來統一法的標準（法度）。

尚賢：尊重人才，把人才問題看成是國家政治中的頭等大事，主張賢者在位，能者稱職。

節葬：厚葬久喪會使國家貧窮、人民寡少、政治混亂，必須加以廢止。

節用：提倡極其刻苦樸素的生活，是極端救世的苦行學派，強調要做到不增加費用又要有利於人民。

非樂：從社會功利的角度出發，認為貴族所採用的禮樂制度，皆屬浪費而不實用，除加重人民負擔，也會影響國家經濟。

非命：不承認有「命定」的存在，認為人力定可勝過命運。如果人們相信命運，則只會使人們聽從命運的安排，進而懈志懶惰、喪失進取之心。

天志：認為天是有人格、有意志的，扮演主宰人倫秩序、施予賞善罰惡的能力與角色。

明鬼：認為鬼神是存在的，並操有賞善罰惡之權。如果人們深信鬼神的存在，並相信鬼神對人的賞罰，便可以改善人的行為。

兼愛：認為天下之所以亂，是因為人們之間缺乏無分別的愛，只要人們平等互愛、視人如己，就不會有不孝慈、盜賊、侵奪之事。

非攻：主張兼愛天下，放棄戰爭，以為攻伐是不義且不利的事。

== 科學研究 ==
墨子建立中国古代第一个逻辑学体系——墨家逻辑，主要以[[三物论|三物论]]为代表，三物分别为[[故|故]]、[[理|理]]、[[类|类]]。<ref>[http://www.cnki.com.cn/Article/CJFDTOTAL-ZDXB201001004.htm 墨家“三物逻辑”及其在《伤寒论》中的应用 杨武金 贺海峰] </ref>

墨子還是一個傑出的[[科學家|科學家]]，在[[力學|力學]]、[[幾何學|幾何學]]、[[代數學|代數學]]、[[光學|光學]]等方面，都有重大貢獻，爲當時諸子所望塵莫及。墨子在《墨经》精到地阐述了[[经典力学|经典力学]]中力的概念和力矩原理，提出“力”、“动”与“止”的定义。关于[[槓桿原理|槓桿原理]]，亦提出了“重”、“权”（力）、“称头”（重臂）、“称尾”（力臂）等概念。<ref>[http://news.btime.com/original/20160816/n385281.shtml 研究《墨经》的老教授说，这里有量子通信的基础]</ref>《墨经》中提出了“端”、“尺”、“区”、“穴”等概念，大致相当于近代几何学上的点、线、面、体。墨子和他的学生做了世界上最早的[[針孔成像|針孔成像]]实验，在当时就知道了光的直线传播<ref>《[[墨子_(書)|墨子]]·经说下》：“景光之人煦若射，下者之入也高，高者之入也下。”</ref>。他也提出了“[[基本粒子|粒子]]论”的雛形，关于“端”的论述，指出“端”是不占有空间的，是物体不可再細分的最小单位。

墨子的科学成就被中外众多学者称赞。[[蔡元培|蔡元培]]认为：“[[先秦|先秦]]唯墨子頗治[[科學|科學]]”。為向墨子在[[光學|光學]]領域的突出成就致敬，中國將全球首顆[[量子科学实验卫星|量子科学实验卫星]]命名為“墨子號”。<ref>{{cite web|title=发射在即！世界首颗量子卫星定名“墨子号”|url=http://news.sciencenet.cn/htmlnews/2016/8/353675.shtm|website=科学网|publisher=中国科学报社|accessdate=2016-08-15}}</ref>
