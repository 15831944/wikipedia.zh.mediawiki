《'''排球少年！！'''》（{{lang-ja|ハイキュー!!}}）是[[日本|日本]][[漫画家|漫画家]][[古舘春一|古舘春一]]的[[漫畫|漫畫]]作品。

== 作品概述 ==
=== 創作背景 ===
[[File:2006-08-19_1442.jpg|left]]
作者古館春一雖然是在[[岩手縣|岩手縣]]出身，但本作主要是以[[仙台市|仙台市]]、[[仙台站|仙台站]]以及{{tsl|ja|仙台市体育館}}之類以仙台地点作为创作背景。這是因為古館春一在高中畢業後，曾在仙台市的仙台設計專門學校內就讀，其前作《{{tsl|ja|詭弁学派、四ッ谷先輩の怪談。|诡辩学派、四谷前辈的怪谈}}》，由於当时的初代責任編輯住在[[宮城縣|宮城縣]]，因此以宮城縣作为该作的故事舞台<ref>[https://twitter.com/haikyu_com/status/261416116524572672 ハイキュー!!.com] - 公式ツイッター</ref>。

本作中对烏野高中和周邊環境的描写，是基于當地風貌所创作的<ref name="月バレ2014年3月号"/>。高中名称虽为虚构，但實際上是以當地學校作藍本的<ref>『ハイキュー!!』9巻130頁</ref>。本作有很多場景也跟岩手縣[[輕米町|輕米町]]十分相似<ref>{{Cite web|url=http://www.daily-tohoku.co.jp/news/kita_ar/20150505/p201505040P070860.html|title=「排球少年！！」的聖地？　以輕米作藍本？|accessdate=2015-05-10|deadurl=yes|archiveurl=https://web.archive.org/web/20150508152001/http://www.daily-tohoku.co.jp/news/kita_ar/20150505/p201505040P070860.html|archivedate=2015-05-08}}</ref>。

另外，{{tsl|ja|全日本中学校バレーボール選手権大会|全日本中學排球選拔大賽}}、{{tsl|ja|全日本バレーボール高等学校選手権大会|全日本排球高中選拔大賽}}（春高排球賽）、{{tsl|ja|全国高等学校総合体育大会バレーボール競技大会|全國高中統合體育大會排球競技大賽}}（{{tsl|ja|インターハイ|IH預賽}}）等實際存在的大賽，在作品中也有展现。

初中和高中時代，古館春一曾是排球部的中槌手<ref name="月バレ2014年3月号">[[日本文化出版|日本文化出版]]『[[月刊バレーボール|月刊バレーボール]]』2014年3月号</ref>。古館在連載之前就一直希望可以描繪一套以排球作為題材的作品，但是，責任編輯建议連載排球漫畫須有一定實力，因此古馆春一选择先创作另一套作品《四谷前輩》<ref name=situmonnjou>{{Cite web|url=http://mantan-web.jp/2013/03/18/20130317dog00m200032000c.html|title=マンガ質問状 ： 「ハイキュー!!」　推敲1年、作者渾身のバレーマンガ|work=[[まんたんブロード|MANTANWEB]]|date=2013-03-18|accessdate=2013-12-03}}</ref>。

在《四谷前輩》的連載完結後，古館春一以取得排球漫畫連載為目標，开始在《少年JUMP NEXT!》2011冬季增刊版上刊載單編，并接下来在《週刊少年JUMP》的2011年20、21合併號上继续刊載單編。雖然后来古馆春一取得連載的許可，但最初繪好的第1話因情節过于豐富，所以才通过逐集整理，變成現在这个樣子。單編刊載一年之后，經過细致推敲，本作品最终在《週刊少年JUMP》2012年12號開始正式連載<ref name=situmonnjou />。

=== 風格設計 ===
{{TransH}}
古館“超級大傳，完美的故事，以配合世爵折騰”，因為，啟動了“Haikyu！”<REF名=採訪> {{引用網絡|標題= Haikyu！ ！古館春一老師面試| URL = HTTP：//vbw.jp/5437/ | accessdate = 2015年5月10日}} </ REF>。

雖然這項運動的描述有誇張，它繪製的發揮，這是在遊戲中的基本現實中使用的基礎上。因此，作為一個男孩卡通“清醒”的風格。但是，只有與思想古館技能的英雄組合的匆忙，其實都是非寫實的手法描繪不可能取得<REF名=採訪/>。

規則說明和發揮容易在人在山谷不感興趣甚至閱讀，以便納入故事。旋轉被利用，以便提供一種宣洩和張力。關於發揮重要的話，象玩詞語是被選擇令人印象深刻的<REF名稱=面試/>。
{{TransF}}

== 故事大綱 ==
{{cquote|挡在我面前的，是一面很高很高的墙。墙的另一头，是怎样的风景呢？在我眼里会呈现出怎样的风景呢？顶端的景色，那是单凭我一个人绝对无法看到的景色。但是，不只是我一个人的话……}}
小時候，日向翔陽從電視上看見排球比賽，烏野高中的一名小個子在球場上的英姿，簡直就是個「小巨人」，對此非常深刻，並開始迷上排球，一直想成為「小巨人」一樣的人。初中時排球社不夠人數，只好默默等待。後來在新成員和友人的協助下，參與了初中首次也是唯一的比賽，卻不幸與最強的隊伍「北川第一中學」對決而落敗。翔陽為了能夠打出「勝利」的比賽，成為站在球場上最長時間的人，實現成為「小巨人」的排球夢想，決定考入烏野高中，卻遇上發誓要打倒的北川第一中學的影山飛雄？兩人在烏野高中互相競爭下，到底能在球場發生什麼火花呢？

== 用語 ==
;排球
{{main|排球}}
:1支隊伍由6人組成（6人制的情況下），隔著球場中央的一道網，2隊互相擊球。不讓球落地，通過最多3次觸球，轉化為進攻的運動。
;球員位置
{{main|排球#球员位置}}
:主攻手：簡稱WS（Wing Spiker），擔當攻擊的中心，是控制攻守平衡的多面手。
:攔網員：簡稱MB（Middle Blocker），又稱副攻手、快攻手或攔中，用攔網阻止對方的攻擊，主要用快攻來得分、進一步引誘對方扣殺。
:舉球員：簡稱S（Setter），又稱為二傳手，舉球給隊友進行扣殺，研擬攻擊戰術的指揮官。
:自由人：簡稱L（Libero），專門負責防守，不能進行扣球、發球等攻擊性動作，也不能從高於球網的位置將球擊回，與後衛交替進行防守。在比賽中可以多次進出球場，不受一般選手替換的規則限制。
;局點（Set Point）
:指只要多拿一分就能贏取「該局」。
;賽點（Match Point）
:指只要多拿一分就能贏取「該場比賽」。
;IH
:全國高中綜合體育大會。
;飄球
:發出不太會旋轉的球使空氣阻力增加，球的軌道會因空氣阻力飄忽不定，進而使對方接發球時容易誤判落球位置。可加入跳躍動作（跳飄）增加球的殺傷力。
;跳躍發球（跳發）
:利用跳躍提高發球時的擊球點，使球的路徑角度更為刁鑽。發出成功的跳發取決於起跳點、拋球位置及擊球位置三者合一，因此有一定難度，但威力強大。
;二傳攻擊
:直接由舉球員進行攻擊，一種出其不意的攻擊方式。但被料到的話，一定會被擋下來，風險很高。
;Pipe(管式)攻擊
:以前排的快攻為幌子，所發動的後排攻擊。
;側傳
:比直傳的線路較柔軟的托球。
;一人時間差
:讓別人看到起跳動作，然後稍微停一下，給攔網員一個假動作後再擊球。
;魚躍
:身體在半空中將球救起的魚躍救球動作。

;'''超手扣'''
:利用身高和高度優勢，將球從攔網者手的上空擊出的扣球。
;選擇性攔網
:看清楚托球的方向再起跳的攔網。
;「烏鴉」與「貓」的故事
:5年前6月IH預選，烏野高中在與白鳥澤學園經過激烈較量後落敗，錯過了全國大賽出場的機會。音駒高中也同樣，在東京代表隊的角逐中遺憾退場。接著在第二年3月，烏野作為宮城的代表，音駒高中作為東京第2的代表，在春季高校排球大會出場。但是，烏野在3回戰中落敗，雖然在四分之一決賽等待著烏野的晉級，但比賽卻沒能成行，音駒也在四分之一決賽時失利。無論進行多少次練習賽，就這樣在正式舞台上從未交鋒過的兩隊，教練相繼引退，在這期間，兩隊逐漸開始衰退。

== VOMIC ==
集英社「[[VOMIC|VOMIC]]」於2012年11月在日本JUMP專門情報番組「{{lang|ja|サキよみ ジャンBANG!}}」中播放<ref>[http://natalie.mu/comic/news/79033 「ハイキュー!!」がVOMIC化、声優陣にKENN＆前野智昭]</ref>，並於同年12月開始在集英社Voics Comics Station網站中發放。

== 電視動畫 ==
2013年10月，集英社公佈了本作品動畫化的消息。电视动画由制作过[[攻殼機動隊|攻殼機動隊]]系列、[[罪惡王冠|罪惡王冠]]与[[黑子的篮球|黑子的篮球]]的[[Production_I.G|Production I.G]]公司製作。第一季由[[滿仲勸|滿仲勸]]负责监督，由[[岸本卓|岸本卓]]担当系列构成与剧本编写，[[人物设计|人物设计]]由参与过[[玲音|玲音]]人物设计的[[岸田隆宏|岸田隆宏]]负责，千葉崇洋、海谷敏久为总作画监督，动作设定由甲斐泰之负责，佐藤真由美负责色彩设计，由中田祐美子担当摄影监督。{{tsl|ja|菊田浩巳}}担当音响监督，音乐由[[林友樹|林友樹]]与橘麻美两人负责。日本电视台[[每日放送|每日放送]]同时参与制作。

== 影響與評價 ==
此作品在[[讀賣新聞社|讀賣新聞社]]主辦的SUGOI JAPAN Award 2015「最想向海外介紹的日本流行文化」評選中獲得漫畫部門第3名<ref>{{cite web |language= ja |title= 世界に紹介したい作品に『まどマギ』国民投票7万7631票獲得 |url= http://www.oricon.co.jp/news/2049850/ |author= |work= ORICON STYLE |publisher= [[ORICON|ORICON]] |date=2015年3月12日  |accessdate=2015年3月25日  }}</ref><ref>{{cite web |language= ja |title= 国民投票 結果発表｜SUGOI JAPAN |url= http://sugoi-japan.jp/result.html |author=  |work=  |publisher= SUGOI JAPAN |date=  |accessdate= 2015年3月25日 |deadurl= yes |archiveurl= https://web.archive.org/web/20150323022823/http://sugoi-japan.jp/result.html |archivedate= 2015年3月23日 }}</ref>。

在2014年10月发售的单行本13卷(首周销量34.7万本)与小说第4卷(7.0万本)，各自获得了[[公信榜|公信榜]]漫画、小说每周排行榜的的第一名，成功制霸了两大主要排行榜<ref>{{Cite web |url=http://www.oricon.co.jp/news/2043098/full/|title=【オリコン】『ハイキュー!!』コミック&ノベライズがW首位 史上4作目 |work=[[オリコン|オリコン]] |accessdate=2015-03-01}}</ref>。在2015年5月，单行本16卷（周销量38.5万本）与小说第5卷（周销量6.9万本）再次登顶于[[公信榜|公信榜]]漫画、小说排行榜，闪耀地独霸了两个排行榜的冠军<ref>{{Cite web |url=http://www.oricon.co.jp/news/2052498/full/|title=【オリコン】『ハイキュー!!』コミック&ノベライズが2度目のW首位を達成 |work=[[オリコン|オリコン]] |accessdate=2015-05-07}}</ref>。在动画放送的2014年，被称为是扩大漫画单行本市场的三部作品之一。<ref>{{Cite web |url= http://www.oricon.co.jp/news/2047585/full/|title=【年間書籍市場】コミック売上額が過去最高2815.1億円 前年比104%で2年連続増  |work=[[オリコン|オリコン]] |accessdate=2015-03-01}}</ref>。

到2014年10月为止，累计发行数达1000万本<ref>『ハイキュー!!』13巻販促ポスター</ref>。

除此之外，该漫画有数次在漫画排行榜中名列前茅的好成绩<ref>{{Cite web |url= http://natalie.mu/comic/news/112477|title=女子が夢中な男子マンガBest20、連載中＆完結の2部門|work=コミックナタリー |accessdate=2015-03-01}}</ref><ref>{{Cite web |url= http://natalie.mu/comic/news/134323|title=エンタミクス「2014年コレ読んで漫画ランキング」1位は「七つの大罪」|work=コミックナタリー |accessdate=2015-03-01}}</ref><ref>{{Cite web |url=http://natalie.mu/comic/news/133048|title=ダ・ヴィンチの「BOOK OF THE YEAR」1位は2年連続で「進撃の巨人」|work=コミックナタリー |accessdate=2015-03-01}}</ref>。

2016年秋季番中繼《[[勇利!!!_on_ICE|勇利!!! on ICE]]》後為[[推特|推特]]趨勢中該季第2熱門的動畫。<ref>{{cite web|language=en|author=Rafael Antonio Pineda|publisher=Anime News Network|accessdate=2016年12月23日|date=2016年12月21日|url=http://www.animenewsnetwork.com/interest/2016-12-21/yuri-on-ice-dominates-fall-anime-on-twitter-rankings/.110186|title=Yuri!!! on Ice Dominates Fall Anime on Twitter Rankings}}</ref>第3季第1卷初動10107張成為秋季番首部破萬的動畫，雖然略不如第1季的24711張和第2季的16245張，但表現依然相對出色。<ref>{{cite web|language=zh-tw|author=曈妍|publisher=宅宅新聞|accessdate=2016年12月23日|date=2016年12月17日|url=https://news.gamme.com.tw/1463649|title=宅腐女最狂《排球少年！！》第三季BD/DVD首卷初動銷售破萬}}</ref>
