《'''億萬富翁少女'''》（{{lang-ja|ビリオネアガール}}）是日本作家[[支倉凍砂|支倉凍砂]]原案、[[桂明日香|桂明日香]]所畫的漫畫作品。以[[股票|股票]]投資累積百億財產的少女與靠[[打工|打工]]度日的大學生兩人間的故事。2009年起在《[[月刊Afternoon|月刊Afternoon]]》連載，單行本由[[講談社|講談社]]發行。

== 簡介 ==
大學生高遠惠受親戚大貫[[教授|教授]]的請託，為教授的[[姪女|姪女]]藤岡紫擔任數學[[家教|家教]]，到了她的[[豪宅|豪宅]]才知道她以股票交易已有170億日元的財產。但藤岡紫卻覺得寂寞。

== 登場角色 ==
*高遠 惠
:是一個平凡度日的男大學生。因大貫教授介紹家教工作因而認識藤岡紫。

*藤岡 紫
:一位年紀輕輕就靠股票交易賺進身價170億的平凡少女。因為在同學的電話裡說是大學生所以想要知道大學的事情，在叔叔大貫教授的介紹下認識高遠惠。有很豐富的財經知識，卻個性內向、涉世未深，外貌雖美但不擅於衣著打扮。因家人不認同她而搬到豪宅獨居，操盤室內堆滿[[垃圾|垃圾]]。

*大貫教授
:與高遠惠及藤岡紫有親戚關係，由於藤岡紫的家人不認同她靠股票賺錢，因此主動要求擔任她的[[監護人|監護人]]。由於工作繁忙且年紀與她相差甚遠，因此以當家教的名義請高遠惠幫忙照顧藤岡紫。

== 專業用語 ==
*當沖交易
:[[當沖|當沖交易]]指資金投入市場的時間短暫，只尋求找出相對的一段大幅度波動作套利，有可能僅數秒或數小時就平倉而不隔夜的交易方式。以這種交易方式投資的人稱為[[當沖客|當沖客]]。
*FX
:FX（Foreign exchange market）稱做[[外匯市場|外匯市場]]，是指用作為外匯交易的市場，其特色是在此市場上有多於一種的貨幣，被交易為另一種[[貨幣|貨幣]]，或稱[[外幣|外幣]]。按交易契约种类，有[[即期|即期]]、[[掉期|掉期]]和[[遠期合約|遠期合約]]等市場存在。
*資本主義
:[[資本主義|資本主義]]指的是一種[[經濟學|經濟學]]或[[經濟社會學|經濟社會學]]的制度，在這樣的制度下絕大部分的生产资料都歸私人所有，並藉著僱傭或勞動的手段以生產工具創造利潤。
*B-S模型
:[[布萊克-舒爾茲模型|布萊克-舒爾茲模型]]（Black-Scholes Model，簡稱B-S模型）是一种为[[期权|期权]]或[[权证|权证]]等[[金融衍生工具|金融衍生工具]]定价的数学模型，由美國經濟學家[[麥倫·休斯|麥倫·休斯]]（Myron Scholes）與[[費雪·布萊克|費雪·布萊克]]（Fischer Black）所最先提出，并由[[罗伯特·墨顿|罗伯特·墨顿]]（Robert C. Merton）完善。
*大學基金
:[[大學基金|大學基金]]是指為教育而成立的基金，往往會以投資人的名字命名。教育基金在市場上取得收益後，會為大學的財政或獎學金等提供支持。
*讓渡益稅
:[[讓渡益稅|讓渡益稅]]是日本的稅種，是對賣出股票或[[土地|土地]]等資產所獲得的利益進行徵收的稅賦。[[證券交易所得稅|證券交易所得稅]]（[[證所稅|證所稅]]）即為讓渡益稅的一種。作品裡的數據是舊的，現行的讓渡益稅率為20%。
