'''根号'''（{{Lang|en|radical symbol}}），即'''[[方根|方根]]'''的[[符號|符號]]，使用“√”表示。

== 用例 ==
=== 平方根 ===
在√下以橫式書寫，必要時須加強橫線長度或以不同大小的根號來表示巢狀算式。
:<math> \sqrt{2\,},\quad \sqrt{x\,},\quad \sqrt{x + y + z + w + \cdots \;},\quad \sqrt{ x + \sqrt{ x + \sqrt{ x + \sqrt{ x + \cdots\ }\;}\;}\,} </math>

[[運算次序|運算次序]]因橫線所及之處而有所不同。在某些印刷情況下，在後方有其他算式時不易判斷運算次序，因此可使用乘號或是括號來分離。
:<math> \sqrt{x}\,y = \left(\sqrt{x}\right)y = \sqrt{x}\cdot y \ne \sqrt {xy} </math>

非負數的平方根代表該數的 1/2 次方，根號有時會以[[冪|冪]]來表示。
:<math> \sqrt x = x ^{1/2} </math>

=== 方根 ===
{{Main|方根}}
√左上有時會加上一個小數字來表示方根。通常只在 3 以上時才會標示，而非整數的情況通常以[[冪|冪次]]表示。
:<math>\begin{align} \sqrt[3] x &= x ^{1/3} \\
\sqrt[4] x &= x^{1/4} \\
&\vdots\\
\sqrt[n] x &= x ^{1/n} \end{align}</math>
