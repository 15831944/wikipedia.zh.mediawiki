'''根管治疗术''' (Root Canal Therapy)，又称'''牙髓治疗''' (Endodontic Therapy)，一般較常聽到的俗稱為'''抽神經'''，粵語稱'''杜牙根'''，是[[牙醫學|牙醫學]]中治疗[[牙髓|牙髓]]坏死和牙根感染的一种[[手术|手术]]<ref name="toothclub">[http://www.toothclub.gov.hk/chi/adu_01_06_08.html 成人口腔護理區 - 根管治療]，[[衞生署_(香港)|衞生署 (香港)]] </ref>。對於未能以一般補牙修復的蛀牙，根管治療可盡量保留自然牙，因而与[[拔牙术|拔牙术]]互补。

==牙齿结构==
牙齿中间空洞的部分包含称为[[牙髓|牙髓]]的软组织。空洞上部宽阔，称为[[牙髓腔|牙髓腔]]，下部有管状的[[根管|根管]]，由之导出牙神经和营养神经的血管。人类一般每颗牙齿有1-4个根管，后部的牙齿根管最多。

如果蛀牙沒有及時處理，蛀牙的細菌會深入牙髓，導致牙髓发生感染，除会造成疼痛，可進而引發颌骨感染，最终牙齿因为牙神经的死亡而变得脆弱。由於牙髓受感染死亡後不能重生，所以在根管治療時必須把牙髓徹底移除，並要消毒殺菌，及以填充物密封。

==过程==

# 拍[[X射线|X射线照片]]，确定患处结构和手术方案。
# 若活性牙髓，則會施以[[局部麻醉|局部麻醉]]，若牙髓已壞死，則不一定，視患者需求。
# 去除腐质并向牙髓腔扩展。
# 打开牙髓腔，取出發炎或壞死的牙髓。
# 用根管銼盡量去除感染源並扩大根管以利封填。
# 用消毒液（例如[[次氯酸钠|次氯酸钠]]或3%[[过氧化氢|过氧化氢]]）和[[生理盐水|生理盐水]]交替冲洗根管内部。
# 用牙胶尖充填根管，盡量保持根管处于无菌封闭状态。
# 視齒質損失的程度及牙位用[[樹脂|樹脂]]或{{le|玻璃离子体|Glass_ionomer_cement}}、{{le|银汞|Amalgam (dentistry)}}等材料充填牙齿的窩洞。
# 由於移除牙髓後的牙齒變得脆弱，一般需要安装[[牙冠_(修复体)|人造牙冠]]保護根管治療後的牙齒，防止出現破裂。

==缺点==
* 手术繁琐，一般要2-4次就诊才能完成。病人要承受的痛苦也较大。
* 价格昂贵，例如在美国一颗牙齿的手术一般需要1,000美元，如加上牙冠则可达2,000美元。
* 可能失败，消毒不彻底导致牙髓腔再次感染，导致需要[[拔牙术|拔牙]]。具有4根根管的臼齿，第四根根管容易被错过，导致失败。另外形状异常的根管，消毒充填过程中也可能不彻底。
* 局部麻醉有時不完全，牙根最深處仍會留有痛覺，這時就必須考驗牙醫的技術。

==治療失敗==
根管治疗手术失败后不一定需要拔牙。

根管治疗迄今仍是牙髓和根尖周病的主要方法。因此人们非常关注根管治疗的疗效，尤其是一些经过高标准治疗仍然失败的原因一直深受医师的关注。究其原因归纳起来有以下几方面：
# 解剖学因素，包括侧支根管、副根管、根尖分歧和根尖分叉等；
# 治疗过程中的医源性因素如根管侧穿未发现，根管超填，冠部封闭状态欠佳和治疗过程中纸尖、棉捻的遗留等；
# 根管内微生物感染未得到控制，近年来根管外的感染尤其受到学者们的关注。

目前对于根管失败的再治疗，首选仍是保守性根管治疗，即通过分析失败原因选择合理的根管治疗手段，如超声、激光、微波等，选择对根管细菌敏感的药物处理根管，采用垂直的热加压充填技术充填侧支根管、副根管等。在保守治疗无效的情况下，可以考虑根尖手术，但要严格地掌握适应征和禁忌症。根管治疗失败的再治疗成功率，采用保守治疗约为66%，失败率为23%，根尖手术成功率为59%，失败率为19%，二者之间无显著性差异。
