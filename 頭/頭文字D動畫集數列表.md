== 頭文字D Extra Stage ==
番外篇，目前共兩集。動畫製作由A.C.G.T製作。
* Extra Stage （OVA系列第1部）
: 片名副題「'''衝擊之藍...（{{lang|ja|インパクトブルーの彼方に...}}）'''」。由佐滕真子、沙雪、庄司慎吾和中里毅主演。2000年2月22日發售。
* Extra Stage2（OVA系列第2部）
: 片名副題「'''～啟程之綠～（{{lang|ja|旅立ちのグリーン}}）'''」。由佐滕真子、沙雪和池谷浩一郎主演，內容為第四部第二集和第三集之間。2008年12月5日發售。

== 頭文字D Battle Stage ==
競賽畫面總匯，目前共兩集。
*Battle Stage
:First Stage～Third Stage的競賽畫面總匯，其中First Stage的賽車畫面，在Battle Stage中以Third Stage所使用的軟體重新製作，3D構圖更為精良，而Second Stage和Third Stage部份則是直接剪入，並多出了原作漫畫和動畫皆未提及的啟介與清次於赤城山的賽事，以及漫畫未收錄、動畫未完整呈現的拓海與涼介的賽事。全1集，艾迴出版。
*Battle Stage2
:Fourth Stage的競賽畫面總匯，並完整播出動畫略過的啟介與川井淳郎還有微笑酒井的比賽。另外路面表現也有若干異動。

== 頭文字D to the Next Stage 〜計畫D的猜測〜 ==
Fourth Stage播映前頭文字D的介紹節目，又稱總集編，主要內容為第一季至第三季的劇情節要，另包含對土屋圭市、宇佐美廉等幕後人員的訪問及製作片段。
