《'''頭條新聞'''》（{{lang-en|'''Headliner'''}}）是[[香港特別行政區政府|香港政府]]轄下的[[香港電台|香港電台]]電視部公共事務組製作的政治時事節目，於1989年4月4日在[[亞洲電視本港台|亞洲電視本港台]]啟播，當時正值[[中国民主運動|中国民主運動]]的時期，節目按季度調配一直斷續播放至今，通常於星期五或六晚上20:30播出。

《頭條新聞》在2008年8月1日起重回亞視本港台播出。其後有指由2013年3月起，頭條新聞固定在[[無綫電視|無綫電視]][[翡翠台|翡翠台]]播放，不會再與[[警訊|警訊]]對調<ref>[http://hk.apple.nextmedia.com/news/art/20130313/18193598 鄧忍光欲炒《頭條新聞》主持 施永遠：感到處長要求我做政治任務] [[蘋果日報_(香港)|蘋果日報 (香港)]] 2013年3月13日</ref><ref>[http://hk.apple.nextmedia.com/news/art/20130314/18194568 港台重臣炮轟處長連小學生也不如不信任鄧忍光運動 如箭在弦] 蘋果日報 2013年3月14日</ref><ref>[http://hk.apple.nextmedia.com/news/art/20130315/18195840 鄧忍光犯眾怒 開會謀計息風波 新聞部高層憤而離場] 蘋果日報 2013年3月15日</ref>。2014年4月25日，頭條新聞再度與[[警訊|警訊]]對調，改於[[亞洲電視本港台|亞洲電視本港台]]播出，結果因[[慣性收視|慣性收視]]影響而大幅度下跌。2015年2月13日，頭條新聞再回流到[[翡翠台|翡翠台]]播放，收視有所回升。

== 歷史與發展 ==
《頭條新聞》於1989年4月4日在[[亞洲電視本港台|亞洲電視本港台]]啟播，當時正值[[中国民主運動|中国民主運動]]的時期，節目一直斷續播放至今，通常於星期五或六晚上20:30播出。第一代主持是[[吳明林|吳明林]]與[[黃志強|黃志強]]。<ref>[http://www.liberalstudies.tv/headliner20/pdf/ls_headliner20_02.pdf 香港電台「通識網」教學建議]</ref><ref>[https://www.facebook.com/RTHKheadliner/videos/1048876828512072/ 頭條新聞Facebook：27年前今天，《頭條新聞》在亞洲電視首播；27年後的今天，亞洲電視已然落幕。]</ref>

《頭條新聞》為「個人意見節目」（2003年以後在節目開頭標明），以輕鬆幽默、嬉笑怒罵方式討論時事。節目有多個環節；如將時事片段配上粵語歌曲以諷刺時弊、另有主持人扮演各種時事人物，當中對白充滿諷刺意味，[[香港特別行政區政府|香港特別行政區政府]]高官是其經常挖苦對象。<ref>[http://www.youtube.com/watch?v=-gOIcoUmjpE&feature=relmfu 在2012年6月23日播出的《頭條新聞》片段]</ref>

《頭條新聞》自啟播至今，為香港電台的王牌節目，獲得不少觀眾的贊同及欣賞，部分視此節目為[[香港電台|香港電台]]編採自主和[[新聞自由|新聞自由]]的象徵。節目內容及新聞題材選擇時被指比較偏向[[民主派_(香港)|民主派]]，客觀性亦曾因此引起部分[[建制派_(香港)|建制派]]政客、政府高官不滿，並點名抨擊及不接受節目訪問。

《頭條新聞》因為涉及敏感的時事題材，時常被[[中國大陸|中國大陸]]封殺，當在無線電視翡翠台直播時，[[廣東有線|廣東有線]]會用預先錄製節目覆蓋部分或者全部內容。

== 爭議 ==
《頭條新聞》節目中主持人經常以幽默對答揶揄香港政府的施政，節目受到親中共人士批評，例如1998年3月（又說1996年1月），前[[中國人民政治協商會議全國委員會香港地區委員|全國政協委員]][[徐四民|徐四民]]曾指該節目「陰陽怪氣」，事件引起爭議。亦有親中共人士指該節目與香港電台角色不符，因香港電台是香港政府轄下的公營電台。

2001年10月，時任[[香港特別行政區行政長官|行政長官]]的[[董建華|董建華]]發表《[[香港施政報告|施政報告]]》後，《頭條新聞》節目中其中一個環節，主持人以阿富汗塔利班武裝份子打扮諷刺當時的董建華的《施政報告》為《施捨報告》，諷刺政府的政策小恩小惠，又把特區政府比喻為塔利班政權，此番言論被董建華批評為「低級趣味」，事件再次引起社會不同聲音的爭議。

另一觀點認為《頭條新聞》以嬉笑怒罵方式跟社會、政治、時事人物開一開玩笑，是多元社會的特色，各界應持鼓勵包容的態度，尊重不同的創作及市民的選擇。<ref>[http://news.sina.com.hk/cgi-bin/nw/show.cgi/469/3/1/1095649/1.html 明報：《頭條新聞》屹立20年 延續傳奇]</ref>

《頭條新聞》是香港電台編輯獨立重要指標，2010年8月28日，有報導指廣播處長黃華麒因不能接受主持批評政府的手法，提出不再與現任主持吳志森和曾志豪續約，民主派立法會議員[[何秀蘭|何秀蘭]]認為若消息屬實，黃華麒再沒資格做廣播處長，要求他公開交代，而黃華麒則不評論事件。<ref>[http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20100828&sec_id=4104&subsec_id=12731&art_id=14391917 黃華麒擬幫政府拔眼中釘整肅《頭條新聞》點名要換兩主持] 蘋果日報 2010年08月28日</ref>

此外，為了可以爭取繼續在香港電台及無線電視中播放，近年頭條新聞的編採及內容已漸趨溫和。

== 節目被無綫電視抽起事件 ==
{{seealso|中共總書記視察香港|習近平總書記視察香港}}
2017年6月30日，[[無綫電視|無綫電視]]原定傍晚於[[翡翠台|翡翠台]]播放的《[[頭條新聞_(香港電台)|頭條新聞]]》節目突然抽起。改播[[黨和國家領導人|國家領導人]][[習近平總書記視察香港|習近平總書記視察香港]]片段及風水節目《[[順峰順水|順峰順水]]》。港台表示在節目播出前8分鐘突然收到無綫電話，指稱「新聞部」有突發新聞，要求港台節目讓路。無綫6時24分向港台發出電郵，要求港台同意轉換節目時段。不過最後未有播出突發新聞，只播放習近平訪港片段和風水節目。《頭條新聞》延至深夜12時半於[[J5|J5]]頻道播出。

香港電台機構傳訊總監伍曼儀指，沒有簽署接受節目調動的文件，並向無綫電視投訴和要求解釋對事件感到awfully displeased（非常不滿）。[[通訊局|通訊局]]在7月4日已收到133宗相關投訴。<ref>{{cite news|title=無綫突抽調《頭條新聞》 通訊局接逾百宗投訴|url=https://topick.hket.com/article/1850080/%E7%84%A1%E7%B6%AB%E7%AA%81%E6%8A%BD%E8%AA%BF%E3%80%8A%E9%A0%AD%E6%A2%9D%E6%96%B0%E8%81%9E%E3%80%8B%20%E9%80%9A%E8%A8%8A%E5%B1%80%E6%8E%A5%E9%80%BE%E7%99%BE%E5%AE%97%E6%8A%95%E8%A8%B4|accessdate=2017-07-05|publisher=香港經濟日報|date=2017-07-05}}</ref>

無綫在7月5日下午以逾700字回覆港台，表明[[習近平|習近平]]總書記首次以[[中共總書記視察香港|中共領導人身分訪港]]是重大新聞。指出「由於講話全文不足15分鐘，只好以一些備用節目（《順峰順水》）去填補剩餘時段」。又點名香港電台傳訊總監伍曼儀罔顧事實，「如伍女士認為國家主席對香港各界人士講話全文不是新聞，或不及《頭條新聞》重要，乃罔顧事實，亦不是專業新聞或廣播人應有的說話」。最後指出播放港台節目是「歷史遺留」，至今已不合時宜，應盡快終止這安排。<ref>{{cite news|title=突抽《頭條新聞》被質疑　無綫700字反駁　指播港台片已不合時|url=https://news.mingpao.com/ins/instantnews/web_tc/article/20170705/s00001/1499241574048|accessdate=2017-07-05|publisher=明報|date=2017-07-05}}</ref>

伍曼儀承認及尊重習近平訪港是重大新聞，但指無綫播出的是預錄片段，而非突發新聞。<ref>{{Cite news|url=http://news.rthk.hk/rthk/ch/component/k2/1340303-20170705.htm|title=伍曼儀承認主席來港是重大新聞　惟無綫播出片段非突發|author=|date=2017-07-05|work=香港電台|accessdate=2017-07-14|language=zh-hk}}</ref>

== 同名電台節目 ==
[[香港電台第一台|香港電台第一台]]在1990年代初，於逢星期二至五在晚上十時至十一時時段（賽馬夜除外，星期一則播放一個名為《政治騷小小》的節目，專門講述如政黨內訌、政客動態等政壇八卦和邀請政黨領袖演唱及參演廣播劇等，[[區麗雅|區麗雅]]主持），曾有一個同樣稱為《頭條新聞》的[[烽煙節目|烽煙節目]]，但與電視版的形式並無關聯，主持先後有港台公共事務組組長[[陳耀華|陳耀華]]、資深記者[[吳明林|吳明林]]、《[[時代論壇|亞洲基督教時代論壇報]]》總編輯[[李錦洪|李錦洪]]、學者[[洪清田|洪清田]]等，並固定在節目開始前播放由[[鍾偉明|鍾偉明]]播讀[[粵語|粵語]]引言的[[越南語|越南語]]對港府越南難民甄別政策之廣播（[[北漏洞拉|北漏洞拉]]）。此節目大約在[[香港回歸|香港主權移交]]前中止。類似的烽煙節目有《[[千禧年代_(電台節目)|千禧年代]]》及《[[自由風自由Phone|自由風自由Phone]]》。
