'''咸興市'''（{{kor|諺=함흥시|漢=咸興市|羅=Hamhŭng si|f=y}}）位於[[咸鏡南道|咸鏡南道]]中部海邊，位於[[朝鮮半島|朝鮮半島]]東海岸[[咸鏡灣|咸鏡灣]]畔，是僅次於[[平壤|平壤]]的[[朝鮮|朝鮮]]第二大城市，擁有化學、紡織、機械等大規模的重工業和輕工業基地，有許多觀光名勝。 亦是[[咸鏡南道|咸鏡南道]]的道府所在地。有人說這裡是阿蘭弗勸[[解夫婁|解夫婁]]立國的地方（即[[東扶餘|東扶餘]]）。也是[[李成桂|李成桂]]龙兴之地。

==历史==
这里最早是[[高句麗|高句麗]]的地方，唐代是[[渤海國|渤海國]]五京之一的南京南海府。高麗時是女真族地方，元代是[[雙城總管府|雙城總管府]]，1356年置咸州，李成桂逊位后回到此地终老，朝鲜王朝把此地与[[鏡城郡|鏡城]]合併为[[咸鏡道|咸鏡道]]。

== 行政區劃 ==
* 咸興市直轄（56洞・1勞動者區・14里）
* {{link-jp|興南區域|興南区域}}（{{lang|ko|흥남구역}}）（43洞・1勞動者區・5里）：沿海港灣的化學工業地帶，咸興的中心。<br>1960年設咸興直轄市（1970年降格為市），興南市併入重組並改稱興南區域，2001年復設興南市，全市2005年再併入咸興市改為區域至今。

=== 已撤銷的行政區 ===
* [[海岸區域|海岸區域]]（{{lang|ko|해안구역}}）：機械工業的中心地，1990年前為龍城區域。2001年廢止，原轄共11洞2里行政區併入興南市。
* [[沙浦區域|沙浦區域]]（{{lang|ko|사포구역}}）：化學纖維的生產基地。1974年將本宮區域併入。2001年廢止，所轄10洞行政區直屬咸興市、餘行政區併入興南市。
* [[城川江區域|城川江區域]]（{{lang|ko|성천강구역}}）：紡織中心，1990年前為城川區域。2001年廢止，原轄共18洞行政區直屬咸興市。
* [[東興山區域|東興山區域]]（{{lang|ko|동흥산구역}}）：1977年前為盤龍區域。2001年廢止，原轄共16洞2里行政區直屬咸興市。
* [[會上區域|會上區域]]（{{lang|ko|회상구역}}）：1974年將德山郡併入。2001年廢止，原轄共12洞11里行政區直屬咸興市。
* [[興德區域|興德區域]]（{{lang|ko|흥덕구역}}）：2001年廢止，原轄共13洞1里行政區併入興南市。

== 港口 ==
[[興南港|興南港]](Hungnam Port)：平均水深高達8.6公呎，岸線全長約2,110公呎，港口受長達837公呎的人造防波堤保護。它設有4個泊船位置，配備多臺10公噸橋式起重機、多臺5公噸港口起重機、多臺貨物裝瀉起重機、170公呎的運輸帶等。新建的處理氧化鎂粉煤渣碼頭, 裝備以100 噸傾瀉貨車, 長達928公呎的運輸帶能每小時運載750噸。 它設有佔地約80,000平方公呎的7個貨倉及露天貨場，其中一個是供氧化鎂粉煤渣專用。每年吞吐量達3.8萬公噸，主要處理：氧化鎂粉煤渣、鹽、五穀、焦炭、礦物及金屬等商品。在不久的未來，建造當中第5及第14號碼頭能使興南港口的每年吞吐量激增至10萬公噸。

==经济==
咸兴是朝鲜化学工业的一个中心。同时其也是朝鲜对外贸易重要的一个港口及工业城市。纺织业，尤其是维尼龙、金属器皿、机械、精炼油与食品加工等都是咸兴主要的工业种类。

=== 畜牧業 ===
* '''青年山羊牧場'''：位於上昌洞之「天義山」(Chonju Peak)，砌築60,000平方公呎的[[圍牆|圍牆]]、許多[[橋樑|橋樑]]、約60公里的現代化[[山羊|山羊]][[牧場|牧場]]，大量生產富有營養的各種乳製品。

== 觀光熱點 ==
* '''東興山'''：位於東興山區，最高峰是海拔319米。這裡有「九天閣」及「宣化堂」等古代建築和近期修建的「解放亭」等朝鮮式亭臺樓閣。
** 「九天閣」坐落在東興山頂，在公元1108年修築的的咸興城牆的北將臺，站使用來作戰爭指揮所。
** 「宣化堂」依傍著「東興山」，坐北向南，初建於公元1416年，重建於公元1764年。
* '''[[周恩來|周恩來]]同志的銅像'''：，坐落在[[朝鮮|朝鮮]]全國化學肥料生產基地的「興南化肥聯合企業」的[[廣場|廣場]]。咸興市的[[民眾|民眾]]經常來這裡向[[銅像|銅像]]獻花。 
* '''黃草嶺真興王巡獵碑'''：紀念新羅真興王出行而立石碑，建於公元568年，原位於咸鏡南道咸興均黃草嶺，在公元1852年，依觀察使：尹定鉉先生的意見遷到了黃草嶺下的「真興里」現址。 該[[花崗岩|花崗岩]]造的石碑高：151.1厘米；闊：46.9厘米；深：24.5-48.5厘米。
* [[麻田遊園地|麻田遊園地]](Majon Pleasure Ground)：[[國際|國際]]聞名的[[海浴場|海浴場]]。

== 風味菜肴 ==
* 咸興[[冷麵|冷麵]]：以[[馬鈴薯|馬鈴薯]]為原料麵條裡放上甜酸脆爽的鱔魚、食鹽、芝麻、香油拌勻後再加上酸甜的佐料醬油食用，莫不令人舌口生津，食慾大振。

== 交通 ==
咸兴是一个朝鲜的交通枢纽，設有完善的電氣化鐵路及高速公路：
* 咸興市至[[平壤|平壤]]市約315公里；
* 咸興市至[[元山|元山]]市約115公里；
* 咸興市至[[咸南|咸南]]港約17公里。
