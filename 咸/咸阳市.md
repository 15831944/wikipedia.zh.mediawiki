'''咸阳市'''是[[中华人民共和国|中华人民共和国]][[陕西省|陕西省]]下辖的[[地级市|地级市]]，位于[[关中平原|关中平原]]中部、[[渭河|渭河]]沿岸；距[[西安|西安]]25千米；[[黄河|黄河]]的第一大支流——[[渭河|渭河]]流经市区。咸阳辖区面积10196平方千米，总人口504万人<ref name="咸阳市行政区划地图">{{cite web|url=http://www.kemap.net/map/china/shxi/5617.html|title=咸阳市行政区划地图|accessdate=2009-05-22|deadurl=yes|archiveurl=https://web.archive.org/web/20100616145050/http://kemap.net/map/china/shxi/5617.html|archivedate=2010-06-16}}</ref>，下辖3个[[市辖区|市辖区]]、2个[[县级市|县级市]]和9个[[县|县]]，拥有一个[[国家级高新技术产业开发区|国家级高新技术产业开发区]]和一个国家高新技术创业服务中心；中国第七个[[国家级新区|国家级新区]]、首个以创新城市发展方式为主题的国家级新区[[西咸新区|西咸新区]]的大部分（包括秦汉新城、沣西新城、[[空港新城|空港新城]]和泾河新城的全部以及沣东新城的部分区域）均位于咸阳市境内；中国目前唯一的农业高新技术产业开发区也于1997年在咸阳市[[杨陵区|杨陵区]]成立。

咸陽是中国著名古都，是中国历史上第一个中央集权王朝——[[秦朝|秦朝]]的都城所在地，在[[周朝|周朝]]、[[西汉|西汉]]和[[唐朝|唐朝]]均为京畿重地。有两千多年的建城史的咸阳，是中国国家级历史文化名城、中国优秀旅游城市、中国甲级对外开放城市、全国双拥模范城、国家卫生城市；2004年，被评为首届中国魅力城市；2005年当选中国十佳宜居地级市。2014年被授予国家园林城市。<ref>{{cite web|url=http://news.shaanxi.gov.cn/shownews.asp?id=34449|title=咸陽市榮獲“中國魅力城市”稱號|accessdate=2009-05-22|deadurl=yes|archiveurl=https://web.archive.org/web/20041012003824/http://news.shaanxi.gov.cn/shownews.asp?id=34449|archivedate=2004-10-12}}</ref>

咸阳市位于[[关中-天水经济区|关中-天水经济区]]核心区域，是[[国家发改委|国家发改委]]确定的西安（咸阳）大都市的两大组成部分之一。

== 历史 ==
据[[李吉甫|李吉甫]]《元和郡县志》解释：“山南曰‘阳’，水北也称‘阳’”。而咸阳正是地处[[九嵕山|九嵕山]]之南，[[渭水|渭水]]之北，山水俱阳，“咸”即为“皆”之意，故名“咸阳”。另据《[[史记|史记]]》和秦都咸阳出土的陶文，以为[[商鞅|商鞅]]在此曾置“咸亭”、“阳里”，[[秦孝公|秦孝公]]将两名合一，为“咸阳”。

咸阳因历史悠久，[[文物|文物]][[遗址|遗址]]众多被人们称为“天然[[历史|历史]][[博物馆|博物馆]]”。它的[[历史|历史]]可以上溯到4000多年以前的[[新石器时代|新石器时代]]，曾是11个王朝的[[京畿|京畿]]属地，历史文物方迹遍布全市。

前350年，[[秦孝公|秦孝公]]由[[栎阳|栎阳]]迁都咸阳，在此营筑翼阙及宫殿。[[秦始皇|秦始皇]]仿建六国宫殿，使[[咸阳城|咸阳城]]成为规模恢宏的帝都。后置扶风郡。西汉建立（公元前206年）之时，[[咸陽縣|咸阳]]先后改名为[[新城县|新城]]和[[渭城县|渭城]]。在今咸阳原上，因[[西汉|西汉]][[五陵|五陵]]置有[[陵邑|陵邑]]，故有“五陵原”之称。[[西晋|西晋]]析置[[始平郡|始平郡]]。[[前秦|前秦]]置[[咸阳郡|咸阳郡]]。[[北魏|北魏]]，并咸阳县于泾阳县。[[隋朝|隋朝]]并入[[京兆郡|京兆郡]]。[[唐朝|唐朝]]重置咸阳县，建[[唐咸阳城|咸阳城]]。[[武则天|武则天]]因其母[[孝明高皇后|杨氏]]的顺陵在咸阳北原，曾改咸阳县为“赤县”。唐神龙元年（公元705年），升为次[[畿|畿]]。[[五代|五代]]、[[北宋|宋]]、[[金朝|金]]皆称“咸阳”。[[元朝|元朝]]初时，一度将咸阳并入兴平，不久又恢复咸阳县制。[[明朝|明]]洪武四年（1371年），将[[明咸阳城|县城]]迁到渭水驿，即现在“秦都区”所在地。[[明朝|明]]、[[清朝|清]]均称“咸阳”，属[[西安府|西安府]]管辖。

[[中华民国|中华民国]]时期，咸阳县先是归[[关中道|关中道]]管辖，不久废道直属陕西省政府。1937年后，属咸阳第十行政督察专员公署。1949年5月18日，[[中国人民解放军|解放军]]进驻咸阳县。

[[中华人民共和国|中华人民共和国]]成立后，设咸阳分区，初辖咸阳、兴平、武功、长安、户县、周至六县<ref>[http://www.xydqw.com/Article_Class2.asp?ClassID=130 咸阳地情网-古今大事] {{webarchive|url=https://web.archive.org/web/20090716131546/http://www.xydqw.com/Article_Class2.asp?ClassID=130 |date=2009-07-16 }}</ref>。1950年，根据[[陕西省人民政府|陕西省人民政府]]成立后发布的区域划分命令，[[咸阳专区|咸阳专区]]辖咸阳、[[三原|三原]]、泾阳、淳化、[[旬邑|栒邑]]、[[礼泉县|醴泉]]、兴平、[[周至|盩厔]]、[[户县|鄠县]]、耀县、铜川、富平、高陵13县。1952年，设县级咸阳市。1953年1月，咸阳分区撤销；下辖咸阳市、咸阳县、鄠县、铜川由省政府直辖，其他郊县分划[[渭南|渭南]]、[[宝鸡|宝鸡]]专署。

1958年，咸阳市、县合并，成立新的咸阳市（县级市）。1961年10月，复设咸阳专区；下辖咸阳市（县级市）和三原、泾阳、淳化、栒邑、长武、邠县、醴泉、永寿、乾县、兴平、盩厔、鄠县、高陵13县。1966年6月，咸阳市（县级市）划归西安市管辖；至1971年，重新划归咸阳地区。

1983年10月5日，经[[中华人民共和国国务院|中国国务院]]批准，[[咸阳地区|咸阳地区]]改为'''咸阳市'''（省辖市），原咸阳市（县级市）改为秦都区，[[周至|周至]]、[[户县|户县]]、[[高陵|高陵]]划归[[西安市|西安市]]，原属[[宝鸡市|宝鸡市]]的[[武功县|武功县]]和[[杨陵区|杨陵区]]划归新设的咸阳市（省辖市）。1984年5月，咸阳地区改为[[省辖市|省辖市]]后，原县级市更名“秦都区”。

2007年7月，所属[[杨陵区|杨陵区]]划归[[杨凌农业高新技术产业示范区|国家杨凌农业高新技术产业示范区]]管辖。但由于农业高新区的特殊性，[[杨凌农业高新技术产业示范区|国家杨凌农业高新技术产业示范区]]并未被[[民政部|民政部]]列为行政区域，其下辖的县级[[杨陵区|杨陵区]]在名义上仍为咸阳市市辖区。

2017年1月，咸阳市所辖的秦都、渭城、泾阳、兴平4个县市区中的15个乡镇街道、60.5万人口交由[[西安市|西安市]]代管。<ref>{{cite news|title=新格局下咸阳如何追赶超越？|url=http://news.sina.com.cn/o/2017-03-13/doc-ifychihc6312504.shtml|accessdate=2017-07-19|work=news.sina.com.cn|agency=新浪}}</ref>

2018年5月，经国务院批准，同意撤销彬县，设立县级[[彬州市|彬州市]]，由陕西省直辖，咸阳市代管。<ref>{{cite news|title=陕西省人民政府关于同意撤销彬县设立县级彬州市的批复|url=http://www.shaanxi.gov.cn/gk/zfwj/110631.htm|accessdate=2018-05-11|work=www.shaanxi.gov.cn|agency=陕西省人民政府|archive-url=https://web.archive.org/web/20180710194600/http://www.shaanxi.gov.cn/gk/zfwj/110631.htm|archive-date=2018-07-10|dead-url=yes}}</ref>

== 地理 ==
咸阳市地处陕西省[[关中盆地|关中盆地]]的中部，地势北高南低，呈阶梯状，高差明显，界限清晰，黄土高原、平原居主导地位。全市最高点位于东北部的石门山峰海拔1826米，最低处在东南部的三原县大程镇清河出境地，海拔366米。北部是黄土高原南缘的一部分，大体以[[泾河|泾河]]为界，西南部是黄土丘陵沟壑区，东北部是残原黄土沟壑和土石低中山，南部为[[渭河盆地|渭河盆地]]，属[[关中平原|关中平原]]的一部分，地势平坦，水土流失轻。盆地又可分为[[泾渭冲积平原|泾渭冲积平原]]和黄土台原，从北向南呈阶梯状分布。

境内水资源主要由河川径流和地下水组成。河流水系属黄河流域渭河水系，渭河干流从南缘流过，在市境汇入的主要支流有漆水河、新河、沣河、泾河、石川河，其中泾河最大，形成了泾河、渭河两大水系。地下水资源南富北贫，由于连续干旱，地表水供给不足，地下水开采过度，致使有的河流和池塘干涸，地下水位持续下降。过境客水较多，但利用难度较大。地表水与地下水分布极不平衡。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为4894834人<ref>{{cite book |author=咸阳市统计局|title=《咸阳市2010年第六次全国人口普查主要数据公报》}}</ref>（不含杨陵区，下同），同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加56080人，增长1.16%。年平均增长率为0.12%。其中，男性人口为2512545人，占51.33%；女性人口为 2382289人，占48.67%。总人口性别比（以女性为100）为105.47。0－14岁人口为746446人，占15.25%；15－64岁人口为3728386人，占76.17%；65岁及以上人口为420002人，占8.58%。

据人口变动抽样调查推算，2013年末全市常住人口为494.22万人。

2014年末，咸阳市常住人口495.68万人，比上年末增加1.46万人。全年出生人口5.03万人，人口出生率10.14‰；死亡人口3.05万人，人口死亡率6.16‰；人口增长率3.98‰，比上年下降0.04个千分点。从城乡结构看，城镇常住人口236.59万人，比上年末增加6.53万人；乡村常住人口259.09万人，减少5.07万人，城镇人口占总人口比重为47.73%。<ref>{{Cite web|url=http://www.xianyang.gov.cn/xyxw/jrtt/238566.htm|title=咸阳市2014年国民经济和社会发展统计公报|accessdate=|author=|date=|publisher=}}</ref>

2016年末，咸阳市常住人口498.66万人，比上年末增加1.42万人。全年出生人口5.2万人，人口出生率10.43‰；死亡人口3.05万人，人口死亡率6.12‰；人口增长率4.31‰，比上年提高0.36个千分点。从城乡结构看，城镇常住人口253.52万人，比上年末增加9.38万人；乡村常住人口245.14万人，减少7.96万人；城镇人口占总人口比重为50.84%，比上年提高1.74个百分点。

== 经济 ==
咸阳是[[陕西省|陕西省]]工业中心之一，以纺织、电子为中心，兼有电力、建材、橡胶、机械等工业部门。棉织、毛纺、化纤产品畅销国内外。咸阳是陕西省重要的粮棉产区，向国家提供的粮、肉、蛋、菜人均数量居全省之冠。著名的企业有：[[彩虹集团|彩虹集团]]（中国最大的显象管生产企业）、西北橡胶总厂（中国大型综合工业橡胶和军用橡胶制品厂，品牌“双西”，始建于1959年），陕西棉纺织系统工厂（包括[[西北国棉一厂（现已破产）|西北国棉一厂（现已破产）]]<ref>{{cite web |url=http://news.163.com/08/1021/03/4OOH91UK0001124J.html |title=西北国棉一厂负债率高达346.54%宣告破产 |accessdate=2009-05-23}}</ref>、二厂、七厂和八厂）。

咸阳又以自己的[[工业|工业]]特色被誉称为“[[纺织|纺织]][[电子|电子]]城”。纺织工业成龙配套，从棉纺、毛纺到[[化纤|化纤]]，从[[纱|纱]]、[[布|布]]、针织到[[印染|印染]]，从纺织大机到[[器材|器材]]、[[配件|配件]]制造，形成了比较完整的纺织工业体系。电子工业由只生产军品，发展到“军民结合”，由只生产元部件，发展到组装整机。闻名全国的陕西彩色显像管总厂就设在这裡。

== 文物与旅游 ==

咸阳是中国历史上第一个封建帝国秦的首都，又是汉、唐等十余个王朝的京畿重地，是闻名世界的古[[丝绸之路|丝绸之路]]的第一站。悠久的历史给咸阳留下了极为丰富的历史文化遗产，古遗址众多，内涵丰富；古墓葬分布广泛，最为集中。古建筑主要有旬邑泰塔、彬县开元寺塔、泾阳崇文塔、三原城隍庙等。石刻主要有彬县大佛寺石窟等。据统计，全市有各类文物点5313处，其中古遗址1037处，古墓葬1135处，古建筑247处。

咸阳市旅游资源丰富，著名景点星罗棋布，是[[陕西省|陕西省]]乃至中国的文物大市。咸阳號稱[[周朝|周朝]]、[[秦朝|秦朝]]、[[漢朝|漢朝]]、[[唐朝|唐朝]]文物宝库，拥有[[世界文化遗产|世界文化遗产]]1处、[[国家考古遗址公园|国家考古遗址公园]]2处、[[国家一级博物馆|国家一级博物馆]]1座、[[国家森林公园|国家森林公园]]1处，另有古邰国、秦[[咸阳宫|咸阳宫]]、[[郑国渠|郑国渠]]等重要古代遗址9处，规模宏大的漢唐帝王陵群21处，千佛塔、秦、漢[[汉兵马俑|兵马俑]]、[[漢茂陵|漢茂陵]]、[[咸阳博物馆|咸阳博物馆]]、[[漢陽陵|漢陽陵]]等。[[東漢|東漢]]断代史家[[班固|班固]]、投筆從戎再通西域的[[班超|班超]]、曹大家[[班昭|班昭]]兄妹（皆為[[班彪|班彪]]之子、扶風安陵人）的故郷在今日咸陽。

咸阳市区境内的亦有以北平街为主要入口之一的明清老城遗址区，老城内存有全国重点文物保护单位——咸阳文庙、安国寺、圣母宫等古建筑和咸阳市内唯一的明城墙遗址[[凤凰台|凤凰台]]。

咸阳湖，是西北地区水域面积最大的人工湖，著名的[[关中八景|关中八景]]之一咸阳古渡遗址就坐落在咸阳湖畔。东起渭城区古渡公园，西至秦都区咸通南路；南邻咸阳湖南苑广场，并于西咸新区的主干道——世纪大道相接，北抵咸阳市区最大的广场——统一广场。
统一广场是2008年北京奥运会火炬接力咸阳站的起点。

== 宗教 ==
明代以前，境内汉族多信佛教或道教，回族信仰伊斯兰教。明末清初，天主教传入境内。1840年鸦片战争后，天主教和基督新教遍及各县，相继成为境内信徒较多、影响较大的宗教。

“[[文化大革命|文化大革命]]”中，境内所有宗教被迫停止活动。[[中共十一届三中全会|中共十一届三中全会]]以后，归还了教堂、寺观等宗教财产，恢复了宗教活动。

== 语言 ==
咸阳市方言大部分属北方官话区中原方言关中片，而杨陵区五泉、武功县游凤部分村庄、彬县永乐、长武巨家、路家等地属中原方言秦胧片。

== 教育科研资源 ==
咸阳市内教育科研资源丰富，是陕西省内仅次于省会西安的科教大市，咸阳高等院校和科研单位之多全国地级市中也实属罕见。咸阳市拥有1所[[985工程|985工程]]和[[211工程|211工程]]院校[[西北农林科技大学|西北农林科技大学]]和1所全国重点大学、[[中西部高校基础能力建设工程|中西部高校基础能力建设工程]]高校[[陕西科技大学|陕西科技大学]]和其他多所高等学校。咸阳市辖多所省级知名中学。同时，咸阳市是西北地区重要的军工科研基地之一，多家兵工集团坐落在咸阳。
