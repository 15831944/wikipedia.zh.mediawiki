'''咸宁市'''是[[中华人民共和国|中华人民共和国]][[湖北省|湖北省]]下辖的[[地级市|地级市]]，位于湖北省东南部；北邻[[长江|长江]]，[[京广线|京广线]]、[[107国道|107国道]]、[[京广高速铁路|京广高速铁路]]、[[武汉城市圈|武汉城市圈]]城际铁路 和[[京珠高速公路|京珠高速公路]]贯穿南北，交通便利，物产丰富，咸宁因背靠[[幕阜山|幕阜山]]、踞临[[长江|长江]]而成为华中首府武汉门户，同时又是武汉1+8城市圈成员之一。

北伐战争中，国民革命军第4軍、第7军在湖北咸宁境内击溃北洋军阀[[吴佩孚|吴佩孚]]的[[直系|直系]]主力，并在两天后包围武昌，旋即占领武汉三镇，史称[[汀泗桥战役|汀泗桥战役]]与贺胜桥战役。

== 历史沿革 ==
夏商属[[荆楚|荆楚]]，[[秦朝|秦朝]]属[[南郡|南郡]]，[[汉朝|汉朝]]属[[江夏郡|江夏郡]]，[[东汉|东汉]]末属[[东吴|东吴]]。[[黄武|黄武]]二年（公元223年）置[[蒲圻县|蒲圻县]]。[[唐代宗|唐代宗]][[大历|大历]]三年（公元768年）置永安镇，[[南唐|南唐]][[保大|保大]]十三年（公元955年）升为永安县，[[宋真宗|宋真宗]][[景德|景德]]四年（公元1007年）为避[[宋太祖|宋太祖]][[永安陵|永安陵]]讳，按《易·乾象》“万国咸宁”与“永安”近义之意，取名咸宁县。[[元朝|元朝]]属[[湖广行省|湖广行省]]武昌路；明清时，属武昌府。[[民国|民国]]时期，先属江汉道，后属湖北省第一行政督察区。

中华人民共和国建立后，先后隶属[[大冶专区|大冶专区]]、[[孝感专区|孝感专区]]。1965年8月成立[[咸宁专区|咸宁专区]]，辖[[咸宁|咸宁]]、[[嘉鱼|嘉鱼]]、[[蒲圻|蒲圻]]、[[通山|通山]]、[[通城|通城]]、[[崇阳|崇阳]]、[[阳新|阳新]]、[[鄂城|鄂城]]、[[武昌|武昌]]9县。1970年咸宁专区改称[[咸宁地区|咸宁地区]]。1975年和1979年，武昌、鄂城县分别划属武汉市、黄冈地区。1983年8月撤销咸宁县，设立咸宁市（县级）。1986年6月撤销[[蒲圻县|蒲圻县]]，设立蒲圻市（县级）。1997年[[阳新县|阳新县]]划属[[黄石市|黄石市]]。1998年10月蒲圻市更名为赤壁市。

1998年12月经国务院批准，撤销咸宁地区，设立地级咸宁市，同时撤销原县级咸宁市，设立咸安区。

== 地理 ==
咸宁东南是[[江西|江西]]、西南为[[湖南|湖南]]，西北与[[荆州|荆州]]隔江相望，北面靠近武汉、[[鄂州|鄂州]]两市，东北面为[[黄石|黄石]]市，是湖北省的“南大门”。整个地形由西南向东北倾斜，基本属低山丘陵区，兼有高山、平原、湖区。 同时咸宁又是“[[桂花|桂花]]之乡”、“楠竹之乡”，其产出的桂花不论质量还是产量在全国都是第一。

== 气候 ==
[[亚热带季风气候|亚热带季风气候]]，环境适宜，森林覆盖率高于56%，空气质量全省最佳。年平均气温在17℃左右，降雨多集中在6～7月，土壤多为黄红土。

== 文化特产 ==
*'''向阳湖文化'''：向阳湖文化，是围绕湖北咸宁市向阳湖畔的文化部“五七干校”而展开的文化学术活动。
位于咸宁市向阳湖畔的“五七干校”是全国规模最大的干校之一。当年6000多文化名人，知识分子曾经在这里劳动生活，并在向阳湖留下了一篇篇不朽之作，如郭小川的《楠竹歌》、陈白尘的《牛棚日记》、张光年的《向阳日记》、臧克家的诗集《忆向阳》等，成为中国文学艺术的宝贵财富。
*竹文化
*桂花
*茶叶
*鱼米之乡
*'''全国最大的“镇”'''姓氏居住地：出自女真族。为镇海后裔。[[镇海_(蒙古帝国)|镇海]]（1160－1251年），为蒙古汗国大臣，克烈氏，南宋嘉泰三年（1203年），[[克烈部|克烈部]]与蒙古部分裂后，归依[[铁木真|铁木真]]。窝阔台汗三年（1231年），受封右丞相，後受赐恩州一千户，世食其赋。據說镇姓人就是镇海南迁后裔。
