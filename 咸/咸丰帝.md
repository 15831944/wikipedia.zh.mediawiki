'''咸豐帝'''（{{bd|1831年|7月17日|1861年|8月22日|catIdx=A愛}}），[[爱新觉罗|爱新觉罗]]氏<!-- 前方請勿加粗 -->，名'''奕{{僻字|詝|左“言”右“宁”}}'''（{{lang-mnc|ᡳ ᠵᡠ|v=I Ju|a=I Ju}}），號'''且樂道人'''，<ref>《[[清稗類鈔|清稗類鈔]]》</ref>是[[清朝|清朝]]第九位[[皇帝|皇帝]]，也是[[清兵入关|清兵入关]]以来的第七位皇帝，1850年3月9日至1861年8月22日在位，[[年號|年號]]「[[咸丰_(年号)|咸豐]]」。[[清朝治藏历史|西藏方面]]尊為「'''[[文殊皇帝|文殊皇帝]]'''」{{NoteTag|1=[[藏傳佛教|藏傳佛教]]以[[文殊菩薩|文殊菩薩]]敬稱[[清朝皇帝|清朝皇帝]]。}}。

咸豐帝是[[道光帝|道光帝]]第四子，生母為[[孝全成皇后|孝全成皇后钮祜禄氏]]，誕於[[北京|北京]][[圆明园|圆明园]]澄静斋，养母為[[孝静成皇后|孝静成皇后博尔济吉特氏]]。20岁登基，1861年崩于[[承德市|承德]][[避暑山庄|避暑山庄]]烟波致爽殿，享年30岁。死后[[庙号|庙号]]'''文宗'''（{{lang-mnc|ᠸᡝᠨᡯᡠᠩ|v=wendzung}}），[[谥号|谥号]]簡稱'''显皇帝'''（{{lang-mnc|ᡳᠯᡝᡨᡠ <br>ᡥᡡᠸᠠᠩᡩᡳ|v=iletu hūwangdi}}），葬于[[清东陵|清东陵]]中的[[清定陵|定陵]]。他也是清朝最後一位掌握實權的皇帝與最後一位[[秘密建儲制|儲位密建]]的皇帝。

== 生平 ==
[[File:《咸丰皇帝便装像》.jpg|thumb]]
[[File:Xianfeng_Bias.jpg|thumb]]

1831年7月17日（道光十一年六月初九日），咸丰帝生于[[北京|北京]][[圆明园|圆明园]]之澄静斋。时道光帝前三个儿子都已去世，咸丰帝出生后即为在世的皇长子。二十六年，按照[[秘密建儲制|秘密立储制度]]，被道光帝立为[[储君|储君]]。三十年正月丁未，道光帝驾崩前，宣召大臣开启[[鐍匣|鐍匣]]，立为[[皇太子|皇太子]]。

咸豐帝即位後便勤於政事，广开言路、明詔求賢，先後將有损国家利益的[[穆彰阿|穆彰阿]]和[[耆英|耆英]]革職，大手笔地对朝政颇有改革。但此時的大清帝国內憂外患不斷，先後爆发[[太平天国|太平天国]]运动以及[[第二次鸦片战争|第二次鸦片战争]]。

在第二次鸦片战争中，俄国[[西伯利亚|西伯利亚]][[总督|总督]][[穆拉维约夫|穆拉维约夫]]迫使[[黑龙江将军|黑龙江将军]][[奕山|奕山]]签订《[[瑷珲条约|瑷珲条约]]》，割去[[黑龙江|黑龙江]]以北、[[外兴安岭|外兴安岭]]以南原属清朝的领土约60万平方公里，咸丰帝拒绝承认该条约。随后[[英法联军|英法联军]]进攻[[北京|北京]]，咸丰帝下诏对英国法国宣战：“兵家胜败何常，该国兵远来即有数万，未可当我中国人民千百之一，其能经几战乎？<ref>《咸丰朝筹办夷务始末》卷62</ref>”最后[[圆明园|圆明园]]、[[清漪园|清漪园]]等被焚掠，以签定一系列[[不平等条约|不平等条约]]收场。

咸丰十一年（1861年）七月十七日，咸丰帝崩于[[行在|行在]][[避暑山庄|避暑山庄]][[烟波致爽殿|烟波致爽殿]]。皇长子[[同治帝|同治帝]]继位。同时依遺詔，由[[肃顺|肃顺]]、[[载垣|载垣]]、[[端华|端华]]、[[景寿|景寿]]、[[穆荫|穆荫]]、[[匡源|匡源]]、[[杜翰|杜翰]]、[[焦祐瀛|焦祐瀛]]为[[顾命八大臣|顾命八大臣]]，[[肅順|肅順]]為首，輔導皇帝施政。在咸丰灵柩启程返京期间，東太后[[慈安|慈安]]、西太后[[慈禧|慈禧]]、[[恭親王|恭親王]][[奕訢|奕訢]]、[[醇亲王|醇亲王]][[奕譞|奕譞]]四人联合發動[[辛酉政变|辛酉政变]]，醇亲王奕譞亲自抓捕肃顺<ref>据老北京满族宫廷禁卫口述史</ref>，随之八大臣非死即貶。同时，政府随即由[[慈安|慈安]]、[[慈禧|慈禧]][[两宫听政|两宫听政]]，咸丰帝生前的輔政遺命宣告廢止。

根據内务府档案的记载，如意馆画士沈振麟曾在同治年间绘制了两幅先帝咸丰帝圣容，这两幅圣容均先画稿，呈览后再进一步绘制。和硕恭亲王奕訢向两宫皇太后和同治帝呈览的两幅墨稿分别为便衣墨稿一件及道装配山洞景致墨稿一件。

== 評價 ==
* 清朝皇帝的評價中，咸豐帝的爭議最大。咸丰爱看戏，爱唱戏，即使到[[热河厅|热河]][[行在|行在]]唱戏，“着升平署三拔至热河”，也表現得乐不思蜀<ref>福成《庸盦笔记》</ref>。咸丰一朝，财政十分困难，要镇压太平天国，对付英法联军，财源枯竭，“户部因军兴财匮，行钞，置宝钞处，行大钱，置官钱总局，分领其事”，[[钞票|钞票]]大量发行，造成[[通货膨胀|通货膨胀]]，“官民交累，徒滋弊窦”<ref>《清史稿肃顺传》卷三八七。</ref>。

* 咸丰帝“任賢擢才，洞觀肆應”，在面對太平天国运动與“三千年未有之政局”的內憂外患中，咸丰指揮若定，重用汉族大臣[[曾国藩|曾国藩]]人等组织团练来对付太平天国，咸丰帝颇思除弊求治，提拔行事果断的肃顺，并支持肃顺等人在朝政上推行的改革。為後來的[[同光中興|同光中興]]打下良好基礎。但也因為採納肃顺對英法兩國強硬的做法，導致引發[[英法聯軍之役|英法聯軍之役]]。

* 咸丰帝临终前对朝政事宜不合理的安排和权力制衡，使朝臣和后宫在有关朝政和国事方面展开逐鹿，间接导致[[慈安太后|慈安太后]]、[[慈禧太后|慈禧太后]]、[[恭親王|恭親王]][[奕訢|奕訢]]、[[醇亲王|醇亲王]][[奕譞|奕譞]]联手，打倒了[[顾命八大臣|顾命八大臣]]，最後慈禧掌权近半个世纪，从而也被有些史家流派认为咸丰帝没有安排好善后事宜，致使后宫干政近半个世纪的局面。

* 英、法等國要求清廷能讓英法兩國在北京設置駐京公使，新任公使到任時能觐見皇帝，但咸丰帝不接受。英國也要求清廷開放中國貿易，咸丰帝也拒絕，雖然[[天津條約|天津條約]]簽訂後，咸豐帝默許了英、法等國的要求，但又對英、法等國的公使刁難(主因是採納肃顺對英法兩國強硬的做法)。咸丰帝本人的守舊，間接導致[[英法聯軍之役|英法聯軍之役]]清朝的慘敗，使清朝失去首都[[北京|北京]]，[[圓明園|圓明園]]也遭聯軍焚毀。

* 太平天国运动與[[英法聯軍之役|英法聯軍之役]]，使咸丰帝的執政遭受打擊，逃往熱河行宮後就病逝了，太平天国运动與[[英法聯軍之役|英法聯軍之役]]的善後工作，直到[[慈安|慈安]]、[[慈禧|慈禧]][[两宫听政|两宫听政]]時期才結束。

== 大事年表 ==
* 道光十一年六月初九日，奕{{僻字|詝|左“言”右“宁”}}出生。后受教于[[杜受田|杜受田]]。
* 道光三十年正月，宣宗去世，奕{{僻字|詝|左“言”右“宁”}}繼位。是年太平天國起事。
* 咸豐三年二月，太平軍攻佔[[江宁府|江寧]]，定都在此，改名天京。九月，太平天國北伐軍逼近[[天津|天津]]。是年[[曾國藩|曾國藩]]建[[湘軍|湘軍]]。
* 咸豐五年四月，[[李開芳|李開芳]]被俘，太平天國北伐軍覆沒。
* 咸豐六年八月，[[天京事變|天京事變]]發生。九月，「[[亞羅號事件|亞羅號事件]]」發生。
* 咸豐八年四月，與俄國簽訂《[[璦琿條約|璦琿條約]]》。五月，先後與俄、美、英、法四國簽訂《[[中俄天津條約|中俄天津條約]]》、《[[中美天津條約|中美天津條約]]》、《[[中英天津條約|中英天津條約]]》及《[[中法天津條約|中法天津條約]]》。十月，太平軍取得[[三河大捷|三河大捷]]。
* 咸豐九年五月，清[[大沽|大沽]]守軍擊退英、法艦隊。
* 咸豐十年七月，英法聯軍攻佔天津和大沽一帶。八月，[[八里橋|八里橋]]和[[大沽口|大沽口]]相繼被攻佔，咸豐帝逃往承德，亦不足兩個月，英法聯軍[[火燒圓明園|火燒圓明園]]，進佔北京。十月，《[[北京条约|中英北京條約]]》及《[[北京条约|中法北京條約]]》立。十一月，《[[北京条约|中俄北京條約]]》立。十二月[[總理各國事務衙門|總理各國事務衙門]]成立。
* 咸豐十一年七月十七日，咸豐皇帝駕崩於[[承德市|承德]][[承德避暑山庄及其周围寺庙|避暑山庄]][[烟波致爽殿|烟波致爽殿]]，年三十。其子[[载淳|载淳]]年仅六岁，继承大统，是為同治帝。咸丰委派[[载垣|载垣]]、[[端华|端华]]、[[景寿|景寿]]、[[肃顺|肃顺]]、[[穆荫|穆荫]]、[[匡源|匡源]]、[[杜翰|杜翰]]、[[焦祐瀛|焦祐瀛]]为[[顧命八大臣|辅政八大臣]]，辅助小皇帝。

==家族==
=== -{后}-妃子女 ===
; 皇后
* [[孝德顯皇后|孝德顯皇后]]，萨克达氏，咸豐即位前即去世。
* '''[[慈安太后|孝貞顯皇后]]'''（1837年8月12日－1881年4月8日），皇后鈕祜祿氏，滿洲镶黄旗人，廣西右江道三等承恩公穆楊阿之女。1852年（咸豐二年二月）選秀入宮，四月二十七進內，冊封為貞嬪，五月晉升為貞貴妃，十月即正式立為皇后。文宗死後，稱為「母后皇太后」（[[慈安太后|慈安太后]]、東太后）與[[慈禧太后|慈禧太后]]（西太后）垂簾聽政。光緒七年崩，年四十四，相傳為西太后所害，曰孝貞慈安裕慶和敬誠靖儀天祚聖顯皇后。
* '''[[慈禧太后|孝欽顯皇后]]'''（1835年11月29日－1908年11月15日），懿貴妃葉赫那拉氏，滿洲鑲蓝旗人，安徽[[徽寧池太廣道|徽寧池太廣道]]道員惠徵之女。1852年（咸豐二年二月）選秀入宮，五月初九進內，冊封為懿貴人，咸豐四年二月乙未晉封為懿嬪，1856年（咸豐六年三月庚辰），懿嬪生下皇長子載淳，當日就晉封懿妃。咸豐七年（1857）正月乙卯[16]，又晉封懿貴妃。文宗死後，稱為「聖母皇太后」（[[慈禧太后|慈禧太后]]、西太后）與東太后垂簾聽政，[[光緒|光緒]]三十四年崩，年七十二。生一子[[同治帝|載淳]]，即同治帝。

; 妃嫔
* [[莊靜皇貴妃|莊靜皇貴妃]]，他他拉氏，主事慶海女，道光十七年二月二十七生，咸豐元年入宮，二年五月初九，封為麗貴人，四年十二月二十四，封為麗嬪，五年五月初七，晉為麗妃，穆宗登極後，尊封麗皇貴妃，同治十三年十一月十六，光緒帝尊封為麗皇貴太妃，光緒十六年十一月薨，年五十四，生一女，[[榮安固倫公主|榮安固倫公主]]。
* [[端恪皇贵妃|端恪皇贵妃]]，佟佳氏。
* [[玫贵妃|玫贵妃]]，徐佳氏，皇二子生母。
* [[婉贵妃_(咸丰帝)|婉贵妃]]，索绰络氏，咸丰帝时为婉嫔，同治帝晋尊为皇考婉妃，再诏晋尊为婉贵妃。
* [[璷妃|璷妃]]，葉赫那拉氏。咸豐五年二月初十日，封璷貴人，二月二十四日，璷貴人如嬪位的內庭宮各退。同治帝尊其為皇考璷嬪。光緒帝再尊其為璷妃，所享的內庭宮分同貴妃之位。
* [[禧妃|禧妃]]，察哈拉氏。
* [[吉妃|吉妃]]，王氏。咸豐八年五月十五日，封為吉貴人。同治帝尊其為皇考吉嬪，後光緒帝再尊其為吉妃。
* [[庆妃|庆妃]]，张氏。
* [[云嫔|云嫔]]，武佳氏，咸丰帝为皇子时，入侍為「格格」。
* [[容嫔|容嫔]]，伊尔根觉罗氏，咸丰三年十月二十日已為容常在，咸豐五年十月初一日已晋為容贵人，咸丰十一年尊封容嫔。尊封伊爾根覺羅氏為容嬪的尊封事宜，於同治八年仍未辦理，因伊爾根覺羅氏突然去世而停止。
* 璹嬪（1840年—1874年），[[叶赫那拉|叶赫那拉]]氏，正白旗滿洲員外郎桂祥之女。道光二十年二月二十日子時出生。咸豐五年二月初十日，經外八旗選秀入宮，封為𤩇贵人，所享的內庭宮分同嬪位。咸豐七年十一月至咸丰八年二月，那拉氏的待遇因故如常在之位。咸豐九年十一月初十日，改號為璹貴人。咸丰十一年尊封璹嫔。与莊靜皇貴妃同居永和宫。尊封叶赫那拉氏為璹嫔的尊封事宜，於同治十三年仍未辦理，因葉赫那拉氏突然去世而停止。同治十三年三月二十四去世，葬在清定陵妃园寝。

* [[玉嬪_(咸豐帝)|玉嬪]]（1843年—1863年），[[叶赫那拉|叶赫那拉]]氏，[[正白旗|正白旗]]满洲员外郎桂祥之女，有一姐[[璹嫔|璹嫔]]，同为[[咸豐帝|咸豐帝]]妃嫔<ref>{{cite book|authors=章乃炜、王蔼人|title=清宫述闻(正续编合编本)|editor=陈晓东|others=封扉设计：李猛|edition=第1版|location=北京|publisher=[[紫禁城出版社|紫禁城出版社]] |date=2009年12月|page = 597|ISBN = 978-7-80047-907-6}}</ref>。咸豐八年三月二十五日，由本家送入圓明園，為玉贵人。咸丰十一年同治帝继位，奉两宫皇太后懿旨晋封为玉嫔。同治元年十一月十六日（1863年1月5日）去世<ref name="qingshigao" />。同治四年九月二十五日奉安妃园寝<ref name="qingshigao" />。

* [[玶常在|玶常在]]，伊尔根觉罗氏，初号英贵人，咸丰二年封英嫔，咸豐三年降為伊貴人，咸豐五年降為伊常在，再降為伊答應，六年升回玶常在。<ref>{{cite book|author=王佩-{環}-|title=清宫后妃|location=沈阳 |publisher=辽宁大学出版社|date=1993年12月|page=351|ISBN=978-7-5610-2144-6}}</ref>

* [[瑃常在|瑃常在]]<ref>{{cite book|authors=章乃炜、王蔼人|title=清宫述闻（正续编合编本）|editor=责任编辑：陈晓东|others=封扉设计：李猛|edition=第1版|location=北京|publisher=紫禁城出版社|date=2009年12月|page=600|ISBN =978-7-80047-907-6}}</ref>
* [[錱常在_(咸豐帝)|錱常在]]（？—1859年），《内务府谱牒》記載為戴氏，戴吉祿原配之女。包衣管领下辛者库人。咸丰九年（1859年）五月初六日去世。
