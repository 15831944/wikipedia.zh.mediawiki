{{日本作品|我的妹妹是黃漫老師|エロマンガ先生|Eromanga Sensei|cn=埃罗芒阿老师|hk=我的妹妹是黃漫老師|tw=情色漫畫老師}}是由[[日本|日本]][[作家|作家]][[伏見司|伏見司]]創作、[[神崎廣|神崎廣]]繪製插畫的[[戀愛|戀愛]][[輕小說|輕小說]]作品，連載於[[電擊文庫|電擊文庫]]，由[[KADOKAWA|KADOKAWA]]（角川）出版發行。[[輕漫畫|輕漫畫]]於2014年5月27日發售的《[[月刊Comic電擊大王|月刊Comic電擊大王]]》開始連載。已於2017年4月8日至6月25日播放12集[[電視動畫|電視動畫]]。

== 故事簡介 ==
{{more plot}}
高中生兼輕小說作家和泉正宗有個[[家裡蹲|家裡蹲]]的妹妹[[和泉紗霧|和泉紗霧]]，兩人一年前才成為兄妹，可是紗霧完全不走出房間。正宗除了每天都幫紗霧準備食物，還天天在想辦法讓她走出房間。而正宗的搭檔插畫家「黃漫老師」，是個能畫出非常棒的[[殺必死|殺必死]]插画的人。然而某一天，正宗突然發現衝擊的事實：「黃漫老師」其實是紗霧。正宗與紗霧的關係瞬間出現變化，正宗身邊也越聚越多个輕小說作家。正宗與紗霧就此展開了一段環繞青春的愛情故事……

== 作品概要 ==
本作是一部以非血緣關係的兄妹間的感情发展為側重點，以作家與插畫家的夢想為軸心的[[戀愛喜劇|戀愛喜劇]]作品。現實的輕小説作品的登場是本作的主要特徵之一。在小説開始企劃期間，原作者伏見司決定放棄[[跨媒體製作|媒體組合]]的傳播形式，並以此為前提進行小説寫作。在決定動畫化之前，小説累計銷售量為60萬本，2017年4月22日累計100萬本，同年6月8日宣佈突破120萬。

2013年12月10日，[[KADOKAWA|KADOKAWA]] 出版社正式出版小説第一冊並負責後續出版工作，由{{tsl|ja|rin (漫画家)|rin}}負責作畫的[[輕漫画|輕漫画]]在2014年5月27日發售的《[[月刊Comic電擊大王|月刊Comic電擊大王]]》2014年7月号開始連載<ref>{{Cite web|url=http://books.rakuten.co.jp/rb/12776691/|script-title=ja:電撃大王 2014年 07月号|publisher=[[樂天_(Rakuten)|樂天]]|accessdate=2017年5月2日|language=ja}}</ref>。2015年11月28日開始通過[[Niconico動畫#niconico.E7.9B.B4.E6.92.AD.EF.BC.88.E3.83.8B.E3.82.B3.E3.83.8B.E3.82.B3.E7.94.9F.E6.94.BE.E9.80.81.EF.BC.89|Niconico直播]]服务進行{{tsl|ja|多数決ドラマ|多數決Drama}}配信。2016年3月10日發表改編電視動畫的消息<ref>{{Cite web|url= http://dengekionline.com/elem/000/001/234/1234702/|script-title=ja: 電撃文庫『エロマンガ先生』のアニメ化が決定！|work=[[電擊文庫|電擊文庫]]|publisher=[[KADOKAWA|KADOKAWA]] [[ASCII_Media_Works|ASCII Media Works]]|date=2016年3月10日|accessdate=2016-03-10|language=ja}}</ref>。動畫于2017年4月8日開始在日本電視臺全網播放。動畫OVA於2019年1月16日發售。

=== 地理位置 ===
在動畫中，男女主角與山田妖精的住宅地理位置設定在[[東京都|東京都]][[足立區|足立區]]一丁目（[[五反野站|五反野站]]附近），從住宅外可以看見[[荒川_(關東)|荒川河]]，這一地區是作者伏見司曾經居住過的地方。千壽村征的住宅地理位置設定在[[千葉縣|千葉縣]]，距離[[千葉站|千葉站]]換乘電車不到一個小時車程的鄉下地方。

=== 與其他作品的聯動 ===
本作與伏見司的前作輕小説《[[我的妹妹哪有這麼可愛！|我的妹妹哪有這麼可愛！]]》進行過多次聯動。小說的第2卷插畫與第3卷第292頁都有對《我的妹妹哪有這麼可愛！》的女主角高坂桐乃的人物描繪<ref>{{Cite web|url=http://blog.livedoor.jp/geek/archives/51505873.html|script-title=ja: 【コラム】 エロマンガ先生５巻発売記念！伏見つかさ先生＆かんざきひろ先生インタビュー|work=秋葉原Blog|date=2015-09-11}}</ref>。在動畫中，本作的[[時間線|時間線]]定位在《我的妹妹哪有這麼可愛！》之後，設定上屬於同一個[[世界觀|世界觀]]<ref group="來源">動畫BD/DVD附帶特典CD[[廣播劇|廣播劇]]{{nihongo|《我的妹妹是黃漫老師×我的妹妹哪有這麼可愛！合作劇》|「エロマンガ先生×俺の妹がこんなに可愛いわけがない コラボレーションドラマ」}}。</ref>。在動畫上映之前，官方便進行過多次聯動宣傳活動，包括推出兩部作品的官方選集、在多數決Drama配信中的作品聯動以及兩部作品的周邊產品共同銷售等。

在動畫開播後，《我的妹妹哪有這麼可愛！》的[[角色|角色]]於動畫第8話與第11話中客串登場。已登場的角色有高坂京介（聲：[[中村悠一|中村悠一]]）、高坂桐乃（聲：[[竹達彩奈|竹達彩奈]]）、五更瑠璃（聲：[[花澤香菜|花澤香菜]]）、槙島沙織（聲：[[生天目仁美|生天目仁美]]）、來-{栖}-加奈子（以梅露露的[[cosplay|cosplay]]形象）、五更日向（聲：花澤香菜）、五更珠希、御鏡光輝、赤城瀨菜、赤城京平、真壁楓與三浦絃一郎。此外，高坂兄妹與和泉兄妹也在動畫特典廣播劇中共同登場。

此外，在動畫第8話中，其他知名輕小說作品的角色如[[司波達也|司波達也]]（聲：[[中村悠一|中村悠一]]；《[[魔法科高中的劣等生|魔法科高中的劣等生]]》男主角）、赫蘿（聲：[[小清水亞美|小清水亞美]]；《[[狼與辛香料|狼與辛香料]]》主角）與桐人（聲：[[松岡禎丞|松岡禎丞]]；《[[刀劍神域|刀劍神域]]》男主角）也以過場形式客串登場。

== 登場人物 ==
=== 主要人物 ===
; {{nihongo|和泉正宗／和泉征宗|和泉 正宗（いずみ まさむね）／和泉 マサムネ（いずみ マサムネ）|聲：[[松岡禎丞|松岡禎丞]]<ref name="character">{{cite web|url=http://eromanga-sensei.com/character/|title=CHARACTER|work=電視動畫「我的妹妹是黃漫老師」官方網站|language=ja}}</ref>}}
: 本作男主角，15歲，第5卷時年16歲。輕小說作家，'''「和泉征宗」'''為筆名，是女主角和泉紗霧無血緣關係的哥哥。從出道作開始都是由「黃漫老師」擔任插畫家。在入學國中的時候是第一位榮獲輕小說新秀獎選考員獎勵賞的新秀作家。
: 其生母喪生後，大概主线剧情一年前，其父親和泉虎徹再婚時，繼母帶著女兒紗霧再組家庭，然而夫婦卻在新婚旅行時喪生。
: 原本以《轉生的銀狼》為代表作的學園異能戰鬥小說為寫作風格，但在與妹妹接觸後改以《世界上最可愛的妹妹》為代表作的妹系風格為寫作方式，在{{nihongo|'''「輕小說天下第一武鬥會」'''|ラノベ天下一武闘会}}中獲得最多投票數，榮獲第一，甚至被文庫本化與動畫化。但這類風格使千壽村征頗為不滿。被山田妖精及責任編輯稱為「雜魚似的存在」。

; {{nihongo|和泉紗霧／黃漫老師|和泉 紗霧（いずみ さぎり）／エロマンガ先生（エロマンガせんせい）|聲：[[藤田茜|藤田茜]]{{r|character}}}}
: 本作女主角，12歳，第4卷慶生后時年13歲。小説插畫家，'''「黃漫老師」'''為筆名，是男主角和泉正宗無血緣關係的妹妹。銀髮藍瞳。《星塵☆小魔女梅露露》的忠實粉絲。
: 於部落格上表示筆名「黃漫老師」的由來是島嶼[[埃羅芒阿島|埃羅-{}-芒阿島]]的名稱，跟黃漫沒關係。當有人當場稱她為「黃漫老師」時，和泉紗霧的回答都是「我不認識叫那種名字的人」，這個回答已經成為了口頭禪。
: 小學二年級時，父母離婚，由此經常拒絕上學，甚至到國中時沒有去過一次學校。因监护人（正宗的姑姑）尝试强行将她带出房间，加之父母過世所造成之精神創傷而成為家裡蹲，只要家中有人就絕不離開臥室，與和泉正宗的溝通都是用踩地板造成的聲響（「床咚」）作為溝通方式。
: 對待插畫相當專注，富含激情，是第一個給正宗發投稿完結感想並畫了畫慶祝的人。常將神野惠與山田妖精作為繪畫模特，讓她們擺出各種姿勢，甚至平靜地要求她們模仿性騷擾的動作。對女孩子的內褲有着異乎常人的渴望，曾當和泉正宗的面脫下神野惠的內褲，以及通過電腦視訊臨摹山田妖精的內褲。
: 擁有自己的部落格，有時也會在視頻網站上進行繪畫直播。直播時會使用變聲器，頭戴梅露露面具。但在某次直播中被和泉正宗發現攝像機身後的飯盤，從而暴露身份。
: 在漫畫中，和泉紗霧的插畫由{{nihongo|大將田中|たいしょう田中}}負責，動畫中則由[[Tiv|Tiv]]負責。
: 在現實中， 「紗霧」一名源自日本[[狹霧號驅逐艦|狹霧號驅逐艦]]<ref>{{cite web|url=http://blog.livedoor.jp/geek/archives/51421359.html|script-title=ja:先生＆かんざきひろ先生インタビュー後編|publisher=秋葉原Blog|language=ja}}</ref>。

2017年11月12日，官方在[[千葉縣|千葉縣]][[浦安市|浦安市]]的[[舞濱圓形劇場|舞濱圓形劇場]]舉辦的{{nihongo|「黃漫老師Fes」|エロマンガ Fes}}活動上宣佈將在2018年製作[[原創動畫錄影帶|OVA]]，於2019年1月16日正式發售<ref>{{Cite web|url=https://prtimes.jp/main/html/rd/p/000000748.000016356.html|script-title=ja:アニメ『エロマンガ先生』2018年OVA制作決定！！！|publisher=PR TIMES|date=2017年11月12日|accessdate=2017年11月12日|language=ja}}</ref><ref>{{Cite news|url=https://prtimes.jp/main/html/rd/p/000001322.000016356.html|script-title=ja:TV アニメーション「エロマンガ先生」OVA詳細が発表！|publisher=PR TIMES|date=2018年10月3日|accessdate=2018年10月4日|language=ja}}</ref><ref>{{Cite news|url=https://mainichi.jp/articles/20181003/dyo/00m/200/019000c|script-title=ja:エロマンガ先生：OVAが19年1月16日発売　「山田エルフのラブソング」「和泉紗霧のファーストキス」の2本立て|publisher=[[毎日新聞|毎日新聞]]|date=2018年10月3日|accessdate=2018年10月4日|language=ja}}</ref>。

=== 網路電台 ===
《'''黃漫老師電台SENSATION！'''》（{{lang-ja|ラジオで エロマンガSENSATION!}}），簡稱《'''黃漫電台'''》（{{lang-ja|Eラジ}}）是一檔網路電台節目，于2017年4月7日到11月3日止隔周星期五在[[音泉|音泉]]電台播放<ref>{{Cite web|url=http://www.onsen.ag/program/eromanga-sensei/|script-title=ja:ラジオで　エロマンガSENSATION！|work=[[音泉|音泉]]|accessdate=2017-03-12}}</ref>。節目由和泉紗霧的聲優[[藤田茜|藤田茜]]與山田妖精的聲優[[高橋未奈美|高橋未奈美]]負責主持。

本作在日本Charapedia舉行的「最期待的2017年春季動畫作品TOP 20！」投票中獲得第2名<ref>{{Cite web|url=http://www.charapedia.jp/research/0158/|script-title=ja:もっとも期待している2017年春アニメ作品 TOP20!|work=CaraPedia|accessdate=2017-07-29|language=ja}}</ref>，在「下集還想繼續追的2017年春季動畫作品TOP 20！」投票中獲得第3名<ref>{{Cite web|url=http://www.charapedia.jp/research/0161/|script-title=ja:次回以降も観続けたい2017年春アニメ作品 TOP20!|work=CaraPedia|accessdate=2017-07-29|language=ja}}</ref>。本作還在[[niconico動畫|niconico動畫]]于2017年6月29日開始的「用戶認真評選霸權動畫2017年春季篇」投票中登上綜合榜首<ref>{{Cite web|url=https://enquete.nicovideo.jp/result/85|script-title=ja:【結果】ユーザーが本気で選ぶ覇権アニメ　2017年春編|work=niconico問卷|accessdate=2017-07-29|language=ja}}</ref>。
