== 剧情 ==
该片主要讲述一个“[[三角恋|三角恋]]”的爱情故事。

惠香（舒淇）和惠宝（舒淇）是[[双胞胎|双胞胎]]亲姊妹，两人都爱上了阿楠（余文乐），为了自己的[[幸福|幸福]]和[[爱情|爱情]]，两姊妹开始一明一暗，斗智斗勇。<ref>{{cite web |url= http://news.xinhuanet.com/society/2012-03/09/c_111630260.htm|title=电影《情迷》：文艺片演绎的悬疑 |accessdate=2012年3月9日 |publisher=新华网 |work= |language =zh-cn}}</ref>

== 花絮 ==
电影中，舒淇一个饰演4个角色，她主演饰演一对双胞胎孪生姊妹。<ref name=" fish">{{cite web|url=http://ent.xinmin.cn/2012/02/15/13655324.html|title=舒淇电影《情迷》秀女伶妆扮 学戏曲身段|accessdate=2012-02-15|publisher=新民网|work=|language=zh-cn}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref>{{cite web|url=http://ent.cn.yahoo.com/ypen/20120305/904431.html|title=《情迷》将映 舒淇：每个导演都想把我掏空|accessdate=2012年3月5日|publisher=中国雅虎|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20130224195432/http://ent.cn.yahoo.com/ypen/20120305/904431.html|archivedate=2013年2月24日}}</ref>

对于电影中的[[床戏|床戏]]，余文乐表示：“我可是很有压力，有几场喝了酒才敢上阵的。”

在电影的发布会上，余文乐表示自己在社会现实生活中不会用情不专和脚踏两只船，“同时爱两个人对我来说很难，首先我不会撒谎，其次这种情况很麻烦，肯定有穿帮的那天。”<ref>{{cite web |url= http://www.hunantv.com/c/20120215/0823514010.html|title=舒淇缺席《情迷》发布 余文乐：不会脚踏两只船 |accessdate= 2012-02-15|publisher=锦衣卫 |work= |language =zh-cn}}</ref>
