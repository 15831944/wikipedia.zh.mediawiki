《'''情聖卡薩諾瓦'''》（[[義大利語|義大利語]]：{{lang|it|''Il giovane Casanova''}}；[[法語|法語]]：{{lang|fr|''Le Jeune Casanova''}}）是一部[[2002年電影|2002年]]的[[義大利電影|義大利]]-[[法國電影|法國]]-[[德國電影|德國]]合製的[[古裝劇|古裝]][[電視電影|電視電影]]，分為兩集，因此也可視為一套迷你影集。由{{tsl|en|Giacomo Battiato|傑可莫·巴提亞托}}導演，[[義大利人|義大利]]演員{{tsl|en|Stefano Accorsi|史帝芬努·阿科西}}出演主角[[傑可莫·卡薩諾瓦|傑可莫·卡薩諾瓦]]。影片於2002年的法國{{tsl|en|Festival International de Programmes Audiovisuels|視聽藝術國際影展}}獲獎<ref>{{cite web |url=http://php88.free.fr/bdff/film/2002/2175/10.jpg |title=Le Jeune Casanova |website=php88.free.fr |accessdate=2017-06-25 |language=fr}}</ref>，並曾由[[公視|公視]]引進至[[中華民國|臺灣]]播出<ref name=pts>{{cite web |url=http://web.pts.org.tw/php/mealc/main.php?XMAENO=549&XMBENO=743 |title=情聖卡薩諾瓦 |work=[[公視|公視]] |accessdate=2017-06-26 |language=zh-TW}}</ref>。

== 主題 ==
橫跨[[威尼斯共和國|威尼斯]]的街道，逃離法律的監管，遊歷整個[[歐洲|歐洲]]尋求新的挑戰，[[傑可莫·卡薩諾瓦|傑可莫·卡薩諾瓦]]以充沛的精力創造屬於他自己的生活。他是[[小提琴家|小提琴家]]、作家、士兵、間諜、賭徒、冒險家，以及充滿魅力的情人和[[勾引|誘惑者]]。同時，他也為女子的姣容所傾倒，為她們的激情所引誘，被她們的神祕所吸引。對於年輕的卡薩諾瓦來說，生活應是如同令人眼花瞭亂的[[威尼斯嘉年華會|威尼斯嘉年華會]]，但愛情不接受面具與偽裝，這就是他最眞實、最脆弱也最赤裸的地方。作為一名女演員的兒子，卡薩諾瓦從小就生活在充滿戲劇氛圍和演藝魅力的環境中。作為法國駐[[威尼斯共和國|威尼斯共和國]]大使的門徒，他開始拓展自己通往上流社會的道路，施展自身魅力去征服最美麗的女子。他為此遭受自己保護人的嫉妒而被關進監獄，但15個月之後，他成功越獄逃到[[巴黎|巴黎]]，並在那裡獲得[[路易十五|路易十五]]的寵妃[[龐巴度夫人|龐巴度夫人]]的賞識。

== 播出 ==
影片於2002年2月5日和2月7日在[[義大利|義大利]]於 [[Canale_5|Canale 5]] 電視臺分兩集首播，隨後於其他國家播出：
* 2002年3月10日於[[法國|法國]]播出，片名《{{lang|fr|Le Jeune Casanova}}》
* 2003年2月16日於[[德國|德國]]播出，片名《{{lang|de|Casanova – Ich liebe alle Frauen}}》
* 2003年3月15日於[[阿根廷|阿根廷]]播出，片名《{{lang|es|El joven Casanova}}》
* 2004年11月6日於[[匈牙利|匈牙利]]播出，片名《{{lang|hu|Az ifjú Casanova}}》

== 製作 ==
此片所有演員均以[[法語|法語]]演出，其中的義大利演員之後再為自己出演的角色作[[義大利語|義大利語]]配音。
