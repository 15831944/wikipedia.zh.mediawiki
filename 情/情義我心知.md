{{engname|《情義我心知》|'' Moonlight in Tokyo ''}}是2005年出品的一部香港電影，由[[黎明|黎明]]、[[杜汶澤|杜汶澤]]、[[葉璇|葉璇]]、[[楊貴媚|楊貴媚]]、[[張耀揚|張耀揚]]等主演，在[[東京|東京]]進行拍攝<ref>{{Cite web |url= http://www.hkbo.com.hk/search-details-new.php?filmid=6476&run=1|title =電影資料庫| publisher = 香港票房有限公司|language = 中文}}</ref><ref>{{Cite web |url= http://www.hkmdb.com/db/movies/view.mhtml?id=11188&display_set=big5|title =情義我心知（2005）| publisher =香港影庫|language = 中文}}</ref>。

==情節介紹==
故事講述黃海在日本借高利貸做本錢，經營皮條生意，結果遭人陷害而血本無歸，更欠下黑幫一筆巨債，被逼於街頭派紙巾還債。黃海在新宿街頭扒竊期間，巧合遇上患有輕度弱智的小學同學李子俊，原來李子俊與哥哥在日本旅遊時失散，獨自流浪街頭。黃海發現李子俊頗受女性歡迎，在經營酒吧的好友燕姐拉攏下，將李子俊推向了男色市場，於是兩兄弟穿梭於新宿的大街小巷之間，兩人之間漸漸產生了兄弟之情，但其後的一連串事件令兩人的生活急劇變化，使到李子俊與黃海陷入了更大的麻煩<ref>{{Cite web |url= http://news.cts.com.tw/cts/entertain/200512/200512240189470.html|title =新片情義我心知 黎明飾智障人士| publisher =華視新聞網|language = 中文}}</ref><ref>{{Cite web |url= http://www.cinespot.com/hkmreviews/%E6%83%85%E7%BE%A9%E6%88%91%E5%BF%83%E7%9F%A5|title =情義我心知| author =晨曦| publisher =動映地帶|language = 中文}}</ref><ref>{{cite news |language =中文|author =阿童牧| url = https://hk.entertainment.appledaily.com/entertainment/daily/article/20060108/5553695 |title =熱爆影評：沒想像中低能，比想像中好看|publisher =蘋果日報| date =2016年1月8日}}</ref>。

==主要角色==
* [[黎明|黎明]]－李子俊，患有輕度弱智，個性天真善良，是黃海的國小同學。
* [[杜汶澤|杜汶澤]]－黃海，為人不學無術，在日本新宿以各種方法謀財。
* [[楊貴媚|楊貴媚]]－燕姐，為人豪爽，不拘小節，在日本新宿替黑幫經營酒吧，經常照顧流落街頭的黃海。
* [[葉璇|葉璇]]－葉璇，黃海的前妻，因為不滿丈夫不學無術而與其分手，但其實一直期盼他浪子回頭。
* [[陳文剛|陳文剛]]－俊兄
* [[李家榮|李家榮]]－俊弟
* 蔡靜文－俊妹
