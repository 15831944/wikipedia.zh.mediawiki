{{engname|《情迷大話王》|'' Everyday Is Valentine ''}}是2001年出品的一部香港愛情喜劇電影，由[[王晶|王晶]] 執導、監製及編劇，[[黎明|黎明]]、[[張柏芝|張柏芝]]、[[吳孟達|吳孟達]]、[[苑瓊丹|苑瓊丹]]等主演，在香港、[[澳門|澳門]]及[[尼泊爾|尼泊爾]]拍攝<ref>{{cite news |language =中文| url =https://hk.lifestyle.appledaily.com/etw/magazine/article/20010403/3_1375226/%E9%BB%8E-%E6%98%8E-%E6%9F%8F-%E8%8A%9D-%E6%83%85-%E8%BF%B7-%E6%BE%B3-%E9%96%80|title =黎明、柏芝情迷澳門|publisher =蘋果日報| date =2001年4月3日 }}</ref><ref>{{Cite web |url= http://www.hkbo.com.hk/search-details-new.php?filmid=6433&run=1|title =電影資料庫| publisher = 香港票房有限公司|language = 中文}}</ref><ref>{{Cite web |url= http://hkmdb.com/db/movies/view.mhtml?id=9062&display_set=big5|title =情迷大話王（2001）| publisher =香港影庫|language = 中文}}</ref>。

==情節介紹==
Wonderful是一個在愛情路上徘徊的失意女子，而OK賴則是一個小古惑。故事講述Wonderful感情失意時，偶然在便利店遇上大話王OK賴。法師在法會中，向Wonderful表示她的姻緣已經近在眼前，但卻連系於尼泊爾的聖地，宣稱在那地方她的姻緣才得成就。法師叮囑OK賴以後不可以過於吹噓，否則會斷送大好的姻緣，並法將兩條彩帶分別送給OK賴和Wonderful，增加二人的運氣。某天，兩人再次相遇，Wonderful誤以為OK賴是富家子弟，接受了OK賴的邀請，與他一起吃午餐。Wonderful本來悉心打扮準備應約，然而Wonderful的父親訛稱住在澳門的祖母生病，於是OK賴約她在澳門見面。原來Wonderful的父親在澳門借下了大筆貴利，OK賴不忍Wonderful被傷害，四出借錢救她，Wonderful非常感動，對OK賴萌生愛念，彼此的緣份逐漸拉近。後來OK賴被揭穿並非富豪，一向最痛恨別人講大話的Ｗonderful 感到再受騙，在沒有安全感之下出走，一段大好的姻緣因而掀起了波瀾<ref>{{Cite web |url= http://www.filmcritics.org.hk/film-review/node/2017/06/09/%E6%83%85%E8%BF%B7%E5%A4%A7%E8%A9%B1%E7%8E%8B%EF%BC%9A%E6%8F%9B%E4%B8%8A%E6%96%B0%E8%A3%9D%E7%9A%84%E8%BF%BD%E5%A5%B3%E6%95%85%E4%BA%8B|title =情迷大話王：換上新裝的追女故事| author =葉念琛| date =2001年| publisher = 2001香港電影回顧|via=香港影評庫|language = 中文}}</ref><ref>{{Cite web |url= http://www.cinespot.com/hkmreviews/%E6%83%85%E8%BF%B7%E5%A4%A7%E8%A9%B1%E7%8E%8B|title =情迷大話王|publisher = 動映地帶| language = 中文}}</ref>。
