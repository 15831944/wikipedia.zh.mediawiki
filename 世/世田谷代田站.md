'''世田谷代田站'''（{{jpn|j=世田谷代田駅|hg=せたがやだいたえき|rm=Setagaya-daita eki|f=y}}）是一個位於[[東京都|東京都]][[世田谷區|世田谷區]][[代田_(世田谷區)|代田]]二丁目，屬於[[小田急電鐵|小田急電鐵]][[小田原線|小田原線]]的[[鐵路車站|鐵路車站]]。車站編號是'''OH 08'''。

== 歷史 ==
* 1927年（[[昭和|昭和]]2年）4月1日：車站啟用，最初名為世田谷中原站。
* 1945年（昭和20年）7月1日：車站受到空襲而損毀，不得不暫時關閉。這是小田急中唯一受到空襲的車站。
* 1946年（昭和21年）
** 6月15日：重新啟用。
** 8月20日：改稱世田谷代田站。
* 2006年（[[平成|平成]]18年）3月26日：由於進行複線化工程，臨時搭建的跨站式站房開始提供服務。
* 2013年（平成25年）3月23日：地下急行線設置臨時月台，車站改建為地下站<ref>{{Cite web |url= http://www.shimochika-navi.com/99_backnumber/pdf/vol27.pdf|title= シモチカナビ vol.27 vol.10|accessdate=2015-09-06}}</ref>。
* 2015年（平成27年）8月29日：地上站房南側部分完工，原先在地下2層的剪票口移至地上<ref>{{Cite web |url= http://www.shimochika-navi.com/99_backnumber/pdf/vol200-10.pdf|title= シモチカナビⅡ vol.10|accessdate=2015-09-06}}</ref>。
* 2018年（平成30年）
** 3月3日：複複線化完工，緩行線月台移至地下2樓。地下3樓臨時月台廢除<ref>{{PDFlink|[http://www.odakyu.jp/news/o5oaa10000005sd3-att/8732_1784221_.pdf 2018年3月3日初電から、代々木上原駅～梅ヶ丘駅間において複々線での運転を開始します]}} - 小田急電鉄（2017年12月15日）、2018年3月3日閲覧。</ref>。
** 3月17日：[[東京地下鐵千代田線|東京地下鐵千代田線]]直通列車（各站停車）開始停靠本站。

== 站名由來 ==
開業時稱「'''世田谷中原'''」，源自當地代田村小字。之後1946年「代田村」細分化後更名為「'''世田谷代田'''」。

== 車站結構 ==
地下2層設有1面2線島式月台。車站設施在地面層，大廳南北兩側皆設有檢票口。地面大廳在2017年3月24日設置「小田急環境室」，展示小田急電鐵的環境保全應對與本站地下化工程介紹<ref>[https://www.odakyu.jp/csr/environment_room/ 小田急環境ルームのご案内] - 小田急電鉄（2018年3月4日閲覧）</ref>。

剪票口旁咖啡廳STREAMER COFFEE COMPANY<ref>[https://www.odakyu.jp/station/setagaya_daita/yard/ 世田谷代田駅 店舗・サービス] - 小田急電鉄（2018年3月4日閲覧）</ref>。

=== 變遷 ===
==== 地下化工程前 ====
為[[相對式月台|相對式月台]]2面2線[[地面車站|地面車站]]。付費區大廳直通下行月台，2009年7月24日以前在上行月台設有臨時檢票口（無人）。月台間有天橋，並有個「富士見窗」，可在晴朗時看見[[富士山|富士山]]。

另外，月台有許多古老月台特有的構造，如木造月台壁面與長椅等。

==== 急行線工程完成以前 ====
小田急在[[東北澤站|東北澤]] - [[梅丘站|梅丘]]間進行交通立體化（地下化）及[[複複線|複複線]]化工程，2006年3月26日至2013年3月22日為止使用臨時跨站式站房，同時下行月台側檢票口往新宿方向移動20公尺。2009年7月24日以前上行月台亦設有臨時檢票口。

==== 緩行線工程完成以前 ====
2013年3月23日，地下3層的急行線先行開通，並設置臨時相對式月台。地下化至2015年8月28日為止使用地下2層的臨時車站設施。地面至地下2樓檢票口之間有個10階的階梯<ref>{{Cite news|url=https://shimokita.keizai.biz/headline/1738/|title=地下化後の世田谷代田駅に106段の階段－ネット上で話題に|newspaper=下北沢経済新聞|date=2013-03-28|accessdate=2018-03-03}}</ref>。

2015年8月29日，地面設置南口剪票口。2015年9月當時，地下1層未使用，地面層有剪票口、站務室，地下2樓有廁所，地下3樓有臨時月台。地面與地下2樓之間設有電扶梯與電梯、階梯，地下2樓與月台間沒有電扶梯<ref>{{Cite web |url= http://www.odakyu.jp/station/setagaya_daita/index.html|title= 世田谷代田駅のご案内・周辺情報｜小田急電鉄|accessdate=2015-09-05}}</ref>。

2018年3月3日複複複線化工程完成，緩行線的正式月台啟用，急行線臨時月台停止使用。
