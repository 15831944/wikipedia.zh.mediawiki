'''世田谷線'''（{{jpn|j=世田谷線|hg=せたがやせん|rm=Setagaya sen|f=y}}）是[[日本|日本]][[東京急行電鐵|東京急行電鐵]]（東急）的{{link-ja|軌道 (鐵道)|軌道 (鉄道)|軌道}}線，連結[[東京都|東京都]][[世田谷區|世田谷區]][[三軒茶屋站|三軒茶屋站]]與[[下高井戶站|下高井戶站]]。本線是東急電鐵唯一的[[路面電車|路面電車]]。

路線圖與車站編號為使用的路線顏色為黃色，路線記號為'''SG'''。

== 路線資料 ==
* 路線距離（{{link-ja|營業距離|営業キロ}}）：5.0公里
* [[軌距|軌距]]：1372毫米
* 複線路段：全線
* 電氣化路段：全線（[[直流電|直流電]]600V）
* [[閉塞_(鐵路)|閉塞方式]]：沒有根據{{link-ja|軌道法|軌道法}}設置閉塞裝置<ref name="signal-comment" />
* 保安裝置：軌道信號連動車內警報裝置
* 最高速度：40公里/小時

== 車費 ==
全線車費採用均一制，截至2014年4月1日大人150日圓、兒童80日圓（使用IC卡大人144日圓、兒童72日圓）。
== 車輛 ==
[[File:Tokyu-80.jpg|thumb]]
現在世田谷線使用的車輛是{{link-ja|東急300系電力動車組|東急300系電車|300系}}10編組，所有列車都以2節編組運行。
