'''世界一流大学和一流学科建设'''，简称'''双一流'''，是[[中华人民共和国|中华人民共和国]]自2015年开始实施的[[中華人民共和國高等教育|高等教育]]政策<ref name=ztfa/>，目标是在21世纪中叶建成高等教育强国<ref>{{Cite news |url =http://money.163.com/15/1106/10/B7NVVRHG00253B0H.html |title = 中国启动大学“双一流”建设： 本世纪中叶建成高等教育强国|publisher =东方早报(上海) |language = zh-hans |date = 2015-11-06 |quote =  }}</ref>。2017年9月21日，[[中華人民共和國教育部|教育部]]公布了高校建設名單，共有42所一流大學、140所高校的465個一流學科入選<ref>{{Cite news |url =http://www.moe.gov.cn/srcsite/A22/moe_843/201709/t20170921_314942.html |title = 教育部 财政部 国家发展改革委关于公布世界一流大学和一流学科建设高校及建设学科名单的通知|publisher =中華人民共和國教育部|language = zh-hans |date = 2017-09-21 |quote =  }}</ref><ref>{{Cite news|url =https://www.xcnnews.com/rd/1042218.html|title =雙一流42所高校如何選出?建設有哪些重點工作?|publisher =中國新聞網|accessdate =2017-09-30|language =中文|date =2017-09-21|quote =|archive-url =https://web.archive.org/web/20170921145117/https://www.xcnnews.com/rd/1042218.html|archive-date =2017-09-21|dead-url =yes}}</ref>。

==背景==
1990年代，[[中华人民共和国|中华人民共和国]]相继实施“[[211工程|211工程]]”和“[[985工程|985工程]]”，部分高校与专业學科取得重大发展，高校的實力增加。但是这两项工程存在入选高校身份固化、競爭缺失、重復交叉、高校资源不均等問題<ref>{{Cite news|url =http://www.sd.chinanews.com.cn/2/2016/0630/18351.html|title=985、211时代或终结 高校发展如何破除身份壁垒？|publisher =中国新闻网|language = zh-hans|author= |date = 2016-06-30|quote =  }}</ref>，有改變計畫、加強資源整合的需要<ref name=aaa>{{Cite news|url =http://news.sina.com.cn/c/nd/2016-06-29/doc-ifxtmwei9463278.shtml|title=985、211不再继续？教育部回应:年内启动双一流|publisher =新京报|language = zh-hans|author=韩雪枫 |date = 2016-06-29 |quote = |accessdate = 2017-09-30 }}</ref>。

從2007年至2014年，[[中华人民共和国教育部|中华人民共和国教育部]]每年都会在工作要点中提及“985工程”和“211工程”，其中2013年首次谈到了“加快建设一流大学和重点学科”，自2014年起的工作要点中都谈到了“双一流”建设，而自2015年起的工作要点中已不见“985工程”和“211工程”<ref>{{Cite web |url = http://old.moe.gov.cn//publicfiles/business/htmlfiles/moe/s5987/201301/147136.html |title = 教育部2013年工作要点 |author =  |date =2013-01-23  |publisher =中华人民共和国教育部  |language =  |accessdate = 2017-09-30 |quote =  }}</ref><ref>{{Cite web |url = http://old.moe.gov.cn//publicfiles/business/htmlfiles/moe/s7049/201402/xxgk_163889.html |title = 教育部关于印发《教育部2014年工作要点》的通知 |author =  |date = 2014-01-22 |publisher = 中华人民共和国教育部 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://old.moe.gov.cn//publicfiles/business/htmlfiles/moe/moe_164/201502/183971.html  |title = 教育部2015年工作要点 |author =  |date = 2015-02 |publisher = 中华人民共和国教育部  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.moe.edu.cn/jyb_xwfb/moe_164/201602/t20160205_229511.html |title =教育部2016年工作要点  |author =  |date = 2016-02-05 |publisher =  中华人民共和国教育部 |language =  |accessdate =  |quote =  }}</ref>。

==历史==
2015年8月18日，[[中共中央总书记|中共中央总书记]][[习近平|习近平]]主持的[[中共中央全面深化改革领导小组|中共中央全面深化改革领导小组]]第15次会议审议通过《统筹推进世界一流大学和一流学科建设总体方案》<ref>{{Cite web |url =http://www.mps.gov.cn/n2255079/n4876594/n4974590/n5143628/c5196461/content.html|title = 习近平主持召开中央全面深化改革领导小组第十五次会议|publisher =中华人民共和国公安部 |language = zh-hans |date = 2015-08-18 |quote =  }}</ref>。

2015年10月24日，[[中華人民共和國國務院|国务院]]印发《[[s:统筹推进世界一流大学和一流学科建设总体方案|统筹推进世界一流大学和一流学科建设总体方案]]》，自2016年起针对大学以及学科建设明确提出了“双一流”的任务要求，分三个阶段制定了时间表，其中将“基本建成高等教育强国”的最终目标节点定于21世纪中叶<ref name=ztfa>{{Cite web |url =http://www.gov.cn/zhengce/content/2015-11/05/content_10269.htm |title = 国务院关于印发统筹推进世界一流大学和一流学科建设总体方案的通知|publisher =中国政府网 |language = zh-hans |date = 2015-11-05 |quote =  }}</ref>。

==内容==
===目标===
根据《统筹推进世界一流大学和一流学科建设总体方案》，统筹推进一流大学和一流学科建设将分三步走：
#2020年，若干所大学和一批学科进入世界一流行列，若干学科进入世界一流学科前列。
#2030年，更多的大学和学科进入世界一流行列，若干所大学进入世界一流前列，一批学科进入世界一流学科前列，高等教育整体实力显著提升。
#21世纪中叶，一流大学和一流学科的数量和实力进入世界前列，基本建成高等教育强国<ref name=ztfa/>。

===特色===
2016年6月，教育部官员表示，“双一流”建设的评审标准、资金分布将更注重学科建设，而不是像“985工程”、“211工程”那样向确定的高校拨款，在遴选上加入滚动淘汰机制，“双一流”建设方案将给之前未入选“985工程”、“211工程”的高校机会，而且以前入选“985工程”、“211工程”的高校也不一定会被列入“双一流”建设<ref>{{Cite web |url = http://www.sohu.com/a/100136433_112831 |title = 985、211将废止？教育部年内启动“双一流”高校建设 |author =  |date = 2016-06-30 |publisher = 搜狐|language =  |accessdate =  |quote =  }}</ref>。惟在最終名單中，原985工程高校均進入了一流大學建設名單，原211工程高校均至少有一個學科進入一流學科建設名單。
