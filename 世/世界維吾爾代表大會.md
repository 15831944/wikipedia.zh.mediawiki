'''世界維吾爾代表大會'''（{{lang-ug|دۇنيا ئۇيغۇر قۇرۇلتىيى}}，[[拉丁维文|拉丁维文]]：{{lang|ug-Latn|Dunya Uyghur Qurultiyi}}；縮寫：{{lang|ug-Latn|DUQ}}， {{lang-en|World Uyghur Congress}}，縮寫：{{lang|en|WUC}}），簡稱世維大會或世維會，起於2004年4月16日至19日於[[德國|德國]][[慕尼黑|慕尼黑]]市召開之[[維吾爾族|維吾爾族]]相關團體代表大會，由東突厥斯坦民族大會與世界維吾爾青年代表大會共同召集各相關之[[東突厥斯坦|東突厥斯坦]]組織而籌建的統一領導機構组织<ref name=wuc>世界维吾尔代表大会,[http://www.uyghurcongress.com/cn/AboutWUC.asp?mid=1095738888 “世界维吾尔代表大会简介”],2006年.</ref>。世界維吾爾代表大會每年接受美國國會支持的[[美国国家民主基金会|美国国家民主基金会]]资助<ref>http://www.ned.org/where-we-work/asia/china-xinjiang/east-turkistan{{en}}</ref>。2006年11月，三度獲得提名[[諾貝爾和平獎|諾貝爾和平獎]]的[[热比娅·卡德尔|热比娅·卡德尔]]被選舉为世界維吾爾代表大會第二屆主席，於2009年5月續任第三屆主席<ref name=wuc/><ref>[http://news.rti.org.tw/index_newsContent.aspx?nid=270605 諾貝爾和平獎熱比婭獲邀觀禮] {{webarchive|url=https://web.archive.org/web/20131002054530/http://news.rti.org.tw/index_newsContent.aspx?nid=270605 |date=2013-10-02 }}, [[中央廣播電臺|中央廣播電臺]] Radio Taiwan International, 2010/12/3</ref><ref>[http://www.libertytimes.com.tw/2006/new/sep/12/today-int4.htm 疆獨鬥士熱比婭 獲諾貝爾和平獎提名] {{webarchive|url=https://web.archive.org/web/20070629181927/http://www.libertytimes.com.tw/2006/new/sep/12/today-int4.htm |date=2007-06-29 }}, [[自由時報|自由時報]], 2006年9月12日</ref><ref>[http://www.amnesty.org.au/china/comments/9233 Fighting for her peoples' rights: Rebiya Kadeer visits Australia], [[國際特赦組織|國際特赦組織]], 2008年2月{{en}}</ref><ref>[http://tech.mit.edu/V127/N27/kadeer.html Visitor Kadeer Calls For Action to Help Uyghur People], The Tech - MIT's Newspaper, June 8, 2007{{en}}</ref>。现任世维会主席的是[[多里坤·艾沙|多里坤·艾沙]]。

== 宗旨 ==
世維大會總部設在德國，被认为是爭取东突厥斯坦（新疆）獨立的主流組織。世維大會發言人迪里夏提表示：「大會不支持、不鼓勵、不行使武裝戰爭來爭取东突厥斯坦（新疆）獨立，而且大會一直提倡以非暴力進行抗爭。大會的工作方向是一直敦促中國當局尊重維吾爾人的政治歸屬，並通過民主的方式來解决雙方的爭端。」<ref>{{cite news|title=维吾尔大会反对以暴力手段解决新疆问题|author=|date=2005-09-30|publisher=[[東突厥斯坦信息中心|東突厥斯坦信息中心]]|url=http://www.uygur.org/china/et/2005/0930c.htm|accessdate=2008-07-20|deadurl=yes|archiveurl=https://web.archive.org/web/20080603143930/http://www.uygur.org/china/et/2005/0930c.htm|archivedate=2008-06-03}}</ref>大会宣布打算与世界各国政府，形成“和平反对”中国政府在新疆的政策<ref name="unpo">[http://www.unpo.org/content/view/547/236/ Newly founded World Uyghur Congress calls for peaceful solution in East Turkestan]. [[非聯合國會員國家及民族組織|非聯合國會員國家及民族組織]]. 2004年4月22日{{en}}</ref>。它指称维吾尔族人受到的对待有把该地区变成“定时炸弹”的危险<ref>{{cite web |url = http://news.bbc.co.uk/2/hi/asia-pacific/4297070.stm |publisher = BBC News |title = China given warning on Xinjiang |date=30 September 2005 | accessdate=24 January 2010}}{{en}}</ref>。本組織主要使用之維吾爾語類型為拉丁维文（{{lang|ug|ULY、Uyghur Latin Yéziqi}}）。

此組織被[[中华人民共和国政府|中华人民共和国政府]]視為進行[[国家分裂|國家分裂]]活动的恐怖組織<ref>{{cite news |title = 揭露热比娅和“世维会”真面目|author=|date=2009-07-07|publisher=[[中国网|中国网]] |url = http://www.china.com.cn/news/txt/2009-07/07/content_18088564.htm|accessdate=2009-07-07}}</ref>。中國政府称该组织和其领导人[[热比娅|热比娅]]为[[乌鲁木齐七·五事件|乌鲁木齐七·五事件]]的幕后主使<ref>{{cite news|title=揭开"恐怖分子"热比娅的画皮|author=|date=2009-07-07|publisher=[[东方网|东方网]]|url=http://www.eeo.com.cn/blog/shwx/2009/07/07/142968.shtml|accessdate=2009-07-07|deadurl=yes|archiveurl=https://web.archive.org/web/20131102230451/http://www.eeo.com.cn/blog/shwx/2009/07/07/142968.shtml|archivedate=2013-11-02}}</ref>。
中国官媒称该组织曾在7·5事件之后广泛制造谣言攻击中国政府“屠杀”维吾尔族民众以煽动仇恨。比如该组织领导人热比娅用湖北[[石首市|石首]]一起民众抗议活动照片指控中国政府镇压维吾尔族和平示威<ref name="谣言1">{{cite web|url=http://china.nfdaily.cn/tf/content/2009-07/09/content_5346902.htm|title=热比娅用石首“调包”照片掩盖真相欺骗世界|work=[[环球时报|环球时报]]|publisher=[[南方报业传媒集团|南方报网]]|access-date=2014-03-03|archive-url=https://web.archive.org/web/20091023142521/http://china.nfdaily.cn/tf/content/2009-07/09/content_5346902.htm|archive-date=2009-10-23|dead-url=yes}}</ref>；又比如该组织在土耳其首都[[安卡拉|安卡拉]]的街头宣传中，将一起发生在[[杭州市|杭州]]的交通事故指为[[中国人民解放军|解放军]]对维吾尔族民众的屠杀。<ref>{{Cite web |url = http://news.xinhuanet.com/politics/2009-07/11/content_11691930_1.htm） |title = 热比娅：一个陷入自相矛盾的人 |author = 俞铮|date = 2009-07-11 |publisher = 新华社 |language = zh-hans |accessdate = 2015-04-09 }}</ref>

== 組織沿革 ==
1990年代在各國設立的各式維吾爾民族運動組織開始進行強化統合機制。

首於1992年、以[[土耳其|土耳其]]退役將校力薩·別金為指導中心，在土耳其[[伊斯坦布尔|伊斯坦布尔]]舉辦「東突厥斯坦民族大會（拉丁维文：{{lang|ug-Latn|Sherqiy Türkistan Milliy Qurultiyi}}、英語：{{lang|en|East Turkestan National Congress}}）」，始將世界各國獨立的維吾爾民族運動組織及個人加以統合。再於1996年、以學生運動家[[多里坤·艾沙|多里坤·艾沙]]為指導中心，在德國慕尼黑設立「世界維吾爾青年代表大會（拉丁维文：{{lang|ug-Latn|Dunya Uyghur Yashliri Qurultiyi}}、英語：{{lang|en|World Uyghur Youth Congress}}）」<small><ref>水谷(如參考文獻[1]) pp.70-88.</ref></small>。

2004年4月16日、以「東突厥斯坦民族大會」與「世界維吾爾青年代表大會」兩個組織為主、且統合其它大小相關的維吾爾組織成為新的單一組織「世界維吾爾代表大會」。被選出的第一屆主席為[[自由欧洲电台|自由欧洲电台]]前職員艾爾肯·阿布甫泰肯（為[[中華民國|中華民國]]前[[新疆维吾尔自治区|新疆省]]維籍幹部及1936年[[中華民國國民大會|國大代表]][[艾沙·玉素甫·阿布甫泰肯|艾沙]]之子）。接著於2006年11月之後[[热比娅·卡德尔|热比娅·卡德尔]]繼任為第二、三屆主席。2017年11月，在世界维吾尔代表大会第六次特别会议上，多里坤·艾沙当选为主席。

2012年5月14日，世界维吾尔会议第四届代表大会在东京开幕，来自世界各地代表以及支援维吾尔族问题的日本国会议员等出席了开幕式。会议主席热比亚说希望与中国政府对话，要求获得民族自决权。<ref>http://yibada.com/news/view/21323415 {{dead link}}世界维吾尔大会（世维会）在东京召开</ref>世维会成员同日参观了[[靖国神社|靖国神社]]。<ref>{{Cite web |url = http://www.boxun.com/news/gb/china/2013/06/201306231043.shtml#.VP4jMVXF8fZ |title = 热比娅主席访问日本出席“日本、维吾尔争取自由互援恳谈会” |author = 荷香 |date = 2013年6月23日 |publisher = 博讯  |language = zh-cn |accessdate = 2015年3月9日 |quote = 2012年5月世界维吾尔会议第四届代表大会期间，热比娅主席曾参拜靖国神社 }}</ref>

* 第二屆主席（2006年11月－2009年5月）：[[热比娅·卡德尔|热比娅·卡德尔]]
* 第三屆主席（2009年5月－2017年11月）：[[热比娅·卡德尔|热比娅·卡德尔]]
* 第四届主席 (2017年11月-至今）： [[多里坤·艾沙|多里坤·艾沙]]

=== 世维大會組織領導機構 ===
其组织机构由世界各地的代表选举产生。任期为3年。设主席，监察长、秘书长各一人，副职若干；新闻发言人两人；下设执行委员会及若干专门委员会。<ref>{{Cite web |url = http://www.uyghurcongress.org/cn/?p=4336 |title = 领导机构 |publisher = 世界维吾尔代表大会 |language = zh-hans |accessdate = 2014-04-22}}</ref>
