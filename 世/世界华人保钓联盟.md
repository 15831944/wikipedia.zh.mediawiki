== 历史 ==
2011年1月2日，世界华人保钓联盟正式在香港成立。

2012年4月19日，世界华人保钓联盟协会在香港成立。

== 保钓运动 ==
2012年1月3日，世界华人保钓联盟的8个成员在[[香港|香港]]乘坐[[渔船|渔船]]前往钓鱼岛宣誓主权，但是遭到香港海事部门的阻止而行动取消。<ref>{{cite web |url= http://news.ifeng.com/mainland/detail_2012_01/03/11731543_0.shtml|title= “世界华人保钓联盟”驾船前往钓鱼岛 被香港水警截回|accessdate=2012年1月3日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>

2012年6月16日，世界华人保钓联盟组织[[两岸四地|两岸四地]]的保钓[[民族英雄|民族英雄]]从大陆前往钓鱼岛宣誓主权，但是中国大陆的参加人士被[[中华人民共和国政府|中华人民共和国政府]]请回，导致行动被迫取消。<ref>{{cite web |url=http://news.ifeng.com/taiwan/3/detail_2012_06/16/15349379_0.shtml |title= 台保钓人士：大陆保钓成员已被请回 行动确定取消|accessdate=2012年6月16日  |publisher=凤凰网 |work= |language =zh-cn}}</ref>

2012年8月12日，15位香港保钓人士，包括出任團長的[[人民力量|人民力量]]成員[[楊匡|楊匡]]、社民連[[2012年香港立法會選舉|立法會選舉]]候選人[[曾健成|曾健成]]、[[香港公民黨|香港公民黨]]新西支部執委[[王化民|王化民]]及保釣委員會委員羅堪就等人从[[尖沙咀|尖沙咀]]乘坐“启丰二号”起航，前往[[台湾海峡|台湾海峡]]，将在[[彭佳屿|彭佳屿]]和[[澳门|澳门]]、[[台湾|台湾]]、[[厦门|厦门]]的保钓人士会和，一同前往钓鱼岛宣示主权，出发前，香港保釣行動委員會主席[[陈妙德|陈妙德]]对[[法新社|法新社]]记者说：“如果此次遭到日方船只的拦截，我们将会反抗。同时我们也希望，[[中国人民解放军|中国人民解放军]]的船只能够为我们保驾护航。”<ref>{{cite web |url= http://news.ifeng.com/mainland/special/diaoyudaozhengduan/content-3/detail_2012_08/13/16761350_0.shtml|title= 香港保钓人士：遇拦截将反抗 望解放军护航|accessdate=2012年8月13日 |publisher= 凤凰网|work= |language =zh-cn}}</ref><ref>{{cite web |url= http://news.ifeng.com/mainland/special/diaoyudaozhengduan/content-3/detail_2012_08/12/16746041_0.shtml|title=台湾保钓人士：香港保钓船若顺利出海 2天后可会合 |accessdate=2012年8月12日 |publisher= 凤凰网|work= |language =zh-cn}}</ref> 台湾的保钓人士由黄锡林带队，计划从钓鱼岛的归属地台湾[[宜兰县|宜兰县]]出发。<ref>{{cite web |url= http://news.ifeng.com/mainland/special/diaoyudaozhengduan/content-3/detail_2012_08/13/16767968_0.shtml|title=港台联动 台湾保钓团体计划宜兰出海 |accessdate= 2012年8月13日|publisher=凤凰网 |work= |language =zh-cn}}</ref>

2012年8月14日，台湾原本打算14日出海，但是船东受到台当局的压力，不同意出海，即使是给香港保钓船的补给，也不同意。厦门保钓人士迫于大陆当局的压力和天气原因而取消前往钓鱼岛宣示主权，面对官方不予支持，世界华人保钓联盟秘书长李义强委婉地说：“在放开民间这个问题上还有很大的保守。现在香港的船已经到达海域，同时又一个台风已经形成，所以现在就算能走，估计也已经来不及了。……我们也希望政府在这个问题上对民间保钓给予跟大空间。”<ref>{{cite web |url= http://news.ifeng.com/mainland/special/diaoyudaozhengduan/content-3/detail_2012_08/14/16800416_0.shtml|title= 内地保钓人士称因天气原因取消赴钓鱼岛行程|accessdate= 2012年8月14日|publisher=凤凰网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://news.ifeng.com/mainland/special/diaoyudaozhengduan/content-3/detail_2012_08/14/16800563_0.shtml |title=台保钓人士赴钓岛行程受阻 船东受压力反悔坚决不开船 |accessdate= 2012年8月14日|publisher= 凤凰网|work= |language =zh-cn}}</ref>

2012年8月15日，香港保钓人士乘坐“启丰二号”抵达钓鱼岛附近海域，由于日本[[海上保安厅|海上保安厅]]派出了20多艘船隻进行拦截、[[直升机|直升机]]进行监视，使用撞击、高压水炮等方法阻止保钓人士登島，他们當中7人跳入海中，游泳登陆钓鱼岛，原本计划插上两岸政府旗帜，但是在钓鱼岛上守候的30多名沖繩縣警察隨即以违反日本《出入境管理及难民认定法》為由拘捕7人，而被日本海上保安廳撞损的“启丰二号”上的8位保钓人士、4位船员、2位[[凤凰卫视|凤凰卫视]][[记者|记者]]也被沖繩縣警帶走，<ref>{{cite web |url=http://world.huanqiu.com/regions/2012-08/3029675.html |title= 香港保钓船上14名保钓人士全部被日方扣留|accessdate= 2012-08-15 |publisher=环球时报 |work= |language =zh-cn}}</ref> 日本首相[[野田佳彦|野田佳彦]]表示将按照日本法律进行严肃处理。<ref>{{cite web |url= http://news.ifeng.com/mainland/special/diaoyudaozhengduan/content-3/detail_2012_08/15/16830652_0.shtml|title=日首相称将“依法”严惩中国香港保钓人士 |accessdate= 2012年8月15日|publisher= 凤凰网|work= |language =zh-cn}}</ref> 同日，[[中国民间保钓联合会|中国民间保钓联合会]]和世界华人保钓联盟的成员在[[北京|北京]][[日本驻华大使馆|日本驻华大使馆]]拉横幅和请愿，要求「中华人民共和国政府收复失地」，[[湖南省|湖南省]][[长沙市|长沙市]]步行街黄兴雕像前也出现了拉横幅抗议「日本侵占钓鱼岛」的保钓人士。<ref>{{cite web |url=http://news.qq.com/a/20120815/001108.htm#p=1 |title=高清：中国多地爆发反日示威游行 |accessdate=2012-08-15 |publisher=腾讯网 |work= |language =zh-cn}}</ref> 保钓人士聲稱在扣押期間「不允许躺着或靠着休息」，受到了「非常粗暴和不公正的对待」。<ref>{{cite web|url=http://news.ifeng.com/mainland/special/diaoyudaozhengduan/content-3/detail_2012_08/19/16912151_0.shtml |title=保钓人士遭日本扣押过程：不给水喝不让大小便 |accessdate=2012年8月19日 |publisher=凤凰网 |work= |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20120819073045/http://news.ifeng.com/mainland/special/diaoyudaozhengduan/content-3/detail_2012_08/19/16912151_0.shtml |archivedate=2012年8月19日 }}</ref><ref>{{cite web|url=http://mil.huanqiu.com/china/2012-08/3040259.html|title=日虐待保钓人士细节：不许躺下休息不准靠着|accessdate=2012-08-19|publisher=环球网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120828092512/http://mil.huanqiu.com/china/2012-08/3040259.html|archivedate=2012-08-28}}</ref>

2012年8月15日深夜，香港特首[[梁振英|梁振英]]召见日本驻香港总领事[[隈丸优次|隈丸优次]]：“港府对事件极度关注，已透过总领事向日本政府重申钓鱼岛和其附属岛屿自古以来就是中国领土，香港居民多年来对这些岛屿有强烈感情，不希望见到日本政府作出任何被香港居民视为挑衅性的行为。（我们）要求日本政府不能危及香港居民和其他中国公民的人身和财产安全，并促请日本政府尽快释放所有香港居民和其他中国公民。”<ref>{{cite web |url= http://news.sohu.com/20120816/n350790908.shtml|title= 梁振英召见日本总领事 入境处官员赴日援助|accessdate= 2012年8月16日|publisher=搜狐网 |work= |language =zh-cn}}</ref> 15日晚上，[[华盛顿中国和平统一促进会|华盛顿中国和平统一促进会]]发表声明，敦促日本政府无条件施放香港保钓人士。<ref>{{cite web|url=http://world.huanqiu.com/regions/2012-08/3030448.html|title=华盛顿资深保钓团体声援香港保钓人士促日放人|accessdate=2012-08-16|publisher=环球时报|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120818152922/http://world.huanqiu.com/regions/2012-08/3030448.html|archivedate=2012-08-18}}</ref> 而在日本国内，有部分較激進的網民在網上表達不滿。<ref>{{cite web |url= http://world.huanqiu.com/exclusive/2012-08/3029823.html|title=中国人登岛震惊日本 日网民叫嚣“枪决” |accessdate=2012年8月16日 |publisher=环球时报 |work= |language =zh-cn}}</ref>

2012年8月16日，中华民国外交部政务次长[[董国猷|董国猷]]约见日本驻台代表[[樽井澄夫|樽井澄夫]]说：“香港保钓人士带[[中华民国国旗|中华民国国旗]]登上钓鱼岛，为自发行为，外交部事先不知情，也与中华民国政府无涉”。<ref>{{cite web |url=http://taiwan.huanqiu.com/news/2012-08/3034960.html |title=台“外交部”向日本解释：事先不知保钓者登岛 |accessdate= 2012-08-17 |publisher=环球时报 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://news.ifeng.com/taiwan/3/detail_2012_08/17/16877256_0.shtml?_from_ralated |title= 台官方：香港保钓人士自发带旗登岛 台方事先不知情|accessdate= 2012年8月17日|publisher=凤凰网 |work= |language =zh-cn}}</ref>
