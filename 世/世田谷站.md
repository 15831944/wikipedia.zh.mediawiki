'''世田谷站'''（{{jpn|j=世田谷駅|hg=せたがやえき|rm=Setagaya eki|f=y}}）是一個位於[[東京都|東京都]][[世田谷區|世田谷區]][[世田谷_(世田谷區)|世田谷]]四丁目，屬於[[東京急行電鐵|東京急行電鐵]][[世田谷線|世田谷線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''SG05'''。

== 年表 == 
* 1925年（[[大正|大正]]14年）1月18日 - 開業。
* 2013年（[[平成|平成]]25年）3月31日 - 伴隨世田谷1號平交道擴充，上下線月台向三軒茶屋方向移動約10公尺。

== 車站構造 ==
[[對向式月台|對向式月台]]2面2線的[[地面車站|地面車站]]。

通常是[[無人站|無人站]]，但每年12月與1月合計4日舉辦{{link-ja|襤褸市|ボロ市}}活動時將配置臨時的站員，接受車費、上下車整理等。

== 利用狀況 ==
