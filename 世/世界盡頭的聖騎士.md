{{日本作品|世界盡頭的聖騎士|最果てのパラディン}}是一部由柳野{{lang|ja|かなた}}創作的輕小說<ref name="秦野0821" />，插畫由{{link-ja|輪くすさが}}負責<ref name="gnn0420" />。小說於2015年5月1日起於小説投稿網站「[[成為小說家吧|成為小說家吧]]」開始連載<ref group="小說家">{{cite web |author=-{柳野かなた}- |url=https://ncode.syosetu.com/n5115cq/ |title=-{最果てのパラディン}- |date=2015-05-01 |website=[[成為小說家吧|小説家になろう]] |language=ja |accessdate= 2018-04-23}}</ref>，其後由[[OVERLAP|OVERLAP]]的OVERLAP文庫出版<ref name="Pineda1117" />。漫畫版於《{{tsl|ja|コミックガルド|COMIC GARDO}}》上連載<ref group="OVL部落格">{{cite web |url=http://blog.over-lap.co.jp/paladin4_0825/ |title=-{『最果てのパラディンⅣ』カバーイラスト公開＆コミカライズ決定！}- |website=-{オーバーラップ広報室}- |date= 2017-08-25 |language=ja |accessdate=2018-04-23  |archiveurl=https://web.archive.org/web/20170901233626/http://blog.over-lap.co.jp/paladin4_0825/  |archivedate=2017-09-01 |dead-url=no}}</ref><ref name="natalie0926" />。

本作分別於2017年及2018年的《[[這本輕小說真厲害！|這本輕小說真厲害！]]》的年度文庫中排名第19位<ref name="玻璃1124" />及第14位<ref name="宝島社2018" />。

== 故事簡介 ==
過去被毀滅的死者之街──在這樣一處冷僻的土地有一名活人小孩，名叫威爾。養育這位少年的，是三名不死族──豪邁的骷髏劍客「布拉德」、端莊的神官木乃伊「瑪莉」以及乖僻的魔法師幽靈「古斯」。在三人的教育與疼愛中，少年日漸茁壯。一天，少年不禁產生疑惑：「……現在的『我』究竟是誰？」不死族們隱藏於世界盡頭之街的祕密終被威爾解開。善良之神們的愛與慈悲，邪惡之神們的偏執與瘋狂。「這是約定。雖然講起來會有些長，不過就讓我告訴你。關於許多英雄們與我們的死……也關係到你之所以活在這裡的緣由。」──明白一切之後，少年決心踏上了聖騎士之路。

== 登場人物 ==
=== 主角 ===
; {{nihongo|威廉|ウィリアム}}
: 本作主角，被布拉德、瑪麗及卡斯暱稱為'''威爾'''。燈火女神葛雷斯菲爾的使徒，人稱「邊境的燈火」及「世界盡頭的聖騎士」。似乎喜歡著燈火女神想向她告白。

=== 死者之街 ===
; {{nihongo|布拉德|ブラッド}}
: 在廢墟之街養育威廉的三名不死族之一。200年前封印上王的三英傑之一，生前為劍士，別名為「{{nihongo|戰鬼|ウォーオウガ}}」。被威爾視作父親。

; {{nihongo|瑪麗|マリー}}
: 在廢墟之街養育威廉的三名不死族之一。200年前封印上王的三英傑之一，生前為神官，別名為「{{nihongo|地母神的愛女|地母神（マーテル）の愛娘}}」。被威爾視作母親。

; {{nihongo|古斯|ガス}}
: 在廢墟之街養育威廉的三名不死族之一。200年前封印上王的三英傑之一，生前為大魔法師，別名為「{{nihongo|徬徨的賢者|ワンダリングセイジ}}」。被威爾視作祖父。

=== 神明 ===

; 燈火之女神/葛雷斯菲爾
<blockquote>掌管轉生與輪迴的女神同時也是不死之女神的姊姊，性格相當寡言，當初將威爾的靈魂指引致這個世界。在威爾初次對戰不死神時，賜與他輪迴女神的加護並成為他的守護神。</blockquote>

; 不死之女神/絲塔古內特
<blockquote>掌管不死與永生的女神同時也是燈火之女神的妹妹，本是善良陣營的神明，卻因無法忍受生死的悲劇，而走向歧路。非常中意威爾想讓其成為自己的下僕。</blockquote>
