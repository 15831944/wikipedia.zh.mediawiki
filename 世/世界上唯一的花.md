《'''世界上唯一的花'''》（{{lang-ja|世界に一つだけの花}}）是[[日本|日本]][[偶像_(日本)|偶像]][[男子音乐组合|男子组合]][[SMAP|SMAP]]的第35张[[单曲|单曲]]，于2003年3月5日由[[勝利娛樂|JVC建伍胜利娱乐株式会社]]发行。

== 单曲背景 ==
最初这首歌收录于SMAP在2002年7月24日的[[专辑|专辑]]《{{Link-ja|SMAP 015/Drink! Smap!|SMAP 015/Drink! Smap!}}》。词曲作者[[槙原敬之|槙原敬之]]表示这首歌曲是一接到任务就开始创作，但直到截止日才完成。<ref name="asashi">[http://www.asahi.com/showbiz/music/TKY200912100253.html 「私」「人生」…　槇原敬之デビュー20年のベスト盤]『[[朝日新聞|朝日新聞]]』2009年12月10日夕刊</ref>

专辑发行之后，这首歌就成为专辑中最受欢迎的歌曲。而在[[关西电视台|关西电视台]]和[[富士电视台|富士电视台]]联合制作的电视节目《[[SMAP×SMAP|SMAP×SMAP]]》特别节目“同学年”中，由[[木村拓哉|木村拓哉]]演绎的这首歌一跃成为当时的热门话题。<ref group="注">正確には木村単独ではなく、出演者との別撮りによるパート分けがされている。</ref>

这首歌曲被更多的日本人注意到是在次年，2003年1月－3月在[[富士电视网|富士电视网]]播出了由关西电视台和[[共同電視|共同电视台]]联合制作的电视剧《{{Link-ja|我的生存之道|僕の生きる道}}》，该剧由[[草彅刚|草彅刚]]主演，而主题曲就选用了本歌。單曲並沒有拍攝[[音樂錄像帶|音樂錄像帶]]，宣傳所用影像片段大多由現場演唱的視頻組成。

== 销售成绩 ==
單曲的銷售取得了巨大的成功。在[[Oricon公信榜|Oricon公信榜]]總共7週佔據冠軍位置，其中有三週還是在2004年由於[[NHK紅白歌合戰|NHK紅白歌合戰]]的效應，時隔39個星期重返榜首達成的。以絕對的優勢成為[[Oricon單曲年榜冠軍作品列表|2003年度日本單曲銷量冠軍]]，到2004年仍然是年度單曲銷量第11位。Oricon統計的總銷量接近260萬張，成為[[日本最暢銷單曲列表|日本史上銷量第9高的單曲]]。

連續兩年（2003年度、2004年度）榮獲[[JASRAC獎|JASRAC獎]]金獎，表明這首歌是年度獲得著作權使用費最多的歌曲。

同時也是2003年度日本音樂界最高榮譽——[[日本唱片大獎|日本唱片大獎]]先前的最大熱門，但是由於SMAP所屬的[[傑尼斯事務所|傑尼斯事務所]]拒絕參與評審類獎項，因此最終並沒有獲得。

是2003年度日本[[卡拉OK|卡拉OK]]點唱次數最多的歌曲。在2005年《[[COUNT_DOWN_TV|COUNT DOWN TV]]》的歷代卡拉OK點唱排名中，僅僅誕生兩年多的《世界上唯一的花》就高居第2名，僅次於[[安室奈美惠|安室奈美惠]]的《[[CAN_YOU_CELEBRATE?|CAN YOU CELEBRATE?]]》（1997年）。

2016年SMAP解散，引發單曲购买熱潮，成為日本單曲史上第三張突破300萬的作品。

== 翻唱纪录 ==
自从SMAP推出该首歌并走红之后，许多歌手都翻唱过这首歌。其中词曲作者槙原敬之也曾翻唱过，而且SMAP后也推出了该首歌的官方中文版。

=== 槙原敬之翻唱版 ===

=== SMAP中文版 ===
2011年3月11日，日本[[東北地方|東北]]和[[關東地方|關東]]發生有紀錄以來最強烈的[[2011年日本東北地方太平洋近海地震|9.0級大地震]]，引發許多次生災害，造成大量民眾傷亡。3月18日，這首歌正式發布中文版，定名為《'''世界上唯一的花'''》，[[台灣|台灣]]製作人[[林明阳|林明阳]]負責中文填詞。由[[avex|avex中國]]發行，在[[中國移動|中國移動]]發布手機彩鈴和全曲下載服務，所得部分收益用於地震賑災。同時中文版歌曲被用作春季AUPRES「世界上唯你的白皙」的活動推廣主題曲。<ref>[http://roll.sohu.com/20110319/n304564525.shtml SMAP新歌部分收入捐日本灾区]</ref>2011年5月21日，SMAP於東京會見[[中國|中國]][[國務院總理|國務院總理]][[溫家寶|溫家寶]]時，現場用中文演唱了本曲<ref>[http://news.sina.com.cn/c/p/2011-05-22/042022507577.shtml 温家宝会见日本人气演唱组合SMAP成员(图)]（2011年5月22日；东方早报）</ref>。

== 反响评论 ==
一般認為這首歌之所以取得這麼大的成功在於歌曲中蘊含的濃厚的[[人文主義|人文主義]]、多元文化與價值和平相處的色彩。歌詞中的一句「無法成為No.1也好，原本就是特別的Only One」（{{lang|ja|No.1 にならなくてもいい　もともと特別な Only One}}）成為日本街知巷聞的名句。

而[[伊拉克戰爭|伊拉克戰爭]]更是與這首歌的成功有很大關係。當時正在伊拉克戰爭爆發前夕，日本雖然沒有直接參戰，但作為[[美國|美國]]的親密盟國，事實上支持對伊拉克的入侵。不過日本民眾普遍厭倦戰爭，反對開戰，在這種情況下，強調「每個人都是獨特的個體，不要作比較，也不用爭第一」的《世界上唯一的花》使廣大民眾取得共鳴。日本知名播報員[[筑紫哲也|筑紫哲也]]也認為這首歌是反戰歌。

2005年，這首歌被選為[[發現號太空梭|發現號太空梭]]第31次太空飛行「[[STS-114|STS-114]]」的第十三日的起床音樂，日本宇航员野口在航天飞机上对这首歌进行了简单的评述（当日NASA “Wake-up calls”的[http://spaceflight.nasa.gov/gallery/audio/shuttle/sts-114/wave/fd13.wav WAV]和[http://spaceflight.nasa.gov/gallery/audio/shuttle/sts-114/mp3/fd13.mp3 MP3]录音）。

2007年，被日本[[文化廳|文化廳]]選定為「[[日本歌百選|日本歌百選]]」之一。<ref>[http://www.bunka.go.jp/uta100sen/index.html {{lang|ja|親子で歌いつごう 日本の歌百選}}] {{webarchive|url=https://web.archive.org/web/20130121095234/http://www.bunka.go.jp//uta100sen/index.html |date=2013-01-21 }}</ref>
