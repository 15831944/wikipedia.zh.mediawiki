'''世新大學'''（'''Shih Hsin University'''，簡稱「'''世新'''」），位於[[中華民國|中華民國]][[台北市|台北市]][[文山區|文山區]]的一所私立大學。以傳播學群起家的大學。

創辦人[[成舍我|成舍我]]於1933年創立「'''北平新聞專科學校'''」，政府遷台後於1956年創立'''世界新聞職業學校'''，並秉持「學校為學生而辦，學生為讀書而來」的辦學原則。

1960年，改名'''世界新聞專科學校'''。

1991年，改制'''世界新聞傳播學院'''。

1997年，改名'''世新大學。'''

== 沿革 ==
[[File:SHU2.JPG|thumb]]
[[File:College_of_Management_Building,_Shih_Hsin_University.jpg|thumb]]
=== 世界新聞高級職業學校（1956~1960） ===
*1956年9月，'''世界新聞高級職業學校'''於[[台北|台北]][[木柵|木柵]]溝子口創校，設置編輯採訪科，分初級班、高級班上課。

=== 世界新聞專科學校（1960~1991） ===

*1960年8月，改制'''世界新聞專科學校'''，設置三年制、五年制專科新聞科。
*1962年，因應新聞產業變化及教學需要，新聞科分拆為報業行政科、編輯採訪科及廣播電視科。
*1963年，增設公共關係科。
*1964年，增設圖書資料科。
*1966年，增設電影製作科，為臺灣第一個電影製作科系。
*1969年，增設印刷攝影科。五年制公共關系科停招。
*1971年，電影製作科分設技術組、編導組。
*1976年，增設觀光宣導科

=== 世界新聞傳播學院（1991~1997） ===

*1991年，改制'''世界新聞傳播學院'''，設置新聞學系<ref>編輯採訪科改制</ref>（分設編輯採訪組、編輯翻譯組）、公共傳播學系<ref>公共關係科改制。</ref>（分設公共關係組、廣告組、觀光組）、視聽傳播學系<ref>廣播電視科及電影製作科合併改制。</ref>（分設廣播組、電影組、電視組）、印刷攝影學系<ref>印刷攝影科改制。</ref>（分設印製組、攝影組）、傳播管理學系<ref>報業行政科改制。</ref>（分設媒體經營組、資訊管理組）。
*1992年，增設口語傳播學系。停招圖書資料科。
*1993年，增設社會心理學系。公共傳播學系觀光組獨立為觀光學系。
*1994年，增設傳播研究所碩士班、經濟學系。印刷攝影系改名平面傳播科技學系。傳播管理學系資訊管理組獨立為資訊管理學系。
*1995年，增設圖書資訊學系、財務金融學系。
*1996年，增設行政管理學系、法律學系。新聞學系編輯翻譯組改名國際傳播組。

=== 世新大學（1997~迄今） ===

*1997年，改名'''世新大學'''，設'''新聞傳播學院'''、'''管理學院'''、'''人文社會學院'''及'''法學院'''等四個學院。視聽傳播學系改名廣播電視電影學系。增設社會發展研究所碩士班、英語學系。
*1998年，增設傳播研究所博士班、中國文學系。
*1999年，增設經濟學系碩士班。
*2000年，增設圖書資訊學系碩士班、行政管理學系碩士班。
*2001年，增設平面傳播科技學系碩士班、傳播管理學系碩士班及碩士在職專班、行政管理學系碩士在職專班、經濟學系碩士在職專班。圖書資訊學系改名資訊傳播學系。
*2002年，公共關係學系改名公共關係暨廣告學系，並成立碩士班。增設平面傳播科技學系碩士在職專班、日本語文學系進修部。
*2003年，增設性別研究所碩士班、新聞學系碩士班、口語傳播學系碩士班、中國文學系碩士班、數位媒體設計學系（分設動畫設計組、遊戲設計組）。
*2004年，新聞學系取消分組。平面傳播科技學系改名圖文傳播暨數位出版學系，並取消分組。傳播管理學系自管理學院改隸新聞傳播學院。日本語文學系成立日間部。
*2005年，增設智慧財產權法律研究所碩士班、行政管理學系博士班、企業管理學系。
*2007年，增設中國文學系博士班。
*2012年，增設國際廉能治理碩士在職學位學程(IMIG)。傳播研究所博士班改名傳播博士學位學程。
*2013年，增設數位媒體設計學系碩士班、傳播匯流與創新管理學士學位學程。
*2015年，增設非營利與社會企業碩士在職學位學程。
*2017年，增設全媒體學士學位學程。
*2018年，1111人力銀行調查，雇主最滿意大學「大眾傳播學群類」第一名。Cheers雜誌調查，「企業最愛碩士生」私校第一名。
*2019年，教育部公佈「107學年度大專校院學制註冊率」，世新大學註冊率93.9％ 穩坐非醫學院私立綜合性大學第一名。與CNN建立夥伴關係，成為CNN在台灣第一所夥伴大學。「DailyView 網路溫度計」2018大學競爭力排名Top50出爐，因開設YouTuber創作者課程培養人才，全國競爭力排名第32名。

== 校園環境 ==
因校地不大且資源有限，宿舍名額因此不足，校方設有「生活之窗」提供租屋資訊及代徵室友等服務，學習資源中心則提供討論室、舉辦讀書會活動、免費學業問題幫助等服務。

靠近校園中心的言論廣場設有英語沉浸園及日語沉浸園，可在此使用英文及日語聊天作普通會話，並不時舉辦下午茶、萬聖節、茶道等具文化特色之活動；除了兩側之英日語沉浸園，此處也是各類活動舉辦的場地，舉凡社團的成果展、學校的各類展示、宣傳活動等。

在舍我樓一樓設有學生餐廳與衛保組。鄰近操場的禮堂大樓，除了是各類社團之社桌聚集地，並常有各類校內外大型活動與研討會在此舉辦。

== 圖書館 ==
世新大學圖書館成立於1956年，最初僅是一小間圖書室，經過二度搬遷，1984年才興建新大樓。目前館舍位於行政大樓左側，1986年正式啟用。

為一獨立六層樓建築，內設有期刊資訊室、圖書資訊室、光碟資料庫檢索區、漫畫區、視聽區、語言學習區、團體視聽室和三間閱覽室。並於1998年於一樓閱覽室前，興建一座玻璃帷幕式閱覽室，2013年玻璃帷幕式閱覽室增設門禁管制，師生可攜帶服務證及學生證進入<ref>[http://lib.shu.edu.tw/about_time.htm 開放時間及行事曆]</ref>。為因應大眾傳播發展的需要，於六樓設立漫畫區，提供該校師生欣賞、研究與創作使用。

1992年開發業務自動化系統，1998年第二學期啟用圖書館新自動化系統，並提供二十四小時服務。

== 教學單位 ==
世新大學設有四個學院，分別為新聞傳播學院、管理學院、人文社會學院、法學院，招收學制分別有大學日間部，以及博、碩士班學生。2000年將進修教育部改制為「終身教育學院」。2003年結合[[中國傳媒大學|中國傳媒大學]]、[[北京電影學院|北京電影學院]]等師資，成立「上海進修學院」。<ref>[http://www.shucn.net/0201.htm 上海世新進修學院簡介] {{webarchive|url=https://web.archive.org/web/20110711124554/http://www.shucn.net/0201.htm |date=2011-07-11 }}</ref>

== 翠谷風情BBS ==

== 校內社團 ==
* 學生自治會：學生自治團體，全校學生為當然會員。含畢聯會、學生自治會、學生議會、學生評議會(尚未成立)。現今學生會已因投票門檻不到而無法成立、學生議長為行管系李靖。
* 學藝性：以學術技藝研究為宗旨。如新聞人報社、校園廣播網、美術社、動漫社、話劇社、文學社、書法社、勵言社、魔術社、機研社…等。
* 服務性：以推展社會服務為目的。如崇德社、羅浮童軍社、社服團、汎愛社、慈青社…等。
* 康樂性：以提倡正當休閒、康樂、體育為目的。如競技啦啦社、[[世新合唱團|合唱團]]、空手道社、電競社、實況社、熱舞社、民俗舞蹈社、爵士鋼琴社、民謠吉他社、登山社、詞曲創作社…等。
* 聯誼性：以促進友誼、砥礪情操為目的。如高屏友會、雲嘉友會、竹苗友會、轉學生聯誼會…等。
* 系學會：由各系系上學生組成，通常舉辦各系迎新茶會、迎新宿營、啦啦隊練習、講座等活動。
* 校內媒體：學生媒體的興盛，為世新的特色。1980年代時曾有十幾個相關社團並存。現在主要有新聞人報社、校園廣播網及成報新聞社等。而世新電台與世新電視台則為獨立營運之機構，小世界則為新聞系實習報紙，編寫社區、世新活動及專題報導，非校內社團。<ref>[http://osa.shu.edu.tw/new/activity_club.htm 世新大學社團一覽（課外組網頁）] {{webarchive|url=https://archive.is/20120713112721/http://osa.shu.edu.tw/new/activity_club.htm |date=2012-07-13 }}</ref>

:大一新生必修課程，安排在星期三的下午一、二節，類似導師時間，和服務教育搭配成單雙週小班課程，內容不一定，熱心的老師和新生們有可能是經驗分享、講座、活動、參觀展覽，志工參與活動等，主要希望能更關懷學生、解決生活/課業上的難題。106學年度終止計畫。

*'''職涯守護神(針對高教深耕計畫重新設計)'''
