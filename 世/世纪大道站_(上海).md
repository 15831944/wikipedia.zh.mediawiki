'''世紀大道站'''位于[[上海|上海]][[浦东新区|浦东新区]][[世纪大道_(上海)|世纪大道]][[东方路|东方路]]，为[[上海轨道交通2号线|上海轨道交通2号线]]、[[上海轨道交通4号线|4号线]]、[[上海轨道交通6号线|6号线]]和[[上海轨道交通9号线|9号线]]的[[換乘站|四线换乘枢纽]]。

== 車站歷史 ==
世纪大道站原名东方路站，為突出本站位於上海著名街道－[[世纪大道_(上海)|世纪大道]]，于2006年7月经上海市政府批准改为现時名称。为配合四线换乘枢纽站（2、4、6、9号线）的施工，本站曾于2005年10月22日关闭，为期一年，并于2006年10月28日开放，并成为2号线、4号线的[[换乘站|换乘站]]。2007年12月29日，6号线開通試營運，世纪大道站成為[[上海轨道交通|上海轨道交通]]第三個[[換乘站|三线换乘站]]。2009年12月31日，9号线二期開通試運營，世纪大道站成為[[上海轨道交通|上海轨道交通]]首個[[換乘站|四线换乘站]]，也是中国大陆第一个四线换乘的地铁站<ref>[http://2010.qq.com/a/20091222/000002.htm 上海世纪大道站成国内首个4线换乘站]</ref>。

===地下一层===
其中，地下一层为[[上海轨道交通2号线|2号线]]、[[上海轨道交通4号线|4号线]]和[[上海轨道交通9号线|9号线]]的站厅层，东西走向的[[上海轨道交通6号线|6号线]]从地下一层站厅中部穿过，为[[侧式站台|侧式站台]]，因此将站厅分为A区，B区两部分。A、B两区在站厅层层面互不连通，乘坐反方向6号线列车的乘客需要通过2号线或4号线站台进行。虽然[[上海轨道交通9号线|9号线]]的站台2009年才启用，为了方便乘客[[换乘站|换乘]]，该线的站厅层已提前建设完毕，并和[[上海轨道交通6号线|6号线]]同步启用。

===2、4、9号线站台===
[[上海轨道交通2号线|2号线]]和的[[上海轨道交通9号线|9号线]]站台分別位于车站中部和车站西南，均为地下二层[[岛式站台|岛式站台]]，两线结构近似于[[混合式站台|双岛式站台]]，但在站台层面互不连通。[[上海轨道交通4号线|4号线]]站台位于车站东北，为地下三层[[岛式站台|岛式站台]]。

== 车站出口 ==
[[File:20170820世纪大道站2号口.jpg|thumb]]
目前世纪大道站开放4个出入口，分别为位于[[世纪大道_(上海)|世纪大道]]、[[张杨路|张杨路]]、[[东方路|东方路]]交汇的六岔路口的1号口、位于[[世纪大道_(上海)|世纪大道]]、[[福山路|福山路]]、[[潍坊路|潍坊路]]交汇的六岔路口的6号口、位于东方路、张杨路的7号口和位于世纪大道、潍坊路的12号口。

== 周邊地區 ==
[[File:Century_Link_Shanghai.jpg|thumb]]]]
世纪大道站处于[[浦东新区|浦东地区]]三条重要的道路——世纪大道、张杨路和东方路的交叉路口，这三条道路分别直通[[延安东路隧道|延安东路隧道]]、[[复兴东路隧道|复兴东路隧道]]和[[大连路隧道|大连路隧道]]。加上该站紧邻浦东的商业中心，且附近有[[第一八佰伴|第一八佰伴]]、[[华润时代广场|华润时代广场]]、[[中融国际广场|中融国际广场]]、[[新梅联合广场|新梅联合广场]]等大型购物中心、[[百思买|百思买]]、[[苏宁电器|苏宁电器]]等家电卖场及[[百脑汇|百脑汇]]、[[太平洋数码广场|太平洋数码广场]]等IT卖场。而且是[[上海轨道交通|上海轨道交通]]首個[[換乘站|四线换乘枢纽]]。因此每天客流量大，是[[浦东新区|浦东新区]]重要的轨道交通车站。
