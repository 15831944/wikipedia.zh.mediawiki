'''坪'''（{{lang-zh|píng}}、{{lang-ja|坪/tsubo}}、{{lang-ko|평/pyeong}}），源于[[日本|日本]]传统计量系统[[日本度量衡|尺貫法]]的[[面積|面積單位]]，主要用於計算房屋、建築用地之[[面積|面積]]，主要应用于日本、[[臺灣|臺灣]]和[[朝鮮半島|朝鮮半島]]。在[[明治|明治]]时期的{{link-ja|度量衡法|度量衡法}}中，1坪被定义为400/121平方米。1坪为边长为6[[日尺|日尺]]（1{{link-ja|间|間}}）的正方形的面积，约为3.305 785 124平方米。

== 起源及定义 ==
「坪」源于与其等值的面积单位“[[步_(單位)|步]]”。步是中国古代单位，[[周代|周代]]时步是表示成年男子两个步幅的距离的长度单位（先迈出右脚，再迈出左脚，则一步指起点到左脚的距离），边长为一步的正方形的面积也被称为“步”，是面积单位“步”最早的起源，当时一步的面积要小于现在的面积，后来演变为等于边长6尺的正方形的面积。

在[[江戶時代|江戶時代]]，1坪的面積等於2塊標準[[疊蓆|疊蓆（榻榻-{米}-）]]面積。而根據[[日本人|日本人]]對疊蓆的嚴格規定，2塊疊蓆合起來就是一個邊長一[[間|間]]（即6日尺）的正方形。

在[[明治时期|明治时期]]，日尺的长度为10/33米，从而造成了1坪 = 400/121平方米的现状，其计算过程如下：

1 坪 = 6 日尺 × 6 日尺 
<math>=(\frac{6\times10}{33})^2</math> [[平方米|平方米]]
<math>=\frac{400}{121}</math> 平方米≈3.305785平方米

而1平方米 = <math>\frac{121}{400}</math> 坪 = 0.3025坪

== 相关单位 ==
坪或者步是日本尺貫法中面积的基本单位，其他面积单位与坪的关系为：
*合 -- 1合 = 0.1坪
*勺 -- 1勺 = 0.1合
*畝 -- 1畝 = 30坪 = 約99.174平方米 = 約1公亩(a)
*反（段） -- 1反 = 10畝 = 約991.74平方米 = 約10公亩
*町（町歩） -- 1町 = 10反 = 約9917.4平方米 = 約1公顷(ha)

由于畝、反、町的值与1公亩、10公亩、1公顷的值非常相近，所以面积单位到[[米制|米制]]的过渡在日本比较顺利。相較之下，坪就無法順利換算成米制。

房子要裝潢、施工 會需要丈量, 然後就會出現很多的單位, 在此列出一些常見的計算方式：

計算方式

⑴坪數 = 長(公尺)×寬(公尺)×0.3025 

⑵坪數 = 長(公分)×寬(公分)÷32400

⑶1坪 = 180cm×180cm = 32400 平方公分

1坪=6[[台尺|台尺]]×6台尺 = 36才 (適用於地坪、磁磚、布料等施工單位)

== 日本之外 ==
坪原本是根据日本特有的[[疊蓆|疊蓆（榻榻-{米}-）]]来定义的单位，原來在没有这个定义基础的台湾和韩国，由於分別在1895年[[乙未割臺|乙未割臺]]及1910年[[日韓合併|日韓合併]]成為日本領土，坪也被引進並广泛使用。台湾和韩国使用的「坪」，与日本等值。

===台灣===
1945年[[日本二戰投降|日本二戰投降]]，[[國民政府接收臺灣|國民政府接收臺灣]]，曾規定以[[平方公尺|平方公尺]]做為法定計算單位，但除了部分政府文件及產權登記使用平方公尺，[[臺灣|臺灣]]民间仍普遍以「坪」計算房屋和建筑面积，建商廣告或房屋交易皆以坪作為單位至今。

===韓國===
1961年[[韩国|韩国]]计量法颁布实施，规定使用[[国际单位制|国际单位制]]（[[米制|-{zh-cn:米制; zh-tw:公制; zh-hk:公制}-]]）[[米_(单位)|-{zh-cn:米; zh-tw:公尺; zh-hk:米;}-]]，但在实际的土地、建筑买卖中仍常使用坪作为单位，直至2007年7月[[韩国政府|韩国政府]]对计量法进行修订，继续使用「坪」等單位，将被课以罚金，但民間依然故我。{{citation needed}}

== 与“平”的区别 ==
在[[中国大陆|中国大陆]]，口语中常使用“平”或“平-{米}-”作为房屋和建筑的面积单位，指的是“[[平方米|平方米]]”(m{{small|<sup>2</sup>}})。
