'''坪石邨'''（'''Ping Shek Estate'''），是[[香港|香港]]的一個[[香港公共屋邨|公共屋邨]]，位於[[九龍|九龍]][[觀塘區|觀塘區]][[平山_(香港)|平山]]，由[[香港房屋委員會|香港房屋委員會]]負責管理，[[香港屋宇建設委員會|香港屋宇建設委員會]]第九條公共屋邨，由[[巴馬丹拿|巴馬丹拿]]建築及工程師有限公司設計<ref>{{cite web|url=http://www.hkej.com/template/lj/jsp/d003.jsp|title=巴馬丹拿與公和洋行 建築與藝術|work=[[信報|信報]]財經新聞 Lifestyle Journal|author=|date=|archiveurl=https://web.archive.org/web/20140415051918/http://www.hkej.com/template/lj/jsp/d003.jsp|archivedate=2014-04-15|access-date=2018-08-21|dead-url=no}}</ref>，於1970年代落成<ref>[http://sites.google.com/site/sjacssecret/1979.jpg 1979.jpg] ，照片攝於1979年，從當時[[英國皇家空軍|英國皇家空軍]][[前皇家空軍基地|九龍灣兵房]]高空俯瞰[[四順|四順]]、彩雲邨、[[清水灣道|清水灣道]]、坪石邨、[[彩虹邨|彩虹邨]]，可見興建中彩雲邨。版權屬[[香港公共圖書館|香港公共圖書館]]所有。</ref>，截至2018年6月30日，住戶數目合共4,500戶<ref>{{cite web |author1=香港房屋委員會 |title=坪石邨, 觀塘, 九龍東 |url=https://www.housingauthority.gov.hk/tc/global-elements/estate-locator/detail.html?propertyType=1&id=2798 |work=房委會物業位置及資料}}</ref>。坪石邨所屬位置剛剛在[[黃大仙區|黃大仙區]]區界外，所以屬於[[觀塘區|觀塘區]]，本邨鄰近[[牛池灣|牛池灣]]村及[[彩虹邨|彩虹邨]]。有鑑於坪石邨的位置與[[啟德機場|啟德機場]]非常接近，所以屋邨的高度在興建時受到限制。屋邨由多條九龍區主要幹道包圍：[[龍翔道|龍翔道]]，[[觀塘道|觀塘道]]及[[清水灣道|清水灣道]]，鄰近[[彩虹交匯處|彩虹交匯處]]，因此交通十分便利。

2010年1月22日，[[香港房屋委員會|香港房屋委員會]]建築小組通過斥資1.3億港元活化坪石邨，由利安建築師有限公司負責設計，也是房委會首個對舊式屋邨引入活化概念，認為可再保存多15年。而該邨居民與商戶對此持不同意見<ref>{{cite news|title=房會1.3億元活化坪石邨|url=http://the-sun.on.cc/cnt/news/20100123/00407_036.html|accessdate=2010-11-27|newspaper=《[[太陽報_(香港)|太陽報]]》|date=2010-01-23}}</ref>。

在1960年代以前，坪石為香港[[新九龍|新九龍]]的十三個規模較大的鄉村—[[九龍十三鄉|九龍十三鄉]]之一。<ref name=orientaldaily>{{cite news|title=2011年5月19日 坦言集：忘了殖民地以前的本|url=http://orientaldaily.on.cc/cnt/news/20110519/00184_011.html|accessdate=2014-05-12|newspaper=東方日報|location=香港|language=中文|date=2011年5月19日}}</ref><ref name="come">[http://www.eastkowloon.org.hk/?action-viewnews-itemid-62 十三鄉由來]</ref>
位於[[平山_(香港)|平山]]，原是坪石村及白屋仔村寮屋區。1968年，屋宇建設委員會宣佈興建坪石邨，並於1970年至1971年建成，由港督[[戴麟趾|戴麟趾]]爵士揭幕。

「九龍皇帝」[[曾灶財|曾灶財]]曾在該邨部分燈柱和電箱留有墨迹<ref>{{cite news|title=政府拒認曾灶財墨寶|url=http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20100114&sec_id=4104&subsec=11867&art_id=13624791|accessdate=2010-11-27|newspaper=《[[蘋果日報_(香港)|蘋果日報]]》|date=2010-01-14}}</ref>，[[民政事務局|民政事務局]]轄下部門已在該墨迹套上膠片保護。

坪石邨入伙時，鑽石樓及玉石樓均沒有升降機，房屋委員會其後在2010-2012年為這些樓宇安裝升降機。

== 活化工程 ==
房委會建築小組在2010年通過斥資1.3億元進行活化工程，包括增加適合長者使用的設施、在露天廣場加建有蓋表演場地，並為住戶在室內加設扶手及改裝晾衣架等，滿足長者及殘疾住戶的需要。同時在該邨社區會堂設立回憶閣。2013年，坪石邨各大廈外牆進行翻新工程，所有工程於2013年11月完成。

== 商場 ==
坪石商場位於地面和平台。分佈在鑽石樓、玉石樓、紅石樓和酒家大樓。主要租戶包括[[百佳超級市場|百佳超級市場]]、[[759阿信屋|759阿信屋]]、[[大昌食品|大昌食品]]、[[屈臣氏|屈臣氏]]、[[萬寧|萬寧]]、[[7-11便利店|7-11便利店]]、[[上海商業銀行|上海商業銀行]]、麵包店、眼鏡店、髮廊、糧油雜貨店、辦館、美容護膚中心、文具店、診所、琴行、和牙醫醫務所。餐廳包括金華茶餐廳、彬記粥麵美食、[[麥當勞|麥當勞]]、[[大快活|大快活]]和[[爭鮮迴轉壽司|爭鮮迴轉壽司]]。

2011年6月，該邨酒家大樓有近40年歷史的第一代屋邨酒樓新坪石酒家（前稱坪石[[酒家|酒家]]）突然拉閘結業<ref>{{cite news|author=盧文烈|title=最低工資下首現酒樓結業|url=http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20110609&sec_id=4104&subsec=11867&art_id=15326661|accessdate=2011-06-13|newspaper=[[蘋果日報_(香港)|蘋果日報]]|date=2011-06-09}}</ref>。現時為富港酒家。
