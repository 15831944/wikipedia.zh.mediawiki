'''软件包格式'''是一種{{link-en|壓縮檔|Archive file}}，裡面包含了檔案以及額外的，可以在[[软件包|软件包]]中找到的元数据<ref>Justin Angelo Cappos, [http://books.google.fr/books?id=77Ejx2DmxzgC&pg=PA128&dq=%22package+formats%22&hl=en&sa=X&ei=YEs_Uof6Kqqs0QWDn4HgCQ&redir_esc=y#v=onepage&q=%22package%20formats%22&f=false Stork: Secure Package Management for VM Environments] {{webarchive|url=https://web.archive.org/web/20140219030221/http://books.google.fr/books?id=77Ejx2DmxzgC&pg=PA128&dq=%22package+formats%22&hl=en&sa=X&ei=YEs_Uof6Kqqs0QWDn4HgCQ&redir_esc=y |date=2014-02-19 }}, ProQuest, 2008, p. 128 ;</ref>。目前有許多種格式可供使用，在[[Linux發行版|Linux發行版]]及[[BSD|BSD]]中均有使用。

主要有以下幾種類型的软件包：
* [[二進制|二進制]]软件包
* [[原始碼|原始碼]]软件包

== 範例 ==
* [[deb|deb]] — '''Deb'''ian 软件包，最初是由[[Debian|Debian]]所開發。被 Debian 及其衍生發行版，如 [[Ubuntu|Ubuntu]] 所使用<ref>{{cite web|url=https://help.ubuntu.com/community/InstallingSoftware |title=InstallingSoftware - Community Ubuntu Documentation |publisher=Help.ubuntu.com |date= |accessdate=2013-04-24}}</ref>。
* {{link-en|PISI}} 是 {{link-en|Pardus|Pardus}} 的開發者所開發。被使用在 Pardus 及其衍生發行版 {{link-en|Pardus-Anka|Pardus-Anka}} 及 {{link-en|Pisi Linux|Pisi Linux}}。 
* [[Tar#縮寫|pkg.tar.xz]] － 被 [[Arch_Linux|Arch Linux]] 的 [[Pacman|Pacman]] 軟體包管理員預設使用<ref>{{cite web|url=https://www.archlinux.org/pacman/makepkg.conf.5.html|title= makepkg.conf(5) Manual Page}}</ref>。
* [[Puppy_Linux|PUP 及 PET]] － 由 [[Puppy_Linux|Puppy Linux]] 所使用 － 點擊並安裝的软件包類型。此作業系統可以安裝在隨身碟上以方便攜帶，而且應用程式也可以直接安裝到裡面<ref name="PupFAQ">{{cite web|url = http://puppylinux.com/faq.htm|title = Pup FAQ|accessdate = 2008-10-27|last = Kauler|first = Barry|authorlink = |year = 2006}}</ref>。
* [[QPKG|QPKG]] － 被 QNAP [[網路附加儲存|NAS]] 裝置所使用<ref>{{cite web|url=http://www.qnap.com/en/index.php?sn=859|title=App Center (QPKG)}}</ref>。
* [[TazPKG|TazPKG]] － [[Slitaz|Sli'''Taz''']] 上基於 [[Cpio|Cpio]] 及 [[gzip|gzip]]/[[LZMA|LZMA]] 作為[[殼層|殼層]]的软件包。
