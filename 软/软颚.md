'''软颚'''或者'''软腭'''（医学拉丁语术语：{{lang|la|Velum palatinum}}或{{lang|la|Palatum molle}}）是[[哺乳动物|哺乳动物]]中[[硬腭|硬腭]]延长形成的一个双褶皱形状的口腔部分。它斜挂在或者垂直挂在[[舌|舌]]根上方，是由[[咽腭肌|咽腭肌]]和其它[[肌肉|肌肉]]的延伸组成的。它是分隔[[口腔|口腔]]与[[咽|咽]]的组织的一部分，因此有分隔[[呼吸系统|呼吸系统]]与[[消化系统|消化系统]]的作用。

== 解剖学 ==
软颚由一层没有角质的[[上皮组织|上皮]]覆盖。许多哺乳动物的软腭的[[粘膜|粘膜]]里有淋巴组织（{{lang|la|Tonsilla veli palatini}}）人和[[食肉目|食肉目]]动物没有这样的[[淋巴结|淋巴结]]。

咽缩肌（{{lang|la|Musculus constrictor pharyngis}}）、[[腭帆张肌|腭帆张肌]]（{{lang|la|Musculus tensor veli palatini}}）和[[腭帆提肌|腭帆提肌]]（{{lang|la|Musculus levator veli palatini}}）可以运动软颚。从软颚的边缘伸展出两道双皱褶组织，即[[腭弓|腭弓]]（{{lang|la|Arcus palatini}}）。[[猴|猴]]和[[人|人]]的软颚中部有一个被称为[[腭垂|腭垂]]的下垂，大多数其它哺乳动物没有这个组织。

软颚的神经是从[[舌咽神经|舌咽神经]]和[[迷走神经|迷走神经]]延伸出来的。

软颚的基线被称为“啊线”，因为它在说“啊”的时候可见。

在软颚末端悬挂的[[小舌|小舌]]在部分语言中用于发出[[小舌音|小舌音]]。

== 功能 ==
软颚有分隔呼吸道和消化道的作用。在[[下咽|下咽]]的时候在咽缩肌的作用下软颚后壁会亚在[[气管|气管]]上。在下咽或者[[打哈欠|打哈欠]]的时候从[[耳咽管|耳咽管]]伸出来的腭帆张肌和腭帆提肌有平衡[[中耳|中耳]]和外部的压力差的作用。

在[[发音|发音]]的时候软颚抬起，把鼻腔与口咽腔分隔开来，这样一来从肺里传出来的气流只能流进口腔来产生口音。在发[[鼻音_(辅音)|鼻音]]的时候[[口腔|口腔]]被封住，气流流过鼻腔。在发[[鼻化元音|鼻化元音]]的时候软颚下沉，让气流同时通过口腔和鼻腔。软颚在[[打呼噜|打呼噜]]的时候也有作用。

触碰小舌或者软腭的深处，对大多数人来说都会引起强烈的[[呕吐反射|呕吐反射]]，属于一种自然的保护机制。

== 医学意义 ==
在校正[[唇顎裂|唇顎裂]]的[[整容|整容手术]]以及[[肿瘤|肿瘤]]手术中软颚有很大的意义。
