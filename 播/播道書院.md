'''播道書院'''（{{lang-en|Evangel College}}）位於[[香港|香港]][[新界|新界]][[將軍澳|將軍澳]][[至善街|至善街]]，是一間[[直資|直資]]中學、小學一條龍學校，由[[中國基督教播道會總會]於2005年創辦。中學和小學部各有30班，學生總人數約1600人，校長是盧偉成。校訓為「播揚真理 道育幼苗」（Proclaim the Truth, Nurture our Youth）。

==開設科目==
===初中===
中國語文、英國語文、數學、綜合人文（包括經濟、地理、歷史，中三分為三科）、中國歷史、綜合科學（包括生物、化學、物理，中三分為三科）、體育、宗教教育、音樂、視覺藝術
===高中===
中國語文、英國語文、數學、通識教育、中國文學、英語文學、中國歷史、經濟、地理、歷史、生物、化學、物理、綜合科學、企業、會計與財務概論、資訊及通訊科技、視覺藝術、數學延伸部分一（微積分與統計）、數學延伸部分二（代數與微積分）、體育、宗教教育

==環保政策==
學校建築本身為環保設計。學校亦有設立環保學會（Environmental Protection Team），定期舉辦活動及宣傳環保意識，例如舉行回收月餅罐活動、參觀海下灣及米埔等等。

==學校設施==
播道書院校舍是從[[教育署|教育署]]首次為香港學校舉行命為「邁向多元化校園」建築設計比賽參賽作品中選出。奪冠作品是由三位畢業於[[港大|港大]]的[[香港建築師學會|香港建築師學會]]會員、[[凱達環球|凱達環球有限公司]]建築師：柳景康、朱海山和盧仕佳設計獨特創新的「Z」形校舍，既配合活動教學概念，又善用空間，符合環保理念，為學生提供一個擴一闊及優美的學習環境。<br>
學校整個建築是南北向，讓空氣對流，亦不會有西斜的情況出現。每一班房設計了一些太陽板，避免室內過熱及多使用天然光。在整個「Z」字形大樓中，右下角設有一個露天舞台，在多個樓層皆可清楚看見。小學禮堂設在建築物下層，同時高層是特別課室，如實驗室或視覺藝術室等，課室設在樓上，目的為善用陽光及空氣對流。大禮堂設在校園另一角，而中小學也分別有不同的操場。學校設計以[[三維空間|三維空間]]的組合，部分一班房的單元可變成兩個班房的單元，以活動及隔音的間板分隔，兩班之間有一些空間讓學生舉行小組活動，各課室褶門可以讓學生繪畫和展覽作品。<br>此外，播道書院道真堂於2006年成立並進駐學校，成為播道書院的福音夥伴，服務學校及將軍澳區社群。

==近年事件==
===教師離職潮===
2017/18學年首屆原校升讀生畢業後，隨即大批教師離職，包括數位中學教師及二十多位小學教師。

===蘇格蘭遊學團內定參與名單===
於2017年首度舉辦的蘇格蘭遊學團被指由校方內定名單，並未經過公開招募及遴選程序，其後證實因為首年舉辦，希望試驗成效，因此未有公開程序，而其後於2018年舉辦的同一遊學團則為公開遴選，未有任何爭議。

===超強颱風山竹吹襲損毀誤會===
有居住播道書院校舍附近的居民上載圖片，指校舍一樓平台被超強颱風山竹吹歪<ref>{{Cite web |url = http://hd.stheadline.com/news/realtime/hk/1318947/ |title = 【山竹襲港】將軍澳街道被淹浸成重災區 海濱長廊變「長江」 }}</ref>，惟後證實為誤會，該處平台本已為不規則形狀，僅為視覺錯覺<ref>{{Cite web |url = http://www.881903.com/page/zh-tw/newsdetail.aspx?ItemId=1030572&csid=261_341 |title = 播道書院澄清平台沒有塌下網傳照片屬誤會 }}</ref>。為建築師的播道書院校監何永業先生更查看校舍建築圖則，確認為因高空俯視引起的錯覺。
另外，校舍中學部一樓F教員室、二至三樓共三間多用途室、二樓G教員室、四至五樓共四間實驗室及六樓視覺藝術室均有窗戶被指受超強颱風山竹吹至碎裂<ref>{{Cite web |url = https://www.hk01.com/%E8%A6%AA%E5%AD%90/236183/%E9%A2%B1%E9%A2%A8%E5%B1%B1%E7%AB%B9-%E9%9B%A3%E6%8A%B5%E5%8D%81%E8%99%9F%E9%A2%A8%E7%90%83-%E5%85%A8%E6%B8%AF%E5%A4%9A%E9%96%93%E4%B8%AD%E5%B0%8F%E5%AD%B8%E6%A0%A1%E5%9C%92%E8%A2%AB%E6%91%A7%E6%AF%80 |title = 【颱風山竹】難抵十號風球　全港多間中小學校園被摧毀 }}</ref>，惟翌日（九月十七日）證實亦為窗簾未有完全關上而引致的視覺錯覺。

'''學生會成立'''

播道書院歷屆均有學生要求成立學生會，但一直未被校方重視。至近年校方一度表示希望成立校內具代表性的學生組織，學生的訴求才開始被重視，而校方亦有與部分自發關注學生會事務的學生代表交流意見。有乃各屆學生多年來的努力，第一屆學生會終於2018年成立。
