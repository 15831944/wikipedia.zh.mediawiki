'''慈禧行宫'''或称'''邯郸行宫'''，位于河北省[[邯郸市|邯郸市]][[邯郸道|城内中街]]原[[邯郸专署|邯郸专署]]院内北侧，建于[[清朝|清]]光绪二十七年（[[1901年|1901年]]），是慈禧太后回銮路北京过[[明清邯郸城|邯郸]]的休憩之处{{参|{{cite book |author1=董海林 |coauthors=李广 李亚 |title=《邯郸名胜古迹》 |date=2011-09 |publisher=河北人民出版社 |location=石家庄 |isbn=9787202059142 |page=197页 |accessdate=2018-06-21}}}}。

== 历史 ==
[[清朝|清]][[光绪|光绪]]二十六年（1900年），[[八国联军|八国联军]]攻占北京后，[[慈禧太后|慈禧太后]]带着[[光绪帝|光绪帝]]逃到[[西安|西安]]，次年订《[[辛丑条约|辛丑条约]]》后回銮。1901年11月慈禧太后驻跸邯郸，此为沿途路过地区所建行宫之一{{参|《邯郸县志》}}。1995年10月，行宫被邯郸市人民政府公布为第一批[[邯郸市文物保护单位|市级文物保护单位]]{{参|{{cite web |title=邯郸市第一批市级文物保护单位名单 |url=http://www.hd.gov.cn/zjhd/wwgj/201712/t20171220_733152.html |website=邯郸市人民政府 |accessdate=2018-06-07}}}}。

== 建筑风格 ==
[[File:The_Cixi_palace_02.jpg|180px]]
建筑风格是典型的中国北方复式[[四合院|四合院]]，原有南北两院，前院接待朝拜、奏请、议事；后院房前植有腊梅，环境幽静。现存北院一大殿及两配殿，大殿坐北朝南，悬山顶，面宽13.10米，进深9.05米，高8.7米，为七架梁前后廊式的砖木结构布瓦顶建筑。东西配殿均面宽三间进深二间，即8.55米×5.70米，为硬山顶建筑。其它均被改建为民房，待全面修复{{参|{{cite news|author1=刘秀君|title=隐现在历史长河中的珍珠|url=http://www.handannews.com.cn/epaper/handrb/html/2013-05/26/content_110118.htm?div=0|accessdate=2017-09-17|agency=邯郸日报社|date=2013年5月26日|archive-url=https://web.archive.org/web/20170917213826/http://www.handannews.com.cn/epaper/handrb/html/2013-05/26/content_110118.htm?div=0|archive-date=2017-09-17|dead-url=yes}}}}{{参|{{cite news |author1=刘秀君 |title=隐现在历史长河中的珍珠 |url=http://www.handannews.com.cn/epaper/handrb/html/2013-05/26/content_110118.htm?div=0 |accessdate=2018-06-07 |work=邯郸日报 |agency=邯郸日报社 |archive-url=https://web.archive.org/web/20180612142408/http://www.handannews.com.cn/epaper/handrb/html/2013-05/26/content_110118.htm?div=0 |archive-date=2018-06-12 |dead-url=yes }}}}。

== 争议 ==
另外根据民间说法，当年[[邯郸县|邯郸县]]地方官实际上还在城北20里的[[黄粱梦|黄粱梦]]镇的吕仙祠为慈禧太后准备了另一处行宫，是由[[吕仙祠|吕仙祠]]内的西王母殿仓促改建的。严格来说这只能算是一座尖站，是慈禧太后离开邯郸城后途中吃早饭时的休息场所{{参|{{cite web |author1=杨红林 |title=慈禧回銮：1901年的一次特殊旅行 |url=http://news.ifeng.com/a/20171205/53878347_0.shtml |website=凤凰网 |accessdate=2018-06-07}}}}。
