'''慈幼中學（{{lang|pt|Instituto Salesiano}}）'''為[[天主教|天主教]][[慈幼會|慈幼會]]在[[澳門|澳門]]所開辦一所[[學校|學校]]，亦是該會在[[遠東|遠東]]地區的第一所學校，由[[聖雷鳴道|聖雷鳴道]]（St.Luigi Versiglia）於1906年所創辦，建校距今已經有112年歷史，是澳門現存的天主教學校中歷史最悠久的英文學校<ref name="劉羨冰2007">{{cite book |author1=劉羨冰 |title=《澳門教育史》 |date=2007 |publisher=澳門出版協會 |isbn=9789993793908 |page=118}}</ref>，現時學校部分範圍有計劃準備重建<ref>{{cite web |title =(2015) 慈幼百年校舍或將拆卸-正報|url=https://www.cyberctm.com/zh_TW/news/detail/1092643}}</ref><ref>{{cite web |title =校友心聲-正報|url=https://www.cyberctm.com/zh_TW/news/detail/1293679#.W8NB0BMzZ24}}</ref>。學校開辦初期為[[孤兒院|孤兒院]]，其後搬遷並命名為'''無原罪工藝學校'''，於1952年再改名為現時的'''慈幼中學'''。2006年12月8日，慈幼中學慶祝了一百週年校慶<ref>{{cite web |title =慈幼母校百週年-鮑思高家庭通訊|url=http://www.sdb.org.hk/sbchinese/191/19113.htm}}</ref><ref>{{cite web |title =圖說香港歷史建築 p.102|url=https://books.google.com/books?id=sYZYCgAAQBAJ&pg=PA102&lpg=PA102&dq=%E7%84%A1%E5%8E%9F%E7%BD%AA%E5%B7%A5%E8%97%9D%E5%AD%B8%E6%A0%A1&source=bl&ots=oUsEWaMjIn&sig=_B2jPG9Dt2P9ewlVTnd2LotAbhU&hl=zh-TW&sa=X&ved=2ahUKEwj7vK_5_4XeAhWNdHAKHayKAvMQ6AEwDXoECAkQAQ#v=onepage&q=%E7%84%A1%E5%8E%9F%E7%BD%AA%E5%B7%A5%E8%97%9D%E5%AD%B8%E6%A0%A1&f=false}}</ref>。

除了歷史悠久，慈幼中學亦是近代工讀教育在澳門的發端，也反映了工業教育在澳門的發展<ref name="劉羨冰2007"/>。

==起源==
慈幼中學現址位處[[聖老楞佐堂_(澳門)|聖老楞佐教堂]]對開的一幅土地，十九世紀時屬[[英國東印度公司|英國東印度公司]]所有。因該建築門口有八對十六條枝柱，故有「十六柱」之稱。後來該建築及其所位處之地段被[[澳葡|澳葡]]政府收回，[[澳門總督|澳門總督]]轉而批示將之撥予[[天主教澳門教區|天主教澳門教區]]使用<ref>{{cite web |title =舊城遺事 五個已經消失的澳門地名-匯澳傳媒 |url=http://recap853.com.mo/2017/05/%E8%88%8A%E5%9F%8E%E9%81%BA%E4%BA%8B-%E4%BA%94%E5%80%8B%E5%B7%B2%E7%B6%93%E6%B6%88%E5%A4%B1%E7%9A%84%E6%BE%B3%E9%96%80%E5%9C%B0%E5%90%8D/}}</ref>。

1906年，總部設在[[意大利|意大利]]的[[鮑思高慈幼會|鮑思高慈幼會]]應當時[[天主教澳門教區|天主教澳門教區]][[鮑理諾|鮑理諾]][[主教|主教]]之邀，派遣[[聖雷鳴道|雷鳴道神父]]到[[澳門|澳門]]傳教辦學。雷鳴道於同年乘船從意大利出到澳門，開始慈幼會在[[中國|中國]]的事業。雷鳴道最初在[[聖若瑟修院|聖若瑟修院]]對面開辦[[孤兒院|孤兒院]]，數年後獲[[天主教澳門教區|天主教澳門教區]]贈予位於[[聖老楞佐堂_(澳門)|聖老楞佐教堂]]對開的一幅土地連建築，於是便開辦慈幼會在遠東首所學校——無原罪工藝學院<ref>{{cite web |title =聖雷鳴道主教及聖高惠黎司鐸 感恩祭禮儀經文 - 鮑思高慈幼會 |url=http://www.sdb.org.hk/?wpdmpro=%E8%81%96%E9%9B%B7%E9%B3%B4%E9%81%93%E4%B8%BB%E6%95%99-%E5%8F%8A-%E8%81%96%E9%AB%98%E6%83%A0%E9%BB%8E%E5%8F%B8%E9%90%B8-%E6%84%9F%E6%81%A9%E7%A5%AD%E7%A6%AE%E5%84%80%E7%B6%93%E6%96%87}}</ref><ref>{{cite web |title =校史-澳門慈幼中學|url=http://ism.edu.mo/%E5%AD%B8%E6%A0%A1%E7%B0%A1%E4%BB%8B/%E6%A0%A1%E5%8F%B2/7}}</ref>。1910年，該校開辦[[木工|木工]]、[[排版|排版]]等工藝課程，使當時的無原罪工藝學校成為一所[[職業學校|職業學校]]，為當年的學生習得一手技藝謀生。1932年擴充校舍，並把校務擴充成完整的學校系統。於1937年獲當時之教育廳批准立案，並且易名為「鮑思高紀念中學」，惟於1952年與[[葡光學校|葡光學校]]的[[葡文|葡文]]校名相同，所以再易名成「慈幼中學」，並沿用至今。1964年，校方為學生增辦了電子和機械兩個工業科目，並以[[英文|英文]]作為[[教學語言|教學語言]]。1972年[[陳日君|陳日君]]樞機在任校長期間，將六個工藝科目全數停辦，令慈幼中學成為一所依照香港編制課程的英文[[工業學校|工業學校]]<ref>{{cite web |title =校史-澳門慈幼中學|url=http://ism.edu.mo/%E5%AD%B8%E6%A0%A1%E7%B0%A1%E4%BB%8B/%E6%A0%A1%E5%8F%B2/7}}</ref>。

==簡介 ==
慈幼中學為澳門一所完全的[[男校|男校]]。學制為六年[[小學|小學]]，三年[[初中|初中]]和三年[[高中|高中]]。高中設純理組、理工組及文商組。學校於1995年9月加入政府[[免費教育|免費教育]]網，提供小學、初中及高中免費教育<ref name="ism_history">{{cite web|title=Institute Salesiano|url=http://ism.edu.mo/學校簡介/校史/}}</ref>。2002年，澳門政府正式批准慈幼中學在保留原有之工科課程下，成為了一所文法中學（[[三三四學制|三三四學制]]）。中學部依據[[英國|英國]][[普通教育文憑|普通教育文憑]]（General Certificate of Education）內容編排課程，並以英文為教學語言為學生授課<ref>{{cite web |title =：簡介-澳門慈幼中學 |url = http://ism.edu.mo/%E5%AD%B8%E6%A0%A1%E9%A1%98%E6%99%AF/ }}</ref>。

[[File:Instituto_Salesiano.JPG|thumb]]
學校設有機械工場,2電子工場及劃則室。隔年會舉行一連數日的工科聯展，供同學及公眾評鑑參觀，而近年慈幼中學亦有參加多個校外科技展覽及比賽。<ref>{{cite web |title =：工科男的浪漫 百年慈幼工房圖輯-匯澳傳媒|url = http://recap853.com.mo/2017/07/%E5%B7%A5%E7%A7%91%E7%94%B7%E7%9A%84%E6%B5%AA%E6%BC%AB-%E7%99%BE%E5%B9%B4%E6%85%88%E5%B9%BC%E5%B7%A5%E6%88%BF%E5%9C%96%E8%BC%AF/ }}</ref><ref>{{cite web |title =：慈幼科普展21日揭幕-盈澳網|url = http://www.in853.com/news/bencandy.php?fid=55&id=157013 }}</ref><ref>{{cite web |title =高中部理工組課程表-澳門慈幼中學|url = http://ism.edu.mo/ssec_st/ }}</ref>

2012-2013學年，原有的初中工業科目（[[機械|機械]]、[[電子|電子]]及[[劃則|劃則]]科）整合為「設計與應用科技科」（Design & Applied Technology），並因應[[计算机辅助设计|電腦科技輔助設計(CAD)]]技術的普及與發展，課程教學上進一步提升電腦輔助設計的應用性<ref>{{cite web |title =慈幼中學招生簡介-澳門慈幼中學|url = http://ism.edu.mo/wp-content/uploads/2018/02/%E6%8B%9B%E7%94%9F201819.pdf}}</ref><ref>{{cite web |title =全國機器人奧賽慈幼獲佳績-澳門日報|url =http://www.macaodaily.com/html/2017-09/01/content_1204544.htm}}</ref>。

在學制上，學生在高中時會分組，視學生志願及成績，決定進入理組（生物組）、理化及科技組（理工組）或文商組<ref>{{cite web |title =慈幼中學招生簡介-澳門慈幼中學|url = http://ism.edu.mo/wp-content/uploads/2018/02/%E6%8B%9B%E7%94%9F201819.pdf}}</ref>。

== 校徽  ==
[[File:IS_School_Macau.png|thumb]]
慈幼中學校徽主體為一藍色[[盾牌|盾牌]]，盾牌上有紅色[[十字架|十字架]]，代表跟隨[[耶穌基督|耶穌基督]]的教導、以[[基督|基督]]作學校的助佑。紅色十字架上有校名的簡寫「I S」。底部一朵白色[[百合花|百合花]]象徵學生應有的純潔精神。盾牌和白色百合花代表著「純潔與剛毅」（PURI ET FORTES）的校訓，而盾牌下的紅絲帶則有[[拉丁文|拉丁文]]的校訓（PURI ET FORTES）。絲帶兩端為「慈幼」及「中學」<ref>{{cite web |title =：校徽校歌-澳門慈幼中學 |url = http://ism.edu.mo/%E5%AD%B8%E6%A0%A1%E7%B0%A1%E4%BB%8B/%E6%A0%A1%E5%BE%BD%E6%A0%A1%E6%AD%8C/}}</ref>。
