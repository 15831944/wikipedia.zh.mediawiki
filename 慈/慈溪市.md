'''慈溪市'''位于[[中国|中国]][[浙江省|浙江省]]东北部，濒临[[杭州湾|杭州湾]]，为[[宁波市|宁波市]]代管的一个经济发达的[[县级市|县级市]]，宁波第二大城市。慈溪市位于[[宁绍平原|宁绍平原]]东北部，面积1,361平方公里，户籍人口104.19万。

== 沿革 ==
===句章时期（唐开元年以前）===

[[春秋|春秋]]时期属[[越国|越国]]地。[[战国|战国]]时期设句章城。<ref name='历史沿革'>{{Cite web|url =http://cx.ningbo.gov.cn/art/2008/1/16/art_14377_170465.html|title =慈溪市历史沿革|author =慈溪政府网|date =|publisher =|language =中文|accessdate =2014-01-24|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20140201181319/http://cx.ningbo.gov.cn/art/2008/1/16/art_14377_170465.html|archivedate =2014-02-01}}</ref>[[秦朝|秦朝]]（[[秦始皇|秦始皇]]二十六年），设[[句章县|句章县]]，隶属于[[会稽郡|会稽郡]]。<ref name='历史沿革' />[[东汉|东汉]]为会稽东部都尉治隶属于[[扬州_(古代)|扬州]][[会稽郡|会稽郡]]。<ref name='历史沿革' />[[东晋|东晋]]时期因战事而废城，后迁至鄞县一带。隶属于扬州会稽郡。<ref name='《慈溪县志》'>{{cite book |author=慈溪市地方志编纂委员会 |authorlink= |coauthors= |title=《慈溪县志》|year=1992 |publisher= 浙江人民出版社|location= |isbn=}}</ref>[[隋朝|隋朝]]，隶属于会稽郡，后属[[吴州|吴州]]，后属[[越州|越州]]。<ref name='历史沿革' />[[唐朝|唐朝]][[武德|武德]]四年（621年）废句章，隶属于越州。<ref name='《慈溪县志》' />
===唐开元年至清===

唐开元二十六年，设[[慈溪县|慈溪县]]，后曾隶属于[[江南东道|江南东道]]越州、明州、余姚郡。<ref name='《慈溪县志》' />唐乾元元年，隶属于浙江东道。<ref name='历史沿革' />[[五代|五代]]时期，属明州望海军，隶[[吴越国|吴越国]]。<ref name='历史沿革' />[[北宋|北宋]]，属两浙路明州奉国军。<ref name='历史沿革' />[[南宋|南宋]]，两浙东路庆元府。<ref name='历史沿革' />[[元朝|元朝]]，属江浙行省浙江东道宣慰使司庆元路。<ref name='历史沿革' />[[明朝|明朝]]，属宁波府<ref name='历史沿革' />，明永乐十六年，慈溪失县印，上报朝廷后，改溪从谷，即改慈溪县为[[慈谿县|慈谿县]]。[[清朝|清朝]]，慈谿县属宁波府，隶[[浙江省|浙江省]]绍台道。<ref name='《慈溪县志》' />

===中华民国大陆时期===

*[[民国元年|民国元年]]至[[民国二年|民国二年]]，直属于浙江省军政府。
*[[民国三年|民国三年]]至[[民国十五年|民国十五年]]，隶属于浙江省会稽道。
*[[民国十六年|民国十六年]]至[[民国二十年|民国二十年]]，直属浙江省政府。
*[[民国二十一年|民国二十一年]]至[[民国二十三年|民国二十三年]]，属浙江省第五[[行政督察区|行政督察区]]。
*[[民国二十四年|民国二十四年]]至[[民国三十五年|民国三十五年]]，属浙江省第六行政督察区。
*[[民国三十六年|民国三十六年]]至[[民国三十七年|民国三十七年]]，属浙江省第三行政督察区。
*民国三十七年四月至[[民国三十八年|民国三十八年]]六月，属浙江省第二行政督察区<ref name='《慈溪县志》' />。

===中华人民共和国时期===

建国初期（1949-1954），慈谿县属宁波专员公署（[[宁波专区|宁波专区]]）。1954年[[镇海县|镇海县]]、[[余姚县|余姚县]]、慈谿县之北成为新的慈谿县。[[县治|县治]]从[[慈城镇|慈城镇]]移至浒山镇（[[浒山街道|浒山街道]]）。1956年因[[汉字简化|汉字简化]]，复-{慈谿}-为-{慈溪}-。1970年，慈溪县隶属于宁波地区。1979年，泗门区划归余姚，余姚龙南区划入慈溪县，形成现有境域。1988年10月13日，慈溪撤县改市（县级市），属[[宁波市|宁波市]]<ref name='《慈溪县志》' />。

== 行政区划 ==

下辖5个街道、14个镇，59个社区、27个居委会、296个行政村：

* 街道办事处：[[浒山街道|浒山街道]]、[[古塘街道|古塘街道]]、[[白沙路街道|白沙路街道]]、[[宗汉街道|宗汉街道]]、[[坎墩街道|坎墩街道]]。
* 镇：[[观海卫镇|观海卫镇]]、[[长河镇|长河镇]]、[[周巷镇|周巷镇]]、[[新浦镇|新浦镇]]、[[崇寿镇|崇寿镇]]、[[胜山镇|胜山镇]]、[[庵东镇|庵东镇]]、[[附海镇|附海镇]]、[[逍林镇|逍林镇]]、[[桥头镇|桥头镇]]、[[横河镇|横河镇]]、[[匡堰镇|匡堰镇]]、[[掌起镇|掌起镇]]、[[龙山镇|龙山镇]]。

慈溪所在之地，原称三北，乃慈北、姚北、镇北之意。1954年由原余姚县、慈溪县、镇海县3县之北部产棉区合并新设慈溪县，县城新设在浒山。而此前的原慈溪县县城则驻在[[慈城镇|慈城镇]]，该镇目前属于[[江北区_(宁波)|江北区]]。

== 物产 ==
金属资源有[[铁|铁]]、[[铜|铜]]。非金属资源有[[石墨|石墨]]、[[石英|石英]]、[[天然气|天然气]]、[[泥炭|泥炭]]等。土特产品有[[杨梅|杨梅]]、[[山麥冬屬|麦冬]]、[[丝瓜络|丝瓜络]]、[[古窑浦水蜜桃|古窑浦水蜜桃]]、[[鸣鹤雷竹笋|鸣鹤雷竹笋]]、[[周巷酱油|周巷酱油]]、[[天元牌黄酒|天元牌黄酒]]、[[龙山黄泥螺|龙山黄泥螺]]、[[三北豆酥糖|三北豆酥糖]]、[[蟹|沙蟹]]、[[天元古旧家具|天元古旧家具]]等。

== 经济 ==

慈溪市现为[[中国百强县|中国大陆百强县]]前10之列，民营经济占绝对主导成分，超95%民营比例，最著名的产业是电器，工业发展迅速，百花齐放。中小企业遍地，一镇一品，一户一厂是慈溪一大特色。2012年[[国内生产总值|GDP总量]]948.29亿元<ref name="GDP">[http://www.cixi.gov.cn/art/2013/1/28/art_14354_975632.html 慈溪GDP解读]</ref>。
