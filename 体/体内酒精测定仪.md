'''体内酒精测定仪'''（{{lang-en|breathalyzer}}）或'''呼气式酒精检测仪'''（breathalyser，breath和analyzer/analyser的[[混成詞|混成詞]]）是一种根据呼气样本测定[[血液酒精濃度|血液酒精濃度]]的设备。'''Breathalyzer'''是一个[[品牌|品牌名称]]（一个[[通用商標|通用商標]]），属于发明家{{le|Robert Frank Borkenstein}}开发的酒精浓度测试仪。其在1954年5月13日被注册为商标，但许多人使用该术语指代测量[[血液酒精濃度|血液酒精濃度]]的任何通用设备。

== 起源 ==

== 化学原理 ==
当用户的呼气进入呼气分析仪时，其呼吸中的任何[[乙醇|乙醇]]被[[氧化还原反应|氧化]]为[[乙酸|乙酸]]在 [[陽極|阳极]]：

CH<sub>3</sub>CH<sub>2</sub>OH(g) + H<sub>2</sub>O(l) → CH<sub>3</sub>CO<sub>2</sub>H(l) + 4H<sup>+</sup>(aq) + 4e<sup>−</sup>

在[[陰極|阴极]]，大气氧气[[氧化还原反应|减少]]：

O<sub>2</sub>(g) + 4H<sup>+</sup>(aq) + 4e<sup>−</sup> → 2H<sub>2</sub>O(l)

总体反应是将乙醇氧化为乙酸和水。

CH<sub>3</sub>CH<sub>2</sub>哦(l)+O<sub>2</sub>(g)→CH<sub>3</sub>二甘醇(水)+H<sub>2</sub>O(l)

由该反应产生的[[电流|电流]]被微控制器测量，并被检测仪显示为近似的[[血液酒精濃度|血液酒精濃度]]。

== 法律执行 ==

== 公共与私人使用 ==
美国执法部门使用的所有呼气式酒精测试仪必须经交通运输部的[[美国国家公路交通安全管理局|国家公路交通安全管理局]]批准。

公共的呼气式检测仪正成为一些消费者测试体内酒精含量的方法。这可用于酒吧、餐馆、慈善机构、婚礼等所有活动。

== 呼气测试证据在美国 ==
[[File:Evidential.JPG|右]]
呼气酒精读数可以两种方式用于刑事诉讼。如果车辆驾驶员的测试读数超过驾驶员法定上限则将被指控{{tsl|en|Illegal per se|本身违法}}：也就是说，在美国当酒精浓度（BrAC）在0.08%以上时驾驶车辆即是非法。一个例外是[[威斯康星州|威斯康星州]]，第一次醉酒驾驶违法通常是违反民事条例。

== 传感器 ==
; 光伏测定
: 光伏测定仅用于过时的Photo Electric Intoximeter（PEI），这是一种现今很少见的呼气测试形式。该流程使用[[光敏电阻|光敏电阻]]来分析[[氧化还原反应|氧化还原反应]]的颜色变化。呼吸样品将冒泡通过一个[[硫酸|硫酸]]、[[重铬酸钾|重铬酸钾]]和[[硝酸银|硝酸银]]的[[水溶液|水溶液]]。[[硝酸银|硝酸银]]起到[[催化|催化]]，使[[醇|醇]]以可观的速率被氧化。反应所需的酸性条件也可以由[[硫酸|硫酸]]提供。{{TransH}}In [[溶液|溶液]], [[乙醇|乙醇]] reacts with the [[重铬酸钾|重铬酸钾]], reducing the [[铬酸盐|铬酸盐]] [[离子|离子]] to the [[铬|铬]] (III) [[离子|离子]]. This reduction results in a change of the [[溶液|溶液]]'s color from red-orange to green. The reacted [[溶液|溶液]] is compared to a vial of non-reacted [[溶液|溶液]] by a [[光感測器|光感測器]], which creates an [[电流|电流]] proportional to the degree of the color change; this current moves the needle that indicates BAC. Like other methods, breath testing devices using [[分析化学|化学分析]] are somewhat prone to false readings. [[化合物|化合物]] that have compositions similar to [[乙醇|乙醇]], for example, could also act as reducing agents, creating the necessary color change to indicate increased BAC.{{TransF}}

== 呼气式分析仪的误读 ==
有许多种物质或技术可以“糊弄”呼吸式分析仪（即产生较低的[[血液酒精濃度|血液酒精濃度]]）。
