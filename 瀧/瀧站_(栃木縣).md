'''瀧站'''（{{jpn|j=滝駅|hg=たきえき|rm=Taki eki|f=y}}）是位於[[日本|日本]][[栃木縣|栃木縣]][[那須烏山市|那須烏山市]]瀧的[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[烏山線|烏山線]]鐵道車站。

車站名來自附近的[[江川_(栃木縣)|江川]][[龍門之瀧_(栃木縣)|龍門之瀧]]。

== 車站構造 ==
[[單式月台|單式月台]]1面1線的無人地上車站。無站舎。

== 車站周邊 ==
* 龍門之瀧（徒步5分）
* 龍門故鄉民藝館（龍門之瀧入口）
* 太平寺（龍門之瀧入口。[[川口松太郎|川口松太郎]]小説中有名的蛇姫之墓）
* [[栃木縣道10號宇都宮那須烏山線|栃木縣道10號宇都宮那須烏山線]]離本站約1.3km

== 历史 ==
* 1954年（[[昭和|昭和]]29年）6月1日：啟用。
* 1987年（昭和62年）4月1日：[[國鐵分割民營化|國鐵分割民營化]]，成為JR東日本車站。
