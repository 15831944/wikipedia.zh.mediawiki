'''瀧溪車站'''位於[[台灣|台灣]][[臺東縣|臺東縣]][[太麻里鄉|太麻里鄉]]多良村，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[南迴線|南迴線]]的[[鐵路車站|鐵路車站]]。

== 站名由來 ==
本地昔日為[[排灣族|排灣族]]「大得吉社」所在地，北邊另有察-{腊}-密社、南邊有大竹高社。1937年臺東廳地方制度改正，大得吉更名「大溪」、察-{腊}-密更名「瀧」、大竹高更名「大竹」。南迴鐵路設站時初訂名大竹，後一度訂名所在地大溪，但因宜蘭線已有同名車站，故將瀧、大溪兩地名合為「瀧溪」作為站名。

== 車站構造 ==
*[[島式月台|島式月台]]一座。

== 利用狀況 ==
*目前為乙種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]，一人站員服務。
*本站是[[台灣|台灣]]鐵路少數幾個有[[對號列車|對號列車]]停靠的[[簡易站|簡易站]]。
*2018/10/12 年度改點之後 實施週日班表
*停靠狀態：
**[[自強號列車|自強號]]：停靠(順行)304次、326次（逢週五、日行駛），(逆行)313次。
**[[莒光號列車|莒光號]]：停靠(順行)（704、706次逢週日停駛）、（6702、6760次逢週日行駛），(逆行)701、751、705次（逢週五、日行駛）。
**[[臺鐵區間車|區間車]]：皆停靠（[[復興號列車|復興號]]車廂編組）
**[[普快車|普快車]]：皆停靠。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*[[南迴線|南迴線]]設線時此站被命名為「大竹」。
*1992年10月5日：[[南迴線|南迴線]]正式營業時設「大溪」站（本站行政區域屬於[[臺東縣|臺東縣]][[太麻里鄉|太麻里鄉]]多良村），因與[[宜蘭線|宜蘭線]][[大溪車站|大溪站]]同名，後來更名為「瀧溪」站。
*2002年6月1日：由[[三等站|三等站]]降為[[甲種簡易站|甲種簡易站]]，並指定由[[大武車站|大武站]]管理。
*2016年6月28日：啟用多卡通刷卡機。
*2017年9月16日：由[[甲種簡易站|甲種簡易站]]調整為乙種[[簡易站|簡易站]]
