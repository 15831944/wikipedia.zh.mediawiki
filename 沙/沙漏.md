'''沙漏'''是一種[[計時儀器|計時儀器]]，又有'''沙鐘'''、'''沙壺'''等別稱。沙漏通常由兩個連在一起的[[流沙池|流沙池]]組合而成，而內裏則視乎計算的時間密封裝有一定的[[流沙|流沙]]。

== 流沙池 ==
形狀各有不同，最傳統及普遍的是[[漏斗|漏斗]]形（大概和半個[[球體|球體]]狀相似，但稍尖），球中央開一孔，連接另一流沙池，另一邊則由[[木板|木板]]蓋著。其餘則五花八門，如[[三角形|三角形]]、[[四角形|四角形]]、[[五角形|五角形]]、[[柱形|柱形]]……[[不規則形|不規則形]]皆有，而且兩個流沙池可有不同的外形。目前最常見的質料是[[玻璃|玻璃]]。

== 流沙 ==
[[粗沙|粗沙]]、[[幼沙|幼沙]]、[[細沙|細沙]]甚至[[小石子|小石子]]也可作流沙，只要能夠在兩流沙池內互相流動就可作流沙，流速快慢不拘。

== 歷史 ==
[[File:Ambrogio_Lorenzetti_002-detail-Temperance.jpg|right]]

=== 發展流程 ===
沙漏最早被發現於[[中國|中國]]、[[希臘|希臘]]、[[義大利|義大利]]等地，被使用於航海，[[澳洲國會|澳洲國會]]使用沙漏作為會議議程計時之用。

計時儀器的發展：
* [[日光鐘|日光鐘]] → [[水鐘|水鐘]]、[[香印|香印]]<ref>把香製作成特定形狀，當燒到特定位置時，就代表某個時辰</ref> → [[沙鐘|沙鐘]] → [[機械鐘|機械鐘]]。
前三種是[[古代|古代]]所用的，[[機械鐘|機械鐘]]現已廣泛被人們採用，成為人們生活中不可缺少的一份。

沙漏正是停在沙鐘這一階段。沙漏是由[[漏刻|漏刻]]（水鐘）演變而成的。

=== 漏刻 ===
漏刻同樣是一計時儀器。[[中國|中國]]古代時，人們發現[[瓷器|瓷器]]內的[[水|水]]會由裂縫中有規律地漏出來。於是便研製了漏刻出來，在一個留有小孔的[[漏壺|漏壺]]內注入水（或[[水銀|水銀]]）後，水會一滴一滴地漏進另一容器 - [[箭壺|箭壺]]中。在箭壺蓋開一小洞，再插一[[箭桿|箭桿]]，箭桿上刻上[[時間|時間]]，再用一[[浮木|浮木]]把箭桿托起便能知道大體時間了。

後來，人們知道壺內水越多，流速越快，水越少，流速越慢，會影響計時的精確度，於是在壺上再加一漏壺，水一流走，馬上補充水份，計算時間得以準確。

据[[梁代|梁代]]《漏刻经》记载:“漏刻之作，盖肇于[[轩辕|轩辕]]之日，宣乎夏商之代”。这说明，早在公元前三、四千年的父系氏族公社时期，中华大地就已经出现了漏刻。

=== 沙漏 ===
漏刻有一壞處，在[[氣候|氣候]]寒冷時壺內水份會變冷甚致結[[冰|冰]]，以致流速極慢或不流，但用水銀又過於昂貴，因而改用[[沙|沙]]代替水，並將之簡化，稱之為「沙漏」。

== 運作原理 ==
流沙由一個流沙池流到另一個流沙池，直至流光，因為內裏是[[密封|密封]]的，所以流完的時間每次都會相同，流沙池上繪有刻度，從而得知時間。同樣可控制流沙量而獲得不同時間。每一邊流完後都可倒轉，再次使用。

== 用處 ==
[[File:Hourglass.jpg|thumb]]
* 計算時間。
* 有欣賞和商業價值，經包裝的沙漏款式新穎，常被用作裝飾品。
* 古人[[斐迪南·麥哲倫|斐迪南·麥哲倫]]的航行中，他就在他的每一艘[[船|船]]上放置18個沙漏，用以計算時間及紀錄[[日誌|日誌]]。

== 最著名的沙漏 - 五輪沙漏 ==
* [[五輪沙漏|五輪沙漏]]於1360年由[[詹希元|詹希元]]研製成功<ref>[[明朝|明]][[游潜|游潜]]《[[博物志補|博物志補]]》：「五輪沙漏：北方水善凍，壺漏不下，新安詹希元以沙代水，人以為古未有也。」</ref>。[[宋濂|宋濂]]為此撰寫一篇《五轮沙漏铭》<ref>《宋学士全集》卷15</ref>。
* 脫離了傳統[[天文儀器|天文儀器]]的輔助，變成一種[[機械性|機械性]]的[[時鐘|時鐘]]，和現代時鐘運作原理接近。
* 流沙流出時帶動初輪，其他各級[[機械齒輪|機械齒輪]]亦被初輪帶動族轉。最後帶動在水面上水平族轉的中輪，這中輪[[軸心|軸心]]有一[[指針|指針]]，指針在一有刻線的[[圓盤|圓盤]]上轉動，可顯示時間。
* 還設有兩[[木人|木人]]站在五輪沙漏上，每過一段時間，便會自動擊[[鼓|鼓]]報時。
