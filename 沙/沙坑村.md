沙坑村是中華民國臺灣[[嘉義縣|嘉義縣]][[竹崎鄉|竹崎鄉]]轄下的行政區。位於嘉義縣東北方，北鄰嘉義縣竹崎鄉坑頭村，西南鄰竹崎鄉獅埜村，東接竹崎鄉復金村。

面積約3.8367平方公里，行政區包含12個鄰、291戶、人口數927人。聚落內的[[沙坑國小|沙坑國小]]以及[[沙坑真武宮|沙坑真武宮]]為該村景點。

==地理==
沙坑村位於竹崎鄉西北角，地形主要是淺山丘陵地，獅子頭溪為主要水系。年均溫大約27到28度，年雨量充沛，動植物生態豐富。

山地、丘陵間的小山谷或沖積扇頂稱為「坑<ref name="坑">{{cite web |author1=內政部 |title=地名應用 |url=http://gn.moi.gov.tw/GeoNames/usage.aspx |accessdate=2018-10-21}}</ref>」，因此與坑有關的地名，多分布在山間谷地或山腳下與平原交界處。

==聚落歷史==
沙坑村目前區域由下厝仔、沙坑、大竹圍、水坑仔、木柵寮以及羗仔寮六個聚落<ref name="地理志">{{cite web |author1=嘉義縣文化觀光局 |title=嘉義縣志-地理志 |url=https://www.tbocc.gov.tw/ModuleMsg/News_View.aspx?NewsClassID=61&lang=tw |accessdate=2018-10-21}}</ref>組成。

==教育==
沙坑村本區域內目前無設立國中，國小學區為沙坑國小<ref name="沙坑國小">{{cite web |author1=SAKPS版權所有 |title=沙坑國小 |url=http://www.sakps.cyc.edu.tw/ |accessdate=2018-10-21}}</ref>與竹崎國小；國中學區為[[民雄鄉|民雄鄉]]大吉國中。

沙坑村沙坑國小的創立於1955年，時為嘉義縣竹崎鄉義仁國民學校沙坑分班(一班)。1968年後改制為嘉義縣竹崎鄉沙坑國民小學。

==文化資源/文化資產==
===信仰===
沙坑村沙坑真武宮<ref name="沙坑真武宮">{{cite web |author1=中央研究院人社中心地理資訊科學研究專題中心 |title=沙坑真武宮 |url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/juchi/1014004-ZWG |accessdate=2018-10-21}}</ref>的主神為[[北極玄天上帝|北極玄天上帝]]，位於嘉義縣竹崎鄉沙坑村水坑61號。寺廟的建立起源於[[康熙|康熙]]十三年 (西元1674年)時，閩粵兩省先民帶著玄天上帝的香火來台灣開墾，因為十分靈驗所以香火隨之鼎盛。於是在[[清朝|清朝]][[嘉慶|嘉慶]]二十三年(西元1818年)，居民提議建廟，在水坑溪仔平奠基，並且在這一年完工，稱作真武宮。

==其他特色==
===村里產業===
早期以生產麻竹筍<ref name="產業">{{cite web |author1=嘉義縣竹崎鄉公所 |title=村里產業 |url=https://chuchi.cyhg.gov.tw/cp.aspx?n=BC8C007404B8C737&s=F6E116B26C6DEE85 |accessdate=2018-10-21}}</ref>聞名，近年來，除麻竹筍外，改種柑橘、柳橙、荔枝、香蕉、火龍果、酪梨、龍眼、鳳梨、高接梨等。

===技藝===
本村沙坑國小設有優質國樂班<ref name="沙坑國小" />，常受各級單位邀約表演，獎項非凡，成績優異。沙坑社區另有民俗技藝班「舞龍舞獅」團隊，為村里凝聚力的象徵。
