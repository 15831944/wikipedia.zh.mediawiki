'''沙文主义'''（{{Lang-fr|Chauvinisme}} ; {{Lang-en|Chauvinism}}）原指极端的、不合理的、过分的爱国主义（因此也是一种[[民族主义|民族主义]]）。如今的含義也囊括其他領域，主要指盲目熱愛自己所處的團體，並經常對其他團體懷有惡意與仇恨。

词源是[[拿破仑|拿破仑]]手下的一名（有可能是位文藝上杜撰的）士兵[[尼古拉·沙文|尼古拉·沙文]]（Nicolas Chauvin），他由于获得军功章对拿破仑感恩戴德，对拿破仑以军事力量征服其他民族的政策狂热崇拜。而“沙文主义”这个名词則首先出现在[[法国|法国]]的一部[[戏剧|戏剧]]“三色帽徽”中，以讽刺的口吻描写沙文的这种情绪。後来这个词被广泛应用，如[[大国沙文主义|大国沙文主义]]、民族沙文主义等，還曾被[[女权运动|女权运动]]的领袖用於“[[男性沙文主义|男性沙文主义]]”（相当于[[汉语|汉语]]中的“[[大男人主义|大男人主义]]”）。[[英語|英語]]中有一個對等的詞，是Jingoism。所以經過演变，如今“沙文主義”這個詞在英語中更多場合中指[[種族歧視|種族歧視]]與[[性別歧視|性別歧視]]，尤其是後者。

沙文主義者一般都是對自己所在的國家、民族和團體感到驕傲，因此看不起其他的國家、民族和團體。

== 極端民族主義 ==
{{see|民族主義|分離主義}}

在19世紀時，歐洲各國興起[[民族主義|民族主義]]，提倡擁有共同文化、語言及歷史的民族建立屬於他們的國家，於是多個民族發動了獨立運動，成功擺脫外族的管治，建立統一及獨立的[[民族國家|民族國家]]。例子有[[比利時|比利時]]脫離[[荷蘭|荷蘭]]的統治，在1831年取得[[獨立運動|獨立]]；[[普魯士|普魯士]]（[[大德意志|德意志]]邦國）先後打敗[[丹麥|丹麥]]、[[奧地利帝國|奧地利帝國]]及[[法國|法國]]於1871年建立統一的國家。

但是，到了19世紀末和20世紀初，歐洲列強為了增進本民族的利益，不惜付出任何代價，試圖加強自己在歐洲及全世界的影響力，從而令這種擁護自己民族的情緒變質，成為一種危險的[[愛國主義|愛國主義]]，譬如[[納粹主義|納粹主義]]。

而部分的極端民族主義會演變成[[排外|排外]]情緒，甚至流於[[恐怖主義|恐怖主義]]；譬如德國與俄羅斯等國的[[新納粹主義|新納粹主義]]、[[北愛爾蘭|北愛爾蘭]]的[[基督教恐怖主義|基督教恐怖主義]]、東南亞的[[排華|排華]]運動等等。

現今典型的極端民族主義有包含美国的白人優越主义、[[阿拉伯國家|阿拉伯國家]]的[[基本教義派|基本教義派]]、[[馬來西亞|馬來西亞]]的[[馬來人至上|馬來人至上]]觀念、[[伊朗|伊朗]][[什葉派|什葉派]]本位主義、[[大漢沙文主義|中國]][[大汉族主义|大漢族主義]]、日韓[[右翼|右翼]]民族主義、朝鲜[[金日成|金日成]]奠基的[[主體思想|主體思想]]、[[南非共和國|南非]]的[[波耳人|波耳人]][[種族隔離|種族隔離]]制度、[[臺灣主體意識|臺灣主體意識]]等。

== 大国沙文主义 ==

大国沙文主义原指极端的、不合理的、过分的爱国主义（因此也是一种民族主义），如苏联发起的[[冬季战争|冬季战争]]、[[阿富汗战争_(1979年)|阿富汗战争]]，美国发动的[[美墨戰爭|美墨戰爭]]、[[越战|越战]]，英国发起的[[鴉片戰爭|鴉片戰爭]]、日本挑起的[[甲午戰爭|甲午戰爭]]、[[中國抗日戰爭|侵華戰爭]]等。如今的含义也囊括其他领域，主要指盲目热爱自己所处的团体，并经常对其他团体怀有恶意与仇恨。大国沙文主义者一般都是对自己所在的国家、团体、民族感到骄傲，因此看不起其他的国家、民族和团体。

== 男性沙文主義 ==
{{main|男性沙文主義}}

男性沙文主義，是一種認為[[男性|男性]]必定優於[[女性|女性]]的[[理念|理念]]。男性沙文主義者被俗稱為「沙豬」。此外，男性沙文主義亦包括認為男性優於女性是因為男性擁有[[陽具|陽具]]。

男性沙文主義常見於[[父權|父權]]社會中的[[家庭|家庭]]，例如[[伊斯蘭教|伊斯蘭教]]的[[穆斯林|穆斯林]]社會，[[丈夫|丈夫]]或[[父親|父親]]的[[權力|權力]]是至高無上的，妻子和孩子應該要絕對服從。 

在阿拉伯部落傳統（非伊斯蘭教義）之中，[[父親|父親]]還可掌握女兒的生殺大權 - [[名譽殺人|名譽殺人]]（以維持名譽為由，殺死被強暴或行了未婚性行為的女兒）。同樣基於以牙還牙以眼還眼的部落傳統，如果女兒失貞，其監護人有權強暴對方家族女性親屬。

== 民族沙文主義==
民族沙文主义及沙文主义，沙文主义奉行极端的、不合理的、过分的爱国主义。因此沙文主义也是民族主义的一种。
