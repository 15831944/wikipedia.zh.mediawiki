== 屬會 ==
[[File:HK_STSA_HQ_LekYuenEstate.JPG|thumb]]壽全樓地下的會址]]
沙田體育會轄下擁有14個體育屬會：
# '''田徑會'''：為[[香港業餘田徑總會|香港業餘田徑總會]]贊助屬會會員。
# '''社交舞會'''
# '''划艇會'''：設有1至3星級訓練班，除教授初學者基本知識外，並組織精英隊伍代表沙田區比賽。
# '''棒球會'''：於1983年由當時沙田民政專員[[曾蔭權|曾蔭權]]支持，成立香港首支華人少棒隊「沙燕隊」，發展至今已成立6支球隊，包括、「勇士隊(白)」、「勇士隊(藍)」、「田龍隊」、「沙燕隊」、「少獅隊」及「飛虎隊」，參與「香港少棒聯盟」及「香港棒球總會」舉辦的各組別比賽，並曾多次奪得聯賽冠軍。
# '''射藝會'''：為[[香港射箭總會|香港射箭總會]]成員屬會。
# '''跆拳道會'''
# '''武術會'''：於2000年7月成立，在區內推廣武術運動，舉辦長期訓練班，由[[世界武術錦標賽|世界武術錦標賽]]冠軍張萬強總教練主理，課程內容包括7種基本腳法、5種步型及基本[[少林拳|少林拳]]術，每年主辦「沙田武術公開賽」。
# '''單車會'''：由前港隊成員[[洪松蔭|洪松蔭]]擔任領隊，定期於大水坑單車公園訓練，並參加由[[香港單車聯會|香港單車聯會]]舉辦之賽事。
# '''[[沙田體育會足球隊|足球會]]'''：由[[立法會|立法會]]兼[[沙田區議會|沙田區議會]][[議員|議員]][[林大輝|林大輝]]擔任榮譽領隊。
# '''-{zh-hk:乒乓球會;zh-hans:乒乓球会;zh-tw:桌球會;}-'''
# '''草地滾球會'''：為[[香港草地滾球總會|香港草地滾球總會]]註冊會員，亦派隊參與公開賽事及聯賽。
# '''網球會'''：設有初班、技術改良及培訓班。
# '''游泳會'''
# '''籃球會'''

== 單車公園 ==
{{主條目|大圍單車公園}}
{{主條目|大水坑單車公園}}
單車公園成立至今一直由沙田體育會管理，原本位於[[沙田|沙田]][[大圍|大圍]]，隨著建立[[馬鞍山綫|馬鐵]]大圍車廠及城市規劃發展，「大圍單車公園」於2001年2月關閉，[[香港政府|香港政府]]以收取象徵式租金的短期租約形式在[[馬鞍山_(香港市鎮)|馬鞍山]]大水坑撥地予沙田體育會興建新的「馬鞍山單車公園」<ref>[http://www.info.gov.hk/gia/general/200106/20/0620159.htm 立法會十題：新單車公園]</ref>。以往沙田體育會的營運開支由單車公園的收入補貼，惟自遷移至大水坑後，由於位置偏僻，單車公園的收入減少，導致沙田體育會每年錄得十多萬港元的虧損。

== 沙田龍舟競賽 ==
沙田體育會由1984年起主辦一年一度「沙田龍舟競賽」，並由[[沙田民政處|沙田民政處]]及[[康樂及文化事務署|康樂及文化事務署]]協辦，獲得[[沙田區議會|沙田區議會]]贊助，是區內一年一度的[[端午節|傳統節日]]盛事。比賽地點位於[[城門河|城門河]][[翠榕橋|翠榕橋]]至[[沙燕橋|沙燕橋]]之間一段河道，通常分為男子中龍（20-26人）、男子小龍（9人）及女子鳳艇（9人）的級別，競逐公開或邀請賽。
