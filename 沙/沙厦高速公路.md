'''厦沙高速公路'''是连接[[福建省|福建省]][[厦门市|厦门市]]与[[沙县|沙县]]的一条[[高速公路|高速公路]]。是海西高速公路“六横”主骨架网中的重要组成部分，途经厦门[[同安区|同安区]]、[[泉州市|泉州市]][[安溪县|安溪县]]、[[永春县|永春县]]、[[德化县|德化县]]和[[三明市|三明市]][[尤溪县|尤溪县]]等地区。2005年，国家交通部支持海西高速公路网建设，拟规划建设一条厦门市至沙县高速公路，推荐方案起点位于厦门田厝互通，经安溪县芹坑口、永春县、德化县的县城、上涌、葛坑、尤溪县的华口、大头桥、下川、西城，沙县的琅口，终于际口[[互通|互通]]接[[福银高速|福银高速]]公路。

厦沙高速公路全长241公里，按双向4车道、时速80公里设计，路基度24米，总投资176.47亿元。

== 厦门-安溪段 ==
厦门至沙县高速公路项目安溪段工程在2011年全面开工建设。2010年12月，厦门同安至泉州安溪段已开工建设。2012年12月31日厦门至安溪（属厦沙高速S30，厦门到沙县）和安溪至南安金淘（属[[福诏高速|福诏高速]]S35，福鼎沙埕到诏安）高速公路正式通车，但只有主线通车，凤南，莲花收费站未启用，2013年8月30日厦门新收费站可以连接到厦沙高速<ref>{{cite news|title=明日起厦沙高速公路厦门收费站启用|url=http://www.xmnn.cn/dzbk/xmwb/20130829/201308/t20130829_3472299.htm|accessdate=2014-04-26|newspaper=厦门网|date=2013-08-29}}</ref>，并且厦沙高速可以连接到[[沈海高速公路|沈海高速公路]][[G15|G15]]，已启用厦门（新）、厦门北、同安凤南、安溪龙门、安溪官桥收费站<ref>{{cite news|title=南安至厦门高速主线通车 只能通过厦门北站互通上高速|url=http://news.163.com/13/0101/10/8K4JOIOK00014JB6.html|accessdate=2014-04-26|newspaper=网易新闻|date=2013-01-01}}</ref>。可以在安溪城关芹关口通过福诏高速至金淘亭川枢纽连接到[[泉南高速公路|泉南高速公路]][[G72|G72]]。

== 安溪段 ==
厦沙高速公路安溪段项目起于泉州市[[安溪县|安溪县]]凤城枢纽互通，接沈海高速复线，经安溪县城厢镇、官桥镇、蓬莱镇、金谷镇，接永春县达埔镇，经岩峰枢纽互通连接泉三高速，全长31.25公里。于2017年11月16日正式通车。<ref>{{cite news|title=厦沙高速安溪段和德化县城至国宝段建成通车|url=https://xw.qq.com/fj/20171117016015/FJC2017111701601500|accessdate=2017-11-18|newspaper=腾讯大闽网|date=2017-11-17}}</ref>

== 德化段 ==

厦沙高速公路德化段项目起于泉州市[[永春县|永春县]]达埔枢纽互通，接[[莆永高速|莆永高速]]公路，利用[[泉三高速|泉三高速]]公路（9.9公里）及德化连接线，经德化县龙浔、浔中、国宝、赤水、上涌、汤头，终于德化县葛坑镇大岭村，顺接厦沙高速公路三明段，路线全长73.694公里，实际新建路线里程54.287公里。
该项目全线路段按双向四车道设计，路基宽度24.5米。全线共设德化、国宝、上涌3处互通立交，全线设计行车速度80km/h，计划工期为36个月，工程概算48.3亿元。
<ref>{{cite news|title=厦沙高速德化段昨开工|url=http://www.qzwb.com/gb/content/2014-12/20/content_5013492.htm|accessdate=2015-03-08|newspaper=泉州网|date=2014-12-20}}</ref>。2017年11月16日，德化县城至国宝段正式通车<ref>{{cite news|title=厦沙高速安溪段和德化县城至国宝段建成通车|url=https://xw.qq.com/fj/20171117016015/FJC2017111701601500|accessdate=2017-11-18|newspaper=腾讯大闽网|date=2017-11-17}}</ref>。2017年12月12日，德化国宝至葛坑段全线通车<ref>{{cite news|title=厦沙高速全线通车 厦门至三明三小时可达|url=http://www.mnw.cn/xiamen/news/1900704.html|accessdate=2018-05-03|newspaper=闽南网|date=2017-12-13}}</ref>
== 尤溪段 ==

厦沙高速公路[[三明市|三明市]]境段全长83.44公里，概算总投资72.47亿元，工期3年，分10个标段施工；沿线共设置华口、坂面、尤溪西、南阳、沙县东、际口6处互通式立交。其中，尤溪县境内59.79公里，概算投资49.66亿元，设7个施工标段。经过坂面、中仙、台溪、西城等4个乡镇19个村,在西城镇玉池村设尤溪西互通，坂面镇下川设坂面互通，西城镇秀村设综合服务区。

2014年6月8日，总长1767米的厦沙高速公路A4标段控制性工程蒋坑岬隧道动工建设，这标志着厦沙高速公路尤溪段工程正式开工建设。6月份尤溪段将全线动工。<ref>{{cite news|title=厦沙高速公路尤溪段正式开工建设|url=http://www.smnet.com.cn/xwpd/economy/201406/t20140610_200264333.htm|accessdate=2015-03-08|newspaper=三明日报|date=2014-06-10}}</ref>。2017年12月12日，厦沙高速尤溪段全线贯通<ref>{{cite news|title=厦沙高速全线通车 厦门至三明三小时可达|url=http://www.mnw.cn/xiamen/news/1900704.html|accessdate=2018-05-03|newspaper=闽南网|date=2017-12-13}}</ref>
