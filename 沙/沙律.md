'''沙律'''（{{Lang-en|salad}}），是西餐中的一道普遍菜式，通常是[[配菜|配菜]]或前菜，有時也會作為一道主菜。沙拉可被視為[[涼菜|涼菜]]。

沙律經常與[[蔬菜|蔬菜]]、[[肉|肉]]、[[海鮮|海鮮]]或[[蛋|蛋]]與[[蛋黃醬|蛋黃醬]]混合食用。蔬菜沙律可以作為一部份菜的調味料，也常用作[[三文治|三文治]]餡料。沙律的種類包括蛋沙律、雞丁沙律、吞拿魚沙律、蝦沙律和火腿沙律。在[[丹麥|丹麥]]，人們將沙律加上肉片製成上層不蓋麵包的{{tsl|en|Smørrebrød|開口三文治}}，其他種類还有黃瓜沙律、辣根沙律、[[意大利沙律|意大利沙律]]和[[俄國沙律|俄國沙律]]等等。

== 詞源 ==
在英語中，沙拉一詞最早出現在十四世紀，當時寫作“salad”或“sallet”。英語“salad”是從[[法語|法語]]“salade”變化而來，而該詞又來源於[[拉丁語|拉丁語]]“salata”，本義爲“鹹的”，是“sal”（鹽）的派生詞。在[[古羅馬|古羅馬]]時期蔬菜常以[[鹽水|鹽水]]或含鹽的油與醋作佐料，沙拉的名稱亦因此與鹽聯繫起來<ref name="etymonline">{{OEtymD|salad}}</ref>。

== 歷史 ==
在[[中世紀|中世紀]]，一個長冬以後，人們會以鹹肉來醃製菜，這被認為是沙拉的源頭。也有说法稱沙拉源於[[羅馬帝國|羅馬帝國]]。

記者[[約翰·伊夫林|約翰·伊夫林]]（John Evelyn）寫了一本有關沙拉的書《Acetaria: A Discourse on Sallets》，這本書中稱沙拉來自[[義大利|義大利]]或[[荷蘭|荷蘭]]。

== 蔬菜沙拉 ==
[[File:GreenSalad.jpg|right]]
蔬菜沙拉大多由未煮過的蔬菜混合而成。蔬菜葉材料包括[[萵苣|萵苣]]、[[蒲公英|蒲公英]]、[[菠菜|菠菜]]或[[芝麻菜|芝麻菜]]等。

其它可用於製作沙拉的蔬菜材料包括[[番茄|番茄]]、[[黃瓜|黃瓜]]、[[胡椒|胡椒]]、[[蘑菇|蘑菇]]、[[蔥|蔥]]、[[大蔥|大蔥]]、[[胡蘿蔔|胡蘿蔔]]、[[蘿蔔|蘿蔔]]和[[酪梨|酪梨]]等。其它食材還有[[麵團|麵團]]、[[橄欖_(木樨科)|橄欖]]、[[馬鈴薯|馬鈴薯]]、[[稻|米]]、油煎方型小麵包片、肉（例如[[煙肉|煙肉]]、[[雞|雞]]）、[[乳酪|乳酪]]或[[鱼|魚]]（例如[[鮪魚|鮪魚]]）。

葉菜類清洗後切絲或撕片，根莖類煮熟後切塊或切片，可生食的瓜果類切塊或切片，不可生食的瓜果類則切塊或切片後需煮熟，肉類、穀類或雜糧類則須熟食。魚若是新鮮可切片做生魚片，若非則須熟食。

== 其它類型沙拉 ==
一些沙拉根據其他食物製成。
[[File:Noodlesalad1.jpg|right]]沙拉]]

[[File:Fruit_salad.JPG|right]]沙拉]]

* [[蛋沙拉|蛋沙拉]] 
* [[水果沙拉|水果沙拉]] （[[楊桃|楊桃]]、[[香蕉|香蕉]]、[[奇異果|奇異果]]、[[木瓜|木瓜]]、[[菠蘿|菠蘿]]、[[石榴|石榴]]等。）

* [[馬鈴薯沙拉|馬鈴薯沙拉]]

* [[水門沙拉|水門沙拉]] （[[開心果|開心果]]、[[布丁|布丁]]、[[菠蘿|菠蘿]]、[[蛋白軟糖|蛋白軟糖]]、[[橘子|橘子]]和打好的[[奶油|奶油]]製成。）

== 安全問題 ==
由於製作沙拉的過程中，蔬菜及不少配料均沒有經過煎煮等處理，因此如擺放太久容易發生[[變質|變質]]，甚至引起[[食物中毒|食物中毒]]<ref>[http://www.takungpao.com/news/07/12/21/GW-840415.htm 24體藝生疑食沙律中毒] {{webarchive|url=https://web.archive.org/web/20071228044054/http://www.takungpao.com/news/07/12/21/GW-840415.htm |date=2007-12-28 }}，載大公網，2007年12月21日</ref>。
