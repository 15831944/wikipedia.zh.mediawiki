'''沙河粉'''，簡稱'''河粉'''、'''河'''或'''粉'''，是一種通過蒸煮米漿而成的粉條食品，是[[中国南方|中国南方]]和[[港澳|港澳]]、[[东南亚|东南亚]]一带常见的一种[[食品|食品]]，常見的煮法為炒或水煮後配湯，例如炒牛河、上湯河粉。正宗山水沙河粉的原产[[广州|广州]][[沙河街道_(广州市)|沙河镇]]。

== 制作方法 ==
[[File:Chow_Kueh_Teow.JPG|thumb]]

沙河粉原料是[[大米|大米]]，将米洗净後磨成粉加水制成米浆，用米浆蒸成薄粉皮、再切成带状而成。{{fact|time=2019-02-08|因它是用[[白云山|白云山]]上的[[九龙泉|九龙泉]]水泡制的}}，所产的粉薄白透明，爽软韧筋兼备，炒、泡、拌食皆宜。

== 烹调方法 ==
[[File:Wok_Cooking.jpg|thumb]]
沙河粉可以用[[蒸|蒸]]、[[炒|炒]]（分干炒和湿炒两种）、[[煮|煮]]等多种方法烹调。　
* 乾炒：加油配[[韭黃|韭黃]]以猛火炒熟，加上[[酱油|酱油]]，可配[[牛肉|牛肉]]、[[牛腩|牛腩]]、猪什等，其中以[[乾炒牛河|乾炒牛河]]最為著名。
* 湿炒：在干炒后，打上[[芡汁|芡汁]]。
* 煮：與一般湯麵麵食類似，煮熟后配上[[大地魚|大地魚]][[上湯|上湯]]作湯底，再將河粉佐以[[魚丸|魚丸]]、[[牛丸|牛丸]]、[[雲吞|雲吞]]或其他肉類等配料。在廣東各地的粉麵食店中相當普遍，店家一般會供客人作為麵食之中的一種選擇（[[粗麵|粗麵]]/[[幼麵|幼麵]]/[[米粉|米粉]]/河粉）。
* 蒸：蒸熟后，自行醮酱。

== 现状 ==
[[File:Sha_Ho_Fun_in_Ipoh.jpg|thumb]]
手工制成的沙河粉色白，近来也有店家在其中加入各种[[蔬菜|蔬菜]]汁或者[[水果|水果]]汁，制成五颜六色的沙河粉。沙河粉已实现机械化生产，在广州随之出现了大量自行生产的米粉，部分厂家甚至在陈化米中加入[[硼砂|硼砂]]制作，影响了饮食安全。

2004年，广州市政府考虑将沙河粉以“广州沙河粉”之名申报“原产地域产品”保护，从而加强对沙河粉制造业的管理。<ref>[http://www.southcn.com/news/gdnews/nanyuedadi/200304020647.htm 广东新闻：产权保护沙河粉将申请地域保护不让随意标称]</ref>
2009年，炒河粉被[[洛杉矶时报|洛杉矶时报]]评为“年度十大食谱”冠军。

沙河粉也隨著20世紀初，大量的[[客家|客家]]與[[廣東|廣東]]移民至[[馬來半島|馬來半島]]，在當地也發展出許多麵食，其中最有名的是[[馬來西亞|馬來西亞]][[霹靂州|霹靂州]]的[[雞絲河粉|雞絲河粉]]、[[芽菜雞|芽菜雞]]等。

==其他称谓==
河粉和其他地方的同类食品，如[[客家人|客家人]]的[[粄条|粄条]]、[[闽南人|闽南人]]的[[粿条|粿条]]极为相似，甚至无法分辨，以致很多时候，在多种方言交叠的地区，[[粄条|粄条]]、[[粿条|粿条]]和河粉都被视为同一种食物，例如在马、新一帶，在福建人、广府人、潮州人和客家人生活交融的环境，河粉和粿条可说是同义词。
