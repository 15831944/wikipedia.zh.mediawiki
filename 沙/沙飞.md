'''沙飞'''（{{bd|1912年|5月5日|1950年|3月4日}}）原名司徒传，广东[[开平|开平]]人，摄影师，1936年10月拍摄发表[[鲁迅|鲁迅]]最后的留影、[[鲁迅|鲁迅]]遗容及其葬礼的摄影作品。抗日战争时期成为中共军旅摄影师，《[[人民日报|人民日报]]》的前身《抗敌报》和《[[解放军画报|解放军画报]]》的前身《晋察冀画报》的创始人。1950年3月因精神失常射杀[[日本|日本]]医生[[津泽胜|津泽胜]]被华北军区政治部军法处判处死刑<ref>{{cite web|url = http://dangshi.people.com.cn/GB/16195131.html|title = 沙飞的悲剧人生 |accessdate = 2012-06-06 |date =2011-10-11 |language =zh-hans |publisher =党史博览|author = 陈辉}}</ref>。

==生平==
[[File:Lu_Xun1.jpg|left]]生平不苟言笑，這是鲁迅照片中唯一大笑的镜头。沙飞摄于1936年10月8日。十一天后，鲁迅逝世。]]

1912年5月5日，沙飞在广东省[[开平市|开平县]]的一个药商家庭出生。1926年，年仅14岁的沙飞在无线电学校毕业后参军，在[[国民革命军北伐|北伐]]队伍中成为了[[无线电|电台]]报务员。他跟随军队先后来到上海、宁波、济南、北京等地。北伐战争胜利后，在广西梧州的军用电台工作了3年时间。1935年6月，加入了上海的摄影组织[[上海黑白影社|上海黑白影社]]。1936年9月，离开汕头，来到了上海。1936年10月8日，沙飞为参加[[上海八仙桥青年会|上海八仙桥青年会]]第二届[[全国木刻展览会|全国木刻展览会]]的鲁迅拍照。在这个月28日，广州《[[民国日报_(广州)|民国日报]]》刊登了沙飞的文章《鲁迅先生在全国木刻展会场裏》。<ref name="陈辉"/>

1936年10月19日清晨五点二十五分鲁迅在上海因肺结核病去世，終年55岁。鲁迅的死讯震惊全中国，沙飞也不例外。沙飞在得知鲁迅逝世后，立即带上照相机，直奔[[上海鲁迅故居|鲁迅所在的寓所]]。沙飞在鲁迅遗体前深鞠三躬，之后用照相机记录了鲁迅葬礼的所有过程。<ref name="陈辉">{{cite web |author1=陈辉 |title=八路军著名摄影师沙飞住院期间为何枪杀主治医生？ |url=http://history.people.com.cn/GB/205396/16324835.html |website=人民网文史频道 |publisher=《党史博览》 |accessdate=2018-12-10 |date=2011-11-21}}</ref>

1937年10月，沙飞在[[河北|河北]][[阜平|阜平]]，经晋察冀军区司令员兼政委[[聂荣臻|聂荣臻]]批准，加入[[八路军|八路军]]，并正式由司徒传改名为沙飞，成为共产党部队第一位专职新闻摄影记者。12月到晋察冀军区《抗敌报》编辑部任副主任<ref>{{cite web|url = http://mil.cnr.cn/jszl/xwdd/201111/t20111115_508783444.html|title =聂荣臻为何挥泪处决八路军著名摄影师|accessdate = 2012-06-11 |date =2011-11-15|language =zh-hans |publisher =党史博览}}</ref>。

[[File:Norman_Bethune_China_1938.jpg|thumb]]（左侧）和[[聂荣臻|聂荣臻]]（中间）见面。沙飞摄。]]
1938年5月，沙飞因为身体不好住进[[山西|山西]][[五台县|五台县]]耿镇河北村的军区卫生部卫生所休养。6月[[白求恩|白求恩]]抵达，沙飞闻讯后前去为白求恩拍摄了照片。1938年8月，松岩口村建立“模范医院”，沙飞在此间为白求恩的医疗队拍摄了大量工作照片。1939年白求恩因感染逝世，沙飞为其拍摄了遗容。根据白求恩去世前的遗嘱，白求恩所带的那部带有柯达镜头的莱丁娜照相机留给了沙飞。<ref>{{cite web|url = http://news.xinhuanet.com/it/2009-10/18/content_12260812_1.htm|title = 诺尔曼·白求恩的遗嘱：照相机给沙飞 |accessdate = 2012-06-11 |date =2009-10-18 |language =zh-hans |publisher =新华网}}</ref>

沙飞在抗日战争期间患有严重的精神疾病。1949年沙飞因结核病住院治疗，但精神失控的他用手枪杀死了为他治病的日本医生[[津泽胜|津泽胜]]。由于事态严重，华北军区司令员[[聂荣臻|聂荣臻]]最终决定枪决沙飞，以整训军规、稳定当时大量日籍医疗人员<ref>{{cite news|title=沙飞五十五年祭|url=http://www.gmw.cn/content/2005-09/26/content_309446.htm|accessdate=2015-01-16|newspaper=光明网}}</ref>。1950年3月4日，沙飞在河北石家庄遭处决，得年37岁。

沙飞死后，他的署名一度从所有作品上被抹掉<ref>{{cite journal|url = http://zqb.cyol.com/content/2009-06/17/content_2715076.htm|title = 共产党第一位摄影记者沙飞：新闻摄影不“新闻”|accessdate = 2012-06-11 |date =2009-06-17 |language =zh-hans |publisher =[[中国青年报|中国青年报]]|author=林天宏}}</ref>。1986年5月19日北京军区法院判决撤销1950年华北军区政治部军法处1950年2月的判决，给沙飞恢复军籍。6月北京军区纪律检查委员会决定恢复党籍。
