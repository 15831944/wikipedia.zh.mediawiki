'''{{FULLPAGENAME}}'''是{{PRC admin/fullname|43/09/21/404/000|3}}曾下辖的一个[[类似乡级单位|类似乡级单位]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|43/09/21/404/000}}}}</ref>。2000年，益阳将原[[大通湖农场|大通湖]]、[[北洲子农场|北洲子]]、[[金盆农场|金盆]]、[[千山红农场|千山红]]等国营农场改为镇，设立[[大通湖管理区|大通湖管理区]]。沙堡洲办事处即是依托大通湖渔场设立<ref>{{Cite web |url=http://www.datonghu.gov.cn/content/detail/57f88c4e1be65ce8f92ce729.html |title=大通湖区建置沿革 第二节 建置沿革|accessdate=2018-06-21 |author=|date=2016-04-17|work= |publisher=中共大通湖区委、大通湖区管委会网站，来源：周卫锋|language=Zh-hans }}</ref>。2013年，[[湖南省人民政府|湖南省人民政府]]批准，以原大通湖渔场和沙堡洲办事处管辖范围设立[[沙堡洲镇|沙堡洲镇]]。驻地在沙堡洲<ref>{{Cite web |url=http://www.iyzx.com/portal.php?mod=view&aid=9092 |title=省政府批准设立沙堡洲镇|accessdate=2018-06-21 |author=记者：李双建，通讯员：李旭、郑春林|date=2013-06-04|work=编辑: 黄萍 |publisher=益阳在线，来源：益阳日报|language=Zh-hans }}</ref>。2015年，湖南省人民政府批准，沙堡洲镇并入[[河坝镇_(南县)|河坝镇]]。大通湖渔场作为企业保留<ref>{{Cite web |url=http://www.hunan.gov.cn/hnyw/szdt/201512/t20151201_4800797.html |title=益阳市乡镇区划调整改革方案获省政府批准|accessdate=2018-06-21 |author=|date=2015-12-01|work= |publisher=湖南省人民政府网站|language=Zh-hans }}</ref>。
