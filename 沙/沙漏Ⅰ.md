'''沙漏'''，2007年1月1日由[[当代世界出版社|当代世界出版社]]出版发行的书籍。作者是中国著名作家[[饶雪漫|饶雪漫]]。

== 内容简介 ==
该书为饶雪漫“[[青春疼痛系列|青春疼痛系列]]”之六。也为“沙漏”系列第一部。第一部主要讲述女孩[[莫醒醒|莫醒醒]]和[[米砂|米砂]]的故事。

== 书籍主题 ==
书籍主题是叙述了[[青春期|青春期]]男生女生成长、痛苦和欢乐以及他们之间的爱情故事。作者[[饶雪漫|饶雪漫]]自己说：“这个长长的[[迷宫|迷宫]]里，这里是最快活的一段，因为我在黑暗的行程里，好像感受到前方风的声音，所以跑得格外欢快和得意。就像一个锁匠给自己打造了一个完美的手铐，然后呼啦一下，自己打通了所有华丽的关口，获得自由。”<ref>{{cite web |url=http://news.163.com/07/1121/00/3TPIVB5C0001124J.html |title=饶雪漫《沙漏》下月将面世 |accessdate=2007-11-21 |work=|publisher=网易 |language= zh-hans}}</ref>

== 作者简介 ==
[[饶雪漫|饶雪漫]]，[[四川|四川]][[自贡|自贡]]人，1994年毕业于[[四川理工学院|四川理工学院]]人文学院文学系，当代青春文学作家。
