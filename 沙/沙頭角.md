'''沙頭角'''（香港部分-{{lang-en|'''Sha Tau Kok'''}}，中國大陸部分-{{lang-en|'''Shatoujiao'''}}）<ref>香港部分的英文名稱“Sha Tau Kok”是「沙頭角」三字的[[香港政府粵語拼音|粵語拼音]]，而中國大陸所使用的英文翻譯“Shatoujiao”是「沙頭角」三字的[[漢語拼音|漢語拼音]]（即以北京语音爲標準音的[[普通話|普通話]]的拼音）。</ref>是位於[[香港|香港]][[新界|新界]][[北區_(香港)|北區]]和[[深圳|深圳]][[盐田区|盐田区]]邊界上的地方，橫跨兩地，以前是人煙興盛的村莊，當1898年[[新界|新界]]正式租借與[[英國|英國]]時，以[[沙頭角河|沙頭角河]]分界，劃分為[[香港邊境禁區|香港邊境禁區]]，當中河尾一段成為[[中英街|中英街]]。早在清代，沙頭角的東和墟已為興盛的墟市，但隨著英國租借新界，沙頭角南面劃為英租界內，令村民來往不便。及後位於香港境內的部分沙頭角範圍劃為禁區，非沙頭角村落的村民不能輕易進入沙頭角，使沙頭角難復當年的興盛。

香港境內的沙頭角已建為一個小市鎮。沙頭角市中央為沙頭角公路石涌凹段，周圍還有多條以「順」字開頭的街道。其中沙頭角邨為大部分居民居住的地方。以前，不少沙頭角居民是海邊柵屋為居，就是住在船上的[[艇民|艇民]]，及後房協的沙頭角邨落成，很多居民因此上岸，搬進公共房屋，而車坪街及新樓街仍然有一定數量的樓宇。沙頭角另一標記為沙頭角碼頭。沙頭角碼頭位於海中心，有一條百多米長的橋將碼頭和陸地連接，碼頭有街渡前往吉澳、三椏等新界東北鄉郊地區。

自[[港英政府|英國]]於1899年接管香港新界後，直到1980年前，港英政府將現今北區範圍劃分四個分區，其中之一就是「沙頭角區」，由於此制度沿用近80年之久，故此現今香港方的「沙頭角」在廣義上也涵蓋原「沙頭角區」所覆蓋的地方，包括[[蓮麻坑|蓮麻坑]]、[[萬屋邊|萬屋邊]]、[[禾坑|禾坑]]、[[石涌凹|石涌凹]]、[[南涌|南涌]]、[[鹿頸|鹿頸]]等地，而新娘潭路以東的半島也被稱爲「[[沙頭角半島|沙頭角半島]]」（包括[[船灣淡水湖|船灣淡水湖]]），故此該半島上的[[烏蛟騰|烏蛟騰]]、[[谷埔|谷埔]]、[[鎖羅盆|鎖羅盆]]、[[荔枝窩|荔枝窩]]等地，以及[[印洲塘|印洲塘]]一帶以[[吉澳|吉澳]]爲首的島嶼都屬於沙頭角的一部分。

==地名來源==
相傳沙頭角得名於一名清朝大臣，當時他巡視沙頭角一帶時，面對[[大鵬灣_(南海)|大鵬灣]]的優美風光，便題了兩句詩：「日出'''沙頭'''，月懸海'''角'''」，是故得名。

==歷史==
[[File:Map_of_The_Convention_for_the_Extension_of_Hong_Kong_Territory_in_1898_-_2.jpg|thumb]]》附圖中的沙頭角（圖右方「Sha-tau-kok」）]]

深圳的沙頭角地區曾經出土新石器文物，但是香港的沙頭角地區則没有相關的發現。

自[[清|清]]初[[復界|復界]]後，大量人士遷移至沙頭角一帶定居，由於復界後沿海[[治安|治安]]不靖，當地鄉村於是組織約，以守望相助，沙頭角有10約，其中兩約位於今日[[中國|中國]][[廣東省|廣東省]][[深圳市|深圳市]]範圍內（其餘8約於[[香港特別行政區|香港特別行政區]]範圍內），[[慶春約|慶春約]]為著名的其中一約。

自1950年代起，沙頭角被香港政府列作[[香港邊境禁區|香港邊境禁區]]，加上[[第二次世界大戰|第二次世界大戰]]後鄉郊生活式微，沙頭角人口數十年來一直下降，通常居住人口變得愈來愈少。相反在[[中國大陸|中國大陸]]一邊，由於改革開放，貼近香港的沙頭角反而吸引到較多人流。

1967年7月8日，為[[六七暴動|六七暴動]]一部份的[[沙頭角槍戰|沙頭角槍戰]]發生，事件以共6死20傷結束，香港政府其後宣佈邊境全面封閉。

1997年[[香港回歸|香港回歸]]後，愈來愈多意見要求解封香港邊境禁區，沙頭角是要求被解封的其中一個重點，好讓生態[[旅遊|旅遊]]等項目可以發展。[[香港政府|香港政府]][[保安局|保安局]]於2006年9月7日表示將會大幅度地縮減香港邊境禁區範圍，但是沙頭角[[中英街|中英街]]不擬開放，表示是基於[[保安|保安]]理由的考慮。其後，香港政府於2012年2月15日凌晨零時起，開放了沙頭角墟口前的香港禁區範圍給[[香港市民|香港市民]]自由進出。當天由[[新界鄉議局|新界鄉議局]][[主席|主席]][[劉皇發|劉皇發]]和時任[[政務司司長|政務司司長]][[唐英年|唐英年]]主持開放儀式。石涌凹檢查站因此隨即正式撤銷，邊境檢查站向北移至沙頭角墟口繼續運作。

==地理==
[[File:HK_Frontier_Closed_Area.JPG|thumb]] 
沙頭角大致可以分為十個約，當中大部份位於香港邊境，另外有兩個約位於華界。現時，沙頭角基本上可以看成三部份的組合：
# 香港的沙頭角部份，包括以下地方：[[中英街|中英街]]、[[沙頭角村|沙頭角村]]、[[週田村|週田村]]、[[山咀村|山咀村]]、[[担水坑村|担水坑村]]等
# 深圳的[[沙頭角街道|沙頭角街道]]：[[大鵬灣_(南海)|大鵬灣]]、[[梧桐山|梧桐山]]、[[紅花嶺|紅花嶺]]、[[沙栏下|沙栏下]]，分為四個社區（[[沙頭角社區|沙頭角社區]]、[[田心社區_(沙頭角)|田心社區]]、[[橋東社區|橋東社區]]、[[中英街社區|中英街社區]]）<ref name=xzqh>{{cite web |url=http://www.yantian.gov.cn/cn/zjyt/csyt/xzqh/ |title= 行政区划 |accessdate=2013-03-26 |language=zh-hans }}</ref>，另外還包括[[東和社區|東和社區]]
# 深圳的[[海山街道|海山街道]]

沙頭角街道及海山街道在過去合稱為[[沙頭角鎮|沙頭角鎮]]，但現時已取消，並分拆為為兩個街道。

香港的沙頭角在廣義上還包括[[新界|新界]]東北部分[[打鼓嶺|打鼓嶺]]以東的大片地區，包括[[蓮麻坑|蓮麻坑]]、[[紅花嶺|紅花嶺]]、[[萬屋邊|萬屋邊]]、[[禾坑|禾坑]]、[[石涌凹|石涌凹]]、[[南涌|南涌]]、[[鹿頸|鹿頸]]、[[沙頭角海|沙頭角海]]等地，[[船灣|船灣]]和[[赤門_(香港)|赤門海峽]]以北、[[新娘潭路|新娘潭路]]以東的整個半島在傳統上也被稱爲[[沙頭角半島|沙頭角半島]]。

==合辦農莊==
沙頭角有一個有機本土種植的合辦農莊，主要以種植葡萄為主。 而現今位於沙頭角中的位置，以前是屬於禁區，而截至2018年仍未被地產商與政府看中<ref>[https://bkb.mpweekly.com/uncategorized/20180816-80600?v=false 【落區】沙頭角有個葡萄園：地理老師退休務農 種好香港]</ref>。

== 出入境 ==
沙頭角亦是香港和中國大陸的出入境管制站之一<ref>{{cite web |url=http://www.immd.gov.hk/chtml/immdctlpts.htm |title=存档副本 |accessdate=2007-09-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20070914184246/http://www.immd.gov.hk/chtml/immdctlpts.htm |archivedate=2007-09-14 }}</ref>，佔1997年跨界流量的 8%<ref>{{cite web |url=http://www.pland.gov.hk/p_study/comp_s/cbl/CBLstudy_stage1_chin/ch02_c.htm |title=存档副本 |accessdate=2008-01-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20080124071941/http://www.pland.gov.hk/p_study/comp_s/cbl/CBLstudy_stage1_chin/ch02_c.htm |archivedate=2008-01-24 }}</ref>。沙頭角出入境口岸適合前往[[大梅沙|大]]、[[小梅沙|小梅沙]]、[[鹽田|鹽田]]、沙頭角、[[南澳街道|南澳]]等地旅客使用。

==未來發展==
{{HK_future_infrastructure}}

2012年2月15日香港時間零時零分開始，[[香港邊境禁區|香港邊境禁區]]範圍再次縮減，沙頭角的6條鄉村獲得解放，其中[[塘肚村|塘肚村]]、[[新村|新村]]及[[木棉頭村|木棉頭村]]早就已經洞悉所帶來的商業機會，陸續興建旅遊景點。居民計劃分階段將42[[公頃|公頃]]土地開闢作[[有機|有機]][[農莊|農莊]]<ref>[https://archive.is/20120711145358/hk.news.yahoo.com/%E4%B8%8D%E6%83%B3%E8%AE%8A%E6%AB%83%E5%A0%B4-%E8%BE%B2%E8%8E%8A%E9%A4%8A%E6%B7%A8%E5%9C%9F-213326087.html 不想變櫃場 農莊養淨土] 《明報》 2012年1月30日</ref>、[[動物園|動物園]]〈飼養了[[黑草羊|黑草羊]]、[[兔|兔]]、[[牛|牛]]及[[鴕鳥|鴕鳥]]等〉，又挖掘了多個[[魚塘|魚塘]]有機飼養[[鯇魚|鯇魚]]，及劃地提供[[野戰|野戰]]區域，當中包括[[氣槍|氣槍]]靶場及9種野戰場景，場內放置了[[吉普車|吉普車]]、[[直升機|直升機]]及[[火箭|火箭]][[模型|模型]]等。全部項目合共投資額達1億港元，預料每逢假日可以接待逾千名遊人。此外，居民亦有開發更多香港本地的有機耕種產品，已經落實的包括佔地20公頃的[[稻田|稻田]]、每月可以生產30公噸[[菌類|菌類]][[食品|食品]]的[[蘑菇|蘑菇]]種植場地及佔地5公頃的[[蘭花|蘭花]]館，可以供應香港酒店。沿海岸方面，居民會闢設[[海星|海星]]館、沿海設立[[蠔排|蠔排]]及飼養[[龍蝦|龍蝦]]及[[珍珠蚌|珍珠蚌]]等，長遠計劃申請興建[[民宿|民宿]]及[[酒店|酒店]]等配套設施，目標為開拓全香港最大的生態主題公園。負責人表示，未來場地會逐步擴展至近海岸線，共逾100公頃土地，建造一處具有[[海|海]]、[[陸|陸]]、[[空|空]]主題的休閒旅遊景點；場內亦可以提供700至1000個就業職位，令到區內居民不用再長途跋涉到區外工作，可以節省時間及交通費用。

沙頭角區鄉事委員會主席[[李冠洪|李冠洪]]表示，將會加強[[宣傳|宣傳]]開放禁區後沙頭角的[[旅遊|旅遊]]元素，例如有機農莊及區內的歷史建築物，期望可以吸引市民到沙頭角旅遊<ref>[https://archive.is/20120710074602/hk.news.yahoo.com/%E7%A6%81%E5%8D%803%E6%9D%91%E5%84%84%E5%85%83%E5%BB%BA%E8%BE%B2%E8%8E%8A%E8%BF%8E%E5%AE%A2-%E5%85%A7%E8%A8%AD%E9%87%8E%E6%88%B0%E5%A0%B4%E5%8B%95%E7%89%A9%E5%9C%92-%E5%A4%A7%E9%80%BE%E5%85%A9%E7%B6%AD%E5%9C%92-213326283.html 禁區3村億元建農莊迎客 內設野戰場動物園 大逾兩維園] 《明報》 2012年1月30日</ref>。

另外，[[深圳地鐵8號線|深圳地鐵8號線]]將以[[深圳地鐵2號線|2號線東延伸段]]名義服務[[鹽田區|鹽田區]]，當中會在深圳一方的沙頭角設立海山站及沙頭角站，使該處居民進出深圳市區交通更為方便。
