== 简介 ==
1973年，伊丽莎白出生于[[美国|美国]][[德克萨斯州|德克萨斯州]]的[[休斯顿|休斯顿]]，其父亲是[[黎巴嫩|黎巴嫩]]移民，<ref name=Askmen>{{cite web|url=http://www.aaiusa.org/arab-americans/23/famous-arab-americans|title=Shannon Elizabeth|accessdate=2008-06-28|publisher=www.aaiusa.org|deadurl=yes|archiveurl=https://web.archive.org/web/20060526173953/http://www.aaiusa.org/arab-americans/23/famous-arab-americans|archivedate=2006-05-26}}</ref> 
她的母亲母亲是[[英国|英国]]、[[爱尔兰|爱尔兰]]、[[德国|德国]]和[[切罗基人|切罗基人]]的血统。<ref>http://articles.orlandosentinel.com/2000-03-12/news/0003080244_1_titus-lange-bruce-abbott/2</ref>
她在[[贫民窟|贫民窟]]长大。<ref name="waco">{{cite web|author=Steven Russell |title=Shannon Elizabeth: First she heated up American Pie. Now the nubile newcomer is ready for the Hollywood main course. |work=Interviews |publisher=[[Maxim_(magazine)|Maxim Online]] |date=2000-02 |url=http://www.maximonline.com/articles/index.aspx?a_id=1554 |accessdate=2006-09-30 |archiveurl=https://web.archive.org/web/20060421112902/http://www.maximonline.com/articles/index.aspx?a_id=1554 |archivedate=2006-04-21 |deadurl=yes }}</ref> 
在[[高中|高中]]时，她感兴趣于[[网球|网球]]运动，甚至考虑过从事专业的网球生涯，伊丽莎白曾在开始从事电影事业前曾是个模特。<ref name="curls up">{{cite web|author = John Crook|title = Shannon Elizabeth Curls up with 'Cuts'|work = TV News|publisher = Zap2it|date = 2005-02-13|url = http://tv.zap2it.com/tveditorial/tve_main/1,1002,271%7C93276%7C1%7C,00.html|accessdate = 2006-09-30|deadurl = yes|archiveurl = https://web.archive.org/web/20060725054044/http://tv.zap2it.com/tveditorial/tve_main/1%2C1002%2C271%7C93276%7C1%7C%2C00.html|archivedate = 2006-07-25}}</ref>
