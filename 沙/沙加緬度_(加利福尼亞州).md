'''萨克拉门托'''（{{lang-en|Sacramento}}，又譯為'''-{zh-hant:薩克拉門托;zh-hans:沙加缅度;}-'''），是一個位於[[美國|美國]][[加利福尼亞州|加利福尼亞州]]北部、[[沙加緬度河|沙加緬度河]]流域上的城市，也是加利福尼亞州的首府，加州政府機關及議會、[[沙加緬度郡_(加利福尼亞州)|沙加緬度郡]]政府皆位於此。最初於1849年12月由John Sutter建立，今天也是加州在人口上的第五大城。沙加緬度最初是由Fort Sutter成長而來，而Fort Sutter也是由John Sutter於1839年所建立的。在19世紀中的掏金潮時，沙加緬度是一個重要的人口集散地，也是一個商業和農業中心、和[[運貨馬車隊|運貨馬車隊]]、[[驿站马车|驛站馬車]]、[[河輪|河輪]]、[[電報|電報]]、[[驛馬快遞|驛馬快遞]]、和[[第一橫貫大陸鐵路|第一橫貫大陸鐵路]]的末端站。

==名称==
Sacramento是來自[[西班牙語|西班牙語]]「[[聖禮|聖事]]」之意，是西班牙人最初抵達此區時給加州中央山谷和沙加緬度河起的名字，後來Fort Sutter發展成市鎮後也成為了市名。中文譯名“-{zh-hans:沙加缅度; zh-hant:沙加緬度;}-”源自19世紀粵籍華僑對「Sacramento」的音譯，至今仍是當地華人及華語媒體唯一使用、以及臺灣採用的中文名稱，但現時中國大陸和香港譯作“-{zh-hans:萨克拉门托; zh-hant:薩克拉門托;}-”。老一輩[[華僑|華僑]]也通常稱沙加緬度為「-{二埠}-」，相對於「大埠」指[[舊金山|舊金山]]、和「三埠」為[[斯托克顿_(加利福尼亚州)|斯托克顿]]，但近年新移民已較少用這名稱。

== 历史 ==
萨克拉门托原本是由瑞士移民建立的城堡，1848年1月马歇尔（James W. Marshall）在修建锯木厂时，偶然在当地的河流中发现了金子，而且纯度很高，消息传出后，震动欧美，各地的淘金者大量涌入加州，掀起了历史上著名的淘金狂潮。萨克拉门托也就从一个小城镇很快变成重要的经济交通枢纽，第一条横贯美国大陆的铁路终点站就建在此处。由于修建大陆铁路时有许多来自中国南方的劳工，随着铁路的修建，许多中国移民就在这里落脚。

在淘金热的带动下，萨克拉门托的商业和农业发展惊人，火车、马车、船只来往不歇，各种货物以及人员在此进进出出，川流不息，一派繁忙景象，萨克拉门托城也随之建立，成为加州最早的自治城市。1854年加州议会以及州政府搬迁到萨克拉门托后成为加州首府。

===氣候===
沙加緬度屬於[[地中海型氣候|地中海型氣候]]（[[柯本氣候分類法|柯本氣候分類法]] ''Csa''），其特徵為潮溼、溫和的冬天和乾燥、炎熱的夏天。雨季分佈於十月到四月。極少降雪，降雪最高紀錄為1888年1月4日在市中心的8.9公分，1900年後只有三次較顯著的降雪紀錄，最後一次為1976年<ref>{{cite web|url=http://www.capradio.org/articles/2011/02/24/forecasters-say-snow-possible-in-sacramento- |title=Forecasters Say Snow Possible In Sacramento |publisher=capradio.org |date=2011-02-24 |accessdate=2012-11-16}}</ref>。

根据1981至2010年在行政机场的数据，日最低气温低于或等于{{convert|0|C}}的平均日数为15天；日最高气温超过{{convert|30|C}}的日数年均有103天，{{convert|35|C}}以上的有38天。<ref name = "NWS Sacramento, CA"/>最冷月（12月）均温{{convert|46.2|F|1|disp=flip}}，极端最低气温{{convert|17|F|0|disp=flip}}（1932年12月11日）。<ref name = "NWS Sacramento, CA"/>最热月（7月）均温{{convert|75.2|F|1|disp=flip}}，极端最高气温{{convert|115|F|0|disp=flip}}（1961年6月15日）。<ref name = "NWS Sacramento, CA"/>年均降水量约{{convert|18.52|in|mm|disp=flip}}，年极端最少降水量为{{convert|5.81|in|mm|disp=flip}}（2013年），最多为{{convert|37.62|in|mm|disp=flip}}（1983年）。<ref name = "NWS Sacramento, CA"/>无霜期平均有299天（2月11日至12月6日）。<ref name = "NWS Sacramento, CA"/>

== 人文 ==
根據2000年的[[人口普查|人口普查]]，薩克拉門托的人口為40萬7018，市區內住有15萬4581戶人家，以及9萬1202個家庭。人口密度為每平方公里1617.4人。市區內共有16萬3957個居住單位，單位密度為每平方公里651.5單位。種族組成為[[美國人口普查種族定義|白人]]佔48.29%、[[美國人口普查種族定義|非裔黑人]]佔15.47%、[[美國人口普查種族定義|原住民]]佔1.30%、[[美國人口普查種族定義|亞裔]]佔16.62%、[[美國人口普查種族定義|太平洋島嶼裔]]佔0.95%、[[美國人口普查種族定義|其他族裔]]佔10.96%、兩種以上族裔佔6.41%。[[拉丁裔美国人|拉丁美洲裔]]佔了人口的21.61%。沙加緬度的2007年估計人口已增加到46萬0242，為加州第二大內陸城市。

== 政治 ==
與多數美國城市一樣，沙加緬度市政府為[[議會-經理制政府|議會-經理制政府]]，市議會由[[市長|市長]]和8位市議員組成，市長為市議會主席，並由全市選出，市議員則分別由8個選區選出。現任市長為2016年就職的[[達雷爾·斯坦伯格|達雷爾·斯坦伯格]]。市議會決議和市政府運行則由[[市經理|市經理]]執行，現任市經理為2001年聘用的John F. Shirey。

華裔民主黨人、現任[[沙加緬度縣|沙加緬度縣]]郡監事（County Supervisor）[[余福慶|余福慶]]（Jimmie R. Yee）曾先後擔任沙市副市長和市長。此外，日裔民主黨人[[松井武男|松井武男]]自1979年起成為代表沙加緬度的[[美國眾議院|聯邦眾議員]]，並持續連任至2005年病逝；在同年的補選中，松井的遺孀、日裔民主黨人[[桃瑞絲·松井|桃瑞絲·松井]]（Doris Okada Matsui）以68.2%的選票贏得議席，也使她成為[[美國國會|美國國會]]史上第四十四名繼承夫志從政的遺孀。

== 經濟 ==
主要工業是高科技製造業、教育、電腦和[[農業|農業]]、[[畜牧業|畜牧業]]。

== 教育 ==
*[[沙加緬度州立大學|沙加緬度州立大學]]提供全面优质的本科与研究生教育。
*[[戴維斯加利福尼亞大學|戴維斯加利福尼亞大學]]的醫學院、護理學院和附屬的醫院都設在沙加緬度。
*[[沙加緬度市聯合學區|沙加緬度市聯合學區]]有学校。

== 體育 ==
* 籃球：[[沙加緬度國王隊|-{zh-hans:萨克拉门托国王队;zh-hk:薩克拉門托帝王隊;zh-tw:沙加緬度國王隊;}-]]，是[[國家籃球協會|國家籃球協會]]的主要球隊，自1948年加入NBA。只有在1951年奪得該年度NBA的總冠軍。主場為[[金州第一中心|金州第一中心]]，位於沙加緬度市北部。
* [[萨克拉门托君主|萨克拉门托君主]]是[[WNBA|WNBA]]的一支年轻的球队，成立于1997年，并于2005奪得WNBA的總冠軍，但已经于2009年解散。
