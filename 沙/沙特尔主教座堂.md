'''沙特尔圣母主教座堂'''（{{lang-fr|'''Cathédrale Notre-Dame de Chartres'''}}，又译'''夏特主教座堂'''、'''夏爾特主教座堂'''）位于[[法国|法国]][[巴黎|巴黎]]西南约70公里处的[[沙特尔|沙特尔]]市。据传[[圣母马利亚|圣母马利亚]]曾在此显灵，並保存了[[马利亚_(耶稣的母亲)|马利亚]]曾穿著的聖衣，沙特尔因此成为[[西欧|西欧]]重要的天主教圣母[[朝圣|朝圣]]地之一。1979年10月26日[[世界遗产委员会|世界遗产委员会]]第3屆會議起列入[[世界文化遗产|世界文化遗产]]。

== 历史 ==
沙特尔主教座堂最初以[[巴西利卡|巴西利卡]]式兴建，公元8世纪到12世纪，沙特尔教堂数次遭火灾。1020年9月7日的大火后，在[[福尔贝|福尔贝]]（{{lang|fr|Fulbert}}）[[主教|主教]]的领导下，有木屋顶的仿[[罗马式建筑|罗马式]]主教座堂在原址重建，取代原有的[[巴西利卡|巴西利卡]]式建築。1194年6月10日的晚上，一場大火摧殘了沙特尔主教座堂的大部分<ref>{{cite book |author = Neil Parkyn |title = 世界建築七十奇蹟 |year = 2004 |publisher = 猫头鹰出版社 |ISBN = 9867879686}}</ref>，但一位神職人員从废墟中找到了[[查理大帝|查理大帝]]取自[[君士坦丁堡|君士坦丁堡]]送給沙特爾主教座堂的[[聖母瑪麗亞|聖母瑪麗亞]]在[[耶稣降生|耶稣降生]]時所穿的衣服，激起了大家重建的信心与热情。1220年，新的主体建筑便已完工。

1594年，[[法国国王|法国国王]][[亨利四世_(法兰西)|亨利四世]]在此登基[[加冕|加冕]]。

== 建筑 ==
[[File:Monografie_de_la_Cathedrale_de_Chartres_-_10_Facade_Meridionale_-_Gravure.jpg|thumb]]
[[File:Chartres_Cathedral_Royal_portal_Central_Bay_Tympanum_2007_08_31.jpg|thumb]]
[[File:Chartres_-_cathédrale_-_rosace_nord.jpg|thumb]]

沙特尔主教座堂高114.6米（西端南塔较早建成，高约105米），面积5940平方米，西正门宽47.5米，最大[[玫瑰窗|玫瑰窗]]直径13.4米。是第一座完全成熟的哥特式主教座堂，法国[[哥特式建筑|哥特式建筑]]高峰时期的代表作，其建筑结构及平面配置成了以后各主教座堂模仿的蓝本。

西面大门有三个拱门，正面门楣上因有著耶穌基督的石雕，故以“王者之门”（Portal Royal）著称，是早期哥特式石雕艺术的经典。北面大門上雕有[[聖母|聖母]]和[[舊約聖經|舊約聖經]]中的人物，而南面翼殿大门的浮雕則描述了[[基督|基督]]的一生。因此沙特尔主教座堂被称为“石雕圣经教堂”。

另外拱門上有三个高窗，这象征着天主教“[[三位一体|三位一体]]”的理念。[[玫瑰窗|玫瑰窗]]是圣母玛丽亚的传统象征。其上方十二道细密的联拱，以及[[玫瑰窗|玫瑰窗]]里十二个小圆形与松子形镂空装饰，都是象征[[耶稣|耶稣]]的十二门徒。

西面的两座尖塔有着明显的差异，因为南塔于1145年-1170年建成，是晚期[[罗曼式建筑|罗曼式建筑]]向[[哥特式建筑|哥特式建筑]]过渡的风格，较为朴素；而北面的一座建成于1507年，有更多的雕刻，并更纤细，是典型的[[哥特式建筑|哥特式建筑]]风格。

教堂内共有170多扇[[花窗玻璃|花窗玻璃]]，當中完整保存了100多扇12世纪—13世纪的[[花窗玻璃|花窗玻璃]]，這此彩色玻璃窗以[[聖經|聖經]]故事為題材，繪畫了4000多個[[拜占庭|拜占庭]]風格的人物，形象鮮明，栩栩如生，其中部分北翼的[[玫瑰窗|玫瑰窗]]是[[路易九世|路易九世]]的母親[[布朗卡皇后|布朗卡皇后]]所捐助的。
