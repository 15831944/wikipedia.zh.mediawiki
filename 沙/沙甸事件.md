'''沙甸事件'''是[[文化大革命|文化大革命]]时期于1975年7月29日至8月4日期间发生在[[中国|中国]][[云南省|云南省]][[红河哈尼族彝族自治州|红河哈尼族彝族自治州]][[个旧市|个旧市]][[沙甸鎮|鸡街公社沙甸大队]]及其附近[[回民|回民]]村落的村民的动乱事件。

《云南文化大革命运动大事纪实》在“大事件”篇中记录：1974年红河州革委会成立了针对回民的‘鸡街民兵指挥部’，沙甸百姓也成立‘回民兵团指挥部’。双方剑拔弩张，持枪火拼，抢劫军械，导致政府判断此为叛乱事件，引发沙甸事件”<ref name="炎黄春秋">{{cite web|title=炎黄春秋|url= http://www.yhcqw.com/30/1928.html}}</ref>。《云南文化大革命运动大事纪实》指出为王洪文下令镇压<ref name="云南文化大革命运动大事纪实">{{cite web|title=云南文化大革命运动大事纪实|url=https://books.google.com/books/about/%E6%96%87%E9%9D%A9%E5%8F%8D%E6%80%9D%E5%9B%9E%E5%BF%86%E5%8F%B2%E6%96%99%E4%B9%8B%E5%85%AB.html?id=ZNx_AQAACAAJ}}</ref>。另有一说据《经济学人》杂志在一篇名为 Deng Xiaoping’s legacy: The great stabiliser 的文章中称，是邓小平下令镇压，镇压最终导致1600人死亡，其中包括300名儿童<ref name="economist">{{cite web|title=Deng Xiaoping’s legacy The great stabiliser|url=http://www.economist.com/node/21533354|publisher=经济学人|accessdate=2018-02-06}}</ref>。

1979年2月，中共十一届三中全会后，中共云南省委和昆明军区党委经中央批准联合发出通知为“沙甸事件”平反，检讨错误，并彻底推翻之前的结论[5]，并释放被捕的数名穆斯林民众[6]。通知称，责任在于当时中共云南省委主要领导人。1987年8月，中共云南省委发出补充通知，“沙甸事件”彻底平反。

==历史背景==
{{see also|文化大革命}}

在中华人民共和国在文化大革命初期，以大中学生红卫兵为主力进行的[[破四旧|破四旧]]，立四新，即“破除旧思想、旧文化、旧风俗、旧习惯，树立新思想、新文化、新风俗、新习惯”相标榜的社会运动，其伴随着红卫兵运动的兴起而席卷了中国大陆。

沙甸位于[[开远县|开远县]]、[[个旧县|个旧县]]和[[蒙自县|蒙自县]]的中心交汇点，地理位置极其重要。文革期间，全国[[武斗|武斗]]之风胜行，对[[中华人民共和国宗教史|宗教亦实行高压政策]]。包括[[中华人民共和国伊斯兰教|伊斯兰教]]在内的[[中华人民共和国佛教|佛教]]、[[中华人民共和国基督教|基督教]]、[[道教|道教]]等诸教[[圣职者|圣职者]]及[[信徒|信徒]]皆受[[批斗|批斗]]，[[宗教场所|宗教场所]]受到冲击。[[红卫兵|红卫兵]]禁止回民参加[[礼拜|礼拜]]，查抄焚毁伊斯兰教典籍，批斗[[阿訇|阿訇]]及[[穆斯林|穆斯林]]。

1968年3月，“[[云南省革命委员会|云南省革命委员会]]”成立，对民众大肆抓捕、[[揪鬥|揪鬥]]。被迫害民众逃往沙甸，造反派称沙甸是“反革命基地”，并宣称要武力清扫。<ref name="chinashadian">[http://www.chinashadian.com/Article/ShowArticle.asp?ArticleID=311 震惊中外的“沙甸事件”]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }},云南沙甸网</ref>同年12月，“云南省革命委员会一、二号文件宣传队”（支左部队）进村，大肆搜查外地民众，进行残酷批斗。并驻扎于沙甸大[[清真寺|清真寺]]，“宣传队”在清真寺中吃猪肉，猪骨头丢在水井，宣称“信教就是反对马列主义、毛泽东思想，就是反对党的领导”。

1971年[[九一三事件|“九一三”事件]]后，沙甸一些民众上书要求平反并开放清真寺，未得回应。1973年10月，回民自行打开被关闭的清真寺，并遭革委会派兵武装阻挠。消息传出后红河州和蒙自县的革委会立即派出由部队和地方官员组成的近百人的工作队进驻沙甸搞“反复辟”宣传，劝说民众理性信教。

1974年4月“批林批孔”运动中，工作队宣传“批林批孔要结合批清真寺”，并点了沙甸代表马伯华、马绍美等人的名，他们开始进京上访。5月14日，中共中央提出由时任省委第一书记、省革委主任的[[刘维新|周兴]]负责落实此事。周兴在草草调查后，向中央报告：“沙甸问题已经解决。”9月云南省委还发出通知规定：“已经关闭或改作他用的清真寺不得再打开作宗教活动场所，已经强行打开的清真寺，要在作好工作的基础上，由群众自行关闭。<ref name="chinashadian" />

== 事件经过 ==
1975年3月，[[中共云南省委|中共云南省委]]向[[中共中央|中共中央]]请求采取军事手段镇压。

1975年7月29日凌晨三時，开始武装镇压沙甸村民（打土围子）。在对沙甸之军事包围部署完毕后，一支先遣侦察部队开始行动。他们在熟悉沙甸地形和道路的嚮導指引下，先除去村边站岗的沙甸[[民兵|民兵]]，然后进入大清真寺后院——沙甸民兵团的办公地点，欲捉沙甸叛乱者的头目。不料头目们当夜皆不在現場，有駐寺人员发现武装军人深夜进入清真寺，登上叫拜楼三楼敲钟报警。

凌晨四時，沙甸村民凭借地形熟悉，利用自制武器向占领大清真寺的部队进行反攻。经过激烈的交火，沙甸村民夺回了大[[清真寺|清真寺]]，并从部队手中抢夺一些现代化武器。这时西营已被部队占领，后山、马家井、林家巷、金鸡寨、川方寨的部分生产队场院和民房也被部队控制。当晚，沙甸村民展开街战、巷战和以房屋为据点的攻守战。

同日下午，沙甸反抗者被围困於金鸡寨。部队决定對群众勸降，让他们交出武器自行出村。兩千多名老、弱、妇、幼、伤、残、病者于31日上午集队出村。出村后所有有问题的人被关起来审查，其他的人被送到沙甸附近的鸡街铅厂、江水地砖瓦厂等地參加強制講習。

8月1、2日两昼夜，战斗集中在争夺金鸡寨、川方寨的民房和制高点。战斗时紧时松，但从未间断过。8月3日，金鸡寨已被部队控制了一半左右，之後進入短兵相接狀態。当时沙甸约有7700人参与叛乱，这场战斗进行了七天八夜，击毙“沙甸村民”900多人，伤、残600多人。而解放軍也有一百多人之伤亡。马伯华等沙甸村民的核心成员共10人，其中有3人被俘，其余都被击毙。马伯华的尸体是在一段被炸毁的地道里找到的，在其周圍还有5位全副武装的年轻女子的尸体，都是死於152加农炮轟炸之下。

政府在事后檢討这次平叛战斗的情况，發現以下問題：
一是对敌情掌握不够，特别是对村庄内的立体式防御工事缺乏了解。
二是对沙甸的地下兵工厂的生产能力完全不清楚。
三是轻视了反叛势力武装集团的战斗力。
四是對一般群众仇視解放軍的情況估计分析不足，造成了不必要的伤亡。
五是没預想到叛軍集团对武裝抵抗之准备充分情況。以至於原计划中僅3小时的作战计划，最後居然打了7天8夜。

参加“沙甸事件”遭到同样武装镇压的还有开远市的新寨，砚山县的车白泥、田心、茂龙、松毛坡，文山县的茂克等回民村寨。整个事件中回教民众死亡1600多人，伤、残近1000人。其中开远市新寨的伤亡比例最高，超过了沙甸。战斗结束之后，省州县(云南省、红河州、蒙自县)沙甸工作组的“武装平叛沙甸叛乱”宣传提纲即30条指出：“沙甸等地回民村武装叛乱，妄图建立[[伊斯兰共和国|伊斯兰共和国]]，沙甸成立‘罕指奔拉协’，派人与[[苏修|苏修]]联系，妄图背叛祖国，反对中央领导同志，打砸抢、挑起武斗，反对省委、反对新生红色[[政权|政权]]”等等。

省州县沙甸工作组还举办沙甸武装叛乱罪行展览。该展览先在鸡街、[[个旧|个旧]]展出。而在“沙甸事件”后，因“沙甸事件”而判死刑和重刑的多达数十人，接受強制講習的也多達数百人。

== 平反 ==
1979年2月，[[中共十一届三中全会|中共十一届三中全会]]后，[[中共云南省委|中共云南省委]]和[[昆明军区|昆明军区]]党委经中央批准联合发出通知为“沙甸事件”平反，检讨错误，并彻底推翻之前的结论<ref>原定（以沙甸为中心的反革命武装叛乱）的结论应予撤销，这个事件中涉及的广大穆斯林干部群众应予平反。”[http://www.musilin.net.cn/2010/0122/33562.html 云南省沙甸事件始末]  {{webarchive|url=https://web.archive.org/web/20150204083247/http://www.musilin.net.cn/2010/0122/33562.html |date=2015-02-04 }}</ref>，并释放被捕的数名穆斯林民众<ref>[http://www.sufi.org.cn/Photo/GoodPicture/200512/7.html 1979年沙甸事件平反出狱留影] {{webarchive|url=https://web.archive.org/web/20121124113418/http://www.sufi.org.cn/Photo/GoodPicture/200512/7.html |date=2012-11-24 }}</ref>。通知称，沙甸事件是地方推行[[林彪|林彪]]、“[[四人帮|四人帮]]”的[[反革命|反革命]][[修正主义|修正主义]]路线的结果，酿成沙甸事件的罪魁祸首是林彪、“四人帮”，要把千仇万恨都集中在林彪、“四人帮”身上。通知还称，中央的指示是正确的，责任在于当时[[中共云南省委|中共云南省委]]主要领导人。

1980年，[[沙甸大清真寺|沙甸大清真寺]]得以重建。

1987年8月，[[中共云南省委|中共云南省委]]发出补充通知，“沙甸事件”彻底平反。官方在宣传中称，沙甸回民称“沙甸事件”的平反为沙甸的第二次解放。

现当地建有[[沙希德|舍西德]]{{NoteTag|舍西德(Shahid)，伊斯兰教对所有为[[安拉|安拉]]之[[主道|道]]而献出生命的穆斯林的尊称。阿拉伯语音译，意为“[[殉教|殉教]]者”、“[[烈士|烈士]]”。}}纪念碑，碑文亦将责任归之为极左派，并称平反后沙甸“各项事业兴旺，民族和睦，军民团结，政通人和，国泰民安”。

如今沙甸地区号称“东方小[[麦加|麦加]]”，有指沙甸已形成“[[政教合一|政教合一]]”的社会形态，清真寺替代地方政府掌握政治权力，在地区内推行[[伊斯兰化|伊斯兰化]]政策，如全面[[禁酒|禁酒]]、[[宗教教育|宗教教育]]、大量女性穿戴[[面纱|面纱]]等。由于[[2014年昆明火车站暴力恐怖袭击事件|昆明3·01恐怖袭击事件]]的策划者在沙甸被捕，有人称沙甸为中国的“[[伊斯兰国|伊斯兰国]]”、“恐怖分子的庇护所”，部分网民也开始对沙甸事件中军方的镇压行动表示肯定，将沙甸事件视为一场阴谋分裂国家的叛乱。当地伊斯兰教[[圣职人员|圣职人员]]则认为沙甸受到误解，同时表示沙甸宗教活动符合中国法律，清真寺支持中国政府<ref>{{cite web |url = https://www.thestandnews.com/china/%E4%B8%AD%E5%9C%8B%E7%9A%84-%E4%BC%8A%E6%96%AF%E8%98%AD%E5%9C%8B-%E5%A4%9A%E9%9D%A2%E6%B2%99%E7%94%B8/ |title = 中國的「伊斯蘭國」？ 多面沙甸|publisher=立场新闻 |author=劉燁 |accessdate=2015-09-22 |date=2015-08-21}}</ref><ref>{{cite web |url = http://history.m4.cn/2014-01/1222719.shtml |title=鲜为人知的云南平远街叛乱 |publisher=四月网 |accessdate=2015-09-22|date=2014-01-14}}</ref>。
