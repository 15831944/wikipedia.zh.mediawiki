'''沙月永遠'''（{{lang-ja|沙月とわ}}，1998年1月18日 - ），[[日本|日本]]的[[AV女優|AV女優]]<ref name="GREEN"/>，[[新潟縣|新潟縣]]出身<ref name="K.M.Produce">{{Cite web |url=http://www.km-produce.com/towa_satsuki |title=沙月とわの公式ページ |publisher=[[KM_Produce|K.M.Produce]] |accessdate=2018-11-10 |language=ja}}</ref>。目前為「[[KM_Produce|K.M.Produce]]」專屬女優<ref name="K.M.Produce"/>，所屬於「{{link-jp|GREEN|ハスラー (芸能事務所)}}」事務所<ref name="GREEN"/>。

== 人物 ==
*興趣是看[[動漫|動漫]]、玩[[遊戲|遊戲]]，擅長[[游泳|游泳]]、[[古典芭蕾|古典芭蕾]]舞<ref name="GREEN"/>。

*從[[小學|小學]]的時候開始[[自慰|自慰]]，初體驗的對象是18歲時在[[迴轉壽司|迴轉壽司]][[兼職|打工]]一位9歲的常客<ref>{{Cite web |url=https://news.dmm.co.jp/article/998867/?page=all |title=新人AV女優･沙月とわ暴走！清純そうに見えてガチオタなGカップは緊縛＆ぶっかけに興味あり❤ごっくんも平気？▼デビュー作での無意識発言「お尻なんて舐められた事ないのに！」→ガンダム好きなのが全世界に発覚。【KMP専属AVデビュー記念ロングインタビュー第2回】 |work=FANZAニュース |publisher=DMM |date=2018-03-21 |accessdate=2018-10-04 |language=ja}}</ref>。

*原本就很喜歡也憧憬AV女優，因為自己喜歡的女優[[佐倉絆|佐倉絆]]、[[成瀨心美|成瀨心美]]與「[[KM_Produce|K.M.Produce]]」有深深的相關，便自己申請加入該公司，並特別為她舉行試鏡<ref>{{Cite web |url=https://xcity.jp/idol/interview/detail/?program_id=117880 |title=【前編】オーディションでグランプリを勝ち取ったアノ子がついに……ミリオン専属女優「沙月とわ」ちゃんデビュー！ 2018年2月9日に、衝撃の素人美女がミリオン専属女優としてデビューする……その名も沙月とわちゃん。自らオーディションに応募し、憧れの"AV女優"となった現在の心境を語ってくれました。【沙月とわ 新人AV女優インタビュー】 |publisher=XCITY |date=2018-02-09 |accessdate=2018-10-04 |language=ja}}</ref>。

*AV出道之前只與初體驗的對象發生過關係，這也是在出道作上標示她1年沒有做愛的理由<ref>{{Cite web |url=https://news.dmm.co.jp/article/998813/?page=all |title=人気爆発AV女優･沙月とわがドスケベ本領発揮！それ以上にガチな本性も…東風克智ボッキ歓喜。剛毛だった思春期からパイパンにした理由は？▼デビュー作を見ながら振り返りトークで赤面！【KMP専属AVデビュー記念ロングインタビュー第1回】 |work=FANZAニュース |publisher=[[DMM.com|DMM]] |date=2018-03-14 |accessdate=2018-10-04 |language=ja}}</ref>。

== 簡歷 ==
2017年，她以[[日語|日語]][[假名|假名]]「ちひろ」參加「[[KM_Produce|K.M.Produce]]」所舉辦的《KMP Dream Girl Audition》2017活動，並獲得星光大道的冠軍<ref>{{cite web|url=http://www.playno1.com/article-25654-1.html#.W-YIppMzaUk |title=票選優勝！美顏巨乳！星光大道冠軍！KMP集團的救星正式現身！ ...  |publisher=AVNo.1-PLAYNO.1 |date=2017-12-27}}</ref>。

2018年2月，以「'''沙月とわ'''」名義成為[[標籤|品牌]]「million」的專屬女優[[AV|AV]][[出道|出道]]<ref>{{cite web|url=http://www.playno1.com/article-26133-1.html#.W-YE4JMzaUk |title=年初最神秘強者？ 沙月とわ完全解剖！ |publisher=AVNo.1-PLAYNO.1 |date=2018-02-12}}</ref><ref>{{Youtube|icm9eD8E8ZA|☆新人Debut☆沙月とわ SATSUKI TOWA official PV / million by kmp}}</ref>。
