'''沙河站'''位于[[北京市|北京市]][[昌平区|昌平区]]，是[[北京地铁|北京地铁]][[北京地铁昌平线|昌平线]]的一座车站。本车站于2010年12月30日投入运营<ref>{{cite web|url=http://zhengwu.beijing.gov.cn/bmfu/bmts/t1147349.htm|title=五条地铁新线最小发车间隔五分钟|author=刘冕 刘欢|work=北京日报|date=2010-12-31|accessdate=2011-01-08}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 概述 ==
沙河站位于北京市昌平区沙河地区，百沙路与南丰路丁字路口南侧。车站呈大致偏向西北–东南向布置。

== 结构 ==
车站为高架车站，[[岛式站台|岛式站台]]设计。站台中部有2组通向站厅的扶梯<ref name="zhandian">{{cite web|url=http://www.bjsubway.com/station/xltcx/linecp/2013-08-26/249.html|title=站点及周边信息：昌平线沙河站|publisher=北京地铁|date=2013-08-26|accessdate=2016-01-13}}</ref>。

== 出入口 ==

车站共有2个出入口：A口通向站厅西侧，B口通向东侧。每个出口分别布置了直梯（A1、B2）和扶梯（A2、B1）<ref name="zhandian"></ref>。
