'''沙基惨案'''发生于1925年6月23日，又稱'''六二三事件'''，[[大英帝国|英国]]士兵开枪[[镇压|镇压]][[广州|广州]]共产国际领导下的[[周恩來|周恩來]]策划的示威和罢工行动，造成严重伤亡的事件。共产国际的计划也随之失败。

== 背景 ==
1925年5月，[[大英帝国|英]][[大日本帝国|日]]镇压在[[上海|上海]]游行工人引发的[[五卅惨案|五卅惨案]]。6月，广州和[[香港|香港]]工人为声援上海工人发动的[[省港大罢工|省港大罢工]]。

== 经过 ==
6月21日，为声援上海的罢工工人，香港、广州工人也进行罢工，工人纷纷离职返回广州。23日，10多万工人、商人、学生、以及持槍的黃埔軍校學生在[[东较场|东较场]]举行集会，要求打倒[[帝国主义|帝国主义]]、废除[[不平等条约|不平等条约]]，并游行到[[沙面|沙面]]租界对岸的[[沙基|沙基]]。下午3时，当游行队伍行经沙基西桥口时，雙方發生衝突。有學生声称[[法蘭西殖民帝國|法國]]兵首先開槍。<ref>《[[華僑日報|華僑日報]]》，民14年6月25日</ref>也有说沙面[[维多利酒店|维多利酒店]]（今[[胜利宾馆|胜利宾馆]]）楼上有外国人用手枪向游行群众射击。而据北京美国公使向美京的机密报告，一个美国人亲目看到苏俄军官指挥黄埔学生军先行向沙面的英，法守军开枪<ref>美国外交部档案USDS893-6554，1925年7月25日，北京美国公使</ref>。

沙面内的英军、法军士兵听到枪声后，从沙面向游行群众开机枪扫射，驻扎在[[白鹅潭|白鹅潭]](今[[白天鹅宾馆|白天鹅宾馆]])的英国军舰同时向北岸开炮。游行队伍毫无防备，四散躲避，当场死亡五十多人，包括东征军一團三營營長，共产党员曹石泉。周恩來僅以身免，身邊的衛士都被打死。一百七十多人重伤，轻伤者不计其数。至10月3日为六二三沙基烈士举行[[国葬|国葬]]仪式时，死亡出殡人数为61人。死者中包括有13岁儿童1人，学生4人，教师1人，商民9人，工人6人，以及[[黄埔军校|黄埔军校]]师生27人（另有兩名俄羅斯人死亡）。廣州[[嶺南大學_(廣州)|私立嶺南大學]]之學運領袖曹耀亦受槍伤。

中国历史学者刘仲敬认为，沙基示威队伍由周恩來率領的[[国民革命军东征|東征軍]]兩個營、學生軍一個營为主力，以“革命群眾”的名義尝试攻下沙面英租界，是一种人肉盾牌策略。如果英國人镇压，就会被宣传为帝國主義屠殺無辜平民。如果英國人选择不镇压，租界就会被占领，而国民政府無須承擔撕毀國際條約的罪名<ref>[https://medium.com/@viewswise/%E5%91%A8%E6%81%A9%E4%BE%86-627cae5e2e37 劉仲敬 | 周恩來 民國及近現代人物縱橫談]</ref>。

== 后续 ==
惨案发生后，[[國民政府#廣州（1925年7月1日—1926年12月5日）|廣東國民政府]]立刻以广东省长[[胡汉民|胡汉民]]的名义向英国和法国驻广州总领事馆提出抗议，26日又再次提交抗议，提出各国派大员向广东革命政府谢罪、惩办相关军事长官、各国军舰撤离、沙面租界交由广东革命政府接管、赔偿死伤人员以抚恤金等要求。但遭到驻粤英、法领事的拒绝。29日，广州各团体提出对英实施经济绝交。同日，香港25万工人开始全面总罢工，并有13万人撤回广州。7月11日，广州市民公祭死亡的烈士。次年，在沙基修成马路，命名为六月二三马路，后改名为[[六二三路|六二三路]]。

英国的对华政策主要目在保护与华通商和香港的利益。在1925年，中国各地反英情绪达到最高潮，英国认为其起端，如上海五卅惨案，广州沙基惨案等事件，都为苏俄阴谋在后煽动而发生的。英国自此不愿支援中国“温和派”，而增加中国“激进派”对英的恶感<ref>英国外交部档案FO371-10949，页79，1925年9月24日，北京英公使密电伦敦外交部</ref>。

== 達姆彈的使用 ==
据当时亲历惨案者回忆，受伤与死亡的中国人傷口极大，难以救治。有传闻指当时英、法军队使用了[[達姆彈|達姆彈]]。<ref>[http://www.gzzxws.gov.cn/gzws/gzws/ml/35/200809/t20080916_7931_1.htm 张采庵，“沙基惨案”亲历见闻，《广州文史》第三十五辑] {{webarchive|url=https://web.archive.org/web/20150418070622/http://www.gzzxws.gov.cn/gzws/gzws/ml/35/200809/t20080916_7931_1.htm |date=2015-04-18 }}</ref>

== 纪念碑历史 ==
{{main|沙基惨案纪念碑}}
1926年1月，广州市政府将沙基马路改名为六二三路，并在[[沙面|沙面]]东桥头六二三路上竖立“'''毋忘此日'''”纪念碑，同年6月23日，举行路名和纪念碑落成典礼。1949年[[中国共产党|中共]]建政后，1950年因东桥扩建，将原碑拆毀，迁移东桥脚，改建一座高大的纪念碑立于桥头，改名为'''沙基惨案烈士纪念碑'''。现时的纪念碑为2001年10月18日重新修建而成。<ref>[http://news.sina.com.cn/s/2001-10-29/387672.html 广州沙基惨案纪念碑小了？新立的碑引来争议]</ref>。
