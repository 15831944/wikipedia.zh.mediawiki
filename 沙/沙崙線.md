{{沙崙線RDT}}
'''沙崙線'''，為[[臺灣鐵路管理局|臺灣鐵路管理局]]之客運支線，定位為[[高鐵臺南站|高鐵臺南站]]的連絡線，2011年1月2日正式通車<ref>[http://www.railway.gov.tw/News/News_Content.aspx?NTC=H&NewsSN=2507 台灣鐵路管理局-沙崙支線100年1月2日通車營運]</ref>。沙崙線開通後，大幅減少高鐵車站至市中心旅行時間（公路約需40－50分鐘、鐵路約23分鐘），沙崙線亦為自1992年[[南迴線|南迴線]]開通以來，臺鐵再次新築路線。

== 路線資料 ==
*經營管轄：[[臺灣鐵路管理局|臺灣鐵路管理局]]
*路線距離：中洲－沙崙間5.3公里
*[[軌距|軌距]]：1,067毫米
*車站數：3（含起訖車站）
*開業時間：2011年1月2日全線開通
*雙線區間：全線
*[[電氣化鐵路|電化]]區間：全線（[[交流|交流]]25KV）
*除中洲端部份採路堤外，全線高架無平交道
*營運時間：06:00至24:00
*營運班距：配合高鐵時刻，約20至40分一班

行駛列車多經過臺南＝沙崙區間，使得單向平均每30分鐘即有一班。而在2017年4月27日[[善化車站|善化車站]]第二月台落成前，每日有8班列車在善化折返時，不辦理客運（載客區間僅至南科）。

== 使用車輛 ==
[[File:TRA_EMU612_on_Shalun_Line_20101207.jpg|thumb]]
沙崙線興建計畫中，有編列[[EMU800|EMU800]]之採購預算；惟最後決定以[[EMU600|EMU600]]五組（共20輛）營運<ref>[http://udn.com/NEWS/LIFE/BREAKINGNEWS9/5827030.shtml 沙崙線列車 加裝幕顯示高鐵車班]，聯合報</ref>。

營運初期，列車外觀為以臺南市風景為主題之彩繪，內部則配有智慧型列車服務資訊系統<ref>[http://www.railway.gov.tw/News/News_Content.aspx?NTC=H&NewsSN=2307 台鐵智慧型列車服務資訊　升值旅客服務品質]，臺灣鐵路管理局</ref>；然由於彩繪貼紙褪色、脫落，加上配合列車運用，後將其改回原塗裝。

目前運用列車以[[台鐵EMU600型電聯車|EMU600]]及[[EMU500|EMU500]]行駛其班次。

== 簡介 ==
[[File:Shalun_Line_0K.JPG|thumb]]
該線興建的目的，是為了解決[[臺南都會區|臺南都會區]]與高鐵站間的交通問題。最初台南至高鐵車站計畫興建[[臺南捷運|臺南捷運]]紅線，但由於政府財政上的困難，[[交通部高速鐵路工程局|交通部高速鐵路工程局]]辦理「民間機構參與興建暨營運台南都會區大眾捷運系統優先辦理路線可行性研究」，經[[行政院|行政院]]核示應優先考量高鐵站與台鐵站間最佳的銜接方式後，高速鐵路工程局提出興建「沙崙線」的構想，並進行初步設計，之後全案轉由[[交通部鐵路改建工程局|交通部鐵路改建工程局]]東部工程處負責後續工程。

沙崙線原定2009年竣工，並計畫配合2008年[[臺灣博覽會|臺灣博覽會]]（後未舉辦）提前完工。但因為[[二仁溪|二仁溪]]整治計畫的關係，於2010年竣工，並移轉給台鐵經營。2011年1月2日正式通車後，臺鐵闢駛經[[中洲車站|中洲]]直達[[臺南車站|臺南]]的區間車，距離約18km、需時廿分鐘，較[[高鐵快捷公車|高鐵快捷公車]]節省約一半時間。

工程經費方面：土木工程建設經費為[[新臺幣|新臺幣]]33.9億元（其中包括跨越[[中山高速公路|中山高速公路]]的困難工程，亦為臺鐵首條上跨高速公路的路線），增購電聯車16輛約為新臺幣8億元；[[中洲車站|中洲車站]]整體改建總經費為新臺幣3.7億元，南160號道路鐵路平交道立體化工程總經費新臺幣1.7億元，合計共需新臺幣47.3億元。而沙崙線除了全線高架，路線也採用無[[道碴|道碴]][[鐵軌|版式軌道]]。

[[2010年中華民國縣市改制直轄市|臺南市縣合併改制]]之後，2012年起[[台南心時代跨年晚會|跨年晚會]]均選在高鐵站區舉行，沙崙線也配合延長營運至凌晨2時。

== 電子票證 ==
目前沙崙線各站設有多卡通驗票閘門，可持[[一卡通_(台灣)|一卡通]]、[[悠遊卡|悠遊卡]]、[[icash|icash2.0]]、[[HappyCash|HappyCash]]搭乘。

== 沿線風景 ==
沙崙線於中洲車站新設第一月台A側及高架第三月台，出站後下行線以半徑200公尺之曲線轉東、上行線亦轉東但以17‰坡度爬升並越過縱貫線，稍後兩線會合掠過高速公路仁德服務區南緣。復續往東轉東北進入大潭，於長榮大學前方設站。出大潭後跨越[[臺南市區道列表|南160線]]，進入高鐵特定區後與[[台39線|台39線]]、高鐵路線平行。最後進入站區並止於高鐵車站南方，台鐵與高鐵站兩間有空廊相銜。
