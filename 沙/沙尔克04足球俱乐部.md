'''盖尔森基兴-沙尔克04足球俱乐部'''（[[德文|德文]]：'''{{lang|de|Fußball-Club Gelsenkirchen-Schalke 04}}'''，简称：'''{{lang|de|FC Schalke 04}}'''）是一家位于[[德国|德国]]西部[[北莱茵-威斯特法伦|北莱茵-威斯特法伦]]的[[盖尔森基兴|盖尔森基兴]]沙尔克地区的足球俱乐部，成立於1904年5月4日，目前在[[德甲|德国甲级足球联赛]]比賽，由于盖尔森基兴曾经是[[欧洲|欧洲]]采煤中心，因而被称为“矿工”。主场为可容納62,271人的[[維爾廷斯球場|維爾廷斯球場]]。

在1933年至1942年之间，沙尔克04闯进20次联赛和杯赛的14次决赛，即每年最少晋身一项决赛，取得6次联赛及一次杯赛冠军，是当時[[德国|德国]]的最佳球队。1963年[[德甲|德甲]]成立，沙尔克04是创始成员之一。

在2017年签约新赞助商凯发娱乐，成为亚洲区合作伙伴。

== 历史 ==
沙尔克04的前身是于1904年成立的'''威斯特法倫1904沙尔克俱乐部'''（{{lang|de|Sportclub Westfalia 1904 Schalke}}），队名的来历是一座叫做“沙尔克”的矿井。因為[[盖尔森基兴|盖尔森基兴]]是煤矿城市，所以成员都是煤矿工人。1924年改名为'''沙尔克04足球俱乐部'''（{{lang|de|FC Schalke 04}}）；5年后的1929年因获得盖尔森基兴市在财政协助兴建新球场，更名'''盖尔森基兴沙尔克04足球俱乐部'''（{{lang|de|FC Gelsenkirchen-Schalke 04}}）。

[[File:2010-06-04_Schalke_04-Geschäftsstelle_01.jpg|thumb]]
当时未成立[[德甲|德甲]]，联赛是[[德国足球冠军|德国锦标赛]]。沙尔克04在1934年首度奪得联赛冠军；1937年沙尔克04赢得了首个[[德国盃|德国盃]]冠军。此後至1958年，沙尔克04一共赢得了17次德国锦标赛冠军。

沙尔克04是1963年[[德甲|德甲]]成立时的创始成员，但由于实力所限，多年來沙尔克04都未曾赢得德甲联赛冠军。1972年[[德国盃|德国盃]]是沙尔克04当时唯一錦標。自1981年至1991年，沙尔克43次降级。至2001年和2002年兩赢[[德国盃|德国盃]]，沙尔克04才得以挽回颜面。
 
2009/2010赛季，沙尔克04获得[[德甲|德甲]]联赛亚军，这是该队近年来在德甲联赛的最佳战绩，也因此直接晋级[[2010/11赛季欧洲冠军联赛|2010/2011赛季]][[欧洲冠军联赛|欧洲冠军联赛]]小组赛。这一赛季的欧洲冠军联赛1/4决赛中，沙尔克04以两回合7比3的悬殊比分淘汰卫冕冠军[[国际米兰|国际米兰]]（客场5-2，主场2-1），首次晋级欧冠四强，然而好景不长，他们在随后的四强中被曼联淘汰。

== 魯爾區德比 ==
{{main|魯爾區德比}}
[[File:Arena_Auf_Schalke_hosting_Schalke_04_vs_Dortmund_in_2009.jpg|210px]]在德甲联赛在2009年。]]
位於[[德国|德国]]西部[[北莱茵-威斯特法伦|北莱茵-威斯特法伦]]境內的[[城市群|城市群]][[鲁尔区|鲁尔区]]內兩支主要的球隊，沙爾克04和[[多特蒙德足球俱樂部|多特蒙德]]之間的對賽被稱為「魯爾區德比」。截至2012年年底，沙爾克04以56勝、34和、47負稍為佔優。

== 沙爾克04二隊 ==
{{main|沙爾克04二隊}}

沙爾克04二隊是德甲聯賽上沙爾克04之下的預備隊，直到2005年之前，沙爾克04二隊都以沙爾克04業餘隊的姿態出場比賽。

== 主場球場 ==
[[File:2010-06-03_Arena_AufSchalke_01.jpg|thumb]]
'''-{zh-hans:费尔廷斯竞技场; zh-hk:維爾廷斯球場;}-'''（'''{{lang|de|Veltins-Arena}}'''），原名'''傲赴沙尔克体育场'''（'''{{lang|de|Arena AufSchalke}}'''），在2001年揭幕成为沙尔克零四的主场球场。在[[2006年世界杯足球赛|2006年世界杯足球赛]]作四场小组赛及一场八強淘汰賽的比赛场地，同时亦为[[欧洲足球协会|欧洲足协]]的[[欧洲足协五星级足球場|五星级足球場]]。2005年7月1日將球场的命名权给予著名的[[德國|德國]]啤酒厂[[:en:Veltins|-{zh-hans:菲尔廷斯; zh-hk:維爾廷斯;}-]]。

沙尔克零四原來的[[:en:Parkstadion|花园体育场]]（{{lang|de|Parkstadion}}）在90年代開始展現老舊，当球队在1997年历史性首夺[[欧洲联盟杯足球赛|-{zh-hans:欧洲联盟杯; zh-hk:歐洲足協杯;}-]]及2004年的百年會慶，球隊決定興建一座全新的多功能運動場，於1998年與[[德國|德國]]的建築公司[[HBM|HBM]]簽定價值1,860萬[[歐元|歐元]]的承建合約。新球場位於花園體育場鄰近地區，但地盤地底800米為兩個煤礦礦井，避免影響球場結構而被迫將球場從最佳的南-北座向改為西南-東北向使球場與礦井平行。

維爾廷斯球場舉行聯賽時容量為61,481人（包括座席及立席），國際賽轉為全座席的容量為53,993人。72個行政廂房一整圈圍繞球場，分隔上下兩層看台。球場設有可開關[[聚四氟乙烯|鐵氟龍]]塗佈的玻璃纖維頂蓋，利用球場上24個鋼鐵支架支撐，除可擋雨及雪外，當舉行演唱會時可將向外聲音減低105分貝。草坪上方25米懸掛四面大屏幕，每面為35平方米。與[[日本|日本]]的[[札幌巨蛋|札幌巨蛋]]及[[荷蘭|荷蘭]]的[[加爾勒球場|加爾勒球場]]相同，草坪可在四小時內移出球場供舉行演唱會等用途。
