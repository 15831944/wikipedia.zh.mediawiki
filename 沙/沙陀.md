'''沙陀'''，又名'''处月'''、'''朱邪'''、'''朱耶'''<ref>《永樂大典》卷7154·〈五代史補〉：太祖武皇，本[[朱耶赤心|朱耶赤心]]之後，沙陀部人也。其先生于[[雕|雕]]窠中，酋長以其異生，諸族傳養之，遂以「諸爺」為氏，言非一父所養也。其後言訛，以「諸」為「朱」，以「爺」為「耶」。</ref>，原是[[唐朝|唐朝]]时[[西突厥|西突厥]]的一部，游牧于今[[新疆|新疆]][[准噶尔盆地|准噶尔盆地]]东南（今[[巴里坤|巴里坤]]）一带，因其地有大沙丘，故而得名。

== 西域争雄 ==
[[唐太宗|唐太宗]][[貞觀_(唐朝)|贞观]]七年（633年），[[處月|處月]]部首领曾随西突厥贵族[[阿史那弥射|阿史那弥射]]至長安覲見。后來，處月隸屬于[[乙毗咄陆可汗|乙毗(亦写作“毘”)咄陆可汗]]。642年，乙毗咄陸可汗進攻[[伊州|伊州]]（今新疆[[哈密|哈密]]），又派遣處月、處密二部圍天山軍（治所在新疆[[托克遜縣|托克遜縣]]西北），皆被唐軍擊敗。<ref>[[資治通鑒|資治通鑒]]. [http://zh.wikisource.org/wiki/%E8%B3%87%E6%B2%BB%E9%80%9A%E9%91%91/%E5%8D%B7196 卷196]：西突厥乙毗咄陸可汗既殺沙缽羅葉護，並其眾，又擊吐火羅，滅之。自恃強大，遂驕倨，拘留唐使者，侵暴西域，遣兵寇伊州；郭孝恪將輕騎二千自烏骨邀擊，敗之。乙毗咄陸又遣處月、處密二部圍天山；孝恪擊走之，乘勝進拔處月俟斤所居城，追奔至遏索山，降處密之眾而歸。</ref>

[[唐高宗|唐高宗]][[永徽|永徽]]年间（653年或654年）唐军灭西突厥[[阿史那贺鲁|阿史那贺鲁]]，处月部亦被击败。[[武则天|武则天]]于[[长安_(年号)|长安]]二年（702年）在[[庭州|庭州]]（今新疆[[昌吉|昌吉]][[吉木萨尔|吉木萨尔]]）设立[[北庭都护府|北庭都护府]]，沙陀部首领[[朱邪金山|朱邪金山]]因参与征[[铁勒|铁勒]]有功，被封为“金满州都督”，受北庭都护管辖，后金山之子[[朱邪辅国|朱邪辅国]]因受[[吐蕃|吐蕃]]压迫，北迁庭州。

[[安史之乱|安史之乱]]之后[[回纥|回纥]]占领今[[新疆|新疆]]地区，而吐蕃则趁乱占据[[河西|河西]]、[[陇右|陇右]]（今[[甘肃|甘肃]]）。沙陀人受回纥挤压，于是投靠吐蕃，于789年与其联军攻占庭州。后吐蕃为防止沙陀与回纥勾结，将其迁至[[甘州|甘州]]（今甘肃[[张掖|张掖]]），封辅国之子[[朱邪尽忠|朱邪尽忠]]（唐将[[李克用|李克用]]曾祖）为统军[[大论|大论]]，常为吐蕃先锋。

== 迁徙中原 ==
800年左右，回纥攻占[[凉州|凉州]]，吐蕃以沙陀驻地靠近凉州，试图将其再次迁往[[黄河|黄河]]以西的高原地区。809年，沙陀不愿西迁，朱邪尽忠率全体部众投奔唐朝。吐蕃追杀，沙陀人且战且走，三万人中仅剩下两千人到达[[灵州|灵州]]（今[[宁夏|宁夏]][[灵武市|灵武市]]），朱邪尽忠亦战死，其子[[朱邪执宜|朱邪执宜]]继位。灵盐[[节度使|节度使]][[范希朝|范希朝]]将他们安定在[[盐州|盐州]]（今宁夏[[吴忠|吴忠]]市[[盐池|盐池]]县），任命执宜为阴山都督府兵马使，招纳旧部。

后范希朝改任[[太原|太原]]府，唐朝为防止沙陀人反复，令其随迁至黄花堆（今天的[[山西|山西]][[朔州|朔州]]市[[山阴|山阴]]县），后又分其兵隶各部，成为唐军重要的兵员来源。[[唐懿宗|唐懿宗]]时，招执宜子朱邪赤心镇压[[庞勋|庞勋]]起兵，赐姓名为[[李国昌|李国昌]]。

== 唐末风云 ==
876年，李国昌之子[[李克用|李克用]]反唐，攻占[[大同市|大同]]，880年，被唐军击败，父子二人率部北逃[[鞑靼|鞑靼]]。[[黄巢|黄巢]]入[[长安|长安]]，唐朝各[[藩镇|藩镇]]不出援军，[[唐僖宗|唐僖宗]]只好招李克用进兵击黄巢，于883年在[[黄陂|黄陂]]击败黄巢。唐朝封李克用为[[河东|河东]]（山西）[[节度使|节度使]]。黄巢死后，其旧将[[朱全忠|朱全忠]]势力日盛，与李克用在华北争战，终于在907年称帝，建立[[后梁|后梁]]，唐亡，中国进入[[五代十国|五代十国]]时期。

== 三代一国 ==
913年，李克用之子晋王[[李存勗|李存勗]]击败了反复无常的[[刘仁恭|刘仁恭]]父子，从而占据了[[幽州|幽州]]。923年灭后梁，建立[[后唐|后唐]]，是為莊宗。[[岐国|岐国]]归附后唐，其余诸国除[[前蜀|前蜀]]、[[杨吴|杨吴]]、[[南汉|南汉]]外皆向后唐称臣。925年，后唐灭前蜀。

莊宗建国后痴迷于戏剧和打猎，把国家大事全盘托付给[[伶人|伶人]]和[[宦官|宦官]]，又好殺多疑，不體恤士卒。926年由[[魏博軍|魏博軍]]的一個士兵[[皇甫暉|皇甫暉]]煽動不能還家的同袍，發動了[[鄴都之變|鄴都之變]]，李克用养子大太保[[李嗣源|李嗣源]]前往平亂，卻被叛軍擁立，莊宗打算御駕親征，卻被伶人發動[[興教門之變|興教門之變]]所殺，嗣源即位，是為明宗。933年明宗李嗣源驾崩，934年闵帝改元当年即被养兄[[李从珂|李从珂]]篡位。李从珂貶謫李嗣源女婿[[石敬瑭|石敬瑭]]，敬瑭投靠契丹，借契丹兵灭后唐。936年建立'''[[后晋|后晋]]'''，自称「儿皇帝」，对契丹（[[辽朝|辽朝]]）的國君[[辽太宗|辽太宗]]称父、割地、上贡。石敬瑭死后他的侄子[[石重贵|石重贵]]反辽，与辽国开战数次，终于被俘。后晋滅亡。

之后原后晋大将，沙陀人[[刘知远|刘知远]]自[[太原|太原]]起兵，建立'''[[后汉|后汉]]'''，不久病死，傳至後漢隱帝劉承祐，因猜忌[[顧命大臣|顧命大臣]][[郭威|郭威]]，郭威起兵殺死[[劉承祐|劉承祐]]，不久篡漢，建立[[后周|后周]]。原后汉皇族[[刘崇|刘崇]]在太原称帝，建立'''[[北汉|北汉]]'''，延续后汉，托庇于[[辽|辽]]，979年被[[宋朝|宋朝]]攻滅，刘氏全族被俘，迁居[[开封|开封]]。

{{or|之后沙陀便和[[汉族|汉族]]、[[契丹|契丹]]等民族融合，不再是一个独立的民族。辽代，游牧于[[阴山|阴山]][[长城|长城]]的黑車子室韋与元代[[汪古|汪古]]一带是沙陀后裔，奉晋王李克用为先祖，后逐渐融入[[蒙古族|蒙古族]]。

有人说[[雅库特人|雅库特人]]出自沙陀萨葛部。}}
