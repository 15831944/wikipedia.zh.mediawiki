'''沙塵天使'''（{{lang-en|Angel Dust}}），本名'''克莉絲汀娜'''（{{lang-en|Christina}}），是[[漫威漫畫|漫威漫畫]]旗下的一名虛構漫畫角色，由[[傑夫·強斯|傑夫·強斯]]與肖恩·馬丁布魯（{{lang|en|Shawn Martinbrough}}）創作。她在漫畫《{{link-en|莫洛克斯 (漫畫)|Morlocks (comics)|莫洛克斯}}》#1（2002年6月出版）中首次登場，是一名{{link-en|變種人 (漫威漫畫)|Mutant (Marvel Comics)|變種人}}。此外，沙塵天使還曾在電影《[[惡棍英雄：死侍|惡棍英雄：死侍]]》（2016年）中出場，由[[吉娜·卡拉諾|吉娜·卡拉諾]]飾演。

== 出版歷史 ==

沙塵天使在漫畫《{{link-en|莫洛克斯 (漫畫)|Morlocks (comics)|莫洛克斯}}》#1（2002年6月出版）中首次登場<ref name="Marvel.com"/>，由[[傑夫·強斯|傑夫·強斯]]與肖恩·馬丁布魯（{{lang|en|Shawn Martinbrough}}）創作<ref name="Comic Vine"/>。她是一位黑髮黃眼，{{convert|110|lb|kg|sigfig=1|abbr=on}}重，約165公分高的女子<ref name="Marvel.com"/>。

== 人物 ==

克莉絲汀娜（{{lang|en|Christina}}）出生在美國[[伊利諾州|伊利諾州]]的[[芝加哥|芝加哥]]，與父母過著幸福快樂的日子，直到一次，她的能力顯現出來，讓她意識到自己是名{{link-en|變種人 (漫威漫畫)|Mutant (Marvel Comics)|變種人}}。因為無法控制自己的能力（她的能力是由[[腎上腺素|腎上腺素]]激發的，每當腎上腺素過高就會失控），且擔心父母得知真相，克莉絲汀娜決定離家出走。她與一群變種人在一座廢棄的地鐵站（變種人團體{{link-en|莫洛克斯 (漫畫)|Morlocks (comics)|莫洛克斯}}在芝加哥的分部）生活<ref name="Moviecomicswhoswho.com"/>。地鐵站的成員包含領導人{{link-en|郵差 (漫威漫畫)|Postman (comics)|郵差}}、{{link-en|細胞 (漫威漫畫)|Cell (comics)|細胞}}、{{link-en|電氣夏娃|Electric Eve}}、{{link-en|垃圾蟲 (漫威漫畫)|Litterbug (comics)|垃圾蟲}}、{{link-en|落葉 (漫威漫畫)|Shatter (comics)|落葉}}與{{link-en|商人 (漫威漫畫)|Trader (comics)|商人}}等人<ref name="Comic Vine"/>。

一次，地鐵站遭到[[哨兵機器人|哨兵機器人]]（專門獵殺變種人的機器人）攻擊，郵差將克莉絲汀娜送回家。克莉絲汀娜鼓起勇氣告訴父母真相，但出乎克莉絲汀娜意料之外，她的父母接納了她<ref name="Moviecomicswhoswho.com"/>。在與同伴們催毀一個哨兵機器人基地後，克莉絲汀娜回家與父母團聚<ref name="Morlocks#3-4"/>。在《{{link-en|屠殺 (漫畫)|Decimation (comics)|屠殺}}》的劇情線中，克莉絲汀娜被[[神盾局|神盾局]]去除了超能力<ref name="New Avengers#18"/>。

== 能力 ==

沙塵天使能讓她的[[腎上腺素|腎上腺素]]升高，讓自身在短時間內獲得超乎常人的力氣。她能舉起{{convert|800|lb|kg|sigfig=1|abbr=on}}至{{convert|25|t|lb|sigfig=2|abbr=in|order=flip}}的物體。除了力氣，她的移動速度、敏捷度與耐力都會增強<ref name="Morlocks#1"/>。腎上腺素消退後，她的能力就會消失，同時也會累癱一陣子。沙塵天使無法控制自身能力，其超能力是經由她的情緒如憤怒和恐懼所激發的，因此要是她被情緒淹沒，就會造成失控<ref name="Moviecomicswhoswho.com"/>。

== 其他媒體 ==

沙塵天使曾在電影《[[惡棍英雄：死侍|惡棍英雄：死侍]]》（2016年）中出場，由[[吉娜·卡拉諾|吉娜·卡拉諾]]飾演<ref name="CaranoCast"/>。在該片中，沙塵天使與{{link-en|阿賈克斯 (漫威漫畫)|Ajax (comics)|阿賈克斯}}一同將傭兵韋德·威爾森（{{lang|en|Wade Wilson}}）改造成擁有癒合能力的超級人類「[[死侍|死侍]]」<ref name="CaranoSRNov2015"/><ref name="SkreinComicCon2015"/>。卡拉諾原本打算戴上黃色[[隱形眼鏡|隱形眼鏡]]來符合沙塵天使在漫畫中的樣貌，但化妝設計師{{link-en|比爾·科索|Bill Corso}}不打算這麼做<ref name="CorsoColliderMar2016"/>。
