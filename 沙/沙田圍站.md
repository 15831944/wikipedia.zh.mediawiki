'''沙田圍站'''（{{lang-en|'''Sha Tin Wai Station'''}}）是[[香港|香港]][[港鐵|港鐵]][[馬鞍山綫|馬鞍山綫]]的一個架空車站，位於[[新界|新界]][[沙田區|沙田區]][[沙田圍|沙田圍]][[沙角邨|沙角邨]]及[[博康邨|博康邨]]之間的沙角街和逸泰街之間地面，於2004年12月21日啟用。

==結構==
沙田圍站共分為2層，大堂位於地面，月台設於大堂之上<ref name="sta_info">{{Cite book|author=港鐵公司|title=沙田圍站車站指南|date=2009年8月|publisher=港鐵公司}}</ref>。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Sha Tin Wai Station 2017 06 part1.jpg | caption1 = 大堂
}}
大堂與月台層之間設有[[升降機|升降機]]和[[電動扶梯|扶手電梯]]，方便乘客上落。

現時，沙田圍站內提供不同類型的商店供乘客購物或進食，例如有[[便利店|便利店]]、[[麵包糕餅店|麵包糕餅店]]及[[醫務所|醫務所]]等服務及[[零售業|零售業]]<ref name="shop"/><ref name="sta_info" />。由於港鐵的收費區是不能進食，因此大部份[[餐馆|食肆]]不會設在收費區內。此外，沙田圍站也有不少自助服務設施供乘客享用，包括各主要[[銀行|銀行]]的[[自動櫃員機|自動櫃員機]]<ref name="shop"/><ref name="sta_info" />。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Sha Tin Wai Station 2017 11 part1.jpg | caption1 = 2號月台
| image2 = Sha Tin Wai Station 2018 06 part2.jpg | caption2 = 月台佈置
}}
沙田圍站設有2個月台，採用[[島式月台|島式月台]]方式排列，月台北面設有預留空間以便馬鞍山綫列車增加至8卡行走，現時擴建工程已經完成；並設有月台閘門，車站站牌以[[沙田圍|沙田圍]]之經典大種[[客家人|客家人]]建築[[曾大屋|曾大屋]]（山下圍、山廈圍）為代表背景設計。

===命名及啟用===
車站在馬鞍山綫規劃初期曾命名為'''沙角街站'''；後來以附近之[[客家|客家]]村落[[沙田圍|沙田圍]]為名。<ref>{{cite web|url=http://www.hk-place.com/view.php?id=325|title=道路及鐵路－香港鐵路（二）馬鞍山綫|work=香港地方|accessdate=2011年2月25日|language=zh-hant}}</ref>

==事件==
;「水舞間」事件
2016年5月10日，一股[[低壓槽|低壓槽]]強雷雨區於早上7時許抵港，沙田於1小時內降雨量達97毫米，達到「[[黑色暴雨警告信號|黑雨]]」水平，並導致沙田圍站月台天花嚴重漏水，並被報章評為儼如「水舞間」。[[沙角_(選區)|沙角]]區議員[[陳兆陽|陳兆陽]]於[[facebook|facebook]]留言諷刺[[港鐵|港鐵]]：「港鐵仲好意思加價？」。<ref>{{cite web|url=http://hk.apple.nextmedia.com/realtime/news/20160510/55084948|title=「黑雨」襲沙田 月台即變「水舞間」|work=蘋果日報|accessdate=2016年5月10日|language=zh-hant}}</ref>
;沙田圍站旁「煙花匯演」事件
2019年2月6日（[[正月初二|正月初二]]）零晨時份，懷疑有人在沙田圍站旁的沙田圍遊樂場非法燃放[[煙花|煙花]]，涉嫌違反《危險品條例》規定不得攜帶爆炸品進入鐵路處所的任何部份、非法藏有或未經許可燃放煙花的條文，當時還有馬鞍山綫列車在旁邊駛過，幸好當時沒有影響到行車安全，警員到場巡邏後未發現任何可疑人士。<ref>{{cite news|url=https://hk.news.appledaily.com/breaking/realtime/article/20190206/59230178|title=初二馬鐵旁「煙花匯演」 網民：嚇到BB喊|work=蘋果日報|date=2019-02-06|accessdate=2019-02-16}} </ref>
