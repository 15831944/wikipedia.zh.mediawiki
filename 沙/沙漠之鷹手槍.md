'''沙漠之鷹手枪'''（{{lang-he|'''נשר המדבר'''}}；{{lang-en|'''IMI Desert Eagle'''}}）在[[中國大陸|中國大陸]]地區被簡稱為'''沙鹰'''，是一種由[[以色列軍事工業|以色列軍事工業]]為[[麥林|麥林]]（Magnum Research，Inc.）所生產之[[半自動手槍|半自動]]、[[氣動式_(槍械)|氣動式]]運作，可使用[[.357麥格農|.357麥格農]]、{{tsl|en|.41 Magnum|.41麥格農}}、[[.44麥格農|.44麥格農]]、{{tsl|en|.440 Cor-bon|.440 Cor-bon}}、[[.50_AE|.50 AE]]等彈藥的[[手槍|手槍]]。

== 歷史 ==
1979年，在[[美國|美國]][[明尼蘇達州|明尼蘇達州]][[明尼阿波利斯|明尼阿波利斯]]附近剛組建成的麥格農研究所（Magnum Research）計畫推出一能射擊.357麥格農（Magnum）子彈的新型手槍，槍枝主要功能為打靶或打獵等，並將此計畫稱為「麥格農之鷹」（Magnum Eagle）。由公司創辦人兼槍械工程師B‧懷特（B‧White）主持技術細節和開發作業，成功於1981年推出最早沙漠之鷹的原型槍，1982年公佈消息並於1983年得到其設計專利。

但當時的技術還不完善，其槍在供彈系統上還存在許多問題，但由於生產商資金和技術實力薄弱，為彌補這些缺陷並將此槍更加推向市場，麥格農研究所便尋求與國內外大公司合作。最後由「以色列軍事工業」（IMI）持續接手改良，經過反復試驗和不斷改進，於1985年得到了目前的沙漠之鷹設計專利。1992年，美國政府開始對進口槍枝進行要求與限制，因此槍枝的製造流程改成以色列軍事工業製造零件，接著運往麥格農研究所負責零件的組裝以及後續加工，如鍍金或進行拋光等，並對槍管進行更加精細的加工。

== 結構 ==
在諸多使用[[短後座行程|短後座行程]]或[[氣體反衝式|氣體反衝式]]機構的手槍之中，沙漠之鷹的設計極為不同。

由於其使用了麥格農這種高威力的子彈，如果使用一般手槍的設計將在閉鎖的強度上不足，它改用較常在[[步槍|步槍]]上使用的[[氣動式_(槍械)|氣動]]機構。

它的{{Link-en|閉鎖式槍機|Closed bolt}}與[[M16突擊步槍|M16突擊步槍]]系列的步槍十分相似。氣動的優點在於它能夠使用比傳統手槍威力更大的[[子彈|子彈]]，這使得沙漠之鷹系列手槍能和使用麥格農子彈的[[左輪手槍|左輪手槍]]競爭。槍管採固定式固定於槍管座上，在近槍口處和膛室下方跟槍身連接。由於槍管在射擊時並不會移動，理論上有助於射擊的準確度。因槍管為固定式，並在頂部有瞄準鏡安裝導軌，使用者可自行加裝瞄準設備。[[手槍套筒|套筒]]兩側均有保險機柄，槍枝可左右手操作。

U字型握把採整體製造，以一彈簧銷固定，通常為硬塑膠製成，但也可請廠商訂做。為能進一步降低後座力，採用2根平行的復進簧。

[[File:Desert_Eagle_Chrome.JPG|left]]

瓦斯導管位於槍囗內約1cm處，導管再連結瓦斯缸管，活塞則位於[[手槍套筒|套筒]]前緣，當滑套裝入槍身兩側復進導槽時，活塞位置正好在瓦斯缸管內。子彈一經擊發後，高壓瓦斯馬上就進入瓦斯通道，藉由瓦斯導管導引至槍口的前方，之後再被導引往下並往後，直接推動固定在滑套上的活塞。滑套後退時會連帶著將槍機向後移動並退彈進入開膛狀態，在滑套往後移動時，亦會將擊槌壓下呈待發狀態。

之後受復進簧彈力影響驅使[[手槍套筒|套筒]]前行，並將彈匣內子彈頂入槍膛後，槍機旋轉成閉鎖狀態；如果沒子彈時，滑套則會固定在後方，等換上新彈匣後以姆指下壓，滑套會立刻將子彈推入槍膛。藉由槍機上突柄的設計，整個槍機會旋轉以解除閉鎖，然後完成拋殼、上膛、並且重新旋轉閉鎖等動作。也因此設計，如果將滑套固定在後時，可以清楚地看到在外槍管和槍身間的空間內會有一活塞暴露在外。

== 設計 ==
[[File:Desert_Eagle_patent_4619184.png|left]]
[[File:Desert_Eagle_.357_Magnum.jpg|thumb]]

沙漠之鷹為一把大且重的手槍，氣動的機械結構使得此槍的體積與重量都超出一般手槍；槍本身連空彈匣就已有1.9公斤。

巨大握把使其難以單手握持，氣動機械結構使[[後座力|後座力]]有所緩衝，女性也能使用（當然持槍姿勢必須正確）。在射擊時所產生的高噪音導致軍、警方拒絕採用，又因沙漠之鷹其貫穿力強，甚至能穿透輕質隔牆，因此沙漠之鷹目前僅少量的用於競技、狩獵，自衛。

儘管如此，它卻被裝備於少數國家的[[特警隊|特警隊]]或[[特種部隊|特種部隊]]；然而，這些手槍會否被用於實戰，仍然是未知之數。

== 型號 ==
[[File:Desert_Eagle.JPG|thumb]]
[[File:Desert-Eagle-dore-p1030142.jpg|thumb]]
沙漠之鷹有三種型號。
=== Mark I ===
鋁製的槍身，有.357麥格農和.44麥格農兩種口徑，已停產。

=== Mark VII ===
鋼製的槍身，有.357麥格農、.44麥格農和.41麥格農三種口徑，此型已停產，不過市場上仍有6吋（152毫米）和10吋（254毫米）長的.44麥格農口徑Mark VII庫存（2005年早期）。

=== Mark XIX ===

Mark XIX有.357麥格農、.44麥格農和.50 AE（Action Express）三種口徑。麥格農公司也曾展示過使用.440 Cor-Bon口徑的樣本（.50 AE的衍生槍種）。

== 變換口徑 ==
改裝一把沙漠之鷹的口徑，只需要相搭配的[[槍管|槍管]]、{{tsl|en|Bolt (firearms)|槍機}}和[[彈匣|彈匣]]，因此在野外能迅速地完成換裝以射擊別種子彈。最廣泛使用的槍管長是6英吋的（也有10和14英吋，但不常見）。槍管上已經做好了[[瞄準鏡|瞄準鏡]]的基座，可以輕易地裝上。

== 相關 ==
IMI生產了沙漠之鷹後，原本稱作[[IWI傑里科941手槍|杰里科941]]的手槍現在被稱作「小沙鷹」(英語：Baby Eagle)，這把槍並無相同運作原理，它採用標準的手槍設計──雙動模式和短後座行程的設計。一個相似之處是彈藥的使用，.41 AE子彈即是為Jericho 941設計，它使用[[縮緣式彈殼|縮緣式彈殼]]，所以只要更換槍管便能使用[[9毫米鲁格弹|9毫米鲁格弹]]子彈。這是因為.41 AE子彈等於將.41 Magnum子彈的[[底緣|底緣]]與[[退殼鉤槽|退殼鉤槽]]換成和9毫米子彈相同。這使得相同的[[退殼器|退殼器]]和[[拋殼頂桿|拋殼頂桿]]能夠在兩種子彈上共同使用。.50 AE正好可以做个类比，它也有縮緣式彈殼，使得槍只需換裝槍管和彈匣便能於.44沙漠之鷹和.50 AE兩種口徑之間切換。

== 犯罪用途 ==
沙漠之鷹在[[墨西哥|墨西哥]]被廣泛的用於販毒和犯罪用途，當中不少是從美國走私到當地。另外在美國和[[巴西|巴西]]等國也有被黑幫用於械鬥。

* 2015年—《[[Fate/Grand_Order|Fate/Grand Order]]》：Archer（設定為黑化的[[Fate/stay_night|Fate/stay night]]從者Archer）所使用的[[雙持|雙槍]]的原型為沙漠之鷹，分別是左手的紅色槍與右手的白色槍，具有握把護環並在槍管下方加裝刀刃而使兩槍變成[[刀銃|刀銃]]，並作為其專屬概念禮裝「[[干將莫邪|干将•莫邪]]（手枪）」中最明顯的武器。
* 2015年—《[[虹彩六號：圍攻行動|-{zh-hans:彩虹六号：围攻; zh-hant:虹彩六號：圍攻行動;}-]]》（Rainbow Six: Siege）：型號為黑色.50口徑Mark XIX型（曾經為鍍銀版本），命名為「D-50」，被[[海豹部隊|海豹部隊]]所使用。
* 2016年—《[[殺戮空間2|殺戮空間2]]》（Killing Floor 2）：型號為銀色.50口徑Mark XIX型，命名為“Desert Eagle”，可[[雙持|雙持]]。
* 2016年—《[[決勝時刻4：現代戰爭|-{zh:決勝時刻：現代戰爭; zh-hans:使命召唤：现代战争; zh-hant:決勝時刻：現代戰爭; zh-cn:使命召唤：现代战争; zh-tw:決勝時刻：現代戰爭; zh-hk:使命召喚：現代戰爭; zh-mo:使命召喚：現代戰爭;}-重製版]]》（Call of Duty: Modern Warfare Remastered）：型號為銀色/金色.50口徑Mark XIX型，外觀比前作更為精細，並修正至不再使用純雙動板機，但奇怪地撃鎚是處於半待發狀態。
* 2018年—《[[極地戰嚎5|-{zh-hans:孤岛惊魂; zh-hant:極地戰嚎;}-5]]》（Far Cry 5）：型號為.50口徑Mark XIX型，使用銀色鍍銘滑套及黑色[[机匣|底把]]，命名為“D50”，8發彈匣。
* 2019年—《[[孤岛惊魂：新曙光|-{zh-hans:孤岛惊魂：新曙光; zh-hant:極地戰嚎：破曉;}-]]》（Far Cry: New Dawn）：型號為.50口徑Mark XIX型，8發彈匣。
* 2019年— 《[[生化危机2_(重制版)|-{zh-hans:生化危机2（重制版）; zh-hant:惡靈古堡2（重制版）;}-]]》：[[里昂·S·甘乃迪|里昂·S·甘乃迪]]使用的大口徑手槍，「沙漠之鷹」被改名為『閃電之鷹』。

=== 輕小說 ===
* 2008年—《[[緋彈的亞莉亞|-{zh-hans:绯弹的亚里亚;zh-hant:緋彈的亞莉亞;}-]]》（Aria the Scarlet Ammo）：黑色.50口徑Mark XIX型被[[遠山金次|遠山金次]]所使用，原為金次的父親遠山金叉的遺物，但由平賀文違法改造成「金次樣式」，只有在[[緋彈的亞莉亞#HSS|爆發模式]]下才可以使用。
