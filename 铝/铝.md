'''铝'''（'''Aluminium''' 或'''Aluminum'''）是一种[[化学元素|化学元素]]，属于[[硼族元素|硼族元素]]，其[[化学符号|化学符号]]是'''Al'''，[[原子序数|原子序数]]是13。相对密度是2.70。铝是一种较软的易延展的银白色[[金属|金属]]。铝是[[地壳|地壳]]中[[地球的地殼元素豐度列表|第三大丰度的元素]]（仅次于[[氧|氧]]和[[硅|硅]]），也是[[化學元素豐度|丰度]]最大的金属，在地球的固体表面中占约8%的[[质量|质量]]。铝金属在化学上很活跃，因此除非在极其特殊的[[氧化还原反应|氧化还原]]环境下，一般很难找到[[游离态|游离态]]的金属铝。被发现的含铝的[[矿物|矿物]]超过270种。<ref>

铝因其低[[密度|密度]]以及耐[[腐蚀|腐蚀]]（由于[[钝化|钝化]]现象）而受到重视。利用铝及其[[铝合金|合金]]制造的结构件不仅在[[航空航太|航空航太]]工业中非常关键，在[[交通|交通]]和结构材料领域也非常重要。最有用的铝[[化合物|化合物]]是它的[[氧化物|氧化物]]和[[硫酸盐|硫酸盐]]。

尽管铝在环境中广泛存在，但没有一种已知生命形式需要铝[[元素|元素]]。

== 性质 ==
=== 物理性质 ===
铝是[[轻金属|轻金属]]，[[密度|密度]]僅是[[鐵|鐵]]三分之一左右。純鋁較軟，在300℃左右失去[[抗張強度|抗張強度]]，[[熔点|熔点]]660.4度。經處理過的[[鋁合金|鋁合金]]較堅韌、易延展。有着金属光泽，光滑时表面银白而发亮，粗糙时呈暗灰色。无[[磁性|磁性]]且不易点燃。反射[[可见光|可见光]]能力强（约92%），反射中远[[红外线|红外线]]可达98%。纯铝的强韧度为7～11MPa，而铝合金可达200～600MPa。铝很容易被加工，切割，塑形。

铝原子以[[立方晶格|立方晶格]]排列。

铝有良好的[[导电|导电]]导热性（都为[[铜|铜]]的59%），而远轻于铜。铝可以在低于1.2K 的温度和[[磁通量|磁通量]]大于100[[高斯_(单位)|高斯]]下[[超导材料|超导]]。

=== 化学性质 ===
铝容易與[[氧|氧]]反應，暴露于[[空气|空气]]中会在其表面生成致密的[[氧化铝|氧化铝]][[氧化铝|Al<sub>2</sub>O<sub>3</sub>]]（此过程为[[钝化|钝化]]），有效的防止其继续[[氧化|氧化]]，所以通常略显银灰色。
:<chem>4Al + 3O2 -> 2Al2O3</chem>

平常我们可见的铝制品，均已经被氧化。而其氧化薄膜又使鋁不易被腐蝕。

在280 °C的温度之下铝开始会被水氧化，生成[[氢|氢气]]和[[氢氧化铝|氢氧化铝]]和热。
:<chem>2Al + 6H2O -> 2Al(OH)3 + 3H2</chem>

同时与一般的[[金属|金属]]不同的是，它既可以和[[酸|酸]]进行反应，又可以和[[强碱|强碱]]进行反应，因此认为铝是[[两性金属|两性金属]]，铝的[[氧化物|氧化物]]称为[[两性氧化物|两性氧化物]]，而[[氢氧化铝|氢氧化铝]]则称为[[两性氢氧化物|两性氢氧化物]]。

铝和碱反应形成[[四羟基合铝酸盐|四羟基合铝酸盐]]（曾被认为是[[偏铝酸盐|偏铝酸盐]]）和氢气。

如与[[氢氧化钠|氢氧化钠]]反应生成[[四羟基合铝酸钠|四羟基合铝酸钠]]:
:<chem>2Al + 2NaOH + 6H2O -> 2Na[Al(OH)4] + 3H2</chem>

铝能够与稀的[[强酸|强酸]]进行反应，生成[[氢气|氢气]]和相应的铝[[盐_(化学)|盐]]。

如与稀[[盐酸|盐酸]]反应生成[[氯化铝|氯化铝]]：
:<chem>2Al + 6HCl -> 2AlCl3 + 3H2</chem>

与稀[[硝酸|硝酸]]反应生成[[硝酸铝|硝酸铝]]:
:<chem>Al + 4HNO3 -> Al(NO3)3 + NO ^ + 2H2O</chem>

该反应为其中的一个反应。

在[[常温|常温]]下，铝在[[浓硝酸|浓硝酸]]和[[浓硫酸|浓硫酸]]中被[[钝化|钝化]]，不与它们反应，所以浓硝酸是用铝罐（可维持约180小时）运输的。

=== 同位素 ===
已知铝的同位素有24个，原子量从19～42，但只有<chem>^27Al</chem>（稳定）和 <chem>^26Al</chem>（有放射性，[[半衰期|半衰期]]为7.2×10<sup>5</sup>年）存在于自然界。其中<chem>^27Al</chem>占 99.9% 以上。<chem>^26Al</chem>的比例可用于测定陨石的年龄。

== 自然分布 ==
铝元素在[[地壳|地壳]]中的含量居金属首位，占地壳总量的8.3 %。

== 历史 ==
相对于其他金属，铝的发现比较晚。1808年[[汉弗里·戴维|汉弗里·戴维]]爵士首次使用了“Aluminum”这个词，并开始尝试生产铝。 1825年丹麦化学家[[汉斯·奥斯特|汉斯·奥斯特]]成功用[[钾|钾]]从[[氯化铝|氯化铝]]中还原出铝：
:<chem>4AlCl3 + 3K -> Al + 3KAlCl4</chem>

[[File:Bolton-davy.jpg|thumb]]爵士]]

1827年[[弗里德里希·维勒|弗里德里希·维勒]]用金属[[钾|钾]]还原熔融的无水氯化铝得到较纯的金属铝单质。由于取之不易，当时铝的价格高于[[黄金|黄金]]。 

[[德维尔|德维尔]]（Henri Etienne Sainte-Claire Deville）在1846年纯化了维勒过程，并发表在1859年的一本书上。由此十年内铝的价格降低了90%。

1886年[[查尔斯·马丁·霍尔|查尔斯·马丁·霍尔]]（Charles Martin Hall）和[[保罗·埃鲁|保罗·埃鲁]]（Paul Héroult）各自独立发现了以命名的[[电解|电解]]制铝法。在1889年[[卡尔·约瑟夫·拜耳|卡尔·约瑟夫·拜耳]]（Carl Josef Bayer）继续-{zh-cn:优化;zh-tw:最佳化}-了从[[铝土矿|铝土矿]]中提取[[氧化铝|氧化铝]]的过程，使得生产铝的原料[[氧化铝|氧化铝]]更加经济易得。迄今以[[拜耳法|拜耳法]]与[[霍尔-埃鲁法|霍尔-埃鲁法]]联用生产铝的方法为大规模工业制铝的主要手段。

== 品种分类 ==
根據鋁[[锭|锭]]主成份含量可以分成三类：高级纯铝（铝的含量99.93%－99.999%）、工业高纯铝（铝的含量99.85%－99.90%）、工业纯铝（铝的含量98.0%－99.7%）。

== 铝的应用 ==
白色、软、易加工，金属铝熔点为660度，是重要工业原料。
* 铝的[[合金|合金]]较轻而强度高。通常的工业用[[铝合金|铝合金]]，如[[6063-T5|6063-T5]]，其强度超过了3Cr13高速不锈钢，因而成為[[飞机|飞机]]、[[汽车|汽车]]、[[軌道車輛|軌道車輛]]及[[火箭|火箭]]的主要生產原料。
* 由于铝有良好的[[导电性|导电性]]和[[导热性|导热性]]且較輕，可取代[[銅|銅]]用作[[高壓電|超高电压]]的电缆材料。高纯铝具有更优良的性能。
* 铝在高温时的[[还原性|还原性]]极强，可以用于冶炼高熔点金属以及[[長焊鋼軌|長焊鋼軌]]铁路铺设时的临时炼铁（这种方法称为“[[铝热法|铝热法]]”）。
* 铝富延展性，可製成[[铝箔|铝箔]]，用于[[包裝|包裝]]。
* 鋁是金屬，所以可以[[回收|回收]]再造，但是回收率不高。
* 鋁的抗腐蝕性（特別是抗氧化，因其氧化物[[氧化鋁|氧化鋁]]在金屬表面形成緻密的膜，反而增加了鋁的抗腐抗熱性）優異，外觀質感佳，價格適中，為電腦機殼的首選材料。
* 鋁可被人體自然排泄。早期研究人員懷疑鋁是導致[[阿茲海默症|阿茲海默症]]的元兇，但幾十年的研究並沒有發現任何證據可以證明鋁會導致老人痴呆。<ref>{{cite web | url = http://alzheimers.org.uk/site/scripts/documents_info.php?documentID=99 | title = Aluminium and Alzheimer's disease | accessdate = 2012-06-15 | deadurl = yes | archiveurl = https://web.archive.org/web/20120311205419/http://alzheimers.org.uk/site/scripts/documents_info.php?documentID=99 | archivedate = 2012年3月11日 | df =  }}</ref>

近五十年来，铝已成为世界上最为广泛应用的[[金属|金属]]之一。除上所述，在[[建筑业|建筑业]]上，由于铝在[[空气|空气]]中的稳定性和[[阳极处理|阳极处理]]后的极佳外观而受到很大应用；在[[航空|航空]]及[[國防工業|国防军工]]部门也大量使用铝合金材料；在电力输送上则常用高强度钢线补强的铝缆，在一些地方因銅製電纜價格較高常遭竊而改用鋁製電纜；[[集装箱|集装箱]]运输、日常用品、家用电器、机械设备等都需要大量的铝。

== 铝期货 ==
目前铝的[[期货交易|期货交易]]主要在[[伦敦金属交易所|伦敦金属交易所]]和[[上海期货交易所|上海期货交易所]]进行。铝是[[伦敦金属交易所|伦敦金属交易所]]的重要品种。
