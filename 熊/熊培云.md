'''熊培云'''{{BD|1973年|1月28日|||catIdx=X熊}}，中国[[江西|江西]][[九江|九江]]人，毕业于[[南开大学|南开大学]]、[[巴黎大学|巴黎大学]]，任教于南开大学。著名自由撰稿人、[[公共知识分子|公共知识分子]]。<ref name=" fish" >{{cite web |url=http://news.qq.com/a/20110412/000970.htm |title=熊培云：华人公共知识分子 |accessdate=2011年4月12日 |publisher=腾讯网 |archiveurl=http://news.qq.com/a/20110412/000970.htm |archivedate= 2012年11月7日|work= |language =zh-cn}}</ref><ref name=" macs" >{{cite web |url= http://cul.sohu.com/20120417/n340811979.shtml|title= 熊培云：我不是左派或右派，我是一个底线派|accessdate=2012年4月17日 |publisher=搜狐网 |archiveurl=http://cul.sohu.com/20120417/n340811979.shtml |archivedate=2012年11月7日|work= |language =zh-cn}}</ref>

== 生平 ==
一九七三年生于江西[[永修|永修]]，毕业于南开大学、巴黎大学，主修历史学、法学、传播学与文学。

曾驻巴黎，任《南风窗》驻欧洲记者，兼国内若干影响力媒体专栏作家及社论作者，《新京报》首席评论员。东京大学客座研究员，牛津大学访问学者，广西师范大学出版社“理想国译丛”创始主编委员，京东文学奖评委<ref>[http://wxy.nankai.edu.cn/People/Detail/109/0 南开大学文学院]</ref>
