'''熊汶銨'''（遊戲代號：'''Never loses'''、'''NL'''，{{bd|1995年|4月2日}}）是一名台灣《[[英雄聯盟|英雄聯盟]]》職業選手，擁有國民電競妹妹之稱<ref>{{Cite web |url = http://game.ettoday.net/article/818281.htm|title = 我狼來襲！IEM賽後，閃電狼探《ETtoday新聞雲》全紀錄|date = 2016年11月25日 |publisher = ETtoday }}</ref>，目前為專職實況主。

NL是一名團戰型最主要的物理輸出，雖然在戰隊中不夠耀眼，但也從未讓隊友失望。而且每到大賽都會爆發潛力，是一名大賽型的選手。個人英雄池擅長英雄均為團隊型最主要的物理輸出，如伊澤瑞爾、路西恩、燼、凱特琳、克黎思妲等英雄，在對線能力方面造詣頗深。<ref>{{Cite web |url = http://www.laonanren.com/news/2016-05/114472p7.htm |title = 2016MSI季中赛FW战队全体成员及教练资料介绍|date = 2016年5月6日 |publisher = 老男人 }}</ref>

== 生平 ==

1995年4月2日出生於[[桃園市|桃園市]][[八德區|八德區]]，身高172公分，體重50公斤，A型<ref>{{Cite web |url = https://docs.google.com/document/d/1ekpVv9xUaIFN1artBdvtD8Gup_P9Vh8NFbuuCfUliqQ/edit |title =NL實況臺 - 關於我}}</ref>，家中有爸爸、媽媽、奶奶、妹妹和一隻[[馬爾濟斯|馬爾濟斯]]。
高中就讀振聲中學資訊科




國中時患有腸躁症，經醫生檢查後大腸沒問題，但是少一顆腎，因此不用當兵。<ref>{{Cite web |url = http://www.ironforum.com.tw/subjdetail/139/2 |title = 《閃電狼》電競選手背後的力量：NL家人篇 |date = 2015年7月22日 |publisher = 鉄人論壇 |access-date = 2016-12-05 |archive-url = https://web.archive.org/web/20161220080542/http://www.ironforum.com.tw/subjdetail/139/2 |archive-date = 2016-12-20 |dead-url = yes }}</ref>
