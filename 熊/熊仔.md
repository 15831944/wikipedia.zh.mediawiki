'''熊仔'''（1990年9月12日－），本名'''熊信寬'''，台灣台北人，[[饒舌|饒舌]]歌手 。

== 早年經歷 ==
成長於書香世家，祖父為[[小學|小學]]校長、父母親皆為[[大學|大學]]教授。學歷為[[建國中學|建國中學]]、[[臺灣大學|臺灣大學]]電機系、電信工程學研究所。 就讀臺灣大學時，加入了台大嘻哈研究社。

== 演藝經歷 ==
2014年加入音樂人[[大支|大支]]創辦的「人人有功練音樂工作室」，於2015年發行首張專輯《無限》，曾為製作專輯休學半年，得[[第6屆金音創作獎|第6屆金音創作獎]]5項提名，並奪得最佳嘻哈專輯、最佳嘻哈單曲、最佳新人等三項大獎。[[第27屆金曲獎|第27屆金曲獎]]提名最佳國語專輯獎與最佳新人獎。[[第29屆金曲獎|第29屆金曲獎]]入圍最佳單曲製作人獎。

=== 豹子膽身份 ===
[[BOWZ_豹子膽|BOWZ 豹子膽]](1998-2019)是由藝人熊仔創造出來的[[虛構角色|虛構角色]]，<ref>{{Cite web|url=https://www.elle.com/tw/entertainment/voice/g22636900/kumachan-new-creation-bowz/|title=熊仔 ≠ 豹子膽，新專輯細節多到像看《全面啟動》！解謎五大彩蛋強到讓人跪|accessdate=2018-12-31|author=|date=2018-08-12|last=Lee|first=Inna Chou 與 Leah|work=ELLE|publisher=|language=zh-TW}}</ref>賦予他一個對於社會現狀不滿、又不得志的地下饒舌歌手身分<ref>{{Cite web|url=https://www.cool-style.com.tw/wd2/archives/303169|title=囂張指稱 "流行音樂" 都是垃圾！「BOWZ 豹子膽」你到底是何方角色？|accessdate=2018-12-31|date=2018-05-05|work=COOL-STYLE 潮流生活網|language=zh-TW}}</ref>。其現身方式以影音平台[[YouTube|Youtube頻道]]和社群網站為主。 目前已發行歌曲有《給愛麗絲》、《禁愛令》、《走紅毯》、《假朋友真兄弟》、《大頭大頭》、《夢中夢》、《卡提諾》、《失真》和《...夢中夢中夢中夢中...》。

BOWZ豹子膽是個代表熊仔的另一個身分，熊仔和他的團隊會利用[[Instagram|Instagram]]在每週不定時間發佈貼文，讓觀眾與其互動、留言，並讓這個角色的支線劇情和主線劇情能夠順利發展<ref>{{Citation|last=嘻哈龍虎門|title=[龍虎門] 給個說法 - 熊仔＆豹子膽|url=https://www.youtube.com/watch?v=HH5yWPYY2bU&t=211s|accessdate=2019-01-01}}</ref>。在到達一定的劇情發展後，這個角色會在[[YouTube|YouTube]]頻道上發佈歌曲，最後集結成一張專輯，收錄在熊仔／BOWZ 豹子膽的《夢想成真》專輯。

於2019年1月7日，在[[Facebook|Facebook]]粉絲專頁及Instagram以經紀人角度發佈貼文，宣佈於2019年1月6日發現BOWZ的遺體在車禍現場，詳細狀況仍待調查。將會將該兩個社群媒體的帳號轉換為紀念性帳號。

== 軼事 ==
*2015年6月13日曾於[[國立臺灣大學|台大]]電機學群畢業典禮上以Rap Style致詞，引起全校轟動。<ref>[https://www.youtube.com/watch?v=4KndYVj_Rtk，https://www.youtube.com/watch?v=4KndYVj_Rtk]，熊仔台大電機學群畢業代表 Rap style 致詞。</ref>
*於2016金曲獎典禮上，被[[張清芳|張清芳]]以「那個胖子」稱呼，廣為流傳，成為新作《那個胖子》創作靈感。張清芳事後解釋，她不認識熊仔，心想，綽號既為「熊仔」，應該身形如小熊一般渾圓可愛，故稱「那個胖子」。
*於2018年5月3日以[[BOWZ_豹子膽|豹子膽]]的名義發布歌曲《給愛麗絲》，也正式讓[[BOWZ_豹子膽|豹子膽]]這位虛擬人物登場
*於2018年8月8日，[[創作歌手|創作歌手]]R-chord [[謝和弦|謝和弦]]因病情復發並在他的臉書留下「你要自己剃光頭，還是我幫你剃，再裝嘛！」等字句引發各網友的撻伐，熊仔則在10日寫了一首歌回應[[謝和弦|謝和弦]]。
