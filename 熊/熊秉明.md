'''熊秉明'''（{{bd|1922年||2002年|12月14日}}）。[[法籍華人|法籍華人]]，[[哲學家|哲學家]]、[[詩人|詩人]]、[[雕塑家|雕塑家]]。著名數學家[[熊慶來|熊慶來]]之子。

== 生平 ==
1922年生於在中国[[南京|南京]]，祖籍云南弥勒县。1939年考入[[国立西南联合大学|国立西南联合大学]]经济系，次年转入哲学系。1944年毕业于[[国立西南联合大学|国立西南联合大学]]。毕业时全班男生全部应征入伍，熊秉明被调配至滇南战场，任美军少尉翻译官。1947年考取公费留法，进入[[巴黎大学|巴黎大学]]攻读博士学位。1949年改行在巴黎国立美术学校学习雕塑艺术。1952年与瑞士女子陶乐蒂结婚并于次年生下长子有瑞。1960年在瑞士[[苏黎世大学|苏黎世大学]]教授汉语及中国哲学。1962年在[[巴黎大学|巴黎大学]]东方语言文化学院担任中文系教授及系主任。1969年父亲[[熊庆来|熊庆来]]逝世。

1983年获[[法国教育部|法国教育部]]棕榈骑士勋章，并举办“巴黎华裔艺术家展览”。次年被[[法国教育部|法国教育部]]任命为教授。1998年5月参加北京大学百年校庆典礼，1999年与第二任妻子陸丙安结婚。旅居[[法國|法國]]50年，2001年被南京大学授予荣誉教授称号。2002年3月為[[南京大學|南京大學]]製作大型雕塑「孺子牛」，陳設於南大校園，作為[[南京大學|南京大學]]百年校慶紀念標記。

在歐洲期間，熊秉明曾是[[歐洲華文作家協會|歐洲華文作家協會]]會員。

2002年12月14日因[[腦溢血|腦溢血]]在法国巴黎突然去世，享年80歲。 <ref>{{cite web |url=http://www.mile.hh.gov.cn/info/1005/5201.htm |title=存档副本 |accessdate=2016-08-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20160816080020/http://www.mile.hh.gov.cn/info/1005/5201.htm |archivedate=2016-08-16 }}，简介。</ref><ref>[http://www.artofxiongbingming.org/chronology_e.php]，大事年表。</ref>

== 逸闻 ==
作为童年好友兼邻居的[[杨振宁|杨振宁]]向中国美术馆捐赠熊秉明的三件雕塑作品，分别是《笔架》《骆驼》和《马》。<ref>[http://culture.ifeng.com/a/20160827/49848895_0.shtml 杨振宁捐赠熊秉明雕塑作品]，凤凰网新闻。</ref>
