[[File:20100915Senkaku_Islands_Uotsuri_Jima_Kita_Kojima_Minami_Kojima.jpg|thumb]]、[[南小島|南小島]]]]
'''漢疆演習'''，或稱'''漢疆計畫'''<ref>根據參與行動官兵回憶書籍，應為【演習】，但媒體仍有使用【計畫】一說。</ref>，是民國79年（1990年）[[中華民國|中華民國]][[國軍|國軍]]计划護衛[[中華民國全國運動會|中華民國全國運動會]]聖火登島[[釣魚台列嶼|釣魚台列嶼]]宣示主權<ref name="國防新聞網">{{cite web |url=http://www.ewdefense.com/poman.asp?pno=566|title=塵封的國家機密～漢疆計畫|publisher=國防新聞網}}</ref>，然被時任[[中華民國總統|總統]][[李登輝|李登輝]]制止的一項作戰計畫<ref>[http://www.yzzk.com/cfm/Content_Archive.cfm?Channel=ac&Path=2208325152/45ae1.cfm　亞洲週刊:國軍保釣漢疆突擊隊血淚史曝光　黃銘俊]</ref>，這是一件沒有官方文件的歷史公案，時任[[總統|總統]][[馬英九|馬英九]]接受《[[亞洲週刊|亞洲週刊]]》專訪時證稱「漢疆突擊隊」突擊登[[釣魚台|釣魚台]]是被[[李登輝|李登輝]]喊停的。<ref>[http://www.nownews.com/2012/11/08/91-2871264.htm　新新聞-郝伯村策劃漢疆突擊隊突擊登釣魚台　馬英九:最後是被李登輝喊停的]</ref>而李登輝則否認有過此一計畫。

==選派==
民國79年（1990年）由時任[[行政院長|行政院長]][[郝柏村|郝柏村]]跳過三軍統帥[[李登輝|李登輝]][[總統|總統]]，囑國防部擬訂漢疆演習，以[[中華民國陸軍|中華民國陸軍]][[陸軍航空特戰指揮部|空降特戰司令部]]獨立第62旅步4營第45連官兵組成突擊隊，護衛高雄市主辦[[臺灣區運動會|台灣區運]]聖火登島釣魚台宣示主權。<ref>{{harvnb|黃銘俊|2012|pp=233-237}}</ref>

==特訓==
特訓於9月下旬實施，時任旅長[[賈輔義|賈輔義]]上校（[[中華民國陸軍軍官學校|陸官]]39期），步4營營長[[張傳善|張傳善]]中校（陸官47期），第45連連長[[關至德|關至德]]上尉（陸官專科6期）。連上約80位弟兄，扣除已婚、待退、獨子，選一個加強排50人兵力實施格鬥、鎮暴、阻擊、暴破等特訓；突擊隊預計搭乘的[[UH-1直升機|UH-1直升機]]，為減輕重量利機動性，拆除直升機的火箭砲與機槍，禁止隊員帶項鍊、戒指，原本兩公斤重的軍皮鞋也換成膠鞋。特訓期間管制休假，最後幾天每日搭直升機加強演訓，晚上播放電影《[[八百壯士|八百壯士]]》、《[[英烈千秋|英烈千秋]]》...等軍教片激勵士氣，隊員切結保密，集體行動，軍服沒有階章、姓名，槍枝無國軍標誌，錄遺照並留下頭髮、指甲與遺書，有做好為國犧牲之心理準備。

==作戰管制==
特訓後於50名訓員中作戰管制36人執行任務，原由45連副連長[[張雄偉|張雄偉]]中尉（陸官專科8期）與排長[[簡宜天|簡宜天]]少尉（預官39期）任正副隊長，復經連長關至德上尉（陸官專科6期）向作戰科長[[林志雄|林志雄]]中校（陸官45期，後因車禍亡故）力爭取代副連長親自率隊，並提議增連輔導長楊立中尉（政戰35期）改任副隊長，與熟捻[[日語|日語]]之[[本部連|旅部連]]士兵二人，共計38人機降登島任務。

==臨戰待命==
10月20日任務當天，UH—1H直升機4架，隊員38人帶著AN/PRC-77通信主機4部、[[57式機槍|57式機槍]]4挺、[[M72輕型反裝甲武器|66火箭彈]]、[[M16|M16]]步槍、[[手榴彈|手榴彈]]、彈藥及救生衣，於晚餐後在停機坪完成待命準備，連長與作戰科長在旅部戰管室受命。突擊隊原定21日凌晨清島以利聖火登島，事實上，突擊隊主在護衛區運聖火登島，與拆除島上燈塔及建物設施，並無意與日員衝突。唯當日凌晨高雄區運聖火「上賓一號」海釣船挺進至釣魚台6浬處時，已被日艦包圍無法再前進，僵持4小時後，聖火隊遙望著釣魚台點燃聖火，高舉國旗並高唱[[中華民國頌|中華民國頌]]、[[梅花_(電影)|梅花]]等歌曲後返航，是而突擊任務並未配合執行。<ref>[http://news.chinatimes.com/politics/50207343/112012110500114.html 中時電子報新聞:漢疆演習揭密，準備空降釣魚島，寫遺書，剪指甲，互道忠烈祠見]</ref>

==任務終止==
該演習海空軍均有配合措施，然美軍亦有海空軍在該域加強戒備，警告台灣不可輕舉妄動意味濃厚。突擊隊雖未護衛區運聖火登島，然任務仍持續待命，至11月5日[[李登輝|李登輝]]打電話給[[郝柏村|郝柏村]]說：「釣魚台是籌碼！不是戰場！」，任務方告終止。<ref>{{cite web |url=http://www.ewdefense.com/weapon.asp?wno=125|title=「保釣」系列2—「保臺」、「保釣」利器（一）|publisher=亞太防務雜誌提供 VOL.4}}</ref><ref>[http://www.cnki.com.cn/Article/CJFDTotal-BQZS201011029.htm 军事保钓掀风云 绝密的台军“汉疆计划” 《兵器知识》 2010年11期]</ref><ref>{{cite web |url=http://www.ewdefense.com/poman.asp?pno=566|title=塵封的國家機密～漢疆計畫|publisher=國防新聞網 2012/4/9}}</ref>

==傳媒報導與揭密成書==
多年來這項沒有官方文件的歷史公案，政府是既不承認也沒否認，在突擊隊隊員未受訪前，傳媒載述多為概要，甚有訛傳情節，直至「漢疆突擊隊」隊員2012年受訪揭密成書「規復釣魚台從漢疆突擊隊出發」，方得一窺實境<ref>[http://www.books.com.tw/exep/prod/booksfile.php?item=0010565470 規復釣魚台從漢疆突擊隊出發]</ref>。
*名稱：初擬以「悍疆計畫」為名，取悍衛疆土之意，因懼李登輝質疑反對，而改以「漢疆演習」為名，同（漢）字（演習），讓李誤以為是「漢光演習」之一部，事實上確也達到欺瞞效果，唯部份載述仍以「漢疆計畫」為名。
*任務：非長期軍事佔領，而是護衛台灣區運聖火登島[[釣魚台列嶼|釣魚台]]宣示主權，插旗與破壞燈塔及硬體設施，快去快回，無意與日員衝突。
*旅長：賈輔義上校（1991年晉升少將），非[[薛石民|薛石民]]少將。
*突擊隊人數：45人是最初核報人數，作戰管制為38人執行。
*直升機：UH—1H直升機4架次，非外傳八架，也沒出勤，由隊長始終在戰管室待命可證。
*武器彈藥：當時從62旅彈藥庫找出批號最新的美造彈藥，以及狀況良好的美造20發短彈匣供突擊隊使用。
*爆材：因是短暫登島宣示主權，僅輕裝攜[[手榴彈|手榴彈]]、[[57式機槍|57式機槍]]、[[M72輕型反裝甲武器|66火箭彈]]簡易爆材，訛傳另攜56式3.5吋火箭筒，因該武器當時非堪用品未帶。
*有人提議用立可白身體留名以利認屍作法但未實施，而遺照是用錄相非拍照。
*郝伯村是跳過三軍統帥逕指示國防部執行，故在10月21日總統李登輝並未知曉與制止登島任務，然該日雖未登島，任務未即撤銷，至李獲知後，於11月5日方要求郝柏村終止任務。
*李登輝本人則否認有此計畫，只是單純要海軍艦隊護航在釣魚台海域作業的台灣漁船，並要求海軍不得進入日本領海。<ref name="新台灣的主張">{{cite book|author=李登輝著|title=新台灣的主張|location=[[台北市|台北]]|publisher=遠足文化|date=2015-08}}</ref>{{rp|178}}
