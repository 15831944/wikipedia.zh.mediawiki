'''汉四郡'''是[[汉武帝|汉武帝]]在[[元封_(西汉)|元封]]二年（前109年）至元封三年（前108年）間派兵征服[[衛滿朝鮮|衛滿朝鮮]]後在其旧地设立的四个[[郡|郡]]的總稱，分别為[[乐浪郡|乐浪郡]]、[[玄菟郡|玄菟郡]]、[[真番郡|真番郡]]及[[臨屯郡|臨屯郡]]。其中，玄菟郡設于前107年，其他三郡設于前108年。四郡并存的情况只存在了二十多年，到前82年，真番、临屯二郡與玄菟郡的東部地區被并入樂浪郡，分別設東部都尉和南部都尉，玄菟郡治遷至[[高句驪縣|高句驪縣]]。之后的[[东汉|东汉]]、[[曹魏|曹魏]]与[[西晋|西晋]]皆保留了乐浪郡与玄菟郡。[[汉献帝|汉献帝]]时割據[[遼東|遼東]]的公孫氏析樂浪郡南部都尉（原真番郡轄地）為[[帶方郡|帶方郡]]，並為曹魏、西晉所承繼。313年，[[高句丽|高句丽]]侵略乐浪郡，据有乐浪、带方二郡的[[张统|张统]]因不堪长期孤军与高句丽、[[百濟|百濟]]作战而率千余家迁到[[辽西|辽西]]投靠[[慕容廆|慕容廆]]。慕容廆後为其在辽西侨置乐浪郡（《资治通鉴》卷八八，[[建兴_(西晋)|建兴]]元年条）。隨著樂浪郡在遼西侨置，管轄朝鮮半島的汉四郡灭亡。而原真番郡轄地[[帶方郡|帶方郡]]由于成为中原王朝的[[飞地|飞地]]，受到了孤立，4世紀時和樂浪故地一起成為高句麗與[[百济国|百济国]]爭霸的場所。

== 高句麗與漢四郡 ==
{{朝鲜半岛历史 |隐藏 = 1}}

《[[三國史記|三國史記]]》記載公元37年[[大武神王|大武神王]]向[[鸭绿江|鸭绿江]]南的[[乐浪郡|乐浪郡]]发动进攻，一度占据<ref>《[[三国史记|三国史记]]》卷14：“二十年　王袭乐浪　灭之”</ref>。七年后（42年），光武帝派兵渡海收复了乐浪，阻止了高句丽的扩张。49年2月，[[慕本王|慕本王]]派遣将军袭击[[東汉|東汉]]的[[右北平|右北平]]、[[渔阳|渔阳]]、[[上谷|上谷]]、[[辽西|辽西]]等四个郡。辽东太守[[祭肜|祭肜]]以恩义及信义向慕本王对质，并透过和亲使两国的关系得以回复。

据《三国史记》[[东川王|东川王]]本纪记载，东川王二十一年（247年），“王以丸都城经乱不可复都，筑平壤城，移民及庙社。平壤者，本仙人[[檀君|王俭]]之宅也，或云王之都王险。”<ref name="中国与朝鲜半岛关系史论">[http://book.ifeng.com/section.php?book_id=512&id=35101 中国与朝鲜半岛关系史论]</ref>不过《[[晋书|晋书]]》卷一四《地理志》记载：“[[咸宁_(西晋)|咸宁]]二年（276）十月，分昌黎、辽东、玄菟、带方、乐浪等郡国五置[[平州|平州]]”，“乐浪郡，汉置，统县六，户三千七百”，“带方郡，[[公孙度|公孙度]]置，统县七，户四千九百。”《三国史记·高句丽本纪》记载，高句丽“侵乐浪郡，虏获男女二千余口”。因此247年东川王所城的平壤应该為國內城的衛城，而不在[[大同江|大同江]]。<ref name="中国与朝鲜半岛关系史论" />傳統觀點認為313年被是高句麗據有樂浪郡的開始，《三国史记·高句丽本纪》记载，高句丽“侵乐浪郡，虏获男女二千余口。”；[[故国原王|故国原王]]四十一年，“百济王率兵三万来攻平壤城”；[[小兽林王|小兽林王]]七年“百济将兵三万来侵平壤”；[[广开土王|广开土王]]四年，“与百济战于浿水之上”。这些都表明公元4世纪初高句丽已夺取了二郡，控制了大同江流域<ref name="中国与朝鲜半岛关系史论" />。

[[前燕|前燕]]慕容氏是[[隋|隋]][[唐|唐]]之前最后一次给[[高句丽|高句丽]]以巨创的中国地方政权。342年冬，[[慕容皝|慕容皝]]毁高句丽[[丸都城|丸都城]]，不过并不能恢复中国以前对[[朝鲜半岛|朝鲜半岛]]北部的控制，只好接受高句丽的臣服<ref name="中国与朝鲜半岛关系史论" />。《资治通鉴》卷97载，故国原王在前燕伐高句丽的次年“遣其弟称臣入朝于燕，贡珍异以千数”。迫于前燕压力，高句丽迁都到[[平壤|平壤]]<ref name="中国与朝鲜半岛关系史论" />。

371年，当时的[[百济|百济]]世子[[近仇首王|近仇首王]]率3万军队拿下樂浪并处死了高句丽[[故国原王|故国原王]]，百济短期獲得樂浪地区。高句丽[[好太王|好太王]]和[[长寿王|长寿王]]两代多次大败百济，百济势力被逐出了乐浪郡。427年前后，百济继续争夺乐浪、带方两郡的控制權。

313年初，据有[[乐浪|乐浪]]、[[带方|带方]]二郡的[[张统|张统]]因不堪长期孤军与[[高句丽|高句丽]]、[[百濟|百濟]]作战而率千余家迁到[[辽西|辽西]]投靠[[慕容廆|慕容廆]]，慕容廆後为其在辽西侨置乐浪郡（《资治通鉴》卷八八，建兴元年条）。不过在乐浪故地发现的4世纪和5世纪初的带有[[东晋|东晋]]年号和官衔的汉人墓铭和砖铭表明，313年後到5世纪初，还有一定数量的汉人居住在朝鲜半岛北部并继续奉东晋正朔。

== 朝鮮半島早期郡縣 ==
在四郡設立之前，汉武帝[[元朔|元朔]]年間曾於朝鮮半島東部的穢貃地區設立[[蒼海郡|蒼海郡]]，由於記載簡略，其建制情況尚未得到確切考定。

== 對朝鮮半島的有效管轄 ==
{{中国东北历史}}

中国王朝[[漢朝|漢朝]]、公孫氏、[[曹魏|曹魏]]和[[西晋|西晋]]對这一地区的先后統治，一直持續到公元[[四世紀|四世紀]]初[[晋朝|晋朝]]陷於[[永嘉之祸|永嘉之祸]]，才被[[高句麗|高句麗]]和百济所取代。[[前燕|前燕]]慕容氏是最后一次给高句丽以巨创的中国地方政权。342年冬，[[慕容皝|慕容皝]]毁高句丽[[丸都城|丸都城]]，不过并不能恢复以前对朝鲜半岛北部的控制，只好接受高句丽的臣服。

[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]發掘的[[安鹤宫|安鹤宫]]遗址被認為是平壤的早期建築。而吉林[[集安|集安]]的将军坟被認為是长寿王的陵墓，即承认长寿王死后又归葬“故国”，这是一种可能，尽管无文献记载。

另外还有一种可能，即长寿王迁都之平壤並非近日之平壤。427年，高句丽长寿王迁都平壤很有可能不是今日大同江畔的平壤，而是國內城衛城或者和國內同一个城。今日之平壤，為586年（[[隋文帝|隋文帝]][[開皇|開皇]]六年）平原王遷都到最後一個首都長安城，新唐書認為長安城就是平壤即[[樂浪郡|樂浪郡]]，於是國內城附近的平壤就與今日之平壤混淆。
