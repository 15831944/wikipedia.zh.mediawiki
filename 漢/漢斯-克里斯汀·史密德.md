'''漢斯-克里斯汀·史密德'''（{{lang-de|Hans-Christian Schmid}}）是一位德國電影導演。

== 生平 ==
漢斯-克里斯汀·史密德曾以《{{link-de|曙光乍洩|Lichter}}》(2003)、《{{link-de|靈界線 (電影)|Requiem (Film)|靈界線}}》(2006)、《{{link-de|控訴風暴|Sturm (2009)}}》(2009)與《{{link-de|週末回家|Was bleibt (2012)}}》(2012)四度進入[[柏林影展|柏林影展]]正式競賽單元<ref>{{cite web|url=https://www.berlinale.de/en/archiv/jahresarchive/2003/02_programm_2003/02_Programm_2003.html |title=PROGRAMME 2003|accessdate=9 June 2014 |work=berlinale.de| archiveurl= https://web.archive.org/web/20131109180214/https://www.berlinale.de/en/archiv/jahresarchive/2003/02_programm_2003/02_Programm_2003.html| archivedate=9 June 2014 <!--DASHBot-->| deadurl= no}}</ref><ref>{{cite web|url=http://www.berlinale.de/en/presse/pressemitteilungen/alle/Alle-Detail_12436.html |title=Press Release, 9th Jan |date=9 January 2012 |accessdate=9 January 2012 |work=berlinale.de}}</ref>，並憑藉《曙光乍洩》與《靈界線》獲得[[柏林影展費比西國際影評人獎|費比西國際影評人獎]]<ref name="PRIZES">{{cite web|url=http://www.berlinale.de/en/archiv/jahresarchive/2003/03_preistr_ger_2003/03_Preistraeger_2003.html |title=PRIZES & HONOURS 2003|accessdate=9 June 2014 |work=berlinale.de| archiveurl=https://web.archive.org/web/20131015120227/http://www.berlinale.de/en/archiv/jahresarchive/2003/03_preistr_ger_2003/03_Preistraeger_2003.html| archivedate=9 June 2014 <!--DASHBot-->| deadurl= no}}</ref><ref>{{cite web|url=http://www.berlinale.de/media/pdf_word/service_7/vor_55_ifb/53_IFB_Awards_2003.pdf |title=Awards announcement 2003 2003|accessdate=9 June 2014 |work=berlinale.de| archiveurl=https://web.archive.org/web/20130516044630/http://www.berlinale.de/media/pdf_word/service_7/vor_55_ifb/53_IFB_Awards_2003.pdf| archivedate=9 June 2014 <!--DASHBot-->| deadurl= no}}</ref>。
