'''肇庆阅江大桥'''，是[[中国|中国]][[广东省|广东省]][[肇庆市|肇庆市]]一条在建的大型公路桥梁，横跨[[西江|西江]]连接[[端州区|端州区]]与[[高要区|高要区]]城区。将是肇庆端州与高要之间'''第三条'''跨西江桥梁，于2017年8月9日竣工通车。建成后有效地提高肇庆市区交通来往西江两岸的效率。

阅江大桥位于端州区与高要区之间，起点位于肇庆市端州区古塔路与端州路交叉路口沿古塔路高架，跨越西江，连接高要区世纪大道。项目规划设计长度为3406米，其中桥梁长度为2922米，主桥的主跨338米，属特大跨径桥梁。

== 建设特点<ref>{{Cite web|url=http://www.shgzw.gov.cn/gzw/detail1.jsp?main_colid=16&top_id=0&main_artid=32384|title=上海市国有资产监督管理委员会|accessdate=2017-03-11|work=www.shgzw.gov.cn|deadurl=yes|archiveurl=https://web.archive.org/web/20170312054524/http://www.shgzw.gov.cn/gzw/detail1.jsp?main_colid=16&top_id=0&main_artid=32384|archivedate=2017年3月12日|df=}}</ref> ==
国内大跨度桥梁领域首用BIM技术

主桥设计为双塔单索面预应力混凝土斜拉桥，宽度33.5米；

结构上采用墩、塔、梁固结，单索面布于桥面中央。

桥梁底部距离江面约33米，钢-混组合结构的主塔高83.8米。

== 周边道路改造<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201608/t20160824_397322.html|title=肇庆市人民政府-阅江大桥主桥将在端州公正路口落地|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref> ==

'''古塔路'''作为连接阅江大桥的主要道路，大桥建成后车流量预计将明显增大，目前的道路状况不能满足未来的通行需求，将对古塔路进行升级改造。

其中，'''星湖大道路口至公正路路口段进行拓宽'''，拆除现有花基及人行道，由现在的双向四车道改造成：'''双向六车道+两侧非机动车道、人行道；'''公正路路口至建设路路口段将架设'''双向四车道主线高架桥'''，地面设'''双向四车道辅路+两侧人行道；'''

原有的'''建设路大花坛拆除，改建为平交，方便各个方向的车辆有序转入上桥匝道，对从匝道下桥的车辆进行疏 导分流。'''

古塔南路路段除了'''架设双向四车道主线高架桥'''，'''地面建设双向四车道辅路'''之外，还将在'''两侧建设上下行匝道桥。'''

== 建设历史 ==
2007年，肇庆西江三桥纳入两地政府的议事日程之中，开展选址工作<ref>{{Cite news|url=http://news.163.com/07/0924/14/3P5ONU0400011229.html|title=肇庆三桥正筹建|last=|first=|date=|work=|accessdate=}}</ref>。

2011年5月，选址确定为定址于端州城区古塔路—高要世纪大道<ref>{{Cite web|url=http://news.sina.com.cn/o/2011-05-06/031622413863.shtml|title=西江三桥定址古塔桥位|accessdate=2011-05-06|author=|date=|publisher=}}</ref>。

2011年6月，启动征名程序<ref>{{Cite web|url=http://news.sina.com.cn/o/2011-06-13/041122629218.shtml|title=肇庆西江三桥万元征名_新闻中心_新浪网|accessdate=2017-03-11|work=news.sina.com.cn}}</ref>，并与11月完成征名程序，收到1103份提名，定名“阅江大桥”<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201111/t20111111_310045.html|title=肇庆市人民政府-西江三桥命名为“阅江大桥”  共征集到1103个提名|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref>。

2011年12月，确定桥址、桥型，完成工程立项<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201112/t20111224_310561.html|title=肇庆市人民政府-阅江大桥建设加紧推进项目桥址、桥型已定，并完成市级立项|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref>。

2012年3月，完成岩土工程初步勘察<ref>{{Cite web|url=http://news.163.com/12/0319/02/7SU737I400014AED.html|title=肇庆西江三桥完成岩土工程初步勘察(图)_网易新闻中心|accessdate=2017-03-11|last=网易|work=news.163.com}}</ref>。

2012年5月，开始地质详勘工作<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201206/t20120615_312038.html|title=肇庆市人民政府-阅江大桥开展地质详勘|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref>。

2012年8月，确定工程中标人，由上海市第一市政工程有限公司，以750,729,402元承揽工程施工<ref>{{Cite web|url=http://www.zqjs.gov.cn/Article.asp?ArtID=2147|title=肇庆市阅江大桥工程施工招标中标候选人的公示|accessdate=2017-03-11|work=www.zqjs.gov.cn|archiveurl=https://web.archive.org/web/20121024162714/http://www.zqjs.gov.cn/Article.asp?ArtID=2147|archivedate=2012-10-24|deadurl=yes}}</ref>。

2012年11月，肇庆市纪委出台《肇庆市阅江大桥工程建设实施方案》，防范工程中的廉政风险<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201211/t20121122_313341.html|title=肇庆市人民政府-阅江大桥建设有序推进|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref>。

'''2012年10月，工程正式动工。'''

2012年11月，完成前期筑坝工程，进入打桩工作阶段<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201211/t20121105_313073.html|title=肇庆市人民政府-阅江大桥 建设进展顺利|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref>。

2013年3月，江滨堤防洪闸门开始施工，同年4月完工<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201303/t20130321_315032.html|title=肇庆市人民政府-阅江大桥应急防洪闸紧张施工|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref>。

2013年11月，北岸房屋征收补偿方案出台<ref>{{Cite web|url=http://zwgk.zhaoqing.gov.cn/zq116/201402/t20140217_231209.html|title=欢迎光临肇庆市人民政府！|accessdate=2017-03-11|work=zwgk.zhaoqing.gov.cn}}</ref>。

2013年12月，进行护筒跟进、溶洞补勘探作业，北岸开始打桩<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201312/t20131205_318760.html|title=肇庆市人民政府-水位较低有利施工 阅江大桥加紧建设|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref>。

2014年4月，38号主墩进行“孤岛”施工。截止4月，阅江大桥南侧立起了12个承台，39#主墩的桩基已经打好，套上了约7.2米高，截面600多平方米的钢 套箱，之后将把套箱沉入水中，注入水泥混凝土。而北侧水中近岸的28#、29#墩桩基完成4根，正在进行的桩基 占水中桩基总量的55%。项目今年已完成投资约7000万元，占年度计划完成投资3亿元的23%，项目开工以来已累计完成投资2.8亿元<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201404/t20140411_320444.html|title=肇庆市人民政府-阅江大桥北侧主墩进行“孤岛”施工|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref>。

2014年9月，南岸39号主墩出水，进入浇筑程序；北岸进行33-38号桩基工程<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201409/t20140918_322892.html|title=肇庆市人民政府-阅江大桥南侧主墩出水  北侧正在开展桩基施工|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref>。

2014年10月，水下桩基全部完工，进行桥墩施工<ref>{{Cite web|url=http://www.xjrb.com/2014/1016/231985.shtml|title=阅江大桥进行桥墩工程施工_肇庆新闻_肇庆_西江网|accessdate=2017-03-11|work=www.xjrb.com}}</ref>。

2014年12月，北岸端州区古塔路沿线房屋开始清拆<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201412/t20141216_324153.html|title=肇庆市人民政府-阅江大桥北岸开始清拆|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref>。

2015年5月，受洪水困扰，于5月23日暂停施工，后于28日复工<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201505/t20150529_326762.html|title=肇庆市人民政府-西江水位下降  阅江大桥复工忙|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref>。

2015年3月，北岸施工栈道建成启用，建好的北侧栈桥长275米、标高7.0米、桥面宽8米，可行驶施工的“田螺车”、重型吊车、运输大型设备的车辆<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201503/t20150325_325663.html|title=肇庆市人民政府-阅江大桥北侧施工栈桥建成启用|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref>。

2016年2月，主塔基本成型。主塔工程高约83米，由5节混凝土底座和16节钢塔组成，“帆”形中间由12道横梁连接。北岸主塔安装了两节钢塔，南岸安装了10节钢塔<ref>{{Cite web|url=http://www.zhaoqing.gov.cn/xwzx/zqyw/201602/t20160221_367089.html|title=肇庆市人民政府-阅江大桥施工忙 主塔显露雏形|accessdate=2017-03-11|work=www.zhaoqing.gov.cn}}</ref>。
