'''赵孟{{ruby|頫|ㄈㄨˇ}}'''<ref>“頫”，[[現代標準漢語|普通话/国语]]中读作“斧”，[[汉语拼音|汉语拼音]]f{{Unicode|ǔ}}，[[注音符号|注音符号]]ㄈㄨˇ。“頫”在此处是“俯”字的[[异体字|异体字]]。</ref>（{{bd|1254年||1322年|catIdx=Z赵}}），[[表字|字]]'''子昂'''，[[号|号]]'''松雪道人'''，[[别号|别号]]'''鸥波'''、'''水精宫道人'''等。[[中国|中国]][[吴兴区|吴兴]]（今[[浙江省|浙江]][[湖州市|湖州]]）人，宋室后代。[[元朝|元代]]官員，书画家。其妻為元朝畫家、詩人[[管道昇|管道昇]]。元朝畫家[[王蒙_(画家)|王蒙]]之外祖父。

==生平==
赵孟頫出身于[[宋朝|宋朝]]宗室（[[赵匡胤|宋太祖]]第十一世孫，宋太祖第四子秦王[[趙德芳|趙德芳]]後代），[[宋孝宗|宋孝宗]]的兄长[[赵伯圭|赵伯圭]]的玄孙。宋亡後，辞官返回故乡吴兴闲居。

至元23年（1286年）经[[程鉅夫|程鉅夫]]推薦出仕[[元朝|元朝]]，授[[刑部|刑部]][[主事|主事]]，於北方遊宦十年。至元末到大德初，仕宦江南。累官至[[翰林学士|翰林学士]][[承旨|承旨]]，[[荣禄大夫|荣禄大夫]]，世稱「趙承旨」。死后晋封魏国公，谥文敏。

赵孟頫在诗、书、画、印上皆有很高造诣。诗作有《松雪斋集》。书法上精通行书、楷体，独创“赵体”，对后代书法[[艺术|艺术]]影响很大；[[篆刻|篆刻]]以“[[圆朱文|圆朱文]]”著称；画法上也有独创性，首次提出书画用笔相同的理论。

=== 朋友 ===
*[[高克恭|高克恭]]，[[色目人|色目人]]畫家，趙孟頫和其友誼最為篤厚，[[至元|至元]]23年（1286年）初到[[元大都|大都]]任官，便與其過從。此後高克恭受其影響開始習畫，成為天才型畫家<ref>衛欣，〈高克恭與趙孟頫交往略考〉，《美與時代》2008年第一期，86-88頁。</ref>。曾與趙孟頫共同創作《奇石古木圖》等畫<ref name="九州四海風雅同">蕭啟慶，《九州四海風雅同：元代多族士人圈的形成與發展》，台北:聯經出版事業公司，2012年6月27日 出版，259頁。</ref>，並於至元末到[[大德|大德]]初年，一同仕宦江南。
*[[鮮于樞|鮮于樞]]，書法家。

== 文人畫 ==
趙孟頫、高克恭、李衎（音同看）與商琦，並稱元初四大家。趙孟頫至大都任官後，吸收北方「用筆簡率」的古意畫風，師法唐[[王維|王維]]、五代[[董源|董源]]的畫風，創造出《[[鹊华秋色图|鹊华秋色图]]》與《水村圖》<ref>蕭啟慶，《元代的族群文化與科舉》，台北:聯經出版事業公司，2008年1月28日，初版，頁44。</ref>。開啟以「寫意」為主的[[文人畫|文人畫]]風，集前代大成，不拘風格。反璞歸真，講求平淡自然<ref>蕭啟慶，《元代的族群文化與科舉》，台北:聯經出版事業公司，2008年1月28日，初版，頁48。</ref>。迄今为止没有证据显示他曾为[[忽必烈|忽必烈]]或宫中的蒙古权贵画过画，并不是严格意义上的[[宫廷画家|宫廷画家]]<ref name="高居翰">{{cite book|title = 隔江山色：元代绘画 |accessdate = 2012-08-06|author =高居翰（James Cahill） |date =2009-8 |publisher = 三联书店 |language = zh-hans|page = 25 |isbn=978-7-108-03010-8 }}</ref>。

赵孟頫的画作题材广泛：山水、人物、动物、花鸟、竹石无所不有，其后各代几乎无人能超过他。他前期的画作设色独到，即“绚丽之极，仍归自然”<ref>清初“六大家”之一的恽南田推崇赵孟頫的设色：“惟赵吴兴洗脱宋人刻画之迹，运以虚和，出之妍雅，秾纤得中，灵气敞恍，愈浅淡愈见秾厚，所谓绚烂之极仍归自然，画法之一变也”。</ref>。后期多作淡墨画，近乎白描。绘画作品有《[[鹊华秋色图|鹊华秋色图]]》。赵孟頫的儿子赵雍、夫人[[管道昇|管道-{昇}-]]皆能作画，元代[[画家|画家]][[王蒙_(画家)|王蒙]]是他的外孙。

== 书法 ==
趙孟頫、[[鮮于樞|鮮于樞]]與[[鄧文原|鄧文原]]，並稱元初三大家。趙孟頫以行書、楷書最為著名，他主張遵從古法，認為書法應用筆為上，所以勤摹古人，探究筆法。主張將「晉書的風情神韻化入精謹森嚴的唐法」<ref name="元代的族群文化與科舉">頁45。</ref>。他在中國書法史中有兩大貢獻，其一是振興章草，章草的復興雖不能歸功他一人，但趙的影響力卻不可小視，當時書法家多數受他影響，愛寫章草。其二是振興小楷，在宋人當中，堪稱用力最勤<ref >{{cite book|title=中國古代文化知識趣談|author=馬重奇|date=1994年第二版|publisher=導師出版社|language=zh-hant|page=102|isbn=962-397-135-4}}</ref>。

趙孟頫的书法深受[[东晋|东晋]]书法家[[王羲之|王羲之]]的影響，《[[行書右軍四事|行書右軍四事]]》卷可以看出趙孟頫對[[王羲之|王羲之]]的敬仰。

== 影響 ==
=== 同代人 ===
*[[高克恭|高克恭]]，[[色目人|色目]]畫家中享名最高，受品題最多者<ref>萧启庆，《九州四海风雅同：元代多族士人圈的形成与发展》，台北:联经出版事业公司，2012年6月27日 出版，259頁。</ref>。趙孟頫和其友誼最為篤厚，至元23年（1286年）高克恭初到大都任官，便與其過從。此後受其影響開始習畫，成為天才型畫家<ref>衛欣，〈高克恭與趙孟頫交往略考〉，《美與時代》2008年第一期，86-88頁。</ref>。

=== 明 ===
