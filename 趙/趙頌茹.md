'''趙頌茹'''（{{lang|en|Chiu Chung Yu}}，{{bd|1982年|7月24日|catIdx=Z趙}}），本名'''趙梓淇'''，原名'''趙潤敏'''，英文名'''Yu'''，[[香港|香港]][[模特兒|模特兒]]、[[歌手|歌手]]、[[藝人|藝人]]，暱稱'''阿Yu'''。由於相貌酷似女星[[鄭秀文|鄭秀文]]，又有「翻版Sammi」之稱。<ref name="hk.l_翻版Sa">{{Cite web |title=翻版Sammi衰黑口黑面|author= |work=Apple Daily 蘋果日報 |date= |access-date=27 September 2018 |url= https://hk.lifestyle.appledaily.com/nextplus/magazine/article/20020725/2_2732453/%E7%BF%BB-%E7%89%88-Sammi-%E8%A1%B0-%E9%BB%91-%E5%8F%A3-%E9%BB%91-%E9%9D%A2 |language= |quote=}}</ref>

== 生平 ==
因貌似著名歌手[[鄭秀文|鄭秀文]]的趙頌茹，有「小鄭秀文」之稱，当年曾参加「Elite Model Look」比赛，並得到季军，之后被唱片公司发掘，邀请试音，签约红音乐唱片，成为公司第一位力捧的新人，2002年推出处女大碟，後來签约英皇。趙頌茹後來淡出娛樂圈，後來轉戰時裝記者，其後轉為形象及設計顧問。

趙頌茹與[[周永恆|周永恆]]在2010年1月28日结束6年爱情长跑，2人同属教徒，於教会註册结婚。当时教友与圈內好友，好似[[关心妍|关心妍]]、[[高皓正|高皓正]]、[[张致恆|张致恆]]、[[徐天佑|徐天佑]]、[[裕美|裕美]]、[[梁雨恩|梁雨恩]]等，均有到场观礼。周永恆仲自弹自唱《Wonderful Tonight》送给太太，令其泣不成声。

周永恆和趙頌茹於婚前却发现周永恆脑生瘤，心情跌入谷底。但是兩人执手同歷患难，更诞下两个女儿。周永恆坦言自2010年由结婚到当爸爸，每件事都来得太快，虽然不习惯，亦添上压力，但肯定自己学会很多。

2014年，周永恆被指有婚外情，早已知丈夫有婚外情的趙頌茹，一直努力維繫這段婚姻。兩周前，即發生[[家暴|家暴]]事件前兩日，趙頌茹曾經上過周永恒公司，要求同對方晚膳，但周拒絕。有指當時趙頌茹情緒激動，向周掉茶杯，激動離去，周永恒追出去。其公司職員向周刊透露指大家對此情此景毫不奇怪，因為周永恒兩夫妻時常在公司吵鬧，只是萬料不到原來背後更發生家暴。<ref>[http://hk.apple.nextmedia.com/realtime/breaking/20140830/52847525?top=24h 周永恆打老婆 趙頌茹報警拉老公]蘋果日報 (香港)2014年8月30日</ref>

2014年8月趙頌茹向警察報案指遭到丈夫周永恆襲擊，導致受傷入院，及後周永恆被揭發出軌。<ref>[http://www.epochtimes.com/b5/15/9/2/n4518183.htm  趙頌茹獨自撫養兩女 寧慳自己不慳女兒]</ref><ref>[http://hkm.appledaily.com/detail.php?guid=19966681&category_guid=462&category=daily&issue=20170323   趙頌茹獨力養女，不用贍養費]</ref><ref>[https://m.mingpao.com/pns/dailynews/web_tc/article/20170409/s00016/1491675045658  34歲趙頌茹27歲廖子妤 新片扮演母女]</ref>

襲擊案件令周被裁定罪名成立，因獲趙頌茹同意，裁判官准周自簽港幣3,000元擔保及守行為18個月。裁判官表示，家暴是嚴重罪行，被告應該愛護妻子和孩子，勸喻周永恒日後面對爭執時，應保持冷靜，不可使用暴力，假若再犯將會嚴懲。
 
周永恆於守行為期間，到趙頌茹的娘家鬧事，並以短訊恐嚇殺死妻兒及自殺，結果被控普通襲擊罪成，案件在2015年1月30日於[[屯門裁判法院|屯門裁判法院]]宣判，周被判監兩個月，緩刑18個月。
 
母兼父職的趙頌茹2015年為香港一間美容院做廣告代言，並獲得不少護膚品、美甲、鞋履、動漫展覽、教育中心、汽車等客戶和品牌的工作機會，她亦在一些活動中擔任[[模特兒|模特兒]]。
 
2015年底，攜同兩個女兒到[[台灣|台灣]][[花蓮|花蓮]]拍攝[[香港有線電視|有線電視]]節目《帶阿B去旅行》。翌年她接受國際醫美品牌的代言邀請並拍攝以「女人要靠自己」為題的短片。2016年底創立個人品牌售賣自家設計服裝。
