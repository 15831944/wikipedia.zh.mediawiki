'''趙曉音'''，[[中華民國|中華民國]][[基督教|基督教]][[新教|新教]][[牧师|牧師]]，[[新北市|新北市]]'''泰山幸福教會'''主任牧師<ref>{{Cite web|url=http://www.cdn.org.tw/News.aspx?key=9709|title=「教育」不僅是傳授知識 更蘊含人格培養、帶動國家進步的深層意義！ 倘若不慎走歪 賠上的也許是家庭和國家未來|accessdate=2017-04-26}}</ref>，反同性婚姻運動者，曾任'''[[下一代幸福聯盟|下一代幸福聯盟]]'''(下福盟)法務<ref>{{Cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20161229/1023461/|title=「束帶」牽緣分？反同牧師獲同婚團體聲援|author=|date=|work=|newspaper=蘋果日報|accessdate=2017-04-26|language=zh-TW}}</ref>，惟在好友[[四叉貓|四叉貓]]與其不同陣營起衝突後，宣布退出下福盟。<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/2024249|title=自責害四叉貓受暴力對待   牧師退出反同陣營 - 生活 - 自由時報電子報|accessdate=2017-04-26}}</ref>

因任職於五股禮拜堂時，為少年「進行釋放禱告」之影片遭上傳，遭網友以「驅魔」為題瘋傳而知名。<ref>{{Cite news|url=http://news.ltn.com.tw/news/society/breakingnews/1322660|title=電玩「邪靈」附身？  少年被「驅魔」全身扭曲嘶吼 - 社會 - 自由時報電子報|accessdate=2017-04-26}}</ref> 趙曉音曾為此多次澄清，並蒐證揚言要對網友提告。<ref>{{Cite news|url=http://www.ettoday.net/news/20150520/509173.htm|title=戰神出去！少年哀號掙扎　牧師駁驅魔：是釋放禱告|author=|last=https://www.facebook.com/ETtoday|date=|work=|newspaper=ETtoday東森新聞雲|accessdate=2017-04-26|language=zh-TW}}</ref>

== 生平 ==
'''趙曉音'''自幼篤信基督教，[[銘傳商專|銘傳商專]]大眾傳播科畢業後，<ref>{{Cite web|url=http://www.bolbookstore.com/front/bin/ptdetail.phtml?Part=9789868771321|title=靈糧供應站 -- 走出黑暗|accessdate=2017-04-26|work=www.bolbookstore.com|language=zh-tw}}</ref>曾任記者、法務專員。本希望成為律師，但後來投身傳道，成為牧師。<ref>{{Cite news|url=http://www.appledaily.com.tw/appledaily/article/headline/20150520/36560334/|title=「驅魔」影片瘋傳 牧師告網友|author=|date=|work=|newspaper=蘋果日報|accessdate=2017-04-26|language=zh-TW}}</ref>

2015年，因其在五股禮拜堂，為沉迷網路遊戲的少年進行釋放禱告的錄影，遭網友以「驅魔」為題上傳而知名。

==  反同性婚姻 ==
趙曉音曾多次在臉書上呼籲反對民法修改同性婚姻，<ref>{{Cite web|url=https://opinion.udn.com/opinion/story/6068/1670241|title=如果同性戀並非天生，那異性戀也不是：四個「你沒想到你自己」的謬誤|accessdate=2017-04-26|author=|date=|work=鳴人堂|publisher=|language=zh-TW}}</ref> 並且投身反對同性婚姻運動，擔任下一代幸福聯盟的法務，1226立院陳情活動負責人。<ref>{{Cite news|url=http://www.coolloud.org.tw/node/87276|title=【記者會】下一代幸福聯盟提告立委助理|date=2016-12-28|newspaper=苦勞網|accessdate=2017-04-26|language=zh-hant}}</ref> 並因此遭到警方以違反《[[集會遊行法|集遊法]]》26條移送，[[臺灣臺北地方法院檢察署|北檢]]調查後做出不起訴處分。<ref>{{Cite news|url=http://www.storm.mg/article/266636|title=「無積極煽動」 趙曉音帶領反同團體陳抗涉違集遊法 北檢不起訴-風傳媒|accessdate=2017-05-21|language=zh-TW}}</ref>

==  與同志互動 ==
趙曉音雖然為反同婚運動領導者之一，但自2016年12月3日之下一代幸福聯盟於凱道舉辦之反同婚活動開始，便與知名挺同婚同志[[四叉貓|四叉貓]]互動頻繁，趙稱其為「可愛的、配合大會的同志」。<ref>{{Cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20161205/1004928/|title=給我網路和鍵盤　四叉貓嗆護家盟：我殺傷力最強|author=|date=|work=|newspaper=蘋果日報|accessdate=2017-04-26|language=zh-TW}}</ref> 自此之後，兩人時常於反同婚活動場合互動，甚至四叉貓曾自備沙發床一類誇張之道具，在活動中一起敷面膜，自嘲為「護膚盟」。<ref>{{Cite news|url=https://udn.com/news/story/1/2195180|title=再次亂入反同團體 四叉貓自備面膜稱「護膚盟」|author=|last=聯合新聞網|date=|work=|newspaper=即時|accessdate=2017-04-26}}</ref> 2016年12月28日，趙曉音遭束帶捆綁之事提告，四叉貓亦路過綁上束帶聲援。<ref>{{Cite news|url=http://news.ltn.com.tw/news/society/breakingnews/1931638|title=挺同、反同和解？ 四叉貓、牧師「串一起」提告他 - 社會 - 自由時報電子報|accessdate=2017-04-26}}</ref>

除在反同婚活動場合，兩人私下也有聚餐互動，趙曉音因而遭到部分反同婚人士指稱為奸細。<ref>{{Cite news|url=http://www.storm.mg/article/230378|title=牧師趙曉音遭反同陣營批為「奸細」四叉貓：神啊～我有罪-風傳媒|accessdate=2017-04-26|language=zh-TW}}</ref>

== 退出下一代幸福聯盟 ==
2017年3月22日，四叉貓在[[麥當勞|麥當勞]]台北市某分店巧遇赴[[疾管署|疾管署]]抗議後之[[滿天星素人連線|滿天星素人連線]]人士，因拿出手機自拍引發衝突，遭控跟蹤。後[[台北市政府警察局|台北市政府警察局]]忠孝東路派出所出面協調，又因被誤傳引發爭議，使忠孝東路派出所所長林俊燁公開蒐證影片澄清<ref>{{Cite news|url=http://news.ltn.com.tw/news/society/breakingnews/2013696|title=「四叉貓」、「滿天星」互槓 警無端被誣|author=|date=|work=|newspaper=自由時報|accessdate=2017-03-23|language=zh-TW}}</ref>。 23日，人在美國的[[趙曉音|趙曉音]]於臉書發文聲援四叉貓，指四叉貓「只會跟蹤食物」<ref>{{Cite news|url=http://www.cna.com.tw/news/asoc/201703235019-1.aspx|title=四叉貓跟反同婚團體起爭執 趙曉音臉書幫聲援[影]|author=|date=|work=|newspaper=中央通訊社|accessdate=2017-03-23|language=zh-TW}}</ref>。  此一聲援引發[[滿天星素人連線|滿天星素人連線]]之不滿，揚言要讓趙曉音「永遠閉嘴」，趙憤而提告<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/2017747|title=內鬨？ 自稱反同牧師自稱收到威脅|author=|date=|work=|newspaper=自由時報|accessdate=2017-03-27|language=zh-TW}}</ref>。3月31日，趙曉音於臉書發表聲明，退出[[下一代幸福聯盟|下一代幸福聯盟]]。趙曉音在接受[[新頭殼|新頭殼]]電話訪問時表示，因其與四叉貓的打賭才引發麥當勞之爭議，為了對朋友負責且避免反同團體間之對立，因而退出下福盟<ref>{{Cite web|url=https://newtalk.tw/news/view/2017-04-01/83869|title=趙曉音退出反同陣營  下福盟惋惜、 四叉貓樂見|accessdate=2017-04-01|author=|date=|work=新頭殼|publisher=|language=zh-TW}}</ref>，並表示未來會把重心放在監督性平教材上，會更多傾聽同志的聲音<ref>{{Cite web|url=http://www.storm.mg/article/243256|title=牧師趙曉音退出反同陣營下福盟 原因竟跟四叉貓有關|accessdate=2017-04-02|author=|date=|work=風傳媒|publisher=|language=zh-TW}}</ref>。
