'''趙樸初'''（{{bd|1907年|11月5日|2000年|5月21日|catIdx=Z趙}}），[[安徽省|安徽省]][[太湖县|太湖县]]人，[[中国佛教|中国佛教]]领袖、作家、诗人、书法家<ref>{{Cite web|title=中国佛教协会官方网-赵朴初居士与金陵刻经处|url=http://www.chinabuddhism.com.cn/special/jlkjc/zpc111/2016-07-04/10790.html|work=www.chinabuddhism.com.cn|accessdate=2019-04-06}}</ref>，被佛教界尊称为「赵朴老」。曾担任[[中国人民政治协商会议第九届全国委员会|中国人民政治协商会议第九届全国委员会]]副主席、[[中国民主促进会|中国民主促进会]]中央名誉主席、[[中国佛教协会|中国佛教协会]]会长，著名的社会活动家、杰出的爱国宗教领袖、[[中国共产党|中国共产党]]的亲密朋友。<ref>{{Cite web|title=赵朴初同志生平|url=http://www.people.com.cn/GB/channel1/10/20000531/83464.html|work=www.people.com.cn|accessdate=2019-04-06}}</ref>

== 生平 ==
赵朴初出身于[[太湖县|太湖县]]四世[[翰林|翰林]]趙家，五世祖[[趙文楷|趙文楷]]为[[嘉庆|嘉庆]]元年[[状元|状元]]，元祖[[趙畇|趙畇]]、曾祖[[趙繼元|趙繼元]]、伯祖（祖父之兄）[[趙曾重|趙曾重]]皆为[[进士|进士]]出身。1926年就读于[[东吴大学_(苏州)|苏州东吴大学]]。

1928年起，赵朴初担任任上海江浙佛教联合会秘书，上海佛教协会秘书，“佛教净业社”社长，[[四明银行|四明银行]]行长。

1936年，趙樸初在上海成立[[中国佛教护国和平会|中国佛教护国和平会]]，参加[[抗日|抗日]]救亡活动。1937年-1945年的[[中国抗日战争|中国抗日战争]]期间，趙樸初先后担任[[上海文化界救亡协会理事|上海文化界救亡协会理事]]，[[上海市佛教协会|上海市佛教协会]]秘书、主任秘书，及[[上海慈联救济战区难民委员会|上海慈联救济战区难民委员会]]常务委员、兼难民收容股主任，[[上海净业社儿童教养院|上海净业社儿童教养院]]副院长，[[上海少年村|上海少年村]]村长等职务。1939年，参加民主促进运动。

1945年，中国抗日战争胜利后，参与发起组织[[中国民主促进会|中国民主促进会]]，担任上海分会副主任。

1946年后，任上海安通运输公司、上海华通运输公司常务董事、总经理。

1949年，任上海临时联合救济委员会总干事，[[中国人民保卫世界和平委员会|中国人民保卫世界和平委员会]]常委、副主席，[[中国亚非团结委员会|中国亚非团结委员会]]常委。1949年10月，[[中华人民共和国|中华人民共和国]]成立，趙樸初代表上海佛教界，参加[[中国共产党|中国共产党]]在北京首次召开的[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]。1950年，趙樸初任[[华东军政委员会|华东军政委员会]]民政部副部长、人事部副部长，[[上海市人民政府|上海市人民政府]]法律委员会副主任。1952年10月，在北京召开的[[亚洲及太平洋区域和平会议|亚洲及太平洋区域和平会议]]上，趙樸初代表中国佛教界，将一尊古铜佛像赠送日本佛教界。

1952年11月4日和5日，趙樸初与[[虚云和尚|虚云和尚]]、[[喜饶嘉措|喜饶嘉措]]、[[噶喇藏|噶喇藏]]、[[圆瑛|圆瑛]]、[[柳霞·土登塔巴|柳霞·土登塔巴]]等二十人在[[广济寺_(北京市)|北京广济寺]]召开发起人会议，发起组织[[中国佛教协会|中国佛教协会]]。[[中共中央统战部|中共中央统战部]]部长[[李维汉|李维汉]]出席致词。会中通过发起书，决定由趙樸初、柳霞·土登塔巴、[[丹巴日杰|丹巴日杰]]、[[巨赞|巨赞]]、[[周叔迦|周叔迦]]、[[郭朋|郭朋]]、[[李一平|李一平]]组成[[中国佛教协会筹备处|中国佛教协会筹备处]]，趙樸初担任筹备处主任。经过半年的筹备，1953年5月30日至6月3日，在北京广济寺举行中国佛教协会成立会议。出席会议者包括[[汉族|汉族]]、[[藏族|藏族]]、[[蒙古族|蒙古族]]、[[傣族|傣族]]、[[满族|满族]]、[[苗族|苗族]]、[[裕固族|撒里维吾尔]]等七个民族的[[活佛|活佛]]、[[喇嘛|喇嘛]]、[[法師_(佛教)|法师]]、[[居士|居士]]代表一百二十人。中央统战部李维汉部长到会讲了话。会议中通过了《[[中国佛教协会章程|中国佛教协会章程]]》和有关决议；会中推选[[第十四世達賴喇嘛|达赖喇嘛]]、[[班禅额尔德尼·确吉坚赞|班禅额尔德尼·确吉坚赞]]、[[虚云法师|虚云法师]]、[[查干葛根|查干葛根]]为中国佛教协会的名誉会长，选举[[圆瑛|圆瑛]]为会长，趙樸初与喜饶嘉错、[[公德林|公德林]]、[[晋美吉村|晋美吉村]]、[[能海法师|能海法师]]、[[当喇藏|当喇藏]]、[[祜巴|祜巴]]、[[阿旺嘉错|阿旺嘉错]]为副会长。会中并产生了中国佛教协会理事和中国佛教协会常务理事会，及以趙樸初为秘书长，巨赞、周叔迦、郭朋为副秘书长。此后趙樸初的副会长、秘书长职务，连任至1980年。

「[[文化大革命|文化大革命]]」期間[[中国佛教|中国佛教]]受极严重摧残，中国佛教协会也停止活动。1978年[[中共十一届三中全会|中共十一届三中全会]]后，在宗教政策方面，采行「文革」前的温和路线，有限度的容许宗教存在。1978年后，停止活动十年之久的[[中国佛教|中国佛教]]也逐渐复苏。1980年12月16日至12月23日，中国佛教协会召开了第四届全国代表会议。趙樸初在工作报告中称：
{{cquote|
中国佛教协会第三届全国代表会议，是一九六二年二月召开的，到现在，已经历时十八年又九个月了。在这十八年期间，我们国家经过了一个剧烈的历史大动荡时期，走过了一段艰难曲折的道路。在党的领导下，粉碎了[[林彪|林彪]]、[[江青|江青]]反革命集团，拨乱反正，祖国大地，又重见光明，今天，我们在渡过十八年的沧桑岁月以后，大家聚会一处，济济一堂，共同举行这个全国性佛教代表会议，抚今追昔，感慨无量……
}}

在这一次大会中，改选了中国佛教协会理事会，趙樸初当选为会长，对劫后中国佛教的复兴，贡献甚多。

1980年以后，趙樸初除了领导中国佛教协会外，还担任[[中国书法家协会|中国书法家协会]]副主席，[[中国民主促进会|中国民主促进会]]中央常委、副主席、[[中央参议委员会|中央参议委员会]]主任，及[[中国民主促进会|中国民主促进会]]中央名誉主席，第六、七、八届[[全国政协副主席|全国政协副主席]]。

趙樸初在90岁以后健康状况不佳。2000年5月21日在北京病逝，享年93岁。

== 书法 ==
赵朴初书法以[[行楷|行楷]]最擅长，脱胎于[[李北海|李北海]]、[[苏东坡|苏东坡]]，字的体势向右上方倾斜，结构严谨、笔力劲健而又有种雍容宽博的气度，隐隐透出一种佛家气象。<ref>{{Cite web|title=新民晚报数字报-朴老翰墨有慈心|url=http://xmwb.xinmin.cn/html/2014-11/15/content_25_2.htm|work=xmwb.xinmin.cn|accessdate=2019-04-06}}</ref>
