==生平==
1984年毕业于综合军事学院。1994年获得敖特根腾格里大学（Отгонтэнгэр Их сургуулийн）硕士学位。2000年获管理学院硕士学位。2013年获管理学院博士学位。2008年起，任[[国家大呼拉尔|国家大呼拉尔]]委员<ref name=zaluu/>。2012年至2016年，先后任[[阿勒坦呼亚格内阁|阿勒坦呼亚格内阁]]、[[赛汗比勒格内阁|赛汗比勒格内阁]]人口与劳动和社会保障部部长<ref name=zaluu/><ref name=sss>{{Cite web|url =http://news.xinhuanet.com/world/2012-08/18/c_112766973.htm|title =蒙古国议会任命13名政府部长|author =|date =2012-08-18|publisher =新华网|language =|accessdate =|quote =}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。自2016年起担任国家大呼拉尔[[民主党_(蒙古国)|民主党]]团团长。2016年当选民主党主席<ref name=zaluu/><ref>{{Cite web |url =  http://www.fmprc.gov.cn/web/zwbd_673032/wshd_673034/t1453643.shtml|title =  驻蒙古大使邢海明会见蒙古民主党主席额尔登|author =  |date = 2017-04-14  |publisher = 中华人民共和国外交部 |language =  |accessdate =  |quote =  }}</ref>。
