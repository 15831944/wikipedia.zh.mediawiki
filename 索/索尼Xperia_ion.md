'''Sony Xperia ion'''是[[日本|日本]]電子公司[[SONY|SONY]]於2012年5月推出的高階[[智慧型手機|智慧型手機]]。（3G版本）但該機已在2012年2月時於[[美國|美國]]與[[AT&T|AT&T]]合作推出[[LTE|4G LTE]]的版本。

==規格<ref>{{cite web |url=http://www.sogi.com.tw/product/productInfo.aspx?pno=8910 |title=存档副本 |accessdate=2012-05-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20120526075534/http://www.sogi.com.tw/product/productInfo.aspx?pno=8910 |archivedate=2012-05-26 }}</ref>==
*通信系統：HSDPA／UTMS／GSM四頻／LTE（僅限於美國版本）
*螢幕：4.55" [[TFT-LCD|Mobile BRAVIA Engine Super LCD]] 1280x720
*處理器：[[Snapdragon|Qualcomm Snapdragon S3 MSM8660, 1.5GHz 雙核心處理器]]
*記憶體：1GB RAM / 16GB ROM
*系統：[[Android_2.3|Android 2.3]]，可升級Android 4.0（Ice Cream Sandwich）及Android 4.1（jelly bean）的軟件升級
*相機：1,200萬畫素Exmor R for Mobile 感光元件鏡頭（有LED閃光燈）(3D拍攝、16倍數位變焦、自動對焦等功能）、130萬畫素前置鏡頭
*NFC
*xLoud、Clearbass
*設計國：{{JPN}}
*産地國：{{CHN}}
*上市日期：2012年5月（{{flagicon|ROC}} [[臺灣|臺灣]]及{{flagicon|HK}} [[香港|香港]]）
