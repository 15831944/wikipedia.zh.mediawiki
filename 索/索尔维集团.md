'''索尔维集团'''（Solvay S.A.，原译“'''苏威集团'''”）是一家总部位于[[比利时|比利时]]首都[[布鲁塞尔|布鲁塞尔]]的跨国性[[化工|化工]]集团，1863年由[[比利时|比利时]][[化学家|化学家]][[欧内斯特·索尔维|欧内斯特·索尔维]]创立。索尔维集团目前在[[布鲁塞尔|布鲁塞尔]]和[[巴黎|巴黎]]的[[纽约泛欧证交所|纽约泛欧证交所]]挂牌上市，在全球55个国家拥有约30,900名员工。公司现任执行总裁是[[让-皮埃尔·柯拉马帝|让-皮埃尔·柯拉马帝]]。 

索尔维集团的产品被广泛应用于各行业领域，旗下90%的销售产品位居全球前三甲。索尔维在[[稀土|稀土]]、[[白炭黑|白炭黑]]、[[工程塑料|工程塑料]]、[[聚酰胺|聚酰胺]]和中间体、[[香料|香料]]及功能化学品、基础化学品、特殊化学品、特种聚合物、新兴生物化学等业务领域占据重要地位。集团2015年全球销售额达124亿[[欧元|欧元]]。

== 历史 ==
[[File:Ernest_Solvay.jpg|thumb]]
[[File:Catalyst_-_2015-08-08_-_Andy_Mabbett_-_02.JPG|thumb]]

1861年，[[欧内斯特·索尔维|欧内斯特·索尔维]]发现了工业化生产[[纯碱|纯碱]]的关键。1863年，他将革新的[[氨碱法|氨碱法]]申请专利，并和弟弟成立了索尔维公司。1865年，第一个纯碱厂在[[比利时|比利时]]古亦埃(Couillet)开始运作。

从1870年起，索尔维开始向全球扩展业务，首先在[[英国|英国]]、[[德国|德国]]、[[俄罗斯|俄罗斯]]和[[美国|美国]]等国家建立了工厂。至2009年，索尔维已成为活跃于[[化工|化工]]、[[塑料|塑料]]和[[制药|制药]]等行业的全球性工业集团<ref>[http://fr.slideshare.net/ElFumieros/solvay-historique 苏威集团的发展历史]</ref>。

2009年9月，索尔维将制药业务以52亿[[欧元|欧元]]的价格（其中现金45亿欧元）出售给[[美國雅培|美國雅培]]<ref>[http://www.mofcom.gov.cn/aarticle/i/jyjl/m/200910/20091006587290.html 苏威以52亿欧元将制药业卖给美国雅培]</ref>，2010年2月完成交易。2011年4月，索尔维又出价66亿[[欧元|欧元]]（其中现金34亿欧元）友好收购[[法国|法国]][[罗地亚|罗地亚]]（Rhodia）集团<ref>[http://www.solvayasiapacific.com/EN/News/20110404_SolvaytocreatewithRhodiaamajorplayerinChemicals.aspx 苏威出价66亿欧元收购法国罗地亚]</ref>，并于9月收购成功。这两大交易使索尔维的业务集中于[[化工|化工]]和[[塑料|塑料]]领域。

2012年1月，新索尔维集团在[[巴黎|巴黎]]上市<ref>[http://info.yup.cn/jrtt/39614.jhtml 苏威将于2012年1月23日上市巴黎股市]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，当年9月进入[[巴黎CAC40指数|巴黎CAC40指数]]取代[[标致雪铁龙集团|标致雪铁龙集团]]<ref>[http://roll.sohu.com/20120910/n352692297.shtml 苏威入围巴黎CAC40指数取代标致雪铁龙]</ref>。2013年10月7日，索尔维宣布以13亿[[美元|美元]]收购[[美国|美国]]化工企业Chemlogics，以加强在快速增长的油气行业服务市场的地位。同年10月，索尔维在[[比利时|比利时]][[布鲁塞尔|布鲁塞尔]]庆祝公司成立150周年，[[比利时国王|比利时国王]][[菲利普一世|菲利普一世]]和[[玛蒂尔德_(比利时王后)|玛蒂尔德王后]]<ref>[http://www.zimbio.com/pictures/Nok9hNvJ-A2/Solvay+Celebrates+150th+Anniversary+Brussels/7rVfMtrVXRh/Queen+Mathilde 150周年庆典：比利时王后、国王，苏威集团CEO]</ref>受邀出席典礼。<ref>{{Cite web |url=http://www.solvay.com/en/media/events/2013-10-6-150-celebration-brussels.html |title=150th anniversary celebrations in Brussels |access-date=2013-10-14 |archive-url=https://web.archive.org/web/20131022211943/http://www.solvay.com/en/media/events/2013-10-6-150-celebration-brussels.html |archive-date=2013-10-22 |dead-url=yes }}</ref>

== 索尔维在全球 ==
索尔维是一家大型世界领先化工集团：
* 40％的售额在快速增长国家（[[中国|中国]]、[[拉丁美洲|拉丁美洲]]、[[印度|印度]]等等），43﹪在[[欧洲|欧洲]]，17﹪在[[北美|北美]]。
* 拥有多元化的终端市场：[[消费品|消费品]]：28﹪，[[建筑|建筑]]：17﹪，[[汽车|汽车]]：15﹪，[[电子产品|电子产品]]：7﹪，[[能源|能源]]：5﹪，[[环境|环境]]：4﹪，农业水产：4﹪，造纸：2﹪，其他：18﹪。<ref>[http://reports.icorcom.eu/solvay/ar2011/fr/files/assets/basic-html/page4.html 苏威全球景况简述] {{webarchive|url=https://web.archive.org/web/20131224102733/http://reports.icorcom.eu/solvay/ar2011/fr/files/assets/basic-html/page4.html |date=2013-12-24 }}</ref>

== 在华情况 ==
罗地亚早在20世纪70年代就通过其前身[[罗纳-普朗克公司|罗纳-普朗克公司]]在[[中国|中国]]开展业务，至2002年底在华投资已超过3亿[[美元|美元]]。1994年索尔维入驻[[中国|中国]]市场，成立索尔维投资有限公司开展业务。

索尔维集团目前在中国拥有2800多名员工，17个生产工厂，4个研发/科技中心，其中包括1座位于[[上海|上海]]的全球研发中心。2011年10月，索尔维在[[上海|上海]]建立了绿色产品与工艺国际联合实验室（E2P2L），合作伙伴包括[[华东师范大学|华东师范大学]]和[[法国国家科学研究中心|法国国家科学研究院]]。<ref>[http://news.ecnu.edu.cn/s/192/t/342/ac/95/info44181.htm 华东师范大学新闻网]</ref>

== 可持续发展与创新 ==
索尔维致力推动可持续性改进、创新与卓越运营，集中投资于增值高、耗能低的行业和新产品<ref>[http://www.industrie.com/chimie/solvay-vise-l-excellence-et-la-croissance,43438 索尔维的目标是卓越运营和盈利发展]</ref>。索尔维集团是世界首架[[太阳能|太阳能]][[飞机|飞机]]“[[阳光动力号|阳光动力号]]”项目的第一个参与者，主要提供制造“阳光动力号”所需要的高性能超轻聚合材料。<ref>{{Cite web |url=http://www.solvay.com/EN/About/SolarImpulse/Home.aspx |title=苏威是太阳能飞机最初参与伙伴 |access-date=2012-10-01 |archive-url=https://web.archive.org/web/20120904074115/http://www.solvay.com/EN/About/SolarImpulse/Home.aspx |archive-date=2012-09-04 |dead-url=yes }}</ref><ref>[http://news.xinhuanet.com/air/2012-06/27/c_123337564.htm 新华网 最大太阳能飞机完成首次洲际飞行] {{webarchive|url=https://web.archive.org/web/20131007221233/http://news.xinhuanet.com/air/2012-06/27/c_123337564.htm |date=2013-10-07 }}</ref>
