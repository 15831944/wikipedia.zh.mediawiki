'''索尼 α7'''（型號：ILCE-7）是2013年10月[[索尼公司|索尼公司]]推出的可换镜头[[全画幅|全画幅]]数码相机<ref name="Sony Alpha ILCE-7">[http://www.dpreview.com/news/2013/10/16/sony-announces-first-full-frame-mirrorless-ilc-cameras-a7-and-a7r Sony announces A7 and A7R: first full-frame mirrorless cameras: Digital Photography Review]. Dpreview.com. Retrieved on 2013-10-30.</ref>，使用[[索尼E接环|索尼E接环]]，[[法兰距|法兰距]]18mm。是世界首台全画幅[[微单相机|微单相机]]。

同时推出的还有无低通高像素版本α7R，具备三千六百万像素[[CMOS|CMOS]]传感器。索尼公司在之后也发布了不少近似模具机型，形成了'''α7系列机型'''。

==概述==
索尼公司在2010年推出[[索尼E接环|E卡口系统]]以来，随着系统不断发展成熟，市场接受度逐渐升高<ref>[http://article.pchome.net/content-1690582.html 无反领域占据日本市场大头]—PCHOME</ref>，“微单热”兴起。无论是转接或者利用现有配套镜头，无反机型都显示出[[单反相机|单反相机]]难以比拟的便携度和灵活性。

2012年索尼公司推出的同样基于E卡口系统的[[NEX-VG900|NEX-VG900]]摄像机，即采用135全画幅规格的传感器，从技术上证明了E卡口全幅化的可能性。在α77和α7R推出前，谣言一度猜测相关机型名称可能被定为''NEX-9''<ref>{{Cite web |title=Sony NEX Full-Frame Rumor Roundup |author=DL Cade |date=October 3, 2013 |work=PetaPixel |accessdate=2014-01-11 |url=http://petapixel.com/2013/10/03/sony-nex-full-frame-rumor-roundup/ }}</ref>；但是随着2013年索尼停用[[索尼NEX系列|NEX]]、而启用ILCE产品编号，让这一猜想落空。“ILC”代表''Interchangeable Lens Camera''，而E则代表E卡口<ref>{{cite web |url=http://blog.sony.com/2013/10/a7-faq/ |title=Your α7 and α7R Questions Answered |publisher=blog.sony.com |date=2013-10-17 |accessdate=2013-10-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20131102173050/http://blog.sony.com/2013/10/a7-faq/ |archivedate=2013-11-02 }}</ref>；新机型使用一贯的先锋机型数字“7”作为标识，也有划时代的意义。

虽然 α7 和 α7R 具备全画幅传感器且拥有类似单反的外观，但是其体积和重量仍然可以较轻量级[[全画幅数码单反相机|全画幅数码单反相机]]有优势。即便与APS-C规格中最小的数码单反[[佳能_EOS_100D|佳能 EOS 100D]]相比， α7也仅仅略重474克，体积略大，而厚度仅有100D的2/3<ref>{{cite web |url=http://www.dpreview.com/reviews/canon-eos-100d-rebel-sl1/2 |title=Canon EOS 100D / Rebel SL1 specifications |accessdate=January 9, 2014}}</ref>。 在中国大陆，α7上市后单机稳定约8000元的价格，也是一个非常不错的全画幅机型的选择。

索尼之前推出的E接环镜头全为APS-C规格设计，虽然可以在物理上直接适配α7，但是无法充分利用全画幅传感器。为此，索尼一并推出了全画幅设计的E卡口镜头，今后将使用“FE”字样与普通E卡口镜头进行区分。

==主要参数==
* Exmor HD CMOS 全画幅传感器，两千四百万像素
* TruBlack XtraFine可翻转3"液晶屏，具备92万点显示能力
* 1.3 cm [[电子取景器|电子取景器]]，具备236万点
* 1200区测光
* 内建[[Wi-Fi|Wi-Fi]]与[[NFC|NFC]]功能，可以支持智能设备遥控拍摄
* 支持[[PlayMemories_Camera_Apps|PlayMemories Camera Apps]]
* [[MI热靴|MI热靴]]

==相近型号==
在2013～2014年。索尼以α7同模具发布了一共三款机型：'''α7''' / '''α7R''' / '''α7S''' 。参数类似，仅在传感器关键指标上存在差异。其中：
*α7R的“R”代表"resolution"，具备三千六百万高像素，采用无低通CMOS传感器。
*α7S的“S”代表"sensitivity"，扩展后的感光度覆盖 ISO 50 到 409600，具备4k影像摄录能力（需使用[[HDMI|HDMI]]口外接设备）。
而在2015年，则推出了更新版模具为基础的'''α7 II'''、'''α7R II'''与'''α7S II'''。

==评价与反响==
该相机上市后获得了广泛好评。 [[DxO_Labs|DxOMark]]赞赏α7R为具有最高级全画幅传感器的'''''无反相机''''' ，并给出了和 [[尼康D800|尼康D800]] 相同的95分高分，但比D800E低一分。<ref>http://www.dxomark.com/Reviews/Sony-Alpha-7R-review-Highest-ever-full-frame-image-quality</ref> α7R的姊妹机α7获得了90分，比专业DSLR相机 [[尼康Df|尼康Df]], [[尼康_D4|尼康 D4]] 和 [[索尼_Alpha_99|索尼 Alpha 99]]要高。<ref>{{cite web |url=http://www.dxomark.com/Cameras/Ratings |title=Camera Sensor Ratings by DxOMark |accessdate=December 22, 2013}}</ref> The Verge 在给出8.3分/满分10分的评论中提到“可能几年后我们才会意识到，但当DSLR被专业摄影师摆到合适的位置和全画幅无反主导市场的时候，我们将会感谢α7s作为开启了这一切的相机。”<ref>http://www.theverge.com/2013/12/16/5213600/sony-alpha-a7-a7r-review</ref> EPhotozine将α7列为了5星，同时称赞了α7的价格并说α7是“最近以来可买到的最便宜的数码全片幅照相机”和相比同级别照相机来说更小更轻的照相机。 <ref>http://www.ephotozine.com/article/sony-alpha-7-ilce-7-full-review-23179</ref> 并赞赏了α7是“年度相机”。 <ref>http://www.ephotozine.com/article/ephotozine-s-best-cameras-of-the-year-awards-2013-23514</ref> Imaging Resource 同样也表达了对α7作为“年度相机”的评价。 <ref>http://www.imaging-resource.com/news/2013/12/11/13-best-cameras-for-2013-ir-cameras-and-lenses-of-the-year</ref>

α7是同一时期最为廉价的数码全画幅机型，上市初期即跌到约8000元人民币（中国大陆行货）；在上市一年半的2015年8月，新机价格以跌破6000元（中国大陆行货）<ref>[http://www.mgpyh.com/recommend/6174509/ 历史新低，最便宜全画幅？索尼 SONY A7 ILCE-7 全画幅 微单机身 ￥5849] — 买个便宜货 </ref>。
===存在的问题===
*'''耀斑''' / '''绿帽'''
经过一段时间的使用，有部分用户发现 A7机型 在拍摄含有高光点的场景时，画面会出现不明耀斑。经过更换镜头测试，情况依然存在；而同一时期的 α7R 相比则并没有该现象<ref>[http://www.5i01.cn/topicdetail.php?f=254&t=4135066 A7在强光源下有飞碟光晕是不正常相机] — Mobile01 </ref>。这一现象在爱好者中被称为『'''飞碟'''』（光斑形状类似）或『'''绿帽'''』（呈现绿色倾向）<ref>[http://forum.xitek.com/thread-1356964-1-1-2.html A7所谓的绿帽子产生的原因或具体场景？] — 色影无忌</ref><ref>[http://www.chiphell.com/thread-936121-1-1.html 关于A7的绿帽子光晕问题，已经确定主要是机身问题了] — CHIPHELL</ref>。虽然在两岸的爱好者论坛，如Mobile01、色影无忌、CHIPHELL等多有讨论，但是索尼公司并未正面就此进行回应。
star eating : 超過30sec 星點會被當noise被抹除 

*'''卡口强度'''
α7以及同时期的α7R，其卡口后的卡爪部分，采用塑料材质（官方答复为复合塑料与玻璃纤维），部分用户对此就卡口总成的强度进行质疑。因FE卡口非唯使用原生E口的镜头，在实际中经常使用转接环搭配各式规则镜头进行拍摄，故成为关注焦点<ref>[http://www.chiphell.com/thread-916920-1-1.html A7/R的塑料卡口爪，你们一点不介意？] — CHIPHELL</ref>。

亦有第三方厂商籍此做文章，推出了可自行更换的卡爪附件。而索尼在后续的α7S，以及α7M2、α7RM2上皆采用了金属卡爪设计。

*'''握把与快门钮设计'''
A7的握把设计较为扁平，而快门按钮为直上设计，以人体工程学眼光来看不适合长期握持；相比自己同为微单产品的[[NEX-7|NEX-7]]与[[NEX-6|NEX-6]]亦大为不及。因外壳模具通用，这一问题也出现在α7R与α7S上。因不影响正常成像与长期耐用度，故关注度不如前述问题。

在续作的α7M2上，采用了全新设计的握把，解决了该问题。
