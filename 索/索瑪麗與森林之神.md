{{日本作品|索瑪麗與森林之神|ソマリと森の神様}}，是一部由{{lang|ja|暮石ヤコ}}創作的[[漫畫作品|漫畫作品]]。2015年4月26日，本作於ZENON COMIC的網路漫畫網站「Web Comic Zenyon」上開始連載<ref>{{Cite web |url = https://news.mynavi.jp/article/20150426-a125/ |title = 人外に支配された世界描く本格ファンタジー「ソマリと森の神様」ぜにょんで開始 |author = mynavi |date = 2015-04-26 |publisher = Mynavi Corporation |language = ja |accessdate = 2019-04-14}}</ref>。單行本由[[德間書店|德間書店]]出版，中文版則由[[青文出版社|青文出版社]]代理。

2019年3月，動畫製作組宣佈將本作動畫化，並定於同年秋季播映<ref>{{Cite web |url = https://www.oricon.co.jp/news/2131945/full/ |title = 漫画『ソマリと森の神様』TVアニメ化で今秋放送 主演は水瀬いのり&小野大輔  |author = Oricon News |date = 2019-03-23 |publisher = oricon ME inc. |language = ja |accessdate = 2019-04-14}}</ref>。

== 故事簡介 ==
這個世界被異形生物所支配，而人類則處於食物鏈的最底層，並面臨滅絕的危機。

某天，一名偽裝成「牛角族」的人類少女與森林守護者「哥雷姆」相遇，並展開踏上尋找人類住民的旅途。

== 登場人物 ==
; {{nihongo|索瑪麗|ソマリ|聲：[[水瀨祈|水瀨祈]]}}
: 本作主角之一。人類種少女，平時戴着有角的帽子，用以偽裝為「牛角族」。在森林里被「爸爸」撿到，其後與它一同展開旅途。

; {{nihongo|哥雷姆|ゴーレム|聲：[[小野大輔|小野大輔]]}}
: 本作主角之一。哥雷姆種，曾經是森林的守護者。
: 沒有名字，被索瑪麗稱作「父親」。在「撿到」索瑪麗後，為了尋找她的真正父母而與她一同展開旅途。
