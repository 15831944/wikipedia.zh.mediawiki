== 區劃沿革 ==
[[File:Somaliland_map_numbered2.png|thumb]]
[[File:somaliland_map_regions.png|thumb]]
[[File:Animin8955.gif|thumb]]
在2008年3月21日之前，[[索馬利蘭|索馬利蘭]]政府仍沿用[[索馬里|索馬里]]統一時所涵蓋的6個行政區︰
#[[阿烏達爾_(索馬利亞)|奧達勒州]]
#[[沃戈伊加勒貝德州|沃戈伊加勒貝德州]]
#[[萨纳格州|萨纳格州]]{{NoteTag|name=索邦之爭}}
#[[索勒州|索勒州]]{{NoteTag|name=索邦之爭}}
#[[托格代爾州|托格代爾州]]
#[[薩錫爾_(索馬利亞)|薩赫勒州]]

同年3月22日，總統[[达希尔·雷亚莱·卡欣|达希尔·雷亚莱·卡欣]]發表了一份「總統新聞聲明」，宣佈從已有的行政區劃中成立6个新的行政區和16个新區<ref>[http://www.somalilandtimes.net/sl/2008/322/080.shtml]</ref>。同年5月15日，總統又發表了一份「總統新聞聲明」，宣佈成立第13個行政區<ref>{{cite web |url=http://harowo.com/2008/05/15/madaxweynaha-oo-gobol-cusub-u-magacaabay-baligubadle/ |title=存档副本 |accessdate=2012-09-06 |0=2008-12-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20081207092348/http://harowo.com/2008/05/15/madaxweynaha-oo-gobol-cusub-u-magacaabay-baligubadle/ |archivedate=2008-12-07 }}</ref>。

至此，索馬利蘭境內現今共有13個州。
