'''-{A|zh:索龙;zh-hans:索龙;zh-tw:梭隆;}-'''（Sorong）又译'''-{zh:梭隆;zh-hans:梭隆;zh-tw:索龍;}-'''，是[[印度尼西亚|印度尼西亚]][[西巴布亚省|西巴布亚省]]的一座海滨城市，是[[西巴布亚省|西巴布亚省]]的最大城市以及[[西新几内亚|西新几内亚]]的第二大城市<ref>{{cite web |url=http://www.citypopulation.de/Indonesia-CU.html |title=INDONESIA: Cities & Settlements |last= |first= |date=2017-04-22 |website=citypopulation.de |publisher= |access-date=2017-07-16 |quote=}}</ref>，唯一与之陆地接壤的[[行政区划|印尼次级行政区]]是[[索龙县|索龙县]]。索龙是进出印尼[[拉贾安帕特群岛|拉贾安帕特群岛]]的重要枢纽，也是印尼蓬勃发展的东部油气产业运输枢纽。2010年人口普查中，索龙市共有190,625人<ref>Biro Pusat Statistik, Jakarta, 2011.</ref>，2014年1月的官方估计人口数达到219,958。

==名称==

'''索龙'''（Sorong）这个名称最早来源于'''Soren'''，在{{le|比亚克语|Biak language|比亚克语}}中意为''深而波涛汹涌的海洋''。这个名字由早期航行游历各岛并在[[拉贾安帕特群岛|拉贾安帕特群岛]]定居下来的[[比亚克农福尔县|比亚克-农福尔]]人所使用。当[[比亚克农福尔县|比亚克-农福尔]]人来到这块被称为'''马拉都大陆'''（[[印尼语|印尼语]]：Daratan Maladum）的地区，他们决定称之为Soren。在与中国商人、欧洲传教士、[[马鲁古群岛|马鲁古]]和[[桑义赫群岛县|桑义赫]]地区人接触后，这一名字逐渐变为索龙（Sorong）<ref>{{Cite web |url=http://www.kemendagri.go.id/pages/profil-daerah/kabupaten/id/92/name/papua-barat/detail/9271/kota-sorong |title=Etymology of Sorong |access-date=2017-03-09 |archive-url=https://web.archive.org/web/20170103171206/http://www.kemendagri.go.id/pages/profil-daerah/kabupaten/id/92/name/papua-barat/detail/9271/kota-sorong |archive-date=2017-01-03 |dead-url=yes }}</ref>

==行政区划==

索龙市属[[西巴布亚省|西巴布亚省]]，与相邻[[行政区划|次级行政区]][[索龙县|索龙县]]互不归属，是省之下独立的地方城市行政实体。索龙市原是[[索龙县|索龙县]]的一部分，也是原[[索龙县|索龙县]]的县治所在。1996年6月3日印尼议会颁布的1996年第31号法案宣布成立'''索龙行政市'''（[[印尼语|印尼语]]：Kota Administratif Sorong），1999年第45号法案又将索龙行政市升格为自治的城市行政区'''索龙市'''（[[印尼语|印尼语]]：Kota Sorong），索龙市政府所在地为'''索龙城市区'''（[[印尼语|印尼语]]：Kecamatan Sorong Kota）<ref>https://petatematikindo.files.wordpress.com/2013/02/administrasi-kota-sorong.jpg</ref>。目前索龙市下辖10个区（Kecamatan或Distrik）<ref>http://www.sorongkota.go.id/sample-page/kondisi-daerah/kondisi-geografis/{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>：索龙（Sorong）、西索龙（Sorong Barat）、索龙群岛（Sorong Kepulauan）、东索龙（Sorong Timur）、北索龙（Sorong Utara）、索龙马诺伊（Sorong Manoi）、索龙城市（Sorong Kota）、Malaimsimsa、Klaurung和Maladom Mes。区下辖41个村（kelurahan）

==交通==

索龙是[[跨巴布亚公路|跨巴布亚公路]]西端的起点。

索龙目前暂无铁路，[[西巴布亚|西巴布亚]]有计划修建一条铁路连接[[查亚普拉|查亚普拉]]和索龙。一期工程在[[西巴布亚省|西巴布亚省]]，从索龙到[[马诺夸里|马诺夸里]]，长390公里，本段线路预计2018年开工，2019年结束。二期工程从[[马诺夸里|马诺夸里]]向南连接巴布亚省的[[纳比雷|纳比雷]]<ref>http://finance.detik.com/berita-ekonomi-bisnis/3110744/ini-rute-jalur-ka-papua-setelah-sorong-manokwari</ref>。该项目由国家财政预算出资，被认为是印尼建设[[爪哇岛|爪哇岛]]之外铁路网络的突破性进展<ref>http://finance.detik.com/berita-ekonomi-bisnis/2988734/ini-peta-rute-kereta-di-papua-rp-10-triliun</ref>。

2012年初开始建造集装箱港口，年集装箱处理能力达50万标准箱，工程成本约9360万美元<ref>[http://www.thejakartapost.com/news/2011/06/20/container-port-be-built-sorong.html Container port to be built in Sorong | The Jakarta Post<!-- Bot generated title -->]</ref>。

[[多米尼克爱德华·奥索机场|多米尼克爱德华·奥索机场]]位于城市南部，有飞往印尼各大城市的航班。

==气候==

索龙属[[热带雨林气候|热带雨林气候]]，全年没有真正的旱季。最湿润的6月份降雨量合计{{convert|373|mm}}，最干旱的2月份降雨量也达{{convert|180|mm}}，索龙过度降水由季风引起。全年气温稳定，最热月和最冷月的温差是{{convert|1.2|C-change|1}}；最热的11月平均温{{convert|27.8|C|F|1}}，最冷的7月平均温{{convert|26.6|C|F|1}}。

==经济特区==

2016年6月14日，印尼经济统筹部长达尔敏（Darmin Nasution）表示，政府决定把索龙定为印尼的第十个经济特区，相关政府条例将在2016年底发出<ref name="国际日报">{{cite web |url=http://www.guojiribao.com/shtml/gjrb/20160616/273255.shtml |title=梭隆成为印尼第十个经济特区 |last= |first= |date=2016-06-16 |website=www.guojiribao.com |publisher=国际日报 |access-date=2017-09-22 |quote=}}</ref>。索龙共有4种行业最具发展潜力，林业、农用工业、物流业和海洋渔业<ref name="国际日报"></ref>。印尼海事与资源统筹部长利萨·兰里（Rizal Ramli）称索龙经济特区将以农用加工业为重点发展行业，政府在索龙经济特区建设的本质是发展加工业中心，并建设为一个物流中心，能够支持附近的采矿活动以及渔业的发展<ref>{{cite web |url=http://www.shangbaoindonesia.com/organisasi/%E5%BC%80%E5%8F%91%E5%B7%B4%E5%B8%83%E4%BA%9A%E7%B4%A2%E9%BE%99%E7%BB%8F%E6%B5%8E%E7%89%B9%E5%8C%BA%E6%94%BF%E5%BA%9C%E5%B0%86%E6%8F%90%E4%BE%9B%E8%B4%A2%E6%94%BF%E5%88%BA%E6%BF%80.html |title=开发巴布亚索龙经济特区 政府将提供财政刺激 |last= |first= |date=2016-06-15 |website=www.shangbaoindonesia.com |publisher=印度尼西亚商报 |access-date=2017-09-22 |quote=}}</ref>。
