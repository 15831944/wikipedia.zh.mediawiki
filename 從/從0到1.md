《'''從0到1：打開世界運作的未知祕密，在意想不到之處發現價值'''》於2014出版 (2014年9月16日發行)。此書由[[彼得·泰爾|彼得·泰爾]]、馬斯特共同完成。提爾是一位[[風險投資|風險投資]]家、PayPal共同創始人，同時也是[[Facebook|Facebook]]早期投資人之一。而馬斯特是其在[[史丹佛大學|史丹佛大學]]的學生，馬斯特於2012年修習提爾教授的創業CS183課程，他將課堂筆記濃縮整理，這份筆記在網路上廣受好評；而本書即是由這份筆記發展而出。<ref>{{cite web|url=http://www.zerotoonebook.com|title = Zero to One|accessdate = September 13, 2014}}</ref><ref>{{cite web|url=http://blakemasters.com/peter-thiels-cs183-startup|title = Notes Essays—Peter Thiel’s CS183: Startup—Stanford, Spring 2012|last = Masters|first = Blake|accessdate = September 13, 2014}}</ref><ref>{{cite web|url=http://www.quora.com/Whats-the-difference-between-Blake-Masters-notes-from-CS183-and-his-book-Zero-To-One|title = What's the difference between Blake Masters' notes from CS183 and his book "Zero To One"?|publisher = [[Quora|Quora]]|accessdate = September 13, 2014}}</ref>

== 行銷方法 ==
為了推廣本書，提爾於2014年接受新聞網站TechCrunch 聯合主編的訪問。2014年9月9日，提爾、知名商業人士費里斯(Timothy Ferriss) 一同擔任播客<ref>{{cite web|url=http://techcrunch.com/2014/09/08/peter-thiel-tweets/|title = Peter Thiel Tweets|date = September 8, 2014|accessdate = September 14, 2014|publisher = ''[[TechCrunch|TechCrunch]]''|last = Buhr|first = Sarah}}</ref> 。2014年9月11日，提爾在娛樂社群網站 Reddit上，做了一個"任意發問"(Ask Me Anything)的活動。.<ref>{{cite web|url=http://fourhourworkweek.com/2014/09/09/peter-thiel/|title = The Tim Ferriss Show: Interview with Peter Thiel, Billionaire Investor and Company Creator|last = Ferriss|first = Timothy|authorlink = Timothy Ferriss|date = September 9, 2014|accessdate = September 14, 2014}}</ref>
2014年9月13日，提爾在[[全國公共廣播電台|全國公共廣播電台]]NPR與主持人韋德一同討論本書。<ref>{{cite web|url = http://www.reddit.com/r/IAmA/comments/2g4g95/peter_thiel_technology_entrepreneur_and_investor|title = Peter Thiel, technology entrepreneur and investor. AMA|publisher = [[Reddit|Reddit]]|date = September 11, 2014|accessdate = September 13, 2014}}</ref><ref>{{cite web|url=http://www.slate.com/blogs/moneybox/2014/09/11/peter_thiel_ama_a_question_session_with_the_billionaire_paypal_founder.html|title = Peter Thiel Reveals His Favorite Rap Artist and More in a Reddit AMA|date = September 11, 2014|accessdate = September 14, 2014|publisher = ''[[Slate_(magazine)|Slate]]''}}</ref><ref>{{cite web|url=http://venturebeat.com/2014/09/11/peter-thiel-goes-after-net-neutrality-uber-bitcoin-and-hollywood-in-4-quotes/|title = Peter Thiel goes after net neutrality, Uber, Bitcoin, and Hollywood (in 4 quotes)|last = Ferenstein|first = Gregory|publisher = ''[[VentureBeat|VentureBeat]]''|date = September 11, 2014|accessdate = September 14, 2014}}</ref><ref>{{cite web|url=http://www.npr.org/2014/09/13/348181054/peter-thiel-in-zero-to-one-how-to-develop-the-developed-world|title = Peter Thiel In 'Zero To One': How To Develop The Developed World|date = September 13, 2014|accessdate = September 14, 2014|publisher = [[NPR|NPR]]}}</ref>

== 市場接受度 ==

=== 早期佳評 ===
2014年5月，本書官方上市前，一篇在美國商業新聞網站《商業內幕》上的文章提到:「這本書聽起來十分有趣! 」<ref>{{cite web|url = http://www.businessinsider.com/peter-thiel-book-zero-to-one-2014-5|title = Billionaire Peter Thiel Has A New Book Coming That Will Tell You How To Be A Startup Billionaire|last = Yarow|first = Jay|date = May 16, 2014|accessdate = September 14, 2014|publisher = ''[[Business_Insider|Business Insider]]''}}</ref>

=== 書評 ===
《[[出版者周刊|出版者周刊]]》Publishers Weekly評論本書:「提爾在書裡談到如何創造成功的商業，然而，他所談及的內容太過抽象，因此無法提供太多幫助給下一個[[賈伯斯|賈伯斯]]，或是提爾。」<ref name=kirkus>{{cite web|url=https://www.kirkusreviews.com/book-reviews/peter-thiel/zero-to-one/|title = ZERO TO ONE Notes on Startups, or How to Build the Future (book review)|publisher = [[Kirkus_Reviews|Kirkus Reviews]]|date = August 6, 2014|accessdate = September 14, 2014}}</ref> <ref>{{cite web|url=http://www.publishersweekly.com/978-0-8041-3929-8|title = Zero to One: Notes on Startups, or How to Build the Future (book review)|publisher = [[Publishers_Weekly|Publishers Weekly]]|date = August 11, 2014|accessdate = September 14, 2014}}</ref>

Jason Shen閱讀此書後，強力推薦讀者購買。<ref>{{cite web|url=http://www.jasonshen.com/2014/best-worst-parts-peter-thiels-new-book-zero-one/|title = The Best and Worst Parts of Peter Thiel’s New Book: Zero to One|last = Shen|first = Jason|date = August 26, 2014|accessdate = September 14, 2014}}</ref>Shen認為本書最棒的地方就是點出，成功創業者都是從一堆秘密中建立出來的<ref>{{cite web|url=http://blakemasters.com/post/22866240816/peter-thiels-cs183-startup-class-11-notes-essay|title = Peter Thiel’s CS183: Startup - Class 11 Notes Essay|last = Masters|first = Blake|date = May 11, 2012|accessdate = September 14, 2014}}</ref> ；不過他也提到，本書最糟糕的地方就是，提爾最喜愛的創業者即是先有了太偉大的計畫。<ref>{{cite web|url=http://blakemasters.com/post/23435743973/peter-thiels-cs183-startup-class-13-notes-essay|title = Peter Thiel’s CS183: Startup - Class 13 Notes Essay|last = Masters|first = Blake|date = May 20, 2012|accessdate = September 14, 2014}}</ref>

西元2014年11月，英國電腦科學家柏內茲-李{Timothy B. Lee)在新聞網站Vox.com評論<ref>{{cite web|url=https://www.vox.com/2014/11/30/7300019/how-peter-thiel-repackaged-conventional-wisdom-as-bold-contrarianism|title = How Peter Thiel repackaged conventional wisdom as bold contrarianism|last = Lee|first = Timothy B.|date = November 30, 2014|accessdate = December 19, 2014|publisher = [[Vox_Media|Vox.com]]}}</ref>
，他認為提爾的書確實有些良好建議，不過有些建議聽起來卻又太不真實，李認為提爾應該提供實際一點的建言，或做出一些較合理的主張。

在期刊《新亞特蘭提斯》中， 一位編輯將提爾與哲學家[[尼采|尼采]]做了比較，他針對提爾提道:「他根本就是一個太過政治與理論性的書呆子!這是從0到1的書中發現的，特別當他從我們認為已經最好、最鮮明的想法中，又提出了明顯的綜合錯誤觀念。」<ref name=Poulos>{{cite web|url=http://www.thenewatlantis.com/publications/competing-to-conform|title = Competing to Conform|publisher = [[The_New_Atlantis_(journal)|The New Atlantis (journal)]]|date = March 2015|accessdate = February 13, 2016}}</ref>
