== 故事簡介 ==
千種由嗣所居住的時代是個競爭非常激烈的社會，各項成績都不如人的由嗣在學校中只有F級的評價。某天，學校打算讓優等生螢塚亞里香和螢塚由乃兩姐妹前往一百年前做田野調查，感到好奇的由嗣在通往過去的門關閉之前也跟著回到一百年前的晶生村。此次的行動對由嗣的人生將有重大改變<ref>{{cite web|url=https://www.tgsmart.jp/article.aspx?a=5390|title=【TG SCOOP】クロシェット最新作『ここから夏のイノセンス！』は"人気絵師・せせなやう×実力派ライター・保住圭"の注目作!!|website=TG Smart|date=2015-04-23|accessdate=2019-04-12|language=ja}}</ref>。

== 登場角色 ==
; {{nihongo|千種由嗣|千種 由嗣（ちぐさ よしつぐ）}}
: 作品男主角，在身處的時代過得不順遂，因而跟著螢塚姐妹來到一百年前的晶生村。後來得到了未來世界的首肯，以螢塚姐妹助手的身分留在晶生村。
; {{nihongo|螢塚亞里香|蛍塚 アリカ（ほたるづか アリカ）|聲：{{lang|ja|あじ秋刀魚}}<ref name="character">{{cite web|url=https://clochette-soft.jp/kokonatsu/character.html|title=メインヒロイン|publisher=[[Clochette|Clochette]]|accessdate=2019年4月23日|language=ja}}</ref>}}
: 未來世界學校的精英，和由嗣是同班同學，和妹妹由乃為做田野調查來到一百年前的晶生村。知道由嗣在未來的評價，但並未瞧不起由嗣。
; {{nihongo|螢塚由乃|蛍塚 ユノ（ほたるづか ユノ）|聲：{{lang|ja|くすはらゆい}}<ref name="character"/>}}
: 亞里香同父異母的妹妹，跟姐姐亞里香一樣是優等生，個性自由奔放且我行我素。對於由嗣感到好奇，經常捉弄由嗣。
; {{nihongo|初姬彩羽|初姫 いろは（はつひめ いろは）|聲：[[桃井莓|桃井莓]]<ref name="character"/>}}
: 晶生村的居民，個性純真且和藹可親。在由嗣來到晶生村時是第一個發現他的人。對由嗣極為親切，並讓由嗣住在自己家。
; {{nihongo|久萬里壽|久万里 寿（くまり ことぶき）|聲：[[遙空|遙空]]<ref name="character"/>}}
: 晶生村村長的女兒，和彩羽是好朋友。個性文靜寡言，給人個性冷漠的感覺。實際上對人親切，一旦接受他人的請求便會完成到底。
; {{nihongo|鴫原紅緒|鴫原 紅緒（しぎはら べにお）|聲：[[櫻川未央|櫻川未央]]<ref name="character2">{{cite web|url=https://clochette-soft.jp/kokonatsu/character2.html|title=サブキャラクター|publisher=Clochette|accessdate=2019年4月23日|language=ja}}</ref>}}
: 比彩羽大一歲的青梅竹馬，家中五姐弟的長女。覺得由嗣跟村裡的男性有所不同而對他很在意。
; {{nihongo|克拉蕾塔·利特利歐|クラレッタ・リットリオ|聲：鶴屋春人<ref name="character2"/>}}
: 未來世界的居民，已在晶生村住了五年，負責未來與晶生村之間的聯絡。
