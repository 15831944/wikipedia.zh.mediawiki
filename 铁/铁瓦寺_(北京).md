'''铁瓦寺'''，位于[[北京市|北京市]][[房山区|房山区]][[河北镇_(北京市)|河北镇]]河北村，是[[汉传佛教|汉传佛教]]寺院，也是[[北京市文物保护单位|北京市文物保护单位]]。

==简介==
铁瓦寺始建于[[明朝|明朝]]<ref name=bjwb/>。约建于明朝[[正德|正德]]年间。清朝[[康熙|康熙]]十八年（1679年），北京地区发生八级以上地震，翌年又发生余震。铁瓦寺在地震中受到严重破坏，当地乡绅合力修复，补齐了破损的铁瓦，恢复了铁瓦殿原貌<ref name=si>{{Cite web |url =http://www.sohu.com/a/211744527_717266  |title = 文化遗产 房山铁瓦寺 |author =  |date =2017-12-20   |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

铁瓦寺坐北朝南。现存山门、东西配殿以及铁瓦殿。铁瓦殿呈圆柱状<ref name=bjwb/>，高6米，直径5.8米，门窗发券，殿表的白灰墙上刷有铁红色。殿顶采用[[攒尖顶|攒尖顶]]，6条脊自上朝6个方向垂下，将顶分为6个扇面<ref name=si/>。顶部的六条垂脊、宝珠、瓦都是铁制。铁瓦直径0.13米，长0.31米，共458块，大部铸有铭文<ref name=bjwb>{{Cite web |url = http://www.bjww.gov.cn/wbsj/bjwbdwDetail.htm |title =铁瓦寺  |author =  |date =  |publisher = 北京文博 |language =  |accessdate = 2018-02-20 |quote =  }}</ref>。垂脊、宝珠、瓦共用铁约3000公斤<ref name=si/>。

2001年7月12日，铁瓦寺被公布为第六批[[北京市文物保护单位|北京市文物保护单位]]<ref name=bjwb/><ref name=si/>。
