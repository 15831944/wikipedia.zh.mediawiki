'''铁力路站'''位于[[上海|上海]][[寶山区|寶山区]][[富锦路|富锦路]][[克东路|克东路]]，为[[上海轨道交通3号线|上海轨道交通3号线]]的地下[[侧式站台|侧式车站]]，也是3号线全线中唯一的一个地下车站。

== 車站特點 ==
本站有一項特色，站頂（即大堂的天花）是利用全透明的「玻璃樹」設計，因此大堂是用天然光照明，車站月台是利用通風井採光，在車站大堂的門外可以看見通風井，本站是[[上海軌道交通|上海軌道交通]]第二個位於地下利用天然光照明的車站，而第一個是[[上海科技館站|上海科技館站]]。

== 周邊 ==
此站邻近[[上海宝钢集团公司|上海宝山钢铁公司]]厂区。

== 公交换乘 ==
172、711、769、永罗线、淞嘉线、淞嘉专线、淞泾专线、宝山8路

== 车站出口 ==
* 1号口：富锦路北侧，克山路西
* 2号口：富锦路北侧，克东路东
