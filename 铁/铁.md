'''铁'''是一种化学[[元素|元素]]，它的[[化学符号|化学符号]]是'''Fe'''，它的[[原子序数|原子序数]]是26，它的[[相对原子质量|相对原子质量]]是56。它是[[过渡金属|过渡金属]]的一种。铁是最常用的[[金属|金属]]，是[[地球|地球]][[外核|外核]]及[[內核|內核]]的重要成份，是[[地球的地殼元素豐度列表|地殼上豐度第四高的元素]]和第二高的金屬。鐵常出現在[[类地行星|类地行星]]中，因為鐵是高質量恆星[[核融合|核融合]]後的產物，[[鎳-56|鎳-56]]是放熱[[核融合|核融合]]反應的最後一個產物，之後會衰變成最常見的鐵同位素。

铁和其他[[8族元素|8族元素]]相同，其[[氧化態|氧化態]]範圍很廣，由−2到+6，但其中+2和+3是最常見的氧化態。在[[流星体|流星体]]及低氧的環境下，鐵會以单质的形式存在，但是鐵很容易和氧氣和水反應。鐵的表面是有光澤的銀灰色，但在空氣中鐵會反應生成[[水合|水合]]的氧化鐵，一般稱為[[铁锈|铁锈]]。許多金屬在氧化後會形成[[钝化|钝化]]的氧化層，保護內部的金屬不被氧化，但氧化鐵的密度較鐵要低，因此氧化鐵會剝落，無法保護內部的鐵不受腐蝕。

== 性质 ==
=== 物理性质 ===
鐵是有光澤的銀白色[[金屬|金屬]]，硬而有[[延展性|延展性]]，[[熔點|熔點]]為1538[[摄氏度|℃]]，[[沸點|沸點]]为2750℃，有很強的鐵[[磁性|磁性]]，並有良好的[[可塑性|可塑性]]和[[導熱性|導熱性]]。[[晶体结构|晶体结构]]为[[体心立方结构|体心立方结构]]，[[晶格常数|晶格常数]]a=2.87[[埃|埃]]。日常生活中的铁通常含有[[碳|碳]]因而暴露在[[氧气|氧气]]中容易在遇到[[水分子|水]]的情况下发生[[电化学|电化学]][[腐蚀|腐蚀]]，而[[纯度|纯度]]较高的铁则不易[[腐蚀|腐蚀]]<ref name=hpfe>超高純度鐵（日文）：http://techon.nikkeibp.co.jp/article/WORD/20060317/115050/</ref>。

=== 化学性质 ===

具有金属[[单质|单质]]的通性，能与[[非金属|非金属]]单质、[[酸|酸]]、[[盐_(化学)|盐]]等反应。

铁在[[氧气|氧气]]中燃烧后生成[[四氧化三铁|四氧化三铁]]
:<math>\rm 3Fe+2O_{2} \rightarrow Fe_{3}O_{4} </math>
在高温下，铁可以与[[水蒸汽|水蒸汽]]反应，生成四氧化三铁和[[氢气|氢气]]。
:<math>\rm 3Fe + 4H_{2}O \rightarrow Fe_{3}O_{4} + 4H_{2}</math>

铁元素可以形成3种[[氧化物|氧化物]]，分别是[[氧化亚铁|氧化亚铁]]（{{chem|FeO}}），[[氧化铁|氧化铁]]（{{chem|Fe|2|O|3}}），和[[四氧化三铁|四氧化三铁]]（{{chem|Fe|3|O|4}}）（{{chem|FeO}}·{{chem|Fe|2|O|3}}）。

铁和非氧化性酸反应得到Fe<sup>2+</sup>（亚铁离子，浅绿色），和氧化性酸反应得到Fe<sup>3+</sup>（铁离子，黄色），铁在[[浓硫酸|浓硫酸]]和[[浓硝酸|浓硝酸]]中钝化。

铁加热、加压下可以和[[一氧化碳|一氧化碳]]反应得到[[羰基|羰基]]化合物：
:<math>\rm Fe + 5CO \rightarrow Fe(CO)_5 </math>

铁和[[氯气|氯气]]反应（点燃）得到三氯化铁，而和[[硫|硫]]反应（加热）只能得到硫化亚铁：
:<math>\rm 2Fe+3Cl_2 \rightarrow 2FeCl_3 </math><ref>《化学 必修1》.人民教育出版社.2007年3月第3版.'''第四章 非金属及其化合物'''.''2 富集在海水中的元素——氯''.P<sub>83</sub>.ISBN 978-7-107-17648-7</ref>
:<math>\rm Fe+S \rightarrow FeS</math>

铁和Fe<sup>3+</sup>反应得到Fe<sup>2+</sup>：
:<math>\rm 2FeCl_3 + Fe \rightarrow 3FeCl_2 </math><ref>《化学 必修1》.[[人民教育出版社|人民教育出版社]].2007年3月第3版.'''第三章 金属及其化合物'''.''2 几种重要的金属化合物''.三、铁的重要化合物.P<sub>61</sub>.ISBN 978-7-107-17648-7</ref>

=== 化合物 ===
{{main article|铁化合物}}

== 发现 ==
铁是古代就已知的金属之一。铁矿石是[[地壳|地壳]]主要组成成分之一，铁在自然界中分布极为广泛，但人类发现和利用铁却比[[黄金|黄金]]和[[铜|铜]]要迟。首先是由于天然的[[单质|单质]]状态的铁在[[地球|地球]]上非常稀少，而且它容易[[氧化|氧化]]生锈，加上它的[[熔点|熔点]]（1812K）又比铜（1356K）高得多，就使得它比铜难于熔炼。

人类最早发现的铁是从天空落下来的[[陨石|陨石]]，[[陨石|陨石]]中含铁的百分比很高，是铁和[[镍|镍]]、[[钴|钴]]等金属的混合物，在融化铁矿石的方法尚未问世，人类无法大量获得[[生铁|生铁]]的时候，铁一直被视为一种带有神秘性的最珍贵的金属。

铁的发现和大规模使用，是人类发展史上的一个里程碑，它把人类从石器时代、青铜器时代带到了[[铁器时代|铁器时代]]，推动了人类文明的发展。至今铁仍然是现代化学工业的基础，人类进步所必不可少的金属材料。

== 名称由来 ==
铁，化学符号'''Fe'''的来源是拉丁文名称Ferrum。

《說文解字》：「-{鐵，黑金也。从金，𢧤聲。銕，古文鐵，从夷。}-」

铁是[[宇宙|宇宙]]中第六[[化學元素豐度|丰富]]的元素，也是最常见的耐火元素。<ref name=apjl717_2_L92>{{cite journal

 | last5=Matsuura | first5=M. | last6=Kraemer | first6=K. E.
 | last7=Bernard-Salas | first7=J. | last8=Markwick | first8=A. J.
 | title=Rusty Old Stars: A Source of the Missing Interstellar Iron?
 | journal=The Astrophysical Journal Letters
 | volume=717 | issue=2 | pages=L92–L97 | year=2010
 | doi=10.1088/2041-8205/717/2/L92 | bibcode=2010ApJ...717L..92M |arxiv = 1005.3489 }}</ref>它是因为大质量恒星的[[硅燃烧过程|硅燃烧过程]]在[[恆星核合成|恆星核合成]]的最后放热阶段形成的。

铁是[[地球|地球]]上分布最广的金属之一。约占[[地壳|地壳]][[质量|质量]]的5.1%，居元素分布序列中的第四位，仅次于[[氧|氧]]、[[硅|硅]]和[[铝|铝]]。

在自然界，[[游离态|游离态]]的铁只能从[[陨石|陨石]]中找到，分布在地壳中的 铁都以化合物的状态存在。铁的主要矿石有：[[赤铁矿|赤铁矿]]Fe<sub>2</sub>O<sub>3</sub>，含铁量在50%~60%之间；[[磁铁矿|磁铁矿]]Fe<sub>3</sub>O<sub>4</sub>，含铁量60%以上，有[[亚铁磁性|亚铁磁性]]，此外还有[[褐铁矿|褐铁矿]]Fe<sub>2</sub>O<sub>3</sub>·nH<sub>2</sub>O、[[菱铁矿|菱铁矿]]FeCO<sub>3</sub>和[[黄铁矿|黄铁矿]]FeS<sub>2</sub>，它们的含铁量低一些，但比较容易冶炼。

== 制备 ==
单质铁的制备一般采用[[冶炼法|冶炼法]]。以[[赤铁矿|赤铁矿]]（Fe<sub>2</sub>O<sub>3</sub>）和[[磁铁矿|磁铁矿]]（Fe<sub>3</sub>O<sub>4</sub>）为原料，与焦炭和助溶剂在熔矿炉内反应，焦炭燃烧产生[[二氧化碳|二氧化碳]]（CO<sub>2</sub>），二氧化碳与过量的焦炭接触就生成[[一氧化碳|一氧化碳]]（CO），一氧化碳和礦石內的氧化铁作用就生成金属铁。

:<math>\rm C + O_{2} \rightarrow CO_{2}</math>

:<math>\rm CO_{2} + C \rightarrow 2CO</math>

:<math>\rm Fe_{3}O_{4} + 4CO \rightarrow 3Fe + 4CO_{2}</math>

:<math>\rm Fe_{2}O_{3} + 3CO \rightarrow 2Fe + 3CO_{2}</math>

:<math>\rm FeO + CO \rightarrow Fe + CO_{2}</math>

以上反应都是[[可逆反应|可逆反应]]，所产生的一氧化碳浓度越大越好，要使反应进行完全必须在800度以上进行。

化学纯的铁是用[[氢|氢]]气还原纯氧化铁来制取，也可由[[五羰基铁|五羰基铁]]来制取，通过其热分解来得到纯铁。

铁也可以通过[[铝热反应|铝热反应]]得到：
:<math>\rm Fe_{2}O_{3}+2Al \rightarrow 2Fe+Al_{2}O_{3}</math><ref>《化学 必修2》.人民教育出版社.2007年3月第3版.'''第四章 化学与自然资源的开发利用'''.''1 开发利用金属矿物和海水资源''.一、金属矿物的开发利用.P<sub>89</sub>.ISBN 978-7-107-17649-4</ref>

== 用途 ==
在日常生活裡，铁可以算是最有用、最廉價、最丰富、最重要的金属。工農業生产中，铁是最重要的基本结构材料，铁合金用途广泛，例如[[鋼|鋼]]就是鐵和碳及其他金屬的合金；国防和战争更是[[钢|钢铁]]的较量，钢铁的年产量代表一个国家的现代化水準，被稱作工業之母。

对于人体，铁是不可缺少的[[微量元素|微量元素]]<ref>{{Cite web
 |url          = http://www.sfncc.org.cn/Z_Show.asp?ArticleID=2124
 |title        = 微量元素铁与人体健康的关系

 |archive-url  = https://web.archive.org/web/20140920065040/http://www.sfncc.org.cn/Z_Show.asp?ArticleID=2124
 |archive-date = 2014-09-20
 |dead-url     = yes
}}</ref>。在十多种人体必需的微量元素中，铁無論在重要性上還是在數量上，都居於首位。

一个正常的成年人全身含有3[[克|克]]多铁，相当于一颗小铁钉的质量。人体血液中的[[血红蛋白|血红蛋白]]就是铁的[[配合物|配合物]]，它具有固定氧和输送氧的功能。人体缺铁会引起[[贫血症|贫血症]]。只要不偏食，不大出血，成年人一般不会缺铁。但由于女生会来月经等而造成血液流失，导致女性的铁质流失，所以女性宜食的食品中很多都含有丰富的铁质。（但是体内铁浓度过高会导致[[血色沉着病|铁过载]]）

所谓[[煤气|煤气]]中毒（[[一氧化碳|一氧化碳]]中毒），也是由于血红素中铁離子（II）上原本氧气的连接位被一氧化碳占据，丧失了吸收氧分子的能力，使人窒息中毒而死亡<ref>{{Cite web
