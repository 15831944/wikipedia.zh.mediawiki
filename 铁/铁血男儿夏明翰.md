《'''铁血男儿夏明翰'''》，2012年上映于中国大陆的战争剧。

== 剧情 ==
1919年，[[五四运动|五四运动]]风起云涌，刚刚建立不久的[[中华民国|中华民国]]陷入了[[军阀割据|军阀割据]]的局面，民众苦不堪言，[[北洋军阀|北洋军阀]][[吴佩孚|吴佩孚]]在攻下[[湖南省|湖南省]][[衡阳市|衡阳城]]的时候，深深影响了衡阳城的夏明翰一家及部分爱国青年，他们争相传阅《[[新青年|新青年]]》等进步书籍，并且加入了[[砂子会|砂子会]]。夏明翰为了改良社会和保守的家庭决裂了，他去了古城[[长沙|长沙]]，在[[毛泽东|毛泽东]]的介绍下，加入了[[中国共产党|中国共产党]]，和[[郑家钧|郑家钧]]结为夫妻。1928年，夏明翰在[[白色恐怖|白色恐怖]]的黑幕下，被中国共产党叛徒出卖，后被[[中国国民党|中国国民党]]擒拿杀害。

== 花絮 ==
夏明翰、孙越泽、章承基三人结为弟兄，但是三人信奉不同的主义和信仰。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2012-09-01/21433728914.shtml |title=《铁血男儿夏明翰》开播预热 |accessdate=  2012年9月1日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

电视剧基本上是用历史说话，定为“青春励志偶像剧”。<ref>{{cite web |url=http://hnrb.hinews.cn/html/2012-09/03/content_521950.htm |title=《铁血男儿夏明翰》将登央视 |accessdate= 2012年9月3日 |publisher=海南日报 |work= |language =zh-cn}}</ref>

主演夏明翰的严宽说：“夏明翰[[抵制日货|抵制日货]]烧了自己家的日货。本以为夏明翰的爷爷会责怪明翰，谁知爷爷当上了国货维持会的会长。爷爷也爱国，原谅了明翰，没有惩罚他。今天的我们又学到了些什么？”<ref>{{cite web|url=http://www.morningpost.com.cn/szb/html/2012-09/21/content_185154.htm|title=《铁血男儿夏明翰》凸显时代精神|accessdate=2012年9月21日|publisher=北京晨报|work=|language=zh-cn}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
