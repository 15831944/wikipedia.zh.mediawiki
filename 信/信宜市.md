'''信宜市'''（[[邮政式拼音|邮政式拼音]]：{{lang|la|'''Sunyi'''}}）是[[中国|中国]][[广东省|广东省]][[茂名市|茂名市]]下辖[[县级市|县级市]]。信宜位于广东省西南部，毗邻[[广西壮族自治区|广西]]，是[[中国肉类生产百强县列表|广东省肉类生产第一大、中国第11大县市]]。信宜现有旅居海外华侨、港澳台同胞53万人，归侨、侨眷39万人，是粤西地区最大的[[侨乡|侨乡]]。<ref name=基本概况>{{Cite web|url=http://www.xinyi.gov.cn/Item/663.aspx|title=基本概况-信宜市人民政府|accessdate=2018-02-01|work=www.xinyi.gov.cn|deadurl=yes|archiveurl=https://web.archive.org/web/20161113122159/http://xinyi.gov.cn/Item/663.aspx|archivedate=2016-11-13}}</ref>据部门统计，全市耕地面积4.05万公顷，粮食播种面积5.23万公顷，粮食产量31.38万吨。林地面积20.58万公顷，森林覆盖率67.98％，活立木蓄积量0.15亿立方米。<ref name=基本概况 />

== 历史 ==
[[File:Guangdong_Sheng_Bowuguan_2012.11.18_11-05-04.jpg|缩略图]]]]
古代属[[百越|百越]]地。[[秦代|秦代]]属[[桂林郡|桂林郡]]，[[汉代|汉代]]属[[苍梧郡|苍梧郡]]，[[晋代|晋代]]属[[晋康郡|晋康郡]]。[[南朝梁|南朝梁]]廢梁德郡，保留梁德县。[[隋代|隋代]]更名为怀德县，屬瀧州。[[唐|唐]]初析置出信义、潭峨、特亮县、扶莱五县。唐乾封二年（667年），[[扶莱县|扶莱县]]划归禺州。[[宋太祖|宋太祖]]年间，怀德、潭峨、特亮三县并入信义县，以该地南有信山，北有招义山而得名。[[宋太宗|宋太宗]]即位后，为避“[[赵光义|义]]”之讳，于[[太平兴国|太平兴国]]五年（公元976年）将信义更名为信宜。1995年9月撤县设[[县级市|县级市]]。<ref name=re>{{Cite web|url=http://www.xinyi.gov.cn/Item/660.aspx|title=行政区域-信宜市人民政府|accessdate=2018-02-01|work=www.xinyi.gov.cn|deadurl=yes|archiveurl=https://web.archive.org/web/20161113081839/http://xinyi.gov.cn/Item/660.aspx|archivedate=2016-11-13}}</ref>

== 地理 ==
信宜四面環山<ref name=tvb>[[翡翠台|六點半新聞]]，2016-05-22，無線電視翡翠台。</ref>，境内地形以山地为主，地势东北高、西南低，为[[亚热带季风气候|亚热带季风气候]]。信宜全市有海拔500米以上的大小山頭{{Fact|371|time=2018-02-05T13:22:21+00:00}}座，最高峰大田頂海拔高度1704米，是粵西第一高峰。<ref>{{Cite book|title=信宜县志|last=陈|first=启著|publisher=广东人民出版社|year=1993|isbn=978-7-218-01230-8|location=广东省信宜市|pages=1173|asin=B00CWIFV84}}</ref>全市年平均气温22.6℃，年降水量1816.2毫米。<ref>{{Cite web|url=http://www.xinyi.gov.cn/Item/659.aspx|title=地理环境|accessdate=2018-02-05|author=|date=2012-02-21|publisher=信宜市人民政府|archive-url=https://web.archive.org/web/20180206002537/http://www.xinyi.gov.cn/Item/659.aspx|archive-date=2018-02-06|dead-url=yes}}</ref>2016年5月20日，[[兩廣|兩廣]]暴雨成災，信宜市由於獨特的地理環境，令市內水淹及腰，當地水庫水位亦高出警戒水位1.5米<ref name=tvb />。

境内森林覆盖率约70%左右，属于山城，市民通常称呼为「山旮旯」（偏僻的角落）。[[锡|锡]]、[[金|金]]、[[玉|玉]]、[[花岗石|花岗石]]储量较大，其中斑岩锡矿储量位居中国第三。农业主产[[水稻|水稻]]及[[小麦|小麦]]、[[玉米|玉米]]等粮食和多种[[中药|中药]]材，盛产[[荔枝|荔枝]]。工业有林业、造纸、电子、采矿、玉器加工等门类。

旅游产业是信宜市的重要产业。目前信宜境内的旅游地点有：大雾岭、大田顶、大仁山、[[西江|西江]]温泉、天马山、太华山、石根山等。

== 方言 ==
信宜方言指的是信宜地区使用的方言，包括[[信宜白话|信宜白话]]和信宜涯话。信宜白话，属于汉藏语系下的汉语族[[粤语|粤语]]支高阳片；信宜涯话，属于汉藏语系汉语族[[客语|客语]]支，之前属于不分片客语，后来经比较可算粤台片[[涯话|涯话]]小片。

== 教育 ==
[[File:Education_Town.jpg|缩略图]]
信宜市目前的教育以中小学教育为主，现有中学53所，小学357所。2002年5月，开始在市区东南郊兴建包括高中、初中、小学及幼儿园的信宜教育城。信宜教育城于2003年秋季大部分建成并开始招生。初中部从原来的[[信宜中学|信宜中学]]分出，[[信宜中学|信宜中学]]只保留高中部。新建成的[[信宜中学|信宜中学]]占地600亩、办学规模120个班。

==特产== 
信宜自然资源丰富。野生及人工栽种的植物共2000余种。药材据统计300多种，其中[[田七|田七]]、[[砂仁|砂仁]]、[[肉桂|清花桂]]等南药尤为著名。水果有[[山楂|山楂]]、[[柿子|柿子]]、[[荔枝|荔枝]]、[[龙眼|龙眼]]、[[三华李|三华李]]等40多种。珍贵野生动物有[[金钱龟|金钱龟]]、[[中国大鲵|娃娃鱼]]、[[蟒蛇|蟒蛇]]等。<ref name=基本概况 />
另外，信宜物产资源丰富。盛产全国唯一的“南方碧玉”，矿产资源有[[玉石|玉石]]、[[滑石|滑石]]、[[高岭石|高岭石]]、[[萤石|萤石]]、[[石英|石英]]、[[金|金]]、[[银|银]]、[[铜|铜]]、[[铁|铁]]、[[锡|锡]]、[[锌|锌]]、[[钼|钼]]等30多种。其中花岗岩总储量达数十亿立方米，居全省首位，银岩锡矿是全国第三大锡矿，东坑金矿为广东省第二大金矿，玉矿是全国唯一的“南方碧玉”，以它雕制的工艺美术品誉满中外。<ref>{{Cite web|url=http://www.xinyi.gov.cn/Item/657.aspx|title=物产资源-信宜市人民政府|accessdate=2018-02-01|work=www.xinyi.gov.cn|deadurl=yes|archiveurl=https://web.archive.org/web/20161113080555/http://xinyi.gov.cn/Item/657.aspx|archivedate=2016-11-13}}</ref>
信宜市是[[鉴江|鉴江]]、[[黄华河|黄华河]]、[[罗定江|南江]]、[[北流河|北流河]]的发源地。土特产有尚文米粉、大成山楂、大成面、怀乡三黄鸡、钱排三华李、仔鱼等。信宜是全省重要的竹器、玉器、松香、花岗岩板材出口基地之一，是全省山地鸡、三华李、荔枝、龙眼、南药、反季节蔬菜的重要生产基地。<ref name=基本概况 />
