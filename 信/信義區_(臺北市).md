'''信義區'''，地理位置位於[[臺灣|臺灣]][[臺北市|臺北市]]的中間略偏東南，[[臺北市政府|臺北市政府]]、[[臺北市議會|臺北市議會]]、[[臺北世界貿易中心|臺北世界貿易中心]]皆位於此。

此區原為[[松山區_(臺灣)|松山區]]的一部份，由於松山區至西元1980年代人口達45萬，轄區過大，因此在西元1990年3月12日臺北市行政區重劃時將松南地區畫出獨立設區，由原松山區縱貫鐵路以南地區與[[大安區_(臺北市)|大安區]]和平東路3段以北合併而成<ref>{{cite web |title=歷史沿革 |url=http://www.xydo.taipei.gov.tw/ct.asp?xItem=21764&CtNode=3508&mp=124091 |date=  |publisher=臺北市信義區公所 |accessdate=}}</ref>，名稱取自[[信義計畫區|信義計畫區]]。因此歷史緣由，常見區內道路以「松」字開頭，而[[松山高中|松山高中]]、[[松山工農|松山工農]]、[[松山家商|松山家商]]等校，也因此出現今天都不位於[[松山區_(臺灣)|松山區]]的現象。

該區[[百貨公司|百貨公司]]林立<ref> [https://www.travel.taipei/zh-tw/attraction/details/1573 台北旅遊網 - 信義商圈]</ref>，許多企業總部、金融總部設於此，區內的[[臺北101|臺北101]]是目前世界第十二高的[[摩天大樓|摩天大樓]]，亦是世界最高[[綠色建築|綠色建築]]。[[信義計畫區|信義計畫區]]因實施都市計畫特別審議制度及重點發展區域，因此其都市景觀成為臺北市的重要亮點，並且吸引臺北眾多金字塔頂層富裕人士群聚於此。2010年臺北市舉辦花卉博覽會而制定區花，信義區為野牡丹，民政局也特別製作區花版門牌。

== 地理位置 ==
北接[[松山區_(臺灣)|松山區]]，東面為[[南港區|南港區]]，西面為[[大安區_(臺北市)|大安區]]，南接[[文山區|文山區]]。
*北界：[[市民大道|市民大道]]五段（光復南路至基隆路）、[[市民大道|市民大道]]六段（基隆路至中坡北路），鄰接[[松山區_(臺灣)|松山區]]。
*東界：[[中坡北路|中坡北路]]／南路、[[中坡南路|中坡南路]]201巷、[[福德街|福德街]]310巷與309巷至[[福德國小|福德國小]]後山，鄰接[[南港區|南港區]]。
*南界：[[拇指山脈|拇指山脈]]、[[四獸山|四獸山]]（福德國小後山至[[莊敬隧道|莊敬隧道]]，鄰接[[南港區|南港區]]、[[文山區|文山區]]）。
*西界：[[莊敬隧道|莊敬隧道]]、和平東路三段、基隆路二段、光復南路，鄰接[[大安區_(臺北市)|大安區]]。

== 歷史 ==
[[File:Jrzj.jpg|thumb]]
[[台灣日治時期|日治時期]]1920年臺灣地方改制，實行街-{庄}-制，設「-{[[松山庄|松山庄]]}-」，隸屬[[臺北州|臺北州]][[七星郡|七星郡]]管轄，其中之'''中坡'''、'''五分埔'''、'''三張犁'''、'''興雅'''等[[大字_(行政區劃)|大字]]在今信義區轄內。1938年松山-{庄}-併入臺北市。80年代末期，當時的松山區人口已達四十五萬餘人，約佔臺北市五分之一，轄有九十八里，不但是臺北市人口最多，編制人員最大的行政區，也是台閩地區最大建制的基層單位。因此1990年臺北市16區改為12區時，分割縱貫鐵路以南的松山區，約為'''中坡'''、'''五分埔'''、'''三張犁'''、'''興雅'''等區域獨立組成信義區。
