'''信阳东站'''为[[京广高速铁路|京广高速铁路]]的一个车站，于2012年9月28日正式开通运营，信阳到[[郑州|郑州]]、[[武汉|武汉]]的时间分别缩短到90分钟和50分钟。<ref name="开通">{{cite web |url=http://news.hexun.com/2012-09-24/146169913.html |language= |title=高铁信阳东站正式开通运营 唯一开行始发列车省辖市 |publisher=河南省人民政府网站 |date=2012年9月24日 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20150627004109/http://news.hexun.com/2012-09-24/146169913.html |archivedate=2015年6月27日 }}</ref>

== 简介 ==
信阳东站位于[[信阳|信阳]][[羊山新区|羊山新区]]，距市中心较远。站前广场西侧正在建设信阳高铁站快乐园<ref>{{cite web |url=http://www.ha.xinhuanet.com/hnxw/2012-09/14/c_113072747.htm |language= |title=信阳东站即将迎来高铁开通两条临时公交去高铁站 |publisher=大河网 |date=2012年9月14日 |accessdate= }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

该站设计有三站台七道，車站南面有预留[[寧西鐵路|寧西鐵路]]增建二线客运连接线。<ref>{{cite web |url=http://news.shangdu.com/xinyang/20120928/132_5704706.shtml |language= |title=信阳东站今日正式投入使用 信阳进入高铁时代 |publisher=信阳网 |date=2012年9月28日 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20121031200503/http://news.shangdu.com/xinyang/20120928/132_5704706.shtml |archivedate=2012年10月31日 }}</ref>

目前该站有5趟始发列车，是[[河南省|河南省]]除[[郑州东站|郑州东]]、[[郑州站|郑州]]外三个开行始发列车的省辖市高铁站之一。
