== 分布 ==
大多数信天翁科的鸟生活在南半球深海區域的範圍內，少数生活在北[[太平洋|太平洋]]和赤道地带。人們通常在大洋航行時，在，海上或石礁島嶼等地方才可看到信天翁。

== 形态 ==
信天翁科的鸟都比较大，拥有窄长的[[翅膀|翅膀]]，它们的嘴比较长和有力，上喙末端形成一个向下的钩。成鸟一般身長達1米多，雙翅展開可達餘3-4米，體重達8至9[[公斤|公斤]]，雄性信天翁一般比雌性體型大。

== 习性 ==
信天翁科的鸟可以在海上长时间飞行，它们可以非常有效地利用空气动力的原理在海面上滑翔，有时数小时不必扇一下翅膀。當信天翁能夠掌握海面的氣流，便可以恣意在空中高速翻飛達2000—3000米，在1小時內可以橫越113[[公里|公里]]的海面。據記載，一隻信天翁可以在12天內飛行達5000公里的旅程。信天翁科的鸟一般只在育幼时上岸，它们甚至可以在海面上睡觉。

在[[南极洲|南极洲]]附近，它们最主要的食物是那里的[[鱼|鱼]]、[[墨鱼|墨鱼]]和[[虾|虾]]。它们也以船丢弃的废物为食，因此它们有时跟船飞行。

信天翁科的鸟在繁殖期会上岸登岛，筑巢产卵，這時碰到人牠們也不逃避。信天翁性成熟较晚，到5—8歲時才開始繁殖。雌鳥產一到两顆蛋。信天翁是[[一夫一妻制|一夫一妻制]]，並輪流負責將蛋孵化，孵化期可達60天之久。初期幼鳥從父母口中得到已消化了的脂肪及半消化了的食物。这时，父母需要[[反芻|反芻]]體內食物。

信天翁科的鸟可以活40—60年。<ref>{{cite book|author1=李敏|title=生物进化密码|date=2015-04-22|publisher=安徽人民出版社|url=https://books.google.com/books?id=GCV0CAAAQBAJ&pg=PT48&lpg=PT48&dq=%E4%BF%A1%E5%A4%A9%E7%BF%81+%E6%B0%B4%E6%89%8B&source=bl&ots=NC9IyodMcP&sig=u_tB2QE893OlqKfzLty0000AHGY&hl=zh-CN&sa=X&ved=0ahUKEwivosLoqYrWAhUhOpoKHcG3AuYQ6AEIaDAN#v=onepage&q=%E4%BF%A1%E5%A4%A9%E7%BF%81%20%E6%B0%B4%E6%89%8B&f=false}}</ref>

== 保护 ==
[[File:Laysan_albatross_chick_remains.jpg|left]]屍體，肚子裡滿是垃圾]]
按[[世界自然保护联盟|世界自然保护联盟]]的估计所有信天翁科的鸟都受到灭绝的威胁。受胁的原因主要是由于赖以繁殖的海岛受到人为的开发和破坏，原因有：
* 有的地方会有人趁繁殖期上岛捡蛋，使繁殖失败，长期如此而导致种群数量锐减。
* 棲息地遭嚴重破壞，每年都有許多因誤食人類產生的垃圾而死的信天翁。

== 分类 ==
信天翁科的分属至今为止还没有定论。

== 「信天翁」一詞的其他用語 ==
在[[高爾夫球|高爾夫球]]運動中，當球手打出低於[[標準桿|標準桿]]數三桿時，會被稱為信天翁。
