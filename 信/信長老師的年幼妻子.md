《'''信長老師的年幼妻子'''》（{{lang-ja|ノブナガ先生の幼な妻}}）是[[日本|日本]][[漫畫家|漫畫家]]{{lang|ja|紺野あずれ}}，以[[戰國時代|戰國時代]]及[[時間旅行|穿越]]為題材創作的[[日本漫畫|漫畫]]作品，[[雙葉社|雙葉社]]出版，於漫畫雜誌《[[月刊Action|月刊Action]]》開始連載。2018年12月宣佈改編為電視動畫，2019年4月6日播映<ref>{{cite web|script-title=ja:紺野あずれの戦国系ラブコメ「ノブナガ先生の幼な妻」TVアニメ化！来年4月放送|url=https://natalie.mu/comic/news/311703|website=[[Natalie_(網站)|Natalie]]|accessdate=2019年4月4日|language=ja|date=2018年12月11日}}</ref>。

== 故事簡介 ==
中學教師「織田信永」一直以來都沒放棄這有如美少女遊戲般的夢想，就是有個超喜歡自己的少女突然出現在身旁，某一天他身邊真的出現了一位自稱 14 歲的少女，這位名為「歸蝶」的女孩似乎來自戰國世代，她立志要和信永生下愛的結晶。<ref>{{Cite web|url=https://gnn.gamer.com.tw/6/172406.html|title=《信長老師的年幼妻子》漫畫改編電視動畫預定明年 4 月開播|accessdate=|author=Blue Moon|date=2018-12-12|publisher=巴哈姆特電玩資訊站|language=zh-tw}}</ref>

== 登場角色 ==
; {{nihongo|織田信永|織田信永（おだ ノブナガ）|聲：[[酒井廣大|酒井廣大]]}}
: 本作品的主人公。28歲，在初中作為社會科老師。
; {{nihongo|齋藤歸蝶|斎藤帰蝶（さいとう きちょう）|聲：[[上原明里|上原明里]]}}
: 從織田家倉庫突然出現的14歲（[[虛歲|虛歲]]15歲）少女。本來是政治聯姻而嫁給織田信長，但卻穿越到了現代。
; {{nihongo|生駒吉乃|生駒吉乃（いこま きつの）|聲：[[小澤亞李|小澤亞李]]}}
: 織田信長的側室。從戰國時代轉生到現代。29歲時去世，以年輕的樣貌轉生。
; {{nihongo|枇杷島萬結|枇杷島万結（びわじま まゆ）|聲：[[結崎木乃美|結崎木乃美]]}}
: 信永的學生。
; {{nihongo|星丘友里|星ヶ丘友里（ほしがおか ゆり）|聲：[[田中茉理花|田中茉理花]]}}
: 在信永的班級擔任助理教師的女教師。 24歲
; {{nihongo|熱田杏南|熱田杏南（あつた あんな）|聲：[[豐田萌繪|豐田萌繪]]}}
: 枇杷島萬結的朋友。

; {{nihongo|熱田蘭真|熱田蘭真（あつた らんま）}}
: 杏南的弟弟。
