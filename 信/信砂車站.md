'''信砂站'''（{{jpn|j=信砂駅|hg=のぶしゃえき|rm=Nobusha eki|f=y}}）是位於[[北海道|北海道]][[增毛郡|增毛郡]][[增毛町|增毛町]]大字舍熊村字信砂，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[留萌本線|留萌本線]]，已廢除的[[鐵路車站|鐵路車站]]。電報略號為'''ノフ'''。

車站名稱可能源於[[阿伊努語|阿伊努語]]「nup-sam-pet」（沿野外流動的河流）<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/91100P.pdf|title=アイヌ語地名リスト|accessdate=2017-08-17|author=北海道庁|date=|publisher=北海道庁}}</ref>或「nup-sa」（原野中靠近海濱的地方）。

本車站與[[舍熊車站|舍熊車站]]之間的車站距離是0.8km，在2016年[[留萌車站|留萌車站]]至[[增毛車站|增毛車站]]被廢線前是北海道中車站距離最短的。

== 歷史 ==
* 1963年（昭和38年）12月1日 - [[日本國有鐵道|日本國有鐵道]][[禮受車站|禮受車站]]至[[舍熊車站|舍熊車站]]之間設立'''信砂臨時站'''（{{lang|ja|仮乗降場}}），亦開始旅客業務。
* 1987年（昭和62年）4月1日 - 隨著[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）接管。同時車站升格為'''信砂站'''。
* 1990年（平成2年）3月10日 - 設定營業距離。
* 1993年（平成5年）2月 - 車站向增毛方向前進42米。
* 2016年（平成28年）12月5日 - 由於[[留萌車站|留萌車站]]至[[增毛車站|增毛車站]]路段廢除，本車站同時廢站<ref>{{cite web|url=http://www.jrhokkaido.co.jp/press/2016/160428-3.pdf |language=ja |title=留萌線（留萌・増毛間）の鉄道事業廃止届の提出について |publisher=[[北海道旅客鐵道|JR北海道]] |date=2016-04-28 |accessdate=2017-06-13 |archive-url=https://web.archive.org/web/20160429052355/http://www.jrhokkaido.co.jp/press/2016/160428-3.pdf |archive-date=2016-04-29 |deadurl=yes }}</ref><ref>{{cite web|url=http://www.jrhokkaido.co.jp/press/2016/160628-1.pdf |language=ja |title=留萌線（留萌・増毛間）の廃止日繰上げの届出について |publisher=[[北海道旅客鐵道|JR北海道]] |date=2016-06-28 |accessdate=2017-06-13 |archive-url=https://web.archive.org/web/20160701083826/http://www.jrhokkaido.co.jp/press/2016/160628-1.pdf |archive-date=2016-07-01 |deadurl=yes }}</ref>。

== 車站構造 ==
廢站時為1面1線的地面車站。[[側式月台|側式月台]]位於路軌的西北方（增毛方向的右方）。沒有[[轉轍器|轉轍器]]的棒線車站。

由臨時車站開始已經是無人車站。月台北方出入口設置了候車室，為搬遷車站時使用[[装配式建筑|預製組件]]興建。沒有設置[[廁所|廁所]]。搬遷車站後月台為木製甲板式，長度為一輛列車，並安裝了[[導盲磚|導盲磚]]。

1993年，由於信砂河進行改道工程，車站向增毛方向前進了42米。而搬遷前的車站，月台位於路軌的東南方（增毛方向的左方），車站建築物位於月台南方出入口附近，外觀與阿分車站的候車室相似。而月台往增毛方向設有斜坡連接車站外的道路。

== 載客量 ==
* 1992年的每天載客量為12人，每天上下車乘客量為24人。
* 2011年至2015年度平均每天載客量少於10人<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |language=ja |title=平成26年度 線区別の収支状況等について |publisher=JR北海道 |date=2016-02-10 |accessdate=2017-06-21 |archive-url=https://web.archive.org/web/20170407133949/http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |archive-date=2017-04-07 |deadurl=yes }}</ref>。
