== 車站構造 ==
橘線車站為地下兩層[[島式月台|島式月台]]車站，共設有5個出入口，黃線車站部分未定。

== 利用狀況 ==
車站位於[[中正路_(高雄市)|中正三路]]上、錦田路與[[民族路_(高雄市)|民族二路]]間，站名取自其北側之「信義國小」。車站代碼為'''O6'''。{{KMRT ridership-date}}日均進出人次為{{KMRT ridership|station=信義國小站}}，在高雄捷運系統中排名第{{KMRT ridership-rank|station=信義國小站}}名；目前最高紀錄為2018年12月的5,071人次，最低紀錄為2009年3月的2,519人次。由於本站與前後兩站站距極近（其與[[文化中心站_(臺灣)|文化中心站]]間更為目前高雄捷運系統最近站距者），服務範圍因而較小，使用人次不多；本站前之中正路為高雄商辦集中地，大樓林立、商業發達。

== 車站出口 ==
出口1位於車站西端北側，出口2位於車站中段，出口3位於車站西端南側，出口4位於車站東端南側，出口5位於車站東端北側
*出口1：錦田路口（北）、信義國小（西）、新興區行政中心、[[高雄市公共腳踏車租賃系統|高雄市公共腳踏車租賃系統]]
*出口2：開封路口（北）、信義國小（東）
*出口3：錦田路口（南）、高雄市財稅行政大樓
*出口4：民族二路口（南）、高雄商旅飯店、鼎新公園
*出口5：民族二路口（北）、國泰中正大樓
