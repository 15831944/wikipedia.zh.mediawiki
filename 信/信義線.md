'''信義線'''為[[臺北捷運|臺北捷運]]營運中的路線，是[[淡水信義線|淡水信義線]]的一部分，為台北市第2條東西向捷運路網，採用[[高運量系統|高運量系統]]，2004年10月22日，由行政院核定捷運信義線(新十大建設)財務計畫，全線長7.9公里，營運長度6.4公里，造價394億元<ref>https://tw.news.yahoo.com/%E6%8D%B7%E9%81%8B%E4%BF%A1%E7%BE%A9%E7%B7%9A%E9%80%B1%E6%97%A5%E9%80%9A%E8%BB%8A-%E9%A6%AC%E9%83%9D%E4%B8%BB%E6%8C%81%E9%80%9A%E8%BB%8A%E5%85%B8-120131992.html</ref>，皆為地下路線。路線起自[[中正紀念堂站|中正紀念堂站]]，往東穿越金華街，再沿愛國東路行進，後轉往北沿杭州南路，接著向東轉沿[[信義路_(台北市)|信義路]]，經[[信義計畫區|信義計畫區]]至福德街廣慈博愛園區，後沿福德街、中坡南路至玉成公園止，並設置儲車軌1處<ref>{{cite web|title=信義線路線說明|url=http://www.dorts.gov.tw/ct.asp?xItem=69577731&CtNode=70158&mp=115001|publisher=臺北市政府捷運工程局|accessdate=2014-03-29|date=2013-11-25|deadurl=yes|archiveurl=https://web.archive.org/web/20140423081457/http://www.dorts.gov.tw/ct.asp?xItem=69577731&CtNode=70158&mp=115001|archivedate=2014年4月23日|df=}}</ref><ref>{{cite web|title=信義線東延段路線說明|url=http://www.dorts.gov.tw/ct.asp?xItem=1126231&CtNode=32188&mp=115001|publisher=臺北市政府捷運工程局|accessdate=2014-03-29|date=2013-09-13|deadurl=yes|archiveurl=https://web.archive.org/web/20140217234305/http://www.dorts.gov.tw/ct.asp?xItem=1126231&CtNode=32188&mp=115001|archivedate=2014年2月17日|df=}}</ref>。營運模式上與[[台北捷運淡水線|淡水線]][[直通運行|直通運行]]，目前通車至[[象山站|象山站]]，東延段部分目前正興建中。

==簡介==
信義線是台北捷運二期路網的路線之一，當初興建主要的目的，是因原本的台北捷運路網僅有一條東西向路線[[臺北捷運板南線|板南線]]，導致南北向旅次與東西向旅次主要集中於[[台北車站|台北車站]]轉乘而過度擁擠、轉乘空間不足的弊病；再加上隨信義計畫區的開發、[[台北101|國際金融中心]]的興建以及[[台北世界貿易中心|世貿展覽館]]的設立而使信義計畫區周邊的交通問題日益嚴重，預估2021年信義計畫區之尖峰小時旅次量將達2萬人次以上。<ref>http://www.dorts.gov.tw/qa/qa_net.asp?t=下午%2011:33:55#11 {{webarchive|url=https://web.archive.org/web/20160304192248/http://www.dorts.gov.tw/qa/qa_net.asp?t=%E4%B8%8B%E5%8D%88%2011:33:55 |date=2016-03-04 }} 台北市政府捷運工程局-問答集：11.捷運初期路網已完成，市區交通已相當便捷，何以再施作信義線？</ref>

信義線共設8個地下車站，全線車站除中正紀念堂站為[[月台閘門|半高式月台門]]，其餘均裝設[[月台幕門|全高式月台門]]。其中[[台北101／世貿站|台北101／世貿站]]和[[東門站_(台北市)|東門站]]則分別配合[[台北101|101大樓]]興建和[[台北捷運新莊線|新莊線]]動工需求而最先進行發包施工。其餘各車站，包括[[大安森林公園站|大安森林公園站]]、[[大安站_(臺北市)|大安站]]、[[信義安和站|信義安和站]]以及[[象山站|象山站]]，則自2005年4月起陸續進行招標施工，並於同年7月份全面動工<ref>{{cite web |url=http://www.tpml.edu.tw/blind/cgi/dlpic.php?filename=9405011.TXT |title=存档副本 |accessdate=2014-08-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20130927231817/http://www.tpml.edu.tw/blind/cgi/dlpic.php?filename=9405011.TXT |archivedate=2013-09-27 }} 搭乘捷運信義線的美夢100年成真文，沈榮欣作</ref>。此外，因鄰近居民要求，捷運局在象山站後沿福德街向東，新增[[信義松德站|R04]]、[[廣慈/奉天宮站|R03站]]，規劃路線全長7.94公里<ref name="egroup.idv.tw">http://www.egroup.idv.tw/enews/9407/new06072701.htm{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }} 捷運信義線全線動工-不動產ｅ族，2005/07/27</ref>，並在此站以東沿福德街、大道路口、中坡南路至玉成公園止，設置儲車尾軌<ref name=III>[http://www.libertytimes.com.tw/2009/new/dec/24/today-taipei11-2.htm 捷運信義線 延伸2站] {{webarchive|url=https://web.archive.org/web/20120114120526/http://www.libertytimes.com.tw/2009/new/dec/24/today-taipei11-2.htm |date=2012-01-14 }}</ref>。東延段部分原計要設置2站，但因R04站的用地無法取得再加上附近的居民反對聯開導致東延段無法如期動工，因此捷運局決議正式取消R04站的設站計畫<ref>{{Cite web |url = http://www.dorts.gov.tw/ct.asp?xItem=2072069&CtNode=32190&mp=115001 |title = 信義線東延段設計現況 |accessdate = 2014-10-06 |deadurl = yes |archiveurl = https://web.archive.org/web/20141006132851/http://www.dorts.gov.tw/ct.asp?xItem=2072069&CtNode=32190&mp=115001 |archivedate = 2014年10月6日 |df =  }}</ref>，東延段已於2016年11月7日開始動工興建。除東延段尚在興建外，其餘部分皆已通車，信義線自象山站後，軌道繼續延伸約600公尺至信義松德路口以東，為儲車尾軌。

信義線規劃初期採「淡水－象山」與[[台北捷運淡水線|淡水線]]直通的營運模式，但因當時[[台北捷運松山線|松山線]]尚未通車，故先維持「淡水－新店」直通運行，「淡水－象山」的營運模式暫時改為「北投－象山」，而23時以後，比照先前「台北車站－台電大樓」運行模式，改為「台北車站－象山」的營運模式。

本線於通車後初期並無採用列車入站廣播提示，於2014年中才開始採用。{{efn|除東門站及象山站外，各站的廣播提示為「1月台列車往淡水（北投）」、「2月台列車往象山（大安）」；大安站為區間車的端點站，列車抵達該站第二月台後就會變成「2月台列車不載客，請改搭下一班列車」。}}

==列車運行方式==
目前營運模式：
*「淡水－象山」列車：為原本規劃的營運模式，2013年因松山線尚未通車，故先切換為「北投－象山」營運模式因應，2014年11月15日正式啟用。<ref>{{Cite web|url=http://www.trtc.com.tw/ct.asp?xItem=74238706&ctNode=22278&mp=122031|title=捷運松山線通車後 路網回歸原設計 各路線依顏色銜接 旅客省時、省錢、更方便|accessdate=2014-04-08|publisher=臺北大眾捷運股份有限公司|date=2014-04-08|deadurl=yes|archiveurl=https://web.archive.org/web/20140408212221/http://www.trtc.com.tw/ct.asp?xItem=74238706&ctNode=22278&mp=122031|archivedate=2014-04-08}}</ref>
*「北投－象山」列車：2013年信義線通車後為不影響「淡水－新店」的運轉，先直通至淡水線北投站，在2014年的春節假期期間為疏運淡水站眾多旅客曾暫時延長為「淡水－象山」，往新店與往象山的列車則在淡水站交互發車。本營運模式在松山線通車後繼續保留，但只在尖峰時段或台北101與世貿中心舉辦大型活動(如資訊展或跨年)導致台北101/世貿站湧入大批人潮時開行。
*「北投－大安」列車：大安站以東三站（信義安和站、台北101／世貿站、象山站）離峰運量較低，故利用大安站東側的袋狀軌在離峰時段開行區間車，23時以後不開行。
*「淡水－大安」列車：這是「北投－大安」列車於春節期間、假期疏運淡水站湧入的人潮而變更的營運模式。
*「民權西路－象山」列車：因應颱風入境風速達停駛標準時原「淡水－象山」列車改為只行駛至民權西路站為止，直到恢復正常營運為止

未來營運模式：
*「淡水－廣慈/奉天宮」列車：東延段通車後原「淡水－象山」的延伸模式。

過去營運模式：
*「台北車站－象山」列車：這是「北投－象山」列車於晚上11時後的變更模式。

==紀念章==
旅客可以在服務台或諮詢服務台詢問站務員，站務員會拿出紀念印章供旅客使用蓋印。本線的紀念章與既有路線不同，採用「車站專屬紀念章」，各車站的紀念章如下：
*中正紀念堂站：以「中正紀念堂主體」為主圖，結合廣場兩旁造景，增添中式建築景觀之美。
*東門站：以「東門市場」為主圖，結合永康商圈著名美食「小籠包」及「牛肉麵」，呈現該站周邊特色及人文景觀。
*大安森林公園站：以「大安森林公園站」為主圖，結合建國花市花卉，充滿花草綠意之景象。
*大安站：以「大安站捷運高架共構站體」為主圖，結合車站周邊復興南路旁的木棉花，徜徉在木棉道的生活愜意，呈現都市文教景觀。
*信義安和站：以「臨江街觀光夜市」及「文昌家具街」為主圖，呈現周邊商圈的意象。
*台北101／世貿站：以「臺北101」、「世貿中心」作為主圖，展現信義區金融貿易商圈風格。
*象山站：以「臺北樹蛙」、「自然步道」為主圖，呈現當地自然生態平衡。
中正紀念堂站、東門站與大安站都是重要轉乘大站，於2015年2月11日起改用黑墨與星型印章。<ref>http://www.trtc.com.tw/ct.asp?xItem=69474054&ctNode=22278&mp=122031 捷運信義線「車站專屬紀念章」 站站不同 各有特色</ref>

==歷史==
*1982年：原有意規劃由南港經信義直通淡水。
*1993年：地質調查發現，原訂為終點站址的R03站前周邊屬於台北斷層，故將終點站改規劃至[[中強公園|中強公園]]。
*2000年10月：地質調查後將臺北斷層自臺灣活動性斷層之列中移除，而將其歸類為非活動性斷層。評估斷層疑慮可解除，故恢復原路線規劃，並以信義線東延段執行。<ref>{{cite web |url=http://www.dorts.gov.tw/ct.asp?xItem=1126231&CtNode=32188&mp=115001 |title=存档副本 |accessdate=2014年3月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20140217234305/http://www.dorts.gov.tw/ct.asp?xItem=1126231&CtNode=32188&mp=115001 |archivedate=2014年2月17日 |df= }} 臺北市政府捷運工程局 ─ 信義線東延段路線說明</ref>
*2002年11月1日：世貿中心站（後更名為台北101／世貿站）主體工程開工。
*2004年10月22日：行政院核定，捷運信義線(新十大建設)財務計畫。
*2005年7月26日：信義線全面開工。<ref name="egroup.idv.tw"/>
*2005年8月：完成信義線向東延規劃後，提報審議。
