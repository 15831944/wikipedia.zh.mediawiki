'''满洲里市'''（{{mong|m=ᠮᠠᠨᠵᠤᠤᠷ<br />ᠬᠣᠲᠠ|k=Манжуур хот}}）是[[中华人民共和国|中华人民共和国]][[内蒙古自治区|内蒙古自治区]][[呼伦贝尔市|呼伦贝尔市]]下辖的准地级计划单列市，中国最大的陆运口岸城市。总面积730平方公里，全市人口23万多。是一座拥有百年历史的口岸城市，素有“亚洲之窗”之稱。

== 地名 ==
20世纪初的满洲里地区是呼伦贝尔大草原的一部分，因为有一个四季喷涌不止的泉眼，被当地蒙古牧民称之为“霍勒金布拉格”（{{mong|m=ᠬᠥᠭᠵᠢᠬᠦ ᠶᠢᠨ<br />ᠪᠤᠯᠠᠭ|k=хөгжихийн булаг}}），意为“旺盛的泉水”。

1902年的[[東清鐵路|東清鐵路]]修通至此，俄国人将[[满洲里站|火车站]]取名为（[[ISO9|ISO9]]：{{Unicode|Man'čžuriâ}}；意爲[[满洲地区|满洲]]），是东清铁路进入满洲地域的第一个车站。{{Unicode|Man'čžuriâ}}的汉语发音近似“满洲里亚”，从俄语转译汉语时，变成了“满洲里”。后来人们就以“满洲里”这个车站名作为地名了。
[[File:Manzhouli3.jpg|thumb]]

== 历史 ==
这个地方是北方[[遊牧民族|游牧民族]]的历史摇篮、先后为[[东胡|东胡]]、[[匈奴|匈奴]]、[[鲜卑|鲜卑]]、[[契丹人|契丹]]、[[女真|女真]]、[[蒙古族|蒙古]]等民族的领地。满洲里[[清朝|清朝]]属[[黑龍江將軍|黑龍江將軍]]呼伦贝尔副都统衔总管（后为呼伦贝尔副都统），辖区位于黑龙江西部，[[大兴安岭|大兴安岭]]岭北中俄边境。清朝[[雍正|雍正]]年间中国同俄国签订了一系列界约后，开始设官镇守，戌守边疆。
* 1901年（清[[光绪|光绪]]二十七年），俄国根据《[[中俄密约|中俄密约]]》（1896年）建成和[[西伯利亞鐵路|西伯利亞鐵路]]相连的[[東清鐵路|東清鐵路]]，中东铁路西部线建成后，建立火车站，因是从俄国进入中国东北地区（当时惯称“满洲”）的首站，故名“满洲里站”。
* 1903年（清光绪二十九年），设满洲里铁路交涉分局。
* 1905年根据《[[中日满洲善后条约|中日满洲善后条约]]》以满洲里为商埠，城市规模迅速扩大。
* 1908年正式设立满洲里[[海关|海关]]，逐渐成为陆运口岸名城。
* 1909年10月（清[[宣統|宣統]]元年九月），黑龙江行省于满洲里正式设置胪滨府，1912年1月被废止。
* 1920年2月，复设胪滨县。[[中华民国|中华民国]]时期，1923年4月，将满洲里铁路交涉分局改为满洲里市政分局，隶属东省特别区管辖。
* 1927年3月满洲里建制市级行政单位。
* 东北沦陷。1933年1月，滿州國取消[[臚濱縣|臚濱縣]]，设满洲里办事处。1936年1月，改为满洲里市政管理处，1940年改为满洲里街。1941年1月，改为满洲里市，隶属兴安北分省管辖。1943年10月，改隶兴安总省管辖。
* 1945年“九三”抗日战争胜利后，隶属[[民國時期的呼倫貝爾_(嶺西)|呼伦贝尔地方自治政府]]。1949年4月，隶属内蒙古自治区[[呼伦贝尔纳文慕仁盟|呼伦贝尔纳文慕仁盟]]。
* 1953年4月，改由内蒙古自治区直辖。1954年5月，改隶呼伦贝尔盟。1969年7月，与呼伦贝尔盟一起划归黑龙江省管辖，仍隶呼伦贝尔盟。1979年7月，划归内蒙古自治区管辖，仍隶呼伦贝尔盟。
* 1988年，满洲里被中国设立为经济体制改革开放试验区。1990年全市总面积696.3平方千米；总人口13.7万，[[汉族|汉族]]占绝大多数。1992年3月满洲里被国务院批准为首批沿边开放城市后，满洲里经济和社会各项事业实现了历史性跨越。边境贸易持续健康发展，与日本国、俄罗斯、蒙古國等40多个国家和地区建立了广泛的贸易关系。
* 1993年7-9月一些日本国人来满洲里地区实地考察。2002年，满洲里和[[深圳|深圳]]一起被确定为“十五”期间国家重点建设和优先发展的两个陆路口岸。曾经丢失的满洲里就这样在中国经济体质上成为了新生婴儿。

== 地理 ==
满洲里市位于内蒙古呼伦贝尔大草原的腹地，东依兴安岭，南濒呼伦湖，西邻蒙古国，北接俄罗斯，是中国最大的沿边陆路口岸。全市总面积730平方公里。气候属于温带半干旱大陆性气候。满洲里市总人口26万，居住着蒙、汉、回、朝鲜、鄂温克、鄂伦春和俄罗斯等20多个[[民族|民族]]，是一座独领中俄蒙三国风情、中西文化交融的口岸城市，素有“[[东亚之窗|东亚之窗]]”的美誉。
满洲里原称“[[霍勒津布拉格|霍勒津布拉格]]”，蒙语意为“旺盛的泉水”。

==经济==
[[滿洲里口岸|滿洲里口岸]]是中国最大的陆路口岸，承担对[[俄罗斯|俄罗斯]]、[[东欧|东欧]]国家60%的进出口货物。

铁路换装能力达5000万吨。2004年已经扩建完成新的铁路换装货场。2007年满洲里口岸进出口货物达2200万吨。满洲里市拥有国家级开发区2个——边境经济技术合作区（进口资源加工园区）、中俄互市贸易区（对俄出口加工区）和[[扎赉诺尔区|扎赉诺尔]]重化工业基地。

中国[[中华人民共和国国家发展和改革委员会|国家发改委]]2010年初发布《关于2009年西部大开发进展情况和2010年工作安排的通知》，表示2010年将推动[[广西壮族自治区|广西]][[东兴市|东兴]]、[[云南省|云南]][[瑞丽市|瑞丽]]、[[新疆维吾尔自治区|新疆]][[喀什市|喀什]]、内蒙古满洲里等为沿边[[经济特区|经济特区]]。
