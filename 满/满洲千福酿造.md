'''满洲千福酿造株式会社'''是一家位于[[满洲国|满洲国]][[奉天市|奉天市]]（今[[中华人民共和国|中华人民共和国]][[辽宁省|辽宁省]][[沈阳市|沈阳市]]），依照满洲国法律建立的，以酿酒和贩卖为主业的公司。它的母公司是日本企业合名会社三宅清兵卫商店（今{{link-ja|三宅本店|三宅本店|株式会社三宅本店}}）。{{r|nenkan|ootsu|page1=485,486}}

满洲千福是第一家日本企業开设于满洲国的从事酿酒的子公司，也是1940年到1945年间满洲国产量最大的酿酒公司。

== 背景 ==
=== 三宅清兵卫商店 ===
1856年（[[安政|安政]]3年），三宅清兵卫在今日的[[廣島縣|廣島縣]][[吴市|吴市]]地区创立“'''河内屋'''”{{efn|自此历代家主都[[袭名|袭名]]“三宅清兵卫”}}，从事[[味醂|味醂]]、[[燒酎|燒酎]]、[[白酒_(日本酒)|白酒]]的制造。1902年（[[明治|明治]]22年），由于预见到[[吴海军工厂|吴海军工厂]]成立后清酒需求的增加，开始了[[清酒|清酒]]的制造事业。{{r|miyake|hiroshima|jizake}}

1916年（[[大正|大正]]5年），河内屋注册了“'''千福'''”商标。河内屋在成为[[大日本帝国海军|大日本帝国海军]]的供应商后，通过了海军的质量测试：海军将其产品装载于练习舰中，经过200天以上数次跨越赤道的航海后，确认清酒品质依然不变{{r|jizake-takara}}。由此千福的产品在海军各[[鎮守府_(日本海軍)|镇守府]]销路大开。{{r|miyake}}

1925年（[[大正|大正]]14年），'''合名会社三宅清兵卫商店'''{{efn|合名会社即[[无限公司|无限公司]]}}成立。1928年（[[昭和|昭和]]3年），三宅清兵卫商店的产品开始统一使用“千福”品牌{{r|miyake}}。千福酒通过大连向满洲输入{{r|hibi1939}}，1932年的输入量为570.82石，占大连港输入日本酒总量的3%{{r|hibi1934}}。

=== 满洲的酿酒业 ===
==== 传统酒类 ====
在满洲国建立之前，满洲地区出产的酒类有[[高粱酒|高粱酒]]、[[药酒|药酒]]、[[奶酒|奶酒]]、[[果酒|果酒]]等。

高粱酒俗称“白干儿”，是满洲地区传统上广受欢迎的酒类。由于满洲天气酷寒，对蒸馏酒的需求较大，而满洲盛产高粱，故而高粱酒流行。在满洲国建立之前，向[[中国内地|中国内地]]、台湾有较多的输出，满洲国建国后出口量减少。据日本资料记载，满洲国时期高粱酒以[[辽阳市|辽阳]]产为最优{{r|rokusho-1|page=337,338}}。满铁中央试验所对高粱酒进行研究，实现了工业化方式生产，对满洲国和华北的酿酒者造成了冲击{{r|马相金|page1=70}}。

药酒是以高粱酒或黄酒浸泡中药材并加入冰糖等制成的传统酒。在满洲国时期代表性的药酒有[[虎骨酒|虎骨酒]]和史国公酒等{{r|rokusho-2|page1=438}}。奶酒是传统的蒙古饮料，以[[牛奶|牛乳]]、[[马乳|马乳]]为原料，有未经蒸馏的和蒸馏酒两类{{r|rokusho-2|page1=439}}。在吉林、奉天省有山葡萄等野生果类，传统上民间有采集酿制果实酒的习惯{{r|rokusho-3|page1=539}}。

==== 其他 ====
[[伏特加|伏特加]]是俄式蒸馏酒，当时一般酒精度在40到50度左右。满洲国时期伏特加主要在北满生产，从事酿造的有俄罗斯人、满洲国人、希腊人。日资公司大同酒精会社也制造伏特加。{{r|rokusho-3|page1=538}}

[[格瓦斯|格瓦斯]]是含有酒精的俄式饮料。当时在满洲里、哈尔滨等地有少量制造。{{r|rokusho-3|page1=538}}

[[日俄战争|日俄战争]]前夕的1903年，随着[[东清铁路|东清铁路]]的建设，俄罗斯人将饮用啤酒的习惯传播到给了一些当地人。此后30年间啤酒的需求缓慢增加，日本的[[札幌啤酒|札幌啤酒]]经由[[符拉迪沃斯托克|符拉迪沃斯托克]]销往满洲。满洲国建立后，随着日本居民的增加，啤酒的需求增加，在满洲国也出现了不少啤酒工厂。{{r|rokusho-3|page1=541}}

满洲国建立前，本地没有中国传统的[[黄酒|黄酒]]包括[[绍兴酒|绍兴酒]]的酿造事业。1932年，满洲造酒株式会社在抚顺成立，开始酿造绍兴酒。至1939年，在奉天年产量200万斤，抚顺年产100万斤{{r|rokusho-2|page1=436}}。而针对一般意义上的黄酒，日本人曾进行尝试酿造但未有明确成果{{r|rokusho-2|page1=435}}。

=== 满洲的清酒 ===
在满洲，清酒酿造发展较早的是大连、旅顺地区，即[[关东州|关东州]]。满铁附属地发展较关东州稍晚。而在北满的[[满蒙开拓移民|满蒙开拓移民]]也有一些人从事清酒酿造。

满洲的清酒诞生于1910年，铃鹿辩三郎在大连试酿{{r|马相金|page1=68}}清酒。然而当时在满洲生产的大部分日本酒基本仍是浊酒。从浊酒到清酒的转向始于1916年，最早是抚顺的大江惟贤开始过滤浊酒制造清酒，此后一年间，满洲的日本酒生产基本都转向了清酒。然而这一时期酿造失败率较高，多有腐败，满洲的清酒酿造仍然处于摸索期。{{r|hosaka|page=五四}}

到1920年，酿造者数量明显增加，但清酒品质仍然一般，乏善可陈。1922年，[[关东州|关东州]]开始征收酿酒税，1925年，关东厅改订酿酒税法规，并组织了“酒造组合”（酿酒协会），并拨给资金，以图改善质量。到1928年，关东州酿造的清酒质量已经有长足的进步，接近日本内地水平。与关东州相比，满铁沿线各附属地的清酒酿造则较落后。{{r|hosaka|page=五四}}

根据1932年的数据，全满洲地区年产清酒15,100[[石_(容量單位)|石]]，而关东州地区的大连、旅顺占9,000石，约为其中6成。仅次于大连旅顺的奉天、抚顺年产仅为3,000石{{r|hosaka|page=五五}}。同一时期，满洲地区清酒的消费量为一年约35,000石到36,000石{{r|kita|page1=599}}。

[[满蒙开拓移民|满蒙开拓团]]的日本人在北满也有酿造酒类，满洲国政府在1936年起颁发了一些酿酒许可。根据1940年的数据，在[[磐石县|磐石]]、[[密山|密山]]、[[绥稜|绥稜]]、[[佳木斯|佳木斯]]等有日本人酿造清酒、葡萄酒和高粱酒。{{r|nagashima1940|page1=995}}

== 设立 ==
1933年（[[大同_(满洲国)|大同]]2年）12月29日，三宅清兵卫商店出资100万[[满洲国圆|圆]]，在奉天市[[满铁附属地|满铁附属地]]建立了满洲千福酿造株式会社{{r|nenkan|page1=485}}。会社占地7146[[坪|坪]]，厂房为砖砌结构{{r|jizake}}。

满洲千福是日系酒商在满洲国建立的酿酒子公司中最早的{{r|kita|page1=599}}。会社位于若松町72号（今和平区南十马路1号附近）。会长由三宅清兵卫的弟弟三宅清一郎担任。{{r|nenkan|page1=485}}

营业内容为酒类及饮料制品的制造和贩卖以及相关附属业务。决算期为每年9月。{{r|nenkan|page1=485,486}}

== 经营 ==
[[File:满洲千福海报.jpg|thumb]]
截至1939年，满洲千福选用朝鲜、日本本土的大米作为原料，以厂内10台精米机进行脱壳、碾白。尽管当时[[安东省_(满洲国)|安东]]产米，质量仍有所不及，故而未选用。水源采用奉天满铁附属地的地下水{{efn|根据[[夏目漱石|夏目漱石]]1909年的散文，奉天城因为自古没有下水设施，大小便的处理也不彻底，几百年间地下水被粪尿污染，变得酸涩{{r|natsume}}。满洲千福厂址位于满铁附属地西端，距离奉天老城较远，地下水质或有不同。}}，以水泵自百尺深的井下汲取。水的硬度符合当时日本酒4.5度到6度的标准。与当时其他日系酿酒公司一样，聘用的员工为日本人和朝鲜人，而满洲国人被认为不适合这类工作。{{r|hibi1939}}

满洲千福1936年产量为约5,000石{{r|hibi1936}}，1938年左右年产10,000石{{r|hibi1939}}，1940年产量达到17,550石{{r|ootsu}}，1943、1944年产量为16,000石{{r|kureshi|page1=348}}{{efn|1940年和1943、1944年产量来自不同文献，相互有矛盾，姑且采用两者数字。满洲国的清酒产量在1941年达到最高，其后逐年减少{{r|hirai|page1=15}}，则满洲千福1940年产量高于1943、1944年是可能的。}}。1940年，在满洲的酿酒公司超过60家，而1940-1945年间，满洲千福都是满洲国清酒产量第一的公司{{r|ootsu}}。依据1936年的记述，[[热河省_(满洲国)|热河省]][[承德|承德]]米价3斗12圆时（同时期奉天、[[新京|新京]]米价9圆），奉天产的金鹤{{efn|{{link-ja|白鹤酒造|白鶴酒造|白嘉纳家}}（满洲国法人嘉纳酒造株式会社）出品{{r|白鹤|hibi1939}}}}价2.4圆，千代乃春、千福1.8圆，大连产志摩锦、晃阳价格在1.7-1.8圆间{{r|sakamoto1936|page1=65}}。

满洲千福在1939年前后曾考虑扩建在奉天的工厂，然而由于工厂位于满铁附属地而非工业区[[铁西区_(沈阳市)|铁西区]]，周围是住宅故而难以扩张。{{r|hibi1939}}

1945年，满洲国灭亡，满洲千福厂区也被苏军占领。{{r|jizake}}

此外，尽管满洲千福已经设立，日本三宅清兵卫商店（1939年改为三宅本店）的千福酒仍然持续出口到满洲国。{{r|hibi1939}}

== 此后的三宅清兵卫商店（三宅本店） ==
1939年，个人所有的三宅清兵卫商店通过法人化，改为株式会社三宅本店。同年，三宅在青岛设立了'''东亚千福酿造株式会社'''。{{r|ootsu}}

1945年7月，三宅本店所在的吴市遭受美军空袭，所属工厂燃烧殆尽。此后随着日本战败，三宅本店失去了其所有海外资产。其后通过第三代三宅清兵卫的努力。战后10年，三宅本店基本恢复到当年的生产能力，产量在[[灘五鄉|灘五鄉]]以西的地区称雄。{{r|ootsu}}
