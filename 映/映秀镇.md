'''映秀镇'''是[[中华人民共和国|中国]][[四川省|四川省]][[阿坝藏族羌族自治州|阿坝藏族羌族自治州]][[汶川县|汶川县]]的一个镇。映秀镇地处[[阿坝州|阿坝州]]南大门，是进出[[九寨沟|九寨沟]]、[[卧龙|卧龙]]、[[四姑娘山|四姑娘山]]的必经之地。映秀镇处于[[国道213线|国道213线]]和省道303线交汇处，[[都汶高速公路|都汶高速公路]]直达映秀镇。<ref>[http://china.rednet.cn/c/2008/05/14/1507491.htm 映秀鎮，圖組]</ref>

==地理概况==
映秀镇面积115.12平方公里。2004年末，映秀镇有农业人口4181人，8个[[行政村|行政村]]29个村民小组，因[[紫水工程|紫水工程]]移民，2005年有农业人口3387人，7个村，25个村民小组<ref> {{cite web |url=http://www.scjgxx.gov.cn/company/mobon4/index.asp?ep_id=13776 |title=汶川县映秀镇人民政府 |accessdate=2008-05-13 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20080520120402/http://www.scjgxx.gov.cn/company/mobon4/index.asp?ep_id=13776 |archivedate=2008-05-20 }}</ref>，7个行政村分别为：中滩堡村、老街村、枫香树村、黄家村、张家坪村、渔子溪村、黄家院村<ref>[http://www.sxjjpc.gov.cn/wjtz/200811070062_1_0.rar 全国行政区划代码] {{webarchive|url=https://web.archive.org/web/20110707041746/http://www.sxjjpc.gov.cn/wjtz/200811070062_1_0.rar |date=2011-07-07 }}（村、社区级，2008年11月发布），来自[http://www.sxjjpc.gov.cn/wjtz/200811070062.htm 山西省经济普查办公室] {{webarchive|url=https://web.archive.org/web/20110707041839/http://www.sxjjpc.gov.cn/wjtz/200811070062.htm |date=2011-07-07 }}</ref>。

2007年，映秀镇有耕地面积1800亩，已退耕地4950亩，常住人口6906人，其中农业人口4193人，素有“水电之乡”之称<ref name="汶川信息港"> {{cite web |url=http://www.wenchuan.net/Article/ShowArticle.asp?ArticleID=4 |title=映秀镇 |accessdate=2008-05-13 |date=2007-06-20 |publisher=汶川信息港 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20080519055132/http://www.wenchuan.net/Article/ShowArticle.asp?ArticleID=4 |archivedate=2008-05-19 }}</ref>。

映秀镇境内物种丰富，粮食作物主要以[[玉米|玉米]]、[[洋芋|洋芋]]、[[红苕|红苕]]为主，经济作物以[[油菜籽|油菜籽]]以及各种[[蔬菜|蔬菜]]为主。

2002年全镇人均收入1885元，粮食总产量658567公斤，农民人平有粮128公斤，乡镇企业总产值2027万元<ref name="汶川信息港"> {{cite web |url=http://www.wenchuan.net/Article/ShowArticle.asp?ArticleID=4 |title=映秀镇 |accessdate=2008-05-13 |date=2007-06-20 |publisher=汶川信息港 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20080519055132/http://www.wenchuan.net/Article/ShowArticle.asp?ArticleID=4 |archivedate=2008-05-19 }}</ref>。

== 汶川大地震 ==

[[File:Yingxiu_Wenchuan_County_07-28-2005.jpg|right]]
[[File:Yingxiu_001.jpg|right]]
* 2008年5月12日，四川省发生罕见[[汶川大地震|大地震]]，[[震中|震中]]位于映秀镇附近，为[[重災區|重災區]]，全鎮一萬多人口中，數千人死亡。<ref>http://tw.news.yahoo.com/article/url/d/a/080517/5/zg4r.html{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>映秀镇在震后曾长达十餘小时完全失去对外联络，外部救援力量由于地势和天气原因，也极难进入映秀。导致该镇在地震中损失巨大。<ref>http://news.163.com/08/0513/19/4BRKK6F60001124J.html</ref>。
*香港[[無綫新聞|TVB新聞部]]派出逾十隊採訪隊約數十人，包括記者，工程人員等到四川採訪，當中有5隊[[TVB|TVB]][[採訪隊|採訪隊]]從不同徙徑進入震央[[映秀鎮|映秀鎮]]，其中香港TVB採訪隊[[柳俊江|柳俊江]]和另一隊TVB採訪隊[[潘蔚林|潘蔚林]]在惡劣環境中徒步約兩天，直達震央，成為全球第一隊及第二隊進入震央映秀鎮的國際媒體記者，也是首隊中國及國際媒體採訪隊以徙步方式進入震央映秀鎮，以直面災情的畫面報道，其拍的畫面被大部分國際媒體使用，包括[[NHK|NHK]]、[[CNN|CNN]]、[[BBC|BBC]]、[[TVBS|TVBS]]及[[CCTV|CCTV]]等。

* 2008年5月24日[[联合国秘书长|联合国秘书长]][[潘基文|潘基文]]抵达[[汶川|汶川]]映秀镇。[[温家宝|温家宝]]在映秀镇救灾现场与[[潘基文|潘基文]]会面。会晤后，两人一同会见中外记者。<ref>汶川地震大事记 http://news.163.com/09/0512/07/593LP5IF000120GR.html</ref>

* 2008年5月31日，[[解放軍|解放軍]]一架[[Mi-8直升機|Mi-8运输直升机]]（mi-171）在執行地震災區救災任務時，在映秀鎮附近墜毀，造成[[5·31四川救災直升機失事事故|機上18人全部罹難]]。

* 2009年5月11日，汶川县映秀镇灾后恢复重建项目开工暨映秀小学奠基仪式在抗震救灾期间直升机机场原址隆重举行。同时开工的有[[映秀小学|映秀小学]]、映秀幼儿园、[[中滩堡大道|中滩堡大道]]、[[渔子溪桥|渔子溪桥]]和映秀镇安居房五个项目，这是震中的映秀镇灾后重建的第一批项目。<ref>映秀镇灾后重建首批启动五项目 http://www.hnjjb.com/pages.php?xuh=48047{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref>
