'''映灣園'''（[[英文|英文]]：'''Caribbean Coast'''）是[[香港|香港]][[長江實業|長江實業]]、[[和記黃埔|和記黃埔]]及[[地鐵公司|地鐵公司]]（今[[香港鐵路有限公司|港鐵公司]]）的[[東涌站_(東涌綫)|東涌站]]物業發展項目第三期，位於[[新界|新界]][[大嶼山|大嶼山]][[東涌_(香港)|東涌]]健東路1號，為東涌規模最大的私人屋苑，於2002年至2007年分階段入伙。

== 規劃 ==
映灣園共有13幢高座樓宇及8幢低座樓宇，提供5,430個單位，逾半數為三房戶，面積由859至1,295方呎。不過映灣園13幢高座樓宇高達65層，加上西面相連的[[藍天海岸|藍天海岸]]、[[影岸‧紅|影岸‧紅]]及[[海堤灣畔|海堤灣畔]]，共24棟樓宇以「一」字排開的佈局，影響東涌市中心的通風，於2006年被[[環保觸覺|環保觸覺]]指為[[屏風樓|屏風樓]]<ref>[http://www.greensense.org.hk/densebuilding/GreenSense_PressRelease_20061203.doc 2006年12月3日環保觸覺新聞稿]</ref>。

由於屋苑鄰近[[香港國際機場|香港國際機場]]，加上自2010年後東涌市區擴展步伐急速加快，以及周邊有多個新落成的私人屋苑[[昇薈|昇薈]]，故吸引不少機組人員及在機場內工作的人士租住，亦引來投資者購買單位作長線收租。

映灣園分為五期發展：
* 第1座至第5座<ref name=empty_block>映灣園不設第4座，第13座和第14座。13座大樓由第1座排到第16座</ref>為第一期：'''賞濤軒'''（'''Monterey Cove'''），位於健東路2號，於2002年12月入伙，提供1,552個單位。
* 第6座至第8座為第二期：'''映濤軒'''（'''Albany Cove'''），於2003年7月開售，於2004年9月入伙。
* 第9座至第12座為第三期：'''悅濤軒'''（'''Carmal Cove'''），於2004年8月開售，於2005年7月入伙，共提供1,664個單位。
* 第15座及第16座<ref name=empty_block/>為第四期：'''聽濤軒'''（'''Crystal Cove'''），於2006年9月落成，於2007年1月開售，共提供824個單位。
* 洋房部分為第5期'''海珀名邸'''，提供56幢大屋，於2010年3月開售，於2011年5月入伙。

== 商場 ==
[[File:Caribbean_Square_2016.jpg|thumb]]
映灣園設有1個小型商場，名為'''映灣薈'''（'''Caribbean Square'''）商場於2011年7月前稱為'''映灣坊'''（'''Caribbean Bazaar'''）。早期商場的日常生活配套大致能夠滿足映灣園居民的日常需要。不過自從2009年起，[[置富產業信託|置富產業信託]]向[[長實集團|長實集團]]購買映灣坊後，陸續有商戶因為未能夠承擔被租金加幅而結業。有報道指，於2011年時，場內30個租戶中9個為[[地產代理|地產代理]]商舖。諷刺的是現時東涌其他地區的日常生活配套已較2000年代多元化，但映灣園屋苑內的日常生活配套卻較2000年代明顯單一化<ref>{{cite news|title=逼走雜貨店便利店　映灣坊1/3是地產舖|url=http://hk.apple.nextmedia.com/news/art/20110306/15044453|publisher=香港蘋果日報|date=2011-03-06}}</ref>

== 會所 ==
[[File:Caribbean_Coast_Swimming_Pool_Overview_2007.jpg|thumb]]
映灣園整體會所及園林面積逾61萬方呎，為全港最大，以強烈度假風情做賣點。<ref>[http://orientaldaily.on.cc/cnt/finance/20110806/00205_002.html 映灣園擁園林會所 《東方日報》 2011-08-06]</ref>會所名為Club Carribbean，分別以5個主題包裝，有3個會所，其中第1期巴哈馬會所已於2002年12月啟用，設施達42項。包括2個戶外園林沙灘泳池，其中一個長達120米、水力按摩池、草地滾球練習地、兒童遊樂場、綜合多用途運動場、6個網球場、高爾夫球練習場、日式香薰泉及美容中心、健康舞室、綜合電腦遊戲室、保齡球場。

不過攀石場及沙灘排球場使用率非常低。而會所內的一個小食亭及電腦遊戲室，因未有申請相關牌照，自2007年入伙至2010年一直未有開放。<ref>[http://hk.apple.nextmedia.com/news/art/20100524/14061400 映灣園會所　兩成設施虛設 《蘋果日報》 2010-05-24]</ref>

== 配套 ==
=== 區內學校 ===
映灣園屬小學98號校網，中學為離島區。學校包括[[嗇色園主辦可譽中學暨可譽小學|嗇色園主辦可譽中學暨可譽小學]]、[[香港教育工作者聯會黃楚標中學|香港教育工作者聯會黃楚標學校]]、青松侯寶垣小學、寶安商會溫浩根小學、[[靈糧堂怡文中學|靈糧堂怡文中學]]、保良局馬錦明夫人章馥仙中學以及在[[逸東邨|逸東邨]]的[[救世軍林拔中紀念學校|救世軍林拔中紀念學校]]、[[東涌天主教學校|東涌天主教學校]]、[[港青基信書院|港青基信書院]]和[[明愛華德中書院|明愛華德中書院]]外，還有昇薈旁的89區用地預留作興建學校等。
