'''映画伦理委员会'''（{{lang-ja|'''映画倫理委員会'''}}），簡稱「映伦」（{{lang|ja|'''映倫'''}}），是[[日本|日本]]針對所有國內公開放映的[[電影|電影]]進行內容[[審查|審查]]及[[電影分級制度|分級]]之機構，由[[日本電影|日本電影]]業者自主組成，因此並不具有[[公法人|公法人]]身分，而是一個類似[[利益團體|利益團體]]的公益財團法人組織。所有在日本發行的電影，都要經過映倫檢閱分級後方可上映。

== 歷史 ==
=== 背景 ===
[[第二次世界大戰|第二次世界大戰]]結束後，日本的電影開始了前所未有的黃金自由時期，日本的電影工業便從戰後重新快速起步。

然自1960年代以後，受到電視與海外進口影片的衝擊影響，使得日本本土的電影產業陷入了泥潭。雖然自1970年代以後，日本電影憑藉[[恐怖片|恐怖片]]、[[情色片|情色片]]、[[動畫|動畫]]的新式題材重現復甦的景象，但隨即衍生的[[社會問題|社會問題]]如[[性暴力|性暴力]]、[[性犯罪|性犯罪]]率和暴力犯罪率的提高，使得[[日本政府|日本政府]]更重視對於未成年人的保護，並加強電影等傳媒內容的審查。

===映倫成立===
1949年，在實施《映畫倫理規程》後，初建時映倫隸屬於日本影像制作者聯盟，為政府承認的公益法人，其模式尤同[[美國電影協會|美國電影協會]]的自律審查型態。

1957年之後，獨立改組獨立。獨立後的映倫吸收了電影界以外的人士參加影片的審查，主要任務是審查電影腳本、影片，除了故事片外，[[紀錄片|紀錄片]]、新聞片、廣告片均在審查之列，並主張「以電影界以外的客觀公正的眼睛去審查電影」。

日本現行媒體倫理審查組織，大多由業者以外之民間公正人士或學者所組成，然也有由業者自律組成。

== 與美國電影評級審查之差異 ==
雖然映倫和美國的電影審查的演進皆是源自電影業的自律規範，但與美國電影協會評級審查不同的是：在日本所有的影片必須經過映倫的審查之後方可上映。審查過的影片標有「映倫」字樣與編號，未經審查的影片不得在日本全國、以及放映環境衛生同業工會聯合會所屬影院放映。

== 具體分級措施 ==
在電影之分級標準受到美國影響很大，分為四級：
*普遍級（G）：所有年齡段都可以觀賞的影片，影片中雖依劇情可有輕微[[性暗示|性]]、[[暴力|暴力]]描寫等鏡頭，但需儘量控制在有限的範圍內。
*保護級（PG-12）：有[[性愛|性]]、[[暴力|暴力]]、殘酷、[[毒品|毒品]]等描寫，未滿12歲的兒童不可單獨觀賞。或需在家長或保護者陪伴下陪同觀看。
*限制級（R-15）：未滿15歲少年，一律禁止其入場與收看（有虐待描寫）。
*成人級（R-18）：未滿18歲一律禁止入場與收看（有強烈的[[性愛|性]]、[[暴力|暴力]]、[[反社會行為|反社會行為]]；或美化吸毒之描寫）。

== 審查流程 ==
===日本國內電影===
#申請：須依照申請格式填寫，並附帶劇本申請之。
#劇本申請：若台詞等表現在[[人權|人權]]或[[道德|道德]]規範上，有其不適當情形，便會回報給原申請人，並做進一步協議。
#分鏡申請：若在電檢時毛片之分鏡有不明微妙的描寫、效果等情形，會拍攝影片該影格部分進行審查，並與原申請人做進一步協議。
#分級審查：完成上述審查的影片，便會依照內容做進一步分級檢定區分。
===外國電影===
#申請：在經由海關檢驗合格後，須依照申請格式填寫，並附帶相關資料申請。
#審查：在附加日語字幕的情況下，於電影院放映的狀況下進行審查。
:預備審查：若原申請者要求，也可在未加日文字幕的原片，進行放映審查。
:預告片、特報：由於預告片因性質需大量播放至各年齡層關係，所以皆以普遍級尺度進行審查。而超過普遍級尺度的影像、台詞、描寫則不得採用。

== 映倫時期下的色情片 ==
在「映倫」的年代，所有的[[色情片|色情片]]都是合情合理，不是「千百樂」（日本古裝俠義劇情）就是[[紅杏出牆|紅杏出牆]]，其特色就是一定都有劇情。

另外，當時[[成人電影|成人電影]]的最大供應商叫「東寶」，東寶是日式的[[紅磨坊|紅磨坊]]歌舞劇團。後來「東寶」解散後，一些女[[演員|演員]]沒頭路，就去拍成人影片，所以當時的女優（日文對女演員的稱呼）常是一副哀怨深情的模樣。

== 映倫之衰微 ==
1980年代「映倫」式微，主要是錄影帶、[[DVD|DVD]]與[[網際網路|網際網路]]的市場大幅崛起，迫使日本政府修法，私人公司可以拍攝成人視頻而只受簡化後的檢閱，但相對代價是要在三點用[[馬賽克_(影像處理)|馬賽克]]遮蓋；但過沒多久，AV業者就以市場因素（賣不出去）為由跟映倫[[遊說|遊說]]；映倫則以「重點位置」的文字空間做為法令之轉寰餘地，雙方以重點部位的「第三點」做遮罩作為妥協。
