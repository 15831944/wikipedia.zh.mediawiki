'''钍'''（'''Thorium'''，{{标音|拼音=tǔ}}，舊譯作'''釖'''、'''鋀'''）是[[原子序数|原子序数]]为90的[[元素|元素]]，其[[元素符號|元素符號]]為'''Th'''，屬[[锕系元素|锕系元素]]，具有[[放射性|放射性]]。其拉丁文名称來自[[北欧神话|北欧神话]]的雷神[[索尔|索尔]]（Thor）。

钍-232会通过吸收慢[[中子|中子]]而变成可作核燃料之用的[[铀|铀]]-233。钍、铀两种元素是[[核能发电厂|核能发电厂]]最重要的燃料。

== 历史 ==
1828年，[[瑞典|瑞典]]化学家[[永斯·贝采利乌斯|永斯·贝采利乌斯]]在来自一个[[挪威|挪威]]岛屿的{{link-en|釷石|Thorite}}矿中发现了新一种元素，此后以北歐神話的雷神[[索尔|索尔]]命名。1898年，[[玛丽亚·居里|玛丽亚·居里]]和{{link-en|格哈特·施密特|Gerhard Carl Schmidt}}分别并同时发现了钍的放射性。发现90年后，化学家才首次得以分离纯的钍。

== 来源 ==
[[File:MonaziteUSGOV.jpg|thumb]]
钍化物往往可以在[[独居石|独居石]]的沙-{}-裡找到（{{lang|la|(Ce,La,Nd,Th)[PO<sub>4</sub>]}}，包含4-12%的[[二氧化钍|二氧化钍]]，{{lang|la|ThO<sub>2</sub>}}），其它矿物包括与[[鋯石|鋯石]]同晶型的[[方釷石|方釷石]]（{{lang|la|(Th,U)O<sub>2</sub>}}）和釷石（{{lang|la|ThSiO<sub>4</sub>}}）。榍石和锆石也含少量钍。

在地壳里平均每千克物质含7至13毫克钍，也就是说钍是铀的含量的两至三倍。由于钍亲土，因此在所有[[硅酸盐|硅酸盐]]中均含少量钍。

[[澳大利亚|澳大利亚]]、[[挪威|挪威]]、[[斯里兰卡|斯里兰卡]]、[[加拿大|加拿大]]、[[美国|美国]]、[[印度|印度]]、[[拉普兰|拉普兰]]和[[巴西|巴西]]开采钍。[[土耳其|土耳其]]有约80亿吨未开采的矿藏。全世界年开采二氧化钍量估计为100万吨。人的骨骼中每千克骨骼质约含0.002至0.012毫克钍。每天通过食物和饮水一个人约进食0.05至3微克钍。<ref name="Emsley">Emsley J. ''The Elements'', '''1992''', Clarendon Press Oxford</ref>

== 特征 ==
[[File:Decay_Chain_Thorium.svg|thumb]], 俗称“钍系”。]]
纯钍是银白色的[[金属|金属]]，在室温下在-{zh-hans:空气里;zh-hant:空氣裡;}-它稳定，可以数月保持其金属光泽。它逐渐氧化为灰色，最后为黑色。钍的物理特性与它含多少氧化物有关。一般纯的钍含有百分之零点几的二氧化钍。但是也可以制造极纯的钍。纯钍柔软，延展性能很好，可以冷压滚和拉长。钍具有[[多晶型性|多晶型性]]，有两个不同的晶态。在摄氏1400度以上它是面心立方的，在这个温度以下它是体心立方的。水对钍的腐蚀作用很慢。在大多数-{zh-hans:稀酸里;zh-hant:稀酸裡;}-以及在浓盐酸和-{zh-hans:磷酸里;zh-hant:磷酸裡;}-它的溶解速度也很慢。在发烟硝酸和[[王水|王水]]中它溶解迅速。粉状或者屑状的钍在-{zh-hans:空气里;zh-hant:空氣裡;}-加温下很容易自燃。其火焰是白色、明亮的。

== 提取 ==
钍是从氧化钍中提取出来的。在[[氩|氩]]空气中或者在真空中氧化钍被[[钙|钙]]在高温下还原。由于钍会与[[氢|氢]]组成化合物因此无法使用氢来还原。还原后使用[[氟化氢|氟化氢]]冲洗，然后钍被过滤出来。

* [[镭|镭]]{{lang|la|Ra-228}}（[[半衰期|半衰期]]5.75年）
* [[锕|锕]]{{lang|la|Ac-228}}（6.15小时）
* 钍{{lang|la|Th-228}}（1.9116年）
* 镭{{lang|la|Ra-224}}（3.66日）
* [[氡|氡]]{{lang|la|Rn-220}}（55.6秒）
* [[钋|钋]]{{lang|la|Po-216}}（0.145秒）
* [[铅|铅]]{{lang|la|Pb-212}}（10.6小时）
* [[铋|铋]]{{lang|la|Bi-212}}（60.55分钟）

* 两者皆衰变为稳定的铅{{lang|la|Pb-208}}

== 应用 ==
钍的[[氧化物|氧化物]]被用来制作[[白炽罩|白炽罩]]。这样的白炽罩是将布料浸在99%的[[硝酸钍|硝酸钍]]和1%的[[硝酸铈|硝酸铈]]溶液中然后点燃制成的。在高温下硝酸钍分解为[[氧化钍|氧化钍]]和{{What|氧化氮}}。由于钍的放射性，现在已经不再使用钍来做白炽罩了。

=== 原子工业 ===
在[[球床反應堆|球床反應堆]]如{{lang|en|[[THTR-300|THTR-300]]}}中钍可以用来制作铀同位素{{lang|la|<sup>233</sup>U}}：通过中子射击钍{{lang|la|<sup>232</sup>Th}}可以变成{{lang|la|<sup>233</sup>Th}}，后者通过[[镤|镤]]{{lang|la|<sup>233</sup>Pa}}衰变为铀{{lang|la|<sup>233</sup>U}}。铀的这个同位素可以裂变，当作核电站的燃料使用。由于钍比铀常见得多，因此假如未来铀矿逐渐消耗的话钍可能成为重要的能源来源。今天要经济地使用这个工艺还需要解决许多技术和安全问题。

<math>\mathrm{{}^{232}Th\ + n \longrightarrow {}^{233}Th \stackrel{\beta^- 22,3 min} \longrightarrow\ {}^{233}Pa \stackrel{\beta^- 26,967 d} \longrightarrow\ {}^{233}U}</math>

-{zh-hans:<small>公式里;zh-hant:公式裡;}-的时间数据是半衰期</small>

比起铀{{lang|la|<sup>235</sup>U}}和[[钚|钚]]{{lang|la|<sup>239</sup>Pu}}钍有一个重要的优点：它每吸收一个中子的产量高。它的生产效率比目前使用的{{lang|la|<sup>238</sup>U}}和{{lang|la|<sup>239</sup>Pu}}還要高。

这个技术价格高的原因之一是因为产生的{{lang|la|<sup>233</sup>U}}中掺有半衰期很短的{{lang|la|<sup>232</sup>U}}，使得它的放射性非常高。此外在回收钍的时候也有类似的毛病，因为其中掺有放射性高的{{lang|la|<sup>228</sup>Th}}。除此之外生产{{lang|la|<sup>233</sup>U}}有导致核武器扩散的危险。

1960-{zh-hans:年代里;zh-hant:年代裡;}-美国[[橡树岭国家实验室|橡树岭国家实验室]]是最早试验钍循环的实验室之一。他们制造了一座[[熔盐反应堆|熔盐反应堆]]来试验这个技术的可行性。由于缺少经费这个项目于1976年停止。

印度拥有大量的钍矿，因此计划在未来完全仅依靠钍作为其核燃料，将铀完全排挤掉。这个雄心勃勃的计划包括快中子和热中子[[增殖反应堆|增殖反应堆]]。

2007年挪威考虑是否将精力集中在钍循环上。

=== 钍造影剂 ===
{{main|钍造影剂}}
从1931年至1940年代末一种稳定的、胶质的二氧化钍[[混悬剂|混悬剂]]在[[血管攝影|血管攝影]]被作为[[放射性对比剂|放射性对比剂]]使用。但是这个用剂会聚集在[[微血管|微血管]]中，导致局部放射性过高和癌症。胆癌明显与钍造影剂有关，钍造影剂还能引发一般非常少见的恶性肝脏癌症肝[[血管肉瘤|血管肉瘤]]。此外还有钍造影剂导致鼻腔癌的纪录。一般病发发生在使用30至35年后。今天人们使用[[硫酸钡|硫酸钡]]和改善的、有香味的碘化合物取代钍造影剂。

=== 其它应用 ===
为了改善钨惰性气体焊接电极产生火花的性能其电极掺入1至4%的氧化钍。为了降低电子[[功函数|功函数]]放电灯的钨丝掺有1-3%的二氧化钍。在造灯业中氧化钍还以薄膜或者薄片的形式作为[[收气剂|收气剂]]。由于二氧化钍的[[折射率|折射率]]比较高，因此在高质量的光学透镜中也掺有钍。

== 安全 ==
钍的化学毒性估计非常小。它的危险性主要来自其放射性。钍毒性小的原因在于纯钍及其最常见的氧化物的水溶性非常小，在一升水中只能溶0.0001微克的纯钍。只有在酸性非常高、[[pH值|pH值]]低于4的溶液中它的可溶性才提高。此外[[草酸|草酸]]也可以提高钍的水溶性。

=== 放射性 ===
钍同位素{{lang|la|<sup>232</sup>Th}}的半衰期为140.5亿年，它的放射性比铀238还要低。原因是因为由于它的半衰期高，因此它每秒钟衰变的次数少，此外它衰变产物中半衰期比较低的也比较少。钍放射[[α粒子|α粒子]]，因此它被吸入或者食入的话危险比较高。尤其粉状的金属和氧化物能够进入肺，非常危险，会致癌。此外在放置钍和它的化合物的时候要注意它的衰变产物亦可能有放射性，如<sup>208</sup>Tl可以放射出260万电子伏特的伽马射线。

== 化合物 ==
按照其在[[元素周期表|元素周期表]]中的位置钍在它的化合物中一般以正四价出现，正三价和正二价比较少见。而其盐的溶液中，'''只'''存在正四价的钍离子。一个特殊点是[[锕系元素|锕系元素]]没有固定[[非整比化合物|整比的]][[碳化合物|碳化合物]]。

[[二氧化钍|二氧化钍]]（{{lang|la|ThO<sub>2</sub>}}）是所有金属氧化物中熔点最高的，达3300摄氏度。只有少数金属如[[钨|钨]]和少数化合物如[[碳化钽|碳化钽]]的熔点比它高。

[[硝酸钍|硝酸钍]]（{{lang|la|Th(NO<sub>3</sub>)<sub>4</sub>}}）是一种无色的、容易在水和酒精中溶解的化合物。硝酸钍是制造二氧化钍和金属钍过程中的一个重要环节，在制造白炽罩的过程中也有应用。

[[氮化钍|氮化钍]]（{{lang|la|Th<sub>3</sub>N<sub>4</sub>}}）由炽热的钍与空气中的[[氮|氮]]反应而成，有青铜光泽。在空气中它与水汽反应在数小时内分解。

[[碳化钍|碳化钍]]（{{lang|la|ThC<sub>2</sub>}}）是黄色的晶体，其熔点为2655摄氏度。碳化钍在约9K时超导。碳化铀和碳化钍的混合物在高温反应堆中作为燃料。这种燃料是将钍和铀的氧化物与碳混合加温到1600至2000摄氏度制成的。
