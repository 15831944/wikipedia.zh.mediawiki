'''家長昭博'''（{{Bd|1986年|6月13日}}）是一位[[日本|日本]][[足球|足球]][[運動員|運動員]]，現時效力[[川崎前鋒|川崎前鋒]]，司職[[中場|中場]]。

從[[中學|中學]]開始加入[[大阪飛腳|大阪飛腳]]青訓系統，2004年正式轉職業，隔年即協助球隊奪得聯賽冠軍。2008年被租借到[[大分三神|大分三神]]兩年，接著2010年再次被租借至[[大阪櫻花|大阪櫻花]]。也曾被[[日本國家足球隊|日本國家隊]]徵召，於2007年3月參加對[[秘魯國家足球隊|秘魯]]的比賽。

2008年12月，他參加[[普利茅斯足球俱樂部|普利茅夫]]測試。<ref>{{cite web |url=http://news.bbc.co.uk/sport1/hi/football/teams/p/plymouth_argyle/7778111.stm |title=普利茅夫試驗日本新星 |publisher=[[BBC|BBC]]體育 |date=2008-12-11 }}</ref>[[普利茅斯足球俱樂部|普利茅夫]]試圖想簽下他，但最後因工作證問題並未成功。<ref>{{cite web |url=http://www.skysports.com/story/0,19528,11705_4803025,00.html |title=朝聖者錯過家長昭博 |publisher=[[英國天空廣播|英國天空廣播]]體育 |date=2009-01-16 }}</ref>

2010年12月，他與[[皇家馬略卡|馬略卡]]簽下五年合約。2012年，被租借到[[K聯賽|K聯賽]]的[[蔚山現代|蔚山現代]]，及後再借至母會[[日職|日職]]的[[大阪飛腳|大阪飛腳]]。

2014年正式返日效力[[日職|日職]]球會[[大宮松鼠|大宮松鼠]]並成為進攻核心，惟球隊該季以尾三降班。續為該會效力的他終助球會以[[日乙|日乙]]冠軍身分重返[[日職|日職]]。

2017年1月，他宣布由[[大宮松鼠|大宮松鼠]]轉投 [[川崎前鋒|川崎前鋒]] 。
