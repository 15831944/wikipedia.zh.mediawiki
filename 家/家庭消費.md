'''家庭消費'''（英文：Household Consumption）是以家庭為統計單位的消費數字，為滿足家庭成員所需的食衣住行所有相關費用，家庭消費的來源為家庭成員所有收入的[[可支配所得|可支配所得]]。影響家庭消費的因素有以下幾點：家庭成員的總收入水平、家庭的規模、家庭成員的年齡結構、家庭成員的消費觀念、家庭所處的社會環境<ref>{{Cite web |url = http://mall.cnki.net/reference/ref_search.aspx?bid=R200806043&inputText=%E5%AE%B6%E5%BA%AD%E6%B6%88%E8%B4%B9 |title = 《統計大辭典》：家庭消费 | editor = 郑家亨，莫曰达，铁大章，蒋光远|publisher =中國知網|language = zh-cn|accessdate = 2018-07-24}}</ref>。依[[經濟合作暨發展組織|經濟合作暨發展組織]]統計，家庭消費通常佔國家的[[國內生產總值|國內生產總值]]（GDP）的60％左右，所以是經濟需求分析的重要變數，分析內容可能含食品、衣服、住房修繕及水電燃料（租金）、交通、購車、醫療保健、休閒娛樂及其它雜項消費<ref>{{Cite web |url = https://data.oecd.org/hha/household-spending.htm |title = Household spending|publisher =OECD Data|language =en|accessdate = 2018-07-25}}</ref>。
