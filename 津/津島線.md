'''津島線'''（{{jpn|j=津島線|hg=つしません|rm=Tsushima sen|f=y}}）是一條位於[[日本|日本]][[愛知縣|愛知縣]]的鐵路路線，由[[名古屋鐵道|名古屋鐵道]]擁有及營運。
== 路線資料 ==
* 路線距離（{{link-ja|營業距離|営業キロ}}）：11.8公里
* [[軌距|軌距]]：1067毫米
* 站數：8個（包括起終點站）
* 複線路段：全線
* 電氣化路段：全線電氣化（直流1500V）
* [[閉塞_(鐵路)|閉塞方式]]：自動閉塞式
* 認可最高速度：105公里/小時（普通90公里/小時）
* 最小曲線半徑：200公尺（須口站構內。只限上行3號月台進入箇所160公尺）
== 歷史 ==
在名古屋以西（[[尾張國|尾張國]]西部）地區鋪設鐵路的計劃始見於1889年6月的「尾勢電氣鐵道」。然而，由於當時的電氣化鐵路技術未成熟，該計劃未曾付諸實行<ref group="參" name="井戶田3">{{cite book|author=井戶田弘|title=東海地方の鉄道敷設史 3|year=2008}}</ref>{{rp|8}}。其後通車的都是傳統蒸汽火車路線，包括後來被收歸國有的[[關西鐵道|關西鐵道]]，以及現為名鐵[[尾西線|尾西線]]的尾西鐵道<ref group="參" name="井戶田3" />{{rp|10}}。

二十世紀初，電氣化鐵路技術取得突破，鐵路公司亦如雨後春筍般出現。1906年，三所分別名為「尾張電氣鐵道」、「津島電氣鐵道」以及「名古屋電氣鐵道」（下稱「名電」）的公司分別發表名古屋至津島鐵路計劃書<ref group="參" name="井戶田2">{{cite book|author=井戶田弘|title=東海地方の鉄道敷設史 2|year=2006}}</ref>{{rp|90}}。其中名電的津島線與{{tsl|ja|名鉄一宮線|一宮線}}和[[犬山線|犬山線]]合稱「郡部線」，涵蓋名古屋西、北兩面的市郊地區。在該計劃中，津島線從一宮線枇杷島橋站附近的分岔（即今[[枇杷島分歧點|枇杷島分歧點]]）起始，沿古道「津島街道」連接津島<ref group="參" name="井戶田2" />{{rp|92}}。

1907年12月，名電的三線計劃獲得批准<ref group="參" name="社史">{{cite book|author=名古屋鐵道株式會社社史編纂委員會|title=名古屋鐵道社史|year=1961}}</ref>{{rp|52}}<ref group="參" name="年報">{{cite web|url={{NDLDC|805361/41}}|title=鉄道院年報. 明治42年度|work=官報|publisher=[[大藏省|大藏省]]印刷局|year=1910|accessdate=2017-05-01}}</ref>。津島線的測量工作於1910年展開<ref group="參" name="井戶田2" />{{rp|93}}，次年12月正式動工，1914年1月通車<ref group="參" name="社史" />{{rp|56}}。當時所有「郡部線」電車皆經名古屋市內電車軌道往返{{tsl|ja|柳橋駅 (愛知県)|柳橋站 (愛知縣)|柳橋站}}，因此津島線甫通車已提供直達名古屋市中心的服務<ref group="參" name="社史" />{{rp|56}}。

津島線對尾西鐵道的生意構成重大影響。尾西鐵道嘗試興建連接名古屋市的「中村線」對抗名古屋電鐵，然而未能成功。1925年，名古屋電氣鐵道收購了尾西鐵道<ref group="參" name="系譜">{{cite journal|author=中川浩一|title=名古屋鉄道の系譜|journal=鉄道ピクトリアル|publisher=鉄道図書刊行会|issn=0040-4047|issue=246|date=1971-01}}</ref>{{rp|44}}，後者原有路段成為名古屋電鐵的[[尾西線|尾西線]]。1931年，津島線西端與尾西線整合，由一座新的津島站取代之前的津島線「新津島站」及尾西線「津島站」<ref group="名" name="Meitetsu_Exhibition">{{cite web|url=http://www.meitetsu.co.jp/recommend/library/exhibition/detail/1237435_5073.html|title=津島線開通100周年記念展|work=名鉄資料館|date=2014-??-??|accessdate=2017-05-01}}</ref>。

1941年，新的名古屋鐵道東西聯絡線啟用，須口站以東的津島線劃歸「名岐線」（即今[[名古屋本線|名古屋本線]]）<ref group="參" name="百年史">{{Cite book|year=1994|title=名古屋鉄道百年史|author=名古屋鐵道廣報宣傳部|publisher=[[名古屋鐵道|名古屋鐵道]]}}</ref>{{rp|743-744}}。

1959年9月吹襲日本的[[伊勢灣颱風|伊勢灣颱風]]令津島線多處遭受水浸，需要停駛。經過約兩個月的維修，該線逐步重開，期間一度設置津島口臨時站服務來往津島站的乘客<ref group="名" name="Meitetsu_Exhibition" /><ref group="名" name="Meitetsu_Typhoon">{{cite web|url=http://www.meitetsu.co.jp/recommend/library/exhibition/detail/1239687_5073.html|title=伊勢湾台風と名鉄電車|work=名鉄資料館|date=2014-??-??|accessdate=2017-04-19}}</ref>。此外，津島線亦曾於1961年<ref group="名" name="Meitetsu_Typhoon" />、1976年<ref group="參" name="廣井">{{Cite journal|author=廣井脩、市澤成介、村中明等|title=2000年東海豪雨災害における災害情報の伝達と住民の対応|issn=0918-8681|url=http://www.hiroi.iii.u-tokyo.ac.jp/index-houkokusho-rist-tokaigou.pdf|format=pdf|accessdate=2017-04-24}}</ref>{{rp|83}}及2000年<ref group="參" name="近20">{{cite book|author=名鐵120年史編纂委員會事務局|title=名鉄120年：近20年のあゆみ|year=2014}}</ref>{{rp|73-74}}因豪雨或颱風使基建受損而需要暫停服務。

津島線原有貨運列車行走，但於1964年7月1日起終止<ref group="名" name="歷史">{{cite web|url=http://www.meitetsu.co.jp/recommend/library/exhibition/detail/tsushima.html|title=津島線の歴史|work=名鉄資料館|date=2014-??-??|accessdate=2017-05-01}}</ref>。津島站及車站附近鐵路於1968年[[高架鐵路|高架化]]，至2002年高架路段東延至勝幡站<ref group="名" name="Meitetsu_Exhibition" /><ref group="名" name="歷史" />。

== 服務概況 ==
津島線的常規服務為每小時4班列車。於平日早上8時前後、平日晚間，以及週末及假日的日間，每小時加開2至3班列車<ref group="名" name="勝幡">{{cite web|url=http://trainbus.meitetsu.co.jp/pc/T2?sf=%E5%8B%9D%E5%B9%A1&rsf=4293-%E5%8B%9D%E5%B9%A1|title=勝幡駅|work=駅時刻表|accessdate=2017-04-19}}</ref>。大部份津島線列車[[直通運行|直通]]名古屋本線及/或尾西線。西行列車多以佐屋或彌富站為終點，東行列車則主要以須口、吉良吉田或豐明站為終點。

另外，平日晚間有數班由[[知多新線|知多新線]][[內海站_(愛知縣)|內海站]]開出的特急列車經津島線前往佐屋站<ref group="名" name="內海平">{{cite web|url=http://www.meitetsu.co.jp/ekibetsu/timetable/timetable/tt241611.html|title=內海站平日時間表|accessdate=2017-04-19}}</ref>。
