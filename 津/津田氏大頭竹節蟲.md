'''津田氏大頭竹節蟲'''（學名：{{lang|la|''Megacrania tsudai''}}）分類地位為[[動物界|動物界]]（Animal）、[[節肢動物門|節肢動物門]]（Arthropod）、[[昆蟲綱|昆蟲綱]]（Insecta）、[[竹節蟲目|竹節蟲目]]（Phasmida）、[[竹節蟲科|竹節蟲科]]（Phasmida）。此名是一說是為了紀念日本學者[[津田松苗|津田松苗]]，然據台灣昆蟲權威[[朱耀沂|朱耀沂]]博士考證，種小名應是紀念當年任職於林業試驗場恆春分場的[[津田德藏|津田德藏]]。

== 生活史 ==
竹節蟲的生活使經過了卵→若蟲→成蟲等3個過程，是一種[[不完全變態|不完全變態]]的昆蟲，這種成長模式與[[蟋蟀|蟋蟀]]、[[螳螂|螳螂]]、[[蝗蟲|蝗蟲]]、[[蚱蜢|蚱蜢]]一樣，[[若蟲|若蟲]]長得跟成蟲很像，經過幾次[[蛻皮|蛻皮]]後，長出翅膀就變成成蟲。這與[[完全變態|完全變態]]的昆蟲，如[[蝴蝶|蝴蝶]]、[[甲蟲|甲蟲]]完全不一樣，少了[[蛹|蛹]]的階段。

本蟲可進行[[孤雌生殖|孤雌生殖]]，{{fact|但是也有發現雄蟲。|time=2017-02-24}}

== 自衛 ==
一般竹節蟲沒有利齒與銳爪，遇到天敵時無法反擊，便以[[假死|假死]]的方式來逃命。然而津田氏大頭竹節蟲演化出三大自衛方式：

*保護色~身體的顏色與林投樹葉很像，平時會躲在林投葉片的凹陷處，或相互交疊的基部。
*因為林投樹葉外緣有兩排順刺，中央還有一排倒刺，所以很安全。為了自己不至於被樹葉的刺割傷，牠自己會先吃掉會絆到自己要行動的刺。
*會從頭部與胸部間，分泌出特殊白色的體液來反擊，其味道類似薄荷味，有強烈的刺激性以達到驅敵的目的。從三齡若蟲開始可以噴射刺激性防禦物質，其中主要成分以[[獼猴桃鹼|獼猴桃鹼]]為主。

== 奶瓶型蟲卵 ==
獨特的蟲卵像顆植物的種子，遠看又像牠自己所排放出來的糞便。其外觀如同人類嬰兒用的奶瓶型狀，大小如[[黃豆|黃豆]]長約7~8mm，寬約5~6mm。在各種竹節蟲中是屬於大型的卵粒，不過卵粒相當脆弱，只要輕輕的受到擠壓或碰撞，非常容易受傷，尤其是卵蓋的接合處。

== 生態習性與分布 ==
津田氏大頭竹節蟲是[[植食性|植食性]]的昆蟲，其主食是[[林投樹|林投樹]]的樹葉，牠會先啃食林投葉背的倒刺，再吃平行葉脈。而林投樹主要分布在海岸的沿線，目前只在台灣的恆春半島以及台東外海的[[綠島|綠島]]鹿場前有發現牠的蹤跡。
