'''津榆铁路基址'''，位于[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[海港区|海港区]]迎宾路引青园内，属于[[全国重点文物保护单位|全国重点文物保护单位]]“[[秦皇岛港口近代建筑群|秦皇岛港口近代建筑群]]”。

==简介==
[[津榆铁路|津榆铁路]]是[[秦皇岛港|秦皇岛港]]煤炭运输专用线，始建于1881年。1891年，[[清政府|清政府]]在[[山海关|山海关]]设北洋官办铁路局，并于1893年将铁路展至秦皇岛汤河和[[山海关|山海关]]。当时在津榆铁路该段担任运营的“龙号”机车是中国较早的[[蒸汽机车|蒸汽机车]]<ref>郭绘宇，浅述秦皇岛近代港口建筑群的历史沿革及现状，企业文化旬刊2015(7)</ref>。

津榆铁路是中国首条自办铁路、首条标准轨距铁路。现存津榆铁路基址长117米、宽23米，是津榆铁路秦皇岛至汤河段在1915年改线前的路基，位于今秦皇岛市海港区迎宾路秦皇岛引青园内。2004年，作为优秀近现代重要史迹，秦皇岛市人民政府将津榆铁路基址公布为第四批秦皇岛市文物保护单位，并于2004年5月1日树立秦皇岛市文物保护单位标志，还复制了中国第一台蒸汽机车“龙号”模型安置在该基址上。2008年，津榆铁路基址作为“[[秦皇岛港口近代建筑群|秦皇岛港口近代建筑群]]”组成部分被[[河北省人民政府|河北省人民政府]]公布为第五批[[河北省文物保护单位|河北省文物保护单位]]。2013年3月，津榆铁路基址作为“秦皇岛港口近代建筑群”组成部分被[[中华人民共和国国务院|国务院]]公布为[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]<ref name=zgrb>{{Cite web |url = http://www.chinadaily.com.cn/hqcj/xfly/2013-10-09/content_10276219.html |title =秦皇岛市对津榆铁路基址“龙号”机车 模型进行保养维修  |author =  |date = 2013-10-09 |publisher =中国日报网  |language =  |accessdate =  |quote =  }}</ref>。

路基上放置着一辆蒸汽机车车头，是1881年制造的“龙号”机车的复制品。“龙号”机车又名“中国火箭号”（Rocket of China），是中国工人自制的第一辆蒸汽机车<ref>{{Cite web |url =  https://baijiahao.baidu.com/s?id=1567771280526073&wfr=spider&for=pc|title = 城市印迹——河北秦皇岛“津榆”线上的小火车 |author =  |date = 2017-05-19 |publisher =百家号  |language =  |accessdate =  |quote =  }}</ref>。因露天存放，机车表层油饰脱落，锈蚀严重，内部木质结构部分腐烂。2013年，秦皇岛市文物局对“龙号”机车模型进行了保养修复<ref name=zgrb/>。
