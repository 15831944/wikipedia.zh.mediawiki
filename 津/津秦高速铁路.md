'''津秦高速铁路'''是[[中国|中国]]一条连接[[天津市|天津市]]及[[河北省|河北省]][[秦皇岛市|秦皇岛市]]的[[高速铁路|高速铁路]]客运专线，并且是中国《中长期铁路网规划》中[[中国高速铁路#“四纵四横”客运专线|“四纵四横”客运专线网]]的组成部分。津秦客运专线虽然不是“四纵四横”路网内八大专线的组成部分，但由于津秦高速铁路与[[秦沈客运专线|京哈铁路秦沈段]]共同连接[[京哈高速铁路|京哈高速铁路]]及[[京沪高速铁路|京沪高速铁路]]两大干线<ref>{{cite web | url=http://news.enorth.com.cn/system/2009/05/21/004039513.shtml | title=津秦铁路将“直通”沈阳 与秦沈客运专线连接 | publisher=北方网 | accessdate=2013-08-05}}</ref>，是[[中国东北|中国东北]]至[[华北|华北]]、[[华东|华东]]地区的主要通道，因此也被纳入为[[中国高速铁路#“四纵四横”客运专线|“四纵四横”客运专线网]]的一部分。津秦客运专线在[[秦皇岛站|秦皇岛站]]与[[秦沈客运专线|秦沈客运专线]]连接，在[[天津站|天津站]]与[[京津城际铁路|京津城际铁路]]连接，并可经[[天津地下直径线|地下直径线]]与[[京沪高速铁路|京沪高速铁路]]、[[津保铁路|津保铁路]]连接。津秦高速铁路已于2013年12月1日开通运营。

== 工程概况 ==
津秦高速铁路全长261公里，其中[[天津市|天津市]]境内69.1公里，[[河北省|河北省]]境内为192.1公里。全线设计时速为350公里，铺设无碴轨道239.79公里，全线桥隧比例为67.6%，规划运输能力为每年单向8000万人。<ref>{{cite web| url=http://www.wolai.com/news/2009-08-26/49894/| title=津秦客运专线连接京津京沪高铁 将汇聚滨海站| accessdate=2013-08-05| deadurl=yes| archiveurl=https://web.archive.org/web/20131202025630/http://www.wolai.com/news/2009-08-26/49894/| archivedate=2013-12-02}}</ref>共设有8个车站，其中[[军粮城北站|军粮城北站]]、[[滨海西站|滨海西站]]、[[滨海北站|滨海北站]]和[[滦河站|滦河站]]为新建车站，其余均为改造既有车站，此外，卢龙南站为预留车站。津秦高速铁路在天津市界内共设有两座220千伏牵引站，一座是位于东丽区的新兴村牵引站，一座是位于汉沽茶淀镇的留庄牵引站。
。津秦高速铁路于2008年11月8日动工。<ref>{{cite web | url=http://news.enorth.com.cn/system/2009/05/25/004049735.shtml | title=津秦客运专线全线开工 正线全长261公里 | publisher=北方网 | accessdate=2013-08-05}}</ref>津秦客运专线原计划于2012年11月建成，后延期至2013年下半年建成并于12月1日通车。<ref>{{cite web | url=http://www.gov.cn/jrzg/2013-11/25/content_2534310.htm | title=津秦高铁将于12月1日开通运营 | publisher=中央政府网站 | accessdate=2013-11-27}}</ref>

== 建设历程 ==
[[File:炫彩津门0天津西站全景Panorama_of_Tianjin_West_Railway_Station.jpg|thumb]]|300px]]
2006年，津秦高速铁路在规划时拟定在[[天津市|天津市]]、[[滨海新区|滨海新区]]、[[唐山市|唐山市]]、[[卢龙县|卢龙县]]、[[北戴河|北戴河]]和[[秦皇岛|秦皇岛]]六地设站。然而这项规划对设站的安排，被途径的迁安市反对，迁安市以“从行政区域上说，高铁站在秦皇岛内有卢龙、北戴河和秦皇岛3站，而在唐山境内却只有一站，不公平。”的理由赢得上级唐山市政府的支持。2006年4月20日，唐山市政府以“唐政呈[2006]30号”公文，向河北省政府申请支持在迁安设站。2008年，在唐山市和河北省政府的协调下，迁安站南移，调整至迁安与滦县交界处，命名为滦河站。2008年2月29日，津秦高速铁路途径的迁安市市政府致函铁道部，申请设立迁安站，并表示愿意承担因设计调整而增加的费用。不久，迁安方面得到消息，设站基本获准通过，站点就设在该市野鸡坨镇。

2008年11月8日，津秦高速铁路正式动工，最终设[[天津西站|天津西站]]、[[天津站|天津站]]、[[军粮城北站|军粮城北站]]、[[滨海西站|滨海西站]]、[[滨海北站|滨海北站]]、[[唐山站|唐山站]]、[[滦河站|滦河站]]、[[北戴河站|北戴河站]]和[[秦皇岛站|秦皇岛站]]，同时预留卢龙南站。<ref>{{cite web | url=http://news.enorth.com.cn/system/2009/05/25/004049735.shtml | title=津秦客运专线全线开工 正线全长261公里 | publisher=北方网 | accessdate=2013-08-05}}</ref>其中，天津西站划归至京沪高速铁路工程中，联通天津西站和天津站的天津地下直径线作为独立工程施工。然而，更改站点后的方案未重新进行环评。

2008年11月27日，[[迁安市|迁安市]]致函唐山市，提出在站房的修建中，应本着“谁多投资谁多受益”的原则，站名设为“迁滦站”，并表示愿意承担多建部分的投资，比滦县多投资2800万元。但该提议遭到财力相对薄弱的[[滦县|滦县]]反对，最后站名以两地交界处的滦河为名。<ref>{{cite web | url=http://news.hexun.com/2011-06-02/130183551_1.html | title=津秦高铁被叫停调查 迁安滦县争设站致规划变更 | publisher=和讯网 | accessdate=2013-08-07}}</ref>

2011年5月18日，[[中华人民共和国环境保护部|国家环保部]]下发“责令改正违法行为决定书”，指出津秦高速铁路滦县段部分施工地点修改后未重新做环评，违反了《环境影响评价法》第24条的规定。“决定书”要求天津至秦皇岛客运专线项目立即停止建设，并在6月15日之前向环保部重新报批环境影响评价文件。<ref>{{cite web | url=http://news.xinhuanet.com/local/2011-05/19/c_121432768.htm | title=环保部叫停津秦铁路、胶济铁路两条高铁建设运行 | publisher=新华网 | accessdate=2013-08-07}}</ref>该“决定书”是环保部成立以后少数引起社会普遍关注的行政裁决，又发生在[[刘志军|铁道部高层人士调整]]之后，引发了坊间的各种猜测。但对于其对津秦线工期的实际影响，有媒体表示不超过两个月。<ref>{{cite web | url=http://www.chinanews.com/gn/2011/05-21/3056718.shtml | title=滦县段线路变动致津秦高铁被叫停 工程或延期2个月 | accessdate=2013-08-07}}</ref>此前，津秦线虽未能达到2011年上半年通车的预计目标，但自2008年开工以来工程进度一直比较稳定，并未发生大的拖延。

2013年5月16日，[[军粮城北站|军粮城北站]]主体完工，站房总建筑面积近3千平方米。<ref>{{cite web| url=http://www.022net.com/2013/5-17/434958272610815.html| title=天津市军粮城北站主体已完工| publisher=人民网| accessdate=2013-08-05| deadurl=yes| archiveurl=https://web.archive.org/web/20131202025639/http://www.022net.com/2013/5-17/434958272610815.html| archivedate=2013-12-02}}</ref>

7月7日，津秦高速铁路滨海段供电工程完成全部施工，具备通电条件。7月20日起，津秦高速铁路全线6座牵引变电所陆续受电投入运行，7月24日10时30分至25日20时30分，全线接触网送电成功，为即将全面展开的联调联试奠定了基础。7月26日8时30分，55011/2次热滑试验车开始在津秦铁路客运专线进行热滑试验。<ref>{{cite web|url=http://news.enorth.com.cn/system/2013/06/23/011089999.shtml|title=供电工程最后施工 津秦客专滨海段下月联调联试|publisher=北方网|date=201-06-23|accessdate=2013年6月23日}}</ref><ref>{{cite web | url=http://news.xinhuanet.com/local/2013-07/26/c_116703341.htm | title=津秦铁路客运专线进行热滑试验 即将进入联调联试阶段 | publisher=新华网 | accessdate=2013-08-05}}</ref>

2013年8月3日，津秦铁路客运专线开始联调联试。<ref>{{cite web | url=http://www.gov.cn/jrzg/2013-08/03/content_2460716.htm | title=津秦铁路客运专线开始联调联试 | publisher=中央政府网 | accessdate=2013-08-05}}</ref>2013年8月5日，津秦高速铁路滨海北站站房各项工程已基本完工。<ref>{{cite web | url=http://news.enorth.com.cn/system/2013/08/05/011198095.shtml | title=津秦客运专线滨海北站站房各项工程已基本完工 | publisher=北方网 | accessdate=2013-08-05}}</ref>

2013年12月1日，津秦高速铁路正式通车，首趟列车为G420次，由[[天津西站|天津西站]]开往[[沈阳站|沈阳站]]

== 车站列表 ==
津秦高速铁路，自京沪高铁的津沪线路所起（距离北京南站116km），过[[天津西站|天津西站]]（津沪所至天津西段与京沪高速铁路同期运营，这段起初称之为津山客专线天津西段），经[[天津地下直径线|天津地下直径线]]途径[[天津站|天津站]]、[[军粮城北站|军粮城北站]]、[[滨海西站|滨海西站]]（原滨海站）、[[滨海北站|滨海北站]]、[[唐山站|唐山站]]、[[老庄子线路所|老庄子线路所]]、[[滦河站|滦河站]]、卢龙南站（预留）、[[北戴河站|北戴河站]]、[[秦皇岛站|秦皇岛站]]，止于京哈铁路的龙家营线路所（距离北京站308km）。全线共设9座车站3个线路所。其中，天津西站可与天津地铁1号线换乘，天津站可以与[[天津地铁2号线|天津地铁2号线]]、[[天津地铁3号线|3号线]]和[[天津地铁9号线|9号线]]换乘，[[军粮城北站|军粮城北站]]可与天津地铁9号线换乘，并在津滨高铁开通后可连接塘沽和于家堡(滨海南站)。
