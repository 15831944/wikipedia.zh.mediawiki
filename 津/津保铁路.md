'''津保铁路'''，铁路系统称'''津霸客专线'''（天津西-霸州西）'''、霸徐线'''（霸州西-徐水）<ref name="jwb1226">{{cite news|url=http://epaper.jwb.com.cn/jwb/html/2015-12/26/content_6_1.htm|title=津霸客专霸徐铁路28日开通|publisher=[[今晚报|今晚报]]|date=2015-12-26|accessdate=2015-12-26}}</ref>，曾一度被称为'''津保客运专线'''，是连接[[天津市|天津市]]和[[河北省|河北省]][[保定市|保定市]]的铁路。线路起自[[天津西站|天津西站]]，终至[[保定站|保定站]]，线路正线全长158公里，最高设计时速为250公里。工程由[[铁道部|铁道部]]、[[天津市|天津市]]和[[河北省|河北省]]共同建设，工程投资估算总额240亿元，已于2010年3月开始兴建，原计划于2013年建成通车<ref>[http://news.enorth.com.cn/system/2010/03/21/004551977.shtml 津保铁路开建 2013年通车]</ref>，但在建设过程中曾遭遇停工事件所以竣工时间推迟至2015年底<ref>{{cite news|url=http://www.tianjinwe.com/tianjin/ms/qjtj/201412/t20141222_752493.html|title=津保铁路明年底建成通车 未来行程缩短至40分钟|publisher=天津网|date=2014-12-22}}</ref>。根据铁路客户服务中心12306网页公告，津保铁路于2015年12月28日正式开通运营，天津到保定的运行时间由原来的2小时36分压缩至1小时左右。<ref>{{Cite web|url = http://www.12306.cn/mormhweb/zxdt_ggxx/201512/t20151228_4163.html|title = 北京铁路局关于津保铁路开通运营的公告|accessdate = 2015-12-28|author = |date = |publisher = }}</ref>

== 概况 ==
津保铁路工程起自[[天津西站|天津西站]]，终至河北省[[保定站|保定站]]，总工程除包括新建天津至保定铁路，还包括建设本线与[[京广客运专线|京广客运专线]]、[[京九线|京九线]]、[[京沪线|京沪线]]的联络线工程，以及建设[[天津枢纽大北环线|天津枢纽大北环线]]等配套工程。津保铁路全线长157.925公里，将新建铁路133.006公里，利用既有线24.919公里，设天津西站、胜芳站、[[霸州西站|霸州西站]]、白沟站、白洋淀站、徐水站、[[保定站|保定站]]等7个车站。新建段的大部分将采用高架形式，全线桥梁地段占线路长度的65.4%。津保铁路天津段从天津西站至与津冀交界处全长只有19.23公里，经过唯一桥梁子牙河特大桥<ref>[http://news.enorth.com.cn/system/2010/03/21/004551977.shtml 津保铁路开工 全线设7站 大部分采用高架]，北方网，2010年10月28日查阅</ref>。

== 背景 ==
在铁道部商定[[京九铁路|京九铁路]]霸州站的[[京广铁路|京广铁路]]联络线时，曾规划修建保霸铁路并连接[[天津|天津]]和保定。但是河北省为了开发[[黄骅港|黄骅港]]，优先建设由[[朔州|朔州]]至黄骅港的[[朔黄铁路|朔黄铁路]]，不同意先行修建[[保霸铁路|保霸铁路]]。1992年，[[朔黄铁路|朔黄铁路]]和黄骅港工程完工后，河北省省长叶连松在保定、廊坊联合报送的《关于合资修建保霸铁路的报告》上批示“同意修建[[保霸铁路|保霸铁路]]，并与[[津霸铁路|津霸铁路]]对接”。1993年3月，河北省政府向[[铁道部|铁道部]]提交《关于请审批保霸铁路的函》<ref>《关于请审批保霸铁路的函》，河北省人民政府，1993年3月</ref>。1998年，[[国家计划委员会|国家计划委员会]]（[[国家发改委|国家发改委]]的前身）批复了修建保霸铁路的立项问题。国家批准保霸铁路立项后，铁道部和北京铁路局、天津市、廊坊市曾多次派关键人专程到保定会商合资建设保霸铁路，该铁路项目仍然久拖不决，建设经费的领取也遭到了时任河北省副省长[[丛福奎|丛福奎]]的阻挠<ref>[http://www.raildoor.com/hb/html/c29/2009-07/138.htm 津保铁路创下了另类中国之最——国家批准十年仍未开工]{{dead link|date=十月 2017 |bot=Amazingloong }}，李诺，2010年10月28日查阅</ref>。2005年11月22日，铁道部与天津市签署《关于天津铁路建设有关问题的会议纪要》，双方约定铁道部、天津市及相关省共同筹资建设[[津秦客运专线|津秦客运专线]]和保霸铁路，并明确将保霸铁路纳入铁道部“十一五”规划，但是该铁路仍无实质性进展。

== 现状 ==
2008年11月8日，铁道部、天津市和河北省三方《加快天津至保定铁路建设会议纪要》签字仪式在万丽泰达酒店会议中心举行，明确了津保铁路将一次性新建，全长145公里，总投资估算240亿元，预计2009年上半年开工，然而这次协商之后仍未如期开工建设<ref>[http://www.chnrailway.com/news/2010324/201032410291592083302.shtml 深度剖析“三线两站”铁路建设发展]，中华铁道网，2010年10月28日查阅</ref>。2010年3月17日，天津市表示，经积极争取，国家发改委正式核准批复天津至保定铁路项目。津保铁路于2010年3月正式开工建设，将于2013年建成，届时从天津到保定可实现1小时内通达。保定市目前正在争取将津保铁路向西沿伸至[[山西省|山西省]]的[[大新站|大新站]]，即修建“天保大”铁路（天津--保定--大新）<ref>[http://leaders.people.com.cn/GB/9059380.html 保定市长回复网友提出的保霸铁路建设等6条留言] {{Wayback|url=http://leaders.people.com.cn/GB/9059380.html |date=20101222172503 }}，人民网，2010年10月28日查阅</ref>。

===停工事件===
在发生[[2011年甬台温铁路列车追尾事故|7.23甬温线动车追尾事故]]后，铁道部大规模压缩铁路建设投资，根据多方面的报道，目前津保线的铁道部一方投资迟迟不能到位，因而陷入停工状态。<ref>{{cite web|url=http://money.163.com/11/0923/20/7ELPIG6V00253B0H.html|title=铁道部断炊资金供给 津保铁路全线停工|publisher=网易|date=2011-09-23|accessdate=2011年10月3日}}</ref>迟至2012年3月，重新全面复工<ref>{{cite news|url=http://zqb.cyol.com/html/2012-03/23/nw.D110000zgqnb_20120323_14-06.htm|title=津保铁路建设吹响复工的号角|publisher=[[中国青年报|中国青年报]]|date=2012年3月23日|accessdate=2012年6月15日}}</ref>，但原定2013年完工的工期已无法实现。

截至2014年10月，由天津至保定的列车仅三次（K7728/7725、K7748/7745及[[T5682/5683、T5684/5681次列车|T5684/5681]]），且仅K7748/7745次列车不经停北京西站。由于津保铁路施工涉及徐水火车站（含京广正线）的改造，京广线上的近百班旅客列车计划于2015年10月中下旬临时停运。<ref>{{cite news|url=http://yanzhao.yzdsb.com.cn/system/2015/08/26/014284608.shtml|title=受铁路施工影响10月中下旬河北省近百趟列车将临时停运|publisher=[[燕赵都市报|燕赵都市报]]|date=2015-08-26|accessdate=2015-08-28|deadurl=yes|archiveurl=https://web.archive.org/web/20151001023319/http://yanzhao.yzdsb.com.cn/system/2015/08/26/014284608.shtml|archivedate=2015-10-01}}</ref>

===联调联试===
2015年10月20日，津保铁路开始联调联试。<ref>{{cite news|url=http://epaper.yzdsb.com.cn/201510/21/560364.html|title=津保铁路昨起联调联试 12月底将正式开通运营|publisher=[[燕赵都市报|燕赵都市报]]|date=2015-10-21|accessdate=2015-10-22|deadurl=yes|archiveurl=https://web.archive.org/web/20151117031151/http://epaper.yzdsb.com.cn/201510/21/560364.html|archivedate=2015-11-17}}</ref>同年11月8日，[[中国铁路高速综合检测列车|综合检测列车]]开始在津保铁路上试运行。<ref>{{cite news|url=http://news.enorth.com.cn/system/2015/11/10/030619801.shtml|title=津保试跑：津保铁路联调联试进入检测试跑阶段|publisher=[[天津日报|天津日报]]|date=2015-11-10|accessdate=2015-11-13}}</ref>津保铁路在同期公布的2016年新增列车信息中分段显示为'''津霸客专'''、'''霸徐线'''。<ref>{{cite news|url=http://www.12306.cn/mormhweb/zxdt_ggxx/201511/t20151113_3876.html|title=济南铁路局关于2015年底调整列车运行图的公告|publisher=[[中国铁路客户服务中心|中国铁路客户服务中心]]|date=2015-11-13|accessdate=2015-11-13}}</ref>津保铁路于2015年12月17日进入最后验收阶段，验收合格后方可投入运营，届时从石家庄至天津高铁直达只需1.5个小时。<ref>{{cite news|url=http://yanzhao.yzdsb.com.cn/system/2015/12/18/014346544.shtml|title=津保铁路昨日进入最后验收阶段|publisher=[[燕赵都市报|燕赵都市报]]|date=2015-12-18|accessdate=2015-12-18|deadurl=yes|archiveurl=https://web.archive.org/web/20151222124339/http://yanzhao.yzdsb.com.cn/system/2015/12/18/014346544.shtml|archivedate=2015-12-22}}</ref>

津保铁路于2015年12月28日开通， 初期开行旅客列车4.5对<ref>{{Cite web|url = http://www.12306.cn/mormhweb/zxdt_ggxx/201512/t20151228_4163.html|title = 北京铁路局关于津保铁路开通运营的公告|accessdate = 2015-12-28|author = |date = |publisher = }}</ref><ref>{{cite news|url=http://epaper.yzdsb.com.cn/201512/28/567690.html|title=今起津保铁路正式开通运营 石家庄至天津实现90分钟高铁直达|publisher=[[燕赵都市报|燕赵都市报]]|date=2015-12-28|accessdate=2015-12-28}}{{dead link|date=十月 2017 |bot=Amazingloong }}</ref>，2016年1月10日增加正式班次，目前每天有26对旅客列车经由该线。<ref name="jwb1226"/>
