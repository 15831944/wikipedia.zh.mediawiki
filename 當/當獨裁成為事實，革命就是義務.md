「'''當獨裁成為事實，革命就是義務'''」（{{Lang-en|'''When dictatorship is a fact, revolution becomes a duty.'''}}）是源自於[[柏林自由大學|柏林自由大學]]哲學系教授[[帕斯卡·梅西耶|帕斯卡·梅西耶]]的文學著作《[[里斯本夜車|里斯本夜車]]》以及同名{{Link-en|里斯本夜車 (電影)|Night Train to Lisbon (film)|電影作品}}所翻譯而成的字句，其中該作品主要是描述1974年於葡萄牙爆發的[[康乃馨革命|康乃馨革命]]<ref name="主場新聞">{{Cite web|url =http://thehousenews.com/politics/%E7%95%B6%E7%8D%A8%E8%A3%81%E6%88%90%E7%82%BA%E4%BA%8B%E5%AF%A6-%E9%9D%A9%E5%91%BD%E5%B0%B1%E6%98%AF%E7%BE%A9%E5%8B%99/|title =「當獨裁成為事實，革命就是權利」|author =|date =2014年3月21日|publisher =[[主場新聞|主場新聞]]|language =zh-hant|accessdate =2014年4月24日|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20140324005321/http://thehousenews.com/politics/%E7%95%B6%E7%8D%A8%E8%A3%81%E6%88%90%E7%82%BA%E4%BA%8B%E5%AF%A6-%E9%9D%A9%E5%91%BD%E5%B0%B1%E6%98%AF%E7%BE%A9%E5%8B%99/|archivedate =2014年3月24日}}</ref>。而該字句的內容則是改編自法國浪漫主義作家[[维克多·雨果|维克多·雨果]]的名言<ref>[http://ikazeiwacu.fr/2013/09/02/la-lettre-du-nord-l-apartheid-dans-le-systeme-de-ladmnistration-du-fpr/ La LETTRE DU NORD: L’ APARTHEID dans le système de l’admnistration du FPR. | IKAZE IWACU] {{webarchive|url=https://web.archive.org/web/20140914114402/http://ikazeiwacu.fr/2013/09/02/la-lettre-du-nord-l-apartheid-dans-le-systeme-de-ladmnistration-du-fpr/ |date=2014-09-14 }} Victor Marie Hugo, dans son livre de 1860 « La légende des siècles », a déclaré que « quand la dictature est un fait, la révolution devient un droit « .</ref>「當獨裁是一項事實，革命就成為一種權利<ref>[http://blog.udn.com/hanklin1018/12209839 「當獨裁是一項事實，革命就成為一種權利」才是正確的譯法 - 林錦泉 的網誌 - udn部落格]</ref>」（When dictatorship is a fact revolution becomes a right）<ref>{{Cite web |url = http://www.korpisworld.com/quotes/Victor%20Hugo.htm |title = Victor Hugo Quotes |author =  |date =  |publisher = Korpisworld.com |language = en |accessdate = 2014年4月24日 |quote = }}</ref>。

==引用 ==
[[File:318太陽花學運_(13553374193).jpg|thumb]]（國會）議場外牆上的黑色噴字「當獨裁成為事實，革命就是義務」]]
2014年台湾的[[太陽花學運|太陽花學運]]中，這段話被喷在立法院現場，並且成為之後反對違反朝野協商結果逕行通過《[[海峽兩岸服務貿易協議|海峽兩岸服務貿易協議]]》的行動宣言之一。3月21日上午，抗議學生持續集結在立法院青島東路側門周邊，同時學生也發起「街頭民主教室」進行相關議題之演講。稍後前民主進步黨主席[[施明德|施明德]]女兒施蜜娜爬到[[立法院|立法院]]議場二樓北側外牆，並且準備使用黑漆噴上「當獨裁成為事實  革命就是義務」字句。然而在噴上「當」和「獨」兩個字的時候施蜜娜立即被在場學生與糾察隊制止，同時現場群眾還齊聲大喊「停止噴漆」。對此施明德的妻子[[陳嘉君|陳嘉君]]進入議場內和抗議學生溝通，並且表示這是公民自主的發聲方式之一。

在中止行動半個小時後，施蜜娜在陳嘉君等人的協助下其重新爬上鋁梯上繼續噴漆，其中陳嘉君在現場還表示：「所有的訴求，都在這個行動裡面已經說明了一切。因為『當獨裁成為事實，革命就是義務』，也就是今天學生占領立法院的所有行動的理由。<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/972799 |title = 佔領國會》「當獨裁成為事實，革命就是義務」|author = |date = 2014年3月21日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2014年4月24日 |quote = }}</ref><ref name = "主場新聞"/>之後施蜜娜針對這次噴漆行動發表聲明，並且表示整個行動如果使政府覺得已經犯罪的話將會負起所有責任。到了下午整個噴漆行動才正式完成，對此聚集在立法院外的抗議群眾紛紛鼓掌支持。<ref name = "主場新聞"/>

學生代表[[江昺崙|江昺崙]]在接受採訪時表示學生尊重他們的行動以及言論自由等權利，因此雖然沒有事先邀請他們噴漆但是也不會特意制止行動，但是必須由參與成員自行負擔法律責任。<ref>{{Cite web |url = http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS1/8562865.shtml |title = 議場外牆噴漆女子 是施明德16歲女兒 |author = 胡宥心 |date = 2014年3月21日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2014年4月24日 |quote = }}</ref>不過活動單位對此也表示，「當獨裁成為事實，革命就是義務」的字句也將會是這次活動唯一會留下來的噴漆文字。<ref>{{Cite web|url =http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS1/8562518.shtml|title =立院外牆噴字「當獨裁成事實 革命就是義務」|author =林澔一|date =2014年3月21日|publisher =[[聯合新聞網|聯合新聞網]]|language =zh-hant|deadurl =yes|archiveurl =https://web.archive.org/web/20140324015901/http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS1/8562518.shtml|archivedate =2014年3月24日}}</ref>
