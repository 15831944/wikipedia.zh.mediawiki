《'''當代男兒'''》是[[華星唱片|華星唱片]]發行的音樂[[合輯|合輯]]，於1991年推出<ref>{{Cite web |url =http://cis.ntl.edu.tw/F/TGV82UQAVR2MVUYLAGYK7GG1ECVSJA5FK3HPTSKUB3NPADR4G8-08149?func=full-set-set&set_number=000028&set_entry=000015&format=999|title =當代男兒[錄音資料]-張國榮, 劉德華, 黎明演唱|date = 2005年5月27日 |publisher =視聽館藏資料-國立臺灣圖書館|language = 中文}}</ref>，專輯收錄了十歌曲<ref>{{Cite web |url =http://comebacktolove.blogspot.com/2016/09/1991_82.html|title =張國榮 劉德華 黎明 - 當代男兒 (1991)|publisher =80-90年代廣東歌-come back to love|language = 中文}}</ref><ref>[[黎明音樂作品列表|黎明音樂作品列表]]</ref><ref>[[張國榮音樂作品列表|張國榮音樂作品列表]]</ref><ref>[[劉德華音樂作品列表|劉德華音樂作品列表]]</ref>，《恕未從俗》為首次收錄的歌曲<ref>{{Cite web |url =http://comebacktolove.blogspot.com/2016/09/1989-tvb.html|title =黎明 - 恕未從俗 (1989) TVB劇"回到唐山"主題曲|publisher =80-90年代廣東歌-come back to love|language = 中文}}</ref>。
