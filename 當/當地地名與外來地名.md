'''外來地名'''是指外方語文對一個地方使用與當地語文不同的名稱。相對地，當地語文所用的地名稱為'''當地地名'''。

== 實例 ==
# 中文的[[北京|北京]]是當地地名，法文的{{lang|fr|Pékin}}是外來地名；中文的[[中國|中國]]是當地地名，英文的{{lang|en|China}}是外來地名。
# 意大利首都[[羅馬|羅馬]]的意大利文{{lang|it|Roma}}是當地地名，英文的{{lang|en|Rome}}是外來地名；俄文的{{lang|ru|Россия}}是當地地名，中文的[[俄羅斯|俄羅斯]]是外來地名。
# 英文的{{lang|en|San Francisco}}是當地地名，中文的[[旧金山|旧-{}-金-{}-山]]是外來地名。 <!-- 那粵語裡的「三藩市」怎麼算呢？ --> 
# 英文的{{lang|en|Christchurch}}是當地地名，中文的[[基督城|基督城]]是外來地名，中國正式的漢譯名[[克賴斯特徹奇|克賴斯特徹奇]]也是外來地名因為發音不同。有時即使拼寫相同，發音差異也會使一個地名在不同語文成為外來地名。
# [[法國|法國]]首都[[巴黎|巴黎]]的法文{{lang|fr|Paris}}是當地地名，英文和德文的{{lang|en|Paris}}是外來地名，因為這兩種語文的發音和法文不同；法文中{{lang|fr|Paris}}的s不發音，但英文和德文都發音。
# [[東京|東京]]在日文為當地地名，在中文為外來地名。
# 泰語的[[曼谷|估忒]]（{{lang|th|กรุงเทพ}}）是當地地名，全世界所稱的曼谷（{{lang|th|บางกอก}}）是外來地名

== 減少使用的趨勢 ==
20世紀後期，使用外來地名引起了爭議。當地人不喜歡有貶低意味的外來地名。人們也避免使用涉及敏感歷史的外來地名。近年來[[地理學家|地理學家]]致力於減少外來地名。例如：現在西班牙文稱[[土耳其|土耳其]]首都[[安卡拉|安卡拉]]為{{lang|es|Ankara}}，不用以前的外來地名{{lang|es|Angora}}。[[聯合國|聯合國]]建議把外來地名的使用盡量減至最少。

有時政府會要求外國應使用當地地名，放棄外國沿用的外來地名。例如：1985年[[科特迪瓦|-{科特迪瓦}-]]政府要求外國一致使用該國的法文國名{{lang|fr|Côte d'Ivoire}}，不用外來地名。中文的「-{象牙海岸}-」或英文的「{{lang|en|Ivory Coast}}」，因此中文改稱「-{科特迪瓦}-」，但在中華民國（台灣）等地仍沿用「{{lang|zh-tw|象牙海岸}}」。2005年[[韓國|韓國]]政府要求首都的漢譯名，從外來地名漢城改為當地地名[[首爾|首爾]]。2018年[[白俄羅斯|白俄羅斯]]要求將其國名的漢譯名改為當地地名[[白羅斯|白羅斯]]。
