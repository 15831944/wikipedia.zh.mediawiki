《'''假面騎士響鬼'''》是2005年1月30日至2006年1月22日，於[[朝日電視台|朝日電視台]]毎週日早上8:00～8:30播出的《[[假面騎士系列|假面騎士系列]]》的第6部[[平成|平成]]系列作品，全48集。

原名为《'''音擊戰士響鬼'''》，但基于商业考量，仍掛名「假面骑士」。

* 劇場版　'''[[幪面超人響鬼與七人的戰鬼|假面騎士響鬼與七人的戰鬼]]'''　（2005年9月3日上映）

台灣於2007年9月30日起於Momo親子台放映，每週日18:00～18:30播出（後改為每週日19:00～19:30播出），2008年8月31日播畢 。

香港於2007年11月24日起於無綫電視翡翠台放映，每週六12:30～13:00播出，2008年11月29日播畢。

== 故事 ==
自古以來，在日本有很多為了保護人類，而在深山進行嚴苛的修行變身成為「鬼」，以淨化之音與「魔化魍」戰鬥的人。這樣做是為了對抗魔化魍。漸漸的，鬼按照自己的能力進行分類，並且變成可以被繼承的流派，更與那些曾被自己所救的人類，組成一個名叫「猛士」的組織。

馬上就要高中應考的少年安達明日夢，在往[[屋久島|屋久島]]的船上，看到了一個奇怪的男子。突然一名小孩從甲板失足掉下，一瞬間，明日夢看到了不敢相信的事情，剛才的男子以超越常人的力量救回小孩。男子對明日夢說了一句：「著實地鍛鍊了一番！」之後離去。

到達[[屋久島|屋久島]]的明日夢，在森林裏，被巨大的土蜘蛛襲擊。這時，之前那名奇怪的男子再次出現，並且變成一個紫色的身影。

這正是「響鬼」與明日夢相遇的一刻。

== 鬼（音擊戰士）（本作登場的假面騎士） ==
又稱為音擊戰士，共分三類，分別是使用「[[打擊樂|打擊樂]]」、「[[管樂|管樂]]」及「[[弦樂|弦樂]]」的三種戰士。由於每種樂器針對的是不同類型的魔化魍，所以音擊戰士們都會獨立行動。而專門負責支援這些鬼的人們，則在幕後組成了名為「猛士」的組織。這個組織的規模甚大，不同的分部在全日本各地互相聯繫，交換魔化魍的情報，再由鬼出擊戰鬥，保護人類。有別於其它假面騎士系列作品，鬼解除變身之後會連同衣服一起消失，所以鬼多數戰鬥完後先解除頭部，然後回去分部的更衣室完全解除變身再更衣。

:210cm  95kg  100m只需3s  一跳即可122m  拳力7.3t  踢力15t（通常时）

:威吹鬼的武器「烈風」有兩種型態，分別是射擊模式和吹奏模式，平時以射擊模式（銃槍型態）應戰，當威吹鬼透過「烈風」將大量子彈「鬼石」打進魔化魍體內後，他就會利用「烈風」的吹奏模式（[[小號|小號]]型態）向魔化魍吹起「清澈音」，令「鬼石」迅速膨脹，從而擊破魔化魍（必殺音擊：'''音擊射 疾風一閃'''）。

:威吹鬼擅長遠程袭击，且速战速决，攻击快准狠，力量在所有鬼之上。同樣能運用不同的光碟獸，有專用的[[水|水]]屬性'''鈍色[[蛇|蛇]]'''，方便於水裡偵測敵蹤。另外，威吹鬼有一名女性弟子'''天美晶'''（[[秋山奈奈|秋山奈奈]]飾），同樣屬於'''管系'''的見習戰士，能變身成天鬼。

=== 非主線情節的鬼 ===
==== 凱鬼（Kachidoki） ====
'''打擊系'''音擊戰士，，現為守護東北區的鬼。(於第3話曾提及過)

==== 曉鬼（Akatsuki） ====
原名'''津村努'''。'''打擊系'''音擊戰士，後放棄成為鬼。(於第25-26話登場)

==== 導鬼（Michibiki） ====
已退休，本姓'''和泉'''。'''管系'''音擊戰士，武器為小號。'''威吹鬼'''的父親，猛士五鬼之一，為守護關西區的鬼。

==== 吹雪鬼（Fubuki） ====
已退休，'''管系'''音擊戰士，武器為銅管樂。'''威吹鬼'''的母親，現與'''導鬼'''於吉野居住。

==== 山吹鬼（Yamabuki） ====
曾在'''響鬼'''的記事簿見過這名字，使用小鼓，其餘一切皆不詳。

==== 弦鬼（Genki） ====
初代能使用鬼力量的人，武器為結他。曾於江戶時代於日本中部戰鬥。

==== 初代斬鬼（First Kamen Rider Zanki） ====
'''財津原藏王丸'''(現任'''斬鬼''')的師父，突然死亡，要'''財津原藏王丸'''的師姐'''朱鬼'''指導接替下任斬鬼。

==== Amaki ====
於'''Hibiki'''世界出現之假面騎士。
漢字為「天鬼」，曾在原作中的第41集出場(假面騎士Decade)，只是單單以「天美晶的變身體」來稱呼，並沒有正式名稱。在'''Decade'''中才有正式的名稱。
* 主體顏色：藍色及黑色
;音擊武器
* 音擊管・烈風
* 音擊鳴・鳴風
;必殺技
* 音擊射・疾風一閃

== 音樂 ==
這個作品，用了很多配樂來凝聚出一個很特別的氣氛：日常場面滑稽的氣氛和跟魔化魍的對峙場面。

在開場曲的「輝」裏用了不同的樂器，在威吹鬼活躍的時候用喇叭，轟鬼活躍的時候用吉他等等。

第56届NHK[[紅白歌唱大賽|紅白歌唱大賽]]在白組的[[布施明|布施明]]唱「少年よ」的時候，響鬼、威吹鬼、轟鬼也在旁邊客串演出。

細川在最後也出現和布施握手，並對觀眾說'''「響鬼，鍛鍊了一番！」'''。
