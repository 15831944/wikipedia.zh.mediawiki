'''赵毅'''（{{bd|1981年|7月16日}}），中国男配音演员、演员。

===生平==
毕业于[[解放军艺术学院|解放军艺术学院]]表演系。父亲[[赵述仁|赵述仁]]与妻子[[褚珺|褚珺]]都是配音演员。代表作有电视剧《[[天龙八部|天龙八部]]》的虚竹、《[[独孤天下|独孤天下]]》的宇文护，网剧《[[无间道|无间道]]》的江子丹<ref>{{Cite web |url=http://www.chinanews.com/yl/2016/12-30/8109722.shtml |title=网剧《无间道》配音遭质疑 幕后配音演员回应|accessdate=2018-04-14 |author= |date=2016-12-30 |work=张曦 |publisher=中国新闻网|language=Zh-hans }}</ref>，动画《[[开封奇谈|开封奇谈]]》的白玉堂，游戏《[[古剑奇谭二：永夜初晗凝碧天|古剑奇谭二：永夜初晗凝碧天]]》的乐无异<ref>{{Cite web |url=http://gjqt.wangyuan.com/gj2/2013/06/05/001007.shtml|title=乐无异——配音演员：赵毅|accessdate=2018-04-14 |author= |date=2013-06-05 |work= |publisher=《古剑奇谭二》官方网站|language=Zh-hans }}</ref>等。
