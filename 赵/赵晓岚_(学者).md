'''赵晓岚'''{{BD|1953年|6月3日|||Z赵}}出生于[[湖南省|湖南省]][[长沙市|长沙市]]，毕业于[[湖南师范大学|湖南师范大学]]中文系，现为[[湖南师范大学|湖南师范大学]]教授。<ref>{{cite web |url=http://kejiao.cntv.cn/20100402/100100.shtml |title=百家讲坛专家资料库之赵晓岚 |accessdate= 2010年|work=|publisher=cctv官方网 |language = zh-hans}}</ref>

== 简介 ==
1953年，赵晓岚出生于[[湖南省|湖南省]][[长沙市|长沙市]]。<ref name="hnsx ">{{cite web|url=http://news.hunnu.edu.cn/index.php/sdywmore/1789.html|title=赵晓岚作客“百家讲坛”广受青睐|accessdate=2008年8月2日|work=|publisher=湖南师范大学官方网|language=zh-hans}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

1970年，赵晓岚在[[长沙市一中|长沙市一中]]毕业，并且在该校留校任教，一度长达12年。<ref name="hnsx " />

1982年，赵晓岚考取[[湖南师范大学|湖南师范大学]]古代文学研究生。3年后获得[[硕士|硕士]]学位，并且在[[湖南师范大学|湖南师范大学]]任教。<ref name="hnsx " />

1998年，赵晓岚考取[[华东师范大学|华东师范大学]]。3年后获得[[博士|博士]]学位。后又去[[复旦大学|复旦大学]]做博士后研究。<ref name="hnsx " />

2003年，赵晓岚回到母校[[湖南师范大学|湖南师范大学]]担任教授至今。<ref name="hnsx " />

2008年和2009年，赵晓岚在[[央视|央视]]《[[百家讲坛|百家讲坛]]》节目先后主讲了《[[南唐后主|南唐后主]][[李煜|李煜]]》和《金戈铁马[[辛弃疾|辛弃疾]]》。<ref>{{cite web |url=http://news.xinmin.cn/rollnews/2009/08/07/2347557.html |title=湖南师大“美女教授”赵晓岚将再上《百家讲坛》 |accessdate= 2009年|work=|publisher= 新民网|language = zh-hans}}</ref>
