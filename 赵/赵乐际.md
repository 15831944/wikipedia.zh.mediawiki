'''赵乐际'''（{{bd|1957年|3月8日|}}），[[陕西省|陕西]][[西安市|西安]]人，生于[[青海省|青海]][[西宁市|西宁]]，为现任[[中国共产党|中国共产党]]和[[中华人民共和国|中华人民共和国]]主要[[党和国家领导人|领导人]]之一。赵乐际是中共第十九届[[中国共产党中央政治局常务委员会|中央政治局常务委员会]]委员（排名第六）。

赵乐际于[[北京大学|北京大学]]哲学系毕业，[[中共中央党校|中共中央党校]]研究生学历。1975年7月加入[[中国共产党|中国共产党]]；是中共第十六至十八届中央委员，[[中国共产党第十八届中央委员会|第十八届]][[中国共产党中央政治局|中央政治局]]委员、[[中国共产党中央书记处|中央书记处]]书记。[[中国共产党第十九届中央委员会|第十九届]][[中国共产党中央政治局常务委员会|中共中央政治局常委]]、[[中国共产党中央纪律检查委员会书记|中央纪律检查委员会书记]]。曾先后任[[中国共产党青海省委员会|中共青海省委]]书记、[[中国共产党陕西省委员会|中共陕西省委]]书记、中央书记处书记、[[中共中央组织部|中央组织部]]部长。

== 早年经历 ==
赵乐际的父母是由[[西安市|西安]]前往[[青海省|青海]]地区支边的干部。赵乐际1957年3月出生在青海，并且长期在这里生活、工作。

1974年9月，赵乐际响应党中央关于[[知识青年|知识青年]][[上山下乡运动|上山下乡]]的号召，在青海[[贵德县|贵德县]][[河东乡_(贵德县)|河东乡]]贡巴大队插队劳动。仅一年之后，1975年8月，赵乐际就有机会返回城市，在[[青海省|青海省]]商业厅办公室当收发兼通讯员。作为最后一届[[工农兵|工农兵]]大学生，赵乐际于1977年2月进入[[北京大学|北京大学]]哲学系学习，1980年1月毕业。

== 从政经历 ==
大学毕业后，赵乐际回到青海省商业厅，历任政治处宣传干部，青海省商业学校教员、教务处副主任，商业厅政治处副主任兼厅团委书记。1984年12月，担任商业厅下属的“青海省五金交电化工公司”任党委书记，后来又兼任经理职务。1986年4月，再次返回商业厅任副厅长、财贸工会主席。1991年2月，晋升省商业厅厅长。

此后，赵乐际的职务更是稳步上升，1993年2月任青海省省长助理、省财政厅厅长。1994年7月任[[青海省人民政府|青海省人民政府]]副省长，1997年3月任中共[[西宁市|西宁市]][[市委書記|市委书记]]。1997年12月任[[中国共产党青海省委员会|中共青海省委]]副书记、西宁市委书记。1999年8月任代理省长，2000年1月正式当选[[青海行政长官列表|青海省人民政府省长]]。赵乐际以42岁的年龄，成为当时中国最年轻的省长。2003年8月，晋升中共青海省委书记，成为当时最年轻的[[省委书记|省委书记]]。2004年1月14日，当选青海省人大常委会主任。

在[[青海|青海]]工作近三十年后，2007年3月25日，[[中国共产党中央委员会|中共中央]]决定赵乐际接替[[李建国|李建国]]，出任[[中国共产党陕西省委员会|中共陕西省委]]书记，2008年1月23日当选[[陕西省人大常委会|陕西省人大常委会]]主任。

在2012年11月15日召开的[[中国共产党第十八届中央委员会第一次全体会议|中共十八届一中全会]]上，55岁的赵乐际当选为[[中共中央政治局|中共中央政治局]]委员，同时当选[[中国共产党中央书记处|中共中央书记处]]书记，进而成为[[党和国家领导人|党和国家领导人]]。2012年11月19日，[[中国共产党中央委员会|中共中央]]决定赵乐际兼任[[中共中央组织部|中共中央组织部]]部长。外媒称，赵乐际和[[中国共产党中央委员会总书记|总书记]][[习近平|习近平]]关系紧密。赵乐际自2012年起主导了党、军、政和国有企业4,000个最高级别职位的人事任命，并对习近平亲信多加提拔<ref>{{Cite web|url=https://cn.wsj.com/articles/CN-BCH-20171025152455|title=解读中共新一届政治局常委名单|accessdate=2018-03-19|date=2017-10-27|publisher=华尔街日报}}</ref>。

在2017年10月25日召开的[[中国共产党第十九届中央委员会第一次全体会议|中共十九届一中全会]]上，60岁的赵乐际当选为[[中国共产党中央政治局常务委员会|中共中央政治局常委]]（排名第六）、[[中国共产党中央纪律检查委员会|中央紀律檢查委员会]]书记。

== 家庭 ==
父亲：赵喜民（1933年1月10日－1999年6月19日）,早年留学前苏联，后来支边青海。曾经任《青海日报》副总编，80年代初期回到西安，后来任职陕西人民教育出版社社长，育有4个儿子。

弟弟：[[赵乐秦|赵乐秦]]（1960年1月－），现任[[广西壮族自治区|广西壮族自治区]]人大常委会副主任，[[中共桂林市委|中共桂林市委]]书记、市人大常委会主任。
