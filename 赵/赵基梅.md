'''赵基梅'''（{{bd|1910年||1947年|12月27日}}）[[湖北|湖北]][[麻城|麻城]]人，[[中国工农红军|中国工农红军]]、[[八路军|八路军]]、[[中国人民解放军|中国人民解放军]]将领<ref name=zgjw>{{Cite web |url = http://www.81.cn/yljnt/2014-01/17/content_5738983.htm |title =赵基梅  |author =  |date = 2014-01-17 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref><ref name=cpc>{{Cite web |url = http://dangshi.people.com.cn/GB/144964/145297/8972853.html |title =赵基梅（简介）  |author =  |date =  |publisher =中国共产党新闻网  |language =  |accessdate = 2017-05-25 |quote =  }}</ref><ref name=macheng>{{Cite web |url = http://www.macheng.gov.cn/info/1030/7910.htm |title = 赵采银 |author =  |date = 2015-05-13 |publisher = 中国麻城 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
赵基梅生于一个贫农家庭，父母早逝，由其兄嫂抚养。赵基梅自幼帮家里劳动。1926年，乘马岗、顺河地区先后秘密成立农民协会及农民武装。赵基梅的大哥、二哥都参加了农民自卫队，赵基梅参加了儿童团。1927年10月6日，大哥赵基松遭[[中国国民党|中国国民党]]右派军队杀害。1928年，17岁的赵基梅报名参加赤卫队，配合[[中国工农红军|中国工农红军]]攻打八里畈。1929年，赵基梅和二哥赵基柏一起在董家畈苏维埃政府报名参加了中国工农红军。1930年，加入[[中国共产党|中国共产党]]。在鄂豫皖地区随中国工农红军转战三年。[[中国共产党第四次反围剿战争|第四次反“围剿”]]之后，随[[红四方面军|红四方面军]]主力西进陕南、川北。1933年冬，任[[红三十军|红三十军]]九十师政治部主任。1935年3月，自[[川陕苏区|川陕苏区]]开始[[长征|长征]]，两次翻越雪山、过草地，最后抵达[[陕甘宁边区|陕甘宁边区]]<ref name=zgjw/><ref name=cpc/><ref name=macheng/>。

[[抗日战争|抗日战争]]时期，1938年初，赵基梅所在部队改称[[八路军|八路军]]太南独立游击支队，赵基梅任司令员，[[涂锡道|涂锡道]]任政治委员，俗称“赵支队”或“赵涂支队”。同年4月下旬，[[晋冀鲁豫军区|晋冀鲁豫军区]]在[[太行山|太行山]]成立，赵支队对内称为晋冀鲁豫军区第五军分区，对外仍称八路军太南独立游击支队，赵基梅继续任司令员。1940年4月底，赵支队和八路军六八九团三个连、豫北大队、独立大队等部队合编成八路军第二纵队教导三旅（又称新三旅、独立旅），赵基梅任副旅长。1940年6月上旬，[[日军|日军]]对冀鲁豫边区发动“大扫荡”，赵基梅率特务营抗击进攻旅部驻地的数百名日伪军，保证旅部与机关转移。反“扫荡”结束之后，新三旅旅长[[韩先楚|韩先楚]]调离，赵基梅代理新三旅旅长。1941年7月，新三旅的七团、八团和新二旅的四团合编为八路军一一五师教导第七旅，赵基梅任政治委员。1941年10月，晋冀鲁豫军区第八军分区成立，赵基梅兼任司令员<ref name=zgjw/><ref name=cpc/><ref name=macheng/>。

1945年底，[[杨勇|杨勇]]任[[晋冀鲁豫军区|晋冀鲁豫军区]]第七纵队司令员，赵基梅任副司令员。[[第二次国共内战|第二次国共内战]]时期，1946年2月，在晋冀鲁豫野战军领导人[[刘伯承|刘伯承]]、[[邓小平|邓小平]]指挥下，赵基梅协助杨勇转战豫东和鲁西南，配合兄弟部队攻克了[[定陶县|定陶县]]、[[单县|单县]]、[[曹县|曹县]]等城镇。1947年2月，晋冀鲁豫野战军第一、第七纵队合编为第一纵队，杨勇任司令员，[[苏振华|苏振华]]任政治委员，赵基梅任副司令员，率部在豫北局部反攻，配合兄弟部队攻克了[[濮阳县|濮阳县]]、[[封丘县|封丘县]]、[[延津县|延津县]]、[[原阳县|原阳县]]、[[原武县|原武县]]、[[汤阴县|汤阴县]]。1947年8月初，在[[山西|山西]][[晋城|晋城]]地区休整的[[新四军|新四军]]第五师主力与嵩岳部队一部，改编成晋冀鲁豫野战军第十二纵队，赵基梅任司令员，[[张才千|张才千]]任政治委员。第十二纵队组建后的第三天，赵基梅、张才千率第十二纵队自晋城出发，[[挺进大别山|挺进大别山]]。1947年9月23日，在[[郑州|郑州]]以南配合[[华东野战军|华东野战军]]围歼[[国民革命军|国民革命军]]整编第十师第一一八旅。此后又在豫东南信（阳）潢（川）公路以北阻击了国民革命军，使第十纵队挺进大别山的军事压力减轻<ref name=zgjw/><ref name=cpc/><ref name=macheng/>。

第十二纵队进入大别山区后，[[刘邓大军|刘邓大军]]首长命该纵队创建[[江汉军区|江汉军区]]。1947年12月6日，第十二纵队与[[张才千|张才千]]、[[李人林|李人林]]率领的中原独立旅在[[红安县|红安县]]华家河合并成立江汉军区，赵基梅兼任司令员，[[刘建勋|刘建勋]]任政治委员。1947年12月14日，江汉军区部队分为三路跨越[[平汉铁路|平汉铁路]]进入江汉地区。此后部队不断取胜，摧毁了[[中国国民党|中国国民党]]方面的[[保甲|保甲]]统治及地方武装，占领了农村地区并成立各级人民政府。至1947年12月底，共歼敌4000多人，攻占[[桐柏县|桐柏县]]、[[枣阳县|枣阳县]]、[[泌阳县|泌阳县]]、[[唐河县|唐河县]]、[[新野县|新野县]]、[[京山县|京山县]]、[[钟祥县|钟祥县]]、[[天门县|天门县]]、[[潜江县|潜江县]]等十多个城镇，配合了主力在大别山的斗争<ref name=zgjw/><ref name=cpc/><ref name=macheng/>。

1947年12月27日，赵基梅因急性胃炎抢救无效在[[随县|随县]]（今[[随州市|随州市]]）三里岗病逝。[[中华人民共和国|中华人民共和国]]成立后，追认赵基梅为[[烈士_(中华人民共和国)|革命烈士]]<ref name=zgjw/><ref name=cpc/><ref name=macheng/>。
