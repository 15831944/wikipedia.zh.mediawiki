'''赵瑞宝'''（{{bd|1960年|3月||}}），[[山东|山东]][[兖州|兖州]]人，[[中国人民解放军少将|中国人民解放军少将]]。

曾任[[中国人民解放军第二炮兵部队|中国人民解放军第二炮兵部队]]政治部干部部部长、[[第二炮兵工程学院|第二炮兵工程学院]]政治委员<ref>{{cite web |title=第二炮兵工程学院领导撰文纪念建院50周年 |url=http://news.163.com/09/1015/18/5LMFTM8V000120GU.html |website=网易 |accessdate=2019-01-07 |date=2009-10-15}}</ref>。2010年7月，晋升少将军衔。2011年，任第二炮兵[[中国人民解放军火箭军第六十五基地|第51基地]]政治委员<ref>{{Cite web |url = http://roll.sohu.com/20120118/n332490862.shtml |title =迎新春军政座谈会在沈阳举行  |author =  |date =2012-01-18  |publisher =  搜狐|language =  |accessdate = 2019-01-07 |quote =  }}</ref>。2014年，改任第二炮兵[[中国人民解放军火箭军第六十一基地|第52基地]]政治委员<ref>{{Cite web |url = http://www.cflac.org.cn/xw/bwyc/201405/t20140506_254628.htm?isappinstalled=1 |title =中国文联文艺志愿服务团与二炮官兵一同唱响中国梦、强军梦 |author =  |date = 2014-05-06 |publisher = 中国文艺网 |language =  |accessdate = 2019-01-07 |quote =  }}</ref>。2017年，任[[中国人民解放军火箭军|火箭军]][[中国人民解放军火箭军第六十一基地|第61基地]]政治委员。2018年4月，升任[[中国人民解放军西部战区|中国人民解放军西部战区]]副政委兼政治工作部主任<ref>2018年6月12日CCTV7《军事报道》节目播送画面。</ref>。

是[[第十三届全国人大代表|第十三届全国人大代表]]<ref>{{cite web |title=代表信息 |url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=200813290894 |website=中国人大网 |accessdate=2019-01-07}}</ref>。
