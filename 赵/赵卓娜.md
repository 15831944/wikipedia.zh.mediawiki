== 生平 ==
1988年，赵卓娜出生在黑龙江省哈尔滨市，曾在中央戏剧学院读书，毕业后从事影视、主持工作，荣获过多项影视奖。她与已婚的[[韩寒|韩寒]]有情感绯闻。<ref>{{cite web |url=http://ent.ifeng.com/idolnews/mainland/detail_2012_09/03/17298904_0.shtml |title=传韩寒感情出轨 与演员赵卓娜已相恋近三年 |accessdate= 2012年9月3日|publisher=凤凰网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://hb.chinadaily.com.cn/2014/1023/94/9303.html |language= |title=韩寒再曝出轨 韩寒出轨门女主赵卓娜个人资料照片曝光 |publisher=中国日报网 |date=2014-10-23 |accessdate=2014-10-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20141023060943/http://hb.chinadaily.com.cn/2014/1023/94/9303.html |archivedate=2014-10-23 }}</ref>
