'''赵正永'''（{{bd|1951年|3月|catIdx=Z赵}}），[[安徽省|安徽]][[马鞍山市|马鞍山]]人。1968年11月参加工作，1973年11月加入[[中国共产党|中国共产党]]。[[中南大学|原中南矿冶学院]]材料系毕业，[[中共中央党校|中共中央党校]]研究生学历。[[中共第十八届中央委员|中共第十八届中央委员]]。曾任[[中国共产党陕西省委员会|中共陕西省委]]书记，[[陕西省人民代表大会常务委员会|陕西省人大常委会]]主任，[[陕西省|陕西省]]省长，[[第十二届全国人民代表大会各专门委员会|第十二届全国人民代表大会内务司法委员会]]副主任委员等职务。2019年1月15日，因涉嫌严重违纪违法，接受中共中央纪委和国家监委纪律审查和监察调查<ref>{{cite web |url = http://www.ccdi.gov.cn/scdc/zggb/zjsc/201901/t20190115_187175.html|title=陕西省委原书记赵正永涉嫌严重违纪违法接受中央纪委国家监委纪律审查和监察调查 |accessdate = 2019年1月15日 |publisher = 中央纪委国家监委网站}}</ref>。

==早年经历==
1968年11月中学毕业后，17岁的赵正永作为[[知识青年|知识青年]]被安排到[[安徽省|安徽省]][[宣城|宣城]]地区水阳公社[[插队|插队]]劳动。1970年通过工厂招工，进入[[馬鋼集團|马鞍山钢铁公司]]，在修建部机动车间当工人；后在秘书科，当秘书。1974年10月，被推荐到[[中南大学|原中南矿冶学院]]（今[[中南大学|中南大学]]）材料系金属物理专业学习。毕业后，回到[[馬鋼集團|马钢]]；历任钢铁研究所物理室技术干部，复查办办事员，钢铁研究所共青团委书记、公司团委副书记等职。

==从政之路==
1982年8月，任[[中国共产主义青年团|共青团]]马鞍山市委书记。1983年5月，任中共马鞍山市委常委、共青团马鞍山市委书记。1985年9月，任中共马鞍山市委常委、秘书长。1988年3月，任中共马鞍山市委副书记。1992年5月，任中共安徽省[[黄山市|黄山市]]委副书记。1993年4月，任中共黄山市委书记，黄山军分区第一书记、第一政委。1998年4月，任安徽省公安厅厅长、党委书记，[[中国人民武装警察部队安徽省总队|武警安徽省总队]]第一书记、第一政委。2000年5月，任[[中国共产党安徽省委员会|中共安徽省委]]政法委书记，安徽省公安厅厅长、党委书记，武警安徽省总队第一书记、第一政委<ref>{{Cite web|url=http://knews.shaanxi.gov.cn/IssuedContentAction.do?dispatch=vContNrForZfld&contentid=63628|title=省长赵正永个人简历|accessdate=2019-01-15|publisher=[[陕西省人民政府|陕西省人民政府]]|archiveurl=http://web.archive.org/web/20121002235852/http://knews.shaanxi.gov.cn/IssuedContentAction.do?dispatch=vContNrForZfld&contentid=63628|archivedate=2012-10-02|deadurl=yes}}</ref>。

2001年6月，赵正永离开家乡安徽，调任[[中国共产党陕西省委员会|中共陕西省委]]常委、省政法委书记；2005年1月起，还兼任陕西省副省长、党组副书记；2010年6月，在[[袁纯清|袁纯清]]调[[山西省|山西省]]任职后，赵正永出任中共陕西省委副书记、陕西省代理省长<ref>{{Cite web|url=http://news.hsw.cn/system/2010/06/02/050525616.shtml|title=赵正永同志任陕西省代省长|accessdate=2019-01-15|date=2016-06-02|publisher=华商报}}</ref>；2011年1月，在陕西省第十一届人大第四次会议上，正式出任省长职务<ref>{{Cite web|url=http://news.xinhuanet.com/politics/2011-01/22/c_13702769.htm|title=赵正永当选陕西省省长|accessdate=2019-01-15|date=2011-01-22|publisher=新华网|archiveurl=https://web.archive.org/web/20110124224538/http://news.xinhuanet.com/politics/2011-01/22/c_13702769.htm|archivedate=2011-01-24}}</ref>。2012年11月，在[[中国共产党第十八次全国代表大会|中共十八大]]上，首次进入中央委员会，成为[[中国共产党第十八届中央委员会委员列表|中国共产党第十八届中央委员会委员]]；次月18日，接任中共陕西省委书记，并不再担任省长职务<ref>{{Cite web|url=http://politics.people.com.cn/n/2012/1218/c1001-19937245.html|title=赵正永任陕西省委书记|accessdate=2019-01-15|date=2012-12-18|publisher=人民网}}</ref>；2013年1月起，兼任[[陕西省人民代表大会常务委员会|陕西省人大常委会]]主任<ref>{{Cite web|url=http://www.chinanews.com/gn/2013/01-31/4538691.shtml|title=赵正永当选陕西省人大主任|accessdate=2019-01-15|date=2013-01-31|publisher=中新网}}</ref>。

2016年3月底，卸任中共陕西省委书记，并于4月1日，辞去省十二届人大常委会主任职务<ref>{{cite news|url=http://news.xinhuanet.com/politics/2016-03/27/c_1118454887.htm|title=陕西省委主要负责同志职务调整|date=2016-03-27|work=新华网|accessdate=2016-03-27}}</ref>。

2016年4月，任[[第十二届全国人民代表大会|第十二届全国人民代表大会]][[全国人民代表大会监察和司法委员会|内务司法委员会]]副主任委员<ref>{{cite news|url=https://www.thepaper.cn/newsDetail_forward_1462674|title=陕西省委原书记赵正永、河南省委原书记郭庚茂赴全国人大任职|date=2016-04-28|work=澎湃新闻|accessdate=2016-04-28}}</ref>。

== 落马 ==

2018年8月初，时任中共陕西省纪委预防腐败室主任的赵正永外甥[[胡传祥|胡传祥]]被带走调查<ref>{{cite web|url=http://www.chinanews.com/sh/2018/08-06/8591360.shtml|date=2018-08-06|title=陕西省纪委原预防腐败室主任胡传祥接受审查调查|publisher=中国新闻网}}</ref>，针对赵正永的调查开始<ref>{{cite web|url=https://globalnews.qq.com/article/TWF2019011600238000|title=对话赵发琦：我为什么有勇气举报赵正永|date=2019-01-16|publisher=腾讯网}}</ref>。

2019年1月15日，因涉嫌严重违纪违法，接受[[中国共产党中央纪律检查委员会|中央纪委]][[中华人民共和国国家监察委员会|国家监委]]纪律审查和监察调查<ref>{{cite web|url=http://www.ccdi.gov.cn/toutiao/201901/t20190115_187173.html|title=陕西省委原书记赵正永涉嫌严重违纪违法  接受中央纪委国家监委纪律审查和监察调查|accessdate=2019-01-15|date=2019-01-15|publisher=中央纪委国家监委网站}}</ref>。

赵正永被查前最后一次公开露面为2018年7月3日参访长安[[香积寺|香积寺]]；当地人称，“去过香积寺，平安又无事”<ref>{{cite web|url=https://www.guancha.cn/politics/2019_01_16_487091.shtml|date=2019-01-16|publisher=观察者网|title=赵正永最后一次露脸 秦岭违建整治前参访当地寺院}}</ref>。

2019年1月，中纪委主管的《中国纪检监察杂志》发文称赵正永“当省长时很多事情自己定，不向[[赵乐际|省委书记]]汇报，当省委书记时，却经常干预政府的事”，带头违反民主集中制原则，被认为是陕西官场乱象的根源<ref>{{cite web|url=https://zgjjjc.ccdi.gov.cn/bqml/bqxx/201901/t20190127_187745.html|date=2019-01-27|publisher=中国纪检监察杂志|title=有权必有责 用权受监督——习近平总书记关于加强权力监督的重要论述揽要}}</ref>。

== 批评与争议 ==
=== 秦岭北麓违建别墅案 ===
{{Main|秦岭北麓违建别墅案}}
[[中国中央电视台|中国中央电视台]]在2019年1月9日播出的《一抓到底正风纪——秦岭违建整治始末》中采访到陕西省、西安市等部分原领导干部。报道称，2014年5月中共中央总书记[[习近平|习近平]]批示要求陕西省关注秦岭违建别墅。时任中共陕西省委书记赵正永未在省委常委会上对此批示进行传达学习，也未进行专题研究，只是简单地批示省委督察室会同中共西安市委尽快查清、向中共中央报送材料。时任省长[[娄勤俭|娄勤俭]]也只是进行了圈阅。同年7月，西安市调查组上报称违建别墅已彻底查清，共计202栋。随后，中共西安市委将202栋的数据向中共陕西省委汇报，省委未做核实便于8月直接向中央上报。同年10月，习近平再次做出批示，赵正永在省委常委会上仅提了原则性要求，要求西安市认真落实。然而，不仅202栋别墅整而未治，之后秦岭北麓仍然不断出现违规新建别墅达600余栋之多。2018年7月底开始，中共中央派出以[[中国共产党中央纪律检查委员会|中纪委]]副书记[[徐令义|徐令义]]为组长的专项整治工作组督导陕西省全面整治秦岭违建别墅，共拆除整治1194栋<ref>{{cite web|url=http://news.cctv.com/2019/01/09/ARTIxDkQvFJBQV3VT5relywf190109.shtml|title=【独家视频】秦岭违建别墅整治始末 一抓到底正风纪|publisher=央视网|date=2019-01-09}}</ref>。当晚，人民日报海外版旗下微博账号“@侠客岛”发表评论称，“在秦岭违章别墅拆除问题上，现任省委书记、时任省领导、几任西安书记市长，甚至县长都出镜做了检讨。但有一个人没有出镜，就是那位‘时任陕西省委主要领导’”<ref>{{Cite web|url=https://news.sina.com.cn/c/2019-01-17/doc-ihqhqcis6925581.shtml|title=除了赵正永 还有这些陕西高官因涉秦岭违建落马|accessdate=2019-01-17|date=2019-01-17|last=界面|work=新浪n}}</ref>。

=== 陕西千亿矿权案 ===
{{Main|陕西千亿矿权案}}
2016年11月3日，赵发琦实名举报前后两任陕西省省长[[袁纯清|袁纯清]]和赵正永通过发密函影响[[中华人民共和国最高人民法院|最高人民法院]]和[[陕西省高级人民法院|陕西省高级人民法院]]办案等手段操纵司法，干预案件执行<ref>{{Cite web|url=https://www.weibo.com/ttarticle/p/show?id=2309404270047076955067|title=实名举报原陕西省委书记赵正永|accessdate=2019-01-10|author=赵发琦|date=2016-11-03|publisher=[[新浪微博|新浪微博]]}}</ref>。据《财经》杂志披露，早在2007年3月25日，时任陕西省常务副省长的赵正永就曾作出批示，要求陕西警方对凯奇莱公司进行侦查，理由是其涉嫌虚报注册资本。赵正永此举被认为是要彻底注销凯奇莱公司，使其丧失在“陕北千亿矿权案”中的主体权从而无法推进诉讼程序。2010年8月30日，在陕西省政府党组会议上，已任陕西省代省长的赵正永直接认定凯奇莱公司与西勘院民事合同无效，并签发了相关文件，造成事实上的政府取代法院的职能，代替法院判案。2010年11月3日，赵正永在省政府专题党组会议后，再次要求公安部门对凯奇莱公司采取行动。此后陕西省公安厅成立了“查处凯奇莱涉嫌经济犯罪督办组”。同年8月19日，赵发琦被榆林市公安局抓捕并羁押，时间长达133天。其后赵发琦被取保候审，后获判无罪。其间，凯奇莱公司就撤销营业执照等向法院提起行政诉讼获胜，公司的营业执照于2013年恢复。<ref>{{cite web|url=https://finance.sina.com.cn/china/2019-01-15/doc-ihqfskcn7452142.shtml|title=赵正永被查：卷入千亿矿权案 对秦岭违建整治不力|publisher=财经杂志|date=2019-01-15}}</ref>目前尚未有官方层面的信息披露赵正永的违纪违法细节与“陕北千亿矿权案”的关联。

据《财经》杂志和《经济参考报》等媒体报道，本案与原陕西省政协常委、现任广西自治区政协常委的香港女商人[[刘娟_(商人)|刘娟]]密切相关<ref>{{cite web|url=http://finance.sina.com.cn/leadership/crz/20130730/111416285883.shtml|title=女港商刘娟坐拥上千亿元煤矿 旗下公司6年纳税35元|publisher=财经杂志|date=2013-07-30}}</ref>，其实际控制的香港益业投资（集团）有限公司涉嫌伙同央企[[中国化学工程集团公司|中国化学工程集团公司]]以3年建成240万吨甲醇MTO项目的虚假承诺获取陕西省政府配置“陕北千亿矿权案”中的特大煤矿；在获得省政府配置的煤炭资源后，中化工程集团零收益退出，香港益业公司将该煤矿以及化工项目51%股权转让给当地省属国有企业[[延长石油集团|陕西延长石油集团有限责任公司]]，虚增被收购企业净资产约3亿元，骗取巨额国有资产<ref>{{cite web|url=http://www.jjckb.cn/2013-06/21/content_451455.htm|date=2013-06-21|title=延长石油被指亿元国有资产流失:女港商联手中化工程集团以“承诺”项目获取特大煤矿|publisher=经济参考报|author=王文志、肖波}}</ref>。但[[赵发琦|赵发琦]]在接受媒体采访时明确否认刘娟为赵正永情妇的传言，称刘娟的后台另有其人<ref>{{cite web|url=https://globalnews.qq.com/article/TWF2019011600238000|title=对话赵发琦：我为什么有勇气举报赵正永|date=2019-01-16|publisher=腾讯网}}</ref>。
