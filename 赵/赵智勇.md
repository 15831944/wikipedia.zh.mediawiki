'''赵智勇'''（{{bd|1955年|4月||catIdx=Z赵}}），[[河北|河北]][[易县|易县]]人。前[[中国共产党|中国共产党]]党员（2014年被开除党籍）、副省级干部（2014年被降级为[[中华人民共和国公务员#公务员的类别和级别#“非领导职务”类|科员]]）。

==生平==
[[西南财经大学|西南财经大学]]货币银行学系专业在职研究生毕业，经济学博士。历任[[中国工商银行|中国工商银行]]管理信息部总经理、江西分行行长，[[江西省人民政府|江西省人民政府]]省长助理、副省长，中共[[九江市|九江市]]委书记等职。2006年12月获任[[中共江西省委|中共江西省委]]常委，2008年6月兼任省委秘书长<ref>{{cite web |accessdate=2012-05-30 |url = http://www.chinanews.com/tw/2012/02-22/3689750.shtml |title=江西省委常委赵智勇会见台北历史博物馆馆长 |publisher=[[中新网|中新网]]}}</ref>。2014年6月因涉嫌违纪被免职<ref>{{cite web |accessdate=2014-06-07  |url = http://district.ce.cn/newarea/sddy/201406/03/t20140603_2915627.shtml |title = 赵智勇涉嫌违纪被免职 |publisher=中国经济网}}</ref>。同年7月16日，经调查赵智勇利用职务上的便利谋取私利严重违纪被开除党籍，并取消其副省级待遇降为[[中华人民共和国公务员#公务员的类别和级别#“非领导职务”类|科员]]，收缴其违纪所得。<ref >{{cite web |url = http://news.163.com/14/0716/09/A192DJ8N00014JB5.html |language= |title = 江西省委原常委赵智勇被开除党籍 降为科员 |publisher = 中央纪委监察部网站 |date=2014-07-16 |accessdate= }}</ref>
