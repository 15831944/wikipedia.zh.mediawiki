'''赵宪庚'''（{{bd|1953年|11月7日||catIdx=}}），[[山西省|山西省]][[忻州市|忻州市]]人。凝聚态物理学家、核武器工程专家、中国工程院院士。[[中国共产党第十七届中央委员会候补委员列表|第十七届]]、[[中国共产党第十八届中央委员会候补委员列表|十八届中共中央候补委员]]，2016年10月被递补為[[中国共产党第十八届中央委员会委员列表|第十八届中央委员]]。

==生平==
1982年毕业于[[山西大学|山西大学]]物理系，1991年获[[中国工程物理研究院|中国工程物理研究院]]研究生部理学博士学位。1993年在[[意大利|意大利]][[国际理论物理中心|国际理论物理中心]]做访问学者，1995年在[[美国|美国]][[加州大学圣巴巴拉分校|加州大学圣巴巴拉分校]]理论物理所做访问学者，1995～1996年在美国[[得克萨斯大学奥斯汀分校|得克萨斯大学奥斯汀分校]]物理系做高级访问学者，1999年在[[瑞典|瑞典]][[于默奥大学|于默奥大学]]物理系做访问教授。1997年被评为博士生导师、晋升为研究员。1997-2003年任[[北京应用物理与计算数学研究所|北京应用物理与计算数学研究所]]研究员、博士生导师、副所长、所长；2003-2007年任[[中国工程物理研究院|中国工程物理研究院]]研究员、博士生导师、副院长；2002-2007年兼任[[中国核学会|中国核学会]]计算物理学会理事长。

2007年8月，任[[中国工程物理研究院|中国工程物理研究院]]院长。2011年，当选为[[中国工程院|中国工程院]]院士<ref>{{cite news|title=刘奇会见赵宪庚一行|url=http://news.163.com/11/0617/08/76O5BR8600014AED.html|accessdate=2012-09-24|newspaper=网易（转自宁波日报）}}</ref>。2015年9月，卸任[[中国工程物理研究院|中国工程物理研究院]]院长一职，转任[[中国工程院|中国工程院]]党组成员。2016年3月，媒体报道已任中国工程院副院长、党组成员<ref >{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1449114| title=中央候补委员、中国工程院党组成员赵宪庚任工程院副院长|publisher=澎湃新闻|accessdate= 2016-03-27}}</ref>。2016年10月，在[[中国共产党第十八届中央委员会第六次全体会议|中国共产党第十八届中央委员会第六次全体会议]]上，被递补為[[中国共产党第十八届中央委员会委员列表|中共第十八届中央委员]]<ref name="十八届六中全会">{{cite web |url = http://www.ccdi.gov.cn/xwtt/201610/t20161027_88691.html |title = 中国共产党第十八届中央委员会第六次全体会议公报 |accessdate = 2016年10月27日 |publisher = 中央纪委监察部网站}}</ref>。

赵宪庚是禁核试后中国[[核武器|核武器]]研究的主要学术带头人和技术负责人之一，参与组织、领导并主持完成多项大型实验和国防重大任务，负责理论方案制定、可靠性论证、重大关键技术攻关和决策，对推动在禁核试条件下发展中国核科学技术做出重要贡献。获国家科技进步奖一等奖3项，省部级一等奖3项。
