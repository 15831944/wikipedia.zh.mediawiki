'''赵普'''{{BD|1971年|4月24日|||catIdx= Zhao赵}}出生于[[安徽省|安徽省]][[池州市|池州市]][[贵池区|贵池区]]，毕业于[[中国传媒大学|中国传媒大学]]和[[北京电影学院|北京电影学院]]，为前[[中国中央电视台|中国中央电视台]]主持人。<ref name=" lion" >{{cite web |url=http://www.zhuchijie.com/show_123_z7_t1.html |title=赵普 |publisher= 主持界|accessdate=2011年 |work= |language =}}</ref>

==生平==
1971年，赵普出生于[[安徽省|安徽省]][[池州市|池州市]][[贵池区|贵池区]]。<ref name=" lion" /> 

1987年，赵普初中毕业后应征到[[中国人民解放军总参谋部|中国人民解放军总参谋部]]装甲兵部队，开始服兵役。<ref name=" lion" /> 

1990年-1995年，退役后的赵普在安徽省体育局任职员。后下岗。下岗后在服装市场上做生意，并在电视台兼职做天气预报员，期间自学播音知识和高中课程。<ref name=" lion" /> 

1995年，赵普到北京广播学院（现[[中国传媒大学|中国传媒大学]]）就读播音系干部专修班。<ref name=" lion" /> 

1997年，大学毕业后的赵普到[[北京电视台|北京电视台]]担任主播。<ref name=" lion" />

2001年，赵普到[[北京电影学院|北京电影学院]]就读制片专业本科班。<ref name=" lion" />

2006年，赵普参加[[中国中央电视台|中国中央电视台]]举办的主持人大赛《魅力新搭档》，进入全国前30强。赛后被录用到[[中国中央电视台新闻频道|央视新闻频道]]（CCTV-13）。主持《[[朝闻天下|朝闻天下]]》节目。同时攻读北京师范大学艺术系硕士学位。<ref name=" lion" />

2011年，赵普转任[[中国中央电视台综合频道|央视综合频道]]（CCTV-1）《[[晚间新闻|晚间新闻]]》。<ref name=" lion" />

2015年，11月1日晚，央视主播赵普被曝辞职，或回母校中国传媒大学任职。<ref>http://ent.china.com/star/news/11052670/20151102/20671433_all.html#page_2</ref>

{{啥时候|近期}}，赵普将会加入[[爱奇艺|爱奇艺]]，担任源自英国的电视游戏节目《{{le|谁会成为百万富翁 (英国游戏节目)|Who Wants to Be a Millionaire? (UK game show)|谁会成为百万富翁}}》的中国内地版本《谁想成为百万富翁》的主持人，并在爱奇艺网上播出。

== 微博事件 ==

2012年4月，赵普在政府正式公开老酸奶的原材料问题前，发微博称，“来自调查记者的短信：不要再吃老酸奶（固体形态）和果冻，内幕很可怕，不细说。”。此后，赵普突然停止了在《晚间新闻》节目的出镜。具体原因不明。随后，一些媒体记者曾和赵普取得过联系，但赵普均未正面回应停止出镜的原因。也没有通过任何方式向外界解释。<ref>{{cite web |url=http://news.ifeng.com/mainland/detail_2012_04/21/14048166_0.shtml |title=赵普回应被停职传言：只能说“无可奉告” |publisher= 凤凰网|accessdate=2012-04-21 |work= |language =}}</ref><ref>{{cite web |url=http://ent.163.com/12/0421/11/7VK2F39K00031H2L.html |title= 赵普曝光明胶疑被封杀 回应:一片冰心在玉壶|publisher=网易 |accessdate=2012-04-21 |work= |language =}}</ref><ref name="deer " >{{cite web |url=http://news.ifeng.com/mainland/detail_2012_04/21/14052887_0.shtml |title=赵普回应因曝料“毒胶囊”遭停职：一片冰心在玉壶 |publisher=凤凰网 |accessdate= 2012-04-12|work= |language =}}</ref><ref>{{cite web |url=http://news.ifeng.com/mainland/detail_2012_04/21/14052887_0.shtml |title=赵普用古诗回应被停职传闻：一片冰心在玉壶 |publisher=凤凰网 |accessdate= 2012-04-12|work= |language =}}</ref> 

在赵普停止出镜期间，坊间传闻其被央视革职。据传在2012年6月中下旬，[[凤凰卫视|凤凰卫视]]派人和赵普取得联系，力邀赵普加盟。又有传闻称[[中国中央电视台|中国中央电视台]]得知赵普有意转投凤凰卫视后，在其网站上大量删除赵普此前录制的有关端午节诗会的节目视频。<ref>{{cite web |url=http://culture.ifeng.com/8/detail_2012_06/26/15558802_0.shtml |title= 网曝赵普终离央视 转投凤凰卫视几成定局|accessdate= 2012年6月26日|publisher= 人民网-传媒频道 宋心蕊|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://ent.qq.com/a/20120626/000035.htm |title=传赵普转投凤凰卫视 央视被激怒删除其所录节目 |accessdate=2012年6月26日 |publisher= 腾讯网|work= |language =zh-cn}}</ref>

2012年8月13日下午4点，赵普在央视新闻频道一档报道[[伦敦奥运会|伦敦奥运会]]节目中出镜，结束了4个多月的停止出镜状态。<ref>{{cite web |url=http://ent.qq.com/a/20120813/000597.htm |title=赵普重新出镜主持央视新闻频道节目 力证未跳槽 |accessdate=2012年8月13日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

2012年12月，赵普接受《[[南方人物周刊|南方人物周刊]]》采访，首度谈及微博事件，指出并非中国中央电视台整他，而央视跟其说明用意是：“先避风头，暂缓出镜。为的是抗住来自‘上面’的压力，也为保护‘站着中枪’的赵普”。<ref>{{cite web |url=http://news.qq.com/a/20121225/001795.htm |title=赵普首谈“封杀”传闻 称被央视保护暂缓出镜  |accessdate= 2012年12月26日|publisher= 腾讯网|archiveurl=http://news.qq.com/a/20121225/001795.htm |archivedate= 2012年12月26日|work=南都网 |language=zh-cn}}</ref><ref>{{cite web |url= http://news.qq.com/a/20121226/000217.htm|title=赵普回应毒胶囊事件 称被央视保护暂缓出镜 |accessdate= 2012年12月26日|publisher= 腾讯网|archiveurl= http://news.qq.com/a/20121226/000217.htm|archivedate= 2012年12月26日|work= 成都商报|language=zh-cn}}</ref>
