'''赵宇正当防卫案'''<ref>{{Cite web |url = http://www.bjnews.com.cn/news/2019/02/20/548770.html |title = 福州 “制止施暴被拘案” 赵宇被以过失重伤罪移送检察院|author =  |date = 2019年2月20日 |publisher = 新京报|language = zh-cn |accessdate = 2019年2月20日 |quote =  }}</ref>，或称'''福州见义勇为被拘案'''、'''制止施暴被拘案'''、'''赵宇过失致人重伤案'''、'''赵宇案'''，事件发生于2018年12月26日[[福州市|福州市]]，当事人赵宇在阻止一名[[强奸未遂|强奸未遂]]的罪犯时，过程中造成对方受伤至伤残二级，后赵宇被警方以涉嫌故意伤害拘留14天，并于2019年1月10日释放。事件在2019年2月16日被福州电视台报道并开始在网络传播，进而引发关于[[见义勇为|见义勇为]]以及[[正当防卫|正当防卫]]的讨论。

== 经过 ==
2018年12月26日晚，赵宇在出租屋听到楼下有人呼救，前去了解情况，他看到一女子正被一男子掐住脖子，便上前拉开。一番拉扯中，赵宇踹到男子腹部。

2018年12月29日，赵宇因“涉嫌故意伤害”被公安机关带走，关进福州市第一看守所；关押为[[刑事拘留|刑事拘留]]性质<ref name="20190220thepaper">{{cite news|url=https://www.thepaper.cn/newsDetail_forward_3014490|title=福州小伙阻止施暴踹伤人被拘后续：被以过失致人重伤移送检方|publisher=[[澎湃新闻|澎湃新闻]]|author=王选辉|date=2019-02-20|accessdate=2019-02-21|language=zh-hans|archivedate=2019-02-20|archiveurl=https://archive.fo/HbwTy}}</ref>。

2019年1月10日，因检察机关不予批捕，赵宇在缴纳1万元人民币保证金后被释放<ref name="20190220legaldaily">{{cite news|url=http://epaper.legaldaily.com.cn/fzrb/content/20190220/Articel01009GN.htm|title=福州男子为“救人”踹伤“施暴者” 是见义勇为还是故意伤害社会期盼回应|author=吴亚东|publisher=[[法制日报|法制日报]]|date=2019-02-20|archivedate=2019-02-21|archiveurl=https://archive.fo/3b8sE}}</ref>。关于释放原因，事件相关证明书显示为 “因检察院不批准逮捕应立即释放”，经晋安公安分局决定，予以释放。计刑事拘留13天，一说14天。

同年2月16日，赵宇注册[[新浪微博|新浪微博]]账号名为“被冤枉的赵宇”进行网络维权求助。

2019年2月20日，福州市公安局晋安分局决定以[[过失致人重伤罪|过失致人重伤罪]]追究赵宇的刑事责任，晋安分局认为，赵宇过失致人重伤一案，犯罪事实清楚，证据确实充分，移送晋安区人民检察院审查起诉<ref>{{Cite web |url = https://news.sina.com.cn/s/2019-02-20/doc-ihrfqzka7649481.shtml |title = 检方回应 “小伙见义勇为反被拘” 曾作不逮捕决定|author =  |date = 2019年2月20日 |publisher = 新浪网|language = zh-cn |accessdate = 2019年2月20日 |quote =  }}</ref><ref name="20190220thepaper" />。

2019年2月21日凌晨1点，认证账号「福州公安」在[[新浪微博|新浪微博]]、[[微信|微信公众号]]发文，公开案情部分内容。福州公安以涉嫌过失致人重伤罪移交晋安区人民检察院对赵宇进行起诉，检察院经审查认为，赵宇的行为属[[正当防卫|正当防卫]]，但超过必要限度，造成了被害人李华重伤的后果；鉴于赵宇有制止不法侵害的行为，为弘扬社会正气，鼓励见义勇为，综合全案事实证据，对赵宇作出不起诉决定；而李华因涉嫌犯非法侵入住宅罪已于2月19日在公安机关指定的地点监视居住，公安机关将视其病况采取相应法律措施。<ref>{{cite web|url=https://www.weibo.com/1925694254/HhB8Izi0T|title=2019年2月21日1时24分，福州公安之新浪微博|website=[[新浪微博|新浪微博]]|date=2019-02-21|language=zh-hans|archivedate=2019-02-21|archiveurl=https://archive.fo/lG43S}}</ref><ref>{{cite web|url=https://mp.weixin.qq.com/s/GEPm8hLiVtl7h_tehTDSZQ|title=案情通报|date=2019-02-21|accessdate=2019-02-21|language=zh-hans|archiveurl=https://archive.fo/gvbVE|archivedate=2019-02-21|website=[[微信|微信公众平台]]}}</ref><ref>{{Cite web |url = https://news.ifeng.com/c/7kSbJgXujjM |title = 福州检方回应 “见义勇为被拘” 案：不予起诉|author =  |date = 2019年2月21日 |publisher = 凤凰网|language = zh-cn |accessdate = 2019年2月21日 |quote =  }}</ref>

3月1日，在[[中华人民共和国最高人民检察院|中华人民共和国最高人民检察院]]指导下，福建省人民检察院指令福州市人民检察院对赵宇案进行了审查。福州市人民检察院经审查认为，原不起诉决定存在适用法律错误，遂指令晋安区人民检察院撤销原不起诉决定，以正当防卫对赵宇作出无罪的不起诉决定<ref name="new">[http://www.xinhuanet.com/legal/2019-03/01/c_1124182940.htm 最高人民检察院就“赵宇正当防卫案”作出回应]</ref>。

3月19日，赵宇收到由福州市公安局晋安分局颁发的“见义勇为确认证书”<ref>[http://www.bjnews.com.cn/news/2019/03/19/557628.html 赵宇收到见义勇为确认证书]</ref>。

== 当事人 ==
===受害者===
邹某（化名）27岁，湖南人。在事发时，声称遭到李华的殴打和试图[[性侵|性侵]]。事发后，邹某被鉴定为轻微伤，脸部多个部位青肿；邹某感谢赵宇的见义勇为行为，并且使用手机录像描述当晚经过愿意为赵宇作证，之后返回老家。

===赵宇===
赵宇，22岁，黑龙江哈尔滨人，保安，现在福州生活。当天深夜他正在家中陪妻子待产，忽然听到一名女邻居呼救便上前制止。事后三天，赵宇被福州市公安局晋安分局以涉嫌[[故意伤害罪|故意伤害罪]][[刑事拘留|刑事拘留]]。

===李华===
李华，50岁，四川人，包工头。被赵宇踢伤出院后内脏损伤鉴定为重伤二级，被踢后曾说过要找人弄死邹某和赵宇。在接受媒体采访时，李华声称自己“站在门口玩”“也不知道为什么赵宇要踢他”；而之前的媒体电话采访中，李华声称自己在打麻将，不方便多说<ref name="new"/>。
