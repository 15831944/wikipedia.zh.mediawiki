'''赵海宸'''（{{bd|1987年|10月13日|catIdx=zhao赵海宸}}），[[黑龙江|黑龙江]][[密山|密山]]人,[[汉族|汉族]]，于[[黑龙江|黑龙江]][[密山|密山]]出生，[[乐视体育|乐视体育]]《[[新吐槽运动|新吐槽运动]]》栏目主持人。2010年毕业于[[中国传媒大学南广学院|中国传媒大学南广学院]]本科。

== 生平 ==
1987年，赵海宸出生在[[黑龙江省|黑龙江省]]中俄边陲小镇，父母都是[[农垦|农垦]]职工。从小赵海宸喜欢模仿广告里的配音，对电视节目很感兴趣。10岁时因为父母工作变动，赵海宸来到[[秦皇岛|秦皇岛]]生活，在少年宫学习朗诵和表演。高三时参加[[艺考|艺考]]，专业排名全国第九。在大学期间对体育，尤其是足球解说产生浓厚兴趣。2010年毕业后到[[优酷|优酷]]体育频道担任编辑和节目策划，配音，并创建品牌节目吐槽运动。2012年在山东体育频道参加解说员比赛，获得最佳评论奖。<ref>[http://www.iqilu.com/html/dianshibao/cehua/2012/0611/1245681.shtml 最佳评论：赵海宸]</ref>同年加盟[[CSPN|CSPN]]，担任英超，德甲解说员。2013年起在[[乐视体育|乐视体育]]<ref>[http://www.lesports.com/news/869838019.html 乐视体育赵海宸专栏：王大雷们应当如何谢场]</ref>担任解说员，主持人。

==代表作==
大话体育<ref>[http://baike.so.com/doc/6907330-7129174.html 大话体育]</ref>,吐槽运动，新吐槽运动

== 节目 ==
*2010年在[[优酷|优酷]]创建大话体育节目。
*2013年更名为吐槽运动<ref>[http://www.360kan.com/va/Zs5kbqNw7pk3Dz.html 吐槽运动]</ref>。
*2017年更名为新吐槽运动

== 工作经历 ==
*2010年毕业后到[[优酷|优酷]]网体育频道担任编辑和节目策划，配音，并创建品牌节目吐槽运动（前身是大话体育）。
*2012年加盟[[CSPN|CSPN]]，担任英超，德甲解说员。
*2013年起在乐视体育担任解说员，主持人。
== 奖项 ==
*2012年在山东体育频道参加解说员比赛，获得最佳评论奖。<ref>[http://www.iqilu.com/html/dianshibao/cehua/2012/0611/1245681.shtml 最佳评论：赵海宸]</ref>
