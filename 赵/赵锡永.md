'''赵锡永'''{{BD|1955年||||catIdx= Z赵}}，[[辽宁省|辽宁省]][[沈阳市|沈阳市]][[铁西区|铁西区]]人，籍贯辽宁[[盖州|盖州]]，[[中国大陆|中国大陆]]人士，因自称“[[中华人民共和国国务院|国务院]]官员”陆续在[[湖南|湖南]][[湘潭|湘潭]]、[[娄底|娄底]]、[[云南|云南]][[昆明|昆明]]、[[玉溪|玉溪]]等地巡查行骗，而湘滇两省从副省级直至县级官员均被其骗，2013年3月，中华人民共和国国务院研究室下达通知，通知中说国务院研究室并没有此人，随后开始实施对赵锡永的逮捕计划。<ref name=" fish">{{cite web|url=http://news.cn.yahoo.com/ypen/20130316/1660267.html|title=三年假“钦差”：从湖南骗到云南|accessdate=2013年3月18日|publisher=雅虎网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20130328190539/http://news.cn.yahoo.com/ypen/20130316/1660267.html|archivedate=2013年3月28日}}</ref><ref name=" bull" >{{cite web |url=http://news.ifeng.com/gundong/detail_2013_03/15/23128875_1.shtml |title=赵锡永何许人也？ |accessdate=2013年3月18日 |publisher=凤凰网 |work= |language=zh-cn}}</ref>3月22日，赵锡永在辽宁省被捕。

== 生平 ==
1955年，赵锡永出生在[[辽宁省|辽宁省]][[沈阳市|沈阳市]][[铁西区|铁西区]]，该地是当时全国最大的工人聚集区。<ref name=" tang" >{{cite web |url=http://news.ifeng.com/shendu/nfzm/detail_2013_03/21/23361312_0.shtml |title=“司长”赵锡永：低调、有学问、有路子、还不捞钱 |accessdate=2013年3月21日 |publisher=凤凰网 |work= |language=zh-cn}}</ref>1990年代，赵锡永在隶属于金杯汽车股份有限公司的沈阳弹簧厂任职该厂的负责人，金杯汽车股份有限公司的董事长名叫“[[赵希友|赵希友]]”；而他的妻子是沈阳市铸铁厂的工人。<ref name=" tang" />1998年，赵锡永注册建立沈阳弓字弹簧有限公司、一汽金杯沈阳弹簧供销公司。<ref name=" tang" />1999年，赵锡永注册建立一汽金杯悬架弹簧有限公司和一汽金杯沈阳轿车有限公司。<ref name=" tang" />

2004年，49岁的赵锡永参加了中华人民共和国国务院的一次招聘考试，但是没有考上，由于专业水平不错，他感到自己怀才不遇。<ref name="hcbu " >{{cite web |url=http://news.gmw.cn/newspaper/2013-04/06/content_1205204.htm |title= 被误认为国务院研究员赵锡永“就坡下驴”主动冒充|accessdate=2013-04-06　 |publisher=光明网 |work= |language=zh-cn}}</ref>2008年，辽宁省的干部在广州市会见到辽宁省铁岭市去投资的企业家，赵锡永化名香港宏基万国汽车有限公司的代表人“赵希涌”，去铁岭市考察。<ref name=" tang" />2010年，赵锡永建立北京宏基万国汽车有限公司，开始到湖南省娄底市去搞汽车生产。<ref name=" tang" />2010年3月11日，“国务院发展研究中心研究员”赵锡永在湖南省娄底市和当地干部出席了“湖南金华车辆公司”与“北京宏基万国汽车有限公司”的签约仪式。<ref name=" tang" />

2010年，赵锡永假扮国务院发展研究中心研究员、国务院政策研究室司长陆续在湖南湘潭、娄底、云南昆明、玉溪等地巡查，[[湖南省人民政府|湖南省副省长]][[甘霖|甘霖]]、娄底市委常委、市经开区党委书记[[谢志雄|谢志雄]]、娄底经开区党委副书记、管委会主任[[罗孝贵|罗孝贵]]、昆明市市委书记[[张田欣|张田欣]]等均遭到其骗。<ref name=" fish" /><ref name=" bull" /><ref>{{cite web |url=http://news.21cn.com/domestic/yaowen/2013/03/15/14793452.shtml |title=资料：湖南副省长甘霖与赵锡永一同出席活动 |accessdate=2013年3月18日 |publisher=21CN |work= |language=zh-cn |archive-url=https://web.archive.org/web/20130318031533/http://news.21cn.com/domestic/yaowen/2013/03/15/14793452.shtml |archive-date=2013年3月18日 |dead-url=yes }}</ref><ref>{{cite web |url=http://news.sohu.com/20130314/n368877097.shtml |title= 资料：昆明市委书记与赵锡永等出席昆明年会|accessdate= 2013年3月14日|publisher=搜狐网 |work= |language=zh-cn}}</ref>

2012年，赵锡永被昆明云内动力股份有限公司聘用为“企业发展顾问”，被云南当地干部称为“北京赵司长”。<ref name=" tang" />不过，赵锡永曾在[[同济大学|同济大学]]的一次演讲中承认自己既不是所谓的“博士”也并非“国务院司级干部”。<ref name=" ma" >{{cite web |url=http://roll.sohu.com/20130326/n370310871.shtml |title=沈阳人赵锡永冒充司长“不收钱 有路子 且低调 |accessdate=2013年3月26日 |publisher=搜狐网 |work= |language=zh-cn}}</ref>

2013年3月8日，国务院研究室发文称该单位没有赵锡永这个人。<ref name=" tang" />得知其身份为虚假之后，娄底市外宣办主任王卫光称赵锡永对汽车行业确实颇有研究，“他对我们开发区的汽车项目确确实实起到了帮助和促进作用。”。<ref name=" tang" />

2013年3月22日，赵锡永在辽宁省被逮捕。<ref name=" opp" />被关押在昆明市看守所。<ref name="wang2">http://blog.sina.com.cn/s/blog_6a2e26690101gvoe.html</ref>4月3日，湖南同升律师事务所主任[[成戊平|成戊平]]接到赵锡永妻子的委托，成为他的辩护律师。<ref>{{cite web |url=http://news.ifeng.com/society/1/detail_2013_04/03/23836079_0.shtml?_114sobiaoqian |title=赵锡永代理律师：他没有骗财行为善意 望宽大处理 |accessdate= 2013年4月3日|publisher= 凤凰网|work= |language=zh-cn}}</ref>云南法闻律师事务所律师[[王汉政|王汉政]]接替湖南的律师成为其辩护律师。<ref name="wang">http://blog.sina.com.cn/s/blog_6a2e26690102v5k6.html</ref>9月10日左右，被换押到昆明市官渡区看守所。<ref name="wang2" />赵锡永最终认罪，被判处缓刑。<ref name="wang" />过完2014年春节，赵锡永就回到了沈阳。<ref name="wang" />

== 家庭 ==
赵锡永已婚，他妻子关岩称：“我完全认可他，我们家里不差钱，他也从未给家里带来什么好处。”，“（其行为）实现他做事情的理想。他想做些实事，有着多年企业管理经验，看到有的企业没有发展就着急，他不是为了给自己谋利，而是真心帮助企业发展。”<ref name=" ma" />
