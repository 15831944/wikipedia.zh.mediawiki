《'''飯沼。'''》（{{lang-ja|めしぬま。}}）是[[日本漫畫家|日本漫畫家]][[あみだむく|あみだむく]]所創作的[[日本漫畫|日本漫畫]]作品。於《[[月刊Comic_Zenon|月刊Comic Zenon]]》（[[Coamix|North Stars Pictures]]）2016年3月號開始進行連載。

pixiv Comic 2017年美食家排行第1名。2018年網路動畫化。

== 登場人物 ==
;飯沼
: 配音 - [[羽多野涉|羽多野涉]]（Anime Beans）、[[花江夏樹|花江夏樹]]（聲優GATARI）<ref>{{Twitter status2|mangahot_jp|870519605030039552|accessdate=2018-06-10}}</ref>
: 本作的主人公。戴著眼鏡、體型矮小、身材瘦弱且陰沈的年輕上班族。<ref>{{Cite web|url=https://natalie.mu/comic/news/282746|title=恍惚の表情で食事するリーマン描く「めしぬま。」アニメ化、アプリで配信|publisher=Comic Natalie|accessdate=2018-06-10|date=2018-05-19
}}</ref>

: 在公司內是個普通上班族，從外表看不出來是個大胃王。沒有討厭的食物，最喜歡肉類。

; 笹原幸惠
: 飯沼的同事。和他一樣是個不起眼的上班族。

; 花山柊二
: 從國中時就一直是飯沼的前輩。和公司的同事都相處的很好。

; 花山社長
: 常常邀飯沼請客的飯友。平時在總公司上班。

; 川田龍馬
: 前陣子調至飯沼所在部門的後輩。個性耿直且認真。

; 飯沼夏花
: 飯沼的妹妹。大學生。和哥哥一樣都是貪吃鬼。

; 山本秋乃
: 飯沼的表妹。大學生。喜歡作料理。

; 竹谷薰
: 飯沼的同事。為公司裡的前輩，個性看似嚴肅死板，其實是個口是心非的人。
