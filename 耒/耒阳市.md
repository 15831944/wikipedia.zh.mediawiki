'''耒阳市'''（英语：'''Leiyang'''；[[汉语拼音|汉语拼音]]：''lěiyáng''；[[注音符號|注音]]：ㄌㄟˇ ㄧㄤˊ），别称'''纸都'''，是[[中华人民共和国|中华人民共和国]][[湖南省|湖南省]][[衡阳市|衡阳市]]下辖[[县级市|县级市]]<ref name=sqjj>{{Cite web |url = http://www.leiyang.gov.cn/detail.ashx?id=26623 |title = 耒阳市情简介 |publisher = 耒阳党政门户网站}}</ref>，因古代[[神农氏|神农氏]]在此地发明创造[[耒耜|耒耜]]（古代农耕工具）而得名，亦有耒水贯穿其中。

耒阳也是中国[[四大发明|四大发明]]之一[[造纸术|造纸术]]的发明者[[蔡伦|蔡伦]]的故乡<ref>{{cite journal|title=耒阳:开拓挖潜,构建土地防护体系|author=刘峥嵘|url=http://kns.cnki.net/KCMS/detail/detail.aspx?dbcode=CJFQ&dbname=CJFD2014&filename=DCSC201401022&uid=WEEvREcwSlJHSldRa1FhdkJkVWI2VUdyV0orbUdIS25KWi9lRkV5Q2ZEND0=$9A4hF_YAuvQ5obgVAqNKPCYcEjKensW4ggI8Fm4gTkoUKaID8j8gFw!!&v=MDU3MzRxVHJXTTFGckNVUkxLZlp1UnRGeXpoVUx6TklTN1liYkc0SDlYTXJvOUhab1I4ZVgxTHV4WVM3RGgxVDM=|date=2014|journal=中国地产市场|accessdate=2018-01-22|issue=1|doi=|others=|year=|volume=|page=54-55|pmid=}}</ref>，中国地名文化遗产--[[千年古县|千年古县]]<ref>{{Cite web |url =http://www.leiyang.gov.cn/detail.ashx?id=375|title = 耒阳市成功申报为全国“千年古县” |publisher = 耒阳党政门户网站}}</ref>，中国油茶之乡。位于湖南省东南部，[[耒水|耒水]]中游。东北邻安仁县；东南、南连永兴县；西南接桂阳县；西滨舂陵水，与常宁县隔河相望；北界衡南县。东经112°38'-113°13'，北纬26°08'-26°43'。南北长62公里，东西宽58公里，成不规模棱形，总面积2656平方公里。总人口143万，常住人口116.2万，民族有汉、苗、侗族等，汉族占总人口99.99%<ref name=rkmz>{{Cite web |url = http://www.leiyang.gov.cn/detail.ashx?id=647 |title = 人口民族 |publisher = 耒阳党政门户网站}}</ref>。辖24个乡镇、6个街道办事处、377个行政村(居)，是全省三座县级中等城市之一，也是全省城区面积最大、城市人口最多的县级城市。

2015年9月，耒阳市成为湖南国土资源省直管县经济体制改革试点县（市）<ref>{{Cite web|url=http://www.hunan.gov.cn/hnyw/__deleted_2017.12.13_10.32.41__zwdt/201506/t20150606_4753553.html|title=13县（市）启动省直管县改革试点|publisher=湖南省人民政府门户网站|accessdate=2018-02-26|archiveurl=https://web.archive.org/web/20180226211823/http://www.hunan.gov.cn/hnyw/__deleted_2017.12.13_10.32.41__zwdt/201506/t20150606_4753553.html|archivedate=2018-02-26|deadurl=yes}}</ref>。

==历史==
耒阳建置很早，在[[夏朝|夏]][[商朝|商]]属[[荆州_(古代)|荆州]]，战国时属[[楚国|楚]]。[[秦始皇|秦始皇]]二十六年（前221年）置耒县，因[[耒水|耒水]]而命名，隶[[长沙郡|长沙郡]]。

* [[西汉|西汉]]高祖五年（前202年），以其治位耒水之北，更名耒阳县，隶属[[桂阳郡|桂阳郡]]。
* 王莽[[新朝|新朝]]开凤元年（14年），改耒阳县为南平亭，隶南平郡（桂阳郡改）。
* [[东汉|东汉]]建武年间（25～26年），废南平亭，复耒阳为县，隶桂阳郡，属荆州刺史部。东汉献帝建安十三年（208年），“[[刘豫州|刘豫州]]”据荆州，取桂阳，耒隶之。蜀汉（222年）隶桂阳郡，属荆州刺史。章武二年（222年）八月荆襄属吴，耒阳仍隶桂阳郡。吴孙亮（252～258年）析县地为新宁、新平、梨阳、耒阳四县，耒阳属桂阳郡。
* [[西晋|西晋]]（256～316年）耒阳仍隶桂阳郡，初属湘州，复属荆州。改梨阳县为利阳县，仍属湘东郡。东晋孝武帝太元二十年（395年），撤利阳并入耒阳，隶桂阳郡，属江洲。
* [[南北朝|南北朝]]时，南朝宋、齐至梁天正元年（420～551年），耒阳县隶桂阳郡，属湘州。梁元帝时（552～554年），迁耒阳县治于列敖山口，改隶湘东郡，上属湘洲。陈朝未变。
* [[隋文帝|隋文帝]]开皇九年（589年）平陈，耒阳县更为涞阴县，以其治位于耒阳水之阴也（水之东为阴），置衡洲，县属之。
* 唐武德四年（621年），复名耒阳县，治所迁回汉晋故治，隶衡州。贞观元年（627年）起，上属江南道。昭宗乾宁三年（896年）马殷据潭州，称楚王，衡州俱系统属，耒属焉，历梁、唐、晋、汉，凡五十六年，皆马殷地。
* [[南唐|南唐]]保大九年（951年）李璟时，刘仁瞻取岳州，命边镐自袁州取长沙而楚天，十年（952年）武平节度使周行逢取潭州，子保叔嗣衡洲刺史，而耒阳属南唐，名仍旧。
* [[宋朝|宋朝]]，改名为来阳县，隶衡州衡阳郡，属荆湖南路安抚司。
* [[元世祖|元世祖]]至元年十九年（1282年），因耒阳民众繁富，升为州，直隶[[湖广等处行中书省|湖广行省]][[湖南道宣慰司|湖南道宣尉司]]。
* [[明朝|明]][[洪武|洪武]]三年（1370年）三月，耒阳因遭兵灾，人口骤减，复降为县，隶衡州府，属[[湖廣等處承宣布政使司|湖广政使司]]。
* 清朝仍为耒阳县，隶衡州府，属衡东郴桂道（初为衡永郴道）。
* 民国三年（1914年），改衡永郴桂道为[[衡陽道|衡阳道]]，耒阳隶衡阳道，属湖南省。
* 民国十一年（1922年），废道存省，耒阳县隶湖南省。
* 民国二十六年（1937年）12月，湖南省分置9个[[行政督察區|行政督察区]]。耒阳县隶湖南省第五行政督察区。
* 民国二十九年（1940年）4月，将全省改划为十个行政督察区，耒阳县隶湖南省第二行政督察区。
* 民国三十三年（1944年）7月3日，日军侵占耒阳城，县政府迁上堡街（今属黄市镇），后迁严塘铺（今属大义乡）。
* 民国三十四年（1945年）8月15日，日本宣布无条件投降，9月15日境内日军缴械，耒阳光复，县政府迁回县城。
* 1949年10月7日，耒阳解放，12月成立县人民政府，隶衡阳专区。
* 1952年11月，衡阳、郴县、零陵三专署合并成立湘南行署（行署驻衡阳），耒阳隶湘南行政区。
* 1954年7月，撤销湘南行署，改设衡阳、郴县两个专署，耒阳隶郴县专区（1960年改名为郴州专区，1979年更名为郴州地区）。
* 1983年7月1日，回归衡阳市。
* 1986年11月，经国务院批准，耒阳县升格为县级耒阳市，隶属衡阳市。以原城关镇为市区。
* 2006年3月，经衡阳市人民政府批准，设置五里牌街道办事处，4月21日正式挂牌成立。办事处总面积29平方公里，人口5.3万人。
* 2010年8月，成立三顺街道办事处，辖1个居委会、6个建制村，总面积26.56平方公里，总人口1.5万人，办事处机关驻三顺村。{{NoteTag|然而，在官方行政区划资料<ref name=czqh>{{Cite web |url = http://www.leiyang.gov.cn/detail.ashx?id=645 |title = 行政区划 |publisher = 耒阳党政门户网站}}</ref>中，三顺并未作为独立的街道列出。}}
* 2015年9月，成为[[湖南省国土资源厅|湖南国土资源]]省直管县经济体制改革试点县（市）。<ref name="lsyg">{{Cite web |url = http://www.leiyang.gov.cn/detail.ashx?id=646 |title = 历史沿革 |publisher = 耒阳党政门户网站}}</ref>

==语言==
耒阳市辖区内的方言主要是[[耒阳话|耒阳话]]，其内部又有东乡口音，西乡口音以及北乡口音。以前的出版物多将耒阳话划为[[客家话|客家话]]区，现在多数学者赞成把耒阳话视作[[赣方言|赣方言]]，但是大多数耒阳人对[[湘方言|湘方言]]有較強认同感。有待学者进一步考证。{{fact}}

==地理==
耒阳地处[[南岭|南岭]]余脉，境内为低山丘陵地带，市境东南多山地，西部则较平缓。属于[[亚热带季风气候|亚热带季风气候]]，既具有阳光丰富的大陆性季风气候特点，又有雨量充沛、空气湿润的海洋性气候特征。全市年平均气温18℃，降水量1337毫米。<ref>{{Cite web|url=http://www.leiyang.gov.cn/zjlyDetail.ashx?id=648|title=气候特征|accessdate=2018-07-11|author=|date=|work=www.leiyang.gov.cn|publisher=耒阳党政门户网}}</ref>

==经济==

==休闲与娱乐==
*烈士陵园，是耒阳自发形成的休闲中心。里面树木繁盛（但在[[2008年中国雪灾|2008年中国雪灾]]中破坏较为严重），早晨市民过来锻炼身体，傍晚饭后市民在这散步，晚上市民则在歌厅，溜冰场等处娱乐。在这里休闲娱乐的中老年人居多。
*五一广场，90年代休闲娱乐的地方较少，市民都在这健身娱乐。特别是晚上七点到九点钟这个时段，人非常多；修建发明家广场和神龙广场后，市民则就近选择地点。
*五一电影院，位于五一路中心，一直是耒阳最为繁华的地段。
*杜甫公园，位于耒水中的东洲上，杜甫在此醉酒身卒，死后葬于此（后迁至耒阳一中）。
*发明家广场和神龙广场，耒阳最大的广场。在这里年青人溜冰、玩滑板；老人们做健身操、谈天。
*蔡伦公园，原大舞台地点，也是市民休闲娱乐地点之一。
*沿河路，无数的小吃、夜宵摊点、歌厅。
近几年，耒阳文化娱乐业年产值近5000万元，利润达3400余万元，上缴税费1600多万元。{{fact}}
