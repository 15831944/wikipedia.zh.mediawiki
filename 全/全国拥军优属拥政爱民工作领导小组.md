'''全国拥军优属拥政爱民工作领导小组'''（简称“全国双拥工作领导小组”），是[[中华人民共和国国务院|中华人民共和国国务院]]成立的[[国务院议事协调机构|国务院议事协调机构]]，领导全国[[拥军优属拥政爱民|拥军优属拥政爱民]]工作。

== 沿革 ==
1991年6月24日，《国务院、中央军委关于成立全国拥军优属拥政爱民工作领导小组的通知》（国发〔1991〕34号）发出，国务院、中央军委决定成立'''全国拥军优属拥政爱民工作领导小组'''（简称“全国双拥工作领导小组”）<ref name="chengli">{{cite journal |url = http://www.gov.cn/gongbao/shuju/1991/gwyb199127.pdf |title = 《国务院、中央军委关于成立全国拥军优属拥政爱民工作领导小组的通知》 |author =  |date =  |journal = 《[[中华人民共和国国务院公报|中华人民共和国国务院公报]]》 |year = 1991年 |number = 第27号 |language =  |accessdate =  |quote =  }}</ref>。

== 职责 ==
1991年《国务院、中央军委关于成立全国拥军优属拥政爱民工作领导小组的通知》（国发〔1991〕34号）规定：“全国双拥工作领导小组在国务院、中央军委领导下工作。其主要职责是：统一指导和协调全国的拥军优属、拥政爱民工作，研究解决双拥工作中涉及全国性的问题，制定有关政策、法规；协调处理军地关系中一些重大问题；组织交流情况，推广先进经验，表彰先进典型；承办国务院、中央军委交办的有关事项。”<ref name="chengli" />

==办事机构==
1991年《国务院、中央军委关于成立全国拥军优属拥政爱民工作领导小组的通知》（国发〔1991〕34号）规定：“全国双拥工作领导小组下设办公室，作为全国双拥工作领导小组的办事机构，负责承办日常工作。”<ref name=chengli/>

2003年《国务院办公厅中央军委办公厅关于调整全国拥军优属拥政爱民工作领导小组组成人员的通知》<ref name=ellsan/>、2008年《国务院办公厅中央军委办公厅关于调整全国拥军优属拥政爱民工作领导小组组成人员的通知》均规定：“全国拥军优属拥政爱民工作领导小组的具体工作由民政部、总政治部承担。”<ref name=ellb/>

'''全国拥军优属拥政爱民工作领导小组办公室'''（简称“全国双拥办”）设在民政部，由民政部、总政治部抽组人员[[合署办公|合署办公]]，负责领导小组的日常工作。下设秘书组和政策研究组<ref>{{Cite web |url =http://military.people.com.cn/GB/1076/52984/6570780.html  |title =首次全国双拥模范城(县)和双拥模范单位评选公示  |author =  |date = 2007-11-24 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref><ref name=jun>{{Cite web |url = http://www.81.cn/zjsymfc/2015-07/31/content_6608946.htm |title = 全国双拥工作领导小组办公室介绍 |author =  |date = 2015-07-31 |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref>。主要职责是：
#负责办理全国双拥工作领导小组全体会议和组长办公会决定的事项。
#承办全国双拥工作领导小组召开的各类会议和活动。
#负责与领导小组成员单位的联系、协调工作，承办向领导小组报告工作事宜。
#负责与各省、自治区、直辖市双拥办联系，承办向领导小组报告工作事宜。
#提出研究制定双拥工作政策、法规的建议;研究协调军政军民关系中一些重要问题。
#开展调查研究，总结工作情况，推广典型经验；组织协调全国性的双拥宣传。
#指导有关部门和地方各级双拥办开展双拥业务工作。
#负责起草、呈报、收发、管理文电工作。
#编印双拥工作刊物和资料，维护双拥工作网站。
#承办全国双拥工作领导小组交办的其他事项<ref name=jun/>。
