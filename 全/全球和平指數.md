'''全球和平指數'''（{{lang-en|'''Global Peace Index'''}}，亦稱'''和平指數'''）是一套用作測量指定國家或地區的[[和平|和平]]程度的指標，該[[指數|指數]]由名為英國[[经济学人信息社|经济学人信息社]]（{{lang|en|Economist Intelligence Unit}}）的專家小組所維持和公布，而該專家小組的成員主要來自各[[和平學|和平學]]系、[[智庫|智庫]]以及澳洲[[悉尼大學|悉尼大學]]和平及衝突學中心（{{lang|en|the Centre for Peace and Conflict Studies}}）所組成。指數引用[[聯合國|聯合國]]和其他[[國際組織|國際組織]]的数据，包括國家[[軍費|軍費]]、國家因組織性[[衝突|衝突]][[死亡|死亡]]人數、聯合國派遣人員數目等，對全球121個[[國家|國家]]及[[地區|地區]]的和平程度及生活穩定程度作出評分以及排名。首個排名於2007年5月30日公布。<ref>{{cite web |url = http://www.visionofhumanity.com/artman/uploads/1/EIU_Report.pdf |title = Global Peace Index - Methodolgy, Results & Findings |accessdate = July 1, 2007 |language = en }}</ref>2010年排名[[紐西蘭|紐西蘭]]連續兩年蟬聯第一。<ref>{{cite web |url = http://www.visionofhumanity.org/wp-content/uploads/2010/06/2010-GPI-Results-and-Methodology-Report.pdf |title = Global Peace Index - 2010 Methodolgy, Results & Findings |accessdate = June 9, 2010 |language = en }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>{{efn|[[香港|香港]]在2007年及2008年排名23，2009年之後不再被列入排名。}}

== 目的及目標 ==
據全球和平指數主席Clyde McConaghy所指，全球和平指數的[[目標|目標]]是摒除以往單以戰爭作為粗陋的[[指標|指標]]，有[[系統|系統]]地量度和平的結構，而指數為和平程度提供[[量化|量化]]的指標，可作跨時間及地域的比較，並且期望指數能引發及影響國際領袖，使他們能就和平問題作出反應。<ref>

First Global Peace Index Ranks 121 Countries  |accessdate=July 1, 2007	|language=en }}
</ref>
