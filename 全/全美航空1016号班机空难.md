== 航班 ==

=== 历史记录的航班 ===
<font style="vertical-align: inherit;"><font style="vertical-align: inherit;">1994年7月2日，一架全美航空的[[麥克唐納-道格拉斯公司|麥克唐納-道格拉斯公司]]DC-9客機（註冊號N954VJ）於[[美國東部時間|美國東部時間]]下午6時15分從哥倫比亞大都會機場起飛，飛行約35分鐘後抵達[[夏洛特/道格拉斯國際機場|夏洛特/道格拉斯國際機場]]，機上載有2名[[機師|機師]]，3名[[機艙服務員|機艙服務員]]及52名乘客（包括兩名嬰兒）。</font></font>

<font style="vertical-align: inherit;"><font style="vertical-align: inherit;">航班飛行途中一直沒有遇上惡劣天氣直至到達</font></font><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">[[夏洛特_(北卡羅來納州)|夏洛特]]上空</font></font><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，在機場附近上空出現幾次強烈的雷暴</font></font><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。</font></font><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">下午6時38分、航班1016被告知使用</font></font>[[仪表着陆系统|<font style="vertical-align: inherit;"><font style="vertical-align: inherit;">儀表著陸系統</font></font>]]<font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> (ILS)降落跑道18(現在18C)。</font></font> 是次進埸由副機長負責<font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，航班在接近跑道尉遇上大雨。</font></font><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">航班被機場控制塔指示降落跑道18R，下午6時39分、塔控制批准1016號航班降落。</font></font><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">機長要求控制塔提供上一架降落客機時的天氣狀況，一架</font></font>[[福克100|<font style="vertical-align: inherit;"><font style="vertical-align: inherit;">福克100</font></font>]]<font style="vertical-align: inherit;"><font style="vertical-align: inherit;">剛剛降落在18R跑道。</font></font> <font style="vertical-align: inherit;"><font style="vertical-align: inherit;">控制塔告知航班1016，福克客機的機師表示"一帆風順。" </font></font> <font style="vertical-align: inherit;"><font style="vertical-align: inherit;">在墜機後的訪問中，多名乘客和乘務員告知國家運輸安全委員會，該航班沒有任何異常直至飛機最後進場時進入了一個暴風雨中。</font></font>

下午6時40分、塔控制发出一个[[风切变|风切变]]警告，但1016號航班卻在不同頻道而未能接到該警告. 大约一分钟后，航班1016正在進行降落，但队长意识到客機正處於一個严重的困境中並试图中止副機長着陆表示"轉右，重新降落"。未己，機長通过无线电通知[[航空交通管制|控制塔]]表示"全美航空1016，取消著陸"；塔控制表示"收到"，并指示航班1016爬升至3,000英尺（910米）。飞机嘗試在惡劣天氣下爬升，但转向右和急速下降。{{Rp|6}} 機師不斷试图控制客机使客機爬升。因软件差异以至降低灵敏度而遲来的风切变的预警系统没有及時提醒他们。[[霍尼韦尔|霍尼韦尔]]工程师指出，飞行员应该在墜機前8-9秒收到警報。

=== 墜毀 ===
[[File:USAir_Flight_1016_(1).jpg|缩略图]]
下午6時42分,1016號航班在距離机场大約0.5英里外墜毀，壓毁机场围栏及树木。機身在地面滑行時斷成數截，飞机分成四个主要部分；前{{Convert|40|ft|m}} 的機身，包括驾驶舱和頭等客舱停在华莱士尼尔路中間。 后部分的机身，包括尾和后安装的发动机停在住宅的車庫中。

52名乘客中，37人死於钝器创伤、烧伤，或一氧化碳中毒。 另外14名乘客重伤，1人轻伤。5名机组人员中，2名[[飞行员|飞行员]]轻伤，2名[[機艙服务员|機艙服务员]]重伤，1名[[機艙服务员|機艙服务员]]轻伤。 地面上沒有人受傷。

== 调查 ==
国家运输安全委员会立即派遣一个调查小组找尋飛行紀錄儀。經過漫长的调查後。结论是，在機場附近上空的雷雨所形成的[[微下擊暴流|微下擊暴流]]。 国家运输安全委员会列出这些因素：
# 駕駛員继续在一個有可能發生[[微下擊暴流|微下擊暴流]]的情況下下降。
# 駕駛員未能迅速识别风切变(失效的风切变警示软件加劇了情況；风切变的预警系统应在墜機前8-9秒發出警報。)
# 駕駛員未能採取适当的行動離開發生風切變的範圍
# 控制塔未能及時提供天氣資訊
