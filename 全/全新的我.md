《'''全新的我'''》（{{lang-en|'''Brand New Me'''}}）是收錄在[[美國|美國]][[節奏藍調|節奏藍調]]歌手[[艾莉西亞·凱斯|艾莉西亞·凱斯]]在2012年發行的第五張專輯《[[烈火琴韻|烈火琴韻]]》裡的一首歌曲，做為專輯中第二首單曲，《全新的我》首次於2012年11月18日在電台上播出。這首歌為艾莉西亞·凱斯與[[蘇格蘭|蘇格蘭]]歌手兼作曲人[[艾蜜莉·珊黛|艾蜜莉·珊黛]]所寫，是一首節奏比較輕的民歌。歌詞中描述一個人的成長與蛻變。<ref name="about" />艾莉西亞·凱斯在2012年的{{link-en|iTunes音樂祭|iTunes Festival}}和VH1 Storytellers等場合表演過這首歌曲。

== 歌曲 ==
艾莉西亞·凱斯在2009年與身兼[[音樂製作人|音樂製作人]]及[[饒舌|饒舌]]歌手{{link-en|史威茲·畢茲|Swizz Beatz}}結婚後，在接下來的3年中也經歷了許多活動。<ref name="billboard cover story" />在接受[[告示牌_(雜誌)|告示牌]]的訪談中，凱斯提到她感覺她到了一個全新的境界，而發想要創作一張新的專輯及歌曲。<ref name="billboard cover story" />2012年11月10日，《全新的我》被公布在凱斯的網站上，成為《烈火琴韻》中的第二章單曲，<ref name="new single" /><ref name="tv3" />歌曲的音訊檔亦上傳到凱斯的官方[[VEVO|VEVO]]頻道，<ref name="new single" />在11月18日首次於電台播出。

《全新的我》是一首鋼琴民歌，作曲者為凱斯和[[蘇格蘭|蘇格蘭]]做曲家[[艾蜜莉·珊黛|艾蜜莉·珊黛]]共同創作，她們在艾蜜莉·珊黛的專輯《Our Version of Events》已有合作過。歌詞上，凱斯不斷描述著成長與發展。<ref name="new single" />[[Allmusic|Allmusic]]的Andy Kellman表示，凱斯將這首歌視為她的[[自傳|自傳]]。<ref name="allmusic" />

== 音樂影片 ==
《全新的我》的[[音樂影片|音樂影片]]於2012年12月17日在[[黑人娛樂電視台|黑人娛樂電視台]]首播，<ref name="mv" />導演為Diane Martel，影片主要是在描述凱斯在追尋的是自我發現。<ref name="dailymail" />

== 現場演出 ==
凱斯首次於2012年9月28日的{{link-en|iTunes音樂祭|iTunes Festival}}上表演這首歌曲，凱斯還在同一場演唱也唱了專輯內另一首新歌《[[錢非萬能|錢非萬能]]》。<ref name="itunes festival" />10月16日，凱斯在[[紐約|紐約]][[林肯中心|林肯中心]]的[[艾弗里·費雪廳|艾弗里·費雪廳]]裡的一場演唱會中表演這首歌曲。<ref name="lincoln center" />

凱斯公布她將會在2013年5月於[[英國|英國]]舉行演唱會，並且確定《全新的我》會在演唱會的上演出。<ref name="tour" />
