'''全国人民代表大会常务委员会立法工作计划'''（简称'''全国人大常委会立法工作计划'''）是[[全国人民代表大会常务委员会|全国人大常委会]]每年制定的对本年度全国人大常委会立法工作的具体安排。

==简介==
1993年，[[第八届全国人大常委会|第八届全国人大常委会]]出台《关于今明两年立法工作的安排意见》，这是全国人大常委会首次制定关于立法工作的计划<ref>{{Cite web |url = http://www.legaldaily.com.cn/index_article/content/2014-09/11/content_5757289.htm?node=5955 |title = 推进科学立法着力提高立法质量 |author =  |date = 2014-09-11 |publisher = 法制网 |language =  |accessdate =  |quote =  }}</ref>。自1993年[[第八届全国人大|第八届全国人大]]开始，全国人大常委会制定年度立法工作计划，称为'''全国人民代表大会常务委员会立法计划'''（简称'''全国人大常委会立法计划'''），主要对年度内提交全国人大常委会会议审议的立法项目作出安排。2008年起，正式改为'''全国人民代表大会常务委员会立法工作计划'''（简称'''全国人大常委会立法工作计划'''），增加了对立法工作层面的一些要求<ref name=cong>{{Cite web |url =  http://npc.people.com.cn/n1/2016/0517/c14576-28356403.html|title = 从“立法计划”到“立法工作计划” |author =  |date = 2016-05-17 |publisher = 人大新闻网 |language =  |accessdate =  |quote =  }}</ref>。

年度立法工作计划主要是贯彻全国人大常委会五年[[全国人民代表大会常务委员会立法规划|立法规划]]的原则要求，落实立法规划确定的立法项目的审议安排。根据经济和社会发展需要，年度立法工作计划可对立法规划进行补充和调整<ref name=cong/>。

2015年3月15日[[第十二届全国人民代表大会|第十二届全国人民代表大会]]第三次会议《关于修改〈中华人民共和国立法法〉的决定》修正后的《[[中华人民共和国立法法|中华人民共和国立法法]]》首次对全国人大常委会立法规划和年度立法工作计划作出了规定<ref name=cong/>。该法第五十二条规定：“全国人民代表大会常务委员会通过立法规划、年度立法计划等形式，加强对立法工作的统筹安排。编制立法规划和年度立法计划，应当认真研究代表议案和建议，广泛征集意见，科学论证评估，根据经济社会发展和民主法治建设的需要，确定立法项目，提高立法的及时性、针对性和系统性。立法规划和年度立法计划由委员长会议通过并向社会公布。”“全国人民代表大会常务委员会工作机构负责编制立法规划和拟订年度立法计划，并按照全国人民代表大会常务委员会的要求，督促立法规划和年度立法计划的落实。”<ref>{{Cite web |url = http://www.npc.gov.cn/npc/dbdhhy/12_3/2015-03/18/content_1930713.htm |title = 中华人民共和国立法法 |author =  |date = 2015-03-18 |publisher =中国人大网  |language =  |accessdate =  |quote =  }}</ref>

立法工作计划的编制程序通常为：在前一年度下半年由[[全国人大常委会法制工作委员会|全国人大常委会法制工作委员会]]编制下一年度立法工作计划，通常在前一年度12月由[[全国人大常委会委员长会议|全国人大常委会委员长会议]]原则通过，对下一年度立法工作作出预安排。等到下一年度3月[[全国人民代表大会|全国人民代表大会]]会议召开后，根据会议精神及[[全国人大代表|全国人大代表]]审议中提出的意见和建议，对预安排加以调整完善，经全国人大常委会委员长会议讨论通过后印发执行，并且向社会公布<ref name=cong/>。

全国人大常委会法制工作委员会在编制立法工作计划时，首先按照[[中共中央|中共中央]]的工作部署，汇总、梳理有关立法项目，根据立法项目的成熟度作出安排。同时，全国人大代表提出的议案和建议中的立法项目建议是立法工作计划项目来源之一。此外，还向[[全国人民代表大会专门委员会|全国人大各专门委员会]]、全国人大常委会有关工作机构、[[中华人民共和国国务院|国务院]]有关部门、[[最高人民法院|最高人民法院]]、[[最高人民检察院|最高人民检察院]]、[[中央军委法制局|中央军委法制局]]等单位发函以征求立法项目建议。立法工作计划草稿多次征求全国人大有关专门委员会及全国人大常委会有关工作委员会、[[国务院法制办|国务院法制办]]、中央军委法制局等的意见。在编制2016年立法工作计划时，首次向部分全国人大代表、各省级人大常委会法制工作机构以及基层立法联系点征求意见。对意见有分歧或立法思路不明确的项目，可召开座谈会，听取项目提出单位及其他单位、议案领衔代表、专家学者的意见<ref name=cong/>。
