'''全军保障社会化工作领导小组'''是已撤销的[[中央军事委员会|中央军委]]议事协调机构，负责领导全军保障社会化工作。

==沿革==
2005年4月15日，'''全军后勤保障社会化工作领导小组'''第一次会议召开，中央军委委员、总后勤部部长[[廖锡龙|廖锡龙]]讲话<ref>{{Cite web |url =http://news.sina.com.cn/c/2005-04-15/16525661693s.shtml  |title =廖锡龙：走具有我军特色的后勤保障社会化路子  |author =  |date =  2005-04-15|publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

2007年5月22日，全军后勤保障社会化工作领导小组第三次会议召开，中央军委委员、总后勤部部长廖锡龙讲话，全军后勤保障社会化工作领导小组副组长、总政治部副主任[[刘振起|刘振起]]，领导小组副组长、总后勤部副部长[[孙志强|孙志强]]，领导小组副组长、总装备部副部长[[朱发忠|朱发忠]]出席<ref>{{Cite web |url =http://news.163.com/07/0523/10/3F5V4T7H000120GU.html  |title = 廖锡龙：进一步推进军队后勤和其他保障社会化 |author =  |date =2017-05-23  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2008年9月17日，全军后勤保障社会化工作领导小组第四次会议召开，中央军委委员、全军后勤保障社会化工作领导小组组长、总后勤部部长廖锡龙讲话，会议审议了军队保障社会化“十化一改”12个分项实施方案，部署了下一阶段工作任务。总参谋长助理[[杨志琦|杨志琦]]、总政治部主任助理[[许耀元|许耀元]]、总后勤部副部长[[丁继业|丁继业]]出席会议<ref>{{Cite web |url =http://news.sohu.com/20080918/n259615932.shtml  |title = 深入推进军队保障社会化改革 |author =  |date =2008-09-18  |publisher =  搜狐|language =  |accessdate =  |quote =  }}</ref>。

2009年，全军后勤保障社会化工作领导小组更名为'''全军保障社会化工作领导小组'''<ref name=geng>{{Cite web |url =  http://news.ifeng.com/mil/gundong/detail_2010_11/24/3210925_0.shtml|title =  保障社会化绘出军队保障新蓝图|author =  |date = 2010-11-24 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

2012年2月14日，全军保障社会化工作领导小组第六次会议召开，中央军委委员、全军保障社会化工作领导小组组长、总后勤部部长廖锡龙讲话，会议审议了2012年军队保障社会化工作要点，总参谋长助理[[陈勇_(将军)|陈勇]]、总后勤部副部长[[丁继业|丁继业]]、总装备部副部长[[牛红光|牛红光]]出席会议<ref>{{Cite web |url = http://military.people.com.cn/GB/172467/17113842.html |title = 努力推动军队保障社会化向深度和广度拓展 |author =  |date = 2012-02-15 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。
