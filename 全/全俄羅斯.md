'''全俄羅斯'''（{{lang-rus|'''Общерусский народ'''|Obshcherusskiy narod}}），也被稱為'''三位一體的俄羅斯人'''或'''泛俄羅斯民族''' 由三個部分組成[[大俄羅斯|大俄羅斯]]、[[白俄羅斯|白俄羅斯]]和[[小俄羅斯|小俄羅斯]]（烏克蘭的舊稱）

根據歷史學家的觀點，[[基輔羅斯|基輔羅斯]]被認為是三個現代[[東斯拉夫人|東斯拉夫人]]國家（白俄羅斯、俄羅斯及烏克蘭）的前身，三國的歷史學家皆承認為其東斯拉夫人的共同祖先，後來[[基輔羅斯|基輔羅斯]]因[[蒙古帝國|蒙古帝國]]的[[金帳汗國|金帳汗國]]入侵而滅亡。

== 詞源 ==
三位一體的俄羅斯人的論文的第一作者被認為是基輔莫希拉學院校長，[[普斯科夫|普斯科夫]]的主教和大主教諾夫哥羅德教授塞奧法尼斯普羅科波維奇<ref>http://www.novyn.kpi.ua/2011-2/29-filos-Synjakov.pdf</ref>。

== 歷史 ==
[[斯拉夫人|斯拉夫人]]是一個廣泛生活在東南歐的民族，斯拉夫人分為東、西、南三個大族群，是為東斯拉夫人、西斯拉夫人、南斯拉夫人。
* 東斯拉夫人就是[[俄羅斯人|俄羅斯人]]、[[烏克蘭人|烏克蘭人]]、[[白俄羅斯人|白俄羅斯人]]
* 西斯拉夫人是[[波蘭人|波蘭人]]、[[捷克人|捷克人]]、[[斯洛伐克人|斯洛伐克人]]、卡舒比人（主要分佈於今波蘭北部地區，人口約30萬人）和[[索布人|索布人]]（主要分佈在德國的薩克森與布蘭登堡兩州境內，人口在5萬至6萬之間，他們居住的地區被稱為盧薩蒂亞）
* 南斯拉夫人是[[塞尔维亚人|塞尔维亚人]]、[[保加利亚人|保加利亚人]]、[[克罗地亚人|克罗地亚人]]、[[斯洛文尼亚人|斯洛文尼亚人]]、[[波斯尼亚人|波斯尼亚人]]、[[马其顿人|马其顿人]]和[[黑山人|黑山人]]。
東斯拉夫人在9世紀前是許多鬆散的部落聯盟，在9世紀時一個名叫留裡克的[[瑞典|瑞典]][[瓦良格人|瓦良格人]][[傭兵|傭兵]]在860年左右被選舉為大諾夫哥羅德的統治者，因而創建了[[留裡克王朝|留裡克王朝]]。北方的留裡克王朝南下統一了東斯拉夫人的其它部落，並在在[[第聶伯河|第聶伯河]]沿岸今日烏克蘭首都基輔建立了[[基輔羅斯|基輔羅斯]]。

12世紀時基輔羅斯已經因為封建制度而分裂成許多的小公國隨後在13世紀被蒙古[[金帳汗國|金帳汗國]]所佔領，其間金帳汗國無法管轄的地區出現權力真空而這時波蘭－立陶碗大公國佔領了烏克蘭。之後[[莫斯科公國|莫斯科公國]]的伊凡雷帝擊敗了[[金帳汗國|金帳汗國]]後蒙古的影響力逐漸在東歐消退，後來成立的[[俄羅斯沙皇國|俄羅斯沙皇國]]逐漸收復了古羅斯的領土。為了共同抵禦波蘭和土耳其的入侵，烏克蘭哥薩克領主和俄羅斯沙皇簽訂了「佩列亞斯拉夫爾協議」結盟300多年，到了彼得大帝時期已經完全收復了古羅斯全部領土。
* 「烏克蘭」這個名詞是波蘭人所取意思為邊界之地或「在邊界上」的意思一開始做為一個歧視性的地理名詞而後一戰才被烏克蘭民族主義者用來作為民族的概念。<ref>http://litopys.org.ua/rizne/nazva_eu.htm</ref>
* 烏克蘭語並非歷史悠久的語言100多年前僅被作為說不同口音的俄語，其誕生歷史在1930年代後才逐漸產生。

=== 俄羅斯與烏克蘭和白俄羅斯 ===
[[蒙古征服俄羅斯|蒙古征服俄羅斯]]後以[[東斯拉夫人|東斯拉夫人]]為主體的古[[基輔羅斯|基輔羅斯]]滅亡，兩國共同的歷史開始出現分歧<ref name="Gumelev">{{cite book|last=Gumilev|first=Lev|title=Ot Rusi k Rossii|publisher=AST|year=2005|isbn=5-17-012201-2}}</ref>，基輔羅斯分裂成3個部份尚未被蒙古征服的西北部份羅斯是[[加利西亞－沃里尼亞王國|加利西亞－沃里尼亞王國]]成為白俄羅斯和西烏克蘭的一部份，東北部基輔羅斯成為殘餘的[[弗拉基米尔-苏兹达尔大公国|弗拉基米尔-苏兹达尔大公国]]及繼承國[[莫斯科大公國|莫斯科大公國]]，東南部烏克蘭和俄羅斯依據被蒙古征服時間的不同分裂成東南烏克蘭和俄羅斯，以及[[波蘭立陶宛聯邦|波蘭立陶宛聯邦]]的軍事干預往後基輔羅斯分裂成大俄羅斯、[[小俄羅斯|小俄羅斯]]（烏克蘭）、白俄羅斯三個國家。

伊凡雷帝時期[[莫斯科公國|莫斯科公國]]從一個小公國逐漸統一了古基輔羅斯分裂的的領土，此時蒙古[[金帳汗國|金帳汗國]]因為內部的分裂和爭奪大汗王位而分裂成許多小國家如[[喀山汗國|喀山汗國]]、[[阿斯特拉罕汗國|阿斯特拉罕汗國]]和[[克里米亞汗國|克里米亞汗國]]，俄羅斯並驅逐了蒙古金帳汗國在東歐和中亞的勢力，成為一個強大的[[東斯拉夫人|東斯拉夫人]]國家此時距離基輔羅斯滅亡250年東斯拉夫人脫離蒙古統治；而西烏克蘭卻被[[立陶宛大公國|立陶宛大公國]]併吞，隨後又成為[[波蘭立陶宛聯邦|波蘭立陶宛聯邦]]的一員。烏克蘭中部的[[哥薩克人|哥薩克人]]和[[哥薩克酋長國|哥薩克酋長國]]拒絕接受波蘭貴族的控制，因波蘭強迫西烏克蘭改信天主教，經常與波蘭聯邦發生衝突，西部烏克蘭的[[利沃夫|利沃夫]]成為[[小波蘭|小波蘭]]的一部份，中部烏克蘭要求改為加入[[文化|文化]]相同的俄羅斯基督教[[東正教|東正教]]派，最終哥薩克議會於1654年通過[[佩列亞斯拉夫條約|佩列亞斯拉夫條約]]，加入[[莫斯科公國|莫斯科公國]]<ref name="Shambarov">{{cite book|last=Shambarov|first=Valery|title=Kazachestvo Istoriya Volnoy Rusi|publisher=Algorithm Expo, Moscow|year=2007|isbn=978-5-699-20121-1}}</ref>。

在[[伊凡雷帝|伊凡雷帝]]死後波蘭國王扶植了假王子，一個自稱是伊凡雷帝幼子德米特裡的人為偽王，波蘭人一度占領了莫斯科後來遭莫斯科人民起義推翻，隨後波蘭國王[[瓦迪斯瓦夫四世|瓦迪斯瓦夫四世]]稱自己為莫斯科大公和俄國沙皇。

17世紀中期，烏克蘭逐漸和[[俄羅斯帝國|俄羅斯帝國]]融合，此後在18世紀俄羅斯女皇[[葉卡捷琳娜二世|葉卡捷琳娜二世]]與普魯士和奧地利聯合瓜分波蘭後西烏克蘭和俄羅斯完成統一，古基輔羅斯領土在此時以完全收復。在200多年之後的一次世界大戰後德國扶植了[[烏克蘭國|烏克蘭國]]傀儡政權其間又有西烏克蘭的[[西蒙·彼得留拉|西蒙·彼得留拉]]綠軍開始屠殺俄羅斯人，而二戰之後納粹德國亦扶植親納粹的班德拉政權和烏克蘭反抗軍，二戰結束後一部份班德拉份子前往美國發展成為反蘇工具，直到蘇聯解體後班德拉主義者回到烏克蘭發展[[新納粹主義|新納粹主義]]並成為2014年初美國支持邁丹獨立廣場革命的參與者，在隨後的烏克蘭內戰中他們加入了烏克蘭國民禁衛軍。

== 俄羅斯的分裂歷史過程 ==
在第一次世界大戰以及1917年十月革命和[[俄國內戰|俄國內戰]]後烏克蘭和白俄羅斯受德國支持獨立，以及列寧簽訂割地賠款的[[布列斯特-立陶夫斯克條約|布列斯特-立陶夫斯克條約]]而獨立出去，列寧在受到一戰德國150萬金馬克金援作為內部間諜，德國支持列寧發動十月革命政變推翻沙皇旦條件為簽訂[[布列斯特-立陶夫斯克條約|布列斯特-立陶夫斯克條約]]為代價，儘管遭到大多數[[蘇共中央|蘇共中央]]大多數人反對，但列寧堅持簽訂該條約，這些檔案在冷戰後解密。

條約主要內容如下：
# 蘇俄終止與德國、奧匈帝國、保加利亞和土耳其的戰爭狀態；
# 蘇俄軍隊全面復員，海軍軍艦駛回海港並解除武裝；
# 蘇俄承認[[芬蘭|芬蘭]]、[[烏克蘭|烏克蘭]]、[[格魯吉亞|格魯吉亞]]之獨立，並有義務同[[烏克蘭人民共和國|烏克蘭人民共和國]]立即締結和約；
# 蘇俄將[[愛斯特蘭|愛斯特蘭]]、[[利夫蘭|利夫蘭]]、[[庫爾蘭|庫爾蘭]]、[[立陶宛|立陶宛]]、[[俄屬波蘭|俄屬波蘭]]等地割與德意志帝國與奧匈帝國，而上列各地今後之政治狀態，僅德奧二國有權決定；
# 蘇俄撤出阿達罕（Ardakhan）、[[卡爾斯|卡爾斯]]、[[巴統|巴統]]等省，由奧斯曼土耳其帝國接管；
# 蘇俄拆除[[阿蘭群島|阿蘭群島]]上所有防禦工事，其未來地位由[[德意志帝國|德意志帝國]]、[[奧匈帝國|奧匈帝國]]、[[瑞典王國|瑞典王國]]和[[芬蘭|芬蘭]]四國共同決定。
此條約對俄國是相當沉重之打擊，俄方所受之損失大致如下：
# 國土：共78萬平方公里。<ref>http://lib.ru/HISTORY/FELSHTINSKY/brestskij_mir.txt</ref>
# 人口：共5600萬人口，相當於全國34%的人口
# 產業：
## 農產損失33%
## 工業損失54%
## 製糖業損失80%
## 鐵礦損失73%
## 煤礦損失75%
# 財政收入：共損失約27%

此外，在8月27日時於柏林所簽訂之補充條款中尚規定蘇俄需給付德國60億馬克賠款，分五期償還，但最後付款日期不得超過1918年底。

隨後德國在一次世界大戰戰敗條約的內容沒有被完全被執行，而二次大戰後蘇聯又將大部分一戰後俄羅斯丟施的領土納入其勢力範圍。

在蘇聯紅軍佔領烏克蘭後不願意成為[[俄羅斯蘇維埃聯邦社會主義共和國|俄羅斯蘇維埃聯邦社會主義共和國]]統治下的領土，希望興復東正教及君主制或民族性傳統的舊俄羅斯，因此烏克蘭當地出現支持白軍的保皇復國運動，於是遭到了蘇俄的進攻。1922年列寧為了加強對烏克蘭地區的控制將俄羅斯領土[[新俄羅斯|新俄羅斯]]的東南部地區割讓給烏克蘭但此時烏克蘭只是作為蘇聯的一個[[加盟共和國|加盟共和國]]。

1991年蘇聯解體在美國支持的親西方政權[[葉利欽|葉利欽]]政府將俄羅斯、烏克蘭和白俄羅斯三國作為主體國家恢復獨立。

== 現在 ==
跟據2000年[[白俄羅斯|白俄羅斯]]進行的一項代表性民意調查，42.6％的人和受訪者表示他們相信三位一體的俄羅斯人民[[白俄羅斯|白俄羅斯]]是全俄羅斯的一部分。<ref>[http://books.google.de/books?id=DiwPRpRYt2kC&pg=PA143&dq=triune+Russian+nation&hl=de&sa=X&ei=39AkUquSH4LNtAbivYCACg&ved=0CEIQ6AEwAg#v=onepage&q=triune Russian nation&f=false von Nelly Bekus (2010) Struggle Over Identity: The Official and the Alternative "Belarusianness" P.143]</ref>

[[2014年克里米亞歸屬公投|2014年克里米亞公投決定]]加入[[俄羅斯聯邦|俄羅斯聯邦]]。2014年烏克蘭東南部8個普遍立場親俄的州，也決定重新建立1922年劃入[[烏克蘭蘇維埃社會主義共和國|烏克蘭蘇維埃社會主義共和國]]管轄的[[新俄羅斯|新俄羅斯]]。俄語地區支持者普遍支持加入[[頓涅茨克人民共和國|頓涅茨克人民共和國]]和[[盧甘斯克人民共和國|盧甘斯克人民共和國]]組成[[新俄羅斯聯邦|新俄羅斯聯邦]]。
