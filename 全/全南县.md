'''全南县'''（旧称“'''虔南'''”，1957年更名为“'''全南'''”<ref>http://www.xzqh.org/quhua/36jx/0729qn.htm</ref>）位于中国[[江西省|江西省]]南部，是[[赣州市|赣州市]]下辖的一个县。

== 历史沿革 ==

[[清朝|清朝]][[光绪|光绪]]二十九年（1903年）析龙南县之大龙、新兴堡和信丰县之镇南、扬溪、步口、回戈堡设置'''虔南厅'''。民国2年（1913）年[[废府州厅改县|废厅建县]]，为[[虔南县_(中华民国)|虔南县]]。1957年改名全南县。

== 行政区划 ==
下辖6镇，3乡。
* 镇：城厢镇、金龙镇、陂头镇、大吉山镇、南迳镇、龙源坝镇。
* 乡：中寨乡、社迳乡、龙下乡。

== 地理 ==
全南县地处江西省最南端，与广东翁源、连平、始兴、南雄、江西龙南、信丰6县（市）交界。

全南属亚热带季风型气候区。

== 交通 ==
县城处于[[105国道|105国道]]与[[106国道|106国道]]、赣粤高速与京珠高速的公路连接线上，距赣州180公里，至韶关站142公里，至龙南站45公里，至广州260公里、至东莞320公里、至深圳380公里，是江西省融入“9+2”泛珠三角经济圈和承接沿海产业转移的前沿阵地。2013年1月通車的[[龙杨高速公路|龙杨高速公路]]把全南融入珠三角二小时交通圈。
