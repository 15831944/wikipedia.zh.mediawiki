'''全国“扫黄打非”工作小组办公室'''，简称'''扫黄打非办'''，开设于1989年，是全国“扫黄打非”工作小组的办事机构，与'''中共中央宣传部反非法反违禁局'''<ref>{{cite web |url = http://www.gapp.gov.cn/govpublic/4198/184191.shtml |title = 反非法和违禁出版物司（全国“扫黄打非”工作办公室） |date = 2013-12-31 |accessdate = 2016-06-14 |website = 国家新闻出版广电总局}}</ref>[[一个机构两块牌子|一个机构两块牌子]]。

官方称，全国“扫黄打非”工作小组隶属于[[中央宣传思想工作领导小组|中央宣传思想工作领导小组]]，由中央宣传部、中央统战部、中央政法委等26个部门组成，成员由各有关部门副部级分管领导担任<ref>[http://www.shdf.gov.cn/shdf/contents/749/45113.html 中国扫黄打非网 关于我们]</ref>，办公室设在中央宣传部。负责人包括[[中共中央宣传部|中共中央宣传部]]部长<ref>[http://www.infzm.com/content/40655 探秘“扫黄打非”办]</ref>。工作任务为“意识形态斗争”<ref>[http://www.gov.cn/gzdt/2012-01/11/content_2042269.htm 柳斌杰：“扫黄打非”工作要严把“五个关口”]</ref>。

== 政策实践 ==
2014年4月，扫黄打非办参与发起“清源2014”“[[净网2014|净网2014]]”“秋风2014”和“固边2014”四个专项行动.

2015年，扫黄打非办参与发起“净网2015”“护苗2015”“秋风2015”三項專項行動。

2016年，扫黄打非办参与发起“净网2016”“护苗2016”“秋风2016”三項專項行動。

2017年4月，扫黄打非办参与发起“净网2017”“护苗2017”“秋风2017”三項專項行動<ref>{{cite web |title=全国“扫黄打非”办公室部署开展“净网2017”“护苗2017”“秋风2017”专项行动 |url=http://www.shdf.gov.cn/shdf/contents/767/318702.html |website=中国扫黄打非网 |accessdate=2018-11-06}}</ref>。

2018年4月，扫黄打非办参与发起“净网2018”“护苗2018”“秋风2018”三項專項行動<ref>{{cite web |title=全国“扫黄打非”办公室推进开展“净网2018” “护苗2018”“秋风2018”专项行动 |url=http://www.shdf.gov.cn/shdf/contents/767/364069.html |website=中国扫黄打非网 |accessdate=2018-11-06}}</ref>。

== 评价 ==
办公室自我评价其目标为“要牢固树立对党和国家高度负责的态度，着力解决社会反映强烈的问题”<ref>[http://news.xinhuanet.com/legal/2013-03/24/c_124496248.htm “扫黄打非”新部署：扫除萎靡之风 重心转到网上]瞭望周刊</ref>。工作内容为 "打击“非法出版物”、“扫除淫秽色情文化垃圾”、打击“假媒体、假记者站、假记者”（“三假”）<ref>{{cite web |url = http://www.china.com.cn/news/2014-04/14/content_32081810.htm |title=2014年“扫黄打非”工作将集中开展四个专项行动 |accessdate = 2014-08-12 |website = 中国网 |language=zh-hans}}</ref><ref>[http://www.shdf.gov.cn/shdf/contents/4482/200184.html 关于开展打击网上淫秽色情信息专项行动的公告]</ref>。
