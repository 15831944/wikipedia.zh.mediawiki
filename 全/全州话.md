'''全州话'''是流行于[[广西壮族自治区|广西]]北部[[全州县|全州]]及其临近地区的一种[[汉语方言|汉语方言]]，属于[[湘语|湘语]][[永全片|永全片]]，与[[湖南省|湖南省]]的[[永州话|永州话]]一脉相承。

全州话里有大量舌叶音。

== 声调 ==
全州话有五个声调：阴平33；阳平13；上声54；去声24；入声22.

== 词汇 ==
人称代词包括老派的“俺”ang54，“侬”long54，tong33；分别表示新派的“我”“你”“他”<ref>《全州话与普通话主要语法差异研究》，广西民族大学硕士论文，2009年．</ref>。

== 语法 ==
全州话的语法极富湘语色彩，又有自己的特点。比如状语后置“我走先”；宾语前置句“他打得我过”“比他不过”等；正正式正反问句如“你去去北京？”。与许多南方方言类似，全州话的动物性别属性的描述是主从结构，如“鸡公”“鸡婆”（公鸡、母鸡）。
