'''全国戏剧文化奖'''，前身是'''中国戏剧文学奖'''和'''中国话剧金狮奖'''，由[[中华人民共和国文化部|文化部]]在1985年建立，主要评选对象是[[舞台剧|舞台剧]]、[[电视剧|电视剧]]、[[广播剧|广播剧]]等[[戏剧|戏剧]]作品，评选规则是《中国戏剧文学奖全国优秀剧目调演章程》，2010年，在经过[[中国共产党中央委员会|中国共产党中央委员会]]、[[中华人民共和国国务院|中华人民共和国国务院]]的同意之后，该奖规格提高，该奖每3年一次。<ref name="fish " >{{cite web |url=http://www.zhongguoxijuchang.com/Article/ShowArticle.asp?ArticleID=4144 |title=新浪网：第二届全国戏剧文化奖启动评选有新招 |accessdate=2012-07-19 |publisher=中国剧场 |archiveurl=https://web.archive.org/web/20121109093928/http://www.zhongguoxijuchang.com/Article/ShowArticle.asp?ArticleID=4144 |archivedate=2012年11月9日 |work= |language=zh-cn |dead-url=yes }}</ref><ref name=" ops">{{cite web|url=http://www.zhongguoxijuchang.com/Article/ShowArticle.asp?ArticleID=1931|title=全国戏剧文化奖章程|accessdate=2011-06-11|publisher=中国剧场|archiveurl=https://web.archive.org/web/20130414082702/http://www.zhongguoxijuchang.com/Article/ShowArticle.asp?ArticleID=1931|archivedate=2013年4月14日|work=|language=zh-cn|deadurl=yes}}</ref>

== 定义 ==
全国戏剧文化奖由[[中华人民共和国文化部|文化部]]旗下[[中国戏剧文学学会|中国戏剧文学学会]]举办，是纯民间的政府奖，但是不代表官方[[意识形态|意识形态]]。<ref>{{cite web |url=http://gb.cri.cn/32464/2011/06/01/2385s3265176.htm |title=全国戏剧文化奖引风波“国家级”身份遭质疑 |accessdate=2011-06-01 |publisher=国际在线 |archiveurl=http://gb.cri.cn/32464/2011/06/01/2385s3265176.htm |archivedate=2012年11月15日 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://beijingww.qianlong.com/1470/2012/07/19/290@162568.htm |title=“全国戏剧文化奖”不需层层申报 |accessdate=2012-07-19 |publisher=千龙网 |archiveurl=http://beijingww.qianlong.com/1470/2012/07/19/290@162568.htm |archivedate=2012年11月15日 |work=|quote =全国戏剧文化奖下设“剧本奖”、“剧目奖”、“戏剧理论评论奖”、“出品人及制作人奖”，与其他国家级奖项不同的是，该奖项由文化部主管社团中国戏剧文学学会主办，是一项纯民间的政府奖。 |language =zh-cn}}</ref>

== 类别 ==
全国戏剧文化奖分为多个小奖，剧目奖为主心骨<ref name="fish " /><ref name=" ops" />：
*剧本奖（舞台剧、影视剧、广播剧等剧本）
*剧目奖（综合奖及编、导、演、音乐、舞美等单项奖，话剧除外）
*戏剧理论评论奖（包含论文和论著）
*出品人及制作人奖（包含单位部门组织奖）
