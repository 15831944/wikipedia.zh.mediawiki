'''全國家長團體聯盟'''（簡稱'''全家盟'''），為[[臺灣|臺灣]]一個由民間組成的[[家長|家長]]團體，於2002年10月26日結盟，正式成立於2005年5月22日。積極支持[[臺灣教育改革|教育改革]]與壓力減輕<ref>[http://www.libertytimes.com.tw/2012/new/aug/26/today-life8-2.htm?Slots=BLife 經濟不佳 家長反對調漲]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。現任理事長為彭淑燕。全家盟主張「推動[[十二年國民基本教育|十二年國教]]」、「推動教師專業評鑑」、「促進家長參與教育」及「發現孩子軟實力」，也曾在[[2010年五都選舉|2010年五都選舉]]發表致五[[直轄市|直轄市]]市長候選人公開信<ref>{{Cite web |url=http://www.napo.org.tw/zhong-da-jiao-yu-zheng-ce-tui-dong/zhi-wu-dou-hou-xuan-ren-gong-kai-xin |title=全國家長團體聯盟致五都市長候選人暨國、民兩黨黨中央公開信 |access-date=2013-07-19 |archive-url=https://web.archive.org/web/20131228032450/http://www.napo.org.tw/zhong-da-jiao-yu-zheng-ce-tui-dong/zhi-wu-dou-hou-xuan-ren-gong-kai-xin |archive-date=2013-12-28 |dead-url=yes }}</ref>，要求儘速施行[[十二年國教|十二年國教]]<ref>[http://blog.udn.com/mybook678/19948466 全家盟挺12年國教，真家長何不組7-11聯盟？]</ref>。全家盟為台灣一個有名的家長組織，但所代表的家長數量不明<ref>[http://blog.udn.com/mybook678/19897811 12年國教爭議，誰是真家長、假家長？]</ref>。

2010年，全家盟與學界一起推動《兒童教育與照顧法》<ref>[http://www.mdnkids.com/info/news/content.asp?Serial_NO=66776 兒照法對象　全家盟支持0到12歲]</ref>；但該法因為涵蓋0-12歲的孩子、範圍過大，最後拆解為《幼兒教育與照顧法》，於2012年3月立法通過<ref>[http://www.cylaw.org.tw/Law2-1-1.asp?id=719 幼兒教育及照顧法」於立法院三讀通過] </ref>。 2013年6月，全家盟呼籲儘速通過12年國教修法 <ref>[http://tw.news.yahoo.com/全家盟籲-盡快通過十二年國教修法-101631205.html 全家盟籲　盡快通過十二年國教修法] </ref>全家盟的十二年國教主張免試及適性，對於學科菁英要求保留特招方式，維持數理、語文智慧較強學生有暢通升學管道。全家盟關注教育資源公平合理分配，主張儘速修改《教師法》，將教師評鑑及修改合理教評會組成比例入法，讓每一個孩子都有公平合理的受教權。

2009年到2012年間，全家盟向[[中華民國教育部|中華民國教育部]]申請多項12年國教宣導相關補助費用。<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20140609/412854/ 12年國教不同調 家長聯盟槓全家盟]</ref>

==宗旨==
{{無來源章節}}

*保障學生學習權益，提升以學生為主體的優質教育。
*增進家長教育權及家長參與教育權之知能發展。
*推展親師合作及愛的教育。
*促進國際教育交流。

== 任務 ==
{{無來源章節}}

*參與及監督中央政府教育政策、法令與制度之訂定。
* 推動各縣市家長團體參與及監督當地教育政策、法令與制度之訂定。
* 建立家長團體間之聯繫網絡、資訊通路與合作關係。
* 協助各級家長組織之成立與發展。
* 增進家長參與教育事務之知能與方法。
* 提供家長親職教育資訊與經驗交流之管道。
* 其他符合本會宗旨之任務。
