'''全子瑞'''，[[中華民國陸軍|中華民國陸軍]][[中將|中將]]，[[臺灣|臺灣]][[高雄|高雄]]人，[[籍貫|籍貫]][[河南省|河南省]][[淅川縣|淅川縣]]<ref>[http://www2.cma.edu.tw/admin/library/newweb/big5/history/in2.htm　歷任校長 ] {{webarchive|url=https://web.archive.org/web/20131006024944/http://www2.cma.edu.tw/admin/library/newweb/big5/history/in2.htm |date=2013-10-06 }} 陸軍官校歷任校長</ref>，第27任[[中華民國陸軍軍官學校|中華民國陸軍軍官學校]][[校長|校長]]<ref>[http://www.cma.edu.tw/ac_principal.php?&page=6　中華民國陸軍軍官學校-歷屆校長] 中華民國陸軍軍官學校 [2013年2月9日]</ref>。曾任[[陸軍|陸軍]][[大膽島|大膽島]]、莒指部、關指部指揮官、六軍團副指揮官、花東防衛指揮部[[指揮官|指揮官]]與陸軍司令部參謀長等要職，於2017年7月1日陸軍教準部指揮官職退伍<ref>[http://mna.gpwb.gov.tw/post.php?id=9&message=84305 李總長接見屆退將官　表彰軍旅服務奉獻] 軍聞社 2017年6月29日 [2017年7月1日]</ref>。

== 班期更迭 ==
陸官早期雖班隊單一，但非故定一年一招生，故僅述陸官第幾期畢業；俟來台後改為四年制一年一招後，有人習慣上加以（年班）；而在1978年招收專科班後，專科班為了和原大學制班隊有所區隔，在（稱呼）與（畢業證書）上，冠以「專科班」再加示（第幾期）、（第幾年班）；故陸官官網之「歷屆校長」期別介紹，在全子瑞前僅以「陸官第幾期」登載，全子瑞起即冠以「正期班」或「專科班」第幾年班做為區分，值得注意的是，陸官在2004年（專科27期）停招專科班後，大學制班隊仍延用「正期班」制至今。

== 軍旅仕途 ==
全子瑞在2010年7月16日接任陸官校長，時任[[陸軍|陸軍]][[司令|司令]]楊天嘯在布達新職指出，新任校長全子瑞少將為陸軍官校51期七十一年班畢業，先後完成陸院、戰院深造教育，並歷練陸軍[[排長|排長]]、[[連長|連長]]、[[營長|營長]]、一五八旅參謀主任、副旅長、陸軍官校專指部與學指部指揮官、花蓮防衛指揮部參謀長，以及莒光地區、關渡地區指揮部指揮官等重要職務，學經歷完整，部隊實務經驗豐富<ref>[http://www.ptt.cc/bbs/Militarylife/M.1279302197.A.EED.html 陸軍官校新任校長全子瑞今就職履新] 轉軍聞社 2010年7月16日 [2013年2月9日]</ref>。全子瑞在2012年7月1日卸任校長，獲頒干城甲種二等獎章，肯定他推動校務發展及完成校務規劃的重大貢獻<ref>[http://www.ptt.cc/bbs/Militarylife/M.1341409823.A.ECE.html　李翔宙主持陸軍官校長布達交接]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }} 轉軍聞社 2012年7月2日 [2013年2月9日]</ref>，隨即接任六軍團副指揮官，再於2013年1月1日接任花東防衛指揮部指揮官<ref>[http://news.gpwb.gov.tw/news.aspx?ydn=026dTHGgTRNpmRFEgxcbfcCSN9Fhd8KFbqLRgMWauV9AENI1z12tlkOe75KgZJ2kSeEtXWiBna7zdEOv1lkVJzdkASNlOkEuJsjCUPUDN6M%3d 李翔宙主持金防部 花防部指揮官任職布達]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }} 青年日報 20102年12月28日 [2013年2月9日]</ref><ref>[http://www.libertytimes.com.tw/2012/new/dec/29/today-p4.htm　國軍組織人事大變動] {{webarchive|url=https://web.archive.org/web/20130101074420/http://www.libertytimes.com.tw/2012/new/dec/29/today-p4.htm |date=2013-01-01 }} 自由電子報 20102年12月29日 [2013年2月9日]</ref>。

== 治校與校際發展 ==
全子瑞認為，陸軍官校是培養[[中華民國陸軍|中華民國陸軍]]軍事幹部的搖籃，更為[[國家|國家]]的安定與發展的基礎，因此官校教學要著重品德教育，形塑允文允武現代化優質幹部。除治理校務，他也致力於校際發展，於2010年8月18日參訪台灣專案管理學會洽談未來相關合作事宜<ref>[http://www.tw-pma.org.tw/news/2010year/news20100818-1.html 陸官校長全子瑞將軍參訪洽談合作事宜] 台灣專案管理學會 2010年8月18日 [2013年2月9日]</ref>。另於2011年12月27日拜訪與敦聘[[成功大學|成功大學]]校長[[黃煌煇|黃煌煇]]為校務委員，交換治校理念<ref>[http://news.secr.ncku.edu.tw/files/14-1054-87087,r81-1.php　吸取成大經驗 陸官校長全子瑞少將27日拜會成大校長黃煌煇] {{Wayback|url=http://news.secr.ncku.edu.tw/files/14-1054-87087,r81-1.php |date=20131202233828 }} 成功大學 2011年12月27日 [2013年2月9日]</ref>。

== 勳獎章 ==
全子瑞服務軍旅卅餘年，共獲頒「[[忠勤勳章|忠勤勳章]]」3座、「陸海空軍獎章」1座、「干城獎章」1座、「陸光獎章」1座、「金甌獎章」3座、「弼亮獎章」3座、「景風獎章」4座與「寶星獎章」4座等共20座勳[[獎章|獎章]]（同類勳獎章不做細分）。<ref>[http://mna.gpwb.gov.tw/post.php?id=9&message=84305 李總長接見屆退將官　表彰軍旅服務奉獻] 軍聞社 2017年6月29日 [2017年7月1日]</ref>
