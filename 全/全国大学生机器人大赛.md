'''全国大学生机器人大赛'''始于2002年。目前由[[中国共产主义青年团中央委员会|共青团中央]]及[[中华全国学生联合会|全国学联]]主办。旗下设立'''Robocon'''（自2002年至2013年由[[中国中央电视台|中国中央电视台]]主办，2013年底移交共青团中央及全国学联主办）、'''RoboMaster'''（另由[[深圳市人民政府|深圳市人民政府]]及[[大疆创新|大疆创新]]联合主办）、'''ROBOTAC'''和'''机器人创业赛'''四项赛事。

== Robocon ==
{{Infobox sports league|title=全国大学生机器人大赛Robocon赛事|motto=让思维沸腾起来、让创新行动起来|champion=[[东北大学|东北大学]] Action|sport=机器人竞赛|last_season=|logo=http://www.cnrobocon.net/static/con-nav.png|website=http://www.cnrobocon.net/}}全国大学生机器人大赛Robocon赛事始于2002年，每年举办一次，系全国大学生机器人大赛的主要赛事。大赛的冠军队将代表中国参加由亚洲-太平洋广播电视联盟（Asia-Pacific Broadcasting Union，ABU）主办的[[亞太廣播聯盟機械人大賽|亚太大学生机器人大赛（ABU Robocon）]]。大赛目前已成为国内技术挑战性最强、影响力最大的大学生机器人赛事。中国在此赛事中已五次夺冠<ref name = ":1">{{Cite web|url=http://www.cnrobocon.net/#/about/origin-introduce|title=ROBOCON - 关于大赛|accessdate=2018-03-15|author=|date=|work=www.cnrobocon.net|publisher=}}</ref>。
该赛事每年都将由ABU Robocon的承办国（每年一换）制定和发布比赛的主题和规则。全国大学生机器人大赛Robocon赛事将采用这个规则进行比赛。参赛者需要综合运用机械、电子、控制、计算机等技术知识和手段，经过约十个月制作和准备，利用机器人完成规则设置的任务。作为高技术的竞赛平台，这个比赛从一开始就吸引了大专院校学生的浓厚兴趣。我国代表队在参加过的16届ABU Robocon中表现不凡，在国际上展现了我国机器人教育的卓越水平<ref name = ":1" />。

[[东北大学_(中国)|东北大学 (中国)]]于''2018年6月21日''在全国大学生机器人大赛Robocon赛事中卫冕三连冠<ref>[http://www.chinanews.com/gn/2018/06-21/8543331.shtml 东北大学将代表中国参加亚太大学生机器人大赛_中新网]</ref>，2019年的Robocon国际赛由[[蒙古国|蒙古]][[乌兰巴托|乌兰巴托]]主办<ref>{{Cite news|url=http://aburobocon2019.mnb.mn/en|title=ABU ROBOCON 2019|newspaper=http://aburobocon2019.mnb.mn/|accessdate=2018-08-27|language=en-US}}</ref>。

全国大学生机器人大赛RoboMaster机甲大师赛由共青团中央、全国学联、深圳市人民政府联合主办，[[大疆创新|大疆创新]]发起并承办。始于2015年，原名Robomasters，自2017赛季起更名RoboMaster，并确定中文名“机甲大师”。是中国国内首个射击对抗类的机器人比赛。<ref>{{Cite web|url=https://www.robomaster.com/zh-CN/robo/about|title=RoboMaster - 机器人大赛/大赛介绍|accessdate=2018-03-15|author=|date=|work=www.robomaster.com|publisher=}}</ref>

赛事融合了[[机器视觉|机器视觉]]、[[嵌入式系统|嵌入式系统设计]]、[[机械设计|机械设计]]、[[惯性导航系统|惯性导航]]、[[人机交互|人机交互]]等众多机器人相关技术学科，采用机器人射击对抗形式开展竞赛。参赛队员需要独立设计、制造机器人。在每局限时7分钟的比赛时间里，双方队员各自操作己方机器人，在复杂的“战场环境”中寻找敌方进行射击对抗。机器人所搭载的“皮肤系统”可感知弹丸的击打数据，并通过“血量”形式呈现出来。当比赛时间耗尽，裁判系统将根据双方基地血量高低或尚且存活的机器人总血量高低判定胜负。

根据RoboMaster2018赛季赛程安排，将依次开展分区赛、国际邀请赛、复活赛、全国赛（年度总决赛）。分区赛和国际邀请赛的优胜战队直接获得晋级年度总决赛的资格，部分战队可获得复活资格。

== ROBOTAC ==
{{Infobox sports league|title=全国大学生机器人大赛ROBOTAC赛|sport=机器人竞赛|website=https://www.robotac.cn/}}ROBOTAC是ROBOT(机器人)+TACTIC(策略、战略)的缩写。从2015年起，ROBOTAC成为共青团中央和全国学联主办的“全国大学生机器人大赛”的赛事项目之一。<ref>{{Cite web|url=http://www.robotac.cn/robotac/introduce|title=ROBOTAC - 大赛介绍|accessdate=2018-03-15|work=www.robotac.cn|language=zh-CN}}</ref>

== 机器人创业赛 ==
在原有全国大学生机器人大赛Robocon赛事、Robomasters赛事、Robotac赛事的基础上，共青团中央、全国学联决定联合[[黑龙江省人民政府|黑龙江省人民政府]]，自2016年起共同组织第十五届全国大学生机器人大赛机器人创业赛。
