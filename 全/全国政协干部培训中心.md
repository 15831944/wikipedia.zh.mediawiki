'''全国政协干部培训中心'''（'''全国政协北戴河管理局'''），位于[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[北戴河区|北戴河区]]康乐路5号，是[[全国政协办公厅|全国政协办公厅]]直属正司局级[[事业单位|事业单位]]。

==简介==
全国政协干部培训中心（全国政协北戴河管理局）是为政协系统干部提供培训、相关会议与接待服务的事业单位，承担全国政协系统干部和地方政协委托的有关培训任务。经费来源为财政补助收入和事业收入<ref>{{Cite web |url = http://www.gjsy.gov.cn/cxzl/ |title =全国政协干部培训中心 全国政协北戴河管理局 |author =  |date =  |publisher = 事业单位在线 |language =  |accessdate = 2018-02-08 |quote =  }}</ref><ref>{{Cite web |url =  http://www.cppcc.gov.cn/zxww/2015/04/09/ARTI1428569386122666.shtml|title =全国政协干部培训中心(全国政协北戴河管理局)  |author =  |date =2015-04-09  |publisher =  中国政协网 |language =  |accessdate =  |quote =  }}</ref>。该单位是全国政协系统的最高培训机构<ref name=wy/>。自1992年始，全国政协面向地方政协干部开展培训工作<ref>乔梁，春天的邀请，中国政协2009(4):82-83</ref>。1994年，全国政协干部培训中心成立<ref>张志国，政协干部向往的学校——全国政协干部培训中心十一年工作成果综述，中国政协2004:22-25</ref>。截至2010年，全国政协培训中心已举办培训班74期，培训学员3万多人<ref name=wy>{{Cite web |url = http://news.163.com/10/0927/14/6HJIT72900014JB6.html |title =重庆北碚区政协主席在全国政协培训中心讲座受欢迎  |author =  |date =2010-09-27  |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。
