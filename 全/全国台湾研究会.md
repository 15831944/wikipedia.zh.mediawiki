'''全国台湾研究会'''（{{lang-en|National Society of Taiwan Studies}}），簡稱'''NSTS'''，是[[中华人民共和国|中华人民共和国]]从事[[台湾问题|台湾问题]]和[[台湾海峡两岸关系|台湾海峡两岸关系]]研究的学者、有关人员和团体组成的全国性民间学术社团。現會址為[[北京市|北京市]][[海淀区|海淀区]]双清路[[清华同方大厦|清华同方大厦]]B座四层。

==简介==
1988年8月16日，'''台湾研究会'''成立<ref name=jj/>（后称'''全国台湾研究会'''<ref>孔翔云，两岸史学界首次聚会研讨台湾史，台声1995(10):4-4</ref>）。该会旨在为[[海峡两岸|海峡两岸]]及海内外从事台湾问题和两岸关系研究的专家学者搭建交流平台，推动学术活动，增进学者间交流、友谊与合作，促进两岸关系和平发展与祖国和平统一<ref name=jj>{{Cite web |url =  http://tyh.taiwan.cn/benhuijianjie/200902/t20090225_835707.htm|title = 全国台湾研究会简介 |author =  |date =2009-02-25  |publisher = 全国台湾研究会 |language =  |accessdate =  |quote =  }}</ref>。

全国台湾研究会的最高权力机构是会员代表大会。理事会是会员代表大会的执行机构。常务理事会由理事会选举产生，在理事会闭会期间行使理事会的部分职权，对理事会负责。本会设会长、副会长、秘书长，其中秘书长为本会法定代表人<ref>{{Cite web |url =  http://tyh.taiwan.cn/benhuizhangcheng/200911/t20091127_1156015.htm|title = 第四章　组织机构和负责人产生、罢免 |author =  |date =  2009-11-27|publisher = 全国台湾研究会 |language =  |accessdate =  |quote =  }}</ref>。

1991年起，全國台灣研究會、[[中華全國台灣同胞聯誼會|中華全國台灣同胞聯誼會]]、[[中國社會科學院台灣研究所|中國社會科學院台灣研究所]]每年聯合主辦“海峽兩岸關係學術研討會”，簡稱“三台會”，截至2018年第27屆均未曾停辦<ref>{{Cite web|url=http://hk.crntt.com/doc/1051/4/8/7/105148747.html|author=趙家麟|language=zh-hk|title=余元傑憂心：南台灣對兩岸關係不了解不在意|publisher=中評社|date=2018-08-03|accessdate=2018-08-05}}</ref>。
