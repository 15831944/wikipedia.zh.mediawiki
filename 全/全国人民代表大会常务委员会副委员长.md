'''全国人民代表大会常务委员会副委员长'''，简称'''全国人大常委会副委员长'''{{NoteTag|根据官方标准，不可过度简称为“全国人大副委员长”。}}，是[[中华人民共和国|中华人民共和国]]最高[[人民代表大会制度|国家权力机关]][[全国人民代表大会|全国人民代表大会]]的常设机关[[全国人民代表大会常务委员会|全国人民代表大会常务委员会]]及其[[全国人民代表大会常务委员会委员长会议|委员长会议]]的组成人员之一，位列[[中华人民共和国|中华人民共和国]][[党和国家领导人|国家领导人]]（[[国家级副职|国家级副职]]）。

==历史==
1954年3月23日，在中华人民共和国宪法起草委员会第一次会议上，[[中共中央|中共中央]]提出《中华人民共和国宪法草案（初稿）》，其中用的是“副议长”这一名称<ref name=pp>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1631918 |title =全国人大为何设“委员长”而不是“议长”  |author =  |date =  2017-03-04 |publisher =澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

1954年9月20日，[[第一届全国人民代表大会第一次会议|第一届全国人民代表大会第一次会议]]通过的《[[五四宪法|中华人民共和国宪法]]》规定，全国人民代表大会常务委员会由“委员长、副委员长若干人、秘书长、委员若干人”组成。由此，“副委员长”这一名称在宪法中获得确定。此后的1975年宪法、1978年宪法、1982年宪法对“副委员长”这一名称都未作修改<ref name=pp/><ref name=wusi>{{Cite web |url = http://www.npc.gov.cn/wxzl/wxzl/2000-12/26/content_4264.htm |title = 中华人民共和国宪法（1954年） |author =  |date = 2000-12-26 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>。

1954年宪法没有规定副委员长的职权。1954年《[[中华人民共和国全国人民代表大会组织法|中华人民共和国全国人民代表大会组织法]]》规定，副委员长的主要作用是在“常务委员会委员长因为健康情况长期不能工作或者缺位的时候，由常务委员会在副委员长中推选一人代理委员长的职务，直到委员长恢复健康或者全国人民代表大会选出新的委员长为止”。另外副委员长可以向常务委员会提出议案<ref name=wusi/><ref name=wusizuzhi>{{Cite web |url =http://www.npc.gov.cn/wxzl/wxzl/2000-12/10/content_4265.htm  |title =  中华人民共和国全国人民代表大会组织法（1954年）|author =  |date = 2000-12-10|publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>。

1975年，[[第四届全国人民代表大会第一次会议|第四届全国人民代表大会第一次会议]]通过了修改后的《[[七五宪法|中华人民共和国宪法]]》，其中没有规定副委员长的职权<ref name=qiwu>{{Cite web |url = http://www.npc.gov.cn/wxzl/wxzl/2000-12/06/content_4362.htm |title = 中华人民共和国宪法（1975年） |author =  |date =2000-12-06  |publisher =  中国人大网|language =  |accessdate =  |quote =  }}</ref>。1978年[[第五届全国人民代表大会第一次会议|第五届全国人民代表大会第一次会议]]通过了修改后的《[[七八宪法|中华人民共和国宪法]]》，其中第二十六条规定，“全国人民代表大会常务委员会副委员长协助委员长工作，可以代行委员长的部分职权。”<ref name=qiba>{{Cite web |url = http://www.npc.gov.cn/wxzl/wxzl/2000-12/06/content_4365.htm |title = 中华人民共和国宪法（1978年） |author =  |date =2000-12-06  |publisher =中国人大网  |language =  |accessdate =  |quote =  }}</ref>1982年[[第五届全国人民代表大会第五次会议|第五届全国人民代表大会第五次会议]]通过了修改后的《[[中华人民共和国宪法|中华人民共和国宪法]]》，对副委员长的职权进行了修改<ref name=baer>{{Cite web |url = http://www.npc.gov.cn/wxzl/wxzl/2000-12/06/content_4421.htm |title = 中华人民共和国宪法（1982年） |author =  |date = 2000-12-06 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>。

1954年宪法、1975年宪法、1978年宪法、1982年宪法均规定，副委员长的人数为“若干人”<ref name=wusi/><ref name=qiwu/><ref name=qiba/><ref name=baer/>。

副委员长和全国人民代表大会的任期相同，每届任期五年（1954年宪法每届任期四年）。1982年宪法第六十六条规定，“委员长、副委员长连续任职不得超过两届”<ref name=baer/>。

==选举==
1954年《中华人民共和国全国人民代表大会组织法》第十七条规定：“全国人民代表大会常务委员会委员长、副委员长、秘书长和委员，由每届全国人民代表大会在第一次会议中选出。”<ref name=wusizuzhi/>

1982年《中华人民共和国全国人民代表大会组织法》第十三条规定：“全国人民代表大会常务委员会委员长、副委员长、秘书长、委员的人选，……由[[全国人民代表大会主席团|主席团]]提名，经各代表团酝酿协商后，再由主席团根据多数代表的意见确定正式候选人名单。”第二十三条规定：全国人民代表大会常务委员会由委员长、副委员长若干人、秘书长、委员若干人组成，“常务委员会的组成人员由全国人民代表大会从代表中选出。”“常务委员会的组成人员不得担任国家行政机关、审判机关和检察机关的职务；如果担任上述职务，必须向常务委员会辞去常务委员会的职务。”<ref name=baerzuzhi/>

==职权==
根据1982年《[[中华人民共和国宪法|中华人民共和国宪法]]》和《中华人民共和国全国人民代表大会组织法》的规定，副委员长、[[全国人民代表大会常务委员会秘书长|秘书长]]协助[[全国人民代表大会常务委员会委员长|委员长]]工作。委员长、副委员长、秘书长组成[[全国人民代表大会常务委员会委员长会议|委员长会议]]，处理全国人民代表大会常务委员会的重要日常工作<ref name=baer/><ref name=baerzuzhi>{{Cite web |url =http://www.npc.gov.cn/wxzl/wxzl/2000-12/06/content_4425.htm  |title =中华人民共和国全国人民代表大会组织法（1982年）  |author =  |date =2000-12-06  |publisher =中国人大网  |language =  |accessdate =  |quote =  }}</ref>。

副委员长受委员长的委托，可以代行委员长的部分职权。在委员长因健康状况不能工作或者缺位的时候，由常务委员会在副委员长中推选一人代理委员长的职务，直到委员长恢复健康或者全国人民代表大会选出新的委员长为止<ref name=baerzuzhi/>。
