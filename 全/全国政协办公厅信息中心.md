'''全国政协办公厅信息中心'''，位于[[北京市|北京市]][[西城区|西城区]][[太平桥大街|太平桥大街]]23号，是[[全国政协办公厅|全国政协办公厅]]直属[[事业单位|事业单位]]。

==简介==
2006年5月11日，全国政协办公厅信息中心成立会在北京举行，全国政协秘书长[[郑万通|郑万通]]出席并讲话。郑万通指出，2006年初颁发的《[[中共中央|中共中央]]关于加强人民政协工作的意见》对政协自身建设提出了两位一体的任务，对政协机关建设提出了明确要求，以电子政务为主线的信息化建设是2016年政协机关建设的两大重点工程之一，是全国政协办公厅工作的重要组成部分<ref name=chl>全国政协办公厅信息中心成立，中国政协2006(5)</ref>。

全国政协办公厅信息中心是全国政协办公厅信息化建设的业务承担者<ref name=chl/>，为政协系统提供信息服务。业务有信息系统建设与资料管理、信息产品开发、信息采集与加工处理、信息分析与发布、相关咨询服务与培训<ref>{{Cite web |url =  http://www.gjsy.gov.cn/cxzl/|title =  全国政协办公厅信息中心|author =  |date =  |publisher = 事业单位在线 |language =  |accessdate =2018-02-08  |quote =  }}</ref>。

中国政协网于2008年开通<ref>{{Cite web |url =http://www.fmprc.gov.cn/ce/cohk/chn/xwfb/t535470.htm  |title =十一届全国人大二次会议和全国政协十一届二次会议采访通知  |author =  |date = 2009-02-05 |publisher = 中华人民共和国外交部驻香港特别行政区特派员公署 |language =  |accessdate =  |quote =  }}</ref>。由全国政协办公厅信息中心主办，[[中国网络电视台|中国网络电视台]]（央视网）提供技术支持。2017年12月27日，全国政协互联网新版门户网站正式上线，[[全国政协副主席|全国政协副主席]]兼秘书长[[张庆黎|张庆黎]]出席开通仪式并启动门户网站<ref>{{Cite web |url =  http://news.jschina.com.cn/scroll/guonei/201712/t20171227_1301500.shtml|title =
全国政协互联网新版门户网站正式开通 张庆黎出席开通仪式  |author =  |date = 2017-12-27 |publisher = 中国江苏网 |language =  |accessdate =  |quote =  }}</ref>。
