'''全鐘瑞'''（{{lang-ko|전종서}}，{{bd|1994年|7月5日}}），[[大韩民国|韓國]][[女演員|女演員]]，於2018年憑[[李滄東|李滄東]]執導的《[[燒失樂園|燒失樂園]]》出道。

== 生平 ==
全鐘瑞出讀前就讀[[世宗大學|世宗大學]]電影藝術系。2018年參與[[李滄東|李滄東]]執導的《[[燒失樂園|燒失樂園]]》的試鏡，並順利獲得角色，與[[劉亞仁|劉亞仁]]、[[史蒂文·連|史蒂文·連]]共同主演。該片入圍[[第71屆坎城影展|第71屆坎城影展]]正式競賽電影，因此全鐘瑞與劇組一同前往法國[[康城|康城]]走上紅地毯，得到了世界傳媒的關注<ref>{{Cite web |url = https://tw.appledaily.com/new/realtime/20180516/1354702/ |title = 【坎城影展】潔西卡美肩尬丘巴卡　劉亞仁首出擊被豬隊友帶塞 |author = 陳薇安 |date = 2018-05-16 |publisher = 蘋果日報 |language = zh-hant |accessdate = 2018-05-17 }}</ref>。全鐘瑞亦憑海美一角，入圍多個電影獎項。
