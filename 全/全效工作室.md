'''全效工作室'''（{{lang-en|Ensemble Studios}}），隸屬於[[微軟|微軟]]的遊戲開發公司。其經典作品包括廣受歡迎的[[世紀帝國|世紀帝國]]系列。

1995年，該公司由[[Tony_Goodman|Tony Goodman]]，[[里克·古德曼|里克·古德曼]]，{{link-en|Bruce Shelley|Bruce Shelley}}和{{link-en|Brian Sullivan|Brian Sullivan}}在[[美國|美國]][[達拉斯|達拉斯]]創建。1997年，他們出品了第一款遊戲[[世紀帝國|世紀帝國]]。隨後陸續製作了《世紀帝國2》、《神話時代》和《世紀帝國3》等遊戲。2001年5月，[[微軟|微軟]]收購了該擁有約100名員工的公司。

== 關閉公司 ==

2008年9月9日，微軟宣布旗下子公司全效工作室將在發行《[[星環戰役|星環戰役]]》之後關閉，據獨立的報告指出，所有與這件專案無關的員工已遭資遣，<ref name="ensembleclosing">{{cite web | url = http://www.shacknews.com/onearticle.x/54648 | title = Ensemble Studios rumoured to be closing | work = Shacknews| accessdate = 2008年11月2日}}</ref> 有媒體分析說，Ensemble Studios遭關閉，在一定程度上是基於PC遊戲市場的相對衰落以及微軟希望旗下製作組更專注於Xbox360遊戲開發等因素造成的<ref>[http://pc.duowan.com/article/20080912/1221153359.html 微軟決定關閉Ensemble Studios] {{webarchive|url=https://web.archive.org/web/20090519050542/http://pc.duowan.com/article/20080912/1221153359.html |date=2009-05-19 }} 多玩游戏网 2008-09-12</ref>。
