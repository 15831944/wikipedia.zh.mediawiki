'''楼楼'''（？－2017年10月14日，原名'''仲二楼'''<ref>{{cite web |author1=微宠物 |title=盘点各大网红猫 |url=https://kuaibao.qq.com/s/20180327B0K9RJ00?refer=spider |website=天天快报 |accessdate=2018-11-11 |date=2018-03-27 |archive-url=https://web.archive.org/web/20181111084800/https://kuaibao.qq.com/s/20180327B0K9RJ00?refer=spider |archive-date=2018-11-11 | deadurl=no}}</ref>）是一只橘白相间的[[中华田园猫|中华田园猫]]，由[[新浪微博|新浪微博]]用户楼板娘和其丈夫楼爸所饲养，因为其洗澡的视频而流行于网络，然后被制作成[[表情包|表情包]]流行于[[微信|微信]]等[[社交网络|社交网络]]。2017年10月11日晚上，楼楼出现下肢[[瘫痪|瘫痪]]和[[呕吐|呕吐]]等严重病症被送往[[宠物医院|宠物医院]]急救。2017年10月14日凌晨0时25分，楼楼因为{{le|肥厚性心肌病|Hypertrophic cardiomyopathy}}引发的{{le|肺水肿|Pulmonary edema}}和[[血栓|血栓]]引发的下肢瘫痪医治无效而被实施[[安乐死|安乐死]]<ref>{{cite web |author1=萌爪医生 |title=楼楼回喵星了，它得的是猫咪中最常被诊断出的疾病 |url=https://daily.zhihu.com/story/9651877 |website=知乎日报 |accessdate=2018-11-06 |archive-url=https://web.archive.org/web/20181111084840/https://daily.zhihu.com/story/9651877 |archive-date=2018-11-11 | dead-url=no}}</ref><ref name="我们的表情包没了">{{cite web |title=我们的表情包没了 网红猫“楼楼”去世 68万粉丝哀悼 |url=http://tech.sina.com.cn/roll/2017-10-16/doc-ifymviyp1548275.shtml |website=新浪科技 |publisher=cnbeta |accessdate=2018-11-05 |date=2017-10-16 | archive-url=https://web.archive.org/web/20181111084853/http://tech.sina.com.cn/roll/2017-10-16/doc-ifymviyp1548275.shtml |archive-date=2018-11-11 | dead-url=no}}</ref>。

== 外貌 ==
楼楼的[[家貓品種列表|品种]]是中华田园猫，体形肥胖，体重达7.7公斤。毛色总体是橘色和白色相间，其身体毛发为白色，鼻子周围和尾巴的毛发为橘色，头顶的毛发则是橘白相间。楼楼虽然外貌并不是非常出众，但经常会做出一些魔性的表情，因此吸引了多达55万粉丝，而楼楼的这些魔性的表情也被网友制作成了表情包<ref>{{cite news |title=及时热点：猫界网红“楼楼”因病去世 |url=https://www.iqiyi.com/v_19rre53ekc.html |accessdate=2018-11-11 |work=[[新京报|新京报]]动新闻 |date=2017-10-16}}</ref>。

== 生平 ==
{{external media|video1=[http://n.miaopai.com/media/8-t2Nwj4c0O-Yg8bV9nric~AgLE_ 楼楼走红的洗澡视频]}}
楼楼一开始是一只流浪猫，后来被楼板娘及其丈夫楼爸所收养。在楼板娘的饲养下，楼楼逐渐长胖<ref>{{cite web |author1=微漫画 |title=感觉身体被掏空？云养猫莫跟风，真养猫需谨慎！ |url=https://wemedia.ifeng.com/34869819/wemedia.shtml |website=大风号 |accessdate=2018-11-11 |date=2017-10-27 |archive-url=https://web.archive.org/web/20181111084950/https://wemedia.ifeng.com/34869819/wemedia.shtml | archive-date=2018-11-11 | dead-url=no}}</ref>，而楼板娘平时也经常给楼楼和收养的另外一只黑色的猫多多拍摄日常生活的照片和视频，并分享到新浪微博上。起初，这些照片和视频并没有引起很大的关注，直到2017年7月23日，楼板娘在新浪微博上上传了楼楼洗澡的视频，在视频中，楼楼的主人在给楼楼洗澡，但是楼楼面对[[蓮蓬頭|蓮蓬頭]]的水一直躲避，蜷缩在浴室角落里，并用爪子抓绕浴室玻璃门，企图逃跑，还时不时发出叫声。这段视频中楼楼可爱的表现吸引了大量网友的注意，此微博也有一万人次的点赞、评论和转发，楼楼也因此走红。随后网友也从楼板娘的微博里发现了楼楼照片中魔性的表情，并把这些照片制作成表情包，这些表情包也受到很多网友的欢迎并被广泛使用<ref>{{cite web |author1=贾大方 |title=你连表情包里猫的名字都不知道，还敢说爱猫？ |url=http://tech.ifeng.com/a/20171019/44721665_0.shtml |website=凤凰网科技 |publisher=PingWest中文网 |accessdate=2018-11-05 |date=2017-10-19 | archive-url=https://web.archive.org/web/20181111084934/http://tech.ifeng.com/a/20171019/44721665_0.shtml | archive-date=2018-11-11 | dead-url=no }}</ref>。

好景不长，2017年10月11日晚上11点左右，楼楼突然出现了呕吐、下肢瘫痪和呼吸急促的病症。楼板娘立刻将楼楼送到一家宠物医院就诊，拍了[[X光|X光]]片后，楼楼被诊断出患有肥厚性心肌病、肺水肿和血栓，情况非常严重，只能通过[[吸氧|吸氧]]的方式暂时减轻一下楼楼的呼吸困难问题。同时，楼板娘也把楼楼病重的消息发布在新浪微博上，引发了大量网友的关注，大量网友对楼楼的病情表示担忧，一些在养宠物方面有经验的网友也纷纷出谋献策。[[北京|北京]]的[[兽医|兽医]]姚海峰医生提出了利用[[外科手术|外科手术]]移除血栓的治疗方式，但由于楼楼和楼板娘一家居住在[[上海|上海]]，距离医疗技术较为完备的北京的兽医院路途遥远，所以要转院到北京不太可行。楼爸又考虑到外科手术风险较大，便于13日中午转院到上海当地的蓝石宠物医院进行药物治疗<ref name="我们的表情包没了" />。14日0时25分，楼楼因为病情恶化医治无效，楼板娘和楼爸决定让楼楼安乐死<ref>{{cite web |title=网红猫楼楼心脏病突发离世 因表情包俘获无数云吸猫网友 |url=https://www.guancha.cn/life/2017_10_14_430842.shtml |website=观察者网 |accessdate=2018-11-05 |date=2017-10-14 |archive-url=https://web.archive.org/web/20180425114615/http://www.guancha.cn/life/2017_10_14_430842.shtml# |archive-date=2018-04-25 |dead-url=no }}</ref>。

== 影响与评价 ==
楼楼在生前因为其表情包而受到很多网友的欢迎，这次去世，也使得很多网友沉痛哀悼。有些网友之前经常使用楼楼的表情包，但却不知到表情包中的猫就叫楼楼，有些网友则是刚认识楼楼不久，有些网友则是在很久之前便已经认识楼楼了，是楼楼的忠实粉丝，这些网友都对楼楼的去世表示可惜，不少网友还为此伤心落泪好几天。很多网友也留言表示对楼楼的想念，向楼楼道别，祝福楼楼在[[天堂|天堂]]过得开心。有网友表示难以接受楼楼的去世，而且楼楼的表情包再也不能更新了<ref>{{cite web |author1=Zhang Xingjian |title=Death of celebrity cat saddens the internet |url=http://www.chinadaily.com.cn/life/2017-10/16/content_33322847.htm |website=[[中国日报|China Daily]] |accessdate=2018-11-11 |language=en |date=2017-10-16 |archive-url=https://web.archive.org/web/20180402094428/http://www.chinadaily.com.cn/life/2017-10/16/content_33322847.htm# |archive-date=2018-04-02 |dead-url=no }}</ref>；有网友则称楼楼只是厌倦了无聊的[[地球|地球]]生活，想回到自己的喵星而已。不过也有少数网友表示楼楼仅仅是一只猫，不应该过度炒作<ref>{{cite web |author1=胖编怪聊 |title=这只猫去世，近百万网友每天以泪洗面 |url=http://news.163.com/17/1019/07/D13H0I01000181BT.html |website=网易新闻 |accessdate=2018-11-05 |date=2017-10-19 |archive-url=https://web.archive.org/web/20171127192028/http://news.163.com/17/1019/07/D13H0I01000181BT.html# |archive-date=2017-11-27 |dead-url=no }}</ref>。

楼楼的死亡除了引起大量网友的惋惜之外，也有一些兽医对楼楼这次患病进行的分析，总结出肥胖是导致楼楼死亡的主要原因，因为肥胖易引发心脏疾病，且平时难以察觉出病症，这也是楼楼此次发病没有预兆的原因<ref>{{cite web |title=网红猫楼楼去世 56万粉丝为它哀悼！警惕：总想把猫养胖 却不知道“胖猫胖狗命难长” |url=http://news.chinaxiaokang.com/dujiazhuangao/2017/1014/265670.html |website=中国小康网 |accessdate=2018-11-05 |date=2017-10-14 |archive-url=https://web.archive.org/web/20181111084948/http://news.chinaxiaokang.com/dujiazhuangao/2017/1014/265670.html |archive-date=2018-11-11 | dead-url=no}}</ref>。一些有经验的宠物饲养者也建议要定期对宠物进行体检，平时也要收集附近的宠物医院的资讯，遇到突发情况时才能够及时找到医院就诊<ref name="让张含韵送赵丽颖祝福还被喷的网红小猫楼楼因胖而死">{{cite web |title=让张含韵送赵丽颖祝福还被喷的网红小猫"楼楼"因胖而死？吸猫虽好更要爱猫，"楼爸"这些反思你一定也要知道！ |url=http://www.haibao.com/notes/article/2472825.htm |website=海报时尚网 |accessdate=2018-11-05 |date=2017-10-17 |archive-url=https://web.archive.org/web/20171018011038/http://www.haibao.com/notes/article/2472825.htm# |archive-date=2017-10-18 |dead-url=no }}</ref>。

楼楼的死亡除了引发了[[医学|医学]]上的思考外，也引发了对于死亡的[[哲学|哲学]]思考，湖南红网新媒体集团上的编辑称要坦然从容地接受楼楼的离世，因为生老病死是自然界的规律<ref>{{cite web |author1=谢华丽 |title=网红猫楼楼去世是一堂死亡课 |url=http://news.163.com/17/1016/00/D0R19QPK000187VG.html |website=网易新闻 |publisher=红网(长沙) |accessdate=2018-11-05 |date=2017-10-16 |archive-url=https://web.archive.org/web/20171209114945/http://news.163.com/17/1016/00/D0R19QPK000187VG.html# |archive-date=2017-12-09 |dead-url=no }}</ref>。
