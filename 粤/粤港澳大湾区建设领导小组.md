'''粤港澳大湾区建设领导小组'''，是[[中华人民共和国国务院|中华人民共和国国务院]]于2018设立的[[国务院议事协调机构|国务院议事协调机构]]，负责领导[[粤港澳大湾区|粤港澳大湾区]]建设。

==沿革==
2017年初，[[粤港澳大湾区|粤港澳大湾区]]第一次写入[[李克强|李克强]]总理的《[[政府工作报告|政府工作报告]]》。2017年底，[[中央经济工作会议|中央经济工作会议]]将粤港澳大湾区列入2018年重点工作。2018年全国[[两会|两会]]上，粤港澳大湾区建设被提升至国家发展战略层面<ref name=wyelyb/><ref name=ppelyb/>。同时，全国两会期间，[[农工民主党|农工民主党]]中央在向[[第十三届全国政协|第十三届全国政协]]一次会议提交的提案《关于构建粤港澳大湾区整体协同发展模式的建议》中，建议由中央统筹设立粤港澳大湾区联合管理机构<ref>{{cite web |url =http://finance.sina.com.cn/china/gncj/2018-08-15/doc-ihhtfwqr4474497.shtml  |title =粤港澳大湾区领导小组会议召开 港澳特首任小组成员  |author =  |date = 2018-08-15 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。2018年5月20日至22日，[[中共中央政治局常委|中共中央政治局常委]]、[[中华人民共和国国务院副总理|国务院副总理]][[韩正|韩正]]赴[[广东省|广东省]]调研自贸试验区建设和深化粤港澳合作等工作<ref>{{cite web |url = http://finance.ifeng.com/a/20180815/16450778_0.shtml |title = 粤港澳大湾区建设领导小组正式亮相，意味着什么？ |author =  |date =2018-08-15 |website =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。2018年6月底，韩正在北京分别会见[[香港行政长官|香港行政长官]][[林郑月娥|林郑月娥]]、[[澳门行政长官|澳门行政长官]][[崔世安|崔世安]]，就《粤港澳大湾区发展规划纲要》听取意见和建议<ref name=wyelyb/><ref name=ppelyb/>。

2018年，为推动粤港澳大湾区规划，[[中华人民共和国国务院|国务院]]成立高层统筹决策和实施的“粤港澳大湾区建设领导小组”，这也是首次有[[香港特别行政区行政长官|香港特别行政区行政长官]]和[[澳门特别行政区行政长官|澳门特别行政区行政长官]]参与中央高层架构的工作。2018年8月15日，粤港澳大湾区建设领导小组第一次全体会议在[[北京|北京]]举行，标志着该领导小组正式运行<ref name=wyelyb>{{cite web |url =  http://money.163.com/18/0814/13/DP63177G002580S6.html |title = 粤港澳大湾区建设领导小组将开首次会议 韩正任组长 |author =  |date = 2018-08-14 |website = 网易  |language =  |accessdate =  |quote =  }}</ref><ref name=ppelyb>{{cite web |url = https://www.thepaper.cn/newsDetail_forward_2346856 |title = 粤港澳大湾区建设领导小组首次会议明日举行，港澳特首参会  |author =  |date = 2018-08-14  |publisher = 澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。
