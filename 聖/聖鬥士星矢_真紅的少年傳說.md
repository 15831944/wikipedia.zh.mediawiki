「'''聖鬥士星矢 真紅的少年傳說'''」是於1988年7月23日上映的「[[聖鬥士星矢|聖鬥士星矢]]」系列的動畫電影第三部。

同時上映的作品有『[[魁!!男塾|魁!!男塾]]』<ref name="booklet">DVD『聖鬥士星矢 THE MOVIE BOX』（[[東映動畫|東映動畫]]、2004年）的內附小冊子的「作品解説」。</ref>。

== 解說 ==
本作沒有延續在「[[東映|東映]]漫畫祭」上映的傳統，而是作為「[[週刊少年JUMP|週刊少年JUMP]]創刊20週年」紀念計劃而上映的作品。上映時間比起前兩作的45分鐘大幅增加至1.5倍的70分鐘，也是「聖鬥士星矢」的四齣劇場版裡最大熱的作品。作畫監督[[荒木伸吾|荒木伸吾]]親自操刀設計了「太陽神 亞伯」的人物形象，以及在原著和電視動畫的十二宮篇裡殞命的[[黃金聖鬥士|黃金聖鬥士]]們的重新登場，都成為粉絲之間的話題<ref name="booklet" />。而且本作比電視動畫系列的海皇篇更早安排了[[天馬座_星矢|星矢]]、紫龍和冰河穿上黃金聖衣的場面。

本作和前作「[[聖鬥士星矢_眾神的激戰|眾神的激戰]]」，雖是完全原創的故事但卻與原作漫畫「聖鬥士星矢」的世界觀沒有產生衝突，受到粉絲們的好評。導演山內重保的執導功力受到認可，在之後的OVA「[[聖鬥士星矢#冥王篇|冥王哈迪斯十二宮篇]]」裡也被起用為導演。

本作也是目前為止所有劇場版中（不算「[[聖鬥士星矢_天界篇_序奏～overture～|天界篇]]」）唯一沒有使用電視動畫的主題歌，而是採用了[[當山瞳|當山瞳]]和Oren Waters合唱的「'''{{lang|ja|YOU ARE MY REASON TO BE ～愛は瞳の中に～}}'''」為主題歌的劇場版。歌曲風格抒情，有別於一貫的熱血風格。

== 故事簡介 ==
[[雅典娜|雅典娜]]的哥哥，傳說中的[[太陽神|太陽神]][[福玻斯|福玻斯·亞伯]]（フォェボス アベル）復活，受眾神之命在地上世界引發大洪水來達到肅清人類的企圖。身為雅典娜的紗織打算阻止亞伯，反被亞伯殺死，靈魂墜入死之國度。為了拯救雅典娜，[[天馬座_星矢|星矢]]等人潛入了聖域的禁地「Dignity Hill」，面前卻被在十二宮之戰中戰死的撒加等[[黃金聖鬥士|黃金聖鬥士]]以及從[[希臘神話|神話時代]]開始就守護亞伯的「日冕聖鬥士（Corona Saint）」所阻擋。面對以往從未遇到過的強敵，星矢等人最大的激戰展開！

:如今在現代復活，企圖消滅污穢的人類並支配大地。
:能在地球各處引起[[火山|火山]]噴發和[[洪水|洪水]]爆發等天地異變的現象，又能將靈魂向「極樂淨土」引導。
:其強大的小宇宙所造就的防禦壁能反彈任何攻擊，此外亦能夠讀取人類甚至神的思考。
:最後被穿上[[射手座|射手座]]黃金聖衣的[[天馬座_星矢|星矢]]用黃金箭射穿左胸，接著被倒塌的日冕神殿所吞噬。
: 在之後的劇場版「[[聖鬥士星矢_最終聖戰的戰士們|最終聖戰的戰士們]]」中以亡靈的形態登場。燃燒跟生前一樣的黃金小宇宙，在地上引發火山噴發造成破壞。星矢對他屈服於[[聖鬥士星矢_最終聖戰的戰士們#登場人物|路西法]]之下感到震驚。

=== 日冕聖鬥士（Corona Saint） ===
不從屬於「[[黃金聖鬥士|黃金]]」、「[[聖鬥士星矢角色列表#白銀聖鬥士|白銀]]」、「[[聖鬥士星矢角色列表#青銅聖鬥士|青銅]]」等任何一個聖鬥士階級，從[[希臘神話|神話時代]]開始就被選中仕奉[[太陽神|太陽神]]的聖鬥士，亦象徵著太陽的力量。

雖只有三人，但他們的實力卻凌駕於[[黃金聖鬥士|黃金聖鬥士]]之上。他們身穿的聖衣被稱為「日冕聖衣」，不是由[[雅典娜|雅典娜]]而是由太陽神亞伯所賜予，擁有無論雅典娜的聖鬥士怎樣攻擊都能夠抵擋的防禦力<ref name="booklet" />。此外在總數是八十八人的聖鬥士裡，黃金十二人、白銀二十四人、青銅四十八人，在三個階級以外餘下還有四名聖鬥士，有說法是日冕聖鬥士是存在於這四名聖鬥士之中<ref name="booklet" />。

; '''[[船底座|龍骨座]]（Carina） [[阿特拉斯|阿特拉斯]]'''（Atlas）（聲：[[神谷明|神谷明]]／香港：[[馮永和|馮永和]]／台灣：[[劉傑|劉傑]]） 
* 必殺技：'''烈燄日輪'''（Burning Corona）
: 日冕聖鬥士的領袖。
:跋扈而強大的實力者，尊亞伯為最高神並對他宣誓絕對的信賴與忠誠。
:當亞伯將雅典娜送往黃泉之國時，[[黃金聖鬥士|黃金聖鬥士]]的[[黃金聖鬥士#水瓶座|卡妙]]和[[黃金聖鬥士#摩羯座|修羅]]向亞伯發難，被他連同貝烈尼凱和薩歐的聯合攻擊鎮壓。
:使用從對手體內燃燒爆發的「烈燄日輪」將紫龍、瞬和一輝打倒，之後將五名[[聖鬥士星矢角色列表#青銅聖鬥士|青銅聖鬥士]]全部打敗，一度向亞伯報告勝利，但被穿上[[天秤座|天秤座]]和[[水瓶座|水瓶座]]的黃金聖衣並重新站起來的紫龍和冰河的一擊重創，再被同樣穿上[[射手座|射手座]]黃金聖衣的[[天馬座_星矢|星矢]]以「天馬彗星拳」殺死。

; '''[[后髮座|后髮座]]（Coma<ref>「[[后髮座|后髮座]]」的英文應該是「Coma Berenices」，但本作中是「Coma」。</ref>） [[貝勒尼基二世|貝雷尼凱]]'''（Berenike）（聲：[[古川登志夫|古川登志夫]]／台灣：[[陳彥鈞|陳彥鈞]]）
* 必殺技：'''奪命金髮絲'''（Golden Death Hair）
: 以利用自身的頭髮將對手的動作封鎖並切斷的絕技“奪命金髮絲”封住卡妙和修羅的動作，再聯合亞多拉斯和薩歐的合擊將兩人打倒。
:身穿太陽神亞伯賜予的日冕聖衣，令冰河的「鉆石星塵」和「極光雷電攻擊」的凍氣都不起作用，但最後被小宇宙爆發的冰河以“曙光女神之寬恕”一擊粉碎。

; '''[[天猫座|山貓座]]（Lynx） 賈奧'''（Jaow）（聲：[[森功至|森功至]]／台灣：[[宋克軍|宋克軍]]）
* 必殺技：'''閃光地獄爪'''（Shining Hell Claw）
: 動作如野獸般敏捷，使用兇猛的拳法。
:三個日冕聖鬥士中只有他身上沒有穿正裝（聖衣外加短袍）<ref name="booklet" />。
:在黃金聖鬥士卡妙和修羅被貝烈尼凱的「奪命金髮絲」封住動作後，以自己的絕招「閃光地獄爪」以及亞多拉斯的「烈燄日輪」的二段攻擊將卡妙和修羅打倒。
:在[[雙子座|雙子座]]的[[黃金聖鬥士#雙子座|撒加]]為了提高星矢的小宇宙不惜與星矢交鋒因而受傷之後出現，撒加為星矢開路與他對峙。
:最後被撒加打出的自殺式「銀河爆裂拳」打倒，與撒加同歸於盡。
