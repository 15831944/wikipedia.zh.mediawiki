本作於2011年以「聖鬥士星矢誕生25周年纪念作」發佈製作消息，2013年10月公佈標題，距離上一部劇場版作品《[[聖鬥士星矢_天界篇_序奏～overture～|聖鬥士星矢 天界篇 序奏～overture～]]》上映相隔了10年以上。本作品是原作者[[車田正美|車田正美]]出道四十週年紀念電影，由原作者親自命名副標題。本作品更是首次採用全[[CG|CG]]製作的系列劇場版作品，劇情則是以在TV版大受歡迎的十二宮篇改編的故事。但人物的[[3D|3D]]造型和原著有較大差異。主角青銅聖鬥士們的造型無甚大變化，黃金聖鬥士們的造型則有較大改動以接近其本身國籍的外觀（髮色、膚色和瞳色的變動，添加了鬍子和眼鏡等等）。

本作的主角，青銅聖鬥士們的[[聲優|聲優]]陣容沒有沿用[[聖鬥士星矢|舊TV版]]的第一代聲優陣、[[聖鬥士星矢#冥王篇|冥王篇]]的第二代聲優陣、[[聖鬥士星矢Ω|Ω]]的第三代聲優陣中的班底，而是起用以[[石川界人|石川界人]]為首的一系列新生代聲優，以及由[[山寺宏一|山寺宏一]]、[[平田廣明|平田廣明]]等實力聲優組成的[[黃金聖鬥士|黃金聖鬥士]]聲優陣容的第四代聲優陣。

而根據據原作者[[車田正美|車田正美]]所說<ref name="message">[http://saintseiya2014.com/comment.html 車田正美先生からのコメント]</ref>，本作在設定和設定上都刻意偏向漫畫原作的設定；而為了打破過去無法達到的表現效果，所以本作才決定採用全[[CG|CG]]製作。而製作本作的最大目的則是希望能吸引更多人接觸原作，並使新舊的支持者能聯繫起來。

影片讲述了城户沙织（佐佐木彩夏配音）——一个为自己所拥有的神秘力量而困惑的女孩，被[[天馬座|天馬座]][[圣斗士|圣斗士]][[天馬座_星矢|星矢]]（石川界人<ref name="CV4"/>配音）从一个被派来杀死她的刺客手中救下。沙织随后得知她是[[女神|女神]][[雅典娜|雅典娜]]的转世，而星矢则是她的88个圣斗士之一，她的使命是前往圣域并向[[聖鬥士星矢角色列表#教皇（Pope）|教皇]]发动战争<ref name="personalities">{{cite web |url=http://saintseiya2014.com/gl_english.html |title=Introduction |publisher=Crunchyroll |date=2014-02-26 |accessdate=2014-04-07 |archiveurl=https://www.webcitation.org/6OfZCGmEi?url=http://saintseiya2014.com/gl_english.html |archivedate=2014-04-08 |language=en |deadurl=yes }}</ref>。为了从即将到来的死亡威胁下拯救他们的女神雅典娜，星矢和他的青铜圣斗士伙伴[[天龙座|天龙座]]紫龙、[[凤凰座|凤凰座]]一辉、[[白鸟座|白鸟座]]冰河、[[室女座|室女座]]瞬一起，必须在十二个小时内穿越圣域的[[十二宫|十二宫]]圣殿。他们必须面对由十二个黄金圣斗士分别守护每个圣殿<ref name="cast">{{cite web |url=http://www.crunchyroll.com/anime-news/2014/02/26-1/main-voice-cast-for-saint-seiya-2014-cg-film-announced |title=Main Voice Cast for "Saint Seiya" 2014 CG Film Announced |publisher=Crunchyroll |date=2014-02-26 |accessdate=2014-04-06|language=en}}</ref>。

《圣域传说》是导演佐藤的第一部CGI的作品，他认为这是一个挑战。影片将故事的背景放在了现代，人物设计风格与车田正美的画风不同，在某些时候CGI的质感被设计的风格类似[[视频游戏|视频游戏]]， 而在某些时候则类似电影风格。除了CGI，人物造型都依稀基于佐藤的想法。原作漫画的作者车田正美担任影片的电影监制。佐藤认为，通过与车田的共同工作， 他们将避免删除“原作漫画的灵魂” 。佐藤还希望CGI来可以给主角们带来经由他们各自的个性所产生的面部表情。他表示因为不是机器人动画，所以尽量避免画面造成类似机器的感觉，并说是以 “在电视上绝对没办法看得到的表现方式以120%的心血作成的映像”。为了突出角色们的充满活力，而在制作中使用了模型制作而不采用真人[[动作捕捉|动作捕捉]]，以[[传统动画|传统动画]]般的“手工制作”的方式来完成CG动画制作。<ref name="060s"/><ref>{{cite web|url=http://news.walkerplus.com/article/45078/|title=“タイバニ”のさとうけいいち監督が語る！新『聖闘士星矢』の気になる内容は？|accessdate=April 6, 2014|language=ja}}</ref><ref>{{cite news|author=盛夏Veya|title=你能接受吗？《圣斗士》新剧场CG人设|url=http://pcedu.pconline.com.cn/402/4024501.html|accessdate=2014-06-27|newspaper=pconline|date=2013-12-18|language=zh-cn|quote=剧场版的CG人设完全打破了车田正美的画风……日本游戏人物风格接近}}</ref>

== 故事簡介 ==

在遙遠的神話時代，有一群守護著戰鬥女神[[雅典娜|雅典娜]]的戰士。他們正是在邪惡於這個世上蔓延的時候必定會出現的希望的戰士·聖鬥士。他們身穿[[星座|星座]]的[[聖衣|聖衣]]，赤手空拳對抗威脅大地的邪惡勢力，據說他們的拳頭能撕裂天空，腳踢能裂開大地。

然後，經過長久的聖戰，在現代，為自己所擁有的不可思議的力量而煩惱的少女·'''[[聖鬥士星矢角色列表#城戶沙織|城戶沙織]]'''，被告知那種力量名為「[[小宇宙|小宇宙]]」，而她正是率領「聖鬥士」守衛大地的愛與和平的雅典娜的轉世。但是作為聖鬥士大本營的聖域裏的教皇則聲稱沙織是冒充雅典娜的逆賊，更派出刺客追殺她。而將沙織從來襲刺客的襲擊中救出來的，正是守護著她的[[青銅聖鬥士|青銅聖鬥士]]'''[[天馬座_星矢|星矢]]'''。

在戰鬥中決心面對自己命運的沙織，和星矢及其他青銅聖鬥士同伴一起前往聖域。但在聖域，教皇早已設下陷阱，更有聖鬥士之中號稱最強的[[黃金聖鬥士|黃金聖鬥士]]們在等待著他們。星矢能夠將沙織守護到底嗎？年輕聖鬥士們的死鬥如今開始！

*'''聖鬥士 (Saint)'''
:守護著女神[[雅典娜|雅典娜]]的戰士。
:據說不拿武器的他們的拳頭能撕裂天空，腳踢能裂開大地。
:聖鬥士分為三個級別，為守護十二宮，最高級的黃金聖鬥士、中級的白銀聖鬥士、最低級的青銅聖鬥士。
:還有，他們身穿與自己守護星座相對應的聖衣。

*'''聖衣 (Cloth)'''
:聖鬥士所身穿的防具。劇中有著能支援小宇宙的追加設定。
:平常壓縮成被稱作'''聖衣箱(Cloth Box)'''大型箱子並放置在聖域。
:在召喚聖衣箱方面，青銅聖鬥士是以[[金屬掛牌|金屬掛牌]]召喚，黃金聖鬥士則以[[石板|石板]]召喚。
:在小宇宙提升到一定程度時，聖衣會自動生成全罩式頭盔以覆蓋頭部，其中大部份黃金聖衣的頭盔就連眼睛的部份也覆蓋起來。

*'''雅典娜'''
:帶領聖鬥士戰鬥的女神。
:但實際上雅典娜本身並不喜歡戰鬥，而每次向雅典娜有關的戰爭全都是為了守護雅典娜而引起的戰鬥。

*'''小宇宙'''
:在人體內存在的宇宙的能源。
:而聖鬥士能透過讓小宇宙燃燒來發揮意外的破壞力，不過失去所有小宇宙的下場就是死。
:還有，雅典娜的小宇宙有著治療傷勢的特殊力量。

*'''Seven Senses'''
:如果小宇宙是人類的第六感的話，Seven Senses就是第七感。
:達到Seven Senses即是擁有與黃金聖鬥士相同的力量，黃金聖鬥士至少都已達到Seven Senses。

*'''聖域 (Sanctuary)'''
:雅典娜和教皇，以及聖鬥士居住的世界。
:原作設定在希臘但本作則在異世界。有十二宮和教皇之間，雅典娜神殿等建築物在上空。
:由於聖域有結界包裹的關係，所以普通人是無法認知其存在。

*'''十二宮'''
:像守護雅典娜神殿一樣地在螺旋型配置在神殿外圍，黃金聖鬥士的宮殿。
:每一宮內部都因其黃金聖鬥士的喜好而有不同造型。

*'''火時鐘'''
:教皇使用的裝置。能吸收、積蓄雅典娜的小宇宙，一邊計算著小宇宙的存量和時間的經過而點火。
:通過火時鐘也能看到聖域和地上的情況。

== 概要 ==
本作於2011年以「聖鬥士星矢誕生25周年纪念作」發佈製作消息，2013年10月公佈標題，距離上一部劇場版作品《[[聖鬥士星矢_天界篇_序奏～overture～|聖鬥士星矢 天界篇 序奏～overture～]]》上映相隔了10年以上。本作品是原作者[[車田正美|車田正美]]出道四十週年紀念電影，由原作者親自命名副標題。本作品更是首次採用全[[CG|CG]]製作的系列劇場版作品，劇情則是以在TV版大受歡迎的十二宮篇改編的故事。

據原作者[[車田正美|車田正美]]所說<ref name="message">[http://saintseiya2014.com/comment.html 車田正美先生からのコメント]</ref>，本作在設定和設定上都刻意偏向漫畫原作的設定；而為了打破過去無法達到的表現效果，所以本作才決定採用全[[CG|CG]]製作。而製作本作的最大目的則是希望能吸引更多人接觸原作，並使新舊的支持者能聯繫起來。

== 故事簡介 ==
在遙遠的神話時代，有一群守護著戰鬥女神[[雅典娜|雅典娜]]的戰士。他們正是在邪惡於這個世上蔓延的時候必定會出現的希望的戰士·聖鬥士。他們身穿[[星座|星座]]的[[聖衣|聖衣]]，赤手空拳對抗威脅大地的邪惡勢力，據說他們的拳頭能撕裂天空，腳踢能裂開大地。

然後，經過長久的聖戰，在現代，為自己所擁有的不可思議的力量而煩惱的少女·'''[[聖鬥士星矢角色列表#城戶沙織|城戶沙織]]'''，被告知那種力量名為「[[小宇宙|小宇宙]]」，而她正是率領「聖鬥士」守衛大地的愛與和平的雅典娜的轉世。但是作為聖鬥士大本營的聖域裏的教皇則聲稱沙織是冒充雅典娜的逆賊，更派出刺客追殺她。而將沙織從來襲刺客的襲擊中救出來的，正是守護著她的[[青銅聖鬥士|青銅聖鬥士]]'''[[天馬座_星矢|星矢]]'''。

在戰鬥中決心面對自己命運的沙織，和星矢及其他青銅聖鬥士同伴一起前往聖域。但在聖域，教皇早已設下陷阱，更有聖鬥士之中號稱最強的[[黃金聖鬥士|黃金聖鬥士]]們在等待著他們。

星矢能夠將沙織守護到底嗎？年輕聖鬥士們的死鬥如今開始！

== 設定上與原著的異同 ==
* 人物的[[3D|3D]]造型和原著有較大差異。主角青銅聖鬥士們的造型無甚大變化，黃金聖鬥士們的造型則有較大改動以接近其本身國籍的外觀（髮色、膚色和瞳色的變動，添加了鬍子和眼鏡等等）。
* 聖域的位置從希臘改為在現實世界之外的異空間，而十二宮等建築物則建築在[[平流層|平流層]]左右的位置，同時十二宮的每一宮內部亦根據守護者的性格而各有不同風格。
* 十二宮所使用的標誌則在原圖案為基礎下重新設計，但基本已和原圖案沒太多相似點。
* 主角青銅聖鬥士們的[[聲優|聲優]]陣容沒有沿用[[聖鬥士星矢|舊TV版]]、[[聖鬥士星矢#冥王篇|冥王篇]]、[[聖鬥士星矢Ω|Ω]]中的班底，而是起用以[[石川界人|石川界人]]為首的一系列新生代聲優；[[黃金聖鬥士|黃金聖鬥士]]的聲優陣容則由[[山寺宏一|山寺宏一]]、[[平田廣明|平田廣明]]等實力聲優組成。
* 沙織對星矢的稱呼由原著的「星矢」變成「星矢君」<ref name="trailor">[http://www.youtube.com/watch?v=7VZ2HA3NkNA 聖鬥士星矢 聖域傳說 動畫電影預告]</ref>，而且跟原著不同有與星矢等人一起進入十二宮。

== 聖衣設定 ==

=== 青銅聖衣 ===
{{WP:OR}}
* 聖衣的設定拋棄了動畫的設定，基本按照十二宮篇的[[漫畫|漫畫]]設定而設計（然而瞬的肩甲設計却和十二宮篇的[[動畫|動畫]]設定一樣）。
* 聖衣平時會壓縮變形成聖衣箱的狀態保存；而使用者平時亦帶著一個刻著所屬星座圖案的金屬掛牌，只要把掛牌擲到地上便能將聖衣轉送到所在位置。同時掛牌還有打開通往聖域的空間閘門。
* 對比原作只有數十件零件，本作的聖衣分開時平均有超過一百件零件。
* 在聖衣上有一道紋路，會隨著使用者大宇宙強度變化來發出不同程度的光芒。
* 在戰鬥時，聖衣會自動生成頭盔以覆蓋頭部，而且和動畫前部份原創的頭盔不同的是，本作的頭盔為全罩式，只會露出眼睛的部份（一輝的頭盔則是露出嘴巴），頭盔同時額頭位置會刻上所屬星座圖案。

=== 黃金聖衣 ===

* 黃金聖衣的造型大部份為重新製作，只留下少量原設計特徵。
* 為配合角色性格或[[星座|星座]]特點，部份黃金聖衣在金色的基礎上增添了其他的顏色。
* 在聖衣的胸部位置都鑲著一顆刻著所屬星座圖案的寶石。
* 同樣的，在戰鬥時聖衣會自動生成全罩式頭盔以覆蓋頭部，跟青銅聖衣的頭盔不同的是，大部份黃金聖衣的頭盔就連眼睛的部份也覆蓋起來。
