'''聖公會牧愛小學'''（[[英語|英語]]：'''S.K.H. Good Shepherd Primary School'''）是[[香港|香港]]一所津貼全日制男女小學，為[[聖公會|聖公會]]牧愛堂主辦，創校於1963年。

==歷史==
聖公會牧愛小學前身為「牧愛堂初級小學」，創立於1956年，校址位於[[九龍|九龍]][[紅磡|紅磡]][[江西街|江西街]]。1963年[[馬頭圍道|馬頭圍道]]樓高四層的新校舍建成，撥交牧愛堂拓展小學教育，為「牧愛小學下午校」。2008年開辦全日制小一及小二班級，隔年推至小三及小四，2010年正式將下午校轉為全日制津貼小學，並改名「聖公會牧愛小學」。<ref>[http://www.gsps.edu.hk/it-school/php/webcms/public/index.php3?refid=719&mode=published&nocache1488197732 公會牧愛小學：學校簡介]</ref>

== 校訓 ==
「仁愛、勇敢」
 
== 設施 ==
校舍樓高四層，開設十三班，每級有兩班，分別是「仁」班和「勇」班，而有一級會加設「愛」班。校舍設有[[禮堂|禮堂]]、[[操場|操場]]、[[多媒體|多媒體]]教室、中央[[圖書館|圖書館]]、輔導室、社工室、模擬飛行駕駛室、多用途室等。課室設備完善，除一般授課設備外，並設有視像廣播系統、[[投影機|投影機]]、[[電腦|電腦]]、[[平板電腦|平板電腦]]、[[錄影機|錄影機]]和[[電視機|電視機]]，配合現代教育所需。

== 學制 == 
每周上課5天，星期一至四上課七節，星期五上課六節 , 每節45分鐘，有兩個小息，每個15分鐘。星期一至四上課時間為上午八時十五分，下課時間為下午三時二十分 。星期五上課時間為上午八時十五分，下課時間為二時二十分。

== 班級結構 == 
共分六級，有五級兩班，有一級三班，共十三班。

== 輔導教學 == 
輔導教學班逢周一至五（每級不同時段）上午七時四十分至八時十五分。

== 事件 ==
=== 前任校長燒屍案 ===  
1993年7月14日發生牧愛小學校長燒屍案。前任校長（王世明）因串謀盜竊公款在校內殺害一名女書記兼會計（劉容梅）後，將屍體運往[[鰂魚涌公園|鰂魚涌公園]]、[[海澤街|海澤街]]停車場附近某隱蔽處燒屍，毀屍滅跡，其間有途人發現有人向一尼龍袋淋潑易燃液體並報警繼而揭發命案。校長燒屍案於[[香港高等法院|高等法院]]審判後被告最終誤殺罪名成立，入獄9年<ref>[[香港命案列表#1990年代|香港命案列表#1990年代]]及[[1993年香港#7月|1993年香港]]</ref><ref>{{cite web|title=東網光影：小學校長為財自保　93年殺人燒屍|url=http://hk.on.cc/hk/bkn/cnt/news/20170709/bkn-20170709060007829-0709_00822_001.html|work=東網光影 |accessdate=2017-07-09}}</ref>。
