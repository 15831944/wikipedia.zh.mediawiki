'''圣地亚哥-德孔波斯特拉'''<ref>{{Cite GB|id = 17693.5|title = 外语地名汉字译写导则 西班牙语|year = 2009|previous = 1999|promulgate_date = 2009-02-06|effective_date = 2009-08-01|publisher_1 = [[中国国家标准化管理委员会|中国国家标准化管理委员会]]|publisher_2 = [[中华人民共和国国家质量监督检验检疫总局|中华人民共和国国家质量监督检验检疫总局]]}}</ref>（{{lang-es|'''Santiago de Compostela'''}}；意為「繁星原野的聖地亞哥」）是[[西班牙|西班牙]][[加利西亞自治區|加利西亞自治區]]的首府，人口不足十万。相傳[[耶穌十二門徒|耶穌十二門徒]]之一的[[雅各_(西庇太的兒子)|大雅各]]安葬於此，是[[天主教|天主教]][[朝聖|朝聖]]勝地之一。自中世纪以来，前来此地的朝圣者络绎不绝，乃至形成了一条有名的朝圣之路，即[[圣雅各之路|圣雅各之路]]。

'''圣地亚哥-德孔波斯特拉古城'''于1985年被列为[[世界文化遗产|世界文化遗产]]。西班牙境内和[[法国圣雅各伯朝圣之路|法国境内]]的「[[圣雅各之路|圣雅各之路]]」分别于1993年和1998年被列为两项单独的世界文化遗产。

[[圣地亚哥-德孔波斯特拉大学|圣地亚哥-德孔波斯特拉大学]]创建于16世纪，早期以教授[[神学|神学]]为主，如今已成为拥有4万学生的综合大学。

== 历史 ==
[[File:Scallop_weathervane.jpg|225px]]
圣地亚哥－德孔波斯特拉是一个被群山环抱的偏僻小城，西班牙语尊称为'''[[圣地亚哥|圣地亚哥]]'''（聖雅各的西班牙語式發音）。

相传[[雅各_(西庇太的兒子)|大雅各]]曾到西班牙传教7年，死后遗骨葬在西班牙。多年过去之后，他的墓地就无人知晓了。直到813年，一位农夫发现了大雅各的墓地。人们在此建立了一座小教堂，并将此地取名圣地亚哥－德孔波斯特拉。德孔波斯特拉是由拉丁语“Campus Stellae”（星之野）或者“Compositum”（墓地）演化来的。

如今，[[圣地亚哥-德孔波斯特拉主教座堂|圣地亚哥-德孔波斯特拉主教座堂]]是城市的中心，也是聖雅各之路的終點。

2013年7月，[[2013年聖地亞哥-德孔波斯特拉火車事故|發生高鐵事故]]。
