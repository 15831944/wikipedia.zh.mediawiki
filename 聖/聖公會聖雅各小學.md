'''聖公會聖雅各小學'''，為[[香港|香港]][[聖公宗|聖公宗]]小學監理委員會屬下的小學，位於[[香港|香港]][[灣仔|灣仔]][[堅尼地道|堅尼地道]]110號。

== 歷史 ==
該校建於1961年。歷屆校長計有李冬清校長、衛寶欽校長、張浩然校長、麥瑜壁校長。現任校長則為張勇邦。

校址於灣仔堅尼地道110號，為配合聖雅各三方重建計劃，2009年至2013年間曾遷到銅鑼灣聖馬利亞堂中學舊址。

重建後分為三部份，即聖雅各堂（連室內運動場、牧宅等，共15層）、聖雅各小學（共11層）、及聖雅各福群會大樓（包括庇護工場、青少年中心等，共14層），於2013年10月13日舉行祝聖儀式。<ref>http://echo.hkskh.org/issue.aspx?lang=2&id=139</ref>

== 編制 ==
該校開辦小一至小六，每級各四班，由2013/14起陸續增至每級各五班（即最終由24班增至30班）。<ref>http://www.skhsjps.edu.hk/it-school/php/webcms/public/index.php3?refid=807&mode=published&nocache1390654969</ref>全體教職員共有49人，包括總校長，副校長1人，主任8人，外籍[[英語|英語]]老師2人，[[工友|工友]]和[[書記|書記]]，教學助理及[[資訊|資訊]]科技員。具有大學學位者占95%，具碩士或以上學歷者占20%。

除此之外，聖公會聖雅各小學設有家長教師會和[[校友會|校友會]]，作為家長及舊生跟學校溝通的橋樑。現任家教會主席為程嘉賢女士，而校友會主席則是梁劍邦先生。

== 設施 ==
該校設施包括課室30個、[[禮堂|禮堂]]1個、[[操場|操場]]2個、[[圖書館|圖書館]]1個、特別室5個（包括音樂室2個，視藝室，電腦室，活動室）。
