'''聖保祿天主教小學'''（'''St. Paul's Primary Catholic School'''）是[[香港|香港]]的一所女子[[小學|小學]]，位於[[跑馬地|跑馬地]][[黃泥涌道|黃泥涌道]]。直屬中學為[[聖保祿中學|聖保祿中學]]。宗教為[[天主教|天主教]]。該校校舍被列為[[香港二級歷史建築|香港二級歷史建築]]。

== 學校歷史 ==
1848年，[[沙爾德聖保祿女修會|沙爾德聖保祿女修會]]由法國來香港收養孤兒，後來服務老人。

1927年創立該校，至1960年轉為政府津貼小學。

於2015年中宣佈，學校需要擴建校舍，在靠後門的位置興建多層式的教學大樓，又興建天橋連接[[聖保祿中學|聖保祿中學]]。在工程進行期間，會暫時借用柴灣的一所小學校舍進行教學（由2016年的[[暑假|暑假]]後實施）。

== 校訓 ==
「為一切人，成為一切」（{{Bible|格林多前書|9|22}}）

== 班級編制 ==
全校现分二十四班，每年級四班；從2016-2017學年起實行全日制。不設精英班。
