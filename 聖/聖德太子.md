'''聖德太子'''（{{bd|572年|1月31日|621年|3月20日<ref name="古事記~往古之追慕">[http://miko.org/~uraki/kuon/furu/furu_index1.htm 古事記~往古之追慕]</ref>|catIdx=Shoutoku Taishi}}，即[[敏達天皇|敏達天皇]]元年[[正月初一|正月初一]]－[[推古天皇|推古天皇]]二十九年[[二月廿二|二月廿二]]（也有二十九年[[二月初五|二月初五]]之說，見《[[日本書紀|日本書紀]]》））是[[日本|日本]][[飛鳥時代|飛鳥時代]]男性皇族，[[推古天皇|推古天皇]]在位期間的的政治改革推行者。為[[用明天皇|用明天皇]]的二皇子。母親為[[欽明天皇|欽明天皇]]皇女－穴穗部間人皇女（{{lang|ja|穴穂部間人皇女}}），姑母為[[推古天皇|推古天皇]]。

聖德太子本名'''厩户'''（{{lang|ja|厩戸}}），因相傳他於馬房前出生。但較為有力的說法是在出生地附近有著厩户（{{lang|ja|厩戸}}）這個地名，引此而取名。別名'''豐聰耳'''（{{lang|ja|豊聡耳}}）、'''上宮王'''（{{lang|ja|上宮王}}）。 《[[古事記|古事記]]》中記載為'''上宮之厩戶豐聰耳命'''（{{lang|ja|上宮之厩戸豊聡耳命}}）。 《[[日本書紀|日本書紀]]》中亦有《厩戸皇子》一條記載為'''豐耳聰聖德'''（{{lang|ja|豊耳聡聖徳}}）、'''豐聰耳法大王'''（{{lang|ja|豊聡耳法大王}}）、'''法主王'''（{{lang|ja|法主王}}）。聖德太子這個名稱雖於平安時代開始被廣泛的使用而為普遍性的稱呼，但因為是後世添上的尊稱（諡名）的緣由，近年日本有部份教科書中將稱號變更為「厩戶王」（{{lang|ja|厩戸王}}）。詳細參見[[#名稱相關|#名稱相關]]章節。

[[大和時代|大和時代]]豪族政爭，[[用明天皇|用明天皇]]二年[[蘇我馬子|蘇我馬子]]滅[[物部氏|物部氏]]；[[崇峻天皇|崇峻天皇]]五年，崇峻天皇遭暗殺，[[蘇我氏|蘇我氏]]外孫女豐御食炊屋姬即位為[[推古天皇|推古天皇]]，由厩户皇子以皇太子之位輔政。聖德太子輔政後即大力進行改革，並遣使入隋、唐學習中國制度。後推行新政，制定冠位十二階、頒佈憲法十七條、採用曆法、編修國史、使用[[天皇|天皇]]名號、興隆佛教。聖德太子是從1958年到1983年間流通使用的[[日本銀行|日本銀行]]券10,000[[日圓|日圓]]的幣面人物。

聖德太子的確實逝世日期傳疑。有說法指應該為推古天皇29年2月5日。

== 生涯 ==
据《扶桑略记》和《圣德太子传历》<ref name="古事記~往古之追慕"/>，敏达天皇元年壬辰春正月朔初一日（572年1月31日）聖德太子出生。

592年，[[推古天皇|推古天皇]]（33任天皇）即位，据《日本书纪》<ref name="古事記~往古之追慕"/>，推古天皇元年夏四月庚午朔己卯（593年5月15日）立侄儿聖德太子為皇太子，輔佐天皇[[攝政|摄行朝政]]。

596年（推古3年），太子向[[高句麗|高句麗]]的僧[[彗慈|彗慈]]（又名[[惠慈|惠慈]]）拜師。

圣德太子仿照[[中国|中国]]制度，展开改革，即“推古朝改革”，正尊卑、定名分，与中国通交，笃信佛教。他于603年制定[[冠位十二阶|冠位十二阶]]，依冠位区别官位高低，由中央朝廷授予，以整顿朝廷贵族官僚的身份制度，加强朝廷纲纪。翌年推行[[十七条宪法|十七条宪法]]，作为朝廷官僚贵族的道德戒条，条文体现中国[[诸子百家|诸子百家]]及[[汉传佛教|汉传佛教]]思想。四度派遣[[遣隋使|遣隋使]]，607年派[[小野妹子|小野妹子]]<ref name="saisyotenno">ベン・アミー・シロニー(著) Ben‐Ami Shillony(原著)『母なる天皇―女性的君主制の過去・現在・未来』大谷堅志郎 (翻訳)、79頁。 (第2章5『日本で最初の「天皇」』)。</ref>，614年派[[犬上御田锹|犬上御田锹]]、[[矢田部造|矢田部造]]，建邦交，吸收先进文物制度。振兴[[佛教|佛教]]，建[[法隆寺|法隆寺]]、[[四天王寺|四天王寺]]，亦使建筑、雕塑艺术得到进步；圣德太子亦著述《[[法华经义疏|法华经义疏]]》《[[维摩经义疏|维摩经义疏]]》《[[胜鬘经义疏|胜鬘经义疏]]》等[[佛经|佛经]]注释，亦即三经义疏<ref name="baike">{{cite book|author=中国大百科全书出版社汇编|year=1998 |title =[[中国大百科全书|中国大百科全书]]|edition=第一版|series=外国历史（二） |publisher=中国大百科全书出版社 |location=北京 |id= }}，第921页。</ref>，被尊为日本佛教始祖。采用历法，编纂《国记》《天皇记》等史书，功绩颇多。

根據一般說法，聖德太子<ref>實際上，《[[隋書|隋書]]》記載遣使致書的，是一位名[[多利思比孤|多利思北孤]]的倭國君主，而不是聖德太子或[[推古天皇|推古女皇]]。一般人會認為是聖德太子，是根據日本本國史書《[[古事記|古事記]]》與《[[日本書紀|日本書紀]]》的年代，去推算隋大業三年（607年）在位的天皇而來。</ref> 遣使致送的[[國書|國書]]說：「日出處天子，致書日沒處天子，無恙。」实际上将日本与[[中国|中国]]置于平等地位。隋皇帝看了很不高興，對[[鴻臚卿|鴻臚卿]]下令說：「蠻夷書有無禮者，勿復以聞。」<ref>[http://www.cns.hk:89/hb/news/2008/12-24/1501285.shtml 古代國書中的元首稱謂]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，表明隋帝並無視於聖德太子對隋國對等關係的看法。但也有學者認為這封國書並不太像是以知書達禮、敬慕中國而聞名之聖德太子所該有的風格。實際上，《[[隋書|隋書]]》記載這封國書，是一位叫[[多利思比孤|多利思北孤]]的[[倭國|倭國]]君王所致送，從未寫說這封國書是聖德太子所呈。若根據《[[新唐書|新唐書]]》中記載，這封國書很明顯應是[[用明天皇|用明天皇]]（31任天皇）所遞交<ref>《新唐書·倭、日本傳》：「'''用明'''，亦曰目'''多利思比孤'''，直隋[[開皇|開皇]]末，始與中國通。」根據這說法，該王明顯是[[用明天皇|用明天皇]]，致書隋帝而引起不悅的傳聞，可能早於大業三年才是。</ref>，與聖德太子毫無關係。

由於聖德太子篤信佛教，加上受到慧慈法師的恩慧，建立[[法隆寺|法隆寺]]。

据《日本书纪》<ref name="古事記~往古之追慕"/>，推古天皇二十九年辛巳春二月己丑朔癸巳（621年3月3日）圣德太子去世，而据《扶桑略记》<ref name="古事記~往古之追慕"/> 是廿九年辛巳二月廿二日（621年3月20日）。

== 傳說 ==
聖德太子留下諸多傳說，如下：
;出生
:據說聖德太子是在馬廄出生，是由[[觀世音菩薩|觀世音菩薩]]在其母親的口內受胎而誕生，這與[[聖經|聖經]]上對[[基督|基督]]降生的敘述類似。
;豐聰耳
:聖德太子經常要聽取人民的取願，他能一次聽多人說話，在《上宮聖德法王帝説》中的記載是八人，《日本書紀》和《日本現報善惡靈異記》的記載是十人，《聖德太子傳歷》的記載是三十六人，另一種說法是、能依序聽完多人說話之後全部重複一次，也就是過人的記憶力。
;兼知未然
:
;南嶽[[慧思|慧思]]轉生

== 聖德太子著作 ==
* 《三經義疏》（{{lang|ja|三経義疏}}）。
* 《四天王寺緣起》（{{lang|ja|四天王寺縁起}}）。
* 《[[十七條憲法|十七條憲法]]》初出於《[[日本書紀|日本書紀]]》章節裡全文引用段落。
* 《天皇記》、《國記》、《臣連伴造國造百八十部及公民等本記》（{{lang|ja|臣連伴造国造百八十部幷公民等本記}}）於《[[日本書紀|日本書紀]]》中只記載書名，無現存物内容不明。
* 《[[先代舊事本紀|先代舊事本紀]]》。
* 《未來記》。並不特指某一本書，是假託聖德太子所作，以「未來記」之名，在鎌倉時代大量出現的偽書群。

篤信佛教的聖德太子於推古十五年(607年)為《[[妙法蓮華經|妙法蓮華經]]》、《[[勝鬘經|勝鬘經]]》、《[[維摩經|維摩經]]》三經作注，書名《[[三經義疏|三經義疏]]》。
