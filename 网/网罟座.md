== 历史 ==
网罟座是[[法國|法國]]天文學家[[拉卡伊|拉卡伊]]创造的一个[[星座|星座]]。它的名称是为了纪念在[[目镜|目镜]]中加上了精密观测用的[[网|网]]，即[[十字丝|十字丝]]。这个很小的远南天星座最突出的特点是几个有趣的[[双星|双星]]和相对明亮的[[星系|星系]]。

== 有趣的星 ==
* [[网罟座ζ|网罟座ζ]] 1和网罟座ζ 2：这是一对用[[双筒望远镜|双筒望远镜]]可见的[[双星|双星]]。两星[[星等|星等]]分别为+5和+5.5,距離地球39光年。从[[地球|地球]]看相距130[[弧秒|弧秒]]。两星组成成分与[[太阳|太阳]]相似。
* [[网罟座θ|网罟座θ]]星：这是由两颗[[星等|星等]]分别为+6和+8的恒星组成是[[双星系统|双星系统]]。从[[地球|地球]]看两星相距4.1[[弧秒|弧秒]]。

== 深空天体 ==
* [[NGC_1313|NGC 1313]]：这是一个明亮的[[漩涡星系|漩涡星系]]。从[[地球|地球]]看它由一个明亮的南北向的棒和暗而带斑点的[[旋臂|旋臂]]，其[[星等|星等]]为+9.4。
