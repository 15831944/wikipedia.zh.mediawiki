'''網絡-{zh-cn:论坛; zh-tw:論壇; zh-hk:討論區}-'''，常簡稱為'''-{zh-cn:论坛; zh-tw:論壇; zh-hk:討論區}-'''，又稱'''-{zh-cn:讨论区;zh-tw:討論區; zh-hk:論壇}-'''、'''討論版'''等，是種提供在線討論的[[程序|程序]]，或由这些程序建立的以在線討論为主的[[网站|网站]]。由[[Usenet|Usenet]]在1980年之后開始流行，网络论坛大多在技术上代替了早期的电话为基础的'''[[BBS|BBS]]'''服务。虽然在技术上代替了BBS，很多论坛还保有“BBS”的名称。

== 論壇的用處 ==
論壇是供人們作討論的地方。討論題材有很多，例如：[[娛樂|娛樂]]、[[新聞|新聞]]、[[教育|教育]]、[[旅遊|旅遊]]、[[消閒|休閒]]等等。有些論壇設有多項討論題材，包羅萬有，有的則只專注討論某題材。在論壇中，很多使用者還會跟他人分享[[資源|資源]]， 例如：[[音樂|音樂]]、[[短片|短片]]、[[圖片|圖片]]等。有些[[公司|公司]]、[[機構|機構]]又或是[[學校|學校]]，都會設有論壇，供該會的成員作討論之用。

== 论坛的管理 ==
[[网站管理员|站长]]拥有论坛所有权，而論壇依讨论主题分成不同版块，各个版块由[[版主|版主]]管理，一些小型论坛的管理员或超级管理员由站长兼任。为了鼓励会员发言，设有会员积分系统，颁发勋章。一般会员有個性头像和個人签名，而破坏论坛秩序的会员会遭到[[禁止发言|禁止发言]]、[[IP封鎖|IP封鎖]]或取消[[会员|会员]]资格的处罚。有些论坛允許非会员发言，有的必须注册会员才能发言。

在版面发无关内容，版主会移动文章到相关的版面或直接[[删帖|删帖]]，多数论坛不鼓励[[灌水|灌水]]并设有专门的灌水区。

== 论坛程式 ==
論壇程式一般都有公司或組織開發，下載或購買所發佈的程式包，再利用[[FTP|FTP]][[上傳|上傳]]至[[網頁|網頁]][[空間|空間]]，常見的[[論壇|論壇]][[程式|程式]]有下列幾種：
=== [[Discuz!|Discuz!]] ===
[[Discuz!|Discuz!]]是由康盛创想（北京）科技有限公司所開發的[[PHP|PHP]]論壇程式，目前最新版本為X3.2。最新版本推出[[BIG5|BIG5]]（[[繁體中文|繁體]]）、[[GBK|GBK]]（[[簡體中文|簡體]]）、UTF8（[[繁體中文|繁體]]、[[簡體中文|簡體]]），也有人自行修改為[[英文|英文]]版本和[[泰文|泰文]]版本。

2010年8月24日，[[騰訊|騰訊]]正式收購康盛創想，而Discuz!也成為騰訊旗下的產品。

=== [[PHPWind|PHPWind]] ===
[[PHPWind|PHPWind]]是由[[杭州|杭州]]德天-{信息}-技术有限公司所開發的[[PHP|PHP]]論壇程式，之前由於操作較難，而且較少著名論壇使用，所以少為人知。但是自从被[[阿里巴巴|阿里巴巴]]收购之后，研发力量投入增加，日益受到广大站长的喜爱，发展速度飞快，目前最新版本為9.0。

=== [[PhpBB|PhpBB]] ===
[[PhpBB|PhpBB]]於2000年6月17日創立，用家覆蓋全球，支援軟件相當的多元化。由於它簡潔易用，而且完全免費，加上[[程式錯誤|程式錯誤]]也較少，所以深受歡迎。

=== [[vBulletin|vBulletin]] ===
商業論壇程式，無免費版。使用[[PHP|PHP]]指令碼語言編寫，並以[[MySQL|MySQL]]為數據庫。

== 爭議與風險 ==
{{see|內容農場|粗口|色情網站|侵權|網絡犯罪|五毛|電腦病毒|網上欺凌|起底|網絡攻擊}}

當在2008年有名的社群網站如[[Facebook|Facebook]]和[[Twitter|Twitter]]開始在部分國家出現、並且在推廣多年之後，某些國家的網路論壇除了瀏覽人數與會員開始流失之外，後期便出現一些光怪陸離的現象，例如被檢舉音樂檔案侵權、大量出現廣告文章（多數是如色情、[[援交|援交]]、[[性藥|性藥]]相關的廣告文宣，有網友推測這跟管理者的怠慢有關）還有不知其意義的惡意詆毀某人的文章還有亂碼等等。
* 此外在台灣，除了在2012年中曾登上各大新聞版面的[[台灣論壇|台灣論壇]]外，大多數的網路-{zh-cn:论坛; zh-tw:論壇; zh-hk:討論區}-因為在Facebook等社群網站大規模的壓倒性競爭下幾乎都不公開與關閉相關的消息（或者是管理方直接用Facebook來發布關閉-{zh-cn:论坛; zh-tw:論壇; zh-hk:討論區}-的消息）然後就使其無聲無息地消失，目前到2017年1月底仍維持運作的幾乎是極少數。
* 香港也有與台灣類似的情形存在，不過仍有大多數是因為政治上的外在因素所致。
