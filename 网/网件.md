美国'''网件公司'''（{{lang-en|Netgear, Inc.}}，品牌及商標為'''NETGEAR'''），是一家1996年成立、總部位於加州[[聖荷西_(加利福尼亞州)|聖荷西]]的[[電腦網絡|電腦網絡]]設備開發商。公司業務遍及[[北美|北美]]、[[歐洲|歐洲]]、[[大洋洲|澳紐]]、[[中東|中東]]、[[非洲|非洲]]及[[亞太|亞太]]25個國家及地區，產品主要面向家用、商用及網際網絡接入服務業者（ISP），主要銷售渠道包括傳統零售、網絡零售、直接銷售、增值分銷及寬頻服務供應商設備提供。

Netgear的路由器及管理交換器產品多以良好的擴充性、[[開放原始碼|開放原始碼]]的韌體以及衆多[[客制路由器韌體清單|第三方韌體]]（像是[[OpenWRT|OpenWRT]]/[[LEDE|LEDE]]、[[Tomato|Tomato]]等）支援著稱，[[安全漏洞|安全漏洞]]的發現也較爲及時，不過原廠韌體的易用性相較於一些競爭對手的產品（像是華碩的AsusWRT）來說則是存在不足。<ref>[https://thenextweb.com/gadgets/2017/01/31/netgear-vulnerability-router-bypass/ Netgear exploit found in 31 models lets hackers turn your router into botnet]</ref><ref>[http://v2.nat32.com/dg834g.htm Netgear DG834G telnet]</ref><ref>[https://web.archive.org/web/20080119003657/http://www.suburbia.com.au/~dan/ DG834G Embedded Linux]</ref><ref>{{cite web|url=http://www.synacktiv.com/ressources/TCP32764_backdoor_again.pdf |title=How Sercomm saved my Easter: Another backdoor in my router: when Christmas is NOT enough! |publisher=Synacktiv|date=2014 |accessdate=2014-12-25}}</ref><ref>{{cite web|url=http://www.synacktiv.com/ressources/TCP32764_backdoor_again.pdf |title=How Sercomm saved my Easter: Another backdoor in my router: when Christmas is NOT enough! |publisher=Synacktiv|date=2014 |accessdate=2014-12-25}}</ref><ref>{{cite web|url=https://github.com/elvanderb/TCP-32764 |title=TCP-32764 GitHub repository |publisher=GitHub |date= |accessdate=2014-06-02}}</ref>

Netgear也販售[[網路附加儲存|網路附加儲存]]裝置（NAS）ReadyNAS（經由收購Infrant公司整合獲得）<ref name="netgear_acquires_Infrant1">{{cite web |url=http://investor.netgear.com/ReleaseDetail.cfm?ReleaseID=244439 |title=NETGEAR, Inc. - NETGEAR® Completes Acquisition of Infrant |publisher=Investor.netgear.com |date= |accessdate=2011-08-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20110818151844/http://investor.netgear.com/ReleaseDetail.cfm?ReleaseID=244439 |archivedate=2011-08-18 }}</ref><ref name="netgear_acquires_Infrant2">{{cite web |url=http://blogs.pcworld.com/staffblog/archives/004307.html |title=Today @ PC World Netgear Acquires Infrant Technologies |publisher=Blogs.pcworld.com |date=2007-05-03 |accessdate=2011-08-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20110928040226/http://blogs.pcworld.com/staffblog/archives/004307.html |archivedate=2011-09-28 }}</ref>，自2009年起也爲使用者提供 ReadyNAS Vault 線上備份轉發服務，該服務Netgear路由器的使用者也可享受，使用者可將電腦或行動裝置上的指定資料備份至NAS裝置或已連接存儲裝置的路由器上。<ref name="netgear_adds_vault2">{{cite web|last=Ngo |first=Dong |url=http://news.cnet.com/8301-17938_105-10186045-1.html |title=Netgear's ReadyNAS Vault taps into cloud storage | Crave - CNET |publisher=News.cnet.com |date=2009-03-03 |accessdate=2011-08-10}}</ref>
