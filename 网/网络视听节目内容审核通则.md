《'''网络视听节目内容审核通则'''》是2017年6月30日由受[[国家新闻出版广电总局|国家新闻出版广电总局]]监管的[[中国网络视听节目服务协会|中国网络视听节目服务协会]]颁布的用于规范网络视听节目审核的[[行业公约|行业公约]]<ref name="on.cc">{{Cite web |url=http://hk.on.cc/cn/bkn/cnt/news/20170630/bkncn-20170630201150257-0630_05011_001.html |title=內地整頓網絡視聽節目禁播同性戀情節 |accessdate=2017-06-30 |date=2017-06-30 |publisher=on.cc 东网}}</ref>，对视频内容施加了众多的限制，提高了监管的力度。<ref name="thepapercn">{{Cite web |url=http://www.thepaper.cn/newsDetail_forward_1721777 |title=《网络视听节目内容审核通则》发布：应坚持以现实题材为主 |accessdate=2017-06-30 |author=陈宇曦 |date=2017-06-30 |publisher=澎湃新闻}}</ref>该通则宣称是为了提升网络原创节目品质，促进网络视听节目行业健康发展。<ref name="qqtech">{{Cite web |url=http://tech.qq.com/a/20170630/028325.htm |title=网络视听节目内容审核通则：先审后播 政治方向正确 |accessdate=2017-06-30 |author=中国网络视听协会 |date=2017-06-30|publisher=腾讯科技}}</ref>该文件中要求视听节目以人民为导向、以[[社会主义核心价值观|社会主义核心价值观]]为引领，传递正能量，讲好中国故事。<ref name="cnsa">{{Cite web |url=http://www.cnsa.cn/2017/06/30/ARTI0Qg4cp7jtd1Z5o0RnfzM170630.shtml |title=中国网络视听节目服务协会发布 《网络视听节目内容审核通则》 |accessdate=2017-06-30 |author=中国网络视听节目服务协会 |date=2017-06-30 |publisher=中国网络视听节目服务协会 |archiveurl=https://web.archive.org/web/20170630135640/http://www.cnsa.cn/2017/06/30/ARTI0Qg4cp7jtd1Z5o0RnfzM170630.shtml |archivedate=2017-06-30 |deadurl=yes }}</ref>

== 历史 ==
2012年，中国网络视听节目服务协会根据《中国网络视听节目服务自律公约》制定发布了《网络剧、微电影等网络视听节目内容审核通则》，而《网络视听节目内容审核通则》则是在前述文件的基础上，针对现状修订完成。<ref name="cnsa" />2017年6月22日，国家新闻出版广电总局发布规定，要求[[新浪微博|新浪微博]]、[[AcFun|AcFun]]等网站关闭视听节目，新浪微博因此关闭了15分钟以上的视频上传服务并声称会加强与[[人民日报|人民日报]]、[[新华社|新华社]]等中央媒体的内容合作。<ref name="hk01">{{Cite web |url=https://www.hk01.com/%E5%85%A9%E5%B2%B8/101705/%E5%85%A7%E5%9C%B0%E7%B6%B2%E7%B5%A1%E8%A6%96%E8%81%BD%E7%AF%80%E7%9B%AE%E5%AF%A9%E6%9F%A5%E5%86%8D%E6%94%B6%E7%B7%8A-%E7%A6%81%E6%92%AD%E5%90%8C%E6%80%A7%E6%88%80%E5%85%A7%E5%AE%B9%E9%81%AD%E7%B6%B2%E6%B0%91%E5%8F%8D%E5%B0%8D |title=內地網絡視聽節目審查再收緊-禁播同性戀內容遭網民反對 |accessdate=2017-06-30 |author=李俊杰 |date=2017-06-30 |publisher=香港01}}</ref>

== 内容 ==
《通则》中的「网络视听节目」包括网络剧、微电影、网络电影、影视类动画片、纪录片，文艺、娱乐、科技、财经、体育、教育等专业类网络视听节目，以及其他网络原创视听节目，并确立了两大内容审核原则：先审后播和审核到位。<ref name="xinhua">{{Cite web |url=http://news.xinhuanet.com/2017-06/30/c_1121242939.htm |title=互联网视听节目审核通则强调先审后播原则 |accessdate=2017-06-30 |author=史竞男、白瀛 |date=2017-06-30 |publisher=新华社}}</ref><ref name="thepapercn" />《通则》列出了8条完全禁止内容，以及10条须裁剪、删除的内容，包括不符合[[中国特色社会主义|国情]]，宣传恐怖暴力易引发犯罪，渲染淫秽色情，侮辱和诽谤，恶搞英雄，泄露国家机密，有损国家形象，伤害民族、国家团结，宣传[[封建迷信|封建迷信]]和对未成年人有害的内容等，其中《通则》要求禁止炒作明星子女并要求真人秀节目减少明星的参与增加人民群众的参与。<ref name="sina">{{Cite web |url=http://news.dayoo.com/china/201706/30/139997_51431520.htm |title=网络视听节目内容审核通则:恶搞英雄内容须删除 |accessdate=2017-06-30 |date=2017-06-30 |publisher=大洋网 |deadurl=yes |archiveurl=https://web.archive.org/web/20170630085134/http://news.dayoo.com/china/201706/30/139997_51431520.htm |archivedate=2017年6月30日 |df= }}</ref><ref name="cnsa" />《通则》还要求坚持[[双百方针|双百方针]]，努力传播体现当代中国价值观念、体现中华文化精神、反映中国人审美追求，思想性、艺术性、观赏性有机统一的优秀作品。<ref name="xinhua" />

== 争议 ==
《通则》中将[[同性恋|同性恋]]与[[性变态|性变态]]、[[近亲性交|乱伦]]等共同列为「不正常的性关系」并将其放置于「渲染淫秽色情和庸俗低级趣味」段落下，禁止其出现在视听节目中。这个举措引起众多网友和群众的反对。<ref name="on.cc" /><ref name="hk01" />微博认证为「中国首本公益性质同志资讯微杂志」的博主「同志之声」发文表达对《通则》的反对。其认为中华人民共和国「自2001年4月起，已将同性恋从《中国精神病障碍分类与诊断标准》中删除，同性恋如今属于正常的性倾向之一，同性恋性关系、性行为与异性恋性关系、性行为一样，同属正常的关系和行为，不应受到区别对待。」<ref name="tongzhier">{{Cite web |url=http://weibo.com/1664065962/FadtYpl6H |title=同志之声的微博 |accessdate=2017-06-30 |author=同志之声 |date=2017-06-30 |publisher=微博}}</ref><ref name="hk01" />
