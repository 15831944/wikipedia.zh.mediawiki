'''渭源站'''，位于[[中华人民共和国|中华人民共和国]][[甘肃省|甘肃省]][[定西市|定西市]][[渭源县|渭源县]][[锹峪镇|锹峪镇]]<ref>{{Cite web |url=http://gsjjb.gansudaily.com.cn/system/2010/08/31/011674278.shtml |title=兰渝铁路渭源火车站征地拆迁安置工作启动 |access-date=2017-09-29 |archive-url=https://web.archive.org/web/20171009195751/http://gsjjb.gansudaily.com.cn/system/2010/08/31/011674278.shtml |archive-date=2017-10-09 |dead-url=no }}</ref>，是[[兰渝铁路|兰渝铁路]]上的火车站，于2017年9月29日启用<ref>{{Cite web |url=http://news.sina.com.cn/o/2017-09-19/doc-ifykyfwq8372777.shtml |title=兰渝铁路全线通车将提前至9月29日 |access-date=2017-09-29 |archive-url=https://web.archive.org/web/20171009194259/http://news.sina.com.cn/o/2017-09-19/doc-ifykyfwq8372777.shtml |archive-date=2017-10-09 |dead-url=no }}</ref><ref>{{Cite web |url=http://www.chinacqsb.com/html/2017-09/29/content_31267.htm |title=兰渝铁路今起全线开通运营 重庆13小时到兰州 硬座115元卧铺下铺214元 |access-date=2017-09-29 |archive-url=https://web.archive.org/web/20171009195504/http://www.chinacqsb.com/html/2017-09/29/content_31267.htm |archive-date=2017-10-09 |dead-url=yes }}</ref>，由[[兰州铁路局|兰州铁路局]][[陇西车务段|陇西车务段]]管辖。<ref>{{cite news|url=http://mini.eastday.com/a/171004002510438.html|title=兰州铁路局积极应对兰渝铁路首场降雪|publisher=[[经济日报|经济日报]]|date=2017-10-04|accessdate=2017-10-10}}</ref>
