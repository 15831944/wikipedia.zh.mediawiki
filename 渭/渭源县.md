'''渭源县'''是[[中国|中国]][[甘肃省|甘肃省]][[定西市|定西市]]下辖的一个[[县|县]]，位于甘肃东南部，[[渭河|渭河]]上游。面积2065平方公里，2004年人口34万。[[邮政编码|邮政编码]] 748200，县政府驻清源镇。

渭源县经济主要以农业为主，经济相比附近其他行政区域比较落后。全县覆盖有G316和G212国道以及S14渭陇高速，分别通向陇西、西安、天水和兰州，是甘东南地区国道的主要交通干线。此外，貫穿全縣的[[蘭渝鐵路|蘭渝鐵路]]已於2017年9月29日通車，打通了甘肅去往西南地區的便捷通道，亦結束甘東南地區不通鐵路的歷史。

== 当地特产 ==
本地特产有[[党参|党参]]、[[黄芪|黄芪]]、[[当归|当归]]；[[马铃薯|马铃薯]]。

== 未来规划 ==
目前，贯穿境内的[[G75兰海高速|G75兰海高速]]由于财政原因，G75渭源段仍在规划中。G75高速是连接兰州、渭源、四川和重庆的重要交通干线，预计2013年渭源至四川段开工，2015年末竣工。
