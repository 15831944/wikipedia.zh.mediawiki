[[File:DiaoChan.jpg|thumb]]
'''貂蟬'''，是知名的文學[[虛構人物|虛構人物]]，[[東漢|東漢]][[司徒|司徒]][[王允|王允]]的[[歌妓|歌妓]]，[[元代|元代]][[雜曲|雜曲]]《[[:s:錦雲堂暗定連環計|錦雲堂暗定連環計]]》中說她是任昂之女，字'''紅昌'''，為[[山西|山西]]一村姑，早年其事跡大多出現在[[說書|說書]][[話本|話本]]當中，最後由《[[三國演義|三國演義]]》作者[[羅貫中|羅貫中]]整理創作出一個完整的形象，乃[[中國古代四大美女|中國古代四大美女]]的「閉月」。

== 史學上的形跡脈絡 ==
在結集史傳與民間故事於一身的著名長篇歷史[[小說|小說]]《[[三國演義|三國演義]]》中，貂蟬的形象非常膾炙人口。可是，在[[正史|正史]]之中並未有發現關於「貂蟬」這名女性的記載，只有[[呂布|呂布]]和[[董卓|董卓]]侍婢[[私通|私通]]的紀錄而已。

據《[[三國志|三國志]]·呂布傳》提及「卓常使布守中閣，布與卓侍婢私通，恐事發覺，心不自安」，<ref>《[[:s:三國志/卷07|三國志·呂布傳]]》</ref>事件中的侍婢並未明示「貂蟬」之名。但在清人[[平步青|平步青]]的戲曲考證書《[[小栖霞说稗|小栖霞说稗]]》中提到：唐《[[開元占經|開元占經]]》卷三十三曾提及：“熒惑犯須女。占注雲：《漢書通志》：‘[[曹操|曹操]]未得志，先誘董卓，進刁蟬以惑其君。’”<ref>《[[小栖霞说稗|小栖霞说稗]]·斬貂蟬》：近出《曲园杂纂》(卷三十八)《小浮梅槛闲话》：杂剧有《关公月下斩貂蝉》，因《后书》《吕布传》有私与董卓传婢通事，附会成之。复卿曰：「《徐文长集》(卷十七)《奉师季先生书》云：『世所传操闭羽与其嫂于一室，羽遂明烛以达旦，事乃无有。盖到此田地，虽庸人亦做得，不足为羽奇；虽至愚人亦不试以此，以操之智，决所不为也。阳节潘氏，盖亦看《三国志》小说而得之者，如所谓「斩貂蝉」之类，世皆盛传之，乃绝无有，此不可不考也。』曲园似本之天池生。」馪旉曰：「《升庵全集》(卷四十八)：『世传吕布妻貂蝉，史传不载。唐李长吉《吕将军歌》：」榼榼银龟摇白马，傅粉女郎大旗下。」似有其人也。』」鄂按：《开元占经》(卷二十三)《荧惑犯须女占》注：「《汉书通志》：『曹操未得志，先诱董卓进刁蝉以惑其君。』」是蝉固实有其人，特非布传所通之傅婢，亦未为圣帝斩。《汉书通志》不知何人所撰，《隋书经籍志》无之，盖《七录》所未收。罗氏演义易「刁」为「貂」，则不知何本。天池谓秉烛达旦事乌有，足证阳节潘氏荣总论之轻信小说。刁蝉则非无其人。</ref>而《漢書通志》一書早已亡佚，現存的《[[:s:唐開元占經_(四庫全書本)|開元占經]]》注無此條。

=== 貂蟬異說 ===
{{來源請求|近世民間歷史文學考究以為，貂蟬在秦漢兩朝時泛指侍中、中常侍等高階朝廷官員配戴的貂蟬冠朝服裝飾，一些漢代文學經常以貂蟬來隱喻顯赫尊貴的社會地位之身份象徵意義，本意上未必能與形容外貌等義。因而衍生貂蟬真身依男身、女身之別有幾種假設主張說法。

*第一種假設:'''寵臣'''，敵視董卓的士閥為謀誅董卓計畫順利，安排一位士人成為董卓的寵臣，利用取得董卓的信任而給予的權力，將董卓的黨羽兵力一一調離長安城外，讓長安城內只剩下由刺董派士閥控制的兵隊。

*第二種假設:'''[[宦官|寵宦]]'''，王允為首的士閥黨羽為了順利誅剿董卓，利用因為先前朝廷宮變影響枉死大量閹宦導致內廷官員人力不足需要補充之機遇，讓該宦官討好董卓而獲得信任，從而掌握董卓一派黨羽的行動訊息，以便尋得適當機會剷除董卓。

*第三種假設:'''寵妾'''，也可能是受寵侍女，這版本借鑒演義通說，安排一貌美女士成為董卓侍女，掌握董卓起居生活上的細節。

*第四種假設:'''寵眷'''，王允間接利用董卓寵待家眷而大肆濫封高等爵位給家眷子女這一點，讓呂布意識到自身的待遇與信任不如董卓家眷重要，從而感到被冷落疏遠，王允趁機勸服呂布參與誅殺董卓。借鑒漢末名士[[王粲|王粲]]所著《[[英雄記|英雄記]]》對董卓的史述，董卓當權時，其遍封宗族，更為孫女董白進爵渭陽君的加冕儀式盛大，在郿城東起壇台，董卓讓董白乘坐青蓋車，讓都尉、刺史、中郎將以上隨侍左右的記載。

*第五種假設:某位'''身份高貴'''之女，本著貂蟬字義上象徵身份地位，基於某種私人理由犧牲貞操名節，委身敵方以圖報讎，事成後自認這不是值得傳誦之事，故而密而不宣。

一個傳說人物有其故事傳頌後世必有其依據根源之原則，借鑒號稱同為中國文學四大美人的西施、王昭君、楊貴妃皆是中華歷世朝代的政治權力鬥爭下的犧牲產物的共通性，假設真有名為貂蟬之女傑，那麼她極有可能已經在董卓被誅身亡之後，隨之遭遇香消玉殞之悲劇人生，只是人們基於好人終有好結局的心態，故而衍生數種不同身世經歷版本故事予以傳頌。

但是無論是哪種假設說法，都純粹反映當時人們有感於董卓專橫暴政之下，藉集體虛構文學之想像創作，塑造出一名能夠解救苦難萬民的女英傑，及早終結暴政亂世的美談故事。}}

== 《三國演義》中的形象 ==
[[File:Wang_Yun_&_Diao_Chan.JPG|thumb]][[长廊|长廊]]绘画：[[王允|王允]]巧设[[连环计|连环计]]，图中女子为貂蝉]]
在《三國演義》中，貂蟬是[[司徒|司徒]][[王允|王允]]的侍女，後成為義女，長得美貌動人，方二八年華（十六歲），深有報國之心，知悉王允對於董卓的亂政深感無助，於是便獻出自己來對付亂臣賊子。貂蟬以[[美人計|美人計]]同時獲得董卓與呂布二人的眷戀，令本以父子相稱的二人產生嫌隙，後來更成功挑動呂布與王允合作，刺殺董卓。<ref>《[[:s:三國演義/第008回|三國演義·第八回]]》</ref><ref>《[[:s:三國演義/第009回|三國演義·第九回]]》</ref>

[[呂布|呂布]]逃離長安後，小說對於貂蟬的敘述已經絕無僅有。唯第十六回提到呂布妻室時交代了呂布「後娶貂蟬為妾」與及「貂蟬亦無所出（子女）」。第十九回時，呂布正與[[曹操|曹操]]交戰，呂布謀士[[陳宮|陳宮]]勸呂布帶輕兵親截曹軍糧道，呂布狐疑未決，便以此計問於妻妾，貂蟬卻建議呂布「勿輕身自出」，於是呂布便決志死守孤城，不作他計，日夜與妻子嚴氏及貂蟬飲酒解愁，士氣敗壞，將領離心，最後呂布滅於曹操之手。自此之後貂蟬便在《三國演義》中銷聲匿跡。
== 貂蝉故里之争 ==
2011年4月28日，[[甘肃省|甘肃省]][[临洮县|临洮县]]正式对外宣布，已成功注册“貂蝉”和“貂蝉故里”共78个类别商标，[[甘肃|甘肃]][[临洮|临洮]]、[[山西|山西]][[忻州|忻州]]、[[陕西|陕西]][[米脂|米脂]]三省四地的貂蝉故里之争暂告一段落。
== 戲劇 ==
民間的戲劇對貂蟬的故事作一點延伸，並常常提到[[關羽|關羽]]，在《關公月下釋貂蟬》中，貂蟬就被關羽釋放，最後不知所終。而《關公蒙面斬貂蟬》就借[[姜太公|姜太公]]蒙面斬[[妲己|妲己]]為藍本並道出[[紅顏禍水|紅顏禍水]]，關羽害怕其傾國傾城之貌會為天下帶來大亂。明劇《關公與貂蟬》，貂蟬成為關羽之妾。
