'''湾里区'''是[[中国|中国]][[江西省|江西省]][[南昌市|南昌市]]的一个市辖区。总面积为254平方公里，全区人口79145人，男性41917人，女性37228人。<ref>{{Cite web|url=http://www.ncwl.gov.cn/News.shtml?p5=56235|title=南昌湾里_湾里区2016年国民经济和社会发展统计公报|accessdate=2017-12-23|work=www.ncwl.gov.cn}}</ref>

==行政区划==
2013年，湾里区辖2个街道、4个镇：
*街道：[[站前街道|站前街道]]、[[幸福街道|幸福街道]]
*镇：[[招贤镇|招贤镇]]、[[梅岭镇|梅岭镇]]、[[罗亭镇|罗亭镇]]、[[太平镇|太平镇]]

== 教育 ==
公办本科高校：[[江西中医药大学|江西中医药大学]]（湾里校区），民办专科高校：[[南昌职业学院|南昌职业学院]](湾里校区)，公办专科高校：[[南昌教育学院|南昌教育学院]]。中小学35所，在校学生9479人，专任教师710人。<ref>{{Cite web|url=http://www.ncwl.gov.cn/News.shtml?p5=56235|title=南昌湾里_湾里区2016年国民经济和社会发展统计公报|accessdate=2017-12-23|work=www.ncwl.gov.cn}}</ref>

== 旅游 ==
景点：[[洪崖丹井|洪崖丹井]]，[[西山積翠|西山积翠]]，[[梅岭国家森林公园|梅岭]]，狮子峰。<ref>{{Cite web|url=http://www.ncwl.gov.cn/News.shtml?p5=53693|title=南昌湾里_湾里概况|accessdate=2017-12-23|work=www.ncwl.gov.cn}}</ref>

== 历史 ==
1969年11月南昌市革委在湾里设置了“湾里地区战备外迁指挥部”<ref>[http://www.nc.gov.cn/ljnc/xqgk/200911/t20091120_187886.htm 湾里区] {{webarchive|url=https://web.archive.org/web/20130524051002/http://www.nc.gov.cn/ljnc/xqgk/200911/t20091120_187886.htm |date=2013-05-24 }}</ref>
1981年3月湾里区设立<ref>[http://www.xzqh.org/html/show.php?contentid=9730 湾里区历史沿革]</ref><ref>[http://www.gov.cn/test/2007-03/23/content_559119.htm 中华人民共和国行政区划（1981年）]</ref>。
