'''鈽'''（'''Plutonium'''，-{zh-tw:香港譯為'''鈽''',大陸譯為'''鈈''';zh-cn:台湾香港譯為'''钸''';zh-hk:香港台灣譯為'''鈽''';}-）是[[原子序数|原子序数]]94、[[元素符號|元素符號]]為'''Pu'''的[[放射性|放射性]][[超鈾元素|超鈾元素]]。它屬於[[錒系元素|錒系]][[金屬|金屬]]，外表呈銀白色，接觸空氣後容易腐蝕、氧化，在表面生成無光澤的[[二氧化鈽|二氧化鈽]]。鈽有六种[[同素異形體|同素異形體]]和四種[[氧化態|氧化態]]，易和[[碳|碳]]、[[鹵素|鹵素]]、[[氮|氮]]、[[矽|矽]]起化學反應。鈽暴露在潮濕的空氣中時會產生[[氧化物|氧化物]]和[[氫化物|氫化物]]，其體積最大可膨脹70%，屑狀的钚能[[自燃|自燃]]。它也是一种[[辐射病|放射性毒物]]，会於[[骨髓|骨髓]]中集中。因此操作和處理鈽有一定的危險。

鈽是天然存在於自然界中質量最重的原子。它最穩定的[[鈽的同位素|同位素]]是[[鈽-244|鈽-244]]，[[半衰期|半衰期]]約為八千萬年，足夠使鈽以微量存在於自然環境中<ref>{{cite web|url=http://www.nature.com/nature/journal/v234/n5325/abs/234132a0.html|title='Detection of Plutonium-244 in Nature,' Nature 234, 132-134 (19 November 1971)|format=PDF}}</ref>。

鈽最重要的[[同位素|同位素]]是[[鈽-239|鈽-239]]，[[半衰期|半衰期]]為2.41萬年，常被用來製造核子武器。鈽-239和鈽-241都易于裂變，即它們的原子核可以在慢速[[中子溫度|熱中子]]撞擊下產生[[核分裂|核分裂]]，釋出能量、[[伽馬射線|伽馬射線]]以及[[中子輻射|中子輻射]]，從而形成[[核連鎖反應|核連鎖反應]]，並應用在[[核武器|核武器]]與[[核反應爐|核反應爐]]上。

[[鈽-238|鈽-238]]的半衰期為88年，並放出[[α粒子|α粒子]]。它是[[放射性同位素熱電機|放射性同位素熱電機]]的熱量來源，常用於驅動[[太空飛行器|太空船]]。

[[鈽-240|鈽-240]][[自發裂變|自發裂變]]的比率很高，容易造成[[中子通量|中子通量]]激增，因而影響了鈽作為核武及反應器燃料的適用性。

分離鈽同位素的過程成本極高又耗時費力，因此鈽的特定同位素時幾乎都是以特殊反應合成。

1940年，[[格倫·西奧多·西博格|格倫·西奧多·西博格]]和[[埃德溫·麥克米倫|埃德溫·麥克米倫]]首度在[[柏克萊加州大學|柏克萊加州大學]]實驗室，以[[氘|氘]]撞擊[[鈾-238|鈾-238]]而合成鈽元素。麥克米倫將這個新元素取名Pluto（意為[[冥王星|冥王星]]），西博格便開玩笑提議定其元素符號為Pu（音類似英語中表嫌惡時的口語「pew」）。科學家隨後在自然界中發現了微量的鈽。[[二次大戰|二次大戰]]時[[曼哈頓計劃|曼哈頓計劃]]則首度將製造微量鈽元素列為主要任務之一，曼哈頓計劃後來成功研製出第一個原子彈。1945年7月的第一次[[核試驗|核試驗]]「[[三一试验|三一试验]]」，以及第二次、投於[[長崎市原子彈爆炸|長崎市]]的「[[胖子原子彈|胖子原子彈]]」，都使用了鈽製作內核部分。關於鈽元素的[[人體輻射實驗|人體輻射實驗]]研究並在未經[[受試者同意|受試者同意]]之下進行，二次大戰期間及戰後都有數次核試驗相關意外，其中有的甚至造成傷亡。[[核動力|核能發電廠]][[放射性廢料|核廢料]]的清除，以及[[冷戰|冷戰]]期間所打造的核武建設在[[核武裁減|核武裁減]]後的廢用，都延伸出日後[[核武擴散|核武擴散]]以及環境等問題。非陸上核試驗也會釋出殘餘的[[放射性落下灰|原子塵]]，現已依《[[部分禁止核試驗條約|部分禁止核試驗條約]]》明令禁止。

=== 核分裂 ===
[[File:Plutonium_ring.jpg|150px]]鈽元素環狀物。環狀物重5.3公斤，直徑約11公分，足夠製作一枚[[氫彈|核彈]]。它的形狀有助於維繫[[核臨界安全|臨界安全]]。|alt=生鏽的圓柱狀金屬]]

鈽是一種具放射性的[[錒系元素|錒系]]金屬。它的[[電子排布|5f電子]]是離域和定域之間的過渡界線；鈽因此常被認為是最複雜的元素之一<ref name="physicsworld.com">{{cite news|url=http://physicsworld.com/cws/article/news/16443|title=Plutonium is also a superconductor|publisher=PhysicsWeb.org|author=Dumé, Belle|date=2002-11-20}}</ref>。它的[[同位素|同位素]][[鈽-239|鈽-239]]是三個最重要的[[核裂變|易裂變]]同位素之一（另外二者為[[鈾-233|鈾-233]]和[[鈾-235|鈾-235]]）<ref>{{cite web|url = http://www.epa.gov/rpdweb00/glossary/termdef.html#f|title = Fissile Material|work = Radiation Glossary|publisher = United States Environmental Protection Agency|year = 2008|accessdate = 2008-11-23|author = EPA contributors}}</ref>；[[鈽-241|鈽-241]]也具有高度易裂變性。所謂的具「易裂變性」（fissile），是指同位素的[[原子核|原子核]]受到[[中子溫度|慢中子]]撞擊後，能夠產生[[核分裂|核分裂]]，並另釋放出足以支持[[核連鎖反應|核連鎖反應]]、進一步促使原子核分裂的中子。

鈽的同位素的[[質量數|質量數]]範圍從228到247不等。其中質量數低於鈽-244（最穩定的同位素）的同位素，主要的[[放射性|衰變方式]]是[[自發裂變|自發裂變]]和[[α衰變|α衰變]]，衰變產物通常生成鈾（92個[[質子|質子]]）和[[錼|錼]]（93個質子）的同位素（忽略裂變過程產生之二子核的大範圍）。質量數大於鈽-244的同位素則以[[β衰變|β衰變]]為主要衰變方式，衰變產物多為[[鋂|鋂]]（95個質子）。鈽-241是[[衰變鏈|錼衰變系]]的[[衰變鏈|母同位素]]，透過β粒子或電子放射衰變成鋂-241<ref name = "NNDC"/><ref name = "Heiserman1992"/><!-- Note: page 340 -->。

鈽-238和鈽-239是最普遍的人造同位素<ref name = "Heiserman1992"/>。鈽-239是使用鈾（U）和中子（n），並以[[鎿|鎿]]（Np）作為中間體，產生β衰變（β<sup>−</sup>），透過下列反應合成<ref>{{cite journal|first=J. W.|last=Kennedy|coauthors=Seaborg, G. T.; Segrè, E.; Wahl, A. C.|title=Properties of Element 94|journal=Physical Review|year=1946|issue=7–8|pages=555–556|doi=10.1103/PhysRev.70.555|volume=70}}</ref>：
:<math>\mathrm{^{238}_{\ 92}U\ +\ ^{1}_{0}n\ \longrightarrow \ ^{239}_{\ 92}U\ \xrightarrow[23.5 \ min]{\beta^-} \ ^{239}_{\ 93}Np\ \xrightarrow[2.3565 \ d]{\beta^-} \ ^{239}_{\ 94}Pu}</math>

鈾-235裂變中的中子被鈾-238原子核[[中子俘獲|俘獲]]、形成鈾-239；[[β衰變|β衰變]]將一個中子轉變成質子，形成鎿-239（半衰期為2.36日），另一次β衰變則形成鈽-239<ref name = "Greenwood1997p1259">{{harvnb|Greenwood|1997|p = 1259}}</ref>。[[合金管工程|合金管工程]]的學者曾在1940年推導出此反應式。

鈽-238是以[[氘|氘核]]（D，重[[氫|氫]]的原子核）撞擊鈾-238，透過下列反應合成<ref>{{cite journal|first=Glenn T.|last=Seaborg|coauthors=McMillan, E.; Kennedy, J. W.; Wahl, A. C.|title=Radioactive Element 94 from Deuterons on Uranium|journal=Physical Review|year=1946|issue=7–8|pages=366–367|doi=10.1103/PhysRev.69.367|volume=69}}</ref>：

:<math>\mathrm{^{238}_{\ 92}U\ +\ ^{2}_{1}D\ \longrightarrow \ ^{238}_{\ 93}Np\ +\ 2\ ^{1}_{0}n \quad;\quad ^{238}_{\ 93}Np\ \xrightarrow[2.117 \ d]{\beta^-} \ ^{238}_{\ 94}Pu}</math>

在此反應過程中，一個氘核撞擊[[鈾|鈾]]-238，生成兩個中子和鎿-238；[[鎿|鎿]]-238再發射負β粒子、產生自發衰變，形成鈽-238。

== 應用 ==
=== 原子彈 ===
[[File:Nagasakibomb.jpg|thumb]]內含一個鈽核。|alt=原子彈爆炸蕈狀雲（灰色蕈梗和白色蕈頂）的照片。]]
同位素鈽-239是[[核武器|核武器]]中最重要的裂變成份。將[[鈽核|鈽核]]置入[[核武器設計|反射體]]（質量數大的物質的反射層）中，能使逃逸的中子再反射回彈心，減少中子的損失，進而降低鈽達到[[臨界質量|臨界質量]]的標準量：從原需16公斤的鈽，可減少至10公斤，即一個直徑約10公分的球體的量<ref>
{{cite book
|title = Physics for Radiation Protection
|last = Martin
|first = James E.
|year = 2000
|publisher = Wiley-Interscience
|isbn = 0471353736
|edition = 1st
|page = 532
}}</ref>。它的臨界質量約僅有鈾-235的三分之一<ref name = "Heiserman1992"/>。

[[曼哈頓計畫|曼哈頓計畫]]期間製造的「[[胖子原子彈|胖子原子彈]]」型鈽彈，為了達到極高的密度而選擇使用易爆炸、壓縮的鈽，再結合中心中子源，以刺激反應進行、提高反應效率。因此，鈽彈只需6.2公斤鈽便可達到[[爆炸當量|爆炸當量]]，相當於2萬噸的[[三硝基甲苯|三硝基甲苯]]（TNT）<ref name="yield">{{cite book
|first = John
|last = Malik
|url = http://www.fas.org/sgp/othergov/doe/lanl/docs1/00313791.pdf
|title = The Yields of the Hiroshima and Nagasaki Explosions
|publisher = Los Alamos
|id = LA-8819
|date = September 1985
|page = Table VI
|accessdate = 2009-02-15
}}</ref><ref name = "FASdesign">
{{cite web
|url = http://www.fas.org/nuke/intro/nuke/design.htm
|title = Nuclear Weapon Design
|publisher = Federation of American Scientists
|year = 1998
|accessdate = 2008-12-07
|author = FAS contributors
}}</ref>（參見[[核武器設計|核武器設計]]）。在理想假設中，僅僅4公斤的鈽原料（甚至更少），只要搭配複雜的裝配設計，就可製造出一個原子彈<ref name = "FASdesign"/>。

=== 核廢料 ===
{{main|核廢料}}
一般[[輕水反應爐|輕水反應爐]]所產生的核廢料中含有鈽，但為鈽-242、鈽-239和鈽-238的混合物。它的濃度不足以製作成核武器，不過可以改用作一次性的[[混氧燃料|混氧燃料]]（MOX fuel）。在反應爐中以慢速熱中子放射線照射鈽時，會偶然發生[[中子俘獲|中子俘獲]]，而增加鈽-242和鈽-240的量。因此反應進行到第二輪之後，鈽只能和[[快中子反應堆|快中子反應堆]]反應、消耗。在反應器中沒有快中子時（普遍情況下），剩餘的鈽通常會被遺棄，形成壽命長、處理棘手的核廢料成分。

=== 能源與熱源 ===
[[File:Plutonium_pellet.jpg|thumb]]
同位素[[鈽-238|鈽-238]]的半衰期為87.74年<ref>{{cite web|url=http://www.ieer.org/ensec/no-3/puchange.html|title=Science for the Critical Masses: How Plutonium Changes with Time|publisher=Institute for Energy and Environmental Research }}</ref>。它會放出大量[[內能|熱能]]，伴隨著低能的[[伽馬|伽馬]]和[[自發裂變|自發裂變]]射線/粒子<ref name="pumper">{{cite journal
|url = http://arq.lanl.gov/source/orgs/nmt/nmtdo/AQarchive/05spring/heart.html
|journal = Actinide Research Quarterly
|title = From heat sources to heart sources: Los Alamos made material for plutonium-powered pumper
|author = ARQ contributors
|location = Los Alamos (NM)
|publisher = Los Alamos National Laboratory
|issue = 1
|year = 2005
|accessdate = 2009-02-15
}}</ref>。它是[[α粒子|α輻射體]]，同時具有高輻射能及低穿透性，故僅需低度防護措施。單一紙張就可以抵擋鈽-238所放射出的α粒子；同時，每公斤的鈽-238可產生約570瓦特熱能<ref name = "Heiserman1992"/><!-- Note: page 338 --><ref name="pumper"/>。以上特性使鈽-238適宜用於製作[[放射性同位素熱電機|放射性同位素熱電機]]。

== 謠傳 ==
大陸百科網站[[百度百科|百度百科]]曾經謠傳只要約一粒[[方糖|方糖]]大小（約5[[克|克]]）的鈽便足以令全球人類死亡。香港[[無線電視|無線電視]]的[[無線新聞|新聞節目]]曾引用這謠言，結果被香港的[[廣播事務管理局|廣播事務管理局]]警告指報導令觀眾驚恐，未有提供足夠證據確保報導準確<ref>[http://news.sina.com.hk/news/2/1/1/2453496/1.html 錯報核新聞 無線遭警告]{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}，《明報》，2011年10月4日</ref>。<br />
