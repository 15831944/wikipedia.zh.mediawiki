'''陸軍工兵第五四群'''，為[[中華民國國軍|中華民國國軍]]駐守於[[高雄|高雄]]的陸軍部隊，編配在[[陸軍第八軍團指揮部|陸軍第八軍團]]，隊名「神斧部隊」。

== 任務 ==
平時支援作戰區各部隊反恐應援任務之機動阻絕設置，災害發生時依令投入災害救援；戰時則實施作戰區內各港口封、毀港作業，並針對重要灘岸構築灘岸阻絕，以及維護反擊部隊機動路線暢通。

歷年參與[[曾文水庫清淤|曾文水庫清淤]]、[[2014年高雄氣爆事故|2014年高雄氣爆事故]]、[[2016年高雄美濃地震|2016年高雄美濃地震]]及大大小小風災

== 沿革 ==
*1954年，五四群原隸屬於陸軍第一軍團工兵指揮部第五七一工兵總隊。
*1960年，嘉禾案改編為五四戰鬥工兵群，編配陸軍第四三軍，曾分別派駐金門、馬祖、嘉義、澎湖等地。
*1989年，陸精六號案，編配陸軍第八軍團後備指揮部，進駐臺南四分子營區。
*1998年，[[精實案|精實案]]納編[[陸軍花東防衛司令部|陸軍花東防衛司令部]]、[[陸軍澎湖防衛司令部|陸軍澎湖防衛司令部]]及[[陸軍機械化步兵第三三三旅|陸軍步兵第三三三師]]工兵營，編成五四工兵群，為陸軍第八軍團直屬部隊。
*2007年，[[精進案|精進案]]納編軍團各聯兵旅直屬工兵連，編成陸軍工兵第五四群，並調整駐地至台南新化知義營區。

== 隊名由來 ==
*1988年，因執行「佳山案」有功，奉前國防部長[[湯曜明|湯曜明]]先生贈予「神工築起溪澗橋，斧威開闢山沿路」對聯，命名為神斧部隊<ref>[http://www.ydn.com.tw/News/229881 【神斧部隊】54工兵群]，黃雋永，[[青年日報|青年日報]]，中華民國國防部青年日報社，2017/04/03</ref>。

== 隊徽涵意 ==
「神斧」二字代表「神工築起溪澗橋，斧威開闢山沿路」之工兵精神，雙穗代表抗戰建國紀念及發揮抗戰建國精神之意。<br />
隊徽中城堡圖樣，代表工兵是全軍的命脈，必須工兵健全，然後攻則能穿山越水，迅赴事機，勇猛邁進，所向克捷；守則能深溝高壘，無隙可尋，固守堅持，牢不可破。<br />
黃金嘉禾代表陸軍軍隊北伐抗戰、戡亂所建立之豐功偉績，乃「寓兵於民」及「執干戈以衛社稷」之意。<ref>[http://www.ydn.com.tw/News/229881 【神斧部隊】54工兵群]，黃雋永，[[青年日報|青年日報]]，中華民國國防部青年日報社，2017/04/03</ref>。
