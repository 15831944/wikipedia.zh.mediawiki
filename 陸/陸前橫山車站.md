'''陸前橫山車站'''（{{jpn|j=陸前横山駅|hg=りくぜんよこやまえき|rm=Rikuzen-yokoyama eki|f=y}}）是一位於[[日本|日本]][[宮城縣|宮城縣]][[登米市|登米市]]津山町、[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[氣仙沼線|氣仙沼線]]沿線的[[鐵路車站|鐵路車站]]。無人看管的陸前橫山車站是由[[石卷車站|石卷車站]]代管，屬{{link-ja|東日本旅客鐵道仙台支社|東日本旅客鉄道仙台支社|JR東日本仙台支社}}管轄範圍。

== 車站構造 ==
[[盛土|盛土]]上[[側式月台|側式月台]]1面1線的[[地面車站|地面車站]]。[[石卷站|石卷站]]管理的[[無人站|無人站]]。

== 車站週邊 ==
* 登米市公所 津山綜合支所橫山出張所
* 登米市立橫山國民小學
* 橫山郵局
* 登米警察署橫山駐在所
* 橫山不動尊
* 國道45號

== 歷史 ==
* 1977年12月11日 - 車站啟用。
* 1987年4月1日 - [[日本國鐵分割民營化|日本國鐵分割民營化]]，本站管轄由東日本旅客鐵道繼承。
* 2011年（[[平成|平成]]23年）3月11日 - [[東日本大地震|東日本大地震]]，停止營業。
* 2012年（平成24年）8月20日 - 以[[公車捷運系統|BRT]]臨時恢復營運<ref>{{PDFlink|[http://www.jr-sendai.com/wp-content/uploads/2012/08/press20120718-kesennumasen.pdf 気仙沼線における暫定的なサービス提供開始について]}} - 東日本旅客鉄道仙台支社プレスリリース（2012年7月18日）</ref>。車站移至國道上。
