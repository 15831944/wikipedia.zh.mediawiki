'''陸前港車站'''（{{jpn|j=陸前港駅|hg=りくぜんみなとえき|rm=Rikuzen-minato eki|f=y}}）是一位於[[日本|日本]][[宮城縣|宮城縣]][[本吉郡|本吉郡]][[南三陸町|南三陸町]]、[[東日本旅客鐵道|東日本旅客鐵道]][[氣仙沼線|氣仙沼線]]沿線的[[鐵路車站|鐵路車站]]。車站屬[[東日本旅客鐵道仙台支社|JR東日本仙台支社]]管轄範圍，並由[[石卷車站|石卷車站]]代為管理。

== 車站構造 ==
[[盛土|盛土]]上[[側式月台|側式月台]]1面1線的[[地面車站|地面車站]]。[[石卷站|石卷站]]管理的[[無人站|無人站]]。

== 車站週邊 ==
* 歌津港簡易郵局
* 縣道225號泊崎半島線
* 國道45號

== 歷史 ==
* 1977年（[[昭和|昭和]]52年）12月11日 - 車站啟用。
*1987年（昭和62年）4月1日 - [[日本國鐵分割民營化|日本國鐵分割民營化]]，車站管轄劃歸由東日本旅客鐵道繼承。
* 2011年（[[平成|平成]]23年）3月11日 - 東北地方太平洋近海發生[[2011年日本東北地方太平洋近海地震|嚴重地震]]，地震引發的[[海嘯|海嘯]]沖毀靠近海岸的許多鐵路設施，陸前港也是23個被沖毀的鐵路車站之一<ref>{{cite web |url=http://www.sankeibiz.jp/business/news/110401/bsd1104010801004-n1.htm |language=ja |title=<span lang="ja">【東日本大震災】ＪＲ東日本、太平洋沿岸の２３駅流失</span> |publisher=[[產經新聞社|產經新聞社]] |date=2011-04-01 |accessdate=2011-04-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20110413050421/http://www.sankeibiz.jp/business/news/110401/bsd1104010801004-n1.htm |archivedate=2011-04-13 }}</ref>。
