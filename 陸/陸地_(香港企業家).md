'''陸地 [[博士|博士]] [[太平紳士|太平紳士]]'''，(Dr. '''Luk Tei, Lewis'''，{{bd|1960年|}})，[[上海|上海]]出生，是已故[[商人|商人]]「玩具博士」[[陸宗霖|陸宗霖]]的長子。陸地為[[香港|香港]]企業家，亦為香港的[[律師|律師]]，具有[[英格蘭|英格蘭]]及[[威爾斯|威爾斯]]、[[新加坡|新加坡]]及[[澳洲|澳洲]]之律師資格。

== 簡歷 ==
陸地 [[博士|博士]]持有[[倫敦國王學院|倫敦大學國王學院]][[法律|法律]][[學士|學士]]學位，並獲該[[大學|大學]]頒發[[院士|院士]]。

在事業方面，陸地為安陸集團有限公司行政總裁。此外，他亦為[[胡百全律師事務所|胡百全律師事務所]]資深律師。

== 聖殿騎士團 ==
陸地博士除了在政府多個委員會擔任公職之外，亦是[[聖殿騎士團|聖殿騎士團]][[專家|專家]]，對[[聖地|聖地]]素有研究。在2006年《[[達文西密碼|達文西密碼]]》電影引起宗教界的一輪風波，他亦因為他的學問而四-{出}-為[[基督徒|基督徒]]進行證道講談。

== 名下馬匹 ==
陸地博士是[[香港賽馬會|香港賽馬會]]會員，曾經擁有馬匹[[玩具男爵|玩具男爵]]、[[玩具勇士|玩具勇士]]。
