'''陸軍航空特戰指揮部'''，簡稱'''陸軍航特部'''、'''航特部'''、'''航特'''或稱為'''陸航'''，總編制兵力員額約10387人，其所屬各空中騎兵旅編制兵力員額約四千餘人，前身為'''陸軍空降特戰司令部'''，簡稱'''空特部'''、'''空特'''。<ref>{{Cite web |url= http://army.mnd.gov.tw/Publish.aspx?cnid=824&p=9662&Level=2 |title= 陸軍航空特戰指揮部 |author=  |date=  |publisher= 中華民國陸軍司令部 |language= zh-tw |accessdate= 2012-06-17 |quote=  |archive-url= https://web.archive.org/web/20131002225343/http://army.mnd.gov.tw/Publish.aspx?cnid=824&p=9662&Level=2 |archive-date= 2013-10-02 |dead-url= yes }}</ref> <ref>[http://museum.mnd.gov.tw/Publish.aspx?cnid=1458&p=36818 陸軍航空特戰部隊特展] {{webarchive|url=https://web.archive.org/web/20131002225248/http://museum.mnd.gov.tw/Publish.aspx?cnid=1458&p=36818 |date=2013-10-02 }} - 國軍歷史文物館，期間：2005年2月21日~2005年4月30日</ref>

== 沿革 ==

源起於1944年1月1日[[中華民國陸軍|中華民國陸軍]]在[[雲南省_(中華民國)|雲南省]][[昆明市|昆明市]]成立的傘兵第一團，並於1958年3月1日在[[臺灣省|臺灣省]][[桃園市|桃園縣]][[龍潭區_(臺灣)|龍潭]]成立「'''陸軍特種部隊司令'''部」。

1974年4月1日，為統一指揮體系，「'''陸軍空降司令部'''」與「'''陸軍特種作戰司令部'''」併編改稱「陸軍空降特戰司令部」（空特部）。

1979年10月執行「反攻專案」，納編「'''陸軍航空指揮部暨航空訓練中心'''」。

1999年10月1日因應[[精實案|精實案]]，改編為「'''陸軍航空特戰司令部'''」(航特部)。

2006年2月16日因應《國防部組織法》，再改稱為「陸軍航空特戰指揮部」。

==裝備==

陸軍航空特戰指揮部於1992年購買42架[[AH-1眼鏡蛇直升機|AH-1W超級眼鏡蛇攻擊直升機]]，1997年再度購買21架AH-1W，共擁有63架AH-1W，但在1999年7月17日、2008年7月16日<ref>[http://www.appledaily.com.tw/appledaily/article/headline/20080717/30760326/ 軍機撞民宅 痛失2飛官] - 蘋果日報，2008年7月17日</ref>失事折損兩架AH-1W <ref>[http://www.chinanews.com/tw/jsdt/news/2008/12-18/1493256.shtml 臺陸軍直升機迫降4人受傷 近10年失事記錄一覽] - 中國新聞網，統計期間：1998年3月17日~2008年12月18日</ref>。目前擁有61架AH-1W，AH-1W超級眼鏡蛇攻擊直升機分別編成兩個攻擊直升機營，平均編配在第601空中騎兵旅（駐守[[北臺灣|北臺灣]][[桃園市|桃園市]]）與第602空中騎兵旅（駐守[[中臺灣|中臺灣]][[臺中市|臺中市]]）所轄。

陸軍航空特戰指揮部購買30架[[AH-64阿帕契直昇機|AH-64E阿帕契攻擊直昇機]]，已經在2013年11月底全部抵達臺灣，但在2014年4月25日失事折損一架，目前擁有29架。AH-64E阿帕契攻擊直昇機也分別編成兩個攻擊直升機營編配在第601空中騎兵旅，還有編制30架[[UH-60黑鷹直升機|UH-60M黑鷹通用直升機]]。第601空中騎兵旅編制兵力2124人，第602空中騎兵旅編制兵力2148人。
