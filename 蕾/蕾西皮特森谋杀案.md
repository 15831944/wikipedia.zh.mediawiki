'''蕾西·丹尼斯·皮特森''' （{{lang|en| Laci Peterson}}，{{bd|1975年|5月4日|2002年|12月24日}}）<ref name="ctv">[http://www.courttv.com/trials/peterson/keyplayers.html CT-LPet Laci Peterson Case – Key Players" (bio)] Court TV, 2003,   {{webarchive|url=https://web.archive.org/web/20070406105007/http://www.courttv.com/trials/peterson/keyplayers.html|date=April 6, 2007}}</ref> 美国人，怀孕八个月时，被丈夫斯科特·皮特森谋杀，该案一度轰动美国。她死前最后一次被人目击时间是2002年12月24日。她的丈夫斯科特·皮特森，2003年被法庭判决谋杀罪成立，判处死刑。斯科特目前关押在美国[[圣昆丁州立监狱|圣昆丁州立监狱]]中，上诉到[[加利福尼亚州最高法院|加州最高法院]]。

== 早期生活和婚姻 ==
蕾西·丹尼斯·罗出生在[[加州|加州]]的[[莫德斯托_(加利福尼亞州)|莫德斯托]]。 她的父母在高中相识，并于毕业后不久结婚。<ref>{{Cite web|url=http://www.wargs.com/other/peterson.html|title=Ancestry of Conner Peterson|accessdate=2011-06-21|publisher=Wargs.com}}</ref> 他们的第一个孩子，布伦特·罗卡，生于1971年。 蕾西是他们的第二个孩子，生于1975年。 她的父母在她的第一个生日分开了。 后来蕾西父亲再婚，并有另一个女儿艾米。蕾西在她父亲位于[[加利福尼亚州|加利福尼亚州]][[埃斯卡隆_(加利福尼亚州)|埃斯卡隆]]的[[酪農業|乳制品农场]]中长大， 并在[[初级中学|初中]]和[[中学|高中]]担任啦啦队长。毕业于[[Thomas_Downey_High_School|托马斯唐尼高中]]之后，她去了[[加州理工州立大學|加州理工学院的国立大学]]<ref name=":0">{{Cite news|url=http://www.modbee.com/news/local/crime/scott-peterson-case/peterson-disappearance-and-arrest/article3096758.html|title=The woman behind the smile|newspaper=modbee|access-date=2016-12-15}}</ref>主修装饰性[[园艺|园艺]]。<ref>{{Cite book|url=https://books.google.com/books?id=Fs42z2VxgWIC&lpg=PA30&ots=MYTCq-4AMt&dq=laci%20peterson%20%22ornamental%20horticulture%22&pg=PA30#v=onepage&q=laci%20peterson%20%22ornamental%20horticulture%22&f=false|title=Laci: Inside the Laci Peterson Murder|last=Fleeman|first=Michael|date=2007-04-01|publisher=Macmillan|isbn=9781429904339|language=en}}</ref> 同时，她在摩洛湾的一个叫太平洋咖啡馆的小餐厅遇见了斯科特·皮特森。

1996年12月，斯科特和拉奇开始恋爱。 之后他们在1997年8月9日（蕾西毕业前几个月）结婚。



婚姻开始的一两年，他们试图延迟要孩子，但蕾西表示有兴趣开始一个新的家庭。 2000年12月，蕾西正式开始准备[[妊娠|怀孕]]。怀孕花了比预期更长的时间。2002年5月，在预约生育率测试的同时，斯科特和拉奇自然受孕了可以。<ref>{{Cite web|url=http://www.courttv.com/trials/peterson/|title=The Laci Peterson Case|accessdate=2011-06-21|publisher=Courttv.com|archiveurl=https://web.archive.org/web/20090415133554/http://www.courttv.com/trials/peterson/|archivedate=April 15, 2009}}</ref><ref>[http://www.crimenews2000.com/lacipeterson Laci Peterson Case Information]  {{Webarchive|url=https://web.archive.org/web/20040219035329/http://www.crimenews2000.com/lacipeterson|date=February 19, 2004}}</ref><ref>{{Cite web|url=http://www.upi.com/view.cfm?StoryID=20030427-103248-4669r|title=The UPI's April 27 report|accessdate=2011-06-21|date=2003-04-27|publisher=Upi.com}}</ref><ref>[http://transcripts.cnn.com/TRANSCRIPTS/0301/29/lkl.00.html CNN: Larry King Live] Panel Discusses Laci Peterson Case</ref><ref>[http://www.cnn.com/2003/US/West/01/12/missing.woman/index.html CNN.com January 12, 2003 article] Detectives Search near Bay for Missing Pregnant Woman</ref>

== 失踪和发现的尸体 ==
除了斯科特，已知在她失踪前最后两个人和蕾西说话的人是她的妹妹艾米，和她的母亲莎伦。 2002年12月23日晚，艾米当着蕾西的面剪下了斯科特的头发。 之后大约晚上8点30分，莎伦与蕾西通过电话。 不久之后，第二天早上的10点15分，一个邻居发现他家的一只叫麦肯齐的黄金猎犬在社区乱跑，脖子上戴着项圈和一条泥泞的皮带。 邻居把它带回院子里。此时，蕾西1996年的 Land Rover Discovery 的[[运动型多用途车|运动型多用途车辆]] 停在车道上。 她的包，里面有她的钥匙和钱包，被放在房子里的桌子上。

== 轰动美国 ==

本案自2002年圣诞节前夕案发以来，引发全美社会大众的普遍关注，在警方办案期间总计接到超过1万余通提供报案线索的电话，更有多达百位民众自愿加入寻找被害人下落的工作，而协助加州莫德斯多警方的办案单位有90个。凶手斯科特婚外情的女主角安柏弗瑞（Amber Frey）后来被蕾西的家人感动，登上法庭证人席，揭露斯科特的花言巧语以及为人的阴暗面。一度任职模特儿的弗瑞，并且亲笔完成一本厚达210页的书《证人》（Witness）。提供大众从个人角度认识这宗10年来美国社会情节最重大的犯罪故事。《证人》一书的售价为25.95美元，2005年1月7日登上[[亚马逊|亚马逊]]与巴恩斯连锁书局（Barnes Bookstore）的畅销书排行榜第2名。<ref>{{Cite web |url=http://www.epochtimes.com/b5/5/3/18/n854265.htm |title=轰动一时的史考特彼德森杀妻案步入尾声 |date=2005-03-18|accessdate=2017-09-14 |work=大纪元}}</ref>
