'''蕾妮·凱絲琳·齐薇格'''（{{lang-en|Renée Kathleen Zellweger}}，{{bd|1969年|4月25日|catIdx=Z}}），[[美国|美国]][[女演员|女演员]]，生于[[得克萨斯州|得克萨斯州]][[休斯顿|休斯顿]]郊区的卡地镇。她獲得多個讚賞及獎項，包括[[奧斯卡金像獎|奧斯卡金像獎]]、[[英國電影學院獎|英國電影學院獎]]、三個[[金球獎|金球獎]]和三個[[美國演員工會獎|美國演員工會獎]]，其中更獲得[[第76届奥斯卡金像奖|第76届奥斯卡金像奖]][[奥斯卡最佳女配角奖|最佳女配角奖]]。至2007年起，她是其中一名最高身價的[[荷里活|荷里活]]女演員。<ref>{{Cite web |url = http://www.thesun.co.uk/sol/homepage/showbiz/film/1338887/Actress-Reese-Witherspoon-tops-rich-list-for-second-year-running.html |title = Reese is still raking it in |author = Nadia Mendoza |date = 2008-06-25 |publisher = The Sun |language = en}}</ref>

== 简介 ==
蕾妮的父母都是[[移民|移民]]，父亲是[[瑞士|瑞士]]人，母亲是[[挪威|挪威]]人。她1987年毕业于Katy高中，曾当过[[啦啦队|啦啦队]]队员，[[体操|体操]]运动员，也加入过[[戏剧|戏剧]]社團。她進入[[德州大學奥斯汀分校|德州大學奥斯汀分校]]英語系学习，选修了一门戏剧课，但唯一原因是为了完成[[美术|美术]]学分。她平常靠在酒吧、饭店中打零工维持生计。

1991年從[[德州大學奧斯汀分校|德州大學奧斯汀分校]]毕业后，她走上了演艺道路，积极为[[德州|德州]]商业广告和小电影试镜。

1994年她凭借在《{{link-en|德州電鋸殺人狂4|Texas Chainsaw Massacre: The Next Generation}}》（''Texas Chainsaw Massacre: The Next Generation''）中的表演一举成名，其后又主演了《{{link-en|杀戮边缘|Love and a .45}}》（''Love and a .45''）。自此之后她的影视事业一帆风顺，参与演出了三部《[[BJ单身日记|BJ单身日记]]》、《[[芝加哥_(電影)|芝加哥]]》与《[[冷山|冷山]]》等一系列广受好评的电影。
