'''球鞋網路'''（{{lang-en|Sneakernet}}）形容使用移動物理媒介的方式傳輸電子資訊。<ref>{{cite web|title = Amazon Snowball：球鞋網路的極致 |author = 傅瑞德 |url =https://rocket.cafe/talks/3454 |date = 2015-10-08 |accessdate = 2017-06-04 |archiveurl = https://web.archive.org/web/20170604011951/https://rocket.cafe/talks/3454 |archivedate = 2017-06-04 |deadurl = no}}</ref><ref>{{Cite web|title=Oxford Dictionary |url=http://www.oxforddictionaries.com/definition/english/sneakernet |accessdate=2017-06-04 |archiveurl=https://archive.is/20170604014413/https://en.oxforddictionaries.com/definition/sneakernet |archivedate=2017-06-04 |deadurl=no }}</ref>

==-{渠道}-==
一般定義被視為將資訊從一台電腦傳到另一台，而非使用[[網際網路|網際網路]]傳輸。包括但不限於：
# [[磁帶|磁帶]]
# [[軟碟片|軟碟片]]
# [[CD|CD]]
# [[USB隨身碟|USB隨身碟]]
# 外接式[[硬碟|硬碟]]

==歷史個案==
# [[澳洲悉尼大學|澳洲悉尼大學]]於1983年使用[[usenet|usenet]]進行學術性交流。
# [[Google_Storage|谷歌]]於2010年以保障消費者私隱理由使用哈勃空间望远镜傳送120TB檔案。
# [[亞馬遜網路服務系統|亞馬遜網路服務系統]]於2015使用AWS Snowball單次輸送50TB(23公斤)數據,2016年使用AWS Snowmobile單次輸送100PB(46公斤)數據
