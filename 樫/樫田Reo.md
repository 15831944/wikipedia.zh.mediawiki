'''{{lang|ja|樫田レオ}}'''（{{lang|ja|かしだ レオ}}）是日本的自由游戏剧本作家。

== 概要 ==
[[岐阜县|岐阜县]]出身，生日是3月5日。<ref name = "Interview"/>主要在[[Key_(公司)|Key]]的作品中担当辅助的剧本作家。

在2005年11月25日Key发售的[[智代After_～It's_a_Wonderful_Life～|智代After ～It's a Wonderful Life～]]中出道，创作了游戏中一部分的剧本。<ref name = "Tomoyo">{{Cite web |url= http://prot.co.jp/ps2/tomoyoafter/index.html |script-title= ja:智代アフター～It’s a Wonderful Life～CS Edition |author= [[PROTOTYPE|PROTOTYPE]]|date= |work= |publisher= |language= ja|accessdate= 2012年5月13日|quote=  }}</ref>在2007年7月27日发售的Key社作品[[Little_Busters!|Little Busters!]]中担任西园美鱼线路的剧本作家，并参与了部分歌曲的作词。<ref name="LB">{{Cite web|url=http://key.visualarts.gr.jp/product/little/products.htm |script-title=ja:リトルバスターズ！製品内容 |author=[[Key_(公司)|Key]] |date= |work=Little Busters!官方网站 |publisher=[[Visual_Art's|Visual Art's]] |language=ja |accessdate=2012年5月13日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20110518025600/http://key.visualarts.gr.jp/product/little/products.htm |archivedate=2011年5月18日 }}</ref>Key社于2008年7月25日发售Little Busters!的成人向化作品[[Little_Busters!|Little Busters! EX]]，其中{{lang|ja|樫田レオ}}依然担任西园美鱼线的剧本作家。

在[[同人|同人]]活动中十分活跃，曾以数篇[[同人小说|同人小说]]参加过3次[[Comic_Market|Comic Market]]，并且已经决定申请参加2012年8月举行的C82。<ref name = "Doujin">{{Cite web |url= http://www7b.biglobe.ne.jp/~kashida_leo/doujin.html |script-title= ja:同人活動 |author= {{lang|ja|樫田レオ}}|date= |work= {{lang|ja|樫田レオ}}官方网站|publisher= |language= ja|accessdate= 2012年5月13日|quote=  }}</ref>

本业是税务顾问（Tax advisor），现在在岐阜县[[岐阜市|岐阜市]]经营着税务顾问事务所。<ref name = "Shigoto">{{Cite web |url= http://www7b.biglobe.ne.jp/~kashida_leo/works.html |script-title= ja:仕事内容 |author= {{lang|ja|樫田レオ}}|date= |work= {{lang|ja|樫田レオ}}官方网站|publisher= |language= ja|accessdate= 2012年5月13日|quote=  }}</ref>

== 人物 ==
* 经常在博客上发表自己对读过的[[小说|小说]]、[[漫画|漫画]]，看过的[[动画|动画]]等的评论。
* 曾经被怀疑是[[久弥直树|久弥直树]]的化名，因为[[笔名|笔名]]倒过来读是“我是鹿”（{{lang|ja|俺だ鹿}}），而“鹿”是Key社的粉丝对久弥的爱称。<ref>{{Cite web |url = http://hi.baidu.com/tiancangmiao/blog/item/98dadf077b0062ca7a894796.html |title= 【某苍个人心得】孤独的价值——再谈久弥直樹 |author= 天苍妙 |date= 2008年12月31日|work= |publisher= |language= zh-hans|accessdate= 2012年5月13日|quote=  }}</ref>
* 刚开始只是出于兴趣做Key的二次创作（以关于[[Kanon|Kanon]]中的天野美汐的居多），后来被[[Visual_Art's|Visual Art's]]邀请加入智代after的制作团队。<ref name="Interview">{{Cite web|url= http://theinterviews.jp/kashidaleo|script-title= ja:樫田レオのインタビュー|author= {{lang|ja|樫田レオ}}|date= |work= THE INTERVIEWS|publisher= |language= ja|accessdate= 2012年5月13日|quote= |deadurl= yes|archiveurl= https://web.archive.org/web/20121008051102/http://theinterviews.jp/kashidaleo|archivedate= 2012年10月8日}}</ref>
* 自我介绍是“脚穿（税务顾问和剧本作家的）两只草鞋，希望能够随心所欲地奔跑，但实际上已经快要跌倒了”。<ref name = "Interview"/>
