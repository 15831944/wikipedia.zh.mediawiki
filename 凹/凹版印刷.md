'''凹版印刷'''（{{lang-it|Intaglio}}），是印纹在印版表面雕刻凹下的[[制版|制版]]技术。

== 概要 ==
一般说来，采用[[铜|铜]]或[[锌|锌]]板作为雕刻的表面，凹下的部分可利用[[腐蚀|腐蚀]]、[[雕刻|雕刻]]、[[铜版画|铜版画]]或[[mezzotint金属版制版法|mezzotint金属版制版法]]，[[Collography|Collographs]]可能按照凹印版印刷。要印刷凹印版，表面覆上油墨，然后用''[[塔勒坦布|塔勒坦布]]''或刮刀从表面擦去油墨，只留下凹下的部分。把被印物覆在印版之上，印版和纸张通过印刷机加压，将油墨从印版凹下的部分传送到纸张上完成印刷过程。

== 種類 ==
'''雕刻凹版'''（Intaglio）
: 即傳統凹版印刷，印紋部份具凸出感，油墨濃厚、結實。適用於[[有價證券|有價證券]]、[[鈔票|鈔票]]的印刷上。<ref>[http://zosterops.myweb.hinet.net/page5/p5-4.htm 選擇與被印物合適的印刷法]</ref>
: 「Intaglio」这来自[[意大利|意大利]]的词主要指的是雕刻木板，用于装饰的目的（如[[家具|家具]]）。有时也用来指雕刻[[印章|印章]]，其在印在要盖章的材料上，留下凸起的图文。

'''{{link-en|照相凹版|Rotogravure}}'''（{{lang|en|Rotogravure}}）
: 或稱'''攝影製版'''、'''照相印刷'''（gravure）<ref>{{Cite web |url=http://www1.ntmofa.gov.tw/artnew/nhtml/5/603.htm |title=名詞解釋 - 國立臺灣美術館 |access-date=2014-09-09 |archive-url=https://web.archive.org/web/20140216223113/http://www1.ntmofa.gov.tw/artnew/nhtml/5/603.htm |archive-date=2014-02-16 |dead-url=yes }}</ref>，以投影的方式通過凹版專用網線版（Contact Screen），曝光於已塗佈感光藥膜的金屬版面，隨後經酸水侵蝕，造成版面印色部位深淺不一之凹槽印紋，看似有隱約網點狀、連續調，最後由機器之滾墨、揩墨作用，印出濃淡色調。<ref>{{Cite web |url=http://www1.ntmofa.gov.tw/artnew/nhtml/5/606.htm |title=名詞解釋 - 國立臺灣美術館 |access-date=2014-09-09 |archive-url=https://web.archive.org/web/20140221170555/http://www1.ntmofa.gov.tw/artnew/nhtml/5/606.htm |archive-date=2014-02-21 |dead-url=yes }}</ref>適用於彩色雜誌、塑膠膜、鋁箔等包裝印刷物上。
: 因採輪轉式凹版印刷機印刷，而又稱作'''輪轉凹印'''。
:照相凹版即採用照相方法製成的凹版製版。用三原色油墨進行套印，即可印出色彩與原稿相符的郵票；為了表現畫面的深暗部分，有時再增加黑版、灰版或其他色版套印，稱為“四色版”、“五色版”……有時還運用專色，以達到墨色淳厚、簡潔明快的效果。它的特徵是：版紋深、網紋細、吃墨量大，表現能力很強，用三色版或四色版，就能印製出刷色效果比較好的郵票；色彩鮮豔、濃郁、純正，層次豐富，柔和而細膩。
