'''鄂州机场'''是一个位于[[湖北省|湖北省]][[鄂州市|鄂州市]]的待建机场，选址地跨鄂州市[[鄂城区|鄂城区]][[燕矶镇|燕矶镇]]、[[杨叶镇|杨叶镇]]、[[沙窝乡|沙窝乡]]，2018年2月9日获得[[中华人民共和国国务院|国务院]]、[[中央军委|中央军委]]批准立项<ref>{{Cite web|url=http://www.gov.cn/zhengce/content/2018-02/23/content_5268245.htm|title=国务院 中央军委关于同意新建湖北鄂州民用机场的批复（国函〔2018〕26号）|accessdate=2018-02-23|work=www.gov.cn|deadurl=no|archiveurl=https://web.archive.org/web/20180223160211/http://www.gov.cn/zhengce/content/2018-02/23/content_5268245.htm|archivedate=2018-02-23}}</ref>。机场性质为客运支线、货运枢纽机场，是湖北省与[[顺丰速运|顺丰速运]]共同建设的'''湖北国际物流核心枢纽'''机场。<ref>{{Cite web |url=http://www.cninfo.com.cn/cninfo-new/disclosure/szse_sme/bulletin_detail/true/1204430698?announceTime=2018-02-26%2007:43 |title=关于公司子公司参与的湖北国际物流核心枢纽项目的进展公告 |access-date=2018-03-03 |archive-url=https://web.archive.org/web/20180303110102/http://www.cninfo.com.cn/cninfo-new/disclosure/szse_sme/bulletin_detail/true/1204430698?announceTime=2018-02-26%2007:43 |archive-date=2018-03-03 |dead-url=no }}</ref>

==概况==
该项目是国家“[[十三五|十三五]]”重大生产力布局项目，湖北省“十三五”时期重大投资项目。根据规划，该项目包括机场工程、转运中心及[[顺丰航空|顺丰航空公司]]基地工程和供油工程等。项目初步用地规模12平方公里，近期投资概算320.63亿元。预计到2025年，实现年货邮吞吐量245万吨、旅客吞吐量100万人次，成为全球前十的货运机场；到2045年，实现年货邮吞吐量765万吨、旅客吞吐量1500万人，成为全球第一大货运机场。<ref>[http://www.ezhou.gov.cn/info/2016/C042280533.htm 亚洲首个全货运机场选址鄂州] {{webarchive|url=https://web.archive.org/web/20170305003447/http://www.ezhou.gov.cn/info/2016/C042280533.htm |date=2017-03-05 }}</ref><ref>[http://www.hubei.gov.cn/zwgk/szsmlm/shzqb/201604/t20160423_824049.shtml 　　亚洲首个全货运枢纽机场选址鄂州] {{webarchive|url=https://web.archive.org/web/20160509001110/http://www.hubei.gov.cn/zwgk/szsmlm/shzqb/201604/t20160423_824049.shtml |date=2016-05-09 }}</ref><ref>[http://www.ezhou.gov.cn/info/2017/C1101104883.htm 鄂州机场预可研报告通过专家评估　王立出席评估会] {{webarchive|url=https://web.archive.org/web/20171107011933/http://www.ezhou.gov.cn/info/2017/C1101104883.htm |date=2017-11-07 }}</ref><ref>{{Cite web |url=http://www.hubei.gov.cn/2015change/2015news/xwfbh/newsfbh/xwfbh201833/ |title=鄂州市推进湖北物流核心枢纽项目进展情况新闻发布会 |access-date=2018-06-30 |archive-url=https://web.archive.org/web/20180630105634/http://www.hubei.gov.cn/2015change/2015news/xwfbh/newsfbh/xwfbh201833/ |archive-date=2018-06-30 |dead-url=no }}</ref>

==机场设施==
===跑道===
机场包含东、西2条远距平行跑道及滑行道系统，跑道长3600米、宽45米，跑道间距1900米，东跑道向南错开200米，[[飞行区等级|飞行区等级]]4E。主降方向均设置Ⅱ类精密进近系统，次降方向均设置I类精密进近系统。平行滑行道4条、垂直联络道2条，快滑12条。滑行道宽23米，两侧道肩各10.5米。
===机坪===
共有机位124个。其中，转运中心近机位56个（33D23E），远机位34个（1C14D30E）；客机机位8个（6C2E）；其他快递货运机位5个（4D1E）；货航机位9个（5D4E）；东跑道北端隔离机位1个。
===塔台===
塔台位于转运中心北侧，高87米，距离西跑道1152.75米，距离东跑道747.25米，规模按A类建设。
===航站楼===
航站楼位于西跑道南端东侧，面积1.5万平方米，配备机位8个（8C）。航站楼分两层，一层为出发与到达厅，包括送客厅、迎客厅、值机厅、行李提取厅、远机位候机厅、行李分拣厅、贵宾厅等。二层为候机厅。
===货运区===
货运区包括普货货运区、快件货运区和转运中心货运区。普货货运区和快件货运区面积2.4万平方米，供位于西跑道东侧南端，供客机机腹带货和除顺丰速运以外的第三方快递公司使用。
====转运中心====
作为[[顺丰速运|顺丰速运]]转运中心使用，位于两条跑道间靠南侧区域，面积67.8万平方米。顺丰速运包裹在此进行分拣，2025年高峰小时处理能力达到包裹37万件，小件51万件。

===顺丰航空基地===
作为[[顺丰航空|顺丰航空]]基地使用，位于西跑道北端东侧，包括机务维修设施、地面服务及勤务设施、航空食品设施等。维修机库含2个窄体维修机位和4个宽体维修机位；改装机库含1个窄体改装机位和2个宽体改装机位；喷气机库含1个机位。
===供油设施===
包含4万立方米的机场油库，临近[[长江|长江]]沿岸有5000吨级的码头泊位1个。<ref>{{Cite web |url=http://www.echeng.gov.cn/upimages/20181109173442599885Doc.pdf |title=湖北鄂州民用机场项目环境评价报告书 |access-date=2019-01-17 |archive-url=https://web.archive.org/web/20190117174230/http://www.echeng.gov.cn/upimages/20181109173442599885Doc.pdf |archive-date=2019-01-17 |dead-url=no }}</ref>

==未来发展==
===远期规划===
远期规划年限为2045年，规划占地面积18.6平方公里，将扩建转运中心等设施，并在东跑道东侧新建客运航站楼和其他快递公司设施。预计此时年业务量为旅客吞吐量1500万人次，邮货吞吐量765万吨。
===远景规划===
远景规划为终端发展目标，规划占地面积21.4平方公里。将在远期2045年规划的基础上，在东跑道东侧新建第三跑道，飞行区等级为4F，最终形成三条远距离独立平行跑道布局。
