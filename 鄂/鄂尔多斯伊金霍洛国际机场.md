'''鄂爾多斯機場'''（{{airport codes|DSN|ZBDS}}）是一个位于[[中国|中国]][[内蒙古自治区|内蒙古自治区]][[鄂尔多斯|鄂尔多斯]][[伊金霍洛旗|伊金霍洛旗]]布爾臺格鄉滿家廟的4E级民航支线机场，由[[內蒙古民航機場集團公司|內蒙古民航機場集團公司]]名下的鄂爾多斯機場管理有限公司管理<ref>鄂爾多斯高原上的銀鷹 《[[中國民航|中國民航]]》2010年11月期</ref>。

2016年11月29日，鄂尔多斯国际航空口岸对外开放新闻发布会宣布机场升级，正式更名为'''鄂尔多斯伊金霍洛国际机场'''。<ref>[http://www.thepaper.cn/newsDetail_forward_1571304 鄂尔多斯机场升级为国际机场，更名鄂尔多斯伊金霍洛国际机场]，澎湃新闻。</ref>

== 歷史 ==
鄂爾多斯機場前身為東勝機場，[[鄂爾多斯|鄂爾多斯]]於1959年修建東勝機場，主要用作橫越[[黃河|黃河]]之用，只能供[[中国军用飞机列表#型號列表|Y-5飛機]]起落，不過隨著[[包頭黃河大橋|包頭黃河大橋]]的落成，東勝機場承擔的責任已大不如前，故機場於1983年10月停航<ref>[http://www.nmg.xinhuanet.com/nmgzfzx/2007-06/29/content_10438961.htm 内蒙古鄂尔多斯机场试飞成功 7月底将通航] {{webarchive|url=https://web.archive.org/web/20160303181019/http://www.nmg.xinhuanet.com/nmgzfzx/2007-06/29/content_10438961.htm |date=2016-03-03 }}</ref>。

== 興建 ==
2005年3月25日[[國務院|國務院]][[中央軍委|中央軍委]]通過批准[[內蒙古自治區|內蒙古自治區人民政府]]遷建鄂爾多斯民用機場的申請<REF>[http://www.gov.cn/xxgk/pub/govpublic/mrlm/200803/t20080328_31913.html 國務院中央軍委關于同意遷建鄂爾多斯民用機場的批復國函〔2005〕16號]</REF>，預算機場的投資額為1.53億元[[人民幣|人民幣]]，跑道長1800米，寬45米，可供[[多尼爾328|多尼爾328]]、[[運-12|運-12E]]、[[衝8|衝8]]等支線飛機升降。機場的修建工程由[[內蒙古西蒙集團|內蒙古西蒙集團]]及[[神東天隆公司|神東天隆公司]]負責，是內地第一家由民營企業投資建設的民航機場<REF>[http://paper.wenweipo.com/2007/07/28/FI0707280037.htm 內蒙鄂爾多斯機場通航]</REF>，建築工程歷時兩年，實際投資額為3.5億元，飛行區等級為4C級，跑道實際長2,400米，寬45米，可供波音737—800及以下機型起降。機場於2007年6月5日完成飛行校驗，于7月26日正式重新通航<ref>[http://paper.wenweipo.com/2007/07/02/FI0707020007.htm 內蒙鄂市機場本月通航]</ref>。

== 擴建 ==
鄂爾多斯機場自建成通航以来，旅客人数快速增長，超出了原先設計的旅客吞吐量，故機場的扩建工程早于2008年着手展开，並於2009年得到[[華北民航局|華北民航局]]的批准。新航站楼工程于2013年1月30日投入使用。航站楼建筑面积10.03万平方米，可满足1200万人次旅客吞吐量需求。航站楼设有13部廊桥，站坪机位14个。楼内现有A、B两座值机岛，值机柜台32个，国内旅客安检双通道7条，国际旅客安检单通道3条，贵宾厅面积1万平方米，共有休息室34间。<ref name=":0">{{Cite web|url=http://www.ordosairport.com/About_us.aspx?CID=131|title=机场公司|accessdate=2018-08-09|author=鄂尔多斯机场管理公司|date=|publisher=}}</ref>

== 使用情况 ==
鄂爾多斯機場原定設計至2015年，年旅客吞吐量為27萬人次。不過在2008年，機場的旅客吞吐量已突破30萬人次，遠超過興建時的預計吞吐量，至2009年，機場的旅客增長數為53.7%，實際人次為47萬2656人次，全國排第66位，2010年頭半年，機場的客量及貨量增長率依然強勁，兩者分別為57.3%及65%<ref>鄂爾多斯的騰飛之翼 《[[中國民航|中國民航]]》2010年9月期</ref>，同年9月，機場客運量更首次突破50萬人次<ref>[http://www.csoca.org/index.php?option=com_content&task=view&id=24149&Itemid=65 鄂爾多斯機場旅客吞吐量首次突破50萬創紀錄]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2011年11月9日，鄂尔多斯机场机场吞吐量突破100万人次<ref>[http://news.carnoc.com/list/205/205281.html 仅仅四年　鄂尔多斯机场吞吐量突破百万人次]</ref>。

2017年，鄂尔多斯机场年客流量为2,105,842人次，较上年增长23.4%。<ref>{{Cite web|url=http://www.caac.gov.cn/XXGK/XXGK/TJSJ/201803/t20180307_55600.html|title=2017年民航机场生产统计公报|accessdate=2018-08-09|work=www.caac.gov.cn}}</ref>

由于硬件设施先进、气象条件优越、空域及净空条件良好、日可供飞行训练时间充足、距离主要航空公司基地较近等优势，鄂尔多斯机场现为[[中国国际航空|中国国际航空]]、[[首都航空|首都航空]]、[[中国联合航空|中国联合航空]]、[[海南航空|海南航空]]等航空公司的训练基地。<ref>{{Cite web|url=http://news.carnoc.com/list/343/343476.html?f=mhs|title=图片 鄂尔多斯机场:起降“大飞机”的时代已正式来临_民航新闻_民航资源网|accessdate=2018-08-09|work=news.carnoc.com}}</ref>

鄂尔多斯机场曾在2015被[[Skytrax|Skytrax]]评为四星级机场，是中国第8家获得该殊荣的机场。<ref>{{Cite web|url=http://finance.ifeng.com/a/20150812/13905129_0.shtml|title=鄂尔多斯机场获评“国际四星级机场”称号|accessdate=2018-08-09|work=finance.ifeng.com}}</ref>

2016年鄂尔多斯机场成功保障了波音787-9和波音747-400等大型客货机任务，成为华北地区除省会机场外首个具备大型客机与货机保障能力的支线机场。<ref name=":0" />
