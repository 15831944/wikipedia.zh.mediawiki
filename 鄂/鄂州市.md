'''鄂州市'''是[[中华人民共和国|中华人民共和国]][[湖北省|湖北省]]下辖的[[地级市|地级市]]，位于[[长江|长江]]南岸，为鄂东地区重要工业基地，属于[[武汉城市圈|武汉城市圈]]。鄂州市是湖北省第一批[[改革开放|改革开放]]试验区，也是[[历史|历史]][[文化|文化]]名城、古铜镜之乡，“[[武昌|武昌]]鱼”的原产地，素有“百湖之市”、“鱼米之乡”等称号。鄂州市面积1596平方公里，为中国面积最小的地级市（只考慮陸地面積的話則為第三小，僅次於[[三沙市|三沙市]]與[[舟山市|舟山市]]）。

== 历史沿革 ==
*[[春秋时期|春秋时期]]，楚王熊渠封儿子[[熊挚红|熊挚红]]为鄂王，在长江南岸梁子湖畔修筑鄂王城，湖北省的简称“鄂”由此而来。
*221年，[[东吴|东吴]][[孙权|孙权]]在此建城，取“以武而昌”之义，改[[鄂县|鄂县]]为武昌，定都于此。在设置[[武昌县_(东吴)|武昌县]]的同时设置[[武昌郡|武昌郡]]，治武昌县。
*[[隋朝|隋朝]]改[[郢州_(南朝宋)|郢州]]为鄂州，治所在[[江夏县|江夏县]]（现[[武汉市|武汉市]][[武昌区|武昌区]]），因此不少人误以为是鄂州和武昌名字互换。
*[[元朝|元代]]改当时的[[鄂州路|鄂州路]]为[[武昌路|武昌路]]，[[明代|明代]]改为[[武昌府|武昌府]]，治所仍在江夏县。
*为了区别，武昌府城被称为“[[上武昌|上武昌]]”，武昌县城被称为“[[下武昌|下武昌]]”，因此又有“上下武昌城”的说法。[[武昌起义|武昌起义]]就发生在武昌府城。
*[[中華民國|民国]]以后，废除府制，原武昌府治所江夏县被改为武昌县。1913年，原武昌县改名[[寿昌县|寿昌县]]。
*1914年，因与[[浙江省|浙江省]][[鄂城县|寿昌县]]重名，又改名[[鄂城县|鄂城县]]。
*1949年后，曾设立[[鄂城区|鄂城市]]，与鄂城县并存。1983年，设立省辖鄂州市。

== 地理 ==
鄂州以湖泊众多而闻名，拥有[[梁子湖|梁子湖]]、[[鸭儿湖|鸭儿湖]]、[[三山湖|三山湖]]、[[花马湖|花马湖]]、[[沐鹅湖|沐鹅湖]]、[[红莲湖|红莲湖]]、[[洋澜湖|洋澜湖]]等大小湖泊133个，水域面积达4.3万公顷。

其中风光秀美的梁子湖是中国重要的自然湿地保护区，是湖北省第二大淡水湖，也是正宗武昌鱼的原产地。

== 经济 ==
2011年，全市实现[[国内生产总值|生产总值]]（GDP）490.89亿元<ref>[http://www.stats-hb.gov.cn/structure/tjgb/ndtjgb/zw_184669_1.htm 《2011年鄂州市国民经济和社会发展统计公报》]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，按可比价格计算，比上年增长16.0%。其中，[[第一产业|第一产业]]实现增加值60.99亿元，增长4.6%；[[第二产业|第二产业]]增加值289.83亿元，增长17.2%；[[第三产业|第三产业]]增加值140.07亿元，增长18.8%。三次产业结构为12.4：59.1：28.5。按常住人口计算，全市人均生产总值达到46756元，折合7238美元。
