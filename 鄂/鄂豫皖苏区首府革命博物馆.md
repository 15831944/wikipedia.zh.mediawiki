'''鄂豫皖苏区首府革命博物馆'''，位于[[河南省|河南省]][[信阳市|信阳市]][[新县|新县]]首府路首府路文博新村004号，是展示新县作为[[鄂豫皖苏区|鄂豫皖苏区]]首府的革命历史的博物馆。

==简介==
1958年，新县建立新县革命烈士纪念馆。1979年，中共新县县委决定将烈士馆用房中的几间辟为'''新县革命历史陈列室'''。新县是当年鄂豫皖苏区的首府，留下了许多革命历史和革命文物，该陈列室不能满足展出要求。经[[中共河南省委|中共河南省委]]、[[河南省人民政府|河南省人民政府]]批准并拨款，在新县创建'''鄂豫皖苏区首府革命博物馆'''，并由[[李先念|李先念]]题写馆名。1985年10月25日奠基，1987年12月22日完工，1990年10月25日开馆<ref name=www>{{Cite web |url = http://www.haww.gov.cn/clzl/2006-11/16/content_115998.htm |title = 鄂豫皖苏区首府革命博物馆 |author =  |date = 2006-11-16 |publisher =  河南文物网|language =  |accessdate =  |quote =  }}</ref>。2002年5月17日，鄂豫皖苏区首府革命博物馆经过改造后重新开馆，并同时举行“爱国主义教育示范基地”揭牌仪式<ref>{{Cite web|url =http://news.xinhuanet.com/newscenter/2002-05/17/content_398112.htm|title =鄂豫皖苏区首府革命博物馆开馆|author =|date =2002-05-17|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20041124064744/http://news.xinhuanet.com/newscenter/2002-05/17/content_398112.htm|archivedate =2004-11-24}}</ref>。2017年，鄂豫皖苏区首府革命博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。

鄂豫皖苏区首府革命博物馆为河南省内规模最大的县级革命博物馆，为仿古建筑，占地面积21000平方米，建筑面积2700平方米。内设7个展厅，展陈面积828平方米，分为基本陈列、专题陈列。基本陈列展示鄂豫皖苏区10年[[第一次国共内战|土地革命]]历史。专题陈列展示新县籍43位授衔将军、50位省军级干部的生平事迹<ref name=www/>。

鄂豫皖苏区首府革命博物馆藏品主要为革命文物，共407件，其中的壁书《中华苏维埃土地法大纲》是革命文物一级品。藏品主要分为：军事用品（如“[[列宁号|列宁号]]”飞机、长矛、大刀、枪支、红军旗帜等）、文献资料（如党政机关的报告、法令、政策、标语、传单等）、烈士遗物、历史照片、其他文物（如各级苏维埃印章、钱币、收税凭单、收条、抚恤证、土地证等）<ref name=www/>。
