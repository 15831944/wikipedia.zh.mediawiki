'''鄂尔多斯博物馆'''，位于[[内蒙古自治区|内蒙古自治区]][[鄂尔多斯市|鄂尔多斯市]][[康巴什新区|康巴什新区]]文化西路南5号，是鄂尔多斯市属综合性博物馆。

==简介==
2006年5月26日，鄂尔多斯博物馆奠基开工建设，2011年主体建筑完工。2012年5月18日[[国际博物馆日|国际博物馆日]]，免费对外正式开放。鄂尔多斯博物馆占地面积27760平方米，建筑面积41227平方米，地下一层、地上四层、局部八层，7个专业展厅的总面积达到10000平方米。建筑造型像一颗坠入沙漠的[[陨石|陨石]]，也像[[蒙古|蒙古]]包，建筑顶部线条创造出丰富的内部空间；表层古铜色金属材料象征着以青铜器为代表的北方草原游牧文明<ref name=jueqi>{{Cite web |url = http://news.ifeng.com/gundong/detail_2013_05/13/25230450_0.shtml |title = 崛起的鄂尔多斯博物馆 |author =  |date = 2013-05-13 |publisher =  凤凰网|language =  |accessdate =  |quote =  }}</ref><ref name=jishi>{{Cite web |url =http://www.ordos.gov.cn/dtxx/bmdt/201305/t20130515_853090.html  |title = 从远古走向今天的记忆——鄂尔多斯博物馆发展纪实 |author =  |date =2013-05-15  |publisher = 鄂尔多斯在线 |language =  |accessdate =  |quote =  }}</ref>。

博物馆开馆后，设四个常规展览：《农耕、游牧·交融、碰撞——鄂尔多斯通史展》、《鄂尔多斯蒙古族历史文化展》、《鄂尔多斯社会发展成就展》、《鄂尔多斯古生物展》。另外还经常举办临时展览。鄂尔多斯博物馆通过征集民间文物，以及收藏乌兰木伦等遗址的出土文物，使馆藏文物在2013年达到23000多件<ref name=jueqi/><ref name=jishi/>。

2017年，鄂尔多斯博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.northnews.cn/2017/0110/2358723.shtml |title = 鄂尔多斯博物馆获评国家一级博物馆 |author =  |date = 2017-01-10 |website = 正北方网 |language =  |accessdate =  |quote =  }}</ref>。
