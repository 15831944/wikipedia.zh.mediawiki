'''磷化氢'''，又名'''[[膦|膦]]'''（拼音：lìn），分子式：{{磷}}{{氢}}{{sub|3}}，一种无色、极毒、有魚腥臭味的气体。<br />

== 製取 ==
实验室中可以用[[白磷|白磷]]与浓[[氢氧化钾|氢氧化钾]]共热制取：

:P<sub>4</sub> + 3KOH(浓) + 3H<sub>2</sub>O → PH<sub>3</sub>↑ + 3KH<sub>2</sub>PO<sub>2</sub>

生成的磷化氢往往因为含有[[聯膦|聯膦]]（P<sub>2</sub>H<sub>4</sub>）而会自燃。磷化氢可用铜、汞、银等较不活泼的金属盐溶液吸收：

:4CuSO<sub>4</sub> + PH<sub>3</sub> + 4H<sub>2</sub>O → 4Cu↓ + H<sub>3</sub>PO<sub>4</sub> + 4H<sub>2</sub>SO<sub>4</sub>
:24CuSO<sub>4</sub> + 11PH<sub>3</sub> + 12H<sub>2</sub>O → 8Cu<sub>3</sub>P↓ + 3H<sub>3</sub>PO<sub>4</sub> + 24H<sub>2</sub>SO<sub>4</sub>

磷化氢还可通过金属磷化物与水作用得到。例如：

:Ca<sub>3</sub>P<sub>2</sub> + 6H<sub>2</sub>O → 3Ca(OH)<sub>2</sub> + 2PH<sub>3</sub>↑
== 性質 ==
磷化氢的密度大于空气，化学性质與[[氨|氨]]相似，也有[[孤對電子|孤對電子]]，所以磷化氫也為一種[[路易斯鹼|路易斯鹼]]，具有配位能力，如低温下和[[三氟化硼|BF<sub>3</sub>]]反应生成H<sub>3</sub>P·BF<sub>3</sub>。與[[氫離子|氫離子]]結合則生成[[鏻|-{zh-hans:𬭸;zh-hant:鏻;}-]]離子(只有与[[强酸|强酸]]如[[氢碘酸|HI]]、[[高氯酸|HClO<sub>4</sub>]]等反应时)。

:PH<sub>3</sub> + H<sup>+</sup> → PH<sub>4</sub><sup>+</sup>

磷化氢还具有强还原性。例如，磷化氢能还原Ag<sup>+</sup>、Cu<sup>2+</sup>等某些金属离子，使Ag<sup>+</sup>还原为Ag,使Cu<sup>2+</sup>还原为[[磷化亚铜|Cu<sub>3</sub>P]]和Cu。

:PH<sub>3</sub> + 6Ag<sup>+</sup> + 3H<sub>2</sub>O → 6Ag + H<sub>3</sub>PO<sub>3</sub> + 6H<sup>+</sup>

此外，磷化氢还能参与有机反应。磷化氢和醛在加热下（100℃）可得到含有[[羟甲基|羟甲基]]的叔膦化合物<ref>有机磷化合物.尹志刚 主编.化学工业出版社. 2.1.3.8 与醛的反应. P<sub>40</sub>. ISBN 978-7-122-09816-0</ref>：
: PH<sub>3</sub> + 3 RCHO → [RCH(OH)]<sub>3</sub>P

磷化氢是剧毒气体。磷化氢在空气中的最高允许量为0.3[[百萬分率|ppm]]。气体中的磷化氢可用[[重铬酸钾|重铬酸钾]]、[[铬酸银|铬酸银]]、[[活性炭|活性炭]]、[[漂白粉|漂白粉]]的[[悬浮液|悬浮液]]消除其毒性。空气中微量的磷化氢可根据用[[硝酸银|硝酸银]]浸过的[[硅胶|硅胶]]变黑情况指示出来。[[磷化铝|磷化铝]]、[[磷化锌|磷化锌]]和空气中的[[水蒸气|水蒸气]]反应生成磷化氢，因此要密封保存。这一特性使其被用作粮食仓库的烟熏[[消毒剂|消毒剂]]。<ref>刘新锦等.无机元素化学(第二版) 氮族元素.北京:科学出版社.2010.01 ISBN 978-7-03-026399-5</ref>

== 安全 ==
磷化氫氣體可能與空氣形成爆炸性混合物並可以[[自燃|自燃]]。磷化氫燃燒時，它會產生白色煙霧[[五氧化二磷|五氧化二磷]]，吸入後可導致嚴重刺激呼吸道。
