'''磷酸钇'''是一种[[钇|钇]]的[[磷酸盐|磷酸盐]]，化学式为YPO<sub>4</sub>。<ref>{{Greenwood&Earnshaw2nd|page=526}}</ref>它在自然界中以[[磷钇矿|磷钇矿]]<ref name=Jane>Jane E. Macintyre: Dictionary of Inorganic Compounds. CRC Press, 1992, ISBN 0-412-30120-2, S. 3704 (eingeschränkte Vorschau in der Google-Buchsuche).</ref>和针磷钇铒矿<ref>Charles Milton, K J Murata, M M Knechtel. [http://www.minsocam.org/ammin/AM29/AM29_92.pdf WEINSCHENKITE, YTTRIUM PHOSPHATE DIHYDRATE, FROM VIRGINIA]. minsocam. [2017-8-14]</ref>的形式存在。

==制备==
磷酸钇可以由[[氯化钇|氯化钇]]和[[磷酸钠|磷酸钠]]反应，或由[[硝酸钇|硝酸钇]]和[[磷酸氢二铵|磷酸氢二铵]]于溶液中反应得到：<ref name=Jane />
:<math>\mathrm{YCl_3 + (NH_4)_3PO_4 \longrightarrow YPO_4 \downarrow + 3 \ NH_4Cl}</math>
:<math>\mathrm{Y(NO_3)_3 + (NH_4 )_2 HPO_4 \longrightarrow YPO_4 \downarrow \ + 2 \ NH_4NO_3 + HNO3}</math>

[[氧化钇|氧化钇]]和磷酸氢二铵的反应也能制备磷酸钇：<ref>Georg Brauer: Handbuch der Präparativen Anorganischen Chemie. 3., umgearb. Auflage. Band I. Enke, Stuttgart 1975, ISBN 3-432-02328-6, S. 1114.</ref>
:<math>\mathrm{Y_2O_3 + 2 \ (NH_4 )_2HPO_4 \longrightarrow YPO_4 + 2 \ NH_3 + 3 \ H_2 O}</math>

氯化钇和[[磷酸|磷酸]]于35~40°C混合，之后滴加[[氨水|氨水]]反应：<ref>{{книга
 |заглавие = Руководство по неорганическому синтезу: В 6-ти т. |ответственный = Ред. Брауэр Г. |место  = М.
 |издательство  = Мир |год = 1985 |том = 4 |страниц  = 447
}}</ref>
:<math>\mathrm{ YCl_3 + H_3PO_4 + 3NH_3\cdot H_2O \ \xrightarrow{35-40^oC}\ YPO_4 + 3NH_4Cl + 3H_2O }</math>

==性质==
磷酸钇属[[四方晶系|四方晶系]]，[[晶胞参数|晶胞参数]]为a=0.68832 nm，c=0.60208 nm。它可以以无水物、一水合物或二水合物的形式存在。二水合物属[[单斜晶系|单斜晶系]]，[[空间群|空间群]]为'''''B 2/b'''''，晶胞参数为a=0.648 nm，b=1.512 nm，c=0.628 nm，β=129.4°，Z = 4。<ref>{{книга

磷酸钇和浓碱反应，生成[[氢氧化钇|氢氧化钇]]。<ref>易宪武 等. 无机化学丛书 第七卷 钪 稀土元素. 科学出版社, 2011. pp 178. ISBN 978-7-03-030574-9</ref>

== 用途 ==
掺杂Ce<sup>3+</sup>的磷酸钇可用作发光材料<ref>文雨水. [http://www.ixueshu.com/document/12715efa7ee0d218318947a18e7f9386.html 铈激活磷酸钇发光材料][J]. 液晶与显示, 1990(z1):27-28.</ref>，Ce<sup>3+</sup>-Tb<sup>3+</sup>等双掺杂材料也有报道。<ref>刘旭焱, 丁楠, 屈重年 等. [http://www.fgxb.org/CN/article/downloadArticleFile.do?attachType=PDF&id=13512 稀土掺杂磷酸钇荧光粉的控制合成和发光性能][J]. 发光学报, 2017(5):587-593.</ref>
