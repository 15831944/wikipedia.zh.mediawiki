'''磷酸亚铁'''是[[磷酸|磷酸]]的[[铁|铁]]盐，[[化学式|化学式]]Fe<sub>3</sub>(PO<sub>4</sub>)<sub>2</sub>。可用于花园里消灭苔藓和蛞蝓。

== 制备 ==
* 存在于自然界的矿物中：
** [[蓝铁矿|蓝铁矿]] — Fe<sub>3</sub>(PO<sub>4</sub>)<sub>2</sub>·8H<sub>2</sub>O，含有[[磷酸铁|磷酸铁]]杂质。

* 混合[[硫酸亚铁|硫酸亚铁]]和[[磷酸钠|磷酸钠]]溶液：
:<math>\mathsf{3FeSO_4 + 2Na_3PO_4 \ \xrightarrow{}\ Fe_3(PO_4)_2\downarrow + 3Na_2SO_4 }</math>

== 物理性质 ==
蓝灰色[[单斜晶系|单斜晶体]]，[[空间群|空间群]]'''''P 2<sub>1</sub>/c'''''，其中''a'' = 0.881 nm，''b'' = 1.1169 nm，''c'' = 0.6145 nm，β = 99.6°，Z = 4。

可形成水合物Fe<sub>3</sub>(PO<sub>4</sub>)<sub>2</sub>·''n'' H<sub>2</sub>O，其中''n'' = 1, 4, 6, 8。

水合物晶体须密封存储，因为它们很容易在空气中氧化。

== 化学性质 ==

* 与[[磷酸|磷酸]]反应，形成[[酸式盐|酸式盐]]：
:<math>\mathsf{Fe_3(PO_4)_2 + H_3PO_4 \ \xrightarrow{}\ 3FeHPO_4 }</math>
:<math>\mathsf{Fe_3(PO_4)_2 + 4H_3PO_4 \ \xrightarrow{}\ 3Fe(H_2PO_4)_2 }</math>

* 与[[氢|氢]]反应生成[[磷化物|磷化物]]：
:<math>\mathsf{Fe_3(PO_4)_2 + 8H_2 \ \xrightarrow{T}\ Fe_3P_2 + 8H_2O }</math>

* 水合物加热分解：
:<math>\mathsf{Fe_3(PO_4)_2\cdot 8H_2O \ \xrightarrow[-H_2O]{T}\ Fe_3(PO_4)_2\cdot 4H_2O \ \xrightarrow[-H_2O]{180^oC}\ Fe_3(PO_4)_2 }</math>
