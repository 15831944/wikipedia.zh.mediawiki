'''磷酸二氢錳'''是一種[[無機化合物|無機化合物]]，屬於磷酸二氢鹽，由[[錳|錳]]、[[磷|磷]]、[[氧|氧]]、[[氫|氫]]組成，化學式為Mn(H<sub>2</sub>PO<sub>4</sub>)<sub>2</sub>。

== 性質 ==
因[[酸性|酸性]]較強，故具有[[腐蚀|腐蚀]]性、與[[氧化物|氧化物]]接触极容易[[变质|变质]]，一般[[常溫|常溫]]下呈磷酸二氢錳[[水合物|水合物]](Mn(H<sub>2</sub>PO<sub>4</sub>)<sub>2</sub>·2H<sub>2</sub>O)，當溫度高于[[100|100]][[攝氏溫標|℃]]时[[脱水|脱水]]生成無水磷酸二氢錳(Mn(H<sub>2</sub>PO<sub>4</sub>)<sub>2</sub>)。

== 制備 ==
工業上常用兩種方法制備'''磷酸二氢錳'''。
===碳酸锰法===
第一種方法叫作碳酸锰法

[[磷酸|磷酸]]和[[碳酸锰|碳酸锰]]进行反应，去除杂质，經分離程序制得:
:MnCO<sub>3</sub>+2H<sub>3</sub>PO<sub>4</sub>→Mn(H<sub>2</sub>PO<sub>4</sub>)<sub>2</sub>+H<sub>2</sub>O+CO<sub>2</sub>↑

===硫酸锰法===
第二種方法叫作硫酸锰法

[[硫酸锰|硫酸锰]]和[[碳酸鈉|碳酸鈉]]反应產生[[碳酸锰|碳酸锰]]，加入[[磷酸|磷酸]]使[[碳酸锰|碳酸锰]]跟[[磷酸|磷酸]]發生反应則生成磷酸二氢錳的混合物，經過分離程序則制得。
:MnSO<sub>4</sub>+Na<sub>2</sub>CO<sub>3</sub>→MnCO<sub>3</sub>+Na<sub>2</sub>SO<sub>4</sub>
:MnCO<sub>3</sub>+2H<sub>3</sub>PO<sub>4</sub>→Mn(H<sub>2</sub>PO<sub>4</sub>)<sub>2</sub>+CO<sub>2</sub>↑+H<sub>2</sub>O


== 用途 ==
'''磷酸二氢錳'''用作金屬表面的磷化處理劑，陶瓷工業中的著色劑，玻璃工業中的澄清劑，國防工業上則可以做為潤滑或防滑的塗料材料。電鍍中，用於黑色金屬製件的防腐處理，其性能僅次於[[磷酸二氫鋅|磷酸二氫鋅]]。
