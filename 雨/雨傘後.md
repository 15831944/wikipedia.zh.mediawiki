《'''雨傘後'''》，是由[[黃家駒|黃家駒]]作曲，[[陳少琪|陳少琪]]填詞的香港流行歌曲。

== 名稱由來 ==
《[[雨傘後|雨傘後]]》最初出現於香港女歌手[[呂珊|呂珊]]1993年大碟《[[今天再生|今天再生]]》TRACK 3，唱片由BMG發行。歌曲近年備受注目其一原因，是因為由已故香港傳奇音樂人，殿堂級搖滾樂隊[[Beyond|Beyond]]靈魂人物[[黃家駒|黃家駒]]作曲。另一方面，[[雨傘後|雨傘後]]可引伸「雨傘之後」的意思，不少有關[[雨傘運動|雨傘運動]]之後討論的文章、講座及藝術作品，不論支持或反對[[雨傘運動|雨傘運動]]，都用了[[雨傘後|雨傘後]]作命題提及對有關運動的反思 。」<ref>{{http://www.yzzk.com/cfm/blogger3.cfm?id=1443067821942&author=%E5%B1%88%E7%A9%8E%E5%A6%8D 亞洲周刊 屈穎妍: 雨傘後糜爛了的人心 2015年10月4日}}</ref> <ref>{{https://www.thestandnews.com/society/%E9%9B%A8%E5%82%98%E5%BE%8C%E7%9A%84%E7%A4%BE%E5%8D%80-%E4%B8%80%E5%80%8B%E8%B6%85%E8%B6%8A%E6%94%BF%E9%BB%A8%E6%94%BF%E6%B2%BB%E7%9A%84%E6%83%B3%E5%83%8F/ 立場新聞 鄭偉謙: 雨傘後的社區：一個超越政黨政治的想像 2015年10月6日}}</ref> <ref>{{https://www.eventbrite.hk/e/19056088273《雨傘後......你和我》講座.... }}</ref> <ref>{{https://www.symedialab.org.hk/talk/%E9%9B%A8%E5%82%98%E5%BE%8C%EF%BC%8C%E8%AA%B0%E5%BE%9E%E8%A8%98%E8%80%85%E5%BF%83%EF%BC%9F/  樹仁新傳網 XIAOYU ZHANG: 雨傘後，誰從記者心？ }}</ref>
 

== 呂珊原唱 ==
由[[黃家駒|黃家駒]]作曲，[[陳少琪|陳少琪]]填詞的[[雨傘後|雨傘後]]，首次出現於香港歌手[[呂珊|呂珊]] 1993年出版的唱片《今天再生》，由於香港殿堂級音樂人[[黃家駒|黃家駒]]在1993年離世，所以歌曲[[雨傘後|雨傘後]]，被視為[[黃家駒|黃家駒]]最後一批遺作之一。

== 出現於戲劇==
===改編[[雨傘後|雨傘後]] (音樂微電影)===
該節目來自[[香港電台|香港電台]]電視部製作《[[傘開以後|傘開以後]]》系列，節目希望說出香港人經歷[[雨傘運動|雨傘運動]]一年後的感受和反思。
根據《都市日報》報道'<ref>{{[http://www.metrohk.com.hk/?cmd=detail&id=289628 都市日報: 雨傘後的追尋和迷失]}}</ref> 其中一集[[香港電台|香港電台]]電視部編導特意找來非「[[黃絲|黃絲]]」、「[[藍絲|藍絲]]」的香港音樂人[[韋啟良|韋啟良]] (文中「韋指自己並非在運動站在前線「抗命」的音樂人」) 把[[雨傘後|雨傘後]]重新編曲，[[韋啟良|韋啟良]]接受訪問時表示，歌曲雖然是「90年代的作品，歌詞簡單但含意強，道出了港人在「雨傘運動」的現況，......」特別是歌詞:「雨傘後人無語退後，不再問不再望不再夢， 天色也哀愁」*，[[韋啟良|韋啟良]]表示「歌詞好似預言咗某啲嘢，人對接受或失去，可能有人會抗拒。」他指港人在[[雨傘運動|雨傘運動]]後多了一份迷失的感覺：「畫面上好似企喺十字路口度好徬徨，前面有好多條路但唔知揀咩方向，嗰份迷失感嚟自無方向嘅迷失，定係唔選擇情況下嘅迷失？其實人可以有好多選擇。」他回顧整個運動：「需唔需要大家行到咁激進？係咪所有嘢都唔可以坐低傾？」，經歷運動，他只想「嘆一口氣」。其實[[韋啟良|韋啟良]]的一番話，也道出了不少非極端「[[黃絲|黃絲]]」、「[[藍絲|藍絲]]」，不是企在最前的香港人心聲。

* 另有歌詞:
「埋藏雨傘後 人群擦過後 
安慰自己 你或者
也在想 也在哭 也難受 
也內疚」


===[[雨傘後|雨傘後]] (音樂微電影)故事大綱===
阿宇和阿珊原本是一對喜歡音樂小情侶，因[[雨傘運動|雨傘運動]]群眾多次唱[[黃家駒|黃家駒]]的《[[海闊天空|海闊天空]]》而吸引他們留心事態發展。但阿宇眼見運動滲入暴力，政府又不願與示威者和談，[[雨傘運動|雨傘運動]]最終在沒達到步向民主下被逼清場完結，阿宇認為選擇以逃避方式自處，阿珊則希望繼續尋覓運動的足印和精神。後來阿珊有機會出唱片，但保守的歌星經理人認為之前已有音樂人(指[[林夕|林夕]])因寫歌詞有「狼」、「無恥」字眼而被勸止 <ref>{{http://www.pentoy.hk/%E6%96%87%E5%8C%96/mpforum2013/2015/01/08/%E6%9E%97%E5%A4%95%E8%AC%9B%E5%A0%82%EF%BC%9A%E4%BD%A0%E9%82%84%E6%B2%89%E9%BB%98%E5%97%8E%EF%BC%9F/  林夕講堂：你還沉默嗎？ }}</ref> ，所以不敢創新以免犯他人禁忌，坊間包括內地正流行[[黃家駒|黃家駒]]式《[[喜歡你|喜歡你]]》和《[[情人|情人]]》般的情歌，經理人決定找阿珊重唱舊歌。阿珊如何抉擇? 阿宇最後會否願意重新面對阿珊，重拾舊好?

劇集特意找來香港音樂人[[韋啟良|韋啟良]]將[[黃家駒|黃家駒]]1993年所作，由[[陳少琪|陳少琪]]填詞的[[雨傘後|雨傘後]]重新編曲，由女主角[[戴萃瑤|戴萃瑤]]唱出，利用[[雨傘後|雨傘後]]的歌名說出[[雨傘運動|雨傘運動]]之後，部份港人的心情，及用[[黃家駒|黃家駒]]作品提起當時群眾齊心的情況。

===2018年香港獨立電影《[[對倒|對倒]]》===
《[[雨傘後|雨傘後]]》為獨立電影《[[對倒|對倒]]》的主題音樂，電影描述記者爸爸孫世凱 (袁富華飾演)，有一個當警察的兒子孫耀偉，([[甄思羽|甄思羽]]飾演)，及一個有參與示威抗爭的女兒孫慧語 (伍詠詩飾演)，三個人面對家庭、香港社會撕裂，各有不同意見。[[黃家駒|黃家駒]]作曲的[[雨傘後|雨傘後]]在片尾結局時播出。<ref>香港獨立電影節2018:當代獨立視野{{cite web |url=http://www.hkindieff.hk/film/programmes01f.html |title=存档副本 |accessdate=2018年1月26日 |deadurl=yes |archiveurl=https://web.archive.org/web/20180126185606/http://www.hkindieff.hk/film/programmes01f.html |archivedate=2018年1月26日 |df= }}</ref>
