'''雨宮 天'''（{{bd|1993年|8月28日|catIdx=Amamiya Sora}}<ref>生日在「声優グランプリ」2013年6月号里公布</ref>）是[[日本|日本]]的[[女性|女性]][[声优|声优]]。[[東京都|東京都]]出身。[[Music_Ray'n|Music Ray'n]]所属。<!--身長160cm。--><ref>{{Cite web|url=http://talent.yahoo.co.jp/pf/detail/pp540427|title=雨宮天 プロフィール Yahoo!人物名鑑|}}</ref><ref>{{cite web|url=http://www.oricon.co.jp/prof/597207/profile/|title=雨宮天のプロフィール|publisher=ORICON STYLE|accessdate=2016-05-12}}</ref><ref>{{cite web|url=http://prtimes.jp/main/html/rd/p/000000028.000016356.html|title=麻倉もも・雨宮天・夏川椎菜によるユニット「TrySail（トライセイル）」が4thシングルを５月11日にリリース決定！TVアニメ「はいふり」のオープニングテーマ！|date=2016-02-14|accessdate=2016-05-12|publisher=PR TIMES}}</ref>

== 来歷 ==
憧憬著配音員[[澤城美雪|澤城美雪]]而嚮往成為一名聲優，在2011年舉辦的第2回Music Ray'n超級聲優試鏡中合格，開始聲優活動<ref name="seiyuu">{{Cite journal|title=[[SEIGURA|SEIGURA]]|issue=2013年6月号|page=40－41|publisher=[[主婦之友社|主婦之友社]]|date=2013-05-10}}</ref>。

2015年，与同为Music Ray'n所属的[[麻倉桃|麻倉桃]]、[[夏川椎菜|夏川椎菜]]一起组成了声优组合「[[TrySail|TrySail]]」进行活动。

2015年獲得第9屆[[聲優獎|聲優獎]]新人聲優賞。

== 人物 ==
喜欢的食物是[[燒肉|燒肉]]。同時還是漢字[[愛好者|愛好者]]。目前正在學習[[漢字檢定|漢字檢定]]準一級<ref name="seiyuu" />。其座右銘為：物以類聚<ref name="精靈般的聲音">{{Cite web |url = https://kknews.cc/zh-tw/comic/68nmop.html |title =《聲優教室》雨宮天——精靈一般的聲音 - 每日頭條 |author =  |date = 2016-06-22 |publisher =  |language = zh |accessdate = 2016-11-25 |quote =  }}</ref>。
在七大罪錄音現場被稱作「梢（こずえ）」
