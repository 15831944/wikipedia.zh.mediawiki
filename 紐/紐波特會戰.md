'''紐波特會戰'''（{{lang-nl|Slag bij Nieuwpoort}}）是[[八十年戰爭|八十年戰爭]]中的一場著名[[戰役|戰役]]，於1600年7月2日在今[[比利時|比利時]][[紐波特_(比利時)|紐波特]]附近會戰，由[[拿騷的毛里茨|拿騷的毛里茨]]率領的[[荷蘭|荷蘭]]軍隊，對抗由[[奧地利大公|奧地利大公]][[阿爾布雷希特七世|阿爾布雷希特]]所帶領的[[法蘭德斯|弗蘭德]]（[[西班牙|西班牙]]）聯軍。

拿騷的毛里茨（後來的[[奧蘭治親王|奧蘭治親王]]）被派往攻打[[敦克爾克|敦克爾克]]市（今法國與比利時邊境附近），遭到阿爾布雷希特的突襲，會戰於是展開。這場會戰造成荷軍攻佔敦克爾克市的計畫被迫取消。

這場會戰是荷蘭歷史上最著名的戰役之一，部分肇因於紐波特會戰的日期容易記得。<ref>{{cite web |last = Verhoef |first = C.E.H.J. |date = 2000/6/28 |title= Nieuwpoort 1600. De bekendste slag uit de Tachtigjarige Oorlog |trans-title = 1600年紐波特會戰：八十年戰爭中最著名的戰役 |publisher = Aspekt |isbn = 9789075323863 |language = nl}}</ref>
