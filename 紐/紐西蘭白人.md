'''紐西蘭白人'''（{{lang-mi|'''Pākehā'''}}）是指在[[紐西蘭|紐西蘭]]居住的[[歐洲|歐洲]][[白人|白人]]<ref>{{cite web |url=http://kupu.maori.nz/ShowKupu.aspx?kupu=545 |title=存档副本 |accessdate=2016-06-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20131103171433/http://kupu.maori.nz/ShowKupu.aspx?kupu=545 |archivedate=2013-11-03 }}</ref>。[[奇異鳥|奇異鳥]]的英語{{lang|en|''Kiwi''}}普遍用來泛指所有在紐西蘭居住的“紐西蘭人”，不論膚色和種族，但 Pākehā 則專指移居當地的歐洲白人。於特殊情況，紐西蘭白人專指[[英國|英國]]、或於19到20世紀移居紐國的[[英格蘭|英格蘭]]/[[凱爾特|凱爾特]]血統的人。其他歐洲白人的來源有：[[荷蘭人|荷蘭人]]、[[斯堪的納維亞人|斯堪的納維亞人]]、[[日耳曼人|日耳曼人]]、[[斯拉夫人|斯拉夫人]]等。從1916年起，紐西蘭政府在其統計數字採用這個名字來把歐裔白人跟原住民區分<ref>{{cite web |url=http://www.stats.govt.nz/surveys_and_methods/methods/classifications-and-standards/classification-related-stats-standards/ethnicity/definition.aspx |title=存档副本 |accessdate=2012年9月21日 |deadurl=yes |archiveurl=https://web.archive.org/web/20120827050952/http://www.stats.govt.nz/surveys_and_methods/methods/classifications-and-standards/classification-related-stats-standards/ethnicity/definition.aspx |archivedate=2012年8月27日 |df= }}</ref>。


==字源==
'''Pākehā''' 一詞源自[[毛利語|毛利語]]，在18世紀後期隨着歐洲人不斷遷入，從毛利語進入當地的英語中。這個字於1815年首成出現於當地的字典裡，原指當地出生的淺色皮膚的人。這個詞語原來的意思已不能考究，但很可能源自 ''pākehakeha'' 或 ''pakepakehā''，一個對一種虛構、淺膚色的人物的名稱。亦有人指 ''Pākehā'' 語源自毛利語的[[跳蚤|跳蚤]]、[[豬|豬]]或[[乞丐|乞丐]]，但這些說法在語言學上沒有支持。
