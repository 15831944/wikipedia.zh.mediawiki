《'''紐瓦克聖人'''》（{{lang-en|''The Many Saints of Newark''}}）是一部即將上映的美國犯罪電影，由[[亞倫·泰勒_(導演)|亞倫·泰勒]]執導，[[大衛·雀斯|大衛·雀斯]]和{{link-en|勞倫斯·康納|Lawrence Konner}}編劇，本片是大衛·雀斯的[[HBO|HBO]]犯罪電視劇《[[人在江湖_(美國電視劇)|人在江湖]]》的前傳<ref>{{Cite web |url = https://www.nj.com/entertainment/2019/04/the-many-saints-of-newark-the-sopranos-prequel-movie-to-start-filming-in-newark-as-production-gets-underway-in-nyc.html |title = ‘The Many Saints of Newark,’ the ‘Sopranos’ prequel movie, to start filming in Newark |author = Amy Kuperinsky |date = 2019-04-06 |publisher = nj.com |language = en}}</ref>。主演是[[亞歷桑德羅·尼沃拉|亞歷桑德羅·尼沃拉]]、[[強·柏恩瑟|強·柏恩瑟]]、[[韋娜·法米加|韋娜·法米加]]、[[寇瑞·史托爾|寇瑞·史托爾]]、[[比利·馬格努森|比利·馬格努森]]、Michael Gandolfini和[[雷·李歐塔|雷·李歐塔]]。故事設定在1960年代{{link-en|1967年紐瓦克騷亂|1967 Newark riots|紐瓦克騷亂}}期間的[[新澤西州|新澤西州]][[紐瓦克_(新澤西州)|紐瓦克]]。

[[華納兄弟公司|華納兄弟公司]]安排本片在2020年9月25日上映。

== 製作 ==
2017年6月，創作人[[大衛·雀斯|大衛·雀斯]]拒絕考慮繼續《[[人在江湖_(美國電視劇)|人在江湖]]》的故事，但暗示了對該劇的前傳有興趣<ref>{{Cite web |url = https://www.digitalspy.com/tv/ustv/a830906/the-sopranos-creator-would-be-open-to-a-prequel-series/ |title = The Sopranos creator would be open to a prequel series |author = Sam Warner |date = 2017-06-15 |publisher = Digital Spy |language = en}}</ref>。2018年3月正式宣布將會製作一部前傳電影，雀斯正在編寫劇本。<ref>{{Cite web |url = https://www.theguardian.com/film/2018/mar/08/david-chase-to-make-sopranos-prequel-movie-the-many-saints-of-newark |title = David Chase to make Sopranos prequel movie The Many Saints of Newark |date = 2018-03-08 |publisher = The Guardian |language = en}}</ref>

2018年7月，曾執導過該劇的[[亞倫·泰勒_(導演)|亞倫·泰勒]]被聘請執導本片<ref>{{Cite web |url = https://www.hollywoodreporter.com/news/sopranos-prequel-movie-taps-director-alan-taylor-1124896 |title = 'Sopranos' Prequel Movie Taps Former Series Director Alan Taylor |author = Mia Galuppo |date = 2018-07-03 |publisher = The Hollywood Reporter |language = en}}</ref>，並且在11月，[[亞歷桑德羅·尼沃拉|亞歷桑德羅·尼沃拉]]被選在電影中主演。2019年1月，雀斯在討論該劇20週年紀念時，透露了年輕的{{link-en|Tony Soprano|Tony Soprano}}會出現在電影中<ref>{{Cite web |url = https://deadline.com/2019/01/the-sopranos-david-chase-20th-anniversary-prequel-film-tony-soprano-the-many-saints-of-newark-1202530544/ |title = ‘The Sopranos’ Creator David Chase Offers Glimpses Of Prequel Movie As Groundbreaking HBO Series Turns 20 |author = Mike Fleming Jr |date = 2019-01-07 |publisher = Deadline |language = en}}</ref>。同一個月，[[強·柏恩瑟|強·柏恩瑟]]、[[韋娜·法米加|韋娜·法米加]]、[[寇瑞·史托爾|寇瑞·史托爾]]和[[比利·馬格努森|比利·馬格努森]]加入了演員陣容。[[占士·簡東費尼|占士·簡東費尼]]的兒子Michael Gandolfini飾演年輕的Tony Soprano<ref>{{Cite web |url = https://deadline.com/2019/01/the-sopranos-prequel-movie-michael-gandolfini-tony-soprano-james-gandolfini-the-many-saints-of-newark-james-gandolfini-david-chase-1202539160/ |title = ‘Sopranos’ Prequel Film Finds Young Tony: Michael Gandolfini Is Chip Off Old Block |author = Mike Fleming Jr |date = 2019-01-22 |publisher = Deadline |language = en}}</ref><ref>{{Cite web |url = https://www.nj.com/entertainment/2019/01/james-gandolfinis-son-michael-to-play-young-tony-in-sopranos-prequel-film-the-many-saints-of-newark.html |title = James Gandolfini's son Michael to play young Tony in ‘Sopranos’ prequel film ‘The Many Saints of Newark’ |author = Amy Kuperinsky |date = 2019-01-23 |publisher = NJ.com |language = en}}</ref>。[[雷·李歐塔|雷·李歐塔]]在2月加入演員陣容，{{link-en|約翰·馬加羅|John Magaro}}、{{link-en|小萊斯利·奧多姆|Leslie Odom Jr.}}和Michela De Rossi在3月加入。

主要拍攝於2019年4月3日開始。<ref>{{Cite web |url = https://hnentertainment.co/update-sopranos-prequel-film-the-many-saints-of-newark-may-wrap-filming-in-august/ |title = UPDATE: ‘Sopranos’ Prequel Film ‘The Many Saints of Newark’ May Wrap Filming In August |author = Christopher Marc |date = 2019-02-16 |publisher = HE Entertainment |language = en}}</ref>
