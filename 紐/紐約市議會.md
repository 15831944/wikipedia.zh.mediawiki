'''紐約市議會'''（{{lang-en|New York City Council}}）為[[紐約市|紐約市]]的代議機構，負責監察市政機構的運作，確定土地用途以及批准城市預算，並且掌握紐約市的立法權。

紐約市議會的權力來自於{{Translink|en|New York Constitution|紐約州憲法}}、紐約州地方法政府法、市政地方自治法以及紐約市憲章。市議會的首領為紐約市議長，他由議員互選產生，其權力也主要是程序性的事項，如召集市議會的會議、確定議事日程、由議長辨公室提交要討論的法案等。

== 歷史 ==

== 議員 ==
紐約市議會有51位議員，他們的議席由51個市議會選區產生。這51個選區就是將[[紐約市|紐約市]]的5個行政區，依據相應的人口比重平均劃分而來，其中[[曼哈頓區|曼哈頓區]]10個選區、[[布魯克林區|布魯克林區]]16個選區、[[皇后區|皇后區]]14個選區、[[布朗克斯區|布朗克斯區]]8個選區、[[史坦頓島|史坦頓島]]3個選區。紐約市議會的選舉採取[[小選區制|小選區制]]，51個小選區由合資格選民以一人一票的原則，選出得票最多的候選人成為該區市議員。

== 選舉 ==
紐約市議會議員的議席由51個市議會選區產生。這51個選區就是將[[紐約市|紐約市]]的5個行政區，依據相應的人口比重平均劃分而來，其中[[曼哈頓區|曼哈頓區]]10個選區、[[布魯克林區|布魯克林區]]16個選區、[[皇后區|皇后區]]14個選區、[[布朗克斯區|布朗克斯區]]8個選區、[[史坦頓島|史坦頓島]]3個選區。紐約市議會的選舉採取[[小選區制|小選區制]]，51個小選區以一人一票的原則，得票最多的候選人則成為該區市議員。

值得注意的是，紐約市議會曾經在1936至1947年間採用[[比例代表制|比例代表制]]，此制度得到共和黨的擁護，因們們可透過此制度得到更多議席，也使支持度較低的社會主義者能夠晉身議會。但於[[第二次世界大戰|第二次世界大戰]]後，隨着美國社會對[[社會主義|社會主義]]的日漸恐懼，紐約市議會廢除了此制度。

== 職權 ==
紐約市議會負責監察市政機構的運作，確定土地用途以及批准城市預算，並且掌握[[紐約市|紐約市]]的立法權。紐約市議會下設35個委員會和6個小組委員會，比如民權委員會、公務員和勞動委員會、經濟發展委員會、教育委員會、藥品濫用小組委員會、公共住房小組委員會。

== 議員工資 ==
現時，議員的基本年資為 $148,500 美元，在2016年前段之前則為 $112,500 美元。即使議員身兼其他公職或擔當議長，他們亦不會得到賠償金。
