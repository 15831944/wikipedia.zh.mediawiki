'''伯南布哥'''（{{lang|pt|Pernambuco}}），是[[巴西|巴西]]的26个州之一，地处东北部。面积98,311.616平方千米，首府是[[累西腓|累西腓]]。伯南布哥北部是[[塞阿腊州|塞阿腊州]]和[[帕拉伊巴州|帕拉伊巴州]]，西部是[[皮奧伊州|皮奧伊州]]，其南部是[[阿拉戈阿斯|阿拉戈阿斯州]]和[[巴伊亚州|巴伊亚州]]。

伯南布哥主要有[[累西腓|累西腓]]、[[雅博阿唐|雅博阿唐]]、[[奥林达|奥林达]], [[卡鲁阿鲁|卡鲁阿鲁]]、[[保利斯塔|保利斯塔]]、[[彼得罗利纳|彼得罗利纳]]、[[卡布-迪圣阿戈斯蒂纽|卡布-迪圣阿戈斯蒂纽]]、[[维多利亚-迪圣安唐|维多利亚-迪圣安唐]]等几个大城市。较大的河流有[[圣弗朗西斯科河|圣弗朗西斯科河]]、[[卡皮贝里比河|卡皮贝里比河]]、[[伊波茹卡河|伊波茹卡河]]、[[乌纳河_(伯南布哥州)|乌纳河]]、[[帕热乌河|帕热乌河]]和[[雅博阿唐河|雅博阿唐河]]。

== 经济 ==
伯南布哥的[[经济|经济]]以[[农业|农业]]为基础。[[农作物|农作物]]主要是[[甘蔗|甘蔗]]，[[木薯|木薯]]，牛肉。[[工业|工业]]有：[[食品|食品]]，[[化工|化工]]，[[矿产|矿产]]，[[纺织|纺织]]和[[电子|电子]]。

== 气候 ==
该地属于[[热带|热带]]高温气候，洲内长期干旱。 

== 历史 ==
[[巴西历史|发现巴西]]不久以后，为了开发这个大的[[殖民地|殖民地]]，[[葡萄牙|葡萄牙]]于1534年在巴西沿海建立了十多个省份（{{lang|pt|Capitania}}），伯南布哥是其中之一。然后葡萄牙国王[[若昂三世|若昂三世]]。把伯南布哥交给[[杜阿尔特·科埃略|杜阿尔特·科埃略]]管理。这个人曾经对本国在[[印度|印度]]建立殖民地有很大的贡献。他在管理时期成立[[奥林达|奥林达]]和[[累西腓|累西腓]]两个城市，还贩卖很多[[非洲|非洲]][[奴隶|奴隶]]，并且开发了伯南布哥的[[甘蔗|甘蔗]]生产（在16世纪， 糖是非常珍贵的），所以伯南布哥成为了巴西唯一个繁荣的省份。

1630年，伯南布哥被[[荷兰|荷兰]]占领，荷兰的开发政策很大程度上影响了该地区的经济发展。1654年，葡荷之战后，荷兰被屈服，放弃这个殖民地。

直到19世纪，伯南布哥还是世界上出口糖最重要的地区。然而，由于糖的价值下降，同时巴西南部开始[[工业化|工业化]]，于是伯南布哥失去了它在本国的重要地位，成为今天巴西最穷的地区之一。
