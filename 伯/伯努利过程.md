'''伯努利过程'''是一个由有限个或无限个的[[独立|独立]][[随机变量|随机变量]] ''X''<sub>1</sub>, ''X''<sub>2</sub>, ''X''<sub>3</sub> ,..., 所组成的[[离散时间|离散时间]][[随机过程|随机过程]]，其中 ''X''<sub>1</sub>, ''X''<sub>2</sub>, ''X''<sub>3</sub> ,..., 满足如下条件：

* 对每个 ''i'', ''X''<sub>''i''</sub> 等于 0 或 1;
* 对每个 ''i'', ''X''<sub>''i''</sub> = 1 的概率等于 ''p''.

换言之，伯努利过程是一列独立同分布的[[伯努利试验|伯努利试验]]。每个''X''<sub>''i''</sub> 的2个结果也被称为“成功”或“失败”。所以当用数字 0 或 1 来表示的时候，这个数字被称为第i个试验的成功次数。

与伯努利过程相关的随机变量有：
*前 n 个试验的成功次数服从[[二项分布|二项分布]]。
*要得到 r 次成功所需要的试验次数服从[[负二项分布|负二项分布]]。
*要得到 1 次成功所需要的试验次数服从[[几何分布|几何分布]]，这是负二项分布的一个特例。

伯努利过程在[[隨機程序|隨機程序]]分類上，屬於discrete-time, discrete-value。
