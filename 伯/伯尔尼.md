'''伯尔尼'''（伯尔尼德語：'''Bärn'''、{{lang-de|'''Bern'''}}、{{lang-fr|'''Berne'''}}、{{lang-it|'''Berna'''}}、{{lang-rm|'''Berna'''}}）位於[[瑞士|瑞士]]西半部領土中央偏北之處，為僅次於[[蘇黎世|蘇黎世]]、[[日內瓦|日內瓦]]、[[巴塞爾|巴塞爾]]和[[洛桑|洛桑]]的第五大城，是[[伯恩州|伯恩州]]首府，亦是該國實際上的[[首都|首都]]。其舊城區今日已成為[[聯合國教科文組織|聯合國教科文組織]]核定的[[世界遺產|世界遺產]]。

== 歷史 ==
伯尔尼市於1191年由Zähringen的Berthold V建立，一般認為這座城市得名於一隻被擊退的熊。（這個地區經常有熊的出沒，人們可以在鄰近Nydeggbrücke的地區看到）1353年，伯恩成為聯邦瑞士的第八個州，於1405年一場大火之後徹底重建。1848年新的聯邦憲法將伯恩定為瑞士首都，或是更精確地稱為聯邦之城。

== 地理 ==
伯恩是位處瑞士的高原，毗鄰阿爾卑斯山，同時[[莱茵河|莱茵河]]支流[[阿勒河|阿勒河]]穿城而过。其山脈有Gurten和Bantiger，分別高864和947米。

== 文化 ==

伯尔尼 拥有众多的歌剧院、剧院和乐团。同时还有大量的电影院、夜总会和舞厅。另外伯恩亦設有各項博物館，例如[[伯恩歷史博物館|伯恩歷史博物館]]。

== 观光 ==
== 交通 ==
[[伯恩站|伯恩站]]位于城中心。[[伯爾尼－貝爾普機場|伯爾尼－貝爾普機場]]（Berne-Belp Airport）位于城市东南9公里处，与[[柏林|柏林]]、[[维也纳|维也纳]]、[[慕尼黑|慕尼黑]]、[[巴黎|巴黎]]通航。从[[伦敦|伦敦]]赴伯尔尼最快的路径是搭航班至[[巴塞尔|巴塞尔]]，再转火车。市内[[公共交通|公共交通]]主要由[[无轨电车|无轨电车]]、[[有轨电车|有轨电车]]、[[地铁|地铁]]和[[公共汽车|公共汽车]]组成。

== 杂项 ==
1886年，[[伯尔尼保护文学和艺术作品公约|伯尔尼保护文学和艺术作品公约]]在此签署。1905年，德国出生的[[爱因斯坦|爱因斯坦]]在伯尔尼专利局工作期间，提出了[[狭义相对论|狭义相对论]]。1907年，在專利局辦公室內，透過電梯自由墜落的想像，首次得到[[广义相对论|广义相对论]]理論核心的[[等效原理|等效原理]]。
