'''手塚治-{虫}-'''（{{bd|1928年|11月3日|1989年|2月9日|Tezuka, Osamu}}），本名'''手塚 治'''，生於[[日本|日本]][[大阪府|大阪府]]，[[漫画家|漫画家]]、[[動畫師|动画制作人]]、[[医学博士|医学博士]]，同時也是現代[[日本動畫|日本動畫]]的創立者之一，在日本漫畫界有著舉足輕重的影響力。手塚治虫是一位多產的漫畫家，其知名代表作有《[[多羅羅|多羅羅]]》、《[[藍寶石王子|藍寶石王子]]》、《[[森林大帝|森林大帝]]》、《[[原子小金剛|原子小金剛]]》、《[[怪醫黑傑克|怪醫黑傑克]]》、《[[小飞龙|小飞龙]]》和《[[三眼神童|三眼神童]]》等等，無一不是膾炙人口的暢銷作品。

==生平==
1928年生於日本大阪府丰能郡丰中町，由於生於[[明治天皇|明治天皇]]誕辰，所以本名为'''手塚治'''；因自幼喜爱昆虫，故于1939年取了「手冢治虫」的笔名，把他的原名「治」改成日文中發音很像的[[步行虫|步行虫]]的日文名称<ref>他的原名「治」（osamu）在日文中發音很像的[[步行虫|步行虫]]（Ground Beetle）的日文名称（[[日语罗马字|日语罗马字]] osamushi）。于是他把其筆名写作「-{治虫}-」(讀音與本名相同)</ref>。[[汉字|汉字]]写作「-{治虫}-」。以後「-{手塚治虫}-」就成為了他的筆名。同时，他以医学家身分的名義仍使用原名'''手塚治'''。他也是日本第一位導入助手制度與企業化經營的漫畫家，對於日後日本漫畫的啟蒙有著極大的推動作用。

1945年進入[[大阪大学|大阪大学]]附属医学专门部，1951年畢業，隔年通過醫師國家考試。1961年於奈良縣立醫科大學以「在異型精子細胞中膜構造的電子顯微鏡研究」論文获得医学博士学位。自1946年起在《少国民新闻》（现《每日小学生新闻》）上连载[[四格漫画|四格漫画]]《小马的日记》，从此走上漫画之路。1947年，根据酒井七马原作改编的漫画《新宝岛》发行，该作将大量电影拍摄法引入漫画，一经推出就受到欢迎，人们由此开始接受"新漫画"的概念。此后陆续创作出[[森林大帝|森林大帝]]、[[原子小金剛|原子小金剛]]、[[怪醫黑傑克|怪醫黑傑克]]、[[小飞龙|小飞龙]]、[[三眼神童|三眼神童]]等大量脍炙人口的长篇作品，屡次获得各种奖项，其中的《蓝宝石王子》是公认的日本第一部少女故事漫画。1967年开始连载的探讨生命奥秘的作品《[[火鸟|火鸟]]》，是其创作生涯的巅峰之作，至他临终时，作品最后一章《现代篇》仍未完成。1961年成立「手冢治虫製作动画部」，翌年以「[[虫制作公司|虫制作公司]]」的名义开始活动，日本第一部多集TV动画[[原子小金剛|原子小金剛]]、第一部彩色多集TV动画[[森林大帝|森林大帝]]、第一部长达2小时的TV动画特辑《100万年地球之旅 熊猫BOOK》等一大批优秀的动画作品均诞生于此。晚年时制作的各种实验性动画在国际社会上得到很高的评价。

1989年2月9日，手冢治虫因[[胃癌|胃癌]]于日本东京都千代田区麹町半藏门医院逝世，享年60岁。

== 年表 ==
* 1928年 - [[大阪府|大阪府]][[丰能郡|丰能郡]]丰中町（现在的[[豐中市|丰中市]]）出生。
* 1933年 - 5岁时，搬到[[兵库县|兵库县]][[川邊郡|川邊郡]]小滨村（现在的[[寶塚市|宝塚市]]）居住。
* 1935年 - 入讀[[池田師範附屬小學|池田師範附屬小學]]（今[[大阪教育大學|大阪教育大學]]附屬池田小學）。
* 1941年 - 入讀[[大阪府立北野中學|大阪府立北野中學]]（今大阪府立北野高級中學）。
* 1945年 - 於北野中學畢業，入讀[[大阪帝國大學|大阪帝國大學]]（今[[大阪大學|大阪大學]]）附屬醫學專門學部。
* 1946年 - 《小马日记》於1月4日初次於《[[少國民新聞|少國民新聞]]》連載。
* 1947年 - 與漫畫家[[酒井七馬|酒井七馬]]共同發表二百頁長篇漫畫《新金銀島》，廣獲好評。
* 1951年 - 留級1年，於大阪大学附屬醫學專門學部畢業。
* 1952年 - 取得醫師執照。
* 1953年 - 搬到[[東京都|東京都]][[豐島區|豐島區]]。
* 1959年 - 與[[岡田悦子|岡田悦子]]結婚。
* 1961年 - 在[[奈良县立医科大学|奈良县立医科大学]]取得[[医学博士|医学博士]][[学位|学位]]；同年，長子[[手塚真|手塚真]]出生。
* 1963年 - 最早的[[日本电视动画|日本电视动画]]《[[原子小金剛|原子小金剛]]》于1月1日在[[富士电视台|富士电视台]]开播。
* 1964年 - 長女[[手塚留美子|手塚留美子]]出生。
* 1965年 - 最早的日本彩色电视动画《[[森林大帝|森林大帝]]》（{{lang|ja|ジャングル大帝}}）于10月6日在富士电视台开播。
* 1973年 - 虫制作公司倒闭；同年，《[[怪醫黑傑克|怪醫黑傑克]]》开始连载。
* 1975年 - 《[[怪醫黑傑克|怪醫黑傑克]]》获得第4次[[日本漫画家协会|日本漫画家协会]]奖特别优秀奖。
* 1989年 - 因[[胃癌|胃癌]]去世於[[半藏門醫院|半藏門醫院]]，享年60岁，[[法號|法號]]為「伯藝院殿覺圓蟲聖大居士」。死後獲頒「勳三等[[瑞寶章|瑞寶章]]」。
* 1990年 - 全部[[作品|作品]]获得第19次日本漫画家协会奖[[文部大臣|文部大臣]]奖。
* 1994年 - 兵库县宝塚市的[[宝塚市立手塚治虫纪念馆|宝塚市立手塚治虫纪念馆]]开幕。

== 成就与影响 ==
=== 漫画之神 ===
手塚治虫对现代日本[[漫画|漫画]]影响巨大的主要影响如下：
* 1947年《新宝岛》－发明以[[电影|电影]]技巧来画漫画。采用[[变焦|变焦]]、[[广角|广角]]、[[俯视|俯视]]等电影技术绘图，以[[分镜头|分镜头]]的方式连接。摆脱以[[四格漫画|四格漫画]]和传统连环画的绘画方式。这种手法沿用至今。成为漫画的構图标准。
* 1953年《[[藍寶石王子|藍寶石王子]]》－被公认的第一部[[少女漫画|少女漫画]]，並奠定[[美少女|美少女]]的常見造型特徵（[[萌|萌]]），如大眼睛和圓臉等。
* 1961年－“虫制作公司成立”－奠定了日本動畫界的发展方向。当时，美国动画界巨头[[迪士尼|迪士尼]]公司采用的是高成本、大制作类似“[[好莱坞|好莱坞]]”一样的制作方式。耗费庞大的资金、时间、人力才能制作一部作品。甚至一部动画就可以使一个工厂破产。而手塚治虫认为：只要故事好，就算是两张纸片也一样可以吸引人。着力刻画人物的内心，极大限度地节约制作成本。配合大量预定义的动作和CG来开发作品。最著名的就是口动、眨眼系统：眨眼三帧、说话三帧的模式。这种方式如今仍然被日本動畫界使用。甚至成为[[冒險遊戲|AVG游戏]]製作根本。至此：日本動畫成为能与美國動畫分庭抗禮的巨匠。美國動畫特点走大制作、大成本的道路；日本動畫则採用小成本、快产出的方式。
* 1963《原子小金剛》－第一部日本国产電視動畫，当时的收视率超过四成。这部電視動畫作品其实已经是比较完善的日本動畫的雏形。甚至有很多看来是习惯的东西也被保留下来，比如：一周播一次动画。
* 1969年的'''海王子'''是日本第一部正統西式[[奇幻|奇幻]]漫畫，並成為第一部公認有深度的動畫原著，並發掘動畫監督新星[[富野喜幸|富野喜幸]]。

=== 作品与思想 ===
手塚治虫是多產的漫畫家，一生畫了約450部漫畫（一說700部），同時作品豐富多樣：兒童生活、科幻、偵探、歷史、傳說，文學、醫學、宗教、音樂、哲學等，除此之外還有數篇醫學論文。這些作品幾乎都影響後來的漫畫家的創作題材，不過這些作品取材和來源分為四大類：
# 其它動漫作品影響：手塚治虫閱讀過大量的東西方動畫、漫畫作品，並受許多漫畫、動畫大師，比如迪士尼、萬籟鳴等的影響。
# 社會環境：手冢治蟲曾受到[[二戰|二戰]]和日本思想變化的影響，作品常常凝重、深邃、帶有「求索」的心情，呼籲和平、反對戰爭。前者比如《佛陀》，後者比如《三個阿道夫》 這些作品在日本動漫乃至世界動漫都是非常少見，但的確也彌足珍貴。
# 小說：手塚治虫改編不少的世界名著，比如《新[[浮士德|浮士德]]》、《[[一千零一夜|一千零一夜]]》、《三隻眼》等。經改編的作品有的保持原貌，但大多加上作者的思想。此外亦曾受梵蒂岡教廷之邀製作《舊約 聖書物語》，除了動畫，[[集英社|集英社]]並於1994年發行漫畫版，全3卷。
# 職業和興趣：手塚治虫是醫學博士，喜歡昆蟲。於是創作《人間昆蟲記》、《BLACK JACK》（中文譯：《怪醫黑傑克》）等作品

=== 影响 ===
手塚治虫对日本漫画家们的影响巨大而深遠，幾乎所有的日本漫画家都是受手塚治虫作品启蒙，而當今許多日本青年也以他為精神支柱，投身漫畫家這一職業。比如[[鸟山明|鸟山明]]就曾经说过：儿时经常涂鸦老师（手塚治虫）的人物。
[[沙耶之歌|沙耶之歌]]的故事与手塚治创作的漫画[[火之鸟|火之鸟]]的复活篇有不少相似之处，而游戏主角某段对白亦有间接地提及火鸟。
* 手塚治虫的漫画的总销量已经超过1亿，许多形象（比如原子小金剛、[[黑傑克|黑傑克]]），不仅受到日本、亚洲动漫爱好者的喜爱。而且成为世界动漫明星。
* 手塚治虫对中国非常友好，称中国漫画家[[万籁鸣|万籁鸣]]为兄。手塚治虫漫画《[[原子小金剛|原子小金剛]]》成为最先进入中国动漫界的日本动画片之一。2000年，北京举办手塚治虫纪念馆（四天）的活动。
* 影響[[藤子不二雄|藤子不二雄]]、[[石之森章太郎|石之森章太郎]]、[[赤塚不二夫|赤塚不二夫]]、[[水野英子|水野英子]]、[[横山光輝|横山光輝]]、[[松本零士|松本零士]]、[[永井豪|永井豪]]、[[古谷三敏|古谷三敏]]、[[大友克洋|大友克洋]]、[[辰巳嘉裕|辰巳嘉裕]]、[[萩尾望都|萩尾望都]]、[[里中满智子|里中满智子]]、[[諸星大二郎|諸星大二郎]]、[[浦澤直樹|浦澤直樹]]、[[寺澤武一|寺澤武一]]、[[安達充|安達充]]等漫畫家，他們多數是在學生時代讀過手塚治虫的漫畫，感受到強烈的衝擊並開始走向漫畫家一途。

=== 死后评价 ===
手塚治虫是在漫畫桌前，握著畫筆死去的。据说手塚治虫弥留时曾喃喃自语：“给我铅笔”，但这种说法并没有得到完全的确定。《火鸟》是手塚治虫未完成的作品。
* 横山光辉著有《手塚治虫漫画大全》，大和书房《手塚治虫的世界》
* 菲律宾著名学者也对手塚治虫作出过很高的评价：“手塚治虫的作品里充满东方的智慧，以及对未来的思索。”
1990年《消除岐视黑人会》因为《森林大帝》中将土著描写成很有攻击性而进行批评，之后的单行本加上解释的文字。（竹内オサム『戦後マンガ50年史』）
