'''手尾蟲'''（[[學名|學名]]：{{lang|la|'''''Cheirurus'''''}}）是生存於早[[奧陶紀|奧陶紀]]至中[[泥盆紀|泥盆紀]]的一[[屬|屬]][[三葉蟲|三葉蟲]]，生活在[[海底|海底]]。這類三葉蟲最明顯的特徵在於其[[胸部|胸部]]與尾甲邊緣的尖刺，以及外觀特殊的頭甲。手尾蟲尾甲上的刺可能有助於牠們固定在海底的軟沙上，不過現今某些學者根據手尾蟲身體的弧度與其圓潤的頭甲，推測牠們是會[[游泳|游泳]]的，而非只會在海底爬行，其[[圓|圓]][[球_(數學)|球]]狀的[[頭部|頭部]]可能內含[[脂肪|脂肪]]，幫助其在海中浮升，不過大多數會游泳的三葉蟲都具有大型的[[眼睛|眼睛]]，手尾蟲的眼睛卻很小，因此有些學者認為牠們平常是棲息在海底的，只有在逃避天敵等特殊狀況時才會游泳。<ref name="Cheirurus">{{Cite web |url = http://tomleetaiwan.pixnet.net/blog/post/32085654-%E4%B8%89%E8%91%89%E8%9F%B2%E5%8C%96%E7%9F%B3%E6%A8%99%E6%9C%AC---cheirurus-(crotalocephalina)-(crotal |title = 三葉蟲化石標本 - Cheirurus (Crotalocephalina) (Crotalocephalus) sp. 手尾蟲屬三葉蟲 @ Tomm Happy Days :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-16 }}</ref>
