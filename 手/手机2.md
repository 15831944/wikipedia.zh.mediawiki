'''手机2'''，由2003年电影《[[手机_(电影)|手机]]》原班人马拍摄的一部电影，于2018年立案、拍摄，导演[[冯小刚|冯小刚]]，编剧[[刘震云|刘震云]]。暂未上映。

== 拍摄 ==
2016年3月，[[华谊|华谊]]H计划发布会上，冯小刚作为[[美拉影业|美拉影业]]负责人上台介绍新片计划时曾重点推介《手机2》和《歌声离我远去》（既2017年上映的电影《[[芳华|芳华]]》），称《手机2》“既是一部[[喜剧|喜剧]]也是一部[[恐怖片|恐怖片]]”<ref>{{cite web |title=冯小刚预计今年开拍《手机2》 探讨科技与人的关系 既是喜剧片也是恐怖片 |url=http://news.mtime.com/2018/02/20/1578287.html |website=Mtime时光网 |date=2018-02-20}}</ref>。2018年3月，《手机2之朋友圈》出现在[[广电总局|广电总局]]最新公布的剧本（梗概）备案、立项公示通知中。根据备案公示，编剧为[[刘震云|刘震云]]，主角仍为《手机》中[[葛优|葛优]]饰演的严守一，但身份从电视主持人变为网站“朋友圈”的主持人<ref>{{cite news |title=冯小刚《手机2之朋友圈》立项曝剧情 |url=http://news.mtime.com/2018/03/27/1579292.html |publisher=Mtime时光网 |date=2018-03-27}}</ref>。根据报道，该片拍摄已经[[杀青|杀青]]<ref>{{cite web |title=《手机2》杀青，忍了61天后，冯小刚终于反击崔永元！ |url=https://finance.ifeng.com/a/20180712/16381337_0.shtml |website=凤凰网-城市金融网 |date=2018-07-12}}</ref>，计划2019年上映<ref>{{cite web |title=华谊H计划公布片单 "手机2"预计19年上映 |url=http://news.mtime.com/2018/06/16/1581523.html |website=Mtime时光网 |date=2018-06-16}}</ref>。

== 早期宣传与争议 ==
2018年5月10日晚，冯小刚在其微博公布《手机2》概念海报既剧组人员照片，包括编剧刘震云，主演[[葛优|葛优]]、[[范冰冰|范冰冰]]、[[张国立|张国立]]、[[徐帆|徐帆]]、[[范伟|范伟]]在内的电影《手机》原班人马悉数回归<ref>{{cite web |author1=苏姗 |title=冯小刚《手机2》开拍 葛优范冰冰等原班人马回归 |url=http://www.xinhuanet.com/ent/2018-05/11/c_1122815644.htm |website=新华网 |date=2018-05-11}}</ref><ref>{{cite web |title=冯小刚《手机2》开机，崔永元怒怼 葛优等原班人马回归 崔永元拒绝被“影射” |url=http://news.mtime.com/2018/05/11/1580487.html |website=Mtime时光网 |date=2018-05-11}}</ref>。5月11日，主持人[[崔永元|崔永元]]在其微博公布其4月7日与刘震云的短信记录，短信中崔表示不希望刘使用“手机2”命名，以避免引起不必要的联想和议论，崔永元曾因电影《手机》中情节映射的问题受到不好的影响，刘称新作品将命名为“朋友圈”，而冯小刚公布的海报中却仍旧使用了“手机2”的命名，崔在这条微博评论称“冯小刚是渣子大家都知道，刘震云变成渣子速度偏快了一些<ref>{{Cite web|url=https://news.dahe.cn/2018/06-01/318520.html|title=冯小刚宣布《手机2》开拍 崔永元得知后大骂"渣子"|accessdate=2018-10-04|work=大河网}}</ref>。”其后，崔永元在其微博曝光了[[范冰冰偷税漏税事件|范冰冰的阴阳合同问题]]<ref>{{cite web |author1=苏三 |title=崔永元详解和《手机2》仇怨 又为何针对范冰冰 |url=http://ent.sina.com.cn/m/c/2018-05-30/doc-ihcffhsv9352805.shtml |website=新浪娱乐 |date=2018-05-30 |access-date=2018-10-04 |archive-url=https://web.archive.org/web/20180612140037/http://ent.sina.com.cn/m/c/2018-05-30/doc-ihcffhsv9352805.shtml |archive-date=2018-06-12 |dead-url=no }}</ref>，后经过税务部门调查，范冰冰及其担任法定代表人的企业因偷逃税款遭巨额处罚<ref name="新华网1003">{{cite news |title=税务部门依法查处范冰冰“阴阳合同”等偷逃税问题 |url=http://www.xinhuanet.com/legal/2018-10/03/c_129965300.htm |agency=新华社 |publisher=新华网 |date=2018-10-03 |access-date=2018-10-04 |archive-url=https://web.archive.org/web/20181003203810/http://www.xinhuanet.com/legal/2018-10/03/c_129965300.htm |archive-date=2018-10-03 |dead-url=no }}</ref>。
