'''手語翻譯'''（'''{{lang-en|Sign language interpretation}}'''）是指翻譯員以[[手語|手語]]、[[口語|口語]]為交流手段，為聾人和健聽人士之間進行傳譯服務。

== 歷史 ==
根據[[香港中文大學手語及聾人研究中心|香港中文大學手語及聾人研究中心]]資料顯示，以前的香港聾人群體並沒有一套有系統的手語，他們自創一些動作，再輔以肢體動作來與別人溝通，直至後來有一對[[上海|上海]]聾人夫婦來港，自資成立華僑聾啞學校教授手語<ref name="溝通橋梁">{{cite web |language=zh-hk | url=http://stedu.stheadline.com/sec/sec_news.php?aid=16336&cat=8&subcat=21 | title= 溝通橋梁 助求診求學 手語翻譯員 打破聽說障礙 | publisher=[[星島日報|星島日報]] | date=2016-11-09 | accessdate=2017-05-13}}</ref><ref>{{cite web |language=zh-hk | url=https://news.mingpao.com/pns/dailynews/web_tc/article/20170219/s00005/1487440672400 | title= 無定向學堂﹕了解無聲語言——手語 捍衛被忽略聲音 |publisher=[[明報|明報]] | date=2017-02-19 | accessdate=2017-05-13}}</ref>。

手語有語法、有詞彙；因歷史文化的演變，更發展出獨特的語言系統。在1960年代，香港已發展出一套手語<ref>{{cite web |language=zh-hk | url=http://hk.apple.nextmedia.com/news/art/20140917/18869185 | title= 恐怖日佔回憶　用手講更血淋淋 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] | date=2014-09-17 | accessdate=2017-05-13}}</ref>。

聾人機構「[[龍耳|龍耳]]」的手語翻譯員指出，香港現時流通的手語主要有兩種，一種是中文手語，意思是直接將整句句字，逐一逐一地順序打出，而另一種是自然手語，以「重點放在前，問句量詞放在後」為原則，方便以視覺作主要接收訊息的聾人迅速明白說話的內容<ref>{{cite web |language=zh-hk | url=http://hk.apple.nextmedia.com/realtime/supplement/20170209/56276998 | title= 
【潮語手語】考吓你「你識條鐵」 點樣打？ |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] | date=2017-02-09 | accessdate=2017-05-13}}</ref>。

根據[[香港統計處|香港統計處]]資料，香港有15萬5千多位聾人及弱聽，當中有部份是以手語作溝通，但手語翻譯員卻只有50多名，出現人手緊絀的情況<ref name="溝通橋梁" />。

== 工作內容 ==
* 為聾人提供將口語翻譯成手語的服務 
* 為健聽人士提供將手語翻譯成口語的服務
* 直接用手語和聾人進行交流

== 入職要求 ==
[[香港社會服務聯會|香港社會服務聯會]]、香港復康聯會和[[勞工及福利局|勞工及福利局]]康復諮詢委員會於2015年共同設立《[[香港手語翻譯員名單|香港手語翻譯員名單]]》，列載在香港可提供手語翻譯服務的人士。手語翻譯員只要在申請日起計，過去兩年內提供不少於200小時的手語翻譯服務，並持有由受僱機構簽發的證明文件，便可把個人資料加入此名單中，與註冊的翻譯員看齊，為聾人提供翻譯服務<ref>{{cite web |language=zh-hk | url=http://hk.on.cc/hk/bkn/cnt/news/20160317/bkn-20160317121015706-0317_00822_001.html | title= 聾人的一天　接駁有聲世界困難重重 |publisher=東網電視 | date=2016-03-17 | accessdate=2017-05-13}}</ref><ref>{{cite web |language=zh-hk | url=http://hk.on.cc/hk/bkn/cnt/news/20160318/bkn-20160318103553931-0318_00822_001.html | title= 手語不統一政府不支援　翻譯認可路難走 |publisher=東網電視 | date=2017-05-13 | accessdate=2016-03-31}}</ref>。

== 職業慨況 ==
時薪大約$150至$450，視乎服務的類別和不同機構的制度而定<ref name="溝通橋梁" />。

出路︰
* 受聘於聾人機構或以聾人為服務對象的社區中心
* 為法庭提供兼職傳譯服務
* 於電視台及電子媒體不定期協助翻譯節目
* 於求職機構協助聾人求職。
