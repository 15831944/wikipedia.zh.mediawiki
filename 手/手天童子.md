《'''手天童子'''》是 [[永井豪|永井豪]]在『[[週刊少年Magazine|週刊少年Magazine]]』（[[講談社|講談社]]）1976年9月5日号到1978年4月30日号連載的[[漫画|漫画]]作品。

== 主要登場人物 ==

; 手天童 子郎（しゅてんどう じろう）
: 声：[[堀川りょう|堀川亮]]

; 柴 竜一郎（しば りゅういちろう）
: 声：[[池水通洋|池水通洋]]

; 柴 京子（しば きょうこ）
: 声：[[高木早苗|高木早苗]]

; 白鳥 美雪（しらとり みゆき）
