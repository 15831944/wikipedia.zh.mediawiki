'''固戍站'''是[[深圳地鐵|深圳地鐵]][[深圳地鐵1号綫|1号綫]]的一個車站，位於[[寶安大道|寶安大道]]與[[固戍二路|固戍二路]]的交叉路中，沿寶安大道路中呈“一字形”佈置。車站於2011年6月15日隨1號綫續建段的建成而投入使用。

==利用状况==
本站周边为城中村和工业区，但随着周边城中村进行公寓化住房改造，选择在该地居住的市民增多，车站客流量连年增长。2017年车站日均客流量为6.8万人次，2018年已突破9.4万人次。由于站厅空间有限，车站在工作日7:00至9:30于出口实施人流管制，导致早高峰期间进站乘客大排长龙。<ref>{{cite web |url=http://static.scms.sztv.com.cn/ysz/zx/szdyxc/27950148.shtml |title=深圳地铁5号线突发故障，上班族纷纷迟到！还有1号线固戍站更虐心！ |publisher=[[深圳广电集团|深圳广电集团]] |date=2018-07-24 }}</ref>

==历史==
本站为1号线续建工程机场段的一个车站，在2007年深圳市政府发布的《深圳特区近期轨道建设交通影响评估及交通疏解规划要点》中曾同时出现“航城站”和“固戍站”两个站名<ref>{{cite web |url=http://www.sz.gov.cn/cn/xxgk/xwfyr/wqhg/fbh_26/fbg/200712/P020081024352499579841.doc |title=深圳特区近期轨道建设交通影响评估及交通疏解规划要点 |publisher=深圳市人民政府 |date=2007-12-12 }}</ref>，站名最终在2008年4月公布的命名方案中确定为固戍站。<ref>[http://www.szplan.gov.cn/main/ghdt/tzgg/200804230217672.shtml 关于我市近期建设地铁线路及站点命名的通告] {{webarchive|url=https://web.archive.org/web/20080430203949/http://www.szplan.gov.cn/main/ghdt/tzgg/200804230217672.shtml |date=2008-04-30 }}，2008年4月23日，深圳市规划局</ref>1号线续建工程机场段土建工程于2008年8月启动。<ref>{{cite web |url=http://www.szmc.net/02constuction/pop.jsp?infoID=381 |title=1号线(罗宝线)续建工程进度计划 |publisher=深圳市地铁有限公司 |date=2008-04-30 |archive-url=https://web.archive.org/web/20080601035827/http://www.szmc.net/02constuction/pop.jsp?infoID=381 |archivedate=2008-06-01 |deadurl=yes}}</ref>2009年10月28日，固戍站主体结构封顶。<ref>{{cite news |url=http://news.sznews.com/content/2009-10/28/content_4145312.htm |title=深圳地铁1号线机场段固戍车站主体结构封顶 |publisher=深圳新闻网 |date=2009-10-28 |archive-url=https://web.archive.org/web/20091217102326/http://news.sznews.com/content/2009-10/28/content_4145312.htm |archivedate=2009-12-17 |deadurl=yes}}</ref>2011年6月15日，本站随1号线续建工程机场段开通而启用。
