'''固定客'''（'''Solidor'''），[[台灣|台灣]]獨立[[樂團|樂團]]，成員共四人，由[[劉鈞輝|導輝]]（[[主唱|主唱]]）、[[陳昭信|阿信]]（[[吉他手|吉他手]]）、[[李炫徵|阿炫]]（[[貝斯手|貝斯手]]）和[[施承甫|小P]]（團長暨[[鼓手|鼓手]]）組成。

== 樂團簡介 ==
固定客發跡於台灣台中，始於2004年11月由原始成員導灰與阿信組成。<BR/>
<BR/>
2006年在經過幾代成員的替換後成員終於底定，成員編制為主唱、2名吉他、貝斯、鼓、DJ六人。<BR/>
憑藉發佈在<ref>[http://tw.streetvoice.com/solidor/songs/10802/]</ref>音樂平台Streetvoice上靈感來自日本知名漫畫的同名歌曲《20世紀少年》開始大受歡迎<BR/>
並於<ref>{{cite web |url=http://tw.streetvoice.com/tkrock/articles/1471/ |title=存档副本 |accessdate=2014-05-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20140520005305/http://tw.streetvoice.com/tkrock/articles/1471/ |archivedate=2014-05-20 }}</ref>06年的台客搖滾嘉年華中獲得全國總優勝<BR/>
在該年與友團LazyLady在自己設立的音樂廠牌"異想音樂"下獨立製作發行合輯單曲《MIXTUREROCK120%》<BR/>
同年度除了相當活躍在各大比賽與活動中，與友團LAZYLADY所合組的阿雷固更在該年底拿下第十九屆[[Yamaha全國熱音大賽|Yamaha全國熱音大賽]]總決賽冠軍，<BR/>
隔年亦代表台灣至馬來西亞出賽YAMAHA亞洲區大賽；同年受邀為07年[[台客搖滾|台客搖滾]]表演團體。<BR/>
<BR/>
07年中旬固定客因團員兵役問題面臨停團；後與團員間理念上有了不同的想法，嚴重的爭執使固定客幾乎形同解散<BR/>
在與成員爭吵期間主唱導輝曾試著加入新團員以重組固定客，但僅僅活動了一段時間後又再度停團。<BR/>
<BR/>
2009年8月由阿信與小P召集固定客重組，重組後編制縮減為主唱、吉他、貝斯、鼓四人<BR/>
<ref>[https://www.facebook.com/events/183653618837/?ref=22]</ref>於2009年12月26日舉行復出演唱會正式回歸舞台；<BR/>
<BR/>
<ref>{{cite web |url=http://news.chinatimes.com/2009Cti/Channel/Showbiz/showbiz-news-cnt/0,5020,110511+132010071000935,00.html |title=存档副本 |accessdate=2014-05-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20140520003721/http://news.chinatimes.com/2009Cti/Channel/Showbiz/showbiz-news-cnt/0,5020,110511+132010071000935,00.html |archivedate=2014-05-20 }}</ref>2010年入圍[[貢寮國際海洋音樂祭|貢寮國際海洋音樂祭]]海洋大賞決賽並於該決賽得到評審團大賞，亦於<ref>{{cite web |url=http://tw.streetvoice.com/wannaplaymusic/articles/861024/ |title=存档副本 |accessdate=2014-05-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20140520005208/http://tw.streetvoice.com/wannaplaymusic/articles/861024/ |archivedate=2014-05-20 }}</ref>該年加入想玩音樂，<ref>{{cite web |url=http://www.wannaplaymusic.com.tw/record/index.php/releases-m/99-solidier-ep1 |title=存档副本 |accessdate=2014-05-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20140520001011/http://www.wannaplaymusic.com.tw/record/index.php/releases-m/99-solidier-ep1 |archivedate=2014-05-20 }}</ref>並發行首張正式單曲《[[SELF|SELF]]》<BR/>
；首張單曲發行後固定客大多與友團LAZYLADY合體阿雷固進行巡迴或是配佩演出，<ref>{{cite web |url=http://www.wannaplaymusic.com.tw/record/index.php/releases-m/101-solidier-a01 |title=存档副本 |accessdate=2014-05-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20140520001402/http://www.wannaplaymusic.com.tw/record/index.php/releases-m/101-solidier-a01 |archivedate=2014-05-20 }}</ref>2012年12月發行第二張單曲《[[HERE_WE_ARE|HERE WE ARE]]》。<BR/>
<BR/>
2012年11月18日《十八銅人搖滾英雄會》<BR/>
表演前夕於<ref>[https://www.facebook.com/Solidortw/photos/a.10150139504285252.403821.316402645251/10152159477620252/?type=1]</ref>官方臉書公佈該場受邀表演為此年度最後一場演出。<BR/>
<BR/>
2012年11月18日之後，雖無任何正式公告但已無任何活動與演出，形同停團。<BR/>
<BR/>
現該團成員已各自各組樂團。<BR/>

==音樂風格==
早期為饒舌搭配樂團及DJ的表演風格<ref>{{cite web |url=http://www.wannaplaymusic.com.tw/record/index.php/releases-m/99-solidier-ep1 |title=存档副本 |accessdate=2014-05-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20140520001011/http://www.wannaplaymusic.com.tw/record/index.php/releases-m/99-solidier-ep1 |archivedate=2014-05-20 }}</ref><BR/>
固定客曲風融合EMO、FUNK 、RAP、METAL等等<ref>{{cite web |url=http://www.wannaplaymusic.com.tw/record/index.php/entertainer-m/80-solidor |title=存档副本 |accessdate=2014-05-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20140520003051/http://www.wannaplaymusic.com.tw/record/index.php/entertainer-m/80-solidor |archivedate=2014-05-20 }}</ref><BR/>
創作大多為批判社會、生活環境的內容。<BR/>

==樂團歷程==

*2005年7月 貢寮國際海洋音樂節

*2005年10月 樂團成員DJ-REX因母喪家中事務繁重離團
*2005年11月 整年度100場現場演出達成
