'''维新党'''（{{jpn|j=維新の党|hg=いしんのとう|rm=Ishin no Tō}}）是[[日本|日本]]的一个已不存在的政党，由[[日本维新会|日本维新会]]和與[[连结党|连结党]]（{{lang|ja|結いの党}}）于2014年9月21日合并而成<ref>[http://www.asahi.com/articles/ASG9P5R2CG9PPTIL00V.html 「維新の党」が結党大会　衆参５３議員で船出]</ref>，结党时为日本第二大在野党。

== 概要 ==
在日本維新會共同代表（黨魁）之一的[[石原慎太郎|石原慎太郎]]因志趣相異於2014年8月1日另組[[次世代党|次世代党]]後，割席的「『新』維新會」在同日與連結黨以兩者合組新政党的前提下進行交涉，此前另一共同代表[[橋下徹|橋下徹]]曽對兩者合作表達出濃厚興趣。此後兩者藉共組[[衆議院_(日本)|衆議院]]黨團進行磨合，並協定於同年9月21日召開創党大会，翌22日透過[[大阪府|大阪府]]{{link-ja|選舉管理委員會 (日本)|選挙管理委員会|選舉管理委員會}}向[[總務大臣|總務大臣]]提出申請。

成立時，維新會與連結黨兩者均推舉党内一人為共同代表、並決定於大阪、東京兩地同設總部。而實際流程方面，連結黨相對做出較大讓步，登記主體實際上主要總部為大阪總部、黨魁為橋下徹，且日本維新會是為續存政黨易名，連結黨則解散併入，某程度上原維新會一方獲得較多優勢。<ref>{{cite news |newspaper=[[朝日新聞|朝日新聞デジタル]] |title=橋下・江田氏、共同代表へ　維新・結い、本拠地は大阪に |url=http://www.asahi.com/articles/ASG975743G97UTFK33W.html |date=2014-09-07 |accessdate=2014-09-21}}</ref><ref>{{cite news |newspaper=MSN産経ニュース |title=新党「維新の党」、代表者の届け出は「橋下氏」 |url=http://sankei.jp.msn.com/politics/news/140918/stt14091819140009-n1.htm |date=2014-09-18 |accessdate=2014-09-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20140920174044/http://sankei.jp.msn.com/politics/news/140918/stt14091819140009-n1.htm |archivedate=2014-09-20 }}</ref><ref>{{cite news |newspaper=MSN産経ニュース |title=橋下、江田両氏の共同代表制　維新・結い新党、党名に「維新」残す |url=http://sankei.jp.msn.com/politics/news/140907/stt14090718580008-n1.htm |date=2014-09-07 |accessdate=2014-09-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20140919101259/http://sankei.jp.msn.com/politics/news/140907/stt14090718580008-n1.htm |archivedate=2014-09-19 }}</ref>

2015年8月14日，幹事長[[柿澤未途|柿澤未途]]私下參與[[山形市|山形市長]]選舉[[民主党_(日本)|民主党]]・[[日本共産党|共産党]]推舉參選人的造勢活動，引發黨內一部反彈。<ref>{{Cite news|url = http://yamagata-np.jp/news/201508/18/kj_2015081800359.php?keyword=%E6%9F%BF%E6%B2%A2%E6%9C%AA%E9%80%94|title = 山形市長選、政党幹部相次ぎ来援 前哨戦ヒートアップ|newspaper = |publisher = 山形新聞|date = 2015-08-18|accessdate = 2015-08-29}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>其後松井一郎要求柿澤辞任被拒、加上松野袒護柿澤而加深党内對立<ref>{{Cite news |url = http://www.sankei.com/politics/news/150821/plt1508210039-n1.html|title = 柿沢幹事長辞任は「不要」 維新・松野氏| newspaper = 産経ニュース|publisher = 産経新聞|date = 2015-08-21|accessdate = 2015-08-29}}</ref>。同月27日，橋下・松井宣布辞任黨務並表明去意<ref>{{Cite news |url = http://www.sankei.com/west/news/150827/wst1508270037-n1.html|title = 橋下氏も離党へ 維新〝分裂〟…松井顧問が離党意向を表明|newspaper = 産経WEST|publisher = 産経新聞|date = 2015-08-27|accessdate = 2015-08-29}}</ref>，翌28日二人宣布另起爐灶<ref>{{Cite news |url = http://www.sankei.com/west/news/150829/wst1508290014-n1.html|title = 橋下氏「年内に新党」 大阪維新を国政政党化へ 維新分裂は必至| newspaper = 産経WEST|publisher = 産経新聞|date = 2015-08-29|accessdate = 2015-08-29}}</ref>，在宣布另籌組新政黨後，與橋下・松井友好的十数位議員表示離意<ref>{{Cite news |url = http://www.sankei.com/west/news/150829/wst1508290053-n1.html|title =「橋下氏についていく」維新の党大阪系議員が来月にも集団離党へ|newspaper = 産経WEST|publisher = 産経新聞|date = 2015-08-29|accessdate = 2015-09-01}}</ref>，另一方面松野・柿澤表示會研究與民主党合流或餘部再組的可能性<ref>{{Cite news |url = http://www.nikkei.com/article/DGXLASFS30H2M_Q5A830C1PE8000/|title = 維新・松野代表、野党再編巡り岡田氏と協議へ|newspaper = |publisher = 日本経済新聞|date = 2015-08-30|accessdate = 2015-09-01}}</ref>，自此維新党分裂確定。

橋下在10月創立新政党、並祭出限制民主党出身者参加的方針<ref>{{Cite news |url = http://www.zakzak.co.jp/society/politics/news/20150901/plt1509011538005-n1.htm|title = 維新の民主出身議員が崖っぷち　“橋下新党”元民主の参加を原則認めない方針|newspaper = ZAKZAK|publisher = [[夕刊フジ|夕刊フジ]]|date = 2015-09-01|accessdate = 2015-09-08}}</ref>，另[[馬場伸幸|馬場伸幸]]国会對策委員長、[[片山虎之助|片山虎之助]]總務会長表明參與新党意向<ref>{{Cite news|url = http://www.nikkansports.com/general/news/1532569.html|title = 橋下氏の国政出馬を条件に片山氏が大阪組新党参加|newspaper = |publisher = [[日刊スポーツ|日刊スポーツ]]|date = 2015-09-01|accessdate = 2015-09-08|deadurl = yes|archiveurl = https://web.archive.org/web/20150905002742/http://www.nikkansports.com/general/news/1532569.html|archivedate = 2015-09-05}}</ref><ref>{{Cite news|url = http://www.yomiuri.co.jp/politics/20150902-OYT1T50141.html|title = 橋下新党に18人参加意向…維新51議員調査|newspaper = |publisher = 読売新聞|date = 2015-09-03|accessdate = 2015-09-08}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。松野在9月8日藉馬場・片山離黨為由，決定解除馬場、片山及柿澤三人職務<ref>{{Cite news |url = http://www.sankei.com/politics/news/150908/plt1509080041-n1.html|title = 松野代表 馬場国対委員長と片山総務会長、柿沢幹事長に解任通告| newspaper = 産経ニュース|publisher = 産経新聞|date = 2015-09-08|accessdate = 2015-09-08}}</ref>。橋下於9月15日宣布新党名為[[大阪維新會_(2015年)|おおさか維新の会]]」（漢語圏同樣譯作「大阪維新會」）<ref>{{Cite news|url = http://www3.nhk.or.jp/news/html/20150916/k10010236391000.html|title = 橋下氏 新党名を「おおさか維新の会」に|newspaper = NHK|publisher = NHK|date = 2015-09-16|accessdate = 2015-09-16|deadurl = yes|archiveurl = https://web.archive.org/web/20150916011018/http://www3.nhk.or.jp/news/html/20150916/k10010236391000.html|archivedate = 2015-09-16}}</ref><ref>{{cite news|url=http://www.sankei.com/politics/news/150916/plt1509160046-n1.html|title=大阪系新党名は「おおさか維新の会」|publisher=[[產經新聞社|產經新聞社]]|date=2015-09-16|accessdate=2015-10-12}}</ref>。

2016年2月22日，維新黨與[[民主黨_(日本)|民主黨]]達成基本協議，兩黨將合併成立新政黨。<ref>[http://www.fnn-news.com/news/headlines/articles/CONN00317003.html 民主・維新両代表、両党が合流し新党を結成することで一致] {{webarchive|url=https://web.archive.org/web/20160222182547/http://www.fnn-news.com/news/headlines/articles/CONN00317003.html |date=2016-02-22 }}</ref>3月27日，兩黨正式合併成「[[民進黨_(日本)|民進黨]]」。
