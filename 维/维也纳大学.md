'''維也納大學'''（{{lang-de|Universität Wien}}）位于[[奥地利|奥地利]]的首都[[维也纳|维也纳]]，始建于公元1365年，是现存最古老的[[德语|德语]][[大学|大学]]，也是中欧最大的学校之一，维也纳大学亦是目前最大的德语大学，学校共设有180个[[学位|学位]]。在奥地利，维也纳大学的也是最大的教学和研究机构，拥有近8600名员工，其中有6500名學者。截止2017年，共有[[各大學諾貝爾獎得主列表|20位校友及教职工]]曾获得[[诺贝尔奖|诺贝尔奖]]。

== 历史 ==
[[File:Wien_-_Universität_(2).JPG|thumb]]
大学的雏形最初在1365年3月12日由公爵[[鲁道夫四世|鲁道夫四世]]和他的兄弟[[阿爾布雷希特三世_(奧地利)|阿尔伯莱希特三世]]及[[利奧波德三世_(哈布斯堡)|利奥波德三世]]创建。它是当时[[神圣罗马帝国|神圣罗马帝国]]的第二所大学，同时也是世界上第二所德语大学（第一所是[[布拉格大学|布拉格大学]]），最初開辦時是仿照[[巴黎大學|巴黎大學]]的模式。1383年維也納大學經營不佳，由魯道夫四世的弟弟和阿尔伯莱希特三世捐款，改革校內行政，成為東部德語地區一流的大學。

现在的学校本部是在公元1877年至1884年间建造的。原来的校本部位于施度本门附近的Ignaz-Seipel广场，老大学教堂和[[奥地利国家科学院|奥地利国家科学院]]目前仍在该处。1878年文學院開始有條件式地招收女學生，11年後，1897年女性第一次被准许以正式生身份入校，但一开始只能被哲学院录取。其余的学院后来也一一开始招收女生：医学院1900年，法学院1919年，新教神学院1923年，到了1946年最保守的天主教神学院也终于开始正式招收女学生。

== 研究与教学 ==
目前学校共有在校生35375人，130个[[学位|学位]]。大学共拥有约6100名专职研究人员和约2000名[[教师|教师]]（往往同时也参与科研工作）。

== 结构 ==
目前的校长是Georg Winckler[[教授|教授]]。维也纳大学实行民主制度，各个学院往往拥有很大的权利。而在学院内部，所有[[学生|学生]]、[[系|系]]和[[教授|教授]]都拥有对学院一切重大事务的参与及[[投票|投票]]权。时任奥地利总理[[沃尔夫冈·许塞尔|沃尔夫冈·许塞尔]]实行将大学权力集中化的计划，将权力主要集中到正教授（系主任）手中。
2004年底著名的医学院自维也纳大学划出，成立[[维也纳医科大学|维也纳医科大学]]。学校的一些其他院系也因之作了一些调整。目前大学共拥有15个[[学院|学院]]以及2个研究中心。他们是：
