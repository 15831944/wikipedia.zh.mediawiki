'''维拉港'''（{{lang-en|'''Port Vila'''}}；{{lang-fr|'''Port-Vila'''}}）是[[瓦努阿图|瓦努阿图]]的首都，位于17°45'S, 168°18'E，约有33,700人<ref>[http://www.greatestcities.com/Oceania/Vanuatu/Port-Vila_town_state_capital.html http://www.greatestcities.com/]</ref>。拥有机场及港口，是该国经济中心。[[第二次世界大战|第二次世界大战]]时维拉港曾是[[美国|美国]]及[[澳大利亚|澳大利亚]]的空军基地。

== 歷史 ==
[[美拉尼西亞|美拉尼西亞]]人已在維拉港居住了逾千年。

在1606年，已有第一群到達該島的歐洲人，其首領是[[彼得·奎羅斯|彼得·奎羅斯]]<ref>[http://www.gospelherald.com/template/view.htm?code=fai&id=5387 基督日報]</ref>。

在19世紀，法國殖民者在維拉港設立市政局，由[[新赫布里底群島|新赫布里底群島]]管轄，在1889年宣佈自治並成為第一個實行無性別之分的普選的自治地區。當時維拉港有50名[[白人|白人]]和500名土著，但只有白人才擁有統治權。

在二次大戰期間，維拉港一度成為澳洲和美國的空軍陸地。1980年 瓦努阿图脱离法国独立建国。1984年发行自己的货币。

在1987年，一個[[氣旋|氣旋]]對維拉港造成巨大傷害，接著在2002年發生的[[地震|地震]]也對維拉港造成巨大傷害，2015年的[[氣旋帕姆|氣旋帕姆]]再度對維拉港造成重大損害。

== 經濟 ==
[[File:Port_Vila_market.jpg|thumb]]
維拉港是國家之中最重要的港口及國家經濟中心，而機場也是座落在維拉港中。

維拉港最大的工業活動是農業和漁業，旅遊業也相當重要，尤其是來自[[澳洲|澳洲]]和[[新西蘭|新西蘭]]的旅客，在1997年該兩國的旅客量達5萬人次。

維拉港是一個[[避稅港|避稅港]]，因此國外資金是相當重要的。

維拉港相當依賴[[外援|外援]]，即使近年有些外援來自[[中國|中國]]，但它們大多來自澳洲和新西蘭。例如新西蘭政府給予已受訓的醫生，並給予那些醫生的第一年部分薪金，而澳洲則給予那些在醫院工作的顧問醫生的薪金。

全國約有35.7%出口輸出自維拉港，86.9%進口進入維拉港。

== 人口 ==
維拉港約有33,700人，主要屬於[[美拉尼西亞|美拉尼西亞]]人，也有少量[[玻里尼西亞|玻里尼西亞]]人、[[亞洲人|亞洲人]]和[[歐洲人|歐洲人]]。

=== 語言 ===
[[比斯拉馬語|比斯拉馬語]]是當地人的日常語言，至於[[英語|英語]]和[[法語|法語]]，在維拉港也相當普及，其它本地語也有人使用。

=== 宗教 ===
[[基督教|基督教]]在萬那杜相當流行，其中最大的派別是[[長老教會|長老教會]]，約三分一人相信；至於[[羅馬天主教|羅馬天主教]]和[[美托尼西亞教|美托尼西亞教]]在當地也相當普及。

== 氣候 ==
維拉港屬於[[熱帶氣候|熱帶氣候]]，有[[旱季|旱季]]和[[雨季|雨季]]之分，年降雨量約2360毫米，吹東南[[信風|信風]]。

== 教育 ==
[[南太平洋大學|南太平洋大學]]位於維拉港，由十二個太平洋國家共同擁有。大學的萬那杜校園是法律學校，除了教授法律外，也會教授語言學。
