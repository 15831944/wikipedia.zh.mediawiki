'''维亚切斯拉夫·亚历克塞维奇·扎伊采夫'''（{{lang-ru|''Вячеслав Алексеевич зайцев''}}；{{bd|1952年|11月12日|||catIdx=ZЗайцев,Вячеслав}}）是前[[蘇聯|蘇聯]]的[[俄罗斯苏维埃联邦社会主义共和国|俄羅斯]]運動員，[[蘇聯男排|國家排球隊]]的二傳手。扎伊采夫生於[[列寧格勒|列寧格勒]]，1976年成為國家排球隊成員<ref name=r1/>。曾於[[1976年夏季奥林匹克运动会|1976年]]、[[1980年夏季奥林匹克运动会|1980年]]及[[1988年夏季奥林匹克运动会|1988年]]三度代表蘇聯參加奧林匹克運動會的排球比賽<ref name=r1/>。

== 簡歷 ==
*  1970年代末-80年代初[[蘇聯男排|蘇聯男排]]'''「五連冠」'''（'''1977年世界盃'''、'''1978年世界男排錦標賽'''、'''[[1980年夏季奧林匹克運動會排球比賽|1980年莫斯科奧運會]]'''、'''1981年世界盃'''、'''1982年世界男排錦標賽''') 主力二傳手。

== 家庭 ==
维亚切斯拉夫·亚历克塞维奇·扎伊采夫的太太是前蘇聯[[游泳|游泳]]代表隊成員、前女子200米蛙泳紀錄保持者[[伊莉娜·波兹尼亚科娃|伊莉娜·波兹尼亚科娃]]（{{lang|ru|''Ирина Позднякова''}}；{{lang|en|''Irina Pozdnyakova''}}）<ref name=r6/><ref name=r2/><ref name=r3/>。二人育有一女一子，均有意大利籍<ref name=r4/><ref name=r5/>：
* 女兒安娜（1975年─）：1993年與一名意大利人結婚，取得意大利籍；
* 兒子[[伊凡·维亚切斯拉夫维奇·扎伊采夫|伊凡·维亚切斯拉夫维奇·扎伊采夫]]（{{lang|ru|''Иван Вячеславович Зайцев''}}；1988年─）：意大利國家排球隊運動員，在意大利出生而入籍。
