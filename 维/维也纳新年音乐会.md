'''维也纳新年音乐会'''（{{lang-de|Das Neujahrskonzert der Wiener Philharmoniker}}）开始于1939年，正式命名于1946年。

== 历史 ==

1939年12月31日，在[[维也纳|维也纳]][[金色大厅|金色大厅]]举办了首届“新年音乐会”，由[[克来门斯·克劳斯|克来门斯·克劳斯]]创办。 第2届于1941年1月1日举行，以后除1945年因战争原因停办一届外，其余都是于每年的1月1日定期在金色大厅举行。1946年，新年音乐会正式命名为“维也纳新年音乐会”。

从1955年起，音乐会由[[威利·博斯科夫斯基|威利·博斯科夫斯基]]指挥，直至1979年。自此年起，音乐会由[[洛林·马泽尔|洛林·马泽尔]]指挥，直至1986年。1987年，[[卡拉扬|卡拉扬]]成为了维也纳新年音乐会的指挥，自此往后，音乐会的指挥每一年都会更换。

1939年没有加演，资料指出直到1945年才有加演。克莱门斯·克劳斯经常随机的加演。令人惊奇的是，{{單雙書號轉換|[[蓝色多瑙河|蓝色多瑙河]]}}华尔兹是在1945年第一次演出的，而且是作为加演。最近几年经常加演的另一曲目老约翰·斯特劳斯的[[拉德茨基进行曲|拉德茨基进行曲]]，是在1946年才开始演出的，也是作为加演。直到1958年这两支曲目才作为常演项目而不是加演，然后它们就成为了音乐会的必演项目。

在闭幕的时候常有非常情况发生，一次是在1967年，博斯科夫斯基把蓝色多瑙河作为了他音乐会曲目的一部分，另一次是2005年，[[洛林·马泽尔|洛林·马泽尔]]把{{單雙書號轉換|蓝色多瑙河}}华尔兹作为了闭幕曲（省略了{{單雙書號轉換|拉德斯基进行曲}}）以作为2004年[[印度洋海啸|印度洋海啸]]殉难者的纪念。

博斯科夫斯基在1936—1979年一直是交响乐队的[[首席小提琴手|首席小提琴手]]。他从1955年—1979年一直指挥维也纳新年音乐会。1980年，洛林·马泽尔成为了首位非奥地利籍的指挥。马泽尔连续多年担任指挥以后，从1987年开始，每年选择明星指挥成为了一项传统。指挥人选一般由交响乐队的成员投票决定。

通常新年音樂會上演奏的作品主要以[[小約翰·史特勞斯|小約翰·史特勞斯]]、[[约瑟夫·施特劳斯|約瑟夫·史特勞斯]]、[[爱德华·施特劳斯|愛德華·史特勞斯]]、[[老約翰·史特勞斯|老約翰·史特勞斯]]、[[約瑟夫·蘭納|約瑟夫·蘭納]]為主，在某些特別的年份中，會加入其他作曲家的作品，如[[羅西尼|羅西尼]](1991)、[[卡爾·米夏埃爾·齊雷爾|齊雷爾]](1996、2012、2017)、[[布拉姆斯|布拉姆斯]](2003)、[[奥托·尼古拉|奧托‧尼古萊]](1992、2010、2017)等等。

== 传统 ==
1987年的新年音乐会上，当{{單雙書號轉換|拉德斯基进行曲}}响起时，观众忍不住开始鼓掌打节拍，卡拉扬转身指挥观众一幕被人们所熟知。但实际上早在博斯科夫斯基时代，在演奏{{單雙書號轉換|拉德斯基进行曲}}时就有观众跟着鼓掌了，网络上85、86年的视频也显示，马泽尔就曾转身示意让观众起身跟着鼓掌。所以认为这一传统的开端始于卡拉扬是不对的。

从现在留存的资料来看，博斯科夫斯基、马泽尔和卡拉揚都曾经进行过致辞，只是并没有在{{單雙書號轉換|蓝色多瑙河}}前面，而在1989年的新年音乐会上，当{{單雙書號轉換|蓝色多瑙河}}前奏响起时，观众的掌声打断了音樂家們的演奏。克莱伯面向观众，说道：“Die Wiener Philharmoniker und ich wünschen Ihnen：Prosit Neujahr!”<small>（[[中文|中译]]：“维也纳爱乐乐团和我敬祝大家：新年快乐！”；前半句由指挥所说，“Prosit Neujahr!”则由指挥及所有乐手一齐说出）</small>。致辞这个传统就固定了在演奏{{單雙書號轉換|蓝色多瑙河}}前，一直延续到现在。而在某些年，一些指挥则会用中文向观众们说道“新年好”（1996年，[[洛林·马泽尔|洛林·马泽尔]]；1999年，洛林·马泽尔；2002年，[[小泽征尔|小泽征尔]]；2005年，洛林·马泽尔）特别是2002年的小泽征尔只用了中文一种语言（马泽尔除了中文外还用其他语言）。

==年度指揮家的選擇==
在維也納新年音樂會的年度指揮家的評選上，主要由[[維也納愛樂樂團|維也納愛樂樂團]]成員共同評選而出的！<ref>http://www.soomal.com/doc/20100004639.htm/</ref>

基本條件則需要：

1.與[[維也納愛樂樂團|維也納愛樂樂團]]有長期合作

2.短期內有密切合作或是合作過重要的音樂會

3.受到樂團成員及維也納當地人民的好評

4.最後是[[指揮家|指揮家]]本身願意研究和指揮[[史特勞斯|史特勞斯]]家族的作品。

==录音==

'''维也纳新年音乐会'''的商业录音最早见于1979年。中国大陆自1994年至2002年有少量原版[[CD|CD]]引进，从2003年至2007年引进的CD虽然在编号上与原版保持一致，但是刻录和印刷质量与在欧洲销售的版本有差异。2008年以后在中国大陆销售CD被称作中国市场定制版本，编号与欧洲市场不同，但是唱片的发行公司Decca宣称他们在全球销售的新年音乐会录音为单一版本，并且否认了“中国版本”的存在性。

== 電視轉播 ==
维也纳新年音乐会于1959年开始第一次进行电视转播，不久便成了欧洲文化的盛事。电视转播中，维也纳歌剧院芭蕾舞团还为新年音乐会配上了古典的维也纳舞蹈，这一传统一直保留至今。在[[中国大陆|中国大陆]]，從1987年开始由[[中央电视台|中央电视台]]进行电视转播（其间曾因西方制裁1989年[[六四事件|天安门事件]]而终止了1990年的转播）。然而在[[中国大陆|中国大陆]]的转播最初并非实时进行，音乐会开始于当地时间11点15分，而[[中央电视台|中央电视台]]的转播开始于北京时间18点45分。在[[台灣|台灣]]，1998年第一次轉播該場音樂會（當時由[[中視|中視]]頻道轉播，該頻道轉播至2000年止），2003年至今的新年音樂會是由[[台灣電視公司|台灣電視公司]]進行電視轉播；[[台中市|台中市]][[圓滿戶外劇場|圓滿戶外劇場]]每年也會戶外同步轉播，也會在[[高雄市|高雄市]][[衛武營國家藝術文化中心|衛武營國家藝術文化中心]]戶外廣場進行轉播。

== 軼事 ==
* 在1990年的年度指揮中，樂團原想邀請[[喬治·蕭提|喬治·蕭提]]來擔任，但蕭提婉拒後，變想邀請[[雷納德·伯恩斯坦|雷納德·伯恩斯坦]]擔任，但伯恩斯坦因工作過於忙碌而無法抽身，便決定由他的門生[[祖賓·梅塔|祖賓·梅塔]]來擔任。
* 1992年的新年音樂會因為是[[維也納愛樂樂團|維也納愛樂樂團]]建團150周年，故樂團當時想邀請[[雷納德·伯恩斯坦|雷納德·伯恩斯坦]]擔任年度指揮，但伯恩斯坦卻於1990年10月14日逝世，而後變改邀請[[卡洛斯·克萊伯|卡洛斯·克萊伯]]擔任，由於要邀請克萊柏指揮音樂會是相當不容易的，故在此年所播映的影片中，故未安排芭蕾舞片段，在事後的演出也相當成功，成就另一個經典的音樂會。
* 2005年的音樂會因受到[[南亞海嘯|南亞海嘯]]的影響，當年指揮[[洛林·馬澤爾|洛林·馬澤爾]]在下半場第一首曲目美麗的加拉蒂亞序曲(Die schöne Galathee, Ouvertüre)結束後，與樂團一起向災區捐贈一筆費用，以重建災區，現行市面上影片已將此段剪去，只能從當年電視轉播中可見，但也讓世人看見古典音樂作為藝術與其人文價值的意義。
