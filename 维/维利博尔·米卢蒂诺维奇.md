'''维利博尔·“博拉”·米卢蒂诺维奇'''（{{lang-sr|Велибор Бора Милутиновић}}，{{bd|1944年|9月7日|||Милутиновић}}），[[塞尔维亚|塞尔维亚]]足球教练。中国人也称其为'''米卢'''，外號“神奇教练”。
== 簡歷 ==
米卢蒂诺维奇在球員時代可謂平淡無奇，但執教生涯卻成就非凡，蓋因他是唯一一个连续帶領五支不同的國家[[足球|足球]]队參與[[世界杯|世界杯]]決賽週的教练：[[1986年世界杯足球赛|1986年]]的[[墨西哥国家足球队|墨西哥]]，[[1990年世界杯足球赛|1990年]]的[[哥斯达黎加国家足球队|哥斯达黎加]]、[[1994年世界杯足球赛|1994年]]的[[美国国家足球队|美国]]、[[1998年世界杯足球赛|1998年]]的[[尼日利亚国家足球队|尼日利亚]]和[[2002年世界杯足球赛|2002年]]的[[中国国家足球队|中国]]。

他在執教墨西哥、哥斯達黎加、美國、尼日利亞時，四次均能打入世界杯決賽第二圈，被稱為「神奇教練」。而在担任中国足球队主教练期间，則成功带领中国队首次闯入世界杯决赛圈，但這次則未能打入第二圈。2009年他曾短暂担任[[伊拉克|伊拉克]]國家隊主教練。

2009年12月8日，塞尔维亚足协宣布，米卢蒂诺维奇担任[[塞尔维亚国家足球队|塞尔维亚国家队]]助理教练一职<ref>[http://sports.sina.com.cn/c/2009-12-08/23244735650.shtml 米卢任塞尔维亚国足助教 备战世界杯搜集对手情报]</ref>。

== 米卢与中国足球 ==
他虽然离开中国队，但仍然时刻关注和支持中国足球。米卢在新浪网有一个中文博客，自2007年起，发表文章几十篇，拥有上千万的访问量。

2018年1月11日，米卢在北京正式接受中国足协邀请，受聘成为中国足协青训顾问<ref>{{Cite web |url=http://www.fa.org.cn/news/other/2018-01-11/528104.html |title=米卢受聘成为中国足球协会青训顾问 |access-date=2018-01-11 |archive-url=https://web.archive.org/web/20180111224123/http://www.fa.org.cn/news/other/2018-01-11/528104.html |archive-date=2018-01-11 |dead-url=yes }}</ref>。
