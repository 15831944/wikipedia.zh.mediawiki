'''维多利亚·纽兰'''（{{lang-en|'''Victoria Nuland  '''}}，{{bd|1961年||||catIdx= N纽兰}}）出生于美国[[纽约州|纽约州]][[纽约市|纽约市]]，曾任[[美国国务院|美国国务院]]发言人。<ref>{{cite web|url=http://www.state.gov/r/pa/prs/ps/2011/05/164502.htm|title=Bureau of Public Affairs Front Office Changes|access-date=2012-02-14|archive-url=https://web.archive.org/web/20120120051650/http://www.state.gov/r/pa/prs/ps/2011/05/164502.htm|archive-date=2012-01-20|dead-url=yes}}</ref>维多利亚·纽兰曾在2005年到2008年间任美国驻[[北约|北约]]大使。纽兰拥有[[布朗大学|布朗大学]]学士学位，会说[[俄语|俄语]]、[[法语|法语]]以及一些[[汉语|汉语]]。

== 工作履历 ==
*2003年至2005年，纽兰出任[[美国副总统|美国副总统]][[迪克·切尼|迪克·切尼]]的国家安全首席副顾问。<ref name="rmws " >{{cite web |url=http://world.people.com.cn/GB/14766230.html |title=美国务院任命维多利亚·纽兰为新发言人 接替克劳利 |accessdate= 2011年5月29日|work= |publisher=人民网 |language =zh-cn}}</ref>
*2005年至2008年，纽兰出任[[美国驻北大西洋公约组织代表|美国驻北大西洋公约组织代表]]。<ref name="rmws " />
*2010年，纽兰出任[[美国驻欧洲常规武装力量特别代表|美国驻欧洲常规武装力量特别代表]]。<ref name="rmws " />
*2011年5月27日，纽兰出任[[美国国务院发言人|美国国务院发言人]]，接替辞职的[[菲利普·克劳利|菲利普·克劳利]]。<ref name="rmws " />
*2013年9月18日, 昇任負責歐洲及歐亞事務的助理國務卿.<ref>{{Cite web |url=http://www.state.gov/r/pa/ei/biog/214343.htm |title=存档副本 |access-date=2013-12-08 |archive-url=https://web.archive.org/web/20131210164810/http://www.state.gov/r/pa/ei/biog/214343.htm |archive-date=2013-12-10 |dead-url=yes }}</ref>

== 个人生活 ==
纽兰是[[美国|美国]][[耶鲁大学|耶鲁大学]][[医学|医学]][[教授|教授]][[舍温·纽兰|舍温·纽兰]]的女儿。1977年，她毕业于[[罗斯玛丽中学|罗斯玛丽中学]]，后来在[[布朗大学|布朗大学]]获得[[学士|学士]]学位。纽兰的丈夫是[[历史学家|历史学家]][[罗伯特·卡根|罗伯特·卡根]]，两人育有两个孩子。纽兰精通[[英语|英语]]、[[俄语|俄语]]、[[法语|法语]]，略知[[汉语|汉语]]。
