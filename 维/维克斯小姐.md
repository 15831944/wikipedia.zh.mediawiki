'''维克斯小姐'''（'''The Misses Vickers'''），美国画家[[约翰·辛格·沙金|約翰·辛格·薩金特]][[油画|油画]]作品。画中人为[[维克斯|维克斯家]]（Vickers，英国军工家族）三位千金，<ref name=1st/>在[[錫菲|錫菲]][[博尔索弗山|博尔索弗山]]（Bolsover Hill）的家族宅邸里。<ref name=3rd>{{cite web |url=http://www.yorkshiresfavourites.org/paintings/the-misses-vickers.html |language=en |title=The Misses Vickers |publisher=Yorkshire's Favourite Paintings |date=Unknown |accessdate=2012-08-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20130511113956/http://www.yorkshiresfavourites.org/paintings/the-misses-vickers.html |archivedate=2013-05-11 }}</ref>此画作于1884年。<ref name=1st/>现藏于錫菲[[錫菲艺术馆|錫菲艺术馆]]（Museums Sheffield）。<ref name=1st/>

画中三人分别为（由左至右）佛罗伦斯·伊芙琳·维克斯（Florence Evelyn Vickers，1867年-1947年），美宝·弗朗西丝·维克斯（Mabel Frances Vickers，1862年-1894年）与克莱拉·米尔德丽德·维克斯（Clara Mildred Vickers，1865年-1952年）。<ref name=2nd>{{cite web |url=http://jssgallery.org/paintings/the_misses_vickers/the_misses_vickers.htm |language=en |title=The Misses Vickers |publisher=John Singer Sargent Virtual Gallery |date=Unknown |accessdate=2012-08-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20120825055504/http://www.jssgallery.org/Paintings/The_Misses_Vickers/The_Misses_Vickers.htm |archivedate=2012-08-25 }}</ref>薩金特受托作画纪念美宝21岁生日。<ref name=2nd/>1885年，该作于巴黎一个沙龙展出，遭到冷落，更被称为“伪委拉斯蓋茲”油画。<ref name=2nd/>次年，该作于[[皇家艺术学院|皇家艺术学院]]展出，被观画者选为[[波迈宪报|波迈宪报]]（Pall Mall Gazette）“年度最差油画”。批评者认为，薩金特运用的透视使得“克莱拉扭转了身体，椅子倾斜，前方两人比较紧凑，背景中在远处的物件是“杂耍人的把戏”，“既肤浅又狂妄””。<ref name=2nd/>

在这幅画中，可见薩金特深受[[委拉斯蓋茲|委拉斯蓋茲]]影响。该作的色调与委拉斯蓋茲的作品一样，比较阴暗，同样运用了空间，在空间后也有光线（右上方）。<ref name=2nd/>这些手法都可以在薩金特的另一件作品，[[爱德华·达里·博伊特的女儿们|愛德華·達裏·博伊特的女兒們]]中见到。<ref name=2nd/>除此之外，在该作中，也可以看出[[弗兰斯·哈尔斯|弗兰斯·哈尔斯]]的痕迹，即其“自发性”对薩金特的影响。<ref name=2nd/>

无论是时人，抑或是作家本人，都不喜欢这幅作品。<ref name=3rd/>薩金特写道：“我在乡下给人画了几幅肖像，又在黑洞錫菲给三个丑女人作了像。”<ref name=3rd/>讽刺的是，在当下，这幅画是錫菲艺术馆里，最受欢迎的视觉艺术藏品之一。<ref name=3rd/>
