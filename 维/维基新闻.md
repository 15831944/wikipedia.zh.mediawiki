'''維基新聞'''（{{lang|en|Wikinews}}），此[[維基|維基]]計劃隸屬於[[維基媒體基金會|維基媒體基金會]]，是一個[[自由內容|自由內容]]的新聞媒體，訴求是讓任何人得以發表各項新闻主题。使命是“提供一個新聞工作者能獨立報導新聞的環境”，现在已经有了二十一种语言的版本。

這計劃的重要目标，是建立一個新聞資料庫。方便為其他计划引述、參考，當然也包括[[维基百科|维基百科]]自己，以作为全球和區域新聞事件的记录。

== 歷史 ==
2004年11月，[[維基媒體基金會|維基媒體基金會]]實驗以wiki共筆協作的方式建立新聞網站的可行性。最終計劃得到基金會理事會允許建立。同年年尾，網站進入公開測試階段，並且[[德語|德語]]版本也已經運作。此外還先後建立了[[荷蘭語|荷蘭語]]、[[法語|法語]]、[[西班牙語|西班牙語]]、[[瑞典語|瑞典語]]、[[保加利亚語|保加利亚語]]、[[波蘭語|波蘭語]]、[[葡萄牙語|葡萄牙語]]、[[羅馬尼亞語|羅馬尼亞語]]和[[乌克兰語|乌克兰語]]等語言版本。

2005年3月13日，維基新聞己發表1,000條新聞條目。同年4月3日註冊用戶人數達2,000名。同年維基媒體基金會發起用戶投票，以決定維基新聞的內容發佈版權，結果由公共領域改為[[知识共享|知识共享 署名2.5版]]。

== 附加計劃 ==
維基新聞主要是以文字方式來發佈新聞，有些用戶們把這個理念延伸到聲音和影像媒體。其中有三個發展中的計劃 — [[n:en:Wikinews:Audio_Wikinews|語音維基新聞]]發佈Ogg Vorbis格式的檔案、 — [[n:en:Wikinews:Broadcast|維基新聞廣播]]是提供影像內容，還有[[n:en:Wikinews:Print_edition|可列印版本的維基新聞]]使它可讓更多人閱讀。

== 中文維基新聞 ==
'''中文維基新聞'''是[[維基新聞|維基新聞]]協作計劃的[[中文|中文]]版本，自2006年3月14日正式成立，由非營利組織──[[維基媒體基金會|維基媒體基金會]]維持負責，目前在各語言[[維基新聞|維基新聞]]新聞稿數排名中列第9名，截至2016年5月15日，中文維基新聞已擁有5,159個新聞稿。<ref>{{Cite web|url=https://zh.wikinews.org/wiki/Special:%E7%BB%9F%E8%AE%A1|title=統計資料|accessdate=2016-05-15|author=|date=|publisher=}}</ref>

===設立===
維基新聞的中文版曾在元維基和中文維基百科上經過多次討論，曾經過兩次是否設立的投票。

第一次投票最終未得到足夠的維基人支持，而未通過。支持的一方認為，不應為了擔心維基新聞報導有政治等敏感話題而遭中國政府封鎖，而放棄整個華文用戶市場。反對的一方則認為，由於維基新聞的自由理念，一定會使得很多不同政見的新聞出現。展望未來，問題在於：如何避免上次維基百科和其他維基媒體基金會的計劃受中國政府封鎖的事件再次重演。

經過一年多後的第二次投票，維基新聞以大比數通過，并設立動議。中文維基新聞於是在2006年3月14日正式成立。
