'''維爾紐斯機場'''（立陶宛語：{{lang|lt|'''Vilniaus oro uostas'''}}；{{airport codes|VNO|EYVI}}），或称'''维尔纽斯国际机场'''（{{lang|lt|'''Tarptautinis Vilniaus oro uostas'''}}），是立陶宛首都维尔纽斯的一座民用机场，启用于1944年。

== 概况 ==
维尔纽斯机场的管理机构是维尔纽斯国际机场公司，为一家国有公司，1991年9月30日由立陶宛交通部设立。<ref name="ff">{{cite web|url=http://www.vno.lt/en/about_us/facts_figures/index.php |title=Facts & Figures |publisher=Vilnius International Airport |date= |accessdate=2008年11月15日 |language=en }}</ref>2006年，该机场客流量最大的航空公司是[[波罗的海航空|波罗的海航空]]，其次为[[立陶宛航空|立陶宛航空]]。此外，[[北欧航空|北欧航空]]、[[汉莎航空|汉莎航空]]、[[芬兰航空|芬兰航空]]、[[俄罗斯航空|俄罗斯航空]]、[[奥地利航空|奥地利航空]]、[[捷克航空|捷克航空]]、[[波兰航空|波兰航空]]以及一些区域性航空公司也在维尔纽斯国际机场开通航线。

维尔纽斯机场是一个快速发展的机场，每年旅客吞吐量在100万人次以上，2007年突破170万<ref name="ff"/>，居立陶宛四个主要机场之首。

== 航站楼 ==
维尔纽斯机场旧[[航站楼|航站楼]]兴建于1950年代，是一座典型[[苏维埃|苏维埃]]风格的航站楼，设计容量为每天20架次。航站楼外部装饰有士兵、工人和飞行员的雕像；内部墙壁及天花板则饰以花环、月季及星星，为苏联战后初期的典型风格。

2007年11月，总建筑面积3.7万平方米的新航站楼投入使用，提高了容量及设施水平，也满足了[[申根协定|申根协定]]的要求。<ref name="ff"/>

== 事故及意外 ==
[[File:SAS_Dash-8-400_after_crash-landing_in_Vilnius_airport.JPG|thumb]]
2007年9月12日，北欧航空一架载有48名乘客和4名机组人员的[[Dash_8|庞巴迪Q400]]客机（注册号LN-RDS）由[[哥本哈根机场|哥本哈根机场]]飞往立陶宛[[帕兰加国际机场|帕兰加国际机场]]，准备降落前发现起落架失灵。机组决定在维尔纽斯机场迫降。飞机迫降后所有人员均及时成功逃离，未造成人员伤亡。<ref>{{cite web|url=http://news.xinhuanet.com/video/2007-09/13/content_6714836.htm |title=丹麦客机起落架失灵 在立陶宛紧急迫降 |publisher=新华网 |date=2007年9月13日 |accessdate=2008年11月12日 }}</ref>这次事故以及同期另两起类似事故，促使北欧航空决定永久停飞庞巴迪Q400客机。<ref>{{cite web|url=http://finance.sina.com.cn/chanjing/b/20071030/13551757477.shtml | title=北欧航空永久停飞27架庞巴迪飞机 |publisher=新浪网，原载中国民航新闻信息网 |date=2007年10月30日 |accessdate=2008年11月12日 }}</ref>

== 地面交通 ==
