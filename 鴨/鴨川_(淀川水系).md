'''鴨川'''（{{jpn|j=鴨川|hg=かもがわ|rm=Kamogawa|f=y}}）是流經[[日本|日本]][[京都府|京都府]][[京都市|京都市]]的[[淀川|淀川]][[水系|水系]]的[[一級水系|一級河川]]。

== 地理 ==
根據日本河川法，鴨川的起點為京都市[[北區_(京都市)|北區]][[雲畑|雲畑]]的出合橋附近。

起源於北區雲畑的棧敷岳東部的山谷的'''神父谷川和'''和起緣於棧敷岳南部的藥師峠的'''雲畑岩屋川'''在雲畑岩屋會流成為'''雲畑川'''。雲畑川和起源於雲畑的魚谷山南部的山谷的'''中津川'''在出合橋附近匯流，在此之後就成為「鴨川」。由於鴨川的起點標記在中津川下游流域，據此可判定其源流為中津川，但這終究是日本河川法上的處理方式，傳統上也有許多人將其源流看作棧敷岳。

另外，除了此鴨川主流之外，還有起源於京都市[[右京區|右京區]]的[[芹生峠|芹生峠]]的'''貴船川'''和起源於京都市[[左京區|左京區]]的[[花脊峠|花脊峠]]的'''鞍馬川'''。兩者在貴船口匯流成為鞍馬川，再於山幸橋和鴨川匯流成為更大的河川。在[[貴船神社|貴船神社]]的社傳之中，[[神武天皇|神武天皇]]的母親[[玉依姬|玉依姫命]]在往[[淀川|淀川]]、鴨川的源流溯溪而上之後，到達之處為[[貴船|貴船]]的土地，因此在[[日本神話|日本神話]]中也將貴船川看作是鴨川的源流。

鴨川和鞍馬川匯流後，於北區上賀茂進入京都盆地。以東南東的方向流經上賀茂神社（[[賀茂別雷神社|賀茂別雷神社]]）、下鴨神社（[[賀茂御祖神社|賀茂御祖神社]]）旁，在賀茂大橋（加茂大橋）前和[[高野川_(京都市)|高野川]]匯流。之後往正南方流經京都市內，於[[四条大橋|四条大橋]]附近轉流向西南方，在[[伏見區|伏見區]]下鳥羽注入[[桂川_(淀川水系)|桂川]]。另外，於[[中京區|中京區]]向西分支出[[高瀨川_(京都府)|高瀨川]]，往南並行流去。

=== 命名 ===
最早河岸開墾土地是西元五世紀末到六世紀的賀茂氏。之後延河土地分成了上賀茂村、西加茂村跟下鴨村，讀法皆為 kamo，只是漢字標記不同。<ref name=":0">{{Cite book|title = 京都原來如此：千年古都的風俗、習慣、飲食文化解謎|last = 博學堅持俱樂部|first = |publisher = 遠足文化事業股份有限公司|year = 2015|isbn = 9789869189620|location = 台灣新北市新店區|pages = 第168到169頁}}</ref>  奈良時代的七一三年寫成的《山城國風土記》來說，書中標記為賀茂川。到了江戶時代的地圖，則多寫成「加茂川」。而到了江戶末期至明治時代，又習慣稱為「賀茂川」。<ref name=":0" /> 明治29年（西元1896年）當時的官員依據新制的河川法，將河川的名稱統一為「鴨川」，但因在地人習慣把上游稱為「賀茂川」<ref name=":0" />，造成一條河有兩種寫法。

=== 標記 ===
根據於1964年（[[昭和|昭和]]39年）公佈、並在隔年實施的日本河川法，在起點之後統一標記為鴨川，但於貫例上，在[[高野川_(京都市)|高野川]]的匯流點之前的上游會標記為'''賀茂川'''或'''加茂川'''。雖然在「[[日本紀略|日本紀略]]」之中，「鴨川」「賀茂川」的標記被混用在一起，但是於[[平安時代|平安時代]]並沒與根據流域來區分標記。{{请求来源}}

由於鴨川流過[[平安京|平安京]]的東部，在古代也被稱為'''東河'''。{{来源请求}}

== 歷史 ==
鴨川上游流域在古代是日本氏族[[賀茂氏|賀茂氏]]的本營。上賀茂神社、下鴨神社為用作祭祀其[[氏神|氏神]]。下游流域則是被分配置[[山背國|山背國]]的[[秦氏|秦氏]]的定居地點。

鴨川之於[[平安京|平安京]]則是被當作城都東邊的界限。另外，據說鴨川被當作平安京東邊界限的理由為，初代造宮大夫的[[藤原小黑麻呂|藤原小黑麻呂]]進入被當作新京城預定地區的[[葛野郡|葛野郡]]視察，並占卜了「地相」<ref>本內容也被記載在京都檢定的官方課本之內。</ref>，以及以根據[[作庭記|作庭記]]，是為了反映[[風水|風水]]的[[四神相應|四神相應]]的思想。但是，在作庭記之中並沒有提及平安京的地勢，而且在打造平安京時的天皇的詔令及被演奏的踏歌中也沒有觸及此一想法，因此並沒有史學上的根據能說明四神相應的思想影響了建都的工程，此被認為是已普及化的後世的推論。

在[[中世紀|中世紀]]時，鴨川和京都的市區鄰接的河岸成了風化區，聚集了許多藝人。[[出雲阿國|出雲阿國]]也是其中之一。此外，鴨川及三條河原一帶曾經是[[織豐時代|織豐時代]]用於公開處刑與首級示眾甚至棄屍亂葬崗的場所。

1614年（[[慶長|慶長]]19年），[[角倉了以|角倉了以]]、[[角倉素庵|素庵]]父子開鑿了[[高瀬川_(京都府)|高瀬川]]，被當作連結京都和大阪之間的水路來利用。1890年（明治23年），由於[[琵琶湖疏水|琵琶湖疏水]]的開通，鴨川和[[琵琶湖|琵琶湖]]也連在了一起。但是進入20世紀後，由於鐵路的開通，水運隨之衰退。

=== 治水 ===
自古以來鴨川被認知為是不斷氾濫肆虐的河川。除了以流經大都市的河川來講，鴨川的坡度算是陡峭之外，平安京建城時砍伐北山的樹木、隨著市區向東方的擴大河岸跟著市區化等等也是原因。

824年（[[天長|天長]]元年）設置了擔任治水的官職「防鴨河使」，但洪水並未因此停止。平安末期掌權的[[白河天皇|白河法皇]]，將「賀茂之水」放在不如自己意之事（天下三大不如意）的第一位。[[豊臣秀吉|豊臣秀吉]]建造的[[御土居|御土居]]的東部沿著鴨川，發揮了堤防的功能。另外1670年（[[寛文|寛文]]10年）時[[今出川通|今出川通]]到[[五條通|五條通]]為止的區間建設了寛文新堤，和御土居之間的土地被開發為新的市區。

==== 昭和的鴨川水災 ====
1935年（昭和10年）6月29日清晨時發生的鴨川水災造成死者12人、負傷者71人、住宅全毀或半毀482戶、37.2平方公里的面積浸水。除此之外，鴨川之上的的26座橋之中，包括[[五條大橋|五條大橋]]等共有15座橋被沖走。由於[[三條大橋|三條大橋]]等上游的橋樑被往下沖刷後，撞上當時為混凝土拱橋的[[四條大橋|四條大橋]]，而將河川攔截住，滿溢出來的水和先斗町及鴨川平行向下沖刷，使得京阪本線的地盤流失、車站建築及月台、琵琶湖疏水的團栗閘門受到損害，東山區側的宮川町也浸水，且河水在早上破壞了團栗橋、松原大橋、五條大橋，堵住了正面橋，產生了更大的災害。在此洪水中沒有受到災害的橋樑只有北大路、加茂、七條共三座橋樑而已。在修復工程進行中的8月10-11日之間，因為再次的豪雨，發生了鴨川上的臨時橋樑被沖走等災害。以此為契機，計劃了[[琵琶湖疏水|琵琶湖疏水]]的暗渠化、塩小路橋附近的水力發電廠建設、京阪線的地下化等大規模的河川整修事業，雖然因戰爭而中斷的京阪線地下化、疏水的暗渠化遭到中止，鴨川工程於1947年（昭和22年）完工。殘留至當時的西側水路（禊川）的河道被從原本的高度向下挖掘了近兩公尺，五条大橋-塩小路橋之間的京阪線的綠地部份被撤走，將河川的幅度加寬、建造多個落差達數十公分的河堰以抑制河水的流速，成了現今的姿態<ref>參考文獻：『京都市水害史』（1936年3月、京都市發行、京都府立綜合資料館收藏（資料編碼1104878531））、植村善博著『京都の治水と昭和大水害』(2011年4月 文理閣 刊) </ref>。另外京阪線和琵琶湖疏水的地下化，則被當作京都市的交通堵塞舒缓政策在1979年動工，並於1987年5月完成地下化。

=== 鴨川替換說 ===
1932年（昭和7年）[[塚本常雄|塚本常雄]]所提倡的學說主張<ref group = *>此學說的根源被認為是來自1985年(明治28年)『平安通史 巻乃一』中湯本文彥等人。</ref>，從賀茂川的下游部分和高野川的匯流地點到和[[白川_(淀川水系)|白川]]的匯流地點為止是人工河川，賀茂川本來為筆直南下和堀川匯流，高野川則從現在的匯流地點流向西南方向，在[[頂法寺|六角堂]]的附近和堀川匯流。其根據為，[[京都盆地|京都盆地]]的地形為東北向西南傾斜，賀茂川與其逆行流動相當不自然，流向本來為低地的[[堀川通|堀川通]]才較為自然。在1980年代之前，此「鴨川替換說」蔚為主流，不僅是書籍，連報紙和觀光手冊中都將此當作事實來看待。<ref>横山(1988) pp.9-40</ref>

但是後來此一說法遭到[[同志社大學|同志社大學]]教授[[横山卓雄|横山卓雄]]的研究否定。地下鐵[[京都市營地下鐵烏丸線|烏丸線]]的施工時期在烏丸中學校附近挖掘時，因發現了岩盤，工程變得相當困難，使用物理探測法對此地下岩盤進行調查之後，發現於東南方向存在著地下山脈。由於地下山脈幾乎垂直橫切過塚本的說法中所提倡的賀茂川的流路，因而產生了矛盾，也確認了現在賀茂川的流路為受該地下山脈的影響之後的自然的流路。<ref>横山(1988)pp.124-131</ref>

== 法規上的限制 ==
根據[[京都府鴨川條例|京都府鴨川條例]]（2008年4月1日施行），某些地方禁止停放自行車、施放[[煙火|煙火]]及爆竹、[[烤肉|烤肉]]、[[塗鴉|塗鴉]]（鴨川全域）等行為。

== 鴨川步道橋問題 ==
[[京都市|京都市]]在1980年代以後，計畫在三條大橋及四條大橋中間架設[[行人|行人]]專用的新橋梁。對於這個計畫，預定地點的東側認為此會帶動街道的活力而贊成，相較之下，西側的[[先斗町|先斗町]]則認為該地成為通往橋的道路會破壞街道的風情而多為反對。

1996年，訪問京都的[[法國|法國]][[總統|總統]][[雅各·席哈克|席哈克]]（前[[巴黎|巴黎]]市長）和市長[[桝本頼兼|桝本頼兼]]的會談之中，提案在鴨川架設模仿[[塞納河|塞納河]][[藝術橋|藝術橋]]的橋樑。因為兩年後的1998年將成為「在日本的法國年（日本におけるフランス年）」和京都市與巴黎市的[[友好城市|友情盟約]]締結40周年，此構想本為當作其紀念事業，但在這個計畫公開之後，除了有因認為「會成為新的觀光勝地」而贊成的意見，也引發了認為「法式風格的設計」會破壞[[景觀|景觀]]的批評聲，輿論分為兩派。在1997年舉行的[[都市計畫|都市計畫]]案的公告中，雖然整體上贊成計畫的意見書比反對的一方更多，但反對的市民團體認為「未充分檢討意見書的內容只憑數字來比較」，批判計畫的決定過於倉促。另外法國的[[世界報_(法國)|世界報]]刊載了批判此計畫的記事等等，連京都市外也被捲入論戰之中，最後在1998年，京都市將藝術橋的計畫作廢。

=== 分流 ===
* 御手洗川
: 又稱明神川。在北區上賀茂和賀茂川分離，流經上賀茂神社境内之後，再次和賀茂川匯流。在神社境內也被稱為御物忌川、楢之小川。這條河川的水被用在[[葵祭|葵祭]]的祭事之上。另外下鴨神社境內也有相同的河川，但其水源為地下泉水，經過高野川之後注入鴨川。
* 禊川（みそそぎ川）
: 在加茂大橋下游和鴨川分離，在西岸的河床（鴨川公園的地下河道，於丸太町橋下游返回地面）和鴨川主流並行。在二條大橋的下游和高瀨川分離後，由五條大橋附近再次和鴨川匯流。夏天的納涼床被設置成跨越此河川的形式。
