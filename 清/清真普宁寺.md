'''清真普宁寺'''，俗称'''回回营清真寺'''，原址位于[[北京市|北京市]][[西城区|西城区]][[西长安街|西长安街]]中段南侧，[[新华门|新华门]]正南，是[[伊斯兰教|伊斯兰教]][[清真寺|清真寺]]，现已无存。

==历史==
[[清朝|清朝]][[乾隆|乾隆]]二十年五月（1755年），清廷派兵平定[[新疆|新疆]][[阿睦尔撒纳|阿睦尔撒纳]]叛乱，解救了[[大和卓|大和卓]]与[[小和卓|小和卓]]两人。但这两人后来组织叛乱活动。新疆和卓家族分成阿帕克和卓与额赖玛特和卓，[[容妃_(乾隆帝)|容妃]]（民间传说中的[[香妃|香妃]]）一家所属的额赖玛特和卓拥护清廷，反对分裂，遭大小和卓欺辱。乾隆二十二年（1757年），清廷再度进军新疆[[平定回部|平定大小和卓叛乱]]，容妃的叔、兄积极配合清军，乾隆二十四年（1759年）平息大小和卓叛乱。乾隆二十五年（1760年），容妃的胞兄、五叔、六叔和其他五位首领等人应召入京，获清廷封官晋爵，史称“八爵进京”。乾隆帝将这支由八位回部首领和[[伊斯兰教|回教]]领袖带领的部众编入[[八旗|八旗]]并下诏封居，安置在西苑宝月楼（今[[中南海|中南海]][[新华门|新华门]]）南侧今[[东安福胡同|东安福胡同]]一带，史称“回回营”，又称红帽子回回。随叔、兄奉旨进京的[[香妃|香妃]]奉召入宫，获乾隆帝册封为和贵人。此后，乾隆帝又下旨在回回营中兴建清真普宁寺，供营中回人礼拜用<ref name=xiangfei/>。

清真普宁寺始建于乾隆二十七年（1762年），乾隆二十八年（1763年）竣工，与西苑宝月楼隔街相望。这是由[[乾隆帝|乾隆帝]]降旨御批，使用内帑并专人督工建造的敕建清真寺。当时附近的[[回部|回部]][[穆斯林|穆斯林]]均在该清真寺礼拜。乾隆帝御制《清真普宁寺碑》，[[中华人民共和国|中华人民共和国]]成立后一度被埋于地下，后被文物部门收藏，现存[[历代帝王庙|历代帝王庙]]内东侧<ref name=xiangfei/>。该碑上有满汉蒙回四体碑文，内容是：

{{引用|为天下共主，俾阻退逖听，壹票我约束，而后戎索所届，风气莫敢以自私、尚已。顾在昔寄象鞮译之掌，必与之达志通欲，修其教不易其宜，厥旨岂相戾哉？盖惟极天下之不齐以致其大齐，而观化者益臻于无外。考前史，[[回纥|回纥]]自[[隋朝|隋]][[开皇|开皇]]时始入于中国，至[[唐朝|唐]][[元和|元和]]初，偕[[摩尼教|摩尼]]进贡，请置寺[[太原|太原]]，额曰大云光明，实为礼拜寺所由。然其致之也，或以假师以通市，于纳土服属我甿我隶之义故无当焉。朕寅承天地祖宗鸿庥，[[平定准噶尔|平准噶尔]]，遂定[[回部|回部]]各城。其[[伯克|伯克]][[霍集斯|霍集斯]]、[[霍什克|霍什克]]等并赐爵王公，赐居邸舍。而余众之不令回其故地者，成居之[[長安右門|长安门]]之西，俾服官执役，受旅处，都人因号称回子营。夫齿繁则见庞，类辨则情涣，思所以统同合异，使瞻听无奇袤，初不在辟其教而揉矫之也。且准部四[[卫拉特蒙古|卫拉特]]内附，若[[普宁寺|普宁寺]]，若[[固尔札庙|固尔札庙]]，既为次第创构，用是宠绥，回人亦吾人也，若之何望有缺耶！爰命将作支内帑羡金，就所居适中之地，为建斯寺。穹门垲殿，翊庑周阿，具中程度。经始以乾隆癸未清和吉月，浃岁落成。回众以时会聚其下，轮年入觐之众伯克等无不欢欣瞻拜。诧[[西域|西域]]所未曾睹，问有叨近日之荣，而兼擅土风之美，如是举者乎？成鞠虞挊曰：然。复重谂之曰：尔回之俗，向惟知有鲁斯讷墨，今则朔奉朝正矣。向惟知有腾格，今则铸颁泉府矣。越及屯赋觐飨诸令典，其大者靡弗同我声教，而国家推以人治人之则，更为之因其教以和其众，揆诸万舞备铜绳之技，九宾缀头之班，此物此志云尔。其谁曰不宜！乃为之记而系以铭。孰为[[天方|天方]]？孰为天堂？花门秘刹，依我云闯。厥城默伽，厥宗墨克，派哈帕尔，传依铁勒。经藏三十，咨之阿浑，西向北向，同归一尊。珉城柟梁，司工所作，会极归极，万邦是若。<br>
乾隆二十有九年岁在甲中仲夏之吉，御制并书。}}

民间曾经误传该清真寺为“香妃寺”，说宝月楼是乾隆帝为香妃而建，香妃的父母思女来京，不能入宫，只好登上清真普宁寺邦克楼，与宝月楼中的香妃隔街相望。所以民间俗称宝月楼为“望家楼”。实际上该寺从未称过“香妃寺”，但回回营、宝月楼确与容妃有关。回回营中居住的多是她的族人。宝月楼虽建于容妃入宫前两年（1758年），但从乾隆帝多次吟咏宝月楼的诗文可知，容妃应多次登临或入住该楼。容妃逝世三年后，乾隆帝在宝月楼前仍作诗慨叹：“卅载画图朝夕似，新正吟咏昔今同。”<ref name=xiangfei/><ref name=anfu/>

[[中华民国|中华民国]]初年，[[袁世凯|袁世凯]]任[[中华民国大总统|大总统]]，将中南海作为大总统府。1913年在[[朱启钤|朱启钤]]主持下，将宝月楼改建为大总统府的大门“[[新华门|新华门]]”，新华门由此取代西苑门，成为中南海的正门。同时，下令将新华门对面的清真普宁寺的寺门、礼拜殿等建筑拆除，修筑了灰色的[[花墙_(西长安街)|花墙]]，以遮挡街南的民居<ref name=xiangfei/><ref name=anfu>{{Cite web |url =  http://finance.ifeng.com/city/msls/20090609/767252.shtml|title = 东安福胡同：一座与香妃有关的清真寺 |author =  |date = 2009-06-09 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref name=zuobie/>。当时政府以拓修新路为名，派出数百名便衣暴徒，将清真寺寺门、礼拜殿等建筑拆毁<ref name=anfu/>。

1913年位于寺北的正门被拆除后，清真普宁寺改在寺南辟建了新的大门，新大门坐北朝南，位于[[东安福胡同|东安福胡同]]中段北侧。[[中华人民共和国|中华人民共和国]]时期，该寺早已不再使用，沦为民居。到2009年之前，该寺仍有遗迹，主要包括：南部的大门遗迹，为一拱雕刻着缠枝花卉的石券；大门遗迹以北的院子里有一座大殿；大殿北面正对新华门处即前文所述的乾隆帝御制《清真普宁寺碑》，2009年之前已运走存历代帝王庙<ref name=anfu/>。

2009年，在西长安街道路拓宽拆迁工程中，整条东安福胡同北侧的建筑物均被拆除。清真普宁寺原址的建筑遗存自此消失<ref name=anfu/><ref name=zuobie>{{Cite web |url = http://news.163.com/09/0420/15/57BQI3T800011SM9.html |title = 作别老长安街 |author =  |date =2009-04-20  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

==建筑==
整座清真寺占地面积二十余亩，建筑恢弘，风格独特。整体采用非对称布局，建筑为砖石结构。正方形绿琉璃瓦顶的大殿坐西朝东，四周各建有角亭一座，绿琉璃瓦重檐[[攒尖顶|攒尖顶]]环廊呈现出[[阿拉伯|阿拉伯]]建筑风格，红色盲柱。该大殿规模为“明九暗七”，形制类似于晚其六年建成的[[北海公园_(北京)|北海]]“小西天”方殿。该寺的拱券形正门位于寺北临街，坐南朝北，穹顶与邦克楼融为一体，券门之上是四柱单檐九脊顶的四壁轩敞楼阁，卷棚重檐[[歇山顶|歇山顶]]，为“邦克楼”，又称唤礼楼。登楼唤礼时，声音俯掠西苑太液池，远传数里<ref name=xiangfei>{{Cite web |url =  http://book.people.com.cn/n/2012/1107/c69360-19520142-1.html|title =  让乾隆念念不忘的神秘女人 揭秘香妃的真实身份|author =  |date =2012-11-07  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref><ref name=anfu/>。
