'''清华八斋'''指位于[[北京|北京]][[清华园|清华园]]的[[清华大学|清华大学]]早期學生宿舍，可分為前五齋與後三齋。前五齋是指明斋、新斋、善斋、静斋、平斋，後三齋是强斋、诚斋和立斋。<ref name=SD>{{cite journal |url=http://doc.qkzz.net/article/d2b61c93-90fa-4b48-9823-8277c9661116.htm|author=冯务中|title=〈清华八斋〉|journal=《寻根》|issue=2005年第06期|date=2005-12|accessdate=2016-02-28|publisher=寻根杂志社|location=|language=zh-cn}}</ref>

==概况==
清華八齋的“斋”字，根據《中国文化辞典》的解释，“原意為在祭祀和典礼之前的潔淨身心的準備<ref>{{cite web |url=http://www.zdic.net/z/29/sw/9F4B.htm |publisher=汉典|title=齋 說文解字
|author=|language=zh-cn|date=|accessdate=2016-02-28}}</ref>，以示虔诚。后引申为能幽居之处所<ref>{{cite web |url=http://www.zdic.net/z/29/kx/9F4B.htm |publisher=汉典|title=齋 康熙字典|author=|language=zh-cn|date=|accessdate=2016-02-28}}</ref>”，以宿舍為齋，能體現其幽雅清静，宜居宜學。清華早期宿舍原本並沒有這麼典雅的名字，而是稱為“几院”，當時的校長[[梅贻琦|梅贻琦]]認為需要較優美的名字，於是在1935年時，決定將四院改称明斋、五院改称善斋、六院改称新斋、七院改称平斋、女生宿舍改称静斋<ref name=SD/>。

清华前五齋的名字源自《[[大學_(經傳)|大学]]》各目，即“大学之道，在明明德”——明斋，“在新民”——新斋，“在止于至善”——善斋。“知止而后有定，定而后能静”——静斋。“家齐而后国治，国治而后天下平”—平斋。[[清华大学|清华大学]]的“明斋”最早建立，其后是一系列“新”、“善”、“静”、“平”各斋。

“前五斋”的得名可能與[[梅贻琦|梅贻琦]]校長的《[[大學_(經傳)|大学]]》思想有关。梅贻琦著有名文《大学一解》<ref name="梅贻琦与清华">{{cite journal |url=http://www.people.com.cn/GB/paper81/6563/643464.html|author=黄延复、王小宁|title=〈梅贻琦与清华〉|journal=《大地》|issue=2002年第九期|date=2002-06-26|accessdate=2016-02-28|publisher=人民日报社|location=|language=zh-cn}}</ref>他在此文中寫道：“及至大学一篇之作，而学问之最后目的，最大精神，乃益见显著。《大学》一书开章明义之数语即曰：‘大学之道，在明明德。在新民。在止于至善。’。若论其目，则格物、致知、诚意、正心、修身，属明明德；而齐家、治国、平天下，属新民。”<ref name="梅贻琦与清华"/>梅校長對《[[大學_(經傳)|大学]]》很熟悉、非常推崇。

==前五齋==
===明斋===
[[File:清華明齋.jpg|thumb]]
明斋建成于1930年，是清华改成大學後的第一個學生宿舍，原稱為'''四院'''，又称为“新洋楼”，是由任職於[[天津|天津]][[基泰工程司|基泰工程司]]的[[杨廷宝|杨廷宝]]先生设计的<ref name=sss>{{cite web|url=http://www.thupxw.com/tesekecheng/shuimuqinghua/94.html|title=寻访清华八斋的前世今生|publisher=清华大学总裁培训班|author=陈伊凡、邱伟怡、张晓妍|language=zh-cn|date=2015-03-24|accessdate=2016-02-28|deadurl=yes|archiveurl=https://web.archive.org/web/20160304082322/http://www.thupxw.com/tesekecheng/shuimuqinghua/94.html|archivedate=2016年3月4日|df=}}</ref>，位于西北操场北端，[[國務院總理|國務院總理]][[朱鎔基|朱鎔基]]也曾住在這裡<ref>{{cite web |url=http://www.expo.tsinghua.edu.cn/ |title=清华大学数字博物馆|publisher=清华大学|author=|language=zh-cn|date=2015-03-24|accessdate=2016-02-28}}</ref>。

明斋的面积為4417[[平方公尺|平方公尺]]，南北长85.4公尺，东西长81.7公尺，是三層的建築物。明齋有四個門，位於它的四個角上，主要大門位在兩個轉角處，建築的中間部分有过街楼，方便學生南北穿行。當時的宿舍美曾有66個房間，每間可住2人，每層樓梯間設有公共厕所盥洗室<ref name=sss/>。

如今的明斋是清华大学社会科学学院的院馆<ref>{{cite web |url=http://www.sss.tsinghua.edu.cn/publish/sss/8033/index.html |title=學校各部門電話|publisher=清华大学社会科学学院|author=|language=zh-cn|date=|accessdate=2016-02-28}}</ref>。

===善斋===
善斋于1932年建成，最初稱為'''五院'''，為男生宿舍，是由Carl L.Anner所设计的，面积是2202[[平方公尺|平方公尺]]，形狀呈“一”字形，主要大門位在南向正中、進門時正對大樓梯，呈現典型的[[意大利|意大利]]式風格。東西方向分成五段，左有各5個立柱<ref name=sss/>。

如今的善斋为[[清華大學馬克思主義學院|马克思主义学院]]的办公楼<ref>{{cite web|url=http://www.qstheory.cn/zl/bkjx/201311/t20131101_285176.htm|title=清華大學思政課因材施教“讀歷史　搞創作”|publisher=中國青年報|author=崔麗|language=zh-cn|date=2013-11-01|accessdate=2016-03-01|deadurl=yes|archiveurl=https://web.archive.org/web/20141019174211/http://www.qstheory.cn/zl/bkjx/201311/t20131101_285176.htm|archivedate=2014-10-19}}</ref>，清华大学绿色办公室也在里面，門外豎立著[[马克思|马克思]]的雕像<ref>{{cite web |url=http://www.tsinghua.edu.cn/publish/marxism/4568/ |title=清华大学马克思主义学院|publisher=清华大学|author=|language=zh-cn|date=|accessdate=2016-02-28}}</ref>。

===静斋===
[[File:清華靜齋.jpg|thumb]]
位於[[近春園|近春園]]與舊清華園的交界上，與[[甲所|甲所]]一丘之隔。早期因做為女生宿舍，門禁很嚴，被男生戏称为為“堡垒”，只有在校慶或重要節日時才開放男生與外賓進入。且因其外观酷似炮台，又稱為“炮台”。

1928年清华大学开始招收女生，因为没有女生专用宿舍，[[古月堂|古月堂]]被临时辟为女生宿舍，同时学校开始筹建女生专用宿舍。1932年“静斋”落成，全校女生和部分女教师迁入住宿。静斋因为是清华第一座女生宿舍而知名度最高。[[中華人民共和國|中華人民共和國]]成立後，静斋曾先後做為慢性病療養處、留學生宿舍、招待所等<ref>{{cite web |url=http://www.expo.tsinghua.edu.cn/ |title=清华大学数字博物馆|publisher=清华大学|author=|language=zh-cn|date=2015-03-24|accessdate=2016-02-28}}</ref>。

===新斋===
[[File:清華新齋.jpg|thumb]]
新齋位於明齋旁邊，建于1934年，由[[天津|天津]][[基泰工程司|基泰工程司]]负责设计與修建，面積4050[[平方公尺|平方公尺]]，形狀為“工”字形。曾經稱為'''六院'''，以前是男生宿舍，著名大師[[钱伟长|钱伟长]]就曾住在这里，现为[[清华大学人文学院|清华大学人文学院]]部分系所在地<ref>{{cite web |url=http://news.guoxue.com/article.php?articleid=32294 |title=清华文学院的前世今生——1952年清华文学院素描|publisher=中华读书报|author=马庆洲|language=zh-cn|date=|accessdate=2016-02-28}}</ref>。

早期的新齋中間部分為楼梯间、盥洗室、值班室等輔助設施，上下段皆為宿舍，並且在北側採用单边走道的格局。齋子立面东西三段式，中央南向正入口<ref name=sss/>。
===平斋===
[[File:清華平齋.jpg|thumb]]
平齋位於善齋後面，建于1934年，也是由天津基泰工程司负责设计與修建，面积2759[[平方公尺|平方公尺]]。最初稱為'''七院'''，目前為教工宿舍，是目前唯一還是當作住所的斋子。

早期的平齋與新齋一樣，中間部分為楼梯间、盥洗室、值班室等輔助設施，上下段皆為宿舍，並且在北側採用单边走道的格局。齋子立面东西三段式，中央南向正入口<ref name=sss/>。

==後三齋==
後三齋均為[[中華人民共和國|中華人民共和國]]成立後興建的齋子，約建於1951年至1952年。當時，[[中國人民銀行|中國人民銀行]]因缺乏金融相關的專業人才，便与清華合作，成立了“银行专修科”，因要方便學員學習，後三齋因而誕生。1952年後三齋同時落成，當時的設計單位是清华大学技术室，也就是当时的基建处。同期間落成的建築物還有新航空馆、西大饭厅、一教和西阶教室等<ref name=sss/>。

===强斋===
[[File:清華強齋.jpg|thumb]]
強齋位於熙春路上，靜齋的旁邊，也是女生宿舍，強齋的名字取於[[易經|易經]]：“天行健，君子以自强不息”<ref name=sss/>。

===诚斋===
誠齋位於[[清華大學圖書館|逸夫馆]]附近，是建筑大师[[梁思成|梁思成]]提议保留大屋顶风格的古建筑，誠齋的名字來自於[[大學_(經傳)|大学]]：“知至而后意诚，意诚而后心正”<ref name=sss/>。而後為了興建[[清華大學圖書館|李文正圖書館]]，已經拆除<ref name=李文正>{{cite web |url=http://www.360doc.com/content/16/0510/19/30513115_558025184.shtml|title=【得体和谐的总体 和而不同的细节】访清华大学图书馆“李文正馆”设计者关肇邺院|publisher=清华大学|author=|language=zh-cn|date=|accessdate=2016-05-18}}</ref>。
===立斋===
立齋位於誠齋的旁邊，也是建筑大师[[梁思成|梁思成]]提议保留大屋顶风格的古建筑，立齋的名字來自於[[論語|論語]]：“夫仁者，己欲立而立人，己欲达而达人”<ref name=sss/>。而後成為清华大學国学院研究院<ref>{{cite web|url=http://tacl.tsinghua.edu.cn/|title=清华大学国学研究院|publisher=清华大学|author=|language=zh-cn|date=|accessdate=2016-02-28|deadurl=yes|archiveurl=https://web.archive.org/web/20160304195323/http://tacl.tsinghua.edu.cn/|archivedate=2016-03-04}}</ref>。但是後來為了興建[[清華大學圖書館|李文正圖書館]]，已經拆除<ref name=李文正/>。
