'''清江浦区'''是[[中国|中国]][[江苏省|江苏省]][[淮安市|淮安市]]所辖的一个[[市辖区|市辖区]]。1983年淮阴地区改设地级市，原县级[[清江市|清江市]]一分为二，分为[[清浦区|清浦区]]和[[清河区|清河区]]。2016年7月由清河区、清浦区合并组成新的清江浦区。

== 地理 ==
清江浦区的三面以河道与其它行政区相分隔：南面隔[[苏北灌溉总渠|苏北灌溉总渠]]与[[洪泽区|洪泽区]]相望；东南隔[[京杭大运河|京杭大运河]]与[[淮安区|淮安区]]相望；西南隔[[二河|二河]]与[[淮阴区|淮阴区]]相望。

清江浦区与周边行政区均处于辽阔的苏北平原，地形极为平坦。

清江浦区境内水系发达，河渠纵横，主要有[[京杭大运河|京杭大运河]]、[[里运河|里运河]]、[[苏北灌溉总渠|苏北灌溉总渠]]、[[淮河入海水道|淮河入海水道]]、[[二河|二河]]等。

==行政区划==

清江浦区辖18个[[街道办事处|街道办事处]]，3个镇，4个乡。

== 人口 ==

2001年，清浦区人口为31万。

== 教育 ==

国家级示范高中[[江苏省淮阴中学|江苏省淮阴中学]]位于清浦区境内。江苏省清浦中学（创建于1952年，曾用名城南民中、东方红中学）为江苏省重点中学。市属重点小学淮安市实验小学位于清浦区境内。区属重点小学有淮安市人民小学（原称人和小学，创办于1940年）、清浦小学（原名工农兵小学）等。
