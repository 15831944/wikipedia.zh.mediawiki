== 故事簡介 ==
山神立貴居住的玉津江町是個背靠山地，並由現人神管理的沿海城市。玉津江町的山側由立貴的妹妹山神水緒里管理，並治理得井井有條；而海側則沒有祭祀的神明。某天立貴在水緒里所管的山上的川中救了差點溺水的女孩春崎-{叶}-，並知道-{叶}-的身分是海側的海神。之後-{叶}-希望立貴協助自己熟悉海神的工作，而立貴也希望-{叶}-能早日獨當一面而伸出援手<ref>{{cite web|url=http://www.tgsmart.jp/article.aspx?a=18009|title=【連載企画第1回！】期待のクロシェット最新作『はるるみなもに！』の連載企画スタート！|website=TG Smart|date=2016-12-25|accessdate=2018-06-03|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|山神立貴|山神 立貴（やまがみ たつき）}}
: 作品男主角，玉津江海神學園二年級生。水緒里的哥哥，祭祀水緒里的神樂-{谷}-神社的宮司之子。
; {{nihongo|春崎-{叶}-|春ヶ崎 叶（はるがさき かなう）|聲：{{lang|ja|步サラ}}<ref name="CHARACTER">{{cite web|url=https://clochette-soft.jp/harumina/character.html|title=CHARACTER|publisher=Clochette|accessdate=2018年6月16日|language=ja}}</ref>}}
: 玉津江町海側的新任海神，個性直率且認真的女孩子，後來轉入立貴的班上就讀。神名為瑞寶-{叶}-比賣命。擁有不幸體質。
; {{nihongo|山神水緒里|山神 水緒里（やまがみ みおり）|聲：[[遙空|遙空]]<ref name="CHARACTER"/>}}
: 立貴的妹妹，神樂-{谷}-神社的二代祭神，神名為稚向津水緒里神，口頭禪是「嗯，我知道」。由於不能下山，因此養成了喜歡玩電玩和看DVD等愛好。
; {{nihongo|松房英麻|松房 英麻（まつふさ えま）|聲：{{lang|ja|花澤さくら}}<ref name="CHARACTER"/>}}
: 立貴和水緒里的青梅竹馬，也是立貴的同班同學，會對親近的人使用暱稱。家中在玉津江町海側的街上開設和菓子店。
; {{nihongo|幡上芽以|幡上 芽以（はたがみ めい）|聲：[[澤澤砂羽|澤澤砂羽]]<ref name="CHARACTER"/>}}
: 雷神，神名為珍魂奇魂甕津雷媛神，個性認真。在玉津江發生窮神事件時前來幫忙-{叶}-解決問題，並因此在玉津江町定居。
; {{nihongo|伊吹明日海|伊吹 明日海（いぶき あすみ）|聲：波奈束風景<ref name="CHARACTER"/>}}
: 玉津江海神學園二年級生，擁有冰山美人氣質的女孩子。父親由於發生船難而過世，所以非常討厭沒有庇護父親的神明，也因此對身為新任海神的-{叶}-相當敵視。
