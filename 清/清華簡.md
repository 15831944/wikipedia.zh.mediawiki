'''清華大學藏戰國竹簡'''，略稱'''清華簡'''<ref>《走近清華簡》，刘国忠，高等教育出版社</ref>，係指2008年7月由[[清華大學|清華大學]]校友捐贈給母校的一批[[竹簡|竹簡]]，被鑒定為[[戰國|戰國]]中期偏晚的重要文物，據信包含《[[尚書_(經)|尚書]]》的部分篇章以及[[周武王|周武王]]時期的樂詩等內容。目前仍在整理研究中。清華大學為此成立了[[清華大學出土文献研究与保护中心|清華大學出土文献研究与保护中心]]，<ref>{{cite web|author1=顾淑霞|author2=刘蔚如|title=清华大学成立出土文献研究与保护中心|url=http://news.tsinghua.edu.cn/new/news.php?id=20356|website=清华新闻网|accessdate=2018-01-26|archiveurl=https://web.archive.org/web/20090924035919/http://news.tsinghua.edu.cn/new/news.php?id=20356|archivedate=2009-09-24|language=zh-cn|date=2009-04-26}}</ref>並承諾資助相關學者。

截至2016年為止，僅三分之一左右的清華簡發表，預計還將出版十輯。<ref>{{cite web|author=張彥|title=中國一場革命性的發現|url=https://cn.nytimes.com/china/20160726/cc26bambooslips/|website=[[纽约时报|纽约时报]]|accessdate=2018-01-26|language=zh|date=2016-07-26}}</ref>

== 歷程 ==
“清华简”形制多种多样，最长的有46-{zh:厘米;zh-hans:厘米;zh-hk:厘米;zh-tw:公分;}-，最短的不到10-{zh:厘米;zh-hans:厘米;zh-hk:厘米;zh-tw:公分;}-。此批竹簡應是[[盗墓|盗墓者]]掠賣而流散海外的，後來清华大学校友[[赵伟国|赵伟国]]从[[香港|香港]]购回竹简，2008年7月15日捐予清華大學收藏。按惯例，这批由清华收藏的竹简被称为“清华简”。同年10月初，对竹简的第一阶段保护清理完成，其年代估計為公元前約三至四世紀；據清華大學宣布，它是“中国战国时期的重要文物，大多在迄今已经发现的先秦竹简中还没有见到过，其涉及到中国传统文化的核心内容，是前所罕见的重大发现”。同年12月受清華委托，[[北京大学|北京大学]][[加速器质谱实验室|加速器质谱实验室]]、[[第四纪年代测定实验室|第四纪年代测定实验室]]对清华简的无字残片样品作[[加速器質譜法|AMS]][[放射性碳定年法|碳14年代测定]]，经[[树轮校正|树轮校正]]的年份是公元前305±30年，即[[战国|战国]]中晚期，与由[[古文_(文字)|古文]]字学观察的结果一致。

第二階段工作由2008年11月起至2009年1月12日清點保护，並於同年3月開始初步释读研究的第三階段工作。為進一步整理研究，清華大學2009年4月25日成立出土文献研究与保护中心（简称出土文献中心），其主任由清華大學歷史系[[教授|教授]]、[[夏商周断代工程|夏商周断代工程]]专家组组长[[李学勤|李学勤]]教授担任，並同时宣布清华简最终确定为2388枚（包括少数残片）、共14支，除還有個別殘斷之處沒找出來外，基本已完整。李学勤指出，這批竹簡既有歷史價值，又有文學意義。書簡中第一批整理出的簡書為[[周文王|周文王]]臨終時對其子[[周武王|發]]的遺言；原文没有篇題，现据篇文试题为《[[保训|保训]]》。而最新的发现是周武王時期的乐诗，李学勤表示，“有的诗竟与《[[诗经|诗经]]·[[國風|國風]]·蟋蟀》一诗有关，前所未见，令人惊奇”；並稱竹簡上恰恰記載戰國時期的勸酒樂詩，可能與本已失傳的《[[樂經|樂經]]》有關係。一部带有纪年性质的著作《[[系年|-{繫}-年]]》，主要记述了从西周初年一直到战国前期的历史。《[[算表|算表]]》被认为是目前发现最早的实用算具，距今2300多年，计算功能超越[[里耶秦简|里耶秦简]][[九九表|九九表]]和[[张家界汉简|张家界汉简]][[九九表|九九表]]等。它能快速计算99.5以内的两个任意整数的乘积，还能计算包含特殊分数「半」的两位数乘法。<ref>[http://news.sciencenet.cn/htmlnews/2014/1/287222.shtm 清华大学发布校藏战国竹简（肆）研究成果]，科学网转自央视新闻：2014-1-7 </ref>

== 質疑 ==
*學者[[姜廣輝|姜廣輝]]等對「清華簡」《保訓》篇內容提出質疑，例如認為周文王在位期間沒有稱王，這點清華簡內容矛盾。然而文王是否稱王，本身就是未解決的歷史懸案；此外，姜也指出該篇中「昔舜舊作小人，親耕於歷丘」不符合傳統說法，因為天子微時應作「躬耕」，上述寫法屬於明顯的錯誤。篇中文句與現存古代文獻有許多雷同之處，有抄襲嫌疑。<ref>{{Cite news| url=http://www.tsinghua.edu.cn/publish/news/4215/2011/20110225232304234871349/20110225232304234871349_.html| title=“清华简”鉴定可能要经历一个长期过程——再谈对《保训》篇的疑问| author=姜广辉| date=2009-06-08| publisher=光明日报| deadurl=yes| archiveurl=https://web.archive.org/web/20160808035721/http://www.tsinghua.edu.cn/publish/news/4215/2011/20110225232304234871349/20110225232304234871349_.html| archivedate=2016-08-08}}</ref><ref>{{Cite journal|journal=《故宫博物院院刊》|date=2013年|issue=第4期|author1=姜广辉| author2=付赞 |author3=邱梦燕|title=《清华简《耆夜》为伪作考》|url=http://wap.cnki.net/qikan-GGBW201304007.html}}</ref><ref>{{Cite journal  |title=清华简《耆夜》没有确证证明为伪作 |author=周宝宏 | journal=《中原文化研究》}}</ref>
