'''清邁府'''（{{thai|t=จังหวัดเชียงใหม่|r=Changwat Chiang Mai}}，{{IPA-th|t͡ɕāŋ.wàt t͡ɕʰīaŋ.màj}}；[[北部泰語|蘭納語]]：{{lang|nod|ᨩ᩠ᨿᨦᩉᩲ᩠ᨾ᩵}}，{{IPA|tɕiaŋ.màj}}），[[泰國北部地區|泰國北部]]邊陲的一個府，北面與[[緬甸|緬甸]]接壤。另外，清邁府與另外5個府為鄰：[[清萊府|清萊府]]（東北）、[[南邦府|南邦府]]（東）、[[南奔府|南奔府]]（東南）、[[達府|達府]]（南）及[[夜豐頌府|夜豐頌府]]（西）。面積為20,107[[平方公里|平方公里]]。府治為[[清迈|清迈]]市。

== 歷史 ==
1274年（至元十一年），[[元朝|元朝]]設置[[雲南等處行中書省|雲南等處行中書省]]，清邁屬於雲南等處行中書省所管轄。

1292年，[[傣阮|傣阮]]族人庸那伽國王[[孟萊王|孟萊王]]（พญาเม็งราย，Phaya Mengrai）於清邁城建立[[蘭納|蘭納]]王國。

1599年，[[阿瑜陀耶王國|阿瑜陀耶王國]]佔領了這裡。

== 地理 ==
面积约20,107平方公里，距离[[曼谷|曼谷]]约700公里。

清邁府位處泰北山區和滨河流域，連綿山脈呈南北走向。本府有泰國最高山峰[[因他暖山|因他暖山]]（海拔2,565公尺高，距府治清邁市西南偏西約40公里。[[昭披耶河|昭披耶河]]的其中一條支流濱河（Mae Ping）源自該府，並流經清邁市順流南下。清迈府有丰富的动植物资源、自然风光和名胜古迹，府內設有7個[[國家公園|國家公園]]。

== 人口 ==
截至2005年清邁府人口為1,649,457人。

主要族群是說[[泰語|泰語]]的[[泰沅|泰沅]]，另外有13.4%的人口是山區少數民族，包括[[阿卡族|阿卡族]]、[[嘎良族|嘎良族]]([[克倫族|克倫族]])、[[拉胡族|拉胡族]]、[[苗族|苗族]]、[[瑤族|瑤族]]等。

== 行政區域 ==

清邁府共分為24縣（''Amphoe''）；再分成共204區（''Tambon''）；並再細分成1,915村（''Muban''）。

== 交通 ==
* '''汽车：'''在清迈市有多处出租汽车的地方，清迈市位于11号公路（Super Highway）。
* '''火车：'''清迈市是泰国火车北线的终点。
* '''[[雙排車|雙排車]]（Songthaew）：'''是一种皮卡改装的两排座的公共交通工具。
* '''[[嘟嘟車|嘟嘟車]]：'''是本地人和外国游客都普遍使用的短程交通工具。
* '''长途大巴：'''清迈市有通往曼谷以及其他各府的大巴，线路繁忙，为府到府直接的主要交通工具，一般以空调车为主。府内的长途车一般没有空调。
* '''自行车和摩托车：'''都很容易在清迈市租到。
* '''飞机：'''[[清迈国际机场|清迈国际机场]]是泰国的七个国际机场之一，是泰北旅游观光的重要门户。

== 旅遊 ==

* '''班達歪木雕村'''（Bantawai Village）——位於清邁古城西南方約25公里。是一處全村幾乎都在從事木材雕刻的小村莊。那些木[[雕塑|雕塑]]成為泰北地區特產之一。
* '''博桑'''（Borsang）——位於清邁以東的訕甘烹縣內的博桑村，是泰國北部最著名的清邁傘製作中心。博桑村民逐漸發展出自己特有的製傘技術，傳統的清邁[[紙傘|紙傘]]和棉布[[傘|傘]]甚受遊客歡迎，已成為當地農民農閒之暇，增添收入的來源。
* '''湄沙'''（Maesa）——位於清邁市以西的Mae Rim區，較多自然風景區。還分別有訓練大象和水牛這兩種泰北重要的動物的學校、Sainanping蘭花園等。
* '''诗丽吉王后植物花園'''（Queen Sirkit Bontanic Garden）——位於清邁市西北27公里的山區內，是泰國第一所具國際水準的植物園，建立於1993年，目的在保護泰國特有的植物生態，為[[植物學|植物學]]研究的基地，佔地960公頃的高科技植物園內，擁有8個玻璃[[溫室|溫室]]及4個大型展示場地。展示內容包括自然科學、泰國[[蘭花|蘭花]]培育、泰國植物、葯草品種、石頭花園等。詩麗吉王后植物花園是泰國政府在1980年代決定發展泰北地區時，因為泰國王后[[诗丽吉王后|诗丽吉王后]]對泰北人文發展甚為關注，所以特別設立這所具研究發展的院所。
* '''Doi Suthep-Pui國家公園'''和'''湄沙瀑布'''——1962年泰王[[普密蓬·阿杜德|普密蓬·阿杜德]]伉儷曾經到瀑布區遊覽，此後便開發成為泰國王室在泰北接待貴賓遊覽的主要景點。目前瀑布風景區的範圍約50平方公里，由10個[[瀑布|瀑布]]所組成。
