'''清齒齦塞音'''是[[輔音|輔音]]的一種，用於一些[[口語|口語]]中。清齒齦塞音、[[清齒塞音|齒音]]和[[清齒齦後塞音|齒齦後音]]在國際音標的符號都是{{IPA|{{尖括号|t}}}}，X-SAMPA音標的符號則是{{尖括号|<tt>t</tt>}}。

/t/十分普遍，幾乎所有語言都有原位/t/音，有的甚至有多於一種/t/的變種。許多[[印度|印度]]的語言，尤其是[[印地語|印地語]]，以及中國各地的方言，都有區分不送氣的/t/和送氣的/tʰ/。為強調不送氣，不送氣的/t/也可寫作/t˭/，但即使不添加不送氣符號「˭」，原位/t/的正常發音就是不送氣的。現時已知的語言中，只有在[[尼依夏島|尼依夏島]]外的[[夏威夷語|夏威夷語]]和口頭的[[薩摩亞語|薩摩亞語]]沒有/t/。薩摩亞語亦沒有/[[齒齦鼻音|n]]/。

== 該音見於漢語 ==
[[普通話|普通話]]中有/t/和/tʰ/兩種，均只出現在字音開首，拼音時分別以d和t表示。/tʲ/亦有存在於普通話，當/t/或/tʰ/出現在以i或ü作開首的[[韻母|韻母]]前，就會被顎音化成/tʲ/和/tʰʲ/。

[[吳語|吳語]]有/t/和/tʰ/，均只出現在字音開首，拼音時分別以t及th表示。依韻書舌頭音端聲透聲作別。

[[粵語|粵語]]中則有/t/、/tʰ/和/t̚/三種，拼音時/t/以d表示，/tʰ/和/t̚/則以t表示。/t/、/tʰ/只出現在字音開首，/t̚/則只出現在入聲字。

== 該音見於英語 ==
英語中有原位/t/和送氣的/tʰ/，兩者是[[同位異音|同位異音]]，均以t表示。

當t出現在重音音節開首或字的開首時會變成/tʰ/，如''try''、''senatorial''、''today''。但若t出現的位置既非字的開首，又非重音音節開首的話，如''palatable''、''theater''，在大部分[[美式英語|北美口音]]中會變成[[齒齦閃音|齒齦閃音]]，在英國南部口音中會變成[[喉塞音|喉塞音]]，而其他地方的口音則會成送氣/tʰ/或原位/t/。如果t出現在s之後，成為st，如stop, strain, 或 register，就會讀成不送氣的/t/。t出現於字末，會讀成/t/，例如pit, waist、和apt。如果/t/和[[齒齦無擦通音|齒齦無擦通音]]/ɹ/並排，串成tr，那麼t會讀成/tʰ/，而原本是濁音的/ɹ/則變成清音，如entrance。

== 該音見於格魯吉亞語 ==
[[格魯吉亞語|格魯吉亞語]]中有/tʰ/和/tʼ/，兩個是獨立的[[音素|音素]]，並非同位音。/tʰ/以თ標示；而/tʼ/則以ტ表示。

== 該音見於德語 ==
[[德語|德語]]和英語一樣，/t/和/tʰ/兩者是同位音。通常字母t、th及在字尾的d都唸成/tʰ/，而字母d則唸成/t/(部分方言唸成/d/)。

== 該音見於葡萄牙語 ==
[[葡萄牙語|葡萄牙語]]只有/t/一個音素。但[[巴西|巴西口音]]的葡萄牙語在/[[閉前不圓唇元音|i]]/前會讀成/tʃ/，通常串成ti或者非重音的te。
