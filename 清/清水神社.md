'''清水神社'''是[[臺灣日治時期|臺灣日治時期]]設於[[臺中州|臺中州]][[大甲郡|大甲郡]][[清水街|清水街]]字清水{{r|金1}}（今[[臺中市|臺中市]][[清水區|清水區]]牛罵頭遺址文化園區）的[[神社|神社]]。

== 歷史 ==
清水神社是[[台灣日治時期|台灣日治時期]]受[[皇民化運動|皇民化運動]]影響作為[[大甲郡|大甲郡]]的神社{{r|金2}}，選址於[[臺中州|臺中州]][[大甲郡|大甲郡]][[清水街|清水街]]字清水{{r|金1}}內原為清水街長周貽楚闢建的運動場處{{r|牛1}}（今臺中市清水區牛罵頭遺址文化園區）興建。耗資67000圓{{r|金2}}，於昭和12年（1937年）動工{{r|蘇}}，同年11月23日鎮座{{r|金1}}，[[近代社格制度|社格]]為鄉社（昭和17年（1942年）10月30日列格）{{r|蘇}}{{r|金1}}。祭祀[[天照大神|天照大神]]、[[開拓三神|開拓三神]]（[[大國主命|大國主命]]、[[大己貴命|大己貴命]]、[[少彥名命|少彥名命]]）和[[北白川宮能久親王|北白川宮能久親王]]{{r|金1}}{{r|蘇}}，例祭日10月17日{{r|金1}}，社紋為[[八咫鏡|八咫鏡]]（外圈）和不明之菊紋變形（內圈）<ref>{{Cite web |url =http://proj1.sinica.edu.tw/photo/subject/search/architech02_pic01.html|title =清水神社社務所前，兩側可見木造吊燈籠與其上的清水神社社紋|author =清水農會|date =1937-11-23|publisher =中央研究院，老照片資料庫檢索|language =中文（台灣）}}</ref>。興建當時可能已有部分牛罵頭遺址文物出土，社掌西村昌隆（此處可能是來源文章之筆誤，1939年時任社掌應為西村昌澄<ref> {{Cite book | author =臺灣總督府文教局社會課| title =[昭和十四年度] 臺灣總督府文教局社會課 臺灣ニ於ケル神社及宗敎，臺灣圖書館，日治時期圖書影像系統（需密碼）| location =臺北市錦町| publisher =臺灣總督府文教局社會課| date =1940-01-19| pages =原書第12頁| url =http://hyerm.ntl.edu.tw:2135/cgi-bin/gs32/gsweb.cgi/ccd=OdqAEp/record?r1=50&h1=0| language =ja}} </ref>）即收藏了部分遺址出土石器及陶器，1943年日本學者國分直一正式發現「牛罵頭遺址」{{r|牛1}}。

戰後神社廢止，1950年代成為中華民國陸軍飛彈（或砲兵{{r|牛2}}）部隊營區直到1997年7月{{r|蘇}}軍方裁併營區將土地還給臺中縣政府{{r|牛2}}。2002年臺中縣文化局將牛罵頭遺址指定為縣定古蹟（現為市定古蹟）{{r|牛2}}，後將此區列為牛罵頭遺址文化園區{{r|蘇}}。2013年將原營區建物與原神社狛犬以「鰲峰山營區及原清水神社遺構」之名公告為[[歷史建築|歷史建築]]<ref>{{Cite web |url =http://www.tchac.taichung.gov.tw/historybuilding?uid=34&pid=23|title = 歷史建築 鰲峰山營區及原清水神社遺構|publisher =臺中市文化資產處|language = 中文（台灣）}}</ref>。

== 現況 ==
原清水神社尚存參道路徑，和位於牛罵頭遺址文化園區內的狛犬1對、集中安置的燈籠構件數枚（含基壇3枚、大石燈籠竿1枚、中台4枚、火袋殘塊數枚、笠1枚、寶珠2枚）、原二之鳥居旁的部分玉垣<ref>{{Cite web |url =https://www.google.com.tw/maps/@24.2681839,120.5803945,3a,15y,333.7h,91.32t/data=!3m6!1e1!3m4!1sXMTvoKsenauAUwX6ry1o5g!2e0!7i13312!8i6656?hl=zh-TW|title = google地圖街景中的原清水神社玉垣遺跡|date =2017-04|language =中文（台灣）}}</ref>，以及位於後參道旁民宅的燈籠構件"中台"和"笠"各1枚<ref>{{Cite web |url =http://mypaper.pchome.com.tw/4astone/post/1372306066|title =臺中清水神社狛犬篇 內文中標註為"神社崎步道旁埋有石燈籠殘件"的照片|author = 磅礴|date =2016-01-08|publisher =PChome個人新聞台|language =中文（台灣）}}</ref>。其中清水神社表參道入口約位於今中航路三段近大街路口處<ref>{{Cite web |url =https://www.google.com.tw/maps/@24.2649154,120.576664,3a,60y,23.63h,89.2t/data=!3m6!1e1!3m4!1skLpLDQVx2X0uy-QjqkUkqw!2e0!7i13312!8i6656?hl=zh-TW|title =google地圖街景中的清水神社表參道入口處一帶|date =2017-04|language =中文（台灣）}}</ref>，此處原建有一之鳥居<ref>{{Cite web |url =https://jj7589.tian.yam.com/album/8886315#380862039|title ="清水神社風景"明信片，可見一之鳥居和兩旁的大石燈籠1對與其後的石燈籠數對|author =心，formosa|date =2011-08-23|publisher =天空部落|language =中文（台灣）}}</ref><ref>{{Cite web |url =http://proj1.sinica.edu.tw/photo/subject/search/history_pic03.html|title =清水神社參道第一鳥居，可見兩旁的大石燈籠1對與其後的石燈籠數對|author =清水農會|date =1937-11-23|publisher =中央研究院，老照片資料庫檢所|language =中文（台灣）}}</ref><ref>{{Cite web |url = http://slowlife-tw.com/zh/blog/qingshuihistory/|title =台中清水（＝牛罵頭）的現在與歷史 內文中標註為"清水舊照片11"的圖片，可見一之鳥居和兩旁的大石燈籠1對與其後的石燈籠數對|author =大沼敬吾|date =2014-05-16|publisher =台中slowlife|language =中文（台灣）}}</ref>，在其後沿中航路三段接鰲海路抵達牛罵頭遺址文化園區<ref>{{Cite web |url =http://www.ccartoa.org.tw/news/2015/150221.html|title =大甲郡鳥瞰地圖 內文下方"大甲郡展望"圖（請自行放大）|author =廖泫銘|publisher =中華民國地圖協會|language =中文（台灣）}}</ref>；另有名為神社崎的後參道{{r|金1}}，是今清水地政事務所旁連接大街路與鰲海路的坡道<ref>{{Cite web |url =https://www.google.com.tw/maps/@24.2690647,120.5785439,3a,40.9y,91.06h,95.7t/data=!3m6!1e1!3m4!1sGIMHgUKfku61eNd4wuAnkA!2e0!7i13312!8i6656?hl=zh-TW|title =google地圖街景中的清水地政事務所與後參道大街路口|date =2017-04|language =中文（台灣）}}</ref>。
