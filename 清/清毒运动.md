'''清毒运动'''，是[[抗日战争|抗日战争]]时期，1943年至1944年[[中国共产党南京市委员会|中共南京工委]]在[[汪精卫政权|汪精卫政权]]的首都[[南京|南京]]领导学生团体开展的一项清除[[鸦片|鸦片]]、捣毁烟馆、焚烧烟土的运动。

== 背景 ==
抗日战争期间，[[侵华日军|侵华日军]]在占领上海、南京后，日本烟商通过[[沪宁线|沪宁线]]贩运大量鸦片烟土进行鸦片生意。1939年，南京市内仅领取执照的贩毒商家就有30余户，领取执照的鸦片烟馆有175家，领取售烟执照的旅馆有14家。全市每天平均卖出鸦片烟3000两，营业收入6.6万元。苏、皖、浙三省的日本鸦片烟商，每月向[[汪精卫政权|汪精卫政府]]行政院交鸦片税多达300万元<ref name=1sa>{{Cite book|title=南京人民革命史党员干部读本|last=中共南京市委党史工作办公室编|first=|publisher=北京：[[中共党史出版社|中共党史出版社]]|year=2011.06|isbn=7-5098-1129-5|location=|pages=69-71}}</ref>。

1943年，[[汪精卫|汪精卫]]意图掌握鸦片[[專賣|专卖]]，授权[[陈公博|陈公博]]向日方交涉取消[[宏济善堂_(华中)|华中宏济善堂]]，并计划在行政院下设禁烟机构。由于日方不答应，陈公博怂恿行政院宣传部长[[林柏生|林柏生]]策动青年学生以游行示威的方式发动禁烟运动（即禁绝烟、赌、舞三害运动）<ref>{{Cite book|title=孤岛见闻 抗战时期的上海|last=陶菊隐著|first=|publisher=上海：上海人民出版社|year=1979.11|isbn=|location=|pages=271-272}}</ref>。1943年冬，林柏生派人到[[國立中央大學_(汪精衛政府)|中央大学]]学生中物色人选，策划发动禁烟运动，并找到[[青年救国社|青年救国社]]（以下简称青救社）社员[[厉恩虞|厉恩虞]]和[[王嘉谟_(1923年)|王嘉谟]]。厉恩虞、王嘉谟两人同青救社的[[潘田|潘田]]、[[姚禹谟|姚禹谟]]等人商量，认为可以利用这个机会发动群众<ref>{{Cite book|title=中外禁毒史|last=牛何兰主编|first=|publisher=昆明：[[云南人民出版社|云南人民出版社]]|year=2012.06|isbn=978-7-222-09389-8|location=|pages=41}}</ref>。[[潘田|潘田]]（中共党员）遂向[[中国共产党南京市委员会|中共南京工委]]书记[[舒诚|舒诚]]汇报，舒诚同意发动清毒运动<ref>{{Cite book|title=南京抗日故事|last=[[姜寒冰|姜寒冰]]编著|first=|publisher=北京：[[中国文史出版社|中国文史出版社]]|year=2005.03|isbn=7-5034-1586-X|location=|pages=155-156}}</ref>。中共地下党认为消灭烟毒并不现实，但运动可以激励群众抗日，发展进步力量，借此利用林柏生掩护中共地下工作<ref>{{Cite book|title=南京党史资料 第4辑|last=中共南京市委党史编写领导小组办公室，南京市档案局编|first=|publisher=中共南京市委党史编写领导小组办公室，南京市档案局编|year=1983.09|isbn=|location=|pages=75-76}}</ref>。

== 经过 ==
1943年12月27日，南京[[国立中央大学_(汪精卫政府)|中央大学]]学生[[厉恩虞|厉恩虞]]等人动员南京各学校学生到[[南京夫子庙|南京夫子庙]]、[[朱雀路|朱雀路]]等地区破坏了大量烟馆并缴获大量烟具，遭到[[日本憲兵|日本宪兵]]队干涉<ref>{{Cite book|title=南京沦陷八年史 下|last=经盛鸿著|first=|publisher=北京：社会科学文献出版社|year=2005.07|isbn=7-80190-582-2|location=|pages=963}}</ref>。12月28日，中共地下党员、青救社成员又发动3000多名学生上街游行示威，去砸烟馆、焚烧烟土<ref>{{Cite book|title=江苏革命史词典|last=叶绪昌主编|first=|publisher=南京：[[南京大学出版社|南京大学出版社]]|year=1993.05|isbn=7-305-02105-9|location=|pages=404-405}}</ref>。游行队伍从夫子庙回到[[南京人民大会堂|国民大会堂]]时，因天色已晚，学生们将没收的烟土、烟具、麻将牌等集中焚烧<ref>{{Cite book|title=南京保卫战史|last=[[孙宅巍|孙宅巍]]著|first=|publisher=南京：南京出版社|year=2014.12|isbn=978-7-5533-0737-4|location=|pages=344}}</ref>。在国民大会堂的学生集会上，厉恩虞、王嘉谟主持并讲话<ref>{{Cite book|title=厉恩虞纪念集|last=贺崇寅主编；中共南京市委宣传部，中共南京市委党史工作办公室编|first=|publisher=上海：[[上海交通大学出版社|上海交通大学出版社]]|year=1999.12|isbn=7-313-02365-0|location=|pages=77-78}}</ref>，并宣告成立“首都学生清毒总会”，由厉恩虞、王嘉谟担任正、副会长<ref>{{Cite book|title=南京百年城市史 1912-2012 9 教育卷|last=徐承德，虞朝东著|first=|publisher=南京：南京出版社|year=2014.09|isbn=978-7-5533-0517-2|location=|pages=146}}</ref>。不久，南京各大、中学校纷纷设立分会，并派[[罗延俊|罗延俊]]、[[余力|余力]]、王士弘持林柏生的介绍信抵达上海<ref>{{Cite book|title=南京革命事典|last=中共南京市委党史工作办公室等编|first=|publisher=南京：[[南京出版社|南京出版社]]|year=2004.04|isbn=7-80614-903-1|location=|pages=264}}</ref>，向上海各大、中学学生代表介绍南京清毒运动情况<ref>{{Cite book|title=不屈的抗争 南京人民反抗日军暴行纪实|last=中共江苏省委党史工办，中共南京市委党史工办，侵华日军南京大屠杀遇难同胞纪念馆编|first=|publisher=北京：[[中央文献出版社|中央文献出版社]]|year=2006.12|isbn=7-5073-2236-X|location=|pages=107}}</ref>。

寒假期间，学生们以青救社社员为核心，组织清毒运动寒假工作团。1944年1月29日，寒假工作团与南京模范女中的“干字运动实践会”联合行动；行动中学生在[[丰富路|丰富路]]的毒枭[[曹玉成|曹玉成]]家搜出一大盒[[海洛因|海洛因]]，随即将曹玉成绑到[[新街口_(南京)|新街口]][[孫中山|孙中山]]铜像前，并向围观群众宣布其贩毒罪行；之后，学生当众焚烧海洛因<ref name=1s>{{Cite book|title=新四军与南京|last=新四军暨华中抗日根据地研究会南京分会，中共南京市委党史工作办公室，南京中共党史学会编|first=|publisher=[[中共党史出版社|中共党史出版社]]|year=2005.06|isbn=7-80199-231-8|location=北京|pages=524-525}}</ref>，并将曹玉成扭送给南京警察厅<ref>{{Cite book|title=新四军 1943-1945|last=胡兆才著|first=|publisher=西安：[[陕西人民出版社|陕西人民出版社]]|year=2013.12|isbn=978-7-224-10910-8|location=|pages=46-47}}</ref>。由于民愤极大，汪精卫政府下令枪毙曹玉成<ref name=1s />。同年2月<ref>{{Cite book|title=日本鸦片侵华资料集 1895-1945|last=中共河北省委党史研究室编|first=|publisher=中共河北省委党史研究室编|year=|isbn=|location=|pages=617}}</ref>，日方同意取消宏济善堂<ref>{{Cite book|title=鸦片 日本侵华毒品政策五十年 1895-1945|last=王宏斌著|first=|publisher=上海：[[上海社会科学院出版社|上海社会科学院出版社]]|year=2016.12|isbn=978-7-5520-1631-4|location=|pages=221-222}}</ref>，并在[[汪精卫政权#政府|汪精卫政权行政院内政部]]下设禁烟总局，宣布“分五年禁绝期”禁绝鸦片（1943年3月30日至1947年3月30日）。汪精卫更换内政部长[[陈群_(民国)|陈群]]为[[梅思平|梅思平]]，并改原戒烟总局为禁烟总局<ref>{{Cite book|title=上海抗日战争史丛书 日军在上海的罪行与统治|last=张铨，庄志龄，陈正卿著|first=|publisher=上海：[[上海人民出版社|上海人民出版社]]|year=2015.08|isbn=978-7-208-13092-0|location=|pages=254}}</ref>。

自禁烟总局成立后，梅思平颁布《限期禁绝售吸所办法》等法令，规定各地售吸所绝对关闭日期，回收特许经营，并新发土行牌照（15张大牌照、128张小牌照）。其中大土行牌照15张中当局留五张、前宏济善堂[[盛文颐|盛文颐]]（[[盛宣怀|盛宣怀]]之侄<ref>{{Cite book|title=粉墨春秋|last=高阳著|first=|publisher=南京：[[译林出版社|译林出版社]]|year=2013.08|isbn=978-7-5447-4077-7|location=|pages=385}}</ref>）等人分配五张，剩余五张分配特业工会<ref>{{Cite book|title=文史资料选辑 合订本 第13卷 第37辑|last=中国人民政治协商会议全国委员会；文史资料委员会《文史资料选辑》编|first=|publisher=北京：[[中国文史出版社|中国文史出版社]]|year=|isbn=|location=|pages=187-188}}</ref>。局方五张分别交给陈公博、[[梅思平|梅思平]]、林柏生、[[袁愈佺|袁愈佺]]和[[章骏|章骏]]各领一张，交由亲戚出面经营<ref>{{Cite book|title=往事今谈|last=[[江上清|江上清]]著|first=|publisher=[[致诚出版社|致诚出版社]]|year=1985.09|isbn=|location=|pages=71}}</ref>。此事引起土商愤怒，到四川路1286号禁烟总局办公室集会抗议，后经潮州帮土行斡旋才停止<ref>{{Cite book|title=汪精卫集团卖国朵敌批叛资料选编|last=南京大学马列主义教研室《汪精卫问题研究组》选编|first=|publisher=南京大学学报编辑部|year=1981.05|isbn=|location=|pages=98-99}}</ref>。

== 后续 ==
1944年4月，中共地下党员、中央大学农学院学生沈新（原名陈建）从日军宪兵队石井处探听日方拟定了有厉恩虞在内的十人黑名单<ref>{{Cite book|title=往事悠悠 赤子情怀 一个南大老校友的心声 絮语|last=戴逸尘著|first=|publisher=南京：[[南京大学出版社|南京大学出版社]]|year=2012.05|isbn=978-7-305-09929-8|location=|pages=150-165}}</ref><ref>{{Cite book|title=不屈的抗争 南京人民反抗日军暴行纪实|last=中共江苏省委党史工办，中共南京市委党史工办，侵华日军南京大屠杀遇难同胞纪念馆编|first=|publisher=北京：中央文献出版社|year=2006.12|isbn=7-5073-2236-X|location=|pages=113}}</ref>；同时，中共地下党外围组织“五兄弟”成员、中央大学工学院学生尚渊如也截获同样消息<ref>{{Cite book|title=青春无悔 南京青年运动和青年工作亲历者口述史|last=邢正军，陈丽君主编|first=|publisher=北京：中共党史出版社|year=2009.04|isbn=978-7-5098-0223-6|location=|pages=116-118}}</ref>。因此，中共地下党与青救社决定暂停各项活动<ref name=sa>{{Cite book|title=南京人民革命史|last=[[朱启銮|朱启銮]]主编；中共南京市委党史办公室编|first=|publisher=南京：[[南京出版社|南京出版社]]|year=1991.10|isbn=7-80560-521-1|location=|pages=217-219}}</ref>。

清毒运动中的学生积极分子，包括厉恩虞、[[王嘉猷|王嘉猷]]、[[朱井熙|朱井熙]]、[[李昭定|李昭定]]、[[程志定|程志定]]、[[江泽民|江泽民]]、[[曹天忠|曹天忠]]、[[余力|余力]]、[[蒋鹏龄|蒋鹏龄]]、[[周效良|周效良]]、[[高锡安|高锡安]]等相继加入中国共产党<ref name=gb />。中共地下党在实力增强后，在1946年4月成立了[[中共南京市委|中共南京市委]]，由[[陈修良|陈修良]]、刘峰担任正副书记<ref name=gb>{{Cite book|title=革命青年 满腔热血 青年江泽民与南京|last=中共南京市委党史工作办公室编|first=|publisher=北京：[[中央文献出版社|中央文献出版社]]|year=2013.12|isbn=978-7-5073-3994-9|location=|pages=192-195}}</ref>。
