'''清畠站'''（{{jpn|j=清畠駅|hg=きよはたえき|rm=Kiyohata eki|f=y}}）位於[[北海道|北海道]][[沙流郡|沙流郡]][[日高町|日高町]]字清畠町，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[日高本線|日高本線]]上的站。電報略號是'''{{lang|ja|ヨハ}}'''。

== 車站構造 ==
[[側式月台|側式月台]]1面1線的地面站。沒有站房，只有一座小型的候車室。 

== 車站周邊 ==
* [[國道235號|國道235號]]
* 清畠郵局
* 慶能舞川
* 慶能舞橋
* 日高町立清畠小學
* 道南巴士「清畠站前」站牌

== 歷史 ==
* 1926年（[[大正|大正]]15年）12月7日 - 由日高拓殖鐵道設立。此時稱為'''慶能舞站'''。
* 1927年（[[昭和|昭和]]2年）8月1日 - 政府買下日高線並收歸國有化，由國鐵管轄。
* 1944年（昭和19年）4月1日 - 改稱為'''清畠站'''。
* 1977年（昭和52年）2月1日 - 廢止貨物和行李處理的業務。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
