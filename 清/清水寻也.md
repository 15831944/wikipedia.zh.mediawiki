'''清水寻也'''（{{jpn|j='''清水 尋也'''|kk=しみず ひろや|rm=Shimizu Hiroya|f=no}}，{{bd|1999年|6月9日|||catIdx=清水}}），[[日本|日本]]男[[演员|演员]]，出身[[日本|日本]][[东京都|东京都]]。曾隸屬{{Link-ja|丰田事务所|トヨタオフィス}}<ref>{{Cite web |url=http://www.toyotaoffice.jp/hiroyashimizu/ |title=清水尋也 オフィシャルページ |accessdate=2015-02-24 |publisher=トヨタオフィス |archiveurl=https://web.archive.org/web/20130916070916/http://www.toyotaoffice.jp/hiroyashimizu/ |archivedate=2013年9月16日 |deadurl=yes |df= }}</ref>、{{Link-ja|Bites|バイツ}}<ref>{{Cite web |url=http://www.bites-inc.co.jp/ |title=Artist|accessdate=2017-12-28 |publisher=バイツ|archiveurl=https://web.archive.org/web/20170911230901/http://www.bites-inc.co.jp/|archivedate=2017-09-11}}</ref>，2017年11月6日之后，移籍至{{Link-ja|Office Saku|オフィス作}}。<ref>{{Link-ja|Office Saku|オフィス作}}公式twitter{{Twitter status|office_saku|927497252741844992|2017年11月6日|accessdate=2017-12-28}}</ref>清水寻也的哥哥是同为演员的{{Link-ja|清水尚弥|清水尚弥}}。

== 早年生活 ==
由于其兄长的影响，清水寻也从3岁开始就学习[[足球|足球]]。但由于清水寻也所念的小学并没有足球俱乐部，因此他在小学及初中期间对[[篮球|篮球]]有了极大的兴趣。<ref name="crea">{{Cite interview|subject=清水尋也|interviewer=くれい響|url=http://crea.bunshun.jp/articles/-/7691|title=話題の映画『ソロモンの偽証』で難役を演じ切った清水尋也|date=2015-04-17|program=CREA|accessdate=2017-12-28}}</ref>

在时间管理上，清水寻也自认比较松散。<ref name="movcol01">[http://www.moviecollection.jp/interview_new/detail.html?id=464 『ストレイヤーズ・クロニクル』清水尋也インタビュー] 2015-07-0''ムービーコレクション''</ref>

== 演艺生涯 ==
清水寻也4岁的时候，由其兄清水尚弥所主演电影《{{Link-ja|瞬间移动的少年|からっぽ (映画)}}》的试映会上，事务所方面向清水寻也发来了试镜邀约。<ref name="crea" /> 清水寻也最初对演艺事业感到恐惧，但他最后还是决定进入娱乐圈，因为他觉得能在这里度过快乐而充实的每一天。<ref name="crea" /> 关于他是否利用其兄长的人际关系时，清水寻也表示“我不在家里谈论工作方面的事情，因为这让我感到很羞耻”，“但我会看我哥哥的电影等作品，所以当我看过的话，我会和他谈及我的感受”。<ref name="movcol02">[https://www.youtube.com/watch?v=kX4cZ0e2z-A ［動画］『ストレイヤーズ・クロニクル』インタビューで俳優の兄・清水尚弥について語る] 2015-07-03''ムービーコレクション''</ref>

2012年，清水寻也出演了他的电影处女作《震动（{{lang|ja|震動}}）》。<ref name="crea2">{{Cite interview|subject=清水尋也|interviewer=くれい響|url=http://crea.bunshun.jp/articles/-/7691?page=2|title=（2ページ目）話題の映画『ソロモンの偽証』で難役を演じ切った清水尋也|date=2015-04-17|program=CREA|accessdate=2017-12-28}}</ref> 随后，他出演了[[富士电视网|富士电视网]]的电视剧《[[高中入学考|高中入学考]]》。<ref name="crea2" />

2014年，清水寻也出演了[[中岛哲也|中岛哲也]]所执导的电影《[[渴望。|渴望。]]》，饰演片中有着激烈霸凌情节的“我”；随后他又在系列电影《{{Link-ja|所罗门的伪证|ソロモンの偽証}}》饰演一个恐怖的不良少年，<ref name="houchi20141222" /> 引发关注。<ref name="modelpress20180131">{{Cite web| title = 【注目の人物】広瀬すず主演「anone」カノン役は清水尋也 「電影少女」にも出演中の最注目俳優| publisher = モデルプレス| date = 2018-01-31| url = https://mdpr.jp/interview/detail/1742430| accessdate = 2018-03-19}}</ref>
