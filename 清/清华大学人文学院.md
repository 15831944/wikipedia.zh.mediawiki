'''清华大学人文学院'''，2012年10月20日建立，前身是1925年建立的'''国学研究院'''，曾有[[王国维|王国维]]、[[梁启超|梁启超]]、[[陈寅恪|陈寅恪]]、[[赵元任|赵元任]]等知名教授，培养过[[曹禺|曹禺]]、[[钱锺书|钱锺书]]、[[季羡林|季羡林]]、[[查良铮|查良铮]]、[[英若诚|英若诚]]等知名作家。

== 沿革 ==
{|class="toccolours" style="float: right; margin-left: 1em; margin-right: 2em; font-size: 85%; background:#c6dbf7; color:black; width:28em; max-width: 40%;" cellspacing="5"
|style="text-align: left;"|　“大学不是职业学校，不只在训练职业人才，大学所训练出来的人只有特殊机能还是不够，他还是一个‘人’，所谓‘人’，就是对于世界社会有他自己的认识、看法，对已往及现在所有有价值的东西——文学、美术、音乐等都能欣赏，具备这些条件者就是一个‘人’。”
|-
|style="text-align: right;"| —— 冯友兰《论大学教育》<ref name="lio " />
|}

1911年，美国退还“[[庚子赔款|庚子赔款]]”，在清华大学建立了“清华留美预备学校”。<ref name="fish " >{{cite web |url=http://www.rwxy.tsinghua.edu.cn/publish/shss/1789/index.html |title=历史沿革 |accessdate=2012年10月21日 |publisher=清华大学 |work= |language =zh-cn}}</ref>

1923年，清华大学大学部正式建立。<ref name="fish " />

1925年，清华大学国学研究院正式建立，王国维、梁启超、陈寅恪、赵元任、李济、吴宓等担任导师、教授。1928年，国学研究院关闭。<ref name="fish " /><ref name="bull " >{{cite web |url=http://news.ifeng.com/mainland/detail_2012_10/21/18409909_0.shtml |title=清华大学成立人文学院 欲重振“国学院”辉煌 |accessdate= 2012年10月21日|publisher=凤凰网 |work= |language =zh-cn}}</ref>

1928年，清华大学组建文学院。<ref name="fish " /><ref name="bull " /><ref name="piac " >{{cite web |url=http://edu.qq.com/a/20121020/000148.htm |title= 清华大学成立人文学院|accessdate= 2012年10月20日|publisher=腾讯网 |work= |language =zh-cn}}</ref>

1952年，清华大学文学院、法学院被政府中止，只留下理工科：调离或搬迁到其他学校。<ref name="fish " /><ref name="bull " />

1978年，[[邓小平|邓小平]]恢复高考制度之后，清华大学慢慢恢复了外语系、社会科学系、中国语言文学系、思想文化研究所、教育研究所、科学技术与社会研究室和艺术教育中心。<ref name="fish " /><ref name="bull " /><ref name="piac " />

1993年12月，清华大学组建人文社会科学学院。<ref name="fish " /><ref name="bull " />

2012年10月20日，人文社会科学学院分为人文学院和[[清华大学社会科学学院|社会科学学院]]。<ref name="fish " /><ref name="bull " /><ref name="piac " />
