'''清源军'''（{{lang-nan|清源軍|poj=Chheng-goân-kun}}；945年－978年），964年后改称'''平海军'''（{{lang|nan|Pêng-hái-kun}}），是[[中國|中國]][[五代十国|五代十国]]时期的[[藩鎮割據|藩鎮割據]]政權，其疆域大致在[[閩南|閩南]]，政治中心位於[[泉州|泉州]]。清源軍前後歷經4位[[節度使|節度使]]或[[留後|留後]]（代理節度使）的統治，由[[閩國|閩國]]軍事將領[[留從效|留從效]]所建立。

== 歷史 ==
[[南唐|南唐]]灭闽（945年）后，闽地三分，南唐得[[建州|建州]]、[[汀州|汀州]]，[[吴越|吴越]]得[[福州|福州]]，而[[泉州|泉州]]、[[漳州|漳州]]为当地人[[留从效|留从效]]、[[陈洪进|陈洪进]]占据，称清源军、平海军，至太平兴国三年（978年）为[[宋|宋]]所灭。

== 外交 ==
西元945年，閩国滅亡。次年，留從效[[罷黜|罷黜]]泉州[[刺史|刺史]]兼[[閩國皇室列表|閩國皇室]]的[[王继勋_(闽)|王繼勳]]，掌控軍政，[[南唐|南唐]]因無法控制而追任留從效為泉州刺史，同年再任[[董思安|董思安]]為漳州刺史。949年，董思安遭留從效的兄長[[留從願|留從願]]毒殺，並自任刺史；因此，南唐只得改設立清源軍，並以留從效為[[節度使|節度使]]統治閩南。南唐後又封其為鄂國公，再進封為晉江王。

== 疆域 ==
清源军辖[[漳州|漳州]]、[[泉州|泉州]]2州14县。
