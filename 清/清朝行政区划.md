'''[[清朝|清朝]]'''版圖遼闊，民族眾多，在'''[[行政區劃|行政區劃]]'''上「因時順地、變通斟酌」<ref>道光皇帝《御製[[大清一統志|大清一統志]]序》</ref>。大體上，清朝在[[内地十八省|漢地]]實行[[行省|行省制]]，在[[中国东北地区|東北地區]]和[[新疆省_(清)|新疆]]實行[[駐劄大臣|軍府制]]，在[[清代蒙古|蒙古]]實行[[清代蒙古盟部旗列表|盟旗制]]，在[[清朝治藏歷史|西藏]]遵循元朝以來的行政區劃設置。

清末，在列強蠶食鯨吞的形勢下，邊疆各地依靠舊有的行政體制已無法維持有效的統治。[[光绪|光绪]]年間，[[新疆省_(清)|新疆]]、[[福建臺灣省|台灣]]與[[奉天省|奉天]]、[[吉林省_(清)|吉林]]、[[黑龍江省_(清)|黑龍江]]相繼建省，實行與[[内地十八省|漢地]]相同的行政體制。[[清代蒙古|蒙古]]、[[清朝治藏歷史|西藏]]、[[海南岛|海南]]也有建省之議，但在[[清朝|清朝滅亡]]之前未能實行。

== 疆域 ==
[[乾隆|乾隆]]二十四年（1759年）之后的清朝疆域：北起[[蒙古高原|蒙古]][[唐努乌梁海|唐努乌梁海]]地区及[[西伯利亚|西伯利亚]]，南至[[南海|南海]]，包括“千里石塘、万里长沙、曾母暗沙”（今[[南海诸岛|南海诸岛]]），西南达[[西藏自治区|西藏]]的[[达旺地区|达旺地区]]、[[云南省|云南省]]的[[南坎|南坎]]、[[江心坡|江心坡]]地区等[[缅甸|缅甸]]北部，西尽[[鹹海|鹹海]]与[[帕米爾高原|帕米爾高原]]地区，包括今[[新疆维吾尔自治区|新疆]]以及[[中亚|中亚]]的[[巴尔喀什湖|巴尔喀什湖]]，东北抵[[外兴安岭|外兴安岭]]，包括[[库页岛|库页岛]]，东南包括[[臺灣|臺灣]]、[[澎湖群島|澎湖群島]]。

清朝最大國土面積達1300多萬平方公里，在中國歷代為第二，并保持了一百多年。晚清國土面積縮減至約1107萬平方公里，不過還是屬俄國之外的世界第二大國，清朝國土之遼闊，由此可以想見。

== 沿革 ==
===漢地===
清代[[山海关|山海關]]以內、[[长城|长城]]以南的漢族地區被稱為「漢地」、「關內」或「內地」。漢地的行政區劃承襲明代「省—府（州）—縣」的層級體制。一級政區為[[承宣布政使司|布政使司]]，通稱“[[行省|行省]]”或“省”。二級政區為府、[[直隸州|直隸州]]。府管轄的州（散州、屬州）不再領縣，形成了單式的三級制。清代初年，原為臨時差官的[[巡撫|巡撫]]取代[[布政使|布政使]]，成為一省的長官。在一些民族雜居之處及戰略要地，設置了新型政區「[[厅_(行政区划)|廳]]」，分為省直轄的[[直隸廳|直隸廳]]和府轄的[[厅_(行政区划)|散廳]]。少數直隸廳下轄縣。

明代布政使司、[[提刑按察司|按察使司]]派出的差官「[[道員|道員]]」，在清代也保留下來。道員的統轄區域是「道」，介於省與府之間，有分巡道、分守道、糧儲道、鹽法道、兵備道等名目。清初的道並不是行政區，道員亦無品級。乾隆以後，定道員秩品為正四品，分巡道、分守道的職權也漸趨一致。有的道下直接領縣。有人認為清末的道實際上已成為省、府之間的行政區劃<ref>《清代政區沿革綜表》王鐘翰序</ref>。

====行省====
在行省設置方面，基本沿襲明代所置的兩京與十三布政使司，即南北二京及山東、山西、河南、陝西、浙江、福建、江西、廣東、廣西、湖廣、四川、雲南、貴州。順治元年（1644年）定鼎北京，以[[沈阳市|盛京]]為留都<ref>《大清一統志》盛京</ref>。二年（1645年）改[[北直隶|北直隶]]為[[直隶省|直隶省]]，改[[南直隸|南直隸]]為[[江南省|江南省]]。康熙三年（1664年），分湖廣為湖北、湖南二省。康熙六年（1667年），江南省正式分為江蘇、安徽二省。康熙七年，陝西省正式分為陝西、甘肅二省，自此形成了所謂「漢地十八省」的格局。

光緒十一年（1885年），分福建省[[臺灣府|臺灣府]]置[[福建臺灣省|臺灣省]]。兩年後臺灣正式建省，稱「[[福建臺灣省|福建臺灣省]]」。光緒二十一年（1895年），因[[甲午战争|甲午战争]]戰敗，臺灣省被割讓予[[日本|日本]]，遂被廢除。光緒三十年十二月（1905年1月），分江蘇江寧、淮安、揚州、徐州四府及通州、海州二直隸州置[[江淮省|江淮省]]，旋即裁撤。此後至清末，內地仍為十八省，與[[東三省|東三省]]、[[新疆省_(清)|新疆省]]合為二十二省。

==== 衝繁疲難 ====
清代的府、州、廳、縣，按照「[[衝繁疲難|衝、繁、疲、難]]」的考語分為不同等次。考語字數越多，地位就越重要。一般以四字俱全者為「最要缺」，三字者（衝繁難、衝疲難、繁疲難）為「要缺」，二字者（衝繁、繁難、繁疲、疲難、衝難、衝疲）為「中缺」、一字或無字者為「簡缺」<ref>《中國歷代行政區劃的變遷》，147頁</ref>。
* 衝：地當孔道者為衝
* 繁：政務紛紜者為繁
* 疲：賦多逋欠者為疲
* 難：民刁俗悍，命盜案多者為難

====土司====
雲南、貴州、廣西、四川、湖南、湖北、甘肅等省設有[[土司|土司]]，分為[[宣慰使司都元帅府|宣慰司]]、宣撫司、招討司、安撫司和長官司（長官為武職），與土府、土州、土縣（長官為文職）。土司的長官以當地各族頭人充任，可以世襲，由朝廷或地方官府頒給印信，歸所在地方之督撫、[[駐紮大臣|駐紮大臣]]管轄。宣慰等司的長官隸屬於[[兵部|兵部]]、土知府、土知州等官隸屬於[[吏部|吏部]]<ref>《清史稿》土司傳</ref>。雍正年間，雲南、貴州、廣西等省的土司開始改行流官制，史稱[[改土归流|改土歸流]]。光緒、宣統之際，[[赵尔丰|赵尔丰]]出任川滇邊務大臣，四川西部的藏人土司、西藏東部的宗也開始改土歸流。

===東三省===
東北為清朝龍興之地。[[顺治|顺治]]年間[[入關戰爭|入關]]後，以駐防八旗留守盛京[[沈阳市|瀋陽]]。康熙至乾隆年間，逐漸形成三個將軍轄區：'''盛京'''（奉天）、'''吉林'''、'''黑龍江'''，地位異於內地之行省。將軍之下設[[都統|專城副都統]]分駐各城，並管理各城的臨近地區。副都統下有[[总管|總管]]統領各旗。在漢民聚居之處，置府、州、縣、廳，如同關內。居於[[黑龙江|黑龍江]]、[[嫩江|嫩江]]中上游的[[巴爾虎|巴爾虎]]、[[达斡尔族|達斡爾]]、[[索伦部|索倫]]（鄂溫克）、[[鄂伦春族|鄂倫春]]、[[锡伯族|錫伯]]等族，編入八旗，由[[布特哈總管|布特哈總管]]、[[呼倫貝爾總管|呼倫貝爾總管]]管轄。黑龍江、[[烏蘇里江|烏蘇里江]]下游及[[庫頁島|庫頁島]]的[[赫哲|赫哲]]、[[尼夫赫人|費雅喀]]、庫頁、奇楞等漁獵部落則分設姓長、鄉長，由[[三姓副都統|三姓副都統]]管轄。

光緒末年的[[甲午戰爭|甲午戰爭]]、[[八國聯軍之役|八國聯軍之役]]與[[日俄戰爭|日俄戰爭]]嚴重動搖了清朝在東北地區的統治，迫使其廢除滿洲的旗民分治制度，設立行省。光緒三十三年（1907年），廢除盛京、吉林、黑龍江三地將軍衙門，改設[[奉天省|奉天省]]、[[吉林省|吉林省]]、[[黑龍江省|黑龍江省]]；隨後裁撤各城副都統、總管，改為府、廳、州、縣。宣統三年（1911年），奉天省領八府、八廳、六州、三十三縣；吉林省領十一府、一州、五廳、十八縣；黑龍江省領七府、六廳、一州、七縣<ref>《清史稿》地理志二、三、四</ref>。

===藩部===
清代蒙古、西藏、青海、新疆与黑龙江[[布特哈|布特哈]]（[[达斡尔族|达呼尔]]、[[索伦部|索伦]]、[[鄂伦春族|鄂伦春]]等族）被称为[[藩部|藩部]]，由[[理藩院|理藩院]]管理。

====蒙古====
{{Main|清代蒙古}}
{{Main list|清代蒙古盟部旗列表}}

明清之際，蒙古分為眾多部落（蒙古語稱為“艾馬克”），部落首領為“部長”（鄂拓克）或“汗”。[[皇太極|清太宗]]時，依照滿洲[[八旗制度|八旗]]的組織形式，將蒙古各部落編為[[旗_(行政区划)|旗]]，完成[[建旗划界|建旗划界]]后，因各旗牧地固定，“旗”成为蒙古地区的基本行政單位，其長官為[[扎薩克|扎薩克]]或[[總管|總管]]。旗下設“佐”（[[苏木_(行政区划)|蘇木]]），相當於[[乡级行政区|鄉]]。自此蒙古各部落被納入統一的行政體系之中。在地域上，蒙古地區大致分為[[察哈尔部|察哈爾]]、[[內蒙古|內蒙古]]、[[西套蒙古|西套蒙古]]、[[外蒙古|外蒙古]]（包括[[土謝圖汗部|土謝圖汗部]]、[[賽音諾顏部|賽音諾顏部]]、[[車臣汗部|車臣汗部]]、[[札薩克圖汗部|札薩克圖汗部]]）、[[科布多|科布多]]與[[唐努乌梁海|唐努乌梁海]]。

清代蒙古又分為[[內屬蒙古|內屬蒙古]]與[[外藩蒙古|外藩蒙古]]。內屬蒙古包括察哈爾、[[歸化城土默特|歸化城土默特]]、唐努烏梁海、[[阿爾泰烏梁海|阿爾泰烏梁海]]等部，各旗由朝廷派遣官員（一般為總管）治理，與內地的[[州|州]]、[[縣|縣]]無異。外藩蒙古各旗則由當地的世襲[[札薩克|札薩克]]管理，處於半自治狀態。在外藩蒙古，以若干旗合為一[[盟|盟]]，設正、副盟長，掌管會盟事宜，對各旗札薩克進行監管。清代的盟是監察機構，並不能視為一級政區。

外藩蒙古又按其歸附清朝的先後分為[[內札薩克蒙古|內札薩克蒙古]]與[[外札薩克蒙古|外札薩克蒙古]]。內札薩克蒙古又被稱為'''內蒙古'''，於天命至康熙初年陸續歸附清朝。乾隆以後定為二十四部，共四十九旗，設六盟。內札薩克各旗不但政治地位很高，還保留了一定的兵權。康熙中期以後歸附清朝的各部落稱為外札薩克蒙古，包括漠北的喀爾喀四部、[[西套蒙古|西套蒙古]]二旗、青海蒙古各部、[[科布多|科布多]]各札薩克旗、新疆[[舊土爾扈特部|舊土爾扈特部]]及[[中路和碩特部|中路和碩特部]]。外札薩克各旗無兵權，隸屬於當地的將軍、[[都統|都統]]、[[駐紮大臣|駐紮大臣]]（西套蒙古二旗除外）。其中喀爾喀四部後來演變為'''外蒙古'''。

====青海====
{{main|西寧辦事大臣|清代青海}}
清代的青海不包括今[[西宁市|西寧]]、[[海东市|海東]]、[[黄南藏族自治州|黃南]]以及[[青海省|青海省]]邊緣的部分地區。統轄青海地方的官員為[[西寧辦事大臣|西寧辦事大臣]]，常駐西寧（屬[[甘肅省|甘肅省]]）。青海大致以[[黄河|黃河]]為界，分為'''青海蒙古'''和'''玉樹等四十族土司'''。黃河以北主要為蒙古人，有[[和硕特|和碩特]]、[[輝特部|輝特]]、[[绰罗斯|綽羅斯]]（準噶爾）、[[土尔扈特|土爾扈特]]、[[喀尔喀蒙古|喀爾喀]]五大部落。雍正三年（1725年），編青海蒙古為二十七旗，後增至二十九旗，由西寧辦事大臣主持會盟。另有[[夏茸尕布活佛|察漢諾門罕]]牧地，實際上單獨為一喇嘛旗。[[道光|道光]]三年（1823年），分黃河以北二十四旗為左、右翼二[[盟|盟]]，每盟設正、副盟長各一人。

黃河以南主要為藏人，設有四十個土司，其中以[[玉樹等四十族|玉樹]]土司最大，故稱玉樹等四十族土司。土司以下有土千戶、土百戶。嘉慶、道光年間，藏人不斷越過黃河向北遷徙，形成了環[[青海湖|青海湖]]一帶的'''[[環海八族|環海八族]]'''。

====西藏====
{{main|駐藏大臣|清朝治藏歷史}}
西藏在清代又稱“唐古忒”、“圖伯特”，分為'''[[衛藏|衛]]'''、'''喀木'''（[[康區|康]]）、'''[[衛藏|藏]]'''、'''[[阿里地区|阿里]]'''四部，以及'''[[霍爾三十九族|霍爾三十九族]]'''地區。西藏地方的行政長官為[[駐藏大臣|駐藏大臣]]，駐[[拉萨市|喇薩]]，會同達賴喇嘛、班禪額爾德尼辦理藏內政務。其政令由[[噶廈|噶廈]]（西藏官府）執行。西藏的基層政區是[[宗_(行政區)|宗]]，大致相當於內地的縣，但規模很小。一些貴族、寺廟的莊園領地稱為“谿卡”，地位比宗低或者平級。宗的長官為“宗本”，谿的長官為“谿堆”，一般由噶廈委派，也有的由特定寺廟委任。後藏[[扎什伦布寺|札什倫布]]附近的幾個宗，由班禪直接管理。

今[[那曲地區|那曲地區]]、[[昌都地區|昌都地區]]北部的各部落統稱[[霍爾三十九族|霍爾三十九族]]，簡稱三十九族，為蒙古人後裔，由駐藏大臣的屬員夷情章京管轄。駐紮於達木（今[[当雄县|當雄]]）的[[達木蒙古|達木蒙古]]八旗，每旗設一佐領，不設總管，直屬於駐藏大臣。

====新疆====
{{main|准噶尔部|回部|清朝治疆历史}}

清代[[新疆维吾尔自治区|新疆]]分為[[天山山脉|天山]]北路的[[准噶尔部|準部]]和天山南路的[[回部|回部]]，統屬於[[伊犁將軍|伊犁將軍]]。其中的蒙古遊牧地區實行盟旗制。[[维吾尔族|維吾爾]]、[[柯尔克孜族|布魯特]]、[[塔吉克族|塔吉克]]等族地區則實行[[伯克制|伯克制]]。蒙古[[舊土爾扈特部|舊土爾扈特部]]與[[巴圖塞特奇勒圖盟|中路和碩特部]]設立旗、盟：舊土爾扈特部為南北東西四路[[舊土爾扈特部|烏訥恩素珠克圖盟]]，和碩特部為[[巴圖塞特奇勒圖盟|巴圖塞特奇勒圖盟]]。凖部地方設[[烏魯木齊都統|烏魯木齊都統]]，統轄烏魯木齊（[[迪化州|迪化州]]）、[[庫爾喀喇烏蘇|庫爾喀喇烏蘇]]、[[吐鲁番厅|吐魯番]]、[[哈密廳|哈密]]、[[奇台县|古城]]、[[巴里坤哈萨克自治县|巴里坤]]（[[镇西府_(清朝)|鎮西府]]）等城。其中迪化州、鎮西府由新疆與[[甘肅省|甘肅省]]雙重管轄。[[塔爾巴哈臺|塔爾巴哈臺]]由塔爾巴哈臺參贊大臣管轄。伊犁及其以西地方由伊犁參贊大臣、領隊大臣管理。回部設總理回疆事務大臣（喀什噶爾參贊大臣），統轄[[喀什市|喀什噶爾]]、[[葉爾羌|葉爾羌]]、[[和闐|和闐]]、[[阿克苏市|阿克蘇]]、[[乌什县|烏什]]、[[庫車直隸州|庫車]]、[[焉耆府|喀喇沙爾]]等城。光緒十年（1884年），[[新疆省_(清)|新疆建省]]，實行與內地相同的府、廳、州、縣體制。

== 屬國 ==
清朝曾有的[[附庸國|屬國]]主要包含（非同時期）：[[李氏朝鲜|朝鲜]]、[[琉球国|琉球]]、[[阮朝|安南]]（越南）、[[琅勃拉邦王国|南掌]]（老挝）、[[泰国|暹罗]]（泰國）、[[贡榜王朝|缅甸]]、[[尼泊爾王國|廓尔喀]]（尼泊尔）、[[锡金王国|哲孟雄]]（锡金）、[[浩罕汗国|浩罕]]、[[哈萨克汗国|哈萨克]]、[[安集延|安集延]]、[[玛尔噶朗|玛尔噶朗]]、[[那木干|那木干]]、[[塔什干|塔什干]]、[[巴达克山|巴达克山]]、[[博罗尔|博罗尔]]、[[杜蘭尼王朝|阿富汗]]、[[坎巨提|坎巨提]]、[[蘇祿蘇丹國|蘇祿國]]。

屬國在列強勢力開始進入遠東後盡數獨立出清朝或是遭列強佔領；中亞的屬國先後被[[俄罗斯帝国|俄罗斯帝国]]佔領，南亞的屬國及緬甸則成為[[大英帝国|大英帝国]]的[[保护国|保护国]]，安南、南掌則成為法國的保護國，而暹羅後也脫離屬國；東亞部份，[[琉球|琉球]]被日本占领；[[李氏朝鲜|朝鲜]]則在[[甲午戰爭|甲午戰爭]]後宣布獨立為[[大韓帝國|大韓帝國]]。

== 与领土相关的条约 ==
* 1689年（康熙二十八年），中俄签订《[[尼布楚條約|尼布楚條約]]》，划定外东北边境。
* 1727年（雍正五年），中俄签订《[[恰克圖界約|恰克圖界約]]》、《[[恰克图条约|恰克图条约]]》，划定[[烏里雅蘇臺將軍|乌里雅苏台]]（外蒙古地区）边境。
* 1842年（道光二十二年），中英签订《[[南京条约|南京条约]]》，将香港岛割让给英国<small>（1997年主權移交，《南京條約》擱置。）</small>。
* 1858年（咸丰八年），中俄签订《[[璦琿條約|璦琿條約]]》，中方割让按《尼布楚条约》规定原属中国的[[黑龙江|黑龙江]]以北约60万平方公里土地。
* 1860年（咸丰十年），英法联军攻入北京，火烧[[圆明园|圆明园]]，后签订中英、中法《[[北京条约|北京条约]]》，将[[九龍半島|九龍半島]]南部割让给英国<small>（1997年主權移交，《北京條約》擱置）</small>。同年中俄《北京条约》签订，将[[乌苏里江|乌苏里江]]以东，包括[[库页岛|库页岛]]在内的约40万平方千米的土地割让给俄国。
* 1864年（同治三年），中国与俄国签定《[[中俄勘分西北界约记|中俄勘分西北界约记]]》，将[[外西北|外西北]]（包括[[巴尔喀什湖|巴尔喀什湖]]之东南、[[伊犁|伊犁]]以西、以及[[噴赤河|喷赤河]]以东的[[伊犁將軍|新疆]][[帕米爾高原|帕米尔]]等地区）的44万平方公里割让给俄国。
* 1881年（光绪七年），中国与俄国签定《[[伊犁條約|伊犁條約]]》。
* 1885年（光绪十一年），中国与法国签定《[[中法新约|中法新约]]》，划定滇越边境。
* 1886年（光绪十二年），中国与英国签定《[[中英缅甸条款|中英缅甸条款]]》，划定滇缅边境。
* 1887年（光绪十三年），《[[中葡和好通商條約|中葡和好通商條約]]》簽訂，中國同意[[葡萄牙|葡萄牙]]“永居、管理[[澳門|澳門]]”<small>（1999年政權移交）</small>。
* 1895年（光绪二十一年），中国与[[日本|日本]]签定《[[马关条约|马关条约]]》，将[[臺灣|臺灣]]以及[[澎湖群島|澎湖列岛]]割让给日本。[[第二次世界大战|二戰]]后由中華民國政府實際統治至今。
* 1898年（光绪二十四年），中国与[[英国|英国]]签定《[[展拓香港界址專條|展拓香港界址專條]]》，将[[香港|香港]]附近[[新界|新界]]等地租借给英国99年<small>（1997年租約期滿，收回。）</small>。
* 1911年（宣统三年），[[辛亥革命|辛亥革命]]爆发后，大清[[外蒙古|外蒙古]]当局与俄罗斯签订《俄蒙协约》，[[中國|中國]]对外蒙古失去控制权，但「名义上」仍然是[[中國|中國]]领土」。
