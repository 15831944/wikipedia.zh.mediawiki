'''清津市'''（{{kor|諺=청진시|漢=淸津市|羅=Chŏngjin si|f=y}}）是[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]][[咸鏡北道|咸鏡北道]]的首府，亦是朝鮮第三大都市，是朝鮮北部的重點工業城市，人口約為700,000人。

== 地理 ==
位於咸鏡北道南部、臨近[[朝鮮東海|朝鮮東海]]的港灣城都市。東北部是[[羅先|羅先]]市、西部是[[富寧郡|富寧郡]]及[[茂山郡|茂山郡]]、南部是[[鏡城郡|鏡城郡]]、北部是[[會寧|會寧]]市。[[北韓|北韓]]的鋼鐵基地，然而[[空氣污染|空氣污染]]亦是[[北韓|北韓]]之最。

== 歷史 ==
古代屬於[[高句麗|高句麗]]、[[渤海国|渤海]]的領土，其後歷經中國的[[金|金]]國及[[元|元]]朝統治。明朝初期，[[朱元璋|朱元璋]]評軍需運糧至此不方便，撤[[衛所制|衛]]，將之劃予高麗，有[[高麗王朝|高麗]]當年文書為證（孟森「清始祖[[布庫里雍順|布庫里雍順]]之考證」），高麗因此設置鏡城郡；[[朝鮮王朝|朝鮮王朝]]時代屬富居縣所管轄的小漁村。
*1904年發生[[日俄戰爭|日俄戰爭]]時，被[[日本|日本]]軍用作軍資補給站。
*1908年時，[[日本|日本]]使[[朝鮮|朝鮮]]開設通商港口；日本殖民時代發展成鋼鐵製造基地。
*1941年，正式升格成為城市。
*1945年8月13日，被[[前蘇聯|前蘇聯]]軍隊佔領、成為第一批解放的[[北韓|北韓]]城市。
*1960年代曾一度脫離[[咸鏡北道|咸鏡北道]]，升格為中央[[直轄市|直轄市]]。
*1985年時，又歸[[咸鏡北道|咸鏡北道]]管轄，並成為首府。
*2012年9月1日，[[中華人民共和國|中国]]延边海华集团在平壤与朝鲜港湾会社签署合营合同成立海港合营公司，共同管理和利用年吞吐能力达700万吨的清津港3、4号连接线码头，合营期限为30年。

== 清津港口 ==
'''清津港口'''(Chongjin Port)被分為「東港」及「西港」兩部分，兩[[碼頭|碼頭]]之間相距4.7公里（2.9英里），由人造防波隄庇護，每年平均膨脹約0.4公呎，由於受暖流影響屬四季永[[不凍港|不凍港]]，[[潮汐|潮汐]]落差約20至30厘米，配備了現代化裝卸貨物設施，設有7個貨站，能容納多部5,000至10,000公噸級的[[貨車|貨車]]，又有[[鐵路|鐵路]]接通[[中華人民共和國|中華人民共和國]]及[[俄罗斯|俄罗斯]]聯邦，每年吞吐能力達800萬公噸、水深：12公呎、能接待2萬公噸的貨船、與[[北韓|北韓]]西岸[[黄海|黄海]]畔的[[南浦|南浦]]港口都是北朝鮮最大的港口、主要連通[[日本|日本]]的[[京都|京都]]市之[[舞鶴港|舞鶴港]]及[[新潟港|新潟港]]。主要處理五穀類、鋼鐵、豬鋼、廢鐵、磁鐵礦和其它礦物等商品，它設有佔地300,000平方公呎的貨倉及露天貨場。

== 交通 ==
* [[清津電車|'''无轨电车'''及'''有軌電車''']]
** 市内道路[[公共交通|公共交通]]以[[无轨电车|无轨电车]]为主。
** 除[[平壤|平壤]]外，它是唯一擁有[[有軌電車|有軌電車]]的[[北韓|北韓]]城市，使用[[中國|中國]]製造的[[車辆|車辆]]。 
* '''鐵路''' 及 '''公路'''
** 鐵路北上聯接[[中國|中國]]的鐵路系統，亦可經[[羅先|羅先]]的寬軌鐵路與[[俄羅斯|俄羅斯]]聯邦的運輸系统連接。
** 鐵路南下經[[金策|金策]]市、[[咸興|咸興]]市，可通[[平壤|平壤]]，至[[平壤|平壤]]的電氣化鐵路長約788公里。

* 航空站可使用位于[[咸鏡南道|咸鏡南道]]的漁郎郡之[[清津機場|清津機場]]，[[北韓|北韓]]政府計劃擴建該[[機場|機場]]成為[[北韓|北韓]]的第二個國際[[機場|機場]]。另外亦可以使用[[中國|中國]]的[[吉林省|吉林省]]的[[延吉朝陽川機場|延吉朝陽川機場]](IATA Code : YNJ)，該[[機場|機場]]距離清津市約166公里。

== 觀光熱點 ==
* [[鏡城|鏡城]]：位於距離清津的西南部約35公里的海濱，它是著名的陶瓷商品生產地、休閒觀光中心。
* [[鹽盤革命史蹟地|鹽盤革命史蹟地]]：位於距離鏡城海濱約4公里的地方 。 
* [[鏡城温泉|鏡城温泉]]：位於距離鏡城約2公里的[[温堡里|温堡里]]，有治療作用用的温泉。 
* [[温堡温泉|温堡温泉]]：位於距離鏡城的西北部約12公里的地方，它是膾炙人口的著名溫泉。 
* [[鏡城南大門|鏡城南大門]]：始建於[[高麗|高麗]]時代，並於[[朝鮮王朝|朝鮮王朝]]時代改建古蹟。

== 觀光飯店 ==
* 清津觀光飯店：位於「青岩洞」的「駱駝山」的山麓，設有18幢客房。
* 天馬山飯店：位於「新岩洞」的「天馬山」的山麓，四層建築物。
== 朝鲜之声发射台 ==
[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]对外广播电台[[朝鲜之声|朝鲜之声]]在本市设有中波发射台，发射机[[功率|功率]]500[[千瓦|千瓦]]，使用[[中波|中波]]621千赫播出，主要向[[日本|日本]]转播[[朝鲜之声|朝鲜之声]]的[[日语|日语]]节目。
