'''清河八郎'''（{{bd|1830年|11月24日|1863年|5月30日}}，即[[天保_(日本)|天保]]元年－[[文久|文久]]3年），[[幕末|幕末]][[庄內藩|庄内藩]]出身的志士，他組成的浪士隊是文久3年[[近藤勇|近藤勇]]和[[芹澤鴨|芹澤鴨]]開創「[[新選組|新選組]]」的開山鼻祖。

== 出生 ==
出生於[[出羽國|出羽國]]庄内藩領[[清川村|清川村]]（現今[[山形縣|山形縣]][[東田川郡|東田川郡]][[庄內町|庄内町]]）一個叫齋藤豪壽的鄉下富豪之子。
[[天保_(日本)|天保]]14年（1843年），向清川關所的[[畑田安右衛門|畑田安右衛門]]學習。[[弘化|弘化]]3年（1846年）和天誅組總裁[[藤本鐵石|藤本鐵石]]見面，兩人有深厚交情。弘化4年（1847年），前往江戸拜[[古學|古學]]派的[[東條一堂|東條一堂]]為師。被東條任命為[[私塾|私塾]]的塾頭，但是他辭職了。轉而向安積艮齋去學習。在他旁邊是[[北辰一刀流|北辰一刀流]]的開祖[[千葉周作|千葉周作]]，是在{{tsl|ja|玄武館||玄武館}}獲得免許證明的劍士，在江戸求得的學問從幕府的學問所{{tsl|ja|昌平黌||昌平黌}}學來的。之後，清河設立私塾講授課程。

== 周遊列國 ==
[[安政|安政]]2年（1855年）3月至9月得這段期間，清河帶著母親從清川村出發。一路經往[[善光寺|善光寺]]、{{tsl|ja|名古屋||名古屋}}、[[伊勢|伊勢]]、{{tsl|ja|奈良||奈良}}、[[京都|京都]]、{{tsl|ja|近江||近江}}、{{tsl|ja|大坂||大坂}}、{{tsl|ja|宮島||宮島}}、[[岩國市|岩國]]、[[天橋立|天橋立]]、{{tsl|ja|鎌倉||鎌倉}}、[[江戸|江戸]]等地，記錄成『西遊草』一書，在旅行中得到的珍貴資料。各國名士爭相閱讀，內容有清河對遊歷過程所評論的犀利話語。
== 虎尾之會 ==

== 急務三策與浪士隊 ==
清河八郎為了名正言順赦免自己的罪過，於是透過熟識的松平上總介（[[松平忠輝|松平忠輝]]後代）向幕府中心人物之一的[[松平春嶽|松平春嶽]]提出「急務三策」。

「急務三策」的意思是：
* 平定攘夷派
* 行使大赦
* 教育天下英才

由於幕府對於攘夷派份子在[[江戶|江戶]]等地攻擊洋商外使等事件，十分煩惱，於是接受了清河八郎的意見，清河八郎又提出「募集英才，創立浪士隊」然後由清河親自率領浪士隊，藉以鎮壓京都的攘夷派份子。

獻策成功後，清河被幕府拔擢領導人，赦免了他的罪，讓清河帶領浪士隊東奔西走，去鎮壓攘夷志士。但清河創立浪士隊是另有所目的，其實清河創立浪士隊，等於是藉由這個名義，招兵買馬，其目的是為了進行[[倒幕運動|倒幕]]運動，並不是真正為幕府辦事，所以為了計畫的進行，才將浪士隊前往京都。

清河徵招浪士隊的消息很快就傳開了整個江戶，紛紛有許多浪士聞訊而來，其中包括[[近藤勇|近藤勇]]等{{tsl|ja|試衛館||試衛館}}的年輕志士，人數超過兩百五十人，由於每個人的津貼是五十兩，所以這樣計算來看總共要花費一萬兩千五百兩左右，就超過了幕府原先預算的五倍，松平上總介因為嫌麻煩就辭去了職務。

突然被找來負責擔任監督人{{tsl|ja|鵜殿鳩翁||鵜殿鳩翁}}只好向浪士們說明預算不足的事情，因此許多武士又紛紛離開清河，清河無奈之下只好招集一些農民和賭徒等烏合之眾充數。

[[文久|文久]]三年（1863年）2月8日，清河以「護衛將軍[[德川家茂|德川家茂]]上京，前往京都維護治安」的名義，前往京都。

沒想到幕府以「[[關白|關白]]命令」為由，要求清河等人立即回江戶，聽到這消息的近藤勇等人相當反對，只留下了24人留在京都，大部分的人都和清河回江戶去了。

== 死亡 ==
幕府開始發現清河八郎的陰謀，於是在文久3年（1863年）4月13日派出刺客[[佐佐木只三郎|佐佐木只三郎]]、[[窪田泉太郎|窪田泉太郎]]等6人在麻布一橋（現今[[麻布十番商店街|麻布十番商店街]]）暗殺清河，享年34歲，在他去世後，幕府贈受他為[[正四位|正四位]]。

據《[[女士道|女士道]]》（[[山岡英子|山岡英子]]於1903年著）的記載，{{tsl|ja|石坂周造||石坂周造}}將其屍體恢復原樣，由山岡英子（{{tsl|ja|山岡鐵舟||山岡鐵舟}}之妻）保管於[[傳通院|傳通院]]安葬，其後遺族再把它安置於[[东京|東京]][[文京區|文京區]]的[[傳通院|傳通院]]裡。清河死後，幕府將浪士隊改名叫「新徴組」，由庄内藩帶領。
