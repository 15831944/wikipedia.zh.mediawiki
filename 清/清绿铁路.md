'''清绿铁路'''南起[[清水站_(甘肃省)|清水站]]，北至[[额济纳旗|额济纳旗]][[额肯查干牧场|额肯查干牧场]]宝日乌拉嘎查（绿园/青山头），是与[[东风航天城|东风航天城]]配套的军用铁路，最初为严格对外保密的军事专用线，近年来因[[中国载人航天工程|中国载人航天工程]]的发展而为人所知<ref>{{cite web|title=为卫星上天而出力_忆清绿铁路的诞生|url=http://www.cnki.com.cn/Article/CJFDTotal-TDZS199803013.htm|author=周转运}}</ref>。线路从[[清水站_(甘肃省)|清水站]]至[[东风站_(额济纳)|绿园]]段区间共331公里，全长356公里。由铁道兵第十师，第3、6师各1个土方机械连担负施工任务。1958年5月开工，1959年4月27日铺轨通车。全线共完成路基土石方247万立方米；桥梁7座，总延长665米；涵梁74座，总延长644米；车站14座，正线铺轨304公里。1960年1月，经检查验收后交付基地铁路管理部门运营。<ref>{{cite web |author1=梅梓祥 |title=铁道兵承建国 防工程：清绿铁路（1） |url=https://www.sohu.com/a/203201676_713283 |accessdate=2019-03-28}}</ref>线路是[[中国人民解放军铁道兵|铁道兵]]承建的国防工程，是为适应0029工程（建设中国第一座导弹试验基地）的需要而修建的军事专用线。<ref>{{cite web |title=铁道兵承建国 防工程：清绿铁路（1） |url=http://www.sohu.com/a/203201676_713283}}</ref>铁路通车于1960年，投资5690万元[[人民币|人民币]]<ref>{{cite news|title=东方红一号发射46周年：实际工作28天，现在仍在太空飞行|url=http://www.thepaper.cn/baidu.jsp?contid=1459444|quote=就拿中心的这条铁路来说，当时就投入了5960万元，而当年全国一年的财政收入才370多亿元|publisher=澎湃新闻|date=2016-04-21}}</ref>。该铁路由[[酒泉卫星发射中心|酒泉卫星发射中心]]铁路管理处管理，所有[[酒泉卫星发射中心|酒泉卫星发射中心]]的大部分火箭、卫星材料、生活用品由此铁路来运输，因此被称为“酒泉卫星发射中心的生命线”<ref>{{cite web|title=守护酒泉卫星发射中心的生命线|url=http://news.xinhuanet.com/tech/2006-10/08/content_5174593_1.htm|publisher=新华网|date=2006-10-08}}</ref>。

==营运状况==
每周一、三、五有军列301/2次在[[清水西站|清水西站]]和[[东风站_(清绿铁路)|东风站]]间往返。
