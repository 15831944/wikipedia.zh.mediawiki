'''繼皇后'''（{{bd|1718年||1766年||CatIdx=U}}），[[辉发那拉氏|辉发那拉氏]]<ref name=皇后>{{harvnb|弘昼等|2002|p=313}}：“莽科，镶蓝旗人，[[王机砮|王机砮]]之孙也。世居辉发地方。国初率兄弟及同里人等来归，编佐领使统之。曾孙……那尔布，原任佐领。元孙……讷理，俱原任佐领……”；<br>{{harvnb|鄂尔泰等|1985|p=177}}：“镶蓝旗满洲第三参领第一佐领系国初以辉发地方来归人丁编立，始以莽库管理……那尔布管理。那尔布因病辞退，以其子讷礼管理。讷礼故，以其子讷苏肯管理。现改为公中佐领……；<br>《清实录》乾隆十五年庚午八月丙戌条：“[[清高宗继皇后|皇后]]之父那尔布、追封为一等公。遣官致祭。造坟立碑如例。妻封为公妻一品夫人。以其孙纳苏肯袭一等侯。”</ref>，[[中國|中國]][[清朝|清朝]]女性[[皇族|皇族]]，滿州鑲藍旗人，為[[乾隆帝|乾隆帝]]的第二任皇后。父親為一等承恩公[[佐領|佐領]]訥爾布。

《[[清史稿|清史稿]]》稱乾隆繼皇后為烏拉那拉氏，所以如今很多人包括史學界的人都以為她的家族姓[[乌拉那拉氏|烏拉那拉]]，甚至還有人附會她是[[孝敬憲皇后|孝敬憲皇后]]的族侄女。其實編寫於民國時期的《[[清史稿|清史稿]]》 多有錯謬，其中也包括乾隆繼皇后的姓氏。《[[欽定八旗滿洲通志|欽定八旗滿洲通志]]》《[[八旗满洲氏族通谱|八旗滿洲氏族通譜]]》裡都記載她父親[[訥爾布|訥爾布]]姓[[辉发那拉氏|輝發那拉]]<small>（詳見[[清高宗继皇后#姓氏考證|姓氏考證]]）。</small>

==早期生平 ==

[[康熙|康熙]]五十七年 (1718年） 二月初十日出生。

按照清初慣例，[[八旗選秀|外八族選秀]]是在臘月中下旬舉行的，雍正初年依然遵循此例，雍正十一年臘月原定要舉行外八族選秀，惟雍正帝下旨改在雍正十二年 (1734年) 二月舉行。自此，在春季舉行外八旗選秀成為了慣例。在新制度下，佐領訥爾布之女輝發那拉氏被指婚為寶親王[[福晉|側福晉]]。

雍正十二年五月，奉旨赏宝亲王侧福晋之父那爾布染貂冬冠一顶等物，賞侧福晋之[[嫡庶|嫡母]]镶有两个珍珠的金耳坠三对等物。及後，由[[欽天監|欽天監]]根據[[八字|八字]]算日子，即所謂「擇吉」，決定在十一月初八日正式娶進。

雍正十三年九月二十四日，奉旨納喇姓側福晉封為妃。同年十月二十九日，將位於[[華嘉胡同|花甲寺胡同]]的华嘉寺胡同舒库入官之宅邸一所共四十二间，賞给她的父親讷尔布，檔案可知她家原本住在河槽沿东。此外，高宗又將[[油坊衚衕|油坊胡同]]的房子賞予那妃的家族。

乾隆二年十二月初四日，正式册封為'''嫻妃'''；乾隆十年（1745年）正月二十三日，詔封為'''嫻貴妃'''，同年十一月十七日行嫻貴妃冊封禮。

乾隆十三年（1748年）三月十一日，皇后富察氏病逝，中宮皇后的位子出現空缺；四月二十一日，乾隆帝到[[观德殿|观德殿]]大行皇后[[梓宫|梓宫]]前奠酒，未刻到[[雍和宫|雍和宫]]行礼，接著待在[[太和斋|太和斋]]。皇贵妃等位則先行到东直门外的静安庄殡宫。翌日，乾隆帝才到該地临奠停灵在那里的[[哲悯皇贵妃|哲悯皇贵妃]]等人。 不久之後的七月十一日午時，皇貴妃內庭等位再一次前往雍和宮，[[翊坤宮|翊坤宮]]、[[永壽宮|永壽宮]]、[[儲秀宮|儲秀宮]]等處各派[[太監|首領太監]]或普通太監若干人跟隨。由此可見，那拉氏已被宮中諸人默認為皇貴妃，只是尚未公諸於眾。

乾隆十三年七月初一日，乾隆帝發出一道上諭<ref>“''朕躬攬萬幾。勤勞宵旰。宮闈內政。全資孝賢皇后綜理。皇后上侍聖母皇太后。承歡朝夕。純孝性成。而治事精詳。輕重得體。自妃嬪以至宮人。無不奉法感恩。心悅誠服。十餘年來。朕之得以專心國事。有餘暇以從容冊府者。皇后之助也。茲奉皇太后懿旨。皇后母儀天下。猶天地之相成。日月之繼照。皇帝春秋鼎盛。內治需人。嫻貴妃那拉氏、系皇考向日所賜側室妃。人亦端莊惠下。應效法聖祖成規。即以嫻貴妃那拉氏繼體坤寧。予心乃慰。即皇帝心有不忍。亦應於皇帝四十歲大慶之先。時已過二十七月之期矣。舉行吉禮。佳兒佳婦。行禮慈寧。始愜予懷也。欽此。朕以二十餘年伉儷之情。恩深誼摯。遽行冊立。於心實所不忍。即過二十七月。於心猶以為速。但思皇后大事。上軫聖母懷思。久而彌篤。歲時令節。以及定省溫凊。朕雖率諸妃嬪 及諸孫、問安左右。而中宮虛位。必有顧之而愴然者。固宜亟承慈命。以慰聖心。且嬪嬙內侍。掖庭之奉職待理者甚眾。不可散而無統。至王妃命婦等、皆有應行典禮。允曠不舉。亦於禮制未協。冊立既不忍舉行。可姑從權制。考之明太祖淑妃李氏寧妃郭氏、相繼攝六宮事。國朝順治十三年、冊立皇貴妃。皇曾祖世祖章皇帝升殿命使翼日頒詔天下。典至崇重。今應仿效前規。冊命嫻貴妃那拉氏為皇貴妃。攝六宮事。於以整肅壼儀。上奉聖母。襄助朕躬。端模範而迓休祥。順成內治。有厚望焉。所有應行典禮。大學士會同禮部、內務府、詳議具奏。尋議、恭查皇貴妃冊封大典。王妃命婦行禮。已有成例。惟貴妃行禮之處。外廷無案可稽。但皇貴妃攝行六宮事。二十七月後即正位中宮。既統理內政。體制自宜尊崇。貴妃亦應一體行禮。所有冊封禮儀 前期一日。遣官祭告太廟。奉先殿告祭禮。上親詣舉行。屆期設鹵簿儀仗、中和韶樂。上御太和殿閱冊寶。大學士等、捧節授持節使。持節使隨冊寶亭、至景運門授內監。皇貴妃具禮服恭迎。宣受如儀。次日上率王以下文武官員、詣皇太后宮行禮。禮畢。皇貴妃率貴妃以下、公主、王妃、命婦、行禮。上御太和殿受賀。頒詔天下。嗣後遇三大節。及慶賀大典。三品以上大臣官員、進箋慶賀。及每歲行親蠶禮。應照例舉行。得旨、依議冊 封典禮。著於明年三月後舉行。其親蠶禮。俟正位中宮後。該部照例奏請。''”</ref>，稱後宮不可無皇后主持，而嫻貴妃端莊惠下，原欲冊立為皇后，但又念及與故孝賢皇后之間的夫妻之情，因此參考[[明朝|明朝]][[朱元璋|太祖]]年間[[孝慈高皇后_(明朝)|孝慈高皇后馬氏]]逝世後由[[李淑妃_(明太祖)|李淑妃]]、[[郭寧妃_(明太祖)|郭寧妃]]相繼攝理後宮事務，以及本朝[[順治帝|順治帝]]冊封[[孝獻端敬皇后|董鄂妃]]為[[皇貴妃|皇貴妃]]等先例，冊封嫻貴妃為攝六宮事的'''皇貴妃'''；同時，乾隆帝也在同一份上諭中許諾，待孝賢皇后喪期（27個月）過後，將正式冊立嫻貴妃為皇后。同年七月三十日，皇贵妃娘家由[[镶蓝旗|镶蓝旗]][[抬旗|抬入]][[上三旗|上三旗]]中的[[正黄旗|正黄旗]]。

==榮登-{后}-位==

乾隆十四年（1749年），乾隆帝破例下令将所有嫻皇贵妃仪仗內的车舆，由金黄色改用明黄色。清制，明黃色為帝-{后}-方可使用的色彩，皇貴妃那拉氏因而成为後宮中唯一能使用明黄仪仗的內庭主位，等同於皇后的規格；同年三月初六日，首領文旦交御筆黃絹「'''懋端壼教'''」匾文一張，安設在[[翊坤宮|翊坤宮]]後殿明間北牆上，亦賜予御筆紅絹「'''德茂椒涂綿福履，教敷蘭掖集嘉祥'''」對聯一副；三月二十九日 ，高宗讓人在翊坤宫前殿佛前，掛墨刻本文一张，款式為锦边壁子匾随匾丁勾；又在翊坤宫后殿正宝座，掛'''御笔画金龙韵对'''一幅、'''御笔红绢金龙福字'''一張。

乾隆十四年（1749年）四月五日，正式冊封那拉氏為'''皇貴妃'''；同年四月九日，嫔妃、[[命妇|命妇]]于[[交泰殿|交泰殿]]內向嫻皇贵妃那拉氏行[[三跪九叩之禮|六肃三跪三叩礼]]。「六肃三跪三叩禮」是只有帝-{后}-才能享有的禮儀，可見那拉氏作為的准皇后，在後宮中已經享有皇后規格的待遇，甚至在乾隆十五年（1750年）正月，部分内廷档案中已經稱嫻皇貴妃为皇后；同年二月初十日，嫻皇贵妃的[[生日|千秋]]慶典亦採用皇后規格辦理；同年八月初二日，正式舉行冊立'''皇后'''之禮，旋即與乾隆帝巡幸[[河南|河南]]嵩洛，並恭谒祖陵。

乾隆十七年（1752年）四月二十五日寅時，皇后那拉氏生皇十二子[[永璂|永璂]]。乾隆十八年（1753年）六月二十三日寅時，生[[乾隆帝皇五女|皇五女]]。乾隆十九年闰四月二十四日，[[員外郎|員外郎]]西寧來說，太監胡世傑交东珠荷包豆一副，高宗命人將此物送予身處[[圓明園|圓明園]]的皇后；五月，乾隆帝带着皇后那拉氏去了一趟祖籍辉发古城（今吉林省通化市辉南县），并在这里留下了几篇诗作。

乾隆二十年（1755年）四月二十二日酉時，[[乾隆帝皇五女|皇五女]]夭折；六月十六日，太監胡世傑給予皇后那拉氏年例[[端午節|西瓜燈]]一對，事後不需像其他妃嬪般用寶蓋交回西瓜燈。乾隆二十年（1755年）十二月二十一日卯時，生皇十三子[[永璟|永璟]]；十二月二十八日，催總管李文照將[[畫|絹畫]]兩张持進翊坤宮內張貼。乾隆二十一年 (1756年) 四月二十七日，高宗特命總管方剛為剛出生的翊坤宫阿哥 (即[[永璟|永璟]]) 的床上鋪涼蓆一塊。

乾隆二十二年（1757年）三月，那拉氏的生母[[郎佳氏|郎佳氏]]去世；七月二十四日子時，皇十三子[[永璟|永璟]]夭折。隨後，那拉氏醉心于佛法之中，曾在崇慶皇太后七旬大庆時，往[[寿康宫|寿康宫]]送进贝叶长生汉字金字无量寿[[佛经|佛经]]一套。期間，那拉氏位下曾有兩位[[清朝後宮|學規距女子]]，拜爾葛斯氏被封為[[慎嬪_(乾隆帝)|伊貴人]]，和卓氏則被封為[[容妃_(乾隆帝)|和貴人]]。那拉氏與伊貴人關係較為密切，兩人常在《穿戴档》中被乾隆帝要求做些[[針線活|女紅]]，如荷包、火燫之類的小物品。乾隆帝更在冊文中稱贊曾受皇后教導的拜爾噶斯氏為「六宮雅範」。

乾隆二十九年二月，為乾隆帝萬壽日念經一日；為皇后那拉氏千秋誕月，分派喇嘛一百八名，在[[雍和宫|雍和宫]]大殿念无量寿佛经九日；正寿一日，派喇嘛五百名在雍和宮唪念，供餑餑桌三張，共用銀四十七兩六錢六分九毫。

== 突然失寵==

乾隆三十年正月，皇后那拉氏陪乾隆帝進行了[[乾隆帝南巡|第四次南巡]]。這次南巡成了繼皇后命運的轉捩點，南巡前期一切都很正常，途中皇帝還為她慶祝四十八歲千秋；閏二月十八日，他們來到[[杭州|杭州]]，在風景秀麗的「蕉石鳴琴」（今屬[[西湖國賓館|西湖國賓館]]範圍內）進早膳時，皇帝還賞賜給皇后攒盘肉一品，但到了當天中午進晚膳時，皇后那拉氏沒有露面，陪著乾隆帝進晚膳的只有當時還是[[孝儀純皇后|令貴妃]]的魏佳氏、[[慶恭皇貴妃|慶妃]]陆氏、以及[[容妃_(乾隆帝)|容嬪]]和卓氏。膳底档上“皇后”二字已被用纸糊上，换以“令贵妃”三字。由此可見，最晚在晚膳准备好了的时候，还没有人发现那拉氏斷髪。

綜合現存的史料，在當日皇后的行为突然一反常态，在皇太后前不能恪尽孝道。在晚膳前，皇后先行遣去宫女三人，断发时是孤身一人的，无论是乾隆帝，还是近侍的宫女，均未察觉到有任何先兆。乾隆帝下令养心殿当差总管潘凤将皇后所拥有的物什查明封贮，並諭令王成查看那拉氏在[[圆明园|圆明园]]的住处、净房，查看有沒有「邪道踪迹」，雖怀疑皇后她「'''平日恨我必深'''」，但還是認為她是被人下了降头才有此舉動。

當天，乾隆帝派[[驸马|額駙]][[福隆安|福隆安]]由水路遣送皇后进宫，並要那拉氏在[[翊坤宫|翊坤宫]]后殿养病，不许见一人。乾隆帝令福隆安日行两站加紧回京，沿路马匹纤夫都必须准备充足；闰二月二十一日，乾隆帝的态度有所软化，令福隆安“酌情行进，无需过急”，回程途中还让地方官员筹备接驾，並且查看沿途水路河道情况等事宜；三月二十三日，乾隆帝甚至还下了一道嘉奖皇后侄子讷苏肯的谕旨，惟同一日乾隆帝又谕令[[福隆安|福隆安]]务必于入京前，等候英廉传旨。乾隆帝谕令阿哥、公主、福晉前来听取他对此事的后续处理，又令当着他們及皇后的面前将涉及此事的三位[[宫女|宫女]]责打六十大板，又将翊坤宫中[[官女子|官女子]]与[[太监|太监]]悉数遣散，另挑老实的女子两位及太监十位侍候；四月二十一日乾隆帝回京，进駐[[圆明园|圆明园]]。五月初二日，侍郎阿永阿因力谏不可废后而被贬往新疆；五月十四日，收缴四份册宝夹纸，並且裁減她位下的部份佣人，但尚未褫奪皇后[[位號|位號]]。七月時，那拉氏位下只剩兩名[[宮女|宮女]]；按清宮制度，只有位分最低微的[[答應|答應]]才只有兩名宮女。

乾隆帝對此事的態度突然轉變，有可能是總管太監王成涿州接駕時所呈上的一道折子。高宗將那拉氏斷髮之故歸究於被人施以邪術，但他不是很確定，便諭令王成看著阿哥们念懲罰那拉氏的旨意時，那拉氏是「怎么听、做何光景」，要王成一一记下並奏上。

乾隆三十一年（1766年）正月初八日的[[内务府|内务府]]满文奏折顯示，已懷孕的皇贵妃魏氏接替被幽禁[[翊坤宮|翊坤宮]]後殿的皇后那拉氏，办理[[元宵節|上元令节]][[清朝后宮|内庭主位]]的饽饽桌张以及诣居[[圆明园|圆明园]]事宜；六月初七日，皇后娘家拨回原旗，並且被削去两个世管佐领。不久，讷苏肯著为三等侍卫，交乌什大臣差遣；七月十四日，皇后那拉氏默默離開人世，終年四十九歲。乾隆帝命那拉氏的喪葬儀式下降一級，即等同依照皇貴妃之例舉行；九月二十八日葬入[[清裕陵|裕陵]]妃園寢。

== 千古疑團 ==

據說那拉氏忽然自行剪髮，在[[滿族|滿族]]的習俗裡頭這是最忌諱的。只在皇太后、皇帝駕崩時，皇后才可以剪髮。當時的皇太后、皇帝都還健在，皇后卻突然剪髮，無疑是在詛咒他們一樣。在當時引出一些傳聞，說皇后隨駕南巡，觸犯龍顏，只得削髮為尼，出家到[[杭州|杭州]]寺庵之中。

在《[[東華續錄|東華續錄]]》乾隆朝卷二十二中的一道諭旨中說「皇后自冊立以來尚無失德，去年春，朕恭奉皇太后巡幸江浙，正承歡洽慶之時，皇后性忽改常，於皇太后前不能恪守孝道。比至杭州，則舉動尤乖正理，跡類瘋迷。因令先其回京，在宮調攝。經今一載有餘，病勢日劇，遂爾庵逝。此實皇后福分淺薄，不能仰承聖母慈眷，長受朕恩禮所致。若論其行事乖違，即予以廢黜，亦理所當然，朕仍存其位號，已為格外優容。但飾終典禮不便復循孝賢皇后大事辦理。所有喪儀止可照皇貴妃例行。」

乾隆帝在[[围场厅|木蘭圍場]]打獵時，聽到皇后病故的消息，只命其子[[永璂|永璂]]回宮。當時，正擔任[[御史|御史]]的[[李玉鳴|李玉鳴]]上疏請示依皇后禮舉喪，結果竟被謫[[伊犁哈萨克自治州|伊犁]]。

十二年後，又有一個名叫金從善的書生，上書乾隆皇帝，談到立-{后}-之事。乾隆還為此發怒道，那拉氏本是朕即位前的側福晉。朕即位後，因[[孝賢純皇后|孝賢皇后]]病逝，她才循序由皇貴妃又立為皇后。後來她自犯過失，朕對她一直優容。國俗最忌剪髮，她卻悍然不顧，朕仍然忍隱，不行廢斥。她病死後，也只是減其儀等，並未削去皇后名號。朕處理此事已經仁至義盡，況且從此未再立皇后。金從善竟想讓朕下詔罪已，朕有何罪應當自責？他又提出讓朕立皇后。朕如今已經六十八歲，豈有再冊立中宮皇后的道理！因此將金從善處斬，從此之後無人再敢提及繼皇后的事<ref>《古稀天子乾隆》劉潞</ref>。

== 喪葬禮仪 ==

實際上，皇后那拉氏的喪葬禮儀也沒有按照[[皇貴妃|皇貴妃]]的標準執行。按照皇貴妃的喪儀規定，每日應有[[大臣|大臣]]、[[公主|公主]]、[[命妇|命婦]]齊集舉哀、行禮一項，惟在皇后那拉氏的喪事中均被取消。

根據[[大清會典|大清會典]]的記載，皇貴妃棺木應用[[梓木|梓木]]，漆飾35道，抬棺夫96人。從內務府記載的檔案中看，皇后那拉氏所用的棺為[[杉木|杉木]]製，抬棺夫64人，僅為嬪等級而已。

皇后那拉氏沒有附葬裕陵或單建陵寢，反而葬在妃園寢內。按照清宮慣例，凡葬在妃園寢內的，無論地位有多低，都各自為券，而皇后那拉氏卻被葬入[[純惠皇貴妃|純惠皇貴妃]]地宮的一側，堂堂的大清皇后反倒成皇貴妃的下屬。

另外，清制：凡妃、[[貴妃|貴妃]]、皇貴妃死後都設神牌，供放在園寢享殿內，祭禮時在殿內舉行，而[[嬪|嬪]]、[[貴人|貴人]]、[[常在|常在]]、[[答應|答應]]則不設神牌。祭祀時，把供品桌抬到寶頂前的月台上，皇后那拉氏既不設神牌，死後也無祭享，入葬以後也隻字不提。根據內務府檔案記載，整個喪事僅用銀二百两七钱九分四厘，還不如一個低級朝廷官員。甚至，皇后那拉氏沒有被賜諡號，僅得'''繼-{后}-'''二字。

公元1929年12月，純惠皇貴妃園寢發現被盜，遜帝溥儀派載澤、載瀛等人赴清東陵處理純惠皇貴妃地宮被盜事宜。學者徐廣源所著的《清皇陵地宫亲探记》記載，在公元1981年11月，諸位學者在開啟和探視純惠皇貴妃地宮時，發現純惠皇貴妃內棺里有二個頭顱骨和一堆遺骨，已無法分辨是屬於何人。

== 姓氏考證 ==

* 《清實錄》記載“皇后之父[[訥爾布|訥爾布]]，追封為一等公，遣官致祭、造墳立碑如例。妻封為公妻一品夫人。以其孫納蘇肯襲一等侯”。

* 《欽定八旗通志 ·卷十六》記載：“[[镶蓝旗|鑲藍旗]]滿洲第三參領第一佐領系國初以輝發地方來歸人丁編立，始以莽庫管理。莽庫故，以纛章京德爾德赫管理。德爾德赫故，以其子護軍參領博伯尼管理，後復以莽庫之子羅和管理。羅和故，以其子羅多管理。羅多升任右衛護軍參領，以其弟訥爾布管理。訥爾布因病辭退，以其子訥禮管理。訥禮故，以其子訥蘇肯管理。現改為公中佐領，以扎蘭泰管理，續以達衝阿管理。達衝阿故，以達忠阿管理。

* 《八旗滿洲氏族通譜》記載：“莽科，鑲藍旗人，王機砮之孫也。世居輝發地方。國初率兄弟及同里人等來歸，編佐領使統之。其孫羅和，原任副都統。曾孫羅多，原任護軍參領。訥爾布，原任佐領。元孫薩賚、四格、訥理，俱原任佐領。武德，現任佐領。四世孫札拉芬，原任佐領。六十八，現任佐領。

因《[[清史稿|清史稿]]》稱乾隆繼皇后為烏拉那拉氏，但根據其他文獻考證，繼皇后的姓氏實為輝發那拉氏，而非烏拉那拉氏。

乾隆繼皇后從妃至后的四次冊文均稱其為“那拉氏”，清代文獻《[[清实录|清實錄]]》、《[[乾隆帝起居注|乾隆帝起居注]]》和《[[皇朝文典|皇朝文典]]》都記載了冊文內容，均從未有“烏拉那拉氏”一說。訥爾布之烈祖父（高祖父之父親）為輝發貝勒王機褚，所以訥爾布實為'''輝發那拉氏'''。

對照以上記載可知，乾隆繼皇后那拉氏家族的脈絡為：

[[莽庫|莽庫]]（或作[[莽科|莽科]]，繼-{后}-祖父的父親）→羅和（繼-{后}-的爺爺）→羅多（繼-{后}-的伯父）、訥爾布（繼-{后}-之父）—訥裡（繼-{后}-的兄弟）—納蘇肯（繼-{后}-的侄子）。其祖上世居[[辉发|輝發]]地方，歸降清室後編入[[镶蓝旗|鑲藍旗]]，世襲四品佐領一職。輝發部為滿族[[扈伦四部|扈倫四部]]之一，因居[[辉发河|輝發河]]得名。始祖星古禮，姓益克得哩，尼瑪察部人，原居薩哈連烏拉（指今第一松花江下游）。而[[孝敬憲皇后|孝敬憲皇后]]所屬的烏拉那拉氏一族世居烏拉地方，[[慈禧太后|慈禧太后]]、[[納蘭明珠|納蘭明珠]]所屬的葉赫那拉氏一族世居葉赫地方，和乾隆繼皇后不是一家子。
