'''清华大学井冈山兵团四一四总部'''分裂自[[清华大学井冈山兵团|清华大学井冈山兵团]]，简称“'''四派'''”或“'''四一四派'''”。领导人[[沈如槐|沈如槐]]、[[孙怒涛|孙怒涛]]等。其观点主要体现在《四一四思潮必胜》。属于[[造反派|造反派]]中的温和派。{{fact|也有人认为属于新保守派。}}

== 历史 ==
{{main|清华大学文革历史}}
1967年4月14日成立“清华大学井冈山兵团四一四(革命)串联队”，5月29日正式成立“清华大学井冈山兵团四一四总部”。与团派之间的冲突不断升级，发展为[[清华大学百日大武斗|清华大学百日大武斗]]。

== 思想 ==
以414派为代表的[[保守派|保守派]]是文革中所谓的“修正主义”力量，虽然他们也打着革命旗号，但并不完全接受“文革”理论，他们所掀起的“[[四一四思潮|四一四思潮]]”从种种方面对“文革”产生疑虑、困惑、失望，从事实上动摇了“文革”正统权威。其观点主要体现在1967年8月出版的《四一四思潮必胜》一文。<ref name="从清华大学两派看">唐少杰，《从清华大学两派看“文化大革命”中群众组织的对立和分歧》，中共党史研究，1998年02期</ref>

据宋永毅、孙大进总结，四一四思潮有三大理论支柱：<ref>{{Cite web|url=http://archives.cnd.org/HXWK/author/SUN-Dajin/zk9908b-2.gb.html|title=清华四·一四思潮：河归旧道十七年|accessdate=|author=宋永毅、孙大进|date=|publisher=}}</ref>
# “阶级关系不变论”。
# “造反派不能坐江山论”。
# “文化大革命必须修整、巩固、妥协论”。

1967年8月发表的《四一四思潮必胜》指出：“清华大学四一四派与团派的原则分歧与组织上的分裂，是具有全国性的普遍意义的。”而四一四派将取得胜利。<ref>[https://ccradb.appspot.com/post/4119 四·一四思潮必胜──给河南造总一战友的一封信]</ref>

== 影响 ==
《井冈山》报1967年9月7日发表《毛泽东思想必胜，机会主义必败——评大毒草〈414思潮必胜〉》。[[清华大学井冈山兵团|清华大学井冈山兵团]]的天安门纵队在清华《井岗山》报上发表《谁敢否定无产阶级文化大革命我们就和他拼命——评 〈四一四思潮必胜〉》。

1968年7月28日凌晨，[[毛泽东|毛泽东]]接见聂元梓、蒯大富、韩爱晶、王大宾、谭厚兰“五大领袖”时说：
{{quote|清华“四·一四”说《四·一四思潮必胜》，我就不高兴，说打江山的人不能坐江山，无产阶级打天下不能坐天下。}}
