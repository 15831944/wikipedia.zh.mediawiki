'''清龙站'''是[[深圳有轨电车|深圳有轨电车]]的一个中间站，位于[[龍華區_(深圳市)|龍華區]][[龙华街道|龙华街道]][[梅龙大道|梅龙大道]]与清龙路的交叉路口，於2017年10月28日随[[深圳有轨电车|深圳有轨电车]]第一期线路开通而啟用。<ref>{{Cite web|url=http://gd.qq.com/a/20171028/003693.htm|title=千呼万唤！深圳龙华有轨电车今天起开通迎客_大粤网_腾讯网|accessdate=2018-07-12|work=gd.qq.com|language=zh-CN}}</ref><ref>{{Cite web|url=http://www.sohu.com/a/152852141_754397|title=龙华的家长注意了！龙华有轨电车月底开通，和清湖地铁站无缝对接（附搭乘攻略+独家照片）|accessdate=2018-07-12|date=2017-06-28|work=www.sohu.com}}</ref>

== 车站结构 ==
本车站有一个岛式月台，月台南端设有超级电容变压设备，月台与马路人行道和附近的大唐时代大厦有天桥（该天桥又叫：'''清龙站天桥'''）连接，天桥与月台及马路人行道间有两台升降机可供使用。
== 车站周边 ==
*大唐时代大厦
*祥和御园
*[[龙华区_(深圳市)|龙华区]][[民政局|民政局]]
*港之龙科技大厦

== 歷史 ==
*2017年10月28日，隨深圳有軌電車開通而啟用。深圳有軌電車的1號綫（清湖至新瀾）、2號綫（清湖至下圍）经过此站<ref name=szsb201710>[http://gd.qq.com/a/20171028/003693.htm 千呼万唤！深圳龙华有轨电车今天起开通迎客]</ref>。
*2017年12月28日，本站发车间隔加密至平峰6分钟一班。<ref>{{Cite web|url=http://www.sztb.gov.cn/jtzx/wycx/dtcx/cxtx/201712/t20171227_10628526.htm|title=龙华有轨电车28日起缩短行车间隔 - 深圳市交通运输委员会|accessdate=2018-07-12|last=深圳市交通运输委员会|work=www.sztb.gov.cn|language=zh-CN}}</ref>
== 未来发展 ==
本站除了现正使用中的西向弯道转向清龙路与[[清湖站_(有轨电车)|清湖站]]相连外，也预留了直行方向的岔道，供日后有轨电车线继续沿[[梅龙大道|梅龙大道]]一直延伸到[[白石龙站|白石龙站]]附近，惟相关规划为远期规划，故暂未有启动时间表。
