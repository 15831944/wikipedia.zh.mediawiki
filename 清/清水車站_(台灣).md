'''清水車站'''(台語:清水車頭 Chheng-chúi chhia-thâu)位於[[台灣|台灣]][[臺中市|臺中市]][[清水區_(台灣)|清水區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[海岸線_(臺鐵)|海岸線]]的[[鐵路車站|鐵路車站]]。該建築為[[臺灣歷史建築百景列表|歷史建築百景徵選活動]]中的第14名<ref name="文"/>，清水本地雖然是海線的重要城鎮，{{Fact|但車站的位置因為離[[清水區_(台灣)|清水]]市區或新的區公所稍遠，市區與火車站之間被特三號公路切割，且站前有一大片為工廠，後站也尚未開發，目前也無天橋或地下道貫串前站，商業活動也無法聚集，所以除了清水高中學生上學及部分通勤族之外，使得短程搭乘之乘客較少。|time=2019-02-08}}

== 車站構造 ==
*[[島式月台|島式月台]]兩座，各有A、B二側月台，可停靠13節客車廂。。
*自動售票機2部，可販售西部幹線[[新竹車站|新竹]]=清水=[[員林車站|員林]]及山線[[成功車站|成功]]=[[豐原車站|豐原]]主要車站之車票。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]，但與一般三等站多以停靠較低等級列車為主的狀況不同，本站一天僅有一班次[[自強號列車|自強號]]不停靠外，其餘各級列車均有停靠。
*[[自強號列車|自強號]]不停靠北上6126次(週日行駛)、及南下109次(週日停駛)。
*其他班次皆停靠。
*對號列車停靠率：95.9%；所有車種停靠率：98.7%。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[icash|icash 2.0]]及[[HappyCash|HappyCash]]付費。

== 歷史 ==
由於隨著臺灣經濟日益繁榮，原有的[[臺中線|山線]]已不足應付運輸量，遂於1919年開始計畫興建海線，其中王田（今[[成功車站|成功]]）到清水間路段為最早完成的部分，<ref name="文"/> 而「清水驛」則於1920年12月15日開業。

原先的車站是木構建築，屋頂為黑瓦，牆面則為真壁——由竹子、黃泥、稻草與石灰混合而成——，外面另以木條交叉為裝飾。然而因為1935年的中部大地震，初代車站被震毀，遂於該年12月重建，1936年5月改建完成。<ref name="文">{{cite book|title=《台灣歷史建築百景專輯》 |date=2004-06-30 |edition=二版 |publisher=[[行政院文化建設委員會|行政院文化建設委員會]] |location=臺北市 |isbn=957-01-2749-X |pages=71頁}}</ref>
*2014年7月16日：為加強疏運需要，開始停靠[[太魯閣列車|太魯閣號]]。
*2015年6月30日：啟用多卡通刷卡機。
*2015年10月15日：原[[太魯閣列車|太魯閣號]]行駛之278、285次改由[[普悠瑪列車|普悠瑪號]]行駛。
*2016年10月20日：原[[普悠瑪列車|普悠瑪號]]行駛之278、285次改由[[太魯閣列車|太魯閣號]]行駛。
*2017年1月5日起成追線2603次區間車改由本站發車。

== 建築特色 ==
清水車站今貌主要是以鋼筋混擬土建成，外牆貼著淺褐色面磚。其門廊下之列柱採用縮柱式設計，表面並施以洗石子工法。<ref name="文"/>
