'''鯔魚'''，又稱'''烏魚'''，俗名青頭仔（幼魚）、奇目仔（成魚）、信魚、正烏、九棍、烏頭，為[[鯔屬|鯔屬]]的一種魚，廣泛分佈于熱帶和亞熱帶的鹹淡水域中。

== 特徵 ==

[[File:Mugil_cephalus_head.jpg|thumb]]
本魚體圓而長，頭部略扁而尾部稍側扁。成魚之脂性[[眼瞼|眼瞼]]發育完善，有時將[[瞳孔|瞳孔]]完全掩蓋。主上頜骨不向下彎至前頜骨下方。背部中央無隆起稜脊。體側有6至7條暗色縱帶，胸鰭基部上半1/2處有一黑色斑塊。具有巨大的背鰭上有硬棘4枚和軟條8枚，臀鰭鰭條8枚。整個身體有櫛鱗，並常有黏液分泌，以增加保護性，無側線。體長可長120公分。

== 生態習性 ==
鯔魚廣泛分布於全球的溫、[[熱帶|熱帶]]海域的沿岸。喜棲息在入海口，水深0至120[[公尺|公尺]]。幼魚常溯河入淡水河川。產卵期時會洄游至外海產卵。利用其鰓耙濾取有機物為食。對環境的適應性很強，遭汙染的港灣、河川皆能生存。

== 經濟利用 ==

[[File:Mullet_roe_by_comicpie_in_Taiwan.jpg|thumb]]]]
屬於高經濟價值的食用魚，魚苗多在每年3、4月間出現，很多漁民均在這段期間打開水閘，將幼苗放進基圍，並投餵[[飼料|飼料]]如[[麻枯|麻枯]]、[[黃豆|黃豆]]渣、[[花生枯|花生枯]]等，以期他們快速成長，趕及冬天前出魚收成。已孕雌魚的[[卵巢|卵巢]]及[[魚卵|魚卵]]統稱「[[烏魚子|烏魚子]]」，是鹽漬水產加工食品<ref name="農訓300-p52">{{cite book|title= 來自海洋的「烏金」——烏魚子 | author= 羅德禎 | coauthors= 李俊賢（攝影） | work = 農訓雜誌Vol.300 |edition= 第32卷第2期 | pages=  |date=2015-02-01}}{{rp|52}}</ref>。雄魚的[[睪丸|精巢]]是一道名菜，即[[烏魚鰾|烏魚鰾]]，肉質美味，加[[酒|酒]][[紅燒|紅燒]]，或是煮湯，甚至煮[[麻油|麻油]]烏魚皆適宜。

== 史籍記載 ==
觀之烏魚，兩眼從額頭突出，看起來非常威嚴，全身都是深綠色斑點，喜歡身居海水清濁之間，在海中蛇行迴游環繞，好動刁鑽，圍入網中遇驚嚇會奮不顧身往上跳躍，至死不屈，不易捕獲，老一輩的人稱之為「鮡烏」。臺海采風圖考<nowiki>云</nowiki>：「鮡魚黑色如鰍，長不盈尺，二目突出於額，身多綠斑。志稱多在海邊泥塗中，善跳躍，土人以為美味。置於地上能跳，亦能行數步。」([[花跳|花跳]]?) 如畜池中，有跳石上斃者。

『[[台灣府志|台灣府志]]』稱：「烏魚，即《本草》之鯔魚；海港所產甚盛。冬至前捕之，曰正頭烏，則肥而味美；至後捕之，曰回頭烏，則瘦而味劣。官徵稅，給烏魚旗，始許採捕。按隋大業六年，吳郡獻海膾四缶井，帝以示群臣曰：昔介象殿庭釣得鯔魚，此幻化耳。今日之膾，乃是真海魚所作，來至數千里，亦是一時奇味。即出數盤以賜近臣，載之《大業拾遺記》，蓋即烏魚也。吾杭素產鯔魚，有江鯔、河鯔二種，其大者不盈尺，與郡中六、七月間所食正同。至秋深，長一、二尺，味始肥美。杭所產，遠不及逮矣。」
