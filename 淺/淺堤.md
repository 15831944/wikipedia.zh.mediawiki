'''淺堤'''是來自[[臺灣|臺灣]][[高雄市|高雄市]]的[[樂團|樂團]]，使用[[台語|台語]]及[[華語|華語]]創作歌曲，[[2015年|2015年]]成立初期以主唱之名組成'''蔡依玲樂隊'''、2016年4月更名為淺堤，現由主唱兼吉他手[[蔡依玲(淺堤)|蔡依玲]]、貝斯手[[方博聖|方博聖]]、鼓手-{[[涂嘉欽|涂嘉欽]]}-、吉他手[[周致宏|周致宏]]組成活動中，目前發行過一張手工 Demo 和一張 [[EP|EP]]，淺堤的成員平均24歲，他們生長在工業發展疲乏、經濟發展無法繼續、貧乏差距日益增加的[[高雄|高雄]]，同時也反應在他們的歌曲裡頭，如〈怪手〉描述紅毛港遷村、〈叨位是你的厝〉創作靈感源自[[新園鄉_(台灣)|新園鄉]]在地阿伯對迫遷與污染的感嘆<ref>{{Cite book | author = 鄭子儀 | title =  淺堤ー焦慮不安  | location = 臺灣高雄市 | publisher =南荒 | date = 2017年12月 | pages = 93頁 | accessdate = 2018年1月3日 | url = | language = zh-tw | quote = 淺堤的團員年齡平均24歲，生長在一個工業發展疲乏的城市之中，經濟發展不再替這塊土地帶來光榮，留下的是環境破壞以及日益增加的貧富差距，政黨輪替沒解決自威權時代遺留下來的問題，反倒土地爭議越演越烈。《怪手》描繪了紅毛港遷村的情境，《叨位是你的厝》的創作靈感從朋友轉述新園鄉在地阿伯對於污染的感嘆而來，面對歲工業發展而來的破遷與污染。}}</ref>。
而淺堤的歌一直被認為能夠反映到台灣社會現象，如講述工業的發展如何搶去平民家園的《怪手》、抒發年輕人鬱悶的《高雄》與《我們的未來》。而主音蔡依玲接受訪問時表示小時候由台灣的外島澎湖搬到高雄生活，眼睜睜看着自然生態被破壞卻未能阻止，而自己根本不能做什麼，長大後能做的就是將無奈寫於歌曲中。並表示當空氣變差，我就寧願少一點<ref>[https://bkb.mpweekly.com/cu0002/20180315-68977 高雄樂隊淺堤寫工業污染　台語譜家鄉故事]</ref>。

== 簡歷 ==
* 2015年3月，由[[蔡依玲(淺堤)|蔡依玲]]、[[李大冠|李大冠]]、[[孩子王_(樂團)|孩子王]][[周致宏|周致宏]]，三人組成'''蔡依玲樂隊'''。
* 2015年10月，舉辦「共同經驗」北中南巡迴。
* 2015年11月，鼓手[[李大冠|李大冠]]為服兵役暫別，並由[[大象體操|大象體操]]-{[[涂嘉欽|涂嘉欽]]}-代打鼓手一職。
* 2016年2月，來自少年白的方博聖加入淺堤負責貝斯手。
* 2016年4月，樂團更名為'''淺堤'''。
* 2016年5月，與電影《[[給親愛的孩子|給親愛的孩子]]》舉辦聯合小巡迴，發行手工demo《Demo. 1》<ref>{{Cite web |url = https://blow.streetvoice.com/20591-淺堤釋出新歌%E3%80%88多崎作〉手工-demo-巡迴首賣/ |title = 淺堤釋出新歌〈多崎作〉手工 demo 巡迴首賣 |author = JohnnyWen |date = 2016年5月11日 |publisher = 吹音樂 |language = zh-tw |accessdate = 2017年10月28日 |quote = 新生代樂團淺堤 shallow levée 來自高雄，日前釋出的新歌〈怪手〉用台語描述台灣史上最大規模的「紅毛港遷村案」，目前點閱次數已破萬；昨日上傳了新歌〈多崎作〉，歌名取自日本作家村上春樹作品《沒有色彩的多崎作與他的巡禮之年》中的主角之名。}}</ref>。
* 2016年9月，舉辦「已是蜃景」北中南巡迴<ref>{{Cite web |url = https://blow.streetvoice.com/24117-淺堤八月甫登大團誕生舞台-九月展開北中南巡迴/ |title = 淺堤八月甫登大團誕生舞台 九月展開北中南巡迴 |author = Shock Lin |date = 2016年8月17日 |publisher = 吹音樂 |language = zh-tw |accessdate = 2017年10月28日 |quote = 來自高雄，去年甫成軍的新生代組合淺堤 shallow levée 在今年四月份釋出了〈怪手〉的 MV，以樸實而清新的方式，描述高雄紅毛港遷村的土地議題，受到不少人的注目，音樂達人馬瓜也特別撰文推薦。樂團更在上周登上大團誕生的舞台，獲得台下觀眾一致好評。接下來，樂團即將展開「已是蜃景－北中南巡迴」，還沒有看過淺堤現場演出的樂迷，可要把握這次難得的機會。}}</ref>。
* 2016年9月，以〈怪手〉入圍第七屆[[金音創作獎|金音獎]]最佳搖滾單曲獎<ref>{{Cite web |url = https://blow.streetvoice.com/24920-2016-第七屆金音獎完整入圍名單/ |title = 2016 第七屆金音獎完整入圍名單） |author = 吹編輯 |date = 2016年9月14日 |publisher = 吹音樂 |language = zh-tw |accessdate = 2017年10月28日 |quote = 文化部影視及流行音樂產業局將於 105 年 9 月 14 日（星期三）14:00 假華山文創園區紅磚區西二館舉行「第七屆金音創作獎入圍」公佈入圍名單，邀請嘻哈天后 Miss Ko、太陽的孩子「舒米恩」以及榮獲第六屆金音獎最佳現場演出獎潛力新秀「巨大的轟鳴」來共同揭曉今年的入圍名單，引領華語創作音樂能量不間斷。}}</ref>。
