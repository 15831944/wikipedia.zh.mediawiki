'''植物園'''或'''植物（學）公園'''是一个收集、繁殖和研究[[植物|植物]]的科学研究机构，同时也为社会提供修养和教育的服务。植物园中的植物一般按其不同的种类有规划地培养，虽然植物园在布局和收藏上一般也考虑到美学观念，但其科学使用价值是最主要，这是它与一般的观赏花园的区别。大多数植物园由[[大学|大学]]或专门的科学研究机构管理。

目前全世界150多个国家分布有2400多个植物园。

== 研究 ==
植物园的科学研究工作包括研究和繁殖珍奇植物作科學用途，並保護[[瀕臨絕種|瀕臨絕種]]的植物。

== 教育工作 ==
植物园的教育工作包括向观众介绍不同的生态环境和其中的职务，植物的生理、种类、功能，植物的用途。许多植物园拥有销售鲜花、草药、蔬菜种子和书籍的商店，有些植物园引入和培养新的观赏植物。

== 历史 ==
最早的现代的植物园由[[意大利|意大利]]北部的大学建立：
* [[帕多瓦植物园|帕多瓦植物园]]（1545年）
* [[比萨|比萨]]（1543年）
* [[博洛尼亚|博洛尼亚]]（1567年）

意大利外的最早的欧洲植物园有：
* [[荷兰|荷兰]][[莱顿|莱顿]]（1590年）
* [[法国|法国]][[蒙彼利埃|蒙彼利埃]]（1593年）
* [[德国|德国]][[海德堡|海德堡]]（1597年）
