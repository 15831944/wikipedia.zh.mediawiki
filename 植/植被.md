'''植被'''是[[地球|地球]]表面所覆盖的[[植物|植物]]的总称。它是一个[[植物学|植物学]]、[[生态学|生态学]]、[[农学|农学]]和[[地球科学|地球科学]]的名词。

植被可以因为生长环境的不同而被分类，譬如[[高山植被|高山植被]]、[[草原植被|草原植被]]、[[海岛植被|海岛植被]]等。

環境因素如[[太阳光|光照]]、[[溫度|溫度]]和[[雨量|雨量]]等會影響植物的生長和分布，因此形成了不同的植被。

== [[熱帶雨林|熱帶雨林]] ==
[[File:Forest_on_Barro_Colorado.png|thumb]]的熱帶雨林]]
多位於[[低緯度|低緯度]][[熱帶|熱帶]]地區，南北緯約10度之間，如[[南美洲|南美洲]]、[[非洲|非洲]]和[[東南亞|東南亞]]附近的熱帶雨林。雨量充足，年雨量在2,000公厘以上，無明顯乾溼季之分；溫差小，平均溫約攝氏25 ~ 27度。以常綠闊葉樹為主。森林層次複雜，可以分成[[地被層|地被層]]、[[草本層|草本層]]、[[灌木層|灌木層]]、[[亞喬木層|亞喬木層]]、[[喬木層|喬木層]]和[[樹冠層|樹冠層]]等。為爭取陽光，有些樹種可以長到80公尺以上。

== 熱帶旱林 ==
[[File:Chacachacare_dry_forest_3.JPG|thumb]]的熱帶旱林]]
位於緯度約10 ~ 25度之間，主要分布於熱帶雨林南北兩側，如非洲、南美洲雨林的南北兩側及[[印度|印度]]的森林。和熱帶雨林比較，熱帶旱林有明顯的乾、溼季之分；隨著季節變化，林相會做出不同的改變。乾季時，樹木的大部份樹葉會掉光，以減少水份散失；當雨季來臨時，充沛的雨量會迅速將森林重新綠化起來。

== [[熱帶草原|熱帶草原]] ==
[[File:Oldoinyolengai.jpg|thumb]]北部的熱帶草原]]
位於緯度約10 ~ 20度之間，但較熱帶旱林更為乾燥，如非洲的疏林草原。由於雨季短、乾季長，使得森林不易生長，而形成草原的自然景觀。雨季多在夏季，其餘時間通常久旱無雨。時常發生的火災使得該區沒有茂密的森林，草地在火災過後很快生成，留存下來的樹木通常有著相當程度的防火能力。

== [[荒漠|荒漠]] ==
[[File:Kalahari_E02_00.jpg|thumb]]]]
荒漠主要分為兩個區域，一個為北緯30度，另一個為南緯30度。其主要分布於大陸內陸區或背風的山區，雨量在300公厘以下，[[蒸發|蒸發]]量大於降水量，日夜溫差大，如[[撒哈拉沙漠|撒哈拉沙漠]]、[[戈壁沙漠|戈壁沙漠]]等。隨著地點的不同，降雨量和氣溫也有很大的不同，如智利[[阿塔卡马沙漠|阿塔卡马沙漠]]位於熱帶，且幾乎全年無雨，北美的[[索诺兰沙漠|索诺兰沙漠]]卻有每年近300公釐的雨量，而蒙古的戈壁沙漠冬季均溫則可降至攝氏零下20度。主要植物為[[仙人掌|仙人掌]]、矮灌木及草本植物等耐旱植物。

== [[地中海灌木林|地中海灌木林]] ==
[[File:Flowering-garrigue4.JPG|thumb]]的硬葉灌木地]]
分布於南北緯30 ~ 40度左右的地區，夏乾熱，秋冬春涼且潮溼，全年氣候溫和，如[[地中海|地中海]]地區、美國[[加州|加州]]、南非[[開普敦|開普敦]]地區等。主要植物以硬葉灌木植物為主。夏季的旱季時常發生火災，因此此區的樹木都有相當的防火能力；灌木則生長週期短暫，在濕季迅速生長，旱季死亡，以避開旱災和火災。

== [[溫帶草原|溫帶草原]] ==
[[File:Konza1.jpg|thumb]]的溫帶草原]]
分布緯度與溫帶森林相似，但年雨量較少，約300 ~ 1,000公厘，如美國中西部、[[阿根廷|阿根廷]]及[[東歐|東歐]]、[[中亞|中亞]]至[[蒙古|蒙古]]的大草原。降雨量不足以使樹林生長，但也不致於形成沙漠，主要植物以[[禾本科|禾本科]]及[[菊科|菊科]]等[[草本植物|草本植物]]為主。

== [[溫帶森林|溫帶森林]] ==
[[File:Autumn_leaves,_Talcott_Mountain_State_Park.jpg|thumb]]的溫帶森林]]
位於緯度約30 ~ 55度之間，然而，[[生物群系|生物群系]]主要分布緯度40 ~ 50度之間，年雨量約650 ~ 3,000公厘以上，如[[東亞|東亞]]、[[西歐|西歐]]及[[美國|美國]]東部的溫帶闊葉林。以闊葉樹為主，如[[楓|楓]]、[[橡|橡]]、[[榆|榆]]等。樹葉在秋、冬季時會凋落，以減少水份及能量喪失，因此被稱為落葉樹。

這個地區是受到人類影響最大的地方。數千年來，人類砍伐樹木，發展[[農業|農業]]。今天，世界大部份的溫帶原始森林均已消失；在北美西部，只有1 ~ 2%的溫帶原始森林仍然留存。

== [[北方針葉林|北方針葉林]] ==
[[File:Mixed_Picea_(Spruce)_forest_from_Vestfold_county_in_Norway.jpg|thumb]]的針葉林]]
分布於高緯度地區，如[[歐亞大陸|歐亞大陸]]北部及[[加拿大|加拿大]]、[[阿拉斯加|阿拉斯加]]的針葉林。冬季長而嚴寒、生長季短。年雨量約200 ~ 600公厘。以[[裸子植物|裸子植物]]為主，如[[松|松]]、[[柏|柏]]、[[杉|杉]]等。終年常綠。樹葉呈針狀，可以減少水份散失。

== [[苔原|苔原]] ==
[[File:Wrangel_Island_tundra.jpg|thumb]]的苔原]]
又稱為寒原或凍原。分布於極地高緯度地區。降水量低、生長季短。主要植物為[[苔蘚|苔蘚]]、[[地衣|地衣]]及矮小灌木等。夏季短，植物在短暫的生長季節迅速開花結果。

== [[山地|山地]] ==
[[File:Mt._Karasawa_(200608).jpg|thumb]][[涸沢岳|涸沢岳]]附近的高山植物]]
山地的環境和植被隨著[[海拔|海拔]]的變化而有所不同。海拔越高，氣溫也越低。一些低緯度地區的高山上，常可以見到類似高緯度地區的植被。

在年平均溫度低於10℃的高山地帶，樹木不易生長，在森林所能到達的最高海拔，稱為[[林線|林線]]。高山地區由於常年低溫、地形陡峭、表面土壤稀少、強風吹襲、烈日照射等自然環境因素，不適合樹木生長，因此[[高山植物|高山植物]]大多為矮小的[[灌木|灌木]]或[[草本植物|草本植物]]。由於生長季節短、動物媒介少，高山寒原的植物大多具有鮮豔的花朵或果實，以吸引動物媒介傳播花粉、種子。
