《'''植物精灵'''》（{{lang-en|''Botanicula''}}）是一个由{{link-en|Amanita Design|Amanita Design}}开发的点击式[[冒险游戏|冒险游戏]]，游戏在2012年4月19日发行于[[Microsoft_Windows|Microsoft Windows]]、[[OS_X|OS X]]和[[Linux|Linux]]平台。为[[iPad|iPad]]用户设计的版本也已发布。<ref>{{cite web|url=
http://amanita-design.net/forum/index.php?topic=3813.0|title=When will be Botanicula for iPad?|date= |accessdate=2012-04-27}}</ref>游戏还计划被移植到到[[Android|Android]]和[[iOS|iOS]]系统中。该游戏正在帮助{{link-en|世界土地信托|World Land Trust}}保护数百英亩的热带雨林。

== 游戏角色 ==
*'''Mr. Lantern''' - 《植物精灵》的主角之一。形体矮小的棕色生物。吸收了种子之后色彩变得鲜亮而富有力量。
*'''Mrs. Mushroom''' - 唯一的女性伙伴。她可以收起[[菌柄|菌柄]]使自己匍匐在地。
*'''Mr. Poppyhead''' - 最为强壮的伙伴，行为莽撞。 
*'''Mr. Twig''' - 伙伴之一。他就像是一根树枝，可以延展他的肢体。身上可以长出鲜花。
*'''Mr. Feather''' - 伙伴之一。身材轻巧，擅长飞行。
*'''Spiders''' - 游戏中的反派角色。它们寄生在树上，吸取生物汁液，破坏生物的家-{}-园。

== 剧情 ==

游戏主要写了五位植物形态的小精灵（Mr. Lantern，Mrs.Mushroom，Mr. Poppy Head，Mr. Twig和Mr. Feather）保护森林生态，除掉污染森林的有害生物的故事。<ref name="Botanicula – Amanita Design"/>

游戏较为明显地分出六个章节。一开始，玩家角色Mr. Lantern受梦中之托，需要将一粒树种带往地面播种。他叫来其余的四个伙伴沿树干而下。途中遭遇到了很多谜题及险境。冲出Spider包围后，五个小精灵帮助一个精灵妈妈找到了她的三个孩子并造访他们所在的部落。在部落帮助下，五个主角准备飞往地面，却遭受了一只Spider的侵扰，主角随即与之一同摔落在地。一番地底历险之后，Mr. Lantern被Spider母体扔入体内……

== 评价 ==

植物精灵收到了绝对多数的正面评价，被认为是继[[银河历险记|银河历险记]]、[[机械迷城|机械迷城]]后，Amanita Design的又一杰作，是真正的集视觉与音乐于一体的艺术作品。<ref>{{cite web|url=http://www.metacritic.com/game/pc/botanicula |title=Botanicula for PC Reviews |publisher=Metacritic |date=2012-04-19 |accessdate=2013-08-17}}</ref><ref>{{cite web|url=http://www.gamerankings.com/pc/625904-botanicula/index.html |title=Botanicula for PC |publisher=GameRankings |date=2012-04-19 |accessdate=2013-08-17}}</ref><ref>{{cite web|url=http://www.mobygames.com/game/botanicula |title=Botanicula for Linux (2012) |publisher=MobyGames |date= |accessdate=2013-08-17}}</ref><ref>{{cite web|url=http://www.hodnoceniher.cz/pc/adventury/botanicula.html |title=Botanicula - recenze, hodnocení |publisher=Hodnoceniher.cz |date= |accessdate=2013-08-17}}</ref>
游戏在可玩性、音乐和视觉效果方面颇受赞赏。游戏不但好玩，而且能讓人感受到悲哀和欢乐的情感，《植物精灵》都能运用声效和图形与之融合。也有评论认为，游戏缺乏耐玩性，重玩一遍《植物精灵》没有太大意义。<ref>{{cite web|url=http://www.ign.com/articles/2012/04/27/botanicula-review |title=Botanicula Review |publisher=IGN |date= |accessdate=2013-08-17}}</ref><ref>{{cite web|author= |url=http://www.gamespot.com/botanicula/reviews/botanicula-review-6372148/ |title=Botanicula Review |publisher=GameSpot.com |date=2012-04-19 |accessdate=2013-08-17}}</ref><ref>{{cite web|author= |url=http://games.tiscali.cz/recenze/botanicula-recenze-59212 |title=Botanicula - recenze - GAMES.cz |publisher=Games.tiscali.cz |date=2012-04-23 |accessdate=2013-08-17}}</ref>

2012年，《植物精灵》在第14届独立游戏节中斩获“最佳音效奖”<ref>{{cite web|url=http://igf.com/2012/03/fourteenth_annual_independent_.html |title=14th annual IGF announces winners |date=March 2012 |accessdate=2012-03-10 |work=[[Independent_Games_Festival|Independent Games Festival]] |deadurl=yes |archiveurl=https://web.archive.org/web/20160304060444/http://igf.com/2012/03/fourteenth_annual_independent_.html |archivedate=2016-03-04 }}</ref>。
同年，该游戏在{{Tsl|en|IndieCade}}(国际独立游戏节)上提名展示，并赢得剧情/世界设定奖(the Story/World Design award)。<ref>{{cite web |url=http://indiegamereviewer.com/indiecade-2012-indie-game-award-winners-the-complete-list/ |title=IndieCade 2012 Award Winners-The Complete List |date=October 2012 |accessdate=2013-03-05 |work=indiegamereviewer.com}}</ref>2012年欧洲游戏奖中，获得最佳冒险类游戏奖。<ref>{{cite web|url=http://www.european-games-award.com/ |title=European Games Award |publisher=European Games Award |date=2011-08-13 |accessdate=2013-08-17}}</ref>
