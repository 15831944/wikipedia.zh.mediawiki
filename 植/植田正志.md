'''植田正志'''{{ref|name|註1}}（{{jpn|j=植田 まさし |hg=うえだ まさし |rm=Ueda Masashi}}，{{bd|1947年|5月27日|||Ueda Masashi}}）為出身日本[[東京都|東京都]][[世田谷區|世田谷區]]的男性四格漫畫家，代表作品有《[[西瓜皮先生|西瓜皮先生]]》、《[[人小鬼大|人小鬼大]]》、《[[頑皮大師|頑皮大師]]》等。

== 經歷 ==
=== 早年 ===
出身於普通上班族家庭，為家中最小的么子，上有兩個兄長，個人血型為A型。個人小時雖然就喜歡繪畫，但並沒有特別熱衷的漫畫作品<ref name="Bunshun">{{Cite web |url=http://bunshun.jp/articles/-/3341?page=3 |script-title=ja:「コボちゃんを描いているときに食事はしない」　“4コマ漫画の巨匠”植田まさしロングインタビュー　#1 |work=[[文藝春秋|文藝春秋]] |language=ja |date=2017年7月15日 |accessdate=2018年6月3日}}</ref>。

青年時在東京都立田園調布高中期間裡曾加入[[橄欖球|橄欖球]]社，當時曾因鍛鍊體能不適把身體操壞，而在醫院住了一段時間。之後評估自己不適合融入團體活動，想投入適合單獨一人完成的事項，便開始將目標放在攝影記者上<ref name="Bunshun"/>。

在就讀[[中央大學_(日本)|中央大學]]的文學系時，植田另在位於[[千代田區|千代田區]]的東京寫真專門學院上夜間班。當時是日本[[學生運動|學生運動]]盛行的期間，某次植田在拍攝學生運動的情況時，發現到有人抱持著玩樂的心態來參加運動而感到失望，因此放棄走上攝影記者的職務<ref name="MangaTime">{{Cite web |url=http://www.manga-time.com/interview/no23.html |script-title=ja:まんがタイム インタビュー |work=Manga-Time |language=ja |accessdate=2012年6月17日}}</ref>。

=== 漫畫家時期 ===
從大學畢業後，植田在哥哥開設的[[補習班|補習班]]裡一邊幫忙，一邊揣摩繪畫四格漫畫的方法<ref name="MangaTime"/>。

1971年植田正志在《週刊漫畫TIMES》上以「{{lang|ja|まち・あみち}}」的筆名，發表首次刊登的作品《{{lang|ja|衝撃マンガ2題}}》<ref name="Sniper">{{Cite web |url=http://sniper.jp/300special_issue/3005four_frame_cartoon/post_1631.html |script-title=ja:四コマ漫画の巨匠・植田まさしを読み解く！来るべき植田まさし批評のために |work=Web Sniper |author={{lang|ja|ばるぼら}} |language=ja |accessdate=2012年6月17日}}</ref>。之後同樣在《週刊漫畫TIMES》推出個人首部創作的四格漫畫系列《 {{lang|ja|ちょんぼ君}}》<ref>{{Cite web |url=https://www.hochi.co.jp/topics/serial/CO019592/20170602-OHT1T50121.html |archive-url=https://web.archive.org/web/20170614003726/https://www.hochi.co.jp/topics/serial/CO019592/20170602-OHT1T50121.html |dead-url=yes |script-title=ja:タウンページ、通販カタログから必死ネタ集め…４コマ漫画年間１１００本の植田まさしさん |work=[[體育報知|體育報知]] |language=ja |date=2017年6月3日 |archivedate=2017年6月14日 |accessdate=2018年6月3日}}</ref>。

之後植田開始長期投入四格漫畫創作，在1980年後陸續發表了《頑皮大師》、《痞子阿正》、《人小鬼大》等代表作。其中《頑皮大師》曾在發行的7集單行本內達到400萬冊的銷售紀錄，<ref name="Sniper"/>而《人小鬼大》也在2010年6月14日達到第一萬篇的連載紀錄，<ref>{{Cite web |url=https://natalie.mu/comic/news/30205 |script-title=ja:6月にお兄ちゃんになる「コボちゃん」が初めて絵本に |work=[[Natalie_(網站)|Natalie]] |language=ja |date=2010年4月11日 |accessdate=2018年6月3日}}</ref>以及在2017年4月1日邁向了第35週年的連載<ref>{{Cite web |url=https://natalie.mu/comic/news/282727 |script-title=ja:アニメ「コボちゃん」初のDVD化！放送開始25周年＆原作35周年記念し |work=Natalie |language=ja |date=2018年5月18日 |accessdate=2018年6月3日}}</ref>。

== 私人 ==
植田正志在1982年透過[[相親|相親]]結婚，育有兩女。

在青年時想從事攝影記者的期間裡，個人所欣賞的攝影師有[[安德魯·弗里德曼|安德魯·弗里德曼]]、[[東松照明|東松照明]]、[[奈良原一高|奈良原一高]]等人<ref name="Bunshun"/>。

私下鮮少在讀者面前公開露面，2011年12月2日播放的由[[松本人志|松本人志]]主持的[[富士電視網|富士電視網]]《{{lang|ja|人志松本の○○な話}}》則是植田少數登場的綜藝節目。
