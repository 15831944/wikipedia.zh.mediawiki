'''綽號'''是人的一種非正式的[[名字|名字]]，是[[別名|別名]]的一種。表示親密或喜愛的綽號也叫'''暱稱'''；表示幽默感或嘲弄的綽號也叫'''諢號'''（渾號）、'''諢名'''（渾名）。

在某些場合下，人們習慣不使用[[本名|本名]]，而是用綽號來代替，表示關係親密、喜愛之情、[[幽默|幽默]]感或嘲弄等意味。從綽號產生、獲得（對於本人來說）的角度來說也稱作'''外號'''，意思是其他人所取的名字，儘管綽號可能並不被本人所接受。

在[[粵語|粵語]]地區，綽號亦被稱作為“'''花名'''”。

== 起名方式 ==
一個人的綽號可能取自於：
* 名字的簡化或某種變形，如諧音或者是形狀類似的其他文字
* 身體的某個顯著特徵
* 對象曾經做過的某件事
* 對象的[[性格|性格]]或[[氣質|氣質]]

這些綽號有時候可能帶有貶意，例如頭部較大的人被稱作「大頭」，而不被本人所接受。由於命名者多半是親朋好友，通常不會強烈表現出自己無法接受的事實。但也有人認為這些稱呼只是親切的表現而覺得無傷大雅。

== 對象事物 ==
* 人類
* 虛構作品中出現的人物或角色
* [[寵物|寵物]]、機車、汽車等，所有物
* 快車之類的列車名（日本新幹線的[[疾風號列車|疾風號]]、[[希望號列車|希望號]]等）
* 特定的鐵道路線
* 商品或服務
* 棒球場等大型設施
* 飛行員之間所個別賦予的愛稱。（[[軍用航空飛行器|軍用機]]的飛行員）
* 異性[[偶像|偶像]]（此情形尤其常作為擬似[[戀愛|戀愛]]的對象）
* 飛機、鐵道火車等
例如在美國將[[紐約市|紐約市]]稱做「大蘋果」（Big Apple）、[[紐約州|紐約州]]稱做「帝國州」（Empire State）一般，有時會幫都市、州、或某個地方取暱稱。

== 暱稱 ==
暱稱與綽號最大的不同在由於關係親近或為了增進感情而另取一個稱呼。日語稱為「{{lang|ja|愛称}}」

== 人格化 ==
對於人以外的事物產生感情時，有時也會對該事物名稱另取綽號。例如在日本，常將身邊的日常用品或交通工具等，另取一個具有親切感的暱稱，稱之為[[擬人化|擬人化]]。維基百科的[[吉祥物|吉祥物]]維基娘，就是由[[日語維基百科|日語維基百科]]的使用者Kasuga所繪製。
