'''菜瓜布'''，或稱'''絲瓜布'''，是由[[絲瓜|絲瓜]]過熟後，纖維膨脹後形成。過熟的絲瓜又稱'''絲瓜絡'''<ref>{{Cite web|url=http://www.setn.com/News.aspx?NewsID=284165|title=絲瓜絡「天然的菜瓜布」 洗碗、洗澡都超好用|accessdate=2018-01-07|author=|date=|last=三立新聞網|publisher=}}</ref>，現今常見的海綿菜瓜布，一面是海綿，一面則有添加[[金剛砂|金剛砂]]增加硬度<ref>{{Cite web|url=http://food.ltn.com.tw/article/4031|title=金剛砂、木漿海綿？各式菜瓜布差在哪？挑選時要注意... - 食譜自由配 - 自由電子報|accessdate=2018-01-07|work=food.ltn.com.tw}}</ref>。常利用於清洗餐具。

== 用途 ==
若在絲瓜過熟不久後製作，質地較輕柔，用做清潔肌膚，可以去除角質<ref>{{Cite news|url=http://mollis2012.pixnet.net/blog/post/145691628-%E7%9A%AE%E8%86%9A%E6%B8%85%E6%BD%94%E8%81%96%E5%93%81------%E7%B5%B2%E7%93%9C%E7%B5%A1|title=皮膚清潔聖品 ---- 絲瓜絡 @ Mollis 茉莉思 :: 痞客邦 PIXNET ::|last=(mollis2012)|first=Mollis 茉莉思|newspaper=Mollis 茉莉思|accessdate=2018-01-07|language=zh-TW}}</ref>；反之，在絲瓜過熟到整顆呈現乾癟狀，此時質地較硬，製作成的菜瓜布多用作清潔餐具。

== 製作 ==
製作菜瓜布需用過熟的絲瓜，在棚架上長至咖啡色後採收。將絲瓜的皮與裡面的肉分離，把果肉拿去浸水搓揉，讓果肉與纖維分開。曬乾後敲打可把絲瓜籽取出，剩下的即是菜瓜布<ref>{{Cite news|url=http://lin139874.pixnet.net/blog/post/305134139-%E8%A3%BD%E4%BD%9C%E8%8F%9C%E7%93%9C%E5%B8%83|title=製作菜瓜布 @ 渝媽的部落格 :: 痞客邦 PIXNET ::|last=(lin139874)|first=渝媽|newspaper=渝媽的部落格|accessdate=2018-01-07|language=zh-TW}}</ref><ref>{{Cite news|url=http://blog.udn.com/frank2001fv/18574547|title=菜瓜布DIY - 阿立的鄉間生活 - udn部落格|accessdate=2018-01-07}}</ref>。

== 衛生問題 ==
使用過的菜瓜布，如果沒有立即晾乾，容易滋生細菌。當拿著充滿細菌的菜瓜布清洗身體時，如果身上有傷口，可能會導致感染[[葡萄球菌属|葡萄球菌]]<ref>{{Cite web|url=https://health.udn.com/health/story/6045/370590|title=「絲瓜絡」藏菌多 刷洗身體大不妙|accessdate=2018-01-07|author=|date=|work=元氣網|publisher=}}</ref>。

因菜瓜布常處於潮濕的情形，建議每4－6週更換一次<ref>{{Cite web|url=http://food.ltn.com.tw/article/6295|title=海綿菜瓜布可以用多久？用 3 點檢視到底該不該換！|accessdate=2018-01-07|author=|date=|work=食譜自由配|publisher=自由電子報}}</ref>。
