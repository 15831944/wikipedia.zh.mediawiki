'''菜系'''是指依据各地饮食风味差异对[[中国菜|中国菜]]进行区分的体系，本身是当代出现的[[汉语|汉语]][[新词|新词]]<ref name="a1"/>。

古代文献中所记载的中国各地饮食差异，仅有物产、食材不同。至宋代经济繁荣，带动[[餐饮业|餐饮业]]，方使人们意识到各地风味的不同<ref name="a1"/>。明代经济和城市繁荣所壮大的[[市民|市民]]阶层，更进一步促进[[中國飲食文化史#明朝|饮食文化的发展]]。期间，[[番茄|番茄]]、[[番薯|番薯]]、[[南瓜|南瓜]]、[[玉米|玉米]]、[[辣椒|辣椒]]、[[大蒜|大蒜]]等外来食材进入中国<ref>{{Cite web |url=http://history.sohu.com/20150526/n413688024_3.shtml |title=舌尖上的古代中国：明朝外来食材掀革命 |accessdate=2018-05-02 |author= |date=2015-05-26 |work= |publisher=搜狐网，来源：南报网|language=Zh-hans }}</ref>。但至清代中期，中国社会方才确立主要[[调味料|调味料]]——辣椒和[[蔗糖|蔗糖]]的运用<ref name="a1"/>。

[[晚清|晚清]]以来，[[中国工业史|中国进入工业化]]、[[中国城市化|城市化]]的时代，[[工人|工人]]、市民阶层的壮大，亦促使餐饮的商品化<ref>{{Cite web |url=http://sh.qq.com/a/20180416/002020.htm?d=123 |title=本帮菜不完全“本帮” 曾经风靡上海滩的是这个菜系 |accessdate=2018-05-03 |author= |date=2018-04-16 |work= |publisher=腾讯网，上观新闻|language=Zh-hans }}</ref>和现代化。类似当代菜系概念的区分——“帮口”，即以'''某帮菜'''的命名方式出现于民国时期。[[本帮菜|本帮菜]]、[[杭帮菜|杭帮菜]]等命名沿用至今。中华人民共和国在1949年成立后，受政治因素影响，以去除“[[封建社会|封]][[資本主義|资]][[修正主义_(马克思主义)|修]]”色彩的'''菜系'''一词替代了'''帮菜'''。研究者认为，这是1950、60年代任商业部长的[[姚依林|姚依林]]在对外交流中发明的[[新词|新词]]。出现在文献中则在1970年代以后。当代社会所熟知的[[八大菜系|八大菜系]]，则出现于[[改革开放|改革开放]]之后。1980年6月20日，《[[人民日报|人民日报]]》刊登的汪绍铨文章《我国的八大菜系》，是最早的文献记录。此后，因经济利益影响，中国菜系之争绵延不绝。[[网络文化|网络文化]]中，将[[食堂|食堂菜]][[恶搞|恶搞]]为第九大菜系<ref name="a1">{{Cite web |url=http://news.ifeng.com/a/20180224/56228765_0.shtml |title=八大菜系的历史，比春晚长不了几年 |accessdate=2018-05-03 |author=吴余 |date=2018-02-24 |work= |publisher=凤凰网|language=Zh-hans }}</ref>。
