'''菜園涌邊街總站'''（[[葡萄牙文|葡萄牙文]]：'''R. MARGINAL DO CANAL DAS HORTAS/TERMINAL'''、{{lang-en|'''Canal Das Hortas St. Terminal'''}}）位於[[澳門|澳門]]台山區菜園涌邊街的一個終點站，[[澳門新福利公共汽車有限公司|新福利]]'''[[澳門巴士9A路線|9A]]'''、'''[[新福利16路線|16]]'''，及'''[[新福利34路線|34]]'''路線以該站為終點站。[[澳門新福利公共汽車有限公司|新福利]]'''[[澳門巴士25X路線|25B]]'''路線停靠該站。

== 沿革 ==

== 歷史 ==
*2008年12月7日，澳門特區政府交通事務局進行第一階段巴士路線重整，其中[[澳門巴士34路線|34]]路線行程縮短，由[[媽閣總站|媽閣總站]]改為本站。<ref>[http://www.dsat.gov.mo/show_news.asp?id=633635823115 交通事務局新聞  - 首階段巴士路線重整下月7日實施]</ref>
*2018年2月3日，澳門特區政府交通事務局優化巴士運作及擴大服務範圍，其中[[澳門巴士25X路線|25X]]路線更改發車站，由「街總社區服務大樓」站改為本站。
*2018年4月28日，[[澳門巴士9A路線|9A]]路線更改發車站，由「台山中街」站改為本站。

== 設施 ==
*[[澳門新福利公共汽車有限公司|澳門新福利]]站長更亭。
*[[澳門新福利公共汽車有限公司|澳門新福利]]自設鋪位，內有洗手間。

== 爭議 ==
*台山坊會指菜園涌邊街巴士總站，自從34號線的起始站由原媽閣總站移往台山菜園涌邊街，使進入巴士總站的公共巴士車輛增多。公共巴士和霸佔巴士專用停車位的車輛，停車時通常沒有關掉引擎，車輛發出的噪音和排出的尾氣，影響了附近居民的生活和健康。<ref>[http://www.macaodaily.com/html/2009-02/04/content_280002.htm 澳門日報電子版  - 菜園涌邊街巴士總站欠秩序 ] {{webarchive|url=https://web.archive.org/web/20090216163143/http://macaodaily.com/html/2009-02/04/content_280002.htm |date=2009-02-16 }}</ref>
