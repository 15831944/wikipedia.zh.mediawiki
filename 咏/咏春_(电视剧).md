《'''咏春'''》，由[[香港|香港]]著名演员[[谢霆锋|谢霆锋]]、[[洪金宝|洪金宝]]、[[元彪|元彪]]以及大陆著名演员[[计春华|计春华]]、[[斯琴高丽|斯琴高丽]]等合作出演的电视剧，该剧讲述了[[广东省|广东省]][[佛山|佛山]][[咏春拳|咏春拳]]传人[[梁赞|梁赞]]父子的人生经历。

== 剧情 ==
该剧改编自1981年电影《[[敗家仔_(1981年電影)|败家仔]]》，当时主演为洪金宝和元彪。<ref>{{cite web |url= http://ent.people.com.cn/GB/8222/86598/98974/98991/6119277.html|title=谢霆锋告别“花拳绣腿”《咏春》展现真功夫 |accessdate=2007年8月15日 |publisher=人民网 |work= |language =zh-cn}}</ref>

佛山赞生堂医馆主人梁赞是[[反清复明|反清复明]]的[[少林寺|少林寺]]弟子[[黄华宝|黄华宝]]之徒弟，他生了两个儿子，一个叫梁壁、另一个叫梁春，梁壁喜好游猎，崇尚武艺，酷似[[八旗子弟|八旗子弟]]，梁赞对此颇为不满，并且不愿意教授他咏春拳，而老二梁春体弱多病，一副病怏怏的样子，也不是习武之材。在一次梁赞和儿子梁壁发生误会之后，梁壁去了[[英属香港|英属香港]]，遇到了梁赞的师傅黄华宝，他传授了梁壁[[咏春拳|咏春拳]]。

高明和妹妹高小翠是广东省的穷苦孩子，妹妹从小就被卖掉，高明上佛山，拜入了[[劈卦门|劈卦门]]，在佛山，他认识了少女小馒头，在小馒头的帮助下找到了妹妹，日久生情，高明也爱上了小馒头，小馒头却爱慕的是梁壁，高明和梁壁就成了情敌，天长日久，高明和梁壁结下了冤仇。由于妹妹得了病，哥哥高明没有钱，他投靠了富商龙金胜，帮忙贩卖[[鸦片|鸦片]]，不料妹妹却被龙金胜强奸，为报仇，高明杀害了龙金胜全家。杀人之后，高明逃匿，他拜师于[[铁线拳|铁线拳]]洪福门下，得到其真传，洪福的女儿洪莲爱上了他，逼他结婚，高明不肯也把师傅和其女儿杀害。

高明自以为武功已练成，再次回到佛山，和梁壁决一死战，被梁壁打败，后绑架小馒头作为人质妄图杀害梁壁却被佛山警察车震忠拘捕，为了给咏春拳延续脉络，梁壁开始在佛山的广场亲自传授大家咏春拳，使得咏春拳开枝散叶，广为传播。

== 花絮 ==
刘梓妍在剧中扮演小馒头，是一个厨艺高超的女子，对此，他表示：“我在剧里是个[[烹饪|烹饪]]高手，可是现实里我是根本不会做菜的。刚进组时，每天我都会苦练刀功，到了现在，我切菜、切丝什么的，可厉害了！我对妈妈说，回到家我就给她做菜，她开心得不行———这个女儿终于肯进厨房了。” <ref>{{cite web|url=http://ent.people.com.cn/GB/8222/86598/98974/98991/6119318.html|title=《咏春》热播 元彪洪金宝谢霆锋表演真功夫|accessdate=2007年8月15日|publisher=人民网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120524142358/http://ent.people.com.cn/GB/8222/86598/98974/98991/6119318.html|archivedate=2012年5月24日}}</ref>
