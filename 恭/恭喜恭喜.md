《'''恭喜恭喜'''》，是中国作曲家[[陈歌辛|陈歌辛]]在1946年创作和发表的中文歌，被广泛传唱为[[春节|贺年]]歌曲。此曲本来是庆祝抗日战争胜利的歌曲，发表后随即流行一时，又因当时碰巧正值戰後首個农历新年，成为全球华人的经典贺年歌曲。

== 创作历程 ==
《恭喜恭喜》原先是庆祝中国[[抗日战争|抗日战争]]胜利的歌曲<ref name="東森新聞"/>。1946年，由于日本已战败投降，[[國民政府|國民政府]]重新把首都定为曾被日本占领的[[南京|南京]]；作曲家陳歌辛为此以笔名“慶餘”创作《恭喜恭喜》一曲，同时作曲、作词，以表达抗战胜利的兴奋之情<ref name="東森新聞"/><ref name="张成觉"/>。此曲最初由歌手[[姚敏|姚敏]]、[[姚莉|姚莉]]两兄妹合唱<ref name="曾慧佳"/>。

== 歌曲特色 ==
《恭喜恭喜》全曲只有四句曲调<ref name="陈钢2"/>。由于此曲最初是纪念战争苦难结束的作品，曲调倾向伤感，并采用通常用来表达悲伤的[[小调|小调]]，而非通常用来表达欢庆的[[大调|大调]]<ref name="Mothership.sg"/>。此曲本来没有[[锣|锣]][[鼓|鼓]]伴奏，后来编曲时加入，以配合新年气氛<ref name="東森新聞"/>。

《恭喜恭喜》有五段歌词，其中以第一段最为闻名<ref name="東森新聞"/>。第一、五段描写人们互道喜讯，第二至四段写寒冬结束后春来大地，并提到歷盡苦难而生的盼望之情<ref name="東森新聞"/>。全曲歌词表面上在说庆祝农历新年的情形，实际上是描写中国人民在抗战胜利后的举国欢腾景象，记叙民众经过长年战火后苦尽甘来<ref name="自由時報"/><ref name="中時電子報"/>。例如，“經過多少苦難，歷盡多少磨練”一句，本意是忆述八年抗战的奋斗，后来解读为告慰人们过去一年来的辛勤工作；“多少心在盼望、盼望春的消息”一句，本意是写战时人民对胜利的盼望，后来解读为人们的新年祝福<ref name="東森新聞"/>。歌词琅琅上口，部分句尾押一韻<ref name="東森新聞"/><ref name="曾慧佳"/>。“恭喜”一词在全曲出现了数十次<ref name="東森新聞"/>。

== 影响 ==
由於符合抗战胜利的时代背景，《恭喜恭喜》发表后随即流行一时<ref name="曾慧佳"/>。此曲发表时碰巧正值农历新年（一说为新年前夕，一说为新年期間），加上歌词与农历新年的气氛也颇为应景，因此被大众当成了贺年歌曲，在往后的春节都经常被传唱<ref name="東森新聞"/><ref name="自由時報"/>。此曲其后流传至台灣、香港、澳门等华人地区，以至海外華人社會，成为全球华人在春节都会唱的经典贺年歌曲<ref name="陈钢2"/><ref name="陈钢1"/>。《恭喜恭喜》也是陳歌辛在中國音樂史上的成名作<ref name="中時電子報"/>。
