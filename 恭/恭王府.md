'''恭王府'''位于[[中國|中國]][[北京市|北京市]][[西城区|西城区]]前海西街17号，是[[清朝|清朝]]后期[[恭亲王|恭亲王]][[奕訢|奕訢]]的府邸及花园，现在是[[全国重点文物保护单位|全国重点文物保护单位]]。恭王府花园在20世纪90年代作为旅游景点对外开放，而恭王府主体建筑也在[[2008年北京奥运会|2008年北京奥运会]]举办时修缮完毕，并向公众开放。

==历史==
恭王府及花园原是[[固倫和孝公主|固倫和孝公主]]在和珅作為二品户部侍郎邸的規制基礎上，花六年增建的公主府，相當於郡王府規制，西路建築是大臣[[和珅|和珅]]的府邸以一品大員建造，嘉慶四年（1799年）正月，和珅賜死，四月「和珅之宅，賞給慶郡王[[永璘|永璘]]居住；和珅之園，賞給成親王[[永瑆|永瑆]]居住。」，咸豐元年后由于庆亲王的孙子[[奕劻|奕劻]]世袭降低为[[輔國將軍|輔國將軍]]<ref>豐元年內務府檔案："將輔國將軍奕劻府第，官為經營，賞給恭親王居住。"咸豐五年十二月初四日（1856年1月11日）內務府檔案記載：「劻貝子門上為呈報事，……現屆醇郡王分府需用府第，自貝子以下居住官府處所及房間數目希即詳查。……今查得本貝子現居府第系咸豐元年三月奉旨賞換定府大街官房一所，共計房一百六十餘間。今門上派委三等侍衛扎拉芬謹持門文赴貴府呈報。」</ref>，換至得定府大街原大學士[[琦善|琦善]]宅，原王府由内务府收回，后于[[咸丰|咸丰]]年间赐予[[奕訢|奕訢]]作为府邸，称为恭王府。1921年，恭亲王[[奕訢|奕訢]]的孙子[[溥伟|溥伟]]、 [[溥儒|溥儒]]（[[溥心畲|溥心畲]]）将恭王府和花园抵押给[[天主教|天主教]]会，后由[[辅仁大学|辅仁大学]]买去作为校舍，后先后为[[北京师范大学|北京师范大学]]、[[中国音乐学院|中国音乐学院]]的校舍。恭王府建筑曾部分为北京空调机厂占用，1980年代腾退。1996年10月起，恭王府花园作为旅游景点供公众参观。2008年8月20日，修复后的恭王府全部对外开放。

2017年，文化部恭王府博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。

==建筑==
恭王府由府邸和花园两部分组成，南北长约330米，东西宽180余米，占地面积约61120平方米，其中府邸占地32260平方米，花园占地28860平方米。

恭王府王府在南，花园在北，由高高的后罩楼将王府与花园分开。

恭王府分中东西三路，分别由多过四合院组成，后为长160米的二层后罩楼。

恭王府花园也分为三路。中路是西洋门，独乐峰，蝠池，安善堂及左右配殿明道堂，棣华轩，福字碑，邀月台，蝠厅。东路是怡神所[[垂花门|垂花门]]，大[[戏楼|戏楼]]，芭蕉院，西路是湖心亭，澄怀撷秀。此外还有[[龙王庙|龙王庙]]、榆关、妙香亭、[[流杯亭|流杯亭]]、艺蔬圃。
