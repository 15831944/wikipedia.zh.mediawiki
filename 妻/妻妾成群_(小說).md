'''《妻妾成群》'''係由[[中國|中國]][[小說家|小說家]][[蘇童|蘇童]]所著，為一中篇小說之合集，於1991年出版，該合集由〈一九三四年的逃亡〉、〈罌粟之家〉及〈妻妾成群〉等三篇作品集結成冊。<ref>{{cite web |author1=博客來 |title=妻妾成群(3版1刷) |url=https://www.books.com.tw/products/0010531859 |publisher=遠流}}</ref>其中〈妻妾成群〉後經改編，成為由[[張藝謀|張藝謀]]所執導的《[[大紅燈籠高高掛|大紅燈籠高高掛]]》的電影原著小說。<ref>{{cite web |author1=博客來 |title=大紅燈籠高高掛 DVD |url=https://www.books.com.tw/products/D020067390 |publisher=豪客}}</ref>此書曾列入[[20世紀中文小說100強|20世紀中文小說100強]]，第81強。<ref>{{cite web |author1=亞洲週刊 |title=二十世紀中文小說一百強排行榜 |url=https://www.yzzk.com/cfm/special_list3.cfm?id=1367574084132 |publisher=亞洲週刊}}</ref>
