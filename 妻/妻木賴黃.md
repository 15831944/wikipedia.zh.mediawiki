'''妻木賴黃'''（{{jpn|j=妻木頼黄|hg=つまき よりなか|rm=Tsumaki Yorinaka}}，{{bd|1859年|2月22日|1916年|10月10日|catIdx=Tsumaki Yorinaka}}）是日本的[[建筑师|建筑师]]。負責許多[[大藏省|大藏省]]等等的官廳建築，確立了[[明治|明治時代]]的官廳營繕組織。十分投入日本[[國會議事堂|國會議事堂]]的建設，但他在世時建築卻尚未開工建設。曾是工手學校（現工學院大學）建築學科教職員。

== 經歷 ==
1859年，作為青山幸道的兒子、繼承[[妻木廣忠|妻木氏]]的幕府之臣[[旗本|旗本]]妻木源三郎賴功的長男<ref>{{cite book |author1=村松貞次郎 |title=日本近代建築の歴史 |date=1977 |publisher=NHK出版 |isbn=4140013001 |page=119}}</ref>，於[[江戶|江戶]]出生，乳名為久之丞。因為父親賴功在1862年前往長崎表立合御用赴任，在當地過世的原因，3歳時妻木賴黃便成為第12代當家。明治9年（1876年）賣掉家產渡海至美國，被告誡要回日本求學而歸國。在1878年，進入工部大學校造家學科（今天的東大建築學系）就讀，跟隨[[喬賽亞·康德|康德]]學習，是[[辰野金吾|辰野金吾]]的後輩。1882年，被退學，前往美國留學，取得[[康乃爾大學|康乃爾大學]]的學士學位<ref>{{cite book |author1=藤森照信 |title=日本近代建築 |date=2014-03-25 |publisher=五南 |page=191}}</ref>，1885年歸國。

回國後在東京府工作，1886年，在因議院（國會議事堂）建設工作所組織的（内閣）臨時建築局裡工作<ref name= 近代1>{{cite book |author1=近江榮 |author2=岡田新一 |author3=村松貞次郎 |editor=村松貞次郎 |title=近代建築史概說 |date=1978 |publisher=彰國社 |page=205}}</ref>。作為官廳集中計畫的一環，為了研究議院建築，和渡邊讓、河合浩藏與職人們一起前往[[德国|德国]]留學，1888年回國<ref>{{cite book |author1=村松貞次郎 |title=日本近代建築の歴史 |date=1977 |publisher=NHK出版 |isbn=4140013001 |page=120}}</ref>。結果國會議事堂建築物的建造，將透過木材來臨時建造，並且該建築物的建造工作已經被停止。

他也負責了大藏省負責港灣、稅關、煙草鹽專賣等等施設的建設工作<ref>{{cite book |author1=村松貞次郎 |title=日本近代建築の歴史 |date=1977 |publisher=NHK出版 |isbn=4140013001 |page=121}}</ref>。1894年[[甲午戰爭|甲午戰爭]]之際，決定在作為大本營的[[廣島市|廣島]]建設臨時議院（廣島臨時假議事堂），因他在短期內便完成，因此功績得到敘勳。此外，也曾參與奈良[[东大寺|东大寺]]大佛殿的修復。

1901年至歐美視察，同年取得工學博士學位。

[[日俄戰爭|日俄戰爭]]後，在[[桂太郎|桂内閣]]任內，要求重建議院建築的聲浪再次升高，辰野金吾要求舉行公開的競圖比賽，並批評妻木等人的議院設計<ref>{{cite book |author1=藤森照信 |title=日本近代建築 |date=2014-03-25 |publisher=五南 |page=193-194}}</ref>。[[桂太郎|桂内閣]]因[[大正政變|大正政變]]被倒閣後，議院建築計畫也延期，妻木辭去官職，1916年因病去世。

== 建築風格 ==
妻木賴黃被建築史界歸類為德國派（相較於辰野金吾的英國派），其作品有強烈的[[巴洛克艺术|巴洛克風格]]，具有厚重的威嚴和一些怪奇的黑暗感<ref name= 近代1/>，但會配合建築的格式採用不同建築的樣式，橫濱正金銀行是採用德國巴洛克，東京府廳、東京商業會議所、神戶地裁是採用德國文藝復興和巴洛克中間樣式<ref>{{cite book |author1=藤森照信 |title=日本近代建築 |date=2014-03-25 |publisher=五南 |page=197}}</ref>，而日本勸業銀行則採用了和風樣式。
