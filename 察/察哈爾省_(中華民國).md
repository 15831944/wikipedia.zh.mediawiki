'''察哈爾省'''，簡稱'''察'''，為[[中華民國|中華民國]][[塞北四省|塞北四省]]之一，省會[[張垣|張垣]]（又稱張家口，當時叫張家口廳，同時張家口以叫張垣而聞名），於1948年取消察哈爾省建制，和[[綏遠省_(中華民國)|綏遠省]]合併為內蒙古自治區。

== 省名由來 ==
以[[察哈尔部|察哈尔部]]蒙古命名。

== 地理 ==
[[萬里長城|萬里長城]]將察哈爾省分為察南及察北。[[察北沙漠|察北沙漠]]延伸至[[外蒙古|外蒙古]]。

== 管轄範圍 ==
民國二年（1913年），以直隸省口北道和綏遠都統、察哈爾部、錫林郭勒盟設置'''[[察哈爾特別區|察哈爾特別區]]'''。1928年改置為省，轄境相當今河北省張家口市，北京市延慶，內蒙古自治區錫林郭勒盟大部，烏蘭察布市東部（包括化德县、商都县）。民國三十六年（1947年），全省土地面積為278957[[平方公里|平方公里]]（一說283675平方公里）。東接[[熱河省_(中華民國)|熱河省]]、[[遼北省_(中華民國)|遼北省]]，北鄰[[興安省_(中華民國)|興安省]]、[[蒙古地方|蒙古地方]]，西界[[綏遠省_(中華民國)|綏遠省]]、[[山西省_(中華民國)|山西省]]，南接[[河北省_(中華民國)|河北省]]。

== 人口 ==
民國三十六年（1947年）上半年統計為211萬4288人，下半年統計為215萬54人。

== 歷史沿革 ==

[[清朝|清朝]]時期稱「[[張垣特區|張垣特區]]」。民國二年（1913年）設置'''察哈爾特別區'''。民國十七年1928年改置為省。

民國二十二年（1933年）[[長城戰役|長城戰役]]之後，[[日軍|日軍]]越過長城，向[[華北|華北]]滲透，並於該年春佔領察哈爾部分地區。[[馮玉祥|馮玉祥]]、[[吉鴻昌|吉鴻昌]]、[[方振武|方振武]]等人在張垣於1933年5月26日成立[[察哈爾民眾抗日同盟軍|察哈爾民眾抗日同盟軍]]，於6月開始向察哈爾和[[熱河|熱河]]的日軍發動進攻，在7月中旬將日軍全數逐出察哈爾，此為[[九一八事變|九一八事變]]以來中國軍隊首次收復失地。同時在[[南京國民政府|南京國民政府]]以實現軍令統一為由，派軍進逼張垣之下，同盟軍被迫解散。

1937年至1945年抗戰期間，察哈爾省被[[日本|日本]]佔領並成為[[德王|德王]]領導的日本控制區[[蒙疆|蒙疆]]的一部份。民國三十四年（1945年）5月至6月，[[晉察冀邊區|晉察冀邊區]]的八路軍，發起[[察南戰役|察南戰役]]，攻下懷安、漱源等縣城，並向平綏路兩測和察北地區發展。8月15日日本投降後，蔣中正命令國軍沿平綏、同蒲、平漢、津浦等鐵路，向華北推進受降。其中，傅作義部隊則沿平綏線已逼近張垣。為了阻止傅作義部隊東進，中共晉察冀軍區、晉綏區軍進結5.3萬主力對傅作義發起攻擊，傅作義被迫西撤。民國三十五年（1946年）中共發起[[大同集寧戰役|大同集寧戰役]]，傅作義乘中共主力在大同之際，率領36集團軍奇襲奪取了中共華北區中心城市張垣。

民國三十七年（1948年）11月，[[平津會戰|平津會戰]]爆發，[[解放軍|解放軍]][[華北軍區第3兵團|華北軍區第3兵團]]向張家口地區守軍發起攻擊，[[國軍|國軍]]第11兵團、傅作義嫡系部隊第三十五軍，救援的第104、第16軍全數消滅。12月24日，解放軍佔領張垣，中華民國政府在察哈爾省的政府機構徹底瓦解。

== 行政區劃 ==
=== 省會 ===
初駐直隸省張北縣，民國十七年改置為省，省會駐萬全縣。民國三十六年（1947年）6月析萬全縣縣城置[[張垣市|張垣市]] ，省會改為張垣，在今[[張家口市|張家口市]]市區。

=== 縣、市、設治局 ===
民國三年（1914年），以直隸省口北道[[張北縣|張北縣]]、[[獨石縣|獨石縣]]（後改稱[[沽源縣|沽源縣]]）和[[多倫縣|多倫縣]]3縣，綏遠特別區[[興和縣|興和縣]]、[[陶林縣|陶林縣]]、[[豐鎮縣|豐鎮縣]]和[[涼城縣|涼城縣]]4縣，察哈爾部8旗，錫林郭勒盟10旗，各旗牧廠設置'''察哈爾特別區'''。先後增設[[商都縣|商都縣]]、[[寶昌縣|寶昌縣]]、[[康保縣|康保縣]]、[[集寧縣|集寧縣]]4縣。民國十七年1928年改置為省。興和、陶林、集寧、豐鎮和涼城5縣劃入綏遠省，並劃入[[河北省_(中華民國)|河北省]]舊[[口北道|口北道]]所轄10縣：宣化、赤城、萬全、懷來、蔚縣、陽原、龍關、延慶、懷安、涿鹿。至民國三十七年（1948年）以前，全省轄1市、19縣、4行政督察區，18旗、1牧場，而旗隸屬於[[錫林郭勒盟|錫林郭勒盟]]、[[察哈爾盟|察哈爾盟]]：

=== 行政督察區 ===
抗戰勝利後，全省劃為4區，張垣市及萬全、宣化、懷來、懷安、張北5縣直屬省政府。行政院於民國三十六年（1947年）6月核准備案。同年8月6日置第四區，民國三十七年（1948年）4月27日置第一區行政督察專署。各區轄縣如下：

*察哈爾省政府直轄區，轄張垣市及萬全、宣化、懷來、懷安、張北5縣。
*第一區，專署駐多倫縣，轄[[多倫縣|多倫縣]]、[[寶昌縣|寶昌縣]]、[[沽源縣|沽源縣]]3縣。
*第二區，專署駐新明縣，轄[[新明县_(中华民国)|新明縣]]、[[尚義縣|尚義縣]]、[[商都縣|商都縣]]、[[康保縣|康保縣]]4縣。
*第三區，專署駐龍關縣，轄[[龍關縣|龍關縣]]、[[崇禮縣|崇禮縣]]、[[赤城縣|赤城縣]]、[[延慶縣|延慶縣]]4縣。
*第四區，專署駐蔚縣，轄[[蔚縣|蔚縣]]、[[涿鹿縣|涿鹿縣]]、[[陽原縣|陽原縣]]3縣。
