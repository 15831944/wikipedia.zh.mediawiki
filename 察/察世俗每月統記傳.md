'''《察世俗每月統記傳》'''（'''Chinese Monthly Magazine'''）创刊于1815年8月5日<ref>{{cite web|url=http://history.cnki.net/defaul.aspx?tjType=subject_search&searchword=%E7%BB%9F%E8%AE%A1|title =历史上的今天|accessdate = 2009年6月4日|publisher = CNKI|language = zh-hans}}</ref>，是世界上第一個以[[華人|華人]]為對象的[[中文|中文]]近代报刊<ref>{{cite book |title = 《中国新闻史新修》|author = 吴廷俊|pages =29 |publisher = 复旦大学出版社 |date = 2008|language = zh-hans |isbn=978-7-309-06146-8|accessdate = 2008年10月}}</ref>，也是[[馬來西亞|馬來西亞]]第一個中文[[报刊|报刊]]。新闻史上也普遍被视为以中文出版的第一種現代報刊 (月刊) <ref>{{cite book |title =中国期刊发展史 |author = 宋应离 |accessdate =2009年6月4日 |language =zh-hans |publisher =河南大学出版社|isbn=9787810417846}}</ref>，內容主要是宗教，次為新聞、新知識。该刊每期五至七页，约2000字，初印500册，后增至1000册，免费在南洋华侨中散发，于1821年停刊，共出80多期<ref>{{cite web|url = http://www.xuebao.tyut.edu.cn/qkxh/News%20%E6%96%87%E4%BB%B6%E5%A4%B9/%E4%B8%AD%E5%9B%BD%E6%9C%9F%E5%88%8A%E4%B9%8B%E6%9C%80.htm|title = 中国期刊之最|publisher = 太原理工大学学报|date = 2008|language = zh-hans|accessdate = 2009年6月4日|deadurl = yes|archiveurl = https://archive.is/20121127060546/http://www.xuebao.tyut.edu.cn/qkxh/News%20%E6%96%87%E4%BB%B6%E5%A4%B9/%E4%B8%AD%E5%9B%BD%E6%9C%9F%E5%88%8A%E4%B9%8B%E6%9C%80.htm|archivedate = 2012年11月27日}}</ref>。

== 歷史 ==
* 1807年9月，作为[[伦敦|伦敦]][[伦敦会|布道会]]派往[[大清国|大清国]]的首名[[传教士|传教士]][[马礼逊|马礼逊]]（Robert Marrison, 1782—1834），在寻求[[美国|美国]]政府援助之后来到大清國[[廣州|廣州]]（今[[中华人民共和国|中华人民共和国]]广州）。为建立能放手开展传教活动的基地。由于[[語言|語言]]不通，以及对[[汉语|汉语]]缺乏认识，当时马礼逊的主要任务是先掌握漢語，然后翻译《[[圣经|圣经]]》、编辑出版英汉辞典，为传教活动克服語言障碍做准备。
* 1811年，大清國政府公布了《嚴定西洋人傳教治罪專條》，实行严格的闭关[[锁国政策|锁国政策]]，严禁西洋人“私自刊刻經卷，倡立講會”，向人民講經傳教，对违反者以“絞决”处治。由於大清國的嚴厲管制，使得[[傳教士|傳教士們]]後來以馬六甲為基地。
* 1812年，伦敦布道会又派遣传教士[[米怜|米怜]]（William Milne,1785—1822）前往[[清朝|大清國]]协助马礼逊从事传教活动，1813年7月抵达[[澳门|澳门]]。鉴于在中国境内公开发行传教宣传品极端危险，米怜建议将对华传教和出版基地暂时置于[[南洋|南洋]]一带，于是马礼逊遂派遣米怜前往考察。1814年冬，米怜先期在[[爪哇|爪哇]]一带进行试探性活动，进展颇为顺利。
* 1814年，[[東印度公司|東印度公司]]的势力已遍及马来半岛的[[马六甲|马六甲]]、[[爪哇|爪哇]]和[[槟城|槟城]]等[[華侨|華侨]]聚居地，由於馬六甲當時為來往東西方的必經國際港口，又較靠近大清國，而清國政府又嚴厲管制傳教活動，所以米怜回广州后便建议定馬六甲作為傳教基地。
* 1815年4月17日，马礼逊带领助手传教士米怜和來自大清國刻字工人[[梁发|梁发]]等35天后到达华人聚居的[[殖民地|英属殖民地]]马六甲，並設立一印刷所。
* 1815年8月5日，“立义馆”免费学校开学；同一天，米怜筹办的中文定期刊物《察世俗每月统记传》创刊。
* 1821年，因为米怜病重，《察世俗每月统计传》停刊，歷時7年，共出7卷80多期。

== 刊名的解释 ==
察世俗一词反映了播道者的世界观、人生觀和价值觀。也就是說，該刊是关于播道者用他们的眼光看待世界、时空、万事万物和人与人、人与神、人与万物之间关系及其内容的。

[[中文|中文]]名稱“察世俗”可能是英文“Chinese”的译音，“每月”即月刊，“统記传”即[[杂志|杂志]]的意思。

== 出版宗旨与任务 ==
马礼逊和米怜受英国基督教新教布道会指令在马六甲设立传教播道基地，其中创办中文刊物的目的在于为大清国播道创造条件。

* 传播[[基督教|基督教]]的教义——“阐发基督教义为根本要务”<ref>{{cite book |title =基督教在华最初10年之回顾 |author = 米怜 |date = 1819 |publisher =马六甲英华书院}}</ref>
* 普及一般知识
* 传播足以促使人们反思及激发其觉悟的公众事物与时事

== 刊载内容大纲 ==
《察世俗每月统记传》在刊发内容上，主要以“神理”（基督教教义）为主，“人道”（伦理道德）其次，再次为“国俗”（各国风土人情）。
此外，还刊载天文地理知识文章，如《论行星》、《论地为行星》、《论日食》等。它还发表一些诗歌，寓言一类作品以及“告贴”（即告示广告）。

* 1815年：《神理》（连载），《圣经之大意》，《解信耶稣之论》；在第二期登载过一篇题为《月食》的预告性新闻，是中文近代报刊上的第一条消息。
* 1816年：《古今圣史记》（连载至1820年），《进小门走窄路解论》；
* 1817年：《万人有罪论》，《神主无所不知无处不在论》；
* 1818年、1819年：《圣事节注》（连载）；
* 1820年：《引圣录句证神原造大地》；
* 1821年：《圣书卷分论》，《旧遗诏书卷分》，《论新遗诏书》。

除了圣经汉译及介绍基督教教義的文章外，《察世俗每月统记传》也刊载一些批判异教及其与基督教区别的短文，不外乎世界歷史、民俗風情、天文等方面的知識，並提供港口貨物商情。 

特色：

* 圖文並茂：中國人之天文知識，仍停留在[[地心說|地心說]]。《察世俗每月統記傳》以淺顯文字搭配插圖，對[[哥白尼|哥白尼]]《[[日心說|日心說]]》作一系統性介紹，闡明並非太陽繞地球轉，而是地球繞太陽轉，破除舊有觀念。
* 將西式[[標點符號|標點符號]]引入：幫助[[中文|中文]]有斷句，包括「、」「，」「。」和地名、人名專用號「――」。

== 出版 ==
* 出刊時間：基本上每陰曆月初出版，连续出版至1821年。
* 出刊量：平均每期6〜9页，共出版7卷，總計524页。
* 印刷方式：使用[[木刻|木刻]][[雕版印刷|雕版]]。
* 編排方式：以書本形式編排，採竹紙印刷，線裝，尺寸-(高19cm，宽12cm)。封面正中为特大号方[[宋体|宋体]]字，刊名自上至下竖排，两竖线将刊名框于正中；右侧写有圣贤警语：“子曰多闻择其善者而从之”；封面左下侧写有“博爱者纂”。
* 发行範圍：免费发行，遍及东南亚各地，藉通信遊歷船舶之便，傳佈於南洋群島、[[暹羅|暹羅]]、[[越南|安南]]各地[[華僑|華僑]]薈萃之區，而中國境內亦可見；若外地[[華人|華人]]需要以函索即寄。
* 總發行量：最初三年，每期印刷份数约500份，1819年每期增至1000份，最后增至2000份。
