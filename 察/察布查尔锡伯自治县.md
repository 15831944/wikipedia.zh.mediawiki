'''察布查尔锡伯自治县'''是[[中华人民共和国|中华人民共和国]][[新疆维吾尔自治区|新疆维吾尔自治区]][[伊犁哈萨克自治州|伊犁哈萨克自治州]]下辖的一个[[自治县|自治县]]。全国唯一的以[[锡伯族|锡伯族]]为主体的自治县。面积4430平方公里，人口15.9万。[[锡伯族|锡伯族]]為察布查尔縣的主體民族，另外還有大量其他民族。

“察布查尔”在[[锡伯语|锡伯语]]中是“粮仓”的意思。锡伯人最初游牧于[[大兴安岭|大兴安岭]]东麓，世代以狩猎、捕鱼为生。16世纪编入[[八旗|八旗]]后， 其社会组织发生了急剧变化， 生产上转入稳定的农业经济。18世纪中叶（[[乾隆|乾隆]]年间），清政府为巩固西北边防，将部分锡伯族迁往新疆，尔后这些锡伯族在[[伊犁河|伊犁河]]谷屯田定居，开拓了自己的第二故乡。在这风沙弥漫的大西北，勤劳的[[锡伯族|锡伯族]]人民凿山筑渠，修筑了长达100多公里的察布查尔大渠。潺潺的流水使荒漠的原野变成了阡陌纵横，树木葱绿，瓜果飘香，美丽富足的地方。东北地区的锡伯族已经失去了自己的民族语言、文字，淡化了自己的风俗习惯，而新疆查布查尔地区的[[锡伯族|锡伯族]]，至今还完整地保留着自己的语言文字（[[锡伯文|锡伯文]]）及浓厚的风俗习惯和宗教信仰（[[萨满教|萨满教]]和[[藏傳佛教|藏傳佛教]]），并经过与兄弟民族的长期交往，取长补短丰富了自己、发展了自己。錫伯族傳統上主要从事渔猎和农业。[[锡伯语|锡伯语]]属[[阿尔泰语系|阿尔泰语系]][[满－通古斯语族|满－通古斯语族]]满语支，跟[[满语|满语]]很接近。[[锡伯族|锡伯族]]除了使用[[锡伯语|锡伯语]]，還兼用[[汉语|汉语]]和[[哈萨克语|哈萨克语]]等多種語言。[[锡伯文|锡伯文]]是[[中華民國|民国]]36年（1947年）在[[满文|满文]]基础上改变而成的，一直沿用至今。 　　

== 历史 ==
伊犁河流域自古以来[[斯基泰人|塞種]]、[[烏孫|烏孫]]、[[突厥|突厥]]、[[突骑施|突骑施]]、[[蒙古族|蒙古]]等游牧民族统治。[[清朝|清朝]]1762年设立了[[伊犁將軍|总统伊犁等处将军]]后，调遣锡伯族官兵携眷移驻到伊犁地区。实际到达伊犁的人数为4030人。

== 地理 ==
位于新疆北部[[天山|天山]]支脉[[乌孙山|乌孙山]]以北、[[伊犁河|伊犁河]]以南的河谷盆地。南部与[[昭苏县|昭苏县]]、[[特克斯县|特克斯县]]相接，东部和[[巩留县|巩留县]]毗邻，北部隔伊犁河与自治州首府[[伊宁市|伊宁市]]和[[伊宁县|伊宁县]]、[[霍城县|霍城县]]相望。县城距伊宁市17公里。

境内有国家一级口岸——[[都拉塔口岸|都拉塔口岸]]。西与[[哈萨克斯坦|哈萨克斯坦]]接壤。

察布查尔属[[大陆性气候|大陆性气候]]，比较温和湿润，是年降水量比较多的地区之一。

== 民族 ==
这里居住着[[锡伯族|锡伯族]]、[[维吾尔族|维吾尔族]]、[[汉族|汉族]]、[[回族|回族]]、[[蒙古族|蒙古族]]、[[哈萨克族|哈萨克族]]等25个民族，人口15.9万。

全国锡伯族人口有19万人，其中10多万人分布在[[辽宁省|辽宁省]]、[[黑龙江省|黑龙江省]]、[[吉林省|吉林省]]、[[内蒙古自治区|内蒙古自治区]]和[[北京市|北京市]]等地；近4万多人在新疆，其中2万多人聚居在察布查尔锡伯自治县内。

== 文化 ==
自治县锡伯族中仍通用属于由[[满语|满语]]、[[满文|满文]]发展而来的[[锡伯语|锡伯语]]和[[锡伯文|锡伯文]]。

察布查尔素有[[射箭|射箭]]之乡美誉，1996年被国家命名为全国体育先进县。射箭和[[摔跤|摔跤]]是锡伯族人民传统的体育项目，几十年来向国家和自治区输送了大批的运动员。县上设有专业射箭队。

== 经济 ==
*自治县森林密布，以[[云杉|云杉]]为主，还有[[松柏目|松柏]]、[[桦木属|桦树]]、[[杨属|杨树]]、[[柳树|柳树]]、[[榆屬|榆树]]等。
*矿产主要有[[煤|煤]]、[[铁|铁]]、[[铜|铜]]、[[鈾|鈾]]、[[石灰石|石灰石]]、[[芒硝|芒硝]]、[[方解石|冰洲石]]、[[石英砂|石英砂]]等，[[石油|石油]]资源也在进一步勘探中。
*自治县经济以农业为主，农牧结合。
