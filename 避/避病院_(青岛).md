'''避病院'''又称'''青岛传染病院'''，旧址位于[[中国|中国]][[山东省|山东省]][[青岛市|青岛市]][[市南区|市南区]]嘉祥路3号，约建于1917-1918年，是青岛第一次日据时期（1914-1922）由日本人设计建造的一座医院建筑群，现为[[青岛中西医结合医院|青岛中西医结合医院]]（即青岛市第五人民医院）所使用，为[[青岛市文物保护单位|青岛市文物保护单位]] <ref Name=dao>{{cite book |author=袁宾久 |title=《青岛老建筑之旅》|year=2016 |publisher=中国海洋大学出版社 |location=青岛 |isbn=9787567010093}}</ref>。

==关于名称==
该建筑在[[青岛市文物保护单位|市级文保单位]]名单中的名称为“德国临时检疫所旧址”，原因是文物部门将建于日据时期的避病院主楼错误地认作德租时期建筑，而曾经存在过的德国临时检疫所没有任何遗迹留下<ref Name=dao/>。

“避病院”为[[日文|日文]]词汇，意为隔离医院、传染病医院。

==历史==
[[File:青岛德军高地营营门.jpg|thumb]]“Höhenlager”]]
[[File:青岛德军高地营.jpg|thumb]]
===清军广武前营及德国临时检疫所===
该址最初为[[清军|清军]]于章高元驻防青岛时期（1892-1897）所建设的广武前营，北临[[胶州湾|后海]]，位于[[台西_(青岛市)#西岭|西岭]]以北。营房布局与其他营房相似，平面呈矩形，以夯土围墙围成，石砌营门朝向西南，营内房屋为平房。

1897年[[德意志帝国海军|德军]]占领青岛后，将其命名为“高地营（Höhenlager）”，营房继续作为兵营使用。海军第三营三连（3. Kompagnie des III. Seebataillons）曾在此驻扎<ref>{{cite web |url=http://www.tsingtau.org/wp-content/uploads/2015/02/Adressbuch_1902.pdf|title=Adress-Buch des deutschen Kiautschou-Gebiets 1902|date=1902-09-08|work=|publisher=Verlag von Otto Rose, Tsingtau}}</ref>。随着[[俾斯麦兵营|俾斯麦兵营]]的建成及投入使用，高地营等原清军兵营逐渐不再作为营房使用。

1906年，为保证市内华人及[[青岛小港|小港]]内中国民船的卫生状况，防止疫情发生，德国当局在高地营内设立了临时检疫所（provisorische Quarantänestation）。检疫所内设有一台巨型蒸汽消毒锅炉和一台可移动式灭鼠机，部分房舍可安置可能患病的人员。该检疫所离主城区较远，易于对外封闭，民船上的人员可从紧靠检疫所北侧的[[青岛栈桥#青岛历史上出现的其他栈桥|阿克纳桥]]登陆到达此处。临时检疫所于次年扩建，并从[[德国|德国]]国内订购了防疫设备<ref>{{cite web |url=http://club.qingdaonews.com/showAnnounce.php?topic_id=2304102&board_id=57 |title=马维立关于“台西镇医院”的答复|date=2005-07-28|work=琴屿飘灯|publisher=}}</ref><ref>{{cite web |url=http://digital.staatsbibliothek-berlin.de/werkansicht?PPN=PPN770823904&PHYSID=PHYS_0055&DMDID=DMDLOG_0001 |title=胶澳租借地发展备忘录 1907-1908年度|date=1908-10|work=德意志帝国海军部|publisher=柏林国家图书馆}}</ref><ref>{{cite web |url=http://digital.staatsbibliothek-berlin.de/werkansicht?PPN=PPN77082420X&PHYSID=PHYS_0053&DMDID=DMDLOG_0001 |title=胶澳租借地发展备忘录 1908-1909年度|date=1909-10|work=德意志帝国海军部|publisher=柏林国家图书馆}}</ref>。

===传染病院===
[[File:青岛中西医结合医院_01.jpg|thumb]]
1914年，日军[[青岛战役|占领]]青岛。日本当局于1915年1月在高地营原临时检疫所设立[[胶澳督署医院|青岛病院]][[台西镇|台西镇]]分院，为一座设有100张床位的隔离医院。当局于1916年绘制城市规划图时，将设立避病院的计划列入第一期工事第17项建筑项目。避病院建筑具体的建设时间暂不可考，但可以推断为1917-1918年间，其设计师可能为日本建筑师[[三上贞|三上贞]] <ref Name=dao/>。

1922年12月，原避病院由[[胶澳商埠|胶澳商埠]]督办公署接管，改名为胶澳商埠传染病院。传染病院后划归山东省总督署卫队军医处，改称市民传染病院。1928年6月，军医处撤退，恢复胶澳商埠传染病院名称<ref Name=shi>{{cite web |url= http://lib.sdsqw.cn/bin/mse.exe?seachword=%u704F%u5FDB%u6F5A%u5B80&K=b2&A=57&rec=2&run=13 |title=青岛市志·卫生志·防疫|publisher=山东省情网}}</ref>。

1929年，胶澳传染病院改名为[[青岛特别市|青岛特别市]]市立传染病院。1931年，市立传染病院改为[[青岛市市立医院|市立医院]]第一分院，1933年7月又改称市立传染病院。1938年1月，传染病院再次被日军当局占用，由日本同仁会防疫处管理。1945年8月，[[日本投降|日本投降]]。同年12月，医院由青岛市卫生委员会接管<ref Name=shi/>。

1949年6月，解放军进驻青岛。1958年8月，青岛市海滨医院迁至传染病院址，并改名为青岛市台西医院。1983年7月，青岛市台西医院改为青岛市第五人民医院并加挂“青岛市中西医结合研究所”牌。1986年8月加挂“山东省青岛中西医结合医院“牌。1996年7月正式确定为“青岛市第五人民医院”和“山东省青岛中西医结合医院”。 

==建筑特色==
避病院旧址现由一栋单层主楼和两栋长条形平房组成。主楼正立面朝向东南方，为三段式划分，中央为三角形山墙，以[[花岗岩|花岗岩]]条石和直线纹样装饰，两翼分布有[[花岗岩|花岗岩]]装饰的双联长窗。坡屋顶以牛舌瓦覆盖，设有波浪形老虎窗<ref Name=dao/>。

屋顶的瓦片中出现了青岛祥利窑厂（Schangli & Co. Tsingtau）生产的带有[[中文|中文]]、[[德文|德文]]铭文的牛舌瓦。祥利窑厂为华商刘明卿于1913年3月投资2.6万元在[[湖岛村|湖岛村]]所开办<ref>{{cite web |url=http://club.qingdaonews.com/showAnnounce_57_2165929_1_1.htm|title=嘉祥路3号的考察（图文并茂版）|date=2005-06-27|work=|publisher=}}</ref>。部分人据此认为该建筑为德租时期所建设，但事实上部分德租时期生产的瓦片直至1920年代仍在工程中使用<ref Name=dao/>。
