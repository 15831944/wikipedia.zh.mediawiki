'''避讳'''是[[中国歷史|中国歷史]]上，必須回避[[君主|君主]]、尊長的「[[名諱|名諱]]」<ref>[[臺灣|臺灣]].[[中華民國教育部|教育部]].國語辭典修訂本.【名諱】「尊長或所尊敬的人的名字。」</ref>的一种要求，通常只限於[[君主|君主]]、尊長之[[本名|本名]]。[[表字|字]][[號|號]]的避讳则较少见<ref>如[[唐玄宗|唐玄宗]]兄长[[李憲_(寧王)|李成器]]避其母[[昭成皇后|昭成皇后]][[尊号|尊号]]，[[改名|改名]]李宪。</ref>。在言谈和书写时，遇到君主尊長的名諱一律要回避，可以用其他字代換，或是刻意將該字缺筆；取名时，不能取他们的名諱中字甚或同音的字；否則，可能觸犯[[大不敬|大不敬]]之罪。[[陆容|陆容]]《菽园杂记》稱，“民间俗讳各处有之，[[吴中|吴中]]为甚”。這種要求也曾經適用於[[漢字文化圈|漢字文化圈]]内中国的臨近國家，例如[[日本|日本]]、[[朝鮮半島|朝鮮半島]]、[[越南|越南]]、[[琉球國|琉球]]。

== 避諱的起源 ==
避讳始於[[周朝|周朝]]<ref>张世南《游宦纪闻》卷三说：“殷人以讳事神，而后有字。”钱希言在《戏瑕》中稱避讳之俗於夏商时已有，证据是《山海经》中往往把夏后启写作夏后开。但陈光坚《讳源略说》一文以为此說证据不足。洪迈《容斋三笔·帝王讳名》提出，避讳之制始于周代，但仅在本庙中避讳。赵翼《陔余丛考·避讳》主张避讳习俗始于东周，即春秋时期。陈光坚《讳源略说》亦稱避讳起自春秋时期。</ref>，《[[左传|左传]]》说：“周人以讳事神，名，终将讳之。”<ref>《左传·桓公六年》-{云}-：“周人以讳事神名，终将讳之。”孔颖达疏：“自殷以往，未有讳法。讳始于周。”</ref>《[[礼记|礼记]]·曲礼》载：“名子者不以国，不以日月，不以隐疾，不以山川。”明文规定取名之避。后来，《[[左传|左传]]》又加上“不以畜牲，不以器帛”这一条款，遂产生“六避”。<ref>《避讳研究》，王新华 著，齐鲁书社</ref>但当时同音或近音的字不用回避。名諱兩字中，只有一个字相同，也不用回避。

== 避諱的種類 ==
* 回避[[君主|君主]]的名字，叫做避'''君讳'''、'''公讳'''或'''国讳'''。一般適用於過去七世代以内。
* 避讳自己家族的长辈的名字，与别人交往时则避对方的长辈之名讳，叫做避'''家讳'''、'''私讳'''。
* 回避[[聖人|聖人]]的名字，叫做避'''聖人諱'''。例如[[邱姓|邱姓]]就因避諱[[孔子|孔子]]的名字「丘」而產生；又例如《[[三国演义|三国演义]]》中称[[关羽|关羽]]为「关公」、「关某」；《[[东周列国志|东周列国志]]》中称孔子为「孔某」。另古人讀經史時，碰到孔子的名字「丘」時，會將之讀作「某」，以示避諱，就連寫的時候也會缺筆。<ref>{{cite web |url=http://win2003.chi.cuhk.edu.hk/bowen/week_topic/Weekly_Topic-View.asp?id=213 |title=存档副本 |accessdate=2013-05-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20080315213308/http://win2003.chi.cuhk.edu.hk/bowen/week_topic/Weekly_Topic-View.asp?id=213 |archivedate=2008-03-15 }}</ref><ref>http://dict.variants.moe.edu.tw/yitia/lda/lda00015.htm</ref>

== 避諱的方法與實例 ==
[[陳垣|陳垣]]《史諱舉例》曰：「避諱常用之法有三：曰'''改字'''，曰'''空字'''，曰'''缺筆'''。」到了[[宋朝|宋朝]]，避諱之風日趨嚴格，又出現「'''[[拆字|拆字]]'''」這一種避諱方法。

=== 改字 ===
改字法：遇到要避諱的字時使用其他別字代替。例如：
# 避[[秦始皇|秦始皇]]嬴政諱，“政月”改“正月”，「正」字改唸平聲，甚至改称“端月”。
# 避[[劉邦|漢高祖]]劉邦諱，“相邦”改“相國”。
# 避[[漢文帝|漢文帝]]劉恆音諱，“姮娥”改[[嫦娥|嫦娥]]。
# 避[[漢景帝|漢景帝]]劉啟諱，“啟蟄”改[[驚蟄|驚蟄]]、“[[启封县|启封县]]”改[[开封县|开封县]]。
# 避[[漢武帝|漢武帝]]劉徹諱，“[[徹侯|徹侯]]”改通侯。
# 避[[漢昭帝|漢昭帝]]劉弗陵諱，否定詞“弗”改“不”。
# 避[[漢宣帝|漢宣帝]]劉詢音諱，“[[荀卿|荀卿]]”改孫卿。
# 避[[劉秀|漢光武帝]]劉秀諱，东汉[[孝廉|孝廉]]“[[秀才|秀才]]”改“[[茂才|茂才]]”。
# 避[[漢明帝|漢明帝]]劉莊讳，[[莊姓|莊姓]]者多半改成[[嚴姓|嚴姓]]。
# 避[[漢桓帝|漢桓帝]]劉志讳，《[[東觀漢記|東觀漢記]]》的“志”改稱“意”。
# 避[[司馬懿|晉宣帝]]司馬懿（追尊）諱，[[三國志|三國志]]將[[吳懿|吳懿]]改“吳壹”。
# 避[[司馬昭|晉文帝]]司馬昭（追尊）諱，[[王昭君|王昭君]]改稱王明君，[[蔡昭姬|蔡昭姬]]改稱蔡文姬，[[韋昭|韋昭]]改稱[[韋曜|韋曜]]。
# 避[[晉愍帝|晉愍帝]]司馬鄴音諱，建業（今[[南京市|南京市]]）改名為建康。
#避[[唐高祖|唐高祖]]李淵祖父[[李虎|李虎]]諱，漢朝以來稱作「虎子」的[[馬桶|馬桶]]改稱為「馬子」。<ref>{{Cite news|url=https://www.xuehua.us/2018/03/20/%E5%94%90%E6%9C%9D%E9%82%A3%E4%BA%9B%E6%9C%89%E6%84%8F%E6%80%9D%E7%9A%84%E7%A6%81%E5%BF%8C%EF%BC%8C%E7%94%9A%E8%87%B3%E5%BD%B1%E5%93%8D%E4%BA%86%E9%A9%AC%E6%A1%B6%E7%9A%84%E7%A7%B0%E5%91%BC/zh-tw/|title=唐朝那些有意思的禁忌，甚至影響了馬桶的稱呼 - 雪花新聞|date=2018-03-20|newspaper=雪花新聞|accessdate=2018-08-16|language=zh-CN}}</ref>
# 避[[唐高祖|唐高祖]]李淵諱，[[陶淵明|陶淵明]]改稱陶泉明或陶深明。
# 避[[唐太宗|唐太宗]]李世民讳，「世」、「民」常改為「代」、「人」。例如：[[董襲|董襲]]，字元世改為元代、[[柳宗元|柳宗元]]《[[捕蛇者说|捕蛇者说]]》中句子“以俟夫观人风者得焉”中的“人风”实际上是指“民风”，是为此而改。
# 避[[唐高宗|唐高宗]]李治諱，文章內“治”字改為“理”。
# 避[[武則天|武則天]]武曌諱，廬陵王（稍後的[[唐中宗|唐中宗]]）之長子李重照改名[[李重润|李重润]]。（曌為照的[[则天文字|则天文字]]）
# 避[[石敬瑭|晉高祖]]石敬瑭諱，有些[[敬姓|敬姓]]人士改为[[端姓|端姓]]，或拆字變為[[苟姓|苟姓]]以及[[文姓|文姓]]。
# 避[[宋太祖|宋太祖]]趙匡胤諱，[[宋太宗|宋太宗]]趙匡義改名为趙光義。
# [[孔子|孔子]]之名丘曾經是[[金朝|金朝]]的'''聖人諱'''，所以有些[[丘姓|丘姓]]人士改为[[邱姓|邱姓]]。
# 「元年」改為「始年」、「原年」，「元來」改為「原來」，非避[[明太祖|明太祖]]朱元璋諱，明代所有名諱皆依古禮：「二名不偏諱，嫌名不諱」。而是避元朝之故。<ref>清·[[郝懿行|郝懿行]]《晉宋書故·元由》：「元，始初也；由，萌蘗也。論事所起，或言元起，或言元來，或言元故，或言元舊，皆是也。今人為書，元俱作原字……蓋起於前明初造，事涉元朝，文字簿書率皆易『元』為原。」</ref>蓋民間追恨元人，不欲書其國號也。<ref>明沈德符《萬曆野獲編》：「俱以原字代元字，蓋民間追恨元人，不欲書其國號也。」</ref>

# 避[[康熙帝|清聖祖]]玄燁讳，[[玄武門|玄武門]]改[[神武門|神武門]]，而“玄孫”改“元孫”。
# 避[[雍正帝|清世宗]]胤禛諱，兄弟[[字輩|字輩]]“胤”改“允”。趙匡胤通稱為「宋祖」甚至改稱趙匡續，王士禛改稱王士正或[[王士禎|王士禎]]。[[真定府|真定府]]改为[[正定府|正定府]]。
# 避清[[乾隆帝|清高宗]]弘曆諱，-{曆}-常改為-{歷}-，而大清曆法[[時憲曆|時憲曆]]改時憲書，[[弘文館|弘文館]]改「宏文館」。
# 避清[[道光帝|清宣宗]]旻寧讳，[[北京|北京]]外城廣寧門改名[[廣安門|廣安門]]。

* 史上亦有皇帝即位時，將名字易為罕用字，使擾民程度降低。例：
# [[漢宣帝|漢宣帝]]劉詢，原名病已。
# [[嘉慶帝|清仁宗]]顒琰（Yóngyǎn, ㄩㄥˊ ㄧㄢˇ），1796年即位前原名永琰。
# [[道光帝|清宣宗]]旻寧，1820年即位前原名綿寧。

=== 空字 ===
空字法：遇到要避諱的字時不填寫。如[[唐朝|唐朝]][[將領|將領]][[李世勣|李世勣]]被改成[[李勣|李勣]]，以避[[唐太宗|唐太宗]]李世民的讳。但最著名的例子是“[[觀世音|觀世音]][[菩薩|菩薩]]”亦因避唐太宗諱，略作“[[觀音|觀音]]”，但這個說法仍待商榷。

=== 缺筆 ===
[[File:Omitting_a_stroke.png|thumb]]
陳垣<ref name="xintangshu">陳垣：《史諱舉例》，（北京：北京科學出版社，1958年），頁5。</ref>認為避諱缺筆，始於[[唐朝|唐朝]]，[[貞觀|貞觀]]十八年蓋的[[文達碑|文達碑]]，有「世子」二字，時[[唐太宗|唐太宗]]稱[[李世民|李世民]]，可見仍未有'''避諱缺筆'''。[[乾封|乾封]]元年《[[賜泰師孔宣公碑|賜泰師孔宣公碑]]》中，「愚智齊泯」中的「泯」作「汦」，因此他認為[[唐高宗|唐高宗]]是避諱缺筆之始。到了[[高宗|高宗]][[顯慶|顯慶]]五年正月詔曰：「至於朕名，或缺其點畫。」說明了當時已經有缺筆避諱。

一般缺筆是缺寫尾一筆。如「愚智泯」的「泯」字，因諱[[唐太宗|唐太宗]][[李世民|李世民]]，而作「汦」。《于志宁碑》中「世武」把唐太宗的名字李世民中的「世」少寫一劃，作「卅武」。[[舊唐書|舊唐書]][[唐穆宗|穆宗]]紀：「恒王房子孫改為汦王房。」汦王應為愍王，因避諱，改民為氏而誤。《[[續漢郡國志|續漢郡國志]]》中載[[敦煌|敦煌]]郡有拼泉，拼泉即淵泉，因[[唐太祖|唐太祖]][[李淵|李淵]]缺筆的淵字缺尾的一筆，後訛為拼。

宋人避[[宋聖祖|宋聖祖]][[趙玄朗|趙玄朗]]諱，「朗」改為「郎」。[[劉焉|劉焉]]，字君朗改為君郎、《[[新唐書|新唐書]]·后妃則天皇傳》：「前鋒左豹韜果毅[[成三朗|成三郎]]為[[唐之奇|唐之奇]]所殺。」

《[[康熙字典|康熙字典]]》將康熙帝玄燁諱各缺一筆。

日本[[明治|明治]]時代，對於[[仁孝天皇|仁孝天皇]]（惠仁）、[[孝明天皇|孝明天皇]]（統仁）、[[明治天皇|明治天皇]]（睦仁）的名諱「惠」、「統」、「睦」缺一筆。

=== 拆字 ===
拆字法：當遇到要避諱的字時，將該字拆開，以描述其字的構造。
# [[宋太宗|宋太宗]]諱趙炅，又名趙耿；「炅」、「耿」二字皆為國諱。宋朝人將「耿」字寫作「從耳從火」。<ref name="rongzhai"/>
# [[宋真宗|宋真宗]]諱趙恒，「恒」為國諱，遂將「恒」寫作「從心從亘」。<ref name="rongzhai">《容齋三筆·卷十一·帝王諱名》：「太宗諱字內有從耳從火者，又有梗音，今為人姓如故。高宗諱從勹從口者亦然。真宗諱從心從亘，音胡登切。」</ref>
# [[宋高宗|宋高宗]]諱趙構，「構」字甚至其音相近的二十餘字皆為國諱。時人為避諱，將與其音相近的「句」寫作「從勹從口」。<ref name="rongzhai"/>

==各地情況==
===中國===
周朝時避諱情況較為寬鬆，只有完全相同的字才要避諱，同音或近音字大致上不用。秦汉时期，大一统的政局形成并得到巩固，君主之尊无以复加，儒学在上层建筑中逐渐占统治优势，这种要求遂变得非常严格。[[地名|地名]]、物名、[[官名|官名]]、[[人名|人名]]都要回避當朝[[天子|天子]]甚至[[皇帝|皇帝]][[祖先|祖先]]的名字。[[郑玄|郑玄]]《[[毛诗|毛诗]]笺注》“天下邦国，至于困穷”言及[[汉高祖|汉高祖]]（刘邦）名讳而并未受到查处，可见汉朝时对避讳的规制不如清代时严格。[[司马迁|司马迁]]《史记·[[张仪|张仪]]列传》“是故天下之游谈士，莫不日夜搤腕”言及其父（[[司马谈|司马谈]]）名讳，但這亦有可能是司馬遷抄錄其父司馬談留下的資料，司馬談自然不需要避諱自己了。

[[东晋|东晋]][[太元_(东晋)|太元]]十三年（388年），侍中[[孔安国_(东晋)|孔安国]][[上表|上表]]，因[[黄门郎|黄门郎]]王愉名犯其父[[孔愉|孔愉]]名讳，不能共同签名，求解。[[有司|有司]]认为：此前，尚书、安众男某某曾上表中兵曹郎王祐犯其父私讳。同意调换部门，这是朝廷制度之外的恩典。但此例一开，官员效仿，不知何时结束。于是断绝了此类请求<ref>《[[晋书|晋书]]·志第十·礼中》太元十三年，召孔安国为侍中。安国表以黄门郎王愉名犯私讳，不得连署，求解。有司议云：“名终讳之，有心所同，闻名心瞿，亦明前诰。而《礼》复云‘君所无私讳，大夫之所有公讳’，无私讳。又云‘诗书不讳，临文不讳’。岂非公义夺私情，王制屈家礼哉！尚书安众男臣先表中兵曹郎王祐名犯父讳，求解职，明诏爰发，听许换曹，盖是恩出制外耳。而顷者互相瞻式，源流既启，莫知其极。夫皇朝礼大，百僚备职，编官列署，动相经涉。若以私讳，人遂其心，则移官易职，迁流莫已，既违典法，有亏政体。请一断之。”从之。</ref>。但避讳却是日渐严格。避各类私讳，亦是后世官员注重的事项<ref>《[[册府元龟|册府元龟]]·卷八百六十三》◎总录部·名讳</ref>。

到了[[唐朝|唐朝]]，有一个字相同甚至是同音，也必须回避了。当时著名诗人[[李贺|李贺]]就是因为其父名“晋肃”中，而不得参加[[进士|进士]]考试，致使终身不得志；[[韓愈|韓愈]]為此事非常生氣，为之作《[[s:諱辯|讳辩]]》驳斥，“父名晋肃，子不得举进士；若父名仁，子不能为人乎！”[[唐律疏议|唐朝法律]]规定，直呼皇帝的名字犯“[[大不敬|大不敬]]”罪，不能赦免。然而[[杜牧|杜牧]]《[[阿房宫赋|阿房宫赋]]》“一旦不能有”、[[柳宗元|柳宗元]]《[[捕蛇者说|捕蛇者说]]》“岂若吾乡邻之旦旦有是哉”皆言及[[唐睿宗|唐睿宗]]（李旦）名讳，[[元稹|元稹]]《连昌宫词》“舞榭欹倾基尚在”又言及[[唐玄宗|唐玄宗]]（李隆基）名讳而并未受到查处，可见唐朝时对避讳的规制亦不如清代时严格。

除了文字避諱，生活上亦有其他形式的名諱。例如[[唐朝|唐朝]]避諱李姓，禁食[[鯉魚|鯉魚]]，抓到鯉魚必須放掉，販賣鯉魚責打60杖。<ref>[[s:酉陽雜俎/卷十七|s:酉陽雜俎/卷十七]]</ref>惟此類規定有時流於形式。另明朝人不禁吃豬肉。<ref>{{Cite web|url=https://zi.media/@yidianzixun/post/q7Fosz|title=唐朝人為啥不能吃鯉魚？那明朝人能吃豬肉嗎？-字媒體 ZiMedia|accessdate=2018-08-16|work=字媒體 ZiMedia|language=zh-Hant}}</ref>

據稱[[五代十國|五代十國]]時[[馮道|馮道]]喜人誦讀《[[道德經|老子]]》，而誦書的[[侍從|侍從]]為了避馮道的名諱，而將「道」這個字給讀成「不可說」，於是他便將《老子》的開頭給讀成了「不可說可不可說，非常不可說」，結果大家都覺得很好笑。<ref>[http://wwwold.hfu.edu.tw/~lbc/BC/5TH/BC0520.HTM 論禪之"不可言說"]</ref>

宋代，讳法大兴，一个帝王的避讳字数量急剧增多，《容齋三筆·十一》云：「本朝尚文之習大盛，故禮官討論，每欲其多，廟諱有五十字者。舉場試卷，小涉疑似，士人輒不敢用，一或犯之，往往暗行黜落。」

元代是蒙古人入主中原，蒙古人本无避讳之事，因此元代不太在意避諱。忽必烈行汉法之后，曾下令避讳，但终元一代讳法不严，一部《元史》，惟《[[程鉅夫|程鉅夫]]列傳》言：「鉅夫名文海，避武宗廟諱，以字行。」餘無所見<ref>[http://140.111.1.40/fulu/fu8/page8.htm 唐以後帝王避諱名-{}-表：元]</ref>，程钜夫生于宋，故习惯避讳，而考察《元史·武宗纪》，武宗朝名中有“海”者多达七人，武宗朝奏章地名有“海”字者众多，并不避讳，故程钜夫避讳一事属于此人特例<ref>《史讳举例》陈垣</ref>。蒙古人中君臣同名者甚多。

明代因之，二字之名在用到其中某一字时不避讳，音近而有称名之嫌，亦不需避諱；所謂「二名不偏讳，嫌名不讳」，故親王可與皇帝共用[[行輩|字輩]]。

满人早期亦不避讳，后来汉化加深，从玄烨开始<ref>陳垣《史諱舉例》：清之避諱，自康熙帝之漢名始。雍乾之世，避諱至嚴，當時文字獄中，至以詩文筆記之對於廟諱御名有無敬避，為順逆憑證。</ref>，才跟着汉人那样有了避讳之事。清乾隆二十二年（1757年），退休的[[布政使|布政使]][[彭家屏|彭家屏]]在族譜《大彭統記》刊登[[河南|河南]][[生員|生員]][[段昌緒|段昌緒]]家存[[吳三桂|吳三桂]]文告時，[[乾隆帝|乾隆帝]]之名諱弘曆不缺筆，成為彭家屏[[自殺|自殺]]及段昌緒[[斬首|斬首]]罪名之一。乾隆四十二年（1777年），江西書生[[王錫侯|王錫侯]]在他刪[[康熙字典|康熙字典]]寫的字典《[[字貫|字貫]]》凡例寫入[[康熙帝|康熙帝]]、[[雍正帝|雍正帝]]、乾隆帝之名諱（玄燁、胤禛、弘曆），沒有缺筆避諱，也有批評康熙字典“然而穿貫之難”的内容，被認為非常不敬，而被依大逆律定罪處斬。<ref>《清代名人傳略》中冊頁143</ref>清朝「末代皇帝」[[溥儀|溥儀]]在位時，“儀”曾經是避諱字。<ref>[http://140.111.1.40/fulu/fu8/page10.htm 唐以後帝王避諱名-{}-表：清]</ref>

[[清朝滅亡|清朝滅亡]]後，中國不再避國諱和圣人讳{{来源请求|reason=上述形式可能以敏感词的方式在中华人民共和国名亡实存|time=2018-9-19}}。现时一般中国人仍避家讳，但仅限于起名，日常使用是不避的，亦有连起名也不避讳的。

=== 日本 ===
日本避諱的相關記錄始見於[[奈良時代|奈良時代]]。[[延曆|延曆]]年間，[[桓武天皇|桓武天皇]]為避其父[[光仁天皇|光仁天皇]]（諱白壁王）的諱，將[[常陸國|常陸國]]的「白壁郡」改名為「[[真壁郡|真壁郡]]」。[[淳和天皇|淳和天皇]]即位的時候，由於天皇名叫「大伴」，氏族「[[大伴氏|大伴氏]]」便被改名為「伴氏」。

進入[[幕府|幕府]]時期以後，[[日本武士|日本武士]]對主君的名字也要避諱。當時[[日本人名|日本人名]]分為「通字」和「偏諱」兩個部份。例如在「[[德川家康|德川家康]]」的名字中，「家」字為通字，由家族時代相傳；「康」字為偏諱。通字無需避諱，但偏諱要避。在[[大坂之役|大坂之役]]中，[[豐臣秀賴|豐臣秀賴]]沒有避德川家康的偏諱，為德川家康出兵大坂提供了口實。不過有時候，日本武士也會將偏諱賜給屬下，以示恩寵。

日本直到[[二戰|二戰]]結束之前依然要避諱，否則是對皇室的[[不敬|不敬]]。例如，為避[[仁孝天皇|仁孝天皇]]（惠仁）、[[孝明天皇|孝明天皇]]（統仁）、[[明治天皇|明治天皇]]（睦仁）的諱，將「惠」、「統」、「睦」三字都缺一筆。此外，對皇后也要避諱。例如，[[大正天皇|大正天皇]]的皇后名叫[[貞明皇后|九條節子]]（{{jpn|j=九条 節子|hg=くじょう さだこ|rm=Kujō Sadako}}），[[秩父宮雍仁親王|秩父宮雍仁親王]]的妃子叫[[松平勢津子|松平節子]]（{{jpn|j=松平 節子|hg=まつだいら せつこ|rm=Matsudaira Setsuko}}）；九條節子成為皇后之後，松平節子為了避諱，將名字改為同音的「勢津子」（{{jpn|j=勢津子|hg=せつこ|rm=Setsuko}}）。

1947年（昭和22年）10月26日，日本國憲法廢除了對皇室不敬的罪名，不再將避諱列入法律條文之內。

=== 琉球 ===
受中國文化的影響，琉球王府也要求避國王的諱。其避諱的方式主要為改字。例如，為避[[尚哲王|尚哲王]]諱，「哲」字改為「明」；<ref>[http://www.tulips.tsukuba.ac.jp/limedio/dlam/B1241188/1/vol05/kume/716.htm 《毛氏家譜（與世山家）·五世維基》]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}：「次男宣哲（哲字因國禁改明）」</ref>避[[尚溫王|尚溫王]]諱，「溫」字改為「和」；<ref>[http://www.tulips.tsukuba.ac.jp/limedio/dlam/B1241188/1/vol05/kume/420.htm 《孫氏家譜（安座間家）·四世有温》]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}：「四世有温（温字因國禁改謂有和）」</ref>避[[尚泰王|尚泰王]]諱，「泰」字改為「春」。<ref>[http://www.tulips.tsukuba.ac.jp/limedio/dlam/B1241188/1/vol05/naha/548.htm 《新参林姓家譜（佐久川家）·新参四世泰雄》]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}：「新参四世泰雄（因泰之字國禁改春）」</ref>此外，琉球人也要避中國皇帝的諱。<ref>[http://www.tulips.tsukuba.ac.jp/limedio/dlam/B1241188/1/vol05/kume/495.htm 《陳姓家譜（仲本家）·十二世》]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}：「十二世諱弘澤（弘字因禁止改宏）」【註：乾隆帝諱弘曆。】</ref><ref>[http://www.tulips.tsukuba.ac.jp/limedio/dlam/B1241188/1/vol05/kume/391.htm 《曾姓家譜（中宗根家）·六世紫金大夫夔》]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}：「男曆（曆之字因禁止改信）」【註：乾隆帝諱弘曆。】</ref>

琉球人要避國王的諱，但不必避國王使用的[[名乘|名乘]]。例如，尚溫王的名乘為「朝克」，<ref name="king">《[http://manwe.lib.u-ryukyu.ac.jp/cgi-bin/disp-img.cgi?page=49&file=iha0210 王代記]{{dead link|date=十月 2017 |bot=InternetArchiveBot }}》，第49頁</ref>但後世仍有一名[[三司官|三司官]]名叫「[[馬克承|馬克承]]」；馬克承的名乘為「小祿親方良泰」，尚泰王繼位後，為避諱，改為「良忠」。<ref>[http://www.tulips.tsukuba.ac.jp/limedio/dlam/B1241188/1/vol05/syuri/534.htm 《馬姓家譜·小祿家·十二世馬克承》]{{dead link|date=十月 2017 |bot=InternetArchiveBot }}：「十二世馬克承童名思龜名乘良泰（因泰之字國禁改忠）」</ref>

1609年[[慶長琉球之役|慶長琉球之役]]之後，琉球成為日本[[薩摩藩|薩摩藩]]的附庸國，薩摩藩在琉球設置[[在番奉行所|在番奉行所]]以監督內政。1755年，[[島津重豪|島津重豪]]就任薩摩藩藩主之後，薩摩藩曾一度要求琉球王府下令避[[江戶幕府|江戶幕府]]將軍的諱。例如，[[毛宣猷|毛宣猷]]為避[[德川家宣|德川家宣]]的諱，改名為[[毛廷柱|毛廷柱]]；<ref>[http://www.tulips.tsukuba.ac.jp/limedio/dlam/B1241188/1/vol05/kume/721.htm 《毛氏家譜（與世山家）·六世宣猷》]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}：「六世宣猷（宣字因國禁改廷柱）」</ref>[[向宣謨|向宣謨]]的名乘原為「今歸仁朝忠」，為避[[德川秀忠|德川秀忠]]諱，改為「今歸仁朝義」。<ref>[http://www.tulips.tsukuba.ac.jp/limedio/dlam/B1241188/1/vol05/syuri/265.htm 《向姓家譜（具志川家）·十世諱宣謨》]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}：「十世諱宣謨 今歸仁王子 童名思德金 名乘朝忠（忠之字因禁止改義）」</ref>薩摩藩又禁止琉球人使用「德」、「千代」、「雄」、「敦」、「信」、「明」、「重」等字作為名字。不過在島津重豪死後，琉球便廢除了這個命令。為避諱而更改[[琉球人名|名乘頭]]的家族，多獲准恢復使用原先的名乘頭；<ref>[http://www.tulips.tsukuba.ac.jp/limedio/dlam/B1241188/1/vol05/naha/468.htm 《平姓家譜（當間家）·七世重富》]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}：「七世重富（重之字因禁止改季因免許如今）」</ref>但仍有家族的名乘頭未被改回。<ref>[http://www.tulips.tsukuba.ac.jp/limedio/dlam/B1241188/1/vol05/syuri/45.htm 《英姓家譜（渡名喜家）·六世恒隆》]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}：「六世恒隆（原是名乘頭重之字因御禁止子孫悉改用恒之字）」</ref>

=== 朝鮮半島 ===
朝鮮半島的避諱可能始於[[統一新羅|統一新羅]]時代。在《[[文武王陵碑|文武王陵碑]]》的碑文里，「[[丙午|丙午]]」被寫作「景午」。[[唐高祖|唐高祖]]之父名叫[[李昞|李昞]]，「昞」字和同音的「丙」字都是需避諱的字，故而[[干支|干支]]紀年中將丙字改作「景」。但是，新羅當時剛剛從[[唐朝|中國]]引進干支紀年，沒有證據顯示新羅是否也從中國引入了避諱制度。

不過在[[高麗王朝|高麗王朝]]時期的法律明確規定需要避高麗國王的諱。高麗時代的避諱方法主要為缺筆，故而在《[[高麗大藏經|高麗大藏經]]》中可以看見不少缺畫字。<ref>[http://www.memorykorea.go.kr:7779/teukjing/teukjing1_02.html 특수표식: 피휘, 각필, 불심지]</ref>此外也有改字的方法，例如[[高麗惠宗|高麗惠宗]]名叫王武，便將「文武兩班」改稱「文虎兩班」，<ref>見[[鳳巖寺|鳳巖寺]]《靜眞大師塔碑文》</ref>後又改稱「龍虎兩班」。又如，《[[三國遺事|三國遺事]]》中，為避[[高麗定宗|高麗定宗]]王堯諱，將「堯」字改為「高」。

1750年，為了避免在[[文廟|文廟]]祭祀中衝犯孔子的諱，[[朝鮮英祖|朝鮮英祖]]下令將「大丘郡」改名為「[[大邱廣域市|大邱郡]]」。<ref>[http://sillok.history.go.kr/inspection/insp_king.jsp?id=wua_12612002_002&tabid=w&mTree=0&inResult=0&indextype=1 《朝鮮王朝實錄·英祖實錄·二十六年十二月》]</ref>

[[興宣大院君|興宣大院君]]重建[[景福宮|景福宮]][[弘禮門|弘禮門]]時，為避清高宗弘曆諱，將「弘禮門」改名為「興禮門」。

=== 越南 ===
[[File:Ti_huy_nha_nguyen.PNG|thumb]]》中的避諱舉例]]
越南的避諱，除了改字、空字、缺筆、拆字之外，'''變音'''是其一大特色。

越南的避諱，最早見於[[陳朝_(越南)|陳朝]]時期。1232年（[[建中|建中]]八年），[[陳太宗|陳太宗]]下達避諱的命令。為了避陳朝皇帝祖先[[陳李|陳李]]的諱，陳太宗下令將全國的[[李姓|李氏]]全部改為[[阮姓|阮氏]]。

在越南，中國的避諱方法改字、空字、缺筆、拆字都有。例如，在[[成泰|成泰]]年間刊行的《[[大南實錄|大南實錄]]》的「大南正編列傳初集」之中，為避[[嗣德帝|嗣德帝]]阮福時諱，改「時」字為「辰」字；為避[[成泰帝|成泰帝]]阮福昭諱，便將「昭」字缺筆寫作「召」。在《大南實錄》的「前編」中，歷代阮主的名諱皆空字。<ref>《[http://lib.nomfoundation.org/collection/1/volume/179/page/25 大南實錄前編·凡例]》：「地名遇廟諱、尊字，或從今名，或缺一筆。至於人名，皆改別字，以示敬也。」</ref><ref>[http://lib.nomfoundation.org/collection/1/volume/162/page/11 《大南正編列傳初集·偽西列傳》]</ref><ref>《[http://lib.nomfoundation.org/collection/1/volume/179/page/36 大南實錄前編·卷一]》</ref>在《[[國朝正編撮要|國朝正編撮要]]》中，為避[[嘉隆帝|嘉隆帝]]阮福-{暎}-諱，寫作「左從日右從英」。<ref>《[http://lib.nomfoundation.org/collection/1/volume/336/page/1 國朝正編撮要·卷之一]》：「世祖高皇帝，諱左從日右從爰，又諱左從日右從英，又諱左從禾右從重。」</ref>

還有一種被稱為變音的避諱方法，即將皇帝名諱的讀音改變，與需避諱字的同音字也隨之變音。變音這種避諱方法為越南特色，始見於[[後黎朝|後黎朝]]時代。[[黎太祖|黎太祖]]諱黎利，「利」字在越南語中本讀作「lì」，為避諱，改為讀「lợi」。[[阮朝|阮朝]]時代，為避歷代皇帝的[[越南人名#垫名|垫名]]「福」之諱，將「福」字的發音從「phúc」改為「phước」，所有讀作「phúc」的字也統統改為讀「phước」。

此外，也存在口頭和書面上不同的避諱方法。例如[[嗣德帝|嗣德帝]]名「阮福時」，在書面上若遇上「時」字，應將「時」字改寫作「辰」字；而在口頭上，「時」字本應讀作「thì」，為避諱改讀「thời」。

在[[鄭阮紛爭|鄭阮紛爭]]時期，[[阮主|廣南阮主]]轄境內的居民必須避阮主之諱。例如，為避[[阮潢|阮潢]]諱，將「hoàng」讀作「huỳnh」。時至今日，越南南部的[[黃姓|黃姓]]居民仍將自己的姓氏讀作「huỳnh」，但北部居民仍讀作「hoàng」。

阮朝[[明命帝|明命帝]]時代，將眾多的阮主支系子孫删去姓氏，冠以「宗室」（{{vie|hn=宗室|qn=Tông Thất}}）二字，标榜其地位。後來[[紹治帝|紹治帝]]繼位，為避紹治帝阮福綿宗之諱，又將「宗室」改姓為近音的「尊室」（{{vie|hn=尊室|qn=Tôn Thất}}），這便是今日越南[[尊室|尊室姓]]的由來。

== 現代 ==
现在虽然已经不讲究这些避讳習俗了，但是仍然有部分人在取名时尽量不取同父辈或祖辈同名或同音的字作为名字。由于避讳的影响，现在的東亞人还是认为直呼长辈的名字为不敬，所以會特別-{注}-意親屬稱謂。

== 例外情況 ==
雖然不取與父輩或祖輩名字相同的用字或同音字是傳統避諱方式之一，但亦有例外。例如[[東晉|東晉]]時[[王羲之|王羲之]]的七個兒子[[王玄之|王玄之]]、[[王凝之|王凝之]]、[[王渙之|王渙之]]、[[王肅之|王肅之]]、[[王徽之|王徽之]]、[[王操之|王操之]]、[[王獻之|王獻之]]都有「之」字，据[[陈寅恪|陈寅恪]]考证“之”字是[[天师教|天师教]]信徒的标志；而汪婕指出，这仅仅是“二名不偏讳”，与天师教信仰无关<ref>汪婕 《对魏晋南北朝"父子不嫌同名"怪现象之思考》 井冈山大学学报·社会科学版 2009第6期</ref>。

==避讳的后果及用途==
避讳徒增文章的理解难度，若不了解相应的避讳情况，可能无法正确认出原词，导致文章难以理解。

另外，避讳本身提供了一种文献断代方法，如果某书中避某字讳，则可以断定该书是相对应的某位皇帝继位后著述的。

==评价==
现代，避讳被认为是强化君主专制的手段而遭到批判。

==漢字文化圈以外==
===猶太人===
{{See also|四字神名}}

由於《[[希伯来聖經|希伯来聖經]]》中的《[[十誡|十誡]]》有「不可妄稱[[四字神名|YHWH]]你神的名」的誡命，犹太教文獻中的四字神名（{{lang-he|יהוה}}）不标[[母音|母音]]，或标上"adonai"的母音作为提醒，而只写4個[[子音|子音字母]]，一般稱為“4字神名”——英語的'''YHWH'''、法語和德語的'''JHWH'''、拉丁語的'''IHVH'''。[[中世紀|中世紀]]時猶太学者為希伯來語的4字神名標上母音時，使用「adonai」（意為「我的主」）的母音符号，（希伯來語单詞的輔音写在中央而元音（母音）写在輔音的上面或下面，以不同组合的点和短线表示，所以可以在「YHWH」下写「adonai」的元音符号），提醒猶太裔讀者要念成「adonai」，不可以直呼神的名。而早期聖經譯者翻譯時，便直接地把輔音和元音拼在一起，而成了「YaHoWaH」（翻譯就是「耶和華」）。

===大溪地人===
在玻里尼西亞的許多地方，重要領導人的名字被認為是神聖的(至今有時依舊如此，詳情可見[[玻里尼西亞文化中的禁忌|Tapu]])，並因此對之需有一些適當的敬重([[瑪那|mana]])。為了避免冒犯，任何類似一個名字的字眼都不可使用，並直到該人死去為止，都需以一個含義相似的詞來代替被禁止使用的字眼。然若一個領袖碰巧活得非常長命的話，這暫時性的替代用法，可能會變成永久性的，進而對語言產生影響。

一個例子是在[[大溪地語|大溪地語]]當中，一些單詞因為避諱的緣故，而被其他的單字給取代。在玻里尼西亞的其他地區，tū或其他同源的字之意為「站立」，但在大溪地語中，該字被以tiʻa取代，因為原本的詞根變成了[[波马雷一世|波马雷一世]]這位國王的名字中Tū-nui-ʻēʻa-i-te-atua的部份；相似地，fetū這個意指「星星」的字眼，在大溪地語中為fetiʻa所取代；而aratū這個意指「柱子」的字眼，也為aratiʻa所取代。

儘管nui(意即「大的」)依舊出現於像Tahiti-nui等合成詞中，一般用以指稱「大」的字眼為rahi(此詞在玻里尼西亞語言中是個指稱「大」的常見字眼)；此外，ʻēʻa這個詞也不再使用，並為purūmu或porōmu取而代之，在現今，ʻēʻa之意為「路徑」，而purūmu之意則為「道路」。

Tū也有個暱稱，叫做Pō-mare (字面含意為「夜咳」)，波馬雷一世的王朝也因此而得名，也因此''pō'' (意即「夜晚」)這個字曾一度為''ruʻi''所取代(然今日ruʻi只用於聖經中，pō這個字根今日又重新變成了日常用語的單詞)；而''mare'' (字面之意為「咳嗽」)則已不可逆地為''hota''所取代。

其他範例如下：
*''vai''(意即「水」)為 ''pape''所取代，並可見於Papeari、Papenoʻo、Papeʻete等名稱中。
*''moe''(意即「睡」)為''taʻoto''(其原始含意為「躺下」)所取代。

一些舊式的單詞依舊用於[[背风群岛_(社会群岛)|背風群島]]一帶。
