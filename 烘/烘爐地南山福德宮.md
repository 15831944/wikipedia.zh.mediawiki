'''中和南山福德宮'''，簡稱'''南山福德宮'''，俗稱'''烘爐地土地公廟'''、'''烘爐地土地廟'''，地址在[[臺灣|臺灣]][[新北市|新北市]][[中和區|中和區]]興南路二段399巷，主祀[[福德正神|福德正神]]（土地公），是位於[[中和區|中和]][[烘爐地山|烘爐地山]]的知名[[土地廟|土地公廟]]，是[[北臺灣|北臺灣]]最有名的[[土地廟|土地公廟]]之一。<ref>{{Cite web|url=http://tour.ntpc.gov.tw/page.aspx?wtp=1&wnd=136&id=123eca0200b00000381c|title=烘爐地南山福德宮|accessdate=2014-01-30|publisher=新北市觀光旅遊網|deadurl=yes|archiveurl=https://web.archive.org/web/20140225223254/http://tour.ntpc.gov.tw/page.aspx?wtp=1&wnd=136&id=123eca0200b00000381c|archivedate=2014-02-25}}</ref>

== 祀神 ==
*頂樓太歲殿入口：[[斗姥元君|斗姥元君]]、[[六十太歲星君|六十太歲星君]]
*頂樓正殿二樓：[[福德正神|福德正神]]、[[土地婆|土地婆]]、[[註生娘娘|註生娘娘]]、[[山神星君|山神星君]]。<ref name=":0">{{Cite web |url=http://www.libertytimes.com.tw/2012/new/oct/20/today-so1-2.htm |title=烘爐地土地公 北台灣最旺 |accessdate=2014-01-30|publisher=自由時報|date=2012-10-20}}</ref>
*二樓財神殿：[[財神|財神]][[玄壇真君|玄壇真君]]、[[月老星君|月老星君]]、[[文昌帝君|文昌帝君]]等。<ref name=":0" />

==簡表==
*清乾隆年初西元約一七三六年祖籍福建省漳州府紹安縣二都秀篆河尾堡盛埧樓，呂姓祖先萬春公派下盛埧房華現公(德進)由原籍背負香火，攜子發貯公及發賞，發質兩公，渡海來台，入墾現址約達面積七十餘公頃在今日[[中和區|中和]][[南勢角|南勢角]]一帶開墾，當時以石板三塊供奉福德正神冀以默佑邦家(現供奉於殿後)於是香火鼎盛日日祭拜。因為靈驗事蹟頗傳，[[香火|香火]]鼎盛。<ref>{{Cite web |url=http://iservice.libertytimes.com.tw/2013/specials/info/news.php?rno=1&no=653942 |title= 中和烘爐地 24小時可參拜|accessdate=2014-01-30|publisher=自由時報|date=2013-02-11}}</ref>
*1952年，[[檀越|檀越]]游興仁發心，改建為小廟。<ref>{{Cite web |url=http://mocfile.moc.gov.tw/htmlfile/201704/5372213d-1803-4183-8854-91ba5a45ab5e/watchtw_tw9/land@.html|title= 從家廟到財神廟之路 發財就來烘爐地 |accessdate=2014-02-03|publisher=國立臺灣歷史博物館 |date=2011-04-08}}</ref>
*1967年，本鄉[[士紳|士紳]]發起鳩資改建，改建正殿，1970年代竣工。而後也建了登山階梯、山腰道路、停車場等，方便信眾參拜。<ref>[http://www.hunglodei.org/index.htm 烘爐地南山福德宮 官方網站] {{webarchive|url=https://web.archive.org/web/20130822122717/http://www.hunglodei.org/index.htm |date=2013-08-22 }}</ref>

==  交通方式 ==
*自行開車：
**[[國道三號|國道三號]]：
#由[[中和交流道|中和交流道]]下，右轉中正路至南山路，右轉興南路至二段399巷口（兩線道），左轉後續行約500公尺在登山口附近路邊停車，或再續行約1公里到達烘爐地的立體停車場（約 2-300 個大小車位）。
#由[[安坑交流道|安坑交流道]]下，往安坑方向走到底右轉安康路，過1個路口右轉安和路，直走續行中和景新街，於興南路左轉，至二段399巷口。

*大眾運輸工具：
**[[公車|公車]]：搭乘[[台北聯營公車|台北聯營公車]]249、670、895，於烘爐地站下車，左轉興南路二段399巷口，步行約500公尺到達石階登山口，沿登山步道步行到達福德宮。
**[[台北捷運|捷運]]：[[南勢角站|南勢角站]]四號出口，沿興南路步行約2公里至烘爐地站牌處；或轉乘捷運接駁公車（自強國中－烘爐地）至終點站烘爐地站；接駁公車於週末假日（星期六，星期日，農曆初二及十六）會延駛至山腰烘爐地的立體停車場。

== 建廟歷程 ==
源起乾隆四年三塊石板

民國四十一年改建石造小廟

民國五十八年篳路藍縷開創重檐歇山頂建築

民國八十五年土地公大神像以一〇八尺之尊立於山巔守護大地

民國一〇〇年主殿二次修繕三殿式雙護龍格局

並增設財神殿為百年立基

南山福德宮在土地公的庇佑及信眾支持下

蛻變為多元信仰之廟宇殿堂
