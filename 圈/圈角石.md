'''圈角石'''（[[學名|學名]]：{{lang|la|'''''Spyroceras'''''}}）是一[[屬|屬]]已[[滅絕|滅絕]]的[[鸚鵡螺亞綱|鸚鵡螺類]]。牠們的[[化石|化石]]產於[[北美|北美]]和[[歐洲|歐洲]]等地。

== 特徵 ==
圈角石屬於[[直角石亞綱|直角石類]]，但和一般的直角石類外觀差異甚大，主要特徵為多個明顯凸出成肋狀的環圈結構，且其外殼並非常見直角石類所具有的[[圓錐|圓錐]]狀殼形。<ref name="Spyroceras">{{Cite web |url = http://liusir.pixnet.net/blog/post/336171584-spyroceras-%E5%9C%88%E8%A7%92%E7%9F%B3-%28%E6%91%A9%E6%B4%9B%E5%93%A5%29 |title = Spyroceras 圈角石 (摩洛哥) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-05-16 }}</ref>
