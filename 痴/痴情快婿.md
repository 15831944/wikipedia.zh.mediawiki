{{engname|《痴情快婿》|'' Fun and Fury ''}}是1992年出品的一部香港電影，由[[陳勳奇|陳勳奇]]執導，以喜劇故事為電影題材，[[黎明|黎明]]、[[周慧敏|周慧敏]]、[[鄭則仕|鄭則仕]]、[[郭秀雲|郭秀雲]]、[[陳勳奇|陳勳奇]]等演出，於1992年3月19日至4月2日期間在香港上映<ref>{{Cite web |url= http://hkmdb.com/db/movies/view.mhtml?id=7586&display_set=big5|title =痴情快婿（1992）| publisher =香港影庫|language = 中文}}</ref><ref>{{Cite web |url= http://www.hkbo.com.hk/search-details-new.php?filmid=8437&run=1|title =電影資料庫| publisher = 香港票房有限公司|language = 中文}}</ref><ref>{{Cite book|author =孫竹、張風蘭|title =銀海愛河: 香港影視明星的情愛逸事|publisher =中國文聯出版公司|ISBN = 7505924893|date =1996年|language =中文|}}</ref>。

==情節介紹==
故事講述韋漢是一名新加坡的反黑警察，馬青霞是其女友，馬青霞的父親馬六甲是已退休的黑幫老大。韋漢陪伴女友回港探望馬六甲，馬六甲得悉女兒的未婚夫竟然是警察，而且曾經在執勤時槍殺了販毒集團的首領，為免禍及女兒，於是想辦法拆散他們二人。販毒集團企圖暗殺韋漢及青霞，馬六甲帶領手下前來營救，卻被槍擊受傷，青霞與販毒集團交涉，結果被擄走。販毒集團脅持青霞，要求與韋漢見面，韋漢的朋友楊義陪同韋漢一起去與販毒集團會面，眾人爆發衝突，最後韋漢與楊義成功將販毒集團的老大消滅，救出青霞<ref>{{Cite web |url= http://big5.ent.qianzhan.com/ent/moviedetail/640dbfe2.html|title =《痴情快婿》電影劇情簡介| publisher =前瞻網娛樂|language = 中文}}</ref><ref>{{Cite web |url= https://moviecool.asia/film/gdMMsC19Rj |title =痴情快婿| publisher =影史庫|language = 中文}}</ref>。

==主要角色==
* [[黎明|黎明]]－韋漢，是一名新加坡的反黑警察。
* [[周慧敏|周慧敏]]－馬青霞，韋漢的未婚妻。
* [[鄭則仕|鄭則仕]]－馬六甲，馬青霞的父親，是個已退休的黑幫老大。
* [[陳勳奇|陳勳奇]]－楊義，韋漢的朋友，為人重情重義。
* [[簡而清|簡而清]]－杭叔，馬六甲的跟班。
* [[徐少強|徐少強]]－笑面虎
* [[郭秀雲|郭秀雲]]－蓉蓉
* [[吳浣儀|吳浣儀]]－娥姐
