'''嵐'''（{{jpn|j=嵐|hg=あらし、アラシ|rm=Arashi|f=y}}；{{lang-en|Arashi}}）是[[日本|日本]][[傑尼斯事務所|傑尼斯事務所]]旗下的男性[[日本偶像|偶像]]團體，由[[相葉雅紀|相葉雅紀]]、[[松本潤|松本潤]]、[[二宮和也|二宮和也]]、[[大野智|大野智]]、[[櫻井翔|櫻井翔]]等5人組成，於1999年11月3日發行首張單曲正式出道，所屬唱片公司為[[J_Storm|J Storm]]。自2010年起5度獲得[[日本金唱片大獎|日本金唱片大獎]]，並在[[oricon|oricon]]舉辦的『樂迷2萬人的選擇“喜歡的歌手”排行榜』中連續7年（10～16）拿下首位<ref>{{cite web | url=http://www.oricon.co.jp/special/2016/favorite-artist-ranking/| title=第13回 音楽ファン2万人が選ぶ 好きなアーティストランキング 2016| publisher=[[Oricon|Oricon]]|date=2016-11-14|accessdate=2017-05-17}}</ref>。组合预定于2020年12月31日起暫時中止活動。<ref>{{cite web |title=嵐、2020年をもってグループ活動休止　ファンクラブサイトで電撃発表 |url=https://headlines.yahoo.co.jp/hl?a=20190127-00010000-spht-ent |website=[[雅虎日本|雅虎日本]] |publisher=[[体育报知|体育报知]] |accessdate=2019-01-27}}</ref><ref>{{cite web |title=嵐　活動休止発表　リーダー大野が１７年６月中旬に提案「それぞれの道を歩んでは」 |url=https://headlines.yahoo.co.jp/hl?a=20190127-00000133-spnannex-ent |website=[[雅虎日本|雅虎日本]] |publisher=Sponichi Annex |accessdate=2019-01-27}}</ref>

== 簡介 ==
1999年從傑尼斯事務所旗下的關東小傑尼斯中挑選而組成，為5人組的男性偶像團體，同年9月15日於[[夏威夷|夏威夷]]召開發表記者會，11月3日發行首張單曲《[[A·RA·SHI|A·RA·SHI]]》正式出道。

目前成員們在音樂活動、[[電視劇|電視劇]]、[[電影|電影]]、[[舞台劇|舞台劇]]、[[綜藝節目|綜藝節目]]、報導等眾多領域活躍中。
