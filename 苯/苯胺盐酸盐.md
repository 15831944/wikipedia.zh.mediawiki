'''苯胺盐酸盐'''是一种芳香胺的的盐酸盐，是无色片状晶体，它可由[[苯胺|苯胺]]和[[盐酸|盐酸]]反应得到。苯胺盐酸盐可溶于水，在30ω%以上浓度的乙醇水溶液中，随着乙醇浓度的增加，其溶解度快速下降。<ref>孙顺平, 李志宝. [https://www.ixueshu.com/document/6d4584ceabbc87bc318947a18e7f9386.html 苯胺盐酸盐在乙醇-水体系中的溶解度测定和混合溶剂电解质模型]. 过程工程学报, 2011, 11(5):777-781.</ref>

== 制备 ==
苯胺盐酸盐可由弱碱性的[[苯胺|苯胺]]和浓盐酸发生中和反应得到：
: <ce>C6H5NH2\ + HCl -> C6H5N^+ H3Cl^-</ce>

此外，实验室还能通过[[锡|锡]]和[[盐酸|盐酸]]还原[[硝基苯|硝基苯]]得到。
: <ce>2C6H5NO2\ + 3Sn\ + 14HCl -> 2C6H5N^+ H3Cl^- \ + 3SnCl4\ + 4H2O</ce>

== 化学性质及用途 ==
苯胺盐酸盐可以和强碱反应，生成苯胺：
: <ce>C6H5N^+ H3Cl^- \ + NaOH -> C6H5NH2\ + NaCl\ + H2O</ce>

苯胺盐酸盐是一种[[染料|染料]]的中間体，将其浸渍到纤维上并对其氧化，可以得到一种叫做[[苯胺黑|苯胺黑]]的不溶性黑色染料，可用于染色。此外，亚硝酸在5 °C以下与之反应，生成重氮盐的氯化物，和酚盐混合，发生重氮偶合反应，产生红色至黄色的颜色，用于染色。
: <ce>C6H5N^+ H3Cl^- \ + HNO2 -> C6H5N^+ NCl^- \ + 2H2O</ce>
: <ce>C6H5N^+ NCl^- \ + C6H5ONa -> C6H5\ - N=N\ - C6H5OH</ce>

苯胺盐酸盐和亚硝胺反应而显色，被用于亚硝胺的检验。<ref>魏天俊, 杨清堂, 潘载炳. 亚硝胺板层分析法的一种新显色剂——苯胺盐酸盐的简易快速制法. 肿瘤防治研究, 1977, 5(2):76-80.</ref>
