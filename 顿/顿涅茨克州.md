'''頓內次克州'''（{{lang-uk|Доне́цька о́бласть|Donets'ka oblast'}}，{{lang-ru|Доне́цкая о́бласть|Donetskaya oblast}}）, 位于[[乌克兰|乌克兰]]东部，面積26,517平方公里，2014年人口433万，首府[[頓內次克|頓內次克]]市。其他重要都市有[[马里乌波尔|马里乌波尔]]和[[馬克耶夫卡|馬克耶夫卡]]。該州在歷史上是[[頓巴斯|頓巴斯]]的重要地帶。州里有大量大工业化公司。

== 地理 ==
頓內次克州位於烏克蘭東南部。面積約26,900平方公里，占全國總面積4.4%。南北橫跨270公里，東西190公里。該州在西南與[[第聶伯羅彼得羅夫斯克州|第聶伯羅彼得羅夫斯克州]]、[[札波羅結州|札波羅結州]]相鄰，東北與[[盧甘斯克州|盧甘斯克州]]接鄰，東為[[俄羅斯|俄羅斯]][[羅斯托夫州|羅斯托夫州]]，南臨[[亞速海|亞速海]]。

在[[斯维亚托戈尔斯克|斯维亚托戈尔斯克]]附近的歷史建築{{link-en|斯维亚托戈尔斯克修道院|Sviatohirsk Lavra}}曾被提名為[[烏克蘭七大奇景|烏克蘭七大奇景]]。
== 行政区划 ==
{{main|顿涅茨克州行政区划}}
顿涅茨克州下辖18个县和28个州辖市，[[乌克兰危机|乌克兰危机]]前首府为[[顿涅茨克市|顿涅茨克市]]，目前实际首府为[[克拉马托尔斯克|克拉马托尔斯克]]。

== 人口統計 ==
頓內次克州在2004年約有470萬人口，占全國人口10%，是全國人口最多也最密集的地區。人口眾多是由於境內有數座工業都市，造成村莊集聚。

[[2004年烏克蘭總統大選|2004年總統大選]]，[[維克托·亞努科維奇|維克托·亞努科維奇]]的支持者威脅如果他們支持的候選人的勝選結果沒有獲得承認，將聯合鄰近州要求[[自治|自治]]。不過支持者未作出任何正式行動。

2001年烏克蘭人口普查，頓內次克州境內的民族有[[烏克蘭人|烏克蘭人]] — 2,744,100 (56.9%)、[[俄羅斯人|俄羅斯人]] — 1,844,400 (38.2%)、[[希臘人|希臘人]] - 77,500 (1.61%)、[[白俄羅斯人|白俄羅斯人]] — 44,500 (0.92%)、[[克里米亞韃靼人|克里米亞韃靼人]] — 19,200 (0.4%)、[[亞美尼亞人|亞美尼亞人]] — 15,700 (0.33%)、[[猶太人|猶太人]] - 8,800 (0.18%)。<ref name="ukrcensus">[http://www.ukrcensus.gov.ua/eng/regions/reg_don/ Ukrcensus.gov.ua — Donetsk region] {{webarchive|url=https://web.archive.org/web/20090604192242/http://www.ukrcensus.gov.ua/eng/regions/reg_don/ |date=2009-06-04 }} URL accessed on January 13, 2007</ref>

2001年，頓內次克州居民的語言使用狀況為[[俄語|俄語]] — 74.9%、[[烏克蘭語|烏克蘭語]] — 24.1%,和[[亞美尼亞語|亞美尼亞語]] — 0.13%<ref name="ukrcensus"/>。

== 經濟 ==
=== 工業 ===
烏克蘭有一半以上的煤炭、成品鋼、焦炭、生鐵和鋼材產量來自頓內次克州。有色冶金、燃料工業和電力工業是工業生產中的重要產業。州內約有882家獨立工業企業，2,095小型工業企業。<ref name="CMU">[http://www.kmu.gov.ua/control/en/publish/article?art_id=124489&cat_id=32596 Cabinet of Ministers of Ukraine — Donetsk Region] URL accessed on January 13, 2007</ref> 

頓內次克州交通基礎設施發達，包括有頓內次克鐵路（占全國運輸40％）、馬里烏波爾港、[[頓內次克國際機場|頓內次克國際機場]]、[[馬里烏波爾|馬里烏波爾]]與[[克拉馬托爾斯克|克拉馬托爾斯克]]（Kramatorsk）的客運機場，還有稠密的道路系統。頓內次克州成立兩處[[經濟特區|經濟特區]]：「頓內次克」與「亞速」，享有優渥稅率。<ref name="CMU"/> 

=== 農業 ===
1999年，頓內次克州糧食產量為999.1千公噸、甜菜27.1千公噸、,葵花子309.4千公噸、馬鈴薯380.2千公噸。<ref name="CMU"/> 另外，肉類134.2千公噸、牛奶494.3千公噸、蛋646.4百萬顆。1999年年初，頓內次克州境內共有2108處農場。<ref name="CMU"/>

== 生態 ==
頓內次克州屬大陸型氣候，夏季炎熱冬季寒冷。東部及東南部在夏季易有強風、高溫及大雨發生。年雨量約524公厘。

[[File:Khomutovska_Steppe.JPG|thumb]]

頓內次克州蘊藏的擴物有：[[煤|煤]]（蘊藏量2500萬噸）、[[石鹽|石鹽]]、碳酸石灰、[[鉀|鉀]]、[[汞|汞]]、[[石棉|石棉]]、[[石墨|石墨]]。另外該州也有豐富的肥沃黑土。

重要的休閒資源有：溫和氣候、[[亞速海|亞速海]]海岸、治療泥、礦物質、氡、礦泉水。由於這些眾多的娛樂資源，許多度假酒店和露營營區座落於此。州內有26座保健中心和膳宿公寓、52處療養院與寄宿處、兒童休息營地等等。<ref name="CMU"/>

== 宣告獨立 ==
2014年4月7日，[[頓涅茨克|頓涅茨克]]的市民佔領了頓涅茨克政府大樓，並宣告成立「[[顿涅茨克人民共和国|顿涅茨克人民共和国]]」。要求从俄罗斯引入维和部队并拟于5月11日前就加入俄联邦举行[[公投|公投]]。<ref>[http://www.bbc.co.uk/zhongwen/trad/world/2014/04/140407_ukraine_donetsk.shtml 烏克蘭危機：頓涅茨克宣佈成立主權國家]</ref>在11日举行独立公投后，顿涅茨克州和[[卢甘斯克州|卢甘斯克州]]于12日宣布成为独立主权国家。<ref >{{cite web|url=http://news.china.com.cn/live/2014-05/13/content_26592862.htm | language= | title=乌克兰东部两州宣布成为独立主权国家 |publisher= 环球网|date=2014-05-13 |accessdate= }}</ref>目前[[顿涅茨克人民共和国|顿涅茨克人民共和国]]控制顿涅茨克州部分领土。
