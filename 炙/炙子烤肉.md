'''炙子烤肉'''最早起源于[[中国|中国]]北方塞外的[[蒙古民族|蒙古民族]]，随着清朝[[满洲|满洲]]入主中原，烤肉技艺也随之带入了北京。炙子烤肉成型于[[清|清]][[康熙|康熙]]二十五年。炙子指的是用铁条钉成的铁板，烧热后可以用来[[炙烤|炙烤]]，烤肉多余的脂肪可以透过缝隙滴走<ref>{{cite web |title=老北京最怀念的炙子烤肉，不只是“南苑北季” |url=http://wemedia.ifeng.com/63759000/wemedia.shtml |website=大风号 |accessdate=2018年11月17日}}</ref>。而烤肉的柴火主要使用带有芳香的木柴，如松木或枣木、梨木等果木<ref>{{cite web |title=炙子烤肉 您是文吃还是武吃？ |url=http://culture.qianlong.com/2017/1221/2269044.shtml |website=千龙网 |accessdate=2018年11月17日}}</ref>。烤肉的肉传统上主要用牛羊肉，多为[[清真|清真]]法宰杀。

在北京，炙子烤肉分为“文吃”和“武吃”。文吃即是由师傅在后厨准备好，吃相较为斯文。武吃即是人人围站在炙子旁，自己取腌好的肉在炙子上烤熟<ref>{{cite web |title=贴秋膘，吃烤肉。您爱文吃还是武吃？ |url=http://wemedia.ifeng.com/73542796/wemedia.shtml |website=大风号 |accessdate=2018年11月17日}}</ref>。

炙子烤肉的配菜，通常是大葱/洋葱和香菜。一同配芝麻烧饼食用。蘸食的小料分干湿两种，干料通常是孜然、茴香、辣椒和芝麻，湿料则是芝麻油、醋。

清末京城以烤肉闻名的饭店有三家，烤肉宛、[[烤肉季|烤肉季]]和烤肉王。烤肉王现已不存在，目前北京的[[中华老字号|老字号]]烤肉店仅剩两家，称“南宛北季”。
