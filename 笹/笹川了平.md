'''笹川了平'''（{{jpn|j=笹川 了平|hg=ささがわ りょうへい|rm=Sasagawa Ryōhei}}），是[[日本|日本]][[漫畫|漫畫]]和[[動畫|動畫]]作品《[[家庭教師HITMAN_REBORN!|家庭教師HITMAN REBORN!]]》的[[家庭教師HITMAN_REBORN!角色列表|虛構角色]]，也是本作品主要角色之一，是彭哥列家族的第十代晴之守護者。死氣炎屬性波動為"晴"，代表顏色為"黃"。

== 人物簡介 ==
笹川京子的哥哥，並盛中學[[拳擊|拳擊]]社主將。首次登場時是拳擊社主將及運動會的級長，讓人覺得他是高年級生，但實際是2年級生，只比[[澤田綱吉|澤田綱吉]]等人大一歲。<ref name="target44">漫畫第六集，第44回「內藤龍祥」。</ref>是個以「'''[[極限|極限]]'''」一詞為自己的做事原則及口頭禪的熱血男人，因為平時都以拚死心情（極限）做事，所以就算被死氣彈擊中後也完全沒有變化。<ref name="target14">漫畫第二集，第14回「笹川了平」。</ref>

實力非常強大，在'''風太'''的「並盛中學打架強人排行榜」中處於第5位，因此被[[里包恩|里包恩]]看中邀請其成為家族的一員，本人卻完全不知道。是個凡事都要與[[拳擊|拳擊]]扯上關係的笨蛋，一旦看到有資質的人便想盡辦法勸對方加入拳擊社，因此在看到恰巧處於'''死氣狀態'''的[[阿綱|阿綱]]後，便深受感動時常勸[[阿綱|阿綱]]加入拳擊社。<ref name="target14" />在熱血的外表下，其實也曾因被多瑪佐家族特有的「嘆息彈」打中陷入自怨自艾的情緒中，抱怨大家都認為[[拳擊|拳擊]]是種冷門的[[運動|運動]]，更認為京子把[[拳擊|拳擊]]當作是一種「只穿[[內褲|內褲]]及護套搏擊」的運動。<ref name="target45">漫畫第六集，第45回「嘆息彈」。</ref>

小學時曾幫助受流氓騷擾的京子，導致額頭留下至今仍依稀可見的疤痕。雖然有時會與京子吵架，但也有身為哥哥心疼妹妹的一面。每次與[[獄寺|獄寺]]見面都因意見不合而吵架。不相信占卜，常說自己的命運是要自己開拓的，其實非常介意自己的抽籤運很差。認為男生是[[處女座|處女座]]很奇怪，所以自稱「拳擊座」。與其外表不符，出人所意料之外的喜歡[[草莓|草莓]]口味[[刨冰|刨冰]]。

== 劇中表現 ==
; 日常篇
: 做為最熱血最極限的男人，了平永遠喜歡和熱血的人打交道。從此他便追隨著阿綱要求他學習拳擊，不知不覺中和眾人產生了深厚的友誼。

; VS瓦利亞篇
: 由家光授予守護[[澤田綱吉|澤田綱吉]]及成為主要家族成員證明的晴之半彭哥列戒指的擁有者，修練期間拜可樂尼洛為家庭教師。利用必殺技「極限太陽」反敗為勝打敗魯斯里亞。最後[[阿綱|阿綱]]成為正統繼任者，正式授予他作為守護者證據的晴之彭哥列戒指。

; 未來篇
*; 未來篇
:: 10年後仍是晴之守護者，且掌握家族內部情況，性格比起十年前更加穏重。<ref group="註">不過，編出不可能的謊話來騙京子的這一點仍舊沒變。</ref>'''彭哥列本部'''毀滅時他正在海外出差<ref group="註">前往[[意大利|意大利]]與魯斯里亞會面</ref>，從[[日本|日本]]出發前曾給黑川花<ref group="註">從二百三十一話中其房間中遺留下來的照片可得知十年後的了平正與黑川花交往，另外在一百五十二話中其中一個部分，在黑川花的房間裡亦有同一張跟了平的合照。</ref>留下口信。在現代的[[阿綱|阿綱]]等人到達未來世界的十三日後回國，並救出在黑曜中心打贏古羅的十年前庫洛姆。向[[阿綱|阿綱]]他們傳達'''彭哥列本部'''即將於五日後突入'''米爾菲歐雷家族'''本部。

*; 未來篇X
:: 在'''米爾菲歐雷家族'''位於[[日本|日本]]的基地中背著負傷的拉爾、[[山本|山本]]等四人行動，但因入江正一的策略和其他人分開，之後和[[獄寺|獄寺]]一起行動。打贏白沙納後，卻在與'''γ'''對戰時受了重傷，之後轉由[[獄寺|獄寺]]對付'''γ'''。
:: 後來在梅洛尼基地的超炎戒指瞬間移動時，帶著晴之彭哥列戒指來到十年後，並拿到入江正一給他的晴之彭哥列匣。和[[阿綱|阿綱]]等人為十日後決戰進行特訓，並在十年後迪諾的分配下，和[[藍波|藍波]]一起接受[[獄寺|獄寺]]的指導，輕鬆開啟了'''漢我流'''的匣子並讓'''漢我流'''聽命於他。

*; 未來決戰篇
:: 在和修羅開匣後的桔梗戰鬥時，成功讓'''漢我流'''的型態產生變化，變成'''初代晴之守護者'''的最強武器「被歌頌為明亮照耀天空的太陽的──納克爾的極限Break」。雖然其出拳速度遠超乎[[桔梗|桔梗]]想像，但了平僅能使用'''超強化肉體'''3分鐘，再加上'''晴之火焰的活性'''的副作用，使得了平之前的傷口再次裂開。
:: 最後因[[阿綱|阿綱]]領悟出覺悟而出現的彭哥列一世令七枚彭哥列戒指解除封印，得到原版晴之彭哥列戒指。

; 繼承式篇
: 與轉學生'''青葉紅葉'''一起參加課後輔導，兩人意外臭味相投。因[[阿綱|阿綱]]遭受到敵對家族襲擊，所以與其他同伴及西蒙家族等人輪流擔任阿綱的護衛。然而繼承式當天依然遭襲，了平與[[阿綱|阿綱]]聯手和古里炎真戰鬥竟意外地被壓倒性打敗，'''彭哥列戒指'''也遭受到破壞。
: 了平以最強的'''覺悟'''點燃火炎，成功將'''彭哥列戒指'''升級成「晴之臂環Ver.X」。並跟隨[[阿綱|阿綱]]前往西蒙家族聖地，搜索西蒙眾人途中意外遇到'''紅葉'''，了平將'''漢我流'''型態變化為'''彭哥列裝備'''，卻只與其戰鬥打成平手，最後兩人一起被復仇者帶走。

; 彩虹的詛咒篇
: 接受[[里包恩|里包恩]]的邀請加入其陣營。在第一場戰事之中，原打算與[[山本武|山本]]等人趕到[[阿綱|阿綱]]身邊幫忙，卻被代表「風」的雲雀阻止，被雲雀不屑的態度激怒的了平，因為大意反被雲雀輕鬆擊敗<ref name="target361" />，最後僅獄寺與山本成功逃脫。<ref name="target363" />

== 技能 ==
=== 招式 ===
; {{nihongo|極限鬥志|極限ストレート（きょくげんストレート）}}
: 因為平時都以拚死心情（極限）做事，所以就算被死氣彈擊中後也完全沒有變化。

; {{nihongo|極限疾走|極限ラッシュ（きょくげんラッシュ）}}
: 不停止的高速長跑，獄寺與山本也無法跟隨。

; {{nihongo|極限太陽|極限太陽（マキシマムキャノン）}}
: 受到[[可樂尼洛|可樂尼洛]]以特殊子彈打擊後所領會的招式，攻擊時雖會使手受傷，威力卻足以一擊破壞巨石和體育館。

; {{nihongo|極限衝鋒槍／極限英格拉姆|極限イングラム（マキシマムイングラム）}}
: 十年後了平所使用的招式，進入敵人懷下後以高速連環三拳擊敗敵人。三重極限太陽。極限太陽改良版。

; {{nihongo|極限聯合|極限コンビネーション（マキシマムコンビネーション）}}
: 裝備「極限Break」後使用的連環快拳招式。

; {{nihongo|極限加農炮|極限キャノン}}
: 裝備「極限Break」後施展的「極限太陽」，能如傳說中'''初代晴之守護者'''可以隔空傷敵的燦爛拳技，從拳頭擊出的晴之炎拳勁光束。

; {{nihongo|陽光計數器|サンシャインカウンター}}
: 裝備「彭哥列齒輪」時的招式，吸收一定傷害後會冒出個火焰的臂鐲。

; {{nihongo|陽光直射|サンシャインストレート}}
: 裝備「彭哥列齒輪」時的招式。

; {{nihongo|日陽升龍| サンシャインアッパー}}
: 裝備「彭哥列齒輪」時的招式，可吸收敵人力量以累積攻擊力。

; {{nihongo|極限日曜上勾拳|極限サンシャインカウンター（マキシマムサンシャインカウンター）}}
: 裝備「彭哥列齒輪」時的招式，使用'''日陽升龍'''吸收敵人力量至100%時，以全力擊出之上勾拳。

; 7<sup><small>3</small></sup>戒指（彭哥列戒指）
: 屬性晴，精緻度S，未來篇的十年後因打碎而不存在，屬於7<sup><small>3</small></sup>戒指的一部份。 
: 繼承式篇被打碎，經彭哥列雕金師塔爾波之手，將動物戒指與彭哥列I世的血「罰」加入後復活並版本升級。

; 動物戒指－漢我流（晴袋鼠）
: 未來篇VS真六弔花結束之後，回到十年前（原故事時間），未來的阿爾柯巴雷諾贈與的禮物，是彭哥列匣的戒指版本。
: 繼承式篇與晴的彭哥列戒指融合並版本升級。

; 一般戒指
: 屬性晴，精緻度不明，十年後了平使用，用作開啟匣子。

=== 匣子 ===
; {{nihongo|動物匣 - 晴 - 晴袋鼠|晴カンガルー（カングーロ・デル・セレーノ）}}
: 10年後了平把牠取名為「漢我流<ref group="註">將袋鼠的英文寫成片假名後再轉成漢字</ref>」，10年後了平所遺留下來的匣子。很快地就被了平馴服。羅倫翠尼設計，可從其腹部的口袋中射出了平專用的晴之拳擊手套，擁有「治癒」的能力，還有可以在空中飛行的推進鞋（F鞋子），大小為1~1.9m。

; {{nihongo|一般匣 - 晴 - 晴平鏟|晴コテ（サルダトーレ・デル・セレーノ）}}
: 十年後了平留下的。肯尼希設計，可以治療小部分傷口，無法像晴孔雀可以治療大部分傷口，而且也有人質疑他的強制超活性會提早讓細胞死亡，與他類似的匣兵器還有能夠緩和疼痛的「雨平鏟」，大小為15cm。

; {{nihongo|一般匣 - 晴 - 死氣之火的繩子|名称不明（縄）}}
: 10年後了平使用，用晴之活性將人加以捆綁，使用者如果失去意識就會鬆開。

; {{nihongo|彭哥列匣 - 晴 - 晴袋鼠Ver.V|晴カンガルーVer.V（カングーロ・デル・セレーノ バージョン・ボンゴレ）}}
: 了平可以從「漢我流」背上的加農炮接受晴之超活性炎光彈，漢我流則可轉換形態變身成初代晴之守護者的最強的武器「被歌頌為明亮照耀天空的太陽的－－納克爾的極限Break」。晴之超活性炎令神經和肌肉強化10倍力量，但普通人身體無法承受此超活性炎的力量，了平亦只可使用3分鐘。
