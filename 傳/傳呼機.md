'''傳呼機'''（大陸通稱'''寻-{}-呼机'''；[[台灣|台灣]]通稱'''-{呼叫器}-'''；[[香港|香港]]通稱'''傳-{}-呼機'''，俗稱-{zh-cn:'''BP机'''、'''BB机'''、'''B.P.Call'''、'''Call机'''; zh-tw:'''BB.Call'''; zh-hk:'''Call機'''}-。法規上名詞是'''無線電叫人業務'''），是一种具有接收和传送简易[[文字|文字]][[信息|信息]]功能的个人[[无线电|无线电]][[通讯|通讯]]工具<ref>http://nrch.culture.tw/twpedia.aspx?id=1007490</ref>。現已式微。

== 功能 ==
传呼机有单向（仅接收）和双向（收发）两种类型。但单向仅能接收的机型是最为常见的。传呼机通常供使用者接收回呼请求，同时也能够接收其他留言信息。寻呼机网络供应商有时也提供各种增值信息，比如[[新闻|新闻]]与[[股市|股市]]信息。

除此之外，因接收的內容不同，又分成四種機型：
# 音樂機（沒有任何的螢幕顯示，只會有鈴聲提示使用者，有人呼叫）
# 數字機（只能顯示數字）
# 文字機（可顯示文字及數字）
# 語音機（收到的是語音訊息，可直接播出人聲）

== 历史 ==
{{globalize|time=2013-11-10T02:53:52+00:00}}
1990年代，傳呼機十分流行，但到了21世纪，功能逐步被[[行動電話|手機]]取代。但在一些国家如[[美国|美国]]，传呼机仍用在一些特殊场合如[[医院|医院]]、[[餐廳|餐馆]]等<ref>[http://gb.cri.cn/12764/2007/03/30/2225@1523076.htm 美国人依然使用传呼机]</ref>，如醫院的醫護人員若使用一般行動電話有可能[[干擾|干擾]]到醫療器材，因此佩帶呼叫器；在沒有服務人員遞送餐點的餐館，客人點餐後會拿到一台呼叫器，則可先就座等候，餐點準備好時餐館便以無線電遙控該呼叫器發出閃光及聲音以提醒客人取餐。

=== 台灣 ===
台灣在1976年開放音型無線電叫人業務。1988年二月，推出顯示型無線電叫人業務，
1992年時的呼叫器用戶使用量正式突破百萬用戶，1996年中有 221萬人使用這項業務。直到1999年時達到432萬用戶的最高峰。呼叫器服務從單純的英、數字到加入中文服務，在當時股市熱潮帶動下也曾推出各類型加值服務。隨著行動電話的普及，呼叫器的用戶使用量也迅速的萎縮。[[國家通訊傳播委員會|國家通訊傳播委員會]]2011年2月底時統計，台灣的呼叫器用戶數尚有約105萬-106萬戶左右，但真正使用呼叫器功能的戶數約只有10萬戶，其餘的用戶多為第二類電信業者提供[[股票機|股票機]]服務使用<ref>[http://blog.yam.com/cliff/article/8609629 懸崖不勒馬: 呼叫器(B.B.Call)輓歌]</ref>。

[[中華電信|中華電信]]表示該公司的呼叫器用戶約剩不到百門，因此預計在2011年9月1日時也將停止呼叫器業務服務，當時所使用呼叫器的人數僅剩76人，但最後[[國家通訊傳播委員會|國家通訊傳播委員會]]決議將全面停止的日期延到2011年12月31日，中華電信呼叫器服務正式走入歷史<ref>[http://www.libertytimes.com.tw/2011/new/apr/23/today-so10-2.htm 呼叫器市場萎縮 中華電年底喊卡] {{webarchive|url=https://web.archive.org/web/20130315074340/http://www.libertytimes.com.tw/2011/new/apr/23/today-so10-2.htm |date=2013-03-15 }}</ref>。台灣廠商長期替呼叫器大廠[[摩托羅拉|摩托羅拉]]代工，呼叫器式微後全球仍有1、2千萬大賣場、工廠、醫院，以及急難救助使用的呼叫器訂單，繼續由台灣金阿波羅等廠商提供。

[[中華國際通訊|中華國際通訊]]0941/0951門號全面停止的日期是2011年12月31日。

=== 香港 ===
{{main|香港傳呼業}}
香港傳呼業自1970年代開始發展，傳呼機（[[香港人|香港人]]俗稱為「Call機」）曾經成為普及的通訊工具之一。1990年代中期，香港[[手提電話|手提電話]]盛行，傳呼業開始沒落，但至今香港仍有一間公司提供傳呼服務<ref>[http://www.coms-auth.hk/tc/licensing/telecommunications/prs/paging/index.html 香港公共無線電傳呼服務營辦商]</ref>，傳呼機尚未被完全淘汰。截至2017年8月，香港還有24310位用戶<ref>[http://www.ofca.gov.hk/filemanager/ofca/tc/content_108/wireless_c.pdf 香港無線通訊服務的主要統計數字]</ref>。

=== 中国大陆 ===
传呼机和无线传呼网络于1980年代出现，经过十多年的发展，于1990年代末期达到了顶峰。在[[手机|手机]]普及之前，人们普遍以拥有传呼机为荣。当时BP机 （beeper機）即是BB机，beeper是呼叫器學術名詞，可發出蜂鳴、警笛聲之意，始自70年代音型無線電叫人業務之時，後傳呼機英文則多使用 pager一詞。
有人誤傳是因为B和P的读音很像而非BP机的音效，BP机的音效是“'''嘀嘀'''”并不是“'''哔哔'''”。1998年，全国传呼机用户突破6546万，名列世界第一。其后逐渐式微。2007年3月信息产业部发表公示称，[[中国联通|中国联通]]申请停止经营[[北京|北京]]等三十省（自治区、直辖市）的198/199、126/127、128/129无线传呼服务。标志着传呼机在大陆地区的运营正式结束<ref>[http://news.ccidnet.com/art/949/20090910/1883285_1.html 别在腰间的传奇 寻呼机时代历史回顾 ]</ref>。

== 技术規格 ==
给传呼机发送信息的方式通常为拨打一个指定的[[电话号码|电话号码]]，而傳呼機就会自动获得发话者的电话号码。而如果需要传送文字信息，则通常会拨打网络供应商的电话，由人工代发。有些网络供应商也提供基于[[Web|Web]]页面或者是[[email|email]]的发送方式。

常见的传呼机協議有[[TAP|TAP]]、[[FLEX|FLEX]]、[[ReFLEX|ReFLEX]]、[[POCSAG|POCSAG]]、[[Golay|Golay]]、[[ERMES|ERMES]]和 [[NTT|NTT]]。在[[亚洲|亚洲]]地区，最多采用的协议是[[FLEX|FLEX]]。 [[FLEX|FLEX]]协议由[[摩托罗拉|摩托罗拉]]开发，此通訊協議提供单向信息传送功能，使用900Mhz频段。
