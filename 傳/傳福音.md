'''傳福音'''是指傳播[[基督信仰|基督信仰]]的行為，亦稱'''傳道'''、'''講道'''、'''傳教'''、'''宣教'''、'''佈道'''（[[新教|新教]]使用較多）、'''福傳'''（[[天主教|天主教]]多用此詞），其字源來自[[耶穌|耶穌]]在世時的[[通用語|通用語]]——[[通用希臘語|通用希臘語]]的「{{polytonic|[[wiktionary:εὐαγγέλιον|εὐαγγέλιον]]}}」，意為「好消息」，後引申為傳播[[基督|基督]]的[[福音|福音]]。

== 簡介 ==
傳福音是[[基督徒|基督徒]]的重要使命。[[新約聖經|新約聖經]]裡數個章節提到基督徒傳福音的重要性：“你们往普天下去，传福音给万民听”<ref>{{cuv|可|16|15}}</ref>，以及“我传福音原没有可夸的，因为我是不得已的；若不传福音，我便有祸了”<ref>{{cuv|林前|9|16}}</ref>。但由于聽者的思想觀念、宗教信仰、教育程度乃至意識形態有所差别，聽者所處的地點、時間不便，基督徒本身可能來自不同的教派因而所傳教義不盡相同的緣故，基督徒在傳福音的時候會經常受到不同程度的排斥或者冷漠對待。有些基督徒在一生中也沒有試過傳福音，但有部分基督徒以傳福音做為畢生的[[志業|志業]]。

傳福音的內容大體上是以[[耶穌基督|耶穌基督]]為中心及[[聖經|聖經]]經文，主要是強調[[救恩|救恩]]的內容，通常也会涉及关于[[原罪|原罪]]和[[大审判|大审判]]等有關[[基督教神学|基督教神学]]内容 有時也可能會有更貼近聽者生活的內容。不同的基督教團體所傳講的教義內容並不完全相同，有時甚至會有較大的差別。

== 方法 ==
基督徒有很多傳福音的方法，當中以三福，四律，五色佈道法較為普遍及流行。一些教会也会印制一些小册子，发给[[慕道友|慕道友]]或是[[路人|路人]]。

傳福音的方式，是隨者時代與社會的轉變而有所不同的，進入[[工業化|工業化]]時代前，基督徒大多以[[講座|到處演講]]、發送[[出版物|出版物]]、創作[[音樂|音樂]]等方式來傳播福音。[[現代|現代]]傳福音的方式則相當多樣化，除了傳統方式之外，[[流行音樂|流行音樂]]、[[傳媒|傳媒]]、[[營隊|營隊]]也常被基督徒運用，以和一般大眾的[[生活|生活]]貼近，使福音更能被瞭解。

== 法律問題 ==
在某些[[政教合一|政教合一]]的[[伊斯蘭國家|伊斯蘭國家]]，傳播基督教是違法行為。雖然[[以色列|以色列]]容許基督教，但禁止向18岁以下的犹太人传教<ref>[http://www.cna.com.tw/SearchNews/doDetail.aspx?id=200708170208 美新教牧師夫婦被下令離開以色列] {{webarchive|url=https://web.archive.org/web/20120117232631/http://www2.cna.com.tw/SearchNews/doDetail.aspx?id=200708170208 |date=2012-01-17 }}</ref>。另外一些非傳統教派，例如[[耶和華見證人|耶和華見證人]]、[[耶穌基督後期聖徒教會|耶穌基督後期聖徒教會]]（[[摩門教|摩門教]]）等，因為傳統教派施壓等緣故，在少部分傳播基督教合法的國家里也遭到不同程度的限制，但在大多數宗教自由的國家里都不會受到限制。在少部分受限制的國家里也通常都可以通過合法途徑加以解決。

在[[朝鲜民主主义人民共和国|北韓]]、[[中华人民共和国|中华人民共和國]]等[[共產主義國家|以共产党为法定执政党的社会主义国家]]，虽然宪法写明公民有宗教信仰的自由，但禁止或限制在非宗教场所传教。
