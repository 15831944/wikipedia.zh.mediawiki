== 故事簡介 ==
不知道自己是誰，失去記憶的主角在下雪的荒野中甦醒。還沒搞清楚狀況的情形下，突然被大型昆蟲襲擊，在關鍵時刻被一名少女「久遠」所救，而讓主角不解的是久遠身上像是獸類的耳朵與尾巴……

== 登場角色 ==
由於動畫版監督[[元永慶太郎|元永慶太郎]]'''並沒有'''玩過遊戲原著，僅僅只是依照遊戲劇本進行動畫化改編<ref>{{cite web|title=バンダイチャンネル·クリエイターズ・セレクション|url=http://www.b-ch.com/contents/feat_creators_selection/backnumber/v27/p02.html|date=2015-11-27|accessdate=2015-11-29|language=ja}}</ref>，因此部分劇情及角色性格在遊戲中與動畫中有相當大的差異，以下以遊戲中為主。
=== 主要人物 ===
; {{nihongo|哈克|ハク|Haku|聲：[[藤原啓治|藤原啓治]]|另譯：白}}
: 在雪山迷路，記憶盡失、遇到襲擊時被久遠所救的青年。本作的主人公。年齡約二十餘歲。
: 與其他有著獸耳、獸尾的亞人相對、哈克是個人類。
: 因為連自己的名字也想不起、久遠給了他「{{lang|ja|ハク}}」這個名字，這個名字是從久遠故鄉傳說「被傳頌之人」的名字而來的。
: 與周圍相比體能極低、連小孩能簡單做到的事也能讓他苦戰，再加上是懶惰者、用著各種各樣的藉口想在工作上偷懶。
: 但是相對洞察力很優秀、頭腦也轉得很快、能下達大膽的作戰與正確的判斷而多次絕處逢生。
: 擁有強大的運算能力與理化知識。雖然讀不懂這個世界的文字，但只是稍為學習一下就逐漸學會。
: 喜歡甜味、根據記憶與擅長料理的露露緹耶一起創造了許多在帝都大受歡迎的甜點。
: 由於亞人基因潛在影響會對「人類」產生親近的好感，因此很容易受到眾人的喜愛。
: 感覺很敏銳、但面對會給自己造成麻煩的事時卻喜歡故意視而不見。

; {{nihongo|久遠|クオン|Kuon|聲：[[種田梨沙|種田梨沙]]|另譯：庫恩}}
: 漂亮可愛又老實的藥師少女，有著一頭順滑光澤的頭髮、毛茸茸的耳朵與被稱為世界第一美的尾巴，身體帶著一股很香的香氣。年齡約十餘歲（已過成年禮）。
: 真實身分是[[传颂之物#.E4.B8.BB.E8.A6.81.E8.A7.92.E8.89.B2|哈克奧羅]]與[[传颂之物#.E5.9D.8E.E7.B4.8D.E8.A5.BF.E7.A7.91.E7.83.8F.E9.AD.AF.E8.B2.9D|柚葉]]之女，由於母親柚葉體弱多病，在她尚在襁褓之中時便因宿疾撒手人寰，因此在名字裡包含了希望她能長壽的意味<ref>朧在遊戲中與哈克聊天時所述</ref><ref>前作中僅有柚葉有懷孕，詳見前作 Windows PC 版（18禁）</ref><ref>白貓尾外觀符合遊戲世界觀中的亞人遺傳法則</ref>。
: 自幼受到哈克奧羅身邊眾女性的寵愛，與艾露露一樣以成為獨當一面的藥師為目的修行中。
: [[#.E5.9C.B0.E5.90.8D|圖斯庫爾]]的皇位繼承人，長大後便要肩負國家重任，隨心所欲也只能在年輕時，因此家人讓其自由的活著。
: 從小便四處不停的旅行，被內心催動著去看更多跟多的世界，見識那些從未見過的景色，感受異地的氣息與人們的風俗，體驗當地才有的飲品，品嘗當地才有的食物。
: 吃吃喝喝的遊山玩水之旅，沒有說教沒有學習的、解放的世界，是久遠心目中的桃源鄉。
: 在離開故鄉後的旅途中與主角哈克相遇，並把照顧哈克當成自己的義務。
: 喜歡甜食，特別是蜂蜜，同時也非常喜歡泡澡，甚至在旅行途中也會為自己準備洗澡水，認為伴隨著一天的結束，泡澡流身汗是非常舒服的一件事。

; {{nihongo|右近|ウコン|Ukon|聲：[[利根健太朗|利根健太朗]]}}
: 哈克與久遠在旅店遇見的男子，鍛煉過的軀體，銳利的目光，還留著絡腮鬍，但年齡給人一種還年輕的印象。
: 雖然衣著上不算用心，但其外貌給人一種豪放青年的感覺，哈哈的笑聲有著一股吸引人的魅力，是猫音的兄長。
: 實際上是大國[[#.E5.9C.B0.E5.90.8D|大和]]的右近衛大將軍[[#.E8.BF.91.E8.A1.9B.E5.A4.A7.E5.B0.87|奧修特爾]]，文武雙精、為官清廉潔白，深受民眾信賴與皇帝重用。
: 原本是低階貴族，在帶著繼承父志、成為與民眾同甘共苦之士的志向從邊境上京後、在恩師的照顧下成了近衛末席，而後又因數次的偶然與幸運、使其功績不斷積累陞遷，而當上了右近衛大將。在此同時也有著因其出身而恥於與他共事的人存在。
: 由於右近衛大將軍的身分過高，不管做什麼都會被世間矚目，難以聽見民眾真正的聲音，因而創造了「右近」這個能避開世人目光、自由行動的身分，以處理一些上不得檯面或不可或缺的事務。
: 看中哈克的才能，也因右近的身分逐漸不敷使用而延攬哈克接替自己行走基層。

; {{nihongo|貓音|ネコネ|Nekone|聲：[[水瀨祈|水瀨祈]]|另譯：喵音、妮可涅、聶可涅}}
: 來自[[#.E5.9C.B0.E5.90.8D|恩納卡姆依]]的少女，右近的妹妹，非常崇拜右近。
: 以最年少的紀錄通過殿試，但由於年紀過小被保留了哲學士頭銜，成為殿學徒。
: 因自小身為天才，周圍的人很難跟上話題、只能獨自在家用功、無法與同年代的孩子們玩到一起，鮮少能看到如同女孩子般天真爛漫年幼稚氣的表情。
: 右近在僱用哈克後，將她交給了哈克，希望她在不懂世故到無法挽回之前，能稍微經歷下世間的艱辛，不要只用自書裡的知識來描述世間。
: 對於哈克的弱氣及懶散度日十分不屑，時常口出諷刺，但遭遇可怕的東西時仍會不自覺的依靠他。
: 為久遠的包容力與氣魄所折服，與久遠締結姊妹之契，憧憬著久遠。
: 喜歡甜食，興趣是研究古代的事物。說話時常在句尾加上「{{lang|ja|です}}」。

; {{nihongo|麻呂呂|マロロ|Maroro|聲：[[杉山大|杉山大]]}}
: 削瘦的身體配著華麗的衣服，但臉上塗抹著的化妝使之顯得十分滑稽可笑的男人，右近團隊的智囊。
: 皇帝欽點，通過最難殿試的殿學士，以高貴之人自居，堅持就算落魄也不能忘記優雅。
: 家人仗其身分揮霍度日，因而使他背負了許多債務，時常處在窮困狀態。
: 崇拜哈克到了讓人懷疑其性取向的地步。
: 同時也是個「術士」，使哈克第一次見識到「法術」的存在。

; {{nihongo|露露缇耶|ルルティエ|Rurutie|聲：[[加隈亞衣|加隈亞衣]]|另譯：璐璐緹艾}}
: 大和國的從屬國[[#.E5.9C.B0.E5.90.8D|九重里]]的小公主，「八柱將」之一九重里之皇歐甄的么女，總共有14位兄姐。
: 因為是公主的身份，所以沒有什麼朋友，楚楚可憐，溫柔婉約。憧憬著「八柱將」之一的宗近。
: 其父皇歐甄因對露露缇耶愛之切，恐其成為籠中之鳥，因而派其前往帝都進貢，望其居於帝都期間能接觸更多的人，拓展對世間的見聞。
: 在押送九重里進貢給帝的貢品途中與哈克等人相識，對哈克有很深的好感，然而哈克從未察覺。
: 騎著一隻一人高的巨大鳥類，稱牠為[[#.E5.85.B6.E4.BB.96.E7.9B.B8.E9.97.9C|可可啵]]，在旁人表現畏懼的同時依然覺得牠很可愛。
: 喜歡料理與縫紉，廚藝優秀，因從小身體柔弱，無法幫忙家族開墾，因此專注在茶飯上。在一行人中負責準備每天的餐點。
: 隱藏著對於男男之戀的興趣，偶爾可見到她出沒於帝都的「文藝之巷」。
:; {{nihongo|可可啵|ココポ|Kokopo|聲：[[米澤圓|米澤圓]]|另譯：可可波}}
:: 跟著露露缇耶的巨大[[#.E7.94.9F.E7.89.A9|赫蘿蘿恩鳥]]，好奇心相當旺盛、如同淘氣鬼般看到有趣的事就會立刻找上去，惹了不少事。
:: 平時雖然很聽話，但一生氣就會狂化。同時也因比正常的赫蘿蘿恩鳥大上非常非常多，被山賊視為被禍日神附身的鳥。

; {{nihongo|奇羽琉|キウル|Kiuru|聲：[[村瀨步|村瀨步]]|另譯：奇烏爾、奇烏魯、裘魯}}
: 「[[#.E5.9C.B0.E5.90.8D|恩納卡姆伊]]」的皇子，身材嬌小給人印象不錯的可愛少年，貓音的同鄉，跟右近是結義的兄弟，另外似乎也與露露缇耶有某種關係。
: 雖然樸素且有些弱氣，但武術的鍛鍊上毫無欠缺。
: 暗戀著貓音，但周圍只有貓音沒感覺出來。
: 被貓音評論為不成熟、黏人、麻煩人、掉鏈子、愛哭鬼、精神脆弱、明明有才能卻輕言放棄、毫無審美觀、總是把樸素跟優雅搞混等……。

; {{nihongo|阿圖依|アトゥイ|Atoui|聲：[[原由实|原由实]]|另譯：阿托伊}}
: 大國大和「八柱將」之一的索楊克庫爾之女，力大無窮，擅使一把巨大的雙尖槍。
: 來自[[#.E5.9C.B0.E5.90.8D|夏赫羅]]，人稱「夏赫羅的狂公主」，猶如狂亂的綻放的鮮花般、容貌美麗的公主。
: 與哈克一行人在帝都相識，隨身帶著一隻類似浮空水母的謎之生物，稱牠為克拉琳。
: 處在憧憬戀愛的年紀，為哈克帥氣的背影所吸引而搭話，轉過臉來卻平凡的使她大失所望。
: 容易墜入愛河，為此鬧出了許多笑話。
:; {{nihongo|克拉琳|クラリン|Kurarin|聲：[[三宅麻理恵|三宅麻理恵]]}}
:: 阿圖依的寵物，半透明的身體加上兩條觸手、如水母般飄在空中的迷之生物。觸手有麻痺的攻擊效果。
:: 雖然什麼都吃，但主要喜歡的都是新鮮的肉還有蟲子跟魚。
:: 喜歡待在阿圖依頭上，但時常被誤認為是帽子。會對阿圖依的話有所反應，似乎有相當的智商。

; {{nihongo|鵟|ノスリ|Nosuri|聲：[[山本希望|山本希望]]|另譯：諾斯莉}}
: 義賊團體「鵟旅團」的首領、面容漂亮的少女，雖然可愛但算不上絕世美女，出身自[[传颂之物#.E7.A8.AE.E6.97.8F|艾文庫魯加族]]。
: 因無法原諒山賊們的行為，而與奧修特爾合作，作為內應對付山賊。
: 雖說是個劫富濟貧的義賊，但其「濟貧」的方法卻常帶給被救濟之人困擾。
: 立志成為一個「好女人」，但大大咧咧、我行我素的性格時常弄巧成拙。

; {{nihongo|扇|オウギ|Ougi|聲：[[櫻井孝宏|櫻井孝宏]]|另譯：歐基}}
: 鵟的弟弟，也是其副手，思維縝密，擅使雙刀，同樣出身自艾文庫魯加族，鵟旅團實質上的領導者。
: 敬愛著姐姐鵟，但也有先行安排行動而後再以明面上的理由哄騙鵟的腹黑一面。

== 世界觀 ==
=== 故事背景 ===
{{HideH
|Head         = 故事背景
|FrameStyle   = background:#F0F0F0
}}
遠古時期，因地表嚴重汙染，世界各地的人類被迫遷居至地底下生活，但因長時間生活在地底設施中，使人類漸漸失去了免疫力，地表環境恢復後卻無法生存在普通環境中。因此，世界各地的科學家開始做各種奇奇怪怪的研究幫助人類重返地面生活，其中不乏會悲鳴的蕪菁、會走路的蘿蔔、會吃人的豬籠草等等……

此時，如同宅男般只吃乾糧、餐具不洗、垃圾不清、生活在自己的房間中的主角。自稱是超級駭客，閒暇之餘著手破解其他國家的研究計畫，並發現了日本研究所的防禦系統越來越強而起疑，認為有些見不得人的是正在進行著。而主角的兄長為一研究計畫——真人計畫的首席科學家，由於真人計畫出現了瓶頸，出生率低下、遺傳病的增加，外加環境汙染，人類這個種族的壽命已經不長了，不管志向多高，要是得不出結果的話都毫無意義，不能如往常般靠藥物及遺傳因子操作研究，而需要技術和奇思妙想。想邀請主角加入研究，但主角卻自認不足而屢屢回絕兄長的研究邀請。

在主角破解了日本研究所的研究資料後，發現他們在做著禁忌般危險的研究，而號稱是研究成果的，是似人又非人類的「'''亞人種'''」創造。在主角將研究資料提供給兄長後，使得真人計畫有了重大性的突破。而後主角也被兄長作為研究的素體被施加了手段、為了進行最終調整而進入低溫睡眠（兄長只對主角說是試驗新藥）。

然而就在調整結束，只待主角醒來的時候，不知為何，發生了巨大的災厄，讓人活生生地融化成史萊姆的怪病在全世界蔓延起來，而人類在變成史萊姆後更將周遭所有生物都吞噬掉，就像是在說將這世上所有生命都吞噬是其使命一般；病原體不明、感染途徑不明、別說治療、連防禦都辦不到！而主角的兄嫂和姪女，也都在兄長面前融化、變成了再也無法溝通、只為吞噬而存在的姿態。（詳見[[受讚頌者|前作]]）

全世界因此陷入了大慌亂，因身邊的人都可能會突然變成史萊姆攻擊你，即使待在與外界隔離、完全密封的避難所都毫無意義，不知是天上神靈所為、還是地底惡魔所致、雖然不知何時發生、但只要被他們發現就無法逃脫了。

而化為史萊姆的人是無法消滅的，拼命存活下來得人們都陷入了恐慌，因不知身邊的人何時會變成怪物來襲，人類在這種疑心暗鬼驅使下、恐懼著其他人，並將周圍所有人都當成了敵人，盡然動用「'''[[#其他|天照]]'''」來消滅他們，接著就是全面戰爭開始……然而那已不能稱做是戰爭，僅僅只是人類之間毫無意義的互相殘殺，因為那時候、已經是連戰爭都無法展開的情況了，數個小型太陽焚盡了地表、地震不斷、海嘯吞沒了一切，城鎮接觸設施完全被破壞、積年累月所再生的自然環境也被破壞了。

如此的混亂中，主角的兄長活下來了，在周圍人不斷便成史萊姆中、拼命的逃竄著，並奇蹟般的找到了即將完成全部設施但因戰爭而被放棄的地下研究所，得以躲藏起來。

在過了不知多久後，兄長用設施裡的系統好不容易控置了「'''天照'''」，並在反覆調試後、重新調整好了局部範圍的自然環境。然而，在兄長終於能回到地表時，一切都結束了，留下來的只有兄長一個人而已，儘管嘗試過各種方法，但不管哪裡都沒有人跡了。孤身一人，被留在這世上的'''帝'''（兄長），無法忍耐那股孤獨，在發現地表上還有代替人類熬過災難的亞人的存在後，也動手染指了禁忌，創造了更多似人又非人類的亞人種，並很不可思議地在模仿了神之所為後卻未感到罪惡感。

'''帝'''為了亞人們創造了民族、教授其文字、計數的方法、更有效率的狩獵方法、耕田的方法等……傳授了各種各樣的知識，眼見無垢的亞人們日益變的聰穎，'''帝'''在愉快之餘，也感到一股空虛，關根究底，亞人並不是人類。

被亞人敬為神、感到與亞人之間的巨大隔閡、感到無法生活在他們之中，'''帝'''漸漸地對地表失去興趣，再次回到地下研究所而閉門不出，為了完成該做的事、為了在一次和家人團聚、為了找出讓變化了型態的人類恢復原種的手段，'''帝'''明知這是何等魯莽的行為，但也只剩此可以做了。

之後，又過了相當長的歲月，百年、兩百年....或是更久（由於'''帝'''曾對自己的身體進行了各種臨床實驗，原本只是想提高基礎體能，卻無意間將壽命延續到了相當的地步，沒變成史萊姆，恐怕也是因為這個原因），'''帝'''感覺到只靠地下研究室的設備的話已經達到研究極限了，於是再一次的離開研究室到了地面上，認為雖因災難而失去了大半的都市和研究設施，但在避難所深處的話，或許還殘留著什麼設備，但是，從地底出來的'''帝'''，卻因眼前看到的風景而啞然，比知道世界毀面而來到地表時感到的衝擊還強：在'''帝'''眼前的是亞人們的國度，並非村落之類、而是已經有好幾個王國建立在地表了。

'''帝'''向亞人提出協助挖掘遺跡，並以設施中栽培出的食物和藥劑作為相應的報酬給予亞人們，但亞人們不知是否因'''帝'''授予了太過多餘的知識，失去了老實聽話的純粹而變的貪婪，對於報酬的要求，也慢慢地逐步升級。'''帝'''雖然感到這並非好的傾向，但對'''帝'''而言無論如何那些研究器材都是必要的，只能回應了亞人們貪婪的要求。而後，亞人更是變本加厲，擅自從遺跡裡拿走器材、並用那些那些對於亞人而言無法理解的、等同路邊石子之物，貪婪的強迫'''帝'''給予更多的東西來做交換，'''帝'''雖憤怒但也不的不答應。

「制鐵技術、新品種的農作物種子、水井的挖掘、精緻的天文學」等……雖然對'''帝'''而言都是些無聊的內容，但對亞人來說卻都是革命性的知識，依靠這些知識，亞人們的社會一夜之間強弱更替，然而這並不關'''帝'''的事，如果亞人們沒有愚蠢到妨礙'''帝'''的話……

但亞人們是如此的愚蠢，從開始的知識爭奪、到最後連想要抓住'''帝'''、獨佔知識的人都出現了，而亞人擅自從遺跡帶出來的精密儀器，也因不懂原理而胡亂擺弄的原因，其大多數變得無法使用。'''帝'''失望了、第二次失望了，亞人不僅外觀像人類，連內在的靈魂都與人類相似。

'''帝'''捨棄了對亞人的感傷，為了達成崇高的目的，決定排除只會造成妨礙的亞人，以卓越的知識與技術、支配亞人建立起自己的軍隊，由於亞人在本能上被植入了對人類的遵從，所以事情比想像中更順利，'''帝'''區使著變的忠實的亞人，佔領遺跡，攻陷周邊妨礙'''帝'''的國家使之至於麾下，建立起了'''大和帝國'''。

而'''帝'''之所以要挖掘遺跡，還有另一個理由便是尋找主角，為主角所施加調整的研究所，是兼做避難所的半永久隔離設施，雖迷失在災難中，但'''帝'''無法捨棄說不定會找到的希望，而帝作夢都沒想到的是，主角會自己出現，然而在得知主角失憶後十分苦惱，喜悅轉瞬即逝。

帝為了拯救人類、拼命存活至今，雖被當作神一般崇拜，但終究只是人類而已，無法違背自然規律，在生命即將消逝之餘，希望主角能繼承他的意志、作為人類的意志，繼承'''帝'''未實現的願望。

: 對於貧寒的他們來說，豐饒的大和土地讓他們垂涎，因此時不時的會對大和進行小規模的掠奪，被大和居民蔑稱為蠻族並厭惡著。
: 原本烏茲魯沙與大和兩兵交接的事很少發生，因其目的是獲得物資，因此幾乎都是襲擊倉庫與搶奪收成，未受到大和重視。
: 在烏茲魯沙之皇「袞杜魯亞」出現後，整合了過百個部落，建立起一大強國，並對周邊各國展開掠奪、吸收，最後對大和舉兵。
: 雖國力弱於大和，但男女老少皆為身經百戰之兵，以比大和多出數倍的軍隊展開侵襲。
: 根據以往經驗，以為只是掠奪而非侵略而對烏茲魯沙有所輕視的周邊鄰國，毫無應對之機便被吞噬。

; {{nihongo|圖斯庫爾|トゥスクル|Tusukuru}}
: 大和西方，海的另一邊的島國，過去與大和僅有商人之間的零碎交流。地理位置相當於現今的[[日本|日本]]。（詳見[[受讚頌者|前作]]舞台）

=== 生物 ===
; {{nihongo|奇奇利|ギギリ|Gigiri}}
: 肉食系的蟲子。喜食腐肉，體型距大、多足、硬殼、且帶有如剪刀般的大顎。
: 會咬人，但正常情況下不會主動襲擊人。帶有會讓身體麻痺的毒性但不致命，被咬到會產生紫色的膿包。
: 通常會在溫暖的地帶進行繁殖。繁殖期間會出現個別個體極端成長，成為{{nihongo|'''巨奇利'''|ボロギギリ}}的情況。
: '''巨奇利'''會為了幼蟲的食物、跟幼蟲一起四處襲擊獵物。
: 在遊戲中外觀近似'''綠色蠍子'''，動畫中則為'''赤色蜈蚣'''。

; {{nihongo|祟|タタリ|Tatari||另譯：塔塔利}}
: 如同紅色史萊姆般的詭異生物，棲息在陽光照射不到的地低深處，以襲擊靠近的生物生物作為糧食，具體數量和生態環境不明。
: 特性是絕對不會死亡，不管燒、切、打、都會馬上復活，不管怎麼做都無法消滅，最多只能徹底打到它離開、或者用巨大聲音和光亮將之驅趕。
: 在亞人社會中相關情報異常稀少，有關塔塔利的事不是被當做禁忌就是被故意避開，跟塔塔利有關的人、不是生病就是被詛咒終生不幸，如同要人們無視塔塔利的存在一樣。
: 實際上是偉大之父的窮途末路，無法死去、面臨永劫的徘徊在世上。詳見[[传颂之物#.E5.AE.97.E6.95.99|前作]]。

; {{nihongo|騎龍|ウォプタル|Uoputaru}}
: 別名：{{nihongo|馬|ウマ}}，[[傳頌之物|前作]]登場的生物，2足步行，用於馬車拉引與騎乘。
: 因氣候不同而大致分為羽毛濃密與光滑鱗片兩種外觀。

; {{nihongo|赫蘿蘿恩鳥|ホロロン鳥|Hororon}}
: 嬌小的、顏色鮮艷的、帶有萌點的鳥。
: 通常被當作食用禽類飼養。

=== 其他 ===
; {{nihongo|天照|アマテラス|Amaterasu}}
: 歷經環境變遷的人類所製造出來的氣候控制衛星，行星生命圈再生的樞紐。
: 依靠方法不同，可將人工太陽投射至地面上、或者捲起超級颶風。
: 要是失去控制便成冰河時期會導致全球凍結。

; {{nihongo|偉大之父|大いなる父|{{lang|ja|オンヴィタイカヤン}}|Onvitaikayan}}
: 在傳說中，於遙遠太古時期建立起如同神技般的高度文明、支配著大地的人。
: 即便如此，還是突然從歷史舞台上消失了蹤影的、被傳頌之人。
: 被亞人們視為如同神一般敬畏的存在。

; {{nihongo|禍日神|ヌグィソムカミ|Nuguisomukami}}
: 泛指給人們降下瘟疫和災害，被畏忌嫌棄的神。

; {{nihongo|常世|コトゥアハムル|Kotouahamuru}}
: 天國。

; {{nihongo|地獄|ディネボクシリ|Dinebokushiri}}
: 地獄。

; {{nihongo|假面|仮面|{{lang|ja|アクルカ}}|Akuruka}}
: 通往強大力量的門扉，只有假面之人才能解放其力量、變成巨大的姿態，是能將一切存在屠盡的根源之力、再多人都無法通過的破滅領域。
: 假面都各自有著不同的力量，不過卻不能輕易使用。傳聞中，假面會慢慢與主人同化、最終吞噬主人的靈魂。
: 雖假面為通往強大力量的門扉，但想要打開門必須要先打開鎖，為此雖只需要極短時間、但必須集中精神，那一瞬間不但無法反擊、更是連自保都做不到。

; {{nihongo|假面之人|仮面の者|{{lang|ja|アクルトゥルカ}}|Akuruturuka}}
: 被皇帝認可、被假面選中的武人。

; {{nihongo|鎖之巫|鎖の巫|{{lang|ja|くさりのカムナギ}}|Kusari no Kamunagi}}
: 凌駕於普通巫（{{lang|ja|カムナギ}}）之上，意義特殊的存在。
: 擁有將敵對於大和之人封印的法術，是能束縛對帝都造成威脅之徒並加以封印的神技。
: 同時掌握著帝都所有應對災厄的技術。傳聞中，得到鎖之巫的人、就如同得到帝都一般。

; {{nihongo|咒法|呪法}}
: 結印後、將念力轉為力量的術法。咒文在遊戲中顯示為[[楔形文字|楔形文字]]。
