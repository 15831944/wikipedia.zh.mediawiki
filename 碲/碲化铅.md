'''碲化铅'''是[[铅|铅]]的[[碲化物|碲化物]]，化学式为PbTe。它是一种窄带半导体，[[带隙|带隙]]为0.32 eV。<ref>{{Cite journal|title = Nanostructured Thermoelectrics: The New Paradigm? †|url = http://pubs.acs.org/doi/pdf/10.1021/cm902195j|journal = Chemistry of Materials|date = 2009-10-07|pages = 648–659|volume = 22|issue = 3|doi = 10.1021/cm902195j|first = Mercouri G.|last = Kanatzidis}}</ref>碲化铅在电流作用下的酸性或碱性溶液中比单质[[碲|碲]]更容易氧化。<ref>张胜义, 何吉阳, 毛淑玲,等. [http://www.ixueshu.com/document/cb3cb0d15a95a604318947a18e7f9386.html 碲化铅纳米棒的电化学性质研究][J]. 分析化学, 2009, 37(a02):64-64.</ref>

碲化铅在自然界中以[[碲铅矿|碲铅矿]]的形式存在。

==制备==
碲化铅可由[[铅|铅]]和[[碲|碲]]的反应得到：<ref>刘倩, 牟强, 张方辉. 碲化铅单晶及薄膜物理特性的研究[J]. 陕西科技大学学报, 2005, 23(2):45-49.</ref>
: Pb + Te → PbTe
[[硼氢化钾|硼氢化钾]]与[[二氧化碲|二氧化碲]]在碱性条件下和[[乙酸铅|乙酸铅]]反应，可以得到PbTe：<ref>汪元元, 蔡克峰, 李晖,等. 温和条件下碲化铅纳米颗粒的制备[J]. 稀有金属材料与工程, 2009, 38(s2):1127-1129.</ref>
: TeO<sub>2</sub> + 2 OH<sup>−</sup> → TeO<sub>3</sub><sup>2-</sup> + H<sub>2</sub>O
: Pb<sup>2+</sup> + TeO<sub>3</sub><sup>2-</sup> → PbTeO<sub>3</sub>
: PbTeO<sub>3</sub> + 6 BH<sub>4</sub><sup>−</sup> → PbTe + 6 OH<sup>−</sup> +3 B<sub>2</sub>H<sub>6</sub>↑
