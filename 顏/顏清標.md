'''顏清標'''（{{bd|1960年|8月25日|CatIdx=Y顏}}），[[綽號|綽號]]「冬瓜標」，[[台灣|台灣]][[臺中市|臺中市]][[沙鹿區|沙鹿區]]出生，私立[[神州高中|神州高中]]畢業，現為財團法人[[大甲鎮瀾宮|大甲鎮瀾宮]]基金會董事長、[[無黨團結聯盟|無黨團結聯盟]]榮譽主席，曾任多屆[[立法委員|立法委員]]，後因[[貪污|貪污]]案遭到[[褫奪公權|褫奪公權]]。在[[台灣地方派系|地方派系]]屬於[[台中|台中]]黑派大老。

== 簡歷 ==
早期打著祖父為其取的「冬瓜標」名號，成為台中[[海線|海線]][[角頭|角頭]]，為中部[[黑道|黑道]]重量級人物。「顏家班」長期稱霸於海線，主要活躍在[[大甲|大甲]]、[[沙鹿|沙鹿]]、[[清水|清水]]、[[大肚|大肚]]、[[龍井|龍井]]等區。 

曾犯下[[貪污|貪污]]、[[槍砲|槍砲]]、[[殺人|殺人]][[未遂|未遂]]、教唆頂替、[[盜匪|盜匪]]和加重[[強盜|強盜]]罪等[[罪行|罪行]]。<ref>[http://www.epochtimes.com/b5/1/2/27/n52103.htm 台中縣正副議長顏清標、張清堂28日凌晨遭收押], [[大紀元|大紀元]], 2001年3月1日</ref><ref>[http://www.epochtimes.com/b5/1/3/1/n52904.htm 發現新證據 顏清標涉嫌殺人未遂、頂替、槍炮等罪], [[大紀元|大紀元]], 2001年2月27日</ref>

1987年因[[一清專案|一清專案]]被送[[綠島|綠島]]管訓。

1991年，三十二歲起踏入政壇，由里長、縣議員選到省議員、立委，連戰皆捷。

2019年，自創「標太郎」品牌賣手工水餃。

== 經歷 ==
早年顏清標之父顏懷以[[柑仔店|柑仔店]]為業，後開始涉入[[柏油|柏油]]、[[瀝青|瀝青]]業。當時年幼的顏清標則與母親一同繼續經營柑仔店。

身為二代的顏清標家族又加入經營砂石業而聞名海線，期間亦有盜採砂石、建築販賣海砂屋的違法行為。但礙於顏清標家族政治及黑道勢力龐大，令許多人敢怒而不敢言。靠著台中傳統黑派及黑道勢力總能讓[[公權力|公權力]]默許，家族事業更加壯大，進而走上政治路。<ref>{{Cite web |url = https://www.youtube.com/watch?v=XQQQDQ_0m84 |title = 顏清標父柑仔店起家　顏家因顏清標成台中望族|date = 2014年2月19日 |publisher = 三立新聞台 |language = zh-hant}}</ref>

但也因砂石業龐大的利益糾紛，與當時槍擊要犯綽號「美國博」的[[林博文|林博文]]結怨。據顏清標表示，林博文要抽佣金，顏清標不肯。

資深媒體人[[蔡玉真|蔡玉真]]說：「顏清標要用黑道與政治的實力來捍衛他們家族，在台中黑派裡面裡頭很多這種比較特殊行業，當然包括除了砂石之外，還包括比較特許的行業。」
<ref>{{Cite web |url = http://www.setn.com/News.aspx?NewsID=14147 |title = 父"瀝青"業起家，顏清標跨足砂石業 |date = 2014-02-19 |publisher = 三立新聞 |language = zh-hant}}</ref>

於《[[黑金|黑金]]》一書，描述顏清標除了砂石業之利益外，亦開設有[[非法|非法]]的[[職業|職業]][[賭場|賭場]]。
<ref> {{Cite book | author = 陳國霖 | title = 黑金 | location = 台灣 | publisher = 商周出版 | date = 2004-06-14 | url = http://www.books.com.tw/products/0010262160 | language = zh-hant }} </ref>

胞弟顏清金（綽號目仔龍）曾因殺人未遂案遭判刑9年定讞。其間顏清標教唆劉文德為顏清金頂罪，案發後，顏清標被判罪八月有期徒刑定讞。<ref>{{cite news |title = 顏清標胞弟 顏清金殺人未遂 判9年定讞 |date = 94-10-28|publisher = 中國時報 |language = zh-hant}}</ref>

顏清標在地方勤於耕耘，身兼[[鎮瀾宮|鎮瀾宮]]的董事長，地方上只要碰到難以解決的問題，都會找顏清標幫忙。大甲鎮瀾宮立場不同的兩派地方人士，為了董事會之爭搞得水火不容，最後在顏清標勢力介入，由自身擔任董事長才將其中一個勢力逐出鎮瀾宮<ref>[http://www.businesstoday.com.tw/v1/content.aspx?a=W19990404024 顏清標推動宗教直航　顏清標愈來愈夠力]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

期間媒體名嘴[[周玉蔻|周玉蔻]]報料，大甲鎮瀾宮副董事長鄭銘坤被綁遭勒索一億元乃是與顏清標自導自演的事件，顏清標暴怒。
<ref>{{Cite web |url = http://www.setn.com/News.aspx?NewsID=30848 |title = 鄭銘坤被綁有黑幕？遭爆與顏清標自導自演　假贖金真A錢 |date = 2014-07-14 |publisher = 三立新聞／綜合報導 |language = zh-hant}}</ref>

顏清標擔任[[大甲鎮瀾宮|大甲鎮瀾宮]]董事長時大力推動對弱勢兒童的慈善活動<ref>[http://tw.news.yahoo.com/%E9%8E%AE%E7%80%BE%E5%AE%AE%E6%9B%B8%E6%B3%95%E7%BE%A9%E8%B3%A3-%E5%8A%A9%E5%BC%B1%E5%8B%A2%E7%AB%A5-20110115-110208-764.html 鎮瀾宮書法義賣　助弱勢童]</ref>。

=== 林明樺黃博廷集團 ===
[[大甲鎮瀾宮|大甲鎮瀾宮]]副董事長鄭銘坤遭林明樺黃博廷集團綁票勒索一億元。

顏清標委請中部有力仲裁者「憨面大仔」和南霸天[[楊雙伍|楊雙伍]]出面談判，最後以境外斡旋、國內現金付贖款方式救出鄭銘坤。

雙方接觸後，外傳約定在國外談判，顏清標請了第三者，林明樺也請了第三者選在[[菲律賓|菲律賓]]討價還價。

最後確定贖款2500萬，綁匪在高雄大寮河濱公園釋放鄭銘坤。

事後，警方發現從台中市議長張宏年、賭博電玩大亨余國柱和這次鄭銘坤綁架案來研判，綁匪幾乎都鎖定黑白兩道具有實力的人士，這些人有黑道對話的窗口，也習慣道上私了的方式，因此警方難以介入。<ref>{{Cite web |url = http://news.tvbs.com.tw/old-news.html?nid=439871 |title = 顏清標救人　請出憨面楊雙伍七大哥 |date = 2005-06-12 |publisher = TVBS |language = zh-hant}}</ref>

=== 彰化陳姓角頭 ===
2005年，九月中旬，彰化重量級角頭陳錦福在其開設在八卦山上餐廳外，遭人開槍狙擊雖僅受輕傷，但因其中可能涉及中部黑道恩怨，甚至波及無黨籍立委顏清標，因此道上傳出有人放話要對顏不利。在一場告別式的場合中，因陳錦福與顏清標同時出席，讓警方繃緊神經並全程嚴密監控。 

一名熟知中部黑幫生態的刑事偵查佐指出，2001年彰化海線重量級角頭陳錦福的手下大將林明勇，在彰化市經營的酒店遭槍殺身亡，林的手下綽號「牛皮」的王俊偉要求「大哥大大」必須拿出「安家費」照顧大哥家人，卻遭陳錦福拒絕，王俊偉心生不滿，去年九月憤而持槍，前往陳錦福公司掃射洩恨，且負氣投靠中縣海線黑道份子吳東皇旗下。 

吳東皇從中市金錢豹南七酒店消費後，在續攤途中，遭歹徒分持長短槍掃射當場身亡，據傳，吳為立委顏清標之手下大將，進而引發顏清標不滿。九月，陳錦福前往其位於八卦山的餐廳，正準備下車時，突遭歹徒連開八槍，臀部中彈，陳錦福遂將矛頭指向王俊偉所為，要求顏清標必須交人，警方表示，目前已從旁找尋證據，預防事態擴大。 <ref>{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20051030/2166296/ |title = 黑幫放話 警護顏清標 |date = 2005年10月30日 |publisher = 頻果日報 |language = zh-hant}}</ref>

=== 彰化過溝仔幫 ===
起因為彰化過溝的重量級人物陳錦福被開21槍險遭狙殺的8天後，海線殺手、名列十大槍擊要犯綽號「牛皮」的王俊偉的二個手下，也在立委顏清標住家不遠處遭人開槍。由於「牛皮」王俊偉跟過的[[大哥|大哥]]都是顏清標胞弟顏青金的手下，以致顏清標的大兒子顏寬恆差點受到池魚之殃，險些中了埋伏而遭狙殺。但海線和過溝二大黑幫勢力，已演變到水火不容的緊繃狀態。<ref>{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/property/20080821/30874914/ |title = 疑護航頭號殺手 顏清標家人成獵殺目標 |date = 2008年8月21日 |publisher = 蘋果日報|language = zh-hant}}</ref>

=== 地方角頭翁奇楠 ===
顏清標次子顏仁賢疑似因停車糾紛與綽號「阿南仔」的翁奇楠發生衝突，險些造成黑幫火拼。
雖然打人一方事後請出中南部大哥級人物致歉，並簽下和解書，但情勢仍然暗潮洶湧，後續發展引起黑白兩道高度關注，警政署長[[王卓鈞|王卓鈞]]獲悉後，也已指示所屬密切注意，防止黑道火併。
<ref>{{Cite web |url = http://news.ltn.com.tw/news/focus/paper/229538 |title = 顏清標子被毆罰站 險肇黑道火併|date = 2008-07-24 |publisher = 自由時報 |language = zh-hant}}</ref>

2010年，「阿南仔」翁奇楠遭槍殺身亡。一度被懷疑與此事件有關，但缺乏具體事證。最終警方以槍手廖國豪、幕後主使者為楊定融宣告破案。

==「顏清標條款」==
從政時官司不斷，常有牢獄之災<ref>[http://issue.udn.com/520/s4-1.htm 中縣議長顏清標遭收押 ] {{webarchive|url=https://web.archive.org/web/20120512130516/http://issue.udn.com/520/s4-1.htm |date=2012-05-12 }}</ref><ref>[http://tps.judicial.gov.tw/970626%E6%9C%80%E9%AB%98%E6%B3%95%E9%99%A2%E6%9C%89%E9%97%9C%E9%A1%8F%E6%B8%85%E6%A8%99%E5%85%88%E7%94%9F%E7%AD%89%E6%A1%88%E5%88%A4%E6%B1%BA%E6%96%B0%E8%81%9E%E7%A8%BF.txt 最高法院有關顏清標先生等案判決新聞稿.]</ref>，例如1999年擔任台中縣議長期間，喝花酒報公帳一千八百餘萬元，於2011年台中高分院更四審宣判，依公務員利用職務詐取財物罪，判處顏清標三年六月、裭奪公權三年。顏清標還表示，此案嚴格來說，絕對不是「貪污」行為，而是法律規定不明確的「使用不當」，希望上訴之後，司法能還給他一個公道<ref>[http://tw.nextmedia.com/applenews/article/art_id/33682283/IssueID/20110921 喝花酒疑雲 顏清標遭判3.5年]</ref>。因此2011年10月立法院上個會期通過首長特別費除罪化、替不少藍綠首長解套，而無黨團結聯盟再提案，欲把除罪範圍擴大到民意代表和村里長，並且在立法院逕付二讀。外界解讀這項提案、是為了幫無盟立委顏清標過去在地方民代任內涉及的司法案件解套<ref>[http://web.pts.org.tw/php/news/pts_news/detail.php?NEENO=194649 "顏清標條款" 民代特別費擬除罪]</ref>。

2013年5月31日，立法院在會期最後一天深夜，由國民黨籍立法院副院長[[洪秀柱|洪秀柱]]主持，國民黨籍[[立法院長|立法院長]][[王金平|王金平]]下令清場後，各黨團一致同意，三讀通過由[[林滄敏|林滄敏]]代表國民黨提出的《會計法》99條之1修正案，將以過去的假發票報帳行為除罪化。這個修法被視為是為顏清標除罪而做<ref>{{cite news |language=zh-tw |author=鄭閔聲、管婺媛、唐筱恬 |url = http://tw.news.yahoo.com/%E7%8E%8B%E9%87%91%E5%B9%B3%E6%96%A1%E6%97%8B-%E9%A1%8F%E4%B9%9F%E9%97%9C%E9%81%8E%E4%BA%86-%E4%BF%AE%E6%B3%95%E9%81%94%E9%99%A3%E9%97%9C%E9%8D%B5-213000631.html| title =王金平斡旋+顏也關過了→修法達陣關鍵| publisher =中國時報 |date=2013年6月2日 |accessdate=2013年6月2日}}</ref>，但大學教授報假帳的行為並沒有除罪<ref>{{cite news |language=zh-tw |author=蘇位榮、林毅璋|url=http://udn.com/NEWS/NATIONAL/NAT4/7941396.shtml| title =會計法修法烏龍！ 漏1字教授未除罪|publisher=聯合報 |date=2013年6月4日 |accessdate=2013年6月4日}}</ref>，因該法條漏了一個「教」字，讓教授可能被排除在外。該修正案的通過引起社會輿論的普遍不滿，總統要求行政院長轉請立法院覆議，此為「[[會計法第99-1條修正烏龍事件|會計法第99-1條修正烏龍事件]]」。6月13日覆議案通過，原決議不予維持。
