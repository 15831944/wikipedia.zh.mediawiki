'''顏重慶'''（'''{{lang|en|Steven Gan}}'''，{{bd|1963年||}}），馬來西亞記者，政治新聞網站《[[当今大马|当今大马]]》的創辦人。《当今大马》被視為馬來西亞的“第一和唯一”獨立新聞來源。<ref>{{cite web |url=http://www.nieman.harvard.edu/reports/article/101376/Virtual-Democracy-in-Malaysia.aspx |title=Steven Gan bio |author= |date=Summer 2002 |work= |publisher=''[[Nieman_Reports|Nieman Reports]]'' |accessdate=29 January 2012}}</ref><ref name=ICIJ />於2000年，他獲[[保護記者委員會|保護記者委員會]]授予[[國際新聞自由獎|國際新聞自由獎]]。<ref name=CPJ />

==生平==
1989年從一間[[澳洲|澳洲]]大學畢業，並獲得政治經濟學學士學位。<ref>{{cite web |url=http://www.guardian.co.uk/technology/2001/apr/29/freespeech.observercampaignpressfreedom |title=Ending the government's monopoly on the truth |author=Steven Gan |date=29 April 2001 |work= |publisher=''[[The_Guardian|The Guardian]]'' |accessdate=30 January 2012}}</ref>之後，他在[[香港|香港]]成為一位自由撰稿人。於1991年，他在[[巴格达|巴格达]]報道了[[海湾战争|海湾战争]]。於1994年，他回到馬來西亞，並成為當地日報《太陽報》的記者。但是，當地的政府監管和媒體自我審查導致新聞自由被約束。<ref name=CPJ /> 

於1995年，他報道了59名孟加拉移民於塞門伊移民拘留營集體死亡的新聞。<ref name=ICIJ /><ref name=NR>{{cite web |url=http://www.nieman.harvard.edu/reports/article/101376/Virtual-Democracy-in-Malaysia.aspx |title=Virtual Democracy in Malaysia |author=Steven Gan |date=Summer 2002 |work= |publisher=''[[Nieman_Reports|Nieman Reports]]'' |accessdate=29 January 2012}}</ref>但是，《太陽報》的編輯因為恐懼政府，所以拒絕發表其報道。因此，顏重慶將報道移交移民權利組織[[力量组织|力量组织]]。<ref name=NR/>力量组织發表了顏重慶的報道，而组织的領袖[[艾琳·費爾南德斯|艾琳·費爾南德斯]]卻因此被恐嚇。<ref name=CPJ />費爾南德斯的審判和上訴持續了13年，最終於2008年被吉隆坡刑事高等法院裁定無罪。<ref>{{cite web |url=http://www.fidh.org/Ms-Irene-Fernandez-finally |title=Ms. Irene Fernandez finally acquitted! |author= |date=25 November 2008 |work= |publisher=[[International_Federation_for_Human_Rights|International Federation for Human Rights]] |accessdate=30 January 2012}}</ref>

於1996年，顏重慶與四名記者被捕。<ref name=CPJ />，他在監獄中度過之後的五天，並因此獲[[國際特赦組織|國際特赦組織]]稱為「[[良心犯|良心犯]]」。<ref>{{cite web|url=http://www.amnesty.org/en/library/asset/ASA28/014/1996/en/ed449b88-b58d-441c-b7ff-78d9480e1652/asa280141996en.pdf|title=Further information on EXTRA 176/96 (ASA 28/13/96, 12 November 1996) - Prisoners of conscience|author=|date=14 November 1996|work=|publisher=[[Amnesty_International|Amnesty International]]|accessdate=30 January 2012|deadurl=yes|archiveurl=https://web.archive.org/web/20130926094633/http://www.amnesty.org/en/library/asset/ASA28/014/1996/en/ed449b88-b58d-441c-b7ff-78d9480e1652/asa280141996en.pdf|archivedate=2013年9月26日}}</ref>出獄後，他寫了一篇專欄文章。他在文中抗議自己無故被捕，並且在獄中獲得不平等的待遇，但《太陽報》的編輯仍然拒絕發表之。作為回應，顏重慶申請辭職<ref name=CPJ />，並成為[[泰國|泰國]]英語日報《[[国民报_(泰国)|国民报]]》的記者。<ref name=ICIJ>{{cite web |url=http://www.publicintegrity.org/investigations/icij/journalists/profile/791/ |title=ICIJ Journalists: Steven Gan |author= |date= |work= |publisher=[[International_Consortium_of_Investigative_Journalists|International Consortium of Investigative Journalists]] |accessdate=29 January 2012}}</ref>

==当今大马==

雖然馬來西亞政府自1984年起開始嚴格控制印刷媒體，但政府仍然於1995年推行[[多媒体超级走廊|多媒体超级走廊]]計劃促使國家科技發展。該計劃禁止政府介入網絡。顏重慶認為網絡將會是一個突破點，故與詹德蘭合辦一個網上獨立新聞來源。<ref name=PBS>{{cite web |url=http://www.pbs.org/newshour/media/ipf/gan.html |title=Steven Gan, editor Malaysiakini |author= |year=2000 |work= |publisher=''[[PBS_NewsHour|PBS NewsHour]]'' |accessdate=29 January 2012}}</ref>

他們因此於1999年11月創辦了《当今大马》。當時的員工只有6人，且預算僅為$100,000。<ref name=PBS />而顏重慶則擔任其首席主編。《当今大马》報道的第一則新聞是對馬來西亞發行量最大的中文報紙《[[星洲日報|星洲日報]]》做法的批評。根據[[英國廣播公司新聞|英國廣播公司新聞]]指出，《当今大马》的報道使《星洲日報》臭名昭著，更迫使《星洲日報》公開道歉。<ref name=BBC>{{cite web |url=http://news.bbc.co.uk/2/hi/asia-pacific/1028657.stm |title=Malaysia's first online paper |author= |date=20 November 2000 |work= |publisher=[[BBC_News|BBC News]] |accessdate=29 January 2012}}</ref>於2001年4月，《当今大马》報道了關於政府秘密拘留10名政治活動家的新聞。<ref name=BW/>

一年之內，《当今大马》的日均瀏覽量達到100,000人次<ref name=PBS />，使之成為馬來西亞其中一個最受歡迎的新聞網站的。<ref name=NR/>其員工數量亦因此上升至12人。<ref name=BBC />於2012年1月，[[Alexa_Internet|Alexa Internet]]的數據顯示《当今大马》是馬來西亞瀏覽量第14高的網站。<ref>{{cite web |url=http://www.alexa.com/topsites/countries/MY |title=Top Sites in Malaysia |author= |date=29 January 2012 |work= |publisher=[[Alexa_Internet|Alexa Internet]] |accessdate=29 January 2012}}</ref>然而，《当今大马》的做法並不受政府及商界的歡迎，因此其廣告告收入較少。<ref>{{cite web |url=http://www.nytimes.com/2008/01/27/technology/27iht-mag28.1.9524336.html?pagewanted=all |title=Combative Indian magazine struggles to sell 'bad news' |author=Sonia Phalnikar |date=27 July 2008 |work= |publisher=''[[The_New_York_Times|The New York Times]]'' |accessdate=29 January 2012}}</ref>

==政府回應==
馬來西亞首相[[马哈迪·莫哈末|马哈迪·莫哈末]]將《当今大马》的員工稱為“叛徒”<ref name=NR/>，並聲稱該網站是在1997年的[[亞洲金融風暴|亞洲金融風暴]]中打壓馬幣的[[乔治·索罗斯|乔治·索罗斯]]資助的。<ref name=BW/> 

於2002年，顏重慶指出《当今大马》經常被黑客入侵。他他相信黑客是政府派來的。<ref name=NR/>該網站於2011年4月和7月再次被黑客入侵，而這次入侵的黑客也很可能是政府派來的。<ref>{{cite web |url=http://www.nytimes.com/2011/09/09/world/asia/09iht-malaysia09.html?pagewanted=all |title=In Malaysia, Freedom of the (Virtual) Press |author=Liz Gooch |date=8 September 2011 |work= |publisher=''The New York Times'' |accessdate=29 January 2012}}</ref><ref>{{cite web |url=http://utse.bangkokpost.com/news/asia/231793/malaysian-news-portal-crippled-by-cyberattacks |title=Malaysian news portal crippled by cyberattacks |author= |date=13 April 2011 |work= |publisher=''[[The_Bangkok_Post|The Bangkok Post]]'' |accessdate=29 January 2012 |deadurl=yes |archiveurl=https://web.archive.org/web/20130921055012/http://utse.bangkokpost.com/news/asia/231793/malaysian-news-portal-crippled-by-cyberattacks |archivedate=2013年9月21日 }}</ref>

於2003年1月20日，《当今大马》辦工室被警方突擊搜查，並被沒收了網站伺服器和15台新聞中心電腦，總值約150,000馬幣。<ref name=CNN />之後，右翼政黨[[馬來民族統一機構|馬來民族統一機構]]以調查煽動叛亂為由調查了《当今大马》的網站和員工。<ref>{{cite web |url=http://news.bbc.co.uk/2/hi/asia-pacific/2676297.stm |title=Malaysian police raid website office |author= |date=20 January 2003 |work= |publisher=BBC News |accessdate=29 January 2012}}</ref>網站於突擊搜查10小時後重啟。<ref name=CNN>{{cite web |url=http://edition.cnn.com/2003/WORLD/asiapcf/southeast/01/21/malaysia.raid/index.html |title=Malaysia's 'independent voice' back online |author=Dave Brewer |date=21 January 2003 |work= |publisher=CNN |accessdate=30 January 2012}}</ref>

==獎項==
於2000年11月，他獲保護記者委員會授予國際新聞自由獎。<ref name=CPJ>{{cite web |url=http://cpj.org/awards/2000/awards00.php |title=IPF Awards 2000 - Announcement |author= |year=2000 |work= |publisher=[[Committee_to_Protect_Journalists|Committee to Protect Journalists]] |accessdate=29 January 2012}}</ref>該獎項旨在表彰面對攻擊、威脅或監禁時，仍然願意捍衞新聞自由的人。<ref>{{cite web |url=http://cpj.org/awards/ |title=CPJ International Press Freedom Awards 2011 |year=2011 |publisher=Committee to Protect Journalists |accessdate=28 January 2012}}</ref>

於2001年7月，《[[彭博商业周刊|彭博商业周刊]]》將顏重慶列為「民意塑造者亞洲之星」。<ref name=BW>{{cite web |url=http://www.businessweek.com/magazine/content/01_27/b3739102.htm |title=Steven Gan: Editor-in-chief, Malaysiakini  |author= |date=2 July 2001 |work= |publisher=''[[Businessweek|Businessweek]]'' |accessdate=29 January 2012}}</ref>翌年，《當今大馬》獲[[國際新聞學會|國際新聞學會]]頒發[[自由媒體先鋒獎|自由媒體先鋒獎]]。<ref name=ICIJ/>
