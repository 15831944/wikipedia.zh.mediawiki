'''《弥勒会见记》'''是一部由28幕（一个序幕和二十七个正幕）组成的[[佛教|佛教]][[曲艺|说唱]]剧本。

== 内容 ==
《弥勒会见记》的内容大略是：[[婆罗门|婆罗门]][[波婆离|波婆离]]已有一百二十岁高龄，在睡梦中受到了天神启示，想去拜渴[[释迦牟尼|释迦牟尼]]。但自己因年事过高无法亲自前往于是派弟子弥勒同伙伴十六人，代表他渴佛致敬。弥勒在梦中也受到相同的天神启示，便欣然前往。波婆离告诉弥勒等说，释迦牟尼身上有三十二大人相。只要看到三十二相，那便是释迦牟尼，就可以把问题如“顶”和“顶坠”等提出来考验如来。弥勒等前往释迎牟尼那里，果然在佛身上看到了三十二相<ref>{{cite journal |author1=[[季羡林|季羡林]] |title=谈新疆博物馆藏吐火罗文A《弥勒会见记剧本》 |journal=文物 |date=1983 |issue=01 |pages=42-44 |doi=10.13619/j.cnki.cn11-1532/k.1983.01.009}}</ref>。
《弥勒会见记》是一部佛典，内容基本上是小乘的，但是已经有了大乘思想的滥觞。在本书中，菩萨还没有像在大乘中那样的地位。但是，大乘色彩已开始显露，比如佛陀受到了异乎寻常的膜拜，信徒们都期望在非常遥远的未来能够同弥勒相会， 等等<ref name="浅议">{{cite journal |author1=季羡林 |title=吐火罗文和回鹘文本《弥勒会见记》性质浅议 |journal=北京大学学报（哲学社会科学版） |date=1991 |issue=02 |pages=66-72+130}}</ref>。
== 文本 ==
根据吐火罗文本及回鹘文本《弥勒会见记》中的题记，《弥勒会见记》应有三种文本，即[[印度文|印度文]]、[[吐火罗文|吐火罗文]]以及[[回鹘文|回鹘文]]<ref name="研究">{{cite journal |author1=[[康颖宽|康颖宽]] |title=《弥勒会见记》研究 |journal=中国艺术研究院 |date=2014}}</ref>。其中《弥勒会见记》原剧本最初是由印度文完成，后由圣月菩萨大师编译为吐火罗文，之后智护戒师又译为回鹘文<ref name="异本">{{cite journal |author1=[[郑玲|郑玲]] |title=《弥勒会见记》异本对勘研究 |journal=中央民族大学 |date=2013}}</ref>。但到目前为止，印度文原本尚未发现，且印度文为梵文或印度某种俗语不得而知<ref name="浅议" />。

其中吐火罗文《弥勒会见记》有德国本和新博本。回鸦文《弥勒会见记》有哈密本、德国胜金口本和德国木头沟本。其中，哈密本《弥勒会见记》在数量上远远超过了德国本<ref name="异本" />。

== 出土 ==
胜金口本与木头沟本回鹊文《弥勒会见记》于20世纪由德国考古队在新疆吐鲁番地区发现的。其中胜金口本保留下的页数最多，完整的有十几张和一些残卷<ref name="研究"/>。

哈密本回鹊文《弥勒会见记》于1959年4月在哈密县天山公社被发现。残卷共有二百九十三张（五百八十六页）<ref name="浅议" />。

新博本吐火罗文《弥勒会见记》于1974年冬季在焉耆县[[七个星佛寺遗址|七个星佛寺遗址]]被发现。残卷的共有四十四张（八十八页）<ref name="浅议" />。

== 成书年代 ==
对于回鹤文哈密本《弥勒会见记》的成书年代，有三种看法分别是为10-11世纪形成说、8-9世纪形成说、公元767年形成说<ref name="异本" />。

== 文体 ==
关于《弥勒会见记》是否属于剧本引起了国内外学术界的争议，其中某些观点截然对立。关于吐火罗文《弥勒会见记》早期，德国学者[[泽格|泽格]]（E.Sieg） 和[[穆勒|穆勒]]（F.W.K.Müller）都否认它是一个剧本。但之后泽格却承认其为一个剧本。季羡林则认为它是不太成熟的剧本。关于回鹘文本的《弥勒会见记》，[[冯家昇|冯家昇]]认为它是一种变文。而[[耿世民|耿世民]]和[[伊斯拉菲尔·玉素甫|伊斯拉菲尔·玉素甫]]认为它是大型佛教剧本。与之相反的是沈尧认为其不能划入戏剧文学，而是叙事文学中的讲唱<ref name="异本" />。
