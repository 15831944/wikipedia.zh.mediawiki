'''杰外动漫'''（'''JY Animation'''）是[[中国大陆|中国大陆]]的一家从事国内和国外正版动画[[新媒体|新媒体]]发行的机构<ref>[http://jya.cn/about 关于杰外动漫] 杰外动漫 | 官方网站</ref>。

== 概要 ==
* 杰外动漫于2010年11月在北京成立<ref>[http://jya.cn/timeline 公司大事记] 杰外动漫 | 官方网站</ref>。
* 杰外动漫致力于动漫产业链的经营，业务涵盖国内外动漫作品的全权代理。包括：动画片/幼教作品的制作、新媒体和传统媒体发行、电子图书发行、网络游戏合作、新媒体整合营销推广、动漫衍生品设计制造、商品化授权、品牌公关等多元化业务。<ref>{{Cite web |url=http://acg.178.com/201707/295031977705.html |title=专访杰外动漫CEO金鹄：一口气公布了四款原创IP，将打造中国最大的国际化IP经纪机构！ - 178动漫频道 |accessdate=2017-12-28 |work=acg.178.com |language=zh |archive-url=https://web.archive.org/web/20171228232018/http://acg.178.com/201707/295031977705.html |archive-date=2017-12-28 |dead-url=yes }}</ref>同时也有参与部分日本电视动画制作的投资。<ref>{{Cite web |url=http://tech.ifeng.com/a/20171213/44801804_0.shtml |title=一年引进170部，中国新番动画市场出现了三个变化 |accessdate=2017-12-28 |work=tech.ifeng.com}}</ref>
