'''傑尼龜'''（{{lang-ja|'''ゼニガメ'''}}，{{lang-en|'''Squirtle'''}}，香港舊譯'''車厘龜'''）是[[精靈寶可夢|精靈寶可夢]]系列登場的{{NUMBEROFPOKEMON}}種虛構[[角色|角色]]（[[怪物|怪獸]]）中的一種。

== 特徵 ==
傑尼龜為水屬性的小龜[[精靈寶可夢|寶可夢]]，身體為淡藍色，圓圓的龜殼為其特徵，甲殼的功能不只是保護身體，其圓滑的外形及表面的淺溝都能降低水中的阻力，這樣就能游得更快。進化速度極快。加上，他的頭十分圓，龜殼十分的硬。

== [[神奇寶貝遊戲列表|遊戲]]中的傑尼龜 ==
初次登場於精靈寶可夢紅／綠版，豐緣編號為209，全國圖鑑編號為7，代表技能為水槍、縮入殼中及火箭頭錘等。
傑尼龜的進化型為[[卡咪龜|卡咪龜]]，[[電玩|電玩]]中在16級進化，36級進化成水箭龜。

== 動畫中的傑尼龜 ==
[[動畫|動畫]]中小智的傑尼龜曾為壞蛋組織－傑尼龜隊的隊長（最後從良，成為消防隊），且在橘子群島時學會了水炮（{{lang|ja|ハイドロポンプ}}）。而小遙的傑尼龜則是很愛哭，並會使用冰凍光束（{{lang|ja|れいとうビーム}}）。

為關都地區中，給新人訓練家的三隻神奇寶貝之一。
