'''喬瑟夫·傑森·那瑪卡哈·摩莫亞'''（{{lang-en|'''Joseph Jason Namakaeha Momoa'''}}，{{bd|1979年|8月1日||}}）是一名[[美國|美國]]男演員、模特兒、導演、編劇和製片人。曾出演過的作品中較著名的是在軍事科幻電視劇《{{link-en|星際奇兵：亞特蘭提斯|Stargate Atlantis}}》（2005年-2009年）中飾演{{link-en|羅南·迪克斯|Ronon Dex}}，並在[[HBO|HBO]]的奇幻電視劇《[[冰與火之歌：權力遊戲|冰與火之歌：權力遊戲]]》（2011年至2012年）中詮釋卓戈卡奧一角。他還主演了2011年的[[劍與魔法|劍與魔法]]電影《[[王者之劍3D|王者之劍3D]]》《[[水行俠_(電影)|水行俠]]》（2018年）<ref>{{cite news | title=Momoa Set for 'Conan' – New Barbarian Set for Franchise Reboot | date=2010-01-21| author = McNary, David| work = [[Variety_(magazine)|Variety]]| url=http://www.variety.com/article/VR1118014163.html?categoryId=28&cs=1| accessdate= 2011-08-18}}</ref>。

2014年驚悚片《{{link-en|帕洛瑪之旅|Road to Paloma}}》是摩莫亞第一部導演、編劇、監製及主演的電影，並於2014年7月11日在美國上映。摩莫亞於2016年的《[[蝙蝠俠對超人：正義曙光|蝙蝠俠對超人：正義曙光]]》開始在[[DC擴展宇宙|DC擴展宇宙]]中飾演[[水行俠|亞瑟·庫瑞／水行俠]]。

== 早期生活 ==
摩莫亞出生於[[夏威夷州|夏威夷州]]的[[檀香山|檀香山]]，母親柯妮（Coni）是一名攝影師，為出生自[[愛荷華州|愛荷華州]]的{{link-en|諾沃克 (愛荷華州)|Norwalk, Iowa|諾沃克}}人且是[[德國|德國]]、[[愛爾蘭|愛爾蘭]]和[[印第安人|印第安人]]的[[混血兒|混血兒]]，而父親喬瑟夫（Joseph）則是一位畫家，且還是擁有夏威夷州血統的[[原住民|原住民]]<ref>{{cite web |author=Sophia Ahmad |url=http://blogs.desmoinesregister.com/dmr/index.php/2010/02/19/interview-with-norwalk-conan-star-jason-momoa |title=Interview with Norwalk’s ‘Conan’ star Jason Momoa | Des Moines Register Staff Blogs |publisher=Blogs.desmoinesregister.com |date=2010-02-19 |accessdate=2014-08-11 |deadurl=yes |archiveurl=https://archive.is/20130408022415/http://blogs.desmoinesregister.com/dmr/index.php/2010/02/19/interview-with-norwalk-conan-star-jason-momoa |archivedate=2013-04-08 }}</ref>。摩莫亞是家中唯一的孩子。由於母親的關係，摩莫亞從小在愛荷華州的諾沃克成長<ref>[http://www.tvguide.com/celebrities/jason-momoa/bio/190112 Jason Momoa's short Biography] on TVguide</ref>。高中畢業後，他在愛荷華州開始了海洋生物學的學業，就讀期間他都在佛羅里達群島上。後來，他轉到科羅拉多州修野生動物生物學，並搬至夏威夷群島再次見到父親<ref name="katiecouric1">{{cite web |url=http://katiecouric.com/videos/game-of-thrones-jason-momoa/ |title="Game of Thrones" Star Jason Momoa – Katie Couric |publisher=Katiecouric.com |date= |accessdate=2014-08-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20140904033741/http://katiecouric.com/videos/game-of-thrones-jason-momoa/ |archivedate=2014-09-04 }}</ref>。

他也是衝浪手{{link-en|布萊恩·庫拉納|Brian Keaulana}}和{{link-en|路斯帝·庫拉納|Rusty Keaulana}}的侄子<ref name=honsb>Ryan, Tim (1999-05-11).  [http://archives.starbulletin.com/1999/05/11/features/story1.html "On Baywatch Duty"].''[[Honolulu_Star-Bulletin|Honolulu Star-Bulletin]]''.  Retrieved 2011-08-08.</ref>。

== 職業生涯 ==
1998年，摩莫亞被國際設計師菊池武夫（Takeo Kikuchi）發掘，展開了自己的[[模特兒|模特兒]]生涯。他於1999年由夏威夷模特兒主辦的妙齡小姐夏威夷比賽。19歲那年，他還曾在衝浪店兼職，在這之前他成功引入到電視劇《[[海灘遊俠|海灘遊俠]]》之中。為了得到這個角色，他向人騙了自己的模特兒簡歷，包括其他過去的工作<ref name="katiecouric1"/>。摩莫亞在動作電視劇《海灘遊俠》中飾演{{link-en|傑森·約阿內|Jason Ioane}}（1999年-2001年）。他除了參演《{{link-en|北海迷情|North Shore (TV series)}}》（2004年-2005年）外，還在2004年電影《{{link-en|強森一家的幸福之旅|Johnson Family Vacation}}》和電視劇《{{link-en|星際奇兵：亞特蘭提斯|Stargate Atlantis}}》（2005年-2009年）中登場，並於2009年的喜劇《[[橄欖球之戀|橄欖球之戀]]》中出演四集的羅馬一角。後來，他在《[[王者之劍3D|王者之劍3D]]》（2011年）中主演[[蠻王柯南|蠻王柯南]]，本片是翻拍於1982年的同名電影，且還和[[阿諾·史瓦辛格|阿諾·史瓦辛格]]飾演同一角色。摩莫亞獲得了[[HBO|HBO]]電視劇《[[冰與火之歌：權力遊戲|冰與火之歌：權力遊戲]]》的面試機會，並上演了一段毛利人使用在戰場上阻赫的[[哈卡舞|哈卡舞]]<ref>{{cite web|url=http://www.newzealand.com/ca/feature/haka/ |title=The Haka - Dance of War - Maori Haka - New Zealand | Tourism NZ |publisher=New Zealand |date=2014-08-07 |accessdate=2014-08-11}}</ref>。

待《星際奇兵：亞特蘭提斯》第四季的結束，他將自己的[[辮子|辮子]]剪去。摩莫亞為戲而留的辮子讓他感覺有些頭疼，在第五季的「Broken Ties」一集中他將此長髮切除。在拍攝了第5季的第一天，他的辮子被重新接回，但在過程中過於繁重，使得他改以{{link-en|特效替身|Stunt double}}的方式使用假髮，直到兩週後才拿到訂製的假髮<ref>{{cite web | author=Sumner, Darren | url=http://www.gateworld.net/news/2008/05/momoa_the_dreads_stay_put.shtml | title=Momoa: The Dreads Stay Put | publisher=[[Gateworld|Gateworld]] | date=2008-05-26 | accessdate=2011-08-18 | deadurl=yes | archiveurl=https://web.archive.org/web/20080711051533/http://www.gateworld.net/news/2008/05/momoa_the_dreads_stay_put.shtml | archivedate=2008-07-11 }}</ref><ref>{{cite web | author=Sumner, Darren | url=http://www.gateworld.net/news/2008/02/new_plot_details_on_broken_ties.shtml | title=New Plot Details on 'Broken Ties' | publisher=[[Gateworld|Gateworld]] | date=2008-02-23 | accessdate=2011-08-18 | deadurl=yes | archiveurl=https://web.archive.org/web/20110616084801/http://www.gateworld.net/news/2008/02/new_plot_details_on_broken_ties.shtml | archivedate=2011-06-16 }}</ref>。值得一提的是，當他第一次詮釋角色時，監製要求他將長髮剪掉，但這卻成為了他角色的象徵，於是製片方允許摩莫亞將長髮留住。

== 個人生活 ==
摩莫亞曾四處旅行，如在[[巴黎|巴黎]]拿起蠟筆畫畫、在[[西藏|西藏]]學習佛法<ref>{{cite web |url=http://ca.movies.yahoo.com/person/jason-momoa/biography.html |title=Jason Momoa Biography - Yahoo Movies Canada |publisher=Ca.movies.yahoo.com |date= |accessdate=2014-08-11 |deadurl=yes |archiveurl=https://archive.is/20140304051819/http://ca.movies.yahoo.com/person/jason-momoa/biography.html |archivedate=2014-03-04 }}</ref><ref>{{cite web|url=http://www.starpulse.com/Actors/Momoa,_Jason/Biography/ |title=Jason Momoa Biography |publisher=Starpulse.com |date= |accessdate=2014-08-11}}</ref>。

2005年，經過共同朋友的介紹下摩莫亞認識了女演員{{link-en|麗莎·博內特|Lisa Bonet}}。2007年7月23日，他和博內特有了女兒蘿拉·伊奧拉尼·莫瑪（Lola Iolani Momoa）。他與博內特後來生了一個兒子諾加-沃夫·曼庫帕·那卡哈·摩莫亞（Nakoa-Wolf Manakauapo Namakaeha Momoa），是於2008年12月15日出生的。摩莫亞的母親在摩莫亞的官方網站上的粉絲論壇公布，她說「Nakoa」的意思是「勇士」（儘管這詞代表著是「勇氣」），而「Kaua（雨）」和「Po（黑暗）」則取於他出生的那天（他晚上出生在洛杉磯時有著罕見的惡劣天氣）。中間名「Namakaeha（那卡哈）」則是取自他的父親<ref>{{cite web|author = Staff | date = 2009-01-09 | url=http://today.msnbc.msn.com/id/28584888/ns/today-entertainment/t/lisa-bonets-new-babys-name-mouthful|title=Lisa Bonet's New Baby's Name Is a Mouthful – Newborn Boy's Name Is Nakoa-Wolf Manakauapo Namakaeha Momoa|publisher = ''[[Access_Hollywood|Access Hollywood]]'' (via [[MSNBC|MSNBC]]) | accessdate = 2011-08-18}}</ref>。

雖然人們普遍認為摩莫亞和博內特是於2007年11月15日結婚，但這對夫妻直到2017年10月才真正地正式結婚<ref name=“UsWeekly(2017-11-02)”>{{Cite news|url=https://www.usmagazine.com/celebrity-news/news/jason-momoa-lisa-bonet-officially-marry-in-secret-wedding/|title=Jason Momoa, Lisa Bonet Officially Marry in Secret Wedding|date=2017-11-02|work=Us Weekly|access-date=2017-11-02|language=en-US}}</ref>。由於摩莫亞和博內特結婚，使他成為了[[柔伊·克拉維茲|柔伊·克拉維茲]]的繼父<ref>{{Cite web|url=http://www.huffingtonpost.com/2014/07/09/jason-momoa-lisa-bonet_n_5570751.html |title=Jason Momoa Gushing About Wife Lisa Bonet Couldn't Be More Adorable|last=Live|first=Emily Tess Katz HuffPost|date=2014-07-09|website=The Huffington Post|access-date=2016-06-30}}</ref>。

2008年11月15日，摩莫亞在[[加利福尼亞州|加利福尼亞州]][[好萊塢|好萊塢]]的一間小酒館中被破碎的啤酒杯弄傷了臉<ref name="NBC1">{{cite web | author = Staff | title=Man Pleads Not Guilty To Assaulting 'Stargate Atlantis' Actor |url=http://www.nbclosangeles.com/news/entertainment/Man-Pleads-Not-Guilty-to-Assaulting-Stargate-Atlantis-Actor.html |publisher = [[KNBC|KNBC]] |date=2009-01-26 |accessdate= 2011-08-18}}</ref><ref name=latimes>{{cite web|url=http://latimesblogs.latimes.com/lanow/2009/10/5year-sentence-for-man-who-attacked-stargate-atalantis-actor-who-required-140-stitches.html|title=Winton, Richard ''Los Angeles Times'' October 7, 2009|author=|date=|publisher=|accessdate=2014-10-15}}</ref>。他事後接受了140針的手術，這使得在他後來的作品中能明顯看出其疤痕<ref name="NBC1"/>。

摩莫亞也是專業的澳式足球俱樂部{{link-en|阿德雷德港足球俱樂部|Port Adelaide Football Club}}的粉絲<ref>{{cite web|url=http://thirat-atthiraride.tumblr.com/post/97463937249/i-wasnt-going-to-see-jason-again-but-i-found-out|title=Thirat Atthiraride|author=|date=|publisher=|accessdate=2014-10-15}}</ref>。
