'''杰里米·托马斯·坎普'''（{{lang-en|Jeremy Thomas Camp}}，生于1978年1月12日）<ref>{{cite web |url=http://www.mtv.com/lyrics/camp_jeremy/longing_heart/3212798/lyrics.jhtml |title=Longing Heart Lyrics |publisher=[[MTV.com|MTV.com]] |accessdate=October 13, 2008}}</ref>是一名来自[[拉斐特_(印第安纳州)|拉斐特 (印第安纳州)]]的[[当代基督教音乐|当代基督教音乐]]家。坎普已发行了七张专辑，其中四张经[[美国唱片业协会|美国唱片业协会]]认证为[[金唱片|金唱片]]，还有两张为[[现场唱片|现场唱片]]，并有17首冠军单曲<ref name="jfh dec3">{{cite web |url=http://www.jesusfreakhideout.com/news/2008/12/03.JEREMY%20CAMPS%20ALBUM%20CHARTS%20ON%20BILLBOARD%20200%20WITH%20PERSONAL%20BEST%20DEBUT.asp |title=Jeremy Camp's Album Charts On Billboard 200 with Personal Best Debut |publisher=[[Jesus_Freak_Hideout|Jesus Freak Hideout]] |date=December 3, 2008 |accessdate=December 12, 2008}}</ref>。他起先的音乐类型是民谣和摇滚的混合物。

坎普也是一名受以圣职的[[牧师|牧师]]<ref name="CCM_29_8">{{cite journal |last=Cowart |first=Jeremy |title=Love Beyond Measure |journal=[[CCM_Magazine|CCM Magazine]] |id={{ISSN|1524-7848}} |volume=29 |issue=8 |pages=32–35 |date=February 2007}}</ref>。

== 早年生活 ==
杰里米·坎普于1978年1月12日生于[[拉斐特_(印第安纳州)|拉斐特 (印第安纳州)]]<ref name="allmusic">{{cite web |url=http://www.allmusic.com/artist/jeremy-camp-p542070 |title=Jeremy Camp Biography |publisher=[[Allmusic|Allmusic]] |accessdate=October 13, 2008 |author=Mark Deming}}</ref>。其父于“收获礼拜堂”（一个位于拉斐特的[[各各他礼拜堂教会|各各他礼拜堂教会]]的教堂）当牧师侍奉神，并曾教他儿子如何弹吉他。坎普完成高中学业之后进入南加州的一所圣经学院学习了两年。他亦有两个弟兄与一个妹妹。
