'''几内亚共和國'''（{{lang-fr|'''République de Guinée'''}}） 通稱'''几内亚'''，是位於[[西非|西非]]的國家，東南接[[科特迪瓦|科特迪瓦]]，南臨[[利比里亞|利比里亞]]，西及[[塞拉利昂|塞拉利昂]]，北鄰[[几内亚比索|几内亚比索]]、[[塞内加尔|塞内加尔]]，北和東北與[[馬里|馬里]]接壤。[[尼日尔河|尼日尔河]]、[[塞内加尔河|塞内加尔河]]、[[冈比亚河|冈比亚河]]在几内亚发源。几内亚本来是指非洲[[撒哈拉沙漠|撒哈拉沙漠]]以南，[[几内亚湾|几内亚湾]]以北的整个地区，它来源于[[柏柏尔语|柏柏尔语]]，意思是“黑人的国家”。

== 历史 ==
{{main|幾內亞歷史}}
在今天的几内亚境内过去曾有过许多帝国，它们兴旺又覆灭。1235年桑迪亚达·凯塔建立的[[马里帝国|马里帝国]]消灭了过去的索索帝国。[[马里帝国|马里帝国]]后来被从北方入侵的人消灭。1591年[[摩尔人|摩尔人]]入侵今天的几内亚。

从1735年到1898年穆斯林在几内亚中部建立了一个拥有宪法的国家。欧洲人从16世纪到19世纪在海岸地区从事[[奴隶贸易|奴隶贸易]]。从1850年开始法国开始对几内亚殖民，但尤其在内陆遭到顽强抵抗。

今天的几内亚的前身是一个法国于1890年建立的殖民地。首都[[科纳克里|科纳克里]]也是1890年创建的。

1958年几内亚通过全民公投，成为唯一選擇不加入[[法蘭西共同體|法蘭西共同體]]而独立的[[法国殖民地|法国殖民地]]，[[艾哈迈德·塞古·杜尔|艾哈迈德·塞古·杜尔]]成为[[几内亚共和国|几内亚共和国]]的第一位总统。

1965年，几内亚总统杜尔以“干涉我国内政”为由，与[[宗主国|宗主国]]法国断绝关系，直至1975年复交。

1970年11月[[葡萄牙|葡萄牙]]入侵几内亚，试图支持流放的几内亚人建立一个新政府，但是失败。

1978年12月，改国号为「'''几内亚人民革命共和国'''」（至1984年4月）。后复名「'''几内亚共和国'''」。

1984年3月26日杜尔逝世，[[兰萨纳·孔戴|兰萨纳·孔戴]]在一个[[军事委员会|军事委员会]]的支持下上台，建立[[第二共和国|第二共和国]]。

1990年在邻国[[利比里亚|利比里亚]]和[[塞拉利昂|塞拉利昂]]发生[[塞拉利昂内战|内战]]，许多难民逃往几内亚。有时甚至有70多万难民在几内亚境内。直到2004年几内亚境内依然有约15万难民。

1993年12月19日几内亚进行第一次民主选举，原[[军政府|军政府]]领导人兰萨纳·孔戴在这次有争议的选举中当选为国家总统。次年1月[[第三共和国|第三共和国]]宣布成立。国内的不安形势有所减轻，但未完全停止。1996年2月一次军事政变未成功。1998年12月18日兰萨纳·孔戴以54%的选票再次当选，反对党指責这次选举舞弊，数日后反对党首领被捕。

从2000年9月到2001年3月几内亚成功地阻挡了塞拉利昂和利比里亚叛军的入侵。

2001年11月通过一次有争议的全民公投修改宪法，使孔戴可以在2003年后继续当任国家总统。2007年2月9日，孔戴任命[[欧仁·卡马拉|欧仁·卡马拉]]为政府总理<ref>[http://news.xinhuanet.com/world/2007-02/10/content_5722787.htm 几内亚总统任命新总理]</ref>，遭到工会及反对派反对；2月26日，孔戴改任[[兰萨纳·库亚特|兰萨纳·库亚特]]为政府总理<ref>[http://news.xinhuanet.com/world/2007-02/27/content_5778363.htm 几内亚总统任命新总理]</ref>。

2008年12月23日，幾內亞政府官員宣布，幾內亞總統蘭薩納·孔戴22日晚在首都科納克裏的醫院因病去世，終年74歲。有消息稱，幾內亞軍人隨即發動政變，幾內亞政府和議會已被解散。

當天，一名叫穆薩·卡馬拉的上尉在幾內亞國家廣播電臺發表講話說，幾內亞從即日起停止一切政治和工會活動，同時成立一個由軍方和民間代表組成的協商委員會，暫時管理國家事務<ref>[http://news.xinhuanet.com/world/2008-12/23/content_10550114.htm 几内亚总统病逝 军官发动政变]</ref>。

== 地理 ==
{{main|幾內亞地理}}
几内亚位于[[非洲|非洲]]西部，北邻[[几内亚比绍|几内亚比绍]]、[[塞内加尔|塞内加尔]]和[[马里|马里]]，东接[[科特迪瓦|科特迪瓦]]，南界[[塞拉利昂|塞拉利昂]]和[[利比里亚|利比里亚]]，西濒[[大西洋|大西洋]]，海岸线长352公里,面积约24.6万平方公里。地形复杂，全境分4个自然区：
* 西部（称下几内亚或沿海几内亚）面积36,133平方公里，为狭长的沿海平原。全区多沼泽，气温湿热，全年6-8个月为雨季，年降水量可达5000毫米，是主要经济作物区。该区大面积淤陷湿地为[[红树林|红树林]]区。 
* 中部（中几内亚）为平均海拔900米的富塔贾隆高原，西非3条主要河流——[[尼日尔河|尼日尔河]]、[[塞内加尔河|塞内加尔河]]和[[冈比亚河|冈比亚河]]均发源于此，被称为“西非水塔”。
* 东北部（上几内亚）面积97000平方公里，为平均海拔约300米的台地。全区地势平坦，多灌乔木树林。布勒地区的黄金闻名于世。主要经济活动为畜牧业。
* 东南部（称森林几内亚）为几内亚高原，面积49 374平方公里，平均海拔500-600公尺，有大片热带原始森林，有海拔1752米的[[宁巴山|宁巴山]]，为全境最高峰。
几内亚沿海地区为热带季风气候，内地为热带草原气候。年降水量内陆为1000～1500毫米，沿海达4000毫米。

== 行政区划 ==
{{main|幾內亞行政區劃}}

几内亚全国划分为首都科纳克里市和七个行政区（REGION），行政区下辖省（PREFECTURE），省下分为专区（SOUS-PREFECTURE），基层行政单位为自然村或城市街道。
* [[金迪亞大區|金迪亚区]]：[[金迪亚省|金迪亚省]]、[[科亚省|科亚省]]、[[杜布雷卡省|杜布雷卡省]]、[[福雷卡里亞省|弗雷卡里亚省]]、[[泰利梅萊省|特里梅莱省]]
* [[博凱大區|博凯区]]：[[博凯省|博凯省]]、[[博法省|博法省]]、[[弗里亚省|弗里亚省]]、[[加瓦爾省|高瓦尔省]]、[[孔达拉省|孔达拉省]]
* [[拉貝大區|拉贝区]]：[[拉贝省|拉贝省]]、[[库比亚省|库比亚省]]、[[莱卢马省|莱鲁马省]]、[[馬里省|马利省]]、[[圖蓋省|杜盖省]]
* [[馬木大區|马木区]]：[[马木省|马木省]]、[[达拉巴省|达拉巴省]]、[[皮塔省|皮塔省]]
* [[法拉納大區|法拉纳区]]：[[法拉纳省|法拉纳省]]、[[达波拉省|达波拉省]]、[[丁吉拉伊省|丁基拉耶省]]、[[基西杜古省|吉西杜古省]]
* [[恩澤雷科雷大區|恩泽雷科雷区]]：[[恩泽雷科雷省|恩泽雷科雷省]]、[[贝拉省|贝拉省]]、[[盖凯杜省|盖凯杜省]]、[[洛拉省|洛拉省]]、[[馬桑塔省|马桑达省]]、[[約穆省|尤木省]]
* [[康康大區|康康区]]：[[康康省|康康省]]、[[凯鲁阿内省|凯鲁阿内省]]、[[库鲁萨省|库鲁萨省]]、[[芒賈納省|芒加纳省]]、[[錫吉里省|西基里省]]

主要城市（2005年1月1日统计）有：科纳克里（187万人口）、恩泽雷科雷（13万人口）、金迪亚（12万人口）和康康（11万人口）。

== 人口 ==
几内亚全国有20多个部族，其中较大的有三个：[[富拉人|富拉人]]（Peul）约占全国人口的40％以上；马林凯人（Malinke，又称马宁加人）约占全国人口的30％以上；苏苏人（Soussou）约占16％。

几内亚最重要的宗教是[[逊尼派|逊尼派]][[伊斯兰教|伊斯兰教]]。全国约87%的居民信奉[[伊斯兰教|伊斯兰教]]，5％信奉[[天主教|天主教]]，其余信奉[[拜物教|拜物教]]。

官方语言是[[法语|法语]]，但此外还有许多当地部落的语言被使用。因为几内亚穆斯林自幼学习[[阿拉伯语|阿拉伯语]]，且操阿拉伯语的黎巴嫩人在几内亚占据着商业领域的半壁江山，因此阿拉伯语也较流行。

2011年7月估計的人口总数为10,601,009 

年龄结构：
* 0-14岁：42.5%
* 15-64岁：54%
* 65岁以上：3.5%

人口增长率为2.645%（2011年）
华人在当地的数量没有准确统计，估计在2万人左右，主要集中在首都科纳克里。

== 经济 ==
[[File:Conakry-1970momunent.JPG|thumbnail]]
[[File:Guinee_Fouta_Djalon_Doucky.jpg|thumbnail]]
{{main|幾內亞經濟}}

几内亚是联合国公布的最不发达国家之一。经济以[[农业|农业]]、[[矿业|矿业]]为主，[[工业|工业]]基础薄弱。粮食不能自给。[[铝礬土|铝礬土]]、[[咖啡|咖啡]]、[[可可|可可]]和[[橡胶|橡胶]]是几经济的主要支柱，但经济作物开发规模不大，难以同西非其他农业强国竞争。出口品种有限，国际收支失衡，绝大多数消费品完全依赖进口。主要进口商品为燃料，采矿和建筑设备，大米，[[水泥|水泥]]，化工产品，电气设备等。主要出口商品为[[铝矾土|铝矾土]]，[[黄金|黄金]]，[[氧化铝|氧化铝]]。

== 与中国的关系 ==
{{主條目|中國-幾內亞關係}}
几内亚于1959年10月4日同[[中華人民共和國|中華人民共和國]]建交，是[[撒哈拉以南非洲|撒哈拉以南非洲]]国家第一个同中華人民共和國建交的国家。

== 教育 ==
几内亚的[[文盲|文盲]]率高达70%，小学入学率为50%，中学入学率只有10%，大学入学率低于1%。

几内亚有多所大学，主要在首都[[科納克里|科纳克里]]，外省[[康康_(几内亚城市)|康康]]也有。
