'''袍哥会'''，即'''哥老会'''，与[[洪门|洪门]]、[[青帮|青帮]]为[[清朝|清朝]]三大[[秘密结社|秘密结社]]，袍哥会的儀式、章程等，與洪门相當類似，袍哥会在清代初期的四川曾经是少部分人的秘密结社，后期与[[湘军|湘军]]裁卒关系密切，清末成為半公開的組織，在[[辛亥革命|辛亥革命]]之后，它长期成为[[四川|四川]]大多数成年男性都直接加入或间接受其控制的公开性组织。袍哥会对[[四川|四川]]、[[重庆|重庆]]、[[湖南|湖南]]社会各方面都有极为重要的影响，这一特点是其他地区未有过的。[[邓小平|邓小平]]的父亲[[邓文明|邓文明]]就是四川袍哥会的成员。

在四川（以下四川均含重庆地区）的哥老会成员被称为'''袍哥'''，有两种解释，一说是取《[[诗经|诗经]]·无衣》:“与子同袍”之义，表示是同一袍色之哥弟；另一说是袍与胞谐音，表示有如同胞之哥弟。两种解释相差無幾。

== 产生 ==
[[啯噜|啯噜]]，或作[[哥老|哥老]]、[[哥佬|哥佬]]，原是[[方言|方言]]，意指「[[结拜|结拜]][[兄弟|兄弟]]」或「[[賭|賭]]徒」。是自[[清朝|清朝]][[乾隆|乾隆]]以后在[[四川|四川]]、[[陕|陕]]南、[[湘|湘]]西、[[鄂|鄂]]西、[[贵州|贵州]]、[[云南|云南]]中出现的以劫夺谋生的异姓[[结拜|结拜]]团体；啯噜具有分散性，各群之间没有联系，成员被称为'''啯噜子'''。 《[[清史稿|清史稿]]》：「[[蜀|蜀]]民失業無賴者，多習拳勇，嗜飲博，寖至劫殺，號為嘓嚕子。」

这些啯噜[[游民|游民]]团伙，主要从事[[赌博|赌博]]、[[搶劫|搶劫]]、[[綁票|綁票]]、[[謀杀|謀杀]]、[[强奸|强奸]]、[[人口販賣|人口販賣]]或者逼良[[賣淫|為娼]]等各种[[犯罪|犯罪]]活动，早期，无明確的[[政治目标|政治目标]]和[[宗教|宗教]]信仰，後期的啯噜受到[[洪門|洪門]]传入的[[反清復明|反清復明]]思想影響，演變為具有[[四川|川]]、[[湖廣|湖]]特色的[[天地會|天地會]]次團體[[哥老会|哥老会]]。

[[嘉庆|嘉庆]]年间，[[四川|四川]]、[[湖北|湖北]]等地[[白莲教|白莲教]]起义时，大批啯噜子加入，有的还成为主力和首领。随着[[白莲教|白莲教]]起义的失败，啯噜在各地都遭到镇压，但仍然在各地活动，还有不少人渗入到清军与[[衙役|衙役]]之中。为了加强他们内部的组织联系，提高其生存、发展和与[[官家|官家]]对抗的能力，他们逐步大量吸收了早已流传入川的[[天地会|天地会]]的组织形式与联络办法，或直接与[[天地会|天地会]]徒融为一家，在[[道光|道光]]年间发展成为[[帮会|帮会]]组织[[哥老会|哥老会]]。

哥老会成立的标志的开山堂，即一个哥老会独立组织的成立仪式。最早在[[四川|四川]]的开山堂一般被认为是[[清|清]][[道光|道光]]二十八年（1848年）[[永宁|永宁]]（今[[叙永|叙永]]）[[郭永泰|郭永泰]]开荩忠山，始有山水香堂，以[[延平郡王|延平郡王]]招讨大将军印信相号召。据后来袍哥首领人物的记述：“会盟者即达四千余人，不久荩忠山[[兄弟|兄弟]]遍及各[[行省|行省]]，而开山立堂者尤以[[四川|四川]]为最。”

哥老会产生之后，在各地迅速发展，[[中國南方|中國南方]]多數省份和[[中國北方|中國北方]]部分省份都有它的组织，从总的来看，哥老会的发展有三个特点：

# [[四川|四川]]数量最多，川东尤其是[[重庆|重庆]]力量最大。
# 在[[湘军|湘军]]等[[團練|團練]]营中发展极快。
# 哥老会虽然分布面广，但仍有很强烈的反清情绪，[[咸丰|咸丰]]年间爆发的历时6年、以[[四川|四川]]为主战场的[[李永和|李永和]]、[[蓝朝鼎|蓝朝鼎]](另参见[[二郎场之戰|二郎场之戰]])[[李蓝农民起义|李蓝农民起义]]的基本力量就是哥老会。以后的历次以反洋教为中心的反清[[教案_(宗教)|教案]]的主力也是哥老会，如“一绅二粮三袍哥”，就是[[四川|四川]]最大教案[[大足教案|大足教案]]的主力。

== 成员 ==
最初的袍哥会成员大多是社会底层人员，[[太平天国|太平天国]]运动与[[李蓝起义|李蓝起义]]失败之后，社会更加动荡，大批的[[流民|流民]]與退伍的[[團練|團練]]軍人加入袍哥会，无钱无势者当袍哥为了求得结援互助。有钱有势的[[門閥|門閥]][[地主|地主]]，为了控制地方，培植羽翼，也积极加入袍哥队伍。在这种风潮之下，不入袍哥会就很难在社会上立足。

===五堂與分類===
由于参加人数越来越多，袍哥会在[[四川|四川]]的各山堂逐渐形成五个有高低之分的堂口，即仁、义、礼、智、信五个班辈，参加仁字堂者以士绅为多；义字堂以商人为主；礼字堂则较多匪盗、地痞和士兵；智字堂多为贫苦农民、手工业者、船夫、车夫；信字堂人数不多，是下九流者，如卖唱、搓澡之流。[[袍哥|袍哥]]一般都不以此为职，但也有少数组织者或骨干以此为职业。还有一部份乡镇山区的袍哥会完全就是[[绿林|绿林]]作风，占山为王，以抢劫为生，这一类被称为'''浑水袍哥'''，另外那些不搞盗、抢等以武力掠夺他人财物的袍哥被称为'''清水袍哥'''。从人数上说清水袍哥占大多数。

===规模===
从[[清朝|清朝]]中期开始[[哥老会|哥老会]]在[[四川|四川]]开山立堂开始，清朝政府都把它视作反清、反社会团体，严加打击，因此人员并不是很多，到了[[清朝|清朝]]末期，因社会动荡且政府对底层的控制力日渐减弱，袍哥会人员迅速增加，最后到了半公开的地步。[[民国|民国]]时期，因袍哥会在四川[[保路运动|保路运动]]发挥了积极作用，再加上[[民国|民国]]初期，时局混乱，袍哥完全成了公开组织。发展到最后的1940年代末，据估计全川成年男子有30-50%参加了袍哥会（也有人估计川东地区，重庆占70-80%）。

== 组织形式 ==

=== 称呼 ===
袍哥会的最初的联络的聚点为'''山头'''或'''香堂'''，成立一个袍哥会组织就叫'''开山立堂'''，山头和香堂的名称就作为袍哥会一个组织的称呼，如'''莲花山富贵堂'''、'''[[峨眉山|峨眉山]]顺德堂'''。后来随着参加人员的逐步增多，聚会场所的变化，袍哥会的组织又改称为'''码头'''、'''公口'''、'''社'''等。

=== 地位 ===
袍哥会没有全国性的组织，甚至没有地区性的组织，每一个袍哥会组织都是独立存在的，没有依存、隶属关系，但会经常有人员联系。因此袍哥会的数量众多，一般一个[[乡|乡]]或[[城市|城市]]的一条[[街道|街道]]就有一个，甚至还更多，如1949年[[成都|成都]]有公口（包括分社、支社）1000多个，[[重庆|重庆]]有公口300多个，然而重庆公口下辖的袍哥组织接近整个街道，总人数接近成都每个公口的10倍，占重庆市区男性居民的2/3之多。

到了[[民国|民国]]后期曾经有些实力较大的袍哥试图进行合并，达到地区性的统一，如曾由[[国民政府|国民政府]]社会部出面支持，重庆袍哥各界成立了'''国民自强社'''，成都相仿成立了'''国民互助社'''，但都是只有名义而已，实际仍是独立存在、活动的。

=== 内部等级 ===
袍哥组织的内部排行分五个等级，分别称为头排、三排、五排、六排、十排。
* 头排：头排大哥即舵头，也称舵把子、社长。另有闲位大哥，亦如一般社会组织的名誉理事，多为有声望的人，也有绅、商依靠袍哥关系便于在社会上活动的，他们挂个名，赞助若干钱取得“大爷”资格，俗称绅夹皮。
* 三排：又称三哥、钱粮。掌管一社经济及经营的商业（如[[茶馆|茶馆]]、[[赌场|赌场]]、[[栈房|栈房]]）。
* 五排：又称五哥、管事、红旗大管事行交际、执法等职，在袍哥中最有社会力量，不少为职业袍哥，也有绅夹皮五哥、闲五。
* 六排：副六也可称五哥，是一般成员。[[绿林|绿林]]则称蓝旗，是负责巡风探事的小头领。 
* 十排：统称老么，有凤尾老么、执法老么、跑腿老么之分。凤尾老么是有家资的年轻后生，可“一步登天海大哥”。执法老么多为流氓凶神，袍哥传堂把守辕门，制裁叛徒充当杀手就是此辈。跑腿老么如茶堂馆、[[赌场|赌场]]杂务均是。一般老么则是新人组织者。 

排行中无二、四、七、八、九。二是不敢僭越[[关帝君|关帝君]]（即关羽，因為《[[三國演義|三國演義]]》的關係，民间称关羽为关二哥、关二爷）。四和七是[[洪門|洪門]]所忌諱的叛徒之數（[[洪門|洪門]]中四、七兩排只能由女性擔任，已婚的稱四姊，未婚的稱七妹）。八、九忌[[杨家将|杨家将]]八姐九妹之称。

===三大会===
袍哥一年集会三次：最隆重的一次[[农历|农历]]五月十三的'''单刀会'''（祭[[关羽|关羽]]）。七月称[[中元|中元]]会，为已故弟兄设祭。[[腊月|腊月]]称吃[[团年饭|团年饭]]。实则都是为吸收新成员，商谈一些有关香堂内部内部及对外事务。新参加袍哥得有人引荐，在香堂上行叩拜礼，听舵爷赏封“排行座次”，之后就是袍哥了。

== 历史事件 ==

=== 保路运动 ===
{{main|保路运动}}
1911年5月9日，[[清政府|清政府]]接受[[盛宣怀|盛宣怀]]的建议，收回地方筑路权，宣布铁路国有，将集股商办的[[川汉铁路|川汉铁路]]路权拱手让给外国列强。由于[[四川|四川]]人民对于清政府的股权赔偿不满，反清情绪迅速高涨，[[四川|四川]]的[[保路运动|保路运动]]在极短期内如火如荼地迅猛展开。两个月中，全川大多数州县建立了保路同志会，其中有[[同盟会|同盟会]]，有立宪派，但最主要的力量是各地的[[袍哥|袍哥]]，[[同盟军|同盟军]]所依靠的力量，也就是袍哥与[[新军|新军]]，四川[[同盟会|同盟会]]负责人[[熊克武|熊克武]]、[[尹昌衡|尹昌衡]]、[[杨庶堪|杨庶堪]]、[[吴玉章|吴玉章]]等人也都加入了袍哥。1911年8月4日，川西各地袍哥大爷到资州（今[[资中|资中]]）罗泉井召开袍哥'''攒堂大会'''，商议保路反清事宜，决定将保路同志会改为[[保路同志军|保路同志军]]，在川西地区举行反清大起义。9月7日[[成都血案|成都血案]]爆发后，以袍哥会为主保路同志军开始全省武装起义。11月22日[[重庆|重庆]]宣布独立，成立[[蜀军|蜀军]]政府，[[张培爵|张培爵]]被推为蜀军[[都督|都督]]，[[夏之时|夏之时]]为副都督。12月8日[[成都|成都]]兵变之后，各路同志军进入成都，[[尹昌衡|尹昌衡]]改组[[四川|四川]]军政府，由[[尹昌衡|尹昌衡]]、[[罗纶|罗纶]]分任正副都督，至此，清政府在[[四川|四川]]的统治宣告结束。

[[保路同志军起事|保路同志军起事]]成功后，[[四川|四川]]各地成立了大量的袍哥组织，此时的[[四川|四川]]，已经成了袍哥的世界。

=== 护国战争 ===
[[袁世凯|袁世凯]]迫使[[宣統退位|宣統退位]]之后，就任[[中华民国|中华民国]]临时大总统。1915年12月12日[[袁世凯|袁世凯]]称帝，改[[中华民国|中华民国]]为[[中華帝國_(國號)|中華帝國]]。前[[云南|云南]]督军[[蔡锷|蔡锷]]、云南将军[[唐继尧|唐继尧]]于1915年12月25日在[[昆明|昆明]]宣布[[云南|云南]]独立，并即组识约二万人之[[护国军|护国军]]讨伐[[袁世凯|袁世凯]]，分由[[蔡锷|蔡锷]]、[[李烈钧|李烈钧]]、[[唐继尧|唐继尧]]任一、二、三军总司令；[[唐继尧|唐继尧]]兼任[[云南|云南]]都督府都督。[[袁世凯|袁世凯]]调派八万人进攻云南，在[[四川|四川]]被[[蔡锷|蔡锷]]大败。在这次护国战争中，同盟会方面又再次拉拢、依靠[[袍哥|袍哥]]，[[袍哥|袍哥]]武装再次集聚，在各地作战。'''四川护国军'''司令部就设在[[温江|温江]]、[[崇州|崇州]]、[[双流|双流]]三县交界的[[袍哥|袍哥]]码头[[刘家濠|刘家濠]]，司令、副司令、参谋长都是原同志军首领、袍哥大爷。

== 消亡 ==
[[国共战争|国共战争]]末期，[[国民政府|国民政府]]希望借助[[四川|四川]]特殊的地形延续政权，就如[[抗日战争|抗日战争]]时一样，把[[首都|首都]]从[[南京|南京]]迁移到[[重庆|重庆]]、[[成都|成都]]，后来又迁到[[西昌|西昌]]，试图在[[四川|四川]]与[[共产党|共产党]]部队抗争下去。大多数[[袍哥|袍哥]]大爷都参加了[[国民政府|国民政府]]所组织抵抗[[共产党|共产党]]武力進攻的部队，五期“游击干部训练班”的学员大多是[[袍哥|袍哥]]，各地成立的'''[[反共救国军|反共救国军]]'''、'''游击挺进军'''之类基本上全是袍哥武装。[[中国人民解放军|解放军]]奪下[[四川|四川]]之后不久，1950年2月，[[国民党|国民党]]残余力量在川西各地组织了一次反攻，其主要力量也都是[[袍哥|袍哥]]。后来被[[共产党|共产党]]作为典型批判的大地主[[刘文彩|刘文彩]]的弟弟[[刘文辉|刘文辉]]组织策划执行了这次反攻，并且拱手交给了[[共产党|共产党]]。[[刘文彩|刘文彩]]就是当时当地[[哥老会|哥老会]]的主要干部。

[[中华人民共和国|中华人民共和国]]成立后，中国政府取缔一切民间黑社会组织以及非法社团，并且对抵抗[[解放军|解放军]]的武装袍哥会进行镇压，其一百多年的风光历史就此基本结束。

=== 茶馆 ===
在袍哥会中[[茶馆|茶馆]]是重要活动场所。各个山头多称XX公，XX社，各占一方“码头”，设茶馆为联络地，也就是香堂所在地，集会多在茶馆中举行。执事袍哥每天必去茶馆处理组织有关事务，接待江湖朋友，称为把水口。茶馆也是袍哥平理、调解纠纷的场所。袍哥会的每年的三大集会也在茶馆举行。
