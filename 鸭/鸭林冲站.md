'''鸭林冲站'''，为[[中国|中国]][[安徽省|安徽省]][[合肥市|合肥市]]的一座[[地鐵車站|地铁车站]]，工程站名'''新蚌埠路站'''，位于[[北二环路_(合肥市)|北二环路]]与[[铜陵北路|铜陵北路]]交叉口，为[[合肥轨道交通3号线|合肥轨道交通3号线]]站点。<ref>{{cite web |title="图书馆"易误解 合肥市民为地铁3号线站名“挑刺”|url=http://www.ahwang.cn/hefei/20190125/1848031.shtml |publisher=安徽网 |accessdate=2019-01-25}}</ref>

== 车站结构 ==
[[竹丝滩站|竹丝滩站]]位于新蚌埠路与站前路交叉口，沿站前路东西向布置，为地下两层单柱两跨岛式站台。共设4个出入口，2组风亭。<ref>{{cite web |title=合肥地铁3号线沿线站点公布！开通运营时间定在 |url=http://wemedia.ifeng.com/44596356/wemedia.shtml |publisher=凤凰网安徽 |accessdate=2019-02-01}}</ref>
