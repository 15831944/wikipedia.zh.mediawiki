'''鸭绿江'''（[[满语|满语]]：{{ManchuSibeUnicode|ᠶᠠᠯᡠ ᡠᠯᠠ}} {{Unicode|''Yalu ula''}}；[[汉语拼音|汉语拼音]]：{{Unicode|Yā lǜ jiāng}}，{{kor|諺=압록강|漢=鴨綠江|羅=Amnokkang|f=y}}）是位于[[中华人民共和国|中华人民共和国]]和[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]之间的一条[[界河|界江]]。[[入海口|入海口]]地区，朝方岛屿——绸缎岛和[[薪岛|薪岛]]等与中方陆地接壤，因此鸭绿江江面末段完全在朝方一侧。目前河口为双方共用。

== 名称 ==
古称[[浿水|浿水]]，[[汉朝|汉朝]]称为'''马訾水'''，[[唐朝|唐朝]]始称鸭绿江（隋唐时期[[浿水|浿水]]为大同江），因其江水清澈，关于鸭绿江其名的来历主要流行着两种说法：一说因江水颜色似鸭头之色而得名（唐朝[[杜佑|杜佑]]《[[通典|通典]]》）；二说因上游地区有鸭江和绿江两条支流汇入，故合而为一，并称为“鸭绿江”。另有源自于[[满族|满族]]先民的语言（满语称为Yalu ula，意为“边界之江”）的说法。史书记载[[建州卫|建州卫]]女真常年生活在[[婆猪江|婆猪江]]畔（鸭绿江支流）。

== 水系 ==
鸭绿江发源于中国[[吉林省|吉林省]]东南，中朝边境[[长白山|长白山]]主峰南麓海拔2300米处的[[天池_(长白山)|长白山天池]]，然后流向西南，流经吉林省的[[长白朝鲜族自治县|长白朝鲜族自治县]]、[[临江市|临江市]]、[[集安市|集安市]]、[[辽宁省|辽宁省]]的[[丹东市|丹东市]]，在辽宁省丹东市[[東港市|东港]]附近入[[黄海|黄海]]北部的[[西朝鲜湾|西朝鲜湾]]。鸭绿江入海口是中国大陆海岸线的最北端。全长795公里，其中流经吉林省境界长575公里，辽宁省220公里；[[流域面积|流域面积]]6.19万[[平方公里|平方公里]]，中国境内流域面积约为3.25万平方公里。

== 水文 ==
鸭绿江上下落差较大，源头与河口落差达到2440米。鸭绿江年降水量约在870毫米，自上游向下游渐增，70％集中6～9月。平均流量1040立方米／秒，年径流量327.6亿立方米。7～8月形成夏汛，最大流量是年均流量的15～25倍

== 气候与生物 ==
鸭绿江全流域气候凉湿，分布以[[红松|红松]]、[[枫桦|枫桦]]为主的针阔叶混交林，下游多[[栎树|栎林]]。有多种野生动、植物，在入海口一带，盛产大[[银鱼|银鱼]]。入海口处建有鸭绿江口滨海湿地自然保护区。

== 桥梁 ==
[[File:Sino-Korean_Friendship_Bridge_across_the_Yalu.jpg|thumb]]
{{seealso|中朝友谊桥}}

20世纪初，鸭绿江上始建铁桥，先后在丹东和朝鲜[[新义州|新义州]]之间建了二座。第一座建于1909年，是座开闭式桥梁。1950年[[朝鲜战争|朝鲜战争]]中被[[美国|美国]]空軍
炸毁，桥墩至今犹存，现辟有“断桥游览区”。第二座桥建于1940年，为[[铁路|铁路]]、[[公路|公路]]两用桥，全长940米，属中朝两国共管。它是中朝两国的交通要道，也是游人观光览胜的景点。

== 水库与水电站 ==
鸭绿江落差大，水利资源丰富，干流上建有[[云峰水库|云峰水库]]、[[渭源水库|渭源水库]]、[[水丰水库|水丰水库]]、[[太平湾水库|太平湾水库]]，支流上建有[[桓仁水库|桓仁水库]]、[[回龙山水库|回龙山水库]]、[[太平哨水库|太平哨水库]]等大中型综合性水利工程。总装机容量188万[[千瓦|千瓦]]，年发电量77.4亿千瓦时（包括水丰水库两侧的扩建电站）。

[[水丰水库|水丰水电站]]是鸭绿江上的第一座大型电站，建于1937年，1941年发电，1955年成立中朝鸭绿江水力发电公司，双方合营水丰发电厂并进行了恢复改建。
