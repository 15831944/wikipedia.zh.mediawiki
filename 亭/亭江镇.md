'''亭江镇'''是[[中华人民共和国|中国]][[福建省|福建省]][[福州市|福州市]][[马尾区|马尾区]]下辖的一个镇，位于福州市东部。亭江镇北面为[[连江县_(福州市)|连江县]][[琯头镇|琯头镇]]，南面为[[马尾区|马尾区]][[罗星街道|罗星街道]]，西面为[[宦溪镇|宦溪镇]]，东面为[[琅岐镇|琅岐镇]]、[[长乐市|长乐市]]。

== 历史沿革 ==
亭江镇[[唐朝|唐朝]]属闽县晋安乡的合浦里和海滨里。[[宋朝|宋朝]]属闽县晋安东乡的合浦北里”和合浦南里”。[[元朝|元朝]]及[[明朝|明朝]]行政区划没有改变。[[清朝|清朝]]，合浦北里和合浦南里改为江右里、江左里。

[[中华民国|中华民国]]时期，亭江镇属闽侯县2区(闽安镇区)。[[中华人民共和国|中华人民共和国]]成立后，属闽侯县第3区(马江区)。1962年，亭江镇划归福州市郊区，为亭江区公所，后改为亭江人民公社。1984年，改公社为乡。1988年改名亭江镇。

== 行政区划 ==
亭江镇共辖3个[[社区|社区]]、17个[[行政村|行政村]]，分别是：

亭头社区、闽安社区、闽亭社区、敖溪村、盛美村、东盛村、笏山村、前洋村、康坂村、白眉村、象洋村、英屿村、东岐村、长柄村、洪塘村、亭头村、闽安村、西边村、长安村、东街村

== 教育 ==
宋朝，亭江就有朱熹讲学的怡山书院。目前，亭江有1所大学，3所中学，18所小学，19所幼儿园。


== 简介 ==
亭江镇山多田少，从清代咸丰、同治年间(1851～1874年)起，就有许多村民被迫背井离乡，到海外谋生。
