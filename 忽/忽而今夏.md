《'''忽而今夏'''》（{{lang-en|'''''Suddenly This Summer'''''}}），2018年[[中国|中国]][[偶像剧|偶像剧]]。改编自[[明前雨后|明前雨后]]同名小说，讲述了章远和何洛跨越十年的青春时光历经千山万水不忘初心的纯爱故事<ref>{{Cite web |url=http://ent.sina.com.cn/s/m/2017-07-27/doc-ifyinvyk1505662.shtml |title=《忽而今夏》首次触网“异地恋” 有青春有内涵|accessdate=2017-12-02 |author= |date=2017-07-27 |work=Blue |publisher=新浪娱乐|language=Zh-hans }}</ref>。该剧由[[腾讯视频|腾讯视频]]、[[完美世界影视|完美世界影视]]联合出品，导演[[吕赢|吕赢]]执导，[[白宇|白宇]]、[[卜冠今|卜冠今]]等主演<ref>{{Cite web |url=http://news.yule.com.cn/html/201708/250348.html |title=《趁我们还年轻》《忽而今夏》双剧同行 完美“写实青春”清风袭来|accessdate=2017-12-02 |author= |date=2017-08-17 |work=Blue |publisher=中国娱乐网|language=Zh-hans }}</ref>。2017年7月26日于青岛<ref>{{Cite web |url=http://yule.bandao.cn/news_html/201709/20170902/news_20170902_2760249.shtml |title=电视剧《忽而今夏》在青岛热拍 讲述纯爱故事|accessdate=2017-12-02 |author= |date=2017-09-02 |work=李敏娜 |publisher=半岛网娱乐|language=Zh-hans }}</ref>开机，历经90天拍摄，于同年10月23日杀青<ref>{{Cite web |url=http://ent.qq.com/a/20171023/032836.htm |title=《忽而今夏》青岛杀青 白宇卜冠今演绎十年成长|accessdate=2017-12-02 |author= |date=2017-10-23 |work=carinayang |publisher=腾讯娱乐|language=Zh-hans }}</ref>。[[腾讯视频|腾讯视频]]于2018年4月19日播出<ref>{{Cite web |url=http://ent.huanqiu.com/movie/yingshi-neidi/2018-04/11763103.html |title=《忽而今夏》定档 卜冠今细腻诠释异地甜酸|accessdate=2018-04-04 |author= |date=2018-04-03 |work=张玮 |publisher=环球网娱乐|language=Zh-hans }}</ref>。

==剧情简介==
少女何洛機靈俏皮但功課欠佳，卻不小心誇下海口要考入名校。意氣飛揚的學霸章遠看似自大高傲，但被何洛的鍥而不捨打動，輔導她不斷進步。兩個人心意漸明，約定報考同一所大學，但陰差陽錯，高考令他們分隔兩座城市。他們不畏時空距離，滿懷憧憬地一起規劃未來。高中同班的一眾好友，也各自經歷成長中的種種磨礪與挑戰。高考失利的章遠急於證明自己，實現早日與何洛團聚的承諾，卻在創業過程中一再受挫。不同的經歷和心境讓他和何洛產生分歧，相互深愛卻漸行漸遠。站在人生的岔路口，他們應該如何抉擇？數年後事業有成卻天各一方，再次重逢時，心中是否還有最初的悸動？一段綿延十年，隔山跨海的眷戀，帶你重溫青澀時光裡的怦然心動，體會追逐夢想的奮力拼搏，回想記憶深處，最初愛過的那個人。

==电视剧取景地==
剧中高中取景于青岛市市南区青岛二中分校。华清大学的取景地是中国海洋大学鱼山校区（操场、校园、食堂）和崂山校区（宿舍，礼堂）。海城大学取景地是在中国石油大学，在青岛市黄岛区（西海岸新区）。<ref>{{cite web |url=https://www.zhihu.com/question/277994465 |website=知乎|accessdate=28 January 2019}}</ref>
