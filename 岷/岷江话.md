'''岷江话'''，又称为'''岷江小片'''，是[[四川话|四川话]]的一个分支，主要分布于[[四川盆地|四川盆地]]西南部[[岷江|岷江]]流域及[[长江|长江]]两岸，同时在盆地中部分布有一个由[[西充|西充]]、[[盐亭|盐亭]]、[[射洪|射洪]]三县组成的[[方言岛|方言岛]]。岷江话的主要特征是[[入声|入声]]保留，独立成调，拥有一套仅用于入声字的[[元音松紧对立|紧元音韵母]]，虽然并不符合“入声归阳平”这一《中国语言地图集》中划定[[西南官话|西南官话]]的标准，但在音系上和成渝话拥有一致的创新点，因此仍为一类。加之，成都等成渝片方言在仅仅一百年前也拥有入声调，因此有无入声调并不是绝对的划分准则<ref name="《西南官话的分区（稿）》">李蓝（2009年第1期），《西南官话的分区（稿）》，方言</ref>。但也有学者认为，岷江小片是由中上古[[巴蜀语|巴蜀语]]直接发展而成，与主要源自移民的西南官话拥有截然不同的起源，但并非主流观点<ref name="《四川官话入声现象的历史文化透视》">杨波（1997年第5期），《四川官话入声现象的历史文化透视》，西南师范大学学报（哲学社会科学版）</ref>。

== 形成 ==
四川地区在元末明初和明末清初的两次“[[湖广填四川|湖广填四川]]”大移民运动中，人口构成发生巨变。在[[明代|明代]]，元之前流行于四川地区的巴蜀语自四川盆地东北部开始，向盆地西南部（今岷江话流行区域）逐步萎缩<ref name="《试论宋代巴蜀方言与现代四川方言的关系》"/>，并且与移民带来的语言相互影响，现今四川话，包括受移民语言影响相对较小的岷江小片和影响巨大的[[成渝片|成渝片]]都在这种背景下开始逐步形成。今岷江话分布地区，特别是核心的[[乐山|乐山]]一带，受战乱影响比较轻微，巴蜀土著居民存留较多，因而语言受“[[湖广填四川|湖广填四川]]”带来的移民语言影响相对较少，但本质仍然是一种新起语言。以乐山西郊苏稽话为例，虽然其拥有符合[[宋代|宋代]]巴蜀语“歌豪通押”、“鱼豪通押”、“家麻与皆来通押”、“‘关’、‘上’押家麻”的少数特征，但是并不能断定是中古巴蜀语在现代的孑遗<ref name="《试论宋代巴蜀方言与现代四川方言的关系》">刘晓南（2009年第8卷第6期），《试论宋代巴蜀方言与现代四川方言的关系——兼谈文献考证的一个重要功用：追寻失落的方言》，语言科学</ref>，反而更可能是后起的语音创新<ref>{{Cite journal|title=《西蜀方言》的音系性质|author=曾晓渝|url=|journal=方言|issue=3|doi=|others=|year=2018|volume=|page=|pmid=}}</ref>。

由于这种特殊的形成背景，从岷江小片核心区的乐山郊区[[苏稽话|苏稽话]]，到[[乐山话|乐山话]]，到岷江小片边缘区成都郊区[[新津话|新津话]]，再到成渝片的[[成都话|成都话]]，语音上存在渐变。以非入声韵为例，我们可以观察到一条较为完整的语音渐变，部分例字如下表所示<ref name="《谈谈乐山方言语音的偏移》">赖先刚（2004年第6期），《谈谈乐山方言语音的偏移》，天府新论</ref>。

== 分布 ==
[[File:Sichuanese.png|240px]]

岷江话主要分布于四川盆地西南部的岷江流域及长江两岸，地跨[[四川|四川]]、[[重庆|重庆]]、[[贵州|贵州]]、[[云南|云南]]4省市，其在四川盆地中部分布有一个由[[西充|西充]]、[[盐亭|盐亭]]、[[射洪|射洪]]三县组成的方言岛，总使用人口约3000万。《中国语言地图集》中也把入声保留的[[西昌话|西昌话]]归入岷江小片，但由于其并不具有岷江小片的其它共同特征，因而存在争议<ref name="《四川省西南官话以外的汉语方育》">崔荣昌（1986年第3期），《四川省西南官话以外的汉语方育》，方言</ref>。使用岷江话的县市如下：

== 音韵 ==
=== 声调 ===
岷江话的调类总共有[[阴平|阴平]]、[[阳平|阳平]]、[[上声|上声]]、[[去声|去声]]、[[入声|入声]]五类。岷江话除入声外调值与成渝片高度一致，阴平一般为高平或半高平（55或44），阳平一般为低降（21或31），上声一般为高降或全降（53或51），去声一般为低升、中升或降升（13、24或213）。而岷江话的入声调值一般为中平、高平或半高平（33、55或44）。部分方言代表点的音调调值如下表所示。

=== 韵母 ===
岷江话的韵母与以成渝片为代表的其它西南官话拥有很大差异。首先，其拥有一套独立的仅用于入声的[[元音松紧对立|紧元音韵母]]来取代入声塞音韵尾，即便是部分喉塞音消失的岷江话也可以利用主要元音的松紧对立来区分入声与舒声字。岷江话的入声韵母包括[iæ]、[uæ]、[ʊ]、[ɘ]、[ɐ]、[iɐ]、[uɐ]、[c]、[yʊ]、[yɵ]等，这些韵母有紧喉作用，在发音时喉头肌肉和口腔肌肉紧张，从而使整个音节表现出一种粗硬紧促的状态<ref name="《四川官话雅棉小片入声归阴平研究》">余江（2004年5月），《四川官话雅棉小片入声归阴平研究》，汕头大学</ref>。其次，岷江话中舒声韵母的归派也与成渝片拥有极其大的差异，有元音高化、咸山摄鼻音脱落等现象，尤以乐山西郊[[苏稽|苏稽]]、[[安谷|安谷]]、[[冠英|冠英]]一带为最。岷江话的韵母如下表（以[[乐山话|乐山话]]为代表），最上为拼音，加（q）者为相配入声韵，[ ]中为国际音标，最下为例字<ref name="《乐山方言》"/><ref name="《谈谈乐山方言语音的偏移》">赖先刚（2004年第6期），《谈谈乐山方言语音的偏移》，天府新论</ref><ref name="《四川方言的鼻尾韵》">甄尚灵（1983年第4期），《四川方言的鼻尾韵》，方言</ref>：

== 危机 ==
与其它四川方言不同，岷江话的生存环境并非仅仅因为[[推广普通话|推广普通话]]的原因而面临危机，其一直以来也受到相对强势的[[成渝片|成渝片]]的强烈冲击，从而流行区域不断萎缩。[[成都话|成都话]]都可能原属岷江小片，据[[中央研究院|中央研究院]]《四川方言調查報告》，其入声直到1940年代都还未完全消失<ref name="《四川方言調查報告》">楊時逢，《四川方言調查報告》，中央研究院歷史語言研究所專刊之82，1984</ref>。岷江话的使用者在对外交流时往往都会将语音尽量向成渝片特别是[[成都话|成都话]]、[[重庆话|重庆话]]等地区中心城市的方言靠拢，因而内部语音越是靠近成渝两地，受成渝片影响越是严重，自身语音特色保留越少<ref name="《谈谈乐山方言语音的偏移》"/><ref name="《方言问题探微：以四川方言为例》">戴伟（2008年第4期），《方言问题探微：以四川方言为例》，重庆社会科学</ref>。在成都近郊的[[温江|温江]]、[[新都|新都]]、[[郫县|郫县]]、[[双流|双流]]等地，岷江话已经有消失的危险<ref name="《张口闭口成都腔》">{{cite web |url=http://news.163.com/09/1124/10/5OSKLLUF000120GR.html |date= 2009-11-24 |title=张口闭口成都腔 谁还能说土掉渣的“岷江话” |accessdate=2010-05-15 |work=网易 |publisher=成都商报}}</ref>。同时，近年来的“[[推广普通话|推广普通话]]”政策也对岷江话产生了巨大影响，部分语音亦开始向普通话靠拢<ref name="《谈谈乐山方言语音的偏移》"/>。
