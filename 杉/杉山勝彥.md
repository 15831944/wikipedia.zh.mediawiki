'''杉山胜彦'''（{{jpn|j=杉山　勝彦|hg=すぎやま かつひこ|rm=Sugiyama Katsuhiko|f=y}}，{{bd|1982年|1月19日|catIdx=Sugiyama Katsuhiko}}），日本作词家、作曲家、编曲家、音乐家、音乐制作人、民谣二重唱“TANEBI”吉他手。

== 经历 ==
杉山胜彦出生于日本[[埼玉縣|琦玉县]] [[入間市|入间市]]。母亲是一位[[钢琴|钢琴]]老师 <ref name="乃木坂×週プレ92-93">「作曲家・杉山勝彦インタビュー」『乃木坂46×週刊プレイボーイ2015』、92-93頁。{{ASIN|B01518F35W}}</ref>。幼儿时期，虽然家中哥哥们都学会了钢琴，但由于对乐谱和练习的厌烦，年龄最小的衫山胜彦跟着母亲学了一年不到的钢琴便放弃了。那时的他非常喜欢游戏《[[勇者斗恶龙|勇者斗恶龙]]》的音乐。初中时，由于一次修学旅行中唱歌的不快经历，衫山胜彦开始在家中用哥哥的吉他练习[[Mr.Children|Mr.Children]]的曲子。<ref name="『音人工坊』　杉山勝彦">{{Cite web |url = https://www.jasrac.or.jp/otobito/interview13/index.html |title = 「音人工房」　杉山 勝彦 |date = 2017-03 |publisher = JASRAC |language = ja |accessdate = 2019-02-09}}</ref> 初二时，衫山胜彦则沉迷于 [[融合爵士樂|融合爵士乐]]乐队 <ref name="乃木坂×週プレ92-93" />。

2006年，衫山胜彦在母校早稻田大学学园祭OB合唱团的舞台上表演时，被RATS & STAR的佐藤善雄发掘。

2007年，成为Sony Music Publishing的专属作曲家后，衫山胜彦又在日本男子组合[[岚|岚]]的单曲”[[Step_and_Go|冬を抱きしめて]]”中以作词家身份出道<ref name="official" />。

2014年1月，在新宿车站前偶遇上田和寛。被他的歌声所吸引后，二人成立组合“USAGI”，衫山在其中担当吉他手，并在后来由[[環球音樂_(日本)|环球音乐]]发行的单曲“イマジン”中出道。同年，衫山作为作曲家的活动也停止了<ref name="official">{{Cite web |url = http://sugiyama-katsuhiko.com/profile.html |title = 杉山勝彦 official site |language = ja |accessdate = 2019-02-09}}</ref>。

2015年，日本女子组合[[乃木坂46|乃木坂46]]在[[第66回NHK紅白歌合戰|第66回NHK紅白歌合戰]]上演唱《[[你就是希望|君の名は希望]]》，作曲者即为衫山胜彦。而在次年的[[第67回NHK紅白歌合戰|第67回NHK紅白歌合戰]]中，该组合同样演唱了由其作曲的《[[再见的意义|サヨナラの意味]]》。在此之前2010年的[[第61回NHK红白歌合战|红白歌会]]上，歌手[[倖田來未|倖田來未]]也演唱了由其作曲的《[[喜歡，喜歡，喜歡。/唯有你|好きで、好きで、好きで。]]》。

2016年，离开Sony Music Publishing，并成立了音乐事务所コライト。<ref name="official" />

2016年10月，从USAGI所属的事务所独立，并将二人组合的名字改为“TANEBI”。<ref name="official" />

2017年12月，为[[家入里歐|家入里欧]]创作的”ずっと、ふたりで”获得第59回[[日本唱片大奖|日本唱片大奖]]作曲奖。<ref>{{Cite web|url=https://www.oricon.co.jp/news/2103390/full/|title=“第59回レコード大賞”各賞決定 大賞候補に乃木坂46&欅坂46ら|accessdate=2019-02-09|date=2017-11-16|work=ORICON NEWS|publisher=oricon ME}}</ref>。

2018年1月，为TV动画”[[DARLING_in_the_FRANXX|DARLING in the FRANXX]]”创作片尾曲”トリカゴ“。<ref>{{Cite web |url = https://www.lisani.jp/0000102372/?show_more=1 |title = TVアニメ『ダーリン・イン・ザ・フランキス』すべてのEDテーマ楽曲制作担当・杉山勝彦インタビュー |date = 2018-05-11 |publisher = リスアニ！WEB|language = ja |accessdate = 2019-02-09}}</ref>{{noteTag|name=note3}}
