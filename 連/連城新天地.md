'''連城新天地'''位於[[深圳|深圳]][[福田區|福田區]]福華路地底，前身是興建[[深圳地鐵|深圳地鐵]][[深圳地鐵1號綫|1號綫]]時一併興建的[[防空洞|人防設施]]，是深圳的一座[[地下街|地下街]]式商場。商場連接[[深圳地鐵|深圳地鐵]][[崗廈站|崗廈站]]、[[會展中心站|會展中心站]]、[[購物公園站|購物公園站]]及[[广深港高速铁路|广深港高速铁路]][[福田站|福田站]]，亦有多個出入口通往附近商廈。現時由[[深圳地鐵集團|深圳地鐵集團]]屬下公司負責營運管理。

== 一期 ==
一期商場於2012年6月28日正式開幕，位於[[會展中心站|會展中心站]]和[[購物公園站|購物公園站]]之間，面積約為27,000平方米。商場分為A區'''流行坊'''、B區'''名品坊'''，及C區'''美食坊'''，其中A區設有手機電訊店，也有一間小型5D影院，和不少小食店。B區設有不少時裝店和精品店。C區及B區近C區的一部分為食肆，包括[[麥當勞|麥當勞]]（已結業）、[[肯德基|肯德基]]、[[德克士|德克士炸雞店]]（已結業）、[[康師傅私房牛肉麵|康師傅私房牛肉麵]]、[[冰雪皇后|冰雪皇后 Dairy Queen]]、[[大家樂|大家樂]]、[[大快活|大快活]]、[[可頌坊|可頌坊]]等等，C區也有emoi、[[萬寧|萬寧]]和Vango便利店。

=== 一期商場特色 ===
商場A區和B區之間設有蘭花園，擺放不少蘭花。此外，音樂噴泉在B區和C區之間，每天定時啟用。
商場位於福華路下方，所以商場在福華路車路中央分隔區安裝了不少玻璃天幕，把室外光線引入商場。

== 二期商場 ==
二期商場於2015年12月2日試業，位於[[會展中心站|會展中心站]]和[[崗廈站|崗廈站]]之間，長約1.3公里，面積約為13,000平方米<ref>深圳晚報-[http://wb.sznews.com/html/2015-12/04/content_3406149.htm 连城新天地二期正式开通] {{webarchive|url=https://web.archive.org/web/20170328204130/http://wb.sznews.com/html/2015-12/04/content_3406149.htm |date=2017-03-28 }}</ref>。商場分為A區至D區。由於商場地面比[[會展中心站|會展中心站]]為低，因此每當列車駛過，除會聽到列車駛過的聲音，亦會感到地面震動。

=== 二期商場特色 ===
二期商場設有水簾舞台，亦有櫥窗展示[[秦始皇兵馬俑|兵馬俑]]。

== 三期美食廣場 ==
美食廣場設於[[會展中心站|會展中心站]]北側，A、B出口之間，設有十多檔食肆，於2015年5月9日開幕。雖名為三期，但實際位置在一、二期之間，亦比二期更早開業。

== 商場代言人 ==
由於業主為[[深圳地鐵集團|深圳地鐵集團]]，商場以卡通化的地鐵列車為代言人，並取名為「地鐵寶寶」。
