'''时光辉'''（{{bd|1970年|1月|catIdx=Shi時}}），籍贯[[安徽|安徽]][[阜阳|阜阳]]，生于上海，[[汉族|汉族]]，[[中华人民共和国|中华人民共和国]]政治人物，[[中共党员|中共党员]]，1993年加入[[中国共产党|中国共产党]]。[[同济大学|同济大学]]毕业，大学学历，经济学硕士，高级工程师。曾任[[上海市人民政府|上海市人民政府]]副市长等职务。现任[[中共贵州省委|中共贵州省委]]常委、政法委书记。

== 早年经历 ==
1990年代初毕业之后，时光辉从[[上海市|上海市]]市政第二公司[[宝钢|宝钢]]分公司的施工员做起，逐级升迁到市政第二公司经理助理、副总经理，后又担任市政工程建设处（公司）处长、总经理；后进入[[上海市市政工程管理局|上海市市政工程管理局]]，担任处长、副局长，开始进入政界。

== 从政履历 ==
进入而立之年之后，时光辉调任[[静安区|静安区]]担任区委常委、副区长<ref>[http://www.sh.xinhuanet.com/2007-06/01/content_10178964.htm 今起进入汛期 区县防汛责任人名单公布] {{webarchive|url=https://web.archive.org/web/20140910195959/http://www.sh.xinhuanet.com/2007-06/01/content_10178964.htm |date=2014-09-10 }} 新华网上海频道_2007年06月1日</ref>，后在[[2008年6月|2008年6月]]转任[[奉贤区|奉贤区]]区委副书记、区长<ref>[http://finance.sina.com.cn/hy/20091030/18426906555.shtml 上海奉贤区区长时光辉：努力改善投资环境] {{Wayback|url=http://finance.sina.com.cn/hy/20091030/18426906555.shtml |date=20131002123201 }} 新浪网</ref><ref>[http://shzw.eastday.com/shzw/G/20110130/u1a33653.html 上海市奉贤区区长时光辉答东方网网友问] 上海政务网</ref>。[[2011年8月|2011年8月]]上海市委决定：时光辉任中共奉贤区委书记<ref>[http://leaders.people.com.cn/GB/15497981.html 上海市委决定：时光辉任中共奉贤区委书记] 人民网_转自东方网_2011年08月24日</ref>。2013年2月1日，上海市十四届人大一次会议举行第四次全体会议，选举时光辉为[[上海市人民政府|上海市人民政府]]副市长，时光辉由此成为中华人民共和国文职官员之中首位“[[70后|70后]]”[[省部级副职|省部级副职]]官员<ref name=jfdaily>[https://www.jfdaily.com/news/detail?id=117527 【人事观察】时光辉任贵州省委常委，曾是全国第一位“70后”副部级]</ref>。

2018年11月，任[[中共贵州省委|中共贵州省委]]常委<ref name=jfdaily/>、政法委书记。
