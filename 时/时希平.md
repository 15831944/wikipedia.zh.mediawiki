'''时希平'''（{{bd|1957年|1月|2015年？|}}）[[河南|河南]][[杞县|杞县]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员<ref name=mian/>。

==生平==
时希平是[[中国共产党|中国共产党]]党员，1974年9月参加工作。1977年进入武汉师范学院（现[[湖北大学|湖北大学]]）政治系学习。后来又成为[[中欧国际工商学院|中欧国际工商学院]]工商管理硕士。历任[[中共湖北省委|中共湖北省委]]组织部干部、[[中共中央组织部|中共中央组织部]]研究室副主任、中共中央组织部企业干部办公室副主任、中共中央组织部干部五局副局长（正局级）、[[国务院国有资产监督管理委员会|国务院国有资产监督管理委员会]]企业领导人员管理一局局长、国务院国有资产监督管理委员会人事局局长<ref name=sina/><ref name=mian/>。

2011年1月起，任[[中华人民共和国国务院|国务院]][[国有重点大型企业监事会|国有重点大型企业监事会]]主席，成为副部级干部，后分管监事会07办事处，负责对[[中国国电集团公司|中国国电集团公司]]、[[中国中化集团公司|中国中化集团公司]]、[[中国北方机车车辆工业集团公司|中国北方机车车辆工业集团公司]]、[[中国民航信息集团公司|中国民航信息集团公司]]履行监事会监督职责。2015年2月，中央第三巡视组进驻[[国家开发投资公司|国家开发投资公司]]开展专项巡视工作，该巡视组组长[[王立英|王立英]]，副组长时希平、[[丁伯东|丁伯东]]<ref>{{Cite web |url = http://district.ce.cn/newarea/sddy/201503/02/t20150302_4702556.shtml |title = 中央第三巡视组进驻国投 组长王立英 副组长时希平、丁伯东 |author =  |date = 2015-03-02 |publisher =中国经济网  |language =  |accessdate =  |quote =  }}</ref>。2015年3月，中央第三巡视组进驻[[东方电气集团有限公司|东方电气集团有限公司]]开展专项巡视工作，该巡视组组长王立英，副组长时希平、丁伯东<ref>{{Cite web |url = http://politics.people.com.cn/n/2015/0306/c1001-26651741.html |title = 中央第三巡视组专项巡视中国东方电气集团有限公司 |author =  |date =  2015-03-06|publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。2015年7月，中央第三巡视组分别进驻[[中国航天科工集团公司|中国航天科工集团公司]]、[[中国航天科技集团公司|中国航天科技集团公司]]开展专项巡视工作，该巡视组组长[[李熙|李熙]]，副组长时希平、[[周清玉|周清玉]]、[[陈善光|陈善光]]<ref>{{Cite web |url = http://district.ce.cn/newarea/sddy/201507/07/t20150707_5860942.shtml |title =中央第三巡视组进驻航天科工 组长李熙 副组长时希平、周清玉、陈善光  |author =  |date = 2015-07-07 |publisher = 中国经济网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://politics.people.com.cn/n/2015/0706/c1001-27262110.html |title =中央第三巡视组近日进驻中国航天科技集团公司  |author =  |date =  2015-07-06|publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref><ref name=sohu/>。巡视组进驻后，中国航天科工集团公司查处了15个单位共31人违反[[中央八项规定|中央八项规定]]精神问题<ref name=sina>{{Cite web |url = http://news.sina.com.cn/c/zs/2015-09-19/doc-ifxhzevf0859487.shtml |title = 落水失踪巡视组长时希平查出央企哪些问题 |author =  |date = 2015-09-19 |publisher =  新浪|language =  |accessdate =  |quote =  }}</ref>。

2015年9月18日[[财新网|财新网]]报道，2015年9月13日前后，国有重点企业监事会主席时希平在[[湖北|湖北]]休假期间落水失踪，原因不明，至今仍未找到<ref name=sina/><ref name=mian/>。
