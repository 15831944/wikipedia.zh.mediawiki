'''时光网'''（{{lang-en|''Mtime''}}）是一家[[中國大陆|中國大陆]]的[[电影|电影]]及[[电视剧|电视剧]][[在线|在线]][[数据库|数据库]]，包括资讯、预告片、海报、写真和影评，開辦於2004年。网站也提供影讯查询、博客、相册和群组等服务。截止至2008年第二季度，网站每月的独立访问者超过40万人次，成为中国最大的电影娱乐社区。2016年7月27日，[[万达院线|万达院线]]宣布以2.8亿美元收购时光网。<ref>[http://news.mtime.com/2016/07/27/1557937.html 万达电影院线全资并购时光网].时光网.2016-07-27.[2016-07-27].</ref>

== 概要 ==
Mtime时光网上有丰富的电影作品信息，包括电影的评分、概述、详细资料、电影图片、演职员表、上映时间、剧情、幕后花絮、制作发行、关联电影、最新电影、技术参数、相关网站、相关资讯、[[音轨|影片音轨]]、屏幕的高宽比、影片的不同版本等等，还有许多影迷写的影评。另有群组和博客提供给影迷讨论的地方，还有电影相关的猜谜互动游戏。

演员、导演、作者和其他工作人员都在数据库中有自己的条目，其中列出他们的个人信息、参加过的影片还有他们的生平。

== 版权 ==
Mtime时光网尊重[[知识产权|知识产权]]，目前本身不提供电影在线观看或下载服务，也不允许用户这样做。但其网站上放出未来会与版权拥有者一道为用户提供高质量的、合法的电影收看服务的信息。<ref>[http://www.mtime.com/help/other/faq_q18.html 新产品新在哪里？]</ref>

== 事件 ==
2010年10月14日时光网的[[域名|域名]]被注册商万网暂停解析，次日也就是15号时光网官方微博表示网站关闭因为系统维护中，2010年11月4日官方微博表示时光网开始逐渐恢复访问，在《致时光网友的一封信》中澄清没有财务或版权的问题以及所谓的低级内容，但没有说明事件的具体原因，只表示有难言之隐，并说道：“以前时光网只是一个网站，现在是你们赋予了它生命。”<ref>[http://news.mtime.com/2010/11/05/1444067.html 致时光网友的一封信 感谢大家的关心和支持]</ref>。
