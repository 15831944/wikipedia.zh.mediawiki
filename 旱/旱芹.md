'''旱芹'''（[[学名|学名]]：{{lang|la|''Apium graveolens''}}），又称'''西洋芹'''、'''西芹'''、'''芹菜'''、'''药芹'''或'''富菜'''，是[[伞形科|伞形科]][[芹属|芹属]]的植物，分布在[[欧洲|欧洲]]、[[美洲|美洲]]、[[亚洲|亚洲]]、[[非洲|非洲]]等地。在[[中国烹饪|中国烹饪]]中是一种常见的[[蔬菜|蔬菜]]，在[[義大利菜|義大利菜]]中也常用作爆香料。芹菜可食用部分主要为[[叶柄|叶柄]]；[[叶|叶]]虽然也可以食用，但因略帶苦味，少有人吃{{來源請求}}。其一種類 ''A. graveolens'' var. ''rapaceum'' 於西方國家栽種，[[根|根]]可食用。現今普遍可見者為改良變種 ''A. graveolens'' var. ''dulce'' ，[[葉|葉]]、[[莖|莖]]、[[根|根]]、[[果實|果實]]全可食用。種子可取油及製[[香辛料|香辛料]]。{{r|薛}}

== 形态 ==
[[File:Celery_cross_section.jpg|thumb]]
芹菜是一年或二年生[[草本|草本]]；基出叶为二回羽状复叶，叶柄发达，中空或实，绿白或绿黄色，有特殊香味；複繖形花序，白色小花。{{r|薛}}

株高约50～80厘米。茎一般都在手指粗细，中间为空。中國品種與西芹莖部粗壯與大小相距很大。{{r|薛}}

中國品種芹菜莖部數分闊。西芹最外圍莖部有一吋闊或以上，四份一至三份一吋厚。

[[根|根]]呈须状。

== 历史 ==
芹菜作為蔬菜食用历史悠久。[[西汉|西汉]]时由[[张骞|张骞]]從[[西域|西域]]传入[[中国|中国]]。在香港常可買到[[西芹|西芹]]和[[中國旱芹菜|中國旱芹菜]]。

加上近代營養師多建議食[[西芹|西芹]]有助[[降血脂|降血脂]]（敗血）、防心腦血管病，而且西芹作為食物易儲易用，廣受華人接受。

== 用途 ==
可以降低血壓，減輕肌肉痙攣以及促進食慾。對關節炎、痛風及腎臟問題有益處。可當作利尿劑、抗氧化劑以及鎮靜劑。

此外，西芹是一種負熱量食物，原因是西芹的纖維量高，使人體要消化這些纖維成份所消耗的熱量，比西芹本身能夠提供的熱量更大的緣故<ref>{{cite episode |url=https://www.facebook.com/jennyhongradio/videos/853912548139713/ |title=什麼食物加速新陳代謝 |series=真情康翠瑩 |date=2018-06-21 |channel=am1430 |start-time=00:18:00 |location=加拿大多倫多 |language=zh-yue }}</ref>。

香芹因其抗氧化物，包括类黄酮、类胡萝卜素和维生素C，也被证明具有抗炎作用。<ref>{{Cite web|title=香芹的功效与作用禁忌，香芹叶子的功效与作用，西芹的功效与作用禁忌|url=https://webmd.cn/2019/03/parsley/|work=WebMD|accessdate=2019-03-01|language=zh-CN}}</ref>

== 烹飪法 ==
=== 中式 ===
中式熟食有做餡、芹菜餃、芹菜肉絲炒苦瓜、腰果炒西芹、芹菜炒干絲等等。

=== 西方 ===
西式生食有：以西芹做其中一種沙律材料<ref>{{cite web |url=http://www.hsph.harvard.edu/nutritionsource/simple-celery-date-salad/ |title=哈佛大學提供的簡易西芹沙律食譜 |publisher=HSPH, [[哈佛大學|哈佛大學]]}}</ref>。亦可搾汁（如西芹雪梨汁、五青汁）或以[[高速攪拌器|高速攪拌器]]做蔬果露。

熟食则有雜菜或煲湯（西芹湯﹑薯仔西芹洋葱忌廉湯）等。

== 种植 ==
[[File:Celery_seed.jpg|left]]
芹菜每年[[六月|六月]]开始播种，一星期左右就可出苗，经过六十天左右生长，就可以食用。芹菜的开花期为[[七月|七月]]至[[十一月|十一月]]，花朵细小呈白色，里面可找到[[种子|种子]]。芹菜耐寒不喜高温，最佳[[温度|温度]]在15到20[[摄氏度|摄氏度]]之间为佳。
