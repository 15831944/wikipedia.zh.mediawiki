'''文面'''或'''刺面'''，在臉上進行的[[刺青|刺青]]，是很多[[民族|民族]]的[[習俗|習俗]]，流行於[[南島語族|南島語族]]之中，如中國大陸滇緬[[台語民族|台語民族]]、紐西蘭[[毛利人|毛利人]]、台灣的[[泰雅族|泰雅族]]及[[賽德克族|賽德克族]]、[[東南亞|東南亞]]及南亞部份民族皆有文面的習俗。這裡所指的文面是指傳統習俗上的文面，與刑罰用的[[黥面|黥面]]意義截然不同。

== 概述 ==
=== 臺灣原住民 ===
[[臺灣原住民|臺灣原住民]][[泰雅族|泰雅族]]、[[賽德克族|賽德克族]]、[[太魯閣族|太魯閣族]]等民族的習俗當中，紋面代表成年及其成就的標記，沒有紋面的族人將無法得到其他族人的尊敬及認同，更無法論及婚嫁。而個人也必須在適齡期達到族人所規範的標準以證明自己有資格紋面，其中男子須要經過[[出草|出草]][[獵首|獵首]]的洗禮，女子則要熟稔[[織布|織布]]技巧。男子一向紋在上額和下巴；女子則是上額及兩頰。另外紋面的另一個重大意義就是以其圖紋分辨家族譜系，將來好在[[彩虹橋|彩虹橋]]上能與親人相認。國立臺灣歷史博物館收藏之泰雅族文面工具組，共1套5件，包括木盤、竹槌、顏料罐、排針、刮血器等工具<ref>{{cite web|author1=國立臺灣歷史博物館|title=斯土斯民-臺灣的故事：紋面工具|url=http://the.nmth.gov.tw/nmth/zh-TW/Item/Detail/e87830af-84b8-4957-bfab-9942c5095d5d|agency=國立臺灣歷史博物館|work=斯土斯民展品列表 - 斯土斯民-臺灣的故事}}</ref>。

== 人物 ==
在台灣，最後一位保有傳統紋面的泰雅族男性為[[高天生|高天生]]（泰雅語：Yawi Noming），在2013年病逝<ref>{{Cite web |url = http://tw.news.yahoo.com/%E5%94%AF-%E7%94%B7%E6%80%A7%E7%B4%8B%E9%9D%A2%E5%9C%8B%E5%AF%B6-%E9%AB%98%E5%A4%A9%E7%94%9F%E7%97%85%E9%80%9D-035515297.html| language=zh-tw |author=管瑞平|title= 唯一男性紋面國寶 高天生病逝 |publisher =自由時報 |date=2013-07-21 |accessdate=2013-07-21}}</ref>。

在2013年苗栗縣泰安鄉梅園村天狗部落的[[高香妹|高香妹]]（享耆壽95歲）、2018年大興村榮灣部落的96歲[[簡玉英|簡玉英]]（享嵩壽103歲）相繼過世後，只剩下[[柯菊蘭|柯菊蘭]]（泰雅語：Lawa Pileg）是台灣最後一位保有傳統紋面的泰雅族女性<ref>{{Cite web |url = http://www.libertytimes.com.tw/2013/new/sep/10/today-life12.htm?Slots=BLife |language=zh-tw |author=張勳騰 |title= 95歲高香妹病逝 紋面國寶僅剩兩人 |publisher = 自由時報||date=2013-09-10 |accessdate=2013-09-11}}</ref>。

== 用作刑罰的刺面 ==
[[中國|中國]]古代有一種刺面的[[刑罰|刑罰]]，稱為[[黥面|黥面]]或[[黥刑|黥刑]]，是在犯人臉上刺字(奴、婢、盗、贼)。
