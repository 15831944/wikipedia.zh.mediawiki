'''牙買加虹蚺'''（[[學名|學名]]：''{{lang|la|Epicrates subflavus}}''）是分佈在[[牙買加|牙買加]]的一種[[虹蚺屬|虹蚺屬]]，現時並沒有確認的[[亞種|亞種]]。<ref name="ITIS">{{ITIS|ID=202184|taxon=''Epicrates subflavus''|year=2008|date=7 July}}</ref>

== 特徵 ==
牙買加虹蚺可以生長達2米長。

== 分佈及棲息地 ==
牙買加虹蚺分佈在[[牙買加|牙買加]]，包括[[山羊島|山羊島]]。<ref name="McD99"/>牠們喜歡棲息在潮濕的[[石灰岩|石灰岩]][[森林|森林]]。

== 保育狀況 ==
牙買加虹蚺被[[世界自然保護聯盟|世界自然保護聯盟]]列為[[易危|易危]]。<ref name="IUCN"/>由於棲息地被破壞及減少，令牠們易於被捕殺，估計牠們在未來的10年或3代數量會下降20%。
