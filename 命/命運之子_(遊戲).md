{{vgname|命運之子|ko=데스티니 차일드|en=Destiny Child}}是一款由Shift Up開發、Next Floor發行的[[角色扮演遊戲|角色扮演遊戲]]，於2016年10月27日在[[Android|Android]]和[[iOS|iOS]]平台上發行（韓國版），並於2018年12月6日推出國際版（多語言版）<ref name="巴哈姆特(2018-12-06)">{{cite news|url=https://gnn.gamer.com.tw/6/172126.html|title=《命運之子 Destiny Child》繁中版上架 化身魔王候補生與眾多美麗使魔們協力作戰|author=Edward|work=巴哈姆特電玩資訊站|date=2018-12-06|accessdate=2018-12-11}}</ref>。該遊戲由韓國插畫家兼遊戲設計師[[金亨泰|金亨泰]]主導和設計，而[[ESTi|ESTi]]則負責配樂<ref name="巴哈姆特(2018-12-06)" />。

== 系統 ==
《命運之子》主要以橫式的方式在手機上遊玩，並在角色和戰鬥場景上運用了Live2D技術<ref name="聯合新聞網(2018-12-06)">{{cite news|url=https://game.udn.com/game/story/10456/3522374|title=《命運之子DESTINY CHILD》正式在台上市！ 體驗金亨泰筆下角色的魅力|author=遊戲角落|work=udn 聯合新聞網|date=2018-12-06|accessdate=2018-12-11}}</ref>。玩家可透過收集和培養各式的「Child」來組隊並與對手展開戰鬥，以及利用裝備和「瑪瑙」來強化Child們。在戰鬥時，玩家須透過手指點選和滑動的方式來對自己的Child角色發出指令（如攻擊或技能），而遊戲也提供了「自動戰鬥」模式來讓Child角色自行展開戰鬥。

== 故事 ==
故事敘述玩家所扮演的「主人公」是實力最弱的魔王候補生，他必須接受訓練與對戰，來與眾多的惡魔爭奪魔界王者的寶座<ref name="聯合新聞網(2018-12-06)" />。

== 衍生作 ==
2017年12月下旬，據報導，《命運之子》將推出改編電視動畫，由[[福田道生|福田道生]]執導，[[子安秀明|子安秀明]]負責劇本<ref name="巴哈姆特(2017-12-25)">{{cite news|url=https://gnn.gamer.com.tw/9/157029.html|title=《命運之子 Destiny Child》動畫製作消息曝光 同步釋出製作團隊名單|author=Edward|work=巴哈姆特電玩資訊站|date=2017-12-25|accessdate=2018-12-11}}</ref>。該動畫定於2018年4月26日在韓國動畫頻道AnimeBox上正式首播<ref name="遊戲基地(2018-04-23)">{{cite news|url=https://www.gamebase.com.tw/news/topic/98997625/|title=《Destiny Child 命運之子》動畫版韓國播放時間正式決定|author=sanchan|work=遊戲基地|date=2018-04-23|accessdate=2018-12-11}}</ref>。
