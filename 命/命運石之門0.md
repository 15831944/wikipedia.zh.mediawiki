{{vgname|命運石之門0|en=Steins;Gate 0{{efn|日文原版也使用「{{Lang|la|Steins;Gate 0}}」为游戏标题，而非其[[片假名|片假名]]名称「{{Lang|ja|シュタインズ・ゲート ゼロ}}」。}}}}是由[[5pb.|5pb.]]開發和發行的[[視覺小說|視覺小說]]遊戲，為[[科學ADV系列|科學ADV系列]]的第五作，也是2009年《[[命運石之門|命運石之門]]》的正統續作。[[PlayStation_3|PlayStation 3]]、[[PlayStation_4|PlayStation 4]]和[[PlayStation_Vita|PlayStation Vita]]版的遊戲在日本地區的發行時間為2015年12月10日，[[Microsoft_Windows|Microsoft Windows]]版是2016年8月26日，[[Xbox_One|Xbox One]]則在2017年2月22日。全球地區的PC版遊戲由[[Spike_Chunsoft|Spike Chunsoft]]代理，並在2018年5月8日公開。其前傳漫畫在2017年刊載，2018年則有一電視動畫在電視台上播放。

遊戲有三個視角，他們分別是前作的主角——大學生岡部倫太郎，[[時間旅行|時間旅行]]者阿萬音鈴羽和[[神經科學|神經科學]]家比屋定真帆。在遇到真帆和其教授阿歷西斯·萊斯基寧（Alexis Leskinen）後，岡部成為[[人工智能|人工智能]]阿瑪迪斯（Amadeus）的受試者。玩家需要細讀系統給出的文字和對話，並通過選擇是否接聽來自阿瑪迪斯的電話來影響劇情的走向；在《命運石之門0》的早期，故事分有兩個分支，然後分支會進入不同的結局。《命運石之門0》由[[志倉千代丸|志倉千代丸]]策劃，並以《命運石之門》的[[廣播劇|廣播劇]]和[[輕小說|輕小說]]為基礎作成其中一條故事線，同時擁有全新的劇情，而不是單純的改編。配樂與前作一樣都是由[[阿保剛|阿保剛]]負責，他在閱讀劇本時記下了自己的情緒波動，並以此撰寫出與作品的世界觀息息相關的音樂。

== 遊戲玩法 ==
[[File:Steins_Gate_0_RINE_system.jpg|thumbnail]]的訊息與角色互動。圖中的角色是椎名真由理]]
《命運石之門0》是一款[[視覺小說|視覺小說]]遊戲，玩家需要閱讀文字框的文字和對話以了解故事的發展，畫面亦伴隨著人物的[[精靈圖|精靈圖]]和背景藝術<ref name="hcg review"/>。遊戲的故事由多個分支組成，而每個分支都有不同的結局。不同於[[命運石之門|前作]]的自始至終式且有著多個分支點的單一路線，《命運石之門0》一開始便會有一個分支點，接著分成兩條主線，再分成不同的結尾，除此之外亦有四條支線<ref name="hcg review"/>。故事的方向大多為根據玩家是否接聽來自人工智能阿瑪迪斯（Amadeus）的電話來確定的，而接聽的時候主角岡部倫太郎便會通過其手機與軟件交流<ref name="kotaku dark">{{cite web |url=http://kotaku.com/steins-gate-0-is-a-dark-time-travel-tale-1763451550 |title=Steins;Gate 0 is a Dark Time Travel Tale |author=Eisenbeis, Richard |date=2016-03-10 |website=[[Kotaku|Kotaku]] |publisher=Gizmodo Media Group |access-date=2017-01-30 |archive-url=https://web.archive.org/web/20170114003000/http://kotaku.com/steins-gate-0-is-a-dark-time-travel-tale-1763451550 |archive-date=2017-01-14 |dead-url=no}}</ref>。此外系統也會在特定情況下讓玩家以其他角色的視角行動，多數是阿萬音鈴羽和比屋定真帆<ref name="rpgfan review psv"/>。

玩家還能以岡部的手機通信軟件RINE{{efn|影射自[[LINE|LINE]]。}}為媒介與他的朋友交流<ref name="destruct review"/>。在某些時候，遊戲會顯示一條通知表明岡部收到了一條消息，玩家可以選擇以不同的方式回信<ref name="rpgfan review psv"/><ref name="gem rine"/>。無論是發送文字還是[[貼圖|貼圖]]，系統都會把遊戲鎖定在對話介面上，而對方則會根據玩家的訊息給予相對的回應<ref name="hcg review"/><ref name="gem rine">{{cite web |url=http://gematsu.com/2015/11/steinsgate-0-video-introduces-rine-trigger-system |author=Romano, Sal |title=Steins;Gate 0 video introduces RINE Trigger system |accessdate=2017-01-30 |website=Gematsu |date=2015-11-13 |archive-url=https://web.archive.org/web/20160304052446/http://gematsu.com/2015/11/steinsgate-0-video-introduces-rine-trigger-system |archive-date=2016-03-04 |dead-url=no}}</ref>。與阿瑪迪斯不同的是，RINE的訊息不會影響遊戲的進度<ref name="rpgfan review psv"/>。

== 設定和角色 == 
{{See also|命運石之門#登場人物}}
[[File:Akihabara_buildings_02.JPG|195px]]]]
《命運石之門0》的故事設在與前作結尾不同的世界線（[[β|β]]），而舞台同樣是[[東京都|東京都]][[千代田區|千代田區]]的[[秋葉原|秋葉原]]。遊戲的主要角色包括[[東京電機大學|東京電機大學]]學生岡部倫太郎、他的朋友椎名真由理、[[神經科學|神經科學]]家牧瀨紅莉栖和綽號為「桶子」的[[駭客|駭客]]橋田至<ref name="Script">{{cite book | title=STEIN;GATE 0 公式資料集 Amadeus's Script| language=ja | publisher = [[KADOKAWA|KADOKAWA]] | isbn = 9784048657990 | accessdate=2018-08-04 }}</ref>{{Rp|66－78}}，他們意外發現可以使用手機和微波爐（角色們合稱它為「電話微波爐（暫名）」）穿越時空，方式為在手機發送訊息的同時把存儲數據數位化。後者被稱為「時空跳躍」，實質上是把記憶傳輸到較早時候的自己身上，此舉引起了秘密研究時間旅行的SERN{{efn|影射自英文簡稱為CERN的[[歐洲核子研究組織|歐洲核子研究組織]]。}}的注意，他們派遣其名為「巡行者」（Rounders）的團隊沒收主角們的時間機器，「巡行者」中包括自稱為記者的桐生萌郁<ref name="Script"/>{{Rp|82－85}}，在执行過程中他們亦殺死了真由理<ref name="CRoll">{{cite web|title=What You Need To Know Before You Start Steins;Gate 0|url=http://www.crunchyroll.com/anime-feature/2018/04/05/what-you-need-to-know-before-you-start-steinsgate-0|author=Skyler Allen|publisher=[[Crunchyroll|Crunchyroll]]|date=2018-04-06|accessdate=2018-08-04}}</ref>。

在《命運石之門0》的世界觀中，時間包含一個分出許多分支且會重新合併的架空歷史（作中稱為「世界線」），這意味著某些事件在到達分歧點時必定會發生，除非過去的事件發生足夠的變化，使歷史出現顯著性的不同，以致其到達另一個有著不同分歧點的世界線集群（即作中所指的「世界線收束範圍」）<ref name="CRoll"/>。其中一個分歧點為2010年的真由理之死，為了避免此事發生，岡部回到過去並導致[[α|α]]世界線收束範圍更改為β，在這世界線中紅莉栖被殺，真由理存活，時間旅行也未被發現。在α和β世界線收束範圍內則有一條不受分歧點影響，名為「命運石之門」的世界線。遊戲中某些人能保留前個世界線的記憶而不受轉移的影響，擁有這現象的人已知有岡部，而岡部則稱該現象為「探知之眼」（Reading Steiner）<ref name="CRoll"/>。

遊戲的其他角色包括時間旅行者，至未來的女兒阿萬音鈴羽、真由理未來的養女椎名篝、紅莉栖的同僚比屋定真帆、維克多孔多利亞大學（Viktor Chondria University）的教授阿歷西斯·萊斯基寧（Alexis Leskinen）和茱蒂·雷耶斯（Judy Reyes）、岡部的朋友漆原琉華和菲利斯·喵喵、真由理的朋友阿萬音由季（鈴羽的母親和至未來的妻子）、中瀨克美和來島楓<ref name="Script"/>{{Rp|86－122}}。

<!--故事-->
評論家多數對遊戲的故事表示讚賞<ref name="rpgfan review psv"/><ref name="destruct review"/><ref name="fam review"/>。《Fami通》的評論家尤其喜愛其氛圍<ref name="fam review">{{cite magazine |title=シュタインズ・ゲート ゼロ |magazine=[[Fami通|Famitsu]] |publisher=[[Enterbrain|Enterbrain]] |language=ja |date=December 2016 |issue=1409}}</ref>。[[Destructoid|Destructoid]]的丹尼斯·卡登（Dennis Carden）認為對那些喜愛《命運石之門》的玩家來說，《命運石之門0》延續前作故事的方式令他們「務必要」玩這部作品，他整體上亦喜歡該部作品，即使其筆調比起前作更為黑暗但也不乏輕浮的時間，使他在遊玩時「笑的時間與哭的時間一樣多」<ref name="destruct review"/>。《RPGFan》的羅布·羅根（Rob Rogan）欣賞《命運石之門0》的故事，稱它發人深省，也令人感到興奮、憂鬱和痛心，但他也表示其劇情進展有時候沒有明確的目的，就像是被「人為拉長了」一樣<ref name="rpgfan review psv"/>。同樣供職於《RPGFan》的羅伯特·芬納（Robert Fenner）認同其同僚的感言，並說劇情中岡部在是否聯絡阿瑪迪斯一事陷入兩難是一個很好的假設，但這樣做就要把遊戲加長五分之一才合理<ref name="rpgfan review ps4"/>。《[[Hardcore_Gamer|Hardcore Gamer]]》的喬丹·赫爾姆（Jordan Helm） 也擁有類似的意見，他指出岡部與阿瑪迪斯的對話是《命運石之門0》的一大亮點，但那些以人物為中心的場景卻總是給人一種是拿來「充數」的感覺<ref name="hcg review"/>。

卡登認為大多數新角都有著出色且到位的表現，但有些角色看似只為了某些情節而存在<ref name="destruct review"/>。羅根表示岡部自前作以來其發展令他越發成為一個有趣的角色<ref name="rpgfan review psv"/>，而芬納則認為岡部的人物刻畫是《命運石之門0》的高潮，稱遊戲精闢地表現出他的{{link-en|自我憎恨|self-hatred}}和[[冒名頂替症候群|冒名頂替症候群]]，使大眾確信其患有重度[[抑鬱症|抑鬱症]]<ref name="rpgfan review ps4"/>。卡登和芬納都對遊戲使用多個視角的做法表示讚賞，稱此舉賦予角色更多的深度和可信度，讓玩家更加了解作中的角色<ref name="rpgfan review psv"/><ref name="destruct review"/>。

<!--遊戲玩法-->
評論家對《命運石之門0》的遊戲玩法意見不一，有些人覺得它過於複雜，亦有人覺得其情節太過簡單<ref name="rpgfan review psv"/><ref name="destruct review"/><ref name="rpgfan review ps4"/>。卡登對玩家若不遵循遊戲指引便難以到達不同結局一事表示不滿，他也說道有時候根本不知道自己做出的選擇會對劇情造成什麼影響<ref name="destruct review"/>；然而其同事芬納就對用首次遊玩時得來的情報，推敲出第二次遊玩時的決擇以探尋另一個結局一事感到有趣<ref name="rpgfan review ps4"/>。羅根認為《命運石之門0》的選擇系統比前作簡單，稱其是系列的一大退步<ref name="rpgfan review psv"/>。而赫爾姆則表示《命運石之門》的選擇大多帶有懸念和悔恨的氣氛，但其續作卻沒有這些東西<ref name="hcg review"/>。芬納還表示欣賞5pb.改進了從前作遺留下來的RINE系統，讓玩家查看他們之前發送的訊息，同時在發送訊息後自動[[存檔|存檔]]遊戲<ref name="rpgfan review ps4"/>。另外，卡登、羅根和赫爾姆都讚揚《命運石之門0》的用語系統，認為它能確保玩家理解遊戲中的概念和術語<ref name="hcg review"/><ref name="rpgfan review psv"/><ref name="destruct review"/>。

<!--內涵-->
《命運石之門0》的藝術指導、演示和音效都收到積極的評價<ref name="hcg review"/><ref name="rpgfan review psv"/><ref name="destruct review"/>，卡登表示其視覺效果「令人印象深刻」<ref name="destruct review"/>，羅根則形容遊戲的角色藝術給人一種「清晰且迷人」的感覺，即使每個角色的[[幀數|幀數]]有限<ref name="hcg review"/>。他還讚揚《命運石之門0》的背景藝術，稱大量的細節為場景增添了個性但又不會分散玩家的注意力，其音樂與每個場景的筆調更是天作之合<ref name="rpgfan review psv"/>。赫爾姆就非常喜歡該作使用多個與環境相關的角色[[精靈圖|精靈圖]]，而不僅僅是幾個靜止的人像，除此之外他亦讚許開發商對岡部精靈圖細節的關注，特別是有否表現出視覺信號和他的精神疲勞<ref name="hcg review"/>。
