'''會展中心站'''是[[深圳地鐵|深圳地鐵]][[深圳地鐵1号綫|1号綫]]和[[深圳地鐵4号綫|4号綫]]的[[換乘站|換乘站]]，位于[[深圳市|深圳市]]中心區[[福華路|福華路]]與[[中心二路|中心二路]]交匯處的地底，於2004年12月28日啟用。同時，本站是深圳地鐵第一個啟用後即為換乘站的車站。

== 車站結構 ==
會展中心站是一個地底車站，呈十字型。车站的結構共分三層，設有四條軌道。

大堂設置在地下一層，呈T字型，由[[深圳市地鐵集團有限公司|深圳市地鐵集團有限公司]]管理。1號線的月台設置在地下二樓，呈东西方向，為一[[島式月台|島式月台]]，由[[深圳市地鐵集團有限公司|深圳市地鐵集團有限公司]]管理，負責接載乘客前往[[羅湖站_(深圳)|羅湖]]或[[機場東站_(深圳)|機場東]]方向。4號線的月台設置在地下三樓，呈南北方向，為兩個[[側式月台|側式月台]]，由[[港鐵軌道交通（深圳）有限公司|港鐵軌道交通（深圳）有限公司]]管理，採港鐵[[東涌線|東涌線]]裝修設計標準，負責接載乘客前往[[清湖站|清湖]]或[[福田口岸站|福田口岸]]方向。

會展中心站的换乘方式为[[换乘站#垂直换乘|垂直换乘]]。乘客在1號線和4號線之間換乘時，需要按指示通過扶手電梯上落。一些不熟悉车站结构的乘客会先上车站大堂，再下月台以进行换乘。有關通道及設備皆由[[深圳市地鐵集團有限公司|深圳市地鐵集團有限公司]]管理。

在會展中心站以西，有一軌道負責接通1號線及4號線，仅供车辆调动使用，不对外运营。該軌道內的限速是每小時50公里。

[[深圳地鐵4号綫|4号綫]]全线開通之後，該站的轉乘客流劇增，在高峰期需實施控制客流的措施，因此現時在該站的大堂設置了[[鐵馬|鐵馬]]以便高峰時期進行客流管制。

車站站廳北邊部分空間被劃為[[連城新天地|連城新天地]]美食廣場，於2015年5月9日開業。

== 車站地下空間 ==
本站和[[崗廈站|崗廈站]]、[[購物公園站|購物公園站]]之間的地下空間，原為[[防空洞|人防設施]]，並改建為[[連城新天地|連城新天地]][[地下街|地下街]]，其中本站和[[購物公園站|購物公園站]]之間的地下空間已於2012年4月開放，而本站和[[崗廈站|崗廈站]]之間的地下空間則於2015年12月試業。另外車站站廳北邊空間（A、B出口之間）被劃為[[連城新天地|連城新天地]]美食廣場，於2015年5月9日開業。

== 歷史 ==
會展中心站隨深圳地鐵一期通車於2004年12月28日啟用。車站建設時，該站曾以金田站為名。
