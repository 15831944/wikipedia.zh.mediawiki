'''會津線'''（{{jpn|j=会津線|hg=あいづせん|rm=Aizu sen|f=y}}）是一條連結[[福島縣|福島縣]][[會津若松市|會津若松市]]的{{link-ja|西若松站|西若松駅}}與同縣[[南會津郡|南會津郡]][[南會津町|南會津町]][[會津高原尾瀨口站|會津高原尾瀨口站]]的鐵路線，由[[會津鐵道|會津鐵道]]經營。此路線原為[[日本國鐵|日本國鐵]]所指定的[[特定地方交通線|特定地方交通線]]'''會津線'''，在1987年[[日本國鐵分割民營化|日本國鐵分割民營化]]時由[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）短暫接收幾個月之後，轉移至為了接收該路線而特別設立的[[第三部門鐵路公司|第三部門業者]]會津鐵道經營。

== 路線資料 ==
* 管轄（事業類別）：會津鐵道（[[鐵路公司|第一種鐵道事業者]]）
* 路段（{{link-ja|營業距離|営業キロ}}）：西若松－會津高原尾瀨口 57.4公里
* [[軌距|軌距]]：1067毫米
* 站數：21個（包括起終點站）
* 複線路段：沒有（全線單線）
* 電氣化路段：會津田島－會津高原尾瀨口 15.4公里（[[直流電|直流電]]1500[[伏特|V]]）
* 非電氣化路段：西若松－會津田島 42.0公里
* [[閉塞_(鐵路)|閉塞方式]]：特殊自動閉塞式（軌道回路檢知式）<ref name = "rj592" />
