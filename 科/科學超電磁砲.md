《'''-{zh-cn:某科学的; zh-tw:科學}-{{Ruby|超電磁砲|RAILGUN||en}}'''》（{{jp|j=とある科学の{{ruby-ja|超電磁砲|レールガン}}|hg=とあるかがくのレールガン|rm=Toaru Kagaku no Rērugan|lead=yes}}，{{lang-en|''A Certain Scientific Railgun''}}）是[[輕小說|輕小說]]《'''[[魔法禁書目錄|-{zh-cn:魔法的; zh-tw:魔法}-{{Ruby]]'''》的外傳漫畫，[[鎌池和馬|鎌池和馬]]原作、[[冬川基|冬川基]]作畫。主角為[[御坂美琴|御坂美琴]]，而原作主角[[上条當麻|上条當麻]]則是此作的男配角{{注|在絕對能力篇为男主角}}。

漫畫版在《[[月刊Comic電擊大王|月刊Comic電擊大王]]》2007年4月號開始連載，目前仍在連載中。第71話最後公布要發展全新章節，連載並於10月份再開。動畫版一期於2010年3月在日本播映完畢，二期《科學超電磁砲S》{{注|2012年10月21日由ASCII MEDIA WORKS在幕張所舉辦的「電擊20周年祭」上宣布制作决定}}於2013年4月開始首播於日本，在當年九月播映完畢。2018年10月宣布第三期製作。

== 劇情 ==
故事發生在面積佔據[[東京都|東京都]]的三分之一，居住著230萬名人口且其中八成人口是學生的巨大都市「學園都市」。學園都市的所有學生均會接受[[超能力|超能力]]開發，藉由[[藥物|藥物]]、[[催眠術|催眠術]]與通電刺激等方式取得超能力。能力者以範圍和威力分為LV0至LV5。

主角[[御坂美琴|御坂美琴]]是學園都市中僅七位LV5（超能力者）的其中一人，排行第三。她是擁有操縱[[電擊|電擊]]能力的「電擊使」，因其特有的絕招（將硬幣以音速的3倍的初始速度發射）「超電磁炮」而得名。

本作不但通過美琴的視角來描繪學園都市的平常而不平凡的日常生活，也敘述學園都市秘密進行非[[人道|人道]]性質的實驗，從而使大家對於本作及本篇《[[魔法禁書目錄|魔法禁書目錄]]》的背景（世界觀中科學側）的認識也慢慢變得清楚。

== 電視動畫 ==
原作為《魔法禁書目錄》的外傳漫畫，於2009年10月至2010年3月19日在日本開始播放24集[[電視動畫|電視動畫]]<ref>{{cite web|url=http://www.mxtv.co.jp/railgun/nextstory.php?p=23|title=第1話|publisher=[[東京都會電視台|東京都會電視台]]|accessdate=2013年8月23日|language=ja}}</ref><ref>{{cite web|url=http://www.mxtv.co.jp/railgun/nextstory.php?p=0|title=第24話|publisher=[[東京都會電視台|東京都會電視台]]|accessdate=2013年8月23日|language=ja}}</ref>。後半（第13集之後）劇情的展開是根據原作者[[鎌池和馬|鎌池和馬]]的意見所製作的原創小故事。

2012年10月21日，電擊文庫創立20周年纪念作品《[[劇場版魔法禁書目錄：恩底彌翁的奇蹟|劇場版 魔法禁書目錄：恩底彌翁的奇蹟]]》後宣布製作第二季，並名為《科學的超電磁砲S》，2013年4月於[[TOKYO_MX|TOKYO MX]]、[[中部日本放送|中部日本放送]]、[[每日放送|每日放送]]、[[BS11|BS11]]、[[AT-X|AT-X]]、[[栃木電視台|栃木電視台]]播送，對應原作的妹妹篇内容。和第一期相同，後半（第17集之後）劇情也為原創劇情。

2018年10月7日，在電撃文庫25周年記念活動「秋之電撃祭」上，官方宣布製作第三季<ref>{{Cite web |url = http://dengekionline.com/elem/000/001/810/1810409/ |title = TVアニメ『とある科学の超電磁砲』第3期の制作が決定！ |author = てけおん |date = 2018-10-07 |publisher = DENGEKI ONLINE |language = ja |accessdate = 2018-10-07}}</ref>。
