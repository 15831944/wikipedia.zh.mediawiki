'''科隆火车总站'''（{{lang-de|Köln Hauptbahnhof}}）是[[德國|德國]][[科隆|科隆]]市的主要火車站，每日服務280,000人次，是德國第五繁忙的火車站。

由於科隆位置靠近[[法國|法國]]、[[荷蘭|荷蘭]]等地，車站成為很多高速火車及過夜火車的停靠站，如[[ICE列車|ICE]]、[[大力士高速列車|大力士]]等。

中央車站位處市中心，旁邊便是著名的[[科隆大教堂|科隆大教堂]]。

== 歷史 ==
科隆第一個中央車站始建於1857年，到1889年決定擴建。[[第二次世界大戰|第二次世界大戰]]之後，車站只剩下頭等及二等候車室完好無缺，故於1953年開始重建，至1957年完工。

1991年車站加設市郊鐵路（S-Bahn）月台，2000年將地面樓層改裝成購物中心。2002年8月1日，[[科隆－萊茵/美因高速鐵路|科隆－萊茵/美因高速鐵路]]正式通車。

== 車站設計 ==
車站共有9個供長途及區域火車使用的月台、2個市郊火車（S-Bahn）月台及2個地鐵（U-Bahn）月台。

== 未來發展 ==
[[德國鐵路|德國鐵路]]早於2007年便計劃開辦由科隆經[[布魯塞爾|布魯塞爾]]直達[[倫敦|倫敦]][[聖潘克拉斯火車站|聖潘克拉斯火車站]]的服務，預計於2015年可投入服務。<ref>http://www.thisislondon.co.uk/standard/article-23801905-the-train-at-st-pancras-will-be-departing-for-germany-via-channel-tunnel.do</ref>
