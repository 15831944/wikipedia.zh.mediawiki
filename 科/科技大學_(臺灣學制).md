'''科技大學'''（簡稱[[科大|科大]]），是[[中華民國|中華民國]][[技職教育|技職教育]]體系的最高層級教學機構。科大的主要學制為[[高級中等學校|高級中等學校]]考試後錄取的四年制技術學院（簡稱'''四技'''），另一體系則是自國中畢業可就讀的五年制專科學校（簡稱'''五專'''），另外還有二年制技術學院（簡稱'''二技'''）供專科生就讀、及附設二年制專科學校（簡稱'''二專'''）供高職生就讀，但目前僅有少部分學校設置。科大亦可設置[[研究所|研究所]]，培育[[碩士|碩士]]及[[博士|博士]]等研究人才。

校名[[英語|英語]]譯名多使用「{{lang|en|'''University of Science and Technology'''}}」或「{{lang|en|'''University of Technology'''}}」，部分學校則直接使用[[大學|大學]]的英文名稱「{{lang|en|'''University'''}}」。

== 課程目標 ==
以培育農業、科技、工程及管理之高級技術專業人才為宗旨。

== 歷史 ==
* 1995年，教育部訂頒「績優專校改制技術學院附設專科部申請辦法」，鼓勵各辦學績優之專科學校改制技術學院，再改名科技大學。
* 1997年，出現臺灣第一批科技大學，為[[國立臺灣科技大學|國立臺灣科技大學]]、[[國立臺北科技大學|國立臺北科技大學]]、[[國立雲林科技大學|國立雲林科技大學]]、[[國立屏東科技大學|國立屏東科技大學]]、[[朝陽科技大學|朝陽科技大學]]等五間科技大學，其中朝陽科技大學是臺灣第一間私立科技大學。
* 1998年，國立高雄技術學院改名[[國立高雄第一科技大學|國立高雄第一科技大學]]，成為臺灣公私立科技大學中第一間加上飾名者。
* 1999年，南臺技術學院改名[[南臺科技大學|南臺科技大學]]。

== 學制 ==
分為[[學士|學士]]學位班、[[碩士|碩士]]學位班及[[博士|博士]]學位班。

*學士學位班
**'''學士後'''：招收'''（[[大學|大學]]'''、'''[[科技大學|科技大學]]'''）畢業生，不接受同等學歷（力）畢業考生入學，限制必須具學士畢業生(含應屆畢業生)考生報考。修業為期一至二年，修業期滿成績及格者，可取得第二學位專業學士後學位學程[[學士|學士]]學位證書。
**'''四技'''：招收'''[[技術型高級中等學校|高職]]'''、'''[[普通型高級中等學校|高中]]'''、'''[[綜合型高級中等學校|綜合高中]]'''畢業生或具同等學歷（力）考生入學，修業為期四年，修業期滿成績及格者，可取得[[學士|學士]]學位證書。
**'''二技'''：招收'''[[專科學校|專科學校]]'''（'''[[五專|五專]]'''、'''[[二專|二專]]'''）畢業生或具同等學歷（力）考生入學，修業為期二年，修業期滿成績及格者，可取得[[學士|學士]]學位證書。

*碩士學位班
招收大學校院及技術學院畢業生或具同等學力考生入學，修業期限一至四年。應修學分數（不含碩士論文學分）24學分以上。修畢應修課程，並提出論文通過考核，授予[[碩士|碩士]]學位。

*博士學位班
招收具有碩士學位者，修業期限二至七年。應修學分數18學分（不含論文學分）以上。修畢應修課程，並通過資格審查及論文考核，授予[[博士|博士]]學位。

== 課程 ==
採學年學分制。原則上以授課滿18小時為一學分。科技大學四年制學生需修滿128學分，五年制專科生需修滿220學分，二年制 二專學生需修滿80學分，二技學生需修滿72學分。研究所碩士班學生需修滿24學分及完成碩士論文，博士班學生需修滿18學分及完成博士論文。

== 主管機關 ==
科技大學的主管機關為'''[[中華民國教育部|中華民國教育部]][[技術及職業教育司|技術及職業教育司]]'''。

== 入學管道 ==
科技大學採多元入學方案，考生可經由報考當年度的'''[[四技二專統一入學測驗|四技二專統一入學測驗]]'''或'''[[二技統一入學測驗|二技統一入學測驗]]'''，並以'''[[推薦甄試|推薦甄試]]'''、'''[[技優甄保|技優甄保]]'''、'''[[聯合登記分發|聯合登記分發]]'''等方式，或直接報名參加部分學校舉辦的獨立招生以取得入學資格。

[[高級中學|高中]]畢業生、[[高級職業學校|高職]]普通科或[[綜合高級中學|綜合高中]]學術導向畢業生，可透過報考當年度的'''[[大學學科能力測驗|大學學科能力測驗]]'''，並以此成績報名'''科技校院日間部四年制申請入學'''，可取得指定校系的入學資格。
