'''科林·克羅爾'''（{{lang-en|'''Colin Kroll'''}}，{{bd|1983年/1984年||2018年|12月16日}}<ref>{{Cite web |url = https://edition.cnn.com/2018/12/16/business/colin-kroll-hq-trivia-dead/index.html |title = Colin Kroll, co-founder of HQ Trivia and Vine, found dead |author = Jackie Wattles, Mark Morales |date = 2018-12-17 |publisher = CNN |language = en}}</ref>），美國企業家。他與[[羅斯·尤蘇波夫|羅斯·尤蘇波夫]]共同創立了影片應用程式[[Vine|Vine]]<ref>{{Cite web |url = https://adage.com/article/digital/hq-trivia-s-scott-rogowsky-face-hot-game-show/311941/ |title = HQ Trivia's Scott Rogowsky Doesn't Want You To Cheat (And Yes Those Eyebrows are All Natural) |author = Megan Graham |date = 2018-01-16 |publisher = AdAge |language = en}}</ref>（被[[Twitter|Twitter]]收購<ref>{{Cite web |url = http://allthingsd.com/20121009/twitter-buys-vine-a-video-clip-company-that-never-launched/ |title = Twitter Buys Vine, a Video Clip Company That Never Launched |date = 2012-10-09 |publisher =  AllThingsD |language = en}}</ref>），以及{{link-en|HQ Trivia|HQ Trivia}}。

== 生平 ==
克羅爾曾就讀[[奧克蘭大學_(密歇根州)|奧克蘭大學]]，修讀電腦科學。

他曾於2007年至2009年在[[雅虎|雅虎]]擔任工程部經理，並於2009年至2013年在Jetsetter擔任首席技術官。<ref>{{Cite web |url = https://www.nytimes.com/2018/12/16/nyregion/hq-trivia-overdose-colin-kroll.html |title = HQ Trivia and Vine Co-Founder Colin Kroll Dies, Apparently of Overdose, Police Say |author = Luis Ferré-Sadurní, Sapna Maheshwari |date = 2018-12-16 |publisher = The New York Times |language = en}}</ref>

== 逝世 ==
克羅爾因為疑似[[吸毒過量|吸毒過量]]而在紐約市逝世，終年34歲。<ref>{{Cite web |url = https://techcrunch.com/2018/12/16/hq-trivia-and-vine-co-founder-colin-kroll-found-dead-of-suspected-overdose/ |title = HQ Trivia and Vine co-founder Colin Kroll found dead of suspected overdose |author = Natasha Lomas |publisher = TechCrunch |language = en}}</ref>
