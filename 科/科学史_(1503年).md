'''[[科学史|科学史]]上的1503年'''发生了众多事件，本条目撷取其中部分罗列如下：

== 天文学 ==
* 7月23日，[[冥王星|冥王星]]运行至[[海王星|海王星]]轨道以外（因[[公转|公转]]轨道[[偏心率|偏心率]]较大，冥王星有时处于比海王星离[[太阳|太阳]]更近的位置），之后冥王星在这段海王星轨道外的[[宇宙空间|宇宙空间]]内运行了233年。但受科学技术限制，当时人们并不知道此事。<ref>{{cite web | title = When Pluto Transits Inside the Orbit of Neptune | url = http://www.auxmaillesgodefroy.com/outer_planet_cycles | deadurl = yes | archiveurl = https://web.archive.org/web/20130105072439/http://www.auxmaillesgodefroy.com/outer_planet_cycles | archivedate = 2013-01-05 }}</ref>

== 航海 ==
[[File:Grand_Cayman_Island.jpg|thumb]]中鸟瞰开曼群岛]]
* 5月10日，[[克里斯托弗·哥伦布|克里斯托弗·哥伦布]]发现了[[开曼群岛|开曼群岛]]，他曾因岛上栖息了数量众多的[[海龟|海龟]]而将海岛命名为“龟岛”（{{lang-es|Las Tortugas}}）。

== 技术 ==
* 4月21日，[[切里尼奥拉战役|切里尼奥拉战役]]（Battle of Cerignola）中[[西班牙帝国|西班牙帝国]]战胜了[[法兰西王国|法兰西王国]]，这次战役被后世认为是历史上第一场以[[火药枪|火药枪]]取胜的战役。
* [[朱利亚诺·达·桑迦洛|朱利亚诺·达·桑迦洛]]（Giuliano da Sangallo）利用[[堡垒|堡垒]]（bastions）这种新技术建造了[[托斯卡纳|托斯卡纳]][[阿雷佐|阿雷佐]]的城墙。<ref>{{cite book|author=James Burke |first=James|last=Burke|title=Connections|location=London|publisher=Macmillan|year=1978|isbn=0-333-24827-9|pages=255–256}}</ref>

== 诞生 ==
* 12月14日，[[诺斯特拉达姆士|诺斯特拉达姆士]]，[[法国|法国]][[物理学家|物理学家]]及[[占星家|占星家]]（[[科学史_(1566年)|1566年]]卒）

== 逝世 ==
* [[彼得·舍费尔|彼得·舍费尔]]（Peter Schöffer），[[德国|德国]][[印刷者|印刷者]]，现代[[字体排版学|字体排版学]]开创者（1425年生）
