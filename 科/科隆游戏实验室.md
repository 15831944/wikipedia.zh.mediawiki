'''科隆游戏实验室'''（{{lang-en|'''Cologne Game Lab''' (CGL)}}）是由 Björn Bartholdy 和 Gundolf S.Freyermuth 在2010年成立的高等教育機構，為[[科隆應用技術大學|科隆應用技術大學]]藝術學院（{{lang-de|Fakultät für Kulturwissenschaften}}）構成院校之一<ref>{{cite web |title=Fakultät für Kulturwissenschaften - TH Köln |url=https://www.th-koeln.de/kulturwissenschaften/fakultaet-fuer-kulturwissenschaften_2458.php |publisher=[[科隆應用技術大學|科隆應用技術大學]] |accessdate=2018-12-27 |location=[[德國|德國]][[科隆|科隆]] |language=de}}</ref><ref name="hp-abt">{{cite web |title=About - Institute |url=http://www.colognegamelab.de/institute/about/ |publisher=科隆遊戲實驗室 |accessdate=2018-12-27 |location=[[德國|德國]][[科隆|科隆]] |language=en}}</ref>，並於初期獲得歐洲區域發展基金（ERDF）和歐洲社會基金（ESF）的「目標2」（Objective 2）資助<ref name="hp-abt"/>。{{ld|科隆国际电影学校|Internationale filmschule köln}}（IFS）、[[丹麥電影學院|丹麥電影學院]]、[[EA|EA]]和[[Ubisoft_Blue_Byte|Ubisoft Blue Byte]]均為其合作夥伴。<ref>{{cite web |title=Faculty & Staff - Institute |url=http://www.colognegamelab.de/institute/people/ |publisher=科隆遊戲實驗室 |accessdate=2018-12-27 |location=[[德國|德國]][[科隆|科隆]] |language=en}}</ref> <ref>{{cite web |title=Gaming-Aid e.V.: Erstes Stipendium vergeben |url=http://www.4players.de/4players.php/spielinfonews/Spielkultur/36136/2179449/Gaming-Aid_eV-Erstes_Stipendium_vergeben.html |website=4players.de |accessdate=2018-12-19}}</ref><ref>{{cite web |title=Gaming-Aid vergibt erstes Stipendium an Studentin der TH Köln |url=http://www.pcgames.de/Spiele-Thema-239104/News/Gaming-Aid-vergibt-erstes-Stipendium-an-Studentin-1270409/ |website=pcgames.de |accessdate=2018-12-19}}</ref><ref>{{cite web |title=Cologne Game Farm: TH Köln eröffnet Inkubator |url=https://www.gameswirtschaft.de/karriere/cologne-game-farm-th-koeln/ |website=gameswirtschaft.de |accessdate=2018-12-19}}</ref>

科隆遊戲實驗室旨在進一步培训互動型內容人才，並著重研究和开发非线性結構的互动游戏和影视。該校提供以英语授課学士及碩士学位课程，涵蓋[[電子遊戲|電子遊戲]]、[[3D動畫|3D動畫]]及[[游戏设计|游戏设计及开发]]三大範疇<ref name="hp-abt"/>。<ref>{{cite web |title=Study |url=http://www.colognegamelab.de/study/ |publisher=科隆遊戲實驗室 |accessdate=2018-12-27 |location=[[德國|德國]][[科隆|科隆]] |language=en}}</ref>目前該校有九名教授及來自30多個國家的300名学生。<ref>{{cite web |title=Faculty & Staff - Institute |url=http://www.colognegamelab.de/institute/people/ |publisher=科隆遊戲實驗室 |accessdate=2018-12-27 |location=[[德國|德國]][[科隆|科隆]] |language=en}}</ref>

== 活動 ==
=== Clash of Realities ===
科隆遊戲實驗室每年都會連同科應大與IFS、EA等學術及產業機構舉辦研討會「Clash of Realities」，探討電子遊戲的藝術設計、技術發展和社會知覺，以及遊戲的文化傳播。<ref>{{cite web |title=Clash of Realities |url=http://clashofrealities.com |publisher=Clash of Realities |accessdate=2018-12-27 |language=en}}</ref>

=== 全球遊戲創作營 ===
科隆游戏实验室自2010年起參與舉辦遊戲即興創作活動「全球遊戲創作」（Global Game Jam）的科隆場，並商借姊妹校{{ld|科隆国际设计学院|Köln International School of Design}}（KISD）場地。<ref>{{cite web |title=CGLjam - Global Game Jam® |url=https://globalgamejam.org/users/cgljam |publisher=Global Game Jam |accessdate=2018-12-27}}</ref>
