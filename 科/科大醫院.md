'''科大醫院'''（{{lang-en|The University Hospital}}）位於[[澳門|澳門]][[氹仔|氹仔]][[偉龍馬路|偉龍馬路]][[澳門科技大學|澳門科技大學]]H座大樓，於2003年10月8日獲[[澳門衛生局|澳門衛生局]]批准成立，隸屬於澳門科技大學基金會，是一間非牟利機構，是澳門第一所具有[[中西醫結合|中西醫互補]]為特色的綜合性醫院。

科大醫院作爲澳門科技大學[[中醫藥|中醫藥]]學院、醫學院及藥學院的教、研基地，匯集國外、[[香港|港]][[澳門|澳]][[臺灣|臺]]地區和[[中國大陸|中國大陸]]的著名醫學專家與教職人員，肩負中醫藥學院的臨床教學工作及醫藥的臨床研究任務。

== 歷史 ==
[[File:University_Hospital,_UCTM.jpg|thumb]] 
醫院前身是為[[澳門科技大學|澳門科技大學]]的'''中醫診療中心'''。

2006年底，爲了拓展科大醫院的西醫服務，醫院的西醫門診、國際醫務中心、國際健康管理中心、醫學影像診斷中心、睡眠診療中心等已於開始投入服務。

== 門診 ==
科大醫院下設西醫部、中醫部、睡眠診療中心、國際健康管理中心、國際醫務中心、中醫藥學院專家臨床中心、臨床化驗診斷中心、醫學影像診斷中心，同時可以提供皮膚及激光、綜合康復診療、腫瘤綜合治療、治未病服務等。

;西醫門診
:全科、普外科、[[皮膚|皮膚]]科、[[心臟科|心臟科]]、[[婦科|婦産科]]、[[兒科|兒科]]、[[骨|骨]]科、[[眼科|眼科]]、[[精神科|精神科]]、[[耳|耳]][[鼻|鼻]][[喉|喉]]科、[[心理|心理]]科、家庭醫學專科、[[腸|腸]][[胃|胃]][[肝|肝]][[膽|膽]]科、[[脊骨|脊骨]]神經科、[[營養|營養]]科、康復科、[[物理治療|物理治療]]、[[泌尿科|泌尿科]]、[[疫苗注射|疫苗注射]]（[[流感|流感]]、[[甲型肝炎|甲]][[乙型肝炎|乙型肝炎]]、[[破傷風|破傷風]]、[[水痘|水痘]]）。

;中醫門診

== 設備 ==
[[File:Ambulance_of_UH_Macau.jpg|thumb]]
醫院設有全科門診、普通及專科診室，治療室40間，近60張專科病床；而醫院的住院部現階段已設有雙人病房、私家病房以及豪華病房，所有病房均設有寬頻上網，實行治療、事務兩不誤。另外醫院更設有[[藥物|藥物]]及健康應用研究所，內設中醫藥及保健研究開發中心、中藥及食物安全品質評估中心以及臨床及競賽化驗中心，擁有一隻一流的專業化醫療團隊。

中醫藥學院專家臨床中心設置有先進的化驗儀器及設施，進行臨床生化學、[[血液學|血液學]]、[[血清|血清]]學、[[免疫學|免疫學]]、[[微生物學|微生物學]]等不同項目的化驗檢查，為臨床醫師及臨床科研等提供優質而快速的化驗服務。中心臨床科室齊全，專科專病具有特色，同時中藥材地道，提供多種劑型、代煎服務，方便病人服用。化驗室的臨床生化學已進行澳洲 NATA 認證，NATA 是全球最多專為化驗所提供認證的機構，為澳洲政府認可，並為國際間認可。
