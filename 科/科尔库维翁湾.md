'''科尔库维翁湾'''（{{lang-gl|Ría de Corcubión}}）是西班牙加利西亚自治区[[拉科鲁尼亚省|拉科鲁尼亚省]]西南部[[下溺湾|下溺湾]]的一部分，也是[[夏亚斯河|夏亚斯河]]的最终[[河口|入海口]]<ref>{{cite web |title=Una cascada con horario de apertura y cierre |url=https://www.elmundo.es/suplementos/viajes/2006/53/1149610004.html |publisher=El Mundo |archiveurl=https://web.archive.org/web/20180123035852/http://www.elmundo.es:80/suplementos/viajes/2006/53/1149610004.html |archivedate=2018-01-23 |language=es |date=2006-06-15}}</ref>。
==地理==
科尔库维翁湾属于[[溺灣|溺灣]]地形，海岸分布有许多[[花岗岩|花岗岩]]<ref>{{cite web |title=La ría de Corcubión se hace más accesible |url=https://www.lavozdegalicia.es/noticia/carballo/corcubion/2018/08/28/ria-corcubion-accesible/0003_201808C28C2991.htm |publisher=[[La_Voz_de_Galicia|La Voz de Galicia]] |language=es |date=2018-08-28}}</ref>，湾中央是洛贝拉斯群岛（{{lang|gl|Illas Lobeiras}}），上面有一座[[灯塔|灯塔]]，方便来往船只的航行<ref>{{cite web |title=Las islas Lobeiras ocupan una posición estratégica en la ría de Corcubión |url=https://www.mundiario.com/articulo/sociedad/islas-lobeiras-ocupan-posicion-estrategica-ria-corucubion/20170304120341081152.html |publisher=mundiario |accessdate=2019-01-19 |language=es |date=2017-03-04}}</ref>。科尔库维翁湾北部是[[菲斯特拉角|菲斯特拉角]]和[[死亡海岸|死亡海岸]]，南临[[穆罗斯-诺亚湾|穆罗斯-诺亚湾]]。
==历史==
1596年10月末，西班牙海军上将{{tsl|es|Martín de Padilla|马丁·德帕迪利亚}}奉命进攻[[英格兰|英格兰]]，他率领的舰队在科尔库维翁湾遭到了猛烈的[[暴风雨|暴风雨]]袭击，约有25至31艘船只沉没<ref name=el>{{cite web |author=Xurxo Salgado |title=Los secretos de la Ría de Corcubión |url=https://www.elmundo.es/elmundo/2011/08/24/galicia/1314203312.html |publisher=[[世界报_(西班牙)|El Mundo]] |archiveurl=https://web.archive.org/web/20170131230535/http://www.elmundo.es:80/elmundo/2011/08/24/galicia/1314203312.html |archivedate=2017-01-31 |location=[[Santiago_de_Compostela|Santiago de Compostela]] |language=es |date=2011-08-24}}</ref>。1803年11月28日，[[拿破仑战争|拿破仑战争]]爆发后没多久，一艘从[[哈瓦那|哈瓦那]]前往[[拉科鲁尼亚|拉科鲁尼亚]]的[[法兰西|法兰西]]32门炮[[护卫舰|护卫舰]]在途径科尔库维翁湾时被英国{{tsl|en|HMS Ardent (1796)|热心号 (1796年)|热心号}}64门炮3等舰击沉<ref name=el/>。

科尔库维翁湾也是[[第一次世界大战|第一次世界大战]]期间[[德意志帝国|德意志帝国]]海军[[U型潜艇|U型潜艇]]的秘密停放点之一<ref>{{cite web |title=Corcubión abastecía clandestinamente a submarinos alemanes durante la I Guerra Mundial |url=https://www.elprogreso.es/articulo/galicia/corcubion-abastecia-clandestinamente-submarinos-alemanes-durante-la-i-guerra-mundial/20141213172300333009.html |publisher=El Progreso |archiveurl=https://web.archive.org/web/20190119072630/https://www.elprogreso.es/articulo/galicia/corcubion-abastecia-clandestinamente-submarinos-alemanes-durante-la-i-guerra-mundial/20141213172300333009.html |archivedate=2019-01-19 |location=Vigo |language=es |date=2014-12-13}}</ref>。
