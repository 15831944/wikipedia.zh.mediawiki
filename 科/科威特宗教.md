[[科威特|科威特]]的官方宗教是[[伊斯蘭教|伊斯蘭教]]，大部分科國公民是穆斯林。<ref name=ssi>{{cite web|url=http://www.strategicstudiesinstitute.army.mil/pdffiles/pub861.pdf |title=The Evolution of U. S.-Turkish Relations in a Transatlantic Context |publisher=[[Strategic_Studies_Institute|Strategic Studies Institute]] |page=87 |quote=Shiites comprise 60 percent of the population in Bahrain, 40 percent in Kuwait, 14 percent in Saudi Arabia, and 35 percent in Lebanon. |deadurl=yes |archiveurl=https://web.archive.org/web/20150318173523/http://www.strategicstudiesinstitute.army.mil/pdffiles/pub861.pdf |archivedate=March 18, 2015 }}</ref>除了伊斯蘭教外，科威特也有259-400名公民是基督徒，<ref name=irfr2007>{{cite web |url=https://www.state.gov/j/drl/rls/irf/2007/90214.htm |title=International Religious Freedom Report for 2007 |work=[[US_State_Department|US State Department]] |year=2007}}</ref>少數公民為巴哈伊教徒。 在科威特的外籍人士大多是穆斯林、印度教徒、基督徒和佛教徒。<ref name=irfr2007>{{cite web |url=https://www.state.gov/j/drl/rls/irf/2007/90214.htm |title=International Religious Freedom Report for 2007 |work=[[US_State_Department|US State Department]] |year=2007}}</ref>

== 伊斯蘭教 ==
{{Main article|{{link-en|科威特伊斯蘭教|Islam in Kuwait}}}}
科威特公民中大部分是穆斯林，估計有85％-95％是遜尼派，35％-40％是什葉派。<ref name=ssi />也存在其他小型穆斯林教派，但數量非常少。在非公民穆斯林中，遜尼派的人數無數據，什葉派估計有10萬人。<ref name=irfr2007 />
== 基督宗教 ==
{{Main article|{{link-en|科威特基督教|Christianity in Kuwait}}}}
基督教是科威特的少數族群的宗教。科威特有一個本地的基督徒社群，有200到400名的基督徒是科威特公民。2014年，有259名基督徒科威特人居住在科威特。<ref name=rel>{{cite web |url=http://stat.paci.gov.kw/arabicreports/#DataTabPlace:ColumnChartEduAge |title=Nationality By Religion and Nationality|work=Government of Kuwait|language=ar}}</ref> 科威特是除[[巴林|巴林]]以外唯一擁有本地基督徒的[[海灣阿拉伯國家合作委員會|海灣阿拉伯國家]]。在非公民中，估計有60萬名基督徒。科威特政府承認的基督宗教教派包括[[羅馬天主教會|羅馬天主教會]]、[[亞歷山大科普特正教會|亞歷山大科普特正教會]]、[[亞美尼亞使徒教會|亞美尼亞使徒教會]]、[[希臘正教會|希臘正教會]]、希臘天主教會（Melkite）、科威特國家福音派教會和[[聖公會|聖公會]]。也有許多基督教宗派團體因人數較少沒有得到官方承認，包括馬蘭卡拉敘利亞正教會（Malankara Orthodox Syrian Church）、馬爾托馬敘利亞教會（Mar Thoma Syrian Church）、[[耶穌基督後期聖徒教會|耶穌基督後期聖徒教會]]（摩門教）和[[基督復臨安息日會|基督復臨安息日會]]。<ref name=irfr2007 /> 在基督徒裡還有一​​些為穆斯林改宗的信徒，大部分不是科威特公民。2015年的一項研究統計，在科威特約有350名從伊斯蘭改宗的基督徒。<ref>{{cite journal|last1=Johnstone|first1=Patrick|last2=Miller|first2=Duane|title=Believers in Christ from a Muslim Background: A Global Census|journal=Interdisciplinary Journal of Research on Religion|date=2015|volume=11|page=16|url=https://www.academia.edu/16338087/Believers_in_Christ_from_a_Muslim_Background_A_Global_Census|accessdate=28 October 2015}}</ref>

== 猶太教 ==
{{main article|{{link-en|科威特猶太人歷史|History of the Jews in Kuwait}}}}
1950年代以前科威特有一些猶太人社群，然而在1980年代，所有的猶太人社群都離開了科威特。目前科威特的猶太人都為科威特外籍人士。<ref name=irfr2007 />

== 印度教 ==
{{main article|{{link-en|阿拉伯國家的印度教|Hinduism in Arab states}}}}
科威特估計有30萬印度教徒，全數非科威特公民。
== 錫克教 ==
科威特有1萬名錫克教徒。<ref name="irfr2007" />
== 佛教 ==
科威特有10萬名佛教徒，全數非科威特公民。
== 巴哈伊 ==
科威特有一小部分公民信仰巴哈伊。<ref name=irfr2007>{{cite web |url=https://www.state.gov/j/drl/rls/irf/2007/90214.htm |title=International Religious Freedom Report for 2007 |work=[[US_State_Department|US State Department]] |year=2007}}</ref>
