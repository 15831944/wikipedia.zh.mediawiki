'''科乐美秘技'''（{{lang-ja|コナミコマンド}}，{{lang-en|Konami Code}}），是[[科乐美|科乐美]]公司一个著名的[[电视游戏|电视游戏]][[秘技|秘技]]，在多個科乐美遊戲中有出現。<ref name="KNM games">The editors of ''GamePro'' and ''GamerHelp.com'', "The 'Contra Code:' We Must Never Forget," ''GamePro'' 235 (April 2008): 63.</ref>儘管如此，此秘技亦在一些非科乐美遊戲中出現。<ref name="autogenerated1">{{cite web |url= http://www.1up.com/do/feature?cId=3108751&did=1 |title= Cracking the Code: The Konami Code |publisher= 1up.com |accessdate= 2008-03-15 |deadurl= yes |archiveurl= https://www.webcitation.org/66ixH5CE7?url=http://www.1up.com/features/cracking-code-konami-code |archivedate= 2012-04-06 }}</ref>

科乐美秘技要用[[電視遊樂器|電視遊樂器]]的搖桿輸入，其輸入碼為：{{key press|up}}{{key press|up}}{{key press|down}}{{key press|down}}{{key press|left}}{{key press|right}}{{key press|left}}{{key press|right}}{{key press|B}}{{key press|A}}。

==起源==
科樂美秘技是由橋本和久創造的。當時，和久正在開發1985年游戏{{nihongo|《[[宇宙巡航艦_(1985年遊戲)|宇宙巡航艦]]》|グラディウス}}。期間，他發現遊戲的難度太高，遂創造了這個秘技以協助玩家完成遊戲。<ref name="autogenerated1"/>

科樂美秘技可以用在很多科乐美的游戏上。<ref name="KNM games"/>它的第一次出现是在科乐美公司的1985年游戏宇宙巡航艦中<ref name="cheat Gradius">{{cite web|url=http://articles.techrepublic.com.com/5100-10878_11-6164935.html |title=Geek Trivia: The cheat goes on |publisher=techrepublic.com |accessdate=2008-04-16 |archiveurl=https://archive.is/20120527064250/http://www.techrepublic.com/article/geek-trivia-the-cheat-goes-on/6164935 |archivedate=2012-05-27 |deadurl=yes }}</ref>。后来，科乐美遊戲{{nihongo|《[[魂斗羅|魂-{斗}-羅]]》|魂斗羅}}更把科乐美秘技发扬光大。<ref name=neseur2>{{cite journal |last1= |first1= |last2= |first2= |year=1991 |title=Roboter in Rage - Probotector |journal=Video Games |volume= |issue=1/91 |pages= |publisher=Markt+Technik Verlag |doi= |url=http://www.kultpower.de/external_frameset.php3?site=%2Fvideogames_datenbank.php3%3Fgame_id%3D2&from_referer=http%3A%2F%2Fwww.kultpower.de%2Fkultpower_search.php%3Fsearch%3Dprobotector |accessdate=11 July 2011 |language=de }}</ref>《魂-{斗}-羅》第2代的密技輸入碼與第1代的有所分別，按法為：
:{{key press|right}}{{key press|left}}{{key press|down}}{{key press|up}}{{key press|A}}{{key press|B}}

此秘技適用在該公司所發行在不同款遊戲上，但也並非每次都適用，且在不同的遊戲中有不同的功效；如《宇宙巡航艦》中是玩家的武器配備立即達最顛峰的滿裝狀態<ref name="cheat Gradius"/>，而在其他遊戲則可能是關卡選擇、甚至是無敵等。

在《[[宇宙巡航艦_(1985年遊戲)|宇宙巡航艦]]》推出後，其續作又修改了科乐美秘技。於[[超級任天堂|超級任天堂]]版本的《[[宇宙巡航艦III－從傳說到神話－|宇宙巡航艦III－從傳說到神話－]]》中，原本的科乐美秘技會使玩家的飛船自爆。可是，若以{{key press|L}}和{{key press|R}}取代{{key press|←}}和{{key press|→}}的話，那麼玩家的飛船就會升級。<ref>[http://www.urbandictionary.com/define.php?term=Konami%20Code Urban Dictionary: Konami Code]</ref>在[[Wii|Wii]]版本的《[[宇宙巡航艦：重生|宇宙巡航艦：重生]]》中，科乐美秘技中的{{key press|A}}和{{key press|B}}分別被{{key press|1}}和{{key press|2}}取代。<ref>[http://www.ign.com/cheats/games/gradius-rebirth-wii-14267799 Gradius ReBirth Cheats & Codes]</ref>

==其他平台==
在其他平台上，科乐美秘技的不同之處主要為{{key press|A}}和{{key press|B}}，全因其他平台沒有這兩個按鈕。在[[Sony|Sony]][[PlayStation|PlayStation]]系列的遊樂器上，此秘技為「{{key press|up}}{{key press|up}}{{key press|down}}{{key press|down}}{{key press|left}}{{key press|right}}{{key press|left}}{{key press|right}}{{key press|×}}{{key press|○}}」，因為PlayStation系列遊樂器沒有B鈕、A鈕，但有相近對應的[[Image:PlayStationX.svg|15px]]鈕、[[Image:PlayStationCircle.svg|15px]]鈕。另外，科乐美秘技在[[手機遊戲|手機遊戲]]上則被改為「{{key press|up}}{{key press|up}}{{key press|down}}{{key press|down}}{{key press|left}}{{key press|right}}{{key press|left}}{{key press|right}}{{key press|5}}{{key press|7}}{{key press|3}}」（573的日語發音為「[[科樂美|科樂美]]」的[[諧音|諧音]]）。<ref>[http://jetcityorange.com/konami-code/ KONAMI CODE ↑ ↑ ↓ ↓ ← → ← →]</ref>而在iOS游戏[[乐动魔方#jubeat_plus|jubeat plus]]和[[动感弹珠#Reflec_Beat_plus|Reflec Beat Plus]]中也作为彩蛋出现，操作方式为在标题画面的空白区域依次按上、上、下、下、左、右、左、右的方向滑动，并点击游戏LOGO上的“B”和“A”字母。jubeat plus中可以令标题画面出现彩蛋（根据所选的主题不同而产生不同效果），Reflec Beat plus中则会令游戏里的判定效果变大。在街机版的jubeat saucer fulfill中，于模式选择画面以相对应的按键输入此秘技，可以进入隐藏模式设定菜单。街機版[[麻雀格鬥俱樂部|麻雀格鬥俱樂部]]以對家名字欄、自己名字欄、上家名字欄、下家名字欄來代替上、下、左、右，以下一步和胡牌按鍵來代替B和A，輸入後可以切換遊戲的一般[[背景音樂|背景音樂]]和特別版背景音樂（只提供給特定使用特別版{{link-ja|e-AMUSEMENT PASS|e-AMUSEMENT PASS}}的玩家）。

== 其他用途 ==
科乐美秘技亦是一個流行文化。<ref>[http://tvtropes.org/pmwiki/pmwiki.php/Main/KonamiCode Konami Code at TV Tropes]</ref>

在2012年[[迪士尼经典动画长片|迪士尼经典动画长片]]《[[无敌破坏王|无敌破坏王]]》中，糖果王利用科乐美秘技重新編程遊戲《甜蜜衝刺》。<ref>{{Cite web | url = http://www.avclub.com/articles/wreckit-ralph,88179/ | title = Wreck-It Ralph | first = Tasha | last = Robinson | date = November 1, 2012 | accessdate = November 19, 2012 | publisher = [[A.V._Club|A.V. Club]] }}</ref>

在卡通頻道動畫《[[探險活寶|探險活寶]]》第二季第16集中，主角阿寶在玩遊戲機嗶莫時，利用科乐美秘技擊敗青蛙頭目。<ref>{{cite web|url=http://tvbythenumbers.zap2it.com/2011/02/23/monday-cable-ratings-wwe-raw-leads-night-pretty-little-liars-skins-rises-big-time-rush-being-human-more/83568|title=Monday Cable Ratings: WWE Raw Leads Night; 'Pretty Little Liars,' 'Skins' Rises + 'Big Time Rush,' 'Being Human' & More|last=Seidman|first=Robert|work=TV by the Numbers|publisher=[[Zap2it|Zap2it]]|date=February 23, 2011|accessdate=2013-05-25}}</ref>

在卡通頻道動畫《[[阿甘妙世界|阿甘妙世界]]》廣告短片中，主角陶阿甘發現同學異形使用科乐美秘技的動作完成許多作弊的事時也開始仿效，結果讓自身產生跳針當機而失敗。

美國樂隊{{tsl|en|Deftones|盲音合唱團}}的第六個專輯《{{tsl|en|Saturday Night Wrist|星期六症候群 (專輯)|星期六症候群}}》中，有一首名為〈U，U，D，D，L，R，L，R，A，B，Select，Start〉的歌。此歌的歌名係改自科乐美秘技。<ref>{{cite journal|title=Albums for 2008: Deftones | journal = [[Kerrang!|Kerrang!]] | issue = 1191 | date = January 5, 2008 | page = 47}}</ref>

在Android版的[[Google_Play_Games|Google Play Games]]中, 以手指依次「上、上、下、下、左、右、左、右」滑動, 會秀出「A, B, >」的圖示。再依次按下「B, A, <」, 即可獲得"[[All_your_base_are_belong_to_us|All your game are belong to us]]"的成就。

===知名网站的彩蛋===

作為一個歷史悠久的秘技，不少知名网站都會製作關於科乐美秘技[[彩蛋_(視覺)|彩蛋]]。<ref name="newsweek_zombies">{{cite news|last=Carvin|first=Andy|title=Geek Trick Of The Week: Newsweek Vs. * [[Google_Reader|Google Reader]]：在阅读界面中输入此秘技后，左侧订阅列表会被替换成左下角有个[[忍者|忍者]]的深蓝色背景，并且在将文章标记为喜欢和取消喜欢时分别会有抛洒红心和心碎的动画。<ref>{{cite web|title=Google Reader 彩蛋升级，加入likes 桃心|url=http://www.guao.hk/posts/google-reader-easter-egg-ninja-likes.html|accessdate=2010-11-12}}</ref>（Google Reader已停止開發）
* [[Facebook|Facebook]]：輸入此秘技后再按一下「Enter」鍵，此后在頁面移動或按鍵時發出昡光。（現已移除）
* [[Twitter|Twitter]]：輸入此秘技后，页面上的小鸟logo会顺时针旋转一圈。
* [[Plurk|Plurk]]：輸入此祕技后，时间轴中的条目会全部墜落，重打一次或F5就會恢復正常。（改打↑↑↓↓←←→→AB會有負片效果）
* [[土豆网|土豆网]]：在任意视频播放页面中输入此秘技后，视频播放器会暂时消失，并且从左侧飞入一个写有“信土豆，死后原地复活，状态全满。MP+100%（有時則是HP+100%）”的标签。提示稍后會自動消失并恢复显示播放器，期间视频播放不会暂停。（已不再支援彩蛋）<ref>{{cite web|title=隐藏有“上上下下左右左右BA”彩蛋的网站|url=http://www.zfreet.com/post/uuddlrlrba-code-on-web.html|accessdate=2010-11-12}}</ref>
* [[球迷网|球迷网]]：在首页输入秘技后即刻出现经典动画片《[[灌籃高手|灌籃高手]]》mv flash，按Esc可退出。
* [[QQ音乐|QQ音乐]]：在网页版（y.qq.com）输入秘技后，页面顶部会出现魂斗罗，同时播放魂斗罗游戏背景音乐（2015年12月測試，已不再支援彩蛋）。
* [[iNDIEVOX_独立音乐网|iNDIEVOX 独立音乐网]]：在网站上任意处输入方向键「上上下下左右左右」与「BA」能开启测试中以情绪效价与唤醒度播放音乐的隐藏功能 VA Radio 。
*[[守望先锋|暴雪娛樂Overwatch官方網站]] 輸入此秘技後會出現D.VA的小彩蛋
