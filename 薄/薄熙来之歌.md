《'''薄熙来之歌'''》{{NoteTag|“'''薄熙来之歌'''”是这首歌曲最通用的名称。词作者孟凡晓在[[中国原创音乐基地|中国原创音乐基地]]网站上使用的歌名为“'''薄熙来之歌'''”{{ref|web1|[外链]}}。曲作者李磊于2012年2月制作和发布的歌曲视频{{ref|video1|[外链]}}中的标题为“'''薄熙来打黑'''”，副标题为“'''献给[[重庆市公安局|重庆市公安局]][[王立军|王立军]]及其战友！'''”。}}，是2009年[[中国|中国]]民间两位音樂人──[[山东|山东]]诗人[[孟凡晓|孟凡晓]]和[[江西|江西]]音乐人[[李磊_(音乐人)|李磊]]合作作词、李磊作曲的一首以[[重庆打黑除恶专项行动|重庆打黑除恶专项行动]]为题材、赞扬时任[[中国共产党|中共]][[中国共产党第十七届中央委员会|第十七届]][[中国共产党中央政治局|中央政治局]]委员兼[[中国共产党重庆市委员会|重庆市委]]书记[[薄熙来|薄熙来]]的[[红歌|红歌]]。[[2010年2月|2010年2月]]初，李磊拍摄、制作了歌曲视频并发表在[[互联网|互联网]]上，此歌迅速走红。

== 作者背景 ==
{{Main|李磊 (音乐人)|孟凡晓}}

[[作曲家|曲作者]]（参与作词）、原唱兼原版视频制作者[[李磊_(音乐人)|李磊]]，男，[[江西人|江西人]]，1962年出生，曾为国有[[摩托车厂|摩托车厂]][[工人|工人]]，2007年“[[下岗|下岗]]”后无固定收入来源，在[[江西|江西]]各地做小生意，创作此歌前从未到过重庆，也没有实地接触过山东的孟凡晓。他爱好[[音乐|音乐]]，此前已创作了约100首歌曲，多与时事政治有关，如《怒吼吧！[[埃及|埃及]]雄獅》（歌颂[[2011年埃及革命|2011年埃及革命]]）、《浩然正氣[[趙紫陽|趙紫陽]]》、《[[打假|打假]]猛士》（“献给[[方舟子|方舟子]]先生”）、《[[十里長街|十里長街]]》、《自由的力量》、《[[中國夢|中國夢]]》、《拥护咱们的[[习近平|习近平]]》等等。他希望拥有自己的录音工作室，自稱“紅衣大叔”，并梦想成为“中国的[[苏珊·波伊尔|苏珊大妈]]”。<ref name="zaobao100302" />

主要词作者[[孟凡晓|孟凡晓]]，男，[[山东人|山东人]]，诗人，时年约23岁，自称是[[孟子|孟子]]的第74代传人。<ref name="zaobao100302" />

== 创作背景 ==
{{See also|薄熙来|王立军|唱红打黑|唱读讲传|重庆打黑除恶专项行动}}

李磊说，他[[下岗|下岗]]后曾在江西摆地摊卖衣服，曾遭“[[黑社会|黑社会]]”[[两抢|抢走货品]]和[[敲诈勒索罪|勒索]]；[[重庆打黑|重庆打黑]]是[[2009年中国|2009年中国]]的社会热点话题，薄熙来领导打黑值得表扬，自己只是希望通过歌曲表达“[[民意|民意]]”，并非搞[[个人崇拜|个人崇拜]]。<ref name="zaobao100302" />

== 创作经过 ==
李磊说，[[2009年8月|2009年8月]]，他把自己作曲赞扬[[薄熙来|薄熙来]]在[[重庆打黑|重庆打黑]]的愿望告诉了孟凡晓，得到孟的同意。李磊花了4个月创作，谱曲后使用[[即时通讯软件|即时通讯软件]][[QQ|QQ]]的聊天功能与远在山东、未曾谋面的孟凡晓一起作词，最后两人在孟凡晓主创的约4到5版歌词中选择了现在这个版本。<ref name="zaobao100302" />

== 歌词结构 ==
这首颂歌的歌词[[#外部链接|<sup>[外链]</sup>]]由两段组成，内容分别是：
* 第一段以“你的目光像刀剑，闪烁着寒光……刺破了腐败和黑暗”开头，称赞薄熙来反腐打黑手腕强硬，以致其威名令贪官“闻风丧胆”；接着，具体肯定了薄熙来治下改善重庆[[治安|治安]]、建设“[[五个重庆|平安重庆]]”的政绩，认为他不愧被誉为“打黑书记”；评价薄熙来为[[人民|人民]]的“靠山”（或“保护伞”）和“和平年代的[[英雄|英雄]]”。
* 第二段开场赞美薄熙来的笑容热情洋溢，对[[群众|群众]]的态度“非常亲善”，称其个性得到了[[老百姓|老百姓]]的“交口称赞”，转而又歌颂薄熙来“嫉恶如仇”的“[[英雄|英雄]]气概”，形成了爱憎分明（形如[[雷锋|雷锋]]所谓“对待同志要像春天般温暖，对敌人要像严冬般冷酷无情”那样[[阶级斗争|立场]]坚定、斗志强）的鲜明对比；随后，[[高度评价|高度评价]]薄熙来在重庆关心[[民生|民生]]、“[[整风|整风]]肃贪”，注重社会[[公平正义|公平正义]]的政策措施，表达了“[[人民|人民]]”寄托在薄熙来或类似人物身上的期盼。最后，全歌以两遍“薄熙来，薄熙来，中国需要千百个英雄！”结尾。

== 视频推广 ==
[[2010年2月|2010年2月]]初，李磊花了2天时间，在[[江西省|江西省]][[南昌市|南昌市]][[洪州|洪州]][[古城公园|古城公园]]使用[[数码相机|数码相机]]和[[三脚架|三脚架]]等设备自拍自唱，然后在自己家中编辑视频（[[钮理安|钮理安]]负责伴奏和[[MIDI|MIDI]]）。该视频{{ref|video1|[外链]}}以洪州古城公园为背景，主唱李磊为唯一演员，多次穿插薄熙来在不同场合的照片，全长约3分钟，片尾有“诚招[[植入式广告|植入式广告]]，收费低廉”、“2010年2月9日制作，拍摄于洪州古城”等[[字幕|字幕]]，被李磊传到网上后迅速走红。<ref name="zaobao100302" />

== 重庆反应 ==
[[2010年3月|2010年3月]]2日，李磊接受《[[联合早报|联合早报]]》记者采访时说，他在2010年[[春节|春节]]（2月14日）前联系过[[重庆电视台|重庆电视台]]，表达了希望受邀去重庆录制[[音樂影片|音乐电视片]]并在[[电视台|电视台]]播放的愿望，重庆台工作人员表示会报告其上级，但此后李磊未获答复。<ref name="zaobao100302" />

2010年3月6日下午，[[第十一届全国人民代表大会第三次会议|第十一届全国人民代表大会第三次会议]][[Template:全国人大代表/11/渝|重庆市代表团]]全体会议审议[[温家宝|温家宝]]代表[[中华人民共和国国务院|国务院]]向[[全国人民代表大会|大会]]所作的《[[政府工作报告|政府工作报告]]》。会后，[[第十一届全国人大代表|第十一届全国人大代表]][[薄熙来|薄熙来]]接受了记者的采访。当记者问及其第二任妻子、[[北京市昂道律师事务所|北京市昂道律师事务所]]（原名“北京市开来律师事务所”）原主任律师[[谷开来|谷开来]]时，薄熙来对夫人帮助打黑表示感谢：
{{Cquote|我的夫人[[谷开来|谷开来]]是[[中华人民共和国|中国]]第一批[[律师|律师]]。不仅法律知识，国际文化的知识也很丰富。她的知识，特别是法律知识在‘[[重庆打黑除恶专项行动|打黑]]’中给了我很大帮助。为了我，她做出了巨大牺牲。十几年前[[律师事务所|律师事务所]]办得正红火的时候急流勇退，专心做学问，我是很感动的。}}
有记者问薄熙来有没有听过网上流行的“薄熙来之歌”，薄熙来连连摆手，笑答<ref>[http://epaper.bjnews.com.cn/html/2010-03/07/content_72994.htm 薄熙来：打黑不干不行——重庆仍有五六百起命案未破，打黑将进行长期斗争；评价要看人民群众是不是满意] {{webarchive|url=https://web.archive.org/web/20100420053006/http://epaper.bjnews.com.cn/html/2010-03/07/content_72994.htm |date=2010-04-20 }}，2010年3月7日《[[新京报|新京报]]》头版。</ref>：
{{Cquote|'''某某人之歌'''要删掉。}}

== 后续反响 ==
民间批评者表示，作者在进行自我[[炒作|炒作]]或对[[薄熙来|薄熙来]]的[[个人崇拜|个人崇拜]]；还有[[阴谋论|阴谋论]]，猜测此歌可能是“[[捧杀|捧杀]]”薄熙来的政治阴谋<ref name="zaobao100302" />。

2010年，[[中国持不同政见者列表|中国著名异议人士]][[艾未未|艾未未]]等人到重庆市某监狱探视被重庆当局判刑的著名[[律师|律师]][[李庄|李庄]]时录制了翻唱《薄熙来之歌》的视频{{ref|video2|[外链]}}并上传到[[互联网|互联网]]，公开讽刺薄熙来。
{{Further|李庄案}}

2012年9月28日，中共中央决定给予薄熙来开除党籍、开除公职处分，对其涉嫌犯罪问题及犯罪问题线索移送司法机关依法处理。9月29日，中央媒体[[光明网|光明网]]发表评论员文章《薄熙来下场再证文革模式是一条死路》<ref>[[光明网|光明网]][[评论员|评论员]]：[http://view.gmw.cn/2012-09/29/content_5243033.htm 薄熙来下场再证文革模式是一条死路]，[[光明网|光明网]] 2012-09-29 08:15:52</ref>，其中提到薄歌：{{Cquote|薄熙来设计打造自己的魅力形象，鼓励放任那些肉麻地吹嘘和“歌颂”自己的文字和歌曲，但对任何不利于自己的言词甚至哪怕仅仅是表示，都要以其不被监督和限制的权力，动用专政的力量给予最严厉的打击。《'''薄熙来之歌'''》的诞生和扩散，反感薄熙来并有所表示的人所受到的非人道迫害，就是再好不过的证明。}}
