'''兴安军'''，[[满洲国|满洲国]]的一支部队，隶属[[满洲国军|满洲国军]]，负责守卫该国蒙古人聚居的[[興安省_(中華民國)|兴安省]]（{{link-ja|兴安四省|興安総省}}）和热河省，该区域大致相当于今[[内蒙古自治区|内蒙古自治区]]东部。兵员主要为[[蒙古族|蒙古人]]<!--当时尚无蒙古族这一概念（来源请求）-->，以骑兵为主力部队。

== 概况 ==
[[File:Hsingan_Army_Cavalry.JPG|thumb]]
1932年4月满洲国军建立，此时关东军制定了将内蒙古自治军（[[九一八事变|九一八事变]]时期以蒙古人为主组成的义勇军）编入满洲国军的方针。4月22日，斋藤恭平少佐、金川耕作大尉、生田吉五郎大尉、本间诚大尉被任命为兴安军的军事顾问。本间顾问在改编内蒙古自治军的同时，在兴安东分省、南分省、北分省选拔蒙古青年并加以训练。<ref name="mori"/>{{rp|115}}

同年5月，'''兴安南警备军'''于兴安南分省成立，司令官为[[巴特玛拉布坦|巴特玛拉布坦]]少将，参谋长为[[甘珠尔扎布|甘珠尔扎布]]。1933年4月，'''兴安北警备军'''于兴安北分省成立，司令官为[[乌尔金|乌尔金]]少将，'''兴安东警备军'''于兴安东省成立，代司令官为绰罗巴图尔上校。热河作战之后，原热河省北部拆分为兴安西分省，察东警备军（义勇军，司令官[[李守信|李守信]]）中由蒙古人组成的第4支队编入满洲国军，成为'''兴安西警备军'''，代司令官为[[乌古廷|乌古廷]]上校<ref name="mori"/>{{rp|115}}。1938年，'''兴安军管区'''成立，巴特玛拉布坦任司令官，统括原四支兴安警备军<ref name="makinami"/>{{rp|132}}。后改编为'''第9军管区'''（通辽）和'''第10军管区'''（海拉尔）<ref name="fujita"/>{{rp|185}}。

满洲国{{link-ja|中央陆军训练处|中央陸軍訓練処}}建立于1932年，负责军官的教育。然而由于语言和习惯等因素，兴安军的军官不适宜在此受训。1934年7月1日，[[满洲国国务院|军政部]]在[[双辽市|郑家屯]]的临时校址设立了专门培养蒙古人军官的'''兴安军官学校'''，校长由巴特玛拉布坦兼任，下永宪次少佐任顾问。第1期学生72人。次年8月1日，学校迁至[[乌兰浩特|王爷庙]]。兴安军事学校以复兴蒙古民族为宗旨，作为当时唯一的蒙古人高等教育机关，为了培训未来的领导者，不仅提供军事培训，也进行其他方面教育。1939年10月，学校改名为'''陆军兴安学校'''。<ref name="mori"/>{{rp|116}}<ref name="ozawa"/>{{rp|46,47}}

根据满洲国军刊行委员会（兰星会）所编《满洲国军》，蒙古士兵具有如下的强项：能够在缺乏地标的广阔草原上辨认方向、夜间到达目的地，战斗勇猛，马术娴熟，可以不执缰绳进行射击，可以长时间骑行。<ref name="makinami"/>{{rp|180}}

== 战历 ==
1932年9月，[[苏炳文|苏炳文]]在[[呼伦贝尔|呼伦贝尔]]地方发起叛乱。满洲国军协助关东军进行讨伐作战，本间诚顾问率领兴安南警备军参战<ref name="mori"/>{{rp|115}}。1933年的热河作战中，兴安南警备军司令官巴特玛拉布坦率2个团的兵力进军[[林西|林西]]、[[林东|林东]]方面，支援关东军。1935年起，兴安北警备军参与了[[哈尔哈庙战斗|哈尔哈庙战斗]]等3次与蒙古国之间的军事冲突。

1937年7月，[[抗日战争|中日战争]]爆发，中方军队越过热河省的长城边界后，满洲国军有部分部队也奉命出战，其中包括兴安南警备军。兴安南警备军推进至[[察哈尔省_(中华民国)|察哈尔省]]，此后接受日军指挥。在山西省[[阳高|阳高]]战斗之后，兴安南警备军的加藤连在9月13日作为先锋占领大同，此后转战山西各处，于10月回师。1938年，共产党的部队进犯热河省境，7月3日，甘珠尔扎布少将担任支队长，率兴安骑兵第5团、第2团约1000人出战，转战冀东各地，至12月历战45场，击溃高翔云部约5000人。<ref name="ozawa"/>{{rp|107,108}}

1939年[[诺门罕事件|诺门罕事件]]中，乌尔金中将率兴安骑兵第1、第2、第8团出战（称为'''乌尔金部队'''）。5月，刚刚编成的国防军'''兴安师'''（师长野村登龟江中将）作为核心组成'''兴安支队'''，率兴安骑兵第4、第5、第6、第12团等6000-7000人出击<ref name="ozawa"/>{{rp|152-155}}。7月1日，兴安支队在准备攻击诺罗高地之时遭受苏军炮击，部队指挥失灵，士气低下。此后的战斗中日系军官多有死伤，相继出现了逃兵<ref name="ozawa"/>{{rp|157,158}}。得到长野支队的增援后，兴安支队在7月10日以残部夺取三角山。7月21日的战斗中野村支队长负伤，返回后方，剩余的兴安支队约300人在8月2日与日军交替掩护回撤<ref name="ozawa"/>{{rp|162}}。兴安支队死伤2,895人<ref name="ozawa"/>{{rp|152}}。

1945年8月，苏联军队入侵满洲，兴安军即第9军管区、第10军管区受命进入战斗配置。第9军管区部队自通辽向奉天移动途中，有部分部队起事，杀死日系军官<ref name="ozawa"/>{{rp|230，231}}。8月16日凌晨，甘珠尔扎布中将出走，第9军管区失去指挥官而在当日解散。第10军管区部队受命向[[锡尼河|锡尼河]]附近移动，8月11日参谋长[[正珠尔扎布|正珠尔扎布]]举事，杀死日系军官29名，此后部队向苏联军队投降<ref name="ozawa"/>{{rp|232,233}}。陆军兴安学校的人员分为3个梯团出发行军，但全部在行军途中失去联系。

兴安师参加过诺门罕战役后改编为'''第2师'''，自1945年1月起守卫西部国境。苏联入侵时，第2师（师长林保治）因一部分部队派往冀东，实际兵力约3000人。第2师收到迎击苏联军队的命令后，部队丧失战意发生了逃兵和内乱。8月11日，师长决定解散残存部队。<ref name="ozawa"/>{{rp|236,238}}一部分起事的部队后来成为[[东蒙古人民自治政府|东蒙古人民自治政府]]的人民自治军。

== 编制 ==
1934年兴安军共3,495人，包括兴安东警备军、兴安西警备军、兴安南警备军、兴安北警备军。兴安东警备军代司令官绰罗巴图尔上校，驻[[博克图镇|博克图]]，辖兴安骑兵第1团、第2团、兴安步兵第1营、[[扎兰屯|扎兰屯]]医院。兴安西警备军代司令官[[乌古廷|乌古廷]]上校，驻[[林西|林西]]，辖兴安骑兵第3团、第4团、独立山炮兵第1连、独立通信第1连、林西医院。兴安南警备军司令官[[巴特玛拉布坦|巴特玛拉布坦]]少将，参谋长[[甘珠尔扎布|甘珠尔扎布]]，驻[[钱家店镇|钱家店]]，辖兴安骑兵第5团、第6团、独立山炮兵第2连、独立通信第2连、钱家屯医院。兴安北警备军司令官[[乌尔金|乌尔金]]少将，参谋长福龄，驻[[海拉尔|海拉尔]]，辖兴安骑兵第7团、第8团、兴安步兵第2营、海拉尔医院。<ref name="fujita"/>{{rp|169}}

1945年兴安军分为第9军管区、第10军官区。第9军管区司令官甘珠尔扎布少将，参谋长德山恭助上校，驻通辽，辖骑兵第48团、第49团（配属于[[铁石部队|铁石部队]]）、独立辎重队、独立炮兵营、独立飞行队、自动车队、第9宪兵团、第9通信队、通辽医院。第10军官区司令[[郭文林|郭文林]]中将，参谋长[[正珠尔扎布|正珠尔扎布]]少将，驻海拉尔，辖第2师、陆军兴安学校（校长乌尔金上将）、海拉尔宪兵团、海拉尔通信队、海拉尔医院、第五十三部队。第2师师长林保治少将，下辖步兵第36团、骑兵第46团、47团、炮兵团、通信队。<ref name="fujita"/>{{rp|185}}
