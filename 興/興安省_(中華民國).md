'''興安省'''是[[中華民國|中華民國]]在[[抗日戰爭_(中國)|抗戰]]勝利後設的[[東九省|東九省]]的其中一省，簡稱'''興'''。该省疆域大約等於今日的[[呼倫貝爾市|呼倫貝爾市]]全境及[[興安盟|興安盟]]部分。

[[国民政府|国民政府]]宣布设立之初，该地区实际由东蒙自治政府统辖，由于国民政府拒不承认东蒙自治政府，1946年5月28日，中共撤消东蒙自治政府，建立兴安省人民政府，由[[东北人民政府|东北人民政府]]和[[内蒙古自治运动联合会|内蒙古自治运动联合会]]双重领导。后中共方面裁撤兴安省，所属地域并入新成立的[[内蒙古自治区|内蒙古自治区]]，而国民政府方面一直沿用兴安省的区划。

== 省名由來 ==
“兴安”是满语，意为“极寒的地方”，与兴安岭同源。满洲国时期首次设立兴安省，后分为兴安东、西、南、北四省。

== 地理 ==
全省疆域大約等於今日的[[呼倫貝爾市|呼倫貝爾市]]，[[興安盟|興安盟]]部分。面積約為278437平方公里，人口1947年調查為32萬7563人，是當時[[中華民國|中華民國]]所轄人口最少的一個行省。西北[[俄羅斯|俄羅斯]]，西界[[蒙古地方|蒙古地方]]，南接[[遼北省|遼北省]]，東接[[黑龍江省|黑龍江省]]、[[嫩江省|嫩江省]]。

[[大興安嶺|大興安嶺]]橫穿中部，東部為 [[東北平原|東北平原]]、西部為[[蒙古高原|蒙古高原]]的延伸。境內主要有[[嫩江|嫩江]]、[[額爾古納河|額爾古納河]]、[[克魯倫河|克魯倫河]]等河流和[[呼倫湖|呼倫湖]]、[[貝爾湖|貝爾湖]]等湖泊。

== 歷史沿革 ==
自古即為[[游牧民族|游牧民族]]的牧場，境內大興安嶺為古代[[鮮卑|鮮卑]]族居住之地。[[清|清]]代時劃規[[黑龍江將軍|黑龍江將軍]]所轄之地，清[[光緒|光緒]]三十三年（西元1907年）設[[黑龍江省|黑龍江省]]，為該省所轄之區。

民國二十一年（1932年），[[奉天事變|奉天事變]]爆發，[[日軍|日軍]]侵佔[[中国东北地区|东北]]，淪為[[滿洲國|滿洲國]]領土。民國三十四年（1945年）[[日本投降|日本投降]]，國民政府於9月上旬，在[[重慶|重慶]]宣布劃分東北三省為九省二市，任命[[吳煥章|吳煥章]]為興安省政府主席，主持在[[海拉爾|海拉爾]]組建興安省政府，併在重慶成立興安省政府籌備處開始策劃接收工作。而[[中共|中共]]在[[蘇聯|蘇聯]]的協助下，率先進入東北。11月17日，蘇聯通知中共地方黨、軍領導機關和武裝力量撤出哈爾濱，準僅將城市政權移交給國民政府接管。為避免衡突，在中共北滿分局[[書記|書記]][[陳雲|陳雲]]主持下，11月22日，中共北滿分局、松江省委、省軍區等全部撤至[[賓縣|賓縣]]。

民國三十五年（1946年）1月，興安省主席吳煥章與松江省主席[[關吉玉|關吉玉]]、黑龍江省主席[[韓駿傑|韓駿傑]]及哈爾濱市市長[[楊綽菴|楊綽菴]]赴哈爾濱市成立各省、市政府辦事處。4月，蘇聯紅軍撤出哈爾濱市，各省、市政府也隨同蘇軍撤出哈爾濱。中共东北民主聯軍派遺山東部隊第七師和359旅一部於28日攻下哈爾濱市，於5月成立「興安省人民政府」，下轄[[興安|興安]]、[[哲里木|哲里木]]、[[那文木仁|那文木仁]]、[[呼倫貝爾|呼倫貝爾]]4盟，共29旗、縣。同年9月，呼倫貝爾盟改為自治區。10月，在海拉爾市成立呼倫貝爾自治政府。

民國三十六年（1947年）4月23日，內蒙古自治政府成立，撤'''興安省'''，所轄盟旗併入[[內蒙古自治區|內蒙古自治區]]。

吳煥章率領部分省府人員赴[[瀋陽|瀋陽]]建立興安省政府辦事處，但興安省政府自始自終未能赴省境辦公。民國三十七年（1948年）年底[[遼瀋會戰|遼瀋會戰]]後，大部分地區為[[中共|中共]]所控。11月2日解放軍正式進入瀋陽，中華民國興安省政府滅亡。

== 行政區劃 ==
民國三十六年（1947年）6月，國民政府公布東北九省的行政區域，其中興安省的省會為[[海拉爾市|海拉爾市]]（今呼倫貝爾市海拉尔区）。全省轄有1市、7縣、11旗，因省境較小，不設行政督察區。但由於抗戰結束後，该省全境先后被[[东蒙古人民自治政府|东蒙古人民自治政府]]、中共[[兴安省政府|兴安省政府]]及[[内蒙古自治政府|内蒙古自治政府]]统辖之下，故國民政府公佈的行政區劃並未得到執行。
