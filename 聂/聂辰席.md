'''聂辰席'''（{{bd|1957年|7月||catIdx=N聂}}），[[河北|河北]][[灵寿|灵寿]]人，曾任原[[国家新闻出版广电总局|国家新闻出版广电总局]]局长、党组书记。现任[[中共中央宣传部|中共中央宣传部]]副部长，[[国家广播电视总局|国家广播电视总局]]局长、党组书记。[[中国共产党第十九届中央委员会委员列表|中共十九届中央委员]]。

==生平 ==
1974年9月参加工作，任[[河北省|河北省]][[灵寿县|灵寿县]]菅村供销社会计。1977年，在[[复旦大学|复旦大学]]计算机科学系程序设计专业学习。1980年，任[[河北省统计局|河北省统计局]]计算中心干部、主任科员、工程师。1987年，任河北省经济信息中心数据库和程序库部主任科员、工程师、副主任（正科级）。其间，1988年6月至1990年6月在[[河北财贸金融函授学院|河北财贸金融函授学院]]涉外经济专业学习。1992年，任[[河北省人民政府研究室|河北省人民政府研究室]]（经济研究中心）综合处主任科员、信息资料处副处长。1993年2月加入[[中国共产党|中国共产党]]<ref name=elyb/>。其间，1991年至1994年在[[中国科学技术大学|中国科学技术大学]]管理科学系管理科学专业研究生班学习，获工学硕士<ref name=elyb/><ref>{{Cite web |url=http://aga.ustc.edu.cn/news/view?id=2244 |title=广电总局副局长聂辰席校友将兼任中央电视台台长 |access-date=2017-07-13 |archive-url=https://web.archive.org/web/20171107024937/http://aga.ustc.edu.cn/news/view?id=2244 |archive-date=2017-11-07 |dead-url=yes }}</ref>。1995年，任河北省政府研究室（经济研究中心）教育科研处处长。1997年，任河北省政府研究室（经济研究中心）副主任、党组成员<ref name=elyb/>。

1999年，任中共[[邯郸市|邯郸市]]委常委、组织部部长。其间，1999年至2000年在[[新加坡南洋理工大学|新加坡南洋理工大学]]商学院管理经济学专业学习，获理学硕士。2001年，任中共邯郸市委副书记、市纪委书记。2003年，任中共邯郸市委副书记、市长。其间，1999年至2003年在[[天津大学|天津大学]]管理学院管理科学与工程专业学习，获管理学博士学位。2004年，任中共邯郸市委书记。其间：2004年至2006年在[[中国人民大学|中国人民大学]]法学院从事民商法[[博士后|博士后]]研究<ref>{{cite web |accessdate = 2012-04-20 |url = http://www.unn.com.cn/BIG5/14748/4657920.html |title = 聂辰席当选邯郸市委书记 王三堂杨慧当选副书记 |publisher = [[人民网|人民网]] }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2006年11月，任[[中共河北省委|中共河北省委]]常委、宣传部部长<ref>{{cite web |accessdate = 2012-04-20 |url = http://www.ce.cn/xwzx/gnsz/gdxw/200611/28/t20061128_9612405.shtml |title = 白克明当选河北省委书记郭庚茂、张毅为副书记 |publisher = [[中国经济网|中国经济网]]}}</ref>。2007年至2011年兼任河北省社会科学界联合会主席。2011年11月，任中共河北省委常委，[[河北省人民政府|河北省人民政府]]副省长、党组副书记<ref>{{cite web |accessdate = 2012-04-20 |url = http://www.chinanews.com/gn/2011/11-27/3488922.shtml |title = 赵勇辞去河北省副省长职务 聂辰席任河北副省长 |publisher=[[中新网|中新网]]}}</ref>。

2012年10月，任[[国家广播电影电视总局|国家广播电影电视总局]]党组副书记、副局长<ref>{{cite web |url = http://www.chinanews.com/gn/2012/11-01/4293853.shtml |title = 聂辰席任国家广电总局党组副书记、副局长 |accessdate = 2012-11-01 |publisher = 中新网}}</ref>。2013年3月[[国务院机构改革|国务院机构改革]]后，改任新设立的[[中华人民共和国国家新闻出版广电总局|中华人民共和国国家新闻出版广电总局]]党组副书记、副局长。2015年4月起，兼任[[中国中央电视台|中央电视台]]台长<ref>[http://ent.163.com/15/0407/19/AMKDRE6H00031H2L.html 官方公布聂辰席接替胡占凡任央视台长]</ref>。2016年9月，接替[[蔡赴朝|蔡赴朝]]任[[中共中央宣传部|中共中央宣传部]]副部长、国家新闻出版广电总局局长、党组书记、[[国家版权局|国家版权局]]局长<ref>{{cite web |accessdate = 2016-09-29 |url = http://finance.ifeng.com/a/20160929/14913793_0.shtml |title = 聂辰席任中宣部副部长 国家新闻出版广电总局局长 |publisher=中国经济网}}</ref>。2018年2月，卸任[[中国中央电视台|中国中央电视台]]台长。2018年3月21日, [[国家广播电视总局|国家广播电视总局]]召开干部大会，宣布了此前中共中央关于组建国家广播电视总局和领导班子任职的决定，聂辰席任中共中央宣传部副部长、国家广播电视总局局长、党组书记<ref name=elyb>{{Cite web |url =http://news.ifeng.com/a/20180321/56922969_0.shtml  |title =聂辰席任国家广播电视总局局长  |author =  |date =  2018-03-21|publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
