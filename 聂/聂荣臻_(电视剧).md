《'''聂荣臻'''》，[[中国大陆|中国大陆]]电视剧，主演[[林永健|林永健]]、[[齐欢|齐欢]]、[[王伍福|王伍福]]、[[刘劲|刘劲]]、[[卢奇|卢奇]]。2013年1月杀青，该剧根据聂荣臻女儿[[聂力|聂力]]《山高水长：我的父亲聂荣臻》改编而成。<ref name="fish ">{{cite web|url=http://sn.people.com.cn/n/2012/1217/c339261-17877321.html|title=林永健再次出演元帅 电视剧《聂荣臻》开机|accessdate=2012年12月17日|publisher=人民网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20131213211348/http://sn.people.com.cn/n/2012/1217/c339261-17877321.html|archivedate=2013年12月13日}}</ref>2013年12月11日在[[央视一套|央视一套]]黄金时间播出<ref name="new " >{{cite web |url= http://ent.sina.com.cn/v/m/2013-12-05/20254055573.shtml|title=林永健出演聂荣臻 曾被质疑能行吗 |accessdate=2013年12月5日 |publisher= 新浪娱乐|work= |language =zh-cn}}</ref>。

== 花絮 ==
该剧的制作方是[[西部电影集团有限公司|西部电影集团有限公司]]、[[重庆市|重庆市]][[江津区|江津区]]委、[[华夏电影发行有限责任公司|华夏电影发行有限责任公司]]、北京中视寰影文化传媒中心。<ref name=" cat" />

该剧投资4500万，历经130多天的拍摄天<ref name="new " />，並曾在[[山西省|山西省]]、[[河北省|河北省]]、[[北京市|北京市]]、[[天津市|天津市]]、[[上海市|上海市]]等地取景。<ref name=" cat" />

林永健首度出演聂荣臻是在电影《[[钱学森_(电影)|钱学森]]》。<ref name="fish " />

女演员齐欢出演聂荣臻妻子张瑞华。<ref name=" nna" >{{cite web |url=http://ent.sina.com.cn/v/m/2012-12-29/15153823881.shtml |title= 《聂荣臻》上海热拍 齐欢致敬革命伉俪|accessdate= 2012年12月29日|publisher=新浪网 |work= |language =zh-cn}}</ref>

男演员墨阳加盟出演该剧。<ref>{{cite web |url=http://ent.qq.com/a/20121215/000046.htm |title=林永健再度演绎“小眼版”聂荣臻 与墨阳演父子 |accessdate=2012年12月25日 |publisher= 腾讯网|work= |language =zh-cn}}</ref>
