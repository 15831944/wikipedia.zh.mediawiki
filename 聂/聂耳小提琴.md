'''聂耳小提琴'''，中国[[云南省博物馆|云南省博物馆]]的一件藏品，[[国家一级文物|国家一级文物]]<ref name="贵阳网" />，中国近代音乐家[[聂耳|聂耳]]曾经使用的[[小提琴|小提琴]]，是他创作后来被用作[[中华人民共和国国歌|中华人民共和国国歌]]的《义勇军进行曲》时所用的乐器。

== 历史背景 ==
被誉为“乐器皇后”的[[小提琴|小提琴]]是17世纪西方最为重要的乐器之一，从1920年开始传入中国。聂耳出生在[[云南|云南]][[昆明|昆明]]，从小热爱音乐，精通[[笛子|笛子]]、[[二胡|二胡]]、[[三弦|三弦]]、[[月琴|月琴]]等多种乐器。1930年初，聂耳离开昆明，前往大都市[[上海|上海]]，在打工的同时从未放弃音乐。1931年初，聂耳帮助朋友代租电影片邮寄回昆明，获得100元酬金，汇给母亲一半后，用剩余的钱买了一把廉价的二手小提琴、几本[[乐谱|乐谱]]和一件冬衣<ref name="中国网">{{cite web |title=云南省博物馆的这把旧琴是当之无愧的“国宝” |url=https://culture.china.com/history2014/11159882/20180109/31925267.html |author=王姗 |website=中国网-新华网 |accessdate=2019-01-12 |date=2018-01-09 |archive-url=https://web.archive.org/web/20180109032551/http://culture.china.com/history2014/11159882/20180109/31925267.html |archive-date=2018-01-09 |dead-url=no }}</ref>。

1932年，聂耳凭借自学小提琴演奏，考入[[联华影业公司|联华影业公司]]歌舞学校。数月后，他成为[[明月歌剧社|明月歌剧社]]的乐队小提琴师，并开始着手创作革命音乐、歌舞。1933年8月，聂耳用小提琴为电影《[[母性之光|母性之光]]》创作了插曲《开矿歌》。之后，聂耳又通过小提琴创作出《码头工人》《[[卖报歌|卖报歌]]》《大路歌》等革命歌曲。1935年，聂耳为电影《[[风云儿女|风云儿女]]》创作主题曲《[[义勇军进行曲|义勇军进行曲]]》，不久在[[日本|日本]][[神奈川县|神奈川县]][[藤泽市|藤泽市]]溺水身亡<ref name="中国网" /><ref name="张蕊">{{cite web |author1=张蕊 |title=云南省三件国宝入选《国家宝藏》第二季 来看看它们稀奇在哪儿 |url=http://yn.yunnan.cn/system/2018/12/03/030131532.shtml |website=云南网 |accessdate=2019-01-07 |date=2018-12-03}}</ref><ref>{{cite news |author1=王健 |title=聂耳遇难地日本藤泽市鹄沼海岸举行聂耳纪念活动(图) |url=http://www.chinanews.com/gj/2015/07-11/7399220.shtml |accessdate=2019-01-12 |publisher=中国新闻网 |date=2015-07-11 |archive-url=https://web.archive.org/web/20150729213600/http://www.chinanews.com/gj/2015/07-11/7399220.shtml |archive-date=2015-07-29 |dead-url=no }}</ref>。

后来，《义勇军进行曲》在1949年被选为中华人民共和国代国歌，1982年被确定为中华人民共和国国歌<ref name="中国网" />。

== 收藏展览 ==
1950年代，聂耳的亲友将近5000件聂耳遗物捐赠给云南省博物馆，其中包括聂耳的个人用品、作品手稿、文章文稿、小说、剧本、日记、书信等，该小提琴也在其中。自2016年10月1日起，云南省博物馆将聂耳小提琴放置在三楼“百年风云”展厅{{efn|“百年风云”展览全名“风云百年——近现代时期的云南”<ref>{{cite web |title=《风云百年——近现代时期的云南》 |url=http://www.ynmuseum.org/detail/641.html |website=云南省博物馆官方网站 |accessdate=2019-01-12 |date=2015-06-21}}</ref>}}的一个独立展柜中展出<ref name="中国网" /><ref name="贵阳网">{{cite web |author1=王姗 |title=聂耳小提琴10月1日起将首次公开在省博长期展出 |url=http://www.gywb.cn/content/2016-10/01/content_5303321.htm |website=贵阳网-昆明日报 |accessdate=2019-01-12 |date=2016-10-01}}</ref>。
