'''苗栗鐵道文物展示館'''，位於[[臺灣|臺灣]][[苗栗縣|苗栗縣]][[苗栗市|苗栗市]]，常被稱為苗栗鐵道博物館、苗栗鐵道公園等。本館座落於[[苗栗車站|苗栗車站]]旁，是利用苗栗機務分駐所之空地建成，目前由[[新竹機務段|新竹機務段]][[苗栗機務分駐所|苗栗機務分駐所]]暫管<ref name="苗栗市觀光旅遊資訊">[http://www.mlcg.gov.tw/mlcg_travelnew/tour/index-1.php?m2=10&id=3 苗栗市觀光旅遊資訊-鐵道文物館]</ref>，並隸屬於[[中華民國交通部|中華民國交通部]][[臺灣鐵路管理局|臺灣鐵路管理局]]。是一座專門保存臺灣鐵道文物與車輛的展示館，除了臺鐵所使用之車輛外，臺鐵亦行文至臺糖與[[行政院農業委員會林務局|林務局]]提供停用車輛至本館展示。目前展示區內（遮雨棚內）可自由並免費參觀，展示區以外如投煤練習場則需向臺鐵申請參觀<ref name="苗栗市觀光旅遊資訊"/>。目前，因苗栗縣政府辦理「苗栗火車站東西站及周邊環境再造工程」而自2019年4月1日閉館至2020年12月底<ref>[https://www.railway.gov.tw/tw/news_Detail.aspx?SN=16446&NewsType=0&n=21560 「苗栗火車站東西站及周邊環境再造工程」暫停火車頭展示空間及投煤練習場參觀導覽]</ref>。

== 歷史 ==
[[File:Taiwan_MiaoliCity_MiaoliRailwayMuseum2.jpg|thumb]]
1999年6月10日，全台首座鐵路展示館舉行揭幕典禮<ref name="中原大學研究報告">[https://www.boch.gov.tw/information_192_49303.html 《鐵路局臺北機廠歷史與其鐵道產業文化資產之基礎研究》，羅世松，第221與224頁]</ref>。

2003年4月，完成'''苗栗鐵路機車展示館'''與興建展示區遮雨棚等設施，並移交苗栗機務分駐所負責管理<ref name="中原大學研究報告"/>。現名為'''苗栗鐵道文物展示館'''。

2014年，台鐵局計畫委託民間經營活化苗栗鐵道文物展示館，公開徵求「苗栗縣苗栗火車站鐵道博物館興建營運移轉案」民間投資者<ref>[http://www.motc.gov.tw/ch/home.jsp?id=14&parentpath=0,2&mcustomize=news_view.jsp&dataserno=201506120006&toolsflag=Y 苗栗火車站鐵道博物館計畫啟動了！]</ref>。

== 館藏文物 ==
本館除下列者外，還曾保存LDR2201柴油客車，後因2011年的[[臺鐵百年文物特展|火車載阮向前行─台鐵百年文物特展]]而移置[[臺北車站|臺北車站]]東側展示<ref>[http://news.ltn.com.tw/news/life/paper/533040 蒸汽火車頭 台北街頭趴趴走 [[自由時報|自由時報]]]</ref>。

=== 鐵路機車 ===
==== [[蒸汽機車|蒸汽機車]] ====
*CT152：1979年6月2日報廢<ref>《台灣鐵道印象》，洪致文 著，南天書局1998年出版（ISBN 957-638-462-1）</ref>，為日本[[汽車製造|汽車製造]]1919年生產的[[台鐵CT150型蒸汽機車|CT150]]型客貨兩用機車。
*DT561：為[[台鐵DT560型蒸汽機車|DT560]]型，美國[[美國機車公司|美國機車公司]]於1920年生產，1979年停用，為貨運用機車。
*28：[[阿里山森林鐵路|阿里山森林鐵路]]的28噸級28號蒸汽機車，由美國[[美國利馬機車廠|利馬機車廠]]於1914年8月26日生產，1969年6月19日停駛<ref>{{Cite web |url=http://citycat.theweb.tw/alishanslch.htm |title=街貓的鐵道網站-阿里山森林鐵路機車表 |access-date=2016-03-17 |archive-url=https://web.archive.org/web/20160209090543/http://citycat.theweb.tw/alishanslch.htm |archive-date=2016-02-09 |dead-url=yes }}</ref>，為阿里山本線之主力機車。
*331：[[臺灣糖業公司|臺糖公司]]虎尾總廠的331號蒸汽機車，1935年由[[日本車輛製造|日本車輛製造]]，1999年1月由台糖虎尾總廠提供<ref>{{Cite web |url=http://citycat.theweb.tw/sgsltw.htm |title=街貓的鐵道網站-臺糖鐵路蒸氣機車分布地點 |access-date=2016-03-17 |archive-url=https://web.archive.org/web/20160324225630/http://citycat.theweb.tw/sgsltw.htm |archive-date=2016-03-24 |dead-url=yes }}</ref>。

==== [[柴電機車|柴電機車]] ====
*R6：屬於[[台鐵R0型柴電機車|R0]]型機車，由日本[[日立|日立公司]]於1960年生產，是臺鐵動力柴油化後第一批購入之機車，因車況不佳曾於1973年改造，最後於1996年停用，僅餘此輛保存。
*S305：為[[台鐵S300型柴電機車|S300]]型，是臺鐵支線與調車用機車，美國[[易安迪|GM-EMD]]於1966年生產，S305在1999年停用報廢後，經臺鐵[[花蓮機廠|花蓮機廠]]於2011年底整修完成後加入本館展示行列<ref>[http://www.railway.gov.tw/Upload/admin/20140106_01.pdf 《鐵路局花蓮機廠廠務史蹟》]</ref>。
*S405：為[[台鐵S400型柴電機車|S400]]型，是臺鐵[[七堵調車場|七堵駝峰式調車場]]專用之調車機，美國GM-EMD於1969年生產，1997年停用。

==== [[柴液機車|柴液機車]] ====
*[[台鐵LDH100型柴液機車|LDH101]]：為臺鐵[[花蓮機廠|花蓮修理廠]]於1970年以動力客車之轉向架<ref>[http://cwhung.blogspot.tw/2017/04/ldh10119701938.html 洪致文 飛行場の測候所-LDH101的身世考證調查（1970的車身，穿著1938的鞋子）]</ref>、引擎等現有零件組裝而成。曾與LDK58、LDR2201、LTPB1813運至澎湖展示<ref>[http://document.hccc.gov.tw/railway/Photoview.php?sn=84 後山鐵道風華文化資產數位博物館]</ref><ref>[http://document.hccc.gov.tw/railway/Photoview.php?sn=85 後山鐵道風華文化資產數位博物館]</ref>，後來因車輛嚴重鏽蝕<ref>[http://document.hccc.gov.tw/railway/Photoview.php?sn=86# 後山鐵道風華文化資產數位博物館]</ref>而運回臺灣修復，並將LDH101、LTPB1813保存於本館。

==== [[柴油機車#機械傳動|柴油機車（機械傳動）]] ====
*[[阿里山25噸級柴油機車#日本三菱11403-1、11403-2（離合器機械傳動）|11403-1]]：阿里山森林鐵路首代柴油機車，1953年由日本[[三菱重工|新三菱重工業]]所生產，用來取代蒸汽機車。但因故障太多，車輛設計也不適合於阿里山森林鐵路行駛，故即使修改設計後也僅能行駛於[[北門車站_(嘉義)|北門]]與[[竹崎車站|竹崎]]間路線標準較佳之路段。本車編號源自公家機關財產編號，並與蒸汽機車同為全黑塗裝，移至本館後才改為橘色塗裝<ref name="世紀風華：阿里山森林鐵路百年車輛史"/>。
*[[阿里山25噸級柴油機車#三菱之11403-3、11403-5（離合器機械傳動）|11403-5]]：阿里山森林鐵路第二代柴油機車，1955年由日本新三菱重工業所生產。此型機車因性能不佳，甚至比不上28噸級蒸汽機車，而未獲重用。不過卻樹立後來的阿里山機車皆以此車為設計概念。編號同樣源自財產編號，並也為全黑塗裝，移至本館後才更改塗裝<ref name="世紀風華：阿里山森林鐵路百年車輛史"/>。

=== 鐵路客車 ===
*25TPK2053：原臺鐵25ES2053鋼樑廠宿營車，此車原已報廢，後來被選為保存車，於1999年由臺鐵[[高雄機廠|高雄機廠]]修復完成並恢復車籍為25TPK2053<ref>[http://cwhung.blogspot.tw/2014/05/tpk2053.html 洪致文 飛行場の測候所-台鐵木造客車TPK2053身世考證]</ref>。目前不定期會迴送至臺鐵高雄機廠進行檢修。
*30SPK2502：1953年10月臺鐵[[台北機廠|台北機廠]]製。原臺鐵30ES2502鋼樑廠宿營車，同為報廢車，但後來被選為保存車之車輛，於1999年由臺鐵高雄機廠修復並恢復車籍為30SPK2502。目前不定期會迴送至臺鐵高雄機廠進行檢修。
*LTPB1813：為1970年日本[[東急車輛製造|東急車輛製造]]臺鐵[[台東線|台東線]]762mm軌距[[舊東線光華號|光華號]]柴油客車之無動力拖車。1982年停用，1985年9月送至澎湖縣立文化中心展示。後因嚴重鏽蝕而於1999年10月運回臺鐵[[台北機廠|台北機廠]]修復，2003年運至本館保存<ref>[http://document.hccc.gov.tw/railway/Photoview.php?sn=71 後山鐵道風華文化資產數位博物館]</ref>。
*SPC2：1971年4月由[[日本車輛製造|日本車輛製造]]，曾為阿里山森林鐵路的光復號對號快車，後因阿里山公路通車之影響而停駛。1999年本車移至本館保存，原塗裝為紅底白線，移至本館後改為橘底白線<ref name="世紀風華：阿里山森林鐵路百年車輛史"/>。

== 歷史建築 ==
[[File:苗栗電力機車庫.png|thumb]]
'''苗栗機關庫'''：由於苗栗位處山線坡度大，致使列車無力爬坡，需要[[輔助機車|輔助機車]]支援，{{fact|故[[台灣總督府交通局鐵道部|台灣總督府交通局鐵道部]]於苗栗車站設置苗栗機關庫，作為輔助機車停放與維修用|time=2016-03-18}}，後來苗栗機關庫於[[1935年新竹－台中地震|1935年新竹－台中地震]]中受損<ref>[http://lib.mlc.gov.tw/webmlh/wd_Content640.asp?SID=1128 苗栗縣政府國際文化觀光局-苗栗老照片]</ref><ref>[http://lib.mlc.gov.tw/webmlh/wd_Content640.asp?SID=1127 苗栗縣政府國際文化觀光局-苗栗老照片]</ref>而不堪使用，並決定新建機關庫，1937年4月24日新式苗栗機關庫落成<ref>[http://lib.mlc.gov.tw/webmlh/wd_Content640.asp?SID=1132 苗栗縣政府國際文化觀光局-苗栗老照片]</ref>，此機關庫成為當時苗栗最大且最新之建築，甚至舉辦落成慶祝會<ref>[http://lib.mlc.gov.tw/webmlh/wd_Content640.asp?SID=1134 苗栗縣政府國際文化觀光局-苗栗老照片]</ref>。此新式機關庫是基於未來鐵路電氣化後，供[[電力機車|電力機車]]使用而設計，所以外觀不同於當時的其他車庫，但是最後從未有電力機車進駐這座機關庫，而是成為臺鐵工程車輛停放、維修之工廠使用至今，目前此車庫仍為工廠重地，故不開放參觀。
