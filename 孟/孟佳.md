'''孟佳'''（{{kor|k='''맹가'''|rm='''Maeng Ga'''}}，{{lang-en|'''Meng Jia'''}}；{{bd|1990年|2月3日|catIdx=孟}}），藝名'''Jia'''（{{kor|k='''지아'''|rm='''Ji Ah'''}}，{{lang-zh|'''佳'''}}），在[[韓國|韓國]]出道的女[[歌手|歌手]]、[[舞者|舞者]]。2007年透過甄選會加入[[JYP娛樂|JYP娛樂]]，2010年7月1日以四人女子團體[[miss_A|miss A]]在韓國出道，擔任主唱及饒舌演唱，曾與[[王霏霏|Fei]]同為[[miss_A|miss A]]的中國籍成員。2016年5月20日選擇不繼續和JYP娛樂續約，退出miss A。<ref>{{cite news|url=http://www.allthatstar.com/tw/view.php?action=5207&postID=243401&t=Jia%E5%90%88%E7%B4%84%E5%88%B0%E6%9C%9F%E9%80%80%E5%9C%98+miss+A%E6%9C%AA%E4%BE%86%E5%B0%87%E4%BB%A5%E6%88%90%E5%93%A1%E5%80%8B%E4%BA%BA%E6%B4%BB%E5%8B%95%E7%82%BA%E4%B8%BB&rtn_sns=fa|title=Jia合約到期退團 miss A未來將以成員個人活動為主|publisher=全星網|date=2016-05-20|language=zh-tw|access-date=2016-05-20|archive-url=https://web.archive.org/web/20160630083322/http://www.allthatstar.com/tw/view.php?action=5207&postID=243401&t=Jia%E5%90%88%E7%B4%84%E5%88%B0%E6%9C%9F%E9%80%80%E5%9C%98+miss+A%E6%9C%AA%E4%BE%86%E5%B0%87%E4%BB%A5%E6%88%90%E5%93%A1%E5%80%8B%E4%BA%BA%E6%B4%BB%E5%8B%95%E7%82%BA%E4%B8%BB&rtn_sns=fa|archive-date=2016-06-30|dead-url=yes}}</ref>同年6月8日，簽約全球華人首富[[王健林|王健林]]之子[[王思聰|王思聰]]旗下公司[[香蕉計劃|香蕉計劃]]，成為該公司旗下藝人，正式回國發展，並發行首張個人專輯。<ref>{{cite news|language=zh|url=http://kpopdata.com/news/48381-%E5%AD%9F%E4%BD%B3%E7%B0%BD%E7%B4%84%E4%B8%AD%E5%9C%8B%E9%A6%99%E8%95%89%E8%A8%88%E5%8A%83%E5%A8%9B%E6%A8%82+%E6%AD%A3%E5%BC%8F%E5%B1%95%E9%96%8B%E5%80%8B%E4%BA%BA%E6%B4%BB%E5%8B%95|title=孟佳簽約中國香蕉計劃娛樂 正式展開個人活動}|date=2016年6月9日|publisher=kpopdata.com}}</ref>

== 生涯 ==
=== 1999年－2009年：出道前 ===
2006年年底就讀北京現代音樂學院時，某次進到教室後才發現裡面正在進行[[JYP娛樂|JYP娛樂]]的甄選會，在一位女性工作人員的勸說下參加，以舞蹈和演唱[[潘瑋柏|潘瑋柏]]的〈Tell Me〉成功入選。{{NoteTag|name=explore|1=2013年11月15日在[[MBC|MBC]]《[[偶像本色|C-RADIO 偶像本色]]》中描述自己被發掘過程。}}2007年3月初到廣州正式和[[JYP娛樂|JYP娛樂]]簽約，2007年3月30日前往韓國開始練習生生涯。<ref>{{cite web| language = zh| url = http://news.bjcma.com/xydt/4369.html| title = 成為韓國天王巨星Rain同門師妹——我院學生孟佳簽約韓國「JYP娛樂」受訓| publisher = 北京現代音樂研修學院| date = 2007-03-29| access-date = 2014-07-08| archive-url = https://web.archive.org/web/20140714121644/http://news.bjcma.com/xydt/4369.html| archive-date = 2014-07-14| dead-url = yes}}</ref>

2009年曾與Sarha、[[王霏霏|Fei]]、黃以霏和[[禹惠林|惠林]]組成5人女子團體「Sisters」<ref>{{cite news|language = zh | url = http://cn.chosun.com/big5/site/data/html_dir/2009/06/01/20090601000018.html|title = 中國版Wonder Girls“Sisters”亮相| publisher = 朝鮮日報中文網| date = 2009-06-01}}</ref>，出演中國[[浙江衛視|浙江衛視]]《越跳越美麗》等節目，最後並未正式出道。2010年6月和[[王霏霏|Fei]]、[[裴秀智|Suzy]]組成「miss A」，尚未出道就成為三星Anycall中國地區代言人，並演唱廣告曲〈Love Again〉。<ref>{{cite news|language = zh|url = http://ent.sina.com.hk/news/-3-465231/1.html|title = JYPE打造亞洲女子天團 miss A未出道先接代言|publisher = 新浪香港|date = 2010-06-07|deadurl = yes|archiveurl = https://archive.is/20140617111108/http://ent.sina.com.hk/news/-3-465231/1.html|archivedate = 2014-06-17}}</ref>6月23日JYP娛樂發佈miss A最終由4名成員組成，除了原有的成員還有1名成員[[李玟暎|Min]]<ref>{{cite news|language = ko| url = http://www.hankyung.com/news/app/newsview.php?aid=2010062398417|title =원더걸스-2PM 'JYP' 신인 걸그룹 'miss A', 7월 초 전격 데뷔!| publisher = 한경닷컴| date = 2010-06-23}}</ref>，同天於Tory Burch開幕儀式上首次以團體形式正式亮相<ref>{{cite news|language = ko| url = http://osen.mt.co.kr/article/G1006240248|title = 'JYP 신인 걸그룹' 미쓰에이, 23일 행사장서 첫 공식 인사| publisher = OSEN| date = 2010-06-24}}</ref>。

=== 2016年－現在：個人活動 ===
2013年7月2日Jia公開自己為公益活動而設計的T恤，首次擔任設計師。<ref>{{cite news|language = zh-tw|url=http://kpopn.com/2013/07/02/183084/|title=寶拉穿Jia 贈送的上衣|date =2013-07-02|publisher =kpopn.com}}</ref><ref>{{cite news|language=zh-tw|url=http://kpopn.com/2013/08/05/187223/|title=Nichkhun：佳的設計很棒！ |date=2013-08-05|publisher=kpopn.com}}</ref>7月10日[[JYP娛樂|JYP娛樂]]透露Jia參與《一又二分之一的夏天》的演出，為個人戲劇處女作<ref>{{cite news|language=zh|url=http://www.bntnews.cn/app/news.php?nid=12174|title = 尼坤孟佳攜手出演內地劇《一又二分之一的夏天》|date =2013-07-10|publisher=bntnews}}</ref>，在劇中飾演一名會計系大三學生——宋青，為了脫離貧窮的環境而專注於學習的書-{呆}-子。

2014年3月15日[[MBC|MBC]]《[[偶像本色|C-RADIO 偶像本色]]》正式播出，首次主持廣播節目，與[[王霏霏|Fei]]、[[周覓|周覓]]一起擔任電台節目主持人<ref>{{cite news|language=zh|url=http://hk.kpopstarz.com/articles/2438/20140315/miss-jia-fei%E5%92%8Csuper-junior-m%E5%91%A8%E8%A6%93%E6%88%90%E7%82%BAmbc%E6%96%B0%E7%AF%80%E7%9B%AE-%E5%81%B6%E5%83%8F%E6%9C%AC%E8%89%B2-%E4%B8%BB%E6%8C%81.htm|title=miss A Jia，Fei和Super Junior-M周覓成為MBC新節目《偶像本色》主持|date=2014-03-15|publisher=KpopStarz|deadurl=yes|archiveurl=https://web.archive.org/web/20140714155746/http://hk.kpopstarz.com/articles/2438/20140315/miss-jia-fei%E5%92%8Csuper-junior-m%E5%91%A8%E8%A6%93%E6%88%90%E7%82%BAmbc%E6%96%B0%E7%AF%80%E7%9B%AE-%E5%81%B6%E5%83%8F%E6%9C%AC%E8%89%B2-%E4%B8%BB%E6%8C%81.htm|archivedate=2014-07-14}}</ref>，每週會訪問不同的韓國偶像明星。7月30日第一次出演電影《第三種愛情》。<ref>{{cite news|language=zh|url=http://www.4hw.com.cn/new/yule/62193.html|title=miss A佳確定出演電影《第三種愛情》與宋承憲劉亦菲首度合作|date=2013-07-30|publisher=kpopn.com|access-date=2014-07-30|archive-url=https://web.archive.org/web/20140806061147/http://www.4hw.com.cn/new/yule/62193.html|archive-date=2014-08-06|dead-url=yes}}</ref>

2016年5月20日，與經紀公司[[JYP娛樂|JYP娛樂]]合約期滿，正式離開公司及組合[[miss_A|miss A]]。同年6月8日，簽約全球華人首富[[王健林|王健林]]之子[[王思聰|王思聰]]旗下公司[[香蕉計劃娛樂|香蕉計劃娛樂]]，成為該公司旗下藝人，正式回國作個人發展，準備發行個人專輯。<ref>{{cite news|language=zh|url=http://kpopdata.com/news/48381-%E5%AD%9F%E4%BD%B3%E7%B0%BD%E7%B4%84%E4%B8%AD%E5%9C%8B%E9%A6%99%E8%95%89%E8%A8%88%E5%8A%83%E5%A8%9B%E6%A8%82+%E6%AD%A3%E5%BC%8F%E5%B1%95%E9%96%8B%E5%80%8B%E4%BA%BA%E6%B4%BB%E5%8B%95|title=孟佳簽約中國香蕉計劃娛樂 正式展開個人活動}|date=2016年6月9日|publisher=kpopdata.com}}</ref>

2016年11月10日，於instagram宣布於11月15日以個人形式回歸樂壇。11月15日公開首張單曲《給我乖》音源，翌日則公開該單曲的MV。

2017年1月3日，公開《Who's That Girl》的MV；4月7日，公開《Candy》的MV；12月27日，公開與[[王嘉爾|王嘉爾]]合作單曲《MOOD》的MV。
