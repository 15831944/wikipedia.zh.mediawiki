'''孟加影縣'''（[[印尼語|印尼語]]：Kabupaten Bengkayang）是[[印度尼西亞|印度尼西亞]][[西加里曼丹省|西加里曼丹省]]的一個[[縣|縣]]。

== 地理 ==
孟加影縣位於西加里曼丹省北部，東接[[上侯縣|上侯縣]] ，與[[馬來西亞|馬來西亞]][[砂拉越|砂拉越州]]接壤，西臨[[納土納海|納土納海]]，南與[[喃吧哇縣|喃吧哇縣]]，[[蘭達克縣|蘭達克縣]]相連，北與[[山口洋|山口洋市]]，[[三發縣|三發縣]]毗鄰。

== 人口 ==
在山口洋市與孟加影縣之間有許多[[華人|華人]]與[[達雅族|達雅族]]通婚的後代居住。<ref>http://www.umsl.edu/~naumannj/professional%20geography%20articles/A%20political%20ecology%20of%20violence%20and%20territory%20in%20West%20Kalimantan.pdf</ref>2006年孟加影人口為211,883人。<ref>{{cite web |url=http://kalbar.bps.go.id/Bengkayang/file/product/kda/ulasankda/ulasanpendudukeng.htm |title=存档副本 |accessdate=2011-06-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20110819061533/http://kalbar.bps.go.id/Bengkayang/file/product/kda/ulasankda/ulasanpendudukeng.htm |archivedate=2011-08-19 }}</ref>孟加影縣當地華人使用[[客家語|客家語]]。<ref>http://www.guojiribao.com/shtml/gjrb/20120815/76802.shtml</ref>

== 經濟 ==
[[農業|農業]]為孟加影縣的主要經濟來源。<ref>{{cite web |url=http://kalbar.bps.go.id/Bengkayang/file/product/kda/ulasankda/ulasanpendapataneng.htm |title=存档副本 |accessdate=2011-06-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20110819061408/http://kalbar.bps.go.id/Bengkayang/file/product/kda/ulasankda/ulasanpendapataneng.htm |archivedate=2011-08-19 }}</ref>[[糧食|糧食]]作物有[[稻米|稻米]]、[[玉米|玉米]]、[[木薯|木薯]]、[[番薯|番薯]]、[[花生|花生]]、[[黃豆|黃豆]]、[[豌豆|豌豆]]等等。[[蔬菜|蔬菜]]類作物有[[黃瓜|黃瓜]]、[[豇豆|豇豆]]等等。經濟作物有[[油棕|油棕]]、[[椰子|椰子]]、[[咖啡|咖啡]]、[[可可|可可]]、[[橡膠|橡膠]]、[[胡椒|胡椒]]等等。<ref>http://kalbar.bps.go.id/Bengkayang/file/product/kda/ulasankda/ulasanpertanianeng.htm{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
