'''孟果大其力戰役'''又稱'''猛果大其力戰役'''<ref>{{Cite web |url = http://www.yunnan.tw/index.php/literature/list-4/yunnanliterature33/413-article3308.html |title = 泰緬地區華人孤軍奮戰史績記要 |accessdate = 2018/07/29 |archive-url = https://web.archive.org/web/20180729112042/http://www.yunnan.tw/index.php/literature/list-4/yunnanliterature33/413-article3308.html |archive-date = 2018-07-29 |dead-url = yes }}</ref>，是1950年[[緬甸聯邦_(1948年－1962年)|緬甸聯邦]]發動對緬甸境內，孟果、大其力一帶的[[中華民國國軍|中華民國國軍]]（[[泰緬孤軍|泰緬孤軍]]）的軍事行動，結果國軍擊潰緬軍，使孤軍議題引起各國重視。

==背景==
{{see also|雲南起義}}
1949年12月[[雲南省主席|雲南省主席]][[盧漢|盧漢]]變節投共，並拘禁[[第八軍|第8軍]][[軍長|軍長]][[李彌|李彌]]與[[第二十六軍|第26軍]]軍長[[余程萬|余程萬]]，脅迫兩人一起投共。第26軍和第8軍得知兩人被拘禁後，圍攻昆明，倒戈部隊（[[第七十四軍|第74軍]]、[[第九十三軍|第93軍]]等）不敵，先後釋放李彌和余程萬。

余程萬獲釋後，率領第26軍撤退至[[法屬印度支那|法屬越南]]。李彌也率領第8軍往[[滇西|滇西]]撤退。第8軍在中緬邊境的[[元江|元江]]被[[解放軍|解放軍]]追上，經過激戰，李彌率領的6萬大軍幾乎被解放軍殲滅，隨後李彌在[[軍統局|軍統局]][[特務|特務]]的安排下來到[[泰國|泰國]][[曼谷|曼谷]]，再從曼谷飛往[[臺灣|臺灣]]。

第8軍237師709團的[[少將|少將]][[團長_(軍隊)|團長]][[李國輝_(中華民國)|李國輝]]率領600多名殘軍從[[雲南|雲南]][[西盟|西盟]][[佤山|佤山]]進入[[緬甸|緬甸]]，在緬境[[孟棒|孟棒]]與第26軍93師278團譚忠部800多人會合<ref>{{Cite web |url = http://www.cares.org.tw/s_2440_detail.asp?sn=502 |title = 中華救助總會 |accessdate = 2019/07/26 |quote = 一九五○年二月，李國輝部隊的六百多人退至緬甸境內，在孟棒與第二十六軍九十三師第二七八團的譚忠部隊約八百多人會合，共約一千五百人，這就是孤軍最早的起源 }}</ref>（278團和第26軍主力分道揚鑣，由副團長譚忠率領自行退入緬甸），這些退入緬甸的國軍部隊被稱為[[泰緬孤軍|孤軍]]。<ref name="泰北孤軍—他們的故事"/>
==戰役經過==
{{see also|泰緬孤軍#打敗緬甸國防軍}}
1950年2月28日孤軍來到緬甸境內，緬北重鎮[[大其力|大其力]]附近的小城鎮[[孟棒|孟棒]]（MongPong，又譯為[[猛捧|猛捧]]）、[[孟果|孟果]]就地休養，並招募新血，兩個月內將部隊擴充到3,500人，部隊改編定名為「復興部隊」。

緬甸政府將孤軍視為威脅，儘管孤軍向緬甸政府表明即將伺機[[反攻大陸|反攻大陸]]，無侵佔緬甸領土野心，緬方仍要求孤軍於4月30日前撤出緬甸，李國輝不予理會。在孟果、大其力附近的[[景棟|景棟]]，駐防在景棟的兩個緬軍軍團蠢蠢欲動，其中一支軍團已經進入大其力。緬軍認為孤軍的主要協助者是[[緬甸華僑|當地華僑]]，於是數百名[[華僑|華僑]]被誣陷擁有武器，遭到緬軍逮捕。<ref>{{cite book|author=覃怡輝|title=金三角國軍血淚史：1950-1981|url=https://books.google.com/books?id=1VHBBwAAQBAJ&pg=PA63|date=2009-08-28|publisher=聯經出版|isbn=978-986-01-9491-3|pages=93}}</ref>三方在6月3日舉行談判，結果緬方拘禁孤軍談判代表[[丁作韶|丁作韶]]、大其力華僑代表[[馬鼎臣|馬鼎臣]]，於6月8日向李國輝發出最後通牒，要去孤軍立即撤出緬甸，否則將使用武力，並拘禁景棟的1,700名華裔居民，以免他們協助孤軍。李國輝要求釋放孤軍談判代表及華裔居民，並請求緬軍不要進入大其力，緬方斷然拒絕。

6月16日緬軍6架轟炸機空襲孟果一帶孤軍陣地，孤軍死傷慘重，李國輝下令棄守陣地，退入原始叢林，緬軍仍不罷休，出動10,000人的搜山部隊進入叢林，企圖一舉消滅孤軍，再出動10,000人的部隊封鎖叢林出口。孤軍在叢林中與緬軍周旋，使緬軍疲憊。

6月28日，孤軍轉守為攻，偷襲緬軍炮兵陣地，繳獲10尊緬軍大炮，以繳獲而來的大炮還擊，緬軍紛紛敗退，孤軍乘勝追擊，奪回孟棒、孟果，並攻陷大其力，並擊落緬軍空軍總司令之飛機，震驚中外，緬方不得不與孤軍和談，孤軍於8月23日撤出大其力，而轉入山區活動，以平息爭端。<ref name="泰北孤軍—他們的故事"/><ref name="金三角，蔣殘軍何以能留下6萬後代？"/>
