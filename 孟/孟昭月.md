'''孟昭月'''（{{bd|1887年||1967年|}}）<ref>生年1887年、卒年1967年均见《台前县志》</ref>[[字|字]]'''子明'''，[[山东|山东]][[寿张|寿张]]北孟村人，[[中華民国|中華民国]]军事将领。<ref name=lai>{{Cite book<!--|和書-->|author = 来新夏等|title = 北洋軍閥史 下冊|year = 2000|publisher =  [[南開大学|南開大学]]出版社|isbn = 7-310-01517-7}}</ref><ref name=xu>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref><ref name=tai>台前县志，中州古籍出版社，2001年，第710页</ref>

==生平==
孟昭月幼年丧父，家境贫寒，以卖豆腐为生。24岁时，孟昭月从军。[[辛亥革命|辛亥革命]]时，孟昭月任工兵营长。后来，孟昭月在[[孫传芳|孫传芳]]手下任陸軍第2師步兵第4旅旅長，1921年升任陸軍第10混成旅旅長。<ref name=lai/><ref name=xu/><ref name=tai/>

1922年（[[民国紀元|民国]]11年）8月，获授陸軍少将加中将衔。<ref name=lai/><ref name=xu/>1924年（民国13年）6月10日，获[[北洋将军府|将军府]]敦威将軍。<ref>1924年6月11日第2953号. 《政府公报》第二〇九册.</ref>同年，被任命为[[金華|金華]]、[[衢州|衢州]]、[[严州|严州]]鎮守使。同年12月，改任[[浙江省_(中華民国)|浙江]]寧台鎮守使。他还被任命为浙江省城守備司令。1925年（民国13年），被任命为浙江省城衛戌司令兼第8師師長。1926年（民国15年）12月，被任命为五省聯軍第3方面軍司令兼浙江（浙軍）总司令代理。<ref name=lai/><ref name=xu/><ref>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = 中華書局|isbn = 7-101-01320-1}}</ref>

孫传芳被北伐军击敗后，孟昭月避居[[天津英租界|天津英租界]]。1938年日军占领华北后，[[汪精卫政权|汪精卫政权]]曾请孟昭月赴山东任职，被孟昭月拒绝。<ref name=tai/>

1967年，孟昭月病逝。<ref name=tai/>
