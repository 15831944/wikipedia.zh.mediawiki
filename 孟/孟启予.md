'''孟启予'''（{{bd|1920年|1月1日||}}）女，原名'''陈元'''，祖籍[[福建省|福建省]][[长乐县|长乐县]]，生于[[北京|北京]]，北京电视台（今[[中国中央电视台|中国中央电视台]]）副台长、代理台长。

==生平==
1936年，孟启予参加[[中国共产党|中国共产党]]领导的革命。1938年加入中国共产党。曾在国民党统治区从事地下工作。1941年到1945年，在[[延安|延安]][[鲁迅艺术学院|鲁迅艺术学院]]学习<ref>赵玉明、王福顺 主编，中外广播电视百科全书，中国广播电视出版社，1995年</ref>。

1945年[[抗日战争|抗日战争]]胜利后，中央决定恢复[[延安新华广播电台|延安新华广播电台]]。1945年8月，延安的无线电技术工程人员安装了一座无线电广播发射台，开始试验播音。首日播送的是延安总部[[朱德|朱德]]总司令于1945年8月10日发布的限令敌伪投降的命令，播音员为[[李慕琳|李慕琳]]、孟启予。1945年9月5日，正式开始播音<ref name=licheng>{{Cite web |url = http://www.dvbcn.com/2009/09/09-38149.html |title = 1936年-2009年中国广播电视发展历程 |author =  |date = 2017-03-06 |publisher = 众视 |language =  |accessdate =  |quote =  }}</ref>。孟启予在延安新华广播电台担任播音员，后任延安新华广播电台播音组组长。1947年3月，孟启予随该电台突破国民党军队封锁，将呼号改为“陕北新华广播电台”继续播音。孟启予任陕北新华广播电台播音组组长。1949年[[北平和平解放|北平和平解放]]，陕北新华广播电台随[[中国人民解放军|中国人民解放军]]总部进驻[[北平|北平]]，孟启予继续在一线从事播音工作<ref name=zgzs>{{Cite web |url =http://zgzs.cnr.cn/archives/3236  |title = 孟启予 |author =  |date = 2014-06-28 |publisher = 中国之声 |language =  |accessdate =  |quote =  }}</ref>。

[[中华人民共和国|中华人民共和国]]成立后，孟启予任中央广播事业局编委。孟启予曾经负责筹建[[中央人民广播电台|中央人民广播电台]]少年儿童广播部，并任主任<ref name=zgzs/><ref name=ningxia>{{Cite web |url = http://roll.nxing.cn/article/481056.html |title = 现任中央电视台台长是谁中央电视台历任台长简历及级别工资 |author =  |date =2015-08-06  |publisher = 宁夏在线 |language =  |accessdate =  |quote =  }}</ref>。1955年，孟启予赴[[苏联|苏联]]学习，任[[莫斯科|莫斯科]]广播电台华语广播部编辑。回国后，1957年参与筹建中国第一个电视台“北京电视台”，并任副主任。1960年9月起，孟启予作为北京电视台副台长，负责台长工作。1963年12月至1966年9月，任北京电视台代理台长。孟启予在任内要到了[[毛泽东|毛泽东]]题写的“北京电视台”台标，该台标自1965年1月1日用到1978年5月1日“北京电视台”更名为“中央电视台”<ref name=deng>邓艳玲，那些掌管CCTV的“一哥”们，滇池天下2011年第23期</ref>。1966年[[文革|文革]]爆发后，孟启予第一批被揪出来，遭到批斗和迫害，后接受[[劳动改造|劳动改造]]<ref name=zhouxun> {{Cite book | author =周迅   | title =大海的一朵浪花：孟启予的广播电视生涯  | location =  | publisher =中国广播电视出版社  | date = 2008年 | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref><ref name=ningxia/>。1973年1月，中央任命[[任继胜|任继胜]]为北京电视台台长，[[徐国盛|徐国盛]]、[[戴临风|戴临风]]、[[王枫|王枫]]、孟启予任副台长。1978年5月1日，“北京电视台”更名为“中央电视台”，英文呼号CCTV。孟启予任中央电视台副台长。1960年代，孟启予指挥了第26届[[世界乒乓球锦标赛|世界乒乓球锦标赛]]现场直播。1970年代，孟启予组织了“[[周恩来|周恩来]]总理遗体告别”、“[[唐山大地震|重大地震]]”、“[[粉碎四人帮|粉碎四人帮]]群众大游行”等电视报道。1980年代，组织协调[[林彪、江青反革命集团案|林彪、江青反革命集团案]]审判大型电视现场直播<ref name=wangfu>王甫，《大海的一朵浪花——孟启予的广播电视生涯》评析，电视研究2008(12):23-23</ref>。1982年，孟启予离休<ref name=ningxia/>。
