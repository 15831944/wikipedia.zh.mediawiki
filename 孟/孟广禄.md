'''孟广禄'''（1962年11月？或1963年— ），中国京剧演员，演[[花脸|花脸]]，[[中国共产党|中国共产党]]党员。第十届全国政协委员，中共十七大代表。[[国家一级演员|国家一级演员]]，享受国务院特殊政府津贴。中国戏剧家协会理事。

== 经历 ==

生于[[天津市|天津市]][[河东区|河东区]]。幼入[[中国戏曲学校|中国戏曲学校]]。1981年入天津市戏曲学校进修班，1984年成为[[天津市青年京剧团|天津市青年京剧团]]演员。1987年拜[[钳韵宏|钳韵宏]]为师，并被[[方荣翔|方荣翔]]收为入室弟子。

2008年，当选第十一届全国政协委员<ref>{{cite news|title=第十一届全国政协委员名单|url=http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|accessdate=2013-04-09|newspaper=新华社北京1月25日电 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过)|date=2008年1月25日|deadurl=yes|archiveurl=https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|archivedate=2013年9月28日}}</ref>，代表文化艺术界，分入第二十七组<ref>{{cite news|title=全国政协信息-孟广禄|url=http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0009410&wytype=wjwy|accessdate=2013-04-09|newspaper=全国政协委员会}}</ref>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。

== 演出剧目 ==

《大保国、探皇陵、二进宫》、《连环套》、《铡美案》、《姚期》、《牧虎关》、《赤桑镇》、《锁五龙》、《探阴山》、《打龙袍》、《曹操父子》等。

== 荣誉 ==

全国中青年京剧演员电视大奖赛最佳演员奖、梅兰芳金奖、文华表演奖,“中国京剧之星”,2000年全国优秀青年演员评比展演荣誉奖、[http://www.chinaopera.net/html/2007-01/1192.html]“全国文化系统劳动模范”、“全国五一劳动奖章”、中国戏剧梅花奖“二度梅”获得者。
