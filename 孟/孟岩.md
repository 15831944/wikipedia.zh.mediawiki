'''孟岩'''{{BD|1980年|9月30日||}}是中國男子田徑運動員，2000年全國田徑錦標賽400公尺跨欄（91.4公分）冠軍。2005年，孟岩在中國全國運動會400公尺跨欄（91.4公分）決賽中奪冠，並以49.19秒打破全國紀錄<ref>{{cite web |author1=新文化报 |title=孟岩跨出第8金 吉林省获得田径16年零的突破 |url=http://sports.sina.com.cn/o/sy/2005-10-22/14441838643.shtml |publisher=新浪網 |accessdate=2005-10-22}}</ref>，2008年於北京中国田径公开赛男子400米跨栏获得亚军<ref>{{cite web |title=中国田径公开赛400米栏孟岩摘银 日本选手夺得冠军 |url=http://2008.sina.com.cn/cn/at/2008-05-24/212088128.shtml |publisher=新浪網 |accessdate=2008-05-24}}</ref>，且在2009年全國運動會衛冕400公尺跨欄（91.4公分）冠軍。
