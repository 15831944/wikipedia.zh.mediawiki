'''孟慶平'''（{{bd|1937年|8月|catIdx=Meng孟}}），男，出生于[[辽宁省|辽宁省]][[黑山县|黑山县]]。曾任[[海南省|海南省]]、[[湖北省|湖北省]][[副省长|副省长]]。

== 生平 ==
孟庆平的祖父从[[河北省|河北省]][[吴桥县|吴桥县]]逃荒去东北的，成为一名[[佃户|佃户]]。他自幼家庭困苦，七岁即开始农务劳动，直到十三岁才开始上学。1962年孟庆平从[[鞍山冶金专科学校|鞍山冶金专科学校]]毕业后[[分配|分配]]至[[海南省|海南省]][[昌江县|昌江县]]石碌铁矿，1965年加入[[中国共产党|中国共产党]]，曾任海南铁矿矿长，[[海南行政区|海南行政区]][[党委|党委]]副书记，1987年9月任海南省筹备组成员、省政府负责人。1988年2月任[[海南省副省长|海南省副省长]]，1993年4月调任[[湖北省副省长|湖北省副省长]]。

在担任海南省副省长期间，孟庆平利用职务之便，多次收受贿赂<ref>{{cite web |url = http://news.sohu.com/70/31/news208503170.shtml |title = 湖北原副省长孟庆平忏悔录 经不起金钱美色诱惑 |publisher=中国新闻网|date=2003年4月15日|accessdate=2010-06-22 |language=Zh-hans}}</ref>。1998年12月29日，[[新华社|新华社]]报道：[[中央纪委|中央纪委]]最近会同[[最高人民检察院|最高人民检察院]]等有关部门，严肃查处了湖北省原副省长孟庆平严重违纪违法案件。中央纪委决定，并报经中央批准，开除孟庆平[[党籍|党籍]]，[[检察机关|检察机关]]已依法将其逮捕<ref>{{cite web |url = http://news.sohu.com/20081229/n261466760.shtml |title = 1998年12月29日 湖北省原副省长孟庆平被查处|publisher=搜狐网搜狐新闻|date=2008年12月29日|accessdate=2010-06-22 |language=Zh-hans}}</ref>。后因[[受贿罪|受贿罪]]被依法判处[[有期徒刑|有期徒刑]]十年。
