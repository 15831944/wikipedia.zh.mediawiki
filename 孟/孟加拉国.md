'''-{孟加拉}-人民共和国'''（{{lang-bn|গণপ্রজাতন্ত্রী বাংলাদেশ}}，{{transl|bn|Gônôprôjatôntri Bangladesh}}），简称'''孟加拉国'''（{{lang|bn|<!--The following spelling is correct. If you see anything odd, your browser isn't Unicode compliant-->বাংলাদেশ}}），中文通稱'''孟加拉'''，[[南亚|南亚]]国家，位於[[孟加拉湾|孟加拉湾]]之北，毗邻[[尼泊尔|尼泊尔]]、[[不丹|不丹]]以及[[中华人民共和国|中国]]，却无接壤；东南山區一小部份与[[缅甸|缅甸]]为邻，其他部份都与[[印度|印度]]接壤。位于[[孟加拉灣|孟加拉灣]]的海域面积大致等同于陆地面积。孟加拉国98%的人口使用[[孟加拉语|孟加拉语]]，[[国教|国教]]为伊斯兰教；人口1.7億，排名世界[[伊斯兰国家|伊斯兰国家]]第三。<ref>{{Cite web |url=http://thediplomat.com/2016/12/the-bay-of-bengal-naval-arms-race/ |title=存档副本 |access-date=2017-09-09 |archive-url=https://web.archive.org/web/20170906134428/http://thediplomat.com/2016/12/the-bay-of-bengal-naval-arms-race/ |archive-date=2017-09-06 |dead-url=no }}</ref> 

第一大城市是首都[[达卡|达卡]]，第二大城市[[吉大港市|吉大港市]]同时是该国最大的港口。

== 历史 ==
{{main|孟加拉国历史}}
[[孟加拉族|孟加拉族]]是南亚次大陆古老民族之一。[[孟加拉|孟加拉]]地区的最早居民是亚澳人。尔后有使用[[藏缅语族|藏缅语族]]的蒙古种人从东北部进来，再后又有与[[达罗毗荼人|达罗毗荼人]]混血的[[雅利安人|雅利安人]]迁入。这几部分人经过长期融合，逐渐形成今天的[[孟加拉人|孟加拉人]]。9世纪已有统一的封建国家。孟加拉地區人民早期多信奉[[佛教|佛教]]、[[印度教|印度教]]。13世纪受外來影響改信[[伊斯兰教|伊斯兰教]]，16世纪时，該地區已經发展成[[次大陆|次大陆]]上人口最稠密、经济最发达、文化昌盛的地区。17世纪被[[莫卧儿帝国|莫卧儿帝国]]征服，1757年[[普拉西戰役|普拉西戰役]]後遭受英国殖民统治，为英属印度的一个省。<ref>{{cite web | url=https://www.britannica.com/event/Battle-of-Plassey | title=Battle of Plassey | access-date=2018-02-21 | archive-url=https://web.archive.org/web/20180221162356/https://www.britannica.com/event/Battle-of-Plassey | archive-date=2018-02-21 | dead-url=no }}{{en}}</ref>

1904年孟加拉地区曾被[[英国|英国]]殖民政府分割成东西两部，后来復合。1947年[[印巴分治|印巴分治]]时，孟加拉地区被再次分割：西孟加拉地区归[[印度|印度]]（今[[西孟加拉邦|西孟加拉邦]]），东孟加拉地区（后改名称[[东巴基斯坦|东巴基斯坦]]）则根据最后一任印度总督蒙巴顿提出的[[蒙巴顿方案|蒙巴顿方案]]归[[巴基斯坦|巴基斯坦]]。但是，地理上的相互隔绝，民族、文化和语言的巨大差异终于使相距约2000公里的东、西巴基斯坦内部矛盾走向不可调和的地步。

1971年3月26日东巴基斯坦宣布独立，并在4月于印度[[加尔各答|加尔各答]]成立[[孟加拉人民共和国临时政府|孟加拉人民共和国临时政府]]，随后东巴基斯坦发生大规模动乱和难民潮。印度以战争手段支持-{zh-tw:孟加拉;zh-cn:孟加拉国; zh-hk:孟加拉國;}-独立以牽制巴基斯坦。11月21日，印军在[[苏联|苏联]]的支持下，大举入侵东巴基斯坦，[[第三次印巴战争|第三次印巴战争]]爆发。12月，战争扩大到西巴基斯坦，发展成全面战争。12月7日，[[联合国|联合国]]大会以压倒多数票通过决议，要求印巴双方停火和撤军。印军仍持续大举进攻，于12月16日攻占达卡，东巴驻军无条件投降。

1972年1月成立-{孟加拉}-人民共和国，[[巴基斯坦总统|巴基斯坦总统]][[佐勒菲卡尔·阿里·布托|佐勒菲卡尔·阿里·布托]]释放了被以“[[叛国罪|叛国罪]]”逮捕的东巴人民联盟领袖[[谢赫·穆吉布·拉赫曼|谢赫·穆吉布·拉赫曼]]，此人就任孟加拉国首任总统。

1974年爆發獨立以來[[1974年孟加拉國飢荒|首次大規模飢荒]]。

1975年8月15日，[[孟加拉国人民聯盟|孟加拉国人民聯盟]]領袖、首任总统[[谢赫·穆吉布·拉赫曼|谢赫·穆吉布·拉赫曼]]遇刺身亡。1981年5月30日，[[孟加拉国民族主義黨|孟加拉国民族主義黨]]領袖、[[齊亞·拉赫曼|齊亞·拉赫曼]]总统在[[吉大港|吉大港]]遇刺身亡，阿卜杜勒·萨塔爾出任代总统。

1982年3月24日，陆军参谋长[[侯赛因·穆罕默德·艾尔沙德|侯赛因·穆罕默德·艾尔沙德]]宣布实行军管，解除萨塔尔的总统职务，自任军法管制首席执行官兼武装部队总司令，后任总统。齐亚·拉赫曼遺孀[[卡莉达·齐亚|卡莉达·齐亚]]领导的民族主义党及穆吉布·拉赫曼長女[[谢赫·哈西娜|谢赫·哈西娜]]人民聯盟同艾尔沙德軍政府展開斗争，1990年12月6日，艾尔沙德辞职，结束他九年的獨裁统治。在1991年大选中，[[卡莉达·齐亚|卡莉达·齐亚]]获胜出任总理。

2006年10月31日，無党派看守军政府成立，总统[[亚杰丁·艾哈迈德|亚杰丁·艾哈迈德]]兼任看守军政府总理。孟加拉国第九届议会选举原定于2007年1月22日举行，但由于当时两大政党前总理卡莉达·齐亚为首的民族主义党和以前总理谢赫·哈西娜为首的人民联盟斗争日趋激烈，政局日益恶化，总统亚杰丁·艾哈迈德被迫于2007年1月11日宣布国家进入紧急状态，取消一切政党活动。2007年1月12日，法赫尔丁·艾哈迈德宣誓就任看守政府总理<ref>{{Cite web |url=http://news.xinhuanet.com/world/2007-01/12/content_5600177.htm |title=存档副本 |access-date=2007-01-13 |archive-url=https://web.archive.org/web/20071205230322/http://news.xinhuanet.com/world/2007-01/12/content_5600177.htm |archive-date=2007-12-05 |dead-url=no }}</ref>。他的看守军政府執政兩年，在國際壓力下，2008年12月29日舉辦國會大選<ref>{{Cite web |url=http://www.china.com.cn/international/txt/2008-05/13/content_15180845.htm |title=孟加拉国看守政府总理宣布今年底举行议会选举 |access-date=2013-05-14 |archive-url=https://web.archive.org/web/20140815144043/http://www.china.com.cn/international/txt/2008-05/13/content_15180845.htm |archive-date=2014-08-15 |dead-url=no }}</ref>，結果人民聯盟大勝，贏得國會300席中262席，黨主席[[谢赫·哈西娜|谢赫·哈西娜]]於2009年1月6日第二次出任總理<ref>{{Cite web |url=http://www.mofa.gov.tw/webapp/ct.asp?xItem=167&ctnode=276 |title=中華民國外交部 - 國情簡介 |accessdate=2011-06-06 |work= |publisher= |date= |archive-url=https://web.archive.org/web/20111128020027/http://www.mofa.gov.tw/webapp/ct.asp?xItem=167&ctnode=276 |archive-date=2011-11-28 |dead-url=no }}</ref>。2009年2月12日[[齊魯爾·拉赫曼|齊魯爾·拉赫曼]]就任第十九任孟加拉國總統。近年孟加拉国社会较为稳定，经济发展迅速。

== 地理 ==
{{Main|孟加拉國地理}}
[[File:Panthera_tigris_tigris.jpg|thumb]]]]

-{zh-tw:孟加拉;zh-cn:孟加拉国; zh-hk:孟加拉國;}-东南部邻[[缅甸|缅甸]]，东、西、北三面与印度毗连，南临[[孟加拉湾|孟加拉湾]]。80％以上的領土位于南亚次大陆东北部的[[恒河|恒河]]和[[布拉马普特拉河|布拉马普特拉河]]下游冲击而成的三角洲上，屬於肥沃、平坦的冲积平原，河道纵横密布，河运发达，河流和湖泊约占全国面积10％，非常适合农业和渔业，但雨季极易泛滥。沿海多小岛和沙洲。國土的大部分低於海拔12米（39呎），如果[[海平面|海平面]]上升1米（3呎），就會有大約10%土地會被淹沒。东南端为[[吉大港山區|吉大港山區]]，平均海拔300—600米，-{zh-tw:孟加拉;zh-cn:孟加拉国; zh-hk:孟加拉國;}-最高峰凯奥克拉东峰，高1229米，剛好位於此地理區上。矿藏有[[天然气|天然气]]、[[煤|煤]]、[[钛|钛]]、[[锆|锆]]等。

=== 气候 ===
-{zh-tw:孟加拉;zh-cn:孟加拉国; zh-hk:孟加拉國;}-大部分地区属于[[热带季风气候|热带季风气候]]，季风具有不稳定性而多旱涝；年降水量西部约1300-1500毫米，东部达2000-2500毫米。月平均最高气温40.5℃，雨量自西向东递增，年雨量1300—2500毫米。1月温度最低，1-4月干旱。6—10月是雨季，雨量占全年80％。常年有从印度洋刮来的[[飓风|飓风]]。

[[黃麻|黃麻]]是孟加拉主要的經濟來源，於[[第二次世界大戰|第二次世界大戰]]至1940年代後期達到高峰，達到世界出口量的80%<ref>{{Cite book

2006年的[[人均年收入|人均年收入]]為1,400美元，比起世界上一些發達國家的人均年收入10,200美元還有很大的差距。黃麻是孟加拉國最主要的經濟出口作物之一，除此之外還出口[[大米|大米]]、[[茶|茶]]、[[芥菜|芥菜]]等，该国于2011年超越印度成为仅次于[[中国大陸|中国大陸]]的世界第二大[[纺织品|纺织品]]出口国<ref >{{cite web|url=http://www.zaobao.com/gj/gj130217_014_2.shtml | language= | title=孟加拉发展前路障碍重重 |publisher=联合早报 |date=2013-02-17 |accessdate= }}</ref>。根據[[世界銀行|世界銀行]]的統計，自從1990年以來，孟加拉的中產階級，以及消費產業都有所擴大。孟加拉有許多的跨國公司例如[[Unocal_Corporation|Unocal Corporation]]和[[雪佛龍|雪佛龍]]，當地公司，例如[[Beximco|Beximco]]、[[SQUARE_(Bangladesh)|Square]]、[[Akij_Group|Akij Group]]、[[M._M._Ispahani_Limited|Ispahani]]、[[Navana_Group|Navana Group]]、[[橫川公司|橫川公司]]、[[Habib_Group|Habib Group]]、[[KDS_Group|KDS Group]]。

孟加拉國一位著名的[[經濟學家|經濟學家]]，[[穆罕默德·尤纳斯|穆罕默德·尤纳斯]]（2006年獲得[[諾貝爾和平獎|諾貝爾和平獎]]），他通過[[格萊玟銀行|格萊玟銀行]]（Grameen Bank）所提出的已廣泛傳播的微額借款，對孟加拉的經濟發展產生了深遠的影響。

== 文化 ==
-{zh-tw:孟加拉;zh-cn:孟加拉国; zh-hk:孟加拉國;}-人原本信仰[[佛教|佛教]]與[[印度教|印度教]]，12世纪[[突厥人|突厥人]]入侵後，[[蘇菲派|蘇菲派]][[傳教士|傳教士]]進入傳播[[伊斯兰教|伊斯兰教]]，到15世纪幾乎全民皆是伊斯兰[[穆斯林|穆斯林]]，只有少数[[印度教|印度教]][[性力派|性力派]]信徒生活在達卡。佛教是孟加拉國第三大宗教，大約佔全國人口0.7%，主要信仰上座部佛教。65％佛教徒是集中在[[吉大港山區|吉大港山區]]的[[藏缅語族|藏缅語族]]地區，而35％的人口是在吉大港古孟加拉語巴魯阿社區。佛教團體也分布在不同的城市，特別是吉大港和達卡。

=== 文化教育 ===
详见：{{Link-en|孟加拉教育|Education in Bangladesh}}

[[学制|学制]]为[[小学|小学]]五年、[[中学|中学]]七年、[[大学|大学]]四年。现政府重视教育，规定八年级以下女生享受免费和[[义务教育|义务教育]]。国立[[大学|大学]]21所，私立大学53所，国立医学院13所，普通学院1225所，工艺学校77所，[[伊斯兰学校|伊斯兰学校]]8410所，专业培训学院64所，中学17386所，小学78000所。5-24岁适龄学生[[入学率|入学率]]57%，教师人数17万。孟[[识字率|识字率]]为62.66%，其中男性为65.94%，女性为58.69%，成人识字率为54.80%。

主要高校有[[达卡大学|达卡大学]]（Dhaka University）、{{Link-en|孟加拉工程技术大学|Bangladesh University of Engineering and Technology}}（BUET）、{{Link-en|拉吉沙希大学|University of Rajshahi}}等。

== 交通運輸 ==
公路：总里程222593千米。其中[[国家公路|国家公路]]3144千米，地区公路1746千米，支线路35454千米，乡村路182249千米。76％的货运及73％的客运由公路运输承担。公路路况整体欠佳。

铁路：总里程2880千米。2003年[[旅客周转量|旅客周转量]]40.2亿人千米，[[货运量|货运量]]为9.5亿吨千米。使用1676毫米 ～1000毫米[[雙軌距鐵路|雙軌距鐵路]]。

水运：海运主要港口为[[吉大港|吉大港]]，承担了该国绝大多数的对外贸易。-{zh-tw:孟加拉;zh-cn:孟加拉国; zh-hk:孟加拉國;}-内河运输公司（BIWTC）拥有船只215艘。孟加拉国运输公司（BSC）拥有船只13艘，运输能力19.6万吨。

空运：[[孟加拉航空|孟加拉航空]]（Biman Bangladesh）现有飞机17架，[[国内航线|国内航线]]7条，[[国际航线|国际航线]]26条。-{zh-tw:孟加拉;zh-cn:孟加拉国; zh-hk:孟加拉國;}-现有国际机场3个（达卡、吉大港、锡莱特），国内机场5个。
