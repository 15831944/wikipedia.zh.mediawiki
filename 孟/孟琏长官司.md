'''孟琏长官司'''是[[明朝|明朝]]直隶于[[云南都指挥使司|云南都指挥使司]]的一个土司，即[[孟连土司|孟连土司]]，治今[[孟连傣族拉祜族佤族自治县|孟连]][[娜允古镇|娜允古镇]]<ref>{{cite web |url=http://cpc.people.com.cn/GB/165240/165562/167249/10105422.html |title=云南娜允 中国最后一个傣族古城 |last= |first= |date=2009-09-23 |website=cpc.people.com.cn |publisher=新华网 |access-date=2017-10-30 |quote=}}</ref>。

[[元朝|元]][[至正|至正]]八年（1348年），原[[木连路|木连路]]被[[麓川路|麓川路]]吞并，明朝初为[[麓川平缅宣慰司|麓川平缅宣慰司]]辖地，1402年后属[[孟定府|孟定府]]。[[永乐_(明朝)|永乐]]四年（1406年）置孟琏长官司，辖境包括元时期的木连路和[[银沙罗甸|银沙罗甸府]]部分地区（今孟连、[[澜沧拉祜族自治县|澜沧]]、[[西盟佤族自治县|西盟]]等县和[[缅甸|缅甸]][[掸邦|掸邦]]东北[[萨尔温江|萨尔温江]]以东部分地区）<ref>{{cite book |last=谭 |first=其骧 |date= |title=《中国历史地图集 第七册》 |url= |location=[[北京|北京]] |publisher=[[中国地图出版社|中国地图出版社]] |page=24 |isbn=978-7-5031-1844-9 |author-link=谭其骧 }}</ref>
，[[车里宣慰司|车里宣慰司]]也赠予孟琏司一片土地（孟连土司曾帮助过车里宣慰，两土司曾经联姻，车里将部分领土划给孟连作为酬谢和陪嫁）<ref name="傣族网">{{cite web |url=http://www.daizuwang.com/Article_Show.asp?ArticleID=1527 |title=孟连土司历史简述 |last=莫 |first=汝德 |date=2009-02-17 |website=傣族网 |publisher=孟连旅游网 |access-date=2017-10-30 |quote=}}</ref>。[[嘉靖|嘉靖]]中期废除，[[万历|万历]]十三年（1585年）复设，称'''猛脸'''。<ref>{{cite book |author=郭红, 靳润成 |date=2007 |title=《中国行政区划通史·明代卷》 |url= |location=[[上海|上海]] |publisher=[[复旦大学|复旦大学]]出版社 |page=211 |isbn=978-7-3090-5598-6 |author-link= }}</ref><ref>《[[明史|明史]]》[[:s:明史/卷315#孟琏〈（即猛脸）|卷三百一十五]]</ref>

[[康熙|康熙]]元年（1662年）孟连土司归附[[清朝|清朝]]，依明建制仍为孟琏长官司。康熙四十八年（1709年）升格[[孟连宣抚司|孟连宣抚司]]。<ref name="傣族网"></ref>
