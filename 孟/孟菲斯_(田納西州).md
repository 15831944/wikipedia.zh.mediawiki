'''孟菲斯'''（{{lang-en|'''Memphis'''}}）是[[美國|美國]][[田納西州|田納西州]][[謝爾比縣_(田納西州)|謝爾比縣]]的一座[[城市|城市]]，也是謝爾比縣[[行政中心|縣治]]。根據2006年的人口預估，孟菲斯的總人口為68萬0768人，是田納西州最大的城市，在美國東南部排名第二、次於[[佛羅里達州|佛羅里達州]][[傑克遜維爾_(佛羅里達州)|傑克森維爾]]，也是全國第17大城市<ref>[http://www.census.gov/popest/cities/tables/SUB-EST2006-01.xls 城市人口排名 - 美國人口普查局(XLS檔案)]</ref>。孟菲斯都會區是田納西州次於[[纳什维尔|纳什维尔]]都會區的第二大都會區，包含在鄰近[[密西西比州|密西西比州]]和[[阿肯色州|阿肯色州]]的部份地區，總人口為126萬0581人。在包含[[諾克斯維爾|諾克斯維爾]]、[[查特怒加_(田納西州)|查特怒加]]、和纳什维尔的田納西州四大城市裡，孟菲斯是人口最多但也是最年輕的。孟菲斯位於[[密西西比河|密西西比河]]岸邊、下[[契卡索陡岸|契卡索陡岸]]（{{lang|en|Chickasaw Bluff}}）上，狼河（{{lang|en|Wolf River}}）的河口。其聯外機場[[孟菲斯国际机场|孟菲斯国际机场]]是世界第二大貨運機場。

==地理与气候==
孟菲斯属典型的[[副热带湿润气候|副热带湿润气候]]，四季分明，气温变化相对和缓，全年降水分配均匀。冬季温和，微潮，时而偏向温暖，日最高气温低于或等于{{convert|0|C}}的平均日数为4.7天，日最低气温低于{{convert|0|C}}的平均日数为47天，低于{{convert|-10|C}}的平均日数为2.7天；夏季炎热潮湿，日最高气温超过{{convert|30|C}}的日数年均有107天，超过{{convert|35|C}}的有23天。<ref name = "NWS Memphis, TN"/>最冷月（1月）均温{{convert|41.2|F|1|disp=flip}}，极端最低气温{{convert|−13|F|0|disp=flip}}（1963年12月24日）。<ref name = "NWS Memphis, TN"/>最热月（7月）均温{{convert|82.7|F|1|disp=flip}}，极端最高气温{{convert|108|F|0|disp=flip}}（1980年7月13日）。<ref name = "NWS Memphis, TN"/>无霜期平均为239天（3月20日至11月13日）。<ref name = "NWS Memphis, TN"/>年均降水量约{{convert|53.7|in|sigfig=3|disp=flip}}，年极端最少降水量为{{convert|33.44|in|disp=flip}}（1916年），最多为{{convert|76.85|in|disp=flip}}（1957年）。<ref name = "NWS Memphis, TN"/>年均降雪量为{{convert|3.8|in|cm|disp=flip}}，<ref name = "NWS Memphis, TN"/>但许多冬季降雪量微少以致难以测量，1917–18年的降雪量最多，积累降雪量为{{convert|25.1|in|cm|disp=flip}}。<ref name = "NWS Memphis, TN"/>
