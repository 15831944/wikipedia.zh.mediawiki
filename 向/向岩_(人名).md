'''向岩'''（{{bd|1872年||1959年|}}）原名'''寿荫'''，字'''少蒨'''，[[湖北|湖北]][[汉川|汉川]]人，[[中华民国|中华民国]]军事将领，[[中华人民共和国|中华人民共和国]]政治人物。

==生平==
1905年到[[日本|日本]]，进入东京陆军东斌学校学习。1906年加入[[中国同盟会|中国同盟会]]。1908年毕业归国，到[[四川|四川]]任陆军速成学堂教官，兼督练公所编译制度等科一等科员。[[武昌起义|武昌起义]]后，以川南宣抚使的名义发布《谕巴蜀檄》。1912年任[[鄂军都督府|鄂军]]第八师参谋长，后因裁军而去职<ref name=rmw/><ref>{{Cite web |url = http://news.ifeng.com/a/20140528/40488306_0.shtml |title =北洋海军招牌被日军藏于靖国神社  |author =  |date = 2014-05-28 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。1913年[[二次革命|二次革命]]爆发，讨[[袁世凯|袁]]军兴起，任江南陆军第一师参谋长。二次革命失败后，到日本加入[[中华革命党|中华革命党]]。曾经回国在[[襄河|襄河]]组织反[[袁世凯|袁]]斗争。[[护法运动|护法运动]]期间，任靖国联军第八军参谋长兼第二混成旅旅长，转战[[湖北|湖北]]、[[四川|四川]]、[[陕西|陕西]]、[[甘肃|甘肃]]、[[云南|云南]]、[[贵州|贵州]]七省。1922年参加北伐，任[[广东|广东]][[韶关|韶关]]大本营将校团副团长、北伐军第五路司令等职务。1925年任[[国民军|国民军]]第三军左翼督战官、游击司令。1926年在湖北组织北伐军别动队。[[抗日战争|抗日战争]]时期曾支援抗战，后曾营救爱国人士。[[中华人民共和国|中华人民共和国]]成立后，历任[[中南军政委员会|中南军政委员会]]参事室参事、[[湖北省政协|湖北省政协]]特邀委员、[[湖北省人民代表大会|湖北省人民代表大会]]代表<ref name=rmw>{{Cite web |url = http://politics.people.com.cn/GB/8198/203099/203103/15707713.html |title = 向岩 |author =  |date = 2001-09-20 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。
