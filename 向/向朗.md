'''向朗''' （{{Bd|？||247年}}），字'''巨達'''，[[襄陽|襄陽]][[宜城|宜城]]人，[[三國|三國]]時期[[蜀漢|蜀漢]]官吏。

== 生平 ==

[[向寵|向寵]]叔父。少年事師於[[司馬徽|司馬徽]]之下，與[[徐庶|徐庶]]、[[韓嵩|韓嵩]]、[[龐統|龐統]]甚有交情。後來[[劉表|劉表]]任其為[[臨沮|臨沮]]長。208年，劉表病故，轉投為[[劉備|劉備]]手下。同年劉備平定荊南，任向朗督[[秭歸|秭歸]]、[[夷道|夷道]]、[[巫山|巫山]]、[[夷陵|夷陵]]四縣的軍、民事。

214年，劉備入主[[益州|益州]]，任向朗為[[巴西郡|巴西]][[太守|太守]]，不久再轉任[[牂牁|牂牁]]、[[房陵|房陵]]。223年，[[劉禪|劉禪]]繼位，升為[[步兵校尉|步兵校尉]]，代[[王連|王連]]領丞相[[長史|長史]]。225年，[[諸葛亮|諸葛亮]]南征，留向朗統領後方之事。

兩年後，跟隨諸葛亮駐軍[[漢中|漢中]]並進行[[诸葛亮北伐|第一次北伐]]，可惜因[[馬謖|馬謖]]之故讓蜀軍於[[街亭之戰|街亭大敗]]。馬謖棄軍逃亡的消息向朗早已知情，因與馬謖私交甚篤故知情不報，諸葛亮對此十分忿怒，免向朗之官並送還[[成都|成都]]。<ref>《向郎传》记载：「向朗平时与（马）谡善，（马）谡逃亡，（向）朗知情不举，亮恨之，免官还成都。」，裴松之的注释为：「（向）朗坐马谡免长史，则建兴六年中也。」</ref>數年後，復職為[[光祿勳|光祿勳]]。

234年諸葛亮死後，轉任[[左將軍|左將軍]]，追論舊功，封為顯明亭侯，地位特進。238年，為劉禪主持[[張皇后_(劉禪)|張貴人]]封皇后的儀式。<ref>《三國志·蜀書·後主張皇后傳》：延熙元年春正月，策曰：「朕統承大業，君臨天下，奉郊廟社稷。今以貴人為皇后，使行丞相事'''左將軍向朗'''持節授璽綬。勉脩中饋，恪肅禋祀，皇后其敬之哉！」</ref>後來開始不問時事，不倦研究經書、古典，於[[延熙|延熙]]十年（247年）逝世，其間優遊無事數十年。<ref>《三國志·蜀書·霍峻等傳第十一》：“自去長史，優遊無事垂三十年，臣松之案：朗坐馬謖免長史，則建興六年中也。朗至延熙十年卒，整二十年耳，此雲「三十」，字之誤也。”</ref>

== 特徵 ==
向朗少時雖有接觸[[文學|文學]]，但沒有大興趣，反而以內政能力見稱。後來在第一次北伐被免官，到死前二十年，開始潛心典籍研究，孜孜不倦。年近八十仍親自[[校對|校對]]書籍，指出謬誤之處，積聚的篇卷，是當時最多的一位。

大開門户，迎接賓客，訓練後起之秀，不過只談論經書、古典，不問時事，上至大臣，下到小童都非常敬重他。

== 評價 ==

[[三國志|三國志]]評曰:「[[霍峻|霍峻]]孤城不傾，[[王連|王連]]固節不移，向朗好學不倦，[[張裔|張裔]]膚敏應機，[[楊洪|楊洪]]乃心忠公，[[費詩|費詩]]率意而言，皆有可紀焉。以先主之廣濟，諸葛之準繩，詩吐直言，猶用陵遲，況庸后乎哉！」
