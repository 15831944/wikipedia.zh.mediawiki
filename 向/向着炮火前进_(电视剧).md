{{engname|《向著炮火前進》|To Advance toward the Fire}}，2012年上映于中国大陆的[[抗日战争|抗日战争]]题材电视剧，导演[[林建中|林建中]]，主演[[吴奇隆|吴奇隆]]、[[甘婷婷|甘婷婷]]、[[王新|王新]]、[[谢孟伟|谢孟伟]]等。<ref>{{cite web |url=http://gb.cri.cn/27224/2012/11/02/1042s3911021.htm |title=《向着炮火前进》剧照 吴奇隆三兄弟血战沙场 |accessdate=2012-11-02 |publisher= 国际在线|work= |archiveurl= |archivedate= |language =zh-cn}}</ref>导演林建中认为电视剧为“偶像抗日剧”。<ref>{{cite web |url=http://ent.qq.com/a/20121028/000170.htm |title= 《向着炮火前进》江苏开播 导演打造偶像抗日剧|accessdate=2012年10月28日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

== 剧情 ==
1945年，抗日战争进入收尾阶段，[[大日本帝国|大日本帝国]]开始溃败和撤退，一架带着机密的[[日军|日军]][[飞机|飞机]]坠毁在团城，飞机坠毁后，日军派人开始搜索和抓捕抢走飞机内物品者，[[游击队|游击队]]的上官于飞也奉命寻找到飞机上的一些物品，日军和游击队长刘建功、上官于飞的部队遭遇，两军交战，日军火力强大，游击队遭到重创，刘建功跳下悬崖幸运地挂在树梢才救回性命。团城著名的飞狐岭狐牙峰山寨头子雷子枫出手营救了上官于飞。雷子枫的母亲看到上官于飞后发现其和雷子枫的前妻相像，心里暗暗地盘算促成二人结婚。日军为了夺回机密，用机枪杀害了团城峡口村的全村人，石敢当、玲珑、胖墩三人因不在村中幸免于难。上官于飞因雷子枫的救助而心生情愫，雷母也非常满意，不久之后，山寨的朱半山投靠日军，趁着雷子枫带人去团城抢夺密码机的时机，掳走雷母，雷子枫回来后带人救出母亲，但是山寨弟兄全部被杀，母亲也因伤逝世，为满足母亲遗愿，雷子枫和上官于飞假装结了婚。当刘建功回来之后，经过与上官于飞、雷子枫的一番磨合，三人决心携手拯救团城，由于朱半山熟知团城地形情势，游击队和日军交战屡屡失败，一次，游击队冲入团城和日军决战，遭到重创，在最后危急关头，幸亏游击队的大部队出兵相助，才解围并且歼灭团城日军。

== 花絮 ==
吴奇隆出演的雷子枫被誉为集“霸气、匪气、孩子气”于一身。<ref>{{cite web|url= http://news.lnd.com.cn/htm/2012-11/02/content_2585407.htm|title= 吴奇隆“炮火”中传绯闻|accessdate= 2012-11-02|publisher= 北国网|work= |language= zh-cn|archive-url= https://web.archive.org/web/20121103092759/http://news.lnd.com.cn/htm/2012-11/02/content_2585407.htm|archive-date= 2012-11-03|dead-url= yes}}</ref>同时，观众认为雷子枫的造型太浮夸，耍酷、卖萌，衣服穿越不符合历史条件下的土匪形象。<ref>{{cite web|url=http://e.gmw.cn/2012-11/01/content_5550396.htm|title="向着炮火前进"惹争议 吴奇隆"偶像土匪"有点雷|accessdate=2012-11-01|publisher=光明网|work=|language=zh-cn}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url=http://ent.163.com/12/1101/06/8F7399UA00032DGD.html |title=《炮火》“土匪”吴奇隆行头拉风 被吐槽太穿越 |accessdate=2012-11-01 |publisher=网易 |work= |language =zh-cn}}</ref><ref>{{cite web |url= http://ent.163.com/12/1031/07/8F4ISGIT00032DGD.html|title= 《炮火》江苏开播 吴奇隆演土匪造型太潮引争议|accessdate=2012-10-31 |publisher=网易 |work= |language =zh-cn}}</ref>

吴奇隆和甘婷婷出演中共游击队的形象突破了以往严肃、古朴的造型。<ref>{{cite web |url=http://news.xinhuanet.com/ent/2012-11/01/c_123899525.htm |title=《向着炮火前进》收视喜人 甘婷婷再突破军旅角色 |accessdate=2012年11月1日 |publisher=新华网 |work= |language =zh-cn}}</ref>

吴奇隆爆料和甘婷婷的吻戏，每次前，甘婷婷都要吃[[大蒜|大蒜]]。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2012-10-29/17313774897.shtml |title=吴奇隆：甘婷婷拍吻戏前爱吃大蒜 |accessdate= 2012年10月29日|publisher=新浪网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://yule.sohu.com/20121029/n355968186.shtml |title=吴奇隆谈深吻“潘金莲” 每次都有一股蒜香味 |accessdate= 2012年10月29日|publisher=搜狐网 |work= |language =zh-cn}}</ref>

甘婷婷在记者会上讲自己将告别以往电视剧中有“[[潘金莲|潘金莲]]”影子的现象，这次很纯情。<ref>{{cite web |url=http://yule.sohu.com/20121029/n355968824.shtml |title=甘婷婷力破“潘金莲”形象：这次我很纯情 |accessdate=2012年10月29日 |publisher= 搜狐网|work= |language =zh-cn}}</ref>

杨奇鸣出演的狙击手阿福被誉为经典角色。<ref>{{cite web |url=http://yule.sohu.com/20121102/n356466535.shtml |title=11月收视榜《炮火》脱颖而出 杨奇鸣塑造经典 |accessdate=2012年11月2日 |publisher=搜狐网 |work= |language =zh-cn}}</ref>

王新出演的刘建功是一个[[教师|教师]]出身，在军中亦严肃、不拘笑容，观众认为造型太呆板、保守。<ref>{{cite web |url=http://entertainment.dbw.cn/system/2012/11/02/054343304.shtml |title=《炮火》队伍重组遇难题 观众认为王新太保守 |accessdate=2012-11-02 |publisher=东北网 |work= |language=zh-cn |archive-url=https://web.archive.org/web/20121124035957/http://entertainment.dbw.cn/system/2012/11/02/054343304.shtml |archive-date=2012-11-24 |dead-url=yes }}</ref>

新秀女演员王碧琪的出现为电视剧赢得了亮点。<ref>{{cite web |url=http://news.xinhuanet.com/ent/2012-10/22/c_123853102.htm |title= 《向着炮火前进》热播 王碧琪反角出镜受关注 |accessdate=2012年10月22日 |publisher= 新华网|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://news.xinhuanet.com/ent/2012-05/03/c_123071096.htm |title= 抗战大戏《向着炮火前进》杀青 王碧琪挑战新角色|accessdate=2012年5月3日 |publisher=新华网 |work= |language =zh-cn}}</ref>
