'''贵阳市快速公交系统'''（{{lang-en|Guiyang Bus Rapid Transit}}，又称“中环快速公交”<ref name="rmw2017">{{cite web |url=http://gz.people.com.cn/n2/2017/0113/c194827-29592397.html |title=贵阳BRT体验流程全报告 换乘路线一图懂 |author=贵阳晚报 |publisher=人民网 |date=2017-01-13 |accessdate=2018-07-02 }}</ref>）是[[贵州省|贵州省]][[贵阳市|贵阳市]]的[[快速公交系统|快速公交系统]]（BRT），于2017年2月13日正式开通。贵阳快速公交设有4条主线及9条支线，均途经贵阳中环路。
==历史==
作为贵州省省会，贵阳中心城区人口密度大，交通出行需求大，但长久以来贵阳的公共交通过度依赖常规公交，且常规公交运行效率低下，2014年贵阳公交高峰时段平均运营时速仅达10公里/小时，堵车时只有7公里/小时。<ref>{{cite web |url=http://news.ifeng.com/a/20140917/42005954_0.shtml |title=贵阳1.5环将建BRT廊道 |author=贵州都市报 |publisher=凤凰网 |date=2014-09-17 |accessdate=2018-07-06 }}</ref><ref>{{cite web |url=http://news.gog.cn/system/2014/09/16/013799163.shtml |title=从"挤"公交到"坐"公交 2018年贵阳建成"公交都市" |author=贵州都市报 |publisher=多彩贵州网 |date=2014-09-06 |accessdate=2018-07-06 }}</ref>2013年，贵阳市政府出台《贵阳市中心城区畅通工程二期实施方案》，提出建设1.5环（中环路），并同步建设1.5环快速公交系统，<ref>{{cite web |url=http://www.gywb.cn/content/2013-12/11/content_353565.htm |title=贵阳市中心城区畅通工程二期实施方案出台 |author=贵阳晚报 |publisher=贵阳网 |date=2013-12-11 |accessdate=2018-07-06 }}</ref>2014年，贵阳中环路开工建设。<ref>{{cite web |url=http://gz.sina.com.cn/news/city/2016-12-30/detail-ifxzczff3427853.shtml |title=贵阳市中环路背后的故事：中心城区施工困难多 |author=贵阳晚报 |publisher=新浪 |date=2016-12-30 }}</ref>

2016年11月，贵阳快速公交首批车辆到达贵阳并开始进行检测。<ref name="firstbuses">{{cite web |url=http://www.sohu.com/a/119685367_119665 |title=贵阳首批“铰接式BRT公交车”亮相 12月将完成检测 |author=贵阳日报 |publisher=新浪 |date=2016-11-23 }}</ref>中环路在同年12月31日开通，快速公交系统同步进行调试和试运营。<ref>{{cite web |url=http://www.gywb.cn/content/2016-12/28/content_5417578.htm |title=中环路12月31日正式通车 BRT将于明年2月下旬正式运营  |publisher=贵阳网 |date=2016-12-28 }}</ref>

2017年1月21日，贵阳快速公交B1、B2、B3路三条主线开通试运行，贵阳公交258、292路调整线路成为快速公交系统支线，在快速公交部分车站与三条主线同台换乘，另有99条常规公交线路接驳快速公交。试运行期间乘客免费乘坐，当天运送乘客约31500人次。<ref>{{cite web |url=http://news.ifeng.com/a/20170123/50617945_0.shtml |title=贵阳BRT试运行首日 超3万人次乘坐体验  |publisher=凤凰网 |author=贵州都市报 |date=2017-01-23 }}</ref><ref name="rmw2017"/>贵阳快速公交于2月13日正式投入服务并开始收费，票价2元/人次，除购买单程票外，乘客也可使用贵阳常规公交IC卡乘车。<ref>{{cite web |url=http://www.myzaker.com/article/58a191731bc8e0b60e00079d/ |title=贵阳 BRT 今日正式开通 乘车购票疑问一一为您解读  |publisher=ZAKER |author=贵阳晚报 |date=2017-02-13 }}</ref>

2017年8月，贵阳快速公交系统新开通B4路，调整B3路，并调整B247、B259、B224、B267、B268、B236、B242七条公交线路行经中环路，快速公交系统主线增加至4条，支线增加至9条，系统覆盖面由原先的[[云岩区|云岩]]、[[南明区|南明]]、[[观山湖区|观山湖]]，增加覆盖[[贵阳站|火车站]]、金阳客车站、花果园、新添寨、[[白云区_(贵阳市)|白云]]、[[花溪区|花溪区]]及老城核心区，运营车辆由50台增加至132台。<ref>{{cite web |url=http://news.gog.cn/system/2017/08/16/015999733.shtml?from=singlemessage |title=贵阳BRT到底有多少条线路？四主线 九支线！(图)  |publisher=多彩贵州网 |date=2017-08-16 }}</ref><ref name="newroutes">{{cite web |url=http://sc.gog.cn/system/2017/08/08/015971476.shtml |title=今起贵阳BRT新增8条线路，6城区全覆盖！来看看经过你家没？  |publisher=多彩贵州网 |date=2017-08-08 }}</ref>
==线路==
目前贵阳快速公交系统共有4条主线和9条支线，主线行驶中环路全线，支线由常规公交调整而来，连接市政道路和中环路快速公交站台实现接驳。

==票务==
贵阳BRT实行2元一票制，使用贵阳公交IC卡则为1.8元。乘客也可使用手机扫码支付乘车。在各主线和支线之间无限换乘2小时内免费。<ref>{{cite web |url=http://www.sohu.com/a/192021749_822192 |title=贵阳BRT实现扫码坐车~同台免费无限次换乘！（附扫码教程）  |publisher=搜狐 |date=2017-09-14 }}</ref>

贵阳BRT在站台设收费区，乘客需提前购票或使用储值票刷卡过闸进站乘车。
==车辆==
贵阳BRT目前共有132辆运营车辆，<ref name="newroutes"/>其中包括6辆18米三门[[铰接客车|铰接客车]]（[[北汽福田|北汽福田]]车型）和44辆12米长[[奇瑞汽车|奇瑞万达贵州客车]]，此50辆车均以[[液化天然气|液化天然气]]为燃料。<ref name="firstbuses"/><ref>{{cite web |url=http://www.sohu.com/a/121105717_398047 |title=贵阳1.5环BRT 首批公交车全部交付 |publisher=搜狐 |date=2016-12-09 }}</ref>6辆18米客车为左侧开门，实行先下后上，44辆12米长客车用于BRT支线，两侧均设有车门，方便在普通道路上停靠。<ref name="roadtransdept">{{cite web |url=http://www.gyyz.cn/content.aspx?id=1166 |title=贵阳BRT本月开通，上19层楼高站台、2小时无限换乘…重点都在这！ |publisher=贵阳市道路运输管理局 |date=2017-01-13 }}</ref>
==设施==
贵阳BRT系统在中环路设置29公里长的专用道，社会车辆只可借道通行，长期占用将被罚款50元，BRT车辆和专用道沿线设有[[闭路电视|闭路电视]]抓拍。但高峰期车流量高时仍有不少社会车辆占用，系统启用半年后BRT车站范围内已发生97起社会车辆导致的交通事故，直接经济损失达百万余元。<ref name="roadtransdept"/><ref>{{cite web |url=http://gz.sina.com.cn/news/sh/2017-07-08/detail-ifyhwefp0265686.shtml |title=贵阳：占用BRT道28000辆车被抓拍 司机将被罚款50元 |publisher=新浪 |author=贵州都市报 |date=2017-07-08 }}</ref><ref>{{cite web |url=http://www.gog.cn/zonghe/system/2017/06/29/015843417.shtml |title=因社会车辆占用BRT道 半年内发生事故97起损失高达百万余元 |publisher=多彩贵州网 |date=2017-06-29 }}</ref>

中环路沿线设26个BRT站点及3个公交枢纽站。车站设在中环路中央或隧道内的专用车道，通过人行天桥或隧道连接普通道路，其中黔春站站台离地面58米，乘客需转乘3次扶手电梯才能到达站台。枢纽站为科技学院、新寨路和黔春路公交枢纽。此外各主线有部分车站设于普通道路沿线。<ref name="roadtransdept"/><ref name="rmw2017"/><ref name="brtgarage"/>

贵阳BRT设4个停保场：贵阳北站停保场、中天一号停保场、贵钢停保场、杨柳井停保场。<ref name="brtgarage">{{cite web |url=https://gy.house.qq.com/a/20160630/014849_1.htm?qqcom_pgv_from=aio |title=解读贵阳1.5环BRT 各路段都经过哪？ |publisher=腾讯网 |author=多彩贵州网 |date=2016-06-30 }}</ref>
