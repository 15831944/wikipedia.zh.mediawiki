'''贵阳轨道交通'''，又称'''贵阳地铁'''，是中国[[贵州省|贵州省]][[贵阳市|贵阳市]]的城市轨道交通系统，由贵阳市城市轨道交通有限公司建设运营。首条线路于2017年12月28日试运营<ref name=":28">{{cite news|title=贵阳地铁今日13时启程|url=http://news.qx162.com/gz/2017/1228/211247.shtml|accessdate=2017-12-28 | publisher =黔讯网| date = 2017-12-28 | language = zh }}</ref>，并于2018年12月1日全线开通<ref name="181123gzdsw">{{cite news|title=贵阳，地铁来了！|url=http://news.gzdsw.com/gz/2018/1123/99960.html|accessdate=2018-11-23 | publisher =贵州都市网| date = 2018-11-23 | language = zh }}</ref>。另有一条线路在建中。

据2015年8月24日贵阳市政府批复的《贵阳市城市轨道交通线网规划修编》，贵阳地铁远期规划将增加到9条线路，总长约466.9公里，将设车站258座。<ref  name=":2">{{cite web |url=http://www.gyurt.com/index.php/article/cmobile/id/1169|title=贵阳市城市轨道交通远景年线网规划情况简介|author=|access-date=2017-07-05|date=2016-08-20|website=|publisher=贵阳市城市轨道交通有限公司|archiveurl=|archivedate=}}</ref>

== 建设历史 ==
===背景与前期规划===
贵阳市是[[中国西南地区|中国西南地区]]重要的交通枢纽和经济中心之一，2008年末全市总人口达到393.86万人。<ref>{{cite web |url=http://tjj.gygov.gov.cn/art/2009/4/9/art_27747_967568.html|title=2008年贵阳市国民经济和社会发展统计公报|author=|access-date=2017-07-06|date=2009-04-09|website=|publisher=贵阳市统计局|archiveurl=|archivedate=}}</ref>贵阳是山地城市，用地资源十分紧张，而随着贵阳城市化水平的提高和居民收入增加，贵阳[[机动车|机动车]]保有量快速增加，至2008年末已经突破38万辆，平均每10人拥有一辆机动车。<ref>{{cite web |url=http://gzdsb.gog.cn/system/2008/12/24/010438244.shtml|title=贵阳机动车保有量突破38万辆|author=廖秋|access-date=2017-07-06|date=2008-12-24|website=金黔在线|publisher=贵州都市报|archiveurl=|archivedate=}}</ref>由此导致贵阳市[[交通拥堵|交通拥堵]]问题严重。基于以上背景，同时按照2005年[[中华人民共和国国务院|国务院]]转发的《关于优先发展[[城市公共交通|城市公共交通]]的意见》，<ref>{{cite web |url=http://www.gov.cn/zwgk/2005-10/19/content_79810.htm|title=国务院办公厅转发建设部等部门关于优先发展城市公共交通意见的通知|author=|access-date=2017-07-06|date=2005-09-23|website=|publisher=国务院办公厅|archiveurl=|archivedate=}}</ref>以及中共贵阳市委2007年做出的“建设生态文明城市”的决定，<ref>{{cite web|url=http://www.gygov.gov.cn/art/2007/12/21/art_10683_273847.html|title=中共贵阳市委关于建设生态文明城市的决定|author=|access-date=2017-07-06|date=2007-12-21|website=|publisher=中共贵阳市委|via=贵阳市人民政府|archiveurl=|archivedate=}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>贵阳市开始规划建设[[城市轨道交通系统|城市轨道交通系统]]。<ref  name=":1">{{cite web |url=http://blog.sina.com.cn/s/blog_bce1d961010182ho.html|title=贵阳市城市快速轨道交通建设规划（2010-2020）环境影响报告书|via=贵阳地铁网官方新浪博客|author=|access-date=2017-07-06|date=2009-03|website=|publisher=中铁二院工程集团有限责任公司|archiveurl=|archivedate=}}</ref>

2008年贵阳市建设局委托[[中国城市规划设计研究院|中国城市规划设计研究院]]编制完成《贵阳市轨道交通线网规划》，<ref  name=":1"/><ref>{{cite web |url=http://www.gyfg.gov.cn/article-2-731.aspx|title=贵阳市建设局2008年工作安排|author=|access-date=2017-07-05|date=2008-02-05|website=|publisher=贵阳市住房和城乡建设局|archiveurl=|archivedate=}}</ref>规划四条地铁线路，连接金阳新区（现为[[观山湖区|观山湖区]]）、[[乌当区|乌当区]]、[[云岩区|云岩区]]、[[南明区|南明区]]及[[小河区|小河区]]（现已并入[[花溪区|花溪区]]），总长139.3公里，<ref  name=":1"/><ref>{{cite web |url=http://news.qq.com/a/20080719/001352.htm|title=贵阳拟年内开建城市轨道交通，向社会征意见|author=帅弋|access-date=2017-07-06|date=2008-07-19|website=|publisher=腾讯网|archiveurl=|archivedate=}}</ref>贵阳市政府同年批复通过该规划。随后，贵阳市建设局再次委托中国城市规划设计研究院于2009年3月编制完成《贵阳市城市快速轨道交通建设规划（2010-2020）》，规划两条近期轨道交通建设任务（[[贵阳轨道交通1号线|1号线]]、[[贵阳轨道交通2号线|2号线]]），总长度55.95km，并于2009年9月编制环境影响报告书，通过[[中华人民共和国环境保护部|国家环境保护部]]评审。<ref  name=":1"/><ref>{{cite web|url=http://www.gygov.gov.cn/art/2009/9/9/art_10682_267014.html|title=快速轨道交通规划环境影响报告书通过国家评审|author=|access-date=2017-07-05|date=2009-09-09|website=|publisher=贵阳市人民政府|archiveurl=|archivedate=}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

2010年9月3日，经国务院批准，[[国家发改委|国家发改委]]正式批复《贵阳市城市快速轨道交通建设规划（2010-2020）》，贵阳市正式启动城市快速轨道交通项目建设。<ref>{{cite web |url=http://xxgk.gygov.gov.cn/xxgk/jcms_files/jcms1/web1/site/art/2010/9/15/art_163_61350.html|title=贵阳市正式启动城市快速轨道交通项目建设 |author=|access-date=2017-07-05|date=2010-09-15|website=|publisher=贵阳市人民政府|archiveurl=|archivedate=}}</ref>

===一期建设===
2013年4月，国家发改委批准了《贵阳市轨道交通1号线可行性研究报告》。<ref>{{cite web |url=http://www.ndrc.gov.cn/fzgggz/nyjt/zhdt/201305/t20130523_542246.html|title=贵阳市轨道交通1号线可行性研究报告通过批准|author=|access-date=2017-07-05|date=2013-04-23|website=|publisher=国家发改委|archiveurl=|archivedate=}}</ref>同年9月[[贵阳轨道交通1号线|贵阳轨道交通1号线]]开工建设，10月全面开工。<ref>{{cite news|author=刘文强|coauthors=|url=http://epaper.gywb.cn/gyrb/page/1/2013-09/24/A01/20130924A01_pdf.pdf|title=举全市之力打响攻坚战，迎接贵阳“轻轨时代”到来|publisher=贵阳日报|pages=A01|date=2013-09-24|accessdate=2017-07-05}}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite book|url=http://www.metroreport.cn/upfile/2015/05/26/20150526144916_796.pdf|author=中国城市轨道交通年度报告课题组|title=中国城市轨道交通年度报告2013|date=|publisher=北京交通大学出版社|year=2014|isbn=9787512119895|location=|pages=130|archiveurl=https://web.archive.org/web/20160211084729/http://metroreport.cn/upfile/2015/05/26/20150526144916_796.pdf|archivedate=2016-02-11}}</ref>2015年9月28日，[[贵阳轨道交通2号线|轨道交通2号线]]一期工程全面开工。<ref>{{cite news|author=路榕|coauthors=|url=http://epaper.gywb.cn/gyrb/page/1/2015-09/29/A01/20150929A01_pdf.pdf|title=贵阳市轨道交通2号线一期工程全面开工|publisher=贵阳日报|pages=A01|date=2015-09-29|accessdate=2017-07-06}}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2017年6月28日，[[贵阳轨道交通1号线|贵阳轨道交通1号线]][[观山湖区|观山湖]]段（[[下麦西站|下麦西站]]至[[贵阳北站_(地铁)|贵阳北站]]）12.9公里建设完成，开始试运行，2017年12月28日正式运营。<ref name=":3">{{cite web |url=http://www.gyurt.com/index.php/article/content/id/1797|title=贵阳轨道交通1号线观山湖段进入试运行|author=孟又新、廖行、吴昊|access-date=2017-07-06|date=2017-06-28|website=|publisher=贵阳市城市轨道交通有限公司|archiveurl=|archivedate=}}</ref><ref name=":28"/>2018年12月1日，1号线后通段开通<ref name="181123gzdsw"/>。

===二期规划与建设===
2014年1月，[[中华人民共和国国务院|国务院]]批准[[贵州省|贵州省]]设立[[贵安新区|贵安新区]]。早在2013年下半年，贵阳城市轨道交通有限公司和贵州贵安建设投资有限公司委托[[中国地铁工程咨询有限责任公司|中国地铁工程咨询有限责任公司]]编制《贵阳市城市轨道交通线网规划修编》及《贵阳市城市轨道交通建设规划（2016-2022）》，《贵阳市城市轨道交通线网规划修编》在原来的远期规划基础上增加四条连接贵阳市和周边地区的市域快线（S1线、S2线、S3线、S4线）和连接贵安新区内部的G1线，总长增至466.9公里。而《贵阳市城市轨道交通建设规划（2016-2022）》所规划的近期建设为延伸[[贵阳轨道交通2号线|2号线]]，新建[[贵阳轨道交通3号线|3号线]]、S1线和S2线，总长增至176公里。<ref  name=":2"/><ref>{{cite web |url=http://www.gyurt.com/index.php/article/content/id/495|title=贵阳市城市轨道交通建设规划（2016-2022）环境影响评价第二次公示（简本）|author=|access-date=2017-07-06|date=2015-09-02|website=|publisher=中铁二院工程集团有限责任公司|via=贵阳市城市轨道交通有限公司|archiveurl=|archivedate=}}</ref>

2015年8月24日贵阳市政府批复《贵阳市城市轨道交通线网规划修编》，<ref  name=":2"/>2016年7月11日[[国家发改委|国家发改委]]批复《贵阳市城市轨道交通建设规划（2016-2022）》，贵阳城市轨道交通第二期建设规划得到批准。<ref>{{cite web |url=http://www.ndrc.gov.cn/zcfb/zcfbtz/201607/t20160715_811366.html|title=国家发展改革委关于贵阳市城市轨道交通第二期建设规划（2016～2022年）的批复|author=|access-date=2017-07-06|date=2016-07-11|website=|publisher=国家发改委|archiveurl=|archivedate=}}</ref>

2017年2月28日，2号线二期工程正式开工，预计2020年投入试运营。<ref name=":6">{{cite web |url=http://www.gyurt.com/index.php/article/content/id/1543|title=贵阳轨道交通2号线二期工程开工建设|author=孟又新、廖行、吴昊|access-date=2017-07-07|date=2017-02-28|website=|publisher=贵阳城市轨道交通有限公司|archiveurl=|archivedate=}}</ref>

==设施==
===车辆===
贵阳市轨道交通1、2号线采用B型车，6辆编组。1号线车辆由[[中车南京浦镇车辆|中车南京浦镇车辆有限公司]]制造，最高运行速度为80公里每小时，最大载客量2062人，计划配置34列车。<ref>{{cite web |url=http://www.gyurt.com/index.php/article/content/id/1319|title=首列地铁电客车接入，贵阳轨道交通1号线开通试运行进入倒计时|author=孟又新、廖行|access-date=2017-07-07|date=2016-10-26|website=|publisher=贵阳城市轨道交通有限公司|archiveurl=|archivedate=}}</ref>
