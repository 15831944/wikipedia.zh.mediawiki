'''贵广客运专线'''，又称'''贵广高铁'''、'''贵广铁路'''，是[[中國|中國]]一條连接[[贵州省|贵州省]][[贵阳市|贵阳市]]與[[廣東省|廣東省]][[廣州市|廣州市]]的铁路[[客运专线|客运专线]]，起自[[贵州|贵州]][[贵阳|贵阳]]，经[[广西|广西]][[桂林|桂林]]、[[贺州|贺州]]至[[廣東|廣東]][[廣州|廣州]]。设计线路长度857公里，广东境内207.5公里、广西境内348.5公里，贵州境内301公里。设计运营速度250km/h<ref>{{cite web | url=http://news.21cn.com/hot/gd/a/2014/0409/08/26924790.shtml | title=贵广高铁8月起调试全程4小时 年底望通车 | deadurl=yes | archiveurl=https://web.archive.org/web/20170422204158/http://news.21cn.com/hot/gd/a/2014/0409/08/26924790.shtml | archivedate=2017年4月22日 | df= }}</ref>，預留300km/h提速條件，最终每日运行列车150对左右<ref>{{cite web | url=http://gzdaily.dayoo.com/html/2010-06/03/content_984504.htm | title=贵广铁路或延至2013年通车 | deadurl=yes | archiveurl=https://web.archive.org/web/20160304072030/http://gzdaily.dayoo.com/html/2010-06/03/content_984504.htm | archivedate=2016-03-04 }}</ref>。该铁路于2008年10月13日开工建设，工程投资858亿元，2014年12月26日建成通车。

贵广高铁极大缩短了中国西南地区与珠三角地区的时空距离，是贵阳、成都、重庆、昆明四省省会到达广州的最快铁路线。贵广高铁走线非常直接，甚至比飞机航线里程还要短15公里<ref>{{cite web | url=https://www.youtube.com/watch?v=d1qbBxM-zas | title=贵广高铁的快与直}}</ref>。相较于普速列车需经沪昆线、京广线绕行湖南株洲，或经过[[黔桂铁路|黔桂铁路]]、[[湘桂铁路|湘桂铁路]]、[[黎湛铁路|黎湛铁路]]、[[广茂铁路|广茂铁路]]绕行广西、广东南部，贵广高铁里程比既有普速铁路大大缩短，故而相对比較便宜及快捷，性价比提高数倍，截止2017年年底，贵广高铁仍是中国大陆性价比最高的高速铁路<ref>{{cite web | url=http://www.iceo.com.cn/com2013/2014/1104/298065.shtml|title=高铁性价比}}</ref>。2018年初[[渝贵铁路|渝贵铁路]]开通、2019年[[成贵客运专线|成贵客运专线]]开通后，贵广高铁将承载整个大西南地区到珠三角的龐大客流。

==历史==
[[File:Train_line_at_Xingping_Reservoir.JPG|thumb]]
*2007年5月18日：国家发改委于2007年5月18日同意新建贵广铁路<ref>{{cite web | url=http://gd.news.163.com/08/0328/09/48457MV10036000Q.html | title=贵广铁路佛山设两站 拟今年8月动工2012年通车 | deadurl=yes | archiveurl=https://web.archive.org/web/20140811033353/http://gd.news.163.com/08/0328/09/48457MV10036000Q.html | archivedate=2014-08-11 }}</ref>。
*2008年10月13日：贵广铁路正式动工，规划输送能力为每日客车100对，每年货运2500万吨<ref>{{cite web | url=http://politics.people.com.cn/GB/14562/8171006.html | title=贵广铁路开工 广州坐火车去桂林只用2小时}}</ref><ref>{{cite web | url=http://www.gd.gov.cn/govpub/zdxm/zdxmjs/200810/t20081013_70475.htm | title=国家发展改革委关于新建贵阳至广州铁路可行性研究报告的批复}}</ref>。
*2010年4月：贵广铁路确认升级为客运专线，设计运行初期由原来的100对以内提高至103至106对，远期将达到147至152对，并取消货运设施<ref>{{cite web | url=http://gcontent.oeeee.com/f/aa/faafda66202d2344/Blog/a2d/344965.html | title=贵广铁路设计时速提高到300公里 广州到贵阳将只需3小时 | deadurl=yes | archiveurl=https://web.archive.org/web/20140809145217/http://gcontent.oeeee.com/f/aa/faafda66202d2344/Blog/a2d/344965.html | archivedate=2014-08-09 }}</ref><ref>{{cite web | url=http://gzdaily.dayoo.com/html/2010-06/03/content_984504.htm | title=贵广铁路或延至2013年通车 | deadurl=yes | archiveurl=https://web.archive.org/web/20160304072030/http://gzdaily.dayoo.com/html/2010-06/03/content_984504.htm | archivedate=2016-03-04 }}</ref>。
*2014年7月31日：贵州至广西段全线贯通<ref>{{cite web | url=http://www.gov.cn/xinwen/2014-08/01/content_2728192.htm | title=贵广高铁贵州至广西段全线贯通 最快年底可通车}}</ref>。
*2014年11月25日：贵广高铁预计在2014年12月26日全线通车，广州至贵阳全程约需4小时52分钟<ref>{{cite web | url=http://gd.qq.com/a/20141125/007886.htm | title=贵广高铁拟12月26日开通 凭车票半价游贵州}}</ref>。
*2014年12月6日：贵广高铁全线空载试运行，沿途车站不停车。
*2014年12月26日：与[[南广铁路|南广铁路]]广东段同期通车。其中最快的D211次列车，贵阳至广州南站只需4小时09分。<ref>{{cite web | url=http://crh.gaotie.cn/guiguang/2014-12-25/206998.html | title=贵阳至广州只需4小时09分}}</ref>桂林西、五通、阳朔、佛山西暂不办理客运业务<ref>[http://guilin.house.sina.com.cn/news/2014-12-25/09435953696191819871334.shtml 贵广高铁开通初期桂林西五通阳朔三站暂不办理客运业务]</ref>。
*2015年9月20日，龙里北至贵阳东段建成，贵阳至广州的动车组线路改为贵阳北—贵阳东—龙洞堡—龙里北，此前的贵阳北—贵阳—龙里北既有线走法停用。<ref>{{cite web | url=http://www.gywb.cn/content/2015-09/21/content_3850665.htm | title=龙洞堡高铁站启用}}</ref>
*2016年1月10日，桂林西站启用客运<ref>[http://ngzb.gxnews.com.cn/html/2016-01/11/content_1221480.htm 欣喜 昨日桂林西站迎送客 郁闷 公交车接驳赶不上趟] {{webarchive|url=https://web.archive.org/web/20160201190727/http://ngzb.gxnews.com.cn/html/2016-01/11/content_1221480.htm |date=2016-02-01 }}</ref><ref>[http://pic.gxnews.com.cn/staticpages/20160111/newgx5692db2b-14247091.shtml 高铁桂林西站开通运营 桂林高铁枢纽“三足鼎立”]</ref>。
*2016年1月24日，阳朔站启用客运<ref>[http://news.gxnews.com.cn/staticpages/20160124/newgx56a4c646-14329888.shtml 贵广高铁阳朔站今日开通 三地驴友不再受转车之苦]</ref>。
*2017年8月18日，佛山西站启用客运。
*2017年11月2日，贵阳东站启用客运。(仅[[贵开城际铁路|贵开城际铁路]]方向)<ref>{{cite web|author1=贵阳网数字报|title=贵阳东站正式投用|url=http://epaper.gywb.cn/gyrb/html/2017-11/03/content_513170.htm|access-date=2017-11-03|archive-url=https://web.archive.org/web/20171107025636/http://epaper.gywb.cn/gyrb/html/2017-11/03/content_513170.htm|archive-date=2017-11-07|dead-url=yes}}</ref>
*2018年3月13日，五通站启用客运<ref>{{cite news|title=广西桂林第九座高铁站五通站今日开站运营|url=http://gx.sina.com.cn/news/gx/2018-03-13/detail-ifyscsmu9453026.shtml|publisher=新浪网|date=2018-03-13}}</ref>。至此贵广高铁全线所有车站均已启用。
*2018年7月5日起，贵广高铁二等座除少部分车次外票价全面上涨，全程票价由267.5元涨至323元，二等座每公里票价由0.31元上涨至0.37元。

==灾害事故==
2017年10月，贵广高铁[[桂林西站|桂林西站]]K415+412.702～K415+692.702(DK413+385～DK413+665)段路基出现下沉，致使线路出现不均匀变形，影响行车安全，桂林西站正线从10月起封闭并拆除（侧线继续使用）。
