'''贵州省高速公路'''是[[中国国家高速公路网|中国国家高速公路网]]和[[贵州省|贵州省]]骨架公路网的重要组成部分。

==沿革==
1986年8月15日，贵州省第一条准高速公路——全长137公里的[[贵黄公路|贵黄高等级公路]]开工建设，历时5年，于1991年5月16日建成通车，它连接[[省会|省会]][[贵阳|贵阳]]与[[黄果树瀑布|黄果树瀑布]]景区，是[[中国西部地区|中国西部地区]]最早开工建设的一条高等级公路，同时也成为贵州省高速公路建设的开端<ref name="历史">{{cite web|url=http://www.gzqss.cn/e/gzsb/html/2015-01/05/content_188351.htm?div=-1|title=四千公里，贵州新起点|author=邹敏|date=2015年1月5日|publisher=贵州商报|access-date=2015-05-12|archive-url=https://web.archive.org/web/20150518113636/http://www.gzqss.cn/e/gzsb/html/2015-01/05/content_188351.htm?div=-1|archive-date=2015-05-18|dead-url=yes}}</ref>。

2010年底，贵州省高速公路总里程达到1507公里<ref name="十二五">{{cite web|url=http://www.qjt.gov.cn/gzsjtyst/web2012/common/xxgk_detail.jsp?s=9&ID=1292&g=9|title=贵州省公路水路交通运输“十二五”发展规划|date=2012年9月20日|publisher=贵州省交通运输厅|deadurl=yes|archiveurl=https://web.archive.org/web/20160304083216/http://www.qjt.gov.cn/gzsjtyst/web2012/common/xxgk_detail.jsp?s=9&ID=1292&g=9|archivedate=2016年3月4日}}</ref>。2011年，全省新增高速公路516公里，总通车里程达到2023公里；同时，全省开通高速公路的[[县|县]]新增10个，达到47个<ref>{{cite web|url=http://news.xinhuanet.com/fortune/2012-01/23/c_111458637.htm|title=贵州省高速公路通车里程超过2000公里|author=李劲峰 王橙澄|date=2012年1月23日|publisher=新华网}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2013年，新增高速公路651公里，全省高速公路通车里程达到3281公里；并新增8个县开通高速公路<ref>{{cite web|url=http://gz.people.com.cn/n/2014/0114/c194827-20378723.html|title=贵州省高速公路通车里程达3281公里|date=2014年1月14日|publisher=贵州日报}}</ref>。截至2014年底，贵州省当年新增高速公路731公里，全省高速公路通车总里程突破4000公里，达到4012公里；新增11个县开通高速公路，全省开通高速公路的县增加到79个<ref name="yukai">{{cite web|url=http://news.gog.com.cn/system/2014/12/30/014016184.shtml|title=贵州省高速公路通车里程达4012公里|author=贺春雨 聂毅 赵惠|date=2014年12月30日|publisher=多彩贵州网|deadurl=yes|archiveurl=https://web.archive.org/web/20150101183711/http://news.gog.com.cn/system/2014/12/30/014016184.shtml|archivedate=2015年1月1日}}</ref>。

==布局规划==
为了优化贵州省境内的[[中华人民共和国国道|国道主干线]]、国家重点[[公路|公路]]和[[国家高速公路|国家高速公路]]与其它主要公路之间的衔接，指导干线公路建设，[[贵州省人民政府|贵州省人民政府]]于2005年批准了由贵州省交通运输厅编制的《贵州省骨架公路网规划（2003～2020）》，以全省所辖[[地区行政公署|地]]（[[自治州|州]]、[[地级市|市]]）、县（[[县级市|市]]）[[政府驻地|政府驻地]]为节点，连接各地（州、市）、县（市）政府驻地、区域经济中心、重要[[交通枢纽|交通枢纽]]以及[[军事战略|军事战略]]要地、主要工[[农业|农业]]生产基地、[[能源|能源]]工业基地、主要[[风景名胜区|旅游风景名胜区]]、[[航空港|航空港]]、[[铁路|铁路]]枢纽，与邻[[省_(行政区划)|省]]（[[自治区|区]]、[[直辖市|市]]）的干线公路顺畅连接，使贵州省公路网融入整个大区域路网中，起到连接东西、沟通南北的作用，利用贵州的区位优势，确立在[[中国西南地区|西南地区]]的交通枢纽地位。该规划确定的骨架公路网采用纵线、横线和联线、支线网格相结合的布局形态，构成由纵贯南北、横连东西的公路交通大通道，包括3条南北纵向线、3条东西横向线和8条联线、8条支线，即“三纵三横八联八支”，简称为“3388网”，总规模约7400公里（扣除重复里程），其中高速公路约2960公里<ref name="骨架">{{cite web|url=http://www.qjt.gov.cn/gzsjtyst/web2012/common/xxgk_detail.jsp?s=9&ID=693&g=9|title=贵州省骨架公路网规划|date=2007年3月1日|publisher=贵州省交通运输厅|deadurl=yes|archiveurl=https://web.archive.org/web/20150518080149/http://www.qjt.gov.cn/gzsjtyst/web2012/common/xxgk_detail.jsp?s=9&ID=693&g=9|archivedate=2015年5月18日}}</ref>。

2008年，贵州省交通运输厅对《贵州省骨架公路网规划》进行修编，并形成了《贵州省高速公路网规划》，该规划经反复[[论证|论证]]和修改完善，于2009年2月获贵州省人民政府批准。根据规划，贵州省高速公路网规划布局为“6横7纵8联”以及4个城市环线，简称“678网”，总规模达到6851公里，其中国家高速公路2251公里，省级高速公路4600公里，到2020年前将基本形成全省高速公路网框架，实现与邻省间至少有一条高速公路通道、[[省会|省会]][[贵阳|贵阳]]与市州地所在地及相邻市州地之间有高速公路连接、县县通高速公路等目标。按2008～2020年、2021～2030年两个阶段分期实施<ref name="高速">{{cite web|url=http://www.qjt.gov.cn/gzsjtyst/web2012/common/detail.jsp?s=51&ID=3|title=贵州省高速公路网规划图|date=2009年8月4日|publisher=贵州省交通运输厅|deadurl=yes|archiveurl=https://web.archive.org/web/20150402103109/http://www.qjt.gov.cn/gzsjtyst/web2012/common/detail.jsp?s=51&ID=3|archivedate=2015年4月2日}}</ref>。

2013年，根据[[中国国务院|中国国务院]]批准的《国家公路网规划（2013年～2030年）》，贵州省约有1400公里的省级高速公路升格为国家高速公路，使全省境内的国家高速规划总里程增加到3600公里，加上省级高速3620公里，全省高速公路规划总里程超过7200公里<ref name="升格">{{cite web|url=http://jt.rednet.cn/c/2013/06/25/3053744.htm|title=贵州高速公路规划总里程达7200余公里|author=姜晓琨 姜明明|date=2013年6月25日|publisher=贵阳日报}}</ref>。

2014年11月25日，贵州省启动了“环贵高速公路”计划，总投资约1978亿元，建成总里程达1952公里（其中895公里已建成），覆盖全省7个市（州）、35个县（市、区）、340个[[乡镇|乡镇]]，最终建成一条沿边环绕贵州全省的高速公路大环线<ref name="历史" /><ref>{{cite web|url=http://difang.gmw.cn/gz/2014-11/26/content_13973876.htm|title=贵州启动“环贵高速公路”修1952公里的高速路|author=陈屹|date=2014年11月26日|publisher=央广网}}</ref>。

2016年10月，贵州省人民政府批复同意由贵州省交通运输厅报请审批的《贵州省高速公路网规划 （加密规划）》，目标是“[[十三五|十三五]]”末或“[[十四五|十四五]]”初实现全省高速公路通车里程1万公里，规划新增高速公路32条<ref>{{cite web |title=图解：《贵州省高速公路网规划（加密规划）》 |url=http://www.gzgov.gov.cn/xxgk/jdhy/zcjd_8115/tjzc/201709/t20170926_991539.html |publisher=贵州省人民政府 |date=2016-10-28}}</ref>。同时印发《贵州省高速公路建设攻坚决战实施方案》进一步明确了建设任务、建设模式及保障措施等<ref>{{cite web |title=省政府办公厅印发贵州省高速公路建设攻坚决战实施方案 |url=http://www.gzgov.gov.cn/xwzx/djfb/201610/t20161025_652253.html |publisher=贵州省人民政府 |date=2016-10-25}}</ref>，提出了53个项目共计4066公里的实施计划，其中优先实施国家和“678”网省级高速公路项目21个1638公里、加快实施新增的省际通道、市（州）绕城和放射线、重要通道扩容项目10个772公里、积极实施其他地方高速公路项目22个1656公里<ref name="youhua">{{cite web |title=《贵州省高速公路网规划（加密规划）》优化调整研究环境影响评价公示 |url=http://www.qjt.gov.cn/xwzx/tzgg/201806/t20180605_2448415.html |publisher=贵州省交通运输厅 |date=2018-06-05}}</ref>。

2018年6月，贵州省交通运输厅公布《贵州省高速公路网规划（加密规划）》优化调整研究，在维持原规划路线布局方案不变的基础上，新增补充路线方案以及对待建高速公路进行合理排序。优化调整后的高速公路加密规划，包含都市圈环线、提升通道能力、强化路网衔接和新增省际出口四个方面，共计新增高速公路17条，总里程855公里<ref name="youhua" />。

==贵州境内国家高速公路==
根据2007年7月[[中华人民共和国交通运输部|中华人民共和国交通运输部]]颁布的《国家高速公路网命名和编号规则》，贵州省交通运输厅于2009年6月起对全省境内的国家高速公路进行了命名调整及标志更换。列入国家高速公路网的每一条高速公路，编号由[[字母|字母]]“G”加[[阿拉伯数字|阿拉伯数字]]组成，并按序列编号<ref>{{cite web|url=http://news.gog.com.cn/system/2009/06/18/010587344.shtml|title=贵州高速公路将全部进行命名调整及标志更换|author=吴蔚|date=2009年6月18日|publisher=金黔在线|deadurl=yes|archiveurl=https://web.archive.org/web/20130702215933/http://news.gog.com.cn/system/2009/06/18/010587344.shtml|archivedate=2013年7月2日}}</ref><ref>{{cite web|url=http://www.qjt.gov.cn/gzsjtyst/web2012/common/detail.jsp?s=1&ID=2117|title=国高网贵州段标志更换完毕|author=李黔刚 陈健蕾|date=2009年12月30日|publisher=中国交通报|deadurl=yes|archiveurl=https://web.archive.org/web/20160304091255/http://www.qjt.gov.cn/gzsjtyst/web2012/common/detail.jsp?s=1&ID=2117|archivedate=2016年3月4日}}</ref>。
