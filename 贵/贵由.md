'''貴由汗'''（{{mong|m=ᠭᠦᠶᠦᠭ <br>ᠬᠠᠭᠠᠨ|Ппп=Güyüg qaγan|n=古余克<ref>见于《[[蒙古秘史|蒙古秘史]]》第270等节，共十处。</ref>，又译库裕克<ref>续资治通鉴所译</ref>，|k=Гүюг хаан}}；{{bd|1206年|3月19日|1248年|4月20日|catIdx=G}}），[[蒙古帝国|大蒙古国]]第三任[[大汗|大汗]]，[[孛儿只斤氏|孛儿只斤氏]]，[[窩闊台|窩闊台]]長子，[[乃馬真后|乃馬真后]]所生，1246年8月24日—1248年4月20日在位，计2年。

[[至元_(忽必烈)|至元]]三年（1266年）十月，[[太庙|太庙]]建成，追尊[[庙号|庙号]]'''定宗'''，[[諡號|谥]]'''简平皇帝'''，在宗庙中列祭于第七室，排在忽必烈之父[[托雷|托雷]]后、忽必烈之兄[[蒙哥|蒙哥]]前<ref>《元史》卷七十四 志第二十五，相关记载：至元三年，“冬十月，太庙成。丞相安童、伯颜言：“祖宗世数、尊谥庙号、配享功臣、增祀四世、各庙神主、七祀神位、法服祭器等事，皆宜以时定”。乃命平章政事赵璧等集议，制尊谥庙号，定为八室。烈祖神元皇帝、皇曾祖妣宣懿皇后第一室，太祖圣武皇帝、皇祖妣光献皇后第二室，太宗英文皇帝、皇伯妣昭慈皇后第三室，皇伯考术赤、皇伯妣别土出迷失第四室，皇伯考察合带、皇伯妣也速伦第五室，皇考睿宗景襄皇帝、皇妣庄圣皇后第六室，定宗简平皇帝、钦淑皇后第七室，宪宗桓肃皇帝、贞节皇后第八室。”节选自 [http://www.guoxue.com/shibu/24shi/yuanshi/yuas_074.htm 《元史·祭祀三·宗庙上》]</ref>。

== 生平 ==
{{main|蒙古西征}}
早年參加征伐[[金朝|金朝]]，俘虜了其親王；又曾經参与西征欧洲。蒙古帝国第三任大汗[[贵由|贵由]]、第四任大汗[[蒙哥|蒙哥]]，以及后来的[[元朝|元朝]]开国[[皇帝|皇帝]][[忽必烈|忽必烈]]，堂兄弟三人都是蒙古第二次西征时[[拔都|拔都]]的部下。

1241年12月11日，窝阔台去世，汗位虚悬，贵由的母亲[[乃马真后|乃马真脱列哥那]]称制，法纪混乱，很多宗王贵族滥发牌符征敛财物，唯有拖雷家族没有这样做，赢得了声誉。乃马真后欲立长子贵由为大汗，[[拔都|拔都]]与贵由不和，一直不肯参加选汗大会，后来，[[成吉思汗|成吉思汗]]幼弟[[铁木哥斡赤斤|铁木哥斡赤斤]]也领兵来争位，帝国面临汗位争夺战和混乱的危险。[[拖雷|拖雷]]的遗孀[[唆鲁禾帖尼|唆鲁禾帖尼]]决定率诸子参加[[忽里勒台|忽里勒台]]大会，1246年8月24日<ref>《元史》载“元定宗（指贵由汗）元年七月十二日”</ref>，宗王大臣们拥立贵由登基，贵由成为[[蒙古帝国|大蒙古国]][[大汗|大汗]]，“全体宗王们脱帽，解开宽腰带，把贵由扶上金王位，以汗号称呼他，到会者对新君九拜表示归顺，在帐外的藩王及外国使臣等也同时跪拜称贺。”<ref name="节选自法国勒内·格鲁塞《草原帝国》">节选自[法国]勒内·格鲁塞《草原帝国》</ref>

贵由登基后，虽然本人很有权威，但是因沉湎酒色、手足痉挛，并没有什么作为，且不理政事，多委于下臣<ref>《新元史·定宗本纪》：“帝严重有威，在位未久，不及设拖。昭慈皇后称制时，君权下替。帝既立，政柄复归于上。然好酒色，手足有拘挛疾，尝以疾不视事，委镇海、喀达克二人裁决焉。”</ref>。

1248年春，貴由親率大軍西征[[拔都|拔都]]，至橫相乙兒（今[[新疆|新疆]][[青河縣|青河縣]]東南）病死<ref>《蒙古秘史》，余大鈞譯，470頁</ref>。一說被拔都系势力毒殺。

1246年8月24日至1248年4月20日在位，在位仅一年零八个月。

==和罗马教宗的交往==
贵由在位期间和[[罗马教宗|罗马教宗]]有交往。歐洲諸國傳言蒙古大汗信仰基督教，因此教宗[[诺森四世|诺森四世]]派遣[[柏郎嘉宾|若望·柏郎嘉宾]]出使，希望勸說蒙古大汗不要傷害基督徒，同時要他深入了解蒙古人的風土民情、作戰方式等。1245年4月16日从法国[[里昂|里昂]]出发，途经神圣罗马帝国、波兰王国和基辅罗斯等国（他于1246年2月3日离开基辅）。1246年4月4日，他在[[伏尔加河|伏尔加河]]下游的萨莱（今伏尔加河下游阿斯特拉罕附近）受到钦察汗[[拔都|拔都]]的接见。拔都派他去蒙古草原见大汗，他经过[[讹答剌|讹答剌]]、伊犁河下游、叶密立河—翻越[[阿尔泰山|阿尔泰山]]，向东抵达蒙古草原。

1246年7月22日，他抵达距离[[哈拉和林|哈拉和林]]只有半天路程的地方，选举大汗的[[忽里勒台|忽里勒台]]大会正在此召开。他目睹了1246年8月24日贵由的当选，并留下了对贵由的生动描述：“在他当选时，约有四十，最多四十五岁。他是中等身材，非常聪明．极为精明，举止极为严肃庄重。从来没有看见他放声大笑，或者是寻欢作乐。”<ref name="节选自法国勒内·格鲁塞《草原帝国》"/> 最後他未能说服贵由皈依天主教，得到贵由的回信后，于1246年11月13日离开蒙古草原，向西踏上归途，经伏尔加河下游的拔都驻地返回西方，1247年9月5日他到达拔都驻地，又经[[基辅|基辅]]返回西方。

== 凉州会盟与吐蕃归附 ==

1247年，[[吐蕃|吐蕃]]诸部宗教界领袖[[萨迦班智达·贡嘎坚赞|萨迦班智达·贡嘎坚赞]]（简称[[萨班|萨班]]）同[[蒙古帝国|大蒙古国]]皇子[[西凉王|西凉王]][[阔端|阔端]]（贵由之弟，[[窝阔台|窝阔台]]之子，[[成吉思汗|成吉思汗]]之孙）在[[凉州|凉州]](今中国甘肃[[武威市|武威市]])议定了吐蕃归附的条件，其中包括呈献图册，交纳贡物，接受派官设治，[[吐蕃|吐蕃]]地区纳入[[蒙古帝国|大蒙古国]]([[蒙古帝国|蒙古帝国]])治下，史称“[[凉州会盟|凉州会盟]]”。<ref>[http://expo.people.com.cn/GB/18225089.html 人民网：“凉州会盟”卷轴唐卡亮相兰州 长达30米（图）]</ref><ref>[http://news.qq.com/a/20080424/002440.htm 腾讯网：还原"凉州会盟" 《月圆凉州》登陆央视六套]</ref><ref>[http://www.lzbs.com.cn/rb/2008-04/24/content_1450996.htm 兰州新闻网：《月圆凉州》今日亮相央视六套] {{webarchive|url=https://web.archive.org/web/20150719055416/http://www.lzbs.com.cn/rb/2008-04/24/content_1450996.htm |date=2015-07-19 }}</ref>

== 窝阔台家族的衰落 ==
[[Image:LetterGuyugToInnocence.jpg|thumb]]
根据《[[新元史|新元史]]》记载，1248年农历三月（1248年4月），贵由以养病为名带兵西巡，途中病逝于横相乙儿（今新疆青河东南），距離[[別失八里|別失八里]]一天路程<ref>《新元史·定宗本纪》：三年春三月，帝不豫，西巡叶密尔河。帝在潜藩，叶密尔河为汤沐地。帝尝谓此地水土宜于朕体，遂决意西巡。未至别失八里，疾大渐，崩于横相乙儿之地。年四十有三。</ref>。

贵由死后，其遗孀[[斡兀立海迷失|斡兀立海迷失]]临朝称制，由於贵由与[[拔都|拔都]]早年不和，拔都拒絕奔喪。为了对抗窝阔台家族，[[拔都|拔都]]以长支宗王的身份遣使邀请宗王、大臣到他在中亚草原的驻地召开忽里台，商议推举新大汗。[[窝阔台|窝阔台]]系和[[察合台|察合台]]系的宗王们多数拒绝前往，海迷失后只派大臣[[八剌|八剌]]为代表到会。[[唆鲁禾帖尼|唆鲁禾帖尼]]则命长子[[蒙哥|蒙哥]]率诸弟及家臣应召前往。

1250年，[[庫力臺大會|庫力臺大會]]在中亚地区拔都的驻地召开，拔都在会上极力称赞蒙哥能力出众，又有西征大功，应当即位，并指出贵由之立违背了[[窝阔台|窝阔台]]遗命（窝阔台遗命[[失烈门|失烈门]]即位），窝阔台后人无继承汗位的资格。大会通过了拔都的提议，推举[[蒙哥|蒙哥]]为大汗。窝阔台、察合台两家拒不承认，唆鲁禾帖尼和蒙哥又遣使邀集各支宗王到斡难河畔召开忽里台，拔都派其弟别儿哥率大军随同蒙哥前往斡难河畔，但窝阔台、察合台两家很多宗王仍不肯应召，大会拖延了很长时间。

由于蒙哥的母亲[[唆鲁禾帖尼|唆鲁禾帖尼]]的威望甚高，并且善于笼络宗王贵族，多数宗王大臣最终应召前来，1251年农历六月在蒙古草原[[斡难河|斡难河]]畔举行[[庫力臺大會|庫力臺大會]]，[[元宪宗|元宪宗]]元年农历六月十一日（1251年7月1日），宗王大臣们共同拥戴[[蒙哥|蒙哥]]即大汗位。此后，为了巩固汗位，唆鲁禾帖尼在镇压反对者时毫不留情，并亲自下令处死贵由的皇后斡兀立海迷失。

自此汗位繼承，便由窝阔台家族转移到了拖雷家族，皇族内部的分裂，为后来[[蒙古帝国|大蒙古国]]的彻底分裂，埋下伏筆。

== 評價 ==
* [[明朝|明朝]]官修[[正史|正史]]《[[元史|元史]]》[[宋濂|宋濂]]等的評價是：“三年戊申春三月，帝崩于横相乙儿之地。……是岁大旱，河水尽涸，野草自焚，牛马十死八九，人不聊生。诸王及各部又遣使于燕京迤南诸郡，征求货财、弓矢、鞍辔之物，或于西域回鹘索取珠玑，或于海东楼取鹰鹘，驲骑络绎，昼夜不绝，民力益困。然自壬寅以来，法度不一，内外离心，而太宗之政衰矣。”<ref>[http://www.guoxue.com/shibu/24shi/yuanshi/yuas_002.htm 《元史》卷二《定宗本紀》]</ref>

* [[清朝|清朝]]史学家[[毕沅|毕沅]]《[[续资治通鉴|续资治通鉴]]》的評價是：“自太宗皇后称制以来，法度不一，内外离心。至是国内大旱，河内尽涸，野草自焚，牛马死者十八九，人不聊生。诸王及各部，又遣使于诸郡征求货财，或于西蕃、回鹘索取珠玑，或于东海搜取鹰、鹘、驿骑络绎，昼夜不绝，民力益困。皇后立库春子实勒们听政，诸王大臣多不服。”<ref>[http://www.guoxue123.com/shibu/0101/01xzztj/171.htm 《续资治通鉴》卷一百七十二《宋纪一百七十二》]</ref>

* [[清朝|清朝]]史学家[[魏源_(清朝)|魏源]]《[[元史新编|元史新编]]》的評價是：“连岁大旱，河水尽涸，野草自焚，牛马十死八九，人不聊生。诸王及各部又遣使于燕京迤南诸郡，征求货财，或于西域、回鹘索取珠玑，或于海东搜取鹰鹘，驿骑不绝，内外离心，故无可纪。然自太祖崩后，拖雷监国者一年，太宗崩后，六皇后称制者四年，定宗之后，皇后临朝者又几四年，前后凡九载无君而国不乱，卒能创业垂统，上竝漢、唐者，则皆宗王宿将维持拱卫，根干蟠据之力。”<ref>《元史新编》卷三《定宗本纪》</ref>

* [[清朝|清朝]]史学家[[曾廉|曾廉]]《[[元书|元书]]》的評價是：“论曰：定宗之世，事多缺漏，而前史曰：‘ 帝崩之岁大旱，河水尽涸，野草自焚，牛马十死八九，人不聊生。诸王及各部又遣使于燕京迤南诸部，征求货财、弓矢、鞍辔，或于西域回鹘索取珠玑，海东索取鹰鹘，驿骑络绎，昼夜不绝，民力益困。然自壬寅以来，法度不一，内外离心，而太宗之政衰矣。’其言壬寅，盖以昭慈皇后称制时言之也。夫定宗即位时，年四十矣，而不能辑诸王侯大将，纪解威亵，此太宗之不谋付以匕图者乎？然在于汉亦孝惠之亚也。惟无良臣为之辅弼，而宗藩党羽遂成，以夺皇阼。炎异之丛，兴其足信耶？而失烈门则太宗遗诏所立也。前史复曰：定宗崩后，三岁无君。蒙哥之党之不欲以为君，非蒙古之无君也。窜之北陲，并逐太宗皇后而弑定宗皇后，可不谓之逆哉！自是而太宗子孙亦不欲以蒙哥兄弟为君，逮于海都，而中原震矣。”<ref>《元书》卷三《定宗本纪》</ref>

* [[中華民国|中華民国]]史学家[[屠寄|屠寄]]《[[蒙兀儿史记|蒙兀儿史记]]》的評價是：“汗严重有威，临御未久，不及设施，惟乃蛮真可敦称制时，威福下移，汗既亲政，纲纪粗立，君权复尊，自幼多疾，成吉思汗尝命亦鲁王之祖忽鲁扎克为之主膳。中年性好酒色，手足有拘挛之病，在位之日，常以疾不视事，事多决于大臣镇海、合答二人云。”<ref>《蒙兀儿史记》卷五《古余克汗本纪》</ref>

* [[中華民国|中華民国]]官修[[正史|正史]]《[[新元史|新元史]]》[[柯劭忞|柯劭忞]]的評價是：“史臣曰：定宗诛奥部拉合蛮，用镇海、耶律铸，赏罚之明，非太宗所及。又乃马真皇后之弊政，皆为帝所铲革。旧史不详考其事，谓前人之业自帝而衰，诬莫其矣。” <ref>
[http://www.guoxue123.com/shibu/0101/00xys/004.htm 《新元史》卷五《定宗本纪》]</ref>
