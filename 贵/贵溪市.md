'''贵溪市'''位于[[江西|江西]]省东北部，是[[中国|中国]]的[[铜|铜]]工业中心。目前贵溪市三面包围了[[鹰潭市|鹰潭市]]区，全市通行赣方言，并与[[福建|福建]]省相毗邻。境内有[[国家重点风景名胜区|国家重点风景名胜区]]、[[道教|道教]]仙山——[[龙虎山|龙虎山]]。

== 历史 ==
[[唐代|唐代]]置贵溪县，以县治周围溪水环绕，县以溪贵而得名。[[明朝|明朝]]屬[[廣信府|廣信府]]。

1957年鹰潭镇单独析出设县级鹰潭镇，1979年改县级[[鹰潭市|鹰潭市]]，1983年设地级[[鹰潭市|鹰潭市]]，贵溪县隶之。1996年贵溪县改为[[县级市|县级市]]，由地级鹰潭市代管。

== 贵溪市市徽 ==
图案简洁明了，由“贵溪”二字的汉语拼音首字母“GX”、运用艺术手段表现的一只鲲鹏和“CU”铜元素符号组成。市徽呈圆形，用红底色象征贵溪老区和红色土地，黄色表明贵溪的铜城特色，蓝色为信江河流。整个市徽的寓意是，座落在这块红色土地上的全国最大铜基地——贵溪，犹如鲲鹏展翅，搏击风浪，正朝着现代代的目标奋起腾飞。
贵溪市市徽设计者：张炳贤、张金华共同设计，张炳贤全国美术家协会会员、全国特级教师。张金华 “中国道教创始人、嗣汉张天师、张道陵第六十五代后裔”，道教祖庭天师府驻京办主任、中华天师道友会执行会长、世界易经风水研究院院长、贵溪撤县设市市徽设计者。天师府、龙虎山历史文化石碑文字的工作者、美术家协会会员、中华赣派风水研究院院长。

== 地理 ==
北为[[怀玉山|怀玉山]]，南为[[武夷山脉|武夷山脉]]，多[[山地|山地]]，建有中型[[水库|水库]]6座：[[塘湾水库|塘湾水库]]、[[硬石岭水库|硬石岭水库]]、[[太平阪水库|太平阪水库]]、[[枫林湾水库|枫林湾水库]]、[[大禾源水库|大禾源水库]]、[[白庙水库|白庙水库]]。

中部为[[信江|信江]]，多[[平原|平原]]和[[丘陵|丘陵]]，并有大片[[丹霞地貌|丹霞地貌]]分布，以西南部[[龙虎山|龙虎山]]周围面积最大。

== 交通 ==
[[File:201608_D333_at_Yingtanbei_Station.jpg|thumb]]]]
[[沪昆铁路|沪昆铁路]]、[[皖赣铁路|皖赣铁路]]在[[贵溪站|贵溪站]]交汇。同时该市管辖范围内还设有[[沪昆高速铁路|沪昆高速铁路]][[鹰潭北站|鹰潭北站]]。同时[[320国道|320国道]]在此交汇，交通便利。

== 经济 ==
[[File:201612_Guixi_Smelter_of_Jiangxi_Copper_Corporation.jpg|thumb]]贵溪冶炼厂]]
贵溪是中国最大、最重要的铜冶炼基地。[[德兴铜矿|德兴铜矿]]通过[[皖赣铁路|皖赣铁路]]、[[永平铜矿|永平铜矿]]通过[[沪昆铁路|沪昆铁路]]在此交汇，目前全國最大銅企業[[江西铜业股份|江西铜业]]的总部就设在贵溪。该公司拥有全国前五大铜矿，并建有规模位居中国第一、世界前十的贵溪冶炼厂。
