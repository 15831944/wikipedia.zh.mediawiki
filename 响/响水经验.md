'''响水经验'''，或称'''11·27响水经验'''，指2007年[[江苏省|江苏省]][[响水县|响水县]]宣传部门在当地化工企业发生事故后阻碍记者采访的一系列做法，源自《沉着应对突发事件 全力做好舆论引导──响水“11·27”事故新闻协调工作的主要做法<ref name="新语丝">{{Citeweb |title=沉着应对突发事件 全力做好舆论引导──响水“11·27”事故新闻协调工作的主要做法 |url=http://www.xys.org/xys/ebooks/others/science/dajia8/xiangshui.txt |publisher=新语丝 |accessdate=2019-03-24 |archiveurl=https://web.archive.org/web/20121106230839/http://xys.org/xys/ebooks/others/science/dajia8/xiangshui.txt |archivedate=2012-11-06 |dead-url=yes }}</ref>》一文（下文中略称《沉着》）<ref name="多维新闻网">{{cite web |title=舆论场：盐城爆炸撕开多少遮羞布 |url=http://news.dwnews.com/china/news/2019-03-22/60125003.html |website=多维新闻网 |accessdate=2019-03-23 |date=2019-03-22}}</ref>。

== 背景 ==
2007年11月27日，[[江苏省|江苏省]][[盐城市|盐城市]][[响水县|响水县]][[陈家港化工集中区|陈家港化工集中区]]联化科技有限公司爆炸事故，造成8人死亡<ref>{{cite web|url=http://www.gov.cn/yjgl/2007-11/28/content_818166.htm|title=江苏省响水县一化工厂发生爆炸事故 已致7人死亡|date=2007-11-28|website=[[中国政府网|中国政府网]]|archivedate=2012-08-03|archiveurl=https://archive.today/20120803133934/http://www.gov.cn/yjgl/2007-11/28/content_818166.htm|access-date=2019-03-23|dead-url=no}}</ref><ref>{{cite news|url=http://news.cctv.com/society/20071130/107726.shtml|author=刘兆权、邓华宁|website=[[中国中央电视台|中国中央电视台]]|date=2007-11-30|title=响水"1127"爆炸事故原因初步查明|archiveurl=https://archive.today/20190322055231/http://news.cctv.com/society/20071130/107726.shtml|archivedate=2019-03-22|access-date=2019-03-23|dead-url=no}}</ref>。爆炸后，众多媒体记者赶赴响水采访，据亲历者《[[中国青年报|中国青年报]]》记者李润文回忆：“事故发生后，盐城市立即启动了一套禁止记者采访的应急预案，不惜采用武力威胁、软禁记者，重金收买、色相利诱等方式收买记者，阻挠采访<ref name="金融界"/>。”

== 文章 ==
2007年底，《沉着》已受到广泛关注，该文署名为“中共响水县委宣传部”，日期为2007年12月24日<ref name="新语丝" /><ref name="新京报2">{{cite web |title=“防火、防盗、防记者”，厉害了“响水经验” |url=http://www.bjnews.com.cn/opinion/2019/03/22/559139.html |website=新京报 |accessdate=2019-03-23 |date=2019-03-22 |archive-url=https://web.archive.org/web/20190324011419/http://www.bjnews.com.cn/opinion/2019/03/22/559139.html |archive-date=2019-03-24 |dead-url=no }}</ref>，但其真伪一直未被证实。文中以官方口吻总结了2007年陈家港化工园联化科技有限公司发生爆炸后当地宣传部门阻碍记者采访的“成功经验”<ref name="金融界" />。

《[[中国青年报|中国青年报]]》记者[[李润文|李润文]]采写的《江苏盐城禁止采访，给记者找小姐》<ref>{{cite web|url=http://xys.org/xys/ebooks/others/science/dajia9/xiangshui2.txt|date=2008-01-02|title=江苏盐城禁止采访，给记者找小姐|author=李润文|work=新语丝（转载）|access-date=2019-03-24|archive-url=https://web.archive.org/web/20121107060409/http://xys.org/xys/ebooks/others/science/dajia9/xiangshui2.txt|archive-date=2012-11-07|dead-url=no}}</ref>因在细节方面与《沉着》互为印证，也被争相传阅。《[[第一财经日报|第一财经日报]]》记者于2011年2月致电李润文，“证实这篇文章的确是他所写”<ref>{{cite web|url=http://vnetcj.jrj.com.cn/2011/02/1501459187683.shtml|date=2011-02-15|work=第一财经日报|author=任绍敏|title=争议响水“舆论引导”经验}}</ref>。

== 评论 ==
2007年12月31日，《南方都市报》在A2版<ref>{{cite web|url=http://epaper.oeeee.com/A/html/2007-12/31/node_2732.htm|archiveurl=https://web.archive.org/web/20130118173308/http://epaper.oeeee.com/A/html/2007-12/31/node_2732.htm|deadurl=yes|title=南方都市报·数字报·完全版 2007年12月31日 (AA02)社论|work=南方都市报|date=2007-12-31|archivedate=2013-01-18}}</ref>发表评论文章《“响水经验”的危险信号》，称《沉着》一文所展现的“响水经验”表明“基层言论管制部门的成熟速度，已经大大超过了媒体拓展生存空间的速度”，媒体的生存空间正越来越狭窄<ref>{{cite web|url=http://www.nanfangdaily.com.cn/southnews/spqy/200712310186.asp|archiveurl=https://web.archive.org/web/20190324011413/http://fuchenlog.blogspot.com/2008/01/blog-post.html|title=“响水经验”的危险信号|author=黄鸣|work=南方都市报|date=2007-12-31|archivedate=2019-03-24}}</ref>。

2011年“响水经验”被再次提起时，上海市新文汇律师事务所主任律师富敏荣认为响水“11·27”事故中有组织地阻挠新闻采访事件，涉及记者的新闻采访权利问题，值得新闻界和法律界高度关注；[[复旦大学|复旦大学]]公共管理与国际事务学院教授唐贤兴认为，“防盗、防火、防记者”的做法说明部分地方政府的治理能力不足<ref name="金融界" />。

2019年“响水经验”被再次提起时，上观新闻发表评论文章称响水经验被旧事重提在一定程度上折射出人们对公共安全事件背后深层问题的担忧和焦虑。这种担忧和焦虑需要靠公权力部门坦诚、公开、务实的处理来逐渐打消。所谓“响水经验”恰应当被当作“教训”来深刻汲取<ref name="上观" />。《新京报》发表评论文章《“不防记者，专防爆炸”的“响水经验”又在哪里？》（后更名《“防火、防盗、防记者”，厉害了“响水经验”》并删减<ref name="新京报2" />），指出响水县相关部门在2007年的所作所为跟受法律与政策保障的舆论监督权相悖，也跟中央保障媒体采访权的明文要求不符，必将成为当地突发事件处置、舆情应对史上的一大污点<ref name="新京报1">{{cite web |author1=佘宗明 |title=“不防记者，专防爆炸”的“响水经验”又在哪里？ |url=http://news.ifeng.com/c/7lFHHcn0gHn |website=凤凰网-新京报 |accessdate=2019-03-23 |date=2019-03-22 |archive-url=https://web.archive.org/web/20190322120507/https://news.ifeng.com/c/7lFHHcn0gHn |archive-date=2019-03-22 |dead-url=no }}稿件之后被删除，然后更名为《“防火、防盗、防记者”，厉害了“响水经验”》。删除撤稿之前的稿件备份[https://web.archive.org/web/20190323175310/https://webcache.googleusercontent.com/search?q=cache%3AZ-BS0eO116QJ%3Awww.bjnews.com.cn%2Fopinion%2F2019%2F03%2F22%2F558983.html+&cd=2&hl=en&ct=clnk&gl=us 在此]</ref>。
