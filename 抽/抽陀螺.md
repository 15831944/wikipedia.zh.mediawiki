'''抽陀螺'''，也叫'''鞭陀羅'''、'''打猴'''，是一項讓陀螺在地上轉動的[[民俗體育|民俗體育]]運動，是老人、小孩都適合的運動。

==沿革==
 
陀螺歷史古老，在山西夏縣境西陰村[[仰韶文化|仰韶文化]]遺址內，曾發現一只陶土製成的陀螺<ref name = ‘史話’>{{cite book | title = 中國歷代體育史話 | url = https://books.google.com/books?id=UcRIDwAAQBAJ&pg=PT152&dq=%E6%8A%BD%E9%99%80%E8%9E%BA&hl=en&sa=X&ved=0ahUKEwiUvIfo0N_cAhW0Ln0KHZ8PAd4Q6AEINjAC#v=onepage&q=%E6%8A%BD%E9%99%80%E8%9E%BA&f=false | editor = 從書編委會 | publisher = 崧博 大佳網 | isbn = 9787119064123 | year = 2018}}</ref>，是約四、五千前新石器時代的產物<ref name="打干樂">{{cite web |author1=魏小芸 |title=打陀螺（打干樂） |url=http://nrch.culture.tw/twpedia.aspx?id=11922 |publisher=臺灣大百科全書 文化部 |accessdate=2018-08-07}}</ref>。據文史記載，在宋朝時期已經出現了類似陀螺的玩具，叫「千千」或「千千車」<ref name = ‘史話’/>。千千是一個長約一寸的針形物體，放在盘中用手擰著旋轉，比賽誰轉得久；這當時嬪妃宮女用來打發時間的遊戲。<ref>{{cite web |title=陀螺的起源及發展 |url=https://media.huayuworld.org/culture/chinese/cul_chculture/vod08html/vod08_09.htm |publisher=中華文化天地 |accessdate=2018-08-07 |archive-url=https://web.archive.org/web/20180704142115/http://media.huayuworld.org/culture/chinese/cul_chculture/vod08html/vod08_09.htm |archive-date=2018-07-04 |dead-url=yes }}</ref> [[臺灣|臺灣]][[國立故宮博物院|國立故宮博物院]]收藏的宋代蘇漢臣的《嬰戲圖》中畫有兩個兒童打陀螺玩耍，也證實了在[[宋朝|宋朝]]已出現了倒鍾形的陀螺 <ref name = ‘史話’/>。《通俗編》中說：「按宋時兒戲物，有千千，見武林舊事。…此皆陀螺類。」<ref>{{cite web |title=陀螺 |url=https://www.ncfta.gov.tw/imediainfo_274_1445.html |publisher=國立傳統藝術中心 |date=2017-11-28 |accessdate=2018-08-07 |ref=2017-11-28 |archive-url=https://web.archive.org/web/20180810175821/https://www.ncfta.gov.tw/imediainfo_274_1445.html |archive-date=2018-08-10 |dead-url=yes }}</ref>。

「陀螺」這個名詞最早出現在明朝劉侗、於弈正合著的《帝京景物略》中{{notetag| 《帝京景物略》：「楊柳兒青，放空鍾；楊柳兒活，抽陀螺；查柳兒死，踼毽子。」}}<ref name = ‘史話’/>，當時已是兒童的玩具，但陀螺是否由千千演變過來的就不得而知<ref>{{cite web |title=陀螺的起源及發展 |url=http://media.huayuworld.org/culture/chinese/cul_chculture/vod08html/vod08_09.htm |publisher=中華文化天地 |accessdate=2018-08-07}}</ref>
。明代的陀螺是木制，中間空心而無柄，用繩子繞住，然後用力一拉陀螺就會旋轉起來；用繩子不斷地抽打，陀螺就會一直旋轉。<ref name="教育大市集">{{cite web |title=抽陀螺 |url=https://market.cloud.edu.tw/content/local/tainan/kunhwa/tolo.htm#%E6%A8%99%E9%A1%8C |publisher=教育大市集 |accessdate=2018-08-07}}</ref>

抽陀螺在10世紀前就已傳至朝鮮、日本等國家<ref>{{cite book |editor=王曉梅 |title=一本书读懂中华民俗知识 |date=2010-06 |isbn=9787511703590 |accessdate=2018-08-07}}</ref>。

== 不同地區的稱呼 ==
不同地區對抽陀螺的稱呼亦有不同。抽陀螺在台灣話中被稱為「打干樂」<ref name="打干樂"/>，湖北稱「刷得螺」<ref name="雪花新聞">{{cite news |title=今年春節我們一起重溫舊時抽陀螺吧~ |url=https://www.xuehua.us/2018/07/03/%E4%BB%8A%E5%B9%B4%E6%98%A5%E8%8A%82%E6%88%91%E4%BB%AC%E4%B8%80%E8%B5%B7%E9%87%8D%E6%B8%A9%E6%97%A7%E6%97%B6%E6%8A%BD%E9%99%80%E8%9E%BA%E5%90%A7/zh-tw/ |accessdate=2018-08-07 |work=雪花新聞}}</ref>。

==原理==
抽陀螺的原理是當物體出現一種〞動者恆動、靜者恆靜〞的狀態時稱為〞平衡〞
，不只是在靜止的時候，動的時候也會達到平衡，有些平衡狀態能持久，有些只是短暫現象，當陀螺受力旋轉時，因各方向離心力總和達到平衡，陀螺能暫時用軸端站立，保持平衡；接著受到空氣阻力、地面摩擦或陀螺重心問題等多種因素之影響，讓旋轉的力道逐漸減弱，到旋轉的動力消失時，陀螺就搖晃倒下<ref>{{cite web |title=陀螺簡介 |url=http://www.wfps.tp.edu.tw/iweb/acrobatics/index.asp?m=5 |publisher=技知能工作坊 |accessdate=2018-08-07}}</ref>。

==好處==
抽陀螺除了有娛樂性，也可以促進血液循環，讓全身關節、肌肉皆得到鍛鍊<ref name="雪花新聞"/>。


==玩法==
抽陀螺玩法有二種：（一）水平抽法─彎身從身側把陀螺往前拋。（二）垂直抽法─陀螺舉高過頭，上往下，邊抽邊打。

抽陀螺比賽方式有二種：（一）分邊法─將參加的人分二組，大家一起抽陀螺，看那一組的陀螺先倒地，倒地的陀螺稱做「死陀螺」，任由對方宰割。輸的一方可以換一個硬而小的陀螺來代替，這時贏的一方，用自己的陀螺舉高對準目標向下猛擊。（二）畫圈法─在地上畫一個圈，圓圈中央，再畫一個小圓圈，各人輪流將自己的陀螺往圈內打，讓陀螺能旋轉出來，如陀螺已固定在一點上旋轉，此時，用繩子將它圈出來，只要到達圈外還在旋轉都不算死，如果陀螺停止在圈內，或一抽就不動都算死，死了要放在當中小圓圈內任別人處罰。若處罰別人的陀螺也停在圈內，一樣要放在小圓圈內任人處罰。如果沒有被擊倒，或被擊到而沒有被分解，就可以拿到一個陀螺，用水平抽法，將自己那個小圓圈內待死的陀螺擊出圈外<ref>{{cite web |title=陀螺的分類及玩法 |url=http://media.huayuworld.org/culture/chinese/cul_chculture/vod08html/vod08_11.htm |publisher=中華文化天地 |accessdate=2018-08-07}}</ref>。
