'''表演性選舉'''（{{lang-en|'''show election'''}}），又稱'''虛假選舉'''、'''橡皮圖章選舉'''，純屬儀式之選舉，無民主成份。

這種[[選舉|選舉]]通常出現在[[獨裁政體|獨裁政體]]、[[威權政體|威權政體]]和[[一黨制國家|一黨制國家]]，為了在形式上表示尊重憲法而定期舉行。通常有接近百分之百的投票率，候選人得票率達95%以上，投票結果一般有利於已掌政的當權者，整個選舉以延續或鞏固政權為目的。投票者如對某議題或某候選人投反對票或[[白票|白票]]會立即引起注意及政權的秋後算帳<ref>{{cite web |title= 第二次世界大战实录·战争背景篇 |url = https://books.google.com.hk/books?id=5lkRBwAAQBAJ&pg=PT106&lpg=PT106&dq=馬泰奧蒂&source=bl&ots=n4zkrAPaxi&sig=uwH4zEGFPxcU0WLTlcYHfr_4ixk&hl=zh-TW&sa=X&ved=0ahUKEwjz_4eS2IXQAhVLpJQKHXUHAz0Q6AEIlgEwGQ#v=onepage&q=%E9%A6%AC%E6%B3%B0%E5%A5%A7%E8%92%82&f=false |author= 马夫 |publisher = 南文博雅 |date = 2015-03-10 }}</ref>，因此經常出現壓倒性的勝利。

== 例子 ==
[[File:Wahlzettel-3.-Reich.jpg|thumb]]的選舉中，選票上可供選擇的只有[[阿道夫·希特拉|阿道夫·希特拉]]一位候選人。]]
歷史上較著名的表演性選舉發生在[[法西斯意大利|法西斯意大利]]於[[1929年意大利大選|1929年]]和[[1934年意大利大選|1934年的選舉]]，以及在[[1933年3月德国国会选举|1933年3月]]、[[1933年11月德国国会选举|11月]]、[[1936年德国国会选举|1936年]]及[[1938年德国国会选举|1938年]][[納粹德國|納粹德國]]的選舉中。在上述5次選舉裡，選民投票時分別只有兩個選項，要麽同意[[法西斯黨|法西斯黨]]及[[納粹黨|納粹黨]]當選，要麽反對，選票上並無其他候選人可供選擇。而獨裁的法西斯黨及納粹黨又以各種暴力與威嚇的手段，令選民在壓力下不得不投下贊成票，故該5場選舉的投票率均接近九成，而贊成票更超過九成，使有關黨派在門面上取得了壓倒性的大勝。

另外，大多數[[共產主義|共產主義]]國家亦充斥著表演性選舉，例子如[[蘇聯|蘇聯]]、[[東方集團|東方集團]]、[[中華人民共和國|中華人民共和國]]、[[朝鮮|朝鮮]]、[[越南|越南]]、[[古巴|古巴]]以及[[敘利亞|敘利亞]]、[[伊拉克復興黨政權|伊拉克復興黨政權]]、[[利比亞民眾國|利比亞民眾國]]、[[白俄羅斯|白俄羅斯]]、[[土庫曼斯坦|土庫曼斯坦]]等[[中亞|中亞]]國家、[[辛巴威|辛巴威]]、[[蘇丹共和國|蘇丹共和國]]、[[厄利垂亞|厄利垂亞]]、[[赤道幾內亞|赤道幾內亞]]、[[喀麥隆|喀麥隆]]、[[烏干達|烏干達]]、[[甘比亞|甘比亞]]等。在這些國家，選舉通常有預定的結果，投票過程只是一種表演及儀式。它們通過壓制反對派、以壓力脅迫選民，或舞弊、操縱偽造的選票（例如[[南韓|南韓]]{{link-en|第四屆總統選舉|South Korean presidential election, March 1960}}和[[十月維新|十月維新]]以後的總統選舉、[[越南共和國|南越]]的[[1955年越南公民投票|1955年越南公民投票]]、[[緬甸|緬甸]]及[[非洲|非洲]]、[[拉丁美洲|拉丁美洲]][[軍事專政|軍事專政]]下的各項選舉等），來製造一面倒的勝利。在一個極端的例子，[[利比里亞|利比里亞]]{{link-en|查爾斯D. B. King|Charles D. B. King}}聲稱，他在1927年的大選中獲得約243,000張選票，超過合資格選民人數15倍。
