== 動作 ==
'''表演（Performing）'''主要指表演者利用技藝或專長來傳達具體的事件或非具體的意象，以達到藝術或是[[娛樂|娛樂]]的目的。這裡所指的技藝或專長包含肢體動作（人或是動物）、聲音（如[[廣播_(大眾媒體)|廣播]]）等等。由於在強調傳達的動作，因此不侷限於任何特定的媒介或型式。


== 劇場部門 ==
{{SeeAlso|演技}}
'''表演（Acting）'''特指戲劇活動的一個部門、或戲劇演出中的表演動作，其擔任演出的人員稱為[[演員|演員]]。在現代，此部門不單止出現在劇場中，也出現在[[電視|電視]]、[[電影|電影]]、[[廣播|廣播]]中的戲劇活動；在這裡主要針對狹義的劇場表演說明。

== 藝術形式 ==
'''表演（Performance）'''指用來表现人的主观意识的一種艺术形式；舉凡[[戲劇|戲劇]]、[[舞蹈|舞蹈]]、歌唱、演奏、[[相聲|相聲]]、[[魔術|魔術]]、[[腹語術|腹語術]]......等，都屬於表演的範疇。這種藝術形式不限制發生的地點，但強調表演者與視聽者之間的直接接觸或互動，因此經過媒界連接表演者與視聽者的形式通常不被直接稱作表演，而常在表演一詞之前冠以媒介的名稱，如電視表演（TV Show）。
