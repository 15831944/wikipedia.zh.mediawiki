'''表演女郎'''（{{lang-en|Showgirl}}）是在舞台娛樂表演上展示自己身材的女性舞者或表演者，尤其是穿著暴露的衣服、甚至[[赤裸上身|赤裸上身]]或者全裸。

「表演女郎」這個術語有時也會指在展銷會和車展上的[[宣傳模特兒|宣傳模特兒]]。

== 歷史 ==
表演女郎可以追溯到1800年代晚期的巴黎[[音樂廳|音樂廳]]及[[卡巴萊|卡巴萊]]，例如[[紅磨坊|紅磨坊]]、[[麗都夜總會|麗都夜總會]]及[[女神遊樂廳|女神遊樂廳]]。<ref>{{Cite web |url = http://www.showgirlsinlasvegas.com/showgirl-history.htm |title = Las Vegas Showgirls: The History of Las Vegas Showgirls |publisher = showgirlsinlasvegas.com |language = en}}</ref>

== 拉斯維加斯表演女郎 ==
[[Image:JubileeDancersByPhilKonstantin.jpg|thumb]]
1950年代，拉斯維加斯開始出現表演女郎，主要為名人提供公開及閉門表演。她們會被介紹到{{link-en|金沙酒店與賭場|Sands Hotel and Casino|金沙賭場}}與{{link-en|Danny Thomas|Danny Thomas}}同場演出。1957年，Harold Minsky在{{link-en|沙漠旅館|Desert Inn}}造就了赤裸上身的表演女郎。隨後，{{link-en|Donn Arden|Donn Arden}}在{{link-en|星塵度假酒店與賭場|Stardust Resort and Casino|星塵賭場}}中長期用她們表演''The Lido de Paris''達31年。<ref>{{Cite web |url = http://www.pbs.org/wgbh/amex/lasvegas/peopleevents/p_entertainers.html |title = Las Vegas: An Unconventional History |date = 2005-07-11 |publisher = PBS |language = en}}</ref><ref>{{Cite web |url = http://lasvegassun.com/news/2008/may/15/showgirls-may-be-fewer-numbers-will-always-be-icon/ |title = Las Vegas Showgirls: Show and (a lot to) tell |author = Mary Manning |date = 2008-05-15 |publisher = Las Vegas Sun |language = en}}</ref>
