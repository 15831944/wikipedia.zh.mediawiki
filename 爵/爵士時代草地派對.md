'''爵士時代草地派對'''（{{lang-en|'''Jazz Age Lawn Party'''}}）又名'''爵士草地音樂派對'''，是一個以1920年代[[禁酒令時期|禁酒令時期]]為靈感的派對，由音樂家Michael Arenella創立，該活動於2005年開始，在美國[[紐約市|紐約市]]的[[總督島|總督島]]上舉行。派對以1920年代為主題，提供了[[爵士樂|爵士樂]]和跳舞表演，還有可以購買食物和飲品。<ref>{{Cite web |url = http://jazzagelawnparty.com/jazz-age-lawn-party/ |title = About Jazz Age Lawn Party |publisher = jazzagelawnparty.com |language = en}}</ref><ref>{{Cite web |url = https://www.timeout.com/newyork/music/jazz-age-lawn-party |title = The Jazz Age Lawn Party 2018 guide |author = Jillian Anthony, Jennifer Picht, Ross Tipograph |date = 2018-05-02 |publisher = Time Out |language = en}}</ref>

該活動最初只是由大約50名Michael Arenella的朋友組成的小型派對，後來越來越受歡迎，每年都吸引了成千上萬的人，現在成為紐約市最受歡迎的活動之一。

== 收費 ==
門票為55至5000美元，12歲以下的小童免費。<ref>{{Cite web |url = http://governorsislandguide.com/?tag=jazz-age-lawn-party |title = The True Story of the Jazz Age Lawn Party |date = 2017-06-08 |publisher = governorsislandguide.com |language = en}}</ref><ref>{{Cite web |url = http://jazzagelawnparty.com/faq/ |title = FAQs |publisher = jazzagelawnparty.com |language = en}}</ref>
