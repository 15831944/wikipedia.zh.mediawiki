{{vgname|抵抗3|en=Resistance 3|cn=抵抗3|tw=全面對抗2}}是一款由[[Insomniac_Games|Insomniac Games]]开发，[[索尼电脑娱乐|索尼电脑娱乐]]发行的PlayStation 3平台独占的第一人称射击游戏。本作是抵抗系列第四部游戏作品，也是主系列三部曲的最后一部<ref name="PS Blog">{{cite web |url=https://blog.us.playstation.com/2010/08/17/resistance-3-from-insomniac-games-coming-exclusively-to-ps3/ |title=Resistance 3 from Insomniac Games Coming Exclusively to PS3 |author=Marcus Smith |date=2010-08-17 |website=[[PlayStation_Blog|PlayStation Blog]] |publisher=[[Sony_Computer_Entertainment|Sony Computer Entertainment]]|language=en |accessdate=2019-02-13}}</ref><ref>{{cite web |url=https://gamerant.com/insomniac-resistance-4-riley-127289/ |title=Insomniac Will Not Be Making Any More 'Resistance' Games |author=Riley Little |date=2012-01-26 |website=Game Rant|language=en|accessdate=2019-02-13}}</ref>。《抵抗3》是抵抗系列中首款支持立体3D图像和PlayStation Move的作品。《抵抗3》的游戏风格由《[[抵抗：灭绝人类|抵抗：灭绝人类]]》和《[[抵抗2|抵抗2]]》的军事科幻转变成末日恐怖类。

2014年3月28日，索尼关闭了《[[抵抗：灭绝人类|抵抗：灭绝人类]]》、《[[抵抗2|抵抗2]]》、《抵抗3》的线上服务。<ref>{{Cite web |url = http://www.ali213.net/news/html/2013-12/93314.html |title = 今后只能玩单机！索尼关闭多款PS3游戏服务器 |author = 乐乐 |date = 2013-12-27 |publisher = 游侠网 |language = zh-cn |accessdate = 2019-02-13 }}</ref><ref>{{Cite web |url = http://www.ali213.net/news/html/2014-1/94084.html |title = SONY宣布旗下《抵抗》1-3系列服务器将停运 |author =  |date = 2014-01-07 |publisher = 游侠网 |language = zh-cn |accessdate = 2019-02-13 }}</ref><ref>{{cite web |url=http://www.eurogamer.net/articles/2014-01-06-sony-turning-off-gran-turismo-5-resistance-servers |title=Sony turning off Gran Turismo 5 and Resistance servers |author=Robert Purchese |date=2014-01-06 |website=[[Eurogamer|Eurogamer]] |publisher=Gamer Network |language=en|accessdate=2019-02-13}}</ref>
