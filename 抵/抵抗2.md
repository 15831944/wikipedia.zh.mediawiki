{{vgname|抵抗2|en=Resistance 2|cn=抵抗2|tw=全面對抗2}}是一款由[[Insomniac_Games|Insomniac Games]]开发，[[索尼电脑娱乐|索尼电脑娱乐]]发行的[[PlayStation_3|PlayStation 3]]平台独占的军事科幻第一人称射击游戏<ref>{{cite web |last=Kollar |first=Philip |date=2008-02-24 |url=http://www.1up.com/previews/resistance-2_4 |title=Resistance 2 (Preview) |website=[[1UP.com|1UP.com]] |publisher=[[Ziff_Davis|Ziff Davis]] |archiveurl=https://web.archive.org/web/20121018005530/http://www.1up.com/previews/resistance-2_4 |archivedate=2012-10-18 |deadurl=yes |accessdate=2019-02-12}}</ref>。本作是[[抵抗：灭绝人类|抵抗：灭绝人类]]的续作。

在合成兽Chimera发动对美国东西海岸全面入侵的情况下，上代的主角内森·黑尔（Nathan Hale）前往美国与合成兽Chimera进行战斗。本作中，黑尔是“哨兵”部队中的一员。“哨兵”部队由Chimera病毒感染者组成的精锐部队，成员需定期注射Chimera病毒抑制剂。

2014年3月28日，索尼关闭了《[[抵抗：灭绝人类|抵抗：灭绝人类]]》、《抵抗2》、《[[抵抗3|抵抗3]]》的线上服务。<ref>{{Cite web |url = http://www.ali213.net/news/html/2013-12/93314.html |title = 今后只能玩单机！索尼关闭多款PS3游戏服务器 |author = 乐乐 |date = 2013-12-27 |publisher = 游侠网 |language = zh-cn |accessdate = 2019-02-13 }}</ref><ref>{{Cite web |url = http://www.ali213.net/news/html/2014-1/94084.html |title = SONY宣布旗下《抵抗》1-3系列服务器将停运 |author =  |date = 2014-01-07 |publisher = 游侠网 |language = zh-cn |accessdate = 2019-02-13 }}</ref><ref>{{cite web |url=http://www.eurogamer.net/articles/2014-01-06-sony-turning-off-gran-turismo-5-resistance-servers |title=Sony turning off Gran Turismo 5 and Resistance servers |author=Robert Purchese |date=2014-01-06 |website=[[Eurogamer|Eurogamer]] |publisher=Gamer Network |language=en|accessdate=2019-02-13}}</ref>
