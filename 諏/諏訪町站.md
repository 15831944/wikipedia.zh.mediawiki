'''諏訪町站'''（{{jpn|j=諏訪町駅|hg=すわちょうえき|rm=Suwachō eki|f=y}}）是位於[[日本|日本]][[愛知縣|愛知縣]][[豐川市|豐川市]]諏訪三丁目的[[名古屋鐵道|名古屋鐵道]][[名鐵豐川線|豐川線]]鐵路車站。當地人稱此站是會略去「町」，稱為諏訪站。
== 概要 ==
車站內的閘機可以使用[[Manaca|Manaca]][[智能卡|智能卡]]，而曾經在2005年尾至2012年2月期間，可以使用Transpass。車站被使用了[[車站集中管理系統|車站集中管理系統]]，為一座[[無人站|無人站]]（由[[國府站_(愛知縣)|國府站]]管理）。在整條豐川線車站中，這是唯一一站是不可[[列車交會|列車交會]]。
== 車站構造 ==
[[側式月台|側式月台]]1面1線的地上車站。車站內的站舍曾經是全日本最小的有人站站舍。

== 利用狀況 ==
根據愛知縣的統計，在2008年，1日乘車人員平均為2,052人，是在豐川線中除了兩端終點站外最多人使用的車站。

== 歷史 == 
* 1945年（[[昭和|昭和]]20年）1月27日 - 車站開業，初時名為'''市役所前站'''。
* 1954年（昭和30年）1月20日 - 車站改名為'''諏訪町站'''。
* 1972年（昭和47年）6月1日 - 月台延長至可容納六卡車廂。
* 1983年（昭和58年）12月24日 - 設置站員，車站內的平交道停用。
* 2007年（平成19年）3月14日 - 開始使用[[車站集中管理系統|車站集中管理系統]]，也及[[TRANPASS|TRANPASS]]導入。
* 2011年（平成23年）2月11日 - [[智能卡|智能卡]][[manaca|manaca]]導入。
* 2012年（平成24年）2月29日 - 不再可以在此車站使用TRANPASS。
