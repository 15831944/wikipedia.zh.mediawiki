'''奔驰失控事件'''是2018年发生于[[中华人民共和国|中国]][[河南省|河南省]]高速公路的一起疑似[[奔驰|奔驰]][[轎車|轎車]][[定速巡航|定速巡航]]失灵事件。當晚车主薛立山驾驶奔驰C200L车型的轎車在[[连霍高速|连霍高速]]河南段以时速120km/h疾驰，期间发现无法取消定速巡航。该事件发生后受到众多网友包括汽车业人士在内的质疑。<ref>{{Cite web|url=http://news.sina.com.cn/o/2018-03-17/doc-ifysiypr3753620.shtml|title=热点 {{!}} “失控奔驰”后续！车主、官方均否认后台操控停车，救人的竟是车主一个动作！|accessdate=2018-05-11|work=news.sina.com.cn}}</ref>

== 事件经过 ==
2018年3月14日20点左右，薛立山驾驶[[奔驰|奔驰]]C200L沿连霍高速驶往成都，期间开启[[巡航定速|定速巡航]]（定速120km/h），在切换成人工驾驶模式时，定速巡航无法取消，奔驰售后为其提供多项解决方案，但定速巡航仍无法退出。随后报警，薛立山询问[[交警|交警]]是否可以追尾大货车以进行停车和跳车，后者予以劝阻，并建议薛立山剐蹭道路护栏，薛立山未采取该方案。<ref>{{Cite web|url=http://www.hnr.cn/news/snxw/201803/t20180316_3079566.html|title=男子奔驰失控时速120狂奔百公里 直闯收费站|accessdate=|author=|date=|publisher=}}</ref>

最终交警清空收费站的3个ETC收费通道，让其通行。薛立山称车辆恢复正常是由奔驰后台操控，事后[[奔驰|奔驰]]声明其不具有后台干预车辆的技术<ref>{{Cite web|url=http://auto.qq.com/a/20180316/032561.htm|title=奔驰回应巡航失控狂奔百公里：不具备后台干预车辆技术_汽车_腾讯网|accessdate=2018-06-03|work=auto.qq.com|language=zh-CN}}</ref>，薛立山改称由开启车门车辆会降速且关车门后车辆继续提速至120km/h，最终通过开启车门退出定速巡航和恢复制动功能。薛立山当晚仍驾驶该车辆继续前往成都。

该事件在网络上受到众多网友包括汽车业内人士质疑（如[[韩寒|韩寒]]<ref>{{Cite web|url=http://news.163.com/18/0317/13/DD3QN2040001875P.html|title=韩寒质疑|accessdate=2018-06-03|last=网易|work=news.163.com}}</ref>），质疑主要有以下：在中国繁忙的高速公路上，如何以匀速120km/h连续行驶1个多小时；为何任何方法都无法退出定速巡航；制动为物理机械，失灵后为何又能恢复正常；为何不公布行车记录仪内容等。<ref>{{Cite news|url=http://news.cqnews.net/html/2018-06/02/content_44418188.htm|title=奔驰车失控事件持续发酵 为何不能降低车速遭质疑|last=杜漩|accessdate=2018-06-03}}</ref>

薛立山最开始称行车记录仪已被覆盖，后称其拥有行车记录仪的内容，在局面进行采访时，又改称其未看过行车记录仪内容，局面王志安多次请求观看行车记录仪的内容，被薛立山多次搪塞。<ref>{{Cite web|url=https://v.qq.com/x/cover/5omjnw76v0ljmgu/n0609d17hk1.html?ptag=qqbrowser|title=局面{{!}}“失控奔驰”最全经过：车主详述失控一小时细节_局面_腾讯视频|accessdate=2018-06-03|last=Team|first=Tencent VFE|work=v.qq.com|language=zh-CN}}</ref>

事件车辆经由北京中机车辆司法鉴定中心进行鉴定，鉴定结果认为车辆并未失控<ref>{{Cite web|url=http://www.sohu.com/a/233565037_120849|title=失控奔驰鉴定结果出炉 车主难道在“说谎”？|accessdate=2018-06-03|date=2018-05-31|work=www.sohu.com}}</ref>。[[奔驰|奔驰]]认可和尊重鉴定结果，薛立山只尊重鉴定结果，并未认可并提出质疑。最初质疑[[U盘|U盘]]插入存在异常，后又称鉴定报告草率、看不懂。<ref>{{Cite web|url=https://www.jfdaily.com/news/detail?id=91403|title="失控奔驰"车主:喜欢刺激,鉴定报告草率敷衍看不懂--上观|accessdate=2018-06-03|work=www.jfdaily.com}}</ref>薛立山称交警将此事件通过媒体发了出去，对其造成伤害。<ref>{{Cite web|url=http://auto.eastday.com/a/180531203851356.html|title=都怪本泽马？“失控奔驰”调查结果打脸，车主律师竟责怪交警_东方汽车|accessdate=2018-06-03|work=auto.eastday.com|language=en}}</ref>

鉴定结果出炉后，一些网友提出质疑，如车主是否会被追责、是否应当承担法律责任<ref>{{cite web |title=“失控奔驰”检测结果不存在失控 车主将面临什么？ |url=http://www.uschinapress.com/2018/0528/1132601.shtml |website=侨报网 |accessdate=2018-06-13}}</ref>，鉴定结果是否可信等。也有文章怀疑事件是否有更多内情<ref>{{Cite web|url=http://auto.mop.com/a/180531151303480.html|title=“失控奔驰”迷雾未散，事件真相是否会就此湮没？_猫扑汽车|accessdate=2018-06-03|work=auto.mop.com|language=en}}</ref><ref>{{cite web |author1=车聚网 |title=检测结果现疑点，奔驰定速巡航失灵案或再现反转？ |url=http://www.sohu.com/a/233638572_115505 |website=搜狐 |accessdate=2018-06-13 |date=2018-06-01}}</ref>。

2019年4月15日，薛立山于新浪微博表示将于一个礼拜时间内正式就此事起诉北京奔驰汽车有限公司。<ref>[http://www.sohu.com/a/308034731_161795 “巡航失控”车主将起诉奔驰！称去年以来故障不断，还遇刹车失灵]</ref>
