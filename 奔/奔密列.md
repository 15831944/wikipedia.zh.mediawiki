'''奔密列'''（{{kmr|k=ប្រាសាទបឹងមាលា|r=Beng Mealea}}），一译'''崩密列'''，是一座位于[[柬埔寨|柬埔寨]][[暹粒省|暹粒省]]的吴哥古迹寺庙风格的[[印度教|印度教]]寺庙。距离[[吴哥古迹|吴哥古迹]]群以东40公里，自暹粒沿公路有77公里。它是吴哥建筑群第一座完全用沙石建筑的庙。<ref name="RB">{{cite book|author= |coauthors= |title=Ancient Angkor |year=2006 |publisher=River Books |location= |isbn= }}</ref>目前它依然被丛林严密包裹着，并且尚未大规模的修复。由于所处位置比较荒僻，所以许多浮雕和塑像已被掠夺。庙宇内遍布树木，以及许多石块倒至四处。
 
== 寺庙 ==
[[File:Beng_Mealea_ប្រាសាទបឹងមាលា_ベンメリア_DSCF4601.jpg|160px]]。]]
[[File:Beng_Mealea_ប្រាសាទបឹងមាលា_ベンメリア_DSCF4599.jpg|thumb]]
[[File:Beng_mealea_map_path.JPG|thumb]]
崩密列的历史还处于不明瞭的状态，据学者推测，崩密列大约建造于11世紀末至12世紀初，即[[蘇利耶跋摩二世|蘇利耶跋摩二世]]时期。这座印度庙宇是用来供奉[[湿婆|湿婆]]神的，虽然它是一座印度教寺庙，可是它有一些雕塑反映的是[[佛教|佛教]]的主题。<ref name="RB"/>它是一座破败、隐秘，未经修缮的寺庙，这座神庙从来没有完工，原本应该有浮雕的寺墙上都空空如也。 
 
和[[吴哥窟|吴哥窟]]一样，崩密列外围有一条护城河，长1.2公里，宽0.9公里。<ref name="RB"/>

由於比較荒僻，很多浮雕和塑像已被盜賊掠奪一空。

[[宮崎駿|宮崎駿]]執導的[[動畫片|動畫片]]《[[天空之城|天空之城]]》，據說原型就是崩密列。

== 地位 ==
1992年9月1日，崩密列被列入联合国世界文化遗产预备名单（UNESCO World Heritage Tentative List）。<ref name="RB"/>

== 入場 ==
[[File:Beng_Mealea-Restaurant_ប្រាសាទបឹងមាលា_ベンメリア_DSCF4786.jpg|170px]]
* 開放時間：6:00至17:00，每周7天。
* 入場費$5[[美元|美元]]。

由於遺址損毀嚴重，故可以徒步參觀的範圍有限。應留意叢林中的[[毒蛇|毒蛇]]。附近的道路上也有被[[搶劫|搶劫]]的風險。此外，遺址周邊地方亦埋有舊[[地雷|地雷]]，當局正在將其撤去。

== 交通 ==
與[[暹粒市|暹粒市]]的距離約77[[公里|公里]]，乘搭[[嘟嘟車|嘟嘟車]]需要約兩個小時。
