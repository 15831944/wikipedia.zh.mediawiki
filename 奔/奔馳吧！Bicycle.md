《'''奔馳吧！Bicycle'''》（{{lang-ja|走れ!Bicycle}}）是日本女子組合[[乃木坂46|乃木坂46]]的第3張[[單曲|單曲]]，於2012年8月22日由[[Sony_Music_Records|Sony Music Records]]發行。

== 發行及宣傳 ==
《奔馳吧！Bicycle》與上一張單曲《[[來吧香波|來吧香波]]》相隔約三個月發售，是乃木坂46於2012年發行的第三張單曲。單曲分為附上DVD的「Type-A」、「Type-B」和「Type-C」，以及只有CD的「通常盤」，共四種形式發行。Type-A、Type-B和Type-C在首批發行期間均附送全國[[握手會|握手會]]應徵券兼特別禮物應徵券（{{lang|ja|全国握手会参加券兼スペシャルプレゼント応募券}}）以及生寫真一張。<ref name="A">{{cite web|url=http://www.nogizaka46.com/discography/?d=SRCL-8058 |language=ja |script-title=ja:走れ！Bicycle【CD+DVD盤/Type-A】 |publisher=[[乃木坂46|乃木坂46 LLC]] |year=2012 |accessdate=2013-02-23}}</ref>發行日期的前一天——8月21日是乃木坂46成立一週年。<ref name="natalie0806">{{cite web|url=http://natalie.mu/music/pp/fujiqnogizaka/page/3|language=ja |script-title=ja:【Super Power Push】 乃木坂46×富士急ハイランド コラボレーション企画 (3/7) |publisher=[[娜塔莉_(網站)|natalie]] |date=2012-08-06 |accessdate=2013-02-23}}</ref>

2012年7月7日，乃木坂46於[[幕張展覽館|幕張展覽館]]舉辦的握手活動中，於約8000名歌迷前首次演唱《奔馳吧！Bicycle》。<ref>{{cite web|url=http://www.oricon.co.jp/news/music/2013983/full/ |language=ja |script-title=ja:乃木坂46、握手会に過去最多8000人　本格ダンスの新曲も初披露 |publisher=[[Oricon|Oricon]] |date=2012-07-07 |accessdate=2013-02-23}}</ref>由於其首張單曲《[[窗簾圍繞|窗簾圍繞]]》和第2張單曲《來吧香波》分別於[[AKB48|AKB48]]的「[[AKB48_重溫時間_最佳曲目100_2012|AKB48 重溫時間 最佳曲目100 2012]]」演唱會和電視上（《[[乃木坂在哪裡？|乃木坂在哪裡？]]》）首次公開，故這首歌曲是她們第一次能夠在自己的活動上披露歌曲。<ref name="natalie0806" />

== 歌曲 ==
A面曲《奔馳吧！Bicycle》（{{lang|ja|走れ!Bicycle}}）共有四名作曲者，分別是Shusui、伊藤涼、木村篤史和{{lang|ja|ヒロイズム}}。四名作曲者都曾為[[傑尼斯事務所|傑尼斯事務所]]的藝人作曲。<ref name="Billboard JAPAN" />這首歌的填詞人為乃木坂46的總製作人[[秋元康|秋元康]]。<ref name="Billboard JAPAN" />根據[[每日新聞|每日新聞]]電子版的報導，這首歌將天愛無邪的戀愛跟迎風前進的自行車重疊描述，是一首明朗的歌曲。<ref>{{cite web|url=http://mantan-web.jp/2012/08/22/20120822dog00m200015000c.html |language=ja |title=<span lang="ja">注目の新譜 ： 乃木坂46「走れ！Bicycle」恋愛模様と風を切る自転車を重ねた爽やかポップ</span> |publisher=MANTANWEB |date=2012-08-22 |accessdate=2013-02-23}}</ref>乃木坂46於[[Oricon|Oricon]]的一次訪問中指，歌中兩人雖然兩情相悅，但是沒有發現對方的愛意，由副歌起描述主角為了取回愛情，拼命地踏着自行車的情境。<ref name="oricon0822" />

《奔馳吧！Bicycle》是[[好侍食品|好侍食品]]（{{lang|ja|ハウス食品}}）「{{lang|ja|メガシャキ}}」的廣告歌曲。<ref name="A" />B面曲《人為什麼奔跑？》（{{lang|ja|人はなぜ走るのか?}}）是[[2012年U20女子世界盃|2012年U20女子世界盃]]的大會官方歌曲。<ref>{{cite web|url=http://www.oricon.co.jp/news/movie/2015674/full/ |language=ja |title=<span lang="ja">乃木坂46、“ヤングなでしこ”にエール　生駒はPK成功に「びっくり」</span> |publisher=[[Oricon|Oricon]] |date=2012-08-15 |accessdate=2013-02-23}}</ref>

== 影片 ==
乃木坂46於上兩張單曲的DVD中都收錄了全體33名成員各自的個人影片。此單曲延續此慣例，以「33人悠閒旅行」（{{lang|ja|33人ぶらり旅}}）為主題，拍攝每名成員在日本國內國外閒逛的情境。每名成員的影片分別都由一名創意工作者執導。例如[[市來玲奈|市來玲奈]]的影片由[[芥川龍之介獎|芥川獎]]得獎作家[[平野啓一郎|平野啓一郎]]執導，出身於[[大分縣|大分縣]]的[[衛藤美彩|衛藤美彩]]則配搭[[大分市|大分市]]市長[[釘宮磐|釘宮磐]]等等。<ref>{{cite web|url=http://www.oricon.co.jp/news/music/2015146/full/ |language=ja |title=<span lang="ja">乃木坂46「33人個別ビデオ」をダンカンらが監督</span> |publisher=[[Oricon|Oricon]] |date=2012-08-12 |accessdate=2013-02-23}}</ref>{{noteTag|1=乃木坂46於同年5月1日被委任為[[大分市|大分市]]觀光大使。<ref>{{cite web |url=http://www.oita-press.co.jp/localNews/2012_133644184514.html |language=ja |title=<span lang="ja">乃木坂４６、大分市観光特使に</span> |publisher=大分合同新聞 |date=2012-05-08 |accessdate=2013-02-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20130511152535/http://www.oita-press.co.jp/localNews/2012_133644184514.html |archivedate=2013-05-11 }}</ref>}}

除了上述的個人影片外，DVD中還收錄了四首歌的[[音樂影片|音樂影片]]（PV），包括《奔馳吧！Bicycle》、《眼淚仍然悲傷的時候》、《人為什麼奔跑？》和《沉默的結他》。

《奔馳吧！Bicycle》的PV由《[[幸福的魔法繪本|幸福的魔法繪本]]》、《[[告白_(電影)|告白]]》導演[[中島哲也|中島哲也]]執導。<ref name="Billboard JAPAN">{{cite web |url=http://www.billboard-japan.com/hs_discs/detail/304 |language=ja |title=<span lang="ja">乃木坂46 待望の3rdシングルはとにかく不思議</span> |publisher=Billboard JAPAN |date=2012-05-08 |accessdate=2013-02-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20120827131644/http://www.billboard-japan.com/hs_discs/detail/304 |archivedate=2012年8月27日 |df= }}</ref>PV甫開始就是一對男女以[[法語|法語]]對話的旁白。<ref name="Billboard JAPAN" /><ref name="oricon0822">{{cite web|url=http://www.oricon.co.jp/music/interview/page/173/ |language=ja |title=<span lang="ja">乃木坂46『理想の告白シチュエーション＆理想の男性って……！？』</span> |publisher=[[Oricon|Oricon]] |date=2012-08-22 |accessdate=2013-02-23}}</ref>組合成員在PV中穿上無袖衣著和戴上[[假髮|假髮]]跳舞。<ref name="Billboard JAPAN" />

== 排行榜成績 ==
《奔馳吧！Bicycle》發行首周的銷量約為18.7萬張，銷量比《來吧香波》的約15.6萬張高。單曲憑此銷量取得同年9月3月[[Oricon公信榜|Oricon公信榜]]單曲周榜首名，成為乃木坂46出道以來第二張冠軍單曲。<ref name="oricon" />該週的專輯周榜首名是其「官方對手」[[AKB48|AKB48]]的《[[1830m|1830m]]》。雖然截至當時為止，AKB48所有相關組合在2012年共有12張作品登上Oricon公信榜首位，卻是首次於同一星期佔有兩榜冠軍。<ref name="oricon" />

此單曲於發行該月（8月）獲[[日本唱片協會|日本唱片協會]]認證為金唱片<ref>{{cite web|title=<span lang="ja">ゴールド等認定作品一覧　2012年8月</span> |url=http://www.riaj.or.jp/data/others/gold/201208.html |publisher=[[日本唱片協會|RIAJ]] |accessdate=2012-09-10 |language=ja}}</ref>，下一個月（9月）再獲認證為白金唱片。<ref name="riaj">{{cite web|title=<span lang="ja">ゴールド等認定作品一覧　2012年9月</span> |url=http://www.riaj.or.jp/data/others/gold/201209.html |publisher=[[日本唱片協會|RIAJ]] |accessdate=2013-02-24 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20121013212752/http://www.riaj.or.jp/data/others/gold/201209.html |archivedate=2012-10-13 }}</ref>

2012年，Oricon公信榜的單曲年榜上，此單曲以約24.1萬張的銷量取得第26名，在乃木坂46三張進榜單曲中排名最高。<ref name="oricon yearly" />
