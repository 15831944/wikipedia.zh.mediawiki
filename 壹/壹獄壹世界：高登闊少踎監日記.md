《'''壹獄壹世界：高登闊少踎監日記'''》（{{lang-en|Imprisoned: Survival Guide for Rich and Prodigal}}）是2015年[[喜剧|喜剧]][[犯罪片|犯罪片]]，改编自[[高登讨论区|高登讨论区]]会员[[小姓奴|小姓奴]]的小说《[[壹狱壹世界|壹狱壹世界]]》。由[[孙立基|孙立基]]导演，[[王宗尧|王宗尧]]、[[張建聲|張建聲]]、[[廖启智|廖启智]]、[[黄光亮|黄光亮]]、[[何家驹|何家驹]]、[[张松枝|张松枝]]主演。电影讲述了王宗尧饰演的“富二代”-{于}-日辰因[[醉酒驾驶|醉酒驾驶]]撞死老妪被判18个月，经历了监狱风云后重新认识人生意义的故事。本片也是演员[[何家驹|何家驹]]闊別銀幕15年後的復出作以及遗作。电影于2015年5月28日在香港上映<ref>{{cite web |url= http://www.hkgolden.com/articles/article.aspx?id=18746 | title=《壹獄壹世界·高登闊少踎監日記》預告鐵窗新秩序？ |date=2015-05-14 |accessdate= 2015-05-27 |work= [[高登讨论区|高登讨论区]]}}</ref><ref>{{cite web |url= http://hk.apple.nextmedia.com/enews/realtime/20150502/53696876 | title=【搵錢嘅好時機】600港人為二萬蚊爭坐監 張建聲已踎監26小時 |date=2015-05-02 |accessdate= 2015-05-27 |work= [[苹果日报|苹果日报]]}}</ref>。

== 制作 ==
[[小姓奴|小姓奴]]是香港[[高登讨论区|高登讨论区]]一名以创作小说著名的会员，自2007年开始创作各种风格各异深受网民欢迎的作品。2013年6月，小姓奴利用署名“-{于}-日辰”在高登发表连载小说《一狱一世界》，共连载五篇帖子。小说分享了作者-{于}-日辰入狱后在狱中生活的点滴，获得网民好评。最初，由于作者“-{于}-日辰”与小姓奴散文集题目《一日之计在于晨》极为相近，小说开始连载时便被读者怀疑，尤其是小说连载到第一篇突然停刊，跟小姓奴因意外入院的时间不谋而合，而更加令人怀疑“-{于}-日辰”就是小姓奴。最终小姓奴在Facebook专页证实《一狱一世界》是他的小说，作者身份的谜团才被解开。之后，碍于小姓奴住院治疗无法续写小说，网民合力打造书中的结局，共写了90000字。2013年，中国3D数码娱乐公司耗资700万买入小说的电影改编权，[[香港电视|香港电视]]艺人[[郭锋|郭锋]]、[[艾威|艾威]]、[[张松枝|张松枝]]和[[黄美棋|黄美棋]]获邀参演<ref>{{cite web |url=http://hk.apple.nextmedia.com/entertainment/first/20131031/18486481 |title=
天無絕人之路 港視藝人跳入大銀幕 |date=2013-10-31 |website=苹果日报}}</ref>。
