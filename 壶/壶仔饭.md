'''壶仔饭'''， 是[[中餐|中餐]]的一种。以陶壶作为炊具，放入糯米、香菇、卤蛋等食材蒸煮而成的蒸饭。发源于中国泉州晋江市，并在该市及临近地区有分布。

== 历史 ==
*壶仔饭起源的时间，大致为[[清末|清末]][[民初|民初]]时期，距今已有百余年的历史。
*而关于壶仔饭的起源原因，有传说为晋江[[深沪镇|深沪镇]]当地一位母亲为儿子捏制的咸饭[[饭团|饭团]]。<ref>{{cite web | title=晋江名小吃深沪壶仔饭 香气扑鼻油而不腻 | url=http://www.mnw.cn/jinjiang/lvyou/ms/688627.html | accessdate=2013-10-25 }}</ref>
*但从实际做法及配料考证，壶仔饭实为[[肉粽|肉粽]]的近亲。<ref>{{cite web | title=晋江深沪壶仔饭：肉粽演变而来的闽南小吃
 | url=http://qz.fjsen.com/2013-01/22/content_10431857.htm | accessdate=2013-01-22 }}</ref>

== 食用 ==
*壶仔饭一般为店铺售卖，较少有自家进行制作。
*出于循环利用及卫生、安全的考量。壶仔饭上桌前，并不是将煮饭的陶壶直接端给顾客。而是由店家将[[陶|陶]]壶内的米饭倒入[[碗|碗]]或[[碟|碟]]中。
*米饭倒出后，最关键的一步。是由店员撒上店家准备好的[[酱汁|酱汁]]、[[花生米|花生米]]、[[葱|葱]]等调味料、佐料，<ref>{{cite web | title=来光顾的多是熟客 六年来价格只涨了1块
 | url=http://qz.fjsen.com/2013-01/22/content_10431857_2.htm | accessdate=2013-01-22 }}</ref>再端给食客进行享用。
*与[[煲仔饭|煲仔饭]]不同，壶仔饭由于系用蒸煮的方式制成，因此没有锅巴。

== 分布 ==
*壶仔饭的分布，以发源地晋江市深沪镇为最多。其次是晋江的其它乡镇及[[石狮市|石狮市]]、泉州市区<ref>{{cite web | title=13年才涨2块钱，这碗深藏中山路的壶仔饭，只有老泉州知道 | url=https://baijiahao.baidu.com/s?id=1578392900671574846&wfr=spider&for=pc | accessdate=2017-09-13 }}</ref>
*另外在泉州各县及临近的[[厦门市|厦门市]]，亦有分布1家及以上。而闽南地区以外，则罕有此类店家。<ref>{{cite web | title=他用20年传承这一碗饭，每个泉州人心中都有属于它的独家记忆 | url=https://baijiahao.baidu.com/s?id=1582650756511093485&wfr=spider&for=pc | accessdate=2017-10-30 }}</ref>
