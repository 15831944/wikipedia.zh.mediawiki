'''犀利哥'''，原名'''程国荣'''{{BD|1976年|10月10日|||CatIdx=C程}}<ref>http://news.zjol.com.cn/08ms/system/2010/03/05/016386418.shtml</ref>，江西鄱阳人，2010年初在[[中国|中国]][[浙江|浙江]][[宁波|宁波]]流浪[[行乞|行乞]]。2010年2月，蜂鸟摄影社区一位用户试相机时拍摄的一组照片，随后以《秒杀宇内究极华丽第一极品路人帅哥！帅到刺瞎你的狗眼！求亲们人肉详细资料》为题转载于[[天涯论坛|天涯论坛]]<ref>{{cite web|author=街头湿人 |url=http://bbs.tianya.cn/post-funinfo-1840563-1.shtml |title=秒杀宇内究极华丽第一极品路人帅哥！帅到刺瞎你的狗眼！求亲们人肉详细资料_娱乐八卦_论坛_天涯社区 |publisher=Bbs.tianya.cn |date= |accessdate=2018-11-29}}</ref>。相片中的乞丐看起来造型独特，衣着另类，表情冷峻，因而获得“犀利哥”之名，收到网民的追捧，迅速成为网络热点人物<ref>{{cite web|url=http://news.ifeng.com/society/4/201002/0224_346_1554311.shtml |title=“最有范儿”乞丐犀利哥受到网友追捧(组图)_资讯_凤凰网 |publisher=News.ifeng.com |date=2010-02-23 |accessdate=2018-11-29}}</ref>。

犀利哥由于生活坎坷可能患有轻微精神疾病。宁波市政府要求救助站务必找到“犀利哥”，救助站和热心网友曾找到“犀利哥”，但“犀利哥”在过度关注下惊慌失措，无法与人正常交流，并拒绝前往救助站<ref>[http://news.xinmin.cn/rollnews/2010/03/03/3832685.html 宁波市政府要求救助站一定要找到“犀利哥”]</ref>，他曾在家鄉與親人團聚<ref>{{cite web|url=http://paper.wenweipo.com/2010/03/07/WY1003070003.htm |title=「犀利哥」一家團圓 離家10年有2兒 |publisher=Paper.wenweipo.com |date= |accessdate=2018-11-29}}</ref>，但不久後又在老家附近開始流浪生活，2013年初被網友在[[天涯论坛|天涯论坛]]提及<ref>[http://bbs.tianya.cn/post-free-3127340-1.shtml 犀利哥重出江湖，再次流浪无人管]，2013年3月13日[[天涯论坛|天涯论坛]]</ref>下，由[[江西電視台|江西電視台]]都市頻道一個節目了解跟進<ref>[http://www.goldia.cn/article/1303/22/1817198722.html 成名后的流浪]，2013年3月21日浙江二套《都市情緣》</ref>之下，被程国圣送到鄱陽精神病醫院安頓治療。

== 质疑与回覆 ==
由于贴中乞丐的举止太过惊世骇俗，有些人甚至质疑是否是真的乞丐。作者“街头湿人”在帖子中自称这是很真实的一张照片，因为没有PS（[[Photoshop|Photoshop]]）痕跡（但事实上，此发言也可能是来源于各大论坛形成的某种回帖句式，即：以发表“毫无PS痕迹！”一类语言来讽刺图片的虚假性或赞扬其PS功力。），街头的背景又说明了这就是一次偶然的抓拍——那苍老黝黑的手，证明了犀利哥就是一个货真价实的乞丐，不是模特或者演员。

== 身世探究 ==
2010年3月5日，“犀利哥”与家人在宁波市求助站相认。“犀利哥”真名叫程国荣，江西鄱阳人，35岁。已婚并育有两个孩子，两年前，父亲和妻子因车祸不幸亡故。家境贫苦，多年前离家打工至今音讯杳无。至于其目前的精神病况，家人亦表示不知情<ref>{{cite web|url=http://news.163.com/10/0306/01/61296U9V00011229.html |title="犀利哥"与家人团聚 出走近10年 |publisher=News.163.com |date= |accessdate=2018-11-29}}</ref><ref>{{cite web|url=http://news.qq.com/a/20100310/000435.htm |title=犀利哥回家后不再“犀利”当地人以他为骄傲 |publisher=News.qq.com |date= |accessdate=2018-11-29}}</ref>。

有网友透露，犀利哥是其战友。犀利哥1997年入伍后在[[南京军区|南京军区]]江西南昌当兵，曾经参加过[[1998年中國水災|1998年中國水災]]的抗洪抢险任務，并且受到嘉奖<ref>{{cite web|url=http://news.qq.com/a/20100303/000292.htm |title=网友称犀利哥是98年抗洪老兵 曾获嘉奖(图) |publisher=News.qq.com |date= |accessdate=2018-11-29}}</ref>。

== 成名後活動 ==
凭借一身乞丐装和其被不少人认为“潇洒”的举动，被称作「潮爆了」（即十分时尚）而紅透网絡的「犀利哥」程國榮在2010年5月1日受聘於順德碧桂園農莊擔任時裝表演部門經理，至5月4日與弟弟跟姐姐與弟弟程國聖單方面回江西老家，據說4日的走秀表演令犀利哥進帳3萬元[[人民幣|人民幣]]。其叔叔程广华坦承，碧桂園農莊看中的是犀利哥的名氣，犀利哥不怕網友誤會他借名氣“炒作賺錢”，“對於犀利哥來說，以自己的能力賺錢，重新找到自己生活的價值，這是非常重要的，所以我們家人全力支援他，也希望網友理解。”

2016年2月24日台湾中时电子报报道，犀利哥已失踪一年<ref>{{cite web|url=http://www.chinatimes.com/realtimenews/20160224005437-260404 |title=揭爆紅「犀利哥」現況 已失蹤1年多 |publisher=Chinatimes.com |date=2016-02-24 |accessdate=2018-11-29}}</ref>。
