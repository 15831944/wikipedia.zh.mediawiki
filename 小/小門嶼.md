'''小門嶼'''為[[澎湖群島|澎湖群島]]之一，屬於[[澎湖縣|澎湖縣]][[西嶼鄉|西嶼鄉]]小門村，因其位於[[吼門水道|吼門水道]]旁，故有'''吼門之珠'''之稱<ref name="multiple">[http://nrch.culture.tw/twpedia.aspx?id=28257 小門嶼-台灣大百科全書]</ref>，其位置位於[[西嶼|西嶼]]北方，以寬約20公尺的水道與西嶼相隔，面積約{{convert|0.5|km2|ha}}，最高處位於島嶼西北側，高23公尺。<ref name="multiple2">楊貴三、沈淑敏（民99）。台灣全志，卷二，土地志，地形篇。南投市：台灣文獻館。</ref>

== 地形 ==
小門嶼整體為一座[[玄武岩|玄武岩]][[方山|方山]]小島<ref name="multiple3">[http://landscape.phc.edu.tw/landscapes/geopark_sy.htm 西嶼鄉小門嶼-澎湖地景資源網] {{webarchive|url=https://web.archive.org/web/20140130231837/http://landscape.phc.edu.tw/landscapes/geopark_sy.htm |date=2014-01-30 }}</ref>，南、北兩岸柱狀[[節理|節理]][[玄武岩|玄武岩]]較為發達。東岸多孔狀玄武岩發達，球狀風化十分普遍。本嶼東北部臺地受雨水侵蝕後造成許多小雨溝，類似惡地地形。<ref name="multiple2">楊貴三、沈淑敏（民99）。台灣全志，卷二，土地志，地形篇。南投市：台灣文獻館。</ref>

[[鯨魚洞|鯨魚洞]]位於本嶼北海岸，此處原為玄武岩的海蝕崖，經過[[海蝕|海蝕]]後，將玄武岩下方之[[砂岩|砂岩]]、[[頁岩|頁岩]]層挖蝕，最後貫穿成一[[海蝕門|海蝕門]]。<ref name="multiple2">楊貴三、沈淑敏（民99）。台灣全志，卷二，土地志，地形篇。南投市：台灣文獻館。</ref>

== 地質 ==
小門嶼的指標地層為小門嶼層，其由近代隆起的海濱沈積物所組成，可分為上層的[[殼灰岩|殼灰岩]]與下層的鐵質[[石英砂岩|石英砂岩]]與疏鬆的[[砂岩|砂岩]]、[[泥岩|泥岩]]互層。<ref name="multiple3">[http://landscape.phc.edu.tw/landscapes/geopark_sy.htm 西嶼鄉小門嶼-澎湖地景資源網] {{webarchive|url=https://web.archive.org/web/20140130231837/http://landscape.phc.edu.tw/landscapes/geopark_sy.htm |date=2014-01-30 }}</ref>

== 對外交通 ==
小門嶼位置位於[[西嶼|西嶼]]北方，以寬約20公尺的水道與西嶼相隔，以小門橋與西嶼連接，為小門嶼唯一的對外橋梁，而小門橋又稱為[[跨海小橋|跨海小橋]]。<ref name="multiple">[http://nrch.culture.tw/twpedia.aspx?id=28257 小門嶼-台灣大百科全書]</ref>而其對外道路編號為[[澎2鄉道|澎2鄉道]]。<ref>[http://blog.yam.com/buslover/article/51876802 20120708 馬公-外垵 搭乘記錄 <返程>-巴拉巴拉巴士愛就愛哈拉]</ref>

== 人文地景 ==
小門嶼除壯麗的玄武岩與鯨魚洞等自然地景外，人文地景也頗為豐富，擁有[[鎮風塔|鎮風塔]]、[[石敢當|石敢當]]、[[菜宅|菜宅]]等。<ref name="multiple">[http://nrch.culture.tw/twpedia.aspx?id=28257 小門嶼-台灣大百科全書]</ref>
