'''小南天福德爺祠'''位於臺灣臺南市中西區，是主祀福德正神的廟宇，又稱'''番薯崎小南天'''{{r|謝奇峰}}{{r|市}}。該廟名稱由來據說是明寧靖王[[朱術桂|朱術桂]]遊覽番薯崎一帶，見風景優美微南天勝境，故而將此祠命名為小南天，並題字懸額，但該匾已不存{{r|謝奇峰}}{{r|市}}。

== 沿革 ==
相傳明永曆廿年（1666年）明寧靖王到蕃薯崎一帶遊覽，見風景優美，為南天勝境，故而將廟名取做「小南天」{{r|謝奇峰}}。此外據說寧靖王還手書「小南天」匾額，但該匾現已不存，目前的「小南天」匾額是嘉慶廿四年（1819年）之作{{NoteTag|謝奇峰《圖解台灣神明圖鑑》一書寫是嘉慶十九年（1814年）之作{{r|謝奇峰}}，但匾上所寫之「嘉慶己卯年」應是嘉慶廿四年（1819年）。}}，出自[[林朝英_(清朝)|林朝英]]之手{{r|福德}}{{r|謝奇峰}}。

清康熙廿三年（1684年）重建{{r|林}}。雍正初年，巡道[[吳昌祚|吳昌祚]]修建此廟{{r|謝奇峰}}{{r|林}}。乾隆五十八年（1792年）巡道[[金溶|金溶]]與地方人士再修，而後嘉慶十九年（1814年）信徒們{{NoteTag|主要有仕紳郭三元、石繼水、魏九、倪國董等人{{r|福德}}。}}再次修廟並於嘉慶廿三年（1818年）獻匾「德厚聿隆」{{r|謝奇峰}}{{r|林}}{{r|福德}}。日大正三年（1914年）董事曾敏卿、許拔臣又發起重修{{r|謝奇峰}}{{r|福德}}。二次大戰後，民國45年（1956年）董事白惠文、蕭魁等人募款重修，並迎開基武廟關帝爺合祀{{r|謝奇峰}}{{r|福德}}。民國74年（1985年）再修，於民國76年（1987年）安座，民國78年（1989）舉行落成大典{{r|謝奇峰}}{{r|林}}{{r|福德}}。

民國105年（2016年）左右再次整修。民國106年（2017年）舉行「丁酉年三朝慶成祈安建醮」儀式{{r|林}}。

== 其他 ==
小南天牆壁上掛有「二十八星宿籤詩板」，是嘉慶廿五年（1820年）的文物{{r|謝奇峰}}。而小南天求取籤詩的方式有別於其他廟宇，不是先抽籤再[[擲筊|擲筊]]，而是直接連擲三筊，在根據結果去找對應的籤詩{{r|謝奇峰}}{{r|程}}。

另外據說清同治年間，[[臺灣鎮總兵|臺灣鎮總兵]][[曾元福|曾元福]]在府城太平街建公館，所以該廟的土地公因有中央級高官在之故而「加官進爵」，戴官帽，披官袍{{r|福德}}。
