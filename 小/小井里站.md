'''小井里站'''（{{kor|諺= 소정리역|漢=小井里驛|羅=Sojeong-ri yeok|f=y}}）是位于[[韩国|韩国]][[世宗特别自治市|世宗特别自治市]]的一个车站，属于[[京釜线|京釜线]]。2017年7月停止旅客營運。
== 車站結構 ==
2面4線雙島式月台的地面車站。

== 历史 ==
*1906年1月20日 - 落成使用。
*1907年9月20日 - 车站毁坏。
*1942年 - 新站舍竣工。
*2002年8月25日 - 车站改建完成。
*2004年3月30日 - 车站起火导致车站毁坏。
*2005年1月 - 车站重建完成。
*2017年7月 - 停止旅客營運。
