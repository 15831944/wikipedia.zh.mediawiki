'''小斑背潛鴨'''（[[學名|學名]]：{{lang | la |''Aythya affinis''}}），是一種產於[[北美|北美]]的小型[[雁鴨科|雁鴨科]][[潛鴨屬|潛鴨屬]]鳥類，冬天會向南遷徙到[[中美洲|中美洲]]。本種與[[斑背潛鴨|斑背潛鴨]]（''A. marila''）關係接近，兩者同屬於一個{{link-en|複合種|Species complex}}
。<ref name = carboneras1992 /><ref name=Livezey/> 

==描述==
成鴨體長為{{convert|38|-|48|cm|in|abbr=on}}，平均體長約為{{convert|41.7|-|43|cm|in|abbr=on}}。體重在{{convert|454|-|1089|g|lb|abbr=on}}之間；雄鴨平均體重約為{{convert|820|g|lb|abbr=on}}<ref name= Cornell/>。翅膀長度（非[[翼展|翼展]]）雄鴨約為{{convert|7.5|-|7.9|in|cm|abbr=on}}，雌鴨則為{{convert|7.3|-|7.8|in|cm|abbr=on}}；[[跗骨|跗骨]]長度約為{{convert|1.4|-|1.5|in|cm|abbr=on}}，[[喙|喙]]長{{convert|1.4|-|1.7|in|cm|abbr=on}}<ref name = carboneras1992 /><ref name=DUb/><ref name = Madge/><ref name=DUa/>，翼展為 {{convert|68|-|78|cm|in|abbr=on}}<ref name= Cornell/>。

成年雄性的[[換羽|繁殖羽]]有一個黑色的頭部，略帶彩虹光澤，後面有一小簇頭羽。胸部為黑色，背部灰白色 。翅膀外圍為黑色，帶有深色漣紋，內側的主要飛行羽則為灰棕色。身體下半部是白色的，側面有一些[[橄欖綠|橄欖色]]的漣紋，[[尾羽|尾羽]]和尾下覆羽則是黑色的。

成年雌鴨在喙的基部有一白色條帶，耳朵區域顏色通常較淺。整體為深褐色，漸次轉為中腹部的白色。

雄鴨的[[蝕羽|蝕羽]]（eclipse plumage），即非繁殖羽外觀變化不大，但頭部和胸部非常黑，頭部很少或沒有白色，翅膀上常有一些灰色的漣狀紋。未成熟的幼鳥外表與雌鴨相似，但顏色較為暗淡，喙的基部也幾乎沒有任何白色帶。雌雄鴨皆有白色的第二飛行羽，藍灰色的喙，一個黑色的“釘子”在尖端，和灰色的腳；雄鴨的[[虹膜|虹膜]]為明亮的黃色，雌鴨呈橙色或[[橙黃色|琥珀色]]，未成熟的幼鳥則呈棕色。<ref name = carboneras1992 /><ref name=DUb/><ref name = Madge/><ref name=DUa/>
