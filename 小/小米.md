'''粟'''（[[学名|学名]]：''{{lang|la|Setaria italica}}''），亦称'''、'''稷'''、'''粟米'''，北方又称'''谷子'''，俗称'''小米'''，属[[禾本科|禾本科]][[狗尾草属|狗尾草属]]的一种[[农作物|农作物]]。早在[[新石器时代|新石器时代]]就已经被种植，在[[青海|青海]]遗址中也有出土。

== 形态 ==
一年生[[草本|草本]]；秆粗壮、分蘖少，狭长披针形叶片，有明显的中脉和小脉，具有细毛；穗状圆锥花序；穗长20～30厘米；小穗成簇聚生在三级支梗上，小穗基本有刺毛。粟的稃壳有白、红、黄、黑、橙、紫各种颜色，俗称“粟有五彩”。卵圆形籽实，粒小多为黄色。

== 分布 ==
原产于[[中国|中国]]北方[[黄河|黄河]]流域；粟耐旱，适合在干旱而缺乏灌溉的地区生长。[[台灣|台灣]]原生種(又稱華南種)小米，生長於山區，為[[台湾原住民|台湾原住民]]早期主食。

== 用途 ==
粟在中国北方俗称小米，是中国古代的主要粮食作物。華夏遠古時期[[夏|夏]]和[[商|商]]代属于“粟文化”，小米作為北方主要糧食作物一直耕種至宋朝，甚至遼金时期中原仍主要以小米为主食。中国最早的[[酒|酒]]也是用小米酿造的。粟的[[茎|茎]]、[[叶|叶]]较坚硬，可用作[[饲料|饲料]]，一般只有[[牛|牛]]能消化。

[[元|元]][[清|清]]時期蒙古和滿[[通古斯|通古斯]]流入粟和[[黍|黍]]后，[[蒙古族|蒙古族]]與[[滿族|滿族]]會做成炒米或粘包食用，但當時是小麦在北方的大扩张时期，粟和黍的食用频率远远不及麦制品。如今蒙古與满族卻皆自称小米是其民族傳統食品，同样在韩国也有自称是其传统食品的说法。

== 其他 ==
[[File:1952-10_1952年河北昌黎县种植小米.png|thumb]]
经过多年人工除草，谷地里的杂草非常像谷子，尤其幼苗时期，叫做“莠”，[[成语|成语]]“良莠不齐”则是由此产生。

[[西方|西方]][[语言|语言]]一般对粟、[[黍|黍]]、[[御穀|御穀]]和其他一些粒小的[[杂粮|杂粮]]有统称，非农业专家一般不分，如英语均称“millet”。

==历史==
