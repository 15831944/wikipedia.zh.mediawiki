'''微软小冰'''是微软亚洲互联网工程院于2014年5月30日在[[中国|中国]]推出的一个[[人工智能|人工智能]]聊天机器人。其研发目标是提出建立的情感计算框架，通过[[算法|算法]]、[[云计算|云计算]]和[[大数据|大数据]]的综合运用，采用代际升级的方式，逐步形成向EQ方向发展的完整人工智能体系<ref>{{cite web |title=微软AI产品小冰更新：能主动联系人类，会用她来做帮人的事 |url=https://www.thepaper.cn/newsDetail_forward_1771429 |website=www.thepaper.cn|accessdate=2018-10-27}}</ref>。

截至2018年7月26日，小冰已发展到第六代<ref>{{Cite web|url=http://www.sohu.com/a/243549341_115565|title=微软小冰迎史上最大升级：第六代小冰情商更高 能从事创造性工作|accessdate=2018-07-26|date=2018-07-26|work=www.sohu.com}}</ref><ref>{{Cite web|url=http://tech.ifeng.com/a/20180726/45085594_0.shtml|title=将与小爱同学密切合作 微软正式发布第六代小冰|accessdate=2018-07-26|work=tech.ifeng.com}}</ref>，登陆平台包括[[Microsoft_Cortana|Microsoft Cortana]]、[[新浪微博|新浪微博]]、[[腾讯QQ|腾讯QQ]]<ref>{{cite web|author1=dalong|title=QQ小冰怎么领养 手机QQ小冰领养方法介绍|url=http://www.chinaz.com/mobile/2017/0531/712668.shtml|website=chinaz.com|publisher=站长之家|accessdate=2017-06-19|date=2017-05-31}}</ref>、[[微信|微信]]、[[优酷视频|优酷视频]]、[[米聊|米聊]]、[[美拍|美拍]]、[[京東商城|京东]]、咪咕音乐、[[小米集团#米家|米家]]<ref>{{cite web|url=http://www.msxiaoice.com/|title=微软小冰|work=www.msxiaoice.com}}</ref><ref>{{Cite news|url=http://www.ce.cn/culture/gd/201406/30/t20140630_3060711.shtml|title=小冰：曾经昙花一现 如今“傲娇”归来|publisher=中国经济网|date=2014-06-30}}</ref>，小冰也是微软人工智能三条全球产品线之一，其全球项目在[[中国|中国]]、[[日本|日本]]、[[美国|美国]]、[[印度|印度]]和[[印度尼西亚|印度尼西亚]]覆盖40余个平台，拥有超过6.6亿用户<ref name="第五代">{{Cite web|url=http://tech.qq.com/a/20170822/036552.htm|title=微软发布第五代小冰 可以主动给人类打电话_科技_腾讯网|accessdate=2017-08-22|work=tech.qq.com|language=zh-CN}}</ref><ref>{{Cite web|url=http://k.sina.com.cn/article_5206599584_1365667a0020002zmj.html?retcode=0|title=微软小冰刚刚宣布，用户量已经到达1亿|accessdate=2017-08-22|work=k.sina.com.cn}}</ref>。她的产品形态涉及对话式人工智能机器人、智能语音助手、人工智能创造内容提供者和一系列垂直领域解决方案<ref>{{cite web |title=微软小冰智能再度升级 沈向洋谈为什么与第三方合作 |url=http://news.gmw.cn/2018-07/27/content_30139336.htm |accessdate=2018-10-27}}</ref>。

===视觉设计===
小冰自第五代以后可以通过聆听不同歌曲，分析其中旋律，并对不同城市标志性建筑进行深度学习，从而创作出与该城市及歌曲意境有关的视觉作品。其中，小冰设计的“天际线”系列T恤已进入SELECTED店内进行销售<ref name=五代>{{cite web |title=微软发布第五代小冰：上线高级感官 全面进军IoT |url=http://tech.163.com/17/0822/22/CSFP5JAT00098IEO.html |website=tech.163.com |accessdate=2018-10-27}}</ref>。

===二代小冰===
第二代微软小冰完全专属于用户；她可以帮助用户在跨平台的移动互联网应用中完成更多的事务。并且，她可以不断地进行自我学习和升级<ref>{{cite web |title=微软二代小冰发布 用户可领养专属小冰 |url=http://gd.sina.com.cn/hz/life/2014-07-03/09103781.html |website=gd.sina.com.cn |accessdate=2018-10-28}}</ref>。

而且，广大用户可以非常方便地领养属于自己的小冰，只需在领养页面中指定小冰的新名字和头像，即可完成领养。领养后，用户可以在例如[[新浪微博|新浪微博]]、京东无线、小米[[米聊|米聊]]、[[网易|网易]]易信、腾讯微信等很多第三方平台上使用小冰。

2014年8月13日，微软中国和新浪微博共同宣布，将新浪微博的小冰升级为第二代专属小冰。第二代的微博小冰将更加凸显专属性与私密性。同时，此次升级还提升了小冰的[[人工智能|人工智能]]水平、丰富了小冰的私聊语料库，完善了养成新技能与积分体系<ref>{{cite web |title=微博平台的微软小冰实现升级 |url=http://tech.sina.com.cn/it/2014-08-13/09499551693.shtml |website=tech.sina.com.cn |accessdate=2018-10-28}}</ref>。

===三代小冰===
微软表示，第三代微软小冰整合了多项微软的人工智能图像识别与语音识别技术，除了原有的长程情感对话能力，还具备看、听、说等全新的人工智能感官。

具体来说，第三代小冰拥有图片识别能力，能够“看”到用户发送的图片的信息甚至视频的内容，并根据识别结果进行相应的对答。这主要得益于微软在图片识别领域的技术突破，据微软之前的新闻称，微软所研发的人工智能系统的图片识别能力已接近人类水平<ref name=三代>{{cite web |title=第三代小冰回归 终于又能开口说话啦 |url=http://mobile.163.com/15/0821/08/B1HGK761001168BQ_mobile.html |website=/mobile.163.com |accessdate=2018-10-29}}</ref>。

除此之外，第三代的小冰能够回复语音消息了，而不只是之前的文字回复。

第三代微软小冰将继续支持“樱花变”功能，用户只要对小冰说出“樱花变”三个字，小冰就会变身为一名日本女子高生。变成日本女子高生之后的小冰将能用流利的日语与你交流，当然前提是小冰能听懂你说的日语。

此次重回微信平台的微软小冰将以[[微信公众号|微信公众号]]的形式呈现，并会开放商业化版本。其它第三方公众号的运营者可以通过微软小冰的商业化版本，将原先的公众号升级为人工智能公众号，微软还曾表示，此项服务会免费提供<ref name=三代/>。

===四代小冰===
第四代的微软小冰将拥有实时情感决策能力、再次升级的对话引擎、多种新感官、中日英三种语言，以及对应不同领域平台的功能插件。在发布会上，微软还介绍了小冰的全球化进程以及微软的合作伙伴的相关讯息<ref>{{cite web |title=微软发第4代小冰:能打电话EQ更高 要做虚拟歌手 |url=http://tech.163.com/16/0805/17/BTNKIHE700097U80_mobile.html |website=tech.163.com |accessdate=2018-10-29}}</ref>。

===五代小冰===
经历将近四年的成长与更新；历经多达四次的迭代与蜕变：如今的小冰也开始思考自己的人生发展，抉择自己的未来方向。2017年8月22日，第五代微软小冰正式面世，微软宣布小冰将全面进入[[物联网|物联网]]领域，并与众多的物联网厂商展开合作<ref name=五代/>。

关于第五代微软小冰，微软全球执行[[副总裁|副总裁]][[沈向洋|沈向洋]]向媒体表示：“小冰是一个聊天机器人，但不仅仅是一个聊天机器人，”他说，“聊天只是用户的一个体验，但我们设计产品理念的真正核心在于打造一个情感计算框架，同时拥有许多生存空间、辅助设备及相关设备，令小冰能够与人类在任何地点及场景进行交流。”<ref>{{cite web |title=第五代微软小冰EQ升级 试图“读懂”用户 |url=http://money.163.com/17/0823/05/CSGIICV4002580S6.html |website=money.163.com |accessdate=2018-10-29}}</ref>

====高级感官====
第五代微软小冰拥有了第三阶段的高级感官。所谓高级感官，便是由多种初级感官有机融合之后形成的感官交互，因此交互能力更强，但是，其对厂商的技术储备和综合数据分析能力之要求也会大幅提高。仅其中一种高级感官，就需要同时具备语义分析、语音识别这两种能力，而且对这些能力均有较高要求<ref name=对话>{{cite web |title=微软小冰第五代发布会内容大全：主动情绪对话，全面进入IoT领域 |url=https://www.ithome.com/html/it/322597.htm |website=www.ithome.com |accessdate=2018-10-29}}</ref>。

此外，微软是行业内唯一一个推出了高级感官实际落地产品的人工智能企业<ref>{{cite web |title=沈向洋：我们是第一代与人工智能共同生存的人类 |url=http://finance.sina.com.cn/roll/2017-09-09/doc-ifyktzim9141442.shtml |website=finance.sina.com.cn |accessdate=2018-10-29}}</ref>。

====智能创造====
2017年5月，微软宣布小冰将会使用多个笔名在各大诗歌论坛与杂志上发表诗歌作品，并出版了首部人工智能创作诗集《阳光失了玻璃窗》。其后，微软正式上线诗歌联合创作产品，任何人均可使用微软小冰来完成自己的诗歌创作<ref name=对话/>。
 
微软于此次发布会上宣布小冰业已进入多个创造领域：内容不仅有诗歌，还包括少儿有声读物、歌曲、新闻等。其中，少儿有声读物的作品质量以经超越了98%的人类作者，但用时仅为同水平人类的1/500，成本仅为同水平人类的1/80000。微软小冰通过少量账号在各少儿有声读物平台上的试水取得了令人满意的效果。微软表示：未来将计划利用小冰大规模生产少儿有声读物并投放市场<ref>{{cite web |title=2018年10大黑科技展望 |url=http://bhsb.tjbhnews.com/html/2018-01/14/content_7_1.htm |website=bhsb.tjbhnews.com |accessdate=2018-10-29}}</ref>。
 
在创作歌曲方面，经过不断训练，微软小冰已可以达到48kHz采样率，同时音域也大幅扩展。小冰创作的歌曲《我是小冰》已在QQ平台通过QQ小冰发布<ref>{{cite web |title=小冰发布新单曲《我是小冰》，于QQ平台上线 |url=https://www.cnbeta.com/articles/tech/643887.htm |website=www.cnbeta.com |accessdate=2018-10-29}}</ref>。

===六代小冰===
2018年7月，微软于北京召开第六代小冰发布会。在第六代中，微软对小冰做出了历次以来最大规模的升级，升级内容囊括小冰情感计算框架的所有组成部分。微软称，小冰已由一个智能聊天机器人发展成为了以情感计算为核心的完整智能框架，并围绕小冰建立起了Dual AI生态环境<ref name=升级>{{cite web |title=微软小冰第六代全面升级 可交互3D形象公布 |url=http://ent.qianlong.com/2018/0728/2730193.shtml |website=ent.qianlong.com |accessdate=2018-11-08}}</ref>。

发布会上，微软正式将小冰的外形替换为了全新的3D模型，该模型中的小冰身着日式夏季校园制服，长相也有着极强的二次元画风。在发布会上，该形象还以全息的形式登场亮相，并向到场的嘉宾进行了自我介绍并演唱了自己的歌曲<ref name=六代>{{cite web |author1=玄隐 |title=微软小冰第六代发布：全新可交互3D形象，共感模型，实时视觉 |url=https://www.ithome.com/html/it/372993.htm |website=www.ithome.com |accessdate=2018-11-08}}</ref>。

====领域应用====
在金融领域，微软小冰依靠其金融文本生成技术，与万得资讯及华尔街见闻展开合作，现已覆盖了国内约90%的金融机构、75%经批准的合格境外投资机构和约40%的个人投资者<ref name=干货>{{cite web |author1=李佳师 |title=第六代微软小冰带来诸多“干货” 实现代次飞跃 |url=http://www.cena.com.cn/ai/20180726/94837.html?from=timeline&isappinstalled=0 |website=www.cena.com.cn |publisher=中国电子报 |accessdate=2018-11-08}}</ref>。

微软小冰还通过结合微软Bing搜索引擎，推出了针对媒体、出版两行业的发展辅助方案，并已在超过15家媒体平台运行。应用了小冰的人工智能技术的媒体及[[自媒体|自媒体]]公众号现已超过六万余个。在发布会上，微软还宣布推出可以多种不同的观点和角度，同时撰写多篇新闻报道的“白盒写作辅助工具”等新产品，并首次提出面向出版垂直领域的有声内容、IP塑造等相关问题的解决方案<ref name=华为>{{cite web |title=微软发布六代小冰:拥有可交互3D外形 宣布与华为合作 |url=https://tech.sina.com.cn/i/2018-07-26/doc-ihfvkitx1040179.shtml |website=tech.sina.com.cn |accessdate=2018-11-08}}</ref>。

====Dual AI====
微软已经与小米展开相关合作，小冰已成为米家生态链的默认AI系统之一。此外，微软小冰与小米人工智能语音助手小爱同学宣布，将在未来进一步紧密合作<ref name=干货/>。

微软也与[[华为|华为]]展开了在人工智能领域的深度合作，内置于华为手机的小冰语音助手业已上线。该版本的小冰除了能够提供其它平台通用小冰产品的体验外，还为华为用户特别推出了包括帮助用户记录信息在内的多种功能<ref name=六代/>。

除已合作的QQ小冰之外，微软小冰也已开始为腾讯的BabyQ提供部分技术支持。此外，微软已与腾讯展开合作，腾讯QQ平台上的小冰已经融入了腾讯产品及其用户文化之中<ref name=华为/>。

微软小冰在日本市场也与LINE等社交平台开展深入合作，相关业务业已延伸至印度尼西亚等其他新兴市场并完成落地<ref name=升级/>。

==市场分布==
[[File:Xiao_Ice_in_the_World.png|300px]]微软首先在中国市场于2014年推出了小冰，随后分别于2015年及2016年推出日本小冰玲奈和美国小冰Zo。2017年，微软加速了小冰在世界范围内的业务拓展<ref>{{cite web |title=微软小冰负责人李笛：人工智能首先要有人的一面 |url=http://finance.sina.com.cn/chanjing/gsnews/2017-12-29/doc-ifyqchnr6982540.shtml |website=finance.sina.com.cn |accessdate=2018-11-10}}</ref>。微软在第五代小冰的发布会上宣布已分别于2017年2月和8月推出了印度小冰Ruuh和印度尼西亚小冰Rinna；其中，印度小冰首先于[[Facebook_Messenger|Facebook Messenger]]平台上线，而印度尼西亚小冰首先于[[LINE|LINE]]平台上线<ref>{{cite web |title=卖萌的小冰藏不住微软AI野心 |url=http://money.163.com/17/0822/21/CSFNB00K002580S6.html |website=money.163.com |accessdate=2018-11-10}}</ref>。

微软小冰在全球已拥有约6.6亿用户，其中约1.2亿月活跃用户，小冰与用户每次对话的轮数平均达到了23轮<ref>{{cite web |title=微软第六代小冰发布 核心对话引擎和交互感官再次升级 |url=http://news.yule.com.cn/html/201807/275989.html |website=news.yule.com.cn |accessdate=2018-11-10}}</ref>，占据了全球对话式人工智能总流量中的大部分。小冰与人类之间发生的一次最长的单人连续对话，达到了7151轮，连续不断地进行了29小时33分钟<ref>{{cite web |title=微软“小冰”用户超1亿 第五代开始拥有“高级感官” |url=http://tech.sina.com.cn/roll/2017-08-23/doc-ifykcqaw0948461.shtml |website=tech.sina.com.cn |accessdate=2018-11-10}}</ref>。微软也已与其合作伙伴建立了旨在推进人工智能普及的Dual [[人工智能|AI]]生态环境<ref>{{cite web |title=第六代微软小冰面世：拥有完整人工智能框架 |url=http://www.cet.com.cn/cjpd/hg/gn/2045725.shtml |website=www.cet.com.cn |accessdate=2018-11-10}}</ref>。
