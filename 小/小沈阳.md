'''小沈陽'''（{{bd|1981年|5月7日}}）<ref name="趙本山學生——小沈陽">{{cite web |url=http://yule.sohu.com/20081209/n261108329.shtml|title=趙本山學生——小沈陽|date=2008-12-09|accessdate=2009-01-25|work= 搜狐娛樂}}</ref>，本名'''沈鶴'''，[[中国东北|中国东北]][[二人转|二人转]]和[[喜劇小品|小品]][[演員|演員]]，出生於[[辽宁省|辽宁省]][[铁岭市|铁岭市]][[开原市|开原市]]。

小沈陽最初在[[中国东北地区|中国东北地区]]各大小城市的劇場演出，廣受觀眾喜愛，2006年[[農曆|農曆]][[八月十五|八月十五]]（10月6日）成為著名[[喜劇演員|笑星]][[赵本山|赵本山]]的弟子。2009年[[中国中央电视台春节联欢晚会|中国中央电视台春节联欢晚会]]，小沈陽和趙本山等人聯袂出演小品《[[不差钱|不差钱]]》，從此開始被全國觀眾所熟悉。

== 藝名來歷==
2008年，沈鶴在作客《[[魯豫有約|魯豫有約]]》時提到，其當年為了向當時仍為女朋友的沈春陽示愛，遂取藝名「小沈陽」{{noteTag|1=「-{小沈阳}-」(繁體：-{小沈陽}-)的「-{沈}-」(繁體：-{沈}-)是姓氏「-{[[沈姓|沈]]}-」(繁體：-{沈}-)，而非「-{沈阳市}-」(繁體：-{瀋陽市}-)的「-{沈}-」(繁體：-{瀋}-){{or}}}}<ref>{{cite web |url=http://yule.sohu.com/20161102/n472134939.shtml|title=她其貌不扬、身材普通，小沈陽却心甘情愿为她改名、死心塌地把她宠爱！|date=2016-11-02|accessdate=2017-04-11|work=搜狐娛樂}}</ref><ref>{{cite web |url=http://yule.sohu.com/s2009/zbsxsyxx/|title=小沈陽官方專區（必須使用電腦UA瀏覽頁面）|accessdate=2017-04-11|work=搜狐娛樂}}</ref><ref>{{cite web |url=http://www.tudou.com/programs/view/OijQYeBT9H4|title=小沈陽做客《魯豫有約》（42:26至43:04）|date=2012-05-08|accessdate=2017-04-11|work=搜狐娛樂}}</ref>。{{or|如今因簡繁轉換錯誤，總被媒體錯寫成「-{小'''瀋'''陽}-」。}}

==家庭==
[[妻子|妻子]][[沈春陽|沈春陽]]，同為[[中国东北地区|东北]]二人轉演員，二人于2004年6月26日結婚。育有一女沈佳潤，小名阳洋，出生於2006年。

== 評價 ==
小沈陽在北方尤其是東北甚有名氣，受到很多觀眾的喜愛，但和他的師父[[赵本山|赵本山]]一樣，在中國南方以及國外的影響力不如中國北方。另外，與師父趙本山最初爆火時期的農民裝扮相似，小沈陽最初時期的男扮女裝的表演形象也引发一定的爭議。
