'''小池百合子'''（{{jpn|j=小池 百合子|hg=こいけ ゆりこ|rm=Koike Yuriko|f=y}}；{{bd|1952年|7月15日|catIdx=Koike Yuriko}}）是[[日本|日本]][[政治家|政治家]]，[[埃及|埃及]][[開羅大學|開羅大學]][[社會學|社會學]]學士，精通[[阿拉伯語|阿拉伯語]]與[[英語|英語]]，現任[[東京都知事|東京都知事]]，也是首位擔任此職的女性；[[新聞主播|新聞主播]]出身，1992年從政後歷任[[參議院_(日本)|參]][[眾議院_(日本)|眾]]兩院[[國會議員|議員]]，以及內閣[[環境大臣|環境大臣]]、[[防衛大臣|防衛大臣]]等要職，是[[日本歷史|日本歷史]]上首位[[女性|女性]]防衛大臣。此外，小池過去曾加入多個政黨，現時為其創立的兩個政黨——[[都民第一會|都民第一會]]的特別顧問、以及[[希望之黨|希望之黨]][[黨魁|黨魁]]。

== 生平 ==
出身于[[兵庫縣|兵庫縣]][[蘆屋市|蘆屋市]]。[[甲南女子中学校|甲南女子中学校]]、[[甲南女子高等学校|甲南女子高等学校]]毕业。1971年（昭和46年）9月[[關西學院大學|關西學院大學]][[社會學|社會學]]部中途退学。1976年（昭和51年）以最优异成绩畢業於[[開羅大學|開羅大學]]社會學部（社会学系），同时也是该大学获取社会学学士学位的第一位女性，通曉[[阿拉伯語|阿拉伯語]]。1977年，成為阿拉伯语的讲师，並在1993年成為日本阿拉伯協會（{{lang|ja|日本アラブ協会}}）顧問。

1992年（平成4年），小池代表[[日本新黨|日本新黨]]當選[[日本參議院|日本參議院]]議員。

2002年（平成14年）12月，小池加入[[自由民主黨_(日本)|自民黨]]。曾經擔任[[日本|日本]][[小泉純一郎|小泉]]內閣的[[環境大臣|環境大臣]]，其後在小泉三次改組內閣時都獲留任。

在2005年（平成17年）[[2005年日本大选|第44屆日本眾議院大選]]中，小池以空降「刺客」身分擊敗[[鮫島宗明|鮫島宗明]]、[[小林興起|小林興起]]等人當選眾議院議員。

2007年（平成19年）7月4日，小池就任[[安倍晉三|安倍第一次內閣]]的防衛大臣，成為首任女性[[防衛大臣|防衛大臣]]。小池上任後，日本防衛省傳出洩漏美國[[神盾艦|神盾艦]]機密醜聞，且在防衛省官員任命案中，和自民黨大老意見相左而捲入政壇內鬥；因此在首相[[安倍晉三|安倍晉三]]重組內閣時，小池即表示不願留任。最後，小池在2007年（平成19年）8月27日發布的新內閣名單後卸任，上任僅54天。

2016年（平成28年）7月31日，小池以291萬2628票當選[[东京都知事|东京都知事]]，成为首位女性都知事。在同年8月2日就職。其後小池退出[[自由民主黨_(日本)|自民黨]]。2017年1月，小池參與創建地方型政黨「都民第一會」，在2017年7月舉行的[[2017年東京都議會議員選舉|東京都議會選舉]]，大獲全勝，使得對手自由民主黨落到二次大戰後議席數最低的一次，東京自民黨吃了次歷史級的大敗仗。

2017年9月25日，面對首相安倍所宣告的[[眾議院解散|解散眾議院]][[第48屆日本眾議院議員總選舉|提前改選]]，小池一方面批評安倍這次解散沒有大義，同時又加緊國政政黨的籌備腳步。之後小池組成了新的國政型政黨「[[希望之黨|希望之黨]]」。

== 政策及主張==
*'''支持利比亞'''：在[[利比亞|利比亞]]前任領導人[[格達費|格達費]]領導時期就與利比亞有交流關係，支持日本應該與利比亞建立交流，與格達費有多年的直接會談交情，在[[利比亞內戰|利比亞內戰]]時期依然堅定交流立場<ref>[http://www.jlfa.gr.jp/association_2.html jlfa.gr.jp]</ref>。
*'''節能減碳'''：小池長期重視環保，耕耘草根婦女運動及環境運動。由於擔任多屆環境大臣，小池除親自設計多款環保產品之外，最為人所津津樂道的是，鼓吹夏季不穿西裝、不打領帶的[[Cool_Biz|Cool Biz]]運動，包括當年首相[[小泉純一郎|小泉純一郎]]、[[豐田汽車|豐田汽車]]總裁[[奧田碩|奧田碩]]等日本知名賢達，紛紛響應號召，在夏天脫下西裝外套與領帶，以節約空調電耗，一時蔚為風尚。[[核能|核能]]議題上認為只要安全科技重新強化後應該繼續核能路線。
:進入九〇年代以後，日本社會開始將降低能耗與保護環境結合起來。1997年12月通過的《[[京都議定書|京都議定書]]》呼吁減少二氧化碳等溫室氣體的排放。為此，日本從政府機構、企業，到學校、團體和家庭紛紛採取各種手段，省能節能，降低溫室氣體的排放。2005年夏天小泉內閣環境大臣小池百合子倡議職員上班時不戴領帶、不穿上裝，以適應空調制冷溫度調高後的辦公室環境。政府機構和企業單位群起響應。據第一生命綜合研究所測算，這項措施的節能效果就達1100億日元。
*'''電線地下化'''：支持電線地下化，去除[[電線桿|電線桿]]。
*'''雙層巴士'''：支持採購英國式[[雙層巴士|雙層巴士]]，去除上下班通勤擁擠問題。
*'''女性宮家'''：支持皇室創設女性[[宮家|宮家]]。<ref>[http://senkyo.mainichi.jp/46shu/kaihyo_area_meikan.html?mid=A13010003003 mainichi.jp/46shu]</ref>

== 軼事 ==
* 由於歷經[[日本新黨|日本新黨]]、[[新進黨|新進黨]]、[[自由黨_(日本1998年)|自由黨]]、[[保守新黨|保守黨]]、[[自由民主党_(日本)|自由民主黨]]等五個政黨，被日本政界稱為「政治渡鳥」，但如同另一政黨領袖[[小澤一郎|小澤一郎]]一樣，被批評為政治[[投機主義|投機主義]]。
* 2006年時因[[肺炎|肺炎]]住院而發熱至攝氏40度，住院兩週以上。
