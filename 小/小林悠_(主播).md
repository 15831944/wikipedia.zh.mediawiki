'''小林 悠'''（{{jpn|kn=こばやし はるか|rm=Kobayashi Haruka|f=h}}，{{bd|1985年|9月26日|||kobayashi}}）是日本女性[[播报员|播报员]]，曾在2010年至2016年担任[[TBS電視台|TBS電視台]]主播。

== 经历 ==
她出身于[[北海道|北海道]][[札幌市|札幌市]]，是[[混血儿|混血儿]]，父亲是[[美国|美国]]人，母亲是[[日本人|日本人]]。父母都是大学教授，[[外祖父|外祖父]]是物理学家[[小林禎作|小林禎作]]（原[[北海道大學|北海道大學]]教授）<ref>{{cite web |title=2月8日（金）小林悠×玉袋筋太郎 |url=https://www.tbsradio.jp/tama954/2013/02/post-386.html |publisher=たまむすび |accessdate=2019-02-09 |archiveurl=https://web.archive.org/web/20180724133111/https://www.tbsradio.jp/tama954/2013/02/post-386.html |archivedate=2018-07-24}}</ref>。[[东京都|东京都]][[御茶水女子大學|御茶水女子大學]]在学期间是{{tsl|ja|セント・フォース|Cent FORCE}}所属的[[日本電視藝人|電視藝人]]和自由播报员<ref>{{cite web |title=『NEWS23』で復帰報道の雨宮塔子 TBSの起死回生なるか |url=http://www-origin.zakzak.co.jp/smp/entertainment/ent-news/news/20160704/enn1607041542012-s1.htm |publisher=ZAKZAK |accessdate=2019-02-09 |date=2016-07-04}}</ref>。[[2010年|2010年]]4月加入[[TBS电视台|TBS电视台]]，担任[[晨间新闻|晨间新闻]]节目主播。2014年10月任《报道特集》新闻节目主播<ref>{{cite web |title=ＴＢＳ小林悠アナ、退社　「健康上の理由」と説明したが… |url=https://www.zakzak.co.jp/entertainment/ent-news/news/20160304/enn1603041700018-n1.htm |accessdate=2019-02-09|publisher=ZAKZAK|date=2016-03-04}}</ref>。

2016年2月1日[[小学館|小学館]]《[[周刊邮报|周刊邮报]]》上刊载了小林悠与某男性[[實業家|實業家]]在某[[酒店|酒店]]约会的[[不伦|不伦]]消息<ref>{{cite web |title=【小林悠さん】元TBSアナが「適応障害」だったと週刊文春に告白、どんな病気？ |url=https://www.huffingtonpost.jp/2016/03/15/yu-kobayashi_n_9467482.html |publisher=[[HuffPost|HuffPost]] |accessdate=2019-02-09 |language=ja |date=2016-03-15}}</ref>，之后她便精神不振，从2月26日起不再参与应由她主持的直播[[綜藝節目|綜藝節目]]<ref>{{Cite news|url=http://www.sponichi.co.jp/entertainment/news/2016/03/04/kiji/K20160304012149640.html|title=なぜ？小林悠アナ失意の退社　デート報道の男性に“二股疑惑”|newspaper=Sponichi Annex|publisher=[[體育日本|體育日本]]|date=2016-03-04|accessdate=2016-03-05}}</ref><ref>{{Cite news|url= http://www.sponichi.co.jp/entertainment/news/2016/03/04/kiji/K20160304012152130.html?feature=related|title=小林悠アナ TBS退社正式発表　健康上の理由で４日付依願退職|newspaper=Sponichi Annex|publisher=スポーツニッポン|date=2016-03-04|accessdate=2016-03-05}}</ref>。TBS电视台播报员分部官方[[網誌|網誌]]「小林悠的悠草子」也在2月中旬关闭<ref>{{Cite news|url= http://www.sponichi.co.jp/entertainment/news/2016/03/04/kiji/K20160304012151230.html|title=小林悠アナ　公式ブログが削除・閉鎖　ＴＢＳ退社と関係？ |newspaper=Sponichi Annex|publisher=スポーツニッポン|date=2016-03-04|accessdate=2016-03-05}}</ref>。3月4日，她以健康状况不良为由从TBS电视台离职<ref>{{Cite news|url= http://www.daily.co.jp/newsflash/gossip/2016/03/04/0008860264.shtml |title= 小林悠アナがTBSを依願退職 |newspaper= デイリースポーツ online |publisher= 株式会社デイリースポーツ |date= 2016-03-04 |accessdate= 2016-03-04}}</ref><ref>{{Cite news|url= http://www.nikkansports.com/entertainment/news/1612393.html |title= TBS小林悠アナ「健康上の理由」での退社発表 |newspaper= ニッカンスポーツ・コム |publisher= 日刊スポーツ新聞社 |date= 2016-03-04 |accessdate= 2016-03-04}}</ref>。

3月16日，她在《[[週刊文春|週刊文春]]》坦言，那位先生目前单身，是她非常珍重的人，周刊邮报的报道让她的身心深受打击。2月10日，她已经被[[身心医学|心療内科]]诊断为{{tsl|en|Adjustment disorder|适应性障碍}}<ref>{{cite web |title=小林悠TBS元アナが初告白「私は適応障害でした」 |url=https://web.archive.org/web/20160429051515/http://shukan.bunshun.jp/articles/-/5972 |publisher=[[週刊文春|週刊文春]] |accessdate=2016-04-29 |date=2016-03-15}}</ref>。

她原预计参加2016年4月改编后的《[[NEWS23|NEWS23]]》节目[[新闻主播|新闻主播]]，但由于上述事件的缘故，职位由{{tsl|ja|駒田健吾||駒田健吾}}和[[皆川玲奈|皆川玲奈]]代替<ref>{{cite web|url=http://www.sponichi.co.jp/entertainment/news/2016/03/03/kiji/K20160303012145380.html|title=「ＮＥＷＳ２３」小林悠アナ加入取りやめ　健康上の理由|publisher=スポニチアネックス|date=2016-03-03|accessdate=2016-03-03}}</ref>。之后，小学馆在7月7日发行的《女性Seven》杂志上刊载了相关的致歉文章<ref>{{cite web |author1=新田龍 |title=捏造発覚のTBS、謝罪文で「演出の一環」「行き過ぎた編集」と強弁…ヤラセ横行の裏事情 |url=https://biz-journal.jp/2016/07/post_15869.html |publisher=Business Journal |accessdate=2019-02-09 |archiveurl=https://web.archive.org/web/20171024020401/http://biz-journal.jp/2016/07/post_15869.html |archivedate=2017-10-24 |language=ja |date=2016-07-12}}</ref>。
