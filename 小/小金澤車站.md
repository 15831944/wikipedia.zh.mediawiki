'''小金澤車站'''（{{jpn|j=小金沢駅|hg=こがねざわえき|rm=Koganezawa eki|f=y}}）是一位於[[日本|日本]][[宮城縣|宮城縣]][[氣仙沼市|氣仙沼市]]本吉町、[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[氣仙沼線|氣仙沼線]]沿線的[[鐵路車站|鐵路車站]]。小金澤是一由[[氣仙沼車站|氣仙沼車站]]代為管理的無人車站，屬[[東日本旅客鐵道盛岡支社|JR東日本盛岡支社]]的管轄範圍。

== 車站構造 ==
[[側式月台|側式月台]]1面1線的[[地面車站|地面車站]]。[[氣仙沼站|氣仙沼站]]管理的[[無人站|無人站]]。

== 歷史 ==
*1957年（[[昭和|昭和]]32年）2月11日 - 車站啟用。
*1987年（昭和62年）4月1日 - [[日本國鐵分割民營化|日本國鐵分割民營化]]，車站管轄劃歸由東日本旅客鐵道繼承。
* 2011年（[[平成|平成]]23年）3月11日 - 東北地方太平洋近海發生[[2011年日本東北地方太平洋近海地震|嚴重地震]]，地震引發的[[海嘯|海嘯]]沖毀靠近海岸的許多鐵路設施，小金澤也是23個被沖毀的鐵路車站之一<ref>{{cite web |url=http://www.sankeibiz.jp/business/news/110401/bsd1104010801004-n1.htm |language=ja |title=<span lang="ja">【東日本大震災】ＪＲ東日本、太平洋沿岸の２３駅流失</span> |publisher=[[產經新聞社|產經新聞社]] |date=2011-04-01 |accessdate=2011-04-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20110413050421/http://www.sankeibiz.jp/business/news/110401/bsd1104010801004-n1.htm |archivedate=2011-04-13 }}</ref>。
