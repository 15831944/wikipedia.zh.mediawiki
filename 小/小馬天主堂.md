'''小馬天主堂'''位於臺灣[[臺東縣|臺東縣]][[成功鎮|成功鎮]]的小馬部落{{NoteTag|據安倍明義之調查，阿美族語原稱「Tsuratsutsuratsu」{{r|東海岸}}。清朝因視該部落為大馬武窟社的分社，稱為「小馬武窟社」，又有寫成「小馬武吻社」{{r|東海岸}}。日昭和十二年（1937年）實施地名簡化，地名改成「小馬」{{r|東海岸}}。當地除天主教堂外，還有[[臺灣基督長老教會|長老教會]]與[[真耶穌教會|真耶穌教會]]的教堂{{r|東海岸}}。}}，是天主教[[白冷外方傳教會|白冷會]]所有的教堂，正式名稱是'''小馬聖尼克老天主堂'''，又稱'''聖尼各老堂'''{{r|東海岸}}{{r|走讀}}，於2005年9月29日公告為歷史建築{{NoteTag|《日出台東 東海岸文化景觀》與《走讀台灣：台東縣 2》則是寫民國92年（2003年）列為歷史建築{{r|東海岸}}{{r|走讀}}。}}{{r|文}}。該教堂被視為臺東的經典建築，有建築雜誌加以報導{{r|東海岸}}{{r|走讀}}。

== 沿革 ==
民國43年（1954年）姚秉彝神父將傳道翻譯員張明祥（Loma'ni Takofan）家作為小馬的宣教會所，舉行主日彌撒{{r|東海岸}}。民國45年（1956年）3月7日，曹經五神父到成功堂區擔任副堂，主日或平日會到小馬宣教和舉行彌撒{{r|東海岸}}。小馬部落的信教者日漸增加{{r|東海岸}}。民國46年（1947年），小馬地區部落興建教堂，工程有不少教友參與{{r|東海岸}}。

民國50年（1961年），瑞士籍神父池作基（Rev. Meinard Tschirky，1930-1992）來到東和堂區宣教{{r|東海岸}}{{r|文}}。之後民國56年（1967年）小馬天主堂成立儲蓄互助社，該年白冷會也向臺東縣政府購買公有地以興建新教堂{{r|東海岸}}。民國57年（1968年）因教徒日增，池作基神父開始教堂的籌建{{r|東海岸}}{{r|文}}。經費來自池作基神父的國外募款，以及教友變賣稻穀所得{{r|東海岸}}{{r|文}}。新聖堂於該年（1968年）8月20日落成，由費聲遠主教祝聖啟用，稱「天主教小馬聖尼各老堂（Hsiaoma, St. Nicolaus）」{{r|東海岸}}{{r|文}}。新聖堂的設計者是[[傅義|傅義]]（Julius Felder）修士，小馬天主堂（聖尼各老堂）是他來臺後的第三件作品{{r|東海岸}}{{r|走讀}}{{r|文}}。

民國74年（1985年）因應臺東市第二公墓遷葬作業，教堂後方設立墓園{{r|東海岸}}{{r|文}}。墓園中葬有池作基、紀守常（Rev. Alfred Giger，1919-1970）、周維道（Rev. Viktor Notter，1906-1992）、郝道永（Rev. Friedrich Hort，1908-1985）、滿海德（Rev. Ernst Manhart，1905-1991）等諸位神父{{r|東海岸}}{{r|文}}。

== 建築特色 ==
小馬天主堂的特色是外觀有類似[[扶壁|扶壁]]的結構，而外牆使用自然石板材，因有不規則線條而使得牆壁看起來像是用真石砌成{{r|東海岸}}{{r|走讀}}{{r|文}}。此外教堂錐狀屋頂（傳義修士稱斗笠屋頂）的採光天窗能將自然光源大量引進，照耀在祭臺上，營造神聖莊嚴的氣氛{{r|東海岸}}{{r|走讀}}{{r|文}}。
