== 暱稱 ==
{{mp|2007 OR|10}}是由[[加州理工學院|加州理工學院]]的天文學家發現，該小行星的發現相關資訊成為[[米高·E·布朗|米高·E·布朗]]的博士生梅格·史康瓦博（Meg E. Schwamb）博士論文的一部分<ref name="scida2011">{{cite web| url= http://www.sciencedaily.com/releases/2011/08/110822124955.htm | title= Astronomers Find Ice and Possibly Methane On Snow White, a Distant Dwarf Planet | publisher= [[Science_Daily|Science Daily]]| date= August 22, 2011 | accessdate=2011-08-22}}</ref>。因為布朗認為該天體可能體積極大或非常亮而在該次巡天中被偵測到<ref name="Brownblog">{{cite web
 |date        = 2009-03-10
 |title       = Snow White needs a bailout
 |publisher   = Mike Brown's Planets (blog)
 |author      = [[米高·E·布朗|Michael E. Brown]]
 |url         = http://www.mikebrownsplanets.com/2009/03/snow-white-needs-bailout.html
 |accessdate  = 2010-02-17
 |archiveurl  = https://www.webcitation.org/5gpP1IbCr?url=http://www.mikebrownsplanets.com/2009/03/snow-white-needs-bailout.html
 |archivedate = 2009-05-17
 |deadurl     = no
}}</ref>，並假設它的顏色是偏白色<ref name="scida2011"/>，而為該天體取暱稱「[[白雪公主|白雪公主]]」（Snow White）。但後來發現它是柯伊伯帶中最偏紅色的天體之一，只有[[创神星|创神星]]的偏紅程度可以與它相比，因此前述的暱稱也就不是那麼適合了。不過布朗的團隊仍將它稱為「第七個矮人」（seventh dwarf），因為創神星發現於2002年、賽德娜發現於2003年、妊神星和[[亡神星|亡神星]]發現於2004年，而[[鳥神星|鳥神星]]和[[鬩神星|鬩神星]]發現於2005年。

{{mp|2007 OR|10}}是目前太陽系中尚未被正式命名的最大天體。2011年布朗認為他終於對該天體有足夠的資訊，因此決定給該天體一個正式名稱<ref name="Brown2011"/>，因為該天體發現了水冰，並可能有甲烷存在，使它被認為是有研究價值的天體<ref name="scida2011"/>。不過，布朗直到2015年都還未提出一個命名的建議，即使他指出直到2017年每個人都可以提出命名的建議<ref>{{cite web|url=http://www.mikebrownsplanets.com/2015/01/ten-years-of-eris.html?showComment=1423266081122#c5069655186376307635|title=Mike Brown's Planets|work=mikebrownsplanets.com}}</ref>。

==距離==
{{mp|2007 OR|10}}大約在1857年到達[[拱點|近日點]]<ref name="jpldata"/>。它目前距離太陽87.5天文單位<ref name=AstDyS-distances>{{cite web |title=AstDyS-2, Asteroids - Dynamic Site |url=http://hamilton.dm.unipi.it/astdys/index.php?pc=3.2.1&pc0=3.2&sb=8&ldfs=59 |quote=Objects with distance from Sun over 59 AU |date=2016-02-26 |accessdate=2016-02-29}}</ref>，並且相對太陽的軌道速度為每秒2.7公里<ref name="horizons">{{cite web
  |url=http://ssd.jpl.nasa.gov/horizons.cgi?find_body=1&body_group=sb&sstr=2007OR10
  |title=Horizon Online Ephemeris System
  |publisher=California Institute of Technology, Jet Propulsion Laboratory

==絕對星等==
行星和矮行星的[[絕對星等|絕對星等]]與天體的體積和[[反照率|反照率]]相關。{{mp|2007 OR|10}}的絕對星等是2.0等<ref name="jpldata"/>，因此是目前已知第五亮的[[海王星外天體|海王星外天體]]<ref name=BrownRedemption2>{{cite web|title=The Redemption of Snow White (Part 2 of 3)|author=Mike Brown| publisher=Mike Brown's Planets|url=http://www.mikebrownsplanets.com/2011/08/redemption-of-snow-white-part-2.html|year=2011|accessdate=2012-02-08}}</ref>，低於賽德娜的1.6等（直徑約1000公里）<ref name="Sednajpl">{{cite web

  |date=2008-12-04 last obs

{{mp|2007 OR|10}}是目前已知最偏紅的海王星外天體之一<ref name="Brown2011">{{cite web

==表面組成和大氣==
{{mp|2007 OR|10}}的光譜顯示了水冰和甲烷的存在，因此與創神星的狀況類似。在{{mp|2007 OR|10}}和創神星表面發現紅色的甲烷霜暗示了這兩顆小天體存在正逐漸散逸入太空中的稀薄甲烷大氣層。雖然{{mp|2007 OR|10}}比創神星接近太陽，因此應會有較高的表面溫度使甲烷被蒸发進入太空，它較大的質量是可能保存大氣層的原因<ref name="Brown2011"/>。水冰在{{mp|2007 OR|10}}的表面被發現也暗示它在早期歷史中有[[冰火山|冰火山]]存在<ref name=BrownRedemption3>{{cite web|title=The Redemption of Snow White (Part 3 of 3)|author=Mike Brown| publisher=Mike Brown's Planets|url=http://www.mikebrownsplanets.com/2011/08/redemption-of-snow-white-part-3-of-3.html#more|year=2011|accessdate=2011-08-23}}</ref>。

==軌道半長軸和軌道週期==
[[深度黃道巡天|深度黃道巡天]]的資料顯示它是在和海王星軌道[[共振海王星外天體|3:10共振]]的狀態下<ref name="Buie"/>。[[小行星中心|小行星中心]]將它列為[[離散盤|離散盤]]天體<ref name=MPC/>。{{mp|2007 OR|10}}是在賽德娜所在區域進行巡天時發現的<ref name="Schwamb2009">{{cite journal
  |last=Schwamb |first= Megan E.
  |coauthors=Michael E. Brown, David L. Rabinowitz
  |title=A Search for Distant Solar System Bodies in the Region of Sedna
  |journal=[[Astrophysical_Journal_Letters|Astrophysical Journal Letters]]
  |year=2009
  |doi=10.1088/0004-637X/694/1/L45
  |arxiv=0901.4173
|bibcode = 2009ApJ...694L..45S }}</ref>。曾经的分类将所有与海王星共振的天体都归于[[柯伊伯带|柯伊伯带]]，但2007OR10的轨道特征（除了共振）属于典型的[[離散盤|離散盤]]天體，因此它将可能是第一个被确认的共振离散盘天体。


自1985年起至被發現前的影像資料中，{{mp|2007 OR|10}}已經被觀測到46次，並且其中7次是它位於[[衝_(天體位置)|衝]]的時候<ref name="jpldata"/>。

== 矮行星地位 ==
如果一個海王星之外的[[矮行星候選者列表|矮行星候選者]]沒有軌道被精確測定的天然衛星，天文學家將無法直接計算它的質量（參見[[三體問題|三體問題]]）以確定是否處於[[流體靜力平衡|流體靜力平衡]]狀態。當時{{mp|2007 OR|10}}沒有已知的衛星，尚無法得知其質量。

基於2016年5月的體積推測<ref name="scida2015" />，{{mp|2007 OR|10}}是已知第三大的矮行星，小於冥王星和[[阋神星|阋神星]]，大於鳥神星、[[妊神星|妊神星]]和穀神星，儘管誤差範圍和鳥神星重疊。[[國際天文聯會|國際天文聯會]]在{{mp|2007 OR|10}}宣布發現後尚未討論增加矮行星的可能性。布朗的個人網頁稱該天體「一定是矮行星，即使它主要以岩石構成」<ref name="Brown-dplist">{{cite web

== 衛星 ==
2016年，天文學家分析哈伯太空望遠鏡於2010年拍攝的{{mp|2007 OR|10}}影像，發現它有一顆直徑約300公里，距離至少15000公里的衛星。該衛星於2016年10月17日美國天文學會行星科學部的會議（DPS48）上宣布發現<ref name = "Lakdawalla2016a" /><ref name="DPS48">{{cite web
  |title      = The moon of the large Kuiper-belt object 2007 OR10
  |publisher  = DPS48 120.22
  |url        = https://aas.org/files/dps-epsc-abstract-book-final.pdf
  |accessdate = 17 October 2016}}</ref>。該衛星因體積過小，且表面亮度低，可能對矮行星體積估計造成影響<ref name = "Lakdawalla2016a" />。
