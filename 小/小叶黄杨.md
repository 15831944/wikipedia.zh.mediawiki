'''小叶黄杨'''（[[学名|学名]]：{{lang|la|''B. s.'' var. ''parvifolia''}}）是[[黄杨科|黄杨科]][[黄杨属|黄杨属]]的一种植物。

== 形态 ==
常绿[[灌木|灌木]]或小[[乔木|乔木]]，高1到3米。小枝有短柔毛。[[茎|茎]]枝呈四棱。宽椭圆形或宽倒卵形革质的叶子对生，钝头或顶上微凹缺，叶子1到3厘米。春季开花，黄绿色，雌雄同株，簇生。近球形的[[蒴果|蒴果]]，短角状突起，果熟期7月，熟时裂为三瓣。

== 分布 ==
原产[[中国大陆|中国大陆]]中部与北部。分布在中国大陆的[[安徽|安徽]]、[[广西|广西]]、[[四川|四川]]、[[江西|江西]]、[[浙江|浙江]]、[[贵州|贵州]]、[[甘肃|甘肃]]、[[江苏|江苏]]、[[广东|广东]]、[[山东|山东]]、[[湖北|湖北]]、[[陕西|陕西]]等地，生长于海拔1,200米至2,600米的地区，多生于[[山谷|山谷]]、溪边和林下，目前各省均有栽培作观赏用。

== 用途 ==
小叶黄杨对[[有毒气体|有毒气体]]有很强的抗性，并且能够净化被污染的空气。<ref name=" bookh" >{{cite book |language=zh-cn|author=布里克尔 |title=《世界园林植物与花卉百科全书》|year=2006年 |publisher=河南科学技术出版社 |location=河南省 |pages= |isbn=9787534953675 }}</ref>

用在医药，祛[[风湿|风湿]]，理气止痛。用于风湿疼痛、跌打损伤。

== 繁殖 ==
小叶黄杨以[[播种|播种]]为主，也可以[[扦插|扦插]]。播种一般是采撷其果实，果壳开裂之后干藏，3月中旬播种。扦插则是在5月到6月之间。<ref name=" bookh" />
