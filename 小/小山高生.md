'''小山高生'''{{BD|1948年|4月21日||}}，[[日本|日本]]資深男[[編劇|編劇]]、[[作家|作家]]。出身於[[東京都|東京都]][[昭島市|昭島市]]{{來源請求}}。

== 簡介 ==
現職[[大阪藝術大學|大阪藝術大學]]角色造形學科教授。前編劇集團「'''{{lang|ja|ぶらざあのっぽ}}'''」主宰（此外這裡也記述{{lang|ja|[[小山高生#ぶらざあのっぽ|ぶらざあのっぽ]]}}的介紹）。[[早稻田大學|早稻田大學]]文學部畢業。

主要參加[[動畫|動畫]]系列的[[編劇|編劇]]、[[劇本統籌|劇本統籌]]工作。1987年10月以前用本名'''小山高男'''持續參加編劇。自稱「[[亞洲|亞洲]]最大的劇作家（身高上）」

=== 來歷、人物 ===
小山於[[早稻田大學|早稻田大學]]就讀期間，曾經在[[朝日電視台|朝日電視台]][[打工|打工]]，至於的工作內容是幫益智猜謎節目《{{link-ja|Quiz TimeShock|クイズタイムショック}}》進行出題。1972年3月，於大學畢業之後，立即進入動畫公司[[龍之子製作公司|龍之子製作公司]]（以下簡稱龍之子）上班，並調任至企劃文藝部。從此以後，小山參加過龍之子製作動畫作品的企劃和編劇工作。此外於負責〈{{link-ja|時間飛船系列|タイムボカンシリーズ}}{{refnest|group="注"|小山在《[[小雙俠|小雙俠 (1977年版)]]》製作期間，曾因生病無法參加編劇而缺席。還有在製作《{{link-ja|頂尖超人|イタダキマン}}》期間，小山因為與[[富士電視台|富士電視台]]所屬製作人[[岡正|岡正]]2人的關係產生摩擦，因此小山在該部電視動畫只負責第7話的編劇，而「{{lang|ja|のっぽとタイムボカン}}<ref>{{Cite web |url=http://www.brother-noppo.com/noppo_bokan.html |script-title=ja:【のっぽとタイムボカン】|publisher=編劇集團「{{lang|ja|ぶらざあのっぽ}}」公式官網|accessdate=2016-07-17 |archiveurl=https://web.archive.org/web/20030620063531/http://www.brother-noppo.com/noppo_bokan.html|archivedate=2003-06-20|deadlinkdate=2017-10|language=ja}}</ref>」一詞意思是說小山否認有參加時間飛船系列的《頂尖超人》編劇。}}〉最終作的企劃時，因為他對時間飛船系列特別的熱愛，因此還投入復活運動。

1975年8月，小山離開龍之子之後，以自由編劇的身分持續參加其它動畫公司作品。但因為受到龍之子公司的照顧，因此對龍之子作品有很強烈的感情。1977年9月，龍之子的創立人和首任社長[[吉田龍夫|吉田龍夫]]因為病逝而使公司面臨困境，小山在作品的工作人員表中曾經發表全力應援支持的話。

另外，小山自身也很熱誠栽培後進新人，因此1986年7月向[[日本電視台|日本電視台]]音樂學院（今：{{link-ja|日視學院|日テレ学院}}）租借教室，開設'''{{lang|ja|アニメシナリオハウス}}'''講座。翌年1987年，[[荒川稔久|荒川稔久]]、[[赤堀悟|赤堀悟]]、[[川崎裕之|川崎裕之]]、{{link-ja|影山由美|影山由美}}、[[隅澤克之|隅澤克之]]、[[松井亞彌|松井亞彌]]等多位線上劇作家輩出。之後更在1988年1月，成立編劇集團'''{{lang|ja|ぶらざあのっぽ}}'''。

2005年4月，小山就任[[大阪藝術大學|大阪藝術大學]]影像學科教授。但是由於無法兩方同時兼顧，因此他參加編劇的次數變少，甚至在2006年沒有生產新的劇本。因此不得宣布停止編劇事業，全面專注於編劇以外的工作<ref>{{Cite web|url=http://www.brother-noppo.com/essay/zakkan/index.php?y=2006&id=20|script-title=ja:【脚本家廃業宣言！】|publisher=編劇集團「{{lang|ja|ぶらざあのっぽ}}」公式官網|date=2006-12-19|accessdate=2017-10-06|archiveurl=https://web.archive.org/web/20071018232716/http://www.brother-noppo.com/essay/zakkan/index.php?y=2006&id=20|archivedate=2007-10-18|deadlinkdate=2017-10|language=ja}}</ref>。2014年8月底，小山於結束栽培新人編劇之後，{{lang|ja|ぶらざあのっぽ}}的規模縮小成為他自己的個人事務所，而經由小山訓練出來的劇作家全都已經獨立<ref>{{Cite web|url=http://www.brother-noppo.com/topics/other/noppo_notice_20140801.php |script-title=ja:【ぶらざあのっぽに関して、ご挨拶】|publisher=編劇集團「{{lang|ja|ぶらざあのっぽ}}」公式官網|date=2014-08-01|accessdate=2017-10-06|archiveurl=https://web.archive.org/web/20140808050235/http://www.brother-noppo.com/topics/other/noppo_notice_20140801.php|archivedate=2014-08-08|deadlinkdate=2017-10|language=ja}}</ref>。

兒子{{link-ja|小山真|小山真}}<ref>{{Cite news|title=2013年2月25日18:10 (JST)|work={{link-ja|小山真|小山真}}的[[Twitter|Twitter]]|newspaper=|date=2013-02-25|url=https://twitter.com/makotoppo104/status/306224724894158852|accessdate=2017-10-06|language=ja}}</ref>為了和父親一樣成為編劇，也加入了父親經營的{{lang|ja|ぶらざあのっぽ}}。

至於{{lang|ja|ぶらざあのっぽ}}的命名、和小山自稱自己是「[[亞洲|亞洲]]最大的劇作家」的由來就在於自身的[[身高|身高]]有194公分。也因此時間飛船系列的導演[[笹川浩|笹川浩]]在製作系列作之一《[[龍威|救難小英雄Yattode]]》時，直接以小山為角色原型，設計了一位身材高大但未露臉、名叫'''小山攝影師'''（{{lang|ja|小山カメラマン}}）的路人（由[[大瀧進矢|大瀧進矢]]擔任配音）。另一方面，小山也有在1993年的OVA《{{link-ja|時空母艦王道復古|タイムボカン王道復古}}》以配音員身分參演（只講「小山{{lang|j|です}}」一小句台詞）。

參加動畫編劇以來，小山多次為《[[七龍珠_(動畫)|七龍珠]]》、《[[以柔克剛|YAWARA！]]》及《[[銀河天使|銀河天使]]》等許多作品提供劇本。在[[特攝片|特攝影集]]方面，與《平成[[假面騎士|假面騎士]]系列》常態劇作家[[井上敏樹|井上敏樹]]是進入業界以來志同道合的好友，因此在井上負責劇本統籌的《銀河天使》和《[[電光快車俠2|電光快車俠2]]》中，曾經有許多小山的弟子參加。

小山除了參加動畫編劇之外，他也是國內新興宗教團體{{link-ja|崇教真光|崇教真光}}的信徒而有名。在團內，小山專門負責寫書和舉辦演講介紹該宗教團體，並在日本國內或海外國家進行傳教活動（詳細參閱下面的[[小山高生#崇教真光信徒活動|崇教真光信徒活動]]）。

== 崇教真光信徒活動 ==
小山是在1977年11月入教，之後就任道場幹部<ref>小山高生《{{lang|ja|霊もピチピチ}}生{{lang|ja|きている 実}}証·真光{{lang|ja|の}}業》，{{lang|ja|{{link-ja|二見書房#里昂社|二見書房|里昂社}}}}，2000年7月1日，第80頁 ISBN 4-576-85091-1</ref>。另外，小山在[[印度尼西亞|印尼]]進行傳教的活動在《崇教真光》誌平成19年（2007年）8月號被刊登。還有，該教團代理教主·{{link-ja|岡田晃彌|岡田晃弥}}的「{{lang|ja|ご}}教示」曾在小山的布教活動中作介紹，並賜予祝福<ref>《崇教真光》誌平成19年（2007年）8月號，第54－57頁。</ref>。

== {{lang|ja|ぶらざあのっぽ}} ==
'''{{lang|ja|ぶらざあのっぽ}}'''，是資深男[[編劇|編劇]]小山高生主宰的動畫編劇集團。1988年1月成立，1990年10月法人化登記成為[[有限公司|有限公司]]。2014年縮小成為個人事務所，最後在2016年3月31日解散，成立時間長達26年（有超過四分之一世紀）<ref>{{Cite web|url=http://www.brother-noppo.com/|script-title=ja:【お知らせ】|publisher=編劇集團「{{lang|ja|ぶらざあのっぽ}}」公式官網|accessdate=2017-10-06 |archiveurl=https://web.archive.org/web/20160402152733/http://www.brother-noppo.com/ |archivedate=2016-04-02|deadlinkdate=2017-10|language=ja}}</ref>。
