{{nihongo|《'''小小英雄－螃蟹與蛋與透明人－'''》|ちいさな英雄ーカニとタマゴと透明人間ー}}是由日本動畫工作室[[Studio_Ponoc|Studio Ponoc]]在2018年夏季公開，內容由三部短篇作品組成的動畫電影。<ref>{{cite web |url=https://animeanime.jp/article/2018/03/28/37206.html |script-title=ja:スタジオポノックが“短編劇場”プロジェクト始動 第1弾「ちいさな英雄」8月24日公開へ |work=Anime Anime |language=ja |date=2018年3月27日 |accessdate=2018年3月30日}}</ref>

== 劇情 ==
=== 卡尼尼與卡尼諾 ===
在河川裡頭，住著由父親{{nihongo|多多|トト}}，以及他的兩名兒子{{nihongo|卡尼尼|カニーニ}}和{{nihongo|卡尼諾|カニーノ}}所組成的[[螃蟹|螃蟹]]家庭。某日，多多在河中意外被水裡的急流給沖走，為了找回失蹤的父親，卡尼尼和卡尼諾兩人踏上了冒險。<ref name="CinemaTodayT0023047">{{cite web |url=https://www.cinematoday.jp/movie/T0023047 |script-title=ja:ちいさな英雄－カニとタマゴと透明人間－」 |work=Cinema Today |language=ja |accessdate=2018年8月12日}}</ref>

=== 武士蛋 ─ Life Ain't Gonna Lose ===
{{nihongo|俊|シュン}}是一名住在[[東京|東京]][[府中市_(東京都)|府中市]]裡喜歡[[棒球|棒球]]的小男孩，但他天生有著[[食物過敏|不能接觸蛋類食物]]的特殊體質。有一天在練習完棒球回家後，俊不小心拿了內含雞蛋成份製成的[[冰淇淋|冰淇淋]]當點心吃，讓他的身體因此起了嚴重反應。<ref name="CinemaTodayT0023047"/>

=== 透明人 ===
一棟舊公寓裡，住著一位不被週遭給注意到的[[透明人|透明人]]。而他的透明人身份，到了最後會連[[重力|重力]]都將其給忽視掉，就這樣慢慢飄浮到天空裡消失。<ref name="CinemaTodayT0023047"/>

== 製作 ==
=== 過程 ===
Studio Ponoc在公開他們創立以來首篇的長篇動畫電影《[[瑪麗與魔女之花|瑪麗與魔女之花]]》前，曾有人向該影片製作人[[西村義明|西村義明]]提出《瑪麗與魔女之花》是否會有出[[外傳|外傳]]之類衍生作品的可能。虽然當時西村義明沒這樣的意圖，但有在心裡想著如果真的要做出外傳的話，应如何使用短篇動畫來表現外傳內容的想法。<ref name="Toho">{{cite web |url=https://www.toho.co.jp/movie/news/1803/01ponoc_eiyu_ib.html |script-title=ja:「メアリと魔女の花」から一年、スタジオポノックが新たな挑戦！短編アニメーションレーベル「ポノック短編劇場」を始動 ポノック短編劇場「ちいさな英雄―カニとタマゴと透明人間―」 |work=[[東寶|東寶]] |language=ja |date=2018年3月27日 |accessdate=2018年3月30日}}</ref>

之後西村義明想到過去在[[吉卜力工作室|吉卜力工作室]]任職時，工作室裡的[[高畑勳|高畑勳]]與[[宮崎駿|宮崎駿]]導演都是會想嘗試新事物的人，有感於之前自己都是在他們所建構出的基礎下製作動畫，便決定以短篇的動畫電影來做挑戰。<ref name="Toho"/>而指名的影片導演人選除了有執導《瑪麗與魔女之花》的[[米林宏昌|米林宏昌]]外，也包括[[百瀨義行|百瀨義行]]與[[山下明彥|山下明彥]]兩名前吉卜力工作室人員。<ref>{{cite web |url=https://natalie.mu/eiga/news/275399 |script-title=ja:スタジオポノックの短編制作プロジェクト始動、西村義明「立ち止まってはいけない」 |work=[[Natalie_(網站)|Natalie]] |language=ja |date=2018年3月27日 |accessdate=2018年3月30日}}</ref>

=== 內容 ===
影片使用的作品標題為{{nihongo|《小小英雄》|ちいさな英雄}}，這出自於西村義明想向孩子們提出在生活四周裡，也許就有著渺小；但努力活著想達成某種目標的人物存在。<ref name="Toho"/>

作品裡包含三個長度各約15分的動畫，<ref>{{cite web |url=http://news.nicovideo.jp/watch/nw3388373 |script-title=ja:スタジオポノックが新プロジェクト「ポノック短編劇場」発表、第1弾を今夏公開 |work=[[niconico動畫|niconico動畫]] |language=ja |date=2018年3月27日 |accessdate=2018年3月30日}}</ref>其中米林宏昌所負責是名為{{nihongo|《卡尼尼與卡尼諾》|カニーニとカニーノ}}的章節，是個將螃蟹給予擬人化的作品，劇情是描述一對名叫卡尼尼和卡尼諾螃蟹兄弟所經歷的冒險故事，這也是米林過去至今執導過動畫裡頭首部由個人所原創的劇本。<ref>{{cite web |url=http://eiga.com/news/20180327/11/ |script-title=ja:ジブリ出身の実力派監督が結集するオムニバスアニメ8月24日公開！ |work=Eiga |language=ja |date=2018年3月27日 |accessdate=2018年3月30日}}</ref>起初米林是計劃弄出一篇將[[青蛙|青蛙]]給擬人化的內容，但之後發現內容不夠有趣而捨棄，便轉向從自己感興趣的澤蟹之類生物來尋找點子。<ref name="Natalie292459">{{cite web |url=https://natalie.mu/comic/news/292459 |script-title=ja:あのキャラに似すぎて設定が変わった？ポノック新作「ちいさな英雄」制作現場レポ |work=Natalie |language=ja |date=2018年7月25日 |accessdate=2018年8月12日}}</ref>畫面的表現手法上，則是採用了融合手繪以及電腦[[電腦圖形|CG]]的方式。<ref name="Natalie292459"/>

百瀨義行所執導{{nihongo|《武士蛋》|サムライエッグ}}是偏向劇情類型的作品，內容改自一位患有嚴重蛋類[[食物過敏|食物過敏]]少年的實有事蹟。<ref name="MyNavi607392">{{cite web |url=https://news.mynavi.jp/article/20180327-607392/ |script-title=ja:「ポノック短編劇場」は挑戦の場、西村代表が短編アニメーションに懸ける思い語る |work=Mynavi |language=ja |date=2018年3月27日 |accessdate=2018年3月30日}}</ref>因作品涉及到醫學症狀，而有向領域人士詢問了一些建議。<ref name="Natalie292459"/>其中百瀨不想讓觀眾覺得這只是一篇關於一名可憐小男孩的故事，而是想表達出生命強韌的一面，<ref name="Natalie292459"/>和少年要如何在天生體質問題下掌控自己的生活，以及親情之間的描寫。<ref name="MyNavi607392"/>作品畫風上有從[[古田足日|古田足日]]和[[田畑精一|田畑精一]]倆人所創作的繪本{{nihongo|《壁櫥裡的冒險》|おしいれのぼうけん}}來作為參考，並採用類似[[色鉛筆|色鉛筆]]所呈現出的繪製效果。<ref name="Natalie292459"/>

最後由山下明彥擔任的{{nihongo|《透明人》|透明人間}}，描述著一名透明人內心裡的孤獨，<ref>{{cite web |url=https://www.cinra.net/news/20180327-ponoc |script-title=ja:スタジオポノック新作『ちいさな英雄』にオダギリジョー、尾野真千子ら参加 |work=Cinra |language=ja |date=2018年3月27日 |accessdate=2018年3月30日}}</ref>並以山下明彥擅長的動作場面來作為表現手法。<ref name="Toho"/>西村義明認為作品裡的透明人，能作為現代人內心常有的孤獨、不安情緒的隱喻，並覺得這是一篇含有哲學性的疑問，以及是三篇裡頭最具前衛風格的作品。<ref name="Natalie292459"/>

=== 配音 ===
第一篇《卡尼尼與卡尼諾》裡，由女影星[[木村文乃|木村文乃]]和[[鈴木梨央|鈴木梨央]]分別擔任了卡尼尼和卡尼諾螃蟹兄弟。<ref name="Mainichi">{{cite web |url=https://www.cinra.net/news/20180618-ponoc |script-title=ja:スタジオポノック新作『ちいさな英雄』に木村文乃、鈴木梨央、坂口健太郎ら |work=Cinra |language=ja |date=2018年6月18日 |accessdate=2018年3月30日}}</ref>第二篇《武士蛋》中的小男孩俊、以及和他的父母一家人，為[[篠原湊大|篠原湊大]]、[[尾野真千子|尾野真千子]]、[[坂口健太郎|坂口健太郎]]三人飾演。<ref name="Mainichi"/>最後一篇的《透明人》，則邀請了[[小田切讓|小田切讓]]和[[田中泯|田中泯]]參與配音。<ref name="Mainichi"/>

=== 配樂與歌曲 ===
配樂部份有過去與米林宏昌多次合作的音樂人[[村松崇繼|村松崇繼]]，流行音樂製作人[[島田昌典|島田昌典]]，以及曾在百瀨義行執導的科幻短篇動畫三部曲《[[可攜式機場|可攜式機場]]》、《[[9號太空站|9號太空站]]》、以及《[[飛天都市計畫|飛天都市計畫]]》裡配樂的[[中田康貴|中田康貴]]擔任。<ref>{{cite web |url=https://mainichi.jp/articles/20180327/dyo/00m/200/006000c |script-title=ja:8月に3編の短編アニメを劇場公開　「メアリと魔女の花」の米林監督らが参加 |work=[[每日新聞|每日新聞]] |language=ja |date=2018年3月27日 |accessdate=2018年3月30日}}</ref><ref>{{cite web |url=http://jp.ign.com/movie/23327/news/38 |script-title=ja:「メアリと魔女の花」のスタジオポノックが短編アニメーション映画のレーベル創設　3本の新作を8月劇場公開 |work=[[IGN|IGN]] |language=ja |date=2018年3月27日 |accessdate=2018年3月30日}}</ref><ref>{{cite web |url=https://news.mynavi.jp/article/20180810-676783/ |script-title=ja:スタジオポノック「ちいさな英雄」三者三様の映像表現＆劇伴が楽しめる予告編 |work=My Navi |language=ja |date=2018年8月10日 |accessdate=2018年8月12日}}</ref>

影片片尾曲{{nihongo|《小小英雄》|ちいさな英雄}}，則由女歌手[[木村KAELA|木村KAELA]]負責演唱。<ref>{{cite web |url=http://skream.jp/news/2018/07/kimurakaela_chiisanaeiyu_analog.php |script-title=ja:木村カエラ、映画"ちいさな英雄―カニとタマゴと透明人間―"EDテーマを表題に据えたアナログ盤『ちいさな英雄』リリース決定。本日7/22にJ-WAVEにてレア・ライヴ音源放送も |work={{lang|ja|激ロック}} |language=ja |date=2018年7月22日 |accessdate=2018年8月12日}}</ref>
