'''小杰尼斯'''（{{jpn|j='''ジャニーズJr.'''|kk=ジャニーズジュニア|rm=Janīzujunia}}，{{Lang-en|'''Johnny's Jr.'''}}）是指隶属于[[日本|日本]]经纪公司[[杰尼斯事务所|杰尼斯事务所]]某特定部分[[偶像_(日本)|偶像]]的统称，他们大多在杰尼斯事务所官网上没有独立页面或尚未以CD形式正式出道。<ref name="hochi20120803">{{cite news|title= ジャニーズJr.に初密着！独自養成システム大公開！|newspaper= スポーツ報知|date= 2012-08-03|url= http://hochi.yomiuri.co.jp/feature/entertainment/johnnys/news/20120803-OHT1T00019.htm|accessdate= 2016-07-24|archiveurl= https://web.archive.org/web/20121215094127/http://hochi.yomiuri.co.jp/feature/entertainment/johnnys/news/20120803-OHT1T00019.htm|archivedate= 2012-12-15}}</ref> 他们也被简称为“{{lang|en|'''Jr.'''}}”。在2016年7月的時間點，當時杰尼斯事务所旗下包含关东與关西的小杰尼斯合計约有300人。<ref name="daily20160706">{{Cite news | author = 上野明彦|title = ジャニーズ事務所 なぜ人材集まるのか 公式募集無し、いつしか履歴書が| newspaper = デイリースポーツ | date = 2016-07-06| url = http://www.daily.co.jp/gossip/2016/07/06/0009256588.shtml| accessdate = 2016-07-24}}</ref>

== 概述 ==
小杰尼斯们主要是为同属[[杰尼斯事务所|杰尼斯事务所]]的已出道的前辈[[偶像_(日本)|偶像]]或[[男子音乐组合|偶像组合]]担任伴舞职能。<ref name="hochi20120803" /> 原本杰尼斯事务所并没有正式的招募候补艺人的计划{{efn|1=具体说明可以参见Johnny's Net内的“{{lang|ja|F&Qよくあるお問い合わせ一覧}}”对事务所招募候补艺人的说明。<ref name="daily20160706" /><ref>{{Cite web| title = よくあるお問い合わせ| work = Johnny's net| publisher = ジャニーズ事務所| url = http://www.johnnys-net.jp/page?id=inquiryJfc| accessdate = 2016-07-24}}</ref>}} 但是每月仍然有许多简历被投递至杰尼斯事务所。事务所[[董事长|董事长]]兼[[总裁|总裁]][[强尼·喜多川|强尼·喜多川]]会亲自审查这些简历，并在事务所举办演唱会、舞台剧等活动的时候不定期在现场举行面试及实际演出等选拔活动。<ref name="daily20160706" /> 如果合格通过事务所的选拔活动，那么事务所接下来就会安排舞蹈培训课程。杰尼斯事务所并不向小杰尼斯收取入社费和培训费等费用。<ref name="daily20160706" /> 入社后，所有小杰尼斯都会获得以个人或组合形式的登台表演机会，并从中累积自己的舞台经验。<ref name="daily20160706" />

小杰尼斯们也有自己统一的粉丝俱乐部，不过一般采用“情报局”这个名字。按照杰尼斯事务所以前的规定，小杰尼斯们在发行首张CD并成立专属自己的粉丝俱乐部之前都会称为“小杰尼斯”，而成立自己的专属粉丝俱乐部则被视为是小杰尼斯们的毕业典礼。需要注意的是，部分小杰尼斯（例如：[[Ya-Ya-yah|Ya-Ya-yah]]）或许发行了自己的首张CD但是因为没有自己的专属粉丝俱乐部，因而他们依旧被归类到小杰尼斯。但是到现在，根据杰尼斯事务所官方网站Johnny's Net和Johnny's Web上显示，部分在舞台剧或影视剧有良好演出记录的小杰尼斯尽管没有发行自己的出道CD，但是也被视为从小杰尼斯中毕业。例如：[[生田斗真|生田斗真]]<ref name="hochi20120803" />、{{Link-ja|佐野瑞树|佐野瑞樹_(俳優)}}、{{Link-ja|屋良朝幸|屋良朝幸}}、[[风间俊介|风间俊介]]、{{Link-ja|长谷川纯|長谷川純}}等人都以[[演员|演员]]身份从小杰尼斯毕业，并在杰尼斯事务所官方网站获得了单独的个人页面。<ref>{{cite web|title=ジャニーズ公式サイトで俳優・風間俊介、屋良朝幸らのページがオープン! ジャニーズJr.からは卒業|url=http://top.tsite.jp/news/johnnys/i/25894755/|publisher=T-Site|accessdate=2018-05-19}}</ref> 此外部分成员（例如：[[内博贵|内博贵]]和[[草野博纪|草野博纪]]等）从小杰尼斯毕业之后会因为一些丑闻或其他绯闻被原所属组合除名并退回到小杰尼斯重新出任研习生。<ref>{{cite news |title= 飲酒騒動NEWSが6人で再出発|newspaper = スポーツニッポン |date= 2006-12-31 |url= http://www.sponichi.co.jp/entertainment/special/2005jya/KFullNormal20061231052.html|accessdate=2016-08-28 |archiveurl= https://web.archive.org/web/20070208133256/http://www.sponichi.co.jp/entertainment/special/2005jya/KFullNormal20061231052.html|archivedate= 2007-02-08}}</ref><ref>{{Cite news | title = 元NEWS内博貴ソロで3年半ぶりに歌手| newspaper = [[日刊スポーツ|日刊スポーツ]]| date = 2008-11-15| url = http://www.nikkansports.com/entertainment/news/p-et-tp0-20081115-429711.html| accessdate = 2016-07-24}}</ref><ref>{{cite news|title= 元NEWSの内と草野が復帰 7月から少年隊恒例ミュージカルで|newspaper= 中日スポーツ|date= 2007-05-17|url= http://www.chunichi.co.jp/chuspo/article/entertainment/news/CK2007051702016699.html|accessdate= 2016-07-24|archiveurl= https://web.archive.org/web/20070519064541/http://www.chunichi.co.jp/chuspo/article/entertainment/news/CK2007051702016699.html|archivedate= 2007-05-19}}</ref>

杰尼斯事务所的发展历史上，曾经出现过两支以“小杰尼斯（{{lang|en|Johnny's Jr.}}）”为名的组合。它们分别是成立于1963年的乐队“{{lang|en|Johnny's Junior}}（{{lang|ja|ジャニーズ・ジュニア}}）”和成立于1973年的团队“{{lang|en|Johnny's Junior}}第1期生”。
# 1963年成立的{{lang|en|Johnny's Junior}}（也被称为“初代Jr.”）是当时杰尼斯事务所旗下[[男子音乐组合|男子音乐组合]]{{Link-ja|Johnny's|ジャニーズ}}的伴奏乐队，他们同时也会在[[爵士乐|爵士]][[咖啡馆|咖啡馆]]展开演出活动。
# 1973年成立的{{lang|en|Johnny's Junior}}第1期生（也被称为“二代Jr.”）则是当时杰尼斯事务所旗下男子音乐组合{{Link-ja|Four Leaves|フォーリーブス}}和男子偶像[[乡裕美|乡裕美]]的伴舞团体。杰尼斯事务所在当时大约维持着人数在25人的小杰尼斯一线队，同时又从中甄选了6人以{{lang|en|Johnny's Junior}}的名义参演了在[[日本剧场|日本剧场]]表演的“第46届西部狂欢节{{·}}乡裕美秀”。随着这支团队的人数规模扩充至9人。1974年秋天，他们分裂成两支组合。其中6人移籍到[[Burning_Production|Burning Production]]，并结成组合“{{Link-ja|Mets|メッツ (ジャニーズ)}}”；而留在杰尼斯事务所的3人则组成了“{{Link-ja|JOHNNY'S Junior Special|JOHNNYS' ジュニア・スペシャル}}”，他们的首度亮相获得了创纪录的成功。当{{lang|en|Johnny's Junior}}第7期生毕业之后，杰尼斯事务所终止了“{{lang|en|Johnny's Junior}}第X期生”的培训方式。

进入1980年代之后，杰尼斯事务所对于旗下小杰尼斯也推出了诸多演艺活动，当时除了那些拥有自己专属粉丝俱乐部的偶像之外，其他隶属杰尼斯事务所的偶像都被归类到小杰尼斯。在1998年到2002年期间，<ref name="sponichi20061002">{{Cite news| title = ジャニーズJr公演に応募殺到| newspaper = スポーツニッポン| date = 2006-10-02| url = http://www.sponichi.co.jp/entertainment/special/2005jya/KFullNormal20061002057.html| accessdate = 2017-03-10| archiveurl = https://web.archive.org/web/20061025033612/http://www.sponichi.co.jp/entertainment/special/2005jya/KFullNormal20061002057.html| archivedate = 2006-10-25}}</ref> 大量名称中夹带“{{lang|en|Junior}}（{{lang|ja|ジュニア}}）”或“{{lang|en|J}}”的节目在日本各大电视台开播。同时，小杰尼斯也在[[武道馆|武道馆]]和包括[[东京巨蛋|东京巨蛋]]在内的三大巨蛋体育馆举行表演，都获得了极大的欢迎。在这个时期，以[[泷泽秀明|泷泽秀明]]和[[涉谷昴|涉谷昴]]为首的“东泷泽西涉谷”“黄金一代Jr.”开始得到媒体及公众的承认。<ref name="sponichi20061002" /><ref>{{Cite web| title = 「今年来る」関西ジャニーズJr.人気の理由| work = 女性自身| publisher = 光文社| date = 2013-04-05| url = https://jisin.jp/serial/johnnys/johnnysinfo/6531| accessdate = 2016-08-28}}</ref> 随后，颇具人气的“关西小杰尼斯”和“名古屋小杰尼斯”也诞生了。通常情况下，[[东海地区|东海地区]]出身的小杰尼斯都会像[[千贺健永|千贺健永]]那样被划归到东京组，但是也有[[平野紫耀|平野紫耀]]这样在关西杰尼斯参加演出活动的个案。<ref>{{Cite journal|author = 関亜沙美|volume = 224|issue = 2015年11月号|title = 関西ジャニーズJr. ダンスより笑い!?関西Jr.になるのはこんな子|journal = 日経エンタテインメント!|page = 32 |publisher = 日経BP社}}</ref>

小杰尼斯的退社或组合解散等相关新闻在原则上是不会在杂志或电视上公开报道的。但是，[[小原裕贵|小原裕贵]]<ref name="nikkan20001016">{{cite news|title= ジャニーズ夏ツアー121万人動員|newspaper= [[日刊スポーツ|日刊スポーツ]]|date= 2000-10-16|url= http://www.nikkansports.com/news/entert/p-et-tp0-001016-01.html|accessdate= 2016-08-28|archiveurl= https://web.archive.org/web/20001027214027/http://www.nikkansports.com/news/entert/p-et-tp0-001016-01.html|archivedate= 2000-10-27}}</ref>、{{Link-ja|秋山纯|秋山純}}<ref>{{cite news |title= 秋山純がジャニーズ辞めていた！|newspaper = サンケイスポーツ |date= 2008-07-16 |url= http://www.sanspo.com/geino/news/080716/gng0807160432001-n1.htm|accessdate=2016-07-24 |archiveurl= https://web.archive.org/web/20080815081016/http://www.sanspo.com/geino/news/080716/gng0807160432001-n1.htm|archivedate= 2008-08-15}}</ref>、{{Link-ja|米花刚史|米花剛史}}<ref>{{Cite news | title = ジャニーズで活躍 米花剛史が30歳前に芸能界引退 | newspaper = スポーツニッポン| date = 2012-08-05| url = http://www.sponichi.co.jp/entertainment/news/2012/08/05/kiji/K20120805003836920.html| accessdate = 2016-07-24}}</ref>等人的退社新闻还是引发了媒体的报道。
