'''小米MIX 3'''是一款由[[小米科技|小米科技]]於2018年10月25日所发布的搭载[[Android|安卓]]系统的全面屏[[智能手机|智能手机]]。是小米MIX系列的第四款产品。

==概述==
小米MIX 3是[[小米MIX_2S|小米MIX 2S]]的换代产品，是全球首款[[钕磁铁|磁动力]]滑盖[[全面屏手机|全面屏手机]]，官方宣称其使用寿命可达30万次<ref>{{cite web |title=小米MIX 3 官网介绍 |url=https://www.mi.com/mix3/ |accessdate=2018-10-25}}</ref>。
==性能==
小米MIX 3采用了高通骁龙845作为手机的[[SoC|SoC]]，并有高达10GB RAM容量的故宫特别版可供选配，也是全球首款拥有10GB内存容量的手机。

==拍摄==
*小米MIX 3的后置摄像头模组采用了1220万有效像素的变焦双摄，摄像头模组与之前发布的小米MIX 2S以及[[小米8|小米8]]基本相同；其中主摄像头采用了索尼IMX 363作为图像传感器，镜头的光圈值为F1.8，物理焦距4.216mm，并支持四轴[[光学防抖|光学防抖]]；而副摄像头则采用了三星S5K3M3+作为图像传感器，镜头光圈值为F2.4，物理焦距5.2mm。<ref>{{cite web |title=小米MIX 3 官网参数 |url=https://www.mi.com/mix3/specs/ |accessdate=2018-10-25}}</ref>

*小米MIX 3自带的相机软件支持AI拍照场景识别和手持超级夜景等功能；此外，小米MIX 3还支持基于AI的[[1080p|1080p]] 960帧慢动作摄像，最长可录制0.25秒；MIX 3也是小米首款支持1080P/60fps以及4K/60fps视频录制的手机。

*[[DxOMark|DxOMark]]给了MIX 3的后置摄像头高达103分的综合得分和108分的拍照得分，其视频得分为93分<ref>{{cite web |title=Xiaomi Mi MIX 3: Excellent all-rounder |url=https://www.dxomark.com/xiaomi-mi-mix-3-camera-review/ |website=dxomark.com|accessdate=2018-10-25}}</ref>。

==充电==
小米MIX 3的有线充电支持[[QC|QC]]4.0+快充，标配最大功率为18W的QC3.0充电器，支持10W[[无线充电|无线充电]]并随机附赠了10W的无线充电底座。

==售价==
小米MIX3发布时的售价为（以[[人民币|人民币]]为单位）：
*6+128GB版：3299元
*8+128GB版：3599元
*8+256GB版：3999元
*10+256GB版：4999元 (故宫特别版)

== 小米MIX 3 5G ==
小米MIX 3 5G外表大致不變，只在背後加上5G標誌，內裡則換上Snapdragon 855和Snapdragon X50 Modem。後者令此機支援26GHz和28GHz的毫米波，以及sub-6 GHz頻段，使下載速度高達2Gbps。
