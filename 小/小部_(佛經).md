《'''小部'''》（[[巴利文|巴利文]]：{{lang|ISAT|Khuddakanikāya}}），也称《'''小尼柯耶'''》，为《[[巴利文大藏经|巴利文大藏经]]》中[[经藏|经藏]]的组成部分，[[上座部佛教|南传上座部佛教]]典籍。

其编订一般认为晚于前四部尼柯耶，具有补遗性质<ref>{{cite book | language = zh-hans | author = 郭良鋆 | coauthors =  | title = 佛陀和原始佛教思想  | url =  | date = 1997年 | location = 中国社会科学出版社 | publisher =  | id =  | isbn = 7500421907 | pages = 8-10页 }}</ref><ref>{{cite web|title=长老偈长老尼偈|url=http://www.yushi666.com/book/%E9%95%BF%E8%80%81%E5%81%88%E9%95%BF%E8%80%81%E5%B0%BC%E5%81%88.pdf|quote= 最初集成的巴利三藏中，是沒有《小部》的。《島史》論述第一次結集時，只提到「品、五十集、相應、集」這中部分，這便是經藏四部的芻型...關於《小部》的集成。前面已經談到，《小部》的集成晚於其它四部。其中大部分經文沒有經過三藏結集，是在其它四部成立之後，將尚未編入者匯集而成的。大約在公元前二世紀，《小部》才匯成一部，與其他四部並列，組成經藏的「五部」。但是，《小部》中的某些經文，卻是相當古老的。 }}</ref>。其他部派或稱此為[[雜藏|雜藏]]。鍚蘭及泰國傳的巴利經藏中的小部有十五部經，而緬甸共有十八部經。

小部雖被認為是晚於前四部尼柯耶集出，但其中有貼近佛世的古老經文如《法句經》、《自說經》、《如是語》、《經集》、《長老偈》、《長老尼偈》等六部古老的經典。同時也收集了接近部派佛教阿毘達磨的過渡典籍，如小誦、天宮事、餓鬼事、義釋、無礙解道、譬喻、佛種姓、所行藏，這些都被認為是屬於較為後期的經典。<ref>A textual and Historical Analysis of the Khuddaka Nikaya – Oliver Abeynayake Ph. D. , Colombo, First Edition – 1984, p. 113.</ref>

==定義==
巴利語Khuddaka，梵語kṣudraka (kshudraka)的意思含有小或雜，漢譯為「小」，或免與雜阿含相混淆。而「尼迦耶」 (nikaya)是南傳上座部獨用，其他部派及大乘則用「阿含」āgama。

因此，小部或小尼迦耶 (Khuddaka nikaya)，相當於 kṣudraka-āgama （古漢語譯為少分阿含、雜類阿笈摩），即是[[雜藏|雜藏]]<ref>{{cite web|title=阿毗达磨俱舍论梵汉对勘 破說我品第九|url=http://www.mldc.cn/sanskritweb/resour/etext/abhk9.html|quote='''kṣudrake' pi câgame''' daridrabrāhmaṇam adhikṛtyôktam—【真】於'''少分阿含'''中。為波遮利婆羅門。說此偈言 【玄】世尊於雜阿笈摩中為婆羅門婆拕梨說|access-date=2018-05-05|archive-url=https://web.archive.org/web/20151002043448/http://www.mldc.cn/sanskritweb/resour/etext/abhk9.html|archive-date=2015-10-02|dead-url=yes}}</ref><ref>玄奘譯《大阿羅漢難提蜜多羅所說法住記》：「有五阿笈摩。謂長阿笈摩。中阿笈摩。增一阿笈摩。相應阿笈摩。雜類阿笈摩。」</ref>。而雖稱為小部，但經數比其他部更多。

按[[善見律毘婆沙|善見律毘婆沙]]記載，小部（屈陀迦）破作十四分<ref>《善見律毘婆沙》：「'''破作十四分'''。悉入《屈陀迦》。」</ref>。無畏山寺派的[[解脫道論|解脫道論]]則記載《義釋》和《無礙解道》為[[阿毗曇|阿毗曇]]。

鍚蘭及泰國傳的巴利經藏中，小部有十五部經，而緬甸共有十八部經。

若據長部註釋書記載，長部師（Dighabhanaka）主張將《小誦》、《譬喻》、《佛種姓》、《所行藏》4部排除，其餘的11部歸入「論藏」。而中部師主張去掉《小誦》，其餘14部經仍作為《小部》保留在「經藏」中<ref name="長老"/>。

《小部》所含經教和諸經的歸屬問題上，眾說紛紜，而莫衷一是，反映了《[[雜藏|雜藏]]》的特性。不過斯里蘭卡的15部經，是一般採取的通說<ref name="長老">{{cite web|title=长老偈长老尼偈|url=http://www.yushi666.com/book/%E9%95%BF%E8%80%81%E5%81%88%E9%95%BF%E8%80%81%E5%B0%BC%E5%81%88.pdf}}</ref><ref>{{cite book|author1=印順|title=原始佛教聖典之集成－第七章 經典部類概論|url=http://yinshun-edu.org.tw/en/book/export/html/3192}}</ref>。

== 分类 ==
鍚蘭巴利經藏《小部》由十五部经组成：
# {{tsl|en|Khuddakapatha|小誦}}（Khuddakapāṭha）：從其它經律中選錄，供初學者誦習的9部短小經文。
# [[法句经|法句经]]（Dhammapada）：佛法之要偈，教義之名句，共26品，423偈。
# [[自说经|自说经]]（Udāna）：佛陀無問自發感興之偈，多處與律藏的「犍度」相同。
# [[如是語|如是語]]經（Itivuttaka）：以增一法集成的重頌，表示輾轉傳來聽聞的話語。
# {{tsl|en|Sutta Nipata|經集}}（Suttanipāta）：為赤銅鍱部所選錄的經文集成，主要由[[偈頌|偈頌]]所組成，包含了許多[[初期佛教|初期佛教]]的經文。
# {{tsl|en|Vimanavatthu|天宫事}}經（Vimānavatthu）：講述行善升往天界的種種樂事。
# {{tsl|en|Petavatthu|餓鬼事}}經（Petavatthu）：講述遭惡報墮為餓鬼的悲慘故事。
# {{tsl|en|Theragatha|長老偈}}（Theragāthā）：佛陀的大弟子比丘所說，以1279偈講述264位長老的事跡。
# {{tsl|en|Therigatha|長老尼偈}}（Therīgāthā）：佛的大弟子比丘尼所說，以522首偈頌講述73位長老尼的事跡。      
# [[本生經|本生經]]（Jātaka）：俗稱「五百五十本生」，收錄本生故事547個，但重複頗多。巴利本生原只有偈頌，長行部分是從[[僧伽羅語|僧伽羅語]]所譯出。 
# {{tsl|en|Niddesa|義釋}}（Niddesa）：對《經集》中的〈義品〉、〈彼岸品〉，以及〈{{tsl|en|Rhinoceros Sutra|犀角經}}〉的義疏，傳說為[[舍利弗|舍利弗]]所作。      
# {{tsl|en|Patisambhidamagga|無礙解道}}（Paṭisambhidāmagga）：論述[[止觀|止觀]]為主的論書，體裁近於部派的[[阿毗達摩|阿毗達摩]]，傳說為[[舍利弗|舍利弗]]所作。 
# [[譬喻_(佛教)|譬喻]]經（Apadāna）：講述佛陀、辟支佛、長老、長老尼的前世傳略和生平事跡及業報因緣。             
# [[佛种姓经|佛种姓经]]（Buddhavaṃsa）：講述佛傳故事的專品，風格與《長部·大本經》相類      。
# {{tsl|en|Cariyapitaka|所行藏}}經（Cariyāpiṭaka）：用詩偈講述了35個本生故事。論述了十[[波羅密|波羅蜜]]多，講釋迦佛前生的大行。
# {{tsl|en|Nettipakarana|指導論}}（Netti-ppakaraṇa）： (緬甸版有，泰國版沒有)，傳說為[[迦旃延|迦旃延]]所作。
# {{tsl|en|Petakopadesa|藏釋}}（Peṭakopadesa）： (緬甸版有，泰國版沒有)，傳說為[[迦旃延|迦旃延]]所作。
# [[彌蘭王問經|彌蘭王問]]（Milindapañha）： (緬甸版有，泰國版沒有)

十八部經在緬甸第五次結集時，被刻於石上，而在第六次結集成印刷及網上版。
