'''小查尔斯·雅各比'''<ref name=bio>northcom official bio <br />{{cite web |url=http://www.northcom.mil/leaders_html/docs/GEN%20Charles%20Jacoby%20Bio.pdf |title=存档副本 |accessdate=2012-10-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20121014021520/http://www.northcom.mil/leaders_html/docs/GEN%20Charles%20Jacoby%20Bio.pdf |archivedate=2012-10-14 }}</ref>（{{lang-en|'''Charles H. Jacoby,Jr.'''}}，{{bd|1954年|6月19日|||CatIdx=Charles H. Jacoby,Jr.}}），退役[[美国陆军|美国陆军]]上将，曾为[[美国北方司令部|美国北方司令部]]（USNORTHCOM）第5任司令，和[[北美空防司令部|北美防空防天司令部]]第22任司令，是首位担任此职的陆军将领，2014年12月5日卸任，并将职位移交给了海军上将[[威廉·戈特尼|威廉·戈特尼]]。此前，他曾担任[[联合参谋部|联合参谋部]]J5战略计划政策总监。 

雅各比1978年毕业于[[美国军事学院|美国军事学院]]，即通称的[[西点军校|西点军校]]，雅各比参加过步兵军官初级和高级课程，并进入[[陆军指挥参谋学院|陆军指挥参谋学院]]、[[高级军事研究学校|高级军事研究学校]]和[[国家战争学院|国家战争学院]]深造，并曾在[[密歇根大学|密歇根大学]]获得历史学硕士学位。

# 1991年6月—1992年1月，[[夏威夷|夏威夷]]，[[斯科菲尔德兵营|斯科菲尔德兵营]]，[[第25步兵师|第25步兵师]]，作战副参谋长，作战参谋主管
# 1992年2月—1993年6月，该师，第1旅，作战参谋
# 1993年6月—1993年10月，该师，作战副参谋长，外部评估科，科长
# 1993年10月—1995年10月，北卡罗莱纳州，布拉格堡，第82空降师，第504伞降步兵团，第1营，营长
# 1995年10月—1996年8月，[[华盛顿特区|华盛顿特区]]，[[陆军参谋长办公室|陆军参谋长办公室]]，管理局，国会活动处，参谋
# 1996年8月—1997年6月，[[国家战争学院|国家战争学院]]，学员
# 1997年6月—1997年10月，华盛顿特区，[[国防语言学院|国防语言学院]]-华盛顿办公室，西班牙语培训，学员
# 1997年11月—1999年2月，[[洪都拉斯|洪都拉斯]]，[[美国南方司令部|美国南方司令部]]，[[布拉沃联合特遣部队|布拉沃联合特遣部队]]，司令
# 1999年2月—2000年11月，[[佛罗里达州|佛罗里达州]]，[[迈阿密|迈阿密]]，美国南方司令部，副总参谋长，后任总司令的执行官
# 2000年11月—2002年7月，华盛顿特区，[[联合参谋部|联合参谋部]]，J-5战略规划政策部，分管全球、多边问题、美洲事务的副总监，
# 2002年7月—2005年6月，夏威夷，斯科菲尔德兵营，第25步兵师（轻型），副师长，分管作战，后分管支援。期间作为[[第76联合特遣队|第76联合特遣队]]分管支援的副司令，前往[[阿富汗|阿富汗]]参加“[[持久自由行动|持久自由行动]]”
# 2005年7月—2007年5月，[[阿拉斯加|阿拉斯加]]，[[理查德森堡|理查德森堡]]，[[美国陆军阿拉斯加部队|美国陆军阿拉斯加部队]]，司令；[[美国阿拉斯加司令部|美国阿拉斯加司令部]]，副司令
# 2007年6月—2009年3月，华盛顿特区，[[刘易斯堡|刘易斯堡]]，[[第1军|第1军]]，司令；刘易斯堡司令
# 2009年3月—2010年6月，[[伊拉克|伊拉克]]，第1军，司令；[[伊拉克多国军|伊拉克多国军]]，司令。参加“[[伊拉克自由行动|伊拉克自由行动]]”
# 2010年6月—2011年8月，华盛顿特区，联合参谋部，J-5战略规划政策总监
# 2011年8月至2014年12月，[[美国北方司令部|美国北方司令部]]（USNORTHCOM），司令；[[北美空防司令部|北美防空防天司令部]]（NORAD），司令
