'''小妖的金色城堡'''，是作家饶雪漫的青春[[爱情小说|爱情小说]]。最早2007年于[[北方文艺出版社|北方文艺出版社]]出版发行。

== 故事概述 ==
本书为[[饶雪漫|饶雪漫]]“青春疼痛系列之一”。叙述了七七，暴暴蓝，优诺三个女子青春期的爱情故事。书的主题如书中所说：“没有人永远十七岁，但永远有人十七岁。”，委婉的劝慰青年珍惜青春，乐观向上。<ref>{{cite book |language =zh-hans|author=饶雪漫|title=《小妖的金色城堡》|year=2007年|publisher=北方文艺出版社|location=|pages=后记|isbn=7-5317-2078-7 }}</ref>

== 作者简介 ==
[[饶雪漫|饶雪漫]]，[[四川|四川]][[自贡|自贡]]人，1994年毕业于[[四川理工学院|四川理工学院]]人文学院文学系，当代青春文学作家。
