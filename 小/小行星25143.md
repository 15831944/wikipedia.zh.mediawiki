'''小行星 25143'''，又名'''糸川'''<ref>“糸”在中文中音同「密」（mì），也是簡化字“纟”的原寫法。在[[甲骨文|甲骨文]]中已有其字，是[[象形字|象形字]]，意為二分之一絲（宋朝研究《说文解字》的学者徐锴说：“一蚕所吐为‘忽’，十忽为“丝”；‘糸’，五忽也。”）。</ref>（[[國際音標|IPA]] {{IPA|[itokawa]}}；{{lang-ja|'''イトカワ'''}}），是一顆會[[火星轨道穿越小行星|穿越火星轨道]]的[[阿波羅小行星|阿波羅小行星]]。日本科学家正在通过[[隼鸟号|隼鸟号]][[太空計劃|太空計劃]]对其进行详细研究。糸川小行星是继[[小行星433|小行星433]]（Eros）後，第二个有人造飞行器着陆的[[小行星|小行星]]，也是第一个被人类採樣，並成為第一個被帶回樣品的小行星。

== 歷史 ==
這顆小行星是在1998年在[[林肯近地小行星研究小組|麗倪耳]]計畫中被發現的，當時的[[臨時編號|臨時編號]]為{{mp|1998 SF|36}}，在2003年，[[日本|日本]]的[[隼鳥號|隼鳥任務]]發射升空後，成為其目的地的這顆小行星就被命名為「糸川」，以紀念日本的火箭科學之父[[糸川英夫|糸川英夫]]（Hideo Itokawa），順帶一提，糸川在二次大戰時期，曾經開發過[[一式戰鬥機|隼型戰鬥機]]。

== 說明 ==
糸川是一顆[[小行星#小行星的构成|S型]]小行星，在[[金石觀測站|金石觀測站]]的[[雷達|雷達]]影像呈現不規則的細長型<ref>參看：{{cite web |url=http://echo.jpl.nasa.gov/~ostro/itokawa.html |title=存档副本 |accessdate=2008-08-11 |0=2008-08-11 |deadurl=yes |archiveurl=https://www.webcitation.org/query?url=http%3A%2F%2Fecho.jpl.nasa.gov%2F%7Eostro%2Fitokawa.html&date=2008-08-11 |archivedate=2008-08-11 }}</ref>。

「隼鳥任務」證實了這項研究結果，團隊並認為糸川是由兩個或多個更小的小行星以重力作用結合在一起的[[聚合體|聚合體]]。

出乎眾人意料之外的，糸川的組成行星的表面缺乏[[撞擊坑|撞擊坑]]，表面佈滿了星羅棋佈的粗糙冰礫。任務團隊形容這些特殊的冰礫為「瓦礫」<ref>{{cite web|url=http://www.planetary.org/news/2005/0916_Hayabusa_Itokawa_Beckons_as_Japans.html |title=存档副本 |accessdate=2008-08-11 |0=2008-05-12 }}</ref>。這意味著糸川不是[[單獨巨石|單獨巨石]]（monolith），而比較像是經過長時間凝聚在一起的瓦礫堆。

==隼鳥任務 ==
日本的探測器「[[隼鳥號|隼鳥號]]」於2003年5月升空，並於2005年9月12日抵達糸川的附近，先是固定在太陽與小行星連線上距離20公里之處，然後移近至7公里的距離上。隼鳥在11月20日花了30分鐘的時間嘗試操作設備登陸並採集樣品，但沒有成功。在11月25日做了第二次登陸和採集樣品的嘗試。隼鳥在離開小行星後，於2010年6月13日深夜返回地球，且將樣品的膠囊艙降落在[[澳洲|澳洲]][[南澳省|-{zh-tw:南澳洲省;zh-cn:南澳大利亚州; zh-hk:南澳大利亞;}-]]的[[烏美拉|烏美拉]]（Woomera）。
