'''小窗格苔蘚蟲'''（[[學名|學名]]：{{lang|la|'''''Fenestrellina'''''}}），又名'''木柵苔蘚蟲'''，是一[[屬|屬]]已[[滅絕|滅絕]]的[[苔蘚蟲|苔蘚蟲]]，生存於[[志留紀|志留紀]]至[[二疊紀|二疊紀]]末期的[[海洋|海洋]]中。<ref>{{Cite web |url = http://fossilworks.org/bridge.pl?a=taxonInfo&taxon_no=25556 |title = Fossilworks: Fenestrellina |language = en |accessdate = 2017-11-07 }}</ref>其[[化石|化石]]分布於世界各地。牠們的單體長約0.5~1[[毫米|毫米]]，但可以組成60[[公分|公分]]或更大的群落，其群落呈[[扇|扇]]狀或[[漏斗|漏斗]]狀。<ref>{{Cite web |url = https://baike.baidu.com/item/%E5%B0%8F%E7%AA%97%E6%A0%BC%E8%8B%94%E8%97%93%E8%99%AB |title = 小窗格苔蘚蟲_百度百科 |language = zh-hans |accessdate = 2017-11-06 }}</ref>
