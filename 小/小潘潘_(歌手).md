'''小潘潘'''（{{bd|1992年|11月23日|}}），本名'''潘文凤'''，中國大陸流行音乐女歌手、-{zh-cn:网络;zh-tw:網路;}-主播，毕业于浙江艺术学院。<ref name=“甲”>{{cite web |title=小潘潘歌曲《笑忘缘》首发 梦回古今爱恋 |url=http://music.yule.sohu.com/20170814/n506690758.shtml |website=搜狐 |accessdate=2018-10-13}}</ref><ref>{{cite news |title=2018 YY粉丝嘉年华开幕首日人气爆棚 摩登兄弟小潘潘引万人合唱 |url=http://news.yule.com.cn/html/201809/280823.html |accessdate=2018-10-14 |date=2018-09-23|work=中国娱乐网}}</ref>歌曲《学猫叫》的原唱之一。<ref>{{cite web |title=YY小潘潘单曲《学猫叫》引发现象级刷屏传播 |url=http://ent.qianlong.com/2018/0522/2589053.shtml |website=千龙网 |accessdate=2018-10-14}}</ref>

==演艺经历==
2017年6月，发行个人EP《只不过》；7月，发行歌曲《只不过》的MV<ref>{{cite web |title=小潘潘新歌EP《只不过》MV首发上线王冕友情出演 |url=http://ent.163.com/17/0815/11/CRSKJ7MN00038793.html |website=网易 |accessdate=2018-10-13}}</ref>；8月，发行单曲《笑忘缘》。<ref name=“甲” />

2018年4月，发行歌曲《[[学猫叫|学猫叫]]》（与[[小峰峰|小峰峰]]共同演唱）<ref>{{cite web |title=歌曲：《学猫叫》 |url=https://y.qq.com/n/yqq/song/004RiqvD4Necim.html |website=QQ音乐|accessdate=2018-10-13}}</ref>；7月，参加央视综艺频道《最佳时刻》节目，演唱成名曲《学猫叫》<ref>{{cite web |title=[最佳时刻 2018世界杯燃情之夜]《学猫叫》 演唱：小潘潘 |url=http://tv.cctv.com/2018/07/10/VIDE69cKSF3SpYvCnRpnkShx180710.shtml |website=CCTV |accessdate=2018-10-13}}</ref>；10月，发行单曲《我是主播》。<ref>{{cite news |title=小潘潘最新单曲《我是主播》诉说主播背后的故事 |url=http://ent.workercn.cn/30024/201810/26/181026131439524.shtml |accessdate=2018-11-10 |work=中工网 |date=2018-10-26}}</ref>12月，和马来西亚知名鬼才原创歌手[[黄明志|黄明志]]合作，演唱《一起做过的蠢事情》。
