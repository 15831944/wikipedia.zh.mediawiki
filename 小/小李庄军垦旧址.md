==地理信息 ==
小李庄地处玛纳斯河东岸的冲积层高台上，距玛纳斯县城15公里，距石河子市16公里，属于玛纳斯县兰州湾乡<ref name="局长">{{cite web |author1=韩炜 |title=兵团旅游局局长王宇科到小李庄军垦旧址进行调研 |url=http://news.ts.cn/content/2008-07/18/content_2707762.htm |website=天山网 |accessdate=2019-04-14}}</ref>。

== 遗迹及维修==
小李庄现存共29栋苏式农庄<ref>{{cite web |author1=刘安新 |title=一四七团军垦旧址小李庄参观者开始数升温 |url=http://www.ts.cn/shz/2013-06/04/content_8247759.htm |website=天山网 |accessdate=2019-04-14}}</ref>。2013年起，新疆维吾尔自治区文物局经国家文物局批准对小李庄军垦旧址进行保护规划<ref>{{cite web |author1=国家文物局 |title=关于小李庄军垦旧址文物保护规划编制立项的批复 |url=http://www.sach.gov.cn/art/2013/10/10/art_1324_101386.html |website=国家文物局 |accessdate=2019-04-14}}</ref>。2016年起，新疆生产建设兵团经国家文物局批准，对小李庄军垦旧址中包括小李庄军垦旧址原三十团团部办公室、原三十团政治处办公室、原农十师资料图书馆、原农十师保卫处办公室、原农十师领导办公室、原农十师政治处办公室、原农十师幼儿园、原农十师供销处办公室、原农十师卫生处办公室、原农十师行政办办公室等10栋文物建筑进行抢险加固<ref>{{cite web |author1=国家文物局 |title=关于小李庄军垦旧址抢险加固工程立项的批复关于小李庄军垦旧址抢险加固工程立项的批复 |url=http://www.sach.gov.cn/art/2016/6/22/art_1324_131927.html |website=国家文物局 |accessdate=2019-04-14}}</ref>。2018年12月，石河子市文体局等多家单位对此次维护工作进行了初步验收<ref>{{cite web |author1=明珠生活 |title=石河子市文体局等多家单位对小李庄军垦旧址的抢险加固工程进行了初步验收 |url=https://m.sohu.com/a/280910663_754552?spm=smwp.comment.author-info.2.1555326359830fvJHipI |website=搜狐 |accessdate=2019-04-15}}</ref>。

== 小李庄历史 ==
1952年4月中旬，新疆军区派运输部副部长林海清等22人至玛纳斯县进行实地考查，并确定在玛纳斯河东岸新建生产点。1953年2月，农业团等在此处盐碱和芦苇中开荒造田，筑公路兴建骑七师师部，建成后改为农十师师部。小李庄房屋建筑呈“日”字型分布。小李庄建成后建筑总面积17931平方米，其中包括办公室建筑1656平方米，大礼堂1300平方米的苏式建筑群<ref>{{cite journal |author1=刘静 |title=一座被珍视的兵团军垦师部旧址建筑群 |journal=环球人文地理·评论版 |date=2016 |issue=05 |accessdate=2019-04-14}}</ref><ref name="文化">{{cite web |author1=许云峰 |title=在新疆兵团体验军垦文化 追忆红色岁月 |url=http://news.sina.com.cn/o/2012-07-11/152424755221.shtml |website=新浪新闻 |accessdate=2019-04-15}}</ref>。
农十师在小李庄逐步发展了农业、工业、手工业、商业等多种行业。除了是行政机关外，共有8个兵团生产连队，并且建立了工程建筑大队、被服厂、水利面粉加工厂、机械厂、银行储蓄所等23个单位。在教育方面，1952年，小李庄先后建立“八一”子弟学校，农十师子女校。1959年后，小李庄又先后开办摸索湾中学、三十团中学、农业中学、农八师师范以及垦区各类专业培训班 <ref name="小李庄" /> <ref name="文化" />。因此，小李庄逐渐成为玛纳斯河东岸政治、经济、文化中心<ref>{{cite journal |author1=刘静 |title=踏勘玛纳斯河东岸奠定兵团小李庄垦荒史 |journal=中国民族博览 |date=2016 |issue=07 |pages=87-88}}</ref>。
1970年起，小李庄转为新疆军区驻军点和生产基地并且有多支部队先后驻扎于此。1989年，驻军撤出，交由新疆军区后勤部管理。从此，小李庄逐渐无人看守，渐渐衰落<ref name="局长" /><ref name="小李庄">{{cite web |author1=石河子新闻网 |title=小李庄 |url=http://www.ts.cn/shz/2013-11/29/content_9008917.htm |website=天山网 |accessdate=2019-04-15}}</ref>。
