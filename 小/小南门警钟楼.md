'''小南门警钟楼'''，又称'''小南门救火联合会钟楼'''，又被简称为'''小南门钟楼'''、'''火警钟楼'''，位于[[上海市|上海市]][[黄浦区|黄浦区]][[中华路_(上海)|中华路]]581号，是曾经的'''上海城厢内外救火联合会'''的所在地。

== 建筑现状 ==
该塔状建筑建于1909年，为[[钢筋混凝土|钢筋混凝土]]结构，塔高35.2米，一度曾为[[上海老城厢|上海老城厢]]的最高建筑。中间有盘旋而上的铁梯至顶，攀缘而上，可俯瞰整个南市老城厢<ref name="thepaper">{{cite web|title=上海小南门警钟楼：敲响埋葬清王朝的丧钟_私家历史_澎湃新闻-The Paper|url=http://www.thepaper.cn/newsDetail_forward_1452498_1|website=澎湃新闻}}</ref>{{NoteTag|据来源写作“上海最高的建筑”。但众所周知，[[龙华塔|龙华塔]]有四十米左右高度，也在当时上海县境内，来源中的上海实则显然指代老城厢。}}。塔内曾有一口警钟，以纯铜铸成，鸣时声传数里（现仅剩塔楼）。楼顶建有一间瞭望室，设望警日夜驻守，遇灾鸣钟报警，各方即迅赴施救<ref name="sh-aiguo">{{cite web|title=小南门警钟楼|url=http://www.sh-aiguo.gov.cn/node2/node4/node8/node31/u1a300.html|website=上海市爱国主义教育基地网站}}</ref>。

== 历史 ==
明末清初，[[上海县|上海县]]还没有专门的消防队伍，而是由驻防的官兵和县衙衙役兼任火政，负责消防。后来，上海先的民间慈善团体开始举办消防队，例如1804年创办的同仁堂、1847年创办的[[厚仁堂|厚仁堂]]、1858年创办的果育堂、1863年创办的济善堂、1869年创办的赞育堂等团体都曾协助消防。但随着上海的发展，人口建筑快速增长，单靠慈善机构办的消防不能完全解决火患，于是，民间士绅纷纷设立“水社”“救火社”等救火团体。<ref name="jsxf">{{cite web|title=上海民间消防组织的发展之路|url=http://js.119.gov.cn/news/fire/2016-06-21/79927.html|website=江苏消防网}}</ref>

1905年11月，[[上海城厢内外总工程局|上海城厢内外总工程局]]{{NoteTag|该机构为[[上海道台|上海道台]]承认的上海城厢内外自治机构。}}的警政科下设有消防处，这是上海县第一次出现半官方的消防机构。由于[[上海租界|上海租界]]的消防队是为了保护租界利益而设立的，因此，当华界地区发生火灾，他们不予理会；而当时的[[上海道台|上海道台]]和[[上海县|上海县]]也没有把消防列入城市建设的范围。深受火灾之苦的市民为了自保自救，纷纷仿效租界火政模式，按路段、行业、社区等为要素组织公益性质的救火社。1907年，上海县城厢内外按地区分片，建立了下东、南、西、中四区设立救火团体。<ref name="jsxf" />

[[File:1907年上海救火联合会成立时官绅合影.jpg|thumb]]
1907年，[[上海城厢内外总工程局|上海城厢内外总工程局]]总董[[李平书|李平书]]联合当时沪南救火会会长[[毛子坚|毛子坚]]、宗教慈善家[[王一亭|王一亭]]等人一道倡议改良内地火政。1907年9月25日晚，召开上海城厢内外救火联合会成立大会，宣布将华界39处救火社会合并成统一的“上海城厢内外救火联合会”，并推举李平书为第一任会长，毛子坚、穆杼斋为副会长。上海知县[[李超琼|李超琼]]拨出小南门内旧粮仓废址2.5亩用于建救火联合会总部。三年后，救火联合会在这里建造了瞭望台暨火警钟楼。这座瞭望台由[[求新轮船厂|求新轮船厂]]创办人[[朱志尧|朱志尧]]负责营造。钟楼为钢架结构，总高35.2米，共分六层，每层有一个大平台，顶层有一可作四面瞭望的平台。在第四层平台上挂有一口高3.9尺，下口径6尺，重4800磅的大钟，大钟由及其控制的铁锤敲响，钟声可传几里之外。联合救火会将老城厢划成几个火政区，并规定，一旦发现火情，现敲响大钟25响，稍作停顿后再敲：敲响一下表示[[肇嘉浜|肇嘉浜]]（今[[复兴东路_(上海)|复兴东路]]）城南起火，二下表示肇嘉浜以北城内起火．三下表示[[十六铺|十六铺]]一带起火，四下表示[[上海南火车站|上海南火车站]]一带{{NoteTag|指的是现在上海市[[南车站路|南车站路]]附近的上海南站，不应和现在的[[上海南站|上海南站]]（新龙华站）混同。}}起火，五下表示[[放生局|放生局]]一带起火<ref>{{cite book|author1=薛理勇|title=旧上海租界史话|date=2002-2-1|publisher=上海社会科学院出版社|isbn=9787806189405}}</ref>。

1911年，[[辛亥革命|辛亥革命]]爆发，[[陈其美|陈其美]]等人决定起义响应。11月3日，在中部[[同盟会|同盟会]]、[[光复会|光复会]]上海支部共同领导下，以小南门警钟楼钟声为信号，上海商团各部、敢死队等聚集南市，宣布上海起义。<ref name="thepaper" />小南门钟楼既是辛亥革命时上海起义的信号楼，也是上海县城内迄今仅存的近代消防建筑。<ref name="jsxf" />

== 保护现状 ==
小南门警钟楼及其附属建筑，在2000年5月22日以“火警钟楼和上海救火联合会旧址”名义列为原[[上海市南市区|上海市南市区]]区级文物保护单位<ref>{{cite book|author1=《南市区续志》编委会|title=南市区续志|date=2003年3月|publisher=上海社会科学院出版社|location=上海|isbn=7-80681-168-0|url=http://shtong.gov.cn/newsite/node2/node4/node2249/node77162/node77199/node77773/node77803/userobject1ai95239.html}}</ref><ref>{{cite web |language =  |publisher = 上海市文化广播影视管理局 |title = 黄浦区县级文物保护单位、登记不可移动文物 |url = http://wgj.sh.gov.cn/wgj/node1257/node1264/u1a75610.html |author = [[上海市文化广播影视管理局|上海市文化广播影视管理局]] |date = 2011-10-28 |accessdate = |archiveurl= https://web.archive.org/web/20120512174125/http://wgj.sh.gov.cn/wgj/node1257/node1264/u1a75610.html | archivedate=2012-05-12 }}</ref>。2005年，该警钟楼以“小南门警钟楼”名义被列为第四批[[上海市优秀历史建筑|上海市优秀历史建筑]]<ref name="四优建">{{cite web|author=上海市房屋土地资源管理局,上海市城市规划管理局|title=上海市第四批优秀历史建筑名单公示|url=http://www.shanghai.gov.cn/nw2/nw2314/nw2319/nw12344/u26aw2359.html|website=上海市人民政府|language=zh}}</ref>。此外，因其在[[辛亥革命|辛亥革命]]中的重要作用，该处建筑被评为[[黄浦区爱国主义教育基地|黄浦区爱国主义教育基地]]<ref>{{cite web|title=黄浦区|url=http://www.sh-aiguo.gov.cn/node2/node4/node11/u1a378.html|website=上海市爱国主义教育基地网站|accessdate=2018-01-27}}</ref>{{NoteTag|在上海市爱国主义教育基地网站上，区级爱国主义教育基地显示为编号；市级爱国主义教育基地则是以火炬表示。}}。
