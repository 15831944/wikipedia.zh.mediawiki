'''小芝風花'''（{{bd|1997年|4月16日|||catIdx=Koshiba Fūka}}）是一名[[日本|日本]][[女演員|女演員]]，出身自[[大阪府|大阪府]]，隸屬於[[奧斯卡傳播|奧斯卡傳播]]旗下。

== 經歷 ==
2012年，小芝風花參演由同屬[[奧斯卡傳播|奧斯卡傳播]]的[[武井咲|武井咲]]主演的電視劇《[[無法呼吸的夏天|無法呼吸的夏天]]》，初次參與電視劇演出。她在劇中飾演主角谷崎玲（武井飾）的妹妹——谷崎麻央<ref>{{Cite web |url=http://career.oricon.co.jp/news/2013164/full/ |script-title=ja:“武井咲の妹”小芝風花が女優デビュー　劇中でも武井の妹に！ |work=[[Oricon|Oricon]] |date=2012-06-19 |accessdate=2014-04-06}}</ref>。

2014年，小芝獲選飾演《[[魔女宅急便_(2014年電影)|魔女宅急便]]》真人版電影的女主角——魔女琪琪，是在超過500人名單中決定的人選<ref>{{Cite web |url=http://www.cinematoday.jp/page/N0053740 |script-title=ja:実写版『魔女の宅急便』おソノ役は尾野真千子！小豆島でロケ |work=Cinema Today |date=2013-06-10 |accessdate=2013-11-30}}</ref>。小芝對此十分高興，獲悉能夠飾演琪琪時更喜極而泣<ref>{{Cite web |url=http://hk.apple.nextmedia.com/entertainment/art/20140404/18678430 |title=好戲派：與小魔女來港互窒 清水崇爭扮龍貓 |work=[[蘋果日報|蘋果日報]] |date=2014-04-04 |accessdate=2014-04-06}}</ref>。不過，小芝飾演琪琪一事，隨後遭不少不看好的網路用戶在她的個人[[部落格|部落格]]上批評<ref>{{Cite web |url=http://www.cinematoday.jp/page/N0052777 |script-title=ja:実写版「魔女宅」キキ・小芝風花、批判に心境吐露…「悲しい気持ちになる事を書かれた」 |work=Cinema Today |date=2013-05-08 |accessdate=2013-11-30 }}</ref>，但《魔女宅急便》原著小說作者[[角野榮子|角野榮子]]在拍攝現場觀看小芝風花的演出後，則表示小芝「十分可愛，跟（她對琪琪的）印象一樣。我一直很想看到活生生的琪琪出現。<ref>{{Cite web |url=http://www.oricon.co.jp/news/2025427/full/ |script-title=ja:実写『魔女の宅急便』おソノ役は尾野真千子　清水崇監督はオリジナル世界観に自信 |work=[[Oricon|Oricon]] |date=2013-06-10 |accessdate=2013-11-30}}</ref>」小芝在[[小豆島|小豆島]]等地拍攝外景的約一個月期間，刻意不跟家人聯絡，以貼近角色獨自修行的背景<ref>{{Cite web |url=http://hk.apple.nextmedia.com/entertainment/art/20140404/18678430 |title=「魔女宅急便」騎掃帚 小芝風花練腕力 |work=[[世界日報|世界日報]] |date=2014-04-06 |accessdate=2014-04-06}}</ref>。她曾於同年3月12日跟導演[[清水崇|清水崇]]到[[香港|香港]]為該電影宣傳<ref>{{Cite web |url=http://paper.wenweipo.com/2014/03/13/EN1403130003.htm |title=初拍戲未有壓力 小芝風花 清水崇會港迷 |work=[[文匯報|文匯報]] |date=2014-03-13 |accessdate=2014-04-06}}</ref>。

== 個人 ==
小芝由小學三年級開始學習[[花式溜冰|花式溜冰]]，曾於[[關西地區|關西地區]]的中小學生的比賽中獲得第8名<ref>{{Cite web |url=http://news.mynavi.jp/news/2011/11/14/019/ |script-title=ja:武井咲の"妹"は14歳のフィギュアスケーター小芝風花さん |work=Mynavi |date=2011-11-14 |accessdate=2014-04-06}}</ref>。
