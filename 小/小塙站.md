'''小塙站'''（{{jpn|j=小塙駅|hg=こばなえき|rm=Kobana eki|f=y}}）是位於[[日本|日本]][[栃木縣|栃木縣]][[那須烏山市|那須烏山市]]小塙的[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[烏山線|烏山線]]鐵道車站。

== 車站構造 ==
[[側式月台|側式月台]]1面1線的無人地上車站。

== 車站周邊 ==
* [[荒川_(栃木縣)|荒川]]

== 历史 ==
* 1934年（[[昭和|昭和]]9年）8月15日 - 開業。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]成為JR東日本車站。
