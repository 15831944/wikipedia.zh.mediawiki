'''小洋房'''，原名'''涵斋'''，又名'''剡溪小筑'''，也常称为'''蒋经国旧居'''，为[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[奉化区|奉化区]][[溪口镇_(宁波市)|溪口镇]]的一幢西式楼房，位于溪口潭墩山东麓，[[文昌阁_(溪口)|文昌阁]]西侧，始建于1930年。建筑原为[[蒋中正|蒋中正]]幕僚居所，1937年后成为[[蒋经国|蒋经国]]与夫人[[蒋方良|蒋方良]]回乡时的住所<ref name="王舜祁">{{cite book|title=宁波文物古迹保护纪实|author=王舜祁|first=|publisher=宁波出版社|accessdate=2018-07-11|year=2000|isbn=9-787-80602-418-8|location=|pages=|chapter=蒋氏故居保护和开放|chapterurl=http://www.nbzx.gov.cn/art/2015/9/18/art_4055_690833.html|archiveurl=https://web.archive.org/web/20180222043756/http://www.nbzx.gov.cn/art/2015/9/18/art_4055_690833.html|archivedate=2018-02-22|deadurl=no|df=}}</ref>。1996年，小洋房作为[[蒋氏故居|蒋氏故居]]的一部分被列入[[全国重点文物保护单位|全国重点文物保护单位]]<ref name="nbwb-detail">{{cite web|url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=830&type=2|title=蒋氏故居|accessdate=2017-09-02|date=2006-06-22|publisher=宁波市文物保护管理所|archiveurl=https://web.archive.org/web/20180222043927/http://www.nbwb.net/pd_wwbh/info.aspx?Id=830&type=2|archivedate=2018-02-22|deadurl=no|df=}}</ref>。

== 历史 ==
小洋房建于1930年，为蒋中正聘请的美国顾问[[威廉·亨瑞·端纳|端纳]]主持建设<ref name="杨韵华">{{Cite journal|title=溪口蒋氏遗迹考察纪行|author=杨韵华|author2=翁沁锦|url=|journal=武汉文博|issue=3|doi=|others=|year=2008|volume=|page=|pmid=}}</ref>，建成后用于接待幕僚和军政要员。[[陈布雷|陈布雷]]曾在[[西安事变|西安事变]]后在此撰写《西安半月记》<ref name="奉化市志">{{Cite book|title=奉化市志|last=|first=|publisher=中华书局|year=1994|isbn=9-787-10101-195-1|location=北京|pages=|chapter=第二十四编  风景旅游}}</ref>。1937年，因[[第二次国共合作|第二次国共合作]]，蒋中正之子蒋经国得以携苏联妻子芬娜（[[蒋方良|蒋方良]]）及子[[蒋孝文|孝文]]回到中国。为消除蒋经国在苏联受到的[[共产主义|共产主义]]思想影响，蒋中正聘请[[徐道邻|徐道邻]]为儿子的老师，讲授[[四书五经|四书五经]]等儒家经典<ref name="杨韵华" />，同时请蒋经国在苏联的同学[[高理文|高理文]]做伴读。除此之外，蒋经国还在此学习了《[[王阳明|王阳明]]全集》和《[[曾文正公|曾文正公]]文集》等典籍，同时将在苏联10年的经历写成《旅俄报告》呈送父亲，蒋中正曾数次批阅此报告。在小洋房居住期间，蒋方良曾在小洋房旁的临溪平台下水游泳，受到乡民议论，后被蒋中正提醒注意<ref name="王凡">{{Cite journal|title=奉化溪口遗迹中的蒋家故事（下）|author=王凡|url=|journal=人物|issue=12|doi=|others=|year=2010|volume=|page=|pmid=}}</ref>。1939年12月12日，日军轰炸溪口，蒋经国生母毛福梅被炸死于[[丰镐房|丰镐房]]，蒋经国手书“以血洗血”并刻碑，其碑现存于小洋房一楼<ref name="杨韵华" />。1949年蒋氏父子迁往台湾后，小洋房被[[中国人民解放军|中国人民解放军]]驻溪口的部队作为首长住所，1958年成为一家工厂的办公室，1979年由当地政府收回后对外开放<ref name="王舜祁" />。

== 建筑 ==
小洋房位于溪口潭墩山东麓，背山面水，为二层西式建筑<ref name="nbwb-detail" />。建筑为三开间，外涂石灰、水泥，因当时称这些材料为“洋灰”因而得名<ref name="杨韵华" />。建筑前后有天井，顶部有阳台可登临，周围设有水泥栏杆，西侧有露天走廊通往[[文昌阁_(溪口)|文昌阁]]<ref name="nbwb-detail" /><ref name="奉化市志" />。

== 保护 ==
{{external media|width=250px|image1=[http://www.nbwb.net/UploadFile/images/2018012213485503.jpg 小洋房保护区划图]，原始内容[https://web.archive.org/web/20180711164427/http://www.nbwb.net/UploadFile/images/2018012213485503.jpg 存档]于2018年7月11日}}1949年以后，虽经历次政治风波，小洋房保存完好。1979年，[[中共中央|中共中央]]为争取[[蒋经国|蒋经国]]与中国大陆谈判以实现两岸统一，决定修葺小洋房、丰镐房和玉泰盐铺。小洋房被按照蒋经国住读时的陈设修复<ref name="王舜祁" />。1996年，小洋房与[[丰镐房|丰镐房]]、[[玉泰盐铺|玉泰盐铺]]一道以[[蒋氏故居|蒋氏故居]]名义被列入[[第四批全国重点文物保护单位|第四批全国重点文物保护单位]]<ref name="nbwb-detail2">{{cite web|url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=830&type=2|title=蒋氏故居|accessdate=2017-09-02|date=2006-06-22|publisher=宁波市文物保护管理所|archiveurl=https://web.archive.org/web/20180222043927/http://www.nbwb.net/pd_wwbh/info.aspx?Id=830&type=2|archivedate=2018-02-22|deadurl=no|df=}}</ref>。建筑的保护范围西至剡溪小庄外墙，东、北至小洋房围墙及跳水台西端，南至剡溪岸边<ref>{{Cite web|url=http://www.nbwb.net/info_bhqh.aspx?id=41654|title=蒋氏故居（小洋房）保护区划|accessdate=2018-07-13|author=|date=2018-01-22|publisher=宁波市文物保护管理所|deadurl=no|archiveurl=https://web.archive.org/web/20180711164423/http://www.nbwb.net/info_bhqh.aspx?id=41654|archivedate=2018-07-11|df=}}</ref>。
