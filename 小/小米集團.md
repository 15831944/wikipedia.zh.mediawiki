'''小米集團'''（{{lang-en|'''Xiaomi Corporation'''}}，簡稱：'''小米'''，{{hkex|1810}}）是[[中國|中國]]一家专注于[[智能硬件|智能硬件]]、[[智能家居|智能家居]]以及[[軟件開發|軟件開發]]的企業，於2010年4月6日成立<ref>{{Cite web|url = http://money.163.com/14/1031/09/A9SHL81G00253G87.html|title = 小米魏来：关于小米的趣史和八大创始人|accessdate = 2016-02-18|author = |date = 2014-10-31|publisher = 网易财经}}</ref>，总部位于中国[[北京市|北京]]，截至2018年3月31日，员工人数近1.45万<ref>{{cite web |url = http://tech.163.com/18/0503/09/DGSEM48E00097U7R.html |title = 小米称员工人数达1.45万名，5515人为研发人员 }}</ref>。

2010年8月及12月，小米發布了基于[[Android|安卓系统]]深度定制的第三方固件[[MIUI|MIUI]]及首款移动应用[[米聊|米聊]]。2011年8月16日，小米正式推出了其第一款硬件产品——[[小米手机|小米手机（一代）]]<ref>{{Cite web|url = http://company.cnstock.com/company/scp_dsy/tcsy_tt/201309/2729536.htm|title = 公司估值达100亿美元 雷军独家解密小米定位|accessdate = 2016-02-18|author = 温婷|date = 2013-09-06|publisher = 上海证券报}}</ref>，开创了以互联网线上抢购高配置、低售价的智能手机销售模式。

通过旗下生态链品牌MIJIA（米家），小米的产品线从智能手机及[[耳机|耳机]]、[[移动电源|移动电源]]等手机周边产品和[[音箱|音箱]]、手环等相关移动智能硬件，扩展到智能电视、[[机顶盒|机顶盒]]、[[路由器|路由器]]、[[空气净化器|空气净化器]]、[[电饭煲|电饭煲]]等家居消费产品。截至2018年3月底，小米已进入全球74个国家和地区的市场，并在其中15个市场智能手机出货量名列前五<ref>{{Cite web|url=http://www.hkexnews.hk/APP/SEHK/2018/2018050202/Documents/SEHK201805030006_c.pdf|title=HKEXNEWS发行人相关资料-小米集团|accessdate=|author=|date=|publisher=|archive-url=https://web.archive.org/web/20180503180631/http://www.hkexnews.hk/APP/SEHK/2018/2018050202/Documents/SEHK201805030006_c.pdf|archive-date=2018-05-03|dead-url=yes}}</ref>。

2012年，小米全资买入北京多看科技有限公司，进入[[电子书|电子书]]阅读领域。[[多看阅读|多看阅读]]是旗下网站，并有相应的App。2018年，业界传闻小米有计划生产[[电子阅读器|电子阅读器]]。

2018年5月3日，小米正式向[[香港交易所|香港交易所]]提交[[IPO|IPO]]申请<ref>{{Cite news|url=http://www.ftchinese.com/story/001077408|title=小米正式在港交所提交IPO申请|newspaper=FT中文网|accessdate=2018-05-03}}</ref>，于2018年7月9日以[[B股|同股不同權]]的方式掛牌上市，并计划于7月23日纳入[[恒生綜合指数|恒生綜合指数]]<ref>{{Cite news|url=http://www.xinhuanet.com/tech/2018-07/10/c_1123101242.htm|title=小米上市首日破发 7月23日将正式纳入恒生綜合指數 |newspaper=新京报|accessdate=2018-07-10}}</ref>。

2018年11月19日，[[美图|美图]]公司与小米集团宣布达成战略合作伙伴关系，合作期限30年<ref>{{Cite web |url=https://tech.sina.com.cn/i/2018-11-20/doc-ihnyuqhi4537521.shtml|title=美图颜劲良：与小米的合作让美图更专注于互联网业务|accessdate=2018-11-20|work=界面|date=2018年11月20日}}</ref>。

== 标志 ==
小米的標誌是一個「MI」形，是“米”字的汉语拼音“mǐ”的大写，同时也是“Mobile Internet”的縮寫，代表小米是一家移動互聯網公司。「MI」另外一个含义为“mission impossible”，意为“完成不能完成的任务”<ref>{{Cite web|url=http://tech.qq.com/a/20110714/000278.htm|title=雷军诠释小米名称寓意：要做移动互联网公司|accessdate=2016-02-18|author=|date=2011-07-14|publisher=腾讯科技}}</ref>。其标志倒过来形似中文少了一点的“心”字，意为“让用户省一点心”<ref>{{Cite web |url = https://www.mi.com/about/ |title = 公司简介 |publisher = 小米科技 |language = zh-cn |accessdate = 2018-07-06 }}</ref>。

== 产品和服务 ==

=== 手机 ===

==== 小米手机系列 ====
{{Main|小米手机}}
[[小米手机|小米手机]]系列是小米公司发布的[[智能手机|智能手机]]主力机型系列，在2015年之前每年仅发布一款年度旗舰，自小米Note开始，小米推出了针对不同消费者人群的若干个分支产品线。后分别开设了主打新零售的X系列、主打无限流量的中低端入门级系列Play系列、主打大屏大电量的Max系列等。

小米Max之前所有的小米手机均在机身正面的屏幕左上角放置一枚Mi字logo，以与正面无Mi字logo的红米手机相区分。自小米Max发布开始，小米手机所有的分支产品线均取消了前置logo设计。此外自小米6开始，小米取消了前置RGB呼吸灯，改为如[[魅族|魅族]]手机类似的白色通知灯。

* '''小米数字系列：'''以“小米+数字”命名的主力旗舰机型，一般一年迭代一次，在每年年中首发，其中，[[小米手机|小米手机（一代）]]、[[小米手机2|小米2]]、[[小米手机3|小米3]]、[[小米手机4|小米4]]、[[小米手機5|小米5]]曾分别推出后续增强机型[[小米手机1S|小米1S]]、[[小米手机2S|小米2S]]、[[小米手機4S|小米4S]]、[[小米手機5s|小米5s]]，而小米7以纪念小米8周年为由被直接跳过，取而代之的是[[小米手机8|小米8]]和[[小米8_SE|小米8 SE]]。数字系列分为以下衍生版本，与数字系列正代机型无直接关系：
**'''小米A系列：'''以“小米+数字+A”命名的低配置衍生机型（A意为“青春版”，Adolescent的首字母），仅有[[小米手机2A|小米2A]]一款，后被子品牌红米取代。2017年重新启用A代号，但格式变为“小米+A+数字”（A意为Android One）；如[[小米A1|小米A1]]，接替小米i系列成为X系列的国际版定制机型，仅在海外市场发售，搭载原生安卓系统。
** '''小米X系列：'''以“小米+数字+X”命名的衍生机型（X意为新零售，汉语拼音Xinlingshou的首字母），如[[小米手机6X|小米6X]]，自[[小米手機5X|小米5X]]时增设。主打线下实体店市场。
** '''小米c系列：'''以“小米+数字+c”命名的衍生机型（C意为China，中国一词的英文首字母），自[[小米手機4c|小米4c]]时增设。小米4c为小米4i在中国大陆地区发布的版本，定位为次旗舰；小米5c定位为试验机型，采用自家芯片“澎湃”。
** '''小米i系列：'''以“小米+数字+i”命名的衍生机型（i意为International，国际一词的英文首字母），针对国际市场推出，仅发布了[[小米手機4i|小米4i]]一款。
** '''小米SE系列：'''以“小米+数字+SE”命名的衍生机型（SE意为Small Edition，小尺寸版的英文首字母）。在小米8发布之前，小米数字系列的屏幕尺寸一直维持在5英寸（对角线）左右，但小米8的屏幕则采用了6.21英寸，相当于16:9比例标准屏幕尺寸为5.5寸的机身投影大小。为照顾部分习惯小尺寸机型的用户的使用习惯，小米便自小米8这一代开始增设数字系列的小屏衍生版本，与同代数字系列旗舰机同时发布。通过缩减部分核心元器件规格以及机身工艺，搭载骁龙自2018年最新推出的骁龙700系列处理器，打造出定位为中高端的“小屏旗舰”。发布了[[小米8_SE|小米8 SE]]和[[小米9_SE|小米9 SE]]<ref>{{Cite web|url=https://www.mi.com/mi9se/|title=小米9 SE - 小米商城|accessdate=2019-02-21|author=|date=|publisher=}}</ref>两款机型。
* '''小米MIX系列：'''以“小米+MIX+数字”命名，是2016年起推出的机型系列，搭载陶瓷机身以及较大尺寸的显示屏。[[小米MIX|初代]]定位为汇集最新可行研发科技的概念机型，之后定位为量产的高端旗舰机型，從MIX 3開始取代小米Note系列作为商务旗舰发布，是目前小米手机各产品线定价最高的机型。
* '''小米Note系列：'''以“小米+Note+数字”命名，是2015年起推出的机型系列，比每年发布的小米手机系列屏幕略大，前两代为针对商务市场打造的高端旗舰机型，因市场反应不佳<ref>{{Cite web|url=http://www.sohu.com/a/190657752_586626|title=小米Note系列回顾与展望：原本心向高端，奈何时运不济|accessdate=2018-05-03|date=2017-09-08|work=www.sohu.com}}</ref>及MIX系列发布，因此在[[小米Note_3|3代]]定位为中端次旗舰机型<ref>{{Cite web|url=http://new.qq.com/omn/20180131/20180131A0O7BL.html|title=过多的小米手机产品线：哪些要精简 哪些又需要调整？-腾讯网|accessdate=2018-05-16|work=new.qq.com|language=zh-CN}}</ref><ref>{{Cite web|url=http://mobile.zol.com.cn/654/6544203.html|title=小米Note3参数首度曝光 定位下落至中端|accessdate=2018-05-07|last=中关村在线|work=mobile.zol.com.cn}}</ref>，該系列自2018年起中断发布。
* '''小米Max系列：'''以“小米+Max+数字”命名，是2016年起推出的机型系列，是尺寸介乎于平板电脑与智能手机之间的“平板手机”，特点为大屏幕及大电池容量。
* '''小米Play系列：'''以“小米+Play”命名的小米系入门级智能手机系列，主打流量不限量，是小米借助其虚拟运营商身份推出的新的产品线，第一代使用了联发科技的P35处理器。

==== 红米手机系列 ====
{{Main|Redmi}}
[[红米手机|红米手机]]是小米公司2013年起<ref>{{Cite web|url=http://tech.sina.com.cn/mobile/n/n/2016-04-08/doc-ifxrcizu3787447.shtml|title=小米4月25日武汉大学发布新机6X|accessdate=|author=|date=2018-04-12|publisher=快科技}}</ref>发布的入门级智能手机子品牌，定价较小米系列低廉及大众化，机身只保留后盖上mi字样的logo，以与小米手机系列相区分。

2019年1月3日，小米官方宣布紅米手機系列將成為小米旗下的獨立品牌。<ref> [https://chinese.engadget.com/2019/01/02/xiaomi-s-redmi-became-independent-brand/ 小米宣佈「紅米 Redmi」成為獨立品牌]</ref><ref> [https://www.eprice.com.tw/mobile/talk/4568/5184480/1/ 紅米將脫離小米成為獨立品牌，首部 4800 萬畫素相機的拍照手機將於 1/10 發表]</ref><ref> [https://ccc.technews.tw/2019/01/03/redmi-brand-independence/ 紅米 Redmi 品牌獨立，小米又開始向華為榮耀學習了]</ref><ref> [http://www.nbd.com.cn/articles/2019-01-03/1288078.html 红米Redmi将独立运营 小米拆分品牌鏖战寒冬]</ref><ref> [https://tw.news.yahoo.com/%E5%B0%8F%E7%B1%B3-%E7%B4%85%E7%B1%B3%E5%B0%87%E4%BD%9C%E7%82%BA%E5%85%A8%E6%96%B0%E7%8D%A8%E7%AB%8B%E5%93%81%E7%89%8C%E9%81%8B%E7%87%9F-034606541.html 小米：紅米將作為全新獨立品牌運營]</ref><ref> [http://3c.ltn.com.tw/news/35508 小米推「紅米」獨立成家？4800萬畫素手機1月10日登場]</ref><ref> [https://tw.news.yahoo.com/%E7%B4%85%E7%8D%A8-%E5%B0%8F%E7%B1%B3%E6%97%97%E4%B8%8B%E5%93%81%E7%89%8C-%E7%B4%85%E7%B1%B3-%E7%8D%A8%E7%AB%8B-083657818.html 紅獨!!! 小米旗下品牌「紅米」獨立!!!]</ref><ref> [https://www.sogi.com.tw/articles/xiaomi_redmi/6252187 小米首次在中國以Redmi宣傳 1月10日紅米手機新品發表]</ref><ref> [https://tw.lifestyle.appledaily.com/gadget/realtime/20190103/1494127 紅米Redmi獨立了！1/10推4800萬畫素手機]</ref><ref> [https://www.cnbeta.com/articles/tech/804551.htm 小米开年第一件大事宣布 红米成全新独立品牌]</ref>。独立运营之后的红米手机品牌将以Redmi作为产品标识，接替之前的小米手机主力系列，作为主打线上市场和极致性价比的手机系列。

独立运营前的红米手机系列亦仿照小米手机系列，针对不同销售渠道及人群推出多个派生机型。

* '''红米数字系列'''：以“红米+数字”命名的常规机型，屏幕大小在4.7[[吋|吋]]~6吋之间，[[红米手机_(2013年)|1代]]、[[红米手机6|6代]]采用了[[联发科技|联发科技]]的[[系统芯片|SoC]]，[[紅米手機2|2代]]、[[红米手机3|3代]]、[[红米手机4|4代]]、[[紅米手機5|5代]]则采用了[[高通|高通]][[骁龙|骁龙]]芯片，其中，红米手机（一代）、[[红米手机3|红米3]]曾推出后续增强机型[[紅米手機1S|红米1S]]、[[红米手机3S|红米3S]]。[[紅米手機5|红米5]]新增大屏派生机型[[紅米手機5_Plus|红米5 Plus]]；[[红米手机6|红米6]]增设定位略高的机型[[红米6_Pro|红米6 Pro]]。
** '''红米S系列：'''以“红米+S+数字”命名的主打自拍的红米手机系列（S意为自拍，Selfie的首字母）。从2代（[[红米自拍手机S2|红米自拍手机S2]]）起步。
** '''红米A系列：'''以“红米+数字+A”命名的低配置派生机型（A意为“青春版”，Adolescent的首字母）。是小米售价最低，同时也是配置最低的一条入门级产品线。[[紅米手機2A|红米2A]]采用展讯芯片，[[红米手机4A|4A]]、[[紅米手機5A|5A]]采用高通骁龙处理器，[[红米手机6A|6A]]则转用联发科技的A22芯片。
** '''红米X系列：'''以“红米+数字+X”命名的派生机型（X意为新零售，汉语拼音Xinlingshou的首字母），是红米3、4代衍生出的一条产品线，主打线下实体店市场。
* '''红米Note系列：'''以“红米+Note+数字”型号命名的大屏主力机型，曾存在各种定制版本，处理器选用也在联发科和高通之间徘徊。在[[红米Note_4|第四代]]另推出主打线下实体店市场的[[红米Note_4X|红米Note 4X]]。
* '''红米Pro系列：'''以“红米+数字+Pro”型号命名的红米旗舰机型，2016年发布[[紅米Pro|红米 Pro（一代）]]后，产品线至今未更新。

此外，个别机型在中国大陆和中国大陆以外市场（如印度）的产品名称是不同的，如中国大陆地区发布的[[紅米手機5_Plus|红米5 Plus]]在印度则称为红米Note 5，而中国大陆地区发布的[[紅米Note_5|紅米Note 5]]在印度市场则称为红米Note 5 Pro。亦有部分红米机型仅在海外市场发售，无法直接在国内市场购买，如Redmi go等。

*小米產品因時常處於缺貨狀態，遭受了業界與部分消費者就供貨方面的負面回應。<ref>{{Cite web|url=http://www.yicai.com/news/5041075.html|title=红米3年卖1.1億台 跌至“谷底”的小米會反彈嗎？|accessdate=2017-02-04|author=|date=2016-07-11|publisher=第一财经}}</ref><ref>{{Cite web|url=http://tech.ifeng.com/a/20150330/41028505_0.shtml|title=解密小米供应链：雷军握主动权 上百公司分散供货|accessdate=2017-02-04|author=|date=2015-03-30|publisher=凤凰科技}}</ref>
*米家空气净化器、MIUI、[[小米手机3|小米手机3]]和[[小米手机8|小米手机8]]等产品曾被控诉抄袭其他厂商设计。<ref>{{Cite web|url=http://tech.163.com/14/0828/10/A4NSFCRJ000915BD.html|title=小米系统涉剽窃国外XPOSED设计|accessdate=2017-02-04|author=|date=2014-08-28|publisher=网易科技}}</ref><ref>{{Cite web|url=http://finance.sina.com.cn/roll/20141216/012421082639.shtml|title=小米推空气净化器：巴慕达指责“抄袭”|accessdate=2017-02-04|author=|date=2014-12-16|publisher=新浪财经}}</ref>
*小米在手机、电视机等设备的系统中强制添加广告增值服务（充值会员可以去广告，手机系统中可手动关闭部分推广广告），导致部分用户不满。<ref>{{Cite web|url=http://tech.ifeng.com/a/20160617/41624546_0.shtml|title=雷军回应MIUI广告问题：不满意可投诉|accessdate=2017-02-04|author=|date=2016-06-17|publisher=凤凰科技}}</ref><ref>{{Cite web|url=http://tech.china.com.cn/elec/20170125/293387.shtml|title=小米电视强植广告遭吐槽 开关机广告过多被消费者埋怨|accessdate=|author=|date=2017-01-25|publisher=中国网}}</ref>
