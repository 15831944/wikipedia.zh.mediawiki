'''小麥啤酒'''（'''Wheat beer'''），或稱'''麥啤'''，為在釀造穀料(grist)使用[[小麥|小麥]]的[[啤酒|啤酒]]，用量依小麥啤酒類型而有所不同，如[[巴伐利亞|巴伐利亞]]的[[德國小麥啤酒|德國小麥啤酒]]用量約占穀料60~70%%<ref name="GBIWEISSBIER"></ref>，而比利時白啤酒，則可能占40%<ref name="PierreCelis">

}}</ref>。

== 種類 ==
根據啤酒釀造的地區和各地釀造法的小變異，麥啤有幾個不同種類：

===德國小麥啤酒===

'''德國小麥啤酒'''(Weissbier)有幾種不同名稱，'''Weissbier'''、'''Hefeweizen'''(Hefe-酵母、weizen-小麥)<ref name="GBIWEISSBIER">、''' Weizenbier'''等都為德國常用名稱

}}</ref>。"Weizen"(小麥)此稱呼被用於[[巴登-符登堡邦|巴登-符登堡邦]]與北德地區，而"Weißbier"或"Weiße"(白色啤酒)此稱呼被用於[[巴伐利亞|巴伐利亞]]。
德國小麥啤酒依德國法律規範，小麥麥芽須占穀料一半以上<ref name="GBIWEISSBIER"></ref>，啤酒花慣用沙士，赫士布鲁克等增加其苦味{{Citation needed}}。
德國小麥啤酒不過濾，留存懸浮的酵母使外觀混濁，這是其被稱Hefeweizen的原因。此外，德國小麥啤酒且常進行瓶中發酵(bottle fermentation)<ref name="GBIKristallweizen"></ref>。被過濾過的德國小麥啤酒稱作'''水晶小麥啤酒'''(Kristall Weizen)，除了比較清澈，與德國小麥啤酒沒有差別<ref name="GBIKristallweizen">

 |url         = http://www.germanbeerinstitute.com/Kristallweizen.html

 |publisher   = German Beer Institute
 |accessdate  = 2013-11-13
 |deadurl     = yes
 |archiveurl  = https://web.archive.org/web/20131111175903/http://www.germanbeerinstitute.com/Kristallweizen.html
 |archivedate = 2013-11-11
}}</ref>。
德國小麥的顏色在淡金色到暗金色間，味道清新，酒體厚度輕，帶有香蕉與丁香味道且果香濃郁<ref name="bjcpWeissbier"></ref>。苦味低(IBU:8~15)，酒精濃度介於4.3~5.6% ABV<ref name="bjcpWeissbier">{{cite web

'''饮用方法：'''
饮用前竖立降温保存两至三天，使瓶中酵母沉淀。饮用时慢慢倾斜将酒倒入杯中。不要搅浑底部的酵母。好的酵母麦啤会立刻产生大量的气泡。如果愿意，可以在倒完啤酒后再轻轻摇晃酒瓶底部，将混浊的酵母倒入另外一个小杯子中品尝。饮用时也可以加一片桔子或者柠檬掩盖酸味{{Citation needed}}。

===小麥勃克===
小麥勃克啤酒(Weizenbock)：小麥啤酒的勃克版─即酵母小麥啤酒的濃烈版<ref name="GBIWEIZENBOCK">
{{cite web
 |url         = http://www.germanbeerinstitute.com/Weizenbock.html
 |title       = WEIZENBOCK
 |publisher   = German Beer Institute
 |accessdate  = 2013-11-11
 |deadurl     = yes
 |archiveurl  = https://web.archive.org/web/20130914192721/http://www.germanbeerinstitute.com/Weizenbock.html
 |archivedate = 2013-09-14
}}</ref>，為勃克啤酒中唯一的愛爾啤酒<ref name="kanweibocom"></ref><ref name="GBIWEIZENBOCK"></ref>。材料上，小麥勃克常以60~70%以上的小麥麥芽取代大麥麥芽<ref name="GBIWEIZENBOCK"></ref>。

小麥勃克顏色介於深琥珀色到深紅棕色<ref name="bjcpWEIZENBOCK">{{cite web
|url=http://www.bjcp.org/2008styles/style15.php#1c

|accessdate=2013-11-11
}}</ref>，具有濃郁的麥芽味與一點丁香辛味<ref name="GBIWEIZENBOCK"></ref><ref name="kanweibocom"></ref>。

===白啤酒===
{{Main|白啤酒}}
白啤酒(Witbier)採[[上層發酵|上層發酵]]方式製作。通常低苦味、清爽、且帶有香料氣息
<ref name="WitbierMartinLodahl">
{{cite web
|url=http://morebeer.com/brewingtechniques/library/styles/2_4style.html
|title=Witbier: Belgian White
|publisher=http://morebeer.com/
|accessdate=2013-11-06
|author=Martin Lodahl 
}}</ref>。
由於未過濾，酒內的小麥蛋白和酵母造成略白與朦朧的外觀，而被稱為白啤酒<ref name="AllAboutWeizenandWheatAleStyles">
{{cite web
|url=http://www.tastings.com/beer/wheat_beers.html
|title=All About Weizen and Wheat Ale Styles
|publisher=tastings.com
|accessdate=2013-11-06
}}</ref><ref name="bjcp">{{cite web
|url=http://www.bjcp.org/2008styles/style16.php#1a
|title=Belgian and French Ale 
|publisher=Beer Judge Certification Program
|accessdate=2013-11-06
}}</ref>。
白啤酒與其他小麥啤酒最大的不同點，在於白啤酒使用的小麥沒發芽且會添加香料<ref name="WitbierMartinLodahl"></ref>
<ref name="AllAboutWeizenandWheatAleStyles"></ref>。
白啤酒最常用的香料有橙皮和胡荽子，亦可有其他香料<ref name="WitbierMartinLodahl"></ref><ref>{{cite web |url=http://beeradvocate.com/beer/style/48|title=Witbier
|publisher=BeerAdvocate
|accessdate=2013-10-20}}</ref><ref name="AllAboutWeizenandWheatAleStyles"></ref>。橙皮可用普通的甜橙皮，或是庫拉索橘(一種苦橙)<ref name="WitbierMartinLodahl"></ref>。白啤酒同其他小麥啤酒一樣使用大量小麥，最傳統的穀料(Grist)配方為54%的大麥麥芽、41%的未發芽小麥和5%的未發芽燕麥<ref name="WitbierMartinLodahl"></ref><ref name="bjcp"></ref>。
