'''小栈教育'''（{{lang-en|Ministack}}），成立于2016年10月，总部设于[[北京|北京]]，主要从事少儿及青少年编程教育<ref>{{Cite web|url=http://stackedu.com/|title=小栈教育|accessdate=2017-12-07|work=stackedu.com|language=zh}}</ref>。

== 发展历程 ==

=== 2016年 ===
2016年10月，小栈教育在北京成立<ref>{{Cite web|url=http://www.sohu.com/a/124437842_362001|title=硅谷归来的他，想让国内创投圈更懂技术|accessdate=2017-12-07|date=2017-01-16|work=www.sohu.com}}</ref>。第一期 Javascript 全栈成人班开班。<br>

2016年12月，第二期 Javascript 全栈成人班开班。<ref>{{Cite web|url=https://mp.weixin.qq.com/s/8pEcqSYVfw8SgAPGudts-w|title=2017年过去了13周，我们花了一半去学习|accessdate=2017-03-28|author=|date=|publisher=}}</ref><br>

=== 2017年 ===
2017年7月，在杭州阿里巴巴园区内开设少儿编程夏令营。<br>

2017年9月，入驻北京顺义国际学校开设少儿编程课。<ref>{{Cite web|url=http://news.juesheng.com/a/36836.html|title=小栈教育李博文：编程教育是下一个教育改革点，任重道远-决胜网|accessdate=2017-12-07|work=news.juesheng.com|language=zh-CN|deadurl=yes|archiveurl=https://web.archive.org/web/20171207141028/http://news.juesheng.com/a/36836.html|archivedate=2017年12月7日|df=}}</ref><br>

2017年10月，入驻北京乐成国际学校开设少儿编程课。<br>
