'''小港區'''是[[台灣|台灣]][[高雄市|高雄市]]的一個[[市轄區|市轄區]]，位於市區南端，北臨[[鳳山區|鳳山區]]，東鄰[[大寮區|大寮區]]且有鳳山丘陵為屏障，西北連[[前鎮區|前鎮區]]，隔海望[[旗津區|旗津區]]，西南濱[[台灣海峽|台灣海峽]]，南接[[林園區|林園區]]，人口約有15.7萬，為[[高雄國際機場|高雄國際航空站]]的所在地。

本區地形平坦，地質以珊瑚[[石灰岩|石灰岩]]為主，氣候上則屬[[熱帶季風氣候|熱帶季風氣候]]，年均溫約為24.7℃，年雨量約為1750~1800公釐。

== 歷史沿革 ==
'''小港區'''舊名「港仔墘」，閩南語稱「邊緣」為「墘」，即因高雄潟湖南岸之鳳山港畔而得名。本區開發甚早，早在[[臺灣荷西統治時期|荷據時期]][[荷蘭東印度公司|荷蘭東印度公司]]就在此建立港口。[[明鄭時期|明鄭時期]]由[[鄭成功|鄭成功]]部將[[吳燕山|吳燕山]]負責開墾，[[清朝|清朝]]時則屬[[鳳山縣_(臺灣)|鳳山縣]]管轄。

日治初期屬[[臺南廳|臺南廳]][[鳳山支廳|鳳山支廳]]。1920年實行五州制，將鳳山支廳所屬港仔墘區、大林蒲區、空地仔區併為一-{庄}-，原訂名港子-{庄}-<ref>街-{庄}-名之改善 (上)，《臺灣日日新報》大正九年八月十三日</ref>，後改為'''[[小港庄|小港-{庄}-]]'''，始得小港之名，並歸[[高雄州|高雄州]][[鳳山郡|鳳山郡]]管轄。

第二次世界大戰結束後臺灣由中華民國政府接收，小港的行政單位改為鄉，隸屬於[[臺灣省|臺灣省]][[高雄縣|高雄縣]]。1979年[[高雄市_(省轄市)|高雄市]]改制[[直轄市|直轄市]]，並將小港鄉併入成為其市轄區。

== 土地規劃利用 ==
小港區地勢平坦、氣候溫暖、雨量豐沛，加上靠近海邊，居民原以農漁業為主，後有[[高雄國際機場|高雄國際航空站]]、高雄第二港口、[[臨海工業區|臨海工業區]]的設置，臨海工業區內先後進駐的387家，工作機會增加，人口快速成長，成為國際門戶及工業中心。

大量工業用地、港口、建築開發外，高雄國際機場邊緣土地禁建限制，農耕、養殖面積大幅短少。[[小港區農會|小港區農會]]以「看飛機」為區域性休閒主題，發展休閒農業，許多咖啡館和休閒農場因應而生。

小港街區已從小港路移轉至二苓重劃區，目前[[漢民路|漢民路]]、[[金府路|金府路]]為小港最繁榮的地段。

== 農產與生態資源 ==
小港區農產品產量以鳳梨最豐，尤以[[大坪頂|大坪頂]]一帶為盛。其它農作物還有稻米、番茄、香蕉、芒果、荔枝...等等；常見植物包括野薑花、大型喬木、向日葵、菱角、香水蓮...等等。

== 文化旅遊景點 ==
;[[紅毛港文化園區|紅毛港文化園區]]：
位於高雄港二港口入口處，利用原有的高字塔結合週邊環境資源整建而成，高字塔是由[[黃啟灣|黃啟灣]]設計師設計建造，以高雄的「高」字為造型的信號台，共有兩座，一座位於[[旗津區|旗津區]]，一座則為位於此，總面積有3.42公頃。由高雄市政府文化局和紅毛港文化協會攜手合作，在園區內規劃了「高字塔旋轉餐廳」、「展示館」、「戶外展示區」、「天空步道」、「碼頭與候船室」、「觀海平台」六大區域，由高雄市政府捷運局興建完成，距海僅六米，能夠進距離欣賞大船入港的景致，是臺灣最接近海洋的展覽館之一。

;[[高雄公園|高雄公園]]：
1979年興建，面積約為5.8公頃，公園三面緊鄰中山四路、大業北路、空中大學街與民益路巷道，正門面向[[高雄國際機場|高雄國際航空站]]，可說是外地遊客下飛機後，對高雄市的第一印象。因應高雄捷運置站，於2006年進行全面翻修改建<ref>[http://pwbgis.kcg.gov.tw/pwb/mrt_park/kh_about.htm 高雄公園緣起  高雄市政府] {{webarchive|url=https://web.archive.org/web/20140814082557/http://pwbgis.kcg.gov.tw/pwb/mrt_park/kh_about.htm |date=2014-08-14 }}</ref>。中鋼集團教育基金會於2007年認養，打造成教育性的飛翔公園。

;[[大坪頂熱帶植物園|高雄熱帶植物園]]：
全國首座以保育、研究、教育及遊憩來建構的都會熱帶植物園，位於[[鳳山水庫|鳳山水庫]]旁，與臨海工業區相鄰，啟用後，逐步恢復原有的自然生態景觀，至今園區盎然生氣，獲有'''綠明珠'''之稱。

[[File:Kaohsiung_Municipal_Social_Education_Hall.JPG|thumb]]
;高雄市立社會教育館（社教館）：
原為青少年活動中心，建於1995年6月，位於學府路，與小港高中及漢民國小相鄰。室內空間包括行政中心、民眾閱覽室、研習教室、展覽室、演藝廳、演講廳、視聽中心和體育活動中心等。戶外園區設露營中心、烤肉區、溜冰場、露天劇場、體能設施等供民眾使用。另與民間社團合作開辦漆彈場。2006年曾在該館演藝廳舉辦臺灣第四十一屆[[電視金鐘獎|電視金鐘獎]]。

;小港區市民農園：
園區約0.8公頃，提供鄉土戶外教學以及農業生態體驗，園區種物多為市民向小港區農會租地耕種。

;機場旁休閒農場和咖啡店：
為近年小港區新興農業休閒產業。臨近機場跑道西北側，就近欣賞飛機起降，近年來吸引許多遊客與市民前往。

;大坪頂：
位於小港機場東南方，與[[大寮區|大寮區]]的相鄰，盛產[[鳳梨|鳳梨]]。由於位居高處，能盡覽高雄夜景並目睹[[高雄國際機場|高雄國際機場]]航班的起降，有多家大型露天餐廳進駐。設有[[大坪頂公園|大坪頂公園]]及[[大坪頂熱帶植物園|高雄熱帶植物園]]，是運動休閒的好去處。

;[[鳳山水庫|鳳山水庫]]：
位於鳳山丘陵東南麓，亦即'''小港區'''、[[林園區|林園區]]、[[大寮區|大寮區]]交界的林內里，由於地處鳳山（山名，詳見[[鳳山丘陵|鳳山丘陵]]）因而取名為鳳山水庫。

;高雄市天文教育館：
位於[[高雄市小港區港和國民小學|港和國小]]內，館內著重天文教學和推廣教育。

*鳳鼻頭
