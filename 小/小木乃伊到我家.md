{{日本作品|小木乃伊到我家|ミイラの飼い方|How to keep a mummy}}是空木翔的[[日本|日本]][[漫畫|漫畫]]作品。目前於[[comico|comico]]（{{link-ja|NHN JAPAN|NHN JAPAN|NHN comico}}）進行連載。[[單行本|單行本]]已經出版8冊。日文直譯譯作《'''木乃伊的飼養方法'''》。

== 故事簡介 ==
{{expand section}}
某一天，作品主角、男高中生柏木空收到一份來自自稱是冒險家的爸爸從埃及寄過來的超大型包裹。他最不期望的就是看到一個嬌小的木乃伊自行打開包裹並從裡面走出來，而那個木乃伊只有空的手掌大小。隨著故事的發展，越來越多不可思議的生物相繼出現，有小鬼、龍甚至是夢貘！

=== 主要人物與夥伴 ===
; {{nihongo|柏木空|柏木 空（かしわぎ そら）|romaji=Kashiwagi Sora|{{配音|[[田村睦心|田村睦心]]<ref name="animatetimes20170901" />、[[津田美波|津田美波]]〈小學生、幼少〉}}}}
: 本作主角。橘色頭髮，瞳孔是橘金色的，頭頂有根會隨著感情晃動的呆毛，呆毛是空的本體（他月：不管發生什麼事，只要抓住-那根-（呆毛）他就會變得乖乖的）。生日是5月5日。身高167公分。運動神經極好，強項是體育及料理，除了英文外無所不能。看的懂甲骨文。很溫柔，愛照顧人。只要被他人依賴，就會赴湯蹈火，有時候會自我犧牲（不過都會被他月阻止）。開朗，社交能力強，因此朋友很多。在故事開始時，空收到遠在埃及的爸爸送來裝著小伊的包裹。雖然一開始空很警惕小伊，有鑒於爸爸的不良前科，但是，空很快地就與小伊培養出感情。
: 別人對他的稱呼：柏木（他月），小空（或空醬）（小茂），空君（空同學）（大地、楓），空（木蓮）

; {{nihongo|小伊|ミーくん|romaji=Mii-kun|{{配音|[[茅野愛衣|茅野愛衣]]}}}}
: 從埃及來的木乃伊，屬於不可思議的生物。目前住在小空家。時常無意識的賣萌，開心的時候會冒小花花。小空取名。

; {{nihongo|神谷他月|神谷 他月（かみや たづき）|romaji=Kamiya Tazuki|{{配音|[[河本啟佑|河本啟佑]]<ref name="animatetimes20170901" />、[[關根明良|關根明良]]〈小學生、幼少〉}}}}
: 藍髮，瞳孔為金褐色，瀏海平時總是蓋住右眼，但沒睡飽或者認真的時候會把瀏海往後梳。生日是7月7日。身高175公分。右手有傷痕。空的青梅竹馬兼摯友兼同班同學。兩人是在幼稚園認識，當時是他月先去找空聊天。個性較為多慮，一般情況下也比小空冷靜。看起來冷酷，其實很有義氣，因為無法不管與自己深交的人，因此會盡量避免與人扯上關係。頭腦很好，常常教空英文，也會教妹妹數學。視力非常好，前弓道社。是個抖S。
: 稱呼：他月（空），小他；他醬（小茂），阿神(神君)（大地），他月君（楓）

; {{nihongo|-{;zh-hans:可尼;zh-tw:可尼;}-|コニー|romaji=konī|{{配音|茅野愛衣}}}}
: 流浪的小鬼，不可思議的生物。住在他月家。雖然一開始是自己賴著不走，但他月也漸漸接受了他的存在，並表示自己右手前臂有跟他月一樣的傷疤。喜歡玩他月的瀏海。特徵是比「讚」。至於名字，他月說因為是「小鬼」所以用-{;zh-hans:可尼;zh-tw:可尼;}-（日文「小鬼 → {{lang|ja|こおに}} → {{lang|ja|こーに}} → {{lang|ja|こにー}} → {{lang|ja|コニー}}」）取名。

; {{nihongo|茂木朝|茂木 朝（もてぎ あさ）|romaji=Motegi Asa|{{配音|[[茜屋日海夏|茜屋日海夏]]<ref name="animatetimes20170901" />}}}}
: 主角群中唯一的女生。有著紅褐色長髮，平時都綁成雙馬尾，斜瀏海。粉色雙眼。身高148公分。個性天真友善、無邪，喜歡被叫暱稱「小茂（{{lang|ja|もぎ}}）」。有著一股驚人的怪力（遺傳自媽媽）。和小空因為龍的事而熟識起來。超級怕蜥蜴。
: 別人對她的稱呼：小茂（空、他月），小朝（大地、楓）

; {{nihongo|-{;zh-hans:阿勇;zh-tw:阿勇;}-|いさお|romaji=Isao|{{配音|[[高橋伸也|高橋伸也]]}}}}
: 龍，為稀有的不可思議生物。能飛，但是飛不遠（五公尺左右）。十分聰明，會寫字。是和主角群有關的不可思議生物之中最成熟的。由小茂取名。

; {{nihongo|立秋大地|立秋 大地（たちあき だいち）|romaji=Tachiaki Daichi|{{配音|[[山下誠一郎|山下誠一郎]]<ref name="animatetimes20170901" />}}}}
: 主角群中最晚出場的人。金色頭髮，綠色瞳孔，身高184公分。生日是1月1日。從小學時期開始只要一睡著就會做噩夢，原因是他在小的時候不小心摔破了昂貴的花瓶被父親懲處。而且每次從噩夢驚醒過來時會大叫和眼神總是憔悴的模樣，這也讓班上同學和其他學生都不敢靠近他。
: 稱呼：大地（空），立秋（他月），小大(大醬)（小茂）

; {{nihongo|胖嘟嘟|ムクムク|romaji=Mukumuku|{{配音|山下誠一郎}}}}
: 夢貘，移動速度跟光速差不多，為了吃掉大地的噩夢而跑來大地家（原因另有隱情）。可以吃掉別人的噩夢和控制在睡夢中的人的行動（像是開窗戶讓牠進去），名字是大地取的，其實大地原本想幫牠取古代風滿滿的名字（左右衛門之類的），但是在定好前牠就先記住胖嘟嘟這個名字了。毛很柔軟。

; {{nihongo|波奇|ポチ|romaji=pochi|{{配音|{{link-ja|高田紗希衣|高田紗希衣}}}}}}
: 空家養的狗。據說是可以活數百年的狗，灰黑色的毛，腳部是白色的。很有教養。

; {{nihongo|小阿|アーヤン|romaji=Aayan|{{配音|[[中尾隆聖|中尾隆聖]]}}}}
: 從埃及來的[[附喪神|附喪神]]（附在[[阿努比斯|阿努比斯]]的雕像上，不會動但是會說話），會預言。暱稱是自己取的，喜歡幫別人取暱稱。

=== 其他人物 ===
; {{nihongo|柏木楓|柏木 カエデ（かしわぎ かえで）|romaji=Kashiwagi Kaede|{{配音|茅野愛衣<ref name="animatetimes20170901" />}}}}
: 小空的姑姑。紫色長髮以及雙瞳。是很出名的藝人。個性十分溫柔但卻少根筋。一戴上眼鏡就會瞬間變成另一個人（楓大人）。

; {{nihongo|柏木木蓮|柏木モクレン|romaji=Kashiwagi Mokuren|{{配音|{{link-ja|松山鷹志|松山鷹志}}}}}}
: 小空的父親，自稱冒險家。從埃及寄小伊＆小阿給空。真實身分為保護官。

; {{nihongo|神谷月夜|かみや つきよ|{{配音|[[中小路雅海|中小路雅海]]}}}}
: 他月的妹妹。有點傲嬌。喜歡空。

; {{nihongo|保護官|}}
: 負責保護不可思議的生物。

; {{nihongo|收藏家|}}
: 捕捉或殺害不可思議的生物，將其拿去賣。也是保護官追捕的對象。

; {{nihongo|神明大人|}}
: 凹十津神社的神明。平常空他們去上學時，為了安全，會把小伊他們寄放在神社裡，由神明大人照顧。

; {{nihongo|地藏菩薩||配音：[[三好翼|三好翼]]（日本）}}
: 凹十津神社外的地藏菩薩，附喪神。

; {{nihongo|帕克|パツク}}
: 惡魔。惡魔很弱小，很容易死掉。死的時候會詛咒離自己最近的人。是死後唯一會轉生的種族。由於世界上只有七隻惡魔，所以很稀有。
: 在被收藏家追捕時受了傷，小時候的大地救了牠。名字是大地取的。

== 合作活動 ==
2016年9月19日起，和[[三麗鷗|三麗鷗]]的角色[[布丁狗|布丁狗]]合作，曾在網路線上發表全8回的漫畫。還有在2017年4月6日，由[[寶島社|寶島社]]發行雜誌書「小木乃伊到我家×布丁狗 SPECIAL BOOK」<ref>{{cite web|url=http://natalie.mu/comic/news/227698|script-title=ja:「ミイラの飼い方」×ポムポムプリンのコラボムック、スピンオフや作者インタビュー収録|publisher=[[Natalie_(網站)|Comic Natalie]]|accessdate=2017-11-03|date=2017-04-06|language=ja}}</ref>。

== 手機遊戲 ==
2019年2月15日宣布預計在2019年的春天，將推出方塊益智類型為內容《{{lang|ja|[https://miiranokaikata-event.nhnpcube.com/ ミイラの飼い方-ミイパズ]}}》的[[智能手機|智慧型手機]][[手機遊戲|社群遊戲]]。

目前已經開放可在遊戲官方網頁展開事前登錄，並推出[[LINE|LINE]]貼圖系列『[https://store.line.me/stickershop/product/1172992/ja LINE POP]』。全部由NHN PixelCube開發擔當。
