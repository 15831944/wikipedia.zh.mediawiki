'''小灰人'''在外星人研究學中也名'''[[羅斯威爾飛碟墜毀事件|羅斯威爾外星人]]'''、'''Zetas'''、和'''Reticulians'''，首次目擊報告是1961年9月19日，也是一種[[外星人|外星人]]或[[外星生命|外星生命]]的通稱，特別是在[[飛碟|飛碟]]相關事件和超常理事件中。雖然有人自稱見到各種不同外星人，但是小灰人依然舉足輕重佔有75%[[美國|美國]]目擊報告，20%的歐洲大陸目擊報告，12%[[英國|英國]]目擊報告。<ref>{{cite book| last =  Bryan

==外觀==
典型的小灰人描述是120cm（4呎）高'''或'''超過五呎高，有灰色（也有說是藍灰／綠灰／紅灰）皮膚，身體細小，沒有大塊肌肉，腳短，步伐不敏捷，手長及膝，手有6指。球狀無髮的頭由細的脖子支撐，頭的比例大，（黑色）大眼睛，扁平鼻子，細的嘴巴沒有外耳。在許多案例，鼻孔也有像是垂直兩條裂口狀的。

有時說他們是穿著灰色緊身衣制服。也有人說是赤裸。而不管是衣服或赤裸都無法分辨性別，都看不到外生殖器。

==小灰人的傳說==
===1890年-1950年===
雖然不同的媒體上有各種小灰人角色出現，但是大體概念上都是小身體的大頭人物, 有大眼，小臉；本形象已經在流行文化存在超過100年。類似小灰人的概念也在多個科幻小說登場，包括[[H·G·威爾斯|H·G·威爾斯]]系列小說，其中1901年《月球上最早的人类》將整個月亮設有居所，外星人設定是頭無鼻，眼睛出現凸塊，步姿古怪。<ref> {{cite book|last=  Wells|first= H G|authorlink= H.G. Wells|title=  The First Men in the Moon|origyear= 1901|date=2004|publisher= Wildside Press|id= ISBN 0-8095-9653-9|chapter= XI|quote = }}</ref> 這個概念在漫畫及科幻雜誌上出現，當中包括[[David_H._Keller|David H. Keller]]作品《The Conquerors》。

===1950-1960年代===
自[[Kenneth_Arnold|Kenneth Arnold]]目睹不明飛行物體以來，1950年代北美公眾對[[extraterrestrial_hypothesis|ETH]]及[[飛碟|飛碟]]關注大幅上升，在電腦及電視作品經常拿他們作業題材。

===1960-1970===
[[File:Zeta_reticuli.png|thumb]]在太空船上看到星圖事後比對，表示外星人來自39光年外Zeta星]]

1960年代，灰人開始在小說中流行，如[[杜立巴族|杜立巴族]]等虛擬外星生命，但是1965 發生了改變因為Boston Traveler出版了他自稱為真實故事的書[[希爾夫婦誘拐事件|Betty and Barney Hill]]. 據說故事來自他被[[催眠|催眠]]後的吐實回憶, 並由Benjamin Simon醫生紀錄了一年催眠治療, 它描述一個[[薄烤饼|薄烤饼]]型太空船降落在山丘上並下來了一些無毛無耳大眼的外星人.<ref>{{cite news|first =  John H
 |last =  Lutrell|title =  Did THEY Seize Couple|publisher = Boston Traveler|date=1965-10-25}}</ref>

===1970-1980年代===
在1977，導演[[史蒂芬·史匹柏|史蒂芬·史匹柏]]用小灰人形象拍攝''[[第三類接觸_(電影)|第三類接觸]]''這部電影。此形象直接來自諸多UFO目擊者和公眾媒體的描述形象。

不明生物研究學家[[Leonard_H._Stringfield|Leonard H. Stringfield]] 提出一個飛碟墜毀故事，美國也找到外星屍體，還驗屍。從許多消息來源指出拼回了尋回的屍體，也看起來像傳統的小灰人。他在1978年的[[MUFON|MUFON]]討論會上提出這故事完整版本。

1980年代，Stringfield 提出第二份文件給MUFON 包含一系列驗屍解剖圖 據稱是來自1950年代參與解剖外星人的醫護員. <ref>Stringfield，Leonard H (1980) "Status Report II, ''The UFO Crash/Retrieval Syndrome:  New Sources, New Data; Jan. 1980, [[MUFON|MUFON]], pp. 10-12, 30-33 (drawings) </ref> 他的報告中的小灰人特徵和當時傳媒電影主流的外星小灰人形象幾乎一樣：

*人形外形，皮膚質地像發霉的肉，包含臉部；
*十分邪恶的物种，遭到多个物种的反感；
*身高3-1/2至4-1/2尺，體重約40磅；
*靠近額頭下方有兩個大眼，杏仁狀，狹長，深邃，兩眼比較分開，有點向上傾斜，高度類似「亞洲人」或「蒙古人」；
*頭部比標準人類為大，西洋梨狀，X光顯示有類似人類的[[下顎|下顎]]、[[上顎|上顎]]、頭蓋骨；
*耳朵只有兩個洞沒有外耳；
*不明顯的鼻子，只有兩個小洞；
*沒有嘴唇的小嘴像是裂縫，看起來不像能用於說話或飲食， 因為只有很小的舌頭，且沒牙齒，嘴巴連結到一個小膜囊但是沒有消化系統；
*脖子很少，有時是看不到的；
*沒有頭髮，身體無體毛；
*軀幹細小；
*手很長，細小，長及膝蓋，腳部短而幼少；
*手部只有四隻手指以及沒有拇指。兩支指頭特長，有時有指甲；
*膚色有許多不同，有的是黃色，咖啡色，褐色，桃色，青灰色，有鱗狀紋路，有彈性和柔軟肌肉，沒有肌肉紋理；
*沒有內外生殖器。

==對公眾文化的影響==
[[File:Europeen_UFO_Congress_2005-03.jpg|thumb]]
小灰人對[[電影|電影]]和文學創作有很多影響，也是靈感來源，in a number of different franchises.
*電視「[[星際之門|星際之門]]」(StarGate SG-1)影集中，「仙宮人」就是小灰人的範本，<ref>{{cite web| url=http://www.scifi.com/stargate/about/aliens/index.html| title=Stargate SG-1: Aliens| publisher=Sci Fi| accessdate=2007-12-26}}</ref>
*卡通系列''[[南方公園|南方公園]]'', 在''Cartman Gets An Anal Probe''此集中小灰人被當成「來訪者」出現，後來在 ''Cancelled''一集中再次出現；其他集中也常以背景或配角出現，特別是較早的幾季。

*「羅斯威爾」影集系列 把小灰人當成陰謀策劃者；本影集中Antar星系來的小灰人墜毀在地球上。

*「七天」影集系列，片中的羅斯威爾外星人明確的被稱為小灰人。
*[[小聯盟|小聯盟]]棒球隊「拉斯維加斯51」使用小灰人當標誌。

*在「Animorphs」系列影集，外星人「Skrit Na」是一個小灰人，但是是來自於另一度空間。

*受歡迎的「Destroy All Humans!」系列遊戲軟體(PS2.PS3.Wii)，主角「Crypto」是一個小灰人。
