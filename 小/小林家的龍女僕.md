==故事簡介==
故事敘述來自異世界的龍托爾因為某些緣故而居住在普通人類[[上班族|上班族]]小林小姐的家裡並擔任[[女僕|女僕]]的日常生活。

=== 龍 ===
; {{Anchor|神奈神威|康娜|康娜卡姆依|坎娜卡梦伊|坎娜}}{{nihongo|神奈神威（康娜卡姆依）|カンナカムイ|Kanna Kamui|聲：[[長繩麻理亞|長繩麻理亞]]<ref name="Character"/>}}{{noteTag|“康娜卡姆依”或“坎娜卡梦伊”（布卡漫畫使用此译名）為“{{lang|ja|カンナカムイ}}”的音译。}}
: 身高：120公分、三圍：B60/W50/H55、生日：西元前12000年。
: 龍形是白色毛髮、羽翼的龍。由于没能力回到原来的世界同时玩恶作剧过大而被放逐，于是追随[[#托尔|托尔]]来到了故事中的世界。喜欢恶作剧，有些怕寂寞的我行我素型，对故事中的人类世界不太了解而好奇心很强。喜欢吃东西，有把好奇的生物（包括蝴蝶等昆蟲）吃掉的性情。色慾很強，但因為仍在幼年懵懂期，所以體現出是「舔遍身體」這類的行為。电是力量的来源，可以把尾巴的尖端插到插座上充电<ref>{{cite web | url = http://cartoon.southcn.com/c/2017-03/29/content_167942261.htm | quote = 还用尾巴插电源充电。 | title = 《小林家龙女仆》出外传漫画 康娜当主角 | date = 2017-03-29 | accessdate = 2017-04-20 | website = 南方網 | archiveurl = https://web.archive.org/web/20170330063628/http://cartoon.southcn.com/c/2017-03/29/content_167942261.htm | archivedate = 2017-03-30  |language=zh-hans}}</ref>。 
: 得到小林小姐收留而居住在其家中，變成人類時為蘿莉造型，使用的身份是小林神奈。
: 名字來自於[[阿伊努|阿伊努]]民間傳說的[[阿伊努拉克爾|阿伊努拉克爾]]之父荒神阿澄雷神。

: 身高：175公分、三圍：B90/W78/H84、生日：當黑暗融入虛無之時。
: 龍形是深藍海龍型。與[[#托爾|托爾]]原本是好友，但是在托爾將神殿摧毀後，便與托爾決裂，所屬調和一派。借着托尔一次回到原来世界找寻食材时打开的通道来到故事的现实世界中。好吃鬼，尤其喜歡甜食。只要有甜食，任何事也會幫忙。
: 變成人類時使用的身份是上井艾爾瑪，为了获得购买美食的费用，而去到小林的公司就職。

; {{Anchor|魁札爾科亞特爾}}{{nihongo|魁札爾科亞特爾|ケツァルコアトル|Ketsarukoatoru (Quetzalcoatl)|聲：[[高橋未奈美|高橋未奈美]]<ref name="Character"/>}}
: 身高：190公分、三圍：B120/W70/H85、生日：天地初分稍前一點。第一人稱為「[[僕娘|僕]]」。
: 簡稱爾科亞，龍形是頸部帶有毛髮的巨蛇型，又有「羽蛇神」的別號。所屬旁觀一派。由于和同伴喝酒时被下诅咒而侵犯自己的妹妹，从而被赶下神位到人类世界中。接受真土翔太的“召唤”，居住在其家中，由於身材姣好（巨乳）而被誤會為[[魅魔|魅魔]]，由于平时着装颇为暴露而被其他人出于道德理由带走。异色异样瞳，平时都是眯眯眼。

: 名字來自於[[阿茲特克神話|阿茲特克神話]]中的羽蛇神[[魁札爾科亞特爾|魁札爾科亞特爾]]。

; {{Anchor|法夫尼爾}}{{nihongo|法夫尼爾|ファフニール|Fafunīru (Fafnir)|聲：[[小野大輔|小野大輔]]<ref name="Character"/>}}
: 又譯法夫納([[bilibili|bilibili]])。是遠古時期就守護洞窟寶物的詛咒之龍，龍形是黑色鱗片、獠牙猙獰的巨龍。有著看到寶箱就會開的習慣。由于经常有人类去掠夺宝物，所以对人类十分厌恶。能變化穿著執事裝人類的形態，居住瀧谷真家中。變成人類時使用的身份是大山猛。由于是守護洞窟寶物，被小林形容为“家里宅”，也喜欢上玩电子游戏。之后被托尔介绍借住在泷谷真家，并且和泷谷真一拍即合。
: 名字來自於北歐神話的惡龍[[法夫納|法夫納]]。據說能和終焉帝不分上下。

; {{nihongo|托爾之父|トールの父|聲：[[菅生隆之|菅生隆之]]}}

: 混沌勢力的領導者。
: 真身是巨大的紅龍，能夠變身為長鬍子的老年男性。曾经来到故事的世界要求托尔回去，最后托尔坚决拒绝下放弃了要求。

; {{Anchor|伊露露}}{{nihongo|伊露露|イルル|Iruru}}
