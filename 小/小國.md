'''小國'''（{{lang-en|Small power}}）是指[[國際|國際]]體系中，[[權力|權力]]資源及影響力不如[[大國|大國]]及[[中等強國|中等強國]]的國家。
== 定義 ==
如同[[大國|大國]]、[[中等強國|中等強國]]或[[微型國家|微型國家]]一样，小國沒有很明確、通用的定義。最常作為小國劃分依據的一个因素是人口，一些观点认为，小國是[[人口|人口]]规模低于1000万的[[主权国家|主权国家]]<ref>{{cite journal|author1=韦民|title=小国概念:争论与选择|journal=国际政治研究|date=2014|volume=51|issue=1|pages=59-83|url=http://www.ixueshu.com/document/0608ae6a66463d07318947a18e7f9386.html|accessdate=2018-01-26}}</ref>。亦有观点从[[国家安全|国家安全]]角度认为，小国是在特定时期下，由于缺少足够的资源，难以自助而必须通过其他途径实现安全的国家行为体<ref>{{cite journal|author1=唐高略|title=小国安全模式研究|journal=华中师范大学|date=2011-05-01|url=http://www.ixueshu.com/document/2d3c8e7941452b31318947a18e7f9386.html|accessdate=2018-01-26}}</ref>。[[基歐漢|罗伯特·基欧汉]]提出过一个定义：小国的领导人认为它不能单独或通过小集团的行动对体系产生重要的影响。而[[罗伯特·罗思坦|罗伯特·罗思坦]]（Robert L. Rothstein）给出了另一个定义：小国认为它不能主要通过运用自身力量来获取安全，必须依赖其他国家、制度或进程<ref>{{cite journal|author1=周方银|title=小国为何能长期存在|journal=国际政治科学|date=2005-02-15|pages=24-47|url=http://www.ixueshu.com/document/f6c03ce0669c7e85.html|accessdate=2018-01-26}}</ref>。

== 數據比較 ==
就人口方面來說，小國普遍人口較少，就經濟方面來說，小國普遍經濟體系較小，但小國的[[人均國內生產總值|人均國內生產總值]]未必不如[[大國|大國]]或[[中等強國|中等強國]]，例如[[盧森堡|盧森堡]]、[[冰島|冰島]]和[[汶萊|汶萊]]。人口數據來自2017年7月的[[世界概況|世界概況]]<ref>{{Cite web |url = https://www.cia.gov/index.html |title = CIA世界概況 |accessdate = 2018/1/24  }}</ref>，[[國內生產總值|國內生產總值]]（GDP）數據來自2018年[[IMF|IMF]]的統計<ref>{{Cite web |url = http://www.imf.org/external/index.htm |title = IMF各國國內生產總值列表 |accessdate = 2018/1/25  }}</ref>。
