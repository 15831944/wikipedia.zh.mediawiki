'''小汤山医院'''又名'''北京小汤山疗养院'''，位于[[北京市|北京市]][[昌平区|昌平区]][[小汤山镇|小汤山镇]]，是北京市属三级综合医院。

==历史==
因山上有温泉，故名小汤山。1950年代，[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]][[中国人民解放军总后勤部|总后勤部]]在此设立[[疗养院|疗养院]]。1982年，更名为'''北京市康复中心'''。1988年，设立为小汤山医院<ref name=A>{{cite web |url = http://news.qq.com/a/20090909/000577.htm |title=小汤山医院的奇迹 |publisher=News.qq.com |date= |accessdate=2013-04-04}}</ref>。

==[[SARS事件|SARS事件]]==
2003年[[SARS|SARS]]疫情最严重时，中国政府动用军队，在原有小汤山医院基础上紧急建设“'''中国人民解放军小汤山非典医院'''”<ref>{{cite web |url = http://news.qq.com/a/20100402/001205.htm |title = 小汤山非典医院拆除 抗非医生连连叹息(图)_新闻_腾讯网 |website = news.qq.com |date=2012-01-17 |accessdate=2013-04-04}}</ref>，常简称为'''小汤山非典医院'''（2010年时北京市卫生局宣传处人员表示：严格地说是没有‘小汤山非典医院’这个说法，它在建立之初就不是一个医院，而是临时的野战医疗点<ref name=cyol>{{Cite web |url = http://www.cyol.net/zqb/content/2010-04/10/content_3176786.htm |title = 专家解读：为什么七年后才拆小汤山 |author = 庄庆鸿 |date = 2010-04-10 |publisher = 中国青年报 |language = zh-cn |accessdate = 2018-06-26 }}</ref>）。2003年5月1日夜，开始接收全国各地的SARS病人<ref name="A" />。SARS疫情结束后，因为其特殊情况，没有利用价值，故被遗弃。长期的废弃，无人看管，使其杂草丛生，破旧不堪，被称为中国的[[寂静岭|寂静岭]]<ref>{{cite web |url = http://house.focus.cn/news/2011-10-31/1557997_1.html |title = 万圣节探秘：中国版寂静岭 北京小汤山医院(组图)-新闻中心-搜狐焦点网 |website = house.focus.cn |date= |accessdate=2013-04-04}}</ref>。

== 拆除 ==
2010年4月2日，[[北京市卫生局|北京市卫生局]]宣布，将拆除北京市小汤山医院非典病房<ref name=cyol />。
