'''小川航基'''{{BD|1997年|8月8日|catIdx=Koki Ogawa}}是[[日本|日本]]職業[[足球|足球]]運動員<ref>[https://data.j-league.or.jp/SFIX04/?player_id=19169 J.League Data Site]</ref>，目前效力於[[磐田喜悅|磐田喜悅]]，司職[[前鋒_(足球)|前鋒]]。

==生涯==
2016年小川航基加入[[日本甲組職業足球聯賽|日本甲組職業足球聯賽]][[磐田喜悅|磐田喜悅]]。4月6日，他在[[日本聯賽盃|日本聯賽盃]]初登場（對上[[甲府風林|甲府風林]]）<ref>[https://data.j-league.or.jp/SFIX04/?player_id=19169 J.League Data Site]</ref><ref>[https://data.j-league.or.jp/SFMS02/?match_card_id=18911 Match report J.League Data Site]</ref>。
