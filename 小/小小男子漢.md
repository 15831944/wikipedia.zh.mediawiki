《'''小小男子漢'''<ref>1995年[[衛視中文台|衛視中文台]]引進電視動畫版播出時譯為《小小男子漢》，[[臺灣電視公司|臺灣電視公司]]引進電視動畫版播出時譯為《振作一下》。</ref>（日文直譯：'''阿強加油'''）》（{{lang-ja|ツヨシしっかりしなさい}}）是[[永松潔|永松潔]]的[[日本漫畫|日本漫畫]]作品，和改編同名電視動畫、電視劇。

== 概要 ==
1986年至1990年在[[講談社|講談社]]青年漫畫雜誌《[[Morning_(漫畫雜誌)|Morning]]》連載。此外第1話是在《Comic Morning Magnum增刊》1986年2月13日號發表。

該作品也是系列漫畫作品，有描寫主角'''井川強'''（暱稱：'''阿強'''／<small>{{lang-ja|'''ツヨシ'''}}</small>）高中時期到大學時期的「'''阿強加油'''（<small>{{lang-ja|ツヨシしっかりしなさい}}</small>）」（單行本全19冊、Best Selection版全3冊）、描寫井川強國中時期的「'''阿強要更加加油'''（<small>{{lang-ja|ツヨシもっとしっかりしなさい}}</small>）」（《{{link-ja|月刊Afternoon|月刊アフタヌーン}}》連載，單行本全5冊）、描寫井川強出社會後已是有2個兒子的父親的「'''阿強加油<sup>2</sup>'''（<small>{{lang-ja|ツヨシしっかり<sup>2</sup>しなさい}}</small><ref group="注">此外該書名的「加油<sup>2</sup>」是念作「加油加油（{{lang|ja|しっかりしっかり}}）」。</ref>）」（單行本全6冊）。

改編方面，曾在1989年播出同名[[小小男子漢#電視劇|電視劇]]，1992年至1994年播出同名[[小小男子漢#動畫|電視動畫]]、並在動畫播出中的1993年上映單篇劇場版動畫。

此外，該作品主角井川強也有在原作者永松的另一部同樣在《Morning》連載的作品「虎男{{lang|ja|さんのお気に}}入{{lang|ja|り}}」中登場。

2007年10月起，同樣在講談社的網路可直接試閱的線上漫畫《{{link-ja|MiChao！|MiChao!}}》連載描寫井川強小學時期的「'''阿強同學加油'''（<small>{{lang-ja|ツヨシくんしっかりしなさい}}</small>）」。

2017年8月發行的[[小學館|小學館]]漫畫雜誌《[[Big_Comic_Original|Big Comic Original]]》17號連載的《{{lang|ja|{{link-ja|テツぼん|テツぼん}}}}》（[[高橋遠州|高橋遠州]]原作）中井川強也以{{lang|ja|ほくほくフーズ}}的社長身分登場。

== 電視劇 ==
1989年2月22日至7月12日的每週三於[[日本電視網協議會|日本電視網]]系列19時30分到20時00分時段首播。與原作不一樣的地方是，主角井川強在家有2個姊姊，電視劇則改成有3個姊姊；還有井川家父親在原作設定是從事派駐在外的工作所以不在家，電視劇則改成在家。

主角井川強由[[SMAP|SMAP]]（當時）成員[[森且行|森且行]]飾演。稍早以前原本決定由{{link-ja|高橋良明|高橋良明}}主演，但是在1989年1月5日高橋因機車意外事故去世，因此在急遽之下指定由森且行飾演。

== 電視動畫 ==
1992年10月4日至1994年12月25日在[[富士電視網|富士電視網]]首播，全112話。還有，本作品是在《[[櫻桃小丸子|櫻桃小丸子]]》第1期完結之後、至櫻桃小丸子第2期開播之前的播出時段墊檔放送。

該動畫在日本播畢之後，從未發行[[DVD|DVD]]或者在[[衛星電視|衛星電視]]再放送，不過從2017年10月31日起在[[Animax|Animax]]（日本當地）播出時，才第一次進行實質再放送（除了後來以[[VHS|VHS]]、[[LD|LD]]等影碟商品化的劇場版之外）。

1994年12月18日播出時的最高收視率為23.7%<ref>{{Cite web|url=https://www.j-cast.com/2017/05/11297697.html?p=2|title=「ツヨシしっかりしなさい」突然話題、アニメ版に「懐かしいなぁ」の声|publisher=Jcast NEWS|date=2017-05-11|accessdate=2017-09-03|language=ja}}</ref>。平均收視率為18.2%。

== 劇場版 ==
1993年於電視動畫播出期間上映，標題名稱《'''小小男子漢：阿強加油的時光機之旅'''（<small>{{lang-ja|ツヨシしっかりしなさい ツヨシのタイムマシーンでしっかりしなさい}}</small>）》（本作是《[[劇場版美少女戰士R|劇場版 美少女戰士R]]》上映之前的併映作品）。

=== 製作人員（劇場版） ===
* 原作：永松潔
* 企劃：清水賢治（富士電視台）
* 導演：三澤伸
* 編劇：[[山田隆司|山田隆司]]
* 人物設定、總作畫監督：一川孝久
* 美術設定：金村勝義（[[Studio_Jack|Studio Jack]]）
* 美術監督：相原澄子
* 攝影監督：森下成一
* 音響監督：三間雅文
* 剪輯：岡安肇（[[岡安Promotion|岡安Promotion]]）
* 宣傳負責：川崎悅子（富士電視台）
* 助理製作人：重岡由美子（富士電視台）
* 製作人：金田耕司（富士電視台）、茂垣弘道
* 配給：[[東映|東映]]
* 動畫製作：STUDIO COMET
* 製作著作：富士電視台
