'''小琉球池隆宮'''位於[[臺灣|臺灣]][[屏東縣|屏東縣]][[琉球鄉|琉球鄉]]漁福村三民路83號，是一座主祀[[池府千歲|池府千歲]]的[[廟宇|廟宇]]<ref>{{cite web |title=琉球鄉寺廟 |url=http://www.taiwantemple.com/2970529699371292354624287.html |website=Taiwan temple     台灣寺廟網 |accessdate=2018-08-06 |language=zh-TW}}</ref>，為漁福村漁埕尾庄頭廟。

== 緣起 ==
*清[[乾隆|乾隆]]期間，陳任的祖先由故鄉[[福建|福建]]恭請[[池府千歲|池府千歲]]金身渡海來臺，並在[[小琉球|小琉球]]定居，並供奉於自宅中，十年後陳任的祖先返回大陸恭請[[保生大帝|保生大帝]]香火至[[小琉球|小琉球]]供奉，並由[[乩童|乩童]]辦事，為民消災解厄，因神威顯赫，族人乃集資雕刻[[保生大帝|保生大帝]]金身供奉<ref>{{cite web |title=池隆宮 |url=http://crgis.rchss.sinica.edu.tw/temples/PingtungCounty/liouchiou/1322003-CLG |website=文化資源地理資訊系統 |accessdate=2018-08-06 |language=zh-tw}}</ref>。
*清[[道光|道光]]期間，則有先民由大陸迎請[[中壇元帥|中壇元帥]]（李哪吒三太子）前來供奉，同時隴西李姓族人也由故鄉迎請伍府千歲前來供奉，並由[[乩童|乩童]]辦事，為民消災解厄。
*清[[同治|同治]]期間，[[小琉球|小琉球]]中福村蔡論的祖先由大陸恭請[[李府千歲|李府千歲]]香火至陳家自宅中，亦由[[乩童|乩童]]辦事，為民消災解厄，因神威顯赫，亦由信眾集資雕刻金身。
*[[池府千歲|池府千歲]]、[[保生大帝|保生大帝]]及後續迎請之[[中壇元帥|中壇元帥]]、伍府千歲、[[李府千歲|李府千歲]]，均供奉在陳家祖宅，由於眾神神威顯赫，且各由[[乩童|乩童]]降鸞救世，形成鸞堂，早期由陳庭為管理人。
*[[1978年|1978年]](民國六十七年)左右，池隆宮手輦辦事時，有一神聖初始降駕時報頭銜為[[紫微大帝|紫微大帝]]，數次後表示化名為天元千歲並欲在[[琉球鄉|琉球鄉]]興基，因神威顯赫，故信眾集資雕刻金身，此為天元千歲由來。 

== 廟宇興建及改建 ==
*陳家祖宅在鸞生日見增加後，諸生深感空間狹窄，[[1955年|1955年]]（民國四十四年）九月，前[[琉球鄉|琉球鄉]]長王海，夢中受[[池府千歲|池府千歲]]指示興建廟事，遂與諸生決議建廟，並由李登無償提供廟地，於[[1956年|1956年]]（民國四十五年）六月完工。五十多年後，廟體已有斑駁頹廢現象，經合議於[[2007年|2007年]]（民國九十六年）農曆八月動工，於[[2009年|2009年]]（民國九十八年）農曆八月十六日舉行入廟安座儀式。
== 祀神 ==
（以神明方向區分左、右側）
*正殿：主祀[[池府千歲|池府千歲]]、左側祀奉[[李府千歲|李府千歲]]及[[保生大帝|保生大帝]]、右側祀奉[[中壇元帥|中壇元帥]]及伍府千歲與五營兵將，左龕奉祀[[境主公|境主公]]，右龕奉祀[[福德正神|福德正神]]及王船<ref>{{cite web |title=小琉球觀光旅遊資訊網 - 屏東縣琉球鄉官方旅遊網 |url=http://liuqiu.pthg.gov.tw/liuqiu/web_tw.php?prog=travel&con_id=72&view=&item=&page=1 |website=liuqiu.pthg.gov.tw |accessdate=2018-08-06}}</ref>（飛輪號）。
*左廂殿奉祀天元千歲、太歲星君牌位。
*右廂殿奉祀[[包府千歲|包府千歲]]、陪祀[[池府千歲|池府千歲]]、伍府千歲、王朝、馬漢、張龍、趙虎，並設置[[虎頭鍘|虎頭鍘]]、[[狗頭鍘|狗頭鍘]]，當地稱為五靈陰陽壇，廟方人員表示早期[[包府千歲|包府千歲]]與[[池府千歲|池府千歲]]共用[[乩童|乩童]]，降駕時必睡由108顆釘子釘成的釘床（存放於內殿），頭部墊金紙外以上半身赤裸躺在釘床上，有時並左右翻身，然後再起身辦案，非常獨特，除了審理陰陽糾結之事外，習俗上若有親人往生則子孫會上香秉告祈求[[包府千歲|包府千歲]]給予親人通關照應。

== 池府千歲來歷 ==
小琉球池隆宮池府千歲，非一般的池姓系統（如廈門市馬巷元威殿池府千歲-池然、臺南市北門區南鯤鯓代天府池府千歲—池夢颩、台南蚵寮保安宮深山尉池王-池金煥），而是屬非池姓系統（如臺南市馬兵營保和宮池府王爺—荊雲騰），依據廟方人員表示小琉球池隆宮池府千歲係黃帝開國朝臣，黃阜人氏，本姓劉，當年因奉命赴洛陽附近池縣接任縣令，沿途遇一奉玉旨下毒之小童，帶藥一包欲毒疫轄內百姓，池府想未上任規事，而百姓先遭瘟疫，於心不忍，便精思良策，藉口好奇想借看藥物，將小童藥物奪取服下，不惜生命，為救百姓捐軀，滿身變黑，玉旨調回池府魂魄審問，認為善行可嘉且因池縣百姓而亡，便敕封為池府千歲並令其為代天巡狩兼監斬官銜可先斬後奏，昇歸正神，受百姓永懷敬仰。
