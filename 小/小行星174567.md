'''（174567） 瓦爾妲''' （{{mp|2003 MW|12}}）是一颗[[海王星外天体|海王星外天体]]，[[绝对星等|绝对星等]]为3.4<ref name="jpldata"/>。它很有可能成为一颗[[矮行星|矮行星]]。它由[[杰弗里·A·拉森|杰弗里·A·拉森]]于2003年6月21日通过[[太空监视|太空监视]]望远镜发现<ref name=Larsen2007>{{Cite journal|last=Larsen|first=Jeffrey A.|coauthors=Roe, Eric S.; Albert, C. Elise ''et al.''|title=The Search for Distant Objects in the Solar System Using Spacewatch|year=2007|journal=The Astronomical Journal|volume=133|issue=4|pages=1247–1270|doi=10.1086/511155|bibcode=2007AJ....133.1247L}}</ref>。

它目前距离太阳47.5天文单位<ref name="AstDys"/>，将于2096年11月到达[[拱点|近日点]]<ref name=Buie/><ref name="Horizons">{{Cite web
  |title=HORIZONS Web-Interface
  |publisher=[http://ssd.jpl.nasa.gov JPL Solar System Dynamics]
  |url=http://ssd.jpl.nasa.gov/horizons.cgi?find_body=1&body_group=sb&sstr=174567
  |accessdate=2009-08-24}}</ref>。它在14次[[冲_(天体位置)|冲]]时已被观测到58次，最早的照片可追溯到1980年<ref name="jpldata"/>。

[[迈克尔·E·布朗|迈克尔·E·布朗]]自动更新的网站将它列为很有可能的[[矮行星|矮行星]]，但它的直径至今还没有测得<ref name="Brown-dplist">{{cite web
