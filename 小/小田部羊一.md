'''小田部羊一'''（{{bd|1936年|9月15日|catIdx=Kotabe Yōichi}}）是[[日本|日本]]的男性[[動畫師|動畫師]]、[[電子遊戲|電子遊戲]]插圖及[[人物設計|角色設計家]]。

代表參與的動畫作品為《[[世界名作劇場|世界名作劇場]]》系列，電子遊戲則有《[[超級瑪利歐|超級瑪利歐]]》、《[[薩爾達傳說|薩爾達傳說]]》系列等。

==經歷==
===出生===
小田部羊一出生於[[台灣日治時期|日治時代]]的[[台灣|台灣]][[台北市|台北市]]，<ref name="Geidai">{{Cite web |url=http://animation.geidai.ac.jp/open2011/#anchor02 |script-title=ja:東京藝術大学大学院映像研究科アニメーション専攻 OPENイベント2011 |work=[[東京藝術大學|東京藝術大學]] |language=ja |accessdate=2013年2月26日 }}</ref>在[[二戰|二戰]]結束後則跟著家人返回日本的[[奈良縣|奈良縣]]生活，之後也曾搬移至[[新潟縣|新潟縣]]、[[茨城縣|茨城縣]]等地居住。

===動畫師時期===
在就讀[[東京藝術大學|東京藝術大學]]日本畫科後，小田部羊一進入了[[東映|東映]]動畫公司就職，<ref name="Geidai" />當時他主要合作的同事有[[大塚康生|大塚康生]]、[[高畑勳|高畑勳]]、[[宮崎駿|宮崎駿]]等人。當時在東映期間小田部羊一與他們合作的代表動畫為《[[太陽王子_霍爾斯的大冒險|太陽王子 霍爾斯的大冒險]]》。

1971年在大塚康生的提議之下，小田部羊一與高畑勳及宮崎駿陸續從東映跳槽到A Production動畫製作公司，以進行將[[阿斯特麗德·林格倫|阿斯特麗德·林格倫]]的著作《[[長襪子皮皮|長襪子皮皮]]》改編成動畫的計畫，<ref>{{Cite book |title= 出發點1979～1996 |location=台灣 |publisher=[[台灣東販|台灣東販]] |author=[[宮崎駿|宮崎駿]] |language=zh-hant |date=2006年2月1日 |pages=441頁  |ISBN=978-957-473-843-4 |accessdate=2013年2月26日 }}</ref>不過此計畫最後因得不到作者阿斯特麗德的允許而告吹。<ref>{{Cite book |title= 出發點1979～1996 |location=台灣 |publisher=[[台灣東販|台灣東販]] |author=[[宮崎駿|宮崎駿]] |language=zh-hant |date=2006年2月1日 |pages=542頁  |ISBN=978-957-473-843-4 |accessdate=2013年2月26日 }}</ref>而2年後小田部羊一又與高畑勳等人再次轉職到[[日本動畫公司|日本動畫公司]]裡，在此間則參與了《[[世界名作劇場|世界名作劇場]]》系列中《[[小天使|小天使]]》、《[[尋母三千里|尋母三千里]]》、《[[小浣熊|小浣熊]]》等動畫製作。至1977年小田部羊一則從日本動畫公司離職，改以自由業身份活動一段時期。<ref name="Yomiuri" />

===任天堂時期===
1985年小田部羊一在先前在東映共事的同事[[池田宏|池田宏]]邀請之下，進入了電子遊戲公司[[任天堂|任天堂]]就職。<ref>{{Cite web |url=http://www.nintendo.co.jp/ds/interview/dsi/vol8/index3.html |script-title=ja:任天堂から誘いを受けて |work=[[任天堂|任天堂]] |language=ja |accessdate=2013年2月26日 }}</ref>在任天堂裡小田部羊一擔任了《[[瑪利歐系列|瑪利歐系列]]》、《[[精灵宝可梦|精灵宝可梦]]》裡眾多作品的遊戲插圖繪製、或是角色設計相關事務。<ref name="Geidai" />2007年則從任天堂離職，再次以自由業創作者身份活動。

===其它===
小田部羊一曾與妻子[[奧山玲子|奧山玲子]]組成兩人專用的工作室「{{lang|ja|アトリエ羚}}」，而倆人共同使用的筆名則為「{{lang|ja|あんていろーぷ}}」。

除曾擔任動畫師及遊戲插圖畫家外，小田部羊一也曾擔任過{{nihongo|東京動畫研究所|東映アニメーション研究所}}及母校東京藝術大學的講師。<ref name="Geidai" />另個人畫展則曾在2010年夏季於[[東京都|東京都]]展出。<ref>{{Cite web |url=http://anime.webnt.jp/news/view/2035 |script-title=ja:「ハイジ」の小田部羊一、サイン会開催 |work=Web NewType |language=ja |date=2010年7月30日 |accessdate=2013年2月26日 }}</ref>
