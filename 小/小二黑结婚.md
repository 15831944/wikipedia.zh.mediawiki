《'''小二黑结婚'''》，是[[中國|中國]]作家[[趙樹理|趙樹理]]创作于1943年的[[短篇小說|短篇小說]]，是中国[[现代文学|现代文学]]中著名的短篇小说之一。

== 概述 ==
《小二黑结婚》由真實事件改編，是一個歌颂婚姻自由的故事。该小说讲述了一對農村青年小二黑和小芹通过与[[迷信|迷信思想]]及[[封建制度_(中国)|封建势力]]作斗争，争取婚姻自由的故事。故事中小二黑和小芹的[[自由戀愛|自由戀愛]]受到流氓惡霸村長金旺的阻撓；二諸葛由於迷信也反對小二黑與小芹的結合，私下给小二黑收養一位[[童养媳|童养媳]]；三仙姑為貪財而出賣女兒，逼小芹嫁给吃喝嫖赌的吴广荣。一天小二黑和小芹暗地商量终身大事时，被金旺等捆住。雙方家長前往村長家要人，發生一連串群殴事件。後來由抗日民主区政府区长出面支持，经过一番斗争，嚴惩恶棍金旺，故事最後以喜劇收場，二诸葛和三仙姑表示支持這件婚事<ref>{{cite book|author1=教育部中文学科教学指导委员会组编|editor=丁帆主编|title=中国新文学史 上|date=2013年4月|publisher=北京：高等教育出版社|isbn=978-7-04-037035-5|pages=351|accessdate=2015-11-05}}</ref>。

== 创作经过和改编 ==
這部小說甫一推出極受好評，[[中国共产党|中国共产党]]晉察冀中央局宣傳部長[[周揚|周揚]]在《新的人民的文藝》一文中稱讚：“反映農村鬥爭的最杰出的作品，也是解放區文藝的代表之作。”[[彭德怀|彭德怀]]也曾在這部小說上題詞：「像這樣從群眾調查研究中，寫出來的通俗故事，還不多見。」<ref>{{cite book|author1=朱德发编|title=现代中国文学史精编 1900-2000|date=2013年1月|publisher=济南：山东教育出版社|isbn=978-7-5328-7703-4|pages=146|accessdate=2015-11-05}}</ref>

《小二黑结婚》于1950年及1964年分别由[[顾而已|顾而已]]、[[于学伟|于学伟]]导演拍摄成两部同名电影搬上银幕。
