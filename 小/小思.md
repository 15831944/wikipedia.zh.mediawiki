'''小思'''（1939年－），原名'''盧瑋鑾'''，筆名「小思」、「明川」、「盧颿」，生於香港，籍貫廣東番禺，香港當代著名散文作家、教育家。

畢業於[[香港中文大學|香港中文大學]][[新亞書院|新亞書院]]中文系，獲[[香港大學|香港大學]]中文系哲學碩士銜。1973年於[[京都大學|京都大學]]人文科學研究所擔任研究員。曾任香港中文大學[[香港文學研究中心|香港文學研究中心]]主任，已於2002年退休。

==學術貢獻==
小思活躍於香港文化活動，是六七十年代《中國學生周報》的作者之一，歷任多屆公開徵文比賽評判，如[[青年文學獎|青年文學獎]]，[[香港文學雙年獎|香港文學雙年獎]]等。

小思多年來研究和整理香港文學及文化史料，曾捐贈超過20,000項個人藏品（包括剪報、期刊、書籍、小冊子及論文等資料）予[[香港中文大學|香港中文大學]]圖書館，協助成立「香港文學特藏」及「香港文學資料庫」。<ref name="lib2015">{{cite web |url=http://www.lib.cuhk.edu.hk/tc/about/news/spc-news-congratulations |title=恭賀盧瑋鑾教授（小思）榮獲「2015香港藝術發展獎」之「終身成就獎」}}</ref><ref>{{cite web |url=http://www.commentshk.com/2010/03/blog-post_31.html |title=關於香港──我知道得愈多，就愈有感情（訪問盧瑋鑾）|date=2010-04-01 |author=梁文道 }}</ref> 小思並開展香港文學口述歷史。她亦曾為中、小學教師。曾在香港《[[星島日報|星島日報]]‧星辰》、《[[明報|明報]]》副刊等撰寫專欄，於2014年10月11日封筆。其著作以散文為主，亦有不少學術論著。

小思曾獲[[香港教育學院|香港教育學院]]第二屆《[[傑出教育家獎|傑出教育家獎]]2003》，以表揚她對香港文學教育的貢獻。2010年獲[[香港藝術發展局|香港藝術發展局]]頒發「2009香港藝術發展獎」之「傑出藝術貢獻獎」。<ref>{{cite web |url=http://www.hkadc.org.hk/?p=3557&lang=tc |title=香港藝術發展獎 得獎名單 2009}}</ref> 2016年4月21日，小思榮獲「2015香港藝術發展獎」之「終身成就獎」。<ref name="lib2015" />

==作品觀點==
著作中有一篇文章提及電影《布拉格之春》、《布拉格的大學生》，述及《布拉格的大學生》以為自己甚麼也沒有，就出賣自己的影子，而《布拉格之春》入面的醫生因拒絕簽署悔過書，而甘願成為抹窗工人。
