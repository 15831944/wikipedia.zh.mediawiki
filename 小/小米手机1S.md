'''小米手机1S'''是一款由[[小米科技|小米科技]]于2012年8月16日推出的[[智能手机|智能手机]]，该机基于[[Android|Android]]操作系统，搭载[[MIUI|MIUI]]。作为[[小米手机M1|小米手机M1]]的升级版本，小米手机1S配备了高通双核1.7GHz处理器，主鏡頭改用F2.2光圈的背照式鏡頭<ref>http://webcache.googleusercontent.com/search?q=cache:E_zON9NTBxoJ:www.lmubbs.net/read.php%3Ftid-3133.html</ref><ref>http://www.cnmo.com/reviews/175471.html</ref>，并添加了一个200万像素的前置摄像头，除此之外，其他的硬件配置则与第一代小米手机相同<ref name="评测">{{cite web| url=http://tech.sina.com.cn/mobile/n/2012-08-28/10467554168.shtml|title=无需等待 双核超值款小米1S现货仅1799| author=新浪网|date=2012年8月28日|accessdate=2012-09-03}}</ref>。

== 发布 ==
2012年8月16日，小米科技在位于[[北京|北京]]的[[798艺术区|798艺术区]]举行了“小米2012年新品发布”，在发布会中，小米科技创始人[[雷军|雷军]]正式发布了两台智能手机，一台为第一代小米手机的升级版小米手机1S，另一台为4核智能手机[[小米手机2|小米手机2]]。在发布会上，雷军公布了小米手机1S的价格为人民币1499元，并开放网络预订<ref name="发布会">{{cite web| url=http://www.cnbeta.com/articles/201785.htm|title=“小米手机1S”1499元 “小米手机2”1999元| author=CNBETA|date=2012年8月16日|accessdate=2012-09-03}}</ref>。

== 发售 ==
小米手机1S于2012年8月23日北京时间上午10点开放购买，共有140万名用户预订，但首轮只放出20万台手机。开放购买29分36秒后，20万台手机即告售罄。小米科技联合创始人黎万强在微博上表示抢购到这20万台小米手机1S的用户将在7天内收到货物，并称下一轮发售将于近期开始<ref name="第一轮">{{cite web|url=http://www.enet.com.cn/article/2012/0823/A20120823153988.shtml|title=小米手机1S首轮20万台29分36秒售完|author=硅谷动力|date=2012年8月23日|accessdate=2012-09-03|deadurl=yes|archiveurl=https://web.archive.org/web/20120824183809/http://www.enet.com.cn/article/2012/0823/A20120823153988.shtml|archivedate=2012年8月24日}}</ref>。2012年9月6日上午10点，小米手机1S第二轮发售开始，此轮发售放出15万台小米手机1S与5万台电信版小米手机1S<ref name="第二轮">{{cite web|url=http://www.enet.com.cn/article/2012/0903/A20120903157730.shtml|title=小米手机1S于9月6日开启第二轮发售|author=硅谷动力|date=2012年9月3日|accessdate=2012-09-03|deadurl=yes|archiveurl=https://web.archive.org/web/20120904095915/http://www.enet.com.cn/article/2012/0903/A20120903157730.shtml|archivedate=2012年9月4日}}</ref>，仅用9分40秒，这二十万台小米手机1S即告售罄<ref name="第二轮售罄">{{cite web| url=http://cnbeta.com/articles/204649.htm|title=9分40秒售磬20万台 小米手机1S持续热卖| author=CNBETA|date=2012年9月6日|accessdate=2012-09-06}}</ref>。

== 争议 ==
[[奇虎360|奇虎360]]董事长[[周鸿祎|周鸿祎]]在小米手机1S首轮发售售罄后认为这是小米科技在变相降价，损害了以人民币1999元购买第一代小米手机的消费者的利益，并建议雷军给在半年内购买第一代小米手機的用户返现700元<ref name="降价">{{cite web| url=http://finance.sina.com.cn/chanjing/gsnews/20120902/095813022255.shtml|title=小米手机营销手段供应链陷争议 售后被指系软肋| author=新浪网|date=2012年9月2日|accessdate=2012-09-03}}</ref>。除此之外，也有人在成本问题上质疑小米手机1S，认为小米手机1S的成本仅有900元。但随后雷军辟谣称小米手机仅有100至200元的利润<ref name="降价"/>。
