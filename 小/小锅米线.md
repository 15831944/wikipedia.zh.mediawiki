'''小锅米线'''是[[云南省|云南]][[昆明市|昆明]]一种特殊烹饪方法制成的[[米线|米线]]<ref name="云游网">{{cite web |url=http://www.mnw.cn/tour/guide/1510884.html |title=云南旅游哪里的米线好吃--云南米线之旅 |author=<!--Staff writer(s); no by-line.--> |date=2016-12-22 |website=www.mnw.cn |publisher=云游网 |access-date=2018-07-01 |quote=}}</ref>，是昆明知名的小吃<ref>{{cite web |url=http://news.163.com/13/0914/06/98NBJ3MB00014AEE.html |title=打造世界知名旅游城市昆明不能只有小锅米线 |author1=肖辉龙 |author2=彭婷婷 |date=2013-09-14 |website=news.163.com |publisher=云信网 |access-date=2018-07-01 |quote=另一方面，吃的大多都只知道“小锅米线”}}</ref>，口味兼有酸味、鲜味、甜味和辣味<ref name="云游网" />。小锅米线的价格较为低廉，少有价格达到20元一碗的米线<ref>{{cite web |url=http://news.ifeng.com/gundong/detail_2013_11/06/30990288_0.shtml |title=昆明首现20元1碗小锅米线 |author=<!--Staff writer(s); no by-line.--> |date=2013-11-06 |website=news.ifeng.com |publisher=云南信息报 |access-date=2018-07-01 |quote=昆明北市区小康大道和谐世纪天膳美食广场有家昆夫小锅的小锅米线卖20块钱一碗，卖得云南第一贵？}}</ref>。

==制作==

小锅米线的煮制使用红铜的小锅，小锅口径较碗略大，使用高汤熬煮<ref name="云游网" />。待汤煮沸后，放入剁肉帽子（肉沫）、[[豌豆|豌豆]]尖、[[酸菜|酸腌菜]]、[[韭菜|韭菜]]、[[西红柿|西红柿]]、[[酱油|酱油]]和油辣椒或糊辣子等，最后放入米线<ref name="云游网" />，可依个人口味加入臭豆腐等其他食材<ref>{{cite web |url=http://news.ifeng.com/gundong/detail_2013_07/30/28054948_0.shtml |title=前卫营的小锅米线生意火爆了十多年 |author=<!--Staff writer(s); no by-line.--> |date=2013-07-30 |website=news.ifeng.com |publisher=云南信息报 |access-date=2018-07-01 |quote=}}</ref><ref>{{cite web |url=http://news.163.com/13/1029/06/9CB7M13R00014AEE.html |title=糊辣子做主调料这小锅米线忒香 |author=陈懿然 |date=2013-10-29 |website=news.163.com |publisher=云信网 |access-date=2018-07-01 |quote=}}</ref>。

==知名小锅米线==

昆明较为知名的小锅米线品牌有端仕小锅米线、万丰小锅米线、青和小锅米线、铁棚小锅米线、昆夫小锅米线等。<ref>{{cite web |url=http://news.ifeng.com/gundong/detail_2013_11/20/31397352_0.shtml |title=昆明“第一次小锅米线大战”开打 |author=<!--Staff writer(s); no by-line.--> |date=2013-11-20 |website=news.ifeng.com |publisher=云南信息报 |access-date=2018-07-01 |quote=}}</ref>
