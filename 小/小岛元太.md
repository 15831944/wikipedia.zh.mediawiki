'''-{zh-hans:小岛元太; zh-hk:小島元太; zh-tw:小嶋元太}-'''（日語：{{lang|ja|小嶋 元太}}/{{lang|ja|こじま げんた}}，Kojima Genta），[[少年侦探团|少年侦探团]]的發起人及成员，是日本[[漫画|漫画]]和[[动画|动画]]作品《[[名侦探柯南|名侦探柯南]]》中的[[名偵探柯南角色列表|虚构角色]]。

命名來自已故的[[日本|日本]][[小說家|小說家]]'''{{tsl|ja|小峰元}}'''。

== 人物 ==
7岁，体重45公斤，就讀於[[东京都|东京都]]米花市帝丹小学一年B班，[[少年侦探团|少年侦探团]]的創辦人；父亲是小島元次，家里经营「小島酒馆」。

可說是個小屁孩，像是爬到郵筒上看遊行，被指責後還會不以為然的說有什麼關係，還會在別人家東摸西碰，有次在停車場足球踢到一位外國人的車，還一副關我甚麼事的樣子，雖說孩子本就頑皮，但元太確實相當白目。時常將少年偵探團帶入危險之中，完全是個不定時炸彈。

在学校裡把自己的鞋柜改成接受委托的信箱，雖然自認是偵探團的团长，並將[[江户川柯南|江户川柯南]]視為手下，但幾乎每次都是由柯南指挥行​​动。此外，其推理能力極為差勁，有时還会与[[圆谷光彦|圆谷光彦]]和[[吉田步美|吉田步美]]做出愚蠢的推理或誇張的想像，令柯南搖頭不已。

== 形象 ==
個性極為頑皮活潑，但也有充满正义感的一面。除了喜欢'''假面超人'''之外，也非常爱吃鳗鱼饭；凡事都會往食物方面想，一听到鉅額的款項时，就會拿鳗鱼饭來比较。

== 情感 ==

由於喜欢步美，所以对柯南多少有所排斥。
