}}'''小米手机'''是中国科技公司[[小米科技|小米科技]]旗下的[[智能手机|智能手机]]系列品牌，于2011年8月16日在北京发布了首款产品。

小米手机系列被定位为小米的智能手机主力系列，主打中高端市场，重视用户体验和技术创新。产品价格初期定为1999元人民币起，为小米集团成立早期在中国市场确立领导地位产生了决定性的作用。目前，小米手机的产品价格区间在1099元至4999元人民币不等，在产品线的划分方面，2015年之前每年仅发布一款年度旗舰，自小米Note开始，小米推出了针对不同消费者人群的若干个分支产品线。

小米手机以及后续迭代产品一经推出便销量火爆，甚至一度因供不应求导致限量抢购。2014年曾位列中国手机市场销量排行第一名，但后期由于产品过于缺货以及其他竞争品牌推出的产品相对小米手机有更强竞争力等原因，小米手机的销量出现了一定程度的下滑。截至2018年末，小米位列中国手机市场排行第五名，落后于[[华为|华为]]（含子品牌[[荣耀|荣耀]]）、[[OPPO|OPPO]]、[[vivo|vivo]]和[[蘋果公司|苹果]]。

2019年1月3日，小米公司通过微博宣布红米手机将从“产品线”升级为独立的子品牌，使得小米手机成为小米集团旗下主打中高端的手机品牌，与华为、OPPO和vivo进行竞争。宣布此战略之后小米发布的第一款手机为[[小米9|小米9]]以及其小屏衍生版本[[小米9_SE|小米9 SE]]。

== 产品分类 ==
截至2019年2月，小米手机系列分为以下几个产品线：
*'''小米数字系列'''：以小米+数字为产品命名的主力旗舰机型。<br/>这一系列的手机均搭载高通当年最新研发的SoC，自小米2开始搭载气压传感器，自小米5开始支持能够进行读卡、写卡和模拟卡的NFC功能、UFS闪存以及Type-C标准的数据传输接口，自小米8开始转为全面屏大屏旗舰，同时暂停发布并取代了小米Note系列，是小米手机产品线中体验最均衡，性价比适中的产品序列。1、2、4、5代均推出了后续加强机型1S、2S、4S、5S<ref>[[小米手机1S|小米1S]]和[[小米手机2S|小米2S]]为基于前代机型的平台上做出硬件规格提升的后续加强机型，[[小米手機4S|小米4S]]为基于[[小米手機4c|小米4C]]平台的优化改进机型，而[[小米手機5s|小米5S]]某种意义上来说则更像是[[小米5|小米5]]的金属后盖版本，性能及硬件提升幅度不大。</ref>。
**'''小米SE系列'''：以小米+数字+SE为产品命名的主力次旗舰机型。衍生代码的SE意为Small Edition，即小屏版。<br/>在小米8升级为大屏主力旗舰之后，为了照顾原有部分喜爱小屏的用户，小米增设了这一产品序列。其与同代数字旗舰同时发布但稍晚上市，采用高通准旗舰处理器序列骁龙700系列，核心配置和外观工艺略有缩水，但和同代数字系列一样主打均衡的体验以及适中的性价比。
**'''小米X系列'''：以小米+数字+X为产品命名的线下市场推广机型。衍生代码X意为Xinlingshou，即“新零售”的汉语拼音首字母。<br/>在小米6发布之后，为促进小米品牌向广大的线下市场推广，小米在2017年增设了这一产品序列。手机主打轻薄的外观以及优良的拍照水准，体验也有部分兼顾。
**'''小米C系列'''：以小米+数字+C为产品命名的实验性质机型。衍生代码C意为China，中国一词的首字母。<br/>这一系列仅推出了小米4C和小米5C两款机型。小米4C为先期发布的小米4i为照顾中国市场用户的产品需求而定制的一款机型，而小米5C则首次搭载了其自主研发的处理器——澎湃S1。
**'''小米A系列'''：以“小米+数字+A”命名的低配置派生机型。A意为“青春版”，是Adolescent的首字母。<br/>这一系列仅有小米2A一款，后被子品牌红米取代。2017年重新启用A代号，但格式变为“小米+A+数字”（A意为Android One）；如小米A1，接替小米i系列成为X系列的国际版定制机型，仅在海外市场发售，搭载原生安卓系统。
**'''小米i系列'''：以“小米+数字+i”命名的派生机型（i意为International，国际一词的英文首字母），针对国际市场推出，仅发布了小米4i一款机型。
*'''小米MIX系列'''：以“小米+MIX+数字”命名，是2016年起推出的机型系列，搭载陶瓷机身以及较大尺寸的显示屏。初代定位为汇集最新可行研发科技的概念机型，之后定位为量产的高端旗舰机型，从MIX 3开始取代小米Note系列作为商务旗舰发布，是当前小米手机各产品线定价最高的机型。
*'''小米Note系列'''：以“小米+Note+数字”命名，是2015年起推出的机型系列，比每年发布的小米手机系列屏幕略大，前两代为针对商务市场打造的高端旗舰机型，因市场反应不佳、产品定位模糊以及MIX系列发布，因此在3代定位为中端次旗舰机型。自2018年起，小米Note系列中断发布，转由提升屏幕大小之后的小米数字系列替代，如[[小米8|小米8]]和[[小米9|小米9]]。
*'''小米Max系列'''：以“小米+Max+数字”命名，是2016年起推出的机型系列，是尺寸介乎于平板电脑与智能手机之间的“平板手机”，特点为大屏幕及大电池容量。处理器平台及影像系统规格基本与同年的红米Note系列相同。
*'''小米Play系列'''：以“小米+Play”命名的小米系入门级智能手机系列，主推版本重点宣传流量不限量，是小米借助其虚拟运营商身份推出的新的产品线，第一代使用了联发科技的P35处理器。

== 机型列表 ==
以下是小米自2011年8月16日发布第一款小米手机以来，小米科技及上市之后的小米集团发布的所有智能手机机型，按照发布时间排序。

需要注意的是，在部分国家和地区销售的手机型号与本列表所列的型号有出入，如小米5X在印度等国际市场的商品名称为小米A1，小米6X在国际市场的商品名称为小米A2。此外，列表中的小米4i并未在中国大陆地区市场销售，该款机型经过提升核心配置后，在中国大陆市场以“小米4c”的名称发布。

小米手机在2018年直接跳过了第7代的发布，改为发布第8代。

== 功能 ==

=== 气压及海拔测量 ===
小米手机数字系列自第2代开始支持气压及海拔的测量，在“指南针”应用中罗盘下方即可显示实时气压以及实时海拔。但海拔测量功能如需相对精确数据仍需进行联网校正。后续的小米Note、小米MIX系列（1、2、2S）也都支持这个功能，自小米9开始取消。

=== 全功能近场通讯技术 ===
[[近距離無線傳輸|近场通讯技术]]（NFC）自小米手机2A首次支持。此后发布的小米3也支持这一功能，通过支持NFC智能卡功能的SIM卡可实现用手机乘坐公交和地铁。但由于该功能使用人数较少，小米4和小米Note一度取消，自小米手机5开始重新支持，并加入了读取NFC智能卡、写入NFC智能卡、模拟公交卡和门卡等功能，小米称之为“全功能NFC”。

目前全功能近场通讯技术仅在[[小米手機5|小米手机5]]（含5S和5s Plus）、6、8、9、9SE，小米Note2、3，小米MIX、MIX2、MIX2S、MIX3上得到支持，均为小米公司旗下高端机型。

=== 红外遥控功能 ===
小米手机自小米手机4开始支持红外遥控功能，其自带的“万能遥控”APP支持若干个品牌的，可通过红外遥控功能操作的电器设备。但该功能在MIX系列一直未获得支持，小米数字系列的小米5S和小米8两款机型也一度取消该功能。此外，小米Note初代也不支持此功能。
=== “小爱同学” ===
小米在小米5X发布会上一同发布了人工智能引擎“小爱同学”，随即替换掉之前在MIUI上内置的语音助手，并采用了全新的虚拟形象以及声源。自小米MIX2S开始凭借骁龙845芯片的NPU模块，支持了语音唤醒“小爱同学”，并在随后的小米MIX3和小米9上添加了AI按键，以便于快捷呼出相应功能。
== 系统界面 ==
{{main|MIUI}}
所有的小米手机系列均搭载由小米公司基于[[Android|Android]]研发的MIUI固件，目前的最新版本为MIUI10。手机出厂时预装“稳定版”系统固件，用户可自行从MIUI网站上下载MIUI开发版固件，也可加入论坛的“内测粉丝组”，通过内测通道安装内测版（分开发版内测与体验版内测）固件。无论是内测版、开发版还是稳定版固件，均通过[[空中编程|OTA]]进行推送更新。

小米的MIUI官方网站提供所有的小米手机固件的下载，并提供平台与其他MIUI用户进行交流。不同于中国国内大多数手机厂商对于固件包的下载和[[Root_(Android)|root]]的限制，小米公司并未阻止手机用户对系统固件进行root，但为安全考量，小米公司在小米手机5之后的机型添加了刷机锁，以避免不法分子对系统进行破解。MIUI官网也提供工厂预装固件所用的应用程序“MiFlash”以备用户下载使用。

值得一提的是，小米支持免费维修用户自行因root导致的硬件故障的手机，系统是否root并不影响手机的保修，这与中国国内其他厂商的售后政策有所不同。

此外，小米在中国大陆以外的国家和地区销售的机型内置了完整的谷歌服务以及谷歌应用商店，而小米A系列机型则不内置MIUI固件，以谷歌官方开发的Android原生固件代替。

== 争议与质疑 ==
小米手机及其迭代产品自发布以来便饱受社会各界质疑。除1999元起的定价能否长期维持小米公司正常运营的质疑以外，其缺货、宣传参数模糊、设计、营销模式、质量等诸多方面也饱受社会争议。

=== 缺货 ===
小米手机自初代发售至今便饱受缺货困扰，几乎任何一代产品自发售始无一例外地陷入产量不足的窘境。小米集团总裁林斌在微博表示，小米手机的缺货一定程度上是由于内部流程和管理上缺乏经验造成<ref>{{Cite web|url=http://www.techweb.com.cn/ucweb/news/id/2726707|title=林斌解释小米9缺货原因：内部的流程和管理上缺乏经验|accessdate=|author=|date=|publisher=techweb}}</ref>，但也有人指出小米在刻意制造货源不足的“假象”以长久保持其产品关注度<ref>{{Cite web|url=http://baijiahao.baidu.com/s?id=1569010034280310&wfr=spider&for=pc|title=华为让雷军慌了! 小米6自称缺货是技术问题, 是真是假?|accessdate=|author=|date=|publisher=}}</ref>。一些观点认为，小米缺货的原因与售价偏低、溢价能力低下、轻资产模式的抗风险手段以及无自有工厂等诸多原因有关。

=== 宣传参数模糊 ===
小米在部分手机产品的宣传上存在一定程度的模糊不清现象，也被一些舆论认为是“虚假宣传”<ref>{{Cite web|url=http://m.qudong.com/pcarticle/252657|title=生于炒作死于宣传：虚假宣传的小米如今被立案调查！|accessdate=|author=|date=2015-9-29 |publisher=驱动中国}}</ref>。如小米3联通/电信版的处理器部件代号分别为MSM8274AB和MSM8674AB，但发布会上宣传则为支持4G网络频段的MSM8974AB。<ref>小米3发售时并未宣传联通/电信版支持4G网络频段，但发布会上宣传的处理器型号则为MSM8974AB，其中部件代号中的“9”为支持4G全网通的型号，“2”和“6”分别对应支持WCDMA网络以及WCDMA/CDMA2000网络的型号。</ref><ref>{{Cite web|url=http://news.mydrivers.com/1/288/288537.htm|title=小米3联通版处理器8974AB变8274AB！高通如是说|accessdate=|author=|date=|publisher=}}</ref>小米官方虽表示接受用户的退款请求<ref>{{Cite web|url=http://news.mydrivers.com/1/288/288504.htm|title=小米的官方声明：小米3联通版可全额退款|accessdate=|author=|date=|publisher=}}</ref>，但在这两个版本上线之后依然引发了轩然大波。此外，小米手机2宣传性能堪比XBOX游戏机<ref>{{Cite web|url=https://www.gamersky.com/news/201208/209648_2.shtml|title=小米2图形性能堪比Xbox 360？解析真相|accessdate=|author=|date=|publisher=}}</ref>、小米手机5低配版采用降频版的高通骁龙820芯片等宣传也有夸大或模糊不清的情况。

=== 设计 ===
小米手机的外观设计一直以来均相较国内其他品牌来说缺乏亮点。小米科技CEO雷军在小米第一款手机发布会上称其设计为“没有设计就是最好的设计”<ref>{{Cite web|url=https://www.ithome.com/0/410/322.htm|title=雷军：没有刻意的设计才是最好的设计|accessdate=|author=|date=|publisher=}}</ref>，一时也被部分网友吐槽至今。2018年推出的小米及红米系列手机则出乎意料地采用了类似于iPhone X的“左上角竖排双摄”设计，有媒体直接指出小米这种行为是“赤裸裸的抄袭”<ref>{{Cite web|url=http://news.ltn.com.tw/news/business/breakingnews/2444181|title=小米8激似iPhoneX 美媒痛批“无耻抄袭”|accessdate=|author=|date=|publisher=}}</ref>。

=== 营销模式 ===
小米除被冠以“饥饿营销”之名以外，其营销模式也引起多方质疑。一些舆论认为，小米擅长以“扁平化的粉丝营销”作为其品牌产品大规模扩张的模式<ref>{{Cite web|url=http://www.woshipm.com/it/36667.html|title=小米如何让用户尖叫（上）：扁平化和粉丝营销真相|accessdate=|author=|date=|publisher=}}</ref>，更有部分观点认为小米对外宣传与树立了一个其CEO[[雷军|雷军]]“良心企业家”的形象，却避而不谈雷军个人从小米公司得到的大量财富，有“人设营销”甚至“牌坊营销”的嫌疑<ref>{{Cite web|url=http://www.sohu.com/a/237465460_373105|title=小米上市前夕，雷军的98亿红包，打了谁的脸！？|accessdate=|author=|date=|publisher=}}</ref>。

=== 质量及品控 ===
虽小米手机的品控及返修率已经随着后续机型的迭代而逐年改善，但小米手机依然被网络上爆出大量品控及质量问题，如花屏、死机、耗电大、WiFi或蜂窝网络断流、边框和后面板玻璃衔接处缝隙大（以小米6为甚）、镀色不均或孔位缺失等。如2019年3月7日，微博有网友爆出新买到的小米9手机无闪光灯<ref>{{cite web |author1=快科技 |title=网友买到小米9无闪光灯：官方紧急调换 补偿保护套 |url=https://tech.sina.com.cn/mobile/n/n/2019-03-08/doc-ihsxncvh0721946.shtml |website=新浪科技 |accessdate=2019-03-08}}</ref>等。部分品控问题小米会立刻派客服人员向用户了解情况，但也有部分网友反应用户在向小米及相关主管人员的微博下反应故障时，不但未获得小米公司的受理，反而被小米公关人员删除评论或封禁账号。
