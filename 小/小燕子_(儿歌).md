'''《小燕子》'''作词：[[王路|王路]]、[[王云阶|王云阶]] 作曲：[[王云阶|王云阶]]。是中国大陆传唱最广的[[儿歌|儿歌]]之一。来自于中国大陆1957年的电影《[[护士日记|护士日记]]》的插曲。

== 产生 ==
1955年5月，大冶师范学校（今[[湖北理工学院|湖北理工学院]]）教师[[王路|王路]]，因眼疾住院治疗。躺在病床上的他，忽见窗外燕子飞过，想起“六一”[[兒童節|儿童节]]马上要到了，当即伏案写下几首儿歌，包括《好菜园》《花婶婶儿》《小黑孩儿》《小老鼠儿》《小燕子》等5首儿童诗。这组诗最早刊登在大冶师范的墙报上，后经人推荐，发表在1956年6月号的《长江文艺》上。

不久，电影[[护士日记|《护士日记》]]在上海开拍，上海电影厂著名作曲家[[王云阶|王云阶]]接到了为该片谱写主题曲的任务。[[王云阶|王云阶]]翻阅了当时全国很多报刊杂志，最后相中了《小燕子》。[[王云阶|王云阶]]当即写信给[[王路|王路]]，希望双方合作。同时，提出该诗长达三个自然段共20行，不能全部作为主题歌词，最适宜为之谱曲入歌的只有第一个自然段，且觉得这个自然段第二行的“穿黑衣”的“黑”字，改作“花”字为好。

[[王路|王路]]收到信后，格外兴奋，感到千里之外遇到了知音，当夜复信表示认可。《小燕子》经过王云阶删除、润色、谱曲，成了[[护士日记|《护士日记》]]的主题曲。后来，[[护士日记|《护士日记》]]放映，演员王丹凤在片中深情一唱。就这样，从未谋面的“三王”，让这只“小燕子”飞入千家万户。<ref>{{Cite web|title=经典儿歌《小燕子》家喻户晓 其作者为黄石教师_大楚网_腾讯网|url=https://web.archive.org/web/20190308135618/http://hb.qq.com/a/20170323/030700.htm|work=web.archive.org|date=2019-03-08|accessdate=2019-03-08}}</ref><ref>{{Cite web|title=金曲《小燕子》家喻户晓 词作者半辈子在黄石_大楚网_腾讯网|url=https://web.archive.org/web/20190308140032/http://hb.qq.com/a/20170323/031000.htm|work=web.archive.org|date=2019-03-08|accessdate=2019-03-08}}</ref>

== 背景 ==
[[护士日记|《护士日记》]]时代背景是[[第一个五年计划_(中国)|一五]]计划的实施，[[第一个五年计划_(中国)|一五]]时期大力发展工业。《小燕子》的第二段表达了高昌平（[[汤化达|汤化达]]饰）和简素华（[[王丹凤|王丹凤]]饰）对未来的美好憧憬和信心以及无私奉献之情。

== 歌词 ==
虽然歌词本身有两段，但最为传唱的只有前半部分。歌曲中重复三遍歌词。
