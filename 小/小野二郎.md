'''小野二郎'''（{{lang-ja|小野 二郎}}，{{bd|1925年|10月27日|||Ono Jirō}}）是出生於[[日本|日本]][[靜岡縣|靜岡縣]][[天龍市|天龍市]](現[[濱松市|濱松市]][[天龍區|天龍區]])的[[壽司|壽司]]師傅，他所經營的壽司店「[[數寄屋橋次郎|數寄屋橋次郎]]」在2007獲得《[[米其林指南|米其林指南]]》的三顆星評價後便連續10年拿下該評比；<ref>{{cite web |url=http://www.sankei.com/life/news/161129/lif1611290039-n1.html |script-title=ja:ラーメン店「鳴龍」が一つ星、「すきやばし次郎」など４店が１０年連続三つ星　ミシュランガイド東京２０１７発表 |work=[[產經新聞|產經新聞]] |language=ja |date=2016年11月29日 |accessdate=2016年12月10日}}</ref>且是在此之前列在《米其林指南》中最高年齡的三顆星評價主廚。<ref>{{cite web |url=http://www.sankei.com/economy/news/161104/prl1611040110-n1.html |script-title=ja:訪日旅行者へ日本最高峰の体験をプレゼント！Voyaginで「すきやばし次郎本店」の鮨を食べよう！ |work=[[產經新聞|產經新聞]] |language=ja |date=2016年11月4日 |accessdate=2016年12月10日}}</ref>

== 經歷 ==
在7歲期間左右小野二郎便已在家鄉附近的料理店工作，<ref name="JiroMovie">{{cite web |url=http://jiro-movie.com/cast |script-title=ja:キャスト・スタッフ |work=Transformer.Inc |language=ja |accessdate=2016年12月10日 |archive-url=https://web.archive.org/web/20160126193340/http://jiro-movie.com/cast |archive-date=2016年1月26日 |dead-url=yes }}</ref>[[二次大戰|二次大戰]]爆發時於1941年被徵召到[[兵工廠|兵工廠]]；後續則在1945年入伍。<ref name="Flap">{{cite book |title=《-{壽司之神全技法：小野二郎的壽司聖經}-》 |publisher=城邦文化 |author=小野二郎 |location=台灣 |isbn=978-986-568087-9  |date=2015年2月1日 |page=折口說明}}</ref>

二戰結束後小野二郎前往到[[濱松市|濱松市]]一帶的餐廳工作，在25歲時則開始立志成為壽司師傅。<ref name="Page81">{{cite book |title=《-{壽司之神全技法：小野二郎的壽司聖經}-》 |publisher=城邦文化 |author=小野二郎 |location=台灣 |isbn=978-986-568087-9  |date=2015年2月1日 |chapter=〈第二章 壽司的技藝及祕訣〉|page=81頁}}</ref>之後前往[[京橋|京橋]]一帶的壽司店「與志乃」工作學習，師傅是吉野末吉，9年後則在位於銀座的與志乃分店開始擔任店主。<ref name="Flap"/>

1965年小野二郎在[[銀座|銀座]]開設自行成立的壽司店[[數寄屋橋次郎|數寄屋橋次郎]]，<ref name="Sankei">{{cite web |url=http://www.sankei.com/life/news/141102/lif1411020013-n1.html |script-title=ja:好々爺、職人技は客を魅了　黄綬褒章　「すきやばし次郎」元社長　すし職人・小野二郎さん（８９） |work=[[產經新聞|產經新聞]] |language=ja |date=2014年11月2日 |accessdate=2016年12月10日}}</ref>在從事壽司領域多年後於2005年得到[[厚生勞動省|厚生勞動省]]表揚為飲食方面的職人之一，<ref>{{cite web |url=http://www.mhlw.go.jp/houdou/2005/11/h1108-1a.html#16 |script-title=ja:第１６部門 飲食物調理及び接客サービスの職業 |work=[[厚生勞動省|厚生勞動省]] |language=ja |accessdate=2016年12月10日}}</ref>所成立的數寄屋橋次郎也在2007年首次得到《米其林指南》東京版的三顆星評價餐館，<ref name="Sankei"/>並於2011年得到日本海外[[西班牙|西班牙]][[瓦倫西亞|瓦倫西亞]]政府認可的能夠優秀調理米類之料理人，<ref name="JiroMovie"/>以及2014年秋季日本政府所頒贈的黃綬褒章。<ref>{{cite web |url=http://www.2nn.jp/newsplus/1414886735/ |script-title=ja:【秋の褒章】「すきやばし次郎」の小野二郎さんが黄綬褒章 |work=2NN |language=ja |date=2014年11月2日 |accessdate=2016年12月10日}}</ref>

== 風格 ==
因本身是[[左撇子|左撇子]]的關係在初期學習捏壽司時為此碰到困擾，之後靠自創的改良捏法而克服。<ref name="Page81"/>而為了保護捏壽司的雙手，在出門時有穿戴手套的習慣。<ref>{{cite web |url=http://japanese.joins.com/article/181/221181.html |script-title=ja:日本に劣らない韓国高級すし、その秘訣は？（１） |work=[[中央日報_(韓國)|中央日報]] |language=ja |date=2016年9月28日 |accessdate=2016年12月10日}}</ref>另外，也不跟男性握手是因為男性的手通常較女性粗糙，而且握手時會較為用力，會傷害小野二郎先生手部的高敏感度。

所捏出來的壽司構造是外觀米粒緊粘著，但內部是米粒之間帶有空隙而又包住空氣，小野二郎對此自評是一些客人認為他做的壽司有著能在口中散開口感的因素。<ref>{{cite book |title=《-{壽司之神全技法：小野二郎的壽司聖經}-》 |publisher=城邦文化 |author=小野二郎 |location=台灣 |isbn=978-986-568087-9  |date=2015年2月1日 |chapter=〈第五章 匠人的心得〉|page=147頁}}</ref>

== 家庭 ==
小野二郎的兩名兒子皆同樣從事壽司師傅職業，長子小野禎一現今頂替父親擔任數寄屋橋次郎本店的店主，<ref>{{cite web |url=http://style.nikkei.com/article/DGXZZO10066880Z21C16A1000000?n_cid=TPRN0016 |script-title=ja:「三十星」が４軒　ミシュランガイド東京、１０周年で |work=NIKKEI STYLE |author=渡邊智哉 |language=ja |date=2016年11月29日 |accessdate=2016年12月10日}}</ref>次子小野隆士則在[[六本木|六本木]]一帶開設數寄屋橋次郎的分店；且曾獲得《米其林指南》兩顆星的評價。<ref>{{cite web |url=http://www.bs-tbs.co.jp/green/bn150.html |script-title=ja:寿司とゴルフに大事なこと　それは"握り"と"リズム" |work=[[TBS電視台|TBS電視台]] |language=ja |date=2013年3月2日 |accessdate=2016年12月10日}}</ref>

== 反應 ==
法國廚師[[喬爾·侯布匈|喬爾·侯布匈]]是小野二郎店鋪的常客，他個人原先討厭品嚐[[章魚|章魚]]，某次吃到小野二郎花費一小時處理後捏製的章魚壽司覺得有如[[龍蝦|龍蝦]]般地口感；之後便會主動點名這道菜餚。<ref>{{cite book |title=《-{壽司之神全技法：小野二郎的壽司聖經}-》 |publisher=城邦文化 |author=小野二郎 |location=台灣 |isbn=978-986-568087-9  |date=2015年2月1日 |chapter=〈第二章 壽司的技藝及祕訣〉|page=89頁}}</ref>

美國總統[[歐巴馬|歐巴馬]]在2014年4月期間訪問日本時，曾與日本首相[[安倍晉三|安倍晉三]]一同前往小野二郎的店鋪用餐。歐巴馬在餐後表示著雖然過去他在[[夏威夷|夏威夷]]生活時期吃過不少日式壽司，但小野二郎做的壽司是他所吃過最棒的。<ref>{{cite web |url=https://www.nytimes.com/2014/04/24/world/asia/obamas-first-order-of-business-in-tokyo-sushi-from-the-master.html?_r=0 |title=Obama’s First Order of Business in Tokyo: Sushi From the Master |work=[[紐約時報|紐約時報]] |author=David S. Joachim |language=en |date=2014年4月23日 |accessdate=2016年12月10日}}</ref>
