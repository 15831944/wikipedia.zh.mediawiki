《'''小惠會使用魔法嗎？'''》（[[日語|日語]]：{{lang|ja|'''メグたんって魔法つかえるの?'''}}）是[[日本電視台|日本電視台]]由2012年7月7日開始在[[星期六|星期六]]25:20 - 25:35播放的[[日本電視劇|電視劇]]<ref>[http://hochi.yomiuri.co.jp/entertainment/news/20120702-OHT1T00038.htm こじはる、パンチラＯＫ！毎回“魔法”を披露…日テレ系新ドラマ「メグたんって魔法つかえるの？」] {{webarchive|url=https://web.archive.org/web/20120708010635/http://hochi.yomiuri.co.jp/entertainment/news/20120702-OHT1T00038.htm |date=2012-07-08 }}</ref>。

== 概要 ==
[[小嶋陽菜|小嶋陽菜]]（[[AKB48|AKB48]]）相隔約4年後第3部連續劇主演作品。腳本和監督由[[福田雄一|福田雄一]]擔任，繼在2012年1月[[季度|季]]・4月季播放《[[繆斯之鏡|繆斯之鏡]]》後被起用。繼《繆斯之鏡》的[[指原莉乃|指原莉乃]]（[[HKT48|HKT48]]）後第2作連續有AKB48成員主演。

== 概要 ==
來人間修行的半調子魔法少女小惠是魔法学校的劣等生。國王命令小惠到「ろくでも荘」[[公寓|公寓]]作為管理員工作。

== 角色 ==
官方網站的飾演名稱。
