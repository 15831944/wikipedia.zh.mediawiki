'''小兴安岭'''（{{lang|en|Lesser Khingan Mountains}}），為[[中華人民共和國|中国]][[黑龙江省|黑龙江省]]北部的一条[[山脉|山脉]]，西北-东南走向，经过[[黑河|黑河]]、[[孙吴|孙吴]]、[[伊春|伊春]]等县市。山脉长约400公里，山地低矮，多以[[丘陵|丘陵]]为主，一般海拔600-1000米左右，最高峰[[平顶山_(小兴安岭)|平顶山]][[海拔|海拔]]1,429米。

== 地形地势 ==

小兴安岭西北段在黑河与[[大兴安岭|大兴安岭]]北段的支脉[[伊勒呼里山|伊勒呼里山]]相连；南部伸入[[松嫩平原|松嫩平原]]与[[三江平原|三江平原]]之间，是[[松花江|松花江]]左岸流域与[[同江市|同江市]]以上[[黑龙江|黑龙江]]流域的[[分水岭|分水岭]]。北段为隆起的[[台地|台地]]；南段为古老褶皱断块山，由[[花岗岩|花岗岩]]、[[变质岩|变质岩]]构成。总面积13万平方公里，其中低山约占37%、丘陵约占53%、浅丘台地约占10%。海拔500~800米。山势和缓，北低南高，[[伊春|伊春]]附近的大管山海拔1203米。

== 地貌 ==
[[File:1963-02_1963年_小兴安岭神树镇3.jpg|thumb]]
地貌相比差异显著，南坡山势浑圆平缓，水系绵长；北坡陡峭，成阶梯状，水系短促。北部多[[丘陵|丘陵]]台地，地表以砂砾岩、[[玄武岩|玄武岩]]为主，河谷多宽谷；南部低山丘陵区，多出露海西期花岗岩，河谷多形成“V”形谷。

== 水文 ==
小兴安岭山区河流，主要有属于黑龙江水系的[[逊别拉河|逊别拉河]]、[[沾河|沾河]]、[[乌云河|乌云河]]等和属于松花江水系的[[呼兰河|呼兰河]]、[[汤旺河|汤旺河]]等，构成黑龙江水系与松花江水系的分水岭。

== 气候 ==
小兴安岭的范围在[[北纬|北纬]]46°28′至49°21′和[[东经|东经]]127°42′至130°14′之间，气候属[[温带大陆性气候|温带大陆性气候]]，冬季严寒漫长，夏季温暖但较短。[[林业|林业]]资源丰富，為中国主要林区之一，另外还有丰富的动物资源和矿产资源。
