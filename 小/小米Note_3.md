'''小米Note 3'''是[[小米科技|小米科技]]於2017年9月11日發表的Android[[智能手機|智能手機]]，搭載基於[[Android|Android]] 7.1的[[MIUI|MIUI]]系統、5.5吋螢幕、[[高通|高通]][[驍龍|驍龍]]660處理器、6GB記憶體、64GB / 128GB儲存空間。中國大陸於9月12日開賣。<ref>[https://udn.com/news/story/7098/2695102 小米Note 3搭載雙鏡頭、臉部識別亮相 等同小米6 Plus產品定位]</ref>

小米Note 3獲得了相機評測機構[[DxOMark|DxOMark]]綜合得分90分的評價，與[[HTC_U11|HTC U11]]和[[Google_Pixel|Google Pixel]]並列。DxOMark給出了「中階的價格，高階的性能」的評論。在相機拍照部分，獲得94分，超越了[[iPhone_8|iPhone 8]]的93分；在攝影部分，則僅獲得82分。<ref>[https://www.dxomark.com/cn/xiaomi-mi-note-3-exceptional-images-with-all-the-bells-and-whistles/ 小米Note 3: 画质出色，功能齐全]</ref><ref>[http://3c.ltn.com.tw/news/32272 小米 Note 3 拍照竟贏過 iPhone 8？評測機構這樣說]</ref><ref>[http://www.eprice.com.tw/mobile/talk/4568/5043933/1/ 小米 Note 3 DxoMark 總分 90 分，靜態攝影比下 iPhone 8]</ref><ref>[https://qooah.com/2017/12/09/xiaomi-note-3-dxo-mark-photo-best-than-iphone-8/ 小米Note 3 DxOmark 拍照評分超 iPhone 8？？]</ref><ref>[https://www.saydigi.com/2017/12/xiaomi-note-3-dxomark-90.html 成績出爐 小米Note 3 在DxOMark評比中拿下90分]</ref>
