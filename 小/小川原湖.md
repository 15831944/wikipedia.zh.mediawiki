'''小川原湖'''（{{jpn|j=小川原湖|hg=おがわらこ|rm=Ogawara-ko}}）是[[日本|日本]][[青森縣|青森縣]][[上北郡|上北郡]][[東北町|東北町]]的湖泊。屬於[[半鹹水湖|半鹹水湖]]。是日本第11大湖泊<ref>{{Cite web|url=http://www.gsi.go.jp/KOKUJYOHO/MENCHO/201410/large_kosyo.pdf|title=平成26年全国都道府県市区町村別面積調 湖沼面積（20傑）|accessdate=2015-03-08|author=国土地理院|date=2015-03-06|format=PDF}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。地址訂為青森縣上北郡東北町大字大浦字小川原湖191番地。

== 地理 ==
小川原湖位於青森縣東部，是該縣面積最大的湖泊。最深處達25公尺，湖岸至離岸200公尺處為深2公尺的較淺部分。海面水位高於湖面水位時期，海水則逆流高瀨川注入湖中。

小川原湖周遭有包括尾駮沼、鷹架沼、市柳沼、田面木沼、內沼、姊沼的湖沼群分布，總稱為小川原湖湖沼群。
* 流入河川：七戶川、花切川、砂土路川、姊沼川 
* 流出河川：高瀨川

== 自然環境 ==
小川原湖湖沼群周遭保有豐富的自然環境，構成多種多樣的生物體系，被[[环境省|環境省]]定為[[日本500重要濕地|日本500重要濕地]]、[[日本音風景百選|日本音風景百選]]。

部分湖沼群有[[毬藻|毬藻]]生存其中、近年來確認小川原湖半鹹水性球藻的近親{{lang|la|''Wittrockiella salina''}}存在，此為日本首次發現。<ref>[http://www.mlit.go.jp/river/toukei_chousa/kasen/jiten/nihon_kawa/82015/82015-1_p3.html 高瀬川の自然環境] {{webarchive|url=https://web.archive.org/web/20131029192634/http://www.mlit.go.jp/river/toukei_chousa/kasen/jiten/nihon_kawa/82015/82015-1_p3.html |date=2013-10-29 }} - 国土交通省</ref><ref name="milt_pdf">[http://www.mlit.go.jp/river/basic_info/jigyo_keikaku/gaiyou/seibi/pdf/takase-4-2.pdf 高瀬川の流域及び自然環境] - 国土交通省</ref>

過去有[[遠東哲羅魚|遠東哲羅魚]]棲息於小川原湖，曾有捕獲約1噸的紀錄，1943年突然滅絕<ref>[http://www.applenet.jp/~naisuimen/fish/file/itou.html イトウ]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }} - 青森県産業技術センター</ref>。鳥類方面，則為[[大天鵝|大天鵝]]、[[小天鵝|小天鵝]]的棲息地。　

== 使用 ==
因水產資源豐富，稱為'''寶沼'''<ref>[http://www.mlit.go.jp/common/000188464.pdf 宝湖活性化協議会(PDF)] {{webarchive|url=https://web.archive.org/web/20130124155849/http://www.mlit.go.jp/common/000188464.pdf |date=2013-01-24 }}</ref>。主要的水產資源有[[西太公魚|西太公魚]]、[[銀魚|銀魚]]、[[鰕虎亞目|鰕虎]]、[[花蜆科|蜆]]([[大和蜆|大和蜆]])、[[鰻魚|鰻魚]]。小川原湖漁業協同組合負責管理及捕撈。

因為湖淺，夏季戲水遊客眾多。冬季湖面結冰，可以穴釣西太公魚。七戶川舊道花切川，是釣[[高身鯽|高身鯽]]的知名地點。

== 淡水化工程 ==
1970年代作為陸奧小川源開發計畫的一環，計畫將小川原湖淡水化，進行供水（小川原湖廣域水道用水供給事業）<ref name="mhlw_pdf">[http://www.mhlw.go.jp/topics/bukyoku/kenkou/suido/tantousya/2012/dl/02-401.pdf 広域的水道計画の策定及び改定状況について] - 厚生労働省</ref>。然而之後得出實際需水量被高估的結論，[[1996年|1996年]]12月，公告撤回淡水化<ref>[http://www.mlit.go.jp/river/shishin_guideline/kankyo/kankyou/kosyo/tec/pdf/6.3.pdf 小川原湖の水理・水質特性] - 国土交通省（90ページなど）</ref><ref name="mhlw_pdf" />。

==污染事件==
2018年2月20日，[[美國空軍|美國空軍]]一架駐防在[[三澤基地|三澤基地]]的[[F-16戰鬥機|F-16戰鬥機]]，起飛後不久因發動機起火而緊急拋棄兩具[[外燃料箱|外燃料箱]]到小川原湖，造成該湖受外漏燃料的污染。由於正值蜆貝和西太公魚等水產的捕撈旺季，在燃料箱未打撈且燃料未清除前，當地漁民無法進行捕撈作業，造成經濟損失<ref>[https://udn.com/news/story/6809/2994991 美戰機故障丟2油箱毀湖 日本漁民想哭]</ref>。
