'''小西湖'''（{{lang-en|Siu Sai Wu}}）位於[[香港|香港]][[九龍|九龍]]的[[九龍塘|九龍塘]]之[[筆架山|筆架山]]的筆架山道、[[歌和老街|歌和老街]]及[[龍翔道|龍翔道]]之間。

小西湖是[[水務署|水務署]]所建築的人造[[蓄水池|蓄水池]]，水深約20英尺，它的儲水主要供[[九龍塘|九龍塘區]]淋花及沖廁之用，並開設了一條地下水道連通[[尖沙咀站_(九廣鐵路英段)|尖沙嘴火車站]]。它共有高低等兩個[[湖泊|湖泊]]，驟似[[中國|中國]][[浙江省|浙江省]][[杭州市|杭州市]]的[[杭州西湖|西湖]]景致，因而得名。

[[南華體育會|南華體育會]]遊藝部在1937年11月14日（星期日）舉辦「小西湖[[旅行團|旅行團]]」，人數高達50人，當日上午9時45分在[[香港島|香港島]][[統一碼頭|統一碼頭]]出發，每人收費[[港幣|港幣]]2毫，提供野餐午膳，加插民眾識詠[[口琴|口琴]][[表演|表演]]及[[遊戲|遊戲]]，下午3時半回程<ref>1937年11月10日（星期三），香港：《[[華字日報|華字日報]]》，第壹張，第肆頁。</ref>。

[[九龍|九龍]]的[[童軍|童軍]]在1939年8月25日至8月27日，連續3天在此處舉行「九龍童子軍第一次大露營」，共有18團參加<ref>1939年8月21日（星期一），香港：《[[大公報|大公報]]》，第二張，第六版。</ref>。

1960年代前，此地是[[香港|香港]]的[[中學|中學]]及[[小學|小學]]舉辦[[學生|學生]]集體旅行的熱門選擇之一。在那時的[[暑假|暑假]]，由於天氣炎熱，吸引許多貧家小童到此處嬉水，部分冒失的泳客常在此地喪命，直至1961年已累計超過百人溺斃，[[香港殖民地時期#香港政府|香港政府]]在1961年7月在四周築起高約8英尺的鐵絲網圍牆，杜絕遊客進入參觀。

小西湖亦指1970年代前期，上述[[蓄水池|蓄水池]]附近的[[貧民窟|貧民窟]]，當年被稱為「小西湖[[寮屋|寮屋]]」（{{lang|en|Siu Sai Wu Huts}}）。目前，於[[九龍|九龍]][[觀塘|觀塘]][[翠屏道|翠屏道]]13號的[[香港歷史檔案大樓|香港歷史檔案大樓]]裡，仍保存有70年代，[[香港殖民地時期#香港政府|香港政府]]把那些寮屋居民遷徒到觀塘的[[秀茂坪|秀茂坪]]徙置區（現在的[[秀茂坪邨|秀茂坪邨]]）的書信函件，供[[香港市民|香港市民]]借閱參考。

1972年，該蓄水池停用，目前已經不屬於水務署所管轄，該湖從此空置至今。

== 地理環境 ==
小西湖位於[[港鐵|港鐵]][[東鐵綫|東鐵綫]]（前稱[[九廣東鐵|九廣東鐵]]）[[筆架山隧道|筆架山隧道]]段的西部，具體位置為筆架山道、[[哥和老街|哥和老街]]及[[龍翔道|龍翔道]]之間的[[峽谷|峽谷]]裡。目前大部分地區被香港[[水務署|水務署]]所架設的[[鐵絲網|鐵絲網]][[圍欄|圍欄]]阻隔，四周遍布茂密[[叢林|叢林]]包圍，一般市民只能在歌和老街的下遊入口遠觀滿溢的小西湖濺出[[堤壩|堤壩]]造成小[[瀑布|瀑布]]。

== 1940及50年代的香港書刊描述的周圍環境 ==
* 1940年5月20日，香港《[[大公報|大公報]]》，第七版，作者：新亞。
**「第一湖之右邊是[[平原|平原]]；第二湖的湖後為平坦的小丘，可供結下[[帳幕|帳幕]]。」
* 1941年2月24日，香港《[[國民日報|國民日報]]》
之「木刻與詩」刊登由[[廣東省|廣東省]]的詩人[[梁儼然|梁儼然]]（梁松生）先生的「小西湖之旅」。
**「初春的小湖，單調而幽鬱，只有朦朧斷續的水聲，墜落荒涼的石隙旁。枯寂的斜橋，跨腳在死水的池上，柔風浴浸著松陰的淡影，心靈像殘葉而飄蕩。年青的詩人，放下一刻的安息，跑上海外的高原，把種子撒下山岡，使烽火裡的詩群底旗幟，展開了新生的光芒。」根據該首[[新詩|新詩]]的內容，小西湖有斷續的水流，有斜[[橋|橋]]橫跨在兩個[[蓄水池|蓄水池]]間，周邊有茂密的[[松樹|松樹]][[叢林|叢林]]。
*《最新香港指南》第104頁，湯建勛先生編輯，香港民華出版社在1950年2月初版。
**「在[[山谷|山谷]]之中，有著[[青松|青松]]綠水，[[小橋|小橋]]幽徑。」
*《香港、九龍、新界旅行手冊》第28頁，[[吳灞陵|吳灞陵]]先生編撰，香港：《[[華僑日報|華僑日報]]》在1950年8月初版。
**「小西湖有一泓綠水，萬樹古松、小橋及幽谷。遊客可以[[游泳|游泳]]、野餐、遊戲及[[唱歌|唱歌]]。」

== 罪案溫床 ==
1939年7月29日（星期六）下午3時，天雨綿綿，24歲[[教師|教師]]黃劍池與21歲[[女朋友|女朋友]]李銀到小西湖[[拍拖|拍拖]]，被3名持短[[劍|劍]][[劫匪|劫匪]]搶去港幣7元5毫、價值港幣10元[[金|金]][[戒指|戒指]]<ref>1939年7月30日（星期日），香港：《華字日報》，第貳張，第叁頁。</ref>。

1940年1月19日（星期五）下午2時45分，23歲女子陳蓮卿與朋友在小西湖，被3名持短劍劫匪搶去港幣13元、價值港幣10元金戒指及港幣13元[[手表|手表]]<ref>1940年1月20日（星期六），香港：《華字日報》，第貳張，第肆頁。</ref>。

1953年10月11日（星期日）下午，「美新貿易公司」徐崇偉[[經理|經理]]與[[妻子|妻子]]任蘇、[[兒子|兒子]]徐錦禮及[[女兒|女兒]]徐錦英在小西湖被一群6至7名劫匪持[[刀|刀]]搶去港幣120元、2[[美元|美元]]、4隻戒指及3隻手表。[[香港警察|香港警察]]隨後[[拘捕|拘捕]][[籍貫|籍貫]][[廣東省|廣東省]][[東莞縣|東莞縣]]的22歲[[大陸|大陸]]劫匪黃林，在同年10月21日（星期三）押解至九龍裁判署提堂<ref>1953年10月22日（星期四），香港：《[[華僑日報|華僑日報]]》，第三張，第一頁：港聞三。</ref>。同年11月23日（星期一）在九龍地方法院判決另外兩位劫匪[[罪名成立|罪名成立]]，17歲香港劫匪羅毓旺被判4年苦工監及笞10籐；18歲籍貫廣東省清遠縣大陸劫匪潘漢全被判18個月苦工監<ref>1953年11月24日（星期二），香港：《華僑日報》，第二張，第三頁：港聞三。</ref>。

30歲的許堅在1961年6月5日（星期一）晚上6時在小西湖[[游泳|游泳]]時，被一位身穿筆挺[[西裝|西裝]]的匪徒搶去放在湖邊的[[衣服|衣服]]，包括港幣12元及一隻手表<ref>1961年6月6日（星期二），香港：《[[工商日報|工商日報]]》，第七頁。</ref>。

住在九龍[[牛頭角|牛頭角]]的25歲梁姓匪徒在1963年10月29日（星期二）中午12時在小西湖，以三角銼威脅行劫一群學生，遇反抗，他刺傷一位16歲[[陳|陳]]姓[[學生|學生]]的[[肩膀|肩膀]]<ref>1963年10月30日（星期三），香港：《大公報》，第一張；第四版。</ref>。

九龍地方法院在1963年11月25日（星期一）下午判決屢次在小西湖行劫的28歲匪徒梁偉培入獄3年，他曾經在同年10月27日（星期日）在小西湖行劫兩男四女共6隻手表；一對小西湖[[情侶|情侶]]兩隻手表及兩隻戒指；同年10月28日（星期一）中午持刀行劫11位到小西湖觀光的學生，劫去港幣8元2毫及一隻手表<ref>1963年11月26日（星期二），香港：《工商日報》，第六頁。</ref>。

1965年9月11日（星期六）下午5時，5位由7歲至11歲的學生在小西湖戲水捉魚，被10多名17歲的[[飛仔|飛仔]]毆打，並搶去[[書包|書包]]<ref>1965年9月12日（星期日），香港：《華僑日報》，第二張，第一頁：港聞一。</ref>。

1968年8月6日（星期二）上午在小西湖濾水池引水道，發現33歲[[婦人|婦人]]李嬌娛被鐵線捆綁的[[屍體|屍體]]。她的38歲籍貫廣東省[[台山縣|台山縣]][[丈夫|丈夫]]梅勝才被控[[謀殺|謀殺]]，在1968年8月14日（星期三）上午狎解至[[北九龍裁判署|北九龍裁判署]]加西亞法庭提訊；1968年8月17日（星期六）再度提堂；1968年10月21日（星期一）在[[高等法院|高等法院]]提堂；<ref>1968年8月15日（星期四），香港：《華僑日報》，第一張，第四頁：香港新聞。</ref><ref>1968年8月18日（星期日），香港：《華僑日報》，第一張，第四頁：香港新聞。</ref><ref>1968年10月22日（星期二），香港：《華僑日報》，第一張，第二頁：香港新聞。</ref>。1968年10月25日（星期五）被判[[誤殺|誤殺罪]]，入獄6年<ref>1968年10月26日（星期六），香港：《工商日報]]》，第七頁。</ref><ref>1968年10月26日（星期六），香港：《大公報》，第二張；第五版。</ref>。

一位男學生與11位女同學在1969年1月8日（星期二）在小西湖野餐時，被6名18歲至22歲的匪徒以破玻璃瓶威脅，搶劫[[價值|價值]]港幣170元的現金、兩部[[相機|相機]]，並刮傷男同學的[[臉孔|臉孔]]<ref>1969年1月8日（星期三），香港：《大公報》，第一張；第四版。</ref>。

* 1965年7月20日（星期二）晚上7時，[[旺角勞工子弟學校|旺角勞工子弟學校]]二年級學生，11歲的石洪標在小西湖遇溺，消防局在晚上11時打撈，直至晚上11時半才把他的屍體撈出<ref>1965年7月21日（星期三），香港：《華僑日報》，第二張，第一頁：港聞一。</ref><ref>1965年7月22日（星期四），香港：《華僑日報》，第一張，第四頁：本港新聞。</ref>。

* 小西湖在1966年4月25日（星期一）上午10時，[[水務署|水務署]][[職員|職員]]開水閘時<ref>1966年4月26日（星期二），香港：《工商日報》，第六頁。</ref>，發現因游泳遇溺[[失蹤|失蹤]]的13歲九龍[[橫頭磡|橫頭磡]]的林才源及馮國源的全裸屍體<ref>1966年4月26日（星期二），香港：《華僑日報》，第二張，第三頁：港聞三。</ref><ref>1966年4月27日（星期三），香港：《華僑日報》，第三張，第二頁：港聞五。</ref>。
