'''小蓝单车'''是由天津鹿鼎科技有限公司研发并运营的无桩[[共享自行车|共享自行车]]品牌，其推出的自行车在众多共享自行车品牌中被用户誉为「拥有骑行最佳体验」<ref name="ifanrbeijing">{{Cite web |url=https://www.ifanr.com/823808 |title=摩拜说：没有 GPS 定位的共享单车都该销毁。这不就是说 ofo 吗？ |author=方嘉文 |date=2017-04-22 |agency=爱范儿}}</ref>。小蓝单车于2016年11月12日在[[中华人民共和国|中华人民共和国]][[深圳市|深圳市]]推出并启用<ref name="sinazhuanfang">{{Cite web |url=http://tech.sina.com.cn/i/2017-03-22/doc-ifycnpiu9486641.shtml |title=专访小蓝单车CEO李刚：未来将吃喝玩乐引入单车平台 |author= |date=2017-03-22 |agency=[[新浪|新浪]]科技}}</ref>并成为中国共享自行车企业中第一个在海外落地的企业<ref name="sinahaiwai">{{Cite web |url=http://tech.sina.com.cn/i/2017-03-22/doc-ifycnikk1488285.shtml |title=ofo/摩拜等入局：中国共享单车争相登陆新加坡 |date=2017-03-22 |agency=澎湃新闻}}</ref>。2017年11月15日，小蓝单车被曝倒闭解散，11月16日小蓝单车负责人李刚发布声明，宣布公司已与拜客出行达成了战略合作，将由拜客出行全权代理小蓝单车未来的运营。2018年1月9日，[[滴滴出行|滴滴出行]]和小蓝单车联合宣布，小蓝单车将由滴滴方面托管。

== 使用方法 ==
用户可通过装有小蓝单车应用程序、[[支付宝|支付宝]]<ref name="jinrongjie">{{Cite web |url=http://finance.jrj.com.cn/tech/2017/05/03211422422680.shtml |title=ofo、永安行、小蓝等百万单车接入支付宝保险，最高可保50万 |author= |date=2017-05-03 |agency=金融界}}</ref>和[[微信|微信]]的智能手机扫描停放于街道两旁小蓝单车上的[[二维码|二维码]]解锁自行车，骑行结束后将自行车停放于道路两旁不影响交通的地方将自行车上锁即可。

== 车型 ==
[[File:Bluegogo_Pro_in_Beijing.jpg|thumb]]
小蓝单车在中国推出了三种车型，分别为bluegogo、bluegogo Pro和bluegogo Pro 2，均有智能电子车锁、车铃并由车筐上的[[太阳能|太阳能]]电池板为智能锁供电。但bluegogo Pro和Pro 2在基本款的基础上添加了三级变速装置，Pro 2更是使用了碳纤维车架，大幅降低车辆的重量。<ref name="firefoxchina"/>小蓝单车称其每款车型均通过了[[國際標準化組織|國際標準化組織]]质量认证。<ref name="ifanrbeijing"/>

在美国运营的小蓝单车是在中国版bluegogo的基础上添加了车灯。<ref name="bluegogous">{{Cite web |url=https://bluegogo.com/us/ |title=bluegogo |deadurl=yes |archiveurl=https://web.archive.org/web/20170405073702/https://www.bluegogo.com/us/ |archivedate=2017年4月5日 |df= }}</ref>

== 争议 ==
[[File:Bluegogo-bike.jpg|缩略图]]
===在舊金山遭禁===
小蓝单车在进入旧金山运营后遭到了市政府的强烈反对，有官员称小蓝单车为「流氓企业」并要严厉惩罚。<ref name="guanchazhe">{{Cite web |url=http://tech.sina.com.cn/roll/2017-04-11/doc-ifyeayzu7487899.shtml |title=吴蒙蒙：共享单车打起“退堂鼓”？ |author=吴蒙蒙 |date=2017-04-11 |agency=观察者网}}</ref>3月15日，旧金山市政府颁布法令禁止小蓝单车或类似无桩共享单车在未获得许可的情况下运营。<ref name="sfexaminer">{{Cite web |url=http://www.sfexaminer.com/sf-passes-new-laws-penalize-bike-share-companies-like-bluegogo/ |title=SF passes new laws to penalize bike-share companies like Bluegogo |author= Joe Fitzgerald Rodriguez |date=2017-03-05 |agency=The San Francisco Examiner}}</ref>但有居住在旧金山的福布斯网记者反对市政府的做法，认为旧金山或美国是一个过度监管、充满繁文缛节的地方。<ref name="qiaobaowang">{{Cite web |url=http://news.uschinapress.com/2017/0119/1092449.shtml |title=从旧金山反对中国共享单车 看东西方文化差异 |author=李怡 |date=2017-01-19 |agency=侨报网}}</ref><ref name="forbes">{{Cite web |url=https://www.forbes.com/sites/bensin/2017/01/18/what-san-franciscos-backlash-against-chinese-start-up-says-about-east-west-cultural-differences/#517dd09b1bf4 |title=What San Francisco's Backlash Against A Chinese Start-Up Says About East-West Cultural Differences |author=Ben Sin |date=2017-01-18 |agency=Forbes}}</ref>

=== 六四坦克宣傳事件 ===
2017年小藍單車北京分公司與[[掌遊天下|掌遊天下]]公司推出聯合活動，宣傳掌遊天下公司的「王牌大作戰」遊戲，活動內容為從6月3日至4日，所有帶有[[坦克|坦克]]圖標的小藍單車騎行免費，並且騎到帶有坦克圖標的小藍單車兩次及以上便有機會獲得奖品<ref>{{Cite web|url=http://www.sohu.com/a/145581401_670685 |archive-url=https://web.archive.org/web/20170602230234/http://www.sohu.com/a/145581401_670685 |title=一骑赢千元 这比王者荣耀还刺激 |author=小蓝单车订阅号 |date=2017-06-02 |publisher=搜狐 |archivedate=2017-06-02 |deadurl=yes }}</ref><ref>{{Cite web|url=http://irunhuayou.com/keji/yejie/2164736.html|title=一骑赢千元 这比王者荣耀还刺激|author=小蓝单车订阅号|date=2017-06-02|publisher=潤匯網}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。因正處於六四敏感時期，該活動因容易联想到[[六四事件|六四事件]]而引發網友熱議，最終短短幾小時後活動便被下架，而中國網絡上所有與此有關的內容均被刪除。<ref>{{Cite web |url=http://tw.on.cc/cn/bkn/cnt/news/20170603/bkncn-20170603140022027-0603_05011_001.html |title=京共享單車六四推尋找坦克活動 遭緊急下架 |author= |date=2017-06-03 |publisher=東網}}</ref><ref>{{cite web|url=https://cnews.com.tw/新創共享公司小藍單車不識「六四」-坦克停天安門/|title=新創共享公司小藍單車不識「六四」 坦克停天安門太狂|author=黃有容|date=2017-06-03|publisher=匯流新聞網}}</ref>但在網上廣為流傳的一張「『坦克』沿[[長安街|長安街]]排列」的圖片則被懷疑是電腦合成，因為長安街沿線禁止擺放單車<ref>{{cite web|url=http://china.dwnews.com/news/2017-06-03/59818249.html|title=六四踩地雷？共享单车坦克活动遭下架|author=|date=2017-06-03|publisher=多維新聞}}</ref>。事發後，警方進入小藍單車內部調查<ref>{{cite web|url=http://www.pingwest.com/bluegogo-die/|title=小蓝单车的死亡日记|author=|date=2017-11-16|publisher=品玩|deadurl=yes|archiveurl=https://web.archive.org/web/20171116092303/http://www.pingwest.com/bluegogo-die/|archivedate=2017-11-16}}</ref>。也因為這次事件，導致小藍單車後續融資受阻。<ref>{{Cite web|url=http://news.dwnews.com/china/news/2017-11-16/60024053.html|title=疑影射六四事件 中国第三大共享单车倒闭|accessdate=2017-11-17|work=news.dwnews.com}}</ref>

===企业危机===
据[[新浪科技|新浪科技]]报道，2017年11月，小蓝单车存在[[押金|押金]]难退的问题<ref>{{Cite web|url=http://tech.sina.com.cn/i/2017-11-15/doc-ifynshev6229747.shtml|title=小蓝单车CEO预言全泡汤：高管分歧 公关总监曾被架空|accessdate=2017-11-16|work=tech.sina.com.cn}}</ref>。11月中旬，小蓝单车创始人李刚发表公开信，承认公司陷入严重危机，表示此后由[[拜客单车|拜客单车]]全权代理小蓝单车未来的运营。<ref>{{Cite web|url=http://tech.sina.com.cn/i/2017-11-16/doc-ifynwnty3549657.shtml|title=小蓝单车CEO公开信：拜客出行接手 尽快解决所欠工资|accessdate=2017-11-25|work=tech.sina.com.cn}}</ref>2018年1月9日，[[滴滴出行|滴滴出行]]和小蓝单车联合宣布，小蓝单车将由滴滴方面托管，未来用户可在滴滴APP内免押金骑行小蓝。此次托管合作之后，滴滴将为小蓝用户额外提供一种转换押金、特权卡和充值余额的方式，用户可自主选择是否转换为等值的滴滴单车券和出行券；而小蓝单车的品牌、押金和欠款等各项事务，则继续归属于小蓝公司<ref>{{Cite web|url=http://news.sina.com.cn/c/2018-01-09/doc-ifyqiwuw8585681.shtml|title=滴滴宣布托管小蓝单车：用户可将押金转换为等值滴滴出行券|accessdate=2018-01-09|work= 新华网 }}</ref>。
