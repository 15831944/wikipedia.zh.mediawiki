'''小峰峰'''（{{bd|1990年|4月26日||}}）{{NoteTag|其出生年份在音乐平台、抖音短视频填写为1990年，在新浪微博填写为1991年}}，本名'''陈峰'''，中国大陆流行音乐男歌手。他是歌曲《[[学猫叫|学猫叫]]》的作者，也是这首歌曲的原唱之一。<ref>{{cite web |title=歌手：小峰峰 |url=https://y.qq.com/n/yqq/singer/001IeM0J4eOqxi.html#stat=y_new.mv_lib.mv_singername |website=QQ音乐 |accessdate=2018-11-21}}</ref><ref name="甲">{{cite news |title=小峰峰酷狗直播被网友“逼疯”：你们都不想跟我聊天吗 |url=http://e.gmw.cn/2018-08/31/content_30905357.htm |accessdate=2018-11-21 |work=光明网 |date=2018-08-31 |archive-url=https://web.archive.org/web/20181123160555/http://e.gmw.cn/2018-08/31/content_30905357.htm |archive-date=2018-11-23 |dead-url=yes }}</ref>

== 早年生活 ==
小峰峰出生于江苏徐州，早年是一名武术体育生，曾多次在武术比赛中取得优异成绩，并成为国家二级运动员。后由于个人原因及社会原因，他选择放弃运动生涯，转战音乐界。<ref>{{cite web |title=小峰峰-艺人介绍 |url=https://music.163.com/#/artist/desc?id=790044 |website=网易云音乐 |accessdate=2018-11-21 |archive-url=https://web.archive.org/web/20181018134211/https://music.163.com/#/artist/desc?id=790044#/artist/desc?id=790044 |archive-date=2018-10-18 |dead-url=no }}</ref>

==演艺经历==
=== 出道初期：2009年－2011年 ===
2009年2月，年仅17岁的小峰峰在网上发表首支个人单曲《替代品》，正式进入音乐界。3月，发行个人单曲《伤之恋》，知名度得到一定提升。<ref>{{cite web |title=小峰峰个人资料 |url=http://m.9ku.com/geshou/7351/info.htm |website=九酷音乐 |accessdate=2018-11-21 |archive-url=https://web.archive.org/web/20171031162050/http://m.9ku.com/geshou/7351/info.htm# |archive-date=2017-10-31 |dead-url=no }}</ref>

2011年6月，发行首张个人录音室专辑《峰の格》。<ref>{{cite web |title=专辑：峰の格 |url=https://y.qq.com/n/yqq/album/000EPfd10aQXwn.html#stat=y_new.singer.album.album_pic |website=QQ音乐 |accessdate=2018-11-24 |archive-url=https://web.archive.org/web/20181124220245/https://y.qq.com/n/yqq/album/000EPfd10aQXwn.html#stat=y_new.singer.album.album_pic#stat=y_new.singer.album.album_pic |archive-date=2018-11-24 |dead-url=no }}</ref>
12月，发行第二张个人专辑《对不起，没关系》。

=== 发展时期：2012年－2017年 ===
2012年，推出单曲《听我唱完这首歌》。

2013年10月，推出第三张个人专辑《你又不是我》。2014年，先后发行个人专辑《我的猫咪公主》、《爱有时差》、《暖男》。

2015年起，侧重于发行单曲，先后发行《渐行渐远》、《梦西厢》、《我又想你了》等歌曲。在2015年9月，小峰峰的首张个人精选集正式发行。

=== 走红时期：2018年至今 ===
2018年4月20日，发行歌曲《[[学猫叫|学猫叫]]》（与小潘潘共同演唱）<ref>{{cite web |title=歌曲：学猫叫 |url=https://y.qq.com/n/yqq/album/4021662_num.html |website=QQ音乐 |accessdate=2018-11-21}}</ref>，这首歌曲发行后不久便走红网络，又一次提升了小峰峰的知名度；6月20日，发布歌曲《学猫叫》的MV。<ref>{{cite news |title=《学猫叫》MV今日发布 亿万点击红歌演唱者小潘潘&小峰峰首曝真容 |url=http://www.dzwww.com/yule/yy/201806/t20180620_17509626.htm |accessdate=2018-11-21 |work=大众网 |date=2018-06-20}}</ref>

2018年6月30日晚，与[[小潘潘_(歌手)|小潘潘]]一同参加[[中国中央电视台|中国中央电视台]]音乐频道（CCTV-15）节目《全球中文音乐榜上榜》，现场演唱《学猫叫》。这是小峰峰从艺9年以来的综艺节目首秀。<ref>{{cite web |title=[全球中文音乐榜上榜]歌曲《学猫叫》 演唱：小潘潘 小峰峰 |url=http://tv.cctv.com/2018/06/30/VIDEHPixGXHhsS7lxhylCmTi180630.shtml |website=CCTV |accessdate=2018-11-21}}</ref>

2018年8月21日，与[[冯提莫|冯提莫]]共同演唱的《左手》（《皇室战争》手游宣传曲）正式上线<ref>{{cite web |author1=新闻游戏 |title=当冯提莫新歌碰上抖音首个超级挑战，《皇室战争》宣传曲或成最大赢家 |url=http://www.sohu.com/a/246544775_646327 |website=搜狐 |accessdate=2018-11-21}}</ref>；8月28日晚，小峰峰做客酷狗直播，现场演唱《学猫叫》、《以父之名》等歌曲<ref name="甲" />；8月31日，发行新歌《绝对不是小黄歌》（与馨哥哥共同演唱）<ref>{{cite news |title=《绝对不是小黄歌》首发 甜蜜暴击撩拨少女心 |url=http://ent.workercn.cn/30024/201808/31/180831133323804.shtml |accessdate=2018-11-21 |work=中工网 |date=2018-08-31}}</ref>；11月11日，在海南三亚参加《2018网络热曲原声秀》。<ref>{{cite news |title=三亚梦幻水上乐园奏响2018网络热曲原声秀 |url=http://www.hi.chinanews.com/hnnew/2018-11-12/476483.html |accessdate=2018-11-21 |work=中国新闻网 |date=2018-11-12}}</ref>
