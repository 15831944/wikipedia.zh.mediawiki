'''小木通'''（[[學名|學名]]：''{{lang|la|Clematis armandii}}''，[[英文|英文]]名稱：{{lang|en|Armand Clematis、Evergreen Clematis}}<ref name="浸大">{{cite encyclopedia | encyclopedia=香港浸會大學中醫藥學院藥用植物圖像數據庫| title= 小木通 | volume= | pages=|url=http://libproject.hkbu.edu.hk/was40/detail?lang=ch&record=1&channelid=1288&searchword=%28name_formal_latin_sort%3D%60Clematis+armandii%60%29+&extension=all&sortfield=%2Bname_formal_latin_sort}}</ref>，別稱'''木通'''、'''大木通'''、'''大淮通'''、'''川木通'''、'''淮木通'''、'''淡木通'''、'''山木通'''、'''花木通'''、'''紫木通'''、'''白木通'''、'''辣木通'''、'''大葉木通'''、'''三葉木通'''、'''野淮通'''、'''蓑衣藤'''、'''簑衣藤'''、'''粗糠藤'''、'''黃防已'''、'''黃防己'''、'''長葉侯培'''、'''疏序鐵線蓮'''、'''鐵馬鞭'''、'''威靈仙'''及'''亂根草'''等<ref name="自然標本館"/>，為[[毛茛科|毛茛科]][[鐵線蓮屬|鐵線蓮屬]]植物<ref name="浸大"/>。模式標本採自於[[四川|四川]][[寶興|寶興]]<ref name="高等植物">{{cite web|url=http://www.cnpc.ac.cn/Taxon/Details/62026|title=Clematis armandii Franch.|publisher=中國高等植物信息系統}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="中國">{{cite encyclopedia| title=小木通| encyclopedia=中國植物誌| volume=28| pages=175-177| url=http://v2.cvh.org.cn/zhiwuzhi/page/28/175a.pdf}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==分佈==
本種現分佈於[[中國大陸|中國大陸]]、[[越南|越南]]及[[緬甸|緬甸]]等國家。中國大陸境內分佈於[[廣西|廣西]]、[[廣東|廣東]]、[[雲南|雲南]]、[[福建|福建]]、[[浙江|浙江]]、[[貴州|貴州]]、[[甘肅|甘肅]]、[[湖南|湖南]]、[[湖北|湖北]]、[[四川|四川]]、[[陝西|陝西]]、[[江西|江西]]、[[西藏|西藏]]等省區<ref name="ars"/><ref>{{cite web|url=http://data.sp2000.cn/2010_cnnode_c/show_species_details.php?name_code=2404569b-b80e-46bf-a448-f9830416ce9a|title=Clematis armandii Franch.|publisher=中國生物物種名錄2010版}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，野外喜生於山谷、山坡、林邊、水溝旁或路旁灌叢之中<ref name="中國"/>。

==形態特徵==

小木通是一種常綠[[攀緣|攀緣]]狀木質[[藤本|藤本]]植物<ref name="西藏">{{cite encyclopedia| title=小木通| encyclopedia=西藏植物誌| volume=2| pages=90| url=http://v2.cvh.org.cn/difangzhi/xizang/page/2/090b.pdf}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="浙江">{{cite encyclopedia| title=小木通| encyclopedia=浙江植物誌| volume=2| pages=291-292| url=http://v2.cvh.org.cn/difangzhi/zhejiang/page/2/291a.pdf}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，高可達6[[米|米]]<ref name="中國"/>。[[莖|莖]]圓柱形，具縱條紋，小枝具棱，披白色短柔毛，後漸脫落至[[無毛|無毛]]<ref name="浸大"/>，乾後黑褐色或微紅褐色<ref name="秦嶺">{{cite encyclopedia| title=小木通| encyclopedia=秦嶺植物誌| volume=1(2)| pages=294| url=http://v2.cvh.org.cn/difangzhi/qinling/page/1(2)/294a.pdf}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

===葉===

[[葉|葉]]為三出[[複葉|複葉]]，對生<ref name="浸大"/>；[[葉柄|葉柄]]基部膨大，具縱條紋<ref name="秦嶺"/>，長約4<ref name="浙江"/>－12<ref name="貴州">{{cite encyclopedia| title=小木通| encyclopedia=貴州植物誌| volume=3| pages=57-59| url=http://v2.cvh.org.cn/difangzhi/guizhou/page/3/057a.pdf}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>[[厘米|厘米]]，與葉片近等長<ref name="西藏"/>；小葉卵形、長圓狀卵形、卵狀披針形，革質，先端較尖，基部圓形、心形或寬楔形，兩面均無毛，全緣，基出脈3－5條<ref name="福建">{{cite encyclopedia| title=小木通| encyclopedia=福建植物誌| volume=2| pages=27| url=http://v2.cvh.org.cn/difangzhi/fujian/page/2/027.pdf}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，於背面隆起<ref name="西藏"/>，[[網脈|網脈]]兩面明顯<ref name="福建"/>，長約4－16[[厘米|厘米]]，寬約2－8[[厘米|厘米]]<ref name="中國"/>；小葉柄扭曲，長約2.5－4[[厘米|厘米]]<ref name="貴州"/>。

===花===

[[花|花]]為聚傘[[花序|花序]]或圓錐狀聚傘花序，頂生或腋生<ref name="中國"/>，常與葉近等長或長於葉<ref name="福建"/>；腋生的花序基部具有多數宿存[[芽鱗|芽鱗]]，芽鱗膜質<ref name="秦嶺"/>，卵形、三角狀卵形至長圓形，長約0.8－3.5[[厘米|厘米]]<ref name="福建"/>；花序軸、分枝及花梗均疏披短柔毛<ref name="西藏"/><ref name="貴州"/>；花為兩性花<ref name="浸大"/>，直徑約2<ref name="秦嶺2">{{cite encyclopedia| title=小木通| encyclopedia=秦嶺植物誌| volume=1(2)| pages=295| url=http://v2.cvh.org.cn/difangzhi/qinling/page/1(2)/295.pdf}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>－4<ref name="河南">{{cite encyclopedia| title=小木通| encyclopedia=河南植物誌| volume=1| pages=459-460| url=http://v2.cvh.org.cn/difangzhi/henan/page/1/459a.pdf}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>[[厘米|厘米]]；總[[花梗|花梗]]粗壯，具細縱條紋，披短毛或近無毛<ref name="秦嶺2"/>，長約3.5－7[[厘米|厘米]]<ref name="河南"/>；[[花瓣|花瓣]]無<ref name="浸大"/>；花序常為3淺裂，苞片膜質<ref name="秦嶺2"/>，上部[[苞片|苞片]]漸小，狹披針形，披[[鏽色|鏽色]]短柔毛<ref name="西藏"/>，下部苞片近長圓形，長約1.2[[厘米|厘米]]<ref name="西藏"/>；[[萼片|萼片]]4－7枚<ref name="浸大"/>，長圓形或長橢圓形，大小變異極大，白色偶帶淡紅色，開展，無毛<ref name="秦嶺2"/>，僅外邊緣密披白色短絨毛<ref name="秦嶺2"/>，長約1－4[[厘米|厘米]]，寬約0.3－2[[厘米|厘米]]<ref name="中國"/>；[[雄蕊|雄蕊]]多數，無毛<ref name="福建"/>，長約7毫米<ref name="秦嶺2"/>；[[花絲|花絲]]絲狀<ref name="秦嶺2"/>；[[花藥|花藥]]線形，比花絲為長，[[藥室|藥室]]側向開裂<ref name="秦嶺2"/>；[[子房|子房]]披柔毛<ref name="貴州"/><ref name="福建"/>；[[心皮|心皮]]多數<ref name="浸大"/>；[[雌蕊|雌蕊]]披白色長柔毛<ref name="秦嶺2"/>。

===果實===

[[果實|果實]]為[[瘦果|瘦果]]，卵形至橢圓形，扁<ref name="福建"/>，中間微凹<ref name="秦嶺2"/>，邊緣凸出<ref name="秦嶺2"/>，疏披伸展短柔毛<ref name="西藏"/><ref name="河南"/>，長約4－7[[毫米|毫米]]，寬約2[[毫米|毫米]]<ref name="秦嶺2"/>；宿存羽毛狀[[花柱|花柱]]<ref name="西藏"/>，具白色長柔毛，長約5[[厘米|厘米]]<ref name="中國"/>。

==用途==
本種全草與[[樟腦|樟腦]]及水後煮沸榨汁，可作[[農藥|農藥]]之用，主治[[菜青蟲|菜青蟲]]、[[橋蟲|橋蟲]]、地老虎、[[瓢蟲|瓢蟲]]等植物蟲害 <ref name="高等植物"/><ref name="秦嶺2"/>。

===醫藥用途===
{{medical}}
本種以藤莖入藥，[[中藥|中藥]]名為川木通，味淡、微苦，性寒<ref name="浸大"/>，有小毒，歸心、小腸及膀胱經<ref name="意古">{{cite web| url=http://www.tcmlib.com/zy/html24/showdetail-323337382ce5b79de69ca8e9809a2c7a79.html| title=川木通| publisher=意古中醫| deadurl=yes| archiveurl=https://web.archive.org/web/20120427042851/http://www.tcmlib.com/zy/html24/showdetail-323337382ce5b79de69ca8e9809a2c7a79.html| archivedate=2012-04-27}}</ref>，[[藥材|藥材]]主產於[[四川|四川]]、[[貴州|貴州]]<ref name = "常用">{{cite book|title=常用中藥材鑑別圖典|author=趙中振, 陳虎彪|publisher=萬里機構|page=第214頁|isbn= 978-962-14-4150-8|date=2010年1月}}</ref>、[[甘肅|甘肅]]、[[湖南|湖南]]<ref name = "常用"/>、[[湖北|湖北]]<ref name = "常用"/>、[[陝西|陝西]]、[[福建|福建]]等地<ref name="浸大"/>，[[中醫|中醫]]歸類為清熱藥、利水滲濕藥<ref>{{cite web |url=http://libproject.hkbu.edu.hk/was40/detail?channelid=35734&searchword=herb_id=D00681| title= 川木通 Armand Clematis Stem Chuan Mu Tong| publisher=香港浸會大學中醫藥學院中藥標本數據庫}}</ref>及活血藥<ref name="意古"/>，具清熱利[[尿|尿]]、消腫<ref name="高等植物"/>、通經下乳等功效<ref name="浸大"/>，主治[[小便|小便]]不利、尿路感染、閉經、[[乳汁|乳汁]]不通、[[風濕|風濕]]、胃痛、淋証熱痛、下焦濕熱所致的水腫、[[小兒麻痺|小兒麻痺]]後遺症等<ref name="浸大"/><ref name="高等植物"/>，外用可治腰腿痛及外傷後的腐肉<ref name="高等植物"/>，小便過多、精滑遺尿、氣弱津傷、腸胃潰瘍者<ref name="高等植物"/>及孕婦禁服<ref name="浸大"/>。於秋季採收，去皮清除雜質後切片曬乾<ref name="浸大"/>。本種[[根部|根部]]及[[花|花]]入藥，具解熱、止痛等功效，主治喉痹、勞傷等；根部主治跌打癆傷；花主治乳娥 <ref name="高等植物"/><ref name="秦嶺2"/>。

==藥材鑑定==

[[中藥|中藥]]川木通以乾燥的藤莖入藥，於[[春|春]]、[[秋|秋]]兩季採挖<ref name = "常用"/>，清除粗皮泥沙等雜質後曬乾或趁鮮時切片曬乾<ref name = "常用"/>；本品呈細長圓柱形，畸為扭曲，表面黃褐或黃棕色，具棱線及縱向凹溝，節膨大，具葉痕及側枝痕<ref name = "常用"/>，直徑3.5－6.5[[厘米|厘米]]<ref name="香港中藥材">{{cite book|title=《香港中藥材圖鑑》|author= 趙中振|publisher=香港浸會大學中醫藥學院|year=2003 |month=第38頁 |page=208|isbn= 988-97448-1-3}}</ref>；飲片外皮多已脫落，邊緣並不整齊<ref name = "常用"/>，殘存皮部黃棕色，木部淺棕色，寬廣，密布小孔<ref name="香港中藥材"/>，髓部較小，偶有空腔，具黃白色放射狀紋理及裂縫<ref name = "常用"/>；質地堅硬，不易折斷<ref name = "常用"/>，氣微無臭，味淡。《[[中國藥典|中國藥典]]》將本種植物列為[[中藥|中藥]]川木通的第一種來源<ref name = "混淆">{{cite book|title=香港容易混淆中藥|author=趙中振, 李應生|publisher=香港浸會大學中醫藥學院|page=第64-65頁|isbn= |date=2010年1月}}</ref>，另與本種同屬的[[繡球藤|繡球藤]]（''Clematis Montana''）同時收錄於《[[中國藥典|中國藥典]]》中，定為法定原植物來源種之一<ref name = "常用"/>；傳統經驗則認為以條粗、斷面黃白色者為佳<ref name = "常用"/>，容易與中藥關木通的原植物來源[[東北馬兜鈴|東北馬兜鈴]]（''Aristolochia manshuriensis''）的藤莖混淆使用<ref>{{cite encyclopedia| encyclopedia=香港容易混淆中藥，中藥數據庫| title=川木通與關木通| volume=| pages=| url=http://hkjccm.com/cm_database/confused/detail.aspx?herb_id=31}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；因東北馬兜鈴含[[馬兜鈴酸|馬兜鈴酸]]類成分，長期大量服用可引致腎小管壞死、慢性或急性[[腎|腎]]衰竭及[[尿道癌|尿道癌]]等病症，因此2004年[[香港衛生署|香港衛生署]]已通報停止使用<ref name = "混淆"/>。
