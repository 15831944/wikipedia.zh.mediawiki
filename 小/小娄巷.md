'''小娄巷'''为位于[[中国|中国]][[江苏省|江苏省]][[无锡市|无锡市]][[梁溪区|梁溪区]]的一处历史文化街区，无锡市四大历史文化街区之一，其范围东西分别至苏家弄和新生路，南北分别至崇宁路和福田巷，现存北面和南面的东半部，占地面积约4万平方米。该片区为[[无锡谈氏|无锡谈氏]]和[[无锡秦氏|秦氏]]两大名门望族的世居之地，始于宋代、盛于明清，自宋代以来曾有一位状元、十三位进士和十五位举人出自此处<ref>{{cite web |url=http://www.js.chinanews.com/news/2009/0331/6188.html|title=《无锡小娄巷历史街区将修复 呈现都市“慢生活”》|accessdate=2012-12-21|work=|publisher=江苏新闻网}}</ref>。现存建筑以晚清-民国时期的为主，2002年其中25个门牌号范围内的古建筑被列入江苏省文物保护单位，另在小娄巷50号存有无锡市古树名木中的唯一一株清代牡丹<ref>{{cite web|url=http://news.thmz.com/col89/2012/04/2012-04-141104536.html|title=《无锡最珍贵古树 小娄巷148岁牡丹开出37朵花》|accessdate=2012-12-21|work=|publisher=江南晚报|deadurl=yes|archiveurl=https://web.archive.org/web/20120710234825/http://news.thmz.com/col89/2012/04/2012-04-141104536.html|archivedate=2012-07-10}}</ref>。然而2008年由无锡市崇安区政府重点项目建设办公室和崇安城市投资发展有限公司提交“修复和建设小娄巷历史文化街区”项目相关拆迁手续，并由无锡市规划局、建设局通过，进而以保护性修复的名义对该区域内的建筑进行了拆迁和改造，并试图将其打造为慢生活体验街区<ref>{{cite web |url=http://news.qq.com/a/20091023/001499.htm|title=《无锡小娄巷遭违法拆迁 文保单位拆迁该谁说了算》|accessdate=2012-12-21|work=方霞、薛庆元|publisher=中国消费者报}}</ref><ref>{{cite web |url=http://news.sina.com.cn/c/2009-10-22/061516479028s.shtml|title=《拆迁许可撤销，老街还要拆？》|accessdate=2012-12-21|work=金辰、陈超|publisher=现代快报}}</ref>。
