'''小堀政一'''（こぼり まさかず，{{bd|1579年||1647年|3月12日|catIdx=Kobori Masakazu}}，[[天正_(日本)|天正]]7年－[[正保|正保]]4年2月6日）是[[安土桃山時代|安土桃山時代]]～[[江戶時代|江戶時代]]前期的[[大名_(稱謂)|大名]]、[[茶道|茶人]]、[[建築家|建築家]]、[[作庭家|作庭家]]。[[備中松山藩|備中松山藩]]2代藩主、後[[近江小室藩|近江小室藩]]初代藩主。'''小堀遠州'''（こぼり えんしゅう）之名較為人所知（「遠州」的由來是[[武家官位|武家官位]]的遠江守）。[[幼名|幼名]]作助、[[元服|元服]]後名為正一、後改為政一。道號是大有宗甫、庵號是[[孤篷庵|孤篷庵]]。

== 生涯 ==
[[天正_(日本)|天正]]7年（1579年），出生，[[小堀正次|小堀正次]]之長男。父・正次原本是[[近江國|近江國]][[坂田郡|坂田郡]]小堀村（現・[[滋賀縣|滋賀縣]][[長濱市|長濱市]]）的[[土豪|土豪]]、[[淺井長政|淺井長政]]之家臣、緣戚。[[淺井氏|淺井家]]被[[織田信長|織田信長]]滅亡後，仕於[[長濱城_(近江國)|長濱城]]主[[豐臣秀吉|羽柴秀吉]]之弟[[豐臣秀長|秀長]]。天正13年（1585年），秀長移封[[郡山城_(大和國)|郡山城]]，正次成為秀長之[[家老|家老]]，共同移到郡山。

此時，秀長招來[[山上宗二|山上宗二]]，並師事[[千利休|千利休]]，郡山在當時和京・堺・奈良並列為茶湯盛行之地。身為[[小姓|小姓]]的政一受父親建議，在[[大德寺|大德寺]]的[[春屋宗園|春屋宗園]]之處參禪。秀長死後，[[文祿|文祿]]4年（1595年），成為秀吉直參移到[[伏見|伏見]]。政一在此向[[古田重然|古田織部]]學習[[茶道|茶道]]。

[[慶長|慶長]]3年（1598年），秀吉没後，正次・政一仕於[[德川家康|德川家康]]。正次因在[[關原之戰|關原之戰]]之功，獲賜[[松山城_(備中國)|備中松山城]]，作為[[備中國|備中]]代官前往松山（現・[[岡山縣|岡山縣]][[高梁市|高梁市]]）赴任。慶長9年（1604年），父没後，政一繼承遺領1萬2460石。

慶長13年（1608年），成為[[駿府城|駿府城]][[普請奉行|普請奉行]]，因修築之功，叙任[[從五位|從五位下]][[遠江國|遠江守]]。以後以此[[官名|官名]]，被稱作'''小堀遠州'''。

[[元和_(日本)|元和]]3年（1617年），兼任[[河內國|河內國]]奉行。元和5年9月（1619年10月），移封[[近江小室藩|近江小室藩]]，元和8年8月（1622年9月），任[[近江國|近江國]]奉行。元和9年12月（1624年1月），任[[遠國奉行|伏見奉行]]。

[[正保|正保]]4年2月6日（1647年3月12日），在伏見奉行屋敷過世，享年69歳。

== 作事 ==
政一在公儀作事的主要業績是[[松山城_(備中國)|備中松山城]]的再建、[[駿府城|駿府城]]修築、[[名古屋城|名古屋城]][[天守|天守]]、[[後陽成天皇|後陽成院]]御所造營等宮中或幕府關係的作事奉行。

作為奉行參畫的建築是[[妙心寺|妙心寺]][[麟祥院|麟祥院]]的[[春日局|春日局]][[靈屋|靈屋]]、[[冰室神社|冰室神社]]拜殿、[[大覺寺_(京都)|大覺寺]]宸殿、[[金地院|金地院]]東照宮、茶室、方丈南庭（鶴龜庭）、[[南禪寺_(京都市)|南禪寺]]本坊方丈南庭、[[大德寺|大德寺]]龍光院密庵席、[[孤篷庵|孤篷庵]]表門前的石橋、前庭、忘筌席露地、[[仙洞御所|仙洞御所]]的庭園。

== 茶湯 ==
政一的茶湯作為[[遠州流|遠州流]]（小堀遠州流）傳承至今。

生涯舉辦約400回茶會，招待的客人據說約2000人。門下有[[松花堂昭乘|松花堂昭乘]]、[[澤庵宗彭|澤庵宗彭]]等人。

== 華道 ==
小堀遠州的美意識也影響[[華道|華道]]的世界，流儀分成正風流・日本橋流・淺草流之三大流派。[[昭和|昭和]]初期，從既成的流派又獨立出許多[[家元|家元]]、[[宗家|宗家]]。
