'''院政'''指[[日本|日本]]政权由[[摄关政治|摄关政治]]转移到[[幕府_(日本)|幕府]]的过渡时期的[[政治体制|政治体制]]。[[天皇|天皇]][[禪位|禪位]]與子侄，而自稱[[上皇|上皇]]，在「院」中[[訓政|訓政]]的政治形態。是皇權為了抵抗攝關政治而發展出來的政治制度。

== 背景 ==

[[藤原氏|藤原氏]]透過[[攝關政治|攝關政治]]掌權下，天皇大權旁落。引起了地方[[豪族|豪族]]與[[武士|武士]]的不滿。到了後三條天皇，因其非藤原氏所生，親政後極力奪回政權。藤原氏的[[外戚|外戚]]集團引起了整個朝廷的矛盾。攝關政治主要透過外戚的身份來干政，而後三條天皇為了擺脫不願當攝關家的傀儡，結合了反對藤原氏的各種勢力，整理莊園、整頓朝政積極恢復皇權。

[[後三條天皇|後三條天皇]]引入了自己的親信[[村上源氏|村上源氏]]的[[源師房|源師房]]，並在1069年發布了[[莊園整理令|莊園整理令]]，整頓藤原氏所屬的莊園。地方反對藤原氏的「受領階級」也參與了反對藤原氏的運動。藤原氏大量的莊園被劃歸為上皇直屬的「院領」。1072年，後三條天皇退位於白河天皇。開設院廳，意圖以上皇之姿重新掌控朝政。卻在隔年病死。院政的確立從[[白河天皇|白河天皇]]才真正開始。

== 制度确立 ==

[[白河天皇|白河天皇]]为对抗[[摄关|摄关]]，退位为[[太上天皇|太上天皇]]。居住在院（白河院）。依靠中下层[[武士|武士]]，招募[[军队|军队]]，建立[[朝廷|朝廷]]百官，频频颁布[[院宣|院宣]]，成为政务的仲裁者，最后[[出家|出家]]为[[太上法皇|太上法皇]]，仍掌握大权。以后[[天皇|天皇]]多遵循此例。白河天皇在居住的院廳中設置了各種官職，並組織軍事力量「[[北面武士|北面武士]]」，作為親衛武裝力量。院廳的官員多為上皇的近臣，多是中下級官僚。代表了皇權與貴族的分立。

上皇以[[院宣|院宣]]的方式發布命令。其權威高過於天皇的詔令。而原屬[[太政大臣|太政大臣]]的職權，雖然經過公卿會議決議，最終的裁定卻在上皇手上。朝政的一切都掌控在上皇中。這百餘年皆由上皇實行院政。而過去攝關政治的權力來源仍是天皇，所以要以天皇的權威作為根基。院政的權威卻直接來自於上皇。透過父親尊親屬的關係控制朝政。

對抗攝關政治的院政形成與其說是皇室與外戚的矛盾所構成，不如說是中下官僚與大貴族的鬥爭。後三條天皇得以整頓莊園，就是因為中下官僚的支持。

== 院政的基礎 ==

攝關政治的基礎是藤原氏掌控的大量[[莊園|莊園]]，而院政卻是[[行國制|行國制]]，是將一個國（行政區）的行政、徵稅權在一定時間賦予特定個人以換取對上皇支持的制度。這樣反而使[[中央集權|中央集權]]的制度加速瓦解。院政接收了藤原氏與豪族的莊園，而積極整頓莊園反而引起了領主的不安。為了避免遭到清算，許多的莊園領主自動向上皇靠攏。上皇為了報答其支持，給予了豁免租稅的優待，反而使莊園的獨立性更強。

由於許多上皇篤信佛教，更遁入[[空門|空門]]，成為[[法皇|法皇]]，使[[佛教|佛教]]的勢力擴張。[[神社|神社]]寺院有自己的[[僧兵|僧兵]]，屢次與地方[[國司|國司]]相爭。朝廷面對寺院的鬥爭，反而束手無策。畏懼神佛的貴族無法反抗，只能倚靠不畏懼神佛的[[武士|武士]]來相抗衡。間接造成了武士的抬頭。

== 院政的意義與影響 ==

院政的是為了對抗攝關政治形成的畸形政治制度，雖然成功的把藤原氏的的勢力逐出朝廷。但上皇與攝關一樣，好大喜功，更因為深信佛教而大興土木興建佛寺，舉辦大規模法會而耗費財政。為了彌補財政空洞，反而加深賣官的情況。院政導致朝政的腐敗，更釀成日後的保元、平治之亂。
