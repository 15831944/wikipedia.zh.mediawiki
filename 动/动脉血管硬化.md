'''动脉血管硬化'''（{{lang-en|Arteriosclerosis}}）是指[[动脉|动脉]]血管壁变厚、变硬并失去弹性的过程。<ref>{{DorlandsDict|one/000008408|arteriosclerosis}}</ref>这一过程会使得脏器、组织的供血逐渐减少，并因[[动脉粥样硬化|动脉粥样硬化]]（动脉血管硬化的一种，因脂肪、[[胆固醇|胆固醇]]在动脉沉积引发）而产生严重的健康问题。<ref>{{Cite web|title = NCBI - WWW Error Blocked Diagnostic|url = http://www.ncbi.nlm.nih.gov/pubmedhealth/PMHT0023229/|website = www.ncbi.nlm.nih.gov|accessdate = 2015-06-19}}</ref>

==形成==
[[低密度脂蛋白|低密度脂蛋白]]（LDL）俗稱「壞的膽固醇」，容易附着在血管壁上，是形成动脉硬化的元兇之一。<ref>{{Cite book |title=圖解循環系統的疾病與機制 |author=砂山 聰 |date=2005 |publisher=世茂出版有限公司}}</ref>{{page|80}}

==症状==
动脉血管硬化的症状有：突发虚弱、面部或下肢[[麻木|麻木]]、{{tsl|en|confusion|思考混乱}}、无法理解语言并注意到问题等。<ref name="ref1">{{Cite web|title = How Is Atherosclerosis Diagnosed? - NHLBI, NIH|url = http://www.nhlbi.nih.gov/health/health-topics/topics/atherosclerosis/diagnosis|website = www.nhlbi.nih.gov|accessdate = 2015-06-19}}</ref>
