'''动吻动物门'''（[[學名|學名]]：{{lang|la|Kinorhyncha}}）是[[动物界|动物界]]的一个[[门_(生物)|门]]。是一类生活在沿[[海|海]]底部泥沙中的、体表分[[节带|节带]]（zonites）、无纤毛的[[假体腔动物|假体腔动物]]。约有100种左右，例如[[动吻虫|动吻虫]]（Echinoderes）等。牠们的近亲是[[鎧甲动物门|鎧甲动物门]]和[[鳃曳动物门|鳃曳动物门]]，此三門統稱為[[有棘動物|有棘動物]]（Scalidophora）。

== 形態 ==
動吻動物的軀幹上有空心的刺，[[體節|體節]]數目少而且寬，體節上覆蓋骨板。有11個體節，刺較小。沒有[[眼睛|眼睛]]。動吻動物的吻部在[[有棘動物|有棘動物]]中比較發達，例如[[动吻虫|动吻虫]]（Echinoderes）等，可見其明顯的吻部膨大結構。<ref>{{Cite web |url = http://chuansong.me/n/1765017752915 |title = 曳鰓、鎧甲、動吻：有棘動物類群 |author = 小白 |date = 2017年4月12日 |language = zh-hans }}</ref>
