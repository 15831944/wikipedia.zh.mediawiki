{{vgname|動物之森 口袋营地|en=Animal Crossing: Pocket Camp|ja=どうぶつの森 ポケットキャンプ}}是一款由[[任天堂|任天堂]]开发并发行在[[iOS|iOS]]和[[Android|Android]]智能手机平台上的网络社交游戏，本作是《[[动物之森系列|動物之森]]》系列在手机平台上的首部游戏作品，也是任天堂继《[[Miitomo|Miitomo]]》、《[[超级马里奥酷跑|超级马里奥酷跑]]》、《[[火焰之纹章_英雄|火焰之纹章 英雄]]》后开发的第四款智能手机游戏。在游戏中，玩家可以与小鎮的居民進行互動、完成任務、从事商业活动或使用家具来装饰露营地点。本作采用了带有付费内容的免费下载形式，将于2017年11月起在全球陆续上线。<ref>[https://www.g-cores.com/articles/93880 《动物之森 口袋营地》11月下旬配信] {{Wayback|url=https://www.g-cores.com/articles/93880 |date=20171107015725 }}.机核网.2017-10-25.[2017-11-04].</ref>

== 游戏玩法 ==

動物森林 口袋营地是一款社交模擬类型遊戲，玩家可以通過交易获得材料和使用家具来定制其[[营地|营地]]。在游戏中，玩家可以周邊地區收集[[木材|木材]]、[[棉花|棉花]]、[[鋼鐵|鋼鐵]]、[[紙|紙]]、[[罐頭|罐頭]]等材料進行交易，也可以与動物角色进行交谈并结为好[[朋友|朋友]]。除外他玩家也可以訪問其他[[玩家|玩家]]的營地，或邀請其他玩家来自己的营地里遊玩。<ref>{{cite web |url=http://www.ign.com/articles/2017/10/25/animal-crossing-pocket-camp-is-coming-to-mobile-in-november |title = O'Brien, Lucy (October 24, 2017). "Animal Crossing: Pocket Camp is Coming to Mobile in November". |publisher = [[IGN|IGN]] |date = 2017-10-25}}</ref>

玩家也可以根據自己的偏好定制人物的頭像、性別，面部特徵和露营房车的駐地。

玩家可以在附近的「娛樂場所」里完成動物們的任務，以獲得材料並將材料交給鐵匠或木匠以便製作成家具。

玩家可以在營地放置自己喜歡的家具以吸引特定的鄰居。每當鄰居拜訪時，會增加其友好值，這也該系列的新特性。<ref>{{Cite web |url= https://www.polygon.com/2017/10/24/16536774/animal-crossing-mobile-ios-android-release-date |title = Frank, Allegra (October 24, 2017). "Animal Crossing: Pocket Camp for mobile out next month". |publisher = [[Polygon|Polygon]] |date = 2017-10-25}} </ref>

遊戲中添加了一個名為Leaf Tickets貨幣，玩家可以在原材料不足時用來補充材料，也可以用來跳過製作時間。<ref>{{Cite web |url = https://www.polygon.com/2017/10/25/16541408/animal-crossing-pocket-camp-tom-nook-kk-slider-microtransactions |title = Frank, Allegra (October 25, 2017). "Animal Crossing: Pocket Camp trades two fan faves for in-game cash" |publisher = [[Polygon|Polygon]] |date = 2017-10-25}}</ref>

遊戲會在特定時期添加特定的主題家具。
