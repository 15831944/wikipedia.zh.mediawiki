'''嬌黃錐拱獸面紋鼎'''，高16.8公分，口徑13.3公分，現藏於臺灣臺北[[國立故宮博物院|國立故宮博物院]]。
 
== 年代 ==
明朝（西元16-17世紀），作者周丹泉是[[景德鎮|景德鎮]]著名的陶瓷仿古專家，生存年代在[[明朝|明朝]][[隆庆_(明朝)|隆慶]]、[[萬曆|萬曆]]年間，此瓷器的生產年代應介於十六世紀末期至十七世紀初期，地點為[[中國|中國]]的景德鎮。<ref>{{Cite book|title = 摶泥幻化 : 院藏歷代陶瓷 | author = 國立故宮博物院| location = 臺灣 | publisher = 馮明珠 | date = 2014-09 | pages = | ISBN = 9789575627126 | accessdate = 2017-05}}</ref>

== 外觀 ==
{{section OR}}
此[[爐|爐]]玲瓏嬌小，約莫兩個手掌大，其外觀為圓鼎造型，爐口兩側各有一立耳為裝飾，底部管狀三足承接其重量，恰成完美的三足鼎立，平均分擔其重量。每足壁皆有精美的裝飾性泥條，恰似脊椎[[形狀|形狀]]、捲曲如雲般的[[紋路|紋路]]。<ref name ="故宮網站">{{Cite web |url = http://theme.npm.edu.tw/selection/Article.aspx?sNo=04001113#inline_content_intro |title = 嬌黃錐拱獸面紋鼎|publisher = 故宮博物院 |language = zh-tw |accessdate = 2017-05}}</ref>

為避免此件精細嬌小的[[瓷器|瓷器]]在燒製的過程破裂，足壁內有中空的設計，每個足壁朝內的方向上各有一透氣小口。

此鼎通體罩施嬌黃色釉（帶點粉紅的淺黃色），[[釉|釉]]質透亮，僅三足底露白胎，若側視則可稍微看見若隱若現的五彩光澤，以上種種特色亦符合了[[北宋|北宋]][[定窯|定窯]]的幾個特徵：胎土白淨、紋飾精美、釉色為溫潤微黃的奶白色<ref>{{Cite book | author = 劉如水 | title = 宋元明清瓷器鑑賞| location = 臺灣| publisher = 五南圖書出版股份有限公司 | date = 2004 | pages = 74 | ISBN =9789861211671 | accessdate = 2017 | language = zh-tw}}</ref>，{{OR|可以判定，此瓷器是周丹泉最擅長模仿之北宋定窯白瓷的傑作，若說它是周丹全的傳世絕作也當之無愧。}}

爐身前後兩面劃獸面[[紋飾|紋飾]]，爐身有象徵財富的金錢紋做裝飾，而左右兩側更飾以朵花象徵富貴，{{OR|可見不單在色彩的使用上（帶粉紅的金黃色），其刻劃與黏貼的紋路和符紋更使整件作品充滿帝王之貴氣。}}

底部雕刻「周丹泉造」四字楷款[[陰刻|陰刻]][[楷體|楷體]][[款識|款識]]，此四字清晰可見，為台北國立故宮博物院典藏唯一刻有周丹全款識的作品<ref name ="故宮網站"/><ref>{{cite web |author3=官方網站 |title=古色特展_展品賞析_明 嬌黃錐拱獸面紋鼎 |url=https://www.npm.gov.tw/exh92/trends/c_artworkpage_c06.htm |website=www.npm.gov.tw |accessdate=2018-07-24 |language=zh-tw |date=2003-12-12}}</ref>。
