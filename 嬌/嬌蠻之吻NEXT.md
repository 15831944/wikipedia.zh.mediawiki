{{日本作品|嬌蠻之吻NEXT|つよきすNEXT}}是CandySoft（{{lang|ja|きゃんでぃそふと}}）在2013年12月20日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]，《[[嬌蠻之吻|嬌蠻之吻]]》系列的全新作品。<ref>{{cite web |title=新世代・強気っ娘恋愛アドベンチャー『つよきすNEXT』がマスターアップ！ ―12月20日発売予定 |url=http://moedigi.com/game/2013/12/tuyokiss-next-master-up/ |website=moedigi |language=ja |date=2013年12月6日}}</ref>2015年12月25日發售續作《嬌蠻之吻FESTIVAL》。<ref>{{cite web |title=【TG SCOOP】CandySoft代表作『つよきす』シリーズ10周年記念の最新作が登場!! |url=https://www.tgsmart.jp/article.aspx?a=7002 |website=TG Smart |language=ja |date=2015/07/22}}</ref>

==故事==
蟹澤佐名木在對馬心秤的引導下加入學生會，卻也因此捲入了學生會和風紀委員會之間的鬥爭中。

;{{nihongo|蟹澤佐名木|蟹沢 さなぎ}}
:本作的男主角，龍鳴館2年C班學生，後來成為[[學生會|學生會]]的副會長。獨自住在遠房親戚家。
;{{nihongo|對馬心秤|対馬 心秤|聲優：上原あおい}}

;{{nihongo|刃牙音子|刃牙 音子|聲優：波奈束風景}}

;{{nihongo|八宮・Yellowdo・Cherish|八宮・イエロード・チェリシュ|聲優：藤咲ウサ}}

;{{nihongo|伊那瀨子羽|伊那瀬 子羽|聲優：[[かわしまりの|かわしまりの]]}}

;{{nihongo|大女乃門龍院澄香|大女ノ門龍院 澄香|聲優：篠原ゆみ}}

;{{nihongo|盡神吉禰|尽神 きつね|聲優：御苑生メイ}}
