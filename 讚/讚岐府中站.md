'''讚岐府中站'''（{{jpn|j=讃岐府中駅|hg=さぬきふちゅうえき|rm=Sanuki-fuchū eki|f=y}}）是一位於[[日本|日本]][[香川縣|香川縣]][[坂出市|坂出市]]府中町，隸屬於[[四國旅客鐵道|四國旅客鐵道]]（JR四國）的[[鐵路車站|鐵路車站]]，車站編號為'''Y05'''。本站現時只停靠普通列車，包括每天上下行各1班各站停車的「南風接駁號」。

由於四國內已有另一個設於[[德島縣|德島縣]]的[[府中站_(德島縣)|府中站]]，為免名稱上的混淆，本車站遂加入舊[[令制國|令制國]]「[[讚岐國|讚岐]]」以茲識別。

本站與[[香西站|香西站]]、[[八十場站|八十場站]]均於1952年設立，設站初期車站規模非常細小，其中本站的月台長度只能容納2輛列車，因此絕大部份的列車都會通過而不停靠，使它們的地位和「[[臨時站|臨時站]]」大致相近。1968年，由於[[國分站_(香川縣)|國分站]]至[[鴨川站|鴨川站]]一段雙軌化，車站由原址向北遷移，新站設於[[堤壩|堤壩]]之上，有效長度仍為2輛，每天只有上下行各1班列車停靠。<ref>[http://df50shikoku.web.fc2.com/ekisha/yosan/kamamowa.html 1986年時的車站月台。]</ref>1986年起班次增加至每天7班。<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/yosan/ST-fuchu.htm |title=存档副本 |accessdate=2013-02-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20150924101516/http://www.shikoku.org.uk/JR/LINE/yosan/ST-fuchu.htm |archivedate=2015-09-24 }} 資料來自非官方車站介紹。</ref>翌年相關路段電氣化後，車站設備才得到提昇，並且將月台的有效長度增至4輛，因此才可以升服務至停靠所有普通列車。<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/yosan/ST-KOUZAI.htm |title=存档副本 |accessdate=2013-02-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20130624095305/http://www.shikoku.org.uk/JR/LINE/yosan/ST-KOUZAI.htm |archivedate=2013-06-24 }} 非官方車站介紹。</ref>

==車站構造==
本站一直以來均以簡易車站模式營運，從未有興建過站舍，車站出入口設於路面，在基堤下另外開設了一個小隧道口，能讓[[汽車|汽車]]和行人通過。<ref>[http://www.kukanzeneki.net/all/taka-tado/6.html] 另一個非官方車站介紹。</ref>
自動售票機則設於往高松方向的出入口。<ref>[http://sta.nuttari.net/photo/1172/006/11720061.jpg] 圖片取自另一個非官方車站介紹。</ref>

設有[[側式月台|側式月台]]2個，提供2線2面的配置。月台上並沒有標示月台編號，而是以上、下行的方式來標示，但在售票機上則有顯示；往坂出方向則為1號月台；而往高松方向的月台是2號月台。<ref>[http://sta.nuttari.net/photo/1172/006/11720060.jpg] 取自非官方車站介紹，圖片攝於2010年。</ref>路軌雖與[[瀨戶大橋線|瀨戶大橋線]]的列車共用，但由於所有瀨戶大橋線列車均不停靠，因此月台的上、下行均以予讚線的方向來命名。

月台全以鋼板及角鐵所築搭而成，有效長度為4輛。兩邊月台均有各自的出入口，位於月台中間位置，由於位處較高，因此月台和地面只能以樓梯連接，並非無障礙通道。<ref>[http://ku-gyou.net/JR-S/s-fuchu.html] 另一個非官方車站介紹。</ref>又因車站建於彎位上，因此列車停靠時，列車和月台間的空隙相對較闊，上落時要留意。

==歷史==
* 1952年1月27日：開業。
* 1968年9月29日：因應國分站－鴨川站間的路段雙軌化，車站被遷移至基堤上。
* 1987年4月1日：[[日本國鐵分割民營化|日本國鐵分割民營化]]，車站的營運由JR四國繼承。
