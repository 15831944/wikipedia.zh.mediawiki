'''古印度文明時期'''，也称'''哈拉帕文明（harappa）'''时期，因起源於[[印度河|印度河]]和[[薩拉斯瓦蒂河|薩拉斯瓦蒂河]]流域，也叫'''印薩兩河文明'''，是指[[世界四大文明|世界四大文明]]之一的[[古印度|古印度]]，約公元前2300年至前1300年之時期。

== 印度河流域文明 ==
'''印度河流域文明'''发生晚於[[两河流域文明|两河流域文明]]、[[尼罗河流域文明|尼罗河流域文明]]，但早于[[商朝|商朝]]。[[考古|考古]]专家在[[印度河|印度河]]、[[薩拉斯瓦蒂河|薩拉斯瓦蒂河]]流域发现[[摩亨佐-達羅|摩亨約达罗]]和[[哈拉帕_(巴基斯坦)|哈拉帕]]两个古代城市遗址，发现了大量[[石器|石器]]、[[青铜器|青铜器]]、[[印章|印章]]和[[农作物|农作物]]遗迹，估计城市人口都在4萬以上。

兩個[[城市|城市]]的中心都有一個人工堆成的土墩，作用未知，考古学家猜测这可能是城市的中心。城市建築以衛城為中心呈網格狀分佈，有市政建築、市場、作坊、儲存區、民居和神廟。每座民居都圍著一個院子建成，有幾個房間、一間廁所和一口水井。建築用基本材料是從燒木頭的窯裡製出的土磚。在摩亨佐-達羅的衛城上建造了一個大浴池，有私人浴池、會所等。

[[印度河|印度河]]流域的農民種植[[大麥|大麥]]、[[小麥|小麥]]、[[棉花|棉花]]、瓜和[[椰棗|椰棗]]。他們還馴養[[大象|大象]]和[[水牛|水牛]]在田裡幹活。這一地區有許多手藝精湛的製陶人，他們用陶輪製作陶器，這在當時是一項嶄新的技術。哈拉帕人使用石器，並用青銅製作刀、武器、碗和雕像。他們建立了發達的廢物處理系統，包括有蓋板的排水系統和倒垃圾的斜槽。

遺址中有許多天秤和砝碼，說明他們有一套度量衡制度好配合其繁榮的商業貿易。而印度河流域文明更證明與古美索不達米亞，阿富汗，緬甸甚至和古中國都有商業往來。

印度河文明持續了800年，尚未有人知道它是如何結束的，但是有幾種說法：河水[[洪水|泛濫]]、[[瘟疫|瘟疫]]、貿易或經濟或國內秩序崩潰等等。

== 文化 ==
{{see|印度河文字}}
在两个古代城市遗址出土大量[[印章|印章]]，由於其文字在歷史上找不到任何與其有關聯的文字系統，並與現代印度梵語毫無關聯，故章上文字至今无人能夠解读，甚至不能确定其究竟是[[文字|文字]]还是图象符号。印章為方形，並刻有各種動物圖案，圖案上方有一種神祕文字。除此外，遺址中的石器，青銅器乃至象牙飾品上皆有其文字出現，並多為簡短的字句。

== 族群 ==
{{commons category|Indus Valley Civilization}}
{{wikivoyage|:en:Mohenjo-daro|摩亨佐-达罗{{en icon}}}}
西方學者認為留下印度河流域文明的居民主要是當地的[[達薩_(梵語)|達薩人]]，他們是[[達羅毗荼人|達羅毗荼人]]。現在這一地區還有母語屬於[[達羅毗荼語系|達羅毗荼語系]]的[[布拉灰人|布拉灰人]]。此古文明的居民與蘇美爾城市的居民做生意，也和印度和中亞部落的人做生意，這一地區與[[蘇美爾|蘇美爾]]的文明有明顯相似之處。
