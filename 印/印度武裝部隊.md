'''印度武裝部隊'''（[[天城文|天城文]]：भारतीय सशस्‍त्र सेनाएं，[[IAST|转写]]：Saśastra Sēnāēṃ）在[[印度共和國|印度共和國]]整體的軍事組織而言，包括印度陸軍、印度海軍、印度空軍，以及各種其他包括印度海岸防衛隊、中央準軍事部隊和核戰略司令部等支援機構。

[[印度共和国军事|印度保持世界第4大武裝力量]]，估計常規部隊總兵力有1,325,000人<ref>http://www.csis.org/media/csis/pubs/060626_asia_balance_powers.pdf</ref> 。印度官方的國防預算達到468億[[美元|美元]]，但實際支出的軍事費用，估計遠遠超過印度官方的數字統計。印度武裝部隊正在迅速擴大和現代化，在未來將有軍事航天計劃<ref>http://www.sfgate.com/cgi-bin/article.cgi?f=/c/a/2008/06/18/MNRR11AJK9.DTL&type=science</ref>，目前則是正在積極開發[[導彈|導彈]]防禦系統<ref>[http://www.iht.com/articles/ap/2009/03/06/asia/AS-India-Missile-Defense.php India successfully tests missile interceptor]</ref>和強大的[[核武|核武]]軍事能力<ref>http://timesofindia.indiatimes.com/India/India_test_fires_submarine-launched_ballistic_missile/rssarticleshow/2817645.cms</ref>。

== 歷史 ==
{{See also|印度军事史}}
印度擁有悠久的軍事歷史，可以追溯到幾千年前。最早紀錄於史册之軍隊是[[吠陀|吠陀]]以及史詩[[羅摩衍那|羅摩衍那]]等。步兵一般是以射箭和武術為主。而與此同時印度也已有海軍記載──公元前1500年在[[梨俱吠陀|梨俱吠陀]]有過海軍遠征之記載。17至18世紀，印度部分小國海軍船隊，已是印度大陸最強大的海軍，曾多次擊敗[[歐洲|歐洲]]艦隊。

19世紀中期印度正式成為[[英國|英國]]殖民地。其時有{{Link-en|英屬印度海軍|HMS Emperor of India}}，直接控制著印度近一個世紀。其時英國對海軍中印度籍軍士有著嚴重之歧視。一個明顯的證據是：1928年才有一個印度人以工程師身份獲授中將軍銜。而在1947年印度獨立之前，一群印度籍水手以抗議軍隊歧視為由，發動兵變，並獲廣泛得到印度全國支持，包括了陸軍和空軍。

[[印度獨立|印度獨立]]後，1950年1月26日成為印度海軍。而陸軍則曾於1947年、1965年、1971年與[[巴基斯坦|巴基斯坦]]打過[[印巴戰爭|三次戰爭]]，也曾於1962年與[[中國|中國]]打過仗。

2012年4月19日成功試射可攜載核彈頭的「[[烈火-5洲際彈道飛彈|烈火5型]]」（Agni V）長程導彈，射程逾5,000公里，聲稱可直擊中國全境。

== 結構 ==
印度武裝力量總部位於[[新德里|新德里]]，印度的[[首都|首都]]。印度[[總統|總統]]擔任武裝部隊的最高統帥；聯合政府為首的印度[[總理|總理]]為武裝部隊所服從，負責執行行政事務。[[國防部|國防部]]則是負責所有的軍事行動，以確保打擊叛亂和國家安全。

== 理念 ==
印度武裝力量有五項主要任務：
* 保衛印度領土的完整。
* 受到外國攻擊則要保衛國家。
* 派遣我方兩棲戰裝備去敵岸作戰。
* 民間遇災要支援（如洪水）。 
* 為國際維和付出行動。

== 军种 ==
=== 陸軍 ===
{{main|印度陸軍}}
[[File:IA_T-90_in_action.jpg|thumb]]的印度[[T-90|T-90]]坦克部隊]]
印度陸軍是现役人数最多的[[陸軍|陸軍]]，完全屬於志願役，印度從未使用[[徵兵制|徵兵制]]過。起初的目標是為了保護國家的邊界，然而近來也對內部提供安全保護，特別是對襲擊[[克什米爾|克什米爾]]和[[阿萨姆邦|東北部]]的武裝恐怖份子。

印度陸軍已參與過許多戰爭，[[克什米爾戰爭|第一次印巴戰爭]]、[[波羅行動|波羅行動]]、[[中印邊境戰爭|中印邊境戰爭]]、[[第二次印巴战争|第二次印巴戰爭]]、[[1971年印巴战争|第三次印巴戰爭]]、[[斯里蘭卡內戰|斯里蘭卡內戰]]、[[卡爾吉爾戰爭|卡爾吉爾戰爭]]等。除此之外，印度也派遣軍隊作為[[聯合國|聯合國]]的維和部隊，於[[塞浦路斯|塞浦路斯]]、[[黎巴嫩|黎巴嫩]]、[[剛果|剛果]]、[[安哥拉|安哥拉]]、[[柬埔寨|柬埔寨]]、[[越南|越南]]、[[納米比亞|納米比亞]]、[[薩爾瓦多|薩爾瓦多]]、[[利比里亞|利比里亞]]、[[莫桑比克|莫桑比克]]和[[索馬利亞|索馬利亞]]等地行動。

=== 海軍 ===
{{main|印度海軍}}
[[File:INS_Viraat_(R22)_Malabar_07.jpg|thumb]]
[[印度海軍|印度海軍]]擁有67000人，其中有5000名[[海軍航空兵|海軍航空兵]]和2000名[[海軍陸戰隊|海軍陸戰隊]]（MARCOS），是世界第五大[[海軍|海軍]]。印度海軍目前擁有超過155艘船艦，包括[[維拉特號航空母艦|維拉特號航空母艦]]。近年印度開始發展海上力量，除了積極向外國購買多艘船艦外，也開發國產潛艇及航空母艦，計畫打造一支[[藍水海軍|藍水海軍]]。

從2008至2013年，印度海軍預計花費約400億美元進行軍事現代化，包括：[[维克拉玛蒂亚号航空母舰|维克拉玛蒂亚号]]、[[维拉特号航空母舰|维拉特号航空母舰]]、[[P-8波塞頓|P-8波塞頓]]反潛機、[[歼敌号|殲敵級潛艇]]、[[什瓦力克級護衛艦|什瓦力克級護衛艦]]、[[加爾各答級驅逐艦|加爾各答級驅逐艦]]、[[鮋魚級潛艇|鮋魚級潛艇]]及改良[[塔瓦爾級護衛艦|塔瓦爾級護衛艦]]。

=== 空軍 ===
[[File:SU-30_MKI_Lajes.JPG|thumb]]]]
{{main|印度空軍}}
正式成立於1932年10月8日作為輔助空軍的印度帝國和皇家前鋒 1945年在二戰期間印度空軍正式進入戰場作戰。印度於1947年獲得獨立後，印度空軍正式為印度聯邦服務，印度成為共和國於1950年前鋒正式離去。自獨立以來，印度空軍同周邊巴基斯坦，並與中國已經參與了四場戰爭。除了衝突，印度空軍一直是積極參與者聯合國 維持和平任務。印度總統擔任印度空軍司令。

=== 海岸警衛隊 ===
{{main|印度海岸警衛隊}}
印度海岸警備隊 （ICG）印度四個武裝部隊的其中之一，其使命是守衛印度共和國的海上利益。印度海岸警衛隊的組織類似於其他的印度武裝部隊。它按海岸警衛隊法創建於1978年8月18日成為一個獨立的單位。它的工作由國防部指揮。

印度海岸警衛隊的任務是保護印度的海上利益，包括它的[[海岸線|海岸線]]，專屬[[經濟區|經濟區]]和航運。它還負責與印度的海上有關海洋的資源，航運，海關和稅務，海洋環境，保護物種以及毒品的執法。

海岸警衛隊與[[印度海軍|印度海軍]] ，印度漁業部，稅務部（[[海關|海關]]）和中央和國家[[警察|警察]]部隊密切合作。印度海岸警衛隊通常是由一名海軍[[中將|中將]]指揮。

== 海外基地 ==
[[File:Yudh_Ahbyas-09_Dhruv-Armour.JPG|缩略图]]
1958年，當時的[[印度總理|印度總理]][[尼赫魯|尼赫魯]]訪問[[不丹|不丹]]，並重申印度支持不丹的獨立，隨後在印度議會宣布任何國家侵略不丹都將被視為侵略印度。

印度也有在[[塔吉克斯坦|塔吉克斯坦]]設立空軍基地。

印度也開始將[[馬爾地夫|馬爾地夫]]納入印度的安全網。

印度可以在對[[巴基斯坦|巴基斯坦]]的戰爭時，使用[[伊朗|伊朗]]的基地。

此外，印度也是三個與[[日本|日本]]簽訂安全條約的國家（另外是[[澳大利亞|澳大利亞]]與[[美國|美國]]）。印度和俄羅斯也有簽訂到到2010年的軍事合作協議。印度和[[緬甸|緬甸]]簽署協議，包括經濟合作、安全與能源協定。印度和[[以色列|以色列]]也建立起軍事和情報交流的外交關係。雖然在冷戰時期印度和以色列呈對立狀態，但隨著[[蘇聯解體|蘇聯解體]]和[[伊斯蘭|伊斯蘭]]恐怖主義的崛起，這兩個國家變成了一個穩固的戰略同盟。
