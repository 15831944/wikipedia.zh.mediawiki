[[印度管理研究所|印度管理研究所]]（IIM）加爾各答分校是1961年11月由印度政府出資針對學士後研究而與[[史隆管理學院|史隆管理學院]]、[[西孟加拉邦|西孟加拉邦]]政府、福特基金會及印度工商界所共同發起創辦的第一所國立商學研究所<ref>{{cite web
|title = About IIM Calcutta
|url = http://www.iimcal.ac.in/campus/default.asp
|accessdate = 2008-07-02
|deadurl = yes
|archiveurl = https://web.archive.org/web/20100219215536/http://www.iimcal.ac.in/campus/default.asp
|archivedate = 2010-02-19
}}</ref>，強項素以電腦系統的支援管理課程而著稱。1975年該校遷至現址；2004年考生錄取率約1%<ref>{{cite web 
|title = 2004 Full Time MBA profile
|author = BusinessWeek
|url = http://www.businessweek.com/bschools/04/full_time_profiles/iimcalcutta.htm
|accessdate = 2008-02-07}}</ref>。
