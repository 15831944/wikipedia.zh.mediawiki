'''卢比'''（{{lang-id|Rupiah}}，简作'''Rp'''）是[[印度尼西亚|印度尼西亚]]的[[法定货币|法定货币]]，[[印尼华人|当地华人]]习称-{'''印尼盾'''}-，由[[印度尼西亚银行|印度尼西亚银行]]发行和控制。印尼卢比的[[ISO_4217|ISO 4217代碼]]為“IDR”。每卢比可分为100[[分_(貨幣)|分]]（sen），但由于[[通货膨胀|通货膨胀]]，印尼盾所有的纸币和硬币都已经不以“分”为单位了。

[[廖内群岛省|廖内群岛]]以及[[新几内亚岛|新几内亚岛]]的印尼部分（[[西伊里安|西伊里安]]）曾分别拥有自己的卢比货币，即{{le|廖内卢比|Riau rupiah}}和{{le|西伊里安卢比|West Irian rupiah}}，但分别在1964年和1971年被纳入了国家卢比。

== 名稱 ==
“卢比”（'''Rupiah'''）一名源自[[印度斯坦语|印度斯坦语]]词语“rupiyaa”（{{lang|ur|روپیہ}}, {{lang|hi|रुपया}}），最源头则是表示“锻打的白银”（wrought silver）的[[梵文|梵文]]词汇“rupya”（{{lang|sa|रूप्य}}）。印尼人通常也用“perak”来代指卢比，这个词在印尼语中表示白银。印尼錢幣上鑄有[[印尼國徽|印尼國徽]]和“Rupiah”字樣。由於印尼被荷蘭殖民期間使用的貨幣稱為「{{en-link|荷屬東印度盾|Netherlands Indies gulden|盾}}」，現時部分華人地區仍使用「印尼-{}-盾」稱呼印尼-{}-盧比。

== 歷史 ==
印尼在1610年至1817年間曾使用荷屬東印度盾作為貨幣，在[[二戰|二戰]]由[[日本|日本]]所佔領的时期開始出現Rupiah，至二戰結束由[[爪哇盧比|爪哇盧比]]（Java Rupiah）取代。

印尼盧比於1949年11月2日首度發行，當時的[[廖內群島|廖內群島]]和[[新幾內亞|新幾內亞]]仍使用非標準的貨幣——[[廖內盧比|廖內盧比]]（Riau rupiah）與[[西伊里安卢比|西伊里安卢比]]（West Irian rupiah），它們分別於1964年和1971年轉用印尼盧比。至1965年12月13日為應付[[通貨膨脹|通貨膨脹]]，當局發行新的印尼盧比，每新盧比兌1000舊盧比。

在1997年至1998年的[[亞洲金融風暴|亞洲金融風暴]]，印尼盧比貶值了35%，成為了[[蘇哈托|蘇哈托]]被推翻的主要因素之一。當時每一[[美元|美元]]相等於2000至3000盧比，至1998年6月更跌至每美元對16800盧比。

印尼盧比的[[輔幣|輔幣]]分別有Rp5、Rp25、Rp50、Rp100、Rp500等幾種面額。而[[紙幣|紙幣]]則有Rp100、Rp500、Rp1,000、Rp5,000、Rp10,000、Rp20,000、Rp50,000、Rp100,000等面額。
