'''印春荣'''（{{bd|1964年|7月||}}）[[云南|云南]][[昌宁|昌宁]]人，[[武警|武警]]上校，[[公安边防部队|公安边防部队]]缉毒英雄<ref name=shiji>{{Cite web |url = http://military.cnr.cn/gz/20170613/t20170613_523797996_6.html  |title = “八一勋章”颁授对象初步候选人简要事迹|author =  |date =2017-06-13  |publisher =央广军事  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1982年10月印春荣入伍<ref name=shiji/>。长期从事缉毒工作，数百次深入边境一线，数十次面对毒贩的枪口，30余次打入贩毒集团卧底开展侦查。2002年5月，印春荣扮成“马仔三哥”独自同毒贩见面，最终将毒贩巧妙引入包围圈，并赤手空拳夺枪，与战友一同生擒2名毒贩，当场缴获[[冰毒|冰毒]]53公斤，摧毁了一个带有[[黑社会|黑社会]]性质的贩毒集团。2003年11月，印春荣打入国际贩毒集团，辗转中国3省7市，与2名毒贩共同吃住19天，成功获取并发出关键情报，最终打掉了日产20公斤的冰毒加工厂，缴获毒品231.86公斤、毒资695万元人民币<ref name=duju>{{Cite web |url = http://news.xinhuanet.com/mil/2017-07/28/c_129666631.htm |title =印春荣：独具虎胆、出生入死的缉毒英雄  |author =  |date = 2017-07-28 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

1998年至2017年，作为侦办主力人员，印春荣先后破获贩毒案件3234起，抓获犯罪嫌疑人4246名，缴获毒品4.62吨、易制毒化学品487吨、毒资3520余万元人民币，印春荣个人参与缉毒量创下了[[公安边防部队|公安边防部队]]之最<ref name=duju/>。

2014年8月，印春荣提任[[云南省公安边防总队|云南省公安边防总队]]普洱市支队支队长。截至2017年，印春荣先后30多次率中方代表团与[[越南|越南]]、[[老挝|老挝]]、[[缅甸|缅甸]]边防部门举行会晤；组织成立了“智慧边防”[[智库|智库]]小组，在边境一线建成了数字化边境管控体系。2017年，普洱边防辖区治安案件发案率比2014年下降67%，刑事案件发案率下降40%。印春荣经多方协调，在[[云南省|云南省]][[西盟佤族自治县|西盟佤族自治县]][[力所拉祜族乡|力所拉祜族乡]]建立了“依网爱心”传递站，为当地少数民族群众募集到78万元人民币的扶贫物资。印春荣还争取到32万元人民币经费，在西盟佤族自治县、[[江城哈尼族彝族自治县|江城哈尼族彝族自治县]]分别建了爱民茶叶加工厂、黑山羊养殖这两个脱贫项目，使辖区贫困群众家庭收入逐年翻番<ref name=duju/>。

2017年7月，中央军委主席[[习近平|习近平]]签署命令，授予印春荣等10位同志“[[八一勋章_(2017年)|八一勋章]]”<ref>{{Cite web |url = http://news.ifeng.com/a/20170728/51522244_0.shtml |title = 景海鹏、程开甲、王刚等10人被授予“八一勋章” |author =  |date = 2017-07-28 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。2017年7月28日，中央军委颁授“八一勋章”和授予荣誉称号仪式在北京[[八一大楼|八一大楼]]举行，中共中央总书记、国家主席、中央军委主席习近平向印春荣等10位“八一勋章”获得者颁授勋章和证书。这是“八一勋章”首次颁授。印春荣同时获授“独具虎胆、出生入死的缉毒英雄”荣誉称号<ref>{{Cite web |url = http://news.ifeng.com/a/20170728/51524452_0.shtml |title = 习近平向“八一勋章”获得者颁授勋章和证书 |author =  |date =2017-07-28   |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
