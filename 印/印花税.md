{{公共财政}}

'''印花稅'''，是[[稅|稅]]的一種，是对[[契約|契約]]、凭证、书据、[[账簿|账簿]]及权利许可证等[[文件|文件]]征收的[[税种|税种]]。[[纳税人|纳税人]]通过在[[文件|文件]]上加贴印花税票，或者盖章来履行纳税义务。

==各地情況==

==={{ROC-TW}}===
{{Wikisource|印花稅法|中華民國印花稅法}}

印花稅屬憑證稅，而憑證之種類繁多，並非各種憑證均需貼用印花稅票，故稅法採列舉式規定，僅規定應稅之憑證始應繳納印花稅。而繳納方式亦非僅貼用印花稅票一途，亦可使用繳款書逐件繳納或彙總繳納，屬於[[地方稅|地方稅]]由各縣市政府徵收。

==== 稅率 ====
依照《[http://law.moj.gov.tw/LawClass/LawAll.aspx?PCode=G0340091 印花稅法]》第7條：
* 銀錢收據：4‰（千分之四）
* 承攬契據：1‰（千分之一）
* 典賣、讓受及分割不動產契據：1‰（千分之一）
* 買賣動產契據：每件稅額12元《[http://www.tpctax.gov.taipei/ct.asp?xItem=1080478&ctNode=78730&mp=103011 台北市稅捐稽徵處]》

=== {{PRC}} ===
目前，中国大陆的印花税征收，是依据於1988年经[[中华人民共和国国务院|中华人民共和国国务院]]颁布施行的《[[s:中华人民共和国印花税暂行条例|中华人民共和国印花税暂行条例]]》。征税范围包括十三大类，并根据相应内容分为税率和定额征收两种。其中，施行税率征收的，自万分之零点五起至千分之一不等。定额则按件收取，每件五元。由书立人双方自行黏贴。印花税目前为地方税，由各地的地方税务局负责征收。

====定义====
以经济活动中-{}-签立的各种合同、产权转移书据、营业帐簿、权利许可证照等应税凭证文件为对象所课征的税。印花税由纳税人按规定应税的比例和定额自行购买并粘贴印花税票，即完成纳税义务。

证券交易印花税：是从普通印花税中发展而来的，属于行为税，即每一笔股票交易成交時皆徵收；亦屬雙向稅，即對买卖双方同时计征。基本税率为0.4%，基金和债券交易不征收印花税。

==={{HKG}}===
'''香港[[印花稅條例|印花稅條例]]'''是《[[香港法例|香港法例]]》之一，第117章。<ref>[http://www.apdnews.com/news/23950.html 香港《2013年印花税（修订）条例草案》刊宪] {{webarchive|url=https://web.archive.org/web/20130407022247/http://www.apdnews.com/news/23950.html |date=2013-04-07 }}，亚太日报，2013年4月7日</ref><ref>{{cite web |url = https://www.elegislation.gov.hk/hk/cap117!zh-Hant-hk|title=第117章 《印花稅條例》}}</ref>《香港法例》指定一些文件必須向[[香港稅務局|香港稅務局]]交付印花稅，否則文件在[[香港|香港]]不具法律效力，例如在法律訴訟時，法庭不會受理。 而且遲交印花稅款必需事前經批准，否則將被重罰。

香港證券的轉讓印花稅，按每張售賣及購買單據所載的代價收取，款額為價值的0.1%，稅款不足1港元之數，須當作1港元計算繳付。

香港物業從價印花稅是劃一稅率，為15%，按交易金額收取。若獲豁免或另有規定(例如香港永久居民於同一文書內購買不多於一個住宅物業時，名下沒有住宅物業)，才可按從價印花稅(第2標準)(即豁免雙倍印花稅)收取。從價印花稅(第2標準)是累進稅率，分為1.5%至4.25%數級，按交易金額收取。最低印花稅率是100港元，最高印花稅率是4.25%，若果超過2174萬港元的有關交易也是以4.25%課稅。

==== 額外印花稅 ====
額外印花稅（{{lang-en|Special Stamp Duty, SSD}}），政府為壓遏樓市過熱而開始額外其增加稅率及適用期。

任何以個人或公司名義在2010年11月20日或以後取得住宅物業，並在取得後24個月內將其轉售，按不同的物業持有期繳付額外印花稅，6個月內出售，稅率為15%，超過6個月到12個月為10%，超過12個月到24個月為5%。

2012年10月27日，政府提高額外印花稅的稅階5%和把適用期延長至36個月。買入後首半年內出售單位的徵收率增至20％；超過半年而不超過一年15％；超過一年而不超過三年，一概徵收10％。<ref name="政府徵15％買家印花稅">[http://www.news.gov.hk/tc/categories/finance/html/2012/10/20121026_172021.shtml《香港政府新聞網》] {{webarchive|url=https://web.archive.org/web/20121028171333/http://www.news.gov.hk/tc/categories/finance/html/2012/10/20121026_172021.shtml |date=2012-10-28 }}，《香港政府新聞網》，2012年10月26日。</ref>

這項措施將提高住宅樓宇投資者的成本和風險。短線投資者須考慮三年內樓市的升幅能否抵消額外印花稅的成本，也要顧及樓市在三年內進入下跌周期的風險。政府相信這項措施可減低短線投資者的投資意欲，而不會對一般置業自住市民造成太大影響。

====雙倍印花稅====
香港法例上並無'''雙倍印花稅'''（{{lang-en|Double Stamp Duty, DSD}}）的名稱，實際是指從價印花稅在標準稅額與優惠稅額比較下的附加徵稅，於2013年2月引入，適用於購買第二間物業的買家或公司名義買家，200萬港元或以下的交易，印花稅的稅率由100港元增加至交易額的1.5%；其他交易的稅率亦全面增加一倍。<ref>[http://property.mpfinance.com/cfm/pa3.cfm?File=20130223/paa01/gaa1h.txt 雙倍印花稅遏炒樓 針對非首套房買家 港人首置或樓換樓豁免]《明報》13年2月23日</ref>
2016年11月4日起，從價印花稅稅率调升至劃一15%，令「雙倍印花稅」實際上已不止多徵一倍稅額。<ref>http://www.ird.gov.hk/chi/faq/index.htm#avd</ref>

==== 買家印花稅 ====
在2012年政府正式新增'''買家印花稅'''（{{lang-en|'''Buyer Stamp Duty''', '''BSD'''}}），非[[香港永久居民|香港永久居民]]及所有公司購買香港住宅，除了繳付一般印花稅，亦需繳付15%的買家印花稅。當局同時將額外印花稅的稅階提高，並把適用期延長至三年：買入後首半年內出售單位的稅率增至20%；半年至一年為15%；一年至三年為10%。<ref>[http://www.news.gov.hk/tc/categories/finance/html/2012/10/20121026_172021.shtml 政府徵15%買家印花稅] {{webarchive|url=https://web.archive.org/web/20121028171333/http://www.news.gov.hk/tc/categories/finance/html/2012/10/20121026_172021.shtml |date=2012-10-28 }}，[[香港政府新聞網|香港政府新聞網]]，2012年10月26日</ref>

==={{AUS}}===
[[澳大利亚|澳大利亚]]联邦政府不征收印花税。但澳大利亚各州对各类文书（即书面文件）和相关交易征收印花税。各州采用不同的印花税率，文件和交易类型不同，税率也会有所差别。如今，有些地区征收不再需要实际的书面文件来征收印花税，现多称为“交易税”。

印花税主要形式包括对[[土地|土地]]、营业、[[股份|股份]]和其他可征税[[财产|财产]]的转让所征的[[流转税|流转税]]，抵押税，租赁税，出租税。初次购房者可以享受流转税与抵押税的折扣或免除。<ref>{{cite web |url = http://yahoo.infochoice.com.au/banking/calculate/flash/stamp-duty.asp |title = Online calculator for stamp duty on house transfers}}（澳大利亞的房屋移轉印花稅網上試算）</ref>

2005年4月20日，若干州或领土的财政主管宣布，他们要在近五年内逐步取消部分印花税。但对于土地所有权转让所征流转税仍将保留。

==={{USA}}===

以前美国联邦政府对交易文档征收印花税，但现在只有州政府征收这类税，特别是在房地产买卖或转让时。

==影响及评价==
==={{GBR}}===
英国历史上的印花税，曾被讥讽为“知识税”（{{lang-en|Tax on Knowledge}}）。由于种类繁多，对出版界带来巨大的负担，另一方面，政府通过此举，并辅之以津贴制度，一定程度上达到控制报业的作用。从1712年开征后到19世纪中叶逐渐废除（1853年广告税废除，1855年印花税废除，1861年纸张税被废除），历时将近1个世纪中，英国报业一直裹足不前。

英國1765年對北美殖民地開徵印花稅，成為[[美國獨立戰爭|美國獨立戰爭]]的導火線之一。
