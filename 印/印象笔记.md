'''印象笔记'''是帮助个人及团队创建、收集、整理和分享信息的知识管理工具，由[[Evernote|Evernote]]于2012年5月正式推出。2018年6月印象笔记重组落地为中美合资的独立实体。

==发展历程==
2008年6月，Evernote公司正式成立。2012年7月19日，印象笔记宣布与[[支付宝|支付宝]]达成合作。2012年12月1日，印象笔记宣布其印象笔记百宝箱服务正式上线。2013年8月29日，印象笔记推出了Windows Phone平台全新的4.0版本。2015年6月，聂坤瑞（Chris O'Neill）接替菲尔·利宾，正式担任Evernote全球CEO。 2015年11月11日，唐毅正式加入印象笔记，全面负责中国业务 <ref>{{cite web |accessdate= |language=zh |deadurl=no |url=http://tech.sina.com.cn/i/2018-03-23/doc-ifysnevm3170340.shtml |title=印象笔记CEO唐毅：正在建中国技术团队 将推特别功能 |first= |last= |publisher=新浪科技 |date= |archiveurl= |archivedate=}}</ref>。2016年6月，Evernote宣布全球现金流转正<ref>{{cite web |accessdate= |language=zh |deadurl=no |url=http://biz.jrj.com.cn/2017/02/23113923143427.shtml |title=EvernoteCEO称公司已实现正现金流 |first= |last= |publisher=金融界 |date= |archiveurl= |archivedate=}}</ref>。2016年6月，印象笔记与[[新东方学校|新东方]]达成战略合作。

2018年3月，印象笔记服务正式搬迁至腾讯云。2018年6月，印象笔记重组落地为中美合资的独立实体并获得红杉宽带数字跨境基金的首轮数亿元人民币融资<ref>{{cite web |accessdate= |language=zh |deadurl=no |url=http://m.iheima.com/article/168499?from=groupmessage&isappinstalled=0 |title=印象笔记宣布重组为独立中国公司 并获得红杉宽带跨境数字产业基金首轮数亿元投资 |first= |last= |publisher=i黑马 |date= |archiveurl= |archivedate=}}</ref>。

==产品Logo==
印象笔记的Logo是一个大象的标志。在英语中有一个说法，“An elephant never forgets.”（大象永远不会遗忘）。根据这个典故，使用大象的形象作为记忆的标志。而大象的“折耳”让人联想到文档图标或是一本书的折叠页。
