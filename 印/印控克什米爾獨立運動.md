'''印控克什米爾獨立運動'''又稱'''查謨-克什米爾獨立運動'''（英語：Insurgency in Jammu and Kashmir，直譯為[[查谟-克什米尔邦|查謨-克什米爾]]的叛亂）或'''喀什米爾起義'''（英語：Kashmir Intifada），是[[喀什米爾|喀什米爾]][[分離主義|分離主義]]者在[[巴基斯坦|巴基斯坦]]的支持下<ref>{{cite news |url = https://global.udn.com/global_vision/story/8663/2235225 |title = 恐怖的曖昧：巴基斯坦，擁恐還是反恐？ |publisher = 聯合新聞網 |date = 2017/01/17 |accessdate = 2018/01/28 }}</ref>，和[[印度政府|印度政府]]之間的衝突，部分分離主義者支持喀什米爾獨立，部分分離主義者則支持加入[[巴基斯坦|巴基斯坦]]。

== 傷亡 ==
據統計，喀什米爾在1988年至2018年間，一共發生47,234起的暴力恐怖事件，累計死亡人數高達44,516人，其中包括14,796名平民，6,361名安全部隊人員，23,356名參與暴力恐怖事件者（截至2018年1月28日的數據）<ref>{{Cite web |url = http://www.satp.org/satporgtp/countries/india/states/jandk/data_sheets/annual_casualties.htm |title = 暴力恐怖事件的死亡人數1988-2018年 |accessdate = 2018/02/01 }}</ref>。

儘管印巴兩國近年來已無大規模的軍事衝突，但喀什米爾每年仍有超過100人死於暴力衝突（2016年有267人死亡，2017年有358人死亡，2018年截至1月28日有16人死亡）。
== 雙方對人權的侵犯 ==
[[印度政府|印度政府]]和[[喀什米爾|喀什米爾]][[分離主義|分離主義]]者，皆被指控犯下[[屠殺|屠殺]]、[[虐待|虐待]]等侵犯人權的[[戰爭罪行|戰爭罪行]]。另外，支持喀什米爾分離主義者的[[巴基斯坦|巴基斯坦]]，也被印度政府指控越界襲擊印度軍人<ref>{{cite news |url =  http://m.ltn.com.tw/news/world/breakingnews/1896136 |title = 衝突再起！ 印度指控巴基斯坦殺害肢解士兵 |publisher = 自由時報電子報 |date = 2016/11/23 |accessdate = 2018/02/01 }}</ref>，甚至殺害喀什米爾平民<ref>{{cite news |url = http://m.huffingtonpost.in/2015/02/24/pakistan-ceasefire_n_6741672.html |title = 巴基斯坦在查謨和克什米爾地區違反停火8個月內685次 |publisher = 哈芬登郵報 |date = 2015/02/24 |accessdate = 2018/02/01 }}</ref>。
=== 印度政府對人權的侵犯 ===
喀什米爾民間團體主張喀什米爾衝突的死亡人數為7萬人（主流說法是4萬多人），衝突中不幸遇難的平民，大多數是被[[印度軍隊|印度軍隊]]殺害<ref>{{Cite web |url = https://kashmirobserver.net/2016/local-news/karadzic-prosecute-all-accused-hr-violations-kashmir-jkccs-4777 |title = 像卡拉季奇一樣，起訴克什米爾境內的所有人權違法行為：JKCCS  |accessdate = 2018/02/01 }}</ref>，2017年喀什米爾甚至發生警察對手無寸鐵的抗議女學生開槍的事件。<ref>{{cite news |url = http://m.ltn.com.tw/news/world/breakingnews/2053415 |title = 印控喀什米爾喊獨立！女學生遭武裝部隊爆頭倒地 |publisher = 自由時報電子報 |date = 2017/05/01 |accessdate = 2018/02/01 }}</ref>
