'''印第安纳波利斯'''（{{lang-en|Indianapolis}}，{{pron-en|ˌɪndiəˈnæpəlɨs}}），简称 Indy ({{IPA-en|ˈɪndi|}})， 是美国[[印地安纳州|印地安纳州]]首府，行政上印第安纳波利斯市与其所在的[[马里昂县|马里昂县]]合一。根据2010年美国人口统计，该市人口为829,718人。它也是该州最大城市、美国第十二大城市，在[[美国中西部|中西部]]面积仅次于[[芝加哥|芝加哥]]。印第安纳波利斯也是美国发展最快的地区之一<ref name=2010forbes>{{cite web|title=Fastest Growing Cities in the U.S.|url=http://www.forbes.com/2010/10/11/cities-innovation-texas-great-plains-indianapolis-opinions-columnists-joel-kotkin_slide_11.html|accessdate=14 March 2011|archiveurl=https://archive.is/20121208154131/http://www.forbes.com/2010/10/11/cities-innovation-texas-great-plains-indianapolis-opinions-columnists-joel-kotkin_slide_11.html|archivedate=2012年12月8日|deadurl=yes}}</ref>。

在印第安纳州历史上，印第安纳波利斯因为其政府以及工业（特别是制造业）而著名。现在，印第安纳波利斯更多着重于该地区的教育、健康以及财政。旅游业也是印第安纳波利斯的主要支柱产业之一，其城市还承办一些著名的娱乐与运动活动，包括每年度的[[印第安納波利斯500|印第安納波利斯500]]等<ref name="PopGrowth">{{cite web|url = http://www.census.gov/population/www/estimates/Estimates%20pages_final.html|title = U.S. Census Figures|publisher = [[United_States_Census|United States Census]]| year = 2006|accessdate = 2008-01-16}}</ref>。

== 歷史 ==
在歷史上此地居民主要是土著[[印第安人|印第安人]]，後來成為美國的一部分。1820年，此城被選為印地安納州首府，並起名做Indianapolis。在十九世紀，此城已是主要地區交通樞紐，連接[[芝加哥|芝加哥]]、[[辛辛那提|辛辛那提]]、[[底特律|底特律]]、[[克里夫蘭|克里夫蘭]]等地。

隨著城市人口的增長，二十世紀起開始向市郊擴張。

=== 气候 ===
印第安纳波利斯气候代表美国中西部温带，属[[溫帶大陸性濕潤氣候|溫帶大陸性濕潤氣候]]，四季分明。冬季寒冷，潮湿，日照少，日最高气温低于{{convert|0|C}}的平均日数为30天，日最低气温低于{{convert|-10|C}}的平均日数为22天，低于{{convert|-20|C}}的有2.9天；夏季炎热潮湿，日最高气温超过{{convert|30|C}}的日数年均有47天，{{convert|35|C}}以上的有2.6天。<ref name = "NWS Indianapolis, IN"/>最冷月（1月）均温{{convert|28.1|F|1|disp=flip}}，极端最低气温{{convert|−27|F|0|disp=flip}}（1994年1月19日）。<ref name = "NWS Indianapolis, IN"/>最热月（7月）均温{{convert|75.4|F|1|disp=flip}}，极端最高气温{{convert|106|F|0|disp=flip}}（上次出现于1936年7月14日）。<ref name = "NWS Indianapolis, IN"/>无霜期平均为184天（4月17日至10月17日）；可测量降雪平均期为11月23日至3月19日。<ref name = "NWS Indianapolis, IN"/>年均降水量约{{convert|42.4|in|sigfig=3|disp=flip}}，年极端最少降水量为{{convert|24.97|in|disp=flip}}（1934年），最多为{{convert|57.65|in|disp=flip}}（1876年）。<ref name = "NWS Indianapolis, IN"/>年均降雪量为{{convert|25.9|in|cm|disp=flip}}；1918–19年的降雪量最少，积累降雪量只有{{convert|5.5|in|cm|disp=flip}}，1981–82年的降雪量最多，积累降雪量为{{convert|58.2|in|cm|disp=flip}}。<ref name = "NWS Indianapolis, IN"/>

== 经济 ==
19世纪末[[工业|工业]]兴起，城市迅速发展，成为州内主要工业中心。如今，工业佔的比重大為降低，而[[教育业|教育业]]、[[医疗保健|医疗保健]]、[[旅游业|旅游业]]、[[金融业|金融业]]共同构成了该市的经济体系。

*印第安納大學醫療：11,810名僱員 
*[[禮來公司|禮來公司]]：10,735名僱員 
*市公立醫療部：10,402名僱員 
*[[沃爾瑪|沃爾瑪]]：8,830名僱員 
*Marsh超級市場：8,000名僱員
*Kroger：7,840名僱員
