'''衣摺加美北站'''（{{jpn|j=衣摺加美北駅|hg=きずりかみきたえき|rm=Kizuri-Kamikita eki|f=y}}）是位於日本[[大阪府|大阪府]][[東大阪市|東大阪市]]衣摺五丁目<ref name="jrw201709"/>之[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）[[大阪東線|大阪東線]]的[[鐵路車站|鐵路車站]]，[[車站編號|車站編號]]是JR-F13。該站名源自東大阪市衣摺和[[大阪市|大阪市]][[平野區|平野區]]加美北<ref name="jrw201709"/><ref>{{Cite news|url=http://www.sankei.com/west/news/170926/wst1709260095-n1.html|title=難読？おおさか東線、新駅の名称は「衣摺加美北」　来年春、ＪＲ長瀬－新加美間で開業|date=2017-9-26|accessdate=2017-9-26|newspaper=[[產經新聞|產經WEST]]}}</ref><ref>{{Cite news|url=https://mainichi.jp/articles/20170927/k00/00m/040/050000c|title=おおさか東線 新駅名は衣摺加美北　長瀬－新加美間に開設 |date=2017-9-26|accessdate=2017-11-6|newspaper=[[每日新聞|每日新聞]]}}</ref>。

== 概要 ==
== 歷史 ==
* 2008年3月15日 - [[大阪東線|大阪東線]]的[[放出站|放出站]]至[[久寶寺站|久寶寺站]]區間開業。此時進行新站設置的準備工作。
* 2013年
** 4月19日 - 新站設置獲得近畿運輸局長同意申請<ref>{{Cite web|url=http://www.westjr.co.jp/press/article/2013/04/page_3707.html|title=おおさか東線　JR長瀬～新加美駅間新駅設置について|date=2013-04-19|accessdate=2017-11-06|publisher=[[西日本旅客鐵道|西日本旅客鐵道]]}}</ref>。
** [[7月|7月]] - 新站設置獲得近畿運輸局長同意<ref>{{Cite web|url=http://www.osr.co.jp/business/outline.html|title=おおさか東線の概要と経緯|accessdate=2017-10-12|publisher=大阪外環状鉄道}}</ref>。
* 2015年9月 - 車站工程開始進行<ref>{{Cite web|url=http://www.city.higashiosaka.lg.jp/0000015744.html|title=JRおおさか東線 新駅工事着手|date=2015-10-14|accessdate=2017-10-12|publisher=[[東大阪市|東大阪市]]}}</ref>。建設時暫定站名為「'''衣摺站'''」<ref name="higashiosaka">{{Cite web|url=http://www.city.higashiosaka.lg.jp/0000011663.html|title=衣摺駅前交通広場|date=2017-05-24|accessdate=2017-10-12|publisher=[[東大阪市|東大阪市]]}}</ref><ref>{{Cite news|url=https://higashiosaka.keizai.biz/headline/307/|title=おおさか東線のJR長瀬－新加美駅間に新駅　JR西日本が概要発表 |date=2016-08-05|accessdate=2017-11-06|newspaper=東大阪経済新聞}}</ref>。
* 2016年8月5日 - JR西日本發表新站概要<ref>{{Cite web|url=http://www.westjr.co.jp/press/article/2016/08/page_9053.html|title=おおさか東線 JR長瀬～新加美駅間、新駅の概要について|date=2016-08-05|accessdate=2017-11-06|publisher=[[西日本旅客鐵道|西日本旅客鐵道]]}}</ref>。
* 2017年9月26日 - JR西日本正式決定站名為「'''衣摺加美北站'''」。名稱來自車站橫跨東大阪市的「衣摺」地區與大阪市平野區的「加美北」地區<ref name="jrw201709">{{Cite web|url=http://www.westjr.co.jp/press/article/2017/09/page_11201.html|title=おおさか東線全線開業に向けて～JR長瀬・新加美駅間新駅の駅名の決定について～|date=2017-09-26|accessdate=2017-09-26|publisher=[[西日本旅客鐵道|西日本旅客鐵道]]}}</ref>。
* 2018年3月17日 - 於大阪東線的[[JR長瀨站|JR長瀨站]]至[[新加美站|新加美站]]之間開業<ref>{{Cite press release |title=平成30年春ダイヤ改正について |url=https://www.westjr.co.jp/press/article/items/20171215_00_jrgroup.pdf |publisher=JRグループ |format=PDF |date=2017-12-15 |accessdate=2017-12-15}}</ref><ref>{{Cite press release |title=2018年春ダイヤ改正について |url=https://www.westjr.co.jp/press/article/2017/12/page_11628.html |publisher=西日本旅客鉄道 |date=2017-12-15 |accessdate=2018-01-12}}</ref><ref>{{Cite press release |title=2018年3月17日にダイヤ改正を実施します |url=https://www.westjr.co.jp/press/article/items/171215_00_keihanshin_1.pdf |publisher=西日本旅客鉄道 |format=PDF |date=2017-12-15 |accessdate=2018-01-12}}</ref>。開業時導入[[車站編號|車站編號]]。
* [[2019年|2019年]][[3月16日|3月16日]] - 隨大阪東線延伸至新大阪，編入成為[[特定都区市内|大阪市内]]站<ref name="jrw_20181227">{{Cite press release |title= おおさか東線開業に伴う運賃に関わる認可申請などについて |url= https://www.westjr.co.jp/press/article/2018/12/page_13598.html |publisher=西日本旅客鐵道 |date=2018-12-27 |accessdate=2019-1-12}}</ref>。

== 車站結構 ==
車站設置2面2線[[側式月台|側式月台]]，為[[高架車站|高架車站]]，8節車廂編成對應<ref name="jrw201709"/>。各月台配有[[電梯|電梯]]、[[手扶梯|手扶梯]]。車站的設計理念是「'''紡織製造業歷史的城鎮'''」<ref name="jrw201709"/>。

車站西側的東大阪市整備作為災害發生的避難場所之站前交通廣場<ref name="higashiosaka"/><ref>{{Cite news |title=おおさか東線、営業中の南区間にも新駅 |newspaper=日経コンストラクション |date=2016-12-22 |url=http://kenplatz.nikkeibp.co.jp/atcl/cntcolumn/15/00011/00002/ |accessdate=2017-10-11}}</ref>。
