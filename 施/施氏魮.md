'''施氏魮'''（[[学名|学名]]：'''''Barbonymus schwanenfeldii'''''），又名[[施瓦氏四鬚魮|施瓦氏四鬚魮]]、'''施瓦氏擬鲃'''、'''施瓦氏四鬚鲃'''、'''泰國鯽'''、'''紅鰭鯽'''，為[[輻鰭魚綱|輻鰭魚綱]][[鯉形目|鯉形目]][[鯉科|鯉科]]的其中一個[[種|種]]。

== 分布 ==
本魚分布於[[泰國|泰國]]、[[越南|越南]]、[[新加坡|新加坡]]、[[印尼|印尼]]、[[馬來西亞|馬來西亞]]、[[寮國|寮國]]、[[柬埔寨|柬埔寨]]的溪流。被引進[[台灣|台灣]]、[[美國|美國]]。

== 特徵 ==
本魚未成年的魚，背部有灰褐色向下至腹側逐漸變成閃亮的銀色，鱗片錯落有致，三角形的背鰭上有一面凹陷。成魚的背鰭為黑色，背鰭基部為紅色，腹鰭和臀鰭為鮮豔的橘紅色。色彩較為單調的尾鰭，上下緣為黑色。體長可達35公分。

== 生態 ==
本魚棲息在淡水溪流、湖泊中，性情溫和，喜群游，屬雜食性。

== 經濟利用 ==
為觀賞性魚類。
