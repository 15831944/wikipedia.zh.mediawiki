'''施耐庵文学奖'''，全称为'''施耐庵长篇叙事文学奖'''，是[[江苏省|江苏省]][[兴化市|兴化市]][[人民政府|人民政府]]主办的文學獎，2011年起頒發，以《[[水浒传|水浒传]]》作者[[施耐庵|施耐庵]]（據說是興化人）命名。<ref name="xljh " >{{cite web |url= http://book.sina.com.cn/news/c/2011-06-02/1353287276.shtml|title=施耐庵文学奖评奖办法 |accessdate= 2011年6月2日|work=|publisher=新浪网 |language=zh-hans}}</ref>该奖旨在鼓励当代汉语长篇叙事艺术的深度探索与发展，推动汉语长篇叙事的创新与繁荣，进一步提升汉语长篇叙事作品的世界地位。原訂每两年评选一次；但第二屆之後停擺了五年，直到2018年5月才宣布启动第三届评选，<ref name="third" >{{cite web|url=http://www.chinawriter.com.cn/n1/2018/0506/c403994-29966967.html|title=第三届施耐庵文学奖启动：打通虚构与非虚构的文体界限--新闻--中国作家网|publisher=中国作家网}}</ref>10月頒發第三屆獲獎作品。<ref>[http://www.chinawriter.com.cn/n1/2018/1019/c403994-30351857.html 第三届“施耐庵文学奖”获奖名单揭晓 宗璞《北归记》等五部作品获奖]</ref>

== 奖项项目 ==
前兩屆的規則是：每两年评选一次，逢单年评奖并颁奖，评奖范围涵盖长篇小说、散文、传记、报告文学、诗歌等长篇叙事文体。每届评出四部作品，其中海外作品一部；另设特别奖，奖励一至二个[[兴化市|兴化]]籍贯作者的奖品，奖金特设。<ref name="xljh " />2018年重啟後，获奖作品由每届4部增加为5部，兼顾海外作品和非虚构作品；特别奖由2部缩减为1部。<ref name="third" />

== 评选标准 ==
*具有[[汉语|汉语]]艺术韵味和[[中华文化|中华文化]]<ref name="xljh " />
*突出人的精神状态和生存状态<ref name="xljh " />
*汉语叙事方式的拓新和改进<ref name="xljh " />

== 设立宗旨 ==
施耐庵文学奖主旨一共为三个：鼓励当代汉语长篇叙事艺术的深度探索与发展，推动汉语长篇叙事的创新与繁荣，进一步提升汉语长篇叙事作品的世界地位。<ref name="xljh " />
