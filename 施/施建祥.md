'''施建祥'''（{{bd|1964年|9月27日|}}{{efn|name=ab1|来源于《中央反腐败协调小组国际追逃追赃工作办公室关于部分外逃人员有关线索的公告<ref name="a1"/>》（31.施建祥）中的信息。}}{{efn|另一个生日是9月20日。有报道提及，2015年9月20日晚，施建祥在上海虹桥宾馆庆祝他与一双儿女的共同生日<ref>{{Cite web |url=http://finance.jrj.com.cn/2018/06/06195024647054.shtml |title=“搅局者”崔永元——揭开被遗忘的“快鹿资本局”|accessdate=2018-10-13 |author=缪凌云 |date=2018-06-06 19:50:15 |work= |publisher=金融网站，来源：野马财经|language=Zh-hans }}</ref>。}}），中华人民共和国企业家，因涉嫌[[集资诈骗罪|集资诈骗罪]]被中国政府通辑。

原[[中华人民共和国户籍制度|户籍]]所在地为上海{{efn|name=ab1}}，持[[香港居民身份证|香港居民身份证]]{{efn|name=ab1}}。生于[[崇明县|崇明县]]的[[长兴岛_(上海)|长兴岛]]，有兄弟施建兴、施建忠。1995年，从印刷厂辞职下海经商。通过成为美国[[西方石油公司|西方石油公司]]在[[华东地区|华东地区]]的总代理，获得第一桶金<ref name="a2">{{Cite web |url=http://news.ifeng.com/a/20160412/48426862_0.shtml |title=快鹿员工谈掌门人施建祥：开会花200多万 辞职是好事 |accessdate=2018-10-13 |author=|date=2016-04-12 |work=胡秋实 |publisher=凤凰网，来源：时代周报|language=Zh-hans }}</ref>。原[[快鹿集团|快鹿集团]]和[[十方控股|十方控股]]董事局主席。2010年代，以投资多部电影、担任电影制片人知名。2016年，快鹿集团旗下的[[P2P借贷|P2P借贷]]网络平台出现兑付危机，事涉[[非法集资|非法集资]]。4月1日，宣布辞去十方控股董事局主席、[[执行董事|执行董事]]职位。4月5日，宣布辞去快鹿集团董事局主席职位<ref name="a2"/>。根据中国政府2018年6月发布的公告<ref name="a1">{{Cite web |url=http://www.ccdi.gov.cn/toutiao/201806/t20180606_173306.html |title=中央反腐败协调小组国际追逃追赃工作办公室关于部分外逃人员有关线索的公告|accessdate=2018-10-13 |author=|date=2018-06-06 |work= |publisher=中纪委、国家监委网站|language=Zh-hans }}</ref>，他已于2016年3月7日外逃，目前所在地为美国。
