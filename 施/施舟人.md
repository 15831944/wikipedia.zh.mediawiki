'''施舟人'''{{noteTag|1=[[道家|道家]][[老子|老子]]是周朝人，而[[庄子|庄子]]名“周”，都有个“周”字，故取其谐音“舟人”，同时包含有作为东西方文化的桥梁“渡人”的意思。}} （{{lang-en|'''Kristofer Marinus Schipper'''}}，{{bd|1934年|10月23日|||catIdx= S施舟人}}）出生于[[瑞典|瑞典]]，[[荷兰|荷兰]]血统，與[[顾彬|顾彬]]、[[施寒微|施寒微]]並稱“[[欧洲三大汉学家|欧洲三大汉学家]]”，他精通八种语言，目前在[[福州大学|福州大学]]担任教授，并且任职[[福州大学世界文明研究中心|福州大学世界文明研究中心]]主任。<ref name="cow " >{{cite web |url=http://china.huanqiu.com/roll/2009-12/656816.html |title=施舟人：无法割舍的中国情缘 |accessdate=2009-12-10 |publisher= 环球时报|work= |language =zh-cn}}</ref>

== 生平 ==
1934年，施舟人出生在[[瑞典|瑞典]]，在[[法国|法国]][[巴黎大学|巴黎大学]]研读[[中文|中文]]、[[日文|日文]]、远东美术史和宗教人类学，师从法国[[汉学家|汉学家]][[康德谟|康德谟]]（Maxime Kaltenmark），专攻中国历史文化。<ref name="cow " />

1962年，施舟人到[[台湾|台湾]][[台南|台南]]学习，并且在台湾出家做了[[道士|道士]]。<ref name="cow " />

1972年，施舟人在[[法国高等研究院|法国高等研究院]]讲中国宗教史。<ref name="cow " />

1976年，施舟人在法国巴黎创办了[[欧洲汉学协会|欧洲汉学协会]]。<ref name="cow " />

1979年，施舟人远赴中国，通过和[[法国科技中心|法国科技中心]]（CNRS）、荷兰[[莱顿大学|莱顿大学]]、[[北京大学|北京大学]]、[[中国社科院|中国社科院]]、[[北京社科院|北京社科院]]的沟通和联系，创建了汉学专项《圣城北京》。<ref name="cow " />

2001年，施舟人定居中国[[福州大学|福州大学]]，并且创建了[[福州大学世界文明研究中心|福州大学世界文明研究中心]]，即俗称的“西观藏书楼”。<ref name="cow " />

== 履历 ==

*1992年，荷兰莱顿大学汉学特邀担讲座教授
*荷兰皇家科学院院士

*2001年，中国福州大学特聘教授

== 思想 ==
施舟人首次提出了建立“文化基因库”，并且历时25年整理了中国《[[道藏|道藏]]》文献。<ref>{{cite web |url=http://news.sohu.com/20070814/n251577514.shtml |title= 施舟人 扎根中国的中西方文化信使|accessdate= 2007年8月14日|publisher=搜狐网 |work= |language =zh-cn}}</ref>
施舟人是欧洲汉学第四代传人，第一代是[[沙畹|沙畹]]，第二代是[[马伯乐|马伯乐]]，第三代是[[康德谟|康德谟]]。施舟人以[[道教|道教]]的研究著称于世。<ref name=" tiger">{{cite web|url=http://www.fj.xinhuanet.com/news/2004-06/26/content_2388543.htm|title=著名汉学家施舟人先生与福州大学缘深情重|accessdate=2004-06-26|publisher=新华网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20041214112525/http://www.fj.xinhuanet.com/news/2004-06/26/content_2388543.htm|archivedate=2004-12-14}}</ref>

== 个人生活 ==
施舟人的妻子是[[袁冰凌|袁冰凌]]，两人1991年在莱顿大学认识，袁冰凌现在担任福州大学人文社会学院副院长，毕业于荷兰莱顿大学，2000年，两人在法国巴黎结婚，育有女儿施心韵。<ref>{{cite web |url=http://news.qq.com/a/20050715/001917.htm |title= 跨越东西方文化　施舟人扎根榕城做中国事|accessdate=2005年7月15日 |publisher= 腾讯网|work= |language =zh-cn}}</ref>
