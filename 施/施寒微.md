'''施寒微'''（{{lang-en|'''Helwig Schmidt-Glintzer'''}}，{{bd|1948年|6月24日|||catIdx= S施寒微}}）出生于[[德国|德国]][[巴特黑斯费尔德|巴特黑斯费尔德]]，毕业于[[哥廷根大学|哥廷根大学]]和[[慕尼黑大学|慕尼黑大学]]，为德国著名的汉学家，與[[施舟人|施舟人]]、[[顧彬|顧彬]]並稱“[[欧洲三大汉学家|欧洲三大汉学家]]”。

== 生平 ==
1948年，施寒微出生在[[德国|德国]][[赫斯费尔德-罗滕堡县|赫斯费尔德-罗滕堡县]]巴特赫尔斯费尔德，大学在[[哥廷根大学|哥廷根大学]]和[[慕尼黑大学|慕尼黑大学]]度过。

1967年，施寒微原本打算学习[[日文|日文]]，但是附近附近学校只有[[中文|中文]]，就放弃了学习日文，转学中文。<ref>{{cite web |url=http://www.chinanews.com/edu/hjhy/news/2007/04-06/909826.shtml |title=国外汉学家：一人一段中国缘 看不惯中文夹洋文 |accessdate=2007年4月6日 |publisher=中国新闻网 |work= |language =zh-cn}}</ref>

1981年，施寒微出任[[路德维希·马克西米利安大学|路德维希·马克西米利安大学]]教授。

1993年，施寒微出任哥廷根大学教授，后来出任[[德国汉学协会|德国汉学协会]]主席。

== 作品 ==
*《中国古今文学通史》
*《中国简史》

== 思想 ==
施寒微早年师从著名汉学家、蒙古史专家[[傅海波|傅海波]]（Herbert Franke），主要钻研中国古代文学和[[佛学|佛学]]。<ref>{{cite web|url=http://iei.bfsu.edu.cn/news/news-198.htm|title=德国著名汉学家施寒微在杜塞尔多夫孔子学院|accessdate=|publisher=中国语言文学学院|work=|language=zh-cn}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
