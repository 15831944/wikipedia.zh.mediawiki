'''酬恤金進軍事件'''或'''補償金事件'''、'''補助金進軍事件'''或（{{lang-en|Bonus Army}}）指1932年[[美國|美國]]一起示威事件，背景為二万名左右的[[第一次世界大戰|第一次世界大戰]][[美軍|美軍]][[退伍軍人|退伍軍人]]要求即時發放戰時服役的[[薪水|薪餉]]，遭到[[美國政府|美國政府]]拒絕而發動人群[[集會|集會]]逗留[[請願|請願]]，最後導致軍方派兵介入而造成流血衝突。

== 起因 ==
在[[第一次世界大戰|第一次世界大戰]]（一戰）中，參戰的美國軍人在每人每日1[[美元|美元]]的薪金之外，還另加25[[1美分硬幣|美分]]做為海外生活補助金。但在戰時，這筆薪金並沒有發放給參戰的美國軍人。戰後，由於美國政府無法負擔這筆費用，因此在1924年訂定並通過《[[服役證明修正法|服役證明修正法]]》（Adjusted Service Certificate Law）。在該法案中，美國政府先發給一戰老兵一張補助金證書，承諾所欠薪金在二十年后依當兵日數，每日以1美元計算，再乘上複利後以現金償付。美國在1930年代發生[[經濟大蕭條|經濟大蕭條]]。1932年，全美有將近三分之一的就業人口找不到工作而急需救濟。於是這些退伍軍人就要求美國政府立刻支付應在1945年才支付的薪金。美國[[海外退伍軍人協會|海外退伍軍人協會]]更認爲，對一個餓肚子的人承諾二十年后才支付買食物的錢，是不合理的。

== 到達華府 ==
6月17日，來自美國各地約一萬二千至二萬名的群眾，其中包括大批[[失業|失業]]的退伍軍人，在前[[美國陸軍|陸軍]][[中士|中士]][[華德·W·瓦特斯|華德·W·瓦特斯]]（{{lang|en|Walter W. Waters}}）的帶領下到達[[华盛顿哥伦比亚特区|华盛顿哥伦比亚特区]]（華府），於[[美國國會大廈|美國國會大廈]]前集會。當時[[美國國會|美國國會]]正在考慮[[德薩斯州|德薩斯州]]的[[約翰·萊特·帕特曼|約翰·萊特·帕特曼]]議員所提出的對一戰老兵支付某種福利補助金的議案（該議案已於6月15日在[[美國眾議院|美國眾議院]]通過）。這些請願群眾後來在與[[美國聯邦政府|聯邦政府]]所在地只隔了[[安那柯斯提河|安那柯斯提河]]的安那柯斯提區[[濕地|濕地]]，搭起類似[[胡佛村|胡佛村]]的簡陋小屋或帳棚以做為集結場所。這些請願者希望能對國會施加壓力，使其修改《服役證明修正法》並立刻支付所欠的薪金。

但是，該議案最後在[[國會|國會]]中被否決。不過，國會表示願意支付請願者回家的路費。這個補助決定只被部分請願者接受。據《[[紐約時報|紐約時報]]》的報道，在7500名的請願人群裏，只有802人接受國會提出的要求，向[[退役軍人管理局|退役軍人管理局]]索取火車票。7月19日晚上，一位以[[左翼|左翼]]言論著名的軍方人物—退休[[美國海軍陸戰隊|海軍陸戰隊]][[少將|少將]][[斯曼得利·巴特勒|斯曼得利·巴特勒]]向請願群眾演說並鼓勵他們的行為；他表示，那些曾穿軍服服役但又不能團結一致的人，是差勁的。同日，有部分人群得到[[汽油|汽油]]、配給資助、或火車票，但仍然不肯離開。

7月28日，請願群眾依然逗留在一處聯邦政府計劃動工的土地上。當[[華府|華府]][[警方|警方]]試圖驅趕他們時，一位[[警察|警察]]開槍誤殺了兩名退伍軍人。憤怒的請願者於是開始以鈍器攻擊警方，打傷了數名警察。在警方撤退之後，[[華府|華府]]向[[赫伯特·胡佛|赫伯特·胡佛]]總統報告他們無法維持秩序；於是胡佛總統下令[[聯邦|聯邦]][[軍隊|軍隊]]進駐華府。

== 軍方介入 ==
同日，在可能違反1878年的《[[民兵團法案|民兵團法案]]》（Posse Comitatus Act）下，由[[道格拉斯·麥克阿瑟|道格拉斯·麥克阿瑟]]將軍指揮並擁有[[步槍|步槍]]和[[催淚瓦斯|催淚瓦斯]]的聯邦部隊開進了請願群眾所在的營地。部隊驅散了請願群眾並焚毀他們的營帳。當時為麥克阿瑟部下的[[德怀特·艾森豪威尔|德怀特·艾森豪威尔]]並不認同這次行動，採取了強烈保留的態度；而當時是少校的[[乔治·巴顿|喬治·巴頓]]則是最先抵達華盛頓特區的聯邦政府官員之一。在事件中，胡佛總統並沒有同意軍隊越過安那柯斯提河進入請願者的主營地。麥克阿瑟將軍則認爲這次事件是意圖推翻美國政府的行為。巴頓的裝甲部隊與步兵在白宮附近的橢圓草坪集合後，巴頓率領佩有[[防毒面具|防毒面具]]、[[刺刀|刺刀]]及軍刀的聯邦軍隊沿[[賓夕法尼亞大道|賓夕法尼亞大道]]前進，投擲[[催淚手榴彈|催淚手榴彈]]來驅散這批請願者。當晚稍後，巴頓及聯邦部隊開始清理安那柯斯提區內的帳篷，有些帳棚或小屋在清除過程中被焚毀。清场期间，军方调运来五辆[[雷诺坦克|雷诺坦克]]协助驱赶人群。到了第二天早上，大部分請願者都離開了特區。

事件發展到最後，有四人死亡，數百人受傷。被槍殺的退伍軍人為[[威廉·赫西卡|威廉·赫西卡]]（William Hushka）和[[埃里克·卡尔森|埃里克·卡尔森]]（Eric Carlson），有一名退伍軍人的妻子流產。一名十一週大的嬰兒由於暴露於[[催淚瓦斯|催淚瓦斯]]中，受到驚駭，导致性命危險。兩名兒童因催淚瓦斯窒息而死。一名十一嵗的男孩因催淚瓦斯傷害，導致失明。一名旁觀者被射傷肩胛。一名退伍軍人被騎兵軍刀傷害。一名退伍軍人的胸口被刺刀插入。至少十名警察被退伍軍人傷害。超過一千名男女兒童、警察、記者、華盛頓哥倫比亞特區居民和救護車司機暴露於催淚瓦斯中。

雖然聯邦部隊燒毀了請願者的帳棚及小屋，有一些報告則表示，一些退伍軍人在軍隊到達前，在營地縱火，以分散軍隊的注意力。

== 事後影響和評價 ==
胡佛總統以財政理由公開反對帕特曼所提出的議案；此外，參與對同行清場的士兵的報告，都對胡佛日後競選連任並無幫助：他下令驅散和甚少資助請願者的行為已經加強了他冷漠對待[[大蕭條|大蕭條]]苦況的負面形象。當時，民主黨候選人羅斯福甚至因此事件稱胡佛是「腦袋裝漿糊」，[[富兰克林·德拉诺·罗斯福|富兰克林·德拉诺·罗斯福]]就任總統以後，一些補助金大軍成員再次在華盛頓集結並向新總統舊事重提。羅斯福一直至其總統任期尾段同樣否決要求。直到他過世後，[[第二次世界大戰|第二次世界大戰]]（二戰）即將完結時，美國政府最終訂立《[[軍人權益法案|軍人權益法案]]》來避免類似衝突。

[[保羅·迪克森|保羅·迪克森]]認爲，集會其後的[[暴力|暴力]]時常被忽略，事件對[[1932年美國總統選舉|1932年美國總統選舉]]的影響亦被低估。他認爲，事件導致美國[[政治人物|政治人物]]對待二戰歸國老兵時用心考慮，促成《軍人權益法案》的訂立。《軍人權益法案》促使百萬人成爲[[中產階級|中產階級]]而改變了美國的狀態。
