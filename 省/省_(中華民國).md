'''省'''是[[中華民國|中華民國]]之[[中華民國行政區劃|第一級]][[行政區劃|行政區劃]]，由[[行政院|行政院]]直轄，下設[[縣_(中華民國)|縣]]、[[市_(中華民國)|市]]。中華民國建國時設22[[省_(行政區劃)|省]]，後增至35省，1949年因[[國共內戰|國共內戰]]失去[[中國大陸|中國大陸]][[領土|領土]]的統治權導致數量銳減，至1955年[[大陳島撤退|大陳島撤退]]後僅餘2省迄今，分別為[[臺灣省|臺灣省]]，以及僅實際管轄[[金門縣|金門]]與[[連江縣_(中華民國)|連江]]兩縣的[[福建省_(中華民國)|福建省]]。

在[[中華民國大陸時期|中華民國大陸時期]]，省是中華民國最主要的[[中國一級行政區|一級行政區]]，主要設置於[[中國本土|中國本土]]。[[中華民國國軍|中華民國國軍]]於國共內戰失利後，中華民國的[[中華民國疆域|有效統治區域]]僅限於[[臺灣地區|臺澎金馬]]。[[中華民國政府|中華民國政府]]在1998年將省制「虛級化」，移除其[[地方自治團體|地方自治團體]]的地位，並大幅裁撤省級機關；2018年，進一步將省級機關「去任務化」而實質廢除。目前僅依照[[中華民國憲法增修條文|憲法增修條文]]保留名義上的省政府主席、省諮議會諮議長之職位，其他行政組織及業務完全移交中央政府。

== 沿革 ==
{{中華民國大陸時期行政區劃單位}}
===北洋政府時期===
中華民國於1912年建立時，省份承襲[[清朝|清朝]]舊制，當時全國共設'''22省'''。當時執政的[[北洋政府|北洋政府]]在22個省之外設置[[京兆地方|京兆]]、[[熱河省|熱河]]、[[察哈爾省|察哈爾]]、[[綏遠省|綏遠]]、[[西康省|川邊]]等幾個特別區，[[西藏地方|西藏]]、[[蒙古地方|蒙古]]、[[青海省|青海]]3個地方，另保留[[阿爾泰區域|阿爾泰]]、[[塔城地區|塔城地區]][[塔爾巴哈臺|塔爾巴哈臺]]、[[伊犁哈薩克自治州|伊犛]]3地區（後均併入[[新疆省|新疆省]]）。

[[北洋政府|北洋政府]]的行政區劃級別為省、[[道_(行政區劃)|道]]、[[縣_(中華民國)|縣]]三級，並有「廢省置道」的計畫。當時全國設置93道，除[[甘肅省_(中華民國)|甘肅省]]最多有7道外，各省平均數量大致在3至4個左右。

===國民政府時期===
1928年，[[國民政府|國民政府]][[北伐|北伐]]成功統一全國後，改[[直隸|直隸]]、[[奉天|奉天]]2省為[[河北|河北]]、[[遼寧|遼寧]]，併京兆特別區入[[河北|河北]]，將[[熱河|熱河]]、[[察哈爾|察哈爾]]、[[綏遠|綏遠]]、[[川邊|川邊]]、[[寧夏|寧夏]]、[[青海|青海]]改建為6個省（川邊特別區改建[[西康省|西康省]]），總計'''28省'''，另外加上[[西藏|西藏]]、[[蒙古|蒙古]]2個地方，以及分自俄、英收回的[[東省特別區|東省特別區]]、[[威海衛|威海衛]]行政區，並增置與省平級的[[直轄市_(中華民國)|特別市]]（即[[直轄市_(中華民國)|直轄市]]）。

國民政府依照[[建國大綱|建國大綱]]制定[[訓政時期約法|訓政時期約法]]，地方制度改採省、[[縣_(中華民國)|縣]]二級制並廢除道制。但由於出現部分省份管轄縣數過多，省對縣的控制力量不彰，又於1931年設置[[行政督察區|行政督察區]]作為中介，但此一行政區劃缺乏法源依據。

1945年[[中国抗日战争|抗戰]]勝利後，國民政府將[[中国东北|東三省]]劃分成[[東北新省區方案|九省]]，增設[[遼北|遼北]]、[[安東|安東]]、[[合江|合江]]、[[松江|松江]]、[[嫩江|嫩江]]、[[興安|興安]]等6省，並增設[[臺灣省|臺灣省]]，共計'''35省'''。1949年，[[海南省|海南島]]和[[南海諸島|南海諸島]]自廣東省劃出，設立[[海南特別行政區|海南特別行政區]]，作為建省之準備。

===政府遷臺後===
1949年底，[[中華民國政府|中華民國政府]]因[[國共內戰|國共內戰]]失利而撤退至[[臺灣|臺灣]]，當時僅有臺灣全省，以及[[江蘇|江蘇]]、[[浙江|浙江]]、[[福建|福建]]、[[廣東|廣東]]、[[雲南|雲南]]、[[西康|西康]]等省，連同[[海南特別行政區|海南特別行政區]]的部分區域尚在中華民國政府的控制之下，其餘位在[[中國大陸|中國大陸]]的省份已全部丟失。1955年[[大陳島撤退|大陳島撤退]]後，中華民國的實際管轄區域僅限於[[臺灣地區|臺澎金馬]]，省份僅餘'''臺灣、福建2省'''；其中，福建省僅管轄[[金門縣|金門群島]]、[[烏坵鄉|烏坵]]與[[馬祖列島|馬祖列島]]。而隨著臺灣本島人口的增加，臺灣省分別劃出6個直轄市，所占全國土地面積也從超過99%下降至69.79%。

== 省虛級化 ==
{{see|臺灣省政府功能業務與組織調整}}

===緣起===
1949年底，[[中華民國政府|中華民國政府]]播遷臺灣，此後便從未對[[中國大陸|中國大陸]]地區實行統治權，僅統治[[台灣省|臺灣省]]全部與[[福建省|福建省]]的極小部分（[[金門縣|金門縣]]<ref>金門群島以及代管[[莆田縣|莆田縣]][[烏坵鄉|烏坵鄉]]</ref>、[[連江縣_(中華民國)|連江縣]][[馬祖列島|馬祖列島]]）。省份的數量雖大幅萎縮，雖然[[臺北市|臺北市]]與[[高雄市_(1979年-2010年)|高雄市]]分別在1967年和1979年升格為[[直轄市_(中華民國)|直轄市]]，臺灣省的行政區域[[面積|面積]]仍佔[[中華民國自由地區|中華民國實際管轄區域]]的98%以上。為了符合當前的社會情勢，地方制度的變革有其必要。

===過程===
1996年，甫連任[[中華民國總統|總統]]成功的[[李登輝|李登輝]]宣布召開[[國家發展會議|國家發展會議]]（簡稱國發會），邀請朝野各界共商國家的未來發展，而省制的廢除與否，則成為會中最受各界矚目的議題。經過朝野不斷的爭論，國發會最終共同做成了省制精簡化的決議。

1998年，朝野依照國發會的共識，合作透過[[立法院|立法院]]修改《[[中華民國憲法增修條文|中華民國憲法增修條文]]》部分內容，並制定《[[s:地方制度法|地方制度法]]》以取代原有的《[[s:省縣自治法|省縣自治法]]》。在這次的修法中，省制遭到虛級化，其原本擁有的「[[地方自治團體|地方自治團體]]」地位遭到移除，無法施行[[地方自治|地方自治]]。這也造成省政機關的重大變革：省政府仍具有[[公法人|公法人]]身分，但成為[[行政院|行政院]]的派出機關，政策制定與人事任命權均由中央掌握，功能與直轄市、市的[[區_(中華民國)|區]][[公所_(中華民國)|公所]]類似；省議會則改制為「[[台灣省諮議會|省諮議會]]」，成為省政的諮詢機關，失去原有[[立法機關|立法機關]]的功能。至於原屬省管轄的縣、市等[[地方政府|地方政府]]，改由中央政府直接管理，主要監理機關為[[中華民國內政部|內政部]]，各縣、市僅在地方區劃上仍隸屬於省。

省虛級化實施後，臺灣省受到的影響最大：[[臺灣省政府|臺灣省政府]]的組織被[[臺灣省政府功能業務與組織調整|大量精簡]]，[[臺灣省議會|臺灣省議會]]則依法改組為[[臺灣省諮議會|臺灣省諮議會]]，兩者的功能均大幅萎縮。而[[福建省_(中華民國)|福建省]]的政府機構自1956年7月為實施戰地政務而精簡化。

===去任務化===
2018年，[[行政院|行政院]][[行政院院長|院長]][[賴清德|賴清德]]宣佈省級機關去任務化預算歸零，所存省級行政組織及業務完全移交中央政府。<ref>[https://www.cna.com.tw/news/firstnews/201806280128.aspx 賴清德：108年省級機關預算全歸零]</ref>省級機關實質上解散，僅依照[[中華民國憲法增修條文|憲法增修條文]]保留[[臺灣省政府|臺灣省政府]]、[[臺灣省諮議會|臺灣省諮議會]]、[[福建省政府|福建省政府]]之機關名號及[[臺灣省政府主席|臺灣省政府主席]]、[[臺灣省諮議會|臺灣省諮議會諮議長]]、[[福建省政府主席|福建省政府主席]]之無給職職位<ref name="tpg">[http://www.cna.com.tw/news/aipl/201806230152-1.aspx 台灣省政府7/1走入歷史 國發會活化中興新村],2018.6.23 中央社</ref>。

2018年7月，[[臺灣省政府|臺灣省政府]]行政組織及業務由[[中華民國國家發展委員會|國家發展委員會]]接收<ref name="tpg"/>；[[臺灣省諮議會|諮議會]]業務由立法院接收<ref>[http://www.ncyu.edu.tw/files/bulletin/secretary/函轉臺灣省諮議會去任務化部分人員業務移撥立法院承接函_1.pdf 臺灣省諮議會去任務化部分人員業務移撥立法院承接函]</ref>。2018年12月，[[福建省政府|福建省政府]]之行政組織及業務由[[行政院金馬聯合服務中心|行政院金馬聯合服務中心]]接收<ref>[https://www.ey.gov.tw/eykm/292D78D38F3A39DF/c2d31cd1-d611-43cc-b8c1-af57c720e878 省府人員將移撥金馬聯服中心服務]</ref>。自此以後，中華民國實體省級機關正式走入歷史。

* 根據現行《[[中華民國憲法增修條文|中華民國憲法增修條文]]》和《[[s:地方制度法|地方制度法]]》的規定，省為非地方自治團體，設置[[省政府|省政府]]和[[臺灣省諮議會|省諮議會]]。其中，省政府為[[行政院|行政院]]派出機關，受行政院指揮監督，辦理下列事項：
# 監督縣（市）自治事項。
# 執行省政府行政事務。
# 其他法令授權或行政院交辦事項。
* 省政府主席依照慣例由[[行政院|行政院]][[政務委員_(中華民國)|政務委員]]兼任。
* 行政機關之編制員額原本依照 《[[:s:臺灣省政府組織規程|臺灣省政府組織規程]]》、《[[:s:臺灣省諮議會組織規程|臺灣省諮議會組織規程]]》及《[[:s:福建省政府組織規程|福建省政府組織規程]]》規定設置，但目前依中華民國憲法增修條文，機關名義保留，實質業務完全移撥[[行政院|行政院]]所屬機關與立法院辦理。
