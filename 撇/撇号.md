'''撇号'''（''' ' '''，{{lang-en|apostrophe}}，又称'''上標點'''、'''略縮号'''、'''省字号'''、'''省年号'''、'''高撇号'''或'''缩写号'''），是一种标点符号，用于英语的缩写词、复数。形式是'''高撇号'''（'''-{’}-'''），加在缩写字詞的'''左上方'''。

須注意的是，撇號的形状和正式[[英文|英文]]出版物和[[繁、簡體中文|繁、簡體中文]]所使用的[[引號|单引号]]（-{‘’}-）的后引号一样，但位置不同。后引号位置是在一格的右上角，而缩写号(-{’}-)的位置在一格的左上角。

==用法==
缩写号来自[[英语|英语]]的Apostrophe。其具体用法如下：
*缩写词，如''do not''可以写成''don't''（不曾），「cannot」等於「can't」（不能），「'06」等於「2006年」。
*[[所有格|所有的]]，如''the cat's whiskers''。
*字母、缩写词、年份的复数，如''P's and Q's''、''the late 1950's''。

近年来标识年份的缩写形式渐渐為人们所接受，所以在中文当中也逐渐出现了这些符号。但是要注意的是年份的縮写只限于本[[世纪|世纪]]，所以在有些时间较长的文献中的年份的所写应指的是那个世纪的年份，而不是現今的世纪。且缩写号应写在年份的前面如「2013年」应是「'13」，而「13'」是错的。
*我们所支持的曾获得过'05全国足球[[甲A联赛|甲A联赛]]的冠军。
*'06年春节联欢晚会

=== 汉语拼音 ===
在[[汉语拼音|汉语拼音]]，撇号称为'''隔音符号'''，以"a，o，e開頭的音節連接在其它音節後面的時候，如果音節的界限發生混淆，用隔音符號（’）隔開，例如：{{lang|la|pí'ǎo}}（[[皮襖|皮襖]]）"<ref>1958, 汉语拼音方案</ref>、{{lang|la|Xī'ān}}（[[西安|西安]]）。[[台北捷運|台北捷運]]不用隔音符號。<ref>{{cite web|url=https://tw.appledaily.com/headline/daily/20121223/34723619/|title=
怪 北捷景安站 英譯如「金幹站」|quote=北市捷運局指出，目前有7大捷運站名英譯沒有隔音符號，常讓外國人問路鬧烏龍，如大安站「Daan」被誤唸為丹站、景安站「Jingan」變成金幹站等，捷運局擬加撇號「’」或橫線「-」，以利分辨音節。|date=2012年12月23日|accessdate=2019年4月2日|website=[[蘋果日報|蘋果日報]]}}</ref>

==相關或形似字符==
*( <nowiki>'</nowiki> ) （撇號兼单引号） U+0027，原为[[ASCII|ASCII]]字符。
*( <nowiki>’</nowiki> ) （右单引号） U+2019，用於正式出版物。
*( <nowiki>′</nowiki> ) （[[角分符號|角分符號]]）U+2032，也是一撇的形狀，有時也稱為「撇號」，但用法不同。
