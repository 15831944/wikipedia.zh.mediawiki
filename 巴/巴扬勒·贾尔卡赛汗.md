'''巴扬勒·贾尔卡赛汗'''（{{bd|1915年||？|}}）[[蒙古族|蒙古族]]，蒙古[[乔巴山省|乔巴山省]]（今[[东方省_(蒙古国)|东方省]]）人，[[蒙古人民共和国|蒙古人民共和国]]政治人物。<ref name=rm>贾尔卡赛汗主席，人民日报1960年4月8日，第1版</ref>

==生平==
1915年，贾尔卡赛汗生于[[乔巴山省|乔巴山省]]的一个牧民家庭，先后毕业于小学、中学、党校和[[莫斯科|莫斯科]]东方研究学院。1934年起，贾尔卡赛汗在[[蒙古革命青年团|蒙古革命青年团]]组织里工作，曾任团委员会主席、团中央的部长等职务。他还历任青年杂志社、学术研究杂志社、文艺杂志社、青年团报社的社长，[[蒙古科学院|蒙古科学院]]院长，[[蒙古作家协会|蒙古作家协会]]主席等职务。<ref name=rm/>

1941年起，贾尔卡赛汗转入外交系统任职，曾任大使馆秘书、大使、外交部司长、部长等职。<ref name=rm/>1950年6月，贾尔卡赛汗成为首任[[蒙古人民共和国|蒙古人民共和国]]驻[[中华人民共和国|中华人民共和国]]大使。1950年7月3日，贾尔卡赛汗向[[中央人民政府|中央人民政府]]主席[[毛泽东|毛泽东]]呈递蒙古人民共和国小呼拉尔主席团的国书。<ref>蒙古人民共和国驻我国大使 贾尔卡赛汗向毛主席呈递国书，人民日报1950年7月4日</ref>

1952年，[[中华人民共和国|中华人民共和国]]建国3周年前夕，蒙古领导人决定访华，这是中华人民共和国首次接待国宾。1952年9月28日，[[蒙古人民革命党|蒙古人民革命党]]中央委员会总书记、部长会议主席[[尤睦佳·泽登巴尔|泽登巴尔]]飞抵[[北京南苑机场|北京南苑机场]]，随行人员共4人：副总理兼外交部长恩·拉姆苏伦、外交部总秘书长普尔布扎勒、东方司副司长都勒布尔金、泽登巴尔的秘书乌勒吉巴雅尔。另两位成员即教育部长[[巴扎尔·锡林迪布|巴扎尔·锡林迪布]]、驻华大使贾尔卡赛汗已先期在北京<ref>{{Cite web |url =  http://paper.people.com.cn/rmrbhwb/html/2014-01/24/content_1383047.htm  |title =  迎宾礼仪回顾（一）（礼仪漫谈）|author =  |date =  |publisher =  人民日报海外版2014年1月24日，第14版|language =  |accessdate =  |quote =  }}</ref>。

贾尔卡赛汗任驻华大使至1953年6月，随后被[[巴扬巴托·奥其尔巴特|巴扬巴托·奥其尔巴特]]接替。<ref>宋思繁，中华人民共和国外交大事记1，世界知识出版社，1997年，第360页</ref>回国后，贾尔卡赛汗出任蒙古外交部长。<ref>我国家领导人员祝贺蒙古国庆，人民日报1954年11月26日</ref>

1947年起，贾尔卡赛汗当选为[[蒙古人民革命党|蒙古人民革命党]]中央委员会委员。1956年起，担任蒙古人民革命党中央委员会对外联络部部长。<ref name=rm/>

1951年起，贾尔卡赛汗当选为蒙古[[大人民呼拉尔|大人民呼拉尔]]委员，1957年当选为大人民呼拉尔外事委员会主席。<ref name=rm/>1959年4月1日，蒙古[[大人民呼拉尔|大人民呼拉尔]]选举贾尔卡赛汗为大人民呼拉尔主席。<ref>20世纪大事记，国际文化出版公司，1991年，第435页</ref>1960年4月8日，大人民呼拉尔主席、[[蒙古人民革命党|蒙古人民革命党]]中央委员会委员巴扬勒·贾尔卡赛汗率蒙古人民共和国大人民呼拉尔代表团，应[[全国人民代表大会常务委员会委员长|全国人民代表大会常务委员会委员长]][[朱德|朱德]]的邀请，到达[[北京|北京]]开始对[[中华人民共和国|中华人民共和国]]进行友好访问。<ref>中华人民共和国日史11，四川人民出版社，2003年，第97页</ref>
