'''巴儿速孛罗'''（{{lang-mn|Барсболд}}；{{bd|1490年||1531年|catIdx=B}}），清朝译作'''巴尔斯博罗特'''，[[蒙古大汗|蒙古大汗]]，[[达延汗|达延汗]]的第三子，封为[[济农|济农]]<ref name="cy">{{cite book |title = 《[[草原帝国|草原帝国]]》 |author = [[勒内·格鲁塞|勒内·格鲁塞]] }}</ref><ref name="zh">{{cite book |title = 《最后的可汗：蒙古帝国余晖》 |author = [[班布尔汗|班布尔汗]] }}</ref>。

== 生平 ==
达延汗本来指定长子[[图鲁博罗特|图鲁博罗特]]的长子[[博迪|博迪]]为继承人。然而，在1517年达延汗死后，巴尔斯博罗特济农以博迪年幼为由，自称[[大汗|大汗]]；达延汗的其他儿子不服，1519年，达延汗的第四子[[阿尔苏博罗特|阿尔苏博罗特]]的支持下，博迪登基，巴尔斯博罗特退位。博迪为了安抚巴尔斯博罗特，封他三个儿子为小汗。巴尔斯博罗特统治[[河套|河套]][[鄂尔多斯|鄂尔多斯]]，后来著名的[[俺答汗|俺答汗]]就是他的儿子。<ref name="cy" /><ref name="zh" /><ref name="cjshl">{{cite web |url = http://www.cjshl.com/news/view/79 |title = 陵寝管理官员济农 |website = 成吉思汗陵 |date = 2007-08-26 |quote = 巴尔斯博罗特，巴图孟克达延汗三子，1484年生，1512年任右翼三万户济农，1531年卒。 |access-date = 2013-06-05 |archive-url = https://web.archive.org/web/20140714165629/http://www.cjshl.com/news/view/79 |archive-date = 2014-07-14 |dead-url = yes }}</ref>
