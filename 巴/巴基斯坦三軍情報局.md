'''三军情报局'''（{{lang-ur|'''انٹر سروسز انٹلیجنس'''}}；{{lang-en|'''Directorate for Inter-Services Intelligence'''}}，简作'''ISI'''）是[[巴基斯坦|巴基斯坦]]最大的情报机构。它的主要职能类似[[美国|美国]]的[[CIA|CIA]]，[[英國|英國]]的[[英国秘密情报局|秘密情報局（MI6）]]。三军情报局成立于1948年。<ref>[http://news.sohu.com/20080325/n255906364.shtml 巴基斯坦三军情报局揭秘]</ref>总部原先位于[[拉瓦尔品第|拉瓦尔品第]]，后迁往[[伊斯兰堡|伊斯兰堡]]。三军情报局现任局长为Zaheerul Islam中將，2012年3月上任。

* '''联合情报处'''（JIB），负责收集政治情报。<ref name = "FAS"/>

* '''联合信号情报处'''（JSIB）
* '''联合情报技术处'''（JIT）

== 與恐怖分子的關連 ==

1990年代後期三軍情報局就開始培養並唆使伊斯蘭極端主義份子，例如虔誠軍。<ref>{{Cite web |url = https://global.udn.com/global_vision/story/8663/2235225 |title = 恐怖的曖昧：巴基斯坦，擁恐還是反恐？ |author = 徐子軒 |date = 2017年1月17日  |publisher = 轉角國際 |language = 中文 |accessdate = 2017年6月22日}}</ref>

2007年，阿富汗当局表示一名被扣押的[[塔利班|塔利班]]發言人哈尼在审讯过程中承认[[穆罕默德·奥马尔|穆罕默德·奥马尔]]正在受到巴基斯坦三军情报局的保护，住在巴基斯坦的奎达市。<ref>[http://www.voachinese.com/content/a-21-n2007-01-18-voa31-63061817/1045811.html 《塔利班称奥马尔躲在巴基斯坦》]，www.voachinese.com，2007年1月17日。</ref>

2008年，阿富汗政府指责巴基斯坦三军情报局介入了先前针对阿富汗总统[[卡尔扎伊|卡尔扎伊]]的未遂暗杀行动。<ref>[http://www.voachinese.com/content/a-21-w2008-06-26-voa71-63174632/962644.html 《阿指责巴策划暗杀巴称指责无根据》]，www.voachinese.com，2008年6月25日。</ref>

2010年，[[伦敦政治经济学院|伦敦政治经济学院]]發表一份报告，指有确凿证据显示巴基斯坦三军情报局不仅为阿富汗的塔利班好战分子提供资金、训练和避难所，甚至还是塔利班领导委员会的成员。<ref>[http://www.voachinese.com/content/article-20100613-afghanistan-pakistan-96264059/820756.html 《有报告称巴基斯坦情报机构支持塔利班》]，www.voachinese.com，2010年6月12日。</ref>

2010年另有報道指巴基斯坦三军情报局要求塔利班杀掉所有人，包括警察、军人和平民，以威吓人民。<ref>[http://www.voachinese.com/content/article-20101007-us-pakistan-104473424/770396.html 《报导称巴基斯坦情报机构鼓励塔利班》]，www.voachinese.com，2010年10月6日。</ref>

2011年，美國法院審訊涉嫌參與[[2008年孟買連環恐怖襲擊|2008年孟買連環恐怖襲擊]]的疑犯[[塔哈瓦尔·拉纳|塔哈瓦尔·拉纳]]，有證人指稱巴基斯坦三军情报局是「[[虔诚军|虔诚军]]」和「穆罕默德军」等好战组织的保护伞，三军情报局向他们提供军事和财务上的支援和训练，在制定攻击孟买的计划时，虔诚军和三军情报局之间合作非常密切。<ref>[http://www.voachinese.com/content/article-20110526-mumbai-us-122685789/840711.html 《孟买恐袭嫌犯美国法院受审》]，www.voachinese.com，2011年5月25日。</ref>
