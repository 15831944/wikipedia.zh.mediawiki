{{巴哈伊}}
'''巴哈伊著作'''像许多其他宗教的著作一样，涵盖了多种主题和形式，包括[[宗教经典|宗教经典]]、启示、解释、[[巴哈伊历史|历史]]、传记、介绍、学习材料等。有时，特定的著作同时包含了前面的多种形式。

巴哈伊的圣作特指[[巴孛|巴孛]]、[[巴哈欧拉|巴哈欧拉]]、[[阿博都巴哈|阿博都巴哈]]、[[守基·阿芬第|守基·阿芬第]]和[[世界正义院|世界正义院]]所著的作品及被认定为真实的[[阿博都巴哈|阿博都巴哈]]谈话记录。巴孛和巴哈欧拉的作品被认为是神圣启示，阿博都的巴哈作品和谈话及守基阿芬第的作品被认为是权威解释、世界正义院的作品则被认为是权威立法及说明。 这些著作都在一定程度上被认为是神圣指引。<ref name = "Smith100">{{harvnb |Smith |2000 |pp=100–101}}</ref><ref name = "Schaefer7">{{harvnb |Schaefer |2007 |p= 7 }}</ref>

巴哈伊信仰大量的依赖其著作。[[识字|识字]]在巴哈伊信仰中是被强烈鼓励的，这样信徒就可以自己来阅读相关著作。<ref name = "Smith227">{{harvnb |Smith |2000 |p=227}}</ref>此外，巴哈伊也总是回到主要著作中去寻找有关教义的问题。<ref name = "Schaefer7" /><ref name = "Smith115">{{harvnb |Smith |2000 |pp= 115–116}}</ref>

巴哈伊许多早期著作为被称作[[书简|书简]]的给个人或社区的信件，<ref name = "Schaefer7" />他们在不同时期被收入各种巴哈伊出版物。时至今日，世界正义院依然使用信件作为主要的交流方法。

==文学形式==
总的来说，特定作品的文学形式从它的名字上就可以得知。

巴哈伊认为信仰的创立者巴孛和巴哈欧拉直接从上帝获得了启示。因此，他们的作品被认为受到了神圣指引，成为[[宗教经典|圣作]]或[[启示|启示]]。<ref name = "Smith227"/><ref name = "Smith294">{{harvnb |Smith |2000 |p= 294}}</ref>

[[阿博都巴哈|阿博都巴哈]]被巴哈欧拉任命为他的继承人，且被授权解释相关宗教经典。因此阿博都巴哈的作品被认为是权威性的指引或解释，同时也是巴哈伊圣作的组成部分。<ref name = "Smith227"/>阿博都巴哈、巴孛和巴哈欧拉被认为是巴哈伊信仰的核心人物。<ref name = "Smith100" />

而守基·阿芬第的解释和指引同样被认为具有权威性，但是其作品并不被认为是启示或宗教经典。<ref name = "Smith227"/>

在巴哈伊看来，世界正义院没有义务解释巴哈欧拉、阿博都巴哈、守基·阿芬第的作品；但是，其承担回答关于这些作品问题的责任。<ref name = "Smith346">{{harvnb |Smith |2000 |pp = 346–350}}</ref>只要尚处有效状态（世界正义院会根据需要调整或废除其自己曾作出的决定），这些回答也被认为是有权威性的，<ref name = "Smith346"/> 并常被结集编入出版物。

巴哈伊核心人物、守基·阿芬第和世界正义院的作品组成了巴哈伊信仰的圣作。<ref name = "Smith100"/>

有一类特别的作品由巴哈伊信仰的核心人物的作品组成，这些作品包括各种祈祷书，其中的祈祷文多出自核心人物的信件。巴哈欧拉的《祈祷与沉思》是一套重要的作品。因为巴哈伊有祈祷、沉思和每日学习圣作的义务，所以该作品在巴哈伊中十分常见。<ref name = "Smith274">{{harvnb |Smith |2000 |pp= 274–275}}</ref>

===历史和传记===
守基·阿芬第所著的唯一书籍《[[神临记|神临记]]》介绍了信仰在1844年至1944年间的历史。<ref name = "Smith317">{{harvnb |Smith |2000 |pp=318–318}}</ref>纳比勒扎兰迪的《[[破晓群英传|破晓群英传]]》介绍了信仰从巴比时期到1853年巴哈欧拉自[[波斯|波斯]]被驱逐阶段的历史。<ref name = "Smith118">{{harvnb |Smith |2000 |p=118}}</ref>

[[鲁伊希·哈努姆|鲁伊希·哈努姆]]所著的《[[监护者司职期|监护者司职期]]》详细介绍了从1957年守基·阿芬第去世至1963年世界正义院完成选举并设立这一过渡阶段的历史。<ref name = "Smith117">{{harvnb |Smith |2000 |p=117}}</ref>

此外，存在许多涉及信仰早期在中东或其他地区历史的书籍，其中大量作品具有传记性质。<ref name = "Smith227"/>H.M.巴柳齐及阿迪卜·塔赫尔扎德聚焦于信仰的核心人物与其同时代人物的历史和传记尤其引人注目。<ref name = "Smith897">{{harvnb |Smith |2000 |pp=89–90}}</ref>

===介绍与学习材料===
主要由哥伦比亚巴哈伊社区制作的儒禧课程系列书籍受其启发的学习材料，被全世界巴哈伊社区认为有相当的重要性。这些材料被广泛应用于[[巴哈伊学习小组|巴哈伊学习小组]]及巴哈伊培训机构。<ref>{{cite web | last = Bahá'í International Community | title = Collaborative Study for Individual and Social Transformation | url = http://www.bahai.org/features/institutes | authorlink = Bahá'í International Community | accessdate = 2006-12-04 }}</ref>

==语言==
包括巴哈欧拉在内的大部分巴哈伊著作，以[[波斯文|波斯文]]或[[阿拉伯文|阿拉伯文]]写就。<ref name = "Smith227"/> 守基·阿芬第用他创造的巴哈伊独特的拼写体系将大量著作翻译成英文。<ref>{{Cite book |author = `Abdu'l-Bahá |authorlink = `Abdu'l-Bahá |origyear = 1901-08 |year = 1992 |title = The Will And Testament of ‘Abdu’l-Bahá |url = http://reference.bahai.org/en/t/ab/WT/wt-1.html#pg11 |page = 11 }} & {{Cite book |first = Shoghi |last = Effendi |authorlink = Shoghi Effendi |year = 1938 |title = The World Order of Bahá’u’lláh |pages = 148–149 |url = http://reference.bahai.org/en/t/se/WOB/wob-40.html#pg148 }}</ref>因守基·阿芬第不仅是翻译人员，还在信仰中具有解释圣作的权威性，因此他的翻译风格极大的影响了巴哈伊著作的翻译工作。<ref>{{Cite book |author = Bahá'u'lláh |authorlink = Bahá'u'lláh |year = 2002 |title = The Summons of the Lord of Hosts |url = http://reference.bahai.org/en/t/b/SLH/slh-14.html }}</ref>

==真实性与权威性==
巴哈伊特别注重特定作品-特别是有权威性人物作品的真实性。<ref name = "Smith100-307">{{harvnb |Smith |2000 |pp= 100–101, 307}}</ref>世界正义院的研究部门及[[国际巴哈伊图书馆|国际巴哈伊图书馆]]的主要职责就是收集、编撰及翻译这些作品。<ref>{{cite web | author = The Universal House of Justice | authorlink = Universal House of Justice | title = Questions about Aspects of the Bahá'í Teachings | date = 1997-08-06 | url = http://bahai-library.com/uhj_aspects_teaching | accessdate = 2006-12-22 }}</ref>

相比之下，朝圣者笔记则是真实性未被验证的信仰核心人物的言论作品。虽然这些作品鼓舞人心，但并不被认为具有权威性。<ref name = "Smith100"/><ref>{{cite web | author = The Universal House of Justice | authorlink = Universal House of Justice | title = Utterances and Tablets of `Abdu'l-Bahá Quoted in Compilations and Letters of the Universal House of Justice | date = 2003-07-14 | url = http://bahai-library.com/uhj_authentic_quotations_pilgrims | accessdate = 2006-12-22 }}</ref>一些阿博都巴哈的谈话集 （如《[[阿博都巴哈在伦敦|阿博都巴哈在伦敦]]》、《[[巴黎谈话|巴黎谈话]]》、《[[弘扬世界和平|弘扬世界和平]]》）即属这一类别，其真实性也待进一步验证。<ref>{{cite web | author = The Universal House of Justice | authorlink = Universal House of Justice | title = Authenticity of Some Bahá'í Texts | date = 1996-10-22 | url = http://bahai-library.com/uhj_authenticity_some_texts | accessdate = 2006-12-22 }}</ref> 1910年至1924年间出版的《[[西方之星|西方之星]]》，包含大量朝圣者笔记及未经验证的阿博都巴哈的信件。

巴哈伊并没有类似[[伊斯兰教|伊斯兰教]]中[[圣训|圣训]]的理论，“圣训”的权威性并不被考虑。<ref name = "Schaefer7" /><ref name = "Smith307">{{harvnb |Smith |2000 |p= 307}}</ref>

巴哈伊社区一直努力扩大被验证及被翻译作品的范围，《[[亚格达斯经|亚格达斯经]]》、《[[圣秘瑰宝|圣秘瑰宝]]》等作品即属晚近添加。
