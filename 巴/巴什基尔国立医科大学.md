'''巴什基尔国立医科大学'''（{{Lang-ru|Башкирский Государственный медицинский университет}}；{{Lang-ba|Башҡорт дәүләт медицина университеты}}）是位于[[俄罗斯联邦|俄罗斯联邦]][[巴什科尔托斯坦共和国|巴什科尔托斯坦共和国]][[乌法|乌法]]的一所国立大学<ref name="contact">{{Cite web|url=http://bashgmu.ru/contacts/|title=Контакты|accessdate=2018-11-09|author=|date=2018|location=Уфа|publisher=Башкирский Государственный медицинский университет|language=ru}}</ref>，也是巴什科尔托斯坦共和国医学和制药科学中心。该大学成立于1932年，<ref name="university">{{Cite web|url=http://bashgmu.ru/en/university/|title=University|accessdate=2018-11-09|author=|date=2018|location=Ufa|publisher=Bashkir State Medical University|archive-url=https://web.archive.org/web/20180420033005/http://bashgmu.ru/en/university/#|archive-date=2018-04-20|dead-url=no}}</ref>直属于[[俄罗斯联邦卫生部|俄罗斯联邦卫生部]]。校园建筑总面积约15万平方米，下设儿科系、预防医学与微生物学系、口腔医学系、药学系等多个院系，为中俄医科大学联盟创始成员之一。<ref name=":0">{{Cite news|url=https://news.jlu.edu.cn/info/1021/45455.htm|title=俄罗斯巴什基尔国立医科大学校长瓦连京·巴甫洛夫院士一行访问吉林大学|author=国际交流与合作处|date=2017-10-27|work=吉林大学新闻中心|accessdate=}}</ref>

== 历史 ==
1932年3月25日，根据[[俄罗斯苏维埃联邦社会主义共和国人民委员会|俄罗斯苏维埃联邦社会主义共和国人民委员会]]第289号决议，巴什基尔国立医学研究所成立。起初，该医学研究所只有普通医学院一个院系，后来创建了儿科系、预防医学学院（后更名为预防医学和微生物学系）、口腔医学系、药学院等院系<ref name="university" />。1995年6月23日，根据[[聯邦委員會_(俄羅斯)|俄罗斯联邦委员会]]第953号和[[俄罗斯联邦卫生部|俄罗斯联邦卫生部]]第209号的命令，巴什基尔国立医学研究所更名为巴什基尔国立医科大学。<ref name=":2">{{Cite web|url=http://bashgmu.ru/about_the_university/|title=О ВУЗе|accessdate=2018-11-11|author=|date=2018|location=Уфа|publisher=Башкирский Государственный медицинский университет|language=ru|archive-url=https://web.archive.org/web/20180326004949/http://bashgmu.ru/about_the_university/#|archive-date=2018-03-26|dead-url=no}}</ref>2010年10月12日，[[乌法医学院|乌法医学院]]合并入巴什基尔州立医科大学<ref name="university" />。2017年，巴什基尔州立医科大学迎来了85周年校庆<ref>{{Cite news|url=https://rb7.ru/news/193584|title=В Уфе ученые из БГМУ празднуют 85-летие университета|author=|date=2017-11-22|work=Новости@RB7.ru|publisher=Спутник FM|accessdate=2018-11-11|language=ru}}</ref>。

== 校园环境 ==
大学建筑的总面积超过15.2万平方米，其中教学区、实验室空间约7万平方米；设有69个临床中心，总面积超过20万平方米；拥有自己的诊所，每年有超过22000名患者接受治疗；并设有疗养院、运动场、宿舍等。该大学通过各种融资，购买了价值超过1亿[[卢布|卢布]]的设备，并于2012年设立模拟中心。当地的学生、居民、医生等都可以到模拟中心接受培训。<ref name=":0" />

== 对外交流 ==
截至2018年，前往巴什基尔国立医科大学学习的留学生来自26个国家，共有230多人。另外，该大学还跟[[中华人民共和国|中华人民共和国]]的多所大学开展了学术合作<ref name="pku" />，并与德国、哈萨克斯坦、土耳其和美国的多所医学院和医疗中心签订了合同<ref name=":2"/>。

2014年，中华人民共和国和俄罗斯联邦的46所医科大学共同成立中俄医科大学联盟，巴什基尔国立医科大学为创始成员之一<ref name=":0" />。每年有超过150名学生和教师到其他地方进行长时间的学术交流<ref name=":2" />。2015年，巴什基尔国立医科大学与[[哈尔滨医科大学|哈尔滨医科大学]]签订合作协议，前者的部分博士生可以到后者继续学习，并有机会学习[[汉语|汉语]]，就读联合博士（3年）和硕士+博士（4年）课程。2018年11月，该校有17名学生正在哈尔滨医科大学学习，还有一些学生则被派往[[四川大学|四川大学]]学习。<ref>{{Cite web|url=http://bashgmu.ru/en/international-cooperation/joint-phd-program-with-harbin-medical-university.php|title=Joint PhD Program with Universities of China|accessdate=2018-11-09|date=2018|location=Ufa|publisher=Bashkir State Medical University|archive-url=https://web.archive.org/web/20180404192022/http://bashgmu.ru/en/international-cooperation/joint-phd-program-with-harbin-medical-university.php#|archive-date=2018-04-04|dead-url=no}}</ref>

2017年到2018年，巴什基尔国立医科大学先后派出访问团，到[[吉林大学|吉林大学]]<ref name=":0" />、哈尔滨医科大学<ref name=":1">{{Cite news|url=http://website.hrbmu.edu.cn/Article/ShowArticle.asp?ArticleID=5557|title=巴什基尔国立医科大学授予杨宝峰校长荣誉教授称号|author=何剑刚|date=2018-04-03|work=哈医大新闻网|accessdate=}}</ref>、[[北京大学人民医院|北京大学人民医院]]<ref name="pku">{{Cite news|url=http://pkunews.pku.edu.cn/xwzh/2018-07/17/content_303730.htm|title=俄罗斯巴什基尔国立医科大学校长一行到访北京大学人民医院|author=凌薇|date=2018-07-17|work=北京大学新闻网|accessdate=2018-07-17}}</ref>等地进行了交流。巴什基尔国立医科大学学术委员会还授予哈尔滨医科大学校长[[杨宝峰|杨宝峰]]“名誉教授”称号<ref name=":1" />。2018年11月8日，哈尔滨医科大学也授予巴什基尔国立医科大学校长瓦连京·巴甫洛夫（{{lang-ru|Валентин Павлов}}）“名誉教授”称号。<ref>{{Cite news|url=http://www.bashinform.ru/news/1236033-rektor-bgmu-valentin-pavlov-priznan-pochetnym-professorom-kharbinskogo-meduniversiteta/|title=Ректор БГМУ Валентин Павлов признан почетным профессором Харбинского медуниверситета|author=|date=2018-11-08|publisher=Информационное агентство «Башинформ»|location=Уфа|accessdate=2018-11-09|language=ru}}</ref>
