{{地區用詞4|cn=巴巴多斯|hk=巴巴多斯|tw=巴貝多|start={{lang-en|'''Barbados'''}}|as=稱為}}是位于[[加勒比海|加勒比海]]与[[大西洋|大西洋]]边界上的[[岛国|岛国]]，是[[西印度群岛|西印度群岛]]最东端的[[岛屿|岛屿]]，首都[[布里奇敦|布里奇敦]]。岛屿[[面积|面积]]430[[平方公里|平方公里]]，主要是低地，在岛的内陆有一些山丘。巴巴多斯岛位于13ºN、59ºW及其附近。距[[委内瑞拉|委内瑞拉]]的东北约434.5[[公里|公里]]。

巴巴多斯有稳定的民主政体，独立于1966年11月30日，是[[英联邦|英联邦]]成员。

巴巴多斯的名字来自于[[葡萄牙语|葡萄牙语]]「{{lang|pt|''Os Barbados''}}」或[[西班牙語|西班牙語]]「{{lang|es|''Los Barbados''}}」，本意皆為「the bearded ones」，意指遍地都是的野生的[[无花果|无花果]]树。

== 历史 ==
{{main|巴巴多斯历史}}
17世纪，英格兰人在巴巴多斯上建立永久殖民地。后巴巴多斯为英国殖民地。1966年，巴巴多斯获得独立。伊丽莎白二世女王为国家元首。

== 地理 ==
{{main|巴貝多地理}}
巴巴多斯岛主要由[[石灰石|石灰石]]构成。常年受稳定的[[信风|信风]]和暖流影响，形成[[热带海洋性气候|热带海洋性气候]]。海岸由[[湿地|湿地]]和[[红树林|红树林]][[滩涂|滩涂]]组成。岛的内陆地区有大片的[[甘蔗|甘蔗]]。巴巴多斯是[[小安地列斯群岛|小安地列斯群岛]]的一员，它位于主岛链的最东端，[[圣卢西亚|圣卢西亚]]和[[圣文森特和格林纳丁斯|圣文森特和格林纳丁斯]]是巴巴多斯最近的邻国。巴巴多斯现在是旅游胜地。

== 经济 ==
{{main|巴貝多经济}}
{{Unreferenced section|time=2017-06-29T12:29:52+00:00}}
[[File:Bridgetown_-_Shopping_Street.jpg|缩略图]]
[[File:Sweet_Beach_in_Dover.jpg|缩略图]]
[[File:Fountain_at_Independence_Square,_Bridgetown,_Barbados-002.jpg|缩略图]]
目前巴巴多斯的GDP总量和人均收入均低于2007年的水平，商品零售价格指数上涨36%，巴元价值缩水。同时[[失业率|失业率]]上升，有些就业人员一周仅工作2-3天，民众的可支配收入减少，即使提高税率也无法改善税收总额不断减少的问题。

[[国际货币基金|国际货币基金]]最新预测显示，2013和2014年巴经济将分别负增长0.8%和1.1%，将是[[西半球|西半球]]唯一经济没有增长的国家。而巴央行只在最新报告中预测2013年巴经济将负增长0.7%，并未提及2014年经济增长预期。

格里菲斯认为，巴最令人担忧的经济问题是高达2.5亿美元的巨额经常账户赤字和不断增加的债务。过去5年中，巴政府经济政策的主要目标是减支，但这会导致各种相关税收和财政收入减少。近几年商业部门一直不景气，失业率持续上升，民众工作时间一减再减，缴纳的个人所得税和企业税也都在相应减少，仅靠提高税率的办法无法实现政府财政收入增收的目标。如果巴政府除了减支以外没有更加适当的措施，赤字和债务问题都不可能得到改善，经济状况将进一步恶化。
