'''巴雷特M82'''（{{lang-en|'''Barrett M82'''}}）是由美國[[巴雷特|巴雷特]]研發生產的重型SASR''（Special Application Scoped Rifle—特殊用途[[狙擊步槍|狙擊步槍]]）''，M82幾乎是主要西方國家的軍隊都有使用，包括美軍特種部隊。美軍暱稱「輕50」''（Light Fifty）''因為其使用[[白朗寧M2重機槍|白朗寧M2重機槍]]的大口徑[[12.7×99mm_NATO|12.7×99 NATO]]（.50 BMG，12.7毫米）彈藥，此槍有兩種衍生型—原本的M82A1（A3）和採用[[无托结构|无托结构]]設計以便於攜帶的M82A2，隨後的[[巴雷特XM500狙擊步槍|XM500]]也繼承它的設計精神，使用了[[无托结构|无托结构]]設計。

== 發展史 ==
M82是由[[朗尼·巴雷特|朗尼·巴雷特]]（Ronnie Barrett）建立，以使用[[12.7×99mm_NATO|12.7×99毫米NATO（.50 BMG）]]口徑彈藥來發展的一套半自動狙擊步槍專案。該口徑彈藥原本是[[白朗寧M2重機槍|白朗寧M2]][[重機槍|重機槍]]所用，於1980年代早期開始研發，之後在1982年造出第一把樣槍，命名為M82，巴雷特繼續在1980年代研發，並於1986年發展出M82A1狙擊步槍。

第一張成功推銷的軍用訂單是[[瑞典|瑞典]]於1989年採購100支M82A1，而最大及最成功訂單緊接在1990年，美軍宣布全面採用M82A1以於科威特的[[沙漠之盾|沙漠之盾]]和[[海湾战争|沙漠風暴]]行動中攻擊伊拉克軍，125支先配備於[[美国海军陆战队|美国海军陆战队]]，陸軍和空軍也接著訂購。M82A1也是美軍唯一的SASR—「特殊用途的狙擊步槍」，可以用於反物資攻擊和引爆彈藥庫。M82具有超過1500米的有效射程，甚至有過2500米的命中紀錄，超高動能搭配高能彈藥（如Raufoss Mk 211），可以有效毀掉[[雷达预警接收器|雷达]]站、[[卡車|卡車]]、停放的[[战斗机|战斗机]]等戰略的物資，因此也稱為「[[反器材步槍|反物資步槍]]」。

更先進的M82A2[[犢牛式|犢牛式]]短槍管設計於1987年研發成功，減低[[後座力|後座力]]的設計使其可以手持射擊而不必使用[[兩腳架|兩腳架]]，但M82A2並沒有很成功地打入市場，很快就停產。然而2006年，巴雷特開始研發XM500，它的犢牛式設計也很類似M82A2。

M82槍族最新的產品是M82A1M，被美國海軍陸戰隊大量裝備並名命為M82A3 SASR。M82A1M與M82A1不同之處在於M82A1M的戰術導軌被大幅度加長，除原本的望遠式瞄準鏡外更可同時加裝其他如夜視鏡等的瞄準裝置，在[[槍托|槍托]]底部加裝單腳架，其他還包括槍身輕量化、改用可拆式[[兩腳架|兩腳架]]及改良的雙室[[砲口制動器|槍口制退器]]。

其他衍生自M82A1還有M82A1A SASR，基本設計相同但專門發射Raufoss Mk 211 Mod 0穿甲燃燒彈。

巴雷特M82的使用國至少有30個，如[[比利時|比利時]]、[[智利|智利]]、[[丹麥|丹麥]]、[[芬蘭|芬蘭]]、[[法國|法國]]、[[德國|德國]]、[[希臘|希臘]]、[[以色列|以色列]]、[[意大利|意大利]]、[[牙買加|牙買加]]、[[馬來西亞|馬來西亞]]、[[墨西哥|墨西哥]]、[[荷蘭|荷蘭]]、[[挪威|挪威]]、[[菲律賓|菲律賓]]、[[葡萄牙|葡萄牙]]、[[中華民國|中華民國]]、[[西班牙|西班牙]]、[[瑞典|瑞典]]、[[土耳其|土耳其]]、[[英國|英國]]及[[美國|美國]]，M82亦是民間市場常見的.50長程射擊競賽用槍，競賽標靶達1000碼（914公尺）甚至更遠可打到1250公尺。

[[美国海岸警卫队|美国海岸警卫队]]使用M82進行反毒作戰，有效的打擊了海岸附近的高速度運[[毒品|毒]][[小艇|小艇]]。同樣地，巴雷特M82也受[[執法機關|執法機關]]所愛用，包括[[纽约市|紐約]]警察局，因為它可以很快地攔截車輛，而且.50口徑一發就能打壞車子引擎，也能很快打穿[[磚牆|磚牆]]和[[水泥|水泥]]，適合[[城市|城市]]戰鬥。

根據文件'''The Brooklyn Connection'''記載，M82由美國的反抗軍支持者走私進[[科索沃|科索沃]]成為[[科索沃革命軍|科索沃革命軍]]的武器。[[北愛爾蘭|北愛爾蘭]]在1990年代，[[臨時愛爾蘭共和軍|臨時愛爾蘭共和軍]]的南阿馬旅（IRA）採用極端式的狙擊戰役對抗英軍和皇家警察，用的就是巴雷特狙擊槍，1997年最後一位死於此戰的英國兵Stephen Restorick就是被狙殺，在北愛爾蘭最強的IRA[[狙擊手|狙擊手]]Michael Caraher被捕後他的狙擊槍也被沒收。

根據側面了解M82A1於2002年被選為實驗性的OSW（Objective Sniper Weapon）平台。此槍改成短槍管的25毫米[[口徑|口徑]]，發射[[25毫米榴彈|25 x 59毫米OCSW高爆連發榴彈]]。實驗性的OSW可以更有效打擊目標，但是後座力過大。這武器稱為'''巴雷特Payload Rifle'''，現在已經被稱為[[巴雷特XM109狙擊榴彈發射器|XM109]]。

== M82到M107 ==
'''XM107'''原始設計是一種[[手動槍機|手動槍機]]狙擊步槍，而且也已經被美軍選中，但是最後美軍並沒採用，事實上，原本送交美軍測試的XM107其實是[[巴雷特M95狙擊步槍|巴雷特M95]]。

當美軍決定需要長程槍枝需求時，已經有預算分配給XM107，為了避免預算問題複雜化，決定變更購買的M82命名為'''M107'''，並用錢購買此型以頂替原本的預算科目。2005年夏天，M82終於進入陸軍測試，並核准備料生產，正式成為官方採用，全名為'''長程狙擊步槍.50口徑，M107'''（Long Range Sniper Rifle，Caliber .50，M107）。M107使用{{Link-en|Leupold公司|Leupold & Stevens|Leupold}} 4.5x14「Vari-X」瞄準鏡。

巴雷特M107是一種.50口徑的[[半自動步槍|半自動]][[狙擊步槍|狙擊步槍]]，像M82一樣有退縮式槍管吸收後座力所以後座力和槍支尺寸不成比例的小，每一次射擊後退的內部機構都由後方的彈簧吸收掉後座力。此外槍支本身[[重量|重量]]和[[砲口制動器|大型槍口制退器]]也吸收不少[[後座力|後座力]]。M82A1做了許多修改才變成M107，新特徵就是加長的[[戰術導軌|戰術導軌]]，後握把和腳架插口。巴雷特近日還打算研發一種輕型版M107比所有「[[反器材步槍|反器材步槍]]」都輕，預計會重新打造槍身框架和內部部件以及用輕型材料製造[[砲口制動器|槍口制動器]]。

巴雷特M107像M82一樣，繼承巴雷特「輕50」的名號。並且在許多方面擠下了M82，像是M107就獲選「美軍2005年十大軍事發明」。<ref>[http://www.policeone.com/police-products/firearms/precision-rifles/press-releases/116526/ Police Precision Rifle Press Releases<!-- Bot generated title -->] {{webarchive|url=https://web.archive.org/web/20071014045606/http://www.policeone.com/police-products/firearms/precision-rifles/press-releases/116526/ |date=2007-10-14 }}</ref>

=== 巴雷特M107CQ ===
新型的M107是一種商業研發產品，M107CQ是一種特別設計的.50狙擊步槍，用以解決過重問題。M107CQ槍管縮短了228.6毫米（9英吋），也比M107輕2.27公斤（5磅），設計者說M107CQ適合用於[[直升機|直升機]]和船艦防衛、斥侯偵查車、都市近戰等方面<ref name="M"/>。但亦有異說為名字是巴雷特M82A1CQ<ref name="M82a1cq"/>。

=== 巴雷特M107A1 ===
2010年10月，巴雷特非官方的宣佈M107已停產，而在2011年，該公司宣佈M107A1為它的繼承者，並提供於商業銷售。顯著的改進包括：減少5磅重量，一個新的圓柱形鈦合金槍口制動器、鈦合金槍管鑰匙及後座緩衝系統，這令武器能夠直接裝上巴雷特設計的[[抑制器|抑制器]]<ref>[http://www.barrett.net/pdfs/M107A1-Sales-Sheet.pdf<!-- Bot generated title -->] {{Cite web |url=http://www.barrett.net/pdfs/M107A1-Sales-Sheet.pdf |title=存档副本 |access-date=2017-07-20 |archive-url=https://web.archive.org/web/20110123041433/http://www.barrett.net/pdfs/M107A1-Sales-Sheet.pdf |archive-date=2011-01-23 |dead-url=yes }}.</ref>。

== 技術性能 ==
[[File:USMC_Barrett_M82A3.jpg|thumb]]使用M82A3 .50口徑半自動長射程狙擊步槍]]
[[File:Marine_with_M82.JPEG|thumb]]
M82是[[后座作用|槍管後座式]][[半自動步槍|半自動槍械]]，當擊發時槍管將會短距縮回（約一吋25mm）後由迴轉式槍機安全鎖住。短暫後退後，槍栓被推入彎曲軌再扭轉把槍管解鎖。當栓機解鎖，槍機拉臂瞬間退回，槍管轉移後座力的動作完成循環。之後槍管固定且栓機彈回，彈出彈殼。當[[撞針|撞針]]歸位，槍機從彈匣引出一顆子彈並送進[[膛室|膛室]]對準槍管。扳機彈回撞針後方位置。此槍的[[彈匣|彈匣]]一次可裝10發，雖然也有12發彈匣的，但是[[海湾战争|第一次波灣戰爭]]後就沒有生產了。

膛室分兩部份（上部和下部），薄鋼板衝壓而成再用十字栓固定。[[槍管|槍管]]有凹孔增加散熱和減重，還裝有大而有效的[[砲口制動器|槍口制動器]]。早期的槍口制動器是圆柱箭頭，新的M82裝有雙膛直角箭頭形（V形）制動器。這制動器減少了接近70%的後坐力。其缺陷是每發射一發槍彈時從制退器噴出的火藥氣體都會在射手附近卷起大量塵土和鬆散顆粒。

M82A1可使用瞄準鏡和摺疊式[[準星|準星]]來瞄準，以備瞄準鏡損壞時還能使用。美軍M82通常裝備{{Link-en|Leupold公司|Leupold & Stevens|Leupold}} Mark 4[[瞄準镜|望遠瞄準鏡]]，而M82A1M（海軍陸戰隊M82A3）有長型[[戰術導軌|戰術導軌]]和美軍[[望远镜|光學望遠鏡]]所以可用更多的瞄準[[配件|配件]]。所有的M82都有可摺疊式[[提把|提把]]和[[兩腳架|兩腳架]]（M82A3中兩者可以拆下）。M82A3有分離式[[槍托|槍托]]，槍托內有柔軟填充物和特种橡胶后座垫可以抵銷射擊後座力，据说后座力比[[霰彈|12鉛徑霰弹]]和7.62公釐口径的步枪更小。M82A1和M82A3能裝於M3或M122[[三腳架|三腳架]]（機槍用的）或是一些載具車輛上的「巴雷特軟拖架」。M82A1有摺疊式準星，然而實戰中M82很難用傳統準星瞄準，因為重量和長度都太大。

M82A2不同於M82A1的地方大多數在於結構：[[握把|握把]]和[[扳機|扳機]]在[[彈匣|彈匣]]之前，[[槍托|槍托]]提前到膛室後面一點處，幾乎靠近彈匣。外加的前握把在靠近槍管的下方，瞄準鏡也往前移了一段距離。

== 爭議 ==
[[File:Barrethit2.jpg|thumb]]]]
.50口徑的子彈，尤其是全金屬套的子彈，可以輕易打穿四級以下的[[防彈衣|防彈衣]]和許多牆壁。此外加的動能使M82比同級槍更能穿透硬[[裝甲|裝甲]]和等級8的[[防弹玻璃|防弹玻璃]]打死目標（有可能的），這些能力使M82被槍枝管制論的支持者列為大敵，[[加利福尼亚州|加州]]已經率先於其他州之前立法禁止.50口徑槍枝販賣給私人槍枝擁有者，規定巴雷特須改提供特製口徑而有類似效能的槍枝，[[.416_Barrett|.416 Barrett]]的口徑體積較小，初速較高，以及改良的彈道系數使其貫穿力較一般步槍為佳，但是槍枝管制論者依然認為不滿，他們認為沒有任何公民的「正常需求」中會需要使用這種大型槍枝。

因為上述對.50口徑步槍的爭議所帶來的問題和壓力，[[巴雷特|巴雷特]]最後中止對加州的[[執法機關|執法機關]]出售所有槍械。<ref>Barrett, Ronnie. [http://www.thegunzone.com/rkba/rkba-50.html ''Draws The Line,'' '''The Gun Zone''' web site] {{webarchive|url=https://web.archive.org/web/20080720011401/http://www.thegunzone.com/rkba/rkba-50.html |date=2008-07-20 }} April 27, 2005, retrieved July 15, 2008.</ref>

另外.50口徑槍械的爭議點在於其尺寸重量成本都大的槍族，對於一般罪犯來說太難使用，但是卻助長了外國和美國本土的[[恐怖组织|恐怖组织]]（例如[[阿富汗|阿富汗]]、[[伊拉克|伊拉克]]等戰區）他們已經從美軍倉庫中偷走不少M82，反對論者藉此認為一開始就不要生產這種槍就沒有這種麻煩。

巴雷特M82已經被美軍大量使用於實戰，此槍也是最初的[[反器材步槍|反物資步槍]]，但是太大及太重也使其很難用於機動戰中的支援火力。

近來有報導敘述有不少[[黑市|黑市]]中的巴雷特.50狙擊步槍開始在墨西哥中用作協助運[[毒品|毒]]及[[犯罪|犯罪]]，以對抗當地穿上防彈衣的[[警察|警察]]部隊<ref>[http://edition.cnn.com/2008/WORLD/americas/03/26/gun.smuggling/index.html CNN]</ref><ref>{{cite web |url=http://www.vpc.org/snipercrime.htm |title=存档副本 |accessdate=2009-03-22 |0=2011-12-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20111202141310/http://www.vpc.org/snipercrime.htm |archivedate=2011-12-02 }}</ref>。

== 流行文化 ==
巴雷特M82同時出現於許多[[好莱坞|好莱坞]][[电影|电影]]中、[[电視|电視]]、[[电脑游戏|电脑游戏]]和[[动画|动画]]裡，但各[[电影|电影]]都有所誇大其性能，誤解包含有能打下[[民航飛機|客機]]。實際上M82-{只}-能打中停放的[[飛機|飛機]]，只有超乎常人的射手才有可能在10發[[彈匣|彈匣]]內打落高速飛行的飛機（假設在射程內）。在[[导弹|飛彈]]出現之前的[[第二次世界大战|第二次世界大战]]中，.50口徑第一次成功用於空戰的空對空射擊，即使有許多缺點，[[美国军事|美軍]]還是堅持用到1950年代，例如[[F-86軍刀戰鬥機|F-86戰鬥機]]上就有6挺[[白朗寧M2重機槍|.50機槍]]。而即使如此，也不代表零速度的人可以在地上用10發彈匣就打掉空中高速飛機，如果是固定盤旋的[[直升機|直升機]]還稍有可能。
