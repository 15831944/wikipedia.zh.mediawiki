[[File:Beelzebub_and_them_with_him.jpg|right]]》中的巴力西卜]]
'''巴力西卜'''（Beelzebub 或 Beel-Zebub），又譯為'''別西卜'''，天主教譯為'''贝耳则步'''，意思為“蒼蠅王”，[[腓尼基人|腓尼基人]]的神，[[新約聖經|新約聖經]]中稱巴力西卜為「'''魔王'''」，[[聖經|聖經]]中[[七宗罪|七宗罪]]的暴食。但是在[[拉比|拉比]]（Rabbi，[[犹太教|犹太教]]的宗教领袖）的文献中，别西卜这个名字也以“蒼蠅王”的意思在使用，被视为是引起[[疾病|疾病]]的恶魔。傳說祂原本是[[熾天使|熾天使]]中聖歌隊的成員，在[[創世紀|創世紀]]戰爭中失敗逃走。[[惡魔學|惡魔學]]裡認爲他是[[地獄|地獄]]的最高統帥，[[約翰·彌爾頓|約翰·彌爾頓]]（John Milton）將其描繪成是外貌威嚴的智者，是最強大的[[墮天使|墮天使]]之一。

== 異教的起源由來 ==
[[巴力|巴力]]（'''Baal'''）在原意上是[[美索不達米亞|美索不達米亞]]人稱神的用法，而'''Zebub'''是爲王子之意，可能是[[腓尼基|腓尼基]]人的聖典中的[[太陽神|太陽神]]，後來卻被[[以色列人|以色列人]]改成了'''惡心'''、'''邪惡'''的代名詞。這名惡魔也曾出現在[[希臘神話|希臘神話]]中。

'''巴力西卜'''（BaalZebul）在[[巴勒斯坦|巴勒斯坦]]的名氣僅次於[[路西法|路西法]]，常將他們相提並論。

== 文學中的形象 ==
《[[失樂園|失樂園]]》中寫著別西卜由[[熾天使|熾天使]]墮天後，在撒旦名號的爭奪戰之中有很高的聲望，因此他的名氣只在[[路西法|路西法]]之後，地位就等同於地獄宰相，別西卜有著統領眾鬼的權力，所以被稱為鬼王，而蒼蠅王的形態亦是很有名的。
