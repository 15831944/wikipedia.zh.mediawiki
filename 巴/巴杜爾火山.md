'''巴杜爾火山'''（{{lang-en|'''Mount Batur'''}}、{{lang-id|'''Gunung Batur'''}}）是一座位於[[印度尼西亞|印度尼西亞]][[巴厘島|巴厘島]][[阿貢火山|阿貢火山]]以北的同心火山口火山，該火山目前屬活火山。較大的火山口面積達{{convert|10|km}}×{{convert|13|km}}，其東南側有一[[火山湖|火山湖]]名為巴杜爾湖。內側的火山口寬{{convert|7.5|km}}，從島上採集到的{{le|熔結凝灰岩|Ignimbrite}}推測，其形成時期約在23,670或28,500年前。<ref name=Smithsonian>{{cite web |url=http://volcano.si.edu/volcano.cfm?vn=264010 |title=Batur General Information |publisher=Smithsonian Institution |work=Global Volcanism Program, National Museum of Natural History |accessdate=January 16, 2016 }}</ref>該火山最早的有紀錄的噴發日期在1804年，最近一次的噴發日期在1999年3月至2000年6月間。<ref name=Eruptions />

==簡介==
巴杜爾火山目前屬於活火山，其特徵是有兩個大型火山口以同心圓方式重疊在一起。內緣的火山口東南側與剛好與巴杜爾湖重疊，並已觀察到有多處新生的火山道在內側火山口，但在NE-SW裂隙噴發口定位系統仍只標示峰頂的3處，分別為巴杜爾1、巴杜爾2、巴杜爾3火山口。歷史紀錄中，巴杜爾火山的爆發程度在輕度至中度等級之間，有時會伴隨著[[熔岩|熔岩]]流。研究顯示，歷史上來自頂峰和側翼火山口的[[玄武岩|玄武岩]]熔岩，流動範圍至火山腳下，以及巴杜爾湖沿岸。<ref name=Smithsonian />

火山口中央包含一座高{{convert|700|km}}的複式火山，其高度超過巴杜爾湖。該火山最早的有紀錄的噴發日期在1804年，並一直持續地有噴發紀錄，最近一次得噴發記錄在1999年3月至2000年6月。<ref name=Smithsonian /><ref name=Eruptions>{{cite news |url=http://www.thejakartapost.com/news/2009/11/11/mt-batur-alert-raised-caution039.html |title=Mt. Batur alert raised to `caution' |date=November 11, 2009 |newspaper=Jakarta Post |accessdate=January 16, 2016 |author=Desy Nurhayati }}</ref>1968年噴發的一次，大量的熔岩流從火山口流出，時至今日，在西南方山脊的{{le|金塔馬尼|Kintamani, Bali}}鎮仍能看到這些痕跡。<ref>{{cite web |url=http://www.ecoroc.com.au/files/pdf/Bali-Geopark-Land-Cruiser-Tour_Jun-15.pdf |title=The Bali Batur Global Geopark & FAQ's (frequently asked questions) |date= |website=Ecoroc |accessdate=January 16, 2017 |author= }}</ref>

==人口分布==
目前有許多人居住在火山口，共有15處村莊在火山口範圍內，包括葛帝山（Kedisan）、松干（Songan）、特魯揚（Trunyan），以及托亞邦卡（Toya Bungkah）等4個主要村莊。 當地人大多依靠農業收入，但由於鄰近火山口，而且觀賞火山的遊客增多，此處的旅遊業近期越來越受歡迎。<ref>{{cite book |last=Langston-Able |first=Nick |title=Playing with Fire: Adventures in Indonesia |publisher=Freakash |year=2007 |location = |pages=85–98 |url=http://www.Adventures-in-Indonesia.co.uk  |isbn=978-0-9553403-4-5 }}</ref>

==世界地質公園網絡==
2012年9月20日，[[聯合國教科文組織|聯合國教科文組織]]將巴杜爾火山列入[[世界地質公園|世界地質公園]]的一部分。<ref>{{cite web |url=http://www.pikiran-rakyat.com/node/205420 |title=Kaldera Gunung Batur Sebagai Taman Bumi Global |trans-title=Caldera of Mount Batur As Global Earth Park |date=September 30, 2012 |accessdate=January 16, 2016 |publisher=Pikiran Rakyat |quote=the geopark is an award from UNESCO for park managers who are able to implement conservation earth geology and also use it as a tourist attraction.}}</ref><ref>{{cite web |trans-title=UNESCO Set Mount Batur Caldera as Global Earth Garden Network |date=September 30, 2012 |author=Nyoman Modern |publisher=Voice of America (Bahasa Indonesia) |title=UNESCO Tetapkan Kaldera Gunung Batur sebagai Jaringan Taman Bumi Global |url=http://www.voaindonesia.com/content/unesco-tetapkan-gunung-batur-sebagai-taman-bumi-global/1517556.html |accessdate=January 16, 2016}}</ref>

==沉積作用==
巴杜爾火山嚴重的沉積作用導致巴杜爾湖水位減少，當局計畫以限制湖泊周圍的旅遊住宿點，向居民宣導在湖泊用魚籠養殖的問題，或是將湖底的火山灰沉積物挖除等。<ref>{{cite web |url=http://www.thejakartapost.com/news/2008/11/19/damaged-lake-batur-approved-top-government-priority.html |title=Damaged Lake Batur approved as top government priority |accessdate=January 16, 2016 |date=November 19, 2008 |publisher=Jakarta Post |author=Luh De Suriyani}}</ref>由於巴杜爾湖受到漁業和農業用水的污染，導致[[鳳眼藍|鳳眼藍]]大量繁殖，加速水質惡化。一部分的人認為，若是將湖底厚重的火山灰沉積物挖除，或許可以消滅鳳眼藍。<ref>{{cite web |url=http://agroindonesia.co.id/index.php/2015/11/24/menyelamatkan-danau-batur-menyelamatkan-kehidupan/ |title=Menyelamatkan Danau Batur, Menyelamatkan Kehidupan |trans-title=Saving Lake Bakur, Saving Lives |date=November 24, 2015 |accessdate=January 16, 2016 |publisher=Agro Indonesia}}</ref><ref>{{cite web |url=http://denpostnews.com/2014/11/14/eceng-gondok-kian-luas/  |title=Eceng Gondok Kian Luas |trans-title=Increase in Water Hyacinth Size |publisher=Denpost Theme |location =Bali |date=November 14, 2014 |accessdate=January 16, 2016}}</ref>
