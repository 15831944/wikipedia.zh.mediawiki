'''巴哈姆特動畫瘋'''是由[[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]]於[[台灣|台灣]]運營的[[OTT|OTT]][[動畫|動畫]]播放平台，並與[[中華電信|中華電信]]合作，採用中華電信[[影音串流|影音串流]]服務，動畫播放中允許用戶發出[[视频弹幕|影片彈幕]]式的回覆訊息，服務力求能達到與日本當季動畫同步播出。<ref name="vote">{{cite web|url= https://gnn.gamer.com.tw/7/126137.html|title= 巴哈姆特免費動畫平台「動畫瘋」即日起正式上線！|publisher= [[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]]|date=2016-01-06 }}</ref><ref>{{cite web|url= http://www.techbang.com/posts/43199-update-faster-than-subtitling-unit-bahamut-animation-crazy-japan-when-ji-xin-animation-synchronized-broadcasting|title= 巴哈姆特「動畫瘋」，日本當季新番動畫同步放送|publisher= T客邦|date=2016-08-14}}</ref>

==成立由來==
巴哈姆特電玩資訊站2014年站內舉行的網路問卷調查中，有超過7成的受訪者願意付費收看網路平台的正版動畫。<ref name="vote"/>因此巴哈姆特開始評估網路平台動畫播放的可行性，並藉此數據尋求合作伙伴。2016年與國內動畫代理商合作，促成動畫瘋的成立。

== 歷史 ==
* 2016年1月6日，成立巴哈姆特動畫瘋。1月25日，取消會員需註冊滿3個月才能觀看限制，新增付費去廣告方案<ref>{{cite web |title=【公告】動畫瘋服務正式開啟 |url=https://forum.gamer.com.tw/C.php?bsn=60404&snA=24057|work=巴哈姆特電玩資訊站|date=2016-01-25}}</ref>。
* 2016年5月，正式推出APP版本<ref>{{cite web |title=免費線上動畫平台「動畫瘋」iOS、Android 版本 APP 正式推出 |url=https://gnn.gamer.com.tw/0/131660.html|work=巴哈姆特電玩資訊站|date=2016-05-11 }}</ref>。
* 2016年12月，APP名列[[Google_Play|Google Play]]的2016年最佳人氣應用程式。
* 2017年4月，公佈30日內非重複觀眾付費比例。
*2017年12月，[[Google_Play|Google Play]]公佈年度獎項，動畫瘋APP名列「最佳娛樂應用」中。<ref>{{cite news |title=Google 公布台灣 2017 年最佳應用程式 App 榜單！你下載了嗎？ |url=http://3c.ltn.com.tw/news/32156}}</ref>
* 2018年2月7日，PC版新增自訂彈幕過濾關鍵字功能；app版3月4日新增。
* 2018年9月19日，新增[[Android_TV|Android TV]]版。<ref>{{cite web |title=公告】動畫瘋 Android TV 版上線測試 |url=https://forum.gamer.com.tw/C.php?bsn=60404&snA=29946|accessdate=2018-12-22}}</ref>
* 2018年10月8日，因用戶增加，為維持觀看品質，原先註冊會員即可免費觀看的1080p畫質轉為付費服務。<ref>{{cite web |title=【站方公告】動畫瘋營運政策調整|url=https://user.gamer.com.tw/notice_detail.php?sn=606|work=巴哈姆特電玩資訊站|date=2018-10-08  }}</ref>
*2018年12月3日，Google Play公佈年度獎項，動畫瘋APP名列「2018年度最受歡迎應用程式」中。<ref>{{cite news |title=Google Play年度最佳榜單出爐！你愛的app得獎了嗎 |url=https://tw.appledaily.com/new/realtime/20181203/1477156/}}</ref>
* 2018年12月21日，新增[[Apple_TV|Apple TV]]版。<ref>{{cite web |title=動畫瘋Apple TV版本上線 |url=https://www.facebook.com/animategamer/posts/2085731514822542?__tn__=-R |accessdate=2018-12-22}}</ref>

== 服務 ==
巴哈姆特動畫瘋服務授權範圍僅限[[台澎金馬地區|台澎金馬地區]]，服務區分為「含廣告免費收看」與「無廣告付費收看」，未登入註冊訪客可使用360p畫質觀看，免費用戶可以最高[[720p|720p]]觀賞，付費用戶可進一步達1080p畫質，並使用[[Android_TV|Android TV]]、[[Apple_TV|Apple TV]]的支援。
