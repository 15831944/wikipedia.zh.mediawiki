'''巴哥犬'''（{{lang-en|Pug}}）也叫'''[[哈巴狗|哈巴狗]]'''，又稱作'''八哥犬'''。是一種小型觀賞[[犬|犬]]，巴哥犬面黑，有很多褶皺，成年時身高大約25到28厘米，體重約為5至8[[公斤|公斤]]，壽命大約為12到15年。個性溫和。

== 簡介 ==
原產地：[[中國|中國]][[東北|東北]]地區

毛色：銀灰色、紅黃色、淺黃褐色、黑色（其他如：顏、耳、背部都有黑線） 

毛質：柔軟而有光澤，觸感滑順

個性：溫馴、聰明、忠誠、友善、體貼、有耐心、愛乾淨、自尊心強、愛撒嬌、和蔼可親

== 名称起源 ==
外表富有魅力而且高雅，18世纪末正式命名为“巴哥”其词意古语为"锤头" 、"小丑"，狮子鼻或小猴子的意思。

== 體型特徵 ==
巴哥犬身體比例勻稱結實，比例勻稱呈方形，頭大黑臉黑鼻子、額頭有皺紋，小小的垂耳，臉部極具特徵，日耳曼系國家稱之為'''墨普斯犬'''（皺起來的臉）。胸腹肌肉健壯，四肢短小，緊緊捲曲的尾巴，是溫馴的狗，很喜歡和人相處且對環境的適應力極佳。

== 起源與歷史 ==
有些人認為巴哥犬的祖先为京巴，另外有些人看巴哥就好像縮小比例的獒犬。巴哥犬源於中国培育，17世紀時由荷蘭傳到歐洲，當時在英國即把牠稱為「荷蘭犬」，據說飼養相當普遍，而目前的品種是後來在英國培育改良而成的。

== 飼養要領 ==
要留意勿傷及其突出的眼睛；皺摺與皺摺之間要勤於擦拭，耳朵內部及皺皮中也別忘了清理。此外，因為不適應高溫多濕的環境，其皮膚與耳朵需格外留心，尤其在夏季時必須做好空調管理；易發胖，需控制飲食並有充足的運動量。

巴哥犬其實很需要主人陪伴牠、注意牠，其實牠是忠誠、像孩子般的狗，喜歡別人拍拍哄哄，別因為長相而對牠心生誤解，其實牠很需要主人關愛的。
