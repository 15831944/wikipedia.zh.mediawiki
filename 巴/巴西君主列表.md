==作为[[葡萄牙|葡萄牙]]殖民地的[[巴西|巴西]]==
===葡萄牙王国巴西亲王===
布拉干萨王朝的葡萄牙国王将巴西亲王的称号分封给自己的继承人，以下是巴西亲王的名单。
*1645-1653：[[提奥多西奥三世_(布拉干萨)|葡萄牙的提奥多西奥王太子]]   <span style="font-size:smaller;">葡萄牙国王[[若昂四世|若昂四世]]长子，布拉干萨公爵</span>
*1656-1683：[[阿方索六世_(葡萄牙)|葡萄牙的阿方索王子]]   <span style="font-size:smaller;">1656年成为葡萄牙国王阿方索六世</span>
*1688：葡萄牙的若昂王太子    <span style="font-size:smaller;">葡萄牙国王[[佩德罗二世_(葡萄牙)|佩德罗二世]]长子，早夭</span>
*1697-1706：[[若昂五世|葡萄牙的若昂王子]]    <span style="font-size:smaller;">1706年成为葡萄牙国王若昂五世</span>
*1711-1712：[[芭芭拉王后_(西班牙)|葡萄牙的玛利亚·芭芭拉长公主]]    <span style="font-size:smaller;">若昂五世长女，后来嫁给[[西班牙君主列表|西班牙王]][[费尔南多六世_(西班牙)|费尔南多六世]]</span>
*1712-1714：葡萄牙的佩德罗王太子    <span style="font-size:smaller;">葡萄牙国王[[若昂五世|若昂五世]]长子，早夭</span>
*1714-1750：[[若泽_(葡萄牙)|葡萄牙的若泽王子]]    <span style="font-size:smaller;">1750年成为葡萄牙国王若泽一世</span>
*1750-1777：[[玛丽亚一世_(葡萄牙)|葡萄牙的玛丽亚·弗朗西丝卡长公主]]    <span style="font-size:smaller;">1777年成为女王玛丽亚一世</span>
*1760-1777：[[佩德罗三世_(葡萄牙)|葡萄牙的佩德罗王子]]    <span style="font-size:smaller;">女王玛丽亚一世的叔叔兼夫婿</span>
*1777-1788：葡萄牙的若泽（二世）王子   <span style="font-size:smaller;">葡萄牙女王玛丽亚一世长子，布拉干萨公爵</span>
*1777-1829：葡萄牙的本妮蒂塔公主        <span style="font-size:smaller;">布拉干萨公爵若泽二世的姑姑兼妻子，若泽二世去世后一直保有巴西亲王夫人的称号</span>
*1788-1816：[[若昂六世|葡萄牙的若昂·玛利亚王子]]    <span style="font-size:smaller;">1816年成为葡萄牙国王若昂六世</span>
*1816：[[佩德罗一世_(巴西)|葡萄牙的佩德罗王太子]]   <span style="font-size:smaller;">1822年成为巴西的第一任皇帝</span>

===[[葡萄牙、巴西和阿尔加维联合王国|葡萄牙、巴西和阿尔加维联合王国]]国王===
[[拿破仑|拿破仑]]战争时代，葡萄牙本土被[[法国|法国]]占领，葡萄牙王室流亡[[巴西|巴西]]，葡萄牙也更名'''葡萄牙、巴西和阿尔加维联合王国'''。葡萄牙国王兼任'''巴西国王'''。1816年，[[佩德罗一世_(巴西)|佩德罗王太子]]自称葡萄牙、巴西和阿尔加维联合王国王室亲王、巴西摄政王统治葡萄牙殖民地巴西。
*1815-1816：[[玛丽亚一世_(葡萄牙)|玛丽亚一世]]
*1816-1825：[[若昂六世|若昂六世]]

==独立后的巴西==
1822年9月7日，巴西摄政王[[佩德罗一世_(巴西)|佩德罗]]宣布巴西独立，并宣称“不独立，毋宁死”（'''''Independência ou Morte!'''''）。同年10月12日，佩德罗成为'''巴西皇帝佩德罗一世'''。
===巴西皇帝===
*1822-1831：[[佩德罗一世_(巴西)|佩德罗一世]]    <span style="font-size:smaller;">1826年兼任葡萄牙国王，称佩德罗四世</span>
*1831-1889：[[佩德罗二世_(巴西)|佩德罗二世]]
1889年11月15日，佩德罗二世在军队与共和党人联合发动的政变中被推翻，巴西君主制覆亡。

====巴西榮譽皇帝====
雖然[[若昂六世|若昂六世]]允許巴西獨立，但也自封自己「巴西榮譽皇帝」。自稱「國王兼皇帝若昂六世陛下」(Sua Majestade o Imperador e Rei Dom João VI)。

==巴西王位觊觎者==
*1889-1891：佩德罗二世
*1891-1921：[[伊莎贝尔_(巴西)|伊莎贝尔（一世）]]   <span style="font-size:smaller;">佩德罗二世长女</span>
*1921-1940：[[佩德罗·恩里克_(巴西)|佩德罗（三世）]]    <span style="font-size:smaller;">伊莎贝尔的孙子</span>
