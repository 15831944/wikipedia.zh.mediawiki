'''巴拉圭共和國'''（{{lang-es|'''República del Paraguay'''}}，{{lang-gn|'''Tetã Paraguái'''}}）  通稱'''巴拉圭'''，是[[南美洲|南美洲]][[內陸國家|內陸國家]]，地據[[巴拉圭河|巴拉圭河]]兩岸。其南邊國境完全與[[阿根廷|阿根廷]]接壤，東北與西北角則分別是[[巴西|巴西]]與[[玻利維亞|玻利維亞]]，為[[南美洲國家聯盟|南美洲國家聯盟]]成員國。首都為[[亞松森|亞松森]]。巴拉圭在西班牙語中是“鹦鹉”的意思。

== 歷史 ==
{{main|巴拉圭歷史}}
巴拉圭原為[[印地安人|印地安人]][[瓜拉尼族|瓜拉尼族]]定居之地，1525年由[[葡萄牙|葡萄牙]]人[[阿雷荷·加西亞|阿雷荷·加西亞]]（Alejo García）發現，1537年[[西班牙|西班牙]]殖民團體首建亚松森市。5年後天主教[[耶穌會|耶穌會]]傳教士也前往该地傳教並教授當地人農牧及手工藝。1767年西班牙殖民者把耶稣会教士驱逐。18世纪，巴拉圭地區是南美洲最大的人口集中地區。

1811年5月15日時巴拉圭脫離西班牙独立，由[[何塞·加斯帕爾·羅德里格斯·德·弗朗西亞|罗德里格斯]]遂行獨裁統治。1840年法學教授[[卡洛斯·安东尼奥·洛佩斯|卡洛斯·安东尼奥·洛佩斯]]繼任後，始採較開放之政策，從事農工商業各項建設。1844年巴拉圭再設議會，已初具獨立國家規模。

洛佩斯之子[[弗朗西斯科·索拉诺·洛佩斯|弗朗西斯科·索拉诺·洛佩斯]]於1862年繼掌政權，醉心[[軍國主義|軍國主義]]。1865年至1870年間，巴拉圭發動了拉丁美洲史上最慘烈的戰爭——[[巴拉圭戰爭|巴拉圭戰爭]]，在5年之間，巴拉圭因為與[[阿根廷|阿根廷]]、[[巴西|巴西]]和[[烏拉圭|烏拉圭]]三國所合組的聯盟對抗作戰，折損了全國約2/3的成年男子人口與諸多领土，也造成該國在戰後半世經濟停滯。

1932年至1935年的[[大厦谷战争|大厦谷战争]]中，巴拉圭战胜[[玻利维亚|玻利维亚]]，從玻國取得大片[[大廈谷|大廈谷]]地區。翌年巴玻戰爭之統帥[[何塞·费利克斯·埃斯蒂加里维亚|埃斯蒂加里维亚]]出任總統，旋於1940年8月重頒憲法，分區治理，政治漸上軌道。

1947年，巴國發生為期六個月之內戰後由[[红党_(巴拉圭)|紅黨]]執政。1954年，屬紅黨的[[阿尔弗雷多·斯特罗斯纳|斯特罗斯纳將軍]]發動政變，同年9月當選總統，嗣獲一再連任迄1989年2月，始為[[安德烈斯·罗德里格斯|罗德里格斯將軍]]發動軍事政變推翻。从此開始實行民主制度选举总统。

== 政治和政府 ==
巴拉圭實行[[民主共和制|民主共和制]]，採[[行政|行政]]、[[立法|立法]]、[[司法|司法]][[三權分立|三權分立]]，[[總統|總統]]是[[國家元首|國家元首]]及政府最高決策者，是由人民選舉產生，任期5年。[[國會|國會]]由[[巴拉圭参议院|參議院]]及[[巴拉圭众议院|眾議院]]兩院組成，現有參議員45人、眾議員80人。

巴拉圭的民主政制始於1989年軍事政變，1989年5月大選舉後軍方即還政於民，軍方雖然仍具影響力，但均己適應民主選舉。1993年5月9日巴國舉行總統大選，由執政紅黨所推舉之候選人[[胡安·卡洛斯·瓦斯莫西|瓦斯莫西]]當選，並於同年8月15日就職，成為巴40年來第一位民選文職總統，巴國遂轉為文人政府。1998年5月10日巴拉圭舉行總統大選，由執政紅黨所推舉之候選人[[勞爾·庫瓦斯·格勞|庫瓦斯]]（Raul Cubas Grau）及副總統候選人阿加尼亞（Luis Maria Argana）當選，於8月15日就職。1999年3月間副總統阿氏遇刺身亡，造成政局不安，總統庫瓦斯因與軍事強人歐威多將軍涉及暗殺案而被迫宣佈辭職流亡巴西。巴拉圭政府依憲法由國會議長[[路易斯·岡薩雷斯·馬基|岡薩雷斯]]繼任總統。他的任期經巴拉圭最高法院裁定至2003年8月，並籌組由紅、藍及國家組合黨組成之聯合政府。2000年8月巴舉行副總統補選，由反對黨[[真正激進自由黨|真正激進自由黨]]（藍黨）候選人佛朗哥（Julio Cesar Franco）當選。

上任總統[[尼卡諾爾·杜阿爾特·弗魯托斯|尼卡諾爾·杜阿爾特·弗魯托斯]]於2003年當選，屬紅黨，上任後致力於政經改革，微有收效。

2008年4月，[[費爾南多·盧戈|費爾南多·盧戈]]在大選勝出，成為新任總統，並邀請自己的親生姐姐擔任[[第一夫人|第一夫人]]。在未擔任總統前，盧戈原是一位[[主教|主教]]，在該國廣大的貧窮人口中擁有非常高的支持度<ref>{{cite news|archiveurl=https://web.archive.org/web/20080430165910/http://hk.news.yahoo.com/080426/12/2t2ud.html |archivedate=2008年4月30日 |url=http://hk.news.yahoo.com/080426/12/2t2ud.html |title=主教總統邀姊當第一夫人 |date=2008年4月27日 |accessdate=2015年6月14日 |deadurl=yes }}</ref>。

2012年6月發生警察與農民的武裝衝突事件，為此事件[[費爾南多·盧戈|費爾南多·盧戈]]被参議院舉行一場院內投票後彈劾下台。[[費德里科·弗朗哥|費德里科·弗朗哥]]為當時的副總統，發生此事件後擔任臨時總統，直到[[2013年巴拉圭大選|2013年總統選舉]]後新任總統[[奥拉西奥·卡特斯|奥拉西奥·卡特斯]]就職。

費爾南多·盧戈被彈劾下台的事情遭到許多鄰近國家以及[[南美共同市場|南美共同市場]]的反對認為此決定未遵守程序且違反民主。

== 外交 == 
{{Main|巴拉圭外交|中華民國－巴拉圭關係}}

自[[烏拉圭|烏拉圭]]於1988年2月4日與中華民國斷交後，巴拉圭是[[南美洲|南美洲]]唯一与[[中華民國|中華民國]]拥有外交关系的国家，也是中華民國目前17個友邦中面積最大者，雙方是於1957年7月8日締結邦交。1988年7月，中華民國政府在[[東方市_(巴拉圭)|東方市]]設立[[中華民國駐東方市總領事館|總領事館]]。兩國合作關係密切友好，巴拉圭對中華民國參與國際事務活動向來堅定支持，每年[[聯合國大會|聯合國大會]]總辯論時都明確為中華民國執言。[[中華民國政府|中華民國政府]]多年來支持巴拉圭政府興建貧民住宅計畫，完成之貧民住宅已经遍佈全國各省；2016年[[蔡英文政府|蔡英文政府]]延續了2007年[[陳水扁政府|陳水扁政府]]與當時巴拉圭政府達成的承諾，資助[[新臺幣|新臺幣]]約23億金援巴拉圭的貧民住宅計劃。

== 地理 ==
{{main|巴拉圭地理}}
巴拉圭全國面積為406,752平方公里，恰巧位於南美洲的中心地帶。雖然巴拉圭為內陸國家，但[[巴拉圭河|巴拉圭河]]從北到南貫通國土中央，為國家[[經濟|經濟]]及[[交通|交通]]運輸起了重要作用。巴拉圭河將該國分為東西兩部份。西半部是廈谷地區（Chaco），面積約247,000平方公里，佔全國總面積61%，這個區域是大片[[草原|草原]]，與鄰國阿根廷、巴西、玻利維亞的草原地區構成[[大廈谷|大廈谷]]大草原。

東半部則以起伏的地形著稱，在一些河川下游區域多[[沼泽|沼泽]]地及肥沃土地，並提供林業、農業與牛隻養殖業的優良環境。

氣候方面，巴拉圭屬[[副热带湿润气候|副热带湿润气候]]，年平均氣溫為24.5℃，夏季為每年10月至翌年3月底，平均氣溫達31℃，但偶爾會有幾天超過40℃。冬季從6月至8月，平均氣溫在14.5℃左右。在[[伊泰普省|伊泰普省]]及[[上巴拉那省|上巴拉那省]]等地區或會在夜間時低於攝氏零度。

== 行政區劃 ==
{{main|巴拉圭行政區劃}}
巴拉圭共和國劃分為17個[[省|省]]。而[[首都|首都]][[亞松森|亞松森]]則被設為首都特別區（districto capital）。

=== 重要城市 ===
* 亞松森——巴拉圭共和國首都及最大城市。
* [[東方市_(巴拉圭)|東方市]]——巴拉圭第二大城市，位於該國東南邊境，與[[阿根廷|阿根廷]]及巴西邊境為鄰的重要商業城市，亦為南美洲知名的[[華人|華人]]及[[阿拉伯|阿拉伯]]僑民聚集地。

== 人口 ==

[[File:Paraguay_population_density.png|250px]]
巴拉圭全國約七百萬人口中，有接近97%集中聚居在東半部。只有約2至3%的人口居住在廣大的西部恰可地區。而全國51%人口都住在城市，其中首都亞松森佔全國人口11.2%。自1970年代以來，來自[[日本|日本]]、[[大韓民國|韓國]]及[[台灣|台灣]]的東亞移民，及來自西亞、巴西等地的移民，為該國的經濟注入力量。

與眾多拉丁美洲國家一樣，巴拉圭由於曾經長期受西班牙殖民統治，故有近95%人口為[[麥士蒂索人|麥士蒂索人]]（印歐混血兒），主要是西班牙人和當地瓜拉尼族人混合而生的後裔。

大多數人口信奉[[天主教|天主教]]，亦有少數是[[門諾宗|門諾宗]]教徒（集中於北部[[廈谷|廈谷]]（Chaco）地區，以農牧為生）或[[基督新教|基督新教]]教徒。

内陆国巴拉圭有一个庞大的非正规因素分化市场经济，把[[进口|进口]]消费品转口到周边国家，包括数以千计的微型企业和城市摊贩的交易。1970到2009年以每年7.2％的平均增长率成为经济增长最快的[[南美|南美]]国家。该国还拥有世界上第三大自由商业区：「[[東方市_(巴拉圭)|東方市]]」，紧随[[迈阿密|迈阿密]]和[[香港|香港]]之后。

另一个占人口很大比例特别是在农村地区，是来自农业活动的經濟，常常在生活的基础。由于非正规因素占经济的重要成分高，准确的经济数据很难获得。按人均计算，实际收入在1980年的水平停滞不前。在2003年和2008年之间巴拉圭经济快速增长，世界商品需求增长与高价格和有利的天气相结合，支持巴拉圭的商品扩大出口。巴拉圭是世界第六大[[大豆|大豆]]生产国。2008年遭受[[旱灾|旱灾]]，在全球经济衰退之前就开始农产品出口减少和经济放缓。

在2010年，巴拉圭目前正在进行该区域最大的经济扩张，也是南美洲最快的，达到14.5％的[[国内生产总值|国内生产总值]]增长率。

== 文化 ==
巴拉圭的官方語言為瓜拉尼語及[[西班牙語|西班牙語]]。
