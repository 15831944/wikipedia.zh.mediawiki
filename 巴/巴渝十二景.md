'''巴渝十二景'''是指[[中國|中國]][[重慶市|重慶市]]及其周圍多處受歡迎的旅遊景點，當中「巴」、「渝」兩字均為古時重慶市的簡稱。

[[明朝|明朝]][[天順_(明朝)|天順]]年間，受到[[湖南省|湖南省]]「[[瀟湘八景|瀟湘八景]]」 的影響，當地人民列出他們心儀的美景。列表隨年代而有所變化，一些之前在名單內的地方因天然或人為的改變已不復存在，同時又因環境變遷加入新的景點。

== 渝城八景 ==
該清單最初稱為「渝城八景」，景點數目與「瀟湘八景」相同。在明[[天順_(明朝)|天順]]二年（1453年），人們將名單刻在崇音寺（後更名為長安寺）的銅鐘上，原址現為重慶市第25中學。該八處景點為：

# 金碧香風：金碧山是重慶的最高點，可俯望全城美景。該處現在是[[渝中區|渝中區]]人民公園的一部份。金碧的意思是金色和藍色。宋朝年間，官吏在山頂打造了一座由金色和藍色瓷磚製成的金碧台。站在台中，清風徐來，暗香撲面，四處卻無草木花開，所以得名。
# 洪岩滴翠：洪岩即洪崖，位於渝中區靠近河邊且臨近重慶市中心的部份。懸崖上有一個洞穴，泉水從穴中流出滴下懸崖，水珠被陽光映透得如粒粒綠珠碧玉，故得此雅號。今天，那裏以[[吊腳樓|吊腳樓]]羣而聞名。
# 龍門浩月：流經重慶的長江，在過[[海棠溪街道|海棠溪]]後，被靠南岸的一排隆起的石梁劈為內外兩流。外流洶湧湍急，內流被卻浩如平鏡，內外不同如此分明，嘆為觀止。長江在此轉了一個大彎，形似彎月，古人稱這波平如鏡的水灣為「浩」。另凸出江面的石梁酷似一尊微微向外流傾斜的[[龍|遊龍]]，順著江勢連綿而居，中間開處正好疏通著內外水系，形如龍門，[[南宋|南宋]]便有人在石梁內外兩側刻上「龍、門」兩字，「龍門浩」就此得名。原址現為[[南岸區|南岸區]][[南濱公園|南濱公園]]的一部分。
# 黃葛晚渡：今天的南岸宏聲路附近，古時為擺渡渡口，連接重慶南北兩岸，因候渡者有[[黃葛樹|黃葛樹]]遮蔭而得名。20世紀80年代[[重慶石板坡長江大橋|石板坡長江大橋]]建成，取代了黃葛渡口，渡輪已不復在。現在這個場景人工保存，並於2005年成為南濱公園的一部分。
# 佛圖夜雨：佛圖關是重慶市中心陸路交通的唯一出口，靠近現在的[[鵝嶺公園|鵝嶺公園]]，是古代抵禦來自西部威脅的一個重要的軍事要塞。夜雨寺為佛圖關一主要寺廟，其名來自寺前的一塊岩石，每晚都有水從岩石滲出，被地熱化為蒸氣，尤如夜雨，使佛圖關煙雨朦朧，彷彿江上仙境。2009年5月，為修建渝中區職教中心的實習操作樓，夜雨寺已被拆毀。
# 孔殿秋香：於今天的重慶第29中學位置，在明朝時是一座孔廟。每到秋天，遍地桂花盛開，蔚為美景，也為寺廟帶來清香。
# 覺林曉鐘：「覺林」指位於[[南岸區|南岸區]]隱於風景如畫的蓮花山麓的覺林寺，始建於宋代。 每天早上，寺廟中的僧人都會響鈴（就像中國佛教寺廟中的其他僧侶一樣），鐘聲使人們感到快樂。明朝末年，寺廟和鍾聲在戰亂中摧毀。至[[清朝|清]][[康熙|康熙]]二年，雪痕禪師重建寺院，增修了山門和園林，構成了曉鐘喚渡，蓮峰垂露，龍橋夜月，泉水流香等[[覺林八景|覺林八景]]，成為盛極一時的觀光勝地。但至清[[光緒|光緒]]年間，覺林寺被清廷關閉。從此，寺廟漸損，終成廢墟，只留下一[[覺林寺報恩塔|報恩塔]]供人緬懷。
# 北鎮金沙：今[[江北區_(重慶市)|江北區]][[江北嘴遺址|江北嘴]]。[[明朝|明朝]]時，江北隸屬於巴縣江北鎮，故稱「北鎮」。冬季水涸時，北鎮在[[嘉陵江|嘉陵江]]和[[長江|長江]]匯合的北岸，河灘多有河沙鋪於岸邊，在陽光的照射下，沙子像黃金一樣閃耀，故稱「金沙」。

== 古巴渝十二景 ==
重慶是一個擁有多元化景觀和文化的主要城市，最初的八景無法呈現重慶的所有面貎。 因此，列表中的數量後來從八個擴展到十二個。第一次擴展成十二景是在清[[乾隆|乾隆]]期間發生的。1760年，當時的[[巴縣_(重慶)|巴縣]][[知縣|知縣]][[王爾鑒|王爾鑒]]在原來的八景中剔除三個，並在重慶及周邊地區增加了其他七個景點。王的巴渝十二景是：

# 金碧流香：即原來的「金碧香風」。見上一節。
#-{zh-hans:黄; zh-tw:黃}-葛晚渡：源自「渝城八景」。見上一節。
#-{zh-hans:统; zh-tw:統}-景峽猿：-{zh-hans:统; zh-tw:統}-景谷是[[渝北區|渝北區]]的一系列河谷，距重慶市中心約65公里。[[御臨河|御臨河]]穿過山脈，在這處形成了幾個深谷。統景原稱「桶井」，因谷-{zh-hans:内; zh-tw:內}-被峭壁包圍，形似桶狀，人在其中有坐井觀天之感，故得名。之後易為「統景」，也有「集自然山川之靈秀、統天下多種美景」之意。清朝以來，該區以大量[[猴|猴子]]棲居聞名。如今它則是[[溫泉|溫泉]]勝地。
# 歌樂靈音：[[歌樂山|歌樂山]]是[[沙坪壩區|沙坪壩區]]的山脈。今天，這座山以[[紅岩村|紅岩村]]而聞名，紅岩村是[[中國共產黨|中國共產黨]]的歷史紀念地。然而，在清朝，這個地方以天籟而聞名。相傳聲音源自山頂的雲頂寺。雲頂寺始建於明朝，寺內有一銅鐘，高四米，重三千餘斤。僧眾做法事鳴鐘，遠至廿里外都可聽到嘹亮的鳴聲，大殿屋簷上還掛著十二個銅鈴，風一吹，銅鈴發出清脆聲音，和著陣陣松濤翻滾十里，動聽之極。惜1950年代末，為修爐煉鋼，雲頂寺已被拆毀。現在松濤沒了銅鈴相伴，也就少了獨有的韻味。
# 雲篆風清：雲篆山距離今天的[[巴南區|巴南區]]魚洞約八公里。山上有雲篆寺，建於明朝[[成化|成化]]九年（1473年），是一座著名寺廟。重慶被認為是中國的爐灶城市之一<ref>{{Cite web|url=https://web.archive.org/web/20141112042247/http://www.gzkp.org.cn/show.aspx?id=180&cid=5|title=为什么重庆、武汉、南京有“三大火炉”之称？|accessdate=2018-12-06|date=2014-11-12|work=web.archive.org}}</ref>，夏季氣溫非常高。然而，由於海拔高，人們在雲篆寺總能感受到涼風颯颯。不幸的是，在[[文化大革命|文化大革命]]期間（1966年至1976年），寺廟被毀壞了。
# 洪崖滴翠：源自「渝城八景」。見上一節。
# 海棠煙雨：海棠溪是[[南岸區|南岸區]][[長江|長江]]的一條小支流河。 海棠溪起源於南山，長約三十英里，流入長江。溪邊以前長滿了海棠樹，每當春雨降臨，淡煙微布，細雨如絲，溪流映帶，其趣無窮，「海棠煙雨」由此得名。[[唐朝|唐代]]女詩人[[薛濤|薛濤]]在一個下雨的春日前往重慶，參觀了海棠溪，看到了在雨霧中綻放的海棠花美麗景象，然後寫了一首叫《海棠溪》的詩，使此美景廣泛流傳。今天，沿著小溪已沒有聚集的海棠花，但是在[[南濱公園|南濱公園]]保留了一部分面貎。
# 字水宵燈：因長江，嘉陵江在朝天門交匯處附近，流過一系列的岩石和礁石，受到這些岩石的干擾，河水以一種看起來像古篆體的「巴」字，故有「字水」之稱，而「巴」也是古時重慶的名字之一。夜幕低垂，遠處華燈初上，映在河中，點亮「巴」字，「宵燈」映「字水」，人們以前能夠從南岸可以看到這種景像。[[長江三峽水利樞紐工程|三峽大壩]]建設後水流減弱，水位上升，減弱了觀賞性。然而，這種景像仍可在南濱公園看到。
# 華鎣雪霽：[[華鎣山|華鎣山]]是[[四川盆地|四川盆地]]東緣的主要山脈，古時為重慶屬地。重慶的[[緯度|北緯]]約30度，且地處山谷，故在重慶市區很難看到積雪。但在海拔較高的華鎣山可以看到雪景，實屬可貴，且華鎣山在大雪初停之時(即「雪霽」)的景緻更使人震撼。
# 縉嶺雲霞：[[縉雲山|縉雲山]]位於今天的[[北碚區|北碚-{zh-hans:区; zh-tw:區}-]]，它是重慶市中心的主要[[背斜|背斜]]範圍之一。提出將縉雲山列入清單的王爾鑒稱忚喜歡縉雲山九峰的夜景，他在其詩作《縉嶺雲霞》的序云：「縉雲山九峰爭秀，色赤如霞。」<ref>{{Cite web|url=http://cq.sina.com.cn/news/b/2013-07-05/072164795_5.html|title=缙岭云霞 九峰漂浮云海中|accessdate=2018-12-06|work=cq.sina.com.cn}}</ref>古人以赤多白少為縉，故名縉雲，遂為山名。
# 龍門浩月：源自「渝城八景」。見上一節。
# 佛圖夜雨：源自「渝城八景」。見上一節。

== 新巴渝十二景(1989年版) ==
1989年4月，重慶晚報，[[重慶廣播電視總台|重慶電視台]]和重慶市園林局發起了「新評巴渝十二景」活動，鼓勵市民評選出代表重慶市新的十二個景點。這項活動持續了一年。1990年，晚報發布了重慶的十二個觀點新名單：

# [[大足石刻|大足石刻]]：世界八大[[石窟|石窟]]之一。建於公元[[7世紀|七世紀]]，位於重慶市[[大足區|大足縣]]，有雕像五萬餘尊。現已列入[[世界遺產|世界文化遺產]]。
# 山城燈海：「山城」是重慶市的別稱，這裡指的是重慶市中心的夜景。
# 四面飛瀑：位於重慶市[[江津區|江津區]]境內的四面山，四週山脈環繞，分-{zh-hant:佈; zh-tw:布}-有上千條瀑布，其中「[[望鄉台瀑布|望鄉台瀑布]]」，有「華夏第一高瀑」之譽。
# 縉嶺雲霞：源自「古巴渝十二景」，見上一節。
# 北泉溫泳：重慶市週遭遍-{zh-hant:佈; zh-tw:布}-溫泉，有「中國溫泉之都」的稱譽。其中四處溫泉最受歡迎：[[重慶北溫泉|北溫泉]]，[[重慶南溫泉|南溫泉]]，[[重慶東溫泉|東溫泉]]和[[統景溫泉|-{zh-hans:统; zh-tw:統}-景溫泉]]。北溫泉是「溫泉寺」的一部分。[[溫泉寺|溫泉寺]]建於公元423年，[[南北朝|南北朝]]時期。 1927年，[[盧作孚|盧作孚]]將這座寺廟變成了向公眾開放的公園。
# 南山醉花：南山位於重慶市中心的東南邊緣，以遍植數百種花卉而聞名。當中表表者為[[南山公園|南山公園]]和[[南山植物園|南山植物園]]。
# 歌樂靈音：源自「古巴渝十二景」，見上一節。
#-{zh-hans:统; zh-tw:統}-景峽猿：源自「古巴渝十二景」，見上一節。
# 長湖浪嶼：[[長壽湖風景名勝區|長壽湖]]位於重慶市[[長壽區|長壽區]]，是重慶市規模最大（65.5平方公里）的人工湖。該湖有數百個島嶼，竟成「壽」字型排列。鳥飛魚翔、天水一色、島影成趣，被譽為「西南明珠」<ref>{{Cite news|url=http://www.yangtse.com/zt/zgcjx/zxxx/2018-05-14/1358156.html|title=重庆长寿湖：生态修复10余年从最差水质变身山青水绿|author=|date=2018-05-14|work=扬子晚报网|accessdate=2018-12-12}}</ref>。
# 獨釣中原：這裡指的是[[合川區|合川]]歷史悠久的[[釣魚城遺址|釣魚城堡]]。釣魚城在[[宋朝|宋朝]]與[[蒙古國|蒙古]]帝國之間的戰爭中發揮了重要作用。當所有其他地方都輸給蒙古人時，釣魚城仍然仡立與之頑抗。1259年，蒙古[[可汗|大汗]][[蒙哥|蒙哥]]在釣魚城附近的一場戰鬥中喪生，迫使蒙古士兵撤離[[敘利亞|敘利亞]]和[[歐洲|歐洲]]。<ref>{{Cite book|title=The Mongols|last=Morgan|first=David|publisher=Wiley-Blackwell|year=2007|isbn=9781405135399|location=|pages=127-128}}</ref><ref>{{Cite web|url=http://www.meiyatravel.com/Attraction/FishingCity.aspx|title=Fishing City – Origin, History, Scenic Spots - Meiya Travel|accessdate=2018-12-11|work=www.meiyatravel.com}}</ref>
# 南塘溪趣：這裡指的是指[[重慶南溫泉|南溫泉]]，位於重慶南郊花溪河畔風景名勝區內，是另一個受歡迎的重慶溫泉區。
# 朝天匯流：這裡指的是[[朝天門|朝天門]]及旁邊的港口和廣場。它是舊重慶最東部的城門。大門外是[[嘉陵江|嘉陵江]]流入[[長江|長江]]的地方。

== 新巴渝十二景(2006年版) ==

# 長湖浪嶼：源自「新巴渝十二景(1989年版)」，見上一節。
# 芙蓉滴翠：指位於重慶[[武隆區|武隆區]][[江口鎮_(重慶市)|江口鎮]][[芙蓉江|芙蓉江]]畔的[[芙蓉洞|芙蓉洞]]，為天然[[溶洞|溶洞]]，被列入[[世界遺產|世界自然遺產]]。

# 金佛崔嵬：指[[南川區|南川區]]的[[金佛山|金佛山]]，是[[世界遺產|世界自然遺產]]。
# 黃水林海：[[黃水鎮_(石柱縣)|黃水鎮]]是[[石柱土家族自治縣|石柱土家族自治縣]]的一個小鎮。它以1998年成立的[[黃水國家森林公園|黃水國家森林公園]]而聞名。

# 瓷器古鎮：指磁器口，是重慶主城區內少有的能領略老重慶風味的地方，被稱爲「小重慶」。詳見「[[磁器口_(重慶)|磁器口]]」條目。
# 山城夜景：與「新巴渝十二景(1989年版)」的「山城燈海」相同，見上一節。
# 彩雲白帝：指的是[[奉節縣|奉節縣]][[白帝城|白帝城]]古廟群的景觀。名稱源自[[唐朝|唐代]][[詩人|詩人]][[李白|李白]]《[[早發白帝城|早發白帝城]]》一詩首句：「朝辭白帝彩雲間」。
# 大足石刻：源自「新巴渝十二景(1989年版)」，見上一節。
# 漢豐新城：[[開州區|開州區]]漢豐鎮，原址位於長江流域附近。[[長江三峽水利樞紐工程|三峽大壩]]的建設提升了長江水位，老漢峰被水淹沒。政府在更高的地方建立了一個全新的，規劃較好的漢豐城。
# 名碑金街：指的是[[抗戰勝利紀功碑暨人民解放紀念碑|解放碑]]及其周圍的廣場。這是重慶最重要的地標。
# 西沱天街：[[西坨|西坨]]是[[石柱土家族自治縣|石柱土家族自治縣]]保存完好的古鎮。西沱天街從是長江沿線唯一垂直江面的街道，被稱為「萬里長江第一街」。街兩旁保存著[[明朝|明]][[清朝|清]]遺留下來的層層疊疊的土家民居[[吊腳樓|吊腳樓]]。從街上穿過長江便可到[[石寶寨|石寶寨]](見下面「石寶瓊閣」)。<ref>{{Cite web|url=http://epaper.gmw.cn/gmrb/html/2013-08/11/nw.D110000gmrb_20130811_1-10.htm|title=西沱天街-光明日报-光明网|accessdate=2018-12-11|work=epaper.gmw.cn}}</ref>
# 釣魚古城：與「新巴渝十二景(1989年版)」的「獨釣中原」相同，見上一節。
# 紅岩豐碑：指的是「紅岩紀念館」，詳見「[[八路軍重慶辦事處舊址|八路軍重慶辦事處舊址]]」條目。
# 石寶瓊閣：指位於[[忠縣|忠縣]][[石寶鎮_(忠縣)|石寶鎮]]長江北岸邊的石寶寨，為一臨江拔起高十多丈的陡壁巨石。相傳此石為女媧補天所遺的一尊五彩石，故稱「石寶」。詳見「[[石寶寨|石寶寨]]」條目。
# 豐都鬼城：位於重慶市下游[[豐都縣|豐都縣]]的長江北岸，相傳是[[酆都大帝|酆都大帝]]與[[十殿閻羅|十殿閻君]]管理諸鬼之所。詳見「[[豐都鬼城|豐都鬼城]]」條目。

== 新巴渝十二景(2015年版) ==
事隔九年後，[[重慶日報_(中華人民共和國)|重慶日報報業集團]]再一次舉辦「新重慶巴渝十二景」，除原有的「自然場景」類改名為「自然景觀」類外，這次將「人文景觀」拆開為「人文勝地」類和「城市地標」類，每個類別各含十二景，一共評選出三十六個景點<ref>{{Cite news|url=http://news.sina.com.cn/o/2015-08-15/doc-ifxfxzzn7492153.shtml|title=“2015新重庆·巴渝十二景”揭晓|author=张莎，付微之|date=2015年08月15日|work=重庆日报|accessdate=2019-01-04}}</ref><ref>{{Cite news|url=http://www.sohu.com/a/66293418_349101|title=重庆 {{!}} 新巴渝十二景，总有一个在你家乡|author=|date=2016-03-28|work=搜狐新聞|accessdate=2018-12-30}}</ref>。

=== 自然景觀類 ===

# 茶山竹海：即位於[[永川區|永川區]]的[[茶山竹海國家森林公園|茶山竹海國家森林公園]]，距重慶市中心55公里，擁有2萬畝大型連片茶園和5萬畝竹海，乃罕見的茶竹共生景觀，武俠片《[[十面埋伏_(2004年電影)|十面埋伏]]》即在此取景拍攝<ref>{{Cite news|url=http://ent.163.com/ent_2003/editor/news/starnews/040108/040108_213268.html|title=《十面埋伏》刘德华章子怡竹林开"打"|author=庞山岚|date=2004年01月08日|work=华西都市报|accessdate=2018-12-30}}</ref>。2013年列為[[國家4A級旅游景區|國家4A級風景區]]。
# 半島燈海：指[[渝中區|渝中半島]]的夜景，即2006年版的「山城夜景」，見上一節。
# 天下龍缸：指位於[[雲陽縣|雲陽縣]]的[[龍缸國家地質公園|龍缸國家地質公園]]，總面積296平方公里。區內遍-{zh-hant:佈; zh-tw:布}-溶洞，奇峰怪石，地貌奇特。2017年列為[[國家5A級旅游景區|國家5A級風景區]]。
#玄天秀水：即[[玄天湖|玄天湖]]，位於[[銅梁區|銅梁]][[巴嶽山|巴嶽山]]，湖邊建有「玄天湖旅遊度假區」。
# 龍水西湖：這裏指的是[[大足區|大足區]]的[[龍水湖水庫|龍水湖]]，因其坐落於[[巴嶽山|巴嶽山]]西部，故譽為「重慶西湖」。龍水湖為人工湖，面積16平方千米，湖內有形態各異的108個小島點綴，延綿十餘千米，景觀獨特。2004年列入第四批[[中國國家水利風景區列表|國家水利風景區]]。
# 古劍雲海：即[[綦江區|綦江區]]的[[古劍山|古劍山]]景區，2012年列為[[國家4A級旅游景區|國家4A級風景區]]。初冬時分，這裏的雲海變化無常而又千姿百態，令人震撼。
#黑山幽谷：源自「新巴渝十二景(2006年版)」，見上一節。
# 照母湧翠：指的是位於[[北部新區_(重慶市)|北部新區]]的[[照母山森林公園|照母山森林公園]]。因[[南宋|南宋]][[狀元|狀元]][[馮時行|馮時行]]曾在此結廬照母，故得名。這裏植被覆蓋綿密，山勢起伏平緩，甚合老少散止。園內亦有明清時期節孝、忠孝牌坊、孝母泉等景。
# 三峽紅葉：[[長江三峽|長江三峽]]兩岸生長著的植物，每逢十一月尾到次年一月初，即火紅滿上，連綿百里，為中國最長天然紅葉觀賞帶。
# 黃水林濤：即[[黃水國家森林公園|黃水國家森林公園]]，位於[[石柱土家族自治縣|石柱縣]][[黃水鎮_(石柱縣)|黃水鎮]]的[[七曜山|七曜山]]山原上，1998年列為[[中華人民共和國國家森林公園列表|國家級森林公園]]，2017年入選第三批[[中國森林氧吧|中國森林氧吧]]<ref>{{Cite news|url=http://news.ifeng.com/a/20171221/54377352_0.shtml|title=山东又有两景区入选"中国森林氧吧",总数已增至8家|author=朱洪蕾|date=2017年12月21日|work=齐鲁晚报|accessdate=2018-12-30}}</ref>。區域總面積32441公頃，可遊面積6000公頃，是重慶國家級森林公園中區域面積最大，也是長江三峽黃金旅遊線上唯一的[[土家族|土家少數民族]]旅遊景區。
# 仙女天成：指位於[[武隆區|武隆區]]的[[仙女山國家森林公園|仙女山國家森林公園]]，距[[重慶主城區|重慶市主城區]]180千米。其以南國罕見的林海雪原，碧野景觀，2011年列為[[國家5A級旅游景區|國家5A級風景區]]。
#青龍飛瀑：即[[萬州大瀑布|萬州大瀑布]]，寬151米，高64米，是亞洲第一大瀑布。2012年列為[[國家4A級旅游景區|國家4A級風景區]]。

=== 人文勝地類 ===

# 湖廣會館：[[重慶湖廣會館|重慶湖廣會館]]位於[[渝中區|渝中區]]，是廣東公所、齊安公所、禹王宮等古建築群的統稱，始建於清乾隆二十四年（1759）。會館內現加建博物館，紀錄了導致會館產生的背後事件，歷史上一次大規模的移民潮——[[湖廣填四川|湖廣填四川]]。
#古城安居：即[[安居古城|安居古城]]，位於[[銅梁區|銅梁區]]，是重慶保存得最完整的古建築城鎮。城內宮、廟、寺林立，「九宮十八廟」，區級文物古蹟多達197處。2014年列為[[國家4A級旅游景區|國家4A級旅遊景區]]。
# 三峽梯城：位於[[雲陽縣|雲陽縣]]，因建有「登雲梯」，橫貫全市中央，故命之。登雲梯全長1388米、寬30米、共1975級，垂直高度達200多米。市內還有[[三峽文物園|三峽文物園]]、[[磐石城|磐石城]]、[[龍脊嶺公園|龍脊嶺公園]]等景點。2014年梯城列為[[國家4A級旅游景區|國家4A級旅遊景區]]。
# 愛情天梯：位於[[江津區|江津區]]高灘村，詳見「[[愛情天梯|愛情天梯]]」條目。
# 華巖禪意：指華巖風景區，位於重慶市西南郊[[九龍坡區|九龍坡區]]。區內有「[[重慶華巖寺|華巖寺]]」，佔地三萬平方米，年代久遠，興建時間已不考，但至少有數百年歷史，被譽為「川東十大古剎」之一。
# 南濱溢彩：即[[南岸區|南岸區]]南濱路，詳見「[[南濱路|南濱路]]」條目。
# 豐盛古韻：即[[豐盛鎮|豐盛古鎮]]，坐落在重慶市[[巴南區|巴南區]]。古鎮可追溯至[[北宋|北宋]]，於[[明清|明清]]達至鼎盛。現鎮所在地佔地尚有半邊、江西、埡口、十字、書院、壽字、公正、長寧等八條老街，不少古建築仍保存完好。
#釣城懷古：即「釣魚古城」及「獨釣中原」，見上一節。
# 白鶴繞樑：[[涪陵區|涪陵區]]長江南岸大江中，有一座天然石梁「[[白鶴梁|白鶴梁]]」。樑上有許多歷代文人（[[黃庭堅|黃庭堅]]、[[朱熹|朱熹]]、[[王士禛|王仕禎]]等人）在那裏題刻，留下約三萬字真跡，被譽為「水下碑林」。因修建三峽工程，白鶴梁題刻被全部淹沒，不能露出水面，因此利用工程壓力原理，在原址修建[[白鶴梁水下博物館|白鶴梁水下博物館]]，遊客可以從岸經過水下通道進入博物館，觀賞白鶴梁題刻。這裏指的正是「[[白鶴梁水下博物館|白鶴梁水下博物館]]」。
# 路孔萬靈：指「[[萬靈鎮|萬靈古鎮]]」，原名「路孔鎮」，位於[[榮昌區|榮昌區]]東部。相傳[[明朝|明朝]]有位叫曾傲的和尚，雲遊到此，發覺坡邊有六個石孔，似與河中相通，便往石孔倒入糠殼上試一試，不久糠殼果然從河中冒出，於是就把這裏叫作「六孔河」，後來改叫「路孔」。路孔鎮依山順勢而建，台階步梯隨處都是，頗有些重慶老城的感覺，所以又叫「小山城」。2014年列為[[國家4A級旅游景區|國家4A級旅遊景區]]。
# 風情濯水：指[[黔江區|黔江]][[濯水鎮_(重慶市)|濯水古鎮]]，是一個集[[土家族|土家]][[吊腳樓|吊腳樓]]群落、水運碼頭、商貿集鎮於一體的千年古鎮。
# 雙桂禪堂：指[[梁平區|梁平縣]]雙桂堂，詳見「[[雙桂堂|雙桂堂]]」條目。

=== 城市地標類 ===

# 十字金街：即[[渝中區|渝中]]解放碑[[中央商务区|核心商業區]]地帶，號稱「十字金街」。詳見「[[解放碑中央商務區|解放碑中央商務區]]」條目。
# 天地湖光：指[[渝中區|渝中]][[重慶天地|重慶天地]]，為大型建築項目，規劃建築面積約348萬平方米，集商業寫字樓、休閒娛樂、商場、酒店、人工湖、濕地公園、住宅等多種設施。當中B11二三期超高層項目建築物高度將為458米，預計2022年完成<ref>{{Cite news|url=http://news.winshang.com/html/063/0430.html|title=原瑞安重庆天地458米超高层复工 万科预计2022年入市|author=|date=2017年11月30日|work=观点地产网|accessdate=2019-01-03}}</ref>。
# 開埠新城：指[[南岸区|南岸]][[弹子石街道|彈子石]][[中央商务区|核心商業區]]，為[[重慶中央商務區|重慶中央商務區]]「金三角」之一，其餘兩個在解放碑(十字金街，見上)及江北嘴(北岸皇冠，見下)。
# 綠園串珠：指[[大渡口区|大渡口]]公園群。大渡口為重點綠化城市，區內建有多個公園<ref>{{Cite news|url=http://cq.people.com.cn/n2/2017/1128/c367650-30971287.html|title=大渡口：绿色让城市更美好|author=吴黎 王彩艳|date=2017年11月28日|work=重庆日报|accessdate=2019-01-04}}</ref>。
#[[樂和樂都|樂和樂都]]：為座落於[[永川区|永川]]的[[主題公園|主題公園]]，佔地三百多[[公顷|公頃]]。內有地球村、歡樂世界、野生動物世界等區域，並設有兩間酒店。
# 北岸皇冠：指位於[[江北区_(重庆市)|江北]]的[[中央商务区|核心商業區]]，詳見「[[江北嘴中央商务区|江北嘴中央商務區]]」條目。
#動物樂園：即位於九龍坡的重慶動物園，詳見「[[重庆动物园|重慶動物園]]」條目。
# 白塔老街：指文峰古街，位於[[合川区|合川區]]城區的[[涪江|涪江]]南側，與[[钓鱼城遗址|釣魚城]]僅一橋之隔，是一處在古代遺址的基礎上修建的訪古式商業街。
# 兩江彩虹：指重慶[[两江大桥|兩江大橋]]，即「[[东水门长江大桥|東水門長江大橋]]」和「[[千廝門嘉陵江大橋|千廝門嘉陵江大橋]]」的合稱。兩橋分別替渝中區打開南、北通道。詳見相關條目。
# 白象老街：即[[渝中区|渝中區]]的白象街，全長425米。短短的一條白象街，曾在十九世纪末二十世纪初為經濟中心，世界三大金融街區之一，比肩[[纽约|紐約]][[华尔街|華爾街]]、[[伦敦|倫敦]][[倫敦市|金融街]]。百年過後，昔日繁華已消失殆盡。重慶市政府近年委託融創發展商重建該區，希望挽回部份往日風光。
#鶯花飛渡：即「鶯花渡」，為[[江北区_(重庆市)|江北區]]鎏嘉碼頭原址的舊稱。現在是商業及娛樂休閒中心。
#四季觀塘：指[[璧山区|璧山縣]]城南部的[[觀音塘濕地公園|觀音塘濕地公園]]，占地四十公頃，為[[中国湿地公园|國家級濕地公園]]。園內有植物677種，吸引了[[白鹭|白鷺]]、[[斑鸠属|斑鳩]]、[[鸳鸯|鴛鴦]]等30餘種野生鳥類棲息。2014年列為[[国家4A级旅游景区|4A級旅遊景區]]。
