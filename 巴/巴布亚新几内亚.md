'''巴布亚新几内亚独立国'''（{{lang-en|Independent State of Papua New Guinea}}），通称'''巴布亚新几内亚'''（Papua New Guinea；[[巴布亚皮钦语|巴布亚皮钦语]]：Papua Niugini；[[希里摩图语|希里摩图语]]：Papua Niu Gini），简称'''巴新'''，是位於[[太平洋|太平洋]]西南部的一個[[島嶼|島嶼]][[國家|國家]]，主要涵蓋[[新幾內亞島|新幾內亞島]]東半部，西鄰[[印度尼西亞|印度尼西亞]]的[[巴布亞省|巴布亞省]]，南部和東部分別與[[澳洲|澳洲]]和[[所羅門群島|所羅門群島]]隔海相望。

== 历史 ==
{{main|巴布亞新幾內亞歷史}}

公元前8000年新幾內亞高地已有人定居。1511年[[葡萄牙|葡萄牙]]人發現新幾內亞島。直至19世紀初，西方對此島了解不多。

[[file:Territorios_coloniales_de_Nueva_Guinea.PNG|thumb]]

1884年英、德相繼瓜分新幾內亞島東半部及附近島嶼。1906年英屬新幾內亞交[[澳洲聯邦|澳洲聯邦]]管理，改稱澳屬巴布亞領地。

[[Image:Australian_infantry_and_armour_at_Buna_(AWM_014008).jpg|thumb]][[澳大利亞國防軍|澳洲軍隊]]攻擊[[日軍|日軍]]陣地。]]1919年一战结束后，澳军佔領[[德属新几内亚|德属新几内亚]]。1920年12月17日[[國際聯盟|國際聯盟]]決定委託澳管理；[[二战|二战]]中新幾內亞一度為日軍佔領，戰後聯合國委托澳繼續管理德屬部分，1949年澳將原英屬和德屬兩部分合併為一個行政單位，稱「巴布亞紐幾內亞領地」。

1973年12月1日實行內部自治。1975年9月16日脫離[[澳洲|澳洲]]獨立，成為[[英聯邦王國|英聯邦]]成員國。

== 地理 ==
{{main|巴布亞紐幾內亞地理}}
[[File:Papua_New_Guinea_map.png|thumb]]
[[File:Goroka.JPG|thumb]]
[[File:GorokaMarket.jpg|thumb]]
[[File:Huli_Wigman_visit_Cooktown,_Australia_2005.jpg|thumb]]

巴布亞紐幾內亞面積462,840平方公里，其国土包括所有島嶼在内位於[[赤道|赤道]]至南纬12度，東經140度至160度之间。

巴布亞新幾內亞為[[赤道多雨氣候|赤道多雨氣候]]區，氣候炎熱且降雨量大，形成一個個[[熱帶雨林|熱帶雨林]]，其國土多被山和[[熱帶雨林|熱帶雨林]]覆蓋。因巴布亞新幾內亞是熱帶雨林為主，所以在生物种类上相當豐富。植物方面都主要是一些高達數十米的闊葉樹木，而動物方面則主要以鳥類和有袋類動物為主。[[极乐鸟|极乐鸟]]是巴布亚新几内亚特有的鸟类，被绘在其国徽中。

== 政治 ==
{{main|巴布亞紐幾內亞政治}}
巴布亚新几内亚是[[議會民主制|議會民主制]]的[[君主立憲|君主立憲]]國家，国家元首是女王[[伊丽莎白二世|伊丽莎白二世]]，女王不在巴布亚新几内亚時由總督代爲行使元首職責。巴布亚新几内亚立法機構為一院制議會，政府首腦是總理大臣，由議會内多數黨的領袖擔任。議會的111個議席中有22席為21個省和首都區的22名地方首長的官守議席。

巴布亞新幾內亞有40多個[[政黨|政黨]]，其中比較具有影響力的政黨有國民聯盟黨、[[人民行動黨_(巴布亞紐幾內亞)|人民行動黨]]、聯合資源黨、聯合黨、潘古黨、人民全國代表大會黨、人民民主運動黨、人民勞動黨、[[人民進步黨_(巴布亞紐幾內亞)|人民進步黨]]、[[巴布亞新幾內亞黨|巴布亞新幾內亞黨]]及新一代黨等。

巴布亚新几内亚是[[英联邦王國|英联邦王國]]。

== 氣候 ==
巴布亞紐幾內亞為[[赤道氣候|赤道氣候]]，其特點是一年四季高溫，高濕度。平地年平均氣溫約24℃，溫度隨高度升高迅速降低，冬季有暴雨。

== 經濟 ==
{{main|巴布亞紐幾內亞經濟}}

[[File:Papua_New_Guinea_Export_Treemap.png|thumb]]
[[File:OkTediMine.jpg|thumb]]

巴布亞新幾內亞擁有豐富的自然資源，包括礦產和可再生資源，如森林，海洋（包括全球主要剩餘[[金槍魚|金槍魚]]資源的很大一部分），以及農業的某些部分，人民所得約為2,084美元（2015年估計）。

巴布亞新幾內亞主要是以農業為主，據2003年資料顯示，農業佔巴布亞新幾內亞經濟34%。主要出口有礦產和農產[[咖啡|咖啡]]、[[可可|可可]]、[[椰乾|椰乾]]、[[棕油|棕油]]、[[橡膠|橡膠]]、木材及海產等。

亚太经合组织领导人非正式会议，简称亚太经合峰会（英语：THE APEC ECONOMIC LEADERS' MEETING）。举办次数：第26次。举办年份：2018年。举办日：11月12日-11月18日。举办国：巴布亚新几内亚。举办都市：莫尔兹比港

==人口==
{{main|巴布亞人|巴布亞紐幾內亞人口}}
巴布亚新几内亚在2014年人口估计为6,552,730，人口中位年龄是22.4岁。人口增长率是1.84%，出生率为24.89‰，死亡率为6.53‰。城市人口占13%。新生儿的死亡率是每千人39.67。人口出生时的平均预期寿命是66.85岁。人口识字率是62.4%。没有读写能力的人中妇女占了大部分。<ref>[https://www.cia.gov/library/publications/the-world-factbook/geos/pp.html CIA世界知识手册-巴纽]</ref>

巴布亞紐幾內亞是在世界上最多元化的國家之一。<ref name=fearon>{{cite journal|title=Ethnic and Cultural Diversity by Country|author=James Fearon|journal=Journal of Economic Growth|volume=8|issue=2|pages=195–222|year=2003|doi=10.1023/A:1024419522867|url=http://telematica.politicas.unam.mx/biblioteca/archivos/040107017.pdf|access-date=2018-11-21|archive-url=https://web.archive.org/web/20130512112705/http://telematica.politicas.unam.mx/biblioteca/archivos/040107017.pdf|archive-date=2013-05-12|dead-url=yes}}</ref> 巴布亞紐幾內亞的土著族群有數百個。大部分被稱為[[巴布亞人|巴布亞人]]。他們的祖先在幾萬年前，來到新幾內亞地區。很多偏遠的部落與外界仍然只有輕微的接觸。其余的土著居民为[[南岛民族|南岛民族]]，在少于四千年前迁移至该地区。

原住民的祖先在不到4萬年前抵達該地區。也有許多世界其他地方的居民，包括[[中國|中國]]，[[歐洲|歐洲]]，[[澳洲|澳洲]]，[[菲律賓|菲律賓]]，[[波利尼西亞|波利尼西亞]]和[[密克羅尼西亞|密克羅尼西亞]]。在1975年，巴布亞[[獨立|獨立]]，有40,000名外籍人（主要是澳大利亞和中國）在巴布亞紐幾內亞。<ref>"[http://www.britannica.com/EBchecked/topic/442191/Papua-New-Guinea Papua New Guinea]". ''Encyclopædia Britannica'' Online.</ref>

== 交通 ==
{{main|巴布亞紐幾內亞交通}}
因其多山崎嶇地形，國內陸路交通十分不便。首都[[莫爾茲比港|莫爾茲比港]]并無公路通往任何國內城市，亦無鐵路連接其他城市。公路也僅有少數由外國捐贈的公共交通車輛行駛。

空運乃國內最重要運輸方式，於2007年全國有578座[[機場|機場]]。其次是水運，全國有[[港口|港口]]50多個。
