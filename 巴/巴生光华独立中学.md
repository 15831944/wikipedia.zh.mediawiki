'''巴生光華獨立中學'''（{{lang-en|Kwang Hua Private High School}}，{{lang-ms|Sekolah Menengah Persendirian Kwang Hua}}），简称'''光华独中'''，是[[馬來西亞|馬來西亞]]一所由華社出資創辦的[[華文中學|華文中學]]，位於[[雪蘭莪州|雪蘭莪州]][[巴生|巴生]]，于1954年2月21日建校。2014年，该校开始出现教室不足的情况，不得不采取限制入学的措施。2018年，该校开始建设新的宿舍楼<ref>{{cite web |author1=于小喆 |title=大马光华独中学生增长破纪录 教室不足限制入读 |url=http://www.chinanews.com/hwjy/2014/01-06/5702437.shtml |website=中国新闻网 |accessdate=2019-02-16}}</ref><ref>{{cite web |title=巴生光华独中最大型发展，筹1千万建宿舍 |url=http://www.sinchew.com.my/node/1728407 |website=星洲网 |accessdate=2019-02-16}}</ref><ref>{{cite web |title=巴生光华独中筹获千万·9层新宿舍明年建好 |url=http://www.sinchew.com.my/node/1785856/%E5%B7%B4%E7%94%9F%E5%85%89%E5%8D%8E%E7%8B%AC%E4%B8%AD%E7%AD%B9%E8%8E%B7%E5%8D%83%E4%B8%87%C2%B79%E5%B1%82%E6%96%B0%E5%AE%BF%E8%88%8D%E6%98%8E%E5%B9%B4%E5%BB%BA%E5%A5%BD |website=星洲网 |accessdate=2019-02-16}}</ref>。
