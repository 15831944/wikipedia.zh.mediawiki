'''巴伐利亚基督教社会联盟'''（{{Audio|De-CSU.ogg|德语：}}{{lang|de|CSU – Christlich-Soziale Union in Bayern e.V.}}），简称'''基社盟'''（CSU），是[[德国|德国]][[巴伐利亞州|巴伐利亞州]]的一个[[政党|政党]]。在联邦层面该党和其姐妹党[[德国基督教民主联盟|德国基督教民主联盟]]（基民盟）在[[德国联邦议会|联邦议会]]共同组成一个党团。作為聯盟黨，基民盟並不會在巴伐利亚和基社盟競爭公職，基社盟只在巴伐利亚參選，而基民盟在其他的聯邦州分參選。

== 党史和执政史 ==
该党以[[基督教民主主义|基督教民主主义]]为指导，在[[二战|二战]]后1945年由昔日的[[反法西斯|反法西斯]]人士成立，1957年击败州内对手[[巴伐利亚党|巴伐利亚党]]并执政延续至今，创造了德国国内政党的执政纪录。

== 党的指导思想 ==
该党奉行[[基督教民主主义|基督教民主主义]]和[[保守主义|保守主义]]。“保守”此处指：保留[[基督教|基督教]][[价值观|价值观]]，保留传统、自然的事物。

== 群众基础 ==
该党主要受到巴伐利亚资本家和劳工、庄园主和农民、独立经营的小业主、[[家庭主妇|家庭主妇]]、退休群体等社会各个阶层和职业的支持。

== 党的外围组织 ==
所属的青年组织：基社盟青年团

==-{zh-hant:邦;zh-hans:州;}-内选举纪录 ==
该党自1957年在每次巴伐利亞州议会选举中所得票数均过半，因此得以独自组成州政府：
* 1999年：64%
* 2002年：58.6%
* 2003年：60.7%

在2008年的州议会选举中，基社盟只获得43.4%的得票率，首次未能获得过半数议席，其後与同屬中間偏右的自民党组成联合政府。2013年的州议会选举中，自民黨未能過得票門檻（5%），但基社盟能夠單獨取得過半數，再次独自组成州政府。

2018年，基社盟只获得37.2%的得票率，不僅未能获得过半数议席，更是70多年來最差的一次選舉結果，因此除了前述的問題，也動搖聯邦政府執政聯盟的政黨。
