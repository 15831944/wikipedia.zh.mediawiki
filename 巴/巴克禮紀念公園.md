'''巴克禮紀念公園'''，或稱'''巴克禮公園'''，編號'''18號公園'''，為紀念英籍[[巴克禮|巴克禮]]牧師而命名。位於[[臺南市|臺南市]][[東區_(臺南市)|東區]][[崇明里|崇明-{里}-]]<ref>{{cite news|url=http://www.nownews.com/2007/10/23/11424-2176255.htm|title=來去台南／垃圾公園拿下建築獎　全靠3名傻瓜|work=今日傳媒|date=2007-10-23}}</ref><ref>{{cite news|url=http://www.nownews.com/2010/08/13/11468-2635994.htm|title=郭添財感謝里長和志工　營造巴克禮公園成綠色奇蹟|work=今日傳媒|date=2010-08-13}}</ref>，公園對面為[[臺南市立臺南文化中心|臺南文化中心]]。

== 命名 ==
臺南市政府為紀念英籍[[巴克禮|巴克禮]]牧師，擇市內公園其一命名。關於地緣關係，最初擬將[[台灣教會公報|台灣教會公報社]]對面的9號公園（位於[[東寧路|東寧路]]與青年路間）改名，但當地居民多不同意。市府轉而選擇18號公園，並在2003年正式公告。

== 歷史 ==
都市計畫中，此地編為18號公園預定地。昔日於地權因素，臺南市與[[臺南縣|臺南縣]]政府對簿公堂，導致土地無法開發，一度淪為垃圾傾倒地。直到2004年在里長[[李仁慈|李仁慈]]與里民們的努力下，運出近170趟垃圾車的垃圾，並加以建設，終於成型。獲得全國10大優良公園、第一屆全國景觀大獎、國家卓越建設獎等，並於2007年獲得「[[全球卓越建設獎|全球卓越建設獎]]（FIABCI Prix d'Excellence Awards）公共建設類」優選肯定。

由於此地地下水源豐富，為[[竹溪_(臺南市)|竹溪]]的發源地之一，地點就在園內的月牙灣狀湖泊。在[[台灣荷西殖民時期|荷據時期]]，此湖泊與臺南市立臺南文化中心的水塘本是一塘，名曰「[[荷蘭埤|荷蘭埤]]」，又名「夢湖」。後按圖索驥，重新發堀夢湖，成為一大特色。

巴克禮紀念公園2期擴建，在2015年1月31日正式啟用。原公園3.3公頃，2期則有5.8公頃，總計9.1公頃。

== 位置 ==
與[[臺南文化中心|臺南文化中心]]隔[[中華東路_(台南市)|中華東路]]相望。
