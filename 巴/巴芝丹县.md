'''巴芝丹县'''（{{lang-id|Kabupaten Pacitan}}）是[[印度尼西亚|印度尼西亚]]（印尼）的一个{{tsl|id|Kabupaten|县 (印度尼西亚)|县}}，位于该国西部的[[东爪哇省|东爪哇省]]，距首都[[雅加达|雅加达]]约{{convert|500|km}}。该县共有人口542960人（2015年）<ref>{{cite web|url=http://www.geonames.org/1633438/kabupaten%20pacitan.html|title=Kabupaten Pacitan, Indonesia}}</ref>，人口密度每平方千米683人，人口密集<ref name = "nasapop">{{Cite web |url= http://neo.sci.gsfc.nasa.gov/view.php?datasetId=SEDAC_POP|title= NASA Earth Observations: Population Density|access-date = 2016-01-30|publisher= NASA/SEDAC}}</ref>。该地年平均[[降水量|降水量]]为2703mm，其中1月477mm，8月29mm<ref name = "nasarain">{{Cite web |url= http://neo.sci.gsfc.nasa.gov/view.php?datasetId=TRMM_3B43M&year=2014|title= NASA Earth Observations: Rainfall (1 month - TRMM)|access-date = 2016-01-30 |publisher= NASA/Tropical Rainfall Monitoring Mission}}</ref>。
