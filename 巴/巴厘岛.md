'''峇里島'''（{{lang-id|'''Bali'''}}；峇里语：{{lang|ban|ᬩᬮᬶ}}），作為行政區時被稱為'''峇里省'''，是[[印度尼西亚|印度尼西亚]]33個[[一級行政區|一級行政區]]之一，也是著名的旅遊勝地。該島距離印尼[[首都|首都]][[雅加达|雅加达]]約1,000[[公里|公里]]，与[[爪哇岛|爪哇岛]]之間僅有3.2公里寬海峽相隔，面积约5,630[[平方公里|平方公里]]，[[人口|人口]]约315万，其[[省會|省會]]設於島上南部的[[登巴萨|登巴萨]]。

== 譯名 ==
[[中國大陸|中國大陸]]使用「-{巴厘}-岛」的譯名。在[[臺灣|臺灣]]，[[中華民國外交部|外交部]]<ref>[https://www.boca.gov.tw/cp-56-4063-f873d-1.html 峇-{}-里島阿貢火山]外交部調升峇-{}-里島之旅遊警示</ref>、[[傳媒|傳媒]]、[[觀光業|觀光業]]者，普遍以「峇-{}-里島」作為Bali的音譯。但教育部辭典有收錄「巴-{}-里島」<ref>[http://dict.revised.moe.edu.tw/cgi-bin/cbdic/gsweb.cgi?ccd=Me25PI&o=e0&sec=sec1&op=v&view=1-1 巴里島]，教育部重編國語辭典修訂本</ref>的譯名。

== 地理 ==
=== 地形 ===
[[File:Bali_Labeled.png|left]]

全岛大部分为山地，地势东高西低，有四座以上的锥形火山峰，其中[[阿贡火山|阿贡火山]]（巴厘峰）海拔3142米，为岛上最高点；也有仍在活动期的[[巴杜爾火山|巴杜爾火山]]。

===行政区划===

峇厘省下设1市（Kota）8县（Kabupaten），市县下再设57个区（Kecamatan）

市县名单：

=== 氣候 ===
[[File:Climate_Diagram_Denpasar.PNG|thumb]]的雨溫圖<ref>各數值為1961至1990年的平均，資料來自[[世界氣象組織|世界氣象組織]]（WMO）。</ref>]]

該島地處[[熱帶|熱帶]]，屬於[[熱帶乾濕季氣候|熱帶乾濕季氣候]]，乾濕季相當分明，[[雨季|雨季]]（10月至隔年3月）時吹西北季風，[[乾季|乾季]]（4月至9月）時則吹東南季風。（這樣的季風型態使得海上交通變得困難，是峇里島與外界聯絡不易的一個原因，因此才有獨立的歷史發展。）乾季期間，東部和北部經常降水不足，此外雨季時可能一整天都無降雨，也可能有兩至三小時的[[颮|颮]]，有時降雨過盛會造成道路30公分以上的積水。

峇里島一整年的[[氣溫|氣溫]]大多沒有變化，最低平均氣溫約24度，最高平均氣溫約31度，平均[[濕度|濕度]]約為78%<ref>1990年統計（Statistik Propinsi Bali 1991: 2-6）。</ref>。天氣大多炎熱潮濕，體感溫度偏高<ref name="Covarru54">Covarrubias (1937=1991: 54)</ref>。

== 交通 ==
[[File:Denpasar_national_terminal&old_tower.jpg|thumb]]]]

=== 島外連結交通 ===
峇里島與島外主要的交通接口為[[伍拉·賴國際機場|伍拉·賴國際機場]]（又稱峇里國際機場），位在峇里島南部，主要連結[[雅加達|雅加達]]、[[東京|東京]]、[[新加坡|新加坡]]、[[香港|香港]]、[[悉尼|悉尼]]和[[墨爾本|墨爾本]]等各國大城市。除了國際航線外，國內航線也通往印尼其它的島嶼。

海路交通則是以[[渡輪|渡輪]]連接[[爪哇島|爪哇島]]、[[龍目島|龍目島]]等其他印尼的島嶼。

=== 島内交通 ===
峇里島內沒有[[鐵路|鐵路]]設施，所以大部分的交通方式為[[汽車|汽車]]，島內有沿岸的環島公路以及通往內陸的南北向公路。村落與村落間連接的道路相當多，因此汽車的通行相對方便。中產階級以下的人民主要的交通方式為[[摩托車|摩托車]]和[[合乘計程車|合乘計程車]]，此外一些地區也有使用[[馬車|馬車]]代步。

由於觀光客眾多，島內有許多接駁車。另外，南部主要的觀光地，如[[丹帕沙|丹帕沙]]（Denpasar）、[[庫塔|庫塔]]（Kuta）和[[沙努|沙努]]（Sanur），都有計程車的行駛，價格相當的低廉。

== 人口 ==
巴厘島的人口在2010年人口普查實為3,890,757人。最新的估算（2014年1月）當地人口增長至4,225,384人。大約有30,000名外籍人士居住在巴厘島。

=== 種姓制度 ===
{{main|峇里種姓制}}
由於峇里島人多數信奉[[印度教|印度教]]，印度教的種姓制度也在峇里島人根深柢固。然而峇里島的種姓制度沒有印度的種姓制度複雜。

光看峇里島人的名字，大概可以猜到他是家中的第幾個小孩。原因是峇里島人取名時會依照小孩在家中的排行。

峇里島人除了王室貴族，一般人是沒有姓的，只有名字，其中不管男女，老大都叫Wayan或Putu，老大的女生也可叫Iluh，打從老大至老四都有一些特訂的名字取用。若然老五出現，便會從頭使用老大的特訂名字（Wayan等等），如此類推。<ref>{{cite book |author = 汪雨菁 |title = 峇里島 Day by Day No.7 |publisher = 墨刻出版 |year = 2014 |isbn = 978-986-289-173-5 |pages = p.55 }}</ref>一般來說，峇里島男生會在名字前冠以 '''I'''（相當於「先生」），峇里島女生則冠以 '''NI'''（相當於「女士」）。

例如後[[蘇哈托|蘇哈托]]時期的頂尖藝術畫家 {{link-en|I Nyoman Masriadi |Nyoman Masriadi }} 是家中三子，而印尼女子羽毛球員[[馬哈德維·伊斯特拉尼·尼·克圖特|馬哈德維·伊斯特拉尼·尼·克圖特]]（NI Ketut Mahadewi Istirani）在家中排行第四。

== 峇里文化 ==
雖然處於印尼這個全球人口最多的[[回教|回教]]國家當中，巴厘島的居民大部分信奉{{link-en|峇里印度教|Balinese Hinduism}}。该地文化资源十分丰富，因种类多样而闻名。其他方面，岛上的雕刻、编织艺术也十分有特色。巴厘岛的别称也和它的文化一样多样：具有“神明之岛”、“恶魔之岛”、“天堂之岛”、“魔幻之岛”、“花之岛”、“旅原之地”等的别称。

=== 語言 ===
峇里島有多種本土峇里語言，多數的峇里人都可說現代[[巴厘语|巴厘语]]，其次[[印尼語|印尼語]]是峇里島最常使用的語言。峇里島最為多數的印尼人幾乎都可說[[多語|兩個或三個]]以上的語言，而部份擁有華人血統的印尼人或峇里人，也略懂[[華語|華語]]。由於[[旅遊業|旅遊業]]繁盛，[[英語|英語]]是當地常用的第三語言，亦由於近年來有大量[[兩岸三地|兩岸三地]]的遊客到訪峇里島，[[中文|中文]]開始在當地流行。其他外國語文例如[[日文|日文]]、[[韓文|韓文]]、[[俄文|俄文]]、[[法文|法文]]及[[德文|德文]]也在遊客區的多語言標示牌中出現。

=== 舞蹈 ===
峇里人的古典舞蹈是印尼民族舞蹈中的一支，具有独特的地位。峇里的舞蹈文化與宗教有著密不可分的關係，透過於各大大小小的宗教儀式，如寺廟典禮﹑婚禮﹑火葬儀式﹑家廟典禮，用舞蹈和音樂禮拜﹑款待神明。舞者們非常重視這些傳統舞蹈，從細緻而華麗的服飾打扮，以至舞蹈時的投入和嚴謹態度，就是將舞蹈昇華成與神祇溝通的途徑。峇里人相信，獻給神明的舞蹈，是修行的一種，期望不斷的修行可以解脫俗世輪迴之苦。<ref>{{cite book |author = 汪雨菁 |title = 峇里島 Day by Day No.7 |publisher = 墨刻出版 |year = 2014 |isbn = 978-986-289-173-5 |pages = p.158-159 }}</ref>

峇里舞蹈舉凡有{{link-en|巴龍舞|Barong (mythology)}}、{{link-en|峇里戰士舞|Baris (dance)}}、[[卡恰舞|卡恰舞]]、{{link-en|雷貢舞|Legong}}、{{link-en|凱比亞嘟嘟|Kebyar duduk}}、{{link-en|潘揚布拉瑪舞|Panyembrama}}、{{link-en|歐蕾唐姆林林安舞|Oleg (dance)}}、{{link-en|托賓假面具舞|Topeng}}、{{link-en|桑揚舞|Sanghyang}}、凱比亞川朋（Kebyar Trompong）、賈克（Jauk）等等。

=== 次文化 ===
自2012年起，島上流行搞怪快閃。既使終年均溫皆高於24度，但仍有快閃族著大外套、口罩、太陽眼鏡，在鬧區市集進行快閃活動。引起許多觀光客爭相模仿，儼然蘊為另一種巴里島時尚風潮。

== 經濟與觀光 ==
=== 主要觀光地與觀光產業 ===
==== 南部的海灘與度假區 ====
峇里島的觀光業從1969年[[伍拉·賴國際機場|伍拉·賴國際機場]]啟用後開始大規模的發展，[[沙努|沙努]]和[[庫塔|庫塔]]成為觀光勝地；1980年代，高級度假村開始往[[努沙杜瓦|努沙杜瓦]]開發；1990年代，開發的風潮開始向庫塔的南北向拓展，範圍從[[水明漾|水明漾]]（Seminyak）、[[雷吉安|雷吉安]]（Legian）、[[金巴蘭|金巴蘭]]（Jimbaran）到沿岸區，形成廣大的觀光地帶。水明漾的北部就是著名景點[[海神廟|海神廟]]的所在。沙努和庫塔在爆炸案發生之後，當局對於治安採取積極的措施，移除了一些街邊的攤販。

峇里島南部的海岸海浪品質佳，許多世界各國的[[衝浪|衝浪]]好手相繼造訪，成為了衝浪的勝地。最近許多靠衝浪產業為生的人增加，包括衝浪商店和衝浪教練等，甚至是靠衝浪商贊助的專業級衝浪手也增加了。
