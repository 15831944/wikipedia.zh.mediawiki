'''巴爾的摩金鶯'''（Baltimore Orioles），[[美國職棒大聯盟|美國職棒大聯盟]]球隊之一，屬地於[[美國|美國]][[馬里蘭州|馬里蘭州]][[巴爾的摩|巴爾的摩]]，球隊隸屬於的[[美國聯盟|美國聯盟]][[美國聯盟東區|東區]]。自{{MLBy|1992年}}起，主場進駐[[金鶯公園|金鶯公園]]至今。球隊的名字來源於馬里蘭州的州鳥。媒體常慣稱'''O's'''或'''Birds'''。

金鶯於1894年創隊，是美國聯盟初創八隊之一，當時球隊根據地為[[威斯康辛州|威斯康辛州]][[密爾瓦基|密爾瓦基]]，時名[[密爾瓦基釀酒人|密爾瓦基釀酒人]]。1902年時，球隊遷移到[[聖路易|聖路易]]，易名為聖路易棕人隊（亦譯為聖路易布朗隊，Saint Louis  Browns）。1954年，球團將球隊再遷移至[[巴爾的摩|巴爾的摩]]，球隊再更名為[[巴爾的摩金鶯|巴爾的摩金鶯]]。

==球隊歷史<ref>{{cite news| url=http://www.washingtonpost.com/wp-srv/sports/orioles/longterm/memories/davey/articles/poorcomm.htm | work=The Washington Post | accessdate=2015-01-09|title=Poor Communication at Heart of Feud|author=Mark Maske |date=November 16, 1997|page=D4}}</ref>==
金莺最近两次进入季后赛是在1996和1997年，分别以外卡和分区冠军进入季后赛，但是都在美国联盟冠军赛分别输给[[纽约洋基|纽约洋基]]和[[克里夫兰印地安人|克里夫兰印地安人]]，而无缘进入世界大赛。

其中1996年美国联盟冠军赛G1最有争议，小球迷Jeffrey Maier手伸过本垒打墙，接住了[[德瑞克·基特|德瑞克·基特]]的高飞球，使得原本的高飞球出局变成了本垒打，也因此进入延长赛，比赛最后结果是由洋基获胜，当时如果没有这个误判，金莺将获胜。

从1998年以来金莺成绩开始下滑，连续6年分区排名都在第4位，只有在2004年获得第3名，但2005年开始连续3年第4名。

===2008年－2009年===
2008年球季，金鶯在新的球團經營人Andy MacPhail帶領下進行重建。在球季開始之前，他們完成幾件交易，包括將[[米格爾·特哈達|米格爾·特哈達]]送至[[休士頓太空人|休士頓太空人]]；[[艾瑞克·貝達德|艾瑞克·貝達德]]送至[[西雅圖水手|西雅圖水手]]，球隊獲得潛力新秀[[亞當·瓊斯|亞當·瓊斯]]、左投手George Sherrill、小聯盟投手Kam Mickolio、Chris Tillman及Tony Butler。球季開始的第一個禮拜，他們的成績獲得分區領先，隊上球員[[尼克·馬卡克斯|尼克·馬卡克斯]]季初打擊率低迷，而新加入的成員Luke Scott有高的打擊率，達到0.500。不過在這之後，球隊的戰績開始一路下滑。今年因坦帕彎光芒戰績突飛猛進的關係，加入球隊在9月的最後12場比賽輸掉11場比賽，中間包括10連敗，最後的34場比賽輸掉28場，這使得他們的分區排名變成最末位，這是至從1988年以來，再次成為分區最後1名。球季結束以後，金鶯改變他們的制服，將代表馬里蘭的圓形小塊加在球衣的左方向，希望能讓球隊在新球季有不一樣的改變。

2009年6月30日，在對[[波士頓紅襪|波士頓紅襪]]的比賽中，金鶯從原本1：10的大幅落後，於第7局狂得10分，最後以11：10險勝紅襪，當時紅襪在分區排為領先<ref>{{cite web|url=http://mlb.mlb.com/news/article.jsp?ymd=20090630&content_id=5618964&vkey=recap&fext=.jsp&c_id=mlb|title=O's stage historic comeback vs. Red Sox|publisher=MLB|author=Spencer Fordin|date=2009-07-01|language=en|accessdate=2015-01-09}}</ref>。雖然有此逆轉演出，但球季結束以後，球隊只獲得64勝98敗的成績，連續2年於分區排名墊底。球季結束以後，金鶯和總教練Dave Trembley續約1年<ref>
{{cite web|url=http://mlb.mlb.com/news/article.jsp?ymd=20091002&content_id=7287970&vkey=news_mlb&fext=.jsp&c_id=mlb|title=Orioles pick up option on Trembley|author=Spencer Fordin|publisher=MLB|date=2009-10-03|accessdate=2015-01-09|language=en}}</ref>。瓊斯在今年獲選參加及[[2009年美國職棒大聯盟全明星賽|全明星賽]]，並獲得[[美國職棒大聯盟金手套獎|金手套獎]]。

===2010年－2011年===
2010年球季開始前，金鶯和德州遊騎兵交易Kevin Millwood做為他們的開幕戰先發，期望帶領隊中年輕的投手<ref>{{cite web|url=http://baltimore.orioles.mlb.com/news/article.jsp?ymd=20100329&content_id=8989470&vkey=news_bal&fext=.jsp&c_id=bal|title=Millwood gets O's Opening Day nod|author=Brittany Ghiroli|publisher=MLB|date=2010-03-30|accessdate=2015-01-09|language=en|deadurl=yes|archiveurl=https://web.archive.org/web/20100402012206/http://baltimore.orioles.mlb.com/news/article.jsp?ymd=20100329&content_id=8989470&vkey=news_bal&fext=.jsp&c_id=bal|archivedate=2010年4月2日|df=}}</ref>。球季支出球隊戰績不佳，於4月12日和坦帕彎光芒的比賽，金鶯進場的觀眾只有9,129人，為隊史紀錄的新低<ref>[http://www.wbaltv.com/sports/23130983/detail.html Deadlink]{{deadlink|2015-01-09}}</ref>。在前18場比賽裡面，戰績是難堪的2勝16敗，是大聯盟歷史上最差的。

6月4日，金鶯開除總教練Dave Trembley，由三壘教練Juan Samuel暫代總教練職位，直到球團任命新的總教練為止<ref>{{cite web |title = O's Fire Trembley, Samuel To Replace Him | url = http://www.msnbc.msn.com/id/37508848}}</ref>。

7月30日，金鶯雇用Buck Showalter為球隊新任總教練<ref>{{cite web |title = Orioles hire Buck Showalter as manager | url = http://nbcsports.msnbc.com/id/38472552/ns/sports-baseball/}}</ref>。在8月2日球隊介紹新的總教練，並在8月3日首次亮相，並選26號最為球衣號碼，以向他前金鶯已過世的前總教練Johnny Oates表示敬意<ref>{{cite web |title = Showalter pays tribute to  friend and mentor |url = http://msn.foxsports.com/mlb/story/Buck-Showalter-to-wear-Johnny-Oates-number-as-Orioles-manager |deadurl = yes |archiveurl = https://web.archive.org/web/20100804202515/http://msn.foxsports.com/mlb/story/Buck-Showalter-to-wear-Johnny-Oates-number-as-Orioles-manager |archivedate = 2010-08-04 }}</ref>。在Showalter上任後的最後幾場比賽，於8月、9月及10月的戰績，總計繳出34勝24敗的成績，雖然還無法擺脫分區墊底的命運，但新的一年讓人對金鶯能有所好的期待。

===2012年:重回到季後賽===
2012年的球季，金鶯帶來了許多的驚喜。2012年在明星賽前的比賽，金鶯拿下45勝40敗的成績，這是至從1998年以後第二次勝率達到五成以後的成績。在明星賽之前的比賽，5月6日於客場對波士頓紅襪的比賽中，指定打擊[[克里斯·戴維斯|克里斯·戴維斯]]於延長賽16局上場投球，結果金鶯於17局上獲得3分，戴維斯獲得勝投。這是隊史至從1925年以後第1位拿下勝投的打擊者。9月13日的比賽中金鶯獲得勝利，順利拿下今年的第81勝，確定今年的勝率至少可以達到5成，也讓金鶯至擺脫至從1997年以來，14個未過半的球季。9月16日的比賽中金鶯獲得勝利，確定今年球季的勝率可以達到5成以上。

9月21日，金鶯的終結者[[吉姆·詹森|吉姆·詹森]]拿下今年球季的第46次救援成功，打破隊史的紀錄，上一個紀錄保持人是1997年Randy Myers所保持的45次救援成功。9月30日對紅襪的比賽中，詹森拿下今年的第50次救援成功，這也是今年大聯盟裡面唯一一位救援成功達到50次的終結者。在今年球季詹森總計拿下51次的救援成功。另外在9月30日的比賽，由於[[德克薩斯遊騎兵_(棒球)|德克薩斯遊騎兵]]擊敗[[洛杉磯天使|洛杉磯天使]]，金鶯順利的拿下暌違15年的季後賽門票。今年球季金鶯總計拿下93勝69敗的優異成績，只差東區冠軍[[紐約洋基|紐約洋基]]2場勝差，而他們上一個球季只獲得69勝93敗的成績，今年看到了大幅度的成長。

2012年季後賽金鶯是獲得外卡的資格，由於賽制的改變，他們必須先跟另一個外卡的球隊打1場外卡驟死戰，他們的對手是遊騎兵，由於例行賽金鶯對遊騎兵勝少敗多，必須在客場比賽，最後他們以5：1擊敗了遊騎兵，晉級[[2012年美國聯盟分區賽|美國聯盟分區賽]]，對手為洋基。在第3戰中，有一個新紀錄產生，洋基的指定打擊[[勞爾·伊巴涅斯|勞爾·伊巴涅斯]]於第9局以代打的身份上場，結果他擊出追平比數的全壘打，之後在延長比賽12局下，伊巴涅斯第1球就擊出再見全壘打，他成為大聯盟歷史上第1位在1場比賽第9局及延長賽都擊出全壘打的非先發球員。金鶯最後以2勝3敗遭到淘汰，不過今年對金鶯來講算是美好的球季。

===2013年:創紀錄的球季===
這年金鶯隊無緣季後賽，但主砲[[克里斯·戴維斯|克里斯·戴維斯]]以53轟的成績拿下大聯盟全壘打王，同時刷新隊史新紀錄。而守備方面也以僅僅54次失誤創下大聯盟新紀錄。

===2014年：睽違17年的ALDS和ALCS===
金鶯於美國時間2014年9月16日8比2擊敗[[多倫多藍鳥|多倫多藍鳥]]，在美聯東區封王，這是金鶯自1997年以來首次分區封王，也是1969年以來首次在主場登基。

ALDS，金鶯以直落3橫掃[[底特律老虎|底特律老虎]]，晉級ALCS，對手是同樣以直落3橫掃[[洛杉磯天使|洛杉磯天使]]晉級的[[堪薩斯皇家|堪薩斯皇家]]。

ALCS金鶯雖然享有主場優勢，但前2戰牛棚表現不佳，移師到堪薩斯的後2戰打擊遭到封鎖，最終以8:6(10局)、6:4、1:2、1:2，直落4遭到橫掃。[[臺灣|臺灣]]投手[[陳偉殷|陳偉殷]]在第3戰(台北時間10月15日上午8時)先發，主投5.1局失2分卻等不到隊友火力支援吞敗。

2014年金鶯表現精彩，尤其是季前和金鶯簽了一張1年8000000美金合約的Nelson Cruz。他在2014年例行賽揮出40轟，不僅為金鶯提供強大而穩定的火力支援，同時也填補了Chris Davis受到對手重兵看管所留下來的火力缺口；陳偉殷也交出16勝6敗、ERA3.54的佳績。

===2015年===
金鶯在14年冬天走了兩名大將:Nick Markskis及Nelson Cruz後，打擊方面降低很多，在這時補進了Travis Snider,Everth Cabrera等，似乎想將他們倆離開後缺少的火力補起來，而且隊中大將Matt Wieters,Manny Machado,Chris Davis也即將重返，對於金鶯來說是個好消息。

15年4月，金鶯的戰績剛好5成，在4月26日對上紅襪時，以18:7大勝，這也是暨2006年對上印地安人後，金鶯得分最高的比賽，同時也是一種面對紅襪最高比分的一次。5月金鶯雖然有小低潮，但6月隨即反彈;6月5日主力捕手Matt Wieters回歸，帶領金鶯以5:2擊敗印地安人，開啟一波連勝潮，而且在6月16日面對[[費城費城人|費城費城人]]，以19:3大勝，金鶯全場更敲出8支全壘打，打破隊史紀錄。6月金鶯打出18勝10敗的好成績，但在連續贏得7個系列賽後，金鶯跟6月比起來，完全是不同球隊，7月至20日為止，只贏得一場系列賽，不免被橫掃的命運，先發投手壓制力堅強但打擊卻時常發揮不出來，導致金鶯在6月底時一度登上的美東第一，瞬間被[[紐約洋基|紐約洋基]]拿走，勝差也逐漸被擴大。

而在731交易大限前，總管雖然決定讓金鶯成為買家，在交易大限結束前幾小時，金鶯與[[密爾瓦基釀酒人|密爾瓦基釀酒人]]交易，以小聯盟投手Zach Davies交易來外野手Gerardo Parra;再與[[芝加哥小熊|芝加哥小熊]]交易，用救援投手Tommy Hunter交易來小聯盟外野手Junior Lake，但最後金鶯還是無緣季後賽。

今年冬天金鶯在自由市場非常掙扎，一次就有3個重量級球員投身自由市場(陳偉殷轉戰[[邁阿密馬林魚|邁阿密馬林魚]]，其他兩人留下)。

為了找到Chris Davis的備胎，金鶯先簽下Mark Trumbo和Pedro Alvarez;再與投身自由市場的牛棚後援Darren O'Day給簽回來;接著在台灣時間12月25日，和[[韓國職棒|韓國職棒]]強打[[金賢洙|金賢洙]]簽下兩年700萬美元（約台幣2.3億元）合約，對波拉斯施加壓力。

===2016年===

金鳥大軍一開季便衝上分區龍頭，最重要的關鍵在於火力徹底炸裂，不只從開路先鋒到中心棒次棒棒開花，甚至連埋伏在第8棒的Jonathan Schoop都轟出25支全壘打!但與前一季不同的是，雖然火力兇猛，但自家先發卻常常核爆，甚至還有人被踢進牛棚裡去，也因此被紅襪與藍鳥迎頭趕上，讓出王座。

儘管紅襪分區封王，但金鶯依舊以89勝73敗的成績，與藍鳥爭奪對決遊騎兵的資格，最終因[[巴克·修沃爾特|巴克·修沃爾特]]調度失誤延賽賽2：5落敗。

===2018年:砍掉重練===
金鳥大軍儘管在季前補強投手，但還是無法獲得外界好評，因為相較於旺盛的打擊火力，金鶯更需要在投手環節大補強。

毫無意外的，金鶯砲火正常發揮，但投手也個個爆炸，讓本季金鶯提前宣布放暑假(全大聯盟最快)，最後在731交易大限把陣中的自家球員包含[[曼尼·馬查多|曼尼·馬查多]]送到[[洛杉磯道奇|洛杉磯道奇]]、[[查克·布里頓|查克·布里頓]]送到同區[[紐約洋基|紐約洋基]]，[[達倫·奧'戴|達倫·奧'戴]]、[[凱文・高斯曼|凱文・高斯曼]]相繼送到[[亞特蘭大勇士|亞特蘭大勇士]]，換取不少小聯盟球員，正式宣告金鶯重建。雖然陣中還剩[[克里斯·戴維斯|克里斯·戴維斯]]和[[亞當·瓊斯|亞當·瓊斯]]，沒有下限的金鶯成為本季最快吞下第100敗的球隊，也是球隊自從1954年搬家到巴爾的摩後，隊史第三度單季吞100敗。

2018年金鶯隊決定與修沃爾特分道揚鑣，結束他執教9年金鶯隊。
