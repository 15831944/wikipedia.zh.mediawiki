'''巴登-巴登'''（{{lang-de|Baden-Baden}}），[[德国|德国]][[巴登-符腾堡|巴登-符腾堡]]州的城市，位于德国的西南部，[[黑森林|黑森林]]的西麓，奥斯河谷，邻近[[卡尔斯鲁厄|卡尔斯鲁厄]]。巴登-巴登是著名的[[温泉|温泉]]疗养地、[[旅游|旅游]]胜地和国际会议城市。著名美国文学家[[马克吐温|马克吐温]]对巴登巴登做出过如下评价：五分钟你会忘了你自己，二十分钟你会忘了全世界。

== 地理 ==
[[File:Stiftskirche_Baden-Baden.JPG|thumb]]
巴登-巴登位於[[德國|德國]]南部[[巴登-符騰堡|巴登-符騰堡]]州的西側，北部[[黑森林|黑森林]]西麓的奥斯河谷，城市的东部依靠着黑森林的边缘，西南部处于莱茵河上游平原。

巴登-巴登距離德国與[[法國|法國]]边界的[[萊茵河|萊茵河]]僅7.5公里，毗邻法国的北[[阿尔萨斯|阿尔萨斯]]地区和[[史特拉斯堡|史特拉斯堡]]。

城区沿着奥斯河谷而建，依山傍水，景色多姿。

== 历史 ==
巴登-巴登的历史可以追溯到[[罗马帝国|罗马帝国]]时期，[[罗马人|罗马人]]在这个地区发现了温泉，巴登-巴登由此出名。12世纪初，赫尔曼二世（{{lang|de|Graf Hermann II.}}）从策林格王朝（{{lang|de|Zähringer}}）继承了巴登-巴登及其周边，自封为[[巴登统治者列表|巴登藩侯]]。1250年，巴登-巴登正式成为城市。

1689年8月24日，巴登-巴登在对[[普法尔茨|普法尔茨]]的争夺战中被法国军队烧毁。

18世纪末，巴登-巴登再次成为疗养胜地，建立起了豪华酒店、赌场和疗养馆，到访过的贵族们把巴登-巴登称为“欧洲的夏季首都”<ref>http://www.bad-bad.de/gesch/romantk.htm</ref>，而把巴黎称为“欧洲的冬季首都”。

1858年起，在周边的伊费兹海姆（{{lang|de|Iffezheim}}）开始了每年5月底到6月初、8月底到9月初的[[赛马|赛马]]传统，它和每年10月的交易和赛马节一起，使得巴登-巴登成为国际赛马中心。

== 文化 ==
[[File:Baden_baden2.jpg|thumb]]
=== 剧院 ===
巴登-巴登的宴会厅（{{lang|de|Festspielhaus}}）是德国最大的[[歌剧院|歌剧院]]和[[音乐厅|音乐厅]]，是继[[法国|法国]][[巴黎|巴黎]]的[[巴士底歌剧院|巴士底歌剧院]]之后的[[欧洲|欧洲]]第二大歌剧院，巴登-巴登的剧院拥有2500个观众座位，超过[[意大利|意大利]][[米兰|米兰]]的[[斯卡拉大剧院|斯卡拉大剧院]]。

此外巴登-巴登还另拥有一个[[剧院|剧院]]。

=== 赌场 ===
巴登-巴登的赌场（{{lang|de|Casino}}）是德国最大和历史最悠久的赌场，源于1809年。它被美国德裔演员兼歌手[[玛莲娜·迪特里茜|玛莲娜·迪特里茜]]誉为“全世界最漂亮的赌场”<ref>http://www.bad-bad.de/casino/c-gesch.htm</ref>。赌场对服装要求严格，男士需穿[[西服|西服]]带[[领带|领带]]才能进入内厅赌博。

=== 温泉 ===
巴登-巴登的温泉疗养馆（{{lang|de|Kurhaus}}）世界闻名，也是这座城市的标志，它建于1824年，不仅是洗温泉的地方，也拥有德国最大和最老的[[赌场|赌场]]，赌场大厅以疗养馆建造者[[弗里德里希·魏布雷讷|弗里德里希·魏布雷讷]]的名字命名。

巴登-巴登满城的温泉浴池中有两个罗马式浴池最为出名，一个是卡拉卡勒浴池（{{lang|de|Caracalla Therme}}），罗马时期的[[罗马皇帝列表#元首制|卡拉卡拉大帝]]曾御临温泉治疗，由此而得名，这里的温泉有十分广泛的治疗功用。另一个是[[文艺复兴|文艺复兴]]风格的腓特烈大浴池（{{lang|de|Friedrichsbad}}），始建于1869年历时8年建成，大浴场内有蒸气浴、灌水浴、流水浴和潜水浴等，还有天然泉泥浴和碳酸浴等多种种类，且有男女混浴。

== 溫泉之特色 ==
[[File:Altes_Dampfbad_Baden-Baden.JPG|thumb]]
“巴登-巴登”（{{lang|de|Baden-Baden}}）在德语里是“温泉温泉”的意思，这个名字的来历可以追溯到统治黑森林地区的[[巴登大公国|巴登大公国]]。巴登-巴登自[[罗马时期|罗马时期]]即為著名的溫泉療養區，巴登-巴登的温泉是食盐泉，可以治疗[[心脏|心脏]][[动脉|动脉]]疾病、[[风湿病|风湿病]]、[[妇女病|妇女病]]和[[呼吸道|呼吸道]]疾病，功效广泛。此區亦為[[葡萄|葡萄]]產區，近世紀則因為氣候外在條件優勢，而進一步成為德國首要的健康休閒區。

*西元211年-217年，[[罗马人|罗马人]]和[[日耳曼人|日耳曼人]]在黑森林一带进行战斗。罗马人发现从奥斯河谷涌出的泉水温度高达68度，因为当时的罗马人热衷温泉浴，[[羅馬皇帝|羅馬皇帝]][[卡拉卡拉|卡拉卡拉]]（Caracalla）立刻在此建造了罗马式的浴池，并多次前往温泉治療其長年的風濕病，他的御用浴池全部用[[大理石|大理石]]砌成，而士兵的浴池则用[[砖|砖]]砌成。1847年在巴登-巴登挖掘出来的浴池遗址，便是罗马时期留下的。
*12世紀，[[神聖羅馬帝國|神聖羅馬帝國]]將此區收歸為帝國屬地。
*16世紀，此地發展配藥及製藥業，與溫泉治療相輔相成。
*1862年，法國人在此設立[[賭場|賭場]]、[[戲院|戲院]]、[[歌劇院|歌劇院]]，法國[[賽馬|賽馬]]俱樂部亦成立賽馬場，爾後此區乃成為全歐洲夏季休閒之都。
