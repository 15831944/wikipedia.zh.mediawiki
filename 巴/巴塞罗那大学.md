'''巴塞罗那大学'''（{{lang-ca|Universitat de Barcelona, UB}}; {{IPA-ca|uniβərsiˈtad də βərsəˈɫonə|IPA}}），为[[巴塞罗那|巴塞罗那]]市的一所公立大学。巴塞罗那大学创建于1450年，1842年重建。巴塞罗那大学为[[加泰罗尼亚|加泰罗尼亚]]最古老的大学。 根據2017年[[QS世界大學排名|QS世界大學排名]]，巴塞羅那大學被認為是加泰罗尼亚以至[[西班牙大學列表|西班牙最好的大學]]，全球排名156位。<ref>{{cite web |url=http://www.topuniversities.com/university-rankings/world-university-rankings/2011?page=3 |title=QS World University Rankings |publisher=Topuniversities |date= |accessdate=2012-02-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20120215000010/http://www.topuniversities.com/university-rankings/world-university-rankings/2011?page=3 |archivedate=2012-02-15 }}</ref>

== 院系设置 ==
巴塞罗那大学设有22个系。

== 校园 ==
