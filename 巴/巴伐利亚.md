'''巴伐利亞自由州'''（{{lang-de|Freistaat Bayern}}），簡稱'''巴伐利亞州'''，又称'''拜恩州、拜-{}-仁州'''，是[[德意志联邦共和国|德意志联邦共和国]]东南部的一个联邦州，其面積位居德国第一（占全国面积1/5）、人口第二（次于[[北莱茵-威斯特法伦|北莱茵-威斯特法伦]]），首府位於[[慕尼黑|慕尼黑]]。

巴伐利亚州的西北部与[[巴登-符腾堡|巴登-符腾堡]]、[[图林根|图林根]]、[[萨克森|萨克森]]、[[黑森|黑森]]诸州接壤，东部与[[捷克|捷克]]，南部与[[奥地利|奥地利]]接壤。州府[[慕尼黑|慕尼黑]]为其最大城市；北部[[中弗兰肯行政区|中弗兰肯行政区]][[纽伦堡|纽伦堡]]为该州第二大城市，北拜仁的最大城市；第三大城市是西南部的[[奥格斯堡|奥格斯堡]]，一座古罗马时代就已存在的当时的古罗马国家的边境城市。

巴伐利亚的文化与德国其他州，尤其与北方不同，主要为宗教信仰（天主教为主）和语言（三大方言）差异。巴伐利亚为德国乃至欧洲的一强劲经济体，许多德国企业总部均位于巴伐利亚境内，诸如[[宝马|宝马]]、[[奥迪|奥迪]]、[[西门子公司|西门子]]、[[英飞凌|英飞凌]]、[[彪马|彪马]]、[[阿迪达斯|阿迪达斯]]等。

== 名稱由來 ==
「巴伐利亚」來自拉丁文「{{lang|en|Bavaria}}」，德文「Bayern」音譯應作「'''拜恩'''」、「'''拜-{}-仁'''」。

== 历史 ==
巴伐利亚出自''{{lang|de|Bayuwaren}}''，是罗马帝国统治该地时罗马人给予该地的名称。指的是“来自[[波西米亚|波西米亚]]的人（Boii人）”。大约于520年[[法兰克人|法兰克人]]也提及此名。在8世纪早期[[圣波尼法爵|圣波尼法爵]]傳播[[天主教|天主教]]，使當地居民一律[[皈依|皈依]]。从此巴伐利亚一直信仰[[罗马天主教|罗马天主教]]，16世纪巴伐利亚抗拒[[宗教改革|宗教改革]]，未曾[[改宗|改宗]][[新教|新教]]，而至今仍然坚持信仰[[罗马天主教|罗马天主教]]。

大约由550年至788年，阿芝诺芬家族统治{{link-en|巴伐利亚公国|Duchy of Bavaria}}，直至最后一位公爵{{link-de|泰西罗三世|Tassilo III.}}被[[查理大帝|查理大帝]]罢黜为止。在接着的400年有很多家族统治公国，很少家族能传至第三代。而那些公爵中最后亦是最重要的一位，就是[[韦尔夫家族|韦尔夫]]王朝的[[狮子亨利|狮子亨利]]，亦是慕尼黑的建城者。

当狮子亨利被他侄子[[腓特烈一世_(神圣罗马帝国)|腓特烈一世]]罢免[[萨克森|萨克森]]与巴伐利亚公爵的名衔之后，在1180年，巴伐利亚被给予'''[[维特尔斯巴赫王朝|维特尔斯巴赫]]家族'''作为封地，从此巴伐利亚由1180年直至1918年都是在[[维特尔斯巴赫王朝|维特尔斯巴赫]]家族统治下。1255年巴伐利亚第一次分裂成若干数个公国，但在1506年巴伐利亚再被统一，而慕尼黑也成为唯一的首都。1623年[[三十年战争|三十年战争]]早期巴伐利亚公爵取代了他们的表亲[[普法尔茨|普法尔茨]]伯爵获得了[[神圣罗马帝国|神圣罗马帝国]]内强大的[[选帝侯|选帝侯]]名衔，从那时起不但决定了巴伐利亚拥有选举罗马人民的国王和神圣罗马帝国皇帝的权利，而且获得在帝国法律之下的特别正式地位。
[[File:Wieskirche1998.jpg|225px]]]]

当[[拿破仑|拿破仑]]撤销帝国名号的时候，巴伐利亚于1806年升格成为了一个[[巴伐利亚王国|王国]]，而在1815年[[普法尔茨|普法尔茨]]也与巴伐利亚合并。在1799年至1817年之间巴伐利亚首相[[马克西米利安一世_(巴伐利亚国王)|马克西米连·约瑟夫·冯·莫约拿斯]]伯爵（Maximilian I. Joseph, 1756/05/27-1825/10/23）跟随了一个严厉的现代化方针，建下了行政结构的基础，即使当时仍是君主政体。而这方针亦沿用至今日。在1818年一个新宪法通过（以当时为标准），设立两院制的议会，包括上议院（"Kammer der Reichsräte"）及下议院（"Kammer der Abgeordneten"）。该宪法到第一次世界大战结束，君主政体倒台之前仍然有效。

在普鲁士崛起后，巴伐利亚计划与[[普鲁士|普鲁士]]和[[奥地利|奥地利]]对抗以维护她自己的独立地位。1866年[[普奥战争|普奥战争]]和[[普法战争|普法战争]]的胜利使得普鲁士势不可挡。1870年巴伐利亚加入北方德意志邦联，并在1871年被并入[[德意志帝国|德意志帝国]]。巴伐利亚王国的內政、軍事（僅在發生戰爭時歸屬帝國中央政府指揮），甚至於外交權力（巴伐利亞王國政府有權向外國派駐大使級外交官）仍有所保留。但是王国内部存在很多反对意见，巴伐利亚民族主义者希望国家能像奥地利一样保留独立地位。年轻的国王[[路德維希二世|路德維希二世]]则把精力用在建设[[新天鹅堡|新天鹅堡]]上。至20世纪初，大部分巴伐利亚人则自我认同为德国人。

在20世纪早期[[瓦西里·康定斯基|瓦西里·康定斯基]]、[[保罗·克利|保罗·克利]]、[[亨利·易卜生|亨利·易卜生]]、与其它知名画家都来到了巴伐利亚，特别是慕尼黑的施华宾区，但该区在[[一战|一战]]中被破坏。

一战德意志帝国灭亡后，国王[[路德维希三世_(巴伐利亚)|路德维希三世]]遭宰相[[库尔特·艾斯纳|库尔特·艾斯纳]]罢免，后者成立了共和国。然而这位持奉[[社会主义|社会主义]]的新州长却在1919年被暗杀，其建立的[[巴伐利亚人民邦|巴伐利亚人民邦]]（Volksstaat Bayern）也随之覆灭。与此同时，伴随着[[德国十一月革命|德国十一月革命]]，更为激进的政权[[巴伐利亚苏维埃共和国|巴伐利亚苏维埃共和国]]成立。共产主义革命在一个月后遭到霍夫曼政府暴力镇压并失败。在1919年8月[[班贝格宪法|班贝格宪法]]（Bamberger Verfassung）通过后，巴伐利亚自由邦成立，并成为了[[魏玛共和国|魏玛共和国]]的一部分。随后，右翼极端民族主义者的活动也开始增加。特别是1923年的[[啤酒馆政变|啤酒馆政变]]，使得慕尼黑和[[纽伦堡|纽伦堡]]成为了[[纳粹党|国家社会主义德国工人党]]的根据地。

作为制造业中心，在[[第二次世界大战|第二次世界大战]]时慕尼黑受到猛烈轰炸，并被[[美军|美军]]占领。[[二战|二战]]後，巴伐利亚恢复为一个繁盛的工业中心，大规模的重建工程努力地恢复了很多慕尼黑的历史上著名的建筑，也主办了[[1972年夏季奥运会|1972年夏季奥运会]]。

== 独立意识 ==
之后普鲁士透过战争手段迫使巴伐利亚加入德意志帝国，但其仍拥有王国地位，并保留独立的邮政、铁路与军队等事务的自主权（即所谓的“保留权利”{{lang|de|Reservatrechte}}）。

现在巴伐利亚仍然有着自己的文化独特性，许多当地的居民都强调自己和德国其他地区（尤其与北方的原普鲁士各邦）的区别。并且强调自己的語言的不同。

== 经济 ==
巴伐利亚是德国最富裕的地區之一。在过去的几十年里，这里完成了从以农业为主到高科技工业的巨大转变。[[汽车|汽车]]制造业是巴伐利亚的核心产业之一，拥有悠久的历史和传统。失业率（2004年7月）为百分之六点五。

== 文化 ==
巴伐利亚文化可以一直追溯到1000多年以前。据自由州[[宪法|宪法]]第3条规定，巴伐利亚应当成为一个“文化-{zh-hans:州; zh-hant:邦}-”。州政府在2003年度的预算中向艺术和文化领域拨款超过550万欧元，除此以外，各级地方政府和个人也对本州的文化事业作出了不可低估的贡献。如巴伐利亚传统的民族舞（der Schuhplattler），就是最古老的欧洲舞蹈，起源于1050年。

== 政治 ==
巴伐利亞州擁有一個[[一院制|一院制]]的州议会（{{lang|de|Landtag}}），並實行普选制度。1999年以前，-{zh-hans:州; zh-hant:邦}-內還有一個參議院（{{lang|de|Senat}}），其成員是從巴伐利亞的社會與經濟團體中挑選出來的，不過已經在1998年的一場[[公民投票|公民投票]]中廢止。目前巴伐利亞的政府首腦稱為'''部长主席'''（{{lang|de|Ministerpräsident}}）。

巴伐利亞長期以來較為支持德國的保守派，其中[[基督教社会联盟|基督教社会联盟]]從1946年開始，便在此地擁有龐大優勢。並且從1957年其成員當選部长主席之後，執政就不曾中斷。

=== 2003年9月21日邦選舉 ===
[[艾德蒙·施托伊伯尔|艾德蒙·施托伊伯尔]]（Edmund Stoiber）在這場選舉中連任部长主席，並以基督教社會聯盟組成政府。
