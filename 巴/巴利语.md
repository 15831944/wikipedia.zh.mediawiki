'''巴利语'''（[[IAST|IAST]]/[[ISO_15919|ISO 15919]]: {{IAST|'''Pāḷi'''}}、पालि）是[[古印度|古印度]]的一种语言，属於[[印欧语系|印欧语系]]、[[印度-伊朗语族|印度-伊朗语族]]、[[印度-雅利安语支|印度-雅利安语支]]的一种[[中古印度-雅利安語|中古印度-雅利安語]]，与[[梵語|梵語]]十分相近，是印度西方所用的俗語（[[普拉克里特諸語言|普拉克里特諸語言]]）。巴利語可以用各种文字书写，比如[[婆罗米文|婆罗米文]]、[[天城文|天城文]]，以及由[[巴利圣典协会|巴利圣典协会]]的英国语言学家{{le|理斯·戴维斯|Thomas William Rhys Davids}}所采用的適用於各種印度語言的[[拉丁字母|拉丁字母]][[轉寫|轉寫]]。

== 起源 ==
從字面上來看，巴利（Pāli），是線、行，或是文章、經典的意思，和註釋、評論（[[義註|Attha-katha]]）相對。按[[南傳上座部|南傳上座部]]的記載，稱傳誦巴利三藏的語言，為「摩揭陀語」（Magadhi），或「根本語」（Mula-Bhasa）。後來才叫其「聖典（Pali）語」，即「巴利語」<ref name="賀">{{cite journal|author1=賀勻、蔡奇林|title=略 談 巴 利 語|url=http://buddhism.lib.ntu.edu.tw/lesson/pali/grammar/D.htm}}</ref>。

據[[錫蘭|錫蘭]][[上座部佛教|上座部佛教]]的傳說，現存的《[[巴利語大藏經|巴利語大藏經]]》就是[[阿育王|阿育王]]的兒子<ref>《[[大唐西域記|大唐西域記]]》有他是阿育王的弟弟的傳說。</ref>[[摩哂陀|摩哂陀]]（巴利語：{{IAST|Mahinda}}，梵語：{{IAST|Mahindra}}）帶到錫蘭去的版本，所以巴利語也就是[[摩揭陀語|摩揭陀語]]（{{IAST|Māgadhā nirutti}}，{{IAST|Māgadhikā bhāsā}}）。在公元前二六年前後以[[僧伽羅文|僧伽羅文]]字母記錄下來<ref name="賀"/>。在公元五世紀，[[覺音|覺音]]尊者將[[僧伽羅語|僧伽羅語]]（sīhaḷa-bhāsa）[[義註|注釋書]]轉譯為巴利聖典語<ref> 《小王統史》：「其時所有譯成錫蘭語之義解，為一切人之根本言語摩羯陀語。在生類所有言語中，如此〔語〕持來利益，所有之長老阿闍梨而見為如聖典。由彼總終達耶！彼〔佛音〕為禮大菩提樹，而去閻浮洲。」</ref>。

然而，據[[季羨林|季羨林]]考證，根據語言學的特徵，巴利語應是屬於古印度西部的方言，與流行於東部的[[摩揭陀語|摩揭陀語]]不同<ref>季羨林《原始佛教的語言問題》，《北京大學人文科學學報》一九五七年第一期：我們還可以從語言特徵上來闡明巴利文不是摩揭陀語。關於巴利文的流行地區問題，學者們有各種不同意見。Westergaard<small>（《Über den ältesten Zeitraum der indischen Geschichte》, p. 87.）</small>和E. Kuhn<small>（《Beiträge zur Pāli-Grammatik》, pp. 6 ff.）</small>認為巴利文是[[優襌尼|優襌尼]]（Ujjayinī）地方的方言。R. O. Franke從碑刻方面來著手研究這個問題，結論是︰巴利文是[[賓陀山|賓陀山]]（Vindhya）中部至西部一帶的方言<small>（《Pāli und Sanskrit》, pp. 131 ff.）</small>。Sten Konow也以為賓陀山地帶就是巴利文的故鄉<small>（《The Home of Paiśāci》, ZDWG. 64, p. 95 ff.）</small>，因為他發現巴利文與[[毗舍遮語|毗舍遮語]]之間相同的地方很多，而[[毗舍遮|毗舍遮]]的故鄉他就定為優襌尼<small>（Grerson，《The paiśāca Language of North-Western India》, Asiatic Society Monographs. Vol. Ⅷ, 1906.書中說毗舍遮語是印度西北部方言。）</small>。H. Oldenberg最初主張巴利文是[[羯陵迦|羯陵迦]]（Kaliṅga）方言<small>（巴利文《律藏》，vol. I. London 1879, pp. L ff.）</small>。附和此說者有E. Müller<small>（《Simplilfied Grammar of the Pāli Language》, London 1884, p.111.）</small>。但是H. Oldenberg後來又放棄了前說，另立新說，說巴利文是[[馬拉地語|馬拉提語]]的前身<small>（《Die Lehre des Upanishaden und die Anfänge des Buddhismus》, Göttingen 1915, p. 283.）</small>。E. Windisch<small>（《Über den sprachlichen Charakter des Pali》, Actes du XIVe Congres International des Orientalistes, prem. Partie, Paris 1906, pp. 252 ff.）</small>和W. Geiger<small>（《Pāli literatur und Sprache》, Strassburg 1916, p. 5.）</small>則復歸舊說，說巴利文就是摩揭陀方言<small>（關於這個問題的文獻不勝枚舉，請參閱︰季羡林《使用不定過去時作為確定佛典年代和來源的標準》“Die Verwedung des Aorists als Kriterium für Alter und Ursprung buddhistischer Texte”。德國《格廷根科學院集刊．語言學歷史學類》，1949, p. 288. Anm, 2.）</small>。上面這些說法雖然紛歧，但也有比較一致的一點，這就是︰多數學者都主張巴利文是一個西部方言。事實上也正是這樣子。巴利文的形態變化與阿育王石刻的[[吉爾那爾|吉爾那爾]]（Girnār）石刻相似，如“於”格的語尾是-amhi﹑-e“業”格複數的語尾是-ne等等。但是另一方面，摩揭陀語則是一個東部方言，r變成l，s變成ś，以-a作尾聲的字“體”格的語尾是 -e等等。兩者的區別是非常大的，無論如何也不能混為一談。</ref>。印度地方現存的三十多件[[阿育王敕令|阿育王敕令]]中，屬於西部群的基爾納爾（Girnar）刻文，最接近巴利語。巴利佛教所屬的上座部，其根據地即是西部印度[[優禪尼|優禪尼]]（Ujjeni）為主的地方，且其附近地區發現的刻文也與巴利語相近。並且，把上座部佛教傳入錫蘭的[[摩哂陀|摩哂陀]]長老，據說生於優禪尼，並在這裡出家<ref name="賀"/>。

在巴利語和[[摩揭陀俗語|摩揭陀俗語]](Ardhamāgadhī)之間有很多顯著的相似性，摩揭陀俗語是保存在古代[[耆那教|耆那教]]經典中的古老形式的摩揭陀語。摩揭陀俗語同巴利語在類似要點上不同於阿育王題刻的東部俗語。例如，摩揭陀俗語也不把{{IAST|r}}變更為{{IAST|l}}，并且在名詞變格中至少在很多押韻的地方顯示了結尾{{IAST|-o}}而非東部俗語的{{IAST|-e}}。這種相似性不是偶然的，因為耆那教的創始者[[筏馱摩那|尼乾陀若提子]]（被教徒尊稱為大雄 Mahāvīra）和[[釋迦牟尼佛|釋迦牟尼佛]]在相同的地域（[[摩揭陀國|摩揭陀國]]）傳教。

長和短元音只對比於[[開音節|開音節]]中；在閉音節中所有元音總是短的。短和長的{{IAST|e}}與{{IAST|o}}是互補分布的：短變體只出現閉音節中，長變體只出現在開音節中。短和長的{{IAST|e}}與{{IAST|o}}因此不是不同的音位。

有叫做{{IAST|[[anusvāra|anusvāra]]}}的聲音（梵語術語；巴利語為：{{IAST|nigghahita}}），用羅馬化字母{{IAST|ṁ|}}（[[ISO_15919|ISO 15919]]）或{{IAST|ṃ}}（[[IAST|IAST]]）表示，在多數傳統字母中用凸出點來表示，最初用作標記前面的元音要鼻音化的事實。就是說，{{IAST|aṃ}}, {{IAST|iṃ}}和{{IAST|uṃ}}表示{{IPA|[ã]}}, {{IPA|[ĩ]}}和{{IPA|[ũ]}}。但是在很多傳統發音中，{{IAST|anusvāra}}被發音得更強，像軟腭鼻音{{IPA|[ŋ]}}，所以這些聲音轉而被發音為{{IPA|[ãŋ]}}, {{IPA|[ĩŋ]}}和{{IPA|[ũŋ]}}。但是{{IAST|ṃ}}永遠不跟隨在長元音之后發音；在{{IAST|ṃ}}被增加到以長元音結尾的詞幹的時候，{{IAST|ā}}, {{IAST|ī}}和{{IAST|ū}}要轉換成相應的短元音，比如{{IAST|kathā + ṃ}}變成{{IAST|kathaṃ}}而非{{IAST|*kathāṃ}}，{{IAST|devī + ṃ}}變成{{IAST|deviṃ}}而非{{IAST|*devīṃ}}。

== 構詞法 ==
{{see also|巴利語語法}}

巴利語是高度詞形變化的語言，其中幾乎所有詞，除了承載基本意義的詞根之外，還包含一個或多個[[詞綴|詞綴]]（通常是后綴），它們某種方式修改意義。名詞要為了[[性_(語法)|性]]、[[數_(語法)|數]]和格而變化詞尾；動詞詞形變化要承載關於[[人稱|人稱]]、數、[[時態|時態]]和[[語氣|語氣]]的信息。

=== 名詞變格 ===
巴利語名詞為三個文法[[性_(語法)|性]]（陽性，陰性，中性）和兩個數（單數，複數）而詞尾變化。名詞還在原則上顯示八個[[格_(語法)|格]]：[[主格|主格]]（{{IAST|paccatta}}），[[呼格|呼格]]，[[賓格|賓格]]（{{IAST|upayoga}}），[[工具格|工具格]]（{{IAST|karaṇa}}），[[與格|與格]]（{{IAST|sampadāna}}），[[離格|離格]]，[[屬格|屬格]]（{{IAST|sāmin}}），和[[方位格|方位格]]（{{IAST|bhumma}}）；但是在很多實例中，兩個或多個格在形式上是同一的；特別是對於屬格和與格。

== 書寫 ==

=== 帶有變音符號的巴利語字母 ===
歷史上，巴利語的最先書寫記錄據信是在[[斯里蘭卡|斯里蘭卡]]基於早先口頭傳承而寫成的。依據斯里蘭卡的大編年史{{IAST|[[大史|Mahavamsa]]}}，由於國家鬧大饑荒，佛教僧侶在公元前100年{{IAST|Vattagamini}}王期間寫下了巴利語圣典。寫下的巴利語的傳播保持了一套通用的字母音值系統，但使用各種各樣的實際文字來表達這些音值。這使很多西方人糊涂，他們意圖假定一種文字不可避免的連結著一組音素。

在[[斯里蘭卡|斯里蘭卡]]，巴利語文本記錄於新哈勒文（Sinhala，[[僧伽羅文|僧伽羅文]]）。其他地區文字，最顯著的有[[高棉文|高棉文]]、[[緬甸文|緬甸文]]、傣文（[[老傣文|传统傣仂文]]）和現代[[泰語字母|泰文]]（自從1893年），[[天城文|天城文]]和东南亚孟族人（Mon）的[[孟文|孟文]]、寮文（寮国Laos）也已經被用來記錄巴利語。 

自從十九世紀，巴利語已經用羅馬文字書寫了。[[Frans_Velthuis|Frans Velthuis]]設計了一個可選方案允許使用普通[[ASCII|ASCII]]碼來打字而不用[[變音符號|變音符號]]，但是它比標準Rhys Davids系統（見下）要更少可讀性。

巴利語字母如下排序（以天城字母的順序為基準，巴利語專用的發音擺在最後）：
:a ( अ )、ā ( आ )、i ( इ )、ī ( ई )、
:u ( उ )、ū ( ऊ )、e ( ए )、o ( ओ )、-ṃ ( अं )、
:k ( क )、kh ( ख )、g ( ग )、gh ( घ )、ṅ ( ङ )、
:c ( च )、ch ( छ )、j ( ज )、jh ( झ )、ñ ( ञ )、
:ṭ ( ट )、ṭh ( ठ )、ḍ ( ड )、ḍh ( ढ )、ṇ ( ण )、
:t ( त )、th ( थ )、d ( द )、dh ( ध )、n ( न )、
:p ( प )、ph ( फ )、b ( ब )、bh ( भ )、m ( म )、
:y ( य )、r ( र )、l ( ल )、v ( व )、
:s ( स )、h ( ह )、ḷ ( ळ )、ḷh ( ऴ )。

{{IAST|ḷh}}儘管是{{IAST|ḷ}}和{{IAST|h}}的連寫，但實際上是一個單一音素，如同kh、gh、ch...等等。

=== 計算機上的巴利語轉寫 ===

有多種字體用於巴利語轉寫，但是不推薦老的ASCII字體比如Leedsbit PaliTranslit, Times_Norman, Times_CSX+, Skt Times, Vri RomanPali CN/CB等，因為它們彼此不兼容并且在技術上過時了。正好相反，推薦基於[[Unicode|Unicode]]標準的字體，因為Unicode是所有字體的前景還有它們易於相互移植。

但是不是所有Unicode字體都包含必須的字符。要正確的顯示用於羅馬化巴利語（或梵語）的所有變音符號，Unicode字體必須包含下列字符范圍：
* Basic Latin: U+0000 – U+007F
* Latin-1 Supplement: U+0080 – U+00FF
* Latin Extended-A: U+0100 – U+017F
* Latin Extended-B: U+0180 – U+024F
* Latin Extended Additional: U+1E00 – U+1EFF

[http://www.palitext.com/subpages/PC_Unicode.htm 巴利圣典协会]向Windows和Linux計算機推薦[http://zencomp.com/greatwisdom/fonts/ VU-Times字體]和[http://depts.washington.edu/ebmp/software.php Gandhari Unicode字體]。[http://www.thdl.org/tools/fonts/diafonts.html 雪域數碼圖書館]推薦[https://web.archive.org/web/20070617033953/http://www.bcca.org/services/fonts/ Times Ext Roman字體]，和到其他一些可用於巴利語的Unicode字體的鏈接。還有一名英國佛教僧侶Bhikkhu Pesala提供了原為自己設計的一些巴利語Unicode [https://web.archive.org/web/20120320233551/http://www.aimwell.org/Fonts/fonts.html 字體]，和用於Windows XP的某些巴利語[https://web.archive.org/web/20120321073228/http://www.aimwell.org/Fonts/Keyboards/keyboards.html 鍵盤]。進一步的，Alanwood的Unicode資源的[http://www.alanwood.net/unicode/fonts.html 字體段落]鏈接了可用於巴利語錄入的一些通用字體，如果它們覆蓋了上述字符范圍的話。

== 巴利語與梵語 ==
儘管巴利語不能被認為是[[古典梵語|古典梵語]]或更早的[[吠陀梵語|吠陀梵語]]的直接後代，這些語言明顯有非常密切關聯，并且巴利語和[[梵語|梵語]]的共同特征總是很容易被熟悉二者的印度人識別出來。實際上，大部分的巴利語和梵語的詞幹都是在形式上同一的，只在詞尾變化的細節上有所區別。

這種聯繫是如此充分的眾所周知，以至於來自梵語的技術術語可以通過一組常規的音韻變換輕易的轉換成巴利語。這種變換模擬出了曾在原巴利語中發生的音韻發展的子集。由於這些變換的流行，不總能得知給定的巴利詞是古俗語詞匯的一部分，還是從梵語借來的變換後的詞。梵語詞有規律的對應於巴利詞的存在性不總是巴利詞源的可靠證據，因為在某些情況下，通過反向構詞可以從俗語詞創造出假造的梵語詞。   

下列音位學處理不意圖作為對從古印度祖先語言產生巴利語的歷史變化的詳盡描述，而是在梵語和巴利語之間最常見的音韻等式的總結，并且不聲稱其完備性。

=== 元音和雙元音 ===

* 梵語的{{IAST|ai}}和{{IAST|au}}總是分別單元音化為巴利語的{{IAST|e}}和{{IAST|o}}
::例如：{{IAST|m'''ai'''trī}} → {{IAST|m'''e'''ttā}}, {{IAST|'''au'''ṣadha}} → {{IAST|'''o'''sadha}}
* 梵語的{{IAST|aya}}和{{IAST|ava}}同樣經常簡約為巴利語的{{IAST|e}}和{{IAST|o}}
::例如：{{IAST|dhār'''aya'''ti}} → {{IAST|dhār'''e'''ti}}, {{IAST|'''ava'''tāra}} → {{IAST|'''o'''tāra}}, {{IAST|bh'''ava'''ti}} → {{IAST|h'''o'''ti}}
* 梵語的{{IAST|avi}}成為巴利語的{{IAST|e}}（就是{{IAST|avi}} → {{IAST|ai}} → {{IAST|e}}）
::例如：{{IAST|sth'''avi'''ra}} → {{IAST|th'''e'''ra}}
* 梵語的{{IAST|ṛ}}在巴利語中出現為{{IAST|a}}, {{IAST|i}}或{{IAST|u}}，經常一致於在隨后音節中元音。{{IAST|ṛ}}有時還在唇音之后變為{{IAST|u}}。
::例如：{{IAST|k'''ṛ'''ta}} → {{IAST|k'''a'''ta}}, {{IAST|t'''ṛ'''ṣṇa}} → {{IAST|t'''a'''ṇha}}, {{IAST|sm'''ṛ'''ti}} → {{IAST|s'''a'''ti}}, {{IAST|'''ṛ'''ṣi}} → {{IAST|'''i'''si}},  {{IAST|d'''ṛ'''ṣṭi}} → {{IAST|d'''i'''ṭṭhi}}, {{IAST|'''ṛ'''ddhi}} → {{IAST|'''i'''ddhi}}, {{IAST|'''ṛ'''ju}} → {{IAST|'''u'''ju}}, {{IAST|sp'''ṛ'''ṣṭa}} → {{IAST|ph'''u'''ṭṭha}}, {{IAST|v'''ṛ'''ddha}} → {{IAST|v'''u'''ddha}}

* 梵語的長元音在兩個隨后輔音的序列之前被縮短。
::例如：{{IAST|kṣ'''ānt'''i}} → {{IAST|kh'''ant'''i}}, {{IAST|r'''ājy'''a}} → {{IAST|r'''ajj'''a}}, {{IAST|'''īśv'''ara}} → {{IAST|'''iss'''ara}}, {{IAST|t'''īrṇ'''a}} → {{IAST|t'''iṇṇ'''a}},  {{IAST|p'''ūrv'''a}} → {{IAST|p'''ubb'''a}}

=== 輔音 ===
==== 音素變更 ====
* 梵語的噝擦音{{IAST|ś}}, {{IAST|ṣ}}和{{IAST|s}}合并到一起為巴利語的{{IAST|s}}
::例如：{{IAST|'''ś'''araṇa}} → {{IAST|'''s'''araṇa}}, {{IAST|do'''ṣ'''a}} → {{IAST|do'''s'''a}}
* 梵語的塞音{{IAST|ḍ}}和{{IAST|ḍh}}在元音之間成為{{IAST|ḷ}}和{{IAST|ḷh}}（同於吠陀梵語）
::例如：{{IAST|cakravā'''ḍ'''a}} → {{IAST|cakkavā'''ḷ'''a}}, {{IAST|virū'''ḍh'''a}} → {{IAST|virū'''ḷh'''a}}

==== 同化 ====
===== 一般規則 =====

* 很多從一個輔音到臨近輔音的[[同化_(語言學)|同化]]出現在巴利語的發展中，產生了大量的[[長輔音|長]]（雙）輔音。因為長輔音的[[送氣|送氣]]只在這個簇的最后輔音上是可識別的，長輔音{{IAST|kh, gh, ch, jh, ṭh, ḍh, th, dh, ph, bh}}出現為{{IAST|kkh, ggh, cch, jjh, ṭṭh, ḍḍh, tth, ddh, pph, bbh}}，而非''khkh, ghgh''等。
* 在同化會在詞首生成長輔音（或不送氣塞音+送氣塞音的序列）的時候，詞首長輔音要簡化為一個單一輔音。
::例如：{{IAST|'''pr'''āṇa}} → {{IAST|'''p'''āṇa}}（非'''''pp'''āṇa''）, {{IAST|'''sth'''avira}} → {{IAST|'''th'''era}}（非'''''tth'''era''）, {{IAST|'''dhy'''āna}} → {{IAST|'''jh'''āna}}（非'''''jjh'''āna''）, {{IAST|'''jñ'''āti}} → {{IAST|'''ñ'''āti}}（非'''''ññ'''āti''）
* 在同化會在詞中間生成三輔音序列的時候，長輔音要簡化直到在序列中只有兩個輔音。
::例如：{{IAST|u'''ttr'''āsa}} → {{IAST|u'''tt'''āsa}}（非''u'''ttt'''āsa''）, {{IAST|ma'''ntr'''a}} → {{IAST|ma'''nt'''a}}（非''ma'''ntt'''a''）, {{IAST|i'''ndr'''a}} → {{IAST|i'''nd'''a}}（非''i'''ndd'''a''）, {{IAST|va'''ndhy'''a}} → {{IAST|va'''ñjh'''a}}（非''va'''ñjjh'''a''）
* 從同化得出的序列{{IAST|vv}}要變更為{{IAST|bb}}
::例如：{{IAST|sa'''rv'''a}} → {{IAST|sa'''vv'''a}} → {{IAST|sa'''bb'''a}}, {{IAST|pra'''vr'''ajati}} → {{IAST|pa'''vv'''ajati}} → {{IAST|pa'''bb'''ajati}}, {{IAST|di'''vy'''a}} → {{IAST|di'''vv'''a}} → {{IAST|di'''bb'''a}}

===== 整體同化 =====

整體同化就是一個音素變為同一於臨近音素，它有兩種類型：隨後同化是被同化的音素同一於隨后的音素；而跟前同化是被同化的音素同一於前面的音素。

====== 隨後同化 ======
* 內部的{{IAST|[[visarga|visarga]]}}同化為隨後的清塞音或噝擦音
::例如：{{IAST|du'''ḥk'''ṛta}} → {{IAST|du'''kk'''ata}}, {{IAST|du'''ḥkh'''a}} → {{IAST|du'''kkh'''a}}, {{IAST|du'''ḥpr'''ajña}} → {{IAST|du'''pp'''añña}}, {{IAST|ni'''ḥkr'''odha}}（={{IAST|ni'''ṣkr'''odha}}）→ {{IAST|ni'''kk'''odha}}, {{IAST|ni'''ḥp'''akva}}（={{IAST|ni'''ṣp'''akva}}）→ {{IAST|ni'''pp'''akka}}, {{IAST|ni'''ḥś'''oka}} → {{IAST|ni'''ss'''oka}}, {{IAST|ni'''ḥs'''attva}} → {{IAST|ni'''ss'''atta}}
* 兩個不類似的梵語塞音的序列中，第一個塞音同化為第二個塞音
::例如：{{IAST|vimu'''kt'''i}} → {{IAST|vimu'''tt'''i}}, {{IAST|du'''gdh'''a}} → {{IAST|du'''ddh'''a}}, {{IAST|u'''tp'''āda}} → {{IAST|u'''pp'''āda}}, {{IAST|pu'''dg'''ala}} → {{IAST|pu'''gg'''ala}}, {{IAST|u'''dgh'''oṣa}} → {{IAST|u'''ggh'''osa}}, {{IAST|a'''dbh'''uta}} → {{IAST|a'''bbh'''uta}}, {{IAST|śa'''bd'''a}} → {{IAST|sa'''dd'''a}}
* 在兩個不類似鼻音的序列中，第一個鼻音同化為第二個鼻音
::例如：{{IAST|u'''nm'''atta}} → {{IAST|u'''mm'''atta}}, {{IAST|pradyu'''mn'''a}} → {{IAST|pajju'''nn'''a}}
* {{IAST|j}}同化為隨后的{{IAST|ñ}}（就是{{IAST|jñ}}成為{{IAST|ññ}}）
::例如：{{IAST|pra'''jñ'''ā}} → {{IAST|pa'''ññ'''ā}}, {{IAST|'''jñ'''āti}} → {{IAST|'''ñ'''āti}}
* 梵語的流音{{IAST|r}}和{{IAST|l}}同化為隨後的塞音、鼻音、噝擦音或{{IAST|v}}
::例如：{{IAST|mā'''rg'''a}} → {{IAST|ma'''gg'''a}}, {{IAST|ka'''rm'''a}} → {{IAST|ka'''mm'''a}}, {{IAST|va'''rṣ'''a}} → {{IAST|va'''ss'''a}}, {{IAST|ka'''lp'''a}} → {{IAST|ka'''pp'''a}}, {{IAST|sa'''rv'''a}} → {{IAST|sa'''vv'''a}} → {{IAST|sa'''bb'''a}}
* {{IAST|r}}同化為隨後的{{IAST|l}}
::例如：{{IAST|du'''rl'''abha}} → {{IAST|du'''ll'''abha}}, {{IAST|ni'''rl'''opa}} → {{IAST|ni'''ll'''opa}}
* {{IAST|d}}有時同化為隨後的{{IAST|v}}，生成{{IAST|vv}} → {{IAST|bb}}
::例如：{{IAST|u'''dv'''igna}} → {{IAST|u'''vv'''igga}} → {{IAST|u'''bb'''igga}}, {{IAST|'''dv'''ādaśa}} → {{IAST|'''b'''ārasa}}（比較於{{IAST|'''dv'''ādasa}}）
* {{IAST|t}}和{{IAST|d}}在出現於詞素邊界上的時候，可以同化為隨後的{{IAST|s}}或{{IAST|y}}
::例如：{{IAST|u'''t'''+'''s'''ava}} → {{IAST|u'''ss'''ava}}, {{IAST|u'''d'''+'''y'''āna}} → {{IAST|u'''yy'''āna}}

====== 跟前同化 ======

* 鼻音有時同化為前面的塞音（在其他插音出現的情況請參見后面）
::例如：{{IAST|a'''gn'''i}} → {{IAST|a'''gg'''i}}, {{IAST|ā'''tm'''an}} → {{IAST|a'''tt'''a}}, {{IAST|prā'''pn'''oti}} → {{IAST|pa'''pp'''oti}}, {{IAST|śa'''kn'''oti}} → {{IAST|sa'''kk'''oti}}
* {{IAST|m}}同化為詞首噝擦音
::例如：{{IAST|'''sm'''arati}} → {{IAST|'''s'''arati}}, {{IAST|'''sm'''ṛti}} → {{IAST|'''s'''ati}}
* 鼻音同化為前面的塞音+噝擦音簇，它接著有如沒有跟隨鼻音那樣發展（參見后面的部分同化）
::例如：{{IAST|tī'''kṣṇ'''a}} → {{IAST|ti'''kṣ'''a}} → {{IAST|ti'''kkh'''a}}, {{IAST|la'''kṣm'''ī}} → {{IAST|la'''kṣ'''ī}} → {{IAST|la'''kkh'''ī}}
* 梵語的流音{{IAST|r}}和{{IAST|l}}同化為前面的塞音、鼻音、噝擦音或{{IAST|v}}
::例如：{{IAST|'''pr'''āṇa}} → {{IAST|'''p'''āṇa}}, {{IAST|'''gr'''āma}} → {{IAST|'''g'''āma}},  {{IAST|'''śr'''āvaka}} → {{IAST|'''s'''āvaka}}, {{IAST|a'''gr'''a}} → {{IAST|a'''gg'''a}}, {{IAST|i'''ndr'''a}} → {{IAST|i'''nd'''a}}, {{IAST|pra'''vr'''ajati}} → {{IAST|pa'''vv'''ajati}} → {{IAST|pa'''bb'''ajati}}, {{IAST|a'''śr'''u}} → {{IAST|a'''ss'''u}}
* {{IAST|y}}同化為前面的非齒/卷舌塞音或鼻音
::例如：{{IAST|'''cy'''avati}} → {{IAST|'''c'''avati}}, {{IAST|'''jy'''otiṣ}} → {{IAST|'''j'''oti}}, {{IAST|rā'''jy'''a}} → {{IAST|ra'''jj'''a}}, {{IAST|ma'''tsy'''a}} → {{IAST|ma'''cchy'''a}} → {{IAST|ma'''cch'''a}}, {{IAST|la'''psy'''ate}} → {{IAST|la'''cchy'''ate}} → {{IAST|la'''cch'''ati}}, {{IAST|a'''bhy'''āgata}} → {{IAST|a'''bbh'''āgata}}, {{IAST|ā'''khy'''āti}} → {{IAST|a'''kkh'''āti}}, {{IAST|saṃ'''khy'''ā}} → {{IAST|saṅ'''kh'''ā}}（{{IAST|saṅ'''khy'''ā}}也同樣）, {{IAST|ra'''my'''a}} → {{IAST|ra'''mm'''a}}
* {{IAST|y}}同化為前面的非詞首{{IAST|v}}，生成{{IAST|vv}} → {{IAST|bb}}
::例如：{{IAST|di'''vy'''a}} → {{IAST|di'''vv'''a}} → {{IAST|di'''bb'''a}}, {{IAST|vedita'''vy'''a}} → {{IAST|vedita'''vv'''a}} → {{IAST|vedita'''bb'''a}}, {{IAST|bhā'''vy'''a}} → {{IAST|bha'''vv'''a}} → {{IAST|bha'''bb'''a}}
* {{IAST|y}}和{{IAST|v}}同化為任何前面的塞擦音，生成{{IAST|ss}}
::例如：{{IAST|pa'''śy'''ati}} → {{IAST|pa'''ss'''ati}}, {{IAST|'''śy'''ena}} → {{IAST|'''s'''ena}}, {{IAST|a'''śv'''a}} → {{IAST|a'''ss'''a}}, {{IAST|ī'''śv'''ara}} → {{IAST|i'''ss'''ara}}, {{IAST|kari'''ṣy'''ati}} → {{IAST|kari'''ss'''ati}}, {{IAST|ta'''sy'''a}} → {{IAST|ta'''ss'''a}}, {{IAST|'''sv'''āmin}} → {{IAST|'''s'''āmī}}
* {{IAST|v}}有時同化為前面的塞音
::例如：{{IAST|pa'''kv'''a}} → {{IAST|pa'''kk'''a}}, {{IAST|ca'''tv'''āri}} → {{IAST|ca'''tt'''āri}}, {{IAST|sa'''ttv'''a}} → {{IAST|sa'''tt'''a}}, {{IAST|'''dhv'''aja}} → {{IAST|'''dh'''aja}}

===== 部分和相互同化 =====

* 梵語的在塞音前的[[噝擦音|噝擦音]]同化為這個塞音，并且如果這個塞音不是送氣的，則變為送氣的；就是說{{IAST|śc, st,  ṣṭ}}和{{IAST|sp}}變為{{IAST|cch, tth, ṭṭh}}和{{IAST|pph}}
::例如：{{IAST|pa'''śc'''āt}} → {{IAST|pa'''cch'''ā}}, {{IAST|a'''st'''i}} → {{IAST|a'''tth'''i}}, {{IAST|'''st'''ava}} → {{IAST|'''th'''ava}}, {{IAST|śre'''ṣṭh'''a}} → {{IAST|se'''ṭṭh'''a}}, {{IAST|a'''ṣṭ'''a}} → {{IAST|a'''ṭṭh'''a}}, {{IAST|'''sp'''arśa}} → {{IAST|'''ph'''assa}}
* 在噝擦音-塞音-流音序列中，流音同化為前面的輔音，并且這個簇表現得如同噝擦音-塞音序列，就是說{{IAST|str}}和{{IAST|ṣṭr}}變為{{IAST|tth}}和{{IAST|ṭṭh}}
::例如：{{IAST|śā'''str'''a}} → {{IAST|śa'''st'''a}} → {{IAST|sa'''tth'''a}}, {{IAST|rā'''ṣṭr'''a}} → {{IAST|ra'''ṣṭ'''a}} → {{IAST|ra'''ṭṭh'''a}}
* {{IAST|t}}和{{IAST|p}}在{{IAST|s}}之前成為{{IAST|c}}，并且這個噝擦音同化為前面的音素并送氣（就是說序列{{IAST|ts}}和{{IAST|ps}}成為{{IAST|cch}}）
::例如：{{IAST|va'''ts'''a}} → {{IAST|va'''cch'''a}}, {{IAST|a'''ps'''aras}} → {{IAST|a'''cch'''arā}}
* 噝擦音同化為前面的{{IAST|k}}并送氣（就是說序列{{IAST|kṣ}}變為{{IAST|kkh}}）
::例如：{{IAST|bhi'''kṣ'''u}} → {{IAST|bhi'''kkh'''u}}, {{IAST|'''kṣ'''ānti}} → {{IAST|'''kh'''anti}}
* 跟隨著{{IAST|y}}的任何齒或卷舌塞音或鼻音都轉換成相應的硬腭音，并且{{IAST|y}}同化為這個新輔音，就是說{{IAST|ty, thy, dy, dhy, ny}}成為{{IAST|cc, cch, jj, jjh, ññ}}；類似的{{IAST|ṇy}}變為{{IAST|ññ}}。在塞音前的鼻音變成共享這種變化的硬腭音。
::例如：{{IAST|'''ty'''ajati}} → {{IAST|'''cy'''ajati}} → {{IAST|'''c'''ajati}}, {{IAST|sa'''ty'''a}} → {{IAST|sa'''cy'''a}} → {{IAST|sa'''cc'''a}}, {{IAST|mi'''thy'''ā}} →  {{IAST|mi'''chy'''ā}} → {{IAST|mi'''cch'''ā}}, {{IAST|vi'''dy'''ā}} → {{IAST|vi'''jy'''ā}} →  {{IAST|vi'''jj'''ā}}, {{IAST|ma'''dhy'''a}} → {{IAST|ma'''jhy'''a}} → {{IAST|ma'''jjh'''a}}, {{IAST|a'''ny'''a}} → {{IAST|a'''ñy'''a}} → {{IAST|a'''ññ'''a}}, {{IAST|pu'''ṇy'''a}} → {{IAST|pu'''ñy'''a}} → {{IAST|pu'''ññ'''a}}, {{IAST|va'''ndhy'''a}} →  {{IAST|va'''ñjhy'''a}} → {{IAST|va'''ñjjh'''a}} → {{IAST|va'''ñjh'''a}}
* 序列{{IAST|mr}}變為{{IAST|mb}}，通過在鼻音和流音之間插入一個塞音，隨后把這個流音同化為這個塞音，并且最后簡化結果的長輔音。
::例如：{{IAST|ā'''mr'''a}} → {{IAST|a'''mbr'''a}} → {{IAST|a'''mb'''a}}, {{IAST|tā'''mr'''a}} → {{IAST|ta'''mb'''a}}

==== 插音 ====
有時在特定輔音序列中插入[[插音|插音]]元音。如對{{IAST|ṛ}}，可以插入{{IAST|a}}, {{IAST|i}}或{{IAST|u}}，依賴於鄰接輔音或在隨后音節中元音的影響。{{IAST|i}}經常出現在{{IAST|i}}, {{IAST|y}}或硬腭音附近；{{IAST|u}}經常出現在{{IAST|u}}, {{IAST|v}}或唇音附近。
* 塞音+鼻音序列有時用{{IAST|a}}或{{IAST|u}}分離
::例如：{{IAST|ra'''tn'''a}} → {{IAST|ra'''tan'''a}}, {{IAST|pa'''dm'''a}} → {{IAST|pa'''dum'''a}}（{{IAST|u}}受唇音{{IAST|m}}的影響）
* 序列{{IAST|sn}}在詞首可以變為{{IAST|sin}}
::例如：{{IAST|'''sn'''āna}} → {{IAST|'''sin'''āna}}, {{IAST|'''sn'''eha}} → {{IAST|'''sin'''eha}}
* {{IAST|i}}可以插入在輔音和{{IAST|l}}之間
::例如：{{IAST|'''kl'''eśa}} → {{IAST|'''kil'''esa}}, {{IAST|'''gl'''āna}} → {{IAST|'''gil'''āna}}, {{IAST|'''ml'''āyati}} → {{IAST|'''mil'''āyati}}, {{IAST|'''śl'''āghati}} → {{IAST|'''sil'''āghati}}
* 在詞首噝擦音和{{IAST|r}}之間可以插入插音元音
::例如：{{IAST|'''śr'''ī}} → {{IAST|'''sir'''ī}}
* 序列{{IAST|ry}}一般變為{{IAST|riy}}（{{IAST|i}}受隨后的{{IAST|y}}的影響），但出於元音縮短的目的仍被當作雙輔音序列
::例如：{{IAST|'''āry'''a}} → {{IAST|'''ary'''a}} → {{IAST|'''ariy'''a}}, {{IAST|s'''ūry'''a}} → {{IAST|s'''ury'''a}} → {{IAST|s'''uriy'''a}}, {{IAST|v'''īry'''a}} → {{IAST|v'''iry'''a}} → {{IAST|v'''iriy'''a}}
* {{IAST|a}}或{{IAST|i}}插入{{IAST|r}}和{{IAST|h}}之間
::例如：{{IAST|a'''rh'''ati}} → {{IAST|a'''rah'''ati}}, {{IAST|ga'''rh'''ā}} → {{IAST|ga'''rah'''ā}}, {{IAST|ba'''rh'''iṣ}} → {{IAST|ba'''rih'''isa}}
* 在其他的輔音序列中有零星的插音
::例如：{{IAST|cai'''ty'''a}} → {{IAST|ce'''tiy'''a}}（非''ce'''cc'''a''）, {{IAST|va'''jr'''a}} → {{IAST|va'''jir'''a}}（非''va'''jj'''a''）

==== 其他變更 ====
* 梵語的任何在鼻音前的噝擦音都變為跟隨著{{IAST|h}}的鼻音序列，就是說{{IAST|ṣṇ, sn}}和{{IAST|sm}}成為{{IAST|ṇh, nh}}和{{IAST|mh}}
::例如：{{IAST|tṛ'''ṣṇ'''a}} → {{IAST|ta'''ṇh'''a}}, {{IAST|u'''ṣṇ'''īṣa}} → {{IAST|u'''ṇh'''īsa}}, {{IAST|a'''sm'''i}} → {{IAST|a'''mh'''i}}
* 序列{{IAST|śn}}成為{{IAST|ñh}}，由於{{IAST|n}}同化為前面的硬腭噝擦音
::例如：{{IAST|pra'''śn'''a}} → {{IAST|pra'''śñ'''a}} → {{IAST|pa'''ñh'''a}}
* 序列{{IAST|hy}}和{{IAST|hv}}經歷[[對調_(語言學)|對調]]
::例如：{{IAST|ji'''hv'''ā}} → {{IAST|ji'''vh'''ā}}, {{IAST|gṛ'''hy'''a}} → {{IAST|ga'''yh'''a}}, {{IAST|gu'''hy'''a}} → {{IAST|gu'''yh'''a}}
* {{IAST|h}}經歷與隨后的鼻音的對調
::例如：{{IAST|gṛ'''hṇ'''āti}} → {{IAST|ga'''ṇh'''āti}}
* {{IAST|y}}在{{IAST|e}}和元音之間成為長輔音
::例如：{{IAST|śr'''ey'''as}} → {{IAST|s'''eyy'''a}}, {{IAST|Maitr'''ey'''a}} → {{IAST|Mett'''eyy'''a}}

* 濁送氣音比如{{IAST|bh}}和{{IAST|gh}}罕見的偶爾變為{{IAST|h}}
::例如：{{IAST|'''bh'''avati}} → {{IAST|'''h'''oti}}, {{IAST|-e'''bh'''iṣ}} → {{IAST|-e'''h'''i}}, {{IAST|la'''gh'''u}} → {{IAST|la'''h'''u}}
* 齒音和卷舌音零星的變為對方
::例如：{{IAST|jñā'''n'''a}} → {{IAST|ñā'''ṇ'''a}}（非''ñā'''n'''a''）, {{IAST|'''d'''ahati}} → {{IAST|'''ḍ'''ahati}}（比較於{{IAST|Pāli}} {{IAST|'''d'''ahati}}）{{IAST|nī'''ḍ'''a}} → {{IAST|nī'''l'''a}}（非''nī'''ḷ'''a''）, {{IAST|'''sth'''āna}} → {{IAST|'''ṭh'''āna}}（非'''''th'''āna''）, {{IAST|duḥkṛ'''t'''a}} → {{IAST|dukka'''ṭ'''a}}（比較於{{IAST|Pāli}} {{IAST|dukka'''t'''a}}）

=== 例外 ===
有對上述規則的一些顯著的例外；其中很多是常用的俗語詞而不是從梵語中借用來的。
* {{IAST|ā'''ry'''a}} → {{IAST|a'''yy'''a}}（比較於{{IAST|a'''riy'''a}}）
* {{IAST|g'''u'''ru}} → {{IAST|g'''a'''ru}}（adj.）（比較於{{IAST|g'''u'''ru}}（n.））
* {{IAST|pur'''uṣ'''a}} → {{IAST|pur'''is'''a}}（非''pur'''us'''a''）
* {{IAST|'''vṛ'''kṣa}} → {{IAST|'''ru'''kṣa}} → {{IAST|'''ru'''kkha}}（非'''''va'''kkha''）
