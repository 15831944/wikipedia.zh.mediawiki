'''巴达赫尚省'''（{{lang-fa|ولایت بدخشان|Velâyat-e Badakhšân}}，{{lang-ps|د بدخشان ولایت‎|Badaxšān wilāyat}}），又譯'''巴達克山'''，位於[[阿富汗|阿富汗]]東北角，包括[[瓦罕走廊|瓦罕走廊]]地区，與[[塔哈爾省|塔哈爾省]]、[[潘傑希爾省|潘傑希爾省]]、[[努爾斯坦省|努爾斯坦省]]等省份及[[塔吉克|塔吉克]]、[[中華人民共和國|中國]]和[[巴基斯坦|巴基斯坦]]控制下的[[克什米尔|克什米尔]]相鄰，是[[巴达克山|巴达克山]]地区的一部分。中华民国政府曾經宣称[[瓦罕走廊|瓦罕走廊]]应该属于[[中华民国版图|中华民国版图]]，属[[中华民国新疆省|中华民国新疆省]]的一部分（详见[[中华民国行政区划|中华民国行政区划]]）。

== 地理 ==
[[File:Wakhan,_Badakhshan.jpg|250px]]
[[File:Afghanistan_Badakhshan_numbered_gray.PNG|250px]]

巴达赫尚主要在北面和东面与[[塔吉克斯坦|塔吉克斯坦]]接壤，东面的一个长条被称作[[瓦罕走廊|瓦罕走廊]]，这里与[[巴基斯坦|巴基斯坦]]北部接壤，在它的最东边，瓦罕走廊的另外一端与[[中華人民共和國|中国]]接壤。整个省的总面积为44,059[[平方公里|平方公里]]，大部分是[[兴都库什|兴都库什]]山和[[帕米尔|帕米尔]]山脉。该省共分为13个行政区。

== 历史 ==
{{See also|巴达克山}}

巴达赫尚是古代[[丝绸之路|丝绸之路]]的一个中转站，巴达赫尚的名字是萨珊王朝起的，名字源于一个词「badaxš」，一个萨珊王朝的官方头衔，这个名字的后缀 -ān，则表示这个区域属于一个拥有badaxš头衔的人<ref name=Iranica>{{cite encyclopedia|last=Eilers |first=W. |encyclopedia=[[Encyclopædia_Iranica|Encyclopædia Iranica]] |title=BADA<u>K</u>ŠĀN |url=http://www.iranica.com/newsite/articles/v3f4/v3f4a011.html |accessdate=January 2008 |edition=Online Edition |publisher=[[Columbia_University|Columbia University]] |location=United States |deadurl=yes |archiveurl=https://web.archive.org/web/20090102233859/http://www.iranica.com/newsite/articles/v3f4/v3f4a011.html |archivedate=2009-01-02 }}</ref>（类似[[德黑兰|德黑兰]]，[[阿塞拜疆|阿塞拜疆]]和[[伊斯法罕|伊斯法罕]]）。

巴达赫尚和[[潘傑希爾省|潘傑希爾省]]是仅有的两个在[[塔利班|塔利班]]扩张中未能控制的省份。但是在战争期间，Mawlawi Shariqi建立了一个非塔利班的伊斯兰酋长国，同时临近的[[努尔斯坦省|努尔斯坦省]]发生了伊斯兰革命。巴达赫尚本地人，阿富汗前总统[[布爾漢努丁·拉巴尼|布爾漢努丁·拉巴尼]]和[[艾哈迈德·沙阿·马苏德|艾哈迈德·沙阿·马苏德]]是反塔利班的[[北方聯盟_(阿富汗)|北方联盟]]在塔利班巅峰时期的最后的剩余。巴达赫尚即将陷落的时候，[[美国|美国]]用其[[空军|空军]]力量和大量的援助让北方联盟恢复了对该省的控制。

该省目前的总督是孟什·阿卜杜拉·马吉德，之前一任是赛义德·阿敏·塔里克。在塔利班失败之后，中国对该地区表现出了巨大的兴趣，帮助重建了道路和基础设施。

== 经济 ==
[[File:Kuchi_caravan_in_Badakhshan,_Afghanistan.jpg|300px]]
尽管拥有巨大的矿藏，巴达赫尚仍然是全世界最贫穷的地区之一。种植[[罂粟|罂粟]]是该省唯一实际的收入，而且，由于缺乏基本的医疗设施，和很多难以接近的区域，以及极为严酷的冬季，这里有着全世界最高的[[产妇|产妇]][[死亡率|死亡率]]。

巴达赫尚的[[天青石|天青石]]从几个世纪之前就开始开采，在古代它曾经是世界上最大，最著名的天青石产地<ref>Deer, William A.; Howie, Robert A, and Zussman, Joseph (1963) "Lapis lazuli" ''Rock-Forming Minerals'' Longman, London, [http://www.worldcat.org/oclc/61975619 OCLC 61975619]</ref><ref>Lapis lazuli was also found in the Urals Mountains in Russia. Deer ''et al.'' above</ref> 。巴达赫尚最近几年的很多采矿活动都是为了开采天青石。这些开采活动被用来支持北方联盟的部队，在北方联盟之前，这些天青石被用来支持反对[[苏联|苏联]]的圣战者。最近的一些地质勘探表明，这一区域有其他宝石矿藏，其中包括[[红宝石|红宝石]]和绿宝石。这些矿藏的开发成了振兴该地区经济的关键。

== 首府 ==
[[File:Feyzabad_international.jpg|300px]]

[[法扎巴德_(阿富汗)|法扎巴德]]（{{lang-ps|فيض‌آباد}}）位于[[阿富汗|阿富汗]]东北部[[科克恰河|科克恰河]]畔，是巴达赫尚省的首府。人口大约有50,000。他是东北阿富汗和帕米尔地区主要的经济和政治中心，在冬季这座城市有时候会因为大雪而与外界隔离。1979年，这里曾经是一个阿富汗游击队抵制苏联入侵的重点地区，但是在1980年法扎巴德落入苏联人手中之后，成为了一座主要的要塞城市。

== 人口和语言 ==
该省人口估计有823,000，他们主要讲[[塔吉克语|塔吉克语]]，同时当地也有一些其他的印度伊朗语系的其他语言。当地有一些[[柯尔克孜族|柯尔克孜族]]。还有一些[[游牧|游牧]]半游牧状态的[[乌兹别克族|乌兹别克族]]人和[[普什图族|普什图族]]人他们随着季节变化做长距离的迁徙。本地居民之中大部分是[[逊尼派|逊尼派]]穆斯林，但是大部分帕米尔人是[[伊斯玛仪|伊斯玛仪]]派。
