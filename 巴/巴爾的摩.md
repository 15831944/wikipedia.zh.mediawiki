'''巴爾的摩'''（{{lang-en|Baltimore}} {{IPAc-en|ˈ|b|ɔː|l|t|ɨ|ˌ|m|ɔr}}；本地發音：{{IPA-all|ˈbɔɫ.mɔɻ|}}）是[[美国|美国]][[马里兰州|马里兰州]]最大的城市，也是美国最大独立城市和主要海港之一。巴尔的摩市被[[巴尔的摩县|巴尔的摩县]]环绕，但不属于巴尔的摩县，是马里兰州唯一的一个[[独立市_(美国)|独立市]]，因此经常称为'''巴尔的摩市'''。

== 历史 ==

巴尔的摩市建立於1729年，名字来源于马里兰地区封地的所有者[[巴尔的摩男爵|巴尔的摩男爵]]。本地的早期发展與海运和制糖业有密切相关。

1814年，在[[1812年战争|1812年战争]]期间英国军队在[[火烧华盛顿|火烧华盛顿]]之后，分海陆两路进攻巴尔的摩。同年9月12日，英国军队和美国民兵在巴尔的摩西南的北角发生交火。英国指挥官羅斯将军（Ross）被击毙。英国海军在对守卫巴尔的摩内港的[[麥克亨利堡|麥克亨利堡]]（Fort McHenry）进行通宵炮击后，並未能趕走守卫堡壘的美国军队，随后英国军队放弃对巴尔的摩的进攻。美国律師{{le|弗朗西斯·斯科特·基|Francis Scott Key}}（Francis Scott Key）在目睹了英国海军对麥克亨利堡的炮击后，写下了后来称为美国国歌的《[[星條旗歌|星條旗歌]]》(The Star-Spangled Banner)。

1851年，巴尔的摩市脱离巴尔的摩县，成为独立市。

1861年，一队美国士兵在巴尔的摩转火车时，遭到亲南方的巴尔的摩市民围攻，士兵向市民开枪後导致了[[1861年巴爾的摩暴動|巴尔的摩暴乱]]。此暴乱的直接结果是[[林肯|林肯]]政府对亲南方的州進行镇压，並宣布巴尔的摩进入军管状态，巴尔的摩市长、警察局长與[[马里兰州|马里兰州]]议会多名议员被捕。在整个美国内战期间，[[马里兰州|马里兰州]]被置于[[美国联邦政府|美国联邦政府]]的直接管理，[[德拉瓦州|德拉瓦州]]则被置为北方军事占领状态。[[阿肯色州|阿肯色州]]和[[田纳西州|田纳西州]]在了解到林肯政府对亲南方的地区的镇压后，宣布脱离美国北方政府加入南方政府。

1904年2月7日，巴尔的摩市发生大火，整个城市几乎被烧毁。从[[哥伦比亚特区|哥伦比亚特区]]、[[费城|费城]]至[[纽约|纽约]]等地都有消防队趕來救援，但由于各處消防水龙头的接头有所差異，因此救援人員面对大火无能为力。这件事件导致了[[國家標準技術研究所|國家標準技術研究所]]（NIST）的诞生。

1980年，[[国家水族馆巴尔的摩分馆|巴尔的摩国家水族馆]]建成。同年在巴尔的摩内港地区海建成一系列的的商业观光区，巴尔的摩内港逐渐从工业区和居民区转变成为观光旅游区。

2003年，[[飓风伊莎贝尔|飓风伊莎贝尔]]帶來的潮水，淹没了巴尔的摩内港地区。

== 地理 ==
巴尔的摩位于马里兰州中部，[[帕塔普斯科河|帕塔普斯科河]]河口地区，紧邻[[切薩皮克灣|切薩皮克灣]]，是优异的港口。面積238.5[[平方公里|平方公里]]，陸地為209.3平方公里，水域有29.2平方公里。

注：平均（气温、降水）数据来自市中心，极端数据来自国际交换站。

巴爾的摩属[[副热带湿润气候|副热带湿润气候]]，四季分明，气温变化相对和缓，全年降水分配均匀。冬季冷凉，微潮，时而偏向寒冷，日最高气温低于{{convert|0|C}}的平均日数为10.2天，日最低气温低于{{convert|0|C}}的平均日数为58天，低于{{convert|-10|C}}的平均日数为1.9天；夏季炎热潮湿，日最高气温达{{convert|30|C}}的日数年均有65天，{{convert|35|C}}以上的有9.9天。<ref name = "NWS Sterling, VA"/>最冷月（1月）均温{{convert|35.8|F|1|disp=flip}}，极端最低气温{{convert|−7|F|0|disp=flip}}（上次出现于1984年1月22日）。<ref name = "NWS Sterling, VA"/>最热月（7月）均温{{convert|80.7|F|1|disp=flip}}，极端最高气温{{convert|107|F|0|disp=flip}}（1936年7月10日）。<ref name = "NWS Sterling, VA"/>无霜期平均为252天（3月21日至11月27日），由于[[热岛效应|热岛效应]]的影响，市中心的无霜期较西、北郊，可多出50天以上；此外，西、北郊平均降雪量可达{{convert|30|in|cm|disp=flip}}。<ref name = "NWS Sterling, VA"/>可测量降雪平均期为12月14日至3月8日。<ref name = "NWS Sterling, VA"/>年均降水量约{{convert|40.9|in|sigfig=3|disp=flip}}，年极端最少降水量为{{convert|21.55|in|disp=flip}}（1930年），最多为{{convert|62.66|in|disp=flip}}（2003年）。<ref name = "NWS Sterling, VA"/>年均降雪量为{{convert|20.1|in|cm|disp=flip}}，<ref name = "NWS Sterling, VA"/>但降雪年际变化较大；1949–50年的降雪量最少，积累降雪量只有{{convert|0.7|in|cm|disp=flip}}，2009–10年的降雪量最多，积累降雪量为{{convert|77.0|in|cm|disp=flip}}。<ref name = "NWS Sterling, VA"/>

== 人文與環境 ==
===人口構成===
根據2006年美國人口調查結果，巴爾的摩64% 的居民是黑人（非洲裔美國人），30% 的人口是白人，4%是亞洲裔美國人（遠高於美國平均數字，美國12%的人口是黑人）。總括來說，巴爾的摩是美國黑人比較集中的城市。

== 文化影響 ==
在美國暢銷軍事小說作家[[湯姆·克蘭西|湯姆·克蘭西]]於1991年出版的「傑克·雷恩」系列第五作：「[[恐懼的總和|恐懼的總和]]」一書中描寫著：在[[新納粹主義|新納粹主義]][[恐怖份子|恐怖份子]]的策劃下，為製造假象煽動[[美國|美國]]與[[俄羅斯|俄羅斯]]全面開戰、其將一枚來自於[[以色列|以色列]]、後卻在[[以阿戰爭|以阿戰爭]]的戰場中失落的[[氫彈|氫彈]]，將外型偽裝成[[自動販賣機|自動販賣機]]、將該枚[[核武器|核武器]]於巴爾的摩職業[[美式足球|美式足球]]場引爆、造成數以萬計的市民喪生。

2002年开始首播的美国[[HBO|HBO]]电视台电视剧，也是被许多媒体和观众认为的历史上最伟大的电视剧《[[火线重案组|火线重案组]]》就以巴尔的摩市为舞台展开。
