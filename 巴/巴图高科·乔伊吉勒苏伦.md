'''巴图高科·乔伊吉勒苏伦'''（{{lang-mn|Баттогтох Чойжилсүрэн}}，Battogtokh Choijilsuren，{{bd|1970年|||}}）[[蒙古国|蒙古国]][[乌布苏省|乌布苏省]]人，蒙古国政治人物。

==生平==
乔伊吉勒苏伦生于[[乌布苏省|乌布苏省]]铁斯[[苏木|苏木]]（Тэс сум）。早年获[[俄罗斯|俄罗斯]][[叶卡捷琳堡|叶卡捷琳堡]][[乌拉尔联邦大学|乌拉尔工学院]]自动遥控机械师资格（Автоматикч телемеханикч мэргэжилтэй）。1993年至1995年，任[[俄罗斯|俄罗斯]][[叶卡捷琳堡|叶卡捷琳堡]]“СП-Каман”公司董事。1995年至1999年，任“Хурд Ч”ХЭАА-н的董事。1999年至2006年，任“Хурд Хүнс”有限责任公司董事。2006年至2008年，任[[蒙古国总统|蒙古国总统]]办公室副主任。2008年当选[[国家大呼拉尔|国家大呼拉尔]]委员，2012年连任<ref>{{Cite web |url = http://ardiinelch.mn/allpoll.php?id=132 |title =  Баттогтох Чойжилсүрэн|author =  |date =  |publisher = АРДЫН ЭЛЧ |language =  |accessdate =2017-06-27  |quote =  }}</ref>。

乔伊吉勒苏伦是[[蒙古人民党|蒙古人民党]]党员。2016年7月，乔伊吉勒苏伦被[[国家大呼拉尔|国家大呼拉尔]]任命为[[额尔登巴特内阁|额尔登巴特内阁]]的财政部部长<ref>{{Cite web |url = http://www.fmprc.gov.cn/web/zwbd_673032/wshd_673034/t1386190.shtml |title = 驻蒙古大使邢海明会见蒙古新任财政部长乔伊吉勒苏伦 |author =  |date = 2016-08-01 |publisher = 中华人民共和国外交部 |language =  |accessdate =  |quote =  }}</ref>。上任伊始，乔伊吉勒苏伦便对外公布了蒙古国宏观经济、金融、外债、外国投资等方面情况，指出蒙古国正面临严重的经济困难。2016年8月15日，乔伊吉勒苏伦表示，蒙古国新政府将采取一系列措施应对经济困难局面，近期将向国家大呼拉尔提交经济救助计划<ref>{{Cite web |url =http://jjckb.xinhuanet.com/2016-08/16/c_135601504.htm  |title =  蒙古国总理称将采取措施应对经济困局|author =  |date = 2016-08-16 |publisher =经济参考  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://finance.huanqiu.com/roll/2016-08/9320079.html |title =经济新政救得了断崖的蒙古吗  |author =  |date = 2016-08-17 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref>。
