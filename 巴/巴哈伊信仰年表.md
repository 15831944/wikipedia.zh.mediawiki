{{巴哈伊}}

以下是[[巴哈伊信仰|巴哈伊信仰]]和[[巴比教|巴比教]]简单的大事年表，欲了解事件详情，请见文末参考文献。
==1795==
* [[谢赫·艾哈迈德|谢赫·艾哈迈德]]开创[[谢赫运动|谢赫运动]]。

==1817==
* 11月12日，[[巴哈欧拉|巴哈欧拉]]诞生。

==1819==
* 10月20日，[[巴孛|巴孛]]诞生。

==1826==
* [[谢赫·艾哈迈德|谢赫·艾哈迈德]]逝世，[[赛义德·卡齐姆|赛义德·卡齐姆]]被指定为谢赫派的首领。

==1828==
* [[巴孛|巴孛]]的父亲，[[米尔扎·穆罕默德·里达|米尔扎·穆罕默德·里达]]去世，[[巴孛|巴孛]]被交由他的舅舅哈吉·米尔扎·赛义德·阿里照顾。

==1835==
* 9月24日-10月22日，[[巴哈欧拉|巴哈欧拉]]与[[阿希耶·哈努姆|阿希耶·哈努姆]]结婚。

==1843==
* [[赛义德·卡齐姆|赛义德·卡齐姆]]去世。去世前他指示自己包括[[穆拉·侯赛因|穆拉·侯赛因]]在内的学生，去寻找被允诺者，即[[马赫迪|马赫迪]]。

==1844==
* 5月23日，[[巴孛|巴孛]]在[[伊朗|伊朗]][[设拉子|设拉子]]向[[穆拉·侯赛因|穆拉·侯赛因]]宣示了自己的使命。
* 同日，[[巴哈欧拉|巴哈欧拉]]和[[阿希耶·哈努姆|阿希耶·哈努姆]]的长子[[阿博都巴哈|阿博都巴哈]]出生。
* 9月底，[[巴哈欧拉|巴哈欧拉]]接受了[[巴孛|巴孛]]的信仰<ref>{{Cite book

== 1845 ==
* 9月，[[巴孛|巴孛]]公开宣布自己是[[马赫迪|马赫迪]]后，[[设拉子|设拉子]]的巴比运动受到了压制。
* 西方政府首次涉及的报道是巴比教的穆拉·阿里-巴斯塔米被逮捕和监禁，首次发表在秋天的[[伦敦|伦敦]][[泰晤士报|泰晤士报]]上。<ref>{{cite web | last = Momen | first = Moojan | title = Early Western Accounts of the Babi and Baha'i Faiths | work = Encyclopedia articles | publisher = Bahá'í Library Online | year = 1999 (online) | url =http://bahai-library.com/momen_encyclopedia_western_accounts | accessdate = 2012-02-02}}</ref>
== 1846 ==
* [[巴哈欧拉|巴哈欧拉]]和[[阿希耶·哈努姆|阿希耶·哈努姆]]生下[[巴希伊·哈努姆|巴希伊·哈努姆]]
* 9月，[[巴孛|巴孛]]离开设拉子去[[伊斯法罕|伊斯法罕]]。 <ref name="Perkins">{{cite book | first = Mary | last = Perkins | title = Hour of the Dawn | publisher =  George Ronald | location = Oxford | year = 1987 | page = 212}}</ref>

== 1847 ==
* 7月，[[巴孛|巴孛]]被囚禁于[[马库|马库堡]]，写下了[[巴扬经|巴扬经]]<ref name="Perkins">{{cite book | first = Mary | last = Perkins | title = Hour of the Dawn | publisher =  George Ronald | location = Oxford | year = 1987 | page = 212}}</ref>

== 1848 ==
* [[巴哈欧拉|巴哈欧拉]]和[[阿希耶·哈努姆|阿希耶·哈努姆]]生下了[[米尔扎·米赫迪|米尔扎·米赫迪]]。
* [[阿博都-巴哈|阿博都-巴哈]]的妻子，[[穆尼丽·哈努姆|穆尼丽·哈努姆]]出生于[[伊斯法罕|伊斯法罕]]一户[[巴比教|巴比教]]望族。
* 3月20日，[[穆拉·侯赛因|穆拉·侯赛因]]在[[马库|马库]]拜访[[巴孛|巴孛]]。
* 4月10日，由于在马库声望与日俱增，巴孛被转送到奇赫里特的监狱。直到被处决前几天，他大部分时间都被关押在此。
* 6-7月，开[[巴达什特会议|巴达什特会议]]。<ref name="amanat">{{cite book | first = Abbas | last = Amanat | title = Resurrection and Renewal: The Making of the Babi Movement in Iran | publisher =  Cornell University Press | location = Ithaca | year = 1989 | page = 324}}</ref>
* 7月，对[[巴孛|巴孛]]进行公开审判，巴孛出人意料地进行了一次公开的宣示。他被转送回奇赫里特。
* 7月21日，[[穆拉·侯赛因|穆拉·侯赛因]]举着黑旗，与202名其他巴比教徒游行至[[马什哈德|马什哈德]]。
* 10月20日，穆拉·侯赛因和其他巴比信徒被围困在塔巴尔西堡。
* 10月20日，[[库都斯|库都斯]]到达塔巴尔西。

== 1849 ==
*巴哈欧拉与法蒂米在[[德黑兰|德黑兰]]结婚。
*2月2日，[[穆拉·侯赛因|穆拉·侯赛因]]在谢赫·塔尔巴西堡的战斗中阵亡。
*5月10日，塔尔巴西堡战斗在和谈投降后结束，战胜方答应放走巴比信徒。<ref>[https://web.archive.org/web/20030815222343/http://www.fortunecity.com/tatooine/acegarp/898/bahai_chrono.htm The NEXUS: Baha'i Chronology]</ref>
*5月16日，[[库都斯|库都斯]]被屠戮处决。

== 1850 ==
* 7月9日，[[巴孛|巴孛]]在[[大不里士|大不里士]]被公开处决。
* 秋天，关于巴比教的简讯在[[英国|英国]]和[[美国|美国]]的几家报纸见报。<ref>{{cite web | title = Early mention of Bábís in western newspapers, summer 1850 | work = Historical documents and Newspaper articles | publisher = Bahá'í Library Online | date = 2010-09-17 |origyear = Autumn 1850 | url = http://bahai-library.com/1850_brief_reports | accessdate = 2012-02-02}}</ref>

== 1852 ==
* 8月15日，愤怒的巴比信徒（出于自身的冲动），策划了一起失败的针对纳赛尔丁·沙阿的暗杀，后者为报复囚禁了[[巴哈欧拉|巴哈欧拉]]，处决了数千名巴比信徒，包括[[塔希莉|塔希莉]]和亚兹迪的赛义德·侯赛因。
* 9-12月，在[[德黑兰|德黑兰]]的“黑坑”被关押了4个月，[[巴哈欧拉|巴哈欧拉]]第一次得到启示，他是[[巴孛|巴孛]]预言的那个[[允诺者|允诺者]]。
* [[巴哈欧拉|巴哈欧拉]]和法蒂米的儿子米尔扎·穆罕默德·阿里在[[巴格达|巴格达]]出生。

== 1853 ==
* 1月12日，巴哈欧拉从[[德黑兰|德黑兰]]被流放至[[巴格达|巴格达]]

== 1854 ==
* 4月10日，巴哈欧拉隐居至库尔德斯坦的[[苏莱曼尼亚|苏莱曼尼亚]]山区，因为他和米尔扎·叶海亚之间的关系日渐紧张。

== 1856 ==
* [[巴哈欧拉|巴哈欧拉]]在库尔德斯坦被发现，在[[阿博都巴哈|阿博都巴哈]]的请求下回到[[巴格达|巴格达]]。

== 1857 ==
* [[巴哈欧拉|巴哈欧拉]]写下了[[隐言经|隐言经]]和[[四谷经|四谷经]]。

== 1860 ==
* 巴哈欧拉写下了[[七谷经|七谷经]]。

== 1861 ==
* 巴哈欧拉在1861年底或1862年初用两天一夜写下了[[笃信经|笃信经]]。

== 1862 ==
* 5月10日，波斯大使请求[[奥斯曼帝国|奥斯曼帝国]]将巴比信徒流放至更远的地方。

== 1863 ==
*4月21日，在被流放至君士坦丁堡（今[[伊斯坦布尔|伊斯坦布尔]]）之前，巴哈欧拉在[[巴格达|巴格达]]的[[里兹万花园|里兹万花园]]宣布宣示自己是[[上帝将要显圣的祂|上帝将要显圣的祂]]。
*12月12日，巴哈欧拉在君士坦丁堡4个月以后，被流放至正式的囚禁地阿德里安堡（今[[埃迪尔内|埃迪尔内]]）。

== 1865 ==
*巴哈欧拉写下了阿拉伯语[[艾哈迈德书简(阿拉伯语)|艾哈迈德书简]]。
*阿瑟·德·戈比诺出版了介绍巴比教的法语书籍。

== 1867 ==
* 巴哈欧拉开始写作并送寄[[致君王书简|致君王书简]]。

== 1868 ==
*8月5日，[[巴哈欧拉|巴哈欧拉]]和他大群的追随者从埃迪尔内被送往[[巴勒斯坦|巴勒斯坦]]的罪犯流放地[[阿卡(以色列)|阿卡]]（今[[以色列|以色列]]境内）。
*8月31日，巴哈欧拉到达阿卡。

== 1869 ==
*[[巴哈欧拉|巴哈欧拉]]给波斯的[[沙赫|沙赫]]纳赛尔丁·沙阿寄去了一封信，信使[[巴迪|巴迪]]被处死。

== 1870 ==
* 6月23日，米尔扎·米赫迪因为从天台跌落而去世。

== 1873 ==
*巴哈欧拉写作[[亚格达斯经|亚格达斯经]].

== 1886 ==
*[[阿希耶·哈努姆|阿希耶·哈努姆]]去世。

== 1889 ==
*布朗在英国的一系列学术讲座和论文中提到巴哈伊信仰。

== 1892 ==
*5月29日，巴哈欧拉逝世，他的遗体被安放在[[巴吉宅邸|巴吉宅邸]]旁的[[巴哈欧拉陵寝|陵寝]]，他最后的几年在[[巴吉宅邸|巴吉宅邸]]读过。在他的遗嘱中，他指定[[阿博都巴哈|阿博都巴哈]]为他的继任者，和信仰的领袖。

== 1893 ==
*9月23日，亨利·H·杰赛普博士在[[芝加哥|芝加哥]]世界宗教议会上第一次公开提到巴哈伊信仰。

== 1894 ==
* Thornton Chase为美国的第一批五个巴哈伊之一。

== 1897 ==
*3月1日，巴哈欧拉的长曾外孙[[守基·阿芬第|守基·阿芬第]]出生。

== 1898 ==
*第一批西方来的朝圣者到达[[阿卡(以色列)|阿卡]]，包括Phoebe Hearst和第一个美国黑人信徒，Robert Turner.

== 1901 ==
*位于[[土库曼斯坦|土库曼斯坦]][[阿什哈巴特|阿什哈巴特]]的第一座[[灵曦堂|巴哈伊灵曦堂]]奠基。

== 1903 ==
*[[伊朗|伊朗]][[亚兹达|亚兹达]]超过100名巴哈伊信徒在20世纪的第一场重大的[[对巴哈伊信仰的迫害|迫害巴哈伊]]运动中被杀。

== 1908 ==
*9月，[[阿博都巴哈|阿博都巴哈]]在64岁时从监禁和流放中释放。

== 1909 ==
*隐藏了59年之后，3月21日，[[巴孛|巴孛]]的遗体被安置在[[巴孛陵寝|巴孛陵寝]]。

== 1910 ==
*[[阿博都巴哈|阿博都巴哈]]8月到达埃及。
*玛丽·麦克斯韦，后被人称作[[鲁伊希·哈努姆|鲁伊希·哈努姆]]，在[[纽约|纽约]]出生。

== 1911 ==
*8月-12月，[[阿博都巴哈|阿博都巴哈]]穿越欧洲旅行，访问了诸多城市，如[[伦敦|伦敦]]，[[布里斯托尔|布里斯托尔]]，[[巴黎|巴黎]]。
* 9月10日，[[阿博都巴哈|阿博都巴哈]]在伦敦的城市教堂做了他的第一次面向西方公众的演讲，由Wellesley Tudor Pole做英语翻译。

== 1912 ==
*4月11日，[[阿博都巴哈|阿博都巴哈]]到达[[纽约|纽约]]，开始北美的访问。
*阿博都巴哈赠与位于[[伊利诺伊州|伊利诺伊州]]威尔米特的北美[[灵曦堂|巴哈伊灵曦堂]]的角石给Nettie Tobin.
*12月5日，阿博都巴哈起航离开北美去往欧洲。

== 1916 ==
*阿博都巴哈写下了十四篇中的八篇《[[神圣计划书简|神圣计划书简]]》。

== 1917 ==
*阿博都巴哈写了其余的六篇《神圣计划书简》。

== 1918 ==
*9月19日，阿博都巴哈在奥斯曼军队输掉了[[美吉多战役|美吉多战役]]前被人以性命威胁。<ref>[http://bahai-library.com/bsr/bsr09/9B3_lambden_armageddon.htm Catastrophe, Armageddon and Millennium: some aspects of the Bábí-Baha'i exegesis of apocalyptic symbolism]</ref>

== 1920 ==
*4月27日，阿博都巴哈被英国授予爵位，以表彰他在二战中的人道主义行为。

== 1921 ==
*11月28日，阿博都巴哈在海法去世，在他的[[阿博都巴哈的遗嘱|遗嘱]]中指定[[守基·阿芬第|守基·阿芬第]]为[[圣护|圣护]]。

== 1932 ==
*7月15日，[[巴哈欧拉|巴哈欧拉]]的女儿，[[巴希伊·哈努姆|巴希伊·哈努姆]]去世。

== 1935 ==
*[[守基·阿芬第|守基·阿芬第]]将《[[巴哈欧拉圣典选集》|巴哈欧拉圣典选集》]]从[[波斯文|波斯文]]和[[阿拉伯语|阿拉伯语]]翻译成英语。

== 1937 ==
*米尔扎·穆罕默德·阿里，被[[阿博都巴哈|阿博都巴哈]]称之为主要的圣约背叛者死亡。
*守基·阿芬第启动了“神圣计划”以在全球范围内传播巴哈伊信仰。
*守基·阿芬第与玛丽·麦克斯韦结婚，后者被称为[[鲁伊希·哈努姆|鲁伊希·哈努姆]]，加拿大著名巴哈伊家庭的女儿。

== 1944 ==
*[[守基·阿芬第|守基·阿芬第]]发表了《[[神临记|神临记]]》，纪念巴哈伊天启周期100年。

== 1951 ==
*全世界有11个运行中的国家灵体会。
*[[守基·阿芬第|守基·阿芬第]]另指定了32名[[圣辅|圣辅]]。
*[[守基·阿芬第|守基·阿芬第]]指定了第一个多国的巴哈伊团体，[[国际巴哈伊理事会|国际巴哈伊理事会]]。

== 1953 ==
* 守基·阿芬第启动了十年拓展计划。
* 威尔米特的北美[[灵曦堂|巴哈伊灵曦堂]]落成。
* [[巴孛陵寝|巴孛陵寝]]的上层结构完工。

== 1957 ==
*11月4日，守基·阿芬第去世，没有留下子嗣，也未指定[[圣护|圣护]]的继承人。临时的“信仰的领袖”的职责由27位[[圣辅|圣辅]]担当，以完成十年拓展计划，以及选举[[世界正义院|世界正义院]]。

== 1963 ==
*摩洛哥一波迫害巴哈伊的狂潮于4月中旬结束，在美国数月的外交和报纸、电视报道努力后，<ref>{{Cite book  | last = Rutstein  | first = Nathan  | title = From a Gnat to an Eagle: The Story of Nathan Rutstein  | publisher = US Baha'i Publishing Trust  | year = 2008  | page = 119  | url = http://books.google.com/books?id=aR2AIrHGrH4C&pg=PA719&hl=en&ei=B7sTTOGpOMWblgfU5oHCDA&sa=X&oi=book_result&ct=result&resnum=2&ved=0CDAQ6AEwAQ#v=onepage&q&f=false  | isbn = 9781931847469}}</ref>皇室为因为信仰原因判处巴哈伊信徒死刑而道歉。<ref>{{Cite news| title = Divinity School Members Protest Verdict on Baha'i| newspaper = The Harvard Crimson | publisher = | date = January 18, 1963 | url =http://www.thecrimson.com/article/1963/1/18/divinity-school-members-protest-verdict-on/| accessdate = 2010-06-07}}</ref>
*4月21日，第一届[[巴哈伊世界大会|巴哈伊世界大会]]在伦敦召开。第一届[[世界正义院|世界正义院]]由各国家灵体会的56名代表在海法选出，暨十年拓展计划结束，巴哈欧拉在[[里兹万花园|里兹万花园]]宣示100周年。<ref>{{cite web|url=http://bci.org/nnby/history/enoch_olinga.htm |title=Excerpts from the lives of early and contemporary believers on teaching the Bahá'í Faith:  Enoch Olinga, Hand of the Cause of God, Father of Victories |first=N. == 1968 ==
*第二次世界正义院选举

== 1973 ==
*第三次世界正义院选举

== 1978 ==
*第四次世界正义院选举

== 1979 ==
*[[伊朗|伊朗]]伊斯兰革命开始，直到2006年，迫害巴哈伊的行动持续了数代人，杀害了超过200名巴哈伊。

== 1983 ==
*1月31日，[[世界正义院|世界正义院]]入驻位于[[巴哈伊世界中心群楼|巴哈伊世界中心群楼]]的永久驻地。
*第五次世界正义院选举

== 1985 ==
*10月，世界正义院发布了《世界和平的承诺》

== 1986 ==
*12月24日，印度[[灵曦堂|巴哈伊灵曦堂]]（即”[[莲花寺_(德里)|莲花庙]]“）落成。

== 1987 ==
*第六次世界正义院选举

== 1992 ==
*4月21日，标志[[巴哈欧拉|巴哈欧拉]]去世100周年的神圣年开始。
*11月22-26日，第二届[[巴哈伊世界大会|巴哈伊世界大会]]在[[纽约|纽约]]举行。
*儒禧研习中心发展的里程碑，成为了一个正式的组织，但早在20世纪70年代，中心就已经在[[FUNDAEC|FUNDAEC]]基金会下开始运作了。

== 1993 ==
*3月21日，带评注的英文版[[亚格达斯经|亚格达斯经]]发布。
*第七次世界正义院选举

== 1998 ==
*第八次世界正义院选举

== 2000 ==
*1月19日，[[鲁伊希·哈努姆|鲁伊希·哈努姆]]去世，标志了[[巴哈欧拉|巴哈欧拉]]家族所有保持了对[[守基·阿芬第|守基·阿芬第]]和[[世界正义院|世界正义院]]忠诚的人全部离世。

== 2001 ==
*围绕着[[巴孛陵寝|巴孛陵寝]]和[[扇形区|扇形区]]周围的[[巴哈伊阶梯花园|阶梯花园]]完工。
*全世界总共有182个国家灵体会代表各个国家。（见[[巴哈伊数据#全世界数据|全世界巴哈伊数据]]）

== 2003 ==
