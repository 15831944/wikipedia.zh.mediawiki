'''巴东勿刹站'''（{{lang-ms|Stesen Kereta Api Padang Besar}}）是[[马来西亚|马来西亚]][[玻璃市州|玻璃市州]][[巴东勿刹|巴东勿刹]]区的火车站，属于[[KTM电动列车服务|KTM电动列车服务]]（ETS）和[[KTM通勤铁路北马区|KTM通勤铁路北马区]]的其中一站，也是北马区巴东勿刹线和[[西海岸线_(马来西亚)|西海岸线]]的北端终点站。巴东勿刹火车站位于[[马来西亚|马来西亚]]与[[泰国|泰国]]边界地带，是[[马来西亚|马来西亚]]主要通往[[泰国|泰国]]铁道交接处，站内设有大马和泰国的边检、海关部门，旅客在此出入境实施[[一地两检|一地两检]]。[[马来亚铁道公司|马来亚铁道公司]]是该车站的主要经营者，于1915年启用。
