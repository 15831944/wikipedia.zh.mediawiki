'''巴西總統'''是[[巴西|巴西聯邦共和國]]的[[國家元首|國家元首]]和[[政府首脑|政府首脑]]，掌握着最高[[行政權|行政權]]。

== 歷史 ==
總統制是自1889年11月15日[[德奧多羅·達·豐塞卡|德奧多羅·達·豐塞卡]]推翻帝制、建立巴西聯邦共和國時開始實行。自該時開始，巴西歷經六個憲政時期、兩個軍事獨裁統治時期、三個民主時期，在民主時期期間均實行[[強制投票|強制投票]]。

== 任期 ==
自1985年巴西結束軍政府時期，恢復民主政治以來，巴西的總統任期為四年，可以連任一次。但在隔了一個任期後，可以再次參選總統。

1992年，[[费尔南多·科洛尔·德梅洛|费尔南多·科洛尔]]總統遭[[巴西國會|巴西國會]][[彈劾|彈劾]]被罷免，是民主化後第一位因彈劾下台的總統。

2016年，[[迪爾瑪·羅賽芙|迪爾瑪·羅賽芙]]總統遭[[巴西國會|巴西國會]][[彈劾|彈劾]]被罷免，是民主化後第二位因彈劾下台的總統。

== 前任總統 ==
截至2019年，共有6位前總統仍然在世：

* [[费尔南多·科洛尔·德梅洛|费尔南多·科洛尔]]
* [[费尔南多·恩里克·卡多佐|费尔南多·恩里克·卡多佐]]
* [[路易斯·伊納西奧·盧拉·達席爾瓦|路易斯·伊納西奧·盧拉·達席爾瓦]]
* [[迪尔玛·罗塞夫|迪尔玛·罗塞夫]]
* [[米歇爾·特梅爾|米歇爾·特梅爾]]

而歷史上最長壽的前總統是[[文塞斯劳·布拉斯·佩雷拉·戈麦斯|文塞斯劳·布拉斯·佩雷拉·戈麦斯]]（1868年2月26日–1966年5月15日），享壽98歲。
