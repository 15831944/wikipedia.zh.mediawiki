'''巴蜀'''，又称为'''川渝'''<ref>四川一词可能包含有多种意义：1. 历史及文化意义上，泛指四川盆地一带。2. 四川省，目前中华人民共和国的一个省级行政区。3. 历史上曾经存在的名为“四川”的行政区，包括宋代的四川制置使、元代的四川行省、明代的四川承宣布政使司、清代和民国时期的四川省等。（{{cite book |author=袁庭栋 |title=《巴蜀文化志》 |year=2009 |publisher=四川出版集团·巴蜀书社 |pages =第2页 |isbn=978-7-80752-357-4 }}）</ref><ref>本文主要介绍第一种意义下的四川，近似等同于今四川省与重庆市之和，但并不包括四川省西部的非汉族聚居区（川边地区）；事实上其与第三种意义下的四川在概念上较为接近，因为自宋代设立四川制置使后，其后名为“四川”的行政区都基本涵盖整个四川盆地，不同之处主要集中在川边地区和汉中盆地的归属上；而第二种意义下的四川由于1997年的行政区划调整，不再包括四川盆地东部的今重庆市。</ref>，是[[中国|中国]]西南部[[重庆市|重庆市]]与[[四川省|四川省]]的合称，历史上其范围还包括与四川盆地文化联系密切的[[汉中盆地|汉中盆地]]、湖北[[宜昌|宜昌]]等地<ref>在元以前，汉中盆地多与四川盆地（或四川盆地的一部分）属于同一行政区，且自汉代就有“巴蜀与汉中同俗”（出自《汉书·地理志》）一说</ref>，屬於[[汉地|汉地]]的一部分，也屬梁州地區<ref>揚雄《蜀都賦》「蜀都之地，古曰梁州。禹治其江，渟臯彌望，郁乎青蔥，沃壄千里。」</ref>，巴蜀地区中的[[古蜀|蜀国]]地区由于历史悠久、物产丰富、风光秀丽，自古被誉为“[[天府之国|天府之国]]”，而深处丘陵地区的原[[巴国|巴国]]则人烟稀少。“巴蜀”得名于[[先秦时期|先秦时期]]存在于重庆四川的两个国家：[[巴国|巴国]]与[[古蜀|蜀国]]。“四川”则得名于[[北宋|北宋]][[宋真宗|真宗]][[咸平|咸平]]年间在四川地区设立的简称为“四川路”的“川峡四路”：益州路、梓州路、利州路与夔州路。川渝地区（巴蜀地区）由于地处[[丘陵|丘陵]]、[[盆地|盆地]]的相对封闭的自然条件，在语言、饮食、风俗、艺术等各方面自成一体却也各不相同，拥有有别于中国其它地区的独特的地方文化，历史上也曾经建立过包括[[成家|成家]]、[[蜀汉|蜀汉]]、[[成汉|成汉]]、[[西蜀|西蜀]]、[[前蜀|前蜀]]、[[后蜀|后蜀]]、[[李蜀|李蜀]]、[[大夏|大夏]]和[[大西|大西]]在内的众多独立国家。重庆四川地区同时也在中国历史上几度扮演极其重要的角色，历时52年的重庆[[合川|合川]][[钓鱼城|钓鱼城抗战]]延迟了[[南宋|南宋]]的灭亡，[[四川保路运动|四川保路运动]]给[[辛亥革命|辛亥革命]]的爆发和成功创造条件<ref>刘永峰：《保路：四川人的家与国》，先锋国家历史，2009年5月</ref>。而[[中国抗日战争|中国抗日战争]]时期，国民政府由[[南京|南京]]迁往重庆，重庆作为中国的大后方不仅承担国民政府三分之一以上的支出，而由大量重庆四川地区的[[袍哥会|袍哥]]组成的[[川军|川军]]抗战伤亡人数也占到中国全国官兵伤亡人数的五分之二<ref>何晓鹏：《四川何以救中国》，先锋国家历史，2009年5月</ref>。

==文化==

{{main|巴蜀文化}}
[[File:Ciqikou18.JPG|缩略图]]
[[秦朝|秦]]灭巴蜀后，原本拥有独立文化体系的四川地区逐渐被纳入[[中原文化|中原文化体系]]，但由于地处[[四川盆地|盆地]]的相对封闭的自然条件，在四川地区逐渐发展出了有别于中原地区的独特的地方文化。虽然在巴蜀时期，巴文化与蜀文化之间交流就十分密切，并且相对于中原文化具有一定的共同特征<ref>袁庭栋：《巴蜀文化志》，四川出版集团·巴蜀书社，2009年5月，ISBN 978-7-80752-357-4</ref><ref>萧易：《古蜀国：故乡还是他乡？》，先锋国家历史，2009年5月</ref>，但巴蜀地区形成统一的文化，成为一个相对独立的文化区域的时间为[[汉朝|汉朝]]<ref>谭继和：《巴蜀文化辨思集》，四川出版集团·四川人民出版社，2004年6月</ref>。在[[宋朝|宋]]末[[元朝|元]]初、[[元朝|元]]末[[明朝|明]]初和[[明朝|明]]末[[清朝|清]]初，四川地区三次经历了长期战乱，经济凋敝，人口锐减，使自[[先秦时期|巴蜀时期]]以来一直繁荣的巴蜀文化遭受重创。之后的清初“[[湖广填四川|湖广填四川]]”的大移民运动奠定了现今[[巴蜀文化|巴蜀文化]]的基础，并使其独特性保留至今，同时形成了以[[四川话|四川话]]、[[川剧|川剧]]、[[川菜|川菜]]、[[川酒|川酒]]、[[川茶|川茶]]、[[川盐|川盐]]、[[川药|川药]]、[[川江|川江]]、[[川军|川军]]、[[蜀绣|蜀绣]]、[[蜀锦|蜀锦]]、[[蜀学|蜀学]]等为代表的具有强烈地方特色的文化符号。

===语言===
[[File:Sichuanese.png|thumb]]
{{main|重庆话}}
{{main|四川话}}

四川话是流行于四川地区的主要语言，属[[汉语|汉语]][[西南官话|西南官话]]，与[[北方官话|北方官话]]和[[湘语|湘语]]、[[赣语|赣语]]、[[客家话|客家话]]等南方汉语都有一定的相似之处，是一种带有过渡性质的[[南方官话|南方官话]]，大致包括西南官话成渝片（成渝话）及赤灌片（赤灌话）。四川话目前约有1亿2千万的使用者，如果把它当成一种独立语言，其使用者人数居于世界所有语言第10位，次于[[日语|日语]]，多于[[法语|法语]]。现今四川话形成于明末清初“[[湖广填四川|湖广填四川]]”的大移民运动时期，是由来自[[湖广|湖广]]、[[广东|广东]]、[[江西|江西]]、[[陕西|陕西]]等地的各地移民方言逐渐演变融合而形成的。[[成都话|成都话]]是[[川剧|川剧]]和各类曲艺的标准音<ref>{{cite book |author=张绍诚 |title=《巴蜀方言浅说》 |year=2005 |publisher=四川出版集团·巴蜀书社 |chapter =第一章第二节：成都话是四川通用语 |isbn=7-80659-707-7}}</ref>，但四川话本身并没有标准音。

除四川话外，四川地区较为常见的语言还有[[客家话|土广东话]]（属客家话）和[[湘语|老湖广话]]（属湘语）等，其都以方言岛的形式分布于四川各地。其中土广东话使用者主要集中于[[成都平原|成都平原]]周边丘陵、川中丘陵和[[川北|川北]]山区等地，人口约100万；而老湖广话使用者主要集中于[[沱江|沱江]]中上游丘陵等地，人口约90万<ref name="《移民与四川人“打乡谈”》">{{Cite journal|author=严奇岩 |title=《移民与四川人“打乡谈”》|journal=成都大学学报（社科版）|date=2007年第4期}}</ref>。由于四川话在四川地区的强势地位，一般而言，土广东话与老湖广话的使用者同时也是四川话的使用者<ref name="《移民与四川人“打乡谈”》"/>。

==艺术==

四川地区传统艺术起源于[[先秦时期|巴蜀时期]]，历史悠久，种类丰富多样，并且具有强烈的地方特色。四川传统艺术主要包括以[[川剧|川剧]]、[[川昆|川昆]]、[[灯戏|灯戏]]、[[四川清音|四川清音]]、[[四川扬琴|四川扬琴]]、[[四川竹琴|四川竹琴]]、[[金钱板|金钱板]]等为代表的传统戏剧曲艺；以[[川江号子|川江号子]]等[[四川民歌|四川民歌]]和[[蜀派古琴|蜀派古琴]]、[[洞经音乐|洞经音乐]]等为代表的传统音乐；以[[蜀锦|蜀锦]]、[[蜀绣|蜀绣]]、[[银花丝工艺|银花丝工艺]]、[[瓷胎竹编|瓷胎竹编]]、[[成都漆艺|成都]]及[[重庆漆艺|重庆漆艺]]、[[绵竹年画|绵竹]]及[[梁平年画|梁平年画]]、[[川派盆景|川派盆景]]等为代表的传统工艺美术；以[[四川民居|四川民居]]、[[四川园林|四川园林]]为代表的传统建筑艺术和本土文学等。

===川剧===
[[File:Sichuan_Opera.jpg|thumb]]
{{main|川剧}}
川剧是流行于四川地区和邻近的[[云南|云南]]、[[贵州|贵州]]部分区域的主要戏剧剧种，因变脸、藏刀、吐火等绝活而闻名于世。清初的“[[湖广填四川|湖广填四川]]”大移民运动中，各省移民大量入川，其带来的各省戏剧艺术与四川本土原有的形成于明朝的“川戏”等艺术形式相结合，逐步演变发展，在18世纪中期的[[清代|清]][[乾隆|乾隆]]年间形成了现今的川剧艺术，迄今已有近300年历史<ref>川剧起源有多种说法，除此说之外，还有起源于唐宋说和起源于明朝说等。（{{cite web |url=http://scopera.newssc.org/system/2009/04/29/011854142.shtml |title=《地方剧种：川剧——历史沿革》 |accessdate=2009-07-07 |work=四川省川剧艺术研究院 |publisher=中国川剧网}}）</ref>。川剧唱腔由独自起源的高腔、昆腔、弹戏、胡琴和灯戏五大声腔“五腔共和”而成，这也是川剧唱腔的最显著特点。川剧分小生、旦角、生角、花脸和丑角五个行当。

川剧拥有被称为“四条河道”的四大流派：川西河，又称为成都河、上河，以[[成都|成都]]为中心流行于[[川西|川西]][[成都平原|坝子]]一带，以胡琴为特色，形成独特的“坝调”；[[资阳河派|资阳河]]，又称为川南河、中河，流行于[[川南|川南]]，，以高腔为主，艺术风格最为谨严；川北河，以弹戏为主，流行于[[川北|川北]]；下川东河，又称为重庆河、下河，以重庆为中心流传于[[川东|川东]]，特点是戏路杂，声腔多样化<ref>{{cite web |url=http://scopera.newssc.org/system/2009/04/29/011854167.shtml |title=《地方剧种：川剧——艺术特点》 |accessdate=2009-07-07 |work=四川省川剧艺术研究院 |publisher=中国川剧网}}</ref><ref>{{cite book |author=袁庭栋 |title=《巴蜀文化志》 |year=2009 |publisher=四川出版集团·巴蜀书社 |chapter =第六章第七节：源远而又多源的川剧 |isbn=978-7-80752-357-4 }}</ref>。20世纪中期之后，各流派差别在逐渐缩小<ref>《川戏》，先锋国家历史，2009年5月</ref>。除川剧外，四川地区还流行有由[[昆曲|昆曲]]演变而来的[[川昆|川昆]]（川剧声腔“昆腔”的原型）、起源于四川传统民俗活动的[[灯戏|灯戏]]（川剧声腔“灯戏”的原型）<ref>{{cite web |url=http://www.ihchina.cn/inc/guojiaminglunry.jsp?gjml_id=221 |title=《国家名录：灯戏》 |accessdate=2009-07-07 |publisher=中国非物质文化遗产网 |deadurl=yes |archiveurl=https://web.archive.org/web/20060712173437/http://www.ihchina.cn/inc/guojiaminglunry.jsp?gjml_id=221 |archivedate=2006-07-12 }}</ref>、[[木偶戏|木偶戏]]、[[皮影戏|皮影戏]]等剧种。

===曲艺===

四川地区传统曲艺最迟起源于[[汉代|汉代]]，目前在巴蜀地区出土的汉代说唱俑多达上百件，是中国出土汉代说唱俑最多的地区。[[唐朝|唐]][[宋朝|宋]]时期，巴蜀地区曲艺进一步发展，并且广为流行。目前四川地区的传统曲艺主要成型于明末清初的“湖广填四川”大移民运动时期，此时湖广、广东、江西、陕西等地的各种说唱形式随移民流入巴蜀，逐渐与巴蜀传统曲艺融合，发展成为新的成熟的曲种，并在[[民国|民国]]初年发展达到高潮<ref>幸晓峰：《移民入川与四川曲艺的成熟和发展》，四川戏剧，2008年12月</ref>。四川地区的传统曲艺种类包括[[四川清音|四川清音]]、[[四川扬琴|四川扬琴]]、[[四川竹琴|四川竹琴]]、[[金钱板|金钱板]]、[[四川评书|四川评书]]、[[四川相书|四川相书]]、[[四川谐剧|四川谐剧]]、[[四川荷叶|四川荷叶]]、[[四川盘子|四川盘子]]、[[四川花鼓|四川花鼓]]、[[四川莲萧|四川莲萧]]等。

===音乐===
川渝地区传统音乐起源于巴蜀时期，[[广汉|广汉]][[三星堆遗址|三星堆遗址]]、[[成都|成都]][[金沙遗址|金沙遗址]]、[[成都|成都]][[羊子山遗址|羊子山遗址]]、[[涪陵|涪陵]][[小田溪遗址|小田溪遗址]]等巴蜀时期遗址中均有大量乐器出土，且形制与此时中原地区同类乐器有所差异，表现出了此时巴蜀地区音乐文化的独特性<ref>华明玲、田彬华：《三星堆出土乐器研究》，西南科技大学学报（哲学社会科学版），2009年4月</ref>，[[巴渝舞|巴渝舞]]也起源于此时，并成为汉代重要的宫廷乐舞<ref>李春刚：《巴国·巴人·巴文化》，巴蜀文化研究通讯，2005年第4期</ref>。唐宋时期，巴蜀地区音乐发展到鼎盛时期，[[杜甫|杜甫]]居于成都时曾以“喧然名都会”、“锦城丝管日纷纷”来形容此盛况。唐时发展形成了蜀国弦<ref>蜀国弦与巴渝舞应当有一定的传承关系，但目前相关研究较少，并不能定论（谭继和：《巴蜀文化研究综议》，中华文研究通讯，总第20期）</ref>，并在[[前蜀|前]][[后蜀|后蜀]]时期进而发展成为[[蜀宫伎乐|蜀宫伎乐]]。

现今巴蜀地区传统音乐的音调结构特色鲜明，大致可分为蜀羽体系与蜀-{徵}-体系两类<ref>王耀华，杜亚雄：《中国传统音乐概论》，福建教育出版社，2004年11月15日，ISBN 978-7-5334-2807-5</ref>。流传至今的巴蜀地区传统音乐主要包括[[洞经音乐|洞经音乐]]、[[蜀派古琴|蜀派古琴]]和[[四川民歌|四川民歌]]。洞经音乐起源于古蜀人的祭祀音乐，自唐代便开始在巴蜀地区流行，并一直传承至今，堪称古代音乐的“活化石”<ref>沙马拉毅、侯光：《中国洞经音乐的源和流》，西南民族大学学报，2008年第9期</ref>；蜀派古琴起源于汉代，并在清代最终成型，成为古琴的一个重要流派，现世流传的古琴名曲《[[流水|流水]]》便是蜀派古琴的代表作<ref>沙曾成伟：《蜀派古琴源流及艺术特色》，音乐探索，1997年2月</ref>；四川民歌包括以[[川江号子|川江号子]]为代表的劳动号子和山歌、小调等，代表曲目包括《[[太阳出来喜洋洋|太阳出来喜洋洋]]》、《[[黄杨扁担|黄杨扁担]]》、《[[康定情歌_(歌曲)|康定情歌]]》等。

川西地区（成都市、德阳市、绵阳市、眉山市），川北地区（南充市、广元市、遂宁市、达州市、巴中市），川南地区（宜宾市、自贡市、内江市、泸州市、乐山市），川东地区（重庆市主城区及其周边各区县、原万县市地区各区县、原涪陵市地区各区县、广安市），川中地区（资阳市），另有时川东地区的原万县市地区各区县专作下川东地区。
