'''巴斯'''（{{bd|1980年||2017年|9月13日}}）生于[[四川省|四川省]][[宝兴县|宝兴县]]，[[大熊猫|大熊猫]]，[[1990年亚运会|1990年亚运会]]吉祥物“[[盼盼|盼盼]]”的原型，一生创造了大熊猫史上多项第一<ref name=panpan>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1794086|title = 北京亚运会吉祥物“盼盼”原型大熊猫巴斯去世，享年37岁 |author =  |date = 2017-09-14 |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref><ref name=lishi>{{Cite web |url = http://news.xinhuanet.com/2017-09/14/c_1121662551.htm |title = 明星大熊猫“巴斯”离世 |author =  |date = 2017-09-14 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
巴斯的家乡在四川省宝兴县，是野生雌性大熊猫。1984年2月22日，4岁的巴斯在觅食时落入巴斯沟大河边（大河边是个地名）巴斯河的冰冷河水中，被宝兴县[[永富乡_(宝兴县)|永富乡]]农村的女村民李兴玉和10岁的侄儿石家明发现并救起。在送交给位于宝兴的蜂桶寨自然保护区时，保护区工作人员要李兴玉给这只大熊猫取名字，李兴玉想到是在巴斯沟附近发现的，便取名为“巴斯”<ref name=panpan/><ref name=lishi/><ref>{{Cite web |url =http://yaan.scol.com.cn/sh/content/2015-06/18/content_51735488.htm?node=110768  |title =李兴玉与大熊猫“巴斯”的不解之缘之一  |author =  |date = 2015-06-18 |publisher = 四川在线 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://2010.163.com/10/1102/06/6KFBLH9F00864IPP.html |title =救了巴斯，这是我和它的缘分  |author =  |date =2010-11-02  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。在蜂桶寨自然保护区居住数月后，巴斯被送到[[四川卧龙国家级自然保护区|卧龙]]的[[中国保护大熊猫研究中心|中国保护大熊猫研究中心]]。后经[[中华人民共和国林业部|中华人民共和国林业部]]调配，1984年5月6日，巴斯来到[[福建省|福建省]][[福州市|福州市]]的[[福州动物园|福州动物园]]<ref name=rensheng/><ref name=jiemi>{{Cite web |url = http://news.163.com/17/0914/19/CUAL3V58000187VE.html |title = 揭秘熊猫明星巴斯的最后100天，饲养员回忆录催人泪下 |author =  |date = 2017-09-14 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。1987年7月至1988年2月，巴斯应邀代表[[中国野生动物保护协会|中国野生动物保护协会]]赴[[美国|美国]][[圣地亚哥动物园|圣地亚哥动物园]]访问半年，观众须排队5小时才能观看巴斯3分钟，共有250万至300万观众前来参观。巴斯前往[[洛杉矶国际机场|洛杉矶国际机场]]离开美国当天，沿途200余公里都是送别她的人<ref name=panpan/><ref name=lishi/><ref name=rensheng>{{Cite web |url = http://news.china.com.cn/rollnews/education/live/2015-05/29/content_32906951_2.htm |title =  “盼盼”原型——大熊猫“巴斯”的传奇“人生”|author =  |date = 2015-05-29 |publisher =中国网  |language =  |accessdate =  |quote =  }}</ref>。

[[File:福州熊猫世界-熊猫巴斯03.jpg|thumb]]
1990年，应[[1990年亚洲运动会|北京亚运会]]组委会邀请，巴斯成为本届亚运会吉祥物“[[盼盼|盼盼]]”的原型。[[1991年中国中央电视台春节联欢晚会|1991年中国中央电视台春节联欢晚会]]上，福州动物园表演《盼盼与巴斯》，巴斯由此亮相中国中央电视台春节联欢晚会，表演了投篮、举重、打电话等，成为家喻户晓的大熊猫明星<ref name=panpan/><ref name=lishi/>。

1998年，巴斯为异种[[克隆|克隆]]大熊猫捐献[[体细胞|体细胞]]，直到早期胚胎形成。该成果被[[中国科学院|中国科学院]]、[[中国工程院|中国工程院]]评为1999年中国十大科技进展<ref name=panpan/>。

2001年，巴斯成为世界上第一只被确诊患有[[高血压|高血压]]的大熊猫<ref name=zqw>{{Cite web |url = http://news.youth.cn/gn/201709/t20170914_10715112.htm |title =  传奇大熊猫巴斯去世 获熊猫界数个第一|author =  |date = 2017-09-14 |publisher = 中国青年网 |language =  |accessdate =  |quote =  }}</ref>。2002年，巴斯患高血压昏迷一周，经抢救苏醒，脱离生命危险，最后挣扎着走出房间<ref name=panpan/>。2002年，巴斯还接受了[[白内障|白内障]]摘除手术，成为世界上第一只接受白内障摘除手术的大熊猫<ref name=zqw/><ref name=panpan/>。2010年11月12日[[2010年亚运会|广州亚运会]]开幕式直播中，巴斯在福州分会场亮相5分钟，戴着黑框眼镜看电视<ref name=youpiao/>。2010年，巴斯突发[[胰腺炎|胰腺炎]]，经过8个昼夜的抢救，最终脱离生命危险。2015年，以巴斯的经历为题材制作的3D动画短片《巴斯向世界人民问好》在美国[[纽约时代广场|纽约时代广场]]大屏幕播放。2017年1月18日，巴斯获颁“世界上现存最长寿圈养大熊猫”[[吉尼斯世界纪录大全|世界吉尼斯纪录]]证书。2017年6月初起，巴斯因[[肝硬化|肝硬化]]、[[肾衰竭|肾衰竭]]、年老体弱，出现多种病症，[[海峡（福州）大熊猫研究交流中心|海峡（福州）大熊猫研究交流中心]]、[[中国人民解放军福州总医院|福州总医院]]的专家多次会诊<ref name=panpan/><ref name=youpiao>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1796404 |title = 福州市民携邮票、旗帜悼念熊猫巴斯，它的器官将用于科学研究 |author =  |date = 2017-09-16 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。为给无法进食的巴斯补充营养，工作人员通过手术在巴斯的颈静脉安置了一个输液管，巴斯也成为世界上第一只接受该手术的大熊猫<ref name=zqw/>。

[[File:福州熊猫世界-熊猫巴斯01.jpg|thumb]]
2017年9月13日，巴斯在海峡（福州）大熊猫研究交流中心病逝，享年37岁<ref name=panpan/>。9月16日，海峡（福州）大熊猫研究交流中心在福州熊猫世界为巴斯举办追思活动，在10月1日[[中华人民共和国国庆节|中华人民共和国国庆节]]之前每天为她播放专题视频，并设立悼念专版<ref name=jiemi/>。大批福州市民来到海峡熊猫世界悼念巴斯，甚至翻出当年“盼盼”的明信片、邮票、助威旗帜等纪念品摆放在兽舍隔离窗外。巴斯的遗体将作为标本安放在巴斯博物馆，部分器官将用于科学研究<ref name=youpiao/>。

==纪念==
巴斯的出生日期不详。曾在1月18日、11月28日等日期庆祝过巴斯的生日<ref>{{Cite web |url =http://news.xinhuanet.com/mrdx/2015-11/30/c_134867464.htm  |title =明星熊猫“巴斯”迎35岁超高龄生日 |author =  |date = 2015-11-30 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.sohu.com/a/124704155_162758 |title =大熊猫“巴斯”的37岁生日  |author =  |date = 2017-01-19  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。

2015年，在巴斯35岁生日会上，海峡（福州）大熊猫研究交流中心提出要将“生物巴斯”转化为“文化巴斯”，设立了“巴斯勋章”奖项，成立了“巴斯基金会”等，并为她制作出目前世界上最大的熊猫石雕（钢筋水泥结构，高2米，重6吨）<ref name=jiemi/>。

2017年1月18日，福州市人民政府为巴斯树立了“世界的巴斯、和平的图腾”代表中国印的石碑<ref name=jiemi/>。
