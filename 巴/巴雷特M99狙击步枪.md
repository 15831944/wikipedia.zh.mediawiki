'''巴雷特M99'''（{{lang|en|Barrett Model 99}}，又名'''{{lang|en|Big Shot}}'''）是[[巴雷特|巴雷特]]在1999年推出的[[狙击步枪|狙击步枪]]，發射[[12.7×99mm_NATO|12.7×99毫米]][[北大西洋公约组织|北約]][[口徑|口徑]]（.50 BMG）及{{Link-en|.416 Barrett|.416 Barrett|.416巴雷特}}[[子彈|子彈]]。

== 設計及歷史 ==
巴雷特在推出大口徑的[[巴雷特M82狙擊步槍|M82]]及[[巴雷特M95狙擊步槍|巴雷特M95]]後，為了再提高精確度及降低長度，以M95為基礎設計出一種[[犢牛式|犢牛式]]結構、[[手動槍機|旋轉後拉式槍機]]、內置彈倉只可放一發子彈的[[狙擊步槍|狙擊步槍]]。M99的槍口仍然裝有高效能的雙室[[砲口制動器|槍口制退器]]，但槍管沒有刻上線坑，亦沒有預設瞄準鏡，只有在[[機匣|機匣]]頂部設有的[[戰術導軌|戰術導軌]]上安裝瞄準鏡，[[兩腳架|兩腳架]]裝在機匣前端的底部。巴雷特M99有[[12.7×99mm_NATO|12.7×99毫米NATO]]（.50 BMG）及{{Link-en|.416 Barrett|.416 Barrett}}兩種口徑，亦提供四種不同槍管選擇，包括[[12.7×99mm_NATO|12.7×99毫米NATO]]（.50 BMG）的25、29、32寸槍管版本，及{{Link-en|.416 Barrett|.416 Barrett}}的32寸槍管版本<ref>[http://www.barrettrifles.com/rifle_99.aspx 巴雷特官方頁面-M99資料]</ref><ref>{{cite web |url=http://world.guns.ru/sniper/sn54-e.htm |title=存档副本 |accessdate=2007年7月28日 |deadurl=yes |archiveurl=https://web.archive.org/web/20070809044731/http://world.guns.ru/sniper/sn54-e.htm |archivedate=2007年8月9日 |df= }}</ref>。

巴雷特為了令M99達到極高精度，因此結構比較簡單，[[彈倉|彈倉]]只可放一發子彈而且不設[[彈匣|彈匣]]，在軍事用途缺乏競爭力，所以現時主要是向民用市場及[[警察|執法部門]]發售，在美國一些禁止民間擁有.50 BMG的州份（例如[[加利福尼亚州|加州]]）只會發售.416 Barrett口徑版本。
