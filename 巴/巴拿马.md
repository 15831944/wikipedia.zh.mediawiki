'''巴拿马共和国'''（{{lang-es|'''República de Panamá'''}} {{IPA-es|reˈpuβlika ðe panaˈma|}}），簡稱'''巴拿馬'''（{{lang-es|link=no|'''Panamá''' }} {{IPA-es|panaˈma|}}; {{IPAc-en|audio=En-us-Panama-2.ogg|ˈ|p|æ|n|ə|m|ɑː}} {{Respell|PAN|ə-mah }}），是[[中美洲|中美洲]]最南部的国家，面积7.55万平方公里，人口393万。

連接[[大西洋|大西洋]]及[[太平洋|太平洋]]的[[巴拿马运河|巴拿马运河]]位於國家的中央，劃分了南北[[美洲|美洲]]，拥有重要的战略与经济地位。

== 地理 ==
{{main|巴拿马地理}}
巴拿马位于[[中美洲|中美洲]]，它的东岸是[[加勒比海|加勒比海]]，西岸是[[太平洋|太平洋]]，北临[[哥斯达黎加|哥斯达黎加]]，南临[[哥伦比亚|哥伦比亚]]。它是连接[[北美洲|北美洲]]和[[南美洲|南美洲]]的地峡。其境內的野生動物多樣性是所有中美洲國家中最高的，並且也是很多北美洲及南美洲野生動物品種的發源地。境內擁有巴拿馬運河。

== 历史 ==
{{main|巴拿马历史|美国入侵巴拿马}}
1501年，巴拿马沦为西班牙殖民地，属新格兰纳达总督区。

1821年巴拿马独立，加入[[西蒙·玻利瓦尔|西蒙·玻利瓦尔]]建立的[[大哥伦比亚共和国|大哥伦比亚共和国]]。

1830年，[[大哥伦比亚共和国|大哥伦比亚共和国]]分裂，巴拿马成为[[哥伦比亚|哥伦比亚]]共和国的一个省。

1880年，法国人[[斐迪南·德·雷赛布|斐迪南·德·雷赛布]]开始動工開鑿连接[[大西洋|大西洋]]和[[太平洋|太平洋]]的[[巴拿馬運河|巴拿馬運河]]。美国总统[[西奥多·罗斯福|西奥多·罗斯福]]试图接管这条运河。

1903年美國在击败英、法后与[[哥伦比亚|哥伦比亚]]政府签定由美修建和租借运河的条约，哥议会拒绝批准。1903年11月3日，美军在巴拿马登陆，策動巴脫離哥倫比亞獨立，成立巴拿馬共和國。同年11月18日，美巴簽訂《[[美巴条约|巴拿馬運河條約]]》，美國取得「永久使用、佔領和控制[[巴拿馬運河|巴拿馬運河]]」的權利，隨後更把持了[[巴拿馬運河區|運河區]]的行政、司法、警備、鐵路和財政等，使運河區成為名副其實的「[[國中之國|國中之國]]」。美国从1904年起恢复开凿运河，于1914年完工。

1968年{{tsl|en|Omar Torrijos|奥马尔·托里霍斯}}将军上台，到他在1981年一次空难中逝世为止他是[[巴拿馬|巴拿馬]]的实际操权者。托里霍斯死后，[[曼紐·諾瑞嘉|曼紐·諾瑞嘉]]於1983年通過政變掌握大权。

1977年9月7日，[[吉米·卡特|吉米·卡特]]的美國政府和托里霍斯的巴拿馬政府簽署了關於巴拿馬運河的新條約。新條約規定，美國將在1999年底以前把運河和運河區的全部主權和管轄權逐步交還給巴拿馬共和國政府。

[[File:Panama_Skyline.jpg|thumb]]
1989年5月总统大选开票统计结果，反对党联合推举之总统候选人[[吉列爾莫·恩達拉|吉列爾莫·恩達拉]]获得选举胜利，但軍政府諾瑞嘉宣布选举无效，同年9月美国对巴国实行[[经济制裁|经济制裁]]。

1989年12月，美国一个海军陆战队士兵被杀，美国以保护美国侨民的安全為由，派兵入侵巴拿马。12月15日诺列加宣布与美进入战争状态，自任国家元首。12月20日，兩萬七千名美軍對巴拿馬展開軍事行動，推翻諾瑞嘉政權，吉列爾莫·恩達拉就任總統。[[曼紐·諾瑞嘉|諾瑞嘉]]逃入[[梵蒂冈|梵蒂冈]]大使馆，1990年1月3日向美軍投降，他立刻被运往[[佛罗里达州|佛罗里达州]]，被起诉和关押。美軍入侵造成約兩千人死亡，包括巴拿馬軍人及一般平民。

1999年12月31日，巴拿馬收回運河的全部管理和防務權，美軍全數撤出。<ref name=xinhua>{{cite web|url=http://news.xinhuanet.com/ziliao/2004-05/12/content_1465235.htm|title=巴拿马概况|publisher=[[新华网|新华网]]|accessdate=2015年9月8日 |language=zh-Hans}}</ref>

== 政治 ==
{{main|巴拿马政治}}
巴拿马的行政机关由其总统和两个副总统領導，立法机关是一个由72个代表组成的議会。总统和立法大会成员由国民选举產生，任期为5年，不能連任。在巴拿马公民有选举义务，但至今为止还没有人因为没有参加选举而受惩罚。一个独立的选举委员会监视选民登记和选举过程。司法机关由一个由九名法官组成的最高法院和地方法院组成。

执政联盟包括：民主变革党和民族主义共和自由运动。

反对党主要包括：民主革命党、巴拿马主义党、民主广泛阵线党、人民党<ref name=xinhua />。

== 经济 ==
[[File:Exclusa_Miraflores_Canal_de_Panama_Panorama.jpg|thumb]]]]
{{main|巴拿马经济}}
鉴于它重要的地理位置巴拿马的经济的重点是[[服务业|服务业]]，以金融、贸易和旅游为主。由於巴拿馬的建立和貿易都和美國有重要關係，巴拿馬從1907年開始使用[[美元|美元]]作為流通货币，本國貨幣为[[巴拿马巴波亚|巴波亚]]，但只发行1分、10分、25分、50分的铸币，与美元等值并同时在境内使用。是世界上第一個[[美國|美國]]以外使用美元作為法定貨幣的國家。

巴拿馬是全世界最主要的[[權宜船旗|權宜船旗]]註冊國，據2007年統計，有4954艘船舶登記為巴拿馬籍<ref>{{cite book |author=王穆衡、張世龍、陳一平 |title= 船舶設籍制度之探討|url= http://oa.lib.ksu.edu.tw/OA/bitstream/987654321/197382/3/96-MBA008_U.pdf|format=pdf|location= 中華民國（台灣）台北市|publisher=中華民國交通部運輸研究所 |isbn= 978-986-01-3820-7|date=2008年4月|pages=頁11}}</ref>。

;财政金融
巴拿马城国际金融中心是拉美地区最重要的金融中心。2008年巴外汇储备24.43亿美元，外债108.85亿美元<ref name=xinhua />。

;对外贸易
巴拿马商品贸易历年有巨额逆差。2013年出口额8.4亿美元，进口额130.4亿美元，同比分别增长2.7%和3.2%。主要出口产品为金枪鱼、虾、鱼粉、咖啡、香蕉等，主要出口对象为美国、瑞典、西班牙、哥斯达黎加等。主要进口石油产品、制成品、药品和食品等，进口主要来自美国、科隆自由区、日本、哥斯达黎加等地。

中华人民共和国主要向巴拿马出口轻工和纺织品、燃料油、橡胶和塑料鞋、计算机和通讯产品等，从巴进口鱼粉、废金属、水泥和皮革等<ref name=xinhua />。
