'''巴哈馬國'''（{{lang-en|Commonwealth of the Bahamas}}），通称'''巴哈马'''，是位於[[大西洋|大西洋]]西侧[[廬卡雅群島|廬卡雅群島]]上的一个[[島國|島國]]，全國包含700座[[岛屿|岛屿]]（以及小岛屿）和[[珊瑚礁|珊瑚礁]]。地處[[美国|美国]][[佛罗里达|佛罗里达]][[佛羅里達礁島群|礁島群]]東南面，[[古巴|古巴]]和[[伊斯帕尼奥拉岛|伊斯帕尼奥拉岛]]（[[海地|海地]]和[[多米尼加|多米尼加]]以北，[[特克斯和凯科斯群岛|特克斯和凯科斯群岛]]东北部，絕大多數的領土位於[[百慕達三角|百慕達三角]]附近。

巴哈馬的首都[[拿骚|拿骚]]位于[[新普罗维登斯岛|新普罗维登斯岛]]上，絕大多數的人口居住於首都拿骚一帶；而旅遊業和金融業是巴哈馬主要的經濟活動。

== 历史 ==
{{main|巴哈马历史}}
巴哈馬群島的原住民為路克揚[[印第安人|印第安人]]，1492年10月12日[[哥倫布|哥倫布]]探索該群島時瞥見他們。據說哥倫布曾登陸[[聖薩爾瓦多島|聖薩爾瓦多島]]。[[西班牙人|西班牙人]]對移民該島無興趣，但到此搜刮奴隸，使人口銳減。

1648年一批來自[[百慕達群島|百慕達群島]]的英國人來到巴哈馬群島，但沒有定居的打算，這裡於是很快又淪為海盜的避難所，而且[[西班牙人|西班牙人]]也不時登陸侵擾，導致後來雖有人但並不特別繁榮。

[[美國獨立革命|美國獨立革命]]之後，當時一些效忠英國的人士逃離美國來到巴哈馬群島，進而帶動了巴哈馬群島的繁榮，陸續建立了[[棉花|棉花]]種植園。美國[[南北戰爭|南北戰爭]]期間成為封鎖走私船的中心。一直到第二次世界大戰後，[[旅遊業|旅遊業]]才開始發展，使當地持續繁盛。

1964年巴哈馬群島獲得內部自治權。1973年成為獨立的'''巴哈马國'''。

== 政治 ==
巴哈馬國是[[君主立憲|君主立憲]]國家，採[[兩院制|兩院制]]。[[國家元首|國家元首]]是[[巴哈馬君主|巴哈馬君主]]（代表[[英國君主|英國君主]]），政府代表是[[巴哈馬總督|總督]]。政府首腦為[[巴哈马总理|巴哈马总理]]大臣。

== 人口 ==
人口約298,000（2001年），人民多為歐洲人和非洲人混血後裔，非洲人為以前奴隸。人口多集中在[[新普罗维登斯岛|新普洛維頓斯島]]。

== 气候 ==
巴哈馬群岛大部分的地區都属温和的[[亚热带气候|亚热带气候]]，[[北回归线|北回归线]]横贯中部。在一年之中8月为最热月份，平均气温介於[[攝氏|攝氏]]{{convert|24|至|32|C}}之間；1、2月为最冷月份，平均气温{{convert|17|至|25|C}}之間。巴哈馬的年均气温約為攝氏{{convert|23.5|C}}，年平均降水量則在1000[[毫米|毫米]]上下。
