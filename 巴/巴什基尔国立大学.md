'''巴什基尔国立大学'''（{{lang-ru|Башкирский государственный университет}}；{{lang-ba|Башҡорт дәүләт университеты}}）是一座位于[[俄罗斯|俄罗斯]][[巴什科尔托斯坦共和国|巴什科尔托斯坦共和国]][[首府|首府]][[乌法|乌法]]的大学，1909年正式建立。

==概况==
1909年10月4日，巴什基尔国立大学的前身——乌法师范学院（{{lang|ru|Уфимский учительский институт}}）在乌法成立<ref>{{cite web |script-title=ru:Первый вуз Башкортостана |url=http://vatandash.ru/index.php?article=441 |website=vatandash.ru |accessdate=2018-11-08 |language=ru |title=存档副本 |archive-url=https://web.archive.org/web/20171226172624/http://vatandash.ru/index.php?article=441# |archive-date=2017-12-26 |dead-url=no }}</ref>，该校也是巴什科尔托斯坦历史上第一所高等教育机构<ref name=intl/>。[[十月革命|十月革命]]后，1919年，乌法师范学院更名为人民教育学院（{{lang|ru|Институт народного образования}}）。之后又几经易名，在1957年改为巴什基尔国立大学<ref>{{cite web |title=乌法 |url=http://cn.russia.edu.ru/russia/cities/ufa/ |website=俄罗斯联邦教育与科技部 |accessdate=2018-11-08 |language=zh-hans |archive-url=https://web.archive.org/web/20170823221020/http://cn.russia.edu.ru/russia/cities/ufa/# |archive-date=2017-08-23 |dead-url=no }}</ref>。

巴什基尔国立大学是俄罗斯的“古典大学”之一<ref>{{cite web |script-title=ru:Адреса университетов-участников Ассоциации классических университетов России |url=http://www.acur.msu.ru/members_list.php |accessdate=2018-11-08 |language=ru |title=存档副本 |archive-url=https://web.archive.org/web/20180616181641/http://www.acur.msu.ru/members_list.php# |archive-date=2018-06-16 |dead-url=no }}</ref>。该校在在有机化学、应用数学、语言学、信息技术、地球物理学和石油工程方面具有一定的实力。根据俄罗斯[[国际文传电讯社|国际文传电讯社]]2018年的排名，巴什基尔国立大学在所有俄罗斯[[国立大学|国立大学]]中综合排名第61位<ref>{{cite web |script-title=ru:Национальный рейтинг университетов |url=https://academia.interfax.ru/ru/ratings/?rating=1&year=2018&page=4 |website=interfax.ru |accessdate=2018-11-08 |language=ru}}</ref>。该校在巴什科尔托斯坦境内的[[斯捷尔利塔马克|斯捷尔利塔马克]]、[[比尔斯克|比尔斯克]]、[[锡拜|锡拜]]和[[涅夫捷卡姆斯克|涅夫捷卡姆斯克]]设有分校<ref name=intl>{{cite web |title=INTERNATIONAL AND INTERREGIONAL RELATIONS DEPARTMENT |url=http://www.bashedu.ru/en/international-relations |website=Bashkir State University |accessdate=2018-11-08 |language=en |archive-url=https://web.archive.org/web/20180428073625/http://www.bashedu.ru/en/international-relations# |archive-date=2018-04-28 |dead-url=no }}</ref>。

截至2018年，该校拥有超过33000名学生，其中包括来自23个国家的350多名国际学生。教职人员人员约2000人，其中包括183名[[博士|博士]]和489名[[副博士|副博士]]。据该校官网介绍，每年有超过500名研究生和博士生在这里学习，占整个巴什科尔托斯坦研究生和博士生的三分之一。<ref name=intl/>
