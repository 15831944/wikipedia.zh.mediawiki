'''巴拉克拉瓦海軍博物館'''（{{lang-uk|Морський музейний комплекс "Балаклава"}}，{{lang-ru|Музей холодной войны}}）是一座地下潛艇基地，位於[[俄羅斯|俄羅斯]][[克里米亞|克里米亞]]的[[巴拉克拉瓦|巴拉克拉瓦]]，一開始這個博物館稱為冷戰博物館K-825，在[[冷戰|冷戰]]期間是一座最高機密的[[軍事設施|軍事設施]]，位於[[Balaclava_Bay|巴拉克拉瓦湾]]。時至今日，除了冷戰博物館之外，也包含有關[[2014年克里米亞危機|2014年克里米亞危機]]的介紹。

== 簡介與宗旨 ==
{{Unreferenced section|time=2017-11-27T15:17:07+00:00}}
這座軍事基地建立的宗旨是可以承受住10萬噸[[核武器当量|核武器当量]]的攻擊，整個地下軍事基地包括地下的水道網絡與停靠港口，修理廠、魚雷儲存倉庫與其他武器。除此之外，也可以保護內部的人員免於被輻射侵襲，出入口位於塔夫羅斯山，兩側都有出入口，如果必要時，沉箱門可以完全將只變成密閉的空間，岩石上的洞口整齊地覆蓋著偽裝裝置。

825GTS計畫目的是維護、修復[[613型潜艇|613型潜艇]]和[[633型潜艇|633型潜艇]]，其中有一條中央水道設施，長度達602公尺，可同時容納7艘潛水艇，最多可以容納14艘不同類型的潛艇，水道深度約為8公尺，寬度約為12到22公尺。整體而言所有設施達到9600平方公尺，其中水體面積達到5200平方公尺。和平時期的軍事武裝在碼頭上進行，同時密切注意可能的敵方軍事間諜衛星活動。一個特殊的隧道用於戰時的裝備設施。整個複雜的結構包括維修與基礎設施，代號為280計畫，可以儲備和維持核武器庫。地下潛艇基地平均溫度保持在[[攝氏|攝氏]]15度左右。

== 歷史 ==
{{来源请求|[[第二次世界大戰|第二次世界大戰]]之後，世界兩大強權，[[美國|美國]]與[[蘇聯|蘇聯]]在這之後開始加強核打擊的力度，目的是要維持[[一次打击|先发制人的打击]]和[[二次打击|报复性袭击]]。當時[[拉夫连季·帕夫洛维奇·贝利亚|拉夫连季·帕夫洛维奇·贝利亚]]給了[[约瑟夫·维萨里奥诺维奇·斯大林|约瑟夫·维萨里奥诺维奇·斯大林]]一項計畫，一開始被設定核計畫之一，目的是為了讓潛艇有一個隱藏的處所，經過了若干年的研究之後，確定落腳於巴拉克拉瓦，該城市立即經過編碼，並併入[[塞瓦斯托波爾市|塞瓦斯托波爾市]]的一個新區。巴拉克拉瓦坐落在狹窄的灣口，寬度僅為200~400公尺。窄小的彎口可以保護城市免於風暴，也是窺探黑海的眼睛，保护该城市不仅从风暴，但也窥视的眼睛，因为它是不可见的，在任何角度离开大海。 此外，鄰近的[[塞瓦斯托波爾|塞瓦斯托波尔]]是現在[[俄羅斯海軍|俄羅斯海軍]][[黑海艦隊|黑海艦隊]]主要基地。}}

1957年特別建設一個名為第528號的特殊部門，主要是整建地下設施的，建造過程持續了從1953~1961年總共8年，施工期間總計有120萬噸岩石從塔夫羅斯山上移除 。為了確保物資可以安全在夜間運送往[[黑海|黑海]]，1993年之後地下潛艇基地就關閉，且大多數都是無人看守的，到了2000年幾乎棄置的設施才轉移給[[烏克蘭海軍|烏克蘭海軍]]。1993~2003年之間因為無人看守而被[[掠夺|掠夺]]，幾乎所有的金屬物品都被竊盜一空<ref>{{Cite web|url=http://submarine.id.ru/cp/z97.shtml|title=Underground harbor of submarines (Russian)|date=|last=Cherkashin|first=N.|publisher=|website=|access-date=|deadurl=yes|archiveurl=https://web.archive.org/web/20070516003051/http://www.submarine.id.ru/cp/z97.shtml|archivedate=2007-05-16}}</ref>

{{来源请求|由弗拉基米爾·斯特凡諾夫斯基領導的'''塞瓦斯托波爾海洋委員會'''提議建造巴拉克拉瓦海軍博物館，該博物館定位在冷戰相關的展覽館，展出修理工廠、核武工廠以及一座地下碼頭、旅遊中心、電影播放室，展現冷戰兩大軍事強權對抗的過程，參展的最後，地下紀念碑緬懷那些永垂不朽的戰士，雖然沒有被子彈所擊中，不過與潛水艇一起長眠在冰冷海洋的深處。}}

2013年，慶祝巴拉克拉瓦海軍博物館10週年<ref>{{Cite web|url=http://meridian.in.ua/news/10624.html|title=Подземный музей отметил юбилей (видео)|website=meridian.in.ua|access-date=2016-11-25}}</ref>。潛艇退伍軍人、過去僱員、[[烏克蘭軍隊|烏克蘭軍隊]]與學生皆出席了慶祝儀式。不過吞併克里米亞之後，巴拉克拉瓦海軍博物館與南部海洋在2014年回歸俄羅斯聯邦管轄<ref>{{Cite web|url=http://muzey-sevastopol.com/|title=俄羅斯聯邦回歸|access-date=2017-11-22}}</ref>

== 重新啟用 ==
根據新聞2014年3月報導，俄羅斯當局正在考慮是否要恢復巴拉克拉瓦潛艇基地<ref>{{Cite news|url=http://www.gazeta.ru/science/2014/03/23_a_5956453.shtml|title=Подводные лодки в степях Балаклавы|newspaper=Газета.Ru|access-date=2016-11-25}}</ref>。

== 類似的基地 ==
* [[瑞典|瑞典]]的{{Link-en|穆斯克海軍基地|Muskö naval base}}亦為地下海軍基地，用於停放[[导弹快艇|导弹快艇]]與[[驱逐舰|驱逐舰]]。
* [[甫洛夫斯克灣|甫洛夫斯克灣]]的輻射庇護所和維修工廠
* 維達亞依佛輻射避難所和潛艇維修工廠
* [[亚得里亚海|亚得里亚海]]地下潜艇基地隸屬於{{tsl|en|Albanian Naval Force|阿尔巴尼亚海军}}，是一个秘密的军事设施，也是在冷戰時期建立，過去用於儲藏蘇聯潛艇，不過到了2009年為[[北大西洋公约组织|北大西洋公约组织]]所控制。
