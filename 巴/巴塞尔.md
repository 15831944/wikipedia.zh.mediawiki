'''巴塞爾'''（又稱為'''巴泽爾'''，{{lang-de|'''Basel'''}}，{{lang-fr|'''Bâle'''}}，{{lang-it|'''Basilea'''}}）是[[瑞士|瑞士]]的第三大[[城市|城市]]（僅次於[[蘇黎世|蘇黎世]]和[[日內瓦|日內瓦]]），為[[巴塞爾城市州|巴塞爾城市州]]（{{lang-de|Basel-Stadt}}，{{lang-fr|Bâle-Ville}}）的首府，坐落於[[瑞士|瑞士]]西北的[[三國交角|三國交角]]，西北鄰[[法國|法國]][[阿爾薩斯|阿爾薩斯]]，東北與[[德國|德國]]南北走向的[[黑森林|黑森林]]山脈接壤；而[[萊茵河|萊茵河]]在此東注北湧穿城而去，將巴塞爾一分為二，版圖較大者位於西岸稱為大巴塞爾区，小巴塞爾区則位於東岸。

巴塞尔与邻近的自治市[[里恩|里恩]]（Riehen）和[[贝廷恩|贝廷恩]]（Bettingen BS）以及与城市附近的古老渔村[[克莱恩许宁恩|克莱恩许宁恩]]（Kleinhüningen）联合形成巴塞尔城市州。

巴塞尔市中心是围绕着[[巴塞尔市政大厅|市政大厅]]及14世纪建成的[[巴塞尔大学|巴塞尔大学]]附近形成的。狭窄的街道及小路，因为被[[莱茵河|莱茵河]]隔断而建立的桥梁都是巴塞尔的特色之处。进入市中心部分则因为保护关系除了搭载游客的电车外，是没有任何[[机动|机动]]交通工具的。

巴塞尔是[[化工|化工]]和[[制药|制药]]工业发达的地区，尤其以知名药业公司[[诺华|诺华]]（Novartis）和[[羅氏|霍夫曼·罗氏集团]]（Hoffmann - La Roche）为首的瑞士最大的药品公司总部都设在巴塞尔。位于巴塞尔附近的[[穆滕茨|穆滕茨]]（Muttenz）是全欧洲最大的[[铁路调路|铁路调路]]及[[Rangierbahnhof|分路车站]]。巴塞尔港口是瑞士仅有的由莱茵河畔向[[北海_(大西洋)|北海]]的出口。[[巴塞尔－米卢斯－弗赖堡欧洲机场|巴塞尔－米卢斯－弗赖堡欧洲机场]]是与邻近的[[法国|法国]]一同管理的，它位于法国境内，由一条高速公路通往瑞士境内。边界关税于是成为了机场范围内的收费项目。

巴塞尔有时被冠以''Romandie''的德語名称，因为巴塞尔的官方语系是德语，其准确的官方称呼是巴塞尔-德语。

== 市徽 ==
[[File:Wappen_Basel-Stadt_matt.svg|thumb]]
巴塞尔市亦即巴塞尔州的的市徽是一个向左弯曲的黑的主教权杖。它镶嵌在白色的背景上。权杖向下变宽，末端是三个尖角，被三个线条中断。巴塞尔权杖的象征是弯曲的主教权杖。徽章柄是狮子，原始的男人，天使和自15世纪的蛇妖-那是一只公鸡头和蛇尾形状的龙。

== 地理 ==

巴塞爾位於瑞士最西北端，橫跨萊茵河兩岸，並與法德兩國交界。[[小巴塞爾|小巴塞爾]]與巴塞爾市北部則位於萊茵河右岸。

巴塞爾市面積為23.91平方公里（2009年）；其中農業用地占4%（0.95平方公里），林地3.7%（0.88平方公里），另有6.1%為河流與湖泊（1.45平方公里），此外則均為市區。

巴塞爾的工業區占市區的10.2%，住宅房舍40.7%，大眾運輸與公共設施為24%，水電供應設施約占2.7%，而公園用地則為8.9%。

=== 行政區 ===

巴塞爾市其下沒有再細分行政區，但依住宅區劃分為19區，分別位於[[莱茵河|莱茵河]]兩側：

* 左岸-四分之一：城镇 （{{tsl|de|Grossbasel|大巴塞尔区}}）, 莱因河以南的区域为大巴塞尔。
* 右岸-四分之一：城镇 （{{tsl|de|Kleinbasel|小巴塞尔区}}）, 莱因河以北的区域为小巴塞尔。

而另两个自治市[[里恩|里恩]]和[[贝廷恩|贝廷恩]]则不属于巴塞尔自治市。

=== 氣候 ===
巴塞爾位於萊茵河谷地，它的地理位置正好接收來自勃艮第省隘口進入的地中海氣流，使此地氣候特别溫和晴朗，为[[海洋性气候|海洋性气候]]区。相較於瑞士其他地區，巴塞爾的秋天較不易起霧。巴塞爾的年均温約為攝氏9.4度，降雨量為791公釐<ref>http://www.klimadiagramme.de/Europa/Schweiz/basel.html</ref>。五月與六月為巴塞爾的雨季，這兩個月的降雨天數均超過十日；二月則為最乾燥的季節。冬天的巴塞爾積雪甚淺，每年平均降雪日僅僅30天。當地風力不強，天氣好時吹東風，不好時則吹西風。<ref name=ClimateBB>{{cite web |url=http://www.meteoswiss.admin.ch/product/output/climate-data/climate-diagrams-normal-values-station-processing/BAS/climsheet_BAS_np8110_e.pdf |title=Climate normals Basel / Binningen, Reference period 1981−2010 |publisher=Swiss Federal Office of Meteorology and Climatology - MeteoSwiss |location=Zürich-Airport, Switzerland |date=2 July 2014 |format=PDF |accessdate=3 April 2015}}</ref>

== 经济 ==

巴塞爾2010年的失業率為4.2%。根據2008年的資料<ref>{{cite web |url=http://www.bfs.admin.ch/bfs/portal/en/index/regionen/02/gemeindesuche.html |title=存档副本 |accessdate=2011-11-09 |0=2011-11-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20111116031935/http://www.bfs.admin.ch/bfs/portal/en/index/regionen/02/gemeindesuche.html |archivedate=2011-11-16 }}</ref>，巴塞爾的經濟活動以二級產業（即工業部門）之從業人口最多，為120,130人，三級產業（服務業）居次（82,449人），最後則是一級產業（農業），34,645人。其中服務業的的從業人口中有46.2%為女性。

一年一度的巴塞爾樣品博覽會（[[Mustermesse_Basel|Mustermesse Basel]]，簡稱MUBA）於萊茵河右岸的小巴塞爾展出。其餘種要的展覽活動包括[[Baselworld|Baselworld]]（巴塞爾世界鐘錶珠寶博覽會）與[[Art_Basel|Art Basel]]（巴塞爾藝術博覽會）等。

=== 出版业与媒体 ===

1460年[[巴塞尔大学|巴塞尔大学]] 的成立极大促进了当地出版与印刷业的发展。当时有50多个大出版家。

== 交通 ==

=== 港口 ===
巴塞爾擁有瑞士唯一的貨運港口。貨物自巴塞爾港口經[[莱茵河|莱茵河]]可到達[[北海_(大西洋)|北海]]的海港[[鹿特丹港|鹿特丹港]]。

=== 機場 ===
[[巴塞尔－米卢斯－弗赖堡欧洲机场|巴塞尔－米卢斯－弗赖堡欧洲机场]]服務巴塞爾及週邊的法國[[米卢斯|米卢斯]]、德國[[弗賴堡|弗賴堡]]。機場全部位於法國境內，但由瑞士和法國共同管理。機場分為兩個獨立的建築區域，分別服務法國和瑞士；在瑞士加入[[申根协定|申根协定]]之前，兩個區域之間設有出入境檢查站。

=== 鐵路 ===
巴塞爾是一個重要的鐵路樞紐站。城中有分屬瑞士、法國、德國的三個主要火車站——瑞士的[[巴塞爾瑞士聯邦鐵路車站|巴塞爾瑞士聯邦鐵路車站]]、法國的[[巴塞尔瑞士车站|巴塞尔法國國家鐵路車站]]、德國的[[巴塞尔巴登车站|巴塞尔巴登车站]]。瑞、法兩國的車站其實共站，但由[[海关|海关]]區隔。巴登車站則在城市的另一側。

由德國[[卡尔斯鲁厄|卡爾斯魯厄]]前往巴塞爾的高速鐵路[[ICE|ICE]]於2008年完工。[[法国高速铁路莱茵河-罗讷河线|法国高速铁路莱茵河-罗讷河线]]則於2011年12月投用。

巴塞爾的本地鐵路服務由{{Tsl|de|Regio-S-Bahn Basel|巴塞爾地區鐵路}}提供。

=== 公路 ===
巴塞爾位於瑞士[[A3高速公路_(瑞士)|A3高速公路 (瑞士)]]上

=== 公共交通 ===
 [[File:Basel_-_Straßenbahnnetzplan.png|thumb]]
巴塞爾有發達的城市公共交通網絡，包括發達的{{Tsl|de|Strassenbahn Basel|電車網絡}}。綠色的[[有軌電車|電車]]和[[巴士|巴士]]由巴塞爾城市半州的{{Tsl|de|Basler Verkehrs-Betriebe|巴塞爾公共交通網絡}}（BVB）運營；而黃色的電車和巴士則由[[巴澤爾鄉村半州|巴澤爾鄉村半州]]的{{Tsl|de|Baselland Transport|巴塞爾鄉村交通}}（BLT）運營。
巴塞爾的BVB和鄰近的法國[[阿爾薩斯|阿爾薩斯]]地區、德國的[[巴登|巴登]]地區共享交通網絡。通勤鐵路網由[[瑞士聯邦鐵路|瑞士聯邦鐵路]]、[[法國國家鐵路|法國國家鐵路]]、[[德国铁路|德國鐵路]]共同提供服務，連接{{Tsl|de|Trinationaler Eurodistrict Basel|巴塞爾都會區}}與週邊城鎮。

== 宗教 ==

巴塞爾約有四分之一的人口[[羅馬天主教|羅馬天主教]]，另外23.5%屬於[[瑞士归正会|瑞士归正会]]（Swiss Reformed Church）。回教人口約為12,368人，占7%。無宗教信仰的人口約為52,321人（31.41%）。<ref>{{cite web |url=http://www.pxweb.bfs.admin.ch/Database/German_40%20-%20Eidgen%C3%B6ssische%20Volksz%C3%A4hlung/40.3%20-%202000/40.3%20-%202000.asp?lang=1&prod=40&secprod=3&openChild=true |title=存档副本 |accessdate=2013-11-19 |0=2014-04-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20140409212530/http://www.pxweb.bfs.admin.ch/Database/German_40%20-%20Eidgen%C3%B6ssische%20Volksz%C3%A4hlung/40.3%20-%202000/40.3%20-%202000.asp?lang=1&prod=40&secprod=3&openChild=true |archivedate=2014-04-09 }}</ref>

== 文化 ==
巴塞尔有着非常丰富的文化生活。（1997年，它曾尝试角逐成为“欧洲文化之都”，虽然最后没有成功），巴塞尔的[https://en.wikipedia.org/wiki/Schola_Cantorum_Basiliensis Schola Cantorum Basiliensis]成立于1933年，自中世纪巴洛克时代就已经成为全球性的音乐表演和研究中心。巴塞尔剧院的歌剧团，芭蕾舞团剧目繁忙。瑞士巴塞尔交响乐团也是瑞士规模最大的乐团。2004年5月，青年合唱团的第五次欧洲音乐节合唱节开幕：这项传统开始于1992年，而这个节日的主人则是当地的巴塞尔童声合唱团。

=== 重要人物 ===
荷蘭哲學家[[伊拉斯謨|伊拉斯謨]]曾於巴塞爾任教，他的晚年也在此度過。宗教改革家與[[喀爾文教派|喀爾文教派]]的創始者[[喀爾文|喀爾文]]也曾在巴塞爾首次發表他的著作《基督教原理》（Institute of Christian Religion）。瑞士網球名將[[罗杰·费德勒|費德勒]]出生於巴塞爾。

=== 博物館 ===

巴塞爾市的博物館眾多，且涵蓋主題相當廣泛，尤其以豐富的藝術收藏見長。巴塞爾的文物收藏與推廣活動可回溯到16世紀。從1980年代起，許多私人收藏也開始對外展出。

今日巴塞爾擁有超過20間的博物館。其中包含[[巴塞爾藝術博物館|巴塞爾藝術博物館]]（Kunstmuseum Basel），館藏以十九與二十世紀的藝術品見稱，其中還包括[[畢卡索|畢卡索]]的展覽間；中世紀與文藝復興時期的歐洲藝術收藏也非常豐富。
