'''巴黎-萨克雷大学'''（Université Paris-Saclay）是一所[[法国|法国]][[巴黎|巴黎]]一所[[大学与院校共同体|大学与院校共同体]]，建于2014年12月29日。

== 办学目标 ==
巴黎-萨克雷大学筹委会主席Dominique Vernay说该大学的办学目标是在[[世界大学学术排名|世界大学学术排名]]中排前十，欧洲大陆第一。<ref>{{Cite news |last = Staley |first = Oliver |date = 13 March 2014 |title = Nations Chasing Harvard Merge Colleges to Ascend Rankings |url = http://www.bloomberg.com/news/2014-03-13/nations-chasing-harvard-merge-universities-to-ascend-rankings.html |work = [[bloomberg.com|bloomberg.com]] |accessdate = 13 March 2014 }}</ref>該大學也希望儘量發揮研究成果的經濟和商業潛力，一部份靠大學衍生企業（university spin-off）和研究衍生企業（research spin-off），但也會與成熟企業緊密合作。<ref name="Campus profile">{{Cite web |title = Towards Université Paris-Saclay |url = http://www.campus-paris-saclay.fr/en/Idex-Paris-Saclay/Towards-Universite-Paris-Saclay |work = [[Campus_du_plateau_de_Saclay|campus-paris-saclay.fr]] |accessdate = 15 March 2013 |deadurl = yes |archiveurl = https://web.archive.org/web/20140604095904/http://www.campus-paris-saclay.fr/en/Idex-Paris-Saclay/Towards-Universite-Paris-Saclay |archivedate = 2014年6月4日 }}</ref>

== 成员 ==
