《'''巴黎宝贝'''》（{{lang-en|'''Perfect Baby'''}}）2011年上映于[[中国大陆|中国大陆]]的爱情片，导演[[王菁|王菁]]，主演[[邓超|邓超]]、[[珍·玛奇|珍·玛奇]]、[[伊能静|伊能静]]等。

== 剧情 ==
该片讲述一个中国[[富二代|富二代]]马小顺和法国美女情爱诗人艾玛之间的姐弟恋爱情故事。<ref name="fish " />

==花絮==
法国导演[[让·雅克·阿诺|让·雅克·阿诺]]是该片的艺术顾问。

该片引爆“[[捐精|捐精]]”和“[[富二代|富二代]]”的话题争论。<ref name="rqslh " >{{cite web |url= http://ent.ifeng.com/zz/detail_2011_08/30/8788890_0.shtml|title= 《巴黎宝贝》首周票房达千万 捐精话题引发热议|accessdate=2011年8月30日  |publisher=凤凰网 |work= |language =zh-cn}}</ref><ref name="nca ">{{cite web |url=http://et.21cn.com/movie/xinpianreping/2011/08/31/9017608.shtml |title=《巴黎宝贝》：富二代也是人 会哭会痛会成长 |accessdate=2011-08-31 |publisher=21CN |work= |language=zh-cn |archive-url=https://web.archive.org/web/20110928062107/http://et.21cn.com/movie/xinpianreping/2011/08/31/9017608.shtml |archive-date=2011-09-28 |dead-url=yes }}</ref>

该片被誉为邓超版本的“[[宝贝计划|宝贝计划]]”。<ref name=" rta" >{{cite web |url=http://news.163.com/11/0906/12/7D95FF9F00014AEE.html |title= 《巴黎宝贝》：邓超版的“宝贝计划|accessdate= 2011-09-06|publisher=网易 |work= |language =zh-cn}}</ref>

该片一定程度展现了[[东方文明|东方文明]]和[[西方文明|西方文明]]之间沟通碰撞的侧影。<ref name=" " >{{cite web |url=http://ent.ifeng.com/movie/news/mainland/detail_2011_09/05/8944673_0.shtml |title=《巴黎宝贝》：一杯洋气的奶油爆米花  |accessdate=2011年9月5日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>

该片继承了欧洲电影的重视艺术的传统，拍摄了许多自然景物。<ref name=" rsqa" >{{cite web |url= http://ent.ifeng.com/movie/news/mainland/detail_2011_09/05/8940000_0.shtml|title=《巴黎宝贝》：中国人在巴黎 |accessdate=2011年9月5日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>

该片有法国式的浪漫情调和幽默效果。<ref name=" shje" />

2011年7月26日，该片制作方在北京举行“[[法兰西之吻|法兰西之吻]]征选启动仪式”，该片中的法国演员和10对中国情侣展示了法国式[[湿吻|湿吻]]。<ref>{{cite web |url= http://www.cqcb.com/cbnews/instant/2011-07-27/1695180.html|title= 《巴黎宝贝》玩湿吻大赛 男女演员、观众齐舌吻|accessdate=2011-07-27 |publisher=晨网 |work= |archiveurl=http://www.cqcb.com/cbnews/instant/2011-07-27/1695180.html |archivedate= 2012年12月16日 |language =zh-cn}}</ref>

该片中有伊能静和邓超跳[[探戈|探戈]]的情节。<ref>{{cite web |url=http://ent.qq.com/a/20101217/000240.htm |title=《巴黎宝贝》开放探班 伊能静与邓超激情探戈 |accessdate= 2010年12月17日|publisher=腾讯网 |work= |language =zh-cn}}</ref>
