'''弘昼'''（{{lang-mnc|ᡥᡡᠩ ᠵᡝᡠ|v=hūng jeo}}；{{bd|1712年|1月5日|1770年|9月2日|catIdx=H弘}}），[[雍正帝|雍正帝]]第五子，[[乾隆帝|乾隆帝]][[弘曆|弘曆]]之弟，封'''和親王'''，諡號'''恭'''，是為'''和恭親王'''。

性情剛強暴躁，好[[飲酒|飲酒]]。在世時，常常假裝自己已死，為自己辦[[喪事|喪事]]取樂，於是被稱'''荒唐王爺'''，歷史學家指他其實是向[[弘曆|弘曆]]宣示他無意爭[[帝位|帝位]]，而假裝“荒唐”，韬光养晦、自污保命。

和亲王府旧址位于[[清陆军部和海军部旧址|清陆军部和海军部旧址]]。弘昼五世孫[[溥良|溥良]]為[[光绪|光绪]]六年（1880年）庚辰科進士，六世孫[[毓隆|毓隆]]為[[光緒|光緒]]二十年（1894年）甲午恩科進士，八世孫[[启功|启功]]為當代書法家、國畫大師。

==生平==
[[康熙|康熙]]五十年[[十一月廿七|十一月廿七]]出生，生母[[纯懿皇贵妃|纯懿皇贵妃]]耿氏，时为[[雍王|雍王]]府[[格格|格格]]，而由[[乾隆帝|乾隆帝]]的生母[[崇慶皇太后|崇慶皇太后]]撫養成人<ref name="啟功">{{cite book |author=侯剛 |title=啟功 |accessdate=2014-01-18 |edition=第一版 |series=中國文博名家畫傳 |publisher=文物出版 |location=北京 |language=zh |isbn=7-5010-1550-3 |pages=2|date=2003年12月}}</ref>。

[[雍正|雍正]]十一年（1733年）封[[和亲王|和亲王]]。

雍正十三年（1735年）二月，设办理[[苗疆|苗疆]]事务处，[[弘历|弘历]]与弘昼、[[鄂尔泰|鄂尔泰]]等共同办理苗疆事务<ref>[http://www.guoxue123.com/shibu/0101/00qsg/219.htm 《清史稿·卷二百二十·列传七》]</ref>。八月二十三日凌晨，雍正駕崩，[[内侍|内侍]]取出[[秘密建儲|密谕]]，宣布弘历即位，是為乾隆帝。十月，乾隆命弘昼管理[[内务府|内务府]]、[[御书处|御书处]]的事务。

乾隆四年（1739年）二月，管理[[雍正帝|雍正帝]]舊宅[[雍和宫|雍和宫]]。八月，授正白旗满洲都统。十二月，管理武英殿。

乾隆五年（1740年）二月，授镶黄旗满洲都统。三月，辦理勘定八旗佐領世職應襲則例的事務。

乾隆十一年（1746年）十二月，任玉牒馆总裁。

乾隆十五年（1750年），管理内务府[[奉宸苑|奉宸苑]]。

乾隆十七年（1752年）四月，解管奉宸苑事务。

乾隆十八年（1753年）正月，弘昼又提升为[[议政大臣|议政大臣]]。五月，解除管武英殿御书处事务。

乾隆二十八年（1763年）八月，管理正黄旗觉罗学事务。

乾隆三十五年（1770年）庚寅七月十三日申時薨，年五十八歲，諡'''恭'''，由第二子[[永璧|永璧]]承袭他的爵位。

==皇兄疼愛==
乾隆帝即位后，把父皇的[[潛邸|潛邸]]及财物全赐给了弘昼，所以弘昼富於其他親王。弘昼倚着兄长的威势，傲慢任性，肆意妄为。<ref>[http://www.chinanews.com/hb/2010/12-29/2755219.shtml 乾隆与弘昼：龙兄御弟]</ref>

弘昼性剛，一次上朝，弘昼因事与[[军机大臣|军机大臣]]、获封一等公的[[讷亲|讷亲]]有了小争执，竟然当着满朝文武的面殴打讷亲，乾隆目睹了整个事情的经过，既不怪罪，也不出声阻止。文武百官见状，从此无人敢惹弘昼。<ref>[http://news.ifeng.com/history/zhongguogudaishi/detail_2011_02/18/4737929_0.shtml 乾隆皇帝为什么放纵两个骄横跋扈的弟弟？]</ref>

乾隆帝曾於[[乾清宮|正大光明殿]]监考[[八旗|八旗]]子弟，到了中午的时候，弘昼请乾隆帝去用膳，乾隆帝認為八旗子弟積習懈怠玩忽，不願離開。弘昼氣著说：「皇上难道以为我要私下[[舞弊|收買士子心]]麼？」乾隆當作沒聽到，真的走了。[[傅恆|傅恆]]責備弘昼：「這難道是臣子可以講的話麼？」弘昼立刻後悔了，次日弘昼去向乾隆脫帽謝罪，乾隆說：「假使朕昨天回了你一句話，[[大不敬|大不敬]]之罪就足夠讓你碎屍萬段了。你的話雖然是憨直，但是你想讓我趕緊用餐的心，是出於兄弟友愛之情，所以朕原諒，但你以後小心，不要再說出這種話來了。」於是兄弟倆友愛如初。<ref>[[昭槤|昭槤]]《[[啸亭杂录|啸亭杂录]]》尝命王监试八旗子弟于正大光明殿，日已晡，上尚未退朝，恭王请上退食。上以士子积习疲玩，未之许，王激烈曰：“上疑吾买嘱士子心耶？”上怡然退。傅文忠责王曰：“此岂人臣之所宜语？”王始悔悟。次日免冠请罪，上方云：“昨朕若答一语，汝身应粉矣！其言虽戆，心实友爱，故朕恕之。然他日慎勿作此语也。”友爱如初。</ref>

== 個人作品 ==
弘昼的詩文存世者很少，精品更少。《金樽吟》是其代表作，这首诗是向乾隆表述了自己无意[[帝位|帝位]]、只想把酒言欢，及时行乐的心迹，被历史学家誉为“救命诗”。诗-{云}-：「世事无常耽金樽，杯杯[[茅台酒|臺]][[郎酒|郎]]醉红尘。人生难得一知己，推盃换盏话古今。」

==人物评价==
乾隆帝：

# 朕弟和碩和親王、秉性純誠。持躬端恪。髫年共學。友愛實深。自備位親藩。懋昭敬慎。方冀六旬同慶。棣萼言歡。
# 吾弟少於吾甫三月。皇父在潛邸時，育吾二人於東西室，及九歲讀書，同受經於傅先生”，“與吾自孩提以至於今，且孺且耽，恰恰如也，余既以同氣之親，相親相勗於平日，晨夕之與俱，筆硯之與共，爾我形骸之悉化。
# 予與王幼同學同課，習為詩古文詞。當是時侍奉皇考膝下，優游書府，日寢饋於經史文字中，世綱塵務，毫髮不以嬰其心。吾兩人者，相規以善，交相勉，相得無間，如是者垂二十年，天倫之樂無過於是。

== 家庭 ==

==== 妻妾 ====
嫡福晉[[吳扎庫氏|吳扎庫氏]]，副都統五什圖之女。

側福晉[[章佳氏|章佳氏]]，護軍參領雄保之女。

側福晉[[崔佳氏|崔佳氏]]，崔奇哲之女。雍正十二年二月的外[[八旗选秀|八旗选秀]]中，時年十六歲的崔佳氏被指婚予和亲王，在当年十月初四日娶进。后来因为崔佳氏没有生育，所以没有被收进《玉牒》。

側妃[[佟佳氏|佟氏]]，乾隆十一年去世。《皇朝文典》有錄入和碩和親王側妃祭文，「恩施嬪侍」等的字眼可知她死前高宗特恩命宮中的一位嬪娘娘侍奉她，未知為前朝妃嬪還是乾清宮主位。

==== 兒子 ====
第一子：[[永瑛|永瑛]]，雍正九年四月十二日生，母嫡福晉吳扎庫氏。

第二子：[[永璧|永璧]]（和碩和勤親王），雍正十一年六月十三日子時生，母嫡福晉吳扎庫氏。

第三子：未有名，雍正十二年四月二十日生，母側福晉章佳氏。

第四子：[[永瑸|永瑸]]（散秩大臣、鎮國將軍，雍正十三年十一月十一日生，母嫡福晉吳扎庫氏。

第五子：未有名，乾隆三年十二月初五日生，母嫡福晉吳扎庫氏。

第六子：[[永瑍|永瑍]]（頭等侍衛、鎮國將軍）。

第七子：[[永琨|永琨]]（將軍不入八分輔國公），母嫡福晉吳扎庫氏。

第八子：[[永璔|永璔]]，乾隆十年八月二十四日生，母嫡福晉吳扎庫氏。

==== 女兒 ====
[[和硕和婉公主|和碩和婉公主]]：母嫡福晉烏札庫氏，雍正十二年六月二十四生。乾隆初撫養宮中，封和碩和婉公主。
