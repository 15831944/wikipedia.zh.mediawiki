'''惠阳区'''（官方音译：{{lang|en|'''Huiyang'''}}；传统外文名：{{lang|en|'''Fuiyong'''}}）是[[中国|中国]][[广东省|广东省]][[惠州市|惠州市]]的一个[[市辖区|市辖区]]，位于[[广东省|广东省]]东南部，东部与广东省[[惠东县|惠东县]]毗邻，南部连着[[大亚湾|大亚湾]]，与[[香港|香港]]隔海相望，西部与广东省[[深圳市|深圳市]]和[[东莞市|东莞市]]接壤，北部与广东省[[惠州市|惠州市]][[惠城区|惠城区]]交界。惠阳区辖区面积915.6平方公里，2016年末全区常住人口59.86万，其中户籍人口38.02万。<ref>[http://www.huiyang.gov.cn/hygk/rkgk/index.shtml 惠州市惠阳区人民政府网站]</ref>

== 行政区划 ==
辖5个街道：[[淡水街道|淡水]]、[[秋长街道|秋长]]、[[三和街道|三和]]、[[澳头街道|澳头]]、[[霞涌街道|霞涌]]，6个镇：[[永湖镇|永湖镇]]、[[良井镇|良井镇]]、[[平潭镇|平潭镇]]、[[沙田镇_(惠州市)|沙田镇]]、[[新圩镇|新圩镇]]、[[镇隆镇|镇隆镇]]。<ref>[http://www.stats.gov.cn/tjbz/cxfldm/2011/44/13/441303.html 中华人民共和国国家统计局关于惠州市惠阳区的行政区划]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 历史 ==
[[宋朝|宋朝]][[天禧_(北宋)|天禧]]四年（1020年）由祯州改为惠州，始有惠州。

[[民国|民国]]二年（1913年），原歸屬惠州的[[歸善縣|歸善縣]]改稱惠阳縣。

1963年始设立[[惠阳专区|惠阳专区]]、惠阳地区。1988年撤销惠阳地区，[[惠州市|惠州市]]升为地级市，惠阳为惠州所辖的一个县。

1994年撤惠阳县设立惠阳市、2003年撤惠阳市设立惠阳区。　　

惠陽客家人來源有三：廣東南雄、福建及嘉應州。惠陽成為客家人移居海外或香港之前的跳板。

1975年5月，三栋公社从惠阳县境划出归惠州管辖。

1994年5月6日，国家民政部批准惠阳县撤销，设置惠阳市（县级）建置，为省辖市，委托惠州市代管。

==民族==
惠阳区的居民几乎都是[[汉族|汉族]]人，大部分属[[客家民系|客家民系]]。惠阳的客家人占全区本地人口的95%以上，是纯客属地区。惠阳本地汉族人中除[[客家人|客家人]]外，其余为[[鹤佬人|鹤佬人]]。

== 语言 ==
{{Main|惠阳话|惠城话}}

惠阳的语言以[[客家话|客家话]]为主，个别地区亦使用[[閩南語|閩南語]]。

惠阳客家话大多属[[客家語|客家語]][[粤台片|粤台片]][[新惠小片|新惠小片]]，称为[[惠阳话|惠阳话]]，以淡水话为代表，是惠阳区通用的客家话。惠阳话是客家话的强势方言之一，广泛流传于惠州、深圳、东莞东南部山区、香港新界及海外，广义的惠阳话等同于新惠小片。除惠阳话外，部分郊区流行[[惠州話|惠城话]]，為一种主要流行于惠州市惠城区，与惠陽客家话差异很大的本地方言。

== 经济 ==
惠阳区农业比较有特色的为优质谷、[[荔枝|荔枝]]、蔬菜、[[西瓜|西瓜]]、速生丰产林等三高农业。[[工业|工业]]以[[外向型经济|外向型经济]]为主，区内外资企业众多，区内比较有实力的企业为[[联想集团|联想集团]]、中建电讯、东亚电子、中狮电子等。惠阳区的第三产业近年来发展迅速，区内宾馆、酒店、旅游、金融等服务设施比较完善，各种商业非常活跃。

== 旅游 ==
惠阳区临山靠海，旅游资源非常丰富，有金果湾乡村俱乐部、大甲岛、沙田温泉、银瓶山等旅游景点。另外惠阳是不少名人的故乡，如港星陳小春、鍾漢良等。这里还有[[叶挺|叶挺]]故居、叶挺纪念馆、[[邓仲元|邓仲元]]故居、[[邓承修|邓承修]]故居等人文景点。历代古迹有淡水祖庙、清泉寺、新墟盘古宫、镇隆和秋长[[客家围屋|客家围屋]]等。

== 交通 ==
惠阳区内有[[惠州机场|惠州机场]]（原名平潭机场,位于惠阳区[[平潭镇|平潭镇]]）及[[惠州港|惠州港]]，[[深汕高速公路|深汕高速公路]]、[[潮莞高速公路|潮莞高速公路]]、[[惠深沿海高速公路|惠深沿海高速公路]]、[[惠澳高速公路|惠澳高速公路]]以及[[厦深铁路|厦深铁路]]贯穿其中，所以交通十分便利。
