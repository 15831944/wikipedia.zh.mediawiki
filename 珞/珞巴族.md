'''珞巴族'''（{{bo|z=ལྷོ་པ་|w=lho pa}}，即藏語“南方人”），居於中國[[西藏自治區|西藏自治區]]東南界與[[印度|印度]][[阿魯納恰爾邦|阿魯納恰爾邦]]（又稱[[藏南|藏南]]）山上，人口60萬，絕大部份在印度。中華人民共和國識別為56民族之一，印度與西方按部落再細分為多個民族，包括Nyishi族，Adi族，Mishing族等，是印度東北的[[黃種人|黃種人]]。母語人口最多三個是Nyishi語，[[阿迪語|阿迪語]]，Mishing語，均屬[[達尼語支|達尼語支]]，但不能完全互通。

== 名称 ==

* blo pa（{{bo|z=བློ་པ་}}），意为“智慧”；

* slo pa（{{bo|z=སློ་པ་}}），意为“附近”；

* lho pa（{{bo|z=ལྷོ་པ་}}），意为“南方”；

* klo pa（{{bo|z=ཀློ་པ་}}），意为“没有开化”。

:受晚期[[西藏农奴制度|农奴制]]影响，多用klo pa作为称呼。1965年规定称呼为lho pa，废除另外三种。
== 部落 ==

使用達尼语支的部落：

* 崩尼（Bengni）

* [[博嘎尔部|博嘎尔]]（Bokar）

* 纳（[[:en:Nga_people|Nga]]）

* 米古巴（Miguba）

* 米新巴（Misinba）

* 德根（[[:en:Tagin_(people)|Tagin]]）

* 希蒙（Shimong）

* 巴达姆（[[:en:Padam_(tribe)|Padam]]）

* 民荣（Minyong）

* 尼西（[[:en:Nishi_(tribe)|Nishi]]）

* 迦龙（[[:en:Galo|Bengni Gallong]]）

* 登尼（[[:en:Apatani|Tanii]]）
其他部落有[[崩如|崩如]]、布瑞（Puroik；苏龙，"Sulung"）、[[義都語|義都]]（Idu）等。

== 中国实际控制区内珞巴族人口分布 ==

是[[中国|中国]][[西藏自治区|西藏自治区]][[山南市|山南市]]、[[林芝市|林芝市]]的一个少数民族，主要分布在西藏东起察隅、西至门隅之间的[[珞隅|珞隅]]地区。据1980年代初[[中国|中国]]的国家民委编写的出版物，珞巴族数量已超过30万，根据人口自然增长情况，目前总人口约60万。其中处于[[中国|中国]]实际控制区内仅有3682人。

== 文化 ==
=== 民居 ===

珞巴族的住宅一般建在河流两岸的半山坡上，方向面向山坡，以避免潮湿的天气和[[洪水|山洪]]。建筑以两层干栏式房屋为主。珞巴族各部落的建築风格各不相同。布瑞部落一般居住在竹木结构的简易房屋。珞渝地区和[[墨脱县|墨脱]]南部的珞巴族的住宅以家长制或氏族家庭风格的长屋为主。达木地区部落居住在小栋房。<ref>陈立明. [http://www.cnki.com.cn/Article/CJFDTOTAL-XZDX200904006.htm 珞巴族的传统文化与环境保护][J]. 西藏大学学报:社会科学版, 2009, 24(4):6-12.</ref>珞巴族传统房屋中心有火塘，族人围绕火塘居住，不同的位置对居住人有严格的划分。<ref>陈立明. [http://www.cnki.com.cn/Article/CJFDTOTAL-XZMZ200303004.htm 珞巴族传统居住习俗及其变化][J]. 西藏民族学院学报:哲学社会科学版, 2003, 24(3):20-24.</ref>
=== 饮食 ===

珞巴族主食以谷物为主，喜辣食，好饮酒。米类以干食为主，制作方式以蒸和焖等方式。面食制作方式为烙与烤，以饼食为主，便于携带和保存。肉类用熏烤和炖煮制作，蔬菜常清煮或生食。珞巴族善于[[酿酒|酿酒]]，酒类以[[稻|稻米]]、[[玉米|玉米]]、鸡爪谷、木糌粑、棕心粉、[[蜜蜡|蜜蜡]]和水果酿成。珞巴族一般不喝茶，款待客人时以酒代茶。<ref>{{Cite journal|title=珞巴族饮食文化|author=于乃昌|url=http://www.cnki.com.cn/Article/CJFDTotal-ZOXH802.013.htm|journal=中国西藏|issue=02|doi=|others=|year=1998|volume=|page=42-44|pmid=}}</ref>在菜肴方面，珞巴族喜食牛肉、猪肉、野牛肉、香獐肉、鱼肉、鼠肉。
=== 婚姻 ===

珞巴族现实行一夫一妻制，但仍存在一夫多妻现象。在神话传说中记载珞巴族历史上存在[[近親婚姻|血缘婚]]和[[对偶婚|对偶婚]]的形式。珞巴语中原本没有“婚嫁”、“娶亲”等词汇。珞巴族的旧式婚姻普遍存在[[童婚|童婚]]、[[抢婚|抢婚]]、买卖婚姻的现象。通婚方面，在[[博嘎尔部落|博嘎尔]]部落中，不能与同氏族、外部落、外氏族的人结婚，在等级方面也有严格的婚姻限制。妇女地位低下，没有财产继承权。但近年来随着社会的发展，买卖婚、等级婚观念在淡化。<ref>{{Cite web|url=http://www.china.com.cn/aboutchina/zhuanti/lbz/2009-07/15/content_18143348.htm|title=珞巴族的旧式婚俗|accessdate=2009-07-15|author=|date=|publisher=中国网}}</ref>珞巴族的订婚仪式称为“雅玛如”，意为”打记号“。男方向女方家庭攀婚时，其婚姻介绍人会在女方脑门涂抹酥油。娶亲日期由杀鸡占卜选定。<ref name=":0">{{Cite book|title=珞巴族社会历史调查二|last=|first=|publisher=民族出版社|year=2009|isbn=9787105087792|location=|pages=}}</ref>
=== 体育娱乐 ===

珞巴族在重要节日、宴庆及“梭白巴”“索苗仁”等重大宗教活动时常对歌和跳舞，在体育方面有[[摔跤|摔跤]]、[[跳高|跳高]]、举重石、[[射箭|射箭]]、断木杆、游泳、比武、拔河等活动。<ref name=":0" />{{rp|468}}

== 宗教 ==
{{Fact|珞巴族很少信仰藏传佛教|time=2008-8-21}}，在上珞瑜地区的珞巴族信仰[[苯教|苯教]]，而在下珞瑜地区的珞巴族则大多数坚持万物有灵，其中部分人信仰太阳神。
珞巴族的宗教信仰主要是[[巫覡宗敎|巫覡宗敎]]，也有部分信仰[[藏传佛教|藏传佛教]]。
== 服饰聚焦 ==

珞巴族服饰独具特色。{{Fact|男子一般穿藏式氇氆长袍，外套黑色羊毛坎肩，长及腹部，为套头式。妇女一般穿无领窄袖对襟上衣，以麻布为原料。下身围紧身筒裙，裙长略过膝部，用带子扎紧裹腿。|time=2019-01-31T02:31:28+00:00}}

== 语言 ==

{{main|珞巴语}}

珞巴族的语言都属于[[藏缅语族|藏缅语族]]。多数部落使用[[塔尼语支|塔尼语支]]的语言，如崩尼-博嘎尔语。另有[[义都语|义都语]]、[[崩如语|崩如语]]、[[苏龙语|苏龙语]]，不属于塔尼语支。
