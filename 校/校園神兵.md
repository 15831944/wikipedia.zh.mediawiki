《'''校園神兵'''》（Martin Mystery)改編自歐洲同名漫畫'''[[神秘馬丁|神秘馬丁]]'''，並以此改編成一部動畫片。

== 故事內容 ==
故事當中描述馬丁、黛安和爪哇同時為一個名為"中心"的秘密組織工作，他們三人宛如"星際戰警"一般為了保衛地球而與外星生物、惡靈、傳說中的生物等等奮戰。
 到第三季以後，他們的任務多半是自己碰上麻煩，較缺少"中心"指派的任務

== 人物介紹 ==
=== 主要人物 ===
==== 神秘.馬丁 ====   
*中文配音員: [[何志威|何志威]]
*年齡: 16歲
*英文名字: Martin Mystery
*就讀: 魁北克多靈頓中學
*身分: 學生
*秘密身分: 秘密組織秘密探員 '''神秘.馬丁''' 
*衣服: 黃色上衣
*道具: 萬能手錶
*能力: 膽識及敏捷
*外表看似平凡的中學學生，可是實際上是組織當中秘密探員，為了保衛地球而與外星生物奮戰。喜歡追逐一些怪異的事情，對不可思議的事情感到好奇，探險中不失領導人風範，擁有相當的膽識及敏捷。
*原著漫畫中和黛安是假結婚的夫妻
*第三季中，馬丁獲得一組由"中心"資助的迷你總部，架設於他的寢室，方便辦案。

==== 黛安.倫巴 ====
*中文配音員: [[許淑嬪|許淑嬪]]
*年齡: 16歲
*英文名字: Diana Lombard
*就讀: 魁北克多靈頓中學
*身分: 學生
*秘密身分: 秘密組織秘密探員 '''黛安.倫巴''' 
*衣服: 粉紅色上衣
*能力: 聰明 推理
*她是馬丁的妹妹，跟馬丁比較不一樣的是黛安比較聰明，做事情比較條理，思考也比較有邏輯性，馬丁喜歡捉弄她，但是在任務中當黛安遇到危險時，馬丁也會去救她。而在組織當中，黛安思考能力是不可或缺的。
*原著漫畫中與馬丁是假結婚的夫妻

====原始人.爪哇====
*中文配音員: [[符爽|符爽]]
*英文名子: Java the Caveman
*就讀: 魁北克多靈頓中學
*身分: 校工 廚師
*真正身分: 20萬年前原始人
*秘密身分: 秘密組織秘密探員 '''原始人.爪哇'''
*能力: 力氣大
*個性很老實，力氣也很大，常常在任務中幫助馬丁和黛安，是團隊中最重要的動力。
*原著漫畫中和黛安是假結婚關係，為了使爪哇得到美國國籍。

=== 組織人物 ===
==== 外星人.比利 ====
*中文配音員: [[林美秀|林美秀]]
*非常膽小，外表看起來小小的，在任務中負責分析馬丁帶來的樣本並且傳送馬丁一行人到任務場所，而且總是無聲無息的突然出現在三人身旁，是大媽的助理。真實身分是外星人頭頭 真名-甘薩。
*在第三季中，比利獲得了一套偽裝衣，使他得以以青少年的外型與馬丁等人出任務。

====大媽====

==道具介紹 ==

=== 超能手錶（U-watch） ===
*使用人:'''神秘.馬丁'''
可以在任務中變出武器或工具，是任務當中好幫手，當遇到危機時，馬丁會變成武器來化解危機，而此萬能手錶也可以能來讀出不明液體的資料，可說是任務當中的好幫手。
:'''神杖'''：可變化伸縮長度的棍子，具有一定程度堅韌性。可作為武器或抵擋、輔助型工具使用。
:'''渦輪彈簧繩'''：用於逃跑躲避敵人，或者可用於纏繞捆綁用。
:'''黏液掃描'''：分析一定程度黏液性質，包括化學成分或者生物DNA來源。但進一步確認可能需將樣本交回總部。
:'''傳說索引'''：查詢各種事物歷史傳說，以提供相關資訊參考或打敗敵人的方法。
:'''鋼網發射器'''：可用來發射網子困住對方。
:'''防護罩'''：可用來形成屏障隔絕物體，多用於抵抗敵人攻擊，少數用來困住小體積敵人。缺點是使用有時間限制。
:'''阿法護目鏡'''：提供照明的眼鏡。
:發訊器：可從手錶側發射小型追蹤器。
