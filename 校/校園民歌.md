'''校園民歌'''是[[台灣|台灣]]在1970年代發展出來的[[流行音樂|流行音樂]]風格。1970年代之前，[[西洋|西洋]][[流行音樂|流行音樂]]為台灣青年人喜愛的音樂主流。但1972年，當時[[美國總統|美國總統]][[尼克森|-{zh-cn:尼克松; zh-tw:尼克森; zh-hk:尼克遜}-]]訪問[[中華人民共和國|中華人民共和國]]及1978年[[中美關係|中美斷交]]後，激發了國內年青人面對自我文化覺醒，因而觸發台灣青年學子不再高唱西洋音樂，轉向「用自己的語言，創作自己的歌曲」的想法，並喊出「唱自己的歌」的口號，被稱為「'''民歌運動'''」，造就了校園民歌的誕生。校園民歌為[[台灣流行音樂|台灣流行音樂]]產業在1980至90年代起飛奠定了基礎，其參與者有不少在日後成為[[華語流行音樂|華語流行音樂]]的要角。

== 起源與發展 ==
===民歌採集運動===
台灣的「民歌採集運動」始於1964年，由[[許常惠|許常惠]]與[[史惟亮|史惟亮]]共同倡導。他們對臺灣全省的田野民俗音樂（包括原住民音樂及漢民族傳統音樂）作了全面性的調查、採集、整理和研究工作，蒐集近三千首臺灣山地與平地原住民歌謠。此活動發掘了多位民間藝人，包括演唱民歌的[[陳達_(歌手)|陳達]]。<ref>{{Cite web|url=http://musiciantw.ncfta.gov.tw/list.aspx?p=M016&c=&t=1|title=台灣音樂群像──許常惠|accessdate=2019-04-11}}</ref>。
===詞曲創作風氣===
1971年由[[洪小喬|洪小喬]]主持的音樂節目《[[金曲獎_(電視節目)|金曲獎]]》（由[[今是有限公司|今是有限公司]]與[[歌林公司|歌林公司]]音樂出版部合作製播，由[[中國電視公司|中國電視公司]]播出），開始在大眾傳播媒體上鼓勵優良歌曲創作。

1970年代初期，在學校社團及[[救國團|救國團]]營隊活動中，已可見到學生創作歌曲的先例，比如由吳統雄和山野服務的朋友們於1973年共同創作的〈[[偶然_(校園民歌)|偶然]]〉<ref>{{Cite web|url=http://tx.liberal.ntu.edu.tw/TxMusic/Musical_Poetry/fortuitous-case.htm|title=「偶然」正名事件|accessdate=2019-04-11}}</ref>；這一類小歌的作者又包括有創作〈盼〉的[[周興立|周興立]]，其創作據稱始於1972年<ref>{{Cite web|url=https://blog.xuite.net/ericlth5/twblog/184785004|title=周興立─在民歌以前就開始創作的人|accessdate=2019-04-11}}</ref>。這些學生創作歌曲連同各地民謠被編成方便攜帶的小歌本，使用於學生的團康活動，在民歌運動開展之前已經在校園中傳唱。<ref>{{Cite web|url=https://blog.xuite.net/ericlth5/twblog/184784987|title=團康─在民歌以前唱的歌|accessdate=2019-04-11}}</ref>。後來著名的民歌手[[葉佳修|葉佳修]]也是早在1974年就寫下了處女作〈流浪者的獨白〉，並且在廣播人凌晨主持的「平安夜」節目中發表。

===台灣現代民歌===
[[File:Prof_Yu_Kwang_Chung.jpg|thumb]]]]
[[File:Tao_Hsiao-ching_in_2015_TIBE.JPG|upright]]]]
[[File:胡德夫接受凤凰URadio专访.jpeg|upright]]]]
1974年，[[楊弦|楊弦]]在[[胡德夫_(歌手)|胡德夫]]的演唱會上發表以-{[[余光中|余光中]]}-的〈鄉愁四韻〉等[[現代詩|現代詩]]譜曲的作品，獲得余光中的欣賞與鼓勵。1975年，楊弦在[[台北中山堂|台北中山堂]]舉辦「現代民謠創作演唱會」，發表八首譜自余光中《白玉苦瓜》詩集中的詩作創作曲，並且由[[洪建全|洪建全]]基金會策畫出版了唱片《中國現代民歌集》，被視為是台灣現代民歌發展的開始。1976年，在[[淡江大學|淡江大學]]的一場民歌演唱會上，音樂人[[李雙澤|李雙澤]]在台上拋掉可樂瓶（但當時的主持人[[陶曉清|陶曉清]]表示當晚李雙澤並沒有拋掉可樂瓶，「那只是後人增添的神話」），拿起吉他開始唱起台灣民謠〈補破網〉和愛國歌曲〈[[國父紀念歌|國父紀念歌]]〉，史稱「淡江事件」，隨後李雙澤創作了〈[[美麗島_(歌曲)|美麗島]]〉和改寫自[[蔣勳|蔣勳]]的〈少年中國〉，揭開「民歌運動」的序幕。

廣播人[[陶曉清|陶曉清]]開放自己的節目，讓年輕的音樂創作者有一個發表的平台。1977年，陶曉清邀請[[朱介英|朱介英]]、吳統雄、[[吳楚楚|吳楚楚]]、[[楊祖珺|楊祖珺]]、[[韓正皓|韓正皓]]、[[胡德夫|胡德夫]]、陳屏、楊弦等八位民歌手灌錄了唱片《我們的歌》，大大帶動了校園民歌風潮。陶曉清因此也被視為台灣民歌發展的重要推手。

然而「民歌」的定義及正名自楊弦《中國現代民歌集》面市已來即引發「嚴肅音樂論述」以及「現代民歌」的兩派論戰。<ref name = "source">{{Cite journal|title=民歌風華專欄|author=郭麗娟|url=http://www.tri.org.tw/per/57/57-58.pdf|journal=源|issue=57|doi=|others=|year=2006|volume=|page=57-58|pmid=}}</ref>比如當時在《中央日報》就曾有標題為〈『民歌』」不是這樣〉的短文，批評《中國現代民歌集》裡的九首歌曲被稱作民歌是「名不符實」。許常惠也曾在1978年具體提出對民歌的嚴謹定義。<ref>{{Cite news|url=http://www.merit-times.com/NewsPage.aspx?unid=445427|title=【我的青春我的歌 】民歌不是這樣？|last=曹郁美|first=|date=|work=|newspaper=人間福報|accessdate=2016-8-29|language=zh-TW}}</ref>

===民歌創作比賽與商業化===
1977年[[滾石唱片|新格唱片]]推出「[[金韻獎|金韻獎]]」，1978年[[海山唱片|海山唱片]]創辦「民謠風」；這兩項民歌創作比賽以及比賽合輯的出版，將民歌帶向商業化，並以「校園民歌」的名稱吸納了「中國現代民歌」<ref name = "source"/>。此外電視歌唱比賽節目如[[五燈獎|五燈獎]]、[[六燈獎|六燈獎]]也推波助瀾的作用，為以後的台灣唱片市場打下基礎。後續的《[[大學城_(電視節目)|大學城]]》（由[[天鼎傳播公司|天鼎傳播公司]]製播，由[[台灣電視公司|台灣電視公司]]播出）[[大學城全國大專創作歌謠大賽|全國大專創作歌謠大賽]]延續了校園民歌的風潮。

很多民歌時期著名的歌手與創作者都經歷過上述的比賽，或者搭上這一波比賽出輯的風潮，進入唱片市場。當時著名的民歌手有[[陳明韶|陳明韶]]、[[包美聖|包美聖]]、[[黃大城|黃大城]]、[[趙樹海|趙樹海]]、[[李建復|李建復]]、[[齊豫|齊豫]]、[[葉佳修|葉佳修]]、[[芝麻龍眼|芝麻龍眼]]、[[王海玲_(歌手)|王海玲]]、[[王夢麟|王夢麟]]、[[蘇來|蘇來]]與[[楊祖珺|楊祖珺]]等人。而〈[[龍的傳人|龍的傳人]]〉、〈[[動不動就說愛我|動不動就說愛我]]〉等是民歌時期的代表性作品，前者作者為[[侯德健|侯德健]]，首唱者為[[李建復|李建復]]，後者作者為[[林秋離|林秋離]]和[[熊美玲|熊美玲]]，演唱者為[[芝麻龍眼|芝麻龍眼]]。

== 特色與風格 ==

===特質===
民歌的特質是歌詞通俗而真摯，曲調優美而樸質、節奏明朗而生動，結構簡短而自然，使人聽了有身歷其境的感覺歌謠具深刻思想內容，反映社會的光明面及純正價值觀。 
===精神===
民歌能夠表現出一個國家民族命運和遭遇，甚至藉著民謠，可加強民族意識和國家觀念，其功效無可限量台灣民歌，充分表現出先民的奮鬥、進取、樂觀開朗的精神。
===題材===
在台灣各地流傳的民歌，包羅萬象，例如：歷史故事、民間傳說，道德教化、農耕漁牧、家庭倫理、男女情愛  ( 例如由[[芝麻龍眼|芝麻龍眼]]所演唱的動不動就說愛我一曲 ) ……等，都是民歌題材，內容非常豐富。

===文化資產===
民歌是人民的心靈寄託。 由一個民族的民歌，可以了解該民族文化內涵，及該民族的心聲。 
[[File:Chiang_Hsun.jpg|thumb]]]]
民歌運動早期歌詠山水、青春，旋律清新自然、歌詞樸素，並有濃厚的中國風和各大學及大專院校校園氣味，包括中國景物懷詠、古典文學意識等等。
初期有[[洪小喬|洪小喬]]、[[李雙澤|李雙澤]]、[[葉佳修|葉佳修]]、[[蘇來|蘇來]]、[[李建復|李建復]]、黃大城、[[劉藍溪|劉藍溪]]、[[邰肇玫|邰肇玫]]、[[包美聖|包美聖]]、[[施碧梧|施碧梧]]、[[金智娟|金智娟]]、[[齊豫|齊豫]]、[[陳明韶|陳明韶]]、[[趙樹海|趙樹海]]等歌手及後期的、[[潘越雲|潘越雲]]、[[鄭怡|鄭怡]]、[[芝麻龍眼|芝麻龍眼]]、[[鄭麗絲|鄭麗絲]]、[[費玉清|費玉清]]、[[潘安邦|潘安邦]]、[[劉文正|劉文正]]、[[林佳蓉_(台灣)|林佳蓉 (台灣)]]、[[黃韻玲|黃韻玲]]、[[許景淳|許景淳]]、[[施孝榮|施孝榮]]、[[王夢麟|王夢麟]]、[[銀霞|銀霞]]、[[王海玲|王海玲]]、[[木吉他合唱團|木吉他合唱團]]、[[南方二重唱|南方二重唱]]等。

作曲作詞者則有靳鐵章、畢心一、畢奐一、林建助、[[周興立|周興立]]、許翰君、陳雲山、黃大城、王新蓮、吳楚楚等民歌主力。

1970年代後期，台灣社會運動風起雲湧，人心鼓動，在[[侯德建|侯德建]]、[[羅大佑|羅大佑]]、[[李泰祥|李泰祥]]的介入，民歌運動有了質的躍進，與初期的淳樸無瑕多了人文關懷和社會責任。此時大量的歌詞是知名作家的詩作，如[[三毛_(作家)|三毛]]、余光中、[[席慕容|席慕容]] 、[[蔣勳|蔣勳]]等人的作品，而隨著侯德建、羅大佑、李泰祥、[[李壽全|李壽全]]的作品的流傳，民歌運動達到前所未有的高度。

===語言===
多數校園民歌使用[[現代標準漢語|國語]]歌詞。[[台語|台語]]歌曲較為知名的有收錄在新格唱片《金韻獎紀念專輯》中由[[簡上仁|簡上仁]]創作並演唱的〈正月調〉，以及木吉他合唱團專輯中的〈拼宵夜〉。海山唱片民謠風系列中，[[林詩達|林詩達]]個人專輯中收錄有兩首由[[李臨秋|李臨秋]]作詞、林詩達作曲的台語歌曲，[[葉佳修|葉佳修]]第二張專輯中也有一首台語創作歌曲。

== 後續與傳承 ==

台灣民歌運動轉變的原因可分列如下：

[[File:陳昇.jpg|thumb]]]]
;社會商業化、現代化的進展
此趨勢使「校園民歌」的經濟與社會環境開始改變，唱片業者以獲利為導向，美歐影音產品大舉進入台灣，「校園民歌」單薄的技術水平和樸素的形式變得難以與時代和市場趨勢相搭配，各唱片公司以大資本的商業行銷手法包裝流行歌手傾銷市場。商業化趨勢也使大學校園早期的社會文藝孤島地位被分割瓦解，所謂「校園」民歌的最後基地--各大學及大專院校校園也不復存在，甚至台灣本土化運動也無法接收「校園民歌」退潮後的校園市場，而必須從新興的小眾場域（pub、社運活動、地方政府舉辦的音樂節活動等）出發，抵抗商業化與市場或的衝擊。1982年金韻獎停辦，海山唱片結束營業，預示了校園民歌運動沒落的命運。1990年代以後，以[[林強|林強]]、[[陳明章|陳明章]]、[[陳昇|陳昇]]等人為代表的台灣新歌謠曾一度在市場佔有一席之地。但未久即被高度商業化的日本偶像生產模式潮流所淹沒。

;民歌作者與演唱者的星散與精神傳承
基於前述政治、社會原因與個人的生涯規劃，校園民歌運動的主要人物、歌手畢業後出國深造、服兵役造成了人才斷層，音樂風格後繼無人。若干知名民歌手如[[侯德健|侯德健]]、齊豫、羅大佑等轉赴[[香港|香港]]與[[大陸|大陸]]發展，少數歌手與作者投入新的商業化音樂和綜藝事業，擔任主持、選秀評審（如[[黃韻玲|黃韻玲]]）的工作。多數人則完全終止創作和演出，轉入廣播界（如[[鄭怡|鄭怡]]、[[芝麻龍眼|芝麻龍眼]]中的芝麻[[陳艾玲|陳艾玲]]）、宗教媒體（如[[殷正洋|殷正洋]]），甚至移民海外而銷聲匿跡，台灣「校園民歌」運動的時代遂全面結束。時至今日，「校園民歌」成為台灣1960年代到1970年代初期出生人士懷舊的象徵，並留存於[[KTV|KTV]]與懷舊的音像製品市場中。

不同於台灣因環境氛圍改變，產生新的音樂潮流，在中國大陸和東南亞各國如[[新加坡|新加坡]]、[[馬來西亞|馬來西亞]]、[[印尼|印尼]]等華人聚居地，台灣的「校園民歌」仍受到廣泛歡迎。受到台灣校園民歌的影響，新加坡在1982年左右也興起創作「新謠」的風潮。<ref>{{cite news  |url = http://www.yan.sg/xinyaobaishenglou/|title = 新加坡民谣”新谣“：那些年，我们唱着的歌|work=新加坡眼|date = 2014-07-07  |accessdate = 2019-01-06 }}</ref>「校園民歌」與部份演唱者於1980年代中後加入中國大陸市場，對其華語歌曲的發展也留下深刻影響。如李泰祥作曲、女作家[[三毛(作家)|三毛]]寫詞、齊豫演唱的《橄欖樹》亦掀起了對[[台灣|台灣]]的「校園民歌熱」，同時也帶動香港「城市民歌」的興起，成為了[[粵語流行音樂|粵語流行音樂]]的一個重要支派。除此之外，[[中視|中視]]2009年八點檔連續劇[[光陰的故事|光陰的故事]]、[[閃亮的日子|閃亮的日子]]裡亦使用校園民歌作為配樂。

2015年適逢台灣民歌四十年，李建復擔任「民歌40」演唱會的召集人、導演馬宜中、「驚喜合唱」以民歌為主題選擇在見證台灣成長交通史的[[板橋車站_(臺灣)|板橋車站]]再次快閃演出，除了在現場令許多遊客叫好，同時在網路上也引起廣大迴響。影片在[[YouTube|YouTube]]網路影片平台播放點閱至今已達700萬人次。<ref>http://www.businessweekly.com.tw/article.aspx?id=11194&type=Blog&p=1 「民歌四十」快閃幕後》我們的十九歲與那條「日光大道」. 商業週刊</ref>

民歌是台灣當時文化界的知識份子和音樂前輩留下的寶貴的文化遺產。尤其質樸優美的台灣民歌，是時代背景的影子，反映出台灣人開拓生活天地的奮鬥史，它潛藏著豐富的民族情感和精神。它是民俗文學之瑰寶，保存民間文化的精華。是中華文化與台灣永恆的文化資產。台灣民歌其詞意貼切，旋律又優美動人，人人應予維護並加以發揚，我們應保持先民所傳下來良好的藝術文化，多從事思維和創造活動，使民間的藝術創造力重現生機；並珍惜傳統的藝術文化。

* '''懷古與歷史'''
龍的傳人（李建復演唱）、唐山子民（黃大城演唱）、中華之愛（施孝榮演唱）、古月照今晨（[[文章|文章]]演唱）、易水寒（施孝榮演唱）、古國今昔（王夢麟演唱）、俠客（施孝榮演唱）、我心似清泉（王海玲演唱）、送你一把泥土（費玉清演唱）、夢駝鈴（費玉清演唱）、變色的長城（費玉清演唱）、中華民國頌（費玉清演唱）、赤壁賦（施孝榮演唱，[[陳雲山|陳雲山]]詞）

* '''鄉愁'''
鄉愁四韻（余光中作品）、歸去來兮（侯德建作品，李建復演唱）、歸人沙城（施孝榮演唱）、出塞曲（[[蔡琴|蔡琴]]演唱）、蘆歌（李建復演唱）

* '''中國詩詞（含新詩）'''
釵頭鳳（包美聖演唱，宋[[陸游|陸游]]詞）、[[雨霖鈴|雨霖鈴]]（包美聖演唱，宋朝[[柳永|柳永]]詞）、秘摩崖月夜（包美聖演唱，[[胡適|胡適]]新詩）、雪花的快樂（王海玲演唱，[[徐志摩|徐志摩]]新詩）、再別康橋（范廣惠演唱，[[徐志摩|徐志摩]]新詩）、浮生千山路（潘越雲演唱，[[陳幸蕙|陳幸蕙]]集詩詞）、歌（羅大佑演唱，[[徐志摩|徐志摩]]譯自英國女詩人 Christina Georgina Rossetti的短詩）。

* '''文化經驗'''
廟會（王夢麟演唱）、月琴（鄭怡演唱）、今山古道（黃大城演唱）、鄉間小路（齊豫演唱）、晚霞滿漁船（胡立武演唱）、夏之旅（蔡幸娟演唱）   

* '''台灣常民生活'''
雨中即景（趙樹海、王夢麟、黃大城演唱）、酒干倘賣無（[[蘇芮|蘇芮]]演唱）、外婆的澎湖灣（潘安邦演唱）、鄉居記趣（葉佳修演唱）、正月調（[[簡上仁|簡上仁]]演唱）、拼宵夜（[[木吉他合唱團|木吉他合唱團]]）

* '''流浪、告別與回歸'''
橄欖樹（李泰祥作品，齊豫演唱）、浮雲遊子（陳明韶演唱）、曠野寄情（李建復演唱）、就要揮別（楊芳儀、徐曉菁演唱）、匆匆的走、露莎蘭（[[蘇格蘭|蘇格蘭]]民謠，[[楊芳儀|楊芳儀]]演唱）、告別憂傷（[[蔡藍欽|蔡藍欽]]演唱）、戲雪（張艾嘉演唱）、愛之旅（洪小喬演唱）

* '''遺忘'''
被遺忘的時光（蔡琴演唱，被用為香港電影[[無間道|無間道]]主題曲）、微風往事（鄭怡演唱）、忘川（李建復演唱）、讓我們看雲去（陳明韶演唱）

* '''童年童稚'''
往事（陳秋霞作品、[[陳秋霞|陳秋霞]]演唱）、捉泥鰍（侯德建作品、[[侯德建|侯德建]]演唱）、童年（羅大佑作品、[[張艾嘉|張艾嘉]]演唱）、老師斯卡也答（楊芳儀演唱）、光陰的故事（羅大佑作品、張艾嘉演唱）
* '''自然詠歎與懷想'''
蘭花草（銀霞演唱）、漁唱（黃大城演唱）、秋蟬（楊芳儀、徐曉菁演唱）、海裡來的沙（芝麻龍眼演唱） 、小貝殼（邰肇玫演唱）、小草（王夢麟演唱）、聽泉（楊芳儀、徐曉菁合唱）、守住這一片陽光（[[林佳蓉|林佳蓉]]、[[許淑絹|許淑絹]]演唱）、七月涼山、露莎蘭（[[蘇格蘭|蘇格蘭]]民謠，[[楊芳儀|楊芳儀]]、[[徐曉菁|徐曉菁]]演唱）、蝸牛與黃鹂鳥（銀霞演唱）、夕陽伴我歸（[[陳淑樺|陳淑樺]]演唱）、微光中的歌吟（鄭怡演唱）
* '''愛情'''
你那好冷的小手（銀霞演唱）、小秘密（銀霞演唱）、風告訴我（陳明韶演唱）、如果（施碧梧、邰肇玫演唱）、動不動就說愛我 （芝麻龍眼演唱）、神話（[[羅吉鎮|羅吉鎮]]、[[李碧華|李碧華]]演唱）、偶然（[[卓秀琴|卓秀琴]]演唱）、木棉道（王夢麟演唱）、拜訪春天（施孝榮演唱）、小茉莉（包美聖演唱）、忘了我是誰（王海玲演唱）、阿美阿美（王夢麟演唱）、離開你走近你（王夢麟演唱）、風中的早晨（[[王新蓮马宜中|王新蓮马宜中]]演唱）、生命中的陽光（[[木吉他合唱團|木吉他合唱團]]）、歡顏、走在雨中（齊豫演唱）、陽光和小雨（潘安邦演唱）、[[守著陽光守著你|守著陽光守著你]]、天天天藍（潘越雲演唱）、讀你、恰似你的溫柔（蔡琴演唱）、野百合也有春天（潘越雲演唱）、夢的衣裳（[[李碧華_(歌手)|李碧華]]演唱）、請擁抱我（[[鄭人文|鄭人文]]演唱）、第一支舞、我深愛過（[[周秉鈞|周秉鈞]]、[[楊海薇|楊海薇]]演唱）、[[夜玫瑰|夜玫瑰]]（[[以色列|以色列]]民謠，楊芳儀、徐曉菁演唱）、想你（芝麻龍眼演唱）、春風（王瑞瑜演唱）
