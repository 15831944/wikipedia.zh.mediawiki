'''枕崎站'''（{{jpn|j=枕崎駅|hg=まくらざきえき|rm=Makurazaki eki|f=y}}）是一座位於[[鹿兒島縣|鹿兒島縣]][[枕崎市|枕崎市]]的[[九州旅客鐵道|九州旅客鐵道]]（JR九州）鐵路車站，是[[地方交通線|地方交通線]][[指宿枕崎線|指宿枕崎線]]的終點站。此車站曾經是{{link-ja|鹿兒島交通|鹿児島交通}}[[枕崎線|枕崎線]]的終點站，但該線已在1984年時廢線。枕崎車站是[[JR|JR]]系統、乃至於日本本土（不包含[[沖繩島|沖繩島]]在內的四大主島）上最南端的鐵路車站，不過此處的最南端是指路線上位於下行方向最遠端之意，但在實際的地理[[緯度|緯度]]上，枕崎車站其實比同樣位在指宿枕崎線上的多座沿線車站緯度高。

== 車站構造 ==
*一個1面1線單式月台，不可以[[列車交會|列車交會]]。
*[[無人站|無人站]]。
*[[地上車站|地上車站]]。
*設有候車室。

在2006年5月1日，車站向鹿兒島中央車站方向遷移100米，新地方設有站舍。

== 历史 ==
此車站首先是由[[鹿兒島交通|南薩鐵道]]（現為鹿兒島交通）建設，此車站也管理鐵路公司其他事業。隨後，[[日本國有鐵道|日本國有鐵道]][[指宿枕崎線|指宿枕崎線]]開通，設置終點站在此。在[[鹿兒島交通枕崎線|枕崎線]]停運後，車站擁有權落在日本國有鐵道上。

* 1931年（昭和6年）3月10日：[[南薩鐵道|南薩鐵道]][[鹿兒島交通枕崎線|枕崎線]]加世田至枕崎開通，車站啟用。
* 1963年（昭和38年）10月31日：[[日本國有鐵道|日本國有鐵道]][[指宿枕崎線|指宿枕崎線]]全線開通，兩線共用車站。
* 1984年（昭和59年）3月17日：鹿兒島交通枕崎線（前南薩鐵道）停運，車站擁有權落在日本國有鐵道上。
* 1987年（昭和62年）4月1日：因[[國鐵分割民營化|國鐵分割民營化]]，本站成為[[九州旅客鐵道|九州旅客鐵道]]的車站。
* 2006年（平成18年）5月1日：車站遷移100米。
* 2009年（平成21年）12月26日：站前廣場設置觀光資訊。

== 使用狀況 ==
* 在2007年，平均1日的乘車人數為131人。
