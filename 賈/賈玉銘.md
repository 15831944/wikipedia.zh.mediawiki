'''贾玉铭'''（{{bd|1880年||1964年|4月12日|}}），[[山东|山东]][[昌乐县|昌乐]]人，是[[中国基督教新教|中国基督教新教]][[福音派|福音派]]的著名[[神学家|神学家]]、[[解经|解经]]家。

== 早年 ==
1880年（清[[光绪|光绪]]六年），贾玉铭出生在中国山东省[[昌乐县|昌乐县]]小岭村。1901年，毕业于[[狄考文|狄考文]]博士在[[登州府|登州府]]创办的[[长老会|长老会]][[文會館|文會館]]（[[齐鲁大學|齐鲁大學]]前身），由于[[庚子事变|庚子事变]]的影响，那一年文會館只畢業了兩名學生，而其中之一就是賈玉銘（德新） <ref>[[于中旻|于中旻]]：[http://www.aboutbible.net/Ab/E.58.ChiaYuming.html 贾玉铭]</ref>。然后他繼續進教士館（神學院），接受[[改革宗|改革宗]][[系统神学|系统神学]]的训练。贾玉铭跟[[丁立美|丁立美]]是长老会[[文会馆|文会馆]]（后来的[[齐鲁大学|齐鲁大学]]）的同学，后来也一起同工很多年。丁立美组织了“学生立志布道团”当第一任的巡行干事，后来1919年到“云南布道会”去。

== 牧师生涯 ==
1904年他从教士館畢業，被按立為牧師，先后在[[美北长老会|美北长老会]]在山东省设立的安邱縣逢旺支會、濟宁州教会和沂州府教会担任牧師，历时12年，为一千多人施洗。<ref>{{Cite web|url=http://www.bdcconline.net/zh-hant/stories/by-person/j/jia-yuming.php|title=賈玉銘|accessdate=2018-02-08|author=|date=|work=www.bdcconline.net|publisher=|language=zh-Hant|archive-url=https://web.archive.org/web/20171120223213/http://www.bdcconline.net/zh-hant/stories/by-person/j/jia-yuming.php|archive-date=2017-11-20|dead-url=yes}}</ref>

== 神学教育 ==
1915年，35岁的贾玉铭前往南京[[金陵神学院|金陵神学院]]执教，从此开始了大半生的神学教育生涯。1920年代，贾玉铭还在南京创办《灵光报》双月刊。

1919年，[[齐鲁大学|齐鲁大学]]神学院院长[[赫士|赫士]]博士（Watson M. Hayes，1857~1944）因坚持保守信仰而辞职离开齐鲁，在潍县另外创办[[华北神学院|华北神学院]]，并於1922年遷往滕縣。賈玉銘牧師被聘為副院長。1930年，贾玉铭出任[[金陵女子神学院|金陵女子神学院]]院长。

1936年10月，贾玉铭和金陵女子神学院教员廖恩荣、[[毕詠琴|毕詠琴]]在南京鼓楼厚载巷53号另外创立了[[中国基督教灵修学院|中国基督教灵修学院]]，旨在培养“仰望主；不募捐，不借債，過信心的生活”的傳道人。不久抗战爆发，该院西迁[[重庆南山|重庆南山]]，战后迁回南京[[山西路_(南京)|山西路]]81号。

1948年，賈玉銘應邀參加在荷蘭舉行的世界福音會議，並當選為副主席。回国后，1949年灵修学院在上海[[巨鹿路|巨鹿路]]367号复院，称上海灵修学院。

1954年，[[中国基督教三自爱国运动委员会|中国基督教三自爱国运动委员会]]成立，年过七十的福音派领袖賈玉銘被安排為七位第二届副主席之一。1956年，上海灵修学院和中国基督教传道人修养院、怀恩业余神学院、上海圣经学社合并成立中国基督教上海灵修神学院，校址设巨鹿路，贾玉铭仍任院长 <ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node75195/node75204/node75305/node75319/userobject1ai92065.html 上海宗教志 >> 第五编基督教 >> 第三章教育和出版 >> 第一节　神学院校] {{webarchive|url=https://web.archive.org/web/20150402160509/http://www.shtong.gov.cn/newsite/node2/node2245/node75195/node75204/node75305/node75319/userobject1ai92065.html |date=2015-04-02 }}</ref>。1957年[[反右运动|反右运动]]期间，賈玉銘受到一定的批判和衝擊。1959年，上海灵修神学院停办。1964年4月12日，贾玉铭在上海去世，享年84歲。

== 著作 ==
《神道学》、《圣经要义》、《完全救法》、《教牧學》、《宣道法》、《靈修日課》，已及聖經66卷專卷釋義
