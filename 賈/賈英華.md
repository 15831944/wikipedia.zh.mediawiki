'''賈英華'''（{{bd|1952年||}}），[[中國|中國]][[作家|作家]]，晚清研究學者，[[中國作家協會|中國作家協會]]全國委員會委員、[[中國傳記文學學會|中國傳記文學學會]]副會長、[[中國電力作家協會|中國電力作家協會]]主席。代表作為《[[末代皇帝的後半生|末代皇帝的後半生]]》，{{Fact|此書日文版是風糜日本的暢銷書。中、英、美、法、日及港、澳、台等上百家國內外權威媒體評價其成功地續補了溥儀的《[[我的前半生|我的前半生]]》|time=2013-02-13T21:09:23+00:00}}。此書中文版暢銷再版九次，並獲中國圖書第五屆[[金鑰匙獎|金鑰匙獎]]。他曾捉刀為末代皇帝溥儀題寫墓誌，為末代太監孫耀庭撰題碑文。其書法作品曾入選《中國首屆名人名作展》、《當代名家書畫展》等。<ref>{{cite web |url=http://www.17k.com/zhuanti/20111229/|title=《賈英華末代皇族系列作品專題》|accessdate=2011-11-29 |work=|publisher=17K小說網|language=zh-hans}}</ref><ref>{{cite web|url=http://www.xinwenren.com/zuojia/201111142617.html|title=《賈英華四部末代皇族系列新作出版》|accessdate=2011-11-14|work=|publisher=中國新聞人網|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20111115195605/http://www.xinwenren.com/zuojia/201111142617.html|archivedate=2011-11-15}}</ref>

賈英華末代皇族系列作品之一《[[末代皇帝的非常人生|末代皇帝的非常人生]]》獲得20113年第四屆中國傳記文學優秀作品獎。。<ref>{{cite web|url=http://news.xinhuanet.com/politics/2013-03/27/c_115184575.htm|title=《第四屆中國傳記文學優秀作品獎揭曉》|accessdate=2013-03-27|work=|publisher=新華網|language=zh-hans}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}}}</ref>

== 工作經歷 ==
曾任北京熱電廠工人、班長、團委書記、總廠團委書記，華北電管局電力報華北記者站負責人，北京熱電總廠團委書記，國務院秘書局副處長、處長、副局長，國家經貿委電力司副司長、國家電監會供電部主任。<br />
中國作協第六、七、八屆全國委員會委員。<ref>{{cite web |url=http://www.chinawriter.com.cn/union/content/117.shtml |title=《中國作協》|accessdate=2012-09-09 |work=|publisher=中國作家網|language=zh-hans}}</ref>
