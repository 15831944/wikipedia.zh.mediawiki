'''陶峙岳'''（{{bd|1892年|9月18日|1988年|12月26日|catIdx=T}}），原名'''陶纪常'''，字'''岷毓'''，曾用名'''陶锄'''，[[湖南|湖南]][[寧鄉縣|寧鄉縣]]人，[[國民革命軍|國民革命軍]]中將，[[中國人民解放軍上將|中國人民解放軍上將]]。<ref name="gf" >{{cite web |url=http://cpc.people.com.cn/GB/64162/126778/127857/7538793.html |title=陶峙岳 |accessdate=2015-07-16 |publisher=中国共产党新闻网}}</ref>

1911年参加[[武昌起义|武昌起义]]后加入[[同盟会|同盟会]]。1916年毕业于[[保定陆军军官学校|保定陆军军官学校]]，后任湖南督军署参谋处上尉参谋。曾任湘军营长、团长。1926年参加[[北伐战争|北伐战争]]，任国民革命军独立第二师团长，第四十军三师副师长、师长，第八师旅长、师长。1937年任国军第七十七军军长。参加[[淞沪抗战|淞沪抗战]]，后任国军第一军军长，第三十四集团军副总司令，第三十七集团军副总司令、总司令，河西警备司令部总司令，西北军政长官公署副长官，新疆警备总司令部总司令。1949年9月，率部在新疆倒戈。后编入[[中国人民解放军|中国人民解放军]]，任[[新疆军区|新疆军区]]副司令员兼第二十二兵团司令员，西北军政委员会委员，新疆军区副司令员兼新疆生产建设兵团司令员。1982年加入[[中国共产党|中国共产党]]。1983年当选为第六届[[全国政协副主席|全国政协副主席]]。

==生平==
=== 早年经历 ===
1904年考入长沙城南师范附属小学。1906年毕业后考入求忠中学。1907年进入陆军小学。1911年毕业后赴武汉考入陆军第三中学，参加[[武昌起义|武昌起义]]<ref name="欧阳青">{{cite book |author=欧阳青|title=大授衔：1955共和国将帅授衔档案|year=2011 |publisher=长城出版社 |location=北京 |pages=239|isbn=978-7-5483-0058-8}}</ref>。1912年加入[[同盟會|同盟會]]。1914年秋，陆军第一预备学校毕业后到北京北苑陆军第八师受入伍训练，充任军官递补生。1915年春转入[[保定陆军军官学校|保定陆军军官学校]]。1916年秋毕业后回到长沙，任湖南省督军公署参谋处上尉参谋。1917年任[[谭延闿|谭延闿]]部“护字营”营参谋长兼第一支队教导员，不久升任营长。1920年，任[[湘军|湘军]]第六混成旅后旅独立工兵营营长。1923年任湘军第一师[[贺耀祖|贺耀祖]]部上校团长<ref name="gf"/>。

1926年，北伐军击败[[吴佩孚|吴佩孚]]，攻克[[长沙|长沙]]，贺耀祖率部接受改编，陶峙岳任国民革命军独立第二师第三旅团长。在[[北伐战争|北伐战争]]中，因功晋级为少将团长、第三师少将副师长。1927年3月参加会攻南京，任国民革命军第40军第3师副师长。4月代理南京卫戍司令，并经[[贺耀祖|贺耀祖]]介绍加入[[中国国民党|中国国民党]]。12月任第3师师长，参加了复克[[徐州|徐州]]之役。1928年，任第40军第3师中将师长，参加[[蒋桂战争|蒋桂战争]]<ref name="gf"/>。

1930年，陶峙岳任第八师少将副师长兼第二十四旅旅长，参加中原大战。1931年7月陶峙岳参加[[中央苏区第三次反围剿战争|对中央苏区第三次围剿作战]]，黄陂之役中，陶峙岳旅大败<ref>{{cite book|title=民国军事史 第二卷|author=姜克夫|publisher=重庆出版社|year=2009|pages=232|isbn=9787229008390}}</ref>。1933年，陶峙岳任第八师师长。

=== 抗日战争与第二次国共内战时期 ===
抗日战争爆发后，出任第76军军长兼第8师师长。1937年9月，率部由陕西经河南驰援上海，参加[[淞沪会战|淞沪会战]]，率官兵坚守阵地22天。1938年9月，陶峙岳任[[第一战区|第一战区]][[国民革命军第一军|第1军]]军长<ref>{{cite book|title=国民党历史上的158个军|author=张明金、刘立勤|year=2007|publisher=解放军出版社|isbn=9787506553872|pages=25}}</ref>。1940年夏任第34集团军副总司令，次年4月任第一战区政治部主任兼陕西民众动员指挥部参谋长。1942年8月任第37集团军总司令，次年秋调任河西警备总司令<ref name="gf"/>。

1946年，陶峙岳应西北行营主任[[张治中|张治中]]之请，任新疆警备总司令、河西警备总司令、[[西北军政长官公署|西北军政长官公署]]副长官。同年夏，协助张治中释放关押在新疆监狱的一大批共产党员，并亲自派人将他们护送到延安<ref name="gf"/>。1949年春曾采取拖延战术，避免了驻疆部队被调入关内，并与新疆省主席[[包尔汉|包尔汉]]等人秘密进行倒戈的准备。9月24日晚12点，率驻酒泉国军官兵通电起义，是谓[[酒泉起义|酒泉起义]]。25日，率新疆的7万余国军官兵通电接受中国共产党领导<ref name="传">{{cite book|author=《彭德怀传》编写组|title=彭德怀传|date=2006-12|publisher=当代中国出版社|isbn=978-7-80092-103-2|pages=225}}</ref><ref name="現代史">{{cite book|author=李守孔|title=中國現代史|publisher=[[三民書局|三民書局]]|location=台北|year=1973|isbn=9571406635|pages=175}}</ref>，是谓[[新疆和平解放|新疆和平解放]]。尔后积极进行宣慰工作，迅速平息骚乱事件，坚决镇压破坏活动。10月中旬，发布《告全疆将士书》，宣传中国共产党的方针政策。

=== 中华人民共和国成立后 ===
[[中华人民共和国|中华人民共和国]]成立后，任[[新疆军区|新疆军区]]副司令员，[[西北军政委员会|西北军政委员会]]委员，中国人民解放军第22兵团司令员<ref>{{cite book |author = 张明金、刘立勤 |title = 中国人民解放军历史上的200个军区 |year = 2010 |publisher = 解放军文艺出版社 |pages = 514 |isbn = 978-7-5033-2252-5}}</ref>。1954年，任[[新疆生产建设兵团|新疆生产建设兵团]]司令员。1955年，担任新疆军区副司令员<ref>{{cite book|author=《中国人民解放军军史》编写组|title=中国人民解放军军史：第五卷|year=2011|publisher=军事科学出版社|pages=22|isbn=978-7-80237-427-0}}</ref>，被授予上将军衔和一级[[解放勋章|解放勋章]]。他领导部队屯垦戈壁，挥汗荒原，创造出在戈壁滩上种植棉花、兴修水利、植树造林、建造城市等许多奇迹，使新疆逐渐走出落后贫困的古道，成为国内外闻名的农业示范区。1979年任[[湖南省人民代表大会常务委员会|湖南省人民代表大会常务委员会]]副主任<ref name="gf"/>。1965年申請加入[[中國共產黨|中國共產黨]]，但直到1982年才成為中共黨員。

曾任[[全國人民代表大會|全國人民代表大會]]第1至5屆代表，第4、5屆[[全國人民代表大會常務委員會|全國人民代表大會常務委員會]]委員，[[中國人民政治協商會議|中國人民政治協商會議]]第1、2、3、4、5、6屆委員，第6屆[[全国政协副主席|全国政协副主席]]。1988年被授予[[胜利功勋荣誉章|胜利功勋荣誉章]]。1988年12月26日病逝于[[長沙|長沙]]<ref>{{cite book |author = 星火燎原编辑部 |title = 中国人民解放军将帅名录：第一卷 |year = 2006 |publisher = 解放军出版社 |pages = 137 |isbn = 7506550555}}</ref>。
