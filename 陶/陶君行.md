'''陶君行'''（'''To Kwan Hang Andrew'''，{{bd|1966年|2月7日|catIdx=T陶}}, 前[[社民連|社民連]]主席；前[[香港|香港]][[黃大仙區|黃大仙區]][[香港區議會|區議員]]，曾任[[香港民主同盟|香港民主同盟]]（[[民主黨_(香港)|民主黨]]的前身）中央委員及「[[前綫_(1996年)|前綫]]」秘書長，1989年始擔任[[香港市民支援愛國民主運動聯合會|香港市民支援愛國民主運動聯合會]]（支聯會）常委，2005年籌組[[領匯監察|領匯監察]]，並獲選為主席，有美國會計師資格。曾為網絡電台[[香港人民廣播電台|香港人民廣播電台]]節目風波裡的龍門陣及波政不分、網絡電台[[MyRadio|MyRadio]]譚志強頻道及足球頻道節目主持。

現為[[花生台|花生台]]擁有者，大君伐及怒抄花生節目主持。其妻子為[[民間人權陣線|民間人權陣線]]前任召集人、香港天主教正義和平委員會幹事[[孔令瑜|孔令瑜]]。

==簡介==
===早期背景===
陶君行是客家人，在[[彩虹村|彩虹村]]長大，小學至預科時就讀於[[喇沙小學|喇沙小學]]及[[喇沙書院|喇沙書院]]，年少時熱愛體育運動，代表學校參與學界田徑、越野賽、籃球及游泳等項目之比賽，並獲取校內甲組最佳運動員的獎項。陶君行講述讀書時期醉心體育，後來接觸到社運便全心投入民主抗爭運動，進入大專後，他熱忱於學生運動，曾擔任[[嶺南大學_(香港)|嶺南學院]]學生會（現[[嶺南大學學生會|嶺南大學學生會]]）會長，及至於1989年出任[[香港專上學生聯會|香港專上學生聯會]]秘書長。1989年4月到北京支援民運，[[六四事件|六四事件]]後冒起，其後成為[[香港民主同盟|香港民主同盟]]為創會會員，踏足政界，其後參加區議會選舉，於[[1991年香港區議會選舉|1991年香港區議會選舉]]參選黃大仙區議會竹園選區當選，成為當時全香港最年輕之區議員。

===退出民主黨及前綫===
陶君行1998年與民主黨內年青議員[[鄭家富|鄭家富]]（[[香港立法會|立法會]]議員）、[[陳國樑|陳國樑]]（[[市政局_(香港)|市政局]]議員）、[[黃仲棋|黃仲棋]]（市政局議員）、[[范國威|范國威]]（西貢區議會議員）等，組成「少壯派」推動黨內改革，提倡黨內民主、採取基層勞工路線、結合議會與街頭運動的策略，與「主流派」及民主黨內的前「匯點」成員發生激烈罵戰 <ref> {{cite web |title = 陶君行出書翻民主黨舊帳 |url = http://paper.wenweipo.com/2010/07/23/HK1007230026.htm}}</ref>。陶君行繼而在2002年退出民主黨，先組成「社會民主論壇」，後加入「前綫」。2004年，陶君行由於反對領匯上市，被黨友排擠，後來宣布淡出前綫<ref> {{cite web |title = 指劉細良策劃 推張炳良下台 陶君行爆與民主黨恩怨
|url = http://news.singtao.ca/toronto/2010-07-23/hongkong1279871009d2622450.html}}</ref>。

===社民連時代===
2006年，與 [[黃毓民|黃毓民]]、「大舊」[[陳偉業|陳偉業]]、「長毛」[[梁國雄|梁國雄]]等，組合前「少壯派」成員，再加上一些前衛社運人士如[[李偉儀|李偉儀]]等，組成[[社會民主連線|社會民主連線]]，並出任秘書長，後獲選為副主席（外務），與會內另一位副主席（內務）[[麥國風|麥國風]]同為會內的第二把手。該會標榜為「旗幟鮮明的反對派」。

2010年，陶君行當選[[社會民主連線|社會民主連線]]主席，接替[[黃毓民|黃毓民]]，成為社民連主席。

==參與選舉==
===立法會選舉===
2000年曾與[[司徒華|司徒華]]、[[李華明|李華明]]、[[胡志偉|胡志偉]]三人合組一條名單，參選九龍東地區直選，唯名單中只有司徒華、李華明二人當選。<ref>
[http://www.elections.gov.hk/elections/legco2000/index_c.htm 2000年香港立法會選舉] {{webarchive|url=https://web.archive.org/web/20081204101359/http://www.elections.gov.hk/elections/legco2000/index_c.htm |date=2008-12-04 }}</ref>

2004年，原定代表「前綫」在九龍東選區參選立法會，但[[鄭經翰|鄭經翰]]被商業電台突然解僱後在九龍東參選，並與陶君行組成一張名單，雖然選前民意調查顯示陶君行有望代表民主派奪得議席，但選舉結果名單得73,479票，鄭經翰成為「票王」當選，但票數不足以令名單中排第二名的陶君行入選。<ref>[http://www.elections.gov.hk/elections/legco2004/tc_chi/results/rs_gc_kln_e.html 2004年香港立法會九龍東選舉結果] {{webarchive|url=https://web.archive.org/web/20081210162408/http://www.elections.gov.hk/elections/legco2004/tc_chi/results/rs_gc_kln_e.html |date=2008-12-10 }}</ref>

2008年7月28日陶君行獨自參選，代表社會民主連線參與九龍東地區直選，以「我肯搏」為競選口號。他在該選舉的九龍東選區中得到28,690票，僅排在公民黨[[梁家傑|梁家傑]]之後，位列第五名。<ref>[http://www.singtao.ca/cal/2008-09-09/1220953501d1277789.html 「癲狗」入局 社民連擁三席，星島日報，2008年9月9日]{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref><ref>[http://www.elections.gov.hk/legco2008/chi/result/rs_gc_LC3.html 2008年香港立法會九龍東選舉結果] {{webarchive|url=https://web.archive.org/web/20080909061707/http://www.elections.gov.hk/legco2008/chi/result/rs_gc_LC3.html |date=2008-09-09 }}</ref>

[[2012年立法會選舉|2012年立法會選舉]]，陶君行單獨組成名單，代表社會民主連線參與九龍東地區直選。根據[[2012年香港政治制度改革|2012年香港政治制度改革]]中通過的立法會選舉辦法，九龍東地區的議席從4席增至5席。但同區參選的還有同樣代表激進泛民勢力的[[人民力量|人民力量]]的[[黃洋達|黃洋達]]名單，分別獲得9.57%和12.85%的選票，均未能當選。陶黃兩張名單在合計獲得總選票22.42%的情況下，拱手將九龍東地區的第5席讓給配票精確的建制派獨立參選人[[謝偉俊|謝偉俊]]（得票率13.54%）。

===區議會選舉===
陶君行於1991年-2011年出任[[黃大仙區|黃大仙區]]議員（[[竹園北邨|竹園北]]）。

2007年10月，[[梁安琪_(香港)|梁安琪]]在香港電台舉辦的區議會選舉論壇與對手'''陶君行'''辯論。其間，梁安琪多次發表出位言論，引起社會各界嘩然。<ref>[http://www.youtube.com/watch?v=ZM7qsByjwf0 區選論壇 - 陶君行舌戰梁安琪]</ref>

由於她在選舉論壇的舉措，有網民在網上討論區討論關於她的一切，她在落敗後指網民因為她在選舉論壇後，在網上不停地抹黑她，認為會認響她的票源。這次選舉中她以1851比2825輸給陶君行，相差20.8%。<ref>[http://www.elections.gov.hk/dc2007/chi/results_wong_tai_sin.html 2007年黃大仙區議會選舉結果] {{webarchive|url=https://web.archive.org/web/20090306140353/http://www.elections.gov.hk/dc2007/chi/results_wong_tai_sin.html |date=2009-03-06 }}</ref>

2011年11月7日，陶君行於[[2011年香港區議會選舉|2011年香港區議會選舉]][[黃大仙|黃大仙]]竹園北區選落敗，只得1660票，大敗於獨立人士[[丁志威|丁志威]]的2754票<ref>[http://hk.news.yahoo.com/%E7%A4%BE%E6%B0%91%E9%80%A3%E4%B8%BB%E5%B8%AD%E9%99%B6%E5%90%9B%E8%A1%8C%E8%90%BD%E9%81%B8-232246937.html 社民連主席陶君行落選]</ref>。於同日下午辭任[[社民連|社民連]]主席一職<ref>[http://hk.news.yahoo.com/%E9%99%B6%E5%90%9B%E8%A1%8C%E8%BE%AD%E7%A4%BE%E6%B0%91%E9%80%A3%E4%B8%BB%E5%B8%AD-084207139.html 陶君行辭社民連主席]</ref>。

==官司==
===被控襲警===
2007年12月29日，[[梁國雄|梁國雄]]及陶君行趁訪港的全國人大常委會副秘書長[[喬曉陽|喬曉陽]]到中區[[禮賓府|禮賓府]]演講時，到禮賓府抗議，在禮賓府東閘外焚燒報紙及車胎，期間兩名警員上前試圖撲滅火種時，梁國雄及陶君行衝前撞向兩名警員，令他們推倒落地受傷。梁、陶二人其後被警方控以襲警罪名。

2008年10月30日，案件在[[東區裁判法院|東區裁判法院]]開審。案情指當時[[梁國雄|梁國雄]]及陶君行焚燒報紙及車胎，被港島[[警察機動部隊|警察機動部隊]]姓李[[警員_(香港)|警員]]及姓關[[警長|警長]]上前阻止。李開動滅火器時，陶君行突然撞擊他，梁國雄同時衝向李及用手推李，令他跌倒。關扶起李時，被陶君行用雙手拉衫袖而倒地。<ref>[http://appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20081031&sec_id=4104&subsec_id=12740&art_id=11787316  《梁國雄、陶君行涉襲警開審》] {{webarchive|url=https://web.archive.org/web/20081103131141/http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20081031&sec_id=4104&subsec_id=12740&art_id=11787316 |date=2008-11-03 }}，香港《蘋果日報》，2008年10月31日。</ref>

同年12月11日，梁、陶二人各被裁定一項襲警罪名成立，押後至2009年1月16日判刑，陶的另一項襲警罪則獲判無罪。<ref>[http://www.takungpao.com/news/08/12/12/EP-1003498.htm 《梁國雄陶君行襲警罪成》] {{webarchive|url=https://web.archive.org/web/20081214202612/http://www.takungpao.com/news/08/12/12/EP-1003498.htm |date=2008-12-14 }}，香港《大公報》，2008年12月12日。</ref>

2009年1月16日，該案件在東區裁判法院判刑。裁判官指斥梁、陶二人不應濫用人權衝擊警方，即使他們不認同現場警員，也不能推撞警員，考慮到警員傷勢輕微，可以用社會服務令代替監禁，於是分別判決陶君行及梁國雄100及120小時[[社會服務令|社會服務令]]，裁判官強調判刑並沒有考慮兩人的地位才輕判。<ref>[http://www.takungpao.com/news/09/01/17/images_0702-1019971.htm 《梁國雄陶君行襲警被判社服》] {{webarchive|url=https://web.archive.org/web/20090118150157/http://takungpao.com/news/09/01/17/images_0702-1019971.htm |date=2009-01-18 }}，香港《大公報》，2009年1月17日。</ref>梁國雄表明會上訴，認為裁判官的判辭，並無解釋示威者表達意見的權利是否合憲。

2009年10月1日示威其間被指襲警。他本人否認，並表示他被警察襲擊。從片段看到警察有襲擊他，最終獲判無罪。<ref>[http://www.youtube.com/watch?v=IxVd4Evki2E 直擊陶君行被指襲警經過]</ref><ref>[http://hk.news.yahoo.com/article/100604/18/ieul.html 陶君行被控襲警罪名不成立]</ref>

===撞車傷人後離去===
陶君行於2017年12月在觀塘倒車時涉嫌撞倒後方另一輛私家車，以及未待警方到場便離去。陶君行今被判不小心駕駛及發生意外後沒有報案兩罪罪名成立，合共罰款3500元，一星期內繳交。
