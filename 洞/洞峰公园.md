'''洞峰公园'''（{{lang-ja|洞峰公園}}）是一个坐落于[[日本|日本]][[茨城县|茨城县]][[筑波市|筑波市]]{{link-ja|二宫|二の宮 (つくば市)}}的[[公园|城市公园]]，于1980年开园<ref>{{Cite web |url=http://www.koen.pref.ibaraki.jp/contents3.html |script-title=ja:茨城県県営都市公園一覧 |language=ja |publisher=[[茨城县|茨城县]] |access-date=2019-02-22 |archive-url=https://web.archive.org/web/20181005014841/http://www.koen.pref.ibaraki.jp/contents3.html |archive-date=2018-10-05 |dead-url=no }}</ref>。

== 历史 ==
在洞峰公园开园之前，由大高建筑设计事务所和都市再生机构茨城地域支社共同设计建设的新都市纪念馆就已于1976年在规划的园区内竣工<ref name="npo"/>。1980年，洞峰公园正式开园<ref name="茨新"/><ref name="茨土"/>。1999年，洞峰公园进行了一次大规模重修工程<ref name="茨土"/>。2005年8月，在洞峰公园内进行环境美化活动的洞峰爱之会（{{lang-ja|洞峰をあいする会}}）获得了茨城县内首个公园支援者的认证<ref>{{Cite web |author=茨城県土木部都市局公園街路課 |url=http://www.koen.pref.ibaraki.jp/supporter/doho20050819.html |script-title=ja:洞峰をあいする会 |publisher=茨城県営都市公園 |date=2005-08-19 |access-date=2019-02-22 |language=ja |archive-url=https://web.archive.org/web/20181005014923/http://www.koen.pref.ibaraki.jp/supporter/doho20050819.html |archive-date=2018-10-05 |dead-url=no }}</ref>。2010年3月，洞峰公园内的游泳池重新对外开放<ref>{{Cite web |url=http://www.joyoliving.co.jp/topics/201002/1002052.htm |script-title=ja:洞峰公園プール リニューアルオープン |publisher={{lang|ja|常陽リビング}} |date=2010-03-01 |language=ja |access-date=2019-02-22 |archive-url=https://web.archive.org/web/20181004183303/http://www.joyoliving.co.jp/topics/201002/1002052.htm |archive-date=2018-10-04 |dead-url=no }}</ref>。

== 自然 ==
洞峰公园内目前栖息的动物有[[红鼹鼠|红鼹鼠]]、[[蟋蟀|蟋蟀]]、[[蝉|蝉]]、[[蜻蜓|蜻蜓]]、[[蝗虫|蝗虫]]等<ref>{{Cite web |publisher={{lang|ja|霞ヶ浦市民協会}} |year=2008 |url=http://www.kasumigaura.com/0810.html |title=TOPICS0810 |language=ja |access-date=2019-02-22 }}{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。除此之外，园内种有[[银杏|银杏]]、[[樟树|樟树]]、[[榉树|榉树]]、[[樱树|樱树]]和[[松树|松树]]等树木<ref name="インスト">{{Cite web |publisher={{lang|ja|森林インストラクター茨城}} |url=http://www.forest-ibaraki.org/topics20090628.php |title=トピックス：森林インストラクター茨城/巨木ってどんな木？？モグラの穴 のぞいたことがありますか？皆で 確かめてみました！！！ |date=2010-06-03 |language=ja |access-date=2019-02-22 |archive-url=https://web.archive.org/web/20181005033445/http://www.forest-ibaraki.org/topics20090628.php |archive-date=2018-10-05 |dead-url=no }}</ref><ref name="茨土"/>。

虽然洞峰公园目前的动植物数量远不如从前的情况，但现在洞峰公园仍被当地人们当作观察自然的好场所<ref>{{Cite web |publisher=筑波市立小野川小学校 |url=http://www.tsukuba.ed.jp/~s-onogawa-e/modules/wordpress/index.php?p=1521 |script-title=ja:小野川小学校ニュース：ありがとうございました！ |date=2010-09-14 |language=ja |access-date=2019-02-22 |archive-url=https://web.archive.org/web/20160304191134/http://www.tsukuba.ed.jp/~s-onogawa-e/modules/wordpress/index.php?p=1521 |archive-date=2016-03-04 |dead-url=yes }}</ref><ref name="茨土"/>。

== 设施 ==
洞峰公园内设有50米[[游泳池|游泳池]]、[[体育馆|体育馆]]、[[网球场|网球场]]、[[棒球场|棒球场]]和草场等运动设施<ref name="茨新"/>。其中，公园使用[[太阳能|太阳能]]从而对游泳池的水进行加热<ref name="茨土"/><ref>{{Cite web |url=http://www.ibaraki-portal.jp/shops/000138/ |script-title=ja:茨城ポータル（イバラキポータル） |publisher=洞峰公園 |date=2010-11-28 |language=ja |access-date=2019-02-22 }}{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。除运动设施外，洞峰公园内还建有一座由大高建筑设计事务所和都市再生机构茨城地域支社共同设计建设的新都市纪念馆，内设有展示厅和咖啡馆<ref name="茨新"/><ref name="npo">{{Cite journal |title={{lang|ja|NPO法人つくば建築研究会}} |year=2005 |language=ja |pages=85 }}</ref>。
