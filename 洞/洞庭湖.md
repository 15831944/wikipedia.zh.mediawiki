'''洞庭湖'''位於[[中華民國|中華人民共和國]][[湖南省|湖南省]]北部，[[长江|长江]][[荆江|荆江]]河段以南，是[[中华民国|中华人民共和国]]第三大[[湖|湖]]，僅次於[[青海湖|青海湖]]、[[鄱陽湖|鄱陽湖]]，也是中国第二大[[淡水湖|淡水湖]]，和[[中国五大淡水湖|中国五大淡水湖]]之一，面积3,968平方公里（1998年）。洞庭湖南纳[[湘江|湘]]、[[资江|资]]、[[沅水|沅]]、[[澧水|澧]]四水汇入，北与长江相连，通过松滋、太平、[[藕池镇|藕池]]，[[调关镇|调弦]]（1958年已封堵）“四口”吞纳长江[[洪水|洪水]]，湖水由东面的[[城陵矶|城陵矶]]附近注入长江。[[唐代|唐代]][[诗人|诗人]][[刘禹锡|刘禹锡]]曾经用诗歌《望洞庭》描写了洞庭湖的平静和美丽。

==与长江关系==
洞庭湖是长江的过水湖，长江经松滋、太平、藕池三口分流入洞庭湖，经城陵矶出湖。汛期长江宜昌流量的一半从湖里过。以前的问题是长江泥沙在湖区沉积，和人为围垦一起淤积洞庭湖库容，逐渐削弱其调蓄洪水的能力。三峡大坝蓄水后，长江清水下泄，荆江河道下切，造成枯水期分流入洞庭三口的水量锐减，除松滋口外的其他两口都大概率断流，严重影响洞庭湖的水量。

这需要全面疏浚挖深洞庭湖的江水入湖分流河网，以保证洞庭湖沿线枯水季节的供水。

== 洞庭湖水系 ==
洞庭湖水系指最终汇入洞庭湖的各水网系统，包括[[洞庭湖平原|洞庭湖平原]]各河网水道和[[湘江|湘江]]、[[资江|资江]]、[[澧水|澧水]]和[[沅水|沅江]]等[[长江|长江]]一级支流。

洞庭湖流域即洞庭湖水系流经的广大地区，流域多年平均年降水量1,427毫米，多年平均年[[径流量|径流量]]为2,016亿立方米，约占[[长江流域|长江流域]][[地表水|地表水]]资源的21%，其比重为长江流域各[[水系|水系]]之首。洞庭湖流域覆盖[[湖南省|湖南省]]大部和[[湖北省|湖北省]]、[[广西壮族自治区|广西壮族自治区]]、[[贵州省|贵州省]]和[[重庆市|重庆市]]部分地区，流域面积262,800余平方公里，约占长江流域面积的14%。因[[湘江|湘]]、[[资江|资]]、[[沅水|沅]]、[[澧水|澧]]四水常遇较大暴雨，[[洪峰|洪峰]]流量常超过河道安全泄量，造成[[洪水|洪水]][[灾害|灾害]]。[[洞庭湖平原|洞庭湖平原]]区为湖南农业主产区，以种植[[粮食|粮食]]、[[棉花|棉花]]为主，也是中国主要淡水养殖区之一，耕地面积约占湖南全省的六分之一，居住人口约占湖南全省九分之一。

== 环境问题 ==
{{main|洞庭湖草原}}

[[2010年|2010年]]春，洞庭湖因[[乾旱|乾旱]]，水域面積銳減，由历年同期的1,649[[平方公里|平方公里]]，減至382平方公里，許多地方變成[[草原|草原]]。当时专家指出，如果6月[[汛期|汛期]]来临较慢，雨量不大水位上涨缓慢，会给湖区[[田鼠|田鼠]]提供充足由洲滩向内垸迁移的时间，那么6月底到7月还可能爆发[[鼠害|鼠害]]。

由于泥沙淤塞、[[围湖造田|围湖造田]]，洞庭湖现已分割为[[东洞庭湖|东洞庭湖]]、[[西洞庭湖|西洞庭湖]]、[[南洞庭湖|南洞庭湖]]、[[大通湖|大通湖]]、[[目平湖|目平湖]]和[[七里湖|七里湖]]等部分。洞庭湖的面积已经大大缩小，以至于比鄱阳湖还小。人们意识到这个问题后，退耕还湖，现在洞庭湖的面积逐渐增大，但是还没有超过鄱阳湖。

== 公共卫生 ==
[[File:1965-4_1965年_洞庭湖.jpg|thumb]]
洞庭湖是湖南省内[[血吸虫病|血吸虫病]]疫情最集中的地区。<ref>[http://news.xinhuanet.com/newscenter/2004-06/07/content_1512034.htm 《经济半小时》记者调查：血吸虫威逼长江]，新华网</ref>附近的岳阳是[[血吸虫|血吸虫]]重点疫区，且近年血吸虫病发病量呈上升趋势。分析认为发病量上升与[[牛|牛]][[羊|羊]]的洲滩敞放（虫卵可粪便传播），以及查（灭）[[钉螺|钉螺]]防疫经费不足有关。<ref>[http://www.people.com.cn/GB/34948/34951/34956/2625849.html “血吸虫病防治”专题调研综述之一 “瘟神”缘何又肆虐？] 人民网—中国政协新闻</ref>

血吸虫尾蚴10秒即可进入人体，为了避免染上血吸虫病，居民应该避免直接接触洞庭湖水。农民赤脚进入洞庭湖疫区劳作亦为染病的高危举动。<ref>[http://www.yyr.cn/News/2013-6/24/290511208.htm 疫区下湖戏水 当心染上血吸虫病！_岳阳新闻_岳阳人网]</ref>

== 历史变迁 ==

历史上洞庭湖曾是中国第一大淡水湖，昔日号称“'''八百里洞庭'''”。由于近现代的围湖造田，以及自然的泥沙淤积，洞庭湖面积由最大时的[[清|清]][[順治|順治]]到[[道光|道光]]年間汛期的湖面面積约6,000[[平方公里|平方公里]]骤减到[[1983年|1983年]]的2,625平方公里，[[中华人民共和国|中华人民共和国]]成立后被[[鄱阳湖|鄱阳湖]]超过而退居为中国境内第二大淡水湖。近年来加强了对湖泊区域的保护，实行退耕还湖。現在天然湖泊面积2,820平方公里，蓄洪堤垸和单退堤垸高水还湖扩大湖泊面积1,343平方公里，总共3,968平方公里（官方数据）。
