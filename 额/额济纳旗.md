'''額濟納旗'''（{{Mong|m=ᠡᠵᠡᠨ᠎ᠡ <br>ᠬᠣᠰᠢᠭᠤ|k=Эзнээ хошуу}}）是[[中華人民共和國|中華人民共和國]][[內蒙古自治區|內蒙古自治區]][[阿拉善盟|阿拉善盟]]轄下的一個[[旗|旗]]。


== 历史 ==
在古代为[[草原丝绸之路|草原丝绸之路]]常经之地，也是[[西汉|西汉]]至[[唐朝|唐朝]][[河西走廊|河西走廊]]的军事战略要地，[[西夏|西夏]]时称[[黑水城|黑水城]]，额济纳乃[[党项语|党项语]]称呼，这座城市也是当今世界现存的唯一一座以党项语命名的城市。元代在此设有[[亦集乃路|亦集乃路]]，属[[甘肃等处行中书省|甘肃省]]。

== 簡介 ==
額濟納旗，地廣人稀。多為無人居住的沙漠區域。除了本有的蒙古人所經營的游牧業之外，最近該旗也發展旅遊業。另外，著名的[[酒泉衛星發射中心|酒泉衛星發射中心]]也位於該旗。額濟納旗原为西北地区的生态屏障，现在已沦为[[沙尘暴|沙尘暴]]的重要发源地<ref>{{cite journal zh |title=祁连山以北地区生态植被的变化规律 |journal=科技导报|date=2006|authors = 金晓媚、万力、李文梅 |volume=25|issue=3|pages=18-21|id= |url=|format=|accessdate=}}</ref>。

== 交通 ==
===公路===
达（达来库布镇）酒（[[酒泉|酒泉]]）线（S312）和达（达来库布镇）银（[[银川|银川]]）线（S315 ）。

还可以从蒙古国通过[[策克口岸|西伯库伦口岸]]进入。

===铁路===
[[临策铁路|临策铁路]]、[[清绿铁路|清绿铁路]]、[[嘉策铁路|嘉策铁路]]贯穿全境。有客运列车在呼和浩特与达来呼布间往返。

===航空===
[[额济纳旗桃来机场|额济纳旗桃来机场]]于2013年12月17日通航。目前有[[幸福航空|幸福航空]]和[[东方航空|东方航空]]来往[[阿拉善左旗|阿拉善左旗]]和[[西安|西安]]的定期航班。<ref>[http://data.carnoc.com/corp/airport/ejn.html 额济纳旗桃来机场]</ref>
