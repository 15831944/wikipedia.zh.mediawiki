'''玛珍塔·迪瓦恩'''（{{Lang-en|'''Magenta Devine'''}}，{{bd|1957年|11月4日|2019年|3月6日|catIdx=Devine}}），[[联合王国|英国]]女性[[主持人|电视主持人]]、[[记者|记者]]和音乐推广人。<ref name="guardian">{{cite web |author1=Janet Street-Porter |title=Magenta Devine, presenter of Network 7 and Rough Guide, dies aged 61 |url=https://www.theguardian.com/tv-and-radio/2019/mar/06/magenta-devine-presenter-of-network-7-and-rough-guide-dies-aged-61 |publisher=[[卫报|卫报]] |accessdate=2019-03-09}}</ref><ref>{{cite web |author1=Melanie McFadyean |title=How We Met: Magenta De Vine and David Okuefuna |url=https://www.independent.co.uk/arts-entertainment/how-we-met-magenta-de-vine-and-david-okuefuna-1511242.html |publisher=[[独立报|独立报]] |accessdate=2019-03-09}}</ref><ref>{{cite web |author1=Hayley O'Keeffe |title='Aylesbury legend' Magenta Devine dies aged 61 |url=https://www.bucksherald.co.uk/news/aylesbury-legend-magenta-devine-dies-aged-61-1-8837467 |publisher={{Link-en|雄鹿先驱报|Bucks Herald}} |accessdate=2019-03-09}}</ref><ref>{{cite web |title=Magenta Devine, TV presenter known for Network 7 and Rough Guide |url=https://www.heraldscotland.com/opinion/17483804.magenta-devine-tv-presenter-known-for-network-7-and-rough-guide/ |publisher={{Link-en|先驱报 (格拉斯哥)|The Herald (Glasgow)|先驱报}} |accessdate=2019-03-09}}</ref>玛珍塔·迪瓦恩因在1990年代为[[英國廣播公司第二台|BBC电视二台]]主持了青年电视节目《简明指南（{{lang|en|Rough Guides}}）》和《报道（{{lang|en|Reportage}}）》而家喻户晓。<ref>{{cite web |author1=Paul McCann |title=Media hypocrites fuelled by cocaine |url=https://www.independent.co.uk/news/media-hypocrites-fuelled-by-cocaine-1180537.html |publisher=[[独立报|独立报]] |accessdate=2019-03-09}}</ref>随后她又为[[独立电视台|独立电视台]]主持了节目《{{Link-en|年轻有天赋却一文不名|Young, Gifted and Broke}}》。<ref>{{cite web |title=Magenta De Vine |url=https://www.bfi.org.uk/films-tv-people/4ce2ba8346c11 |publisher=[[英国电影协会|英国电影协会]] |accessdate=2019-03-09}}</ref>

== 职业生涯 ==
玛珍塔·迪瓦恩最初是为{{Link-en|托尼·布兰斯比|Tony Brainsby}}工作，后者是[[皇后乐队|皇后乐队]]、{{Link-en|瘦李奇乐队|Thin Lizzy}}和[[白蛇_(樂團)|白蛇乐队]]的[[公关|公关]]。<ref>{{cite web |author1=David Stubbs |title=Magenta Devine: an 80s TV icon of effortless style and substance |url=https://www.theguardian.com/tv-and-radio/2019/mar/06/magenta-devine-network-7-rough-guide-80s-tv-icon-of-style-and-substance |publisher=[[卫报|卫报]] |accessdate=2019-03-09}}</ref> 在与男友{{Link-en|托尼·詹姆斯|Tony James (musician)}}一起生活的同时，她也成为詹姆斯新乐队{{Link-en|跟随卫星|Sigue Sigue Sputnik}}的音乐推广人。<ref name="BBC">{{cite web |title=Magenta Devine, TV presenter, dies aged 61 |url=https://www.bbc.com/news/entertainment-arts-47473388 |publisher=[[BBC新闻|BBC新闻]] |accessdate=2019-03-09}}</ref> 迪瓦恩从1986年开始了她的电视生涯，她在[[BBC威尔士|BBC威尔士]]主持一档名为《果汁（{{lang|en|Juice}}）》的流行音乐节目。<ref name="juice">{{cite web |title=Juice |url=https://www.tvcream.co.uk/telly/tv-a-z/a-m/j-is-for/juice/ |publisher=TV Cream |accessdate=2019-03-09}}</ref> 在迪瓦恩与詹姆斯分手之后，制片人{{Link-en|雅内·史崔特-波特|Janet Street-Porter}}预定了迪瓦恩去主持[[第四台_(英國)|四频道]]的青年节目《{{Link-en|第七网络|Network 7}}》。<ref name="BBC" /> 然而迪瓦恩却去了[[英國廣播公司第二台|BBC电视二台]]，去主持一档名为《{{Link-en|DEF II|DEF II}}》的电视节目。简单明了的指南是其节目特色，后来发展成为独立节目《简明指南（{{lang|en|Rough Guide}}）》。<ref name="BBC" />

== 晚年生活 ==
1990年代，玛珍塔·迪瓦恩曾经寻求过医疗手段来解决自己的[[海洛因|海洛因]][[药物成瘾|上瘾问题]]和[[抑郁症|抑郁症]]。<ref name="BBC" /> 2003年，她被伦敦的[[英格蘭及威爾斯高等法院|高等法院]]宣布[[破产|破产]]。<ref name="Standard"/>

2019年3月6日，玛珍塔·迪瓦恩因疾病进入伦敦中心医院治疗短暂时间后去世，享壽61岁。<ref name="BBC" />
