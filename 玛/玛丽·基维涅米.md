'''玛丽·基维涅米'''（{{lang-fi|'''Mari Johanna Kiviniemi'''}}，{{bd|1968年|9月27日|catIdx=Kiviniemi}}），[[芬兰|芬兰]]政治人物，'''芬兰第二位女[[芬兰总理|总理]]'''。

== 早年 ==
1968年9月27日，基维涅米生于芬兰的[[塞伊奈约基|塞伊奈约基]]。

1988年，入学于[[赫尔辛基大学|赫尔辛基大学]]经济学专业。

== 政坛 ==

1995年，成为[[芬兰议会|芬兰议会]]议员。

2003年，出任中间党副主席。

2005年9月至2006年3月，基维涅米出任芬兰外贸与开发事务部长。

2007年4月至2010年6月22日出任行政与地方事务部长。

2010年6月基维涅米当选为中间党主席，而且由于中间党为芬兰最大的执政党，基维涅米获芬兰总统塔里娅·哈洛宁任命为芬兰新一届政府总理，成为'''芬兰历史上第二位女总理'''<ref>[http://news.xinhuanet.com/world/2010-06/23/c_12249813.htm 芬兰总统任命玛丽·基维涅米为新一届政府总理]，新华网，2010年6月23日。</ref>。

== 个人==
1996年，玛丽与商人Juha Louhivuori结婚，育有一女Hanna（1997年出生）和一子Antti（2000年出生）。

玛丽的业余爱好包括古典音乐，她会弹钢琴和吹长笛。

玛丽精通四门语言：芬兰语、德语、英语、瑞典语。
