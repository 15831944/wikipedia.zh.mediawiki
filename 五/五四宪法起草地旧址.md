'''五四宪法起草地旧址'''位于[[中国|中国]][[浙江省|浙江省]][[杭州市|杭州市]][[西湖区_(杭州市)|西湖区]][[北山街道_(杭州市)|北山街道]][[葛岭|葛岭]]南麓[[北山街|北山街]]84号院内30号楼（原葛岭路30号），原为[[汤恩伯|汤恩伯]]别墅<ref>{{cite web|url=http://www.npc.gov.cn/npc/xinwen/2014-12/04/content_1888168.htm0|title=《你不知道的宪法诞生地》|accessdate=2018-07-06|work=|publisher=中国人大网}}{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

此处建筑建于1930年代，由东侧的二层楼房和西侧的平房组成。1949年5月后与周边其它别墅被改为[[中共浙江省委|中共浙江省委]]干部宿舍（即北山街84号院，今西湖山庄）<ref>{{cite web |url=http://hzdaily.hangzhou.com.cn/hzrb/html/2009-02/06/content_602365.htm|title=《探秘北山街84号》|accessdate=2018-07-06|work=仲向平、韩斌|publisher=杭州日报}}</ref>。1953年12月28日至次年3月14日，[[毛泽东|毛泽东]]率领宪法起草小组成员以此为办公地，起草了中华人民共和国第一部宪法（[[中华人民共和国宪法_(1954年)|“五四宪法”]]）草案初稿<ref>{{cite web |url=http://hzdaily.hangzhou.com.cn/hzrb/html/2016-12/05/content_2417365.htm?from=timeline&isappinstalled=0|title=《全国首家以宪法为主题的陈列馆史料珍贵》|accessdate=2018-07-05|work=郑莉娜|publisher=杭州日报}}</ref>。

2016年12月4日该建筑辟为'''五四宪法历史资料陈列馆'''对外开放，为首个中国宪法纪念场馆。现状平房内设有会议室、会客室、毛泽东办公室、休息室和值班室等复原陈列，以保护和展示当年留存的实物，二层楼房内设有《西子湖畔 制宪奠基》主题陈列，展示“五四宪法”从起草、讨论、通过到实施的全过程<ref>{{cite web |url=http://www.ccrnews.com.cn/index.php/Index/content/id/66027.html|title=《西子湖畔 制宪奠基 “五四宪法”历史资料陈列馆基本陈列》|accessdate=2018-07-05|work=王永翔|publisher=中国文物信息网}}</ref>。
