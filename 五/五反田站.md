'''五反田站'''（{{jpn|j=五反田駅|hg=ごたんだえき|rm=Gotanda eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[品川區|品川區]][[東五反田|東五反田]]一丁目與二丁目，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）、[[東京急行電鐵|東京急行電鐵]]（東急）、[[東京都交通局|東京都交通局]]（[[都營地下鐵|都營地下鐵]]）的[[鐵路車站|鐵路車站]]。

== 历史 ==
[[File:JR_Gotanda_sta_001.jpg|thumb]]
* 1911年（[[明治|明治]]44年）10月15日：[[鐵道省|鐵道院]]山手線五反田站開業。
* 1928年（[[昭和|昭和]]3年）6月17日：池上電氣鐵道（現東急池上線）五反田站開業。
* 1945年（昭和20年）5月24日：[[太平洋戰爭|太平洋戰爭]]時被美軍[[空襲|空襲]]，造成站房全毀。
* 1968年（昭和43年）11月15日：都營地下鐵1號線五反田站開業。
* 1978年（昭和53年）7月1日：都營地下鐵1號線改名淺草線。
* 1987年（昭和62年）4月1日：[[國鐵分割民營化|國鐵分割民營化]]，本站由JR東日本接手管理。
* 2000年（[[平成|平成]]12年）3月1日：JR東日本與東急的連絡通道啟用自動驗票閘門<ref>{{Cite journal|title=五反田駅に自動改札を設置|journal=HOT ほっと TOKYU|date=2000-03-01|issue=216|url=http://hot.tokyu.co.jp/railway/hot/2000/2000-03.pdf|publisher=東京急行電鉄|accessdate=2017-01-21|format=PDF|deadurl=yes|archiveurl=https://web.archive.org/web/20150116192614/http://hot.tokyu.co.jp/railway/hot/2000/2000-03.pdf|archivedate=2015-01-16}}</ref>。
*2001年（[[平成|平成]]13年）11月18日：智慧卡系統[[Suica|Suica]]正式投入服務。
* 2007年（平成19年）3月18日：東急、都營地下鐵車站啟用IC卡「[[PASMO|PASMO]]」。
* 2012年（平成24年）：五反田站東急、JR連絡通道設置電扶梯與電梯。
* 2015年（平成27年）3月28日：山手線[[月台門|月台門]]啟用（山手線第18個啟用的車站）。<ref>{{Cite web |url=http://shinagawa.keizai.biz/headline/2311/ |title=JR山手線の五反田駅のホームドア運用開始 山手線で18駅目|publisher=品川経済新聞 |accessdate=2016-08-17}}</ref>

=== 站名由來 ===
1911年，官設鐵道在[[荏原郡|荏原郡]][[大崎_(品川區)#歷史|大崎町]]上大崎[[小字|字]]子之神（現地）開設車站，選擇以近鄰下大崎的字、已形成市區的'''五反田'''（現在品川區東五反田一、二丁目一部分）作為站名。

五反田意為「五反之田」（一反約992平方公尺）。

本站是[[島式月台|島式月台]]1面2線的[[高架車站|高架車站]]。月台位於轉彎處，車站下方有[[國道1號_(日本)|國道1號線]]通過。剪票口有JR管轄的一樓剪票口與東急管轄的東急五反田大樓口兩處。

2010年，站內進行改良工程。付費區內的廁所與剪票口外的[[綠色窗口|綠色窗口]]、自動售票機皆進行更新，月台與1樓剪票口之間新設電梯。

2008年（[[平成|平成]]20年）3月14日，車站西側直通的複合商業設施「[[atre|atre vii五反田]]」開幕<ref name="atre">{{Cite web |url=http://www.ekipara.com/building/K1040R17.html |title=アトレヴィ五反田（駅ビルポータルサイト「駅パラ」より） |publisher=東日本旅客鉄道 |accessdate=2014-07-24}}</ref>，吸引高級[[超市|超市]]「[[成城石井|成城石井]]」與餐飲店進駐<ref name="atre"/>。而付費區內也開設藥妝店，剪票口外正面（月台下方）開設{{link-ja|NewDays|NEWDAYS}}與麵包店<ref>{{Cite web |url=http://sasp.mapion.co.jp/b/suica/attr/?srt_yomi=%E3%81%94%E3%81%9F%E3%82%93%E3%81%A0%E3%81%88%E3%81%8D |title=JR東日本：Suicaポイントがたまるお店検索＞店舗検索結果（ごたんだえき） |publisher=東日本旅客鉄道 |accessdate=2014-07-24 |deadurl=yes |archiveurl=https://archive.is/20140728220330/http://sasp.mapion.co.jp/b/suica/attr/?srt_yomi=%E3%81%94%E3%81%9F%E3%82%93%E3%81%A0%E3%81%88%E3%81%8D |archivedate=2014-07-28 }}</ref>。

東急五反田大樓口的自動售票機為東急樣式。因此，在此購入JR站入場券會得到一張寫為「精算書」的紙張。

本站為島式月台1面2線的高架車站。月台位於比山手線更高的位置。

車站與車站大樓東急五反田大樓（地上9層，地下2層）一體化，4樓直通池上線月台。身障人士可使用東急五反田大樓內的電梯前往。該大樓於2008年（平成20年）<ref name="store">{{cite press release|title=東急五反田ビルのリニューアルを実施 |url=http://www.tokyu-store.co.jp/news/pdf/2007.4.18%E6%9D%B1%E6%80%A5%E4%BA%94%E5%8F%8D%E7%94%B0%E3%83%93%E3%83%AB%E3%81%AE%E3%83%AA%E3%83%8B%E3%83%A5%E3%83%BC%E3%82%A2%E3%83%AB%E3%82%92%E5%AE%9F%E6%96%BD.pdf |publisher=東急ストア |date=2007-04-18|accessdate=2014-07-24}}p.2</ref>4月23日起全面整修，之後以購物中心「remy gotanda（レミィ五反田）」重新開幕。主題為「PleasantWalk」（宜人的開放步道），連通車站的4樓採用挑高方式打通5樓<ref name="store"/>，因此{{link-ja|東急store|東急ストア}}縮小為1至3樓。

池上線 - 山手線間的連絡通道在過去僅有階梯，2012年8月底全面啟用電扶梯與電梯，完成無障礙化<ref>{{Cite web |url=http://www.tokyu.co.jp/contents_index/guide/webcata2013-2014/index.html?highlightwords=%E4%BA%94%E5%8F%8D%E7%94%B0%E9%A7%85#page=21 |title=TOKYU CORPORATION 2013-2014（東急電鉄会社案内：デジタルカタログ） |publisher=東京急行電鉄 |accessdate=2014-07-24 |archive-url=https://web.archive.org/web/20150426135054/http://www.tokyu.co.jp/contents_index/guide/webcata2013-2014/index.html?highlightwords=%E4%BA%94%E5%8F%8D%E7%94%B0%E9%A7%85#page=21 |archive-date=2015-04-26 |dead-url=yes }}</ref><ref>{{cite press release|title=２０１１年度の鉄軌道事業設備投資計画は総額３６７億円 |url=http://www.tokyu.co.jp/file/2011513-1.pdf|publisher=東京急行電鉄 |date=2011-05-13|accessdate=2014-07-24}}p.2</ref>。

原先池上線的終點站為下一站[[大崎廣小路站|大崎廣小路站]]，與本站採用步行轉乘。而且當時池上電氣鐵道有計畫跨越山手線延伸<ref>週刊「歴史でめぐる鉄道全路線 大手私鉄06 東京急行電鉄」（朝日新聞出版刊） 27ページ</ref>，因此本站比高架的大崎廣小路站更高。1960年代中期規劃東急泉岳寺線時，曾考慮廢除池上線部分區間，並將本站從高架轉為地下，線籍改為泉岳寺線，之後因計畫變更中止泉岳寺線建設計畫，本站維持高架車站構造。

蒲田站發往五反田站的末班車到達本站後，會停放至隔天作為往蒲田站的始發列車。

本站為[[島式月台|島式月台]]1面2線的[[地下車站|地下車站]]。剪票口層與月台之間、近JR五反田站的「A3」出入口與大廳之間設有電梯與電扶梯。<!-- 都交通局Webサイト・五反田駅の立体図 -->副站名為「立正大學前」。

當初曾規劃先行通車至本站，因此戶越方向可增設剪式橫渡線<ref>「都営地下鉄建設史 - 1号線 - 」（東京都交通局、1971年）</ref>，但至今仍未裝設。

作為大門站管區五反田站務區的所在站，本站管理淺草線[[西馬込站|西馬込站]] - [[三田站_(東京都)|三田站]]間各站<ref>三田駅の他の路線は管轄外である。</ref>。

本站除了定期券售票處（委託給{{link-ja|東京都營交通協力會|東京都営交通協力会}}），月台還有商店「merci」。

== 使用情況 ==
* '''JR東日本''' - 2017年度1日平均上車人次為'''139,030人'''<ref group="使用人數">[http://www.jreast.co.jp/passenger/index.html 各駅の乗車人員] - JR東日本</ref>。
*: 該公司車站中次於[[船橋站|船橋站]]位居第24位。
* '''東京急行電鐵''' - 2017年度1日平均上下車人次為'''113,193人'''<ref group="使用人數">[http://www.tokyu.co.jp/railway/data/passengers/ 各駅の1日の平均乗降人員]{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }} - 東京急行電鉄</ref>。
*: 池上線內排名第1位。

*: 淺草線全部20個車站當中次於[[東銀座站|東銀座站]]名列第9位。
