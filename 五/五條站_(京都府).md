'''五條站'''（{{jpn|j=五条駅|hg=ごじょうえき|rm=Gojō eki|f=y}}）是一個位於[[日本|日本]][[京都府|京都府]][[京都市|京都市]][[下京區|下京區]]大坂町，屬於[[京都市營地下鐵|京都市營地下鐵]][[烏丸線|烏丸線]]的[[鐵路車站|鐵路車站]]。車站編號是'''K10'''。

== 歷史 ==
* 1981年（[[昭和|昭和]]56年）5月29日 - 烏丸線的[[北大路站|北大路]]－[[京都站|京都]]間開始營業之際開業<ref name=sone04>{{Cite book|和書|author=[[曽根悟|曽根悟]]（監修） |title=週刊 歴史でめぐる鉄道全路線 公営鉄道・私鉄 |editor=朝日新聞出版分冊百科編集部（編集） |date=2011-04-03 |publisher=[[朝日新聞出版|朝日新聞出版]] |series=週刊朝日百科 |volume=4号 京福電気鉄道・叡山電鉄・嵯峨野観光鉄道・京都市交通局 |pages=22-23}}</ref>。

== 車站結構 ==
此站位於[[烏丸通|烏丸通]]與[[五條通|五條通]]交差的交差點（烏丸五条）的地下，月台是島式1面2線。閘口南北各設1個，只有北閘口配備有站員。
