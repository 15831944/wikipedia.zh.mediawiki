'''五邑大学'''（[[英文|英文]]：{{lang|en|'''Wuyi University'''}}）建于1985年，是[[中国|中国]][[广东省|广东省]]一所全日制本科综合性[[大学|大学]]。学校位于[[珠江三角洲|珠江三角洲]]西部的[[广东省|广东省]][[江门市|江门市]]，学校占地约1000亩，天沙河蜿蜒横贯其中，一河两区，毗邻[[香港|港]][[澳門|澳]]，北距[[广州|广州]]、南距[[珠海|珠海]]各约100[[公里|公里]]。江门市辖[[新会|新会]]、[[台山|台山]]、[[鹤山|鹤山]]、[[开平|开平]]、[[恩平|恩平]]五市（区），简称五邑，'''五邑大学'''由此得名。

=== 图书馆 ===
五邑大学图书馆是江门五邑地区最大的图书文献和信息情报中心。它始建于1985年，总建筑面积为24438平方米。

截止到2015年12月30日，图书馆馆藏纸质文献158.61万册，其中中文纸质图书141.86万册，外文纸质图书5.03万册，中外文过刊7.22万册；图书馆还订阅了中外文期刊、报纸共1475种。图书馆电子资源丰富，馆藏电子图书132.5万册，订阅了28种（类）中外文数据库（[[中国知网|中国知网]]、国研网，北大法宝、EBSCO、SPRINGER等），拥有自建数据库3个（英语语音学习数据库、华人华侨数据库、随书光盘数据库）；五邑华侨文献资源库为此馆的特色馆藏，正在不断地积累和建设中。<ref>{{Cite web|url=http://lib.wyu.edu.cn/Html/Aboutus_5.html|title=五邑大学图书馆|accessdate=2016-09-23|work=lib.wyu.edu.cn|deadurl=yes|archiveurl=https://web.archive.org/web/20090417114555/http://lib.wyu.edu.cn/Html/Aboutus_5.html|archivedate=2009-04-17}}</ref>

== 争议 ==
2018年9月7日，[[豆瓣|豆瓣]]出现一篇以机电工程院学生黄某的口吻叙述的文章。黄某在文中自述，今年5月他潜入女生宿舍楼，持自己的宿舍钥匙开门进入大楼其中一间房，[[猥亵|猥亵]]了正在熟睡的一名大二女学生。黄某在事后立即逃跑，但将道歉字条交给同幢大楼的另一名学生，拜托她交给受害者，受害者报警求助，警方根据信中附上的[[腾讯QQ|QQ]]号码[[中国网络实名制|找到]]黄某。学校辅导员要求他坦白自己的行为，并休学回家治疗[[抑郁症|抑郁症]]。但是三个月后，黄某接到学校建议其自行退学的通知。不能接受结果的黄男发帖“喊冤”，曾多次尝试自杀作结，但没有成功。消息公布后，校方表示正研究处理对策，若黄某在网帖中供述的事实属实，会严肃处理。最终，黄某家长到学校办理了退学。而受害女生已返校上课，情绪正常，并一直参加学校的各项活动。<ref>{{cite web |title=广东五邑大学男生猥亵女生后主动退学，患抑郁症正接受治疗  |url=http://www.sohu.com/a/253273415_161795 |website=南方都市报 |accessdate=2018-09-11 |date=2018-09-11}}</ref><ref>{{cite web |title=男生发帖称“辛苦考大学摸下女生全没了”，校方：若属实严处 |url=https://www.thepaper.cn/newsDetail_forward_2428833 |website=澎湃新闻 |accessdate=2018-09-11 |date=2018-09-11}}</ref><ref>{{cite web |title=【無恥】猥褻女生被勸退　廣東男大生：辛苦考大學點解摸下就冇咗 |url=https://www.hk01.com/%E5%A4%A7%E5%9C%8B%E5%B0%8F%E4%BA%8B/234149/%E7%84%A1%E6%81%A5-%E7%8C%A5%E8%A4%BB%E5%A5%B3%E7%94%9F%E8%A2%AB%E5%8B%B8%E9%80%80-%E5%BB%A3%E6%9D%B1%E7%94%B7%E5%A4%A7%E7%94%9F-%E8%BE%9B%E8%8B%A6%E8%80%83%E5%A4%A7%E5%AD%B8%E9%BB%9E%E8%A7%A3%E6%91%B8%E4%B8%8B%E5%B0%B1%E5%86%87%E5%92%97 |website=香港01 |accessdate=2018-09-11 |date=2018-09-11}}</ref>。
