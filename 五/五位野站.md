'''五位野站'''（{{jpn|j=五位野駅|hg=ごいのえき|rm=Goino eki|f=y}}）是一座位於[[鹿兒島縣|鹿兒島縣]][[鹿兒島市|鹿兒島市]]平川町的[[九州旅客鐵道|九州旅客鐵道]]（JR九州）[[指宿枕崎線|指宿枕崎線]]的鐵路車站。周邊為平川動物園。

== 車站構造 ==
*一個2面2線相對式月台，可以[[列車交會|列車交會]]。
*[[無人站|無人站]]。
*[[地面車站|地面車站]]。
*設置為[[自動售賣機|自動售賣機]]。

== 車站周邊 ==

== 历史 ==
* 1930年（昭和5年）12月7日：車站啟用。
* 1983年（昭和58年）3月8日：成為[[無人站|無人站]]。
* 1987年（昭和62年）4月1日：因[[國鐵分割民營化|國鐵分割民營化]]，本站成為[[九州旅客鐵道|九州旅客鐵道]]的車站。
