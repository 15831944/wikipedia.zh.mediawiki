'''五星体育传媒有限公司'''（{{Lang-en|'''Great Sports Media Co.,Ltd.'''}}）是由[[上海文化广播影视集团有限公司|上海文化广播影视集团有限公司]]完全控股的一家从事体育类电视节目制作、发行及[[有线电视|有线频道]]运营的公司。<ref name="about" /> 目前该公司拥有并运营[[上海广播电视台五星体育频道|五星体育频道]]、劲爆体育频道等两家体育专业类有线电视频道。<ref name="about" /><ref name="smg">{{cite web |title=五星体育频道 |url=https://www.smg.cn/review/channel/channel_9/index.html |publisher=[[上海文广集团|上海文广集团]] |accessdate=2018-08-22}}</ref><ref>{{cite web |title=五星体育全新打造《劲爆体育》 6月1日倾情开播 |url=http://sports.sohu.com/20120529/n344336331.shtml |publisher=[[搜狐网|搜狐网]] |accessdate=2018-08-22}}</ref> 五星体育传媒有限公司拥有[[中国足球超级联赛|中国足球超级联赛]]、[[中国篮球联赛|中国篮球联赛]]、[[钻石联赛上海站|国际田径钻石联赛上海站]]、[[上海大師賽|上海ATP1000大师赛]]、[[斯诺克上海大师赛|斯诺克上海大师赛]]、[[上海马拉松|上海国际马拉松赛]]、[[英格兰足球超级联赛|英格兰足球超级联赛]]、[[西班牙足球甲级联赛|西班牙足球甲级联赛]]、[[德国足球甲级联赛|德国足球甲级联赛]]、[[澳大利亚网球公开赛|澳大利亚网球公开赛]]、[[温布尔登网球锦标赛|温布尔登网球锦标赛]]、[[一级方程式赛车|一级方程式赛车]]等赛事在中国大陆的转播权。<ref name="about" /><ref name="smg" />
