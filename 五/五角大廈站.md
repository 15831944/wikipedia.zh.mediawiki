'''五角大廈轉運中心'''（{{lang-en|Pentagon Transit Center}}）是[[美國|美國]][[維珍尼亞州|維珍尼亞州]][[阿靈頓縣|阿靈頓縣]]鄰近[[五角大樓|五角大樓]]，屬於[[華盛頓地鐵|華盛頓地鐵]]的[[分離式月台|分離式月台]]車站。車站在1977年7月1日啟用<ref name="July 1, 1977">{{Citation |last =Feaver |first =Douglas B. |title = Today, Metro could be U.S. model |newspaper = The Washington Post |page = A1 |date =July 1, 1977}}</ref>，並由[[華盛頓都會區交通局|華盛頓都會區交通局]]。此站有[[華盛頓地鐵藍線|藍線]]和[[華盛頓地鐵黃線|黃線]]停靠，亦是兩條路線分岔之處，亦是轉乘車站。向北，藍線繼續通過維珍尼亞州，黃線則跨過[[波多馬克河|波多馬克河]]進入[[華盛頓特區|哥倫比亞特區]]。

車站位於地下，鄰近[[五角大廈|五角大廈]]，並一度有直接前往五角大廈的入口和地下購物中心。此入口在2001年關閉以進行{{link-en|五角大廈翻新計劃|Pentagon Renovation Program}}<ref name="June 16, 2000">{{Citation |last =Layton |first =Lyndsey |title =Pentagon, Metro quarrel escalates; Security concerns would interfere with convenience |newspaper =The Washington Post |page = A10 |date =June 16, 2000}}</ref>。前往五角大廈的通道現時改為途經地面一個新的帶有安檢設施、鄰近巴士車廠的入口以及到地鐵站的入口。
