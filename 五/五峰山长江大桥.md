'''镇江长江大桥'''<ref>{{Cite web|url=http://www.zjna.gov.cn/zjxqgb/xqgl/xqjj/index.html|title=镇江经济技术开发区官方网站-新区概览|accessdate=|author=|date=|publisher=}}</ref>是一座正在建设的[[公路铁路两用桥|公路铁路两用]][[长江大桥|长江大桥]]<ref name = "连镇铁路五峰山长江特大桥主塔墩首根钻孔桩开钻">{{cite web | author = 祁曙光, 戴杰 | editor = 易爽 | title = 连镇铁路五峰山长江特大桥主塔墩首根钻孔桩开钻 | publisher = 中铁大桥局集团有限公司 | date = 2016-01-04 | language = zh | accessdate = 2016-01-05 | url = http://www.ztmbec.com/tabid/3075/InfoID/70661/frtid/3052/Default.aspx}}</ref><ref name = "五峰山长江特大桥两岸施工全面铺开">{{cite web | author = 中铁大桥勘测设计院集团有限公司 | title = 五峰山长江特大桥两岸施工全面铺开 | publisher = 中铁大桥勘测设计院集团有限公司 | date = 2015-12-29 | language = zh | accessdate = 2016-01-05 | url = http://www.brdi.com.cn/Article1.aspx?ID=1566}}</ref>，位于[[长江|长江]]江苏段，为[[连淮扬镇铁路|连淮扬镇铁路]]和[[江宜高速公路|江宜高速公路]]的共用过江通道。该桥全长6408.909m，跨江部分1432m，正桥主航道桥采用（84+84+1092+84+84）米五跨双塔连续[[钢桁梁|钢桁梁]][[悬索桥|悬索桥]]桥式方案。根据设计，镇江长江大桥上层为双向八车道[[高速公路|高速公路]]（桥面宽度40.5米），下层为四线[[铁路|铁路]]<ref name = "中国第一座公铁两用悬索桥镇江五峰山长江大桥即将开建"/>。

镇江长江大桥的设计单位是[[中铁大桥勘测设计院集团有限公司|中铁大桥勘测设计院集团有限公司]]<ref name = "新建连云港至镇江铁路初步设计获批"/><ref name = "新建连云港至镇江铁路可行性研究报告获批"/>；施工单位为[[中铁大桥局|中铁大桥局]]和[[中交二航局|中交二航局]]<ref name="世界最大荷载 武汉建桥军团建功 中国高铁将在悬索桥上飞驰">{{cite web| author =廖志慧, 刘家华, 王虎| title =世界最大荷载 武汉建桥军团建功 中国高铁将在悬索桥上飞驰| publisher =湖北日报| date =2015-10-24| language =zh| accessdate =2015-10-24| url =http://ctdsb.cnhubei.com/html/hbrb/20151024/hbrb2764679.html}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name = "我国首座超千米跨径公铁两用悬索桥开建 中国高铁将再次飞跃长江创世界纪录">{{cite web | author = 乔宣 | editor = 易爽 | title = 我国首座超千米跨径公铁两用悬索桥开建 中国高铁将再次飞跃长江创世界纪录 | publisher = 中铁大桥局集团有限公司 | date = 2015-10-22 | language = zh | accessdate = 2015-10-24 | url = http://www.ztmbec.com/tabid/3075/InfoID/66221/frtid/3052/Default.aspx}}</ref><ref name = "公司铁路桥梁市场再获新突破 中标全国首座超千米跨径公铁两用悬索桥 五峰山长江大桥开工在即">{{cite web | author = 郑婕, 任婷婷 | title = 公司铁路桥梁市场再获新突破 中标全国首座超千米跨径公铁两用悬索桥 五峰山长江大桥开工在即 | publisher = 中交第二航务工程局有限公司 | date = 2015-10-20 | language = zh | accessdate = 2015-10-24 | url = http://www.sneb.com.cn/web.do?action=getNewsInfoById&Module_Id=35&newsId=8174}}</ref>。

== 设计 ==
2014年，[[国家发展改革委|国家发展改革委]]批复可行性研究报告：上层双向八车道高速公路，下层双线铁路，主跨1036米<ref name = "新建连云港至镇江铁路可行性研究报告获批"/><ref name = "国家发展改革委关于新建连云港至镇江铁路可行性研究报告的批复 发改基础[2014]2227号"/>。2015年，[[中国铁路总公司|中国铁路总公司]]批复初步设计：上层双向八车道高速公路，下层四线铁路，主跨1092米<ref name = "中国第一座公铁两用悬索桥镇江五长江大桥即将开建"/><ref name = "新建连云港至镇江铁路初步设计获批"/>。為了滿足鐵路、公路兩用且跨越1400公尺的跨度，採用了[[桁架悬索桥|桁架悬索桥]]的設計<ref>{{cite web | author = 须臾千秋 | title = 对变形极其敏感的高铁列车，如何通过悬索桥？ | publisher = [[网易新闻|网易新闻]]，[[知乎|知乎]] | date = 2017-12-25 | language = zh | accessdate = 2017-12-25 | url = https://zhuanlan.zhihu.com/p/32316847?group_id=928567243453878272}}</ref>。

=== 技术标准 ===
按四线铁路和八车道高速公路标准合建，采用公路在上、铁路在下的分层布置设计。

* 连淮扬镇铁路
** 铁路等级：[[客运专线|客运专线]]
** 正线数目：双线
** 设计活载：ZK-活载
** 限制坡度：20‰
** 行车速度：≤250公里/小时

* 待定
** 铁路等级：[[客运专线|客运专线]]
** 正线数目：双线
** 设计活载：ZK-活载
** 限制坡度：20‰
** 行车速度：≤250公里/小时
