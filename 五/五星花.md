'''五星花'''（[[学名|学名]]：{{lang|la|'''''Pentas lanceolata'''''}}），又名'''繁星花'''，为[[茜草科|茜草科]][[五星花属|五星花属]]下的一种開花植物<ref>{{cite web |title= Pentas lanceolata on the British Database of World Flora and Fauna |url= http://british-towns.net/nature/07_specie_detail.asp?GetLSID=235247 |accessdate=2008-09-15}}</ref>，原產於非洲大部分地區及[[也門|也門]]<ref name="GRIN">{{cite web |url=http://www.ars-grin.gov/cgi-bin/npgs/html/taxon.pl?27317 |title=Taxon: ''Pentas lanceolata'' (Forssk.) Deflers |work=Germplasm Resources Information Network |publisher=United States Department of Agriculture |date=2010-04-30 |accessdate=2012-12-02 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>，是一種[[蝴蝶园|蝴蝶园]]中的常見的園藝植物<ref>{{cite web |title= Pentas lanceolata Floridata |url= http://floridata.com/ref/p/pen_lanc.cfm |accessdate=2008-09-15}}</ref>。
