'''五塘广场站'''是[[南京地铁3号线|南京地铁3号线]]车站，位于[[江苏省|江苏省]][[南京市|南京市]][[鼓楼区_(南京)|鼓楼区]]中央北路与幕府东路交叉路口、五塘广场南侧，沿中央北路西北、东南方向布置。该站为地下三层岛式站台结构，设4个出入口。2015年4月1日投入使用。<ref>{{cite web|url=http://www.njmetro.com.cn/about_01_08show.aspx?Id=175 |title=大事记 |publisher=南京地铁 |quote=4月1日 地铁3号线工程开通试运营 |accessdate=2018-07-29 }}</ref>五塘广场站是3号线9座“[[红楼梦|红楼]]”主题车站之一，主题文化墙为《[[太虚幻境|太虚幻境]]》。<ref>{{cite web |url=http://news.to8to.com/weiyi/13266.html |title=南京地铁三号线开通 站内红楼梦主题背景美瞎了！|publisher=土巴兔新闻 |date=2015-04-02 |accessdate=2018-08-01}}</ref><ref>{{cite web |url=http://js.people.com.cn/n/2015/0330/c360302-24315796.html |title=盘点南京地铁3号线红楼主题站点“之最” |publisher=人民网江苏 |date=2015-03-30 |accessdate=2018-08-01}}</ref>
