'''五四運動'''發生於[[1919年|1919年]]（中华民国8年）[[5月4日|5月4日]]的[[北洋政府|北洋政府]]治下的[[京兆地方|京兆地方]]，是一场以青年學生為主的[[學生運動|學生運動]]，以及包括廣大公民、市民和工商人士等中基層廣泛參與的一次示威遊行、請願、罷課、罷工和暴力對抗政府等多形式的行動。事件起因是在[[第一次世界大戰|第一次世界大戰]]結束後舉行的[[巴黎和會_(1919年)|巴黎和會]]中，[[列強|列強]]將戰敗國[[德意志帝國|德國]]在[[山東|山東]]的權益轉讓給[[大日本帝国|日本]]，即[[山東問題|山東問題]]。當時北洋政府未能捍衛國家利益，有國人極度不滿，從而上街游行表達不滿。當時最著名之口號是「外爭國權（對抗列強侵權），內除國賊（懲處媚日官員）」。

广义五四运动则是指自1915年中日签订《对华[[二十一条|二十一条]]要求》(简称二十一条)到1926年[[北伐战争|北伐战争]]这段时间，中国知识界和青年学生反思及[[批判|批判]][[華夏文化|華夏传统文化]]，追隨「[[德先生|德先生]]」（“[[民主|民主]]”的英文“[[:Wikt:democracy|Democracy]]”）和「[[賽先生|賽先生]]」（“[[科學|科學]]”的英文“[[:Wikt:science|Science]]”）<ref>陳獨秀《〈新青年〉罪案之答辯書》“拥护那德谟克拉西（民主）和赛因斯（科学）两位先生”，“本志同人本来无罪，只因为拥护那德莫克拉西（democracy）和赛因斯（science）两位先生，才犯了这几条滔天的大罪。”“西洋人因為擁護德、賽兩先生，鬧了多少事，流了多少血，德、賽兩先生才漸漸從黑暗中把他們救出來，引到光明世界。我們現在只認定只有這兩位先生，可以救治中國政治上道德上學術上思想上一切的黑暗。若因為擁護這兩位先生，一切政府的壓迫，社會的攻擊笑罵，就是斷頭流血，都不推辭。”</ref><ref>《红旗》1979年第5期，侯外廬《五四时期民主和科学思潮》:「五四時期思想解放運動的特點,就是揭櫫民主與科學,批判與它不相容的舊思想、舊道德和舊文化,提倡新思想、新道德和新文化」。</ref>，探索强国之路的[[新文化運動|新文化運動]]的繼續和發展<ref>{{cite web |url = http://news.xinhuanet.com/theory/2006-09/19/content_5108611.htm |title=光明日報:五四新文化運動與近代中國對民主與科學的追求 |publisher=News.xinhuanet.com |date= |accessdate=2017-05-23}}</ref><ref>{{cite web |url = http://cpc.people.com.cn/GB/68742/84762/84763/5806243.html |title=北京师范大学政治学与国际关系学院韩璐:近年来国内五四运动研究述评 |publisher=cpc.people.com.cn |date= |accessdate=2017-05-23}}</ref><ref>{{cite web |url = http://news.southcn.com/community/shzt/youth/spirit/200404280506.htm |title = 教育部邓小平理论研究中心:五四时期的民主和科学精神 |publisher=News.southcn.com |date= |accessdate=2017-05-23}}</ref>。1924年4月19日，[[中共中央局|中共中央局]]委員長[[陳獨秀|陳獨秀]]和[[秘書|秘書]][[毛澤東|毛澤東]]聯名發出通告，第一次要求各地的黨團體的組織展開“五一”、“五四”、“五五”、“五七”紀念和宣傳活動，強調恢復國權運動、[[新文化運動|新文化運動]]，紀念五五（[[卡爾·馬克思|馬克思]]生日），目的在於傳播[[馬克思主義|馬克思主義]]<ref>{{cite web |url = http://cpc.people.com.cn/GB/75289/5686277.html |author = 中共中央宣传部副部长[[龚育之|龚育之]] |year = 1999 |title = 《纪念“五四”的历史回顾和当代意义》 |website = cpc.people.com.cn |date= |accessdate=2017-05-23}}</ref>。1939年[[八路軍|八路軍]]總政治部、中共中央青委發出《關於部隊紀念“五四”青年節工作的指示》，明確指出中央青委決定每年5月4日為中國青年節（1949年後成中国大陸的固定節日）<ref>{{cite web |author=1059 |url = http://cpc.people.com.cn/GB/64162/124333/124343/7418583.html |title=1930年至1939年青年工作大事记 |publisher=Cpc.people.com.cn |date=2008-06-24 |accessdate=2017-05-23}}</ref>。

== 背景 ==
[[File:54-4.jpg|thumb]]
=== 新思想的傳播與發展 ===
西方思想在[[晚清|晚清]]尤其[[甲午戰爭|甲午戰爭]]之後大量傳入[[中國|中國]]並影響年輕一代，而在民國初年這種影響隨著《[[新青年|新青年]]》等刊物的發展以及[[新文化运动|白話文運動]]的推展，自由和反抗傳統權威等思想影響了學生以及一般的[[市民|市民]]。

[[新文化運動|新文化運動]]提倡[[民主|民主]]和[[科學|科學]]，從思想、政治和文化領域激發並影響了華人尤其是華夏青年的愛國救國熱情，這兩種理念從根本上為五四運動奠定了思想和智力基礎。

社團組織在民國的發展包括[[少年中國學會|少年中國學會]]、工學會、新民學會、新潮社、平民教育講演團和工讀互助團等等，讓五四運動在全國展開有了組織上的基礎。

=== 高等教育进步 ===
中國的教育制度在清末的新政中，學習西方及日本學制而改革。到了民初，高等教育獲得進一步的發展。尤其是[[北京大學|北京大學]]，在校長[[蔡元培|蔡元培]]的領導下，引進了開放的學風，提出了「思想自由，兼容並包」辦學方針，[[李大釗|李大釗]]、[[朱家骅|朱家骅]]、[[陳獨秀|陳獨秀]]、[[章士釗|章士釗]]、[[胡適|胡適]]、[[辜鴻銘|辜鴻銘]]（[[英國文學|英國文學]]）、[[劉師培|劉師培]]、[[錢玄同|錢玄同]]（教[[音韻學|音韻學]]）、[[吳梅|吳梅]]（教[[戲曲史|戲曲史]]）、[[劉半農|劉半農]]（教新文學）、[[嚴復|嚴復]]（[[翻譯家|翻譯家]]）等被聘請於北大任教，同時培養學生獨立自主開放進步的思想和精神，這種思想和精神成為五四運動的重要動力。
（注：[[魯迅|魯迅]]并不是北大职工，而只是兼职。因为当时北大规定，到北大任职，必须辞去原来职务，而鲁迅是[[中華民國教育部|教育部]]裡资料抄写员。）

=== 民族主义情緒 ===
1914年8月23日，日本對德國宣戰，經70多日激戰，於11月7日全部佔領德國[[胶州湾|膠州灣租借地]]。1915年1月，日本向中國提出[[二十一條|二十一條]]，北洋政府在5月9日，接納了其中大多數的要求，這原本日方要求保密的協定，為新聞界所得知，并發佈該協定，激起了民族主義的情緒，使中國知識分子及民眾對於日本以及賣國政府強烈的不滿，被認為這是[[五九國恥|國恥]]，同時也引發了不少反日的活動，這種情緒在五四運動中進一步發展而發揮作用。

1917年8月14日，北京政府向德國宣戰，成為第一次世界大戰的參戰國。1918年初，日本向段祺瑞控制下的北京政府提供了大量貸款，並協助組建和裝備一支中國參戰軍，其貸款還被用於[[安福國會|安福國會]]龐大的賄選開支。同年9月，北京政府與日本交換了關於向日本借款的公文，作為借款的交換條件之一，又交換了關於山東問題的換文，其主要內容為： 1、[[膠濟鐵路|膠濟鐵路]]沿線之日本國軍隊，除濟南留一部隊外，全部均調集於[[青島|青島]]。2、關於膠濟鐵路沿線的警備：日軍撤走，由日本人指揮的巡警隊代替。 3、膠濟鐵路將由中日兩國合辦經營。 北京政府在換文中，對日本的提議「欣然同意」。駐日公使[[章宗祥|章宗祥]]向日本政府親遞換文，後被北京學生痛毆。在中國對德宣戰，與日本同為戰勝國，但德國在山東的權益沒有收回，反而被日本擴大了，這一換文成為[[巴黎和会_(1919年)|巴黎和會]]上日本強佔山東的借口。

=== 一戰与工商业成长 ===
清末以來，中國的工商業雖有所發展，但在西方產品的輸入情形下，中國本土工商業的發展仍然有限，[[第一次世界大戰|一戰]]的發生使歐洲各國產業無力東顧，中國的工商業獲得很大的發展，參與工商業的人口持續增加，民族工業，尤其是輕工業得以巨大發展，城市中的工商階層在中國社會中的地位也更显重要，在五四運動中，他們成為聲援愛國學生的主要力量。

=== 北京政府派系鬥爭 ===
「五四運動」前後，北京政府總統、內閣、國會之間達成了某種平衡，故政治運作較先前為暢順。當時政府領導者為大總統[[徐世昌|徐世昌]]、國務總理[[錢能訓|錢能訓]]，及「[[安福國會|安福國會]]」幕後領袖[[段祺瑞|段祺瑞]]。但政府仍深受各個黨派、各地軍閥的制約，政策因之時有變異。「五四運動」發生之後，「[[研究系|研究系]]」以政府外交失敗做鬥爭「[[安福系|安福系]]」工具，藉傳媒煽動學生舉辦愛國示威遊行，並進一步造成「[[六三运动|六三運動]]」，使得學運風潮擴大，最終逼使相關政府領導人下臺負責。除此之外，「文治派」與「[[安福系|安福系]]」也藉機相互攻訐，以謀求自我派系之利益。是以「五四運動」絕對不可以單單理解為群眾在愛國意識之下的自發行為；事件源起、擴大都深受黨派鬥爭之影響。<ref>試論北京政府對五四運動的態度及舉措[http://blog.yam.com/ritla/article/22854586]</ref>

===韩国三一运动===
{{main|三一运动}}
1919年3月1日，在五四运动的前两个月，韩国爆发[[三一运动|三一运动]]。韩国独立人士在首尔[[塔洞公园|塔洞公园]]宣读《独立宣言书》引发数万名民反日游行示威。三一运动很快席卷整个[[朝鲜半岛|朝鲜半岛]]。有200万以上群众参加了上千起反日示威和暴动。<ref name="论文">[http://www.jlrtvu.jl.cn/xb/file.asp?fileid=20070314013 三一运动与五四运动背景简析] {{Wayback|url=http://www.jlrtvu.jl.cn/xb/file.asp?fileid=20070314013# |date=20141006124337 }}，论文摘编 《吉林广播电视大学学报》 2007年第0期</ref><ref name="China">[http://www.china.com.cn/chinese/WISI/986557.htm 1919年朝鲜三一起义 200万人参加]，中国网</ref>

[[三一运动|三一运动]]受到当时中国各界密切关注，得到当时中国广大知识分子的高度支持与赞扬,成为推进五四运动的强大思想动力。<ref name="中国新闻">[http://cpc.people.com.cn/GB/68742/84762/84763/5806243.html 近年来国内五四运动研究述评]，中国共产党新闻</ref> 通过三一运动，中国人民加速了对[[帝国主义|帝国主义]]本质的认识，看透了[[北洋军阀|北洋军阀]]的反动面目，在反帝救亡方式上给中国人民极大的启迪，<ref name="中国新闻" /><ref>[http://wenku.baidu.com/view/82550a258e9951e79b89277c.html 朝鲜三一运动对中国五四运动的影响]，百度文库</ref>成为五四运动的催化剂和行为的楷模。<ref name="世纪桥">[http://wuxizazhi.cnki.net/Search/LJDS200901032.html 浅析朝鲜三一运动与中国五四运动],《世纪桥》</ref><ref>[http://detail.bookuu.com/1649993.html 《新编五四运动史》]黑龙江人民出版社</ref>两者相互呼应，有着相似的国际、国内背景、运动过程和运动结果。<ref name="世纪桥" /><ref>[http://www.qikan.com.cn/article/xkji201201136.html 试论朝鲜三一运动与中国五四运动的关系]，期刊网</ref>

== 近因 ==
[[File:Tianjin_students_54-2.jpg|thumb]]
從1917年7月起，任交通兼財政總長的曹汝霖與駐日公使章宗祥等人受命於內閣總理[[段祺瑞|段祺瑞]]以出賣主權的代價向日本政府大舉借款。1918年，曹汝霖與日本外相[[後藤新平|後藤新平]]商定，以髙徐、濟順兩路路權為抵押，借款三千萬日元，並密電駐日公使章宗祥在日本簽字。

1919年1月，[[第一次世界大戰|第一次世界大戰]]戰勝國在[[法蘭西第三共和國|法國]][[巴黎|巴黎]]召開「和平會議」。4月15日，中國代表團向巴黎和會提出《廢除[[中日民四條約|中日民四條約]]說帖》，提出废除外国在华势力范围、撤退外国在华驻军等七项希望取消日本强加的“[[二十一条|二十一条]]”及换文的陈述书，遭到拒绝。巴黎和會不顧中國也是戰勝國之一，根據《關於山東之條約》決定將[[德意志帝國|德國]]在中國[[山東|山東]]的權益轉讓給[[大日本帝國|日本]]，只是归还了[[八国联军|八国联军]]侵入北京时被德国掠去的天文仪器。

[[梁啟超|梁啟超]]致電[[汪大燮|汪大燮]]、林長民，建議警醒國民和政府，拒絕在和約上簽字。其電文稱：“汪、林二總長轉外協會：對德國事，聞將以青島直接交還，因日使力爭，結果英、法為所動，吾若認此，不啻加繩自縛，請警告政府及國民嚴責各全權，萬勿署名，以示決心。”5月1日，中國談判代表、外交總長[[陸徵祥|陸徵祥]]將此事電告北京政府，並稱如不簽約，則對撤廢領事裁判權、取消庚子賠款、關稅自主及賠償損失等等有所不利。5月2日，北京政府密電中國代表可以簽約。<ref>{{cite web|url=http://news.xinhuanet.com/blog/2007-05/07/content_6065682.htm |title=五四运动：文化的运动，还是政治的运动？ |publisher=News.xinhuanet.com |date=2006-06-13 |accessdate=2017-05-23}}</ref>

[[林長民|林長民]]《外交警報敬告國民》一文刊載在5月2日的《晨報》頭版頭條：“膠州亡矣!山東亡矣!國不國矣!此惡耗，前兩日僕即聞之。今得任公電，乃證實矣。聞前次四國會議時，本已決定德人在遠東所得權利，交由五國商量處置，惟須得關係國之同意：我國所要求者，再由五國交還我國而已，不知因何一變其形勢也。更聞日本力爭之理由無他，但執一九一五年之二十一款及一九一八年之膠濟換文及諸鐵路草約為口實。嗚呼!二十一款出於脅迫，膠濟換文以該路所屬確定為前提，不得逕為應屬日本之據。濟順、高徐草約，為預備合同，尚未正式訂定。此皆我國民所不能承認者也。國亡無日，願合我四萬萬眾誓死圖之。”<ref>{{Cite book|title=《中国近代史通鉴 1840--1949 7 五四运动与国民革命》|last=戴|first=逸|publisher=红旗出版社|year=1997|isbn=7-80068-722-8|location=北京|pages=405}}</ref>此消息傳到中國後，北京學生群情激憤，學生、工商業者、教育界和許多愛國團體紛紛通電，斥責日本的無禮行徑，並且要求中國政府堅持國家主權。

在這種情況下，和會代表提交了關於山東問題的說帖，要求歸還中國在山東的德[[租界|租界]]和[[膠濟鐵路|膠濟鐵路]]主權，以及要求廢除《二十一條》等合法條件。但結果，[[英國|英]]、[[美國|美]]、[[法國|法]]、[[日本|日]]、-{zh-cn:[[意大利|意]];zh-tw:[[義大利|義]]}-等國不顧中國民众呼聲，在6月28日終於簽訂《[[凡爾賽和約|協約國和參戰各國對德和約]]》，即《凡爾賽和約》，仍然將德國在山東的權利轉送日本<ref>{{cite web |publisher=''[[凡爾賽和約|Treaty of Versailles]]''|title=German Rights and Interests Outside Germany|url=http://www.firstworldwar.com/source/versailles118-158.htm |accessdate=2009-05-03}}</ref>。在巴黎和會中，中国政府的外交失敗，直接引發了中國民众的强烈不滿，进而引发了五四运动。

== 過程 ==
[[File:Burn_Japanese_goods,_Tsinghua_School,_1919.jpg|thumb]]
5月1日，北京大學的一些學生獲悉巴黎和會拒絕中國要求的消息。當天，學生代表就在北大[[西齋飯廳|西齋飯廳]]召開緊急會議，決定5月3日在北大法科大禮堂舉行全體學生臨時大會。

5月3日晚，[[北京大學|北京大學]]學生舉行大會，[[北京高等師範學堂|高師]]、[[北京法政專門學堂|法政專門]]、[[北京高等工業學堂|高等工業]]等學校也有代表參加。學生代表發言，情緒激昂，號召大家奮起救國。最後定出四條辦法，其中就有第二日齊集[[天安門|天安門]]示威的計劃。这四条办法是：（一）联合各界一致力争；（二）通电巴黎专使，坚持不在和约上签字；（三）通电各省于5月7日国耻纪念举行游行示威运动；（四）定于5月4日（星期日）齐集天安门举行学界之大示威。

[[File:zhaojialou.jpg|thumb]]
5月4日，上午10时，各校学生召开碰头会，商定游行路線。下午1点，北京大学等十三所院校三千余名学生汇集天安门，举行了声势浩大的示威活动。提出了“外争国权，内除国贼”、“取消二十一条”、“拒绝合约签字”等口号，並在集会上宣读了[[罗家伦|罗家伦]]起草的，《北京學界全體宣言》：「中国的土地可以征服而不可以断送！中国的人民可以杀戮而不可以低头！国亡了！同胞起来呀！」。然后，总指挥[[傅斯年|傅斯年]]扛着大旗走在遊行队伍最前面，队伍隨即向東交民巷使馆区进发，受到巡捕阻拦，学生代表求会见四国公使，[[羅家倫|羅家倫]]、[[江绍原|江绍原]]前往各國使館遞送英文的備忘錄。仅美国使馆人员接受了学生的陈词书，英法意使馆均拒绝接受。在“外爭國權”而不得的情況下，隊伍中喊出一句：“去找曹汝霖算账去！”矛盾遂轉向“內懲國賊”——學生開始向位于北京长安街东端之北的趙家樓曹宅所在地移動，一路上高喊「收回[[山東|山東]]權利」、「拒絕在[[巴黎和会_(1919年)|巴黎和會]]上簽字」、「廢除二十一條」、「抵制日貨」、「寧肯玉碎，勿為瓦全」、「外爭国權，內惩國賊」、「還我青島」等口號，並且要求懲辦交通總長[[曹汝霖|曹汝霖]]、幣制局總裁[[陸宗輿|陸宗輿]]、駐日公使[[章宗祥|章宗祥]]。下午兩點多鐘，大規模遊行隊伍到達曹宅，數百名軍警早把胡同口封住了，隊伍不得入內。學生向軍警講明來意說：“我們是愛國學生，來這裡是找曹總長談談國事，交換意見，要他愛中國。我們學生手無寸鐵，你們也是中國人，難道你們不愛中國嗎？”軍警於是放行。學生進入胡同，發生暴亂痛打了在曹家串門的[[章宗祥|章宗祥]]，並火燒曹宅，引發「[[火燒趙家樓|火燒趙家樓]]」事件。隨後，軍警給予鎮壓，並逮捕了學生代表32人。<ref>中華民國史事日誌．中華民國八年己未．第437頁</ref>

[[File:54-6.jpg|thumb]]
學生遊行活動受到廣泛關注，各界人士給予關注和支持，抗議逮捕學生，[[蔡元培|蔡元培]]、[[朱家骅|朱家骅]]等“歐美同學會”會員（時會長蔡元培）當時為了營救學生，不惜發動全國工商界罷工罷市，上至政府官員；下至務農人員。北京軍閥政府頒布嚴禁抗議公告，大總統[[徐世昌|徐世昌]]下令鎮壓。但是，學生團體和社會團體紛紛支持。11日，上海成立學生聯合會。14日，天津學生聯合會成立。廣州，南京，杭州，武漢，濟南的學生和工人也給予支持。5月19日，北京各校學生同時宣告罷課，並向各省的省議會、教育會、工會、商會、農會、學校、報館發出罷課宣言。[[天津|天津]]、[[上海|上海]]、[[南京|南京]]、[[杭州|杭州]]、[[重慶|重慶]]、[[南昌|南昌]]、[[武漢|武漢]]、[[長沙|長沙]]、[[廈門|廈門]]、[[濟南|濟南]]、[[開封|開封]]、[[太原|太原]]等地學生，在北京各校學生罷課以後，先後宣告罷課，支持北京學生的鬥爭。

6月，由於學生影響不斷擴大，《[[五七日刊|五七日刊]]》和學生組織宣傳，學生抗議不斷遭到鎮壓。3日，北京數以千計的學生湧向街道，開展大規模的宣傳活動，被軍警逮捕170多人。學校附近駐紮著大批軍警，戒備森嚴。4日，逮捕學生800餘人，此間引發了新一輪的大規模抗議活動。

6月5日，上海工人開始大規模罷工，以響應學生。上海日商的內外棉第三、第四、第五紗廠、日華紗廠、上海紗廠和[[商務印書館|商務印書館]]的工人全體罷工，參加罷工的有兩萬人以上。6日、7日、9日，上海的電車工人、船塢工人、清潔工人、輪船水手，也相繼罷工，總數前後約有六、七萬人。上海工人罷工波及各地，[[京漢鐵路|京漢鐵路]]長辛店工人，[[京奉鐵路|京奉鐵路]]工人及[[九江|九江]]工人都舉行罷工和示威遊行。

6日，上海各界聯合會成立，反對開課、開市，並且聯合其他地區，告知上海罷工主張。通過上海的三罷運動，全國22個省150多個城市都有不同程度的反映。6月11日，[[陳獨秀|陳獨秀]]等人到北京前門外鬧市區散發《[[北京市民宣言|北京市民宣言]]》，聲明如政府不接受市民要求，「我等學生商人勞工軍人等，惟有直接行動以圖根本之改造」。陳獨秀因此被捕。各地學生團體和社會知名人士紛紛通電，抗議政府的這一暴行。面對強大社会舆论壓力，曹、陸、章相繼被免職，總統[[徐世昌|徐世昌]]提出辭職。6月12日以後，工人相繼復工，學生停止罷課。6月28日，中國代表顧維鈞等人沒有在和約上簽字。

=== 後期 ===
1921年11月11日至1922年2月6日，美國、英國、日本、法國、義大利、荷兰、比利時、葡萄牙、中國九國在美國首都華盛頓舉行國際會議。中國政府時值顔惠慶内閣，派出施肇基、頋維鈞、王寵惠三人爲全權代表，余日章、蔣夢麟爲國民代表，[[朱佛定|朱佛定]]爲華盛頓會議中國代表團秘書，1922年2月4日，中國和日本還在華盛頓簽訂了《解決山東問題懸案條約》及其附約。條約規定，日本將德國舊租借地交還中國，中國將該地全部開為商埠；原駐青島、[[膠濟鐵路|膠濟鐵路]]及其支線的日軍應立即撤退；[[青島海關|青島海關]]歸還中國；膠濟鐵路及其支線歸還中國等。附約中規定了對日本人和外國僑民的許多特殊權利，但是中國通過該條約收回了[[山東半島|山東半島]]主權和膠濟鐵路權益。

== 學生代表 ==
=== 北京 ===
* [[方豪_(教育家)|方豪]]（1894-1955），[[浙江|浙江]][[金华|金华]]人。[[北京大學|北京大學]]學生，[[北京学生联合会|北京学生联合会]]及其后的[[中华民国学生联合会|中华民国学生联合会]]的首任主席。後在[[安徽|安徽]]、[[浙江|浙江]]等地長期從事教育工作。
* [[羅家倫|羅家倫]]（1896-1969），[[浙江|浙江]][[紹興|紹興]]人，字志希。[[北京大學|北京大學]]學生，《北京學界全體宣言》起草人。倡導新文化運動、白話文運動。五四後留學海外，後來擔任過[[國立中央大學|國立中央大學]]校長、[[清華大學|清華大學]]校長（1928年9月18日就职）。逝世於[[台北|台北]]。
* [[段錫朋|段錫朋]]（1897-1948），[[江西|江西]][[永新|永新]]人，字書貽。[[北京大學學生會|北京大學學生會]]會長、[[北京中等以上學校學生聯合會|北京中等以上學校學生聯合會]]會長、[[中华民国学生联合会|中华民国学生联合会]]會長。逝世於[[上海|上海]]。
* [[王自治|王自治]]（1889-1965），[[甘肅|甘肅]][[正寧|正寧]]人，字立轩。[[北京大学|北京大学]]學生，繼[[段錫朋|段錫朋]]之後任[[北京學生聯合會|北京學生聯合會]]會長。后長期在[[甘肅省|甘肅省]]任職，並從事教育工作，曾任[[甘肃学院|甘肃学院]]（今[[蘭州大學|蘭州大學]]前身）院長。曾參加[[中國國民黨|中國國民黨]]。1949年后留在[[中國大陸|中國大陸]]，加入[[中國國民黨革命委員會|中國國民黨革命委員會]]。

* [[誠學會|誠學會]]： 是四川學生[[鄒附初|鄒附初]]、[[陳銘德|陳銘德]]、[[鄭獻徵|鄭獻徵]]等所組織，是企圖插手四川政局的地方性組織。大部分是民治主義同志會會員，與四川軍閥[[劉湘|劉湘]]等有聯繫。

== 影響 ==
五四運動從形式上是中國學生的愛國運動，但從整個社會背景社會發展來說，它的影響遠遠不止於此，除了波及中國思想文化，政治發展方向，社會經濟潮流，教育，亦對中國共產黨的發展有重要的作用，同時它對現時中國共產黨領導下的中國社會亦有著不可低估之影響。<ref>毛泽东《新民主主义论》</ref>因強調五四新文化運動在中國現代思想、文化與文學發生中的獨立性與資源性,以此將以科學和民主為旗幟的啟蒙主義的「五四」,與“無產階級領導的人民大眾的反帝反封建的文化”捆綁在一起。<ref>[http://news.xinhuanet.com/ziliao/2004-04/27/content_1441831.htm 五四運動] {{webarchive|url=https://web.archive.org/web/20111018004946/http://news.xinhuanet.com/ziliao/2004-04/27/content_1441831.htm |date=2011-10-18 }}</ref><ref>《「五四」運動六十週年紀念論文集》:我们当前讲的民主是社会主义的民主,这是比以往任何的"民主"都更广泛更彻底的无产阶级的民主。</ref><ref>中国社会科学院近代史研究所《紀念五四运动60周年學朮讨论会文选》“列宁所说: 无产阶级民主比任何资产阶级民主要民主。苏维埃政权比最民主的资产阶级共和国要民主百万倍。”</ref>

=== 五四精神 ===
五四的根本精神是什麼，有不同看法。在一般的看法中，一種看法：科學，民主。1949年後，中共闡述為：愛國，进步，科學，民主。

五四運動，其主力是學生和青年，他們的愛國精神、為真理和正義而戰的精神、不畏強暴和黑暗政治精神值得任何時代的青年和學生學習。

==== 林毓生論「五四精神」 ====
[[林毓生|林毓生]]<ref>林毓生著，穆善培譯，《中國意識的危機》，第335至336頁，貴陽：貴州人民出版社，1986。</ref>認為應當區分五四精神、五四目標與五四思想。林以為，五四精神是一種中國知識份子特有的[[入世|入世]]使命感。這種使命感承襲[[儒家|儒家]]「先天下之憂而憂，後天下之樂而樂」與「家事、國事、天下事、事事關心」的精神，與[[沙皇|俄國沙皇]]時代的讀書人與國家威權同制度發生的「疏離感」，因而產生的知識階級（intelligentsia）激進精神，以及西方社會[[政教分離|政教分離]]為背景而發展出來的近代西方知識份子多元化專業研究的風格，有很大出入。這種使命感使中國知識份子以為真理本身應該指導[[政治|政治]]、[[社會|社會]]、[[文化|文化]]與[[道德|道德]]的發展。其最高境界乃[[孔子|孔子]]「知其不可為而為」的悲劇精神。

而林教授認為「使國家強盛」、「[[自由|自由]]」、「[[民主|民主]]」、「[[法治|法治]]」和「[[科學|科學]]」為五四目標。

而五四思想則包含「全盤反傳統」和「[[全盤西化|全盤西化]]」等傳統[[一元論|一元論]]模式的思想。

=== 思想文化 ===
在1919年10月，總統舉行秋定[[祭孔|祭孔]]，同時組織了[[四存學會|四存學會]]，以「[[昌明|昌明]]」「[[周公|周公]]孔子之學」為宗旨，1921年更修訂了《褒揚條例實施細則》，進一步強化禮教。[[梁啟超|梁啟超]]和[[梁漱溟|梁漱溟]]則高唱中國[[文化優越論|文化優越論]]。而在五四運動中，某些反對儒家思想文化的人士，推行了[[新文化運動|新文化運動]]，提出「[[打孔家店|打孔家店]]」<ref>{{cite web|url=http://history.book.163.com/10/0215/09/5VI8RLJI009244K2.html|title=五四新文化运动无人提出"打倒孔家店"口号|date=2010-02-15|publisher=国家历史}}</ref>、「[[推倒貞節牌坊|推倒貞節牌坊]]」等口號。五四運動進一步促進了反[[儒家|儒家]]的發展，與獨尊[[儒術|儒術]]的思潮形成針鋒相對的局面。

中國語言文字政策的思想淵源大部分都來自於五四時期的西方理論，包括使用白話文（[[現代漢語|現代漢語]]）、國語（[[普通話|普通話]]）作為[[官方語言|官方語言]]，以及[[漢字簡化|漢字簡化]]、[[漢語拼音|漢語拼音]]等。

=== 对外关系 ===
中國代表沒有在巴黎和平會議上簽字，這並不代表中國的利益和權益能夠保留下來，實際上剛好相反，多國利益及國際秩序被排除後，中國成為日本及蘇聯競爭利益的場所（因為日本在華有高額投資及移民、蘇聯則認為中國適合推行[[共產主義|共產主義]]），日本為了保障其僑民及利益、也不滿中國不遵守條約及迫害日僑，逐步的佔領了中國。由於北京政府的失敗，各地軍閥利用北京政府的無力，相互攻伐，加劇了中國的內亂。

1921年到1922年召開了[[華盛頓會議|華盛頓會議]]，中國代表提出「[[十項原則|十項原則]]」，山東問題得到解決，日本把[[青島|青島]]交還給中華民國政府但是[[膠濟鐵路|膠濟鐵路]]仍由日本控制，山東實際仍由日本控制，中國自認權益沒有完全恢復，但實際上逐出列強勢力就等於失去列強保護、也失去了各列強制衡以維持中國基本權益的情勢，日本在中國投資移民最多、加上中國突然由[[日俄戰爭|日俄戰爭]]時及之後的中日友好轉為反日，故懷抱最大的怨念，因此中國實際上失去更多的權益。

[[中國國民黨|中國國民黨]]與[[中國共產黨|中國共產黨]]的對外政策與五四運動的思想相近，日本在二戰戰前獲得的情報是此兩黨都是蘇聯的傀儡，而實際上中國國民黨在中國大陸時期，內部確實有大量的共諜。

=== 社會經濟 ===
學生和群眾的抵制外國貨、外資、外僑，對中國沒有幫助，卻惹得日本全面入侵中國，因為日本在華投資移民最多，不像其他西方國家可以承受退出中國的損失。

=== 教育 ===
[[約翰·杜威|約翰·杜威]]的實用主義教育思想和「[[自動主義|自動主義]]」「[[自律輔導主義|自律輔導主義]]」等教育思想在五四推動下，教育界引起巨變，北京高等師範學校廢除[[學監制|學監制]]，在「開放女禁」呼聲下，1919年秋，[[北京大學|北京大學]]、[[南京高等師範|南京高等師範]]、[[嶺南大學_(廣州)|嶺南大學]]開始招收女生，在教育制度上，婦女權利在五四影響下發生變革。1921年10月，北京政府頒布了[[《學校系統改革案》|《學校系統改革案》]]（壬戊學制），小學六年，初中四年，高中兩年，大學四至六年，以美國為藍本，教育得到革新。

=== 文學藝術 ===
五四運動開創了中國文學新時代。五四以後，中國出現了大小40多個文藝社團，如[[文學研究會|文學研究會]]和[[創造社|創造社]]，對中國文學發展影響深遠。這個時期，出現了大批文學巨匠，如[[沈雁冰|沈雁冰]]、[[鄭振鐸|鄭振鐸]]、[[葉紹鈞|葉紹鈞]]、[[郭沫若|郭沫若]]、[[郁達夫|郁達夫]]等。

===美術===
{{main|中國油畫}}
五四運動後，出國學畫的人逐漸增加。在各地，許多正規的美術院校相繼成立，油畫也在中國紮根。當時湧現出來的出名畫家有[[顏文樑|顏文樑]]、[[劉海粟|劉海粟]]、[[徐悲鴻|徐悲鴻]]、[[林風眠|林風眠]]、[[方幹民|方幹民]]、[[吳大羽|吳大羽]]、衛天霖等，其中以所謂「[[四大校長|四大校長]]」，即顏文樑、劉海栗、徐悲鴻和林風眠的影響最大。

=== 政治 ===
五四運動使[[俄羅斯蘇維埃共和國|蘇俄]]在中國播下無產階級暴力革命的種子，[[中國共產黨|中國共產黨]]的成立和[[國民黨|國民黨]]的改組便充份顯示出：當時很多人認爲“[[布爾什維克|布爾什維克]]革命是當時半封建半殖民地中國的唯一出路”。但也有人認為當時對暴力革命的絕對化，對中國的長遠發展是一種破壞而非建設作用；有些人甚至認為被中國國民黨或中國共產黨統治，還不如成為殖民地。

五四運動以後，工人罷工和政治鬥爭依然不斷，1920年五一，北京學生聯合會散發《[[五一歷史|五一歷史]]》傳單，使馬克思主義，共產黨理論傳入工人階層，為共產黨誕生創造了群眾基礎。在[[中國共產黨|中國共產黨]]黨史上寫下了重要篇章。陳獨秀也在五四之後由激進的民主主義者向馬克思主義者轉變,並接受了階級鬥爭和無產階級專政等觀點。

== 評價 ==
与共产党斗争失利的势力一般认为五四運動直接影響了[[中國共產黨|中國共產黨]]的誕生和發展，為後來的國共內戰打下始端。[[中國共產黨|中國共產黨]]黨史一般將其定義為「反帝反封建的愛國運動」（注意這裡的「[[封建|封建]]」一詞是歐洲人馬克思意義上的封建觀，不是中國先秦時代“分封建國”之封建），並以此運動作為[[舊民主主義革命|舊民主主義革命]]和[[新民主主義革命|新民主主義革命]]的分水嶺。五四運動促進了馬克思主義理論在中國的廣泛傳播，為中國共產黨的成立在思想上和幹部上作了準備。
* 五四运动表现了反帝反封建的彻底性。
* 五四运动是一次真正的群众运动。
* 五四运动促进了马克思主义在中国的传播及其与中国工人运动的结合。
[[中國大陸|中國大陸]]一直存在紀念五四運動的活動，尤其在黨活動中被視為重要紀念，各級黨組織、[[中国共产主义青年团|共青團]]組織、[[學校|學校]]、[[機關|機關]]都對其形式非常重視，每年都有大型的紀念活動。

中国史学界界定五四运动具有重大意义，可概括为：
# 彰顯了學生當時救亡圖存的愛國精神
# 促進了社會各階層覺醒
# 加深國人自立圖強之意識
# 擴大了新文化運動的影響

[[歷史學|歷史學]]家[[林毓生|林毓生]][[教授|教授]]<ref>林毓生，1998，《熱烈與冷靜》，第109至111頁，上海：上海文藝出版社。</ref>提到了五四運動的意識形態，认为其可分內容與思維模式兩個方面來看，其內容又可分形式與實質兩個層次。從形式層次上，五四人物喊出了如[[自由|自由]]、[[民主|民主]]、[[科學|科學]]、[[理性|理性]]、[[思想革命|思想革命]]、[[文學革命|文學革命]]等口號。

=== 其他以「五四」為名的事件 ===
1980年代的新五四运动（New May Fourth Movement），當時稱為就是新古诗运动（Neo-classical Poetry Movement），由[[范光陵|范光陵]]博士（Dr. Fan Kuanling）等主张新古诗运动，主要提倡新古诗运动、诗意油画、与四位先生打一桌牌，即[[德先生|德先生]]、[[赛先生|赛先生]]、诚信先生（又名良心）、礼义先生（又名孝经）。

2014年，由[[中華民國|中華民國]]（[[台灣|台灣]]）的[[新黨|新黨]]所提出的新五四運動，以發揚五四精神、捍衛民主與法治為號召，呼籲國人手持[[中華民國國旗|國旗]]走上[[凱達格蘭大道|凱達格蘭大道]]，一同譴責近日來[[太陽花運動|太陽花運動]]以及[[426凱道反核|426凱道反核]]的行動。<ref>{{Cite web |url=http://www.appledaily.com.tw/realtimenews/article/politics/20140502/390372/%E9%83%81%E6%85%95%E6%98%8E%E8%99%9F%E5%8F%AC%E3%80%8C%E6%96%B0%E4%BA%94%E5%9B%9B%E9%81%8B%E5%8B%95%E3%80%8D%E3%80%80%E5%91%A8%E6%97%A5%E4%B8%8A%E5%87%B1%E9%81%93|title=郁慕明號召「新五四運動」　周日上凱道|author = 劉宛琳|date = 2014年5月4日 |publisher =[[蘋果日報_(台灣)|蘋果日報]] |language = zh-hant |accessdate = 2014年5月4日 }}</ref>根據警政署的統計，截至當日下午2點20分左右，約有1萬餘人參與。<ref>{{Cite web |url =http://newtalk.tw/news/2014/05/04/46931.html|title= 凱道「新五四運動」 警方估有萬人參與|author =翁嫆琄 |date = 2014年5月4日 |publisher = 新頭殼newtalk |language = zh-hant |accessdate = 2014年5月4日 }}</ref>[[張安樂|張安樂]]、[[王炳忠|王炳忠]]<ref>{{Cite web |url=http://www.appledaily.com.tw/realtimenews/article/politics/20140504/391455/%E6%96%B0%E4%BA%94%E5%9B%9B%E9%81%8B%E5%8B%95%E3%80%80%E7%8E%8B%E7%82%B3%E5%BF%A0%E6%89%B9%E9%99%B3%E7%82%BA%E5%BB%B7%E7%B4%8D%E7%B2%B9%E6%B3%95%E8%A5%BF%E6%96%AF|title =新五四運動　王炳忠批陳為廷納粹法西斯|author = 陳詩璧|date = 2014年5月4日 |publisher =[[蘋果日報_(台灣)|蘋果日報]] |language = zh-hant |accessdate = 2014年5月4日 }}</ref>以及正逢母喪的[[馬以南|馬以南]]、[[馬冰如|馬冰如]]<ref>{{Cite web |url =https://tw.news.yahoo.com/%E9%A6%AC%E5%AE%B6%E5%A7%90%E5%A6%B9%E5%87%B1%E9%81%93%E7%8F%BE%E8%BA%AB-%E7%99%BD%E7%8B%BC%E7%8B%A0%E6%89%B9%E5%AD%B8%E9%81%8B-%E7%BD%8C%E7%B2%9F%E8%8A%B1-091327472.html |title =馬家姐妹凱道現身　白狼狠批學運「罌粟花」|author = |date = 2014年5月4日 |publisher = [[NOWnews|NOWnews]] |language = zh-hant |accessdate = 2014年5月4日 }}</ref>皆有參與。
