'''五氟化溴'''，为[[溴|溴]]的一种[[氟化物|氟化物]]，化学式为BrF<sub>5</sub>。

== 物理性质 ==

室温下，为无色[[液体|液体]]。[[密度|密度]](0℃，100kPa)为 2.557g/m。[[熔点|熔点]]-60.6℃，[[沸点|沸点]]40.9℃。

== 生成 ==

: 3 BrF → BrF<sub>3</sub> + Br<sub>2</sub>

: Br<sub>2</sub> + 5 F<sub>2</sub> → 2 BrF<sub>5</sub>

: KBr + 3 F<sub>2</sub> → KF + BrF<sub>5</sub>

== 用途 ==

用于[[火箭|火箭]]燃料、合成中间体等。
