'''五华县'''（传统外文：{{lang|pinyin|'''Ngwa'''}}），位于[[中国|中国]][[广东省|广东省]]东北部，属于[[梅州市|梅州市]]管辖。东边与梅州[[丰顺县|丰顺县]]相邻，南边与揭阳[[揭西县|揭西县]]、汕尾[[陆河县|陆河县]]相邻，西边与河源[[紫金县|紫金县]]、[[东源县|东源县]]、[[龙川县|龙川县]]相邻，北边与梅州[[兴宁市|兴宁市]]相连，总面积为3226平方公里。2002年人口为119万，有华侨和港澳台以五華為祖籍者30多万人。县政府驻地在水寨镇。

本縣原稱長樂縣，民國三年（1914年）改名五華縣（因為與[[福建省|福建省]][[長樂縣_(唐朝)|長樂縣]]、[[湖北省|湖北省]]長樂縣重名。福建省長樂縣保留原名，今為福建省[[福州市|福州市]][[長樂區|長樂區]]；湖北省長樂縣改名五峯縣，今為湖北省[[宜昌市|宜昌市]][[五峰土家族自治縣|五峰土家族自治縣]]）。

== 行政区划 ==
2005年5月下辖16个镇：水寨镇、河东镇、郭田镇、双华镇、棉洋镇、安流镇、梅林镇、龙村镇、横陂镇、华阳镇、周江镇、长布镇、潭下镇、岐岭镇、转水镇、华城镇。

== 历史 ==
宋朝熙宁四年（1071年）开始置'''长乐县'''，民国三年（1914年）长乐县更名为五华县。

== 民族 ==
五华县以[[汉族|汉族]][[客家人|客家人]]为主。

== 语言 ==
全县通用[[客家语|客家语]][[五華話|五华话]]，属纯客家县。

== 经济 ==
2004年全县[[国内生产总值|国内生产总值]]为32.2亿元。农业主要以[[水稻|水稻]]为主，[[烟叶|烟叶]]、[[青梅|青梅]]等[[种植业|种植业]]也发展迅速。[[工业|工业]]以汽车配件、制酒、制药、五金配件等[[制造业|制造业]]为主。[[旅游业|旅游业]]逐步兴旺，2004年旅游总收入为1.1亿元。

== 旅游 ==
五华地处山区，旅游资源丰富，旅游景点有：西湖温泉、长乐学宫、狮雄山塔、汤湖热矿泥山庄、益塘水库、七目嶂自然保护区。

== 文化 ==
五华县民间艺术丰厚，[[木偶戏|木偶戏]]和[[采茶戏|采茶戏]]都是其中杰出的代表。五华还拥有省的足球训练基地和[[中国足球协会甲级联赛|中甲联赛]]首个县级城市球队-[[梅州客家足球俱乐部|梅州客家足球俱乐部梅州五华队]]，并曾被[[国家体委|国家体委]]誉为“足球之乡”。五华的石制工艺非常出名，全国各地和东南亚都有五华石工的杰作。 

== 交通 ==
[[广梅汕铁路|广梅汕铁路]]、[[205国道|205国道]]和[[梅河高速公路|梅河高速公路]]从五华县境内通过。
