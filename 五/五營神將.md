'''五營神將'''，或稱'''五營將軍'''、'''五營元帥'''、'''五營神軍'''、'''五營兵馬'''等，簡稱[[五營|五營]]。是[[道教|道教]]與民間信仰中，保護[[廟|廟]]境或[[村莊|村莊]]的重要[[護法神|護法神]]。'''五營'''分為「東營九[[夷|夷]]軍－青旗（[[青龍|青龍]]）」、「南營八[[蠻|蠻]]軍－紅旗（[[朱雀|朱雀]]）」、「西營六[[戎|戎]]軍－白旗（[[白虎|白虎]]）」、「北營五[[狄|狄]]軍」－黑旗（[[玄武|玄武]]）、「中壇[[三秦|三秦]]軍－黃旗（[[黃帝|黃帝]]）」等五支[[神|神]][[軍|軍]]以抵禦[[邪魔|邪魔]]。

五營之中，以[[中壇元帥|中壇元帥]]為主，統領中營，與其他四營（東營、南營、西營、北營）元帥統領，統號稱'''五營元帥'''。

== 淵源 ==
[[File:2015-03-18_文興宮五營元帥令牌.jpg|thumb]]
五營信仰是[[道教|道教]]以至於[[民間信仰|民間信仰]]的重要一環，[[台灣民間信仰|台灣民間信仰]]中的神明結構，其創作模式一以傳統帝王的君權政體為範本， 自有一套分工精細、組織嚴密的神政體系。   
   
最早是由[[玉皇大帝|玉皇大帝]]是掌理[[三界|三界]]的最高統治者，下設文職與武職兩大官系；武職官系又分為「情治系統」和「軍事系統」兩支，前者如[[城隍|城隍]]組織，掌管三度空間的秩序維護，後者則包括天兵（36天罡或軍將）、地兵（72地煞）和五營神兵，專責擔負陰陽兩界的作戰防禦。如果說合稱為「神軍」的天兵、地兵是中央軍的話，那麼五營神兵就是地方的防衛部隊了。   
    
這套五營神兵的信仰，最遲在[[漢代|漢代]][[魏晉|魏晉]]以前即已形成，並出現於文獻中：從有社開始，就逐漸有「五營」的概念，文獻中較早提到的是在漢晉之間的道經，這只是下限，因為「五營」的觀念一定是流傳很久之後才被記錄到書中的，因此可能在[[漢朝|漢朝]]或更早就已形成。 
  
[[臺灣|臺灣]]因係移民社會，信仰沿襲原鄉[[中國大陸|中國大陸]]，自是承襲這套五營信仰於生活中；不過，後來也因地制宜，逐漸在地化，形成臺灣民間信仰的重要元素，尤以和[[王爺信仰|王爺信仰]]、[[城隍|城隍]]信仰與聚落（庄頭；自然村落）的結合，更突顯台灣五營信仰的本土化。

== 說法 ==
五營神將為何許[[神明|神明]]、何種[[姓氏|姓氏]]的說法相當多，以「東南西北中」為序，有「張蕭劉連李」、「龍朱白武李」、「溫康馬趙李」、「溫康王馬李」、「張康馬趙李」、「辛池蔣洪李」、「康趙白黑李」、［溫康高趙馬］等說法。

絕大多數廟宇五營神將的中央主帥是「[[中壇元帥|中壇元帥]]」[[哪吒|哪吒]]（哪吒，俗稱其姓李），李哪吒也同時是[[佛教|佛教]]與[[道教|道教]]的著名[[護法神|護法神]]。

大多數廟宇採用「張蕭劉連李」的說法，一般說法認為居首的東營張將軍，即是[[法主真君|法主真君]][[張其清|張其清]]。南營將軍為輔天真君[[蕭其明|蕭其明]]（法主真君的[[結拜|結拜]]兄弟），又稱蕭法主。西營將軍[[劉其秀|劉其秀]]，北營將軍[[連其亮|連其亮]]。此等神祇在世時，皆是著名的[[閭山派|閭山派]][[道士|道士]]。臺灣的許多廟宇寫作：張其清（張基清）、蕭其明（蕭其靈、蕭其龍）、劉武秀、連宗宮（連忠宮），這是因為法壇多以口耳相傳，疏於文字，但傳說故事大致相同。

「五營旗」不但是內五營的主要象徵物，表示「五營兵馬在此」，同時也是法師或[[乩童|乩童]]調營遣將的作法信物，通常與「五寶」搭配使用。

== 設置 ==
五營分為「內五營」和「外五營」，即是在五個方位布署的營陣，其設置有大有小、有廣有近。大者、廣者有以鄉間村落為單位，於村頭村尾之五方各設置一營；其次以宮廟為單位，在宮廟五方各設置一營；小者、近者為集中設置一處代表五營，於廟埕建有五營厝，也有在部份宮廟壇上安置五營旗。[[新營車站|新營車站]]為全台灣唯一設有五營的車站<ref name="曾鴻儒">{{Cite news|url=http://news.ltn.com.tw/news/life/paper/292968|title=新營火車站區 竟有5座廟|author=曾鴻儒、楊金城|date=2009-04-05|work=|publisher=《自由時報》|accessdate=2017-01-15|language=zh-tw}}</ref>。

== 五營神咒 ==
[[File:白沙坑文德宮五營樓.jpg|thumb]]五營樓]]
*依照各道壇傳承，文字略有出入，大同小異。

*東營：
拜請法天張聖者，身居[[福州|福郡]]延[[泉州|泉]][[汀州|汀]]。

赤腳修行行正法，兜蛇妙相顯威靈。

金沙橋上翻光斗，青龍潭裏早修行。

腳踏火輪驅邪穢，手持寶劍斬妖精。

行雲致雨霑世界，書符出相渡凡塵。

護國救民端妙相，[[代天巡狩|代天行化]]現真身。

四十五年[[南贍部洲|閻浮世]]，遊行國土救生民。

辰月卯日飛身化，化身應現在壇前。

左右伽羅五官將，前後駕馬二威兵。

三界祖師盧太宰，三壇祖師江舍人。

更有蕭劉連聖者，去瘟治病歷苦辛。

法門弟子專拜請，張公法主降臨來。

神兵火急如律令。

*南營：
拜請輔天蕭聖者，法雨廣布施有情。

威靈顯赫耀乾坤，親身下降斬邪靈。

毫光照耀三途路，廣大感應度眾生。

又有麾下諸猛將，點起營中百萬兵。

山精鬼怪皆降伏，威震[[城隍|城隍]]與[[土地神|社神]]。

慈悲親臨降道場，為吾凡間救萬民。

法門弟子專拜請，蕭公法主降臨來。

神兵火急如律令。

*西營：
◎拜請

七臺山上劉聖者，降龍伏虎大慈悲。

巖中瑜伽四十載，修道參見[[龍樹菩薩|龍樹王]]。

助天護民先出洞，千里以外早彎弓。

射殺妖邪誅鬼怪，顯赫威靈展神通。

獅子岩前伏猛虎，金幢樹下降青龍。

法門弟子專拜請，劉公法主降臨來。

神兵火急如律令。

*北營：
◎拜請

七臺嶺下連聖者，林氏六郎有神通。

剪髮光頭惟吾願，少年捨身入瑜伽。

頂禮[[如來|如來]]將身捨，一生發願救萬民。

不怕天高并地厚，不怕山遙江海深。

不怕城隍并[[土地廟|社廟]]，不怕為非不正神。

惟吾爐中行正法，行罡步斗到壇前。

法門弟子專拜請，連公法主降臨來。

神兵火急如律令。

*中營：
◎拜請

三壇海會大尊者，威靈顯赫大將軍。

三頭六臂多變化，腳踏五雷[[風火輪|風火輪]]。

剔骨割肉還父母，至心皈命朝[[世尊|世尊]]。

如來[[靈山|靈山]]來搭救，蓮花水上化生魂。

或說昔日朝[[太乙|太乙]]，鍊得法術造乾坤。

立感即應降祥雲，消災治病速除瘟。

立志廣護[[佛家|佛家]]門，如入[[法界|法界]]之[[善才|善才]]。

身居大羅玉殿內，為救萬民下金階。

金鎗挑起天門開，繡球拋落五湖海。

統掌三壇身自在，主管中營是英才。

扶助真主保江山，與父同尊神威大。

攝魔屏穢真仙術，百萬神兵四邊排。

法門弟子專拜請，中壇元帥降臨來。

神兵火急如律令。

註：關於中壇元帥的請神咒，也有一說如下：
◎拜請

三十三天都元帥，統領天兵下瑤台。 

金鎗一轉天門開，繡球現出五湖海。

頭上日月耀乾坤，腳踏七星毫光大。

扶在三壇真自在，主掌法界高英才。

龍王殿前展威能，飛沙走石洞門開。

收斬江海蛟龍將，治病救苦速消災。

三歲郎君照北斗，百萬天兵四邊排。

法門弟子全拜請，中壇元帥降臨來。

神兵火急如律令。

== 祭祀方式 ==
有的廟宇會奉祀青、紅、黑、白、黃五支[[旗|旗]]子象徵五營神將，也有用[[神位|神位]]、[[佛祖漆|神圖]]、[[符|符]]咒、令牌、小神像、木偶頭像等作為象徵的。

祭拜五營神稱作「犒軍」，所用的[[紙錢|紙錢]]是「甲馬」（一種印有[[武士|武士]]、[[軍旗|軍旗]]、[[盔甲|盔甲]]與[[馬匹|馬匹]]的紙錢）、「[[金白錢|金白錢]]」（[[黃色|黃色]]、[[白色|白色]]長條狀），[[北臺灣|北臺灣]]或用「刈金」；[[南臺灣|南臺灣]]或用「九金」；如奉請[[哪吒|哪吒]][[元帥|元帥]]或另外四營元帥時，可以用一般奉祀[[神明|神明]]的「壽金」等「四色金」祭拜。

在祭拜時，許多信徒除了[[茶|茶]]、[[酒|酒]]、[[餅|餅]]、[[糖果|糖果]]、[[牲禮|牲禮]]、[[水果|水果]]等普通供品以外，尚會供奉生的牧[[草|草]]、[[蔬菜|蔬菜]]（如生的[[蕃薯|蕃薯]]、[[小黃瓜|小黃瓜]]、[[紅蘿蔔|紅蘿蔔]]）等，以供神軍之戰[[馬|馬]]食用。
