'''五·一二命令'''，是指1968年5月12日，[[中国共产党中央委员会|中国共产党中央委员会]]、[[中华人民共和国国务院|中华人民共和国国务院]]、[[中央军事委员会|中央军事委员会]]、[[中央文化革命小组|中央文化革命小组]]联合发布的《命令》，宣布对中国大陆体育系统实行军事接管。

== 内容 ==
《联合命令》中称[[国家体育运动委员会|国家体育运动委员会]]（包括国防体育俱乐部）系统，是[[贺龙|贺龙]]、[[刘仁|刘仁]]、[[荣高棠|荣高棠]]等人成立的“独立王国”<ref>{{Cite book|title=中共党史教学参考资料 26 中 文化大革命研究资料|last=中国人民解放军国防大学党史党建政工教研室编|first=|publisher=[[党史出版社|党史出版社]]|year=1988.10|isbn=|location=|pages=115-116}}</ref>。它还规定：
# 各级体育运动委员会机关，及所属运动系（队），由所在地区的大军区、省军区接管。
# 各航空俱乐部（包括飞机、器材）和滑翔机制造厂由各军区空军接管。
# 各航海、潜水俱乐部（包括船只及航海器材）由海军各舰队接管。 
# 各无线电俱乐部（包括通信设备、器材）由各军区通信部门接管。
# 各射击、摩托俱乐部（包括射击场、武器、弹药、摩托运动器材）由各大军区、省军区指派军训、军械部门分别接管。接管国家体育系统的工作，由各大军区统一组织实施。根据实际需要由陆军、海军、空军分别派出军管小组或军代表<ref>{{Cite book|title=历史的丰碑 中华人民共和国国史全鉴 2 政治卷|last=[[中共中央党校理论研究室|中共中央党校理论研究室]]编|first=|publisher=北京：[[中共中央文献出版社|中共中央文献出版社]] |year=2005.01|isbn=7-5073-1612-2|location=|pages=813-814}}</ref><ref>{{Cite book|title=中国人民解放军通鉴 1927-1996|last=《中国人民解放军通鉴》编辑委员会编|first=|publisher=兰州：[[甘肃人民出版社|甘肃人民出版社]]|year=1997.07|isbn=|location=|pages=1984}}</ref>。

随着命令实施，国家体委所属的各类运动队均被解散，运动员下放到工厂、煤矿和农村。比如，中国围棋开拓者[[陈祖德|陈祖德]]、[[吴淞笙|吴淞笙]]、[[王汝南|王汝南]]、[[曹志林|曹志林]]、[[华以刚|华以刚]]、[[邱鑫|邱鑫]]和[[黄德勋|黄德勋]]；中国象棋大师[[傅光明|傅光明]]；跳伞运动员[[李荣荣|李荣荣]]；1966年世界推举纪录冠军[[邓国银|邓国银]]等下放到[[北京第三通用机械厂|北京第三通用机械厂]]<ref>{{Cite book|title=魏洲平方剩 千古一叹 下|last=门关沟文化馆编|first=|publisher=北京：[[团结出版社|团结出版社]]|year=2011.05|isbn=978-7-5126-0412-4|location=|pages=483-484}}</ref>。

1971年7月，第一次全国体育工作会议召开，时任总理的[[周恩来|周恩来]]代表国务院对国家体委召开第一次体工会进行批示，称“总的来说，毛主席的革命路线在广大群众和多数干部中还是占主导地位的，成绩是主要的”。这使得体育事业的定义产生变化，从“修正主义黑线”中划了出来<ref>{{Cite book|title=论中国现代普通高校体育制度的变迁|last=李冬梅著|first=|publisher=北京：北京体育大学出版社|year=2009.02|isbn=978-7-5644-0133-7|location=|pages=68}}</ref>。[[九一三事件|九一三事件]]爆发后，中国政治局势出现了转机。同年9月，周恩来主持中央工作，他对国家体委军管会领导做了批评后，提名由[[王猛_(少将)|王猛]]负责领导国家体育运动委员会；国家体委脱离[[总参谋部军训部|总参谋部军训部]]管辖，又恢复成为国务院的一个部委<ref>{{Cite book|title=北京体育大学校史 1953-2003|last=《校史》编委会编|first=|publisher=北京：北京体育大学出版社|year=2003.10|isbn=7-81100-028-8|location=|pages=68}}</ref>。1972年，很多体育运动员逐步返回原单位，国家运动队的基本队伍得以保留<ref>{{Cite book|title=艰难的起点与落点的辉煌 中国体坛备忘录|last=谷世权编著|first=|publisher=长春：吉林教育出版社|year=1994.03|isbn=7-5383-2142-X|location=|pages=124-125}}</ref>。各级革委会相继撤销了体委军管组，成立了体育局；体委系统组织机构逐步恢复，开始行使管理职能<ref>{{Cite book|title=竞技体育强国之路|last=许敏雄著|first=|publisher=北京：[[光明日报出版社|光明日报出版社]]|year=2012.10|isbn=978-7-5112-2998-4|location=|pages=58-60}}</ref>。

== 撤销 ==
文化大革命结束后，1979年2月12日，国家体委召开有4000多人的群众大会，会上宣读了中央批准的国家体委党组关于建议撤销1968年5月12日对全国体育系统的命令的报告（《关于建议撤销中发（68）71号文件的报告》<ref>{{Cite book|title=北京体育大学校史 1953-2003|last=《校史》编委会编|first=|publisher=北京：北京体育大学出版社|year=2003.10|isbn=7-81100-028-8|location=|pages=78}}</ref>）。会上，宣布为受打击、迫害的荣高棠、[[李達_(哲學家)|李达]]彻底平反，恢复名誉；同时宣布为受株连的刘仁等彻底平反，恢复名誉<ref>{{Cite book|title=新中国体育60年|last=[[熊晓正|熊晓正]]，[[钟秉枢|钟秉枢]]主编|first=|publisher=北京：[[北京体育大学出版社|北京体育大学出版社]]|year=2010.02|isbn=978-7-5644-0349-2|location=|pages=160}}</ref>。中共中央政治局委员、国务院副总理[[余秋里|余秋里]]、[[陈锡联|陈锡联]]，国务院副总理[[王任重|王任重]]出席主持大会<ref>{{Cite book|title=中国改革开放20年纪事|last=[[董兆祥|董兆祥]]，[[彭小华|彭小华]]主编|first=|publisher=上海：[[上海人民出版社|上海人民出版社]]|year=1998.12|isbn=7-208-03047-2|location=|pages=22}}</ref>。

1980年，《[[中国体育报|体育报]]》评选出1979年中国十大体育新闻，其中撤销五·一二命令是“十大体育新闻”第一<ref>{{Cite book|title=体育百科大全 39 新闻 奖励 组织 团体|last=吴兆祥|first=|publisher=合肥：[[安徽人民出版社|安徽人民出版社]]|year=1998.08|isbn=7-212-01571-7|location=|pages=20-22}}</ref>。
