'''五指山山脈'''是[[台灣|台灣]][[加里山山脈|加里山山脈]]一條支脈，一般通稱'''五指山列'''，或'''五指山系'''，主要為東北東－西南西走向，東起[[基隆|基隆]]海岸，西至[[台北|台北]][[圓山|圓山]]一帶沒入[[基隆河|基隆河]]，大約是新生高架橋、中山北路交會一帶。全長約25公里，東寬西狹，最寬有8.5公里，呈一楔形之山脈。<ref>楊貴三、沈淑敏（民99）。台灣全志，卷二，土地志，地形篇。南投市：台灣文獻館。</ref>

== 範圍 ==
五指山山脈東起[[基隆|基隆]]海岸，西至[[台北|台北]][[圓山|圓山]]一帶沒入[[基隆河|基隆河]]，大約是新生高架橋、中山北路交會一帶。全長約25公里，東寬西狹，最寬有8.5公里。山脈之北緣有[[崁腳斷層|崁腳斷層]]，北坡呈狹小之陡崖，南坡呈寬闊之緩坡。

== 地質 ==
本區地質主要有五指山層及大寮層，五指山層形成於[[新生代|新生代]][[漸新世|漸新世]]，為大台北地區古老之地層，其主要岩性特徵為含有煤層的白[[砂岩|砂岩]]夾灰黑色[[頁岩|頁岩]]<ref>{{cite web |url=http://geo3w.ncue.edu.tw/bsrapage/geoscience/geology/tb0403.htm |title=存档副本 |accessdate=2012-11-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20160529124126/http://geo3w.ncue.edu.tw/bsrapage/geoscience/geology/tb0403.htm |archivedate=2016-05-29 }} 西部麓山帶地質區（新第三紀沉積盆地）</ref>，且[[砂岩|砂岩]]與[[頁岩|頁岩]]之[[硬度|硬度]]懸殊，故[[層階地形|層階地形]]發達。<ref>楊貴三、沈淑敏（民99）。台灣全志，卷二，土地志，地形篇。南投市：台灣文獻館。</ref>大寮層則於[[新生代|新生代]][[中新世|中新世]]形成，主要為厚層細粒[[砂岩|砂岩]]、[[頁岩|頁岩]]及砂頁岩互層。[[崁腳斷層|崁腳斷層]]通過兩層之間，五指山層位於上盤，大寮層位於下盤。<ref>台北市政府工務局大地工程處（民97）台北小溪頭石頭厝-地質解說牌</ref>
