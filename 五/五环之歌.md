《'''五环之歌'''》是一首自2010年代起在中国流行的网络歌曲。歌词据信由一名可能姓杨的不具名作者于2009年发表在[[饭否|饭否]]上，<ref>[http://www.ihowandwhy.com/z/如何看待不明真相的上海彩虹合唱团误称《五环之歌》的作词者是岳云鹏老师？ 如何看待不明真相的上海彩虹合唱团误称《五环之歌》的作词者是岳云鹏老师？]{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>旋律则来自[[蒋大为|蒋大为]]演唱的《牡丹之歌》。<ref name = "gcc">[http://www.guancha.cn/video/2016_04_11_356598.shtml 上海彩虹室内合唱团北京音乐会表演《五环之歌》]，观察者网</ref>2011年，[[德云社|德云社]]相声演员[[岳云鹏|岳云鹏]]在一档相声中首次演唱了《五环之歌》，使之迅速成为话题。有多名歌手曾经在商业演出中[[翻唱|翻唱]]过本曲，并收录在至少3个专辑中。《五环之歌》有非官方的北京市市歌之称。<ref name ="doo">[http://dsj.voc.com.cn/article/201604/201604121027335732.html 《五环之歌》是北京市市歌吗？]，《经略》，纪武</ref>

== 现实的反响 ==
虽然本曲因岳云鹏的表演而闻名，实际上词曲均非岳云鹏原创。但公众多将本曲误会为岳云鹏作词，<ref name = "gcc"/>甚至曾经产生其他翻唱者侵权的误会。<ref>[http://ent.163.com/14/0729/11/A2AMKUGA00031H0O.html 王麟新“神曲”《五环之歌》被指抄袭岳云鹏]，网易娱乐</ref>至于使用的《牡丹之歌》的旋律，据岳云鹏表示已经得到蒋大为的口头[[授权|授权]]。<ref>[http://ent.sina.com.cn/z/v/2016-07-07/doc-ifxtwiht3292216.shtml 岳云鹏谈《五环之歌》授权 原唱:随便改]，新浪娱乐</ref>

《五环之歌》具有[[咏叹调|咏叹调]]的旋律和无厘头的歌词，因此受到欢迎。曾经演唱过本曲的毛阿敏表示初次听到《五环之歌》时被“逗得乐到”，觉得有一种"死皮赖脸的感觉"。<ref name ="mam"/>有观点认为《五环之歌》的流行代表了北京市民的无奈情绪。<ref name ="doo"/>[[北京市|北京市]]近年来面对人口无序增长，城市快速扩张和[[北京交通|交通状况恶化]]的情况。虽然官方喜欢推广歌功颂德式的城市歌曲，但相比来说，《五环之歌》里描述的交通拥堵和城市扩张导致的焦虑心态才是北京市民每天面对的日常。因此本曲的流行，某种意义上是处于弱势的北京市民面对无力改变之现状的一种无风险和低姿态的反抗。<ref>[http://news.ifeng.com/a/20160412/48436221_0.shtml 《五环之歌》怎么成“北京市歌”]，凤凰评论，江夜雨</ref>

2015年7月，电影《[[煎饼侠|煎饼侠]]》使用一版《五环之歌》作为推广曲、插曲<ref>[http://ent.sina.com.cn/m/c/2015-07-01/doc-ifxesfuc3463020.shtml 《煎饼侠》推广曲《五环之歌》MV首曝光]，新浪娱乐</ref>。

2016年時，岳雲鵬在此曲基礎上改編《新五環之歌》，由[[美团网|美团网]]將其作為[[廣告曲|廣告曲]]。2018年，《牡丹之歌》版權所有者北京眾得文化傳播有限公司將岳雲鵬、廣告製作公司以及美團網運營公司三方告上法庭，要求立即停止侵權並賠償經濟損失50萬人民幣。<ref>{{Cite web|url=http://ent.sina.com.cn/y/yneidi/2018-06-20/doc-iheauxwa0641280.shtml|title=岳云鹏《新五环之歌》被指侵权 原告求偿50万|accessdate=2018-06-20|date=2018-06-20|work=新浪娱乐}}</ref>

2016年8月，[[首都地区环线高速公路|北京七环路]]（即G95高速公路）全线通车。因这首歌的歌词中提到北京“终有一天”会“修到七环”，此事被一些媒体调侃为预言成真。<ref>[https://news.sina.com.cn/c/2018-08-20/doc-ihhxaafy9684744.shtml 还在唱“五环之歌”？“北京大七环”来了(图)] - [[新浪网|新浪网]]</ref><ref>[https://www.chinatimes.com/cn/realtimenews/20180821001413-260409 北京大七环全线通车　神曲「五环之歌」再次爆红] - [[中时电子报|中时电子报]]</ref>
