'''五箇山'''（ごかやま）是[[富山縣|富山縣]]西南端的-{[[南砺市|南砺市]]}-的舊[[平村_(富山縣)|平村]]、舊[[上平村_(富山縣)|上平村]]、舊[[利賀村|利賀村]]所構成的地域。

== 地名由來 ==
赤尾谷、上梨谷、下梨谷、小谷、利賀谷的5個[[谷|谷]]所組成的「五箇谷間」，轉變為「五箇山」的[[地名|地名]]。

== 歷史 ==
傳說為[[平家落人|平家落人]]定住之地。1183年，[[源平合戰|源平合戰]]時，木曾義仲（[[源義仲|源義仲]]）和[[平維盛|平維盛]]（[[平清盛|平清盛]]之孫）在[[富山縣|富山縣]]和[[石川縣|石川縣]]縣境的[[俱利伽羅峠|俱利伽羅峠]]發生戰爭（[[俱利伽羅峠之戰|俱利伽羅峠之戰]]）。當時，義仲用火牛之戰法大勝平家。殘黨成為[[落人|落人]]逃亡隱匿在五箇山。

== 地域特色 ==
五箇山是世界有數的[[豪雪地帶|豪雪地帶]]，因為此風土而產生傾斜度急大的屋頂的[[合掌造|合掌造]]家屋。現在在-{南砺市}-（舊[[平村_(富山縣)|平村]]）的[[相倉|相倉地區]]、同市（舊[[上平村_(富山縣)|上平村]]）的[[菅沼_(南砺市)|菅沼地區]]也還保存有合掌造集落，1970年12月4日，作為「越中五箇山相倉集落」「越中五箇山菅沼集落」指定為國之[[史跡|史跡]]。1994年，選定為[[重要傳統的建造物群保存地區|重要傳統的建造物群保存地區]]。1995年12月，和隣接的[[岐阜縣|岐阜縣]][[大野郡_(岐阜縣)|大野郡]][[白川村|白川村]]的[[白川鄉|白川鄉]]（[[荻町_(白川村)|荻町地區]]）以「[[白川鄉與五箇山的合掌造聚落|白川鄉・五箇山的合掌造集落]]」共同登錄為[[世界遺産|世界遺産]]。2009年3月，[[法國|法國]][[米其林|米其林]]發行的旅行指南日本編「米其林・綠色指南・日本」選為最高的3星觀光地，是日本國內17處3星觀光地之一。相倉集落・菅沼集落被選為2星觀光地。
