{{japanese|japanese=迦具土|kana=カグツチ|romaji=Kagutsuchi}}
'''迦具土'''（{{lang|ja|カグツチ}}）乃是[[日本神話|日本神話]]中的[[火神|火神]]，《[[古事記|古事記]]》記錄成'''火之迦具土神'''（{{lang|ja|ヒノカグツチノカミ}}）、'''火之夜藝速男神'''（{{lang|ja|ヒノヤギハヤヲノカミ}}）、'''火之炫毘古神'''（{{lang|ja|ヒノカカビコノカミ}}）等，而《[[日本書紀|日本書紀]]》則記成'''軻遇突智'''、'''火產靈'''等；另有別名'''火之迦倶槌神'''（{{lang|ja|ホノカグツチノカミ}}）、'''火產靈神'''（{{lang|ja|ホムスビノカミ}}）、'''火牟須比命'''（{{lang|ja|ホムスビノミコト}}）等。

== 概要 ==
=== 日本書紀之記載 ===
根據《日本書紀》卷第一所載之第二種說法，伊奘冉尊生火神'''軻遇突智'''時被燒死，臨終之際臥生土神[[波邇夜須毘古神與波邇夜須毘賣神|埴山姬]]（{{lang|ja|ハニヤマヒメ}}）及水神[[彌都波能賣神|罔象女]]。軻遇突智娶埴山姬，生下[[和久產巢日神|稚產靈]]，此神頭上生[[蠶|蠶]]與[[桑树|桑]]，[[肚脐|臍中]]生五穀。第三種說法謂伊奘冉尊生'''軻遇突智'''時，為子所焦而神退（亦云神避）。神退之時生水神罔象女及土神埴山姬，又生天吉葛（一云與曾豆羅）。第四種說法則曰伊奘冉尊生火神'''軻遇突智'''之時，悶熱懊惱而嘔吐，穢物化作[[金山毘古神|金山彥]]、[[尿|小便]]化作罔象女、[[糞便|大便]]化為埴山媛。

第六種說法則謂[[伊奘諾尊|伊奘諾尊]]拔所帶[[十握劍|十握劍]]斬'''軻遇突智'''為三段：劍刃垂血化為天安河邊之五百箇磐石，即[[經津主神|經津主神]]之祖；劍鐔垂血化為[[甕速日神|甕速日神]]（亦曰甕速日命）、[[樋速日神|熯速日神]]（亦曰熯速日命），其中甕速日神乃[[建御雷|武甕槌神]]之祖；劍鋒垂血化作[[石拆神根拆神|磐裂神、根裂神]]、[[石筒之男神|磐筒男命]]（或云磐筒男命與磐筒女命）；劍頭垂血化為[[闇淤加美神|闇龗]]、闇山祇、[[闇御津羽神|闇罔象]]。

第七種說法提及[[伊奘諾尊|伊奘諾尊]]拔[[天之尾羽張|劍]]斬殺'''軻遇突智'''成三段：一段為雷神、一段為[[大山津見|大山祇神]]、一段為[[闇淤加美神|高龗]]；又云斬'''軻遇突智'''時其血液染紅了天八十河中的五百箇磐石，因而化成神：[[石拆神根拆神|磐裂神、根裂神]]、兒[[石筒之男神|磐筒男神]]、次為磐筒女神、兒[[經津主神|經津主神]]。而第八種說法則謂伊奘諾尊斬軻遇突智命爲五段，各化為五山祇：頭部化為[[大山津見|大山祇]]、身體化為中山祇、手部化為麓山祇、腰肢化為正勝山祇、足部化為鵪山祇<ref>原字作左酓右隹，音安，受限於[[Unicode|Unicode]]編碼無法顯示，故以異體字「鵪」代替。</ref>。

=== 古事記之記載 ===
依《古事記》上卷之記述，[[伊奘諾尊|伊邪那岐命]]和[[伊奘冉尊|伊邪那美命]]生下[[鳥之石楠船神|鳥之石楠船神]]，亦名天鳥船；次生[[大宜都比賣|大宜都比賣]]神以及'''火之夜藝速男神'''，又名'''火之炫毘古神'''、'''火之迦具土神'''。伊邪那美命生下後者時，[[女陰|下陰]]遭灼傷而臥病在床。其嘔吐物生成[[金山毘古神|金山毘古神、金山毘賣神]]；[[糞便|糞便]]生成[[波邇夜須毘古神與波邇夜須毘賣神|波邇夜須毘古神、波邇夜須毘賣神]]；[[尿|尿液]]生成[[彌都波能賣神|彌都波能賣神]]、[[和久產巢日神|和久產巢日神]]等二神，後者之女名曰[[豐宇氣毘賣神|豐宇氣毘賣神]]；不久伊邪那美命身故。

盛怒之下的伊邪那岐命拔出[[天之尾羽張|十拳劍]]，砍下了迦具土的頭顱，其[[血液|血液]]從劍上落到岩石。其中由劍鋒落下的血生成[[石拆神根拆神|石拆神、根拆神]]和[[石筒之男神|石筒之男神]]；劍身落下的血生成[[甕速日神|甕速日神]]、[[樋速日神|樋速日神]]和[[建御雷|建御雷之男神]]（亦名建布都神、豐布都神）；劍柄落下的血生成[[闇淤加美神|闇淤加美神]]和[[闇御津羽神|闇御津羽神]]。而迦具土的[[屍體|屍體]]亦化為眾多山神：頭化為[[大山津見|正鹿山津見神]]，胸部化為[[大山津見|淤縢山津見神]]，腹部化為[[大山津見|奧山津見神]]，[[陽具|陽具]]化為[[大山津見|闇山津見神]]，左手化為[[大山津見|志藝山津見神]]，右手化為[[大山津見|羽山津見神]]，左腳化為[[大山津見|原山津見神]]，右腳化為[[大山津見|戶山津見神]]。所斬之刀名謂[[天之尾羽張|天之尾羽張]]，亦名伊都之尾羽張。

== 解說 ==
茲根據神名的含義解釋如下：
# 火之迦具土神：「迦具（{{lang|ja|カグ}}）」同「輝く」，為火光搖曳之明亮貌，「土（{{lang|ja|ツチ}}）」裡的「ツ」為[[連詞|連接詞]]、「チ」為神靈之意<ref>參看{{ja}}[http://www.zb.em-net.ne.jp/~kiokunomori/html/mukashi/shinnmei/shinnmei-3.html 神名釈義　名義ミニ事典]，「火之迦具土の神 （ヒノカグツチ）」的段落。</ref>。
# 火之夜藝速男神：「夜藝（{{lang|ja|ヤギ}}）」為「燒」，乃燃燒之意，「速（{{lang|ja|ハヤ}}）」表火勢迅速猛烈，「男神（{{lang|ja|ヲノカミ}}）」則點出此神明之[[性別|性別]]<ref>參看{{ja}}[http://www.zb.em-net.ne.jp/~kiokunomori/html/mukashi/shinnmei/shinnmei-3.html 神名釈義　名義ミニ事典]，「火之夜芸速男の神 （ヒノヤギハヤヲ）」的段落。</ref>。
# 火之炫毘古神：「炫（{{lang|ja|カカ}}）」乃「輝く」，即明亮的意思；「毘古神」（{{lang|ja|ビコノカミ}}）也可寫作「彥神」，表男神之意<ref>參看{{ja}}[http://www.zb.em-net.ne.jp/~kiokunomori/html/mukashi/shinnmei/shinnmei-3.html 神名釈義　名義ミニ事典]，「火之炫毘古の神 （ヒノカカビコ）」的段落。</ref>。

== 信仰 ==
迦具土神既為火神，在[[日本|日本]][[民间宗教|民間信仰]]中祂與[[灶君|竈神]]息息相關。此外，金屬鍛冶業、製陶業等與火相關的行業也祭拜此神祇。位於[[京都府|京都府]][[京都市|京都市]][[右京區|右京區]]的[[愛宕神社|愛宕神社]]乃全日本9百餘座愛宕神社的總本社，將迦具土神視為防火之神。故主祭迦具土的[[神社|神社]]除8百餘座之秋葉神社、9百餘座之愛宕神社、5座之野野宮神社外，主要計有下列：
* 火男火賣神社（[[大分縣|大分縣]][[別府市|別府市]]）：主祭火之加具土命、火燒速女命等男女二神。
* [[秋葉山本宮秋葉神社|秋葉山本宮秋葉神社]]（[[靜岡縣|靜岡縣]][[濱松市|濱松市]]）
* [[愛宕神社|愛宕神社]]（[[京都府|京都府]][[京都市|京都市]][[右京區|右京區]]）：稱作'''迦遇槌命'''。
* 野野宮神社（京都府京都市右京區）

此外，有人認為[[島根縣|島根縣]][[安來市|安來市]]的意多伎神社祭神也跟迦具土有關。
