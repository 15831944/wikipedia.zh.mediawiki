'''匈牙利汤'''（{{lang-hu|Gulyás}}，{{IPA-hu|ˈɡujaːʃ||hu-gulyás.ogg}}）又译'''古拉希'''<ref name=dgb />、'''古拉什'''<ref>{{Cite web |url = http://go.huanqiu.com/guide/2016-06/9092566.html |title = 匈牙利“国汤”与中国有神奇渊源 |author = 陈恒富 |date = 2016-06-28 |publisher = 环球时报 |language = zh-cn |accessdate = 2018-05-14 }}</ref>、'''古亚士'''，是一种肉和蔬菜为主料加入[[辣椒粉|辣椒粉]]，具甜味的[[汤|汤]]或者[[浓汤|浓汤]]。<ref>{{cite book |first=Karoly |last=Gundel |title=Gundel's Hungarian cookbook |publisher=Corvina |location=Budapest |year=1992 |pages= |isbn=963-13-3600-X |oclc=32227400}}page 20</ref>

匈牙利汤最初是匈牙利草原牧民的传统饮食，起源于公元9世纪<ref name="Britannica">[http://books.google.co.uk/books?id=Pf6cAAAAQBAJ&pg=PA356&dq=Bulgaria,+Hungary,+Romania,+the+Czech+Republic,+and+Slovakia&hl=en&sa=X&ei=RgFJU7m_IbGV7AbR-QE&ved=0CDMQ6AEwAA#v=onepage&q=goulash&f=false Bulgaria, Hungary, Romania, the Czech Republic, and Slovakia], Britannica Educational Publishing, 2013, p. 94</ref>，现亦成为从[[波罗的海|波罗的海]]到[[黑海|黑海]]周边，[[东欧|东欧]]、[[中欧|中欧]]很多国家在内的广大地区的日常饮食之一。其名称Gulyás一词原本意为“放牛的人”，匈牙利汤则被称作“gulyáshús”（意為「牧牛人烹制的肉」），如今gulyás一词既指牧牛人，也指这种匈牙利汤。

匈牙利汤也被[[肉汤共产主义|肉汤共产主义]]以及[[中苏交恶|中苏交恶]]时期中国[[毛泽东|毛泽东]]的诗词《念奴娇·鸟儿问答》（“土豆烧熟了，再加牛肉”）{{noteTag|1=“土豆烧牛肉”这一译法初见于1958年，实为误译<ref name=dgb>{{cite web|url=http://news.takungpao.com/paper/q/2013/0416/1549434.html|title=“土豆烧牛肉共产主义”由来|publisher=[[大公报|大公报]]|date=2013-04-16|accessdate=2014-11-22}}</ref>}}提到，指[[冷战|冷战]]时期比较注重民生的东欧共产主义模式。
