'''囉哩人'''是[[多姆人|多姆人]]的一個分支。目前主要生活在[[中亞|中亞]]的[[塔吉克斯坦|塔吉克斯坦]]、[[烏茲別克斯坦|烏茲別克斯坦]]、[[吉爾吉斯斯坦|吉爾吉斯斯坦]]。語言和[[吉普賽語|吉普賽語]]接近。信[[伊斯蘭教|伊斯蘭教]]。其内部社會對外族封閉。

中國史書對他的記載最早出現在《[[元史|元史]]》，稱之爲“'''囉哩回回'''”。還有中國史書又稱之爲'''“羅里”'''、'''“剌里”'''、'''“盧里”'''、'''“柳里”'''等。

== 中國的囉哩人 ==
元代就已有囉哩人從中亞進入中國，《元史·刑法志·雜犯》有：{{quote|諸'''囉哩回回'''為民害者，從所在有司禁治。}}
《元史·[[元成宗|成宗]]紀》有：{{quote|[[大德|大德]]六年九月丁未，中書省臣言：‘'''羅里'''等擾民，宜依例决遣置屯田所’。從之。}}
《元史·[[元順帝|順帝]]紀》有：{{quote|是月，回回'''剌里'''五百餘人渡河寇掠解、吉、隰等州。}}
《元史·[[唵木海|唵木海]]傳》有：{{quote|癸丑，從宗王旭烈兀征'''剌里西番'''}}

[[清代|清代]][[王士禛|王士禛]]《[[池北偶谈|池北偶谈]]·谈献四·秦襄毅公年谱》也記載了該族内部社會封閉，以至於[[近親結婚|近親結婚]]：

{{quote|秦州有'''囉哩'''戶，乃回回别種。漢人不與通婚姻，自相嫁娶，有以兄弟娶姊妹者，有以姑姨配甥侄者。予訪得清水、秦安等縣，亦有囉哩，乃移文各縣，令其共为婚姻。}}

== 吉爾吉斯斯坦的囉哩人 ==

囉哩回回住在吉爾吉斯斯坦南部的[[奧什州|奧什州]]。他們的生活水平很低，很多人沒有身份證件。兒童沒有機會接受母語教育。囉哩社區現在努力提高自己的生活水平、保留自己的文化。

== [[俄羅斯|俄羅斯]]的囉哩人 ==
[[Image:Lyuli_woman_with_child_in_Kazan,_Russia2.JPG|thumb]]街頭的一名囉哩婦女]]

1990年代初，有囉哩人開始遷入俄羅斯的城市，以火車站和市場附近為落腳點。開始俄羅斯一度誤以爲他們是來自塔吉克的難民，依據民族服飾把他們認作[[塔吉克人|塔吉克人]]或[[烏茲別克人|烏茲別克人]]。

俄羅斯[[吉普賽人|吉普賽人]]認爲囉哩人和自己不同，不過俄羅斯把囉哩人認作吉普賽人。
