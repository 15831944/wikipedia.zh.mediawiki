'''翟志剛'''（{{bd|1966年|11月21日|}}），男，汉族，[[中华人民共和国|中国]][[黑龙江省|黑龙江省]][[齐齐哈尔市|齐齐哈尔市]][[龍江縣|龍江縣]]人。他是[[神舟七號|神舟七號]][[太空船|太空船]]三名正選[[航天员|航天员]]之一，他的隊友還有[[劉伯明_(宇航員)|劉伯明]]和[[景海鵬|景海鵬]]。

翟志剛於2008年9月25日在[[甘肅|甘肅]][[酒泉|酒泉]]隨[[神七|神七]]發射升空，任务代号01，並在9月27日作[[太空漫步|太空漫步]]，成為第一位進行太空漫步的中国航天员。

== 早年生活 ==
1966年10月10日，翟志刚出生在一个[[黑龙江省|黑龙江省]][[齐齐哈尔市|齐齐哈尔市]][[龙江县|龙江县]]的小乡村。

他于1985年6月进入[[中国人民解放军|中国人民解放军]]，1989年毕业于[[空军第三飞行学院|空军第三飞行学院]]，曾任[[中国人民解放军空军|空军]]航空兵[[战斗机|战斗机]][[飞行员|飞行员]]。他曾任空军试训中心某团飞行教员，有1000多小时的飞行经验，是空军一级飞行员，当时已是[[中校|中校]]。<ref>{{cite web |url = http://www.022net.com/2008/9-22/444958323049928.html |title = 航天英雄翟志刚刘伯明天津杨村起飞 |publisher = 人民网·天津视窗 |language = zh-cn |accessdate = 2010-01-27 |deadurl = yes |archiveurl = https://web.archive.org/web/20100405070547/http://www.022net.com/2008/9-22/444958323049928.html |archivedate = 2010-04-05 }}</ref>

== 航天员生涯 ==
1996年，翟志刚被选拔参加航天计划测试，後来经过重重选拔，於1998年成为第一批[[航天员|航天员]]14人中的一员，从空军改隶属总装，改为陆军。[[神舟五号|神舟五号]]计划中，他是首飞梯队3名航天员中的一名，不过最终[[杨利伟|杨利伟]]被任命驾驶飞船，而翟志刚在几次测试结束後排名第二，超过[[聂海胜|聂海胜]]。[[神舟六号|神舟六号]]计划中，翟志刚是飞行成员组6名宇航员中的一名。《[[大公报|大公报]]》认为翟志刚和聂海胜将会是第一人选，因为他们曾参与过神舟五号计划。不过，翟志刚与[[吴杰_(飞行员)|吴杰]]被分入一组训练，而[[费俊龙|费俊龙]]和聂海胜最终被选拔乘坐神舟六号。

=== 神舟七号飞行任务 ===
{{main|神舟七号}}
神舟七号飞行任务中，翟志刚（代号01）与[[刘伯明_(宇航员)|劉伯明]]（02）、[[景海鹏|景海鹏]]（03）被选拔为神舟七号的正式航天员乘组，而翟志刚於2008年9月17日被任命为任务指令长。2008年9月25日，北京时间21:10:04，他们乘坐飞船升入太空，这是中国第一次有3名航天员一起执行任务的航天计划，也是中国第三次[[载人航天|载人航天]]任务。<ref> CCTV1 live telecast </ref><ref> http://www.chinadaily.com.cn/china/2008-09/26/content_7060750.htm </ref>2008年9月27日，翟志刚成为第一名[[舱外活动|太空漫步]]的中国人，并且是完全出舱。帮助其出舱的航天员刘伯明处於舱内，但上身探出舱外，跨在舱门上。北京时间18:25:55，翟志刚进入舱门，成功地完成了太空漫步。<ref> CCTV4 live telecast </ref> 翟志刚所穿的是中国自主研制的[[飞天号航天服|飞天号航天服]]，而刘伯明所穿的是俄罗斯研制的[[海鹰航天服|海鹰航天服]]（Orlan-M space suit）。

2013年，获晋升为少将军衔<ref>{{cite news|title=13年间中国有10位航天员“飞天”，至少6位都成了少将|url=http://www.thepaper.cn/newsDetail_forward_1544363|accessdate=2016-10-16|newspaper=澎湃新闻}}</ref>。

== 家庭 ==
翟志剛出身於[[農民|農民]]家庭，是-{幺}-子，有三位姐姐和两個哥哥。翟志剛已婚，育有一子。[[妻子|妻子]]張淑靜在航天食物研究部門工作。

== 爱好 ==
他爱好[[书法|书法]]、跳[[交谊舞|交谊舞]]、[[玩具|玩具]]，喜欢模仿表演[[赵本山|赵本山]]的[[小品|小品]]。

== 趣事 ==
2008年9月27日，翟志刚首次太空漫步时的开场白“我已出舱，感觉良好”，被国内股民篡改为“我已出仓，感觉良好”的股市离场口吻，一时成坊间趣谈。<ref>{{cite news|url=http://qjwb.zjol.com.cn/html/2009-04/10/content_4679843.htm|title=我已出仓，感觉良好|date=2009年4月10日|publisher=钱江晚报|language=zh-cn|quote=去年下半年，沪深大盘一落千丈，不少此前满仓的股民纷纷割肉离场，于是股民们将这句话略做修改，将“舱”改为“仓”，用来自我解嘲。例句如：“我已出仓，感觉良好；少亏十万，胜过捂牢；持币过节，来年再炒”，“向全国股民们汇报，我已出仓，感觉良好！”。}}</ref>
