'''翟天临学术风波'''，又名'''翟天临学术门'''，是指在2019年2月初，[[中国大陆|中国大陆]]男演员[[翟天临|翟天临]]因涉嫌学术不端而引发的一起争议事件。在这起事件中，翟天临除了被曝抄袭论文之外，其高考成绩也遭到网友质疑。事件在网络上发酵数日后，翟天临本人发表致歉信，并申请退出[[北京大学|北大]][[博士后|博士后]]科研流动站的相关工作<ref name=A>{{cite web |url=http://ent.sina.com.cn/s/m/2019-02-14/doc-ihqfskcp5149791.shtml |language= |title=翟天临发布致歉信：申请退出北大博士后相关工作 |publisher=新浪娱乐 |date=2019-02-14|accessdate= |archive-url=https://web.archive.org/web/20190214103631/http://ent.sina.com.cn/s/m/2019-02-14/doc-ihqfskcp5149791.shtml |archive-date=2019-02-14|dead-url=no }}</ref>。之后，翟天临的电影学博士学位被[[北京电影学院|北京电影学院]]撤销。<ref name=":0">{{Cite web|title=北京电影学院：撤销翟天临博士学位，取消陈浥博导资格_教育家_澎湃新闻-The Paper|url=https://www.thepaper.cn/newsDetail_forward_3007076|work=www.thepaper.cn|accessdate=2019-03-31}}</ref>

== 发展 ==
=== 翟天临被曝学术不端 ===
网友先后使用知网等查重工具对翟2018年8月发表在《广电时评》杂志上的论文进行查重，发现全文字数不足3000而且出现大段抄袭，除去本人已发表文字外复制比超过40%<ref>{{Cite web|title=翟天临论文查重达40%惹议，原文作者：要我打假？{{!}}翟天临{{!}}论文{{!}}白孝文_新浪网|url=https://k.sina.com.cn/article_1887344341_707e96d501900jgld.html|work=k.sina.com.cn|date=2019-02-10|accessdate=2019-02-15|last=观察者网|archive-url=https://web.archive.org/web/20190215235620/https://k.sina.com.cn/article_1887344341_707e96d501900jgld.html|archive-date=2019-02-15|dead-url=no}}</ref>。其中被抄袭最多的是黄立华教授2006年发表在《黄山学院学报》上的文章，亦引得原作者抨击<ref>[http://ent.163.com/19/0209/19/E7JLS0L400038FO9.html 实锤？翟天临论文涉嫌抄袭 原作者在朋友圈发声了] {{Wayback|url=http://ent.163.com/19/0209/19/E7JLS0L400038FO9.html |date=20190209181917 }}网易娱乐 2019-02-09 19:50:02 2019-2-9 21:10:40</ref>。
2月14日，有媒体发现翟天临撰写的北京电影学院硕士学位论文《“英雄”本是“普通人”——试论表演创作中的英雄形象与人性》在知网的查重结果中，重复字数过万，重复比高达36.2%。其中，单篇最大文字复制比达到5.5%，来源为[[陈坤|陈坤]]的毕业论文《性格化表演之我见》，已经达到知网“疑似剽窃观点”和“疑似剽窃文字表述”的指标。<ref>{{cite news|language = zh-cn|author = 周慧晓婉|author2 = 徐美琳|url = http://www.bjnews.com.cn/ent/2019/02/13/546769.html|title = 翟天临疑似抄袭陈坤论文，陈坤团队表示不回应|publisher = 新京报|date = 2019-02-13|accessdate = 2019-03-25|archive-url = https://web.archive.org/web/20190325041551/http://www.bjnews.com.cn/ent/2019/02/13/546769.html|archive-date = 2019-03-25|dead-url = no}}</ref>

在被曝抄袭论文之后，翟天临先前的采访报道以及博文也被网友扒出。网友发现，翟天临曾自曝[[普通高等学校招生全国统一考试|高考]]成绩在580多分，但又说自己的数学成绩只有19分，而文综成绩相当高，接近满分。网友质疑他的高考成绩并表示，在数学仅有19分的情况下总成绩达到580多分，文综总分需高达270以上，语文和英语平均分也要分别达到145分；而当年[[山东省|山东省]]文科状元的文综总成绩只有267分{{NoteTag|翟天临是在山东参加的高考。}}。此外，翟天临亦曾表示，自己当年的高考成绩过了一本线，也刚好达到了[[国际关系学院|国际关系学院]]的录取分数线，但自己最后还是选择就读[[北京电影学院|北京电影学院]]。有网友据此又查阅了网上资料，发现当年[[国际关系学院|国际关系学院]]在山东省的文科录取分数线是641分<ref>{{Cite web|title=翟天临高考成绩再遭质疑 数学19分总分却580+|url=http://ent.163.com/19/0210/16/E7LSSHI200038FO9.html|work=ent.163.com|date=2019-02-10|accessdate=2019-02-15|last=网易|archive-url=https://web.archive.org/web/20190212130718/http://ent.163.com/19/0210/16/E7LSSHI200038FO9.html|archive-date=2019-02-12|dead-url=no}}</ref>。网友通过查阅2006年北京电影学院表演系在山东录取的学生数据得知，当年仅录取两名山东籍学生：翟天临与马晓灿，两人的分数分别为348分和402分。结合马晓灿曾以全校第一的成绩考入表演系研究生等情况，网友推断翟天临高考分应为348分<ref>{{Cite web|title=翟天临真实高考成绩被扒 疑似348分远低一本线{{!}}翟天临{{!}}高考成绩{{!}}博士_新浪娱乐_新浪网|url=http://ent.sina.com.cn/s/m/2019-02-14/doc-ihrfqzka5659449.shtml|work=ent.sina.com.cn|date=2019-02-14|accessdate=2019-02-14|last=实习生懿娴|archive-url=https://web.archive.org/web/20190214075627/http://ent.sina.com.cn/s/m/2019-02-14/doc-ihrfqzka5659449.shtml|archive-date=2019-02-14|dead-url=no}}</ref>。
