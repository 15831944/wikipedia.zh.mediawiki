'''護照'''或'''護證'''是一個[[國家|國家]]或[[地區|地區]]的[[政府|政府]]发放給本國[[公民|公民]]或[[國民|國民]]的一種[[旅行|旅行]][[证件|证件]]，用於證明持有人的[[身分標識方式|身分]]與[[國籍|國籍]]，以便其出入本國及在外國旅行，同時亦用於請求有關外國當局給予持照人通行便利及保護。

護照關係到在國外所受合法保護的權利與進入本籍國的權利。护照上通常有持有者的[[照片|照片]]、[[签名|签名]]、[[出生日期|出生日期]]、[[国籍|国籍]]和其它个人身分的证明。许多国家正在開發將[[生物识别|生物识别]]技术用於護照，以便能够更精确地确认护照的使用者是其合法持有人。

如今国际间旅行通常要求出示护照，但也有例外的情况。护照实际上仅仅是一种国际认可的用来对旅行者身份鉴定的手段。而这样的鉴定要求在很多情况下或针对某些旅行者可以免除。例如现在[[美国|美国]]公民凭[[駕駛執照|駕駛執照]]即可进入[[墨西哥|墨西哥]]，而[[欧盟|欧盟]]国民在欧盟内部旅行也不需要护照。 

同样的，护照在一个国家内部也可以被当成身份证件而使用。

[[國際民航組織|國際民航組織]]已经就护照的标准格式和特点制定了Doc 9303标准。 

最近有许多关于在护照中采用[[生物识别|生物识别]]技术以改进身份识别安全的讨论，美国在[[911|911]]事件后已经针对27个免签证国家要求启用含有此种技术的护照，它是通过内嵌非接触式[[芯片|芯片]]来儲存生物信息。

== 历史 ==
[[File:QingPassport.jpg|thumb]]護照]]
全世界最早出現的護照似乎是在在[[公元前|公元前]]450年，根據《[[聖經|聖經]]》《[[尼希米記|尼希米記]]》二章7節記載：「我又對王說，王若以為美，請賜我詔書，通知大河以西的諸省長准我經過，直到我抵達[[猶大王國|猶大]]」<ref>http://www.recoveryversion.com.tw/Style0A/026/read_List.php?f_BookNo=16&f_ChapterNo=2</ref>。[[波斯|波斯]]王發給[[巴勒斯坦|巴勒斯坦]]的一封詔書，要求沿途官員保障波斯官員的旅行安全。

以[[中國|中國]]历史上為例，不同历史时期有不同的称呼，例如称为[[符節|符]]、传、[[过所|过所]]、公验、[[度牒|度牒]]、路证、[[通关文牒|通关文牒]]、安全证书、通行证、暂住证等。到了[[清朝|清朝]]，根据通行证件的证明身份、提请边境关防检查机关予以“保护”和“关照”、给予通行的便利和必要的协助之功能，将“PASSPORT”一词，于1845年首次译为“护照”，并沿用至今。

{{fact|西方觀點，護照的[[西方|西方]]稱呼「Passport」並不是來於「seaport」（海港），而是來於[[中世纪|中世纪]]用於通過城牆大門（「porte」）的文書。在中世纪的[[欧洲|欧洲]]，一些文件可以由当地的统治机关颁发给旅行者，一般包括持有者被允许通过城镇的名单。这种制度延续到[[法国|法国]]，直到1860年。在这段时间，护照并不是旅行到海港的必须，这关系到[[自由貿易|自由貿易]]，但是从海港到内陆城填则需要护照。早期的护照经常(但不总是)包含持有者体貌特征的描述，照片是直到20世纪初才被添加上去的。}}

在[[第二次世界大战|二战]]后，[[國際聯盟|國際聯盟]]、[[联合国|联合国]]和[[国际民航组织|国际民航组织]]，颁布了在护照的设计和特征的指导标准，这些指导很大方面上形成了现代的护照。近幾年則有MRP（Machine Readable Passport, [[可機讀護照|可機讀護照]]）統一規定持照人資料頁在封面內側頁，並有可利用掃描方式讓電腦判讀的資料，以節省移民暨出入國管理人員查驗證照時的時間。

== 類型 ==
* '''普通護照'''：发放的对象是普通[[公民|公民]]。
* '''外交護照'''：发放的对象是[[外交官|外交官]]和外交代表、或者是根據某些国家法规而发放给该国的公务员。持有外交護照並不自動享有[[外交豁免权|外交豁免权]]。持有外交護照，同时在一些國家會给予不同於非外交護照的[[簽證|簽證]]禮遇。另外，大多數國家的元首到別國進行訪問時，若不持用護照，則會由該國外交部部長簽發「元首通行狀」供元首及其陪同出訪的配偶使用，此通行狀會記載訪問國家的國名及出訪事由，不需事先申請簽證，可獲禮遇免除通關及證照查驗程序，僅會由邀訪國另行蓋印以茲證明，因此在每次出訪前必需簽發一份新的元首通行狀，出訪完成後交由外交部歸檔列為[[政府檔案|政府檔案]]。<ref>{{cite web|url=http://www.mofa.gov.tw/webapp/fp.asp?xItem=23416&ctnode=1038|title=外交部表示：外交護照包括元首通行狀|accessdate=2010-05-26}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref>{{cite web|url=http://tp.50webs.com/page17e.htm|title=元首通行狀 首度曝光|accessdate=2010-05-27}}</ref>。
* '''公務護照'''：发放的对象是政府派出執行公務的雇員（或者技术和行政人员），因为他們不具有外交官資格或[[維也納公約|維也納公約]]认可的外交身分。持有公務護照，在一些國家會有不同於普通護照的簽證禮遇。
* '''特殊護照'''：发放的对象是高级政府官员和他们的随从，作为对其免除签证要求的官方许可。
* '''集體護照'''：在某些情況，如学校团体旅行，同團的所有兒童在旅行期间共用一本集體護照。
* '''[[永久居留證|永久居留證]]'''：某些國家發給没有公民身份的居民的身份證明及入境/居留許可。
* '''國內護照'''：某些國家為管控本國人口遷徙而發行使用。例如[[沙特阿拉伯|沙特阿拉伯]]的iqama，[[中國大陸|中國大陸]]的[[中华人民共和国户籍制度|户口登记制度]]，还有實行[[計劃經濟|計劃經濟]]的[[蘇聯|蘇聯]]，是一種為監控人口流動情況而發行的內部護照。
* '''加急護照'''或'''临时護照'''：发放给那些原先护照遗失或失窃但急需出国旅行的人。
* '''商务護照'''：发放给那些频繁旅行的人，这种护照的页数较普通护照的多。
[[File:UN-laissez-passer.jpg|thumb]]
* '''[[聯合國护照|聯合國通行證]]'''（Laissez-passer）：[[联合国|联合国]]和各专门机构根据《联合国特权和豁免公约》和《专门机构特权和豁免公约》向本机构的职员颁发一种具有护照效力的身份证件。有红皮通行证和蓝皮通行证两种，其效力分别相当于外交护照和公务护照。「Laissez-passer」是[[法文|法文]]「請予通行」之意（註：聯合國部分機構設立在使用法文的[[巴黎|巴黎]]和[[布魯塞爾|布魯塞爾]]，以及法國之[[常任理事國|常任理事國]]地位，法文亦為聯合國官方語言之一。）。
* '''家庭護照'''：是一种只发放给家庭（家长和儿童）而不是个人的护照，美国政府已经不再发放此种护照。
* '''[[难民旅行证|难民旅行证]]'''：某些国家的政府根据1951年7月28日联合国《关于难民地位的公约》第28条及1967年1月31日《难民地位议定书》的规定颁发给合法在其领土内居留的难民的旅行证，供持证人到发证国领土以外旅行。

== 国际民航组织标准 ==
[[国际民航组织|国际民航组织]](ICAO)提供了护照规格的标准化建议，如今被大部分国家政府所采用。护照的大小需要符合 ISO/IEC 7810 标准，即 125x88 毫米，B7 规格。

== 特別情況 ==

君主制国家的君主一般无护照，如[[英女王|英女王]][[伊莉莎白二世|伊莉莎白二世]]不需要護照和簽證就可以自由進出各個國家，因為英國護照是以英國女王陛下的名義簽發的，因此女王本人不需要護照。

=== 國際旅行不需護照的情況 ===
护照通常在国际旅游中都必须要用到，通常在跨国与国的边境时需要使用。但是有些时候，某些国家的公民出境到另外一个国家时，由于国与国订立了某些契约（如[[欧盟|欧盟]]），可能就无需出示护照，但可能会以其他身份证明文件来证明身份，如驾驶执照，个人身份证，护照上的签证。

=== 國內旅行需要護照的情況 ===
某些國家，例如實行或曾實行[[計劃經濟|計劃經濟]]的國家，為了監控人口流動或管控人民遷徙，要求人民在國內旅行或遷徙時，申請並持有國內護照。例如蘇聯。

由於[[英國|英國]]的[[殖民地|殖民地]]與國籍法複雜，在[[大不列顛|大不列顛]]與[[屬地|屬地]]之間，或英國各屬地之間旅行，也常需要護照。

1963年[[沙巴|沙巴]]和[[砂拉越|砂拉越]]与[[新加坡|新加坡]]和[[马来亚|马来亚]]共同参组[[馬來西亞聯邦|馬來西亞聯邦]]時，依據各方事前同意的決議，[[沙巴|沙巴]]和[[砂拉越|砂拉越]]兩州可保有原先獨立的邊境管制。[[西馬來西亞|西馬來西亞]]人民進入沙巴或砂拉越，以及兩州之間相互旅行，均需檢查護照，但沙巴、砂拉越人民進入[[西馬來西亞|西馬來西亞]]時則不需檢查護照。


=== {{flagicon|CHNML}} 中國大陸 ===
中國大陆居民前往[[香港特别行政区|香港]]或[[澳门特别行政区|澳門]]，需要[[往來港澳通行证|往來港澳通行证]]（俗稱雙程證）或者[[前往港澳通行证|前往港澳通行证]]（俗稱單程證，前往港澳地區定居使用）；定居外国的中国大陆居民而无往来港澳通行证者，需要在中国驻外领事机构申请进入许可；中国大陆居民经港澳过境至其他国家者，可用[[中华人民共和国护照|中華人民共和國護照]]，在港澳逗留时间各不超过7天。

依據[[中華人民共和國憲法|中華人民共和國憲法]]，中华人民共和国主張[[台灣地区|台灣地区]]（包含[[台澎金馬|台澎金馬]]）是中華人民共和國一部分。中華人民共和國[[中国大陆|大陆地区]]公民前往臺灣地区，亦須持用替代護照的「[[大陸居民往來台灣通行證|大陸居民往來台灣通行證]]」在中華人民共和國（中國大陸）出境，並持由[[中华民国|中华民国]]内政部移民署签发的「[[中華民國台灣地區入出境許可證|中華民國台灣地區入出境許可證]]」入境。

=== {{flagicon|Hong Kong}}{{flagicon|Macau}} 香港及澳門===
根據[[香港特別行政區基本法|香港特別行政區基本法]]及[[澳門特別行政區基本法|澳門特別行政區基本法]]，有簽發護照、進行入境管控、與外國訂定單獨簽證協定等權力。[[中國內地|中國內地]]、[[香港特別行政區|香港特別行政區]]、[[澳門特別行政區|澳門特別行政區]]之間有各自的邊境控制。

香港居民進入澳門，需於兩地邊防出示香港居民身份證（有'*','***','R'的[[香港居民身份證|香港居民身份證]]）或[[回港證|回港證]]。

澳門居民進入香港，則需於兩地邊防出示澳門居民往來香港特別行政區旅遊證（非永久性居民及其他人士）或澳門永久性居民身份證。

港澳居民中的中国公民前往中國大陸，則不得持港澳特区護照，除需於港澳邊防出示上述港澳地區的證件外，還需要於中國邊防出示「[[港澳居民來往內地通行證|港澳居民來往內地通行證]]（俗稱回鄉證）」（舊為「[[港澳同胞回鄉證|港澳同胞回鄉證]]」）或「中華人民共和國旅行證」或「[[中華人民共和國出入境通行證|中華人民共和國出入境通行證]]」。

=== {{flagicon|Taiwan}} 中華民國===
[[中華民國|中華民國]]自[[国共内战|退守台湾]]後的管轄領土僅治及于[[台澎金馬地區|台澎金馬地區]]（包含[[臺灣省|臺灣省]]及[[福建省_(中華民國)|福建省部分地區]]，即包含[[澎湖|澎湖]]、[[金門|金門]]和[[馬祖|馬祖]]）、[[東沙群島|東沙群島]]和[[南沙群島|南沙群島]]及其它島嶼，而[[中華民國憲法|中華民國憲法]]及其[[中華民國憲法增修條文|附屬條款]]上的[[中華民國領土|領土]]仍包含[[中國大陸|大陆地区]]、台湾地区以及[[蒙古国|蒙古国]]。

而[[中華民國|中華民國]]、[[中華人民共和國|中華人民共和國]]與[[蒙古國|蒙古國]]三方之間存在各自獨立的[[法制|法制]]与边境管制。值得一提的是，中華民國與中華人民共和國均不認可對方護照為有效國際旅行證件，不在對方護照上注記或蓋章。

中華民國國民前往中國大陸，須持用「[[台灣居民來往大陸通行證|台灣居民來往大陸通行證]]」或「[[中華人民共和國旅行證|中華人民共和國旅行證]]」於中華人民共和國（中國大陸）口岸入境。
