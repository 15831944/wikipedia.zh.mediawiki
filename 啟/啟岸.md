'''啟岸'''（{{lang|en|'''The Vantage'''}}），是位於[[香港|香港]][[九龍|九龍]][[紅磡|紅磡]][[鶴園|鶴園]][[馬頭圍道|馬頭圍道]]的單幢住宅，樓高共29層，由[[恒基兆業|恒基兆業]]發展，[[尊家管業|尊家管業]]管理，預計於[[2021年|2021年]][[4月30日|4月30日]]入伙。

== 歷史 ==
啟岸 前身為位於[[馬頭圍道|馬頭圍道]]57-57K、65-69號及[[庇利街|庇利街]]2-20號在[[1958年|1958年]]建成的「馬頭圍大廈」，佔地約2.3萬[[平方呎|平方呎]]，屬於住宅（甲類）用途<ref>[https://ps.hket.com/article/45084/%E9%A6%AC%E9%A0%AD%E5%9C%8D%E5%A4%A7%E5%BB%88%20%E6%81%92%E5%9C%B0%E7%94%B3%E8%AB%8B%E5%BC%B7%E6%8B%8D 馬頭圍大廈 恒地申請強拍]</ref>。發展商[[恒基兆業地產|恒基兆業地產]]早於[[2008年12月|2008年12月]]委託[[田生地產|田生地產]]開始併購此項目，至[[2009年10月|2009年10月]]獲得接近八成業主願意出售，住宅單位平均每[[平方呎|平方呎]][[收購|收購]]價為3,200[[港元|港元]]，後巷舖位每呎則4,000元，而面向[[庇利街|庇利街]]及[[馬頭圍道|馬頭圍道]]舖位每呎達10,000元，估計收購價約六億五千萬元<ref>[http://www1.hkej.com/dailynews/articlePrint/id/489424 田生6.5億購馬頭圍大廈]</ref>。[[2010年|2010年]]，發展商趕及於發水新例生效前獲批圖則，獲准興建兩幢26層高的商住物業，並設有五層平台。至[[2014年1月|2014年1月]]發展商已收集290份業權中的195份，即已收逾百分之八十五業權，故向[[土地審裁處|土地審裁處]]提出申請進行[[強制拍賣|強制拍賣]]，拍賣底價約九千五百多萬元<ref>[http://std.stheadline.com/daily/article/detail/135807-%E5%9C%B0%E7%94%A2-%E9%A6%AC%E9%A0%AD%E5%9C%8D%E5%A4%A7%E5%BB%88%E7%94%B3%E5%BC%B7%E6%8B%8D+%E5%BA%95%E5%83%B9%E9%80%BE9500%E8%90%AC 馬頭圍大廈申強拍 底價逾9500萬]</ref>。

其後恒地再與8名業主達成[[收購|收購]]協議，惟仍有4名業主未能聯絡，故法庭批出於[[2016年11月|2016年11月]]進行強制拍賣，最終在無競爭對手下，以拍賣底價17.39億元奪得項目，其執行董事[[黃浩明|黃浩明]]表示，整個項目收購期達7年，以拍賣成交價計算，加上建築成本，預計總投資額約30億元，將會發展商住項目，提供500至600伙，另有部分零售樓面<ref>[https://ps.hket.com/article/1535202/%E6%81%92%E5%9C%B0%E9%80%BE17%E5%84%84%20%E7%B5%B1%E4%B8%80%E9%A6%AC%E9%A0%AD%E5%9C%8D%E5%A4%A7%E5%BB%88%E6%A5%AD%E6%AC%8A 恒地逾17億 統一馬頭圍大廈業權]</ref>。

== 簡介 ==
啟岸 共設29層（包括地下、低層會所及高層會所但不設4、13、14及24樓），除於3至30樓提供551伙[[實用面積|實用面積]]介乎170方呎至425方呎的開放式至2房住宅單位外，其中開放式及1房單位佔全盤逾八成，個別單位實用面積僅170[[平方呎|平方呎]]，料屬[[2013年|2013年]]一手新例實施後[[紅磡|紅磡]]面積最細的新盤單位，以5至16樓標準樓層計，項目每層25伙，設4部[[升降機|升降機]]<ref>[https://www.mpfinance.com/fin/dailyp2.php?node=1551466889907&issue=20190302 啟岸主打1房 全盤送衣櫃 開放式170呎起 細絕紅磡]</ref>，其餘樓層分別設有
* 地下：住宅入口大堂、商場
* 1樓：商場
* 2樓：有蓋園景、機電房
* 17樓：[[庇護層|庇護層]]
* 低層會所及高層會所：[[會所|住客康樂設施]]

項目的[[管理費|管理費]]每呎5近元，住戶每月需繳付介乎848至1983元的管理費，其中面積最小只約170平方呎的3樓A7室，每月管理費為848元。

長達2年的[[樓花|樓花]]於[[2019年4月|2019年4月]]開售，而[[示範單位|示範單位]]曾設於[[尖沙咀|尖沙咀]][[美麗華廣場|美麗華廣場]]一期。 

== 會所 ==
會所 設於大廈頂層，為[[複式|複式]]設計，面積約11,563[[平方呎|平方呎]]，命名「天艦會」，設施包括25米天際[[泳池|泳池]]、船長吧、宴會廳、[[健身|健身]]室、池畔影院等，其中以斜面落地玻璃設計的「船長瞭望艙」位於頂層，外型呈三角型，猶如[[郵輪|郵輪]]中的控制室，內裡設有[[望遠鏡|望遠鏡]]及太空椅，供住戶欣賞[[啟德郵輪碼頭|啟德郵輪碼頭]]及[[維港|維港]]海景休息<ref>[https://ps.hket.com/article/2276836/%E5%95%9F%E5%B2%B8%E6%9C%83%E6%89%80%E3%80%8C%E5%A4%A9%E8%89%A6%E6%9C%83%E3%80%8D%20%E7%89%B9%E8%A8%AD%E8%88%B9%E9%95%B7%E7%9E%AD%E6%9C%9B%E8%89%99%E4%BE%9B%E4%BD%8F%E5%AE%A2%E8%B3%9E%E6%99%AF 啟岸會所「天艦會」 特設船長瞭望艙供住客賞景]</ref>。

[[發展商|發展商]]表示其室內設計靈感源自「星空下的[[郵輪|郵輪]]」，而整體建築以豪華郵輪為藍本<ref>[https://paper.hket.com/article/2279647/%E9%80%BE%E8%90%AC%E5%91%8E%E4%BC%91%E9%96%92%E6%9C%83%E6%89%80%20%E5%91%BD%E5%90%8D%E3%80%8C%E5%A4%A9%E8%89%A6%E6%9C%83%E3%80%8D 逾萬呎休閒會所 命名「天艦會」]</ref>。

== 商場 ==
啟岸 基座地下及1樓為約34,500[[平方呎|平方呎]]的[[商場|商場]]，發展商稱為美食消閒大道，其中中庭部分為無頂式長廊設計，而1樓一部分則為露天平台<ref>[https://www.thevantage.com.hk/data/Van_Pamphlet.pdf 發展項目資料傳單 - 啟岸The Vantage]</ref>。
