'''贾启允'''（{{bd|1914年|3月|2004年|4月12日|catIdx=J}}），[[山西省|山西省]][[闻喜县|闻喜县]]人。[[中华人民共和国|中华人民共和国]]政治人物<ref name=liren>{{Cite web |url =http://www.stats.gov.cn/ztjc/zthd/50znjn/200206/t20020617_35560.html  |title =国家统计局历任局长  |author =  |date =2004-10-18  |publisher = 中华人民共和国国家统计局 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
早年参加[[山西牺牲救国同盟会|山西牺牲救国同盟会]]，后担任[[山西青年抗敌决死队|山西青年抗敌决死队]]三纵32团政治主任，后来改任8团政委。1940年，改任9团政委；同年随部改编入[[八路军|八路军]]太行第三军分区，任军分区政治部副主任<ref>{{cite book|author=王生甫, 任惠媛|title=牺盟会史|year=1987年|publisher=山西人民出版社|pages=315页}}</ref>。[[第二次国共内战|第二次国共内战]]期间，任中共太岳区三地委副书记兼组织部部长、中共太岳区运城地委副书记、中共晋绥分局第十一地委书记。[[中华人民共和国|中华人民共和国]]成立后，担任中共[[温江|温江]]地委书记、中共[[成都市|成都市]]委第三书记。是[[中共八大|中共八大]]代表。1954年1月至1958年11月，任[[中华人民共和国国家统计局|国家统计局]]副局长，1958年11月至1961年6月，任国家统计局局长<ref name=liren/>。1961年，任[[中共四川省委|中共四川省委]]书记处书记<ref>{{cite book|author=《当代四川》丛书编辑部|title=当代四川大事辑要|year=1991年|publisher=四川人民出版社|pages=165页}}</ref>。1965年，任[[中共贵州省委|中共贵州省委]]第一书记<ref>{{cite book|author=贵州省地方志编纂委员会|title=贵州省志·气象志|year=1998年|publisher=方志出版社|pages=199页}}</ref>。[[文革|文革]]爆发后，1967年1月贾启允在[[一月风暴|一月风暴]]全国夺权的形势下下台<ref>贵州“文革”触及“贵州事件”，炎黄春秋2012年第8期</ref>。

1975年10月，贾启允任[[中共云南省委|中共云南省委]]第一书记兼[[云南省革命委员会|云南省革命委员会]]主任，任至1977年2月<ref>{{cite book|author=云南省地方志编纂委员会|title=云南省志/卷首/中华人民共和国地方志丛书|year=2004年|publisher=云南人民出版社|pages=373页}}</ref>。1975年10月至1977年2月，兼[[昆明军区|昆明军区]][[政治委员|政治委员]]。1975年10月，贾启允出任中共云南省委第一书记行前，[[邓小平|邓小平]]要贾启允放手“整顿”，“不要怕当还乡团”。（“还乡团”出自[[土地革命战争|土地革命战争]]时期，农民造反，地主逃离，农民分了地主的土地及财产，地主组织“还乡团”对农民“反攻倒算”。文革时“还乡团”的含义是：“[[走资派|走资派]]”复职后否定[[文化大革命|文化大革命]]，报复[[造反派|造反派]]。）类似的话邓小平同期还对即将赴任中共四川省委第一书记的[[赵紫阳|赵紫阳]]说过。在[[反击右倾翻案风|反击右倾翻案风]]中，贾启允将邓小平说的这些话揭发出来。“还乡团”由此成了邓小平的罪状之一。贾启允揭发后，“[[四人帮|四人帮]]”找赵紫阳了解这次谈话的内容，被赵紫阳掩盖过去<ref>{{Cite web |url =http://www.artsbj.com/show-189-423605-1.html  |title =  杨继绳：我对文革的思考|author = 杨继绳 |date = 2013-12-24 |publisher =  北京文艺网|language =  |accessdate =  |quote =  }}</ref><ref> {{Cite book | author =杨继绳  | title = 中国改革年代的政治斗争 | location = 香港 | publisher =  辉煌文化出版社 | date =  2004年| pages =  | ISBN =  | accessdate =  | url =  | language =  | quote = 第八节 大平反 大改组 }} </ref>。1976年[[粉碎四人帮|粉碎“四人帮”]]后，贾启允很快就在1977年2月下台。

1979年3月，贾启允出任河北省石家庄地区行署副专员<ref>{{Cite web |url = http://www.sjz.gov.cn/col/1275377215238/2004/10/09/1275396580014.html |title =政权 政协  |author =  |date =2004-10-09 |publisher = 中国石家庄 |language =  |accessdate =  |quote =  }}</ref>。1983年4月，调任[[河北省政协|河北省政协]]副主席、党组副书记。

2004年4月12日，贾启允在[[石家庄|石家庄]]病逝，享年90岁。
