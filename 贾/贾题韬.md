'''贾题韬'''{{BD|1909年||1995年|1月8日|catIdx=J贾}}出生于[[大清|大清]][[山西省|山西省]][[洪洞县|洪洞县]]赵城（今今洪洞县罗云村），号'''玄非'''，法名'''定密'''，毕业于[[山西大学|山西大学]]法学院法律系，为知名[[中国象棋|象棋]]理论家、[[大乘佛教|大乘佛教]]大师、[[藏传佛教|藏传佛教]]大师，1995年在四川省成都市圆寂。<ref name="fish " >{{cite web |url= http://fo.ifeng.com/special/dadexuezhe/dade/detail_2010_08/23/2153902_0.shtml|title=贾题韬 |accessdate= 2010年8月23日|publisher= 凤凰网|archiveurl=http://fo.ifeng.com/special/dadexuezhe/dade/detail_2010_08/23/2153902_0.shtml |archivedate=2012年11月11日 |work= |language =zh-cn}}</ref>

== 生平 ==
1909年（大清[[宣统|宣统]]元年），贾题韬出生在[[山西省|山西省]][[洪洞县|洪洞县]]赵城，曾在太原第一中学读书，儿时就对[[中国象棋|象棋]]有着浓厚兴趣。<ref name="fish " />

1925年11月，贾题韬荣获“山西省青年象棋比赛”的冠军。

1927年9月，贾题韬入读[[山西大学|山西大学]]法学院法律系，后来得病，无意间看到[[佛教|佛教]]典籍，读过之后，疾病痊愈，自此开始信仰佛教。1931年，贾题韬大学毕业，留在学校教书，后出任山西省教育厅秘书。

1935年，[[释能海|释能海]]到山西省[[太原市|太原市]]讲经说法，贾题韬拜入释能海的门下，学习[[大乘佛教|大乘佛教]]和[[藏传佛教|藏传佛教]]。

1937年，[[大日本帝国|大日本帝国]]全面侵华，山西大学被迫搬迁，贾题韬回到家里，卖掉家业组织了[[抗日游击队|抗日游击队]]，后编入[[国民革命军|国民革命军]]第二战区司令部，贾题韬担任第二战区司令部秘书兼第六保安区中校军法官。<ref name="fish " /><ref name="fish " />

1938年，贾题韬因病退职，搬迁到成都，陆续在各中学执教，恰巧释能海在近慈寺，贾题韬经常去请教佛法。<ref name="fish " />

{{better source|1940年，贾题韬在成都和中国棋王[[谢侠逊|谢侠逊]]对弈17局，贾题韬小胜。<ref name="tiger " /> }}后来，贾题韬拜师赵升桥，学习[[道家|道家]]。

1941年，贾题韬出版了著作《象棋指归》，后来在成都各大学讲学。

1943年，贾题韬和[[袁焕仙|袁焕仙]]、[[朱之洪|朱之洪]]、[[但懋辛|但懋辛]]、[[萧静轩|萧静轩]]、[[傅真吾|傅真吾]]一起建立了“[[维摩精舍|维摩精舍]]”。<ref name="tiger " >{{cite web |url=http://sichuandaily.scol.com.cn/2009/11/20/20091120618594178311.htm |title=追忆贾题韬先生  |accessdate= 2009年11月20日|publisher=四川日报 |archiveurl= http://web.archive.org/web/20160304224804/http://sichuandaily.scol.com.cn/2009/11/20/20091120618594178311.htm|archivedate=2012年11月11日 |work= |language =zh-cn |dead-url=yes }}</ref>

1948年，贾题韬出任[[国民大会|国民大会]]代表，[[中国人民解放军|中国人民解放军]]进入四川之后，贾题韬跟随解放军远赴[[西藏地区|西藏地区]]，在[[拉萨|拉萨]]学习[[藏文|藏文]]和藏传佛教，建立了“江阳医学院”。<ref name="fish " /><ref name="tiger " />

1965年，贾题韬出任宗教事务委员会委员兼西藏佛教协会副秘书长。<ref name="fish " />

1966年，[[文化大革命|文化大革命]]爆发，贾题韬被以“[[反革命|双料反革命]]”罪[[批斗|批斗]]、殴打等。

1979年，贾题韬返回成都市。

1980年，贾题韬担任[[中国佛教协会|中国佛教协会]]理事。

1981年，贾题韬担任四川省佛教协会常务理事兼副秘书长。

1983年，贾题韬担任四川省政协常委、[[中国人民政治协商会议第六届全国委员会委员名单|中国人民政治协商会议第六届全国委员会委员]]。

1987年，贾题韬担任中国佛教协会常务理事。

1988年，贾题韬出任[[中国人民政治协商会议第七届全国委员会委员名单|中国人民政治协商会议第七届全国委员会委员]]。

1995年1月8日，贾题韬在四川省成都市圆寂。

== 个人生活 ==
贾题韬曾师从[[释能海|释能海]]学习佛教，又曾和赵升桥学习道教，还曾远赴西藏研习藏传佛教，与[[格鲁派|格鲁派]]密悟格西为挚友。

贾题韬的弟子有：[[冯学成|冯学成]]、[[严永奎|严永奎]]、[[门世海|门世海]]等人。<ref name="fish " />
