'''贾斯汀·丹尼尔·马斯特森'''（'''Justin Daniel Masterson'''，{{bd|1985年|3月22日}}出生于[[牙买加|牙买加]][[金斯敦|金斯敦]]）是效力于[[美国职棒大联盟|美国职棒大联盟]][[匹茲堡海盜|匹茲堡海盜]]的投手。他身高6英尺6英寸，体重250英磅。马斯特森是1912年以来第一位在[[芬威公园|芬威公园]]的前四次先发都没有输球的红袜投手。

==早年棒球生涯==

===大学之前时期===
马斯特森出生于牙买加的首都金斯敦，因为当时他的父亲在牙买加神学院担任牧师<ref>{{cite web |url=http://goaztecs.cstv.com/sports/m-basebl/mtt/masterson_justin00.html |title=Profile: #21 Justin Masterson | work=[[圣迭戈州立大学|圣迭戈州立大学]] |accessdate=2008-04-17}}</ref>。几年之后马斯特森搬到了[[印地安那州|印地安那州]]。他高中就读于[[俄亥俄州|俄亥俄州]]的毕佛克里克高中，在那里他第一次接触到棒球，打的位置包括[[捕手|捕手]]、[[投手|投手]]和[[一垒手|一垒手]]。

之后他在伯特利学院读书，成为了先发投手。不过打击依然出色，他大二那年打出过10支全垒打。后来他去了圣迭戈州立大学，转为中继投手。2005年在科德角联盟投出了防御率1.15的成绩，并有10次救援成功。

===小联盟时期===
2006年红袜在选秀大会中的第二轮挑中了马斯特森。这一年他在''美国棒球''杂志的希望之星评选中名列第64位。

马斯特森从红袜短1A的[[洛威旋球|洛威旋球]]起步，在那里担任中继投手。他被选入了2006年美国棒球杂志短赛季全明星阵容<ref>{{cite web |url=http://www.baseballamerica.com/today/minors/features/262512.html |title=BaseballAmerica.com: Minors: 2006 Minor League All-Stars |accessdate=2008-04-14}}</ref><ref>{{cite web |url=http://www.thebaseballcube.com/players/M/Justin-Masterson.shtml |title=Justin Masterson Statistics — The Baseball Cube |accessdate=2008-04-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20080417171908/http://www.thebaseballcube.com/players/M/Justin-Masterson.shtml |archivedate=2008-04-17 }}</ref><ref>{{cite web |url=http://www.minorleaguebaseball.com/milb/stats/stats.jsp?n=Justin%20Masterson&pos=P&sid=milb&t=p_pbp&pid=475416 |title=Minor League Baseball: Stats: Player |accessdate=2008-04-14}}</ref>。

2007年他转为了先发投手，并从高阶1A的[[兰卡斯特喷射鹰|兰卡斯特喷射鹰]]升上了2A的[[波特兰海狗|波特兰海狗]]，这一年他一共赢了12场比赛，在红袜小联盟所有投手中排名第二<ref>{{citeweb|url=http://mlb.mlb.com/news/article.jsp?ymd=20080306&content_id=2408395&vkey=spt2008news&fext=.jsp&c_id=bos|publisher=mlb.com|title=Second place among all Red Sox minors.|accessdate=2008-06-01}}</ref>。

升上海狗队之后，马斯特森说：“我一直很有信心，相信自己会成为一位优秀的投手。我曾经去了一所较小的学校，在那里证明了我有做投手的能力。无论未来有多少的路要走，我从来不担心。”<ref>[http://media.www.thedailyaztec.com/media/storage/paper741/news/2007/08/29/Sports/A.Diamond.Discovered.In.San.Diego-2941534.shtml The Daily Aztec] {{webarchive|url=https://web.archive.org/web/20080420181842/http://media.www.thedailyaztec.com/media/storage/paper741/news/2007/08/29/Sports/A.Diamond.Discovered.In.San.Diego-2941534.shtml |date=2008-04-20 }}</ref>

==大联盟生涯==

===2008年===
马斯特森被红袜队邀请参加了2008年的春训。4月24日，由于[[松坂大辅|松坂大辅]]的流行性感冒，马斯特森被紧急叫上大联盟，先发对上[[洛杉矶安那罕天使|洛杉矶安那罕天使]]。这场比赛他主投6局，仅掉1分，很好地完成了他的任务。比赛之后就被送回了小联盟。5月20日，马斯特森第二度在大联盟登板先发，这次他投了6又1/3局失1分，拿下了他在大联盟的第一场胜投。之后还是被下放回了下联盟<ref>{{cite web
|url=http://www.boston.com/sports/baseball/redsox/articles/2008/05/21/nice_encore_by_masterson/
|title=Nice encore by Masterson | date=2008-05-21 |publisher=波士顿环球报 | author=Gordon Edes |accessdate=2008-05-21}}</ref>。

在大联盟5次先发之后，球队宣布马斯特森将留在大联盟。由于红袜队先发轮值表已满，主教练[[特里·弗兰克纳|特里·弗兰克纳]]希望马斯特森能转为中继投手。7月7日他被下放回3A锻炼<ref>{{cite web
|url=http://www.bostonherald.com/sports/baseball/red_sox/view/2008_07_08_Masterson_off_to_become_reliever/srvc=redsox&position=5 |title=Justin Masterson off to become reliever | date=2008-07-08 |publisher=波士顿先驱报 | author=Rob Bradford |accessdate=2008-07-08}}</ref>。由于[[大卫·阿德斯马|大卫·阿德斯马]]的受伤，7月20日马斯特森被叫回大联盟顶替阿德斯马中继的位置<ref>{{cite web
|url=http://sports.yahoo.com/mlb/news;_ylt=ApeZ.iBsYvUgcevRmDgWueaFCLcF?slug=rotowire-ustinastersonackinos&prov=rotowire&type=fantasy
|title=Justin Masterson: Back in Boston | date=2008-07-20 |publisher=rotowire.com | author= |accessdate=2008-07-23}}</ref>。在他第一次中继出场的比赛中，马斯特森表现出色，主投2又2/3局无失分，封锁了[[西雅图水手|西雅图水手]]的打线。

在2008年的季后赛中，马斯特森几乎场场登板中继，他也成为了红袜季后赛最倚赖的中继投手之一。在他上场的9场比赛中，一共投了9.2局，防御率仅为1.82，拿下4个中继点，还在[[美联冠军赛|美联冠军赛]]第五场对[[坦帕湾光芒|坦帕湾光芒]]的大逆转比赛中获得胜投<ref>[http://www.telegram.com/article/20081017/NEWS/810170732/1009/SPORTS A miracle comeback: Red Sox 8, Rays 7]</ref>。

==球探报告==
马斯特森采用的是四分之三侧投，和[[丹尼斯·埃克斯利|丹尼斯·埃克斯利]]（Dennis Eckersley）有些相似。他主要的球种包括了快速球，伸卡球，滑球和变速球。其中他的伸卡球下降幅度很大，被认为是他最具威力的球种<ref>[http://www.soxprospects.com/players/masterson-justin.htm Justin Masterson | SoxProspects.com<!-- Bot generated title -->]</ref>。球速可在84到94英里/小时之间变化，让打者抓不到节奏。

马斯特森的快速球也十分多样化，球速可在87至95英里/小时之间。变速球则在77到81英里之间。

==私人生活==
2007年11月3日马斯特森和梅丽尔·哈姆（Meryl Ham）结婚。现在他们定居于俄亥俄州的毕佛克里克<ref>[http://redsoxnation.net/forums/index.php?showtopic=21573 Justin Masterson - Red Sox pitching prospect - Royal Rooters<!-- Bot generated title -->]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
