'''贾平凹'''（{{lang-pinyin|Jiǎ Píngwā}}<ref>{{cite web |url=http://ciku.us.chinesecio.com/zh-hans/cio_msg_forum_content/id/291 |language=zh-cn |title=66个易错词拷问国人的汉语态度 |publisher=美国堪萨斯大学孔子学院 |date=2014-03-31 |access-date=2015-01-03 |archive-url=https://web.archive.org/web/20150103105212/http://ciku.us.chinesecio.com/zh-hans/cio_msg_forum_content/id/291 |archive-date=2015-01-03 |dead-url=yes }}</ref>  {{bd|1952年|2月21日|catIdx=J贾}}），原名'''贾平娃'''，后改称“平凹”（取“洼”解，是指“坦途”之意），<ref name="multiple1">{{cite web|url=http://news.cn.yahoo.com/050405/346/2al62.html|title=《西北文豪贾平凹悲凉一笑》|accessdate=2005-04-05|work=|publisher=雅虎|language=zh-hans}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>[[中国大陸|中国大陸]]当代著名作家<ref>[http://book.sina.com.cn/people/jiapingao/ 作家贾平凹 名人堂 新浪网]</ref>。[[陕西省|陕西省]][[商洛市|商洛市]][[丹凤县|丹凤县]]人。[[中国作家协会|中国作家协会]]理事、中国作家协会陕西分会副主席。现为[[西安建筑科技大学|西安建筑科技大学]]人文学院院长。2016年12月当选第九届中国作家协会副主席。<ref >{{cite web|url=http://news.sina.com.cn/c/2016-12-03/doc-ifxyhwyy0561948.shtml | language= | title=著名作家贾平凹当选中国作协副主席 |publisher=中华网 |date=2016年12月3日 |accessdate= }}</ref>

==生平==
1952年，出生于乡村教师家庭，从小父亲对他的教育非常严格，经常使用棍棒教育，结婚后都曾打过他。<ref>{{cite web|url=http://www.chinaqking.com/rm/2009/30678.html|title=《贾平凹：人生几多情》|accessdate=2009-03-27|work=|publisher=中国期刊网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20090413151151/http://www.chinaqking.com/rm/2009/30678.html|archivedate=2009-04-13}}</ref>

1975年，毕业于[[西北大学_(中国)|西北大学]]中文系。<ref name="multiple1">{{cite web|url=http://news.cn.yahoo.com/050405/346/2al62.html|title=《西北文豪贾平凹悲凉一笑》|accessdate=2005-04-05|work=|publisher=雅虎|language=zh-hans}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>毕业后从事过几年文学编辑工作，曾任陕西人民出版社文艺编辑、《长安》文学月刊编辑。

1983年，開始就职西安市文联，专职作家，从事专业创作。1993年，作品《废都》由北京出版社出版第一版后，毁誉参半，被冠之以“流氓作家”称号，<ref>{{cite web |url=http://culture.people.com.cn/GB/70806/70984/9760320.html|title=《《废都》“解禁” 贾平凹摘下“流氓作家”帽|accessdate=2009年7月31日|work=|publisher=人民网|language= zh-hans}}</ref>《废都》也随后被禁。理由是：书中大量的性描写争议太多。<ref>{{cite web |url=http://news.sohu.com/20090729/n265571479.shtml|title=《废都》被禁16年后解禁 专家呼吁明确审查标准|accessdate=2009年7月29日|work=|publisher=搜狐网|language= zh-hans}}</ref>

2009年，《废都》解禁，[[作家出版社|作家出版社]]率先再版。

== 风格 ==
在文学界，贾平凹以“[[商州|商州]]系列”为代表的农村作品写得非常优秀。<ref name="multiple1">{{cite web|url=http://news.cn.yahoo.com/050405/346/2al62.html|title=《西北文豪贾平凹悲凉一笑》|accessdate=2005-04-05|work=|publisher=雅虎|language=zh-hans}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 评价 ==
=== 正面评价 ===
中国当代著名作家[[王蒙_(作家)|王蒙]]点评贾平凹时说：“贾平凹这人说话时陕西味十足，他的眼睛非常亮，很有观察力，事实上，他的农民模样掩盖了他的精明、智慧。他是个聪明、善于不露痕迹搞幽默的作家。”<ref>{{cite web |url=http://www.gmw.cn/01wzb/2001-06/03/05-B8674F610035B20448256A600000DBB8.htm|title=《王蒙点评当今诸作家》|accessdate=2011年10月20日|work=常江|publisher=文摘报|language= zh-hans}}</ref>

=== 负面评价 ===
著名文艺批评家[[肖鹰|肖鹰]]评点贾平凹时说：“贾平凹的《秦腔》，从表述方面来说，他的语言表达很圆熟，但是，我从中看不到一个站在中国文学高峰上的巨著所应该具有的品质和内涵。我看到的更多的是琐碎的、低迷的、阴暗的、甚至猥亵的写作趣味。作家在对现实的苦难和困厄进行揭示的时候，心中应该怀着真挚的大爱，应该带着对社会、人生的悲悯和祝福的情怀。如果把《秦腔》这样的作品推崇为中国文学（小说）的高峰之作，既是很可笑的事，更是很可悲的事。《废都》在1993年出版时，遭遇到批评界、尤其是中青年批评家的普遍抨击，这是对的；其后被查禁，也是对的。因为这部书，不仅语言和结构仿旧，而且思想和旨趣酸腐，其中那种旧式没落文人的自以为是和淫逸幻觉，是完全违背我们现时代的人文精神的。现在，《废都》不仅被解禁，而且个别当年狂批《废都》的新锐批评家现在摇身一变，变脸成为飙捧《废都》的‘权威幹将’。在这‘狂’与‘飙’的大转换之间，是这些批评家换了大脑，还是我们的文学标准、乃至于基本的是非观出现了问题呢……我认为，《废都》和《檀香刑》这样的作品被批评家追捧，只能表明在当下批评界，文学标准和个人标准都瓦解了，当下中国文学批评确实在整体上颓败了。”<ref>{{cite web |url=http://book.people.com.cn/GB/69360/10637139.html|title=《肖鹰：阿来写旅游招贴 贾平凹写的是变态文学》|accessdate=2009年12月23日|work=|publisher=人民网|language= zh-hans}}</ref>
