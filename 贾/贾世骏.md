'''贾世骏'''（{{bd|1930年||2011年|9月4日}}）祖籍[[山东|山东]][[单县|单县]]，生于[[南京|南京]]，[[中华人民共和国|中华人民共和国]]男高音歌唱家、作曲家<ref name=wy>{{Cite web |url = http://news.163.com/11/0905/15/7D6R1QMT00014AED.html|title =  歌唱家贾世骏肺癌去世|author =  |date = 2011-09-05 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref name=rmw>{{Cite web |url =http://politics.people.com.cn/GB/77065/15613469.html  |title =  著名男高音歌唱家贾世骏因病去世|author =  |date = 2011-09-07 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1930年生于南京。上小学时便学会了吹[[口琴|口琴]]、弹[[月琴|月琴]]等乐器，并是小小乐队的主要成员之一。1949年8月在[[北平|北平]]参加[[中国人民解放军|中国人民解放军]]，在六十六军文工团、一九八师文工队任[[大提琴|大提琴]]演奏员，歌唱演员。1950年10月到[[朝鲜|朝鲜]]参加[[抗美援朝战争|抗美援朝战争]]，任东北某英雄担架队军代表。1954年10月调入[[北京军区|北京军区]]政治部战友歌舞团（当时是[[华北军区|华北军区]]政治部文工团），历任北京军区战友歌舞团合唱队员、独唱演员、歌队队长、编导室声乐艺术指导等职务。1956年6月加入[[中国共产党|中国共产党]]。1956年加入[[中国音乐家协会|中国音乐家协会]]。曾获[[中华人民共和国国务院|国务院]][[政府特殊津贴|政府特殊津贴]]、[[中央军事委员会|中央军委]]颁发的“[[胜利功勋荣誉章|胜利功勋荣誉章]]”。1986年10月[[离休|离休]]<ref name=rmw/>。

贾世骏曾先后师从[[中央音乐学院|中央音乐学院]][[俞宜萱|俞宜萱]]、[[周美玉|周美玉]]，[[中央乐团|中央乐团]][[陈灵|陈灵]]、[[蔡敬宜|蔡敬宜]]，北京中国师范学院[[秦宗惠|秦宗惠]]等学习声乐，并在[[保加利亚|保加利亚]]声乐专家邱尔金以及迪雅克维奇教授指导下，进修声乐一年半<ref name=rmw/>。

贾世骏曾多次随[[周恩来|周恩来]]、[[陈毅|陈毅]]、[[罗瑞卿|罗瑞卿]]等[[党和国家领导人|党和国家领导人]]率领的中国艺术团、解放军艺术团到[[苏联|苏联]]、[[罗马尼亚|罗马尼亚]]、[[阿尔巴尼亚|阿尔巴尼亚]]、[[朝鲜|朝鲜]]、[[越南|越南]]、[[印度尼西亚|印度尼西亚]]、[[日本|日本]]等国及中国[[香港|香港]]、[[澳门|澳门]]地区演出。1986年离休后，参加了老战友合唱艺术团，继续演唱《长征组歌》；还被[[北京|北京]]、[[青岛|青岛]]等地多所院校聘为艺术顾问、校外讲师团成员、校外辅导员。1999年10月，参加了在[[北京音乐厅|北京音乐厅]]举行的《我的祖国——新中国50年优秀歌曲音乐会》，[[江泽民|江泽民]]、[[胡锦涛|胡锦涛]]等中央领导观看了此次演出。2009年10月，为庆祝[[中华人民共和国|中华人民共和国]]成立60周年，参加了《神州大舞台》、《流金岁月》、大歌舞《东方红》45年再聚首等节目的演出及录制<ref name=rmw/>。2010年4月，在[[李光羲|李光羲]]独唱音乐会上，贾世骏最后一次登台表演，他也是此次音乐会上唯一一位自费从外地赶来的嘉宾<ref name=wy/>。

2011年9月4日，贾世骏因[[肺癌|肺癌]]在[[北京|北京]][[中国人民解放军海军总医院|中国人民解放军海军总医院]]逝世，享年81岁。逝世前几天，贾世骏的朋友[[王昆|王昆]]、[[李光羲|李光羲]]、[[徐有光|徐有光]]、[[胡宝善|胡宝善]]、[[姜嘉锵|姜嘉锵]]、[[于淑珍|于淑珍]]、[[邓玉华|邓玉华]]、[[金家勤|金家勤]]、[[刘斌_(歌手)|刘斌]]、[[于乃久|于乃久]]、[[柳石明|柳石明]]等都到医院探望，坐在轮椅上的王昆还领着大家为贾世骏唱起了《过雪山草地》<ref name=wy/>。

==作品==
贾世骏演唱音域宽广，音质圆润纯净，音色甜美明亮，气魄雄伟，吐字清晰，受到军民喜爱。1960年代至1980年代，他首唱了许多著名歌曲，如《[[大海航行靠舵手|大海航行靠舵手]]》、《读[[毛主席|毛主席]]的书》、《战士歌唱东方红》、《我骑着马儿过草原》、《颂歌献给毛主席》、《[[打靶归来|打靶归来]]》等。他演唱的代表性曲目还有《[[牧歌|牧歌]]》、《[[康定情歌|康定情歌]]》、《战士想的是什么》、《草原之夜》、《祖国颂》、《赠[[缅甸|缅甸]]友人》等等，以及大合唱《祖国颂》男声领唱，[[印度尼西亚|印度尼西亚]]歌曲《美丽的梭罗河》、[[苏联|苏联]]歌曲《故乡》、[[朝鲜|朝鲜]]歌曲《月飞山—英雄山》、[[越南|越南]]歌曲《最美丽的名字—[[胡志明|胡志明]]》等外国歌曲。被部队指战员誉为战士歌唱家，被[[莫斯科|莫斯科]]人誉为中国的“列米舍夫”（俄国抒情男高音歌唱家），被印度尼西亚人民誉为“金嗓子”<ref name=rmw/><ref name=wy/>。

在1965年大型声乐套曲《[[长征组歌|长征组歌]]——红军不怕远征难》六曲中，贾世骏领唱《过雪山草地》。在1964年大型音乐舞蹈史诗《[[东方红_(音乐剧)|东方红]]》中，领唱《红军不怕远征难》中演唱《[[七律·长征|七律·长征]]》。在[[中南海|中南海]]春藕斋为[[毛泽东|毛泽东]]主席演唱时，毛泽东握着他的手夸他唱得好。[[周恩来|周恩来]]总理在审查《长征组歌》、接见演员时，拉着他的手，让他坐在自己身边和大家合影。周恩来还让他教自己唱《过雪山草地》<ref name=rmw/>。

贾世骏还是一位作曲家，发表并演唱了自己创作的《我爱伟大的祖国》、《祖国啊母亲》、《[[长江|长江]]啊长江》、《飞吧我的歌声》、《做毛主席的好战士》、《听毛主席的话》、《手捧请柬想[[周恩来|总理]]》、《人民的好总理——周恩来》、《我爱[[西沙群岛|西沙]]》、《演习场上重相逢》、《我从[[罗湖桥|罗湖桥]]上过》、《[[香港回归|香港回归]]唱新歌》、《[[上海|上海]]抒情曲》、《美丽的[[青岛|青岛]]》<ref name=wy/>。他曾为[[国家审计署|国家审计署]]《审计工作者之歌》谱曲。还曾为少年儿童作词谱曲，创作出大量少儿歌曲、歌舞曲、[[木偶剧|木偶剧]]音乐，如《我是一个小歌手》，《小豆苗》，为[[北京市东城区史家小学|北京市东城区史家小学]]作《史家小学校歌》。他创作的200余首儿童歌曲在[[中国人民解放军总后勤部|中国人民解放军总后勤部]]幼儿园、[[北京军区|北京军区]]司令部幼儿园和政治部幼儿园等地演出，并在[[中央人民广播电台|中央人民广播电台]]、[[中央电视台|中央电视台]]等媒体播出<ref name=rmw/>。

2013年，贾世骏的自传《飞吧，我的歌声：我的歌唱艺术生涯》由[[解放军出版社|解放军出版社]]出版<ref name=wy/>。
