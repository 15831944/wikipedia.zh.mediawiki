'''贾餗'''（{{bd|？||835年|12月17日|catIdx=J}}<ref name=AS>[http://sinocal.sinica.edu.tw/ 中央研究院兩千年中西曆轉換]</ref><ref name=ZZTJ245>[[:s:資治通鑑/卷245|《资治通鉴》卷二百四十五]]</ref>），字'''子美'''，爵封'''姑臧县男'''，[[唐朝|唐朝]]官员，[[唐文宗|唐文宗]]年间短暂拜相。文宗年间，他卷入朝臣和宦官间的大规模权力争斗即[[甘露之变|甘露之变]]，和其他三名宰相[[李训|李训]]、[[王涯|王涯]]、[[舒元舆|舒元舆]]一同被宦官所杀。

== 家世和早期仕途 ==
贾餗生年不详，河南府（即东都[[洛阳|洛阳]]地区）人氏。<ref name=BT169>[[:s:舊唐書/卷169|《旧唐书》卷一百六十九]]</ref>世居[[姑臧|姑臧]]。祖父贾渭、父亲贾宁在《[[新唐书|新唐书]]》宰相世系表中均无官职记载，意指他们为平民。贾餗有兄贾竦。<ref>新唐书卷七十五{{cite web |url=http://www.sidneyluo.net/a/a17/075.htm |title=存档副本 |accessdate=2009-12-19 |0=2009-12-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20091219203329/http://www.sidneyluo.net/a/a17/075.htm |archivedate=2009-12-19 }}{{cite web |url=http://www.sidneyluo.net/a/a17/table/form101.htm |title=存档副本 |accessdate=2008-12-27 |0=2009-12-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20091220195605/http://www.sidneyluo.net/a/a17/table/form101.htm |archivedate=2009-12-20 }}</ref>

贾餗早孤，旅居江淮之间。从父[[贾全_(唐朝)|贾全]]任[[浙东观察使|浙东观察使]]，贾餗前往投靠。贾全惊异于他的才能，待他也很优厚。<ref name=NBT179>[[:s:新唐書/卷179|《新唐书》卷一百七十九]]</ref>

贾餗通过[[进士|进士]][[殿试|殿试]]，因文史出众而扬名。又通过贤良方正异等考试，被任为[[渭南|渭南]]尉、集贤校理。后四度升迁为[[考功员外郎|考功员外郎]]，[[知制诰|知制诰]]。<ref name=NBT179/>

== 穆宗、敬宗年间 ==
[[唐穆宗|唐穆宗]][[长庆|长庆]]（821年）元年十一月，贾餗和[[中书舍人|中书舍人]][[白居易|白居易]]、缮部郎中[[陈岵|陈岵]]被任为选拔贤良的策论考官，<ref>[[:s:旧唐书/卷16|《旧唐书》卷一十六]]</ref><ref name=BT166>[[:s:旧唐书/卷166|《旧唐书》卷一百六十六]]</ref>时人认为他们公正。不久，贾餗迁[[库部郎中|库部郎中]]，仍知制诰。<ref name=BT169/>贾餗善于作文，机敏有决断，但性格暴厉急躁，常凌辱同僚。[[谏议大夫|谏议大夫]][[李渤|李渤]]厌恶贾餗并报告宰相，但因[[李逢吉|李逢吉]]、[[窦易直|窦易直]]爱其才，贾餗没有被贬。<ref name=NBT179/><ref>《新唐书》此处记载意指李逢吉、窦易直当时都是宰相，但造成了时间争议，因在唐穆宗生前，窦易直从未拜相。见《资治通鉴》卷二百四十三。</ref>

四年（824年），穆宗崩，子[[唐敬宗|唐敬宗]]继位，<ref name=ZZTJ243>[[:s:資治通鑑/卷243|《资治通鉴》卷二百四十三]]：穆宗長慶四年（甲辰、八二四年）：壬申，……是夕，上崩于寢殿。……丙子，敬宗卽位于太極東序。</ref>贾餗是众多被派到各镇告哀的使者之一，四月被派往江浙地区为宣慰使。<ref>[[:s:永福寺石壁法华经记|元稹《永福寺石壁法华经记》]]</ref>途中因[[张又新|张又新]]构陷，<ref name=BT169/>出为[[常州|常州]]刺史。当时，官员出使由红衣小吏为前导，贾餗就任常州后也如此。其上司[[浙西观察使|浙西观察使]][[李德裕|李德裕]]命他停止使用红衣小吏，贾餗很记恨。<ref name=NBT179/>

== 文宗年间 ==
{{see also|甘露之变}}
唐敬宗弟唐文宗[[太和_(唐朝)|太和]]（827年 - 835年）初年，贾餗被召回[[长安|长安]]任[[太常少卿|太常少卿]]。二年（828年）三月，文宗御宣政殿亲试制策举人，以贾餗和[[左散骑常侍|左散骑常侍]][[冯宿|冯宿]]、库部郎中[[庞严|庞严]]为考制策官，<ref>[[:s:旧唐书/卷17上|《旧唐书》卷一十七上]]</ref><ref>[[:s:试制举人诏|:s:试制举人诏]]</ref>以[[裴休|裴休]]为甲等制科之首。<ref name=BT166/>三人都是当时文士，目睹进士[[刘蕡|刘蕡]]的对策指宦官为祸乱根本，为之嗟叹，以为即使是[[汉朝|汉朝]]的[[晁错|晁错]]、[[董仲舒|董仲舒]]也无以过之。但当时宦官当道，三人不敢录用刘蕡，<ref name=NBT179/><ref name=NBT178>[[:s:新唐書/卷178|《新唐书》卷一百七十八]]</ref>物论喧然不平。<ref>[[:s:旧唐书/卷190下|《旧唐书》卷一百九十下]]</ref>同年贾餗再任知制诰。三年（829年）七月，以[[朝散大夫|朝散大夫]]、守[[太常少卿|太常少卿]]、知制诰、[[上柱国|上柱国]]守中书舍人。<ref>[[:s:贾餗等中书舍人制|李虞仲《贾餗等中书舍人制》]]</ref>四年（830年）九月，权知礼部科举。五年（831年），科举发榜后，拜[[礼部侍郎|礼部侍郎]]。他经历三届科举，选拔了75人，多至公卿、宰相。[[杜顗|杜顗]]中进士，贾餗对人说：“得杜顗，足敌数百人。”授其[[秘书省|秘书省]]正字。<ref>[[:s:新唐书/卷166|《新唐书》卷一百六十六]]</ref>任内还奉敕为时任[[西川节度使|西川节度使]]的李德裕撰德政碑。<ref>[[:s:赐李德裕立德政碑敕|:s:赐李德裕立德政碑敕]]</ref>六年（832年）四月，贾餗奏复考功别头试。<ref>[[:s:新唐书/卷44|《新唐书》卷四十四]]</ref>七年（833年）五月，转任[[兵部侍郎|兵部侍郎]]。八年（834年）十一月，迁[[京兆尹|京兆尹]]，授[[御史大夫|御史大夫]]。<ref name=BT169/>

九年（835年）四月，唐文宗设宴[[曲江|曲江]]。按规矩，京兆尹应该在外门下马对御史作揖。贾餗以自己身居高位且和宰相[[李宗闵|李宗闵]]、文宗近臣[[郑注|郑注]]有交为傲，没有下马，继续骑行。殿中侍御史[[杨俭|杨俭]]、[[苏特|苏特]]与贾餗争辩，被贾餗痛骂：“黄面儿敢尔！”杨俭说：“公为御史，能嘿嘿吗？”结果，大夫[[温造|温造]]上表弹劾贾餗，贾餗被罚俸。贾餗以为耻，请求出京，被任为浙西观察使，未行，以前浙西道都团练观察处置等使、朝议大夫、检校礼部尚书、持节润州诸军事兼润州刺史、御史大夫、[[上柱国|上柱国]]、姑臧县开国男、食邑三百户、赐紫金鱼袋被任为守[[中书侍郎|中书侍郎]]，授[[同中书门下平章事|同中书门下平章事]]，拜为宰相。<ref name=BT17-2>[[:s:旧唐书/卷17下|《旧唐书》卷一十七下]]</ref><ref name=NBT8>[[:s:新唐书/卷8|《新唐书》卷八]]</ref><ref>[[:s:授贾餗中书侍郎平章事制|:s:授贾餗中书侍郎平章事制]]</ref>[[柏杨|柏杨]]认为，此次贾餗在出事后突然升职是郑注试图表现自己对皇帝的影响力的事例之一。当时文宗不满朋党，故大用贾餗等无党之人。<ref name=ZZTJ245/><ref name=BT172>[[:s:旧唐书/卷172|《旧唐书》卷一百七十二]]</ref>不久，贾餗又被授[[银青光禄大夫|银青光禄大夫]]、充[[集贤殿|集贤殿]]大学士，监修国史。<ref name=BT169/><ref>[[:s:授李固言崇文馆大学士贾餗集贤殿大学士制|:s:授李固言崇文馆大学士贾餗集贤殿大学士制]]</ref>正逢李宗闵获罪，贾餗指杨俭、苏特为其党，罢斥之。<ref name=NBT179/>

此时，文宗、郑注、贾餗同僚宰相李训正筹划屠杀当权宦官，贾餗并不知情。12月14日，<ref name=AS/>李训实施计划（即后世所谓“甘露之变”），宦官挟持文宗，计划失败。李训逃出长安，贾餗和同僚宰相王涯、舒元舆回到宰相官署，相信文宗很快会召他们去延英殿处理后事，称不知有何事，命属官继续照常办公。但不久，宦官[[神策军|神策军]]中尉[[仇士良|仇士良]]等相信朝臣参与图谋，命左、右神策副使刘泰伦、魏仲卿等统率神策军五百人攻打官邸。<ref name=BT184>[[:s:旧唐书/卷184|《旧唐书》卷一百八十四]]</ref>宰相们正要用午餐，闻讯逃走。贾餗穿便服躲了一夜，次日，自知逃不掉，换上丧服骑驴至[[兴安门|兴安门]]，对士兵说：“我是宰相贾餗，被奸人诬陷，请把我交给神策军。”士兵照做了。同时，宦官将王涯受刑后写的称自己和朝臣意欲推翻文宗立郑注为帝的供状呈上，并得文宗认可。<ref name=ZZTJ245/><ref>[[:s:开成改元赦文|:s:开成改元赦文]]</ref>

故[[岭南节度使|岭南节度使]][[胡证|胡证]]为京城富豪，且生前素日和贾餗交好。这时禁军贪胡家钱财，称胡证子胡溵藏匿贾餗，杀入其家，一日之内抢光其家财，军人擒胡溵入左军，仇士良命斩之以徇。<ref>[[:s:旧唐书/卷163|《旧唐书》卷一百六十三]]</ref><ref>[[:s:新唐书/卷164|《新唐书》卷一百六十四]]</ref>有人说太子詹事[[浑鐬|浑鐬]]藏匿贾餗，百骑捕之，浑鐬苦辩得免，但其家被士兵抢完。<ref>[[:s:新唐书/卷155|《新唐书》卷一百五十五]]</ref>

12月17日，<ref name=AS/>神策左右军各三百人押送贾餗、王涯、[[河东节度使|河东节度使]][[王璠|王璠]]、京兆少尹[[羅立言|羅立言]]、[[邠宁节度使|邠宁节度使]][[郭行餘|郭行餘]]、舒元舆、[[御史中丞|御史中丞]][[李孝本|李孝本]]及李训首级献于京郊庙社，后押赴子城西南隅独柳树下腰斩。<ref>[[:s:诛王涯等敕|:s:诛王涯等敕]]</ref>贾餗临刑愤怒地大声责骂；<ref name=NBT179/>他们与郑注、左金吾卫大将军[[韩约|韩约]]、凤翔少尹[[魏逢|魏逢]]共计十一人，横尸宫阙之下。<ref name=NBT8/><ref name=BT184/>他们的家属也遭屠戮，<ref name=ZZTJ245/><ref name=NBT178/><ref name=BT17-2/>亲属门人从坐者数十百人，<ref>[[:s:旧唐书/卷170|《旧唐书》卷一百七十]]</ref><ref>[[:s:诛王涯郑注加恩中外德音|:s:诛王涯郑注加恩中外德音]]</ref>资产都被军卒瓜分。死非其罪，世人多以为冤。<ref name=BT169/>

贾餗小时候与[[沈传师|沈传师]]交好，沈传师已死，曾在贾餗梦中说：“君可休矣！”贾餗醒后在卧室祭祀他，又梦到他说：“事已如此，奈何！”贾餗终于遭祸。<ref name=NBT179/>

事变当夜，文宗召[[右仆射|右仆射]][[郑覃|郑覃]]与[[尚书左仆射|尚书左仆射]][[令狐楚|令狐楚]]入宫，共商作诏书，都想用为宰相。令狐楚因王涯、贾餗冤死，叙其罪状时泛泛而谈，仇士良等不悦，故令狐楚不得拜相。<ref name=ZZTJ245/><ref name=BT166/><ref name=BT172/>

此后宦官愈发骄横，文宗最终忧死。<ref name=NBT178/>

[[宋代|宋代]]史学家[[司马光|司马光]]在所著《[[资治通鉴|资治通鉴]]》中如是评价贾餗和王涯之死：<ref name=ZZTJ245/>“当时的评论家都说王涯和贾餗善文学，有名声，并不知道李训、郑注的图谋，横遭灭族之祸。他们为此气愤，哀叹他们死得冤枉。臣不以为然。社稷危亡，却不去匡正，当宰相有什么用！王、贾安于高位，享受财富和荣耀，同时李、郑这样的小人用尽奸险手段谋取将相之位。王、贾和他们比肩，不以为耻；国家面临危亡，他们不以为忧。他们苟且偷安，日复一日，自以为得保身良策，无人能及。如果人人为此且无灾祸，奸臣谁不愿意如此呢？一旦灾难降临，就不免家破人亡。这是上天要族诛他们，而非仇士良！”

贾餗子贾庠等在甘露之变中逃奔[[昭义军节度使|昭义军节度使]][[刘从谏|刘从谏]]。[[唐武宗|唐武宗]][[会昌_(唐朝)|会昌]]四年（844年）九月，刘从谏侄[[刘稹|刘稹]]败亡，贾庠等也被杀。时李德裕为宰相，下诏称“已在昭义军诛杀逆贼王涯、贾餗子孙”，宣告中外，有识者非之。<ref name=ZZTJ248>[[:s:資治通鑑/卷248|《资治通鉴》卷二百四十八]]：又杀刘从谏父子所厚善者[[张谷|张谷]]、[[陈扬庭|陈扬庭]]、[[李仲京|李仲京]]、[[郭台|郭台]]、[[王羽|王羽]]、[[韩茂章|韩茂章]]、[[韩茂实|茂实]]、[[王渥|王渥]]、贾庠等凡十二家，并其子侄甥婿无遗。仲京，训之兄；台，[[郭行馀|行馀]]之子。羽，涯之从孙；茂章、茂实，[[韩约|约]]之子；渥，[[璠|王璠]]之子。庠，餗之子也。甘露之乱，仲京等亡归从谏，从谏抚养之。</ref>

后在文宗叔父[[唐宣宗|唐宣宗]]年间，贾餗和王涯被平反，官爵追复。<ref name=ZZTJ249>[[:s:資治通鑑/卷249|《资治通鉴》卷二百四十九]]：宣宗元圣至明成武献文睿智章仁神聪懿道大孝皇帝下大中八年（甲戌，公元八五四年）：上以甘露之变，惟李训、郑注当死，自馀王涯、贾餗等无罪，诏皆雪其冤。</ref>

白居易曾作《看常州柘枝赠贾使君》《戏和贾常州醉中二绝句》。
