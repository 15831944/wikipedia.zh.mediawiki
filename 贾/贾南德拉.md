'''贾南德拉·比尔·比克拉姆·沙阿'''（[[尼泊尔语|尼泊尔语]]：{{lang|ne|'''ज्ञानेन्द्र वीर बिक्रम शाहदेव'''}}；[[IAST|IAST]]：{{transl|尼泊尔语|''Jñānendra Vīra Bikrama Śāh''}}，{{bd|1947年|7月7日|catIdx=G賈南德拉}} ）是[[尼泊爾|尼泊爾]][[沙阿王朝|沙阿王朝]]的末代國王。

== 簡歷 ==
1950年，當賈南德拉只有三歲的時候，其祖父，時任尼泊爾國王的[[狄里布凡|狄里布凡]]與其他家人流亡國外，於是把王位傳於他（1950年—1951年在位）。但這次傳位並沒有得到國際承認。兩個月後狄里布凡回到尼泊爾重掌王權，1955年傳位于賈南德拉的父親[[馬亨德拉|馬亨德拉]]。

五十年後，因尼泊爾暴發[[尼泊尔王室枪击事件|王室血案]]。狄潘德拉王儲因婚姻問題導致精神不穩定，最終在槍殺包括[[畢蘭德拉|畢蘭德拉]]國王在内的數名王室成員后自殺。因此，賈南德拉再次登上國王的寶座。但也有人懷疑他涉嫌暗殺兄長[[畢蘭德拉|畢蘭德拉]]國王，並把罪名推到[[狄潘德拉|狄潘德拉]]王儲身上；待後來又暗殺接掌了王位的狄潘德拉。

隨後賈南德拉宣布接掌王位，並宣佈全國進入緊急狀態，實行軍法統治，並推翻他的兄長畢蘭德拉在1990年推行的[[君主立憲|君主立憲]]制度。他這舉動引來國內強烈的反彈，但這些反彈聲音都被他大幅鎮壓。他對國內的異見份子絕不手軟，特別是國內[[尼泊爾共產黨（毛派）|尼泊爾共產黨（毛派）]]的叛軍。他亦在三年內兩度因為對抗叛軍不力而解除民選首相的職務，並自行任命首相。2005年2月1日，他透過電視演說宣布解散由時任首相[[德烏帕|德烏帕]]（Sher Bahadur Deuba）領導的內閣，因為他們未能同反政府武装游擊隊舉行和談的[[聯合政府|聯合政府]]。解散內閣之後，他立即在全國實施戒嚴，並於翌日自行籌組內閣，任命十個部長，直接控制國家。

2006年4月6日，尼泊爾各地爆發大規模反國王示威。示威並得到反對國王的毛派遊擊隊支持。4月21日，尼泊爾反國王示威和罷工持續兩星期後，在美國和印度施壓下，國王賈南德拉終於屈服，表示願意還政於民。他發表電視演說，宣布會解散內閣，將行政權力交予人民，並將舉行選舉，呼籲反對派推舉首相人選。4月30日，[[柯伊拉腊|柯伊拉腊]]前往王宫接受国王贾南德拉任命并宣誓就职尼泊尔首相。<ref>[http://news.xinhuanet.com/newscenter/2006-04/30/content_4495533.htm]</ref>5月18日，尼泊爾國會一致通過，剝奪國王贾南德拉包括軍權在內的權力，使尼泊爾日後不受王室控制。如果公告內所有建議落實，尼泊爾國王將只成為象徵元首。<ref>{{cite web |url=http://hk.news.yahoo.com/060518/12/1nye1.html |title=存档副本 |accessdate=2006-09-04 |0=2006-09-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20060904092525/http://hk.news.yahoo.com/060518/12/1nye1.html |archivedate=2006-09-04 }}</ref>6月11日，尼泊爾國會通過剝奪國王對國會已通過的議案之否決權，亦即國會議員在將法案簽署成法律前，毋須先徵求國王之同意。<ref>{{cite web |url=http://hk.news.yahoo.com/060611/12/1onji.html |title=存档副本 |accessdate=2006-06-15 |0=2012-07-09 |deadurl=yes |archiveurl=https://archive.is/20120709063445/http://hk.news.yahoo.com/060611/12/1onji.html |archivedate=2012-07-09 }}</ref>政府和尼共（毛派）开始谈判决定国家的未来。11月7日，尼共（毛派）和政府达成协议，放弃武装斗争，加入政府。<ref>[http://www.ndtv.com/template/template.asp?template=nepalmaoists&slug=Koirala+hails+peace+deal&id=96016&callid=1]</ref>

=== 王室被罷黜 ===
2007年12月23日，尼泊尔政府与[[尼泊爾共產黨_(毛派)|毛派]]反政府组织达成协议，同意废除持续了240年之久的[[君主政体|君主政体]]<ref>{{cite web|url = http://news.bbc.co.uk/chinese/simp/hi/newsid_7150000/newsid_7158600/7158690.stm|title = 尼泊尔同意废除君主政体|date=2007年12月24日||accessdate=2007年12月24日||publisher=BBC中文网}}</ref>，以換取毛派保留在聯合政府及交還在武裝衝突時所佔領的土地及物業。在王室被廢黜後，尼泊爾會舉行國民議會選舉<ref name=abolish>{{cite web| title =Nepalese monarchy to be abolished| publisher =BBC| date =2007年12月24日| url =http://news.bbc.co.uk/1/hi/world/south_asia/7158670.stm| accessdate =2007-12-25}}</ref>。

尼泊爾國會在12月28日進行投票，以確認與毛派的和平協議。結果國會以270票對3票大比數通過協議<ref>{{cite web|url=http://news.bbc.co.uk/2/hi/south_asia/7163034.stm |title=Vote to abolish Nepal's monarchy|publisher=BBC|date=2007-12-28|accessdate=2007-12-28}}</ref>。

根据协议，2008年5月28日，新选出的制宪会议正式确认了议会关于废除君主制的决议，尼泊爾宣告成立聯邦民主共和國，从而在宪法上终止了君主制的存在。賈南德拉成為尼泊爾最後一位君主。

== 家庭 ==
賈南德拉已婚，有一子一女。在君主政體被廢除之前，他的兒子[[帕拉斯·比爾·比克拉姆·沙阿·德瓦|帕拉斯]]是王位的繼承人。他現在生活在當時臨時政府安排的[[龍樹|龍樹]]宮。他的新住所包括10座建築，在他離開後他的皇宮成了博物館，而他用鑽石和紅寶石鑲嵌的皇冠和皇家權杖，以及所有其他皇冠珠寶和皇家財產，也成為政府財產。
