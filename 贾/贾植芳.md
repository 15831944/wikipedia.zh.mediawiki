'''贾植芳'''（{{bd|1916年||2008年|4月24日|}}）作家、诗人。曾任[[震旦大学|震旦大学]]中文系主任、[[复旦大学|复旦大学]]中文系教授，兼复旦大学图书馆馆长等职<ref>
{{cite web 
|url = http://chinese.fudan.edu.cn/d7/a4/c12493a120740/page.htm
|title = 贾植芳
|publisher = 复旦大学中国语言文学系
|language = zh-hans
}}</ref>。曾是复旦大学中国现当代文学和比较文学学科的『学科带头人』。也曾为中国作家协会、上海作家协会等诸多机构会员、理事。<ref>
{{cite web 
|url = http://www.fudan.edu.cn/entries/view/1980/
|title = 贾植芳
|publisher = 复旦大学
|language = zh-hans
}}</ref>。[[中华人民共和国|中华人民共和国]]建立后，因与胡风私交受到牵连，成为[[胡风反革命集团案|胡风反革命集团案]]中的『骨干分子』，蒙冤入狱并遭受迫害多年，因其在胡案中未出卖朋友，甚至仗义执言而受到后人敬仰<ref>{{cite journal 
|url = http://rmfyb.chinacourt.org/paper/html/2017-09/01/content_129674.htm?div=-1
|title = 写好“人”字最后一捺
|publisher = 人民法院报
|author = 杨立波
|language = zh-hans
}}</ref>。著有学术自选集《历史的背面》、回忆录《狱里狱外》等，部分论著辑为《贾植芳文集》。

==生平==

1955年受[[胡风反革命集团案|胡风反革命集团案]]牵连入狱，1967年正式获刑12年，出狱后在复旦印刷厂上班。1979年获得平反<ref>{{cite journal 
|url = http://www.thepaper.cn/baidu.jsp?contid=1701866
|title = 高考故事｜作家李辉：考进复旦后偶遇恩师贾植芳，改变一生
|publisher = 澎湃新闻
|author = 陈竹沁 彭琪
|language = zh-hans
}}</ref>。
