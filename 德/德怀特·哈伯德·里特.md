== 生平 ==
1956年，里特出生在[[美国|美国]][[俄亥俄州|俄亥俄州]][[克利夫兰_(俄亥俄州)|克利夫兰]]，他是美国的电影导演，著名作品有《越狱》、《识骨寻踪》和《铁拳》。<ref>{{cite web |url=http://people.mtime.com/899076/ |title=德怀特·H·里特的作品 |accessdate= |publisher=时光网 |work= |language =zh-cn}}</ref>
