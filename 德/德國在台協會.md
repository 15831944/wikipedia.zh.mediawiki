'''德國在台協會'''（{{lang-de|Deutsches Institut Taipei}}）是[[德國|德國]]派駐在[[台灣|台灣]]的[[德國駐外機構列表|代表處]]，負責維護、促進非邦交性的[[中華民國－德國關係|德台實質雙邊關係]]<ref name="協會">{{Cite web|title=德國在台協會|url=http://www.taipei.diplo.de/Vertretung/taipei/zh-tw/02-German__Institute/0Seite__Deutsches__Institut.html|publisher=德國在台協會|accessdate=2015-03-13|deadurl=yes|archiveurl=https://web.archive.org/web/20150402013851/http://www.taipei.diplo.de/Vertretung/taipei/zh-tw/02-German__Institute/0Seite__Deutsches__Institut.html|archivedate=2015年4月2日|df=}}{{zh-hant}}</ref>，並照管德國在台利益<ref name="德國外交部">{{lang|en|{{Cite web |title=Taiwan |url=http://www.auswaertiges-amt.de/EN/Aussenpolitik/Laender/Laenderinfos/01-Nodes/Taiwan_node.html |publisher=[[德國外交部|Federal Foreign Office, Germany]]|accessdate=2015-03-13}}}}{{en}}</ref>。協會的業務除了增進德國與台灣在[[文化|文化]]、[[經濟|經濟]]等領域的往來之外<ref name="協會"></ref>，還包括處理[[簽證|簽證]]、[[護照|護照]]相關的[[領事|領事]]事務、替在台德僑提供[[公民|公民]]服務<ref name="領事曁公民服務">{{Cite web|title=領事曁公民服務|url=http://www.taipei.diplo.de/Vertretung/taipei/zh-tw/03-Citizen__Services/0Informationen__Buergerdienste.html|publisher=德國在台協會|accessdate=2015-03-13|deadurl=yes|archiveurl=https://web.archive.org/web/20150225154759/http://www.taipei.diplo.de/Vertretung/taipei/zh-tw/03-Citizen__Services/0Informationen__Buergerdienste.html|archivedate=2015年2月25日|df=}}{{zh-hant}}</ref>、推廣赴德旅遊等<ref name="首頁"></ref>，在部分場合或事務下亦會作為德國的代表與台灣政府單位接洽，或扮演台灣官方派員赴德時的聯絡單位<ref name="華夏導報">{{Cite news |url=http://epaper.pccu.edu.tw/newsDetail.asp?NewsNo=4812 |title= 
德國在台協會副處長Herr Helmut Lüders與德外交部Frau Groneick蒞臨文大參觀 |publisher=華夏導報 |author=蔡玉芳 |date=2004-04-30 |accessdate=2015-03-13}}{{zh-hant}}</ref>。德國在台協會的最高負責人稱作處長（{{lang|de|Generaldirektor}}），為德國駐台代表<ref name="八十九年">{{Cite book |title=中華民國八十九年外交年鑑 |publisher=[[中華民國外交部|外交部]]外交年鑑編輯委員會 |year=2001 |location=台北 |isbn=957-02-8186-3 |page=頁726}}{{zh-hant}}</ref>。

==沿革==
[[德意志聯邦共和國|德意志聯邦共和國]]自1949年起在西德地區成立，但並未與該年因[[國共內戰|國共內戰]]而遷至台灣的[[中華民國政府|中華民國政府]]締結邦交關係。1972年聯邦德國和[[中華人民共和國|中華人民共和國]]建交後，同台灣間的政治關係也受到[[波昂|波昂]]對華政策所限<ref name="我國與德國的雙邊關係">{{Citation |title=我國與德國的雙邊關係 |author=[[胡為真|胡為真]] |date=2003-01-23 |accessdate=2015-03-13|publisher=[[中華民國外交部|中華民國外交部]] |url=http://npl.ly.gov.tw/npl/report/921203/14.pdf |format=pdf}}{{zh-hant}}</ref>{{rp|p4}}，因此在台德國單位中並沒有正式的[[外交代表機構|外交代表機構]]，而僅有1950年代起駐台的[[德國學術交流總署|德國學術交流總署]]<ref name="德國外交部"></ref>、1963年6月成立的德國文化中心<ref name="八十八年">{{Cite book |title=中華民國八十八年外交年鑑 |publisher=[[中華民國外交部|外交部]]外交年鑑編輯委員會 |year=2000 |location=台北 |isbn=957-02-5836-5 }}{{zh-hant}}</ref>{{rp|p709}}（2009年7月1日更名為[[歌德學院|台北歌德學院]]<ref name="九十九年">{{Cite book |title=中華民國九十九年外交年鑑 |publisher=[[中華民國外交部|外交部]]外交年鑑編輯委員會 |year=2012 |location=台北 |isbn=978-986-03-2422-8 |page=頁124}}{{zh-hant}}</ref>）與1981年5月26日創設的{{Link-de|德國經濟辦事處|Deutsches Wirtschaftsbüro Taipei}}<ref name="八十八年"></ref>{{rp|p709}}。此後，台灣與西德關係，乃至於[[兩德統一|兩德統一]]後的德台關係有了進一步的開展<ref name="我國與德國的雙邊關係"></ref>{{rp|p5}}，而德國也已是台灣在歐洲最大的貿易伙伴<ref name="八十八年"></ref>{{rp|p193}}。2000年2月1日時，德方始於[[台北市|台北市]]建立德國在台協會<ref name="協會"></ref>，並派遣外交官接掌會務<ref name="我國與德國的雙邊關係"></ref>{{rp|p5}}。創立後的協會辦事處座落於[[中山區_(台北市)|中山區]]的[[民生東路|民生東路]]與[[建國南北路_(臺北市)|建國北路]]交叉口，地址為民生東路三段2號4樓<ref name="九十六年">{{Cite book |title=中華民國九十六年外交年鑑 |publisher=[[中華民國外交部|外交部]]外交年鑑編輯委員會 |year=2008 |location=台北 |isbn=978-986-01-5731-4 |page=頁576}}{{zh-hant}}</ref>。

2008年，德國在台協會的處址經{{Link-de|德國聯邦建築暨區域規劃局|Bundesamt für Bauwesen und Raumordnung}}（{{lang|de|BBR}}）確認，判定其存有結構缺陷的問題，已經無法繼續使用，因此[[德國外交部|德國外交部]]在當時世界第一高樓——[[台北101|台北101]]內租下第33樓、作為協會新址，工程於2009年1月8日展開計劃。建築區劃局為了新址的闢建，提出了內含[[地震|地震]]安全說明的[[可行性研究|可行性研究]]，根據此份報告，在樓板面積約2,600[[平方公尺|平方公尺]]的高層大廈中，提供的可用空間不僅有1,700平方公尺，如此一來，便得同時符合德國標案件築品質的要求、及台灣當地的建築概念法則。此外，進度亦由於在[[招標|招標]]時物色合適的台灣總承包商、還有在取得建築許可時，與台北101管理方進行的協議上遭遇困難，一度嚴重落後<ref name="BBR">{{lang|de|{{Cite web |url=http://www.bbr.bund.de/BBR/DE/Bauprojekte/Ausland/BotschaftenKonsulate/Taipei/taipei.html |title=Deutsches Institut Taipei / Taiwan |publisher={{Link-de|德國聯邦建築暨區域規劃局|Bundesamt für Bauwesen und Raumordnung|Bundesamt für Bauwesen und Raumordnung (BBR)}} |accessdate=2015-03-13}}}}{{de}}</ref>。

2011年1月，[[歐盟|歐盟]]通過台灣居民赴歐盟國家的旅遊免簽政策，此後德國外交部再度修改其送交給建築區劃局的規劃申請書，這使得建築區劃局暫停施工，而規劃書也為了讓台灣當地的建築事務所、台北101管理方取得必要的追加[[施工許可|施工許可]]，而進行更新。同年7月1日，德國在台協會新址開工，至11月18日宣告完成<ref name="BBR"></ref>。2012年3月初，德國在台協會正式遷入台北101的33樓辦公<ref name="協會遷">{{Cite web |url=http://www.taipei.diplo.de/Vertretung/taipei/zh-tw/02-German__Institute/250512/BG-2505.html |title=2012年5月25日德國在台協會喬遷晚會與音樂饗宴活動照片集錦 |publisher=德國在台協會 |accessdate=2015-03-13 |archive-url=https://web.archive.org/web/20121226161834/http://www.taipei.diplo.de/Vertretung/taipei/zh-tw/02-German__Institute/250512/BG-2505.html |archive-date=2012-12-26 |dead-url=yes }}{{zh-hant}}</ref><ref name="經濟辦事處遷">{{Cite web |url=http://www.taiwan.ahk.de/cn/news/details/artikel/the-german-institute-taipei-is-relocating/?cHash=11496b50b8ed0af11fc712cd30adfab4 |title=德國在台協會搬新家! |publisher={{Link-de|德國經濟辦事處|Deutsches Wirtschaftsbüro Taipei}} |date=2012-02-24 |accessdate=2015-03-13 |archive-url=https://web.archive.org/web/20161216032606/http://www.taiwan.ahk.de/cn/news/details/artikel/the-german-institute-taipei-is-relocating/?cHash=11496b50b8ed0af11fc712cd30adfab4 |archive-date=2016-12-16 |dead-url=yes }}{{zh-hant}}</ref>。

==組織與職掌==
德國在台協會作為德國駐台代表處，設有若干領事事務單位（{{lang|de|Konsularabteilung}}），對[[中華民國護照|中華民國護照]]持有人或第三國國籍者的的服務項目包括審理、核發[[申根簽證|申根簽證]]、各類文件認證等<ref name="簽證資訊">{{Cite web |title=簽證資訊 |url=http://www.taipei.diplo.de/Vertretung/taipei/zh-tw/01-Welcome/Visabestimmungen/Seite__Visa.html |publisher=德國在台協會 |accessdate=2015-03-13 |archive-url=https://web.archive.org/web/20150220051602/http://www.taipei.diplo.de/Vertretung/taipei/zh-tw/01-Welcome/Visabestimmungen/Seite__Visa.html |archive-date=2015-02-20 |dead-url=yes }}{{zh-hant}}</ref>。與{{Link-en|德國國籍法|German nationality law|德國公民}}相關的業務範疇則有提供建議、緊急救助等協助項目，和[[德國護照|德國護照]]申辦等<ref name="領事曁公民服務"></ref>。

德國在台協會的其他單位還有經濟部門（{{lang|de|Wirtschaftsabteilung}}）、文化部門（{{lang|de|Kulturabteilung}}）和新聞部門（{{lang|de|Presseabteilung}}）<ref name="部門">{{Cite web |title=處長及各相關部門 |url=http://www.taipei.diplo.de/Vertretung/taipei/zh-tw/02-German__Institute/Generaldirektor__und__Abteilungen/Generaldirektor__und__Abteilungen.html |publisher=德國在台協會 |accessdate=2015-03-13 |archive-url=https://web.archive.org/web/20150402014046/http://www.taipei.diplo.de/Vertretung/taipei/zh-tw/02-German__Institute/Generaldirektor__und__Abteilungen/Generaldirektor__und__Abteilungen.html |archive-date=2015-04-02 |dead-url=yes }}{{zh-hant}}</ref>，以支援德商協助工作、[[德國文化|德國文化]]在台工作，以及包括德國留學資訊、德語促進活動等方面在內的交流計畫<ref name="華夏導報"></ref>。此外，德國在台協會尚於2010年起成立「持續力小組」（{{lang|de|Team Nachhaltigkeit}}），推動德國與台灣在[[永續發展|永續發展]]上的合作<ref>{{Cite web |title=持續力-德國在台協會的中心理念 |url=http://www.taipei.diplo.de/Vertretung/taipei/zh-tw/08-Nachhaltigkeit/Ziele__und__Team/Seite__Leitmotiv__des__Deutschen__Institutes.html |publisher=德國在台協會 |accessdate=2015-03-13 |archive-url=https://web.archive.org/web/20150316131531/http://www.taipei.diplo.de/Vertretung/taipei/zh-tw/08-Nachhaltigkeit/Ziele__und__Team/Seite__Leitmotiv__des__Deutschen__Institutes.html |archive-date=2015-03-16 |dead-url=yes }}{{zh-hant}}</ref>。
