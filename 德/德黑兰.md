== 歷史 ==

[[File:Shamsol-emereh-golestan-pal.jpg|thumb]]
[[File:Green-Palace-Tehran.JPG|thumb]]

德黑蘭古稱「拉伊」（Ray），這一詞在語源上與[[古波斯語|古波斯語]]及《[[波斯古經|波斯古經]]》裡的[[雷伊_(伊朗)|剌伽]]（Rhaga）有關係<ref>{{Harvnb|Adle|Hourcade|1992|p=22}}</ref>。剌伽是使用伊朗語的米底人及阿契美尼德人的活躍地區。在祆教的波斯古經裡，剌伽是[[阿胡拉·馬茲達|阿胡拉·馬茲達]]所創造的第12個聖地<ref name="Minorsky (1997)">{{Harvnb|Minorsky|Bosworth|1997}}</ref>，在古波斯文的銘文裡提到剌伽是一個省份，[[大流士一世|大流士一世]]在剌伽派遣援軍幫助他的父親敉平安息的動亂<ref name="Minorsky (1997)" />。剌伽附近的[[德馬峰|德馬峰]]在《列王紀》裡所描述為是弗雷頓綑綁龍形魔鬼[[查哈克|查哈克]]（Zahhak）的地方，德馬峰在波斯神話和傳說裡都顯得十分重要<ref name="Tafazolli">{{Harvnb|Tafazolli|}}</ref>，祆教裡的原人、《[[列王紀_(伊朗)|列王紀]]》裡的第一位國王蓋約馬爾特據說就居住在德馬峰<ref name="Tafazolli" />。在傳說裡，達馬萬德的建立也是歸功於他<ref name="Tafazolli" />。傳說中的射手[[阿拉什|阿拉什]]（Arash）就在德馬峰犧牲他的性命來射出一箭，箭矢所落之處便形成了[[伊朗|伊朗]]與[[圖蘭|圖蘭]]的分界<ref name="Tafazolli" />，每年的[[星節|星節]]（Tiregan）就是紀念和慶祝這個傳說。根據一個波斯傳說，國王馬努切赫爾出生在達馬萬德<ref name="Tafazolli" />。據說在1815年8月31日，達馬萬德舉行慶典以慶祝[[查哈克|查哈克]]的死忌<ref name="Tafazolli" />。根據祆教的傳說，查哈克會在末日之前被伊朗英雄加沙斯帕殺死<ref name="Tafazolli" />。雖然現代史學家認為祆教的發源地是在[[大呼羅珊|大呼羅珊]]，但一些中古波斯文獻指出拉伊是祆教創始人[[查拉圖斯特拉|查拉圖斯特拉]]的出生地<ref>{{Harvnb|Curtis|Stewart|2005|p=37}}</ref>。

[[File:Tehran_Conference,_1943.jpg|thumb]]]]

在[[薩珊王朝|薩珊王朝]]統治時期，[[伊嗣俟三世|伊嗣俟三世]]（Yazdagird III）在641年於拉伊向他的子民作出最後的呼籲後便逃向[[大呼羅珊|呼羅珊]]，以逃避阿拉伯人的侵略<ref name="Minorsky (1997)" />。舍赫爾·巴努神殿位於德黑蘭南郊，神殿只向女子開放，它是用以紀念耶斯提澤德的女兒，據說她是第三位[[什葉派|什葉派]]伊瑪目[[侯賽因·本·阿里·本·阿比·塔利卜|塔利卜]]的妻子<ref name="Minorsky (1997)" />。拉伊是抵抗阿拉伯人侵略的巴赫拉姆六世的後裔及米倫家族的邑地<ref name="Minorsky (1997)" />。在那次侵略裡，阿拉伯人攻陷了拉伊，並摧毀了城鎮，其後又把它重建<ref name="Minorsky (1997)" />。在10世紀，許多伊斯蘭地理學家都在文獻裡詳細描述拉伊<ref name="Minorsky (1997)" />。雖然阿拉伯人希望可以在拉伊體現[[巴格達|巴格達]]的繁華，但是他們在當地勢孤力弱，而且當地的波斯人遍及各個階層<ref name="Minorsky (1997)" />，[[烏古斯人|烏古斯人]]分別在1035年及1042年將拉伊廢棄，拉伊在塞爾柱及花剌子模統治時期一度恢復過來<ref name="Minorsky (1997)" />。根據當時伊斯蘭史學家的記載，蒙古人屠殺了拉伊的居民，並完全廢棄拉伊。在薩珊王朝的編年史裡，拉伊已經淪為一個次要的城市<ref name="Minorsky (1997)" />。

德黑蘭的名字由來不明<ref>{{cite web |url = http://universityandheritage.net/SIFU/XII_Hanoi_2009/en/abstracts/html/14.html |title = ''A Research on Adaptation of Historic Urban Landscapes ; The Case of The Historical City of Tehran'' |publisher = Tehran Historical City Office |accessdate=2011-07-07 |author = Samira Behrooz、Katayoun Karampour |language = en}}</ref>，它在9世紀時期是一個村落，但是名氣不及當時較繁華的[[雷伊_(伊朗)|剌伽]]。[[蘇菲主義|蘇菲主義]]學者納吉姆丁·拉齊認為拉伊在蒙古人入侵前的人口達500,000人。13世紀蒙古人毀滅了拉伊後，許多居民逃到德黑蘭。在一些早期的文獻裡，它被稱為「剌伽的德黑蘭」。地理學家[[哈姆杜拉·穆斯圖菲|哈姆杜拉·穆斯圖菲]]（Hamdollah Mostowfi）在他的《心之喜》（Nuz'hat al-Qulub）裡描述德黑蘭是一個著名的村落。

紀念[[舍赫爾巴努|舍赫爾巴努]]（Shahrbanu）的神廟位於德黑蘭附近，她是第四位什葉派伊瑪目[[阿里·宰因·阿比丁|阿里·宰因·阿比丁]]的母親，鄰近的山岳都以她的名字命名。不過，一些文獻指出神廟原本是供奉水和豐產女神[[阿娜希塔|阿娜希塔]]（Anahita），在伊斯蘭時代伊朗人皈依伊斯蘭教後，人們為了保護神廟而改變了它的用途{{NoteTag|根據一些說法，舍赫爾巴努曾經到過神廟敬拜她的祖先信仰（祆教的水之女神阿娜希塔）<ref>{{Harvnb|Jubber|2010|p=44}}</ref>。}}。

[[卡斯提亞王國|卡斯提亞王國]]使節[[克拉维约|魯伊·岡薩雷斯·克拉維約]]（Ruy Gonzáles de Clavijo）可能是第一位到訪德黑蘭的歐洲人，他在1404年7月前往[[帖木兒|帖木兒]]的首都[[撒馬爾罕|撒馬爾罕]]（位於今[[烏茲別克|烏茲別克]]）途中路徑德黑蘭，當時帖木兒控制著伊朗，而當時的德黑蘭仍沒有城牆保護<ref>{{Harvnb|Encyclopaedia Britannica, inc|2006|p=192}}</ref>。

18世紀初，[[卡里姆汗|卡里姆汗]]下令在德黑蘭興建一所皇宮及政府辦公室，或許是要表明德黑蘭是首都，但後來卻把政府辦公室遷往[[設拉子|設拉子]]。1796年，[[卡扎爾王朝|卡扎爾王朝]]的[[阿迦·穆罕默德·汗|阿迦·穆罕默德·汗]]加冕為王，同時德黑蘭成為了首都{{NoteTag|德黑蘭取代了設拉子成為首都<ref>{{Harvnb|Singh|2002|p=130}}</ref>。}}，直至現在。

在1920年代至1930年代，德黑蘭在伊朗[[沙阿|沙阿]][[禮薩汗|禮薩汗]]的統治下幾乎從零開始建設<ref>{{Harvnb|Khosravi|2008|p=9}}</ref>，禮薩汗認為諸如[[古列斯坦皇宮|古列斯坦皇宮]]、[[吐普哈內廣場|吐普哈內廣場]]（Toopkhaneh）、城堡、要塞等古建築不符合城市的現代化發展，這些建築都被有系統地摧毀，如國家銀行、警察總部、電報局、軍事學院等現代化伊朗式建築在原址上建立起來。[[室內大市集|室內大市集]]被一分為二，一些古蹟被拆除，用以建造寬直的大道。許多波斯花園都被納入重建項目裡。

在[[第二次世界大戰|第二次世界大戰]]期間，[[英國|英國]]及[[蘇聯|蘇聯]]軍隊進入過德黑蘭{{NoteTag|英國軍隊佔領南部，蘇聯軍隊佔領北部，以保護德黑蘭免受德軍入侵<ref>{{Harvnb|Brune|2003|p=617}}</ref>。}}。1943年，[[美國總統|美國總統]][[富蘭克林·德拉諾·羅斯福|羅斯福]]、[[英國首相|英國首相]][[溫斯頓·丘吉爾|丘吉爾]]及[[蘇聯部長會議主席|蘇聯部長會議主席]][[史太林|史太林]]出席在德黑蘭舉行的[[德黑蘭會議|德黑蘭會議]]<ref>{{Harvnb|Miscamble|2007|p=55}}</ref>。

在沙阿[[穆罕默德·禮薩·巴列維|穆罕默德·禮薩·巴列維]]的統治下，德黑蘭在1960年代及1970年代的發展迅速，現代化的建築重塑了城市的面貌，未來數十年的城市發展計劃也制定了出來，1979年的革命使這些計劃中止，新政權落實了這些計劃的一些部分，如[[默德塔|默德塔]]的建造。

在1980年至1988年的[[兩伊戰爭|兩伊戰爭]]期間，德黑蘭多次被[[飛毛腿飛彈|飛毛腿飛彈]]襲擊{{NoteTag|伊朗為了報復伊拉克的空襲，向巴格達市中心以南發射了三枚飛毛腿飛彈，伊拉克亦以其蘇制飛毛腿飛彈還擊，德黑蘭被飛毛腿飛彈擊中33次<ref>{{Harvnb|Hume|1994|p=149}}</ref>。}}。

== 氣候 ==
[[File:Damavand_from_dizin.jpg|thumb]]滑雪場眺望[[德馬峰|德馬峰]]。]]

德黑蘭屬於[[陸性率|大陸性]][[半乾旱氣候|半乾旱氣候]]（[[柯本氣候分類法|柯本氣候分類法]]裡的寒冷半乾旱氣候）。德黑蘭的氣候在很大程度上受到地理位置的影響，[[厄爾布爾士山脈|厄爾布爾士山脈]]（Alborz）在德黑蘭以北，荒漠則在南面。城內各地區的海拔高度不一，北部山丘地帶的氣候通常較南部的平原涼爽。夏季炎熱、乾旱、少雨，相對濕度較低，晚間天氣清涼。冬季冷凉、干燥，但降水多于夏季。大部分的[[降水|降雨]]都發生在晚秋至春季之間，全年濕度較平均。1月为最冷月（最低溫度-1 °C，最高溫度8 °C），7月为最熱月（平均最低溫度24 °C，平均最高溫度37 °C）<ref>{{cite web |url = http://www.irantour.org/Iran/Climate.html |title = Climate of Tehran |publisher = Iran Tour |accessdate=2011-07-08 |language = en}}</ref>。

德黑蘭的氣候較伊朗许多地區溫和，但极端的天氣狀況也并不罕见。极端最高气温为43 °C，极端最低气温为-15 °C。2008年1月5日和1月6日，在過往數年降雪量較少的情況下，德黑蘭突然遭受暴雪及低溫侵襲，積雪結冰的情況嚴重，部長會議宣佈進入[[緊急狀態|緊急狀態]]，1月7日及1月8日停工停課<ref>{{cite web |url = http://www.news24.com/World/News/21-die-in-Iran-snow-storms-20080107 |title = 21 die in Iran snow storms |publisher = News24 |date = 07-01-2008 |accessdate=2011-07-08 |language = en}}</ref><ref>{{cite web |url = http://news.bbc.co.uk/2/hi/middle_east/7178192.stm |title = Dozens killed in Iran blizzards |publisher = BBC |date = 09-01-2008 |accessdate=2011-07-08 |language = en}}</ref>。

== 人口 ==

[[File:Tehran_Population_(1956-2016).png|thumb]]

在2006年，德黑蘭的人口總數約為780萬人<ref>{{Harvnb|Sorensen|2010|p=85}}</ref>。由於受到各國文化影響，德黑蘭的居民包括了全國各個民族及語族，他們代表了伊朗的各個民族（百分率不一）。德黑蘭的母語是帶有德黑蘭方言特点的[[波斯語|波斯語]]，大部分的居民都是[[波斯人|波斯人]]<ref name="Abbasi-Shavazi (2009) 100-101" />。居住在德黑蘭的少數民族包括[[阿塞拜疆人|阿塞拜疆人]]、[[庫爾德人|庫爾德人]]、[[俾路支人|俾路支人]]、[[亞美尼亞人|亞美尼亞人]]、巴克提爾人（Bakhtiari）、[[亞述人|亞述人]]、塔里什人（Talysh）等。

根據[[德黑蘭大學|德黑蘭大學]]社會學系在2010年的調查發現，在德黑蘭各個地區的所有社會經濟階層當中，63%的德黑蘭人在德黑蘭出生、98%的德黑蘭人懂波斯语、67%的德黑蘭人在民族上自認為波斯人、13%的德黑蘭人略懂歐洲語言<ref>"Chand Darsad Tehranihaa dar Tehran Bedonyaa Amadand"（在德黑蘭出生的德黑蘭人佔多少百分率）[http://www.tabnak.ir/fa/news/133668][http://www.aftabnews.ir/vdchimnzw23nzid.tft2.html][http://www.jahannews.com/vdcgw39qzak9tn4.rpra.html]</ref>。

德黑蘭的民族社會狀況在1980年代發生了急劇的轉變。1979年的[[伊朗伊斯蘭革命|伊朗伊斯蘭革命]]導致了政治、社會及經濟的改變，許多伊朗人承受的政治、社會及宗教壓力增加而離開伊朗，當中不少是德黑蘭人，他們大多出走到[[加拿大|加拿大]]、[[美國|美國]]、[[法國|法國]]、[[瑞典|瑞典]]及其他歐洲國家，當中最多人移居到[[美國|美國]]、[[加拿大|加拿大]]及[[法國|法國]]{{NoteTag|移居出伊朗的人數估計達750,000至1,500,000人<ref>{{Harvnb|Federal Research Division|2004|p=113}}</ref>。}}。

隨後的[[兩伊戰爭|兩伊戰爭]]爆發，伊拉克入侵伊朗，又有一批德黑蘭居民為了逃避戰難而離開，特別是在伊拉克空襲德黑蘭期間。當時伊拉克得到世界強權的支持，經濟上的孤立成為了德黑蘭居民離開的更重要原因，戰爭結束後，他們在習慣了外國的生活後大多沒有再回國。在戰爭期間，大量伊朗西部及伊朗西南接壤伊拉克地區的居民湧入了德黑蘭<ref>{{Harvnb|Khosravi|2008|p=58}}</ref>。

不穩的局勢加上鄰國[[阿富汗|阿富汗]]及[[伊拉克|伊拉克]]的戰事使上百萬名難民湧入伊朗，德黑蘭如同一塊磁石一樣吸引著勞工來到幫助城市進行戰後的修復工作，他們所要求的工資較本地勞工廉宜{{NoteTag|這些難民從事一般伊朗人不願意做的低報酬工作，幾乎所有建築工人都是阿富汗人<ref>{{Harvnb|Burke|Elliot|2008|p=52}}</ref>。}}。在[[聯合國難民署|聯合國難民署]]的協助下，許多難民得以被遣送回國，但德黑蘭仍有可觀數量的阿富汗和伊拉克難民不願意離開，他們對祖國的前景感到悲觀。許多阿富汗難民是通曉波斯語的[[哈扎拉族|哈扎拉族]]及[[塔吉克族|塔吉克族]]，他們會說波斯語的方言，而伊拉克難民主要是來自[[美索不達米亞|美索不達米亞]]的[[什葉派|什葉派]]穆斯林，通曉阿拉伯語。

伊斯兰教[[十二伊瑪目派|十二伊瑪目派]]（[[什叶派|什叶派]]中最大的支派）是目前伊朗的國教，大部分德黑蘭人都被認為是[[十二伊瑪目派|十二伊瑪目派]]的溫和信徒<ref>{{Harvnb|Zonn|Kostianoy|Kosarev|Glantz|2010|p=397}}</ref>，但是由於缺乏獨立的統計，這個說法仍未得到證實。德黑蘭的少數宗教族群包括伊斯兰教[[遜尼派|遜尼派]]，伊斯兰教[[苏菲派|苏菲派]]，[[琐罗亚斯德教|琐罗亚斯德教]]，[[巴哈伊教|巴哈伊教]]，[[猶太教|猶太教]]，[[基督教|基督教]]（包括[[東方亞述教會|東方亞述教會]]、[[亞美尼亞使徒教會|亞美尼亞使徒教會]]、迦勒底天主教會、[[罗马天主教會|罗马天主教會]]、[[基督復臨安息日會|基督復臨安息日會]]、伊朗基督新教教會、亞美尼亞福音教會、伊朗神召會、亞美尼亞天主教會、[[俄羅斯正教會|俄羅斯正教會]]及[[長老宗|長老宗]]教会的信徒），以及[[不可知論|不可知論者]]和[[無神論|無神論者]]。基督教各教派在德黑兰仍有一定影响。

== 遷都 ==

關於遷都的計劃已經被提出過許多次，[[伊朗政府|伊朗政府]]在2010年宣佈出於「安全和行政理由」而遷都的計劃已經有定案<ref name="Finalized">{{cite web |url = http://www.payvand.com/news/10/may/1322.html |title = For Security and Admnistrative Reasons: Plan to Move Capital From Tehran Finalized |publisher = Payvand |date = 2010-05-30 |accessdate=2011-07-08 |author = Maryam Ahamdi |language= en}}</ref>。[[伊朗伊斯蘭會議|伊朗國會]]選定[[沙赫魯德|沙赫魯德]]（Shahroud）、[[伊斯法罕|伊斯法罕]]及[[塞姆南市|塞姆南市]]為未來首都的候選城市。為了避免地震帶來的潛在破壞，政府也有搬遷163所國有企業及一些大學的計劃。總統[[馬哈茂德·艾哈邁迪內賈德|馬哈茂德·艾哈邁迪內賈德]]認為至少5,000,000人應遷出德黑蘭<ref name="Finalized" /><ref>{{cite web |url = http://www.payvand.com/news/10/jul/1064.html |title = Iran Moots Shifting Capital from Tehran |publisher = Payvand |date = 2010-07-07 |accessdate=2011-07-08 |author = Nima Tamaddon |language= en}}</ref>，伊朗當局現正將國防相關的企業遷出德黑蘭<ref>{{cite web |url = http://edition.presstv.ir/detail/137793.html |title = Iran relocating defense industry |publisher = PressTV |date = 07-08-2010 |accessdate = 2011-07-08 |language = en |archive-url = https://web.archive.org/web/20140408131725/http://edition.presstv.ir/detail/137793.html |archive-date = 2014-04-08 |dead-url = yes }}</ref>。

=== 舊城區 ===

在[[巴列維王朝|巴列維]]時期，舊城區經歷了重大的轉變，德黑蘭現存的舊城區包括[[大市集|大市集]]、[[查萊赫梅伊丹|查萊赫梅伊丹]]（Chaleh Meydan）、[[杜拉_(德黑蘭)|杜拉]]（Doulat）、[[彭梅納爾|彭梅納爾]]（Pamenar）。查萊赫梅伊丹是當中最古地的地區。

== 飲食 ==

德黑蘭有許多現代化和時髦的餐廳，款待伊朗人及全球各國的人。德黑蘭最受歡迎的菜式是[[切洛喀巴|切洛喀巴]]（Chelow kabab，「喀巴」源自波斯語，即是烤肉）<ref>{{Harvnb|Marks|2010|p=296}}</ref>。西式快餐也受到年青一代的歡迎。銷售比薩、三文治及烤肉的店舖遍佈全城。

== 經濟 ==
{{seealso|伊朗經濟}}

德黑蘭是伊朗的經濟中心<ref>{{cite web |url = http://www.britannica.com/EBchecked/topic/585619/Tehran/276311/Economy |title = Tehrān |publisher = Britannica Online Encyclopedia |accessdate=2011-07-09 |language = en}}</ref>。約30%公營機構的勞動人口及45%的大型企業都位於德黑蘭、當中近半數的勞動人口都為政府機關工作<ref>{{cite web |url = http://www.comw.org/rma/fulltext/070417cordesman.pdf |title = Warfighting and Prolifeation in the Middle East |publisher = Strategy Center for Strategic and International Studies |date = 2007-04-17 |accessdate=2011-07-09 |author = Anthony H. Cordesman、Arleigh A. Bruke |language = en}}</ref>。其他的勞動力是工廠工人、店主、體力勞動者及運輸工人。由於伊斯蘭政府與西方的關係緊張，只有少數外國企業在德黑蘭運作，而在[[伊朗伊斯蘭革命|伊朗伊斯蘭革命]]之前，許多外國企業都在這個地區運作。現在，汽車制造業、電子及電力設備、軍工、紡織、制糖、水泥及化工都是德黑蘭的主要現代產業，德黑蘭還是首屈一指的地毯及家具銷售中心。城市南部靠近[[雷伊_(伊朗)|雷伊]]還有煉油廠。德黑蘭有四個機場，分別是[[梅赫拉巴德國際機場|梅赫拉巴德國際機場]]、[[伊瑪目霍梅尼國際機場|伊瑪目霍梅尼國際機場]]、[[加勒赫莫爾吉機場|加勒赫莫爾吉機場]]（Ghal'eh Morghi airfield）及[[杜森塔佩空軍基地|杜森塔佩空軍基地]]（Doshan Tappeh Air Base）。

德黑蘭相當倚賴私家車、巴士、摩托車和的士，她是世界上最倚賴汽車的城市之一。[[德黑蘭證券交易所|德黑蘭證券交易所]]是[[國際證券交易所聯會|國際證券交易所聯會]]的正式成員，同時是[[歐亞證券交易所聯會|歐亞證券交易所聯會]]的創始會員，它是近年來表現最好的證券交易所之一<ref>{{cite web |url = http://news.bbc.co.uk/2/hi/business/3129995.stm |title = Iran blocks share price gains |publisher = BBC News |date = 06-08-2003 |accessdate=2011-07-10 |language = en}}</ref>。

[[德黑蘭地鐵|德黑蘭地鐵]]是世界上最潔淨的的地鐵系統之一，日载乘客达两万多人，<ref name="Reuters">{{cite news |url = http://in.reuters.com/article/2011/03/05/idINIndia-55349520110305 |title = Tehran Metro head resigns in row with Ahmadinejad |newspaper=Reuters |date = 2011-03-05 |accessdate=2011-03-05}}</ref>年客运量2010年为480.1万人，<ref>http://metro.tehran.ir/Portals/1/articlepdf/1-Statistics.pdf</ref>同時是中東最大型的地鐵系統<ref>{{cite web |url = http://www.eurotransportmagazine.com/news/industry-news/china-invests-in-tehran-metro/ |title = China invests in Tehran Metro |publisher = Eurotransport Magazine |date = 2010-10-19 |accessdate=2011-07-10 |author = Dubai Metro |language = en}}</ref>。德黑蘭地鐵的可行性研究及理論計劃早在1970年代便開展。在8條規劃的地鐵線路當中，2條地鐵線路在2001年開始營運。德黑蘭地鐵現時有4條地鐵線路正在營運，现總長度達152公里，共85个站。<ref>{{cite news|title = "12km section of Tehran Metro inaugurated"|url = http://www.tehrantimes.com/economy-and-business/115284-12km-section-of-tehran-metro-inaugurated|accessdate = 2014-09-06|newspaper = TehranTimes|archive-url = https://web.archive.org/web/20150317035633/http://www.tehrantimes.com/economy-and-business/115284-12km-section-of-tehran-metro-inaugurated|archive-date = 2015-03-17|dead-url = yes}}</ref>另外還有2條地鐵線路正在施工中，预计所有地铁修建完成时总长度为430公里。<ref>{{cite web |url = http://www.railway-technology.com/projects/tehranmetro/ |title = Tehran Metro, Iran |website=Railway-Technology.com |date = |accessdate=2014-06-29}}</ref>搭乘任意一般地铁并不管距离需要4500[[伊朗里亚尔|伊朗里亚尔]]

=== 道路公共交通 ===

[[File:Shariati_street_traffic.jpg|缩略图]]

[[公共汽车|公共汽车]]、[[無軌電車|無軌電車]]及[[巴士快速交通系統|巴士快速交通系統]]都納入了德黑蘭的交通系統。

公共汽車早在1920年代已投入服務。德黑蘭的4個巴士總部提供轉車服務，該4個巴士總站分別位於南部、東部、西部及貝伊哈格希公園<ref>{{cite web |url = http://chutti.pk/Tehran/Transportation/Getting-Around/Buses.html |title = Buses |publisher = Chutti.pk |accessdate=2011-07-10 |language = en}}</ref>。

無軌電車在1992年開始投入使用，採用[[斯柯達公司|斯柯達公司]]出產的65部[[铰接客车|雙節]]電車<ref name="Murray (2000) p57,99">{{Harvnb|Murray|2000|pp=57,99}}</ref>，這是伊朗第一個無軌電車系統，也是目前唯一的無軌電車系統<ref name="Murray (2000) p57,99" />。2005年，無軌電車系統的5條路線全部以[[伊瑪目侯賽因廣場|伊瑪目侯賽因廣場]]（Imam Hossein Square）及[[德黑蘭地鐵|德黑蘭地鐵]]2號線的[[伊瑪目侯賽因地鐵站|伊瑪目侯賽因地鐵站]]為起點<ref name="Trolleybus">Trolleybus Magazine No.265（2006年1－2月） 第17-18頁</ref>，當中2條路線向東北延伸，這些路線都設在廣闊[[行車道|行車道]]的中央（沿著[[達馬萬德街|達馬萬德街]]），約每500米都設有一個無軌電車專用的車站。另外3條路線向南而行，以混合交通的形式運行。2組路線以[[半直通|半直通]]及本地形式（所有車站都會停車）運作<ref name="Trolleybus" />。

2008年，[[德黑蘭巴士快速交通|德黑蘭巴士快速交通]]在德黑蘭市長穆罕默德-巴蓋爾·加利巴夫的主持下正式啟用。德黑蘭巴士快速交通有3條路線，60個車站，網絡覆蓋德黑蘭各個地區。截至2011年，德黑蘭巴士快速交通路線的總長度達100公里，每天的人流量達1,800,000人。

== 空氣污染 ==
[[File:Northern-Tehran.JPG|thumb]]及德黑蘭市面都清晰可見。]]

德黑蘭面臨嚴重的[[空氣污染|空氣污染]]，市内經常出现大范围的[[煙霧|煙霧]]，使人呼吸困難及導致肺部疾病。據估計，德黑蘭每天有27人死於空氣污染引發的疾病<ref>{{cite web |url = http://www.bbc.co.uk/persian/iran/story/2006/09/060909_mv-pollution-tehran.shtml |title = هر روز 27 تهرانی به دليل آلودگی هوا می ميرند |publisher = BBCPersian.com |date = 2006-09-18 |accessdate=2011-07-11 |language=fa}}</ref>。據當地官員所說，恶劣的空氣质量在單一個月內可導致3,600人死亡<ref>{{cite web |url = http://news.bbc.co.uk/2/hi/middle_east/6245463.stm |title = Iran smog 'kills 3,600 in month' |publisher = BBC News |date = 09-01-2007 |accessdate=2011-07-11 |language = en}}</ref>。空氣污染的源頭8成來自汽車尾气<ref>{{cite web |url = http://www.payvand.com/news/09/jan/1007.html |title = Car exhaust fumes blamed for over 80% of air pollution in Tehran |publisher = Payvand |date = 2009-01-01 |accessdate=2011-07-11 |language = en}}</ref>，其餘2成來自工業污染。一些估計認為單是摩托車已經佔了德黑蘭空氣污染源頭的3成及噪音污染源頭的5成<ref>{{cite web |url = http://www.payvand.com/news/10/oct/1239.html |title = Motorcycles Account for 30% of Air Pollution in Tehran |publisher = Payvand |date = 2010-10-29 |accessdate=2011-07-11 |language = en}}</ref>。

伊朗在2007年實施[[2007年伊朗汽油配給計劃|汽油配給計劃]]，但在減少污染上的收效甚微。2011年，在交通系統得到改善及由於[[伊朗資助改革計劃|資助改革計劃]]而導致汽油價格上升的情況下，政府期望可以改善交通和污染問題<ref>{{cite web |url = http://www.turquoisepartners.com/iraninvestment/IIM-Feb11.pdf |title = Iran Investment Monthly |accessdate=2011-07-11 |language = en}}</ref>。

德黑兰嚴重的空氣污染主要是以下的原因造成的：

*大部分工業都分佈在德黑蘭市郊，城市充斥著老舊的車輛，不符合當今的排放法規。
*許多市民都使用私家車或的士代步，造成嚴重的交通擠塞。
*德黑蘭的北方是廣袤的[[厄爾布爾士山脈|厄爾布爾士山脈]]，阻隔了裏海吹來的溫暖潮溼氣流，时常出现的[[逆溫|逆溫]]现象令德黑蘭的空氣污染物难以稀释消散。常年乾燥少雲的天氣使德黑蘭日照极为充足,而紫外線加上污染物令空气中臭氧含量上升。此外，據報導伊朗生產的汽油質素欠佳，使污染問題加劇<ref>{{cite journal |title = The smoggiest of all capitals |journal=[[經濟學人|經濟學人]] |pages=40|date = 2011年1月}}</ref>。

政府致力於減少空氣污染，鼓勵的士和巴士以[[壓縮天然氣|壓縮天然氣]]引擎取代汽油引擎。此外，自伊朗伊斯蘭共和國在1979年建立以來便在市中心的交通繁忙時間期間設立了「交通分區」，進入「交通分區」的車輛受到特殊的限制{{NoteTag|不過，商業活動傾向於私家車可通行的地區，通勤者都將他們的車輛停泊在靠近工作的地方，使鄰近「交通分區」地區的交通變得更繁忙<ref>{{Harvnb|Moavenzadeh|Hanaki|Baccini|2002|p=145}}</ref>。}}。政府也希望提高市民對污染危害的意識，例如在全市安裝污染指標顯示器<ref>{{Harvnb|Schuppe|2008|p=23}}</ref>，監測[[懸浮粒子|懸浮粒子]]、[[二氧化氮|二氧化氮]]、臭氧、[[二氧化硫|二氧化硫]]及[[一氧化碳|一氧化碳]]的含量，顯示器還會顯示[[空氣污染指標|空氣污染指標]]，空氣污染指標是以上5種污染物的空氣质量測量的綜合指數，將每種污染物的水平區分為安全、有害及危險。

== 教育及研究 ==
{{seealso|伊朗教育}}

德黑蘭是伊朗最大及最重要的教育中心，現今約有50所高等院校及大學都坐落在[[大德黑蘭|大德黑蘭]]。

自[[精英學校|精英學校]]（Dar ul-Funun）在19世紀中成立後，許多高等教育機構都聚集在德黑蘭，一些高等教育機構在伊朗的政治事件當中展示出它們的重要角色。[[塞繆爾·馬丁·喬丹|塞繆爾·馬丁·喬丹]]（Samuel Martin Jordan）是[[厄爾布爾士高級中學|厄爾布爾士高級中學]]的創立者之一{{NoteTag|厄爾布爾士高級中學的原名是德黑蘭美國學院<ref>{{Harvnb|Ansari|Martin|2002|p=64}}</ref>。}}，德黑蘭的喬丹徑以他的名字命名。在德黑蘭眾多教育機構當中，[[謝里夫理工大學|謝里夫理工大學]]（Sharif University of Technology）是伊朗最享負盛名的理工大學，[[德黑蘭大學|德黑蘭大學]]是伊朗最大及歷史最悠久的國立大學<ref>{{Harvnb|Wedgeworth|1993|p=389}}</ref>，也是中東及中亞最古老的大學之一。

德黑蘭的[[阿米爾卡比爾理工大學|阿米爾卡比爾理工大學]]（Amirkabir University of Technology）、[[沙希德·贝赫什提大学|沙希德·贝赫什提大学]]（Shahid Beheshti University）、[[KN圖什理工大學|KN圖什理工大學]]（K.N.Toosi University of Technology）、[[伊朗科技大學|伊朗科技大學]]（Iran University of Science and Technology）、[[伊朗醫科大學|伊朗醫科大學]]（Iran University of Medical Sciences）、[[薩伊德大學|薩伊德大學]]（Shahed University）及[[塔比阿特莫達勒斯大學|塔比阿特莫達勒斯大學]]（Tarbiat Modares University）都是伊朗排名較高的大學。

一些宗教學校、神學院及國內最大的軍事學院都位於德黑蘭。

== 體育 ==

德黑蘭是第一個主辦[[亞洲運動會|亞洲運動會]]的中東城市，[[1974年亞洲運動會|1974年亞洲運動會]]吸引了25個國家2,363名運動員及官員參與<ref>{{cite web |url = http://en.olympic.cn/games/asian/2004-03-27/121726.html |title = 7th Asian Summer Games: Tehran 1974 |publisher = Official Website of the Chinese Olympic Committee |date = 2004-03-27 |accessdate=2011-07-12 |language = en}}</ref>。

[[File:Tochal-telecabin.jpg|thumb]][[托烏恰爾建築群|架空纜車]]。]]
[[File:Azadistadium_tehran_iran.jpg|thumb]]是全球第4大足球場、中東第1大足球場。]]

阿薩迪體育場是伊朗國家足球隊的主場，配備100,000個[[座位數|座位]]，阿薩迪足球場是伊朗及中東最大的足球場，也是全球最大的足球場之一。許多伊朗足球超級聯賽的頂部賽事都在這裡舉行。2005年，由於發生了致命意外及安全措施不足，[[國際足球聯合會|國際足球聯合會]]下令伊朗限制觀眾進入阿薩迪足球場{{NoteTag|阿薩迪足球場在當時舉行2006世界杯外圍賽，伊朗國家足球隊在賽事當中以2-1擊敗日本國家足球隊，賽事結束後，大批觀眾湧向出口並發生意外，至少5人死亡<ref>{{cite web |url = http://news.bbc.co.uk/2/hi/middle_east/4383939.stm |title = Five die after Iran football game |publisher = BBC News |date = 2005-03-26 |accessdate=2011-07-12 |language = en}}</ref>。}}。[[沙希德達斯特蓋爾迪球場|沙希德達斯特蓋爾迪球場]]（Shahid Dastgerdi Stadium）、[[塔赫季球場|塔赫季球場]]（Takhti Stadium）、沙希德施魯迪球場（Shahid Shirudi Stadium）都是位於德黑蘭的球場。

[[蒂津|蒂津]]滑雪勝地位於德黑蘭以北的[[厄爾布爾士山脈|厄爾布爾士山脈]]，[[托烏恰爾建築群|托烏恰爾滑雪勝地]]是世界上海拔第5高的滑雪勝地<ref>{{Harvnb|Gavin|2009|p=18}}</ref>，海拔高度超過3,730米，托烏恰爾滑雪勝地在1976完工，恰好在沙阿被推翻前不久。

要登上托烏恰爾滑雪勝地必須乘搭8公里長的架空纜車，這可能是世界上最長的架空纜車<ref>{{cite web |url = http://www.tochal.org/en/telecabin_introduction.asp |title = Lines of Telecabin |publisher = Tochal Telecabin |accessdate = 2011-07-12 |language = en |archive-url = https://web.archive.org/web/20081121200006/http://www.tochal.org/en/telecabin_introduction.asp |archive-date = 2008-11-21 |dead-url = yes }}</ref>，最高點的第7站連接3個斜坡，最長的斜坡在南方，那是一個U型的斜坡，由第7站一直向下延伸至第5站。另外2處斜坡位於第7站的北方，有兩條平行的登山吊椅纜車，可到達3,900米接近托烏恰爾的最高點，比架空纜車的第7站更高，其海拔高度是歐洲滑雪勝地都不及的。

托烏恰爾滑雪勝地第7站在1年的8個月內都可以滑雪，靠近德黑蘭也有一些冰川和雪地可供滑雪。伊朗現在有12個滑雪勝地，[[托烏恰爾|托烏恰爾]]、[[蒂津|蒂津]]及[[謝姆沙克|謝姆沙克]]（Shemshak）是當中最著名的滑雪勝地，距德黑蘭的車程只需1至3小時。

德黑蘭是[[伊朗足球超級聯賽|伊朗足球超級聯賽]]多支球隊的根據地，包括[[德黑蘭獨立足球俱樂部|德黑蘭獨立]]（Esteghlal Tehran FC）、[[柏斯波利斯足球俱樂部|柏斯波利斯]]及[[德黑蘭石油足球俱樂部|德黑蘭石油]]（Naft Tehran F.C.）。

== 建築 ==
[[File:Khalvat_Karimkhani_Golestan2.jpg|thumb]] 

德黑蘭是相對古老的城市，具有獨特的建築傳統風格。德黑蘭的考古研究和挖掘都發現其文明可以追溯到公元前6000年的拉伊，如今拉伊已經併入德黑蘭。在歷史裡，德黑蘭在大部分時間裡都是一個人口稀少的村落，直到18世紀末成為首都後才變得相對重要。雖然德黑蘭在[[卡扎爾王朝|卡扎爾王朝]]時期及之前發生過多次地震，但仍保留著一些古代建築。德黑蘭是當今伊朗的[[首都|首都]]，建有現代化的基礎建設。舊城區的[[士紳化|士紳化]]的現象及文化象徵建築物的毀壞引起了一些關注<ref>{{cite web |url = http://www.loc.gov/loc/lcib/0408/iran.html |title = The Style of Tehran |publisher = Library of Congress Information Bulletin |accessdate=2011-07-12 |author = Donna Urschel |language = en}}</ref>。

[[阿扎迪塔|阿扎迪塔]]已是德黑蘭的一個地標，建造阿扎迪塔是為了紀念[[波斯帝國成立2500周年慶典|波斯帝國建國2500周年]]，阿扎迪塔的原名是「沙赫雅德塔」（Shahyad Tower），[[伊朗伊斯蘭革命|伊朗伊斯蘭革命]]後才改名為阿扎迪塔，意思是「自由之塔」。新近建成的[[默德塔|默德塔]]（Milad Tower）可能會取代阿扎迪塔成為德黑蘭的新象徵。默德塔是[[高塔列表|世界第6高的高塔]]，默德塔建築群建有數間餐廳、一所5星級酒店、一所會展中心、一所世貿中心及一個科技園。由於區內的地震活動頻繁，德黑蘭的樓房層數較低，但為了應付人口增加而不得不建造現代化的高樓建築。自1830年以後，德黑蘭再也沒有發生過嚴重的地震<ref>{{cite web |url = http://www.guardian.co.uk/world/2010/apr/19/women-blame-earthquakes-iran-cleric |title = Women to blame for earthquakes, says Iran cleric |publisher = guardian.co.uk |date = 2010-04-19 |accessdate=2011-07-12 |language = en}}</ref>。

位於[[優素福阿巴德|優素福阿巴德]]地區以北的[[德黑蘭國際大廈|德黑蘭國際大廈]]（Tehran International Tower）是德黑蘭最高的住宅建築，其設計與[[美國|美國]][[內華達州|內華達州]][[克拉克縣_(內華達州)|克拉克縣]][[天堂市|帕拉代斯]][[拉斯維加斯大道|拉斯維加斯大道]]的[[曼德勒海灣酒店|曼德勒海灣酒店]]相似。德黑蘭國際大廈位於優素福阿巴德的北方、哈姆基公路的南方，科爾德斯頓公路的東方、謝赫巴哈公路的西方，這些公路可通往德黑蘭各個地區。

== 旅遊及觀光勝地 ==
[[File:Museh_Melli.jpg|right]]的[[阿契美尼德王朝|阿契美尼德王朝]]收藏品。]]

作為伊朗的首都，德黑蘭具有眾多文化景點。波斯皇帝的[[孔雀寶座|孔雀寶座]]可在[[古列斯坦皇宮|古列斯坦皇宮]]裡找到。[[伊朗國家博物館|伊朗國家博物館]]、[[巴列維皇宮|巴列維皇宮建築群]]（Sa'dabad Palace），[[伊朗玻璃制品及陶瓷博物館|伊朗玻璃制品及陶瓷博物館]]（Glassware and Ceramics Museum of Iran）、[[伊朗地毯博物館|伊朗地毯博物館]]（The Carpet Museum of Iran）、[[德黑蘭釉下彩繪博物館|德黑蘭釉下彩繪博物館]]（Tehran's Underglass painting Museum）、[[尼雅瓦朗皇宮建築群|尼雅瓦朗皇宮建築群]]（Niavaran Palace Complex）及[[薩菲爾辦公室設備博物館|薩菲爾辦公室設備博物館]]（Safir Office Machines Museum）是德黑蘭較知名的博物館。[[德黑蘭現代藝術博物館|德黑蘭現代藝術博物館]]的收藏品包括[[梵高|梵高]]、[[巴勃羅·畢卡索|巴勃羅·畢卡索]]及[[安迪·華荷|安迪·華荷]]的作品，這些作品原本是皇后法拉赫·迪巴的收藏品<ref>{{Harvnb|Robertson|2008|p=238}}</ref>。

德黑蘭還藏有[[伊朗御寶|伊朗御寶]]（又稱波斯御寶），據稱伊朗御寶是世界上最大、最耀眼炫目、最有價值的寶石收藏品，當中包括王冠、權杖、古波斯頭飾、羽飾、鑲上寶石的劍及盾、寶玉、翡翠、紅寶石及鑽石，還有伊朗沙阿在2500年來收藏的各種物品。伊朗御寶在德黑蘭的[[伊朗伊斯蘭共和國中央銀行|中央銀行]]供公眾參觀<ref>{{Harvnb|Sawhney|2008|p=406}}</ref>。

[[德黑蘭國際書展|德黑蘭國際書展]]是國際出版界公認為亞洲及中東最重要的出版界事件<ref>{{cite web |url = http://www.tibf.ir/ |title = TIBF |publisher = TIBF |accessdate=2011-07-12 |language = en}}</ref>。

=== 娛樂休閒 ===


年輕一代最流行的社交活動都圍繞著[[電影院|電影院]]，許多電影院都位於市中心，德黑蘭最大的電影院[[阿扎迪電影院|阿扎迪電影院]]在2008年落成{{NoteTag|阿扎迪電影院的前身也是電影院，它在1997年被一場大火摧毀<ref>{{cite web |url = http://www.tehrantimes.com/index_View.asp?code=169589 |title = ISFA ready to screen short films in Tehran’s Azadi Cinema |publisher = Tehran Times |date = 2008-05-26 |accessdate=2011-07-13 |language = en}}</ref>。}}。對家庭來說，伊爾姆動物園和伊爾姆城市遊戲城是流行的聚會地點。

藝人經常在藝人之家會面。赫爾劇院在1962年開幕，是德黑蘭及中東最大的劇院。德黑蘭電視一台、德黑蘭劇院電視、奧米德電視及德黑蘭演藝電視是德黑蘭最受歡迎的電視台。

=== 宗教場所 ===

許多新舊的宗教場所散落在德黑蘭各處。各種宗教的信徒可到清真寺、教堂及猶太教堂進行宗教活動。

德黑蘭的週五禮拜通常在[[德黑蘭大學|德黑蘭大學]]舉行<ref>{{Harvnb|Aghaie|2004|p=133}}</ref>，由一位週五領拜者帶領，[[伊朗最高領袖|伊朗最高領袖]]有時也會出席週五禮拜。數十萬民眾會參與週五禮拜，期間全城都會陷入靜止狀態。

== 大事 ==
* 第7屆[[亞洲運動會|亞洲運動會]]在1974年9月1日至1974年9月16日於伊朗德黑蘭舉行。運動會在[[阿扎迪綜合體育館|阿扎迪綜合體育館]]（Azadi Sport Complex）進行，這是亞洲運動會第一次中東舉行，25個國家3,010位運動員參與了該屆亞洲運動會，這是自亞洲運動會舉辦以來參與人數最多的一屆<ref>{{cite web |url = http://www.ocasia.org/Game/GameParticular.aspx?GPCode=11 |title = Tehran 1974 |publisher = Olympic Council of Asia |accessdate = 2011-07-13 |language = en |archive-url = https://web.archive.org/web/20110522121505/http://www.ocasia.org/Game/GameParticular.aspx?GPCode=11 |archive-date = 2011-05-22 |dead-url = yes }}</ref>。
* [[1976年亞洲盃足球賽|1976年亞洲盃足球賽]]是第6屆舉辦的亞洲盃足球賽，賽事在伊朗德黑蘭舉行，6隊參賽球隊分為2組進行比賽，伊朗在該屆賽事奪得連續第3屆冠軍，伊朗國家足球隊在決賽裡以1-0擊敗[[科威特|科威特]]<ref>{{cite web |url = http://www.the-afc.com/en/afc-asian-cup-news/afc-asian-cup-features/33007-asian-cup-launch-pad-for-iran-superstars |title = Asian Cup launch pad for Iran superstars |publisher = the-afc.com |date = 2011-01-28 |accessdate=2011-07-13 |language = en}}</ref>。
* 第1屆[[西亞運動會|西亞運動會]]在1997年11月19日至11月28日於德黑蘭舉行，賽事的成功促使了西亞運動會聯盟的成立<ref>{{cite web |url = http://www.rsssf.com/tablesw/westasiagames.html |title = West Asian Games |publisher = West Asian Games |accessdate=2011-07-13 |language = en}}</ref>。
