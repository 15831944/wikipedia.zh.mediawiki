'''德宏傣族景颇族自治州'''（[[傣那语|傣那语]]：ᥖᥬᥳᥑᥨᥒᥰ {{IPA|[taɯ˥˧xoŋ˥]}}，[[傣仂语|傣仂语]]：ᦺᦎᧉᦆᦳᧂ {{IPA|[tai˩˧xuŋ˥˩]}}，[[载瓦语|载瓦语]]：{{Lang|atb|Sikung Sam Zaizo Byumyu Yumsing Upkang Mau}}，[[景颇语|景颇语]]：{{Lang|kac|Sakhkung Sam Jinghpo Amyu Madu Uphkang Mungdo}}），简称'''德宏州'''，是[[中华人民共和国|中华人民共和国]][[云南省|云南省]]下辖的[[自治州|自治州]]，位于云南省西部。除东面与[[保山市|保山市]]相邻外，其余三面皆与[[缅甸|缅甸]]接壤。地处[[横断山脉|横断山脉]]南段边缘，[[高黎贡山|高黎贡山]]西部延伸的切割山原谷地，地势东北高而陡峻，西南低而宽缓。[[怒江|怒江]]、[[大盈江|大盈江]]、[[瑞丽江|瑞丽江]]流经境内。全州总面积1.12万平方公里，人口127.9万，[[汉族|汉族]]人口比例约52%，[[傣族|傣族]]和[[景颇族|景颇族]]人口比例分別约29%和11%。自治州首府驻[[芒市|芒市]]。

== 名称由来 ==
「德宏」是[[傣语|傣语]]译音，“德”为下面，“宏”为[[怒江|怒江]]，意思是“怒江下游的地方”，因其自然环境优美，历史文化灿烂，民族风情独特，有「孔雀之乡」、「神话之乡」、「歌舞之乡」等美誉。<ref name="历史沿革">{{Cite web|url=http://www.xzqh.org/html/2008/0424/18901.html|title=德宏州历史沿革|publisher=行政区划网|date=2010-11-12|accessdate=2011-08-01}}</ref>

== 历史 ==
约公元前四世纪，中国历史上最早的一条国际陆路交通线“西南丝路”开通，中印贸易便已开始，德宏即为西南丝路的必经之地。

今州境古为[[西南夷|西南夷]][[哀牢国|哀牢国]]地。[[西汉|西汉]][[元封|元封]]二年（前109年），[[汉武帝|汉武帝]]开西南夷，出兵攻打哀牢，哀牢辖地缩減，德宏仍为哀牢地。[[东汉|东汉]][[永平_(东汉)|永平]]十二年（69年）哀牢王内附，汉朝遂于哀牢地置[[永昌郡|永昌郡]]，并置[[哀牢县|哀牢县]]于今[[盈江县|盈江县]]境。[[建安|建安]]十九年（214年），[[刘备|刘备]]平蜀，于南中诸郡设[[庲降都督|庲降都督]]。[[西晋|西晋]]分[[益州|益州]]置[[宁州|宁州]]，永昌郡属之。[[东晋|东晋]][[咸康|咸康]]八年（342年）省宁州，罢永昌郡。[[南朝齐|南朝齐]]复置永昌郡，[[隋朝|隋朝]]再罢。

[[唐朝|唐朝]]年间，[[南诏|南诏]]建国，今州境分属[[永昌节度|永昌节度]]和[[丽水节度|丽水节度]]，于今[[芒市|芒市]]置茫施城，又于今[[盈江县|盈江县]]东北置押西城，皆属永昌节度。[[大理国|大理国]]时，州境分属[[永昌府|永昌府]]和[[腾冲府|腾冲府]]，押西城改称干额。

[[元朝|元]]初置[[南甸路|南甸路]]（治今[[梁河县|梁河县]][[曩宋阿昌族乡|曩宋阿昌族乡]]）。[[至元|至元]]十三年（1276年）置[[茫施路|茫施路]]（治今[[芒市|芒市]]）、[[镇西路|镇西路]]（治今[[盈江县|盈江县]][[旧城镇_(盈江县)|旧城镇]]）、[[平缅路|平缅路]]（治今[[陇川县|陇川县]]）、[[麓川路|麓川路]]（治今[[瑞丽市|瑞丽市]]），皆属[[金齿等处宣抚司|金齿等处宣抚司]]，后改[[大理金齿宣慰司|大理金齿宣慰司]]。至元二十六年（1289年）改南甸路为[[南甸军民府|南甸军民府]]。

[[明朝|明]][[洪武|洪武]]十五年（1382年）改南甸军民府为[[南甸府|南甸府]]；改茫施路为[[芒施府|芒施府]]。洪武十七年（1384年）废麓川路、平缅路，改置[[麓川平缅宣慰司|麓川平缅宣慰司]]（治今瑞丽市）。[[永乐|永乐]]元年（1403年）析麓川平缅司置[[干崖长官司|干崖长官司]]（治今[[盈江县|盈江县]][[新城乡_(盈江县)|新城乡]]）。永乐十二年（1414年）改南甸府为[[南甸州|南甸州]]。[[正统|正统]]六年（1441年）废麓川平缅宣慰司。正统八年（1443年）升南甸州为[[南甸宣抚司|南甸宣抚司]]。正统九年（1444年）改芒施府置[[芒市长官司|芒市长官司]]，属金齿军民指挥司；升干崖长官司为[[干崖宣抚司|干崖宣抚司]]；于麓川平缅军民宣慰司故地置[[陇川宣慰司|陇川宣慰司]]（治今陇川县）。正统十一年（1446年）改陇川宣慰司为[[陇川宣抚司|陇川宣抚司]]（治今陇川县[[陇把镇|陇把镇]]）。正统十四年（1449年）析干崖宣抚司置[[盏达副宣抚司|盏达副宣抚司]]（治今盈江县[[莲花山乡|莲花山乡]]）。[[万历|万历]]十二年（1584年）置宣抚同知安于麓川。曾是[[孟卯|孟卯]]王国的中心（[[曼尼普尔|曼尼普尔]]人口中的[[蓬王国|蓬王国]]），[[阿薩姆邦|阿薩姆邦]]的[[阿豪姆人|阿豪姆人]]也來自這里。

[[清朝|清]][[顺治|顺治]]十六年（1659年）析陇川宣抚司置[[勐卯安抚司|勐卯安抚司]]（治今瑞丽市），属[[腾越厅|腾越厅]]。清代又分芒市长官司置[[芒市安抚司|芒市安抚司]]、[[遮放副宣抚司|遮放副宣抚司]]（治今芒市[[遮放镇|遮放镇]]）、勐板土千总（治今芒市南），属永昌府；分陇川宣抚司置[[户撒长官司|户撒长官司]]（今陇川县[[户撒阿昌族乡|户撒阿昌族乡]]）、[[腊撒长官司|腊撒长官司]]（今陇川县北腊撒），属腾越厅。

[[民国|民国]]元年（1912年）合芒市安抚司和勐板土千总为[[芒板弹压委员|芒板弹压委员]]；改干崖宣抚司为[[干崖弹压委员|干崖弹压委员]]；改盏达副宣抚司为[[盏达弹压委员|盏达弹压委员]]；改陇川宣抚司为[[陇川弹压委员|陇川弹压委员]]；合勐卯安抚司、遮放副宣抚司设[[勐卯弹压委员|勐卯弹压委员]]；合芒市安抚司、勐板土千总设[[芒板弹压委员|芒板弹压委员]]；将南甸宣抚司并入[[腾冲县|腾冲县]]。民国四年（1915年）遮放划入芒板弹压委员改[[芒遮板行政区|芒遮板行政区]]。1916年改勐卯、陇川、盏达弹压委员为[[勐卯行政区|勐卯行政区]]、[[陇川行政区|陇川行政区]]、[[盏达行政区|盏达行政区]]。1917年合勐卯行政区与腊撒土司置[[勐撒行政区|勐撒行政区]]，夏秋驻腊撒，冬春驻勐卯；合干崖和户撒置[[干户行政区|干户行政区]]。1929年置第一殖边督办公署，各行政区属之。1932年设[[畹町镇|畹町镇]]，隶潞西行政区。民国二十四年（1935年）改勐撒行政区置[[瑞丽设治局|瑞丽设治局]]；改芒遮板行政区为[[潞西设治局|潞西设治局]]；改陇川行政区置[[陇川设治局|陇川设治局]]；改[[腾冲县|腾冲县]]原南甸宣抚司置[[梁河设治局|梁河设治局]]（治今[[梁河县|梁河县]][[大厂乡|大厂乡]]）；以原干崖弹压委员地置[[盈江设治局|盈江设治局]]；以原盏达弹压委员地置[[莲山设治局|莲山设治局]]，皆属第一殖边督办区。1938年撤销第一殖边督办公署。民国三十一年（1942年）各设治局属云南省第六行政督察区。民国三十五年（1946年）改属第十二区。

[[中华人民共和国|中华人民共和国]]建立后，1949年各设治局改为设治区，属[[保山专区|保山专区]]；梁河设治区迁治[[遮岛镇|遮岛镇]]。1952年改潞西设治区置[[潞西县|潞西县]]；改瑞丽设治区置[[瑞丽县|瑞丽县]]；改莲山设治区置[[盈江县|盈江县]]（治今[[旧城镇_(盈江县)|旧城镇]]）；改陇川设治区为[[陇川县|陇川县]]（治今[[城子镇|城子镇]]）；改梁河设治区置[[梁河县傣族景颇族自治区|梁河县傣族景颇族自治区]]；同年12月，畹町镇升为县级镇。1953年7月24日置[[德宏傣族景颇族自治区|德宏傣族景颇族自治区]]，驻潞西县（同时梁河县自治区改置[[梁河县|梁河县]]），辖潞西、盈江、莲山、陇川、瑞丽、梁河六县和畹町镇，1956年改为[[德宏傣族景颇族自治州|德宏傣族景颇族自治州]]，将保山专区并入德宏州。1958年撤销莲山县并入盈江县，县治迁今址[[平原镇_(盈江县)|平原镇]]。1963年恢复保山专区。1969年将德宏州并入保山地区，1973年又析出复置。1985年1月，撤销畹町镇，改设[[畹町市|畹町市]]。1992年6月，撤销瑞丽县，改置[[瑞丽市|瑞丽市]]。1996年12月，撤销潞西县，改置[[潞西市|潞西市]]。1998年4月，陇川县驻地迁至[[章凤镇|章凤镇]]。1999年1月，撤销[[畹町市|畹町市]]，并入瑞丽市，设立畹町经济开发区。2010年7月，潞西市更名为[[芒市|芒市]]。

== 地理 ==
德宏的地理位置介于东经97°31′至98°43′、北纬23°50′至25°20′之间，总面积11526平方公里。南、西和西北三面与[[缅甸联邦|缅甸联邦]][[克钦邦|克钦邦]]接壤，[[边界|国境线]]长达503.8公里，东和东北2面与[[保山市|保山市]]的[[龙陵县|龙陵县]]、[[腾冲县|腾冲县]]两县相邻。德宏是古代"[[南方丝绸之路|南方丝绸之路]]"的出口，境内有九条公路与[[缅甸|缅甸]]北部城镇相通，有瑞丽、畹町两个国家级口岸；盈江、章凤两个省级口岸。

===气候===
德宏傣族景颇族自治州热量丰富，气候温和，属南亚热带气候。年平均气温18.3～20℃，1月平均气温10.9～12.5℃，6月平均气温22.8～24.3℃，年较温差11.8～12.8℃，因而“冬无严寒，夏无酷暑”。

德宏属多雨区，年降雨量1376～1649毫米，5～10月降雨量占全年降雨量的88～90%，冬、春旱较突出。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全州[[常住人口|常住人口]]为1211440人<ref>{{cite book |author=德宏傣族景颇族自治州统计局|title=《德宏傣族景颇族自治州2010年第六次全国人口普查主要数据公报》}}</ref>。同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加了140153人，增长13.08%。平均每年增加14015人，年平均增长率为1.24%。其中，男性为624774人，占总人口的51.57%；女性为586666人，占总人口的48.43%。总人口性别比（以女性为100）为106.50。0－14岁的人口为252174人，占总人口的20.82%；15－64岁的人口为883264人，占总人口的72.91%；65岁及以上人口为76002人，占总人口的6.27%。居住在城镇的人口为414070人，占34.18%；居住在乡村的人口为797370人，占65.82%。

=== 民族 ===
常住人口中，[[汉族|汉族]]人口为629147人，占总人口的51.93%；各[[少数民族|少数民族]]人口为582293人，占总人口的48.07%。

德宏州和附近县市的傣族多数属于[[傣那|傣那]]，说[[傣那语|傣那语]]。

== 经济 ==
2005年，全州生产总值完成58.84亿元，人均生产总值5533元，其中第一产业完成19.03亿元，第二产业完成13.7亿元，第三产业完成26.11亿元，一、二、三产业比重为32：23：45；全社会固定资产投资34.13亿元，建立了除粮、糖、茶传统产业以外的外贸加工业、旅游产业、水电及电冶产业、生物资源开发产业、建筑建材业、畜牧产业六大产业；地方财政一般预算收入完成4.06亿元；外贸进出口总额完成3.9亿美元，其中进口1.19亿美元，出口2.71亿美元；城镇居民人均可支配收入8395元，农民人均纯收入1504元。全州发电量突破10亿千瓦小时，水泥突破60万吨，猪牛羊产量突破4万吨。

2008年实现[[生产总值|生产总值]]99.7亿元，比上年增加11.5%。城镇居民[[人均可支配收入|人均可支配收入]]11450元，增长9.1%。[[农民人均纯收入|农民人均纯收入]]2439元，增长19.2%，城镇居民人均消费支出8787元，增长8.6%。农村居民人均消费支出2208元，增长24.2%。<ref name="德宏概况">{{Cite web|url=http://www.dh.gov.cn/dhzrmzfgzxxw/3963735020085968896/20091019/243021.html|title=德宏傣族景颇族自治州经济社会发展概况（2008年）|publisher=德宏州人民政府门户网站|date=2009-10-19|accessdate=2011-08-01}}</ref>

==名人==
*[[卞章戛|卞章戛]]，男，勐卯镇弄冒寨人，云南省德宏傣族史学家。他曾入奘寺为僧，并担任“召帕雅坦玛铁”（大佛爷）。在1886年3月25日，完成了史学价值较高的《嘿勐沽勐》一书为，研究德宏勐卯及德宏附近地方民族史，提供了极为珍贵的资料《嘿勐沽勐》（勐卯古代诸王史）的成书。
*[[李根源|李根源]]（1879（己卯年）-1965），字印泉，又字养溪、雪生，号曲石，别署[[高黎贡山|高黎贡山]]人，祖籍山东益都（今山东青州），生于云南腾越（今云南[[腾冲|腾冲]]）。近代名士、国民党元老。
*[[龚全国|龚全国]]，男，1950年4月出生，云南德宏州梁河县人，葫芦丝演奏家。
*[[万小散|万小散]]（1963年--2013年），女，国家一级演员，原云南省德宏州傣剧团演员、副团长，云南省青年表演艺术家，云南省山茶花奖获得者，云南省有突出贡献优秀奖，云南省戏剧家协会理事，德宏州政协委员。曾代表中国和云南少数民族艺术团出访过美国、日本、新加坡、缅甸等国家和地区演出。
*[[毛相|毛相]]（1918（戊午年）～1986年），男，云南省瑞丽市姐相乡贺赛村人，原云南省德宏州民族歌舞团著名民间舞蹈家。
*[[邵梅罕|邵梅罕]]，女，傣族，1968年出生，云南省德宏傣族景颇族自治州芒市风平镇弄么村人，云南省剪纸民间艺人。邵梅罕继承和发扬了傣族民间剪纸艺术的传统，同时融入了中原地区剪纸艺术的长处，形成了独特的风格，其作品曾多次获得国内外比赛的金奖，被当地群众称作“傣乡剪纸王”。
