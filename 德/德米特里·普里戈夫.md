'''德米特里·亞歷山德羅維奇·普里戈夫'''（{{lang-ru|Дми́трий Алекса́ндрович При́гов}}；{{bd|1940年|11月5日|2007年|7月16日}}<ref name="latimes">{{cite web |url = http://articles.latimes.com/2007/jul/17/local/me-passings17.2 |title = Dmitri Prigov, 66; artist and influential poet of the post-Soviet era |trans-title = 普里戈夫：後蘇聯時代最具影響力的藝術家與詩人，享年六十六歲 |publisher= Los Angeles Times | date = 16 July 2007 |language=en |accessdate=2018-07-16 }}</ref>），[[俄羅斯|俄羅斯]]先鋒詩人與[[觀念藝術|觀念藝術家]]<ref>{{cite web |title = 冬宫博物馆着力推当代艺术 |url = http://art.china.cn/haiwai/2012-11/09/content_5480832.htm |publisher = 艺术中国 |date = 2012-11-09 |accessdate = 2018-07-17}}</ref>。他是[[蘇聯|蘇聯]]時代最具名的[[持不同政見者|異議人士]]之一，曾在1986年被送入[[精神病院|精神病院]]<ref name=20prigov>{{cite web |last = Kishkovsky |first = Sophia |url = https://www.nytimes.com/2007/07/20/books/20prigov.html?ex=1342584000&en=2abefdd4279f2682&ei=5088&partner=rssnyt&emc=rss |publisher = New York Times |title = Dmitri Prigov, 66, Poet Who Challenged Soviet Authority, Dies |trans-title = 曾挑戰蘇聯威權的俄羅斯詩人普里戈夫逝世，享年六十六歲 |date= 20 July 2007 |language=en |accessdate=2018-07-16 }}</ref>。

== 早年 ==
普里戈夫出生於[[蘇俄|蘇俄]][[莫斯科|莫斯科]]，在青少年時期便開始寫詩。他後來於{{link-en|斯特羅加諾夫莫斯科國立藝術與工藝大學|Stroganov Moscow State University of Arts and Industry}}鑽研雕塑，畢業後成為建築師，也曾為市立公園設計雕塑藝術品。<ref name=20prigov/>

== 藝術家生涯 ==

普里戈夫和他的友人列夫·魯賓斯坦（Lev Rubinstein）是俄國[[觀念藝術|觀念藝術]]的先驅<ref name="penguin" />。普里戈夫著名的觀念藝術作品包括在錫罐上寫詩句。<ref name=20prigov/>

普里戈夫是一位多產的詩人，截至2005年，已作有近三萬六千首詩作<ref name=20prigov/><ref name="penguin" />。在蘇俄時代，他的詩作屬於[[秘密出版物|秘密出版物]]（{{lang-ru|самиздат}}），僅在地下詩界流傳，直到共產時代結束後才獲得正式發行；不過在此之前，普里戈夫的作品便已經廣在[[流亡分子|流亡文學界]]及{{link-en|斯拉夫研究|Slavic studies}}刊物上流傳。

1986年，普里戈夫因為在街頭發送詩作給路人的行徑，而被[[國家安全委員會_(蘇聯)|國家安全委員會]]（K.G.B）逮捕<ref name="penguin">{{cite book |title = The Penguin Book of Russian Poetry |last1 = Chandler |first1 = Robert |first2 = Irina |last2 = Mashinski |first3 = Boris |last3 = Dralyuk |publisher = Penguin |location = UK |isbn = 0141972262 |chapter = 6: Three More Recent Poets |language = en}}</ref>。他被送到[[精神病院|精神病院]]，直到在[[貝拉·阿赫瑪杜琳娜|貝拉·阿赫瑪杜琳娜]]等詩人的嚴正抗議之下，才獲釋。<ref name=20prigov/><ref name="penguin" />

1987年起普里戈夫開始公開出版與展覽，1991年他加入作家聯盟。此外，他自1975年起便是藝術家聯盟的成員之一。

1993年，普里戈夫獲得{{link-en|阿弗雷德·杜佛基金會|Alfred Toepfer Stiftung F.V.S.}}的{{link-en|普希金獎|Pushkin Prize}}。2002年，再獲頒[[鮑里斯·巴斯特納克|鮑里斯·巴斯特納克獎]]。

2007年，普里戈夫因心臟病在莫斯科逝世，享年六十六歲。<ref>{{cite web |url = http://www.gif.ru/eng/news/dmitri-prigov/ |title = Russian poet Dmitri Prigov, leader of conceptualist school, dies at age 66 |trans-title = 俄國詩人、概念藝術先鋒普里戈夫六十六歲逝世 |website = GiF.Ru |publisher = Art of Russia |location = Mosco |date = 16.07.2007 |accessdate = 2018-07-18 |language = en}}</ref><ref name="penguin" />

2011年，在第五十四屆威尼斯雙年展期間，[[艾米塔吉博物館|艾米塔吉博物館]]舉辦了一場重要的普里戈夫藝術專題展。
