'''德姆其格·莫洛姆扎木茨'''（Demshigiyn Molomjamts，{{bd|1920年|||}}）[[蒙古族|蒙古族]]，[[中戈壁省|中戈壁省]][[巴彦扎尔格朗县|巴彦扎尔格朗县]]人，[[蒙古人民革命党|蒙古人民革命党]]中央政治局委员、党中央书记<ref name=sh>当代国际人物词典，上海辞书出版社，1980年</ref>。

==生平==
1920年，莫洛姆扎木茨生于中戈壁省巴彦扎尔格朗县。1938年至1940年，在[[乌兰巴托|乌兰巴托]]财经学校学习。毕业后，到[[苏联|苏联]][[伊尔库茨克|伊尔库茨克]]财经学院学习。1946年归国后，出任财政部副部长。1947年加入[[蒙古人民革命党|蒙古人民革命党]]。1954年[[蒙古人民革命党|蒙古人民革命党]]十二大起，当选为历届中央委员。1948年，任财政部部长。1957年，任部长会议副主席。同年，在[[蒙古人民革命党|蒙古人民革命党]]十二届六中全会上，当选为中央政治局侯补委员。 1959年[[蒙古人民革命党|蒙古人民革命党]]十三届四中全会起，当选为历届中央政治局委员。 1964年[[蒙古人民革命党|蒙古人民革命党]]十四届六中全会起，当选为历届中央书记。1959年至1965年，任部长会议第一副主席。1959年4月，兼任国家计划委员会主席。1963年至1965年，任部长会议经济合作委员会主席、驻[[经互会|经互会]]常任代表。1965年5月起，专事党务工作<ref name=sh/>。

1958年，蒙古人民共和国部长会议副主席莫洛姆扎木茨率蒙古政府经济代表团访问[[中华人民共和国|中华人民共和国]]<ref name=sh/>。1958年12月至1959年1月，先后访问[[北京市|北京市]]、[[上海市|上海市]]、[[杭州市|杭州市]]等地<ref>{{Cite web |url = http://www.shtong.gov.cn/newsite/node2/node2245/node69969/node69987/node70084/node70219/userobject1ai70167.html |title = 上海外事志 下编·第五篇国际来往 第一章与苏联和社会主义各国的来往 1952～1993年访沪的社会主义国家元首和政府、议会、政党领导人名录  |author =  |date =  |publisher = 上海通 |language =  |accessdate = 2017-06-28 |quote =  }}</ref>。1960年，《中华人民共和国和蒙古人民共和国通商条约》签订，[[中华人民共和国主席|中华人民共和国主席]]特派[[中华人民共和国对外贸易部|中华人民共和国对外贸易部]]部长[[叶季壮|叶季壮]]、蒙古人民共和国大人民呼拉尔主席团特派蒙古人民共和国部长会议第一副主席德·莫洛姆扎木茨议定并签字<ref>{{Cite web |url = http://www.npc.gov.cn/wxzl/gongbao/2000-12/25/content_5000799.htm |title = 中华人民共和国和蒙古人民共和国通商条约 |author =  |date =2000-12-25  |publisher =  中国人大网|language =  |accessdate =  |quote =  }}</ref>。

1984年8月23日，[[蒙古人民革命党|蒙古人民革命党]]中央委员会举行第八次非常会议，决定解除68岁的[[尤睦佳·泽登巴尔|泽登巴尔]]的蒙古人民革命党中央委员会总书记和政治局委员职务，并且选举58岁的政治局委员、部长会议主席[[姜巴·巴特蒙赫|巴特蒙赫]]为新任总书记。同日，蒙古[[国家大呼拉尔|大人民呼拉尔]]举行非常会议，决定解除泽登巴尔的大人民呼拉尔主席团主席（即国家元首）职务。蒙古人民革命党中央政治局委员莫洛姆扎木茨在蒙古人民革命党中央全会上发表的讲话称：“泽登巴尔同志的健康状况使他不可能继续担任党中央总书记的工作。”向中央全会提出解除其职务的建议是“征得他本人同意的。”<ref>解之，蒙古最高级人事更迭，国际展望1984(19):8-10</ref>
