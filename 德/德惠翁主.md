'''德惠翁主'''{{BD|1912年|5月25日|1989年|4月21日}}，[[李氏|李氏]]，本名不詳，為[[日治朝鲜|日治朝鲜]][[李王家|李王家]][[翁主|翁主]]，[[大韓帝國|大韓帝國]]開國[[皇帝|皇帝]][[朝鲜高宗|李熙]]的庶三女，母[[貴人梁氏|貴人梁氏]]。

== 幼年 ==

1911年，受高宗寵愛的[[純獻皇貴妃|純獻皇貴妃]]去世後，尚宮梁氏得到了寵幸，並在次年5月生下一女，這是高宗最後一個女兒。由於是[[庶出|庶出]]身分（母為宮人），因而封為翁主，號'''德惠'''。其母也連帶的被擢昇為[[貴人|貴人]]，並賜堂號福寧堂（當時日韓已經合併，但宮中仍維持舊制）。

由於老年得女，又是么女，德惠翁主頗受高宗疼愛。也正因為如此，在稍成長後，高宗擔心德惠翁主會被日本政府所帶走，便暗地裡為德惠翁主安排婚事，期望能以婚姻的方式將其留住。不久，[[一次大戰|一次大戰]]結束，戰勝國於[[巴黎|巴黎]]舉行[[巴黎和會|和會]]，高宗企圖派遣密使以求列強協助獨立，但此事不久被日本發現。而高宗不久突然駕崩，年僅7歲的德惠翁主失去了父親，也等於失去了保護者。

== 前往日本 ==
1925年1月，[[朝鮮純宗|朝鮮純宗]]被通知，時就讀小學五年級的德惠翁主即將被送往日本“深造”。對於異母妹即將遠行消息，純宗和其妻[[純貞孝皇后|純貞孝皇后]]都感到無能為力。其生母梁貴人更是萬般不捨，但卻完全阻止不了。

同年3月25日，13歲的德惠翁主啟程前往日本，5天後平安到達，並由身在日本兄長[[李垠|李垠]]的妻子－[[李方子|李方子]]前往迎接。據[[李方子|李方子]]女士在多年後的回憶，當時她所見到的德惠翁主：「臉上毫無表情、對周遭的世界默不作聲，看起來相當的沉默和孤獨。」

身在日本的德惠翁主，被安排進入[[學習院|學習院]]就讀，而[[李垠|李垠]]夫婦本想讓德惠翁主一起同住，但遭到日本當局回絕。此時的德惠翁主，相當沉默和封閉，整日不發一語。

1929年5月30日，德惠翁主的生母－福寧堂貴人梁氏病逝的消息傳到了日本。這消息對德惠翁主而言是相當大的打擊，從此她將自己關在房內不再踏出一步。數日後，日本當局決定讓她回國奔喪。15天後德惠翁主又被匆匆送回日本。她回到日本的時候，仍由她在日本唯一的親人李垠夫婦迎接，而李垠夫婦再見到她時，德惠翁主臉上毫無表情，眼中也沒有淚水，只有無盡的冷漠和茫然。

很快的，德惠翁主被安排就診，並被醫生診斷出，德惠翁主患了一種神經性疾病，至此，德惠翁主已經無法繼續她在學習院未完的學業，於是日本方面安排她前往和李垠夫婦同住。這段期間的德惠翁主，不吃也不喝，對於外界的事物毫無感覺，夜裡有時會如[[夢遊症|夢遊症]]患者般走來走去。之後她才被確定患了「早發性[[癡呆症|癡呆症]]」，日本方面認為是先天疾病、韓國則認為是後天疾病。

== 婚姻生活 ==
次年秋天，日本政府決定將德惠翁主嫁給[[對馬島|對馬島]]藩主之子－[[伯爵|伯爵]][[宗武志|宗武志]]，這件事情遭到了[[李垠|李垠]]的強烈反對和抗議，但由於德惠翁主的病情相當不穩定，因此並未立即出嫁，而是在1931年，19歲的她病情稍微好轉之後，於同年5月8日她被告知了婚期已定下的消息，而德惠翁主對於此事的反應是：往後四天水米不沾、正日呆坐著流淚。但她仍然被安排強制出嫁了。
[[File:Princess_dukhye_and_takeyuki_so,_1931.JPG|right]]在對馬嚴原]]
雖然說是強逼的婚姻，但是她在新婚期與丈夫的關係不錯，病情也進入平穩期，1933年8月14日，德惠翁主同宗武志生下一女[[宗正惠|宗正惠]]。但在生下女兒後的數年內，她的病情越來越嚴重，幾乎無法將女兒撫養長大，即使丈夫及夫家相當照護她<ref>[http://www.epochtimes.com/b5/12/12/27/n3762615.htm%E6%82%B2%E6%88%9A%E4%B8%80%E7%94%9F%E7%9A%84%E6%9C%9D%E9%AE%AE%E6%9C%AB%E4%BB%A3%E5%85%AC%E4%B8%BB.html 悲戚一生的朝鮮末代公主]</ref>，德惠翁主的病情依然持續地惡化。而德惠翁主也開始以醫院為家。

德惠翁主即使進入醫院接受治療，病情卻幾乎沒有好轉，她精神恍惚茫然地渡過每一天。而在此期間，外界已發生了很大的變化：[[中國抗日戰爭|中國抗日戰爭]]、她的祖國分裂（[[韓戰|韓戰]]）。1953年（一說1955年），她的宗族[[李王家|李王家]]及宗武志的家族都要求兩人離婚，宗武志雖然反對，但這要求後來被實現了，德惠翁主又變的孤苦無依，不久，在1955年，德惠翁主一生唯一的子女'''宗正惠'''過世。

離婚之後的德惠翁主便一直住在當時頗著名的精神病專科醫院－松澤醫院，該醫院院長對德惠翁主的遭遇十分同情，因此給了德惠翁主相當細心的照顧，才使德惠翁主得以存活。雖然活了下來，但德惠翁主最後終於完全失去意識。

== 返回祖國 ==
1961年，德惠翁主的祖國發生政變，得權的[[朴正熙|朴正熙]]在同年11月出訪[[美國|美國]]。途中經過[[東京|東京]]時，朴正熙會見了[[李垠|李垠]]的夫人[[李方子|李方子]]。當李方子向他說起德惠翁主的悲慘遭遇時，朴正熙感到相當吃驚，因為他完全不曉得德惠翁主這號人物，隨後李方子夫人向朴正熙詳述了德惠翁主的身世和遭遇，朴正熙聞後不禁鼻酸，並保證會安排德惠翁主回國安養。

次年，在迅速的處理完德惠翁主的回國手續後，德惠翁主回到了闊別38年的祖國。回到[[漢城|漢城]]之後，德惠翁主見到了久未相見的[[純貞孝皇后|尹大妃]]－[[朝鮮純宗|純宗]]的遺孀。但尹大妃相當震驚，尹大妃一時無法接受，印象中活潑的德惠翁主，竟然變成了一個失去意識且言神呆滯茫然不知的老婦。

此時，德惠翁主幼年時的[[乳母|乳母]]－卞福童（변복동，1889～1972）仍在世，在見到德惠翁主後亦相當的激動，也相當地悲痛，因而在德惠翁主進入漢城大學附屬醫院後，卞氏在德惠翁主身邊全天候的看照著，這次，在乳母卞氏所給予的親情溫暖之下，德惠翁主恢復意識，病情日見好轉，之後還親自寫了兩封簡短的問候信，分別給尹大妃和在日本頗照顧她的[[李垠|李垠]]夫妻。

再之後，德惠翁主的病情穩定了下來，之後出院住進了往日的王宮[[昌德宮|昌德宮]]一角－樂善齋，在那裡還住著幾經動蕩的[[純貞孝皇后|純貞孝皇后]]。

1989年4月21日，德惠翁主逝世，享壽七十七歲。
