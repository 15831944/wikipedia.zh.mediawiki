'''德宏师范高等专科学校'''，简称'''德宏师专'''，是[[云南省人民政府|云南省人民政府]]主办、[[云南省教育厅|云南省教育厅]]主管的一所普通高等师范[[专科学校|专科学校]]，位于[[云南省|云南省]][[德宏傣族景颇族自治州|德宏傣族景颇族自治州]][[芒市|芒市]]，是德宏州两所高校之一。

==历史==

1955年，德宏自治区在[[德宏州民族第一中学|潞西民族中学]]内附设民族师范学校，称“潞中师范部”，为中等专业教育，1974年独立建校，正式命名'''德宏州民族师范学校'''<ref>{{cite book |author=云南省潞西县志编纂委员会 |title=《潞西县志》 |location=[[昆明市|昆明]] |publisher=[[云南教育出版社|云南教育出版社]] |date=1993年 |isbn=7-5415-0685-0 }}</ref>{{rp|338}}。1978年，云南省政府根据全省中学发展情况与师资缺乏情况，批准成立了德宏师范专科班，设在德宏州民族师范学校内，实行[[一个机构两块牌子|一套班子两块牌子]]办学。1984年，省政府批准成立'''德宏教育学院'''、德宏州广播电视大学，1985年教育学院从民族师范迁出。1987年，德宏州委将教育学院与广播电视大学合并，实行一套班子两块牌子办学。2004年，德宏州民族师范学校并入教育学院。2005年，教育学院从[[成人高等学校|成人高等学校]]改制为普通高等师范专科学校。2006年，撤销德宏教育学院建制，成立德宏师范高等专科学校，于当年6月28日揭牌。同年德宏农业学校并入师专<ref>{{cite book |author1=田大余 主编 |author2=《德宏改革开放30年》编委会 |title=《德宏改革开放30年 下》 |location=[[芒市|芒市]] |publisher=[[德宏民族出版社|德宏民族出版社]] |date=2008.11 |isbn=978-7-80750-133-6 }}</ref>{{rp|490-492}}。

==校园==

德宏师专共有2个校区，土地总面积1095.20亩，校舍总建筑面积25万平方米，学校图书馆藏书66.7万册<ref>{{cite web |url=http://www.dhecw.cn/xb/xxgk/xxjj.htm |title=德宏师范高等专科学校简介 |website=www.dhecw.cn |publisher=德宏师范高等专科学校 |access-date=2018-12-31 }}</ref>。2016年9月，在芒市城北拉怀村后山建立的新校区投入使用<ref>{{cite news |author=刘祥元 |title=德宏师专新校区投入使用 |url=http://api.yunnan.cn/dehong/html/2016-09/30/content_4558987.htm |newspaper=[[云南日报|云南日报]] |date=2016-09-30 |access-date=2018-12-31 }}</ref>。

2017年，德宏师范高等专科学校建立“[[江应樑|江应樑]]傣族博物馆”，展出[[傣族|傣族]]铜器、历史典籍以及江应樑傣学研究的成果和他搜集、使用过的物品、在德宏考察拍摄的照片，博物馆藏有的数千件傣族文物均是江应樑及其子江晓林多年搜集的藏品。该博物馆也是中国唯一一所真正意义上的傣族博物馆。<ref>{{cite web |url=http://yn.chinadaily.com.cn/a/201704/13/WS5bdbe120a3101a87ca90a02a.html |title=全国唯一一所真正意义上的傣族博物馆在云南德宏师专开馆 |date=2017-04-13 |website=yn.chinadaily.com.cn |publisher=中国日报网 |access-date=2018-12-19 |quote=}}</ref><ref>{{cite web |url=http://zqb.cyol.com/html/2017-04/17/nw.D110000zgqnb_20170417_4-06.htm |title=父子两代用80年建成一个傣族博物馆 |author=张文凌 |date=2017-04-17 |website=zqb.cyol.com |publisher=《[[中国青年报|中国青年报]]》 |access-date=2018-12-19 |quote=}}</ref>
