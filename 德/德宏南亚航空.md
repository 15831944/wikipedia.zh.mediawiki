'''德宏南亚通用航空有限公司'''（[[英语|英语]]：Dehong South Asian General Aviation Co.,Ltd<ref>{{cite web |url=https://www.icao.int/APAC/Meetings/2017%20RASMAG22/WP25%20China%20RMA%20LTHM%20Burden%20Estimate%20Update.pdf#search=Dehong%20South%20Asian%20General%20Aviation |title=CHINA RMA LTHM BURDEN ESTIMATE UPDATE
 |last= |first= |date=2017-07-13 |website=www.icao.int |publisher=[[国际民用航空组织|International Civil Aviation Organization]] |access-date=2017-08-28 |quote=}}</ref>）简称'''德宏南亚航空'''，是[[云南景成集团|云南景成集团]]独资成立的一家[[通用航空|通用航空公司]]，注册资本3亿[[人民币|元]]<ref name="经营许可证"></ref>。德宏南亚航空成立于2013年9月27日，其前身是'''德宏公务机有限公司'''和'''瑞丽景成直升机有限公司'''，后在2013年11月18日获得中国民用航空西南空中交通管理局颁发的通用航空企业经营许可证<ref name="经营许可证"></ref><ref name="刘祥元、项陆才"></ref>。在正式获得经营许可之前，德宏南亚航空的直升机和公务机分别由[[中国南方航空|中国南方航空]][[中国南方航空珠海直升机分公司|珠海直升机分公司]]和[[海南航空|海南航空]][[金鹿公务|金鹿公务机有限公司]]托管<ref name="2012年7月29日">{{cite web |url=http://www.jcjtgs.com/content-20-123-1.html |title=景成航空产业强势“起飞” |last= |first= |date=2013-09-23 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref>。

==历史==

2012年9月24日，德宏公务机有限公司和瑞丽景成直升机有限公司正式获得筹建批复<ref group="注">注册资本分别为2亿元和1亿元</ref><ref>{{cite web |url=http://dehong.yunnan.cn/html/2013-08/20/content_2852710.htm |title=景成集团下大单：订购18架飞机 |last= |first= |date=2013-08-20 |website=dehong.yunnan.cn |publisher=云南网 |access-date=2017-08-30 |quote= |archive-url=https://web.archive.org/web/20160910165316/http://dehong.yunnan.cn/html/2013-08/20/content_2852710.htm |archive-date=2016-09-10 |dead-url=yes }}</ref>。2013年9月27日，二公司合并更名德宏南亚航空<ref name="经营许可证"></ref><ref name="刘祥元、项陆才"></ref>。

2014年2月21日，德宏南亚航空通过《一般运行和飞行规则》（CCAR-91-R2部）文件审查<ref>{{cite web |url=http://www.jcjtgs.com/content-20-268-1.html |title=德宏南亚航空通过CCAR-91部文件审查 |last= |first= |date=2014-03-07 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref>，2014年3月26日通过其验证飞行<ref name="取三证"></ref>。

2014年3月26日，德宏南亚航空正式接收中国民用航空西南地区管理局颁发的维修许可证和维修项目许可证书，成为《民用航空器维修单位合格审定规定》（CCAR-145-R3部）维修单位<ref name="取三证">{{cite web |url=http://news.carnoc.com/list/282/282134.html |title=德宏州首家通航企业5月18日正式开航 |last= |first= |date=2014-05-17 |website=news.carnoc.com |publisher=民航资源网 |access-date=2017-08-30 |quote=}}</ref>。

2014年5月18日，德宏南亚航空与[[瑞丽航空|瑞丽航空]]一道完成首航<ref>{{cite web |url=http://www.jcjtgs.com/content-20-295-1.html |title=瑞丽航空、德宏南亚通用航空首航圆满成功 |last= |first= |date=2014-05-20 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref><ref name="王楠">{{cite web|url=http://yn.yunnan.cn/html/2014-05/18/content_3215506.htm|title=瑞丽航空、南亚航空同日首航 云南民营航空企业展翅高飞|last=王|first=楠|date=2014-05-18|website=yn.yunnan.cn|publisher=云南网|access-date=2017-08-28|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20171001031619/http://yn.yunnan.cn/html/2014-05/18/content_3215506.htm|archivedate=2017年10月1日|df=}}</ref>。

2014年9月24日，德宏南亚航空完成《小型航空器商业运输运营人运行合格审定规则》（CCAR-135部）文件审查<ref>{{cite web |url=http://www.jcjtgs.com/content-20-395-1.html |title=德宏南亚航空完成135部文审 |last= |first= |date=2014-10-07 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref>，2015年1月5日完成其验证试飞<ref>{{cite web |url=http://www.jcjtgs.com/content-20-447-1.html |title=德宏南亚通用航空完成CCAR-135验证试飞 |last= |first= |date=2015-02-11 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref>。2015年2月15日，德宏南亚航空获得CCAR-135部《航空运营人运行合格证》<ref>{{cite web |url=http://www.jcjtgs.com/content-20-454-1.html |title=德宏南亚航空获颁CCAR-135部运行合格证 |last= |first= |date=2015-03-10 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref>。

==运营基地==

[[File:德宏南亚通用航空有限公司01.jpg|thumb]]的办公楼]]

德宏南亚航空的运营基地设在[[德宏芒市机场|德宏芒市机场]]<ref name="机场"></ref>，机场内建有公务机和直升机的航材库、工具库房以及一幢综合办公楼<ref>{{cite web |url=http://www.jcjtgs.com/content-9-159-1.html |title=德宏南亚航空运营基地办公场所开工 |last= |first= |date=2013-12-02 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref>。德宏南亚航空芒市机场综合楼共4层，建筑面积约4,700平方米，一二层为德宏南亚航空办公区，三四层为综合管理用房，办公楼主体于2015年4月落成<ref>{{cite web |url=http://www.jcjtgs.com/content-20-563-1.html |title=南亚航空芒市机场综合楼 内外装修快速推进 |last= |first= |date=2015-10-08 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref>。2015年12月28日该运营基地启用<ref>{{cite web |url=http://www.jcjtgs.com/content-20-597-1.html |title=德宏南亚通用航空芒市机场主运营基地正式启用 |last= |first= |date=2016-01-05 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-29 |quote=}}</ref>。

德宏南亚航空在[[瑞丽市|瑞丽市]][[勐秀乡|勐秀乡]]建有[[瑞丽景成直升机场|瑞丽景成直升机场]]，是西南地区规模最大的通用航空机场<ref name="王楠"></ref>。

==经营范围==

德宏南亚航空获批在以下领域具备经营资格<ref name="经营许可证"></ref>：

*甲类：陆上石油服务、海上石油服务、直升机机外载荷飞行、医疗救护、空中游览、公务飞行、航空器代管业务、出租飞行、通用航空[[包机|包机]]飞行。
*乙类：航空摄影、空中广告、气象探测、科学实验、城市消防、空中巡查。
*丙类：航空护林、空中拍照。

===购机历史===

2012年7月29日，景成集团向美国[[波音公司|波音公司]]订购了一架[[波音公务机|波音BBJ]][[波音737新世代#737-700|737-700公务机]]<ref name="2012年7月29日"></ref>、向[[湾流航太|湾流航太]]订购一架[[湾流G550|湾流G550公务机]]<ref>{{cite web |url=http://www.jcjtgs.com/content-20-274-1.html |title=德宏南亚航空首架公务机飞抵芒市湾流G550 编号B-8296 |last= |first= |date=2014-04-14 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref>、向[[西科斯基飞机公司|西科斯基飞机公司]]订购一架{{le|西科斯基 S-92|Sikorsky S-92#S-92A|S-92A直升机}}<ref>{{cite web |url=http://www.jcjtgs.com/content-20-56-1.html |title=景成航空产业首架飞机起运 预计8月底抵达瑞丽 |last= |first= |date=2013-08-29 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref><ref>{{cite web |url=https://www.helis.com/database/modelorg/1286/ |title=Sikorsky s-92 Helibus in Sikorsky Helicopters |last= |first= |date= |website=www.helis.com |publisher=Helicopter History Site |access-date=2017-08-30 |quote=}}</ref>。2012年9月8日向西科斯基公司订购了一架[[赛考斯基S-76精灵直升机|西科斯基S-76D型直升机]]<ref>{{cite web |url=http://www.jcjtgs.com/content-20-125-1.html |title=景成集团采购第四架飞机 |last= |first= |date=2013-09-23 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref>。2013年5月30日，景成集团再次与西科斯基公司签订协议，订购一架S-92A型直升机<ref>{{cite web |url=http://www.jcjtgs.com/content-20-60-1.html |title=景成直升机公司再订购一架S-92 |last=吉 |first=斯瑞 |date=2013-08-29 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref>；2014年7月26日与西科斯基公司商谈后决定将其置换为S-76D型直升机，并计划于2015年6月交付<ref>{{cite web |url=http://www.jcjtgs.com/content-20-346-1.html |title=董勒成董事长赴珠海洽谈直升机置换事宜 |last= |first= |date=2014-08-06 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref><ref group="注">此后再无下文</ref>。

2013年8月22日，德宏南亚航空首架西科斯基S-92A抵达[[德宏芒市机场|德宏芒市机场]]，8月26日飞抵[[瑞丽景成直升机场|瑞丽景成直升机场]]<ref>{{cite web |url=http://www.jcjtgs.com/content-20-130-1.html |title=瑞丽景成直升机有限公司首架直升机S-92A抵达瑞丽 |last= |first= |date=2013-09-23 |website=www.jcjtgs.com |publisher=云南景成集团有限公司 |access-date=2017-08-28 |quote=}}</ref>。2014年3月29日，湾流G550公务机抵达[[德宏芒市机场|德宏芒市机场]]<ref name="刘祥元、项陆才">{{cite web |url=http://society.yunnan.cn/html/2014-03/29/content_3149958.htm |title=德宏南亚航空首架湾流公务机抵达芒市 |author=刘祥元、项陆才 |date=2014-03-29 |website=www.yunnan.cn |publisher=云南网 |access-date=2017-08-28 |quote= }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2014年4月24日，西科斯基S-76D交付德宏南亚航空<ref name="取三证"></ref>。2015年8月20日，波音BBJ737-700公务机在[[昆明长水国际机场|昆明长水国际机场]]交付德宏南亚航空<ref>{{cite web |url=http://news.carnoc.com/list/321/321868.html |title=云南省内首次引进波音BBJ公务机 |last=秦 |first=宁宁 |date=2015-08-20 |website=news.carnoc.com |publisher=民航资源网 |access-date=2017-08-30 |quote=}}</ref>，8月23日飞抵德宏芒市机场<ref>{{cite web |url=http://www.ynta.gov.cn/Item/22277.aspx |title=本土航空迎来新成员 国内一流公务机飞抵芒市 |last=韩 |first=海林 |date=2015-08-24 |website=www.ynta.gov.cn |publisher=云南旅游政务网 |access-date=2017-08-29 |quote=}}</ref>。
