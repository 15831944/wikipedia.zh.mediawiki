'''德米特里·尤里耶维奇·孔德拉季耶夫'''（{{lang-ru|Дми́трий Ю́рьевич Кондра́тьев}}，{{bd|1969年|5月25日|}}）是[[俄罗斯|俄罗斯]]已退役[[宇航员|宇航员]]。

==早年==
1986年毕业于[[阿拉木圖|阿拉木圖]]的一所中学，1990年毕业于[[卡恰|卡恰]]高等军事航空学校（ВВАУЛ）。曾在[[彼得罗扎沃茨克|彼得罗扎沃茨克]]附近的航空军团服役。<ref name=autogenerated1>{{Cite web |url=http://rk.karelia.ru/news/letchik-kosmonavt-dmitriy-kondratev-iz-karelii-stal-geroem-rossii/ |title=Летчик-космонавт Дмитрий Кондратьев из Карелии стал Героем России |accessdate=2012-05-05 |archiveurl=https://web.archive.org/web/20120516104431/http://rk.karelia.ru/news/letchik-kosmonavt-dmitriy-kondratev-iz-karelii-stal-geroem-rossii/ |archivedate=2012-05-16 |dead-url=yes|language=ru}}</ref>
==宇航员生涯==
1997年12月入选[[加加林宇航员培训中心|加加林宇航员培训中心]]宇航员候选人。1998年1月至2000年3月接受了基础太空训练。2000年获得测试宇航员资格。

2006年5月2日至2007年4月9日担任[[俄罗斯航太|-{zh-hant:俄羅斯航太; zh-hans:俄罗斯航天集团}-]]驻美国[[德克萨斯州|德克萨斯州]][[休斯敦|休斯敦]][[林顿·约翰逊太空中心|林顿·约翰逊太空中心]]主管。2007年3月获得[[美国联邦航空管理局|美国联邦航空管理局]]（FAA）私人飞行员执照；2009年3月获得FAA航空运输飞行员执照。<ref>{{cite web |title=Cosmonaut Bio: Dmitri Yurievich Kondratyev |url=https://web.archive.org/web/20160329160939/http://www.jsc.nasa.gov:80/Bios/htmlbios/kondratyev-dy.html |publisher=[[NASA|NASA]] |accessdate=2019-04-13 |language=en}}</ref>

===远征26/27===
[[莫斯科时间|莫斯科时间]]2010年12月15日22时09分（[[北京时间|北京时间]]16日凌晨3时09分），他作为指令长，和[[欧洲航天局|欧洲航天局]]意大利宇航员[[保罗·内斯波利|保罗·内斯波利]]和美国宇航员[[凯瑟琳·科尔曼|凯瑟琳·科尔曼]]乘座[[联盟TMA-20|联盟TMA-20]]载人飞船，从[[哈萨克斯坦|哈萨克斯坦]]境内的[[拜科努尔发射场|拜科努尔发射场]]升空，前往[[国际空间站|国际空间站]]执行任务。<ref>{{cite web |title=俄罗斯向国际空间站发射载人飞船 |url=http://www.beijingreview.com.cn/2009news/todaynews/2010-12/16/content_319371.htm |publisher=新华网 |archiveurl=https://web.archive.org/web/20190413031658/http://www.beijingreview.com.cn/2009news/todaynews/2010-12/16/content_319371.htm |archivedate=2019-04-13 |date=2010-12-16}}</ref>

莫斯科时间2011年1月21日17时29分（北京时间22时29分），他和[[奥列格·斯克里波奇卡|奥列格·斯克里波奇卡]]穿着舱外宇航服“奥兰-MK”，打开空间站的[[码头号对接舱|码头号对接舱]]舱门，开始了2011年的首次舱外作业。

他们的主要任务是将一个高速通讯系统的壳体安装到[[星辰号服务舱|星辰号服务舱]]表面，并铺设和连接相关电缆。随后，他们还成功拆卸了一件仪器，并安装了一架摄像机。整个工作持续了5个多小时。<ref>{{cite web |author=耿锐斌 |title=国际空间站两名俄罗斯宇航员完成太空作业 |url=http://news.sciencenet.cn/sbhtmlnews/2011/1/241047.html?id=241047 |publisher=科学时报 |archiveurl=https://web.archive.org/web/20190413030133/http://news.sciencenet.cn/sbhtmlnews/2011/1/241047.html?id=241047 |archivedate=2019-04-13 |date=2011-01-24}}</ref>

莫斯科时间2011年2月16日，他和斯克里波齐卡一起完成了他的第二次舱外任务。<ref>{{cite web |url=http://forum.nasaspaceflight.com/index.php?topic=61.1080 |title=Schedule of ISS flight events |publisher=NASAspaceflight.com |date=2010-07-17 |accessdate=2010-07-24|language=en}}</ref>
