'''德国之声'''（{{lang-de|'''Deutsche Welle'''}}，德文缩写为“{{lang|latin|'''DW'''}}”）是按《德国之声法案》设立，並由德國政府提撥預算資助的[[政治宣传|宣傳]]機構，是[[德国广播电视联合会|德国广播电视联合会]]（ARD）的成员，前身為[[西德|西德]]政府設立的廣播公司。總部設於[[波昂|波昂]]和[[柏林|柏林]]。在60多個國家共有1500多名正式員工與1500多名自由員工，與波昂及柏林的同事合作製作廣播、電視以及網際網路的資訊服務。以29种语言播出广播节目，以[[德语|德语]]、[[英语|英语]]、[[西班牙语|西班牙语]]和[[阿拉伯语|阿拉伯语]]播出电视节目。內容上側重國際時事，也包含介紹德國，以及德國和其他國家之間的雙邊交流。

德国之声网站是德国之声多媒体、互动的网上服务项目。德国之声网站以30种语言提供有关德国、[[欧盟|欧盟]]以及国际的时事新闻、背景報導。德国之声中文网每天更新时事、经济、文化和体育新闻，提供包括如何留学德国以及欧洲旅游等信息，并提供部分被中国大陆当局[[中华人民共和国出版物审查制度|禁止出版图书]]的在线阅读，但不提供德國當局禁止出版图书的在线阅读。

德國之聲的官方數據指出，該公司每週在世界各地觸及1億5700萬名閱聽眾。

== 概况 ==
德国之声从1953年5月3日开始广播節目，至2003年主要的驻地在[[莱茵河|莱茵河]]畔的[[科隆|科隆]]。2003年值德国之声成立50周年之际，德国之声的广播部和网络部搬迁到德国旧都[[波恩|波恩]]以前的政府区。德国之声的电视节目在[[柏林|柏林]]制作。

== 资金出處 ==
总体来说，德国之声的资金来自于纳税人，每年德国之声从德國联邦政府得到拨款。也就是说，德国之声的经费并非来自广播电视用户的收费。在有限的范围内德国之声可以提供一定的广告时间。

2003年，德国之声的经费大约为2.77亿[[欧元|欧元]]。经费支出的具体详情记载于每年的年度报告之中。德国之声的目标和计划也在中期计划報告中详细说明。自1999年以来，联邦政府不断缩减开支，德国之声也受到影响，不得不采取裁员，缩减短波广播语种等一些节约措施。但是，德国之声也看到因特网发展的前途，并逐渐扩充德国之声网站的内容和语种，并开发除新闻之外其他的因特网服务项目。

== 历史==

[[File:DW-TV.jpg|thumb]]]]
[[File:DWTV_Ident.jpg|thumb]]
早在1924年就出现了“德国之声有限公司”，这是一个德国所有地区的广播联合机构，座落在柏林。但是这和1953成立的“德国之声”毫无干系。德国之声于1953年5月3日首次以[[短波|短波]]播出[[德语|德语]][[节目|节目]]。当年的[[德国联邦总统|德国联邦总统]][[特奧多爾·豪斯|特奧多爾·豪斯]]向“在全世界的同胞们”问好。几周之后，也就是1953年6月11日，[[德广联|德广联]]的成员签署[[合约|合约]]成立共同的德国之声短波节目。这个节目首先由当年的[[北德广播公司|北德广播公司]]负责，之后又由座落在科隆的[[西德广播公司|西德广播公司]]接管。该公司的台长对德国之声的节目负责。一年之后，德国之声启动[[英语|英语]]、[[法语|法语]]、[[西班牙语|西班牙语]]和[[葡萄牙语|葡萄牙语]]节目。

1960年通过立法“德国之声”正式成为独立的广播电台。1962年起，德国之声陆续扩充其他语言丰富广播节目。1990年，随着[[两德统一|两德统一]]，原[[民主德国|民主德国]]的[[柏林国际广播电台|柏林国际广播电台]]终止对外广播。该电台的一些工作人员和其播出[[频率|频率]]和播出设备被德国之声採納。

两年之后，也就是1992年4月1日，德国之声在柏林以德语和英语播出DW-TV[[电视节目|电视节目]]。该[[电视|电视]]节目通过[[卫星|卫星]]播出，在以后的几年内不断扩充。现在德国之声共有5个电视频道，分别为DW-TV（英语频道，面向全球播出）、DW-TV Arabia（阿拉伯语频道）、{{lang|es|DW-TV Español}}（西班牙语频道）、DW-TV Deutsch+（美洲频道，每天播出20小时德语节目和4小时英语节目）和DW-TV Deutsch（德语频道，面向亚太及欧洲播出）。

1994年，德国之声以www.dwelle.de的网址推出网站，这在德国所有公法性的[[媒体|媒体]]中开了首例。

2003年德国之声从科隆迁址到波恩。据2002年德国之声年度报告中报道，此次搬迁的费用为1500万[[欧元|欧元]]。

== 现况 ==
经费削减对德国之声造成很大的影响。2004年，削减的金额就达到了75万欧元。德国之声重新进行内部结构重组，并调整节目的方向和重点，以适应不断变化的政治气候和媒体市场的需求。最近，通过立法，德国之声的网站DW-World正式与广播、电视并立，成为德国之声多媒体的三大支柱之一。德国之声网站的重点语言是：德语、英语、华语、俄罗斯语、西班牙语、葡萄牙语和阿拉伯语。德国之声2014年上任台长是彼得·林堡（{{lang|en|Peter Limbourg}}）。德国之声已于2011年10月30日停止除非洲、亚洲以外的短波广播。

== 部門 ==
=== 中文部 ===
德國之聲中文廣播成立於1965年3月7日，目前有近20名工作人員，並在全球範圍內擁有一個通訊員網路。德國之聲中文部通過信件、電話、電郵與聽眾聯繫，並於2006年開通手機簡訊平台。2018年開始,德國之聲在台北設立亞洲辦公室。

==== 中文广播 ====   
中文广播每天通过互联网向全球各地的[[华人|华人]]用[[現代標準漢語|華語]]播音60分钟。德国之声中文节目以国际时政为主要内容，此外还有文化地理历史等专栏节目。

==== 亚洲广播 ====
德国之声在斯里兰卡、新加坡、韩国都建有发射台，斯里兰卡的发射台设在亭可马里（Trincomalee），有一部400千瓦中波发射机，3部250千瓦的短波发射机，新加坡的发射台有100千瓦的短波发射机，韩国的金堤发射台有250千瓦的短波发射机。

==== 台湾广播 ====
德国之声在台湾也使用[[调频|调频]]和[[中波|中波]]落地播出。如花莲的FM97.3和台北的AM1494，现由[[国立教育广播电台|国立教育广播电台]]转播。

==争议==

=== 张丹红事件 ===
{{争议}}
2008年8月4日，[[北京奥运会|北京奥运会]]开幕前四天，时任德国之声中文部副主任的张丹红受邀参加[[德国电视二台|德国电视二台]]一访谈话节目时，宣称[[中共|中共]]在过去的几十年中成功「使四亿人脱贫」，说明「[[中国共产党|中国共产党]]比世界上任何一支政治力量在实践人权宣言第三条方面的贡献都要大」。<ref>{{cite web |url=http://heqinglian.net/2008/12/06/dw-zhangdanhong-3/ |title=张丹红为什么不能面对事实？|publisher=heqinglian.net |date=2009-01-28 |accessdate=2014-08-21}}</ref>此前的7月24日，张丹红在德国电视二台节目中谈到[[西藏|西藏]]时表示，中共「为保护[[西藏文化|西藏文化]]做了很多事情」等。德国《焦点》杂志称张丹红「向[[共匪|中国共产党]]献媚」，表示张违反「德国之声」的宗旨，张丹红因这两次的「不实」言论停职。<ref>{{cite web | url=http://column.creaders.net/columnViewer.php?id=285606&actid=279920 |title= 红色渗透与西方价值的自卫 |publisher=万维 |date=2008-10-02 |accessdate=2014-08-21}}</ref>

此事之后，[[共匪|中共]]党媒一边倒地指责「[[反华|反华]]情绪」和「反华势力」，説张丹红「因言获罪」与[[西方|西方]]标榜的「[[新闻自由|新闻自由]]不符」。靡下[[人民网|人民网]]声称，德国之声标榜客观公正，但自從2008年張丹紅因言論被免職事件之後，改組後的德國之聲中文部對中华人民共和国之抨擊與日俱增，上万份中文报道「遭審查」。並且不再像以往那樣每周刊出讀者反饋。而以往的讀者反饋，批評德國之聲的評論佔有很大的一部分<ref name="人民网">[http://media.people.com.cn/GB/40606/8544791.html 德国之声中文部主任被撤职 张丹红事件骤然升级]</ref>。[[环球网|环球网]]声称，德国议会插手此事被親中者斥为「是要搞新闻检查」<ref name="huanqiu">{{cite book |url=http://world.huanqiu.com/roll/2008-12/320070.html |title=德议会激辩“张丹红事件” 反张派被追问露马脚 |publisher=环球网 |date=2008-12-21 |accessdate=2014-06-03}}</ref>。不过也有中国大陆媒体报道称张丹红在任职期间，也发表过不少「反华言论」，呼吁公众理性看待该事件<ref>{{cite book |url=http://www.china.com.cn/review/txt/2008-09/05/content_16390731.htm |title=理性解读“张丹红事件”|publisher=中国网 |date=2008-09-05 |accessdate=2014-06-03}}</ref>。

2008年9月，詩人[[徐沛|徐沛]]批評張丹紅用流利的德语讲出[[中宣部|中宣部]][[謊言|论调]]，正如[[德國社會民主党|德國社會民主党]][[议员|议员]]{{link-de|Dieter Wiefelspütz}}所形容的「独一无二的灾难」<ref name="beijingspring2008">[http://beijingspring.com/bj2/2008/300/2008928234018.htm 《白玫瑰》与《小花》]</ref><ref>[http://m.spiegel.de/kultur/gesellschaft/a-580036.html Spiegel: Eklat bei der Deutschen Welle]</ref><ref>[http://www.spiegel.de/kultur/gesellschaft/kampagne-gegen-deutsche-medien-der-nazi-geist-kehrt-zurueck-a-577971.html Kampagne gegen deutsche Medien: "Der Nazi-Geist kehrt zurück"]</ref>，并且譴責張丹紅在中共[[對中共摘取法輪功學員及良心犯器官的指控|活摘法輪功學員器官]]正發生時還美化中共<ref>[http://www.newsmax.de/deutsche-welle-ein-instrument-der-kp-chinas-pressemitteilung30226.html Newsmax：Deutsche Welle ein Instrument der KP Chinas?]</ref>。

[[何清涟|何清涟]]在2008年11月接受阿波罗网采访，谈到张丹红先是于2005年3月邀请何女士在德国之声发表评论，后因中共新闻机构的意见，在当年8月出尔反尔称何女士没有资格在德国之声发表评论。在2008年8月张丹红公开为中共辩护被揭露之后，10月7日德国之声中文部主任冯·海因又邀请何清涟女士为中文部写评论。此时正好是德国国内就张丹红事件对德国之声报道倾向质疑之际，何女士认为冯·海因的邀请明显带有利用异议人士为德国之声信誉背书的嫌疑，感到受到侮辱，认为德国之声缺乏自爱自重，以本身功利主义立场去推想他人，是对他人人格的侮辱。<ref>{{cite web |url=http://www.aboluowang.com/2008/1107/109202.html#sthash.fhVgJ7TC.dpbs | title= 外媒及德国之声中文部 不为人知秘密 |publisher=阿波罗网 |date=2008-11-07 |accessdate=2014-08-21}}</ref>

2008年12月，张丹红在接受德国之声采访中回应[[何清涟|何清涟]]的指控，指出何清涟的说法「完全是谎言」。称「我通知何清涟不能写评论最迟是2005年秋，何清涟甚至认为更早，而中国国务院新闻办对我们的指责是2006年11月，中间相隔一年多的时间，而且是我给何清涟写邮件在先。这样颠倒时间上的顺序如果不是有意的，那么在接受采访时以自己的推测为依据，最起码也可以说是不严肃；如果是有意而为，那简直就是无耻了。」<ref>{{cite web |url=http://www.naol.cc/news/world/0812/1205-6.html | title= 德国之声：张丹红驳何清涟颠倒事实的侮辱 |publisher=北美在线 |date=2008-12-05 |accessdate=2014-09-08}}</ref>

=== 华人编辑被停职事件 ===
