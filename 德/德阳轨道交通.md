'''德阳轨道交通'''是四川省[[德阳市|德阳市]]规划的大中运量的城市及市域轨道交通系统。线路类型包括[[地鐵|地铁]]、[[磁懸浮列車|磁悬浮]]与中运量轨道交通。德阳市的轨道交通系统建成后，将承担德阳市内的交通和与[[成都市|成都市]]的互联互通需求。

== 德阳到成都的中低速磁悬浮 ==
该线路由[[西南交通大学|西南交大]]与成都市政府合作主导，建成后将作为西南交大的中低速磁悬浮试验线与德阳到成都的市域快线使用。目前该线路处于可行性研究阶段，有两条备选线路。<ref>{{Cite web|url=http://news.chengdu.cn/2016/0805/1807360.shtml|title=德阳到成都有望开跑磁悬浮 成都北延线力争9月开工|accessdate=2017-07-18|author=|date=|publisher=}}</ref>

=== 规划线路 ===

==== 德阳客运南站至成都天府国际机场方案 ====
德阳客运南站-[[金堂县|金堂]]（换乘[[成都地铁11号线|成都地铁11号线]]、24号线）-淮口-龙简新城-[[成都天府国际机场|天府国际机场]]（换乘[[成都地铁18号线|成都地铁18号线]]）

==== 德阳火车站至杜家碾方案 ====
[[德阳站|德阳火车站]]-扬嘉-金轮-兴隆-高坪-成都国际商贸城-杜家碾（与[[成都地铁1号线|成都地铁1号线]]、[[成都地铁5号线|5号线]]换乘）<ref name=":0">{{Cite web|url=http://deyang.house.qq.com/a/20161114/036595.htm|title=探秘德阳未来轨道交通 磁悬浮+地铁与成都“接轨”|accessdate=2017-07-18|author=|date=|publisher=}}</ref>

=== 车辆 ===
该线路规划采用西南交大自主研制的中低速磁悬浮列车运营。

== 成都地铁向德阳境内延伸计划 ==
德阳市与成都地铁近期正在研究1号线、3号线、5号线和15号线向德阳延伸的方案。<ref>{{Cite web|url=http://news.163.com/16/0125/05/BE5C4KEF00014AED.html|title=未来 成都4条地铁延伸到德阳(图)|accessdate=2017-07-18|author=|date=|publisher=}}</ref>同时，根据成都地铁远期规划，亦有多条线路将延伸至广汉或德阳市区。

== 德阳到成都的中运量轨道交通 ==
德阳市政府规划有自德阳到成都青白江的中运量轨道交通系统。该系统采用BOT模式建设，制式为有轨电车或轻轨。目前该项目仍处于初步规划研究阶段。

=== 线路 ===
德阳城北客运枢纽-德阳市区-德阳城南客运枢纽-成都青白江

=== 换乘 ===
规划在成都青白江与[[成都地铁3号线|成都地铁3号线]]换乘。<ref>{{Cite web|url=http://sc.sina.com.cn/news/m/2015-04-29/detail-iavxeafs6510625.shtml|title=德阳到成都将建轨道交通 对接地铁3号线|accessdate=2017-07-18|author=|date=|publisher=}}</ref>

== 德阳市区内的轨道交通系统 ==
德阳市规划有5条市区内轨道交通线路，制式未定。目前这五条线路仍处于初步规划研究阶段。

=== 线路 ===

==== R1 ====
德阳二环路-德阳北站-德阳市区-德阳汽车南站

==== R2 ====
什邡-德阳西站-南北两支线

===== 北支线 =====
德阳北站-中江

===== 南支线 =====
德阳市区-寿丰

==== R3 ====
庐山北路-庐山路-经开区-银山南路

==== R4 ====
秦岭路-德阳市区-德阳汽车南站-林山路
