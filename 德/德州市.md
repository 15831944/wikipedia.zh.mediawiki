'''德州市'''是[[中华人民共和国|中华人民共和国]][[山东省|山东省]]下辖的[[地级市|地级市]]，位于山东省西北部，[[黄河|黄河]]下游。东与[[济南|济南]]、[[滨州|滨州]]两市相邻，南与[[聊城市|聊城市]]相连，西接[[河北省|河北省]][[衡水|衡水]]、[[邢台|邢台]]两市，北与河北省[[沧州市|沧州市]]相接。市区邻近冀、鲁两省边界，[[京杭大运河|京杭大运河]]流经境内。全市总面积10,358平方公里，人口574.23万。

== 历史沿革 ==
[[秦朝|秦]]置[[鬲县|鬲县]]在今德州城区东南，属[[济北郡|济北郡]]。[[两汉|两汉]]及[[魏晋|魏晋]]鬲县属[[平原郡|平原郡]]。[[西汉|西汉]]置[[安德县|安德县]]在今[[陵城区|陵城区]]东南，属平原郡。[[北魏|北魏]]年间，安德县为平原郡治，徙鬲县至今陵城区，[[北齐|北齐]]省鬲县。[[隋朝|隋]][[开皇|开皇]]九年（589年）置[[德州|德州]]，治安德县，即今陵城区；又分置[[将陵县|将陵县]]于今陵城区北。[[大业|大业]]初年，改德州为平原郡。

[[唐朝|唐]]初复为德州，[[天宝|天宝]]元年（742年）复改为平原郡，[[乾元|乾元]]元年（758年）复为德州。[[北宋|北宋]]移将陵县至今德城区。[[元朝|元]][[元宪宗|宪宗]]三年（1253年）升将陵县置[[陵州|陵州]]，[[至元|至元]]二年（1265年）改置为[[陵县|陵县]]，翌年复改为陵州。[[宋代|宋代]]至[[元代|元代]]安德县仍为德州治。[[明朝|明]][[洪武|洪武]]元年（1368年）省安德县入德州，降陵州为陵县，洪武七年（1374年）移德州至故陵县，即今德州城区，属[[济南府|济南府]]；洪武十三年（1380年）置陵县于今陵城区，属济南府。[[清代|清代]]因之。

[[民国|民国]]二年（1913年）废德州为[[德县|德县]]，属[[东临道|东临道]]。1928年直属山东省。[[中华人民共和国|中华人民共和国]]成立后，1950年5月始建[[德州专区|德州专区]]，由渤海行署区的沧南、泺北两专区合并而成。1952年10月，[[河北省|河北省]]的[[恩县|恩县]]、[[武城|武城]]、[[夏津|夏津]]3县划回德州专区；[[吴桥|吴桥]]、[[东光|东光]]、[[南皮|南皮]]、[[庆云|庆云]]、[[宁津|宁津]]、[[盐山|盐山]]6县划回河北省；撤销陵县，并入德县。

1956年，撤销德州专区，辖县分別划归[[惠民|惠民]]、[[聊城|聊城]]两专区，同时撤销[[德平县|德平县]]和[[恩县|恩县]]。1961年7月，恢复德州专区，同年10月改德县为陵县。1964年12月，河北省的[[宁津|宁津]]、[[庆云|庆云]]两县划入德州专区。1967年德州专区改为[[德州地区|德州地区]]。1989年12月，将[[商河|商河]]、[[济阳|济阳]]两县划归[[济南市|济南市]]。1994年12月，撤销德州地区，改设地级德州市，原县级德州市改为德城区。2014年12月，撤销陵县，设立陵城区。

== 地理 ==
地处[[华北平原|华北平原]]，位于山东省西北部，[[京杭大运河|京杭大运河]]边上，是[[京沪铁路|京沪线]]、[[石德铁路|石德铁路]]、[[京沪高铁|京沪高铁]]、[[青太客运专线|青太客运专线]]（在建）、[[德龙烟铁路|德龙烟铁路]]（在建）上重要的中等城市及全国重要的交通枢纽之一，2011年名列中国城市GDP总量第71位，《[[福布斯|福布斯]]》中文版发布的2013年度 “中国大陆最佳商业城市排行榜”第84位。德州物产丰富，盛产棉花和西瓜，尤以“德州西瓜”、“[[德州扒鸡|德州扒鸡]]”、“[[乐陵金丝小枣|乐陵金丝小枣]]”闻名。

*接壤地区：[[济南|济南]]（南）， [[聊城|聊城]]（西南）， [[滨州|滨州]]（东），[[河北|河北]][[沧州|沧州]]（北），[[河北|河北]][[衡水|衡水]]（西北）

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为556.82万人<ref>{{cite book |author=德州市统计局|title=《德州市2010年第六次全国人口普查主要数据公报》}}</ref>，与[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加了27.45万人。增长5.19%，年平均增长0.51%。其中，男性为281.87万人，占总人口的50.62%；女性为274.95万人，占总人口的49.38%，总人口性别比（以女性为100）为102.52。0－14岁的人口为92.69万人，占16.65%；15－64岁的人口为414.68万人，占74.47%；65岁及以上的人口为49.45万人，占8.88%。

== 交通 ==

== 教育 ==
德州市的教育程度在[[山东省|山东省]]处于中游水平。教育质量较好的学校多位于德城区内。[[德州学院|德州学院]]（二本）为德州市的较高的教育机构。德州市比较有名的学校有：[[德州五中|德州五中]]、[[德州九中|德州九中]]、[[乐陵市实验中学|乐陵市实验中学]]（初中）、[[德州一中|德州一中]]（高中）、[[德州学院|德州学院]]（二级大学本科）、[[武城二中|武城二中]]（高中）、[[德州市实验小学|德州市实验小学]]（小学）等。
