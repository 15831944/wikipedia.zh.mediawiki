'''-{德}-克萨斯共和国'''（1836年－1845年，[[清|清]][[徐繼畬|徐繼畬]]於1844年所著《[[瀛寰志略|瀛寰志略]]》中譯作'''得撒'''），又译'''-{得}-克萨斯共和国'''或'''-{zh-hans:德萨斯共和国;zh-hk:德克薩斯共和國;zh-tw:德薩斯共和國}-'''，亦称'''孤星共和國'''，於1836年自[[墨西哥|墨西哥]]獨立出來的國家，九年後（1845年）經議會決議合併加入[[美國|美國]]，成為其一州。其領土範圍涵蓋現今美國[[德克薩斯州|德克薩斯州]]以及部份的[[新墨西哥州|新墨西哥州]]、[[科羅拉多州|科羅拉多州]]、[[懷俄明州|懷俄明州]]、[[奧克拉荷馬州|奧克拉荷馬州]]、[[堪薩斯州|堪薩斯州]]。其東界及於1819年由美國和[[西班牙|西班牙]]簽署的《[[亚当斯奥尼斯条约|亚当斯奥尼斯条约]]》所訂疆界；南界與西界在其國祚九年中一直與[[墨西哥|墨西哥]]爭執不下：德克薩斯共和國宣稱其疆界遠及於[[格蘭德河|格蘭德河]]；墨西哥則稱其邊界僅及較東的[[努埃塞斯河|努埃塞斯河]]。這個邊界未定的問題，除了是德克薩斯共和國加入美國聯邦的原因之一，在德克薩斯共和國併入美國成為一州之後，演變成為[[美墨戰爭|美墨戰爭]]的主要導火線。

== 独立 ==

在墨西哥独立之前，1821年得克萨斯是[[新西班牙|新西班牙]]的[[西班牙|西班牙]][[殖民地|殖民地]]的一部份。直到独立前，得克萨斯一直属于墨西哥，作为{{le|科阿韋拉及德克薩斯州|Coahuila y Tejas}}的一部分。从1820年代开始，大量美国移民涌入得克萨斯。1835年12月20日，得克萨斯人在戈里亚得签署了第一份独立声明。得克萨斯正式宣布独立是在1836年3月2日，建立了得克萨斯共和国。

四天以后，墨西哥当时的独裁者[[安東尼奧·洛佩斯·德·桑塔·安納|安東尼奧·洛佩斯·德·桑塔·安納]]将军在两星期的进攻之后，殲滅了据守[[阿拉莫_(得克薩斯州聖安東尼奧)|阿拉莫]]约200名的得克萨斯人，「毋忘阿拉莫」变成了得克萨斯独立的口号。1836年4月21日在圣哈金托（现在的[[休士頓|休士頓]]）附近，安東尼奧·洛佩斯·德·桑塔·安納将军所率领的墨西哥军队被[[山姆·休士頓|山姆·休士頓]]将军所率领的800名得克萨斯人所击败，1600名墨西哥人被杀或被俘，安東尼奧·洛佩斯·德·桑塔·安納将军本人也被俘了，而得克萨斯人只有9人战死。这次决战导致得克萨斯真正从墨西哥独立。

== 代表人物 ==
* [[山姆·休士頓|山姆·休士頓]]，弗吉尼亚人，1836年-1838年和1841年-1844年間两次担任得克萨斯共和国总统。他在1859年到1861年担任得克萨斯州的州长。

* 1836年10月，得克萨斯共和国的第一国会在哥伦比亚(现在西部哥伦比亚)召开。斯蒂芬.F.奥斯汀(被稱為得克萨斯之父)在擔任新共和国国务卿兩個月後，於1836年12月27日逝世。

== 首都 ==

从1836年开始，直到[[山姆·休士頓|山姆·休士頓]]将首都迁往[[休士頓|休士頓]]之前，五个城鎮担当了得克萨斯（布拉索斯河畔的華盛頓、哈里斯堡、加尔维斯敦、维拉斯克和哥伦比亚）临时首都。1839年，首都被迁往向[[奥斯汀|奥斯汀]]新镇。

== 政治 ==

共和国的内部有二个相互冲突的政治派别。一派是民族主义的派别，领袖是[[米拉波·波拿巴·拉馬爾|米哈博.B.拉马尔]]，主张得克萨斯继续维持独立并扩展到[[太平洋|太平洋]]。对立派，领袖是總統[[山姆·休斯敦|山姆·休斯敦]]，主张得克萨斯与美国合并，与美国和当地美国人和平共处。

== 合并 ==
1845年2月28日，美國眾議院以132票對76票，通過參議院版聯合決議與得克萨斯共和國合并。7月4日，得克萨斯共和國憲法大會以55票對1票，通過接受美國提案。10月13日，得克萨斯共和國公投以7664票對430票通過合併案。12月16日，美國眾議院以141票對58票（21票棄權）通過接納得克萨斯。12月22日，美國參議院以31票對14票（7票棄權）通過接納得克萨斯。得克萨斯共和國至此併入美國。<ref>{{Cite web | url=https://www.tsl.texas.gov/exhibits/annexation/part5/question12.html | title=Texas Annexation Questions and Answers | publisher=得克萨斯州政府}}{{en}}</ref><ref>{{Cite web | url=https://www.tsl.texas.gov/treasures/earlystate/annex-01.html | title=Annexation | publisher=得克萨斯州政府}}{{en}}</ref>
