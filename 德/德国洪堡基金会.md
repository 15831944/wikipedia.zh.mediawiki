'''德国洪堡基金会'''全称亚历山大·冯·洪堡基金会（[[德语|德语]]：Alexander von Humboldt-stifurg）是[[德国|德国]]政府为了纪念自然学家[[亚历山大·冯·洪堡|亚历山大·冯·洪堡]]于1860年创建的具有司法权能的基金会。
== 简介 ==
德国洪堡基金会促进来自国外和德国的优秀科学家和学者之间的学术合作<ref>{{cite web |author1=Alexander von Humboldt-Stiftung |title=About the Foundation |url=http://www.humboldt-foundation.de/web/about-us.html |website=Alexander von Humboldt-Stiftung |accessdate=2019-04-16}}</ref>。德国洪堡基金会不但为杰出的科学家提供赞助，而且还将他们整合在一个跨越世界的网络中。这个“洪堡家庭”将来自世界的学术精英与德国联系在一起<ref>{{cite web |author1=Alexander von Humboldt-Stiftung / Foundation |title=strategic focus |url=http://www.humboldt-foundation.de/web/strategic-focus.html |website=Alexander von Humboldt-Stiftung / Foundation |accessdate=2019-04-16}}</ref>。洪堡基金会以资助全世界各国优秀科学家而闻名，主要学科有：[[自然科学|自然科学]]、[[人文|人文]]、[[生命科学|生命科学]]、[[工程|工程]]等<ref>{{cite web |author1=Alexander von Humboldt-Stiftung / Foundation |title=Fellowships and Awards Granted in 2017 |url=http://www.humboldt-foundation.de/web/statistics.html |website=Alexander von Humboldt-Stiftung / Foundation |accessdate=2019-04-16}}</ref>。德国洪堡基金会每年为2000余名来自全世界的青年科学家提供资助。2018年止，已资助了来自140多个国家的2.9万名青年科学家，其中包括55名诺贝尔奖获得者<ref>{{cite web |author1=顾钢 |title=德国洪堡奖学金青睐中国学者 |url=http://univst.com/index/h1t18/2018-07/02/content_686585.shtml |website=中国科技网 |accessdate=2019-04-16}}</ref>。

洪堡基金会理事会由主席、德国科学界各大学术组织主席、联邦各州文教部长联席会议主席以及两位联邦政府部长组成<ref name="德国研究">{{cite journal |author1=赵震声 |title=德国洪堡基金与中国洪堡学者 |journal=德国研究 |date=1999 |volume=14 |issue=01}}</ref>。现任德国洪堡基金会秘书长为[[恩诺·奥夫德海德|恩诺·奥夫德海德]]（Enno Aufderheide）。洪堡基金会的资金95%来自联邦政府，1%来欧盟，4%来自其他捐赠<ref>{{cite web |author1=Alexander von Humboldt-Stiftung/Foundation |title=Partners, funders and finances |url=http://www.humboldt-foundation.de/web/partners.html |website=Alexander von Humboldt-Stiftung/Foundation |accessdate=2019-04-16}}</ref>。
1945年，基金会随着德意志帝国的瓦解，停止了活动，基金会于1953年12月10日恢复<ref name="德国研究" />。

== 与中国 ==
1933年，顾班昌（Gu Ban-Tschang 音译）成为第一位获得洪堡奖学金的中国学者，1979年，[[中华人民共和国|中华人民共和国]]第一批学者获得洪堡奖学金，1992年，北京[[中央民族大学|中央民族大学]][[耿世民|耿世民]]和[[中国科学院|中国科学院]]应用数学所[[朱力行|朱力行]]获得[[洪堡科研奖|洪堡科研奖]]，中国科学院应用数学所马志明获得[[马克思·普朗克科研奖|马克思·普朗克科研奖]]<ref name="德国研究" />。2011年，[[同济大学|同济大学]]德语系[[赵劲|赵劲]]教授作为首位中国学者获得[[青年杰出学者洪堡奖|青年杰出学者洪堡奖]]<ref name="同济大学" />。
从1953年到2017年间，先后有2170名中国学者获得洪堡基金会各类奖学金资助<ref>{{cite news |author1=张毅荣 |author2=张远 |title=德国洪堡基金会：中国学者为两国科研合作搭建桥梁 |url=http://www.xinhuanet.com/2018-06/30/c_1123059153.htm |accessdate=2019-04-16 |work=新华网 |agency=新华社 |date=2018-06-30}}</ref>。

德国洪堡基金会秘书长[[恩诺·奥夫德海德|恩诺·奥夫德海德]]上任后多次访问中华人民共和国，2011年2月24日，[[中国科学院|中国科学院]]院长路[[甬祥|甬祥]]在京会见来访恩诺·奥夫德海德<ref>{{cite web |author1=中国科学院 国际合作局 |title=路甬祥会见德国洪堡基金会新任秘书长Aufderheide博士一行 |url=http://www.cas.cn/xw/ldhd/201102/t20110227_3076283.shtml |website=中国科学院 |accessdate=2019-04-16}}</ref>。2013年9月19日，同济大学裴钢校长、董琦副校长在[[同济大学|同济大学]]旭日楼会见恩诺·奥夫德海德<ref name="同济大学">{{cite web |title=洪堡基金会秘书长来访 |url=https://fao.tongji.edu.cn/fa/df/c7472a64223/page.htm |website=同济大学 |accessdate=2019-04-16}}</ref>。2019年3月22日，恩诺·奥夫德海德于[[浙江大学|浙江大学]]玉泉校区举办讲座<ref>{{cite web |title=德国洪堡基金会秘书长Dr. Enno Aufderheide讲座 |url=https://www.zju.edu.cn/2019/0319/c504a1083807/pagem.htm |website=浙江大学 |accessdate=2019-04-16}}</ref>。2019年3月23日，恩诺·奥夫德海德访问[[浙江工业大学|浙江工业大学]]，校长李小年会见来宾<ref>{{cite web |author1=浙江工业大学国际交流与合作处 |title=李小年会见德国洪堡基金会秘书长恩诺?奥夫德海德一行 |url=http://www.wsc.zjut.edu.cn/newsDetail.jsp?id=1688 |website=浙江工业大学 |accessdate=2019-04-16}}</ref>。
