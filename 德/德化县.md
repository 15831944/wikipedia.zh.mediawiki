'''德化县'''（{{lang-nan|德化縣|poj=Tek-hòe-kūiⁿ|tl=Tik-huè-kuīnn}}）是[[中华人民共和国|中国]][[福建省|福建省]][[泉州市|泉州市]]下辖的一个[[县|县]]。中国当代[[瓷器|瓷器]]的著名产地，1996年被国务院发展研究中心命名为“中国陶瓷之乡”，2003年又被称评为“中国民间（陶瓷）艺术之乡”，获“中国瓷都·德化”之称。有些媒体将德化县与[[景德镇|景德镇]]、[[醴陵|醴陵]]并称为中国古代三大瓷都<ref>{{cite news|url=http://www.toutiao.com/i6413568675458056705/|title=细读中国三大瓷都之一德化|author=泉州新闻通|publisher=[[今日头条|今日头条]]}}</ref>。面积2220平方千米，人口30万。县人民政府驻[[浔中镇|浔中镇]]。

== 历史沿革 ==
唐[[光启|光启]]二年（886年），[[王潮|王潮]]克[[泉州|泉州]]，其弟[[王审知|王审知]]創立[[閩國|閩國]]。[[閩國|閩國]]龙启元年（933年），以[[永泰县|永泰县]]归德场置德化县。

[[宋代|宋代]]属[[泉州|泉州]]，[[元代|元代]]属[[福建行省|福建行省]]泉州路。明朝属福建省泉州府。

[[清|清]][[雍正|雍正]]時改属[[永春直隶州|永春直隶州]]。1950年10月劃歸泉州。

* 镇：[[浔中镇|浔中镇]]、[[龙浔镇|龙浔镇]]、[[三班镇|三班镇]]、[[龙门滩镇|龙门滩镇]]、[[雷峰镇|雷峰镇]]、[[南埕镇|南埕镇]]、[[水口镇_(德化县)|水口镇]]、[[赤水镇_(德化县)|赤水镇]]、[[葛坑镇|葛坑镇]]、[[上涌镇|上涌镇]]。
* 乡：[[盖德乡|盖德乡]]、[[杨梅乡_(德化县)|杨梅乡]]、[[汤头乡|汤头乡]]、[[桂阳乡|桂阳乡]]、[[国宝乡|国宝乡]]、[[美湖乡|美湖乡]]、[[大铭乡|大铭乡]]、[[春美乡|春美乡]]。

== 地理 ==
以中低山地为主，最高峰[[戴云山|戴云山]]，海拔1856米，位于县中心。

== 经济 ==
主要以[[陶瓷器|陶瓷]]出口为主，经济产业较为单一，德化丧失了很多经济亮点，近年来随着政府将绿色生态作为一个卖点，旅游是一个重要的突破口。

== 历代名人 ==
*[[何朝宗_(德化)|何朝宗]]，德化县浔中乡隆泰村后所人。明代驰名中外的瓷雕艺术大师。何朝宗的祖籍在江西省临川县甘二都，后移居南城县十六都四脚桥。其先祖何昆源，号德举，明洪武二年（1369）任江西建昌府卫军，七年调泉州右卫所任右营旗宫，十七年奉命拨军到德化县隆泰社厚苏村（即今后所）屯垦，并定居开基该地。其子世样、孙尚志皆习文，先后出任莆田县儒学训导和古田县儒学教谕。
