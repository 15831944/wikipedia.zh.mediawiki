'''德惠市'''位于中国[[吉林省|吉林省]]中北部，是[[长春市|长春市]]代管的一个县级市。市区位于[[长春|长春]]、[[哈尔滨|哈尔滨]]、[[吉林|吉林]]三大城市中间，处于东北物流中心[[大连|大连]]—[[沈阳|沈阳]]—[[长春|长春]]—[[哈尔滨|哈尔滨]]经济带上。人口83万，有汉、满、回、朝等15个民族，汉族占99%以上。德惠市为[[中国肉类生产百强县列表|吉林肉类生产第一大、中国肉类生产第1大县市]]。

== 历史 ==
德惠在[[秦|秦]]以前为[[肃慎|肃慎]]地，[[汉代|汉代]]起属[[夫余|夫余]]国。[[隋|隋]][[唐|唐]]时期属[[渤海国|渤海国]]扶余府。[[辽代|辽代]]属辽东京、金代属金上京管辖。[[元代|元代]]属辽阳中书行省开元路，为蒙古游牧区。[[明|明]]初属辽东都指挥司和奴尔干都指挥司，后属兀良哈三卫和科尔沁部。[[清|清]][[康熙|康熙]]年间筑“[[柳条边|柳条边]]”，这里划为“边外”，属郭尔罗斯前旗领主牧场。[[乾隆|乾隆]]五十六年（1791年）后，山东、河北一带民户入境垦种。[[嘉庆|嘉庆]]年间设怀惠乡、沐德乡，属[[长春|长春]]厅。光绪十六年（1890年）长春府于朱家城子设分防照磨，管理沐德、怀惠二乡缉捕事宜。

清末[[宣统|宣统]]二年四月十六日（1910年5月24日）钦批设县，取沐德、怀惠之尾字名为德惠，县城设于[[大房身镇_(德惠市)|大房身]]，仍隶属[[长春|长春]]府。沦陷时期，1936年4月1日伪县公署迁到张家湾街(窑门站)，张家湾遂改名[[德惠镇|德惠]]。1994年7月经国务院批准撤县设市，所辖地域不变。

== 地理 ==
德惠市位于[[吉林|吉林]]省中北部，松辽平原腹地。全市地势南高北低，呈波状起伏，但相对落差不大，大部分海拔高度在149-241米之间。地处东经125°14'- 126°28'，北纬44°33'- 44°51'，幅员面积3435平方公里。全市年平均气温4℃，降水量520毫米。东南与[[九台|九台]]市接壤，西南毗邻[[长春|长春]]市郊，西部与农安县以伊通河为界，东北与扶余县、[[榆树|榆树]]市隔江相望。境内有[[松花江|松花江]]、[[饮马河|饮马河]]、[[伊通河|伊通河]]等河流。属中温带半湿润季风气候。

== 经济 ==
===农牧业===
农业盛产[[玉米|玉米]]、[[水稻|水稻]]、[[高粱|高粱]]、[[大豆|大豆]]等，是国家重要的商品粮基地和玉米出口基地。是国家粮食生产先进市、菜篮子工程先进市，中国松花江大米之乡、和中国肉鸡之乡。是国家高产优质高效农业示范区和[[吉林|吉林]]省绿色食品基地。全国首批“生态农业建设示范县（市）”、全国首批“无公害蔬菜生产试点县（市）”，省内惟一被农业部确定的“科教兴农与可持续发展两大战略示范县（市）”。特产有德惠大曲酒、肉鸡、草编。

===工业===
工业有食品、饲料、建材、机械、医药、包装等行业。是[[吉林|吉林]]省食品工业强县（市）。设有省级经济开发区。

== 交通 ==
[[京哈铁路|京哈铁路]]，[[102国道|102国道]]和[[京哈高速公路|京哈高速公路]]从境内通过。

== 旅游 ==
德惠市有阳光明媚、空气清新的跃进水库连花自然保护区，生态园，有远近闻名的[[宝山寺|宝山寺]]，有风光秀丽的[[半拉山风景区|半拉山风景区]]，有独具特色的[[鳇鱼圈|鳇鱼圈]]天然娱乐岛，另外还有清代时期的[[贡江碑|贡江碑]]。
