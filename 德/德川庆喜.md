'''德川慶喜'''（{{bd|1837年|10月28日|1913年|11月22日|catIdx=Tokugawa, Yoshinobu}}），[[江戶幕府|江戶幕府]]（[[德川幕府|德川幕府]]）第15代[[征夷大將軍|將軍]]，也是末代將軍。

== 將軍生涯 ==
德川庆喜出生於德川[[德川御三家|御三家]]之一的[[水戶家|水戶家]]，父為[[水戶齊昭|水戶齊昭]]，母[[登美宮吉子|登美宮吉子]]為齊昭的正室；<ref>為正室所出的將軍僅有三代[[德川家光|德川家光]]與慶喜。</ref>慶喜是[[德川家康|德川家康]]么子[[德川賴房|德川賴房]]的第十代子孫。慶喜在眾多兄弟中排行第七。11歲時奉將軍[[德川家慶|德川家慶]]之命，過繼到御三卿之一的一橋家為養子，直到出任將軍為止。

慶喜面對日漸衰落的[[德川幕府|德川幕府]]，原希望在繼承後借助[[法國|法國]]協助進行西化（取得法國貸款600萬美元以招募法國軍官、購買兵器及船隻，並組成日法的公司發展工商貿易），重提[[公武合體|公武合體]]以改革幕政。不料1866年7月，法國新外相穆斯蒂（Léonel, marquis de Moustier）出掌[[政府|政府]]，不再支持法國公使羅叔亞〔或譯陸修〕（Léon Roches）聯結幕府的外交路線，反而與英國採取同一陣線，傾向組成解除幕府的公議政體；慶喜頓失外援，加上在1866年尾，一直支持公武合體，反對倒幕的[[孝明天皇|孝明天皇]]暴斃（因此有傳是倒幕派所為），更使慶喜維護幕府政權合法性的美夢幻滅。

1867年6月，[[土佐藩|土佐藩]]的[[坂本龍馬|坂本龍馬]]及[[後藤象二郎|後藤象二郎]]定下「船中八策」，主張幕府奉還大政，組成以天皇領導的大名公議政體，更取得[[薩摩藩|薩摩藩]]、[[土佐藩|土佐藩]]及[[廣島藩|廣島藩]]簽訂約定書支持。9月，[[薩摩藩|薩摩藩]]及[[長州藩|長州藩]]達成出兵協議，後再加入廣島藩組成倒幕同盟；10月，三藩代表更在京都集合，取得天皇討幕密詔，決意出兵。慶喜為避免一場日本內戰，主動在二條城裡舉行了[[大政奉還|大政奉還]]的儀式，將奉還大政予[[明治天皇|明治天皇]]。天皇頒佈《王政復古令》，廢除幕府。

慶喜希望以此作為苦肉計，以令最終能在新政體下保留實力，以便爭回主導權。不過倒幕同盟並不信任幕府舉動，決心發動以「王政復古」為名的政變，建立由天皇主導的新政府。慶喜面對朝廷及倒幕派的苛刻要求，迫於一戰，帶領幕府軍一萬五千人由[[大阪|大阪]]進攻京都，決戰為數五千人的政府軍，幕府軍卻於[[鳥羽伏見之戰|鳥羽伏見之戰]]大敗，慶喜只得逃回江戶。

最後，慶喜在[[海軍奉行|海軍奉行]][[勝海舟|勝海舟]]遊說下，同意投降，在1868年5月3日，德川慶喜交出江戶，改封於[[靜岡|靜岡]]，領70萬石地，德川幕府在[[江戶|江戶]]265年的統治結束。

== 歸隱生活 ==
退任將軍後，慶喜為免被人猜疑，對政治興趣明顯減退。在新政府任職的舊家臣，慶喜亦盡量避不見面。慶喜熱衷於投入[[攝影|攝影]]、[[狩獵|狩獵]]、[[民謠|民謠]]歌曲研究等生活興趣之中，並專注於這些工作，甚少理會政治。且因為自小便受[[水戶藩|水戶藩]][[水戶學|水戶學]]盛行的[[尊皇攘夷|尊皇攘夷]]理論熏陶，認為自己身屬亂臣，故一直都無意覲見[[天皇|天皇]]。德川本家的[[家督|家督]]，則由他的養子[[田安家|田安家]]的[[德川家達|德川家達]]接任。

[[明治|明治]]三十年（1897年），他從[[靜岡|靜岡]]回到[[東京|東京]]居住。翌年他前往[[皇居|皇居]]（即前[[江戶城|江戶城]]），首次覲見[[明治天皇|明治天皇]]。

明治三十五年（1902年），慶喜被冊封為[[公爵|公爵]]。

[[大正|大正]]二年（1913年），慶喜因[[肺炎|肺炎]]而逝世，享年七十六歲。

== 軼事 ==
* 與生父齊昭一樣都很喜歡吃[[薩摩藩|薩摩藩]]出產的[[豬肉|豬肉]]，所以被人稱為'''{{lang|ja|豚一様}}'''（就是喜好豬肉的一橋大人之意）。對西洋文物十分關心，晚年喜好[[麵包|麵包]]及[[牛奶|牛乳]]，對[[照相機|照相機]]及[[寫真|寫真]]攝影、[[釣魚|釣魚]]、騎[[腳踏車|腳踏車]]、[[顯微鏡|顯微鏡]]、手藝（[[刺繡|刺繡]]）頗有興趣。在將軍時代曾向[[西周_(啓蒙家)|西周]]學習[[法語|法語]]。
* 擁有[[駕駛執照|駕駛執照]]，明治时期在靜岡市内駕駛[[汽車|汽車]]時，曾在蕎麥屋前引起了交通事故。
* 爱好[[攝影|攝影]]，但技術不佳。虽屡次向寫真雜誌上投稿，却没有被採用过。與他有共同興趣生活的友人則是慶喜的異母弟昭武。

德川庆喜出身於[[水戶德川家|水戶德川家]]，後過繼[[一橋家|一橋家]]繼承家督。成為征夷大將軍亦同時是[[德川宗家|德川宗家]]的當主，1902年與宗家分家，創立[[德川慶喜家|德川慶喜家]]（已絕嗣）。

安政2年12月3日，與[[一條美賀子|一條美賀]]結婚。與美賀之間的長女（瓊光院殿池水影現大童女）於安政5年7月16日誕生，7月20日時夭折。之後便與美賀之間再也沒有生下孩子，明治年間一共生下了10男11女，皆是與兩位側室所生。子女有繼承德川慶喜家公爵的七男慶久、[[勝海舟|勝伯爵家]]的[[婿養子|婿養子]]十男精與成為伏見宮博恭王妃的九女經子。慶久的女兒即是[[高松宮宣仁親王|高松宮宣仁親王]]妃{{tsl|ja|宣仁親王妃喜久子|喜久子}}。
