《'''德意志之歌'''》（{{lang-de|'''Das Deutschlandlied'''}}）或《'''德意志人之歌'''》（{{lang|de|'''Das Lied der Deutschen'''}}）是[[德國|德國]]的現行[[國歌|國歌]]。

曲譜為著名[[古典音樂|古典音樂]]家[[海頓|海頓]]於1797年所作，後正式定為[[奥地利帝國|奥地利帝國]]國歌（《[[帝皇頌|帝皇頌]]》）。[[德國|德國]]版歌詞為[[自由主義|自由主義]]詩人奥古斯特·海因利希·霍夫曼·冯·法勒斯雷本教授（Prof. August H. H. von Fallersleben, 1798年-1874年）作於1841年8月26日晚。1922年全部歌詞曲被[[魏玛共和国|魏玛共和国]]第一任总统[[弗里德里希·艾伯特|弗里德里希·艾伯特]]首次定為德國國歌。在[[納粹德國|納粹德國]]統治時期，第一段特別受重視。

[[第二次世界大戰|第二次世界大戰]]後的1952年，在联邦总统[[特奥多尔·豪斯|特奥多尔·豪斯]]和联邦总理[[阿登纳|阿登纳]]之间的一次通信中，[[西德|西德]]政府定此曲為國歌，但除了第三段歌詞'''統一、正义和自由'''（'''Einigkeit und Recht und Freiheit'''）以外，其餘幾段歌詞可能引起爭議，不予採用為官方版歌詞。1991年8月19日，联邦总统[[魏茨泽克|魏茨泽克]]和联邦总理[[科尔|科尔]]在通信中确认了“德意志之歌”对统一德国的传统意义。“德意志之歌”遂正式經確認為統一後德國的聯邦國歌。這首歌也也常以首句歌詞「德意志勝過一切」稱呼，但事實上從未以此為正式曲名。

二戰後同一時期，[[德意志民主共和國|德意志民主共和國]]（[[東德|東德]]）的國歌是《[[从废墟中崛起|从废墟中崛起]]》。1990年[[兩德統一|德國統一]]，只有第三組歌詞統一、正义和自由確認爲統一的共和國國歌。

== 法治 ==
在德国历史上的三月前（Vormärz），当德意志个別分裂的邦国逐渐退出历史舞台的时候，自由派反对党呼吁统一所有德意志的邦国（参见[[德意志帝国|德意志帝国]]），倡导民族自立，政治自由。所以'''[[法治|法治]]'''，'''自由'''，'''手足情'''（Brüderlichkeit）及[[自由、平等、博爱|自由、平等、博爱]]都是歌曲的主要中心思想。

== 歌詞 ==

所有歌词均由霍夫曼·冯·法勒斯雷本所作，目前德国国歌选取第三段作为官方唱词。

== 历史背景 ==
{{Main|德意志统一}}
当[[法国大革命|法国大革命]]发生时，[[神圣罗马帝国|神圣罗马帝国]]日趋衰弱。接踵而来的[[拿破仑战争|拿破仑战争]]更是改变了[[中欧|中欧]]的政治版图。甚至在[[德意志解放战争|德意志解放战争]]前，[[恩斯特·莫里茨·阿尔恩特|恩斯特·莫里茨·阿尔恩特]]写下了著名诗歌《[[什么是德意志祖国|什么是德意志祖国]]？》。在1815年拿破仑被击败后，渴望人权、民主、自由和共和政体的思想被[[维也纳会议|维也纳会议]]所击败，会议复辟了许多專制君主。1819年，[[卡尔斯巴德决议|卡尔斯巴德决议]]后，奥地利首相[[克莱门斯·梅特涅|梅特涅]]加强了[[秘密警察|秘密警察]]对大学校园的监控，反对国内进行政治改革并镇压[[民族主义|民族主义]]及[[自由主义|自由主义]]情绪。

1815年[[德意志邦联|德意志邦联]]建立，由35个德意志邦国和4个自由市组成。各个成员国保持主权独立，随后邦国又纷纷加入[[德意志关税同盟|德意志关税同盟]]以促进贸易往来。然而在政治上，仍然存在着所谓[[德国问题|德国问题]]，即“什么是德国？”。

1848年3月[[德意志1848年革命|德意志各邦人民起义]]后，普鲁士首开先河召开立宪会议，东普鲁士加入邦联。随后德意志代表于美茵河畔法兰克福召开[[法兰克福国民议会|国民议会]]，希望预备一份宪法。但由于普鲁士、奥地利最大两邦因私利考虑联合将议会破坏，并最终掀起[[普奥战争|普奥战争]]。

== 霍夫曼 ==
霍夫曼在这一背景下的1841年创作了诗词，配合海顿的旋律发表。歌词首句“{{lang|de|Deutschland, Deutschland über alles, über alles in der Welt}}，德意志、德意志，高于一切，高于世间的一切”正是希望[[德意志地區|德意志地區]]各個[[諸侯|諸侯]]能够让权建立一个统一的[[德国|德国]]，而非仅着眼于自己的独立小邦。第三段歌词的“{{lang|de|Einigkeit und Recht und Freiheit}}，统一、法制与自由”，也是霍夫曼希望能够建立一个全新的[[法治|法治]]国家，而非君主专权<ref>{{cite web |url = http://www.bundestag.de/blickpunkt/103_Parlament/0502/0502014.html |title = Staatssymbole Zeichen politischer Gemeinschaft |first=Nadja |last=Bareth|work=[[Bundestag|Blickpunt Bundestag]] |date=February 2005|accessdate=1 December 2009 |language=de}}</ref>。

在“德意志之歌”写成后的第二年，霍夫曼因创作这首歌曲和其他革命性作品而失去了他在普鲁士[[布雷斯劳|布雷斯劳]]的教授工作。

== 地理 ==
[[File:Maas_memel_etsch_belt.svg|thumb]]及歌词中的四个[[水体|水体]]]]
当歌词写就时，实际上并不存在一个德意志国家而仅有一个邦联。它包含了一些非德语使用者的地区，但却排除了一些德语区。霍夫曼研究日耳曼文学和传说，根据他在语言学上对德国的定义，他描述了一个当时德语区的大致区域。19世纪的民族主义者也通常是根据语言使用者的分布情况来决定他们企盼的[[民族国家|民族国家]]疆界。
