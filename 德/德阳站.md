'''德阳站'''，位于[[中华人民共和国|中华人民共和国]][[四川省|四川省]][[德阳市|德阳市]][[旌阳区|旌阳区]]，是[[宝成铁路|宝成铁路]]、[[西成客运专线|西成客运专线]]、[[成绵乐客运专线|成绵乐客运专线]]、[[德天铁路|德天铁路]]上的火车站，建于1952年，在2008年的汶川大地震中严重受损，与2011年异地重建。现站房面积9995平方米，4站台面11线，由[[成都铁路局|成都铁路局]]管辖。<ref>[http://search.huochepiao.com/huochezhan/jieshao/2053.htm 德阳火车站介绍]</ref>

== 车站设计 ==
德阳站以砖红色为主色调，借鉴[[三星堆遗址|三星堆博物馆]]的外墙设计，使用圣理尼陶砖覆盖站房外墙。站房地下局部一层，地上两层，与站台通过天桥和地道连接。站台全部采用无柱雨棚，为四川省除[[成都东站|成都东站]]外仅有的采用无柱雨棚的车站。车站上升广场设置有出租车和社会车辆送站通道，地下一层设置有出租车接客平台和通向公交场站的连接通道。

* 荣聚生态园
* 德阳市公路局

==歷史==
* 建于1952年。
* 2009年11月站房重建。
* 2011年3月22日新站房正式启用。

== 未来发展 ==
德阳站将在西侧新建西站房以配合[[宝成铁路|宝成铁路]]成都至德阳段公交化改造。
