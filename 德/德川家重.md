'''德川家重'''（{{bd|1712年|1月28日|1761年|7月13日}}），[[德川幕府|德川幕府]]第九代將軍，在職時間：1745年—1760年。八代將軍[[德川吉宗|德川吉宗]]的長子，母親是側室大久保[[深德院|於須磨之方]]，乳名是長福丸。由於父親吉宗和正室理子之間並沒有子嗣，因此家重被認定是世子，享保十年（1725年）元服後改名為家重。

== 生涯 ==
[[正德|正德]]元年十二月二十一日（1712年1月28日）德川家重出生於[[武藏國|武藏國]][[江戶|江戶]]赤坂[[紀州藩|紀州藩]]邸，是第八代[[將軍|將軍]][[德川吉宗|德川吉宗]]的長子，[[母親|母親]]是[[側室|側室]]於須磨之方。家重天生體質就比較虛弱，無法用很清楚地言語表達事情，且年幼時[[大奧|大奧]]沉淪在酒色之中，健康狀況十分不好。因為這個緣故，外人對擅長文武的家重長弟[[德川宗武|德川宗武]]（田安德川家初代）都較為看好，也很自然的認為宗武將來會繼承將軍。不過吉宗依然選家重為繼嗣。[[延享|延享]]二年九月二十五日（1745年10月20日），德川家重三十三歲時父親[[德川吉宗|德川吉宗]]讓出將軍的位子，[[延享|延享]]二年十月初七日（1745年10月31日）德川家重受[[櫻町天皇|櫻町天皇]]下詔敕封為第九代[[征夷大將軍|征夷大將軍]]，官位晉升到[[正二位|正二位]][[內大臣|內大臣]]兼右近衛大將，並得到源氏長者的稱號。不過到寶曆元年六月二十日（1751年7月12日）父親[[大御所|大御所]][[德川吉宗|德川吉宗]]去世前，家重都只是名義上的[[大將軍|大將軍]]而已，父親[[吉宗|吉宗]]還是掌握[[幕府|幕府]]大半的權力。

家重時代因為有吉宗推進的享保改革，初期十分安定，而家重也充實[[德川綱吉|綱吉]]時期的勘定吟味役，進行了幾個獨立的經濟政策。寶曆5年（1755年）因為農作物歉收，造成一揆的暴動，社會不安再一次增加。

同時，健康情況每況愈下的家重言語表答方式也越來越不清晰，不過家重的親信[[大岡忠光|大岡忠光]]卻能聽出自己在說什麼，也因此他十分器重忠光，並再度啟用側用人制度，大岡忠光因而得以濫用權力、實施暴政，[[寶曆|寶曆]]十年（1760年）大岡忠光去世，同年的五月十三日德川家重讓位給與側室於幸之方所生的長子[[德川家治|德川家治]]，改稱大御所。翌年，[[寶曆|寶曆]]十一年六月十二日（1761年7月13日）德川家重病逝於[[江戶城|江戶城]]內，享年四十九歲，安葬於[[增上寺|增上寺]]，追贈[[正一位|正一位]][[太政大臣|太政大臣]]，法號“淳信院”。

== 官職位階履歷 ==
* 享保9年（1724年）成為將軍繼承人。
* 享保10年（1725年）任從二位權大納言。元服，改名家重。
* 寬保元年（1741年）任右近衛大將。
* 延享2年（1745年）正二位內大臣兼右近衛大將。征夷將軍、源氏長者宣下。
* 寶曆11年（1760年）任右大臣。征夷將軍辭任。
* 寶曆12年（1761年）逝世。贈正一位太政大臣。

== 人物 ==
家重與被稱為[[幕府中興|幕府中興]]的父親吉宗相比較，通常都不太被看好。家重時代也沒有幾個有力老中。同時，引退的大御所吉宗，並沒進行幕府的二元政治，反倒是由將軍親自打理政事。

家重喜歡下將棋，也有有關將棋的著作。而家重也因為有尿失禁這種症狀，被難聽的戲稱為「小便公方」。

戰後，隨著增上寺的修建，寺院內的德川將軍墓地也被挖掘出來進行遷移。這個時候也對德川歷代將軍和德川氏相關人物的遺骨進行調查，而下面是從家重的遺骨中找出的事實。

* 家重是同寺所埋葬的歷代將軍中，面孔（五官）最端整的一位。在「德川實記」有一小段記述：家重在各種各樣的儀式謁見大名時，看起來是十分的高尚。 

* 從家重的[[牙齒|牙齒]]在約45度的磨損來看，家重在[[乳齒|乳齒]]長為[[恆齒|恆齒]]之間，一天到晚都咬著牙齒。而家重在許多記錄上是說無法用言語清楚表達，不過這樣看來那應該是家重自己的癖好。 

* 家重的血型是A型。身高156.3公分，比當時男性的平均身高157．1公分略低，比當時女性的平均身高145.6高了10多公分。

* 有實為女性的說法。根據對遺骨的調查，頭蓋骨尤其是骨盆呈現女性的特徵，而且收納方式跟歷代將軍的盤腿姿勢不同，採用當時女性埋葬的正座姿勢。因為身體構造緣故，身為女性的確也會比較頻尿，另外為了掩飾聲音故一切發言都經由側用人大岡忠光傳達。其父吉宗當年為了跟已有長男的吉通競爭八代將軍的繼承，非常需要擁有自己的後繼者，便將出生的長女當成長男養育。而實質上的長男也就是後來的次子宗武，是到了七代將軍家繼過世的那一年才出生的。
