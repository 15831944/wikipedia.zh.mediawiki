'''德兴市'''位于[[中国|中国]][[江西省|江西省]]东北部，是[[上饶市|上饶市]]代管的一个[[县级市|县级市]]。东接浙江省开化县，东南与玉山县、上饶县毗邻，南和横峰县、弋阳县相接，西接乐平市，北连婺源县。德兴富铜等矿产资源，被称为中国的“铜都”。

== 历史 ==
[[东汉|东汉]][[建安|建安]]八年（203年）置县，已有1800多年的治县历史，1990年12月撤县设市。 

== 地理 ==
德兴全境以[[山地|山地]]为主，南部为[[怀玉山|怀玉山]]。西部小片低山[[丘陵|丘陵]]。[[乐安江|乐安江]]为主要河流。

== 资源 ==
德兴拥有丰富的矿产资源，包括铜、金、银、铅锌、煤等。有中国最大的铜矿、亚洲最大的露天铜矿[[德兴铜矿|德兴铜矿]]，德兴铜矿是世界五大[[斑岩铜矿|斑岩铜矿]]之一、亚洲第一大斑岩矿山。

== 区划人口 ==
===行政区划===
德兴市辖3街道5镇6乡和1开发区：
*银城街道、新营街道、香屯街道
*泗洲镇、花桥镇、新岗山镇、海口镇、绕二镇
*黄柏乡、万村乡、张村乡、皈大乡、李宅乡、龙头山乡
*大茅山经济开发区（省级）

===人口===
德兴市2006年人口为31万，其中1/3人口来自浙江（多为[[新安江水库|新安江水库]]移民），1/3来自全国各地（下放[[知青|知青]]及铜矿工作人员）。

== 交通 ==
[[景婺常高速公路|景婺常高速公路]]途经德兴新岗山；[[昌德高速公路|昌德高速公路]]终于德兴与景婺常高速公路对接（已通车）；[[上德高速公路|上德高速公路]]终于德兴与昌德高速对接（已通车）。

[[乐德铁路|乐德铁路]]为德兴铜矿提供运输为货运单线铁路；[[九景衢铁路|九景衢铁路]]途经德兴新岗山（在建）；[[京福高铁|京福高铁]]至北向南穿越德兴（在建）。

== 风景名胜 ==
*[[三清山|三清山]]：位于德兴东南部与[[玉山县|玉山县]]交界处，为道教名山，2008年7月被列入[[世界遗产名录|世界遗产名录]]，是江西省唯一的[[世界自然遗产|世界自然遗产]]。
*[[大茅山|大茅山]]：为[[国家级风景名胜区|国家级风景名胜区]]，有[[梧风洞|梧风洞]]等景点。
