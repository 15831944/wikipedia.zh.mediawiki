'''德島自動車道'''（{{jpn|j=徳島自動車道|hg=とくしまじどうしゃどう|rm=Tokushima jidōshadō|f=y}}）是以[[德島縣|德島縣]][[鳴門市|鳴門市]][[鳴門系統交流道|鳴門系統交流道]]為起點，至[[愛媛縣|愛媛縣]][[四國中央市|四國中央市]][[川之江東系統交流道|川之江東系統交流道]]，總長106.2[[公里|公里]]的[[日本高速公路|高速公路]]。略稱為'''德島道'''（{{lang|ja|徳島道／とくしまどう}}，TOKUSHIMA EXPWY）。

本道路之高速公路路線編號在鳴門系統交流道 - [[德島系統交流道|德島系統交流道]]（建設中）間與[[高松自動車道|高松自動車道]]、[[松山自動車道|松山自動車道]]（[[川之江系統交流道|川之江系統交流道]] - [[松山交流道_(愛媛縣)|松山交流道]]間）共同編為 {{Ja Exp Route Sign|E11}}，德島系統交流道（建設中）- 川之江東系統交流道間與[[高知自動車道|高知自動車道]]（川之江系統交流道 - [[高知交流道|高知交流道]]間）共同編為 {{Ja Exp Route Sign|E32}}<ref>{{Cite web|url=http://www.mlit.go.jp/road/sign/numbering/list/index.html|title=高速道路ナンバリング一覧|accessdate=2017年2月26日|publisher=国土交通省}}</ref>。

== 概要 ==
[[國土開發幹線自動車道|國土開發幹線自動車道]]的路線名在[[鳴門系統交流道|鳴門系統交流道]] - [[德島交流道|德島交流道]]間為[[四國橫斷自動車道|四國橫斷自動車道]]的一部份；在德島交流道 - [[川之江東系統交流道|川之江東系統交流道]]間則是[[四國縱貫自動車道|四國縱貫自動車道]]的一部分。

[[2000年|2000年]][[3月11日|3月11日]]，境目嶺的新境目隧道貫通，[[井川池田交流道|井川池田交流道]] - 川之江東系統交流道間通車，自此連接[[四國|四國]]四縣的縣廳所在地[[X|X]]型路網（通稱為{{link-ja|X Highway|四国8の字ネットワーク}}／{{lang|ja|エックスハイウェイ}}）完成。

幾乎全線都沿著[[吉野川|吉野川]]走行，相對於[[國道192號|國道192號]]沿著吉野川右岸（南側）走行，德島道除了在三好市内，大致沿吉野川左岸（北側）的山區表面走行。

[[土成交流道|土成交流道]] - [[阿波停車區|阿波停車區]]間有6座連續的[[隧道|隧道]]。連續95公里為[[2車道高速公路|暫定2車道]]（未來預定為完成4車道），{{link-ja|道路規格|道路構造令}}與設計速度在鳴門系統交流道 - [[脇町交流道|脇町交流道]]為第1種第2級、100 [[km/h|km/h]]，脇町交流道 - 川之江東系統交流道為第1種第3級、80 km/h。全線除交流道附近外，速限為70 km/h。

== 歷史==
* [[1994年|1994年]]（[[平成|平成]]6年）[[3月17日|3月17日]]：藍住交流道 - 脇町交流道間通車。
* [[1995年|1995年]]（平成7年）[[8月9日|8月9日]]：德島交流道 - 藍住交流道間通車，上板服務區啟用。
* [[1997年|1997年]]（平成9年）[[12月3日|12月3日]]：脇町交流道 - 美馬交流道間通車，阿波停車區啟用。
* [[1999年|1999年]]（平成11年）[[3月30日|3月30日]]：美馬交流道 - 井川池田交流道間通車。
* [[2000年|2000年]]（平成12年）[[3月11日|3月11日]]：井川池田交流道 - 川之江東系統交流道間通車<ref>{{Cite web |date=1999-12-17 |url=http://response.jp/article/1999/12/17/595.html |title=来年3月に高速道路5区間が開通、総延長6615kmに |work=レスポンス |publisher=株式会社イード |accessdate=2018-03-15}}</ref>，同時與高知自動車道連接，完成X型高速公路路網<ref>{{Cite web |url=http://corp.w-nexco.co.jp/corporate/release/hq/h29/0927c/pdfs/04.pdf |title=高知自動車道 これまで果たした役割の振り返り |format=PDF |publisher=西日本高速道路株式会社 |page=1 |accessdate=2018-03-15}}</ref>。
* [[2004年|2004年]]（平成16年）[[10月31日|10月31日]]：吉野川服務區內的吉野川智慧型交流道開始實施社會實驗。
* [[2006年|2006年]]（平成18年）[[10月1日|10月1日]]：吉野川智慧型交流道永久化，正式啟用。
* [[2013年|2013年]]（平成25年）[[6月4日|6月4日]]：廢除德島收費站。自此新設德島交流道收費站啟用。
* [[2015年|2015年]]（平成27年）[[3月14日|3月14日]]：鳴門系統交流道 - 德島交流道間通車<ref>{{Cite web |date=2015-01-27 |url=http://corp.w-nexco.co.jp/corporate/release/hq/h27/0127/ |title=徳島自動車道（鳴門（なると）JCT〜徳島（とくしま）IC間）が平成27年3月14日（土曜）に開通します |publisher=西日本高速道路株式会社 |accessdate=2018-03-15}}</ref>，自此'''全線通車'''。

2010年度的調査時，全線平均交通量為6,864台。交通量最大的区間為藍住交流道 - 脇町交流道間，超過8,000台。與2005年的調査相比較，全線的交通量皆有增加但是不超過1.0，所以全線幾乎沒有交通堵塞的問題。另外，受到暫定2車道的影響，全線速限為70 km/h，但是實際旅行速度在白天的12小時於全区間內皆超過70 km/h，最大來到81 km/h。
