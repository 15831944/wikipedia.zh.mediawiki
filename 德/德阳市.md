'''德阳市'''是[[中华人民共和国|中华人民共和国]][[四川省|四川省]]下辖的[[地级市|地级市]]，位于四川省中部。市境西南邻[[成都市|成都市]]，东南达[[资阳市|资阳市]]、[[遂宁市|遂宁市]]，东北临[[绵阳市|绵阳市]]，西北界[[阿坝州|阿坝州]]。地处[[四川盆地|四川盆地]]西北部，西北部为[[龙门山|龙门山]]区，中南部为[[成都平原|成都平原]]，东部为川中丘陵区。[[绵远河|绵远河]]自西北往东南流经城区，于南部与[[石亭江|石亭江]]、[[鸭子河|鸭子河]]汇合，后段称[[沱江|沱江]]，东南部有[[凯江|凯江]]、[[郪江|郪江]]等[[涪江|涪江]]支流。全市面积5,911平方公里，人口351.32万。德阳是全省重要的工业城市，以重型机械与磷矿工业为核心产业。著名的[[三星堆遗址|三星堆遗址]]于[[广汉市|广汉市]]出土，为[[古蜀国|古蜀国]]的重要遗迹，现为[[全国重点文物保护单位|全国重点文物保护单位]]。

== 历史 ==
[[先秦|先秦]]前为[[古蜀国|古蜀国]]地。[[秦惠文王|秦惠文王]]九年（前316年），[[秦|秦]]灭蜀国建[[蜀郡|蜀郡]]，今境属之。[[西汉|西汉]][[汉高帝|高帝]]六年（前201年）分[[巴郡|巴郡]]、蜀郡置[[广汉郡|广汉郡]]，治雒县乘乡（治今[[旌阳区|旌阳区]][[孝泉镇|孝泉镇]]），属[[益州|益州]]；又置[[雒县|雒县]]（治今[[广汉市|广汉市]][[北外镇|北外镇]]）、[[绵竹县|绵竹县]]（治今旌阳区北）、[[什方县|什方县]]（治今[[什邡市|什邡市]]），皆属广汉郡。[[东汉|东汉]]雒县为广汉郡治，兼为益州治。[[三国|三国]][[蜀汉|蜀汉]]分绵竹县置[[阳泉县|阳泉县]]（治今旌阳区孝泉镇），又置[[五城县|五城县]]（治今[[中江县|中江县]]），皆属广汉郡。[[延熙|延熙]]年间分广汉郡置[[东广汉郡|东广汉郡]]，五城县改属之。

[[东晋|东晋]]置[[万安县|万安县]]（治今[[罗江区|罗江区]]）。[[隆安|隆安]]二年（398年）置[[晋熙县|晋熙县]]（治今[[绵竹市|绵竹市]]），又于县置[[晋熙郡|晋熙郡]]。[[南朝宋|南朝宋]]改五城县为[[伍城县|伍城县]]。[[南朝梁|南朝梁]]万安县更名[[潺亭县|潺亭县]]。[[西魏|西魏]]潺亭县复名万安县，并于县置[[万安郡|万安郡]]。[[北周|北周]]置[[玄武郡|玄武郡]]，治伍城县；省晋熙县、绵竹县。[[北周闵帝|闵帝]]时什方县更名[[方亭县|方亭县]]，[[北周武帝|武帝]]时省方亭县入雒县。

[[隋朝|隋]][[开皇|开皇]]元年（581年）废晋熙郡，复置晋熙县；废广汉郡，雒县属益州；废玄武郡，改伍城县为[[玄武县|玄武县]]，属益州。开皇十八年（598年）晋熙县改名[[孝水县|孝水县]]；雒县改名[[绵竹县|绵竹县]]，徙治今广汉市。[[仁寿|仁寿]]初置[[凯州|凯州]]，玄武县为州治。[[大业|大业]]二年（607年）废凯州；绵竹县复名雒县；孝水县改名[[绵竹县|绵竹县]]，属蜀郡。

[[唐朝|唐]][[武德|武德]]三年（620年）析雒县置'''[[德阳县|德阳县]]'''（治今旌阳区）；复置[[什邡县|什邡县]]，属[[益州|益州]]。[[调露|调露]]元年（679年）析[[郪县|郪县]]、[[飞鸟县|飞鸟县]]置[[铜山县|铜山县]]（治今中江县[[广福镇|广福镇]]），与玄武县同属[[梓州|梓州]]。[[垂拱|垂拱]]二年（686年）于雒县置[[汉州|汉州]]，并划德阳、什邡、绵竹县来属。[[天宝|天宝]]元年（742年）改万安县为[[罗江县|罗江县]]，属[[绵州|绵州]]。[[北宋|北宋]][[大中祥符|大中祥符]]五年（1012年）改玄武县为[[中江县|中江县]]。

[[蒙古|蒙古]][[中统|中统]]元年（1260年）省雒县入汉州。[[元朝|元]][[至元|至元]]八年（1271年）升德阳县置[[德州|德州]]。至元十三年（1276年）德州复改德阳县。至元二十年（1283年）省铜山县入中江县，属[[潼川府|潼川府]]。元末[[明玉珍|明玉珍]][[大夏|大夏]]政权复置雒县。[[明朝|明]][[洪武|洪武]]四年（1371年）复省雒县入汉州。洪武十年（1377年）省什邡县入绵竹县。洪武十三年（1380年）复置什邡县。[[清朝|清]][[顺治|顺治]]十六年（1659年）并罗江县入德阳县。[[雍正|雍正]]五年（1727年）复置罗江县。[[乾隆|乾隆]]三十五年（1770年）省罗江县入绵州，并移绵州治至。[[嘉庆|嘉庆]]六年（1801年）绵州还旧治，复置罗江县。

[[民国|民国]]二年（1913年）裁府州留县，改汉州为[[广汉县|广汉县]]，与德阳、绵竹、罗江、什邡等县划属[[川西道|川西道]]，次年改[[西川道|西川道]]；中江县划属[[川北道|川北道]]，次年改[[嘉陵道|嘉陵道]]。民国十九年（1930年）废道制。民国二十四年（1935年）德阳、广汉、绵竹、罗江、什邡五县属四川省第十三行政督察区，中江县属第十二行政督察区。

[[中华人民共和国|中华人民共和国]]成立后，德阳、广汉、绵竹、罗江、什邡五县属[[川西行署区|川西行署区]][[绵阳专区|绵阳专区]]，中江县属[[川北行署区|川北行署区]][[遂宁专区|遂宁专区]]。1952年各专区由四川省直接领导，广汉、什邡二县改属[[温江专区|温江专区]]。1958年中江县划归绵阳专区。1959年罗江县并入德阳县。1960年什邡县并入广汉县。1962年复置什邡县。1968年各专区改称地区。1983年3月3日，广汉、什邡二县划属[[成都市|成都市]]；8月18日，析德阳县城区街道、[[汉旺镇|汉旺镇]]和旌阳、城区、八角井三个公社，设立地级[[德阳市|德阳市]]，将[[绵阳地区|绵阳地区]]的德阳、中江、绵竹三县和成都市的广汉、什邡两县划归德阳市管辖；9月12日，撤销德阳县并入德阳市，并设立[[市中区_(德阳市)|市中区]]。1988年2月24日，撤销广汉县，设立县级[[广汉市|广汉市]]。1995年10月27日，撤销什邡县，设立县级[[什邡市|什邡市]]。1996年8月3日，撤销德阳市市中区，改设[[旌阳区|旌阳区]]和[[罗江县|罗江县]]；同年10月8日，撤销绵竹县，设立县级[[绵竹市|绵竹市]]。2017年8月，国务院批复同意撤销罗江县，设立罗江区。

== 地理 ==
属于[[长江|长江]]上游[[沱江|沱江]]流域。东北接[[绵阳市|绵阳市]]，东南与[[遂宁|遂宁]]、[[资阳|资阳]]两市交界，西南连[[成都市|成都市]]，西与[[阿坝州|阿坝州]]接壤。距省会成都50余公里，是成都旅游门户圈的重要组成部分。

德阳地形西高东低，西北部为[[山地|山地]]，中部为[[平原|平原]]，东南部为[[丘陵|丘陵]]，属[[亚热带季风气候|亚热带季风气候]]，四季分明，雨量充沛，年降雨量1000毫米左右，平均气温15.7－16.7℃。

=== 行政区划 ===
全市辖2个[[市辖区|市辖区]]、1个[[县_(中华人民共和国)|县]]，代管3个[[县级市|县级市]]，是国家和四川省重点规划的百万人口大城市之一。
* 市辖区：[[旌阳区|旌阳区]]、[[罗江区|罗江区]]
* 县级市：[[广汉市|广汉市]]、[[什邡市|什邡市]]、[[绵竹市|绵竹市]]
* 县：[[中江县|中江县]]

此外，[[德阳经济技术开发区|德阳经济技术开发区]]是德阳市设立的[[国家级经济技术开发区|国家级经济技术开发区]]。

== 经济 ==
德阳是中国重大装备制造业基地、国家首批新型工业化产业示范基地和四川省第二大工业城市。拥有[[中国二重|中国二重]]、东方电机、东方汽轮机、宏华石油等一批国内一流、世界知名的重装制造企业。德阳重大装备制造业集群在中国乃至世界都具有巨大的影响力，全国60%以上的核电产品、40%的水电机组、30%以上的火电机组和汽轮机、50%的大型轧钢设备和大型电站铸锻件、20%的大型船用铸锻件都是由德阳制造装备，发电设备产量连续多年居世界第一，石油钻机出口居全国第一；食品工业享誉中外，拥有中国名酒剑南春、长城雪茄、冰川时代矿泉水等一批优质品牌，建成了亚洲最大的雪茄烟生产基地；化学工业以磷化工、氯碱化工、钛化工、天然气化工为主，是全国重要的磷化工基地和化肥生产基地；德阳着力培育新能源装备制造战略性新兴产业，大力发展以核电、风力发电、太阳能、潮汐发电、生物能、燃料电池等为重点的新能源装备制造业，被联合国列为清洁技术与新能源装备制造业国际示范城市；新材料、医药等新兴产业快速发展，是国家新材料产业化基地和中药现代化生产基地。德阳经济技术开发区为国家级经济技术开发区。雄厚的工业基础，奠定了德阳在中国西部重要工业城市的地位。
*机械制造业有：三大重装制造企业[[中国第二重型机械集团公司|中国第二重型机械集团公司]]（控制上市公司二重重装（SHA：601268））、[[东方电机厂|东方电机厂]]和[[东方汽轮机厂|东方汽轮机厂]]；四川宏华石油设备有限公司、四川蓝星机械有限公司、科新机电（SHE：300092）、广汉广达机械设备有限公司、德阳迪泰机械有限公司等。
*石油、钻机企业:[[四川宏华石油设备有限公司|四川宏华石油设备有限公司]]（HK：0196）
*化工工业重要企业有：[[金路集团|金路集团]]（SHE：000510）、[[宏达股份|宏达股份]]（SHA：600331）、龙蟒集团、川恒股份等。
*食品工业重要企业有：[[剑南春|剑南春]]、华润蓝剑啤酒、什邡卷烟厂。
*建材及其它工业重要企业有：四川森普管材股份有限公司、特变电工（德阳）电缆股份有限公司、绵竹市盘龙化建有限责任公司等。

德阳经济发展较快，年产值500万元以上工业企业接近1000家、资产总额达1022亿，2009年，全市实现地区生产总值780.00亿元，同比增长14.4%，固定资产投资746.86亿元，同比增长241.2%，实现社会品零售总额245.8亿元，同比增长22.6%，财政总收入129.64亿元，同比增长32.04%，地方财政收入64.99亿元，其中一般预算收入31.88亿元，城镇居民人均可支配收入达到16349元，[[农村居民人均纯收入|农村居民人均纯收入]]达到5625元，四川省200强乡镇中，德阳占51个。 

德阳经济的特征是县域经济比较发达，其下辖的旌阳区、绵竹市、什邡市、广汉市以前均为四川省经济综合实力“十强县”，因受[[汶川大地震|特大地震]]影响，2008年度绵竹和什邡的经济总量负增长30%以上，连同广汉市均未能进入当年四川十强县。德阳一区五县在四川省2008年县级经济中的排位分别是旌阳（8）、广汉（13）、什邡（19）、绵竹（34）、中江（56）、罗江（95）。随着灾后重建的进行，德阳经济加速复苏，绵竹，什邡，广汉有望在2011年全部重返四川十强县之列。前身为广汉市工业开发小区的德阳高新技术产业园区2015年升级为国家高新技术产业开发区，在2014年高新技术产业占44.5%。

== 教育 ==
有各类学校663所，在校学生52.81万人，专任教师26647人。其中：普通高校7所([[中国民用航空飞行学院|中国民用航空飞行学院]]等)，在校普通本（专）科学生5.1万人，增30.10%；普通中学211所，在校生18.79万人；小学405所，在校生24.08万人。学龄儿童入学率99.15%。
