'''德川家茂'''（{{bd|1846年|7月17日|1866年|8月29日|catIdx=Tokugawa, Iemochi}}）是[[德川幕府|德川幕府]]第14代[[征夷大將軍|征夷大將軍]]。第13代將軍[[德川家定|德川家定]]的[[堂弟|堂弟]]。就任將軍前是[[德川御三家|德川御三家]][[纪州藩|纪州藩]]第13代[[藩主|藩主]]。原名'''慶福'''（{{lang|ja|よしとみ}}）。

德川家茂是[[德川齊顺|德川齊顺]]（[[清水德川家|清水德川家]]同時也是[[纪州德川家|纪州德川家]]的當主）的長男、[[德川齊順|德川齊順]]在家茂出生前就去世。德川家茂的祖父是[[德川家齊|德川家齊]]、祖母是[[妙操院|妙操院]]。[[御台所|御台所]]是[[孝明天皇|孝明天皇]]的妹妹[[和宮親子內親王|和宮親子內親王]]（静寬院）。官方说法未娶[[側室|側室]]（然而據說和其他女性有過關係、根据對家茂墓所的調查结果，也证明这种说法）。家茂4岁就成为纪州藩主、是德川家家谱中與没有子嗣的第13代將軍[[德川家定|德川家定]]血统最近的人。<ref>当时家茂叔父[[松平齊民|松平齊民]]、[[蜂須賀齊裕|蜂須賀齊裕]]仍然活着，但是由于他们已经离开德川家。具有讽刺意味的是这两人比家茂活得更长。</ref>家茂在[[井伊直弼|井伊直弼]]等[[南纪派|南纪派]]的支持下13岁就成为第14代將军。不光是血统上，其英明果断的办事风格也得到[[胜海舟|胜海舟]]等阁僚重臣的深厚信赖。

== 生涯 ==
[[弘化|弘化]]3年（1846年）闰5月24日、作为长男出生于[[江戶|江戶]]的纪州藩邸（现：[[东京都|东京都]][[港区|港区]]）。父亲是[[纪州藩|纪州藩]]第11代藩主[[德川齊順|德川齊順]]。[[幼名|幼名]]'''菊千代'''。[[嘉永|嘉永]]2年（1849年）家茂叔父第12代藩主[[德川齊彊|德川齊彊]]去世、以[[养子|养子]]身份继任[[家督|家督]]成为[[纪州藩|纪州藩]]第13代藩主。

[[安政|安政]]5年（1858年）、同[[一桥派|一桥派]]在后继之争中取胜、第13代將军[[德川家定|德川家定]]死后继任为第14代將军。家茂當时只有13岁、以第11代將军[[德川家齊|德川家齊]]孙子的身份压倒德川庆喜继任將军。直到[[文久|文久]]2年（1862年）一直受到[[德川庆赖|田安庆赖]]<ref>但是庆赖自己也是[[大老|大老]]・[[井伊直弼|井伊直弼]]的[[傀儡|傀儡]]</ref>、[[德川庆喜|一桥庆喜]]<ref>庆喜的伯父[[德川齊脩|德川齊脩]]的正室是家茂的父亲德川齊顺同父异母姐姐峯姫。</ref>等人监视和制约。

文久2年（1862年）作为[[公武合体|公武合体]]政策的一环和[[和宮亲子内亲王|和宮]]结婚。虽然是政略结婚、对和宮非常殷切，頻頻送禮、2人關係相當良好。被认为德川家历代將军中和正室关系最为和谐。

文久3年（1863年）相隔229年后以將军身份[[上洛|上洛]]（上一位是三代將軍[[德川家光|德川家光]]）、向妻兄[[孝明天皇|孝明天皇]]立下[[尊皇攘夷|尊皇攘夷]]的誓言。[[庆应|庆应]]元年（1865年）、[[朝廷|朝廷]]处罚了作出[[兵库|兵库]]开港決定的[[老中|老中]][[阿部正外|阿部正外]]等人、家茂以辞去將军职位作为要挟。孝明天皇十分惊慌马上撤销处分决定、并约定此后不再干预幕府人事。

庆应2年（1866年）、家茂在[[第2次长州征伐|第2次长州征伐]]途中在[[大阪|大阪]]病倒。孝明天皇得知后派[[典药寮|典药寮]]医师[[高阶经由|高阶经由]]和[[福井贞宪|福井登]]前往治疗。[[江户|江户]]的[[天璋院|天璋院]]和[[和宮亲子内亲王|和宮]]也急忙分别派遣侍医[[大膳亮弘玄院|大膳亮弘玄院]]、多纪养春院（[[多紀安琢|多紀安琢]]）、[[远田澄庵|远田澄庵]]、[[高岛祐庵|高岛祐庵]]、[[浅田宗伯|浅田宗伯]]等人前往。然而徒劳无功、家茂于同年7月20日在大坂城病逝。享年僅21岁，之後下葬於[[增上寺|三緣山廣度院增上寺]]（[[東京都|東京都]][[港區_(東京都)|港區]]），戒名[[昭德院|昭德院]]殿光蓮社澤譽道雅大居士。

[[昭和|昭和]]33年（1958年）到35年（1960年）[[増上寺|増上寺]]德川將軍家墓地改葬时，根据对德川家遗骨进行过调查的[[铃木尚|铃木尚]]所写《[[骨は語る_徳川將軍・大名家の人びと|骨は語る 徳川將軍・大名家の人びと]](遗骨所叙说的將军一家)》记载、家茂去世时没有剃发。家茂有很厉害的[[龋齿|龋齿]]、残存的31颗牙齿中30颗是蛀牙。和家茂牙齿不好还非常喜欢吃甜食的历史纪录吻合。有人认为龋齿造成体力虚弱、[[脚气|脚气]]攻心、并且医师之间各执一词怠誤治疗（高阶等中医认为是脚气、西医的幕府医师认为是风湿）是夺去家茂生命的原因。

墓地改葬的时候、在和宮墓中發現疑似家茂的男性肖像照。由於当时资料中並無任何記載家茂像妻子和宮的兄長孝明天皇一樣有拍過照的記錄，推测是去世前在大阪拍摄后送给[[江户|江户]]的和宮。发现后的第二天这张照片似乎因日照而失去影像、只留下玻璃板。也有說照片裡的男性不是家茂而是和宮最初婚约人[[有栖川宫炽仁親王|有栖川宫炽仁親王]]。

同時棺木中還發現了封好的女性長髮，髮質同和宮的完全不同，由於棺木是在大阪就已經封好，此時和宮一直在江戶，所以推測頭髮是和家茂一起的女性所有，家茂應該有隨行的侍妾（日本武士的側室與妾是不同的，側室的地位較高）。

== 评价 ==
家茂公只有21年的短暫人生，但是幕府末期得到了很高的评价。[[胜海舟|胜海舟]]曾经评价说：「{{lang|ja|'''若さゆえに時代に翻弄されたが、もう少し長く生きていれば、英邁な君主として名を残したかもしれない。武勇にも優れていた人物であった'''}}（如此年轻且浮沉於动乱年代，如果他能长寿一些，或許能以明君之名流芳于世。是一位武勇杰出的人物)」。

在幕府家臣之间有威望，尤其对家臣和女性以及动物有极其温情的一面，同时性格刚毅，明治以后也为旧幕府的家臣们所称颂。

== 人物 ==
* 喜欢[[羊羹|羊羹]]、[[冰糖|冰糖]]、[[金平糖|金平糖]]、[[卡斯提拉|卡斯提拉]]等甜食。
* 从四肢骨推算身高156.6cm左右。
* 从遗骨看脸很长鼻子高、反牙。肖像画也表现了这些特点。
* 庆喜来探望时、很平静地和他对话，并没有因为他是將军继任的竞争者而有不自然。
* 根据《[[増上寺徳川將軍家墓とその遺品・遺体|増上寺徳川將軍家墓とその遺品・遺体]]》、家茂[[血型|血型]]是A型。

== 逸事 ==
* 少年时喜欢养鱼养鸟玩耍。13岁继任將军元服后抛却这些爱好，专心致志努力修学文武两道，立志当一个有德有能的將军。当时的家臣在他生前死后都十分感叹。
* 幕臣[[户川安清|户川播磨守安清]]是书法名人。年过70仍然被推举为家茂书法先生。有一次在授课中、家茂突然將沾了墨水的手擦在安清头上笑道「明天再学吧」随即走出走出书房。同席的侍从都唏嘘家茂举止异常、安清却哭了起来。侍从问他是不是因为家茂的玩笑太过分，安清说原来是自己年老小便失禁。按当时惯例在给將军上课时发生类似情况会被严惩，家茂觉察到安清的失态故意有此举动使安清規卻责任。（安清的亲戚户川残花「幕末小史」中所写）。
* [[文久|文久]]3年（1863年）4月、家茂奉朝廷之命实行攘夷政策、乘坐幕府[[军舰|军舰]]顺动丸去大阪视察。指挥顺动丸的[[胜海舟|胜海舟]]说明军舰的性能时家茂表现出非常强的领悟能力。胜海舟顺势提出培养海军人材的问题、家茂当场下令设置[[神户海军操练所|神户海军操练所]]。同年12月上洛时、采用胜海舟的进言乘坐顺动丸（也有说是因为上次上洛去的时候陆路花了22天，返回江户乘顺动丸只用了3天）。海中大浪晕船人很多的时候、侧近建议走陆路的时候家茂回答「海上的事情就交给海军」、表现出对[[胜海舟|胜海舟]]极大的信任。[[胜海舟|胜海舟]]也因此对家茂感激不尽忠心耿耿。家茂死后胜海舟近乎绝望、在日记里写下「'''德川家、今日滅ぶ'''(德川家，今天灭亡了)」。此后胜海舟为了保护德川家的存续、为了自己不喜欢的[[德川庆喜|德川庆喜]]而奔走效命。晩年谈起家茂、以病弱的躯体担负时代变革的重责时总是含着泪说「お気の毒の人なりし(对不起他啊)」。
* 欧洲的[[法国|法国]]、[[意大利|意大利]]是[[绢|绢]]的产地、1850年由于原生生物的原因流行[[蚕|蚕]]的传染病、两国的养蚕业几乎毁灭。家茂收集了蚕卵送给[[拿破仑三世|拿破仑三世]]。法国研究了日本蚕之后、改良了品种。拿破仑三世在文久3年（1863年）送了26匹[[阿拉伯马|阿拉伯马]]作为回礼，因為拿破仑三世听说日本幕府末期的动乱，需要好马改良。但是当时幕府首脑没有理解拿破仑三世的意图，把马分给了幕臣和大名。
