'''德川秀忠'''（{{bd|1579年|5月2日|1632年|3月14日|Tokugawa, Hidetada}}），[[日本|日本]][[江戶幕府|江戶幕府]]的第二代將軍（1605年－1632年），1616年至1632年實際掌權。是初代將軍[[德川家康|德川家康]]三子，母親為側室[[西鄉局|西鄉局]]，幼名長丸。因為大哥[[松平信康|松平信康]]早年1579年被迫切腹自盡，二哥[[結城秀康|結城秀康]]又於1584年的[[小牧長久手之戰|小牧長久手之戰]]後，作為父親德川家康與羽柴秀吉達成議和的條件之一，以養子的身分（實際上是人質）交給秀吉，於是身為三男的秀忠便成為德川家的後繼者。

==生平==
[[天正|天正]]七年四月七日（1579年5月2日）德川秀忠出生於[[遠江國|遠江國]][[濱松城|濱松城]]（今[[靜岡縣|靜岡縣]][[濱松市|濱松市]]），是[[江戶幕府|江戶幕府]]初代[[將軍|將軍]][[德川家康|德川家康]]三子，母親是側室[[西鄉局|西鄉局]]<ref>{{cite web |title=德川秀忠：江户幕府第二代征夷大将军 |url=http://sh.qihoo.com/pc/90fa12de28cb5d6fa?cota=1&sign=360_e39369d1 |publisher=娱乐快乐播报 |accessdate=2018-10-16}}</ref>。
秀忠曾經和織田信雄的長女小姬訂親，甚至有在天正十八年（1590年）結婚之說，若只有婚約，在家康和信雄反目後就取消婚約。

[[文祿|文祿]]四年(1595年)，經由[[豐臣秀吉|豐臣秀吉]]安排，娶[[淺井長政|淺井長政]]三女[[阿江|阿江]]為妻。1600年參與[[關原之戰|關原之戰]]，卻因為在上田城被真田軍絆住，耽誤了參與主戰的時間，引起德川家康大怒。德川家康在1603年創立幕府後，為了幫毫無戰功與威名的秀忠鋪路，便在1605年將位置傳給秀忠，自稱「[[大御所|大御所]]」並移居駿府館，但是實權還是掌握在家康手上，到家康死前秀忠都只是名義上的大將軍而已。[[大坂之戰|大坂之戰]]時與家康一同出戰，救出女兒[[德川千姬|千姬]]，滅豐臣家。大坂之戰前，據說極力主張對豐臣家用兵，以洗刷自己在[[第二次上田城之戰|第二次上田城之戰]]的恥辱。在消滅豐臣家以後，秀忠致力推行「[[武家諸法度|武家諸法度]]」、「[[禁中並公家諸法度|禁中並公家諸法度]]」等制度：一般公認秀忠的政治能力遠勝於其軍事上的才能。

[[元和|元和]]二年(1616年)，父親[[德川家康|德川家康]]過世，享壽七十三歲。他正式掌握天下大權。為了有力統御大名，改易了[[福島正則|福島正則]]等外樣大名，並建立御三家，同時嚴格對待朝廷，將五女[[德川和子|德川和子]]嫁給[[後水尾天皇|後水尾天皇]]。為政風格上，起用了酒井忠世、土井利勝等人擔任幕府中樞的老中，發揮自己的領導風格。另一方面對家臣、親屬，甚至朝廷管制都相當嚴格，對外貿易僅限於[[平戶|平戶]]、[[長崎|長崎]]兩地。

[[元和|元和]]九年三月五日(1623年)，德川秀忠把將軍一職讓給長子[[德川家光|德川家光]]，自己效法父親家康成為大御所退後掌握實權，在江戶城進行二元政治<ref>{{cite web |author1=現代快報 |title=幕府第三代將軍德川家光靠奶媽上位 逼死親弟弟 |url=http://culture.people.com.cn/BIG5/n/2014/1011/c22219-25811515.html |publisher=人民網 |accessdate=2014-10-11}}</ref>。最初是考慮隱居小田原城遙控政務，但最後是居住在江戶城的西之丸(現在的皇宮所在)。

1631年，因次子[[德川忠長|德川忠長]]多有亂行，命他隱居。從此之後身體健康狀況日益不佳，[[寬永|寬永]]九年一月二十四日（1632年3月14日）德川秀忠病逝於[[江戶城|江戶城]][[西之丸殿|西之丸殿]]內，享年五十三歲，安葬於[[東京|東京]][[增上寺|增上寺]]
。法號台德院，追贈[[正一位|正一位]]。

== 特徵與評價 ==
根據[[德川實紀|德川實紀]]所示，排行老三的秀忠比起其他兄弟來說是唯一個性溫厚、具有文人氣息的子嗣。大哥[[松平信康|松平信康]]生前就被認為是將才而深受父親家康喜愛，且接受其岳父[[織田信長|織田信長]]的偏諱「'''信'''」字，代表對大哥的認可，二哥[[結城秀康|結城秀康]]是曾被[[羽柴秀吉|羽柴秀吉]]稱讚其武勇過人的猛將，接受其養父秀吉的偏諱「'''秀'''」字，而四弟[[松平忠吉|松平忠吉]]與其岳父德川四天王的[[井伊直政|井伊直政]]則在關原之戰迎戰使用[[捨奸|捨奸]]戰法撤退的島津軍並擊殺了[[島津豐久|島津豐久]]，相對之下秀忠的軍事能力是較差的。

[[關原之戰|關原之戰]]中率領德川軍主力的秀忠強攻[[上田城|上田城]]，被[[真田昌幸|真田昌幸]]成功地拖延步伐，戰略上造成了東軍的重大損失，而在[[大坂之役|大坂之役]]中的[[天王寺岡山之戰|天王寺岡山之戰]]，其部隊更是由於秀忠的指揮不當而造成了很大的混亂。然而根據遺骨推定，秀忠身高約159公分，身上有多處的子彈傷痕，推測作戰時可能都身居第一線，所受傷都深入骨頭。

其高度的政治手腕，正是家康指定為繼承人的一大原因。[[海音寺潮五郎|海音寺潮五郎]]曾經如此形容[[德川三代|德川三代]]：「家康是所有事自己決定，秀忠是一半自己決定，家光則全部仰賴重臣。」

近代認為秀忠是一個[[懼內|懼內]]的人，不過也有說他並非如此。秀忠在1611年曾經跟侍女靜有私生子幸松(保科正之)，但近代武家社會由於要維持正室的面子以及後宮的秩序，當將軍要納妾時，必須有正室的許可，包括侍女也需要。靜沒有收到阿江的許可，因此幸松為了活命，只好過繼給保科家當養子，且秀忠一生並未承認這個兒子。
