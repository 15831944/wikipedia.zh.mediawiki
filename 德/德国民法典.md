《'''德国民法典'''》（{{lang-de|BGB, Bürgerliches Gesetzbuch}}）是[[德意志帝国|德意志帝国]]从1881年开始研究编纂，于1896年8月公布，1900年1月1日施行的[[民法|民法]]法典，為[[歐陆法系|歐陆法系]]中最重要的民法典之一。歷經德意志帝國時期、威瑪共和、納粹統治、兩德分裂至統一後，仍然施行至今，其內容及形式所樹立之風格，不但成為大陸法系之代表性體例之一，也影響日本、中國、韓國等民法典之制定。

== 法典体系 ==
[[Image:Reichsgesetzblatt_1896_Seite_195.png|thumb]]
法典参考了[[罗马法|罗马法]]《[[国法大全|国法大全]]》中的《[[学说汇纂|学说汇纂]]》的编纂体系，采用五编制，分别为总则（{{lang|de|Allgemeiner Teil}}）、[[债务关系法|债务关系法]]（{{lang|de|Recht der Schuldverhältnisse}}）、[[物权法|物权法]]（{{lang|de|Sachenrecht}}）、[[家族法|家族法]](或譯[[親屬法|親屬法]])（{{lang|de|Familienrecht}}）和[[继承法|继承法]]（{{lang|de|Erbrecht}}）。

内容既借鉴了罗马传统法和《[[法国民法典|法国民法典]]》的立法经验，同时又有很大创新。

== 与《法国民法典》的差异 ==
该法典不同于《[[法国民法典|法国民法典]]》之处：
#法国的使用三编制，德国的使用五编制；
#德国的总则篇独立成一篇，规定了民事主体、法律行为和时效等具有总论性质的内容；
#德国的将规定债权关系的条文移至物权法之前，反映了新型的[[债权法|债权法]]理念。
#德国的详细规定了[[法人|法人]]制度，来规范[[公司|公司]]以适应社会发展的需要。

== 影响国家 ==
《[[日本民法典|日本民法典]]》，《[[中华民国民法|中华民国民法]]》，中国大陆，泰国，韩国，希腊和乌克兰等国的立法都深受影响。
