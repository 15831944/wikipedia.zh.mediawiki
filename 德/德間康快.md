'''德間康快'''（{{bd|1921年|10月25日|2000年|9月20日}}），日本企業家、電影製作人。

曾任[[德間書店|德間書店]]社長（第1任）、[[大映|大映]]社長、學校法人逗子開成學園理事長（第5任）、逗子開成高等學校校長（第12任）、[[吉卜力工作室|吉卜力工作室]]社長（第1任、第3任）、[[東京都寫真美術館|東京都寫真美術館]]館長（第3任）。

== 經歷 ==
出生於[[神奈川縣|神奈川縣]][[橫須賀市|橫須賀市]]。經歷[[逗子開成中學校|逗子開成中學校]]、[[早稻田大學|早稻田大學]]商學部畢業。<ref name="Nikkei">{{Cite web |url=http://www.nikkeibp.co.jp/archives/113/113423.html |script-title=ja:徳間康快氏逝って映画界にささやかれる二つの不安 |work=Nikkei BP |date=2000年10月3日 |accessdate=2013年2月23日 |deadurl=yes |archiveurl=https://web.archive.org/web/20100407235047/http://www.nikkeibp.co.jp/archives/113/113423.html |archivedate=2010年4月7日 }}</ref>

青年時曾為[[日本共產黨|日本共產黨]]黨員。1943年進入讀賣新聞社（現今的[[讀賣新聞|讀賣新聞]]東京本社），<ref name="Nikkei" />1946年退社。之後在1954年3月19日成立[[德間書店|德間書店]]，就任第一任社長。<ref>{{Cite web |url=http://www.tokuma.jp/company/profile.html |script-title=ja:会社概要 |work=[[德間書店|德間書店]] | accessdate=2013年2月23日 }}</ref>

2000年9月20日，因[[肝病|肝病]]離世。<ref name="asahi" />

== 事業 ==
=== 電影事業 ===
{{main|大映}}
{{main|吉卜力工作室}}
德間康快於1974年以2億日圓收購當時瀕臨破產的[[大映|大映]]電影公司，之後將大映以{{nihongo|「大映電影株式會社」|大映映画株式会社}}新名稱、成為德間書店底下專門拍攝電影的子公司。<ref>{{Cite web |url=http://www.daiei70th.jp/history.html |script-title=ja:大映の歴史 |work=大映 |accessdate=2013年2月23日 |deadurl=yes |archiveurl=https://web.archive.org/web/20130106102326/http://www.daiei70th.jp/history.html |archivedate=2013年1月6日 }}</ref><ref>{{cite book |script-title=ja:日本映画発達史5 映像時代の到来 |publisher=中央公論社 |author=田中純一郎 |id=B000J7X38A |page=299頁 |language=ja |date=1980年6月 |accessdate=2013年2月23日 }}</ref>

在動畫事業方面，德間則在1985年協助當時拍攝完《[[風之谷|風之谷]]》的[[宮崎駿|宮崎駿]]、[[高畑勳|高畑勳]]等人成立了「[[吉卜力工作室|吉卜力工作室]]」。<ref>{{Cite web |url=http://www.ghibli.jp/30profile/000152.html |script-title=ja:スタジオジブリの歴史 |work=[[吉卜力工作室|吉卜力工作室]] | accessdate=2013年2月23日 }}</ref>

=== 教育事業 ===
{{main|逗子開成中等學校}}
{{main|逗子開成高等學校}}
1984年2月，母校逗子開成學園聘請德間康快擔任學校的理事。因先前在1980年發生逗子開成學園的職員前往[[長野縣|長野縣]][[白馬村|白馬村]]一帶的八方尾根登山發生死亡事故，當時校方以此為教員私人旅行的理由拒絕向罹難家屬賠償，而德間上任時則協助了校方與家屬的和解。<ref>{{Cite web |url=http://www.zushi-kaisei.ac.jp/about/history.html |script-title=ja:110年の歩み |work=逗子開成學園 | accessdate=2013年2月23日 }}</ref>

=== 文化事業 ===
{{main|東京都寫真美術館}}
2000年4月1日德間康快在當時的東京都知事[[石原慎太郎|石原慎太郎]]的邀請下，擔任東京都寫真美術館的館長。在德間於當年逝世後則改由[[資生堂|資生堂]]公司的會長福原義春擔任。<ref>{{Cite web |url=http://www.cinema-st.com/mini/m023.html |script-title=ja:東京都写真美術館 |work=東京都寫真美術館 | accessdate=2013年2月23日 }}</ref>
