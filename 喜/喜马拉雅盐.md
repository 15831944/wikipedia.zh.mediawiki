'''喜马拉雅盐'''，又名'''玫瑰盐'''，是一种产自[[巴基斯坦|巴基斯坦]]东部[[旁遮普省|旁遮普省]]{{link-en|克乌拉盐矿|Khewra Salt Mine|克乌拉盐矿}}的[[石盐|石盐]]。没有科学证据能证明喜马拉雅盐相比普通[[食盐|食盐]]对人体有更多的好处，因此任何声称喜马拉雅盐对人体有特殊益处的说法应被视为[[伪科学|伪科学]]<ref name="DAJSD">{{cite web |title=David Avocado’s Himalayan Salt Debunked |url=https://badscidebunked.wordpress.com/2016/01/18/your-worst-day-ever-david-avocados-himalayan-salt-debunked/ |date=January 18, 2016 |website=Bad Science Debunked |accessdate=July 20, 2017 |archiveurl=https://web.archive.org/web/20170721032712/https://badscidebunked.wordpress.com/2016/01/18/your-worst-day-ever-david-avocados-himalayan-salt-debunked/ |archivedate=July 20, 2017}}</ref><ref name="DIHAHB">{{cite web |title=Pink Himalayan Salt: Does It Have Any Health Benefits? |url=http://www.medicalnewstoday.com/articles/315081.php |website=Medical News Today |accessdate=10 June 2017 |archiveurl=https://web.archive.org/web/20170610052359/http://www.medicalnewstoday.com/articles/315081.php |archivedate=10 June 2017 |date=10 June 2017}}</ref><ref name="IIAS">{{cite web |url=http://www.lisabarger.com/is-it-a-scam/salt-lamps.html |title=Salt Lamps - Is it a Scam? |author=Lisa Berger |work=Today in Alternative Medicine |accessdate=2012-10-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20120414221520/http://www.lisabarger.com/is-it-a-scam/salt-lamps.html |archivedate=2012-04-14}}</ref><ref name="Weil">{{cite web |title=Are Himalayan Salt Lamps Worthwhile? |url=https://www.drweil.com/health-wellness/balanced-living/healthy-home/are-himalayan-salt-lamps-worthwhile/ |accessdate=July 20, 2017 |author=Dr Andrew Weil |archiveurl=https://web.archive.org/web/20170721034408/https://www.drweil.com/health-wellness/balanced-living/healthy-home/are-himalayan-salt-lamps-worthwhile/ |archivedate=July 20, 2017}}</ref>。

==矿物成分==
喜马拉雅盐含有95-98%的[[氯化钠|氯化钠]]、2–4%{{link-en|杂卤石|Polyhalite|杂卤石}}（含有钾、钙、镁、硫、氧、氢）、0.01%的[[氟化物|氟化物]]。

喜马拉雅盐通常是[[白色|白色]]至透明的颜色，而微量的[[铁|铁]]杂质使它呈[[粉红色|粉红色]]或[[红色|红色]]<ref>[http://www.pmdc.gov.pk/?p=KhewraSaltMines Khewra Salt Mines; Pakistan Mineral Development Corporation.]</ref><ref name="Freeman">{{cite web |url=http://science.howstuffworks.com/innovation/edible-innovations/salt2.htm |title=How Salt Works |last1=Freeman |first1=Shanna |publisher=[[HowStuffWorks|HowStuffWorks]] |accessdate=Oct 20, 2014 |archiveurl=https://web.archive.org/web/20170721024605/http://science.howstuffworks.com/innovation/edible-innovations/salt2.htm |archivedate=July 20, 2017}}</ref>，因此被商家命名为玫瑰盐<ref>[http://www.guokr.com/article/439809/]，玫瑰盐：与“健康营养”无关。</ref>。

==用途==
*调味：喜马拉雅盐可在日常生活中代替普通食盐使用。关于喜马拉雅山盐的摄入有很多说法，但是没有科学证据证明它比普通食盐更有益于健康<ref name="DAJSD">{{cite web |title=David Avocado’s Himalayan Salt Debunked |url=https://badscidebunked.wordpress.com/2016/01/18/your-worst-day-ever-david-avocados-himalayan-salt-debunked/ |date=January 18, 2016 |website=Bad Science Debunked |accessdate=July 20, 2017 |archiveurl=https://web.archive.org/web/20170721032712/https://badscidebunked.wordpress.com/2016/01/18/your-worst-day-ever-david-avocados-himalayan-salt-debunked/ |archivedate=July 20, 2017}}</ref><ref name="DIHAHB">{{cite web |title=Pink Himalayan Salt: Does It Have Any Health Benefits? |url=http://www.medicalnewstoday.com/articles/315081.php |website=Medical News Today |accessdate=10 June 2017 |archiveurl=https://web.archive.org/web/20170610052359/http://www.medicalnewstoday.com/articles/315081.php |archivedate=10 June 2017 |date=10 June 2017}}</ref><ref>[http://jiankang.163.com/15/0912/14/B3APN44R0038002S.html]，玫瑰盐真的比普通食盐有益？。</ref>。
*烹饪：大块的盐也可以用于[[盐焗|盐焗]]做法的菜。
*照明：采用一整块的大盐块掏空内部放入[[灯泡|灯泡]]做成的盐灯可用于照明，没有科学证据证明盐灯除了照明及美觀功能之外有其他功效<ref name="IIAS">{{cite web |url=http://www.lisabarger.com/is-it-a-scam/salt-lamps.html |title=Salt Lamps - Is it a Scam? |author=Lisa Berger |work=Today in Alternative Medicine |accessdate=2012-10-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20120414221520/http://www.lisabarger.com/is-it-a-scam/salt-lamps.html |archivedate=2012-04-14}}</ref><ref name="Weil">{{cite web |title=Are Himalayan Salt Lamps Worthwhile? |url=https://www.drweil.com/health-wellness/balanced-living/healthy-home/are-himalayan-salt-lamps-worthwhile/ |accessdate=July 20, 2017 |author=Dr Andrew Weil |archiveurl=https://web.archive.org/web/20170721034408/https://www.drweil.com/health-wellness/balanced-living/healthy-home/are-himalayan-salt-lamps-worthwhile/ |archivedate=July 20, 2017}}</ref>。
