{{日本作品|喜歡與喜歡的三角戀|スキとスキとでサンカク恋愛}}是[[HOOKSOFT|ASa Project]]在2016年11月25日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web|url=https://www.game-style.jp/special/201610/11/00san_ap.php|title=Game-Style「喜歡與喜歡的三角戀」特集|website=Game-Style|date=2016-10-11|accessdate=2018-05-12|language=ja}}</ref>。2019年1月24日由Entergram發售[[PlayStation_4|PlayStation 4]]與[[PlayStation_Vita|PlayStation Vita]]版本<ref>{{cite web|url=https://www.4gamer.net/games/437/G043726/20181011123/|title=「スキとスキとでサンカク恋愛」の公式サイトが本日オープン。完全生産限定版や店舗特典の情報も公開|website=4Gamer.net|date=2018-10-11|accessdate=2019-01-24|language=ja}}</ref>。

== 故事簡介 ==
小森江宗介是御宅族社的成員，並且每天跟妹妹小森江七瑠及青梅竹馬鳴瀧真帆過著愉快的學校生活。某一天，宗介被對他一見鍾情的學姐木須志衣菜告白，而小時候就跟宗介分開的親妹妹小森江鈴也回到了家裡。這兩件事情讓宗介的日常生活有了重大變化<ref>{{Cite web|url=https://www.game-style.jp/soft/201610/03/01san_ap.php|title=“二つの三角関係”に翻弄される、カオスでオタクなドタバタ青春ラブコメディ!? 『スキとスキとでサンカク恋愛』11月25日発売！|publisher=Game-Style|date=2016-10-03|accessdate=2018-05-12|language=ja}}</ref>。

== 登場角色 ==
* 聲優為PC版／PS4、PSV版<ref>{{Cite web|url=https://www.4gamer.net/games/437/G043726/20181018101/|title=「スキとスキとでサンカク恋愛」，登場キャラクターのキャストが公開に|publisher=4Gamer.net|date=2018-10-18|accessdate=2019-01-24|language=ja}}</ref>
=== 主要角色 ===
; {{nihongo|小森江宗介|小森江 宗介（こもりえ そうすけ）}}
: 作品男主角，陽和學園二年級生，在學校加入御宅族社。
; {{nihongo|小森江七瑠|小森江 七瑠（こもりえ ななる）|聲：五十鈴みゆう／[[清水愛|清水愛]]}}
: 宗介沒有血緣關係的妹妹，陽和學園一年級生，御宅族社成員。個性天真活潑，喜歡有趣的事物。
; {{nihongo|小森江鈴|小森江 すず（こもりえ すず）|聲：鹿野まなか／熊乃可菜}}
: 宗介的親妹妹，年幼時跟著宗介的生母一起搬到別的地方，後來被接回小森江家居住，並轉入陽和學園一年級。性格溫厚。
; {{nihongo|鳴瀧真帆|鳴滝 真帆（なるたき まほ）|聲：[[卯衣|卯衣]]／[[大橋步夕|大橋步夕]]}}
: 陽和學園二年級生，御宅族社成員。從幼稚園便和宗介一起玩耍的青梅竹馬，以前曾拒絕過宗介的告白。喜歡格鬥電玩。
; {{nihongo|木須志衣菜|木須 志衣菜（きす しいな）|聲：北見六花／[[小野涼子|小野涼子]]}}
: 陽和學園三年級生，宗介的學姐。對宗介一見鍾情並不斷對他展開攻勢，後來更主動加入御宅族社。喜歡玩[[日本成人遊戲|成人遊戲]]。

=== 其他角色 ===
; {{nihongo|卡拉·奧麗維亞|カーラ・オリヴィア|聲：[[遠野そよぎ|遠野そよぎ]]／[[岡嶋妙|岡嶋妙]]}}
: 可攻略角色。從國外來到日本的留學生，就讀陽和學園一年級，宗介的學妹，御宅族社成員。[[歷女|歷女]]，喜歡看時代劇，言語措辭亦受時代劇影響。有角色扮演的愛好，一直想讓真帆穿上自己設計的衣服。原本體型肥胖，但因為中暑而數日不吃不喝瘦下來後變成了美少女。
; {{nihongo|和呂田茜|和呂田 茜（わろた あかね）|聲：步サラ／岡本理繪}}
: 可攻略角色。陽和學園三年級生，宗介的學姐，御宅族社成員。是名[[腐女|腐女]]。

== 評價 ==
喜歡與喜歡的三角戀於「[[萌系遊戲大賞|萌系遊戲大賞]]」2016年11月的月間賞獲得第3名<ref>{{cite web|url=http://moe-gameaward.com/ranking/index.html?year_id=2016&month_id=11|title=2016年11月発売タイトル月間賞投票|work=萌系遊戲大賞|accessdate=2018-05-12|language=ja}}</ref>，並於年間排名獲得第46名<ref>{{cite web|url=http://moe-gameaward.com/annual/2016.html|title=萌えゲーアワード2016年間ランキング確定|work=萌系遊戲大賞|accessdate=2018-04-10|language=ja}}</ref>。
