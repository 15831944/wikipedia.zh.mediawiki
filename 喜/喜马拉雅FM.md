'''喜马拉雅FM'''是中国[[上海证大喜马拉雅网络科技有限公司|上海证大喜马拉雅网络科技有限公司]]于2013年3月上线的一个在线音频分享平台，平台采用UGC（用户原创内容）模式。平台提供了音频播放、下载、查找服务，也提供了用户个性化的个人推荐服务。用户也可以申请成为主播上传音频文件。

现支持[[iOS|iOS]]及[[Android|Android]]平台。
<ref>{{Cite web|url=https://www.ximalaya.com/|title=有声小说，有声书，儿童故事，相声，鬼故事等MP3有声读物尽在在线听书电台 - 喜马拉雅FM|accessdate=2018-05-28|work=www.ximalaya.com}}</ref>

由于采用[[UGC|UGC]]（用户原创内容）模式，喜马拉雅FM平台吸引了许多音频制作者，其中不乏知名的动漫声优、配音师、知名主持人、原创音乐人和一些草根明星。<ref name = 'finance'>{{Cite web|url=http://finance.ifeng.com/a/20140522/12386247_0.shtml|title=国内最大音频分享平台喜马拉雅获1150万美元风投|accessdate=2018-05-28|work=finance.ifeng.com}}</ref>

== 功能 ==
用户可以在喜马拉雅FM中收听有声书，收听在线广播，观看直播，收听新闻音频和其他用户上传的音频文件。大部分音频内容都是免费的，但也不乏收费内容。

== 公司历史 ==

* 2012年8月 喜马拉雅FM团队正式组建<ref name = 'finance'/>
* 2013年3月 喜马拉雅应用正式上线<ref name = 'finance'/>
* 2014年5月 喜马拉雅激活用户突破5000万大关<ref name = 'finance'/>
* 2014年5月 获得来自SIG（海纳亚洲）、[[凯鹏华盈|KPCB]]（凯鹏华盈）、Sierra Ventures的共1150万美元的 A 轮风险投资 <ref name = 'finance'/>
* 2017年11月 入选[[FT中文网|FT中文网]]与嘉宾传媒共同发布的中国商业案例TOP30榜单<ref>{{Cite news|url=http://finance.cnr.cn/gundong/20171109/t20171109_524019089.shtml|title=中国商业案例Top30发布 微信滴滴人人贷等入选|author=央广网|date=2017-11-09|work=央广网|accessdate=2018-05-28}}</ref>
