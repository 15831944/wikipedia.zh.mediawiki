'''庞晔'''，出生于[[江苏省|江苏省]][[江阴市|江阴市]]，毕业于[[中国传媒大学|中国传媒大学]]，为[[中国中央电视台|中国中央电视台]]主持人。<ref name="fox " >{{cite web |url=http://www.zhuchijie.com/show_283_z7_t1.html |title= 庞晔|publisher= 主持界|accessdate= 2011年|work= |language =}}</ref>

== 简介 ==
庞晔出生于[[江苏省|江苏省]][[江阴市|江阴市]]，他的父亲是科学工作人员，母亲是教师，具备良好的家庭环境和教养，使得她多才多艺，擅长[[歌舞|歌舞]]、[[书法|书法]]、[[绘画|绘画]]和[[琵琶|琵琶]]弹奏。<ref name="fox " />

1991年，庞晔考入[[南菁高级中学|南菁高级中学]]。<ref name="fox " />
1994年，庞晔考入[[中国传媒大学|中国传媒大学]]。<ref name="fox " />

1998年，庞晔就职于[[江苏电视台|江苏电视台]]《[[开心假日|开心假日]]》栏目组。<ref name="fox " />
2000年，庞晔在江苏电视台主持《[[家庭互联网|家庭互联网]]》栏目。<ref name="fox " />
2001年，庞晔在江苏电视台主持《[[无敌智多星|无敌智多星]]》和《[[女人百分百|女人百分百]]》。<ref name="fox " />

2005年，庞晔到中国中央电视台主持[[春节联欢晚会|春节联欢晚会]]。<ref>{{cite web |url= http://news.sina.com.cn/o/2005-02-09/07395092739s.shtml|title=庞晔北上主持春晚感慨多 |publisher=新浪网 |accessdate=2009-02-05 |work= |language =}}</ref>

== 个人生活 ==
庞晔的丈夫也是中国传媒大学学生，两人同为校友在学校认识的。<ref>{{cite web |url=http://ent.sina.com.cn/2004-12-29/0910613564.html |title=美食家老公“勾走”好吃庞晔 |publisher=新浪网 |accessdate=2004年 |work= |language =}}</ref>
