'''庞巴迪CRJ100''' 和 '''CRJ200''' (旧称[[加拿大航空工业|加拿大航空工业]]'''CRJ100''' 和 '''CRJ200''') 是由[[庞巴迪公司|庞巴迪飞机制造公司]]设计和制造的同一系列的[[支线客机|支线客机]]。

CRJ系列的灵感来源自Canadair挑战者[[商务喷气机|商务机]]。这种被称为挑战者610E的公务机项目于1981年终止。在Canadair被[[庞巴迪公司|庞巴迪公司]]收购后不久的1989年初，Canadair[[支线客机|支线客机]]项目正式启动。1991年5月10日,第一架CRJ100原型机完成了处女航。CRJ200与CRJ100在很大程度上是相同的，但200系列采用了更高效的涡扇发动机。

== 发展 ==

=== 起源 ===
CRJ家族的原型是加拿大航空工业的挑战者公务机。 上世纪70年代末, 挑战者相对较宽的机身与舒适的座椅备受商务人士的喜爱。 因此, 在八十年代, 庞巴迪决定要对这种机型进行改进拓宽（即挑战者610E项目）<ref>[https://www.flightglobal.com/pdfarchive/view/1980/1980%20-%201501.html "Canadair Challenger."] ''Flight International'', 21 June 1980. p. 1437.</ref> 然而, 早期对于挑战者的重新设计并没有发展得太顺利，这个项目不久后就被放弃了。
[[File:Lufthansa.crj-100.d-aclp.arp.jpg|缩略图]] 的CRJ100在进近]]
610E项目尽管失败，但它的灵魂并没有消失。 在1987年，一个更加雄心勃勃计划的萌芽出现了。这个萌芽在加拿大航空工业被庞巴迪收购后得到了蓬勃的发展。

1991年10月，''CRJ100的一架原型机'' 执行了其第一次在 [[蒙特利尔-米拉贝勒国际机场|蒙特利尔米拉贝尔国际机场]]的试飞。 1993年7月26日，第一架原型机(注册号C-FCRJ)在试飞中心附近因进入[[尾旋|尾旋]]而失事<ref>{{cite web|url=http://www.rjet.ca/blog/2012/08/28/the-history-of-canadair-regional-jet-msn-7001/|title=The History of Canadair Regional Jet MSN 7001|accessdate=20 November 2014|work=Winglets}}</ref><ref>{{cite web|url=http://aviation-safety.net/database/record.php?id=19930726-2|title=ASN Aircraft accident Canadair CL-600-2B19 Regional Jet CRJ-100 C-FCRJ Byers, KS|accessdate=20 November 2014|author=Harro Ranter|date=26 July 1993|publisher=Aviation-safety.net}}</ref>。 这起事故发生在这种飞机获得在1992年获得适航证 ，交付给客户以后的一年。

=== 进一步的发展 ===
[[File:Canadair_CL-600-2B19_Regional_Jet_CRJ-200ER_-_Air_Nostrum_(Iberia_Regional)_-_EC-JEN_-_LEMD_-_200503051641.jpg|右]]的CRJ200ER]]
CRJ100 ER的是CRJ100的第一个改装机型。它的货门得到了改进，使其比早期的CRJ100寿命更长。<ref name="Flight20June2017">{{cite news|url=https://www.flightglobal.com/news/articles/insight-from-bombardier-the-rise-and-rise-of-region-438479/|title=The Rise and Rise of Regional Aircraft|author=Bombardier|date=20 June 2017|work=FlightGlobal}}</ref>

CRJ200是另一种亚型号，采用更有效率的发动机。它还有一个货运版本，名为CRJ200 PF(货运型)<ref>{{cite web|url=http://www.flightglobal.com/news/articles/west-air-launches-cargo-crj-as-cascade-aims-for-kit-deal-208314/|title=West Air launches cargo CRJ as Cascade aims for kit deal|author=Shannon, Darren and David Kaminski-Morrow|date=8 August 2006|publisher=Flightglobal}}</ref><ref>{{cite web|url=http://www.airteamimages.com/50403.html|title=Bombardier – CRJ – 200|accessdate=20 November 2014|publisher=AirTeamImages.com}}</ref>

== 设计 ==
[[File:CRJ_cockpit.jpg|缩略图]]
[[File:United_Express_Bombardier_CRJ_200s.jpg|缩略图]]。]]
庞巴迪CRJ100/200大致采用了与挑战者CL-600差不多的设计。机身比CL-600长5.92米，一个加强的燃油系统，增强型[[起落架|起落架]] ，以及一个额外的紧急出口。 CRJ100一般可容纳50名乘客，在最大客运布局下可承载52名乘客。 它采用一对 [[通用电气CF34发动机|通用电气CF34-3A1]] [[渦輪扇發動機|涡轮风扇]] 发动机，它们能够产生高达41.0 kN（4,180公斤/9,220磅）的推力。 CRJ100采用了[[罗克韦尔柯林斯|柯林斯]]的航电设备，包括一个[[气象雷达|气象雷达]]。

== 业务历史 ==
[[File:UsairwaysN419aw_07302009.jpg|缩略图]]
在客机的头100天的运营服务中，CRJ100共执行了1237架次航班，调度可靠性据说达到了99%,而其燃油经济性达到了8%。

美国控股公司顶峰航空公司运营配备44座的CRJ飞机随后被转化为50座飞机。

在2006年期间，CRJ100/200项目被终止并停产。一些航空公司引进了的更大的版本，例如CRJ700。
[[File:Comair_CRJ.JPG|缩略图]]]]
据报导，在2010年后，各种航空公司已经加速退休CRJ机队，因为CRJ100/200经济性低下。 在最近几年，一些航空公司用[[巴西航空工業E系列|巴西航空工业E系列]]飞机和较大的[[庞巴迪CRJ700系列|庞巴迪CRJ-700/900]]来替换CRJ100/200机队。

== 不同型号 ==
[[File:Canadair_RJ-100SE_N877SE_Kenosha_WI_10.06.97R_edited-2.jpg|右]]
[[File:Canadair_Regional_Jet_CRJ-200LR_2006_04_07.jpg|右]]
几种CRJ100/200型的不同型号。
; CRJ100<span> </span>
: 这是最初的版本。配备 [[通用电气航空|通用电气]] [[通用电气CF34发动机|CF34-3A1]] 引擎。 目前的经营者，包括格鲁吉亚航空和盧安達航空。

; CRJ100SF<span> </span>
: CRJ100的客改货型。

; CRJ200<span> </span>
: CRJ200与CRJ100除了引擎之外相同，200升级到CF34-3B1发动机，以提高效率。

; CRJ200PF<span> </span>
: 货运版本的CRJ200。

; CRJ200SF<span> </span>
: CRJ200的客改货型。

; CRJ440<span> </span>
: 44座版本，这个版本有较少的座位，以便满足需要的一些主要的美国航空公司。

; [[庞巴迪挑战者850|挑战者800/850]]<span> </span>
: 一种商务机型。

; CRJ500<span> </span>
: 50座版本，在2001年被叫停。
