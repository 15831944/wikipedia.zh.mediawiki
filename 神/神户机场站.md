'''神户机场站'''（{{jpn|j=神戸空港駅|hg=こうべくうこう|rm=Kōbe kūkō eki|f=y}}）是位于[[日本|日本]][[兵库县|兵库县]][[神户市|神户市]][[中央區_(神戶市)|中央区]][[神户机场|神户机场]]1号的神户新交通港湾人工岛线铁路车站。车站编号为'''P09'''。车站整体位于神户机场所在的神户机场岛上。

==历史==
神户机场站随港湾人工岛线的线路延长而开业，在2006年2月16日神户机场开业前的同月2日正式投入运营。<ref>{{cite web |title=第19回　全国街路事業コンクール応募資料 |url=http://www.gaisokkyo.jp/doromanage/wp-content/uploads/2017/10/pro-19-26.pdf |accessdate=2018-11-16 |language=ja}}</ref>
===年表===
*2006年2月2日：港湾人工岛线从市民广场站延伸至本站，本站投入运营。

==车站构造==
本站为设一个[[岛式站台|岛式站台]]的两台两线高架站。

站房内，二层为站厅层，三层为站台层。站房整体建于航站楼前的道路旁，通过位于连接通道可直接通向航站楼。<ref name="terminal 2F">{{cite web |title=[ 2F（出発階）]（館内ご案内） - 神戸空港ターミナル |url=http://www.kairport.co.jp/terminal/2f |accessdate=2018-11-15 |language=ja}}</ref>在车站的流线设计中与航站楼一样充分采用了通用化设计的理念，例如将出站闸机与机场的值机柜台步行距离缩到了最短。车站的付费区域内有扶梯、直梯、无障碍厕所、[[自動體外心臟去顫器|AED]]等设施。<ref>{{cite web |title=路線図・駅情報  神戸新交通 |url=http://www.knt-liner.co.jp/station/#facility |accessdate=2018-11-16 |language=ja}}</ref>

该站的车站印章置于检票口旁的人工窗口处,乘客可自行盖印留念。

== 发车间隔 ==
工作日一般时段以一小时九班的频率发车。早晚高峰期时发车间隔为2-5分钟。<br/>
周末及休息日发车间隔基本为十分钟一班。<ref>{{cite web |title=神戸空港 神戸新交通 |url=http://www.knt-liner.co.jp/stationp09/?tt=1 |accessdate=2018-11-15 |language=ja}}</ref>
