'''神户市'''（{{jpn|j=神戸市|hg=こうべし|rm=Kōbe shi|f=y}}）是位於[[日本|日本]][[兵庫縣|兵庫縣]]東南部的[[都市|都市]]，為兵庫縣[[都道府縣廳所在地|首府]]，也是[[政令指定都市|政令指定都市]]之一<ref>{{cite web |url=http://www.soumu.go.jp/main_sosiki/jichi_gyousei/bunken/shitei_toshi-ichiran.html |script-title=ja:指定都市一覧 |accessdate=2013-11-24|publisher={{lang|ja|総務省統計局}} |language=ja}}</ref>，下轄有9個[[市轄區|區]]。神戶市的面積為552.83平方公里。2016年6月，神戶市有人口1,538,053人<ref name="Kobe Population1">{{cite web |url=http://www.city.kobe.lg.jp/information/data/statistics/toukei/jinkou/suikeijinkou.html |script-title=ja:毎月推計人口 |accessdate=2014-01-23 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204758/http://www.city.kobe.lg.jp/information/data/statistics/toukei/jinkou/suikeijinkou.html |archivedate=2014-01-04 }}</ref>，是日本[[日本城市人口排名|人口第七多的都市]]。神戶市與[[大阪市|大阪市]]、[[京都市|京都市]]並為[[京阪神|京阪神]]都會區的核心都市<ref>{{cite web |url=http://www.kkr.mlit.go.jp/kensei/japanese/index02.html |script-title=ja:京阪神都市圏の概要 |accessdate=2013-12-21 |publisher={{lang|ja|国土交通省近畿地方整備局}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20130209084002/http://www.kkr.mlit.go.jp/kensei/japanese/index02.html |archivedate=2013-02-09 }}</ref>，同時也和其附近的[[衛星都市|衛星都市]]組成[[神戶都市圈|神戶都市圈]]<ref>{{cite web |url=http://www.cao.go.jp/chiiki-shuken/doc/2shiryou13.pdf |script-title=ja:地方公務員制度改革についての提案 |accessdate=2013-11-19 |publisher={{lang|ja|内閣府}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20130121115118/http://www.cao.go.jp/chiiki-shuken/doc/2shiryou13.pdf |archivedate=2013-01-21 }}</ref>。

「神戶」這一地名是起源於現在神戶市中心的[[三宮|三宮]]、元町地區在古代曾是[[生田神社|生田神社]]的領地，並居住有生田神社的{{link-ja|神封|神封|神封戶}}<ref>{{cite web |url=http://haikara.kimec.ne.jp/01/001.html |script-title=ja:「神戸」という地名の由来について |accessdate=2013-12-22 |publisher={{lang|ja|神戸の歴史や文化について知る}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20150308164749/http://haikara.kimec.ne.jp/01/001.html |archivedate=2015-03-08 }}</ref>。神戶地處[[大阪灣|大阪灣]]沿岸，風浪平緩且沿岸水深較深，地形極為適宜建設[[港口|港口]]<ref name="Kobe Geography">{{cite web |url=http://www.city.kobe.lg.jp/safety/prevention/foundation/chishitsu/index.html |script-title=ja:神戸の地形・地質 |accessdate=2013-12-21 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204807/http://www.city.kobe.lg.jp/safety/prevention/foundation/chishitsu/index.html |archivedate=2014-01-04 }}</ref>。在古代時期，神戶就是京都及大阪的外港之一<ref name="Kobe History">{{cite web |url=http://www.city.kobe.lg.jp/information/public/media/shisei/img/111718.pdf |script-title=ja:神戸の歴史 |accessdate=2013-12-21 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140104204732/http://www.city.kobe.lg.jp/information/public/media/shisei/img/111718.pdf |archivedate=2014-01-04 }}</ref>。1868年，神戶成為日本最早開放對外國通商的五個港口之一，之後神戶迅速發展為日本最重要的港灣都市之一<ref>{{cite web|url=http://www.5city.or.jp/kobe/history.html|script-title=ja:日本開港五都市【神戸市｜歷史】|accessdate=2013-12-21|publisher={{lang|ja|日本開港五都市観光協議会}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140104213242/http://www.5city.or.jp/kobe/history.html|archivedate=2014-01-04}}</ref>。也正因其是日本最早的通商港口，神戶以開放和國際化的氣氛而聞名。1995年的[[阪神大地震|阪神淡路大震災]]雖然給神戶帶來巨大的打擊，然而經過多年的重建，神戶的都市建設和人口都已超過地震之前的水準<ref name="Kobe Demographics">{{cite web|url=http://www.city.kobe.lg.jp/information/data/statistics/toukei/img/sinsaimaekoeru.pdf|script-title=ja:神戸市人口，震災前人口を超える|accessdate=2013-12-21|publisher={{lang|ja|神戸市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131107145428/http://www.city.kobe.lg.jp/information/data/statistics/toukei/img/sinsaimaekoeru.pdf|archivedate=2013-11-07}}</ref>。神戶也是一座宜居都市，並曾在2007年入選[[福布斯|福布斯]]雜誌評出的「世界最清潔的25座城市」<ref>{{cite web|url=http://www.forbes.com/2007/04/16/worlds-cleanest-cities-biz-logistics-cx_rm_0416cleanest.html|title=Which Are The World's Cleanest Cities?|accessdate=2013-12-21|publisher=Forbes|deadurl=no|archiveurl=https://web.archive.org/web/20131225022141/http://www.forbes.com/2007/04/16/worlds-cleanest-cities-biz-logistics-cx_rm_0416cleanest.html|archivedate=2013-12-25}}{{en}}</ref>，其後也於2012年在[[瑞士|瑞士]]的諮詢公司ECA國際評選出的世界宜居都市排名中排名第五位，是唯一入選前10位的日本都市<ref>{{cite web|url=http://www.eca-international.com/news/press_releases/7647/#.UrVXNbTTRDe|title=Singapore is best place to live for Asian expatriates, ECA International's research shows|accessdate=2013-12-21|publisher=ECA International}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}{{en}}</ref>。

== 地理 ==
=== 地形及地質 ===
[[File:Amsterdam_at_Kobe10s3872.jpg|thumb]]
神戶市地處兵庫縣東南部，[[六甲山|六甲山]]系和[[瀨戶內海|瀨戶內海]]構成神戶依山傍海的都市景觀。六甲山東西橫貫神戶市域北部，構成神戶市中心街區和郊區的天然分界線。六甲山最高峰的海拔高度是931.25米，也是神戶市內的最高點<ref>{{cite web |url=http://www.city.kobe.lg.jp/information/public/online/rokkosanpo/model_course/a06/ |script-title=ja:六甲山歩モデルコース6 岡本から六甲山最高峰、有馬温泉へ |accessdate=2013-12-21 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140104204800/http://www.city.kobe.lg.jp/information/public/online/rokkosanpo/model_course/a06/ |archivedate=2014-01-04 }}</ref>。山地北側是神戶市內保存較多自然的地區，關西著名的溫泉旅遊地[[有馬溫泉|有馬溫泉]]就位於這裡<ref>{{cite web|url=http://www.arima-onsen.com/onsen.html|script-title=ja:温泉について|accessdate=2013-12-21|publisher={{lang|ja|有馬温泉観光協会}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104212921/http://www.arima-onsen.com/onsen.html|archivedate=2014-01-04}}</ref>。在六甲山地北麓和其附近的丘陵地帶也有一些[[新市鎮|新市鎮]]<ref>{{cite web|url=http://www.lib.kobe-u.ac.jp/directory/eqb/book/4-991/pdf/10_chapter7.pdf|script-title=ja:西神ニュータウンはいま|accessdate=2013-12-21|publisher={{lang|ja|神戸大学付属図書館}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20130204215643/http://www.lib.kobe-u.ac.jp/directory/eqb/book/4-991/pdf/10_chapter7.pdf|archivedate=2013-02-04}}</ref>，有多條鐵路和公路穿過六甲山連接這些新市鎮和神戶市中心市區。在六甲山麓的南側，有眾多神戶開港早期時修建的外國人住宅<ref>{{cite web|url=http://www.kobeijinkan.com/history|script-title=ja:神戸北野ヒストリー|accessdate=2013-12-21|publisher={{lang|ja|神戸北野異人館街}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104205535/http://www.kobeijinkan.com/history|archivedate=2014-01-04}}</ref>，這一區域也是日本最早的近代高檔住宅區。神戶市的大部分中心市區則位於六甲山系和瀨戶內海之間的沿海平地。這一地區的南北長度僅有約三公里，使得神戶的中心街區呈現沿海岸的東西細長的形狀。為了擴充市區和港口用地，神戶曾多次實施[[填海|填海]]造地<ref>{{cite web |url=http://ci.nii.ac.jp/els/110003967384.pdf?id=ART0005440640&type=pdf&lang=jp&host=cinii&order_no=&ppv_type=0&lang_sw=&no=1387640402&cp= |script-title=ja:神戸の埋立の歴史|accessdate=2013-12-21|publisher={{lang|ja|社団法人地盤工学会}} |language=ja}}</ref>。神戶市沿海的兩大人工島—[[港灣人工島|港灣人工島]]和[[六甲人工島|六甲人工島]]均是利用六甲山的土石填海造成的人工島<ref>{{cite web|url=http://www.feel-kobe.jp/area_guide/portisland/|script-title=ja:ポートアイランド/神戸空港/六甲アイランド|accessdate=2013-12-21|publisher={{lang|ja|神戸国際観光コンベンション協会}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140103124748/http://www.feel-kobe.jp/area_guide/portisland/|archivedate=2014-01-03}}</ref>。神戶市內的主要河流則有{{link-ja|住吉川 (兵庫縣)|住吉川 (兵庫県)|住吉川}}、{{link-ja|生田川|生田川}}、{{link-ja|新湊川|新湊川}}等<ref>{{cite web|url=http://www.city.kobe.lg.jp/life/town/river/shirou/01group.html|script-title=ja:神戸の川マップ～神戸の川は主に4つのグループに分けられます～|accessdate=2013-12-21|publisher={{lang|ja|神戸市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204557/http://www.city.kobe.lg.jp/life/town/river/shirou/01group.html|archivedate=2014-01-04}}</ref>，這些河流大多發源自六甲山且在流經神戶市區後就直接入海，因此長度十分短小。

神戶市的地質可大概分為兩個區域。六甲山地主要由[[花崗岩|花崗岩]]構成，其地質構造形成在約7000萬年之前<ref name="Kobe Geography" />。由於花崗岩具有天然的過濾作用，累積的降雨在經過花崗岩的過濾後滲入地下，使得六甲山成為神戶重要的天然水脈。而自六甲山湧出的優質水源也讓神戶生產的[[礦泉水|礦泉水]]和[[日本酒|日本酒]]享有盛譽<ref>{{cite web|url=https://www2.env.go.jp/water-pub/mizu-site/meisui/data/index.asp?info=58|script-title=ja:布引渓流|accessdate=2013-12-21|publisher={{lang|ja|環境省}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20130105194603/https://www2.env.go.jp/water-pub/mizu-site/meisui/data/index.asp?info=58|archivedate=2013-01-05}}</ref>。而市區的平地部分則形成在約3000萬年之前，在基盤岩類的上方覆蓋有較厚的沖積層<ref>{{cite web |url=http://www.lib.kobe-u.ac.jp/directory/eqb/book/11-197/html/pdf/23-28.pdf |script-title=ja:神戸の地質 ・ 地盤と被害の関係 |accessdate=2013-12-21 |publisher={{lang|ja|神戸大学付属図書館}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204331/http://www.lib.kobe-u.ac.jp/directory/eqb/book/11-197/html/pdf/23-28.pdf |archivedate=2014-01-04 }}</ref>。神戶市內有多條活斷層，這些斷層大致可分為須磨断層・諏訪山断層、淡路－六甲活斷層系和大阪灣斷層系三個系統<ref name="Kobe Geography" />。

=== 氣候 ===
[[File:Kobe_Port_island.jpg|thumb]]
神戶市屬於[[瀨戶內海式氣候|瀨戶內海式氣候]]區域，全年氣候較為溫和<ref name="Kobe Weather">{{cite web|url=http://www.city.kobe.lg.jp/life/recycle/environmental/heat/gashaishutu/suishinkeikaku/Chap3.html|script-title=ja:市域の概況|accessdate=2013-12-21|publisher={{lang|ja|神戸市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140104204727/http://www.city.kobe.lg.jp/life/recycle/environmental/heat/gashaishutu/suishinkeikaku/Chap3.html|archivedate=2014年1月4日|df=}}</ref>。六甲山地北側的地區則具有內陸型氣候的特徵，和市區相比有較大的溫差。和日本其他地區相比，神戶的降雨相對較少，全年平均降水量約在1,200毫米。[[櫻花|櫻花]]盛開的春季是神戶最為宜人的季節。在經過潮濕的[[梅雨|梅雨]]季節之後，神戶進入炎熱的夏季。由於神戶空氣較為乾燥且有[[海風|海風]]吹拂，使得神戶的夏季雖然炎熱，但相對[[近畿地方|近畿地方]]其他大都市而言白天仍較為舒適<ref>{{cite web|url=http://allabout.co.jp/gm/gc/396851/2/|script-title=ja:大阪京都神戸、暑くない街は？～夏の暑さを比べてみた|accessdate=2013-12-21|publisher=All About|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204134/http://allabout.co.jp/gm/gc/396851/2/|archivedate=2014-01-04}}</ref>。在夏秋之交的時節，神戶有時會受到颱風的影響。神戶的秋季頗為短暫。而冬季的神戶則由於山地地形的影響，常有被稱為{{link-ja|六甲颪|六甲颪}}的[[季風|季節風]]吹襲<ref>{{cite web|url=http://www.chs.nihon-u.ac.jp/institute/nature/kiyou/2009/pdf/2_10.pdf|script-title=ja:六甲山地南麓における寒候期のおろし風の解析|accessdate=2013-12-21|publisher={{lang|ja|日本大学}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204347/http://www.chs.nihon-u.ac.jp/institute/nature/kiyou/2009/pdf/2_10.pdf|archivedate=2014-01-04}}</ref>。雖然由於六甲颪的影響使得神戶冬季氣溫較低，然而因冬季空氣乾燥使得神戶市區少有降雪。

===人口===
{{日本人口統計|location=Kobe, Hyogo|code=28100|name=神戶市}}

[[File:Population_of_Kobe.svg|thumb]]
神戶在開港之初時人口僅有約2.5萬人<ref name="Kobe Population2">{{cite web |url=http://www.city.kobe.lg.jp/information/project/construction/green/img/keikaku01.pdf |script-title=ja:神戸の概況 |accessdate=2013-12-23 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204930/http://www.city.kobe.lg.jp/information/project/construction/green/img/keikaku01.pdf |archivedate=2014-01-04 }}</ref>，之後神戶的人口快速增加。在1889年實施市制時，神戶已有人口約13萬人<ref name="Kobe Population2" />。在1920年日本首次[[人口普查|人口普查]]時，神戶的人口已經超過60萬，僅次於東京和大阪，是當時日本人口第三多的城市<ref>{{cite web |url=http://www.city.kyoto.jp/sogo/toukei/Publish/Monthly/Topics/201303-01.pdf |script-title=ja:大都市の人口の推移－国勢調査結果から |accessdate=2013-12-23 |publisher={{lang|ja|京都市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104211021/http://www.city.kyoto.jp/sogo/toukei/Publish/Monthly/Topics/201303-01.pdf |archivedate=2014-01-04 }}</ref>。1939年，神戶市人口超過100萬人<ref name="Kobe Population2" />。然而在二戰期間，由於美軍多次空襲使得大量人口離開城市至農村避難，神戶的人口劇減至38萬人<ref name="Kobe Population2" />。戰後隨著神戶市復興進程的進展，暫時逃至農村的人口也回到城市。1956年，神戶人口數再次超過100萬人。之後神戶的人口持續增加，在1992年超過了150萬人<ref>{{cite web |url=http://www.city.kobe.lg.jp/information/about/energy/rekishi/nenpyo.html |script-title=ja:近代神戸略年表 |accessdate=2013-12-23 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204821/http://www.city.kobe.lg.jp/information/about/energy/rekishi/nenpyo.html |archivedate=2014-01-04 }}</ref>。1995年的阪神大地震導致神戶的人口一度大幅減少至約142萬人<ref name="Kobe Demographics" />。之後神戶的人口恢復成長，並在2004年超過阪神大地震之前水準。然而在近年，受到高齡化和少子化的影響，神戶市人口出現減少<ref>{{cite web|url=http://www.city.kobe.lg.jp/child/grow/shinseido/img/siryou9.pdf|script-title=ja:神戸市の子ども・子育てをめぐる現状|accessdate=2013-12-23|publisher={{lang|ja|神戸市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204613/http://www.city.kobe.lg.jp/child/grow/shinseido/img/siryou9.pdf|archivedate=2014-01-04}}</ref>。2015年，神戶的人口數被[[福岡市|福岡市]]超過，變為日本人口第七大都市<ref>{{cite web|url=http://www.sankei.com/west/news/160218/wst1602180089-n1.html|script-title=ja:神戸市、人口で「５大市」から転落…福岡市に抜かれ６位に　若者が減少|accessdate=2016-02-21|publisher={{lang|ja|産経ニュース}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20160221104633/http://www.sankei.com/west/news/160218/wst1602180089-n1.html|archivedate=2016-02-21}}</ref>。2016年1月時，神戶市的人口約有153.8萬人。

神戶的人口高度集中於沿海的中心市區。沿海的垂水區、須磨區、長田區、兵庫區、中央區、灘區、東灘區的合計總面積雖只佔神戶市總面積的約31%，然而人口數卻佔神戶總人口的約69%<ref name="Kobe Population1" />。相反地處內陸的北區和西區雖佔去神戶市過半面積且在戰後因新市鎮的開發使得人口大幅增加，合計總人口仍只有約47萬人<ref name="Kobe Population1" />。而阪神大地震也給神戶的人口地理帶來變化。靠近大阪的東灘區、灘區和位於市中心的中央區在震後人口持續增加，而受地震影響嚴重的長田區的人口則仍然大幅低於地震前水準<ref name="Kinki" />{{rp|62}}。和日本其他地方相同，神戶也面臨嚴重的高齡化和少子化問題。2013年，神戶人口的高齡者比率超過24%<ref>{{cite web |url=http://web.pref.hyogo.lg.jp/hw07/documents/01jyunni_1.xls |script-title=ja:市町別高齢化率 |accessdate=2013-12-23 |publisher={{lang|ja|兵庫県}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140104205359/http://web.pref.hyogo.lg.jp/hw07/documents/01jyunni_1.xls |archivedate=2014-01-04 }}</ref>。相反的是在2010年，神戶的[[總和生育率|總和生育率]]則只有1.29<ref>{{cite web |url=http://web.pref.hyogo.lg.jp/hw07/documents/h22shussyouritu.xls |script-title=ja:市区町別合計特殊出生率 |accessdate=2013-12-23 |publisher={{lang|ja|兵庫県}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140104205453/http://web.pref.hyogo.lg.jp/hw07/documents/h22shussyouritu.xls |archivedate=2014-01-04 }}</ref>。神戶市是日本最國際化的都市之一，市內也居住有不少外國人。2011年時，有約44,000人外國人生活在神戶<ref name="Foreigner">{{cite web |url=http://www.city.kobe.lg.jp/information/project/taikousannkousiryou.pdf |script-title=ja:参考資料 |accessdate=2013-12-23 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204719/http://www.city.kobe.lg.jp/information/project/taikousannkousiryou.pdf |archivedate=2014-01-04 }}</ref>。其中以韓國和朝鮮人最多，之後依次是中國人、越南人、美國人、印度人和菲律賓人。神戶的外國人人口比例在政令指定都市中位列第四，僅次於大阪市、濱松市和名古屋市<ref name="Foreigner" />。神戶市內還有神戶中華同文學校等8家外國人學校<ref>{{cite web |url=http://www.city.kobe.lg.jp/information/public/online/ksnavi/vol8/question01.html |script-title=ja:神戸がめざす国際化先進都市って、どんなまち？ |accessdate=2013-12-24 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140104204922/http://www.city.kobe.lg.jp/information/public/online/ksnavi/vol8/question01.html |archivedate=2014-01-04 }}</ref>。

== 歷史 ==
=== 古代神戶 ===
[[File:Ikuta-jinja_Kobe07n4272.jpg|thumb]]
神戶的歷史可追溯至[[舊石器時代|舊石器時代]]，在神戶市內多地都發現舊石器時代和[[繩文時代|繩文時代]]的文物<ref>{{cite web|url=http://www.feel-kobe.jp/about_kobecity/history/|script-title=ja:神戸の歴史探訪|accessdate=2013-12-22|publisher={{lang|ja|神戸国際観光コンベンション協会}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204249/http://www.feel-kobe.jp/about_kobecity/history/|archivedate=2014-01-04}}</ref>。進入[[彌生時代|彌生時代]]後，隨着稻作技術和金屬器的傳入，神戶的人口進一步增加。據[[日本書紀|日本書紀]]記載，神戶三大神社中的[[生田神社|生田神社]]<ref>{{cite web|url=http://www.ikutajinja.or.jp/about/|script-title=ja:生田神社の紹介|accessdate=2013-12-22|publisher={{lang|ja|生田神社}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131219042103/http://www.ikutajinja.or.jp/about/|archivedate=2013年12月19日|df=}}</ref>和{{link-ja|長田神社|長田神社}}<ref name="Ancient Kobe">{{cite book |author={{lang|ja|坂江渉}}|script-title=ja:《神戸・阪神間の古代史》 |year=2011 |publisher={{lang|ja|のじぎく文庫}} |location=神戸市 |isbn=978-4-343-00618-9 |language=ja}}</ref>{{rp|65}}的歷史都可追溯至彌生時代後期。而在[[古墳時代|古墳時代]]之後，隨著豪族的興起，在日本各地開始出現巨大的{{link-ja|前方後圓墳|前方後円墳}}。神戶市內最為著名的古墳是{{link-ja|五色塚古墳|五色塚古墳}}，這處古墳也是兵庫縣內最大的古墳<ref>{{cite web|url=http://www.city.kobe.lg.jp/culture/culture/folk/estate/bunkazai/syokai/goshiki.html|script-title=ja:国指定記念物（史跡）五色塚古墳|accessdate=2013-12-22|publisher={{lang|ja|神戸市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204811/http://www.city.kobe.lg.jp/culture/culture/folk/estate/bunkazai/syokai/goshiki.html|archivedate=2014-01-04}}</ref>，也顯示在這一時期神戶的已經有一定程度的文明發展。在奈良時代，當時的朝廷在兵庫區修建{{link-ja|大輪田泊|大輪田泊}}，開始神戶作為港口的歷史<ref name="Ancient Kobe" />{{rp|10}}。大輪田泊是攝播五泊之一，是當時日本重要的貿易據點<ref>{{cite web|url=http://www.uminet.jp/know/history/|script-title=ja:瀬戸内海の歴史|accessdate=2013-12-22|publisher={{lang|ja|瀬戸内・海の路ネットワーク推進協議会}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140104212746/http://www.uminet.jp/know/history/|archivedate=2014-01-04}}</ref>。然而在平安時代後期，由於國家財政無力改修港口，大輪田泊陷入荒廢。

大輪田泊的蕭條狀況在[[平氏政權|平氏政權]]興起後發生巨大的變化。[[平清盛|平清盛]]在大輪田泊修築人工島{{link-ja|經島|経が島}}，試圖將大輪田泊發展為[[日宋貿易|日宋貿易]]的據點<ref>{{cite web |url=http://www.city.kobe.lg.jp/information/institution/institution/library/furusato/kyogashima.html |script-title=ja:平清盛と神戸　経ヶ島(きょうがしま) |accessdate=2013-12-22 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204913/http://www.city.kobe.lg.jp/information/institution/institution/library/furusato/kyogashima.html |archivedate=2014-01-04 }}</ref>。1180年，平清盛更強行宣布遷都[[福原京|福原京]]，其位置位於現在神戶市的中央區和兵庫區北部<ref>{{cite web |url=http://kinki.mof.go.jp/content/000042061.pdf |script-title=ja:平清盛の夢の都「福原京」 |accessdate=2013-12-22 |publisher={{lang|ja|近畿財務局}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204639/http://kinki.mof.go.jp/content/000042061.pdf |archivedate=2014-01-04 }}</ref>。然而由於反對勢力過於強大，福原遷都只實施半年後日本的首都就遷回[[平安京|平安京]]。平氏政權也在平清盛去世後迅速崩潰，而神戶更是平氏政權瓦解過程中重要戰役[[一之谷之戰|一之谷之戰]]的戰場。福原京的修築雖可說是僅停留在計劃層面<ref>{{cite web |url=http://www.hyogo-c.ed.jp/~rekihaku-bo/historystation/legend/html/013/013.html |script-title=ja:平氏伝説の旅 ～大輪田泊と、福原京から煙島まで |accessdate=2013-12-22 |publisher={{lang|ja|兵庫県立教育研修所}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20130216065508/http://www.hyogo-c.ed.jp/~rekihaku-bo/historystation/legend/html/013/013.html |archivedate=2013-02-16 }}</ref>，然而在鎌倉時代初期，高僧重源又重振大輪田泊的港口<ref>{{cite web |url=http://www.city.kobe.lg.jp/information/institution/institution/document/year/cyuusei.html |script-title=ja:神戸歷史年表　中世（室町・戦国時代まで） |archivedate=2014-01-04 }}</ref>。在14世紀前期這一自[[建武新政|建武新政]]轉換至[[室町幕府|室町幕府]]的劇變時期，神戶是[[湊川之戰|湊川之戰]]的舞台。也是在同一時期開始，除了[[嘉吉之亂|嘉吉之亂]]至[[應仁之亂|應仁之亂]]之間的20多年間神戶曾由[[山名氏|山名氏]]統治之外，神戶進入約200年的[[赤松氏|赤松氏]]統治時期<ref>{{cite web |language=ja}}</ref>{{rp|65}}。當時的兵庫津是自西日本前往[[北海道|北海道]]的{{link-ja|北前船|北前船}}的重要據點港口<ref>{{cite web |url=http://www.westjr.co.jp/company/issue/bsignal/09_vol_123/feature01.html === 開港至二戰期間的神戶 ===
[[File:Kitano_Street_Kobe02s3s4272.jpg|thumb]]
在[[江戶幕府|江戶幕府]]末期，日本在歐美的壓力之下被迫打開國門。1858年，幕府與[[美國|美]][[英國|英]][[法國|法]][[俄羅斯|俄]][[荷蘭|荷]]五國簽訂[[安政五國條約|安政五國條約]]，條約規定開放五個港口為通商港口。列強雖本慾選擇已有一定規模的兵庫津作為關西地區的開放港口，然因兵庫津排外風氣甚盛加上兵庫津附近的神戶村的沿岸水深較深適合建港<ref>{{cite book }}</ref>。1868年，神戶正式開港<ref name="autogenerated1">{{cite web |url=http://www.city.kobe.lg.jp/culture/leisure/history/kindai/index.html |script-title=ja:近代 |accessdate=2013-12-22 }}</ref>。市區也同樣以湊川為界，湊川以西的兵庫是舊市區，而湊川以東的神戶則有較多外國人居住，使得當時的神戶呈現雙子都市的格局<ref name="Kinki">{{cite book |author={{lang|ja|平岡昭利・野間晴雄}} |script-title=ja:《地図で読む百年　近畿Ⅱ》 |year=2006 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=978-4772230544 |language=ja}}</ref>{{rp|58}}。隨著市區範圍的擴大，兩大市區合二為一，市中心也逐漸往東側的神戶移動，而「神戶」也成為包括兵庫在內的全市的正式名稱。在神戶開港的同時，神戶市內還開闢專供外國人居住的[[神戶外國人居留地|神戶外國人居留地]]<ref name="autogenerated1" {{cite web |url=http://www.kobe-kyoryuchi.com/kobe_kyoryuchi/miserarete/rekishi_unei.html |script-title=ja:居留地の運営 |accessdate=2013-12-28 |publisher={{lang|ja|神戸旧居留地オフィシャルサイト}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20120510032451/http://www.kobe-kyoryuchi.com/kobe_kyoryuchi/miserarete/rekishi_unei.html |archivedate=2012-05-10 }}

}}</ref>。在大正及昭和初期，神戶和[[倫敦|倫敦]]、[[紐約|紐約]]、[[漢堡|漢堡]]並列為世界四大海運市場<ref>{{cite web |url=http://www.jica.go.jp/kansai/office/pdf/profile_04.pdf |script-title=ja:地域に密着した国際協力の拠点として |accessdate=2013-12-22 === 戰後神戶 ===

1995年1月17日，在神戶市近海發生[[阪神淡路大地震|阪神淡路大地震]]。神戶市內部分地區的烈度達到7級，市內死者人數多達4,571人，受災情況極為嚴重<ref>{{cite web |url=http://www.city.kobe.lg.jp/safety/fire/hanshinawaji/higai.html |script-title=ja:被害の状況 総論 |archivedate=2014-01-04 }}</ref>。地震雖然神戶受到巨大的打擊，然而也意味神戶新的復興的開始。交通網的整備是神戶震後復興的重要內容。[[明石海峽大橋|明石海峽大橋]]的開通使得神戶市和淡路島之間的交通情況得到大幅改進<ref>{{cite web |url=http://www.jb-honshi.co.jp/technology/akashi.html |archiveurl=https://web.archive.org/web/20130601110954/http://www.jb-honshi.co.jp/technology/akashi.html |archivedate=2013-06-01 }}</ref>。而[[神戶機場|神戶機場]]的開業使得神戶成為擁有海空雙港的城市<ref>{{cite web |url=http://www.city.kobe.lg.jp/life/access/harbor/rekishi.html |script-title=ja:神戸港の歴史 |accessdate=2013-12-23 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20131218085858/http://www.city.kobe.lg.jp/life/access/harbor/rekishi.html |archivedate=2013-12-18 }}</ref>。然而神戶的震後復興也同樣面臨諸多問題。諸如長田等因地震嚴重受損的街區雖然在震後實施[[都市更新|都市更新]]事業，然而因人口減少等問題使得開發事業停滯不前。在進入21世紀之後，神戶市在2008年被[[聯合國教科文組織|聯合國教科文組織]]評選為設計都市，是亞洲首個獲此殊榮的城市<ref>{{cite web |url=http://www.city.kobe.lg.jp/information/public/online/photo/number22/special_features/ |script-title=ja:アジア初！ユネスコのデザイン都市に認定 |accessdate=2013-12-23 |publisher={{lang|ja|神戸市}} |language=ja == 行政區劃 ==
{{File2|zh-hant=Administrative divisions of Kobe in zh-t.svg|zh-hans=Administrative divisions of Kobe in zh-s.svg|thumb|left|220px|神戶市市域擴大史}}
神戶市在1889年設市，是日本最早的[[市|市]]之一。設市當時的神戶市的面積為21.28平方公里，範圍只包括現在的中央區和兵庫區的部分地區<ref name="Kobe Wards">{{cite web |url=http://www.city.kobe.lg.jp/information/about/energy/rekishi/index.html |script-title=ja:市域変遷図 |accessdate=2013-12-24 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20131031024832/http://www.city.kobe.lg.jp/information/about/energy/rekishi/index.html |archivedate=2013-10-31 }}</ref>。此後神戶市在1896年、1920年、1929年、1941年和附近的町村進行合併，市域範圍逐漸擴大。至1941年時，神戶市的市域範圍已經包括除東灘區和灘區部分地區之外所有的今日神戶市的沿海地區<ref name="Kobe Wards" />。1947年，神戶市合併六甲山北側的眾多町村，市域面積也因此由115.05平方公里大幅擴大至390.50平方公里。之後神戶市亦在1950年4月1日、1950年10月10日、1951年、1955年、1958年合併附近町村，加上填海使得土地面積進一步擴大，今日神戶市的面積已經達到552.83平方公里<ref name="Kobe Wards" />。

由於神戶市為政令指定都市，所以得以劃區。神戶市現在管轄有9個區<ref name="Kobe Population1" />。

== 政治 ==
[[File:Kobe_City_Hall_Kobe01-r.jpg|thumbnail]]
神戶市政治版圖頗為多樣化，日本各主要政黨均能在神戶獲得一定支持率，這也使得神戶的選舉區成為日本歷次選舉中選戰最激烈的選舉區之一。在日本[[眾議院_(日本)|眾議院]]選舉仍然實施中選舉區制的時期，神戶市全市均屬於{{link-ja|兵庫縣第1區 (中選舉區)|兵庫県第1区 (中選挙区)|兵庫縣第1區}}。兵庫縣第1區的歷次選舉結果均呈現[[自由民主黨_(日本)|自民黨]]、[[公明黨|公明黨]]等右派政黨和[[日本社會黨|社會黨]]、[[日本共產黨|共產黨]]等左派勢力瓜分議席的格局。在日本眾議院選舉改制為小選舉區制之後，神戶市的大部分地區分屬[[兵庫縣第1區|兵庫縣第1區]]、[[兵庫縣第2區|第2區]]（2017年以後[[西宮市|西宮市]]北部亦屬該區）、[[兵庫縣第3區|第3區]]；西區則和北播地區的[[西脇市|西脇市]]、[[三木市|三木市]]、[[小野市|小野市]]、[[加西市|加西市]]、[[加東市|加東市]]、[[多可町|多可町]]共同組成[[兵庫縣第4區|兵庫縣第4區]]。如同中選舉區時期，這些選舉區也多是選舉的激戰區，右派政黨和左派政黨互有勝負。在[[民主党_(日本1998年)|民主黨]]取得大勝的[[2009年日本眾議院選舉|2009年日本眾議院選舉]]時，民主黨包攬了這四個選舉區的議席<ref>{{cite web |url=http://www.city.kobe.lg.jp/information/senkyo/data/img/01-H210830_shuugiin_kekka.pdf |script-title=ja:平成21年衆議院議員総選挙の結果調 |accessdate=2013-12-25 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204813/http://www.city.kobe.lg.jp/information/senkyo/data/img/01-H210830_shuugiin_kekka.pdf |archivedate=2014-01-04 }}</ref>。而在[[2012年日本眾議院選舉|2012年日本眾議院選舉]]時，保守派的自民黨和公明黨又在這四個選舉區都取得勝利<ref>{{cite web |url=http://www.city.kobe.lg.jp/information/senkyo/data/img/01-H241216_shuugiin_kekka.pdf |script-title=ja:平成24年衆議院議員総選挙の結果調 |accessdate=2013-12-25 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204735/http://www.city.kobe.lg.jp/information/senkyo/data/img/01-H241216_shuugiin_kekka.pdf |archivedate=2014-01-04 }}</ref>。[[第48屆日本眾議院議員總選舉|2017年日本眾議院選舉]]時，自民黨在兵庫1區、3區、4區獲勝、公明黨在兵庫2區取得議席<ref>{{cite web |url=http://www.yomiuri.co.jp/election/shugiin/2017/kaihyou/ya28.html|script-title=ja:【小選挙区】兵庫県 |accessdate=2017-10-24 |publisher={{lang|ja|読売新聞}} |language=ja}}</ref>。

在地方自治層面，神戶市的市長選舉的一大特徵是自1949年以來，歷任神戶市長的當選者均曾擔任神戶市副市長<ref>{{cite web |url=http://sankei.jp.msn.com/west/west_affairs/news/131027/waf13102723350018-n1.htm }}</ref>。在1973年的市長選舉中，因自民黨撤回對時任市長{{link-ja|宮崎辰雄|宮崎辰雄}}的支持，加上之後宮崎辰雄當選，使得神戶也和日本眾多大都市相同，一時成為由社會黨、共產黨等革新派政黨執政的{{link-ja|革新自治體|革新自治体}}。然而宮崎辰雄原系保守派人士，加上之後自民黨重新加入神戶的執政黨之列，使得神戶市政一度出現市議會內主要政黨皆為執政黨的局面<ref>{{cite |archiveurl=https://web.archive.org/web/20171024154447/https://www.kobe-np.co.jp/news/senkyo/2017/kobesityou/news/201710/0010666604.shtml |archivedate=2017-10-24 == 經濟 ==
神戶市在日本經濟中擁有重要地位。2010年，神戶市內生產總值約為62,178億日元，比上一年名義增加1.9%<ref name="data">{{cite web |url=http://www.city.kobe.lg.jp/information/data/statistics/toukei/keizai/data/26kk_kiji.pdf |script-title=ja:平成26年度神戸市民経済計算 |accessdate=2017-10-24 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171025022534/http://www.city.kobe.lg.jp/information/data/statistics/toukei/keizai/data/26kk_kiji.pdf |archivedate=2017-10-25 }}</ref>。神戶市的經濟總量佔日本經濟總量的約1.3%<ref name="Kobe Economic">{{cite web |url=http://www.city.kobe.lg.jp/information/project/industry/examination/img/chapter1.pdf |script-title=ja:神戸の経済2012 |accessdate=2013-12-26 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140815142755/http://www.city.kobe.lg.jp/information/project/industry/examination/img/chapter1.pdf |archivedate=2014-08-15 }}</ref>。2009年，神戶市人均所得約為290萬9000日元，高於日本平均水準<ref name="Kobe Economic" />。和日本其他都市類似，農業生產只佔神戶市經濟活動中很小的比重。神戶市生產總值中農業產值僅佔約0.1%<ref name="Kobe Economic" />。與農業相對的是，工業生產佔神戶經濟活動總量的比重達18.5%，高於日本全國平均的17.8%，更大幅高於大都市的平均水準10.8%<ref name="Kobe Economic" />。服務業則是神戶經濟中比重最大的產業，其中又以零售業的產值最高<ref name="Kobe Economic" />。
=== 農業 ===
神戶市的農業屬於都市近郊型農業，生產規模較小且產品主要出售給是神戶市。神戶的主要農業區是市北部和西部地區，北部地區的主要農產品是[[大米|大米]]，而西部地區則是葉菜和果菜類<ref name="Kobe Agriculture">{{cite web |url=http://www.city.kobe.lg.jp/information/project/industry/vision2010/img/2015s04dai2.pdf |script-title=ja:神戸の農漁業の現状 |accessdate=2013-12-26 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140104204628/http://www.city.kobe.lg.jp/information/project/industry/vision2010/img/2015s04dai2.pdf |archivedate=2014-01-04 |title=存档副本 }}</ref>。神戶農產品中最為重要的是肉牛和[[乳牛|乳牛]]，其次則是蔬菜和大米。[[神戶牛|神戶牛]]位列日本三大和牛之列，是高檔牛肉的代名詞<ref>{{cite web |url=http://www.city.kobe.lg.jp/life/health/lab/meat/topcs/m0404.html |script-title=ja:神戸ビーフとは |accessdate=2013-12-25 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140104204820/http://www.city.kobe.lg.jp/life/health/lab/meat/topcs/m0404.html |archivedate=2014-01-04 }}</ref>。神戶的主要漁業活動為漁船漁業，市內有垂水、鹽屋、舞子三處漁港<ref>{{cite web |url=http://www1.kaiho.mlit.go.jp/KAN5/siryouko/naname/kobe/harima_gyoko.htm |script-title=ja:兵庫県（瀬戸内海）の漁港 |accessdate=2013-12-25 |publisher={{lang|ja|海上保安庁管区海洋情報部}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20130526163012/http://www1.kaiho.mlit.go.jp/KAN5/siryouko/naname/kobe/harima_gyoko.htm |archivedate=2013-05-26 }}</ref>。
=== 工業 ===
[[File:Kawasaki_HI_Hyogo_factory.jpg|thumb]]
在神戶開港之後，造船等港口相關產業即開始發展。2014年，神戶市工業品生產總值有28,318億日元<ref name="data"></ref>。神戶戰前的主要工業是造船業和化學產業<ref name="Chishi">{{cite book |author={{lang|ja|石川義孝・金田章裕}}|script-title=ja:《日本の地誌8　近畿圏》 |year=2006 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=4-254-16768-7 |language=ja}}</ref>{{rp|233}}，戰後更發展為日本屈指可數的重工業都市。神戶的工業大多集中在沿海地區和內陸的西神工業園區，沿海地區的工業多以鋼鐵、機械等重工業為主，而西神工業園區則聚集了不少公司的研發中心和高科技企業。就生產總值來看，食品飲料、交通運輸機械、化學、鋼鐵等產業是神戶工業中的主要產業。神戶是日本最早接觸西洋服裝的城市之一，這使得服裝、製鞋、珍珠加工等時尚相關產業也在神戶工業中擁有重要地位<ref name="Kobe Economic" />。總部設在神戶的知名工業企業企業則有[[川崎重工業|川崎重工業]]<ref>{{cite web |url=http://www.khi.co.jp/company/base/office_jp/kobe.html |script-title=ja:神戸本社 |accessdate=2013-12-26 |publisher={{lang|ja|川崎重工業}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104205327/http://www.khi.co.jp/company/base/office_jp/kobe.html |archivedate=2014-01-04 }}</ref>、[[神戶製鋼所|神戶製鋼所]]<ref>{{cite web |url=http://www.kobelco.co.jp/about_kobelco/outline/history/index.html |script-title=ja:歷史・歩み |accessdate=2013-12-26 |publisher={{lang|ja|神戸製鋼所}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104212739/http://www.kobelco.co.jp/about_kobelco/outline/history/index.html |archivedate=2014-01-04 }}</ref>、[[UCC上島咖啡|UCC上島咖啡]]<ref>{{cite web |url=http://www.ucc.co.jp/company/info/outline/ |script-title=ja:企業情報 |accessdate=2013-12-26 |publisher={{lang|ja|UCC上島珈琲}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204605/http://www.ucc.co.jp/company/info/outline/ |archivedate=2014-01-04 }}</ref>等。也有[[寶潔|寶潔]]<ref>{{cite web |url=http://jp.pg.com/about/profile02.jsp |script-title=ja:日本での事業展開 |accessdate=2013-12-26 |publisher=P＆G Japan |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20131226150511/http://jp.pg.com/about/profile02.jsp |archivedate=2013-12-26 }}</ref>等一些外資企業將其日本總部設在神戶。

== 文化 ==
=== 方言 ===
{{link-ja|神戶方言|神戸弁}}和{{link-ja|大阪方言|大阪弁}}、[[京都方言|京言葉]]並為關西地區具代表性的[[日本語言|方言]]，然而由於神戶方言和其他關西方言差異頗大，因此也有觀點認為神戶方言應劃至播州方言<ref>{{cite book |author={{lang|ja|崎山昌廣}}|script-title=ja:《神戸学》 |year=2006 |publisher={{lang|ja|神戸新聞総合出版センター}} |location=神戸市 |isbn=9784343003539 |language=ja}}</ref>{{rp|127-131}}。神戶方言有使用{{lang|ja|テヤ}}敬語體系（使用「{{lang|ja|て}}」來表示尊敬之意，而大阪則多使用「{{lang|ja|はる}}」），區分進行態和完成態等特點。較為知名的神戶方言有「{{lang|ja|しっとう}}」（意為「知道」）、「{{lang|ja|山側、浜側}}」（{{lang|ja|山側}}意為北方，{{lang|ja|浜側}}意為南方）等<ref name="KobeChiikigaku">{{cite book |author={{lang|ja|川越栄子}}|script-title=ja:《神戸地域学　神戸の魅力再発見！》 |year=2012 |publisher={{lang|ja|大学教育出版}} |location=岡山市 |isbn=978-4-86429-111-8 |language=ja}}</ref>{{rp|82-85}}。
=== 文學 ===
神户在《[[源氏物語|源氏物語]]》<ref name="KobeChiikigaku" />{{rp|88-91}}、《[[枕草子|枕草子]]》等日本古代文學作品中就已经出现。近代以後，獨特的異國風情使得许多作家選擇神戶作為文學作品的故事舞台。以神戶為故事舞台的文學作品有[[谷崎潤一郎|谷崎潤一郎]]的《{{link-ja|貓和莊造和兩個女人|猫と庄造と二人のをんな}}》<ref name="KobeChiikigaku" />{{rp|91-93}}、《{{link-ja|細雪|細雪}}》；[[野坂昭如|野坂昭如]]的《[[螢火蟲之墓|螢火蟲之墓]]》；[[村上春樹|村上春樹]]的《[[挪威的森林_(小说)|挪威的森林]]》等<ref>{{cite web |url=http://www.kobe-u.ac.jp/info/public-relations/magazine/style/17/dormitory-history.pdf |script-title=ja:学生寮の今昔 |accessdate=2013-12-27 |publisher={{lang|ja|神戸大学}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140104205642/http://www.kobe-u.ac.jp/info/public-relations/magazine/style/17/dormitory-history.pdf |archivedate=2014-01-04 }}</ref>。而文學也在神戶的戰後復興和震後復興中發揮了重要的作用<ref>{{cite web |url=http://www.kobe-np.co.jp/info/bungakukan/ |script-title=ja:館の概要 |accessdate=2013-12-27 |publisher={{lang|ja|神戸文学館}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20131208080138/http://www.kobe-np.co.jp/info/bungakukan/ |archivedate=2013-12-08 }}</ref>。

=== 藝術 ===

神戶是日本最早接觸近代西洋文明的城市之一，因此神戶的美術也充滿國際性和開放的特質。二戰之後，神戶的洋畫家創建了神戶洋畫會，在日本洋畫界有著重要地位。神戶出身的知名美術家有{{link-ja|金山平三|金山平三}}、{{link-ja|川西英|川西英}}<ref>{{cite web |archiveurl=https://web.archive.org/web/20150416180224/http://www.city.kobe.lg.jp/information/public/online/hyakkei/profile/ |archivedate=2015-04-16 }}</ref>、{{link-ja|小磯良平|小磯良平}}<ref>{{cite web === 飲食 ===
神戶的飲食文化受到外國的強烈影響，[[西餐|西餐]]和[[中國菜|中餐]]在神戶的飲食文化中有重要的地位。神戶是日本最早普及肉食文化的都市之一，这不仅因为神戶比日本其他都市更早接受西餐文化，也和神戶是高檔牛肉品种神戶牛的產地有關。神戶中餐館最為聚集的地區是[[南京町|南京町]]。神戶的西洋糕點產業特別發達。自開港至二戰期間，有很多外國人糕點師聚集在神戶。神戶的糕點文化得以普及<ref>{{cite web |url=http://www.ucc.jp/enjoy/column/column.php?cc=46 |script-title=ja:WE LOVE SWEETS！ ～港町神戸と洋菓子文化～ |accessdate=2013-12-28 |publisher={{lang|ja|UCC上島珈琲}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20131227174823/http://www.ucc.jp/enjoy/column/column.php?cc=46 |archivedate=2013-12-27 |title=存档副本 }}</ref>。现今神戶每年舉辦糕點製作大賽，眾多日本糕點企業的總部都設在神戶<ref name="KobeChiikigaku" />{{rp|52-55}}。神戶東部的東灘區和灘區是重要的[[日本酒|日本酒]]產地<ref>{{cite web |url=http://www.nadagogo.ne.jp/siryo/siryo.html |script-title=ja:灘五郷資料館 |accessdate=2013-12-28 |publisher={{lang|ja|灘五郷酒造組合}} |language=ja }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

=== 媒體 ===
神戶新聞創刊於1898年2月11日<ref name="Kobenp">{{cite web |url=http://club.kobe-np.co.jp/corporate/coporate_syasi.html |script-title=ja:社是・社史 |accessdate=2013-12-28 |publisher={{lang|ja|神戸新聞}} |language=ja }}</ref>，是神戶市唯一的一份地方報紙。在阪神大地震期間，神戶新聞社的總部大樓全毀，然神戶新聞社仍借用[[京都新聞|京都新聞]]的設施堅持出版，並未休刊，並且在震後神戶新聞持續對阪神大地震給神戶帶來的變化進行追踪報導，成為日本報業界的佳話<ref name="Kobenp" />。神戶新聞的發行範圍包括了兵庫縣全縣，在神戶市及其近郊地區是市場佔有率第一的報紙<ref>{{cite web |url=http://www.kobe-np.co.jp/ad-data/contents/pdf/2012ad-data.pdf |script-title=ja:広告のご案内 |accessdate=2013-12-28 |publisher={{lang|ja|神戸新聞}} |language=ja |deadurl=no name="KobeChiikigaku" />{{rp|56-59}}。以神戶市為總部的[[電視台|電視台]]有兩家，分別是[[SUN電視台|SUN電視台]]<ref name="SUN-TV">{{cite web|url=http://www.sun-tv.co.jp/company/index.html|script-title=ja:サンテレビの歩み|accessdate=2013-12-28|publisher={{lang|ja|サンテレビ}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20130707160936/http://www.sun-tv.co.jp/company/index.html|archivedate=2013-07-07}}</ref>，其播出範圍包括了兵庫縣和[[大阪府|大阪府]]全境<ref>{{cite 神戶是日本最早上映[[電影|電影]]的都市之一<ref>{{cite web |url=http://www.nikkei.com/article/DGXZZO43148210Z20C12A6000000/ |script-title=ja:大阪・神戸・京都に「日本映画発祥の碑」　本当はどこ？ |accessdate=2013-12-28 |publisher={{lang|ja|日本経済新聞}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204249/http://www.nikkei.com/article/DGXZZO43148210Z20C12A6000000/ |archivedate=2014-01-04 }}</ref>，[[美利堅公園|美利堅公園]]建有電影紀念碑，更顯出神戶在日本電影史擁有特別地位<ref>{{cite web|url=http://www.city.kobe.lg.jp/information/institution/institution/library/furusato/eiga_kinenhi.html|script-title=ja:神戸を知る　映画記念碑|accessdate=2013-12-28|publisher={{lang|ja|神戸市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204542/http://www.city.kobe.lg.jp/information/institution/institution/library/furusato/eiga_kinenhi.html|archivedate=2014-01-04}}</ref>。[[新開地|新開地]]地區在過去電影院鱗次櫛比，是日本電影文化的一大中心<ref>{{cite web|url=http://www.shinkaichi.or.jp/outline/history.html|script-title=ja:「新開地」まちの変遷|accessdate=2013-12-28|publisher={{lang|ja|新開地オモシロ情報サイト}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140104211008/http://www.shinkaichi.or.jp/outline/history.html|archivedate=2014-01-04|title={title}}}</ref>。神戶市極為重視映像產業的發展，積極吸引電影和電視劇在神戶拍攝。在神戶拍攝的著名電影包括了{{link-en|櫻花戀|Sayonara}}、[[鐵金剛勇破火箭嶺|鐵金剛勇破火箭嶺]]、[[華麗一族|華麗一族]]、[[黑雨_(美國電影)|黑雨]]、[[寅次郎的故事_寅次郎紅之花|寅次郎的故事 寅次郎紅之花]]、[[哥吉拉_最後戰役|哥吉拉 最後戰役]]、[[梦比优斯奥特曼&奥特兄弟|梦比优斯奥特曼&奥特兄弟]]、[[新宿事件|新宿事件]]、[[極惡非道|極惡非道]]、[[殺戮都市|殺戮都市]]、[[挪威的森林_(電影)|挪威的森林]]和[[阪急電車_(小說)|阪急電車 單程15分鐘的奇蹟]]等作品<ref name="KobeChiikigaku" />{{rp|65-79}}。神戶每年還舉辦新開地電影節等電影文化活動<ref>{{cite web|url=http://eigasai.shinkaichi.or.jp/concept.html|script-title=ja:映画祭について|accessdate=2013-12-28|publisher={{lang|ja|新開地映画祭}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140104210834/http://eigasai.shinkaichi.or.jp/concept.html|archivedate=2014-01-04}}</ref>。

=== 體育 ===
[[File:Inside_View_of_Kobe_Wing_Stadium.jpg|thumb]]
神戶最受歡迎的職業體育運動是[[棒球|棒球]]和[[足球|足球]]。雖然並無職業棒球球隊將其主場設在神戶市內，然而關西的兩大職業棒球球隊[[阪神虎|阪神虎]]和[[歐力士野牛|歐力士野牛]]均和神戶市有密切的關係。其中歐力士野牛更在1991年至2004年期間將主場設在神戶。現在神戶綜合運動公園棒球場也仍是歐力士野牛的主場球場之一，每年均有數場職棒賽事在這裡舉行<ref>{{cite web|url=http://www.buffaloes.co.jp/stadium/|script-title=ja:スタジアム|accessdate=2013-12-28|publisher={{lang|ja|オリックス・バファローズ}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140208003736/http://buffaloes.co.jp/stadium/|archivedate=2014-02-08}}</ref>。[[神戶勝利船|神戶勝利船]]是神戶唯一的男子職業足球隊，現在參加[[J聯賽|J聯賽]]的賽事<ref>{{cite web|url=http://www.vissel-kobe.co.jp/club/|script-title=ja:クラブ概要|accessdate=2013-12-28|publisher={{lang|ja|ヴィッセル神戸}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140118025052/http://www.vissel-kobe.co.jp/club/|archivedate=2014-01-18}}</ref>。[[INAC神戶雌獅|INAC神戶雌獅]]是日本女子職業足壇的豪強，曾多次獲得日本女足聯賽的冠軍<ref>{{cite web|url=http://inac-kobe.com/club_info/history.html|script-title=ja:ヒストリー|accessdate=2013-12-28|publisher={{lang|ja|INAC神戸}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104210937/http://inac-kobe.com/club_info/history.html|archivedate=2014-01-04}}</ref>。兩家球隊的主場都設在御崎公園球技場。神戶也曾多次舉辦國際體育賽事，其中最為著名的當屬1985年世界大學生運動會和[[2002年世界盃足球賽|2002年世界盃足球賽]]。御崎公園球技場就是為承辦2002年世界盃而興建，在這裡舉辦了3場世界盃的賽事<ref>{{cite web|url=http://www.noevir-stadium.jp/guide/summary/|script-title=ja:スタジアム概要|accessdate=2013-12-28|publisher={{lang|ja|ノエビアスタジアム神戸}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131119090851/http://www.noevir-stadium.jp/guide/summary/|archivedate=2013-11-19}}</ref>。神戶市每年均舉辦神戶馬拉松等普通市民也能參加的體育活動<ref>{{cite web|url=http://www.kobe-marathon.net/subject/index.html|script-title=ja:大会テーマ|accessdate=2013-12-28|publisher={{lang|ja|神戸マラソン}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140104210755/http://www.kobe-marathon.net/subject/index.html|archivedate=2014-01-04}}</ref>。

== 教育 ==
[[File:Kobe-Univ-Rokkodai-Honkan.jpg|thumb]]
神戶市的教育水準發達。2012年時，神戶市內有150所幼稚園、172所小學、102所初中、56所高中和20所大學<ref>{{cite web|url=http://www.city.kobe.lg.jp/information/data/statistics/toukei/gakkou/data/25gakkou_kiji.pdf|script-title=ja:平成25年度学校基本調査|accessdate=2013-12-28|publisher={{lang|ja|神戸市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204703/http://www.city.kobe.lg.jp/information/data/statistics/toukei/gakkou/data/25gakkou_kiji.pdf|archivedate=2014-01-04}}</ref>。神戶的大學中以[[神戶大學|神戶大學]]的教學和研究水準最高。神戶大學創建於1949年，系由神戶經濟大學、姬路高等學校、神戶工業專門學校和兵庫師範學校等學校合併創建<ref>{{cite web|url=http://www.kobe-u.ac.jp/info/outline/history/zentai.html|script-title=ja:沿革・歷史|accessdate=2013-12-28|publisher={{lang|ja|神戸大学}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104205637/http://www.kobe-u.ac.jp/info/outline/history/zentai.html|archivedate=2014-01-04}}</ref>。現在的神戶大學已經發展為擁有11個學院，超過16,000人學生和3,000人教職員的大型綜合性大學<ref>{{cite web|url=http://www.kobe-u.ac.jp/info/outline/outline/index.html|script-title=ja:神戸大学概覧|accessdate=2013-12-28|publisher={{lang|ja|神戸大学}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104205649/http://www.kobe-u.ac.jp/info/outline/outline/index.html|archivedate=2014-01-04}}</ref>。在科研成果方面，神戶大學是日本除了舊[[帝國大學|帝國大學]]之外最頂尖的大學之一。2012年[[諾貝爾醫學或生理學獎|諾貝爾醫學或生理學獎]]得主[[山中伸彌|山中伸彌]]就畢業於神戶大學<ref>{{cite web |url=http://www.kobe-u.ac.jp/topics/top/t2012_10_09_01.html |script-title=ja:本学卒業生の山中伸弥教授にノーベル生理学・医学賞 ～受賞を祝い医学部で記者会見～ |accessdate=2013-12-28 |publisher={{lang|ja|神戸大学}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140104205635/http://www.kobe-u.ac.jp/topics/top/t2012_10_09_01.html |archivedate=2014-01-04 }}</ref>。神戶大學的研究團隊也在[[希格斯玻色子|希格斯玻色子]]的研究中做出了重要貢獻<ref>{{cite web|url=http://www.kobe-u.ac.jp/topics/top/t2013_10_08_02.html|script-title=ja:2013年ノーベル物理学賞に「ヒッグス粒子」決定でATLAS実験に寄与した神戸大グループが記者会見|accessdate=2013-12-28|publisher={{lang|ja|神戸大学}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140104205640/http://www.kobe-u.ac.jp/topics/top/t2013_10_08_02.html|archivedate=2014-01-04}}</ref>。神戶的私立大學中則是以創立於1919年的[[甲南大學|甲南大學]]規模最大<ref>{{cite web|url=http://www.konan-u.ac.jp/gakuen/ayumi/index.html|script-title=ja:学園のあゆみ|accessdate=2013-12-28|publisher={{lang|ja|甲南大学}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204445/http://www.konan-u.ac.jp/gakuen/ayumi/index.html|archivedate=2014-01-04}}</ref>。除了這兩所大學之外，神戶規模較大的大學還有[[兵庫縣立大學|兵庫縣立大學]]和[[神戶市外國語大學|神戶市外國語大學]]等。為了促進各大學之間的學術交流並對地域振興做出貢獻<ref>{{cite web|url=http://www.consortium-hyogo.jp/setsuritsu/index.html|script-title=ja:設立趣旨|accessdate=2013-12-28|publisher={{lang|ja|大学コンソーシアムひょうご神戸}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104234341/http://www.consortium-hyogo.jp/setsuritsu/index.html|archivedate=2014-01-04}}</ref>，神戶市和[[西宮市|西宮市]]的部分大學在2006年設立了兵庫神戶大學聯盟（{{lang|ja|大学コンソーシアムひょうご神戸}}）<ref>{{cite web|url=http://www.consortium-hyogo.jp/setsuritsu/keii.html|script-title=ja:設立経緯|accessdate=2013-12-28|publisher={{lang|ja|大学コンソーシアムひょうご神戸}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104210836/http://www.consortium-hyogo.jp/setsuritsu/keii.html|archivedate=2014-01-04}}</ref>，加盟校的學生可聽講其他加盟校的課程並取得學分。神戶市也是日本科研界的重鎮。包括理化學研究所計算科學研究機構在內的一些科研機構和日本的[[超級電腦|超級電腦]][[京_(超級電腦)|京]]就位於神戶<ref>{{cite web|url=http://www.aics.riken.jp/jp/access/access.html|script-title=ja:アクセス|accessdate=2013-12-30|publisher={{lang|ja|独立行政法人理化学研究所計算科学研究機構}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204102/http://www.aics.riken.jp/jp/access/access.html|archivedate=2014-01-04}}</ref>。

== 觀光 ==
[[File:Arima_Grand_Hotel01s4s4272.jpg|thumb]]
神戶是日本重要的觀光都市。六甲山的自然景觀和有馬溫泉度假區、充滿異國風情的市區、港灣景觀是神戶最重要的觀光資源。[[六甲山|六甲山]]雖位於都市近郊，卻保有豐富的自然景觀。早在開港初期，歐美人就在這裡修建別莊和高爾夫球場<ref>{{cite web|url=http://www.rokkosan.com/theme/|script-title=ja:行こう！六甲山|accessdate=2013-12-28|publisher={{lang|ja|六甲山ポータルサイト}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140123123404/http://www.rokkosan.com/theme/|archivedate=2014-01-23}}</ref>，開始將六甲山建設為日本最早的都市近郊型度假地。六甲山還是日本近代登山運動的發祥地<ref>{{cite web |url=http://www.hyogo-tourism.jp/tabinet/furusato100/meizan/rokkousan/ |script-title=ja:ひょうご旅ネット - 六甲山 |accessdate=2013-12-28 |publisher={{lang|ja|ひょうごツーリズムガイド}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104205635/http://www.hyogo-tourism.jp/tabinet/furusato100/meizan/rokkousan/ |archivedate=2014-01-04 }}</ref>。今日的六甲山也仍然是深受神戶市民喜愛的郊遊地。而位於六甲山地北側的[[有馬溫泉|有馬溫泉]]是日本三大溫泉之一，也是日本最早得到開發的溫泉療養地，在古代就有眾多達官貴人和文人墨客到訪這裡<ref name="KobeChiikigaku" />{{rp|131-140}}。今日的有馬溫泉亦憑藉其靠近大都市圈的優越地理位置吸引眾多遊客到訪。2012年時，有馬溫泉的遊客數達到152萬人<ref name="Tourist" />。神戶市內最能體驗異國風情的地區則是集中眾多西洋建築的北野町山本通、舊居留地，以及[[唐人街|唐人街]][[南京町|南京町]]。北野町山本通的異人館中，以風見雞館最為聞名，其屋頂的雞型風向標現在已經成為神戶的象徵<ref name="Nikkei" />{{rp|162-165}}。雖然阪神大地震使得北野的不少異人館嚴重毀壞，然而受損的建築之後大多按照原樣忠實重建。北野的一些洋館現在更被改建為餐廳或者咖啡廳<ref>{{cite web|url=http://www.starbucks.co.jp/store/concept/kobe/|script-title=ja:神戸北野異人館店|accessdate=2014-01-23|publisher={{lang|ja|スターバックス}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140202102422/http://www.starbucks.co.jp/store/concept/kobe/|archivedate=2014-02-02}}</ref>，成為熱門的觀光景點。南京町則和[[橫濱中華街|橫濱中華街]]、[[長崎新地中華街|長崎新地中華街]]並列日本三大中華街之一。在神戶開港之初，因清朝並未和日本締結條約，使得中國人無法居住在居留地內而多聚居在靠近居留地的南京町<ref>{{cite web|url=http://www.nankinmachi.or.jp/about/taizen/|script-title=ja:南京町大全|accessdate=2013-12-28|publisher={{lang|ja|南京町}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20131125183502/http://www.nankinmachi.or.jp/about/taizen/|archivedate=2013-11-25}}</ref>。南京町聚集了眾多中餐館，並且在[[春節|春節]]等中國傳統節日舉辦節慶活動<ref>{{cite web|url=http://www.nankinmachi.or.jp/about/event/|script-title=ja:年中行事|accessdate=2014-01-23|publisher={{lang|ja|南京町}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140201203250/http://www.nankinmachi.or.jp/about/event/|archivedate=2014-02-01}}</ref>。{{link-ja|神戶港灣區|神戸ハーバーランド}}、[[美利堅公園|美利堅公園]]、[[神戶港塔|神戶港塔]]和[[神戶海洋博物館|神戶海洋博物館]]則是神戶港一帶地區最著名的旅遊景點。神戶港灣區是神戶重要的都市更新區域之一，倉庫等部分昔日港口設施現在也改建為觀光景點，使得昔日的工業區呈現出完全不同於過往的氣氛，成為港灣都市神戶獨具特色的旅遊資源<ref>{{cite web|url=http://www.harborland.co.jp/guide/|script-title=ja:ハーバーランドガイド|accessdate=2014-01-23|publisher={{lang|ja|神戸ハーバーランド}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140201200626/http://www.harborland.co.jp/guide/|archivedate=2014-02-01}}</ref>。美利堅公園<ref>{{cite web|url=http://www.feel-kobe.jp/area_guide/meriken/index.html|script-title=ja:メリケンパーク/ハーバーランド|accessdate=2014-01-23|publisher={{lang|ja|神戸国際観光コンベンション協会}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104235014/http://www.feel-kobe.jp/area_guide/meriken/index.html|archivedate=2014-01-04}}</ref>緊鄰神戶港灣區，內有{{link-ja|魚舞|フィッシュ・ダンス}}、繡球花之鐘（{{lang|ja|オルタンシアの鐘}}）等雕塑和電影紀念碑、移民紀念碑等象徵性紀念物。公園東側的部分地區在阪神淡路大地震之後並未重建而是保留原貌，是記錄地震的重要記憶遺產。[[神戶港塔|神戶港塔]]<ref>{{cite web|url=http://www.kobe-meriken.or.jp/port-tower/question.html|script-title=ja:神戸ポートタワーって？|accessdate=2014-01-23|publisher={{lang|ja|神戸ポートタワー}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20130717101639/http://www.kobe-meriken.or.jp/port-tower/question.html|archivedate=2013-07-17}}</ref>和[[神戶海洋博物館|神戶海洋博物館]]<ref>{{cite web|url=http://www.kobe-meriken.or.jp/maritime-museum/about.html|script-title=ja:神戸海洋博物館について|accessdate=2013-12-28|publisher={{lang|ja|神戸海洋博物館}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140119024256/http://www.kobe-meriken.or.jp/maritime-museum/about.html|archivedate=2014-01-19}}</ref>位於神戶港灣區和美利堅公園之間，兩座建築均因其獨特的外形成為神戶的地標。

{{cite web|url=http://www.arima-toys.jp/floors.html|script-title=ja:フロアマップ|accessdate=2014-01-23|publisher={{lang|ja|有馬玩具博物館}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131218231303/http://arima-toys.jp/floors.html|archivedate=2013-12-18}}
</ref>、{{link-ja|有馬郵票文化博物館|有馬切手文化博物館}}<ref>{{cite web|url=http://www.kitte-museum-arima.jp/tatemono/index.html|script-title=ja:当館について|accessdate=2014-01-23|publisher={{lang|ja|有馬切手文化博物館}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140201180208/http://www.kitte-museum-arima.jp/tatemono/index.html|archivedate=2014-02-01}}</ref>、{{link-ja|神戶時尚美術館|神戸ファッション美術館}}、[[兵庫縣立美術館|兵庫縣立美術館]]、[[魚鱗之家|魚鱗之家美術館]]<ref>{{cite web|url=http://kobe-ijinkan.net/index.php?id=77|script-title=ja:うろこの家|accessdate=2013-12-28|publisher={{lang|ja|うろこの家}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204308/http://kobe-ijinkan.net/index.php?id=77|archivedate=2014-01-04}}</ref>、[[孫文紀念館|孫文紀念館]]<ref>{{cite web |url=http://sonbun.or.jp/jp/index.php?option=com_content&task=view&id=14&Itemid=31 |script-title=ja:孫文記念館（移情閣）紹介 |accessdate=2013-12-28 |publisher={{lang|ja|孫文記念館}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204351/http://sonbun.or.jp/jp/index.php?option=com_content&task=view&id=14&Itemid=31 |archivedate=2014-01-04 }}</ref>、{{link-ja|UCC咖啡博物館|UCCコーヒー博物館}}<ref>{{cite web|url=http://www.ucc.co.jp/museum/about/|script-title=ja:コーヒー博物館について|accessdate=2013-12-28|publisher={{lang|ja|UCCコーヒー博物館}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204523/http://www.ucc.co.jp/museum/about/|archivedate=2014-01-04}}</ref>等。

==== 巴士 ====
[[神戶市巴士|神戶市巴士]]是神戶市中心地區的主要交通方式之一，由[[神戶市交通局|神戶市交通局]]運營。神戶市巴士的票價分為「普通區」和「近郊區」兩大系統。在普通區內乘車時，票價統一為210日元（兒童為110日元）；若乘車至近郊區時，則需要另外加付票款<ref>{{cite web|url=http://www.city.kobe.lg.jp/life/access/transport/bus/busryoukin/index.html|script-title=ja:市バスの乗車料金|accessdate=2013-12-29|publisher={{lang|ja|神戸市交通局}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140121104233/http://www.city.kobe.lg.jp/life/access/transport/bus/busryoukin/index.html|archivedate=2014-01-21}}</ref>。現在神戶市巴士共有92條路線<ref>{{cite web|url=http://www.city.kobe.lg.jp/life/access/transport/bus/keito.html|script-title=ja:神戸市営バス系統一覧|accessdate=2013-12-29|publisher={{lang|ja|神戸市交通局}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204725/http://www.city.kobe.lg.jp/life/access/transport/bus/keito.html|archivedate=2014-01-04}}</ref>，518輛車輛運行<ref>{{cite web|url=http://www.city.kobe.lg.jp/life/access/transport/bus/syaryou/index.html|script-title=ja:市バス車両について|accessdate=2013-12-29|publisher={{lang|ja|神戸市交通局}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140103031641/http://www.city.kobe.lg.jp/life/access/transport/bus/syaryou/index.html|archivedate=2014-01-03}}</ref>，絕大多數巴士車輛均採用白色和綠色的塗裝。除了神戶市巴士外，還有山陽巴士、神姬巴士、阪急巴士等其他公司的巴士在神戶市運行。神戶市內亦有主要供觀光客乘坐的環行巴士<ref>{{cite web |url=http://feel-kobe.jp/transport/bus/index.html |script-title=ja:神戸市内の交通機関 バス・タクシー |accessdate=2014-01-21 |publisher={{lang|ja|神戸国際観光コンベンション協会}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140203021834/http://feel-kobe.jp/transport/bus/index.html |archivedate=2014-02-03 }}</ref>。

=== 海運 ===
[[File:Kobe_Mosaic06s4s3200.jpg|thumb]]
神戶的歷史和[[神戶港|神戶港]]的發展有著密不可分的關係。神戶港的歷史可追溯至[[奈良時代|奈良時代]]的大輪田泊。自[[鎌倉時代|鎌倉時代]]之後，大輪田泊改稱兵庫津。在[[室町時代|室町時代]]，兵庫津作為[[日明貿易|日明貿易]]的據點曾是一座國際港口。然而由於日本在江戶時代實施[[日本鎖國|鎖國]]政策，兵庫津轉變為國內港口，但仍然擁有重要的地位<ref name="Chishi" />{{rp|243}}。神戶港在1868年開港之後再次發展為國際港口<ref>{{cite book |author={{lang|ja|大国正美}}|script-title=ja:《古地図で見る神戸》 |year=2013 |publisher={{lang|ja|神戸新聞総合出版センター}} |location=神戸市 |isbn=978-4-343-00603-5 |language=ja}}</ref>{{rp|48-51}}。在[[明治|明治]]和[[大正|大正]]時期，神戶港就已經多次實施築港工程。1919年時，日本的貿易額中有34%的都是經由神戶港進行<ref name="Chishi" />{{rp|243}}。戰前的神戶港是日本最大的港口，進口和出口的商品也隨著日本經濟的發展日趨多樣化。當時亦有眾多海運企業將總部設在神戶<ref>{{cite web|url=http://www.ymf.or.jp/wp-content/themes/yamagata/images/56_3.pdf|script-title=ja:神戸港競争力復活へのシナリオ|accessdate=2013-12-30|publisher={{lang|ja|山縣記念財団}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204548/http://www.ymf.or.jp/wp-content/themes/yamagata/images/56_3.pdf|archivedate=2014-01-04}}</ref>。神戶港在二戰期間嚴重受損，戰後神戶港雖然很快就開始復興，然其日本第一大港的地位在1960年代被[[橫濱港|橫濱港]]取代<ref name="Chishi" />{{rp|244}}。雖說如此，神戶港在戰後仍取得了巨大的發展。1967年，神戶港的摩耶碼頭建設了日本首個[[貨櫃碼頭|貨櫃碼頭]]<ref>{{cite web |url=http://www.city.kobe.lg.jp/information/public/online/ksnavi/vol4/question01.html |script-title=ja:まちを元気にする海と空のみなと ～産業を支える神戸港・神戸空港～ 神戸港と産業のつながりは古いの？ |accessdate=2013-12-29 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140104204829/http://www.city.kobe.lg.jp/information/public/online/ksnavi/vol4/question01.html |archivedate=2014-01-04 }}</ref>。在1970年代時，神戶港曾是世界上規模最大的貨櫃港<ref>{{cite web|url=http://www.isfj.net/ronbun_backup/2008/1303.pdf|script-title=ja:神戸を例とした日本港湾への提言|accessdate=2013-12-30|publisher={{lang|ja|ISFJ日本政策学生会議}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140713001707/http://www.isfj.net/ronbun_backup/2008/1303.pdf|archivedate=2014-07-13}}</ref>。然而在1980年代中期之後<ref>{{cite web |url=http://www3.boj.or.jp/kobe/kouhyou/report/report131120.pdf |script-title=ja:神戸港の質的変貌. ～集荷力低下と将来像 |accessdate=2013-12-30 |publisher={{lang|ja|日本銀行神戸支店}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140104204159/http://www3.boj.or.jp/kobe/kouhyou/report/report131120.pdf |archivedate=2014-01-04 }}</ref>，隨著[[新加坡|新加坡]]港等東亞其他港口的迅速崛起，加上阪神大地震給港口設施帶來嚴重打擊，神戶港的地位出現下降<ref>{{cite web |url=http://www.city.kobe.lg.jp/information/public/online/ksnavi/vol4/question02.html |script-title=ja:まちを元気にする海と空のみなと ～産業を支える神戸港・神戸空港～ 世界から見た神戸港は？ |accessdate=2013-12-29 |publisher={{lang|ja|神戸市}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140104204908/http://www.city.kobe.lg.jp/information/public/online/ksnavi/vol4/question02.html |archivedate=2014-01-04 }}</ref>。為了重振神戶港的競爭力，日本政府將神戶港和其鄰近的[[大阪港|大阪港]]合併指定為超級中樞港灣<ref>{{cite web|url=http://www.city.kobe.lg.jp/information/project/port/port/supergatewayport.html|script-title=ja:スーパー中枢港湾の実現に向けた取り組みについて|accessdate=2013-12-29|publisher={{lang|ja|神戸市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204713/http://www.city.kobe.lg.jp/information/project/port/port/supergatewayport.html|archivedate=2014-01-04}}</ref>及國際貨櫃戰略港灣<ref>{{cite web|url=http://www.city.kobe.lg.jp/information/project/port/port/senryakukowan/|script-title=ja:阪神港が「国際コンテナ戦略港湾」に選定されました|accessdate=2013-12-29|publisher={{lang|ja|神戸市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204921/http://www.city.kobe.lg.jp/information/project/port/port/senryakukowan/|archivedate=2014-01-04}}</ref>。希冀能通過降低港口使用費、擴充港口設施等方法提振神戶港的競爭力。神戶港不僅是貨運港，也是日本重要的客運港。神戶港建有日本最大的客船用碼頭<ref>{{cite web|url=http://www.city.kobe.lg.jp/culture/leisure/harbor/passenger/terminal/index.html|script-title=ja:ターミナル紹介|accessdate=2013-12-29|publisher={{lang|ja|神戸市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204652/http://www.city.kobe.lg.jp/culture/leisure/harbor/passenger/terminal/index.html|archivedate=2014-01-04}}</ref>，有眾多大型[[郵輪|郵輪]]停靠神戶港<ref>{{cite web|url=http://www.city.kobe.lg.jp/culture/leisure/harbor/passenger/introduction/index.html|script-title=ja:クルーズ客船情報|accessdate=2013-12-29|publisher={{lang|ja|神戸市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204911/http://www.city.kobe.lg.jp/culture/leisure/harbor/passenger/introduction/index.html|archivedate=2014-01-04}}</ref>。神戶港亦開通有前往[[上海市|上海]]等地的定期客船航線。近年神戶市將神戶港的西半部地區劃為親水區域，期望能開發觀光和文化等新的港口價值並讓神戶港成為更易被市民和觀光客所親近的港灣<ref>{{cite web|url=http://www.city.kobe.lg.jp/information/project/port/plan/img/p-00-15.pdf|script-title=ja:みなと神戸－いきいきプラン|accessdate=2013-12-30|publisher={{lang|ja|神戸市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204943/http://www.city.kobe.lg.jp/information/project/port/plan/img/p-00-15.pdf|archivedate=2014-01-04}}</ref>。
=== 空運 ===
[[神戶機場|神戶機場]]建設在港灣人工島南面海域的人工島上，是日本第三座市營機場，在2006年開業<ref>{{cite web|url=http://www.city.kobe.lg.jp/life/access/airport/index_04.html|script-title=ja:神戸空港の経緯|accessdate=2013-12-29|publisher={{lang|ja|神戸市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140104204925/http://www.city.kobe.lg.jp/life/access/airport/index_04.html|archivedate=2014年1月4日|df=}}</ref>。神戶機場的[[IATA机场代码|機場代碼]]是UKB，擁有一條長2,500米的跑道。機場距離市中心的距離較近，在三宮可乘坐港灣人工島線直達機場<ref>{{cite web|url=http://www.city.kobe.lg.jp/life/access/airport/index_01.html|script-title=ja:神戸空港のプロフィール|accessdate=2013-12-29|publisher={{lang|ja|神戸市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20140104204919/http://www.city.kobe.lg.jp/life/access/airport/index_01.html|archivedate=2014-01-04}}</ref>。現在在神戶機場有航班起降的航空公司包括[[全日空|全日空]]、[[天馬航空|天馬航空]]、[[Air_Do|Air Do]]、[[亞洲天網航空|亞洲天網航空]]。神戶機場開通有至[[札幌市|札幌]]郊區的[[新千歲機場|新千歲機場]]、[[東京國際機場|東京國際機場]]（羽田機場）、[[長崎機場|長崎機場]]、[[鹿兒島機場|鹿兒島機場]]、[[那霸機場|那霸機場]]等機場的航班<ref>{{cite web|url=http://www.city.kobe.lg.jp/life/access/airport/01.html|script-title=ja:就航路線|accessdate=2013-12-29|publisher={{lang|ja|神戸市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140104204927/http://www.city.kobe.lg.jp/life/access/airport/01.html|archivedate=2014-01-04|title=存档副本}}</ref>。

== 國際交往 ==
