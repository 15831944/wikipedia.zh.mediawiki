'''神木車站'''位於[[台灣|台灣]][[嘉義縣|嘉義縣]][[阿里山鄉|阿里山鄉]]，為[[行政院農業委員會林務局|林務局]][[阿里山森林鐵路|阿里山森林鐵路]][[阿里山線|阿里山線]]、神木線之[[鐵路車站|鐵路車站]]。

== 車站構造 ==
站房為一木造車站，[[折返式鐵路|折返式車站]]，海拔2138公尺，位於阿里山鐵路第三分道，[[島式月台|島式月台]]、候車亭各一座。

== 利用狀況 ==
*阿里山線（主線）的停靠站
*阿里山─神木區間車（又稱神木線）
現今只剩下一個股道，另一股道暫做為步道使用

== 車站週邊 ==
*[[阿里山神木|阿里山神木]]
*神木紀念亭

== 歷史 ==
*1914年：設立「御神木驛」。
*時間不詳，改為「神木站」。
