本列表是有關于'''[[神話|神話]]及[[傳說|傳說]]中的[[龍|龍]]'''：

==[[東亞文化圈|東亞文化圈]]==
===[[中國|中國]]===
; 龍
中國龍（或東方龍）是中國神話中的一種傳說生物，它具有蝦眼、鹿角、牛嘴、狗鼻、鯰鬚、獅鬃、蛇尾、鱼鳞和鷹爪的形象，在中國文化信仰中，它是吉祥的象征。[[五爪金龍|五爪金龍]]是皇帝的象征符號。

===[[日本|日本]]===
; {{lang|ja|竜}}
與中國龍相似，有3或4爪，是吉祥的象征，通常出現在河川和瀑布。

===[[朝鮮|朝鮮]]===
; {{lang|ko|용}}、{{lang|ko|龍}}
天龍，具有與中國龍相似的形象，通常與雲霧為伴，是吉祥的象征。

; {{lang|ko|이무기}}
海龍，無角，具有與大海蛇相似的形象。

; {{lang|ko|교}}
山龍。

===[[越南|越南]]===
; {{lang|vi|Rồng}}、{{lang|vi|龍}}
越南龍具有和其他東亞龍相似的形象，特别是突出的眼睛、長鬚、腮鬃、蛇尾和鳞片，但是無角；它口中所含的寶石代表的是和藹、高尚和智慧的象徵。

==[[印度文化圈|印度文化圈]]==
===[[印度|印度]]===
; {{lang|sa|नाग}}
印度龍（或那加）的傳說流傳在興都教的信仰當中，它具有[[眼鏡蛇|眼鏡蛇]]的形象，頭部的數量會隨着等級的高低而不同；在佛教傳說中，它是佛教的八部眾之一。

===[[印尼|印尼]]和[[馬來群島|馬來群島]]===
; {{lang|id|Naga/Nogo}}
印尼龍（或馬來龍）是隨着印度文化的腳步流傳到本地區來的，它具有印度龍相似蛇形象，通常會出現在高山、深林和大海中。

===[[高棉|高棉]]===
; {{lang|km|នាគ}}
高棉龍具有與印度龍相似的形象，頭部的數量會隨着等級的高低而不同，最高級的高棉龍拥有9個頭。

===[[菲律賓|菲律賓]]===
; {{lang|en|Bakunawa}}
菲律賓龍具有與大海蛇相似的形象，傳說每當日食的時候它就會四處做亂。

==[[中東文化圈|中東文化圈]]==
===[[伊朗|伊朗]]===
; {{lang|en|Azi Dahaka}}
阿吉塔哈卡有著三頭六眼三口，背生羽翼、能使上千法術，為伊朗自古流傳的怪物。

===[[土耳其|土耳其]]===
; {{lang|tr|Ejderha, Evren}}
傳說土耳其龍是拥有火炎般尾巴的大蛇。

==[[西方文化圈|西方文化圈]]==
===[[加泰羅尼亞|加泰羅尼亞]]===
; {{lang|ca|Drac}}
加泰羅尼亞龍有象蛇般粗壯的脖子，長着兩條腿、一對翅膀和類似牛或獅子一樣的頭部，鼻子噴出帶有熱氣和毒氣的鼻息，它的鼻息會把身邊的事物侵蝕。雌性的加泰羅尼亞龍稱為{{lang|ca|Víbria}}。

===[[法國|法國]]===
; {{lang|fr|Dragon}}
在法國，龍通常是[[龍騎兵|龍騎兵]]的象征符號。

===[[撒丁尼亚|撒丁尼亚]]===
; {{lang|it|Scultone}}
撒丁尼亚龍通常住在灌木叢并拥有长生不老的生命，傳說它有使用目光就將人類殺死的力量，這和[[翼蜥|翼蜥]]相似。

===[[斯堪的納維亞|斯堪的納維亞]]和[[日爾曼|日爾曼]]===
; {{lang|en|Lindworm}}, {{lang|non|Linnormr}}, {{lang|no|Linnorm}}, {{lang|sv|Lindorm}}, {{lang|de|Lindwurm}}
斯堪的納維亞龍（或日爾曼龍）是北歐神話里的一種傳說生物，有蛇的形象，通常有兩條腿或無腿。也就是[[林德蟲|林德蟲]]。

===[[英格蘭|英格蘭]]===
; {{lang|en|Wyvern}}
英格蘭龍（或[[飛龍_(傳說生物)|飛龍]]）是中世紀傳說中最常見到的龍，一般有兩條腿和一對翅膀。

===[[威爾斯|威爾斯]]===
; {{lang|cy|Y Ddraig Goch}}
威爾斯龍（或[[紅龍_(威爾斯)|紅龍]]）是威爾斯民族的象征符號。

===[[蘇格蘭|蘇格蘭]]===
; {{lang|gd|Bheithir}}
蘇格蘭龍是[[凯爾特神話|凯爾特神話]]中的一種傳說生物。

===[[匈牙利|匈牙利]]===
; {{lang|hu|Zomok}}
住在沼泽地的一種大蛇，經常會四處獵食豬只和羊群。

; {{lang|hu|Sárkánykígyó}}
一種有翼的大蛇，主宰狂風和暴雨。

; {{lang|hu|Sárkány}}
一種有人類形象的傳說生物，拥有巨大和多數的頭，當它的頭受傷的時候，就會變的衰弱。

===[[斯拉夫|斯拉夫]]===
; {{lang|bg|Змей}}, {{lang|ru|Zmey}}, {{lang|be|Smok}}, {{lang|uk|Zmiy}}, {{lang|sr|Змај/Zmaj}}
斯拉夫龍與典型的西方龍相似，但有較多的頭部、火炎般的鼻息和一對翅膀。在傳統的斯拉夫傳說中，它是邪惡的象征。

===[[亞美尼亞|亞美尼亞]]===
; {{lang|en|Vishap}}
亞美尼亞龍是一種典型的西方龍。

===[[西伯利亞|西伯利亞]]===
; {{lang|ru|Йилбегән}}
西伯利亞龍是一種典型的斯拉夫龍。

===[[羅馬尼亞|羅馬尼亞]]===
; {{lang|ro|Balaur}}
羅馬尼亞龍是一種典型的斯拉夫龍，有較多的頭部、火炎般的鼻息和一對翅膀。

===[[楚瓦什|楚瓦什]]===
; {{lang|ru|Вěри Çěлен}}
楚瓦什龍是在伊斯蘭文明傳入[[楚瓦什|楚瓦什]]地區以前就在當地流傳的一種傳說生物。

===[[西班牙|西班牙]]===
; {{lang|es|Cuélebre}}
在[[阿斯圖里亞斯|阿斯圖里亞斯]]和[[里昂|里昂]]的傳說中，西班牙龍是一種有着巨大翅膀的大蛇，通常居住在洞窟中守護它的寶藏。它的鼻息帶有火炎和毒氣，能够獵食黃牛，并且能够活數世紀之久。

===[[阿爾巴尼亞|阿爾巴尼亞]]===
; {{lang|sq|Dragua}}
在[[阿爾巴尼亞神話|阿爾巴尼亞神話]]中，阿爾巴尼亞龍長有四條腿和一對蝙蝠般的翅膀，頭上有一支角和一對大耳朵，拥有隐形的能力，通常會在森林中出没。當[[奥斯曼帝國|奥斯曼帝國]]入侵阿爾巴尼亞的時候，它成為了高地人的守護者。

===[[葡萄牙|葡萄牙]]===
; {{lang|pt|Coca}}
葡萄牙龍是[[葡萄牙神話|葡萄牙神話]]中的一種傳說生物。

===[[希臘|希臘]]===
; {{lang|el|Δράκων}}
希臘龍是希臘神話中的一種傳說生物，它的存在通常是為了保護某一個有價值的人或事物。

===[[韃靼|韃靼]]===
; {{lang|tt|Зилант}}
[[Zilant|Zilant]]的形象類似[[飛龍_(傳說生物)|飛龍]]或[[蛇雞|蛇雞]]，它是[[喀山|喀山]]的象征符號，同時它可能也是古代土突厥民族的一个圖騰。

===[[立陶宛|立陶宛]]===
; {{lang|lt|Drakonas}}
立陶宛龍大多數的時候只會展示出一個頭部，但其實它是拥有多個頭部的大蛇。
