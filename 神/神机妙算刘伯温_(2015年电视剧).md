《'''神机妙算刘伯温'''》，中国大陆古装剧，导演韩刚，主演何冰、于震、王刚、王姬、姜寒。<ref>{{cite web |url=http://news.163.com/14/0814/02/A3IU708Q00014AEE.html |title=《神机妙算刘伯温》 何冰变身明朝反腐斗士 |accessdate=2014 |publisher= 网易|work= |language=zh-hans}}</ref>该剧中，加入了应时局2013年起的[[中纪委|中纪委]]反腐运动，显示了朱元璋和刘基的反腐手段。<ref>{{cite web |url=http://news.xinhuanet.com/legal/2014-08/10/c_1112008960.htm |title= 何冰王刚主演《神机妙算刘伯温》 讲述明朝反腐那些事儿|accessdate=2014 |publisher= 新华网|work= |language=zh-hans}}</ref>2014年6月殺青，2015年2月20日晚22:30分登陆[[北京卫视|北京卫视]]。2018年5月31日MOD龍華戲劇台播出。<ref>{{cite web |url=http://www.weibo.com/1599743193/C4J614Lw7?type=comment#_rnd1424238844828 |title=北京卫视剧集地 |accessdate=2015-02-18 |publisher= |work= |language=zh-hans}}</ref>

==剧情==
该剧讲述了元末明初[[刘基|刘基]]（刘伯温）“三分天下[[诸葛亮|诸葛亮]]，一统江山刘伯温”辅助[[朱元璋|朱元璋]]驱逐蒙古军队，重复恢复汉室王朝的功绩。<ref>{{cite web |url=http://ent.163.com/14/0812/10/A3ELBF9J00031GVS.html?f=jsearch |title=六“神”来袭 《神机妙算刘伯温》为反腐代言 |accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref>
