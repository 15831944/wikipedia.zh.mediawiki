《'''神奈川沖浪裏'''》是[[日本|日本]][[浮世繪|浮世繪]][[畫家|畫家]][[葛飾北齋|葛飾北齋]]的著名木版畫，於1832年出版，是《[[富岳三十六景|富岳三十六景]]》系列作品之一。它描述巨浪威脅[[神奈川|神奈川]]沖（神奈川外海）的船隻，與該系列的其它作品一樣，以[[富士山|富士山]]為背景。<ref>[http://w3.sce.pccu.edu.tw/fujiart/worldpaint/w35.htm 浮世繪版畫]</ref>圖中描繪的惊涛巨浪掀卷着渔船，船工们爲了生存而努力抗爭的圖像，遠景是富士山。这幅作品是北齋最有名的作品，也是世界上最有名的日本美术作品之一<ref name="british">{{cite web | url = http://www.britishmuseum.org/research/search_the_collection_database/search_object_details.aspx?objectid=3097579&partid=1&searchText=JCF20008&fromADBC=ad&toADBC=ad&numpages=10&orig=/research/search_the_collection_database.aspx&currentPage=1 | title = Kanagawa-oki nami-ura 神奈川沖浪裏 (Under the Wave off Kanagawa) |accessdate= 2013-02-04|language= en|publisher=[[大英博物館|大英博物館]]}}</ref>。也被记述为《神奈川沖'''波'''裏》<ref>在『日本史B　新訂版』（高等学校地理歴史科用教科書。平成9年3月31日文部科学省検定済。教科書编号：7実教 日B 582）p 215中被记载为「『冨嶽三十六景』神奈川沖波裏の場面」。</ref>。

==作品==
===构成===
本作品高25.7厘米，宽37.9厘米，是一副大画幅横绘画作<ref name="met">{{cite web | url = http://www.metmuseum.org/toah/works-of-art/JP1847 | title = Katsushika Hokusai: The Great Wave at Kanagawa |accessdate=2013-02-05 |language= en|publisher=[[大都会艺术博物馆|大都会艺术博物馆]]}}</ref>。

是由大浪、3只船以及背景的富士山这三个要素构成的画作。完成之后在画面左上角署名补完。
==== 富士山 ====
作为[[富岳三十六景|富岳三十六景]]主题的[[富士山|富士山]]被绘于画面中央靠下的背景中。对于日本来说，富士山是神圣的国家的象征<ref>{{cite web | url = http://www.fujiarts.com/cgi-bin/item.pl?item=56389 | title = Under the Wave off Kanagawa (The Great Wave) by Hokusai (1760–1849) | language = en | accessdate = 2010-07-10 | archiveurl = https://web.archive.org/web/20110711053740/http://www.fujiarts.com/cgi-bin/item.pl?item=56389 | archivedate = 2011-07-11 | deadurl = yes }}</ref>，也是美的象征<ref>{{cite web | url = http://web-japan.org/nipponia/nipponia35/es/topic/index.html | title = El Monte Fuji como Objeto Artístico |accessdate=2013-02-05 |language= 西班牙语|publisher=Nipponia}}</ref>。

将本来雄伟的富士山画得很小，和前景的巨浪形成了鲜明的对比<ref name="chiba">{{Cite web |url= http://www.ccma-net.jp/collection_img/collection_02-06_hokusai.html|title= 冨嶽三十六景　神奈川沖浪裏| language=日语 |publisher= {{link-ja|千叶市美术馆|千葉市美術館}}|accessdate=2013-02-05}}
</ref>。

地平线附近的暗色和仿佛被照亮的，被雪覆盖的山顶，表达了现在是清晨，而太阳是从观赏者这侧升起的。上空的[[积雨云|积雨云]]让人感觉现在应该有暴风雨，然而画中却并没有下雨<ref name="royal">{{cite web |url= http://rsnr.royalsocietypublishing.org/content/early/2009/02/23/rsnr.2007.0039.full | title = What kind of a wave is Hokusai's Great wave off Kanagawa? |accessdate=2013-02-05|language= en|publisher=[[王立协会|王立协会]] }}</ref>。

==== 船 ====
画面中描绘了3只被巨浪翻弄的船。这些船是当时运送活鱼的{{link-ja|押送船|押送船}}<ref name="royal"/>。

船上有8名抱紧船桨的划船手，在船头还能看到2名以上的乘客，画面中大约有30人<ref name="royal"/>。人们僵直在船中，和运动的浪形成对比<ref name="chiba"/>。
==== 浪 ====
画面截取了大海波涛汹涌，风大浪高的瞬间。构图中大浪的曲线描绘成弧形，令富士山成为这弧形的中心。浪尖飞散出的浪花四溅，仿佛降在富士山的雪<ref name="british"/>。最后面的船和浪高几乎相等，押送船一般长约12到15[[米|米]]，北齋将垂直高度伸长了30%左右，因此推测浪高应在10到12米<ref name="royal"/>。

有人认为这个巨浪就是[[海啸|海啸]]<ref>[[联合国教科文组织|联合国教科文组织]][http://itic.ioc-unesco.org/index.php 国际海啸情报中心]的插图。2013年2月11日查阅。</ref>。这个解释最早可以追溯到1960年代。再之前的130年中，都认为是通常常见的大浪。因为北齋生活的年代中，关东、关西均没有大的波浪发生过，因此有可能是听说过1792年在九州发生的{{link-ja|肥後迷惑|島原大変肥後迷惑}}的情况后创作的。但是本作品中描绘的大浪并不是缩短波长的海啸<ref name="royal"/>。

==== 署名 ====
画面左上角的[[落款|落款]]写着画的标题和署名。长方形的框里写着标题是「冨嶽三十六景 / 神奈川沖 / 浪裏 」。它左边写着的署名是「北斎改爲一筆 」，直接把「北斎」改[[号|号]]为「為一」（いいつ）。北齋在其生涯中改过30次号<ref name="british"/>。

富岳三十六景中，除了「北斎改爲一筆 」之外，他还用过「前北斎爲一筆 」以及「北斎爲一筆 」的署名<ref>{{cite web | url = http://expositions.bnf.fr/japonaises/grand/083.htm | title = Hokusai, Les Trente-six vues du mont Fuji |accessdate=2013-02-06 |language=法语 |publisher=[[法国国家图书馆|法国国家图书馆]]}}</ref><!--参照先では「zen saki no Hokusai Iitsu hitsu」も挙げられているが確認できず-->。

== 折绘 ==
从最新的折绘的版木的损耗状态来看，共印出了数千枚作品，在当时属于人气颇高的作品<ref name="british"/>。现存数百枚，从迄今为止的研究来看，现存的本作品的折绘在世界上也能排进前20<ref name="british"/>。折绘主要被日本和欧美的美术馆收藏，另外也有个人收藏家收藏的情况。

到现在也有机会获得当初印刷的折绘。在2003年3月7日，Huguette Bérès收藏会上估价23000[[欧元|欧元]]，在2002年，包含本作品在内的富岳三十六景46张套，拍卖135万欧元<ref>{{cite web | url = http://www.le-japon.info/articles.php?lng=fr&pg=33 | title = Retour de vague nippone – Vente Bérès de 2003 : prix d'adjudication de ''La Vague'' d'Hokusai | accessdate = 2005-11-06 | language = 法语 | archiveurl = https://web.archive.org/web/20071222004937/http://www.le-japon.info/articles.php?lng=fr&pg=33 | archivedate = 2007-12-22 | deadurl = yes }}</ref>。

== 影响 ==
一部分折绘在1870年代后半运到了欧洲<ref name="british"/>。[[荷蘭|荷蘭]][[后印象派|后印象派]]画家[[文森特·梵高|梵高]]非常赞赏这幅画，法國作曲家[[阿希尔-克洛德·德彪西|德彪西]]也受其影响创作出交响诗《海》，也给予了其他艺术家不同程度的影响<ref>{{Cite web |url= http://www.museum.pref.yamanashi.jp/4th_fujisan/01fugaku/4th_fujisan_01fugaku36_01.htm|title= 神奈川沖浪裏（かながわおきなみうら）| language=日语|publisher= {{link-ja|山梨县立博物馆|山梨県立博物館}}|accessdate=2013-02-05}}</ref>。

20世纪以来，因其在国际上的知名度，也被广泛应用在商业广告和大众文化中<ref name="british"/>。

預定於2024年發行的改版[[一千日圓紙幣|一千日圓紙幣]]之背面將採用神奈川沖浪裏<ref>{{cite news|url=https://www3.nhk.or.jp/news/html/20190409/k10011877791000.html?utm_int=news_contents_news-main_001 |title=新１万円札「渋沢栄一」紙幣デザインを発表 ５年後めど発行 |publisher=NHK |date=2019-04-09 |language=ja}}</ref><ref>{{cite news|url=https://tw.appledaily.com/new/realtime/20190409/1547352/ |title=日將全面更換紙鈔　防偽升級「偉人」換人當 |publisher=蘋果日報| date=2019-04-09}}</ref>。
