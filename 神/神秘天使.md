《'''神秘天使'''》（{{lang-en|'''''Strange Angel'''''}}）是一部美國的網絡電視連續劇，於2018年6月14日在{{link-en|CBS All Access|CBS All Access}}上首播。本劇是改編自{{link-en|喬治·彭德爾|George Pendle}}所寫的傳記書《''Strange Angel: The Otherworldly Life of Rocket Scientist John Whiteside Parsons''》，本劇由{{link-en|馬克·赫曼|Mark Heyman}}創作，他也是本劇的執行製作人和編劇。2018年10月29日，宣布CBS All Access續訂第二季。

== 故事簡介 ==
[[傑克·帕森斯|傑克·帕森斯]]是一位1930年代洛杉磯的傑出而雄心勃勃的藍領工人，他在一家化工廠擔任門警，但夢幻般的夢想導致他在美國火箭領域中開發未知學科，同時，他也陷入了一個神秘的世界，其中包括在夜晚進行性愛魔法儀式，並且成為了神秘學家[[阿萊斯特·克勞利|阿萊斯特·克勞利]]的信徒，帕森斯利用克勞利的自我實現教誨來支持他不可思議和前所未有的努力。<ref>{{Cite web |url = http://deadline.com/2018/01/strange-angel-jack-reynor-star-cbs-all-access-drama-series-ridley-scott-1202236420/ |title = ‘Strange Angel’: Jack Reynor To Star In CBS All Access Drama Series |author = Denise Petski |date = 2018-01-05 |publisher = Deadline Hollywood |language = en}}</ref>

== 製作 ==
=== 開發 ===
[[File:Strange_Angel_.jpg|thumb]]飾演[[傑克·帕森斯|傑克·帕森斯]]。]]
2014年10月，據報導說，馬克·赫曼正在為[[AMC_(電視頻道)|AMC]]編寫一部改編喬治·彭德爾所寫的《''Strange Angel: The Otherworldly Life of Rocket Scientist John Whiteside Parsons''》的電視連續劇。由[[列尼·史葛|列尼·史葛]]和他的{{link-en|Scott Free Productions|Scott Free Productions}}製作。到2016年，本劇已經搭起了編劇室並為本劇考慮而製作多個劇本。AMC最終沒有製作本劇。<ref name="CBS">{{Cite web |url = https://deadline.com/2017/08/cbs-all-access-strange-angel-1-no-activity-more-big-brother-1202140299/ |title = CBS All Access Greenlights Drama Series ‘Strange Angel’ & ‘$1’, First Comedy Series ‘No Activity’, Mulls More ‘Big Brother’ – TCA |author = Nellie Andreeva |date = 2017-08-01 |publisher = Deadline |language = en}}</ref>

2017年8月1日，本項目被{{link-en|CBS All Access|CBS All Access}}訂購成電視劇。AMC完成的電視劇劇本會在CBS和AMC達成協議之後使用。<ref name="CBS"/>

2018年4月19日，宣布本劇將於2018年6月14日首播。第一季包括了十集，每周星期四播放。<ref>{{Cite web |url = https://deadline.com/2018/04/cbs-all-access-premiere-strange-angel-jack-parsons-jack-reynor-1202367613/ |title = CBS All Access Sets Premiere Date For ‘Strange Angel’ |author = Dino-Ray Ramos |date = 2018-04-19 |publisher = Deadline |language = en}}</ref>
