《'''神奇樹屋'''》（{{lang-en|'''Magic Tree House'''}}）為[[美國|美國]]知名兒童作家[[瑪麗·波·奧斯本|瑪麗·波·奧斯本]]女士所寫的學習性冒險故事，中文翻譯版由小天下出版，內容描述主角傑克和安妮利用神奇樹屋穿梭時空，前往世界各地，也曾遇見世界名人，以及進入神話中的場景。目前從2005年出版至今，已出版了55集，台灣也出版了53集。在2009年也開始出版神奇樹屋小百科系列，至今已出版了35集，台灣則出版了20集。

{{link-en|同名日本動畫電影|Magic Tree House (film)}}於2012年1月7日在日本上映<ref>{{Cite web |url=http://www.animenewsnetwork.com/news/2011-03-08/magic-tree-house-books-2012-film-confirmed-as-anime |title= Magic Tree House Books' 2012 Film Confirmed as Anime |date=2011-03-08 |accessdate= 2013-03-06|publisher= Anime News Network}}</ref>。

== 作者簡介 ==
[[瑪麗·波·奧斯本|瑪麗·波·奧斯本]]女士為[[美國|美國]]知名兒童文學作家，[[北卡羅萊納大學|北卡羅萊納大學]]主修戲劇與比較宗教學。喜愛不同文學的她，畢業之後，曾到世界各地去旅行。寫作至今已有30年以上，作品超過100本，曾經獲得許多獎項，擔任過兩屆的[[美國|美國]]作家協會主席。[[瑪麗·波·奧斯本|瑪麗·波·奧斯本]]女士也因神奇樹屋系列而榮獲[[藍燈書屋|藍燈書屋]]的終身成就獎和美國教育平裝書協會的勒丁頓紀念獎。

== 角色介紹 ==
;傑克
八歲半，來自蛙溪鎮，是神奇樹屋中的主角之一，做事仔細，謹慎細心，是安妮的哥哥，發現了新事物會馬上記在筆記本上，與妹妹安妮一同穿梭時空。
;安妮
七歲，來自蛙溪鎮，是神奇樹屋中的主角之一，大膽，愛冒險，傑克的妹妹，是個不折不扣的行動派，與哥哥傑克一同穿梭時空，前往世界各地。
;摩根
神奇樹屋的主人，來自古老的國度卡美拉，會出任務給傑克和安妮，與[[圓桌武士|圓桌武士]]的首領[[亞瑟王|亞瑟王]]以及[[梅林_(亞瑟王傳說)|大魔法師梅林]]交情要好。

== 劇情簡介 ==
蛙溪鎮的傑克和安妮意外地在森林裡發現了一座神奇樹屋，發現了只要在樹屋中指著書裡的文字或圖片就可以抵達那個地方，他們倆使用這個神奇樹屋穿越時空，完成神奇樹屋的主人摩根交給他們的任務，並且在其中學到了許多新知識、道理及品德。
