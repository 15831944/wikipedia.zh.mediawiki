'''神經膠質細胞'''（英語：'''neuroglial cell'''、'''glial cell'''），又稱'''神經膠細胞'''、'''膠質細胞'''，是[[神經系統|神經系統]]中的組成單位之一，其角色主要為提供支持、供給營養、維持環境[[恆定|恆定]]及提供絕緣；近來亦有研究指出膠質細胞可參與訊息的傳遞。在[[人類|人類]]的[[腦|腦]]中，膠質細胞對[[神經元|神經元]]的比例估計約為10:1<ref name="sfn">[http://www.sfn.org/index.cfm?pagename=brainBriefings_astrocytes sfn.org Society for Neuroscience, 2000]</ref>。其英文名的字根"'''glia'''"在希腊语中的意思为“胶水”。

== 功能 ==
膠質細胞在神經系統中，扮演許多不同的角色。有些的主要功能為替[[神經元|神經元]]提供物理性支持，常被視為神經系統中的「膠水」。有些膠質細胞則能為神經元提供養分及氧氣並維持周遭的液體環境恆定。亦有膠質細胞可形成[[髓鞘|髓鞘]]以維持絕緣的效果、或是摧毀[[病原體|病原體]]，並移除死亡的神經元。

除此之外，膠質在神經發育上也扮演了重要的角色，可於發育早期導引神經元的遷移、並分泌可影響[[軸突|軸突]]及[[樹突|樹突]]生長的分子，並在[[突觸可塑性|突觸可塑性]]和突觸發生（synaptogenesis）的過程中均扮演著關鍵的角色。最近的研究亦指出，起碼在[[海馬體|海馬體]]及[[小腦|小腦]]中，膠質細胞可主動而積極地參與[[突觸|突觸]]間訊息傳遞的過程：[[星形膠質細胞|星形膠質細胞]]可調控突觸間隙中神經傳導物的清除，以區分不同時間來臨的神經衝動、並避免某些神經傳導物因過度累積而產生毒性，如[[麩胺酸|麩胺酸]]之[[興奮毒性|興奮毒性]]（excitotoxicty）。除此之外，星形膠質細胞可分泌突觸前神經元的調節分子，如[[三磷酸腺苷|三磷酸腺苷]]（ATP），且起碼在實驗室的環境下（[[in_vitro|in vitro]]），可在特定刺激下釋放神經傳導物。{{Fact|time=2007-11-01T17:05:56Z|date=2007年八月}}另一種特殊的膠質細胞，[[少突先驅膠質細胞|少突先驅膠質細胞]]（Oligodendrocyte precursor cell, OPC），有架構及功能相當完善的突觸，可接受來自至少兩類的主要神經元的訊號。{{Fact|time=2007-11-01T17:05:56Z|date=2007年八月}}

在傳統的觀念中，膠質細胞缺少神經元具備的許多特徵，例如[[突觸|突觸]]的構造及[[神經傳導物|神經傳導物]]的釋放，也不被認為在神經訊息傳遞上具有主動、積極的角色。然而，近年的研究逐漸推翻了此種看法。在現今的定義下，神經元和膠質細胞唯一明顯的差異，就是產生[[動作電位|動作電位]]的能力及神經元的極性，即膠質細胞不具有的[[樹突|樹突]]和[[軸突|軸突]]的分別。將膠質細胞單純視為神經系統中的「膠水」已不再合適。

膠質細胞在修復受損神經元上也具有調控的角色。在[[中樞神經系統|中樞神經系統]]中，[[星形膠質細胞|星形膠質細胞]]會增大、增生而形成傷痕，並產生抑制受傷軸突重新生長的分子以抑制神經修復。在[[周圍神經系統|周圍神經系統]]中，[[許旺細胞|許旺細胞]]則會在軸突受傷後回到較早期的發育狀態，以促進軸突的重新生長及修復。{{Fact|time=2007-11-01T17:05:56Z|date=2007年八月}}此項在中樞神經系統及周圍神經系統間的差異，顯示在未來有希望發展技術，以促發中樞神經系統中神經組織的再生、並幫助脊髓損傷的病患。

== 分類 ==
=== 小膠質細胞 ===
小膠質細胞是膠質細胞中最小的一种，在中樞神經系統中大約占總細胞數的15%，在腦及脊髓中各處可见。它的细胞核小、深染，呈扁平状或三角形，胞体发出细长有分支的突起，突起表面还有很多小的棘突。[[小膠質細胞|小膠質細胞]]為特化的[[巨噬細胞|巨噬細胞]]，在[[中樞神經系統|中樞神經系統]]中進行[[吞噬作用|吞噬作用]]，清除病原體或受損的神經元。雖然小膠質細胞是由造血前驅細胞衍生而來，而非[[外胚層|外胚層]]組織的一部分，因此嚴格說來不算膠質細胞，但因其在神經系統中支持性的角色，通常仍將其歸為膠質細胞。

== 分裂能力 ==
膠質細胞在成熟的階段仍保留細胞分裂的能力，而大多數的神經元則失去了此項能力。這個觀點是基於以下觀察：在成熟的神經系統受損（例：[[中風|中風]]、外傷）後，一般來說神經元無法被更換，但經常可在受損處附近觀察到膠質細胞的[[膠質細胞增生|增生]]。然而，詳細的研究並沒有顯示「成熟」的膠質細胞，例如[[星形膠質細胞|星形膠質細胞]]或[[少突膠質細胞|少突膠質細胞]]，仍保有[[有絲分裂|有絲分裂]]的能力：似乎只有原先存在的[[少突先驅膠質細胞|少突先驅膠質細胞]]能保有此種能力。另一方面，成熟神經系統中的少數區域，例如[[海馬體|海馬體]]的[[齒狀回|齒狀回]]（dentate gyrus）中及[[腦室下區|腦室下區]]（subventricular zone）中，仍可以觀察到神經元的新生。

== 胚胎發育 ==
大部分的膠質細胞自發育中胚胎的[[外胚層|外胚層]]組織衍生而來，特別是[[神經管|神經管]]及[[神經脊|神經脊]]；唯一例外者為自造血幹細胞衍生而來的[[小膠質細胞|小膠質細胞]]。在成人的身體中，小膠質細胞為可自我更新的一個族群，與[[中樞神經系統|中樞神經系統]]受損時會滲入的[[巨噬細胞|巨噬細胞]]及[[單核細胞|單核細胞]]有明顯不同。

在中樞神經系統，膠質細胞發育自[[神經管|神經管]]的[[腦室區|腦室區]]（ventricular zone）；此類細胞包括[[少突膠質細胞|少突膠質細胞]]、[[室管膜細胞|室管膜細胞]]、[[星形膠質細胞|星形膠質細胞]]。在[[周圍神經系統|周圍神經系統]]，膠質胞發育自[[神經脊|神經脊]]；此類細胞包括神經中的[[許旺細胞|許旺細胞]]及神經節中的[[衛星細胞|衛星細胞]]。

== 歷史 ==
膠質細胞是在1856年，由[[病理學|病理學家]][[魯道夫·魏爾嘯|魯道夫·魏爾嘯]]，在他試圖尋找腦中的「結締組織」時所發現。

人類腦中膠質細胞的含量大約為神經元的十倍。<ref name="sfn"/>在19世紀下葉膠質細胞被發現後，受到媒體報導的扭曲及斷章取義，產生了「我們只使用了腦的十分之一」這個有名的迷思。膠質細胞藉由管理突觸間訊息的傳遞從而影響學習的角色，在最近才被發現（2004）。
