'''神在堂'''，为[[中国|中国]][[江苏省|江苏省]][[扬州市|扬州市]]的一座老教堂，位于[[广陵区|广陵区]]泰州路[[扬州大学|扬州大学]]商学院内，由[[美国圣公会|美国圣公会]]会长韩忭明于1924年兴建在美汉中学内。神在堂为[[哥特式建筑|哥特式建筑]]，尖拱券，彩色窗玻璃，可容纳600人聚会礼拜<ref>{{cite web |url=http://news.163.com/11/1023/07/7H1LIBEQ00014AED.html|title=盐阜东路洋楼典型哥特式建筑|accessdate=2012-08-10|work=|publisher=扬州晚报}}</ref>。

神在堂长期闲置，被学校用于堆积杂物。鉴于扬州长期仅开放一座可容纳800人的基督教礼拜堂（[[萃园路礼拜堂|萃园路礼拜堂]]），但每次礼拜近2000人聚集，因此有恢复开放神在堂的呼吁<ref>{{cite web|url=http://www.yzzx.gov.cn/article.asp?articleid=6327|title=尽快恢复泰州路上的“神在堂”，以适应信徒宗教活动的需要，并成为旅游线的景点和亮点上|accessdate=2012-08-10|work=|publisher=扬州政协}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2010年，教会方面与扬州大学对神在堂和两幢牧师楼的修缮与使用的问题初步达成一致<ref>{{cite web|url=http://www.yangzhou.gov.cn/jhzj/201110/c31aa2313bb1451fba3a8c5549b157c6.shtml|title=扬州市民宗局2010年工作总结和2011年工作打算|accessdate=2012-08-10|work=|publisher=}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。2011年11月，神在堂被列为扬州市历史建筑<ref>{{cite web |url=http://roll.sohu.com/20111122/n326460942.shtml
|title=37座“草根”老建筑列入保护|accessdate=2012-08-10|work=|publisher=扬州晚报}}</ref>。2012年，又将其列入第五批文保单位推荐名单<ref>{{cite web |url=http://www.yznews.com.cn/yzsbs/html/2012-06/06/content_367333.htm
|title=扬州56处文物遗迹有望跻身“市级文保”|accessdate=2012-08-10|work=|publisher=扬州时报}}</ref>。
