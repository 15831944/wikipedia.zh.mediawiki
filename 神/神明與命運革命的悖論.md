{{Vgname|神明與命運革命的悖論|ja=神様と運命革命のパラドクス|en=The Guided Fate Paradox}}，是由[[日本一SOFTWARE|日本一SOFTWARE]]在[[PlayStation_3|PlayStation 3]]上推出的[[战略角色扮演游戏|战略角色扮演游戏]]。游戏原本預定2012年12月20日發售，后改为在於2013年1月24日推出此作。《[[電玩通|電玩通]]》於2012年10月4日推出“日本一SOFTWAREx電玩通合作計畫”，推出動畫、漫畫等活動。

== 故事 ==
成為『神』的主角，其使命就是「實現人們的願望」。
 
在天界中設置的『命運革命回路』所蘊育出的擬似世界裡發揮力量，神在不為人知的情況下，革命著人們的命運。

地球上光是人類就有70億。動物、植物、宇宙人，還有在神話故事中登場的人物等等各式各樣，在暗地裡支持著萬物的幸福，神明大人的工作就這樣週而復始的持續著——

== 登場人物 ==
本作另外和『SUNRISE×Lantis×電撃G's magazine』的新世代偶像計劃「[[LoveLive!|LoveLive!]]」全面合作。九位校園偶像全員，都有參與本作演出角色配音。

:支援連亞的其中一位天使。
:而她是在這些天使當中，最為親密接近的「神明的專屬天使」。
:個性率直而且溫柔，作為精神的支持幫助著因突然面臨的狀況而一頭霧水的連亞。在商店街的抽獎活動中也是她找到連亞的。
;{{nihongo|龍崎庫羅艾爾|竜崎 クロウエル|聲優：高坂穗乃果（[[新田惠海|新田惠海]]）}}
:支援連亞的其中一位天使。
:擅長照顧人的性格，總是擔心著伙伴的樣子，是天使們之間的好氣氛核心。興趣是製作小點心。在天使們的茶會中，絕對欠缺不了她親手製作的餅乾和蛋糕。
;{{nihongo|綾小路雪莉艾爾|綾小路 シェリエル|聲優：東條希（[[楠田亞衣奈|楠田亞衣奈]]）}}
:支援連亞的其中一位天使。
:擁有令人煩惱的身材與性感的言行，擁有誘惑著青少年主角的小惡魔氣質。不經意出口那辛辣的發言總是使週圍的人感到期待和興奮。
;{{nihongo|東條奈兒艾爾|東條 ネルエル|聲優：小泉花陽（[[久保由利香|久保由利香]]）}}
:支援連亞的其中一位天使。
:經常窩在自己的房間裡幾乎都看不到人，身邊充滿了謎團。能進入她房間的人，就算是天使的伙伴們之中，也只有少數特定幾個人而已
;{{nihongo|白鳥拉娜艾爾|白鳥 ラナエル|聲優：絢瀨繪里（[[南條愛乃|南條愛乃]]）}}
:支援連亞的其中一位天使。
:年幼可愛的外表，與尖銳的言行反差，給與她一種獨特的存在感。平常雖然會說著自己的第二個名字，但有的時候會依照心情改變內在人格。
:是名重度[[中二病|中二病]]患者。
;{{nihongo|姬川卡爾席翁|姫川 ガルシオン|聲優：[[木村明人|木村明人]]}}
:支援連亞的其中一位天使，負責管理不同特色的六位天使。
:面對成為「神明」的連亞，較任何人都表現出更高的敬意。殷勤的態度為其特徵，充滿中年特色的男性天使。
;{{nihongo|伊集院雷崎艾爾|伊集院 レキエル|聲優：[[杉山紀彰|杉山紀彰]]}}
:支援連亞的其中一位天使。
:懷疑著連亞身為「神明」的資質等等，因此態度是所有天使之中最帶刺的一位。因為是責任感很強烈的性格，所以雖然對他人也很嚴格，但對自己更加的嚴厲。
;{{nihongo|西園寺米薩艾爾|西園寺ミザエル|聲優：[[伊月唯|伊月唯]]}}
:第八位天使，是名死亡天使（''今は亡き天使''）。生前在天界進行各種研究跟開發。她是『命運革命回路』的製作者，不過開發途中意外喪失性命。

== 遊戲系統 ==
「神明與命運革命的悖論」其主要顯現出來的中心點在於「複製世界」，以本來的故事設定進行，玩家在遊戲之中所做的一舉一動都將改變故事走向。改變世界的副本也給了原來的世界發生了變化，從而導致了最終革命的命運以及結果。每個段落一開始都是從等級1開始計算，最後全數換算至總等級。使用遊戲之中所得到的「革命迴路」進行神體改造，強化所操縱角色的各種素質及發動的附加效果。<ref>{{Cite web|date=2013-01-17|url=http://news.dengeki.com/elem/000/000/580/580931/index-4.html|title=発売迫る！ 『ディスガイア』制作チームの最新作・運命革命RPG『神様と運命革命のパラドクス』情報を総まとめ【電撃日本一】|work=電撃オンライン|page=4|accessdate=2013-01-17}}</ref>
