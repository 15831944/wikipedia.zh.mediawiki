《'''神雕俠侶'''》（{{lang-en|'''The Condor Heroes 95'''}}；又名《'''九五神雕俠侶'''》），是[[香港電視廣播有限公司|香港電視廣播有限公司]]根據[[金庸|金庸]]所著的[[武俠小說|武俠小說]]《[[神鵰俠侶|神鵰俠侶]]》改編而拍攝的经典鉅著[[電視劇|電視劇]]，全劇共32集，監製为[[李添勝|李添勝]]。

=== 神雕俠侶 ： 雁丘詞 - 问世间情是何物 ===

{{cquote|
問世间 情是何物，直教生死相許。

天南地北双飞客，老翅几回寒暑。

歡樂趣，離別苦，就中更有痴儿女。

君應有語，渺万里層云，千山暮雪，隻影向誰去。}}

== 劇情綱要 ==

楊過 ( 古天樂 飾 ) 一次誤闖進活死人墓中，幸好拜得古墓派掌門小龍女 ( 李若彤 飾) 為師，因厭惡師父二字稱謂，故稱小龍女為「姑姑」。二人相依為命，但不知不覺間卻萌生微妙的男女感情，且達到生死相逢之境地 ! 對此情堅貞不渝，縱使小龙女被尹志平 ( 陳啟泰 飾 ) 所侮辱，仍能摒除世俗眼光而當眾結為連理成親。 後二人相分，決定十六年後終在[[絕情谷|絕情谷]]再續前緣 ……

另一廂，郭靖 ( 白彪 飾 ) 、黃蓉 ( 魏秋樺 飾 ) 將女兒 郭芙 ( 傅明憲 飾 ) 許配予楊過，但為楊過所拒絕 ; 自始郭芙與楊過便積怨日深，於是在一次誤會中，郭芙更斬下楊過之左手。然而，小東邪郭襄 ( 李綺虹 飾 ) 則對楊過豁達情深，更鼓勵楊過踏遍天涯海角去覓回小龍女 ……

==影碟發行==
以下所有影碟發行均由[[電視廣播(國際)有限公司|電視廣播(國際)有限公司]]授權：

香港發行代理商[[現代音像(國際)有限公司|現代音像(國際)有限公司]]於2003年推出發行了《神鵰俠侶》[[VCD|VCD]]影碟零售版本，此影碟將已播放的集數並製作成20隻碟作為片段完整及無刪剪版本，一套共分為兩盒影碟，每隻碟跟電視劇的每集片長約60分鐘一樣，設有[[粵語|粵語]]及[[國語|國語]]發音版本並配上[[繁體中文|繁體中文]]字幕。

中国大陆發行代理商[[中凯文化集团|中凯文化集团]]推出發行了《神鵰俠侶》[[H_DVD|H DVD]]影碟零售版本，总共30集，共四碟，出版社为九洲音像出版公司，設有[[粵語|粵語]]及[[國語|國語]]發音版本並配上[[简体中文|简体中文]]字幕。

台灣發行代理商[[弘音多媒體科技股份有限公司|弘音多媒體科技股份有限公司]]於2007年推出發行了《神鵰俠侶》[[DVD|DVD]]影碟零售版本，此影碟燒錄VCD香港版的20隻碟作為集數，共5隻碟，設有[[粵語|粵語]]及[[國語|國語]]發音版本並配上非隱藏[[繁體中文|繁體中文]]字幕。

==重播==
本劇於2014年5月在TVB星河頻道，2017年9月1日在TVB經典台重播。
