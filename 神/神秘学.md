'''神秘學'''（英語：'''Occultism'''，Occult Studies），或稱'''祕術主義'''

*指[[超自然|超自然]]的[[形而上學|形而上學]]，亦即有關神秘力量的學問。
*指研究「秘密知識」或是「隱瞞起來的知識」的學問。

與神秘學相對的是“可見的學問”或“可以測量的學問”，即"科學"。有時[[新教|新教]]徒認為[[天主教|天主教]]本身也持神秘學行為。至於玩角色扮演遊戲、聽重金屬音樂或讀《[[哈利波特|哈利波特]]》小說，由於實際上沒有任何隱蔽的或神秘的知識和學問，所以與神秘學毫不相關。

==非一般平庸科學的歷史源頭==
一般認為，非一般平庸科學有三大源頭：[[古埃及|古埃及]]、[[希伯來|希伯來]]、[[希臘|古希臘]][[羅馬|羅馬]]，東方學說的四大源頭：[[古印度|印度]]、[[古中國|中國]]、[[古西藏|西藏]]、[[薩滿教|薩滿教]]各個傳統都有不同的特點。

古埃及文明是西方神秘學中最古老的源頭，在後期的羅馬帝國征服之中，漸漸失去了獨立性，進入了希臘與羅馬的文化之中，因此許多理論都失傳，而大部分資料都來源於希臘旅行者的記錄。到現代，古埃及在現代的西方神秘學中，最為有代表性的就是儀式魔法時常用的“[[何魯斯|何魯斯]]（Horus）之眼”，塔羅，煉金術等。

[[希伯來|希伯來]]文明本是中東地區的一支，因戰爭，旅行而吸收了一部分[[迦勒底|加爾底亞]]人、[[蘇美|蘇美]]人、[[埃及|埃及]]人、[[巴比倫|巴比倫]]人和[[腓尼基|腓尼基]]人的文化，而他們信仰的[[猶太教|猶太教]]，建立了最初的“一神論”，也從中發展出神秘學中極具影響力的教派之一 ——[[卡巴拉|卡巴拉]]（Kabalah）。

[[希臘|希臘]]文明造就了神秘學的核心思想，神秘哲學、[[占星|占星]]、[[靈數|靈數]]，從中誕生。

===密傳心理（Esoteric Psychology）===

神秘學中把人分為四大層面：
　　  
:      層面      分支    呈現方式     掌管能力　　
:    '''精神層面'''              靈魂        直覺 
:    '''心智層面'''           頭腦/神智    理智/思想  
:    '''以太層面'''   以太體   基本模式     生長藍圖  
:              情緒體     脈輪       影響情緒  
:              理智體    黃色光     判斷/思考  
:              星靈體   精細結構     出體經驗  
:             以太模板  高頻以太     能量治療  
:              精妙體    純靈性      和諧能量  
:               因果體   前世記憶 
:    '''肉體層面'''              物質         五感 

　　（以太層面由美國靈媒Barbara.Ann.Brennan整理）
