'''神盾戰鬥系統'''（Aegis combat system），正式編號是Weapon System Mk7，是全世界第一種全數位化的艦載戰鬥系統，是美國海軍第一種具備決策輔助功能的系統，[[美國海軍|美國海軍]]現役最重要的整合式水面艦艇作戰系統，由[[洛克希德|洛克希德]]所設計與製造。
起初美國海軍提出一個「先進水面飛彈系統」（ASMS/the '''A'''dvanced '''S'''urface '''M'''issile '''S'''ystem）的提案，經過不斷發展，在1969年12月改名為'''空中預警與地面整合系統'''（'''A'''dvanced '''E'''lectronic '''G'''uidance '''I'''nformation '''S'''ystem/'''A'''irborne '''E'''arly-warning '''G'''round '''I'''ntegrated '''S'''ystem），英文縮寫Aegis剛好是希臘神話中[[埃癸斯|宙斯之盾]]，台灣翻譯為「-{神盾}-戰鬥系統」、中華人民共和國則翻譯為「-{宙斯盾}-系統」。

== 系統設計 ==
[[美國|美國]]在[[越戰|越戰]]時期的經驗以及與[[蘇聯|蘇聯]]的競爭下，分析出美國海軍水面艦艇對反潛、反艦、防空、近迫防禦及區域戰情管理中心之需求，提出新一代作戰中心系統的規劃，因此神盾系統是海軍版的聯合打擊概念的延伸，而非單一種武器系統。

神盾系統的核心是一套電腦化的指揮決策與武器管制系統，雖然在表面上神盾系統很強調對於空中目標的追蹤與攔截能力，不過神盾系統的核心接收來自於整合艦上[[雷達|雷達]]及各種武器控管系統，跨作戰單位可以由[[戰術資訊網路|戰術資訊網路]]交換情報。透過武器管制系統的整合與指揮，艦上的作戰系統得以發揮最大的能力進行必要的攻擊與防禦措施。武器管制系統轄下包括輕型空載多用途系統（LAMPS）、[[AGM-84魚叉反艦飛彈|AGM-84魚叉反艦飛彈]]、[[RIM-161飛彈|標準三型防空导弹]]、[[密集阵近程防御武器系统|密集阵近程防御武器系统]]、[[魚雷|魚雷]]發射系統以及[[海妖反魚雷裝置|海妖反魚雷裝置]]等。

神盾作戰系統最重要，也是最顯眼的就是[[AN/SPY-1_3D相位陣列雷達|AN/SPY-1 3D相位陣列雷達]]，這一套雷達共有四片，成八角形，分別裝置在艦艇上層結構的四個方向上，以[[相位|相位]]雷達波重繪的方式判斷目標，取代傳統的旋轉式回波雷達。第一代的SPY-1A雷達每片重量高達12000磅，上面有140套模組，每個模組包含32具發射/接收與相位控制單元。這一套雷達於1965年開始發展，1974年展開海上測試，第一套系統隨[[提康德罗加级导弹巡洋舰|提康德羅加級巡洋艦]]第一艘提康德羅加號（CG-47）於1983年進入美國海軍服役，後來又發展到驅逐艦，[[阿利·伯克級驅逐艦|阿利·伯克級驅逐艦]]第一艘[[阿利·伯克号驱逐舰|阿利·伯克号（DDG-51）]]於1991年進入美國海軍服役。

2017年因應北韓飛彈威脅的名義，日本傾向部署一種陸基[[神盾系統|神盾系統]]，有助減輕配備神盾系統的日本軍艦全天候警戒負擔。<ref>[http://www.chinatimes.com/newspapers/20170515000310-260119 中時-陸基神盾]</ref>
=== 系統設計 ===
美國發展神盾作戰系統時，依據專案開發的時空背景訂下不同的「BaseLine」（基线）版本。這些架構是以前一代架構為基礎改良或者是增加新的裝備，部分基線架構僅安裝在巡洋艦（CG 47級）或是驅逐艦（DDG 51級）上

;初版 
基線零是早的神盾作戰系統架構，安裝在頭兩艘[[提康德羅加級飛彈巡洋艦|提康德羅加級飛彈巡洋艦]]（CG 47/48）上。基線零主要使用UYK-7和UYK-20電腦，控制各項系統，[[SPY-1雷達|SPY-1雷達]]，資料傳遞與武器射擊管制。整套設備重達610噸，電腦程式只有802500字元（Word）。

;基線一 
基線一是在基線零的架構上，增加LAMPS III直升機系統操作能力，這套系統安裝在CG 49到CG 51等數艘巡洋艦上。

;基線二 
基線二改用[[Mk_41垂直發射系統|Mk 41垂直發射系統]]，提升[[反潛作戰|反潛作戰]]裝備，整合[[戰斧巡弋飛彈|戰斧巡弋飛彈]]等。電腦仍舊使用UYK-7/20，以及UYA-4戰情顯示器。基線二於1986會計年度展開測試的工作，戰斧巡弋飛彈則是到1987年才整合完成。基線二原本是要使用在被放棄的打擊巡洋艦（Strike Cruiser）上，這個計畫被放棄後則全部使用在提康德羅加級巡洋艦當中沒有配備SSQ-89聲納的船上。安裝的巡洋艦包括CG 59到CG 54等數艘。

== 作戰歷史 ==
波斯灣戰爭、伊拉克戰爭及所有美國海外介入行動中都有神盾艦的身影，但多半是當作偵查和[[戰斧巡航導彈|戰斧巡航導彈]]的發射平台，防空飛彈功能还未曾使用過，因為至今並未有足以和美軍海上大規模攻防的軍隊衝突。

1988年7月3日，裝有神盾系統的美國海軍[[提康德羅加級飛彈巡洋艦|提康德羅加級飛彈巡洋艦]]{{le|文森斯號飛彈巡洋艦|USS Vincennes (CG-49)|文森斯號}}（USS ''Vincennes'' CG-49）在与伊朗交火过程中誤擊[[伊朗航空655號班機空難|伊朗航空655號班機]]，造成290名平民死亡。

== 使用神盾系統的國家 ==
神盾系統由於體積過大以及技術問題，一開始美國僅釋出給{{fact|一級盟邦}}例如：[[日本|日本]]，[[中華民國|中華民國]]曾試圖取得並計劃將該套系統用於田單號巡防艦上，但因神盾系統[[田單計畫|小型化研發]]技術難度大且成本過高而放棄，最後田單號以[[成功級巡防艦|成功級巡防艦]]的規格建造（目前又重新啟動計劃）。從2000年以後由於系統小型化成功，使得该系統在5000噸級別的艦船上得以裝備，因此獲得多國採用。

== 类似舰种 ==
   
除美國以外，其他國家也參考美國發展自有的作戰系統，如[[中国海军|中国海军]]的[[中華神盾|中華神盾艦]]、[[英国皇家海军|英国皇家海军]]的[[45型驱逐舰|45型驱逐舰]]等，这种命名归类其实类似于当年英国海军[[无畏号战列舰|无畏级主力舰]]服役以后，其他国家建造的类似军舰也被称为[[无畏舰|无畏舰]]一样。
