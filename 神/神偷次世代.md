{{engname|《神偷次世代》|'' Skyline Cruisers ''}}是2000年出品的一部香港電影，由[[葉偉信|葉偉信]]執導，[[黎明|黎明]]、[[陳小春|陳小春]]、[[舒淇|舒淇]]、[[杜德偉|杜德偉]]、[[雪兒|雪兒]]、[[李燦森|李燦森]]、[[王合喜|王合喜]]、[[尹子維|尹子維]]等演出<ref>{{Cite web |url= https://moviecool.asia/film/gdbU8vu1Okj|title =神偷次世代| publisher =影史庫|language = 中文}}</ref>，在香港、[[吉隆坡|吉隆坡]]及[[怡保|怡保]]進行拍攝<ref>{{Cite web |url= http://www.hkbo.com.hk/search-details-new.php?filmid=5836&run=1 |title =電影資料庫| publisher = 香港票房有限公司|language = 中文}}</ref><ref>{{Cite web |url= http://hkmdb.com/db/movies/view.mhtml?id=8849&display_set=big5|title =神偷次世代（2000）| publisher =香港影庫|language = 中文}}</ref> <ref> {{Cite book |title =《電視月刊》第209期﹔第212-219期|publisher =電視月刊雜誌社| date =2000年| url = https://books.google.com.hk/books?id=8ul5AAAAIAAJ&q=%22%E7%A5%9E%E5%81%B7%E6%AC%A1%E4%B8%96%E4%BB%A3%22&dq=%22%E7%A5%9E%E5%81%B7%E6%AC%A1%E4%B8%96%E4%BB%A3%22&hl=zh-TW&sa=X&ved=0ahUKEwjQiuu8yqTgAhWGXbwKHRzxAj0Q6AEIRDAE |language =中文|}} </ref>。

==情節介紹==
電影內容講述Mac、Bird、Sam和Michelle是國際神偷組織的頂尖成員，Mac和Bird身手敏捷，Sam的偷竊技巧一流，而Michelle則是一名電腦專家，他們四人向來合作無間，能夠完成各種艱巨的任務。以Mac為首的神偷組織接到新任務，負責到馬來西亞監察科學家金元正，目標是偷去一種剛剛完成研究的治癌藥物。四人潛入藥廠，以為任務成功之際卻遭到埋伏，原來任務是一個騙局。四人成功逃脫，Mac遇見一個神秘女子June，使他想起自己的女友被舊拍檔出賣而死的往事。June擁有一個火柴盒，是破解謎團的重要線索<ref>{{Cite web |url= http://big5.ent.qianzhan.com/ent/moviedetail/fa577d82.html|title =神偷次世代 電影劇情簡介| publisher = 前瞻網娛樂|language = 中文}}</ref>。
