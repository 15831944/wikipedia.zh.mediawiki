{{Vgname|神界|en=Divine Divinity|cn=神界|tw=神諭}}是一款2002年由CDV发行，Larian Studios开发的[[动作角色扮演游戏|动作角色扮演游戏]]。

==剧情==
在游戏初始剧情的两千年前，Rivellon大陆的七英雄为了保护Rivellon大陆而在与邪恶巫师的战斗中牺牲。为了纪念七英雄，为保护大陆为初衷的神界秩序（Divine Order）被建立了起来。<ref>{{Cite web |url = http://www.mobygames.com/game/divine-divinity |title = Divine Divinity for Windows |author = |date = |publisher = [[MobyGames|MobyGames]] |language = en |accessdate = 2018-02-18 }}</ref> 

在游戏的开始，[[玩家角色|玩家]]在医者之镇——Aleroth中的一间房屋中醒来。
