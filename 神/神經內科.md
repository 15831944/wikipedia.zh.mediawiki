'''神經內科'''（neurology）是[[醫學|醫學]]的一個分支，專門應付神經系統的疾病，但不包括手術（其相對應的外科為[[神經外科|神經外科]]）。<ref name=WorkWYourDoc>{{cite web|language=en|title=Working with Your Doctor|url=http://patients.aan.com/go/workingwithyourdoctor|publisher=American Academy of Neurology|accessdate=28 October 2012}}</ref> 作為[[應用科學|應用科學]]的一種，其與神經外科均有別於隸屬自然科學的[[神經科學|神經科學]]。前者負責在病人身上運用已有知識，而後者則專門研究與發現新的科學概念。

==神經失調==
神經失調是失調影響中央神經系統、周邊神經系統、或位於中央和周邊神經系統的自主神經系統。常見情況包括：
# 頭疼失調,譬如[[偏頭痛|偏頭痛]]和緊張頭疼（群族頭疼） 
# 情景引發的失調，譬如[[癲癇症|癲癇症]]
# 神經退化性失調，例如[[老年癡呆|老年癡呆]]，包括[[阿茲海默病|阿茲海默病]]
# 腦血管的疾病，譬如瞬變局部缺血的攻擊和衝程
# [[失眠|失眠]]
# [[大腦痲痺|大腦痲痺]]
# [[中央神經系統|中央神經系統]]（腦炎的）[[細菌|細菌]]、[[黴菌|黴菌]]、[[病毒|病毒]]和寄生傳染，[[腦膜炎|腦膜炎]]並且周邊神經（[[神經炎|神經炎]]）
# 瘤- [[腦腫瘤|腦腫瘤]]、[[脊髓腫瘤|脊髓腫瘤]]、周邊神經的[[腫瘤|腫瘤]]
# 運動混亂譬如[[帕金森氏症|帕金森氏症]]的疾病、[[舞蹈病|舞蹈病]]、hemiballism 
# 中央神經系統的壞死疾病、譬如多發性硬化症、和周邊神經系統
# 脊髓神經失調：腫瘤、傳染、精神創傷、畸形（即myelocelle 、meningomyelocelle）
# 周邊神經失調、肌肉和神經連接點的神經失調
# 腦部創傷、[[脊髓|脊髓]]和周邊神經創傷
# [[昏迷|昏迷]]和有各種各樣的起因的昏迷

神經學家負責診斷和對待所有上述條件，除了外科干預外，就成了[[神經外科|神經外科]]醫師的責任，並且在某些情況下更涉及神經放射學家。在一些國家，神經學家的另外法律責任包括為被懷疑的已故患者尋找[[腦幹死亡|腦幹死亡]]的證據，並且發出死亡證書。

== 與精神病學的重疊 ==
雖然許多精神病被認為是神經的失調影響中央神經系統，他們傳統上被分開地分類，並由[[精神科|精神科醫師]]、[[臨床心理學|臨床心理學家]]和[[心理治療|心理治療師]]處理。

但是有強烈的跡象表明神經化學機制在[[躁鬱症|躁鬱症]]和[[精神分裂症|精神分裂症]]的發展上（以實例來說）扮演著一個重要角色。同樣，神經病學疾病常有精神病學表現，這也是[[行為神經學|行為神經學]]（Behavioral Neurology）暨[[神經精神醫學|神經精神醫學]]（Neuropsychiatry）學者的研究範疇。
