'''神泉站'''（{{jpn|j=神泉駅|hg=しんせんえき|rm=Shinsen eki|f=y}}）是一個位於[[東京都|東京都]][[澀谷區|澀谷區]][[神泉町|神泉町]]，屬於[[京王電鐵|京王電鐵]][[井之頭線|井之頭線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''IN02'''。

== 歷史 ==
* 1933年（[[昭和|昭和]]8年）8月1日 - 作為帝都電鐵所屬的車站開始營業。
* 1940年（昭和15年）5月1日 - 合併到小田原急行鐵道、成為該公司所屬帝都線的車站。
* 1942年（昭和17年）5月1日 - [[小田急電鐵|小田急電鐵]]與[[東京急行電鐵|東京急行電鐵]]（大東急）合併。
* 1948年（昭和23年）6月1日 - 以京王帝都電鐵從東急分離出來，成為該公司井之頭線的車站。
* 1995年（[[平成|平成]]7年）9月28日 - 月台延長工程完成，取消選擇性車門操作措施。
* 1996年（[[平成|平成]]8年）12月2日 - 舊站臺拆除後大幅改良成現今所使用的樣貌，新車站落成。

== 車站構造 ==
神泉車站為跨橋式站房結構的[[地面車站|地面車站]]，月台為2面2線的[[側式月台|岸式月台]]。有趣的點是，車站本身幾乎建構在[[隧道|隧道]]中，彷彿是[[地下車站|地下車站]]一般。該站到澀谷車站的直線距離僅有500公尺，但因為與澀谷站之間的[[道玄坂|道玄坂]]一帶地形起伏大，加上此區域內沒有其他鐵道路線經過，促成了該站設立的必要性。

月台的使用有效長度本來只可容納3節18公尺長的車廂，停靠此站時吉祥寺端的2節車廂車門不予開啟，（在這以前曾經有一段時期，部分列車採用[[選擇性車門操作|選擇性車門操作]]的措施），車站本身也僅有設置於澀谷側的簡易閘口，不像現在車站中設有販賣舖以及松濤方向的出入口。

之後，採用了長20公尺車廂設計的[[京王1000系電車_(2代)|1000系電車]]投入營運同時，在隧道的延伸工程中也將月台進行延伸。直至1995年9月28日，停靠此站時，列車車門才可全部開啟。在選擇性車門操作措施實施的時代，沒入隧道的車廂車門前，皆有張貼「此車門無法開啟」的告示。這與同樣鄰接澀谷站的[[東急東橫線|東急東橫線]][[代官山站|代官山站]]狀況類似。

1995年，神泉站連同隧道工程進行大幅度的改良工程，現今使用的車站建築於1996年12月2日落成。兩個月台與車站閘口之間皆設有電梯。廁所設置於2樓的閘口內。基於[[通用設計|通用設計]]，也設置了無障礙廁所。

此站入選[[關東車站百選|關東車站百選]]。

== 車站週邊 ==
周邊的地形起伏，坡道較多。車站東側是[[情侣酒店|情侣酒店]]及[[風俗店|風俗店]]林立而出名的[[圓山町|圓山町]]，該區域也有其它一般的旅館或小餐廳，2000年代以後新開設了許多年輕人取向的[[迪斯可|俱樂部]]或是[[音樂展演空間|音樂展演空間]]。

西南側是環境閑靜的[[住宅區|住宅區]]，北側的[[松濤|松濤]]則是高級住宅區。站前是小規模的商店街，延伸至[[東京都道317號環狀六號線|山手通]]，小型餐飲店為數不少。該站是距離設在[[南平台町|南平台町]]的[[東京急行電鐵|東京急行電鐵]]總公司最近的車站。
