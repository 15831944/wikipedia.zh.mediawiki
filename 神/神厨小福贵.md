《'''神厨小福贵'''》，2007年上映于[[中国大陆|中国大陆]]的国产动画片，导演[[王宏_(导演)|王宏]]，讲述中国饮食文化。<ref>{{cite web |url=http://news.cntv.cn/20120605/112890.shtml |title=国产精品动画《神厨小福贵》赏析 |accessdate=2012年6月5日  |publisher=央视 |archiveurl=http://news.cntv.cn/20120605/112890.shtml |archivedate=2012年12月18日 |work= |language=zh-cn}}</ref>

== 剧情 ==
[[清朝|清朝]]末年，[[八国联军|八国联军]]入侵大清，正值[[慈禧太后|慈禧太后]]和[[光绪帝|光绪帝]]争权夺利，慈禧太后想要废黜他，于是找人下毒毒死光绪帝，[[太监|太监]]小李子是慈禧身边的红人，他是[[御膳房|御膳房]]二总管，他去给慈禧物色放毒的人选，洪福贵的爷爷因为厨艺出众而被选中，结果卷入政治斗争中，洪福贵继承了爷爷的洪家菜厨艺，后离开宫廷，走入民间。<ref>{{cite web |url=http://sd.people.com.cn/GB/166203/172859/14319394.html |title=神厨小福贵 |accessdate= 2011年4月6日|publisher= 人民网|archiveurl=http://sd.people.com.cn/GB/166203/172859/14319394.html |archivedate=2012年12月18日 |work= |language=zh-cn}}</ref>
