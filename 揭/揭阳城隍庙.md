'''揭阳城隍庙'''是位于[[广东省|广东省]][[揭阳市|揭阳市]][[榕城区|榕城区]]东门的一座[[城隍庙|城隍庙]]，始建于[[宋|宋]]绍兴十年（1140年），明洪武二年（1369年）重修。

== 城隍殿 ==
城隍庙前供有左右将军石像、[[千里眼|千里眼]]和[[顺风耳|顺风耳]]，左门和右门匾额题写“你来了么”和“也有今日”。城隍殿内供奉城隍伯府大人像，上方牌匾为“威灵显赫”；左方和右方前往夫人厅大门的匾额分别是“彰善瘅恶”和“南天保障”。每年农历正月二十三日时举行伯府大人和伯府夫人的[[圣驾巡游|圣驾巡游]]仪式<ref>{{cite web |url=http://news.hqcr.com/projects/news/view.aspx?nid=431541 |title=揭阳城隍庙将举行城隍庙伯府大人巡游活动（附路线） |last= |first= |date=2017-02-16 |website= 华人频道|publisher=国际在线 |access-date=2019-02-03 |quote=}}</ref>。

== 夫人厅 ==
厅内供奉城隍伯府夫人像，上有“慈恩广布”匾额。厅右边为夫人的房间，但不允许游客进入。重修捐资芳名中列出了[[泰国华人|泰国华人]]李瑞典等的肖像<ref>{{cite web |url=http://www.csyyuedu.com/article.php?artid=666&tid=0 |title=榕城区志 第一章 华 侨 |last= |first= |date= |website=潮汕文化云阅读 |publisher= |access-date=2019-02-03 |quote=}}</ref>。另外，夫人厅前设有[[放生池|放生池]]，饲养[[锦鲤|锦鲤]]和[[巴西龟|巴西龟]]。

== 周围景点 ==
=== 雷神庙 ===
雷神庙位于城隍庙左边，始建于[[清|清]][[康熙_(年号)|康熙]]2年（1663年），[[乾隆_(年号)|乾隆]]44年（1779年）重建。雷神殿内匾额题记“神明音佑”，供奉[[雷神|雷神]]、[[电母|电母]]、[[雨师|雨师]]和[[风伯|风伯]]。

=== 龙起禅院 ===
龙起禅院位于雷神庙后面，院内供奉[[西方三圣|西方三圣]]坐像、[[文殊菩萨|文殊菩萨]]和[[地藏菩萨|地藏菩萨]]。
