'''揭阳市'''是[[中华人民共和国|中华人民共和国]][[广东省|广东省]]下辖的[[地级市|地级市]]，位于广东省东部。市境东接[[汕头市|汕头市]]、[[潮州市|潮州市]]，北连[[梅州市|梅州市]]，西邻[[汕尾市|汕尾市]]，南临[[南海|南海]]。地处[[韩江三角洲|韩江三角洲]]西部，粤东沿海丘陵与榕江冲积平原区，地势西高东低，低山、谷地与平原相间，分布不均，西北、西南为丘陵山地，中部、南部和东南部为榕江和滨海平原区。西北部为[[莲花山脉|莲花山脉]]，中南部为[[大南山_(粤东山脉)|大南山]]、[[小北山_(粤东山脉)|小北山]]。[[榕江|榕江]]横贯市境中北部并流经市区，中部为[[练江|练江]]，南部为[[龙江_(广东)|龙江]]，均自西向东流入南海。全市总面积5265平方公里，人口605.89万。揭阳是[[粤东|粤东]]古邑，广东省历史文化名城，全国著名[[侨乡|侨乡]]，有华侨320多万人，遍居世界各地，还有归侨、侨眷180万人<ref>{{cite web |url=http://www.jieyang.gd.cn/list.aspx?q_cid=3 |title=揭阳概况 |publisher=揭阳市人民政府 |access-date=2017-02-14 |archive-url=https://web.archive.org/web/20170215064733/http://www.jieyang.gd.cn/list.aspx?q_cid=3 |archive-date=2017-02-15 |dead-url=yes }}</ref>，亦是[[海峡西岸经济区|海峡西岸经济区]]成员城市。

== 歷史 ==
揭阳历史悠久，见诸史载已有2200余年，得名于古[[五岭|五岭]]之一的揭阳岭，[[春秋|春秋]][[战国|战国]]时属百越地。[[秦始皇|秦始皇]]平越后，于秦始皇三十三年（前214年）设立揭阳戍守区，隶属南海郡。[[汉武帝|汉武帝]]元鼎六年（前111年）建制揭阳县，管辖今[[潮汕|潮汕]]、[[梅州|梅州]]和闽南的龙溪、漳浦等地方。东晋成帝咸和六年（331年）揭阳拆为海阳、潮阳、海宁、绥安4县。以后几经复废，至宋绍兴十年（1140年），又设立揭阳县。中華人民共和國成立後，揭阳县先后隶属潮汕专区、粤东行政区、汕头专区、汕头市。

揭陽縣本地方言為潮州話、北部及西部為客家方言區。1983年7月設立梅州市（梅州地區）之前，揭陽縣北部客家方言區的湯坑、八鄉、豐良、潘田8個[[公社|公社]]劃入[[豐順縣|豐順縣]]。西部客家方言區13[[公社|公社]]成立为[[揭西县|揭西县]]。至此揭陽縣一縣分拆為三縣。

1991年12月7日，国务院批准揭阳撤县建市（地级），拆揭陽縣榕城等地為[[榕城區|榕城區]]，其餘地方分為[[揭東縣|揭東縣]]。现辖榕城区、揭东区、揭西、惠来2县2区，代管普宁市（县级）。市区榕城是[[广东省|广东省]]历史文化名城，被誉为“水上莲花”。市人民政府驻东山片区沿江路。

2012年12月，国务院作出《关于同意广东省调整揭阳市部分行政区划的批复》，批准对揭阳市部分行政区划实施调整，揭东县撤县设区，将揭阳市榕城区的磐东街道划归[[揭东区|揭东区]]管理，以原揭东县(不含地都镇、炮台镇、登岗镇)和榕城区磐东街道的行政区域为揭东区的行政区域；原揭东县的地都镇、炮台镇、登岗镇划归榕城区管理。

== 地理 ==
揭阳市地处广东省东部，东邻汕头、潮州，西接汕尾，南濒南海，北靠梅州，[[北回归线|北回归线]]从南部穿过。陆地面积5240.5平方公里。大陆海岸线长136.9公里，沿海岛屿30多个。地势自西向东倾斜，低山高丘与谷地平原交错相间，分布不均，西北部和西南部多为丘陵、山地，中部、南部和东南部都是广阔肥沃的[[榕江|榕江]]冲积平原和滨海沉积平原。 最高峰[[李望嶂|李望嶂]]海拔1222米，位于[[揭西|揭西]]北部和梅州[[五华|五华]]交界处，东经115°51'，北纬23°40'，为莲花山脉东段支脉大北山主峰，东北方向与鸿图嶂相望。

=== 气候 ===
揭阳属[[亚热带季风性湿润气候|亚热带季风性湿润气候]]，日照充足，雨量充沛，终年无雪少霜，年平均气温21.4摄氏度，平均降水量1723毫米。夏秋间常受[[强热带风暴|强热带风暴]]袭击，有时因季风活动反常或寒潮侵袭，会出现冬春干旱或早春低温阴雨天气。

揭阳市是广东省人口较多且较为稠密的地区之一。根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为587.70万人<ref>{{cite book |author=揭阳市统计局|title=《揭阳市2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加了63.96万人，增长12.21%。年平均增长率为1.16%。其中，男性为299.91万人，占51.03%；女性为287.79万人，占48.97%。总人口性别比（以女性为100）为104.21。0—14岁的人口为145.07万人，占24.68%；15—64岁的人口为399.47万人，占67.98%；65岁及以上的人口为43.16万人，占7.34%。

===民族===
汉族人口占总人口的99.99％以上，另有极少数的回、满、苗、壮、黎、傣、京等少数民族。

===侨乡===
揭阳是全国著名侨乡，港澳台同胞和旅外华侨320多万人，遍居世界各地，还有归侨、侨眷180万人。

===语言===
地方語言主要是[[潮汕話|潮汕話]]，也有部分[[客家语|客家语]]。其中潮汕話分布在大部分县区而客家语则主要分布在揭西县和揭东区西北部。

===公交车===
以前出租车及公共汽车未投入运营時，市民普遍依靠人力三轮车出行。随后逐渐被出租车及公共汽车代替。目前老城区仍保留部分人力三轮车。

==经济==
揭阳市属于广东省欠发达地区，2011年GDP为1228.2亿元，人均GDP远低于广东省平均水平和全国平均水平。民营企业比较发达，主要有五金、玉石加工、服装等行业。

===公共传媒===
{{seealso|揭阳广播电视台}}
南方广播影视传媒集团[[揭阳广播电视台|揭阳广播电视台]]于2004年3月23日正式挂牌成立。

根据《关于印发南方广播影视传媒集团揭阳广播电视台机构编制方案的通知》（揭市机编[2005]57号），揭阳广播电视台是揭阳市委宣传部直接领导下的正处级事业单位和南方广播影视传媒集团的成员单位。广播、电视自办节目分别使用 “揭阳人民广播电台”和“揭阳电视台” 呼号。 到2009年度，拥有固定资产8000多万元，电视发射塔一座，光纤传输网450多公里，广播电视平均人口覆盖率为97%。广播电台的综合广播频率FM103.9MHz，农业广播频率FM106.5MHz，交通旅游广播频率FM95.2MHz；电视台发射频率为 535.25 MHz（21频道），发射功率10千瓦；有线广播电视网络带宽750兆赫兹，共传输125套节目（其中数字125套、模拟44套） 。 <ref>{{zh}}{{cite web|author=揭阳广播电视台|title=揭阳广播电视台业务职能|url=http://www.jyrtv.cn/btjs/index.html|publisher=揭阳广播电视台|accessdate=2011-11-18}}</ref>

在报业方面，揭阳日报社发行中共揭阳市委机关报《揭阳日报》。

==著名人物==
*[[林鸣鸾|林鸣鸾]]（1504 - **）揭阳县霖田都钱家寨人（揭西县钱坑镇）。明朝任都匀府知府。
*[[郑大进|郑大进]]（1709年－1782年），字誉捷、号谦基、退谷，广东揭阳梅岗人。清朝政治人物。
*[[曾习经|曾习经]]（1867年9月18日－1926年5月18日），又名曾刚甫，号蛰庵居士，广东揭阳棉湖镇（今属揭西县）人。著名诗人，梁启超认为其是“有清易代之际第一完人”。
*[[许涤新|许涤新]]（1906年－1988年），广东揭阳棉湖镇（今属揭西县）人。经济学家，中国共产党党员。
*[[许地山|许地山]]（1894年2月3日－1941年8月4日），20世纪上半叶中国作家、学者、基督徒。名赞堃，字地山，笔名落花生（落华生）。
*[[丘之纪|丘之纪]]（1902年-1937年），广东揭阳人，[[中華民國陸軍軍官學校|黄埔军校]]三期学生。20世纪30年代先后任民国财政部税警总团团长、副总团长。1937年10月28日，奉令于上海刘家宅阻击日军。11月2日与敌激战时，肉搏殉国。牺牲后，[[国民政府|国民政府]]追赠陆军少将。
*[[杨遵仪|杨遵仪]]（1908年10月7日－2009年9月17日），广东揭阳洋淇乡寨内村（今属揭东县登岗镇）。中国大陆古生物学和地层学的奠基人以及地层古生物教育事业的开拓者。
*[[洪子诚|洪子诚]]（1939年4月－），北京大学中文系教授、博士生导师，中国文学理论研究者。
*[[林兴胜|林兴胜]]揭西县钱坑人，星河奖（潮汕诺贝尔）创始人之一。
*[[许银川|许银川]]（1975年－），中国广东惠来人，中国象棋特级大师。
*[[傅海峰|傅海峰]]（1984年1月2日－），中国羽毛球男子双打运动员，广东揭阳市惠来县人，与蔡赟搭档后成为中国继李永波及田秉毅之后的全英公开赛冠军(2005年)。以后场大力杀球著称。
*[[孙淑伟|孙淑伟]]（1976年1月－），中国广东揭阳人，著名跳水运动员。
*[[林毛根|林毛根]]（1929年－2007年6月8日），中国广东揭阳人，著名古筝演奏家，中国音乐学院民族音乐研究班客席导师，中国音乐家协会会员，中国音乐家协会古筝学会顾问，汕头市音乐家协会名誉主席，一代潮乐宗师。
*[[林世鏗|林世鏗]]（1931年－）中国广东惠来縣葵潭镇人，香港慈雲閣永遠主席
*[[林永青|林永青]]（1951年5月－2011年6月17日）中国广东[[惠来縣|惠来縣]]靖海镇人，中國解放軍海軍少將
*[[戎鐵文|戎鐵文]]（1952年10月－）中国广东惠来縣靖海镇人，歷任第十屆全國人大代表、中共汕尾市委書記、廣東省委組織部副部長、廣東省人大華僑民族宗教委員會主任
*[[揭姓|揭姓]]起源于地名[[揭阳|揭阳]]，始祖为[[秦朝|秦]][[西汉|汉]]时期揭阳县令[[揭猛|揭猛]]。揭猛原名史定，因战功由皇帝赐姓“揭”。<ref>[http://news.sina.com.cn/o/2005-05-28/17426015124s.shtml 岭南写真：中华揭姓源自粤东揭阳]</ref>
