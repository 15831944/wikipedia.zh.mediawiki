'''瑟莫蘇克'''（{{lang-kl|Kommuneqarfik Sermersooq}}，意為「多冰之地」）是[[格陵蘭|格陵蘭]]一個新的自治市，自2009年1月1日起開始運作。<ref name="sooq"/>首府為[[努克|努克]]（曾被稱為戈特霍布）同為[[格陵蘭|格陵蘭]]首府，亦是全格陵蘭人口最多的自治市，2013年1月共有21,868名居民。<ref name="stat2013"/>自治市包括過去在格陵蘭東部和西南部的自治市，全部以當時最主要聚居地命名：<ref>[http://www.sermersooq.gl/da/om_kommune/byerne Sermersooq Municipality] {{Wayback|url=http://www.sermersooq.gl/da/om_kommune/byerne |date=20121118054119 }} {{da icon}}</ref>

== 地理 ==
自治市位於格陵蘭東部和西南部，面積達53.2萬平方公里，<ref name="sooq"/>是世上按面積計算第二大自治市，<ref name="sooq"/>僅次於[[卡蘇伊特薩普|卡蘇伊特薩普]]。南邊與[[庫雅雷克|庫雅雷克]]接壤。西北面與[[凱克卡塔|凱克卡塔]]接壤，更北的地方則與卡蘇伊特薩普接壤。後者的邊界按西經45度劃定。瑟莫蘇克最北端與東北格陵蘭國家公園接壤。東南則因[[丹麥海峽|丹麥海峽]]與[[冰島|冰島]]分開。

== 交通 ==
瑟莫蘇克是兩個橫跨東西兩岸的格陵蘭自治市之一，不過亦是唯一一個連接兩側需要依賴航空連接的自治市。從努克機場到庫盧蘇克機場有由[[格陵蘭航空|格陵蘭航空]]全年營運的定期航班。<ref>{{cite web |url=http://book.airgreenland.com/ |publisher=Air Greenland |title=Booking system |accessdate=12 June 2010 |deadurl=yes |archiveurl=https://web.archive.org/web/20100422211033/http://book.airgreenland.com/ |archivedate=2010年4月22日 }}</ref>在西岸亦有短途來往努克和帕繆特機場的航班。

== 語言 ==
{{main|格陵蘭語}}
西岸居民主要說西格陵蘭方言「Kalaallisut」。[[丹麥語|丹麥語]]亦在主要城鎮中使用。東岸居民則主要說東格陵蘭方言「Tunumiit oraasiat」。
