'''穴熊圍'''（日語：穴熊囲い，平假名：あなぐまがこい），是[[將棋|將棋]]中圍玉（守備的陣形）之一，無論[[居飛車|居飛車]]或[[振飛車|振飛車]]都有使用的圍玉。雖然到完成需要花很多手數，卻是最堅固的圍玉。

== 概要 ==
把端（第1或第9行）的[[香車|香車]]往前移一格再將[[玉将|玉]]潛入其下的形狀正如[[熊|熊]]潛藏進洞穴，這正式名稱的由來。古代也有「岩屋」、「獅子入洞」的說法。

原本穴熊是[[振飛車|振飛車]]用來對付[[居飛車|居飛車]]的圍玉，此'''[[振飛車穴熊|振飛車穴熊]]'''是從[[江戸時代|江戸時代]]就存在的戰法。然而在以前的職業棋士間評價很低，有''「下穴熊之類的不會變強」''<ref>『日本将棋用語事典』p.8。</ref>的偏見。然而在進入1970年代後[[大内延介|大内延介]]等人證明了它的優秀性。其後，[[田中寅彦|田中寅彦]]等人在對振飛車戰中圍出了居飛車的穴熊。之後「'''[[居飛車穴熊|居飛車穴熊]]'''」越來越整備，席捲了將棋界。現代，如何對抗居飛車穴熊也已經成為了[[振飛車|振飛車]]方最大的課題之一。另外，在現代將棋中穴熊也不僅只出現在居飛車對振飛車的對抗型中，在[[相振飛車|相振飛車]]、甚至是[[矢倉|矢倉]]和[[角交換|角交換]]等相居飛車的戰型也常出現見機就換圍成穴熊的情況，可見穴熊已經成為了重視自玉堅固程度的現代將棋的象徴。

另外，桂馬跳離的樣子俗稱「脫褲子」，圍玉大大的弱化{{sfn|週刊将棋|2004|p=66}}。

== 優點 ==
金銀的連結密集因而非常堅固，再加上[[玉將|玉將]]在端離戰場十分遙遠，形成了「絕對不會被[[王手|王手]]（別稱'''Z'''）」的形，對手也因此不得不避免在攻略穴熊的時候犧牲大量棋子。基於這些優點，穴熊方得以採取大膽捨[[飛車|飛車]]和[[角行|角行]]等大子的作戰，被稱為「只有穴熊才能做到的攻擊」。

== 缺點==
首先，圍玉所需的手數非常多，對手常常在完成之前就挑起戰事。再來，圍玉完成之後因為棋子集中在盤上一側，自陣有許多空隙容易被對手打入[[角行|角行]]。雖然橫向的守備力很高，對上方或端攻較弱，在終盤還因為[[玉將|玉將]]躲在棋盤的角落，在被攻擊的時候無處可逃。最後，因為自陣空隙太多容易被對手[[入玉|入玉]]，這種時候就已完全沒有勝算。這種雖然圍玉完全沒有被壞卻投了得狀況被稱為「姿燒穴熊(烤全熊)」。

此外，居飛車穴熊則多有遭對手角道直射的疑慮。許多居飛車穴熊的攻略法都包含角道的利用。

== 攻略法 ==
利用桂、香、歩「小子的攻擊」很有效，因為即使被吃走了也很難做為防守用的棋子。特別是利用「と金」攻擊的方法，雖然速度比較緩慢卻很難招架。避開對手圍玉中守備的金銀轉攻端也是十分有効的作法，像是[[一間飛車|一間飛車]]、[[地下鐵飛車|地下鐵飛車]]等狙擊端的戰法。

作為居飛車穴熊對策，[[四間飛車|四間飛車]]的「[[藤井系統|藤井系統]]」是最有名的。省略自己的圍玉手數（通常都維持[[居玉|居玉]]），併用角道和端攻，企圖在對手完成居飛車穴熊前就攻略完成。其他對策也有[[中田功XP|中田功XP]]等戰法。

== 變形 ==
穴熊有許多種變形。

只用[[金將|金]][[銀將|銀]]2枚圍玉雖然不甚堅固卻保有自陣的均衡，攻擊的棋子也變得更多了。

用金銀3枚圍玉（如上圖）則是最一般的。然而振飛車穴熊的情況會有將2枚[[金將|金將]]放置於3九和4八與放置在3九和4九兩種樣子。

利用金銀4枚圍玉的穴熊非常堅固，以前大多由4枚[[銀冠|銀冠]]發展而來，不過也有一開始就以4枚穴熊為目標的情況。有業餘強手[[田尻隆司|田尻隆司]]發明的田尻穴熊、[[松尾歩|松尾]]流穴熊、還有Big4。

此外，振飛車穴熊也有將[[金將|金將]]擺在4七的情況。是一種不單單考慮圍玉堅硬更重視攻守兼備和自陣均衡的陣形。（[[広瀬章人|広瀬章人]]稱這個構型為「現代穴熊」並多用，留下了許多好成績。）。

==「居飛車穴熊戰法」訴訟==
關於將棋戰法「居飛車穴熊」的開山始祖是誰，曾獲支部名人戰優勝1回、赤旗名人戰優勝3回好成績的業餘強手大木和博曾表示「居飛車穴熊是自己發明的」，並向職業棋士[[田中寅彦|田中寅彦]]提起訴訟索取300萬円的賠償金，要求對方不要再承襲居飛車穴熊始祖的假名。

1999年6月，一審[[東京地方裁判所|東京地裁]]認為「二人皆為創始者」，駁回賠償金的請求。2000年3月，二審[[東京高等裁判所|東京高裁]]也支持一審判決｡2001年2月22日[[最高裁判所_(日本)|最高裁]]第1小法庭決定不予此件上訴。根據此決定，以二審東京高裁的判決定案。

其實，1968年第27期名人戰（大山4-升田0）第2局先手的[[升田幸三|升田幸三實力制第四代名人]]即以居飛車穴熊的概念<ref>當時的解說者以「稀有的左穴熊」記載。</ref><ref>棋譜在週刊将棋編「不滅の名勝負100」（毎日コミュニケーションズ）中可以確認。</ref>實踐於對抗後手[[大山康晴|大山康晴十五世名人]]的四間飛車。然而，實際上將居飛車穴熊作為現代戰法再編・体系化，使其作為真正的對[[振飛車|振飛車]]戰法流行於職業棋士間的是[[田中寅彦|田中寅彦]]。此外，在田中以前也有在實戰中無數次採用居飛車穴熊的[[西村一義|西村一義]]，可見田中多少也受到了西村的影響。
