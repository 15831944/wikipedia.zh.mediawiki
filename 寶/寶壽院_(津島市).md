== 歴史 ==
根據寺傳，在[[弘仁|弘仁]]9年（[[818年|818年]]）拜訪此地的[[空海|空海]]（弘法大師）為了受傳染病蔓延之苦的人們，在津島牛頭天王社（現在的[[津島神社|津島神社]]）旁邊，建設了奉祀祭拜[[藥師佛|藥師佛]]的建築為起源<ref name="rekishi">{{Cite web|url=http://houjyuinn.net/annai/history.html|title=宝寿院の歴史|accessdate=2012-12-18|date=2008-01-10|publisher=宝寿院|archive-url=https://web.archive.org/web/20091121191812/http://houjyuinn.net/annai/history.html|archive-date=2009-11-21|dead-url=yes}}</ref><ref>{{Cite web|url=http://www.tsushima-cci.or.jp/mati/wakuwakuweb/area/a_05.html|title=宝寿院|accessdate=2012-12-18|date=2012-01-25|publisher=津島入口網站|archive-url=https://web.archive.org/web/20071118064430/http://www.tsushima-cci.or.jp/mati/wakuwakuweb/area/a_05.html|archive-date=2007-11-18|dead-url=yes}}</ref>。至[[江戶時代|江戶時代]]，作為[[神宮寺|神宮寺]]的天王社在牆內設置了本地堂，玉牆西邊則分別建立了實相院、明星院、寶壽院，離神社較遠則建立了觀音坊<ref>津島市史、pp.208</ref>。

[[明治|明治時代]]，因為[[神佛分離|神佛分離]]而全面廢除了神宮寺，在[[1868年|1868年]]（[[明治|明治]]元年）舊曆5月至7月在境內的本地堂和鐘樓、[[宝箧印式塔|宝篋印塔]]被摧毀<ref>津島市史、pp.210-213</ref>。此時，應寶壽院的住持宥三等社僧之請求，將本地堂的本尊和經文等等曾全部移至實相院，但因為實相院、明星院的社僧決定[[還俗|還俗]]，在同年8月，從本地堂移來的佛像和佛畫、寶器，加上實相院、明星院的佛像等都一起移置寶壽院<ref name="p123">津島市史、pp.213-214</ref>。另外，這時候無人管理的觀音坊也轉移給寶壽院管理之下。宥三在此之後，進行神社和本地堂的再建<ref>津島市史、pp.217</ref>，但重建工作並未實現。

此段神佛分離的經緯仍殘存紀載在〈神仏分離顛末記宥三日記〉，現在被指定為津島市文化財。

=== 近代 ===
[[1959年|1959年]]（[[昭和|昭和]]34年）的[[颱風薇拉_(1959年)|薇拉颱風]]造成了很大的損壞；[[1974年|1974年]]（昭和49年）地藏堂、[[1981年|1981年]]（昭和56年）的本堂重建。

== 寶壽院舊藏文化資產 ==

* 舊寶壽院[[寶篋印塔|寶篋印塔]]（[[重要文化財|重要文化財]]）

: 有[[貞和|貞和]]三年（1347年）銘刻的石塔。在1967年變更了所有人，被移放在[[名古屋市|名古屋市]]内的個人邸宅内。之後，再移放至[[滋賀縣|滋賀縣]][[大津市|大津市]]坂本本町<ref>[http://www.pref.shiga.lg.jp/edu/sogo/iinkai/ma00/kaigiroku/h19/0324/files/houkoku-1.pdf 「重要文化財の所在地変更について」]（滋賀県網站）</ref><ref>[http://bunka.nii.ac.jp/db/heritages/detail/174422 文化遺産online]</ref>。

* 絹本著色佛涅槃圖（重要文化財）

: 1918年4月8日指定為重要文化財（當時的國寶），為寶壽院所有<ref>[{{NDLDC|2953814}} 國立國會圖書館數位資料收藏]</ref>。現在則由[[奈良国立博物馆|奈良國立博物館]]收藏<ref>[http://www.narahaku.go.jp/collection/756-0.html 收藏品基本資料]（奈良国立博物館網站）</ref>。
