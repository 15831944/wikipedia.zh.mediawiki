'''寶山車站'''是一曾存在於[[台灣|台灣]]桃園縣桃園市（今[[桃園市|桃園市]][[桃園區|桃園區]]），已廢線的[[臺灣鐵路管理局|臺灣鐵路管理局]][[林口線|林口線]]沿線的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]一座。

== 營運狀況 ==
* 在林口線試辦客運期間，本站以鄰近之寶山街住宅大樓居民為其主要客源。

== 車站週邊 ==
*新光三越桃園大有店
*大業黃昏市場
*慈濟桃園靜思堂
*桃園大飯店

== 歷史 ==
*2005年11月28日：正式通車，站名取自鄰近道路「寶山街」之名。
*2012年12月28日：隨著林口線停駛而廢止。
