'''寶林禪寺'''，或稱'''[[寶林寺|寶林寺]]'''，為[[香港|香港]]一座[[佛教|佛教]]寺廟，位於[[新界|新界]][[大嶼山|大嶼山]][[鳳凰山_(香港)|鳳凰山]]半麓，地塘仔半216號地段。除年邁高僧外，寺內僧侶都要下田耕種，糧食自給自足。<ref name="ap080326-9841">{{cite news|url=https://hk.news.appledaily.com/local/daily/article/20080326/10909841 |title=出家藝員何寶生寶林寺抗賊 遭鐵枝打傷「個賊第日有因果報應」 |publisher=蘋果日報 |date=2008-03-26 }}</ref>

寶林禪寺在[[寶蓮禪寺|寶蓮禪寺]]以東約三公里，僅有山路可達，從[[昂坪|昂坪]]前往，要走大約一小時山路。由[[東涌|東涌]]前往，亦要走山路半小時。因位處偏僻，該寺被譽為香港最清靜的佛門地。<ref name="ew110425">{{cite news|url=http://eastweek.my-magazine.me/main/12158 |title=涉騙藝人千萬元 九指狂僧大鬧寶林禪寺內幕 |publisher=東周網 |date=2011-04-25 }}</ref>

該寺由[[悟明|悟明]]法師於1955年開創。悟明法師以苦行見稱，在鳳凰山獨居二十多年。1956年，悟明法師過世後，寶林禪寺由[[聖一|聖一]]和尚接管。寶林禪寺殿堂建築較為簡樸，包括：大殿（兼作禪堂）、毗盧遮那殿、地藏殿及地藏塔等。禪寺大殿有「寶林開鳳山，梵刹接曹溪」門聯。<ref>{{cite web |url=http://www.openlife.com.hk/%E9%A6%99%E6%B8%AF%E5%BB%9F%E5%AE%87%E5%90%8D%E5%86%8A/%E5%AF%B6%E6%9E%97%E7%A6%AA%E5%AF%BA |title=香港廟宇名冊：寶林禪寺 |publisher= |date= |access-date=2018-02-05 |archive-url=https://web.archive.org/web/20170904152525/http://www.openlife.com.hk/%E9%A6%99%E6%B8%AF%E5%BB%9F%E5%AE%87%E5%90%8D%E5%86%8A/%E5%AF%B6%E6%9E%97%E7%A6%AA%E5%AF%BA/ |archive-date=2017-09-04 |dead-url=yes }}</ref><ref>{{cite web|url=http://www.fushantang.com/1005d/e4033.html |title=寶林禪林 |publisher= |date= }}</ref>

==著名修行者==
曾經在寶林禪寺修行的名人包括：

*[[黃元申|黃元申]]：香港前藝人，1989年在寶林禪寺出家為僧，法號衍申。<ref name="ap170912">{{cite web |url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20170912/20149753 |title=【藝人落髮】何寶生剃度棄百億資產 | date=2017-09-12}}</ref><ref>{{cite web |url=http://www.book853.com/show.aspx?id=2165&cid=147 |title=黃元申出家後的生活 | date=2012-05-04}}</ref><ref>{{cite web |url=http://blog.udn.com/acewang3005/6956100 |title=永遠的霍元甲黃元申信佛原因 | date=2012-10-18}}</ref>

*[[莊文清|莊文清]]：香港前藝人，1991年在寶林禪寺出家為尼，法號衍藏。<ref>{{cite web |url=http://blog.sina.com.cn/s/blog_6a61c0ff0102e4mp.html |title=藝人莊文清出家廿多年 |publisher= |date=2013-09-12}}</ref><ref>{{cite web |url=http://www.beihaichanyuan.org/fomen/gat/2010-06-25/3354.html |title=出家逾廿載 莊文清自揭身世 |publisher=佛門動態 |date=2010-06-25 |access-date=2018-02-06 |archive-url=https://web.archive.org/web/20110114054220/http://www.beihaichanyuan.org/fomen/gat/2010-06-25/3354.html |archive-date=2011-01-14 |dead-url=yes }}</ref>

*[[何寶生|何寶生]]：香港前藝人，2005年在寶林禪寺出家為僧，法號道生。<ref name="ap170912" />
