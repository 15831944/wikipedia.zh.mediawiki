'''寶琳站'''（{{lang-en|'''Po Lam Station'''}}）是[[香港|香港]][[港鐵|港鐵]][[將軍澳綫|將軍澳綫]]在新界的終點站之一，位於[[香港|香港]][[新界|新界]][[西貢區|西貢區]][[將軍澳|將軍澳]][[寶琳|寶琳]]貿業路旁，[[新都城中心|新都城中心]]一至三期中間，2002年8月18日啟用，承建商是[[前田建設|前田建設]]工業株式會社。寶琳站雖然鄰近[[寶林邨|寶林邨]]，但寶琳站名則取自附近的街道[[寶琳北路|寶琳北路]]，二者的「林」和「琳」寫法不同。

==車站設計==
寶琳站為半開放設計，月台、大堂同位於地面，設有空調。此站是少數可直接由月台層走出車站的地面車站之一，在將軍澳綫車站中，此乃獨一無二。

鐵路路軌由地面在隔音牆內慢慢下降至地底，在寶琳站南端一路沿著[[景林邨|景林邨]]旁建有一個綠化[[公園|公園]]，一方面覆蓋路軌，另一方面可以為區內提供綠化空間，可沿路步行至坑口站。

=== 大堂 ===
{{ multiple image | align = right | direction = vertical | width = 220

大堂非收費區內，及A、B出入口通道旁，均設有各類型的商店，如[[便利商店|便利店]]、[[時裝店|時裝店]]、[[麵包糕餅店|麵包糕餅店]]、診所等，以及不少自助服務設施供乘客使用，包括[[自動櫃員機|自動櫃員機]]、[[自動販賣機|自動售賣機]]和自動照相機<ref name="shop"/>。客務中心位於大堂中央，近出入口行人電動扶梯旁則設有[[香港郵政|香港郵政]][[郵箱|郵箱]]，方便乘客投寄[[郵件|郵件]]<ref>{{cite web|url=http://www.mtr.com.hk/chi/getting_around/post.html|title=鐵路四處遊－郵箱服務|work=港鐵網頁|publisher=港鐵公司|accessdate=2014年4月14日|language=zh-hant}}</ref><ref>{{cite web|url=http://app1.hongkongpost.hk/CGI/location/box_content.jsp?region=%27New%20Territories%27&lang=zh_TW#saikung |title=郵箱位置表－西貢區 |work=香港郵政網頁 |publisher=香港郵政 |accessdate=2014年4月14日 |language=zh-hant |deadurl=yes |archiveurl=https://web.archive.org/web/20140415110401/http://app1.hongkongpost.hk/CGI/location/box_content.jsp?region=%27New%20Territories%27&lang=zh_TW |archivedate=2014年4月15日 }}</ref>。而B出入口旁亦設有「[[會員服務站|會員服務站]]」<ref>{{cite web|url=http://www.mtr.com.hk/ch/customer/services/csp.html|title=鐵路四處遊－「會員服務站」|work=港鐵網頁|publisher=港鐵公司|accessdate=2016年9月26日|language=zh-hant}}</ref>。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Po Lam Station 2017 part1.jpg | caption1 = 1號月台﹙往北角方向﹚（北面）
| image2 = Po Lam Station 2017 07 part2.jpg | caption2 = 1號月台﹙往北角方向﹚（南面）
}}
與很多港鐵站不同，月台與大堂位處同一層。寶琳站只有一個[[側式月台|側式月台]]，並設有[[月台幕門|月台幕門]]。[[列車|列車]]只能在同一個方向到達和離開，因此列車只會停留一段短時間，以免妨礙下班列車進站；正因如此設計，不時列車會為了保持[[閉塞_(鐵路)|安全距離]]而在未到站時減速，甚至停下，直至前方列車離開。另外，在車站1號月台後方，設有一條不對外開放的路軌供列車停泊，但列車不能從該路軌駛進1號月台上客；而列車進入1號月台後，將無法駛進該段路軌。

寶琳站主要出入口設於貿業路，有五條[[行人天橋|行人天橋]]連接港鐵站及公共運輸交匯處，可以直接由港鐵站前往[[新都城|新都城]]各期和叠翠軒，亦可經行人天橋到達茵怡花園、寶林邨及欣明苑。車站外設有升降機，可以經C出口前往，連接行人天橋。

==鄰接車站==
寶琳站是將軍澳綫的北行終點站之一，乘客可在此乘坐往北角列車，往返[[香港島|香港島]]、[[九龍|九龍]]及[[新界|新界]]各地。如果乘客需要在該站往[[康城站_(香港)|康城]]，需在[[將軍澳站|將軍澳站]]過對面月台轉乘往康城方向的列車。

==未來發展==
===東九龍綫===
{{main|東九龍綫 (2014年方案)}}
據《[[鐵路發展策略2014|鐵路發展策略2014]]》，[[香港特區政府|香港特區政府]]建議興建[[東九龍綫_(2014年方案)|東九龍綫]]，連接[[將軍澳綫|將軍澳綫]]寶琳站及[[觀塘綫|觀塘綫]][[鑽石山站|鑽石山站]]，走線途經[[觀塘|觀塘]]半山北部，中途設[[康景站|康景站]]、[[寶達站|寶達站]]、[[秀茂坪站|秀茂坪站]]、[[順天站|順天站]]及[[彩雲站|彩雲站]]。報告指出東九龍綫可充當[[將軍澳|將軍澳]]地區和[[九龍|九龍]]之間行程的替代路線，如將軍澳綫的列車服務受阻，受影響的乘客可取道這條替代路線以轉乘觀塘綫<ref name="rds2014">《[http://www.thb.gov.hk/tc/psp/publications/transport/publications/rds2014.pdf 鐵路發展策略2014]》（第30-32頁），2014年9月，運輸及房屋局</ref>。
