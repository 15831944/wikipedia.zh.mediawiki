'''寶雲道'''（[[英文|英文]]：'''{{lang|en|Bowen Road}}'''）是[[香港|香港]]的一條道路，連接[[香港島|香港島]][[半山區|半山區]]至[[黃泥涌峽|黃泥涌峽]]一帶。道路西接[[中環|中環]][[馬己仙峽道|馬己仙峽道]]，穿過[[灣仔峽|灣仔峽]]的[[灣仔峽道|灣仔峽道]]，東接[[司徒拔道|司徒拔道]]、[[黃泥涌峽道|黃泥涌峽道]]及[[大坑道|大坑道]]交界。道路以第九任[[香港總督|香港總督]][[寶雲|寶雲]]命名。[[香港日治時期|香港日治時期]]，道路曾被改名為'''霧島通'''。

== 歷史 ==
寶雲道於1888年落成，不少部份以[[棧道|棧道]]型式興建，底下敷設了水管，以英式橋墩支撐，將[[食水|食水]]由香港島[[南區_(香港)|東南部]][[大潭_(香港)|大潭]]的[[大潭水塘|大潭水塘]]引到[[中環|中環]]雅賓利谷的儲水庫，稱為'''寶雲渠'''或'''寶雲輸水道'''，其'''21孔拱券段'''更於2009年9月18日被列為[[香港法定古蹟|香港法定古蹟]]。

由於寶雲道是香港繼[[皇后大道|皇后大道]]及[[堅尼地道|堅尼地道]]之後興建的第三條主要道路，因此當時亦有'''三馬路'''之稱。

1903年，寶雲道近司徒拔道立下了[[維多利亞城界碑|維多利亞城界碑]]，標誌著[[維多利亞城|維多利亞城]]的南部邊界。

== 特色 ==
寶雲道離海平線大約100米，兩旁種有大量樹木，大部份為平坦道路及不可行車，離開市區不遠，從灣仔或中環步行20分鐘可以到達，此20鐘步行剛好充當跑步前熱身，所以甚受緩步跑人士歡迎。寶雲道沿途也建有不同之健身器具供不同人士健身用。

遊人可以留意道路兩旁，很多時可以找到松鼠的芳蹤或在天空飛翔的麻鷹。

== 沿路著名地點 ==
* [[姻緣石|姻緣石]]
* 寶雲道16號，獨立大宅 [[楠樺居|楠樺居]]
* [[蘭谷|蘭谷]]，位於寶雲道與[[寶雲徑|寶雲徑]]交界處的[[峽谷|峽谷]]裡。
* 活水，在寶雲道花園後方，60年代提供山水給不喜有氯化物自來水的市民沖茶用，現已不能飲用。
* 灣坊亭，由晨運客集資興建，建於1991年，前身為1961年建成的寶雲亭，提供茶水供應及避雨用途，1980年代後因為膠樽裝水普及而停止。

== 交通 ==
寶雲道並沒有任何巴士行走，只有[[馬己仙峽道|馬己仙峽道]]至[[波老道|波老道]]一段有[[香港島專線小巴9號線|香港島專線小巴9號線]]途經。此外亦可以乘坐 6, 15, 19, 41A, 63, 66 或 76 號巴士於[[司徒拔道|司徒拔道]]、[[大坑道|大坑道]]及[[黃泥涌峽道|黃泥涌峽道]]之交界（迴旋處）下車步行前往寶雲道。

寶雲徑至司徒拔道一段寶雲道禁止汽車駛入。
