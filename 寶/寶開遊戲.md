'''宝开遊戲'''（{{lang-en|'''PopCap Games'''}}）是一家[[美國|美國]][[休閒遊戲|休閒遊戲]][[游戏开发者|開發商]]和[[游戏发行商|发行商]]，總部設在[[華盛頓州|華盛頓州]][[西雅圖市|西雅圖市]]。它由John Vechey、Brian Fiete和Jason Kapalka於2000年成立，在[[上海|上海]]、[[旧金山|旧金山]]、[[芝加哥|芝加哥]]、[[温哥华|温哥华]]和[[都柏林|都柏林]]设有研发基地。大多數宝开的遊戲版本能夠在自由的形式收費。宝开遊戲的《[[宝石迷阵|宝石迷阵]]》系列已售出超過2500萬份，下载超过1.5亿次。《[[祖玛|祖玛]]》也是其原创的重量级作品。宝开遊戲支持涵盖[[浏览器|浏览器]]、 [[Windows|Windows PC]]和[[Mac|Mac]]、[[Xbox|Xbox]]、[[Xbox360|Xbox360]]、[[PlayStation_3|PlayStation 3]]、[[手機|手機]]、[[掌上電腦|掌上電腦]]、[[iPod_nano|iPod nano]]、[[iPhone|iPhone]]和[[iPod_touch|iPod touch]]等觸摸和移動裝備。

== 开发 ==
其构架宝开是一套社区形式开放性的[[软件框架|软件框架]]——SexyApp Framework(PopCap Games Framework)，这套软件开发工具包使用C++。由此衍生出的[http://sourceforge.net/projects/tuxcap/ TuxCap]支持Linux和Mac OS X，基本兼容宝开并提供额外的python绑定，物理引擎等功能。

== 影响 ==
宝开的开发套件造福了公司之外的开发者，[http://www.tweeler.com/ Tweeler]是一个为这些独立游戏开发者专门提供服务的网站，包含了很多用[[TuxCap|TuxCap]]移植到[[Linux|Linux]]和[[Mac_OS_X|Mac OS X]]的游戏。

== 争议 ==
宝开游戏在AppStore中国区售卖的《植物大战僵尸》iOS版并未像其他大多数公司的做法，而是采用了添加中文语言重新单独售卖的方式，遭到许多玩家不满，同时宝开未对此进行正面回应。<ref>[http://itunes.apple.com/cn/app/plants-vs.-zombies/id350642635?mt=8 植物大战僵尸 AppStore售卖页]</ref>

宝开在《植物大战僵尸》iOS版参加圣诞促销活动的同样是英文版，更加重了玩家对宝开不重视中国大陆市场的看法。<ref>[http://www.iapps.im/archives/25210/ 【双版促销】植物大战僵尸：Plants vs. Zombies] {{webarchive|url=https://archive.is/20120804013938/http://www.iapps.im/archives/25210/ |date=2012-08-04 }}</ref>

== 收購 ==
2011年7月，寶開遊戲被[[美商藝電|美商藝電]]以7.5億美元收購。
