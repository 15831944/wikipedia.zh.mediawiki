'''寶瓶星號'''（'''SuperStar Aquarius'''）是[[麗星郵輪|麗星郵輪]]旗下的一艘豪華[[郵輪|郵輪]]，[[巴哈馬|巴哈馬]]籍，於2007年加入麗星郵輪船隊。現時以[[台灣|台灣]][[基隆港|基隆港]]為[[母港|母港]]。

== 服務歷史 ==
* 2007年，寶瓶星號加入麗星郵輪船隊。
* 2011年11月，寶瓶星號[[中國|中國]][[三亞|三亞]]母港首航。
* 2012年11月，開始第二季三亞母港航次。
* 2013年11月，寶瓶星號逢周日會由[[沙巴亞庇|沙巴亞庇]]出發，前往[[汶萊|汶萊]][[穆阿拉|穆阿拉]]與[[砂拉越|砂拉越]][[民都魯|民都魯]]。<ref>{{cite web |url=http://www.starcruises.com/media/837820/20131112%20SSQ%20Bintulu_CHI.pdf |title=存档副本 |accessdate=2015-11-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304105609/http://www.starcruises.com/media/837820/20131112%20SSQ%20Bintulu_CHI.pdf |archivedate=2016-03-04 }}</ref>
* 2014年11月，開始沙巴亞庇至[[菲律賓|菲律賓]][[巴拉望島|巴拉望島]][[公主港|公主港]]航次。<ref>{{cite web |url=http://www.starcruises.com/media/1309277/20141110%20SSQ%20first%20call%20to%20Puerto%20Princesa%20CHI.pdf |title=存档副本 |accessdate=2015-11-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20160126162808/http://www.starcruises.com/media/1309277/20141110%20SSQ%20first%20call%20to%20Puerto%20Princesa%20CHI.pdf |archivedate=2016-01-26 }}</ref>
