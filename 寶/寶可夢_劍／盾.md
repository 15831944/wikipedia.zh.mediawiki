{{vgname|寶可夢 劍·盾|ja=ポケットモンスター ソード·シールド|en=Pokémon Sword & Shield}}是一款由[[Game_Freak|Game Freak]]開發，[[神奇寶貝_(企業)|Pokemon Inc.]]發行，並在[[任天堂Switch|任天堂Switch]]平台上發售的[[電子角色扮演遊戲|角色扮演遊戲]]。本作為[[精靈寶可夢|精靈寶可夢]]系列第八世代的正統續作。

本作於2019年2月27日的[[任天堂直面會|任天堂直面會]]上公佈，並將於2019年冬季進行全球發售<ref>{{Cite web |url=https://www.iwate-np.co.jp/article/oricon/2130393|title=ポケモン完全新作は“ソード・シールド”　今冬に世界同時発売|publisher=ORICON NEWS|date=2019-02-27|language=ja}}</ref><ref>{{Cite web |url=https://variety.com/2019/gaming/news/pokemon-sword-shield-switch-1203150753/|title=‘Pokemon Sword and Shield’ Announced For Nintendo Switch|publisher=[[綜藝_(雜誌)|綜藝]]|date=2019-02-27|language=en}}</ref>。

== 开发 ==
[[File:Pokémon_Galar_Region.jpg|thumb]][[大不列顛|大不列顛]]島嶼作為設計籃本<ref>{{cite web |last1=Barder |first1=Ollie |title=The Galar Region In 'Pokémon Sword and Shield' Looks A Lot Like The United Kingdom |url=https://www.forbes.com/sites/olliebarder/2019/02/27/the-galar-region-in-pokemon-sword-and-shield-looks-a-lot-like-the-united-kingdom |website=Forbes |accessdate=February 28, 2019 |date=February 27, 2019}}</ref>]]
2017年，「The Pokémon Company」的總裁 Tsunekazu Ishihara 在[[E3|E3游戏展会]]中宣布新的[[宝可梦系列|宝可梦系列]]正处于开发阶段,同時表示开发时间不会超过一年<ref>{{cite web |url = https://www.ign.com/articles/2017/06/13/e3-2017-a-core-pokemon-rpg-is-in-development-for-nintendo-switch |title = "E3 2017: A Core Pokemon RPG Is in Development for Nintendo Switch" |publisger =[[IGN|IGN]] |date = 2019-03-20]]</ref>

2018年，「The Pokémon Company Janpan」的Junichi Masuda宣布新的宝可梦系列将于2019年下半年推出于[[Nintendo_Switch|Nintendo Switch]]平台上。同时，也表示本作将会增加更多精灵以及更多的polished graphics<ref>{{Cite web |url = https://nintendoeverything.com/full-comments-from-pokemons-president-on-the-lets-go-games-and-2019-entries/ |title = Full comments from Pokemon's president on the Let's Go games and 2019 entries - Nintendo Everything |date = 2019-03-20 |publisher = nintendoeverything.com}}</ref>

2019年，游戏开发商在「Pokémon Dierct」释出游戏宣传片，并将发售日期定于当年冬季。<ref>{{Cite web |url = https://www.bbc.com/news/newsbeat-47387396 |title =  "Pokemon: Nintendo announces two new games, Sword and Shield, for the Switch" |publisher = [[BBC|BBC]] |date = 2019-02-27}}</ref>
