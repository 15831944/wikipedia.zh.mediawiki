《'''鄉村路'''》（{{lang-ja|カントリー・ロード}}）為[[日本|日本]][[女性|女性]][[聲優|聲優]][[本名陽子|本名陽子]]在1995年發表的個人生涯第一張音樂單曲作品，為同年[[吉卜力工作室|吉卜力工作室]][[動畫|動畫]]《[[心之谷|心之谷]]》的片尾曲。

== 歌曲內容 ==
此為基於由美國鄉村民謠歌手[[約翰·丹佛|約翰·丹佛]]在1971年發表的單曲《[[Take_Me_Home_Country_Roads|Take Me Home Country Roads]]》所改寫歌詞的翻唱歌曲，並由在動畫中為女主角「月島雫」配音的本名陽子擔任演唱。<ref>{{Cite book |script-title=ja:スタジオジブリ作品関連資料集 Vol.5 |location=日本 |publisher=[[吉卜力工作室|吉卜力工作室]] |pages=52頁 |ISBN=4-19-860660-9 |accessdate=2012月12月4日 }}</ref>

歌詞內容主要由該動畫製作人[[鈴木敏夫|鈴木敏夫]]的女兒[[鈴木麻實子|鈴木麻實子]]改寫，而製作成員之一的[[宮崎駿|宮崎駿]]亦協助歌詞方面補充。另原先麻實子寫的一句歌詞{{nihongo|「孑然一身、不帶著任何雲彩朝向城鎮邁進」|ひとりで生きると 何も持たずに まちを飛びだした}}，之後被宮崎駿提議改成{{nihongo|「別害怕孤單、勇敢地生存，夢想著」|ひとりぼっち おそれずに 生きようと 夢見ていた}}，雖然當時此片導演[[近藤喜文|近藤喜文]]是支持麻實子所寫的方案內容。<ref>{{Cite web |url=http://www.asahi.com/travel/traveler/TKY200608260136.html |script-title=ja:月島雫と天沢聖司―東京・多摩 |work=Asahi |language=ja |accessdate=2012年12月4日 }}</ref>

本名陽子在歌曲發表十數年後的2007年、與音樂家[[DE_DE_MOUSE|DE DE MOUSE]]合作將《鄉村路》重新演唱，並收錄在同年發行的重新詮釋吉卜力動畫系列歌《{{lang|ja|キラキラジブリ}}》專輯中。<ref>{{Cite web |url=http://www.jetsetrecords.net/p/816003246122 |script-title=ja:HAPPY SYNTHESIZER / DE DE MOUSEと本名陽子 - となりのトトロ / カントリーロード |work=Jet Set Records |language=ja |accessdate=2012年12月4日 }}</ref>
