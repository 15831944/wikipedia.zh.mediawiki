
《守望時光》是一首由香港男子組合[[C_AllStar|C AllStar]]主唱的[[粵語流行曲|粵語流行曲]]，於2014年推出。<ref>[https://www.facebook.com/0809callstar/posts/%3C%E5%AE%88%E6%9C%9B%E6%99%82%E5%85%89%3E%E4%BF%82%E9%A6%99%E6%B8%AF%E5%BF%83%E7%90%86%E8%A1%9B%E7%94%9F%E6%9C%8360%E9%80%B1%E5%B9%B4%E4%B8%BB%EF%BF%BD/817340311629165/ C AllStar官方Facebook專頁16-5-2014]</ref>

==背景==
《守望時光》同為香港心理衛生會60周年主題曲<ref>[http://www.mhahk.org.hk/chi/sub1_anniversary60_02.htm 香港心理衞生會60周年鑽禧慶祝宣傳活動]</ref>。官方[[MV|MV]]於2014年6月27日在[[YouTube|YouTube]]平台首發，歌曲內容以描述香港[[情緒病|情緒病]]人士面對人生高低跌盪的心路歷程，及反映出情緒病或[[憂鬱症|憂鬱症]]群眾面對無助受困的處境，及能從黑暗走出光明的心底渴望。

==網上回應==
2014年6月22日，《守望時光》MV推出後，《[[明報|明報]]》報導了這首歌的社會意義。<ref>[https://www.facebook.com/mhahk60/photos/a.1446673058904224.1073741840.1399310520307145/1467750443463152/?type=3&theater 香港心理衛生會60周年誌慶 C AllStar演唱主題曲], 明報, 2014年6月22日</ref>。同年5月30日，《[[成報|成報]]》也有相關報導了C AllStar對情緒病的看法及正視。<ref>[https://www.facebook.com/mhahk60/photos/a.1446673058904224.1073741840.1399310520307145/1457025667868963/?type=3&theater C AllStar錄歌安慰人心], 成報，2014年5月30日</ref>

2014年9月，正值香港社會[[雨傘運動|雨傘運動]]發生，《守望時光》曾被一些群眾於網上發表歌唱cover及加上雨傘運動標誌圖片、社交平台[[Hashtag|Hashtag]]等以[[隱喻|隱喻]]方式憑歌寄意。<ref>[https://www.youtube.com/watch?v=SfsMwsQKvLg 雨傘革命之歌- 守望時光CALLSTAR ( cover ) ]</ref>

2015年1月，由Cmusic Chart網站的Cool Music Forum選出的2014非共和榜頒獎禮－非派台歌曲推介獲得第二位。<ref>[http://www.cmusichart.com/discuz/viewthread.php?tid=7690&extra=page%3D1&page=59 Hit Music Chart Award Ceremony 2014 ], 2015年1月</ref>

年度最佳非派台歌曲大獎: 時代廣場 -許志安

2016年6月10日，《圍港噏Day》一篇C AllStar的賞析文章中，提及了其中一些心水歌曲包括《守望時光》。<Ref>[http://talkupday.com/callstar-%E7%B0%A1%E5%96%AE%E5%98%85-a-cappella-%E6%9C%80%E5%8B%95%E4%BA%BA/ C AllStar-簡單的A Cappella最動人], 圍港噏Day, 2016年6月10日</ref>

2017年，內地的[[QQ音樂|QQ音樂]]網一篇分析港樂發展的文章中，也有提及C AllStar的守望時光一曲說明「进入新生代时期的港樂」擁有紀錄一個城市、一個時代的功能，港樂未死。<ref>[https://y.qq.com/msa/204/1_3831.html QQ音樂：每個人心裡都有一首港樂]</ref>
