《'''守護者'''》（{{lang-en|'''''Watchmen'''''}}）是一部[[美國|美國]][[劇情|劇情]][[電視劇|電視劇集]]，改編自[[阿蘭·摩爾|阿蘭·摩爾]]和{{le|戴夫·吉本斯|Dave Gibbons}}創作的[[守護者_(漫畫)|同名漫畫]]，由[[戴蒙·林道夫|戴蒙·林道夫]]開發製作，他同時擔任編劇和執行製片。預訂於2019年在[[HBO|HBO]]首播。

== 概要 ==
故事設定在[[架空|架空]]的歷史時代，[[超級英雄|超級英雄]]被視為不法之徒。劇集在原著基礎上改編，以不同的視野與觀點賦予作品嶄新風貌和全新故事<ref name="Cast1">{{cite web |last1=Andreeva |first1=Nellie |last2=Petski |first2=Denise |title=‘Watchmen’: Regina King, Don Johnson, Louis Gossett Jr., Andrew Howard Among Six Cast In Damon Lindelof’s HBO Pilot |url=https://deadline.com/2018/05/watchmen-regina-king-don-johnson-louis-gossett-jr-andrew-howard-among-six-cast-in-damon-lindelofs-hbo-pilot-1202396744/ |website=[[Deadline_Hollywood|Deadline Hollywood]] |date=2018-06-29 |accessdate=2019-01-03|language=en-US}}</ref>。

== 製作 ==
=== 開發 ===
2015年10月1日，報道稱[[HBO|HBO]]與2009年上映的電影《[[守護者_(電影)|守護者]]》導演[[查克·史奈德|查克·史奈德]]進行了初步溝通，有意將其改編為電視劇<ref>{{cite web |last1=Trumbore |first1=Dave |title=Exclusive: HBO Eyeing 'Watchmen' TV Series from Zack Snyder |url=http://collider.com/watchmen-tv-series-hbo-zack-snyder/ |website=Collider |date=2015-10-01 |accessdate=2019-01-03|language=en-US}}</ref>。隨後HBO證實了這一消息<ref>{{cite web |last1=Andreeva |first1=Nellie |title=HBO Confirms “Preliminary Discussions” For ‘Watchmen’ TV Series |url=https://deadline.com/2015/10/watchmen-tv-series-hbo-discussions-1201564953/ |website=Deadline Hollywood |date=2015-10-01 |accessdate=2019-01-03|language=en-US}}</ref>。

2017年6月20日，有消息稱[[戴蒙·林道夫|戴蒙·林道夫]]將接手該劇集，並已與HBO達成協議，同時確認查克·史奈德不再參與製作<ref>{{cite web |last1=Petski |first1=Denise |title=Damon Lindelof Eyeing An Adaptation Of ‘Watchmen’ For HBO |url=https://deadline.com/2017/06/damon-lindelof-adaptation-watchmen-hbo-1202117245/ |website=Deadline Hollywood |date=2017-06-21|accessdate=2019-01-03|language=en-US}}</ref>。9月19日，報道稱該劇的編劇工作室已經成立<ref>{{cite web |last1=Ramos |first1=Dino-Ray |title=Damon Lindelof Teases Start Of ‘Watchmen’ Series At HBO |url=https://deadline.com/2017/09/damon-lindelof-watchmen-hbo-television-series-alan-moore-dave-gibbons-1202173526/ |website=Deadline Hollywood |date=2017-09-19|accessdate=2019-01-03|language=en-US}}</ref>。一天後，HBO正式確定預訂試播集和備份劇本<ref>{{cite web |last1=Andreeva |first1=Nellie |title=‘Watchmen’ HBO Adaptation From Damon Lindelof Officially A Go With Pilot & Backup Script Order |url=https://deadline.com/2017/09/watchmen-hbo-pilot-order-damon-lindelof-backup-scripts-1202174167/ |website=Deadline Hollywood |date=2017-09-20|accessdate=2019-01-03|language=en-US}}</ref>。

2018年1月30日，{{le|妮可·卡索|Nicole Kassell}}確定將執導試播集，並擔任劇集執行製片，該集劇本由戴蒙·林道夫執筆<ref>{{cite web |last1=Andreeva |first1=Nellie |title=‘Watchmen’: Nicole Kassell To Direct HBO Pilot From Damon Lindelof |url=https://deadline.com/2018/01/watchmen-nicole-kassell-to-direct-hbo-pilot-damon-lindelof-1202274631/ |website=Deadline Hollywood |date=2018-01-30|accessdate=2019-01-03|language=en-US}}</ref>。5月22日，林道夫向漫畫原著粉絲發出長達五頁的公開信，透露電視劇不會直接改編原漫畫或是其續集，而是描繪一個全新的原創故事<ref>{{cite web |last1=Ramos |first1=Dino-Ray |title=Damon Lindelof’s Open Letter To ‘Watchmen’ Fans Says HBO Adaptation Will Be An Original Story |url=https://deadline.com/2018/05/damon-lindelof-watchmen-open-letter-hbo-series-1202396090/ |website=Deadline Hollywood |date=2018-05-22|accessdate=2019-01-03|language=en-US}}</ref>。8月17日，HBO宣佈預訂該劇的整季，第一季定於2019年首播<ref>{{cite web |last1=Andreeva |first1=Nellie |last2=Petski |first2=Denise |title=‘Watchmen’ Is A Go: Damon Lindelof’s Pilot Picked Up To Series By HBO |url=https://deadline.com/2018/08/damon-lindelof-watchmen-pilot-picked-up-series-hbo-1202447586/ |website=Deadline Hollywood |date=2018-08-17|accessdate=2019-01-03|language=en-US}}</ref>。9月20日，報道稱[[特倫特·雷澤諾|特倫特·雷澤諾]]和{{le|阿提喀斯·羅斯|Atticus Ross}}將為劇集作曲<ref>{{cite web |last1=Pedersen |first1=Erik |title=‘Watchmen’: Trent Reznor & Atticus Ross To Compose Music For Damon Lindelof’s HBO Series |url=https://deadline.com/2018/09/watchmen-trent-reznor-atticus-ross-to-compose-music-for-damon-lindelofs-hbo-series-1202468375/ |website=Deadline Hollywood |date=2018-09-20|accessdate=2019-01-03|language=en-US}}</ref>。
