'''前海深港现代服务业合作区'''，位于[[中華人民共和国|中華人民共和国]][[广东省|广东省]][[深圳市|深圳市]]西部的[[前海湾|前海湾]]畔、[[蛇口|蛇口]]西侧、[[珠江|珠江]]口[[伶仃洋|伶仃洋]]东岸，陆上分别以双界河、[[月亮湾大道|月亮湾大道]]、妈湾大道为界<ref>[http://cppcc.people.com.cn/GB/34962/35010/12691634.html 高起点大手笔规划深圳的“曼哈顿”]，人民网</ref>，行政区-{zh-hans:划; zh-hant:劃}-上属于[[南山区_(深圳市)|南山区]]，於2010年8月26号由[[中华人民共和国国务院|国务院]]批覆。規劃利用毗鄰[[香港|香港]]的區位優勢，特別制定《前海深港现代服务业合作区总体发展规划》，将前海深港现代服务业合作区定位为深港合作先导区、体制机制创新区、现代服务业聚集区和结构调整引领区。<ref>[http://finance.sina.com.cn/g/20100728/12288378833.shtml 特区中的特区——深圳前海深港合作区即将启动]，新浪网 2010年07月28日</ref>屬[[深圳經濟特区|深圳經濟特区]]的一部分。

== 地理 ==
前海深港现代服务业合作区占地面积18.04平方公里，其中3.71平方公里为前海湾保税港区，位于整个珠三角湾区的核心位置，均是填海造地而成的土地。

== 片區規劃 ==
以地域劃分為隶属于[[宝安区|宝安区]]的大鏟灣片區、寶安中心片區和隶属于[[南山区_(深圳市)|南山区]]的桂灣片區、荔灣片區及媽灣片區<ref>[http://www.szqh.com.cn/ljqh/ghjs/ghgl/ghjs_zhgh/ 前海綜合規劃] {{webarchive|url=https://web.archive.org/web/20141129080124/http://www.szqh.com.cn/ljqh/ghjs/ghgl/ghjs_zhgh/ |date=2014-11-29 }}</ref>。由於合作區鄰近[[深圳寶安國際機場|深圳寶安國際機場]]，為免影響飛機升降，部分地塊設有高度限制（目前各在建項目中最高的是鴻榮源金融中心，高度為320米）<ref>[http://news.163.com/air/15/0513/10/APG60GID00014P42.html 深圳前海受机场净空限制 楼高不许超200米]</ref>。

== 发展历程 ==
根據中共中央、國務院的決策部署訂下的前海發展目標，利用香港[[服務業|服務業]]發達的優勢，在前海合作發展金融等现代服务业，到2020年，建成基礎設施完備、聚集具有世界影響力的現代服務業企業，成為亞太地區重要的生產性服務業中心，世界服務貿易重要基地<ref>[http://www.chinalegal.com.hk/index.php?option=com_content&view=article&id=445%3A1010&catid=111&Itemid=85 前海深港現代服務業合作區總體發展規劃] {{webarchive|url=https://web.archive.org/web/20140714220426/http://www.chinalegal.com.hk/index.php?option=com_content&view=article&id=445%3A1010&catid=111&Itemid=85 |date=2014-07-14 }}中國法律服務（香港）有限公司 2010年10月</ref>
<ref>{{cite web|accessdate=2014-05-24|url=http://district.ce.cn/zg/201307/26/t20130726_24608896.shtml|title=前海深港合作区基础设施2017年建成 滞后《规划》2年|publisher=[[中国经济网|中国经济网]]}}</ref>。

2011年6月27日，深圳市第五屆人民代表大會常務委員會在第九次會議中，通過《深圳经济特区前海深港现代服务业合作区条例》，條例在前海合作區推行，以推進前海合作區的發展<ref name="law">[http://www.sz.gov.cn/cn/xxgk/szgg/tzgg/201107/t20110706_1672662.htm 深圳政府在线-《深圳经济特区前海深港现代服务业合作区条例》公布施行] {{webarchive|url=https://web.archive.org/web/20160304190151/http://www.sz.gov.cn/cn/xxgk/szgg/tzgg/201107/t20110706_1672662.htm |date=2016-03-04 }}</ref>。

截至2016年底，在前海注册的金融类企业已达到5.09万家，占前海企业总数的46.6%。其中，银行类金融机构40家，证券业金融机构88家，保险业金融机构22家，另有融资租赁公司2000余家、商业保理公司4000余家、小额贷款公司36家、信用担保公司280余家<ref name="2016年深圳金融业发展基本情况">{{cite web|author1=深圳市人民政府金融发展服务办公室|title=2016年深圳金融业发展基本情况|url=http://www.jr.sz.gov.cn/zwgk/tjsj/zxtjxx/201702/t20170209_5961384.htm|website=深圳市人民政府金融发展服务办公室网站|accessdate=2017-02-11|deadurl=yes|archiveurl=https://web.archive.org/web/20170211080012/http://www.jr.sz.gov.cn/zwgk/tjsj/zxtjxx/201702/t20170209_5961384.htm|archivedate=2017-02-11}}</ref>。

== 与香港关系 ==
前海配合中央提供政策，規劃訂下的藍圖發展，照照規劃大綱，前海如果成功發展，其體制和金融業為主模將會與香港極為相近。香港經貿商會會長李秀恒博士表示，前海與香港正面競爭、香港地位遭受削弱已經是無可避免的趨勢<ref>{{Cite web |url=http://www.hketa.com.hk/acticleDetail?id=518 |title=香港經貿商會 |access-date=2016-07-17 |archive-url=https://web.archive.org/web/20160817064014/http://www.hketa.com.hk/acticleDetail?id=518 |archive-date=2016-08-17 |dead-url=yes }}</ref>。香港雖然國際化，而且國際精英雲集，不過中國共產黨始終無法對其全權操控，因此必需部署，另外建設一座城市「對沖」香港。前海也許不能夠完全取代香港，但在必要時作為香港的後備金融城市卻完全可能。

[[香港|香港]][[建制派_(香港)|建制派]][[立法會議員|立法會議員]][[葉劉淑儀|葉劉淑儀]]公開質疑、認為香港正被「綁上戰車」，無可避免要參與，屆時會有大批服務業工序遷到前海，引發第二次產業北移，惟[[香港政府|香港政府]]卻後知後覺。<ref>[http://forum.hkej.com/node/60537 信報社評] 2010年12月22日</ref>《[[信報|信報]]》創辦人[[林行止|林行止]]則認為；前海的最終目的在取代香港，成為由北京操控、全面為中國服務的地域性金融中心。他以[[新加坡|新加坡]]為例，稱新加坡被專權體制掌控，世人皆知，但是該國在金融業上有與香港爭一日雄長甚且超越香港的架勢。<ref>[http://www.hkej.com/template/dailynews/jsp/detail.jsp?dnews_id=3603&cat_id=6&title_id=569627&txtSearch=%E6%9E%97%E8%A1%8C%E6%AD%A2%E5%B0%88%E6%AC%84 為前海做好準備 國產人才治香港]《信報》網上版 林行止專欄 2013年1月9日</ref>
