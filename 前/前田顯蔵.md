'''前田顯蔵'''（{{jpn|j=前田 顕蔵|kn=まえだ けんぞう|rm=Maeda Kenzō}}，{{bd|1982年|6月27日}}），生於[[日本|日本]][[大阪府|大阪府]]，日本篮球教練，目前是[[秋田北部喜悅|秋田北部喜悅]]助理教練<ref>https://northern-happinets.com/team/staffs/teamstaff03/</ref>。

==经历==
小学4年级起开始接触篮球。{{link-ja|大商学園高等学校|大商学園高等学校}}毕业后赴美国[[内布拉斯加大学卡尼尔分校|内布拉斯加大学卡尼尔分校]]留学。

2008年担任[[bJ联盟|bJ联盟]]{{link-ja|高松五箭|香川ファイブアローズ}}队（现香川五箭）担任助理教练兼翻译<ref>{{Cite web |url=http://www.fivearrows.jp/news/team/entry-527.html |title=前田顕蔵ヘッドコーチ契約基本合意のお知らせ |access-date=2018-10-27 |archive-url=https://web.archive.org/web/20160305021000/http://www.fivearrows.jp/news/team/entry-527.html# |archive-date=2016-03-05 |dead-url=yes }}</ref>，同时担当该队的梯队教练。

2011年7月，接替{{link-ja|金澤篤志|金澤篤志}}出任高松五箭队主教练。当赛季因球队主力伤病，仅取得2场胜利，球队排名垫底。2012-13赛季获得20胜，脱离了垫底的位置。2013-14赛季前田还担任了bj联盟全明星赛西部队的教练。2014-15赛季带领高松五箭进入季后赛，赛季结束后从球队离任。2015年8月，前田至[[秋田北部喜悦|秋田北部喜悦]]队担任助理教练<ref>{{Cite web |url=http://www.happinets.net/news-cate02-entry/2015/08/06/22489/ |title=チームスタッフ契約基本合意のお知らせ }}{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2018年9月，作为[[日本国家男子篮球队|日本国家男子篮球队]]的教练兼翻译，随队备战[[2019年男篮世界杯|2019年男篮世界杯]]亚洲区预选赛。
