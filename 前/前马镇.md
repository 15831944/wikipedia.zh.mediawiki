'''前马镇'''，是{{PRC admin/fullname|32/04/81/113/000|3}}下辖的一个已撤销[[乡镇|乡镇]]级行政单位，位于溧阳市中部。2007年并入[[竹箦镇|竹箦镇]]。

==历史==
前马镇辖境宋代称'''來蘇鄉'''，前马是其下辖一村，名为前馬裡。明代改称來蘇區，區以下设15裡。清代在區内设王贅步（今竹箦镇王渚村）。
中华民国初撤区建市、乡，设置來蘇鄉（乡公所驻[[别桥镇|甓橋]]，下设111图）。1929年改市乡制为区－乡二级制，属竹箦桥区（1935年改称第三区）。
1949年建立來蘇鄉，属县辖竹箦区。当年又拆分为來蘇、洙湯、東湖、余橋、瀨陽5个鄉。次年來蘇鄉改称'''前馬鄉'''。1956年5个乡合为前马乡。1957年后直属於[[溧阳县|溧阳县]]，1958年后改称前马[[人民公社|公社]]，1961年分设余桥公社，1983年后公社恢复为乡。

2000年初前马乡、余桥乡合并为前马镇，2007年撤销前马镇并入[[竹箦镇|竹箦镇]]<ref>{{cite web |title=省政府关于同意调整溧阳市部分镇行政区划的批复 |url=http://www.jiangsu.gov.cn/art/2007/3/17/art_46143_2543681.html |website=江苏省政府网 |publisher=江苏省人民政府 |accessdate=2018-08-12 |date=2007-03-17}}</ref>。

== 行政区划 ==
撤销前前马镇下辖以下地区：

前马镇社区、前马村、洙汤村、潘家村、西芮村、水西村、神塘圩村、余桥村、直埂村、濑阳村、道人渡村和清水塘村。
