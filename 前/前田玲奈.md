== 人物 ==
2011年在《[[緋彈的亞莉亞|緋彈的亞莉亞]]》飾演女學生B<ref name="interview_maeda">{{Cite web |url=http://ddnavi.com/anime-bu/210576/ |title=「前田玲奈」新人声優のインタビュー＆ミニグラビア【新人声優図鑑】　|accessdate=2015-05-15}}</ref>為聲優出道作。2014年在《[[即使如此世界依然美麗|即使如此世界依然美麗]]》飾演的妮可·魯梅露希耶則為初次擔任主角的作品<ref>{{Cite web |url=http://mantan-web.jp/2014/04/04/20140404dog00m200012000c.html |title=注目アニメ紹介：「それでも世界は美しい」　おてんば姫と少年王の絆描く　|accessdate=2015-05-15}}</ref>。
