'''前龜山陸光三村活動中心'''，是[[桃園市|桃園市]]所有[[眷村|眷村]]中，第一座登錄為[[歷史建築|歷史建築]]的眷村，也是全台灣第一座以原建築物直接改建為眷村故事館。
==沿革==
前龜山陸光三村活動中心，「'''陸光'''」代表的意思是「'''陸軍的榮耀'''」<ref>{{cite web |title=龜山鄉千禧新城大榕樹旁眷村故事館 |url=https://lov.vac.gov.tw/Village/Content.aspx?i=184&c=2 |publisher=榮民文化網 |accessdate=2018-12-10}}</ref>。於1966年竣工<ref>{{cite web |author1=地理資訊科學研究專題中心 |title=前龜山陸光三村活動中心 |url=http://crgis.rchss.sinica.edu.tw/assets/tangible/723 |accessdate=2018-12-11}}</ref>。陸光三村文康中心{{notetag|前龜山陸光三村活動中心曾作為龜山鄉自治會辦公室，之後逐漸成為里民，交流集會的場所。}}則於2-3年內陸續完工。2004年3月3日正式公告登錄成為歷史建築；建築樣式屬於1960年代現代式樣建築<ref name=”文化資產網”>{{cite web |author1=國家文化資產網 |title=前龜山陸光三村活動中心 |url=https://nchdb.boch.gov.tw/assets/overview/historicalBuilding/20040303000004 |accessdate=2018-12-11}}</ref>。陸光三村入口有[[宋美齡|蔣宋美齡]]女士的題字「'''陸光三邨'''」<ref>{{cite web |author1=蔡佳峻 |title=前龜山陸光三村活動中 |url=http://nrch.culture.tw/twpedia.aspx?id=13795 |publisher=台灣大百科全書 |accessdate=2018-12-11}}</ref>。陸光三村拆除改建成為[[公營住宅|國宅]]後，在旁邊的陸光三村活動中心則未被拆除<ref name="桃園市政府文化局">{{cite web |author1=桃園市政府文化局 |title=前龜山陸光三村活動中心 |url=http://culture.tycg.gov.tw/home.jsp?id=145&parentpath=0%2C130%2C133&mcustomize=onemessages_view.jsp&dataserno=201607110112&aplistdn=ou=data,ou=chculture3,ou=chculture,ou=ap_root,o=tycg,c=tw&toolsflag=Y |accessdate=2018-12-11}}</ref>。2014年桃園市爭取經費，開始著手館內整修<ref name="桃園市政府文化局"/>。2015年10月正式對外開放，供眾參觀<ref name=”文化資產網”/>。
==眷村故事館==
眷村故事館，是一棟灰色建築物，二層樓高<ref name="民報">{{cite news |author1=顏毓瑩 |title=【民報文化】桃園眷村的故事 |url=http://www.peoplenews.tw/news/0f4912f4-bd89-40e0-ab2f-dfee0da3f21d |accessdate=2018-12-11 |work=民報 |date=2015-07-03}}</ref>，眷村故事館是由「陸光三村活動中心」改建而成<ref>{{cite news |author1=徐嘉祺 |title=龜山陸光三村眷村活動中心變身故事館　述說光陰的故事  原文網址: 龜山陸光三村眷村活動中心變身故事館　述說光陰的故事 |url=https://travel.ettoday.net/article/263749.htm |accessdate=2018-12-11 |work=ETtoday |issue=2013-09-6}}</ref>，經桃籽園文化協會的策劃，及當地里民的參與，將活動中心營造出有「'''家庭味'''」的眷村故事館<ref name="自由時報">{{cite news |author1=李容萍 |title=〈北部〉眷村故事館 龜山陸光三村珍寶 |url=http://news.ltn.com.tw/news/local/paper/800177 |accessdate=2018-12-11 |work=自由時報 |date=2014-07-14}}</ref>。

故事館一樓陳設當時住在眷村軍人物品，包含有皮箱、廣播機、軍靴等，物品是由桃籽園文化協會收集而來的，有些是眷村里民提供。有一項最不一樣的物品是「'''眷戶名牌'''」<ref name="民報"/>。

二樓陳設則是還原當時眷村客廳的樣子，紅門綠窗、老電視、裁縫車等；居民自動提供自己家當年的物品。在當時跳脫[[博物館|博物館]]式的展覽方式，改用生態博物館方式展示，讓故事館與國宅結合<ref name="民報"/>。

故事館中收集來的眷村田調文史資料，經由紀錄片、聲音箱方式表達，並且在各主題區設計互動[[遊戲|遊戲]]，讓大家更了解眷村<ref name="自由時報"/>。


眷村故事館每年會在11月及12月間不定期舉辦活動<ref>{{cite web |title=龜山鄉千禧新城大榕樹旁眷村故事館 |url=https://lov.vac.gov.tw/Village/Content.aspx?i=184&c=2 |publisher=榮民文化網 |accessdate=2018-12-11}}</ref>。

===眷村故事館歷程<ref>{{cite web |title=龜山眷村故事館大事年表 |url=http://tynewvillage.tyccc.gov.tw/ch/m1_life/tyccc_w1_m1_s2_s4_s5_C.php#gsc.tab=0 |publisher=桃園市政府文化局 |accessdate=2018-12-11}}</ref>===
*2002年：自治會與桃籽園文化協會開始收集眷村物品和策劃展示
*2004年：登錄為歷史建築，並命名為「眷村故事館」
*2014年：取得經費，館內整修
*2015年：眷村故事館重新開放參觀
