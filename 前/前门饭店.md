'''前门饭店'''<ref name=bjta>{{Cite web|url =http://www.bjta.gov.cn/tsfwzt/qyml/373570.htm|title =星级饭店|author =|date =2015-04-29|publisher =北京市旅游发展委员会|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20160317085035/http://www.bjta.gov.cn/tsfwzt/qyml/373570.htm|archivedate =2016-03-17}}</ref>，又称“'''前门建国饭店'''”，位于[[北京市|北京市]][[西城区|西城区]][[永安路_(北京)|永安路]]175号。

==简介==
1956年，前门饭店开业<ref name=yuebing>{{Cite web |url = http://www.ce.cn/cysc/sp/subject/2009/yuebing/10/200909/11/t20090911_19672399.shtml |title =建国饭店月饼  |author =  |date = 2009-09-11 |publisher = 中国经济网 |language =  |accessdate =  |quote =  }}</ref>。1958年10月，[[毛泽东|毛泽东]]亲自试乘的第一辆中国自主研制的东风牌轿车开进前门饭店<ref name=liyuan/>。[[北京饭店|北京饭店]]、[[民族饭店|民族饭店]]、前门饭店、[[华侨大厦|华侨大厦]]、[[京西宾馆|京西宾馆]]曾经并称北京“五大饭店”<ref>{{Cite web |url = http://zj.zjol.com.cn/news/259672.html |title =  《京西宾馆那些年那些事》：它见证了诸多历史|author =  |date = 2016-01-26 |publisher = 浙江新闻 |language =  |accessdate =  |quote =  }}</ref>，1964年时这五家饭店是北京建筑和设备最好的饭店<ref>{{Cite web |url = http://cul.qq.com/a/20141115/030632.htm |title = 京西宾馆：“文革”和改革相继上演 |author =  |date =  2014-11-15|publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。前门饭店是北京在[[计划经济|计划经济]]时代的八大饭店之一。2007年，前门饭店建筑被北京市列为《[[北京优秀近现代建筑保护名录|北京优秀近现代建筑保护名录]]》<ref name=yuebing/>。[[文革|文革]]爆发前后，1966年5月[[中共中央华北局|中共中央华北局]]在前门饭店召开了著名的[[前门饭店会议|前门饭店会议]]。

1979年，[[清华大学|清华大学]]教授[[金国芬|金国芬]]为前门饭店开发出具备查询功能的酒店管理软件，在[[中华人民共和国|中华人民共和国]]酒店管理中开创先河<ref>{{Cite web |url = http://www.jiemian.com/article/346992.html |title = 六大门派22家国内外酒店PMS企业大盘点 |author =  |date = 2015-08-07 |publisher =  界面|language =  |accessdate =  |quote =  }}</ref>。

1978年[[改革开放|改革开放]]后，前门饭店由[[事业单位|事业单位]]改制成企业<ref>单志明、李景林，前门建国饭店招牌菜，中国食品2004年04期</ref>。前门饭店是北京最早的涉外酒店之一，入住来宾以海外旅游团队为主<ref name=guocui>{{Cite web |url =  http://news.ifeng.com/gundong/detail_2011_06/27/7262061_0.shtml|title = “国粹”雕琢酒店文化 |author =  |date = 2011-06-27 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。1990年，前门饭店建成梨园剧场并开业。2000年代起，前门饭店定位为中国第一家“京剧文化主题饭店”，将部分单间餐厅用京剧剧名及情节装修为主题餐厅<ref name=guocui/><ref name=liyuan/><ref name=sixing/>。例如兰芳厅得名自[[梅兰芳|梅兰芳]]。作为主题酒店，前门饭店内遍布各种京剧剧照和历史照片，并展示戏服、道具，还为宾客开设有“京剧教室”教京剧表演<ref name=guocui/>。2008年，前门饭店获得[[国家旅游局|国家旅游局]]颁发的四星级酒店牌匾<ref name=sixing>{{Cite web |url = http://finance.sina.com.cn/roll/20080318/02332076054.shtml |title = 前门建国饭店升四星玩“京剧” |author =  |date = 2008-03-18 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

1998年，北京首都旅游集团有限责任公司（简称“首旅集团”）成立<ref>{{Cite web |url = http://www.btg.com.cn/about/about.asp?tid=1 |title = 首旅概况 |author =  |date =  |publisher = 首旅集团 |language =  |accessdate =2017-05-02  |quote =  }}</ref>。1999年，成立北京首都旅游国际酒店集团有限公司（简称“首旅酒店集团”），为首旅集团旗下的二级企业。该公司将“建国”定为集团的统一品牌。所以前门饭店自此又称“前门建国饭店”<ref>{{Cite web |url = http://www.gmw.cn/01gmrb/2001-01/19/GB/01%5E18670%5E0%5EGMA3-106.htm |title = 本报举行2001年迎春联欢会 |author =  |date =  2001-01-19|publisher = 光明网 |language =  |accessdate =  |quote =  }}</ref><ref>绿色“东方”：倡导健康新时尚，厂长经理日报，2000-12-27</ref><ref>陈敏，北京首旅酒店集团的“建国”品牌营销之路，四川大学硕士学位论文，2003年</ref>。

1999年，北京首都旅游股份有限公司（简称“首旅股份”）经[[北京市人民政府|北京市人民政府]]京政办函〔1999〕14号批复批准，由北京首都旅游集团有限责任公司作为主发起人成立。2000年发行[[A股|A股]]，在[[上海证券交易所|上海证券交易所]]上市<ref>{{Cite web|url =http://www.btghg.com/about_history.html|title =发展历程|author =|date =|publisher =首旅酒店集团|language =|accessdate =2017-05-02|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170204111743/http://www.btghg.com/about_history.html|archivedate =2017-02-04}}</ref>。2007年9月12日，北京首都旅游股份有限公司第三届董事会公告称，2007年9月10日北京首都旅游股份有限公司第三届董事会第十二次会议审议通过了《关于公司2007年度公开增发A股方案的提案》，募集资金用途为：（1）收购北京首都旅游集团有限责任公司、香港紫大国际投资有限公司分别持有的[[北京诺富特和平宾馆|北京和平宾馆]]有限公司股权（共计51%股权）；（2）对北京市[[新侨饭店|新侨饭店]]有限公司增资，取得该公司增资完成后的51%股权；（3）收购北京市首都旅游国际酒店集团有限公司前门饭店的整体经营性资产（含负债）并且交纳土地出让金<ref>{{Cite web |url = http://finance.sina.com.cn/stock/s/20070911/22271659749.shtml |title =  首旅股份(600258)第三届董事会第十二次会议决议公告|author =  |date = 2007-09-11 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。

2009年1月16日，北京首都旅游股份有限公司股东大会通过收购前门饭店的决议。2009年，收购前门饭店<ref>{{Cite web |url =  http://app.finance.ifeng.com/data/stock/tab_ggzw.php?symbol=600258&id=13558658|title = 首旅股份：2009年半年度报告|author =  |date = 2009-08-20 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。2013年，北京首都旅游股份有限公司更名为“北京首旅酒店（集团）股份有限公司”（简称“首旅酒店”）<ref>{{Cite web |url = http://finance.ifeng.com/a/20130808/10381731_0.shtml |title = 首旅股份更名“首旅酒店” 淡旅游定位更明确 |author =  |date =  2013-08-08|publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。其旗下的前门饭店全称为“北京首旅酒店（集团）股份有限公司前门饭店”<ref>{{Cite web |url = http://app.finance.ifeng.com/data/stock/tab_ggzw.php?symbol=600258&id=16002755 |title =  首旅酒店：2016年半年度报告|author =  |date = 2016-08-31 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。2017年3月28日，首旅酒店、[[如家酒店|如家酒店]]集团宣布，合并后的酒店集团更名为“北京首旅如家酒店（集团）股份有限公司”<ref>{{Cite web |url = http://finance.sina.com.cn/roll/2017-03-28/doc-ifycsukm3998850.shtml |title = 首旅酒店正式更名首旅如家 三大事业部架构浮出水面 |author =  |date =2017-03-28  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

自2013年起，受“[[中央八项规定|中央八项规定]]”以及2014年初实施的《中央和国家机关会议费管理办法》影响，包括前门饭店在内的高端酒店利润连续下滑<ref>{{Cite web |url = http://finance.ifeng.com/a/20140416/12131718_0.shtml |title = 政策持续影响高端酒店利润 |author =  |date = 2014-04-16 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

==梨园剧场==
1990年，前门饭店与[[北京京剧院|北京京剧院]]合作，在前门饭店内1层创办梨园剧场。梨园剧场所在处，之前是前门饭店附设的礼堂，专用来接待中央各部委会议，后来由于常年不用而改成仓库。改建成梨园剧场后，专演[[京剧|京剧]]，主要接待境外游客晚间观剧<ref name=liyuan>{{Cite web |url =http://culture.people.com.cn/GB/22219/7338079.html  |title = 前门建国饭店：住在店里看京戏 |author =  |date =2008-06-04  |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。2010年代初，梨园剧场每年接待400万游客<ref name=guocui/>。
