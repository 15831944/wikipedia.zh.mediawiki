'''前田實'''（{{jpn|j=前田 実|kn=まえだ みのる}}、{{bd|1954年|3月24日||}}），白羊座，有時會用前田みのる的名義表示。

前田實是日本資深的原畫師、動畫師和人物設計師。也是動畫講師和動畫作家，在国际上评价很高的动画家<ref>{{Cite web|url=https://anabuki.exblog.jp/5049225/|title=講師の前田稔先生がパネリストとして参加します！|accessdate=2007年2月12日|author=|date=|publisher=}}</ref>。出生於[[日本|日本]]的[[北海道|北海道]]，現居日本[[東京都|東京都]]的[[府中市_(東京都)|府中市]]<ref>前田 実的個人FaceBook</ref>。<br>他的興趣是小孩子、騎單車和運動、攝影。他的FB臉書好友多數是動畫工作者、單車的車友<ref>前田 実的個人FaceBook</ref>。<br>

前田製作许多代表日本的动画片作品的人物设计·作画监督，所以許多日本國民都曾經見過他的作品。與其他動畫師比起來，線條通常略帶圓滑、俏皮和簡潔。在Junio的時代，前田因為細膩和活潑的風格而聞名，變成Junio的核心Animator成員之一。<br>

他的代表作有<ref>《七龍珠動畫畫集 金色之戰士》前田實的訪問，開頭第一段的介紹</ref>：『[[怪博士與機器娃娃|怪博士與機器娃娃]]』『[[七龍珠_(動畫)|七龍珠]]』『[[七龍珠Z|七龍珠Z]]』『[[棒球英豪|鄰家女孩]]』『[[麵包超人|麵包超人]]』『{{link-ja|Bug甜心|Bugってハニー}}』『看見風的少年』等。

主要擔任：總經理、設計師、原畫師、動畫導演、動畫人物設計師、動畫機械設計師、作畫監督、總作畫監督、動畫師、首席動畫師。

現在是[[Synergy_SP|Synergy SP]]的董事會經理，也是該公司的創辦人之一。此外也在[[TMS娛樂|TMS娛樂]][[衍生作品|改編]]繪本作家[[柳瀨嵩|柳瀨嵩]]「福祿貝爾館」[[幼兒|幼兒]][[故事|故事]][[繪本|繪本]]刊登的『麵包超人』動畫系列擔任人物設計師、總作畫監督(電視版)、作畫監督(電視特別篇、劇場版)、動畫主管的核心動畫製作人。

背景監督、美術監督的前田實為同名同姓的異人。

==簡歴==
1970年16歲時由於動畫製作公司Studio Junio（後來的：ジュニオブレイントラスト株式會社，以下稱：junio）的第一次的動畫片繪製募集而進入公司。從動畫製作人前田庸生和[[大塚康生|大塚康生]]那裡學習技術，Junio承包的東映動畫作品、東京電影（A生產）的動畫開始他的工作生涯。<br>

經過TV動畫『山林小獵人（はじめ人間ギャートルズ）』等原畫成長為Junio的核心動畫製作人，特別是1980年代，以角色設計師出道的TV動畫『怪博士與機器娃娃』（1981年）為開端，後續『TOUCH鄰家女孩』（1985年）、『七龍珠』（1986年）等暢銷作品擔任角色設計和總作畫監督（チーフ作畫監督）

1980年代，{{link-ja|Group Tac|グループ・タック}}的『[[棒球英豪|鄰家女孩]]』和東映動畫的『怪博士與機器娃娃』、『七龍珠』、『七龍珠Z』主要是以{{link-ja|岡崎稔|岡崎稔}}、前田實為中心。但是二人一個月的訂單數合計-{高達}-20部動畫作品，是因他們的常務董事{{link-ja|香西隆男|香西隆男}}積極的錄用非原包員工來提高發言權。<br>

1984年因為動畫的知識加入了Nintendo Company Limited，[[宮本茂|宮本茂]]對前田實的才華印象深刻建立了友誼關係。在Nintendo Entertainment System平台上面設計任天堂電玩ExciteBike和Vs. Arcade，直到1999年離開任天堂遊戲開發<ref>[http://iwataasks.nintendo.com/interviews/#/wii/punchout/0/2 Drawing Pixels on Giant Graph Paper]</ref>。

1988年開始為『麵包超人』製作後，也持續很長時間地為東映動畫『七龍珠Z』製作。負責大部分的動畫和原畫審查批准、系列作品的人物設定，在一些關鍵劇集擔任監督、原畫的工作一直到1993年離開Junio。隨即成為[[TMS娛樂|東京電影]]的幼兒動畫『麵包超人』系列的核心製作人至今，加上和{{link-ja|杉井儀三郎|杉井ギサブロー}}共同監督『青春野球部Nine』以來二人就長期維持良好的合作關係，在期間一直擔任人物設計和動畫導演的團體工作。<br>

1998年5月1日，前田和岡崎稔及我妻宏一起創立シナジージャパン，就任了该公司的董事。同年的9月24日以「Synergy Japan」的名義公司化成為[[有限會社|有限會社]]。2005年小學館Production和其他資金積極購入公司股份，並把名稱改為「Synergy SP」。

== 關於《廣島落日The Sun Was Lost》<ref>{{Cite web|url=http://www.hiroshimapeacemedia.jp/hiroshima-koku/museum/course_d.html|title=アニメ作品「太陽をなくした日」＝前田稔さん（36）（広島市佐伯区）|accessdate=|author=|date=|publisher=}}</ref><ref>{{Cite web|url=http://hiroshima.moe-nifty.com/blog/2009/06/post-61aa.html|title=アニメ原爆映画「太陽をなくした日」|accessdate=|author=|date=|publisher=}}</ref> ==
我從1970年開始從事動畫的工作至今36年，原子弹爆炸的第二世代<ref group="注">1939年~1945年[[第二次世界大戰|第二次世界大戰]]。</ref>。毕业于大阪专科学校的动画专业，在广岛作为插画家致力于动画电影的制作。

这部是動畫電影被选为第九届（2002年）国际动画节招待作品的19分钟短篇作品。我从动画制作作为目标的时候开始「希望在能夠製作動畫的時候，通過超越文字來表達動畫的世界。」過程花費十年的時間<ref group="注">1992年開始。</ref>。我的童年...... 經常和家人親戚聚集在一起的時候提出這個想法，我的祖父母和父親是經歷十多個原子彈後的倖存者，他們談論“當時正在做什麼，發生了什麼”的原子彈爆炸經歷。在逃跑的路上，我說：“如果我忘了我的學習工具，學校就不再破壞了”和“我很高興我沒有真的經歷過......”。“那个时候的桥是这样的”“我也看到了，但是已经燃烧了，没办法过桥了”等等意外的事实，店头若无其事地放着的混凝土箱子就是当时的防火用水，真的是很有趣。另一方面，許多不了解和平教育的原子彈倖存者的學校孩子，常常覺得他們害怕這種悲慘的結果，認為他們會成為“他”。我想透過動畫將它傳達給不知道的人們。我意識到我可以從孩子的日常生活角度來傳達我的想法。

对于我们这个年龄階段的人来说，几乎是不需要的东西，但是对于年轻人来说，这是不可或缺的，也期待着這部作品多國语言化。“广岛动画节”的倡导者與被称为原爆动画圣经作品的木下莲三先生的《ピカドン》，也是完全没有旁白和字幕的作品，在10分钟充分传达廣島發生的過程。我到中歐和東歐上小學發表放映時，裡面人類融化的場景讓校長驚呼馬上停止播映。“人類融化......”是无法想象的恐怖，在教育上不受欢迎的抗拒……人融化的事實是不可取的。事實上，5000度的高溫足以燒毀人的皮表，校長在學生面前再次發出讚嘆的聲音，我從頭開始重新放映“我很抱歉廣島......”再次向孩子們傳遞事實。這是一個動畫傳授的強大力量的時機，「广岛失去了什么？而现在，有什么是你最不想失去的？」

《太陽をなくした日》虽不像《ピカドン》帶來强烈的冲击，但对于不了解战争和近代史和没有接受过和平教育的孩子们来说，只要知道消失的廣島，然後知道原子彈倖存者的經歷這點就很好了。我在片裡繪製熱輻射線和爆炸，希望在評論中寫下一些線索，即原子彈是一種“輻射曝露”，與其他炸彈有決定性的區別。為了讓這部作品更加生動，我敢於承擔希望，再次鼓掌和大聲呐喊，“如果有更多的人能以看过这部作品为契机，到原子弹资料馆等地方，我會很高興。”

== 關於《七龍珠》 ==
2003年DRAGON BALL DVD BOX DRAGON BOOK，前田實分享了製作感想：“從開始做動畫的時候開始，我就一直抱有疑問，為什麼明明有原作，但是仍然畫得亂糟糟的呢？在某些情況下，作品常常像是很草率隨性的帶過就交到我手上。之所以會這樣，是因為動畫工作者的傲慢？ 我一直想動畫工作者難道不會漫畫家的畫。確實，即使是漫畫家也有不擅長繪製的人物。 但是，即使畫得不好，我覺得不能原封不動地表現出來的話，這樣很奇怪。”“當時有很多作品的漫畫版及動畫版的畫風有很大分別，我很喜歡鳥山老師的畫作，因此我很討厭任何對‘原作的畫進行改動’之類的想法。正因為這樣，工作時會有一種感覺‘我要把漫畫的分鏡讓他保持原狀動起來，不會刪除任何一條線！’我當時真的太年輕了(笑)。那些畫作只用了少許線條，因此要平衡人物的面部五官及四肢相當困難，它們的位置關係只要稍有不同，就會完全變成另一個人。” “在成為人物設定和總作畫監督的立場之後，我想盡辦法讓它忠於原作。但是這是一個團體工作，所以要讓所有人的風格和畫工一致是一項困難的任務，工作人員對這套動畫的製作容易陷入緊張而大驚小怪。有時製作室來了新的年輕原畫師、動畫師，我會先把原作遞給他們，說"先看看這個"……，只是通常並不能馬上就跟上團隊其他成員技術的步伐，因為這需要一些時間練習。我確保他們仍然像原作來提高團隊的工作水準，有時候我會把畫作稍微-{修}-的和鳥山老師比較接近的畫風．有時會稍微講一下，但是講完工作人員會覺得我太苛刻。”“在『七龍珠』後期與『七龍珠Z』之後，整體大家的技術都已經改善很多。但是看了那個"難以描繪的不同畫法"， 一般右撇子的人從右上往左底線，鳥山老師卻從左下往右上畫了線，我覺得這樣的畫法很特別。”<ref>DRAGON BALL DVD BOX DRAGON BOX、2004年7月7日。</ref>雖然在1993年不明原因離開了七龍珠，但仍然繪製『七龍珠Z』的魔人普烏篇３張海報的下一任人物設計師中鶴勝祥的工作。”<ref>DRAGON BALL DVD BOX DRAGON BOX、2004年7月7日。</ref>

雖然前田實是受僱於Junio，但他在東映動畫公司所指導的動畫小組很有才華，許多動畫師在他缺席期間能夠扮演動畫導演的職務，其中最有名的動畫師是[[中鶴勝祥|中鶴勝祥]]、[[佐藤正樹|佐藤正樹]]、[[井手武生|井手武生]]、[[宮原直樹|宮原直樹]]與[[江口寿志|江口寿志]]。『七龍珠Z』第164話是他的最後一次為七龍珠製作後再也沒有擔任主管的職責，『七龍珠Z』第199話1993年離開東映動畫和Junio。東映動畫立即將動畫師[[増永計介|増永計介]]聘用進來，以減輕其他動畫師的工作量的龍珠支援動畫師。<br>
前田實：「七龍珠Z的孫悟空已經成為一個孩子的父親，所以我總是想把孫悟空塑造成很有男子氣概、有擔當的一個角色，我覺得它的故事很有可能和孫悟飯的成長有關。現在的畫技不論是肌肉的曲線或是動作比起七龍珠時期更加地出色了，我很高興，因為在這裡我遇到優秀的工作夥伴，這是一個很好的機緣，無論怎樣都會抱著不屈不撓的精神。」<br>

前田實離開後，他的工作由中鶴勝祥與[[山室直儀|山室直儀]]來共同分擔『[[七龍珠Z|七龍珠Z]]』的後半部分。但即使在『七龍珠Z』結束之後，前田實所指導的動畫小組的許多成員仍然持續為『七龍珠』系列製作，一直到1997年11月結束。他的成員佐藤正樹也在那個時候離開七龍珠，為『[[灌籃高手|灌籃高手]]』擔任人物設計師的工作。


在2010年的『七龍珠動畫畫集：金色之戰士』裡面，他提到「回想起來，我初次去繪鳥山老師的畫，是我擔任『七龍珠』的前作『怪博士與機器娃娃』的pilot版原畫師時的事情。Pilot版是指在正式播出前的測試版影片，大約是在1979年左右。當時我很喜歡老師的圓滑畫風，因此我很開心可以畫這些原畫。當然，甚至後來我記得我的同事認識週刊少年JUMP的『怪博士與機器娃娃』後，就訴說著「如果它被製作成動畫的話，會很有趣」。<br>
我很榮幸可以參與『七龍珠』系列的製作，我也因此變得擅長繪畫。其實平常在和別人自我介紹時，我很少提到自己曾經參與製作『七龍珠』，但是他們都會主動提起「他們在電視開場畫面上的字幕有看到我的名字」。我覺得自己可以長時間參與製作一套人氣作品真的是很開心，而且那段時間氣氛相當地愉快，尤其在觀賞重播的時候，我總會想起那時候發生過的事情，它對我來說是一個很有價值的經驗。如果未來還有新作，我想再次參與『七龍珠』的製作，不過我覺得應該要讓新的年輕人來負責，我想讓他們向我展現二十一世紀的孫悟空帶給我新的啟發。在這個意義上，『七龍珠』是我心目中無可取替的寶物，但另一方面來說，它並非佔據了我心目中的一角，我為何這樣說……？因為這套作品與其說是以某個形態存在於我心目中，不如說它讓我整個人稍微增添了色彩。」<ref>《七龍珠動畫畫集 金色之戰士》前田實的訪問</ref>

== 評價 ==
網路上有很多來自[[中國|中國]]、[[香港|香港]]和[[臺灣|臺灣]]的七龍珠[[粉絲文化|粉絲]]經常在抱怨和不滿，他們非常討厭第一任的人物設計師與總監的[[前田實|前田實]]把七龍珠的主角[[孫悟空_(七龍珠)|悟空]]描寫成英雄救世主。七龍珠的粉絲認為，"'''''真正的悟空是只想找強者打架的人，而不是維護和平正義'''''"。很有可能是由他開始從[[七龍珠Z|七龍珠Z]]一直沿用到[[七龍珠GT|七龍珠GT]]結束，不過在[[龍珠超|七龍珠超]]已經獲得-{修}-正，七龍珠超已將悟空變回單純只想與強者戰鬥的性格。<br>

前田從事動畫的工作至今已有五十年沒有停歇，特別是1970年代~1990年代初期非常活耀，在很多動畫片經常可以看到關於他的作品。然而從離開Junio之後，除了麵包超人之外很少做其他動畫。不少日本民眾覺得除了麵包超人，感覺還是麵包超人。まいせ評論道“我還是喜歡前田實先生的麵包超人的作畫，男性人物帥，女性人物超級可愛”<br>
[[川越淳|川越]]先生“繪本原作家[[柳瀬嵩|柳瀬]]老師辭世，人物設計師的工作更加更加地艱難。劇本的[[米村正二|米村]]先生寫了的故事原案，我也照案畫著跟廠商洽談。可是廢柴機器人無論怎麼畫，我和前田先生２人都會感覺"這個人物好像曾經在哪個時候已經登場了"（笑）。前田先生也畫過很多次各式各樣的畫，雖然最後還是畫出來，但還是很辛苦。畫分鏡的時候，我一直煩惱要怎麼讓他變形（笑）。”<ref>[https://abematimes.com/posts/1003737 劇場版『アンパンマン』最新作監督インタビュー　子どもたちへ伝えるメッセージとは？]</ref>

[[清水保行|清水]]先生:“我在年輕的時候參與過動畫片『麵包超人』與『TOUCH鄰家女孩』的製作。我很感激前田實先生這樣的動畫師前輩關照，他的指導和教學方式很獨特。前田先生雖然-{修}-改年輕人畫的畫作，但是動作和時機他絕對不會改。這樣一來，也不失自己的風格，我的動力也提高了不少，所以我希望我也能像前田先生這樣的方式指導新人。”<ref>アニメミライ 第3回＊『黒の栖-クロノス-』作画監督 清水保行さんスペシャルインタビュー！！</ref><br>

[[井上俊之|井上]]先生:“大概是我立志成為動畫製作人之後。那個時候印象最深刻的是『巨人之星』系列後半部，前田実一個人原畫的那一話。比起[[芝山努|芝山]]先生，前田先生獨特的色彩更濃厚更加的有個性。 近年來前田先生只擔任動畫人物設計、作畫監督和主管的身份，所以我認為能看到關於他的原畫作品的機會較少。由於我們在同一個工作室所以能經常碰面，我能親眼看見前田先生的漂亮的原畫。”<ref>アニメスタイル 2005/07/19 第3回 井上俊之『ギャートルズ』と作画を観る喜び</ref><br>

[[小黑祐一郎|小黑祐一郎]]先生:“非常強大，親自製作了『[[銀河鐵道之夜|銀河鐵道之夜]]』和劇場版『TOUCH鄰家女孩』的動畫導演。前田實先生擔任總作畫監督也一邊檢查『TOUCH鄰家女孩』全集，同時期『怪博士與機器娃娃』，結束後看『七龍珠』看全話的原畫，劇場版『TOUCH鄰家女孩』和『怪博士與機器娃娃』的作畫監督也擔任。這個工作量簡直是超人！”<ref>アニメスタイル 2009/11/26 第257回 『タッチ』（ＴＶ版）小黑祐一郎</ref>

[[佐藤正樹#2017年1月Hobby_Consolas採訪|佐藤]]先生:“我初次遇見前田師傅時，當時正在讀大學三年級，像是工讀性質的實習。第一次去面試的時候，那一天師傅正在繪製第一部「鄰家女孩」的劇場版海報，而且很漂亮。當時師傅正在為「鄰家女孩」系列動畫片製作，另一方面還有「怪博士與機器娃娃」的結尾和「七龍珠」準備要開始的動畫專案，所以師傅也正在畫七龍珠動畫版的OP，好像還有做其它的動畫作品。我因為從其他公司有傳聞關於他過度辛勤於工作的工作狂傳聞，所以我有點害怕。後來有一天師傅找我去七龍珠製作團隊，我們每個星期都要跟他去吃飯.....前田師傅是一位有趣的人，我自己不會喝酒，可是在師傅面前真的很難做到，不管怎麼樣最少都要上喝一杯，雖然師傅現在已經戒掉了，改喝別的。當七龍珠動畫快要追上原作漫畫進度的時候，師傅先看了原作漫畫，說：「拿去吧，飲茶死了。」因為我還沒有讀過，我們全部人都很吃驚，起初我還以為師傅是在開玩笑，但是......飲茶真的已經死了，我覺得飲茶死了很好笑。”<ref>Hobby Consolas 2016/12 採訪龍珠偉大的動畫師－佐藤正樹</ref>
