'''琼州海峡'''，是[[海南省|海南省]]的[[海南岛|海南岛]]与[[广东省|广东省]]的[[雷州半岛|雷州半岛]]之间所夹的水道，因海南岛的别称“琼州岛”而得名，是[[海峡#中國接鄰的著名海峡|中国三大海峡]]之一。琼州海峡东西长约80公里，南北平均宽为29.5公里，最宽处直线距离为33.5公里，最窄处直线距离仅18公里左右，平均水深44米，最大深度114米，水域面积2300余平方千米。<ref name="a">{{cite web |url = http://www.cqgeo.cn/dlbaike/ShowArticle.asp?ArticleID=22823 |title = 《琼州海峡》 |accessdate = 2010-06-09 |publisher = 重庆教科院地理室 |language = zh-hans }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 历史 ==
[[海南島|海南島]]因瓊州海峽之隔，歷史上被視為荒蠻之地。唐代高僧[[鑑真|鑑真]]第五次東渡日本，被困於海南島，唐代宰相[[李德裕|李德裕]]也曾有“一去一萬里，千之千不還”之哀，宋代[[蘇東坡|蘇東坡]]有“滄海何曾斷地脈，珠崖從此破天荒”之嘆。元符三年（1100年）六月蘇東坡渡过琼州海峡返北，在悍江边吟道：“我心本如此，月满江不湍”。1605年7月，海峡地域发生大[[地震|地震]]，引发两岸部分陆地沉降，导致一些村庄淹没在海水中。

1949年12月，[[毛泽东|毛泽东]]向[[中国人民解放军|解放军]]第四野战军发出强渡琼州海峡的命令，准备占领中华民国政府控制的海南岛。1950年3月起，解放軍在海南地方武装瓊崖縱隊的配合下开始[[海南島戰役|海南島戰役]]，[[邓华|邓华]]为总指揮。解放军首先派小股先遣队潜渡琼州海峡并成功登陆，4月16日发起大规模渡海强攻，23日占领[[海口市|海口市]]，5月1日攻佔海南全島，中华民国国军则退居台湾。<ref>{{cite web|url=http://www.chinanews.com.cn/cul/news/2010/04-19/2234442.shtml|title=解放海南：渡海作战部队和琼崖纵队完美配合|accessdate=2010-06-09 |publisher=中国新闻网 |language=zh-hans }}</ref>

1988年3月21日，[[中华人民共和国|中国]][[北京体育学院|北京体育学院]]教师[[张健_(游泳运动员)|张健]]以游泳方式横渡海峡成功，耗时9小时12分，涉水长度29.5公里，此举轰动一时<ref>{{cite web|url=http://sports.sina.com.cn/o/2010-04-24/16354954314.shtml|title=《张健横渡队大事记：1988首渡琼州海峡 横渡英吉利》|accessdate=2010-06-09 |publisher=[[中华人民共和国|中国]]新浪网 |language=zh-hans }}</ref>。

== 地理 ==
[[File:Olivine_basalt2.jpg|thumb]]]]

琼州海峡是[[中国大陆|中国大陆]][[雷州半岛|雷州半岛]]与[[海南岛|海南岛]]的连接水域，东西沟通[[中国南海|中国南海]]北部与[[北部湾|北部湾]]<ref name="a"/>。海峡海水的透明度5米，盐度数值为30，年平均温度25-27摄氏度<ref>{{cite web|url=http://blog.sina.com.cn/s/blog_6183a1e40100eeqp.html|title=《畅游中国三大海峡之一——琼州海峡》|accessdate=2010-06-09 |publisher=新浪网（[[中华人民共和国|中国]]）|language=zh-hans }}</ref>。琼州海峡在[[地质|地质]]史上与[[海南岛|海南岛]]和[[雷州半岛|雷州半岛]]为完整陆地，由于[[第四纪|第四纪]][[冰川|冰川]]时期的地壳运动，在现今的海峡区域形成断陷，海水将断陷区覆盖形成[[海峡|海峡]]。海峡两岸蜿蜒曲折，凸出的海角与凹入的海湾交替分布。北岸为[[玄武岩|玄武岩]]地质构造；南岸为熔岩地台；[[火山岩|火山岩]]在两岸均有分布。海峡南北较浅，中部最深，南北横切面大致呈“V”字形构造。琼州海峡[[海流|海流]]汹涌，强烈侵蚀着海地地貌，形成海底冲槽和沙垅。<ref name="a"/>

== 海上交通 ==
琼州海峡是中国大陆连接[[海南岛|海南岛]]的陆海交通咽喉。公路交通由大陆的G75高速公路通过海安汽车轮渡至[[海口市|海口]]连接[[海南省|海南]]公路网；铁路则通过海安的[[粤海铁路|粤海铁路]]专用码头将火车轮渡到海口与海南铁路接驳。瓊州海峽東部海況險惡，被視為远洋航行畏途。
===交通瓶颈与跨海大桥计划===
[[广东省|广东省]][[徐闻县|徐闻]]海安港的官员认为，海安港汽车轮渡的旅客及汽车流量巨大，并以每年以百分之十五的幅度增长，海峡运输压力明显，建造桥梁为可行之举，可以缓解当前交通压力<ref name="b"/>。据[[中华人民共和国|中国]]媒体报道，[[中华人民共和国|中华人民共和国]]将在琼州海峡建设跨海大桥，桥梁分上下两层，上层为公路，下层为[[铁路轨道|铁路轨道]]，总投资达1400亿元[[人民币|人民币]]，预计2020年建成。媒体引述专家观点认为，琼州海峡大桥建成后，对南海资源开发及推动中国与[[东南亚国家联盟|东盟]]国家的自由贸易区建设具有重大意义。<ref name="b">{{cite web|url=http://news.163.com/10/0109/16/5SJM3VAG000120GU.html|title=《海南将建琼州海峡跨海大桥 总投资1400亿元》|accessdate=2010-06-09 |publisher=网易网（[[中华人民共和国|中国]]） |language=zh-hans }}</ref>
