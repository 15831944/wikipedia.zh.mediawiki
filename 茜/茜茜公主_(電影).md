《'''茜茜公主'''》（''{{lang|de|Sissi}}''）是[[奥地利|奥地利]]拍摄的[[历史|历史]]题材的[[德语|德语]][[三部曲|三部曲]][[电影|电影]]。第一部公映于1955年，电影讲述了奥地利皇后[[伊莉莎白_(巴伐利亞公主)|伊丽莎白]]（1837年—1898年）的早年生活，故事改编自1952年出版的玛丽·布兰克-艾斯曼（{{lang|de|Marie Blank-Eismann}}）的同名[[小说|小说]]。由于第一部《茜茜公主》在[[票房|票房]]收入上获得了巨大成功，1956年和1957年又相继拍摄并公映了续集《'''茜茜公主2：年轻的皇后'''》（''{{lang|de|Sissi – Die junge Kaiserin}}''）和《'''茜茜公主3：皇后的命运'''》（''{{lang|de|Sissi – Schicksalsjahre einer Kaiserin}}''）。

《茜茜公主》三部曲是历史上最成功的德语电影之一，也是在[[德国|德国]]获得最高票房收入的电影之一，第一部观众约6,538,000人，第二部观众约6,385,000人<ref>{{Cite web |url=http://www.sissi.de/filme/presse_sissi_weltweit.php |title={title} |access-date=2007-03-22 |archive-url=https://web.archive.org/web/20070225093133/http://www.sissi.de/filme/presse_sissi_weltweit.php |archive-date=2007-02-25 |dead-url=yes }}</ref>。

这部电影的[[编剧|编剧]]和[[导演|导演]]是奥地利[[剧作家|剧作家]]恩斯特·马利施卡（{{lang|de|Ernst Marischka}}，1893年1月2日—1963年5月12日），饰演[[奥匈帝国|奥匈帝国]][[皇后|皇后]]伊丽莎白（茜茜）的是德国籍和[[法国|法国]]籍[[维也纳|维也纳]]女演员[[罗密·施奈德|罗密·施奈德]]，饰演奥匈帝国[[皇帝|皇帝]][[弗朗茨·约瑟夫一世|弗朗茨·约瑟夫一世]]的是奥地利演员[[卡尔海因茨·伯姆|卡尔海因茨·伯姆]]。

== 第一部 ==

伊丽莎白（昵称“茜茜”，罗密·施奈德饰）是[[巴伐利亚王国|巴伐利亚王国]][[马科西米利安·约瑟夫|马克思公爵]]和[[卢多维卡·威廉明娜|卢多维卡公主]]的次女，生性活泼，热爱自然。而父亲和母亲对子女有着截然相反的教育观念。故事开始于她同她的母亲卢多维卡和姐姐海伦（昵称“内那”）在从[[巴伐利亚|巴伐利亚]]去奥地利以舍尔的旅行途中。她们此行的目的是为了让内那与卢多维卡的[[外甥|外甥]]——23岁的年轻奥地利[[皇帝|皇帝]][[弗朗茨·约瑟夫一世|弗朗茨·约瑟夫]]（[[卡尔海因茨·伯姆|卡尔海因茨·伯姆]]饰）[[订婚|订婚]]。这份婚事是由卢多维卡的姐姐、弗朗茨母亲——专制的[[苏菲公主_(巴伐利亚)|索菲]]钦定的。为了向马克思公爵，隐瞒此次行程的目地，母女两人决定以“旅行”的名义带上茜茜，故茜茜对这份[[婚约|婚约]]一无所知。到达以舍尔后，她在独自钓鱼的途中偶遇出行的弗朗茨，两人[[一见钟情|一见钟情]]。

弗朗茨向茜茜表白时，茜茜才得知弗朗茨内定的订婚对象是姐姐内那。茜茜未能告之弗朗茨自己的真实身份，匆忙与弗朗茨分别。当晚将为弗朗茨举行生日晚宴。晚宴后的[[舞会|舞会]]上，弗朗茨和内那订婚的消息即将公开。茜茜未能拒绝参加晚宴。晚宴上，再次见到茜茜的弗朗茨，终于得知了她真实的身份。无可救药地爱上了茜茜的弗朗茨，违背母亲索菲的旨意，宣布和茜茜订婚，茜茜将要成为[[奥地利皇后|奥地利的皇后]]。

此后，姨母索菲对这位来自巴伐利亚带着豪放狂野作风的外甥女茜茜十分不满，但却又无能为力，只能努力地教她宫廷礼仪。茜茜乘船凯旋般地回到[[维也纳|维也纳]]，同弗朗茨在施特凡大教堂举行了隆重的[[皇家婚礼|皇家婚礼]]，成为了奥地利的皇后。

== 第二部 ==
[[File:Wappen_Kaiserin_Elisabeth.png|thumb]]
茜茜开始了她皇后的生活，但倔强的茜茜和专制的婆婆索菲之间的矛盾却始终不断，在茜茜的女儿出生后，索菲不愿意让来自巴伐利亚的狂野的茜茜抚养小公主，而是要亲自抚养并把小公主从茜茜身边夺走。弗朗茨也顺从了母亲的旨意，茜茜一气之下逃回了巴伐利亚，弗朗茨亲自带着小公主才把她又接回了维也纳，他们和好如初。

此时的奥地利同[[匈牙利|匈牙利]]之间发生摩擦，将要发生[[战争|战争]]，尤其是野心勃勃的革命党领袖、匈牙利伯爵[[久洛·安德拉什|安多西]]不服奥地利对匈牙利的统治，鼓动匈牙利不甘于俯首称臣的贵族们起来反抗。茜茜以她的个人魅力征服了安多西伯爵，同时也征服了匈牙利的人民，最终，茜茜和弗朗茨加冕成为[[奥匈帝国|奥匈帝国]]的皇后和皇帝。

== 第三部 ==
[[File:Franz_joseph1.jpg|thumb]]

奥匈帝国成立后，茜茜居住在匈牙利，她同弗朗茨在一起的时间越来越少，[[久洛·安德拉什|安多西]]伯爵也爱上了茜茜并向她示爱，遭到了茜茜的婉言拒绝。婆婆索菲却在背后散布茜茜与伯爵关系不寻常的谣言。

与此同时，茜茜染上了严重的[[肺结核|肺结核]]，只得去[[大西洋|大西洋]]上的[[马德拉岛|马德拉岛]]修养，茜茜和弗朗茨再次分离。这个消息令十分爱戴茜茜的奥匈人民非常不安，所幸茜茜在母亲卢多维卡的悉心照料下最终痊愈。

病愈后，茜茜与弗朗茨一同访问奥地利统治下的[[意大利|意大利]]，却遭到了意大利人民的冷遇，在[[威尼斯|威尼斯]]，意大利人紧闭窗门，用沉默来表达自己对外族统治的憎恶。此时出现了感人的一幕，等候在[[圣马可广场|圣马可广场]]上小公主跑向她的妈妈，茜茜紧紧地抱住女儿，这一举动感动了意大利人民，他们高呼“Viva la Mama”（意大利语的“母亲万岁”），茜茜为奥匈帝国赢得了意大利人民的心，电影背景响起了奥匈帝国的[[国歌|国歌]]。

== 背景资料 ==
=== 电影 ===
* 小说和电影《茜茜公主》中“Sissi”的名字其实是个误写，伊丽莎白皇后在家中的昵称其实是“Sisi”，她自己在签名时有时候也使用“Lisi”。
* 电影《茜茜公主》的编剧和导演恩斯特·马利施卡曾计划拍摄第四部，并为第四部定下茜茜同[[法国皇后|法国皇后]][[欧根妮|欧根妮]]（Eugènie）之间争风吃醋的故事。但是由于在前三部饰演茜茜的罗密·施奈德拒绝再出演茜茜而作罢<ref name="niewieder">{{Cite web |url=http://www.sissi.de/filme/nie_wieder_sissi.php |title={title} |access-date=2007-03-22 |archive-url=https://web.archive.org/web/20070225082518/http://www.sissi.de/filme/nie_wieder_sissi.php |archive-date=2007-02-25 |dead-url=yes }}</ref>。
* 电影《茜茜公主》在讲述这个闻名遐迩的爱情故事的同时，也向人们展现了奥地利的迷人风光，至今仍对奥地利的[[旅游业|旅游业]]起着积极作用。
* 半个世纪过去了，人们对电影《茜茜公主》的热情不减，它依旧是爱情故事作品的模范。

=== 演员 ===
* 由于在电影《茜茜公主》中塑造人物形象的成功，反而使得主演罗密·施奈德和卡尔海因茨·伯姆都无法走出茜茜和弗朗茨的影子，他们都表示不愿意一生的演艺被框在这两个形象中，也因此拒绝再次出演《茜茜公主》<ref name="niewieder"/>。
* 扮演茜茜的德籍和法籍维也纳女演员罗密·施奈德当年年仅16岁，《茜茜公主》使她一下子出了名。她那迷人的微笑驱散了[[二战|二战]]后奥地利人民心中的阴霾，给人们带来了重建家园的信心。她也迅速征服了欧洲人的心，赢得了“战后欧洲第一美人”的赞誉<ref name="people">http://www.people.com.cn/GB/paper68/6243/617365.html</ref>。但无论是现实中的茜茜还是饰演茜茜的罗密·施奈德，最后都十分不幸地结束了自己的一生。历史上的茜茜公主，即奥匈帝国的皇后于1898年在[[日内瓦|日内瓦]]养病时被一名意大利[[无政府主义|无政府主义]]者谋杀。而在电影中饰演茜茜的罗密·施奈德在1982年遭遇了独生子不幸夭折的打击，44岁的她被发现死于[[巴黎|巴黎]]的家中，[[媒体|媒体]]认为她是[[自杀|自杀]]，但验尸报告表明她死于心力衰竭，属于正常死亡<ref>{{Cite web |url=http://www.sissi.de/filme/romy_schneider.php |title={title} |access-date=2007-03-22 |archive-url=https://web.archive.org/web/20070225131943/http://www.sissi.de/filme/romy_schneider.php |archive-date=2007-02-25 |dead-url=yes }}</ref>。她留下一个女儿莎拉·比阿西尼也是演员，同样拒绝扮演茜茜公主<ref>http://www.dw-world.de/dw/article/0,2144,1449646,00.html</ref>。
* 扮演弗朗茨的奥地利演员卡尔海因茨·伯姆于2014年5月29日逝世，从事为[[埃塞俄比亚|埃塞俄比亚]]募捐的事业，2005年他出席在德国[[斯图加特|斯图加特]]举行的纪念《茜茜公主》上演五十周年、名为“你好，茜茜！”的活动。参加活动的还有弗朗茨与茜茜的重孙——[[哈布斯堡王朝|哈布斯堡王族]]的大公爵马尔库斯·萨尔瓦托<ref>http://www.dw-world.de/dw/article/0,2144,1830587,00.html</ref>。卡尔海因茨·伯姆给他1955年出生的女儿也取名为“Sissi”<ref>{{Cite web |url=http://www.sissi.de/filme/kurios_namenspatin.php |title={title} |access-date=2007-03-22 |archive-url=https://web.archive.org/web/20070503023507/http://www.sissi.de/filme/kurios_namenspatin.php |archive-date=2007-05-03 |dead-url=yes }}</ref>。

=== 真实历史 ===
[[File:Kaiser_Franz_Joseph_tomb_-_Vienna.jpg|thumb]]在维也纳的棺墓]]
历史上真实的茜茜公主并非完全像电影《茜茜公主》中那样。

* 茜茜与弗朗茨的爱情，完全没有电影中描述的那么完美。弗朗茨受过严格的宫廷教育，而茜茜是在[[巴伐利亚|巴伐利亚]]的湖光山色中自由自在地成长的。两种不同的气质最初相互吸引，逐渐地却显得格格不入。成为了伊丽莎白皇后的茜茜虽然享尽荣耀富贵，却总是郁郁寡欢。
* 茜茜从内心里一直拒绝扮演传统的妻子、母亲、皇后乃至一个大帝国形象代表的角色。晚年的茜茜心灰意冷，周游欧亚非列国。1898年，她在日内瓦被一名无政府主义者杀害。尽管如此，这位皇后仍以美貌、魅力和浪漫的忧郁气质而受到臣民的爱戴<ref name="people"/>。
