'''井冈山市'''位于[[江西省|江西省]]中部偏西，隶属于[[吉安市|吉安市]]。以中共[[井冈山革命根据地|井冈山革命根据地]]而闻名。

==历史沿革==
*自西晋太康元年（280）起，此地域分属庐陵郡西昌、遂川县地，随着朝代的转换和建置的不同，此地域仍分属永新、遂川和泰和（即拿山乡小通村）辖地。<ref>{{Cite web |url=http://www.jgs.gov.cn/html/2015/8-11/n69776274.html |title=井岡山市人民政府網-行政区划 |access-date=2017-11-18 |archive-url=https://web.archive.org/web/20171201033032/http://www.jgs.gov.cn/html/2015/8-11/n69776274.html |archive-date=2017-12-01 |dead-url=yes }}</ref>
*1928年2月，中共湘赣边界党和工农兵政府在茨坪建立新遂边陲特别区工农兵政府，属遂川县工农兵政府管辖。10月底，新遂边陲特别区改称大小五井特别区，归中共宁冈县委管辖。<ref>{{Cite web |url=http://www.jgs.gov.cn/html/2015/8-11/n69776274.html |title=井岡山市人民政府網-行政区划 |access-date=2017-11-18 |archive-url=https://web.archive.org/web/20171201033032/http://www.jgs.gov.cn/html/2015/8-11/n69776274.html |archive-date=2017-12-01 |dead-url=yes }}</ref>
*1950年在[[茨坪|茨坪]]设立了井冈山特别区
*1959年成立省辖井冈山管理局
*1981年撤局设县
*1984年经[[中華人民共和國國務院|国务院]]批准撤县设市
*2000年5月经国务院批准将原井冈山市与原[[宁冈县|宁冈县]]合并组建新的井冈山市。
