《'''井冈山'''》，[[中国大陆|中国大陆]]电视剧，导演[[金韬|金韬]]，主演[[王霙|王霙]]{{noteTag|1=“霙”读音“Yīng”，音同“英”。}} 、[[王伍福|王伍福]]、[[潘雨辰|潘雨辰]]、[[黄鹏|黄鹏]]、[[宋佳伦|宋佳伦]]。

== 剧情 ==
1927年，[[中华民国总统|中华民国总统]][[蒋中正|蒋中正]]为了消灭地方割据势力，发动了对[[中国共产党|中国共产党]]的围剿行动，几次大围剿，使得[[红军|红军]]损失惨重，红军内部高参的[[左倾|左倾]]和正面直接对抗战略方针，使得红军不断转弱，为了挽救危局，毛泽东转入[[井冈山|井冈山]]，开辟了井冈山革命根据地，后朱德、陈毅率领部队到井冈山与毛泽东会师。

== 花絮 ==
该剧是“中国人民解放军建军80周年”献礼剧。<ref>{{cite web |url=http://zqb.cyol.com/content/2007-07/23/content_1834341.htm |title=重大革命历史题材电视剧《井冈山》开播 |accessdate=2007 |publisher=中国青年报 |work= |language =zh-cn}}</ref><ref>{{cite web|url=http://www.wuhunews.cn/wenti/tiyu/2008/04/2008-04-2538544.html|title=电视剧《井冈山》今天登陆央视|accessdate=2008|publisher=芜湖新闻网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20160304060832/http://www.wuhunews.cn/wenti/tiyu/2008/04/2008-04-2538544.html|archivedate=2016-03-04}}</ref>

该剧的制作方是中央电视台文艺中心影视部、[[南京军区|南京军区]]政治部前线文工团、[[江西电视台|江西电视台]]和[[中央纪委|中央纪委]]电教中心，出品方是[[中国中央电视台|中国中央电视台]]、[[解放军总政治部|解放军总政治部]]宣传部、江西省委宣传部、南京军区政治部和中央纪委宣教室。<ref name="cat"/>

该剧曾在[[江西省|江西省]]的[[井冈山市|井冈山市]]、[[兴国县|兴国县]]、[[吉安市|吉安市]]，[[浙江省|浙江省]]的[[宁波市|宁波市]]、[[金华市|金华市]]，[[广东省|广东省]]的[[中山市|中山市]]，[[俄罗斯|俄罗斯]]的[[莫斯科|莫斯科]]、[[圣彼得堡|圣彼得堡]]取景。<ref name="cat">{{cite web |url= http://news.sohu.com/20070731/n251324006.shtml|title=电视剧《井冈山》阵容强大 |accessdate= 2007|publisher= 搜狐网|work= |language =zh-cn}}</ref>摄影师是顾克利。<ref>{{cite web|url=http://ent.people.com.cn/GB/8222/86598/93082/93087/6039572.html|title=资料：电视剧《井冈山》主创人员|accessdate=2007年7月27日|publisher=人民网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120531120854/http://ent.people.com.cn/GB/8222/86598/93082/93087/6039572.html|archivedate=2012年5月31日}}</ref>

2007年7月22日，该剧在[[人民大会堂|人民大会堂]]首映。<ref>{{cite web |url=http://jiangxi.jxnews.com.cn/system/2007/07/23/002526440.shtml |title=电视剧《井冈山》昨在京首映 |accessdate=  2007-07-23 |publisher=江西新闻中心 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://news.qq.com/a/20070722/001627.htm |title= 重大革命历史题材电视连续剧《井冈山》在京首映|accessdate= 2007年7月22日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

该剧讲述了毛泽东、杨开慧、贺子珍的爱情纠葛，袁文才、王佐被冤杀的故事。<ref>{{cite web |url=http://media.people.com.cn/GB/40606/6933819.html |title=电视剧《井冈山》编剧邵钧林：《井冈山》点燃激情 |accessdate=2008 |publisher=人民网 |work= |language =zh-cn}}</ref>

由于该剧的热播，致使井冈山的游客大增。<ref>{{cite web |url=http://news.sina.com.cn/c/2007-10-04/124112675264s.shtml |title= 电视剧《井冈山》热播搅热井冈十一游|accessdate=2007 |publisher=新浪网 |work= |language =zh-cn}}</ref>
