'''井上真樹夫'''（{{bd|1938年|11月30日|catIdx=Makio INOUE}}<ref>長らく、各種『俳優名鑑』には『1940年生まれ』と紹介されていたが、2012年頃のTwitterでのファンとのやり取りで1938年の同日生まれである事を公言した。</ref>）是[[日本|日本]]的男性[[演員|演員]]、[[聲優|聲優]]，本名：井上孝夫<ref> 『声優名鑑』、事務所発表、360頁、成美堂出版、1999年、ISBN 978-4415008783</ref>。所屬[[青二Production|青二Production]]。 [[山梨縣|山梨縣]]出身。

== 人物 ==
在16歲時，已經開始出演電視劇、在配音的黎明時期十分活躍。

動畫『魯邦三世』的電視第2期直至2010放送的『魯邦三世 the Last Job』<ref>[http://www.cinematoday.jp/page/N0035923 「ルパン三世」最新作は新声優で!銭形警部の声優に山寺宏一!峰不二子、石川五ェ門も一新!]シネマトゥデイ 2011年10月9日</ref>，石川五右衛門都由他聲演的。

競爭對手役很多，因配演[[宇宙海賊哈洛克船長|宇宙海賊哈洛克船長]]的主角哈洛克而出名。

於2013年作為演員在[[木村拓哉|木村拓哉]]主演的TBS電視劇[[安堂機器人|安堂機器人]]客串演出大物政治家、在續年2014年再於電視劇[[家族狩獵|家族狩獵]]出演[[癡呆症|癡呆症]]老人。<ref>{{Cite web |author=木俣冬 |date=2014-07-18 |url=http://www.excite.co.jp/News/reviewmov/20140718/E1405612252356.html |title=ハーロックが認知症。アニメ世代に老いの現実を実感させる仕掛けなのか「家族狩り」2話 |publisher=エキサイト |accessdate=2014-07-26}}</ref>
