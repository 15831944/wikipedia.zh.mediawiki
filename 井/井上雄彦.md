'''井上雄彥'''（{{jpn|j=井上 雄彦|hg=いのうえ たけひこ|rm=Inoue Takehiko|f=h}}，{{bd|1967年|1月12日|||catIdx=Inoue, Takehiko}}），[[日本|日本]][[漫畫家|漫畫家]]。男性。[[鹿兒島縣|鹿兒島縣]][[大口市|大口市]]（現[[伊佐市|伊佐市]]）出身。B型[[血型|血]]。<br />本名：'''成合雄彥'''（{{lang|ja|成合 雄彦／なりあい たけひこ}}）。暱稱「'''{{lang|ja|イノタケ}}'''（Inotake）」。

== 簡介 ==
代表作是《[[灌籃高手|灌籃高手]]》、《[[浪人劍客|浪人劍客]]》、《[[REAL|REAL]]》。井上雄彥的作品以描述青年在歷經[[體育|運動]]或戰鬥中成長之過程為主。

1988年，井上使用本名（成合雄彥）以短篇「紫色的楓（<small>{{lang-ja|楓パープル}}</small>）」獲得[[手塚獎|手塚獎]]出道。1990年，他開始改用現在的筆名在《[[週刊少年Jump|週刊少年Jump]]》（[[集英社|集英社]]）連載籃球題材漫畫《[[灌籃高手|灌籃高手]]》。連載期間，日本國內和海外鄰近國家掀起了打籃球的熱潮<ref name="AER">《日本的漫畫 手塚治虫文化獎10週年紀念》，[[朝日新聞社|朝日新聞社]]，2006年，第60～75頁。</ref>。1998年開始，轉往《[[Morning_(漫畫雜誌)|MORNING]]》（[[講談社|講談社]]）開始連載《[[浪人劍客|浪人劍客]]》。次年1999年開始，在《[[週刊YOUNG_JUMP|週刊YOUNG JUMP]]》（集英社）連載輪椅籃球題材漫畫《[[REAL|REAL]]》。之後，《浪人劍客》獲頒[[文化廳媒體藝術祭|媒體藝術祭]]大獎和[[手塚治虫文化獎|手塚治虫文化獎]]大獎。2012年，井上本人獲頒{{link-ja|平城遷都1300年紀念ASIA COSMOPOLITAN獎|平城遷都1300年記念アジアコスモポリタン}}文化獎。

== 來歷 ==
井上雄彥在小、中學時期是[[劍道|劍道]]社所屬，高中開始則加入在當時仍冷門的籃球社，並擔任主將。另一方面，他從小就對繪畫十分有興趣，高中畢業時已經確立要成為漫畫家的志向。特別是幼少時期很喜歡{{link-ja|水島新司|水島新司}}的漫畫《{{link-ja|大飯桶|ドカベン}}》，其他還有受到[[池上遼一|池上遼一]]的《[[男組|男組]]》、[[小林誠_(漫畫家)|小林誠]]的漫畫作品所影響<ref name="MAS">{{lang|ja|ティムリーマン}}《{{lang|ja|マンガマスター}}》，美術出版社，2005年，第113～128頁。</ref>。

高中3年級（{{link-ja|鹿兒島縣立大口高等學校|鹿児島県立大口高等学校}}）時，井上雖然以進學藝術大學為前提，前往美術預校接受暑期講習，但因「很耗金錢」等理由而改選擇當地較近的[[熊本大學|熊本大學]]就讀。20歲（1987年）時投稿到[[集英社|集英社]]《[[週刊少年Jump|週刊少年Jump]]》時，被編輯[[中村泰造|中村泰造]]發掘<ref>{{Cite web|date=2009-05-09|url=http://doraku.asahi.com/hito/runner2/100105_03.html|script-title=ja:漫画家 井上雄彦（いのうえ·たけひこ）さん（3/3）|publisher=[[朝日新聞|朝日新聞]] DO樂|accessdate=2011-08-16|language=ja|archive-url=https://web.archive.org/web/20121101034358/http://doraku.asahi.com/hito/runner2/100105_03.html|archive-date=2012-11-01|dead-url=yes}}</ref>。並在導師的鼓勵支持下，離開已就讀三年的[[熊本大學|熊本大學]][[文學|文學]]部，投身於漫畫業界發展。在[[北條司|北條司]]（連載《[[城市獵人|城市獵人]]》期間）門下做了將近10個月的助理，磨練漫畫製作的基礎<ref name="MAS" />。次年1988年，他投稿短篇「紫色的楓」入圍第35屆[[手塚獎|手塚獎]]，從此正式出道。

出道之後，井上至1990年進行《灌籃高手》的首部長期連載之前，在《週刊少年Jump》發表過許多短篇的籃球漫畫，及第一次負責作畫連載《變色龍傑爾》（原作：渡邊和彥）。灌籃高手開始連載之後，日本國內掀起了打籃球的熱潮，而且每更新一回閱讀人數也不斷的逐步攀升。1993年至1996年播出同名電視動畫，更奠定了井上穩固日本漫畫的地位。1995年榮獲第40屆[[小學館漫畫獎|小學館漫畫獎]]，1996年連載終了。2006年也入圍[[文化廳|文化廳]]舉辦問卷調查企劃「{{link-ja|日本媒體藝數百選|日本のメディア芸術100選}}」的漫畫部門第1名<ref name="interview_inoue">{{Cite web|url=http://bj.shueisha.co.jp/interview_inoue/page01.html|script-title=ja:ビジネスジャンプ·井上雄彦インタビュー|work=Business Jump|publisher=[[集英社|集英社]]|date=|accessdate=2019-01-02|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20101006095507/http://bj.shueisha.co.jp/interview_inoue/page01.html|archivedate=2010-10-06}}</ref>。

1998年開始，井上以[[吉川英治|吉川英治]]的[[小說|小說]]《{{link-ja|宮本武藏 (小說)|宮本武蔵 (小説)|宮本武藏}}》為劇本，開始在[[講談社|講談社]]《[[Morning_(漫畫雜誌)|MORNING]]》連載《[[浪人劍客|浪人劍客]]》。次年1999年開始，連載輪椅籃球題材漫畫《[[REAL|REAL]]》在集英社《[[週刊YOUNG_JUMP|週刊YOUNG JUMP]]》，並說這是一部讓他掀起興趣看電視的漫畫。

2018年現在，《浪人劍客》和《REAL》等等所有長期連載作品處於休載狀態。這對井上來說，《浪人劍客》盡量不使用漫畫先驅製造出「漫畫形式的手法和漫畫符號」以力求創新，反過來《REAL》令他畫工發揮到最大限度<ref>「大特集 井上雄{{lang|ja|彦}}『{{lang|ja|リアル}}』」《{{link-ja|DA VINCI|ダ・ヴィンチ}}》2007年12月號，[[Media_Factory|Media Factory]]，第15頁～33頁。</ref>。

== 與籃球的邂逅 ==
井上說高中入學之後，在劍道社活動的哥哥發現他有打籃球的天份，於是他照哥哥的建議加入籃球社。根據井上接受專訪時表示，自己本身一開始對籃球不是很有興趣，而是在朋友的邀請下就「突然加入了」。他加入籃球社後，因為身高因素，經常擔任後衛<ref name="SW1">今井榮一「INTERVIEW 從《灌籃高手》到《浪人劍客》」《SWITCH》2002年 Vol.20 No.3，SWITCH Publishing，第40－51頁。</ref>。

學生時期的井上，已經有當上漫畫家之後一定要創作出籃球漫畫的目標，當時他覺得以籃球為題材的漫畫並不多見<ref name="AER" />。《灌籃高手》連載開始後，井上自行成立了小型的籃球隊叫「TAKECHANS」，並擔任後衛。連載終了之後，[[NHK_BS1|NHK-BS1]]播出的[[NBA|NBA]]中繼轉播至今多次特別邀請他擔任球評。

2004年，井上基於對籃球運動的感謝和回饋之意<ref name="interview_inoue" />，構思決定設立獎學金。2006年年底，他與集英社、日本籃球協會等單位共同設立「{{link-ja|灌籃高手獎學金|スラムダンク奨学金}}（<small>{{lang-ja|スラムダンク奨学金}}</small>）」，希望能幫助於日本高中畢業後，以美國大學為目標，擁有想去美國挑戰的意志和能力<ref name="interview_inoue" />。卻因為經濟或其他因素而無法圓夢的年輕選手完成夢想，讓夢想不再只存在於漫畫之中<ref name="interview_inoue" />。

此外，《灌籃高手》一些登場人物的臉是井上以高中和大學時期的社團同學和好友作為原型。

== 年譜 ==
* 1988年：以短篇「紫色之楓」入圍第35屆[[手塚獎|手塚獎]]（以本名成合雄彥名義），在漫畫業界正式出道。
* 1989年：在《[[週刊少年JUMP|週刊少年JUMP]]》（[[集英社|集英社]]）開始連載《變色龍傑爾》。
* 1990年：在《週刊少年JUMP》開始連載《[[灌籃高手|灌籃高手]]》（連載時期逐漸改現用筆名，至1996年連載終了）。
* 1996年：在{{link-ja|Sports-i ESPN|スポーツ・アイ ESPN}}網站開始連載《[[零秒出手_(漫畫)|零秒出手]]》（連載至1998年終了）。
* 1998年：在《[[Morning_(漫畫雜誌)|MORNING]]》（[[講談社|講談社]]）開始連載《[[浪人劍客|浪人劍客]]》。
* 1999年：在《[[週刊YOUNG_JUMP|週刊YOUNG JUMP]]》（集英社）開始連載《[[REAL|REAL]]》。
* 2004年：《灌籃高手》在日本國內發行超過一億本，於六份早報刊載了全版紀念廣告，並在神奈川縣的廢棄高中舉辦「一億冊感謝最終回（<small>{{lang-ja|1億冊ありがとうファイナル}}</small>）」活動。
* 2007年：擔任[[Xbox_360|Xbox 360]]電玩遊戲《[[失落的奧德賽|失落的奧德賽]]》主要角色設計。
* 2008年～2010年：在日本三個城市舉辦「[[浪人劍客#井上雄彥_最後的漫畫展|井上雄彥 最後的漫畫展]]」。
* 2010年：受到日本籃球協會的特別表揚。

: 以23歳的孤獨殺手為主的短篇作品。在《灌籃高手》連載中進行短期集中連載。

; 耳環（{{lang|ja|ピアス}}，集英社《週刊少年Jump》，1998年9號）
: 以海邊的小鎮為舞台，描述小學六年級時代的[[宮城良田|Ryota]]和少女[[彩子|Ayako]]2人之間的短篇故事。後來也在《週刊YOUNG JUMP》2001年49號重新發表。

; I LOVE THIS GAME（Adidas MANGA FEVER，2002年）
: 以足球為題材的短篇作品。
