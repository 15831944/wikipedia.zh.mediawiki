'''井之頭公園站'''（{{jpn|j=井の頭公園駅|hg=いのかしらこうえんえき|rm=Inokashira-kōen eki|f=y}}）是一個位於[[東京都|東京都]][[三鷹市|三鷹市]]{{link-ja|井之頭|井の頭}}三丁目，屬於[[京王電鐵|京王電鐵]][[井之頭線|井之頭線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''IN16'''。

== 歷史 ==
* 1933年（[[昭和|昭和]]8年）8月1日 - 帝都電鐵澀谷 - 當站（當時站名使用的是井'''ノ'''頭公園）區間開通營運。
* 1934年（昭和9年）4月1日 - 帝都電鐵當站 - 吉祥寺間路段延伸開通營運。
* 1940年（昭和15年）5月1日 - 與小田原急行鐵道合併，成為該公司帝都線的車站。
* 1942年（昭和17年）5月1日 - [[小田急電鐵|小田急電鐵]]與[[東京急行電鐵|東京急行電鐵]]（大東急）合併。
* 1948年（昭和23年）6月1日 - 從東急分離，成立京王帝都電鐵，成為該公司井之頭線的車站。
* 2006年（[[平成|平成]]18年）7月 - 車站改建完工。

== 車站構造 ==
屬於[[側式月台|相對式月台]] 2面2線的[[地面車站|地面車站]]。車站的主建築設置於吉祥寺方向的月台一側，以[[地下道|地下道]]連接往澀谷方向的乘車月台。

2006年7月，車站改建竣工。改善了無障礙設施、裝設電梯、廁所，以及車站辦公室的改建等工程。
