'''井號'''（#），是[[Unicode|Unicode]]和[[ASCII|ASCII]]字元[[35|35]]或0x0023。

在[[漢語|漢語]]，通常以其外觀喚為「[[井|井]]號」或「-{zh-cn:井字; zh-tw:井字號;}-」{{NoteTag|在其他，称为[[编号|编号]]记号或数字[[记号|记号]]。}}。在[[英語|英語]]，Unicode建議稱為number sign（數字記號），可能因為它常用於表示[[序數|序數]]，日語稱為「番号記号」或「井桁」。另一個常用於表示序數的符號是[[№|№]]。

== 不同类型的# ==
;注释符號（comment sign）
:在部分[[編程語言|編程語言]]如[[Perl|Perl]]和[[Python|Python]]，#表示一句[[注释_(计算机语言)|注释]]的開始

;pound sign
:美式英語名稱。

;hash / hash key / hash mark
:美國以外最流行的名稱；在英國和澳洲，稱呼[[電話|電話]]鍵盤中的「#」

;hex
:「#」在編程語言如[[HTML|HTML]]用以表示[[十六進制|十六進制]]

;anchor
:「#」在[[HTML|HTML]]的[[超链接|超链接]]中表示[[HTML锚|HTML锚]]。

;sharp
:外觀如[[樂譜|樂譜]]中[[变音记号|变音记号]]的[[升号|升号]]，如“C#”便讀作「C sharp」。需要注意的是，乐谱的升号和井号的字形不完全一样。标准井号字符（#）横线水平，而竖线向右倾斜；而乐谱的升号（♯）为了在五线谱中容易识别，横线改为斜向上但竖线垂直。{{NoteTag|升记号[[♯|♯]]、降记号[[♭|♭]]和[[还原音乐调|还原]]记号[[♮|♮]]不同。}}

;tic-tac-toe （美式） / noughts-and-crosses （英式）
:[[井字棋|井字棋]]的名字，因為像其棋盤

;crosshatch（交叉線）、fence（欄）、gate（閘）、grid（格）、gridlet、pig pen
:外觀相近

;[[質數階乘|質數階乘]]符號
:n的'''質數階乘'''計為'''[[自然質數階乘|n#]]'''。

;[[主題標籤|主題標籤]]（Hashtag）
:在Twitter等社群網站，用一個井號加上一個詞、單字，或沒有空格的一句話可用來標示一個特定的主題，將各篇獨立的貼文串連在一起。而在新浪微博等中文社交媒体中，多用两个#号中间夹带主题的形式，因为中文没有在词语间空格的习惯。
