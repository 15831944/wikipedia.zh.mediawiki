'''井上嘉浩'''（{{jpn|j=井上嘉浩|kn=いのうえ よしひろ|rm=Inoue Yoshihiro}}，{{bd|1969年|12月28日|2018年|7月6日}}），是一名[[奧姆真理教|奧姆真理教]]教徒、前教团高級幹部（正悟师）。井上在1988年加入奥姆真理教，因参与[[东京地铁沙林毒气事件|东京地铁沙林毒气事件]]，以及{{link-ja|仮谷清志绑架事件|公証人役場事務長逮捕監禁致死事件}}等罪行而被判处死刑。2018年7月6日，井上与包括奥姆真理教教主[[麻原彰晃|麻原彰晃]]在内的7名奥姆真理教相关死刑犯被执行死刑。

==早年生活==
井上1969年出生于日本[[京都府|京都府]][[京都市|京都市]][[右京区|右京区]]，早年父母常常吵架。井上幼时喜欢观看[[纪录片|纪录片]]，对世界的贫困问题，以及犬只的[[人道毁灭|人道毁灭]]问题都很关心。高中时代，井上嘉浩便开始与奥姆真理教接触。他相当敬佩教主[[麻原彰晃|麻原彰晃]]，认为他才是自己理想的父母。1988年，井上从京都的私立名校{{link-ja|洛南高等学校・附属中学校|洛南高等学校・附属中学校|洛南高等学校}}毕业后，在当年3月1日出家，加入奥姆真理教。随后，在麻原彰晃的要求下，井上从就读的[[日本文化大学|日本文化大学]][[法学部|法学部]][[退学|退学]]<ref>{{cite book |author1=降幡 賢一 |title=オウム法廷〈９〉諜報省長官井上嘉浩 |year=2002 |publisher=朝日新聞 |isbn=9784022613882 |pages=32, 34, 67}}</ref>。

==罪行与审判==
在奥姆真理教教团期间，井上颇受教主麻原器重，是奥姆真理教“谍报省”的负责人，负责绑架、偷听、袭击等工作<ref>{{cite web |title=“オウムの申し子” 刑を執行された井上嘉浩死刑囚　「麻原を信じたことが罪のはじまり」 |url=https://www.fnn.jp/posts/00333350HDK |website=FNN |accessdate=2018-11-08}}</ref>。井上曾参与奥姆真理教一系列与[[VX神經毒劑|VX神經毒劑]]相关的犯罪、[[东京地铁沙林毒气事件|东京地铁沙林毒气事件]]（负责幕后协调），以及{{link-ja|仮谷清志绑架事件|公証人役場事務長逮捕監禁致死事件}}等罪行。1995年5月15日，井上与同是奥姆真理教高级幹部的{{link-jp|豐田亨|豊田亨}}一同被捕。同年12月，井上要求与奥姆真理教脱离关系。井上一审时，[[检方|检方]]要求判处井上死刑，但法庭认为井上并不是主犯，而且有悔罪的表现，因此判处井上[[终生监禁|终生监禁]]，但检方不服判决结果，提起抗诉。2004年，[[東京高等裁判所|東京高等裁判所]]终审认定井上在仮谷清志绑架事件一案中犯下逮捕監禁致死罪，而且在东京地铁沙林毒气事件中井上虽然没有实际参与，但在幕后担任协调工作，确实造成了大规模平民伤亡的后果，因此改判井上死刑。2009年，井上的上诉被驳回，2010年，井上死刑定谳<ref>{{cite news|url = http://www.47news.jp/CN/201001/CN2010011301000753.html|title= オウム事件、井上被告の死刑確定 9人目|publisher = 共同通信|date = 2010-01-13|accessdate  = 2010-1-14|archiveurl = https://web.archive.org/web/20130514222921/http://www.47news.jp/CN/201001/CN2010011301000753.html|archivedate = 2013-05-14}}</ref><ref name=TokyoshinbunInoue/>。

在狱中，井上曾与毒物学家[[杜祖健|杜祖健]]合作分析造成[[金正男|金正男]]死亡的VX毒素<ref>{{cite web |title=「沒想到會變這樣」奧姆真理教幹部井上嘉浩最後遺言曝光 |url=https://www.ettoday.net/news/20180714/1212543.htm |website=ETtoday新聞雲 |accessdate=2018-11-08}}</ref>。

==死刑执行==
截至2018年3月，井上與其他十二名奧姆真理教相關死刑犯一直都被關押在{{link-ja|東京拘置所|東京拘置所}}。2018年3月21日，這些死刑犯被分散到到能執行死刑的{{link-ja|東京拘置所|東京拘置所}}等5處設施，其中井上被送到{{link-ja|大阪拘置所|大阪拘置所}}<ref name="オウム死刑囚7人移送 p.1">{{Cite news|url=http://www.sankei.com/affairs/news/180314/afr1803140030-n1.html|title=【オウム死刑囚】中川智正らオウム死刑囚７人を移送　執行施設ある５拘置所に分散か（1／2ページ）|newspaper=産経新聞|publisher=産業経済新聞社|date=2018-03-14|accessdate=2018-03-15|archivedate=2018-03-15|archiveurl=https://web.archive.org/web/20180315132452/http://www.sankei.com/affairs/news/180314/afr1803140030-n1.html}}</ref><ref name="移送7人判明 p.1">{{Cite news|url=http://www.sankei.com/affairs/news/180314/afr1803140037-n1.html|title=【オウム死刑囚】移送７人は新実智光、林泰男、早川紀代秀、井上嘉浩、岡崎一明、横山真人、中川智正の各死刑囚（1／2ページ）|newspaper=産経新聞|publisher=産業経済新聞社|date=2018-03-14|accessdate=2018-03-15|archivedate=2018-03-15|archiveurl=https://web.archive.org/web/20180315132717/http://www.sankei.com/affairs/news/180314/afr1803140037-n1.html}}</ref>。2018年7月6日，經時任日本[[法務大臣|法務大臣]][[上川阳子|上川阳子]]批准，井上嘉浩与奥姆真理教教主[[麻原彰晃|麻原彰晃]]，以及{{link-jp|早川纪代秀|早川紀代秀}}、{{link-jp|新实智光|新実智光}}、{{link-jp|土谷正实|土谷正实}}、{{link-ja|中川智正}}和{{link-ja|远藤誠一}}共7名与奥姆真理教相关的死刑犯被执行绞刑，其中井上在大阪拘置所被执行绞刑<ref name="mainichi20180706">{{cite web|url=https://mainichi.jp/articles/20180706/k00/00e/040/198000c|title=「松本死刑囚ら７人死刑執行」法務省が発表|publisher=每日新闻|date=2018-07-06|access-date=2018-11-08|archive-url=https://web.archive.org/web/20180707005810/https://mainichi.jp/articles/20180706/k00/00e/040/198000c#|archive-date=2018-07-07|dead-url=no}}</ref>。执行前，井上曾说出“没想到会变成这样”的遗言。他遗言的最后一句是“首先、很好”（{{lang|ja|まずは、よし}}）<ref>{{cite web |title=執行直前「こんなことになるとは」　井上嘉浩元死刑囚 |url=https://www.fnn.jp/posts/00396128CX |website=FNN |accessdate=2018-11-08}}</ref>。

2018年10月，在井上家人的要求下，井上嘉浩的案件将会进行再审<ref name=TokyoshinbunInoue>{{cite web |title=井上元死刑囚側、再審請求へ　元オウム幹部、来週にも |url=http://www.tokyo-np.co.jp/s/article/2018101201001096.html |website=東京新聞 |accessdate=2018-11-08}}</ref>。
