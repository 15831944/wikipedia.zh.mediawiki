'''井川遙'''（{{lang-ja|いがわ・はるか}}，{{lang-en|Igawa Haruka}}，{{bd|1976年|6月29日|catIdx=Xiuhui秀}}），[[日本|日本]]的[[演員|女演員]]、[[模特兒|模特兒]]。

== 生平 ==
1976年在東京都的[[墨田區|墨田區]]出生，卻在[[練馬區|練馬區]]接受教育及成長，並進入[[東橫學園女子短期大學|東橫學園女子短期大學]]就讀。大學畢業後，曾短暫當過一名普通的[[辦公室女職員|女性上班族]]，但不到半年就辭職，因為自小就有當一名模特兒的夢想，於是就進入{{link-ja|FMG|エフ・エム・ジー}}經紀公司。在1999年出席《東洋泳衣夏季運動女孩》、《朝日啤酒形象女孩》等活動後正式出道，在當時即為甚具人氣的[[治癒系|療癒系]]女偶像。

2006年11月22日跟大他14歲的時裝設計師《松本與》結婚
<ref name="本名趙秀惠結婚日期">{{cite web |url= http://zyouhot.com/archives/1427 |title= 井川遥が結婚して出産？旦那のブランドは？ |author=|date=2014-06-09 |website= |publisher=Zyouhot |access-date=2017-08-25 |quote=}}</ref>，公公為日本知名科技業[[先鋒_(公司)|先鋒公司]]的社長《松本冠也》，2009年6月生下第一個女兒，2012年7月再生下一個兒子。<ref name="本名趙秀惠育有兩子">{{cite web |url= http://healtycoffee.goodlifef.info/%E3%83%80%E3%82%A4%E3%82%A8%E3%83%83%E3%83%88/%E4%BA%95%E5%B7%9D%E9%81%A5%E3%81%AE%E3%83%80%E3%82%A4%E3%82%A8%E3%83%83%E3%83%88%E3%82%84%E7%BE%8E%E5%AE%B9%E6%B3%95%EF%BC%9F%E3%82%AB%E3%83%83%E3%83%97%E3%82%84%E3%82%B9%E3%83%AA%E3%83%BC%E3%82%B5/ |title= 井川遥のダイエットや美容法？カップやスリーサイズ、身長体重は？ |author=|date=2016-05-04 |website= |publisher= |access-date=2017-08-25 |quote=}}</ref>

結婚後，改變原本的偶像路線，轉變成實力派演員，以演技繼續活躍日本[[藝能界|藝能界]]，在不少部日本連續劇、電影中擔任主要演員。
