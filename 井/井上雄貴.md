== 人物 ==
2013年在就讀東京廣播學院廣播聲優科時，參加「[[少年好萊塢|ZEN THE HOLLYWOOD]]」的甄選並合格<ref>{{Cite web |url =http://ddnavi.com/news/165742/a/|title=少年ハリウッド候補生 「秋チーム公開オーディション＆春チーム楽曲披露イベント」開催！|accessdate=2016-08-05}}</ref><ref>{{Cite web |url=http://shingakunet.com/gakko/SC002114/campusLife/9000780396/ |title=専門学校東京アナウンス学院／在校生のキャンパスライフレポート |accessdate=2016-08-05 |deadurl=no |archiveurl=https://archive.fo/20150126175456/http://shingakunet.com/gakko/SC002114/campusLife/9000780396/ |archivedate=2015-01-26 }}</ref>。於2015年6月27日舉辦的活動中宣布從「ZEN THE HOLLYWOOD」畢業<ref>{{Cite web |url = https://www.youtube.com/watch?v=1ey9z0DpPdk  |title= 【ZEN THE HOLLYWOOD】2015.6.27　夏のぜんハリ アチチチセーラーPLAY |accessdate=2016-08-05 }}</ref>。

2016年於次世代男性聲優甄選活動「{{lang|ja|ツキプロ}} Music Grand Prix 2016」合格。並成為「{{lang|ja|ツキクラ}}」的一員<ref>{{Cite web |url = http://www.animatetimes.com/news/details.php?id=1457595941  |title= 2.5次元芸能事務所、ツキノ芸能プロダクションの新人発掘プロジェクト「ツキクラ」メンバー13名が決定！　『ツキウタ。』メンバーからのコメントも到着 |accessdate=2016-08-05 }}</ref>。

[[Mausu_Promotion|Mausu Promotion]]附屬演員養成所30期生<ref name="profile"/>。

2017年4月加入Mausu Promotion<ref name="profile2"/>。
