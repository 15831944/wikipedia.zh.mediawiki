'''黑釉葉紋碗'''

== 年代 ==
[[南宋|南宋]]時期，約西元12－13世紀，生產於位在[[江西省|江西省]][[吉安市|吉安市]][[永和鎮|永和鎮]]的[[吉州窯|吉州窯]]。

== 外觀 ==
碗壁多斜敞呈[[斗笠|斗笠]]式，圈足小，底淺，底心有的凸起，有的平。外壁多烏亮，有些微泛褐色斑。器內黑釉間飾一大片黃色帶點橘紅色的木葉紋，[[葉子|葉子]]躺在盞壁和盞心上。[[黑釉|黑釉]]葉紋，又稱天目木葉，為吉州窯獨特的裝飾手法，題材上有別於過去的陳規舊律，巧妙地運用[[大自然|大自然]]生長的狀態，放進[[陶瓷|陶瓷]]的圖樣。而此種裝飾所選用樹葉多為葉形肥大的桑葉，因為含水量適中，所以容易留下清晰的[[紋理|紋理]]。

== 背景 ==
[[宋朝|宋朝]]的社會上盛行[[飲茶|飲茶]]和[[鬥茶|鬥茶]]風尚，而黑釉瓷與其併行而來，為中國陶瓷史添上一道特別又崇尚[[自然|自然]]與[[禪|禪]]意的風采。講究飲茶藝術的文士們喜愛用黑色茶碗來襯托白色的[[茶湯|茶湯]]，而宋代人在[[喝茶|喝茶]]的時候，習慣先將茶餅輾成細小的茶末，放在茶碗中並注入熱水，經過攪拌後飲用；茶湯表面因此浮起一層白色泡沫，茶碗本身的黑釉就能襯托它，而鬥茶要看茶湯經攪拌後的呈色和茶粉泡開後的浮沫狀況。還有另一方面，是因為[[佛教|佛教]]與茶文化的相互影響。[[吉州窯|吉州窯]]在唐宋時期[[中國|中國]][[禪宗|禪宗]]源頭的[[贛南|贛南]]地區，位於吉州窯所在地永和鎮郊外的青原山，不僅供給吉州窯制瓷原料，山上的多處[[佛寺|佛寺]]，也會向其地購買瓷器，生活相當貼近吉州窯。

== 製作方法 ==
匠者先在器上施淡[[黃釉|黃釉]]，而後貼上處理過的[[樹葉|樹葉]]，再施一層富含鐵份的黑釉，入[[窯|窯]]燒成。其實就是在經高溫氧化下燒造，使樹葉所殘留的灰質成份在釉面上留下的痕跡而形成。它真實完整的留下了樹葉的原貌，如[[化石|化石]]般經久不損<ref>{{cite web| url = http://antiquities.npm.gov.tw/Utensils_Page.aspx?ItemId=309875 |title = 南宋 吉州窯 黑釉木葉紋茶盞 |website = 器物典藏資料檢索系統 |publisher = 國立故宮博物院 |accessdate = 2017-11-10 |archiveurl = https://web.archive.org/web/20171110095736/http://antiquities.npm.gov.tw/Utensils_Page.aspx?ItemId=309875 |archivedate = 2017-11-10 |deadurl =no |language = zh-tw}}</ref><ref>{{cite web |url = http://catalog.digitalarchives.tw/item/00/0c/c4/b3.html |title = 黑釉枯葉盞 |website = 典藏台灣 |publisher = 中央研究院數位文化中心 |accessdate = 2017-11-10 |archiveurl = https://web.archive.org/web/20171110095941/http://catalog.digitalarchives.tw/item/00/0c/c4/b3.html |archivedate = 2017-11-10 |deadurl = no |language = zh-tw}}</ref>。

== 失傳與後續 ==
然而，因為後來戰亂緣故，吉州窯隨著宋代的衰弱逐漸廢滅，至[[明代|明代]]中期斷燒，木葉天目盞的燒造技術也隨之失傳。不過在[[清代|清代]]已被[[皇室|皇室]]列為世代珍藏之物。然後，經過數年的摸索與實踐，現已基本掌握了木葉碗的工藝過程。
