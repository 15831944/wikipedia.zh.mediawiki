'''黑川站'''（{{jpn|j=黒川駅|hg=くろかわえき|rm=Kurokawa eki|f=y}}）是一個位於[[神奈川縣|神奈川縣]][[川崎市|川崎市]][[麻生區|麻生區]]{{link-ja|黑川 (川崎市)#黑川東地區·黑川站前|黒川 (川崎市)#黒川東地区・黒川駅前|南黑川}}，屬於[[小田急電鐵|小田急電鐵]][[多摩線|多摩線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''OT 03'''。

== 歷史 ==
* 1974年（[[昭和|昭和]]49年）6月1日 - 多摩線開通同時開業。為各站停車停靠站。
* 2004年（[[平成|平成]]16年）12月11日 - 新增區間準急班次，為其停靠站。
* 2006年（平成18年）3月 - 改建工程完工。

=== 站名由來 ===
以車站所在地地名「'''黑川'''」命名。

== 車站構造 ==
本站是[[側式月台|側式月台]]2面2線[[地面車站|地面車站]]，為[[跨站式站房|跨站式站房]]。

2006年1月31日，隨著站內整修，月台屋頂鋪上太陽能發電板做為站內自動售票機與自動驗票閘門的電力來源<ref>{{Cite web|date=|url=http://funtoshare.env.go.jp/archive/movement/tm6/2006/02/060214a.html|title=小田急多摩線5駅で太陽光発電システムを導入|publisher=|accessdate=2015-08-21}}</ref>。2010年度設置列車資訊顯示器<ref>{{cite web|url=http://www.odakyu.jp/program/info/data.info/5353_6545114_.pdf|title=小田急電鉄 2010年度の鉄道事業設備投資計画 ②.行先案内表示装置の新設|accessdate=2012-05-10|format=pdf|deadlinkdate=2015-04-17|deadurl=yes|archiveurl=https://web.archive.org/web/20100602055852/http://www.odakyu.jp/program/info/data.info/5353_6545114_.pdf|archivedate=2010-06-02}}</ref>。

2012年度，因應東北澤～和泉多摩川間複複線化工程完工後，各站停車將改為10輛編組，本站月台往新百合丘方向實施延伸工程<ref>{{cite web |url=http://www.odakyu.jp/program/info/data.info/7093_4803266_.pdf |publisher=小田急電鉄|accessdate=2012-05-13 |title=2012年度の鉄道事業設備投資計画 輸送サービスの向上に291億円|format=pdf}}</ref><ref>「各駅停車の10両化に向けた工事を進めます！」ODAKYU VOICE Station 2012年度鉄道事業設備投資計画 2012年5月21日発行</ref>。

== 車站周邊 ==
車站周邊有小規模集合住宅、住宅、兒童公園等，以及川崎市開發的工業園區{{link-ja|川崎micon city|かわさきマイコンシティ}}<ref>{{cite web|url=http://www.kawasaki-net.ne.jp/micomcity/index.html|title=かわさきマイコンシティ|publisher=川崎市|accessdate=2012-06-06|deadlinkdate=2015-04-17}}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>。幾乎沒有商業設施。北口往唐木田方向的鶴川街道下坡旁有小型商店。鶴川街道對向的部分黑川地區因被指定為{{link-ja|市街化調整區域|市街化調整区域}}，免於開發，保留較完整的自然環境。是川崎市內稀少的[[螢火蟲|螢火蟲]]與{{link-ja|斑北鰍|ホトケドジョウ}}棲息地。

* 川崎市黑川青少年野外活動中心
* {{link-ja|劇團民藝|劇団民藝}}
* {{link-ja|三澤川 (多摩川水系)|三沢川 (多摩川水系)|三澤川}} 
* {{link-ja|鶴川街道|鶴川街道}}
* {{link-ja|神奈川縣道·東京都道137號上麻生連光寺線|神奈川県道・東京都道137号上麻生連光寺線|尻手黑川道路}}
* 汁守神社
* Farmers Market Ceresamos（ファーマーズマーケット セレサモス） - {{link-ja|CERESA川崎農業協同組合|セレサ川崎農業協同組合}}（JA CERESA川崎）直營的大規模農產品直銷所
* [[明治大學|明治大學]]黑川農場<ref>{{cite web|url=http://www.meiji.ac.jp/agri/kurokawa/access/index.html |accessdate=2012-05-10|publisher = 明治大学|title = 明治大学黒川農場交通アクセス}}</ref>
* 神奈川縣企業廳{{link-ja|柿生發電所|柿生発電所}}
* {{link-ja|電源開發|電源開発}}（J-POWER）西東京[[變電所|變電所]] 
* [[東京電力|東京電力]]黑川變電所
* [[若葉台站|若葉台站]]（[[京王相模原線|京王相模原線]]） - 步行約10分。

=== 巴士路線 ===
最近的巴士站是鶴川街道旁的「'''黑川'''」巴士站。可搭乘以下由{{link-ja|小田急巴士|小田急バス}}與{{link-ja|神奈川中央交通|神奈川中央交通}}運行的路線。通勤時間於站前有往micon city的專用巴士。

'''鶴川、柿生方向'''（鶴川街道、黑川站側）
* 神奈中 鶴21、鶴22：往[[鶴川站|鶴川站]]
* 小田急 柿24：往[[柿生站|柿生站]]北口
* 神奈中 柿26：經柿生站北口往[[市尾站_(神奈川縣)|市尾站]]（假日1班）
* 神奈中 柿27：往柿生站北口（假日1班）

'''稻城、調布方向'''（鶴川街道、道路對側）
* 小田急 若11：往若葉台站
* 神奈中 鶴21：往若葉台站
* 神奈中 鶴22：經若葉台站、[[矢野口站|矢野口站]]往[[調布站|調布站]]南口（假日1班）
* 小田急 柿24：往[[稻城站|稻城站]]（不經過若葉台站）／往調布站南口（不經過若葉台、站矢野口站。假日1班）／經[[春日野站|春日野站]]入口往若葉台站（假日1班）
