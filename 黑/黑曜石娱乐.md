'''黑曜石娱乐'''（{{lang-en|Obsidian Entertainment}}）是[[Interplay|Interplay]]旗下的[[黑岛工作室|黑岛工作室]]于2003年解散后，吸收其部分成員而成立的一家[[美国|美国]]电子游戏开发公司。黑曜石目前主要爲[[PC|PC]]與家用[[電子遊戲機|主機]]平臺開發遊戲。

==公司成員==
黑曜石娱乐现由创始人共同管理：{{tsl|en|Feargus Urquhart|Feargus Urquhart}}、Chris Parker、{{tsl|en|Darren Monahan|Darren Monahan}}、Chris Avellone和Chris Jones。2005年7月，黑曜石聘请了曾开发过《[[冰风谷|冰风谷]]》的{{le|乔什·索亚|Josh Sawyer}}。2011年10月，[[輻射系列|輻射系列]]的創始人之一{{le|蒂姆·坎|Timothy Cain}}宣佈加盟黑曜石娱乐<ref>http://www.gamasutra.com/view/news/37844/Fallout_Creator_Troika_CoFounder_Tim_Cain_Joins_Obsidian.php</ref>。

==公司簡史==
2006年3月23日，黑曜石娱乐與[[世嘉公司|世嘉公司]]宣佈雙方將共同開發一款新的角色扮演遊戲<ref>{{cite web |url=http://findarticles.com/p/articles/mi_m0EIN/is_/ai_n26804618?tag=artBody;col1 |title=SEGA Partners with Obsidian Entertainment to Create Original Content for Next Generation Consoles |accessdate=2008-11-01 |date=2006-03-23 |work=BusinessWire |archiveurl=https://archive.is/20120712032145/http://findarticles.com/p/articles/mi_m0EIN/is_2006_March_23/ai_n26804618/ |archivedate=2012-07-12 |deadurl=yes }}</ref>；遊戲名為《'''[[阿尔法协议|阿尔法协议]]'''》，主題圍繞現代間諜戰爭展開。該遊戲于2010年6月1日正式發行<ref>{{cite web |url=http://www.sega.com/news/?n=1851 |title=SEGA And Obsidian Set To Redefine The RPG Genre With Alpha Protocol |work=SEGA News |publisher=SEGA Corporation |date=2008-03-13 |accessdate=2008-11-01}}</ref>。

2006年12月13日，[[世嘉公司|世嘉公司]]宣佈黑曜石娱乐將為其開發一款改編于'''[[異形|異形]]'''電影系列的角色扮演遊戲<ref name="Aliens">{{cite web |url=http://www.sega.com/news/?n=1962 |title=SEGA Signs Obsidian Entertainment To Develop Alien Title For Next-generation Systems |work=SEGA News |publisher=SEGA Corporation |date=2006-12-13 |accessdate=2008-11-01}}</ref>。這款遊戲的開發隨後被取消<ref>{{cite web |url=http://www.giantbomb.com/news/obsidians-aliens-rpg-officially-finished/1492/ |title=Obsidian's Aliens RPG Officially Finished |accessdate=2009-06-26 |date=2009-06-26 |work=Giant Bomb}}</ref>。另一款正在開發的遊戲'''Seven Dwarves'''在這一時期被取消<ref>{{cite web |url=http://www.ianameling.com/resume.pdf |title=Former Obsidian artist Ian Ameling's resume |deadurl=yes |archiveurl=https://web.archive.org/web/20111007073358/http://www.ianameling.com/resume.pdf |archivedate=2011-10-07 }}</ref><ref>{{cite web |url=http://www.warcry.com/articles/view/interviews/devprofiles/4784-Neverwinter-Nights-2-Dev-Profile-Q-A-Kevin-Saunders-NWN2-Lead-Designer |title=Dev Profile Q&A: Kevin Saunders, Lead Designer on NWN2: MotB |work=Warcry Network |publisher=Themis Media |accessdate=2007-05-30 |quote=I was the lead designer on project "New Jersey," which was never announced, and then helped with finishing up NWN2 over its last six months or so. |author=Massey, Dana |date=2007-06-29 |archive-url=https://web.archive.org/web/20081207030250/http://www.warcry.com/articles/view/interviews/devprofiles/4784-Neverwinter-Nights-2-Dev-Profile-Q-A-Kevin-Saunders-NWN2-Lead-Designer |archive-date=2008-12-07 |dead-url=yes }}</ref>。

2009年4月20日，[[贝塞斯达软件公司|贝塞斯达软件公司]]宣佈其旗下'''[[辐射系列|輻射]]'''系列的新作品《'''[[輻射：新維加斯|輻射：新維加斯]]'''》將交付黑曜石開發。遊戲後于2010年10月發行<ref name= fnvveil>{{cite web |url=http://www.eurogamer.net/articles/fallout-new-vegas-unveiled |title=Fallout: New Vegas unveiled |work=Ellie Gibson |publisher=Eurogamer |date=2009-04-20 |accessdate=2009-04-20}}</ref>。

2010年2月11日，[[Red_Eagle_Games|Red Eagle Games]]與黑曜石宣佈雙方將共同開發一款基於科幻小説系列《'''[[時間之輪|時間之輪]]'''》的角色扮演遊戲<ref>{{cite web |url=http://www.dragonmount.com/forums/blog/2/entry-294-red-eagle-games-partners-with-obsidian/ |title=Red Eagle Games partners with Obsidian |accessdate=2010-02-13 |date=2010-02-11 |work=Dragonmount |deadurl=yes |archiveurl=https://web.archive.org/web/20110719102046/http://www.dragonmount.com/forums/blog/2/entry-294-red-eagle-games-partners-with-obsidian/ |archivedate=2011年7月19日 |df= }}</ref>。2010年度[[E3|E3]]遊戲展上的消息<ref>{{cite news|url=http://www.gamasutra.com/view/news/28977/E3_Square_Enix_Owns_Dungeon_Siege_IP_Moves_Into_Western_RPG_Market.php/|title=GamaSutra article regarding Square Enix announcing Dungeon Siege 3|}}</ref>指出黑曜石將為[[史克威尔艾尼克斯|史克威尔艾尼克斯]]開發《'''[[地牢圍攻3|地牢圍攻3]]'''》。遊戲後于2011年6月21日發行。

2011年12月1日，GameInformer報道黑曜石娱乐正在開發一款改編自動畫[[南方公園|南方公園]]的[[角色扮演遊戲|角色扮演遊戲]]；名稱定爲《[[南方公園：真理之杖|南方公園：真理之杖]]》。2012年3月14日，媒體報道黑曜石取消了一款次世代遊戲的開發，並從該項目開發組與南方公園開發組中裁員共約20至30人<ref>{{cite web|url=http://gamebanshee.com/news/107270-obsidian-entertainment-layoffs-project-cancelled.html |title=Obsidian Entertainment Layoffs, Project North Carolina Cancelled |publisher=Gamebanshee.com |date= |accessdate=2012-05-26}}</ref>。

2018年11月11日，[[微軟|微軟]]宣佈收購黑曜石娛樂<ref>{{Cite news|url=https://www.polygon.com/2018/11/10/17959500/obsidian-xbox-acquired-x018-new-vegas-pillars-eternity|title=Microsoft acquires Obsidian Entertainment|work=Polygon|access-date=2018-11-10}}</ref><ref name="GuardianObsidian">{{cite web|url=https://www.theguardian.com/games/2018/nov/10/microsoft-buys-two-new-video-game-studios|title=Microsoft buys two more video game studios|author=MacDonald, Keza|publisher=The Guardian|date=10 November 2018}}</ref><ref name="VergeNov18">{{cite web|url=https://www.theverge.com/2018/11/10/18082922/microsoft-xbox-inxile-obsidian-entertainment-studio-acquisition|title=Microsoft’s Xbox exclusives push continues with new studio acquisitions|author=Warren, Tom|date=10 November 2018}}</ref>。
