== 正教會 ==
{{main|{{link-en|黑山正教會|Eastern Orthodoxy in Montenegro}}}}
[[File:Monasterio_de_Ostrog,_Montenegro,_2014-04-14,_DD_14.JPG|thumb]]]]
[[正教會|正教會]]是黑山的主要宗教。黑山的東正教信徒主要是[[黑山人|黑山人]]和[[塞爾維亞人|塞爾維亞人]]。塞爾維亞人主要為[[塞爾維亞正教會|塞爾維亞正教會]]的信徒，黑山人則分屬塞爾維亞正教會和
{{link-en|黑山正教會|Eastern Orthodoxy in Montenegro}}的信徒。

== 伊斯蘭教 ==
{{main|{{link-en|黑山伊斯蘭教|Islam in Montenegro}}}}
[[File:Pljevlja_City-center_Mosque.JPG|thumb]]{{link-en|侯賽因-帕薩清真寺|Husein-paša's Mosque}}]]
穆斯林是黑山最大的少數宗教社群。黑山共有118,477名穆斯林，占總人口的19.11％。<ref name="monstat 2011"></ref>


黑山的穆斯林分為兩大類，此外還有其他社群：
*[[斯拉夫人|斯拉夫]]穆斯林
**[[波士尼亞克人|波士尼亞克族穆斯林]]（[[波士尼亞語|波士尼亞語]]使用者）
**[[黑山人|黑山族穆斯林]]（[[黑山語|黑山語]]使用者）
**[[穆斯林族|穆斯林族]]
*[[阿爾巴尼亞人|阿爾巴尼亞族穆斯林]]

伊斯蘭教是黑山東北部城市的主要宗教，是[[桑扎克|桑扎克]]區域的一部分，為阿爾巴尼亞人佔多數的地區'。伊斯蘭教是[[羅扎伊|羅扎伊]]，[[普拉夫|普拉夫]]，[[古西涅|古西涅]]，[[烏爾齊尼|烏爾齊尼]]和[[佩特尼察|佩特尼察]]的主要宗教。

== 天主教 ==
{{main|{{link-en|黑山天主教|Catholic Church in Montenegro}}}}
[[File:Marktplatz_Kotor_2014.jpg|thumb]]
天主教信徒主要分布在[[科托爾灣|科托爾灣]]地區，主要為[[克羅埃西亞族|克羅埃西亞族]]，也有一些[[阿爾巴尼亞族|阿爾巴尼亞族]]。[[東儀天主教|東儀天主教]]的信徒超過了2萬人。

== 猶太教 ==
{{main|{{link-en|黑山猶太人歷史|History of the Jews in Montenegro}}}}
2012年2月，黑山總理[[伊戈爾·盧克希奇|伊戈爾·盧克希奇]]與黑山猶太社群簽署了一項協議，正式承認猶太人是黑山少數民族。該協議還將猶太教與天主教會，正教會和伊斯蘭教一樣立為該國的官方宗教。<ref>{{cite web|url=http://www.jewishvirtuallibrary.org/jsource/vjw/Montenegro.html |title=Jewish history of Montenegro |publisher=[[Jewish_Virtual_Library|Jewish Virtual Library]] |accessdate=2012-11-16}}</ref>

== 無神論 ==
大多數黑山國民（98.69％）表明自己有宗教信仰，雖然他們不一定常參與宗教活動。

在2011年的人口普查中，無神論者佔黑山總人口的1.24％，不可知論者佔0.07％。

有宗教信仰的比例在[[科托爾灣|科托爾灣]]地區和首都[[波德戈里察|波德戈里察]]最低。

無神論者所佔比例最高的城市為[[新海爾採格|新海爾採格]]（2.43％），[[科托爾|科托爾]]（2.03％），[[波德戈里察|波德戈里察]]（1.99％）和[[蒂瓦特|蒂瓦特]]（1.7％）。相比之下，[[羅扎伊|羅扎伊]]擁有最少的無神論者比例，只有0.01％。<ref name="monstat 2011"></ref>

然而，在一些城市，超過一半的人口是未表態的。
