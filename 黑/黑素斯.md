'''黑素斯'''，本名'''Jesus Trapero(Jesus Sandoval)'''，出生於[[西班牙|西班牙]][[馬德里|馬德里]]。現為[[臺灣|臺灣]]知名[[Youtuber|Youtuber]]、[[演員|演員]]、[[西班牙|西班牙]]語教師。截至2019.03在[[Youtube|Youtube]]頻道訂閱者為30萬及2656萬次觀看次數。<ref>{{Cite web|title=noldoazul, Youtube|url=https://www.youtube.com/user/noldoazul/about|publisher=noldoazul|website=YouTube}}</ref>其影片作品經常被知名媒體如[[蘋果日報_(台灣)|蘋果日報]]<ref>{{ Cite web|title = 為何老外愛「醜女」？阿兜仔解答千人按讚！|url=https://tw.appledaily.com/new/realtime/20180311/1312692/|}} </ref>、[[東森電視|東森電視]]<ref>{{ Cite web|title =台灣人跟外國人交往 男女標準差很大？|url=https://news.ebc.net.tw/news.php?nid=63950|}}</ref> 、[[自由時報|自由時報]]<ref>{{ Cite web|title =台灣「下班」有潛規則？ 阿兜仔怒罵X…|url=http://news.ltn.com.tw/news/life/breakingnews/1397368|}}</ref>等轉載。大部分的題材源自生活中所觀察到台灣社會現象，常引起各界熱烈討論。

== 個人經歷 ==
黑素斯自小熱愛東方文化，因想更加深入體驗喜愛的文化及全中文語言環境，於2006年飛至台灣唸書。原本僅規劃暫居一段時間，卻於畢業後定居下來至今已有十餘年。常自稱為[[臺灣|臺灣]][[台北|台北]][[台電大樓|台電大樓]]人、天龍國人。因經常被觀眾詢問「什麼時候要回西班牙？」，故而多次在影片中公開回覆「我家就在台灣！不要再叫我回西班牙了」。擁有西班牙馬德里內布里哈大學([https://es.wikipedia.org/wiki/Universidad_Nebrija Universidad Antonio Nebrija])學士學位、西班牙語教學語言碩士(Maestría en Lingüística Aplicada del Español como Lengua Extranjera)學位以及實際教學資歷。

黑素斯的Youtuber生涯於2011年展開，首支百萬觀看次數的影片為2014.6.6上傳的「台灣人的口頭禪」。而電視媒體則於台灣知名談話性節目[[康熙來了|康熙來了]]中首次公開亮相。播放日期2015.06.01，節目主題為[https://www.youtube.com/watch?v=29tG7du-Mwg 康熙來了　他們是網路世界璀璨的一顆星《上》] 。同期節目的來賓中也有知名youtube寵物主題頻道[https://www.youtube.com/channel/UCw2W7GIqJNB-UMUxncnMuiw 黃阿瑪的後宮生活]的飼主志銘一同參與。

於2018.09.19第一次加入[[2分之一強|2分之一強]]電視/網路節目，本期主題為[https://www.youtube.com/watch?v=2G60E7aLp_A&t=117s 羨慕嫉妒恨！外國福利好到讓人想移民！沒錢買房政府送給你？！] 。此節目為新晉的知名談話性節目，內容為外國人在台灣的生活經驗以及相關文化話題主題。

== 風格及特色 ==
最初會開始拍影片上傳Youtube，是為了和學生分享《阿兜仔教你課本上學不到的西班牙文！》。<ref> {{Cite book|title=阿兜仔教你課本上學不到的西班牙文！（001）|url =https://www.youtube.com/watch?v=hNMbHDRV5Is }} </ref>後來進一步加入自己觀察到的台灣社會現象，例如《台灣人對老外的分類》<ref> {{Cite book|title=台灣人對老外的分類|url =https://www.youtube.com/watch?v=a2vgdZgfAMY}} </ref>、《台灣人怕外國人》<ref> {{Cite book|title=台灣人怕外國人|url =https://www.youtube.com/watch?v=Mwl_poJLnh8}} </ref>、《ㄈㄈ尺 （CCR：Cross Cultural Romance）》<ref> {{Cite book|title=ㄈㄈ尺 （CCR：Cross Cultural Romance）|url =https://www.youtube.com/watch?v=EPbWPGeHQy0}} </ref>等，點出許多常被台灣當地民眾忽略的現象。經由長期經營及多次嘗試，以幽默自嘲且無厘頭的風格逐漸受到歡迎，累積人氣。是台灣Youtuber代表性人物之一。<ref>{{Cite book|title=台灣女生很好把？ 這阿兜仔用他經驗說...|url =http://www.chinatimes.com/realtimenews/20160703001796-260405}} </ref>

除了與好友合製影片並自已寫腳本外，黑素斯時常在影片中穿插小劇場。經常一人分飾多角，例如黑貓Miko、黑素貓、身上非常多配件的阿嬤、穿上台灣知名高中女校制服的小綠綠、虛擬哥哥José…等。藉由多元角色的刻板形象，讓話題重點更容易被觀眾理解。

== Youtuber之路 ==
從2011年12月起與具有電影製作專長的法籍好友Thomas<ref> {{Cite book | author = Thomas B | title=Thomas B|url =https://www.youtube.com/user/ThomasBuffaz|}} </ref>開始拍攝《'''阿兜仔教你課本上學不到的西班牙文！'''》、上傳影片至[[Youtube|Youtube]]。最初以和學生分享有趣的西班牙語為目的，意外吸引許多來自世界各地的觀眾留言發表感想，讓他感受到接收迴響、和觀眾互動的樂趣，因此更熱衷於精進拍攝、編輯影片的技巧。

2012年開始拍攝《'''阿兜仔不教美語！'''》系列，固定於每週五晚上上傳影片。改以台灣社會觀察為題，陸續發表了《台灣人跟台灣人講英文》<ref>{{cite news|title=阿兜仔不講英語：會說英語就等於國際化嗎？| url =https://www.youtube.com/watch?time_continue=2&v=v7LPIBAz9K8|}} </ref>、《下班，就是下班！》<ref>{{cite news|title=老外解析「台灣人客套話」 慢慢來＝快點啦！| url =http://www.nextmag.com.tw/realtimenews/news/363416|}} </ref>、《解碼台灣人說話》<ref> {{cite news|title=台灣「下班」有潛規則？ 阿兜仔怒罵X…| url =http://news.ltn.com.tw/news/life/breakingnews/1397368|}} </ref>等影片作品，除了獲得了更廣大的觀眾支持，也引起新聞媒體轉載其內容

他開始和觀眾分享明顯較具爭議性的話題。如曾引起許多觀眾認為有歧視意味的[[玖壹壹|玖壹壹]]《歪國人》MV<ref> {{cite news|title=《歪國人》種族歧視惹議 阿兜仔黑素斯：真的很靠盃啦| url =http://news.ltn.com.tw/news/life/breakingnews/1354584|}} </ref>、跨文化戀愛等。在2016年婚姻平權修法引發各界爭議時，他也發表了《台灣同志婚姻？摩天輪也可以結婚嗎？》<ref>{{cite news|title=台灣人看同性戀表面善良？　阿兜仔：很多歧視很誇張 | url =https://www.ettoday.net/news/20150711/533658.htm|}} </ref>鼓勵觀眾以更開放的態度面對與自己不同的人事物。 

2016年與[[東森電視台|東森電視台]]合作，在頻道播出《黑素斯の熱吵店》談話性網路節目。在具台灣在地特色的熱吵店，以各種不同的文化差異為議題，訪問來自不同國家的來賓。<ref> {{cite news|title=抄襲 Buzzfeed 模式踢鐵板 東森噪咖為新媒體上了一堂課| url =http://punchline.asia/archives/26816|}} </ref>
除此之外，他個人也開始推出新的短片系列《'''挖啥米！'''》「開箱」或「解析」他所喜歡的人事物。同年，他開始邀請其他行業領域的代表，如同為網路紅人的[[蔡阿嘎|蔡阿嘎]]、[[陳孜昊|陳孜昊]]等參與談話性節目，並在[[FACEBOOK|FACEBOOK]] LIVE直播。

2018年6月，他宣布暫停拍攝新片，暫時將分享重心轉移至[[Instagram|Instagram]]，表示希望能給自己多一些時間，思考新的發展方向。<ref> {{Cite web|url = https://www.youtube.com/watch?v=ous4g2adZcA&t=2s|title =我不當Youtuber（I stop making videos）|}} </ref>

2018年10月，開始上傳新節目《COW杯の星期一》，12月上傳闊別六個月以來的第一支《阿兜仔不教美語！》節目影片。<ref> {{Cite web|url = https://www.youtube.com/watch?v=qaQl9jXKvO4&t=3s|title =怎麼回答我老闆的問題? How do you answer her question?|}} </ref>
並於2018/12/31<ref> {{Cite web|url = https://www.youtube.com/watch?v=Rg88qMoE0nM|title =電動滑板改變我人生 this electric skateboard change my life 2018 rewind|}} </ref>、2019/1/1<ref> {{Cite web|url = https://www.youtube.com/watch?v=jtoMq3dgcIg|title =2019年最大的秘密 2019 BIG SECRET|}} </ref>連續公開二支影片，談2018的心路歷程，以及2019年的計劃。在影片中提到目前正在籌備個人品牌，於2019年春節後，陸續啟動並在Instagram上揭露相關的初步企劃。
