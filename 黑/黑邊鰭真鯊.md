'''黑邊鰭真鯊'''（[[學名|學名]]：''{{lang|la|Carcharhinus limbatus}}''），又名'''黑邊鰭白眼鮫'''、'''黑梢真鯊'''，是一種體型較大的[[鯊魚|鯊魚]]，是[[熱帶|熱帶]]及[[溫帶|溫帶]][[海洋|海洋]]的[[大陸棚|大陸棚]]及島基台的住民。

黑邊鰭真鯊是一種強壯的鯊魚，身體呈[[灰色|灰色]]，一般都有黑邊的鰭。牠們有著長、窄及尖銳的前端，長鰓裂，一條大的主[[背鰭|背鰭]]及次背鰭。

== 分類 ==
黑邊鰭真鯊最先被分類為[[錐齒鯊屬|錐齒鯊屬]]（學名''Carcharias''），學名是「'' Carcharias (Prionodon) limbatus''。其他的名稱亦有「''Carcharias (Prionodon) pleurotaenia''」、「''Carcharias microps''」、「''Carcharias (Prionodon) muelleri''」、「''Carcharias maculipinna''」、「''Carcharias ehrenbergi''」、「''Carcharias aethlorus''」、「''Gymnorrhinus abbreviatus''」、「''Carcharias phorcys''」及「''Carcharhinus natator''」。而被接受的科學分類學名則是「''Carcharhinus limbatus''」，當中「limbatus」就是指牠的鰭上的黑邊。

== 行為及攝食 ==
與牠的近親[[短鰭真鯊|短鰭真鯊]]（學名''C. brevipinna''）相似，黑邊鰭真鯊是一種游速很快的[[鯊魚|鯊魚]]，不單能躍上水面，更能在期間多次轉體才回到水中。牠們不帶有攻擊性，且不會攻擊[[人類|人類]]。

有一些證據顯示牠們會分開族群，雄性及沒有懷孕的雌性會聚集一群，而懷孕的雌性及幼鯊會另外聚集成一群。

黑邊鰭真鯊主要是以[[硬骨魚|硬骨魚]]為食糧，如[[沙丁魚|沙丁魚]]、[[鯡魚|鯡魚]]、[[鯔魚|鯔魚]]、[[鰺科|鰺科]]及[[馬鮫|馬鮫]]，甚至會吃其他幼小的[[鯊魚|鯊魚]]包括[[灰真鯊|灰真鯊]]，及一些[[頭足綱|頭足綱]]與[[甲殼亞門|甲殼亞門]]動物。

黑邊鰭真鯊是[[胎生|胎生]]的，並在卵黃囊的胎盤內每胎有著1-10隻幼鯊（平均為4-7隻）。[[妊娠期|妊娠期]]相信為10-12個月，而相信雌鯊每年會生產一次。

== 對人類的重要性 ==
黑邊鰭真鯊的身體可以供[[人類|人類]]吃用，包括新鮮的、乾的或鹽腌的。牠的皮可以製成皮革，而[[肝臟|肝臟]]則可獲得肝油。牠亦會被釣魚者作為遊釣魚類。普遍來說鯊魚因為食性，皆不會主動攻擊人，黑邊鰭白眼鮫因平時獵食對象為小型魚類，並不會攻擊人。
