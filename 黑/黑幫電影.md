'''黑幫電影'''（{{lang-en|'''Mob films'''<ref>[http://www.filmsite.org/crimefilms.html Crime and Gangster Films<!-- Bot generated title -->]</ref>或'''gangster films'''}}），或作'''黑道電影'''、'''幫派電影'''，是與[[黑社會|組織犯罪]]有關的[[犯罪電影|犯罪電影]]的次類型。起源于19世紀中葉以後世界主要[[資本主義|資本主義]][[工業國家|工業國家]]社會急遽貧富不均，衍生出社會寫實[[黑幫|黑幫]]的爭鬥故事，通常特別是指[[黑手黨|黑手黨]]。在早期的黑幫電影中，相當大的程度與[[黑色電影|黑色電影]]有著相同之處。

== 歷史 ==
美國電影《[[黑手_(1906年電影)|黑手]]》（1906年）被認為是最早期尚存的黑幫電影。<ref>{{Cite web |url = http://flavorwire.com/390383/a-beginners-guide-to-gangster-movies |title = A Beginner's Guide to Gangster Movies |author = Jason Bailey |date = 2013-05-09 |publisher = Flavorwire |language = en}}</ref>1912年，[[大衛·格里菲斯|大衛·格里菲斯]]導演的短片《[[佩吉巷的火槍手|佩吉巷的火槍手]]》講述[[紐約市|紐約市]]街頭上的罪案（然而，卻在[[新澤西州|新澤西州]]的{{link-en|利堡 (新澤西州)|Fort Lee, New Jersey|利堡}}拍攝），有傳聞指出包括真實的黑幫份子作為臨時演員。評論家也舉出《[[新生_(1915年電影)|新生]]》（1915年）為早期犯罪電影。

==美國的黑幫電影==
[[File:Edward_g_robinson.jpg|thumb]]》（1931年）]]
[[File:Humphrey_Bogart_in_The_Petrified_Forest_film_trailer.jpg|thumb]]》（1936年）預告片]]
早期電影背景多集中在美國四大都會的[[江湖|江湖]]爭鬥：[[紐約|紐約]]、[[邁阿密|邁阿密]]、[[芝加哥|芝加哥]]、[[洛杉磯|洛杉磯]]。最膾炙人口黑幫電影，當屬1970年代奪得美國[[奧斯卡金像獎|奧斯卡金像獎]]最佳影片的《[[教父_(電影)|教父]]》（''The Godfather''）一至三集。
*1970年代代起，[[法蘭西斯·柯波拉|法蘭西斯·柯波拉]]所執導的《[[教父_(電影)|教父]]》，開啟以描述[[義大利裔美國人|義大利裔美國人]]於[[紐約|紐約]]建立地盤的[[五大家族|五大家族]]及[[黑手黨|黑手黨]]故事，廣受好評。
*近年則又加入[[哥倫比亞|哥倫比亞]]及[[邁阿密|邁阿密]][[西班牙裔|西班牙裔]]美國人幫派爭奪素材。
*另有[[日本|日本]]移民（[[極道|極道]]）、[[華人|華人]]移民（[[華青幫|華青幫]]）在美國幫派火併，搬上銀幕。
