《'''黑街法則'''》（{{lang-en|'''''Brooklyn Rules'''''}}）是一部2007年的美國[[犯罪電影|犯罪電影]]，由{{link-en|麥可·寇倫特|Michael Corrente}}執導，[[提倫斯·維達|提倫斯·維達]]編劇。主演是[[艾力·寶雲|艾力·寶雲]]、[[史考特·肯恩|史考特·肯恩]]、[[小弗雷迪·普林斯|小弗雷迪·普林斯]]、{{link-en|傑瑞·費拉拉|Jerry Ferrara}}和[[米娜·蘇瓦麗|米娜·蘇瓦麗]]。<ref>{{Cite web |url = https://variety.com/2007/film/reviews/brooklyn-rules-1200559298/ |title = Brooklyn Rules |author = John Anderson |date = 2007-05-17 |publisher = Variety |language = en}}</ref>
