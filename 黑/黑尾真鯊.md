'''黑尾真鲨'''，又稱'''黑印白眼鮫'''，为[[軟骨魚綱|軟骨魚綱]][[板鰓亞綱|板鰓亞綱]][[真鯊目|真鯊目]][[真鲨科|真鲨科]]的其中一[[種|種]]。

== 分布 ==

本魚分布魚[[印度|印度]][[太平洋|太平洋]]區熱帶及亞熱帶海域。

== 深度 ==

水深0至280公尺。

== 特徵 ==

本魚體壯碩，屬中型鯊魚，吻長，吻端寬圓。眼大，前鼻瓣呈短三角形，上頷齒短且具鋸齒緣。胸鰭窄如鉤狀，背鰭兩枚，第二背鰭高大，第一背鰭呈鉤狀。體背灰色，腹部白色，第一背鰭具不規則白色邊緣，尾鰭具寬黑邊緣，故名，第二背鰭、臀鰭、腹鰭尖端黑色。脊椎骨211至221枚，體長可達255公分。

== 生態 ==

本魚棲息於[[大陸棚|大陸棚]]、島嶼邊緣或[[珊瑚礁|珊瑚礁]]區，屬肉食性，主要以[[魚類|魚類]]與[[無脊椎動物|無脊椎動物]]為食。常成群巡游，活動力強，卵胎生，每胎產1至6尾仔魚。

== 經濟利用 ==

食用魚，魚肉可製成沙魚煙，魚肝可製成魚油。
