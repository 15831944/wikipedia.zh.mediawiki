'''黑色棘皮症'''（[[拉丁语|拉丁语]]: '''Acanthosis nigricans'''），又名'''棘皮症'''、'''黑棘皮症'''、'''棘皮病'''等，常伴有[[肥胖|肥胖]]的一種[[皮膚病變|皮膚病變]]。屬副[[腫瘤性疾病|腫瘤性疾病]]、先天性/發育性病症<ref>{{cite web |language=中文 |url=http://www.mydrnet.com/disease/5a960d4481431b5816951e88ce461939e22bcf120756bea78317e834d0f37d04.html |title=黑棘皮症的病因、症狀、診斷、治療方法 |publisher=e生網 |access-date=2011-05-22 |archive-url=https://web.archive.org/web/20111110192643/http://www.mydrnet.com/disease/5a960d4481431b5816951e88ce461939e22bcf120756bea78317e834d0f37d04.html |archive-date=2011-11-10 |dead-url=yes }}</ref>，病發部位多於[[皮膚|皮膚]]、皮下、[[外分泌|外分泌]]、[[內分泌|內分泌]]、[[代謝系統|代謝系統]]。

== 症狀 ==

多發於[[皮膚|皮膚]][[褶皺|褶皺]]處，如[[腋|腋]]下、[[頸|頸]][[頜|頜]]、[[股溝|股溝]]、[[外陰|外陰]]。顧名思義，患處會呈現暗黑、污褐色、灰色、斑塊狀、色素沉著，伴有絨毛狀、乳頭瘤狀、雞冠皮嵴狀的凸起、增生、皮膚損傷，亦可伴[[皮膚贅瘤|皮膚贅瘤]]發。

== 病因 ==

黑色棘皮症成因有很多，包括：

=== 肥胖 ===

這是最常見的病因，患處呈暗黑斑狀、圈狀或帶狀分佈[[皮膚|皮膚]][[褶皺|褶皺]]之間，常被人誤認為“洗不乾淨”。

=== 良性型 ===

此種由[[顯性遺傳|顯性遺傳]]引起，多有[[家族集聚|家族集聚]]情況。注意，此種情況只表現於[[皮膚|皮膚]]異常，與[[內臟|內臟]]無關<ref>{{cite web |language=中文 |url=http://www.huoguan.com/disease/d6_d0/40110/ | title=黑棘皮症 |publisher=火罐網 }}</ref>。

=== [[症候群|症候群]]引發 ===

某些症候群的症狀就有黑色棘皮症：

# [[糖尿病|糖尿病]]：糖尿病引發的黑色棘皮症也很常見，常稱「[[高雄性素血症|高雄性素血症]]/[[胰島素|胰島素]]抗性-黑色棘皮症」（HyperAndrogenemia/Insulin Resistance-Acanthosis Nigricans，簡稱HA/IRAN），常用作糖尿病患特徵之一。
# [[結締組織|結締組織]]病變：如[[硬皮症|硬皮症]]、[[Rud_syndrome|Rud syndrome]]、[[Bloom_syndrome|Bloom syndrome]]、[[Lawrence-Seip_syndrome|Lawrence-Seip syndrome]]、[[Crouzon_syndrome|Crouzon syndrome]]等。
# [[多囊卵巢症候群|多囊卵巢症候群]]：黑色棘皮症為[[多囊卵巢症候群|多囊卵巢症候群]]<ref>{{cite web |language=中文 |url=http://www.flyxg.com/news/77/n-117177.html |title=多囊卵巢綜合症的危害 |access-date=2011-05-22 |archive-url=https://web.archive.org/web/20110606073810/http://www.flyxg.com/news/77/n-117177.html |archive-date=2011-06-06 |dead-url=yes }} ,</ref>臨床表現危害之一，屬於較為複雜的[[婦科病|婦科病]]。

=== 癌症 ===

常有[[胃癌|胃癌]]、[[胃腺癌|胃腺癌]]（gastric adenocarcinoma）、[[支氣管癌|支氣管癌]]（bronchogenic carcinoma）等引發，患部多為[[口腔|口腔]][[黏膜|黏膜]]、[[手掌|手掌]]等。可能與[[TGF-alpha|TGF-alpha]]（transforming growth factor-alpha）濃度上升有關。

===藥物===

* [[類固醇|類固醇]]（包括口服型、注射型）
* [[避孕藥|避孕藥]]
* [[雌性激素|雌性激素]] 如''diethylstilbestrol''（DES）
* [[胰島素|胰島素]]如皮下[[胰島素|胰島素]]注射（subcutaneous insulin injection）
* [[抗生素|抗生素]] 如[[夫西地酸|夫西地酸]]（fusidic acid）
* [[雄性激素|雄性激素]]
* 其他
# [[菸鹼酸|菸鹼酸]]（niacin, nicotinic acid)
# [[三嗪環|三嗪環]]（triazinate，簡稱TZT)
# [[生長激素|生長激素]]（somatotrophin）

=== 內分泌 ===

* [[甲狀腺機能亢進|甲狀腺機能亢進]]（[[甲狀腺激素|甲狀腺激素]]過高）
* 甲狀腺激素不足
* [[肢端肥大癥|肢端肥大癥]]

== 機理病因 ==

上述各種成因引起的患者的[[表皮層|表皮層]][[角質|角質]][[細胞|細胞]]、[[真皮層|真皮層]][[纖維母細胞|纖維母細胞]]增生過度、導致[[棘細胞層|棘細胞層]]增厚，皮膚毛囊[[毛囊角化症|角化]]過度引起。此為患病直接原因。

== 治療 ==

# 一般肥胖引起的黑色棘皮症只需減重<ref>{{cite web|language=中文|url=http://www.120live.com/xinchendaixiexitong/743-120live.html|title=黑色棘皮症|publisher=壹貳零健康知識問答網|deadurl=yes|archiveurl=https://web.archive.org/web/20080516133444/http://www.120live.com/xinchendaixiexitong/743-120live.html|archivedate=2008-05-16}}</ref>後可以自然痊癒。而且無論患者或是已經痊癒的人都須注意血糖的濃度<ref>{{cite web |language=中文 |url=http://cn.24drs.com/Daily/article.asp?x_no=7512&page=1 | title=發生黑棘皮症 須注意血糖控制 |publisher=國際厚生健康園區編輯群 }}</ref>。
# 目前一般認為[[維他命A酸|維他命A酸]]是治療此病的有效藥物<ref>{{cite web |language=中文 |url=http://www.pf110.com/dxjyyza/hjpb/7224.html | title=黑棘皮病怎麼治療 |publisher=北京京城皮膚病醫院 |date=2010年7月7日 }}</ref>，可外用0.1%[[維他命A酸|維他命A酸]]軟膏，酌情加用角質軟化劑。有報導食用[[魚油|魚油]]可改善本病<ref>{{cite web|language=中文|url=http://medicaldb.haoyisheng.com/medicaldb/search/viewAllDisease.do?diseaseId=geydanjsga|title=黑棘皮病|publisher=北京健康在線}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
