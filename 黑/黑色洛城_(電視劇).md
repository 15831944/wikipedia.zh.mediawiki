《'''黑色洛城'''》（{{lang-en|'''''Mob City'''''}}，又名《'''洛城黑幫'''》）是一部美國[[新黑色電影|新黑色]][[犯罪片|犯罪]]電視劇，由[[法蘭·達拉本特|法蘭·達拉本特]]創作，[[特納電視網|TNT]]播放。本劇是基於[[洛杉磯警察局|洛杉磯警察局]]和1940年代[[洛杉磯|洛杉磯]]黑幫的真實描述，記錄在John Buntin的著作《''L.A. Noir: The Struggle for the Soul of America's Most Seductive City''》。<ref>{{Cite web |url = http://ew.com/article/2012/10/17/frank-darabont-la-noir/ |title = Frank Darabont TNT drama picked up, reunites 'Walking Dead' actors |author = James Hibberd |date = 2012-10-17 |publisher = EW.com |language = en}}</ref>本劇於2013年12月4日開播。<ref>{{Cite web |url = http://www.cinemablend.com/television/TNT-TBS-Add-Frank-Darabont-Steven-Spielberg-Diablo-Cody-More-Big-Names-Lineup-55710.html |title = TNT And TBS Add Frank Darabont, Steven Spielberg, Diablo Cody And More Big Names To Lineup |author = Katey Rich |publisher = Cinema Blend |language = en}}</ref>

2014年2月10日，TNT取消了《黑色洛城》。<ref>{{Cite web |url = http://tvbythenumbers.zap2it.com/cable/mob-city-canceled-by-tnt/235976/ |title = 'Mob City' Canceled By TNT |date = 2014-02-10 |publisher = TV by the Numbers |language = en}}</ref>在[[德國|德國]]，本劇於2015年7月2日經由polyband公司發行DVD和自由[[區域限制|區域限制]]的藍光光碟，不過，沒有已知打算在美國發行本劇的家庭錄像。<ref>{{Cite web |url = http://polyband.de/ceemes/?article/show/185689/page=827/layout=poly_dvd_bluray_main_public |title = Mob City DVD |publisher = Polyband |deadurl = yes |archiveurl = https://web.archive.org/web/20150417000121/http://polyband.de/ceemes/?article%2Fshow%2F185689%2Fpage=827%2Flayout%3Dpoly_dvd_bluray_main_public |archivedate = 2015-04-17 }}</ref>
