-{zh-tw:'''黑冠麻鷺''';zh-cn:'''黑冠鳽''';zh-hk:'''黑冠鳽'''}-（[[學名|學名]]：{{lang|la|''Gorsachius melanolophus''}}）又名-{zh-tw:'''黑冠鳽''';zh-cn:'''黑冠麻鹭''';zh-hk:'''黑冠麻鷺'''}-或'''黑冠虎斑鳽'''，有時亦俗稱'''大笨鳥'''<!--限台灣？--><ref>{{cite web |author = 高維奇 |website = 台灣動物新聞網 |publisher = [[中華民國保護動物協會|中華民國保護動物協會]] |title = 我是大笨鳥 不是夜鷹！ |url = http://www.tanews.org.tw/info/12577 |date = 2017-05-27 |accessdate = 2017-11-02 |archiveurl = https://web.archive.org/web/20171102105847/http://www.tanews.org.tw/info/12577 |archivedate = 2017-1102 |deadurl = no |language = zh-hant}}</ref>，中等大小，分佈在[[亞洲|亞洲]]南部、東部和東南亞，在[[印度|印度]]、[[中國大陸|中國大陸]]、[[日本|日本]]、[[台灣|台灣]]和[[菲律賓|菲律賓]]等一帶繁殖。

黑冠麻鷺站立時大約47公分高。棲息於森林裏並常夜間活動，遇警時會伸長脖子偽裝，在繁殖期入夜後會站在樹枝上向四方大聲起哄。成鳥有黑冠、面孔和脖子為紅棕色，背部為黑暗的紅棕色，腹部有細條紋，翼黑。亞成鳥是灰棕色有黑和白色斑點，並且下部有斑紋。<ref>{{Cite web|url=http://www.zoo.gov.taipei/ct.asp?xItem=53284035&ctNode=22735&mp=104031|title=臺北動物園-黑冠麻鷺育幼中|accessdate=2017-10-18|date=2013-05-24|last=林惠珍|work=www.zoo.gov.taipei|language=zh-TW}}</ref>

因普遍出現在廣闊的地區，黑冠麻鷺在[[IUCN紅色名錄|IUCN紅色名錄]]中被評估為可最低關注的物種。
