'''黑部川第四發電廠'''（{{lang-ja|黒部川第四発電所}}）位於[[富山縣|富山縣]][[黑部市|黑部市]][[黑部川|黑部川]]上游，為隸屬於[[關西電力|關西電力]]的水力[[發電廠|發電廠]]。通常簡稱為'''黑四發電廠'''或是'''黑四'''。自[[黑部水壩|黑部水壩]]引水進行發電。

== 摘要 ==
[[1958年|1958年]]，向[[世界银行|世界銀行]]貸款3700萬美元，並於[[1963年|1963年]]竣工。此外，為了從[[黑部水壩|黑部水壩]]取水而興建的「[[關電隧道|關電隧道]]」，其工程期間與冷水及[[破碎帶|破碎帶]]奮戰的過程，改編成[[小說|小說]]、[[電影|電影]]、[[連續劇|連續劇]]『[[黑部的太陽|黑部的太陽]]』。現在發電程序為遠距離操控。

[[世界银行|世界银行]]自[[1961年|1961年]][[1月15日|1月15日]]開始[[發電|發電]]。最大輸出功率為33萬5000 kW（建設當時為25萬8000 kW）。[[2007年|2007年]]日本一般水力發電廠（水壩類）發電能力排名為日本第四（第一為[[奧只見發電廠|奧只見發電廠]]），使用水量72. 0 m3/s，有效545.5m。

基於環境保護及防止嚴冬期間的[[雪崩|雪崩]]，此發電設施及送電、送水管線皆設置於地下。

== 到達發電廠的途徑 ==
到達發電廠有'''[[黑部水壩|黑部水壩]]路線'''以及'''[[宇奈月溫泉|宇奈月溫泉]]路線'''兩條種方式。無論是哪種都必須經過地下。但是，只限關西電力相關人員使用，原則上，一般人都無法使用。

*黑部水壩路線是從黑部水壩出發經由[[黑部隧道|黑部隧道]]抵達。若是自[[大町市|大町市]]方向出發，全年皆可通行。黑部隧道雖然和[[立山黑部阿爾卑斯山脈路線|立山黑部阿爾卑斯山脈路線]]中的[[關電隧道|關電隧道]]互相連結，但是，與一般對外開放的關電隧道不同，原則上只有關西電力的相關人員能夠通行。

*宇奈月溫泉路線則是利用起始於[[宇奈月站|宇奈月站]]的[[黑部峽谷鐵道|黑部峽谷鐵道]]到終點站[[櫸平站|櫸平站]]，並於櫸平站搭乘[[關西電力黑部專用鐵道|關西電力黑部專用鐵道]]。雖然關西電力黑部專用鐵道終年運行，但是黑部峽谷鐵道冬季停駛，因此並非全年可利用之路線。關西電力黑部專用鐵道原則上也是相關人士專用。

此外，若報名並抽中關西電力主辦、[[富山縣|富山縣]]協辦的「[[黑部路線|黑部路線]]參觀會」，則可以經由黑部隧道到黑部川第四發電廠參觀，再搭乘關西電力黑部專用鐵道至櫸平站；或是經由上述的相反方向路線。「黑部路線參觀會」是一般人能夠參觀此發電廠的唯一機會。

== 建築資材的運輸 ==
建築資材由[[北陸本線|北陸本線]]（現 [[愛之風富山鐵道線|愛之風富山鐵道線]]）[[黑部站|黑部站]]，透過[[富山地方鐵道本線|富山地方鐵道本線]]黑部支線（舊 黑部鐵道）搬運。

省線三日市站（現 黑部站）相鄰的舊三日市站及宇奈月站之間，原本有為了開發黑部峽谷而鋪設的黑部鐵道。至舊三日市站的路線於[[1969年|1969年]]8月17日廢止營業。鐵軌移除後的現在，仍然可以在富山地方鐵道黑部站或是黑部站的月台見到歷史的痕跡，現行路線也有與舊路線拆除後空地並行的區間。

== 登場作品 ==
[[NHK|NHK]]的「[[PROJECT_X_～Challengers～|PROJECT X ～Challengers～]]」節目中，曾介紹此發電廠的建設過程。並且，[[2002年|2002年]]第53屆[[NHK紅白歌唱大賽|NHK紅白歌唱大賽]]中，[[中島美嘉|中島美嘉]]在與此發電廠相鄰的關西電力黑部專用鐵道車站隧道內，以[[現場直播|現場直播]]演唱[[主題曲|主題曲「]][[地上之星|地上之星]]」。由於關西電力黑部專用鐵道車站及黑部川第四發電廠幾乎都是地下構造，因此，訊號和冬季傳送路線一樣，地底傳播20km後，從[[長野縣|長野縣]][[大町市|大町市]][[扇澤|扇澤]]送出訊號，透過衛星轉播。

2010年起，關西電力以相同發電廠為舞台，中島美雪的「糸」為背景音樂，拍攝系列企業廣告。
