'''長腳鷸'''（[[学名|学名]]：{{lang|la|'''''Himantopus himantopus'''''}}），又名'''高蹺鴴'''、'''黑翅长脚鹬'''、'''红腿娘子'''，为[[長腳鷸科|長腳鷸科]][[長腳鷸屬|長腳鷸屬]]下的一个种。

== 分类 ==
黑翅长脚鹬的分类一直有争议，有人认为可以分为五个种，也有人认为这些种有的是亚种。北美的黑翅长脚鹬在眼后有一块白斑，在[[鸟类DNA分类系统|鸟类DNA分类系统]]中，认为是一个种''Himantopus mexicanus''。
[[File:Black-necked_Stilt.jpg|thumb]]

== 生态环境 ==
黑翅长脚鹬常结群活动，喜沿海浅水及淡水沼泽地。可见于湖泊、沼泽、鱼塘、虾池以及河流平缓水面开阔处，栖息于近水的草地农田等处。由于腿较长可在水位较深的池塘种涉水觅食。

== 分布地域 ==
黑翅长脚鹬分布在[[欧洲|欧洲]]东南及南部、[[印度|印度]]、[[中国|中国]]及[[东南亚|东南亚]]、[[北非|北非]]、[[澳大利亚|澳大利亚]]、[[加拉帕格斯群岛|加拉帕格斯群岛]]、[[夏威夷|夏威夷]]、[[加拿大|加拿大]]中部和南部、[[美国|美国]]西部和东南部。在中国[[新疆|新疆]]西部、[[青海|青海]]东部及[[内蒙古|内蒙古]]西北部繁殖，越冬于[[台湾|台湾]]、[[廣東|廣東]]及[[香港|香港]]等地，全境有过境纪录。

== 特徵 ==
黑翅長腳鷸體態修長，體長越35釐米，通體黑白分明，一雙紅腿很容易辨認。黑翅長腳鷸的[[喙|喙]]黑色細長，雄鳥[[換羽|繁殖羽]]的從眼後到頭頂和頸後黑色，背部和翅為黑色，與頸後的黑色不相連，其他部位為白色；雌鳥的繁殖羽在頭和頸後沒有黑色，眼後有灰色斑。亞成體的鳥色塊分布接近成鳥，但顏色較淺，為灰褐色。黑翅長腳鷸的腿和跗趾修長，為紅色，亞成體鳥的腿顏色略淺，顯橘紅色；飛行時長腿拖於尾後，是重要的辨識特徵。虹膜粉紅色。

== 食物 ==
黑翅长脚鹬以[[昆虫|昆虫]]、[[蝌蚪|蝌蚪]]等为食。

== 繁殖与保护 ==
[[File:Himantopus_himantopus_MHNT.jpg|thumb]]
常集群筑巢于水边土堆或稻田的田埂上，与[[燕鸻|燕鸻]]、[[凤头麦鸡|凤头麦鸡]]、[[泽鹬|泽鹬]]等灰群营巢。巢材采用小石子、泥土、芦苇、枯草等。

本物种未列入保护。
