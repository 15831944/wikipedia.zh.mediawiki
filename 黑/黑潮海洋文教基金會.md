'''黑潮海洋文教基金會'''（[[英語|英語]]：Kuroshio Ocean Educational Foundation）是由曾當過漁民的台灣海洋文學作家[[廖鴻基|廖鴻基]]於1998年發起成立，從鯨豚調查記錄工作為開端，進而以「關懷台灣海洋環境、生態與文化」為宗旨，期望促進民眾海洋保育觀念。<ref>[http://www.kuroshio.org.tw/index.php?load=read&id=160 基金會簡介]</ref><br>

該會目前以花蓮市與台灣東部為活動重心，以推動海洋環境教育為主軸，另外也積極參與[[蘇花高|蘇花高]]議題與[[海洋|海洋]]生態保育議題。

== 具體行動與工作 ==

# 海洋生態觀察 - 進行海上觀察與解說，以及東部鯨豚調查紀錄。比較重要的行動為1999年起投入花蓮[[賞鯨|賞鯨]]解說培訓，趁勢帶動了花東賞鯨業的發展。每年夏天另辦理「海上觀察與解說營」，並拍攝生態影像[[紀錄片|紀錄片]]等。<br> 
# 海洋教育推廣推廣講座、攝影展覽、營隊研習、教材設計、刊物繪本 - 除了舉辦海洋主題展覽、營隊與座談外，黑潮自2007年起，出版刊物海洋軟性議題的《海人》與[[繪本|繪本]]。<br> 
# 海洋文史調查  漁業紀實、文化調查 - 紀錄消逝中的漁業文化，包括紀錄並重現傳統漁法與工法。<br> 
# 海岸活動體驗  體驗行旅、海岸寫真、地圖繪製 - 以踏查方式記錄全台灣海洋與海岸環境，2006年起繪製發送自導式海岸地圖。<br> 
# 海岸環境監測海灘廢棄物監測計畫、國際淨灘行動、環境議題 - 2000年引進海廢監測方法，並參加每年國際淨灘活動，長期記錄垃圾種類數量瞭解台灣海岸面臨的潛在問題。<br> 
# 出版品與觀點論述 - 曾在報章與網路上推出「黑潮觀點」專欄、擔任「小地方社區新聞網」觀察員、發行電子報等，並擁有廖鴻基人的相關出版品。
