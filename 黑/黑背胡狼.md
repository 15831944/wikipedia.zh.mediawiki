'''黑背胡狼'''，又叫黑背豺，学名为{{lang|la|''Canis mesomelas''}}，英文名为{{lang|en|''Black-backed Jackal''}}。

== 主要特征 ==
肩高{{convert|40|cm|in}}，体重{{convert|9|-|14|kg}}。因背部为黑色而得名。

== 习性 ==
多栖息于[[非洲|非洲]][[草原|草原]]。黑背豺很少自己捕食，通常在[[狮子|狮子]]旁边等候，当狮子不注意时，黑背豺趁机偷走食物，而狮子却拿它们没办法。妊娠期2月，每胎多达7-8只小豺。
== 分布 ==
分布于[[东非|东非]]和[[南非|南非]]。
