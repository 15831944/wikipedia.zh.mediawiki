'''黑背信天翁'''（[[学名|学名]]：{{lang|la|''Phoebastria immutabilis''}}）为[[信天翁科|信天翁科]][[信天翁屬|信天翁屬]]下的一个种。

== 命名由來 ==
英文名「Laysan Albatross」意指「[[列山島|列山島]]的信天翁」。列山島是[[西北夏威夷群島|西北夏威夷群島]]中的一個島，是黑背信天翁繁殖地之一。

種名「''immutabilis''」有「不會改變」之意，意指成鳥與幼鳥羽色類似，就像不變一樣。

== 分布地區 ==
廣泛分布於北太平洋，繁殖期時遷徙至[[夏威夷群島|東南夏威夷群島]]的[[歐胡島|歐胡島]]、[[考艾島|考艾島]]、[[尼豪島|尼豪島]]、[[勒哈島|勒哈島]]（Lehua Island）、[[考拉島|考拉島]]（Kaʻula Island），以及[[西北夏威夷群島|西北夏威夷群島]]。

== 生態 ==
黑背信天翁採終生交配制<ref>[http://www.ettoday.net/news/20130208/163380.htm 全球最老的鳥　62歲黑背信天翁「智慧」又當媽]，ETToday東森新聞雲</ref>，已知最老的野外的野外繁殖鳥—黑背信天翁「智慧」（Wisdom）在年約66歲時（2016年）仍持續孵蛋<ref>[http://www.cna.com.tw/news/aopl/201612100139-1.aspx 全球已知最老海鳥現蹤 66歲再度當媽]，[[中央社|中央社]]</ref>。
