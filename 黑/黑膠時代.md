{{作品名稱|name=''Vinyl''|hk=黑膠時代|tw=唱片啟示錄|equiv-cn=hk}}是一部美國[[時代劇|時代]]電視劇，由[[米克·傑格|米克·傑格]]、[[馬田·史高西斯|馬田·史高西斯]]、{{link-en|里奇·科恩|Rich Cohen}}和[[泰倫斯·溫特|泰倫斯·溫特]]創作。本劇主演是[[波比·簡拿威|波比·簡拿威]]，飾演一名1970年代的音樂製作人Richie Finestra。本劇於2016年2月14日在[[HBO|HBO]]上首播<ref>{{Cite web |url = http://deadline.com/2015/10/vinyl-girls-togetherness-february-premiere-dates-hbo-1201597306/ |title = 'Vinyl', 'Girls' & 'Togetherness' Get February Premiere Dates On HBO |author = Denise Petski |date = 2015-10-29 |publisher = Deadline.com |language = en}}</ref>，並於2016年4月17日結束。

科恩、傑格、史高西斯和溫特編寫故事，由史高西斯執導首集。第一季由十集所組成。史高西斯曾希望能夠執導本劇更多的集數。由於創作差異，溫特在第一季結束時離開了他的職位，留下職位給{{link-en|斯科特·Z·伯恩斯|Scott Z. Burns}}。<ref>{{Cite web |url = http://variety.com/2016/tv/news/terence-winter-vinyl-quit-leaving-hbo-1201749342/ |title = Terence Winter Out as 'Vinyl' Showrunner Following Creative Differences |author = Elizabeth Wagmeister |date = 2016-04-08 |publisher = Variety |language = en}}</ref><ref>{{Cite web |url = http://deadline.com/2016/04/vinyl-showrunner-terence-winter-exits-hbo-1201734449/ |title = 'Vinyl' Showrunner Terence Winter Exits HBO Series, New Showrunner Named |author = Nellie Andreeva |date = 2016-04-08 |publisher = Deadline |language = en}}</ref>

HBO在首集播放後不久的2016年2月18日宣佈訂續《黑膠時代》第二季。<ref>{{Cite web |url = http://deadline.com/2016/02/vinyl-renewed-for-season-2-by-hbo-1201705325/ |title = 'Vinyl' Renewed For Season 2 By HBO |author = Nellie Andreeva |date = 2016-02-18 |publisher = Deadline.com |language = en}}</ref>可是，在2016年6月22日，HBO扭轉這個決定並取消了本劇。<ref>{{Cite web |url = http://deadline.com/2016/06/vinyl-canceled-hbo-season-2-martin-scorsese-and-mick-jagger-1201777356/ |title = 'Vinyl' Canceled: HBO Scraps Plans For Revamped Season 2 |author = Nellie Andreeva |date = 2016-06-22 |publisher = Deadline.com |language = en}}</ref>HBO節目總監Casey Bloys表示：「它沒有立足點。資源有限，如果僅僅將唱針移動一點，我們就不認為調整是值得花費製作人的時間。」
