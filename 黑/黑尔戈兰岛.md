'''海姑蘭島'''<ref>{{Cite book |date=1972 |isbn=9570509767 |title=地理學論文集 |publisher=[[臺灣商務印書館|臺灣商務印書館]] |author=[[沙學浚|沙學浚]]|page=頁75}}</ref>，亦譯為'''黑尔戈兰岛'''<ref>中国地名委员会  《[[外国地名译名手册|外国地名译名手册]]》ISBN 978-7-100-00798-6</ref>（{{lang-de|'''Helgoland'''}}；[[黑尔戈兰语|黑尔戈兰语]]：'''deät Lun'''）是位于[[北海_(大西洋)|北海]]东部的小型[[群岛|群岛]]，行政上隶属于[[德国|德国]][[石勒苏益格-荷尔斯泰因州|石勒苏益格-荷尔斯泰因州]]。

== 地理 ==
[[File:Map_helgoland_1910.jpg|thumb]]
[[File:Langeanna.jpg|thumb]]

海姑蘭島位于德国海岸线外约70公里处，由两个小岛组成：面积1km²、有人居住的主岛（Hauptinsel），以及位于其東边、面积为0.7 km²的多恩岛（德语:Düne, 黑尔戈兰语: de Halem）。岛屿周围的水域被称为“[[海姑蘭湾|海姑蘭湾]]”（Helgolander Bucht）或“德意志湾”（Deutsche Bucht）。

在1720年被一场风暴冲为两截之前，主岛和多恩岛是联为一体的。主岛在地貌上可分为下岛（Unterland、deät Deelerlun）和上岛（Oberland、deät Boperlun），下岛位于岛的东南方，接近[[海平面|海平面]]，上岛则为海拔约50米的高台。1947年4月，由于英国皇家海军的爆破，在岛的东边形成了被称为“中区”（Mittelland）的平地。在岛的西北端有一座孤立的、高约47米的岩柱，被称为“高安娜”（Lange Anna）。主岛的南端、东南端有沙滩，西部和北部则为悬崖，直降至海平面以下约50米才形成缓坡。主岛最高处海拔为61米。

海姑蘭島的地质为形成于始新世的红色[[水成岩|水成岩]]，类似的地貌在北海周围地区非常罕见。在水成岩下方为形成年代更久的[[白垩|白垩]]层，类似英国多佛尔附近的白色峭壁。

== 文化 ==
海姑蘭島居民的语言和文化接近[[北弗里西亚群岛|北弗里西亚群岛]]。该岛的方言属于弗里西亚语的变种。

海姑蘭島的名称通常被认为是“神圣之地”的意思，<ref>[https://books.google.com/books?id=8HO-fAcUrgsC&pg=PA39 Heligoland, Past and Present, p. 39], Alex Ritsema</ref>主要是因为在[[北欧神话|北欧神话]]中，该岛是正义、和平和真理之神弗里斯提（Forseti）的居所。

== 政治 ==
海姑蘭島在政治上隶属石勒苏益格-荷尔斯泰因州的Klein Pinneberg区管辖。该岛主要的政治力量为：

*德国自由民主党（FDP） 37.1 %
*基督教民主联盟（CDU） 32.2 %
*德国社会民主党（SPD） 30.6 %

=== 旗帜和徽章 ===
[[File:Flag_of_Helgoland.svg|thumb]]
[[File:DE_Helgoland_COA.svg|thumb]]
海姑蘭島的旗帜由绿、红、白三色组成，三色自上向下排列。绿色代表土地，红色代表红色悬崖，白色代表沙滩。这种对颜色的比喻也反映在该岛的箴言中：

Grün ist das Land,<br />
rot ist die Kant',<br />
weiß ist der Sand,<br />
das sind die Farben von Helgoland.

中文意思如下：

: 赤壁連青草，
: 白沙見海邊；
: 海姑蘭小島，
: 三色顯其顏。

海姑蘭島的岛徽形成于1694年，为盾形徽，图案为自上向下排列的绿、红、白三色水平横纹。

== 历史 ==
[[File:Flag_of_British_Heligoland.svg|200px]]

在史前时代，海姑蘭島即有人类居住的痕迹。697年，[[弗里西亚|弗里西亚]]国王在被[[法兰克人|法兰克人]]击败后退踞该岛。1231年，海姑蘭島成为[[丹麦|丹麦]]国王[[瓦尔德马二世|瓦尔德马二世]]的领地。此后该岛所有权在丹麦国王与[[荷尔斯泰因公爵|荷尔斯泰因公爵]]之间多次易手，期间还曾被[[汉堡|汉堡]]和[[汉萨同盟|汉萨同盟]]占据过一段时间。1714年，丹麦出兵夺得了海姑蘭島。

[[拿破仑战争|拿破仑战争]]期间，海姑蘭島在1807年被[[英国皇家海军|英国皇家海军]]占领，成为向[[欧洲大陆|欧洲大陆]]进行[[走私|走私]]和[[间谍|间谍]]渗透的基地。在英国占据期间，海姑蘭島逐渐成为一个度假胜地，许多来自德国和[[奥地利|奥地利]]的文人作者前往岛上居住，以逃避后两国严厉的文字审查制度。1830年和1848年，海姑蘭島还成为德国革命者的避难所。[[德国国歌|德国国歌]]的歌词就是[[奥古斯特·海因利希·霍夫曼·冯·法勒斯雷本|奥古斯特·海因利希·霍夫曼·冯·法勒斯雷本]]1841年在海姑蘭島度假時写出的。

1890年，英国将海姑蘭島转让给德国，以换取[[东非|东非]]的[[桑给巴尔岛|桑给巴尔岛]]。德国[[海军|海军]]领袖[[提尔皮茨|提尔皮茨]]在海姑蘭島的南端修建了[[防波堤|防波堤]]，将其改建为一处可供轻[[巡洋舰|巡洋舰]]停泊的海军基地。

[[第一次世界大战|第一次世界大战]]期间，英国皇家海军的[[战列巡洋舰|战列巡洋舰]]分队对海姑蘭島进行了猛烈的炮轰。一战结束后，[[法国|法国]]曾提议让[[澳大利亚|澳大利亚]]接管这个岛屿，但是岛上的防波堤和防御工事都被炸毁之后，英国最终放弃了对该岛的要求。

[[File:Wd_b159.JPG|thumb]]

两次世界大战之间，海姑蘭島成为旅游胜地。[[维尔纳·海森堡|维尔纳·海森堡]]在岛上度假期间发展出了他的[[量子理论|量子理论]]。

[[纳粹党|纳粹党]]上台后，重新修复了海姑蘭島上的岸炮阵地、地堡和其他防御工事。1939年12月3日，海姑蘭島成为[[二战|二战]]中[[盟军|盟军]]轰炸的第一块德国土地。

1944年10月15日，[[英国皇家空军|英国皇家空军]]出动飞机对海姑蘭島进行了相当猛烈的空袭，1945年4月18日英国的另一次空袭出动了上千架轰炸机，将海姑蘭島上的所有地面建筑全部炸毁。岛上有128人在空袭中丧生，全部是防空部队成员。次日德国海军悄悄撤出了海姑蘭島上的全部幸存居民。

[[第二次世界大战|第二次世界大战]]结束后，[[英国保守党|英国保守党]]领袖[[安东尼·艾登|安东尼·艾登]]提出将海姑蘭島移交给英国，在岛上建设海空军基地，将其改成监视[[威廉港|威廉港]]、[[库克斯港|库克斯港]]、[[埃姆登|埃姆登]]、[[汉堡|汉堡]]等德国重要军港、控制[[基尔运河|基尔运河]]西出口的前哨。但是当时执政的[[英国工党|英国工党]]政府反对[[大英帝国|大英帝国]]的任何领土扩张。在确信无法获得海姑蘭島之后，1947年4月18日，英国海军向该岛运去6800吨炸药，试图将这个小岛完全炸毁。这次爆炸虽然没能完全抹掉海姑蘭島，但却炸飞了岛的南端，并完全改变了该岛的形状，使其面积从1.27平方公里减为1平方公里。《[[吉尼斯世界纪录|吉尼斯世界纪录]]》将这次爆炸列为“世界上规模最大的常规爆破”。

无人居住的海姑蘭島此后一直作为[[英国皇家空军|英国皇家空军]]的轰炸训练靶场。疏散到德国本土的海姑蘭島民举行了一系列示威，要求返回家乡。1952年3月1日，海姑蘭島的主权被英国正式归还给[[联邦德国|联邦德国]]。

== 现状 ==
海姑蘭島目前在行政上隶属于[[德国|德国]][[石勒苏益格-荷尔斯泰因州|石勒苏益格-荷尔斯泰因州]]。该岛现为旅游胜地，也是[[免税|免税]]地区，有渡轮和飞机与德国大陆相连。海姑蘭島也是德国海军的搜索与营救基地。岛上建有[[灯塔|灯塔]]和信号发射塔。
