'''黑旗'''（{{rtl-lang|ar|راية السوداء ''rāyat al-sawdā' ''}}，又称“'''鹰旗'''”（{{rtl-lang|ar|راية العقاب ''rāyat al-`uqāb''}}），简称“'''旗'''”（{{rtl-lang|ar|الراية ''al-rāya''}}），根据[[圣训|圣训]]的记载，是由[[伊斯兰教先知|伊斯兰教先知]][[默罕默德|默罕默德]]升起的旗帜之一。在历史上，[[阿布·穆斯林|阿布·穆斯林]]将其运用在公元747年的导致[[阿拔斯革命|阿拔斯革命]]的起义中，因此黑旗与[[阿拔斯王朝|阿拔斯王朝]]密切相关。它也是[[伊斯兰教末世论|伊斯兰教末世论]]（预示着[[马赫迪|马赫迪]]<ref name="cookshia">{{cite book |title = Studies in Muslim Apocalyptic |author=David Cook |year=2002 |publisher=Darwin Press |page = 197 }} from Majlisi,</ref>的到来）的一个标志。自1990年代末以来，它也被运用于[[伊斯兰主义|伊斯兰主义]]和[[圣战主义|圣战主义]]中<ref name="smh._Flag">{{Cite web |title = Flag being held by Lindt Chocolat Cafe hostages is not an Islamic State flag |author1 = David Wroe | author2 = James Massola |work = [[The_Sydney_Morning_Herald|The Sydney Morning Herald]] |date = December 16, 2014 |accessdate = 2015-03-03 |url = http://www.smh.com.au/national/flag-being-held-by-lindt-chocolat-cafe-hostages-is-not-an-islamic-state-flag-20141215-1279s0.html |quote =  the black banner which was used in the 1990s}}</ref>。

由于其在伊斯兰教传统中的重要地位，黑旗依旧作为宗教标志被使用着。 使用一面简单的黑色旗帜，或包含[[清真言|清真言]]的黑色旗帜一直是主流的做法，因为出于其自身的历史与逊尼派清真言的重要性，穆斯林们相信黑旗不属于极端分子。
