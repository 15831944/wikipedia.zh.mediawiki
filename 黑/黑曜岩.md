'''黑曜石'''（{{lang-en|Obsidian}}）又名'''十勝石'''，是一種自然產生的[[玻璃|玻璃]]。成因是因為火山熔岩迅速冷卻凝結，沒有足夠的時間讓礦物晶體長出，而形成玻璃質。因為熔岩流外圍冷卻的速度最快，所以黑曜石通常都是在熔岩流外圍發現。主要成分：[[二氧化硅|二氧化硅]]，[[硬度|硬度]]：5，[[比重|比重]]：2.339-2.527，[[折射率|折射率]]：1.48-1.51，含水1-2%，化学組成为SiO<sub>2</sub>(H<sub>2</sub>O)。

== 产地 ==
分布在有或曾有火山活动的地区，诸如美国的[[夏威夷|夏威夷]]、[[日本|日本]]和[[印度尼西亚|印度尼西亚]]。其他还有[[冰岛|冰岛]]、[[匈牙利|匈牙利]]、[[意大利|意大利]]附近的[[利帕里群岛|利帕里群岛]]、[[俄罗斯|俄罗斯]]、[[墨西哥|墨西哥]]、[[厄瓜多尔|厄瓜多尔]]和[[瓜地马拉|瓜地马拉]]。深色结核分布在美国[[亚利桑那州|亚利桑那州]]和[[新墨西哥州|新墨西哥州]]。

== 颜色 ==
大部分呈深绿至黑色。少部分也会有红色、绿色、橙色和黄色的例外。天然黑曜石在阳光下会有“彩眼”效果。

== 作用 ==
[[File:Arrowhead.jpg|thumb]]
黑曜石因具备玻璃的特性，敲碎后断面呈贝壳断状口，十分锋利。远古[[石器时代|石器时代]]曾用作刀、箭头等切削用具，現代人利用黑曜石來製作[[外科手術|外科手術]]用[[解剖刀|解剖刀]]的刀片。另外将黑曜石以1000℃以上温度加热后，由于内部所含水分，烧却后会变成白色颗粒状，质地疏松多孔，可用作土壤改良剂。<br />

== 傳說 ==
在美國亞利桑那州所產的黑曜石称为「阿帕契之淚」，傳說[[阿帕契族|阿帕契部落]]內的一支隊伍被敵方騎兵追到山上，50人死於第一波開火，剩下的25人選擇跳崖自殺，全軍覆沒，噩耗傳來，家人們痛哭的眼淚落到地上，變成了一顆顆黑色的小石頭，黑曜石因此得名「阿帕契之淚」。<ref>{{cite web|url=https://www.ideal.azed.gov/p/system/files/GR8_ApacheTears.pdf |title=The Legend of the Apache Tears |accessdate=2018-10-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20140209140634/https://www.ideal.azed.gov/p/system/files/GR8_ApacheTears.pdf |archivedate=2014-02-09 |df= }}</ref><ref name=legend1>[http://www.firstpeople.us/FP-Html-Legends/Apache_Tear_Drop-Apache.html Apache Tear Drop - An Apache Legend]</ref>
