《'''駭客任務系列'''》（{{lang-en|'''The Matrix'''}}）是一系列由[[華卓斯基兄弟|華卓斯基兄弟]]編劇兼導演的[[科幻片|科幻]][[動作電影|動作電影]]，主要包括三部作品。第一部電影《駭客任務》於1999年3月發行；票房大賣后，在2003年先後推出了兩部續集，《駭客任務2：重裝上陣》和《駭客任務3：最後戰役》。除了電影之外，故事中的角色及所描述的虛幻世界還在[[動畫|動畫]]、[[漫畫|漫畫]]、[[小說|小說]]和[[電玩遊戲|電玩遊戲]]中出現。

== 電影 ==
{{Main|駭客任務|駭客任務2：重裝上陣|駭客任務3：最後戰役}}

駭客任務系列電影包含三部曲，皆由[[華卓斯基兄弟|華卓斯基兄弟]]執導及編劇，[[喬·西佛|喬·西佛]]身任製片人，[[基努·李維|基努·李維]]、[[勞倫斯·費許朋|勞倫斯·費許朋]]、[[凱莉-安·摩絲|凱莉-安·摩絲]]及[[雨果·威明|雨果·威明]]等人主演。該系列在澳大利亞拍攝，首部曲為1999年的《駭客任務》，描述駭客[[尼歐|尼歐]]受招募加入了對機器的反叛。電影在上映後獲得了廣泛回響，全球票房總計4.6億美元，且成為美國第一部DVD版本銷售達300萬份的電影<ref name="dvdsales">''{{cite web|url=http://whatisthematrix.warnerbros.com/rl_cmp/rl_press_August_01_00.html|archiveurl=https://web.archive.org/web/20080305104849/http://whatisthematrix.warnerbros.com/rl_cmp/rl_press_August_01_00.html|archivedate=March 5, 2008|title=Press release – August 1, 2000 – The Matrix DVD: The first to sell 3 million|publisher=Warner Bros., Inc.|work=Whatisthematrix.warnerbros.com|location=Burbank|date=August 1, 2000|accessdate=April 10, 2012}}</ref>。
