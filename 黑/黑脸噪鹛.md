'''黑臉噪鶥'''（[[学名|学名]]：{{lang|la|''Garrulax perspicillatus''}}）是[[畫眉科|畫眉科]][[噪鶥屬|噪鶥屬]]的一种。

== 生态环境 ==
以吵鬧的小群於相當密度的植被出現。

== 分布地域 ==
[[香港|香港]]、[[澳门|澳门]]、[[老挝|老挝]]、[[中国大陆|中国大陆]]和[[越南|越南]]。栖息地包括亚热带或热带的湿润低地林、乡村花园、耕地和亚热带或热带的（低地）干燥疏灌丛。

== 特征 ==
黑脸噪鹛的平均体重约为118.0克，体长30cm，灰褐色，额及眼罩黑色；上体暗褐；外侧尾羽端宽，深褐；下体偏灰渐次为腹部近白，尾下覆羽黄褐。虹膜褐色；[[喙|喙]]近黑色，嘴端较淡；脚红褐色；叫声－刺耳的"piew piew piew"聲。

== 食物 ==
以[[昆蟲|昆蟲]]及[[果實|果實]]为主。

== 繁殖 ==
[[File:Garrulax_perspicillatus_MHNT_227_HdB_Mongolie.jpg|thumb]]
巢搭成杯狀，甚不整齊，置於濃密的下層林叢或短樹。
