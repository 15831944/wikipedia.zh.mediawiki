{{engname|《黑太陽731》|''Men Behind the Sun''}}為刻劃[[第二次世界大戰|第二次世界大戰]]時期，[[大日本帝國|大日本帝國]][[關東軍|關東軍]][[731部隊|731部隊]]進行[[人體實驗|人體實驗]]的[[香港電影|香港電影]]。本電影為[[香港|香港]]改行[[香港電影分級制度|香港電影分級制度]]三級制度後，首部被評級為[[三級片|三級]]（未满18岁不准观看）的電影。

== 概述 ==
故事劇情描述日本關東軍[[731部隊|731部隊]]在[[黑龍江省|黑龍江省]][[哈爾濱市|哈爾濱市]]生產[[細菌|細菌]]，部署[[細菌戰|細菌戰]]的過程與對[[中國人|中國人]]所作的暴行。背景在黑龍江省哈爾濱市取景，運用紀錄片手法呈現日本軍人進行人體實驗的各種血腥殘酷手段，特效畫面逼真，令人不忍卒睹。

== 故事劇情 ==
{{Cquote|
友好歸友好，歷史歸歷史。

Friendship is friendship, History is history|影片聲明—引自片頭字幕}}

1945年2月，日軍找來一批少年隊員作為「[[731部隊|731部隊]]」接班人，前往哈爾濱接受最嚴格的軍事訓練。期間，上級長官教育少年隊員將[[中國人|中國人]]視為可以咨意殘害的「原木」，並陸續見識與協助各項慘絕人寰的人體實驗。

# 凍傷實驗：將已遭凍傷的雙手泡入溫水中，經取出後再使勁一扯，皮肉被撕開而手骨露現。
# 受凍實驗：將雙手放進零下190度的機器裏，經取出後再用力把手指逐一敲碎。
# 野外實驗：將人綑綁於十字架上，然後進行爆破，死者四肢斷裂，眼球突現。
# 毒氣實驗：將人和白鴿帶入毒氣室內，隨即施放毒氣，俱於室內窒息。
# 负壓實驗：將人趕入负壓艙內，然後將艙內氣壓遞減，直至人體膨脹，腸臟溢出。
# 解剖實驗：將人施以麻醉後，立即進行解剖，收集出來的器官皆是活生生的。

== 爭議 ==
上映時，影片中一位同髮型身高五官等神似啞巴少年演員的真屍遭到解剖的過程；將一頭活貓掉進一大堆飢餓的鼠群中，讓牠活生生遭吃掉；將一群老鼠潑淋汽油後，隨即點火燒死，都引來極大爭議。然而，部份觀眾則認為，老鼠把活貓吃掉，象徵[[日本侵華|日本侵華]]的殘酷，比喻[[中國人|中國人]]尤如那頭活貓般掉進鼠群中一樣。因此，本電影的製作的合法性受到質疑。導演[[牟敦芾|牟敦芾]]回應那隻貓是被染了色的蜜糖包著，然後先再扔進老鼠坑，那裡的老鼠只是把貓身上蜂蜜的舔走。而如果有觀眾曾經仔細觀察電影中的場景，就會看到老鼠從未咬到那隻貓，血液和染了色的蜂蜜根本是呈現一樣的效果，而且貓從來沒有被看到有任何真實的肉體上的傷口和貓從來沒有完全停止移動。<ref>[http://www.imdb.com/title/tt0093170/faq]</ref>

根據其中一個執導[[陳洪海|陳洪海]]所回憶，當時道具師用刀把大花貓殺死，在用蜂蜜食色調和的血漿塗在貓的身上，老鼠是愛吃甜食的，都跑來吃血漿，就彷佛老鼠在吃貓一樣，
而貓確實已死。<ref>[http://blog.sina.com.cn/s/blog_4b3968180100bke7.html 记忆中电影《黑太阳731》拍摄的背后故事 3 ]</ref>
