'''黑门山'''（[[希伯来语|希伯来语]]：'''הר חרמון'''，''Har Hermon''；[[阿拉伯语|阿拉伯语]]：'''جبل الشيخ'''，''Jabal el-Shaiykh'', ''Djabl a-Shekh'', "mountain of the chief" and“雪山”；英语：'''Mount Hermon'''，{{coord|33|24|N|35|51|E}}），又名'''西雲山'''、'''赫爾蒙山'''，是一座位於[[東黎巴嫩山脈|東黎巴嫩山脈]]<ref>譯名參考：{{cite web|url=http://www.christianstudy.com/data/dict/biography/mount_hermon.html|title=黑門山|publisher=武林英雄帖─中文基督教神學網站|accessdate=2008-04-24}}，英語原名：{{link-en|Anti-Lebanon|Anti-Lebanon}} mountain range。</ref>南部的山，最高峰海拔为2,814米。顶峰位于[[叙利亚|叙利亚]]和[[黎巴嫩|黎巴嫩]]。在1967年[[六日战争|六日战争]]中以色列获胜后，黑门山的南坡和西坡归属以色列控制。1980年，山脉的这一部分和[[戈兰高地|戈兰高地]]被以色列单方面合并。

== 地理 ==
黑门山占地面积约为1000[[平方千米|平方千米]]，其中大约70 km²在以色列控制之下。

== 圣经历史 ==

== 現況 ==
自1981年起，以色列在佔領區以戈蘭高地法律管治。黑門山現時是以色列的唯一滑雪渡假區，包括有一系列適合初學者、中級及專家級的滑雪愛好者使用的場地。

== 以色列村落 ==
現時在以色列控制地區有三條村落，分別為：
*Majdal Shams（Druze）
*Neve Ativ
*Nimrod
