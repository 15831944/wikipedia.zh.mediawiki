'''黑話'''，本指舊時[[江湖|江湖]][[幫會|幫會]]人物的暗語、暗號，往往見於小說，後指流行於某一特殊行業中，非局外人所能了解的語言<ref>[http://dict.revised.moe.edu.tw/cgi-bin/newDict/dict.sh?cond=%B6%C2%B8%DC&pieceLen=50&fld=1&cat=%20&ukey=-1013202273&serial=3&recNo=0&op=%20&imgFont=1 《重編國語辭典修訂本》，中華民國教育部網站]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在互联网时代，[[网络流行语|网络流行语]]是[[网民|网民]]间的黑话。<ref>{{cite web |title=skr、xswl、666…大陸網路「黑話」 你看懂幾個 |url=https://theme.udn.com/theme/story/6774/3451957 |website=联合新闻网時事話題 |accessdate=2019-02-20 |archiveurl=https://web.archive.org/web/20181102205117/https://theme.udn.com/theme/story/6774/3451957 |archivedate=2018-11-02 |date=2018-10-31}}</ref>

== 常見黑話 ==
江湖上的黑話，又稱作切口，也叫春點、寸點、唇點、唇典。許多的幫派都創作出一套複雜的切口體系，有些是方言。例如：
* 「居米」：[[銀幣|銀幣]]，賭博用語。
* 「一方」：一萬，賭博用語。
* 「三兩三」：中醫認為膽囊約重三兩三錢，而後被代指為「勇氣」，黑道、賭博用語。
* 「一塊」：古董界隱語，可指一百、一千、一萬等，各地不相同。古玩商人收購時往往會問價格，以此問價試探賣家是否內行。
* 「米米」：[[錢|錢]]，[[長沙|長沙]]隱語。
* [[北京|北京]]、[[天津|天津]]黑話：「砲兒」為「流氓」，「[[佛爺|佛爺]]」為「竊賊」，「[[狗肉|狗肉]]」為「牌九」，「柯郎馬」為「農夫」或「鄉下人」。
* [[奉天|奉天]]黑話：「裹住」為「加入黑幫」，「鬍子」為「土匪」，「掌櫃」為「首領」，「字匠」為「文員」，「花舌子」為「通信員」，「插千」為「稽查員」，「料水」為「衛兵」。
* [[蘇州|蘇州]]黑話：「如日中」為「天」，「腳踏實」為「地」，「滿面春」為「風」，「滿城風」為「雨」，「名落孫」為「山」，「八仙过」為「海」，「旦底」為「一」，「挖工」為「二」，「橫川」為「三」，「侧目」為「四」、「缺丑」為「五」、「劈大」為「六」、「皂末」為「七」、「分頭」為「八」、「未丸」為「九」、「田中」為「十」。
* [[銀樓|銀樓]]黑話：「旦」為「一」，「竺」為「二」，「清」為「三」，「罷」為「四」、「吾」為「五」、「交」為「六」、「旨」為「七」、「令」為「八」、「旭」為「九」、「平」為「十」。
* [[墓碑|墓碑]]業黑話：「尹」為「一」，「-{于}-」為「二」，「秦」為「三」，「羅」為「四」、「石」為「五」、「路」為「六」、「左」為「七」、「招」為「八」、「仇」為「九」、「古」為「十」。
* [[天地會|天地會]]黑話：「打鷓鴣」為打劫，「鷹爪孫」為官差，「併肩子」為朋友，「踩盤子」或「踩點子」：指事先偵察要行劫、[[偷竊|偷竊]]的對象。
* [[青幫|青幫]]黑話：「撲風」為劫匪；「爬」為搶劫；「尋」為偷竊；「走沙子」為販賣私鹽；「開條子」為販賣女性；「搬石頭」為販賣兒童；「劈黨」為殺人。
* [[江相派|江相派]]黑話：
** 數字：一稱「流」或「柳」、二稱「月」或「葉」、三稱「汪」或「王」、四稱「則」、五稱「中」、六稱「孫」或「神」、七稱「心」、「情」或「青」、八稱「張」、九稱「爱」、十稱「足」、「局」或「臺」、百稱「牌」或「尺」、千稱「干」、萬稱「方」、元稱「皮錦」、毫稱「星星」。
** 名詞：「天」為「父」、「追」為「子」、「比」為「兄弟姊妹」、「七」為「男性」、「八」為「女性」、「生孫」為「商人」、「一哥」為「[[迷信|迷信]]而愚昧的人」、「火底」為「曾加入[[幫派|幫派]]的人」、「畜生」為「[[混吉|混吉]]人士」、「工水」為「勞工工資」、「花底」為「妓女」、「照子」為「眼睛」、「帝壽」為「愚人」。
* [[臺灣|臺灣]][[臺語|臺語]][[黑話|黑話]]：「賊頭」為[[警察|警察]]、「帶頭」為[[老大|老大]]、「舉攕」（giah8-tshiam2）為發難、「兄弟」為[[幫派|幫派]]分子、「坎站」為[[地位|地位]]、「眉角」為[[脾氣|脾氣]]（原意為[[關鍵|關鍵]]、[[技巧|技巧]]）、「七仔」為年輕女性或[[女朋友|女友]]、「角頭」為地方幫派（原意為[[聚落|聚落]]）或幫派人士、「土地公」為地方[[領袖|領袖]]（原意為[[土地神|土地神]]）、「肉粽」或「麵線」為[[送肉粽|縊死鬼]]、「阿拉」或「鴨頭」或「鐵仔」為[[槍械|槍械]]、「土豆」（花生）為[[子彈|子彈]]、「芭樂」為[[手榴彈|手榴彈]]、「桶子」為小船艇、「燃柴」 （hiânn-tshâ）為犯毒癮、「阿思巴拉」為[[癟三|癟三]]、「神經」為理睬。「[[搓圓仔湯|搓圓仔湯]]」，指透過利誘、[[恐嚇|恐嚇]]等手段，迫使他人放棄權利。「[[仙人跳|仙人跳]]」，指透過女色，詐騙他人錢財。「沙庫仔」（日文，さくら，櫻花），假客人、假裝[[競標|競標]]，幫助攤販哄抬價錢的人。「[[日本人|阿本仔]]」，日本人。「[[外省人|阿山仔]]」，來自[[中國大陸|中國大陸]]的[[外省人|外省人]]。「[[半山仔|半山仔]]」，指[[日治時代|日治時代]]旅居中國、[[二次大戰|二次大戰]]後返臺從政的臺籍人士。
* 臺灣[[眷村黑話|眷村黑話]]：「挂」為幫派，動詞為「死去」、「坏掉」；「條子」為[[警察|警察]]；馬子為女性或[[女朋友|女友]]；「性子」為男性或[[男朋友|男友]]等。

== 进入普通话的黑话 ==
* 下海：原指女性到[[青樓|青樓]]工作，在[[中國大陸|中國大陸]][[改革開放|改革開放]]之后，下海一詞的意思轉變為：離開國家機關或者國有企業而自己經商的人。而在[[台灣|台灣]]、[[马来西亚|马来西亚]]則仍沿用其原意。
* 頭寸：[[銀元|銀元]]。現指金錢。
* 号、班房：[[监狱|监狱]]、[[看守所|看守所]]之类。
