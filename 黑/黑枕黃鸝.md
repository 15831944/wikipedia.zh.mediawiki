'''黑枕黄鹂'''（[[學名|學名]]：''{{lang|la|Oriolus chinensis}}''）又名'''-{zh-cn:黄鹂;zh-tw:黑枕黃鸝;}-'''、'''仓庚，'''俗稱'''黃鶯'''、'''黃鳥'''或'''金衣公子'''，是屬於[[雀形目|雀形目]][[黃鸝科|黃鸝科]]的中型[[鳥類|鳥類]]，分布於[[季風亞洲|季風亞洲]]，包含[[俄羅斯|俄羅斯]][[黑龍江|黑龍江]]流域、[[中國|中國]][[中國東北地區|東北]]至[[華南|華南]]一帶、[[朝鮮半島|朝鮮半島]]、[[台灣|台灣]]、[[菲律賓|菲律賓]]、[[中南半島|中南半島]]、[[印度|印度]][[西高止山脈|西高止山脈]]與[[印度東北部|東北地區]]、[[孟加拉|孟加拉]]、[[斯里蘭卡|斯里蘭卡]]西南部、[[巽他群島|巽他群島]]部分島嶼以及[[安達曼-尼科巴群島|安達曼-尼科巴群島]]<ref name=IUCN/>，分布範圍約2,110萬平方公里<ref name = "birdlife">{{ cite web | title = Species factsheet: ''Oriolus chinensis'' | url = http://datazone.birdlife.org/species/factsheet/black-naped-oriole-oriolus-chinensis | author = [[國際鳥盟|國際鳥盟]]（BirdLife International） | accessdate = 28 November 2018 }}</ref>。

== 特征 ==
黑枕黄鹂的體羽大部分呈金黃色，兩翅及尾呈黑色，在頭部通過眼有一條寬闊的黑紋，嘴呈粉紅色，腳呈鉛色，是一種非常美麗的鳥。

== 习性 ==
黑枕黄鹂是一種樹棲性鳥類，常成對穿梭於綠樹叢間。牠的主食是對農林有害的昆蟲，也吃些果實、種子等植物性食物。黑枕黄鹂的巢築在大樹的枝梢間，狀似搖籃，一窩可產卵4-5枚，雛鳥叫聲響亮，在樹下能聽到巢內雛鳥的叫聲。雛鳥生長較快，孵出一個多月後即能隨親鳥逐漸向南遷移。黑枕黃鸝的歌聲猶如流水般的婉轉動聽。

黑枕黄鹂是一種[[候鳥|候鳥]]，夏天飛到[[中國|中國]]東部地區繁殖，冬季南遷[[印度|印度]]、[[斯里蘭卡|斯里蘭卡]]和[[馬來半島|馬來半島]]等地越冬。
