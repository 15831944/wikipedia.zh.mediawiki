'''黑田孝高'''（{{bd|1546年|12月22日|1604年|4月19日}}）[[日本|日本]]戰國[[安土桃山時代|安土桃山時代]]武將，亦為[[天主教|天主教]]徒大名。小寺家臣、[[姬路城|姬路城]]主[[黑田职隆|黑田职隆]]之子，母親是[[小寺政職|小寺政職]]之養女小寺氏，其妻為{{tsl|ja|櫛橋伊定}}之女，兒子[[黑田長政|黑田長政]]為[[筑前國|筑前國]][[福岡藩|福岡藩]]初代藩主。

[[諱|諱]]為「祐隆」（すけたか）、「孝隆」（よしたか），後改為「孝高」，一般通稱'''黑田官兵衛'''，出家[[法號|法號]]為'''黑田如水'''。以身為[[豊臣秀吉|豊臣秀吉]]心腹的立場，在軍略與外交工作方面都相當活躍。與[[竹中重治|竹中重治]]（半兵衛）是秀吉身邊最重要的參謀雙璧，對其取得天下的過程有重大影響，後世並稱為'''「兩兵衛」'''。

== 播州時代 ==
黑田家的發祥地在[[近江|近江]][[伊香郡|伊香郡]]黑田村，家系是源自[[鎌倉時代|鎌倉時代]]宇多源氏佐佐木一黨（京極氏）分出的播磨源氏。孝高的祖父黑田重隆出仕於[[播磨國|播磨國]]御着城主[[小寺政職|小寺政職]]，政職對黑田一族相當重視，任命重隆為重臣，於[[天文_(後奈良天皇)|天文]]14年（1545年）任命為姬路城主。並將自己的養女嫁於重隆之子[[黑田職隆|黑田職隆]]，還將自己的姓氏賜與黑田家。

孝高於天文15年11月29日（1546年12月22日）生於播磨國的[[姬路城|姬路城]]<ref name="kafu">{{tsl|ja|貝原益軒}}『黒田家譜』</ref>，幼名'''万吉'''。[[永祿|永祿]]2年（1559年）母親去世後醉心[[文學|文學]]。永祿5年（1562年）成為主君[[小寺政職|小寺政職]]的近習。同年初次上陣，與父親黑田職隆一同征伐土豪。永祿7年（1564年）孝高的妹妹嫁給{{tsl|ja|浦上清宗}}，在婚禮當日遭到[[播磨國|播磨國]]龍野城主[[赤松政秀|赤松政秀]]率軍襲擊而死亡<ref>[[永禄|永禄]]元年（1558年）、[[浦上政宗|浦上政宗]]らの後見を得た嫡子の[[赤松義祐|赤松義祐]]に追放された[[赤松晴政|赤松晴政]]を、娘婿の[[赤松政秀|赤松政秀]]が受け入れ、晴政の復権に関わった。</ref>。

永祿10年（1567年）時，孝高繼承父親職隆的家督和家老職，迎娶[[小寺政職|小寺政職]]姪兒櫛橋伊定的女兒光（てる）為正室，受命為姬路[[城代|城代]]。永祿12年（1569年），赤松政秀得到擁戴[[足利義昭|足利義昭]]的[[織田信長|織田信長]]麾下的[[池田勝正|池田勝正]]、[[別所安治|別所安治]]、[[宇喜多直家|宇喜多直家]]等人支援，率領3千士兵攻打姬路城，但孝高施展突襲等戰術，僅以300兵擊退{{tsl|ja|三木通秋}}援軍，是為「{{tsl|ja|青山・土器山の戦い|青山土器山之戰}}」。政秀投降於[[浦上宗景|浦上宗景]]。

[[天正_(日本)|天正]]元年（1573年）小寺氏等播磨的大名，被討伐了[[淺井長政|淺井長政]]、流放了[[幕府將軍|幕府將軍]][[足利義昭|足利義昭]]並成功在[[近畿|近畿]]擴張勢力的織田信長、以及取得[[山陰地方|山陰]][[山陽地方|山陽]]的[[毛利輝元|毛利輝元]]2大勢力夾於其中（浦上宗景投靠信長、宇喜多直家則投靠輝元）。天正3年（1575年），孝高由於其才能得到[[織田信長|信長]]賞識，勸說主君小寺政職臣服在[[長篠之戰|長篠之戰]]中大破[[武田勝頼|武田勝頼]]的[[織田氏|織田氏]]。同年7月，在[[豐臣秀吉|羽柴秀吉]]的安排下到[[岐阜城|岐阜城]]謁見信長。而政職亦與{{tsl|ja|斎村政広|赤松廣秀}}、[[別所長治|別所長治]]等人一同前往京都謁見。另一方面，同年9月，宗景敗於投靠毛利氏的宇喜多直家。

天正4年（1576年），迎接了流放中的將軍・足利義昭的毛利氏，派遣水軍大將[[小早川隆景|小早川隆景]]麾下的[[乃美宗勝|浦宗勝]]，以5千兵從毛利同盟的三木通秋轄下的英賀上陸，進軍播磨，但孝高僅以500兵擊退毛利・三木軍（{{tsl|ja|英賀合戦|英賀合戰}}）<ref name="bunsho">『黒田家文書』</ref>。此役後，孝高將長子松壽丸（其後的[[黑田長政|黑田長政]]）送往信長當人質。天正5年（1577年）秋、信長在[[信貴山城之戰|信貴山城之戰]]中討伐[[松永久秀|松永久秀]]後，命羽柴秀吉進駐播磨。孝高讓一族移到父親隠居的飾東郡國府山城甲山，把居城的姬路城提供予秀吉，自己則住在二之丸，以[[参謀|参謀]]的身分活躍。其後跟從[[豐臣秀長|羽柴秀長]]攻打太田垣景近管轄的[[竹田城|竹田城]]（[[但馬國|但馬國]]），與[[蜂須賀正勝|蜂須賀正勝]]一同參與。

天正6年（1578年），曾一度降服[[織田信長|織田信長]]的[[荒木村重|荒木村重]]再舉反旗（[[有岡城之戰|有岡城之戰]]），孝高為顯誠意赴伊丹城（有岡城）勸其降伏，但反遭拘禁，信長以為孝高背反，命令將松壽丸處刑，幸有[[竹中重治|竹中重治]]相救（藏匿於[[竹中重治|竹中重治]]的居城：岩手山城下）。一年後有岡城陷落，脫險的孝高與松壽丸一同返回[[姬路城|姬路城]]，由於孝高被囚於環境差又狹小的牢房裡一整年，故左腿萎縮且膝關節變形，頭也長滿頭癬，從此不良於行，也無法穿戴[[烏帽子|烏帽子]]。

== 豐臣家臣時代 ==
在[[豐臣秀吉|豐臣秀吉]]麾下，他以智谋出众而与[[竹中重治|竹中重治]]（竹中半兵衛）併稱兩兵衛。天正14年（1586）朝廷賜與從五位下勘解由次官，参与了[[九州|九州]]攻伐，在對[[島津義久|島津]]的戰爭中取得勝利有著極大的貢獻。翌年豐臣秀吉統一九州。戰後，被賜與[[豐前國|豐前國]][[中津城|中津城]]12萬5千石。豐臣秀吉原本允諾[[宇都宮鎮房|宇都宮鎮房]]保有豐前國六郡領地，後卻被改封[[伊予國|伊予國]]（愛媛縣），黑田孝高則由[[山崎|山崎]]（播磨國）改封至豐前國，在入主豐前國時，{{tsl|ja|城井谷城|城井谷}}的宇都宮鎮房、{{tsl|ja|長岩城 (豊前国)|長岩城}}的{{tsl|ja|野仲鎮兼}}舉兵叛亂，黑田孝高給予各個擊破，天正16年（1588）為了領內的安定，黑田長政誘殺了宇都宮鎮房一族；黑田孝高於鎮壓[[肥後國|肥後國]]叛亂的陣前斬殺{{tsl|ja|宇都宮朝房}}。

高山右近比黑田孝高還早接受[[天主教|天主教]]洗禮，由於天正15年7月（1587年）豐臣秀吉頒布禁教令，高山右近因反抗被流放，黑田孝高以身作則宣佈棄教；使得天主教與宣教師無法傳播信仰，對當時的天主教諸侯產生衝擊。可由當時[[路易斯·弗洛伊斯|路易斯·弗洛伊斯]]留下的書簡殘篇可知。

天正17年（1589年）將[[家督|家督]]之位讓予黑田長政而退隱，但仍舊在豐臣秀吉身旁活躍，隔年，對北條小田原征伐，在協調終戰過程有很大的功勞，豐臣秀吉賜予名刀日光一文字（原氏直所有，現為國寶，福岡市博物館所藏）。文祿2年（1593），豐臣秀吉對朝鮮出兵（[[文祿慶長之役|文祿慶長之役]]），黑田孝高與[[五奉行|五奉行]]之一的石田三成產生爭執，惹怒豐臣秀吉，黑田孝高以「如水圓清」法號出家。

== 關原之役與晩年 ==
豐臣秀吉逝世（1598年9月18日）後的[[慶長|慶長]]5年（1600），[[五大老|五大老]]之一的德川家康向東討伐會津的[[上杉景勝|上杉景勝]]，[[石田三成|石田三成]]卻於關原率西軍討伐德川家康。黑田長政隸屬東軍正於關原作戰，黑田孝高在九州豐前國中津城率九千黑田軍，擊破得到毛利家支援企圖恢復舊領的[[大友義統|大友義統]]，並認為[[上洛|上洛]]與取得天下不是難事而燃起野心{{citation needed}}，隱藏在東軍德川家康名義下以奪取天下為目標{{citation needed}}攻城掠地，攻佔了豐前、豐後、筑前、筑後、肥前、肥後、日向等七國（僅餘薩摩、大隅），黑田孝高的行動有促成九州統一的趨勢。不過當[[關原之役|關原之役]]一天內大勢底定後，黑田孝高放棄爭霸，隱居而不問政事。

慶長9年3月20日（1604年4月19日）辰時，病逝於京都伏見藩邸，享年59歲。死後舉行天主教儀式葬禮，於下葬後十五或二十日後，其子長政才舉行佛教式葬禮。法名是'''龍光院殿如水圓清大居士'''。墓所在[[福岡市|福岡市]][[博多區|博多區]]千代橫嶽山崇福寺和[[京都市|京都市]]北區龍寶山大德寺搭頭寺院龍光院。

在1902年（[[明治|明治]]35年）11月13日贈[[從三位|從三位]]。

==報竹中氏之恩==
由於[[竹中重治|竹中重治]]冒死罪之險私護孝高嫡子長政免遭殺害，所以孝高父子對重治滿懷感激。日後黑田孝高擔任失去父親的[[竹中重門|竹中重門]]元服時的[[烏帽子親|烏帽子親]]，[[黑田長政|黑田長政]]也將重門庶子收為黑田家重臣。
