'''黑襪醜聞'''（{{lang-en|'''Black Sox Scandal'''}}）是[[美國職業棒球大聯盟|美國職業棒球大聯盟]]的一場[[打假球|打假球]]比賽，其中八名[[芝加哥白襪|芝加哥白襪]]的成員被指控在[[1919年世界大賽|1919年世界大賽]]中故意輸給[[辛辛那提紅人|辛辛那提紅人]]，以換取由[[阿諾·羅斯汀|阿諾·羅斯汀]]指揮的[[體育賭博|賭博]]集團的錢。醜聞的後果導致任命了法官{{link-en|肯尼索·茂騰·藍迪斯|Kenesaw Mountain Landis}}成為第一位[[美國職業棒球大聯盟主席|美國職業棒球大聯盟主席]]，授予他對這項運動的絕對控制權來恢復其完整性。

雖然在1921年公開審判中無罪釋放，但藍迪斯法官判處八名球員終身禁賽。懲罰最終被定義為還包括從[[棒球名人堂|棒球名人堂]]中被驅逐。儘管在接下來的幾十年間要求恢復參賽（特別是[[無鞋喬·傑克森|無鞋喬·傑克森]]的案件），但禁令仍然有效。<ref>{{Cite web |url = http://www.chicagotribune.com/sports/baseball/whitesox/ct-spt-0705-buck-weaver-black-sox-reinstatement-20150703-story.html |title = Buck Weaver's family pushes to get 'Black Sox' player reinstated |author = John Owens |publisher = chicagotribune.com |language = en}}</ref>
