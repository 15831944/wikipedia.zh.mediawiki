{{engname|《童年往事》|''A Time To Live, A Time To Die''}}，由[[侯孝賢|侯孝賢]]導演，於1985年上映的電影。

== 劇情概述 ==
故事劇情描寫一家族的悲歡喜樂，面臨生老病死之泰然的豁達態度，透露出台灣人敦厚本質，以及對生命成長的關懷，亦屬於一個時代的特殊記憶。此電影以侯孝賢導演童年至大學聯考前之回憶自述為背景主軸，自傳性色彩強烈。主角居所主要在[[嘉義市|嘉義市]][[嘉義舊監獄|嘉義舊監獄]]旁的宿舍區取景，從這處充滿日式風格的建築群，構築出那個年代台灣社會民眾的集體回憶<ref>[http://enews.ccu.edu.tw/modules/news/article.php?storyid=3278 冰冷鐵幕新風貌　嘉義舊監人文之旅]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}，〈中正報〉</ref><ref>[http://www.libertytimes.com.tw/2002/new/jul/28/life/art-1.htm 嘉義舊監]，〈[[自由時報|自由時報]]〉</ref>。

== 拍片緣起 ==
導演[[侯孝賢|侯孝賢]]指出，此前執導《[[風櫃來的人|風櫃來的人]]》多參考自身經驗，《[[冬冬的假期|冬冬的假期]]》則多是編劇[[朱天文|朱天文]]的經驗。從中也藉此了解別人的家庭、成長背景與自己的不同，也會對自己家庭了解更透徹，才決定拍自己的自傳《童年往事》。<ref name=":0">[http://www.bilibili.com/video/av2855691/?from=search&seid=17875855641707778483 法國紀錄片《Portrait of Hou Hsiao-Hsien（侯孝賢畫像）》]</ref>

侯孝賢說明，例如拍《冬冬的假期》就注意到朱天文外祖父的摩托車、時鐘都有40、50年，還有手搖唱片機，除了顯示前人珍惜物品，也表現出長久、穩定地生活在台灣。跟侯孝賢家庭就相當不同，因為從大陸來台的侯父，以為只是短期在台灣工作，隨時要離開，「這種疑慮跟不安，你不自覺得會受到影響。」<ref name=":0" />

== 逸事 ==
[[台灣導演|台灣導演]][[侯孝賢|侯孝賢]]依據本身的成長記憶寫成，故此本片有很濃厚的自傳意味。侯導演原先拍攝主角高中時期的場景時曾向其母校[[國立鳳山高級中學|省鳳中]]申請拍攝許可，但遭拒絕，故後來電影中關於[[國立鳳山高級中學|省鳳中]]的所有校內場景皆是在其他學校校園(主要為高雄中學)所攝得。
