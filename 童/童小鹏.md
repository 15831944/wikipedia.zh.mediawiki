'''童小鹏'''（{{bd|1914年|9月20日|2007年|7月18日|catIdx=T童}}），[[福建省|福建省]][[长汀|长汀]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，早年曾任[[毛泽东|毛泽东]]、[[周恩来|周恩来]]的秘书，后任[[中共中央统战部|中共中央统战部]]副部长、[[国务院副秘书长|国务院副秘书长]]。

==生平==
童小鹏1930年6月参加[[中国工农红军|中国工农红军]]，并加入[[中国共产党|中国共产党]]；在[[红四军_(红一方面军)|红四军]]、[[红一军团|红一军团]]政治部、[[政治保卫局|政治保卫局]]任秘书。1934年随部队参加[[长征|长征]]；到[[陕北|陕北]]后，曾任[[毛泽东|毛泽东]]秘书。1936年12月，[[西安事变|西安事变]]后，一直随[[周恩来|周恩来]]在[[西安|西安]]、[[南京|南京]]、[[武汉|武汉]]、[[桂林|桂林]]、[[重庆|重庆]]等地八路军办事处负责秘书、机要工作；历任机要科科长，[[中共中央南方局|中共中央南方局]]秘书处处长，中共中央驻重庆、南京代表团副秘书长，重庆中共中央南方局副秘书长。[[第二次国共内战|第二次国共内战]]时期，曾任[[中共中央城市工作部|中共中央城市工作部]]、[[中共中央统战部|中共中央统战部]]秘书处处长<ref name=shishi/>。

[[中华人民共和国|中华人民共和国]]成立后，先后担任中共中央统战部副秘书长、秘书长；1958年，调任[[国务院总理办公室|国务院总理办公室]]主任，再次跟随周恩来工作，至[[文化大革命|文化大革命]]开始；期间，还兼任过[[最高国务会议|最高国务会议]]秘书、[[国务院副秘书长|国务院副秘书长]]、国务院直属机关党组成员等职。文革初期，升任[[中共中央办公厅|中共中央办公厅]]副主任兼[[中共中央办公厅秘书局|秘书局]]局长。1967年1月，被下放干校劳动。1973年6月恢复工作，和[[刘友法|刘友法]]、[[李金德|李金德]]一起负责中共中央统战部日常工作；并于1977年出任中共中央统战部副部长。后任[[中共中央党史研究室|中共中央党史资料征集委员会]]副主任（部长级待遇）<ref name=shishi/>。

童小鹏是中共第八、十一次全国代表大会代表，第二、三届全国人民代表大会代表，第五、六届全国政协常委<ref name=shishi/>。

2007年7月18日，童小鹏在[[北京|北京]]病逝，享年93岁<ref name=shishi>{{Cite web |url = http://www.caijing.com.cn/2007-07-27/100025548.html |title =  原周恩来总理办公室主任童小鹏逝世|author =  |date = 2007-07-27 |publisher =  财经网|language =  |accessdate =  |quote =  }}</ref>。
