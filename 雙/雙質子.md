== 質子﹣質子鏈反應 ==
<sup>2</sup>He是[[質子﹣質子鏈反應|質子﹣質子鏈反應]]最初的[[中間體|中間體]]。 質子﹣質子鏈反應的第一步可以分為兩個階段，首先，兩個質子熔合形成一個雙質子（氦-2）：
:{{nuclide|link=yes|hydrogen|1}} + {{nuclide|link=yes|hydrogen|1}} + 1.25 MeV → {{nuclide|helium|2}},
接著發生[[正電子發射|正電子發射]]，[[氦-2|氦-2]]衰變成[[氘|氘]]：
:{{nuclide|helium|2}} → {{nuclide|link=yes|deuterium|2}} + {{SubatomicParticle|link=yes|Positron}} + {{SubatomicParticle|link=yes|Electron Neutrino}} + 1.67 M[[電子伏特|eV]],
完整的反應為：
:{{nuclide|link=yes|hydrogen|1}} + {{nuclide|link=yes|hydrogen|1}} → {{nuclide|link=yes|deuterium|2}} + {{SubatomicParticle|link=yes|Positron}} + {{SubatomicParticle|link=yes|Electron Neutrino}} + {{val|0.42|ul=MeV}}.

R. A. W. Bradford考慮了雙質子若能穩定與大霹靂和[[恆星核合成|恆星核合成]]的結合探討其所造成的假設效應<ref name=Bradford/>。
