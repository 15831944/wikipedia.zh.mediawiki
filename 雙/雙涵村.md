== 地理 ==
雙涵村於六腳鄉東側，牛稠溪 （[[朴子溪|朴子溪]]上游）右岸，對岸為嘉義縣六腳鄉灣內，昔日稱為埤頭，因1736年~1795年（[[乾隆|乾隆年間]]）為灌溉大塗獅、六腳的土獅子陂埤圳取水口，因此地設有兩函管，故名雙涵。此村聚落包含雙涵（舊稱埤頭）、新部落、廍仔、蔀仔、五戶子、大庄、公館及後庄<ref>{{Cite web|url=http://gn.moi.gov.tw/geonames/index.aspx|title=地名資訊服務網|accessdate=2017-07-05|author=|date=|publisher=}}</ref>。

== 聚落歷史 ==
西元1733年（雍正11年）時，清政府第一次開放大陸移民搬眷來台，台灣地區的人口結構得到了一定程度的改善，泉州府南安人陳星吉便移居此地。故此中型集村以陳姓為主。此地為「土獅仔陂」源頭，村民祖籍與灣內庄相同。因在河流凹岸處容易受侵，自日治時代，洪患和嘉南大圳的修築，居民不斷往東北方遷徙，形成二個子聚落，即為六腳鄉內的「新部落」，以及新港鄉安和村的「新結庄」。而廍仔及蔀仔則因早期是一草仔廍（清朝以茅草搭建的舊式製糖工場，故稱草仔廍）。位於雙涵之西南方為五戶子，為本村最南的居民點，因只有五戶人家居住，故得此名。大庄為雙涵村內最大的聚落，全村居民的信仰中心「保安宮」位於此地， 公館則因為昔日的館事處而取其名，後庄則為本村後邊的施姓聚落。本村行政區域，是承襲日本時代雙涵[[大字|大字]]，沿用為村名 <ref>{{Cite web|url=http://lioujiao.cyhg.gov.tw/cp.aspx?n=B2F097B8B70D4221&s=55DFE3306329FFE1|title=嘉義縣六較鄉公所|accessdate=2017-07-05|author=|date=|publisher=嘉義縣六較鄉公所}}</ref><ref>{{Cite web|url=http://seabiard.blogspot.tw/2014/06/blog-post_9185.html|title=地名由來嘉義|accessdate=2017-07-05|author=|date=|publisher=}}</ref> <ref>{{Cite web|url=http://www.nanchens.com/xqxx/xqxx14/xqxx14033.htm|title=永春小岵南山陳氏宗親網|accessdate=2017-07-05|author=陳名實|date=|publisher=泉州師範學院泉州學研究所}}</ref>。

== 教育 ==
雙涵村隸屬學區為嘉義縣六腳鄉[[蒜頭國小|蒜頭國民小學]]。蒜頭國小創立於民國前一年四月十五日．最初校名為蒜頭公學校，於民國三十六年三月一日改稱為"蒜頭國民學校"，民國五十四年八月一日縣指定本校為辦理重點輔導國民學校，民國五十七年八月一日因實施九年義務教育，故校名改為"蒜頭國民小學"<ref>{{Cite web|url=http://puzih-hro.cyhg.gov.tw/cp.aspx?n=C4B185FF957E611E） http://www.shtes.cyc.edu.tw/yearlist.html|title=嘉義縣朴子戶政事務所轄內各國民小學學區所屬村里別一覽表|accessdate=2017-07-05|author=|date=|publisher=}}</ref>。

== 文化資源及資產 ==
保安宮：位於雙涵村42號，創立於光緒年間，主祀為[[五府千歲|五府千歲]]，原本是村民之私家神，建廟後歸入廟共祀[[媽祖|天上聖母]]<ref name=":0">{{Cite web|url=http://crgis.rchss.sinica.edu.tw/|title=文化資源地理資料庫|accessdate=2017-07-05|author=|date=|publisher=}}</ref>。

== 景點 ==
[[王得祿墓|王得祿墓]]：位於新港鄉與六腳鄉雙涵村交界處，是目前台灣現存最大的[[后土|后土]]，墓碑立於西元1844年（清道光二十四年），民國七十二年時經評定為國家一級古蹟。王得祿生於西元1770年（清乾隆三十五年），為清朝時台灣官位最高的水師提督，一生擁多次功績，曾受皇帝封為太子太保，也是現今[[太保市|太保市]]的由來<ref>{{Cite web|url=http://okgo.tw/butyview.html?id=1104|title=玩全台灣|accessdate=2017-07-05|author=|date=|publisher=}}</ref><ref name=":0" /><ref>{{Cite web|url=http://www.ttvs.cy.edu.tw/kcc/94531wan/wan1.htm|title=一級古蹟-王得祿墓園|accessdate=2017-07-05|author=|date=|publisher=}}</ref>。
