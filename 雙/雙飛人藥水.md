'''雙飛人藥水'''（{{lang-fr|Alcool de menthe de Ricqles}}），在中國大陸又名'''利佳薄荷水'''，是一種源自[[法國|法國]]的藥用飲料，由法國[[藥劑師|藥劑師]]亨利·德·里克莱斯（{{lang|fr|Henri de Ricqles}}）於1838年在法國[[里昂|里昂]]發明，是一種含有乙醇的薄荷味液體，由於具有藥用價值，因此包裝為小瓶發售。雙飛人藥水由德國Haribo全資擁有的法國利佳製藥廠（{{lang|fr|HARIBO RICQLES ZAN}}）生產，目前利佳的品牌由{{tsl|en|Dr Pepper Snapple Group}}持有，該集團同時擁有[[七喜|七喜]]、[[玉泉汽水|玉泉汽水]]等[[軟性飲料|軟性飲料]]的品牌。

== 歷史 ==
[[File:No_3791,_30_Octobre_1915,_Ricqlès.jpg|thumb]]
法國藥劑師亨利·德·里克莱斯在1838年進行一次實驗期間，無意中製造了一種[[薄荷|薄荷]]飲料，因為具有藥用價值，於是製作為小瓶以“利佳”為名出售，小瓶上以印有兩個天使拿起瓶子的標貼，結果在市場上大受歡迎，隨於法國本土銷售外，在1900年代更隨法國人帶入[[東南亞|東南亞]]地區。

== 名稱 ==
Ricqles藥水的商標是兩個正在飛翔的天使拿起一個藥水瓶子，在中文地區便把天使形容為飛行中的人，又把兩個人稱為一雙，所以被稱為「雙飛人」。

== 成分 ==
雙飛人藥水的主要成分是由天然植物提煉的[[薄荷|薄荷油]]、[[乙醇|乙醇]]和[[蒸餾水|蒸餾水]]，具有濃烈的薄荷味，具有揮發性。

== 用途 ==
雙飛人藥水可以外塗，也可服用。藥廠宣稱雙飛人藥水可用於痾嘔肚痛、蚊蟲叮咬、湯火灼傷、驅風救急、舟車暈浪、除菌除味等用途。香港的官網Facebook更稱雙飛人藥水可作為[[調味料|調味料]]<ref>[https://topick.hket.com/article/1453408/%E9%9B%99%E9%A3%9B%E4%BA%BA%E8%97%A5%E6%B0%B4-%E9%9B%AA%E7%B3%95%E3%80%80%E7%AD%89%E6%96%BC%E8%96%84%E8%8D%B7%E5%91%B3%E9%9B%AA%E7%B3%95%EF%BC%9F?mtc=10016 雙飛人藥水+雪糕 等於薄荷味雪糕？]，香港經濟日報，2016-06-28</ref>，宣稱把數滴雙飛人藥水加在朱古力[[雪糕|雪糕]]，就會成為有薄荷味的「薄荷[[朱古力|朱古力]]雪糕」<ref>[http://www.e-zone.com.hk/channelnews.php?id=15816 勇士實拍 試食「薄荷味」雪糕 雙飛人其實係調味料？！]，e-zone，2016-06-24</ref>，可是藥廠既稱呼雙飛人是具有藥效的[[藥物|藥物]]，但又稱可作為調味料加進[[食物|食物]]，其宣傳手法備受非議<ref>[http://hk.apple.nextmedia.com/news/art/20160624/19668047 雙飛人變調味料！教人溝雪糕食咁好笑？]，蘋果日報，2016-06-24</ref>。
