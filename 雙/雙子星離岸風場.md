'''雙子星離岸風場'''，又譯'''雙子座離岸風力發電園區'''（{{lang-nl|Gemini windpark}}），是[[荷蘭|荷蘭]]一座600[[百萬瓦|百萬瓦]][[標示容量|等級]]的[[離岸風力發電|離岸風力發電場]]，2015年開工，於2017年5月8日正式啟用，是目前世界上第二大的離岸風力發電場，僅次於[[倫敦陣列|倫敦陣列]]。<ref name="經濟部">{{cite web |title = 荷蘭最大、世界第二，雙子星離岸風電場正式啟用 |url = https://www.greentrade.org.tw/zh-hant/purchasing_info/荷蘭最大、世界第二，雙子星離岸風電場正式啟用/ |date = 2017年5月8日 |publisher = 經濟部推動綠色貿易專案辦公室 |accessdate = 2018-07-10}}</ref><ref>{{cite web |title = 荷蘭雙子座離岸風力發電園區啟用 |url = https://taiwan.investinholland.com/荷蘭雙子座離岸風力發電園區啟用/ |publisher = 荷蘭投資局（NFIA） |date = 2017年6月2日 |accessdate = 2018-07-10}}</ref><ref name="guardian">{{cite web |author = Agence France-Presse |title=Full tilt: giant offshore wind farm opens in North Sea |trans-title = 全速完工：巨型離岸風場在北海啟用 |url=https://www.theguardian.com/environment/2017/may/09/full-tilt-giant-offshore-wind-farm-opens-in-north-sea |publisher=Guardian |date = 2017/05/09 |accessdate=2018-07-10 |language=en}}</ref>

雙子星離岸風場座落在[[格羅寧根省|格羅寧根]]北側外海85公里處，總面積達68平方公里，共設有150個[[西門子公司|西門子]]SWT-4.0[[風力發電機|風力發電機組]]（每台容量4 MW）。風場由兩個海域組成，分別落在[[阿默蘭島|阿默蘭島]]以北的區域，以及[[斯希蒙尼克島|斯希蒙尼克島]]以北55公里處。兩個區域的面積各為34平方公里，其中各有75座機組<ref name="經濟部" />。風場由四家公司共同建造：{{link-en|北方電力公司|Northland Power}}（60%計畫股份）、[[西門子公司|西門子公司]]（20%）、{{link-nl|范歐德公司|Van Oord}}（10%）以及[[HVC公司|HVC公司]]（10%）。<ref name="facts">{{cite web |title = Facts and figures |url = http://www.geminiwindpark.nl/e_facts--figures.html |website = geminiwindpark.nl |language = en}}</ref>

雙子星離岸風場預計將提供78.5萬戶家庭所需的電力，並可削減125萬噸的碳排放<ref name="facts" />。在最初十五年的補貼期，雙子星風場每減少一噸碳排放，荷蘭政府便提供240歐元的補貼；這個金額比[[歐盟排放交易體系|歐盟排放交易體系]]所規範的金額高出了30歐元<ref>{{cite web |trans-title = 最低補貼價格可望促成有效的碳排放交易系統 |title = Minimumprijs biedt beste kansen voor effectief CO2-emissiehandelssysteem  |url = http://www.pbl.nl/nieuws/nieuwsberichten/2013/minimumprijs-biedt-beste-kansen-voor-effectief-co2-emissiehandelssysteem |publisher = Planbureau voor de Leefomgeving |date = 25-04-2013 |accessdate= 2018-07-10 |language = nl}}</ref>。

荷蘭的水電瓦斯事業{{link-en|三角洲公司|DELTA}}（DELTA）在2014年初與雙子星離岸風場簽訂了十年的購電合約，使三角洲公司成為雙子星離岸風場的最大客戶<ref>{{cite web |url = https://www.gic.nl/nieuws/park-gemini-geeft-windenergie-in-nederland-vleugels |title = Park Gemini geeft windenergie in Nederland vleugels |trans-title = 雙子星離岸風場為荷蘭的風能提供了翅膀 |website = gic.nl |date = 1 May 2017 |accessdate= 2018-07-10 |language = nl}}</ref>。截至目前為止，三角洲公司尚未透露購電金額。
