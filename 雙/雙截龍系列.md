{{vgsname|雙截龍|ja=ダブルドラゴン|en=Double Dragon|日语也用汉字“{{nihongo||双截龍}}”表记}}是[[日本|日本]]的[[TECHNŌS_JAPAN|TECHNŌS JAPAN]]公司所開發出來的一套[[電子遊戲|電子遊戲]]系列，曾在1980年代末期～1990年代初期風靡一時，並成為[[清版動作遊戲|捲軸式格鬥遊戲]]的始祖。[[北美洲|北美洲]]與[[歐洲|歐洲]]發行權由[[TAITO|TAITO]]取得。

由於《雙截龍系列》的開發群是[[李小龍|李小龍]]的[[影迷|影迷]]，所以在設定上可見到李小龍與[[截拳道|截拳道]]的影子。例如兩位主角分別叫作「李比利」（Billy Lee）與「李吉米」（Jimmy Lee），而李小龍英文名為「Bruce Lee」。李比利與李吉米是[[雙胞胎|雙胞胎]]兄弟，經營[[武術|武術]][[道場|道場]]，功夫高強，所屬[[流派|流派]]稱為「雙截拳」，代表絕招為「龍尾嵐風腳」（即短暫懸停在空中數圈的[[迴旋踢|迴旋踢]]），猶如[[龍捲風|龍捲風]]一般。人稱這對雙胞胎為「雙截龍」。

== 歷史與特色 ==
TECHNŌS JAPAN於1986年推出[[街機|街機]]系統的格鬥遊戲《[[熱血硬派|熱血硬派]]》並取得成功後，便以它為基礎來製作《[[雙截龍_(遊戲)|雙截龍]]》<ref>{{cite web|title=奇跡のシリーズ最新作『ダブルドラゴン IV』開発スタッフインタビュー！ 開発スタッフの平均年齢は50歳以上!?|url=https://www.famitsu.com/news/201702/15126398.html|website=ファミ通.com|language=ja|date=2017年2月15日}}</ref>，隔年推出後結果一炮而紅也因此TECHNŌS JAPAN繼續開發《雙截龍》的後續作品，形成了《雙截龍系列》，與《熱血硬派》為首的《[[熱血系列|熱血系列]]》成為TECHNŌS JAPAN的兩大遊戲[[品牌|品牌]]，同為構築起TECHNŌS JAPAN黃金時代的雙柱。

不同於《熱血硬派》是以日本[[校園|校園]][[不良少年|不良少年]]為主題，《雙截龍》走的是[[美國|美國]]街頭幫派的路線，不過初期除了《雙截龍》可以兩人同時協同遊玩之外，兩者的格鬥風格並沒有很大的差異。後來《熱血系列》往日式搞笑風格發展，不但人物設定為可愛的[[Q版|Q版]]二頭身，動作、表情與對話也都帶有誇張與逗趣的特色，而《雙截龍系列》則保持一貫的肅殺感，和《熱血系列》清楚地區隔了目標市場。儘管如此，TECHNŌS JAPAN仍富創意地以[[戲仿|戲仿]]的形式將《雙截龍系列》的特色融入走搞笑路線的《熱血系列》，其成果就是《熱血系列》中的龍一與龍二兄弟，完全可說是《雙截龍》的[[可愛版|可愛版]]。

==旁支==
=== 超級雙截龍 ===
{{main|超級雙截龍}}
新一代任天堂主機超級任天堂推出，任天堂對主機表現強大的機能作為宣傳，包括色彩、聲效、處理迴旋放大縮小等運算能力，都使玩家有很大的期望，因為過去的移植作品都因機能問題，而未能有一個100%或有一種震撼性的新作。無奈於CPU處理能力低，畫面延慢情況嚴重，在新主機上，角色面積比例變得更細小。此作在音樂和音效上是家用版之中的一大突破，音樂悅耳，聲效是有功夫電影的效果。於是乎，遊戲的不足，聲效是帶來很大的補缺。這也是本系列招式最多元化的，角色動作也注入李小龍的風格，使動作顯得維妙維肖，一切突破點都使這款運行不流暢的遊戲，仍然能表現出獨有趣味性，整體上都總算能滿足電玩迷。

=== 雙截龍ADVANCENCE ===
掌機GBA的遊戲，保有街機的原始風格，且加強遊戲流暢性和招式多樣性，並新增關卡以及敵人。

=== 雙截龍 霓虹 ===
Wayforward和Majesco開發的雙截龍 霓虹2012年登陸XBOX360與PS3，故事仍舊講訴Lee兄弟拯救瑪莉安，敵人變成了Skullmageddon。

=== 雙截龍 重生 ===
於Android系統上的手遊，未在IOS發售，BOSS變成了皮鞭女琳達，第一次出現多結局，一是救出瑪莉安的好結局，二是通關後琳達挾持瑪莉安與比利的對白選擇上，若選擇投降則是比利和瑪莉安都會被她擒獲且被琳達告知要以自己的方式懲罰他們的壞結局。通關畫面只有報社3個月後才收到的兩人被綁縛的照片。

== 登場人物 ==
; 比利（Billy Lee）
: 男主角，女主角瑪莉安的男友，為救被Skullmageddon擄走的瑪莉安再次踏上征程。添加很多招式。
; 吉米（Jimmy Lee）
: 男主角，Billy的兄弟兼搭檔。
; 瑪莉安（Marian）
: 系列女主角，被Skullamgeddon綁架後改造成復仇瑪莉安，被擊敗後清醒。
; Skullmageddon
: 最終BOSS，遠古時代亡者靈魂彙聚而成的強大靈體，招式強橫不知怎麽的喜歡瑪莉安，將她綁架，想讓瑪莉安成為它的妻子，雇傭衆人想打敗並殺死比利以絕後患，最後一關將瑪莉安改造成復仇瑪莉安一起對抗Billy。
; 威廉姆斯（William）
: 主要敵兵之一，後期有很多版本，綽號迅捷之火箭。遊戲開場打暈瑪莉安拐走。
; 琳達（Linda Lash）
: 性感漂亮的SM女王，綽號性虐待者，善用皮鞭。
; 阿波波（Abobo）
: 強大的小頭目，招式全名豐富，相當難打，接受Skullmageddon雇傭。

== 衍生作品 ==
=== 漫畫 ===
1991年，[[驚奇漫畫|驚奇漫畫]]在[[美國|美國]]出版《雙截龍》[[漫畫|漫畫]]版，共6回，每回22頁，第1至4回為[[Dwayne_McDuffie|Dwayne McDuffie]]所作，第5至6回為[[Tom_Brevoort|Tom Brevoort]]與[[Mike_Kanterovich|Mike Kanterovich]]所作。

=== 電視動畫 ===
1993年9月12日至1994年12月4日，[[DiC娛樂|DiC娛樂]]製作的《雙截龍》[[電視動畫|電視動畫]]版在美國首播，共26集分為2季各13集。<ref>{{cite web|title=Double Dragon|url=http://www.imdb.com/title/tt0242197/|website=[[IMDb|IMDb]]|language=en}}</ref>臺灣由[[台視|台視]]在1994年12月7日至1995年7月20日期間播放，首播時間為[[台灣時間|台灣時間]][[星期三|星期三]]17:30～18:00（1994年12月7日至1994年12月28日）與星期四17:00～17:30（1995年2月23日至1995年7月20日）。

=== 電影 ===
1994年11月4日在美上映的真人版本電影<ref>{{cite web|title=Double Dragon (1994)|url=http://www.imdb.com/title/tt0106761/|website=IMDb|language=en}}</ref>，中譯片名為《'''雙龍奇兵'''》；另外在移植至PS上的格鬥版雙截龍，有截錄部分影片使用於遊戲當中。
