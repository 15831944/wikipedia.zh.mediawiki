'''雙鴨山事件'''是2016年3月，發生在[[中華人民共和國|中華人民共和國]][[黑龍江省|黑龍江]][[双鸭山市|双鸭山]]矿区的討薪事件<ref>{{cite web |url =http://www.rfa.org/mandarin/yataibaodao/renquanfazhi/yf2-03152016101315.html|title=双鸭山讨薪事件发酵 吉林通钢再有千人上街维权|publisher=自由亞洲}}</ref>。

==背景==
自2012年以來，因经济危机煤炭价格不断降低，环渤海动力煤报价从最高时的853元人民幣/吨跌至不足600元。許囉產業如[[钢铁|钢铁]]、[[电力|电力]]等需要煤炭的行业都在萎缩，煤炭产能过剩，許多企業不斷虧損，甚至發不出薪資<ref>{{cite web|url =http://groundbreaking.cn/shehui/guonei/6671.html|title =双鸭山事件：直面煤矿工人的”中国痛“}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

==導火線==
2016年3月6日，黑龙江省省长[[陆昊|陆昊]]在两会上表示，在产能调整关闭煤矿过程中，工人得到妥善安置，“没欠井下职工一分钱”，成为激怒被欠薪工人及家属上街请愿的导火索。

==經過==
2016年3月9日起，[[黑龙江|黑龙江]][[双鸭山市|双鸭山市]]数万名矿工與其家属連續幾天發起[[集會|集會]]、[[遊行|遊行]]、[[請願|請願]]，要求[[双鸭山矿业集团|双鸭山矿业集团]]支付被拖欠的六個月薪水，並請求与矿业集团、当局对话解决<ref>{{cite web |url =http://blog.boxun.com/hero/201603/minzhudangtongxun/4_1.shtml|title=关于双鸭山事件的声明|publisher=民主党通讯}}</ref>。被欠薪矿工及家属手持“我们要活着，我们要吃饭”“共产党还我们钱”“陆昊睁眼说瞎话”等横额，高呼“打倒贪污犯”“省长太过分”等口号。
