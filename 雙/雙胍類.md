'''雙胍類'''（{{lang-en|biguanides}}）{{notetag|{{標音|字=胍|注音=ㄍㄨㄚ |拼音=guā|同音字=瓜}}}}是可以只一種分子或者是一系列基於這個分子所發展的藥物。此類藥物可以做為[[糖尿病|糖尿病]]或[[前期糖尿病|前期糖尿病]]的治療藥物；也可以作為[[抗瘧疾藥物|抗瘧疾藥物]]。

* [[二甲雙胍|二甲雙胍]]（Metformin）：廣泛用於[[2型糖尿病|2型糖尿病]]
* [[苯乙雙胍|苯乙雙胍]]（Phenformin）：因為它的毒性，已經退出了大多市場
* [[丁福明|丁福明]]（Buformin）：因為毒性下市
* [[氯胍|氯胍]]（Proguanil）：為抗瘧疾藥物

== 歷史 ==
過去幾個世紀，[[山羊豆|山羊豆]]（學名：{{lang|la|''Galega officinalis''}}）一直作為糖尿病的傳統療法。直到1920年代，在山羊豆萃取物中發現[[胍|胍]]類化合物，[[動物實驗|動物實驗]]也發現這些化合物可以降低血糖濃度。一些毒性較低的衍生物，像是[[十烷双胍|十烷双胍]]（synthalin）A和B，也都用於糖尿病的治療。但是自從發現[[胰島素|胰島素]]後，這些東西就被遺忘了數十年。到了1950年代，雙胍類又重新的用於[[2型糖尿病|2型糖尿病]]。一開始，[[苯乙雙胍|苯乙雙胍]]被廣泛的使用，不過後來因為它具有致死[[乳酸中毒|乳酸中毒]]後果，所以就漸漸的從各國[[藥典|藥典]]移除了。[[二甲雙胍|二甲雙胍]]有較高的安全性，且目前是最常使用的雙胍類。

== 藥理機轉 ==
雙胍類的確切機轉目前還沒有完全了解。會減少[[肝臟|肝臟]]的[[糖質新生|糖質新生]]，使的血中的[[血糖|血糖]]下降；也會促進細胞對[[葡萄糖|葡萄糖]]的利用度增加，使葡萄糖可以加速進入細胞中。

不像[[硫醯基尿素類|硫醯基尿素類]]和[[苯丙胺酸衍生物|苯丙胺酸衍生物]]這兩個降血糖藥物，雙胍類並不會影響[[胰島素|胰島素]]的釋放。所以不只可以用於[[2型糖尿病|2型糖尿病]]，也可以和胰島素一起用於[[1型糖尿病|1型糖尿病]]。

== 副作用 ==
最常見的副作用是[[腹瀉|腹瀉]]跟[[消化不良|消化不良]]，發生率高達30%以上。最重要且嚴重的副作用是乳酸中毒，所以[[腎衰竭|腎衰竭]]是使用二甲雙胍的禁忌。在使用二甲雙胍前要先評估腎功能。
