'''雙節棍'''（[[琉球語|琉球語]]：{{jp|j={{Audio|help=no|Ja-Nunchaku.oga|ヌンチャク}}|rm=Nunchaku}}，{{lang-en|'''nunchuks'''或'''chainsticks'''}}），[[琉球|琉球]]的一種[[冷兵器|冷兵器]]，由兩支中間以短鐵鏈或繩索連在一起的鐵棍或木棍所組成。

== 起源 ==
傳說[[宋太祖|宋太祖]][[趙匡胤|趙匡胤]]发明[[大盘龙棍|大盘龙棍]]（大扫子）和[[小盘龙棍|小盘龙棍]]（小扫子）两种，與現在雙節棍不同的地方是大掃子是一長一短，用以長截破盾，與現在雙節棍應用技法上均大異其趣<ref name="简明武术辞典">{{cite book |author=楊武|title=《简明武术辞典》 |date=1986|publisher=黑龙江人民出版社|location=中國 |language=zh-hans |isbn=}}</ref>，但從宋代起連枷即成為中原地區的制式武器，<ref>《武經總要》</ref>，但名稱隨地區與朝代不同各有演變，如宋代與西夏的鐵鍊夾棒，<ref>《天盛改旧新定律令》</ref>，元明的馬上鞭棍、鞭棍，一直沿續到清代的虎頭棒、哨子棍等，皆有演練之法。现代的双节棍式样则源自[[琉球|琉球]]。双节棍的[[英文|英文]]名称為「Nunchaku」，從[[琉球語|琉球語]]的「{{lang|ja|ヌンチャク}}，{{transl|ja|''nunchyaku''}}」直譯而來。

[[File:Muge.jpg|thumb]]
琉球双節棍作為[[馬轡|馬轡]]使用，亦不做武器使用，農民同時也從福建等地引進[[連枷|連枷]]，成為雙節棍的可靠起源之一，一說琉球双節棍源自日本的拍子木，而日本目前的双節棍亦與拍子木同樣由繩索而非鐵鍊相連。

{{FACT |琉球禁武時代，農民為了自衛而將農具改造成看似不起眼但卻可傷人的雙節棍。操弄雙節棍的技術並不多，因此多半是農民或漁民練習拳術之餘才會練習的一種兵器，因此並無特別的流派或傳承歷史。}}但由於雙節棍的特殊構造，得以衍生出各種華麗動作，加上攜帶便利，因此現受到很多武術運動愛好者的練習，且常用於武術表演。

雙節棍真正名揚世界是始於武打明星[[李小龙|李小龙]]，因为在电影中多次使用，許多文献都提到他是向菲律宾著名武术家[[伊诺山度|伊诺山度]]学习雙節棍，但李小龙的女儿[[李香凝|李香凝]]則否认此一说法，她提出自己父亲的雙節棍功夫是出於自学。

== 法條規範 ==
*雙節棍不屬於[[中華民國|中華民國]]《槍砲彈藥刀械管制條例》規範的武器。
*在《[[香港法例|香港法例]]》中，雙節棍被列入「重力操作鋼棒」類（[[警察|警用]]伸縮[[警棍|警棍]]也屬此類）<ref>《香港法例》第217章附表</ref>。在[[中華人民共和國|中華人民共和國]]的[[中華人民共和國法律|法律]]中，双截棍不属于被管制的武器，造价也十分便宜，几元至几十元的[[人民币|人民币]]就可以買到一副。

== 结构 ==
[[File:Nunchaku_Routine.gif|thumb]]
由棍体和棍链组成，通常一根叫龍棍，另一根叫陽棍，又或叫青棍和白棍。对于单节棍体来说，靠近棍链一端称为内端，另一端为外端。

== 握法 ==
共分為正握、反握、叠握、垂棍、架棍、夹棍、背棍、藏棍、举棍等。
* 正握：虎口朝向棍的内端
* 反握：虎口朝向外端称为反握
