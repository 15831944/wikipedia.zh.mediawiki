《'''雙子殺手'''》<ref name="自由時報 (2018-03-01)"/><ref name="蘋果時報 (2018-02-28)"/><ref name="中時電子報 (2018-02-28)"/><ref name="TVBS (2018-01-14)"/><ref name="自由時報 (2017-04-23)"/>（暫譯，{{lang-en|''Gemini Man''}}）是一部於2019年上映的美國[[科幻片|科幻]][[動作片|動作]]電影，由[[李安|李安]]執導，[[大卫·贝尼奥夫|大衛·班紐夫]]、{{link-en|強納森·漢斯雷|Jonathan Hensleigh}}、{{link-en|達倫·萊姆克|Darren Lemke}}、[[安德魯·尼可|安德魯·尼可]]、史蒂芬·J·李佛（{{lang|en|Stephen J. Rivele}}）與{{link-en|克里斯多福·沃金森|Christopher Wilkinson}}編劇。該片由[[威爾·史密斯|威爾·史密斯]]、[[克萊夫·歐文|克萊夫·歐文]]、[[瑪麗·伊莉莎白·文斯蒂德|瑪麗·伊莉莎白·文斯蒂德]]與[[黃凱旋|黃凱旋]]主演，劇情講述一名美國[[美國國家安全局|國安局]]（NSA）特工（史密斯飾演）被比自己年輕且強壯的複製人追殺。

該片源於達倫·萊姆克構思的一個概念。[[華特迪士尼影業|華特迪士尼影業]]與[[傑瑞·布洛克海默|傑瑞·布洛克海默]]於1997年開始製作《雙子殺手》，原定導演為[[東尼·史考特|東尼·史考特]]和[[柯蒂斯·汉森|柯提斯·韓森]]，但最終因當時技術不夠發達而未能拍成。到了2016年，{{link-en|天空之舞製片公司|Skydance Media}}從迪士尼手中買下該片。電影的[[主體拍攝|主體拍攝]]於2018年2月27日在美國[[喬治亞州|喬治亞州]]的[[薩凡納|薩凡納]]開始。《雙子殺手》將於2019年10月4日在全球上映。

== 劇情 ==

一名準備退休的美國[[美國國家安全局|國安局]]（NSA）特工亨利（{{lang|en|Henry}}）遭到殺手追殺，經調查後他發現殺手是比自己年輕25歲，各方面能力都較他強的複製人<ref name="自由時報 (2018-03-01)"/><ref name="蘋果時報 (2018-02-28)"/><ref name="中時電子報 (2018-02-28)"/><ref name="Release"/><ref name="WinsteadCast"/>，而自己的一舉一動似乎都在對方的掌握之中<ref name="Score"/>。

== 角色 ==

* [[威爾·史密斯|威爾·史密斯]]飾演亨利，一名準備退休的美國國安局特工<ref name="自由時報 (2018-03-01)"/>。

* [[克萊夫·歐文|克萊夫·歐文]]飾演反派，複製人計畫的負責人<ref name="WinsteadCast"/>。

* [[瑪麗·伊莉莎白·文斯蒂德|瑪麗·伊莉莎白·文斯蒂德]]飾演丹妮（{{lang|en|Danny}}），監視亨利的女特工<ref name="WinsteadCast"/>。[[伊莉莎白·戴比基|伊莉莎白·戴比基]]與{{link-en|塔緹安娜·瑪斯蘭尼|Tatiana Maslany}}曾是該角的演員人選<ref name="TVBS (2018-01-14)"/><ref name="WinsteadCast"/>。

* [[黃凱旋|黃凱旋]]

* {{link-en|琳達·艾蒙德|Linda Emond}}<ref name="Daily Mail (2018-03-28)"/>

== 製作 ==

=== 籌拍 ===

該片基於{{link-en|達倫·萊姆克|Darren Lemke}}構思的一個概念。[[華特迪士尼影業|華特迪士尼影業]]與[[傑瑞·布洛克海默|傑瑞·布洛克海默]]於1997年開始製作該片<ref name="Deadline.com (2017-07-06)"/>。《雙子殺手》原定由[[東尼·史考特|東尼·史考特]]和[[柯蒂斯·汉森|柯提斯·韓森]]執導，[[克林·伊斯威特|克林·伊斯威特]]主演<ref name="自由時報 (2017-04-23)"/><ref name="Deadline.com (2017-07-06)"/>。當時迪士尼的電腦動畫與特效工作室「秘密實驗室」（{{lang|en|The Secret Lab}}，現已關閉）開發了一部名為《人臉計畫》（{{lang|en|''Human Face Project''}}）的測試短片，計畫打造電影中的特效，如呈現片中比主角本人年輕的複製人所需的返老還童特效<ref name="Cartoon Brew (2017-09-05)"/>。《雙子殺手》在迪士尼手上時並未有任何進展，因該技術不夠發達，無法拍攝該片<ref name="Deadline.com (2017-07-06)"/>。

2016年，{{link-en|天空之舞製片公司|Skydance Media}}從迪士尼手中買下該片，布洛克海默依舊擔任製片人<ref name="Deadline.com (2017-07-06)"/>。{{link-en|大衛·艾利森|David Ellison}}也是製片人之一<ref name="WongCast"/>。[[李安|李安]]於2017年4月簽約執導<ref name="Ang Lee"/>。該片由[[威爾·史密斯|威爾·史密斯]]主演<ref name="Release"/>。[[克萊夫·歐文|克萊夫·歐文]]<ref name="OwenCast"/>與[[瑪麗·伊莉莎白·文斯蒂德|瑪麗·伊莉莎白·文斯蒂德]]<ref name="WinsteadCast"/>分別於2018年1月12日和1月16日加盟劇組。

=== 拍攝 ===

該片的[[主體拍攝|主體拍攝]]於2月27日在美國[[喬治亞州|喬治亞州]]的[[薩凡納|薩凡納]]開始<ref name="自由時報 (2018-03-01)"/><ref name="WongCast"/><ref name="WSAV (2018-02-28)"/>。薩凡納當地多蟲，劇組人員都得戴著蚊帳頭罩才能到室外<ref name="自由時報 (2018-03-01)"/><ref name="蘋果時報 (2018-02-28)"/><ref name="中時電子報 (2018-02-28)"/><ref name="WSAV (2018-02-28)"/>。開拍同時，演員[[黃凱旋|黃凱旋]]加入了劇組<ref name="WongCast"/>。接著，劇組前往[[哥倫比亞共和國|哥倫比亞]]的[[卡塔赫納_(哥倫比亞)|卡塔赫納]]拍攝一場街頭槍戰戲<ref name="El Heraldo-1"/><ref name="El Heraldo-2"/><ref name="自由時報 (2018-04-29)"/>。劇組接下來移師至[[匈牙利|匈牙利]]的[[布達佩斯|布達佩斯]]<ref name="TVBS (2018-01-14)"/>。7月13日，在拍完布達佩斯的部分後，導演李安與製片人布洛克海默在[[Twitter|Twitter]]上宣布該片的外景拍攝作業已結束<ref name="自由時報 (2018-07-14)"/>。

同李安先前的作品《[[比利·林恩的中場戰事|比利·林恩的中場戰事]]》（2016年），該片也是以每秒120[[帧|幀]]、4K、3D的高規格拍攝<ref name="VFX"/><ref name="自由時報 (2019-04-06)"/>，不過未來還是會有每秒60幀的版本上映<ref name="ETtoday新聞雲 (2019-03-13)"/>。

=== 特效 ===

該片的特效部分由[[维塔数码|Weta數位]]打造<ref name="VFX"/>，其知名作品包含[[魔戒電影三部曲|《魔戒》電影三部曲]]和《[[阿凡達|阿凡達]]》（2009年）等<ref name="Weta"/>。

== 配樂 ==

2019年2月底，曾為《[[驚聲尖叫|驚聲尖叫]]》系列電影、《[[决斗犹马镇|決戰3:10]]》（2007年）和《[[羅根_(電影)|羅根]]》（2017年）等片創作配樂的[[馬可·貝爾特拉米|馬可·貝爾特拉米]]加入劇組<ref name="Score"/>。

== 上映 ==

《雙子殺手》將於2019年10月4日在全球上映<ref name="自由時報 (2018-03-01)"/><ref name="蘋果時報 (2018-02-28)"/><ref name="中時電子報 (2018-02-28)"/><ref name="Release"/>。該片的首支片段於2019年4月4日在美國[[拉斯維加斯|拉斯維加斯]]的國際電影產業博覽會（{{lang|en|CinemanCon}}）上釋出，廣受媒體好評<ref name="自由時報 (2019-04-06)"/><ref name="自由時報 (2019-04-05)"/>。
