'''雙溪車站'''位於[[台灣|台灣]][[新北市|新北市]][[雙溪區|雙溪區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[宜蘭線|宜蘭線]]的[[鐵路車站|鐵路車站]]。未來可以站外步行前往[[北宜新線|北宜新線]][[雙溪新站|雙溪新站]]，[[北宜新線|北宜新線]]乃在計畫中。

== 車站構造 ==
*[[島式月台|島式月台]]兩座，車站標高約海拔35公尺。
*與牡丹車站間路段為坡度千分之16的陡坡，被鐵道迷稱為「牡丹坡」。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#二等站|二等站]]。
*[[自強號列車|自強號]]
**順行停靠：204(208)、272、172次、246車次；
**逆行停靠：407、175、177、239次。
*[[莒光號列車|莒光號]]除[[觀光列車|觀光團體列車]]、順行：562次、逆行：633次外，皆停靠。
*[[復興號列車|復興號]]：停靠688次。
*[[太魯閣列車|太魯閣號]]、[[普悠瑪列車|普悠瑪號]]：不停靠。
*[[臺鐵區間快車|區間快車]]
**順行停靠：4052次。
**不停靠北上(順行)4026次、4030次，南下(逆行)4021次、4027次。
**逆行停靠：1551次。
**順行4028次為終點，逆行4023次為起點。
*[[臺鐵區間車|區間車]]皆停靠。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費<ref>[http://www.railway.gov.tw/tw/CP.aspx?sn=1312&n=1333 臺鐵多卡通電子票證乘車營運規則]</ref>。
*本站常駐電力機車，作為補助機車或救援機車用。

== 歷史 ==
*1923年10月11日：初名'''雙溪停車場'''，與溪州停車場（今[[南州車站|南州車站]]）同日開始運輸營業<ref>告示第百六十七號，府報第三千六十二號，臺灣總督府</ref>。
*1924年12月1日：更名為'''頂雙溪驛'''；因雙溪「そうけい」（soukei）和礁溪「しょうけい」（shoukei）二站的日文發音類似，為避免混淆才更名<ref>告示第百三十一號，府報第三千三百八十三號，大正十三年十一月二十日，臺灣總督府</ref>，同日宜蘭線鐵道全通。
*1967年10月10日：恢復原名'''雙溪車站'''，與[[成功車站|成功]]、[[豐富車站|豐富]]、[[水里車站|-{水里}-]]等車站同時改稱今名。
