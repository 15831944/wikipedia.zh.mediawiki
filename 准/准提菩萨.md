'''准提菩薩'''（{{lang-sa|चुन्दी|Cundī}}），为[[佛教|佛教]]中[[显教|显教]]、[[密宗|密教]]所共尊的大[[菩萨|菩萨]]；因為音譯或用字不同，又作「準提佛母」、「准胝菩薩」、「尊那菩薩」、「宗達天女」、「純陀天女」等，[[密宗|密宗]]稱之「莲花部诸尊之母」、「七[[俱胝|俱胝]][[佛母|佛母]]」，密号'''最胜金刚'''。

有宗派認為準提菩薩是[[觀世音菩薩|觀世音菩薩]]的[[化身|化身]]，又称'''準提觀音'''、'''天人丈夫观音'''，歸入[[六观音|六观音]]之一，為救度「人道」之[[觀音菩薩|觀音菩薩]][[化身|化身]]。六觀音即：[[慈航觀音|慈航觀音]]（救拔[[餓鬼道|餓鬼道]]）、[[千手千眼觀世音菩薩|千手觀音]]（救拔[[地獄道|地獄道]]）、[[馬頭觀音|馬頭觀音]]（救拔[[畜生道|畜生道]]）、[[十一面觀音|十一面觀音]]（救拔[[修羅道|修羅道]]）、[[準提觀音|準提觀音]]（救拔[[人道|人道]]）、[[如意輪觀音|如意輪觀音]]（救拔[[天人_(佛教)|天道]]）。

因菩薩具有十八隻手，故又俗稱為「[[十八手觀音|十八手觀音]]」，其造象常讓人誤以為是「[[千手觀音|千手觀音]]」。

準提菩薩誕辰普傳為農曆三月十六日，一說三月初六日，或以觀音菩薩誕辰二月十九日再次禮拜。

== 简介 ==
[[準提菩薩|準提菩薩]]一般為相貌庄严的[[天女|天女]]像，與[[摩利支菩薩|摩利支菩薩]]相似，[[漢傳佛教|漢傳佛教]]之[[菩薩|菩薩]]形象多为三目十八臂，手持莲花、利剑、青珠、五股杵、花鬘、澡瓶、绳索、海螺、梵箧等，表示消灾、去惑、增福、延命、超脱[[轮迴|轮迴]]、往生[[净土|净土]]等願；[[藏傳佛教|藏傳佛教]][[覺囊派|覺囊派]]所傳為三面、二十六臂之相。由於菩薩化身無限，亦有戎裝像、出家像等。
準提佛母菩薩的[[準提咒|準提咒]]法也是[[唐密|唐密]][[漢傳獨部密法|漢傳獨部密法]]中的四大法之一。

==分部==
通說準提菩薩即是觀音菩薩，少數宗派則認為準提菩薩是[[大日如來|大日如來]]的化身，如[[東密|東密]]廣澤流認為準提佛母應屬於「佛部」，不應屬於「觀音部」，小野流則認為此菩薩應屬「觀音部」，認為坊間所謂「十八手觀音」就是「準提觀音」<ref>[http://www.sjskim.org/cunde/ex/1/ex1_01.htm 準提佛母，概說——部別與名號]</ref>。

==聖誕==
準提菩薩聖誕為[[農曆|農曆]]三月六日或三月十六日，現代佛教團體普遍在三月十六日舉行聖誕法會等，三月十六日一說，可見於民間流傳之[[農民曆|農民曆]]所附神佛聖誕表中，以及部份傾向傳統或宗教團體印刷之日、月曆等；古代記載卻是三月初六日<ref>《百丈叢林清規證義記》([http://www.cbeta.org/result/normal/X63/1244_003.htm 卷第三] {{webarchive|url=https://web.archive.org/web/20160521101835/http://www.cbeta.org/result/normal/X63/1244_003.htm |date=2016-05-21 }})記載準提佛母菩薩聖誕為三月初六日。</ref>。

== 神咒 ==
该位大菩萨的[[準提咒|準提神咒]]以灵验而著称，且持诵者只要虔誠又具善心，无论[[居士|在家]]、[[出家|出家]]，[[素食|茹素]]、食肉，一律可得福佑、如愿<ref>《七俱胝佛母准提大明陀罗尼经》:“诵持准提陀罗尼，能得到光明独照。所有罪障尽皆消灭。寿命延长。福德智慧增进。并得致诸佛的庇护。生生世世远离诸恶趣。迅速证得无上菩提的佛果。准提咒印能够灭除十恶五逆一切重罪。成就一切白法。具戒清净，速得菩提。若在家人。纵然不断酒肉、妻子。但依法修持。无不成就。”</ref>。此乃準提佛母菩薩之大悲願力，然行者若能漸次斷酒斷肉，亦善也。

其咒曰：
稽首皈依蘇悉諦
頭面頂禮七俱胝
我今稱讚大準提
為願慈悲垂加護
*南無。颯多喃。三藐三菩陀。俱胝喃。怛姪他。唵。折隸。主隸。準提。娑婆訶。
*Namaḥ Saptānāṃ Samyaksaṃbudda Koṭīnāṃ Tadyathā Oṃ Cale Cule Cunde Svāhā

== 脅侍 ==
[[難陀龍王|難陀龍王]]與[[跋難陀龍王|跋難陀龍王]]。在準提佛母菩薩的塑、畫像中，二者位於菩薩身側，形象有作二名龍王相，亦有作二名長者相。

== 化現 ==
在[[漢傳佛教|漢傳佛教]]，'''[[摩利支菩薩|摩利支菩薩]]'''或被視為準提佛母菩薩<ref>[[密藏道開|密藏道開]]《如來淵源攷》：「或以准提像，造摩利支菩薩像，一為菩薩無邊化身，無形無相；二則法身即准提也。」</ref>[[化身|化身]]。[[摩利支菩薩|摩利支菩薩]]即[[道教|道教]]奉祀的[[斗姥元君|斗姥元君]]，斗姥元君與準提佛母菩薩的胸前手印也極其近似。

在[[民間信仰|民間信仰]]，[[準提佛母菩薩|準提佛母菩薩]]是滿願的大菩薩，有時也稱為[[母娘|母娘]]，為[[靈山派|靈山派]]歸入「五母信仰」中。五母之組合各有說法，總歸為五位母性形相之神祇或菩薩。

在[[藏密|藏密]]有一種說法，準提佛母是[[普賢王如來|普賢王如來]]的化現，法身[[準提佛母|準提佛母]]，報身[[觀世音菩薩|觀世音菩薩]]，化身[[綠度母|綠度母]]。準提佛母的不共功德是『事業』，所相應的[[淨土|淨土]]是『[[極樂世界|極樂世界]]』 <ref>{{cite web|url=http://www.ucchusma.net/samanta/cunde/ex/2/ex2_18.htm|title=準提佛母是三世一切諸佛的慈悲、智慧和力量的總集|publisher=七俱胝佛母準提王 阿知上師|access-date=2018-04-30|archive-url=https://web.archive.org/web/20180501093332/http://www.ucchusma.net/samanta/cunde/ex/2/ex2_18.htm#|archive-date=2018-05-01|dead-url=yes}}</ref>。

== 十八臂準提 ==
在[[漢傳佛教|漢傳佛教]]或唐密、日本真言宗都以十八臂準提佛母為本尊，在藏傳佛教很少見到以十八臂準提為本尊(多以四臂、二十六臂為本尊)，藏傳準提法唯有以頂果欽哲法王所傳承的十八臂準提法，甚為稀有難得 <ref>{{cite web|url=http://www.pema-rigzin.com/index.php?do=product&id=79|title=2009年十八臂準提佛母灌頂法會|publisher=虛空法林佛學會}}</ref>。

十八臂準提佛母胸前的手結印契，以'''總持印'''與'''說法印'''最為常見。
