'''立榮航空'''（{{lang-en|UNI Air}}）是一間[[台灣|台灣]]的[[航空公司|航空公司]]，於1996年成立，與[[長榮航空|長榮航空]]同屬[[長榮集團|長榮集團]]旗下，以經營國內航線、[[兩岸航線|兩岸航線]]與區域[[國際航線|國際航線]]為主。

立榮未隨長榮航空公司一同加入[[星空聯盟|星空聯盟]]，因此不屬於星空聯盟成員。

== 沿革 ==
[[File:EVA_Air_Chang'an_Building_20101014.jpg|thumb]]
[[File:UNI_Air_logo.png|thumb]]
[[File:General_Manager_of_Great_China_Air_to_ROC-MOTC-CAA_19980623.jpg|thumb]]合併，合併後以立榮航空為存續公司]]
* 立榮航空的前身為'''[[馬公航空|馬公航空]]'''（{{lang|en|Makung Airlines / Makung International Airlines}}，取名自[[澎湖縣|澎湖縣]][[馬公市|馬公市]]），成立於1988年8月6日，是[[中華民國交通部|中華民國交通部]]實施「開放天空」政策後第一家成立的航空公司。
* 馬公航空分別於1988年11月及1989年2月，引進兩架{{link-en|HS-748|Hawker Siddeley HS 748}} ，執行台北/高雄-馬公航線為主。
* 1990年自1995年，馬公航空引進五架[[BAe-146|BAe-146]]， 執行台北-高雄及台北/高雄-馬公航線為主。
* 1994年3月，馬公航空引進一架[[波音757|波音757]](全台第一架[[波音757|波音757]])，執行高雄-菲律賓佬沃的包機及台北-高雄航線。
* 1995年，[[長榮集團|長榮集團]]購買馬公航空之股權，於1996年3月將馬公航空更名為'''立榮航空'''。
* 1998年7月1日，為了提升競爭力、並響應政府的航空資源共享政策，立榮航空與[[台灣航空|台灣航空]]（Taiwan Airways）以及[[大華航空|大華航空]]（Great China Air）合併，合併後以立榮航空為存續公司，立榮航空同時承接長榮航空之國內航線，成為國內唯一「五金齊發」(五座島內主要機場均飛航金門)、「五馬奔騰」(五座島內主要機場均飛航馬公)的航空公司。
* 2004年政府開放[[小三通|小三通]]後，立榮航空設計出「金廈一條龍」(即可一次訂好船票、機票及金門水頭碼頭與尚義機場間的雙向接駁、行李運送等服務)的營運模式，成為當時立榮航空重要的獲利來源之一。
* 2012年3月30日，立榮航空更換企業識別標誌。
* 2012年9月27日，立榮航空首架訂購之[[ATR-72|ATR-72-600]]型客機於法國交機，於同年10月11日首次在台灣進行商業飛行<ref name = "ATR-2">[http://www.atraircraft.com/newsroom/press-releases-details-1191-en.html ATR delivers first ATR 72-600 to fly in Asia], 27 September 2012</ref>。
* 2016年6月1日，立榮航空最後一架[[MD-90|MD-90]]型(國籍編號B-17920)的客機在完成最後一趟金門至台北航線(B7-8890飛行任務)後光榮除役。此機型立榮航空自1996年起陸續引進14架，曾是該公司飛行國內線的主力機種，也在兩岸開放直航後扮演重要的角色。<ref>[http://61.222.185.194/?FID=6&CID=316242 立榮MD-90客機 走入歷史]</ref>
* 2016年9月13日，立榮航空推出首架酷企鵝彩繪機，每日往返[[臺北松山機場|台北]]─[[馬公機場|馬公]]航線。<ref name = "b17001">[https://www.uniair.com.tw/uniweb/News_Page.aspx?ANNID=20160913134235475&Language=zh-tw], 2016/09/13</ref>

== 事故 ==
* 1999年8月24日，[[立榮航空873號班機|立榮航空873號班機]]，[[波音|波音]][[MD-90|MD-90]]型機（B-17912），由台北飛往花蓮，在花蓮機場的跑道上滑行時，因不明原因爆炸，使得機身中段炸毀，造成1人喪生。當初由於喪生者為乘客[[古金水|古金水]]胞兄，檢方曾懷疑失事原因是由於古金水攜帶爆裂物登機而導致爆炸；但經查證後已還古金水清白。<ref>{{cite news|title=立榮爆炸案 古金水更五審無罪定讞|url=http://news.ltn.com.tw/news/focus/paper/491992|publisher=自由時報|date=2011-05-13|accessdate=2015-01-01}}</ref>
* 2001年1月15日，[[立榮航空695號班機|立榮航空695號班機]]，DASH 8-300型機（B-15235），由臺南飛往金門。降落在金門時，因為遭到[[風切變|風切變]]，導致重落地，使該機的主起落架折損，無人員傷亡。
* 2008年1月15日，[[立榮航空690號班機|立榮航空690號班機]]，DASH 8-300型機（B-15235），由金門飛往嘉義，因為遭到風切變，導致重落地，使該機的主起落架折損，1人受傷。
* 2011年7月22日，[[立榮航空684號班機|立榮航空684號班機]]，DASH 8-300型機（B-15235），由台南飛往澎湖，因為遭到風切變，導致重落地，使該機的主起落架折損，1人受傷。
* 2015年6月19日，[[立榮航空881號班機|立榮航空881號班機]]，MD-90型機（B-17925），由台北飛往金門，上升到約4500呎時，引擎異常溫度升高，機長立即折返松山機場，159位乘客、機組人員均平安。
