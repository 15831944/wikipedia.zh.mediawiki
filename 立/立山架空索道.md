'''立山架空索道'''<ref name="alpen-trans-tc">{{Cite web|url=https://www.alpen-route.com/tw/transport_new/modes.html|title=阿爾卑斯路線內的交通：各交通機關的說明 |publisher=[[立山黑部阿爾卑斯山脈路線|立山黑部阿爾卑斯山脈路線]]|language=zh-hant|accessdate=2018-11-22}}</ref>（{{lang-ja|'''立山ロープウェイ'''}}，又名'''立山空中纜車'''<ref>{{Cite web|url=http://www.info-toyama.com/toyamakanko/statics/foreign/cht/tlibrary/tateyama.html|title=富山百景 / 推薦 - 立山黑部阿爾卑斯山脈路線|website=富山觀光指南|publisher=富山縣觀光・交通・地域振興局|language=zh-hant|accessdate=2018-11-22}}</ref>）是日本一條由{{link-ja|立山黑部貫光|立山黒部貫光}}經營，往返[[大觀峰站|大觀峰站]]與[[黑部平站|黑部平站]]的[[索道|架空索道]]<ref>{{Cite web |url=http://watchizu.gsi.go.jp/watchizu.html?longitude=137.6369667&latitude=36.56886111|script-title=ja:地図閲覧サービス|publisher=[[国土地理院|国土地理院]]|language=ja|accessdate=2011-09-05}}</ref>。該線因地制宜採用無支柱式設計（{{lang|ja|ワンスパン方式}}）<ref name="alpen-navi"/><ref name="omachi">{{Cite web|url=http://www.kanko-omachi.gr.jp/alpine_route.html|script-title=ja:アルペンルート > アルペンルートの乗物|website=信濃大町導航|publisher=大町市觀光協会|language=ja|accessdate=2018-11-22}}</ref>，沿途不設任何支柱<ref name="alpen-navi"/>，為日本最長的無支柱式空中纜車<ref name="alpen-trans-tc"/><ref name="omachi"/><ref>{{Cite web|url=https://www.info-toyama.com/doc/loculture/vol16/page07.html|script-title=ja:ロカルちゃ！富山 > Vol.16 鉄道編 > 立山黒部アルペンルート|website=ロカルちゃ！富山|publisher=富山觀光及地域振興局　觀光課|language=ja|accessdate=2018-11-22}}</ref>，也是[[立山黑部阿爾卑斯山脈路線|立山黑部阿爾卑斯山脈路線]]構成路線之一。立山架空索道與[[黑部登山纜車|黑部登山纜車]]、[[立山隧道無軌電車|立山隧道無軌電車]]等載具同屬於鐵路，受《鐵道事業法》規管<ref name="alpen-navi">{{Cite web|url=https://www.alpen-route.com/enjoy_navi/ropeway/|script-title=ja:立山ロープウェイ - 楽しみ方ナビゲート - 立山黒部アルペンルート|publisher=立山黑部阿爾卑斯山脈路線|language=ja|accessdate=2018-11-22}}</ref>。

== 路線資料 ==
; 路線概要
立山架空索道全線位於富山縣中新川郡境內，於1968年10月20日動工<ref name="tkk-co-biz"/>，1970年7月25日開始營業<ref name="tkk-co-biz"/>，並分別1988年和2012年兩次更新車廂<ref name="alpen-veh">{{cite web|url=https://www.alpen-route.com/enjoy_navi/vehicles/#sec5|script-title=ja:のりもの大図鑑 - 楽しみ方ナビゲート - 立山黒部アルペンルート|publisher=立山黑部阿爾卑斯路線|language=ja|accessdate=2018-11-22|archive-url=https://web.archive.org/web/20180718083902/https://www.alpen-route.com/enjoy_navi/vehicles/#sec5|archive-date=2018-07-18|dead-url=no}}</ref>，目前使用的車廂為第三代。路線全長{{convert|1,710|m}}<ref name="tkk-co-biz"/>，水平距離{{convert|1,638|m}}<ref name="tkk-co-biz"/>。

; 運行形態 
立山架空索道全程需時約5.5至7分<ref name="tkk-co-biz"/>，每年冬季（12月1日 - 翌年4月中旬）封山期間暫停營運<ref>{{cite web|url=https://www.japantimeline.jp/jp/0000000642|script-title=ja:【立山黒部アルペンルート】今年の営業は11月30日（日）まで！|author=富山縣|website=JAPAN TIMELINE|language=ja|date=2014-11-26|accessdate=2018-11-17}}</ref>。
