'''立川断层'''（{{lang-ja|立川断層}}）是一条位于[[日本|日本]][[关东地方|关东地方]]的西北－东南向[[活断层|活断层]]，因主要位于[[东京都|东京都]][[立川市|立川市]]而得名。

== 概况 ==
立川断层从东京都[[青梅市|青梅市]]发端，经立川市向[[府中市_(东京都)|府中市]]发展，全长约20千米。立川断层常与发端于其西北端约2千米的[[名栗断层|名栗断层]]被合称为立川断层群<ref name=honbu />。2000年，经贝塚爽平等人分析，立川断层西北部的平均上下变位速度为0.1－0.25米每千年，中部为0.15－0.29米每千年，东南端附近为0.08－0.17米每千年<ref name=bc />。

== 地震可能性评估 ==
日本{{link-ja|地震调查研究推进本部|地震調査研究推進本部}}于2007年发布的立川断层群评估报告中指出，根据经验式<math>\log{L} = 0.6M - 2.9</math>得出，若组成立川断层群的立川断层和名栗断层同时活动，则该断层群发生的最大[[震级|震级]]的地震为7.4级，活动周期约为1万－1.5万年。如果最大震级地震发生的设想成为现实，则相对于西南侧，该断层群的东北侧将隆起2－3米左右<ref name=honbu />。但根据日本{{link-ja|地质调查综合中心|地質調査総合センター}}和[[东京大学地震研究所|东京大学地震研究所]]的分析，立川断层若发生地震，则出现的情况可能为“东北侧相对落下—活动停止—东北侧反弹隆起”<ref>{{Cite journal |url=https://doi.org/10.11462/afr1985.1998.17_54 |script-title=ja:反射法による立川断層の地下構造 |journal=活断層研究 |year=1998 |volume=17 |pages=54-64 |language=ja |access-date=2019-02-21 |doi=10.11462/afr1985.1998.17_54 }}</ref>。

除地震调查研究推进本部的分析外，东京都于2003年自行进行了一次活断层评估。评估委员会认为，立川断层群理论上可能发生的最大规模地震的震级为7.1－7.3级，活动周期约为5000年。同时，报告还指出，一旦这种规模的地震发生，会造成附近地区约6300人遇难<ref>{{Cite web |url=http://www.bousai.metro.tokyo.jp/smart/taisaku/1005822/1005825.html |script-title=ja:東京の活断層 立川断層帯を調査する（平成11年3月） |language=ja |publisher=東京都総務局総合防災部 |access-date=2019-02-21 }}</ref>。

== 后续研究 ==
2015年5月，东京大学地震研究所的佐藤比吕志教授在东京都立川市召开的演讲会上，报告了从2012年开始3年之间对立川断层进行调查的结果<ref>{{Cite web |url=https://www.city.tachikawa.lg.jp/bosai/event/tachikawadannsoukouennkaikaisaihoukoku.html |script-title=ja:立川断層講演会を開催いたしました |publisher=立川市 |language=ja |access-date=2019-02-21 |archive-url=https://web.archive.org/web/20160318195049/http://www.city.tachikawa.lg.jp/bosai/event/tachikawadannsoukouennkaikaisaihoukoku.html |archive-date=2016-03-18 |dead-url=yes }}</ref>。他认为，在此之前推定全长为33千米的断层群中，仅能确认的为位于[[瑞穗町|瑞穗町]]{{link-ja|箱根崎村|箱根ヶ崎村|箱根崎}}的12千米段，同时得出了该断层群不存在于立川市内的结论。因此，在报告书中，他将该断层的名称变更为'''箱根崎断层'''（{{lang-ja|箱根ヶ崎断層}}）<ref>{{Cite news |url=https://www.city.tachikawa.lg.jp/koho/shise/koho/kohotachikawa/h27/documents/062502.pdf |title=広報たちかわ 2015（平成27）年6月25日号2面 |date=2015-06-25 |publisher=立川市役所 広報紙 |language=ja |access-date=2019-02-21 |archive-url=https://web.archive.org/web/20160304082346/https://www.city.tachikawa.lg.jp/koho/shise/koho/kohotachikawa/h27/documents/062502.pdf |archive-date=2016-03-04 |dead-url=no }}</ref><ref>{{Cite news |url=http://s.mxtv.jp/mxnews/kiji.php?date=465195 |publisher=[[TOKYO_MX|TOKYO MX]] |script-title=ja:検証！首都防災 立川断層帯の新事実 被害想定は？ |language=ja |access-date=2019-02-21 |archive-url=https://web.archive.org/web/20170818220625/http://s.mxtv.jp/mxnews/kiji.php?date=465195 |archive-date=2017-08-18 |dead-url=no }}</ref>。
