'''立马关帝庙'''，位于[[北京市|北京市]][[海淀区|海淀区]]西顶路，是[[道教|道教]]宫观，也是[[北京市海淀区文物保护单位|北京市海淀区文物保护单位]]。

==简介==
立马关帝庙位于[[广仁宫|广仁宫]]（西顶）以东，[[金源时代购物中心|金源时代购物中心]]东北，[[京密引水渠|京密引水渠]]西岸、蓝靛厂北路西侧，西顶路北侧，火器营路东侧。该庙始建年代已不可考，[[明朝|明朝]][[嘉靖|嘉靖]]二十二年（1543年）道士李明道募化重建。[[清朝|清朝]][[雍正|雍正]]四年（1726年），道士王本阳自行出资购买庙产，该庙由此成为私庙，后他将该庙传给徒弟黄义通等人。传到王宗岳时，王宗岳未继续收徒，又恐死后自己的亲属侵占庙产，乃将此庙移交给“众善人等”接办<ref name=sohu/>。

[[清朝|清朝]][[光绪|光绪]]年间，[[慈禧太后|慈禧太后]]的大[[太监|太监]][[刘诚印|刘诚印]]主持重修该庙，重修后自光绪七年（1881年）起成为由太监道士掌管，供出宫太监养老的道教庙宇<ref name=ww/>。所有入庙太监须为道教（[[全真道|全真道]]霍山派）信徒，通过拜师入庙，无需缴纳会费，这对宫中地位低、财产少的太监吸引力很强。清朝末代太监[[孙耀庭|孙耀庭]]便居住于此，起道名“孙善福”<ref name=sohu>{{Cite web |url = http://www.sohu.com/a/212468191_391289 |title = 探秘海淀“立马关帝庙”，为何还叫“老公庙”？ |author =  |date = 2017-12-24 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

该庙起初主要靠刘诚印购置的田产、房产等维持。清朝[[宣统|宣统]]年间，因大太监[[崔玉贵|崔玉贵]]捐献680亩香火地，立马关帝庙有了近十六顷田地，大顺庄六顷，六郎庄一顷一，北务一顷六，蓝靛厂四顷多旱田等。一亩地起初收租二斗，后上涨为三斗，一年可收到一百多石地租。另外还有十余亩菜园、数百间房产出租。此外，刘诚印创办的“德元成”酱菜也是该庙的财源<ref name=sohu/>。

该庙祀[[关羽|关羽]]，因山门左侧殿内塑有一匹红色的“[[赤兔马|赤兔马]]”，故得名“立马关帝庙”。立马关帝庙分两进院落，第一进院落有中殿，第二进院落有正殿，鼎盛时期该庙房屋共40多间<ref name=ww>{{Cite web |url = http://whw.bjhd.gov.cn/whyc/hdww/wwbh/201307/t20130722_518891.htm |title = 立马关帝庙 |author =  |date =2013-07-22  |publisher =北京市海淀区文化委员会  |language =  |accessdate =  |quote =  }}</ref>。1936年北京寺庙调查，该庙“不动产房基地三亩七分余，房屋三十九间；附属房基地二十六亩七分，耕地七顷四十二亩余，房屋二十九所，共计三百四十二间半（内坍塌十一间）。”房屋用来“供神、办理学校，自住外出租。”<ref name=sohu/>

立马关帝庙原先有多通石碑，现均无存，仅存两通清朝光绪年间碑文的拓片。第一通是光绪六年（1880年）二月初九日立，记载了该庙的历史。第二通是光绪辛巳七月立，[[潘祖荫|潘祖荫]]书<ref name=sohu/>。

[[中华人民共和国|中华人民共和国]]成立后，该庙收归国有，庙内太监全部迁居[[万寿兴隆寺|万寿兴隆寺]]。该庙成为蓝靛厂中学班的校舍。后又曾作为盲人五金厂职工宿舍<ref name=sohu/>。在2017年腾退前，立马关帝庙长期由世纪金源公司占用，部分房屋被金源时代购物中心当做保安宿舍。2001年11月，海淀区人民政府公布立马关帝庙为海淀区重点文物保护单位。2002年4月，海淀区文化委员会立海淀区重点文物保护单位牌<ref>{{Cite web |url = http://news.163.com/09/1021/07/5M4OA50P000120GR.html |title =重点文保单位立马关帝庙咋无人管  |author =  |date =2009-10-21  |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。

2017年9月15日，[[北京市道教协会|北京市道教协会]]与世纪金源公司签订立马关帝庙腾退移交协议书，9月25日北京市道教协会派人员进驻立马关帝庙。10月10日，76户职工全部清退，北京市道教协会全面接收该庙，北京市道教协会立马关帝庙修复筹备工作组当天在该庙挂牌成立。10月11日，[[北京市宗教事务局|北京市宗教局]]副局长[[范宝|范宝]]、北京市海淀区人民政府副区长[[陈双|陈双]]召集北京市和海淀区相关单位来到该庙现场调研，北京市道教协会会长[[黄信阳|黄信阳]]，北京市宗教局落政办、北京市道教协会秘书处、北京市海淀区民宗侨办和[[曙光街道_(北京市)|曙光街道]]负责人随同参加调研<ref>{{Cite web |url = http://www.bjethnic.gov.cn/xinwenzhongxin/gzdt/20171016/4569.shtml |title = 道教立马关帝庙移交工作顺利完成 |author =  |date =2017-10-16  |publisher = 北京市民族事务委员会 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.daoisms.org/article/sort028/info-32803.html |title =  北京市道教协会全面接收立马关帝庙|author =  |date =2017-10-17   |publisher = 道教之音 |language =  |accessdate =  |quote =  }}</ref>。10月23日，曙光街道党工委书记冀国瑞、曙光街道办事处主任胡宏宇、办事处副主任段泽明会见北京市道教协会人员，会商该庙后期恢复原址原貌事宜<ref>{{Cite web |url = https://www.sohu.com/a/203966932_99960317 |title = 曙光街道同道教协会会商立马关帝庙相关事宜 |author =  |date = 2017-11-12 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。
