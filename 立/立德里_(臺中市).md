'''立德里'''，為臺灣臺中市大里區轄下的一個里<ref>{{cite book|title=臺灣全志: 政治志·建置沿革篇|year=2007|publisher=國史館臺灣文獻館}}</ref>，位於大里區中央偏東，為臺中市郊的住宅區，多為連棟式透天。

== 里名由來 ==
本境原屬「立仁里」管轄，建里時，以立仁里的「立」字，另取雅字「德」字，定名為「立德里」。

== 地理 ==
立德里面積0.5926平方公里，劃分為30鄰。本里東隔立仁路與[[新仁里_(大里區)|新仁里]]為界，西以[[大里溪|大里溪]]界[[中新里_(大里區)|中新里]]，南隔[[頭汴坑溪|頭汴坑溪]]與[[仁化里|仁化里]]對望，北為[[立仁里_(大里區)|立仁里]]。

== 行政區劃沿革 ==
立德里原屬大里市立仁里管轄。民國91年（2002年），調整行政區，以立仁里南境置「'''立德里'''」。
