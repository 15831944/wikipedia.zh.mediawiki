'''立陶宛共和國'''（{{lang-lt|Lietuvos Respublika}}），通稱'''立陶宛'''（{{IPAc-en|audio=en-us-Lithuania.ogg|ˌ|l|ɪ|θ|juː|ˈ|eɪ|n|i|ə}}; {{lang-lt|Lietuva}}; {{IPA-lt|ˈlʲɛtʲʊvaː|}}），位于欧洲东北部<ref>{{cite web |title=Composition of macro geographical (continental) regions, geographical sub-regions, and selected economic and other groupings |publisher=联合国统计司|url=http://unstats.un.org/unsd/methods/m49/m49regin.htm#europe |accessdate=2008-11-09}}</ref>，是[[北欧|北欧]]和[[波罗的海三國|波罗的海三國]]之一，首都[[维尔纽斯|维尔纽斯]]。

立陶宛北接[[拉脱维亚|拉脱维亚]]、东南接[[白俄罗斯|白俄罗斯]]、南接[[波兰|波兰]]、西南接[[俄罗斯|俄罗斯]]的[[加里宁格勒州|-{zh-hans:加里宁格勒; zh-hant:加里寧格勒;}-州]]，西濱[[波羅的海|波羅的海]]。

立陶宛是[[北欧理事会|北欧理事会]]、[[欧洲联盟|欧洲联盟]]、[[北大西洋公约组织|北大西洋公约组织]]、[[欧洲委员会|欧洲委员会]]、[[联合国|联合国]]、[[波羅的海國家理事會|波羅的海國家理事會]]、[[國際貨幣基金組織|國際貨幣基金組織]]、[[北歐投資銀行|北歐投資銀行]]、[[歐洲安全與合作組織|歐洲安全與合作組織]]、[[申根公约|申根公约]]及[[欧元区|欧元区]]等國際机构組織的成員國，立陶宛属于[[高收入經濟体|高收入經濟体]]并是较高[[人类发展指数|人类发展指数]]国家。

== 历史 ==
[[File:LithuanianHistory.svg|225px]]
{{Main|立陶宛歷史}}
[[立陶宛人|立陶宛人]]屬遠古[[印歐語系|印歐語系]]的一支，於公元前即已來到[[波羅的海|波羅的海]]沿岸定居。12世紀起受[[日耳曼人|日耳曼人]]侵略。1240年[[立陶宛大公國|立陶宛大公國]]成立。

1370年[[波蘭王室|波蘭王室]]絕嗣，1386年波蘭公主[[雅德維加|雅德維加]]嫁給立陶宛大公[[約蓋拉|約蓋拉]]，兩國的結合使國力倍增成為歐洲大國。1569年，立陶宛與波蘭締結[[盧布林聯合|盧布林聯合]]正式合併，成立[[波蘭立陶宛聯邦|波蘭立陶宛聯邦]]。由於對外長期戰爭，加上國內動亂不止，國力迅速衰退，最終在1772年、1793年和1795年為[[俄羅斯|俄羅斯]]、[[普魯士|普魯士]]及[[奧地利|奧地利]]的三次瓜分後滅亡。立陶宛大公國的領土，自此成為[[沙俄|沙俄]]版圖的一部份。

[[第一次世界大战|第一次世界大战]]期间，立陶宛被[[德意志帝國|德意志帝國]]佔領。1917年俄国接連爆发[[二月革命|二月革命]]及[[十月革命|十月革命]]，最後由[[列寧|列寧]]主導成立[[蘇維埃政權|蘇維埃政權]]。1918年2月16日，立陶宛宣佈獨立。

1939年德蘇簽訂《[[苏德互不侵犯條約|苏德互不侵犯條約]]》，立陶宛被划入苏联势力范围，1940年6月14日，即德军进入[[巴黎|巴黎]]的当天，苏联称立陶宛政府“破坏《苏立互助条约》”，“进行反苏活动”，要求“马上逮捕挑衅祸首”，保证苏军在立陶宛境内自由通行，在战略要地驻军，立陶宛被迫应允，苏军遂於15日进驻立陶宛<ref>{{cite web|title=波罗的海三国：再对俄罗斯翻“老账”|url=http://news.xinhuanet.com/world/2005-06/22/content_3120348.htm|publisher=新华网|date=2005-06-22}}</ref> 。1941年[[德蘇戰争|德蘇戰争]]爆發，德軍迅速佔領了立陶宛。並與德國佔领下的拉脱維亞、爱沙尼亞及白俄羅斯部份地區合併為[[奥斯蘭總督轄區|奥斯蘭總督轄區]]。

[[第二次世界大战|第二次世界大战]]之后，立陶宛被[[苏联|苏联]]吞并。1990年3月11日立陶宛宣布独立，直到1991年9月6日苏联才承认其独立。

2004年5月1日，立陶宛正式加入欧盟。

2006年5月30日，阿达姆库斯总统对来自[[劳动党|劳动党]]的两名涉嫌腐败的部长表示不信任，劳动党因此于5月31日宣布退出执政联盟。6月1日，总理布拉藻斯卡斯向总统递交辞呈并获批准。担任财政部长的[[巴尔齐蒂斯|巴尔齐蒂斯]]被阿达姆库斯任命为代总理<ref>{{cite news|title=立陶宛总统向议会提名总理候选人|url=http://news.xinhuanet.com/newscenter/2006-06/15/content_4703830.htm|publisher=新华网|date=2006-06-15}}</ref>。

2015年1月1日，立陶宛正式成为[[欧元区|欧元区]]第19个成员国。

== 政治 ==
{{Main|立陶宛政治}}
[[File:Seimas_litauensparlament.jpg|thumb]]]]
立陶宛政制為[[半总统制|半總統制]]（semi-presidentialism），總統由全民直選產生，任期5年，最多可任兩屆。凡年齡在40歲以上，且居住滿3年以上者，均可參選總統。

若總統未能履行其職務時，由國會議長代理。

總理由總統提名，經國會同意任命。

國會稱為[[立陶宛议会|塞瑪斯]]，141席，任期4年，其中71席由人民選出，70席由政黨得票比率分配。

== 经济 ==
[[File:Vilnius_city.jpg|thumb]]
[[File:Klaipeda_Port_20060913_01.jpg|thumb]]港]]
[[File:Lithuania_Klaipeda_1.jpg|thumb]]郊區旅店]]
* 2015年1月1日起，使用[[歐元|歐元]]。
* 国内有3个自由经济区：[[考纳斯|考纳斯]]、[[克莱佩达|克莱佩达]]和希奥利艾。
* 有欧洲标准的公路网，4个国际机场，不冻港（东波罗的海岸）和该地区第一个卫星通讯系统。
* 盛产[[琥珀|琥珀]]，有少量的粘土、砂石、石灰、石膏、泥炭、铁矿石、磷灰石及石油，所需石油和天然气靠进口。西部沿海地区发现有少量石油和天然气资源，但储量尚未探明。
* 森林面积197.55万公顷，森林覆盖率为30％以上。多野生动物，有60多种哺乳动物、300多种鸟类和50多种鱼类。
* 工业是立陶宛的支柱产业，主要由矿业及采石业、加工制造业以及能源工业3大部门组成。工业门类比较齐全，以食品、木材加工、纺织、化工等为主，机械制造、化工、石油化工、电子工业、金属加工工业等发展迅速，生产的高精度机床、仪表、电子计算机等产品行销全世界80多个国家和地区。
* 首都[[维尔纽斯|维尔纽斯]]是全国工业中心，全市工业产值占立陶宛工业总产值的2/3以上。
* 农业以水平较高的畜牧业为主，占农产品产值的90％以上。农作物有[[亚麻|亚麻]]、马铃薯、甜菜和各种蔬菜，谷物产量很低。

== 人口 ==
立陶宛总人口为298.8（2012年9月）万，其中立陶宛族人占84.2%，波兰族人占6.6%、俄罗斯族人占5.8%、白俄罗斯族占1.2%。

主要的宗教为[[天主教|天主教]]，不过也有少数地区信仰[[东正教|东正教]]、[[新教|新教]]或[[犹太教|犹太教]]。有部分人口是[[立陶宛韃靼人|立陶宛韃靼人]]，是[[脫脫迷失|脫脫迷失]]被帖木兒打敗時，被[[立陶宛大公|立陶宛大公]]收留而形成。有15000人，多是[[穆斯林|穆斯林]]。

== 能源 ==
立陶宛曾经是世界上[[核能|核能]]发电占全国总发电比例最高的国家。据[[国际原子能机构|国际原子能机构]]统计，截至2002年底，立陶宛的核能发电在全国总发电比例中占到了80.1%。不過在2009年12月31日最後一個小時前，立陶宛關閉了其最後一座[[伊格納利納核電站|伊格納利納核電站]]（{{lang|lt|Ignalinos atominė elektrinė}}），境內二個[[核電站|核電站]]分別於2004年及2009年先後關閉，成為全世界第一個完全脫離核電的國家。這是由於立陶宛要加入[[歐盟|歐盟]]的條件之一，就是需關閉蘇聯時期所建造、採用[[RBMK|RBMK反應爐]]之核電廠；為此寻找新的替代能源非常重要，立陶宛原計畫與愛沙尼亞、拉脫維亞及波蘭合資在伊格納利納核電站附近興建一座採用[[ABWR|ABWR反應爐]]的[[維薩基納斯核電站|維薩基納斯核電站]]（{{lang|lt|Visagino atominė elektrinė}}），但因2012年10月的公投遭63%反對票所反對，而中止建廠計劃，故改計劃建造[[天然氣發電廠|天然氣發電廠]]，預計在2013年以前完工，不過它還是無法滿足現有用電需求，仍得從[[俄羅斯|俄羅斯]]、[[愛沙尼亞|愛沙尼亞]]、[[烏克蘭|烏克蘭]]和[[白俄羅斯|白俄羅斯]]等國家採購能源，彌補用電缺口。
<ref>[http://news.chinatimes.com/2007Cti/2007Cti-News/2007Cti-News-Content/0,4521,50402510+132010010100522,00.html 立陶宛關閉核電廠 完全脫離核能]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 体育 ==
该国主要体育活动为[[篮球|篮球]]。立陶宛国家男子篮球队现世界排名第3位。作为一支欧洲劲旅，曾经三次获得欧洲锦标赛冠军，曾四次参加奥运会男篮比赛，三次获得铜牌。中国男篮前主教练[[尤纳斯·卡兹劳斯卡斯|尤纳斯]]也是立陶宛人。立陶宛队中曾涌现出克莱扎、马奇扬斯卡斯和桑盖拉等NBA球星。作为老牌的欧洲劲旅，立陶宛男篮秉承了传统的欧洲实用派打法，同样以整体打法为特点。队员技术娴熟、配合意识良好并善于打硬仗。<ref>{{cite web|title=立陶宛籃球隊簡介|url=http://wenwen.soso.com/z/q117794355.htm|date=2009-02-07|deadurl=yes|archiveurl=https://web.archive.org/web/20111105111350/http://wenwen.soso.com/z/q117794355.htm|archivedate=2011-11-05}}</ref>
