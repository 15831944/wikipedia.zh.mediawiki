在[[幾何學|幾何學]]中，'''立方半八面體'''是一種非凹多面體，屬於星形多面體及[[均勻多面體|均勻多面體]]，也可以歸類在非凸均勻多面體，其索引在均勻多面中是U<sub>15</sub><ref>{{cite web | url = https://archive.lib.msu.edu/crcmath/math/math/c/c835.htm | title = Cubohemioctahedron | publisher = [[密西根州立大學|密西根州立大學]][[圖書館|圖書館]] | archiveurl = https://web.archive.org/web/20130620081222/http://archive.lib.msu.edu/crcmath/math/math/c/c835.htm | archivedate = 2013-06-20}}</ref>、[[溫尼爾多面體模型列表|溫尼爾多面體模型]]中是W<sub>78</sub><ref>{{cite web| archiveurl = https://web.archive.org/web/20160831133741/http://colinspics.org/life/maths/polyhedra/W78%20Cubohemioctahedron.htm | url = http://colinspics.org/life/maths/polyhedra/W78%20Cubohemioctahedron.htm | archivedate = 2016-08-31 | title = W<sub>78</sub> Cubohemioctahedron | publisher = colinspics}}</ref>。立方半八面體外觀看起來像所有[[三角形|三角形]]面都凹進去的[[截半立方體|截半立方體]]。

== 性質 ==
立方半八面體共有10個面、24條邊和12個[[頂點|頂點]]<ref>{{cite web | url = https://www.mathconsult.ch/static/unipoly/15.html | title = Uniform Polyhedra 15: Cubohemioctahedron | publisher = mathconsult | archiveurl = https://web.archive.org/web/20160327004546/http://www.mathconsult.ch/static/unipoly/15.html | archivedate = 2016-03-27}}</ref><ref>{{cite web | url = http://bulatov.org/polyhedra/uniform/u20.html | title = Cubohemioctahedron | publisher = bulatov.org | archiveurl = https://web.archive.org/web/20160401210345/http://bulatov.org/polyhedra/uniform/u20.html | archivedate = 2016-04-01}}</ref>，每個頂點都是2個[[正方形|正方形]]和2個六邊形的公共[[頂點|頂點]]。

== 面的組成 ==
立方半八面體由10個面組成，在其十個面中，有6個正方形面和4個六邊形面，其中的4四個六邊形面互相相交，且皆穿過該立體的[[幾何中心|幾何中心]]。
:[[File:Cuboctahedron_equator.png|200px]]

=== 二面角 ===
八面半八面體[[二面角|二面角]]為[[3的算術平方根|三平方根]]的[[倒數|倒數]]之[[反餘弦|反餘弦]]值<ref>{{cite web | url = http://dmccooey.com/polyhedra/Cubohemioctahedron.html | title = Versi-Regular Polyhedra: Cubohemioctahedron|publisher=dmccooey.com| archiveurl = https://web.archive.org/web/20160324081711/http://dmccooey.com/polyhedra/Cubohemioctahedron.html | archivedate = 2016-03-24}}</ref><ref>Jean Paul Albert Badoureau, Mémoire sur les Figures Isocèles,
Journal de l'École polytechnique 49 (1881), 47-172.</ref>：
:<math>\cos^{-1} (\frac{\sqrt{3}}{3}) \approx 0.9553166 \approx 54.7356 ^{\circ}</math>

=== 頂點座標 ===
由於立方半八面體[[凸包|凸包]]為[[截半立方體|截半立方體]]，因此其12[[頂點|頂點]]會與[[截半立方體|截半立方體]]相同，為{{nowrap|(0, ±1, ±1)}}、 {{nowrap|(±1, 0, ±1)}}, {{nowrap|(±1, ±1, 0)}}，若邊長為a，則[[座標|座標]]要[[縮放|縮放]]<math>\frac{\sqrt{2}}{2}a</math>倍。
