'''立山'''（{{jpn|j=立山|hg=たてやま}}）是[[日本|日本]][[富山縣|富山縣]][[飛驒山脈|飛驒山脈]]（[[北阿爾卑斯|北阿爾卑斯]]）北部{{link-ja|立山連峰|立山連峰}}的主峰，為[[中部山岳國立公園|中部山岳國立公園]]的代表山岳之一<ref name="park">[http://www.env.go.jp/park/chubu/intro/index.html 中部山岳国立公園]（[[環境省|環境省]]）、2010年12月16日閲覧。</ref>，也是[[日本三名山|日本三名山]]與[[日本百名山|日本百名山]]之一。立山是'''雄山'''（{{jpn|j=雄山|hg=おやま}}，[[標高|標高]]3,003 m）、'''大汝山'''（{{jpn|j=大汝山|hg=おおなんじやま}}，標高3,015 m）、'''富士折立'''（{{jpn|j=富士ノ折立|hg=ふじのおりたて}}，標高2,999 m）三座山峰的總稱。雖然有時也單指雄山，但嚴格上來說立山連峰沒有單稱為立山的山峰。立山與[[劍岳|劍岳]]是日本現在少數有[[冰河|冰河]]的山<ref name="ice">{{Cite web |url=http://mainichi.jp/select/news/20120404k0000m040155000c.html |title=立山連峰：日本初の「氷河」、学会が認定 |publisher=[[每日新聞|每日新聞]] |date=2012-04-04 |accessdate=2012-04-04}}</ref>。

== 概要 ==
[[File:Turugitateyama.JPG|thumb]]
「立山」非單獨的地理名稱，而是指{{link-ja|室堂|室堂}}與地獄谷、彌陀原、立山火山口等地的廣域地理名，同時也包含立山信仰與遙拜登山等精神。過去山體在{{link-ja|立山火山口|立山カルデラ}}，但其山頂已因侵蝕而消失。其火山熔岩與堆積物形成了台地{{link-ja|彌陀原 (立山)|弥陀ヶ原 (立山)|彌陀原}}與{{link-ja|五色原 (立山連峰)|五色ヶ原 (立山連峰)|五色原}}，而御庫裏池（{{lang|ja|ミクリガ池}}）、綠池（{{lang|ja|ミドリガ池}}）則是[[火口湖|火口湖]]。現在立山火山的主要火山活動在{{link-ja|地獄谷溫泉 (富山縣)|地獄谷温泉 (富山県)|地獄谷}}周邊的[[火山氣體|火山氣體]]噴發與溫泉。

入選[[日本三名山|日本三名山]]、[[日本百名山|日本百名山]]<ref name="fukada">『日本百名山』深田久弥（著）、朝日新聞社、1982年、ISBN 4-02-260871-4、pp.188-191</ref>、{{link-ja|新日本百名山|新日本百名山}}<ref>『新日本百名山登山ガイド〈上〉』 岩崎元郎（著）、山と渓谷社、2006年、ISBN 4-635-53046-9、pp148-150</ref>及{{link-ja|花之百名山|花の百名山}}<ref name="f100">『花の百名山』田中澄江（著）、文春文庫、1997年、ISBN 4-16-352790-7、P213-216。該書介紹代表花{{link-ja|岩銀杏|イワイチョウ}}。</ref>的立山是[[富山縣|富山縣]]的象徵之一。

雄山山頂有{{link-ja|雄山神社|雄山神社}}本宮。峰本社神殿右端前方有做為測量基準的大型黑色[[花崗岩|御影石]]標石（標高點3,003 m）<ref name="toyama">『改訂版 富山県の山』山と渓谷社、2010年、ISBN 978-4-635-02367-2、pp.16-19</ref>，其南南西方約70公尺處有[[三角點|一等三角點]]（標高2,991.59 m，點名為立山）標石<ref>[http://sokuseikagis1.gsi.go.jp/ 基準点成果等閲覧サービス]、国土地理院、2010年12月16日閲覧。</ref><ref>雄山是『一等三角點百名山』（山と渓谷社、1988年、ISBN  4-635-17030-6 ）。</ref>

立山在[[萬葉集|萬葉集]]中寫為「多知夜麻」<ref>万葉集 巻17、4000、4024</ref>。

== 地理 ==
[[File:Mount_Tate_and_Tsurugi_from_Mount_Goryu.jpg|thumb]]]]
[[File:Tateyama_from_Raicho-zawa.JPG|thumb]]
立山是日本3,000公尺以上高山中的最北端，也是富山縣最高峰、以及[[北陸|北陸]]4縣與環日本海地區最高峰。
; 御前澤冰河
: 位於雄山東側斜面的御前澤冰河全長約700公尺，面積約0.1平方公里。是日本少數現存的冰河，為{{link-ja|黑部川|黒部川}}水源之一<ref name="ice" />。
; 山崎圈谷
: {{link-ja|山崎圈谷|山崎圏谷}}位於雄山西側斜面。1905年（[[明治|明治]]38年）由地理學者、帝國大學理科大學（現[[東京大學|東京大學]]）教授{{link-ja|山崎直方|山崎直方}}博士發現，是日本首次發現的冰河地形。1945年（[[昭和|昭和]]20年）與藥師岳東側圈谷群<ref>1952年（昭和27年）指定為特別天然紀念物。</ref>一同指定為國家[[天然紀念物|天然紀念物]]。

== 自然 ==
冬季有[[西伯利亞|西伯利亞]]寒風帶來日本海水氣，為立山一帶帶來大量降雪，是世界知名的豪雪地帶，最低溫在零下20[[℃|℃]]以下。

室堂周邊往上是[[林木線|森林界限]]的[[偃松|偃松]]帶，有許多[[岩雷鳥|岩雷鳥]]棲息。作為{{link-ja|花之百名山|花の百名山}}之一，室堂周邊在融雪後可見許多[[高山植物|高山植物]]開花，雖然有[[叢生龍膽|立山龍膽]]、{{link-ja|立山靫草|タテヤマウツボグサ}}、立山薊（{{lang|ja|タテヤマアザミ}}）、立山金梅（{{lang|ja|タテヤマキンバイ}}）、立山黃耆（{{lang|ja|タテヤマオウギ}}）等以立山命名的高山植物，但皆非立山的[[特有種|特有種]]。

由於登山者與觀光客的增加，為室堂周邊帶來外來植物與病原菌，對當地生態系造成嚴重威脅。為保護當地環境，立山是日本國內先行禁止私家車進入（[[My_car規制|My car規制]]）的地區之一，現在也持續有{{link-ja|割草十字軍|草刈十字軍}}等民間志工團體推動環境保護活動。

== 人類歷史 ==
* 701年（[[大寶_(日本)|大寶]]元年） - {{link-ja|佐伯有賴|佐伯有頼}}（慈興上人）{{link-ja|開山 (佛教)|開山 (仏教)|開山}}<ref name="fukada" />。（白鷹傳說）
* 1695年（[[元祿|元祿]]8年） - [[加賀藩|加賀藩主]]在{{link-ja|室堂平|室堂平}}為了立山寺（立山權現）參拜者設置參籠所（現在室堂山莊的前身）<ref name="sanmei">『コンサイエンス日本山名辞典』三省堂、1992年、ISBN 4-385-15403-1、P326-327</ref>。<!--室堂小屋は信仰施設であり、単純に「日本最古の山小屋」と言い切ってしまうのは誤解を招く。下の「1995年」の項を参照。-->[[江戶時代|江戶時代]]由於立山信仰興盛，有許多信眾登上立山參拜。
* 1858年（[[安政|安政]]5年）4月9日（新曆） - {{link-ja|飛越地震|飛越地震|安政飛越地震}}，原為[[鳶山|鳶山]]一部分的大鳶山與小鳶山崩落消失，大量土石流入{{link-ja|立山火山口|立山カルデラ}}<ref name="sangaku">『新日本山岳誌』ナカニシヤ出版、2005年、ISBN 4-779-50000-1、P914-916</ref>。
* 1872年（[[明治|明治]]5年） - [[太政官|太政官]]通達，神社佛閣地解除[[女人禁制|女人禁制]]。在此之前的女性只能至{{link-ja|蘆峅寺|芦峅寺}}的姥堂參拜。
* 1880年（明治13年） - {{link-ja|立山新道|立山新道}}開通，連結[[大町市|大町市]]野口 - {{link-ja|針之木峠|針ノ木峠}} - ザラ峠 - 富山市原村。
* 1894年（明治27年） - {{link-ja|陸地測量部|陸地測量部}}的{{link-ja|館潔彥|館潔彦}}前往立山進行測量。
* 1934年（[[昭和|昭和]]9年）12月4日 - 劃入中部山岳國立公園，此山域為特別保護地區<ref name="park" />。
* 1947年（昭和22年） - 佐伯房治於{{link-ja|地獄谷溫泉 (富山縣)|地獄谷温泉 (富山県)|地獄谷溫泉}}建設房治小屋<ref name="koya">『北アルプス山小屋物語』[[東京新聞|東京新聞]]出版局、1990年、ISBN 4-8083-0374-4、P161-178</ref>。
* 1953年（昭和28年）6月15日 - 立山頂上郵便局於室堂開設<ref name="kokuji770">1953年（昭和28年）6月15日郵政省告示第770号「白山頂上郵便局等設置」</ref>。同年7月5日開始業務<ref name="kokuji770"/>。
* 1954年（昭和29年） - [[富山地方鐵道|富山地方鐵道]]{{link-ja|立山站|立山駅}}至{{link-ja|美女平站|美女平駅}}的[[立山黑部貫光立山纜車|立山纜車]]開業。
* 1963年（昭和38年） - [[關西電力|關西電力]]{{link-ja|黑部川第四發電所|黒部川第四発電所}}完成。
* 1970年（昭和45年）7月10日 - 立山頂上郵便局改名為立山山頂郵便局<ref>1970年（昭和45年）6月4日郵政省告示第480号「郵便局改称の件」</ref>。
* 1971年（昭和46年）6月1日 - [[立山黑部阿爾卑斯山脈路線|立山黑部阿爾卑斯山脈路線]]全線開通。
* 1981年（昭和56年）7月18日 - {{link-ja|立山山頂簡易郵便局|立山山頂簡易郵便局}}於室堂開設<ref name="kokuji534">1981年（昭和56年）7月18日郵政省告示第534号「簡易郵便局を設置する件」</ref>。同年7月20日開始業務<ref name="kokuji534"/>。
* 1981年（昭和56年）7月20日 - 立山山頂郵便局廢止，由小見郵便局繼承業務<ref>1981年（昭和56年）6月17日郵政省告示第423号「郵便局を廃止する件」</ref>。
* 1986年（昭和61年） - 在此之前作為山間小屋的{{link-ja|室堂小屋|室堂小屋}}（立山室堂山莊）指定為富山縣指定有形文化財，翌年西側建造新的山間小屋<ref name="koya" />。
* 1995年（[[平成|平成]]7年）6月27日 - 進行拆解修護的室堂小屋指定為國家[[重要文化財|重要文化財]]<ref>1995年（平成7年）6月27日文部省告示第100号「文化財を重要文化財に指定する件」</ref>。是日本最古老的山間小屋。
* 2000年（平成12年）4月28日 - [[郵政省|郵政省]]發售『立山連峰與[[鬱金香|鬱金香]]田·富山縣』的50日圓郵票<ref>2000年（平成12年）4月11日郵政省告示第239号「ふるさと八十円郵便切手及び五十円郵便切手（北陸）を発行する件」</ref><ref>『切手と風景印でたどる百名山』ふくろう舎、2007年、ISBN 978-4-89806-276-0、P92</ref>。
* 2008年（平成20年）8月 - 縣、地方市町以「立山、黑部〜防災大國日本的典範―信仰、砂防、發電―〜」的名義，申請立山進入[[文化廳|文化廳]]的[[世界遺產|世界遺產]]推薦名單，但被「未有國際評價」的理由延後<ref>富山新聞2010年2月7日</ref>。
* 2012年（平成24年）6月- {{link-ja|日本冰雪協會|日本雪氷学会}}在御前澤雪溪發現的冰體確認是日本稀少的現存[[冰河|冰河]]。
* 2012年（平成24年）7月3日 - 彌陀原與{{link-ja|大日平|大日平}}登錄[[拉姆薩公約|拉姆薩公約]]。
* 2014年（平成26年）8月28日 - {{link-ja|立山黑部地質公園|立山黒部ジオパーク}}登錄[[日本地質公園|日本地質公園]]。
