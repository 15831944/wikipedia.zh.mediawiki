'''电池'''，一般狹義上的定義是將本身儲存的[[化學能|化學能]]轉成[[電能|電能]]的裝置，廣義的定義為將預先'''儲存'''起的能量'''轉化'''為可供外用電能的裝置<ref name="Battery">{{cite web |url=https://batt.lbl.gov/what-is-a-battery/ |title=Feedline:WHAT IS A BATTERY?。 |deadurl=yes |archiveurl=https://web.archive.org/web/20130406165634/https://batt.lbl.gov/what-is-a-battery/ |archivedate=2013-04-06 }}</ref>。因此，像太陽能電池只有轉化而無儲存功能的裝置不算是電池。其他名稱有'''電瓶'''、'''電芯'''，而中文「池」及「瓶」也有儲存作用之意。

英文中，單一個電池結構叫做「Cell」（單電池），內部有多個Cell並連或串連的結構叫做「Battery Cell」（電池組）。市售一般乾電池其實構造上是「Cell」但英文上習慣稱「Battery」，汽車用[[鉛酸電池|鉛酸電池]]與方形[[9V電池|9V電池]]則是真正的「Battery」。

== 電池發展史 ==

電池的[[歷史|歷史]]可以追溯到兩千多年前的古[[伊拉克|伊拉克]][[時代|時代]]。是在首都[[巴格達|巴格達]]發現的[[素燒|素燒]][[陶壺|陶壺]]（[[巴格達電池|巴格達電池]]）它是一種使用[[銅|銅]]和[[鐵|鐵]]的電池。[[考古|考古]]者[[發現|發現]]的物品被認為是至今發現的最早的電池[[證據|證據]]。
而[[現代|現代]]真正作為化學能的儲藏體，根據人們的需要可控制地放出電能的[[裝置|裝置]]，首先由[[亞歷山德羅·伏特|亞歷山德羅·伏特]][[發明|發明]]，當時稱為Volta Pile（[[伏打電堆|伏打電堆]]）。

== 電池種類 ==
=== 化學電池 ===
'''[[電化電池|化学电池]]'''、'''電化電池'''、'''電化學電池'''或'''电化学池'''是指通过[[氧化还原反應|氧化还原反應]]，把[[正极|正极]]、[[负极|负极]][[活性物质|活性物质]]的化学能，转化为电能的一类装置。与普通氧化还原反应不同的是[[氧化|氧化]]和[[还原|还原]]反应是分开进行的，氧化在负极，还原在正极，而[[电子|电子]]得失是通过外部[[线路|线路]]进行的，所以形成了电流。这是所有电池的本质特点。经过长期的研究、发展，化学电池迎来了品种繁多，应用广泛的局面。大到一座[[建筑|建筑]]方能[[容纳|容纳]]得下的巨大装置，小到以毫米计的類型。现代[[电子技术|电子技术]]的发展，对化学电池提出了很高的要求。每一次化学电池技术的突破，都带来了[[电子设备|电子设备]][[革命|革命]]性的发展。世界上很多电化学[[科学家|科学家]]，把兴趣集中在做为[[电动汽车动力|电动汽车动力]]的化学电池领域。
[[File:Fotothek_df_n-10_0000553.jpg|缩略图]]]]
==== 乾電池和液體電池 ====
[[乾電池|乾電池]]和[[液体电池|液体电池]]的区分仅限于早期电池发展的那段[[时期|时期]]。最早的电池由装满[[电解液|电解液]]的[[玻璃|玻璃]]容器和两个电极组成。后来推出了以糊状[[电解液|电解液]]为基础的电池，也称做乾電池。

现在仍然有“[[液体|液体]]”电池。一般是[[体积|体积]]非常庞大的品种。如那些做为[[不间断电源|不间断电源]]的大型固定型[[鉛酸蓄電池|鉛酸蓄電池]]或與[[太阳能電池|太阳能電池]]配套使用的铅酸蓄电池。对于[[移动设备|移动设备]]，有些使用的是全[[密封|密封]]，免维护的[[铅酸蓄电池|铅酸蓄电池]]，这类电池已经成功使用了许多年，其中的电解液[[硫酸|硫酸]]是由[[硅凝胶|硅凝胶]]固定。

==== 一次性電池和可充電電池 ====

;一次性電池
[[原電池|一次性電池]](Primary Battery)俗稱「用完即棄」電池及原電池，因為它們的[[電量|電量]]耗盡後，無法再充電使用，只能丟棄。常見的一次性電池包括：<br />
* [[鋅錳電池|鋅錳電池]]—電壓約1.5V，電池容量較低，能輸出的電流也較低，幾乎被鹼錳電池所取代，唯獨是不會在長期存放後漏出有害腐蝕液體，所以仍被使用於低用電量同時需長期使用的裝置，例如鐘、紅外線搖控等。
* [[鹼錳電池|鹼錳電池]]—電壓約1.5V，電池容量及輸出的電池較鋅錳電池高，但不及[[鎳氫電池|鎳氫電池]]，長期存放後漏出有害腐蝕液體。
* [[鋰電池|鋰電池]]Li—電壓約3V，電池容量及輸出的電池極高，可以存放十年仍有相當電力，但價錢較貴。
其他的一次性电池包括有[[鋅電池|鋅電池]]、[[鋅空電池|鋅空電池]]、[[鋅汞電池|鋅汞電池]]、[[水銀電池|水銀電池]]、[[氫氧電池|氫氧電池]]和[[鎂錳電池|鎂錳電池]]。

;可充電電池
[[蓄電池|可充電電池]]又稱二次電池(Secondary Battery)或二級電池、[[蓄電池|蓄電池]]。可充電電池按製作[[材料|材料]]和[[工藝|工藝]]上的不同，其優點是在充電後可多次循環使用，它們可全充[[放電|放電]]兩百多次甚至達2500次，充電電池的輸出電流[[負荷力|負荷力]]要比大部分一次性電池高。常見的類型有：<br />
* [[鉛酸蓄電池|鉛酸電池]]—每個Cell的電壓約2V，容量低但可輸出較大的功率、電池，常使用於汽車中作啟動引擎用，或用於不斷電系統(UPS)、無線電機、通信機。
* [[鎳鎘電池|鎳鎘電池]]NiCd—電壓約1.2V，有較強烈的[[記憶效應_(電池)|記憶效應]]，而且容量較低，含有毒物質，對環境有害，現已被淘汰。
* [[鎳氫電池|鎳氫電池]]NiMH—電壓約1.2V，有極輕微的記憶效應，容量較鎳鎘電池及鹼性電池大，可充放電循環使用數百至二千幾次。舊鎳氫電池有較大的自放電，新的低自放電鎳氫電池自放電低至與鹼性電池相約，而且可在低溫下使用(-20℃)，充電裝置、電壓與鎳鎘電池相同，已取代了鎳鎘電池，同時也可取代絕大部份鹼性電池的用途，也有用於[[混合動力車|混合動力車]]的。
* [[鋰離子電池|鋰離子電池]]Li-ion—電壓約3.6V、3.7V，鋰離子電池具有重量輕（容量是同重量的鎳氫電池的1.5倍~2倍）、容量大、無記憶效應等優點，具有很低的自放電率，因而即使價格相對較高，仍然得到了普遍應用，包括許多電子產品，而且不含有毒物質，但這類用於[[消費性電子產品|消費性電子產品]]的Li-ion電池在存放一段時期後電量會永久減少。另也有用於[[純電動車|純電動車]]及[[混合動力車|混合動力車]]的Li-ion電池，用於這用途的鋰離子電池容量相對略低，但有較大的輸出、充電電流，也有的有較長的壽命，但成本較高。

==== 燃料電池 ====
[[File:Microbial_electrolysis_cell.png|thumb]]
[[燃料電池|燃料電池]]是一種将[[燃料|燃料]]的化學能透過電化學反應直接轉化成電能的裝置，與一般電化電池不同的是反應物不儲存在電池內，只在反應時灌入電池。

燃料電池是利用[[氫氣|氫氣]]在陽極進行氧化反應，將氫氣氧化成[[氫離子|氫離子]]，而[[氧氣|氧氣]]在陰極進行還原反應，與由陽極傳來的氫離子結合生成水。氧化還原反應過程中就可以產生電流。

燃料電池的技術包括了出現[[鹼性燃料電池|鹼性燃料電池]](AFC)、[[磷酸燃料電池|磷酸燃料電池]](PAFC)、[[質子交換膜燃料電池|質子交換膜燃料電池]](PEMFC)、[[熔融碳酸鹽燃料電池|熔融碳酸鹽燃料電池]](MCFC)、[[固態氧化物燃料電池|固態氧化物燃料電池]](SOFC)，以及[[直接甲醇燃料電池|直接甲醇燃料電池]](DMFC)等，而其中，利用[[甲醇|甲醇]]氧化反應作為正極反應的燃料電池技術，更是被業界所看好而積極發展。<br />

==== 水啟動電池 ====
{{link-en|水啟動電池|Water-activated battery}}本身不含電解質，出廠時還不會產生電力，須待加入水之後，才讓成份間的化學反應得以開始，並產生電力。

=== 核電池 ===
[[Image:Plutonium_pellet.jpg|thumb]]238加热至烧红的程度。]]
'''[[核電池|原子能电池]]'''（又称'''核电池'''，氚电池或放射性同位素發电装置）是指使用[[放射性|放射性]][[同位素|同位素]][[衰变|衰变]]时产生之能量来产生电力的装置。这会使人误解成[[核反应堆|核反应堆]]，但实际上这种电池不是利用[[链式反应|链式反应]]来产生能量。核电池比起一般'''电池'''有很长的寿命且其输出能量远比一般化学电池为高，可惜其制作成本也相对很高，使这种电池多用于一些需长时间运作又难以更换电池的仪器之上。
[[File:BlackTerror2452.jpg|缩略图]]
核电池之技术早在1913年已经被[[亨利·莫塞莱|亨利·莫塞莱]]所发明，使众科学家都期望此技术能够用于太空仪器上。但由于一直无法提高能源效率，这技术到近年纳米技术研发出更有效之[[半导体|半导体]]后再被关注。此種電池現應用於衛星、宇航工具，也有於地面上使用。第一個離開[[太陽系|太陽系]]的太空探測器[[航行者一號|航行者一號]]就是用這類電池作為能源，供應電力給探測器上的儀器。
核电池大致分成两种类，分别是[[热转换型核电池|热转换型核电池]]（例如[[放射性同位素熱電機|放射性同位素熱電機]]、[[斯特林放射性同位素高階熱電機|斯特林放射性同位素高階熱電機]]）及[[非热转换型核电池|非热转换型核电池]]。

== 電池容量 ==
電池容量是指電池所能儲存的[[電荷|電荷]]量，電池容量的符號為Q，單位為[[庫倫|庫倫]]（C），但日常生活中多以[[安培小時|安培小時]]（Ah）為單位，由於日常生活使用的電池也有容量相對較少，所以也有用[[毫安培小時|毫安培小時]]（mAh）單位，也即千分之一[[安培小時|安培小時]]，例如[[手機|手機]]所使用的電池通常以後者為標記。

決定電池容量的因素有：
電池的種類（也即制造電池的物質）：同一體積，不同種類的電池有不同的容量，例如鋰電池的容量較很多其他電池為高。
電池的體積：由於物質的化學能的[[能量密度|能量密度]]是固定的，因此體積越大，總藏能量就越多，例如一枚AA電池的容量比AAA電池為大。
電池的溫度：一般情況下，溫度越低，電池的有效容量會減小，不同種類的電池減小的程度各有不同，所以在寒冷地區使用電池時需要特別留意。
放電速率：放電電流越大，同一電池的有效容量會越小，所以推高耗電的電器時電池的容量會減少，例如一枚能點亮2W燈泡一小時的電池，推動4W燈泡時就不能有半小時，必定比半小時短些，短多少就視乎電池種類、溫度…等因素而定。

所以同一枚電池在不同環境下會有不同容量，所以一般電池所標示的容量只可作參考，實際使用仍會因環境及其他工作條件而有所變化。而標示容量一般都以[[室溫|室溫]]情況下作準。

===容量、放電電流與C值===
要計算一電池的在某一放電流下能連續放電多久，方法是將容量除以電流：

<math> t = \frac{Q}{I} </math>

當中，t（h）是放電時間（單位是小時）、Q（Ah）是電池容量（單位是安倍小時）、I（A）是電流（安培）。

而放電（或充電）電流也有用C值來表述，1個C的放電電流會剛好在一小時把電池完全放電，也就是1個C的電流是相對電池容量而定。例如一枚600mAh的電池，1個C的電流即是600mA，以這個電流放電會在一小時內用完電量。同樣地，對一枚2500mAh的電池，1C就是2500mA。又例如以0.5C對電池放電的話，不管電池容量多小，電池都會在2小時用完（1/0.5=2）。

用C作表述的特點在於在相同電池種類、操作環境下，不同容量的電池在同一C值放電率下，理論上都應該有相約放電時間。所以當要比較不同電池性能時，會選擇同一C值的放電或充電速率作比較。電池生產商在電池的規格上也多以C值表述放電電流及充電電流的速率。
