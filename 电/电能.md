'''电能'''（Electrical energy），是指[[电|电]]以各种形式[[做功|做功]]（即產生[[能量|能量]]）的能力。电能被广泛应用在[[动力|动力]]、[[照明|照明]]、[[冶金|冶金]]、[[化学|化学]]、[[纺织|纺织]]、[[通信|通信]]、[[广播|广播]]等各个领域，是[[科学技术|科学技术]]发展、[[国民经济|国民经济]]飞跃的主要动力。

== 種類 ==
有[[直流|直流]]电能、[[交流电|交流电]]能、[[高频|高频]]电能、[[中頻|中頻]]電能、[[低頻|低頻]]電能等。这几种电能均可相互转换。

== 形式 ==
日常生活中使用的电能，主要来自其他形式能量的转换，包括[[水能|水能]]（[[水力发电|水力发电]]）、[[热能|热能]]（[[火力发电|火力发电]]）、[[原子能|原子能]]（[[核电|核电]]）、[[风能|风能]]（[[风力发电|风力发电]]）、[[化学能|化学能]]（[[电池|电池]]）及[[光能|光能]]（[[光电池|光电池]]、[[太阳能|太阳能]]电池等）等。

电能也可转换成其他所需能量形式，如[[熱能|熱能]]、[[光能|光能]]、[[動能|動能]]等等。

电能可以靠[[有线|有线]]或[[无线|无线]]的形式，作远距离的[[传输|传输]]。

== 單位 ==
电能的单位是“'''[[焦耳|焦耳]]'''”，简称“'''焦'''”，以[[拉丁文|拉丁文]]字母'''J'''表示。电能等于[[有功功率|有功功率]]对[[时间|时间]]的[[积分|积分]]。

由于焦耳这个单位很小，在生活和生产中使用不方便，所以人们又使用一个比焦耳大的[[能量|能量]]单位——[[千瓦时|千瓦时]]（俗称“[[度|度]]”）千瓦时的符号是[[kW·h|kW·h]]。

电能的单位焦耳与千瓦时的换算[[公式|公式]]是1千瓦時=3.6×10<sup>6</sup>焦耳
