'''电动自行车'''俗称'''电动车'''或'''电瓶车'''、又稱'''-{zh-hans:电动单车;zh-hk:電動自行車;zh-tw:電動自行車}-'''、'''-{zh-hans:电动脚踏车;zh-hk:電動腳踏車;zh-tw:電動單車}-'''，是一种以[[蓄电池|蓄电池]]为主要[[能源|能源]]的轻便[[交通工具|交通工具]]，外观与[[自行车|单车]]或[[摩托车|摩托车]]相似，屬[[摩托化自行車|摩托化单车]]的一種，部分車種還带有[[脚踏板|脚踏板]]、[[链条|链条]]，可以用[[人力|人力]]作为[[辅助动力|辅助动力]]（稱為'''电动輔助单车'''）。

在中国大陆，根据1999年《电动自行车通用技术条件》的规定：电动[[自行车|单车]]归为助动车，时速规定不得超过20公里，整车[[重量|重量]]不得超过40公斤，是[[非机动车|非机动车]]的一种<ref>[http://www.gd.xinhuanet.com/newscenter/2008-12/17/content_15209582.htm 超标助力车究竟是什么车？] {{webarchive|url=https://web.archive.org/web/20090104040653/http://www.gd.xinhuanet.com/newscenter/2008-12/17/content_15209582.htm |date=2009-01-04 }}</ref>。而根据2018年发布的《电动自行车安全技术规范》，最高车速由20km/h调整为25km/h，整车质量(含电池)由40kg调整为55kg，电机功率则由240W调整为400W。<ref>{{Cite web|url=http://www.xinhuanet.com/local/2018-01/17/c_1122268871.htm|title=电动车新国标公示 电动自行车时速不得超25公里-新华网|accessdate=2018-06-02|work=www.xinhuanet.com}}</ref>
{{Wikibooks|中華民國法律註解/道路交通管理處罰條例/第69-1條}}
在臺灣，电动自行车指指經過型式審驗合格，以電力為動力來源，最高时速為25公里，且不含電池車重在40公斤以下之二輪車輛，具備免考駕照、免掛車牌、免牌照稅與燃料稅、不排放汙染、低噪音、電費成本低廉等。

在澳門，經過[[交通事務局|交通事務局]]的測試後已經批出了全澳門首個電動摩托車牌照，並准許在市面道路行駛，再經過近1年的測試推介，反應理想<ref>{{Cite web |url=http://www.macauzine.net/?action-viewnews-itemid-130 |title=電動摩托車駛上澳門街 |date=2011-05-31}}</ref>。

在香港則有較嚴格的法例，根據[[香港法例|香港法例]]第374章《道路交通條例》，所有電動或機械驅動的車輛不論大小都要向[[運輸署|運輸署]]申領牌照，而電動單車（包括機動三輛車）屬於此類車輛，駕駛者須持有-{zh-hans:摩托车;zh-hk:電單車;zh-tw:機車}-及機動三輪車的有效[[駕駛執照|駕駛執照]]，若未申領牌照而駕駛該車輛，將會干犯駕駛未經登記車輛罪、駕駛沒有第三者保險車輛罪等<ref>{{Cite web |url=https://www.youtube.com/watch?v=gA7dsjEznAw |title=警訊精選 - 使用電動單車相關法例 |date=2013-03-30 |website=YouTube}}</ref>；另根據該法，任何人也不得出售、供應或出租任何不符合以上條例之車輛，違法者可被罰款港幣20,000元，除非被告能證明他／她有合理理由該車輛不會在香港路面上使用<ref>{{Cite web |url=http://www.td.gov.hk/filemanager/sc/content_4808/motorcycle.pdf |title=小型電單車、電動單車及電動滑板 |date=2004-09-28 |publisher=香港特別行政區政府運輸署}}</ref>。小型電動單車在領牌前，也須送交署方檢定是否符合《道路交通（車輛構造及保養）規例》，以檢驗車輛在設計、照明、[[制動|制動]]和警報儀器等是否合乎所定要求及是否適合在道路行走。早前署方和[[香港警方|香港警方]]進行過的抽查測試結果發現，一輛售價5,000元的小型電動單車結構不合標準，手柄及座椅不穩固、剎車不完善、無照明裝置和倒後鏡，也沒有[[速度表|速度表]]。<ref>{{Cite web |url=https://www.consumer.org.hk/sites/consumer/files/w_choice_article/304/W0202_electricscooter.pdf |title=玩小型電動單車要領牌 |publisher=[[消費者委員會|消費者委員會]]}}</ref>

==组成==
===能源===
一般都采用[[铅酸蓄电池|铅酸蓄电池]]，电压大多为48V，也有少部分采用36V的。一般由4个12AH的12V免维护铅酸蓄电池[[串联|串联]]而成。

现在也有不少厂商开发研制采用[[鋰離子電池|锂电池]]为能源的小型电动[[自行车|单车]]，并且已经有成品出现在市场。

===控制器===
[[控制器|控制器]]是利用[[霍尔传感器|霍尔传感器]]来控制[[电机|电机]][[转速|转速]]的一个重要部件，一般具备[[无级调速|无级调速]]、欠压保护、[[过流|过流]]保护等功能。
===电机===
电动自行车[[电机|电机]]主要采用[[直流无刷电动机|直流无刷电动机]]/[[永磁同步电动机|永磁同步电动机]]驱动系统和无刷直流电机驱动系统，亦有少部分采用[[交流异步电机|交流异步电机]]驱动系统，[[开关磁阻电机|开关磁阻电机]]驱动系统等，早期曾广泛使用[[有刷直流电机|有刷直流电机]]，随着科技进步，维护麻烦、相对不可靠的有刷直流电机驱动系统逐渐被无刷直流电机/永磁同步电机与电调（[[变频器|变频器]]）组成的无刷驱动系统取代。

===刹车===
机械[[刹车|刹车]]与摩托车刹车相似，有[[抱剎|抱剎]]和[[碟刹|碟刹]]。同时由于电动自行车本身的特性，也有使用'''再生电刹车'''(eabs)的。

==分类==
*有刷电动自行车：电机采用[[碳刷|碳刷]]，优点是功率大，控制器结构简单。缺点是碳刷为消耗品，现在在中国大陆地区基本已淘汰。
*无刷电动自行车：电机无需碳刷，现在一般都采用此类解决方案，优点是电机免维护。缺点是控制器[[工艺|工艺]]复杂。但随着技术的发展，控制器技术发展迅速，[[矢量控制|矢量控制]]技术已在逐步运用。
*电动摩托车：无刷设计，功率更大。由于安全性欠缺，在中国已经不允许上路，但仍有厂家生产。
*锂动力电动自行车：采用[[锂离子电池|锂离子电池]]作为动力，结构轻巧，可折叠。缺点是价格不菲，且电池功率不够。
*锂离子磁动车：结构轻巧，可折叠，并且续航时间长。缺点是价格不菲，载重量80公斤，一般只适合女性使用。

==充电器==
一般采用二阶段充电，即先[[恒压|恒压]]充电，充到一定程度再用小电流进行[[涓流|涓流]]充满。

==日常保养==
启动时应慢速启动，有助力装置的应用脚踏启动。不可频繁刹车。电池应随用随充，避免造成深度放电。
