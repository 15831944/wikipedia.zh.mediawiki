'''电路'''（{{lang-en|Electrical circuit}}）或稱电子迴路，是由电气设备和'''-{zh-hans:元器件; zh-hant:元件;}-'''，按一定方式連接起来，为[[电荷|电荷]]流通提供了路径的總體，也叫电子线路或稱電氣迴路，簡稱网络或迴路。如[[電源|電源]]、[[电阻|电阻]]、[[电容|电容]]、[[电感|电感]]、[[二极管|二极管]]、[[電晶體|電晶體]]、[[集成電路|集成電路]]和[[电键|电键]]等，构成的网络、[[硬體|硬體]]。负电荷可以在其中运动。

== 術語定義 ==
幾個元件通過導線互相連接，形成「電路」，也可以稱為「網路」。更特定地，電路是可以形成閉合迴路的網路。「支路」是電路的一部分，每一個元件都有它獨屬的支路。任意兩條或多條支路的相交點，稱為「節點」。

== 電路種類 ==
电路的大小，可以相差很大，小到硅片上的[[集成电路|集成电路]]，大到高低壓[[输电|输电]]网。
=== 電子電路 ===
{{main|電子電路}}
根据所处理信号的不同，电子电路可以分为[[模拟电路|模拟电路]]和[[數位電路|數位電路]]。

==== 模拟电路 ====
{{main|類比電路}}
* 自然界產生的連續性物理自然量，將連續性物理自然量轉換為[[連續性電信號|連續性電信號]]，運算[[連續性電信號|連續性電信號]]的電路即稱為類比電路。

* 模拟电路对電信号的連續性[[电壓|电壓]]、[[电流|电流]]进行处理。

最典型的模拟电路应用包括：[[放大电路|放大电路]]、[[振盪电路|振盪电路]]、[[线性运算电路|线性运算电路]]（加法、减法、乘法、除法、微分和积分电路）。運算[[連續性電信號|連續性電信號]]。

==== 数字电路 ====
{{main|數位電路}}
數位電路又名邏輯電路，是一種將連續性的電訊號，轉換為[[不連續性定量電信號|不連續性定量電信號]]，並運算[[不連續性定量電信號|不連續性定量電信號]]的電路。数字电路中， 信号大小為不連續並定量化的[[電壓|電壓]]状态。多数采用-{zh-hans:[[布尔代数|布尔代数]]; zh-hant:[[布林代數|布林代數]];}-逻辑電路对定量後信号进行处理。典型数字电路有，[[正反器|正反器]]、-{zh-hans:[[寄存器|寄存器]]; zh-hant:[[暫存器|暫存器]];}-、[[加法器|加法器]]、[[减法器|减法器]]等，來運算[[不連續性定量電信號|不連續性定量電信號]]。

=== 積體電路 ===
* [[積體電路|積體電路]]亦稱為IC。
* 運用積體電路設計程式（IC設計），將一般電路設計到[[半導體|半導體]]材料裡的半導體電路（一般為矽片），稱為[[積體電路|積體電路]]。
* 利用[[半導體|半導體]]技術製造出[[積體電路|積體電路]]。

== 電路定律 ==
所有的电路都遵循一些基本电路定律。
* -{zh-hans:[[基尔霍夫电流定律|基尔霍夫电流定律]]; zh-hant:[[克希荷夫電流定律|克希荷夫電流定律]];}-：流入一个节点的电流总和，等于流出节点的电流总和。
* -{zh-hans:[[基尔霍夫电压定律|基尔霍夫电压定律]]; zh-hant:[[克希荷夫電壓定律|克希荷夫電壓定律]];}-：环路电压的总和为零。
* [[欧姆定律|欧姆定律]]：线性元件（如[[电阻|电阻]]）两端的电压，等于元件的阻值和流过元件的电流的乘积。
以下两条定理仅适用于线性电路
* [[诺顿定理|诺顿定理]]：任何由独立源，线性受控源与线性元件构成的两端网络，总可以等效为一个理想电流源与一个电阻的并联网络。
* [[戴维宁定理|戴维宁定理]]：任何由独立源，线性受控源与线性元件构成的两端网络，总可以等效为一个理想电压源与一个电阻的串联网络。

分析包含非线性器件的电路，则需要一些更复杂的定律。实际电路设计中，电路分析更多的通过计算机分析模擬来完成。

== 電路功率 ==
* 所有的電路在工作時，每一個元件或線路都會有[[能量|能量]]的工作運用，即[[電能|電能]]運用，而所有電路裡的[[電能|電能]]工作運用即稱為電路[[功率|功率]]。
* 電路或電路元件的[[功率|功率]]定義為：[[功率|功率]]=[[電壓|電壓]]*[[電流|電流]]（P=I*U）。
* 自然界裡能量不會消滅，固有一定律[[能量不滅定律|能量不滅定律]]。
* 電路總功率=電路功率+各電路元件功率。例如：電源（I*U）=電路（I*U）+ 各元件（I*U）.
* 在電路中的能量有時會變為[[熱能|熱能]]或[[輻射|輻射]]能等其他能量到空氣中，這就是電路或電路元件會發熱的原因，不會全部形成電能於電路中，根據能量不滅總能量=電能+熱能+輻射能+其他能量。

== 概括種類 ==
* [[電源電路|電源電路]]：產生各種[[電子電路|電子電路]]的所需求[[電源|電源]]、如:[[電池|電池]]、[[直流電|直流電]]、[[交流電|交流電]]、降電壓電路、升電壓電路。
* [[電子電路|電子電路]]亦稱電氣迴路。
* [[保護電路|保護電路]]：ESD、EMC、EMI。

* [[高頻電路|高頻電路]]，[[高頻|高頻]]，高頻率，使用[[高頻元件|高頻元件]]。

== 電路元件種類 ==
{{main|電子元件}}
* [[被動元件|被動元件]]：如[[電阻|電阻]]、[[電容|電容]]、[[電感|電感]]、[[二極體|二極體]]等，有分基頻被動元件、高頻被動元件。
* [[主動元件|主動元件]]：如[[電晶體|電晶體]]、[[微處理器|微處理器]]，[[IC|IC]]等有分基頻主動元件、高頻主動元件。

== 用途種類 ==
* 微處理器電路：亦稱微控制器電路，形成[[計算機|計算機]]、[[遊戲機|遊戲機]]、[[印表機|印表機]]、[[投影機|投影機]]、([[播放器|播放器]]影、音)、[[監視器|監視器]]、各式各樣[[家電|家電]]、[[滑鼠|滑鼠]]、[[鍵盤|鍵盤]]、[[觸控|觸控]]等。
* 電腦電路：為[[微處理器|微處理器]]電路進階電路，形成[[桌上型電腦|桌上型電腦]]、[[筆記型電腦|筆記型電腦]]、[[掌上型電腦|掌上型電腦]]、[[工業電腦|工業電腦]]等各种[[電腦|電腦]]。
* 通訊電路：形成[[電話|電話]]、[[手機|手機]]、[[對講機|對講機]]、[[有線網路|有線網路]]、[[有線傳送|有線傳送]]、[[無線網路|無線網路]]、[[無線傳送|無線傳送]]、[[光通訊|光通訊]]、[[紅外線|紅外線]]、[[光纖|光纖]]、[[微波通訊|微波通訊]]、[[衛星通訊|衛星通訊]]等。
* 顯示器電路：形成[[螢幕|螢幕]]、[[電視|電視]]、[[儀表|儀表]]、[[平面顯示器|平面顯示器]]等各類[[顯示器|顯示器]]。
* 光電電路：如[[太陽能|太陽能]]、[[電子顯微鏡|電子顯微鏡]]、電子[[望遠鏡|望遠鏡]]、[[相機|相機]]、[[攝影機|攝影機]]等各類電路。
* 電機電路：常運用於大電源設備、如電力設備、運輸設備、醫療設備、工業設備等。

== 電路拓樸 ==
* 電路是由各式光、電、磁零件，操控信號組合而成。
* 設計各種電路的架構，為電路的[[拓樸|拓樸]]。
* 一般電路設計至積體元件（IC）即為IC電路拓樸。
