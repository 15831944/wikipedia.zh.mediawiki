'''电影片（数字）技术合格证'''，曾是[[中华人民共和国|中华人民共和国]]境内以数字影片形式公映的电影的前置审查文件之一。

依据2014年实行的《电影数字母版技术审查管理办法（暂行）》，取得该合格证后，即意味着该数字影片的视听质量符合广电行业标准《电影送审数字母版声画技术质量主观评价办法》的要求，可以获得本许可证，在中国境内上映。

== 历史 ==
{{wikisource|电影数字母版技术审查管理办法（暂行）}}
[[File:电影片（数字）技术合格证.jpg|thumb]]
2014年开始，原[[国家新闻出版广电总局|国家新闻出版广电总局]]电影局下设的技术部门曾在电影申请[[电影公映许可证|电影公映许可证]]进入终审阶段时，同时对数字母版进行技术审查，以确保电影片的视听质量符合广电行业标准《电影送审数字母版声画技术质量主观评价办法》（GD/J033-2010）的要求。在数字母版经过技术审查后，审查机关会在发给纸质版《电影片公映许可证》的同时发放纸质版、光盘版《电影片（数字）技术合格证》，并要求电影制片单位将《技术合格证》数字影像画面自行接在留存的电影数字母版片尾“摄制单位”字幕画面之后<ref name="szmb">{{cite wikisource|author=[[国家广播电影电视总局|国家广播电影电视总局]]|title=电影数字母版技术审查管理办法（暂行）|date=2011-05-13|wslanguage=zh}}</ref>。

在2011年5月前<ref name="szmb" />，对于送审的数字母版，由[[中国电影资料馆|中国电影资料馆]]视情况发放三千五百元或三万元[[人民币|人民币]]补贴，母版由电影审查机关送资料馆保存<ref>{{cite web |title=广电总局电影局关于数字电影母版实行有偿收集的通知-中国电影发行放映协会 |url=http://www.chinafilm.org.cn/Item/Show.asp?m=1&d=6168 |website=中国电影发行放映协会 |accessdate=2018-08-28}}</ref>。《[[s:中华人民共和国电影产业促进法|中华人民共和国电影产业促进法]]》于2017年3月1日正式生效后，国家新闻出版广电总局电影局为贯彻落实《电影产业促进法》相关精神，宣布《电影片公映许可证》是电影片准予公映的唯一有效证明，不再单独发放《电影技术合格证》。在进行电影完成片审查时，只审一个主要发行版，其他发行版本的质量由出品单位、发行单位或其他版权主体负责。<ref>{{cite web |title=电影主管部门：拟不再单独发放《电影技术合格证》 |url=http://www.dmcc.gov.cn/mainSite/xw/jszx/892655/index.html|website=电影数字节目管理中心|accessdate=2018-12-11|date=2017-03-07}}</ref>
