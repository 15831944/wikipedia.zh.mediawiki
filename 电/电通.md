'''電通'''（{{lang-ja|電通}}），為當今[[日本|日本]]第一大也是世界最大單一[[廣告|廣告]][[公司|公司]]。總部位於[[東京都|東京都]][[港區_(東京都)|港區]][[東新橋|東新橋]][[汐留|汐留]]，並在多個國家設有據點。日本五大民營電視台之一的[[TBS電視|TBS電視]]也與其有密切關係。

== 沿革 ==
*前身為1901年創立的「株式会社日本廣告」（{{lang|ja|株式会社日本広告}}）和「電報通訊社」（{{lang|ja|電報通信社}}）。
*1906年，電報通信社改組為「株式会社日本電報通信社」（{{lang|ja|株式会社日本電報通信社}}）。
*1907年，與株式会社日本廣告合併。
*1936年，轉讓新聞通訊部門，改為專營廣告代理至今。
*1955年，社名改為「株式会社電通」（{{lang|ja|株式会社電通}}）。
*1981年，針對[[亞太區|亞太區]]市場，與[[美國|美國]]「揚雅<ref>「Y&R」的舊譯，現僅台灣使用，中國大陸、香港現譯為「揚羅必凱」。</ref> 廣告公司」（[[:en:Young_&_Rubicam|Young & Rubicam Inc.]]，Y&R）[[合資|合資]]成立「電通揚雅株式会社」（{{lang|ja|電通ヤング・アンド・ルビカム株式会社}}，Dentsu Young & Rubicam Inc.，DY&R）<ref>於2004年進行整合將亞太區分公司納入Y&R體系，除日本本社、台灣外更名為「Y&R」（即揚雅廣告）。</ref>。
*2012年7月12日，日本電通宣布，將支付31.6亿英镑（约合49.1亿美元）收购英國安吉斯媒體集團（Aegis Group PLC (AGS.LN)）<ref>{{Cite web |url = http://www.bnext.com.tw/article/view/id/23907 |title = 電通集團斥資50億美元買下安吉斯 |date = 2012-07-13  |publisher = 數位時代 |language = zh-tw }}</ref>。
*2013年3/月27日，日本電通發出新聞稿宣布購併的程序完成，同時公布新全球布局。未來電通集團將分為負責日本市場的「電通」，以及負責日本以外市場的「[[電通安吉斯|電通安吉斯]]」（Dentsu Aegis Network）兩個品牌，其中電通安吉斯總部設在倫敦，旗下仍分為電通與安吉斯兩個系統。<ref>{{Cite web |url = http://www.brain.com.tw/News/NewsPublicContent.aspx?ID=18435 |title = 電通完成收購安吉斯 未來日本與海外事業部分頭並進 |date = 2013-03-27  |publisher = 動腦新聞 |language = zh-tw}}</ref>。
*2016年12月28日，日本厚生勞動省以逼迫下屬違法加班的罪名，起訴25日自殺的员工高橋茉莉的直屬上司。隨後社長[[石井直|石井直]]引咎請辭(2017年1月生效)。<ref name="沿革2">{{Cite web |url = https://theinitium.com/article/20161229-dailynews-japan-dentsu/ |title = 再因員工過勞自殺惹非議，日本電通社長宣布辭職 |date = 2016-12-30  |publisher = 端傳媒科技 |language = }}</ref>
