'''摩-{}-打'''（[[英文|英文]]：Electric motor），又稱為'''馬-{}-達'''、'''電動機'''或'''電動馬-{}-達'''，是一種將[[電能|電能]]轉化成[[機械能|機械能]]，並可再使用[[機械能|機械能]]產生[[動能|動能]]，用来驅動其他裝置的電氣設備。大部分的電動摩打通過磁場和繞組電流，为摩打提供能量。

電動機與[[發電機|發電機]]原理基本一樣，其分別在於能量转化的方向不同：發電機是藉由負載（如水力、風力）將機械能、動能轉為電能；若沒有負載，發電機不會有[[電流|電流]]流出。電動機和電力電子、微控器配合已形成一新學門，稱為電動機控制。

== 歷史 ==
[[File:Jedlik_motor.jpg|220px]]
1740年，第一个电动马达是由[[苏格兰|苏格兰]]僧侶安德鲁·戈登（Andrew Gordon）创建的简单的静电设备。1827年，[[匈牙利|匈牙利]][[物理学家|物理学家]]安幼思·傑德利克（ÁnyosJedlik）开始尝试用[[电磁线圈|电磁线圈]]進行實驗。傑德利克解决一些技术问题後，稱他的设备為“电磁自转機”。虽然只用于教学目的，但第一款傑德利克的设备已包含今日[[直流馬達|直流馬達]]的三个主要组成部分：定子，转子和换向器。

1836年，[[美國|美國]]一位鐵匠湯馬斯·達文波特（Thomas Davenport）製作出世界上第一台能驅動小電車的應用馬達，並在1837年申请了[[专利|专利]]。由于主要动力电池成本極高，在商业上不成功，達文波特破产。一些发明家繼續发展應用馬達，但都遇到了同样电池发电成本的问题。

1845年，[[英國|英國]][[物理學家|物理學家]][[惠斯頓|惠斯頓]]（Wheatstone）申請[[線性馬達|線性馬達]]的專利，但原理於1960年代才被重視，而設計了實用性的線性馬達每次目前已被廣泛在工業上應用。

1870年代初期，世界上最早可商品化的馬達由比利時電機工程師Zenobe Theophile Gamme所發明。

1888年，美國著名發明家[[尼古拉·特斯拉|尼古拉·特斯拉]]應用[[法拉第|法拉第]]的電磁感應原理，發明[[交流馬達|交流馬達]]，即為感應馬達。

1902年，[[瑞典|瑞典]]工程師丹尼爾森利用特斯拉感應馬達的旋轉磁場觀念，發明了[[同步馬達|同步馬達]]。1923年，[[蘇格蘭|蘇格蘭]]人James Weir French發明三相可變磁阻型（Variable reluctance）[[步進馬達|步進馬達]]。

1962年，藉[[霍爾元件|霍爾元件]]之助，實用之DC無刷馬達終於問世。1980年代，實用之[[超音波馬達|超音波馬達]]開始問世。

== 原理 ==
[[File:3phase-rmf-noadd-60f-airopt.gif|thumb]]
馬達的旋轉原理的依據為[[佛来明左手定則|佛来明左手定則]]或是[[右手開掌定則|右手開掌定則]]，當一導線置放於磁場內，若導線通上電流，則導線會切割[[磁場線|磁場線]]使導線產生移動。[[電流|電流]]進入[[線圈|線圈]]產生[[磁場|磁場]]，利用[[電流的磁效應|電流的磁效應]]，使[[電磁鐵|電磁鐵]]在固定的[[磁鐵|磁鐵]]內連續轉動的裝置，可以將[[電能|電能]]轉換成[[動能|動能]]。與永久[[磁鐵|磁鐵]]或由另一組[[線圈|線圈]]所產生的[[磁場|磁場]]互相作用產生[[動力|動力]]。

電動機的種類很多，以基本結構來說，其組成主要由[[定子|定子]]和[[轉子|轉子]]所構成。定子在空間中靜止不動，轉子則可繞軸轉動，由軸承支撐。定子與轉子之間會有一定空氣間隙（氣隙），以確保轉子能自由轉動。機殼（場軛）需要用高導磁係數材料製成，要當作磁路用。

[[直流馬達|直流馬達]]的原理是[[定子|定子]]不動，[[轉子|轉子]]依相互作用所產生作用力的方向運動。[[交流馬達|交流馬達]]則是定子繞組線圈通上交流電，產生旋轉[[磁場|磁場]]，旋轉磁場吸引轉子一起作旋轉運動

== 種类 ==
===依磁場方向分類===
* [[轴向磁场电动机|轴向磁场电动机]] 
* [[径向磁场电动机|径向磁场电动机]] 

=== 依使用電源分類 ===
* [[直流馬達|直流馬達]]：使用永久磁鐵或電磁鐵、電刷、整流子等元件，電刷和整流子將外部所供應的直流電源，持續地供應給轉子的線圈，並適時地改變電流的方向，使轉子能依同一方向持續旋轉。 
* [[交流馬達|交流馬達]]：將交流電通過馬達的定子線圈，設計讓周圍磁場在不同時間、不同的位置推動轉子，使其持續運轉
* [[脈衝馬達|脈衝馬達]]：電源經過數位IC晶片處理，變成脈衝電流以控制馬達，步進馬達就是脈衝馬達的一種。

=== 依構造分類 ===
* [[同步馬達|同步馬達]]：特點是恆速不變與不需要調速，起動轉矩小，且當馬達達到運轉速度時，轉速穩定，效率高。
* [[感應馬達|感應馬達]]：特點是構造簡單耐用，且可使用電阻或電容調整轉速與正反轉，典型應用是[[風扇|風扇]]、[[壓縮機|壓縮機]]、[[冷氣機|冷氣機]]。
* [[可逆馬達|可逆馬達]]：基本上與感應馬達構造與特性相同，特點馬達尾部內藏簡易的剎車機構（摩擦剎車），其目的為了藉由加入摩擦負載，以達到瞬間可逆的特性，並可減少感應馬達因作用力產生的過轉量。
* [[步進馬達|步進馬達]]：特點是[[脈衝馬達|脈衝馬達]]的一種，以一定角度逐步轉動的[[馬達|馬達]]，因採用開迴路（Open Loop）控制方式處理，因此不需要位置檢出和速度檢出的回授裝置，就能達成精確的位置和速度控制，且穩定性佳。 
* [[伺服馬達|伺服馬達]]：特點是具有轉速控制精確穩定、加速和減速反應快、動作迅速（快速反轉、迅速加速）、小型質輕、輸出功率大（即功率密度高）、效率高等特點，廣泛應用於位置和速度控制上。
* [[線性馬達|線性馬達]]：具有長行程的驅動並能表現高精密定位能力。
* 其他：旋轉換流機（Rotary Converter）、旋轉放大機（Rotating Amplifier）等。

== 使用安全 ==
在使用馬達前需先了解其使用的電源是[[直流電|直流電]]還是[[交流電|交流電]]，如果是交流電，還需知道它是三相還是單相的交流電，接錯電源會導致不必要的損失和危險。馬達轉動後若沒有接負載或負載很輕使得馬達轉速快，則感應電動勢較強，此時馬達兩端電壓為電源提供電壓減去感應電壓，因此電流減弱。

若馬達的負載很重，轉速慢則相對感應電動勢較小，也因此電源需提供較大電流（功率）以對應所需的較大功率來輸出／作功。
