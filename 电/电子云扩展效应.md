'''電子雲擴展效應'''是在[[過渡金屬|過渡金屬]][[物理化學|物理化學]]中使用的[[術語|術語]]，它指的是[[拉卡参量|拉卡参量]]（Racah Parameter）的一种减少的趋势，以符号''B''表示，其发生于一种过渡态金属的自由电子形成具有[[配位基|配位基]]的[[配合物|配合物]]。名字来源于[[希腊语|希腊语]]''cloud-expanding''。由于这种效应可以部分解释金属-金属作用间的共价特点，从而其存在使得[[晶体场理论|晶体场理论]]具有缺陷。

''B''的下降指出了配合物中两个电子在给定的双占据金属''d''-[[原子轨道|轨道]]中排斥力较分别为M<sup>n+</sup>气相金属离子时小，而这又意味着配合物中轨道体积较大。这种电子云扩展效应可能由于一下一种或两种原因发生：

#金属中的正电荷作用效果降低。由于金属中的正电荷被配位键中的负电荷中和而降低，''d''-轨道会有轻微的扩展。
#因为[[分子轨道|分子轨道]]是由两个[[原子轨道|原子轨道]]形成的，其结果导致了配位键轨道的重叠以及形成[[共价键|共价键]]增加了轨道体积。
''B''的自由离子值的减少通常以[[电子云扩展参数|电子云扩展参数]]，β，的形式表示。

:β=''B''(配合物)'''/'''''B''(自由离子)
实验上可以观察到电子扩展的体积总是与配体的固有属性保持一致，如下所示：

:[[氟化物|F<sup><nowiki>−</nowiki></sup>]]<[[水|H<sub>2</sub>O]]<[[氨|NH<sub>3</sub>]]<[[乙二胺|en]]<[[硫氰酸盐|[NCS-N]<sup><nowiki>−</nowiki></sup>]]<[[氯化物|Cl<sup><nowiki>−</nowiki></sup>]]<[[氰化物|[CN]<sup><nowiki>−</nowiki></sup>]] < [[溴化物|Br<sup><nowiki>−</nowiki></sup>]] < [[叠氮化合物|N<sub>3</sub><sup><nowiki>−</nowiki></sup>]] < [[碘化物|I<sup><nowiki>−</nowiki></sup>]]

尽管这一序列某种程度上与配体的[[光谱化学序列|光谱化学序列]]很类似——比如，氰化物，乙二胺和氟看起来占据了两者之间的相同位置——其余物质中的氯，碘和溴则占据了完全不同的位置。这一排序粗略地反映了配体与金属形成较好的共价键的能力——有较强该效应的配体位于序列前端，反之则位于序列末端。

电子云扩展效应不仅仅依赖于配体的类型，也与中央金属离子有关。这两种因素共同作用下，可以按如下增序排列电子云扩展效应：

:Mn(II) < Ni(II) ≈ Co(II) < Mo(II) < Re(IV) < Fe(III) < Ir(III) < Co(III) < Mn(IV)
