[[File:Film_Release_License_of_China.png|thumb]]签发給国产故事电影片《[[一齣好戲|一齣好戲]]》，编号电审故字〔2018〕第526号]]
'''电影公映许可证'''，旧称'''电影片公映许可证'''，俗称'''影片通过令'''，或因其常以片头形式加载于电影正片之前且带有抽象化龙形标志而被称为'''龙标'''<ref>{{cite web |author1=北京青年报记者 |title=＂潘金莲＂出征多伦多之前到底过审拿龙标了吗？_网易新闻 |url=http://news.163.com/16/0920/16/C1DVIH2J00014SEH.html |website=网易新闻}}</ref>，是一份用于在[[中国内地|中华人民共和国境内]]发行、放映电影片的[[前置许可|前置许可]]，现由[[国家电影局|国家电影局]]签发。根据《[[s:电影管理条例_(2001年)|电影管理条例]]》，电影片只有审核合格，依法取得《电影片公映许可证》后方可发行、放映。

== 授权法源 ==
{{wikisourcehas|中华人民共和国电影产业促进法|电影管理条例}}
依据[[朱镕基|朱镕基]]于2001年12月25日签发[[s:中华人民共和国国务院令第342号|国务院第342号令]]公布的《[[s:电影管理条例_(2001年)|电影管理条例]]》及[[习近平|习近平]]于2016年11月7日签署第54号《[[中华人民共和国主席令|中华人民共和国主席令]]》公布的《[[中华人民共和国电影产业促进法|中华人民共和国电影产业促进法]]》规定，中国境内实施电影审查制度，未经国务院审查部门审查不得发行、放映、进口、出口。摄制完成后，片方应报请审查，而审查部门（现为[[国家电影局|国家电影局]]）必须在三十日内给出书面答复。同时，电影制片单位或者电影进口经营单位应将《电影片公映许可证》证号印制在该电影片拷贝第一本片头处<ref name="dytl">{{cite wikisource|title=电影管理条例 (2001年)|wslanguage=zh|date=2001-12-25|author=[[中华人民共和国国务院|中华人民共和国国务院]]}}</ref><ref name="cjf">{{cite wikisource|title=中华人民共和国电影产业促进法|wslanguage=zh|date=2006-11-07|author=[[中华人民共和国国务院|中华人民共和国国务院]]}}</ref>。

== 审批流程 ==
许可证审批分为四步，即初审、领取片头、终审、技术审查。其中，第四步技术审查系针对电影画面和音响质量进行的审查，虽然《数字电影技术合格证》并非《电影片公映许可证》的一部分，但实际上没有该许可证影片也无法于电影院上映，故一并列出。

=== 初审 ===
影片摄制完成后，制片方需要向在[[国家电影局|国家电影审查机关]]的网站上在线申请，并等待省级广电部门或电影审查委员会的批准。与此同时，片方需要给相关部门上交如下材料<ref name="ss">{{cite web |title=电影片送审须知 |url=http://dy.chinasarft.gov.cn/html/www/article/2015/014e6ba2d3d355a7402881a74df69fd7.html |website=国家新闻出版广电总局电影局 |accessdate=2018-08-28 |archive-url=https://web.archive.org/web/20180211130122/http://dy.chinasarft.gov.cn/html/www/article/2015/014e6ba2d3d355a7402881a74df69fd7.html |archive-date=2018-02-11 |dead-url=no }}</ref>：
#若是数字电影，则送高清数字节目带一套；若是胶片电影，则可选择混录双片一套、高清数字节目带一套或经过技术审查申请批准的[[Betamax|BETA]]带一套；
#若是数字电影，则上交《国产影片（数字）送审清单》；若是胶片电影，则上交《国产影片（胶片）送审清单》一份；
#上交《国产影片审查报批表》一式三份，须加盖第一出品单位的公章；
#上交《主创人员登记表》一份；
#上交影片完成台本的电子文档（附光盘）；
#上交该片的《电影剧本（梗概）备案回执单》(原件)。

下列内容，是涉及特殊情况的前置审批事项：
#若涉及片名英文译名，需要提交省级广电部门批复同意；
#若审批片名和剧本备案时的片名不同，需要提交省级广电部门批复同意；
#若影片需增加、变更出品单位的，须提前申报，并经省级广电部门批复同意；
#若影片聘请境外主创人员，需提交省级广电部门批复；

无论审查是否合格后，省级广电部门或电影审查委员会均会发给《影片审查决定书》及《影片审查意见表》；若不能通过，意见会在发回的《决定书》及《意见表》中做出说明。经过修改完成的电影若重新提请审查，则重复上述流程。

=== 发放龙标 ===
[[File:公映许可证.gif|从国家电影管理部门获得了《电影片公映许可证》片头意味着一部电影通过初审。此为故事片《[[小时代：青木时代]]》的许可证片头，公映许可证号为電审故字〔2013〕第347号。这段片头中一只由抽象胶片变幻出的红底黄色龙形标章也俗称为“龙标”|thumb]]
[[File:《你的名字》公映许可证.png|此为进口批片《[[你的名字]]》（数字）的许可证片头，公映许可证号为电审进字〔2016〕第055号|thumb]]
经省级广电部门审查通过后，片方需要持有下列材料，亲自前往[[国家电影局|国家电影审查机关]]，经核对无误后方能领取《电影片公映许可证》片头（即龙标）；值得注意的是，此时相关片方已经可以从片头中获知许可证编号，但并没有正式拿到许可证，这段视频也必须要提前加载在电影片里才能送终审<ref name="ss" />。

#影片DVD一套（已按省级广电部门的意见修改完毕）；
#省级广电部门出具的《影片审查决定书》（须加盖电影审查委员会或省级广电部门公章）；
#省级广电部门出具的《影片审查意见表》（需领导签字、加盖[[电影审查委员会|电影审查委员会]]或省级广电部门公章）；
#《国产影片审查报批表》一式二份（须加盖第一出品单位公章；电影审查委员会或省级广电部门公章）并附电子文档（光盘）；
#《主要创作人员登记表》一份并附电子文档（光盘）；
#影片完成台本的电子文档（附光盘）；
#交回该片《电影剧本（梗概）备案回执单》（可复印件）；
#经办人领取电影片头的授权书（须加盖第一出品单位公章）；

下列内容，是涉及特殊情况时所需的材料：
#省级广电部门同意更改影片片名的批复；
#省级广电部门同意影片英文片名批复；
#省级广电部门同意增加、变更出品单位、摄制单位的批复，及联合出品、摄制合同（出品人数量与出品单位数量须相符）；
#省级广电部门同意聘请境外主创人员的批复；
#影片如涉及如[[中华人民共和国人民警察|公安]]、[[中华人民共和国司法|司法]]、少数民族等特殊题材，需提供相关单位的协助审查意见。

现行版本的龙标自2018年12月12日起启用，较上一版相比，新版龙标的时长缩短为4秒，并带有浮雕立体效果，发证机关也因应[[深化党和国家机构改革|深化党和国家机构改革]]随之变更为“国家电影局 CHINA FILM ADMINISTRATION”。第一部启用新版龙标的电影为《[[照相师|照相师]]》（电审故字[2018]第891号）；而第一部启用新版龙标的进口电影为《[[大黄蜂_(电影)|大黄蜂]]》（电审特(进)字[2019]第001号）<ref>[http://www.chinanews.com/cul/2019/01-10/8725186.shtml 电影片头“龙标”换新 更名为“国家电影局”]</ref>。

=== 提交终审 ===
{{see|电影片（数字）技术合格证}}
在获取了对应的片头后，片方需要将下列材料提交国家电影审查机关，进行影片终审。审查通过后，方能正式获得《电影片公映许可证》<ref name="ss" />。值得注意的是，此时提交的内容均将纳入国家电影艺术档案管理（即送交[[中国电影资料馆|中国电影资料馆]]保存），审查机关不予退还<ref>{{cite web |title=国产片审查电影故事片送审须知 |url=http://dy.chinasarft.gov.cn/html/www/article/2010/0129dabd6cf2003f4028811629da935f.html |website=国家新闻出版广电总局电影局 |access-date=2018-08-28 |archive-url=https://web.archive.org/web/20180214071322/http://dy.chinasarft.gov.cn/html/www/article/2010/0129dabd6cf2003f4028811629da935f.html |archive-date=2018-02-14 |dead-url=no }}</ref>。

#提交数字电影母版一套（必须事先加载“龙标”片头）；
#提交《电影片数字母版制作送审单》；
#影片[[HDCAM|HDCAM]]或[[Betamax|BETA]]带一盘；
#影片[[DVD|DVD]]三套；
#《电影片终审信息表》一份并附电子文档（光盘）；
#影片国际乐效一套（光盘）；
#相关剧照和海报，并附电子文件（光盘）；
#经办人领取公映许可证的授权书（须加盖第一出品单位公章）；
#影片字幕（制作须参照《国产电影片字幕管理规定》及《关于规范电影片署名相关事项的通知》）。

下列内容，是涉及特殊情况时所需的材料：
#影片如有修改，需要送交最终完成台本的电子文档（光盘）；
#影片如涉及多家出品单位或联合摄制单位，需要提交出品、摄制单位之间的合同复印件。

== 未获得公映许可 ==
{{expand list}}
下表中罗列了1996年《电影管理条例》实施以来，未获得公映许可证的中国大陆电影和有证据证明曾申请在中国境内公映而未获批准的境外电影。

需要说明的是，据《电影管理条例》第六十一条：“未经批准，擅自举办中外电影展、国际电影节，或者擅自提供电影片参加境外电影展、电影节的，由国务院广播电影电视行政部门责令停止违法活动，没收违法参展的电影片和违法所得；违法所得2万元以上的，并处违法所得5倍以上10倍以下的罚款；没有违法所得或者违法所得不足2万元的，并处2万元以上10万元以下的罚款。”这意味着若一部中国大陆电影在参加境外电影节时未能取得许可证，则属违法行为，不能被电影审查部门批准在中国境内公映也属正常情况。
