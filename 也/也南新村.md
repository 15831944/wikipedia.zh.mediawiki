'''也南新村'''（{{lang-ms|Kampung Baru Jeram}}，旧称「古背」<ref name="sinchew.com.my">{{cite news|url=http://www.sinchew.com.my/node/1617312|title=双胞胎出生率激增．也南新村又叫“孖仔村”|date=2017-02-23|publisher=[[星洲网|星洲网]]}}</ref>）是[[马来西亚|马来西亚]][[霹雳州|霹雳州]][[金宝县|金宝县]]的一个[[华人新村|华人新村]]，位于[[金宝|金宝]]和[[务边|务边]]之间，于1949年开埠。<ref name="sinchew.com.my"/>

== 别名 ==
也南新村也称“双胞胎村”（也称“孖仔村”）。据说是1998年根据风水师的指示，把牌楼建在“大象山”对面，之后便有许多双胞胎和三胞胎出生，所以也南新村会有这别名。<ref name="sinchew.com.my"/>

== 美食 ==
也南新村有很多河婆人 ，所以美食也是“河婆擂茶”最出名。

== 观音庙 ==
也南新村里有一座观音庙，每逢观音诞（农历二月十九、六月十九和九月十九），庙里就会人山人海。许多信男信女都会前来参拜祈求平安。
