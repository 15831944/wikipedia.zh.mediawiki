'''也拉府'''（{{thai|t=จังหวัดยะลา|r=Changwat Yala}}，{{IPA-th|t͡ɕāŋ.wàt já(ʔ).lāː}}），是位于[[泰国|泰国]]最南部的一個府，也是泰南一个不甚安宁的府份。當地華人稱其為'''惹拉府'''。首府為[[也拉|也拉]]（惹拉）。

== 地理位置 ==
也拉府位于[[马来亚半岛|马来亚半岛]]上。

== 历史渊源 ==
历史上也拉府曾经是[[北大年苏丹国|北大年苏丹国]]的一部分，北大年由于历来向[[素可泰王国|素可泰王国]]及[[大城王国|大城王国]]的泰王[[纳貢|纳貢]]而保持其半独立性地位。

后来，大城王国于1767年被[[缅甸|缅甸军]]攻陷，北大年苏丹宣告脱离大城王国完全独立。但是[[扎克里王朝|泰王]]并没有承认这个独立宣言，[[拉玛一世|拉玛一世]]在击败缅甸之后又将北大年苏丹领地置于[[暹羅|暹羅]]的直接控制之下，直到1909年[[英国|英国]]与暹羅订立条约，北大年被分为[[那拉提瓦府|那拉提瓦府]]、[[北大年府|北大年府]]和也拉府归[[泰国|泰国]]管辖，而[[吉打|吉打州]]和[[吉兰丹|吉兰丹州]]则归英属[[马来亚|马来亚]]管辖。

在泰国南部地区的[[分离主义|分离主义]]分子从不承认这一条约，他们在2004年1月认为时机成熟，开始袭击泰国军方及警方，制造叛乱气氛。

== 人口划分 ==
也拉府人口66.1%为[[马来族|马来民族]]，习操[[马来语|马来语]]。68.9%信奉[[回教|回教]]，另26.2%信奉[[佛教|佛教]]。

== 府徽与其他象征 ==

府徽上示意描画出一群使用[[锄头|锄头]]、十字摇臂、吊蓝的[[矿工|采矿工人]]。也拉府原先盛产[[锡|锡矿石]]和[[钨|钨矿石]]。府树是 {{tsl|en|Saraca declinata||Red Saraca}}（学名：''Saraca declinata''）。府花則是 {{tsl|en|Bullet Wood||Bullet Wood}}（学名：''{{tsl|en|Mimusops elengi||Mimusops elengi}}''）。

== 行政与管理 ==
本府行政上被划分为8个县（ampoe），又进一步被划分为56个区（tambon）及341个村（muban）。
