'''地球同步卫星运载火箭'''（'''G'''eosynchronous '''S'''atellite '''L'''aunch '''V'''ehicle，英文缩写：GSLV），[[印度|印度]]自行研发为主的[[运载火箭|运载火箭]]。

== 簡介 ==
[[印度太空研究機構|印度太空研究機構]]利用地球同步衛星運載火箭將[[印度|印度]]全國[[衛星|衛星]][[系統|系統]]類型的[[衛星|衛星]]送至[[地球同步軌道|地球同步軌道]]。[[火箭|火箭]]多為[[俄羅斯|俄羅斯]]協助建造，並非[[印度|印度]]獨立建造。地球同步衛星運載火箭為[[極地衛星運載火箭|極地衛星運載火箭]] 之改良版，增加[[綑綁式|綑綁式]][[液態|液態]]輔助[[火箭|火箭]]為一三節式火箭。第一節為[[固態推進器|固態推進器]]；第二及第三為[[液態推進器|液態推進器]]。固態及輔助火箭是[[極地衛星運載火箭|極地衛星運載火箭]]之延續，[[低溫|低溫]][[液態|液態]][[引擎|引擎]]由[[俄羅斯|俄羅斯]]提供，共買了七個末端節引擎。印度試著去建造低溫末端節引擎並向俄羅斯買技術，但遭[[美國|美國]]施壓，因此俄羅斯並未提供此項技術給印度。

== 發射紀錄 ==
第一次發射失敗，在2001年4月18日发射實驗用通訊衛星地球同步衛星一號，但發射的轨道远低于预定轨道，由于轨道偏差太大，卫星無法自行机动进入地球同步軌道。

第二次發射成功，在2003年5月8日發射實驗用通訊衛星地球同步衛星二號。

第三次發射成功，在2004年9月20日發射[[EDUSAT通訊衛星|EDUSAT通訊衛星]]。

第四次發射失敗，在2006年7月10日下午5點38分，[[印度|印度]]軍方在印[[中央邦|中央邦]][[薩提什達萬航太中心|薩提什達萬航太中心]]發射一枚“地球同步衛星運載火箭-F02”，攜帶著實驗用通訊衛星地球同步衛星四號C型通訊衛星，距離地面約70[[公里|公里]]處出現箭體傾斜、連續翻轉、空中爆炸，跌落在[[孟加拉灣|孟加拉灣]]。

地球同步軌道運載火箭除2010年4月15日与2014年1月5日的两次發射外，其餘全部使用俄羅斯提供的[[低溫|低溫]][[液態|液態]]末端節[[引擎|引擎]]。

== 酬載能力 ==
地球同步軌道運載火箭使用四支L40液態助推火箭可酬載5000[[公斤|公斤]]（11000磅）到[[近地軌道|近地軌道]]；使用[[俄羅斯|俄羅斯]]KVD-1[[低溫|低溫]]末端節[[引擎|引擎]]可將2200[[公斤|公斤]]（4850磅）送到[[地球同步軌道|地球同步軌道]]。

== 各節簡介 ==
=== 第一節 ===
地球同步軌道運載火箭第一節直徑為2.8[[公尺|公尺]]，他用M250及的馬釘鋼作外殼材料，而[[推進劑|推進劑]]重達129吨；第一節之a（L40）有四十吨重的[[液態推進劑|液態推進劑]]（[[四氧化二氮|四氧化二氮]]/[[聯氨|聯氨]]），直徑為1.2公尺。

=== 第二節 ===
地球同步軌道運載火箭第二節直徑為2.8[[公尺|公尺]]，而且有[[液態推進劑|液態推進劑]]（[[四氧化二氮|四氧化二氮]]/[[聯氨|聯氨]]）分裝在兩個鋁合金槽裡，重達37.5吨。

=== 第三節 ===
地球同步軌道運載火箭第三節的直徑也是2.8[[公尺|公尺]]，用[[低溫|低溫]][[液態推進劑|液態推進劑]]（[[液態氫|液態氫]]/[[液態氧|液態氧]]）並放在兩個鋁合金槽裡，其質量為12.5[[吨|吨]]。
