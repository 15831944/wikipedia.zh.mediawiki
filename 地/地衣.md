'''地衣'''是[[真菌|真菌]]和[[绿藻|绿藻]]（或[[蓝细菌|蓝细菌]]）的[[共生|共生体]]，呈灰白、暗绿、淡黄、鲜红等多种颜色，长在干燥的岩石或树皮上。[[光合自养|光合自养]]的绿藻（或蓝细菌）提供营养物质，真菌（通常属于[[子囊菌門|子囊菌門]]，少数属于[[擔子菌門|擔子菌門]]）提供水和矿物质、提供保护，防止水分过度蒸发。地衣可以在严峻的环境条件下生长，常是生物占领新陆地的先锋。在极度干燥的条件下，地衣也可以脱去水分，进入[[休眠|休眠]]状态，待到条件好转，再恢复高速度增长。地衣可以长寿，有的寿命可达千年<ref>10.1016/j.cub.2009.04.034</ref>。

地衣通过包含真菌和绿藻（或蓝细菌）二者的地衣碎片在空气中传播，长出新地衣的方式进行[[无性生殖|无性生殖]]。他们各自也可以单独进行有性和无性的生殖。其中真菌部分单独生殖的后代，需要与有关绿藻（或蓝细菌）重新组合才可生存。

地衣植物約有1.8萬餘種，地球陸地面積的6％被地衣覆蓋。北极地区的地衣是北极[[驯鹿|驯鹿]]的主要食物，地衣也已用於製造[[染料|染料]]（如[[石蕊|石蕊]]）和[[香料|香料]]，以及在[[傳統醫學|傳統醫學]]作为藥物。<ref>{{cite web|url=http://mic.sgmjournals.org/content/156/3/609.full|title=Metals, minerals and microbes: geomicrobiology and bioremediation|journal=Microbiology|author=Geoffrey Michael Gadd|volume=156|date=March 2010|pages=609–643}}</ref>

== 詞源 ==
實際上，地衣（Lichen）中生長于地上者并不多，然而何為稱之地衣？最近[[日本|日本]]學者[[久保輝幸|久保輝幸]]考察地衣的詞源<ref name="久保1">[http://ci.nii.ac.jp/Detail/detail.do?LOCALID=ART0009116428 科學史研究第48巻] {{webarchive|url=https://web.archive.org/web/20130514110049/http://ci.nii.ac.jp/Detail/detail.do?LOCALID=ART0009116428 |date=2013-05-14 }}{{ja}}</ref> <ref name="久保2">[http://mayanagi.hum.ibaraki.ac.jp/students/kubo/chap1.html Lichen如何翻成地衣（日文）] {{webarchive|url=https://web.archive.org/web/20131102214732/http://mayanagi.hum.ibaraki.ac.jp/students/kubo/chap1.html |date=2013-11-02 }}</ref>：據此，地衣一詞首見于早期[[道教|道教]]書籍。但是，所指植物為現今所見的[[車前|車前]]。其后，唐人[[陳藏器|陳藏器]]在本草書中首次記載[[地衣草|地衣草]]，這個地衣草應是現在的地衣（Lichen）的詞源。[[李時珍|李時珍]]將之與《日華子諸家本草》中所記載的地衣視為同一種植物并記載至《[[本草綱目|本草綱目]]》中。而後至[[清朝|清朝]]，[[李善蘭|李善蘭]]同兩位傳教士編輯《植物學》<ref name="植物學">[http://catalogue.nla.gov.au/Record/854803/Details?lookfor=author:%22Williamson,+Alexander,+1829-1890%22&max=7&offset=7 韋廉臣輯訳；李善蘭筆述，《植物學》]，這是復刊影本出版，於合輯的後半部，此連結為澳洲國家圖書館的連結。正文中所述的二位傳教士在書中似乎沒有提到，而是韋廉臣（Alexander Williamson, 1829-1890）翻譯的。但在註解一的網路文件中，則詳提了第三位作者艾約瑟（J. Edkins, 1823-1905）。</ref>時，對「lichen」的詞譯采取本草書中的植物名稱「地衣」。{{Fact|當時西方人最常見的地衣可能就是可用藥的[[冰島苔|冰島苔]]，它生長在地上，所以，他們可能會毫不疑問的使用「地衣」這個名字|time=2011-07-30T14:00:25+00:00}}。

== 形态和结构 ==
地衣生活在各種表面上：[[土壤|土壤]]、[[樹木|樹木]]、[[岩石|岩石]]和牆上。他們通常會在一些環境惡劣的地方生長，如海拔數千米的高山、[[沙漠|沙漠]]和接近[[極地|極地]]的[[凍土|凍土]]。由於地衣對[[二氧化硫|二氧化硫]]相當的敏感，所以[[生態學|生態學]]上，地衣常被當成一種空氣品質的指標。

形態上地衣有三種：
# {{le|殼狀地衣|Crustose lichen}}：植物體成殼狀，緊貼在樹皮和石頭上。底面和基質緊密相連，難以分離。例如，{{le|茶漬屬|Lecanora}}（Lecanora）、{{le|文字衣屬|Graphis (lichen)}}（Graphis）。
# {{le|葉狀地衣|Foliose lichen}}：植物體平鋪，僅由假根狀的菌絲與附著的和體相邊，易於分開。例如，{{le|梅花衣屬|Parmelia (fungus)}}（Parmelia）、蜈蚣衣屬（Physica）。
# {{le|枝狀地衣|Fruticose lichen}}：植物直立或下垂如絲，多數片段有分枝。例如，[[石蕊|石蕊]]属（Cladonia）、[[松萝|松萝]]属（Usnea）。

而地衣結構一般可分為上皮層、藻胞層、髓層和下皮層。
# 上下皮層：由菌絲緊密交織而成，特稱為假皮層，下皮層一般能長出假根。
# 藻胞層：上皮層的下部，在排列疏鬆的藻胞層之間夾雜有許多藻細胞，這些[[細胞|細胞]]成層排列稱異層細胞，若散生則稱為同層細胞。
# 髓：藻細胞下面為髓層，由比較疏鬆而粗大的菌絲體交織而成，專門貯存[[空氣|空氣]]、[[水分|水分]]。

== 地衣的同名異物 ==
* 一種類似于{{link-en|木耳|Nostoc commune}}的[[珠藻科|珠藻科]][[蓝藻|蓝藻]]，又稱作[[地耳|地耳]]或[[地木耳|地木耳]]，亦于一些大陸地區中稱作「地衣」，以之作為食品。此種地衣不同于一般所稱的地衣（Lichen）。
* 現今所見的[[車前草|車前草]]<ref name="久保2" />。
* 一種[[唐朝|唐朝]]時舖於地上的[[織品|織品]]（敷物）<ref name="久保2" />。
