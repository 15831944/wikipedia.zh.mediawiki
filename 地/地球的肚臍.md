世界上有許多地方都被稱作'''地球的肚臍'''，其中包括自然景觀和富有文化价值的地區。在地理上，地球的肚臍可能是高於或低於附近地區，而附近較為平坦，或者是地球表面最低的地方。

在人類文化中，因為古代交通不方便，就將某些地方視為世界的中心，而用地球的肚臍稱呼，有時具有[[宗教|宗教]]上的意義。但並非世界的中心就代表地球的肚臍。

[[德爾斐|德爾斐]]位於[[希臘|希臘]]，所有古希臘城邦共同的聖地，在[[埃斯庫羅斯|埃斯庫羅斯]]（Eschyle）的《[[復仇神|復仇神]]》（les Euménides）中指出，德爾斐是世界的中心<ref>《塔穆德四講》, Quatre lectures Talmudiques, Paris, Minuit, 1968</ref>。

據說[[宙斯|宙斯]]曾經相反方向放出兩隻[[老鷹|老鷹]]來測量大地，而它們相遇的地點就是德爾斐。<ref>Pind. P. 4.74</ref>人們在可能是老鷹相遇的地點也就是德爾斐的Adyton豎立了[[翁法洛斯|翁法洛斯]]（意指「大地肚臍」，事實上，在[[地中海|地中海]]地區的很多地方都豎有翁法洛斯。）

1987年被列入[[文化遺產|文化遺產]]。

== 死海 ==
{{coord|31|20|N|35|30|E|type:isle}}

[[死海|死海]]位於[[約旦|約旦]]和[[以色列|以色列]]交界，其表面是地球上已露出陸地表面的最低點，湖面海拔負418[[公尺|公尺]]<ref>[http://isramar.ocean.org.il/DeadSea/default.htm Monitoring of the Dead Sea] {{webarchive|url=https://web.archive.org/web/20081011060847/http://isramar.ocean.org.il/DeadSea/default.htm |date=2008-10-11 }}. ''Israel Marine Data Center (ISRAMAR)''.</ref>，同時也是世界上最深的[[鹹水湖|鹹水湖]]，因此有「地球的肚臍」的稱呼。

== 烏魯汝 ==
{{coord|25|20|S|131|02|E|type:isle}}
[[File:Uluru.jpg|thumb]]
[[烏魯汝|烏魯汝]]位在[[澳大利亚|澳大利亚]][[愛麗斯泉|愛麗斯泉]]市西南470多[[公里|公里]]處，是世界上最大的獨立[[岩石|岩石]]，長3.6公里、寬2[[公里|公里]]，高出周圍平地約384[[公尺|公尺]]，因而有「地球的肚臍」之稱{{來源請求|time=2017-7-24}}。當地的原住民非常尊崇此巨石。

1987年被列入[[自然遺產|自然遺產]]，1987年被列入[[文化遺產|文化遺產]]。

在[[日文|日文]]小說《[[在世界的中心呼喊愛情|在世界的中心呼喊愛情]]》中，「世界的中心」就是指烏魯汝。

== 復活節島 ==
[[File:Orthographic_projection_centred_over_Easter_Island.png|thumb]]
{{coord|27|07|S|109|21|W|type:isle}}

[[復活節島|復活節島]]位於[[智利|智利]]首都[[聖地亞哥|聖地亞哥]]以西3700公里，[[大溪地|大溪地]]以東4000公里的太平洋上，離最近有人定居的[[皮特開恩群島|皮特開恩群島]]有2075公里，距離最近的島也有415公里，是標準的海上[[孤島|孤島]]，全島周長60公里，面積大約180平方公里。因為復活節島是附近唯一的陸地，因此當地人民稱這座島為'''Rapa Nui'''，也就是「世界的肚臍」<ref>[http://taipei.tzuchi.org.tw/rhythms/magazine/content/37/3/content3.htm 黃馨慧‧蕭耀華：復活節島 世界的肚臍] {{webarchive|url=https://archive.is/20120716182504/http://taipei.tzuchi.org.tw/rhythms/magazine/content/37/3/content3.htm |date=2012-07-16 }}</ref>。

1995年被列入[[文化遺產|文化遺產]]。

== 阿貢火山 ==
{{coord|8|20|S|115|30|E|type:isle}}

[[阿貢火山|阿貢火山]]又稱[[峇里峰|峇里峰]]，是[[峇里島|峇里島]]的最高峰，海拔3142公尺，被當地人稱為「世界的肚臍」，而著名的[[百沙基陵廟|百沙基陵廟]]就建在阿貢火山的山坡上，用來祭祀這座火山之神<ref>[http://big5.ctrip.com/html/magazine/2008/0807/m_03.htm 網上雜誌-峇里島]{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 庫斯科 ==
{{coord|13|30|S|71|58|W|type:isle}}

[[庫斯科|庫斯科]]是[[秘魯|秘魯]]東南方的城市，海拔3400公尺，是古老的[[印加帝國|印加帝國]]的搖籃，被[[安地斯山脈|安地斯山脈]]環繞，[[克丘亞語|克丘亞語]]是「大地之臍」的意思，印加人相信他們帝國的中心就是在大地的中心位置上，而肚臍隱含世界中心意味<ref>[http://blog.mook.com.tw/noahsu/271/20071212/1217 雙魚流浪日記「世界的中心─庫斯科」] {{webarchive|url=https://archive.is/20090209161250/http://blog.mook.com.tw/noahsu/271/20071212/1217 |date=2009-02-09 }}</ref>。

1983年被列入[[文化遺產|文化遺產]]。
