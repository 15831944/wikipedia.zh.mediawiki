{{vgname|地球冒险3|ja=MOTHER3}}是一款由[[1-UP工作室|Brownie Brown Inc]]与[[HAL研究所|HAL研究所]]制作、[[任天堂|任天堂]]在[[Game_Boy_Advance|Game Boy Advance]]发行的游戏。本作是《[[地球冒险系列|地球冒险]]》系列的第三作，也是目前为止，系列的最后一作。游戏曾作为在[[超级任天堂|超级任天堂]]以及[[任天堂64|任天堂64]]和[[Nintendo_64DD|64DD]]上开发的游戏，但在2000年宣布停止开发；直到2003年开始继续开发，游戏最终于2006年4月20日在日本地区发行。<ref>{{cite web|last1=Goldfarb|first1=Andrew|title=Mother 3 Coming to Wii U Virtual Console in Japan|url=http://www.ign.com/articles/2015/11/13/mother-3-coming-to-wii-u-virtual-console-in-japan|publisher=IGN|accessdate=2015-11-18}}</ref>

== 玩法 ==
与前几作类似，《地球冒险3》是一个单人的角色扮演游戏。玩家控制着一群可供游戏的人物角色，在2D的虚拟世界探索。游戏的主要视角是俯视。

== 发行 ==
《地球冒险3》于2006年4月20日在日本地区发行，并成为最佳销售游戏。在发行之前，游戏被列为《Fami通》五大最为期待的游戏之一。

===非官方翻译===
{{See|地球冒险3非官方英文版}}
《地球冒险3》并未在日本地区之外发行。2008年10月17日，Starmen.net发行了一个非官方英文补丁。

== 评价 ==

在日本地区第一周，《地球冒险3》共售出20万份。周刊《Fami通》给予35/40的分数。
