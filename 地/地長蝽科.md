'''地長蝽科'''（學名：{{lang|la|Rhyparochromidae}}）是[[半翅目|半翅目]]的一個[[科_(生物)|科]]<ref>{{cite web |url=http://www.hkwildlife.net/Forum/viewthread.php?tid=90367 |title= 地長蝽科 Rhyparochromidae ID Please |date=2017-08-06 |access-date=2019-01-28 |language=zh-hant }}</ref><ref name="sciencenet">{{cite web |url=http://blog.sciencenet.cn/blog-1292052-953945.html |title=半翅目蝽总科中首次发现拟蚁种类 |date=2016-02-01 |access-date=2019-01-28 |language=zh-hant }}</ref>，由兩個亞科、368 個屬、超過1,800個物種組成<ref>{{cite web |publisher={{tsl|en|Australian Government|}}: {{tsl|en|Department of the Environment and Heritage|}} |url=http://www.discoverlife.org/20/q?search=Rhyparochromidae |title=Rhyparochromidae |date=2002-02-15 |language=en }}</ref>。

本科物種體型皆為細小，還包含數個拟蚁种类{{r|sciencenet}}本科物種一般為棕色、或有斑駁花紋。

==分類==
除Plinthisinae亞科只包含一個屬，餘下接近四百個屬均為地長蝽亞科成員，分屬於11個族，另外一個屬亞科地位未定，茲詳列如下{{r|ITIS}}：

* Tribe	[[Gonianotini|Gonianotini]] {{small|Stål, 1872}}
* Tribe	[[Lethaeini|Lethaeini]] {{small|Stål, 1872}}
* Tribe	[[Megalonotini|Megalonotini]] {{small|Slater, 1957}}
* Tribe	[[Myodochini|Myodochini]] {{small|Boitard, 1827}}
* Tribe	[[Ozophorini|Ozophorini]] {{small|Sweet, 1964}}
* [[地長蝽族|地長蝽族]] [[Rhyparochromini|Rhyparochromini]] {{small|Amyot and Serville, 1843}}
