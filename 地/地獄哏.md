'''地獄哏'''（{{lang en|Hellish gags}}）是指以他人或是自身的[[殘障|殘障]]、[[悲劇|悲劇]]、[[歧視|歧視]]或其他足以引人悲傷、憤怒的事情做為[[幽默|幽默]]、笑點，製作成[[網路爆紅事物|梗圖]]或是影片<ref>[https://www.thenewslens.com/amparticle/86573 【插畫】拒絕爛「地獄梗」，別把無聊當有趣]</ref>。

==概述==
地獄哏為一種人與人之間[[幽默|幽默]]的表達方式，屬於[[迷因|迷因]]的一種。
隨著[[網際網路|網際網路]]的普及，使得[[網路爆紅事物|網路迷因]]快速普及，其中的一部分愛好者，便自然的聚集，並以他人之殘缺、歧視等作為笑點，因為愛好者們自知其不道德，會下[[地獄|地獄]]，遂以「地獄哏」稱之。

地獄哏笑點在於圖片與文字描述之間的落差大太，因而造成觀看者的出乎意料，國立交通大學羅仕宇教授認為，地獄梗挑戰的是文明社會裡看到別人面臨苦難應保有同情心的特點，當面對地獄梗時，「稍微」失諧的感覺產生，會讓觀看者覺得好笑；但當失諧的感覺太嚴重，可能會感到震驚或者出現負向情緒。<ref>{{cite web |title = 地獄梗　迷因的黑暗分支|date = 2017/11/26 | url = http://castnet.nctu.edu.tw/castnet/article/11352?issueID=671}}</ref>

==舉例==
*[[黑人|黑人]]常為地獄哏的題材，大多是基於美國早期的[[黑人法令|黑人歧視]]以及[[黑奴|黑奴]]等事件作嘲諷。
*[[史蒂芬·霍金|史蒂芬·霍金]]在被診斷罹患[[肌萎縮性脊髓側索硬化症|肌萎縮性脊髓側索硬化症]]後，隨著名氣提高，其坐輪椅且頭傾靠在右肩的形象逐漸廣為人知，遂出現了以此相關的地獄哏。
*[[2016年內湖隨機殺人事件|2016年內湖隨機殺人事件]]中，被害人「小燈泡」遭主謀王景玉以菜刀猛砍頸部，導致其當場頭身分離、倒地死亡。事件發生後，就出現了「小燈泡掉頭就走」的諧義地獄哏。

==批評==
多數的地獄哏愛好者，自知其為不[[道德|道德]]的，因此多不在[[愛好者群體|愛好者群體]]以外的地方開玩笑，通常而言，以黑人做題材的玩笑，多是[[諷刺|諷刺]][[黑人法令|黑人歧視]]，而非[[歧視|歧視]]黑人。儘管如此，仍引來了許多的批評。例如2018年3月14日，[[霍金|霍金]]的死訊震驚許多人，網路上卻有許多人放置霍金的地獄哏圖，引發批評。<ref>{{cite web |title = 霍金地獄梗圖狂洗版！網憶起鐘鉉燒炭圖 傻眼：當初護航迷妹消失了|date = 2018-03-14 | url = https://www.ettoday.net/news/20180314/1130196.htm}}</ref>
以及反[[年金改革|年改]]團體800壯士中的[[繆德生|繆德生]]因為攀牆墜樓身亡，卻有[[王浩宇|議員]]公開貼文「800-1=799」遭到許多人撻伐。<ref>{{cite web |title = 議員po「800-1=799」　網怒：缺德地獄梗|date = 2018-03-06 | url = https://tw.news.appledaily.com/column/article/438/rnews/20180306/1309376}}</ref>
