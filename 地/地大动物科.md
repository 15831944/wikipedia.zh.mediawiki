'''地大动物科'''（[[学名|学名]]：{{lang|la|Didazoonidae}}）是[[古蟲動物門|古蟲動物門]]中的一科，包括三個物種，分別為[[地大動物|郝氏地大動物]]<ref name = "Shu"/>、[[圆口虫|具腹圆口虫]]<ref name = "Luo"/>和[[西大動物|皇冠西大蟲]]<ref name = "Xidazoon"/>。全長約 5 至 20 公分。生活於[[寒武紀|寒武紀]]早期，化石皆發現自[[雲南|雲南]][[澄江县|澄江]]的[[帽天山页岩|帽天山页岩]]。
