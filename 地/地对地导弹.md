'''地对地导弹'''是从陆地发射打击地面目标的[[导弹|导弹]]。它由[[弹体|弹体]]、[[弹头|弹头]]、[[动力装置|动力装置]]、[[飛彈導引|制导系统]]等部分组成。按照作战任务的不同，地地导弹可以分为地地战略导弹和地地战术导弹。

== 地地战略导弹 ==
地地战略导弹是由地面发射，对地面战略目标进行打击的[[弹道导弹|弹道式导弹]]，通常装有核弹头。从导弹发射井发射的地地战略导弹，由于阵地固定，易被对方侦察发现，已逐渐趋向发射井和机动发射并重。

== 地地战术导弹 ==
地地战术导弹是指从陆地发射，攻击敌方战术纵深内地面目标的导弹武器，它的最大射程一般不超过1000千米。海湾战争的经验说明，地地战术导弹如果没有准确的命中精度，就不能发挥巨大的杀伤力。伊拉克发射的早期型[[飞毛腿导弹|“飞毛腿”导弹]]有15%的导弹没有正確引爆或遭拦截。但是，由于该导弹的制导精度太低，这15%的导弹大部分都未击中预定的目标。
==导弹种类==
地对地导弹通常分为以下几种：
*[[弹道导弹|弹道导弹]]在高轨道上行驶，发动机在飞行中途工作
**[[战术弹道导弹|战术弹道导弹]]：射程在150公里至300公里之间
***{{tsl|en|Battlefield range ballistic missile||战役战术弹道导弹}}（BRBM）：射程小于200公里
**[[战区弹道导弹|战区弹道导弹]]（TBM）：射程在300公里至350公里之间
***[[短程弹道导弹|短程弹道导弹]]（SRBM）：射程小于1000公里
***[[中程弹道导弹|中程弹道导弹]]（MRBM）：射程在1000公里至3500公里之间
**[[远程弹道导弹|远程弹道导弹]]（IRBM）：射程在3500公里至5500公里之间
**[[洲际弹道导弹|洲际弹道导弹]]（ICBM）：射程大于5500公里
**[[潜射弹道导弹|潜射弹道导弹]]（SLBM）：自[[弹道导弹潜艇|弹道导弹潜艇]]（SSBN）发射，现役类型均为洲际弹道导弹
* [[巡航导弹|巡航导弹]]在地面低空飞行，整个飞行过程中发动机均正常工作，一般射程为2,500公里
* [[反坦克导弹|反坦克导弹]]在地面低空飞行，发动机在飞行过程中不一定是否一直工作，一般射程为5公里
* [[反舰导弹|反舰导弹]]在地面和海面低空飞行，在撞击目标船之前经常改变姿态或闪避；一般射程为130公里

不同机构以不同的形式定义导弹类型。例如，[[美国国防部|美国国防部]]没有远程弹道导弹的定义，并将洲际弹道导弹定义为射程超过5,500公里（3500英里）的导弹。[[国际战略研究所|国际战略研究所]]也没有确定远程弹道导弹的射程范围，并且短程弹道导弹的定义射程范围为比国防部使用的定义范围更短。
