{{vgname|地球異世界|en=Defiance}}，又譯'''生存聖戰'''，是一款[[科幻|科幻]]主題的[[第三人稱射擊遊戲|第三人稱射擊]][[大型多人線上遊戲|大型多人線上遊戲]]，由{{link-en|Trion Worlds|Trion Worlds}}公司開發。《地球異世界》發生在未來被[[地球化|地球化]]的地球。同時還開發了[[Syfy|Syfy]]的[[地球異世界|同名電視劇]]，作為[[商業搭配|商業搭配]]。遊戲於2013年4月2日在[[Microsoft_Windows|Microsoft Windows]]、[[PlayStation_3|PlayStation 3]]和[[Xbox_360|Xbox 360]]上發售，<ref>{{Cite web |url = http://tvbythenumbers.zap2it.com/network-press-releases/syfys-defiance-commences-production-in-toronto-july-24/142442/ |title = Syfy's ‘Defiance’ Commences Production in Toronto July 24 |date = 2012-07-23 |publisher = Screener |language = en}}</ref>同時也在[[Steam|Steam]]上發售。遊戲於2014年6月4日在PC上轉為[[網絡遊戲免費模式|免費遊戲]]，PS3為2014年8月14日，而Xbox 360為2014年11月18日。

== 開發 ==
===《地球異世界2050》===
2017年12月，Trion Worlds確定該遊戲的PlayStation 4版正在開發中<ref>{{Cite web |url = https://www.dualshockers.com/defiance-development-current-generation-hardware-heres-trion-worlds-planned/ |title = Defiance is in Development for Current Generation Hardware; Here's What Trion Worlds Has Planned |author = Azario Lopez |publisher = DualShockers |language = en}}</ref>。他們還表示沒有製作續集的計劃，相反他們將繼續為遊戲升級新內容。名為《地球異世界2050》的新版本於2018年7月10日在PC、Xbox One和PS4上推出，而該遊戲會是重製版，並且可以利用新系統的能力<ref>{{Cite web |url = https://www.pcgamer.com/defiance-2050-is-a-reimagined-sci-fi-shooter-coming-this-summer/ |title = Defiance 2050 is a 'reimagined' sci-fi shooter coming this summer |author = Andy Chalk |date = 2018-02-27 |publisher = PC Gamer |language = en}}</ref>。遊戲在推出時會是免費的。
