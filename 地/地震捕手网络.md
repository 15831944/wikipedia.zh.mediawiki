'''地震捕手网络'''（'''Quake-Catcher Network'''）是[[南加州大学|南加州大学]]运行的一个倡议，其目标是使用基于计算机的[[加速規|加速規]]探测地震<ref>{{Cite web |url = http://quakecatcher.net/ |title = Main project page |accessdate = 2016-10-31|publisher=University of Southern California }}</ref>。它使用[[BOINC|BOINC]]{{tsl|en|Volunteer computing|志愿者计算}}平台（一种[[分布式计算|分布式计算]]，类似[[SETI@home|SETI@home]]）。

它目前支持有内置加速度计的[[移动设备|移动设备]]（[[智能手机|智能手机]]和部分[[平板电脑|平板电脑]]/[[笔记本电脑|笔记本电脑]]）<ref>{{cite web |url = http://quakecatcher.net/about-qcn/about-network|title=Introduction to the Network |accessdate = 2016-10-31 |publisher = University of Southern California }}</ref>，并且目前也支持三种USB设备——codemercs.com JoyWarrior 24F8、ONavi传感器和MotionNode Accel<ref>{{Cite web |url = http://quakecatcher.net/sensor-performance |title=Sensor Comparison |accessdate=2016-10-31 |publisher = University of Southern California }}</ref>。

2011年，项目科学家伊丽莎白·科克兰（Elizabeth Cochran）被美国总统[[奥巴马|奥巴马]]授予{{tsl|en|Presidential_Early_Career_Award_for_Scientists_and_Engineers|总统青年成就奖}}<ref>{{cite web |title = Elizabeth Cochran accepts the Presidential Early Career Award for Scientists |url = https://ucrtoday.ucr.edu/1176/20111014_2pa4252 |website = UCR Today |accessdate=2016-12-2 3}}</ref>，这很大程度上是因为她创建了地震捕手网络项目。
