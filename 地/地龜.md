'''地龜'''（[[學名|學名]]：''Geoemyda spengleri''；[[英文|英文]]：''Black－breatsed Leaf Turtle''），亦名'''黑胸葉龜'''、'''長尾山龜'''、'''十二棱龜'''、'''金龜'''，[[地龜科|地龜科]][[地龜屬|地龜屬]]，半[[水棲龜|水棲龜]]，主要分布於[[越南|越南]]和[[中國|中國]]之[[廣西|廣西]]、[[廣東|廣東]]、[[湖南|湖南]]、[[海南|海南]]等地。

== 居住環境 ==
[[攝氏溫度|攝氏溫度]]介於18℃～28℃之[[山區|山區]][[叢林|叢林]]、[[小溪|小溪]]及[[山澗|山澗]]（山磵）[[小河|小河]]邊。多屬陰暗潮濕之地，其亦不能太過靠境水深處（水位不能超過龜殼高度的2倍）。

== 食物 ==
地龜喜歡吃鮮活的小蟲子。例如：[[蚯蚓|蚯蚓]]、[[蟋蟀|蟋蟀]]、[[麥皮蟲|麥皮蟲]]（[[步行蟲|步行蟲]]之[[幼蟲|幼蟲]]）等。另外亦吃[[西紅柿|西紅柿]]、瘦[[豬肉|豬肉]]、[[黃瓜|黃瓜]]、小魚、[[水龜糧|水龜糧]]等。

== 特徵 ==
地龜體型較小，成體[[背甲|背甲]]長僅約120毫米，寬約78毫米。其頭部呈[[淺棕色|淺棕色]]，頭較小，背部平滑，上喙鉤曲，眼大且外突，自吻突側沿眼至頸側有淺黃色縱紋。

背甲為[[金黃色|金黃色]]或[[桔黃色|桔黃色]]，中央具三條[[嵴棱|嵴棱]]，前後緣均具齒狀，共十二枚。腹甲呈[[棕黑色|棕黑色]]，兩側有[[淺黃色|淺黃色]][[斑紋|斑紋]]，[[甲橋|甲橋]]明顯，背腹甲間借骨縫相連。後肢淺棕色，散布有紅色或黑色斑紋，指、趾間蹼，尾細短。

=== 雌雄鑑別 ===
雌龜的腹甲平坦，尾短，[[泄殖腔|泄殖腔]]孔距腹甲後緣較近；雄龜的腹甲中央凹陷，尾長且粗，泄殖腔孔距腹甲後緣較遠。

== 習性 ==
[[雜食性|雜食性]]半水棲龜，[[幼年期|幼年期]][[肉食性|肉食性]]偏強，[[卵生|卵生]]，每次產下4～6枚卵，每年6月－8月為其[[產卵期|產卵期]]。

== 亞種 ==
一度認為分布於日本[[沖繩島|沖繩島]]北部、[[久米島|久米島]]、[[渡嘉敷島|渡嘉敷島]]的[[琉球長尾山龜|琉球長尾山龜]]（''Geoemyda japonica''；[[日本地龜|リュウキュウヤマガメ]]），是地龜的亞種，但該種已於1992年成為獨立一[[物種|物種]]。

== 保護 ==
在1988年11月8日第七屆[[全國人民代表大會|全國人民代表大會]][[常務委員會|常務委員會]]第四次會議上通過的《[[中華人民共和國野生動物保護法|中華人民共和國野生動物保護法]]》中，定為國家Ⅱ級重點保護野生動物。並於1989年3月1日施行。並於[[九萬山|九萬山]][[水源林|水源林]]、[[大瑤山|大瑤山]]水源林（廣西）、[[東洞庭湖|東洞庭湖]]、[[南洞庭湖|南洞庭湖]][[濕地|濕地]]等地嚴加保護著。
