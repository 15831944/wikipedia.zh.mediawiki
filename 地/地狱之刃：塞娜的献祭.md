{{Vgname|地狱之刃：塞娜的献祭|en=Hellblade: Senua's Sacrifice|cn=地狱之刃：塞娜的献祭|tw=地獄之刃：賽奴雅的獻祭}}，是一个由[[忍者理论|忍者理论]]工作室开发的[[黑暗奇幻|黑暗奇幻]]风格的[[动作冒险游戏|动作冒险游戏]]。制作组把它视为一个[[3A游戏|3A游戏]]。它于2017年8月在全球范围发售<ref>[https://news.sina.com.tw/article/20180623/27274412.html 《地獄之刃：塞娜的獻祭》銷量已突破百萬大關]，[[北京新浪網|北京新浪網]]，2018年6月23日</ref>，适用于[[Windows|Windows]]和[[PlayStation_4|PlayStation 4]]平台，并于2018年4月推出[[Xbox_One|Xbox One]]版本<ref>[https://gnn.gamer.com.tw/2/159892.html 《地獄之刃：賽奴雅的獻祭》台灣軟體分級現身 Xbox One 版本]，[[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]]，2018年3月8日</ref>，并计划于2019年春季推出[[任天堂Switch|任天堂Switch]]版本。<ref>[https://www.polygon.com/nintendo-switch/2019/2/13/18224028/hellblade-nintendo-switch-release-date Hellblade: Senua’s Sacrifice coming to Nintendo Switch this spring].Polygon.2019-02-13.[2019-02-16].</ref>

== 簡介 ==
游戏故事基于[[北欧神话|北欧神话]]和[[凯尔特人|凯尔特人]]文化，发生在8世纪晚期。女主角塞娜是一名来自[[奥克尼|奥克尼]]的凯尔特女战士，她的家园被维京人洗劫，她的爱人迪利安被敌人俘虏。她必须经历重重挑战才能到达[[冥界_(北欧神话)|海尔海姆]]，将死去的爱人的灵魂从[[赫尔_(北欧神话)|海拉]]那里拯救出来。游戏中含有大量[[精神病|精神病]]的暗示，女主角头脑中充斥着许多声音，而她认为自己是被“黑暗”所困扰<ref>[http://www.macaodaily.com/html/2017-08/23/content_1202322.htm 《地獄之刃》迎末日危機]，[[澳門日報|澳門日報]]電子版，2017年8月23日</ref>。为了正确表现精神病的特点，开发者与神经科学家，心理健康专家以及患病者展开了密切合作<ref>[http://www.gamersky.com/news/201611/839612.shtml 《地狱之刃：塞娜的献祭》新开发日志 走进精神病人的世界]，[[游民星空|游民星空]]，2016年11月26日</ref>。

== 开发 ==
=== 动作捕捉 ===
主角与玩家的交流不仅通过言语进行，还通过CG动画和角色动作进行。为此，开发者创建了自己的低成本演播室<ref name="gs-mocap">{{cite web|author=Tamoor Hussain|title=Hellblade Dev Diary Explores Intricacies of Movement and Control|url=https://www.gamespot.com/articles/hellblade-dev-diary-explores-intricacies-of-moveme/1100-6434365/|date= |publisher=[[Gamespot|Gamespot]]|accessdate=2017-08-08|language=en|archiveurl=https://web.archive.org/web/20160727163010/http://www.gamespot.com/articles/hellblade-dev-diary-explores-intricacies-of-moveme/1100-6434365/|archivedate=2016-07-27}}</ref>，有大约20人的团队一起拍摄。忍者理论工作室对此已经有了丰富的经验。塞娜的配音和[[动作捕捉|动作捕捉]]由德国自由摄影师梅丽娜·朱尔根斯完成。Technoprops为团队提供了捕捉朱尔根斯表情和声音的头盔。头盔上只安装两个摄像头，因为他们觉得四个摄像头会让朱尔根斯不舒服，并影响她的表演。该设备中最昂贵的部分是录音设备，被放置在她的额头周围，以避免录制到呼吸音。因为朱尔根斯所有的音频都是现场录制的，所以团队无法直接给她指示或播放背景音乐。而其他演员预先录制的音轨将同时在她的耳机上播放，因此他们可以彼此同步地进行表演<ref>{{cite web |author1=Ninja Theory |title=地狱之刃开发日志18：拍摄设置 |url=https://www.youtube.com/watch?time_continue=250&v=FACTByOjqyQ}}</ref>。

=== 女主角 ===
塞娜的脸部模型是通过三维扫描创建的，[[塞尔维亚|塞尔维亚]]的3Lateral公司使用其摄影测量技术创建了梅丽娜·朱尔根斯脸部的三维拷贝。梅丽娜是该工作室的视频编辑，进行了塞娜的配音，并提供表情和动作捕捉。<ref name=":0">{{cite web |author1=JEFF CORK |title=揭开地狱之刃神秘女主角的细节 |url=http://www.gameinformer.com/b/features/archive/2017/04/10/revealing-the-finer-details-of-hellblades-mysterious-heroine.aspx |website=gameinformer}}</ref>开发者还使用Artec Eva扫描仪扫描了一名女健身教练维基·王尔德的身体。<ref>{{cite web |author1=Denis Michel |title=Hellblade：Entwicklertagebuch zeigt die Entstehung desKörpersder Protagonistin |url=http://www.gamersglobal.de/news/107064/hellblade-entwicklertagebuch-zeigt-die-entstehung-des-koerpers-der-protagonistin |website=Gamers Global}}</ref>塞娜的头发风格依据凯尔特人历史，上面撒上了石灰，装饰着各种小石头和珠子。塞娜的胸针是凯尔特女性常戴的胸针。<ref name=":0" />

=== 精神病形象 ===
开发团队与专家队伍展开了密切合作，其中包括[[剑桥大学|剑桥大学]]的教授[[保羅·弗萊徹|保罗·弗莱彻]]和[[杜伦大学|杜伦大学]]的[[幻聽|幻听]]专家、心理学教授查尔斯·费尼霍。忍者理论获得了来自威康信托基金的资助。他们也帮助工作室与精神病患者取得联系。<ref>{{Cite web|url=http://www.sciencefocus.com/article/mind/hellblade-senua%E2%80%99s-sacrifice-psychosis-interview|title=地狱之刃：赛如那的献祭是如何处理心理学问题的|accessdate=|author=Lloyd, James|date=2017-10-10|publisher=}}</ref>
VideoGamer.com的爱丽丝·贝尔指出，虽然塞娜的精神病是她人生悲剧的一个原因，但游戏也将她的疾病状况描写为可以给她带来满足的事情，并指出她的幻觉有时“实际上也很可爱，反映了精神病可以表现出的颜色和美丽。”对于整个故事，她说：“你（玩家）了解到塞娜的创伤是来源于她对于精神病的挣扎以及他人对她缺乏了解而导致的。制作组非常谨慎，得到了包括精神病专家和患者一步步的帮助，给我们带来尽可能真实的精神病体现。迪利安说，最困难的战场在心中，虽然塞娜也经历了肉体的战场。<ref>{{cite web |author1=Alice Bell |title=《地狱之刃：赛娜的献祭》评测 |url=https://www.videogamer.com/reviews/hellblade-senuas-sacrifice-review1 |website=videogamer}}</ref>”
