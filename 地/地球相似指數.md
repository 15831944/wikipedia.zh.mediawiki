'''地球相似指數'''（{{lang|en|'''Earth Similarity Index'''}}，[[縮寫|縮寫]]作'''{{lang|en|ESI}}'''）是一個標定其他[[行星|行星]]和[[地球|地球]]相似程度的指數，範圍在0和1之間，地球自身的相似指数以1表示。地球相似指数是针对[[行星|行星]]设计出来的，但也可以用于大型[[天然卫星|天然卫星]]和其他天体。地球相似指數可以經由行星半徑、密度、[[脫離速度|脫離速度]]和表面溫度代入公式計算得知<ref name=esi>{{cite web |title=Earth Similarity Index (ESI) |url = http://phl.upr.edu/projects/earth-similarity-index-esi |publisher=Planetary Habitability Laboratory}}</ref>。

該指數在0.8到1之間的行星代表擁有[[岩石|岩石]]組成的表面、可以在氣候溫和條件下保有類似地球大氣的行星。根据这个标准，太阳系中没有任何与地球类似的行星或卫星：除了地球自身相似指数为1之外，排名第二的金星的地球相似指数为0.78。但最近发现有多个太阳系外行星在此范围内，如[[克卜勒438b|克卜勒438b]]是已证实的系外行星中地球相似指数最高的，达0.88。若开普勒候选星[[KOI-4878.01|KOI-4878.01]]得到证实，其相似指数高达0.98。<ref name="upr.edu">{{cite web |url = http://phl.upr.edu/projects/habitable-exoplanets-catalog/data|title=HEC: Data of Potentially Habitable Worlds - Planetary Habitability Laboratory @ UPR Arecibo |work=upr.edu}}</ref> 

值得一提的是，地球相似指数并不是衡量[[行星适居性|行星适居性]]的指标，该指标由[[行星适居指数|行星适居指数]]来表征。

地球相似指数由下列公式计算：

: <math> ESI = \prod_{i=1}^n \left(1 - \left|\frac{x_i - x_{i_0}}{x_i + x_{i_0}}\right|\right)^\frac{w_i}{n} </math>

其中<math> x_i </math>是行星的某个性质（如表面温度），<math> x_{i_0} </math>是地球上对应性质的参考值（如288K），<math> w_i </math>是该性质的权重，n是行星性质的总数量。权重用于调整不同性质的敏感程度。右表中列出了这些性质及对应的参考值和权重。

地球相似指数可以分为两部分：'''地球内部相似指数'''和'''地球表面相似指数'''。其中平均半径和堆密度用于衡量内部相似指数，而表面温度和逃逸速度用于衡量表面相似指数。两者综合起来的'''全局相似指数'''则用于整体测量。

== 高相似指数行星 ==
[[File:Kepler22b-artwork.jpg|thumb]]。该行星因为与地球相似的大小和温度而被认为具有高地球相似指数，但是其密度当前还未知。]]

著名的地球相似天体都是[[系外行星|系外行星]]。然后由于许多[[系外行星侦测法|系外行星探测方法]]对于地球相似指数的参数无法量化，该排名也仅作参考。例如使用较成功的[[凌日法|凌日法]]可以高精度地测得半径，但是对于质量和密度则常常只能估计。又如[[径向速度法|径向速度法]]能够更精确地测得质量，但是测得精准的半径则较困难。因此通过多种方法来测定可以提高地球相似指数的精度，但多数场合下这是无法做到的。

== 非行星的相似指数 ==
[[File:Io_Earth_Moon_Comparison.png|thumb]]和地球、月球按比例的对比。太阳系的一些卫星和矮行星尽管明显比地球小，但是密度和温度与地球类似，这样它们的地球相似指数也相对较高。理论上地球大小的[[太阳系外卫星|太阳系外卫星]]和其他非行星天体也能有高地球相似指数。]]

地球相似指数还能使用在其他非行星天体上，如[[天然卫星|天然卫星]]、[[矮行星|矮行星]]及[[小行星|小行星]]。但至少对于我们已知的太阳系里这部分天体来说，因为它们的平均密度和温度较低，使得它们的全局地球相似指数也处于一个较低的水平。<ref>pg 143. [http://www.statslab.cam.ac.uk/~pat/misc.pdf Multivariate and other worksheets for R (or S-Plus): a miscellany] P.M.E.Altham, Statistical Laboratory, University of Cambridge. January 10, 2013</ref>
