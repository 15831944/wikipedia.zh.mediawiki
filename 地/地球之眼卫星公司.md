'''GeoEye卫星公司'''（'''{{lang-en|GeoEye Inc.}}'''）是[[美国|美国]]一家商用[[卫星|卫星]]图像公司，坐落于[[弗吉尼亚州|弗吉尼亚州]][[赫恩登|赫恩登]]<ref>{{cite news| url=http://www.reuters.com/finance/stocks/companyProfile?rpc=66&symbol=GEOY.O | work=Reuters}}</ref>，為世界上最大的卫星遥感影像公司，母公司为[[博龍資產管理|博龍資產管理]]。<ref name="globalsecurity.org">[http://www.globalsecurity.org/intell/library/news/2006/intell-060112-geoeye.htm ORBIMAGE Completes Acquisition of Space Imaging; Changes Brand Name to GeoEye]</ref>

其前身是1992创立的[[Orbital图像公司|Orbital图像公司]]旗下的一个部门，当时土地遥感政策法刚刚允许私人公司从事卫星图像领域。1997年从该公司分离出来后于2006年更名为GeoEye卫星公司。<ref name="globalsecurity.org">[http://www.globalsecurity.org/intell/library/news/2006/intell-060112-geoeye.htm ORBIMAGE Completes Acquisition of Space Imaging; Changes Brand Name to GeoEye]</ref>公司目前的主要资产是一颗名为IKONOS的商用卫星。

==公司概况==
GeoEye公司向[[微软|微软]]和[[雅虎|雅虎]]的搜索引擎提供超过253,000,000平方公里的卫星图像。但是[[谷歌|谷歌]]表示公司从未使用GeoEye的服务。<ref name="news.cnet.com">[http://news.cnet.com/8301-1023_3-10028842-93.html?part=rss&subj=news&tag=2547-1023_3-0-5 Google to buy GeoEye satellite imagery -cnet.com - August 29, 2008]</ref>并称其所用的卫星图像来自[[美國國家地理空間情報局|美國國家地理空間情報局]]。

总部位于赫恩登市，并在赫恩登市和[[科罗拉多州|科罗拉多州]]的[[桑顿|桑顿]]市均设有卫星控制中心，在[[密苏里州|密苏里州]]的[[圣路易斯|圣路易斯]]进行图像处理。公司在全球还有用数个卫星监测中心。

从2001年其，公司由前[[宇航员|宇航员]]兼[[中将|中将]]的[[詹姆斯·艾伦·亚伯拉罕|詹姆斯·艾伦·亚伯拉罕]]担任董事长。<ref>{{cite news| url=http://www.reuters.com/finance/stocks/officerProfile?symbol=GEOY.O&officerId=842914 | work=Reuters}}</ref>

GeoEye目前主要的竞争对手是美国[[DigitalGlobe|DigitalGlobe]]公司和法国[[Spot_Image|Spot Image]]公司。

==卫星==
GeoEye卫星拥有多颗[[地球观测卫星|地球观测卫星]]，能够拍摄精度1米以下的可视或者近红外图像。

===IKONOS===
IKONOS于1997年发射，能够以每分钟2000平方公里的速度拍摄地球。其黑白(全色)图像的拍摄精度为82厘米，彩色(多谱段)圖像的拍攝精度為3.2米。它在681千米的高空运行并且每98分钟绕地球一圈。它採用太陽同步軌道，出現在地球同一點上空的時間為上午10:30[[平太陽|平太陽]]時。这颗卫星目前由公司位于[[科罗拉多州|科罗拉多州]]的[[桑顿|桑顿]]的卫星控制中心控制。<ref>{{cite web |url=http://www.geoeye.com/CorpSite/products/imagery-sources/Default.aspx?keywords=ikonos&creative=1350458344&gclid=COS9staay5UCFQxxHgodvCAIiQ |title=存档副本 |accessdate=2012-05-21 |0=2010-05-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20100520075619/http://www.geoeye.com/CorpSite/products/imagery-sources/Default.aspx?keywords=ikonos&creative=1350458344&gclid=COS9staay5UCFQxxHgodvCAIiQ |archivedate=2010-05-20 }}</ref>

===OrbView-2商用卫星===
OrbView-2商用卫星由Orbital公司于1997年负责发射，拍摄地球陆地及海洋表面的全彩图像。渔业工作者用OrbView-2提供的海洋讯息绘制渔业地图。除此之外，它还提供范围达2800公里的大型地图以供海军作战，环境监控以及农作物评估等使用。OrbView-2还携带有[[NASA|NASA]]的[[SeaWiFS|SeaWiFS]]传感器。它由位于公司弗吉尼亚州赫恩登市的卫星控制中心控制，<ref>{{cite web |url=http://eospso.gsfc.nasa.gov/eos_homepage/mission_profiles/docs/Orbview-2_SeaWiFS.pdf |title=存档副本 |accessdate=2008-09-08 |0=2008-09-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20080917183112/http://eospso.gsfc.nasa.gov/eos_homepage/mission_profiles/docs/Orbview-2_SeaWiFS.pdf |archivedate=2008-09-17 }}</ref> 并于2010年12月11日停止拍摄图像。<ref name=OceanColor>{{cite web |url=http://oceancolor.gsfc.nasa.gov/cgi/browse.pl?sen=sw&typ=GAC|title=Ocean Color Browse|author=NASA, Goddard Space Flight Center|date= December 1, 2011|accessdate=December 1, 2011}}</ref>

===OrbView-3===
OrbView-3商用衛星由Orbital公司於2006年負責發射，以每分鐘拍摄2000平方公里的地球表面图像。其黑白(全色)圖像的拍攝精度為1米，彩色(多谱段)圖像的拍攝精度為4米。这颗卫星每天搜集约210,000平方公里的图像数据。它每三天会回到同一个位置，运行倾角为50度。同IKONOS商用衛星一样，它也采用[[太陽同步軌道|太陽同步軌道]]，出现在同一点上空的时间为上午10:30[[平太阳|平太阳]]时。

2007年4月23日，GeoEye向[[美国证监会|美国证监会]]递交了一份[[8-K表格|8-K表格]],声称这颗卫星将永久性的暂停服务。因为尽管公司还能控制OrbView-3在地球轨道上的运转，这颗卫星已不能提供任何有价值的图像。<ref>[http://www.spacetoday.net/Summary/3691 OrbView 3 Satellite Malfunctions]</ref><ref>[http://www.sec.gov/Archives/edgar/data/1040570/000095013307001029/w31696e8vk.htm SEC Form 8-K Regarding Orbview-3 Technical Problem]</ref> 2011年3月13日，该卫星在人为的控制下坠毁于太平洋地区。

=== GeoEye-1 ===
{{主条目|地球之眼-1}}

GeoEye-1商用衛星于2008年9月6日上午11点57分发射，并于58分钟56秒后和美国[[麦道公司|麦道公司]][[德尔塔-2运载火箭|德尔塔-2运载火箭]]成功分离。<ref>{{cite web |url=http://spaceflightnow.com/delta/d335/status.html |title=Delta 2-335 Mission Status Center |accessdate=2008-09-05 |author=Justin Ray}}</ref>它提供范围15.2千米的地球表面图像，其黑白(全色)圖像的拍攝精度為41厘米，彩色(多谱段)圖像的拍攝精度為1.65米。它採用太陽同步軌道，飞行高度681千米，运行倾角98度。出現在地球同一點上空的時間為上午10:30[[平太陽|平太陽]]時。这颗卫星由[[通用动力公司|通用动力公司]]在[[亚利桑那州|亚利桑那州]]负责制造，目前由GeoEye公司位于Herndon的分部控制。

===GeoEye-2===
GeoEye公司和[[ITT公司|ITT公司]]合作，计划于2013年发射这枚卫星，它的拍摄精度将达到25厘米。<ref name="news.cnet.com"/> [[洛克西德马丁|洛克西德马丁]]公司将取代[[通用动力公司|通用动力公司]]为这颗卫星制造运载火箭。<ref>http://www.spacenews.com/earth_observation/100311-lockheed-selected-geoeye2.html</ref>

==拍摄军用设施==
2007年7月7日，Geo Eye公司用自己的一颗GeoEye卫星拍到停泊在中国大连港内的“晋级”(094型)核潜艇。<ref name="china.com">{{Cite web |url=http://club.china.com/data/thread/1013/2525/66/53/5_1.html |title=美国GeoEye卫星公司声称拍到三艘中国航母 |access-date=2013-01-26 |archive-url=https://web.archive.org/web/20150531034846/http://club.china.com/data/thread/1013/2525/66/53/5_1.html |archive-date=2015-05-31 |dead-url=yes }}</ref>

2009年9月29日，GeoEye-1卫星拍摄到在德黑兰西南方100哩处的军事设施，疑似伊朗的铀浓缩地下工厂。<ref name="ww.cnetnews.com.cn">{{Cite web |url=http://www.cnetnews.com.cn/2009/0929/1476889.shtml |title=GeoEye卫星拍下伊朗核网站照片 |access-date=2013-01-26 |archive-url=https://web.archive.org/web/20130212144104/http://cnetnews.com.cn/2009/0929/1476889.shtml |archive-date=2013-02-12 |dead-url=yes }}</ref>

2013年1月1日，GeoEye-1卫星在西安[[阎良|阎良]]军用机场拍摄到[[运-20|运-20]]运输机在跑道上的现场测试图。<ref name="http://www.godeyes.cn">[http://www.godeyes.cn/html/2013/01/05/google_earth_14164.html 国外媒体曝光我国运-20测试GeoEye卫星照]</ref>
