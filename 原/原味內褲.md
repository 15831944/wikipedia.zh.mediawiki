'''原味內褲'''一詞源自[[日本|日本]]，主要販售商品為女性穿過後不清洗的貼身衣物，包括[[內衣|內衣]]、[[內褲|內褲]]等，以女性體味、汗味、[[陰道分泌物|陰道分泌物]]及尿漬等帶有濃烈個人氣味為賣點的變態商品。

==原味內褲興起==

英國《[[獨立報|獨立報]]》報導自[[美國電視影集|美國電視影集]]《[[勁爆女子監獄|勁爆女子監獄]]》(Orange is the New Black)在2015年6月開始出現拍賣原味內衣褲的橋段後，原味內褲逐漸走入一般人的生活中，越來越多的人開始進行拍賣原味內衣褲的生意。<ref>{{Cite web |url = https://www.nownews.com/news/20161001/2256231 |title = 拍賣原味內衣褲　穿越久更受歡迎 |date = 2016/10/01 11:13 |publisher = 今日新聞 |language = 中文 }}</ref>

[[JK經濟|JK經濟]]是在日本一種遊走法律邊緣的行業，由女高中生服侍男性顧客，並提供各種服務，像是散步、占卜、按摩等等，不過變成性交易也是常有的事，為導正此歪風，東京警封更設立「JK Business管制條例」，間接促使JK賣家轉向原味衣物事業進而造成風潮。<ref>{{Cite web |url = https://www.ettoday.net/news/20171012/1029689.htm |title = 日本女高中生網賣原味內衣褲 |date = 2017年10月12日 |publisher = ETtoday新聞雲 |language = 中文 }}</ref>

隨著網路的發展，人們進而透過社群軟體或拍賣網站等管道販售及尋求原味商品，社群軟體來說以 Reddit和 Facebook為最多相關資料可搜尋，Reddit因其本身註冊隱密性高，廣受人們喜愛，Facebook則因幾乎人人皆使用，對賣家來說是現成的廣大市場，但申請帳號較多隱私上限制。近期則在 Instagram上也可發現蹤跡；網站平台部分主流以歐美為主，PantyDeal及 Sofia Grey在美劇《[[勁爆女子監獄|勁爆女子監獄]]》撥出後，也開始逐漸被買賣家關注使用；最後亞洲部分至今年剛設立的 The Panty Drawer原味內褲拍賣網站，主打以中文市場為主。

== 購買動機 ==
[[戀內衣|戀內衣]]：透過觀看、穿脫、嗅聞某些類型的貼身衣物（包括內衣、內褲、絲襪、襪子）等而產生[[性刺激|性刺激]]，由於此類衣物雖輕易可於賣場購買，但為增加想像空間進而延伸出要求由其他女性／男性穿著後購買。

不黯性事：此類通常為[[青少年|青少年]]居多，在實際有[[性行為|性行為]]前或性行為較少的情況下，對異性／同性產生的好奇心促使消費行為。

女裝癖：此類通常為男性居多，通常是為體驗成為女性的感覺而產生消費行為。

素人：[[職業|職業]]涵蓋[[學生|學生]]、OL、[[護士|護士]]、[[空姐|空姐]]、[[家庭主婦|家庭主婦]]等等，通常是為貼補家用或學費而開始販售，品質優劣參差不齊。由於身份雖平易近人，但不同的職業也造就不同的買氣，以女性賣家來說，學生、護士、空姐較熱門。<ref>{{Cite web |url = https://panties.com.tw/%E5%A6%82%E4%BD%95%E5%9C%A8%E7%B6%B2%E8%B7%AF%E4%B8%8A%E8%B3%BA%E9%8C%A2/ |title = 『原味內褲也可以很有質感』，我如何靠販售內褲在網路賺錢？ |date = 2018年05月21日 |publisher = The Panty Drawer原味內褲拍賣網站 |language = 中文 }}</ref>

直播主、小模：此類將原味內褲視為增加人氣的另類方式，一般直播主將原味衣物作偽直播中的小禮物，以增加觀賞人數並達到與粉絲互動的效果中的小禮物，以增加觀賞人數並達到與粉絲互動的效果。<ref>{{Cite web |url = https://star.ettoday.net/news/1105324 |title = 小模羽沫賣起原味內褲，吸引男性粉絲熱烈迴響 |date = 2018年02月01日 |publisher = ETtoday新聞雲 |language = 中文 }}</ref>

公司規模化：這類通常為有專職人員協助賣家行銷接洽買方，將利潤分成的方式合作。
