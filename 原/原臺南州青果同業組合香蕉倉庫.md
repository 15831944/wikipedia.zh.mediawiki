'''原臺南州青果同業組合香蕉倉庫'''位於[[臺南市|臺南市]][[中西區|中西區]]，位於[[西市場|西市場]]南側，為[[西市場|西市場]]的附屬建築。於民國九十二年（2003年）5月13日被公告為臺南市市定古蹟{{r|文}}。目前與[[西市場|西市場]]正整修中。

== 沿革 ==
該建築前身是[[臺灣日治時期|日治時期]]臺灣漁業株式會社所有的魚賣場，在會社與[[臺南舊魚市場|魚市場]]遷到[[臺南運河|臺南運河]]旁後，才改由臺南青果同業組合作為香蕉倉庫使用{{r|王}}。二次大戰後，該建築由臺南市農會管理，直到[[西市場|西市場]]一帶的青果市場遷移他處才遭到廢棄{{r|王}}。

== 建築特色 ==
該建築內設有冷藏、加溫等設施{{r|王}}，而位置則平行西市場南翼，格局為長條的一字型平面，其結構為磚造一層樓高，有著雙坡水斜屋頂，與西市場之間以近四公尺的走道相串聯{{r|文}}。

== 其他 ==
由於在一些網站上有關該建築的圖片常放成隔壁的西市場，導致該建築常和西市場被搞混<ref>王晨羽、張芳瑋.[http://www.peopo.org/portal.php?op=viewPost&articleId=38201 古蹟成廢墟 香蕉倉庫嚴重受創].公民新聞.2009-06-30</ref>。
