'''原築'''（[[英文|英文]]：{{lang|en|'''La Grove'''}}）是[[新鴻基地產|新鴻基地產]]發展的低密度私人屋苑，位於[[香港|香港]][[新界|新界]][[元朗|元朗]]十八鄉路83號，共設4座物業，提供542個住宅單位及161個車位，項目在2008年11月21日獲批核同意書/公契，同月以預售樓花形式首次開售，曾經於[[元朗|元朗]] [[新元朗中心|新元朗中心]]設有示範單位，於2009年9月末入伙。屋苑由[[新鴻基地產|新鴻基地產]]旗下的[[啟勝管理服務有限公司|啟勝管理服務有限公司]]負責管理。

== 簡介 ==
物業為低密度設計住宅，由於屋苑開售時正值[[2007年－2008年環球金融危機|2007年－2008年環球金融危機]]，呎價只約港幣$3,100起。

據發展商售樓書所述，屋苑是以年輕專業人士為目標銷售對象，屋苑整體設計以年輕化路線為主，針對年輕人對優質而簡約的生活<ref>[http://www.hkcd.com.hk/content/2008-06/14/content_2110936.htm  元朗原築走年輕化路線] {{webarchive|url=https://web.archive.org/web/20131015105419/http://www.hkcd.com.hk/content/2008-06/14/content_2110936.htm |date=2013-10-15 }} 《香港商報》2008-06-14</ref>，擁有專屬的生活空間而命名，「原」代表原創的生活，「築」代表簡潔高雅的建築，並推廣「有情、有序、有機及有閑」的生活概念。<ref>[http://www.mpfinance.com/htm/Finance/20081129/Special/gm_gma1.htm 原築 元朗「潮」盤] 《明報》2008-11-29</ref>

== 單位特色 == 
原築共有4座住宅物業（第1-3座及第5座）。大廈樓高17至19層，一梯為1至8伙不等，合共提供542個住宅單位。

單位密度不高，每戶均建有環保[[露臺|露臺]]，主要提供2房至3房單位，[[建築面積|建築面積]]由約586至1,060方呎以上，當中以兩房戶為主，約佔七成，而兩房單位[[建築面積|建築面積]]由約586至662方呎; 至於三房單位分為兩類型，分別為三房連套房，以及三房連套房加工人套房。

除了標準分層戶外，項目提供少量特色戶，分別為連平台、相連戶及複式戶，其中複式戶只設有2伙，面積約2,583及2,706方呎，分佈於各座頂層。

== 住客會所 ==
原築[[住客會所|住客會所]]名為「原築‧會」，佔地6萬平方呎，合共3層，提供多項設施。包括室外游泳池、水力按摩池、[[桑拿|桑拿]]室、兒童遊樂室、[[健身室|健身室]]、[[健康舞|健康舞]]及[[瑜伽|瑜伽]]室、[[水療|水療]]室、[[電子遊戲|電子遊戲]]室、活動室、[[卡拉OK|卡拉OK]]室、[[桌球|桌球]]室、茶座、燒烤區。

會所特設宴會廳，提供大型投射螢幕、專用燒烤區、用餐區等，方便住客隨時宴客舉行派對，每節4小時租金約$800。

為方便住戶預訂會所設施，住戶可透過登入網上住客系統，預訂會所設施外，更可查閱會所最新活動及興趣班內容。<gallery mode=packed>

== 爭議 ==

於2008年興建期間，因高度、設計等問題上，引起了鄰近龍田村村民的不滿及抗議。最終發展公司以六十多萬元與村民達成和解<ref>http://orientaldaily.on.cc/cnt/news/20090221/00176_022.html{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 交通 ==
屋苑位於[[元朗南|元朗南]]，住客步行出[[元朗市中心|元朗市中心]]約7-10分鐘。

居民巴士方面，由於入伙前已有一班業主主動找區議員幫忙，直接與[[運輸署|運輸署]]書信及開會商討，申請屋苑的村巴（居民巴士），因此原築2009年入伙時，於2009年10月17日已經可以正式開辦居民巴士NR964線，循環來往原築及[[新元朗中心|新元朗中心]]，為住戶提供接駁[[西鐵綫|西鐵綫]]、[[輕鐵|輕鐵]]及元朗對外巴士服務。

其後業主們曾經向[[運輸署|運輸署]]申請鳳翔路中途站，但[[運輸署|運輸署]]以道路繁忙及與專線小巴39/39A路線重疊為理由不允許。

NR964線居民巴士服務只限原築住戶乘搭，入伙至今皆由[[陽光巴士|陽光巴士]]（[[新鴻基地產|新鴻基地產]]旗下[[載通國際|載通國際]]有限公司的附屬公司）營辦。

私家車方面，[[停車場|停車場]]有161個車位，包括住宅車位148個 (包括傷殘人士專用車位1個)、電單車位 13個、上落貨車位 4個。<br />
