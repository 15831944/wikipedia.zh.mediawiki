'''原宿'''（{{jpn|j=原宿|hg=はらじゅく|rm=Harajuku|f=y}}，{{Audio|help=no|Ja-Harajuku.oga|發音}}）是[[日本|日本]][[東京都|東京都]][[澀谷區|澀谷區]]的一個地區。
# [[原宿站|原宿站]]周邊至[[表參道|表參道]]周邊的泛稱地名。
#: 大部分屬現在{{link-ja|住居表示|住居表示}}的「[[神宮前_(澀谷區)|神宮前]]」町域。
# 1965年（[[昭和|昭和]]40年）廢止的町名。
#: 原宿一丁目至三丁目，位於表參道北側，相當於現在的神宮前一丁目至四丁目，原宿站（當時町名為隱田三丁目）與[[竹下通|竹下通]]周邊（竹下町）在町域外。

原宿是[[東京都|東京都]][[澀谷區|澀谷區]]的一個地區，範圍約為[[明治神宮|明治神宮]]、[[代代木公園|代代木公園]]、[[竹下通|竹下通]]、[[表參道|表參道]]、[[國立代代木競技場|國立代代木競技場]]一帶，是[[日本|日本]]著名的「年輕人之街」。

== 概要 ==
目前的原宿地區範圍比過去住居表示的「原宿」更大，[[明治通|明治通]]與舊澀谷川（穩田川。現在稱為「貓街」（キャット・ストリート）的遊步道（{{link-ja|舊澀谷川遊步道路|旧渋谷川遊歩道路}}））附近低地周圍，過去稱為「{{link-ja|穩田|穏田}}（おんでん）」。與1965年（[[昭和|昭和]]40年）以前「原宿」町名相當的地區為[[表參道|表參道]]北側、現在[[神宮前_(澀谷區)|神宮前]]2丁目內靠近[[青山_(東京都港區)|青山]]的台地部分、神宮前3丁目大部分地區、神宮前1丁目內{{link-ja|東鄉神社 (澀谷區)|東郷神社 (渋谷区)|東鄉神社}}的台地部分。另一方面[[原宿站|原宿站]]至[[竹下通|竹下通]]周邊稱為「竹下町」。1965年（昭和40年）以後，這一帶實施住居表示，統一稱為「神宮前」，廢止「原宿」町名。

== 歷史 ==
=== 江戶時代 ===
[[File:Watermill_at_Onden.jpg|thumb]]「隱田的水車」]]
江戶時代以前，此地前是[[鎌倉街道|鎌倉街道]]的宿場町。[[後三年之役|後三年之役]]時，[[源義家|源義家]]在這附近整合軍勢，因此當地稱為聚集坡（勢揃い坂，現在的神宮前2丁目）。1582年[[本能寺之變|本能寺之變]]時，[[德川家康|德川家康]]因成功從[[堺市|堺]]回到[[三河|三河]]的「{{link-ja|伊賀穿越|伊賀越え}}」（伊賀越え）而行賞，1590年伊賀者獲得{{link-ja|穩田|穏田|穏田村}}與原宿村。

[[江戶時代|江戶時代]]，伊賀眾為了[[江戶|江戶]]防衛而在[[甲州街道|甲州街道]]南邊的原宿建造組屋敷。另外，此地也有[[安藝藩|安藝藩]]藩主[[淺野家|淺野家]]{{link-ja|江戶屋敷|江戸藩邸}}（現在神宮前4、5丁目）等許多{{link-ja|幕臣|幕臣}}的屋敷。當地農民利用{{link-ja|澀谷川|渋谷川}}等的[[水車|水車]]進行精米、製粉等工作。但是此地土地貧瘠，生產量不高，生活困苦，因此當地農民時常進行[[祈雨舞|求雨]]。傳聞他們當日來回{{link-ja|丹澤山地|丹沢山地|丹澤}}的{{link-ja|大山阿夫利神社|大山阿夫利神社}}與[[榛名山|榛名山]]。

=== 明治時代以後 ===
[[江戶時代|江戶時代]]結束後的1868年（慶應4年），原宿村屬於[[幕府領|幕府領]]。同年，任命{{link-ja|武藏知縣事|武蔵知県事}}松村長為管轄現在澀谷區區域的町村。原宿村在11月編入[[東京府|東京府]]管轄。

1879年（明治11年），{{link-ja|郡區町村編制法|郡区町村編制法}}施行，原宿村與舊武藏國豐島郡地區設立[[南豐島郡|南豐島郡]]與[[北豐島郡|北豐島郡]]，原宿村屬南豐島郡。

1889年（明治22年），[[町村制|町村制]]施行，原宿村與{{link-ja|穩田|穩田|穩田村}}、{{link-ja|千駄谷村|千駄ヶ谷町}}合併為[[南豐島郡|南豐島郡]]千駄谷村。7年後的1896年（明治29年），南豐島郡與[[東多摩郡|東多摩郡]]合併為[[豐多摩郡|豐多摩郡]]，原宿屬豐多摩郡千駄谷村。

==== 二戰後 ====
[[File:Meiji-Jingu-Shrine-05.jpg|thumb]]]]
{{see also|表參道#歷史}}
1906年（[[明治|明治]]39年），[[山手線|山手線]]延伸的[[原宿站|原宿站]]開業，1919年（[[大正|大正]]8年）創建[[明治神宮|明治神宮]]，整備[[表參道|表參道]]。

[[太平洋戰爭|太平洋戰爭]]末期的1945年（昭和20年），[[美軍|美軍]]發動的[[東京大空襲|東京大空襲]]讓這一帶化為焦土<ref name="takeshitaofficial">[http://www.takeshita-street.com/about.html 原宿竹下通の歩み] 原宿竹下通商店会について 『竹下通オフィシャルマップ』 平成23年7月15日閲覧</ref>。終戰後，美軍接收[[代代木公園|代代木練兵場]]舊址，改建為[[美國空軍|美國空軍]]軍舍「華盛頓高地」（ワシントンハイツ），表參道沿線Kiddy Land（キディランド）、Oriental Bazzar（オリエンタルバザー）、富士鳥居等以美軍官兵家庭為主的商店陸續開業。

==== 1960年代 ====
[[File:Harajuku-Apartments.jpg|thumb]]
[[File:Coop-Olympia-Harajuku-02.jpg|thumb]]
1960年代實施{{link-ja|住居表示|住居表示}}，[[原宿站|原宿站]]周邊至[[青山通|青山通]]、[[澀谷區|澀谷區]]「[[竹下通|竹下町]]」、「{{link-ja|穩田|穏田}}1～3丁目」、「原宿1～3丁目」等地區成立澀谷區「[[神宮前_(澀谷區)|神宮前]]」。1964年（[[昭和|昭和]]39年），近隣的[[國立代代木競技場|代代木體育館]]成為[[1964年夏季奧林匹克運動會|東京奧運]]場館。

1966年（[[昭和|昭和]]41年），在[[明治通|明治通]]與表參道交差點附近營業的「ドライブイン・ルート5<ref>所在地在現在的[[La_Foret原宿|La Foret原宿]]附近</ref>」每晚吸引了富家子弟前往。稱為「原宿族」的年輕人，開車飛馳在[[表參道|表參道]]的噪音<ref>這些車是要前往[[赤坂|赤坂]]與[[橫濱市|橫濱]]（[[山下町|山下町]]、{{link-ja|本牧|本牧}}）等地</ref>與公共道德等問題造成地方居民的困擾。

另外，隨著1962年（[[昭和|昭和]]37年）公寓大廈法（マンション法）的實施，「第一次公寓大廈熱潮」興起，高級公寓陸續興建<ref name="nomucom">[http://www.nomu.com/mansion/library/theater/history.html 分譲マンションの歴史]　『不動産情報サイト nomu.com』　野村不動産アーバンネット株式会社、平成23年12月22日閲覧</ref>。雖然郊外有{{link-ja|住宅都市整備公團|住宅・都市整備公団}}大量供給「[[團地|團地型]]」集合住宅，但[[地產商|地產商]]在都心部規劃了全新居住方式的高級公寓住宅<ref name="nomucom"/>，而原宿、澀谷地區是早期高級公寓建築最為集中的地區<ref name="allabout">[http://allabout.co.jp/gm/gc/26400/ 高級マンションのルーツは赤坂にあり]　坂根康裕　『オールアバウト』　2008年2月26日掲載、平成23年12月23日閲覧</ref>。當時原宿建設多棟高級公寓是希望[[日本戰後經濟奇蹟|高度成長期]]的日本能追隨歐美的都會生活<ref name="allabout"/>。

原宿最早的高級公寓之一是1958年（昭和33年）第一生命住宅（現在的相互住宅）<ref>後由第一Mansion Series建設。</ref>在原宿站前建設的豪華租賃[[公寓|公寓]]「原宿公寓」（原宿アパートメンツ）。1960年（昭和35年），以{{link-ja|淺井慎平|浅井慎平}}與[[塔摩利|塔摩利]]等文化人居住而聞名的原宿中央公寓（原宿セントラルアパート）<ref>以淺井慎平與[[塔摩利|塔摩利]]等文化人居住而聞名，1986年（昭和61年）拆除</ref>竣工<ref name="takeshitaofficial"/>。1965年（昭和40年）完成的{{link-ja|Co-op Olympia|コープオリンピア}}是第一次公寓大廈熱潮中高級公寓的代表作<ref name="nomucom"/>。同年的「Green Fantasia」（グリーン・ファンタジア）、「Park Twin」（パーク・ハイツ）、1966年（昭和41年）的「Corpo Olympia Annex」（コーポ・オリンピア・アネックス）<ref>與Co-op Olympia沒有關係。</ref>與表參道沿線的豪華公寓群，成為當時雜誌拍攝的愛用地點<ref>装苑、Dress making、MEN'S CLUB、mc Sister等</ref>。

1959年（昭和34年），原宿地區第一個模特兒經紀公司「Eddie Arab Model Production」（エディ・アラブ・モデル・プロダクション，現E-promotion（イイプロモーション））成立。1966年（昭和41年），原宿地區最早的高級服裝店「Mademoiselle non-non」（マドモアゼルノンノン）開幕。1967年起，現在的神宮前交差點起北側的明治通沿線，高級喫茶店與配件店興起，有裏原宿之稱。

==== 1970年代 ====
1970年代，以時尚為中心的[[青年文化|年輕人文化]]從原來的[[新宿|新宿]]、原宿逐漸往[[澀谷|澀谷]]方向移動。1970年（昭和45年）創刊的女性雜誌《{{link-ja|anan|an・an}}》（アンアン）與翌年創刊的女性{{link-ja|時尚雜誌|ファッション雑誌}}《[[non-no|non-no]]》都以原宿的時尚街道為背景。受此影響，拿著這些時尚雜誌走在原宿街頭的年輕女性被稱作「安儂族」（アンノン族）。

1971年（昭和46年），位於[[竹下通|竹下通]]出口附近的明治通、販售歐洲時尚服裝、配件、家具的複合大樓「Palais France」（パレフランス）竣工<ref name="takeshitaofficial"/>。翌年，[[帝都高速度交通營團|營團地下鐵]]的[[東京地下鐵千代田線|千代田線]]全面開通，設置[[明治神宮前站|明治神宮前站]]<ref name="takeshitaofficial"/>。1976年（昭和51年）左右，時尚浪潮也進一步擴展到竹下通<ref name="takeshitaofficial"/>。

1978年（昭和53年），時尚大樓「[[La_Foret原宿|La Foret原宿]]」開幕，此時原宿已經成為時尚、服飾的中心。設計師與攝影師的工作室陸續進駐本地公寓。原宿中央公寓一樓的喫茶店 『レオン』是淺井慎平、[[塔摩利|塔摩利]]、[[渥美清|渥美清]]、[[伊丹十三|伊丹十三]]、{{link-ja|操上和美|操上和美}}等藝人與有錢人聚集的場所。

==== 1980年代 ====
1980年代，女性偶像歌手因為拍攝娛樂雜誌需要大量的服裝。之後文化屋雜貨店開幕引起轟動。

1982年～1984年，{{link-ja|桑原茂一|桑原茂一}}開設日本最早的Live bar「直立猿人」（ピテカントロプスエレクトス）。

1980年代，受到[[竹之子族|竹之子族]]影響，[[竹下通|竹下通]]開始發展。80年代後半，{{link-ja|藝人商店|タレントショップ}}逐漸增加。1977年（昭和52年）開始的[[步行街|步行街]]聚集大量年輕人，搖滾族與街頭樂團浪潮興起，最盛期每日人數可達10萬人。1996年（[[平成|平成]]8年）步行街中止。

==== 1990年以後 ====
[[File:Omote-sando03s3200.jpg|thumb]]]]
1990年代，海外知名的時尚品牌旗艦店陸續在表參道開店。[[神宮前_(澀谷區)|神宮前]]三丁目、神宮前四丁目的住宅區內，新興的時尚潮流商店林立，成為「[[裏原宿|裏原宿]]」的一區。2006年（平成18年）[[表參道之丘|表參道之丘]]開幕。2008年（平成20年）[[東京地下鐵副都心線|東京地下鐵副都心線]]開業，預期將會有進一步的發展。
