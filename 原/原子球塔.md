'''原子球塔'''（'''Atomium'''）是为1958年[[比利時|比利時]][[布鲁塞尔|布鲁塞尔]]舉辦[[世界博览会|世界博览会]]而建的金属结构的纪念性建筑物，André Waterkeyn设计，高102米，包括9个直径18米的球体，与连接圆球的钢管构成相当于放大1650亿倍的α铁的正方体[[晶体结构|晶体结构]]。目前（2008年）三个球体向游客开放，顶部的球体提供观察布鲁塞尔全景的观景地。<ref>{{cite web |url=http://www.atomium.be/Article.aspx?lang=en&id=14&noback=1 |title=存档副本 |accessdate=2009-04-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20090420035924/http://www.atomium.be/Article.aspx?lang=en&id=14&noback=1 |archivedate=2009-04-20 }} Technical Features of the Atomium.</ref>.

12條管道連接著不銹鋼球形成一個立方體，另外立方體中心球體連接8條管道通往立方體外圍8個球體。每個連接的管道內都有電動扶梯與電梯，連接包含展覽廳與公共場所等的六個可開放的球體。頂部球體可以觀賞布魯塞爾全景。[[美國有線電視新聞網|美國有線電視新聞網]]把它命名為歐洲最怪異的建築。<ref>{{cite web|last=Benavides Canepa|first=Jessica|title=11 of Europe's most bizarre buildings|url=http://edition.cnn.com/2013/01/18/travel/europe-bizarre-buildings/index.html|work=[[CNN|CNN]]|publisher=Turner Broadcasting System, Inc|accessdate=25 January 2013|date=24 January 2013}}</ref>

==整修作業==
[[File:Atomium_-_Bruxelles,_Belgium_-_October_31,_2010_03.jpg|thumb]]
原子球塔的整修作業開始於2004年3月；同年10月，為了整修而開始休館，直到2006年2月18日才重新開放，整修包括將原子球體上已退色的鋁板更換成不銹鋼材料。為了能籌措裝修費用，館方將更換的鋁板當紀念品販售。約2公尺長，三角形形狀的鋁板以1000歐元售出。<ref>{{Cite web |url=http://veerle-v2.duoh.com/blog/comments/the_atomium_restored/ |title=存档副本 |access-date=2014-07-23 |archive-url=https://web.archive.org/web/20170923194316/http://veerle-v2.duoh.com/blog/comments/the_atomium_restored/ |archive-date=2017-09-23 |dead-url=yes }}</ref>
