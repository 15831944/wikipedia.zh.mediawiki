'''原子实'''（core或atomic kernel），物理学界又称之为“原子芯”,是一个应用越来越广泛的概念。<ref>{{cite web |title=原子实究竟代表什么?, 原子实究竟代表什么? |url=http://www.dxhx.pku.edu.cn/CN/10.3866/PKU.DXHX19920120 |website=大学化学 |pages=51–52 |language=cn |doi=10.3866/PKU.DXHX19920120 |date=1992-01-15}}</ref>在书写[[电子排布式|电子排布式]]时，通常把内层电子已达到[[稀有气体|稀有气体]]结构的部分写成[[稀有气体|稀有气体]]的元素符号外加方括号的形式来表示，这部分称为“原子实”。<ref>{{cite book |author1=吉林大学、武汉大学、南开大学  |author2=宋天佑、程鹏、徐家宁、张丽荣等编 |title=无机化学(第三版)上册 |date=2018年11月 |publisher=高等教育出版社 |location=北京 |isbn=978-7-04-042426-3 |language = zh|page=165 |accessdate=2019-01-09}}</ref>比如钪的基态原子[[电子排布式|电子排布式]]为：1s<sup>2</sup>2s<sup>2</sup>2p<sup>6</sup>3s<sup>2</sup>3p<sup>6</sup>3d<sup>1</sup>4s<sup>2</sup>，也可以表示为[Ar]3d<sup>1</sup>4s<sup>2</sup>。
根据定义可知，不可由原子实直接表示对应[[稀有气体|稀有气体]]的[[电子排布式|电子排布式]]。

=== 争议 ===
原子实有另一种定义：原子中，由除[[价电子|价电子]]以外的内层电子组成原子实。<ref>{{Cite web|url=http://xuewen.cnki.net/R2006073280000312.html|title=原子实_CNKI学问|accessdate=2019-01-09|work=xuewen.cnki.net}}</ref>但是这种定义有弊端，比如对于<sub>82</sub>Pb：[Xe]4f<sup>14</sup>5d<sup>10</sup>6s<sup>2</sup>6p<sup>2</sup>来说，价电子构型是6s<sup>2</sup>6p<sup>2</sup>，[[价电子层|价电子层]]以里是[Xe]4f<sup>14</sup>5d<sup>10</sup>，即使是这种定义的书也认为[Xe]才是原子实，而[Xe]4f<sup>14</sup>5d<sup>10</sup>不属于原子实。<ref>{{cite web |title=原子实究竟代表什么?, 原子实究竟代表什么? |url=http://www.dxhx.pku.edu.cn/CN/10.3866/PKU.DXHX19920120 |website=大学化学 |pages=51–52 |language=cn |doi=10.3866/PKU.DXHX19920120 |date=1992-01-15}}</ref>
