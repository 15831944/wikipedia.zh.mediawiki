'''原子弹'''又称'''裂变弹'''（{{lang-en|Atomic bomb}}）{{r|原子弹}}，是一种利用[[原子核|核]]原理制成的[[核武器|核武器]]。由[[美国|美国]]最先研制成功，具有极强的破坏力，在爆炸的同时会放出强烈的[[核辐射|核辐射]]，危害生物和非生物组织。第一个裂变（原子弹）试爆释放出的能量为约20,000吨炸藥（见[[三位一體核試|三位一体核试爆]]）的相同的当量。第一个热核（氢弹）试爆释放相同的能量为10,000,000吨炸藥的当量。

==原子弹原理及其结构==
===原子弹爆炸原理===
[[File:Nuclear_fission.svg|250px]] 
{{main|链式反应|核裂变}}
原子弹是利用[[铀|铀]]和[[鈈|鈈]]等较容易裂变的[[重原子核|重原子核]]在[[核裂变|核裂变]]瞬间可以发出巨大能量的原理而发生爆炸的{{r|原子弹}}。

[[铀-235|铀-235]]和[[钚-239|钚-239]]此类重原子核在[[中子|中子]]的轰击后，通常会分裂变成两个中等质量的核{{r|原子弹}}，同时再放出2到3个中子和200兆电子伏的能量{{r|原子弹}}。在裂变中放出的中子，一些在裂变系统中损耗了，而一些则继续进行重核裂变（继续轰击重原子核）反应{{r|原子弹}}。只要在每一次的核裂变中所裂变出的中子数平均多余一个（即中子的[[增值系数|增值系数]]大于1），那么核裂变即可以继续进行，一次一次的反应后，裂变出的中子总数以指数形式增长{{r|原子弹}}，而产生的能量也随之剧增。如果不加控制，最终，这个裂变系统会变为一个剧烈的链式裂变反应{{r|原子弹}}。

在此类重核裂变反应中，系统可以在极短的时间内释放出大量的能量{{r|末日武器}}。当“下一代”中子数定位两个时，在不到一微秒的时间内，一千克的铀或鈈中会有<math>2.5\times 10</math><sup>24</sup>个原子核发生裂变反应{{r|原子弹}}，而就在这不到一微秒的时间内，此反应所产生出能量相当于2万吨TNT当量{{r|原子弹}}。这也是原子弹那极具破坏性威力的来源。

而在原子弹的实际使用及爆炸中，需要提高爆炸的威力，为了利用“快中子裂变体系”，需要使用高浓度的裂变物质作为装药{{r|原子弹资料}}，同时装药量必须远远超过[[临界质量|临界质量]]，使得中子的增值系数远远大于一{{r|原子弹资料}}。

===原子弹结构及其装药===
原子弹是由引爆控制系统，高能炸药，反射层，含有核装料的核部件，中子源及弹壳所组成{{r|原子弹资料}}。
====高能炸药====
高能炸药是推动和压缩反射层以及核装料的能量来源{{r|原子弹资料}}。
====反射层====
反射层一般由铍或铀-238组成，其作用为反射链式反应中射出反应系统的中子，使其回到反应过程中继续参与链式反应{{r|原子弹资料}}。

铀-238不止可以反射中子，因为其密度较大，可以减缓核装料在释放能量过程中的膨胀，使得链式反应可以维持较长时间{{r|原子弹资料}}。

====核装料====
现在，能够大量得到且可以使用于原子弹的装药有[[铀-233|铀-233]]，[[铀-235|铀-235]]和[[鈈-239|鈈-239]]铀235里面有钚239，[[钍232|钍232]]里面有铀233{{r|原子弹资料}}。

核装药为原子弹引爆的主体，只有它的体积或质量超过一定的临界值，原子弹才可能发生爆炸{{r|原子弹资料}}。

===原子弹类型===
根据原子弹的引发机制的不同，原子弹可分为“枪式”和“收聚式（内爆式）”两种{{r|原子弹类型}}。现代原子弹通常综合了这两种结构的特点，以提高核装药的利用率和增强破坏力。
[[File:Fission_bomb_assembly_methods_zh-hans.svg|right]]

====枪式原子弹====
枪式原子弹内，两块皆小于临界体积的半球形的裂物质分开一定距离放置{{r|原子弹类型}}，中子源置于两瓣裂物质中间{{r|原子弹类型}}。在核装药的球面上包裹了一层反射中子的材料（反射中子，提高链式反应效率），而中子反射层外是高速炸药、传爆药以及雷管{{r|原子弹类型}}。雷管再与起爆器相连，起爆器能够自动的起爆炸药{{r|原子弹类型}}。

在起爆原子弹时，两块半球形裂变物质在炸药的轰击下迅速压缩为一个扁球形，即刻即达到超临界状态{{r|原子弹类型}}。中子源此时释放出大量的中子参与到链式反应中，使得裂变物质在极短的时间内释放出巨大的能量，最终使得原子弹起爆{{r|原子弹类型}}。
====收聚式原子弹====
收聚式原子弹与枪式原子弹不同，普通的烈性炸药被制成球形装置，并且把多个小于临界体积的核装药制成小球置于炸药球当中{{r|原子弹类型}}。

炸药起爆时，核装药小球被迅速压紧超过临界体积并起爆原子弹{{r|原子弹类型}},收聚式原子弹的结构较为复杂，但是装药利用率高，破坏性大{{r|原子弹类型}}。

==原子弹的破坏效应==
原子弹的破坏力和杀伤破坏方式主要有光辐射、冲击波、早期核辐射、电磁脉冲及放射性沾染等{{r|原子弹类型}}。
===光辐射===
{{main|光辐射}}
在原子弹引爆后，核爆过程会释放出强烈的辐射光。1枚当量在2万吨左右的原子弹在当空爆炸后，距离爆炸核心7000米的地方人会受到比阳光强13倍的光辐射的照射{{r|原子弹类型}}。而在2800米范围内，光辐射会使人迅速致盲，且皮肤会因为光辐射照射而大面积灼伤溃烂，一些物体也会燃烧{{r|原子弹类型}}。

===冲击波===
{{main|冲击波}}
原子弹爆炸后，核爆会产生出一种巨大的气流超压{{r|原子弹类型}}。一枚三万吨当量的原子弹爆炸后，在离爆炸核心800米处，冲击波会以200米每秒的速度襲捲一切{{r|原子弹类型}}。

===早期核辐射===
{{main|核辐射}}
在原子弹最初起爆的几十秒中内，核爆会释放出中子流和γ射线{{r|原子弹类型}}。一枚两万吨当量的原子弹爆炸时，离它1100米以内的人员单位会受到射线和中子流的严重杀伤{{r|原子弹类型}}。

===电磁脉冲===
{{main|电磁脉冲}}
原子弹爆炸所造成的核爆会制造出电磁脉冲，而电磁脉冲的电场强度可达1万至10万伏，完全可以摧毁起爆点周围的一切电子设备{{r|原子弹类型}}。

===放射性沾染===
{{main|放射性落下灰}}
在原子弹爆炸后，随着蕈狀雲的飘散会有大量的放射性粉尘飘落到地面，会对人体造成照射或皮肤灼伤，严重者最终导致死亡{{r|原子弹类型}}。

==原子弹历史==
早在1938年年底，纳粹德国科学家[[奥托·哈恩|奥托·哈恩]]和[[弗里茨·施特拉斯曼|弗里茨·施特拉斯曼]]发现了[[核裂变|核裂变]]现象，此現象也被刊登於科學雜誌上。至1939年9月，科學界普遍同意製成原子彈的可能性，唯技術細節尚未明朗<ref name="世紀大決戰168-171">《世紀大決戰：第二次世界大戰的總體分析》，第168頁-第171頁。</ref>。除了德國外，當時研究原子彈的國家另有[[法國|法國]]、[[美國|美國]]、[[蘇聯|蘇聯]]、[[英國|英國]]和[[日本|日本]]。
===軸心國===
{{seeAlso|德国核武器开发计划|日本核研究}}
[[File:German_Experimental_Pile_-_Haigerloch_-_April_1945.jpg|right]]
德國因為汉堡大学的教授向德国军方进言的要求而研发核武器。1939年9月26日，德国军备规划局制定了铀规划，并着手开始进行对核武器的研制{{r|原子弹研究}}，以[[维尔纳·海森堡|维尔纳·海森堡]]為首組成特別小組<ref name="世紀大決戰168-171"/>。接著在1940年於德國佔領的[[挪威|挪威]]設立由[[法本公司|法本公司]]管理的[[重水|重水]]製造廠以大量生產重水來製造[[鈽|鈽]]，也研究著如何分離[[鈾-235|鈾-235]]。1941年尾，由於德國進攻蘇聯的作戰沒有達到期望的結果，戰爭轉為長期，絕大多數的資源被移到急需的[[坦克|坦克]]和[[飛機|飛機]]上，並得到了「3年內無法將原子彈實用化」的結論<ref name="世紀大決戰168-171"/>。1942年初，德國取消了研究計畫，雖然研究小組仍持續在工作並建造了數座小型實驗用核反應爐，但[[納粹德國|納粹德國]]的核武研究計畫就停留於理論上，已無有系統的生產計畫。雖然後來生產鈽的挪威工廠被英國突擊隊摧毀，但實質上對德國核武開發沒有決定性影響<ref name="世紀大決戰168-171"/>。

与此同时，日本也正筹划他们的核武器研制计划。1941年5月，日本陆军大臣[[东条英机|东条英机]]批准了空军科技署的“制造铀弹报告”，近百名科学家在[[仁科芳雄|仁科芳雄]]的带领下开始执行“仁方案”{{r|原子弹研究}}。而此时，日本的核计划却遇到阻力。由于国内缺乏可供研究的“铀”，且由於各大研究所並未合作而使得資源分散、進度緩慢。日本的核计划在1943年初就宣告終止，原因是日本估計即使是進行單一的鈾分離工程也必須消耗全國10％的電力、半年的銅產量和十年的時間完成其必要設施，代價過於龐大<ref name="世紀大決戰168-171"/>。但研究人員仍持續在工作，1945年5月，德国U-234潜艇因德国战败，在大西洋上浮，向盟国投降，所载运往日本的540kg氧化铀235被美军秘密截获并被用于曼哈顿计划，并可能成为以铀235为燃料的小男孩（Little Boy）原子弹的材料来源；美国对外宣称，其于马六甲被美舰击沉；日本之後也因為盟軍越來越頻繁的轟炸，研究工作一直受阻，核武發展直到戰爭結束都未脫離理論階段。

===同盟國===
{{main|曼哈顿计划}}

由于[[纳粹德国|纳粹德国]]对德国本土的[[犹太人|犹太人]]的迫害，使得许多犹太人科学家逃到[[美国|美国]]避难，在当中，很多人提醒美国当局，德国人正在制造一种“超级武器”{{r|曼哈顿计划}}。

1939年，[[匈牙利|匈牙利]]科学家[[利奥·西拉德|利奥·西拉德]]担心德国制造出核武器，希望美国政府能够先行研制出，但是当他找到一些官员时，他们认为只是天方夜谭，最终，他只得找到[[爱因斯坦|爱因斯坦]]<ref>{{Cite web

 |title=爱因斯坦自认为犯下一生最大的错误
 |work=人民网
 |accessdate=2010-08-11
}}</ref>。

1939年8月2日，在爱因斯坦与其他几名科学家一同致信[[美国总统|美国总统]][[罗斯福|罗斯福]]，建议美国政府务必在纳粹德国前研制出核武器{{r|爱因斯坦建议}}。罗斯福并没有对此表示重视，而负责转交信件的罗斯福的科学顾问亚历山大·萨克斯担负起了说服罗斯福的责任，他引用了一个著名的例子：当年拿破仑没有采取富尔顿蒸汽船的建议，使其没有建立起强大的海军，最终不敌英国海军。罗斯福被萨克斯说服，决定研制核武器{{r|曼哈顿计划}}。

1939年10月19日，罗斯福正式签署了研制原子弹的文件，两日后，美国成立了“研究原子武器的委员会”，代号为S-11<ref>{{Cite web

1941年12月7日，[[日本|日本]]轰炸的[[美国海军|美国海军]]军港“[[珍珠港|珍珠港]]”，这使得美国政府更加下定决心加速执行他们的“核计划”。1942年6月，美国原子弹研制计划正式开始，由于总部设在美国[[纽约|纽约]]的[[曼哈顿区|曼哈顿区]]，所以此项计划又被稱为“曼哈顿计划”{{r|曼哈顿计划}}。同月，美国总统罗斯福与[[英国首相|英国首相]][[丘吉尔|丘吉尔]]在[[华盛顿|华盛顿]]会晤，两国决定联合研制原子弹，即把英国原有的“合金管”计划融入到“曼哈顿计划”中{{r|曼哈顿计划}}。而此前逃亡到英国的一些法国科学家此时也加入到原子弹研究中{{r|曼哈顿计划}}。

===新墨西哥州的蘑菇云===
{{main|三位一體核試}}
1942年，[[洛斯阿拉莫斯实验室|洛斯阿拉莫斯实验室]]在[[新墨西哥州|新墨西哥州]]一望无际的沙漠上建立了起来{{r|曼哈顿计划}}。

1945年7月初，美国终于研制出了三枚原子弹，他们立即准备着手实验{{r|曼哈顿计划}}。当得知要进行原子弹实验时，美国总统杜鲁门推迟了波茨坦会议时间，因为其希望该武器能够在波茨坦会议期间爆炸，以提高美国的国际地位{{r|曼哈顿计划}}。

7月15日，杜鲁门到达波茨坦。7月16日5时29分45秒，人类历史上的首枚原子弹在阿拉莫戈多沙漠上爆炸。实验极其成功{{r|曼哈顿计划}}。

===从广岛到长崎===
[[File:Nagasakibomb.jpg|250px]] 
{{main|廣島市原子彈爆炸|長崎市原子彈爆炸}}
在1943年后，美军在太平洋战场上损失惨重，[[硫磺岛战役|硫磺岛战役]]和[[冲绳岛战役|冲绳岛战役]]使得美军伤亡巨大，经过这几场战争的伤亡比率推算，美军需要牺牲掉100万陆战队员的生命才能夺下日本本土，所以在得知原子弹成功爆炸后，军方极力要求对日本人使用这种新式武器{{r|曼哈顿计划}}。

但是此时许多科学家却开始反对美国使用原子弹，奥本海默认为日本的失败已是必然，没有必要使用原子弹。一些物理学家也联名致信要求美国不要使用原子弹{{r|曼哈顿计划}}。

但是美国坚持使用原子弹，除了希望能够尽快解决太平洋战争以外，也希望削弱苏军对日作战的意义{{r|曼哈顿计划}}。

1945年7月30日，美英中三国在波茨坦向日本发出最后通牒，威胁说日本如果不立即投降，“日本即將迅速完全毀滅”<ref>波茨坦公告</ref>。日本于同日拒绝接受[[波茨坦公告|波茨坦公告]]。波茨坦公告后，[[杜鲁门|杜鲁门]]在回国途中向军方下达命令：“去投掷那颗大炸弹吧，现在没有任何选择的余地了。”{{r|曼哈顿计划}}

美国投向广岛和长崎的两颗原子弹都是从[[提尼安岛|提尼安岛]]装载起飞的。现在岛上留有当时安放原子弹的铁架等遗迹。

1945年8月6日早上8點15分，艾诺拉·盖号在广岛上空投下人类历史上第一枚用于战争的槍式原子弹（[[小男孩原子弹|小男孩原子弹]]），造成广岛十万多居民死亡。

1945年8月9日，B-29轰炸机大貨櫃在长崎上空投下收聚式原子彈[[胖子原子弹|胖子原子弹]]，导致长崎市近四万人直接死亡<ref>[http://www.yale.edu/lawweb/avalon/abomb/mp10.htm The Avalon Project : The Atomic Bombings of Hiroshima and Nagasaki] {{webarchive|url=https://web.archive.org/web/20110625010544/http://www.yale.edu/lawweb/avalon/abomb/mp10.htm |date=2011-06-25 }}</ref>，总计十四万人员死伤{{r|曼哈顿计划}}。

1945年8月15日日本標準時間中午12點，[[裕仁天皇|裕仁天皇]]宣布[[日本戰敗|日本戰敗]]，[[第二次世界大战|第二次世界大战]]结束。

===二次世界大战之后===
[[Image:Chinese_nuclear_bomb_-_atom_missile_A2923.jpg|thumb]]
{{main|拥有核武器的国家列表}}
苏联得知美国成功研制原子弹后，根据谍报立即展开制造原子弹的紧急计划，并于1949年试爆第一枚原子弹。英国在参考曼哈顿计划的情况下，于1952年试爆了自己的原子弹。1960年，法国试爆原子弹成功。

中华人民共和国在原子弹制造初期获得少量苏联援助，后来在旅美归来的[[邓稼先|邓稼先]]带领下，于1959年6月开始研制核武器，1962年成功，于1964年引爆了自己的原子弹。1974年，印度引爆了自己的原子弹，而巴基斯坦也于1997年试爆了自己的核武器。至2017年为止，朝鲜则于2006年10月，2009年5月，2013年2月，2016年1月和9月共进行了五次核试验。媒体报道2017年4月15日朝鲜太阳节可能会进行第六次核爆试验。
