'''原阳县'''是[[中华人民共和国|中华人民共和国]][[河南省|河南省]][[新乡市|新乡市]]下属的一个县。面积1329平方公里，人口71.4万<ref>{{Cite web |url= http://www.yuanyang.gov.cn/web/mlxxx/xxxq/ |title= 原阳县情 |publisher= 原阳县人民政府 |accessdate= 2011-05-21 |quote= 全县辖6镇11个乡，人口71.4万，区域面积1329平方公里。 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[邮政编码|邮政编码]]453500，县政府驻城关镇。

== 历史沿革 ==
[[秦朝|秦]]时始置[[阳武县|阳武县]]，[[西汉|西汉]]析置[[原武县|原武县]]。[[宋神宗|宋神宗]][[熙宁|熙宁]]年间，并原武入阳武，[[宋哲宗|宋哲宗]][[元祐|元祐]]年间复置原武县。

1936年[[原武县|原武县]]与阳武县合并，改设[[博浪县|博浪县]]。同年又复旧。[[抗日战争|抗日战争]]期间，原武县、阳武县均沦陷。1945年3月，[[解放区|解放区]]两县境内置“原阳县”政权。而光复后的[[国民政府|国民政府]]仍置原武县、阳武县。[[第二次国共内战|第二次国共内战]]爆发后，国共双方对此地曾有争夺。1947年3月[[豫北战役|豫北战役]]中，[[晋冀鲁豫野战军|晋冀鲁豫野战军]]占领[[濮阳县|濮阳]]、[[封丘县|封丘]]、[[延津县|延津]]、[[原武县|原武]]、[[阳武县|阳武]]等[[县城|县城]]及据点<ref>{{Cite web |url=http://www.jconline.cn/Contents/Channel_6942/2012/0528/822244/content_822244.htm|title=豫北战役——回顾|accessdate=2016-05-18 |author=责编：靳敏 |date=2012-05-28 |work= |publisher=原阳县档案信息网|language=Zh-hans }}</ref>。至1948年10月，中共政权控制原武县、阳武县全境。后[[华北人民政府|华北人民政府]]要求撤消战时行政区划，恢复旧行政建制。原阳县官员因原阳县人口过少，请求保留。1949年[[中华人民共和国|中华人民共和国]]建国后，属[[新乡专区|新乡专区]]。最终在1950年3月13日，[[平原省人民政府|平原省人民政府]]通令“奉准原武、阳武二县合并，定名为原阳县”<ref>{{Cite web |url=http://www.hada.gov.cn/w_NewsShow.asp?ID=257:25545|title=原阳原样|accessdate=2016-05-18 |author= |date=2013-09-11 |work= |publisher=原阳县档案信息网|language=Zh-hans }}</ref>。

==行政区划==
[[File:Yuanyanggov.jpg|thumb]]
2005年之前，原阳县辖2个镇、20个乡：陡门乡、城关镇、原武镇、太平镇乡、大宾乡、王杏兰乡、包厂乡、齐街乡、阳阿乡、师寨乡、陡门乡、官厂乡、郭庄乡、祝楼乡、桥北乡、黑洋山乡、梁寨乡、韩董庄乡、蒋庄乡、路寨乡、葛埠口乡、福宁集乡、靳堂乡。县政府驻城关镇。<br />
2005年，原阳县乡镇区划调整：撤销黑洋山乡，将其行政区域划归葛埠口乡管辖，葛埠口乡人民政府驻地不变；撤销王杏兰乡，将其行政区域划归福宁集乡管辖，福宁集乡人民政府驻地不变；撤销包厂乡，将其行政区域划归靳堂乡管辖，靳堂乡人民政府驻地不变；撤销梁寨乡，将其行政区域划归太平镇乡管辖，太平镇乡人民政府驻地不变；撤销郭庄乡，将其行政区域划归陡门乡管辖，陡门乡人民政府驻地不变。<ref>{{Cite web|url=http://www.xzqh.org/html/2009/0204/13315.html|title=原阳县历史沿革|date=2010-11-03|publisher=行政区划网|accessdate={{CURRENTYEAR}}-{{CURRENTMONTH}}-{{CURRENTDAY2}}查阅|deadurl=yes|archiveurl=https://web.archive.org/web/20110420140219/http://www.xzqh.org/html/2009/0204/13315.html|archivedate=2011-04-20}}</ref><br />
目前下辖：[[城关镇|城关镇]]、[[原武镇|原武镇]]和[[师寨镇|师寨镇]]，[[葛埠口乡|葛埠口乡]]、[[福宁集乡|福宁集乡]]、[[祝楼乡|祝楼乡]]、[[桥北乡|桥北乡]]、[[蒋庄乡|蒋庄乡]]、[[官厂乡|官厂乡]]、[[大宾乡|大宾乡]]、[[陡门乡|陡门乡]]、[[阳阿乡|阳阿乡]]、[[路寨乡|路寨乡]]、[[韩董庄乡|韩董庄乡]]、[[太平镇乡|太平镇乡]]、[[齐街乡|齐街乡]]和[[靳堂乡|靳堂乡]]。

==自然环境==
县境地势由西南向东北倾斜。[[黄河大堤|黄河大堤]]以南为高滩，多旱地。中部为低洼易涝地，文岩渠以北为岗地。主要河流有[[黄河|黄河]]、天然渠、文岩渠、幸福渠等。年平均气温为14.3℃，年平均降水量556毫米，全年无霜期227天。
==名胜古迹==
[[張良|張良]]行刺[[秦始皇|秦始皇]]的[[博浪沙|博浪沙]]位於此縣。名胜古迹主要有：明末清初民宅夏家院，以及[[毛遂|毛遂]]故里、[[陈平|陈平]]祠、[[张苍|张苍]]墓、玲珑塔等。
==经济==
2009年GDP 70.9亿元，同比增长12%；全社会固定资产投资105.8亿元，同比增长37.1%；社会消费品零售总额20.7亿元，同比增长19.1%;财政一般预算收入1.97亿元，同比增长15.4%;城镇居民人均可支配收入9516元，同比增长12.1%；农民人均纯收入4404元，同比增长7.8%<ref>{{Cite web|url=http://www.yuanyang.gov.cn/web/mlxxx/jjsl/gnsc/2010-04-26/910.html|title=国内生产总值及构成|publisher=原阳县人民政府|date=2010-11-03}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
==农业==
农业以[[小麦|小麦]]、[[水稻|水稻]]两大作物为主，[[玉米|玉米]]、[[花生|花生]]、[[苹果|苹果]]、[[杏|杏]]具有传统优势。<br />
原阳为[[优质强筋小麦生产基地县|优质强筋小麦生产基地县]]，全县优质小麦种植面积50万亩，2004年通过河南省农业标准示范县验收。<br />
优质水稻种植面积50万亩，[[原阳大米|原阳大米]]在市场上较有名气的品种有[[黄金晴|黄金晴]]等。<br />
==工业==
汽车零部件行业——河南万向系统制动器有限公司、原阳县万安制动系统有限公司和原阳县伟航汽车制动器有限公司等<br />
综合化工行业——河南银田精细化工有限公司等<br />
制药行业——新乡佐今明制药股份有限公司原阳分公司等<br />
板材行业——原阳县兴建精密复合板有限公司（河南金龙精密铜管股份有限公司投资），原阳县宏达木业有限公司等<br />
食品加工行业——新乡市冰馨食品有限公司（郑州市三全食品有限公司的加工基地）等<br />
轻纺行业——新乡市鑫源祥印织有限公司、原阳县纺织厂、丝达纺业公司等
