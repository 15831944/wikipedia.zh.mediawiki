'''原田彩楓'''（{{bd|1997年|12月23日|catIdx=Harata Sayaka}}<ref>{{Cite web|publisher=TBS電視台|work=烏菈菈迷路帖 Blu-ray&DVD 第1卷 映像特典（先行上映会〜共に過ごそう大吉の夜〜ダイジェスト映像）|title=Blu-ray＆DVD|url=http://www.tbs.co.jp/anime/urara/disc/bddvd.html|accessdate=2017-03-24}}</ref>）為[[日本|日本]]的[[配音員|女性配音員]]。[[CLARE_VOICE|CLARE VOICE]]所屬。出身於[[千葉縣|千葉縣]]。身高158公分。<ref name = "profile" />興趣為早起、散步、朗讀、鋼琴。<ref name = "profile" />

== 經歷 ==
高中決定進路時立志成為聲優，於2015年進入養成所。<ref name = "urara_interview">{{Cite web|url=https://www.animatetimes.com/news/details.php?id=1484194083|date=2017-01-14|accessdate=2019-01-23|title=TVアニメ『うらら迷路帖』主人公・千矢役・原田彩楓さんが語る作品の魅力|language=ja}}</ref>

進入養成所前不太擅長唱歌。<ref name = "urara_interview" />

2016年7月22日，原田彩楓以無名新人之姿，確認於遊戲「[[偶像大師_灰姑娘女孩|偶像大師 灰姑娘女孩]]」出演三船美優一角，因此引發了一波網路上的搜尋熱潮。<ref>{{Cite web |date=2016-10-12 |url=http://hameets.com/%E5%8E%9F%E7%94%B0%E5%BD%A9%E6%A5%93%E3%81%A8%E3%81%AF%E8%AA%B0%EF%BC%9F%E3%83%9D%E3%82%B1%E3%83%A2%E3%83%B3go%E3%83%88%E3%83%AC%E3%83%B3%E3%83%89%E3%81%AB%E9%A2%A8%E7%A9%B4/ |script-title=ja:原田彩楓とは誰？ポケモンGOで埋め尽くされたトレンドに風穴をあける。 |publisher=HAMEETS |accessdate=2017-1-13 |language=ja}}</ref>

2019年1月1日從準所屬升為正所屬<ref>{{Cite web|date=2019-01-01|url=http://clare-voice.co.jp/info#date190101 |title=所属のお知らせ|accessdate=2019-1-21 |language=ja}}</ref>，並於同日開設個人推特帳號。
