「'''原則'''」英文譯為：[http://en.wikipedia.org/wiki/Principle Principle]反譯回來也包括「[[定律|定律]]」、「[[原理|原理]]」一詞，是在「[[方法論|方法論]]」中我们所认知的集合范围内可以预测现象的'''[[自然规律|自然规律]]'''，常是客观存在的。使用原則概念的有科學、哲學、宗教、法律等。原則通常被定义为与[[事实|事实]]或[[实在|实在]]有潛在可推論性。然而，由於并没有任何一个'''[[真理|真理]]'''的[[定义|定义]]被学者普遍接受的情況下，[[邏輯|邏輯]]性地使人产生了对'''近似真理'''的'''「原則」'''權充自然绝对性规律的認知，常使'''「原則」'''成为指导行动的"最高准则"或"標準"。许多不同的原則定义一直被广泛争论。许多与原則定义相关的主题同样无法获得共识。[[普世價值|普世價值]]的'''「原則」'''更與絕對'''「真理」'''是兩個不完全等同的概念，儘管它們經常性地被人們所混淆，甚至有人造出「真理原則」的矛盾覆詞產生。

== 引言 ==
回顾关于「原則」这一课题的[[思想史|思想史]]，會發現少有人專門的注意這個詞而進行研究，因為這詞就像葉子般的前面被冠上各式各樣的名詞，幾乎使原則成為語尾助詞。

原則与[[意义|意义]]的[[主体|主体]]通常被认为密切相关，只有有意义的事物才能有慣常性。这一关联在古代就确立了，长期令人感兴趣的关联是類比於「真理」与[[逻辑|逻辑]][[有效性|有效性]]的关系。“因为逻辑最基础的概念是有效性，而它可通过真与假来定义。”（[[涅尔|涅尔]]夫妇，16）而讓各種現象類比套用於「原則」方法之下。

== 对于原則的哲学探讨 ==
原則是[[哲学家|哲学家]]面对人生的意義、存在問題的時候绕过問題核心的概念化處理方式之一。许多哲学家都亟欲以原則代表真理，然而彼此之间可能毫无共同之处，甚至针锋相对。日常生活中，不少人也自称自己的观点是個定律（原則）。探求原則可能是[[人性|人类的本性]]之一，然而有意识地歸納為近似真理与禁忌。

== 科學的原則 ==
''请参看：[[科學|科學]]''

科學的原則是利用[[邏輯|邏輯]]的[[實驗|實驗]]方法[[觀察|觀察]]方法獲得或經過驗證而來。近代的科學旨在理性、客觀的前提下，用[[知識|知識]]（[[理論|理論]]）與完整[[實驗|實驗]]證明出原則。在將知識的研究歸納和系統化後，便成為科學中的原則。
