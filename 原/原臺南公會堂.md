'''原臺南公會堂'''位於[[臺南市|臺南市]][[中西區_(臺灣)|中西區]]，民國八十七年（1998年）6月26日公告為臺南市市定古蹟{{r|文}}。市定古蹟。該建築所在地是之前清朝道光年間士紳[[吳尚新|吳尚新]]的[[吳園|吳園]]，現於其後仍留有吳園的一部分{{r|傅}}，而更早之前相傳這裡是荷治時期向[[鄭成功|鄭成功]]建議攻臺的荷蘭通事[[何斌|何斌]]的府邸，1661年5月3日鄭成功並在此接見經何斌引導的荷蘭使者湯馬斯‧范‧伊伯倫（Thomas van Iperen）與李奧納多‧范‧李奧納杜斯（Leonard van Leonardus）等人{{r|范}}。

== 沿革 ==
[[File:台南公會堂後棟.JPG|200px]]
[[File:台南公會堂後棟屋頂內部.JPG|200px]]
[[File:臺南公館.png|200px]]
該建築物於[[臺灣日治時期|臺灣日治時期]]的1910年9月{{r|石}}由矢田貝季陸所設計，以[[財團法人|財團法人]]型態官民合資4萬多圓興建{{r|傅}}，於1911年2月完工{{r|石}}。當時是叫做臺南公館，1923年改稱臺南公會堂{{r|石}}，是當時市民重要的集會場所，也兼具教化百姓的功能{{r|傅}}。當時臺南市甚至[[臺南州|臺南州]]、[[高雄州|高雄州]]重要公共政策之制定和演講、集會等均在此舉行，而[[臺灣文化協會|臺灣文化協會]]成立後，[[蔡培火|蔡培火]]、[[連橫|連橫]]等人亦在此處演講{{r|石}}。

二次大戰後，改稱為中山堂，為空軍新生社和軍友社使用{{r|遠}}，後來又於1955年6月改為省立臺南社教館{{r|范}}並予以整修{{r|傅}}。而在社教館於1994年{{r|石}}移到臺南市五期重劃區後，該建築一度因位於市中心而有拆除作為商業區興建大樓之議{{r|石}}，最後此議作廢，該建築改為吳園藝文館{{r|傅}}。近年有再整修過，而呈今貌。

== 建築特色 ==
臺南公會堂的空間是由前棟與後方的大集會堂所組成，其中大集會堂是木構造，位於一基座上，講求機能性而未使用西方歷史建築語彙裝飾{{r|傅}}。前棟則有較多的西方歷史建築語彙裝飾，在中央入口部分的二樓設有陽臺，更上面的三角型山牆內則有繁複的裝飾浮雕，而山牆之上則是該建築最高的屋頂，上頭突出部分為老虎窗（Dormer Window），其內則開有牛眼窗{{r|傅}}。而主體屋頂則使用了法國式屋頂（French Roof），並鋪有魚鱗狀板瓦和加上冠飾{{r|傅}}。

前棟兩翼部分在一樓正面牆上開有三個稍微內凹的圓拱窗，而窗子的拱心石與壁體水平線連成一體；二樓正面牆上則使用了四根臺灣少見的[[愛奧尼柱式|愛奧尼克柱式]]方壁柱來承接楣樑與挑簷而成三個開間，而柱子之下則有曲線優美的托架，此一處理方法也是少見的{{r|傅}}。而在兩翼屋頂部分前的矮牆上裝有臺灣傳統建築使用的綠釉花磚，使得該建築增添了一些本土味{{r|傅}}。

而在屋側有圓井，大集會堂下方的基座中有半圓井。

== 周圍 ==
臺南公會堂右側有日治時期的柳屋料理，後方為昔日吳園僅剩的部分飛來峰與水榭，再後面便是[[遠東百貨|遠東百貨公園店]]（原[[臺南市立圖書館|臺南市立圖書館]]，1974年遷館售地）。
