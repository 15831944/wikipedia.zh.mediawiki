'''原画'''是動漫、電子遊戲製作過程中由手繪的描述角色關鍵造型、動作的畫，用於加工後期作品。

== 动画 ==

'''原画'''在动画制作的分工制度中指，由手繪構成[[動畫|動畫]]的製作過程中，描繪動畫關鍵處或動作起始、結束的畫（又称[[關鍵格|關鍵格]]）。一般是以線條稿形式繪製，並加入陰影與分色的層次線。另外，對描繪原畫的人也稱作為原畫者、原畫家、原畫擔當者、或是只稱作原畫。填入原畫作業所描繪的動作關鍵處的是，稱為插入（{{lang|ja|中割り}}）的動畫。原畫與動畫總括起來稱之作畫。在紙上用[[鉛筆|鉛筆]]所描繪的原畫，於愛好家之間或専門店買賣的情況，一般視為比動畫等級要高。

[[作畫監督|作畫監督]]層級的[[動畫師|動畫師]]所繪製的原畫，常常不用再做修正直接使用。

整體或原畫製作時程吃緊的情況，有時會將原畫師畫好的線稿，交由另外的原畫師清理線條完成原畫，此時前者在工作人員名單上標記為第一原畫（一原），後者稱作第二原畫（二原）。第二原畫也可能配置來培育新人原畫師或實踐練習。當作畫牽涉到3D動畫，可能描繪線稿以供位置或動作的引導。或者相反地，先輸出3D畫面再配合作畫。

第二原畫的制度是21世紀以後的事情。最早的雛形出在1979年《[[機動戰士GUNDAM|機動戰士GUNDAM]]》（無印），其人設[[安彥良和|安彥良和]]的發明，不過並沒有發揮太大效果，及後安彥離開動畫界專攻漫畫，故並沒有發展下來。2002年時[[BONES|BONES]]的《[[翼神傳說|翼神傳說]]》中負責動畫畫面管理的佐野浩敏再次引進第二原畫的制度，日後逐漸成為日本動畫業界的常規作業制度。<ref>{{cite web|language=zh-cn|publisher=[[動畫魂|動畫魂]]|author=III|accessdate=2016年9月1日|date=2016年8月30日|url=http://m.anitama.cn/article/c1216047385c5296 |title=第二原畫的前世今生：北久保弘之談第二原畫的的今昔|work=}}</ref>

== 电子游戏 ==

另外，在[[電子遊戲|電子遊戲]]的製作過程裡，關於加工成[[電腦圖形學|電腦圖畫]]前的畫，有時也稱作為原畫（主要在常利用稱作一枚絵的全畫面[[電腦圖形學|CG]]的[[冒險遊戲|冒險遊戲]]或[[模擬遊戲|模擬遊戲]]被使用此稱呼）。

== 漫画 ==

[[漫畫家|漫畫家]]的親筆原稿。通常用「原稿」一詞來稱呼，不過在展示親筆原稿之類的時候，以「原畫展」的形式來使用本詞語（不稱作「原稿展」，但理由不詳）。在漫畫的領域中，「原畫」與「原稿」是同義語。
