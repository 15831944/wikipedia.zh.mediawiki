'''阿秒'''<ref>{{cite web |url=http://www.nsc.gov.tw/sd/unitran1_show.asp?letter=A |title=統一字詞用法表 |accessdate=2012-11-17 |publisher=[[行政院國家科學委員會|行政院國家科學委員會]] |deadurl=yes |archiveurl=https://web.archive.org/web/20050527063714/http://www.nsc.gov.tw/sd/unitran1_show.asp?letter=A |archivedate=2005-05-27 }}</ref>（{{lang-en|attosecond}}），符號 '''as'''，是一種時間的[[國際單位制|國際單位]]，為 10<sup>-18</sup> 秒，或 1/1000 [[飛秒|飛秒]]<ref>{{cite web |url=http://www.memidex.com/attosecond |title=attosecond |work=Memidex/WordNet Dictionary/Thesaurus |accessdate=2011-04-14 |language=en}}</ref>。比例上，一阿秒之於一[[秒|秒]]，如同一秒之於 317.1 億年，約為[[宇宙年齡|宇宙年齡]]的兩倍<ref>[https://web.archive.org/web/20080304183157/http://www.photonics.com/content/news/2008/February/28/90764.aspx Electron Motion Filmed], 28 Feb. 2008{{En}}</ref><ref>[https://web.archive.org/web/20071111131259/http://steacie.nrc-cnrc.gc.ca/personal/corkum/atto5_e.html Exploring "Attosecond" Time. Visualising an Attosecond... How short is an attosecond?]{{En}}</ref>。

==詞源==
阿秒的英語為 ''attosecond''，由表示「10<sup>-18</sup>」的[[國際單位制詞頭|前綴]] ''atto''〈[[阿托|阿托]]〉和表示「秒」的 ''second'' 組合而成。''atto'' 來源自[[丹麥語|丹麥語]]的 ''atten''，意思是「十八」<ref>[http://www.wordinfo.info/words/index/info/view_unit/246 atto- [A toh] (Danish or Norwegian: eighteen; a decimal prefix used in the international metric system for measurements)]. Wordinfo.info (2007-04-05). Retrieved on 2011-01-23.{{En}}</ref>。

==時間間距==
*1阿秒：[[光|光]]飛越3粒[[氫原子|氫阿子]]的時間。
*24阿秒：時間的[[原子单位制|原子單位]]。
*200阿秒：[[鈹的同位素|鈹-8]]的[[半衰期|半衰期]]。
