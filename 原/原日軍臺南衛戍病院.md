'''原日軍臺南衛戍病院'''位於[[臺南市|臺南市]]東區，於民國九十二年（2003年）6月18日公告為臺南市市定古蹟，為全臺灣僅存的完整[[軍醫院|軍醫院]]建築群{{r|文}}{{r|管}}。該建築目前為[[國立成功大學|國立成功大學]]臺灣文學系系館{{r|管}}。

== 沿革 ==
[[File:原日軍衛戍病院(二).JPG|250px]]
臺南衛戍病院是[[臺灣日治時期|日治時期]][[日軍|日軍]]在臺所設的軍醫院之一，初期原本將醫院設在[[赤崁樓|赤崁樓]]西側[[蓬壺書院|蓬壺書院]]，後來於[[大正|大正]]六年（1917年）才在[[原日軍步兵第二聯隊營舍|步兵第二聯隊營舍]]北側建成規模完整的正式院舍{{r|傅}}，為當時臺灣規模最大且完善者{{r|王}}。

[[二次大戰|二次大戰]]後，該建築被[[中華民國國軍|中華民國國軍]]接收，成立[[陸軍訓練司令部|陸軍訓練司令部]]軍醫院。隨後醫院曾數次更名，1960年改稱陸軍第804總醫院（今[[國軍桃園總醫院|國軍桃園總醫院]]），並於1987年遷移至[[桃園縣|桃園縣]]（今[[桃園市|桃園市]]）。醫院遷離後，原址在2000年撥交給[[國立成功大學|國立成功大學]]作為力行校區校舍，歷經修復之後，於2013年成為該校臺灣文學系系館。

== 建築特色 ==
此醫院為[[大日本帝國陸軍|日本陸軍]]經理部營繕組設計{{r|王}}，由數排彼此有廊道相連的南北向建築所組成，主要建材是磚頭，有不同形式的拱廊，其地板有抬高，有的屋頂上面有通氣塔{{r|傅}}，內部有大廳、掛號領藥廳、候診室、門診室、病房與醫官辦公室等房間{{r|王}}。除了建築之外，附近也有不少老樹{{r|傅}}。

另外在「家寶醫學研究室」的隔壁棟紅磚建築北側牆上，上半部留有二次大戰美軍留下的彈孔，下半部的平行彈孔，據說是二二八事件時軍隊進行掃射所留下{{r|管}}。

== 其他 ==
[[管碧玲|管碧玲]]《爸爸的吉他：管碧玲的初衷》一書中寫說當年成功大學有意將醫院建築拆毀改建為新的醫學大樓，但時任臺南市副市長[[許陽明|許陽明]]對臺南衛戍病院提出古蹟審議{{r|管}}。反對此事的成大長官曾將多位擔任審查委員的成大教授找去「訓話」（管碧玲與許陽明當晚請這些委員在臺南新天地百貨的餐廳請這些委員吃飯慰勞），另外審議通過後成大曾提出再議，但最後還是指定成古蹟{{r|管}}。管碧玲在該書中寫說此一古蹟案是她唯一一次為古蹟「關說」的案例{{r|管}}。
