'''原日本陸軍臺南偕行社'''，為[[臺南市文化資產|台南市歷史建築]]，位於[[台南公園|台南公園]]對面，最早見於[[大正|大正]]13年（1924年）的臺南市街圖。台南偕行社共有三棟主建物，但其正面皆被其他建物所遮蔽，難易看到其全貌。此歷史建築屬國防部資產，在2007年登錄為文化資產前，仍有軍眷居住在此，並一度面臨拆除的危機<ref>{{Cite web|url=http://news.ltn.com.tw/news/supplement/paper/149511|title=冒雨搶救古屋 化解拆除危機─台南陸軍偕行社登錄歷史建築|accessdate=|author=|date=2007-08-22|publisher=}}</ref>。2015年曾傳出將由國防部撥交給台南市政府代管使用<ref>{{Cite web|url=http://news.ltn.com.tw/news/local/paper/927883|title=歷史建築偕行社 「養蚊」惹怨|accessdate=|author=|date=2015-10-29|publisher=}}</ref>，而至今仍處於閒置中。

== 建築特色 ==
原日本陸軍臺南偕行社有二層木石造及一層木造建築。二層木石造建築，立面採三段式處理，開設有通風口之大型石塊基座，一樓外牆採拋泥式設計，二樓木構外露灰作壁面，上承斜屋頂石片瓦作屋面。主要入口內凹，挑高之室內大廳設置有L型木造樓梯，獨具特色。一層木造建築呈L型，包圍內部庭園之窗戶採落地設計，為用餐聯誼觀賞之用，上承人字型屋架，屋面覆以石版瓦片，用料講究。<ref>{{Cite web|url=https://nchdb.boch.gov.tw/assets/overview/historicalBuilding/20070911000001|title=文化部文化資產局國家資產網-原日本陸軍臺南偕行社|accessdate=|author=|date=|publisher=}}</ref>

== 相關 ==
'''{{link-jp|偕行社|偕行社}}'''在明治10年（西元1877年）創立，為類似[[國軍英雄館|國軍英雄館]]的集會、社交場所。台灣在日治時期建有台北、北投、台南三處偕行社。

* 台北偕行社（台北市市定古蹟[[婦聯總會|婦聯總會]]）
* 北投偕行社（現為私人住宅）

== 其他 ==
原文化資產登錄名稱為「原臺南陸軍偕行社」，種類為「公共建築」，於2018年12月28日更名為「原日本陸軍臺南偕行社」，種類改為「集會堂」。
