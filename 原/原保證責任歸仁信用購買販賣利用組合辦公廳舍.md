== 沿革 ==

* 1925年，成立「保證責任歸仁信用購買販賣利用組合」
* 1944年，改稱「歸仁庄農業會」
* 1946年，改組為「歸仁鄉合作社」和「歸仁鄉農會」
* 1949年，再合併為「臺南縣歸仁鄉農會」<ref>{{Cite web|url=http://www.naffic.org.tw/webpad/webpad.aspx?EpfJdId9UuCZSUj%2FFtLl5bRGbCeIio5N|title=歸仁區農會 關於本會|accessdate=2019-04-13|author=|date=|publisher=歸仁區農會}}</ref>
* 1967年，歸仁鄉農會遷移至新廳舍<ref>{{Cite web|url=https://news.ltn.com.tw/news/local/paper/1042394|title=歸仁農會老金庫 變身商品補給站|accessdate=2019-04-13|author=|date=2016-10-16|publisher=自由時報}}</ref>
* 2016年，於地震中損毀，之後登錄為歷史建築

== 建築特色 ==
建築物外部立面形式、天花板、門窗以及外牆飾材作法，為[[装饰风艺术|藝術裝飾風格]]，呈現日治中晚期之工藝風格，具時代特色。<ref>{{Cite web|url=https://nchdb.boch.gov.tw/assets/overview/historicalBuilding/20161005000001|title=原保證責任歸仁信用購買販賣利用組合|accessdate=|author=|date=|publisher=文化部文化資產局-國家文化資產網}}</ref>
