'''原麗淇'''（{{lang-en|Eugenia Yuan}}，{{bd|1976年|1月22日}}），[[香港|香港]][[女演員|女演員]]，香港著名藝人[[鄭佩佩|鄭佩佩]]的大女兒，曾和母親合演過武俠片《[[龍騰虎躍|龍騰虎躍]]》。有兩位妹妹，三妹[[原子鏸|原子鏸]]亦為演員。

== 早年經歷 ==
原麗淇於美國[[洛杉磯|洛杉磯]]出生，4歲開始習舞，對[[芭蕾舞|芭蕾舞]]、[[中國舞|中國舞]]、[[爵士舞|爵士舞]]無一不曉，大學畢業於[[加州大学洛杉矶分校|加州大学洛杉矶分校]](俗稱UCLA)語言人類學系<ref name="早年經歷">{{cite web |url=http://news.xinhuanet.com/newscenter/2002-08/21/content_533343.htm| title=原麗淇不願步媽媽鄭佩佩後塵當"俠女"[圖文] |publisher=東方網|date=2002-08-22}}</ref>；原麗淇曾是[[奧林匹克運動會美國代表團|美國國家隊]]的[[藝術體操|藝術體操]]隊成員，並曾在8年間代表過美國到澳洲、俄羅斯等地演出<ref name="早年經歷"/>。

== 演藝經歷 ==
原麗淇在1999年脫離國家隊後簽約了經理人公司，在美國參與多部獨立制作電影和電視劇<ref name="早年經歷"/>。並曾憑《[[三更_(電影)|三更之回家]]》獲得第22屆[[香港電影金像獎|香港電影金像獎]][[香港電影金像獎最佳新演員|最佳新演員]]。
