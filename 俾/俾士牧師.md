'''俾士牧師'''（{{lang-en|The Rev. George Piercy}}，{{bd|1829年|||1913年}}）為英國牧師，被譽為[[香港基督教循道衞理聯合教會|中華循道公會]]之父，建立「中華循道公會華南教區」，是循道公會在華發展的重要奠基者。他21歲隻身來華，在中國傳道31年，開辦學校、醫療工作，並翻譯西方屬靈書籍，例如《天路歷程》。1882年回國後，繼續向[[倫敦|倫敦]]碼頭地區的中國水手傳教至離世為止。

== 簡介 == 
1851年1月，俾士牧師以宣教士身份自費抵港，同年12月北上[[廣州|廣州]]傳道。1853年，俾士在增沙設一福音堂，獲准組成廣州教區。

《天津條約》簽訂後，避地澳門的俾士等傳教士再度入粵，設堂傳教，並在廣州增沙創設聖道書院，培養華籍牧師。<ref>{{cite web|url=http://www.methodist.org.hk/media/monthlynews/pdf/2011/11/03/2011_iss317-eVersion.pdf|title=《香港基督教循道衛理聯合教會會訊》 2011年9-10月，第317期，俾士牧師抵港160週年紀念，第6頁}}</ref><ref>{{cite web|url=http://www.methodist.org.hk/media/monthlynews/pdf/2009/02/03/2001-08.pdf|title=《香港基督教循道衛理聯合教會會訊》 2001年8月，第217期，第5頁}}</ref><ref>{{cite web|url=http://www.methodist.org.hk/media/monthlynews/pdf/2009/02/02/2004-12.PDF|title=《香港基督教循道衛理聯合教會會訊》 2004年12月，第257期，第14頁}}</ref><ref>盧龍光、湯開建、張照、顏小華：《苦難中成長的教會－英國循道公會佛山傳教發展史：1851-1949》香港：基督教中國宗教文化研究社，2011。ISBN 9789627706250</ref>

== 家庭 ==
元配溫樂鍾女士（{{lang|en|Joan Wannop}}）於1858年在廣州南門外增沙創辦[[私塾|寄宿女學]]，為[[華英中學|華英女子中學]]前身之一。
兒女包括約翰（John）、俾士牧師次子同名[[俾士|佐治·俾士]]（George Piercy, Jr.，後稱小俾士）、愛德華（James Edward）、珍妮（Jane）和安妮（Anne Frances），五個孩子都生在中國。除珍妮夭折於繈褓外，其餘四個孩子都長大成人。約翰在英國任職律師，佐治·俾士擔任[[拔萃書室|拔萃書室]]校長，安妮為醫生，愛德華從事商業。<ref>{{cite web|url=http://news.takungpao.com.hk/paper/q/2014/0317/2356372.html|title=百年滄海念心香—英國俾士家族的粵港相冊}}</ref>
