'''俾斯麦兵营'''（{{lang-de|Bismarck-Kaserne}}），旧址位于[[中国|中国]][[山东省|山东省]][[青岛市|青岛市]][[市南区|市南区]][[鱼山路|鱼山路]]5号[[中国海洋大学|中国海洋大学]]鱼山路校区院内，靠近校园[[红岛路|红岛路]]5号后门，是驻青[[德意志帝国海军|德军]]于1903-1909年间建设的一座大型[[兵营|兵营]]建筑群。[[国立山东大学|国立山东大学]]使用该兵营旧址时曾在此基础上扩建校舍。整个建筑群现为[[全国重点文物保护单位|全国文物保护单位]][[青岛德国建筑群|青岛德国建筑群]]的一部分<ref>{{Cite news|url=http://news.sina.com.cn/o/2003-12-21/16301395504s.shtml|title=德军在青岛的俾斯麦兵营|author=|date=2003-12-21|work=青岛晚报|accessdate=2017-05-08}}</ref><ref Name=he>{{Cite news|url=http://qd.sohu.com/20130903/n385730667.shtml|title=俾斯麦兵营和毛奇兵营一豪华一简陋|author=|date=2013-09-03|work=城市信报|accessdate=2017-05-08}}</ref><ref Name=xiao>{{Cite news|url=http://qd.ifeng.com/a/20160503/4514264_0.shtml|title=俾斯麦兵营：德国人百年前修建 校园与兵营的更迭|author=|date=2016-05-03|work=青岛日报|accessdate=2017-05-08}}</ref>。

==建筑历史==
===清军嵩武中营===
俾斯麦兵营旧址的原址是建于章高元驻防时期（1892-1897）的[[清军|清军]]'''嵩武中营'''<ref Name=xun>{{cite web |url=http://club.qingdaonews.com/touch/show_57_3824049_1_0.htm|title=寻访德国三大兵营的前世今生|accessdate=2010-11-12|work=|publisher=}}</ref>。该营房位于群山环绕的山谷中，东北侧与东南侧紧邻今天的[[青岛山|青岛山]]和[[八关山|八关山]]，西侧和西北侧不远处是今天的[[信号山|信号山]]和[[伏龙山|伏龙山]]，后来被部分学者称为“青岛河”的一条地表径流流经兵营西侧。营房平面呈矩形，营房和营门均朝向西南{{注|后来的俾斯麦兵营基本上沿用了原有建筑的房屋朝向}}。营房环绕以夯土围墙，营门以石料砌成，门洞上方有一石匾，刻有[[阴文|阴文]]“嵩武中营”四字。营房全部为中式平房。

1897年11月14日，德军占领青岛，原嵩武中营继续作为兵营使用，并改名为'''“东营”（Ostlager）'''，营门石匾也以德文牌匾覆盖。部分德国当局军政官员（如华人事务专员[[单威廉|单威廉]]）也在此处暂居。为改善居住条件，德国当局营房进行了临时性的修缮，并设置排水系统、安装路灯<ref Name=chen>{{Cite news|url=http://wanbao.qingdaonews.com/html/2008-09/21/content_1028490.htm|title=伊尔蒂斯兵营：沉默的见证者|author=|date=2008-09-21|work=青岛晚报|accessdate=2017-05-08}}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。原有的夯土围墙也被拆除。该营房后来为建设俾斯麦兵营而被拆除。

===德军兵营===
[[File:青岛俾斯麦兵营I、II号营房.jpg|thumb]]
1903年，德国当局开始在嵩武中营的原址建设继[[伊尔蒂斯兵营|伊尔蒂斯兵营]]之后的第二座永久性兵营，由总督府建设局（Bauverwaltung）设计，广包公司（Baufirma F. H. Schmidt）承建{{注|广包公司只参与了第一批工程，III号及IV号营房及相关工程并未参与}}，洛塔尔·马尔克斯（Lothar Marcks）负责工程监督，至1905年完工。首批完成的建筑为设计相同的I号与II号营房以及兵营礼堂<ref Name=xiao/><ref Name=xun/><ref Name=qing>{{cite book |author=袁宾久 |title=《青岛德式建筑》|year=2009 |publisher=中国建筑工业出版社 |location=青岛 |isbn=9787112109050}}</ref>。兵营邻近俾斯麦山（Bismarck-Berg，今[[青岛山|青岛山]]），并因此与该山峰一同使用了“铁血宰相”[[奥托·冯·俾斯麦|奥托·冯·俾斯麦]]的名字。两座营房北侧的马厩在营房与礼堂建成后不久也开始建造并建成投入使用，营房西南侧的空地则用作操场。建筑群初步建成后，海军第三营（III. Seebataillon）指挥部及部分连队自[[伊尔蒂斯兵营|伊尔蒂斯兵营]]迁至此处驻扎。

1905年，德国与[[清政府|清政府]]签订《[[胶高撤兵善后条款|胶高撤兵善后条款]]》。海军第三营驻扎在[[胶州|胶州]]、[[高密|高密]]的两个[[连|连]]于此后撤回青岛。为安置自胶州撤回的第三营一连（1. Kompagnie），总督府于1906年开始建设俾斯麦兵营III号营房，与之同时建设的是兵营礼堂西南侧的车库及I号营房西北侧的士官公寓<ref Name=xun/>。III号营房东南侧的IV号营房在前者建成一段时间后开始建设，同时建设的是礼堂与车库之间的军官公寓。所有工程最终在1909年结束，耗资75万[[德国金马克|金马克]]<ref Name=xun/><ref Name=xiao/>。

[[File:青岛俾斯麦兵营旧址_01.jpg|thumb]]
===日治时期及1920年代初===
1914年11月，日军[[青岛战役|占领]]青岛，该兵营改名为万年兵营{{注|青岛山亦改名为万年山}}，为日军所使用，在此驻扎的是[[青岛守备军_(日本陆军)|青岛守备军]]步兵第二大队、第三大队、独立步兵第七大队<ref Name=xiao/><ref>{{Cite|author=三船写真馆|title=日独战役青岛名所写真帖|date=1914}}</ref>。1922年12月，[[北洋政府|北洋政府]]收回青岛，[[王翰章|王翰章]]所部[[北洋陆军|北洋陆军]]第五师第十旅驻扎于此。

===作为校舍使用===
[[File:青岛俾斯麦兵营旧址_国立山大校园.jpg|thumb]]，近处为化学馆和科学馆]]
{{see also|私立青岛大学|国立青岛大学|山东大学|中国海洋大学}}
1924年5月29日，[[胶澳商埠|胶澳商埠督办]][[高恩洪|高恩洪]]成立[[私立青岛大学|私立青岛大学]]筹备处，筹备建校事宜，并选定原俾斯麦兵营址作为大学校址<ref Name=xiao/>。该决定遭到兵营驻军方面反对，双方争执不下，并请时任直鲁豫三省巡抚使的[[吴佩孚|吴佩孚]]裁定。吴佩孚下令，“令开：奉督办公署第3239号准予备案。9月15日，奉870号公函，准将青岛兵营拨给本校永做校址。”裁定兵营作为校址<ref>{{Cite news|url=http://news.sina.com.cn/o/2012-07-03/114924704432.shtml|title=老山大选址俾斯麦兵营 军阀赖着不走惹恼吴佩孚|author=|date=2012-07-03|work=半岛都市报|accessdate=2017-05-08}}</ref>。9月20日，私立青岛大学正式开学。1925年11月，吴佩孚等[[直系军阀|直系军阀]]在[[第二次直奉战争|第二次直奉战争]]中失败，高恩洪受牵连去职，学校运作出现困难。校舍部分空闲房屋借给军方使用。

1928年4月，[[北伐军|北伐军]]进军[[济南|济南]]，日军趁势自青岛登陆[[五三惨案|出兵山东]]，学校在此局势下停办。1929年4月，[[南京国民政府|南京国民政府]]接收青岛，[[蔡元培|蔡元培]]、[[何思源|何思源]]、[[赵太侔|赵太侔]]等人筹建[[国立青岛大学|国立青岛大学]]。7月2日，私立青岛大学一切校产交予国立青岛大学筹办委员会<ref Name=si>{{cite web|url=http://www.liveqingdao.cn/mp/yc/mingxiao/201108/03367.html|title=私立青岛大学|date=2011-08-03|language =zh-hans|work=胡耀杰}}</ref>。9月21日，[[国立青岛大学|国立青岛大学]]开学，沿用俾斯麦兵营作为校舍。1932年9月，国立青岛大学因抗日学潮改组为[[国立山东大学|国立山东大学]]。此后，国立山东大学在兵营原址的基础上建设了科学馆、化学馆、工学馆等新校舍。

===1938年后===
[[File:国立山东大学1951年开学典礼.jpg|thumb]]
1937年7月7日全面[[抗战|抗战]]爆发后，国立山大于11月迁往内陆地区。1938年1月，日军再次占领青岛，再次将校址占用为兵营。1945年8月，[[日本投降|日本投降]]。{{link-en|美国海军陆战队第六师|6th Marine Division (United States)}}进驻青岛后将原俾斯麦兵营连同其南侧的[[青岛日本中学校|原日本中学校舍]]一并作为兵营使用。经[[司徒雷登|司徒雷登]]从中斡旋，美军撤出两处建筑群，国立山大迁回原俾斯麦兵营校址，同时将原日本中学校舍一并接收，并于该年10月25日复校<ref Name=xiao/>。

1949年6月，[[解放军|解放军]]进驻青岛。中共设立的华东大学于1951年在青岛与山东大学合校。1958年10月，山东大学大部分师资自青岛迁往[[济南|济南]]。1959-1960年，留在青岛的部分专业经调整后成立山东海洋学院，1988年更名为青岛海洋大学，2002年更名为[[中国海洋大学|中国海洋大学]]。俾斯麦兵营旧址现为海大鱼山路校区的一部分，2006年列入[[全国重点文物保护单位|全国文物保护单位]][[青岛德国建筑群|青岛德国建筑群]]之内。

==建筑特色==
===俾斯麦兵营营房===
[[File:青岛俾斯麦兵营_17.jpg|thumb]]
俾斯麦兵营的整个营房建筑群总共有大小建筑十余栋，各建筑的外立面风格或极为相似，或互相保持一致。四座先后建成的大营房在外立面与结构上极为相似，均为二层建筑，矩形红瓦坡屋顶[[阁楼|阁楼]]，平面呈“H”字形，正立面呈三段式对称划分，外立面装饰大量使用[[花岗岩|花岗岩]]。而四栋建筑的外立面风格则体现了由注重装饰到装饰趋简而注重实用的变化，这也刚好迎合了当时世界建筑艺术潮流的趋势。另外值得注意的是，为了防止疾病的发生，营房的设计采用了新的卫生标准，除了宿舍配有与厕所分开的盥洗间以外，甚至还引入了更为先进的抽水马桶，这在当时是一项先驱性的举措<ref Name=xun/>。

====I号、II号营房====
I号、II号营房建于1903-1905年，是两座出自同一张图纸的营房。建筑两层加红瓦坡屋顶阁楼。正立面中央与两端的檐口处起山花，中央处的山花顶部刻有[[德国国徽#德意志帝国|德国国徽]]。两个楼层临西南面（即正立面）为[[敞廊|敞廊]]，其中二楼有两端敞廊是用花岗岩砌成的立柱支撑的，柱头为[[混合柱式|混合柱式]]的变体{{注|值得注意的是，希腊[[柱式|柱式]]的涡卷纹均为向外卷起，俾斯麦兵营的柱头则是向内卷起的}}。外立面部分装饰细节受[[哥德复兴式建筑|哥特式复兴建筑]]的影响<ref Name=qing/>。建筑内部除门廊使用了[[罗曼式建筑#肋架拱顶|肋架拱顶]]外，装饰较少。台阶以花岗岩石块砌成，地面铺瓷砖。

两座营房的敞廊于1950年代后封闭为拱窗，山花上的鹰徽被水泥抹平。两座营房现均为海洋大学水产学院所使用，门楣上均有“水产馆”牌匾。2013年8月-9月，[[中国海洋大学|中国海洋大学]]校方在未申报文物修缮、无合法施工手续的情况下，对II号营房内部进行野蛮施工，媒体曝出后，文物部门有所回应，但未见任何实质性措施，施工方也未整改，此事之后不了了之<ref Name=wei>{{cite web |url=http://qd.sohu.com/20130910/n386261530.shtml|title=俾斯麦兵营旧址遭违法施工 文物局称改造未申报|accessdate=2013-09-10|work=王法艳|publisher=半岛都市报}}</ref>。目前，两座营房正门设有打卡玻璃门，实质上已不对外开放。

====III号、IV号营房====
III号营房建于1906年后，外立面布局与先前建成的两座营房相似，但外立面的花岗岩装饰大大减少，同时值得注意的是，该营房的正立面完全放弃了前两座营房采用的敞廊，而以联排拱窗代替。该建筑后来曾为国立山东大学文学院所在地，现为“海洋馆”。目前，营房正门设有打卡玻璃门，实质上已不对外开放。

IV号营房建于III号营房建成后，1909年完工。外立面布局与其他营房相似，部分设计与III号营房相似（如正立面两端），而包括门窗、山花在内外立面装饰大大简化，正立面单个窗户的面积与III号营房相比较大大减小。该建筑现为海洋大学海洋发展研究院，门楣上有“地质馆”牌匾。该营房目前是四座营房中唯一一座不设打卡玻璃门的营房。

====礼堂、军官公寓与车库====
营房建成后，夹在I号营房与III号营房之间有三栋附属建筑，由东北向西南分别为礼堂、军官公寓与车库。

礼堂（即士兵活动中心）建于1903-1905年间，与I号、II号营房同期建造，为一单层建筑，红瓦坡屋顶，外立面装饰使用了与两座大营房相似的风格，但山花与窗户的形状十分特别。该建筑后来沿用为礼堂。原建筑于1983年拆除，原址现为海大鱼山校区图书馆与海洋法学、海事、海商研究所<ref Name=xun/>。

军官公寓的建造时代与IV号营房相同，是一栋两层坡屋顶阁楼的小型楼房，建筑规模与布局与先前建成的士官公寓（闻一多故居）相似，但外立面装饰极为简洁。该建筑于1924年私立青岛大学筹办后作为女生宿舍使用。1929年6月，[[蔡元培|蔡元培]]来青岛筹办国立青岛大学时，曾与家眷在此暂居。原建筑于1983年拆除，原址现为海大鱼山校区图书馆<ref Name=xun/><ref Name=si/>。

车库建于1906年后，与III号营房时代相同。建筑共三层加坡屋顶阁楼，一楼外墙全部以花岗岩砌成，设有车库。正立面（朝向东南）中央偏北的檐口处起山花，其外立面的装饰特点被看做是向简洁风格的过渡。该建筑后来被用作大学图书馆{{注|作家[[梁实秋|梁实秋]]曾任图书馆馆长兼外国文学系主任}}<ref Name=xun/>。现为海洋大学档案馆和水产学院所使用。

====闻一多故居====
闻一多故居，又称“一多楼”，位于I号营房北侧，邻近今海大学术交流中心及[[大学路_(青岛)|大学路]]、[[红岛路|红岛路]]路角，约建于1906年，与III号营房属于同一批建造的建筑物，最初为士官公寓，是一座两层坡屋顶的住宅类建筑。后作为教职员宿舍使用。1930年夏，近代著名诗人[[闻一多|闻一多]]来到青岛任[[国立青岛大学|国立青岛大学]]文学院院长兼中文系主任，1931年迁居至此，次年离开青岛。与此同时，数学家[[黄际遇|黄际遇]]自1930年9月至1936年2月任理学院院长兼数学系主任{{注|1935年始不再兼任数学系主任}}时居住在该楼一楼<ref>{{cite|author=黄小安|title=爷爷居住了六年的小楼|journal=市南人文历史研究（第20期）|date=2016-06-15}}</ref>。1978年，该楼辟为闻一多故居展览室，并在楼前设立闻一多雕像，刻有其学生[[臧克家|臧克家]]所撰碑文。1984年12月14日，该建筑列入[[青岛市文物保护单位|青岛市文物保护单位]]<ref>{{cite web|url=http://lib.sdsqw.cn/bin/mse.exe?seachword=&K=b2&A=53&rec=7&run=13|title=青岛市志·旅游志·旅游资源·市区资源·名人故居|accessdate=|work=青岛市史志办|publisher=山东省情网}}</ref><ref>{{cite web|url=http://lib.sdsqw.cn/bin/mse.exe?seachword=&K=b2&A=66&rec=55&run=13|title=青岛市志·人物志·传记·闻一多|accessdate=|work=青岛市史志办|publisher=山东省情网}}</ref>。

====马厩及其他附属建筑====
除上述建筑外，兵营还建有若干附属建筑、临时建筑，均为平房。其中规模最大的为兵营马厩，位于I号营房北侧，其原址与今闻一多故居隔[[红岛路|红岛路]]相望。马厩约建于1905年，为一栋平房，规模虽小，却有众多红砖、山花等装饰，也是兵营建筑群内唯一一栋使用红砖作为建筑材料的建筑。该建筑后曾为海洋大学印刷厂，1999年拆除<ref Name=xun/>。

====海军第三营阵亡官兵纪念碑====
[[File:青岛海军第三营纪念碑.jpg|thumb]]
[[File:青岛海大鱼山校区广场.jpg|thumb]]
{{see also|义和团运动|八国联军侵华战争}}
1900年5月28日，[[德国驻华公使|德国驻华公使]][[克林德|克林德]]（Clemens von Ketteler）向胶澳总督府发电报称[[北京|北京]]周边发生[[义和团运动#保定|骚乱]]，要求派出特遣队保护使馆。总督府于第二天派出由1名军官、5名军士和40名海军士兵组成的特遣队乘军舰前往大沽。6月8日，应[[东亚分舰队_(德国)|德国海军东亚分舰队]]的要求，海军第三营又派出25人前往天津。6月19日，海军第三营营长约翰内斯·克里斯特（Johannes Christ）率领两个[[连|连]]共256名官兵乘军舰前往大沽。6月23日，[[俄罗斯帝国陆军|俄军]]将领命令德军占领[[北洋机器局|北洋机器局]]东局及[[天津火车站|天津火车站]]，海军第三营在战斗中有1名中尉军官及8名士兵[[阵亡|阵亡]]，多人受伤。与此同时，总督叶世克还曾派军进驻胶州、高密。

后来，曾有人提出为海军第三营在[[八国联军侵华战争|八国联军侵华战争]]中的阵亡士兵建立纪念碑，海军第三营为此还设立一个纪念碑筹委会，负责预算、设计和施工工作。纪念碑落成于1911年6月23日，位于I号营房与II号营房前方的台阶上。这座纪念碑虽然体量较小，设计却极为讲究。整个纪念碑沿中轴对称，碑体采用黑色[[大理石|大理石]]。两侧的大理石小圆柱柱头采用了[[花岗岩|花岗岩]]。三角形的碑首还有一只展翅雄鹰的[[浮雕|浮雕]]。碑的正面镌刻着“为纪念在1900-1901年的战斗中阵亡的第三海军陆战营的同事们”（Zum Gedächtniss der während der Kämpfe 1900-1901 für das Vaterland gefallen u. verstorbenen Kameraden des III Seebataillons.），碑的背面是阵亡官兵的姓名。前方踏步两侧[[斯芬克斯|人面狮身像]]雕塑，显得极为引人注目<ref>{{cite web |url=http://www.taidongzhen.com/HTML/News/Qingdao/qdrw/2017/0209/69216.html|title=海大校园内的德国第三海军营|accessdate=2017-02-09|work=钧甯|publisher=城市信报}}</ref><ref name=san>{{cite web|url=http://wanbao.qingdaonews.com/html/2008-04/27/content_650811.htm|title=三座消亡的殖民纪念碑|accessdate=2008-04-27|work=王栋|publisher=青岛晚报|archive-url=https://web.archive.org/web/20160304032929/http://wanbao.qingdaonews.com/html/2008-04/27/content_650811.htm|archive-date=2016-03-04|dead-url=yes}}</ref>。

该纪念碑于1914年日军占领青岛后拆除。纪念碑原址附近于1940年代还曾有一栋[[摄末社|小型神社]]，后来同样被拆除。

===国立山大教学楼===
国立山东大学于1932年成立后，在原兵营建筑群的基础上建设了一批新的教学楼，其中不乏史学、建筑学价值及纪念意义较高者。

====科学馆====
国立山东大学科学馆大楼位于原I号、II号营房南侧，桃海路{{注|海大校园内部道路，曾为[[齐河路|齐河路]]的一部分}}北侧。科学馆的建造最初为1931年夏国立青岛大学校务会议的决定，以供理学院的生物、化学、物理三系使用。科学馆大楼由著名建筑师[[董大酉|董大酉]]设计，1932年3月奠基，1933年1月完工，4月1日举行开幕典礼，《[[科学_(中国杂志)|科学]]》第17卷第6期《青岛国立山东大学科学馆开幕记》和1933年《[[建筑月刊|建筑月刊]]》第1卷第3期，都有详细、完整记录。[[青岛市市长|青岛市市长]][[沈鸿烈|沈鸿烈]]、本地其他学术机构和公共团体领袖，以及北京、杭州等地科学界名流都到场参加。沈鸿烈在致辞中称该建筑为青岛回归以来市内落成的第一栋伟大建筑物，并称“科学为救亡根本要图，斯馆之建成，实有重大意义与使命”<ref Name=jin>{{cite book|author=金山|title=《青岛近代建筑 1922-1937》|year=2016|publisher=同济大学出版社|location=上海|isbn=978-7-5608-6665-9}}</ref>。该建筑抗战期间曾为[[日本海军|日本海军]]青岛方面{{link-ja|海军根据地队|海軍根拠地隊|根据地队}}司令部。现为海洋生命学院所使用。2013年9月，该建筑遭到[[中国海洋大学|校方]]野蛮维修，许多旧建筑构件被拆卸。

建筑高三层，外墙全部以花岗岩石板覆盖，楼内有木制护墙板和全木制讲堂，是青岛市境内少有的[[哥德复兴式建筑|哥特式复兴建筑]]<ref Name=wei/><ref>{{cite web |url=http://www.weibo.com/u/1404199805?refer_flag=1005055013_&is_all=1&is_search=1&key_word=科学馆#_0|title=国立山大科学馆|accessdate=2013-11-04|work=|publisher=}}</ref>。

====化学馆====
国立山东大学化学馆大楼位于科学馆西南侧、IV号营房东北侧、桃海路{{注|海大校园内部道路，曾为[[齐河路|齐河路]]的一部分}}北侧。国立山大化学系曾使用科学馆三楼，但原有空间逐渐不敷使用。1936年11月14日的校务会议最终决定建设化学馆大楼。1937年1月19日，国立山大向青岛市工务局呈送建筑图样并获批准。同年3月，化学馆新楼奠基，7月竣工<ref>{{cite web |url=http://www.weibo.com/u/1404199805?refer_flag=1005055013_&is_all=1&is_search=1&key_word=科学馆#_0|title=国立山大科学馆|accessdate=2013-11-04|work=|publisher=}}</ref>。化学馆建成仅四个月后，国立山大因抗战内迁，日军占用校舍作为兵营时将加建一层<ref Name=jin/>。该楼现为海洋生命学院所使用。

建筑高三层（原高两层），奠基石上刻有教育家[[王世杰_(中華民國)|王世杰]]题写的“实学渊泉”四字，正门上方题有“化学馆”三字。奠基石题字的落款部分于[[文革|文革]]时被凿掉。

====工学馆====
国立山东大学工学馆位于闻一多故居（教职工宿舍）与大礼堂之间，I号、II号营房西侧，1935年11月建成。楼高二层，具有典型的[[现代主义建筑|现代主义风格]]，与[[包豪斯|包豪斯建筑]]类似。该建筑于大约1980年代被拆除，原址现为海大学术交流中心<ref>{{cite web |url=http://www.weibo.com/u/1404199805?refer_flag=1005055013_&is_all=1&is_search=1&key_word=工学#_rnd1497105401348|title=国立山大工学馆|accessdate=2014-06-30|work=|publisher=}}</ref><ref>{{cite web |url=http://www.weibo.com/u/1401647125?profile_ftype=1&is_all=1&is_search=1&key_word=山大#1497082624606|title=工馆新筑|accessdate=2014-06-28|work=|publisher=}}</ref>。

====体育馆====
国立山东大学体育馆位于图书馆（今海大档案馆）北侧，邻近[[大学路_(青岛)|大学路]]，1936年建成<ref>{{cite web |url=http://www.weibo.com/u/1401647125?profile_ftype=1&is_all=1&is_search=1&key_word=山大#1497082624606|title=工馆新筑|accessdate=2014-06-28|work=|publisher=}}</ref>。[[宋君复|宋君复]]教授任国立山东大学体育部主任时曾将这里作为[[第十一届奥运会|第十一届奥运会]]中国运动员训练场地。现已不存在。
