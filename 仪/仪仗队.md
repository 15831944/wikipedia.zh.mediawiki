'''仪仗队'''或'''儀隊'''是指儀式或典禮上的衛兵，在平時執行軍務，通常由慎於其體能及靈巧度者自願擔任，只有操行良好、儀態端正，並表現出對儀式責任感者才被認為是合格人選。儀隊是國家軍隊組織的象徵，從國家重大慶典、外賓來訪、敦睦艦隊或基地開放等，檢閱儀隊都是重要項目。

== {{USA}} ==
美國的五個軍種——[[美國陸軍|陸軍]]、[[美國海軍|海軍]]、[[美國空軍|空軍]]、[[美國海軍陸戰隊|海軍陸戰隊]]、[[美國海岸防衛隊|海岸防衛隊]]，及各州的國民警衛隊都有自身的儀隊。每個軍種的儀隊都駐紮於[[華盛頓特區|華盛頓特區]]，然而各地的軍事單位也都為了地方典禮節慶而設有儀隊。

== {{ROC}} ==
{{main|中華民國儀隊}}
[[中華民國儀隊|中華民國儀隊]]，是指[[中華民國國軍|中華民國國軍]]儀隊（又稱三軍儀隊）。中華民國的三個軍種：[[中華民國陸軍|陸軍]]、[[中華民國海軍|海軍]]、[[中華民國空軍|空軍]]都有各自的儀隊，平時在[[中正紀念堂|中正紀念堂]]、[[國父紀念館|國父紀念館]]、[[忠烈祠|忠烈祠]]、[[慈湖陵寢|慈湖陵寢]]、[[頭寮陵寢|頭寮陵寢]]等地駐守。在國慶日等國家重大慶典時，三個軍種的儀隊會聯合演出。此外還有[[中華民國海軍陸戰隊|海軍陸戰隊]]儀隊（簡稱陸戰儀隊）。陸戰儀隊與海軍儀隊的陸戰排不同，後者雖然服裝與陸戰儀隊相同，但在編制上屬於海軍儀隊。陸戰儀隊隸屬陸戰隊司令部，分成警衛連跟儀隊連，且由後者負責北部的禮兵駐防勤務。民國86年因為國軍精實案的關係將兩者合併，民國90年後政府撤除陸戰儀隊的駐防勤務，改由三軍儀隊擔任之。

而各高中職學校也從1950年代開始，基於當時[[軍訓|軍訓教育]]推廣的需要，在國軍儀隊的協助下逐漸創立學生儀隊，至今已約有逾八十所學校有學生儀隊。

中華民國最早的學生儀隊是私立開南商工(臺北市)在民國46年11月16日，由陳有諒校長核准成立之「開南商工大隊榮譽忠勤隊」；這個隊伍隸屬「中國青年反共救國團」臺北市支隊。而隊名「忠勤」的意義，是來自陳校長勉勵所有隊員要「忠於事，勤於業，爭取校、國榮譽」。

== {{PRC}} ==
{{Main|中国人民解放军陆海空三军仪仗队}}
[[中国人民解放军陆海空三军仪仗队|中国人民解放军陆海空三军仪仗队]]，隶属于[[中国人民解放军北京卫戍部队|中国人民解放军北京卫戍部队]]，与[[中国人民解放军军乐团|中国人民解放军军乐团]]、[[中国人民武装警察部队礼炮队|中国人民武装警察部队礼炮队]]共同承担不同规格的司礼任务。三军仪仗队的迎接外宾任务通常分为三种规格：

#第一种规格（最大阵容）由151人组成，仪仗队执行队长（走在军旗前面，持刀下口令者）1名；陆、海、空军执行队长队长（持指挥刀者）3名；旗手1名；护旗手两名；队员144名，用来迎接外国[[国家元首|国家元首]]或[[政府首脑|政府首脑]]。
#第二种规格是由127人组成的陆海空三军仪仗队，用来迎接外国军队的高级将领。
#第三种规格是由101人组成的单[[军种|军种]]仪仗队，用来迎接外国军队的单军种司令。

此外，仪仗队还担负着外国领导人向[[天安门广场|天安门广场]][[人民英雄纪念碑|人民英雄纪念碑]]献[[花圈|花圈]]，重大活动升旗的仪仗任务。另外[[中国人民解放军驻香港部队|中国人民解放军驻香港部队]]也设有三军仪仗队。

== {{RUS}} ==

另外於特定節日向莫斯科紅場列寧墓及[[無名烈士墓_(莫斯科)|無名烈士墓]]獻花環。

== {{BRA}} ==
[[巴西軍事|巴西陸海空三軍]]及[[憲兵|憲兵]]亦擁有多支兵力用於典禮用途。其中最重要的是[[巴西總統|巴西總統]]的儀隊，擔任第一騎兵團。

== {{IND}} ==
