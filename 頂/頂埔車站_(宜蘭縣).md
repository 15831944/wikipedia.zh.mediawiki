'''頂埔車站'''位於[[台灣|台灣]][[宜蘭縣|宜蘭縣]][[頭城鎮|頭城鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[宜蘭線|宜蘭線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]兩座。
*設於[[平交道|平交道]]旁。
*站內設有[[天橋|天橋]]連絡兩月台。（兩月台有設置出入口）
*車站站房已停用。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]。
*僅停靠區間車。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1937年5月1日：設站。
*1960年4月10日：復設招呼站，155、158、163、166次普通客車增停本站。
*2004年9月6日：管理站由[[頭城車站|頭城站]]改為[[礁溪車站|礁溪站]]。
*2005年1月16日：由簡易站降為招呼站。
*2015年6月30日：啟用多卡通刷卡機。
