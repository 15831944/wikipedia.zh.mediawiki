'''頂埔站'''位於[[台灣|台灣]][[新北市|新北市]][[土城區|土城區]]，為[[台北捷運|台北捷運]][[板南線|板南線]]（[[土城線|土城線]][[台北捷運土城線延伸頂埔段|頂埔延伸段]]）營運中的捷運車站及興建中的[[新北捷運|新北捷運]][[三鶯線|三鶯線]]捷運車站。將是[[MRT|捷運系統]]中第二個雙路線端點車站，另一個為[[南港展覽館站|南港展覽館站]]。

== 車站概要 ==
車站位於中央路四段頂埔里，接近頂埔高科技產業園區用地。車站代碼為'''BL01／LB01'''。站名取自於所在地的地名「'''頂埔'''」。

興建中的[[新北捷運|新北捷運]][[三鶯線|三鶯線]]在本站為起點站，是已經預留延伸的高架車站。

== 車站結構 ==
地下、高架（興建中）交會車站（[[臺北捷運|台北捷運系統]][[土城線|土城線]][[台北捷運土城線延伸頂埔段|頂埔延伸段]]為地下車站，[[新北捷運|新北捷運系統]][[三鶯線|三鶯線]]為高架車站。）

地下四層車站，一個[[島式月台|島式月台]]。四個出入口。

===車站出口===
出口1、2位於車站南端，出口3、4位於車站北端，[[無障礙|無障礙]]電梯位於出口1、3、4。

* 出口1：頂埔街（頂埔派出所共構大樓）[[File:Accessibility-directory.svg|20px]]
* 出口2：永福岩祖師廟（[[鴻海精密|鴻海精密]]大門旁）
* 出口3：大暖路（本出口預留未來聯合開發大樓的基座，為[[捷運|捷運]]<font color={{新北捷運色彩|三}}>█</font>[[三鶯線|三鶯線]]轉乘出入口）[[File:Accessibility-directory.svg|20px]]
* 出口4：三民路（階梯出入口設置於[[中央路|中央路]]四段52巷口西側，福利國大樓前；無障礙電梯設置於中央路四段64號，中央第一城大樓與泰隆福利國大樓間）。[[File:Accessibility-directory.svg|20px]]

== 歷史 ==
*2008年11月30日：土城線延伸到頂埔段動工。
*2010年1月20日：地下車站（土城線）正式動工。
*2015年2月：工程完工。
*2015年4月18日：土城線永寧站配合頂埔站展開初勘前穩定性測試作業，永寧站改以固定月臺候車，列車抵達永寧站後，再以空車方式延長行駛至頂埔站進行測試。
*2015年6月6日：台北市政府辦理頂埔站初勘作業。
*2015年6月23日：交通部辦理頂埔站履勘作業。<ref>{{cite web|url=http://www.bcc.com.tw/newsView.2595536|title=板南線延伸頂埔站履勘 頂埔至北車僅30分鐘|work=bcc.com.tw}}</ref>
*2015年7月6日：土城線「[[永寧站_(新北市)|永寧]]－頂埔」段於上午六點起正式通車<ref>[http://www.chinatimes.com/realtimenews/20150701004411-260405 板南線再延伸 頂埔站 6日正式通車《中時電子報》]</ref>。
*2016年6月15日：1號出入口啟用<ref>{{cite web|url=http://www.dorts.ntpc.gov.tw/link_data/index.php?mode=detail&id=27858&type_id=2|title=捷運板南線頂埔站1號出入口 6月15日開放啟用|work=新北市捷運工程局|deadurl=yes|archiveurl=https://web.archive.org/web/20160825214341/http://www.dorts.ntpc.gov.tw/link_data/index.php?mode=detail&id=27858&type_id=2|archivedate=2016-08-25}}</ref>
*2016年7月21日：三鶯線正式動工。
===預定時程===
*2024年：三鶯線正式通車。
