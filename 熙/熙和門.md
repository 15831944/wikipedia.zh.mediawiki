==历史==
熙和门始建于[[明朝|明朝]][[永乐_(明朝)|永乐]]十八年（1420年），初名“右顺门”。明朝[[嘉靖|嘉靖]]三十六年四月（1557年），紫禁城三大殿发生火灾，右顺门被殃及而焚毁，翌年重建。嘉靖四十一年九月（1562年）更名“归极门”。[[万历|万历]]二十五年六月（1597年），再次因火灾而烧毁，[[天启|天启]]年间重建。[[清朝|清朝]][[顺治|顺治]]二年五月（1645年）更名为“雍和门”。清朝[[乾隆|乾隆]]元年（1736年）为避[[雍正帝|雍正帝]]讳而更名“熙和门”。乾隆二十三年（1758年）第三次遭火灾烧毁，同年十二月重建。<ref name=xihemen/>

熙和门是从紫禁城前朝出[[西华门|西华门]]的必经之路，也是紫禁城外朝中路和西路的[[武英殿|武英殿]]、内阁等区相互联系的交通要道。明朝时，此门的梢间曾是百官奏事之所。<ref name=xihemen/>

2012年12月29日，故宫教育中心揭幕仪式举行。故宫教育中心设在熙和门南北庑房内，面积约800平方米，由四个不同主题功能的教室及辅助空间组成，针对不同年龄观众开展教育或研修活动。<ref>{{Cite web |url = http://news.xinhuanet.com/photo/2016-12/29/c_1120215964.htm |title = 故宫教育中心揭幕 |author =  |date = 2016-12-29 |publisher =  新华网|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.xinhuanet.com/2016-12/30/c_1120222437.htm |title = 故宫"文物医院"挂牌 观众可预约参观文物修复全过程 |author =  |date = 2016-12-30 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>

==建筑==
[[File:北京故宫2.JPG|thumb]]]]
熙和门是一座屋宇式大门，坐东朝西，面阔五间，进深两间，黄琉璃瓦单檐[[歇山顶|歇山顶]]。七檩中柱式大木结构，彻上明造，檐下为单昂三踩斗栱，龙锦枋心金线大点金旋子彩画。明次间脊步大门三槽，各安有朱红色宫门两扇，门上嵌有鎏金铜钉。梢间前后檐是坎墙槅扇窗，侧面开门。现存的建筑是清朝重建，所以和[[协和门|协和门]]虽然形制相仿但略有差异，体现了清朝中期官式建筑之特征。<ref name=xihemen/>

熙和门位于高2.62米的基座上，前后出礓磋慢道，各长17.75米。[[太和门|太和门]]广场一侧的慢道边缘向外侧倾斜，与内金水河的走势一致。<ref name=xihemen/>

熙和门南、北两侧各有11间庑房，均为黄琉璃瓦顶，连檐通脊，前出廊，后为风火檐。清朝时，北侧庑房为[[繙书房|繙书房]]，南侧庑房为[[起居注馆|起居注馆]]。 <ref name=xihemen/>
