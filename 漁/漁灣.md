'''漁灣'''是香港[[東區區議會|東區區議會]]下轄的[[選區|選區]]，代號'''C34'''，現任區議員為[[傘下爸媽|傘下爸媽]]的[[徐子見|徐子見]]，於2016年1月1日正式上任。

== 範圍 ==
選區範圍為[[漁灣邨|漁灣邨]]、[[樂軒臺|樂軒臺]]和[[柴灣邨|柴灣邨]]<ref name="獨立媒體151112">{{cite web|url=http://www.inmediahk.net/node/1038821 |title=中年大叔徐子見挑機鍾樹根 誓連「根」拔起 |publisher=[[香港獨立媒體|香港獨立媒體]] |date=2015-11-12 |accessdate=2015-12-13}}</ref>，與其相鄰的選區有[[翠灣_(選區)|翠灣]]、[[小西灣_(選區)|小西灣]]、[[環翠_(選區)|環翠]]、[[翠德|翠德]]和[[佳曉|佳曉]]。

== 沿革 ==
=== 鍾樹根 ===
漁灣選區首次出現於[[1994年香港區議會選舉|1994年香港區議會選舉]]，該次選舉由[[民主建港協進聯盟|民建聯]]的[[鍾樹根|鍾樹根]]與[[民主黨|民主黨]]的古南爭奪，最後鍾樹根勝出。

[[1999年香港區議會選舉|1999年香港區議會選舉]]，當時人口估算約18,594人，其中有9,812位是選民，投票站設於[[漁灣社區會堂|漁灣社區會堂]]。選區議席由鍾樹根和民主黨成員[[鄧徐中|鄧徐中]]爭奪。最終，鍾樹根以2,360票大比數擊敗取得1,253票的鄧徐中而勝出<ref>{{cite web|url=http://www.eac.gov.hk/pdf/distco/ch/1999dc_report/appendix%20iv_trad_chi.pdf |title=附錄四 - 一九九九年區議會選舉結果 |publisher=選舉管理委員會 |date= |accessdate=2015-12-13}}</ref>。

[[2003年香港區議會選舉|2003年香港區議會選舉]]，因為舊[[柴灣邨|柴灣邨]]完全清拆，當時人口估算減少到約13,507人，其中有7,314位是選民。今次鍾樹根自動當選<ref>{{cite web|url=http://www.legco.gov.hk/yr03-04/chinese/panels/ca/papers/ca0419cb2-2010-c.pdf |title=二零零三年區議會選舉報告書 |publisher=選舉管理委員會 |date= |accessdate=2015-12-13}}</ref>。

[[2007年香港區議會選舉|2007年香港區議會選舉]]，當時人口估算約13,630人<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/summaryc.html |title=區議會選舉2007 - 選區分界建議概要(東區) |publisher=[[選舉事務處|選舉事務處]] |date=2012-06-19 |accessdate=2015-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20080501203936/http://www.elections.gov.hk/dc2007/chi/summaryc.html |archivedate=2008-05-01 }}</ref>，其中有7,849位選民<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/num_electors_eastern.html |title=區議會選舉2007 - 選舉簡介 |publisher=[[選舉事務處|選舉事務處]] |date=2012-06-19 |accessdate=2015-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20150621034533/http://www.elections.gov.hk/dc2007/chi/num_electors_eastern.html |archivedate=2015-06-21 }}</ref>，鍾樹根今次與[[社民連|社民連]]的[[陳裕南|陳裕南]]爭奪，最後鍾樹根以1,922票勝出。

[[2011年香港區議會選舉|2011年香港區議會選舉]]，因為新柴灣邨落成，當時人口估算增加到約15,243人<ref>{{cite web|url=http://www.elections.gov.hk/dc2011/chi/summaryc.html |title=2011年區議會選舉 - 選區分界概要 |publisher=[[選舉事務處|選舉事務處]] |date=2011-09-05 |accessdate=2015-12-13}}</ref>，其中有8,428位是選民<ref>{{cite web|url=http://www.elections.gov.hk/dc2011/chi/ebriefsc.html |title=2011年區議會選舉 - 區議會選舉簡介 |publisher=[[選舉事務處|選舉事務處]] |date=2011-09-20 |accessdate=2015-12-13}}</ref>。今次鍾樹根笫二次自動當選。

=== 徐子見 ===
[[2015年香港區議會選舉|2015年香港區議會選舉]]，鍾樹根這次與最後一天才報名參選的[[徐子見|徐子見]]爭奪。最後無黨派的徐子見以2,026票擊敗鍾樹根的1,863票，鍾在對手險勝163票之下失落已經擔任24年的區議員職位。因徐子見從提名到當選只有六星期的地區工作經驗，而鍾樹根則曾任六屆區議會議員，因此各界對選舉結果相當驚訝。

徐子見在[[雨傘運動|雨傘運動]]後，有感自己這一代人只關注經濟發展，甚少參與政治，造成[[建制派|建制派]]壟斷區議會，禍害下一代；亦曾參與「傘下爸媽」及「香港民主宣傳小組」等組織。他眼見臨近區議會選舉提名截止日期，包括漁灣在內的多個選區即將自動當選，而且他曾在[[柴灣|柴灣]]上學，對該區有一定認識，所以決定在提名期最後一天報名參選漁灣選區<ref name="獨立媒體151112" />。他的政綱包括「建議設立東區牙科保健基金」、「打破建制派壟斷區議會」、「提升交通道路、環境衛生水平」、「增設長者康體設施」、「開展兒童義務補習」等等<ref>{{cite web|url=http://hk.apple.nextmedia.com/dc2015/candidate/C34/徐子見 |title=區議會選舉 2015 - 徐子見 |publisher=[[蘋果日報_(香港)|蘋果日報]] |date=2015 |accessdate=2015-12-13}}</ref>。

徐在當選後表示希望將義務補習的理念帶到柴灣，亦會逐步實踐政綱。他亦指勝出的原因是鍾當區議員和[[香港立法會|立法會]]議員的表現並不好，令該區選民求變<ref>{{cite web|url=https://www.thestandnews.com/politics/徐仔揼樹根-徐子見-漁灣居民求變心切-誰選都可以贏/ |title=【徐仔揼樹根】徐子見：漁灣居民求變心切　誰選都可以贏 |publisher=[[立場新聞|立場新聞]] |date=2015-11-24 |accessdate=2015-12-13}}</ref>。

有網民於選舉結果公布的凌晨（11月23日凌晨）在社交網站[[Facebook|Facebook]]開設「全港市民熱烈慶祝treegun落選委員會」專頁，慶祝鍾樹根落敗。專頁內容為鍾落選的新聞和他以往的負面新聞，包括曾把「明目張膽」錯誤說成「明張目膽」的新聞等等<ref>{{cite web|url=http://news.memehk.com/posts/12205 |title=網民「全港市民熱烈慶祝treegun落選委員會」贈慶 徐子見：贏咗係開心嘅 |publisher=謎米新聞 |date=2015-11-23 |accessdate=2015-12-13}}</ref>。當日晚上8時，約150人應網上號召，在位於[[漁灣邨|漁灣邨]]漁安樓地下的區議員辦事處外舉行「連根拔起」慶祝會，長約40分鐘，期間參加者開香檳慶祝、施放多響禮炮、唱出[[徐小鳳|徐小鳳]]名曲《喜氣洋洋》慶祝<ref>{{cite web |url=http://hk.apple.nextmedia.com/dc2015/news/all/54461236 |title=【短片】「連根拔起」街坊勁High 香檳禮炮齊齊拉 |publisher=[[蘋果日報_(香港)|蘋果日報]] |date=2015-11-23 |accessdate=2015-12-13}}</ref>。
