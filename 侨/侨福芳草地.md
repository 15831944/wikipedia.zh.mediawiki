'''侨福芳草地'''，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[朝外街道|朝外街道]][[东大桥路|东大桥路]]9号，由北京侨福置业有限公司开发管理<ref>{{cite web|title=侨福集团|url=http://www.parkviewgreen.com/cn/about/parkview-group-1/|publisher=Parkview Green芳草地}}</ref>，是集写字楼、购物中心、艺术中心、酒店于一体的创新建筑。

整个建筑由四座塔楼构成。通透的玻璃幕墙及钢架结构，结合顶部采用的ETFE膜材料（与国家游泳中心所使用的材料相同）组成独特的节能环保罩，在内部实现独立的微气候环境。室内空气质量达到MERV标准的第11级，是综合性建筑所能达到的最高级别（国内建筑的MERV水平通常在7级左右）。楼顶的斜面设计，充分保证了周围社区的日照权，做到与环境共生的和谐相处。<ref name=tese>{{Cite web |url = http://sz.winshang.com/news-416469-2.html |title =盘点上海K11、侨福芳草地等国内特色体验式购物中心  |author =  |date =2014-12-02  |publisher =赢商网  |language =  |accessdate =  |quote =  }}</ref>。

侨福芳草地的能源使用量仅为同等规模建筑的50%，是中国第一个获得由美国绿色建筑协会(USGBC, U.S. Green Building Council)颁发的 LEED®（Leadership in Energy & Environmental Design Building Rating System）“铂金级”认证的综合性商业体。此外，侨福芳草地建筑曾获多项建筑奖，其中包括2010年MIPIM Asia“年度最佳绿色建筑”，2012年GBA环保建筑大奖中的亚太区绿色建筑大奖等<ref>{{Cite web |url = http://www.parkviewgreen.com/html/cn/about/parkview-group/parkview-group/ |title = 侨福芳草地 |author =  |date =  |publisher =  侨福芳草地|language =  |accessdate =2017-07-21  |quote =  }}</ref>。

==侨福芳草地购物中心==
[[File:Interior_of_Parkview_Green_(20170315171717).jpg|thumb]]
2012年9月16日，侨福芳草地购物中心正式开放。购物中心内摆放有[[萨尔瓦多·达利|达利]]等艺术家的500多件艺术作品，其中主要是雕塑。艺术家[[王鲁言|王鲁言]]专为购物中心创作了巨幅腕表系列油画作品<ref>{{Cite web |url = http://news.winshang.com/html/012/8520.html |title = 北京侨福芳草地购物中心9月16日开放 |author =  |date = 2012-09-16 |publisher =  |language =  |accessdate =  |quote =  }}</ref>。侨福芳草地拥有236米中国最长室内长桥、2000平方米公共艺廊<ref>{{Cite web |url = http://hb.winshang.com/news-479398.html |title = 盘点：北京侨福芳草地等全球特色购物中心吸客策略 |author =  |date = 2015-05-17 |publisher =赢商网  |language =  |accessdate =  |quote =  }}</ref>。购物中心开放后，很快成为北京知名的特色体验式购物中心<ref name=tese/><ref>{{Cite web |url =  http://www.soupu.com/news/622079|title = 极致体验：北京侨福芳草地购物中心的商业经营逻辑 |author =  |date = 2015-03-24  |publisher =搜铺  |language =  |accessdate =  |quote =  }}</ref>。

==侨福当代美术馆·北京==
'''侨福当代美术馆·北京'''，位于侨福芳草地D座10层。2012年8月，芳草地展览馆建成。2016年10月更名为“侨福当代美术馆·北京”。由侨福集团执行长黄建华创办，是非营利性私立美术馆。占地总面积4000多平方米，拥有两个大型展厅及演讲厅等设施<ref>{{Cite web |url = http://www.parkviewgreenmuseum.com/index.php?s=/About/index/lang/cn.html |title = 关于我们 |author =  |date =  |publisher = 侨福当代美术馆·北京 |language =  |accessdate =  2017-07-21|quote =  }}</ref>。

==北京怡亨酒店==
'''北京怡亨酒店'''，位于侨福芳草地，2013年3月开业，是小型奢侈酒店联盟成员。内部布满艺术品，是[[亚洲|亚洲]]最大的达利作品收藏地。还收藏有[[曾梵志|曾梵志]]、[[郑路|郑路]]、[[陈文令|陈文令]]、[[任哲|任哲]]、[[管勇|管勇]]、[[王鲁炎|王鲁炎]]、Pierre Matter、安迪·沃荷、[[高孝午|高孝午]]、[[周良|周良]]、Maarten Baas， Bertjan Pot、阿纳·雅格布森、门迪尼、菲利普·斯达克等人的作品<ref>{{Cite web |url = http://travel.163.com/13/1008/10/9ALK198O00064L1N.html |title = 北京侨福芳草地怡亨酒店的三大创新亮点 |author =  |date = 2013-10-08 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://bj.winshang.com/news-201173.html |title =北京怡亨酒店演绎奢华体验 主题艺术成吸引客流亮点  |author =  |date =2013-11-21  |publisher = 赢商网 |language =  |accessdate =  |quote =  }}</ref>。
