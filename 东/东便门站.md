'''东便门站'''，位于今[[北京市|北京市]][[东城区_(北京市)|东城区]][[北京城东南角楼|内城东南角楼]]西南侧，[[北京明城墙遗址公园|北京明城墙遗址公园]]内，1959年拆除。

==简介==
东便门站是[[京奉铁路|京奉铁路]]及[[北京环城铁路|北京环城铁路]]的车站。1900年末，开工建设[[京奉铁路|京奉铁路]]，起点是[[正阳门东站|正阳门东站]]，第二站便是东便门站”。1901年，京奉铁路[[北京南站|永定门站]]至正阳门东站段通车，东便门站投入使用<ref name=laobeijing>{{Cite web |url =https://baijiahao.baidu.com/s?id=1568358366166171&wfr=spider&for=pc|title =老北京东便门火车站（一）  |author =  |date = 2017-05-25 |publisher =百家号  |language =  |accessdate =  |quote =  }}</ref>。

1915年6月16日，北京开始兴建环城铁路，自[[北京北站|西直门站]]经[[德胜门站|德胜门站]]、[[安定门站_(铁路)|安定门站]]、[[东直门站_(铁路)|东直门站]]、[[朝阳门站_(北京环城铁路)|朝阳门站]]至东便门站沟通[[京奉铁路|京奉铁路]]，终点是[[正阳门东站|正阳门东站]]。1916年1月1日，环城铁路全线建成通车<ref name=qi>环城铁路——为老北京的交通松口气儿，和谐之旅2011年4月</ref>。

[[File:Ming_walls.jpg|thumb]]初年，1915年修建京师环城铁路，拆断了内城东南角楼附近的城墙和敌台。这是当年京师环城铁路穿过内城南城墙处的券洞，位于内城东南角楼以西]]
东便门站位于[[北京城东南角楼|北京内城东南角楼]]西南侧，[[北京内城|北京内城]]城墙南侧，护城河以北。初期建有站房、跨铁路天桥等设施<ref name=laobeijing/><ref>{{Cite web |url =http://www.sohu.com/a/143692486_115482  |title =  老北京东便门火车站（二）|author =  |date =2017-05-26   |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。1915年建环城铁路时，在北京内城东南角楼北侧的内城东墙和西侧的内城南墙上各开了一个券洞，并在两个券洞之间的内侧砌筑高墙，从而使环城铁路避开了北京内城东南角楼。这是历史上第一次在北京内城城墙上开券洞。其中，内城东南角楼西侧的内城南墙上的券洞保留至今，位于东便门站北侧<ref>{{Cite web |url =http://www.sohu.com/a/143975861_115482  |title =老北京东便门火车站（三）  |author =  |date = 2017-05-27  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

1954年7月，北京铁路管理局根据[[北京市人民政府|北京市人民政府]]的城市建设总体规划，计划封闭北京环城铁路的德胜门站、安定门站、东直门站、朝阳门站，以便减少环城铁路对北京城市地面交通的影响。1958年开始，西直门站至[[广安门站|广安门站]]的铁路联络线停用，并改建北京环城铁路，拆除了从西直门站经德胜门站、安定门站、东直门站至朝阳门的铁路。1959年[[北京站|北京站]]建成后，朝阳门站至东便门站间的铁路也被拆除<ref name=qi/>。

东便门站西端的号志房1901年与该站同建，保存至今，现位于[[北京明城墙遗址公园|北京明城墙遗址公园]]内。东便门站东端原先有岔房，旧址位于今[[北京二环路|北京二环路]][[东便门桥|东便门桥]]处<ref>张书剑、海妹，老北京东便门火车站情结，北京纪事2010(10):108-109</ref>。
