'''东南亚闽南语'''，又称'''东南亚福建话'''，是[[闽南语|闽南语]][[泉漳片|泉漳片]]的一种域外分支。这种闽南语是在17世紀到20世纪初叶由[[泉州|泉州]]、[[漳州|漳州]]等地[[移民|移民]]因过番到[[南洋|南洋]]（[[东南亚|东南亚]]的旧称）而带过去的[[语言|语言]]。广义上东南亚闽南语一词也包括若干个[[潮汕片|潮汕片]]在东南亚的域外变体。

== 语言特点 ==
这种闽南语的域外变体有以下两个特点：

1）这种闽南语是由早期[[中国|中国]][[福建省|福建省]]的移民带到南洋的，因此保留了许多比较古雅的词汇和用法，有些在中国闽南本土已经消失了。

2）也因为是早期带过去的语言，东南亚闽南语在发展过程中因缺乏很多现代生活所需要的词汇（尤其是科技、商业、东南亚特殊生活环境用语）而向东南亚若干个本地语言和方言借词，形成多种不同掺杂了土语词汇的闽南语。

== 发展历史 ==

;新加坡
在新加坡，福建话指的是[[閩南話|閩南話]]，新加坡人把閩南話当成是福建话。
早期[[新加坡閩南話|新加坡福建话]]受到马来语的影响，后来又受到英语和华语影响，已经发展成独特的闽南方言，虽然英语和华语普遍使用使得它面临被淘汰的厄运，但是老一辈的华人习惯上还是使用福建话作为沟通工具。一般来说，新加坡華人把[[泉漳片|泉漳片]]福建话当成一种没有文字的语言使用。

== 东南亚闽南语的前景 ==
东南亚闽南语属于派生的闽南语次方言。由于东南亚各国原本都有自己的国语，加上英语在现今社会还拥有者相当重要的经济地位，许多东南亚国家政府的教育政策都偏重让国人学习通用语如馬來語(馬來西亞、印尼推動)、英语以及越来越普遍的华语（普通话），汉语方言在东南亚的发展前景如何依然是个未知数。
