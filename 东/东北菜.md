{{中国菜}}

'''东北菜'''是指在[[中国东北地区|东北]]，包括[[黑龙江省|黑龙江]]、[[吉林省|吉林]]、[[辽宁省|辽宁]] 、[[河北省|河北]]东北部（[[秦皇岛市|秦皇岛市]]为主）、[[内蒙古自治区|内蒙古]]东部的烹饪菜系，东北菜的特点是量大，用料广泛。以火候足而滋味浓郁的各式[[炖|炖菜]]闻名。

东北菜融合了[[中國菜系|中国其他菜系]]的一些特点，尤其受[[鲁菜|鲁菜]]影响巨大。辣味稍敛的[[四川菜|四川菜]]（[[鱼香肉丝|鱼香肉丝]]）、改良后的[[河北菜|河北菜]]在东北菜谱中也是屡见不鲜。东北菜以炖为主，腌制的菜也是在寒冷的东北冬天热情好客的东北人桌上少不了的一道菜，例如[[酸菜|酸菜]]。以及一些融会了少数民族特色的菜类，包括东北[[朝鲜族|朝鲜族]]人的烧烤和[[新疆|新疆]]人的[[羊肉串|羊肉串]]。

==代表性的菜==
# 酸菜白肉鍋：也称東北火鍋。主材料為[[酸菜|酸菜]]與[[豬|豬]][[五花肉|五花肉]]，是流行於[[中國東北|中國東北]]民眾间的家庭傳統菜餚，後傳入北京。做法是使用生炭火的銅鍋，將酸白菜與高湯煮成鍋底，白肉下锅涮熟，佐[[麻油|麻油]]、[[生抽|生抽]]、蒜泥等食用<ref>关云德。满族的火锅。满族文学，2003年第4期：64頁。</ref><ref>徐文苑。中国饮食文化概论。北京：清华大学出版社，2005年：第10頁。ISBN 978-7-81082-370-8。</ref>。
# [[白肉血肠|白肉血肠]]：与酸菜白肉锅类似，但同时加入[[血肠|血肠]]一同炖煮。
# [[猪肉炖粉条|猪肉炖粉条]]：猪肉、白菜与土豆粉条一同炖煮。口感软烂，猪肉肥而不腻。
# [[小鸡炖蘑菇|小鸡炖蘑菇]]：山鸡与东北特产[[蜜環菌|榛蘑]]一同炖煮，味道鲜美
# [[豆角炖排骨|豆角炖排骨]]：主料为土豆、豆角与排骨，炖煮过程中加入大酱。汤汁浓厚香醇
# [[东北乱炖|东北乱炖]]：东北著名的杂烩菜，主料包括芸豆、土豆、排骨、茄子、蘑菇，不同地区也有加入西红柿、玉米等。
# [[锅包肉|锅包肉]]：大片猪里脊肉挂粉炸好后，回锅与糖醋汁（或番茄酱）及葱姜香菜翻炒而成。
# [[扒熊掌|扒熊掌]]
# [[地三鲜|地三鲜]]：茄子、青椒、土豆炸制后，再回锅爆蒜淋薄芡而成。
