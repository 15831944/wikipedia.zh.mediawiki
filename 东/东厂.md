'''東廠'''，其全名為'''東緝事廠'''，[[廠衛|廠衛]]之一。[[中國|中國]][[明朝|明朝]]時期的由[[宦官|宦官]]执掌的特權[[監察官|監察]]、[[情報機構|情治機構]]，偵查[[異見人士|異見人士]]，以鎮壓反對力量。

东廠對[[官吏|官吏]]、[[士大夫|士大夫]]甚至於一般[[庶民|庶民]]製造了大量[[冤案|冤案]]，在當時頗受士人反感。執行公務時，與[[錦衣衛|錦衣衛]]相同，持有「[[駕帖|駕帖]]」以証明自己代[[皇帝|皇帝]]行事，並且由刑科[[给事中|给事中]]的「[[佥签|佥签]]」。廠衛的主要偵查以反[[叛亂|叛亂]]、捉拿[[異議分子|異議分子]]為主，与其他两厂（[[西厂|西厂]]、[[内行厂|内行厂]]）一卫（锦衣卫）合称“厂卫”，是明朝“特务治国”的象征。清代「以軍法從事」常態化遂罷。



東廠約相當於現代社會的[[中央情報局|中央情報局]]

== 起源 ==
原為[[藩王|藩王]]的[[明成祖|明成祖]]，以[[靖難之役|靖難之役]][[起事|起事]]起兵奪位，篡奪了[[建文帝|建文帝]]的中央[[政權|政權]]，故[[異議分子|異議分子]]紛起。[[永樂|永樂]]十八年（1420年），[[明成祖遷都北京|明成祖遷都北京]]後，為了鎮壓政治上的反對力量，決定設立一個稱為「東緝事廠」，簡稱“東廠”的新[[衙門|衙門]]，地點位於[[燕京|燕京]]（今[[北京|北京]]）[[東安門|東安門]]之北，一說[[東華門|東華門]]旁。（今北京東城區[[東廠胡同|東廠胡同]]，據說系原東廠所在地。）<ref>《明史·卷七》</ref>

== 職掌 ==
明成祖創立東廠，是特務與[[秘密警察|秘密警察]]機關，由[[皇城|大內]][[宦官|宦官]]擔任[[行政長官|行政長官]]「總督東廠」。東廠的主要職責是[[監視|監視]][[政府|政府]]、[[軍隊|軍隊]]、[[官員|官員]]、[[士人|士人]]、社會名流、學者等各種[[政治|政治]]力量，並將監視結果直接向[[皇帝|皇帝]]匯報，整肅[[異議份子|異議份子]]、[[政治犯|政治犯]]或者其他[[罪犯|罪犯]]。依據[[偵查|偵查]]得到的[[情報|情報]]，對於那些地位較低的政治[[反對派|反對派]]，不經司法[[審判|審判]]，東廠可以直接[[逮捕|逮捕]]甚或[[刑求|刑求]]；而對於[[名人|名人]]、[[門閥|門閥]]、高官、政要或者有[[皇室|皇室]]、[[貴族|貴族]]身份的高地位[[反對派|反對派]]，東廠在得到皇帝或[[朝廷|朝廷]]的授權後也能夠對其執行刑訊。

明代知名的掌權太监[[王振|王振]]、[[刘瑾|刘瑾]]、[[冯保|冯保]]、[[魏忠贤|魏忠贤]]都曾统领东厂。

== 編制 ==
東廠的[[行政長官|行政長官]]為[[欽差|欽差]][[掌印太監|掌印太監]]，全稱職銜為：欽差總督東廠官校辦事太監，簡稱總督東廠、提督東廠，[[宦官|宦官]]與[[錦衣衛|錦衣衛]]尊稱其為督主，[[士大夫|士大夫]]稱其為廠督、廠璫（璫，[[宦官|宦官]]的雅稱），尊稱為廠公。初設時由[[司禮監|司禮監]]掌印太監（俗稱宗主）兼任，後因掌印太監事務繁雜，改由司禮監[[秉筆太監|秉筆太監]]中位居第二、第三者擔任。

東廠的屬官有「掌刑千戶」、「理刑百戶」各一員，由「錦衣衛千戶」、「錦衣衛百戶」來擔任，稱「貼刑官」<ref>《明史刑法志》提到東廠的制度，“凡中官掌司禮監印者，其屬稱之曰宗主，而督東廠者曰督主。東廠之屬無專官，掌刑千戶一，理刑百戶一，亦謂之貼刑，皆衛官。”</ref><ref>《[[玉光剑气集|玉光剑气集]]》：“东厂始于永乐十八年，设有掌贴刑千、百户二员，掌班司房四十余名，十二伙當头办事百余名，番役可千名。凡中府等处会审大狱，北镇抚司拷讯重犯，本厅皆有人听记，具口词一本，于本日晚或次日奏进。每日兵部访看，有无进部，有无塘报。京城各门，皇城各门，关防出入，俱有事件，或地方失火，或雷击何物，亦奏闻。每月晦日，奏报在京杂粮米豆油面之价。凡禁地人命，亦皆有事件。”</ref>。

[[隸役|隸役]]（稱掌班、領班、司房，共四十餘人）、「緝事」（稱「役長」和「番役」）等[[軍官|軍官]]由[[錦衣衛|錦衣衛]]撥給。

東廠入內即擺設大幅[[岳飛|岳飛]]畫像，提醒東廠人員辦案毋枉毋縱。

===制式武器===
*東廠錦衣衛制式[[兵器|兵器]]：雙刀（[[繡春刀|繡春刀]]、同一刀鞘），一刀之刀柄刻「東」字、另一刀之刀柄刻「廠」字。合刀「東廠」二字相對面置內側。<ref>收藏家/王度，"錦衣衛制式兵器：雙刀"，{{YouTube|1Bh1Emp3kaM|明朝宮廷鬥爭(4/6)：明成祖兇殘 "砲烙"宮女至死(film)}}。</ref>

== 評價 ==
東廠和[[錦衣衛|錦衣衛]]雖然皆是[[明朝|明朝]]的特權監察機關，但是東廠地位相對較高，和專為皇帝一人服務的各形態秘警機構一樣。由於東廠鎮壓[[言論自由|言論自由]]的手段極其殘酷，且有宦官因向皇帝邀功或謀自身的不法私利而為之，所以容易製造大量的[[文字獄|文字獄]]與[[冤案|冤案]]，以致於東廠在《[[明史|明史]]》的口碑極差。
