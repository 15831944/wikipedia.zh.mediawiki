'''东华理工大学'''位于中国[[江西省|江西省]]，是一所以核工程，地质工程为主要特色的综合性理工大学，学科门类比较齐全。原称'''华东地质学院'''，学校注册地位于江西省[[抚州市|抚州市]]，在江西省[[南昌市|南昌市]]有分校区。2010年后逐渐将南昌校区作为校本部建设。

'''东华理工大学'''创建于1956年，是[[中华人民共和国|中华人民共和国]]原核工业部第一直属高校，是[[中国|中国]]教育部“'''中西部基础建设工程'''”二期入选高校，是“'''111计划'''”高校，拥有一个“'''国家级国际联合研究中心'''”，是目前[[江西省|江西省]]省内唯一的一个。多年来，东华理工大学科研经费长期位居江西省第二位。自2010年以来，南昌校区发展迅速，综合实力大幅增强。

东华理工的校庆日是中国第一颗原子弹爆炸成功的10月16日。

== 历史 ==
[[File:Taigu_Kong_Xiangxi_Jiuju_2013.08.27_09-27-19.jpg|thumb]]创立]]
1956年6月，为培养铀矿探矿技术人员，[[第二机械工业部|第二机械工业部]]在山西[[太谷县|太谷县]]创立'''太谷地质学校'''。1958年6月，在太谷地质学校的基础上，建立了'''太原地质专科学校'''（本科），校址迁往太原市。1959年9月学校迁至邻近特大型铀矿区（核工业721矿）的江西省抚州市，校名改为'''抚州地质专科学校'''（本科）。1969年至1973年，受“文革”的影响，学校停办。1974年开始筹建复校。其间培养了三届中专生。1978年4月复办大学，校名为'''抚州地质学院'''。1982年1月更名为'''华东地质学院'''。1999年4月，学院由中国核工业总公司转属地省管理，实行江西省人民政府与国防科工委共建，以江西省人民政府为主进行管理。2002年1月，'''江西省国防科技工业学校'''并入。2002年4月学院更名为'''东华理工学院'''。2003年'''抚州师范专科学校'''并入。2007年3月更名为东华理工大学。<ref>[http://www.ecit.edu.cn/lshyg/xshyg.htm 校史沿革]</ref>

== 院系设置 ==
东华理工大学设有[[地球科学|地球科学]]学院、地球物理与测控技术学院、核科学与工程学院、[[水资源|水资源]]与[[环境工程|环境工程]]学院、建筑工程学院、[[信息工程|信息工程]]学院、[[化学|化学]][[生物|生物]]与[[材料科学|材料科学]]学院、[[机械|机械]]与电子工程学院、[[经济|经济]]与[[管理|管理]]学院、[[文法|文法]]学院、[[艺术|艺术]]学院、外国语学院、理学院、[[体育|体育]]学院、抚州师范学院、高等职业技术学院、继续教育学院、国防科学技术学院、[[海军|海军]]后备军官学院、[[研究生|研究生]]部、思想政治理论教学科研部、[[测绘工程|测绘工程]]学院、长江学院（抚州）、软件学院以及教育心理学教学部等20多个教学单位和二级学院。

== 学生培养 ==
目前校本部现有各类在校学生29000余人。是全国首批学士学位和第四批硕士学位授权单位。近几年与[[南京大学|南京大学]]、[[中国地质大学|中国地质大学]]等单位在4个专业合作培养博士生。全院现有65个一级、二级硕士学位授权点、 11个工程硕士授权点、57个本科专业、11个国防军工专业（含5个国家管理专业）、1个国防重点专业、14个省部级重点学科。

== 教师队伍 ==
现有教职工2800余人，教授、副教授600余人；教师中具有博士学位168人、硕士学位585人，兼职博士生导师15人，中国工程院双聘院士2人，“井冈学者”1人、“东华学者”2人；有省级学科带头人40人，省级骨干教师52人，省部级有突出贡献专家4人，享受国务院政府特殊津贴28人，全国优秀教师4人，省级教学名师7人，江西省新世纪百千万人才工程人选27人。同时，学校还聘请了200多位国内外兼职教授。学校由中国工程院钱七虎院士担任名誉校长。
