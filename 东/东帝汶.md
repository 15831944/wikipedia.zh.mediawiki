'''東帝汶民主共和國'''（[[德頓語|德頓語]]：Repúblika Demokrátika
Timor-Leste），通称'''东帝汶'''（[[德頓語|德頓語]]：Timor Lorosa'e；[[葡萄牙語|葡萄牙語]]：Timor-Leste），是位於[[東南亞|東南亞]]地區[[帝汶島|帝汶島]]東端的[[國家|國家]]，曾是葡萄牙在[[远东|远东]]仅有的两个殖民地之一（另一個為[[澳門|澳門]]）。1999年8月底在[[澳洲|澳洲]]等國家的支持下通過公投決定[[獨立|獨立]]，2002年5月20日零时獨立，2002年9月27日正式加入聯合國，成為第191個聯合國會員國<ref>{{cite web|url=https://macauhub.com.mo/zh/2015/09/01/president-of-timor-leste-begins-first-official-visit-to-china|title=东帝汶总统开始访华之旅|work=Macauhub|date=2015-09-01|publisher=Macauhub|accessdate=2017-07-03}}</ref>。

東帝汶政府採用“Timor-Leste”同時作為其[[葡萄牙語|葡萄牙語]]的國名。leste是葡語「東」的意思；而Timor本來是[[印尼語|印尼語]]timur，也是「東」的意思。所以東帝汶這個國號是一個[[恆真式|恆真式]]名稱。而在印尼統治時期的東帝汶，採用“Timor Timur”作為地名，並簡稱東帝汶為“Tim-tim”。“Lorosa'e”則是[[德頓語|德頓語]]的「東」，意思是日出的太陽。

== 历史 ==
{{main|东帝汶历史}}
[[葡萄牙|葡萄牙]]在16世纪早期开始与[[帝汶岛|帝汶岛]]进行贸易，并进而实行殖民统治，直到20世纪中叶。1859年因为与[[荷兰|荷兰]]发生冲突而签订了一个条约，葡萄牙割让了岛屿西边的一部分（[[西帝汶|西帝汶]]）。[[二战|二战]]期间，[[日本|日本]]在1942－1945年占领了[[葡属帝汶|葡属帝汶]]，但日本战败后葡萄牙又恢复了统治。

葡萄牙政府於1974年開始實行[[非殖民化|非殖民化]]，放棄海外[[殖民地|殖民地]]，導致世界各葡屬殖民地紛紛脫離葡萄牙統治而獨立。東帝汶於1975年11月28日從葡萄牙獨立，成立東帝汶民主共和國，但在9天之後馬上就[[东帝汶印度尼西亚占领时期|被印尼入侵並佔领]]，1976年7月成为印尼的'''[[东帝汶省|东帝汶省]]'''（Timor Timur）。接下来的20年-{zh-hans:里;zh-hant:裡;}-，不断发生抗争运动，并爆发了持續二十多年的獨立戰爭，造成了10－25万人的死亡。

[[File:East_Timor_Demo.jpg|thumb]]
1999年8月30日，在[[联合国|联合国]]的监督下举行了[[1999年东帝汶独立公投|全民公投]]，绝大多数的东帝汶人投票同意从印尼独立。在公投到1999年9月下旬多国维和部队进驻期间，由印尼军方和反东帝汶独立者组建的东帝汶民兵展开了[[1999年东帝汶危机|大规模的焦土运动]]进行报复。大约有1,400名东帝汶人被殺害，并强行驱离了30万人到[[西帝汶|西帝汶]]成为[[难民|难民]]。国家大部分的基础设施（包括住宅、灌溉系统、供水系统、学校、电网等）几乎100％被摧毁。1999年9月20日，以[[澳大利亚|澳大利亚]]为首的国际部队介入以后，才结束了这場暴力行动。 

經「聯合國東帝汶過渡行政當局」管理约兩年半后，于2002年5月20日零时正式恢復獨立，國名為東帝汶民主共和國，成为国际公认的一个独立国家。

2006年4月下旬，新国家领导人上台后造成国内局势紧张，法治接近崩溃。在东帝汶政府的要求下，以澳大利亚为首的国际维和部队（ISF）于5月下旬进驻东帝汶。8月，[[联合国安理会|联合国安理会]]设立[[联合国东帝汶综合特派团|联合国东帝汶综合特派团]]（UNMIT），其中包括有超过1,600名警察。东帝汶局势随之恢复稳定，2007年4月和6月的总统选举及议会选举基本上和平进行。2008年2月，反对派对总统和总理进行了一次不成功的攻击。反对派头目在袭击中丧生，大多数叛军于2008年4月投降。自此以後，东帝汶局势一直非常稳定。

== 政治 ==
{{main|东帝汶政治}}
[[File:Gouverneurspalast_klein.jpg|200px]]

2002年3月22日，東帝汶制憲會議的88名議員，開會通過東帝汶的第一部憲法。

2009年改革前，东帝汶的司法体系以[[印度尼西亚|印度尼西亚]]为基础，2009年议会通过并颁布了以[[葡萄牙|葡萄牙]]法律体系为基础的新的民事法律制度，2011年實行新的民法典。

===行政部门===
东帝汶公民17岁后拥有投票权。

总统由人民選出，任期5年，僅可連任一屆。[[东帝汶总统|东帝汶总统]]只是[[虚位元首|虚位元首]]，但是可以否决议案、解散国会和呼吁发起选举。最近一次的总统选举已在2012年3月举行。总统当选后成为多数党或多数党联盟的领袖，并任命总理。总理是政府首脑；内阁是部长会议。

主要政党有[[東帝汶獨立革命陣線|東帝汶獨立革命陣線]]、[[民主黨|民主黨]]、[[社會民主黨|社會民主黨]]、[[東帝汶社會民主協會|東帝汶社會民主協會]]。

=== 立法机关 ===
东帝汶的国家立法机关是[[一院制|一院制]]的[[国民议会_(东帝汶)|国民议会]]。议员数在52－65之间，由全民投票选举，任期5年。

== 地理 ==
{{Main|东帝汶地理}}

東帝汶位处[[东南亚|东南亚]]，位于[[印度尼西亚群岛|印度尼西亚群岛]]东部的[[小巽他群岛|小巽他群岛]]，东南临[[澳大利亚|澳大利亚]]。領土包括帝汶島東部和西部北海岸被印尼所包圍而孤立的[[飞地|飞地]][[歐庫西區|歐庫西區]]以及附近的[[阿陶羅島|阿陶羅島]]等。西部與印尼[[西帝汶|西帝汶]]相接。

境內多山，[[平原|平原]]、谷地屬[[熱帶草原氣候|熱帶草原氣候]]，全年高温，干湿季明显；其他地區为[[熱帶雨林氣候|熱帶雨林氣候]]，终年高温多雨。常见的[[自然灾害|自然灾害]]是[[洪水|洪水]]和山体[[滑坡|滑坡]]，还有[[地震|地震]]、[[海啸|海啸]]和[[热带气旋|热带气旋]]。

[[自然资源|自然资源]]有[[黄金|黄金]]、[[石油|石油]]、[[天然气|天然气]]、[[锰|锰]]、[[大理石|大理石]]。

由于农业还大量采用[[刀耕火种|刀耕火种]]的方式，导致森林砍伐和水土流失。

== 经济 ==
[[File:Dili_warehouse.jpg|thumb]]

1999年底，东帝汶约有70%的经济基础被反独立的民兵和印尼军队消耗。30万人逃到了西帝汶成为难民。此后三年间，5,000名维和部队和1,300名警察帮助对城市和农村地区进行了重建。到2005年底难民已经返回东帝汶或者在印尼定居。

东帝汶经济仍然面临巨大挑战，包括重建基础设施、加强公务员管理，解决大量年轻人的就业问题。近海的石油天然气开发极大的补充了政府的收入。天然气通过管道输送到澳大利亚。

2005年6月，国会批准了一个石油基金，作为所有石油收入的存储，并为东帝汶的后代保护石油财富。截至2010年10月，该基金已经有6.6亿美元资产。該國缺乏金融人才，因此選擇投資最保守的[[美國國債|美國國債]]，躲過[[2007年－2008年環球金融危機|2007年－2008年環球金融危機]]，在那幾年成為績效最好的[[主權基金|主權基金]]。

東帝汶的[[经济|经济]]结构（2001年）是[[服務業|服務業]]占57%、[[農業|農業]]25%、[[工業|工業]]17%，有[[石油|石油]]及[[天然气|天然气]]资源，但此二资源被澳洲企业掌握。2001年[[進口值|進口值]]达2億3,700萬美元；[[出口值|出口值]]800萬美元。主要進口機械產品。主要出口产品为[[咖啡|咖啡]]、[[橡膠|橡膠]]、[[紫檀木|紫檀木]]、原油、天然氣。

東帝汶是世界上极[[貧窮|貧窮]]落後的國家之一，大部分物資都要靠外國援助，雖然[[經濟|經濟]]正在嘗試进一步地发展，但目前的經濟状态并不良好。

== 人口与宗教 ==
{{Main article|東帝汶宗教}}
[[File:Klein-g20-V1_Ermera.jpg|200px]]

2011年7月有117萬。首都帝力是主要城市，人口大约16万人。其中東帝汶原住民（[[巴布亞|巴布亞]]與[[馬來族|馬來族]]或[[波利尼西亞|波利尼西亞]]的混血人種）佔78％，印尼人佔20％，亦有2％[[東帝汶華人|華人]]。約91.4％居民为[[羅馬天主教|羅馬天主教]]徒，[[基督教|基督教]][[新教|新教]]徒佔全國人口2.6％，1.7％是[[穆斯林|穆斯林]]，0.3％是[[印度教|印度教]]徒，0.1％为[[佛教|佛教]]徒。

1975年东帝汶独立时，以葡萄牙语为官方语言，印尼入侵东帝汶后，禁止使用葡萄牙文。2002年独立建国时，明定葡萄牙文与德顿文（Tetum）为官方语言，还有16种语言日常使用，国民每天还使用数十种方言。此外，印尼语和英语也是其中最常用的语言。

由于地处近赤道地区，气候炎热而潮湿，多发疾病。主要的传染病有[[腹泻|腹泻]]、[[甲肝|甲肝]]、[[伤寒|伤寒]]、[[基孔肯雅热|基孔肯雅热]]、[[登革热|登革热]]和[[疟疾|疟疾]]。

=== 足球 ===
[[東帝汶足球協會|東帝汶足協]]在2001年成立，在2002年加入[[亞洲足聯|亞洲足聯]]。2005年，東帝汶以198比1的壓倒性優勢，獲[[國際足協|國際足協]]成員協會投票通過，正式加入國際足联。
