'''东华大学'''（[[英文|英文]]：'''Donghua University'''，缩写：'''DHU'''，简称：'''东华'''），是一所位於[[中国|中国]][[上海市|上海市]]的[[公立大学|公立大学]]。是教育部直属、国家“[[211工程|211工程]]”重点建设的高等院校。前身為创建于1951年的'''华东纺织工学院'''，1985年改制为'''中国纺织大学'''，1999年更为現名。

該校是一所以[[工科|工]]为主，理、管、文等学科为辅的[[综合大学|综合大学]]。<ref >{{cite web|url=http://www.dhu.edu.cn/5943/list.htm|language= |title=学校介绍 |date=2017-03|accessdate= }}</ref>1960年，被[[中華人民共和國教育部|中華人民共和國教育部]]确定为[[重点大学|重点大学]]，是[[中国|中国]]首批具有[[博士|博士]]、[[硕士|硕士]]、[[学士|学士]]三级学位授予权的大学之一。

== 概觀 ==
校內設有12個專業學院及「旅遊系」，另有「體育部」、「國際文化交流學院」，國際合作辦學的「東華大學-拉薩爾國際設計」等學院。

擁有3個[[博士後|博士後]]流動站、24個博士學位授權點、48個碩士學位授權點、39個[[本科|本科]]專業，學科涉及[[工學|工學]]、[[理學|理學]]等九大學科門類。共有4個[[國家重點學科|國家重點學科]]，3個上海市重點學科，同時設有[[國家重點實驗室|國家重點實驗室]]、[[國家工程技術研究中心|國家工程技術研究中心]]、[[教育部重點實驗室|教育部重點實驗室]]、[[教育部工程研究中心|教育部工程研究中心]]，[[國家大學科技園|國家大學科技園]]。

在校學生2萬余名，教職工3300余名，其中[[工程院|工程院]]院士6名，有<光華工程科技獎>、<[[國家自然科學基金委員會|國家傑出青年科學基金]]>、<教育部跨世紀人才>和<全國回國留學人員先進個人>等榮譽稱號獲得者，[[長江學者|長江學者]]特聘教授、高級職稱教師近600名。

== 校史 ==
最初前身為[[民國|民國]]年間四所高校—[[私立上海紡織工業專科學校|私立上海紡織工業專科學校]]、[[私立中國紡織工程學院|私立中國紡織工程學院]]、[[文琦染織專科學校|文琦染織專科學校]]、[[誠孚紡織專科學校|誠孚紡織專科學校]]在1950年合併而成的[[私立上海紡織工學院|私立上海紡織工學院]]。

*1951年6月院系調整，[[交通大學|交通大學]]紡織系、[[上海工業專科學校|上海工業專科學校]]紡織科併入，組建成為'''華東紡織工學院'''。
*1952年7月，併入[[南通學院|南通學院]]紡織科、[[武漢|武漢]][[中南紡織專科學校|中南紡織專科學校]]。
*1953年7月，併入[[華東交通工業專科學校|華東交通工業專科學校]]機械系、[[蘇州|蘇州]][[蘇南工業專科學校|蘇南工業專科學校]]紡織科、[[四川|四川]][[樂山技藝專科學校|樂山技藝專科學校]]印染班。
*1956年7月，併入[[青島工業學院|青島工業學院]]紡織系。
*1972年4月，更名[[上海紡織工學院|上海紡織工學院]]。
*1980年8月，復名[[華東紡織工學院|華東紡織工學院]]。
*1985年9月，更名[[中國紡織大學|中國紡織大學]]。
*1994年11月，併入[[中國紡織總會管理幹部學院|中國紡織總會管理幹部學院]]。
*1999年8月，併入[[上海紡織高等專科學校|上海紡織高等專科學校]]，更名[[東華大學|東華大學]]。
*1999年9月1日，併入[[國家輕工業局|國家輕工業局]][[上海玻璃搪瓷研究所|上海玻璃搪瓷研究所]]。

== 教學 ==
近年來，連續兩年有論文入選「[[全國百篇優秀博士學位論文|全國百篇優秀博士學位論文]]」，在[[ACM國際大學生程式設計競賽|ACM國際大學生程式設計競賽]]、美國大學生數模競賽、挑戰杯中國大學生創業計畫競賽、全國大學生電子設計競賽、[[中央電視臺|中央電視臺]]CCTV杯全國模特大賽、[[上海市|上海市]]發明創造三槍杯競賽等重大競賽中多次獲得特等獎、金獎或冠軍。

学校有[[中国科学院院士|中国科学院院士]][[张友尚|张友尚]]，[[中国工程院院士|中国工程院院士]][[郁铭芳|郁铭芳]]、[[周翔|周翔]]、[[王浩|王浩]]、[[周勤之|周勤之]]、[[季国标|季国标]]、[[俞建勇|俞建勇]]。

== 成果 ==
由該校的[[紡織|紡織]]工程（在全[[中國|中國]][[高等院校|高等院校]]同類[[一級學科|一級學科]]排名第一）、[[時裝設計|時裝設計]]與[[工程|工程]]、[[材料學|材料學]]、紡織[[化學|化學]]與染整工程等[[國家重點學科|國家重點學科]]，發展傳統優勢學科和新興特色學科。以[[國家重點實驗室|國家重點實驗室]]、[[國家大學科技園|國家大學科技園]]等諸多科技創新基地為依託，為國家科技、經濟發展和社會進步服務。

近年來，大批科研成果廣泛應用於[[航太|航太]]、[[軍事|軍事]]、[[建築|建築]]、[[環保|環保]]、[[材料學|材料學]]等領域。有多項科研成果獲得「國家科技進步獎」、「國家發明獎」及「省部級科技進步獎」，連續三年有「艙外航太服外層防護材料的研製」等科研成果入選「[[中國高校十大科技進展|中國高校十大科技進展]]」，連續兩年獲「國家科技進步二等獎」；學校申請和授權的專利數也不斷提升，於2004年分列全中國高校第15和14位。
