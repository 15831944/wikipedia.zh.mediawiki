《'''东山学堂'''》，2013年中国大陆电视剧，讲述[[毛泽东|毛泽东]]1909年在湘乡市[[东山学校|东山学堂]]的读书生涯。<ref>{{cite web |url=http://news.hexun.com/2013-07-11/156031530.html |title=青年毛泽东走出韶山第一站——东山学堂 |accessdate=2013 |publisher=和讯网 |work= |language=zh-hans}}</ref>2013年12月24日在[[央視八套|央視八套]]上午檔播出。

==剧情==
该剧讲述[[中国共产党|中国共产党]]领导人[[毛泽东|毛泽东]]少年时期在湘乡市东山学堂的求学生涯。<ref>{{cite web |url=http://dianshiju.cntv.cn/20120926/100105.shtml |title=电视剧《东山学堂》简介 |accessdate=2012 |publisher=中国网络电视台 |work= |language=zh-hans}}</ref>
