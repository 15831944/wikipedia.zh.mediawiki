'''上海广播电视台東方衛視频道'''（简称'''東方衛視'''，{{lang-en|Dragon Television}}，2003年10月23日前为'''上海衛視'''）是[[上海广播电视台|上海广播电视台（上海文化广播影视集团）]]旗下的一个省級衛星電視频道。東方衛視国内版本的信號已覆蓋了中國大陆百分百省市，覆盖范围位居中国省级卫视第一，同時東方衛視国际频道在[[北美|北美]]、[[日本|日本]]、[[澳洲|澳洲]]、[[歐洲|歐洲]]及[[香港|香港]][[澳門|澳門]]等國家和地區落地。2009年9月28日起，該頻道国内版本实现標高清同播。東方衛視也是中国最具品牌价值和影响力的综合频道之一，被业内权威机构评为“最具影响力省级卫视”<ref name=smg/>。因東方衛視紅圓內星的臺標酷似一個熟透了的帶葉番茄，故被一些觀眾戲稱為番茄台。

東方衛視以“大台气象、都市气质、年轻气息”为频道定位，以城市年轻人群为收视对象，节目涵盖新闻、影视剧、综艺节目等。其中新闻栏目数量和播出时长方面均位居中国省级卫视第一，而该频道播出的电视剧、综艺节目也深受观众好评<ref name=smg>{{cite web |url =https://www.smg.cn/review/channel/channel_5/index.html|title = 東方衛視频道简介|website = SMG网站}}</ref>。

==发展历史==
===上海衛視时期===
1998年10月1日，上海衛視开播，频道全称“上海电视台卫星频道”，也是[[上海电视台|上海电视台]]的第三套节目，此为東方衛視的前身，该频道也是中国最晚一批开播的省级卫星频道。上海衛視开播也成为了[[中共上海市委|中共上海市委]]和[[上海市人民政府|上海市人民政府]]当年文化实事之一。频道开播初期以编播为主，汇总上海各电视台、各影视频道和上海各影视机构的优秀节目，播放“具有上海特色、代表上海影视水平的精品节目”。制播地点位于浦东东视大厦，设有“新上海精品电视”等7档以编为主的综艺类栏目和纪录片等。还辟有“财经新闻”、“卫视新闻”、“星期视点”等栏目<ref>{{cite web |url =http://www.shtong.gov.cn/newsite/node2/node2245/node4510/node13115/node63849/userobject1ai13255.html|title = 上海广播电视志 >> 补记 >> 一、广电建设 
|website =上海市地方志办公室}}</ref><ref name=sina>{{cite web |url =http://finance.sina.com.cn/chanjing/b/20090831/23536687705.shtml|title = 东方卫视再“战”江湖 |website =《传媒》杂志|date=2009-08-31}}</ref>。频道开播时，台标为透明银色[[白玉兰|白玉兰]]台标，右边标注透明综艺体“上海卫视”，字幅与台标相同大小。2002年，该频道改为上海电视台的第七套节目，并微调台标，台标字体进行了缩小。至更名为東方衛視前，台标颜色为灰色透明。

上海衛視同时还开设专门的国际版本（现东方卫视国际频道），并于2002年1月1日，上海衛視落地[[日本|日本]]（后于2009年退出落地）。2003年1月16日，经原[[国家广播电影电视总局|国家广播电影电视总局]]批准，上海电视台与[[电视广播（国际）有限公司|电视广播（国际）有限公司]]签订了《上海电视台卫星频道落地澳大利亚的协议》，至此上海衛視国际版正式覆盖[[澳大利亚|澳大利亚]]<ref>{{cite web |url =http://ent.sina.com.cn/v/2003-01-17/1605126580.html|title = 上海卫视落地澳洲签约仪式隆重举行|website =新浪网|date=2003-01-17}}</ref>。

上海卫视改版前，虽然全国覆盖率已列省级卫视的前10位，但因上海对待卫视的态度一度比较保守，曾对外设置壁垒，不让外省卫视落地上海，对内重视不够，投入不足，仅依靠播出其他频道节目，加上频道采取低成本运作，最终导致上海卫视收视率和影响力较低，在全国市场收视排名第13位，仅占全国1.5％的份额<ref name=dfws2003>{{cite web |url =http://xwjz.eastday.com/eastday/xwjz/node8898/node8900/userobject1ai41574.html|title =东方卫视：打造上海“软力量”|website = 新闻记者|accessdate=2017-10-18}}</ref>。2002年，上海卫视广告收入仅为5600万元人民币<ref name=sina/>。

===東方衛視时期===
2003年初，为了使东方卫视真正覆盖全国，上海市制定了“全面开放，对等落地，规范管理，市场运作”16字方针。2003年7月，“上海东方卫视传媒有限公司”注册成立，下设新闻中心、节目中心、营运中心、总编室等部门，频道实行公司化核算。此外，东方卫视更享有广告自主经营权，同时享有独立购买、制作、发行电视剧的权利，不再受SMG统一购买电视剧的限制<ref name=dfws2003/>。

2003年10月23日，经国家广电总局批准，上海卫视正式更名为东方卫视，并进行频道改版<ref name=sina/>。频道改版后，推出了《看东方》、《城际连线》、《环球新闻站》、《直播上海》、《今日新观察》等节目<ref>{{cite web |url =http://www.people.com.cn/GB/14677/14737/22037/2146479.html|title =上海卫视更名东方卫视 全新开播看什么|website = 解放日报|accessdate=2013-10-22}}</ref>，同年12月31日还举行了《[[上海新天地|上海新天地]]新年倒计时晚会》现场直播<ref>{{cite web |url =http://media.people.com.cn/GB/40606/4039638.html|title =成长中的“东方巨人”——东方卫视探秘|website = 人民网|accessdate=2006-01-18}}</ref>。

2005年底，上海文广集团开始对新闻、娱乐资源进行整合，原上视新闻综合频道、东视新闻娱乐频道和东方卫视的全部新闻资源合并组建SMG电视新闻中心（今[[SMG融媒体中心|SMG融媒体中心]]）。2006年元旦频道改版，其中《看东方》延长至150分钟，成为当时中国大陆最长的新闻直播节目；原《城际连线》和《环球新闻站》合并为《[[东方新闻|东方新闻]]》。同时推出了一系列综艺节目<ref>{{cite web |url =http://media.people.com.cn/GB/40641/5257433.html|title =东方卫视两翼起舞|website = 人民网|accessdate=2007-01-08}}</ref>。

2009年6月18日，频道再次改版。在改版前，俄罗斯歌手维塔斯与[[韦唯|韦唯]]合作演唱了东方卫视台歌《风从东方来》<ref>{{cite web |url =http://ent.sina.com.cn/v/m/p/2009-05-05/12392503761.shtml|title =东方卫视全新改版 VITAS韦唯合唱主题歌|website = 新浪娱乐|accessdate=2009-05-05}}</ref>。同年9月28日起，该频道实现高、标清同步播出<ref>{{cite web |url =http://news.sina.com.cn/c/2009-09-30/082216383690s.shtml|title =深圳卫视有了高清版|website = 南方日报|accessdate=2009-05-30}}</ref>。

[[2010年上海世博会|2010年上海世博会]]期间，东方卫视与SMG其他频道一起，成为其合作媒体，并推出了与世博会有关的特别节目（如《欢聚2010》）。

2011年1月1日，该频道再次改版。除原有的九点档综艺节目外，东方卫视在晚间十点档开辟第二线综艺节目时段，新增《谁能百里挑一》《谁是大人物》《加油！老爸》《星厨大战》等多档新节目<ref>{{cite web |url =http://news.cntv.cn/20101214/104416.shtml|title =东方卫视将全新改版|website = 新闻晨报|accessdate=2010-12-14}}</ref>。同年2月13日起推出周日体育系列节目，包括《东方体育周刊》、《五星足球》、《篮球风云》和《五星周末赛场》<ref>{{cite web |url =http://ent.ifeng.com/tv/news/mainland/detail_2011_01/20/4363775_0.shtml|title =五星体育联手东方卫视推出卫视体育版|website = 凤凰网娱乐 |accessdate=2017-10-18}}</ref>。

2011年6月20日，东方卫视新闻演播室改造工程完工并投入使用。改造后的演播室面积约500平方米，以DLP大屏加PDP屏为背景，设演播区域6个，功能集全景类新闻、嘉宾访谈、实时连线报道等互动形式为一体。

2014年3月15日，原有东方卫视、[[上海娱乐频道|娱乐频道]]、[[星尚频道|星尚频道]]、[[艺术人文频道|艺术人文频道]]、七彩戏剧频道、生活时尚频道合并组建成全新的“东方卫视中心”，原有频道版面、节目带，正式转入东方卫视中心的运营系统。同时东方卫视的重要节目会在同中心旗下地面频道的电视荧屏右上角角标进行宣传，角标镜面与东方卫视的一致<ref>[http://www.ttacc.net/a/news/2014/1216/32702.html 深度揭秘东方卫视中心]</ref>。同时取消与[[上视新闻综合频道|上视新闻综合频道]]并机直播的特别新闻报道。

2017年10月1日，为贯彻落实《[[中华人民共和国国歌法|中华人民共和国国歌法]]》相关规定，东方卫视（国内版）会在国庆节、国际劳动节等重要的国家法定节日、纪念日期间须在上午10点整播放国歌，国歌片带与央视播放版本一致<ref>{{cite web |url =http://news.ifeng.com/a/20171001/52251501_0.shtml |title = 广电总局：电台电视台应在重要法定节日播放国歌 |website = 凤凰资讯 }}</ref>。

2019年1月1日，东方卫视再次改版，在新闻、文化、综艺、影视方面进行革新。其中新闻播出量将从2018年的的4小时扩至5小时，同时在每周一至周四晚间9:30开设公益文化节目带，以及在每天17:50开设日播传统文化类节目《诗书画》<ref>{{cite web | title = 东方卫视进行全新改版“三箭齐发”凸显“梦想的力量” | publisher = 东方网|author=通讯员陈路、记者刘轶琳 | url= http://sh.eastday.com/m/20181228/u1ai12118815.html| date = 2018-12-28 | language = zh }}</ref>。

== 频道文化 ==
===台标===
东方卫视的台标由[[红色|红色]]圆球和[[白色|白色]]五角星构成。主设计师为胡羿如。东方卫视台标的设计灵感来源于[[水果|水果]]鲜[[橙|橙]]，意念是“[[维生素|维生素]]有ABC，知天下看东方TV”。起初台标为橙色，但由于上海已有[[地面電視|地面電視]]频道使用橙色作为频道识别颜色，所以经过几次修改，最后使用了目前的红色。该台标也与上海东方电视台的台标（旭日加海鸥）有传承、延伸的意味。根据官方介绍，鲜橙可以带给人扑面而来的新鲜、丰润和亮丽之感，与新开播的卫视的理念契合。新鲜有“每天播报最新鲜的新闻”之意；丰润有“影视剧多样而丰富”之意；而亮丽则有“每天为观众带来时尚、欢乐的娱乐资讯和流行话题”之意。至于台标内的五角星，则代表了胜利与美好，其中白色也是西方国家的标志性色彩。红色与白色的交融代表了中西方文化的融合与交流。该台标是由由十多家知名设计公司参与竞标，经过两个多月的甄选产生的<ref>{{cite web |url =http://ent.sina.com.cn/2003-09-23/0211204890.html|title =东方卫视亮台标|website = 新闻晨报|date=2003-09-23}}</ref><ref>{{cite web |url =http://news.ifeng.com/society/2/detail_2009_08/29/883760_0.shtml|title =网友看卫视台标起绰号：湖南芒果台 东方番茄台|website = 法制晚报|date=2009-08-29}}</ref>。

该台标使用后，由于最终的台标与一个熟透了的带叶[[番茄|番茄]]酷似，所以被一些观众和[[网友|网友]]戏称为番茄台。电视台方面也顺势使用了番茄台的名称。例如在东方卫视官方[[新浪微博|新浪微博]]帐号就使用了“东方卫视番茄台”的名称。

2016年5月25日凌晨1点开始，东方卫视（国内版）标清节目由原来上下信箱或左右切边改为16:9挤压全屏播出（台标保留4:3尺寸），至此东方卫视开始步入宽屏时代。同样国际版虽然依旧4:3播出（实则上东方卫视国际频道并不在上海SMG总部负责制作），但与国内版同步新闻节目也开始挤压播出。2016年5月28日，东方卫视新闻节目包装改版，结束了切边时代。2016年7月19日，东方卫视（国内版）标清频道台标改为16:9尺寸，与高清频道一样，仅通过屏幕右上角的高清字样以及转播CCTV1信号所对应的线路作区别（即高清转播CCTV1高清，标清转播CCTV1标清；不过SMG其他地面频道高、标清转播CCTV节目信号统一为高清信号）。

2017年1月27日，东方卫视（国内版）高、标清信号转播[[2017年中央电视台春节联欢晚会|2017年中央电视台春节联欢晚会]]期间统一为高清信号，在转播结束之后转播CCTV1信号恢复对应线路转播，即高清转播CCTV1高清，标清转播CCTV1标清。

2018年12月31日23时22分左右，东方卫视更换新台标。新台标在原有台标的基础上增加了立体效果。东方卫视（国际版）亦同时更换台标和压缩尺寸为16:9，但大多节目依然4:3压缩播出。

===频道口号===
2009年6月18日东方卫视改版之时，该频道启用“风从东方来”口号。2011年5月起进行全新的改版，并启用全新的频道口号“梦想的力量，你我同在”<ref>{{cite web |url =http://ent.qq.com/a/20110721/000295.htm|title =《东方直播室》召开研讨会 打造骆新个人品牌|website =腾讯娱乐 |accessdate=2011-07-21}}</ref>，2015年简化为“梦想的力量”。

===台歌===
频道曾先后推出过两首台歌，一个是《番茄红了》，另一个是《风从东方来》，皆于2009年推出<ref>[http://ent.sina.com.cn/v/m/2009-01-20/12082350803.shtml 《番茄红了》贺新春 东方卫视春晚推出台歌拜年]</ref>，其中《风从东方来》为原[[上海东方电视台|上海东方电视台]]台歌。

== 节目 ==

*上午时段：《看东方》（周六日播出周播新闻节目）、《潮童天下》（精编）、《东方看大剧》、重播电视剧或综艺节目

*凌晨时段：重播综艺节目或展播[[中国梦|中国梦]]主题纪录片
