'''东兴市'''位于[[中华人民共和国|中华人民共和国]][[广西壮族自治区|广西壮族自治区]]南部，是[[防城港市|防城港市]]代管的一个[[县级市|县级市]]。市政府驻[[东兴镇|东兴镇]]，面积540.7平方公里，2007年人口为11万。

== 地理 ==
东兴市地处[[北仑河|北仑河]]流入[[北部湾|北部湾]]河口，东南濒临[[北部湾|北部湾]]，东部和北部与[[防城港|防城港]]市[[防城区|防城区]]相邻，西南与[[越南|越南]]接壤，与[[越南|越南]][[芒街市|芒街市]]隔[[北仑河|北仑河]]相望；背靠大西南，面向[[东南亚|东南亚]]，也是[[中国|中国]][[少数民族|少数民族]][[京族|京族]]的唯一聚居地。

境内地势平坦，有少量低矮山丘，陆地边境线长27.8公里，海岸线长54公里，降雨量充沛，年均气温22.4℃，属亚热带海洋性气候。

== 历史 ==
[[File:Dong_xing_Boundary_marker.jpg|thumb]]
清光绪十三年（1887年）置辖防城县，东兴境为防城县辖地。

1958年12月1日，撤消防城县，并入东兴各族自治县，县治东兴镇。1978年12月15日，东兴各族自治县县治迁往防城镇，改名防城各族自治县。1996年4月29日，经[[国务院|国务院]]批准，设立东兴市（县级），辖东兴、江平、马路等3个镇，市政府驻东兴镇，属防城港市辖。

== 行政区划 ==
2000年，东兴市下辖东兴镇、江平镇、马路镇3个镇。

*东兴镇下辖：东郊、深沟、花溪、中山、七星、公园、北郊7个社区；竹山、松柏、楠木山、长湖、大田、江那、河洲、牛轭岭8个行政村。
*江平镇下辖：城南、城北2个社区；万尾、江龙、吒祖、长山、潭吉、横隘、班埃、交东、山心、黄竹、榕树头、思勒、那漏、贵明、巫头15个行政村。
*马路镇下辖：兴桂社区；马路、大桥、平丰、竹围、大旺、冲榄、吊应、北联8个行政村。 

== 交通 ==
东兴市辖区内有公路59条，总里程285.597公里。东兴市有中越公路大桥与越南联通。

[[东兴口岸|东兴口岸]]是[[中国|中国]]和[[越南|越南]]唯一海陆相连，进行[[旅游|旅游]]和[[商品|商品]][[物资|物资]][[贸易|贸易]]的国家一类[[口岸|口岸]]。
