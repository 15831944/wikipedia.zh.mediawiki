'''东营市'''是[[中华人民共和国|中华人民共和国]][[山东省|山东省]]下辖的[[地级市|地级市]]，位于山东省北部，毗邻[[渤海|渤海]]，[[黄河|黄河]]流经这里进入渤海。东营市为新兴[[石油工业|石油工业]]城市，属于[[黄河三角洲|黄河三角洲]]地区、[[山东半岛城市群|山东半岛城市群]]。东营市还是[[国家卫生城市|国家卫生城市]]、[[全国文明城市|全国文明城市]]。

== 历史 ==
1961年4月，华北石油勘探处在原广饶县辛店公社东营村附近打成第一口勘探井——华八井，获日产8.1吨工业油流；1962年9月23日，在东营构造上打的营2井获日产555吨油流，为当时全国日产量最高的油井；为纪念打出这一高产油井之日，始称“923厂”。

1965年3月，为支援石油会战，惠民地委决定成立中共惠民地区东营工作委员会和东营办事处，并组建一批为矿区服务的商业、金融、邮电等机构。1971年6月11日，923厂更名为[[胜利油田|胜利油田]]。

1982年11月10日，划原惠民地区的垦利、利津2县，广饶县的4个公社，沾化县的4个公社，博兴县的1个公社和3个大队，成立地级东营市。1983年8月30日，广饶县划归东营市。

2016年，垦利县撤县建区。

== 地理 ==
东营市地处山东北部的[[黄河三角洲|黄河三角洲]]地区，[[黄河|黄河]]在东营市境内流入渤海。东营市坐标为北纬36°55′～38°10′，东经118°07′～119°10′。东、北临渤海，西与滨州市毗邻，南与淄博市、潍坊市接壤。南北最大纵距123公里，东西最大横距74公里，面积7923平方公里。

黄河东营段上起滨州界，自西南向东北贯穿东营市全境，在垦利东北部注入渤海，长138公里。黄河水径流量年际变化大，年内分配不均，含沙量大。

== 经济 ==
东营市为[[胜利油田|胜利油田]]所在地，以石油为主的工业包括石油开采、制造和化工业初具规模，服务业、信息技术、中小型民营企业等发展水平较低。[[石油化工|石油化工]]、[[轮胎|轮胎]]制造、[[纺织|纺织]]已形成完整的产业链，[[铜|铜]]冶炼、[[食盐|食盐]]生产、[[造纸|造纸]]、机械、建筑材料、[[食用油|食用油]]、[[饲料|饲料]]生产在[[中国内地|中国内地]]占有较大比重。

2010年，全市[[地区生产总值|地区生产总值]]2,359.94亿元（348.61亿美元），居内地城市（含直辖市）[[中国大陆国内生产总值过千亿元地级以上城市列表|第45位]]；其中工业增加值达到1,612.02亿元（238.13亿美元），占GDP比重达到68.31%；人均GDP为116,448元（17,202美元），居内地城市的[[中国大陆人均国内生产总值过万美元地级以上城市列表|第4位]]。地方财政总收入104.88亿元，[[城镇居民|城镇居民]][[人均可支配收入|人均可支配收入]]23,796元（3,515美元），[[农村居民人均纯收入|农村居民人均纯收入]]7,938元（1,173美元）<ref name="2010年社会与经济指标">2010年东营市社会和经济发展指标来自[http://www.dytjj.com/dytj/Columns/19374178653.shtml 《东营统计年鉴2011》] {{webarchive|url=https://archive.is/20130425172716/http://www.dytjj.com/dytj/Columns/19374178653.shtml |date=2013-04-25 }}</ref>。

辖区内全年[[原油|原油]]产量2,743.52万吨，占内地当年产量的13.51%；[[天然气|天然气]]产量5.08亿立方米。主要工业品中，[[布|布]]产量达到7.69亿米，占内地当年产量的0.96%；精炼[[铜|铜]]产量20.69万吨，占内地当年产量的4.51%；食用植物油产量44.22万吨，占内地当年产量的1.14%；[[轮胎|轮胎]]产量4,135.09万条<ref name="2010年社会与经济指标"/><ref name=全国数据2010>东营市主要产品产量占全国的比重：2010年度全国数据出自[http://www.stats.gov.cn/tjsj/ndsj/2011/indexch.htm 《中国统计年鉴2011》]    {{Wayback|url=http://www.stats.gov.cn/tjsj/ndsj/2011/indexch.htm |date=20171225003852 }}，其中内地全年原油产量20,301万吨，东营占13.51%；内地原盐产量7,037.76万吨，东营占1.46%；内地食用植物油产量3,878.54万吨，东营占1.14%；内地精炼铜产量458.65万吨，东营占4.51%。</ref>。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为203.53万人<ref>{{cite book |author=东营市统计局|title=《东营市2010年第六次全国人口普查主要数据公报》}}</ref>，与[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加了24.23万人。增长13.51%，年均增长1.28%。其中，男性为103.08万人，占总人口的50.65%；女性为100.45万人，占总人口的49.35%，总人口性别比（以女性为100）为102.62。0－14岁的人口为31.69万人，占15.57%，15－64岁的人口为153.70万人，占75.52%；65岁及以上的人口为18.14万人，占8.91%。

== 交通 ==
=== 城市交通 ===
东营市内道路开阔，市区道路两旁有较为宽广的[[人行道|人行道]]与[[公共绿地|公共绿地]]，临近建筑的人行步道宽度在5-7米，而且多高出地面20-50厘米，其中均规划有免费的公共停车场，市内泊车较为方便。东营市公共交通系统为公交车，采取投币及[[IC卡|IC卡]]付款，较为方便。

=== 高速公路 ===
由山东东部[[威海|威海]]至[[乌海|乌海]]高速公路在东营市南广饶县与[[东营|东营]]至[[青州|青州]]高速公路（[[荣乌高速公路|威乌高速]]/[[长深高速|长深高速]]/沿海高速共用路段）汇合，纵穿东营市中心地区，到达北部的垦利县城，在黄河北岸分为威乌高速公路及东港高速公路，在东营境内的高速公路从北部的东营港纵穿到南部并与[[济南|济南]]至[[青岛|青岛]]的[[济青高速公路|济青高速公路]]连接。通过高速公路在5小时内可以到达山东半岛的大部分城市。

=== 铁路 ===
[[东营站|东营站]]为[[淄东铁路|淄东铁路]]上的三等站，也是该铁路的终点站。横贯东营的[[德大铁路|德大铁路]]已于2015年建成通车，在东营设有[[东营南站|东营南站]]等三个中间站，并与淄东铁路设有联络线。目前，东营开行有往返北京、济南、滨州等地的客运列车。

=== 航空 ===
{{main|东营胜利机场}}
[[File:Linbo6619.JPG|thumb]]
[[东营胜利机场|东营胜利机场]]位于东营市区的东北部[[永安镇_(东营市)|永安镇]]境内，距离市区9公里左右，距离东营市市中心25公里左右。机场自1985年起归属胜利石油管理局及[[济南军区|济南军区]]，自2001年起管理局及军区不再使用并开始服务旅客。目前，机场开通了至北京、上海、深圳、杭州、西安、成都、郑州、武汉、天津、宁波、大连、佛山等地的往返航班<ref>{{cite web|title=东营胜利机场航班时刻表|url=http://dyjt.gov.cn/contents/83/14124.html|website=东营市交通运输局|accessdate=2017-09-05|date=2017-05-09|deadurl=no|archiveurl=https://web.archive.org/web/20170905095712/http://dyjt.gov.cn/contents/83/14124.html|archivedate=2017-09-05}}</ref>。

2016年，[[中国商飞|中国商飞]]的东营试飞基地落成，主要负责[[C919|C919]]和[[ARJ21|ARJ21]]项目的试飞工作。{{Fact|另外东营机场内有小型飞机租赁公司，旅客可以在这里学习飞行或者租赁小型飞机。|time=2013-02-11T15:21:21+00:00}}

东营市区有182路公共汽车前往东营胜利机场，在市公交公司北和胜利机场之间来回行驶。东营市区每日有10班以上前往[[济南遥墙国际机场|济南遥墙国际机场]]的公共汽车，7班前往[[青岛流亭国际机场|青岛流亭国际机场]]的汽车，另有在航班起飞前开往机场的机场巴士。

[[垦利县|垦利县]][[黄河口镇|黄河口镇]]另有一座农业机场，跑道较短（仅适用运5型飞机起降）。主要用于防治蝗虫，观光，以及运输农业物资而用。

=== 港口 ===
[[东营港|东营港]]是东营市的主要港口，主要承担本区物资的海上运输任务，同时还开有客运航线。2013年，东营市水路客运量66万人，水路货运量177万吨，港口货物吞吐量达1420万吨<ref>{{Cite web|url = http://www.dystjj.gov.cn/dytj/pic/2014/%E7%BB%9F%E8%AE%A1%E5%B9%B4%E9%89%B4-2014_9_%E5%BB%BA%E7%AD%91%E4%BA%A4%E9%80%9A%E9%82%AE%E7%94%B5%E6%97%85%E6%B8%B8.pdf|title = 东营统计年鉴-2014|accessdate = 2016-02-13|author = 东营市统计局|date = 2015-05-12|publisher = 东营统计信息网|deadurl = yes|archiveurl = https://web.archive.org/web/20161009153447/http://www.dystjj.gov.cn/dytj/pic/2014/%E7%BB%9F%E8%AE%A1%E5%B9%B4%E9%89%B4-2014_9_%E5%BB%BA%E7%AD%91%E4%BA%A4%E9%80%9A%E9%82%AE%E7%94%B5%E6%97%85%E6%B8%B8.pdf|archivedate = 2016-10-09}}</ref>。2016年货物吞吐量达4300万吨。

此外，东营市还有一些渔港，例如广利渔港，红光渔港。

== 环境保护 ==
由于开发无度，东营市的生态环境质量相比过去已严重退化。包括空气质量、饮用水源等污染较为严重，也是中国冬季雾霾最为严重的城市之一。市区人为的光污染（夜间）、扬尘（例如建筑地区不对渣土等废料采取覆盖等措施）现象等较为普遍。

== 社会文化 ==
东营区的牛庄镇是山东吕剧的发源地，因此东营又被称为“吕剧之乡”。六户镇民间的“打老虎”戏也十分具有地方特色。

=== 新闻媒体  ===
东营市由于是胜利油田所在地，所以同时有两套的新闻媒体：东营市市属的[[东营日报|东营日报]]、[[东营电视台|东营电视台]]；而胜利油田的[[胜利日报|胜利日报]]、 [[胜利油田电视台|胜利油田电视台]]，主要面向胜利油田职工和家属。

=== 医疗 ===
市内主要医院有位于西城中心城区的胜利油田中心医院、东营区人民医院、鸿港医院，位于东城的东营市人民医院、东营市儿童医院，位于八分场的胜利医院。

=== 科技 ===
胜利油田的许多科研单位有一定的科技基础设施和发展能力，根据2007年6月公布的全球高性能计算排行榜[[TOP500|TOP500]]，胜利油田物探研究院的高性能计算系统（[[超级计算机|超级计算机]]），以每秒18.60万亿次[[浮点运算|浮点运算]]的实测LINPACK峰值位列全球第43，成为当时中国大陆运算速度与信息处理能力最高的计算系统<ref>{{cite web|title=Top500 List - June 2007|url=https://www.top500.org/list/2007/06/?page=1|website=Top500|accessdate=2017-09-05|deadurl=no|archiveurl=https://web.archive.org/web/20170905141014/https://www.top500.org/list/2007/06/?page=1|archivedate=2017-09-05}}</ref>。

=== 酒店 ===
由于与各大城市公务往来密切的原因之一东营市区有多座豪华饭店及快捷旅舍。其中较为著名的有：[[东胜大厦|东胜大厦]]、[[蓝海国际大酒店|蓝海国际大酒店]]、[[大明大厦|大明大厦]]、[[新悦大酒店|新悦大酒店]]、[[黄河饭店|黄河饭店]]、[[如家酒店|如家酒店]]。拥有高空旋转餐厅2家，分别是位于胜利电视塔的胜利电视塔观光层及济南路的新悦饭店。此外还有类似“胜利宾馆”及“东营宾馆”的园林饭店。广饶县比较著名的有[[宇通尊悦大酒店|宇通尊悦大酒店]]、[[广饶蓝海大饭店|广饶蓝海大饭店]]、[[广饶金岭国际大酒店|广饶金岭国际大酒店]]、[[孙武湖温泉度假酒店|孙武湖温泉度假酒店]]。而垦利、利津县的住宿饭店较少，但其距离东营市区的车程都不超过1个小时。因此较多旅客选择在东营市区的饭店居住。

=== 商业 ===
东营市的商业较发达，商业机构众多。市区的济南路为最主要的商业地段，集中了市区内的重要大型商业机构。自济南路西段往东依次有东营市百货大楼、利群商厦、供销商场、商业大厦、振华商厦、银座商厦。青岛路则集中了许多的餐饮机构，夜生活丰富。

=== 地方特产 ===
东营市紧临渤海，又是黄河入海之地，物产丰富。地方特产有：[[黄河刀鱼|黄河刀鱼]]、黄河口大米、中华绒螯蟹、渤海湾虾皮、渤海大对虾、[[广饶肴驴肉|广饶肴驴肉]]、河口冬枣、中国毛笔四大名笔之一的[[齐笔|齐笔]]、[[佛头黑陶|佛头黑陶]]等。

=== 美食===
东营的饮食因为受黄河与渤海的共同影响，逐渐形成了有自己地方特色的美食，典型代表有：[[广饶肴驴肉|广饶肴驴肉]]、[[利津水煎包|利津水煎包]]、北岭丸子、孤岛鲜鱼汤等。
