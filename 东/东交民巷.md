'''东交民巷'''，原名'''东江米巷'''，是[[北京市|北京市]][[东城区_(北京市)|东城区]]的一条[[胡同|胡同]]，西起[[广场东侧路|广场东侧路]]，东至[[崇文门内大街|崇文门内大街]]，全长1552[[千米|千米]]（与[[东长安街|东长安街]]平行并大致等长），是[[老北京|老北京]]最长的一条胡同。现已改为道路。

== 历史 ==
[[元朝|元朝]]时，[[大都城|大都城]]外，东交民巷和广场西侧的[[西交民巷|西交民巷]]是连在一起的一条胡同，名叫“江米巷”。当时这条胡同有元代控制[[漕运|漕运]]米粮进京的税务所和[[海关|海关]]，因而成为南粮北运的咽喉要地，故得名“江米巷”。[[明代|明代]]时城牆南移並修建[[棋盘街|棋盘街]]，将原来的江米巷截断成为东江米巷和西江米巷。在东江米巷设有[[六部|六部]]中的[[礼部|礼部]]以及[[鸿胪寺|鸿胪寺]]和[[会同馆|会同馆]]，但主要只接待来自[[越南|安南]]、[[蒙古|蒙古]]、[[朝鲜|朝鲜]]、[[缅甸|缅甸]]等四个[[藩属国|藩属国]]的使节，因此会同馆又被称作“四夷馆”。到了[[清代|清代]]，会同馆改名[[四译馆|四译馆]]，并修改政策，只允许外国使节在这里居住四十天。

1860年[[第二次鸦片战争|第二次鸦片战争]][[清軍|清軍]]战败后，根据清政府与[[大英帝国|英]]、[[法蘭西第二帝国|法]]、[[美国|美]]、[[俄罗斯帝国|俄]]签订的《[[天津条约|天津条约]]》中相关条款规定，1861年3月英国[[公使|公使]]正式入住东江米巷的[[淳亲王府|淳亲王府]]（当时名为[[梁公府|梁公府]]，系[[康熙皇帝|康熙皇帝]]第七子淳王[[允祐|允祐]]的府邸）；法国公使正式入住[[安郡王府|安郡王府]]（当时名为[[纯公府|纯公府]]，系[[努尔哈赤|努尔哈赤]]之孙安郡王岳乐的府邸）；美国公使进驻[[美国公民|美国公民]]Dr S.SWilliam位于东江米巷的私宅；俄国公使则入住清初在这里修建的[[东正教|东正教]]教堂俄罗斯馆。随后各国公使馆均选择东交民巷一带作为馆址，到1900年[[义和团运动|义和团运动]]之前这里有法国、[[大日本帝国|日本]]、美国、[[德意志帝国|德国]]、[[比利时|比利时]]、[[荷兰|荷兰]]等多国使馆。义和团运动爆发后，这里因为洋人糜集而被作为攻击的重点，曾有[[童谣|童谣]]念道“吃麵不搁醋，炮打[[西什库|西什库]]；吃麵不搁酱，炮打交民巷”，前者指的是位于北京西皇城根的[[西什库教堂|西什库教堂]]，后者即指东交民巷。

1900年义和团事变之后，根据《[[辛丑条约|辛丑条约]]》的规定，东江米巷改名Legation Street（使馆街），其在中方绘制的[[地图|地图]]中则正式更名为东交民巷，成为由各个使馆自行管理的使馆区；清政府在这条街上的衙署，仅保留了吏部、户部、礼部和[[宗人府|宗人府]]，其余尽数迁出。随后在这里出现了英国[[汇丰银行|汇丰银行]]、[[麦加利银行|麦加利银行]]，俄国[[华俄道胜银行|华俄道胜银行]]，日本的[[横滨正金银行|横滨正金银行]]，德国[[德华银行|德华银行]]，法国[[东方汇理银行|东方汇理银行]]等外资银行，还开办了法国[[邮局|邮局]]、[[医院|医院]]等设施，并出现了大量西式建筑。这块使馆区在[[辛亥革命|辛亥革命]]后一直保留。直到1937年[[抗日战争|抗日战争]]爆发后，除德国、意大利等[[轴心国|轴心国]][[外交官|外交官]]之外，其他国家的外交人员撤出[[北平|北平]]，这块使馆区才移交给[[国民政府|国民政府]]。

1949年以后这里仍被作为使馆区，与中华人民共和国建交的[[民主德国|民主德国]]、[[匈牙利|匈牙利]]、[[缅甸|缅甸]]等国的使馆和[[英國|英國]]的代表処沿用了这里旧有的建筑，直到1959年迁往[[朝阳门|朝阳门]]外[[三里屯|三里屯]]的第一使馆区。[[文化大革命|文化大革命]]期间，这里由于其历史的特殊性，再次受到冲击，街名被改为“'''反帝路'''”，很多西式风格的建筑遭到破坏，1980年代以来，随着北京城市建设的发展，东交民巷的建筑亦受到冲击，[[汇丰银行|汇丰银行]]、[[怡和洋行|怡和洋行]]、俄罗斯使馆的旧址因拓宽马路被拆除；[[德华银行北京分行旧址|德华银行]]于1992年被拆除；日本使馆旧址被[[北京市人民政府|北京市人民政府]]使用；街上还兴建了很多高层建筑和现代建筑，整条街的风貌遭到了极大的破坏。2001年，“东交民巷使馆建筑群”被列为[[北京全國重點文物保護單位列表|全国重点文物保护单位]]。

==建筑==
[[File:Peking_legation_quarter.jpg|thumb]]
《[[辛丑条约|辛丑条约]]》签订后，整个使馆界的面积约1.3平方公里，比原各使馆占地总面积增加了大约20倍。《辛丑条约》第14个附件对东交民巷使馆区的四至做了明确约定。
*东界：划为[[崇文门内大街|崇文门内大街]]，南起[[崇文门|崇文门]]，北到[[东单牌楼|东单牌楼]]（今东单路口北）；
*西界：划为南起[[正阳门|正阳门]]箭楼东侧，向北至今[[新大陆_(北京)|新大陆]]（自此以北至东长安街南侧，自北至南依次为原清[[宗人府|宗人府]]、[[吏部|吏部]]、[[户部|户部]]、[[礼部|礼部]]四个衙署所在地）西口附近向东，绕过原礼部东南墙角直线向北到[[东长安街|东长安街]]南侧（原清[[北京皇城|皇城]]南城墙）再向东，到今[[中华人民共和国公安部|公安部]]大门偏西处（原[[东三座门_(东长安街)|东三座门]]南端）再向北，直到东长安街北侧80米处（现存清皇城南城墙）。
*北界：划为今东长安街北侧80米处（紧邻现存清皇城南城墙），西起今公安部大门偏西处（原[[东三座门_(东长安街)|东三座门]]以北），东到东单牌楼；
*南界：划在今[[前门东大街|前门东大街]]（原[[北京内城|内城]]南城墙根），东起崇文门，西到正阳门箭楼东侧<ref name=men>{{Cite web |url = http://www.sohu.com/a/143533360_773040 |title =东交民巷消失的“门”  |author =  |date =  2017-05-26 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。

使馆界四周有高墙：南墙利用内城南城垣，东、西、北三面建6米高的界墙，墙上砌有雉堞、射击孔，相隔一定距离砌有红漆钢顶的炮台。整个围墙有八个碉堡并带铁门，作为使馆界出入口。围墙由各国按统一图纸分别建造。八个碉堡及铁门的位置是：
*崇文门西城根东口一座：今[[崇文门西大街|崇文门西大街]]北侧，紧邻内城城墙，坐西朝东。东北方为今[[新侨饭店|新侨饭店]]。
*前门东城根西口一座：今[[前门东大街|前门东大街]]北侧，紧邻内城城墙，坐东朝西。西南方紧邻[[正阳门|正阳门]]城楼。东北方为原美国兵营，兵营东侧为[[美国使馆旧址|原美国使馆]]。
*东交民巷东口一座：今东交民巷东口向西数十米处，[[北京医院|北京医院]]东南，坐西朝东。门外东南侧为原跑马场（今[[新侨饭店|新侨饭店]]）。由德国军队把守。
*东交民巷西口一座：今[[毛主席纪念堂|毛主席纪念堂]]东南侧，正对东交民巷处，坐东朝西。门外（西侧）为原东交民巷西口的“敷文”牌楼。东南方为原美国兵营。
*台基厂北口一座：今[[台基厂大街|台基厂大街]]北口以南，坐南朝北。门外向北稍偏西为[[东长安街|原东长安街牌楼]]。由意大利军队把守。
*东河沿北口一座：今[[正义路|正义路]]北口以南的东侧，原御河东岸，坐南朝北。东南方为[[意大利使馆旧址|原意大利使馆和兵营]]。由日本军队把守。
*西河沿北口一座：今正义路北口以南的西侧，原御河西岸，坐南朝北。西南方为[[淳亲王府|原英国使馆]]。由英国军队把守。
*兵部街中段一座：今[[中国国家博物馆|中国国家博物馆]]馆址中部偏西，坐东朝西。位于原英国兵营西南角。门内为原英国兵营。<ref name=men/>。

另外使馆界四周各使馆还另开了一些出入口：
*英国兵营西侧门：今中国国家博物馆馆址北部，坐南朝北。位于原英国兵营西北角。门内为原英国兵营。由英国军队把守。
*英国兵营东侧门：今公安部大门西侧，原[[东三座门_(东长安街)|东三座门]]正南，坐西朝东。位于原英国兵营东北角。门内为原英国兵营。由英国军队把守。
*意大利使馆和兵营北大门：今[[长安俱乐部|长安俱乐部]]南侧，坐南朝北。位于原意大利使馆和兵营正北中央。由意大利军队把守。
*[[水关门|水关门]]：今正义路南口，利用原内城正阳门东水关改建而成<ref name=men/>。

#[[淳亲王府|淳亲王府]]：[[东长安街|东长安街]]14号。现为[[中华人民共和国公安部|中华人民共和国公安部]]。
#[[淳亲王府|英国使馆旧址]]：东长安街14号。现为[[中华人民共和国公安部|中华人民共和国公安部]]。

#[[意大利使馆旧址|意大利使馆旧址]]：[[台基厂大街|台基厂大街]]1号。建于1910年前后。现为[[中国人民对外友好协会|中国人民对外友好协会]]等单位。

#[[国际俱乐部旧址|国际俱乐部旧址]]：台基厂大街8号
#[[法国兵营旧址|法国兵营旧址]]：台基厂三条3号
#[[比利时使馆旧址|比利时使馆旧址]]：[[崇文门西大街|崇文门西大街]]9号。建于1910年前后。现为紫金宾馆。
