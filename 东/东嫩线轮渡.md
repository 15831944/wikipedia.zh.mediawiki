'''东嫩线轮渡'''，为[[中华人民共和国|中国]][[上海市|上海市]][[黄浦江|黄浦江]]上的一条对江轮渡航线，于1977年7月1日建成通航。航线西起[[杨浦区|杨浦区]]嫩江路轮渡站，东至[[浦东新区|浦东新区]]东塘路轮渡站。东嫩线原为车客渡航线，但目前只运营客渡航线。

== 历史 ==
[[File:Nenjiang_Ferry_3.jpg|thumb]]
[[File:Dongtang_Rd_Ferry_1.jpg|thumb]]
1968年，为建设高桥石油化工区，由上海市内河航运公司负责勘察、设计，在浦西军工路共青苗圃（现[[上海共青国家森林公园|共青森林公园]]）南侧及浦东[[东塘路|东塘路]]间新辟一条对江车客渡航线。1970年2月，东嫩线轮渡正式开工建设。工程分两期建设，其中第一期工程为建设车辆渡，于1972年4月建成并于同年5月2日正式启用、第二期工程为建设客渡，整条航线于1977年7月1日建成投入使用<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node4501/node56018/node56020/userobject1ai43385.html 上海市地方志 专业志 >> 上海内河航运志 >> 第六章渡运 >> 节] {{webarchive|url=https://web.archive.org/web/20140101093158/http://www.shtong.gov.cn/newsite/node2/node2245/node4501/node56018/node56020/userobject1ai43385.html |date=2014-01-01 }}</ref><ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node4501/node56018/node56020/userobject1ai43387.html 上海市地方志 专业志 >> 上海内河航运志 >> 第六章渡运 >> 节] {{webarchive|url=https://web.archive.org/web/20140101093424/http://www.shtong.gov.cn/newsite/node2/node2245/node4501/node56018/node56020/userobject1ai43387.html |date=2014-01-01 }}</ref>。2003年11月20日，东嫩线轮渡线因客流原因而缩短运营时间<ref>[http://old.jfdaily.com/gb/node2/node142/node149/userobject1ai334811.html 东嫩线轮渡运营时间有变] {{webarchive|url=https://web.archive.org/web/20160304193120/http://old.jfdaily.com/gb/node2/node142/node149/userobject1ai334811.html |date=2016-03-04 }}</ref>。2007年12月31日24时起东嫩线汽车渡停运，目前只运营客渡航线<ref>[http://sh.eastday.com/qtmt/20071220/u1a385040.html 上海嫩江轮渡站汽车渡航线将从明年起停用]</ref>。
