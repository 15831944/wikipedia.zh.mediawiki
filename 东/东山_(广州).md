'''东山口'''位于广州市越秀区的[[农林下路|农林下路]]與商业区上。

== 地理和历史 ==
[[File:東山電話分所.jpg|thumb]]
[[File:東山百貨大樓.JPG|thumb]]
现在一般所说的“东山口”，是指地铁[[东山口站|东山口站]]附近的地区，即[[署前路|署前路]]、[[中山路_(廣州)|中山一路]]、中山二路、[[农林下路|农林下路]]、[[東華路|东华北路]]相交的交通路口及附近地区。南为[[东山湖|东湖]]、龟岗、东接[[梅花村|梅花村]]（民國政要寓所群）、北接[[東風路_(廣州)|东风东路]]商业区，西面系马棚岗。

东山得名于[[东山寺_(广州)|东山寺]]，一座落成于明朝成化十六年（1480年）的佛教寺庙，由太监韦眷修建，初名'''永泰禅寺'''，俗称“大鉴寺”。顺治年间顺治七年（1650年）重修时，前殿改建为真武庙，俗称东山庙；后殿改建为东山寺。到了民国初年便有记载“城东三里曰东山”。<ref>{{cite book|author=番禺县|title=番禺县续志·舆地志·山脉}}</ref>民国二十年（1931年），广州市推行保甲制，始设东山区公所，这是市属行政区以“东山”命名之始。 <ref>{{cite book|title=广州地方志}}</ref>

东山地方在民国初年系广州城外寺院和[[广东农林试验场|郊野]]，随着广州城的发展，[[广九铁路|广九铁路]]通车设[[大沙头站|大沙头站]]以及教会通过[[望厦条约|条款]]划地建设而逐渐演变为商业和居民区。在中共攻占广州之前的1940年代末期，东山已经成为精致时尚[[新河浦|高档住宅区]]。<ref>{{cite journal|author=叶农|title=美南浸信会与广州东山口}}</ref>

== 地标 ==
本区段内其知名建筑物和场所有[[东山电车总站|东山电车总站]]、[[广东省人民医院|广东省人民医院]]、[[中山大学附属第一医院|中山大学附属第一医院]]东山院区（原东山区人民医院）、[[越秀区人民医院|越秀区人民医院]]、{{link-yue|东山百货大楼|東山百貨大樓}}、[[廣州市培正中学|廣州市培正中学]]、[[新河浦|新河浦]]和[[越秀区图书馆|越秀区图书馆]]等。其中东山电车总站是广州市最古老的电车总站之一。

== 公共交通 ==
附近的巴士服务有东山口总站（电车）、东山口站、龟岗总站、东华东路站、新河浦站、东湖路站、海印桥站、大沙头站执信南路站18、22、40、44、45、50、102、106、107、108、112、129、183、222、227、243、247、285、502、508、541、548。而[[广州地铁一号线|广州地铁一号线]]、[[广州地铁六号线|六号线]]也在这里设“东山口站”。
