'''东山县'''（{{lang-nan|'''Tang-soaⁿ-koān'''}}）位于[[中国|中国]][[福建省|福建省]]东南部的[[东山岛|东山岛]]上，是[[漳州市|漳州市]]下辖的一个县，邻接[[广东省|广东省]]。

==历史==
东山，明初始称铜山。《诏安县志》载：“铜山所城，原为民间牧薮，土名‘东山”’。1916年设东山县（因江苏省已有铜山县，为避免重名，復称“东山县”）。閩南語中“东山”与“铜山”同音。

[[File:1965-01_福建东山岛.jpg|thumb]]
[[秦朝|秦朝]]属[[闽中郡|闽中郡]]，[[汉朝|汉朝]]隶[[揭阳县|揭阳县]]。[[晋朝|晋朝]]义熙九年（413年）划归[[绥安县_(东晋)|绥安县]]，[[隋朝|隋朝]]开皇十二年（592年）随绥安入[[龙溪县|龙溪县]]。[[唐朝|唐朝]]垂拱二年（686年）置[[怀恩县|怀恩县]]，开元二十九年（741年）入[[漳浦|漳浦]]。

明洪武二十年（1387年），江夏侯[[周德兴|周德兴]]布置福建海防，设16个城堡，45个巡司，在东山岛建起了[[铜山城|铜山城]]与铜山水寨（全闽五大水寨之一，管辖[[闽南|闽南]]海防）。洪武二十一年（1388年）信国公[[汤和|汤和]]在福建设立5个卫、12个守御千户所，东山岛为[[镇海卫_(福建)|镇海卫]][[铜山千户所|铜山千户所]]，官兵1220人。嘉靖后期，倭乱最猖獗。嘉靖四十三年（1564年）、万历十年（1582年），[[戚继光|戚继光]]两次亲临东山岛，并于万历九年（1581年）在东山岛设“浙兵营”，[[把总|把总]]一员、兵额450人，基本消除了倭患。

[[明朝|明朝]]嘉靖九年（1530年）置[[诏安县|诏安县]]。1549年[[葡萄牙帝國|葡萄牙]]欲佔領東山島做中國沿海據點，與明帝國發生[[走馬溪之戰|走馬溪之戰]]，葡戰敗退出。

民国时，1916年由漳浦县、诏安县析置东山县（因江苏省已有铜山县，为避免重名，复用古称“东山县”）。

抗日战争时期，日军发动了三次东山岛进攻战役：
*1939年7月11日，日军进攻东山。[[国民革命军|国民革命军]][[蒋鼎文|蒋鼎文]]第25路军[[韩文英|韩文英]]第75师（原为豫西绿林队伍改编的[[镇嵩军|镇嵩军]][[宋天才|宋天才]]第32军缩编）[[史克勤|史克勤]]第224旅(辖第449团、第450团）和盐警队（福建税警第五区600余人）、福建省保安第9团、东山县地方团队和壮丁队严密戒备对抗日军登陆企图，最终于7月16日日舰撤走。
*1939年8月23日日军发动第二次东山岛进攻战役，日军指挥涩谷，包括汪精卫[[和平建国军|和平建国军]][[郑天福|郑天福]]部，军舰5艘、汽艇20余艘、木船30余艘、作战飞机8架，在亲营、白埕等地登陆，占领荟冬、亲营、古雷庄、林边及梧龙庙；另一路日伪军从岐下、港口登陆，进占陈城、港口等地。随后进占进占湖塘、白埕、黄山母、赤山、大路口，守军退守西埔、坑内和城关。史克勤旅主力登岛支援，<ref>原第224旅旅长史克勤：《东山、诏安抗日御寇》，发表于《漳州文史资料（第18期）》1993年，转载于《东山文史资料（第11辑）》1994年1月。</ref>9月4日日伪军撤走。此役近30个村庄受到严重破坏，有167人被炸死，186人负伤，826间民房和120艘民船被炸毁。
*1940年2月12日，日军发动第三次东山岛进攻战役，出动日舰4艘，汽艇7艘，日军陆战队200多人乘汽艇于港口村登陆，汪精卫和平建国军[[黄大伟|黄大伟]]部2000多人在三支尖和宫前湾登陆，猛扑陈城、北山，进攻城区和西埔等地，守军退据坑内，与登岛援军匯合，经过五昼夜战斗，2月16日日伪军撤退。

中共建國後，退守台灣的國民政府亦曾向東山島發動進攻，分別如下：
*1953年7月15日－7月18日[[東山島戰役|东山岛之战]]，金门防卫司令[[胡琏|胡琏]]率第19军第45师、第18师第53团及[[中華民國空降特戰部隊|伞兵总队]]第一大队487人、陆战队步兵第1旅，共计1万多人，在军舰与作战飞机支援下，登陆猛攻东山岛。守军为解放军福建军区公安第80团及1个水兵连共1200人，团长[[游梅耀|游梅耀]]。最后闽南、粤东紧急派出的援军机动兵力登岛，胡琏部败退。
*1965年8月6日[[東山海戰|東山海戰]]。

==地理==
东山县位于[[福建省|福建省]]南部，北纬23度42分，东经117度25分附近。东面是[[台湾海峡|台湾海峡]]，西面是[[诏安县|诏安县]]，西北面是[[云霄县|云霄县]]，东北面是[[漳浦县|漳浦县]]；面积247平方公里。

东山岛由主岛和周边67个小岛组成，状如七星拱月，形如栩栩如生蝴蝶，故称蝶岛。主岛面积194平方公里，原与大陆相连，后[[地壳|地壳]]断裂下降，与大陆分离，为福建省的第二大岛，全岛海岸线总长141.3公里。1961年修建八尺门海堤与陆地相连，成为半岛。2014年八尺门海堤拆除，恢复成全岛。<ref>{{Cite web|url=http://fj.sina.com.cn/news/m/2014-03-18/093753098.html|title=八尺门海堤下月将拆除 东山岛将从半岛变回全岛_新浪福建城事_新浪福建|accessdate=2016-07-25|work=fj.sina.com.cn}}</ref>

东山县属亚热带海洋性气候，1月份平均气温13.1℃，7月份平均气温27.3℃。

== 文化 ==
全县通行[[闽南语|闽南语]]。东山县的东山歌册被列入[[国家级非物质文化遗产名录|国家级非物质文化遗产名录]]。

== 交通 ==
东山境内无高速公路，首条高速公路漳州市高速公路东山支线预计2010年下半年开工<ref>[http://www.fjsen.com/fujian/2009-08/15/content_757535.htm 东山县即将结束境内无高速公路历史] {{webarchive|url=https://web.archive.org/web/20090819030335/http://www.fjsen.com/fujian/2009-08/15/content_757535.htm |date=2009-08-19 }}</ref>。

== 资源 ==
* 矿产资源有[[硅砂|硅砂]]、[[辉绿岩|辉绿岩]]、[[高岭土|高岭土]]、[[花岗岩|花岗岩]]、砖土等。
* 植物有[[柏|柏]]、[[松|松]]、[[杉|杉]]、[[木麻黄|木麻黄]]、[[橡胶|橡胶]]等。
* 土特产有[[柑橘|柑橘]]、[[龙眼|龙眼]]、[[荔枝|荔枝]]、[[番石榴|番石榴]]、[[芦笋|芦笋]]
* 東山島盛產[[東山羊|東山羊]]，此外东山鲍鱼为[[中国地理标志产品|中国地理标志产品]]<ref>[http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm 《地理标志产品保护规定》实施以来批准的地理标志名录（2005年7月至2009年6月）] {{webarchive|url=https://web.archive.org/web/20110311120900/http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm |date=2011-03-11 }}</ref>。

== 名胜古迹 ==
* [[东山风动石|东山风动石]]：为[[国家4A级旅游景区|国家4A级旅游景区]]，位于在铜山古城东门海滨石崖上。
* [[东门屿|东门屿]]：又称“塔屿”，位于东山岛[[铜陵镇|铜陵镇]]的东侧，为“中国四大名屿”之一。
* [[东山关帝祖庙|东山关帝祖庙]]：为[[全国重点文物保护单位|全国重点文物保护单位]]、台湾1000多座关帝庙的香源祖庙。
* [[马銮湾|马銮湾]]沙滩度假区：为大众型沙滩度假区。
* 水寨大门：为民族英雄[[戚继光|戚继光]]、[[郑成功|郑成功]]扎寨练兵的旧址。
* [[金銮湾|金銮湾]]休闲区：为海滨康乐休闲度假区。
* [[乌礁湾|乌礁湾]]：为海滨森林公园。
* [[苏峰山|苏峰山]]：古名东山、川陵山。海拔274米，是东山岛的最高峰。
