'''東歐'''，多半是指[[波蘭|波蘭]]、[[白俄罗斯|白俄罗斯]]、[[乌克兰|乌克兰]]、[[俄罗斯|俄罗斯]]以西的歐洲，直至[[德國|德國]]、[[瑞士|瑞士]]、[[義大利|義大利]]（不含）；但缺乏明确的定义和边界，一般有自然地理和人文地理两种划分方法：

自然地理方面的東歐，是指較少受到[[大西洋|大西洋]]和[[地中海|地中海]]海流影響，接近[[烏拉爾山|烏拉爾山]]、[[乌拉尔河|乌拉尔河]]和[[伏爾加河|伏爾加河]]（[[窩瓦河|窩瓦河]]）流域，實際上主要是指[[俄羅斯聯邦|俄羅斯聯邦]]和[[哈萨克斯坦|哈萨克斯坦]]的[[歐洲|歐洲]]部分和其他[[獨立國協|獨立國協]]的歐洲會員國家。

人文地理（主要指[[政治|政治]]上）的东欧是指自[[第二次世界大戰|第二次世界大戰]]後，或遭[[蘇聯|蘇聯]]併吞，或成為蘇聯的[[附庸國|附庸國]]，政治上由[[共产党|共产党]][[一黨專政|一黨專政]]，經濟上多採用[[計划經濟|計划經濟]]體制的[[社會主義國家|社會主義國家]]，这点与[[西欧|西欧]]相反。大多东欧国家以[[斯拉夫民族|斯拉夫民族]]為主體，居民多信奉[[東正教|東正教]]，但共產黨時代，官方推行[[无神论|无神论]]。

本條目再把東歐分為兩區：東歐和[[獨立國家國協|獨立國家國協]]（以下內容，簡稱為國協）。<ref>這裡指的獨立國家國協，是指曾經參加過國協的國家，包括[[俄羅斯|俄羅斯]][[歐洲俄羅斯|歐洲部分]]、[[白俄羅斯|白俄羅斯]]、[[烏克蘭|烏克蘭]]、[[摩爾多瓦|摩爾多瓦]]以及[[南高加索|南高加索]]共和國。</ref>

== 地形 ==
=== 東歐 ===
[[File:AIRM_-_Building_on_Ștefan_cel_Mare_avenue_132,_Chișinău_-_aug_2015.jpg|thumb]]首都[[基希涅夫|基希涅夫]]街頭，濃厚東歐色彩]]
[[File:Vkraina-Cosaques-Okraina_small.png|thumb]]

本區的地形大致可以[[喀爾巴阡山|喀爾巴阡山]]為分界，分成南北兩部分。

北部的[[波蘭|波蘭]]和[[波羅的海三小國|波羅的海三小國]]平原廣闊，為[[歐俄平原|歐洲大平原]]的東段，地勢由南向北傾斜。第四季的大陸[[冰河|冰河]]在平原南部留下肥沃的[[冰河|冰河]]外洗黃土，也在平原的中北部地區堆積了成群的冰磧丘，使平原成波狀起伏。冰磧丘間的東西向窪地，自古以來即為東西向交通的必經之地。

=== 獨立國協 ===
東歐以東、[[烏拉山|烏拉山]]以西的歐俄部份，是一片遼闊且起伏平緩的古老陸塊，地層十分穩定堅實，全境多屬久經侵蝕的[[準平原|準平原]]，又主要位於[[俄羅斯|俄羅斯]]境內，故稱[[俄羅斯平原|俄羅斯平原]]。平原中部為[[瓦耳代丘陵|瓦耳代丘陵]]，是河流的[[分水嶺|分水嶺]]。

位於[[裏海|裏海]]和[[黑海|黑海]]之間的[[高加索山|高加索山]]，是歐亞的界山之一，山勢高大，地形崎嶇複雜，夾雜著高原、平原和低地。

== 氣候 ==
=== 東歐 ===
受位置影響，過渡性質是本區氣候最主要的特徵。北部各國介於西歐和國協之間，屬於[[溫帶海洋性氣候|溫帶海洋性氣候]]和[[溫帶大陸性氣候|溫帶大陸性氣候]]間的過渡性氣候，冬天酷寒，地面積雪自西部的數週，向東逐漸增加到俄羅斯邊境的兩個半月，雖冷於[[西歐|西歐]]，卻較俄境溫暖；夏季較[[西歐|西歐]]熱，卻又較[[俄羅斯|俄羅斯]]涼爽許多。南部的前[[南斯拉夫|南斯拉夫]]各國、[[阿爾巴尼亞|阿爾巴尼亞]]、[[羅馬尼亞|羅馬尼亞]]、[[保加利亞|保加利亞]]等國位居[[巴爾幹半島|巴爾幹半島]]北部，屬於[[地中海型氣候|地中海型氣候]]和[[溫帶大陸性氣候|溫帶大陸性氣候]]之間的過渡性氣候。

東歐距離溫暖的[[北大西洋|北大西洋]]已遠，全區最冷月均溫皆在零度以下；夏季則因地處內陸，缺乏水氣調劑而顯炎熱，最暖月溫度自北向南見增，北部平原四國不足20℃，南部各國則多超過20℃。

歐俄大部分位於[[溫帶大陸性氣候|溫帶大陸性氣候]]內，冬季漫長而嚴寒，夏季短促而溫暖，春秋季甚短，年溫差甚大，而且愈東愈明顯。夏季是主要雨季，水氣主要由西風自北大西洋帶入，年雨量約為500公釐，且由西向東遞減。

本區冬溫甚低，故河流冰封，雪橇通行，冬季（以月均溫小於10℃為準）長達六個月至八個月。由於氣溫甚低，空氣中水汽含量稀少，故溼度甚低，空氣十分乾冷。夏季由於地處內陸，氣溫較西歐暖，受緯度影響顯著。沿海地區受海洋影響，夏季氣溫低於內陸，但差異不大，沿海於冬季亦較暖。

冬季降雪區域廣泛，每年11月至翌年三月，時常落雪，北部泰卡（Taiga）針葉林區，冬雪最多，經常積雪二、三呎，春季雪融緩慢，融雪水滲入土中，可供植物吸收，對[[寒帶森林|寒帶森林]]的發育裨益甚大。

南部和[[高加索地區|高加索地區]]因山脈阻擋[[俄羅斯|俄羅斯]]酷寒氣流南下，氣候較暖，有助於旅遊業的發展。高加索地區由於緯度較低，氣候上已帶有濃厚副熱帶色彩。

== 民族 ==
* [[斯拉夫民族|斯拉夫民族]]發源於今波蘭東南部維斯杜拉河上游一帶，於西元[[一世紀|一世紀]]時開始向外遷徙，至[[六世紀|六世紀]]時期居地已經遍布東歐以及俄羅斯地區。依居住地的不同，斯拉夫民族可分成東（主要為俄羅斯人、烏克蘭人、白俄羅斯人等）、西（主要為波蘭人、捷克人、斯洛伐克人等）、南（主要為塞爾維亞人、斯洛文尼亞人、克羅地亞人、馬其頓人、黑山人、保加利亞人）三支，其中東支主要分布於俄羅斯境內，分布在東歐各地者以西、南兩支為主。

* [[馬札兒人|馬札兒人]]原為烏拉山西側的草原民族，於西元[[九世紀|九世紀]]建立匈牙利王國。在向西遷徙的過程中，曾與斯拉夫人、[[西突厥|西突厥]]人混居，移居匈牙利平原之後又受日耳曼文化的影響。

* [[阿爾巴尼亞人|阿爾巴尼亞人]]主要分布於今[[阿爾巴尼亞共和國|阿爾巴尼亞共和國]]和[[科索沃共和國|科索沃共和國]]，由於[[十四世紀|十四世紀]]以後一直受到[[土耳其|土耳其]][[奧斯曼帝國|奧斯曼帝國]]的統治，因而文化上充滿伊斯蘭色彩，宗教上也以信仰[[伊斯蘭教|伊斯蘭教]]（57%）為主。

* [[羅馬尼亞人|羅馬尼亞人]]是東歐唯一的拉丁語民族，但跟大多数斯拉夫民族一样，信奉东正教，他們是古羅馬人的後裔，因羅馬帝國曾在此建省並統治由留居東歐。

== 經濟 ==
東歐盛產[[小麥|小麥]]、[[馬鈴薯|馬鈴薯]]、[[甜菜|甜菜]]、[[向日葵|向日葵]]，養畜業發達。共產制度瓦解（因蘇聯解體）後紛紛加入歐盟，以廉價勞工吸引各國企業設廠。東歐人口眾多，城市密布、交通網發達，分布有許多著名工業區。

*俄羅斯：莫斯科、烏拉山、庫斯內次（庫斯巴次，西部西伯利亞平原的東南方）、貝加爾湖工業區（貝加爾湖（L.Baykal）的西北方）；
*烏克蘭：前蘇聯最大軍火工業區頓內次工業區。

整體來說，該區的經濟情況自二戰前始終落後於西歐、北歐和南歐的[[西班牙|西班牙]]及[[義大利|義大利]]，只有[[捷克|捷克]]、[[匈牙利|匈牙利]]、[[斯洛伐克|斯洛伐克]]、[[愛沙尼亞|愛沙尼亞]]和[[斯洛維尼亞|斯洛維尼亞]]的經濟情況較為富庶，但與[[第三世界|第三世界]]國家相較，東歐國家的生活水平仍然算是不錯的。

== 文明分区 ==
按照[[塞繆爾·P·亨廷頓|亨廷顿]]的文明冲突论，东欧还可以细分为以下数个文明圈：
* 被[[神圣罗马帝国|神圣罗马帝国]]统治过的匈牙利、捷克、斯洛伐克、斯洛文尼亚、克罗地亚，以及被[[德意志帝国|德意志帝国]]和[[奥地利帝国|奥地利帝国]]统治过部分领土的波兰属于西方文明下属的天主教文明圈，这几个国家不同程度地受[[德意志|德意志]]文化的影响（尤以曾有大量德国地主移民拓殖的波兰和捷克为最），大多数民众信奉[[罗马天主教|罗马天主教]]（捷克例外，捷克如今已是无宗教信仰者比例最高的国家之一），较早建立[[代議民主制|代议制民主]]政体（如波兰早在1791年就颁布了欧洲第一部成文宪法——[[五三宪法|五三宪法]]），在东欧剧变后民主转型较为平和。这些国家大多在一战前就已开展[[工业革命|工业革命]]，经济虽然落后于西欧但仍然领先全球大多数国家（捷克在二战前还拥有能与德国[[克虏伯|克虏伯]]匹敌的先进军工企业）。科技和文化也较为发达，如[[萧邦|萧邦]]、[[居里夫人|居里夫人]]、[[沃卓斯基姐弟|沃卓斯基姐弟]]、[[罗曼·波兰斯基|罗曼·波兰斯基]]为世界知名的波兰裔文化与科技名人，[[约翰·冯·诺伊曼|约翰·冯·诺依曼]]、[[西奥多·冯·卡门|西奥多·冯·卡门]]、[[裴多菲|裴多菲]]、[[斯坦因|斯坦因]]、[[乔玛|山陀尔·乔马]]、[[索罗斯|索罗斯]]等皆为匈牙利裔科学与文化名人，[[德沃夏克|德沃夏克]]、[[米兰·昆德拉|米兰·昆德拉]]则为捷克的文化名人。
* 被[[奥斯曼土耳其帝国|奥斯曼土耳其帝国]]长期统治的[[巴尔干半岛|巴尔干半岛]]诸国（斯洛文尼亚和克罗地亚除外）大多属于东正教文化圈，这些国家的文明来自对[[拜占庭|拜占庭]]文化的学习和吸收，民众大多信奉[[东正教|东正教]]。按照亨廷顿的理论，巴尔干的东正教国家没有如西欧国家一般经历[[文艺复兴|文艺复兴]]、[[宗教改革|宗教改革]]、[[启蒙运动|启蒙运动]]等思想解放革命，人文精神薄弱，未能树立个人主义和自由主义传统。这些国家在脱离奥斯曼帝国独立后大多实行君主专制政体，而且在二战后又都建立了共产集权政体（希腊除外），冷战后的民主转型较之波兰、捷克斯洛伐克等中欧国家亦困难得多。这些国家和中欧国家的另一显著区别就是国内民族众多，民族矛盾复杂，[[南斯拉夫內戰|南斯拉夫]]和阿尔巴尼亚在冷战结束后都爆发了激烈的动乱。由于两次[[工业革命|工业革命]]都没有赶上，直到二战后才开始工业化，巴尔干诸国经济普遍落后（冷战期间被西欧诸国倾全力扶助了几十年的希腊除外），不仅落后于西欧和中欧的国家，就是跟台湾、韓國及新加坡等亚洲先进国家相比也算是贫穷国家。巴尔干地区的人口贩卖活动尤其猖獗。
* 俄罗斯、乌克兰、白俄罗斯三国都是[[東斯拉夫民族|东斯拉夫人]]的国家，种族血缘和语言文字都很相近，也都属于东正教国家。但该三国较之巴尔干诸国专制传统更为深厚，民主转型更为困难，因而被更早的文明史观论者[[阿諾爾得·約瑟·湯恩比|汤因比]]视为独立的俄罗斯文明圈。乌克兰和白俄罗斯亦属于经济落后国家，尽管乌克兰仍能维持世界一流的[[軍工業|军工业]]。俄罗斯因为大量出口石油、天然气和武器军备而尚能维持高收入水平。
* 阿尔巴尼亚、波黑和马其顿属于穆斯林和东正教徒混合杂居的地区。历经[[霍查|霍查]]对宗教的几十年铁拳打压，阿尔巴尼亚的大多数民众如今已无宗教信仰。
