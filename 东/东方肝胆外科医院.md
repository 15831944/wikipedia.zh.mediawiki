'''东方肝胆外科医院'''、'''东方肝胆外科研究所'''（'''中国人民解放军海军军医大学第三附属医院'''），位于[[上海市|上海市]]，隶属[[中国人民解放军海军军医大学|中国人民解放军海军军医大学]]，是[[三级甲等医院|三级甲等综合医院]]<ref name=gk>{{Cite web|url = http://www.ehbh.cn/intro.php|title = 医院概况|author = |date = |publisher = 第二军医大学第三附属医院|language = |accessdate = 2017-06-15|quote = |archive-url = https://web.archive.org/web/20170705100115/http://www.ehbh.cn/intro.php|archive-date = 2017-07-05|dead-url = yes}}</ref>。

==简介==
医院由[[中国科学院院士|中国科学院院士]][[吴孟超|吴孟超]]创建并领导。1958年，吴孟超、[[张晓华|张晓华]]、[[胡宏楷|胡宏楷]]三人组成[[上海长海医院|长海医院]]肝胆外科研究小组<ref name=renming>我校隆重举行东方肝胆外科医院领导班子任命大会，第二军医大学学报1996年第05期</ref>。1963年，吴孟超成功完成世界第一例中肝叶切除术<ref name=kjrb>{{Cite web |url = http://www.cas.cn/zt/rwzt/wumengchao/mtbd/201106/t20110609_3285008.html  |title = 肝脏禁区创神奇——人民医学家 吴孟超 |author =  |date = 2011-04-27 |publisher =科技日报  |language =  |accessdate =  |quote =  }}</ref>。1978年，吴孟超在长海医院申请成立了[[中华人民共和国|中华人民共和国]]第一个肝胆外科<ref name=doushi>{{Cite web |url = http://www.lifeweek.com.cn/2011/0503/32105_4.shtml |title =肝癌斗士吴孟超  |author =吴丽玮  |date =  |publisher =  三联生活周刊2011年第18期|language =  |accessdate =  |quote =  }}</ref>。1993年，吴孟超提议成立肝胆外科医院。不久，肝胆外科成为长海医院的“院中院”<ref name=kjrb/>。1993年5月，[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、[[中央军委主席|中央军委主席]][[江泽民|江泽民]]题写了“东方肝胆外科医院、研究所”的院名、所名<ref name=hxjww>{{Cite web |url = http://www.huaxia.com/hxjk/zkzn/gb/2008/04/919708.html   |title = 东方肝胆外科医院 |author =  |date =   2008-04-24|publisher =华夏经纬网  |language =  |accessdate =  |quote =  }}</ref>。1996年，吴孟超获[[中央军事委员会|中央军委]]授予“模范医学专家”称号。1996年9月，'''中国人民解放军第二军医大学第三附属医院'''（'''东方肝胆外科医院'''、'''东方肝胆外科研究所'''）成立，这是中华人民共和国最大的肝胆外科医院和研究所<ref name=kjrb/>，也是中华人民共和国第一个肝胆外科专科医院<ref name=gk/>。1996年9月10日，第二军医大学举行宣布东方肝胆外科医院领导班子任命大会，吴孟超任院长，杭开才任政委<ref name=renming/>。1996年12月被[[中国人民解放军总后勤部|中国人民解放军总后勤部]]批准为“三级甲等”专科医院<ref name=hxjww/>。此后该医院又发展成为以肝胆外科为特色的三级甲等综合医院<ref name=gk/>。

2006年，该医院计划西迁安亭。2010年12月安亭院区开工建设，2012年6月结构封顶<ref>{{Cite web |url =  http://china.huanqiu.com/hot/2015-10/7790044.html|title = 东方肝胆医院安亭新院开张:专科变综合 |author =  |date = 2015-10-19 |publisher =环球网  |language =  |accessdate =  |quote =  }}</ref>。2015年10月18日，安亭院区试运行<ref name=gk/>。2017年，更名'''中国人民解放军海军军医大学第三附属医院'''。

国家肝癌科学中心与医院毗邻，是由吴孟超院士、[[王红阳|王红阳]]院士牵头成立的国家级肝癌研究中心，以安亭新院为依托<ref name=gk/>。
