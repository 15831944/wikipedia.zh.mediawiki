'''东至县'''位于[[中国|中国]][[安徽省|安徽省]]南部，是[[池州市|池州市]]下辖的一个县，位于[[长江下游|长江下游]]南岸，北望安庆，南邻江西。

== 历史 ==
[[File:Nieuhof-Ambassade-vers-la-Chine-1665_0781.tif|thumb]]在1665年所画的东流镇风景图]]
[[南唐|南唐]][[保大_(南唐)|保大]]十一年（953年）升东流场为[[东流县|东流县]]，属[[江州_(隋朝)|江州]]；《明一统志》：“东流县，以大江自湓城而下，逶迤东注，故名”。民国21年（1932年）10月，[[秋浦县_(1914年)|秋浦县]]更名至德县。1959年由东流、至德两县合并成东至县。

境内舜耕山（又名大历山），相传为舜躬耕之地，尧访舜时由此渡河，留下众多遗址和传说，县城遂称“尧渡”，自古就有“尧舜之乡”美誉。

== 地理 ==
东至县西北滨临长江，与[[怀宁县|怀宁县]]、[[望江县|望江县]]一水之隔；东与[[贵池区|贵池区]]、[[石台县|石台县]]、[[祁门县|祁门县]]接壤；北部和[[安庆市|安庆市]]区隔江相望；南部和西南部与江西省[[浮梁县|浮梁县]]、[[鄱阳县|鄱阳县]]、[[彭泽县|彭泽县]]接壤。县域坐标：东经116º39"—117º18"、北纬29º34"—30º30"，南北长108公里，东西宽65公里。面积占全省总面积的2.3%。

地势南高北低，最高峰仙寓山（1375.7米），北部西北部地势最低，最低处七里湖及新丰圩（9.5米）。

气候温和，四季分明。多年平均气温16.9℃，多年平均降雨量1554.4mm，无霜期223天。

==行政区划==
东至县辖12个镇、3个乡：尧渡镇、东流镇、大渡口镇、胜利镇、张溪镇、洋湖镇、葛公镇、香隅镇、官港镇、昭潭镇、龙泉镇、泥溪镇、花园乡、木塔乡、青山乡

== 资源 ==
东至县资源丰富，矿产以石灰石、白云石、方解石储量大，分别达20亿吨、4亿吨、2亿吨以上。

== 经济 ==
农业生产条件得天独厚，已形成优质米、优质油、优质棉、食用菌、茶叶、禽畜、水产、经果林等八大农产品开发基地，是全国“商品粮基地县”、“全国优质棉基地县”、“全国速生丰产林基地县”、“全国绿化先进县”、“沿江渔业高效示范区”和“全国红茶出口基地县”。

工业有纺织服装、化工、机械、玩具、新型非金属材料和绿色食品生产加工等六大支柱产业。

== 交通 ==
东至县交通便捷，尧渡镇北距省会[[合肥市|合肥市]]247km，南距江西省[[景德镇市|景德镇市]]152km。
* 公路：[[206国道|206国道]]和[[济广高速公路|济广高速公路]]纵贯南北，[[318国道|318国道]]横穿东西；
* 铁路：[[铜九铁路|铜九铁路]]；
* 水路：长江黄金水道流经县内85公里，沿江有香口、东流、大渡口3个港口。

== 名胜古迹 ==
* [[升金湖|升金湖]]，是安徽省第一个以水禽及其栖息环境为保护对象的[[国家级自然保护区|国家级自然保护区]]，又是亚洲重要湿地之一。
* [[历山|历山]]（舜耕山），距县城12千米，集“尧舜遗迹、佛教文化、岩溶景观”于一山；为省级风景名胜区、国家3A级旅游景区。
* [[东流镇|东流古镇]]，因濒临长江，取“大江东流”之意为名；东晋[[陶渊明|陶渊明]]留下了“采菊东篱下，悠然见南山”的千古名句。
* 仙寓山，距县城40余千米，海拔1375米，林木苍翠，溪流碧透，奇石峥嵘；有近百处瀑布，为皖南山区最大的瀑布群。
* 陶公祠，为省级重点文物保护单位。
* 红鱼池，位于东至县张溪镇，曾被央视《北纬三十度中国行》报道。

== 特产 ==
*葛粉、麦鱼、升金湖咸鸭蛋、官港花菇、升金湖青虾、东至蕨菜、东至香菇、菜糕、葛公豆腐、建新乡珍珠、紫石塔毛尖等。
