'''东安县'''位于[[湖南省|湖南省]]西南边缘、[[永州市|永州市]]西北部，为永州市代管县。[[国内生产总值|国内生产总值]]294，935万元（2002），总人口为587,036人（2002年），其中城镇人口69,260人，乡村人口517,776人<Ref>[https://web.archive.org/web/20060527063510/http://www.hntj.gov.cn/tjgb/xqgb/yzgb/default.htm]</ref>。[[县治|县治]][[白牙市镇|白牙市镇]]。[[东安子鸡|东安子鸡]]是其最出名的特产。

== 历史 ==

明清时，东安县属[[永州府|永州府]]。中华民国初年，全国[[废府州厅改县|废府]]，直属[[湖南省_(中華民國)|湖南省]]。1949年，[[第二次国共内战|第二次国共内战]]进入尾声。[[衡宝战役|衡宝战役]]后，中国人民解放军攻占湖南全省指日可待。10月9日至12月5日间，在当地游击队配合，解放军先后攻占了今永州市各县。10月21日，[[中国人民解放军第41军|中国人民解放军第41军]]401团攻占东安县城<ref>{{Cite web |url=http://lib.huse.cn/lzy/news_view.asp?newsid=6312 |title=第十六章 国民党统治的崩溃与永州人民的解放（二）第四节永州全境的解放|accessdate=2018-06-21 |author= |date=2012-07-05 |work= |publisher=湖南科技学院图书馆——潇湘文化专题网，来源：《永州史话》|language=Zh-hans }}</ref>。其后，东安县历属[[永州专区|永州专区]]、[[零陵专区|零陵专区]]、[[湘南行政区|湘南行政区]]、[[零陵地区|零陵地区]]。1995年，零陵地区改为永州市，隶属至今。

== 地理 ==

面积2204平方千米，相邻的[[县级行政区|县级行政区]]，北与[[邵阳县|邵阳县]]交界，西接与[[新宁县|新宁县]]，南邻[[广西壮族自治区|广西壮族自治区]][[全州县|全州县]]，东连[[祁东县|祁东县]]和永州[[市辖区|市辖区]][[零陵区|零陵区]]。
