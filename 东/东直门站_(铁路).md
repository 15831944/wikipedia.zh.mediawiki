'''东直门站'''，位于今[[北京市|北京市]][[东城区_(北京市)|东城区]][[东直门桥|东直门桥]]南侧，1958年拆除。

==简介==
东直门站是[[北京环城铁路|北京环城铁路]]的车站。1915年6月16日，北京开始兴建环城铁路，自[[北京北站|西直门站]]经[[德胜门站|德胜门站]]、[[安定门站_(铁路)|安定门站]]、东直门站、[[朝阳门站_(北京环城铁路)|朝阳门站]]至[[东便门站|东便门站]]沟通[[京奉铁路|京奉铁路]]，终点是[[正阳门东站|正阳门东站]]。沿线经过的各城门瓮城城墙全部拆除，仅保留箭楼及部分整修过的城墙。1916年1月1日，环城铁路全线建成通车<ref name=qi>环城铁路——为老北京的交通松口气儿，和谐之旅2011年4月</ref><ref name=laobeijing>{{Cite web |url =http://www.sohu.com/a/155979515_115482 |title =老北京东直门火车站  |author =  |date = 2017-07-10  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。

东直门站刚通车时，[[东直门|东直门]]城楼、箭楼尚未拆除，而东直门瓮城已拆除。东直门站站房位于[[北京内城|北京内城]]东直门城楼东南侧，东直门箭楼西南侧，东直门瓮城旧址以南，北京内城城墙东侧不远处。东直门站以北，在东直门城楼和箭楼之间中央设有铁路道口。东直门站铁路下方有涵洞，水从城里经东直门南水关及该涵洞通往护城河<ref name=laobeijing/>。

1954年7月，北京铁路管理局根据[[北京市人民政府|北京市人民政府]]的城市建设总体规划，计划封闭北京环城铁路的德胜门站、安定门站、东直门站、朝阳门站，以便减少环城铁路对北京城市地面交通的影响。1958年开始，西直门站至[[广安门站|广安门站]]的铁路联络线停用，并改建北京环城铁路，拆除了从西直门站经德胜门站、安定门站、东直门站至朝阳门的铁路<ref name=qi/>。
