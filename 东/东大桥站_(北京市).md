'''东大桥站'''位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]]，是[[北京地铁|北京地铁]][[北京地铁6号线|6号线]]的一座车站。

== 位置 ==
东大桥站位于[[北京二环路|东二环]]和[[北京三环路|东三环]]之间；[[工体东路|工体东路]]－[[东大桥路|东大桥路]]（南北向）和[[朝阳门外大街|朝阳门外大街]]（西－东南走向）、[[朝阳北路_(北京)|朝阳北路]]（自此处向东）的交汇处东侧，位于朝阳北路地下，呈东西走向。<ref>{{cite web|url=http://www.bjghw.gov.cn/web/static/articles/catalog_14300/article_ff808081262b02de01262c2fa1a90005/ff808081262b02de01262c2fa1a90005.html|title=轨道交通6号线一期工程规划方案公告|publisher=北京市规划委员会|date=2010-01-14|accessdate=2012-02-10|archive-url=https://web.archive.org/web/20101227055616/http://www.bjghw.gov.cn/web/static/articles/catalog_14300/article_ff808081262b02de01262c2fa1a90005/ff808081262b02de01262c2fa1a90005.html|archive-date=2010-12-27|dead-url=yes}}</ref>

== 结构 ==
东大桥站为地下二层车站，部分为一层，[[岛式站台|岛式站台]]设计<ref>{{cite web|url=http://www.crtg-2.com/main/newsread.asp?id=200|title=北京地铁六号线三标三工区东大桥车站主体双层段初支扣拱开始施工|accessdate=2012-02-10|publisher=中铁二局}}</ref>，双柱三跨结构，车站工程总长255米<ref>{{cite news|url=http://www.cenn.cn/News/20120504-158299.shtm|title=北京地铁六号线03标段东大桥车站达到铺轨条件|date=2012-05-04|accessdate=2012-05-04|deadurl=yes|archiveurl=https://web.archive.org/web/20120627094937/http://www.cenn.cn/News/20120504-158299.shtm|archivedate=2012-06-27}}</ref>。

== 接驳交通 ==
东大桥是现状北京东部重要的公共交通换乘中心站，现有场站在路口东北角，市民广场北，与6号线车站北侧两个出入口衔接。
