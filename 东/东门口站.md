'''东门口站'''，全称'''东门口(天一广场)站'''，建设时曾命名'''天一广场站'''，是[[浙江省|浙江省]][[宁波市|宁波市]]一座地下轨道交通车站，属于[[宁波轨道交通|宁波轨道交通]][[宁波轨道交通1号线|1号线]]。车站于2014年5月30日开通运营<ref name="main">{{cite web|url=http://daily.cnnb.com.cn/dnsb/html/2009-06/26/content_96812.htm|title=详解1号线一期站点|publisher=东南商报|date=2009-06-26|accessdate=2013-01-11}}</ref>。

== 车站形制 ==
东门口站位于[[海曙区|海曙区]]中山东路与日新街路口，中山东路下方，为地下三层[[岛式站台|岛式车站]]。车站净长172米，净宽19.3米，地下建筑面积为9979平方米。

== 装饰 ==
东门口站是宁波轨道交通1号线的6个文化站之一。车站的文化主题为“江厦·天下”，取材于有“走遍天下，不如宁波江厦”之称的[[江厦街|江厦街]]和[[宁波商帮|宁波商帮]]启航的东渡门码头。车站艺术墙由多个大小一致的三角柱拼接而成，右侧绘有宁波画家[[金林观|金林观]]绘制的古代东渡门和[[三江口_(宁波)|三江口]]商贸往来的繁华景象，左侧则设有当代三江口的夜景照片。从不同角度观看艺术墙可以得到不同图像，展现了三江口的变迁<ref>{{cite web|url=http://news.cnnb.com.cn/system/2014/05/13/008061217.shtml|title=宁波轨道交通1号线6个站点文化墙惊艳亮相|publisher=中国宁波网|date=2014-05-13|accessdate=2014-05-13}}</ref>。

== 交通接驳 ==
该站接驳公交东门口站<ref>{{cite web|url=http://www.nbmetro.com/line_info.php |title=宁波轨道交通车站信息查询 |publisher=宁波市轨道交通集团有限公司 |accessdate=2014-06-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20150922060232/http://www.nbmetro.com/line_info.php |archivedate=2015-09-22 }}</ref>。此外，车站附近还设有1个[[宁波公共自行车|公共自行车]]租赁点，安设有50辆自行车<ref>{{cite web|url=http://news.cnnb.com.cn/system/2014/04/04/008029106.shtml|title=宁波轨道交通1号线一期公交接驳方案发布|publisher=东南商报|date=2014-04-04|accessdate=2014-05-18}}</ref>。
