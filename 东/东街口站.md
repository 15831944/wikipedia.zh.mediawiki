'''东街口站'''位于[[福建省|福建省]][[福州市|福州市]][[鼓楼区_(福州市)|鼓楼区]][[杨桥路|杨桥路]]、东街与[[八一七路|八一七路]]的交叉路口，是[[福州地铁1号线|福州地铁1号线]]的一座中间站，预留与[[福州地铁4号线|福州地铁4号线]]实现换乘，该站于2017年1月6日投入使用<ref name=":0">{{cite news|author=原浩|url=http://news.fznews.com.cn/dsxw/20170104/586c38fd119eb.shtml|title=福州地铁1号线（一期）全线1月6日正式开通试运营|publisher=福州新闻网|date=2017-01-04|accessdate=2017-01-05}}</ref>。

== 站名来源 ==
东街口位于福州市中心，在唐代时始名左通衢，宋代改名凤池坊，明代俗称东街通衢，也称左通衢，由此定今名东街，此后将靠近八一七路的路口称为[[东街口|东街口]]。<ref>{{Cite web|url=http://fz.lanfw.com/show.php?contentid=278998|title=东街口为什么叫东街口？福州118个地名解读-福州蓝房网|accessdate=2017-01-07|work=fz.lanfw.com}}</ref>

== 车站结构 ==
东街口站为地下三层车站，地下一层为站厅层；地下二层为1号线站台层，岛式站台，全长196.2米，宽23.7米；地下三层为预留的4号线站台换乘段，岛式站台，已预留部分全长62米，宽25.1米。<ref name=":1"/>

=== 站厅 ===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 =  20170107 Concourse of Dongjiekou Station.jpg | caption1 = 站厅闸机口
| image2 =  20170107 Concourse 2 of Dongjiekou Station.jpg | caption2 = 站厅付费区
}}
东街口站站厅位于八一七路、杨桥路与东街的交叉路口下方，与东方百货和南街改造工程的地下一层相通。站厅为圆形站厅，装修采用白色波浪形吊顶，设置客服中心3处，售票机分布于站厅南北侧方形空间，部分售票机提供榕城通储值卡的充值功能，站厅另设有智能导乘机、自动售货机、自动取款机等服务设施，站厅北部区域设有[[洗手间|洗手间]]。

=== 周边 ===
东街口站周边主要为商业与住宅用地，车站所在位置即为东街口商圈，车站南侧对接[[八一七北路|南街]]地下商业街与东百中心；[[三坊七巷|三坊七巷景区]]可由C口出向西步行300米到达；福建省少年儿童图书馆可由A口出向东步行250米到达；福建省立医院可由A口出向东步行900米到达。

== 历史 ==
[[File:Derng-gar-kau.jpg|缩略图]]
* 2009年6月25日，福州市环境保护局发布了福州地铁1号线的环评报告，其中在八一七路与杨桥东路交叉路口设'''东街口站'''。<ref>
{{Cite web |url=http://www.fzhjbh.gov.cn/1254-38/affair_note/InfoShow.aspx |title=福州市轨道交通1号线工程环境影响报告书（简本） |access-date=2017-01-07 |archive-url=https://web.archive.org/web/20161018204222/http://www.fzhjbh.gov.cn/1254-38/affair_note/InfoShow.aspx |archive-date=2016-10-18 |dead-url=yes }} {{webarchive|url=https://web.archive.org/web/20161018204222/http://www.fzhjbh.gov.cn/1254-38/affair_note/InfoShow.aspx |date=2016-10-18 }}
</ref>
* 2011年4月29日，省发改委批复了福州地铁1号线（一期）的初步设计，其中在八一七路与杨桥路交叉路口设'''东街口站'''，并与福州地铁4号线实现十字换乘。<ref>
[http://www.fjdpc.gov.cn/zfshow.aspx?id=27917 福建省发展和改革委员会关于福州市轨道交通1号线一期工程初步设计的批复]{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}
</ref>
* 2010年4月28日，东街口站开始勘测。<ref>
{{Cite web |url=http://house.fznews.com.cn/news/jszx/2010-4-29/2010429GAnLApFSxx75920.shtml |title=地铁东街口站南门站 有望国庆前开建 |access-date=2016-08-11 |archive-url=https://archive.is/20160811042645/http://house.fznews.com.cn/news/jszx/2010-4-29/2010429GAnLApFSxx75920.shtml |archive-date=2016-08-11 |dead-url=yes }} {{webarchive|url=https://archive.is/20160811042645/http://house.fznews.com.cn/news/jszx/2010-4-29/2010429GAnLApFSxx75920.shtml |date=2016-08-11 }}
</ref>
* 2011年2月20日，为配合东街口站建设，拥有26年历史的东街口天桥开始拆除。<ref>[http://fj.sina.com.cn/news/s/2011-02-21/073692288.html 福州东街口天桥昨晚正式开始拆除]</ref>
* 2011年4月7日，东街口站1号线车站主体与4号线车站部分区域开始围挡施工。<ref name=":1">[http://www.fjsen.com/d/2012-11/15/content_9839433.htm 探访地铁1号线东街口站:主体结构已挖到地下16米]</ref><ref>[http://fj.sina.com.cn/news/m/2011-04-08/071896723.html 福州地铁东街口站9日起围挡施工]</ref>
* 2015年2月8日，1号线东街口站至屏山站区间隧道贯通。<ref>[http://fz.fjsen.com/2015-02/09/content_15664955.htm 福州地铁1号线屏山站至东街口站上行线昨贯通]</ref>
* 2015年4月24日，东街口站主体部分围挡拆除，开始进入附属结构施工。<ref>[http://fz.fjsen.com/2015-04/25/content_16000095.htm 福州地铁1号线东街口站 施工围挡一撤一扩]</ref>
* 2016年9月，东街口站至[[南门兜站|南门兜站]]沿线剩余围挡逐渐拆除。<ref>[http://news.163.com/16/0908/11/C0EH2OGB00014AEE.html 福州东街口至南门兜拆围挡 两侧共种百余棵樟树]</ref>
* 2017年1月6日，东街口站启用。<ref name=":0" />
* 2017年10月1日，东街口站西南侧站厅与C出入口试开放。

== 事故 ==
* 2011年11月14日，东街口站施工工地内一辆吊车突然侧翻，砸中工地外一辆小轿车，事故未造成人员伤亡。<ref>{{Cite web|url=http://news.xinhuanet.com/legal/2011-11/15/c_122283777.htm|title=福州地铁施工吊车侧翻 10米长吊臂砸中过往轿车(组图)_法治频道_新华网|accessdate=2017-04-03|work=news.xinhuanet.com}}</ref>
* 2013年12月17日，东街口站施工工地内一吊车吊臂突然断裂倒下，吊车旁一名工人被吊臂砸中当场身亡。<ref>{{Cite web|url=http://fj.qq.com/a/20131217/012873.htm#p=1|title=福州东街口地铁工地 吊臂断裂砸死一工人_大闽网_腾讯网|accessdate=2017-04-03|work=fj.qq.com|language=zh-CN}}</ref>
