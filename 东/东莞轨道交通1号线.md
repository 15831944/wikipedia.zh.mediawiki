'''東莞軌道交通1號線一期工程'''，又稱'''R1線'''或'''深圳地鐵6號線支線'''，是[[東莞軌道交通|東莞軌道交通]]繼[[东莞轨道交通2号线|2號線]]後開建的一條[[地鐵|地鐵]]線路。該線路原计划於2014年底開工，总工期为5年，但東莞段預計要2019年才獲准動工，預計最早要到2024年通車；<ref name=":0">{{cite web |title=东莞地铁1号线最新消息！设21个站点，有望在春节前后开始动工！ |url=https://mp.weixin.qq.com/s/xsvLThcADqeNqluW-75FCA}}</ref>而深圳段已於2018年1月10日動工，並將於2022年12月28日先行通車，最初兩年會先開通深圳境內的其中3個站（暫以中山大學站為終點，臨時控制中心亦設於此），並暫由深圳地鐵營運。一期全線通車後，全線將移交予中標公司營運，並聯通[[广州地铁|广州地铁]]和[[深圳地铁|深圳地铁]]線網。<ref name="最快2019年通车">{{Cite news |url=http://news.sun0769.com/dg/sh/201407/t20140722_4192961.shtml |title=R1线最快2019年通车 市民去广深一小时搞定 |publisher=东莞阳光网 |date=2014-07-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20140726185024/http://news.sun0769.com/dg/sh/201407/t20140722_4192961.shtml |archivedate=2014-07-26 }}</ref>直至2018年4月，東莞市政府正式宣佈將此項目納入「美麗東莞建設」三年期建設工程中，並將本線26年專營權以[[民間興建營運後轉移模式|民間興建營運後轉移模式]]批出予中標財團，最快9月動工<ref>[http://www.gaoloumi.com/forum.php?mod=viewthread&tid=3118958&extra=page%3D1 高樓迷論壇東莞版-東莞軌道交通1號線最新消息]</ref>，及後再改於10月招標，2019年初動工。

== 規劃 ==
R1線原規劃為用於連接[[广州市|廣]][[深圳市|深]]、只設數個站點的快線，後來經過數次修改，曾一度增至24站，至申报[[环境影响评价|环评]]审批阶段時縮減為21站，后於2017年尾调整至27座（若包括[[光明區|光明區]]路段，將設站31個）<ref>{{Cite news |url=http://news.timedg.com/2013-09/05/content_14075315.htm |title=东莞地铁1号线少了11公里和3站点 |publisher=东莞时报 |date=2013-09-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20140728040429/http://news.timedg.com/2013-09/05/content_14075315.htm |archivedate=2014-07-28 }}</ref><ref>{{Cite news |url=http://www.dggdjt.com/publicfiles//business/htmlfiles/gdjt/zxdt/201712/1783.html |title=东莞市城市轨道交通线网规划调整（2017-2020年）公示 |publisher=东莞市轨道交通有限公司 |date=2017-12-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20140728040429/http://www.dggdjt.com/publicfiles//business/htmlfiles/gdjt/zxdt/201712/1783.html |archivedate=2017-12-25 }}</ref>。此外，[[深圳地鐵6號線|深圳地鐵6號線]][[翠湖站（深圳）|翠湖站]]亦已預留條件予本線作站廳接駁<ref>贏商網-[http://sz.winshang.com/news-522927.html 深圳东莞地铁实现无缝接驳 四条接驳线路初定]</ref>，並將有聯絡線連接兩線。當中在光明區路段（以東深市界為分界線，該線位於黃江南站與科學城東站間明挖回填段隧道內YAK6+130.000里程處），將由[[深圳市地鐵集團|深圳市地鐵集團]]出資建造及擁有，稱為深圳地鐵6號線支線。

== 線路 ==
R1线全長约64.4公里，其中東莞段內高架线25.8公里，地下线30.3公里，地面线及过渡段1.9公里；深圳段則以長5.3公里的隧道（主要以[[隧道鑽挖機|隧道鑽挖機]]建造，其次為明挖回填）為主，末端0.8公里為地面及架空路段。線路起于西端[[望牛墩|望牛墩]][[望洪路站|望洪路站]]，止于深圳[[光明區|光明區]][[翠湖站（深圳）|翠湖站]]，設站31個，分別是：望洪路站、道滘站、道滘东站、人民医院站、汽车总站、滨江体育馆站、莞太路站、[[鸿福路站|鸿福路站]]（本站為與[[東莞軌道交通2號線|東莞軌道交通2號線]]的轉車站，同時設有聯絡線接駁2號線）、新源路站、东城南站、水濂山站、大岭山北站、大岭山站、大岭山东站、松山湖站、大朗西站、大朗站、湿地公园站、黄江北站、黄牛埔站、黄江中心站、黃江南站、武漢大學站、中山大學站（[[中山大學深圳校區|中山大學深圳校區]]）、新明醫院站、翠湖站及光明中心站（預留）（最後五個站位於深圳；其中翠湖站位於地面，其餘為地下車站）。<ref name="最快2019年通车"/><ref>[https://mp.weixin.qq.com/s/PccagCHElCrzGBjOOznNMw?fbclid=IwAR1xVFEj3jwGR95Qv6hAauLEKX5trVtUUeQ-LpUC-HTpoWaJHAT46fSEb7E 2019迎来“地铁年”，两条线路有望年底通车！附深圳在建地铁工程进展]</ref>

線路西北端將與[[广州地铁5号线|广州地铁5号线]]延長段在[[黄埔客运港站|黄埔客运港站]]接駁<ref>{{Cite news|url=http://www.gd.xinhuanet.com/newscenter/2014-07/04/c_1111459413.htm|title=广州五号线有望接驳东莞一号线|publisher=南方日报|date=2014-07-04|deadurl=yes|archiveurl=https://web.archive.org/web/20140727051550/http://www.gd.xinhuanet.com/newscenter/2014-07/04/c_1111459413.htm|archivedate=2014-07-27}}</ref>，而東南端將以「深圳地铁6号线支線」名義延入深圳，在翠湖站與[[深圳地铁6号线|深圳地铁6号线]]接駁。

在支援設施方面，本線設有兩座車廠（位於[[道滘鎮|道滘鎮]]污水處理廠以南的道滘車輛段，以及鄰近黃江中心站、預留與4號線共用的黃江停車場），以及四座牽引變電站（旗峰變電站已於2號線工程中建成，與本線共用電力）。在東莞境內的車輛段2024年完工前，列車會暫配屬於深圳地鐵6號線的長圳車輛段，並經位於翠湖站的聯絡線（由6號線北行軌道連接本線南行軌道，設有第三軌供電-架空電纜間[[中性區|中性區]]）進入正線。

1號線支線為遠期規劃，預計在原1號線濕地公園站至黃江北站之間，新增富民南路站分拆支線。支線途徑大朗鎮，黃江，樟木頭，塘廈，鳳崗，到達深圳邊界與[[深圳地鐵22號線|深圳地鐵22號線]]對接。1號線支線也會接駁未來[[赣深客运专线|贛深客運專線]]的東莞南站。<ref>{{Cite web|url=http://www.timedg.com/2018-12/15/20780591.shtml|title=东莞市轨道交通1号线支线有望接驳赣深高铁东莞南站|accessdate=|author=|date=|publisher=}}</ref>

==建設進展==
深圳市部分已於2017年11月展開工程單一招標（由[[中國交建|中國交建]]中標），並於2018年1月10日動工<ref>[http://shenzhen.sysu.edu.cn/content/304 中山大學深圳校區-深圳新聞：大手笔！明日深圳5条新地铁齐开工，6号线直通中大深圳校区]</ref>；直至2018年6月，此段前期工程已經展開。而在同年9月27日，中大站開展連續牆工程，是東莞軌道交通1號線首個展開主體工程的車站。

東莞方面，則於2018年10月初展開[[民間興建營運後轉移模式|民間興建營運後轉移模式]]招標工作，同時為望洪路至水濂山站施工監理合約進行招標。獲准投標BOT的公司包括：東莞發展控股（東莞市政府附屬公司）-[[中國中鐵|中國中鐵]]-[[中國鐵建|中國鐵建]]等聯營、北京建工集團，以及宏润建设集团（寧波市政府關聯公司）。本線26年BOT專營權已於2019年1月10日批予東莞發展控股-中國中鐵-中國鐵建等聯營公司（並由中國中鐵-中國鐵建負責東莞段全部土木及機電工程），2019年2月20日麻涌至黃江中心站區段動工（但黃江南站至科學城東站段，則於稍後動工），2024年通車。

2019年3月18日，濱江體育公園站亦展開連續牆工程，成為全線第二個及東莞段首個主體動工的車站。

== 使用列车 ==

东莞轨道交通一号线采用6节编组的CIVAS B型车<ref name="dggj">{{cite web|url=http://www.dggdjt.com/publicfiles/business/htmlfiles/gdjt/regcgk/201104/409.html|title=R2线工程概况|publisher=东莞轨道交通有限公司|date=2011年4月11日|language=zh-cn|access-date=2018-01-26|archive-url=https://web.archive.org/web/20150923214109/http://www.dggdjt.com/publicfiles/business/htmlfiles/gdjt/regcgk/201104/409.html|archive-date=2015-09-23|dead-url=yes}}</ref>，列车和轨道线路设计最高运行速度是120km/h<ref>{{cite news|url=http://news.timedg.com/2011-03/25/content_3153916.htm|title=轨道交通 构建幸福东莞快车道|publisher=东莞日报|date=2011年3月25日|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20110824040638/http://news.timedg.com/2011-03/25/content_3153916.htm|archivedate=2011年8月24日}}</ref>。一、二期工程全线配备20列列车，每节车厢每侧设有3个车门，由南京[[中車南京浦鎮車輛|中车浦镇]]城轨车辆有限责任公司负责制造。预计一、二期工程通车后平均旅行速度（含停站和折返时间）将不小于55km/h。首列列车已于2014年10月30日使用平板车运抵东城车辆段<ref>[http://news.timedg.com/2014-10/30/14959631.shtml {{Wayback|url=http://news.timedg.com/2014-10/30/14959631.shtml |date=20160310134555 }} 东莞进入地铁时代 带你与首列车亲密接触[组图]],东莞时间网</ref>。將來，部分列車亦會調至本線上使用。

== 功能作用 ==
R1线建成后，可贯通广深线网，东莞市民乘坐地铁到广州或深圳只需1小时左右。<ref name="最快2019年通车">{{Cite news |url=http://news.sun0769.com/dg/sh/201407/t20140722_4192961.shtml |title=R1线最快2019年通车 市民去广深一小时搞定 |publisher=东莞阳光网 |date=2014-07-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20140726185024/http://news.sun0769.com/dg/sh/201407/t20140722_4192961.shtml |archivedate=2014-07-26 }}</ref>

== 大事記 ==
*2017年1月14日，東莞軌道交通1號線通過廣東省住建廳專家初步審核。
*2017年4月12日，1號線收地動員大會舉行，象徵相關土地收回工作展開。
*2017年尾，東莞軌道交通第二輪建設正式訂定，本線在東莞境內站點確定。
*2018年1月10日，作為深圳地鐵第四期工程一部分的東莞軌道交通1號線深圳段動工。
*2018年4月，東莞市政府決定加快本線東莞段建設，並將以[[BOT模式|BOT模式]]批出本線建造及營運權。
*2019年2月，東莞軌道交通1號線東莞段動工。<ref name=":0" />
