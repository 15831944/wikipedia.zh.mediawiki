'''东宝·仲巴·白玛塔清'''（{{bd|1968年|12月||}}）俗名'''泽仁旺杰'''，[[藏族|藏族]]，[[四川省|四川省]][[甘孜藏族自治州|甘孜藏族自治州]][[九龙县|九龙县]]人，第十七世[[东宝·仲巴活佛|东宝·仲巴活佛]]<ref name=fjzx>[http://picture.fjnet.com/fjnw/201003/t20100317_150126.htm 东宝·仲巴呼图克图仁波切，佛教在线，2010-03-17]</ref><ref name=foxie>{{Cite web |url = http://www.chinabuddhism.com.cn/djc/jjlsh/2015-04-25/8636.html |title =东宝·仲巴呼图克图  |author =  |date = 2010-03-17 |publisher = 中国佛教协会 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
[[藏传佛教|藏传佛教]][[噶玛噶举派|噶玛噶举派]]第十六世东宝·仲巴活佛圆寂后，[[甘孜藏族自治州|甘孜藏族自治州]][[稻城县|稻城县]][[邦坡寺|邦坡寺]]高僧、第十六世东宝·仲巴活佛的经师八图喇嘛得知其圆寂的消息，乃祈请其早日转世，并请[[第十六世噶玛巴|第十六世噶玛巴]]在正观中观察东宝·仲巴活佛何时转世，转世于何地。1970年某日，[[第十六世噶玛巴|第十六世噶玛巴]]召来八图喇嘛，告之东宝·仲巴活佛已转世于[[康区|康区]]南部山中一个两河交汇之所，他家后面为高山，屋前有一株古柏，他名字的藏文首字母为“嚓”。当时，因处于[[文化大革命|文化大革命]]期间，转世灵童的寻访工作只能搁置，等待将来回中国寻访的机会<ref name=xdjy>[http://www.myfo.org/sf_57D92AF46EB14A3B8423E707E31AC071_101_lingyu.html 第十七世东宝·仲巴白玛塔清，心地家园，2006-9-6]</ref>。

[[四川省|四川省]][[九龙县|九龙县]]内的呷尔河两侧群山交错，在呷尔河谷中部形成一巨大的海螺状，海螺顶在上游，海螺口朝向下游。在“海螺口”处的山梁之上，有“华丘村”（藏语，意为“大雁背上的铺子”），为当年第十六世东宝·仲巴活佛在九龙县驻锡之所，村民在此为其修建了[[华丘寺|华丘寺]]。在“海螺”中部距[[呷尔河|呷尔河]]不远的平坝上，有一座[[藏族|藏族]]为主的自然村查尔村。查尔村东部有[[嚓尔河|嚓尔河]]（又译“查尔河”）。嚓尔河自北向南流入呷尔河，汇合后流向[[雅砻江|雅砻江]]。嚓尔村位于[[四川省|四川省]][[九龙县|九龙县]]内[[嚓尔河|嚓尔河]]同[[呷尔河|呷尔河]]（又称“[[九龙河|九龙河]]”）交汇之处<ref name=xdjy/>。

泽仁旺杰生于嚓尔村。其父名叫央中扎西，汉姓王，是[[苯教|苯教]]帕布派（白苯）第十九代掌门人，擅长风水术、咒术。央中扎西与第十六世东宝·仲巴活佛有深交，受后者感化，遂在其苯教传承中融入佛教思想。央中扎西的妻子达瓦志玛是[[噶玛噶举派|噶玛噶举派]][[贡噶活佛|贡噶活佛]]的侄女。泽仁旺杰家有八个兄弟，泽仁旺杰排行第四<ref name=xdjy/>。

泽仁旺杰3岁时，[[阿热寺|阿热寺]]的[[根桑·彭措|根桑·彭措]]秘密为其[[灌顶|灌顶]]。此后十年，泽仁旺杰秘密跟随根桑·彭措学习文化及密法。后来，泽仁旺杰又依止瑜伽大师[[阿尼吾金|阿尼吾金]]学习[[空行母|空行母]]之密法，获得“空行密中密”传承。在朝拜大善知识[[喜绕俄热|喜绕俄热]]时，泽仁旺杰获赐法名“白玛塔清”。他还被大修行者[[噶玛顿珠|噶玛顿珠]]收为入室弟子，秘密学习佛法近十年<ref name=xdjy/>。1975年到1982年，他在四川省甘孜藏族自治州九龙县读书。1983年到1989年，在[[中国人民解放军|中国人民解放军]]35101部队服役。1989年8月正式出家<ref name=foxie/>。

1991年，一直在海外的[[大司徒活佛|大司徒活佛]]回到中国，在[[德格|德格]][[八蚌寺|八蚌寺]]举办了为期三个多月的传法灌顶法事，期间还认定了几位[[活佛|活佛]]的[[转世灵童|转世灵童]]。当时，[[贡噶活佛|贡噶活佛]]的转世灵童一直未找到，贡噶活佛转世灵童寻访小组通过多方考证，认为泽仁旺杰为贡噶活佛的转世，乃将其请至八蚌寺，请大司徒活佛确认。大司徒活佛见到泽仁旺杰之后，认为泽仁旺杰为东宝·仲巴活佛的转世，并当即为泽仁旺杰赐法名“噶玛德列丹增”<ref name=xdjy/>（一说大司徒活佛赐法名“噶玛益西尼玛赤列土登宁杰白桑布”<ref name=fjzx/>）。

当时，[[四川省|四川省]][[稻城县|稻城县]][[邦坡寺|邦坡寺]]和[[卓杰寺|卓杰寺]]的许多喇嘛均来[[八蚌寺|八蚌寺]]参加大司徒活佛的法会，得知大司徒活佛认证了东宝·仲巴活佛的转世，便纷纷求见。[[邦坡寺|邦坡寺]]和[[卓杰寺|卓杰寺]]的大喇嘛依照[[噶玛噶举派|噶玛噶举派]]活佛转世仪轨，对照第十六世东宝·仲巴活佛的预言，验证了泽仁旺杰的肉身灵异胎记，并进行了传统的考察，证明大司徒活佛认证无误，泽仁旺杰确系第十六世东宝·仲巴活佛的转世。泽仁旺杰依止[[大司徒活佛|大司徒活佛]]为根本上师，在八蚌寺接受一系列灌顶，获得历代东宝·仲巴活佛所持的大部分教法传承<ref name=xdjy/>。

1991年4月，[[稻城县|稻城县]]邦坡寺、卓杰寺的喇嘛将泽仁旺杰迎请至卓杰寺，由邦坡寺、卓杰寺的僧团代表大司徒活佛为他举行了坐床大典，他由此正式成为第十七世东宝·仲巴活佛，并成为邦坡寺、卓杰寺的寺主<ref name=fjzx/><ref name=xdjy/>。

经[[云南省|云南省]][[大理州|大理州]][[宾川县|宾川县]]人民政府批准，1993年初，东宝·仲巴活佛率弟子到[[鸡足山|鸡足山]]，为重建[[石钟寺|石钟寺]]奠基。当时，东宝·仲巴活佛的身份虽然已为教派内部认可，但尚未获得政府确认<ref name=xdjy/>。

后来，[[云南省|云南省]][[丽江市|丽江市]]人民政府宗教部门以及广大僧俗信众请求第十七世噶玛巴证实第十七世东宝·仲巴活佛的身份。1995年藏历六月，[[第十七世噶玛巴|第十七世噶玛巴]]下文确认泽仁旺杰为第十六世东宝·仲巴活佛的转世，并赐法名“南喀嘉措噶玛德列土登宁杰白桑布”，还颁发了认证书。[[宁玛派|宁玛派]]大成就者[[夏扎·桑杰多杰|夏扎·桑杰多杰]]活佛也在此期间发表讲话，同意噶玛巴及大司徒活佛的认证，并称要给予第十七世东宝·仲巴活佛共与不共的教法传承<ref name=xdjy/>。

1995年起，东宝·仲巴活佛数次赴[[印度|印度]]、[[尼泊尔|尼泊尔]]、[[不丹|不丹]]、[[锡金|锡金]]等圣地朝拜，并向大司徒活佛、[[杰曹活佛|杰曹活佛]]、夏扎活佛等请求[[噶玛噶举派|噶玛噶举派]]和[[宁玛派|宁玛派]]教法的[[灌顶|灌顶]]、口诀及教授，同时还拜会了[[萨迦派|萨迦派]]、[[格鲁派|格鲁派]]教法持有者，从而获得了[[藏传佛教|藏传佛教]]各教派的教法传承<ref name=xdjy/>。

1999年，[[甘孜藏族自治州|甘孜藏族自治州]]宗教局颁发证书，聘任东宝·仲巴活佛为邦坡寺、卓杰寺住持<ref name=xdjy/>。

1998年、2001年，东宝·仲巴活佛先后当选为第九届[[中甸县|中甸县]]政协常委，第十届[[迪庆藏族自治州|迪庆藏族自治州]]人大代表、委员<ref name=xdjy/>。2000年至2003年，经地方宗教部门推荐，[[云南省|云南省]]党政有关部门讨论并通过决议，推荐东宝·仲巴活佛到[[北京|北京]]的[[中国藏语系高级佛学院|中国藏语系高级佛学院]]学习<ref name=xdjy/><ref name=foxie/>。

2001年，依照历史沿革，东宝·仲巴活佛驻锡[[云南|云南]][[丽江|丽江]][[指云寺|指云寺]]，任丽江地区十三大寺教主<ref name=fjzx/>。2001年，东宝·仲巴活佛再次受到[[噶玛噶举派|噶玛噶举派]]摄政[[大司徒活佛|大司徒活佛]]、三宝法王[[嘉察活佛|嘉察活佛]]和[[宁玛派|宁玛派]]大圆满持有者[[夏扎活佛|夏扎活佛]]的[[灌顶|灌顶]]和教授，并在夏扎活佛座下闭关三个月。由于其教证圆通，获[[第十七世噶玛巴|第十七世噶玛巴]]、四宝法王[[大司徒活佛|大司徒活佛]]、三宝法王[[嘉察活佛|嘉察活佛]]委任，管理[[噶玛噶举派|噶玛噶举]]中国事务，颁发委任书并赐法印<ref name=xdjy/>。

2006年4月，当选为丽江市佛协副会长。2008年1月，被任命为丽江市政协副主席。2009年，当选为丽江市佛协会长<ref name=foxie/>。

2010年2月，东宝·仲巴活佛当选为[[中国佛教协会|中国佛教协会]]第八届理事会常务理事<ref name=fjzx/>。2015年4月，在中国佛教协会第九次全国代表会议上当选为第九届理事会副会长<ref name=foxie/>。他是第十、十一、十二届全国政协委员<ref>{{cite news|title=第十一届全国政协委员名单|url=http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|accessdate=2013-04-09|newspaper=新华社北京1月25日电 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过)|date=2008年1月25日|deadurl=yes|archiveurl=https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|archivedate=2013年9月28日}}</ref>，代表宗教界，分入第五十一组。<ref>{{cite news|title=全国政协信息-东宝仲巴·呼图克图|url=http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0018080&wytype=wjwy|accessdate=2013-04-09|newspaper=全国政协委员会}}</ref>。
