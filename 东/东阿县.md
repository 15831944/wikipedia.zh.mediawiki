'''东{{ruby-py|阿|-{zh-cn:ē;zh-tw:ㄜ;zh-hk:o<sup>1</sup>}-}}县'''位于[[中华人民共和国|中华人民共和国]][[山东省|山东省]]西部、[[黄河|黄河]]北岸，是[[聊城市|聊城市]]所辖的一个[[县_(中华人民共和国)|县]]。

==历史==
东阿置邑，始见《[[春秋_(史书)|春秋]]》，名柯；原属[[卫国|卫国]]，后属[[齐国|齐国]]。[[战国|战国]]改称阿，仍隶齐国。[[秦朝|秦]]始称东阿，属东郡。旧志称：“东阿，古之名邑也，会盟征战，废垒遗墟，见于经史者不一而足。”

汉置东阿县，属东郡。

历经两千多年，东阿县多次更名，隶属关系也多次变动。隶属关系变更除因改朝换代外，黄河多次改道也影响了东阿县隶属关系变动。

1947年冬，县机关迁驻铜城。

1949年8月，原属东阿黄河以东村镇划归平阴县，西南张秋一带划归阳谷县，原属平阴河西部分、聊城的部分地域并入东阿，茌平与东阿交错为邻的村庄，以赵牛河为界北归茌平、南属东阿。

新中国成立，作为聊城地区下辖县归平原省管辖。1952年，平原省撤销，聊城全部划归山东省管辖。

==地理位置==

东阿县位于山东省聊城市东南部，地形地貌特点为黄河下游冲击平原。正东方向与山东省省会济南市下属平阴县隔黄河相望，东南方向与泰安市东平县隔黄河相望，西面与聊城市东昌府区相邻，北面与聊城市茌平县相邻，东北方向与德州市齐河县相邻，西南与聊城市阳谷县相邻。黄河位于东阿县东南部，黄河也为东阿县与东平县、平阴县的分界河。地处东经116°02′至116°33′，北纬36°07′至36°33′。土地总面积729平方千米。

==经济==

聊城市为山东省最穷的市之一。作为聊城市辖县，东阿县整体比较贫穷。县城内人均收入2000多元/月，农村平均每月收入不过1000元/月。东阿县整体工业经济薄弱，全县只有东阿阿胶一家大型企业，东阿县对其税收并没有管辖权，税收全部上交聊城市。这使得东阿县财政匮乏。由于东阿县委、县政府领导由聊城市领导任命，为了自己的官位，东阿县领导对于这一安排敢怒不敢言。<br>

东阿县政府官网公布：2017年上半年，全县生产总值完成109.2亿元，增长7.5%；社会消费品零售总额达到21亿元，增长9.9%；公共财政预算收入完成7.1亿元，同口径增长5.38%；固定资产投资完成39.78亿元，增长17%；城乡居民人均可支配收入分别达到11626元、6283元，增长8.3%和8.9%。<ref>{{cite web |title=县情介绍 |url=http://www.sdde.gov.cn/zjde/xqjs/ |website=东阿县人民政府 |accessdate=2018-09-07}}</ref>该数据真实性存疑。

东阿县房价高企不下，大多数房子价格在6000元/平方米左右，房地产泡沫明显。这与东阿的个人收入严重倒挂。年轻人为了准备婚房，耗尽了夫妻二人和双方父母的积蓄。

==语言==

东阿本地话属于冀鲁官话。东阿县西南刘集镇少数居民也说中原官话。

==民族==

全县绝大部分人口为汉族，极少数为其它少数民族。

==气候==

东阿县气候属于典型的温带季风气候。夏季高温多雨，冬季寒冷干燥、多西北风，季风性特点显著。最低温度通常出现在每年元月份，可达-10°C以下；最高温度通常出现在每年七月份，可达35°C以上。<br>

冬春季存有寒潮、沙尘暴、霜冻等有害天气；夏季诸如雷雨、大风、冰雹强对流天气时有发生。<br>

自2011-01-01到2018-07-01，东阿共出现多云1383天，晴743天，雨417天，阴76天，雪48天，10天气象资料不详。<ref>{{cite web |title=东阿历史天气 |url=https://lishi.tianqi.com/donge/index.html |website=天气网}}</ref>

==行政区划==
东阿县辖1个省级工业园区、2个街道、7个镇、1个乡：
*街道：铜城街道、新城街道
*镇：牛角店镇、刘集镇、大桥镇、高集镇、姜楼镇、姚寨镇、鱼山镇
*乡：陈集乡

顾官屯乡原属东阿县，与聊城市东昌府区为邻。大型化工企业[[鲁西化工|鲁西化工]]主要位于顾官屯乡。聊城市领导极其看重鲁西化工税收并想将该企业税收划归市有，故向山东省委、省政府提出将顾官屯乡划入东昌府区（但未向省领导汇报鲁西化工就位于该乡，也没有说明鲁西化工税收大户）。山东省领导不知道聊城市真正用意，只是认为这是正常的行政管辖权变更，同意了这一请求。由此，2010年7月，顾官屯乡脱离东阿县，划归东昌府区。

==特产== 
*[[东阿阿胶|东阿阿胶]]：为中医药文化的代表，以独特的医药保健价值及浓厚的文化底蕴闻名。阿胶在汉唐至明清一直为贡品。李时珍《[[本草纲目|本草纲目]]》载：“阿胶，本经上品，弘景曰出东阿故名阿胶”。

==旅游资源==
'''曹植墓'''：曹植，是三国时期魏国著名文学家、曹操三子。曹植墓位于山东省聊城市东阿县鱼山镇鱼山村，依山而建。据《三国志-魏志-任城陈萧王传十九》记载：“初，植登鱼山，临东阿，喟然有终焉之心，遂营为墓。”<ref>{{cite web |author1=陈寿 |title=三国志-魏志-任城陈萧王传十九 |url=http://www.guoxue.com/shibu/24shi/sangzz/sgzz_019.htm |website=国学网 |accessdate=2018-11-15}}</ref>，这是有关曹植墓位置最为权威的记载。<br>

曹植墓室分甬道、前室、后室3部分。1951年，平原省文物管理委员会曾清理出土文物132件，其中除 玛瑙珠、玛瑙泡、青玉璜数件较精外，其他大部为陶制明器，象车、案、壶、盆、鸡、狗、鹅、鸭之类的文物。<br>

1996年11月，曹植墓被国务院公布为全国重点文物保护单位。

1998年，山东省文物局拨出专款，对曹植墓进行了修建，建成了现在的陵园。<br>

程咬金（589年－665年2月26日），后更名知节，济州东阿（今山东东阿）人。<ref>{{cite web |author1=苏轼，宋祁 |title=新唐书-卷九十 列传第十五 |url=http://www.guoxue123.com/shibu/0101/00xts/102.htm |website=国学导航 |accessdate=2018-11-14}}</ref>唐朝开国大将，凌烟阁二十四功臣之一。
