'''東凛'''（{{lang-ja|東凛}}，1986年7月21日 - ），[[日本|日本]]的[[AV女優|AV女優]]<ref name="livedoor"/>。目前為「{{Link-jp|溜池ゴロー|溜池ゴロー (アダルトビデオ)}}」專屬女優<ref>{{Cite web|url=http://www.tameikegoro.jp/actress/-/detail/=/id=1025776/ |title=AV女優【東凛】|publisher={{Link-jp|溜池ゴロー|溜池ゴロー (アダルトビデオ)}} |accessdate=2018-08-02}}</ref>，所属于「{{link-ja|Diaz Group|ディアスグループ}}」事務所<ref>{{Cite web|url=http://diaz-g.com/model/ |title=所属モデル |publisher={{link-ja|Diaz Group|ディアスグループ}}|accessdate=2018-08-02}}</ref>。

== 人物 ==
*興趣是喝[[紅茶|紅茶]]還有{{Link-jp|醃醬菜|糠漬け}}<ref name="livedoor"/>，擅长[[舞蹈|舞蹈]]、[[歌曲|歌曲]]和[[妄想|妄想]]<ref name="livedoor"/><ref>{{Cite news|url=http://www.chinatimes.com/realtimenews/20160827000002-260404 |title=正港辣人妻 AV女優東凜就是不讓你「一步到位」 |newspaper=[[中時電子報|中時電子報]] |date=2016-08-27}}</ref>。

*在3歲的時候成為[[童星|童星]]，從此開始以[[女演員|女演員]]以及[[日本電視藝人|藝人]]身分活躍著，即使在結婚之後仍然從事這樣的工作<ref name="第1回">{{Cite web |url=https://news.dmm.co.jp/article/938871/ |title=リアル人妻AV女優・東凛インタビュー第1回 初めての野外露出に爽快感！？ |work=FANZAニュース |publisher=[[DMM.com|DMM]] |date=2015-04-07 |accessdate=2018-09-19 |language=ja}}</ref>。

*她表示想要趁自己的身體還很漂亮的時候拍攝[[裸體攝影|裸體攝影]]，後來也被介紹給了[[AV監督|AV監督]]{{link-ja|溜池GORO|溜池ゴロー}}，並在監督熱情的決定之下拍攝了寫真DVD<ref name="第1回"/>。

*雖然與丈夫已經結婚六年了，但是她並沒有告訴對方自己是一名AV女優<ref>{{Cite web |url=https://news.dmm.co.jp/article/941753/ |title=主人にナイショという罪悪感が、私を淫らにする。【東凛インタビュー第2回】 |work=FANZAニュース |publisher=DMM |date=2015-04-14 |accessdate=2018-09-19 |language=ja}}</ref>。

== 簡歷 ==
2014年5月，发行成人{{Link-jp|寫真DVD|イメージビデオ}}<ref>{{cite web|url=http://www.playno1.com/article-13704-1.html#.Wvh80oiFOUk |title=這個人妻好辣 - 東凛 |publisher=AVNo.1-PLAYNO.1 |date=2014-06-23}}</ref>，以[[人妻|人妻]]的身份[[出道|出道]]。同年9月成为「{{Link-jp|溜池ゴロー|溜池ゴロー (アダルトビデオ)}}」专属AV女优<ref>{{cite web|url=http://www.playno1.com/article-14015-1.html#.W7XPuGgzaUk |title=結婚5年的人妻東凛被男優插進去了! |publisher=AVNo.1-PLAYNO.1 |date=2014-080-12}}</ref>。

2015年5月5日，她獲得《[[DMM成人獎|DMM成人獎]]》2015「特別主持人獎」<ref>{{cite web|url=http://www.dmm.co.jp/adultaward/archive/winner/2015_html/=/navi=none/ |title=DMM.R18アダルトアワード2015受賞結果 |publisher=[[DMM.com|DMM.com]] |date=2015-05-05 |language=ja}}</ref><ref>{{cite web|url=https://nikkan-spa.jp/846408 |title=No.1セクシー女優は誰の手に!? AV女優の祭典［DMM.R18 アダルトアワード2015］写真レポ |publisher={{link-ja|日刊SPA!|SPA!}} |date=2015-05-05 |language=ja}}</ref>。

2017年6月，一度傳出因身分差點曝光而暫停所有公開活動<ref>{{cite web|url=http://www.playno1.com/article-23527-1.html#.W9GiNGgzaUk |title=東凛、即將消滅？ |publisher=AVNo.1-PLAYNO.1 |date=2017-06-02}}</ref>。

2018年10月23日，相隔6個月之後再度更新個人官方部落格<ref>[https://twitter.com/Rin_Azuma0721/status/1054390600278007809 @Rin_Azuma0721 下午11:14 - 2018年10月22日]</ref>。
