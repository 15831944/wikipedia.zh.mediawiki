'''宾俊宅'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[礼士胡同|礼士胡同]]129号，是[[北京市文物保护单位|北京市文物保护单位]]。

==历史==
此宅位于礼士胡同西部，北靠[[前炒面胡同|前炒面胡同]]。原是[[清朝|清朝]]末年[[武昌府|武昌府]]知府[[宾俊|宾俊]]的宅邸，后来日伪时期被其子锡琅出售给投机米商李彦青，后又为律师汪颖所有。不久转给[[天津|天津]]某盐商之子李颂臣，经过改建成如今规模。[[中华人民共和国|中华人民共和国]]成立后，此宅曾作为[[印度尼西亚|印度尼西亚]]驻华大使馆，后曾为[[中国青年报|中国青年报]]社社址。此后为单位办公使用<ref name=dhly>{{Cite web |url = http://www.bjdclib.com/subdb/laneculture/historybuilding/200908/t20090801_14686.html |title = 明瑞宅 |author =  |date = 2009-08-01 |publisher = 东华流韵 |language =  |accessdate =  |quote =  }}</ref>。

1984年5月24日公布为[[北京市文物保护单位|北京市文物保护单位]]<ref name=bjwb>{{Cite web |url =  http://www.bjww.gov.cn/wbsj/bjwbdwDetail.htm|title =东城区礼士胡同一百二十九号四合院  |author =  |date =  |publisher = 北京文博 |language =  |accessdate = 2018-02-20 |quote =  }}</ref>。 

==建筑==
此宅坐北朝南，由住宅和花园两部分组成，占地面积约1200平房米。现存建筑包括<ref name=dhly/>：
*大门：位于东南端，为[[广亮大门|广亮大门]]一间，建在五级双垂带踏跺之上，[[硬山顶|硬山顶]]筒瓦过垄脊屋面，檐柱带有[[雀替|雀替]]，双扇红板门上带有四枚雕花门簪。门外廊心墙、戗檐、门两侧八字墙、沿街院墙都有砖雕图案<ref name=dhly/>。
*倒座房：大门内两侧倒座房各两间，东连后添建的车库两间，西侧是新开的大门，西有临街倒座房五间。
*东院和西院：一进院内，北面为两个并列的四合院<ref name=dhly/>。
**东院：一殿一卷式垂花门建在五级台级上，门左右有一对石兽，两侧看面墙上有什锦窗，砖雕窗框。门内正房三间，东、西厢房各三间，各房都带前廊。各房之间有[[抄手游廊|抄手游廊]]连接，廊子带有倒挂楣子和坐凳。后院有六间北房。1986年在东南院墙处添建一座二柱三楼牌楼，上面绘有苏式彩画。过牌楼向北有一组四合院坐西朝东，院门是一殿一卷式垂花门，门前有坐东朝西的影壁，影壁正中雕有条幅，四角雕有高浮雕岔角花纹。院内南、北、西三面房屋各三间，西房是两卷勾连搭的过厅，可通向西侧的花园，抄手游廊环绕四隅<ref name=dhly/>。
**西院：与东院毗邻，有带前后廊的南房三间是过厅，也是西院的院门。北房五间，带前后廊。东厢房也是过厅，与东院的西厢连接。该院西侧有单披顶游廊，可通向花园。东、西两院正房之间有座重檐顶圆亭，四面有门廊道与东、西、南、北各房间相连<ref name=dhly/>。
*花园：位于宅院西北部，面积小，有水池、假山、树木、花草。东北角有一座单檐八角亭，顶覆绿琉璃瓦<ref name=dhly/>。

此宅院各房都是硬山顶清水脊覆合瓦屋面，墙体都是磨砖对缝。此宅院虽是中华民国时期改建，但建筑完整、布局紧凑。砖雕尤其精致，特别是正房、厢房的廊门走马板上的砖雕匾额，有“撷秀”、“抗风”、“舒华”、“蕴秀”、“竹幽”、“含珠”、“隐玉”、“摘芳”、“拧月”等等<ref name=dhly/>。
