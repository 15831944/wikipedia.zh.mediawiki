'''宾阳县'''（[[邮政式拼音|邮政式拼音]]：Pinyang）位于[[中国|中国]][[广西壮族自治区|广西壮族自治区]]中南部，[[南宁市|南宁市]]东北部，是[[南宁市|南宁市]]所辖的一个[[县|县]]。县城[[宾州镇_(宾阳县)|宾州镇]]距南宁市城区公路里程90公里。<ref name="BYGK">{{cite news|url=http://www.gx.xinhuanet.com/2012-11/26/c_113803973.htm|title=宾阳概况|publisher=新华网|date=2012年11月26日|accessdate=2014年3月7日|language=zh-cn}}</ref>

== 地理 ==
县境介于北纬22°54′～23°27′，东经108°32′～109°15′之间。东与[[贵港市|贵港市]]交界，南接[[横县|横县]]、[[青秀区|青秀区]]，西连[[武鸣区|武鸣区]]、[[兴宁区|兴宁区]]，西北衔接[[上林县|上林县]]，北与上林县及[[来宾市|来宾市]]为邻。<ref name="BYGK"/>

== 历史沿革 ==
今宾阳县境，古为[[百越|百越]][[西瓯|西瓯]]之地。[[夏朝|夏]]、[[商朝|商]]、[[周朝|周]]为[[荆州_(古代)|荆州]]地。[[秦朝|秦]]一统后（公元前214年），地属[[鬱林郡|桂林郡]]。[[西汉|西汉]]时属[[鬱林郡|郁林郡]]。[[元鼎|元鼎]]六年（公元前111年），设'''[[领方县|领方县]]'''（县境内有领方山而得名），为县地建立行政建置之始，治所位于今宾州镇古城村一带。[[唐朝|唐]][[贞观_(唐朝)|贞观]]五年（631年），在岭方县域置'''[[宾州_(广西)|宾州]]'''（因州境内有宾水而得名），为县地设州行政建置之始。

[[中华民国|民国]]元年（1912年），改名'''宾阳县'''（县城在宾水之北故名）。1949年12月[[中国人民解放军|解放军]]攻占宾阳县后，划属[[南宁专区|南宁专区]]管辖。1971年，南宁专区改称[[南宁地区|南宁地区]]。2003年6月27日，南宁地区撤销，宾阳县划归[[南宁市|南宁市]]管辖。<ref name="BYGK"/>

== 文化 ==
===炮龙节===
正月期间，民间每年都有组织特色炮龙节，宾阳炮龙节在正月十一开始进行，被称为“东方狂欢节”。20岁左右的年轻小伙舞龙过街，居民街坊都会用点燃鞭炮以示迎接。据说炮龙拜年会带来好运，抢到龙珠的来年必会得贵子。炮龙节给正月增添了许多喜庆。
===语言===
宾阳[[语言|语言]]资源丰富，境内分布有宾阳话（本地话），白话（[[粤语|粤语]]），新民话（[[客家话|客家话]]），[[壮话|壮话]]等方言，其中全县90%的人口说宾阳话。宾阳话属于桂南[[广西平话|平话]]的一种，桂南平话传统上属于粤语的一支，但近年来有学者试图把桂南平话把粤语的一支中独立出去。
