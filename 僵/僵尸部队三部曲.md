{{vgname|僵尸部队三部曲|en=Zombie Army Trilogy}}是一款由[[Rebellion_Developments|Rebellion Developments]]制作发行的[[第三人称射击游戏|第三人称]][[战术射击游戏|战术射击游戏]]，为[[狙击精英系列|狙击精英系列]]的衍生作品。游戏于2015年3月6日在[[Microsoft_Windows|Microsoft Windows]]、[[PlayStation_4|PlayStation 4]]和[[Xbox_One|Xbox One]]平台发布<ref>{{cite web|url=http://www.ign.com/articles/2015/01/08/zombie-army-trilogy-coming-to-ps4-xbox-one-pc|title=Zombie Army Trilogy coming to PlayStation 4, Xbox One, PC|author=Jordan Sirani|work=[[IGN|IGN]]|date = 2015-01-08|accessdate = 2015-01-09}}</ref>。

三部曲包含《[[狙击精英V2#纳粹僵尸部队|狙击精英：纳粹僵尸部队]]》、《[[狙击精英V2#纳粹僵尸部队|狙击精英：纳粹僵尸部队2]]》及其续作，并且将前两部高清化<ref>{{Cite web|url = http://www.gamersky.com/z/zombiearmy/|title = 游民星空僵尸部队三部曲专区|accessdate = 2015-08-27|author = |date = |publisher = [[游民星空|游民星空]]|language = zh-cn}}</ref>。第三部的主要内容为对抗僵尸化元首率领的军团，游戏的合作模式中有8名人物可供选择，其中4名是新加入的女性角色。2015年8月，游戏将《[[求生之路|求生之路]]》和《[[求生之路2|求生之路2]]》的8名角色加入，至此玩家可操作的人物变为16个<ref>{{Cite web|url = http://www.laozu.com/news/38247.html|title = 《僵尸部队三部曲》加入求生之路角色|accessdate = 2015-08-27|author = |date = 2015-08-21|publisher = 老祖游戏网|language = zh-cn}}</ref><ref>{{cite web|url=http://www.pcgamer.com/zombie-army-trilogy-announced-by-rebellion/|title=Zombie Army Trilogy announced by Rebellion [Updated]|author=Phil Savage|work=[[PC_Gamer|PC Gamer]]|date = 2015-01-08|accessdate = 2015-01-10}}</ref>。游戏包含战役和部落模式，玩家可单独战斗亦可选择与其他玩家合作。
