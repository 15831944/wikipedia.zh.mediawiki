'''聚才路站'''是建设中的[[杭州地铁5号线|杭州地铁5号线]]的地下车站，工程名为'''江虹路西站'''，位于[[滨江区|滨江区]]滨康路聚才路交叉口西北侧的空地内。<ref>{{cite web |title=江虹路西站工程方案公告图 |url=http://attachment.hangzhou.com.cn/forum/201810/24/134957g6pikuufwmkkkwww.jpg}}</ref>

== 结构 ==
聚才路站为地下岛式站台，为地下两层双柱三跨箱型框架结构，标准段基坑深约15.74 m，端头井基坑深约17.05 m；站台宽12.6m，标准段宽21.7m，车站外包总长316.288m。车站共设4个出入口、1个疏散口及2组风亭。<ref>{{cite web |title=杭州地铁5号线一期工程土建施工SG5-16-1标段江虹路西站招标_招标网_浙江省招标 |url=http://www.bidchance.com/info.do?channel=calgg&id=18908383 |website=招标网}}</ref>
