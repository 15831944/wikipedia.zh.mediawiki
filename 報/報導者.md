'''報導者'''（[[英語|英語]]：The Reporter），是由「財團法人報導者文化基金會」於2015年成立的非營利網路媒體<ref name=":1" />，致力於[[公共議題|公共議題]]的深度[[調查報導|調查報導]]，其內部成員大多是資深[[傳媒工作者|媒體工作者]]，創辦人為[[臺灣|臺灣]]知名媒體人[[何榮幸|何榮幸]]。<ref>{{Cite web|url=https://sparks.ntustudents.org/3491/|title=【花火大講堂】何榮幸《我的報導者之路——我們的新聞小革命》現場紀實（下）|accessdate=2018-7-13|author=花火編輯群|date=2016-12-1|publisher=花火}}</ref>

報導者因以非營利[[基金会|基金會]]的模式經營，平日靠募款和讀者捐款維持運作<ref>{{Cite news|url=https://www.inside.com.tw/2015/09/01/the-reporter|title=專做深度調查報導，何榮幸、張鐵志宣佈網路新媒體「報導者」成立|author=Lisa|date=2015-9-1|work=|publisher=INSIDE|accessdate=}}</ref><ref>{{cite news|url=http://www.epochtimes.com.tw/n139522/%E6%96%B0%E5%AA%92%E9%AB%94%E5%A0%B1%E5%B0%8E%E8%80%85%E6%88%90%E7%AB%8B-%E4%B8%8D%E9%87%8D%E5%8D%B3%E6%99%82%E9%87%8D%E6%B7%B1%E5%BA%A6.html|title=新媒體報導者成立 不重即時重深度|date=2015-09-02|agency=[[大紀元時報|大紀元時報]]|author1=黃筠芸}}</ref><ref>{{cite news|url=http://www.epochtimes.com/b5/15/9/1/n4517418.htm|title=台網路新媒體報導者成立 贊助者不干預|author=鄭景雯|last=|first=|date=2015-9-1|work=|publisher=大紀元|accessdate=|agency=[[中央通訊社|中央通訊社]]}}</ref>，其網站內容採用[[創用CC授權條款|創用CC授權條款]]。<ref name=":1">{{Cite news|url=https://www.twreporter.org/a/about-us-footer|title=關於我們 - 報導者 The Reporter|accessdate=2017-03-09|language=zh-TW}}</ref>

==概略==
報導者的營運宗旨是改變臺灣的[[新聞|新聞]][[環境|環境]]和[[公共領域|公共領域]]，該傳媒致力於深度調查報導的產出。其未來發展包括，[[開放原始碼|開放原始碼]]、實施攝影共有[[版權|版權]]與開放編輯室。報導者的報導內容著重國際與[[中國|中]][[香港|港]]事務及[[文化|文化]]觀察，其中包涵深度題目報導、人物專訪及深度文化觀察，並穿插以[[社會|社會]][[人民|大眾]]關注之解釋性新聞及相關[[圖表|圖表]]。<ref name=":0">{{cite news|url=http://www.coss.ntu.edu.tw/zh_tw/announcement/5/%E6%8F%90%E7%87%88%E7%85%A7%E5%89%8D%E8%B7%AF-%E3%80%8A%E5%A0%B1%E5%B0%8E%E8%80%85%E3%80%8B%E9%96%8B%E5%89%B5%E9%9D%9E%E7%87%9F%E5%88%A9%E5%AA%92%E9%AB%94%E7%B6%93%E9%A9%97-82497977|title=提燈照前路 《報導者》開創非營利媒體經驗|last=|first=|date=2015-10-07|work=|publisher=[[國立臺灣大學|國立臺灣大學]]社會科學院|accessdate=|author1=黃硯琳}}</ref><ref name="storm">{{cite news|url=http://www.storm.mg/article/64112|title=新媒體「報導者」創立 致力公共議題深度報導|author=陳耀宗|date=2015-9-1|work=|accessdate=|agency=[[風傳媒|風傳媒]]}}</ref>

報導者的網站有兩點特色：一、網站不登廣告。二、所有報導都不會有點閱數字，讓記者不被文章流量影響報導方向，他們同時也不會冷落較少人關注的議題。<ref name=":3">{{Cite web|url=https://sparks.ntustudents.org/3491/|title=【花火大講堂】何榮幸《我的報導者之路——我們的新聞小革命》現場紀實（下）|accessdate=2018-8-14|author=花火編輯群小編YS|date=2016-1-12|publisher=花火}}</ref>

== 成立背景 ==
{{see|台灣媒體亂象}}[[臺灣|臺灣]]媒體產業在台灣已被視為社會亂源，因「即時新聞」當道，[[新闻工作者|新聞工作者]]為了大量產製即時新聞，使新聞專業下降，且面臨勞動條件低劣，必須靠點閱流量存活等問題。<ref>{{Cite web|url=http://translocal.asia/index.php?m=Article&a=show&id=1124|title=「報導者」何榮幸總編輯教會我的一件事|accessdate=2018-8-14|author=林淑卿|date=2017-9-20|publisher=串樓}}</ref>為了解決當今[[臺灣|臺灣]]新聞氾濫的困境，報導者執行長[[何榮幸|何榮幸]]決定「自己的新聞自己救」，他認為媒體應回歸新聞初衷，新聞應是原生內容，且應給予[[新闻工作者|新聞工作者]]充足的時間製作深度報導，而報導內容也應該經過查證。因此，何榮幸離開主流媒體「獨立評論＠[[天下雜誌|天下]]」，在9月1日記者節這天宣布成立報導者，致力於[[調查報導|調查報導]]的產出。<ref name=":0" /><ref name=":3" />

== 編制 ==
報導者組織分為編輯部、數位部、攝影部，以及基金會。<ref name=":2">{{Cite web|url=https://www.twreporter.org/about-us|title=關於我們|accessdate=2018-8-9|author=報導者|date=|publisher=}}</ref>

* 編輯部：總編輯是前《天下雜誌》總主筆[[何榮幸|何榮幸]]，總主筆是前《天下雜誌》記者[[李雪莉|李雪莉]]，副總編輯為方德琳和楊惠君，編輯主任是張詩芸。<ref name=":2" />
* 數位部：數位部主任是李法賢，數位專案經理陳貞樺。工程師有三位，分別是余崇任、方泰鈞和曾清陽。<ref name=":2" />
* 攝影部：攝影主任是余志偉（曾任《[[台灣日報|台灣日報]]》、《[[台灣蘋果日報|台灣蘋果日報]]》、[[風傳媒|風傳媒]]攝影記者），資深攝影為吳逸驊，攝影記者為林佑恩。<ref name=":2" />
* 基金會：「財團法人報導者文化基金會」的董事長是[[世新大學|世新大學]]教授[[翁秀琪|翁秀琪]]，董事有李怡志和林以涵。<ref name=":2" />

==營運模式==
「財團法人報導者文化基金會」起初由[[和碩聯合科技|和碩聯合科技]]董事長[[童子賢|童子賢]]捐助新臺幣500萬元成立<ref name=":2" />，營運初期三年均依靠企業贊助，但捐助者不得介入及干預。長遠發展則規劃以[[公益金融|公益基金]]模式採[[小額捐款|小額捐款]]方式招募會員維持運作。<ref>{{Cite web|url=https://www.inside.com.tw/2015/09/01/the-reporter|title=專做深度調查報導，何榮幸、張鐵志宣佈網路新媒體「報導者」成立|accessdate=2017-03-09|author=|date=2015-09-01|last=Lisa|work=[[INSIDE_硬塞的網路趨勢觀察|INSIDE 硬塞的網路趨勢觀察]]|publisher=}}</ref>

報導者初期捐助者代表童子賢曾宣示三不原則「不擁有、不干預、不回收」，以減少商業干預。<ref name="storm" /><ref>{{cite news|url=http://news.ltn.com.tw/news/life/breakingnews/1430998|title=不追求即時！新媒體「報導者」成立 致力深度報導|date=2015-09-01|agency=[[自由時報|自由時報]]即時新聞}}</ref>。2015年12月16日起，報導者在每一季結束時，會將每季接受捐助的狀況公布在官網上。至於財務運作，報導者會聘請[[會計師|會計師]]稽核，並於官網公布每年的[[財務報表|財務報表]]。<ref name=":1" />

=== 接受捐助原則 ===
經由報導者文化基金會董事會決議的捐助原則如下<ref name=":1" />：

# 所有捐款者皆須接受「三不原則」：不擁有、不干預、不回收。 
# 不接受任何政黨、政治人物捐款。
# 凡新臺幣100萬元以上捐款，由董事會捐款審查小組決定是否接受。
# 凡新臺幣100萬元以上捐款，應予具名。

== 工作倫理規範 ==
經由報導者文化基金會董事會決議的工作倫理規範如下<ref name=":1" />：

# 基金會董監事，不得涉入與干預《報導者》任何新聞處理個案與新聞內容。
# 所有捐款者皆秉持「三不原則」：不擁有、不干預、不回收。《報導者》任何新聞處理，皆不受捐款者影響。新聞處理內容若涉及捐款者，由監察人監督內容是否維持獨立自主。
# 凡基金會專職人員，在外不得有任何支領固定酬勞之兼職，並不得於商業媒體撰寫文稿。

== 爭議 ==
2017年6月21日報導者刊登張子午撰述〈當房思琪成為實體——專訪作者密友與編輯談林奕含的出版歷程〉的報導，文中以匿名的方式，指稱「一間知名出版社」的總編與[[林奕含|林奕含]]「簽約旋即解約」的過程，「對林奕含不啻是一個重擊」，是對精神疾病「另一個屢見不鮮的歧視與排除」<ref name="apple" />。報導刊出後，寶瓶文化總編輯朱亞君兩度於臉書發表聲明，駁斥報導者刊載的報導未盡平衡報導之責，單方面引用林奕含密友的說法，造成輿論效應，形同[[網絡公審|公審]]，使得她走投無路，於6月25日於自宅意圖輕生獲救<ref name="ltn" />。報導者編輯部隨後於26日凌晨在臉書發文致歉，並刪除報導中相關段落<ref name="apple2" />。
