'''鎮轅境頂土地廟'''（'''鎮轅境頂土地公廟'''）位於臺灣[[臺南市|臺南市]][[北區_(臺南市)|北區]]，是主祀[[福德正神|福德正神]]的廟宇{{r|福德}}。該廟與軍民合建的[[總祿境下土地廟|總祿境下土地廟]]相對，護佑著崇安街（[[總爺街|總爺街]]），是典型「街頭街尾土地公」的例子{{r|傅}}{{r|市}}{{r|府城舊街新路}}，兩廟也是清末府城聯境「十八境」的成員{{r|福德}}。該廟相傳是位在總爺街「蜈蚣穴」的頭部，廟兩側呈V字型的道路是蜈蚣的觸角{{r|市}}。

除福德正神外，還陪祀有齊天大聖與臨水夫人，後殿福德正神兩側還有供奉大型的文、武判官{{r|傅}}{{r|福德}}。廟中保有道光十二年（1832年）的石香爐、與「顯靈保固」、「保我黎民」古匾{{r|傅}}{{r|福德}}。

== 沿革 ==
關於鎮轅境頂土地廟的建廟年代有數種說法，一說是清乾隆五年（1740年），總兵何勉在總鎮署周邊建土城「大營盤城」時{{r|福德}}。另一說則是乾隆十三年（1748年），跟總祿境同時興建{{r|傅}}{{r|市}}{{r|重建廟記}}。該廟與總祿境均是總爺街的土地廟，但是鎮轅境是官建廟宇，總祿境則是軍民集資合建{{r|福德}}{{r|聯境}}。該廟舊稱「總鎮署東轅門土地祠」，於《[[重修臺灣縣志|重修臺灣縣志]]》（1752年）有寫說[[臺灣府城|府城]]文武衙門左邊均建有土地祠，在農曆每月初一、十五都會由主官帶領主祭{{NoteTag|《重修臺灣縣志‧卷六》：「又文武各衙門左，俱有土地祠。朔望日本衙門行香祭，則本官主之。其居民所祀土王曰福德祠。在東安坊者六：嶺頂、番薯崎、觀音亭邊、諸羅倉邊、嶽帝廟右、龍川井。在西定坊者四：南巷口、新街尾、海防廳邊、[[佛頭港景福祠|佛頭港]]。在寧南坊者五：打石街、安海街、磚仔橋、大南門邊、[[大埔福德祠|大埔尾]]。在鎮北坊者六：禾寮港街、[[總祿境下土地廟|總爺街]]、熟皮寮、赤嵌樓左、粗糠崎、林投井。在永康里者二：彌陀寺右、燒{{僻字|𥕟|左「石」右「屚」}}嵌。他如大南門外仙草寮、小北門外柴頭港及安平鎮新豐里、長興里、土庫、歸仁北里、舊社街，在在有祠。每歲二月二日、八月十五日沿戶鳩資演劇，張燈慶讚，亦春祈秋報之意。」}}{{r|福德}}。

清道光、咸豐年間重修過{{r|市}}{{r|重建廟記}}{{r|府城舊街新路}}。二次大戰後，於民國47年（1958年）在當地信眾黃江、王堯醮、謝萬發等人發起下重修{{r|重建廟記}}。民國82年（1993年）重建成今貌{{r|福德}}{{r|重建廟記}}。

== 傳說 ==
相傳太子太保[[王得祿|王得祿]]因受到鎮轅境土地公所激而前往嘉義，因緣際會下救助嘉慶君{{r|福德}}{{r|傅}}。而因為王得祿感念受到鎮轅境土地公護佑，受其刺激而奮發之恩，故奏請朝廷，所以該廟土地公戴官帽，並有文、武判官，建築也採用燕尾脊{{r|福德}}{{r|傅}}{{r|重建廟記}}{{r|府城舊街新路}}。又有一說是總兵要參拜土地公，因總兵為二品官，所以將土地公升為一品{{r|府城舊街新路}}。

另外有說法認為鎮轅境土地公左右是文、武判官，庇祐求取功名者，而總祿境土地公左右是招財、進寶童子，庇祐求取利祿者{{r|福德}}。又有一說是因為總祿境地勢較低，可擋住財富，而有「求官上土地，求財下土地」的說法{{r|府城舊街新路}}。
