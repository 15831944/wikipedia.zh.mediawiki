'''鎮安車站'''位於[[屏東縣|屏東縣]][[林邊鄉|林邊鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[屏東線|屏東線]]的[[鐵路車站|鐵路車站]]。過去曾為臺灣鐵路管理局[[東港線|東港線]]（現已廢止）的端點站。

== 車站構造 ==
*[[側式月台|側式月台]]、[[島式月台|島式月台]]各一座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]。
*停靠狀態：
**[[臺鐵區間車|區間車]]：皆停靠（[[復興號列車|復興號]]車廂編組或是單組[[臺鐵柴聯車|柴聯車]]編組）。
**[[臺鐵區間快車|區間快]]：皆停靠(2組柴聯車編組)
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 車站歷史 ==
*1940年7月19日：開業，初名'''社邊驛'''（[[社邊|社邊]]目前位於南州鄉南安村）。<ref name=":0">{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>
*1962年4月1日：因行政區域屬[[林邊鄉|林邊鄉]]鎮安村，站名和地名一同改為「鎮安站」；與[[龍泉車站|龍泉]]、[[源泉車站|源泉]]、[[大山車站|大山]]、[[山佳車站|山佳]]、[[猴硐車站|侯硐]]、[[平和車站|平和]]等車站同時改稱今名。<ref name=":0" />
*1995年5月1日：由三等站直接降為招呼站，並指定由[[南州車站|南州站]]管理。<ref name=":0" />
*2016年1月1日：啟用多卡通刷卡機。
