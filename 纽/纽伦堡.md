'''纽伦堡'''（{{lang-de|Nürnberg}}﹐{{lang-en|Nuremberg}}）是[[德国|德国]][[巴伐利亚州|巴伐利亚州]][[中弗兰肯行政区|中弗兰肯行政区]]的中心城市，巴伐利亚州的第二大城市，仅次于首府[[慕尼黑|慕尼黑]]，是世界著名大企业集团[[西门子公司|西门子公司]]的诞生地。纽伦堡与它的两座姊妹城市[[菲尔特|菲尔特]]和[[埃尔朗根|埃尔朗根]]相邻极近，城区相连，常被称作“大纽伦堡地区”。[[第二次世界大战|第二次世界大战]]结束之后曾在此举行针对[[纳粹德国|纳粹德国]]战犯的[[纽伦堡审判|纽伦堡审判]]。

== 華語譯名 ==	
據德語音譯為「'''紐倫堡'''」或「'''紐倫伯格'''」。實際上，德語詞尾 "-berg" 乃指「山丘」，與意指「城堡」的詞尾 "-burg" 有所不同。舊據英語音譯為「'''努連堡'''」。

== 故事与传说 ==
[[德国|德国]]著名童话作家[[E·T·A·霍夫曼|E·T·A·霍夫曼]]的[[童话故事|童话故事]]《[[胡桃夹子与老鼠王|胡桃夹子与老鼠王]]》就发生在[[圣诞夜|圣诞夜]]的纽伦堡[[市政厅|市政厅]]，这个故事后来被[[俄罗斯|俄罗斯]]大作曲家[[柴科夫斯基|柴科夫斯基]]谱写成[[芭蕾舞剧|芭蕾舞剧]]《[[胡桃夹子|胡桃夹子]]》而名扬天下。

== 歷史 ==
纽伦堡是中世纪多位德意志皇帝诞生和居住的城市，有浓厚的历史气息，曾经是德意志的玩具制造中心，被称作是欧洲的“玩具都城”（Toy capital）。目前市内仍有别具特色的玩具博物馆，而每年在此地所舉辦的[[紐倫堡國際玩具展|紐倫堡國際玩具展]]（Nuremberg International Toy Fair），也是玩具產業最重要的商展之一。

纽伦堡有两座姊妹城市：[[菲尔特|菲尔特]]和[[埃尔朗根|埃尔朗根]]。从菲尔特到纽伦堡的铁路是德国历史上第一段铁路，就此掀开了德国[[工业化|工业化]]的篇章，成为德国工业化的起点。因此，[[德国铁路|德国铁路]]的铁路博物馆设于纽伦堡，但2005年一场大火使博物馆的馆藏几乎损失殆尽。

历史上，纽伦堡是“[[德意志民族神圣罗马帝国|德意志民族神圣罗马帝国]]”皇帝直辖的统治中心城市之一。使纽伦堡在[[纳粹德国|纳粹德国]]时代成为[[纳粹党|纳粹党]]一年一度的党代会会址，纽伦堡在[[希特勒|希特勒]]德国时代风光无限，并因此使该城沦为英美[[盟军|盟军]]的重点轰炸对象，颇具[[中世纪|中世纪]]古风的老城区就此夷为平地，虽战后重建，亦不复当年风采。[[第三帝国|第三帝国]]著名的反犹太[[纽伦堡法案|纽伦堡法案]]就是在此处出炉的，掀起了种族清洗的风浪。

[[第二次世界大战|第二次世界大战]]后的清算纳粹战犯罪行的[[纽伦堡审判|纽伦堡审判]]是在此举行的。在战后，纽伦堡渐渐归于平淡，回复到地区中心城市的角色。

== 文化與風俗 ==

=== 皇帝堡 ===
這座古堡建於12世紀。現在看到的建築是15至16世紀的遺跡。

=== 日耳曼國家博物館 ===
紐倫堡的[[日耳曼國家博物館|日耳曼國家博物館]]呈現日耳曼地區的豐富文化，也是德國最大的日耳曼文化博物館。

=== 聖塞巴杜斯教堂 ===
始建於1225年的教堂。

=== 圣诞市场 ===
[[File:Christmarkt.jpg|thumb]]
如同大部分的德國城市一樣，纽伦堡舊城區中央的“主市场”（{{lang|de|Hauptmarkt}}）在每年[[圣诞节|圣诞节]]前几周就會開始舉辦[[聖誕市場|聖誕市場]]活動，而在紐倫堡的聖誕市場一般被認為是德国历史最悠久、规模最大的传统圣诞市场。在聖誕市場中最常見的商品包括[[熱紅酒|熱紅酒]]（{{lang|de|Glühwein}}）與[[德式薑餅|德式薑餅]]（{{lang|de|Lebkuchen}}）。圣诞市场一般都是于[[圣诞夜|圣诞夜]]來臨前的下午闭幕。

=== 國際發明展 ===
是全球首創、歷史最悠久的創意商展，因其規模宏大，參展者眾，在國際間享有權威之聲譽。

== 交通 ==
[[File:Siemens_Nürnberg_Vogelweiherstr_Nordeingang.jpg|thumb]]
[[File:germany_supermarket_norma.jpg|thumb]]
大纽伦堡地区是[[德国|德国]]第七大经济区，因为地处中欧中部位置，是一个繁忙的交通枢纽，城市位于布拉格——巴黎、维也纳——布鲁塞尔——伦敦、米兰——苏黎世——斯德哥尔摩、罗马——柏林等[[高速公路|高速公路]]的交汇处，并且是[[ICE|ICE]]、EC和IC铁路的交叉点，从市区旁流过的[[莱茵—多瑙运河|莱茵—多瑙运河]]连接[[莱茵河|莱茵河]]、[[多瑙河|多瑙河]]、[[北海_(大西洋)|北海]]和[[黑海|黑海]]，运河的纽伦堡港每年吞吐货物700万吨，市区北部的纽伦堡国际机场每天提供50条航线通向20多个欧洲城市。

== 经济 ==

自中古時期以來就一直是中歐工業重鎮的紐倫堡，在今日仍然是巴伐利亞的工業中心，以高精密的電子、機械與光學產品產業著稱。除了這些現代化的產品之外，紐倫堡當地的傳統產品，例如[[德式薑餅|德式薑餅]]與手工[[玩具|玩具]]，仍然是具有極高代表性的出口商品。

=== 就业 ===
在德国一片[[失业|失业]]浪潮中，以纽伦堡为中心的[[弗兰肯|弗兰肯]]地区被认为是失业率最低的模范州巴伐利亚的模范地区。

=== 著名企业 ===

* [[西门子|西门子]]：西門子最早创建于纽伦堡的老城区，后来因为纽伦堡老城区内土地稀缺，地价较贵，选择在纽伦堡的姊妹城市[[埃尔朗根|埃尔朗根]]作为扩大发展的基地，在[[第二次世界大战|战后]]，随着[[国际化|国际化]]的浪潮，公司的部分管理职能部门搬迁到[[柏林|柏林]]和[[慕尼黑|慕尼黑]]，在[[埃尔朗根|埃尔朗根]]保留相当的[[研发|研发]]部门和部分管理部门。
* [[Norma超市|Norma超市]]：最早由纽伦堡的姊妹城市[[菲尔特|菲尔特]]的一家企业派生出来，如今已经成为一家在[[欧洲|欧洲]]知名的跨国经营连锁[[超市|超市]]。
* [[GfK公司|GfK公司]]：1934年由后来的德国[[经济部长|经济部长]][[路德维希·艾哈德|艾哈德]]等人成立于纽伦堡，是德国最大、全球第五大的[[市场调研|市场调研]]公司。
* [[SUSE_Linux|SUSE Linux]]：歐洲最大，世界第二大的商業Linux作業系統發行版。專注於超級電腦，資料中心，高效能運算領域。是世界最早的企業用Linux作業系統。除了Linux, 也發展開源雲端堆疊，開源容器技術。

== 教育 ==
纽伦堡原先有一所独立的“纽伦堡经济社会学院”，曾出过不少著名人物，比如德国的“[[社会市场经济|社会市场经济]]之父”前联邦总理[[路德维希·艾哈德|路德维希·艾哈德]]、[[欧洲中央银行|欧洲中央银行]][[首席经济学家|首席经济学家]]金融专家[[奥特马·伊兴|奥特马·伊兴]]（''Otmar Issing''）。后来该学院被姊妹城市[[埃尔朗根|埃尔朗根]]的埃尔朗根大学合并作为该大学的经济社会学系，大学名称也就此改为[[埃尔朗根-纽伦堡大学|埃尔朗根-纽伦堡大学]]。该大学的[[企业经济学系|企业经济学系]]（BWL，相当于美国的商学院、中国的管理学院）据说是德国的五大[[企业经济学系|企业经济学系]]之一（按注册在读大学生人数）。
