'''豆雁'''（[[学名|学名]]：''Anser fabalis''），又名寒林豆雁、大雁、鸿、东方豆雁、西伯利亚豆雁、普通大雁、麦鹅。

== 生态环境 ==
豆雁广泛分布于[[古北界|古北界]]的江河、湖泊以及海岸环境，有趣的是在迁徙途中豆雁还经常光顾农田，时常会取食越冬的冬小麦等作物，因而引起农夫的不满。

== 分布地域 ==
繁殖于[[西伯利亚|西伯利亚]]大部、[[欧洲|欧洲]]北部以及[[冰岛|冰岛]][[格陵兰|格陵兰]]东部，[[中国东北|中国东北]]北部，越冬于包括台湾在内的中国大部，甚至远达香港都有本种越冬记录、[[欧洲|欧洲]]中南部，[[朝鲜|朝鲜]][[日本|日本]]等地，迁徙时几乎会途径整个古北界。

== 身體特征 ==
体形大小有如家鹅，额部不似鸿雁具有一圈耀眼的白色羽毛，体羽深褐色，上体颜色较深呈褐色，下体颜色较浅呈污白色，但不似[[鸿雁|鸿雁]]具有泾渭分明的深浅色快分界线。各羽片边缘颜色较浅。另外豆雁区别于同属近似种的一大特征是黑色的嘴，端部具有一个明亮的黄色色块，远远看去有如口衔一枚黄豆，所谓“豆雁”指的就是这个“豆子”。虹膜颜色暗棕；脚为橘黄色。叫声为较深沉的似hank-hank声。

== 食物 ==
豆雁主要以植物為食，经检验[[白薯|白薯]]、[[小麦|小麦]]、[[蚕豆|蚕豆]]等作物的嫩叶、幼芽；[[慈姑|慈姑]]、[[菱角|菱角]]、[[荸荠|荸荠]]等野生植物都是豆雁的食物，不过偶尔豆雁也会取食少量的[[软体动物|软体动物]]

== 繁殖与保护 ==
豆雁未列入濒危名单，但被作为医药成分捕猎而受到一定程度的威胁。
