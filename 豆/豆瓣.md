'''豆瓣'''（'''{{lang|en|douban}}'''）是一家社区网站。网站由杨勃（网名“阿北”）<ref name=xinhua>{{cite news|author=王乐|title=网络新生领域扫描：一个人运营的“豆瓣”网|url=http://news.xinhuanet.com/ec/2005-11/26/content_3838371.htm|newspaper=新华网}}</ref>创立于2005年3月6日。该网站以书影音起家，提供关于[[书籍|书籍]]、[[电影|电影]]、[[音乐|音乐]]等作品的信息，无论描述还是评论都[[用户生成内容|由用户提供]]，是中国[[Web_2.0|Web 2.0]]网站中具有特色的一个网站。网站还提供书影音推荐、线下同城活动、小组话题交流等多种服务功能，它更像一个集品味系统（读书、电影、音乐）、表达系统（我读、我看、我听）和交流系统（同城、小组、友邻）于一体的创新网络服务，一直致力于帮助都市人群发现生活中有用的事物。2012年，豆瓣閱讀上线，开始进入网上[[电子书|电子书]][[版权|版权]]领域。

2012年8月，豆瓣宣布其月度覆盖独立用户数（Unique Visitors）已超过1亿，日均PV为1.6亿。<ref>{{Cite web |url=http://tech.163.com/12/0817/08/893L4F6N000915BF.html|title=豆瓣称月度覆盖用户数超过1亿 已接近盈利|accessdate=2014-12-09 |author=|date= |work= 2012-08-17|publisher= 网易科技}}</ref> 2013年第二、三季度的豆瓣月度覆盖独立用户数均达2亿，较去年同期增长一倍。<ref>{{Cite web |url= http://www.techweb.com.cn/internet/2013-11-13/1356287.shtml|title=豆瓣宣布月覆盖用户数达2亿 同比增长一倍|accessdate=2014-12-09 |author=安东 |date= |work= 2013-11-13|publisher=  TechWeb.com.cn }}</ref>

== 简介 ==
豆瓣的名称由来是创始人杨勃创建该网站的时候在[[北京|北京]][[豆瓣胡同|豆瓣胡同]]附近的咖啡馆写该网站的源代码，于是取名为豆瓣网。<ref>{{Cite news|url=http://tech.sina.com.cn/i/2015-03-21/doc-iawzuney0948206.shtml|title=10年之豆瓣：一亿文艺青年在一起就成了风格|author=|date=2015-03-21|work=|accessdate=2018-05-27}}</ref>

像大多数 Web 2.0 网站那样，豆瓣网内容可由用户添加，对里面所列的音乐、书籍、电影等可以作收藏、评论、打分等操作。该网站程序算法可以根据用户收藏的电影、书等进行同类推荐，即在用户提供了一定量的收藏信息后，豆瓣会根据其[[算法|算法]]自动选出与此成员口味最像的其他成员，成员以此可以找到和自己“臭味相投”的用户。豆瓣还会给出“豆瓣猜你会喜欢”的列表，向用户推荐可能合其口味的音乐、书籍或电影。

用户可以通过加入同城，小组，友邻等交流方式和其住在同一地区，或喜欢某个共同主题，或互相感兴趣的成员们进行图书，电影和音乐的交流和推荐以及二手转让。

由于豆瓣、[[时光网|时光网]]不同于[[烂番茄|烂番茄]]网站上的专业影评人评分，任何人都可以在上面打分（甚至在影片上映前就可以打分），所以存在着有人为了各自的利益和目的而恶意刷高或低分的现象。<ref >{{cite web|url=http://www.1905.com/news/20161227/1146056.shtml | language= | title=豆瓣被质疑"刷分"面临信用危机 评分还能不能信？ |publisher=中国电影报 |date=2016-12-27 |accessdate= }}</ref>创始人阿北在每日豆瓣微信公众号中的《豆瓣电影评分八问》的文章内，公开回答了豆瓣电影评分的反作弊与反刷分程序，并声称后台不存在“修改电影评分的功能”以保障评分的公正性<ref>{{cite web|author1=杨勃|title=豆瓣电影评分八问|url=https://blog.douban.com/douban/2015/12/18/3060/|website=豆瓣日志|date=2015-12-18}}</ref>。实际上，根据2018年的[[南方周末|南方周末]]报道，豆瓣虽然未能完全避免水军刷分的现象，但其平台壁垒和防刷分机制导致刷分只能采用人工刷分，导致其刷分成本相对许多其他平台要高几倍<ref>{{Cite news |url=https://new.qq.com/omn/20181025/20181025A1KFR3.html |title=一个赞多少钱 ？那些“刷量”买来的繁荣 |author=张玥 陈爱晨}}</ref>。

=== 功能 ===

觀看豆瓣的评分不需要註冊賬戶，寫評價和其他社交功能需要用戶註冊賬戶。

用戶可以自主註銷賬戶；由于违反社区管理原则，帐号會被管理员注销<ref>{{Cite web |url=https://www.douban.com/accounts/suicide/?ck=ludG |title=删除帐号 |accessdate=2019-03-29 |author=豆瓣 |publisher=豆瓣}}</ref><ref>{{Cite web |url=https://help.douban.com/account?app=1#t2-q5 |title=帐号 使用帮助 |accessdate=2019-03-28 |author=豆瓣 |work=help.douban.com |publisher=豆瓣}}</ref>

==== “即使变成甲壳虫卡夫卡也进不去城堡”事件 ====
2011年2月，[[新浪微博|新浪微博]]网友“雪盲”发文称，自己一年前凭空编造了一部名为《即使变成甲壳虫卡夫卡也进不去城堡》的“电影”，并在豆瓣上创建了相关的页面，所有“电影内容”都源自自身经历。结果一年后，这部根本不存在的“电影”被豆瓣网友评出了8.9分的高分，与《[[指环王|指环王]]》和《[[让子弹飞|让子弹飞]]》持平，甚至超过了《[[阿凡达|阿凡达]]》和《[[泰坦尼克号|泰坦尼克号]]》；在评分的235位网友中，给出五星满分评价的高达88.8%，同时有2457人表示想看这部电影，甚至有人对这部完全凭空捏造的“电影”的“剧情细节”做出了详细点评。在“雪盲”自曝这一恶作剧的真相之后，豆瓣删除了这部所谓的“电影”的页面。<ref>{{cite news |title=一部不存在的电影 很多人说看过还有人写影评 |url=http://news.xhby.net/system/2011/02/20/010881148.shtml |accessdate=2019-04-02 |publisher=新华报业网 |date=2011-02-20}}</ref>

==== 2016年12月 ====
2016年12月27日，[[人民日报|人民日报客户端]]转发《中国电影报》文章并改名为《豆瓣、猫眼电影评分面临信用危机恶评伤害电影产业》。文章指出，豆瓣、猫眼对于3部主打档期新片《[[铁道飞虎|铁道飞虎]]》、《[[擺渡人_(2016年電影)|摆渡人]]》和《[[长城_(电影)|长城]]》的评分过低，让不少观众对贺岁档国产影片大失所望，甚至因此拒绝观看国产影片<ref>{{cite web|title=豆瓣、猫眼电影评分面临信用危机 恶评伤害电影产业|url=http://rmrbimg2.people.cn/html/items/wap-share-rmrb/#/index/home/3/normal/detail/1983945131131904_cms_1983945131131904/normal|website=人民日报|accessdate=2017-01-05|date=2016-12-27}}</ref>。

部分网友担心当局保护国产电影，可能会收紧对豆瓣等网络社区的管制，于是在网上发声抗议，反驳文中水军恶意“刷分”的指控<ref name=“端传媒”>{{cite web|title=人民日报客户端抨击电影打分网站，一人一票给电影打分的权利受到威胁？|url=https://theinitium.com/article/20161229-dailynews-chinese-movie-douban/|website=端传媒|accessdate=2017-01-05|date=2016-12-29}}</ref>。但人民日报评论部于12月28日发表文章《中国电影，要有容得下“一星”的肚量》，指出承认观众有“用脚投票”的权利，也就要承认观众有“打星评级”的权利，这都是一种选择，与《恶评》一文彻底撇清关系<ref>{{cite web|author1=张凡|title=【锐评】中国电影，要有容得下“一星”的肚量|url=http://mp.weixin.qq.com/s?__biz=MzA4OTIyMjUyOQ==&mid=2654620247&idx=1&sn=38a8d32d89e92dc27d92f178089f68c9&chksm=8bd0d6b9bca75faf68c92dc8e684a2475e49614193e1db6aed9b20ae1e719f7ec678bdb6ca0e&mpshare=1&scene=1&srcid=1228wprcg8nEJ3haGBvJ38b3#rd|website=微信公众平台|publisher=人民日报评论微信公众号|accessdate=2017-01-05|date=2016-12-28}}</ref>。随后，大批豆瓣网友对三部影片报复性打出一星，导致影片分数比此前更低。网友热传2015年12月豆瓣的创始人兼CEO阿北发布的详细介绍了豆瓣电影的评分机制文章《豆瓣电影评分八问》<ref name=“端传媒”/>。

媒体曾报道[[国家广电总局|国家广电总局]]电影局约谈豆瓣负责人，但电影局局长张宏森在微信朋友圈否认了这一说法，表示相关报道过度“猜想”了事实，电影局没有与豆瓣有过任何接触，也不认识豆瓣任何人，但很想有机会接触，共商促进电影进步对策<ref>{{cite web|title=谁引发了"信用危机" 中国电影还是中国影评？|url=http://media.people.com.cn/n1/2016/1229/c40606-28984510.html|website=人民网|publisher=北京青年报|accessdate=2017-01-05|date=2016-12-29}}</ref>。

==== 2017年CCTV春晚 ====
由于[[2017年中国中央电视台春节联欢晚会|2017年中国中央电视台春节联欢晚会]]的观众评分较低，仅得2.4分（对应满分是10分），负面评论遭官方删除，豆瓣上亦无法搜索到有关“2017央视春晚”的词条<ref>{{cite web|title=2017年中央电视台春节联欢晚会|url=https://movie.douban.com/subject/26729145/|website=豆瓣}}</ref>，其评论功能也遭关闭<ref>{{cite web|url=http://www.dw.com/zh/%E5%A4%A7%E5%86%99%E7%9A%84%E5%B0%B4%E5%B0%AC%E5%85%9A%E5%AA%92%E5%90%90%E6%A7%BD%E6%98%A5%E6%99%9A%E8%A7%82%E4%BC%97%E5%8F%A3%E5%91%B3%E5%88%81/a-37312169?&zhongwen=simp|title=大写的尴尬!党媒"吐槽"春晚观众口味"刁"|publisher=[[德国之声|德国之声]]|date=2017-01-28|accessdate=2017-01-28}}</ref>。目前[[2016年中国中央电视台春节联欢晚会|2016年春晚]]词条也无法评分<ref>{{cite web|title=2016年中央电视台春节联欢晚会 (2016)|url=https://movie.douban.com/subject/26613473/|website=豆瓣|accessdate=2017-02-05}}</ref>。

==== 《纯洁心灵·逐梦演艺圈》 ====
电影《[[纯洁心灵·逐梦演艺圈|纯洁心灵·逐梦演艺圈]]》上映后在豆瓣的评分为创网站新低的2.0分，导致影片仅公映5天便撤档。片方认为负评太多是导致影片票房遭遇“滑铁卢”的诱因，便于9月22日发布公开信，不满该片在豆瓣评分只有2分，称“一个青年导演花十二年心血认认真真给中国拍电影，被豆瓣一天毁了<ref>{{cite web|title=争议国产片《纯洁心灵》上映5天撤档 时光网评分2.4分票房225万 曾发文谴责豆瓣评分低|url=http://news.mtime.com/2017/09/26/1573951.html|website=Mtime时光网|accessdate=2018-01-21|date=2017-09-26}}</ref>！”6月1日，豆瓣电影正式反告毕志飞及其公司至[[北京市朝阳区人民法院|北京市朝阳区人民法院]]，称毕志飞在“致国家电影局的一封信”中的部分内容属于“污蔑诽谤，与事实严重不符”，要求被告立即删除其在微博、微信上发布的侵权文章，并请求判令被告分别在微博和微信公众号上赔礼道歉，及承担诉讼费用<ref>{{cite web|title=2.1分电影又起波澜 豆瓣起诉"逐梦演艺圈"导演侵权|url=http://media.people.com.cn/n1/2018/0602/c40606-30030074.html|website=中国新闻网|accessdate=2018-06-02|date=2018-06-02}}</ref>。

==== 《厉害了，我的国》 ====
2018年的中國紀錄電影《[[厲害了，我的國|厲害了，我的國]]》在豆瓣網站上禁止公眾以個人名義進行評論，取消了網友的自主評分功能，僅有的數條評論則是以中國大陸官方的主流媒體為主，對影片的評分也是以媒體的形式表現：媒體評分8.5分<ref>{{cite web|url=http://culture.dwnews.com/renwen/big5/news/2018-03-05/60043828.html|title=《厲害了，我的國》禁評論？破紀錄背后的“中國奇跡”|publisher=多維新聞網}}</ref>。

==== 《流浪地球》 ====
2019年春节期间，电影《[[流浪地球_(电影)|流浪地球]]》上映后，在豆瓣的评分一度达到8.5，但由于大量用户给出一星评价，再加上疑似有高赞好评被收买改为差评，《流浪地球》评分跌落至7.9，豆瓣的评分机制再次受到质疑。为此包括《流浪地球》粉丝在内，大量网友前往各大应用市场[[评论轰炸|刷差评以表达不满]]，豆瓣APP在各大应用市场的评分也遭遇雪崩式下滑。应用市场中，不乏模仿豆瓣差评的语气加以评论者（如“若批评不自由，则赞美无意义”“给一星平衡一下”等），由此衍生出所谓“豆学”，即模仿豆瓣差评的口吻调侃其它事物。<ref>{{cite web |title=知乎上这个“豆学”挺有意思的 |url=https://user.guancha.cn/main/content?id=78972&s=fwzwyzzwzbt |website=观察者 |accessdate=2019-02-18}}</ref>
