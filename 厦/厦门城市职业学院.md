'''厦门城市职业学院'''位于[[中国|中国]][[福建省|福建省]][[厦门市|厦门市]]，是福建省的一所[[公立|公立]][[大专|大专]]院校，创办于2005年。该校现有“思明校区”和正在建设中的“集美校区”，“思明校区”位于[[厦门岛|厦门岛]]东海岸的前埔南路，邻近[[厦门国际会展中心|厦门国际会展中心]]。校内建有三维动画模拟导游、数字录音技术、工业设计、汽车检测与维修等实训室，并有馆藏纸质图书的图书馆。<ref>{{Cite web |url= http://gaokao.fjedu.gov.cn/ztc/index.php?s=/pages/index/aid/43 |title= 院校直通车——厦门城市职业学院 |author=  |date= 2013-11-11 |publisher= 福建省教育厅阳光高考信息平台 |language= zh-hans |accessdate= 2013-11-25 |quote= 建有三维动画模拟导游、数字录音技术、工业设计、汽车检测与维修等实训室72间 |deadurl= yes |archiveurl= https://web.archive.org/web/20131202230638/http://gaokao.fjedu.gov.cn/ztc/index.php?s=%2Fpages%2Findex%2Faid%2F43 |archivedate= 2013-12-02 }} </ref>

== 学院历史 ==
厦门城市职业学院是在“厦门广播电视大学”、“厦门教育学院”和“厦门职工大学”三所学校的基础上合并组建而成的[[全日制|全日制]]公办高等职业院校。（后又有“厦门市城市建设中等职业学校”划归该学院。<ref>{{Cite web|url=http://www.xmedu.gov.cn/publish/portal0/tab250/info14663.htm|title=厦教局办〔2005〕105号 厦门市教育局关于成立厦门市城市建设中等职业学校实质性划归厦门城市职业学院管理工作协调领导小组的通知|author=|date=2005-06-13|publisher=厦门市教育局|language=zh-hans|accessdate=2013-11-25|quote=}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>）2005年四月份学院通过福建省、市专家组的建院评估，同年五月正式成立。<ref>{{Cite web |url= http://www.66163.com/Fujian_w/dskx/20050517/xs167974.html |title= 厦城市职业学院酝酿组建 |author= 佘峥 |date= 2005-05-17 |publisher= 福建之窗 |language= zh-hans |accessdate= 2013-11-25 |quote= “厦门城市职业学院”已经在四月份通过省、市专家组的建院评估 |archive-url= https://web.archive.org/web/20131202234118/http://www.66163.com/Fujian_w/dskx/20050517/xs167974.html |archive-date= 2013-12-02 |dead-url= yes }}</ref>

== 学院设置 ==
学院下设9个系和2个教研部，分别为：
*商贸管理系
*财会金融系
*旅游系
*应用外语系
*人文社科与艺术系
*学前教育系
*城市建设管理系
*电子与[[信息工程|信息工程]]系
*机械与自动化工程系
*思想政治理论课教研部
*体育教研部
