'''厦门北站'''，又称'''厦门新站'''，位于[[厦门市|厦门市]][[集美区|集美区]][[后溪镇_(厦门市)|后溪镇]]<ref>[http://news.huochepiao.com/2010-4/201041095749.htm 新火车站更名为厦门北站(组图)] {{webarchive|url=https://web.archive.org/web/20110712231816/http://news.huochepiao.com/2010-4/201041095749.htm |date=2011-07-12 }}</ref>，西北是后溪生活区，南面是大学园区。对外交通方便，距离[[厦门高崎国际机场|高崎国际机场]]约10公里，到[[厦门|厦门]]本岛、[[海沧|海沧]]、[[同安|同安]]和[[翔安|翔安]]中心区的距离均在20公里左右。直接与[[沈海高速公路|沈海高速公路]]连接，与[[高崎国际机场|高崎国际机场]]、厦门本岛以及岛外各片区均有[[城市快速路|城市快速路]]相通。

厦门北站于2007年开始设计，并于不久后开工。车站于2011年竣工，并随[[福厦铁路|福厦铁路]]一齐开通。<ref>{{cite web |url=http://www.csadi.com.cn/xmhj/4024.html |title=存档副本 |accessdate=2016-03-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20160308004042/http://www.csadi.com.cn/xmhj/4024.html |archivedate=2016-03-08 }}</ref>

==车站结构==
===站房===
{{multiple image | direction = vertical | align = right | width = 220
|image1 =20151003厦门北站出发层.JPG|caption1=出发层
|image2=20150930厦门北站出站通道.JPG|caption2=厦门北站出站通道
}}
厦门北站站房由[[中南设计院|中南设计院]]设计，外观采用了[[閩南|闽南地区]]特有的燕尾脊意向，带有鲜明的闽南风格；整体形象充分展现[[厦门|厦门]]的城市精神面貌，充满时代特征，彰显地域特色。

建筑面积约12万平方米。本项目采用“上进下出”模式，站房分为出站层、站台层、高架层三层，高架候车厅采用新型大跨度结构，双向跨度为132mx220m。

建筑形态塑造地域文化特色的同时，也充分体现了结构受力的真实性与合理性，实现了建筑与结构的完美统一，并创造了三项[[世界纪录|世界纪录]]：世界上已建成具有最大跨候车室的高速铁路站房、世界上首次采用双向巨型混合框架结构体系的铁路客运站房、世界上首次应用双向不等高交叉桁架组成的屋盖结构。<ref>http://www.guancha.cn/Project/2015_05_30_321496.shtml</ref>

===地铁===
*主条目：[[厦门轨道交通|厦门轨道交通]]、[[厦门北站站|厦门北站站]]
厦门北站在建设时已经预先建设了地铁站——[[厦门北站站|厦门北站站]]，它是厦门地铁第一个建设的车站。目前是[[厦门轨道交通1号线|1号线]]的车站，也是未来和[[厦门轨道交通4号线|4号线]]的换乘车站，1号线车站以南北走向设于国铁'''厦门北站'''地下，4号线车站以东西走向设于地下。目前车站主体结构已建成，1号线车站已經于2017年12月31日随1号线開通。<ref>{{cite web|url=http://www.doc88.com/p-8846033850340.html|title=组合围护结构技术在厦门北站地铁穿越高铁中的应用 }}</ref>

==未来发展==
[[File:20151003厦门北站北广场（未启用）.JPG|thumb]]
[[福廈客運專線|福厦高铁]]厦门站将选址厦门北站，因此在将来会在现厦门北站站房的北侧即北广场位置，要新建一个高铁站房，用于停靠福厦高铁列车，并预留对接漳州。<ref>{{cite news|url=http://xm.house.sina.com.cn/news/2016-02-01/11416099768409179101413.shtml?wt_source=newshp_news_07|title=福厦高铁厦门站选址厦门北站 厦门段争取今年开工|publisher=新浪网|date=2016-02-01}}</ref>届时，厦门北站将扩建8台18线，并将建设[[动车运用所|动车运用所]]。由于厦门站客运能力和客车整备能力均已饱和，而厦门北站能力却有较大富余，厦门将实施“普速出岛”工程，迁建客整所，改造厦门北站的站台和设施，将所有普速列车调整至厦门北站始发终到。<ref>{{cite |title= 厦门北站办理普速客车的运输组织|author=姚悦亮|publisher=运营与维护|url=http://www.docin.com/p-1154116747.html|year=2013}}</ref>
