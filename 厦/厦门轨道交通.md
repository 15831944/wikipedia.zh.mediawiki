'''厦门地铁'''，又称'''厦门轨道交通'''，是中国[[福建省|福建省]][[厦门市|厦门市]]的城市轨道交通系统，於2017年12月31日起運作。依据厦门市城市总体规划和综合交通规划，厦门市城市轨道交通2020年线网由6线组成，总长度约267公里，共设车站139座，含换乘站18座；远景年线网由11条线路组成，总长度404公里，共设车站188座，含换乘站42座。

== 历史 ==
厦门市早在世纪初就有轨道交通规划，主要线路制式为轻轨。建设规划曾于2006年上报国务院，但是由于厦门人口不足条件，批复难度大，以及轻轨造价高等问题<ref>[http://news.sohu.com/20061130/n246699256.shtml 缓建轻轨快速公交先行]，搜狐</ref>，厦门市便决定先行建设[[厦门快速公交|快速公交]]以应对交通压力，建设时预留升级轻轨条件<ref>[http://www.xm.gov.cn/zfxxgk/xxgkznml/zdjsxmn/ssqk/201208/t20120801_500143.htm 城市快速公交建设专报：确保快速公交车准点]，厦门市人民政府</ref>，首期3条线路已经于2008年8月31日完工并投入使用<ref>
[http://www.fj.xinhuanet.com/nmtsj/2008-09/01/content_14272437.htm 厦门快速公交系统昨通车] {{Wayback|url=http://www.fj.xinhuanet.com/nmtsj/2008-09/01/content_14272437.htm |date=20141025111754 }} {{webarchive|url=https://web.archive.org/web/20141025111754/http://www.fj.xinhuanet.com/nmtsj/2008-09/01/content_14272437.htm |date=2014-10-25 }}，新华社
</ref>。

2010年，厦门市重新启动轨道交通规划建设工作，成立了厦门市轨道交通规划建设工作领导小组及工作机构，并于2011年11月21日挂牌成立国有独资企业[[厦门轨道交通集团有限公司|厦门轨道交通集团有限公司]]<ref>[http://www.xm.gov.cn/zt/xmgdjtjs/jsdt/201403/t20140305_832809.htm 厦门轨道交通集团有限公司揭牌成立]，厦门市人民政府</ref>。原计划启动BRT改造轻轨方案<ref>
[http://news.xmnn.cn/xmxw/201009/t20100914_1563466.htm 厦重启"将BRT升级为轻轨"计划 方案将报国家发改委] {{Wayback|url=http://news.xmnn.cn/xmxw/201009/t20100914_1563466.htm |date=20100918094435 }} {{webarchive|url=https://web.archive.org/web/20100918094435/http://news.xmnn.cn/xmxw/201009/t20100914_1563466.htm |date=2010-09-18 }}，厦门网
</ref>，但在论证中，认为直接进行改造会导致交通情况恶化，所以决定保留快速公交，新建地铁线路，等到时机成熟时，再进行改造<ref>[http://news.xmnn.cn/a/xmxw/201409/t20140918_4062276.htm BRT将改造成轻轨 轨道交通1、2、3号线建成后启动]，厦门网</ref>。2011年5月11日，国家发改委批准了《厦门市城市轨道交通近期建设规划（2011～2020年）》<ref>[http://www.sdpc.gov.cn/xmsphz/t20120905_503816.htm 厦门市城市轨道交通近期建设规划通过批准]，中华人民共和国发展和改革委员会</ref>，厦门成为第28个获批建设地铁的城市<ref>[http://bj.house.sina.com.cn/news/2011-06-30/1018367137.shtml 28城建设国家轨道交通获国家批准]，新浪</ref>。

2011年，民众发现厦门北站出现地铁标志，相关部门表示，北站已经在建站时预先建设了地铁站，预留了地铁位置，何时建设地铁还未确定。<ref>[http://news.163.com/11/1104/10/7I0RS2BA00014AEE.html 厦门北站现地铁标示牌引网友热议 工作人员称将清理(组图)]</ref>

建设规划获国家批准后，厦门市轨道交通1号线一期工程用地预审于2012年11月17日获国家国土部批复。其后，厦门市开始进行线路施工前期准备工作，包括线路环评和工可的研究。其中由于跨海段涉及国家中华白海豚保护区，1号线的环评报告经过了较长时间的论证审批，导致施工时间推迟了两年。最终，环境影响评价报告于2013年6月3日获国家环保部批复，工程可行性研究报告也于2013年8月23日正式获批获批<ref>
[http://www.xmgdjt.com/about.asp?ClassId=3 大事记录] {{Wayback|url=http://www.xmgdjt.com/about.asp?ClassId=3 |date=20141025110932 }} {{webarchive|url=https://web.archive.org/web/20141025110932/http://www.xmgdjt.com/about.asp?ClassId=3 |date=2014-10-25 }}，厦门地铁办公室
</ref>。2013年11月13日，厦门市地铁办公室对外发布消息，宣布1号线开工建设<ref>[http://finance.chinanews.com/cj/2013/11-13/5499670.shtml 厦门开建首条地铁 正式迈向轨道交通时代]，中新网</ref>。

2016年10月8日，《厦门市城市轨道交通第二期建设规划（2016-2022年）》获国家批准建设。规划2号线二期、3号线二期、4号线和6号线一期共4个项目，总长度152.2公里。到2022年，形成5条线路、总长约224公里的轨道交通网络。

=== 运营中线路 ===
====1号线====
[[File:Xiamen_AMTR_Linemap.svg|thumb]]
{{main|厦门轨道交通1号线}}
厦门轨道交通1号线为南北走向的骨干线路。线路沿镇海路、嘉禾路等岛内主要交通干道向北敷设，利用高集、集杏海堤跨海后，连接城市北部集美、同安组团，是贯穿城市南北的重要交通出行走廊。线路全长45.52公里，设站31座。线路选用标准B型空调车辆，六辆编组，可载客约1400人，全程通行时间约为50分钟<ref>[http://www.xm.gov.cn/zt/gclsgwygyzc/xmdt/201311/t20131114_763245.htm 厦门地铁1号线开建]，厦门市人民政府</ref>。1號綫于2017年12月31日開通。

1号线原规划有支线，连接杏林和灌口组团，后经过专家咨询、论证研究，取消了地铁1号线支线。<ref>
[http://www.xmgh.gov.cn/gzcy/xfzl/201405/t20140527_43313.htm 信访专栏] {{Wayback|url=http://www.xmgh.gov.cn/gzcy/xfzl/201405/t20140527_43313.htm |date=20150110101727 }} {{webarchive|url=https://web.archive.org/web/20150110101727/http://www.xmgh.gov.cn/gzcy/xfzl/201405/t20140527_43313.htm |date=2015-01-10 }}，厦门市规划委员会
</ref>

=== 建设中线路 ===
====2号线====
{{main|厦门轨道交通2号线}}

厦门轨道交通2号线呈东西走向，连接思明区和海沧区，构建了本岛与海沧区快速跨海连接通道，全长41.63公里，设站32座。<ref>
[http://www.cztb.xm.fj.cn/newasite/ShowFile.aspx?id=705256705751716947505653724754577158475956585547515556685168505653515257 厦门市轨道交通2号线一期工程土建施工监理1标段招标公告] {{Wayback|url=http://www.cztb.xm.fj.cn/newasite/ShowFile.aspx?id=705256705751716947505653724754577158475956585547515556685168505653515257 |date=20150109154628 }} {{webarchive|url=https://web.archive.org/web/20150109154628/http://www.cztb.xm.fj.cn/newasite/ShowFile.aspx?id=705256705751716947505653724754577158475956585547515556685168505653515257 |date=2015-01-09 }}，厦门工程招投标信息网
</ref>

====3号线====
{{main|厦门轨道交通3号线}}

厦门轨道交通3号线呈西南－东北走向，连接思明区、湖里区、翔安区，构建了本岛与翔安片区快速跨海连接通道，是连接厦门本岛与翔安东部副中心的西南—东北向骨干线。线路起于厦门火车站，终至翔安机场，串联了厦门火车站及周边、湖里老城区、五缘湾片区、翔安海西商贸中心、翔安南部新城、翔安机场及空港经济区等重点区域，预留厦大翔安校区支线条件，并在浦边站预留支线接轨条件。线路全长37.86公里，全线设车站26座，平均站间距1.492公里，于岛内设五缘湾停车场，于翔安区设蔡厝车辆段。

==== 4号线 ====
{{main|厦门轨道交通4号线}}

厦门轨道交通4号线规划构建厦门新机场与翔安、同安各个组团及厦门北站枢纽快速连接通道。线路起于翔安机场，经过翔安南部新城、同安新城、厦门北站、集美后溪、灌口、海沧马銮湾，止于海沧嵩屿码头，长69.6km。

====6号线====
{{main|厦门轨道交通6号线}}

厦门轨道交通6号线：海沧马銮湾为起点站，终点由同安影视城调整为翔安舫阳附近。（西端预留向漳州角美方向延伸条件）

=== 近期规划 ===
====5号线====
厦门轨道交通5号线为厦门岛到翔安、同安的放射骨架线，构建厦门岛与翔安、同安快速跨海连接通道，并服务于厦门岛东部会展中心和同安旧城。

====7号线====
厦门轨道交通7号线从厦门大学出发，沿岛内西侧、北侧由集美大桥附近跨海到翔安，最终到达翔安马巷。（南端预留向漳州港方向延伸条件）

====8号线====
厦门轨道交通8号线从海沧新阳片区出发，经岛内到翔安，与4号线汇合到达翔安机场。（预留城际轨道直通），为机场快线，于翔安片区的4号线位与8号线共线，8号线大嶝站—翔安机场站与3、4号线并行。

=== 远期规划 ===

====9号线====
厦门轨道交通9号线起于翔安浦边村附近，中间与4号线汇合绕翔安南部新城，最终到达同安五显附近。

====10号线====
厦门轨道交通10号线北起于同安洪塘附近，一路经翔安火炬园、东部市级中心、厦门大学翔安校区等，终点为大嶝岛西南部。

====11号线====
厦门轨道交通11号线为原规划4号线，即：BRT改造轻轨线。该线路是南北向辅助线，构建了本岛与集美和大同组团快速连接通道，是1号线北向出岛的有力补充。线路大部分沿用厦门快速公交2号线线位，全长40.4公里。全线设车站31座，平均站间距1.3公里。岛内设置县后停车场，大同组团设置四口圳车辆段，同时作为线网备选综合车辆维修基地。

== 广播 ==
廈門地鐵車廂内廣播使用三種語言，先後依次為[[普通話|普通話]]、[[廈門話|閩南語廈門話]]和[[英語|英語]]<ref>廈門地鐵語音報站錄影 https://www.youtube.com/watch?v=gn32_QaAlsI</ref>，與[[台北捷運|台北捷運]]、[[高雄捷運|高雄捷運]]及[[福州地鐵|福州地鐵]]相似。
繼[[廣州地鐵|廣州]]、[[深圳地鐵|深圳]]、[[佛山地鐵|佛山]]、[[東莞軌道交通|東莞]]及[[福州地鐵|福州]]後，[[廈門地鐵|廈門地鐵]]是第六座全線提供方言廣播的城市。

== 票务 ==

=== 定价 ===
'''起步价：'''4公里（含）2元。

'''里程价：'''

超过4公里后，4-12公里（含），按每4公里加收1元计价（不足4公里按4公里计价）；

12-18公里（含），按每6公里加收1元计价（不足6公里按6公里计价）；

18-28公里（含），按每10公里加收1元计价（不足10公里按10公里计价）；

28-43公里（含），按每15公里加收1元计价（不足15公里按15公里计价）；

43公里以上按每20公里加收1元计价（不足20公里按20公里计价）。

乘客普遍认为厦门地铁票价偏高。在民众最常乘坐的里程上，厦门的票价比上海、北京的更贵。

=== 优惠政策 ===

持易通卡刷卡的，按票价9折优惠。

本市学生持学生卡刷卡的，按票价5折优惠。

本市65周岁以上老人持专用易通卡、本省70周岁以上老年人持《福建省老人优待证》每日高峰时段（早上7:00-9:00，晚上17:00-19:00）按票价5折优惠，其余时段免费。

本市的劳模、烈属、离休人员、重点优抚对象等持专用的易通卡刷卡的，给予免费。

持现役军人（含武警）的相关证件，解放军的《离休干部荣誉证》、《军官退休证》、《文职干部退休证》，伤残人员的《残疾人证》、《残疾军人证》、《伤残国家机关工作人员证》、《伤残人民警察证》、《伤残民兵民工证》，《老干部离休荣誉证》，《反扒执勤证》等乘车的，给予免费。

每位成人可以免费携带一名身高1.2米以下儿童。
