'''厦门国际马拉松赛'''，是由[[中國田徑協會|中國田徑協會]]和[[厦门市人民政府|厦门市人民政府]]主辦的体育比赛，經[[國家體育總局|國家體育總局]]批准，于2003年创办，并在每年的一月份举行。赛事由[[中央电视台|中央电视台]]和[[厦门电视台|厦门电视台]]直播，国内和国外电视台也有转播。厦门国际马拉松赛在2008年1月4号被授予“２００８年国际田联金牌路跑赛事”证书，从而成为国际田联2008年的金牌赛事<ref>{{Cite web |url= http://news.xinhuanet.com/sports/2008-01/04/content_7365737.htm |title= 国际田联授北京、厦门国际马拉松赛金牌证书|author=  |date=2008-01-04|publisher= 新华网 |language= |accessdate= 2014-01-02 |quote= }} </ref>。

==2014年厦门国际马拉松赛==
'''2014年厦门国际马拉松赛'''于2014年1月2日在[[厦门市|厦门市]]举行。比赛的冠军将得到高达4万美元奖金，如果打破赛会纪录，还将会另外得到1万美元，而创造世界纪录的选手则更会被重奖100万美元。最后，男子组冠军由[[肯尼亚|肯尼亚]]选手马里科.凯伯春姆巴（另译为"马里科.基普楚巴"）以2小时08分03秒的成绩夺得，但没能延续连续6年打破男子纪录的势头；而女子组冠军则由来自[[埃塞俄比亚|埃塞俄比亚]]的马瑞-迪巴巴以2小时21分37秒夺得，并打破尘封了6年的赛会纪录<ref>{{Cite web |url= http://sports.sina.com.cn/o/2014-01-02/10296962815.shtml?c=spr_mthz_hao360_sports_home_t001|title= 厦门马拉松39岁肯尼亚老将夺冠 猛女打破赛会纪录|author=  |date=2014-01-02|publisher= 新浪体育 |language= |accessdate= 2014-01-02 |quote= }} </ref>。虽然该赛事在中国举行，然而已连续5年无中国选手夺冠。<ref>{{Cite web |url=http://sports.21cn.com/integrate/other/a/2014/0102/11/25829187.shtml |title=厦门马拉松女子纪录告破 中国选手连续5年无冠 |access-date=2014-01-02 |archive-url=https://web.archive.org/web/20140102192511/http://sports.21cn.com/integrate/other/a/2014/0102/11/25829187.shtml |archive-date=2014-01-02 |dead-url=yes }}</ref>
