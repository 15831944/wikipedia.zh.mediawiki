'''何辉'''（{{bd|1911年||1996年|11月5日}}）[[湖南省|湖南省]][[平江县|平江县]]人，[[中国人民解放军开国少将|中国人民解放军开国少将]]。

==生平==
1929年加入[[中国工农红军|中国工农红军]]，1932年加入[[中国共产主义青年团|中国共产主义青年团]]，1933年转入[[中国共产党|中国共产党]]。历任宣传员、宣传队长、湘赣军区后方医院俱乐部主任、科长、师敌工部部长、支队政委、旅政治部主任、[[山西省军区|山西军区]]政治部主任、[[天津警备区|天津警备区]]政治部主任兼塘沽水警区副政委、海军第二航空学校政治委员等职。曾先后参加[[中央苏区|中央苏区]]历次反“围剿”、[[二万五千里长征|二万五千里长征]]、晋西北1940年反“扫荡”和[[晋中战役|晋中战役]]、[[太原战役|太原战役]]等战役<ref name=cpc>{{Cite web |url =http://dangshi.people.com.cn/GB/165617/165618/166491/167909/12099009.html  |title = 何辉 |author =  |date =  |publisher =中国共产党新闻网  |language =  |accessdate = 2017-04-29 |quote =  }}</ref>。1955年，授予[[中国人民解放军少将|中国人民解放军少将]]军衔<ref>{{cite news|title=揭秘：中共首批开国将军的级别是怎样划分的？|url=http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml|accessdate=2012-06-15|newspaper=凤凰网（转自摘自《党史博览》2010年第5期）|date=2010-06-13|deadurl=yes|archiveurl=https://web.archive.org/web/20120620091504/http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml|archivedate=2012-06-20}}</ref><ref name=cpc/>。

1996年11月5日，何辉在[[北京|北京]]病逝，享年85岁<ref name=cpc/>。
