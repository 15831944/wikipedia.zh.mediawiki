'''何景賢'''（{{bd|1935年|1月}}）是[[中華語文研習所|中華語文研習所]]的創始人。他出生於[[中國|中國]][[廣州|廣州]]，致力於中文教育推廣與文化保存<ref>{{Cite journal |title=文化产业，全力冲刺当有可为——专访中华语文研习所校长何景贤 |author=時光 |url=http://www.cqvip.com/qk/84003x/200809/28186630.html |date=2008 |journal=中國報導 |issue=9 |pages=106–107}}</ref>，早年與[[中華民國|中華民國]]政府密切合作以中文教學來推廣外交，近期則將重心轉向推動[[中國|中國]]與[[台灣|台灣]]中文用法交流。

==早年生活==
何景賢於1931年在[[中國|中國]][[廣州|廣州]]出生，起初由外祖母撫養，後於1937年搬往[[北京|北京]]與父母同住<ref name="autobib">{{Cite news |url=https://www.worldcat.org/oclc/911051309 |title=漢教之父：何景賢口述歷史 |author=何景賢 |date=2015-01-31 |isbn=9789574320240 |oclc=911051309}}</ref>。據其在自傳中回憶，在北京居住的十年中，他注意到來自[[台灣|台灣]][[嘉義|嘉義]]的父親與來自中國廣州的母親的[[口音|口音]]都與北京當地的口音有很大的差異，這個差異是他後來投入漢語教育的遠因之一<ref name="autobib"/>。1947年，何景賢隨[[中國國民黨|中國國民黨]]政府來到台灣之後就讀於[[淡江大學|淡江英專（今淡江大學）]]，畢業後至[[密西根大學|密西根大學]]取得語言學碩士學位<ref name = "top10">{{Cite news |url=http://cdp.sinica.edu.tw/cchsieh/news/hcc-news-04-2.pdf |title=十大傑出青年　昨經投票選出 |date=1967-10-07 |work=[[聯合報|聯合報]]}}</ref>。

==推廣中文教育==
當時在台灣宣教的安篤思牧師（{{lang-en|Rev. Egbert W. Andrews}}）由於希望提供傳教士學習中文的管道而與何景賢於1956年在[[紐澤西|紐澤西]]創立了基督教語文學院（{{lang-en|Missionary Language Institute}}）<ref name = "mandarinteach">{{Cite book |url=https://www.worldcat.org/oclc/1047559164 |title=Early 21st-century power struggles of Chinese languages teaching in US higher education |last=Chen |first=Ya-chen |isbn=9781527513501 |location=Newcastle upon Tyne, UK |oclc=1047559164}}</ref>，這是華語區第一個針對非母語使用者的中文教學為目標的機構<ref name = "mandarinteach"/><ref>{{Cite news |url=https://usatoday30.usatoday.com/money/world/2008-03-23-mandarin_N.htm |title=China exudes opportunity for new generation |author=Tini Tran |date=2008 |work=[[美聯社|美聯社]] |accessdate=2019-01-20}}</ref>。何景賢於1966年自[[布朗大學|布朗大學]]結束進修後接任[[台北語文學院|台北語文學院]]{{noteTag|1958年時基督教語文學院為反映其教學範圍不僅限於傳教士，因而更名為「台北語文學院」。}}院長<ref name="autobib"/>。由於[[中華人民共和國|中華人民共和國]]政府在[[冷戰|冷戰]]結束前不允許人民自由出入境，此時期的[[中華民國|中華民國]]政府遂積極投入中文教學活動<ref name = "mandarinteach"/>。1967年，時任[[中國青年救國團|中國青年救國團]]主任的[[蔣經國|蔣經國]]邀請何景賢協助辦理海外青年暑期返國研習營，開啟了其與政府的長期合作<ref name="autobib"/>。同年，何景賢受[[德里大學|德里大學]]邀請赴[[印度|印度]]協助該校規劃中文教學體系<ref>{{Cite book |url=https://www.worldcat.org/oclc/974287285 |title=India and Taiwan : from benign neglect to pragmatism |last=1950- |first=Tripathi, D. P. (Devi Prasad), |last2=R., |first2=Deepak, B. |isbn=9789384464912 |location=New Delhi, India |oclc=974287285}}</ref>，並因推廣中文教育的成績獲選為[[中華民國十大傑出青年|十大傑出青年]]<ref name = "top10"/>。1976年，為強化執行以中文教學為國民外交手段的政策<ref name = "mandarinteach"/>，時任[[教育部|教育部]]長[[蔣彥士|蔣彥士]]指示何景賢籌備中華語文研習所，強化與政府的合作。該年稍後，台北語文學院改制為中華語文研習所<ref name="autobib"/>。

除了在中華語文研習所了開發上百種海外中文教學教材外<ref>{{Cite web |url=http://iweb.ntnu.edu.tw/tcsl/Newsletter/news1998-2002/1999spring.htm#langcen |title=對外華語通訊：1999春季號 |accessdate=2019-01-20 |work=iweb.ntnu.edu.tw |publisher=國立師範大學國語中心 |archiveurl=https://web.archive.org/web/20190120045855/http://iweb.ntnu.edu.tw/tcsl/Newsletter/news1998-2002/1999spring.htm |archivedate=2019-01-19}}</ref>，何景賢亦積極參與設計[[台灣|台灣]]的中文課程。其曾於1984年參與設計以[[羅馬字|羅馬字]]表記的國語注音符號第二式<ref>{{Cite book |title=注音符號的文化演現 |author=何秋堇 |publisher=秀威資訊 |year=2012-05-10}}</ref>、與[[中華民國僑務委員會|僑務委員會]]合作編寫兒童華語課本、初中華語課本<ref>{{Cite web |url=https://gpi.culture.tw/search?q=%E4%BD%95%E6%99%AF%E8%B3%A2&qt=3 |title=政府出版品資訊網：何景賢 |accessdate=2019-01-20 |author=[[中華民國文化部|中華民國文化部]] |language=zh-Hant}}</ref>、以及指導修訂教育部國語辭典等等<ref>{{Cite web |url=http://dict.revised.moe.edu.tw/xddic_utf8/portal_c1_cnt.php?button_num=c1&folder_id=18 |title=教育部重編國語辭典修訂本 |accessdate=2019-01-20 |author=[[中華民國教育部|中華民國教育部]] |work=dict.revised.moe.edu.tw}}</ref>。在中國[[改革開放|改革開放]]之後，何景賢致力於與中國的文化交流<ref>{{Cite news |url=http://mall.cnki.net/magazine/Article/SIJI200210006.htm |title=中国语言文化的传播者 记语言学家、教育家何景贤博士 |author=石頭 |date=2012 |work=21世紀 |publisher=中国国际青年交流中心;中日青年交流中心;中国青年国际人才交流中心 |issn=1005-2984}}</ref>，為避免雙方的漢字使用差異妨礙溝通，其與[[北京語言大學|北京語言大學]]合作，在2005年推出《两岸现代汉语常用词典》<ref>{{Cite web |url=http://www.cctv.com/taiwan/20050608/102075.shtml |title=专访台湾中华语文研习所董事长何景贤  |accessdate=2019-01-20 |author=李峰 |date=2005-06-04 |work=www.cctv.com |publisher=央視國際}}</ref>。在2009年的[[两岸经贸文化论坛|國共論壇]]中，何景賢也建議中國與台灣民間應合作共同編纂一套新的《中华大辞典》<ref>{{Cite web |url=http://news.sina.cn/?sa=t124d2197637v71&from=wap&vt=4 |title=两岸建议缩小汉字使用差异 |accessdate=2019-01-20 |date=2009-07-13 |work=news.sina.cn |publisher=新京報}}</ref>。

==其他經歷==
為處理[[二二八事件|二二八事件]]，[[中華民國行政院|中華民國行政院]]於1991年成立[[行政院研究二二八事件小組|行政院研究二二八事件小組]]，何景賢為小組委員之一，參與分析官方一手資料與採訪事件見證人的過程，其結果發表為《「二二八事件」研究報告》<ref>{{Cite web |url=http://www.228.org.tw/228_overview.html |title=二二八事件介紹 |accessdate=2019-01-20 |author=二二八事件紀念基金會 |work=www.228.org.tw |publisher=二二八事件紀念基金會 |archiveurl=https://web.archive.org/web/20190120061953/http://www.228.org.tw/228_overview.html |archivedate=2019-01-20}}</ref>。
