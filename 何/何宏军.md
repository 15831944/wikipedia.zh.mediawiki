'''何宏军'''（{{bd|1961年|8月||}}）[[陕西|陕西]][[洋县|洋县]]人，[[中国人民解放军少将|中国人民解放军少将]]，[[第十三届全国人民代表大会代表|第十三届全国人民代表大会代表]]<ref>{{cite news|title=全国人民代表大会|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=200813284688|accessdate=2018-06-23}}</ref>。

==生平==
何宏军曾任[[新疆军区|新疆军区]]某师政委。2012年12月任[[青海省军区|青海省军区]]政治部主任<ref name=caixin>{{Cite web |url =http://china.caixin.com/2017-05-06/101087160.html  |title = 何宏军少将任军委政治工作部主任助理 |author =  |date = 2017-05-06 |publisher = 财新网 |language =  |accessdate =  |quote =  }}</ref>。2014年，何宏军任[[中国人民解放军总政治部干部部|中国人民解放军总政治部干部部]]副部长兼老干部局局长<ref>{{Cite web|url =http://news.cyol.com/content/2014-06/05/content_10176452.htm|title =总政黄寺第二干休所：老干部满意是最高标准|author =|date =2014-06-05|publisher =中青在线|language =|accessdate =|quote =}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。2017年，任[[中央军委政治工作部|中央军委政治工作部]]主任助理。同时兼任[[国务院军队转业干部安置工作小组|国务院军队转业干部安置工作小组]]副组长<ref name=caixin/><ref>{{Cite web |url =http://news.163.com/17/0506/19/CJPCEUHT000187VE.html  |title = 何宏军任中央军委政治工作部主任助理 |author =  |date =2017-05-06  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2013年7月晋升少将军衔。
