'''何雨虹'''（{{bd|1996年|05月02日}}），[[中国|中国]]新人女演員，出生于[[山東|山東]][[青島|青島]]，擅长中国舞，咏春拳。<ref name="a3">{{Cite web |url=http://ent.cri.cn/20171212/417edc63-1769-88f5-bac1-b1a7fc536956-3.html|title=《最美的青春》启用新人 何雨虹搭刘智扬引期待|accessdate=2018-11-27 |author= |date=2017-12-12 |publisher=国际在线综合|language=Zh-hans }}</ref>大三期間在學校排練話劇時，因良好的專業素養和獨特的氣質受到導演[[郭靖宇|郭靖宇]]賞識，進而出演電視劇《[[最美的青春|最美的青春]]》進入演藝圈<ref name="a1">{{Cite web |url=http://wemedia.ifeng.com/74051284/wemedia.shtml |title=濰坊這位95後姑娘成了央視正在熱播電視劇女主角，快看看是誰|work=|publisher=大风号|language=Zh-hans|access-date=2018-11-04|archive-url=https://web.archive.org/web/20181104165957/http://wemedia.ifeng.com/74051284/wemedia.shtml|archive-date=2018-11-04|dead-url=yes}}</ref><ref>{{Cite web |url=http://news.yule.com.cn/html/201808/278417.html |title=《最美的青春》女主何雨虹：演這部戲是一種洗禮|accessdate=2018-11-04 |author= |date=2018-08-24 |publisher=中國娛樂網 |language=Zh-hans }}</ref> 2018年接拍電視劇《新˙小娘惹》<ref>{{Cite web |url=https://www.zaobao.com.sg/zentertainment/movies-and-tv/story20181011-898274 |title=10年後重拍 新《小娘惹》主角換中台演員|accessdate=2018-11-04 |author= |date=2018-10-11 |publisher=新明日報 |language=Zh-hans }}</ref>

==生平==
從未有過表演基礎的她，高中時感覺表演很有趣，抱著體驗一下的心態參加了藝考，卻同時斬獲了[[上海戲劇學院|上海戲劇學院]]導演系第5名、[[解放軍藝術學院|解放軍藝術學院]]表演第12名、[[北京電影學院|北京電影學院]]表演系第17名和[[中央戲劇學院|中央戲劇學院]]的合格证<ref name="a1">{{Cite web |url=http://wemedia.ifeng.com/74051284/wemedia.shtml |title=濰坊這位95後姑娘成了央視正在熱播電視劇女主角，快看看是誰|work=|publisher=大风号|language=Zh-hans|access-date=2018-11-04|archive-url=https://web.archive.org/web/20181104165957/http://wemedia.ifeng.com/74051284/wemedia.shtml|archive-date=2018-11-04|dead-url=yes}}</ref><ref name="a2">{{Cite web |url=https://www.yikaosheng.com/teacher/217029/index.html |title=何雨虹-介紹|work=|publisher=藝考生-為夢想不將就-百萬藝考生都在用！|language=Zh-hans|access-date=2018-11-04|archive-url=https://web.archive.org/web/20181104121235/https://www.yikaosheng.com/teacher/217029/index.html|archive-date=2018-11-04|dead-url=yes}}</ref>後選擇中央戲劇學院就讀。

大三在學校排練話劇時，導演[[郭靖宇|郭靖宇]]來中戲參訪，因良好的專業素質和天然、清新的外形被導演發現，於2017年9月加盟電視劇《[[最美的青春|最美的青春]]》的拍攝正式進入演藝圈。<ref name="a3">{{Cite web |url=http://ent.cri.cn/20171212/417edc63-1769-88f5-bac1-b1a7fc536956-3.html|title=《最美的青春》启用新人 何雨虹搭刘智扬引期待|accessdate=2018-11-27 |author= |date=2017-12-12 |publisher=国际在线综合|language=Zh-hans }}</ref>
