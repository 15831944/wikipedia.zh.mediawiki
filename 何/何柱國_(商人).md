'''何柱國'''{{大紫荊}}（{{bd|1949年|6月23日午時||catIdx=何}}），[[香港親共人士|香港親共人士]]、商人、[[星島新聞集團|星島新聞集團]]、[[香港煙草有限公司|香港煙草有限公司]][[董事會主席|董事會主席]]，[[香港煙草|香港煙草]]創辦人[[何英傑|何英傑]]長孫，祖籍[[福建|福建]]<ref>[http://www.fjqw.gov.cn/qbcmsapp/view/zh-cn/news.jsp?cateunid=489CA54746BBEF46B93299CB1299E65A&unid=6846A8947754ED8526B32705E630BC66　福建省政府福建僑網政協常委名單] {{Cite web |url=http://www.fjqw.gov.cn/qbcmsapp/view/zh-cn/news.jsp?cateunid=489CA54746BBEF46B93299CB1299E65A&unid=6846A8947754ED8526B32705E630BC66 |title=存档副本 |access-date=2013-04-07 |archive-url=https://web.archive.org/web/20140819095623/http://www.fjqw.gov.cn/qbcmsapp/view/zh-cn/news.jsp?cateunid=489CA54746BBEF46B93299CB1299E65A&unid=6846A8947754ED8526B32705E630BC66 |archive-date=2014-08-19 |dead-url=yes }}</ref><ref>[http://www.fjqw.gov.cn/qbcmsapp/view/zh-cn/news.jsp?cateunid=489CA54746BBEF46B93299CB1299E65A&unid=FFA5E13BA742E3E44D723078EB3C7B18 福建省政府福建僑網政協常委及政協委員名單]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

何柱國亦有踏足政界，與[[中國大陸|中國大陸]]官方關係良好，1998年獲委任為[[全國政協委員|全國政協委員]]<ref>[http://www.cppcc.gov.cn/CMS/icms/project1/cppcc/wylibary/changWuWeiYuanList.jsp 全國政協常委及政協委員名單]</ref>。他亦是[[北京大學|北京大學]]校董。

== 家族 ==
何柱國在[[拔萃男書院|拔萃男書院]]畢業後，赴[[美國|美]]升學，1990年代返港，接手家族生意。現為[[香港煙草|香港煙草]]董事會主席、[[星島新聞集團有限公司|星島新聞集團有限公司]]董事會主席。何柱國繼承的遺產估計超過二百億港元。

何柱國已婚，妻子是黃敏珊，二人育有一子一女。何柱國的胞弟[[何定國|何定國]]是[[加拿大華人|加拿大華人]]企業家，是[[加拿大|加拿大]][[國龍航空|國龍航空]]的創辦人。

== 與梁振英的關係 ==
2011年12月，正值參選[[香港行政長官|香港行政長官]]的[[梁振英|梁振英]]稱：《[[星島新聞集團|星島新聞集團]]》多篇報導指其主業公司[[戴德梁行|戴德梁行]]面臨要清盤，發表質疑梁的管治能力等言論。他又不點名批評《星島》記者大聲向他提問，故意令他難堪，是針對其本人，有動機地抹黑。何柱國對此指控不滿，並召開記者會高調反駁，稱早前《星島》旗下報刊指梁於戴德梁行「一鋪清袋」（一無所有），內容有事實基礎，何柱國又認為，梁振英出來選舉，作為政客，要有心理準備向記者清楚交代，不然不要出來，但不要趁機罵記者等多番言論，但指星島將繼續支持特區政府。

== 反對佔中 ==
何柱國反對[[雨傘革命|佔領中環]]，指佔領行動有源源不絕的資源是有組織有預謀，批評學生霸佔馬路的行為自私，又表示「[[周永康_(香港)|周永康]]去到旺角都唔代表旺角嘅」，學生不能代表很多香港人，只能代表他們自己，更稱「學生無資格代表我」，若他是警方「一早已經清場」。何柱國直斥佔中三子被別人騎劫佔領行動，「無用㗎啦」，他指學生不可能要求特首梁振英下台以及人大撤回決定，又強調市民需以合法方式爭取民主，聲稱佔領行動會令未來一至兩個月出現失業潮，或者令部分店舖關閉。他稱警方近日的行動十分克制，專業行動值得尊重，又指示威者用雨傘作為工具會插傷別人，高舉雙手時用腳踢警察，並不是和平示威。<ref>[http://hk.apple.nextmedia.com/realtime/news/20141018/53032294 【雨傘革命】何柱國斥學生自私「無資格代表我」]香港蘋果日報</ref>

==「傻仔白痴」論 ==
2016年4月3日，何柱國指前[[學民思潮|學民思潮]]召集人[[黃之鋒|黃之鋒]]不了解香港經營環境。黃之鋒反駁指，何柱國對他的批評是幼稚的情緒發洩。何柱國指黃之鋒所說，大企業沒有裁員，屬「傻仔」及「白痴」想法，並指實情是滿街皆有舖位空置。4月4日，黃之鋒反駁指：「作為一個在社會上有名望的傳媒大亨，用這種純情緒式的發洩字眼去攻擊和自己政見不同的人，我會覺得這是頗幼稚的行為。」他又表示：「[[港澳個人遊|自由行]]拉高租金，令很多小商戶承擔不了租金，然後引致最後他們被迫結業，現在就是因為依賴自由行那種泡沫經濟爆破了，才導致現在的局面。」何柱國不同意黃之鋒指他幼稚的講法：「我在2014年[[讓愛與和平佔領中環|佔中]]時已說過，經濟會因為佔中而帶來很多失業和後遺症。」何柱國繼續表示：「我現在向他挑戰，我和他一起做去[[銅鑼灣|銅鑼灣]]及[[尖沙咀|尖沙咀]][[廣東道|廣東道]]做記錄，大家逛街，叫他拿出計算機，數一下有多少空置舖位，以及那些舖在2014年是否存在。如果他是對的，我向他下跪叩三個響頭，我承認我是『白痴』。如果他是錯的，就叫他閉嘴，向我鞠三個躬，自認是『傻仔』、『白痴』。」黃之鋒回覆時指，不會接受何柱國的挑戰。<ref>{{cite web|url=https://www.youtube.com/watch?v=4XZK8-WZNVY&nohtml5=False | title=何柱國發挑戰令圖證經濟狀況 黃之鋒拒接受 |publisher=[[無綫新聞|無綫新聞]] |date=2016年4月4日}}{{zh-hant}}</ref>
