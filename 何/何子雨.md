'''何子雨'''（{{bd|1989年|6月10日|2017年|11月7日|}}），[[中華民國空軍|中華民國空軍]][[上尉|上尉]]、[[幻象2000|幻象2000]][[戰鬥機|戰鬥機]][[飛行員|飛行員]]，[[台北市內湖區|台北市內湖區]]人，畢業於[[國立成功大學|國立成功大學]]中文系，自成大學士畢業後報考[[空軍軍官學校|空軍軍官學校]]飛行常備軍官班。2017年11月，駕機執行訓練時[[失踪|失踪]]。2018年5月，因公失踪半年后，军方按程序宣告死亡。6月，[[追晋|追晋]]少校军衔。12月4日，空軍於光點消失附近海域1.3公里處尋獲戰機鼻輪、減震支柱及發動機零件等項，初步研判應屬失聯幻象戰機殘骸<ref>{{cite news|title=找了超過一年！空軍宣布尋獲失蹤幻象部分殘骸|publisher=聯合新聞網|url=https://udn.com/news/amp/story/10930/3518336|date=2018-12-4|author=程嘉文|language=zh-hant}}</ref>。

== 生平 ==
何子雨就讀[[臺北市立麗山高級中學|臺北市立麗山高級中學]]，畢業於[[國立成功大學|國立成功大學]]中國文學系學士<ref name = T808989>{{cite news|title=結婚剛滿1周年！ 飛官何子雨驚傳墜機意外|publisher=TVBS|url=http://news.tvbs.com.tw/local/808989|date=2017-11-07|author=陳頡|accessdate=2018-03-23|language=zh-hant}}</ref>。何子雨的大學導師蘇敏逸對他的印象是擔任班代時，常要服務老師學生，都很負責細心的完成<ref name = E9817357/><ref>{{cite news|title=飛官何子雨大學是橄欖球校隊　運動細胞強|publisher=蘋果日報|url=https://tw.appledaily.com/new/realtime/20171108/1237054/|date=2017-11-08|author=辛啟松|accessdate=2018-03-23|language=zh-hant}}</ref><ref>{{cite news|title=上尉飛官何子雨失聯 成大師生心急|publisher=自由時報|url=http://news.ltn.com.tw/news/politics/breakingnews/2246926|date=2017-11-08|author=劉婉君|accessdate=2018-03-23|language=zh-hant}}</ref><ref>{{cite news|title=飛官何子雨大學唸中文轉飛官 在班上特別顯眼|publisher=聯合報|url=https://udn.com/news/story/10930/2804781|date=2017-11-08|author=修瑞瑩|accessdate=2018-03-23|language=zh-hant}}</ref><ref>{{cite news|title=幻象機失聯 成大老師：何子雨內歛不外放|publisher=中央社|url=http://www.cna.com.tw/news/firstnews/201711080136-1.aspx|date=2017-11-08|author=張榮祥|accessdate=2018-03-23|language=zh-hant}}</ref>。

在大學畢業後，2011年以第一名考取[[中華民國國防部|中華民國國防部]]空軍飛行常備軍官班，於2013年畢業<ref>{{cite news|title=幻象機失聯飛官　飛行時數近500小時|publisher=今日新聞|url=https://www.nownews.com/news/20171107/2640150|date=2017-11-07|author=呂烱昌|accessdate=2017-11-13|language=zh-hant}}</ref><ref>{{cite news|title=影》何子雨榜首考取飛官 曾現身蔡英文秋節餐會|publisher=中國時報|url=http://www.chinatimes.com/realtimenews/20171108004019-260405|date=2017-11-08|author=鄭仁宗、陳文越、杜宇喬|accessdate=2017-11-14|language=zh-hant}}</ref>。何子雨除了駕駛幻象戰機外，還飛過[[F-5戰鬥機|F-5戰鬥機]]、[[AT-3自強號高級教練/輕攻擊機|AT-3自強號高級教練/輕攻擊機]]等機型，但未出國受過訓<ref>{{cite news|title=何子雨成大畢業報考飛官 在校表現不錯|publisher=中央社|url=http://www.cna.com.tw/news/firstnews/201711070455-1.aspx|date=2017-11-07|author=劉麗榮|accessdate=2017-11-13|language=zh-hant}}</ref>。2016年5月，何子雨因傷停飛，於2017年1月復飛<ref name = E9817357>{{cite news|title=台幻象機失聯 飛官何子雨曾因傷停飛|publisher=大紀元|url=http://www.epochtimes.com/b5/17/11/8/n9817357.htm|date=2017-11-08|author=鍾元|accessdate=2018-03-23|language=zh-hant}}</ref>。

== 失联与搜寻 ==

事发时，何子雨服役于[[新竹空軍基地|新竹空軍基地]]499聯隊<ref name = T808989/>。2017年11月7日，何子雨駕駛空軍編號2040的[[幻象2000-5|幻象2000-5]]單座戰機，於晚間執行夜間訓練時失聯。[[行政院海岸巡防署海洋巡防總局|行政院海岸巡防署海洋巡防總局]]表示，11月7日晚間7時10分接獲通報，一架[[中華民國空軍|中華民國空軍]][[幻象2000戰鬥機|幻象2000戰鬥機]]於[[基隆市|基隆市]]北北東90浬處失聯；即刻指派北部地區機動海岸巡防隊基隆艦、東部地區機動海岸巡防隊花蓮艦及基隆10033艇緊急出勤，前往現場進行搜尋<ref>{{cite news|title=幻象機基隆外海失聯 海巡2艦1艇搜尋|publisher=中央社|url=http://www.cna.com.tw/news/aipl/201711070397-1.aspx|date=2017-11-07|author=王朝鈺|accessdate=2017-11-16|language=zh-hant}}</ref>。

11月8日，在中华民国空军召开的记者会上，[[國防部空軍司令部|國防部空軍司令部]]副司令[[张哲平|张哲平]]中将表示，除王同义在法国受训失事外，这是幻象战机第5次意外，共有4位飞行员[[殉职|殉职]]。因事涉机密，过往失事调查并未完整公布<ref name = E9817357/>。當時記者會上，[[美國|美國]]的[[自由亞洲電台|自由亞洲電台]]記者詢問「這架飛機有沒有可能貼海飛行，避過雷達搜索，未排除駕機投共的可能？」，张哲平回应「不太可能」<ref name="a2"/>。

9日，[[立法院|立法院]]答询，立委[[王定宇|王定宇]]質詢國防部部長[[馮世寬|馮世寬]]，當飛官何子雨還在海上等著被救，居然有人質疑是投共，國防部長怎麼想？馮世寬說非常氣憤，接著馮世寬拉高音量說，「我不在現場，我在現場會罵『他媽的』，怎麼可以這樣講！」王定宇則說，國會殿堂不建議說這三個字（「他媽的」），但馮世寬豪爽回應：「我就會這樣說！簡直可惡！」國民黨立委[[馬文君|馬文君]]質詢時，馮世寬再說，他非常遺憾。當時他不在現場。在現場可能『不只罵[[三字經|三字經]]』。他說，飛行員為了保護國人，夜間也在訓練，犧牲與家人在一起的時間，沒有得到一點掌聲，卻說是投共，「我簡直無法容忍」。他還說國軍，「我們一向對媒體的回應都太軟弱了」。對於立委詢問，是那個媒體在記者會提出投共問題的，這時國防部發言人陳中吉靠近說，是[[自由亞洲電台|自由亞洲電台]]提問。馮世寬聽到後則下令，「下次我們任何軍事會議，不能邀請他<ref name="a2">{{cite news|title=外媒質疑何子雨投共　馮世寬：他媽的！怎麼可以這樣子想？|publisher=上報|url=http://www.upmedia.mg/news_info.php?SerialNo=28578|date=2017-11-09|author=趙翊妏|accessdate=2018-03-23|language=zh-hant}}</ref><ref>{{cite news|title=飛官投共？馮世寬怒斥：他媽的！以後不准這媒體來|publisher=新頭殼|url=https://newtalk.tw/news/view/2017-11-09/103220|date=2017-11-09|author=林朝億|accessdate=2018-03-23|language=zh-hant}}</ref>。」

11月17日，[[中華民國海軍|中華民國海軍]]完成殘駭分布定位<ref>{{cite news|title=海軍已完成殘駭分布定位　最快下周二打撈幻象黑盒子|publisher=蘋果日報|url=https://tw.news.appledaily.com/politics/realtime/20171118/1243435/|date=2017-11-18|author=陳培煌|accessdate=2017-11-18|language=zh-hant}}</ref>。

2018年1月的报道，国防部表示，已出动空军军机201架次、海军军舰152艘次，海巡舰艇46艘次，空勤直升机7架次，另委商打捞船25艘次。参与搜寻的官兵仍不放弃任何机会，将继续搜寻何子雨<ref>{{cite news|title=失蹤飛官妻訴哀情 馮世寬：國軍從未忘記何子雨|publisher=聯合報|url=https://udn.com/news/story/10930/2909924?from=udn-referralnews_ch2artbottom|date=2018-01-03|author=程嘉文|accessdate=2018-03-23|language=zh-hant}}</ref>。

2018年3月報導，海空搜索、打撈至今，機上飛官何子雨上尉仍生死未卜。嚴德發上午列席立法院外交及國防委員會備詢，他在回應民進黨立委羅致政質詢時表示，國軍關心袍澤，幻象戰機打撈委商雖然已告一段落，並提出報告，但國軍對該海域的海空搜索仍然持續進行，國軍不放棄搜救，海空機、艦搜索何子雨行動將持續下去<ref>{{cite news|title=影／回應何子雨妻關切 嚴德發：搜救行動持續|publisher=聯合新聞網|url=https://udn.com/news/story/10930/3042878|date=2018-3-21|author=洪哲政|accessdate=2018-03-21 09:46|language=zh-hant}}</ref>。

2018年5月報導，歷經半年的搜索打撈，始終沒有尋獲殘骸。由於軍人失蹤半年後就宣告死亡，進入撫卹程序。對於因公失蹤的官兵，前六個月仍照常發放薪水，滿六個月後則宣告死亡，進入撫卹程序<ref>{{cite news|title=影／回應何子雨妻關切 嚴德發：搜救行動持續|publisher=聯合新聞網|url=https://udn.com/news/story/10930/3139814|date=2018-05-13|author=程嘉文|accessdate=2018-05-13|language=zh-hant}}</ref>。

2018年12月4日，空軍晚間宣布，已經尋獲戰機鼻輪、減震支柱與發動機零件等部分殘骸，但尚未尋獲飛行紀錄器（FDR）與飛行員相關物件。2040號機失聯至今以來，共計派遣566批、569架次各型航空器實施搜救，並委由專業打撈公司執行14航次搜尋，日前已於光點消失附近海域1.3公里處尋獲戰機鼻輪、減震支柱及發動機零件等項，初步研判應屬失聯幻象戰機殘骸<ref>{{cite news|title=找了超過一年！空軍宣布尋獲失蹤幻象部分殘骸|publisher=聯合新聞網|url=https://udn.com/news/amp/story/10930/3518336|date=2018-12-4|author=程嘉文|language=zh-hant}}</ref>。

== 追晉空軍少校 ==
2018年（中華民國107年）6月5日，總統府發布總統令「追晉故空軍[[上尉|上尉]]何子雨為空軍[[少校|少校]]，此令自中華民國106年11月7日生效」。空軍司令部將官表示，已認定何子雨為因公殉職，撫卹會依規定從優辦理，並舉辦追晉典禮<ref>{{cite news|title=總統令：追晉何子雨為空軍少校|publisher=中央社|url=http://www.cna.com.tw/news/firstnews/201806050381-1.aspx|date=2018-06-05|author=葉素萍|accessdate=2018-06-06|language=zh-hant}}</ref><ref>{{cite news|title=幻象2000失聯7個月 總統追晉何子雨為空軍少校|publisher=上報|url=https://www.upmedia.mg/news_info.php?SerialNo=42275|date=2018-06-06|author=趙翊妏|accessdate=2018-06-06|language=zh-hant}}</ref>。2018年6月23日星期六於空軍[[新竹機場|新竹基地]]中正堂舉行祈福會。
