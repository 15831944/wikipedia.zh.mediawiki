'''何德华'''（{{lang-en|Teck-Hua Ho}}，{{bd|1961年|5月10日}}）[[新加坡国立大学|新加坡国立大学]]高级常务副校长兼教务长、陈振传百年纪念教授<ref>{{cite web|url=http://news.nus.edu.Sg/highlights/prof-tan-eng-chye-be-next-nus-president|title=Prof Tan Eng Chye to be next NUS President|work=news.nus.com}}</ref> <ref>http://tanfoundation.com.sg/wp-content/uploads/2014/10/Prof_HoTeckHua.pdf</ref>。

== 职业经历 ==
在加入新加坡国立大学之前，何德华曾经担任[[加州大学伯克利分校|加州大学伯克利分校]]{{link-en|哈斯商学院|Haas School of Business}} 威廉·哈尔福德家族（William Halford Jr Family）营销学教授、哈斯商学院亚洲商业中心署长、副院长（学术事务）、营销方向团队主席<ref>{{cite web |url=http://facultybio.haas.berkeley.edu/faculty-list/ho-teck |title=Teck Ho |work=Faculty Directory |publisher=[[Haas_School_of_Business|Haas School of Business]]}}</ref>。

他兼任国立大学医学组织与国家环境局董事局成员<ref>{{cite web|url=http://www.nuhs.edu.sg/about-us/leadership/board-members.html|title=Board Members|work=nuhs.edu.sg}}</ref>、新加坡公共服务学院董事<ref>{{cite web|url=https://www.cscollege.gov.sg/About%20Us/Pages/Fellows.aspx|title=Fellows|work=www.cscollege.gov.sg}}</ref>、陆路交通管理局研究咨询小组主席、新加坡全国人工智能核心计划执行主席<ref>https://www.aisingapore.org/management/</ref>、新加坡数据科学联合平台主席<ref>http://sdsc.sg/?page_id=127</ref>。

== 教育背景 ==
何德华就读于[[新加坡华侨中学|新加坡华侨中学]]，是公共服务委员会（PSC）奖学金的获得者<ref>http://www.zaobao.com.sg/special/face2face/story20081206-28494</ref>。随后在新加坡国立大学获得电机工程一等荣誉学位、电脑及信息科学理学硕士学位。于1993在宾夕法尼亚大学沃顿商学院获得决策科学博士学位<ref>http://cn.nus.edu.sg/about-nus/nus-structure/management/ho-teck-hua</ref>。

== 研究领域 ==
何德华的研究领域包括行为经济学，以数据驱动的决策科学。<ref>{{cite web |url=https://scholar.google.com/citations?user=Wy7ZZTEAAAAJ&hl=en&oi=ao |title=Google Scholar Profile - Teck Ho |publisher=[[Google|Google]]}}</ref> 在2014-2017期间，他担任国际权威杂志《{{link-en|管理科学 (期刊)|Management Science (journal)|管理科学}}》的第一位非美国籍主编。

== 荣誉与获奖情况 ==
2014年，何德华与合作者的研究获得 Wickam Skinner生产和运作管理年会最佳论文奖第二奖；研究论文曾入围市场营销研究领域国际论文最具权威的三大奖项：2011年度威廉·奥戴尔（William F O'Dell）奖、2006年度约翰·利特尔（John DC Little）最佳论文奖、以及2005年度保罗·格林（Paul Green）最佳论文奖。在哈斯商学院任职期间，何德华荣获2015年哈斯商学院最高学院奖《威廉姆森奖》（Williamson Award） <ref>http://newsroom.haas.berkeley.edu/article/prof-teck-ho-receives-prestigious-williamson-award</ref> 。何德华曾于2010年荣获伯克利杰出教学奖 （Berkeley Distinguished Teaching Award）<ref>http://teaching.berkeley.edu/programs/distinguished-teaching-award/past-dta-recipients</ref>，并于2004年、2005年、2006年连续三年荣获厄尔·切特（Earl F Cheit）杰出教学成就奖<ref>http://www.haas.berkeley.edu/faculty/previous_awards.html</ref>。
