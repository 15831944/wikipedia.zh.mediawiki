'''何桂清'''（{{bd|生年不詳||1862年|12月21日|catIdx=H何}}），字'''根雲'''。[[雲南省|雲南省]][[雲南府|雲南府]]昆明縣（今雲南省[[昆明市|昆明市]]）。[[清朝|清朝]][[翰林|翰林]]、政治人物、軍事人物。

== 早年 ==
何桂清早年是[[王英九|王英九]]的[[書僮|書僮]]，兩人也是好友，因為何桂清的父親是王英九之父[[王燮|王燮]]的[[家丁|家丁]]。王燮擔任太守，請來家教老師教英九，桂清伴讀。英九愚鈍又喜歡玩樂嬉鬧，不喜歡[[八股文|八股文]]，因而讀書失敗，何桂清思緒清明，過目不忘，反而造就了何桂清。後來王英九[[捐納|捐納]]為[[鹽使|鹽使]]時，何桂清出資，報答此恩。

== 歷任 ==
*[[道光|道光]]十五年（1835年）乙未科第二甲第四十九名[[進士|進士]]出身，點[[翰林院|翰林院]][[庶吉士|庶吉士]]。 
*十六年（1836年）：散館授翰林院[[編修|編修]]。 
*十七年（1837年）：翰林院[[編修|編修]]，[[河南|河南]][[鄉試|鄉試]]副主考。 
*十九年（1839年）：翰林院[[編修|編修]]，[[貴州|貴州]]鄉試正主考。 
*二十二年（1842年）：翰林院[[編修|編修]]，[[司經局|司經局]][[洗馬_(官名)|洗馬]]，[[右贊善|右贊善]]，改左贊善，[[日講起居注官|日講起居注官]]，[[南書房|南書房]]行走。
*二十三年（1843年）：升翰林院[[侍講|侍講]]，洗馬。
*二十四年（1844年）：翰林院[[侍講|侍講]]，升[[太僕寺|太僕寺]][[少卿|少卿]]，[[廣東|廣東]][[鄉試|鄉試]]正主考，[[會試|會試]]同考官。
*二十五年（1845年）：太僕寺少卿，升[[光祿寺卿|光祿寺卿]]，改[[太常寺卿|太常寺卿]]
*二十六年（1846年）：太常寺卿，山東[[學政|學政]]。
*二十七年（1847年）：太常寺卿，[[內閣學士|內閣學士]]，山東學政。
*二十八年（1848年）：內閣學士，山東學政。二月五日遷[[兵部右侍郎|兵部右侍郎]]。[[丁憂|丁憂]]去職，服滿後補任原官。
*[[咸豐|咸豐]]元年（1851年）：五月二十日署理[[吏部右侍郎|吏部右侍郎]]，閏八月十三日授兵部右侍郎。十二月十四日<ref>1852年2月3日</ref>改[[戶部右侍郎|戶部右侍郎]]，兼管[[錢法堂|錢法堂]]事務，[[南書房|南書房]][[行走|行走]]，[[順天府|順天]]鄉試覆試閱卷大臣，順天武鄉試較射大臣，兼管[[三庫|三庫]]事務，充[[實錄館|實錄館]]副總裁。
*二年（1852年）：戶部右侍郎兼管錢法堂事務，署理[[經筵講官|經筵講官]]。三月六日充會試副總裁、覆試閱卷大臣，大考翰詹閱卷大臣，考試廕生閱卷大臣，八月六日授[[江蘇|江蘇]]學政。
*三年（1853年）：戶部右侍郎兼管錢法堂事務，仍江蘇學政，十一月二日改[[禮部左侍郎|禮部左侍郎]]。
*四年（1854年）：三月十日改吏部右侍郎，仍江蘇學政，四月十八日改[[倉場侍郎|倉場侍郎]]，九月二十一日升[[浙江巡撫|浙江巡撫]]。
*六年（1856年）：浙江巡撫，十一月六日以病免。
*七年（1857年）：咸豐七年（1857年）兩江總督[[怡良|怡良]]因病出缺，同寅(同年登科)[[彭蕴章|彭蕴章]]保薦何桂清上任。四月十二日何桂清署理[[兩江總督|兩江總督]]，六月六日實授兩江總督，功加[[太子少保|太子少保]]，十二月二十六日<ref>1858年1月29日</ref>授欽差大臣、[[南洋通商大臣|辦理通商各口事務]]。  
*十年（1860年）：兩江總督，加[[太子太保|太子太保]]。四月十九日革職拿辦。

== 政績 ==
*咸豐二年（1852年）太平天國襲江南，何桂清以江蘇學政身份疏陳軍事，抨擊封疆大吏軟弱，無所顧忌侃侃而談，[[咸豐帝|咸豐帝]]感到驚奇。
*四年（1854年）授為浙江巡撫。太平軍攻陷[[江寧|江寧]]，東南震動。[[安徽|安徽]][[徽州|徽州]]、[[寧國|寧國]]是浙江屏障，何桂清駐兵[[黄池|黄池]]，扼守要衝，與浙江[[提督|提督]][[鄧紹良|鄧紹良]]擊退太平軍。
*五年（1855）令[[道員|道員]][[徐榮_(清朝)|徐榮]]攻[[黟縣|黟縣]]、石埭，太平軍主力到達後徽勇敗，徐榮戰死。何桂清上疏主張安徽浙江互為唇齒，主客一心方有勝算。[[咸豐帝|咸豐帝]]上諭令地方官吏不分畛域剿敵。
*安徽巡撫移駐[[廬州|廬州]]，徽、寧二郡暫由何桂清管轄。桂清令徽州[[知府|知府]][[石景芬|石景芬]]、[[副將|副將]][[魁齡|魁齡]]等收復徽州府城及[[休寧|休寧]]，佈防堵敵軍來路。
*[[江西|江西]]太平軍侵浙江，攻陷[[開化|開化]]，進犯[[遂安|遂安]]，檄令鄧紹良等夾攻，擊退之，[[周天受|周天受]]、石景芬等收復黟縣、石埭。何桂清奏請增補道員缺額，准石景芬任[[徽寧池太道|徽寧池太道]]；[[總兵|總兵]][[豫祺|豫祺]]戰績不佳，革職，以[[江長貴|江長貴]]代；前侍郎[[張芾|張芾]]駐皖南[[團練|團練]]，督辦徽、寧防務，兼顧浙江[[衢州|衢州]]、[[嚴州|嚴州]]。
*六年（1856）檄令鄧紹良、[[秦如虎|秦如虎]]、[[都興阿|都興阿]]等合攻[[寧國府|寧國府]]，江長貴擊敗來襲的太平軍，收寧國府城。朝廷嘉獎。
*好友[[杭州府|杭州]]知府[[王英九|王英九]]被[[通判|通判]][[徐徵_(清朝)|徐徵]]訐控。桂清復奏有微詞，被中央責問，以病請辭，慰留。年底兩江總督[[怡良|怡良]]解職，[[文淵閣|文淵閣]][[大學士|大學士]][[彭蘊章|彭蘊章]]舉薦桂清可勝任江督統籌軍餉。
*七年（1857）春以二品頂戴署理兩江總督，隨即實授。何桂清舉薦王英九任江蘇[[布政使|布政使]]，負責軍餉。以[[江寧將軍|江寧將軍]][[和春|和春]]負責軍事、[[江南省|江南]][[提督|提督]][[張國梁|張國梁]]為幫辦、前總督怡良為運饋。何桂清奏陳軍略稱旨，上諭和春商酌。冬，攻克[[鎮江|鎮江]]，以辦理軍餉功加太子少保。
*八年（1858）11月，會同[[東閣|東閣]]大學士[[桂良|桂良]]、[[吏部尚書|吏部尚書]][[花沙納|花沙納]]與英、法、美三國改訂税則、通商章程：
**1.有限承認鴉片貿易。
**2.依《[[中英五口通商章程|中英五口通商章程]]》在上海創建[[關稅總局|關稅總局]]。
**3.進出口貨物，一律按時價抽5％關稅，洋貨遠銷內地一律按時價抽2.5％子口税。
**4.各口税收劃一辦理，聘請英人[[李泰國|李泰國]]幫辦海關稅務等。
*十年（1860）春，收復江寧[[九洑洲|九洑洲]]，清軍合圍江寧，功加太子太保。
*太平天國忠王[[李秀成|李秀成]]分兵由安徽[[廣德縣|廣德]]奇襲杭州，[[杭州將軍|杭州將軍]][[瑞昌|瑞昌]]死守，[[咸豐帝|咸豐帝]]下詔令桂清、和春儘速救援。桂清急檄[[廣西|廣西]]提督[[張玉良|張玉良]]馳援，到達後以600兵擊破太平軍，並收復[[臨安|臨安]]、[[孝豐|孝豐]]、[[安吉|安吉]]。咸豐帝下詔嘉獎。

===末路===
*[[太平军二破江南大营|太平军二破江南大营]]，圍[[金壇|金壇]]、攻陷[[江陰|江陰]]，何桂清傳檄總兵[[馬得昭|馬得昭]]、[[熊天喜|熊天喜]]、[[曾秉忠|曾秉忠]]，[[副將|副將]][[劉成元|劉成元]]水陸分路進勦，太平軍合兵十餘萬出[[建平|建平]]、[[東垻|東垻]]，一由東垻進江寧，一由[[溧陽|溧陽]]攻[[常州|常州]]，馬得昭、[[周天孚|周天孚]]分別救援[[蘇州|蘇州]]、常州。太平軍攻陷[[句容|句容]]，江南大營後路斷絕。
*張玉良回軍守常州，和春飛檄求援，何桂清不准張玉良赴援，和春又請調馬得昭，也未得回應。浙江巡撫[[王英九|王英九]]遞書，告誡何桂清勿逃離常州：「事棘時危，身為大臣，萬目睽睽，視以動止。一舉足則人心瓦解矣。」
*江南大營遇大雪，缺餉，噪亂潰敗。和春、張國梁退守丹陽。何桂清上疏推諉責任：「丹陽以上軍務，和春、張國梁主之。常州軍務，臣與張玉良主之。」 
*部署稍定後進軍溧陽，而太平軍已攻破[[丹陽|丹陽]]，[[張國樑|張國樑]]戰死，[[和春|和春]]逃奔常州，何桂清震驚。[[總理糧台|總理糧台]][[查文經|查文經]]等洞悉其意，建議退守蘇州。桂清便奏言軍事已交託和春，自赴蘇州籌餉。
*出逃時，[[戈什哈|戈什哈]][[槍殺|槍殺]]跪留的常州士紳死19人方脫身，張玉良不久也出走。常州士民自籌登陴守城。數日後常州城破，以人民抵抗而屠城。 
*蘇州守軍聞訊大為憤恨，何桂清到達後，江蘇巡撫[[徐有壬|徐有壬]]拒絕開門，並彈劾何「棄城喪師」。和春兵潰重傷退至[[無錫|無錫]]，自殺殉節。
*何桂清託言借外兵，逃進[[上海租界|上海租界]]。數日後蘇州城陷，徐有壬於[[巷戰|巷戰]]被殺殉節，遺疏再彈劾何桂清多條罪狀。
*何桂清統領十萬清軍竟棄城潛逃，導致常州、常熟、蘇州皆失守，咸豐下詔革職，捕送北京審訊。凑巧遇[[英法聯軍|英法聯軍]]，咸豐帝避難熱河，何案擱置兩年。期間王有齡及江蘇巡撫[[薛煥|薛煥]]等相繼上疏乞恩，不准。 
*同治元年（1862）言官紛紛彈劾，[[恭親王|恭親王]]與租界交涉，正式逮捕下獄，擬[[斬監候|斬監候]]。
*[[秋審處|秋審處]]總辦、[[直隸司|直隸司]][[郎中|郎中]][[余光倬|余光倬]]，將其由封疆大吏失守城池應處之「斬監候」加上罪名「擊殺執香跪留父老十九人，忍心害理，罪當加重」而改擬「[[斬立決|斬立決]]」。
*[[刑部尚書|刑部尚書]][[趙光|趙光]]，因與幫辦江南軍務的[[許乃釗|許乃釗]]有交情，瞭解何桂清失陷蘇常殃民細節，覆奏：「不殺何桂清，何以謝江南百萬戰難生靈？」
*[[慈安太后|慈安太后]]、[[慈禧太后|慈禧太后]]降旨命[[大學士|大學士]][[六部|六部]][[九卿|九卿]][[翰林|翰]][[詹事|詹]][[給事中|科]][[御史|道]]會議。[[大理寺卿|大理寺卿]][[李棠階|李棠階]]認為何桂清清為兩江總督，驕橫顢頇，治軍無方，克扣軍餉導致將士譁變、全軍覆沒，棄城而逃又嫁禍他人，其罪不容誅。會議結果如刑部議，而太后另有旨：「何桂清曾任一品大員，用刑宜慎，如有疑義，不妨各陳所見。」 
*有17人上疏論救，[[體仁閣|體仁閣]][[大學士|大學士]]、[[禮部尚書|禮部尚書]][[祁寯藻|祁寯藻]]，引用[[嘉慶帝|嘉慶帝]]諭旨：「刑部議獄，不得有加重字樣。」認為刑部所擬不合制。另有[[工部尚書|工部尚書]][[萬青藜|萬青藜]]、御史[[高延祜|高延祜]]，而薛煥以重金疏通，故拖延深秋仍未決。 
**禮部[[給事中|給事中]][[卞寶第|卞寶第]]（浙江道御史任內就曾參劾桂清）抗章駁[[祁寯藻|祁寯藻]]：「仁宗上諭，只就承平時期尋常罪名而言。」又言：「道光年間浙江提督[[余步雲|余步雲]]失[[定海|定海]]，咸豐年間[[湖北|湖北]][[巡撫|巡撫]][[青麟|青麟]]失[[武昌|武昌]]，皆以失陷封疆[[伏法|伏法]]，其時祁寯藻當[[軍機大臣|軍機大臣]]，何獨於何桂清護惜若此？」
**大理寺卿李棠階以東南戰事尚未結束，求顧全大局再上密折：「刑賞大政，不可為謬悠之議所撓，今欲平賊，而先庇逃帥，何以作中興將士之氣？」
**刑部審訊時，桂清提出一份薛煥等人具名之公稟，請其退到蘇州，以保餉源重地，作為其本不欲棄地的證明。
**公稟出於當時或事後偽造已無從查究，朝廷命[[兩江總督|兩江總督]][[曾國藩|曾國藩]]查核具奏。曾國藩身在江南，瘡痍遍地，且負破敵責任，考量士氣影響，覆奏：「疆吏以城守為大節，不當以僚屬一言為進止；大臣以心跡為罪狀，不必以公稟有無為權衡。」判處[[斬立決|斬立決]]定讞，12月21日在京師[[菜市口法场|菜市口法场]][[棄市|棄市]]。
*《[[清史稿|清史稿]]》評論：「以才敏负一时之望，膺江表重寄。桂清无料敌之明，又失效死之节。身名俱陨，罪实难辞。」
