'''何鲁丽'''（{{bd|1934年|6月|catIdx=H何}}），女，[[中国人|中]][[法兰西人|法]][[混血儿|混血儿]]，[[山东|山东]][[菏泽|菏泽]]人<ref>{{cite web |url = http://news.xinhuanet.com/ziliao/2002-01/23/content_250839.htm |title=何鲁丽简历 |accessdate = 2008-06-28 |website = 新华网 |language = zh-hans }}</ref>，前任[[中国国民党革命委员会|中国国民党革命委员会]]（民革）中央主席。歷任[[中國人民政治協商會議|中國人民政治協商會議]]第八屆全國委員會副主席（1996年增選），第九、十屆[[全國人民代表大會常務委員會副委員長|全國人民代表大會常務委員會副委員長]]。

== 生平 ==
何鲁丽1957年毕业于[[北京医学院|北京医学院]]医疗系，曾任北京市儿童医院住院医师，北京市西城区儿童医院住院医师，北京市第二医院[[主治医师|主治医师]]、[[儿科|儿科]]主任、副主任医师。 1984年被选为[[北京市|北京市]][[西城区|西城区]][[人民政府|人民政府]]副区长，后任[[北京市人民政府|北京市人民政府]]副市长、民革中央副主席、[[全国妇联|全国妇联]]副主席、[[民革|民革]]中央主席、[[全国政协副主席|全国政协副主席]]、[[中央社会主义学院|中央社会主义学院]]院长等职位。1998年至2008年任[[全国人大常委会副委员长|全国人大常委会副委员长]]。

== 家庭 ==
何鲁丽的父亲[[何思源|何思源]]是[[中华民国|中华民国]][[国民政府|国民政府]]的[[北平|北平]]市市长，曾在[[巴黎大学|巴黎大学]]留学，与出生于[[波尔多|波尔多]]市的法国女同学宜文妮·詹姆斯结婚，婚后宜文妮改名为何宜文，是何鲁丽的母亲。由于何思源主张[[北平和平解放|北平和平解放]]，[[国民党当局|国民党当局]][[特务|特务]]在其家中埋藏炸弹，其小女儿在1949年1月17日被炸死。
