'''何煖軒'''（{{bd|1953年|1月5日|catIdx=H何}}），[[臺灣|臺灣]][[新竹|新竹]]人，[[中央警察大學|警察大學]]畢業、[[美國|美國]][[明尼蘇達曼徹州立大學|明尼蘇達曼徹州立大學]]碩士、[[中華大學|中華大學]]博士。曾任臺北市議長[[張建邦|張建邦]]總務主任，隨後任[[交通部|交通部]]參事及公關主任，後拔擢升[[交通部高速公路局|高公局]]長；又歷任[[交通部高速鐵路工程局|高鐵局]]長、[[臺灣鐵路管理局|台鐵局]]長、[[中華郵政|中華郵政]]董事長等職，並獲重用升[[中華民國交通部|交通部]]次長。2016年6月，受[[桃園市市長|桃園市市長]][[鄭文燦|鄭文燦]]推薦<ref>{{Cite web|title=千萬元大肥缺！公股人事爭奪戰|url=https://newtalk.tw/news/view/2016-08-10/76095|work=新頭殼 Newtalk|date=2016-08-10|accessdate=2019-02-08}}</ref>下，何煖軒由[[桃園捷運公司|桃園捷運公司]][[董事長|董事長]]轉任公股投資的[[華航|華航]]董事長直至2019年4月。

何煖軒也任[[台北市航運商業同業公會|台北市航運商業同業公會]](TAA)理事長，另在[[警察大學|警察大學]]、[[國立交通大學|國立交通大學]]、[[中國文化大學|中國文化大學]]、[[開南大學|開南大學]]、[[中華大學|中華大學]]兼課任教。

== 生平 ==
1992年，受當時和[[新國民黨連線|新國民黨連線]]有諸多恩怨[[中華民國交通部|交通部]]部長[[簡又新|簡又新]]找來接替公關主任的職務，以和緩部會與[[立法院|立法院]]的關係。

1999年，時任[[高速公路局|高公局]]局長何煖軒指出，為紓解高速公路因收費造成的交通擁塞，縮短用路人旅行時間，高速公路局除了在民國九十年一月在[[高速公路|高速公路]]全線使用電子收費，並計畫「走多少、收多少」按行駛里程多少，以[[匝道|匝道]]收費方式替代[[收費站|收費站]]收費，以合乎「使用者付費」的公平理念。

2007年，台鐵通訊曾評人稱鐵腕局長的何代兼局長煖軒，是令台鐵員工印象深刻的局長，在職短短的八個月，卻徹底顛覆了台鐵的傳統倫理觀念與作法，影響深遠。

2014年12月，何煖軒進入[[桃園市市長|桃園市市長]][[鄭文燦|鄭文燦]]小內閣，任[[桃園捷運公司|桃園捷運公司]][[董事長|董事長]]。

2016年6月，因[[2016年中華航空空服員罷工事件|空服員罷工事件]]，交通部長[[賀陳旦|賀陳旦]]原本希望前華航董事長[[江耀宗|江耀宗]]回鍋；但考慮諸多因素，最後在[[行政院院長|行政院院長]][[林全|林全]]拍板下定案，何煖軒接任華航董事長<ref>{{Cite web|url = http://www.appledaily.com.tw/realtimenews/article/new/20160624/893018/|title = 華航董座二人在爭　何煖軒出線有內幕|accessdate = |author = |date = 2016-06-24|publisher = 蘋果日報}}</ref>。6月24日，何煖軒讓[[2016年華航空服員罷工事件|2016年華航空服員罷工事件]]落幕；在落幕前，何煖軒已先換掉華航資深客運副總[[董孝行|董孝行]]，改由[[臺灣航勤|臺灣航勤]]董事長[[張揚|張揚]]接任。<ref>{{Cite web|url = http://www.chinatimes.com/realtimenews/20160624005422-260407|title = 何煖軒換掉客運副總董孝行 由張揚接任|accessdate = 2016-06-24|author =張佩芬 |date =2016-06-24 |publisher =工商時報 }}</ref>。

何煖軒進入華航進行人事改革還爆發監委關說疑雲案外案，在6月24日因華航空服員罷空事件，臨危受命，接任董事長，與空服員工會經過談判，取得共識，讓罷工事件落幕後，即陸續進行公司內部人事調整，上任約兩個月，發布25位異動主管名單，其中，地勤服務處副總俞智調任青島辦事處總經理，中國時報曾報導<ref>{{Cite news|url=https://www.chinatimes.com/newspapers/20160803000370-260102|title=監委為夫 關切華航人事|last=周慧如、崔慈悌|newspaper=中時電子報|date=2016-08-03|accessdate=2018-10-31}}</ref>，俞智擔任監委的妻子林雅鋒在獲知俞智有此人事調任前，曾透過監院高層向交通部長賀陳旦表示關切，希望擋下此案；但賀陳旦表示，尊重華航人事決策權，未加干涉。華航在1日發布人令後，俞智以心情沉重為由，自請退休。中國時報引述不具名消息，據了解，林雅鋒透過監院高層向賀陳部長表達關切之意，希望賀陳旦要求何煖軒勿做此人事調動。但賀陳認為，應該尊重華航董座何煖軒的人事決定。中國時報報導，林雅鋒曾在2014年8月任監委後，對何煖軒及前交長林陵三提出彈劾<ref>{{Cite news|url=https://www.chinatimes.com/newspapers/20160805000394-260102|title=華航人事案外案 監委邊關說邊查辦|last=王正寧、呂雪彗|newspaper=中時電子報|date=2016-08-05|accessdate=2018-11-01}}</ref>，監院並在次年2月以「造成國家財政嚴重損失」為由，通過彈劾案。因此當林雅鋒為夫關說一事近日傳開後，令人驚訝其御史形象之轉變。林雅鋒當時回應，她沒有賀陳部長的電話，也不認識賀陳，也不知所謂「監院高層」是指誰。但中國時報數日後又再度追蹤報導，法官出身的林雅鋒，過去針對長生集團與高鐵局的仲裁糾紛，就曾對時任高鐵局長的何煖軒提出彈劾，彈劾雖經監察院通過，但送請司法院公務員懲戒委員會審議時，公懲會在2015年8月以彈劾事由不足採，做成「不受懲戒」的決議。其間，監院曾經三度送公懲會皆無功而返，讓何煖軒頗為感概。事後行文監院及監院紀律委員會，要求監院還他清白，監院在受理後尚未回覆說明處理結果。且監委沒辦倒「高鐵局長」何煖軒，卻已著手調查「桃捷公司董事長」何煖軒。據指出，監委針對自2006年起承包機場捷運機電系統的日商丸紅，一再延宕工程，迄今尚未通車，要對2014年出任桃園捷運公司董事長的何煖軒進行調查。至於林雅鋒本人再三否認關說，亦即究竟有無在監委考察時請時任交通部次長吳盟分帶話給交通部長賀陳旦，關說其夫俞智在華航的人事？賀陳旦當時在行政院會後答覆媒體問題時表示，他沒有直接從吳盟分那裡獲知監委的事，但同仁有轉告他，而他的回答就是「我知道了，但人事處理尊重華航職權」，賀陳旦的說法等於證實監院介入華航人事關說。事後中國時報也特別以社論批評監察院「關說不成查辦伺候」，社論指林雅鋒曾因機場捷運BOT案，對時任高鐵局長的何煖軒提出彈劾，監察院雖以造成國家鉅大損失及官商勾結等弊端為由通過彈劾案，但送司法院公務員懲戒委員會之後，被認為事由不足採信而「不受懲戒」。社論質疑主張彈劾有理的監委，鍥而不捨地再將何連送兩次公懲會皆遭駁回，更批評監委理應自行迴避退出調查，特別是在爆出「為夫關說」之後，倘若法官出身的監委，連「利益迴避」這個公職人員的基本底線都守不住，不要說沒有臉去糾正別人，恐怕日後重回法庭的資格都讓人有所存疑。同樣地，身為監察院院長，張博雅豈能對這些關說指控置若罔聞，她自己也是當事人，究竟有無在場幫腔，甚至出面請託，監察院大家長也有說分明的必要。

2016年9月，何煖軒表示，上級從未要求[[中華航空更名事件|公司更名]]<ref>{{cite news|title=華航正名台灣航空？ 董事長何煖軒這樣說...|publisher=自由時報|url=http://news.ltn.com.tw/news/politics/breakingnews/1842311|date=2016-09-30|author=|accessdate=2018-03-28}}</ref>。2017年間，何煖軒積極在客運、貨運、維修市場佈局，開發電商零售<ref>焦家卉.[https://tw.news.yahoo.com/%E5%9C%A8%E6%88%91%E4%BB%BB%E5%85%A7%E8%8F%AF%E8%88%AA%E4%B8%8D%E6%9C%83%E5%86%8D%E7%BD%B7%E5%B7%A5-%E4%BD%95%E7%85%96%E8%BB%92-%E8%B7%A8%E5%85%A5%E9%9B%BB%E5%95%86%E6%8F%90%E9%AB%98%E7%8D%B2%E5%88%A9-103000991.html 「在我任內華航不會再罷工」 何煖軒：跨入電商提高獲利].風傳媒.2017-08-04</ref>，帶領華航營收創新高，本業獲利近增四倍<ref>{{Cite news|url=https://udn.com/news/story/7252/3046771|title=客貨運暢旺 華航去年本業獲利增近四倍|last=韓化宇|newspaper=聯合新聞網|date=2018-03-22 |accessdate=2018-03-27}}</ref>，華航集團旗下[[台灣虎航|台灣虎航]]首度轉虧為盈<ref>{{Cite news|last=韋樞|url=http://www.cna.com.tw/news/afe/201803140202-1.aspx|title=華航集團營收創新高 虎航首度獲利 |newspaper=中央社|date=2018-03-14|accessdate=2018-03-27}}</ref>。

== 事蹟 ==
過去何煖軒曾全面推動騎[[機車_(公路)|機車]]戴安全帽以及強制規定繫[[座位安全帶|安全帶]]的措施，讓車禍的死亡率出現明顯下降趨勢，而這些改革方案也是他最有成就感的地方。

== 爭議 ==

== 興趣 ==
2006年，何煖軒擔任[[中華民國交通部|交通部]]次長時接受媒體專訪表示，自己是個工作狂，向來給人嚴肅的感覺，殊不知他私底下是個風趣的人，身材壯碩的他，多年前在擔任高公局長時因食道上長了一顆腫瘤，動了大手術，住院住20幾天，但一出院就馬上回工作崗位。他的養生之道「吃得簡單，但是運動是一定要的啦！」

2018年10月，何煖軒接受[[蘋果日報|蘋果日報]]專訪，平日靠著運動紓解壓力，每天6點晨跑數十年如一日，練就強健的體魄<ref>{{Cite news|last=陳慜蔚|url=https://tw.appledaily.com/new/realtime/20181011/1439549/|title=【蘋果頭家1】華航老董何煖軒愛運動　徑賽籃球都曾創紀錄|newspaper=蘋果日報|date=2018-10-11 |accessdate=2018-10-15}}</ref>。談及交通運輸產業勞資爭議，最重要的就是解決問題，帶給社會及大眾正向發展，也希望政府應針對[[罷工|罷工]]修訂相關法規，保障消費者的權益，呼籲同為交通運輸業的夥伴理性溝通<ref>{{Cite news|last=陳慜蔚|url=https://tw.appledaily.com/new/realtime/20181011/1439551/|title=【蘋果頭家2】何煖軒初掌華航勞資對抗　作夢也會夢到「罷工」|newspaper=蘋果日報|date=2018-10-11 |accessdate=2018-10-15}}</ref>，對於媒體不實的報導，何煖軒感嘆地表示，交通產業的假新聞受害者始主為己，抹黑、造謠樣樣來，累積了這些經驗，也讓他擁有的更多，現已豁達來看待<ref>{{Cite news|last=陳慜蔚|url=https://tw.appledaily.com/new/realtime/20181011/1439553/|title=【蘋果頭家3】何煖軒：我是交通界「假新聞第1個受害者」 |newspaper=蘋果日報|date=2018-10-11 |accessdate=2018-10-15}}</ref>，亦持續帶領[[華航集團|華航集團]]積極發展航太維修及電商，成果豐碩可望開花結果<ref>{{Cite news|last=陳慜蔚|url=https://tw.appledaily.com/new/realtime/20181011/1439555/|title=【蘋果頭家4】華航明年航太維修開花　電商結果|newspaper=蘋果日報|date=2018-10-11 |accessdate=2018-10-15}}</ref>。
