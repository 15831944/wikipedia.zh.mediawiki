'''何廣沛'''（{{lang-en|Matthew Ho Kwong Pui}}，{{bd|1988年|8月3日}}），香港男演員及主持，現為[[無綫電視藝員列表|無綫電視經理人合約]]藝員<ref>[http://artiste.tvb.com/main/artistedetail/matthewho 何廣沛 Matthew Ho - TVB藝人資料 - tvb.com]</ref>。

== 簡歷 ==
何廣沛父親曾任職[[懲教署|懲教署]]，而母親則是[[港安醫院|港安醫院]]護士<ref>{{cite web |url=https://archive.fo/E8kV0/90efa98e3ef15f1fbf304db89adb9990ab4993ed.png |title=Matthew Ho's Family |work=[[Facebook|Facebook]] - Matthew Ho（matthew.hokwongpui） |accessdate=2018-11-05}}</ref>。他曾就讀[[聖士提反書院|聖士提反書院]]，唸至中三時與父母及胞姊一同移民[[三藩市|三藩市]]，並入讀[[洛思阿圖斯山丘|洛思阿圖斯山丘]]山麓學校（Foothill College）；2011年修畢[[加州州立大學|加州州立大學]]會計系學士課程後曾於網台 [[9DTV|9DTV]] 擔任節目主持<ref>[http://tvb4life.pixnet.net/blog/post/178116405-%E9%99%B3%E5%98%89%E5%AF%B6-24%E6%AD%B2-%E5%88%9D%E6%88%80-%E5%8E%9A%E5%BA%95%E9%97%8A%E5%B0%91%E3%80%80%F0%A1%83%81%E7%89%88-%E6%9D%8E%E8%8B%A5%E5%BD%A4-%E5%AE%88 陳嘉寶24歲初戀厚底闊少　𡃁版李若彤守處：唔會婚前性行為]，[[FACE周刊|FACE周刊]] 第363期，2014年5月7日</ref>。2012年，他認為不論會計師或演員都要吃苦，兩者相較下還是對演藝工作深感興趣<ref>[https://www.recruit.com.hk/Resources/ArticleDetail.aspx?articleId=36574&sectionId=STA1 何廣沛　放下包袱]，[[Recruit.com.hk|Recruit.com.hk]]，2018年3月27日</ref>，故決定加入[[無綫電視藝員訓練班|無綫電視第26期藝員訓練班]]<ref>[http://event.tvb.com/26thtalent/candidate.php?kuid=6 第26期藝員訓練班學員介紹 - 何廣沛]</ref>，2014年因在該台劇集《[[女人俱樂部|女人俱樂部]]》裡飾演「年青姚子倫」而為人熟悉<ref>[http://www.chinadaily.com.cn/dfpd/dfwhyl/2014-05-01/content_11651643.html 《女人俱樂部》捧紅七少女 何廣沛成 TVB“新帥哥”] {{Wayback|url=http://www.chinadaily.com.cn/dfpd/dfwhyl/2014-05-01/content_11651643.html |date=20140502004909 }}，[[中國日報|中國日報]]網，2014年5月1日</ref><ref>《[[TVB周刊|TVB周刊]]》第929期“來自新星的你”專訪</ref>，並受到重視。

2016年，何廣沛於《[[星和無綫電視大獎2016|星和無綫電視大獎2016]]》頒獎典禮中奪得「TVB最佳新人」殊榮，同年也首次獲提名《[[TVB_馬來西亞星光薈萃頒獎典禮2016|TVB 馬來西亞星光薈萃頒獎典禮2016]]》「最喜愛 TVB 飛躍進步男藝員」、《[[萬千星輝頒獎典禮2016|萬千星輝頒獎典禮2016]]》「最佳男配角」及「飛躍進步男藝員」。2017年，他憑《[[超時空男臣|超時空男臣]]》內「[[李進忠|李進忠]] ／ 李小冬」一角獲得《[[星和無綫電視大獎2017|星和無綫電視大獎2017]]》「TVB 飛躍進步藝人」。

== 軼聞 ==
* 他曾有機會成為香港男子歌唱組合「[[BOP_(組合)|天堂鳥]]」其中一員，卻遭無綫電視打壓而放棄，遂專注當演員<ref>[http://nextplus.nextmedia.com/news/ent/20140527/43908 無綫打壓 麥當當王子冇得做天堂鳥]，[[壹週刊|壹週 Plus]]，2014年5月27日</ref><ref>[http://www.hk01.com/%E5%A8%9B%E6%A8%82/46690/-%E5%BE%8B%E6%94%BF%E5%BC%B7%E4%BA%BA-%E4%BD%95%E5%BB%A3%E6%B2%9B%E6%8D%B1%E9%81%8E%E8%97%9D%E8%A8%93%E7%8F%AD-%E6%94%B6%E6%9C%80%E4%BD%8E%E5%B7%A5%E8%B3%87-%E6%9C%89%E5%98%A2%E5%AD%B8%E6%9C%89%E9%8C%A2%E6%94%B6%E4%BB%B2%E6%83%B3%E9%BB%9E- 《律政強人》何廣沛捱過藝訓班　收最低工資：有嘢學有錢收仲想點？]，[[香港01|香港01]]，2016年10月4日</ref>。
* 他曾被指與廣州男子歌唱組合「[[拾音社|拾音社]]」成員[[何柏誠|何柏誠]]樣貌相似，而何柏誠亦多次在公開場合內認同此說法，更有影迷笑說二人同樣姓何，是失散多年的兄弟。
* 他因拍攝《[[跳躍生命線|跳躍生命線]]》而與[[羅天宇|羅天宇]]、[[郭子豪|郭子豪]]及[[張彥博|張彥博]]結成好友，並組成「跳躍 F4」<ref>[https://hk.ulifestyle.com.hk/topic/detail/206127/%E8%B7%B3%E8%BA%8D%E7%94%9F%E5%91%BD%E7%B7%9A-%E4%BD%95%E5%BB%A3%E6%B2%9B-%E9%83%AD%E5%AD%90%E8%B1%AA%E7%B5%84-%E8%B7%B3%E8%BA%8Df4-%E9%87%8D%E7%8F%BE%E7%B6%93%E5%85%B8mv-%E6%B5%81%E6%98%9F%E9%9B%A8-%E7%AC%91%E7%88%86%E5%85%A8%E5%A0%B4 【跳躍生命線】何廣沛、郭子豪組「跳躍 F4」　重現經典 MV《流星雨》笑爆全場]，[[港生活|港生活]]，2018年10月31日</ref><ref>[https://www.orientalsunday.hk/316864/%E6%9C%80%E6%96%B0%E5%A8%9B%E8%81%9E/%E8%B7%B3%E8%BA%8Df4-%E8%B7%B3%E8%BA%8D%E7%94%9F%E5%91%BD%E7%B7%9A-%E6%B5%81%E6%98%9F%E9%9B%A8/ 《跳躍生命線》救護四子組「跳躍 F4」重現經典《流星雨》MV 畫面！]，[[東方新地|東方新地]]，2018年10月31日</ref>。
