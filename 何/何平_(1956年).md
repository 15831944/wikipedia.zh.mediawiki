'''何平'''（{{bd|1956年|3月||}}），[[中国民主同盟|民盟]]盟员，中国历史学家，东南亚民族史专家。何平对“泰傣古国”有全面且深入的研究，是中国质疑“泰傣古国”最具代表性的学者<ref>{{cite journal |author=杜树海 |date=2007 |title=追寻失落的记忆——泰傣民族起源研究述论 |journal=《广西民族研究》|issue=3 |pages=159 |doi=10.3969/j.issn.1004-454X.2007.03.023 }}</ref>。

==生平==

1985年大学毕业后在[[云南省社会科学院|云南省社会科学院]]东南亚研究所从事研究工作，1995年起执教[[云南大学|云南大学]]，现为云南大学历史系教授、[[博士|博士]]生导师。2013年12月获云南省委宣传部“云南省优秀社科专家”称号。著有《东南亚民族史》、《中南半岛的民族渊源与流变》、《东南亚的封建——奴隶制结构与古代东方社会》、《从云南到阿萨姆: 傣-泰民族历史再考与重构》等专著。<ref>{{cite web |url=http://www.gx211.com/news/20141218/n4599232310.html |title=何平教授——云南大学 |date=2014-12-18 |website=www.gx211.com |publisher=中国高校之窗 |access-date=2018-12-19 |quote=}}</ref><ref>{{cite web |url=http://www.ha.ynu.edu.cn/info/1015/1145.htm |title=何平教授 |date=2017-09-21 |website=www.ha.ynu.edu.cn |publisher=[[云南大学|云南大学]]历史与档案学院 |access-date=2018-12-19 |quote=}}</ref>

因何平撰有《傣族历史上并没有一个“果占璧王国”》等论文驳斥德宏州傣学学会的研究，被网络中傣族文化追从者贬斥为“整天窥视傣族、以炒作傣族话题混政府经费的异族政治学者”<ref>{{cite web |url=http://www.sohu.com/a/212543947_703096 |title=“勐蓬”、“勐卯”等傣族历史古国的雅号皆称“果占璧” |author= |date=2017-12-25 |website=www.sohu.com |publisher=傣族文化 |access-date=2018-12-24 |quote=}}</ref>。
