'''何啟明'''（{{lang-en|'''Ho Kai-ming'''}}，1985年1月6日－），祖籍廣東[[東莞|東莞]]，是[[香港工會聯合會|香港工會聯合會]]成員，立法會議員（勞工界），觀塘區議會議員（2012年－）。

2011年，何啟明首次參與[[觀塘區議會|觀塘區議會]][[栢雅|栢雅]]選區選舉，挑戰擔任17年區議員的[[鄧志豪|鄧志豪]]。他最終以2019票(得票率:55.5%)對1617票(得票率:44.5%)當選。<ref>[http://www.elections.gov.hk/dc2011/chi/results_kwun_tong.html 2011年區議會選舉結果 - 觀塘區]</ref>[[2015年香港區議會選舉|2015年香港區議會選舉]]成功連任。<ref >{{Cite web |title = 2015年區議會選舉 - 選舉結果(觀塘) |work = 香港特別行政區政府|date = |accessdate = 2015-11-23 |url = http://www.elections.gov.hk/dc2015/chi/results_kwun_tong.html?1448245478339 }}</ref>

[[2016年香港立法會選舉|2016年香港立法會選舉]]，何啟明與[[建制派|建制派]]黨友[[陸頌雄|陸頌雄]]於勞工界功能界別自動當選。
