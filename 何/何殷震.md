'''何殷震'''（{{bd|约1884年||约1920年}}），本名'''何班'''，又名'''何震'''，别名'''何振'''，笔名'''志剑'''、'''小器'''。“何殷震”是其笔名，“殷”字为其母姓氏。[[中国|中国]]近代[[女性主義|女性主義者]]、[[无政府主义|无政府主义者]]<ref>{{cite web |title=Rebecca E. Karl谈“中国女权主义的诞生：何殷震的劳动本体论与二十世纪无政府女权主义” |url=http://www.skc.ecnu.edu.cn/s/117/t/325/8b/36/info166710.htm |website=华东师范大学社科处 |accessdate=2019-01-27}}</ref>。[[武进|武进]]县学教谕何承霖次女，生于[[江蘇省|江蘇省]][[揚州府|揚州府]][[儀徵縣|儀徵縣]]。其夫为著名学者[[劉師培|劉師培]]。

== 生平 ==
1907年赴日本，7月，加入[[亚洲和亲会|亚洲和亲会]]。旋与丈夫[[刘师培|刘师培]]及张继等成立无政府主义团体[[社会主义讲习会|社会主义讲习会]]、[[女子复权会|女子复权会]]；并和[[陆恢权|陆恢权]]，[[周大鸿|周大鸿]]等人发起创办《天义报》（半月刊），任主编<ref>{{Cite book|title=实斋笔记|last=章开沅著|first=|publisher=西安：陕西人民出版社|year=2008.03|isbn=7-224-08230-9|location=|pages=245}}</ref>；又办《[[衡报|衡报]]》；宣传[[无政府主义|无政府主义]]。中华民国成立后，曾在[[太原女子师范|太原女子师范]]任教，又担任过[[阎锡山|阎锡山]]家庭教师。刘师培病故后不久，她也因病去世<ref>{{Cite book|title=中华民国史大辞典|last=张宪文等主编|first=|publisher=南京：江苏古籍出版社|year=2001.08|isbn=7-80643-456-9|location=|pages=977}}</ref>。

1908年，《天义报》刊登了[[共产党宣言|共产党宣言]]最早的中译本（第一章）。
