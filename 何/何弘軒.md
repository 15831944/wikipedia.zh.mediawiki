'''何弘軒'''（{{lang-en|'''Alan Ho'''}}，{{bd|1985年|10月31日|}}），本名'''何偉圖'''，[[香港|香港]][[創作歌手|創作歌手]]；他於1995年與[[樂意唱片|樂意唱片]]簽約成為旗下童星歌手，並於2012年正式出道。

==簡歷==

[[何偉圖|何偉圖]]在5歲時曾與[[黎明|黎明]]在《[[歡樂今宵|歡樂今宵]]》同台演出<ref>{{cite web |language=zh-hant | title=對不起 我愛你 (黎明 & 何偉圖) |publisher=[[電視廣播有限公司|電視廣播有限公司]] |accessdate=2012-05-30 |url=http://www.youtube.com/watch?v=PS-TxXSTIi0 }}</ref><ref>{{cite web |language=zh-hant | title=新秀 何偉圖 最近 (電視 VERSION) |publisher=[[新時代電視|新時代電視]] |date=2007 |accessdate=2012-05-30 |url=http://www.youtube.com/watch?v=huI9eB5vQMM }}</ref>。1995年，與[[樂意唱片|樂意唱片]]簽約成為旗下藝人<ref name="bio" />。

何偉圖在中學畢業後前往[[加拿大|加拿大]][[多倫多大學|多倫多大學]]攻讀[[統計學|統計學]]。何偉圖在2007年參加由[[新時代電視|新時代電視]]主辦的選秀節目《[[全球華人新秀歌唱大賽|全球華人新秀歌唱大賽]]2007（[[多倫多|多倫多]]選拔賽）》，最後在該選秀節目中奪得季軍；同年開始在[[多倫多|多倫多]]為[[新時代電視|新時代電視]]娛樂新聞節目《[[熒幕八爪娛|熒幕八爪娛]]》擔任主持工作<ref name="bio" />。

何偉圖於2009年返回香港，並在[[YouTube|YouTube]]發布其自己創作的歌曲《後備情人》，除獲受[[網民|網民]]好評外<ref name="bio">{{cite web |language=zh-hant | title=何偉圖 Alan Ho  |publisher=imusic.hk |accessdate=2012-05-30 |url=http://imusic.hk/25E425BD259525E52581258925E5259C2596_Alan_Ho/ }}</ref>，也獲香港樂壇天后[[陳慧琳|陳慧琳]]讚許<ref>{{cite web |language=zh-hant |title=一張陳慧琳的唱片封面 |accessdate=2012-05-30 |url=https://fbcdn-sphotos-a.akamaihd.net/hphotos-ak-ash2/25455_422257770800_180474_n.jpg }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。何偉圖在2012年正式出道，並於11月推出首張個人專輯《[[Getting_There|Getting There]]》；碟中收錄已在互聯網發布並由他創作的《〈未命名〉》（舊名《後備情人》）、《幸福太遠》歌曲<ref name="gt" />

2014年6月，何偉圖轉往一馬來西亞唱片公司[[無極巨星|無極巨星]]<ref>{{cite web |language=zh-hant |title=何弘軒飛大馬錄製新歌 |publisher=城報網 |accessdate=2014-07-29 |url=http://www.singpao.com/yl/mx/201407/t20140709_517404.html |deadurl=yes |archiveurl=https://web.archive.org/web/20141112162027/http://www.singpao.com/yl/mx/201407/t20140709_517404.html |archivedate=2014-11-12 }}</ref>，並將藝名改為'''何弘軒'''<ref>{{cite web |language=zh-hant | title=何偉圖改名再戰樂壇 |publisher=大公網 |accessdate=2014-07-29 |url=http://news.takungpao.com.hk/paper/q/2014/0625/2559372.html }}</ref>，同時發放派台歌《不做普通朋友》。
