'''何木火'''（{{bd|1958年|}}）字'''世懷'''，號'''水墨行者'''，[[臺灣|臺灣]][[臺中市|臺中市]][[大里區|大里區]]人，是國內外知名[[國畫|國畫]]大師。其三叔何金旺在日治時代是其村內有名的肖像畫家，何木火在他教導下展現繪畫天分。他曾計畫於國中畢業後攻讀藝術科系，然其父親於畢業前夕在工作時發生意外，手臂受傷，使他必須開始工作養家。他成家後生活困頓，此時卻又受了職業傷害而無法工作。1980年，何木火決定投入專業藝術創作，何木火運用父母的資助於大里成立「日新藝廊」，賣畫並開課招生。1983年起，何木火的作品各美展中嶄露頭角，並在許多中日韓的美展中獲獎。<ref>{{cite web |author = 楊睿林 |website = 科技人文雜誌 |publisher = 科技人文雜誌 |url = http://www.taichung-life.com.tw/?REQUEST_ID=cGFnZT1jb2x1bW5fY29sdHVyZQ==&CID=18605 |title = 小人物成就大藝術的何木火 |accessdate = 2017-11-29 |archiveurl = https://web.archive.org/web/20171129070349/http://www.taichung-life.com.tw/?REQUEST_ID=cGFnZT1jb2x1bW5fY29sdHVyZQ==&CID=18605 |archivedate = 2017-11-29 |deadurl = no |language = zh-tw}}</ref><ref>{{cite web |author = 林宜靜 |website = [[中國時報|中國電子報]] |publisher = 中國時報 |url = http://www.chinatimes.com/realtimenews/20161122001185-260405 |title = 【揮灑台灣之美—水墨行者何木火彩墨展】史博館展出 |date = 2016-11-22 |accessdate = 2017-11-29 |archiveurl = https://archive.today/20171129071340/http://www.chinatimes.com/realtimenews/20161122001185-260405 |archivedate = 2017-11-29 |deadurl = no |language = zh-tw }}</ref>
