'''何娟'''（{{bd|1970年|10月8日|}}），出生于[[四川省|四川省]][[彭县|彭县]]（今[[彭州市|彭州市]]），居住在四川省成都市，旅行社导游、领队，是维权人士[[幸清贤|幸清贤]]的妻子。

* 2015年10月6日，[[幸清贤|幸清贤]]、[[唐志顺|唐志顺]]、包卓轩（[[王宇_(律師)|王宇]]、[[包龙军|包龙军]]之子）在缅甸被中国政府跨境抓捕。<ref>{{Cite web|url = http://www.boxun.com/news/gb/china/2015/10/201510091411.shtml#.Vr_ctZMrLsF|title = 紧急扩散：中国公民包卓轩、唐志顺、幸清贤在缅甸失联|accessdate = |author = 博讯|date = 10/09/2015|publisher = 博讯网}}</ref> <ref name=":0">{{Cite web|url = http://www.rfa.org/mandarin/yataibaodao/renquanfazhi/ql2-10102015114247.html|title = 维权人士幸清贤确认被捕 公民联署促当局确保三人安全|accessdate = |author = 乔龙|date = 2015-10-10|publisher = 自由亚洲}}</ref>
* 2015年10月8日，成都市公安局北巷子派出所受内蒙古乌兰浩特公安局委托到[[幸清贤|幸清贤]]家抄家。<ref name=":0" />
* 2015年10月9日，何娟离开中国，前往老挝，转至泰国曼谷<ref name=":1">{{Cite web|url = http://www.rfa.org/mandarin/yataibaodao/renquanfazhi/ql2-10122015100746.html|title = 助王宇之子出境唐志顺家被抄幸清贤妻辗转抵达旧金山|accessdate = |author = 乔龙|date = 10/12/2015|publisher = 自由亚洲}}</ref>，于10月12日，何娟辗转到达美国旧金山，并在机场接受了自由亚洲的采访，呼吁中国政府释放[[幸清贤|幸清贤]]等。<ref name=":1" /><ref>{{Cite web|url = http://www.rfa.org/mandarin/yataibaodao/renquanfazhi/ck-10272015113104.html|title = 义救包蒙蒙幸清贤被捕 其妻何娟问中共要人|accessdate = |author = CK|date = 2015-10-27|publisher = 自由亚洲}}</ref>
