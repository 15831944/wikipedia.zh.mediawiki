'''何康'''（{{bd|1923年||||H何}}），[[福建|福建]][[福州|福州]]人。於1983年至1990年擔任[[中華人民共和國農牧漁業部|中華人民共和國農牧漁業部]]、[[中华人民共和国农业部|中华人民共和国农业部]]部長<ref name='s'>{{cite web |title=原农业部部长何康来柳重访母校 |url=http://news.sina.com.cn/s/2003-12-17/14421365362s.shtml |website=新浪新闻 |accessdate=2019-01-17}}</ref>，在擔任部長期間，進行農業改革，因此令他於1993年獲得[[世界糧食獎|世界糧食獎]]。

== 生平 ==
1936年毕业于南京[[金陵中学|金陵中学]]，同年入读[[马尾海军学校|马尾海军学校]]，1937年退学。1938年进入时在重庆办学的[[天津市南开中学|天津南开中学]]<ref>{{cite web |title=何康：从橡胶园到农业部部长 |url=https://www.xzbu.com/7/view-3085358.htm |website=www.xzbu.com}}</ref>。1939年，受家庭革命影响的何康也要去延安学习，偷跑到湖北，被八路军办事处接待。何康父亲何遂对周恩来说：“我有两个儿子去延安了，老三再走，我也呆不下去了。”何康没去成延安，周恩来把何康找到[[重庆八路军办事处|重庆八路军办事处]]谈话，由董必武介绍何康入党，并任重庆南开中学地下党支部书记，时年仅16岁。1941年[[皖南事变|皖南事变]]后，何康奉命退学离校。时值广西的[[白崇禧|白崇禧]]、[[李济深|李济深]]请何康父亲[[何遂|何遂]]去做[[桂林行营|桂林行营]]总顾问，何康遂随父来到[[桂林|桂林]]，进入[[广西大学|广西大学]]就读经济系。<ref>{{cite web |title=农业部原部长 - 何康 |url=http://www.jlhs.net/DisplayInfo.jsp?pageID=7662&menuID=303 |website=金陵中学 |accessdate=2019-01-17}}</ref>董必武指示何康：要学门技术，将来建国好有用。何康自以为数理基础不是太好，但却非常喜欢生物，而且看到中国很多农村落后，农民很苦，觉得要为中国的国计民生做贡献，农业是更直接的，从经济系转到农学院农艺系。1946年夏天毕业于[[广西大学|广西大学]]农学院<ref name='s' />。

1946年下半年，回到南京的家中。此时[[国共内战|国共全面内战]]已经爆发，由于担心失去党的关系，冒险闯进[[梅园新村|梅园新村]]找到即将回解放区的[[董必武|董必武]]。董必武指示他回家隐蔽，会有人上门联络。不久联络人带何康去见了[[中共上海局|中共上海局]]组织部长[[钱瑛|钱瑛]]。[[钱瑛|钱瑛]]说：何世庸、何世平、何康三兄弟的组织关系已经转至[[中共上海局|中共上海局]]布置工作任务。[[何遂|何遂]]和[[缪秋杰|缪秋杰]]合办了瑞明公司，何康在公司任总经理，财务主任缪希霞。

1949年5月上海解放后，任上海市军管会农林处处长、[[华东军政委员会|华东军政委员会]]农林部副部长。1952年至1957年任[[林业部|林业部]]特种林业司司长，[[农业部|农业部]]、[[农垦部|农垦部]]热带作物司司长。1957年至1978年，在海南工作，任[[华南热带作物科学研究院|华南热带作物科学研究院]]院长、党委书记，[[华南热带作物学院|华南热带作物学院]]院长、党委书记。1971年底任广东省农垦总局副局长，主管教育和科研，同时兼两院院长。

1978年1月至1979年2月任[[中华人民共和国农林部|农林部]]副部长。1979年2月至1982年5月任[[国家农业委员会|国家农业委员会]]副主任、党组成员兼[[农业部|农业部]]副部长、党组成员，1982年4月起兼任[[国家计划委员会|国家计划委员会]]副主任。1982年5月至1983年6月任农牧渔业部副部长、党组成员兼国家计委副主任。

1983年起任[[農牧漁業部|農牧漁業部]]部長、党组书记，国家计委副主任（兼），[[中国科协|中国科协]]副主席。在擔任部長期間，推行農業改革，使農業產品在80年代增長達百分之八，並設立農業研究所，推動了中國的農業改革，這使他於1993年獲得[[世界糧食獎|世界糧食獎]]榮譽。

於1993年卸任農業部長職務後，何康被選為[[全國人民代表大會常務委員會|全國人民代表大會常務委員會]]委員，致力研究農業及農業的法律。並且擔任[[聯合國糧食及農業組織|聯合國糧食及農業組織]]顧問。
