== 渊源 ==
何出自[[姬姓|姬姓]]，根据《元和姓纂》记载：“[[周成王|周成王]]弟唐叔虞裔孙韩王安，为[[秦朝|秦]]所灭，子孙分散，[[江淮|江淮]]间音，以韩为何，遂为何氏。”即周成王分封其弟叔虞於韓。其孫韓王安，原居韓邑 ( 一說在[[山西|山西]][[河津縣|河津縣]] ) 為秦國所滅。子孫分散於江淮之間。江淮方言，音「韓」變「何」，逐為「何」氏。因此，何姓是由韓姓分化出來的。唐朝林寶的《[[元和姓纂|元和姓纂]]》、宋代鄭焦的《通志·氏族略》以及《[[廣韻|廣韻]]》、《[[氏族大全|氏族大全]]》、《[[古今姓氏書辯證|古今姓氏書辯證]]》、明代的《[[萬姓通谙|萬姓通谙]]》、《[[氏族博考|氏族博考]]》、《[[姓觿|姓觿]]》，直至近人岑仲勉、孫望的《[[元和姓纂|元和姓纂]]》四校記，無不記載“何氏爲韓氏”，何韓爲一家。

以民族而言，何氏并非仅为[[汉族|汉族]]所专有，亦有部分外族改姓何。[[唐朝|唐朝]]的[[昭武九姓|昭武九姓]]之一即有何氏；清末，更有[[荷蘭|荷蘭]]姓氏Bosman漢化為何姓、詳見[[何東家族|何東家族]]。

何姓被冒用和赐姓的记录有： 
*《汉书五行志》：“何苗，本姓朱，[[冒姓_(行為)|冒姓]]何。”
*《五代史》：“[[吐谷浑|吐谷浑]]亦有何氏。”
*《兰州府志》：“元永吐蕃宣慰使锁南，其子铭入明为河州卫指挥同知，[[赐姓|赐姓]]何。”

==迁徙 ==
[[西晋|西晋]]末年，中原何氏有移居福建者，即[[永嘉入闽八族|永嘉入闽八族]]之一；[[唐高宗|唐高宗]]时，[[光州|光州]][[固始县|固始]]人何嗣韩随[[陈元光|陈元光]]入闽；[[唐僖宗|唐僖宗]]时，又有固始人随王潮、王审之入闽；[[南宋|南宋]]淳佑年间，[[何逖基|何逖基]]由螺阳迁居温陵、浔江，后隐居清源洞；[[宋朝|宋朝]]。

== 現代名人 ==
*[[何大一|何大一]]：美籍著名華裔科學家，發明雞尾酒療法。1996年被美國時代週刊評選為風雲人物。2001年獲總統公民獎章。
*[[何東|何東]]：香港太平紳士、富豪，香港開埠初期代表華人跟英國人爭取權益。第一個華人（中荷混血兒，但自認為中國人）獲准居住半山貴族區。在1890年代是香港四大家族之首。「何東爵士慈善基金」每年捐千餘萬給慈善機構。
*[[何福堂|何福堂]]：香港第一代傳教士，第一個將聖經翻譯為中文。「何福堂書院」以他命名。
*[[何啟|何啟]]：香港貴族，何福堂兒子。孫中山啟蒙老師。「啟德機場」以他命名，創立「何妙齡那打素醫院」。香港第一位獲封為爵士的華人。
*[[何賢|何賢]]：澳門銀行家、慈善家、何厚鏵父親。澳門開埠初期代表華人跟葡國人爭取權益。50年代，何賢家族為澳門三大家族之首。胞兄'何添'為恆生銀行創辦人。
*[[何厚鏵|何厚鏵]]：澳門第一、二屆特別行政區行政長官。第十一屆全國委員會副主席。2011年獲葡萄牙功績大十字勳章。
*[[何鴻燊|何鴻燊]]：（何東家族成員，何東之弟何福之孫）港澳知名商家、慈善家、博士，外號「賭王」。曾任香港地產建設商會會長。曾獲美國博彩業協會成就獎及北京榮譽市民獎。唯一個人獲港澳政府頒最高榮譽勳章——[[大紫荊勳章|大紫荊勳章]]及[[大蓮花榮譽勳章|大蓮花榮譽勳章]]。澳門一條馬路名為「何鴻燊博士大馬路」，是澳門首位中國人在世時獲授此項殊榮。屢獲海外各地政府頒授榮譽勳銜，其中包括葡萄牙、英國、法國、比利時、西班牙、日本等。 
*[[何香凝|何香凝]]：孫中山親密戰友, 曾積極參加抗日救亡運動。全國人大常委會副委員長、全國婦聯名譽主席、中國美術家協會主席、著名畫家。深圳「何香凝美術館」是中國第一個以個人名字命名的國家級美術館。
*[[何應欽|何應欽]] : 中華民國陸軍一級上將主持中國戰區日本投降簽字典禮，曾獲美國總統羅斯福、英皇喬治六世、法國政府與墨西哥政府等授予各式勳章。
*[[何英傑|何英傑]]：香港慈善家、有'何伯'之稱。成立「何英傑基金」捐出數以億元給香港及國內慈善事業。
*[[何善衡|何善衡]]：發明香港「恆生指數」。創立香港「恆生商學書院」、香港中文大學「善衡書院」。成立「何氏教育基金」協助國內教育事業。
*[[何添|何添]]：香港「恆生銀行」創立人之一。與何善衡等人成立「何梁何利基金」(有「中國諾貝爾獎」之稱)，是內地規模最大、影響最廣和最具權威性的科學技術奬勵基金。
*[[何子淵|何子淵]]：辛亥革命元老、中國現代教育奠基人、同盟會嘉應州主盟人、黃岡起義領導人之一。
*[[何享健|何享健]]：中國美的集團創始人，美的2015年收入超過187億歐元，全球擁有10萬名員工，是首家獲取標普、惠譽、穆迪三大國際信用評級的中國家電企業，集團捐公益累計7億元。
*[[何佐芝|何佐芝]]：香港「商業電台」、「佳視」創立人。2001年獲香港金紫荊星章。何東私生子。
*[[何承天_(建筑师)|何承天]]：香港著名建築師、前立法會議員。規劃香港首個大型屋苑「美孚新邨」，其十字型樓宇設計成為日後香港住宅樓宇的藍本。
*[[何弢|何弢]]：香港區徽設計者，著名建築及設計師，曾獲瑞士文化藝術最高榮譽水晶獎。
*[[何文匯|何文匯]]：香港教育家、博士。首個推動粵語正音運動。
*[[何寶生|何寶生]]：香港90年代明星、歌星，接觸佛教後出家為僧，法號'道生'。是香港首個出家的明星。
*[[何俊仁|何俊仁]]：[[香港立法會|香港立法會]]議員、律師、前區域市政局議員、新界西區議員。
*[[何守信|何守信]]：香港男藝人、名司儀。
*[[何家勁|何家勁]]：香港俊俏男演員、歌手，曾演活漫畫'中華英雄'主角華英雄。
*[[何潤東|何潤東]]：美國出生台灣男演員和歌手。活躍中港台，曾獲最受歡迎男演員、全能藝人等多個獎項。
*[[何潔|何潔]]：中國著名女歌手。'超級女聲'第四名。
*[[何炅|何炅]]：中國著名主持人、演員、歌手。歌曲《梔子花開》深受好評。
*[[何國禧|何國禧]]：香港70年代歌星，成名曲：'親親嗲哋'。
*[[何超儀|何超儀]]：香港女歌星和演員、何鴻燊女兒。
*[[何超瓊|何超瓊]]：港澳名媛、商人、何鴻燊女兒，曾為亞洲女首富。
*[[何猷龍|何猷龍]]：港澳著名商人、何鴻燊兒子。創立澳門「新濠天地(City Of Dream)」、「新濠影滙(Studio City)」。
*[[何家英|何家英]]：中國著名水墨畫家。
*[[何多苓|何多苓]]：中國著名油畫家。
*[[何明憲|何明憲]]：台灣勤美集團董事長。在台、港擁有兩家上市公司—'勤美'、'勤美達'，總市值高達新台幣二百三十億元。
*[[何耀珊|何耀珊]]：新加坡女歌手。歌曲'Without Love'和'One With You' 曾榮獲美國告示排行榜舞曲榜和英國 MusicWeek 俱樂部榜的第一名。
*[[何維健|何維健]]：新加坡男歌手。首張國語專輯創下新加坡有史以來「單日銷售最高紀錄」發行第一天，賣出2000張創下當地紀錄。榮獲《新加坡金曲獎》。
*[[何韻詩|何韻詩]]：香港著名唱作女歌手，[[亞太區|亞太區]]極具影響力和地位的香港巨星[[梅艷芳|梅艷芳]]的唯一女徒弟。
*[[何愛瑜|何愛瑜]]：'李小龍'母親、富商何甘棠(何甘棠生父是中國人，何東爵士同母異父之弟)之養女。
*[[何振梁|何振梁]]：體育外交家。曾任國際奧會執行委員會委員。中國奧會名譽主席，一手促成「中國第一屆奧運會」。
*[[何雯娜|何雯娜]]：為中國奪得蹦床歷史上第一塊奧運金牌。
*[[何可欣|何可欣]]：北京奧運高低槓金牌冠軍。
*[[何沖|何沖]]：北京奧運男子三米板跳水冠軍，有'難度王'之稱。
*[[何華武|何華武]]：中國鐵道部總工程師，為中國高鐵創最高時速世界記錄。
*[[何禮軒|何禮軒]]：香港基本法名嘴大賽冠軍、標語比賽冠軍、電腦開發家。
*[[何艷娟|何艷娟]]：[[香港|香港]]女演員、[[2014年度香港小姐|2014年度香港小姐]]季軍、港大法律系及校花。
*[[何依婷|何依婷]]：[[香港|香港]]女演員、[[2017年度香港小姐|2017年度香港小姐]]亞軍。
*[[何念茲|何念茲]]：馬來西亞女歌手
*[[何遠東|何遠東]]：無線電視男藝員
*[[何廣沛|何廣沛]]：無線電視男藝員
*[[何晶晶|何晶晶]]：香港歌手、配音員
*[[何璐怡|何璐怡]]：香港配音員
*[[何寶珊|何寶珊]]：香港配音員
