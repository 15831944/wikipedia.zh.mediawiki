《'''楚国八百年'''》，2014年播出的[[中国大陆|中国大陆]]人文[[纪录片|纪录片]]，由[[中国中央电视台|中国中央电视台]]、[[湖北广播电视台|湖北广播电视台]]联合出品，[[中国中央电视台纪录频道|央视纪录频道]]和[[湖北卫视|湖北卫视]]联合摄制，[[张晓敏|张晓敏]]执导。该片分为《初生》、《觉醒》、《受挫》、《称霸》、《歧途》、《劫难》、《变革》、《涅槃》等八集，每集45分钟，从2014年2月24日至3月3日在[[CCTV-1|CCTV-1]]、[[CCTV-9|CCTV-9]]、湖北卫视三个频道同步播出<ref name="中国新闻网">{{cite web |title=大型历史纪录片《楚国八百年》首次“复活”历史 |url=http://www.chinanews.com/cul/2014/02-20/5864107.shtml |website=中国新闻网 |accessdate=2019-02-04 |date=2014-02-20 |archive-url=https://web.archive.org/web/20190204174551/http://www.chinanews.com/cul/2014/02-20/5864107.shtml |archive-date=2019-02-04 |dead-url=no }}</ref>。该片首次完整地讲述[[楚国|楚国]]八百年的历史，以全新的视角解读楚文明，全面梳理楚国的诞生、发展、高峰、衰落和灭亡的轨迹，展现一幅楚国兴衰的历史长卷，让观众清晰看到楚国的历史和楚文化盛衰的背景及成因<ref name="央视复播">{{cite web |title=《楚国八百年》央视复播 楚文化再度"发光" |url=http://ent.163.com/14/0716/16/A19P6AR300031GVS.html |website=网易娱乐 |accessdate=2019-02-04 |date=2014-07-16 |archive-url=https://web.archive.org/web/20190204174529/http://ent.163.com/14/0716/16/A19P6AR300031GVS.html |archive-date=2019-02-04 |dead-url=no }}</ref>。

== 拍摄制作 ==
该片由[[中国中央电视台|中国中央电视台]]和[[湖北广播电视台|湖北广播电视台]]联合出品，经历近两年的策划、摄制和打磨<ref name="央视复播" />。该片的总撰稿是知名作家、历史学者张宏杰，曾用两三个月时间重新阅读、审视楚史，确定以“中原化”对楚国兴衰的影响为贯穿《楚国八百年》的主要精神线索，并以主要人物和事件作为推动故事叙事的主要线索<ref name="深圳特区报">{{cite web |title=评《楚国八百年》：打捞失落的楚文明 |url=http://www.chinanews.com/cul/2014/03-20/5972914.shtml |website=中国新闻网-深圳特区报 |accessdate=2019-02-14 |date=2014-03-20 |archive-url=https://web.archive.org/web/20140321082114/http://www.chinanews.com/cul/2014/03-20/5972914.shtml |archive-date=2014-03-21 |dead-url=no }}</ref>。楚国历史悠长，全盛时期地域广阔，为了在八集篇幅内对其进行充分的呈现，该片的导演们化繁为简，将楚国历史分为蛰伏萌芽期、图强发展期、春秋争霸期、中衰复兴期、变法争雄期、衰落灭亡期等阶段。每集选取一个时期最为关键的人物，情节也以其为中心进行展开，从而能够以充足的篇幅来还原历史情境，发掘事件背后的历史规律<ref>{{cite web |author1=张晓敏 |title=纪录片《楚国八百年》导演手记：让历史不再遥远 |url=http://1118.cctv.com/2014/03/07/ARTI1394161332722711.shtml |website=央视网 |accessdate=2019-02-14 |date=2014-03-07 |archive-url=https://web.archive.org/web/20170820114545/http://1118.cctv.com/2014/03/07/ARTI1394161332722711.shtml |archive-date=2017-08-20 |dead-url=no }}</ref><ref name="张晓敏2014">{{cite journal |author1=张晓敏 |author2=杨光照 |author3=崔伊 |author4=陈爽 |title=《楚国八百年》导演创作札记 |journal=中国电视(纪录) |date=2014 |issue=7 |pages=76-81}}</ref>。
