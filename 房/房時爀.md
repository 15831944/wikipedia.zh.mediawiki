'''房時爀'''（{{kor|k='''방시혁 '''|rm='''Bang Si Hyuk'''}}，{{bd|1972年|8月9日|catIdx=B房}}），藝名'''Hitman Bang'''（{{lang-ko|히트맨 뱅}}）。[[韓國|韓國]]知名作曲家、音樂製作人，[[Big_Hit娛樂|Big Hit娛樂]]創辦人與現任代表理事 。[[首爾大學|首爾大學]]美術學系畢業，在校期間於{{link-ko|柳在夏音樂比賽大會|유재하 음악경연대회}}獲得銅獎。之後成為作曲家和音樂製作人，製作出許多熱門歌曲。曾在選秀節目《偉大的誕生》擔任評審導師。<ref>{{cite web |title=방시혁도 심사위원..'위대한탄생' 5인 멘토 확정 |url=http://entertain.naver.com/read?oid=108&aid=0002055442 |date=2010-11-05 |work=STARNEWS}}</ref>

== 生涯 ==
房時爀畢業於韓國知名的{{link-ko|京畿高中|경기고등학교}}和首爾大學。父親[[房極允|房極允]]（朝鲜语：방극윤）曾任職{{link-ko|韓國勤勞福祉公團|근로복지공단}}理事長、{{link-ko|首爾地方勞動廳|서울지방고용노동청}}廳長，現任韓國社會保險研究所理事長；妹妹畢業於[[梨花女子大學|梨花女子大學]]音樂作曲系，現在是專業珠寶設計師。<ref>{{cite web |title=쿠키人터뷰 - ‘히트 메이커’ 방시혁 |url=http://news.kukinews.com/article/view.asp?page=1&gCode=ent&arcid=1284052836 |date=2010-09-09 |work=쿠키뉴스}}</ref>

從國中時候開始進行音樂活動，沉醉於各種樂器。國中時期學習吉他而夢想成為音樂人，但作為學習良好的模範生，聽從父母，為了專心學習而放棄對音樂的熱情，直到考取首爾大學，開始展開正式的音樂活動。1995年，擔任男子二重唱男子團體Check〈人魚小姐〉作曲人而出道，並且在同年舉辦的柳在夏音樂比賽大會獲得銅獎。1997年，成為[[JYP娛樂|JYP娛樂]]製作人，從[[朴軫永|朴軫永]]第三張專輯〈이별탈출〉開始，到[[g.o.d|g.o.d]]〈Friday Night〉〈天藍色氣球〉、[[朴志胤|朴志胤]]〈我陷入了愛情〉、[[Rain|Rain]]〈壞男人〉〈I Do〉等擔任作曲。2005年獨立，成立Big Hit娛樂，並且成為代表理事，擔任[[林正姬|林正姬]]、[[8eight|8eight]]、[[2AM|2AM]]、[[防彈少年團|防彈少年團]]等歌手的音樂製作人。知名作品有[[白智榮|白智榮]]〈像中槍一樣〉、8eight〈沒有心臟〉、2AM〈死也不能放開你〉、[[現代汽車|現代汽車]]i-30廣告歌曲等。<ref>{{cite web |title=방시혁, 명문대 졸업한 엄친아…‘총 맞은 것처럼’ 등 히트곡 제조기 |url=http://news.donga.com/3//20110225/35108910/1 |date=2011-02-25 |work=동아일보}}</ref>2010年至2011年，擔任[[文化廣播_(韓國)|MBC]]選秀節目《偉大的誕生》評審導師。2011年4月27日，受邀回母校首爾大學《媒體訊息文化論壇系列》進行演講，演講主題為「選秀節目的票房」。<ref>{{cite web |title=방시혁 모교강의 화제, 서울대 출신 스타 |url=http://m.stylem.mt.co.kr/view.html?no=2011042909334013362#_adtep |date=2011-04-29 |work=stylem}}</ref>
