'''房宁'''{{BD|1957年|6月|||catIdx= F房}}出生于[[北京市|北京市]]，毕业于[[首都师范大学|首都师范大学]]政教系，曾在[[美国|美国]]游学，现为[[中国社科院政治学研究所|中国社科院政治学研究所]]所长，[[中国政治学会_(中华人民共和国)|中国政治学会]]副会长，<ref>{{cite web|url=http://china.huanqiu.com/info/2011-12/2242188.html|title=环球时报年会嘉宾：中国社科院政治学所所长房宁|accessdate=2011-12-07|publisher=环球网|archiveurl=https://web.archive.org/web/20120205132135/http://china.huanqiu.com/info/2011-12/2242188.html|archivedate=2012年2月5日|work=|language=zh-cn|deadurl=yes}}</ref> 2007年起享受[[国务院|国务院]]政府特殊津贴。<ref name=" fish" >{{cite web |url=http://news.xinhuanet.com/2011-08/09/c_121831048.htm |title=房宁：共同富裕不能等不能拖 重庆的举措值得期待 |accessdate= 2011年8月9日|publisher= 新华网|archiveurl=http://news.xinhuanet.com/2011-08/09/c_121831048.htm |archivedate=2012年11月4日 |work= |language =zh-cn}}</ref>

== 生平 ==
1957年，房宁出生在北京市的一个[[中医|中医]]世家。[[文革|文革]]爆发后，[[上山下乡|上山下乡]]，房宁到北京市郊区上地村参加农业劳动，恢复高考之后，房宁考入[[北京师范学院|北京师范学院]]政教系（今首都师范大学），毕业后留校任教。1987年，房宁远赴美国游学，1990年，房宁升格为北京师范学院副教授，1994年，房宁升格为教授，2001年，房宁出任中国社会科学院任政治学所副所长。<ref>{{cite web |url=http://theory.people.com.cn/GB/40764/127620/174371/10411575.html |title= 房宁教授简介|accessdate=2009年11月19日 |publisher=中国共产党新闻网 |archiveurl=http://theory.people.com.cn/GB/40764/127620/174371/10411575.html |archivedate=2012年11月4日|work= |language =zh-cn}}</ref>

=== 军队国家化问题 ===
房宁反对[[军队国家化|军队国家化]]，认为这是一个“假问题”，[[中国人民解放军|中国人民解放军]]必须听从[[中国共产党|中国共产党]]的指挥。<ref>{{cite web |url=http://theory.people.com.cn/GB/9935040.html |title=房宁：“军队国家化”是一个假问题 |accessdate=2009年8月26日 |publisher=中国共产党新闻网 |archiveurl=http://theory.people.com.cn/GB/9935040.html |archivedate=2012年11月4日 |work= |language =zh-cn}}</ref>

=== 重庆模式问题 ===
房宁支持[[薄熙来|薄熙来]]在[[重庆市|重庆市]][[唱红打黑|唱红打黑]]，指出“现阶段的‘唱红’并不是简单作为一种[[意识形态|意识形态]]的宣示，而是一种[[文化|文化]]自觉和文化建构。不能把‘唱红’看作是为了某种政策制造意识形态氛围。那样看就太离谱了。‘唱红’，正是[[中国特色社会主义价值观|中国特色社会主义价值观]]的一个很好载体。”，否认唱红打黑是文化大革命的延续和[[左倾|左倾]]。<ref>{{cite web |url= http://news.sina.com.cn/c/sd/2011-10-27/120623371313.shtml|title=重庆“唱红”：对市场崇拜的一种“对冲” |accessdate=2011年10月27日 |publisher=新浪网 |archiveurl= http://news.sina.com.cn/c/sd/2011-10-27/120623371313.shtml|archivedate=2012年11月4日 |work= |language =zh-cn}}</ref> 对薄熙来在重庆市采取的“[[共同富裕|共同富裕]]”措施深表期许。<ref name=" fish" />

===法治和人治===
面对[[习近平|习近平]]等新一届政府提出的建立“法治中国”的理念，房宁反对纯粹法治社会，主张人治和法治结合，指出：“不能说法治比人治好，人治比法治坏。不能把人治妖魔化，把法治神圣化。”<ref>{{cite web |url=http://www.zaobao.com/special/zbo/story20141210-422661 |title=http://www.zaobao.com/special/zbo/story20141210-422661 |accessdate=2014 |publisher= |work=新加坡联合早报网 |language=zh-hans }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
