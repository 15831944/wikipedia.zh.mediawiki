'''过氧化钠'''化学式为'''Na<sub>2</sub>O<sub>2</sub>'''，又称二氧化钠或双氧化钠，是[[钠|钠]]在[[氧气|氧气]]或[[空气|空气]]中[[燃烧|燃烧]]的产物。

== 物理性质 ==
在常温下，纯品过氧化钠为白色，但一般见到的过氧化钠呈淡黄色（混有 [[超氧化钠|超氧化钠]] 的缘故），[[密度|密度]]为2.805g／cm<sup>3</sup>，熔点为460[[摄氏度|摄氏度]]。加热时，过氧化钠的颜色会变黄。过氧化钠易变质，应密封保存。过氧化钠对人有强烈的腐蚀性，与易燃物接触易引起火灾，在高温下甚至会发生爆炸。过氧化钠着火不能用水扑灭，必须用沙土或盐盖灭。　

== 制取 ==

过氧化钠主要靠钠的燃烧来制取（得到的产物一般含有约10％的[[超氧化钠|超氧化钠]]），工业上将[[钠|钠]]装入[[铝|铝]]盘，通入干燥且不含二氧化碳的[[空气|空气]]，先保持180~200摄氏度，使钠氧化成氧化钠（Na<sub>2</sub>O），再增加通入气体量，同时升温至300~400摄氏度，即可得到纯净的过氧化钠。

其化学反应方程式为：

<math>\rm4Na + O_2 \rightarrow 2Na_2O\,</math>

<math>\rm2Na_2O + O_2 \rightarrow 2Na_2O_2\,</math>

过氧化钠也可以通过过氧化氢与氢氧化钠的反应制取，化学方程式为：

<math>\rm H_2O_2 + 2NaOH \rightarrow Na_2O_2 + 2H_2O\,</math>

== 化学性质 ==

过氧化钠是[[离子化合物|离子化合物]]，其中氧元素显-1价，钠元素显+1价，其水合物有Na<sub>2</sub>O<sub>2</sub>‧2H<sub>2</sub>O和Na<sub>2</sub>O<sub>2</sub>‧8H<sub>2</sub>O两种。

过氧化钠是强[[氧化剂|氧化剂]]，可以与多种金属单质以及非金属化合物发生反应。

过氧化钠与水反应，生成[[氢氧化钠|氢氧化钠]]和[[氧气|氧气]]：

<math>\rm 2Na_2O_2 + 2H_2O \rightarrow 4NaOH + O_2 </math>

这是一个[[放热反应|放热反应]]，又由于生成物中含有氧气，因此极易引起可燃物的燃烧和爆炸。

过氧化钠与[[二氧化碳|二氧化碳]]反应，生成[[碳酸钠|碳酸钠]]（Na<sub>2</sub>CO<sub>3</sub>）和氧气：

<math>\rm 2Na_2O_2 + 2CO_2 \rightarrow 2Na_2CO_3 + O_2</math>

过氧化钠与稀[[硫酸|硫酸]]反应，生成[[硫酸钠|硫酸钠]]（Na<sub>2</sub>SO<sub>4</sub>）、[[水|水]]和氧气：

<math>\rm 2Na_2O_2 + 2H_2SO_4 \rightarrow 2Na_2SO_4 + O_2 +2H_2O</math>

可以把过氧化钠溶解在低温的硫酸中，然后减压蒸馏即可得到[[过氧化氢|过氧化氢]]（H<sub>2</sub>O<sub>2</sub>）：

<math>\rm Na_2O_2 + H_2SO_4 \rightarrow H_2O_2 + Na_2SO_4</math>

类似的还有与盐酸的反应：

<math>\rm 2Na_2O_2 + 4HCl \rightarrow 4NaCl + O_2 +2H_2O</math>

在碱性环境中，过氧化钠可以把化合物中+3价的[[砷|砷]]（As）氧化成+5价，把+3价的[[铬|铬]]（Cr）氧化成+6价。利用这个反应可以将某些岩石矿物中的+3价铬除去，方法为：在600~700摄氏度的高温下用过氧化钠将铁矿中的3价铬氧化成Na<sub>2</sub>CrO<sub>4</sub>，而Na<sub>2</sub>CrO<sub>4</sub>可以用水溶掉：

<math>\rm 2Fe(CrO_2)_2 + 7Na_2O_2 \rightarrow Fe_2O_3 + 4Na_2CrO_4+3Na_2O</math>

另外，过氧化钠可以将[[铁|铁]]单质氧化成含FeO<sub>4</sub><sup>2−</sup>的[[铁酸盐|铁酸盐]]，还可以在一般条件下将[[有机物|有机物]]氧化成[[乙醇|乙醇]]和[[碳酸盐|碳酸盐]]，也可以与硫化物和氯化物发生剧烈反应。

过氧化钠的[[热稳定性|热稳定性]]好，可加热到[[熔融|熔融]]状态而不分解。

== 用途 ==

由于过氧化钠具有强氧化性，因此可以用来漂白织物、麦杆、羽毛等，也可用做除臭剂或消毒剂。在[[分析化学|分析化学]]上用于处理矿样。由于它和二氧化碳反应放出氧气，因此可以用来在[[潜水艇|潜水艇]]等缺少氧气的环境中提供供人呼吸的氧气。
