《'''过春天'''》（{{lang-en|'''''The Crossing'''''}}）是2018年中国、香港聯合製作的社會議題青春[[犯罪片|犯罪]][[剧情片|剧情片]]、导演[[白雪_(导演)|白雪]]的处女作，由[[黄尧_(演员)|黄尧]]、[[孙阳_(演员)|孙阳]]、[[汤加文|汤加文]]、[[倪虹洁|倪虹洁]]、[[江美仪|江美仪]]、[[廖启智|廖启智]]和[[焦刚|焦刚]]主演。影片讲述了[[双非问题|单非]]家庭的16岁高中女学生“佩佩”游走于深圳和香港两地间的[[香港境內水貨客問題|水货客]]冒险生活。影片于2018年9月7日在[[2018年多倫多國際影展|第43届多伦多国际电影节]]全球首映，2019年3月15日在中国正式上映<ref>{{cite news|url=http://ent.huanqiu.com/movie/yingshi-neidi/2019-02/14376605.html?agt=1720|title=《过春天》改档上映 3月15日水客少女硬核开春|publisher=环球网综合|date=2019-02-22|accessdate=2019-02-27}}</ref>。

== 剧情 ==
16岁[[双非问题|单非]]家庭女孩“佩佩”（[[黄尧_(演员)|黄尧]] 饰），她的城市既是香港、也是深圳，一边有身份，一边有生活。为了和闺蜜去日本看雪，佩佩努力打工但还是来不及賺够去日本的机票钱；一次偶然的机会让佩佩结识了走私水货的闺蜜男友阿豪并凭自己聪明伶俐走上了走私苹果手机的道路。很快佩佩賺够了机票钱，却因为闺蜜发现自己和她男友暖昧而公开翻脸，和阿豪的单干走私生意也被老大发现，眼看没好果子吃的时候警察刚好破门而入，反而救了佩佩。

== 发行 ==
影片于2018年9月7日在[[2018年多倫多國際影展|第43届多伦多国际电影节]]全球首映，获得媒体的赞扬<ref>{{cite web |title=田壮壮监制《过春天》多伦多首映获赞 以少女视角展现社会问题 被称“中国新声音” |url=http://news.mtime.com/2018/09/10/1584119.html |website=Mtime时光网 |accessdate=2018-12-21 |date=2018-09-10}}</ref><ref>{{cite web |title=《过春天》多伦多获外媒盛赞 中国最有力度的新声音 |url=http://fun.youth.cn/gnzx/201809/t20180910_11722985.htm |website=中国青年网 |date=2018-09-10 |accessdate=2018-12-21}}</ref>。在第二届[[平遥国际电影展|平遥国际电影展]]上，影片斩获最佳影片和最佳女演员两项大奖<ref>{{cite web |title=平遥电影节：《过春天》获最佳影片最佳女主角 |url=http://ent.sina.com.cn/m/c/2018-10-18/doc-ihmhafis3397690.shtml |website=新浪网 |accessdate=2018-12-21 |date=2018-10-18}}</ref>。

影片入围[[第69屆柏林影展|第69屆柏林影展]]新生代单元的消息公布当天，影片宣布定档2019年3月8日上映<ref>{{cite web |title=《过春天》《第一次的离别》入围柏林 均为女导演处女作 《过春天》定档明年妇女节 |url=http://news.mtime.com/2018/12/19/1587329.html |website=Mtime时光网 |accessdate=2018-12-21 |date=2018-12-19}}</ref><ref>{{cite web |title=《过春天》入围柏林电影节新生代单元 |url=http://ent.huanqiu.com/movie/yingshi-neidi/2018-12/13853847.html |website=环球网 |accessdate=2018-12-21 |date=2018-12-20}}</ref>，后改档到3月15日上映。
