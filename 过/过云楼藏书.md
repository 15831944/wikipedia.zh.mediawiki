== 起源 ==
"过云楼”是江南著名的私家藏书楼，位于苏州市干将路，世有“江南收藏甲天下，过云楼收藏甲江南”之称，现为苏州市文物保护单位。过云楼藏书源自清末官员[[顧文彬|顾文彬]]、[[顧承|顾承]]父子的一批書画收藏<ref>[http://www.subaonet.com/cul/2012/0422/910716.shtml 苏州新闻网《揭秘过云楼藏书》2012 年 4 月 22 日]</ref>。时值乱世，私人珍藏都急于转手，顧文彬父子趁机收购了大量珍贵字画，这些字画的存在奠定了过云楼藏品的基础。过云楼在顾鹤逸的管理下经过六代人150年的传承，成为集宋元古椠、精写旧抄、明清佳刻、碑帖印谱擁有800余种藏品的大型藏書楼，楼中宋元旧刻、精写旧抄本、明清精刻本、碑帖印谱等藏品琳琅满目，数不胜数。

<br />

== 规模 ==
过云楼藏书包括珍贵的宋版书2部（宋版《锦绣万花谷》前、后集，宋杜大桂编纂的《皇朝名臣续编碑传婉琰集》）、元刻书3部（元[[胡一桂|胡一桂]]撰《周易启蒙传三篇外传一篇》，元[[黄瑞节|黄瑞节]]附录《易学启蒙朱子成书》，元太监[[王公|王公]]编《针灸资生经》七卷）。此外，这批藏书中还有清代[[黄丕烈|黄丕烈]]、[[顾广圻|顾广圻]]、[[鲍廷博|鲍廷博]]等名流大家的批校手迹。其中最为难得的是自清朝以来就在藏书界极负盛名的宋刻《[[锦绣万花谷|锦绣万花谷]]》。《锦绣万花谷》于南宋淳熙十五年刊刻发行，为南宋大型类书，也是目前已知海内外最大部头的宋版书。此书是作者把自己曾经看过的书按通晓的知识分门别类、汇编而成的。这部“宋代的百科全书”，将现存百科全书的年代拉至历史上限。
== 收藏情况 ==
1992年，江苏省南京图书馆花費40萬元<ref>{{cite web |author1=薛颖旦 龚倩 王宏伟 董晨 |title=过云楼藏书：17部宋元珍本人间罕见 |url=http://finance.ifeng.com/money/special/jjgyl/20120628/6675393.shtml |accessdate=2018-08-29}}</ref>购得过云楼藏书共计541部3707册，覆盖了经、史、子、集四大類，几乎涵盖了所有的纸本古籍类型，其中包含9部宋刻本以及大量批校题跋本、四库底本、稿本、抄本、拓本、钤印本、活字印本、套印本，甚至一些来自朝鲜和日本的刻本，时间上跨越宋元明清，拥有较长的时间跨度。<ref>[http://www.jslib.org.cn/njlib_gqsb/201206/t20120627_112804.htm 南京图书馆〈南图召开四分之三过云楼藏书新闻发布会〉2012年6月27日]</ref>

2012年6月4日，北京匡时拍卖夜场举槌开拍《[[锦绣万花谷|锦绣万花谷]]》，江苏凤凰传媒集团以2.162亿元人民币成交价购得四分一之藏書，创下中国古籍拍卖的世界纪录。<ref>[http://www.chinanews.com/cul/2012/06-05/3938008.shtml 中国新闻网〈“过云楼藏书”2.162亿元成交 创古籍拍卖世界纪录〉2012年6月5日]</ref>
2012年9月，部分“过云楼藏书”於台北“第八届海峡两岸图书交易会”上展出，吸引了台湾各界的高度关注，成为展会亮点之一。

2012年12月12日至14日，所藏的四分之一“过云楼藏书”在南京市规划建设展览馆向公众免费展出。
