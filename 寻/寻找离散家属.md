《'''寻找离散家属'''》（{{朝|'''이산가족을 찾습니다'''||'''Isangajog-eul Chajseubnida '''}}）是[[韩国放送公社|韩国放送公社]]（KBS）[[KBS第1频道|第1台]]于1983年6月30日至11月14日，为韩国离散家庭寻找亲属而播出的特别电视直播节目。

== 简介 ==
{{UTC time|韩国标准时|+9:00}}
1983年，为了纪念[[朝鲜战争|朝鲜战争]]停战三十周年，KBS策划了一期名为《寻找离散家属》的寻找失散亲人的节目。节目最初计划仅为90分钟，但由于观众反应强烈，KBS继续制作该节目，共计持续了138天。节目播出后，韩国各地的群众来到KBS总部前的[[汝矣岛公园|汝矣岛广场]]上，搭起帐篷，张贴寻人启事。<ref name=cha>{{cite web |url=http://chn.cha.go.kr/chinese/html/sub3/sub13.jsp |language= |title=KBS电视台特别节目直播《寻找离散家属》 |publisher=[[大韩民国文化财厅|大韩民国文化财厅]] |date= |accessdate=2018-09-12 |archive-url=https://web.archive.org/web/20160402074045/http://chn.cha.go.kr/chinese/html/sub3/sub13.jsp |archive-date=2016-04-02 |dead-url=no }}</ref><ref>{{cite web |url=https://innews.pts.org.tw/?p=6079 |language= |title=他們如何透過韓戰，喚起全世界的共鳴？ |publisher=[[独立特派员|独立特派员]] |date= |author=卓冠齊 |accessdate=2018-09-12 |archive-url=https://web.archive.org/web/20180514134940/https://innews.pts.org.tw/?p=6079 |archive-date=2018-05-14 |dead-url=no }}</ref>

《寻找离散家属》展现了战争与分裂之痛，通过家属的重逢缓解朝鲜半岛的紧张局势，提醒人们南北统一之重要性。节目的播出也促成了1985年9月的第一次朝韩离散家属团聚活动。<ref>{{cite web|url=http://m.news.cctv.com/2018/08/05/ARTIx5WtBpDUFJXyI3OcnJjZ180805.shtml | language= | title=曾经有一档节目在韩国特别火……他们关注的这个主题，如今有了新进展 |publisher=[[中国中央电视台|中国中央电视台]] |date=2018-08-05 |accessdate=2018-09-12}}</ref>

节目于1983年6月30日22:15首次播出至同年11月14日4:00结束，共直播了453小时45分。报名参加节目的申请者有100,952名，最终参与节目者有53,536名，并留下20,522份相关文件记录。节目的播出促成了10,189名离散家属与亲人重聚。<ref name=cha/>
