《'''寻找刘三姐'''》（{{lang-en|'''A Singing Fairy'''}}）导演朱枫，主演苏有朋、黄圣依、魏宗万、车永莉、爱新觉罗·启星、钱柳吟。<ref>{{cite web |url=http://ent.qq.com/a/20100430/000305.htm |title=《寻找刘三姐》今上映 苏有朋领衔传说新篇章 |accessdate=2010 |publisher=腾讯网 |work= |language=zh-hans}}</ref>

== 剧情 ==
该片讲述[[美籍华裔|美籍华裔]]韦文德热爱[[音乐|音乐]]热爱祖地“[[广西壮族自治区|广西壮族自治区]]”，因而来到[[中华人民共和国|中华人民共和国]]寻找“刘三姐”，并且找到心爱的女子“刘甜甜”，两人结为连理。

== 花絮 ==
该片在[[广西壮族自治区|广西壮族自治区]][[桂林|桂林]]、[[宜州|宜州]]、[[罗城|罗城]]等地取景。<ref>{{cite web |url=http://ent.qq.com/a/20100429/000600.htm|title=《寻找刘三姐》首映 苏有朋黄圣依“终成眷属”  |accessdate=2010 |publisher=腾讯网 |work= |language=zh-hans}}</ref>

苏有朋虽然是歌手，但是并不会民族山歌，为了演好这部电影，他拜师当地民族山歌高手学艺。<ref>{{cite web |url=http://ent.qq.com/a/20100424/000081.htm |title=《寻找刘三姐》举行首映会 苏有朋现场飙山歌 |accessdate=2010 |publisher=腾讯网 |work= |language=zh-hans}}</ref>
