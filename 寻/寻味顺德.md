《'''寻味顺德'''》（{{lang-en|''A Bite of Shunde / In Search of Shunde''}}）是2016年[[中国中央电视台纪录频道|中国中央电视台纪录频道]]和[[佛山电视台|佛山电视台]]顺德分台联合拍摄的一部美食[[纪录片|纪录片]]，共有三集，介绍了中国[[广东省|广东省]][[佛山市|佛山市]][[顺德区|顺德区]]的美食。这部纪录片由导演刘硕和费牖明所执导，陈晓卿担任总顾问，[[李立宏|李立宏]]担任普通话旁白解说，制作团队大部分由之前制作《[[舌尖上的中国|舌尖上的中国]]》的制作人员组成。2016年4月19日，顺德华桂园举行了《寻味顺德》的首映仪式，4月30日起接连3日，纪录片在中国中央电视台纪录频道首播。<ref>{{cite web |title=《寻味顺德》五一在央视播出 |url=http://ent.ifeng.com/a/20160422/42609902_0.shtml |website=凤凰娱乐 |publisher=信息时报 |accessdate=2019-02-01 |date=2016-04-22}}</ref>

== 制作 ==
2015年，《寻味顺德》的制作组到顺德进行调研和拍摄，在一年里共录制了长达600小时的原始素材，后经过筛选和剪辑，制作成每集50分钟的三集记录片，其中有很多[[微距摄影|微距摄影]]镜头，对食物进行[[特写|特写]]<ref>{{cite web |title=舌尖团队最新力作，《寻味顺德》五一首播！ |url=http://jishi.cctv.com/2016/04/20/ARTIb7wuFl66XYuO2x5yFpoU160420.shtml |website=央视网 |accessdate=2019-02-01 |date=2016-04-20 |archive-url=https://web.archive.org/web/20170731051446/http://jishi.cctv.com/2016/04/20/ARTIb7wuFl66XYuO2x5yFpoU160420.shtml |archive-date=2017-07-31 |dead-url=no }}</ref><ref>{{cite web |title=看完这部《寻味顺德》，我哭着点了一份外卖 |url=http://www.myzaker.com/article/59d63c021bc8e04070000001 |website=ZAKER新闻 |publisher=橘子娱乐 |accessdate=2019-02-01 |date=2017-10-06}}</ref><ref>{{cite web |title=《寻味顺德》导演手记：充满滋味的一年 |url=http://jishi.cctv.com/2016/04/28/ARTI8dAEI7kXEzIYa8dKOwIj160428.shtml |website=央视网 |accessdate=2019-02-01 |date=2016-04-29 |archive-url=https://web.archive.org/web/20180702062403/http://jishi.cctv.com/2016/04/28/ARTI8dAEI7kXEzIYa8dKOwIj160428.shtml |archive-date=2018-07-02 |dead-url=no }}</ref>。2016年6月3日，《舌尖上的中国·寻味顺德》丛书发售。这套丛书附赠了《寻味顺德》纪录片的[[DVD|DVD]]光盘，丛书内容则是在纪录片的基础上进行延伸研究，将一些研究顺德饮食文化的学者和美食家的研究成果著录成书<ref>{{cite web |title=《寻味顺德》新书发布南京站 舌尖上的中国精彩继续 |url=http://js.ifeng.com/a/20160710/4741282_0.shtml |website=凤凰网江苏 |accessdate=2019-02-01 |date=2016-07-10}}</ref>。2018年6月，制作团队对顺德美食进行进一步的调研，准备拍摄第二季<ref>{{cite web |title=《寻味顺德》第二季正在酝酿 |url=http://news.southcn.com/community/content/2018-06/23/content_182327703.htm |website=南方网 |publisher=广州日报 |accessdate=2019-02-01 |date=2018-06-23 |archive-url=https://web.archive.org/web/20180623072405/http://news.southcn.com/community/content/2018-06/23/content_182327703.htm |archive-date=2018-06-23 |dead-url=no }}</ref>。
