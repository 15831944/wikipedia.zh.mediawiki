{{日本作品|雲之彼端，約定的地方|雲のむこう、約束の場所 -The place promised in our early days-}}是[[日本|日本]][[導演|導演]][[新海誠|新海誠]]於2004年製作的第一部長篇[[動畫電影|動畫電影]]。本作品獲選為2005年第九屆日本[[文部省|文部省]][[文化廳媒體藝術祭|文化廳媒體藝術祭]]動畫部門推薦的作品，以及第59屆每日電影競賽動畫電影獎<ref>{{cite news|url =http://hk.apple.nextmedia.com/supplement/culture/art/20070918/10169787 |title = 秒速5厘米 搗東京私竇 後宮崎駿 新海誠 |publisher = 香港蘋果日報 |date = 2007年9月18日}}</ref><ref>{{cite web|url=http://acg.gamer.com.tw/acgDetail.php?s=341 | title=【動畫】雲之彼端 約定的地方|publisher=巴哈姆特|accessdate=2014-01-21 |language=zh-Hant}}</ref>。

== 製作 ==
本作品是[[新海誠|新海誠]]繼《[[她與她的貓|她與她的貓]]》（{{lang|ja|彼女と彼女の猫}}）與《[[星之聲|星之聲]]》（{{lang|ja|ほしのこえ}}）之後的第三部作品，同時也是首部放映時間達91分的長篇動畫作品。2004年11月20日於[[澀谷|澀谷]]Cinemax公開首映，其他的戲院也在數日後開始放映。[[台灣|台灣]]於2006年8月6日在[[衛視電影台|衛視電影台]]以中文配音方式放映。

本作品延續前作的高製作水準，細膩生動的作畫品質與配樂上的巧妙搭配在日本上映當地獲得極高的評價。擊敗[[宮崎駿|宮崎駿]][[監督|監督]]的《[[哈爾的移動城堡|-{zh-cn:哈;zh-tw:霍;zh-hk:哈;}-爾的移動城堡]]》拿下[[日本|日本]]第59屆每日電影競賽動畫電影獎（{{lang|ja|毎日映画コンクールアニメーション映画賞}}）。

2005年12月26日推出由[[加納新太|加納新太]]所描寫的小說版<ref>{{cite web|url=http://www.amazon.co.jp/dp/4757725884/|title=雲のむこう、約束の場所|publisher=[[Amazon.com|Amazon.co.jp]]|accessdate=2014年10月10日|language=ja}}</ref>，同一時間也在{{lang|ja|月刊アフタヌーン}}推出由{{lang|ja|佐原ミズ}}所作畫的漫畫版；小說版在台灣由[[尖端出版|尖端出版]]代理發行<ref>{{cite web|url=http://www.spp.com.tw/spp2006/all/asp/search/bookfile.asp?bc=25037018 | title=雲之彼端‧約定的地方|publisher=尖端出版|accessdate=2014-01-21 |language=中文}}</ref>。

=== 背景 ===
故事背景設定在以現實世界為基礎所衍生的'''架空世界'''，敘述日本在[[津輕海峽|津輕海峽]]另一側的[[北海道|北海道]]遭到佔領，形成南北兩側對立的舞台。

1945年，[[蘇聯|蘇聯]]背棄日蘇中立條約，於十月攻佔北海道。日本恢復主權後，北海道也改名「蝦夷」，歸入蘇聯的體制之下。1965年[[赫魯晓夫|赫魯晓夫]]在[[蘇共|蘇共]]第二十屆大會上宣佈統合蘇聯、東歐與西亞所有社会主義國家的統一政體「聯邦國」誕生。

1960年代後半，蝦夷內部民族主義運動高漲，為了因應這個情勢，聯邦國於1975年與日本斷交，使日本南北分裂情勢延續至今。

聯邦國在蝦夷上建造有著高度直達雲端、叫做「聯邦國巨塔」的神秘建築物，該神秘巨塔因此造成[[美國|美國]]與聯邦國的軍事衝突點。

== 劇情 ==
住在[[青森|青森]]的中學三年級生「浩紀」與「拓也」，夢想著能夠到「神秘巨塔」所在的海峽對岸去。於是兩個人為了實現夢想而開始製造名為「'''維席拉'''」（{{lang|it|Velaciela}}，音近維拉切拉，意為白色之翼）的飛機。

兩人的同班同學「澤渡佐由理」在因緣牽絆下遇見了浩紀與拓也兩人，並對於他們要在暑假搭乘維席拉到神秘巨塔的計畫非常的寄予厚望，也和浩紀許下一起搭乘維席拉到海峽對岸去的'''約定'''；佐由理因此成為了浩紀與拓也兩人完成夢想不可或缺的女孩。

在此之後，佐由理在沒告知兩人的情況下就突然地轉學並從此下落不明。受到此一打擊的兩人便停止維席拉的計畫；為了忘記曾經存在的夢想，浩紀進入東京的[[高中|高中]]就讀，而拓也則是就讀青森當地的高中。

三年後的1999年，聯邦國與[[美國|美國]]之間的軍事衝突即將一觸即發，神秘巨塔的秘密也日漸趨於明朗化。另一方面，佐由理的行蹤也因神秘巨塔的解密而曝光。佐由理是從中學三年級的夏天開始的三年內，因不明原因所引發的疾病而持續睡眠，並在發病的那年夏天即轉往[[東京|東京]]的[[醫院|醫院]]住院觀察。

在得知神秘巨塔與佐由理有著重大關係的衝擊秘密後，浩紀與拓也兩人便再次完成中學當年所無法實現的夢想。而浩紀在謹記著當時的約定下，載著佐由理飛往那雲之彼端、約定的地方……

== 角色 ==
; {{nihongo|藤澤浩紀|藤沢浩紀（ふじさわ ひろき）|[[配音員|配音員]]：[[吉岡秀隆|吉岡秀隆]]（日本）；[[羅偉傑|羅偉傑]]（香港亞洲影帶）（香港）；[[魏伯勤|魏伯勤]]（台灣）}}
: 本作男主角。中學三年級生，同時也是[[弓道|弓道]]社的社員，有著些許孩子氣般的個性，偷偷暗恋着佐由理。高中時進入東京的學校就讀並寄宿於學生宿舍。會選擇東京的學校就讀是因為佐由理的不告而別，為了不要再見到那曾與佐由理約定的'''神秘巨塔'''所以遠赴東京。受到佐由理當時的影響，高中時代的浩紀也開始學起演奏[[小提琴|小提琴]]，並有不錯的成績。但当他上了高中之后，个性也开始变地内向封闭。在遥远的东京感到无比的寂寞。之后的三年里他经常莫名地进入佐由里的梦境，最终辗转得到了佐由里住院后所写的信才得知她的消息。为了让佐由理能苏醒，他想尽办法让小型飞机能成功起飞，带着佐由里穿越边界飞向约定的地方，最终使佐由理苏醒。
; {{nihongo|白川拓也|白川拓也（しらかわ たくや）|[[配音員|配音員]]：[[萩原聖人|萩原聖人]]（日本）；[[何偉誠|何偉誠]]（香港亞洲影帶）（香港）}}
: 浩紀的同班同學兼好友，滑冰社社員。跟浩紀比較起來顯得有理性、像大人般的個性，但略帶有些許寂寞的氣氛。有物理學方面的才能，高中時代受到岡部的推薦而進入富澤研究室擔任研究聯邦國巨塔的要員之一。
; {{nihongo|澤渡佐由理|沢渡佐由理（さわたり さゆり）|[[配音員|配音員]]：[[南里侑香|南里侑香]]（日本）；[[莎拉|莎拉]]（香港亞洲影帶）（香港）}}
: 本作女主角，有俄羅斯血統，其祖父為艾科森·季柯諾艾，蘇聯工程師。浩紀和拓也寄予思念的同班同學，对藤泽浩纪存有好感。有著鄰家女孩般的美麗外表、開朗活潑的個性中卻帶有一絲害羞，有如夢幻般的美少女。擅長演奏的項目是[[小提琴|小提琴]]。中學三年級的夏天因不明原因時而引發[[嗜睡症|嗜睡症]]，轉往[[東京|東京]]的醫院後便一直持續睡眠；之後便從兩人的面前消失。外表还维持着15岁的模样，最终 藤泽浩纪为了履行与她的约定，在拓也的帮助下，乘坐那架飞机飞往那座塔，而此时少女在梦中也对浩纪强烈的呼唤，最终少女从睡眠中醒了过来，不过她却忘记了最想说的事情。
; {{nihongo|岡部|岡部（おかべ）|[[配音員|配音員]]：[[石塚運昇|石塚運昇]]（日本）}}
: 為了賺取飛機的[[零件|零件]]費用而打工的浩紀和拓也所屬之工廠「蝦夷製作所」的社長；妻子因南北分裂原因而離婚。製作所其真實身分是[[美軍|美軍]]軍武的[[轉包|轉包]][[工廠|工廠]]；與富澤是舊識。
; {{nihongo|富澤常夫|富澤常夫（とみさわ つねお）|[[配音員|配音員]]：[[井上和彦_(聲優)|井上和彦]]（日本）}}
: 屬於青森軍事學院的戰爭時期特殊情報處理研究室的室長，做著神秘巨塔的研究，與岡部是舊識。從前與笠原一起製作飛機。
; {{nihongo|笠原真希|笠原真希（かさはら まき）|[[配音員|配音員]]：[[水野理紗|水野理紗]]（日本）；[[劉惠雲|劉惠雲]]（香港亞洲影帶）、[[廖欣怡|廖欣怡]]（香港得利影視）（香港）}}
: 富澤研究室研究腦科學的年輕研究員。對年紀比她小的拓也有好感。
