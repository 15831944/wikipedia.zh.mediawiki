'''私立正心高級中學'''為一所[[完全中學|完全中學]]，位於中華民國雲林縣斗六市，與同在雲林縣境內的永年中學和文生中學亦為天主教嘉義教區所屬。
兼具國中部與高中部，依照服裝顏色搭配可以明顯辨別[[高中|高中]]與[[國中|國中]]學生。

== 創校背景 ==
西元1961年創校於[[臺灣|臺灣]][[雲林縣|雲林縣]][[斗六市|斗六市]]郊[[虎尾溪|虎尾溪]]畔，當時的[[斗六市|斗六鎮]]鎮長[[吳開興|吳開興]]及鎮公所秘書葉世珍等地方人士向[[天主教|天主教]][[天主教嘉義教區|嘉義教區]]的[[牛會卿|牛會卿]]主教爭取在斗六設校，[[牛會卿|牛會卿]]乃指派[[郭藩|郭藩]]副主教及服務於斗六玫瑰天主堂的方懷正神父等人向[[羅馬教廷|羅馬教廷]]及國外教區募款籌備建校事宜。

第一屆初中一年級計有正、心、誠、意四班共二百五十一人，隔年招收女生兩班，到現在仍維持男女分班，民國57年開始招收高中生。

== 學校命名 ==
正心中學英文校名在創校時牛主教特别把正心託付給耶穌聖心與聖母聖心，所以正心中學的英文校名為 Sacred Hearts High School (有耶穌聖心與聖母聖心兩顆心，所以是Hearts)，選定12月8日聖母無染原罪慶日為正心的校慶日。

== 學校願景 ==
'''正心中學'''致力塑造每位學生「允文允武」，講求分別是「德」、「智」、「體」、「群」、「美」、「聖」等六育之均衡發展，並於第二任校長[[吳友梅|吳友梅]]27年任內新增了許多大型活動。

== 學校編制 ==

===高中部===
各年級有8班普通班，普通班班級名稱以「正、心、誠、意、修、身、齊、家」作為班級識別；另有一音樂班。

===國中部===
各年級有10班普通班，班級名稱為「正、心、誠、意、修、身、齊、家、治、國」作為識別。另有音樂班兩班(音甲、音乙)及美術班兩班(美甲、美乙)，其中音甲班(男生班)音乙(女生班)各分為管樂組與專業組。

因少子化，107學年度入學之普通班縮減為9班。

== 軼事 ==
校園內原有噴水池一座，民國68年時改為[[蔣介石|蔣介石]]戎裝騎馬銅像座，為[[台灣|台灣]]雕塑大師[[謝棟樑|謝棟樑]]作品，因[[921地震|921地震]]時基座震出裂痕，為安全起見將基座拆除，銅像移到男生部游泳池旁草地。
