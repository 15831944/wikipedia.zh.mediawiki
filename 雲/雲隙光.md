'''雲隙光'''（{{lang-en|Crepuscular Ray}}）是一種大氣光學現象，分為兩類：
*從雲霧的邊緣或多塊雲層間隙滲射出的陽光，有時俗稱「耶穌光」
*天空中光暗相間的現象，常見於日出或日落前後，又稱「曙暮暉」（{{lang-en|Crepuscular Ray}}）。這種雲隙光可以在太空或衛星雲圖上看到。

對地面的觀測者而言，只要有[[雲|雲]]遮擋住太陽，就有可能看到此現象<!-------，但最重要的還是水氣與灰塵的條件。因此雲隙光在雲量較多的日子天氣比較常見；晴朗的日子裡，則常發生於[[日落|日落]]。比較好的觀測地點在海濱或濕氣重的山谷地區------->。

== 成因 ==
[[File:富貴角燈塔夕照FUJI9853b.jpg|thumb]][[富貴角|富貴角]]]]
[[File:耶穌光照福隆DSC_6381.jpg|thumb]]]]
[[File:Crepuscular_Rays,_India.JPG|thumb]]經過[[印度|印度]]上空時拍攝到的雲隙光]]]]

*當太陽處於較低角度，天空有較厚的雲塊（多為[[積雲|積雲]]或[[層積雲|層積雲]]）處於[[太陽|太陽]]與觀測者視線中間，太陽被雲塊遮蔽，太陽光從雲後或邊緣散出。對於觀測者而言，猶如雲層自身發光。
*當太陽處於較高角度，天空有較厚的延綿雲層（多為層積雲），陽光經雲隙滲透出來。觀測者會看見從「雲隙」透出「光」。

「曙暮暉」、「反曙暮暉」（{{lang-en|Anti-Crepuscular Ray}}）：
*在日出或日落前後，太陽處於極低仰角。太陽光照射到地面時，受到較高的物件（例如高山或[[直展雲|直展雲]]）遮擋，出現較長的影子，繼而影響天空的亮度，在天空上形成光暗相間的情況。其中，天空較亮部分是有太陽光照射的部分，而較暗的部分就是影子。若影子較長，影子會延伸至太陽位置的相反方向。對於觀測者而言，影子看起來在太陽位置相對的同一位置（[[反日點|反日點]]）聚合。這種背向太陽觀測到的曙暮暉稱為「反曙暮暉」。由於光線於大氣中散射的關係，一般而言，曙暮暉比反曙暮暉來得明亮。

== 文化 ==

太陽仰角較高時照耀地面的雲隙光（耶穌光），常被賦予神聖性，又名「佛祖光」、「上帝之梯」。許多電影、畫作、動漫畫等藝術作品，也常使用耶穌光作為神聖、崇高、救贖的象徵。此外，這種耶穌光也像動漫中不明飛行物體打開艙門時發出的光芒。<ref>[http://www.youtube.com/TJyRm8x_0z8 香港天文台YouTube頻道：氣象冷知識之雲隙光]</ref>

<!-------== 顏色 ==
雲隙光大多是紅色或黃色的，因為在[[日出|日出]]及[[日落|日落]]時的光線通過大氣層比正午太陽的光線次數超過四十倍。通過[[瑞利散射|瑞利散射]]，在空氣中的微粒分散的較短的波長光（綠色和藍色）比較長的紅光和黃光波長多。----->

== 反云隙光 ==
[[File:Anticrepuscular_rays_20050918_in_Ye_Liou.jpg|thumbnail]]
在日落与日出时产生的兩道雲隙光，若其夾角較小，對地面觀測者來說，就好像是兩條光芒從日落的西天射出，輻合於天頂對面的東邊。此現象即為反雲隙光。
