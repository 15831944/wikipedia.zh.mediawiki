'''雲門舞集'''，是一個[[台灣|台灣]]的現代[[舞蹈|舞蹈]]表演團體，於西元1973年由[[林懷民|林懷民]]創辦，也是台灣的第一個職業舞團。雲門之名來自於中國古書《[[呂氏春秋|呂氏春秋]]》中的一句話：「黃帝時，大容作雲門，大卷……」，也就是傳說中[[黃帝|黃帝]]時代舞蹈的名稱。雲門舞集曾推出多個舞蹈作品，當中包括有[[薪傳|薪傳]]，[[九歌_(舞剧)|九歌 (舞剧)]]，[[家族合唱|家族合唱]]，[[流浪者之歌|流浪者之歌]]，[[水月|水月]]，[[竹夢|竹夢]]，[[行草_(舞剧)|行草(舞剧)]]等等。雲門舞集的附團「[[雲門舞集2|雲門舞集2]]」於西元1999年成立，藝術總監為[[羅曼菲|羅曼菲]]。

== 創立與紮根 ==
[[File:Curtain_Call_of_Crossing_the_Ocean,_Legacy_20140712.jpg|thumb]]
[[File:Opening_of_Cursive_it_20140712.jpg|thumb]]
37年來（不包括曾於1988至1991年間暫停運作的三年），雲門的舞台上呈現一百六十多齣舞作。包括古典文學、民間故事、台灣歷史、社會現象的衍化發揮，乃至前衛觀念的嘗試。從[[台北|台北]]的[[國家戲劇院|國家戲劇院]]，到各縣市文化中心、體育館、鄉鎮學校禮堂，雲門在台灣定期與觀眾見面；每年輪流在各城市舉行的戶外演出，平均每場觀眾高達六萬人<ref>{{Cite web|url = http://www.cloudgate.org.tw/news/show/74|title = 屋漏痕台北戶外公演 六萬觀眾熱情抗高溫|accessdate = 2015-06-09|author = |date = 2011-07-26|publisher = }}</ref>。

台灣許多知名的現代舞者均自雲門舞集發跡，如[[林秀偉|林秀偉]]、[[羅曼菲|羅曼菲]]、[[劉紹爐|劉紹爐]]等。這些知名舞者有的也各自創辦了不錯的現代舞劇團，林秀偉創辦了[[太古踏舞團|太古踏舞團]]，劉紹爐創辦了[[光環舞集|光環舞集]]等。雲門舞者多為國內舞蹈科系畢業生，他們的訓練包括現代舞、芭蕾、京劇動作、太極導引、靜坐與拳術。

1998年，雲門創立雲門舞集舞蹈教室，以多年專業經驗創造「生活律動」教材，透過啟發性的教學，認識自己的身體，創造自己的生命律動。1999年5月，雲門在創立26年後成立子團「雲門２」(全名並不是雲門舞集2)，深入台灣各地校園和社區，為更多的觀眾演出。舞團的年度公演「春鬥」，以演出台灣年輕編舞家的作品為主。2000年啟動的藝術駐校活動，已有近2千3百位學生選修。2007年開啟「藝術駐縣活動，舞團進駐城鄉，深耕地方藝術推廣。」此外，從1999年[[九二一大地震|九二一大地震]]到2009年[[八八水災|八八水災]]等，雲門均參與巡迴義演公演募款，以協助災區重建工作。

== 創新風格 ==
林懷民除了身為雲門舞集的創始人和靈感泉源外，也是《蟬》、《變形虹》等短篇小說集作者，並在1960年代密切參與臺灣文學活動。在1970年代誕生的雲門以創新舞蹈和嶄新方向震撼了臺灣文化舞台。林懷民於1972年率領舞團在[[臺北中山堂|臺北中山堂]]舉行的首次公演中<ref name="NINE">{{cite book|title=Cloud Nine:Lin Hwai-Min and Cloud Gate Dance Theater|last=Yang|first=Meng-Yu|date=1998|publisher=Tian xia yuan jian chu ban gu fen you xian gong si|location=Taipei}}</ref>，對新形式展現不懈的好奇心，因此雲門融合了亞洲神話、民俗以及現代美感的動作風格，舞者運用諸如太極導引、冥想、武術、現代舞、芭蕾舞等多種運動方式和藝術手法。

在中國和亞洲的文化背景脈絡中，雲門創造了許多兼具臺灣獨特經驗的舞蹈，如《白蛇傳》改編自中國傳統民間故事；《渡海》舞蹈豐富表現出幾代中國人跨越臺灣海峽來到臺灣；《廖添丁》臺灣傳奇羅賓漢人物，在日本佔領臺灣期間挫敗了暴虐官兵<ref name="NINE" />{{rp|139–142}}； 以及《吳鳳》描述來自中國大陸的官員參與早年平息臺灣原住民的活動<ref name="NINE" />{{rp|111}}。

== 演出評價 ==

雲門也經常應邀赴海外演出，是國際重要藝術節的常客。舞團在台灣及歐、美、亞、澳各洲兩百多個舞台上，演出超過一千七百場，以獨特的創意，精湛的舞技，獲得各地觀眾與舞評家的熱烈讚賞，如中時晚報：「當代台灣最重要的活文化財產」、[[英國|英國]][[倫敦|倫敦]]《[[泰晤士報|泰晤士報]]》評為「[[亞洲|亞洲]]第一當代舞團」、[[德國|德國]]《[[法蘭克福匯報|法蘭克福匯報]]》亦曾贊其為「世界一流現代舞團」。<ref>[http://www.qaf.org.tw/events/2012/how_can_i_live_on_without_you/about.htm 關於「雲門舞集」]</ref>
2003年，《紐約時報》首席舞評家安娜．吉辛珂芙將雲門的《水月》列為該年最佳舞作的首選；為澳洲墨爾本藝術節揭幕的《行草　貳》，榮獲時代評論獎及觀眾票選最佳節目；2004、2005年，《行草　貳》、《狂草》獲台新年度表演藝術獎；2006年，德國《國際舞蹈》與《今日劇場》雜誌邀請歐陸重要舞評家，共同遴選「行草三部曲」為「年度最佳舞作」。<ref>[https://tw.entertainment.appledaily.com/daily/20070203/3230914/ 《行草三部曲》獲評選最佳舞作]</ref>

1999年，林懷民獲頒被稱為「亞洲[[諾貝爾獎|諾貝爾獎]]」的[[菲律賓|菲律賓]]「[[麥格塞塞獎|麥格塞塞獎]]」「新聞、文學與創造性溝通藝術獎」。

2003年，[[台北市政府|台北市政府]]將雲門三十週年特別公演的首演日8月21日，訂定為「雲門日」，並將雲門辦公室所在地的復興北路231巷定名為「雲門巷」，「肯定並感謝雲門舞集三十年來為台北帶來的感動與榮耀。
2012年，香港南華早報，更稱以九歌這齣輝煌的長篇，將林懷民這位亞洲的巨人，提升到瑪莎‧葛蘭姆、模斯‧康寧漢等少數人所佔有的層級：二十世紀偉大編舞家之一。

[[國立中央大學|國立中央大學]][[鹿林天文台|鹿林天文台]]在2007年7月25日發現的[[小行星200025|小行星200025]]命名為雲門（Cloud Gate）<ref>[http://news.rti.org.tw/index_newsContent.aspx?nid=240836 雲門小行星 宇宙發光]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>雲門小行星 表彰林懷民</ref><ref>[http://www.epochtimes.com/b5/10/5/4/n2896803.htm 第一顆以世界一流台灣舞團命名之小行星]</ref><ref>[http://tw.nextmedia.com/applenews/article/art_id/32475975/IssueID/20100430 小行星命名雲門 林懷民分享榮耀]</ref>。

== 火焚與重生 ==
[[File:Cloud_Gate_Theater_Tamsui_02.jpg|thumb]]
2008年2月11日凌晨時分，雲門在臺北縣八里鄉（今 [[新北市|新北市]][[八里區|八里區]]）的排練場發生火災，演出道具及歷年資料幾乎焚毀殆盡<ref>[http://www.nownews.com/2008/02/11/10844-2229517.htm 東森新聞報：雲門舞集倉庫大火　道具、資料付之一炬損失慘重]</ref>。台北縣政府（今 [[新北市政府|新北市政府]]）採「[[ROT|ROT]]」加上「[[BOT|BOT]]」模式，引用「促進民間參與公共建設法」，耗資3億多元，協助雲門舞集，進駐淡水鎮（今 [[淡水區|淡水區]]）[[中央廣播電台|中央廣播電台]]舊址，並興建[[淡水文化藝術教育中心|淡水文化藝術教育中心]]。<ref>[http://www.libertytimes.com.tw/2009/new/apr/14/today-taipei4.htm 雲門舞集重生 進駐淡水央廣舊址] {{webarchive|url=https://web.archive.org/web/20160403213538/http://www.libertytimes.com.tw/2009/new/apr/14/today-taipei4.htm |date=2016-04-03 }} 自由電子報</ref>

== 相關記錄 ==
林懷民與雲門的故事，已於1998年由天下文化公司出版楊孟瑜所撰《飆舞：林懷民與雲門傳奇》(ISBN 9576214971)一書。[[張照堂|張照堂]]監製的《踊舞．踏歌　雲門30》紀錄片，已由公共電視製作發行。多齣雲門作品已拍攝為舞蹈影片問世。在荷蘭攝製的《流浪者之歌》，在法國攝製的《水月》，在德國拍攝的《竹夢》以及在瑞士拍攝的《行草　貳》，為多國電視台播放，並製成DVD發行全球。《雲門‧傳奇》舞作套裝DVD、《行草三部曲》、《紅樓夢》、《九歌》、《屋漏痕》由金革發行。
