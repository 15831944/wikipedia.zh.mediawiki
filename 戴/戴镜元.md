'''戴镜元'''（{{bd|1919年||2008年|4月3日}}）[[福建省|福建省]][[永定县|永定县]]人，[[中国共产党|中国共产党]]及[[中国人民解放军|中国人民解放军]]官员，中国共产党领导的军队中技侦情报工作的创始人之一及早期领导人<ref name=ydxq>{{Cite web|url =http://www.ydxq.cn/right.asp?bookid=1&juanid=111&zhangid=773|title =永定县志·续志（1988～2000） 第二章 人物简介 第二节 军界人物|author =|date =|publisher =永定方志网|language =|accessdate =2017-05-24|quote =}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==生平==
===第一次国共内战===
戴镜元生于民国八年（1919年），是永定县[[岐岭乡|岐岭乡]]竹联村人。民国十七年（1928年）5月，加入[[中国共青团|中国共青团]]。同年，参加永定农民暴动，并历任乡儿童团团长、区儿童团团长、共青团金丰区委书记<ref name=ydxq/><ref name=zgw/>。民国十八年（1929年）5月，[[红四军|红四军]]第二次入闽时，转为[[中国共产党|中国共产党]]党员<ref name=zgw/><ref name=mao>{{Cite web |url =http://dangshi.people.com.cn/n/2014/0812/c85037-25452375.html  |title = 毛泽东嘉奖解读“天书”的戴镜元 |author = 苏俊才 |date = |publisher =福建党史月刊2014年4月（上） |language =  |accessdate =  |quote =  }}</ref><ref name=ydxq/>。任共青团永定县委书记。民国二十年（1931年）12月，调任共青团[[龙岩县|龙岩县]]委书记。民国二十一年（1932年）春，调任共青团[[连城县|连城县]]委书记。民国二十二年（1933年）春，参加[[中国工农红军|中国工农红军]]<ref name=ydxq/>。1933年3月，戴镜元到[[中华苏维埃共和国中央革命军事委员会|中革军委]]二局工作，从此成为中国共产党领导的军队的技侦情报工作的创始人之一<ref name=mao/>。民国二十三年（1934年）10月，在中革军委电台当译电员的戴镜元随部队参加[[长征|长征]]。当时，中革军委代号“红星纵队”，电台代号“第四分队”，分队队长是[[曾希圣|曾希圣]]，副队长是[[钱壮飞|钱壮飞]]，党支部书记是戴镜元。在长征中，电台担负通信联络以及侦听敌方情报的任务，受到中共中央的关注<ref name=ydxq/><ref name=mao/>。在中革军委二局，戴镜元历任中革军委二局党支部书记、研究员、参谋、股长<ref name=zgw/>。

民国二十五年（1936年）2月，依照[[中共中央|中共中央]]决定，戴镜元随同[[李克农|李克农]]（中共代表团团长）、[[钱之光|钱之光]]到[[洛川|洛川]]和[[东北军|东北军]][[张学良|张学良]]、[[王以哲|王以哲]]会谈。同年4月，随[[周恩来|周恩来]]、李克农在[[延安|延安]]和张学良再度会谈，仍然负责机要工作。这两次会谈为日后[[第二次国共合作|第二次国共合作]]的实现、[[抗日民族统一战线|抗日民族统一战线]]的建立奠定了基础<ref name=ydxq/>。

===抗日战争===
[[抗日战争|抗日战争]]时期，戴镜元历任[[中国共产党中央军事委员会|中央军委]]二局副科长、科长、处长、系主任、第一副局长、代局长，二局党总支书记。参加了[[平型关战斗|平型关战斗]]等战役战斗的情报保障任务。民国三十四年（1945年）春，出席[[中共七大|中共七大]]<ref name=ydxq/><ref name=zgw/>。

抗日战争全面爆发后，遵照中共中央、中央军委的指示，中央军委二局自1938年夏开始破译国外密电。该局先后自外训班调来一批人，在戴镜元领导下，经一年多努力，在1939年7月1日首次将破译所获成果报告中共中央、中央军委、总参谋部。1939年7月7日，中共中央、中央军委联名向戴镜元等人发来一封嘉奖信，并特派参谋长[[滕代远|滕代远]]前来慰勉。这是中共中央、中央军委第一次联名写信嘉奖一个工作单位。在奖给戴镜元的大日记本上，有毛泽东1939年7月9日的题词：“步步前进，就步步胜利！”<ref name=mao/>

1939年夏，戴镜元等人破译了[[日军|日军]]一套密电码，从而使中央军委得以掌握日军情报。同年11月7日，[[晋察冀军区|晋察冀军区]]第一分区[[杨成武|杨成武]]所部在[[黄土岭战斗|黄土岭战斗]]中击毙[[阿部规秀|阿部规秀]]。战斗结束后，杨成武把缴获的阿部规秀的军大衣从前方送至[[延安|延安]]，毛泽东和中央军委决定将其奖给戴镜元<ref name=mao/>。

1939年秋，毛泽东坐车自延安到真武洞参加[[中共中央西北局|中共中央西北局]]的一个会议。途经中央军委二局驻地时，因汽车陷入河中，毛泽东决定在中央军委二局驻地过夜。当晚毛泽东与戴镜元谈话。次日上午，毛泽东看望了中央军委二局全体同志<ref name=mao/>。

===第二次国共内战===
[[第二次国共内战|第二次国共内战]]时期，戴镜元任中央军委二局代局长、局长兼政治委员、二局党委书记，[[中央后方委员会|中央后方委员会]]委员。参加了[[中原突围|中原突围]]、[[挺进中原|进军大别山]]、[[辽沈战役|辽沈战役]]、[[淮海战役|淮海战役]]、[[平津战役|平津战役]]、[[渡江战役|渡江战役]]、[[西南战役|进军西南]]等战役战斗的情报保障任务<ref name=ydxq/><ref name=zgw/>。

1945年冬，[[东北民主联军|东北民主联军]]发起[[本溪战役|本溪保卫战]]。本溪战役结束后，戴镜元将对此次战役的意见写成《本溪战役之管见》，汇报给毛泽东，他认为未集中优势兵力全歼敌人是个教训。毛泽东在报告第一页批示：“很有见地。”<ref name=mao/>

1946年6月，[[中国国民党|中国国民党]]军队围攻中国共产党方面的[[中原军区|中原军区]]，全面内战爆发。当时，中原军区部队按照中共中央指示实施[[中原突围|中原突围]]。1946年7月3日，中央军委二局代局长、党总支书记戴镜元致信毛泽东，汇报了中原突围前后的敌军动态以及该局的工作。1946年7月5日，毛泽东亲自复信嘉勉<ref name=mao/>。

1947年9月，在毛泽东关怀下，周恩来、[[任弼时|任弼时]]召集机要、通信、情报工作负责人到[[陕北|陕北]][[佳县|佳县]]神泉堡汇报并研究工作。1947年12月中旬，戴镜元准备回晋绥地区[[临县|临县]]三交镇驻地时，毛泽东、周恩来特来看望<ref name=mao/>。

1949年初，中共中央准备在[[西柏坡|西柏坡]]举行[[中共七届二中全会|中共七届二中全会]]。此时，戴镜元等人也在开工作会议。1949年2月28日，毛泽东、周恩来等领导来看望大家<ref name=mao/>。

1949年4月，戴镜元出席[[中国新民主主义青年团|中国新民主主义青年团]]第一次全国代表大会，当选为中国新民主主义青年团中央委员。同年9月，作为[[中国人民政治协商会议第一届全体会议代表列表|候补代表]]出席[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]<ref name=ydxq/><ref name=zgw/>。

===共和国时期===
[[中华人民共和国|中华人民共和国]]成立后，戴镜元历任[[中央人民政府人民革命军事委员会|中央军委]][[中国人民解放军总参谋部情报部|总情报部]]副部长、[[中国人民解放军总参谋部技术侦察部|中央军委技术部]]部长、中共中央机要局副局长<ref name=ydxq/><ref name=zgw/>。

1950年春，戴镜元所在单位召开工作会议。会前，戴镜元向毛泽东、周恩来、[[朱德|朱德]]进行了综合报告。毛泽东批示：“已阅，很好。应予奖励。”工作会议召开前两天，毛泽东要戴镜元来[[中南海|中南海]]参加一个晚会，中间休息时，戴镜元向毛泽东汇报了会议准备情况，并请毛泽东接见与会人员。1950年4月9日，毛泽东接见了参加工作会议的人员。1950年12月29日，毛泽东又给戴镜元等人题词：“你们的工作很有成绩，配合了正义战争的胜利。希望你们继续努力，为争取完全的胜利而奋斗。”<ref name=mao/>

在营房建设中，戴镜元擅自挪用了进口[[楠木|楠木]]为自己装修住房。在“[[三反五反|三反五反]]”运动中，此事被人揭发。材料上报后，有人建议严惩戴镜元。按当时的惩办条件，该案可以移送人民法院追究刑事责任。[[朱德|朱德]]调阅案卷后，亲自找戴镜元谈话核实情况。鉴于戴镜元的贪污数额处于违纪和判刑之间，且戴镜元属初犯，认错态度很诚恳，按照中国共产党“惩前毖后、治病救人”的方针，朱德与[[中共中央纪律检查委员会|中共中央纪律检查委员会]]的人员研究后决定，对戴镜元作出留党察看的纪律处分，以及免去职务的组织处理，收回其住房改为公用，同时取消其1955年[[中国人民解放军|中国人民解放军]]授衔参评资格，下放到[[北京市|北京市]][[崇文区|崇文区]]一家工厂[[劳动改造|劳动改造]]<ref name=zhude>{{Cite web |url = http://news.sina.com.cn/c/nd/2016-07-22/doc-ifxuhukz0829439.shtml |title =朱德外孙批“军老虎”:谷俊山对徐才厚唯唯诺诺  |author =  |date = 2016-07-22 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。后来，戴镜元出任中共北京市[[东城区_(北京市)|东城区]]委书记处书记、区长等职<ref name=zgw/>。

[[文革|文革]]后期，[[中共中央军委|中共中央军委]]恢复了戴镜元的职务和工作<ref name=zhude/>。戴镜元任[[中国人民解放军总参谋部技术侦察部|中国人民解放军总参谋部第三部]]部长、中国共产党中国人民解放军总参谋部第三部委员会书记。他还先后当选为[[中共十一大|中共十一大]]、[[中共十二大|中共十二大]]代表<ref name=ydxq/>。离休后，为副大军区职离休干部<ref name=zgw/>。

2008年4月3日，戴镜元在北京病逝，享年89岁<ref name=zgw>{{Cite web |url = http://www.china.com.cn/policy/txt/2008-06/19/content_15856628.htm |title =总参谋部某部原部长戴镜元同志逝世  |author =  |date =2008-06-19  |publisher = 中国网 |language =  |accessdate =  |quote =  }}</ref>。
