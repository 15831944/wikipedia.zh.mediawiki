'''戴葆庭'''（{{bd|1895年|8月8日|1976年|}}），号足斋，[[浙江|浙江]][[绍兴|绍兴]]人。中国钱币收藏家、钱币学家、古钱商人。

== 生平 ==
光绪二十一年六月十八日<ref>{{cite book |author1=王荫嘉等 |title=寿泉集拓甲编 |date=1940-10 |publisher=寿泉会 |location=上海 |chapter=寿泉会齿录}}</ref>（1895年8月8日），戴葆庭出生于[[浙江省_(清)|浙江省]][[宁绍台道|宁绍台道]][[绍兴府|绍兴府]][[山阴县_(浙江省)|山阴县]]兰亭陈家葑（今属浙江省[[绍兴市|绍兴市]][[柯桥区|柯桥区]][[兰亭街道|兰亭街道]]）。戴家祖居陈家葑，世代务农，戴葆庭父亲戴炳荣是[[扇子|扇]]庄的画工。由于家境贫寒，戴葆庭早年仅读过两年[[私塾|私塾]]，11岁时就到米店学徒<ref name="绍兴市志">{{cite book |author1=绍兴市地方志编纂委员会 |title=绍兴市志 |date=1997-04 |publisher=浙江人民出版社 |chapter=卷44 人物 第一章 人物传}}</ref>。

戴葆庭自幼酷爱收藏钱币，1920年辞去米店工作，开始经营古钱买卖{{efn|name=寿泉集拓|《[[寿泉集拓|寿泉集拓]]戊集》戴葆庭自序“因得朋友之信用。民国九年，董君弢厂邀余同搜古泉，是为正式集泉之始。余尤不畏艰苦，好涉远程……频年所获，珍品奇泉，止前千百，半归齐斋。故奇斋所藏首选，多为余所让渡<ref name="草根"/>。”}}<ref name="草根">{{cite web |title=草根出身的钱币学家——戴葆庭(组图) |url=http://news.163.com/15/0325/10/ALHV3IUD00014AED.html |website=网易-东方早报}}</ref>。戴葆庭最早是一名下乡收货的“地皮客”，常年挑着担子串街走巷的收购古钱，经常露宿在外，由于读过两年私塾，他就带着书籍和古钱图样去收货，边干边学，学以致用。他收货的足迹遍及大半个中国，曾在[[北京|北京]]、[[天津|天津]]、[[江苏|江苏]]、[[浙江|浙江]]、[[江西|江西]]、[[河南|河南]]、[[陕西|陕西]]、[[山东|山东]]、[[广东|广东]]等省市收购古钱币。也因此，戴葆庭曾收购到很多的珍稀钱币，如1925年戴葆庭在江西[[鄱阳县|鄱阳]]购得一枚“[[大齐通宝|大齐通宝]]”，因为原是小孩[[毽子|毽子]]上的配重，钱身钻有四个孔，被称为“四眼大齐”，回到上海后，戴葆庭将其以500块[[银元|大洋]]转让钱币收藏家[[张叔驯|张叔驯]]{{efn|戴葆庭、戴葆湘兄弟曾是张叔驯的主要供货商<ref name="草根"/>。}}，张因此自号其居曰“齐斋”<ref name="草根"/>。

戴葆庭曾在上海市[[江西中路|江西中路]]67号的[[中国古物商场|中国古物商场]]开设“粹源古玩店”，1945年10月28日改为“戴葆记”。1940年，戴葆庭在[[福州路|福州路]]杏花楼旁边、大新街口开了一家“源昌钱庄”，专门买卖稀有的古钱<ref name="马传德回忆">{{cite journal |author1=马传德 |title=回忆戴葆庭先生 |journal=中国钱币 |date=2015 |issue=1 |pages=71-75}}</ref>。1949年，[[中华人民共和国|中华人民共和国]]建立后，在上海[[广东路_(上海)|广东路]]的上海市古玩市场（后改称[[上海文物商店|上海文物商店]]）工作。曾为[[上海博物馆|上海博物馆]]、[[中国历史博物馆|中国历史博物馆]]等单位鉴定钱币，提供文物定级意见。1966年退休<ref name="四十周年" />。

== 泉界活动及著作 ==
1940年，戴葆庭参与发起成立[[中国泉币学社|中国泉币学社]]，负责日常工作，担任会计职务<ref name="马传德回忆" />。学社创办双月刊《[[泉币_(杂志)|泉币]]》杂志，前后共出刊32期，戴葆庭是学社评议员，曾在《泉币》上发表了多篇文章<ref name="马传德回忆" />。

1940年10月，戴葆庭与[[丁福保|丁福保]]、[[张絅伯|张絅伯]]、[[张季量|张季量]]、[[郑家相|郑家相]]、[[陶庭耀|陶庭耀]]、[[王荫嘉|王荫嘉]]、[[陈亮声|陈亮声]]、[[蔡季襄|蔡季襄]]、[[罗伯昭|罗伯昭]]等十人组成“[[寿泉会|寿泉会]]”，约定每逢会员生辰，会员各出泉拓三种集为“[[寿泉集拓|寿泉集拓]]”，赠给寿翁和其余会员留念<ref>{{cite web |title=寿泉集拓十编 附二种 |url=http://www.xlysauc.com/auction5_det.php?id=117106&ccid=848&n=1112 |website=西冷印社}}</ref>。

1930年代有《足斋泉拓》3册传世。二十世纪五六十年代，戴葆庭曾为教育学生编写一部钱币学讲义，在[[文化大革命|文化大革命]]期间散失。晚年曾花费大量精力撰写自传，但在文化大革命初期亲手焚毁<ref name="戴志强" />。晚年整理集成的《足斋泉拓集珍》在1990年以《戴葆庭集拓中外钱币珍品》为名由[[中华书局|中华书局]]出版<ref>{{cite book |author1=戴葆庭 辑 |coauthors=戴志强,沈鸣镝 整理 |title=戴葆庭集拓中外钱币珍品 |date=1990-4-1 |publisher=中华书局 |isbn=9787101006841}}</ref>。辑著的《珍泉集拓》于1991年由[[新华出版社|新华出版社]]出版<ref>{{cite book |author1=戴葆庭辑著 |title=珍泉集拓 |date=1991-08 |publisher=新华出版社 |location=北京}}</ref>。

== 收藏 ==
戴葆庭的藏品在1949年之后，相继捐赠给[[中国历史博物馆|中国历史博物馆]]<ref>{{cite web |author1=藏品保管一部 |title=中国国家博物馆古代藏品征集与捐献 |url=http://www.chnmuseum.cn/portals/0/web/zt/100n/article_atc-1.html?id=17 |website=中国国家博物馆}}</ref>、[[上海博物馆|上海博物馆]]、[[天津博物馆|天津市历史博物馆]]等单位。其中1959年将119枚太平天国珍稀钱币捐赠中国历史博物馆，受到[[中华人民共和国文化部|中华人民共和国文化部]]的嘉奖<ref name="戴志强" />。

== 家庭 ==
戴葆庭妻子沈燕三也是绍兴人，曾随戴葆庭到各地蒐集古钱，精于钱币墨拓。胞弟戴葆湘，也从事货币收藏，精于钱币鉴定<ref name="绍兴市志" />。

长子[[戴志强|戴志强]]、次子[[沈鸣镝|沈鸣镝]]（戴志茂<ref name="绍兴市志" />）<ref name="四十周年">{{cite web |title=传承匠心精神 弘扬钱币文化——“纪念中国著名钱币学家戴葆庭先生逝世四十周年纪念活动”在绍举行 |url=http://www.cjiyou.net/html/2016-04/419855.htm |website=中国集币在线}}</ref>均从事钱币学研究。戴志强1944年出生于[[上海|上海]]，曾任中国钱币博物馆馆长<ref>{{cite journal |author1=戴莹 |title=平生只言古泉币 不顾现世富贵钞——访钱币鉴定专家戴志强 |journal=收藏界 |date=2007 |issue=6 |pages=99-101}}</ref>。

== 轶事 ==
二十世纪三四十年代，戴葆庭与北京泉商[[骆泽民|骆泽民]]曾被钱币学界誉为“南戴北骆”<ref name="戴志强" />。

== 纪念 ==
戴葆庭墓位于浙江省绍兴市绍兴县兰亭镇（今浙江省绍兴市柯桥区兰亭街道）娄宫村横河里八宝山头，于1998年被列为绍兴县文物保护单位<ref name="戴志强">{{cite journal |author1=戴志强 |title=著名钱币学家戴葆庭 |journal=东方收藏 |date=2010 |issue=1 |pages=14-17}}</ref>。
