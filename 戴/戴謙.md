'''戴謙'''{{BD|1949年|8月20日|||catIdx=D戴}}，出生於[[台南市|台南市]]。有養[[鴨|鴨]]博士之別稱<ref>{{cite news |language = zh-tw |author = 呂雪彗 |coauthors = 黃文博 |url = https://www.chinatimes.com/realtimenews/20171029000012-260410?chdtv |title = 獨家》「養鴨博士」也能賣油 中油董事長竟然是他 |publisher =  [[工商時報|工商時報]] |pages =  |date = 2017-10-29 |accessdate = 2019-04-21}}</ref>，為台灣[[畜牧|畜牧]]學者，曾任[[臺灣大學|臺灣大學]]兼任教授、[[成功大學|成功大學]]生物科技研究所教授兼所長、[[南台科技大學|南台科技大學]]校長。經歷：前[[畜產試驗所|畜產試驗所]]所長、前[[南科管理局|南科管理局]]局長、前國科會副主委、前[[台灣中油|中油]]董事長等職。<ref>{{cite news |language = zh-tw |author = 調查組 |coauthors =  |url = https://www.nextmag.com.tw/realtimenews/news/463582 |title = 
〈直擊〉中油董座戴謙畢業宴　王金平神祕現身 |publisher = 壹週刊 |pages =  |date = 2019-03-05 |accessdate = 2019-04-21}}</ref>

== 學歷 ==
*[[國立台南一中|國立台南一中]]
*[[國立臺灣大學|國立臺灣大學]]畜牧系學士（1968年－1972年）
*國立臺灣大學畜牧研究所碩士（1974年－1976年）
*美國[[加利福尼亞大學戴維斯分校|加利福尼亞大學戴維斯分校]]遺傳學博士（1980年－1984年）
*1972年[[公務人員高等考試|公務人員高等考試]]建設人員畜牧科優等及格

== 經歷 ==
*[[臺灣省畜產試驗所|臺灣省畜產試驗所]]宜蘭種畜繁殖場技正（1976年8月－1981年1月）
*臺灣省畜產試驗所宜蘭分所畜產系主任（1981年1月－1984年3月）
*臺灣省畜產試驗所宜蘭分所所長（1984年3月－1985年3月）
*臺灣省畜產試驗所育種系主任（1985年3月－1989年3月）
*臺灣省畜產試驗所所長（1989年3月－1996年3月）
*國立臺灣大學兼任教授（1990年8月－1998年7月）
*[[臺灣省政府農林廳|臺灣省政府農林廳]]副廳長（1996年12月－1998年10月）
*[[國立成功大學|國立成功大學]]生物科技研究所教授兼所長（1998年8月－2000年1月）
*國立成功大學生物科技研究所教授（2001年1月－2006年7月）
*南科籌備處主任（2000年1月－2003年1月）
*[[南科管理局|南科管理局]]局長（2003年1月－2006年7月）
*[[行政院國家科學委員會|行政院國家科學委員會]]副主任委員（2006年7月－2007年8月）
*[[南台科技大學|南台科技大學]]校長（2007年8月－2017年11月）
*[[台灣中油公司|台灣中油公司]]董事長（2017年11月－2019年3月）
