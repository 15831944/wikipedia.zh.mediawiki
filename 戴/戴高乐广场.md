'''戴高乐广场'''（{{lang-fr|Place Charles de Gaulle}}）是[[法国|法国]][[巴黎|巴黎]]市中心主要[[广场|广场]]之一，也是[[巴黎凯旋门|巴黎凯旋门]]的所在地，始建于1892年，1899年落成。原名'''星形广场'''（{{lang|fr|Place de l'Étoile}}），1918年为纪念[[一战|一战]]胜利而改名为'''胜利广场'''；[[二戰|二戰]]法國遭[[納粹德國|納粹德國]]占領期間，被改名为[[贝当|贝当]]广场。1970年11月13日，為紀念逝世不久的前法國總統[[夏爾·戴高樂|夏爾·戴高樂]]為國家做出的巨大贡献而改为现名。

== 概覽 ==
戴高乐广场位于[[塞纳河|塞纳河]]以北，是十二条主要道路的交汇点，其中最著名的就是向东南延伸的[[香榭丽舍大街|香榭丽舍大街]]。这十二条大街按照顺时针从北方依次为：

# [[瓦格兰大街|瓦格兰大街]]（{{lang|fr|Avenue de Wagram}}） 
# [[奥什大街|奥什大街]]（{{lang|fr|Avenue Hoche}}）
# [[弗里德兰大街|弗里德兰大街]]（{{lang|fr|Avenue de Friedland}}）
# [[香榭丽舍大街|香榭丽舍大街]]（{{lang|fr|Avenue des Champs-Élysées}}）
# [[马索大街|马索大街]]（{{lang|fr|Avenue Marceau}}）
# [[耶拿大街|耶拿大街]]（{{lang|fr|Avenue d'Iéna}}）
# [[克勒贝尔大街|克勒贝尔大街]]（{{lang|fr|Avenue Kléber}}）
# [[雨果大街|雨果大街]]（{{lang|fr|Avenue Victor-Hugo}}）
# [[福煦大街|福煦大街]]（{{lang|fr|Avenue Foch}}）
# [[大军团大街|大军团大街]]（{{lang|fr|Avenue de la Grande-Armée}}）
# {{link-fr|卡諾大街 (巴黎)|Avenue Carnot (Paris)|卡諾大街}}（{{lang|fr|Avenue Carnot}}）
# [[麦马汉大街|麦马汉大街]]（{{lang|fr|Avenue Mac Mahon}}）

2005年，冒着极大的争议，巴黎市政当局对戴高乐广场进行了整修。
