'''戴明盟'''（{{bd|1971年|8月||}}），[[重庆市|重庆市]][[江津区|江津区]]人，[[中国人民解放军海军|中国人民解放军海军]]大校，[[中华人民共和国|中华人民共和国]]第一个驾机完成阻拦着舰和滑跃起飞的飞行员<ref name=pp>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1380348  |title = “中国舰载机首飞第一人”戴明盟任某舰载航空兵部队部队长 |author =  |date = 2015-09-29 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref><ref name=zgjw>{{Cite web |url =  http://www.81.cn/2016hjcllqzn/2016-04/21/content_7017007.htm|title = “航母战斗机英雄试飞员”戴明盟 |author =  |date = 2016-04-21 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
戴明盟出生于[[重庆市|重庆市]][[江津区|江津区]][[石门镇_(重庆市)|石门镇]]，高中在江津第五中学就读<ref>{{Cite web |url = http://news.cqjjnet.com/html/2012-11/28/content_21907681.htm |title = 戴明盟：江津五中走出的舰载机第一人 |author =  |date =  2012-11-28|publisher = 江津网 |language =  |accessdate =  |quote =  }}</ref>。1990年8月，高中毕业后的戴明盟参加[[中国人民解放军|中国人民解放军]]<ref name=zgjw/>。进入[[保定|保定]]的飞行学院学习，毕业后加入“[[海空雄鹰团|海空雄鹰团]]”<ref>{{Cite web |url =http://cq.people.com.cn/n/2015/1010/c367698-26721200.html  |title = 戴明盟：刀尖上的领舞者 |author =  |date = 2015-10-10 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。1994年9月加入[[中国共产党|中国共产党]]<ref name=zgjw/>。戴明盟是中国第一批舰载战斗机试飞员，曾在[[中国人民解放军海军舰载机综合试验训练基地|中国人民解放军海军舰载机综合试验训练基地]]接受训练，创下中国第一个完成地面高速滑跑阻拦，第一个完成飞行阻拦着陆，第一个完成绕舰飞行、触舰复飞等纪录<ref name=pp/>。2012年11月，戴明盟第一个驾驶[[歼-15战斗机|歼-15舰载战斗机]]在[[辽宁舰|辽宁舰]]上实施阻拦着舰和滑跃起飞，实现中国舰载战斗机自“岸基”到“舰基”的突破<ref name=zgjw/>。2014年8月，中央军委授予海军某舰载航空兵部队副部队长戴明盟“航母战斗机英雄试飞员”称号<ref>{{Cite web |url =  http://mil.news.sina.com.cn/2014-08-28/0744797923.html |author =|title =  习近平授予戴明盟航母战斗机英雄试飞员荣誉称号 |date = 2014-08-28 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref><ref name=zgjw/>。2014年9月1日，在北京举行了中央军委授予戴明盟“航母战斗机英雄试飞员”荣誉称号命名大会，中央军委委员、海军司令员[[吴胜利|吴胜利]]宣读了中央军委主席[[习近平|习近平]]签署的命令，并向戴明盟颁发勋章及证书。2015年，戴明盟出任[[中国人民解放军海军舰载航空兵部队|中国人民解放军海军舰载航空兵部队]]部队长。2015年9月3日，在[[纪念中国人民抗日战争暨世界反法西斯战争胜利70周年大会|纪念中国人民抗日战争暨世界反法西斯战争胜利70周年大会]]阅兵中，戴明盟率领5架歼-15舰载战斗机组成“V”字形梯队参加，这是中国人民解放军海军固定翼舰载机首次参加大阅兵<ref name=pp/>。为此，戴明盟被“9·3”阅兵联合指挥部表彰为阅兵优秀指挥员<ref name=zgjw/>。截至2016年，戴明盟曾立二等功、三等功各3次，曾获中航工业集团“航空报国金奖”、国务院和中央军委联合授予的“航母工程建设重大贡献奖”，曾被评为全军爱军精武标兵、2014年第十届“海军十杰青年”、海军功勋飞行员<ref name=zgjw/>。

2018年5月，戴明盟从[[中国人民解放军海军舰载航空兵部队|中国人民解放军海军舰载航空兵部队]]部队长离职，出任[[中国人民解放军海军舰载机综合试验训练基地|中国人民解放军海军舰载机综合试验训练基地]]司令员。
