'''戴季陶'''（{{bd|1891年|1月6日|1949年|2月11日|catIdx=D戴}}），[[原名|原名]]'''良弼'''，[[表字|字]]'''選堂'''，[[號|號]]'''天仇'''，后改名'''傳賢'''、[[表字|字]]'''季陶'''，通稱'''戴季陶'''，[[中华民國|中华民國]][[政治家|政治家]]、[[中國國民黨|中國國民黨]]元老之一，[[中華民國國旗歌|中華民國國旗歌]]的作詞者<ref>關於國旗歌的作詞者尚有爭議，詳見[[中華民國國旗歌#歷史|中華民國國旗歌#歷史]]</ref>，也是[[中国|中国]][[马克思主义|马克思主义]]最早的研究者之一。他在[[訓政時期|訓政時期]]任第一任[[考試院院長|考試院院長]]長達20年，也是歷史上最年輕的五院院長（就任時年僅37歲）。

== 早年經歷 ==
戴季陶[[祖籍|祖籍]][[浙江|浙江]][[湖州|湖州]][[烏程|烏程]]戴山（今屬[[吳興區|吳興區]][[八里店鎮|八里店鎮]]），生於[[四川|四川]][[廣漢|廣漢]]，其高祖戴敏勤入川经商，定居[[汉州|汉州]]，在中西街开碗铺，世继其业，其父戴小轩兼业[[中医|中医]][[外科|外科]]。

1905年，到[[日本|日本]]就讀[[东京高等師範學校|东京高等師範學校]]。1906年4月，在[[東京|東京]]認識東渡[[日本|日本]]就讀東京清華學校的[[蔣介石|蔣介石]]。1907年轉讀[[日本大學|日本大學]][[法律|法律]]系。

1909年返[[華|華]]，並於1911年加入[[同盟會|同盟會]]，屢於[[報章|報章]]批評[[滿清|滿清]][[朝廷|朝廷]]。1911年加入[[同盟会|同盟会]][[辛亥革命|辛亥革命]]时参加[[陈其美|陈其美]]、[[钮永建|钮永建]]的[[上海|上海]]起义，1912年擔任[[孫中山|孫中山]]的[[秘書|秘書]]。

1912年主持[[上海|上海]]《民權報》进行二次[[革命|革命]][[军事|军事]]联络活动，曾發出警告稱：[[蒙古|蒙古]]去，而[[中華民國|中華民國]]亦隨之去矣。

1913年[[二次革命|二次革命]]失败，逃亡日本，戴季陶與[[蔣介石|蔣介石]]是[[結拜|結拜]]好友，兩人在日本同賃一室。與此同時，戴季陶有一位日本籍[[護士|護士]]女友重松金子，重松金子與戴、蔣二人均「過從甚密」，爾後重松金子甚至未婚替戴季陶產下一子。經過跟蔣介石的協調，決定[[過繼|過繼]]給蔣介石，即日後之[[蔣緯國|蔣緯國]]。經蔣家[[王采玉|王太夫人]]斟酌後，[[兼祧|兼祧]]蔣介石與蔣介石之[[胞弟|胞弟]][[蔣瑞青|蔣瑞青]]，並決定由蔣介石[[側室|側室]][[姚冶誠|姚冶誠]]撫養蔣緯國<ref>曾文述：〈[http://books.sina.com/bg/funny/talent/20140422/232949803.html]〉，新浪博客</ref>，據[[李敖|李敖]][[考證|考證]]，蔣中正原有一個幼弟蔣瑞青，早年過世，無子，[[王采玉|王太夫人]]不忍蔣瑞青絕後，著蔣介石收養[[蔣緯國|蔣緯國]]，爾後由緯國奉祀瑞青[[神主|神主]][[香火|香火]]<ref>李玉玲：〈[http://mag.udn.com/mag/people/storypage.jsp?f_ART_ID=392443 李敖：據蔣介石日記考證 蔣緯國不是蔣公之子]〉，台北：《聯合報》，1995年1月2日</ref>。

1916年返[[中國|中國]]，在[[上海|上海]]创办《星期评论》[[周刊|周刊]]，1917年任[[护法军政府|护法军政府]]法制委员会委员长兼[[大元帅|大元帅]]府秘书长；1918年4月代理外交次长。

== 中年經歷 ==
1924年1月，出席[[中國國民黨第一次全國代表大會|中国国民党一大]]，当选为中央执行委员、常务委员，任中央宣传部部长。5月，[[黃埔軍校|黃埔軍校]]成立，任政治部主任。

1925年3月，[[孫中山|孫中山]][[逝世|逝世]]後改名'''傳賢'''，[[表字|字]]'''季陶'''。11月，参加[[反共主义|反对中國共产党]]的“[[西山会议|西山会议]]”。

1926年，任[[國立中山大學|國立中山大學]]校長。1927年，參與策划[[四一二事件|四一二事件]]。1928年，著《日本論》。2月，升为国民党宣传部长。10月起，擔任[[国民政府委员会|国民政府委员会]]委员、[[考試院院長|考試院院長]]長達20年。至1948年6月辞职，改任国史馆馆长。

戴常作許多國事與世局分析與預測，所以衰老得特別快。<ref>「南京撤退時，[[蔣介石|先君]]交待[[鄭彥棻|鄭彥棻]]先生轉告親伯，勸他早一點離開南京，但他老人家卻堅不肯走。……當時，他老人家住在[[湯山|湯山]]的別墅裡。我啣命前往，親伯說：『我在這裡這麼安靜，為什麼要叫我走呢？』我說：『[[蔣介石|父親]]說的，如果您不走，要我在此地陪您。』……親伯說：『……應知  [[孫中山|總理]]說的我生則國死，我死則國生的道理和軍人魂的氣節！你立即回報你父親，我一定儘早起程前往廣州。』」見見蔣緯國：〈我的親伯〉，刊高仕隱著：《蔣緯國進乎？退乎？》，台北長歌出版社1990年1月25日版，第44－45頁</ref>

== 對西北地區教育的貢獻 ==
1932年10月，国民党中央执行委员会，通过戴季陶[[于右任|于右任]]等人“筹建建设西北专门教育初期计划”议案，成立了“筹建建设西北专门教育委员会”。同年12月，“筹建建设西北专门教育委员会”更名为“建设国立西北农林专科学校筹建委员会”，委员有于右任、[[张继|张继]]、戴传贤、[[王世杰_(中華民國)|王世-{杰}-]]、王陆一、王应瑜、[[朱家骅|朱家骅]]、[[吴敬恒|吴敬恒]]、[[李石曾|李石曾]]、沈鹏飞、[[邵力子|邵力子]]、焦易堂、[[杨虎城|杨虎城]]、[[褚民谊|褚民谊]]等15人，于右任、张继、戴传贤被公推为常务委员。委员会办公处设于国民政府教育部，筹划建设“国立西北农林专科学校”。戴传贤具体主持了校址选择和筹建事务。戴传贤著有《关于西北农林教育之所见》，针对西北教育工作阐述了较为系统的办学思想，对学校的创建具有指导意义。

1934年4月20日，[[国立西北农林专科学校|国立西北农林专科学校]]教学大楼（现西北农林科技大学北校区3号教学楼）奠基典礼，戴季陶到会祝词，宣告国立西北农林专科学校正式成立。祝词曰：“民为国本，食为民天。炎黄立国，首裕民食。姜原后稷，弘兹天职。衣食既足，礼义斯舆。树德务滋，树基务坚。木贵松柏，宝重金刚。坚贞之性，百物之良。立教兴学，志在成人。建国之业，教学为先。民德归厚，百业兴焉。万众一心，教有次第。学有师承，事有始终。德有本根。克勤克俭，创业之源。脚踏实地，步步向前。光荣历史，从此开篇。奠基礼成万众欢。祝我学校万万年。”<ref>[http://file.nwsuaf.edu.cn/laozhaopian/laozhaopian04.html 西北农林科技大学档案馆资料] {{webarchive|url=https://web.archive.org/web/20101128111128/http://file.nwsuaf.edu.cn/laozhaopian/laozhaopian04.html |date=2010-11-28 }}</ref><ref>[http://news.nwsuaf.edu.cn/xsg/index.php?start=3 西北农林科技大学校史长廊]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>国立西北农林专科学校后与[[国立西北联合大学|国立西北联合大学]]农学院、[[国立河南大学|国立河南大学]]农学院畜牧系合并成立国立西北农学院，历经西北农学院、西北农业大学和院系调整，于1999年与同处[[杨陵|杨陵]]的1所大学和5个研究所合并成立为[[西北农林科技大学|西北农林科技大学]]，现为[[中华人民共和国教育部|中华人民共和国教育部]]直属高校。

== 逝世 ==
1949年2月12日，[[國史館|國史館]]館長戴傳賢歿於廣州<ref name="年表">{{cite book |authors=陳-{布}-雷等編著 |title=《蔣介石先生年表》 |location=台北 |publisher=傳記文學出版社 |date=1978-06-01 }}</ref>{{rp|58}}，享年58歲。死因有「[[藥物過量|用藥過量]]」說（晚年飽受神經痛之苦的戴常需要大量[[安眠藥|安眠藥]]助睡<ref>{{cite web |url=http://m.sohu.com/n/408905092 |title=探秘戴季陶之死：自杀可能有多大？ }}</ref>），也有「憂憤自盡」說，因為感於[[國共內戰|國共內戰]]中之[[國民黨|國民黨]]敗局而自殺。另外也有「因病死亡」說，其子[[蔣緯國|蔣緯國]]則認為是心臟病突發致死。<ref>「親伯的死因，有人說他老人家是自殺身亡的，因為桌上和床上都發現了安眠藥。當時我並不在場。不過事後聽別人描述現場情況，我肯定地認為親伯不是一般傳說的自殺，而是[[心臟病|心臟病]]突發過世的。因為親伯的床上灑了好幾顆安眠藥，而在靠床的桌子也發現了安眠藥的瓶子，瓶子裡還有半瓶藥。就我所知，他在晚年，一定要安眠藥才能入睡。那天很可能是他在拿安眠藥的時候，心臟病突發，所以藥才會灑在床上。如果我是自我了斷的話，他儘可以把整瓶安眠藥吃光！」見蔣緯國：〈我的親伯〉，刊高仕隱著：《蔣緯國進乎？退乎？》，[[台北|台北]]：長歌出版社1990年1月25日版，第46頁</ref>戴季陶逝世，蔣介石聞耗悲痛，故人零落，中夜唏噓。<ref>蔣經國：〈危急存亡之秋〉，刊《風雨中的寧靜》，台北：[[正中書局|正中書局]]，1988年1月</ref>{{rp|154}}蔣為之哀誄以「痛失勳耆」。2月15日出殯，蔣率中央執行委員、監察委員全體出席。<ref>安淑萍、王長生著：《蔣介石誄辭說屑》，台北：[[傳記文學|傳記文學]]出版社，2009年5月15日，ISBN 9789578506695</ref>{{rp|17}}3月12日頒發[[褒揚令|褒揚令]]。3月31日又頒令，[[國葬|國葬]]。<ref>《戴季陶傳》</ref>{{rp|134-245}}<ref>当时《广东商报》报道：戴季陶“因鉴于内战苦无了期，民生日形痛苦，忧心过度，病势反而加剧”。</ref>

戴季陶逝世後，蔣緯國為他生父心喪3年，而且每年都在[[考試院|考試院]]為故院長冥誕紀念日在[[善導寺|善導寺]]誦經時，著布衣素袍，行跪拜大禮。<ref name="退乎">高仕隱：《蔣緯國進乎？退乎？》，台北：長歌出版社，1990年1月25日</ref>{{rp|46－47}}

戴季陶有一女二子——戴家祥、[[戴安國|戴安國]]與[[蔣緯國|蔣緯國]]。

戴[[留學|留學]][[日本|日本]]時，與[[蔣介石|蔣介石]]合租一室。戴交一[[護士|護士]]重松金子為[[女友|女友]]，育有一子，但是重松金子與[[蔣介石|蔣介石]]亦有感情，戴因而將此子[[過繼|過繼]]予[[蔣介石|蔣介石]]，即為[[蔣緯國|蔣緯國]]，此經蔣緯國於晚年著書證實。蒋纬国与戴安国的关系乃是同父异母的兄弟。

== 評價 ==
蔣緯國：「親伯在我幼小的心靈中，一直就是一位可敬可愛的長者。」<ref name="退乎"/>{{rp|42}}「大家都傳誦親伯的文章，氣勢澎湃，影響深遠。在我記憶裡……他老人家寫文章是從不草稿的。每次都是跟  [[孫中山|總理]]、[[蔣介石|先君]]……確定原則，然後和黨政有關人士，溝通觀念，博采周諮。……他所撰寫的歷史文件，到今天讀來，猶是擲地有聲。他……字跡工整，一筆不苟，煞是秀美勁拔。」<ref name="退乎"/>{{rp|43}}「親伯寫文章的習慣和功力，可能與革命初期從事新聞工作有關……他那深厚[[中國文化|中國文化]]的涵養，橫溢的才華及堅定不撓的革命意志，蘊育了他……內歛氣質，和鑠為不朽的作品。他曾自署為『戴天仇』，就可以顯出……意志與情操了。」<ref name="退乎"/>{{rp|43}}
