'''戴耀廷'''{{MH}}（{{lang-en|Benny Tai Yiu-ting}}，{{bd|1964年|7月12日|||}}），[[基督徒|基督徒]]，[[香港|香港]][[法學|法學]]學者，現任[[香港大學|香港大學]][[香港大學法律學院|法律系]]副教授。其於2013年提出以「[[讓愛與和平佔領中環|讓愛與和平佔領中環]]」而聞名，他認為[[香港|香港]]若未能有「[[真普選|真正普選]]」，應該參與佔領行動以爭取[[2017年香港行政長官選舉|2017年香港行政長官選舉]]得以有「真正選擇」。<ref>{{Cite web |url= http://www.chinareviewnews.com/doc/1024/4/8/8/102448891.html?coluid=7&kindid=0&docid=102448891&mdate=0225001614 |title= 學者戴耀廷倡“佔領中環”爭普選　要會官員 |date= 2013-02-25 |publisher= 中國評論通訊社 |language= zh }}</ref><ref>{{Cite web |url= http://hk.news.yahoo.com/%E9%A0%90%E6%BC%94%E4%BD%94%E9%A0%98%E4%B8%AD%E7%92%B0%E6%88%B4%E8%80%80%E5%BB%B7%E4%B8%8D%E5%8F%83%E8%88%87-220523292.html |title= 預演佔領中環戴耀廷不參與 |date= 2013-02-25 |publisher= 星島日報 |language= zh }}</ref>

== 背景 ==
戴耀廷中學畢業於[[拔萃男書院|拔萃男書院]]，後來於[[香港大學|香港大學]]獲得[[法律學位|法律學士]]學位，當時擔任[[香港大學學生會|香港大學學生會]]的外務秘書<ref>[http://news.hk.msn.com/highlight/%E8%B6%99%E7%B4%AB%E9%99%BD%E3%80%8C%E6%B0%91%E4%B8%BB%E6%B2%BB%E6%B8%AF%E3%80%8D%E8%A6%86%E5%87%BD%E6%9B%9D%E5%85%89-84%E5%B9%B4%E8%A8%B1%E8%AB%BE%E6%B8%AF%E5%A4%A7%E7%94%9F-%E6%88%B4%E8%80%80%E5%BB%B7%EF%B9%95%E6%88%91%E6%B0%91%E4%B8%BB%E5%95%9F%E8%92%99%E4%B9%8B%E4%B8%80 趙紫陽「民主治港」覆函曝光 84年許諾港大生 戴耀廷﹕我民主啟蒙之一]</ref>，及後取得[[法律深造證書|法律深造證書]]，及於[[倫敦大學|倫敦大學]]獲得法律碩士學位。與戴耀廷為同屆的學生包括[[資深大律師|資深大律師]]、前[[律政司司長|律政司司長]][[袁國強|袁國強]]和香港傳媒人士[[劉進圖|劉進圖]]等等。其研究興趣為[[憲法|憲法]]、[[憲政|憲政]]、[[公共法|公共法]]、[[法律|法律]]和[[政治|政治]]及[[宗教|宗教]]的關係、[[人權|人權]]及[[參與式民主|參與式民主]]等。

戴耀廷自言在求學時期，與那個時代許多[[泛民主派|民主派]]人士一樣有濃烈的「中國情懷」，喜歡讀[[中文|中文]]、[[中史|中史]]。讀[[清朝歷史|清朝歷史]]講到中國被列強入侵時「會很傷心」；講到[[日軍侵華|日軍侵華]]時則希望中國日後可以強大。大學期間，吸收了更多民主自由的思想，並在愛國情懷中加添了一份批判，認為強大的中國不只是富國強兵，而是要讓它走向憲政<ref name="01周報">{{Cite news |url=https://www.hk01.com/article/147114/ |title=一國兩制的前世今生　那些年法律學者這樣想像一國兩制 |author=周嘉俊 |date=2018-01-08 |work=01周報 深入報道}}</ref>。

戴耀廷自香港大學畢業後，曾經於[[泛民主派|民主派]]領袖[[李柱銘|李柱銘]]的辦公室擔任法案助理，也是《[[香港基本法|香港基本法]]諮詢委員會》的兩位學生代表之一<ref>{{Cite web |url= http://www.law.hku.hk/faculty/staff/tai_benny.html |title= Faculty of Law, The University of Hong Kong » Faculty » Academic Staff |language= en }}</ref><ref name="天下">[http://www.cw.com.tw/article/article.action?id=5051520 佔領香港中環，專訪運動發起人戴耀廷]，[[天下雜誌|天下雜誌]]，2013-08-20</ref>。該段時期戴對[[一國兩制|一國兩制]]的實施帶有批判和懷疑，他在法律系的學士論文是研究一國兩制如何在[[台灣|台灣]]實施，認為在實施上要面對很多問題。一國兩制本身是充滿矛盾的產物，兩制衝突是必然。中港兩地無論經濟制度、社會狀況均有不少差異，要結合並不容易，戴稱「當時希望有一國兩制，但又不是天真到覺得一國兩制無問題。」<ref name="01周報"/>

2006年，戴以學者身份協助[[香港教育電視|教育電視]]製作面向全港中學生的「認識基本法」公民教育節目<ref name="01周報"/>。

戴耀廷現為[[香港大學|香港大學]][[香港大學法律學院|法律系]]副教授、並曾在2000年至2008年出任法律學院副院長。

== 推動「佔領中環」運動 ==
{{main|讓愛與和平佔領中環}}
[[File:Occupy_Central_forum_HKBU_201304.JPG|thumb]]舉行的佔領中環論壇（2013年4月）]]
[[File:Occupy_Central_(2014)_from_VOA_(7).jpg|thumb]]三位發起人之一]]
戴耀廷於2013年1月16日在[[信報|信報]]專欄拋出「要爭取香港落實[[真普選|真普選]]，可能要準備『殺傷力』更大的武器——佔領中環」。3月27日，他與[[香港中文大學|香港中文大學]]副教授[[陳健民_(社會學者)|陳健民]]與牧師[[朱耀明|朱耀明]]等發布「讓愛與和平佔領中環」信念書。他說「佔中是為了不佔中，就是要透過對抗產生一種張力，佔中是未來的緊張，讓大家回到談判裡。」目的是「用運動帶動政治文化的轉變」。提出[[公民抗命|公民抗命]]的主張，表面上是對抗；但他重視的是民主的商討，透過討論過程，希望找到解決紛爭之方法。佔中運動有個社運少見的設計，他希望能募集一萬人願意簽下「誓約」，對外宣布自己支持非暴力佔中。如果最後真的必須走上街頭，40歲以上的人要走在前面，結束後這一萬人要負起責任，一起走到警察局去投案，聽從警方處理。「我們這一代願意為下一代付出代價」。[[李柱銘|李柱銘]][[資深大律師|資深大律師]]與[[天主教香港教區|天主教香港教區]]前主教[[陳日君|陳日君]][[樞機|樞機]]等均表態支持。<ref name="天下"/>戴耀廷說「這不算是個政治活動，對我自己而言，這是一個宗教活動，我在傳道。我把我一生所教都放進去了。因為《[[聖經|聖經]]》裡說，要行公義（Do justice），重點是要行動。」<ref name="天下"/>

=== 商討及公投 ===
2013年6月至2014年5月，在學校教[[參與式民主|參與式民主]]的戴舉辦了「商討日」。700多個來自各個不同團體的群眾，加上200位義工，分為40個小組討論7大議題。除了主辦單位，沒有人致詞，不分階級做討論，家庭主婦身邊坐的可能是立法會議員，也可能是商人或小學老師。他也設計一套模式，讓各民間、社區組織自行舉辦「商討日」。<ref name="天下"/>

2014年6月22日，和平佔中委托[[香港大學民意研究計劃|香港大學民意研究計劃]]舉辦[[6.22民間全民投票|6.22民間全民投票]]，以選擇[[真普選|民間認可普選方案]]。

=== 啟動 ===
2014年8月31日，[[中华人民共和国全国人民代表大会常务委员会|中华人民共和国全国人民代表大会常务委员会]]決定，[[2017年香港行政长官选举|2017年香港行政长官选举]]必須先經[[提名委员会|提名委员会]]提名，並規定了提名委员会之構成，候選人須獲過半委员提名，最多有三名候選人。2014年9月2日，戴耀廷承認，逼迫北京讓步之策略目標已經失敗，而市民對他們靜坐示威之支持度也在下降。北京不妥協之立場導致「加入抗議行列的人數將比預期少，因為香港人思維講究務實。」戴表示會在公眾假期行動，令更多人能參加，當時傳聞是10月1日<ref>{{Cite news |url= https://tw.appledaily.com/international/daily/20140904/36064624 |title= 萬人佔中 戴耀廷：有信心 |date= 2014-09-04 |publisher= 蘋果日報 }}</ref>。

2014年9月26日晚上，[[2014年香港學界大罷課|學界罷課集會]]演變成[[重奪「公民廣場」行動|重奪「公民廣場」行動]]。9月27日，添美道聲援集會氣氛趨向熾熱。9月28日凌晨1時，戴耀廷與陳健民、朱耀明宣布即時啟動「佔領中環」，並提出兩點訴求：一、撤回人大常委會政改決定，二、馬上重新啟動政改諮詢。<ref>{{Cite web |url= http://hk.apple.nextmedia.com/realtime/news/20140928/52951682 |title= 戴耀廷：佔領中環正式啟動！|date=2014-09-28 |work= (香港)蘋果日報}}</ref>此舉引起部分學生不滿，質疑「佔中」騎劫罷課集會，<ref>{{Cite web |url= http://hk.apple.nextmedia.com/realtime/news/20140928/52951973?top=12h |title= 有學生質問戴耀廷 |date=2014-09-28 |work= (香港)蘋果日報}}</ref>戴耀廷否認此說，強調佔中與罷課目標一致，並對學生表示感激。<ref>{{Cite web |url= http://www.appledaily.com.tw/realtimenews/article/international/20140928/477975/ |title= 否認綁架罷課 戴耀廷：佔中是學生後盾 |date=2014-09-28 |work=[[蘋果日報_(香港)|蘋果日報 (香港)]]}}</ref>其後[[9·28催淚彈驅散行動|9·28催淚彈驅散行動]]演變成[[雨傘革命|雨傘革命]]，戴耀廷對此感觸，承認情況失控超出想像，但對港人爭取政治改革之決心深感自豪。<ref>{{Cite web |url= http://hk.apple.nextmedia.com/news/art/20140929/18883427 |title= 戴耀廷哽咽：香港人好嘢 |date=2014-09-29 |work=[[蘋果日報_(香港)|蘋果日報 (香港)]]}}</ref>

=== 捐款 ===
2014年10月月底，一名自稱「一個愛大學的學者」向部份[[香港傳媒|香港傳媒]]提供一批[[香港大學|香港大學]]內部文件及電子郵件紀錄，涉及戴耀廷於2013年5月及2014年1月收取一份捐予[[港大民意研究計劃|港大民意研究計劃]]、法律學院和人文學院，合共145萬港元的匿名捐款，主要用於2014年[[6.22民間全民投票|6.22民間全民投票]]及[[和平佔中|和平佔中]]前期會議。和平佔中解釋，該批款項由「佔中三子」之一的[[朱耀明|朱耀明]]捐出，當中130萬港元來自支持朱耀明推動香港民主發展的熱心市民，另外15萬港元在朱耀明壽宴上籌得。該批外洩內部文件亦顯示，香港大學知悉該4筆匿名捐款，至同年12月月底，香港大學校務委員會決定由轄下的審核委員會審核該些捐款有否違反規條。 

2015年3月25日，香港大學校務委員會討論[[朱耀明|朱耀明]]捐款審核報告，其中兩筆合共35萬港元捐予人文學院的款項被指控為用作聘用研究助理，在佔領中環期間則成為戴耀廷的秘書，涉嫌公器私用，並不符合捐款原意。校務委員會於當日突然將該份審核報告由終期報告改稱中期報告，並且要求審核委員會提供終極報告，以釐清戴耀廷的責任所在<ref>[http://hk.apple.nextmedia.com/news/art/20150326/19090591 戴耀廷風波港大報告：35萬捐款違原意 建制派委員窮追猛打] 《蘋果日報》 2015年3月26日</ref>，校委會借此事[[香港大學副校長任命事件#佔中捐款調查報告|拖延任命陳文敏為副校長]]<ref>[https://hk.news.appledaily.com/local/daily/article/20150514/19147123 戴耀廷反駁港大報告：捐款合慣常做法]. 蘋果日報. 2015年5月14日. </ref>。

=== 审判 ===

== 風雲計劃 ==
2017年4月18日，戴耀廷提出風雲計劃。民主派要在2019年區議會選舉中，在港九、新界分別贏得105個、112個或以上的民選議席，使民主派在2021年取得選委會第四界別政界中港九各區議會、新界各區議會的57個及60個選委席位，總席位由300多個增至500多個，最終阻止北京政府操控2022年的特首選舉，迫使中共改革現行制度以達真普選。<ref>{{cite web |author1=港大法律系副教授　戴耀廷 |title=2019區選「風雲計劃」 |url=https://hk.news.appledaily.com/local/daily/article/20170418/19993198 |accessdate=2018-07-25}}</ref>戴耀廷為此開設了培訓班，協助有志參選者出戰區議會。<ref>{{cite web |title=【專訪】風雲計劃辦區選培訓班　戴耀廷：讓市民看到泛民有決心贏 |url=https://www.hk01.com/政情/179249/專訪-風雲計劃辦區選培訓班-戴耀廷-讓市民看到泛民有決心贏}}</ref>中國大陸媒體環球視野認為這完全就是港獨奪權計劃。<ref>{{cite web |title=港媒调查揭戴耀廷“风云计划”：本质就是“港独夺权计划” |url=http://www.globalview.cn/html/societies/info_24703.html |accessdate=2018-07-25}}</ref>
