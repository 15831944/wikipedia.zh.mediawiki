'''戴先中'''（{{bd|1954年|4月4日|||戴先中}}）生于[[江苏省|江苏省]][[昆山市|昆山市]]，现为[[东南大学|东南大学]]“控制理论与控制工程”国家重点学科学术带头人、[[博士生导师|博士生导师]]。<ref name="zongti">{{cite web |title=戴先中——神经网络专家|url=http://www.zgkjcx.com/Article/ShowArticle.asp?ArticleID=7050# |website=[[中国科技创新网|中国科技创新网]] |accessdate=2018-08-17}}</ref>

== 生平 ==
戴先中现为“复杂工程系统测量与控制”教育部重点实验室主任，[[国家杰出青年基金|国家杰出青年基金]]获得者，国家教学名师，国家精品课程和国家教学团队负责人等，兼任[[中国自动化学会|中国自动化学会]]电气自动化专委会副主任兼学术委员会主任、[[高等学校自动化专业教学指导委员会|高等学校自动化专业教学指导委员会]]委员、[[国家自然科学基金委|国家自然科学基金委]]（自动化）学科评审组成员、[[国际先进机器人组织|国际先进机器人组织]]的中国代表、[[IEEE|IEEE]]等国际刊物的审稿人等。曾两次赴美国[[康乃尔大学|康乃尔大学]]，一次赴德国[[埃尔朗根-纽伦堡大学|埃尔朗根-纽伦堡大学]]进行合作研究。<ref name="zongti"/>

戴先中的主讲课程为：动化学科概论、微机系统与接口。<ref name="zongti"/>

2018年，戴先中担任第五届“[[台达电子|台达]]杯”高校自动化设计大赛主办方代表。他表示，随着[[智能制造|智能制造]]时代的到来，现代职业教育更需要像[[台达电子|台达]]这样具有丰富经验和技术实力的企业介入，此次大赛可以强化高校学生的应用技术能力，当他们步入社会后，可以快速助力企业提升竞争力，实现转型升级。<ref>{{cite web |author1=[[新潮电子杂志|新潮电子杂志]] |title=74支国际队伍竞逐“智造高手”桂冠，第五届“台达杯”高校自动化设计大赛决赛隆重展开 |url=http://tech.ifeng.com/a/20180725/45083052_0.shtml |website=[[凤凰网|凤凰科技]] |accessdate=2018-08-17|date=2018-07-25}}</ref><ref>{{cite web |title=74支高校队伍竞逐“智造高手”桂冠 |url=http://www.vos.com.cn/news/2018-07/27/cms903028article.shtml |website=[[海峡之声|海峡之声]] |accessdate=2018-08-17|date=2018-07-27}}</ref>

2018年8月7日开幕的第十二届“[[三菱电机|三菱电机]]杯”全国大学生电气与自动化大赛中，戴先中为评委之一。<ref>{{cite news |author1=[[迟嵩|迟嵩]]（记者）、 [[徐状|徐状]]（通讯员） |title=全国大学生电气与自动化大赛在哈理工举行 |url=http://hlj.cnr.cn/htpxw/20180807/t20180807_524325671.shtml |accessdate=2018-08-17 |agency=央广网 |date=2018-08-07}}</ref>
