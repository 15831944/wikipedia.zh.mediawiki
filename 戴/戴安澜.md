'''戴安澜'''（{{bd|1904年|11月25日|1942年|5月26日}}<ref>《外侮需人御，将军赋-{采}-薇--忆抗日爱国将领戴安澜》一文中戴安澜的出生日期为1905年10月19日</ref>），原名'''戴炳陽'''，字衍功，自號'''海鷗'''，[[安徽省|安徽省]][[无为县|无为县]]人。[[国民革命军第五军|国民革命军第五军]]第二〇〇師[[師長|師長]]，著名抗日將領，以血战[[昆仑关|昆仑关]]、遠征[[缅甸|缅甸]]聞名。
[[File:戴安瀾.jpg|thumb]]

== 生平 ==
=== 北伐时期 ===
1924年加入[[北伐军|北伐军]]，当二等兵，1925年入[[黃埔軍校|黃埔軍校]]第三期畢業，翌年參加[[國民革命軍北伐|國民革命軍北伐]]。先後在教導第二師、第四師、第二五師任連、營、團長等職。北伐剿匪，無役弗從。

===抗日战争时期 ===
==== 国内 ====
1933年春，日军协同[[满洲国军|满洲国军]]全面进攻[[热河|热河]]，危及[[北平市|平]][[津|津]]，[[長城戰役|长城抗战爆发]]，戴安澜当时在[[中華民國國軍|中華民國國軍]][[國民革命軍第十七軍|十七军]]第二十五师[[关麟征|关麟征]]部任第三十七旅第一五三團[[团长|团长]]，隨軍北上，擔任完備古北口將軍樓迤右至龍兒峪以東五百米之高地線；参加了[[长城抗战|长城抗战]]，获得三等[[云麾勋章|云麾勋章]]一枚。之後累升至[[旅長|旅長]]。[[七七事變|七七事變]]後，在1938年的[[鲁南会战|鲁南会战]]中，曾率部在中艾山与日军激战4昼夜，因战功卓著，升任第八十九师副师长。同年8月参加[[武汉会战|武汉会战]]。35歲時，升任第五軍第二〇〇師師長。1939年12月，在[[廣西|廣西]][[昆仑关|昆仑关]]與日军[[第5師团|第5師团]]激烈鏖战，戴安澜指挥有方，重伤不下火线，击毙日军[[旅团|旅团]]长[[中村正雄|中村正雄]][[少将|少将]]，贏得著名的[[昆仑关大捷|昆仑关大捷]]。<ref name="name"> http://news.xinhuanet.com/ziliao/2004-07/01/content_1561651.htm</ref>

==== 緬甸 ====
1942年3月，戴安瀾率第二〇〇師西出[[雲南|雲南]]，入[[緬甸|緬甸]]作戰。他不惜冒孤軍深入的危險，開進[[同古|同古]]，逐次接替了[[英軍|英軍]]的防務。為了掩護英軍安全撤退，充分作好迎戰準備，戴安瀾率部日夜搶修工事，布下三道防線，阻擊遲滯敵軍前進。 <ref>蔣介石單獨召見戴安瀾，詢問第二〇〇師能否在同古堅守一兩週，打個勝仗？戴立正誓言：「此次遠徵，係[[唐|唐]][[明|明]]以來揚威國外之盛舉，戴某雖戰至一兵一卒，也必定挫敵兇焰，固守同古。」見《戴安瀾列傳》，北京：解放軍出版社，1988年</ref>他在致夫人[[王荷馨|王荷馨]]的信中寫道：「-{餘}-此次奉命固守同古，因上面大計未定，後方聯絡過遠，敵人行動又快，現在孤軍奮鬥，決心全部犧牲，以報國家養育。為國戰死，事極光榮。」他帶頭立下遺囑：只要還有一兵一卒，亦需堅守到底。如本師長戰死，以副師長代之，副師長戰死以參謀長代之。參謀長戰死，以某某團長代之。全師各級指揮官紛紛效仿，誓與同古共存。3月25日，以8000人挡住日军精锐的[[第55师团|第55师团]]20000餘众的进攻，取得[[同古保卫战|同古保卫战]]胜利（今称[[东吁|东吁]]），贏得中外讚賞。同古保卫战历时12天，第二〇〇师以高昂的斗志与敌鏖战，以牺牲800人的代价，打退了日军20多次冲锋，歼灭敌军4000多人，俘敌400多人，予敌重创，打出了国威<ref name="name"/>。同古战役后期，由于右翼英军只顾逃命，放任日军长驱直入，左翼国军行动迟缓，以致让日军迅速前出并偷袭成功，致使第五军三面受敌，第二〇〇师一度处于被围歼之势，势不得已，第二〇〇师奉命向北突围。为挽颓势，戴安澜指挥第二〇〇师奋勇夺占棠吉，再立战功。后缅甸战场形势日下，曼德勒会战计划流产，5月8日上午，缅北远征军基地[[密支那|密支那]]被占日军攻占，与国内的联系被切断，后援断绝，中国远征军有被围歼之势，指揮官[[杜聿明|杜聿明]]按蒋中正7日的命令，下令第五军经缅北野人山后撤回国。第三十八师师长[[孫立人|孫立人]]認為撤退路途遙遠，拒絕接受命令，接受中缅印战区美陆军司令兼中国战区总司令蒋介石的参谋长[[约瑟夫·史迪威|约瑟夫·史迪威]]的指挥，率所部第三十八师撤往[[印度|印度]]。戴安瀾接受撤退命令率領第二〇〇師經缅北野人山、滇西縱谷撤往雲南。日軍沿途追擊第五軍，因地形不利行軍，五軍各部遭受嚴重打擊。5月16日拂晓，第二〇〇師师部在朗科地區行进途中遭敵伏兵袭击，戴安瀾胸、腹部重伤。<ref>上海教育學院組織編，沈起煒主編：《歷史》（八年級第二學期課本），上海教育出版社，ISBN 7-5320-5131-5</ref>至5月26日時，因傷口感染嚴重，已糜爛穿孔，由于缅北复杂的地形和连绵的阴雨，戴安澜终因缺乏药物医治，伤口化脓溃烂<ref name="name"/>，戴自知來日無多，命左右衛士將之扶起，隨後向北面高呼「反攻！反攻！中華民國萬歲！」後，故於[[緬甸|緬甸]][[茅邦村|茅邦村]]，享年38歲。

戴安瀾犧牲的消息傳回國，蔣電令：務必將戴師長遺體运回祖國。5月29日，因為天氣炎熱，戴遺體流膿水發臭，不能再繼續行走，又不能留在緬甸，乃決定火化。200师官兵将戴棺材遺體在原木上火化後，揀出遺骨，按部位用綢布包好，裝在木箱裡，跟五九八團團部行進。其遺體由其屬下揹負，隨同部隊撤回雲南。第二〇〇师在撤退回国沿途損失超過2,000人以上。

== 身后 ==
[[File:戴安澜烈士墓.jpg|thumb]]
[[File:戴安澜将军像.jpg|thumb]]

戴安瀾將軍殉國後，[[中華民國政府|中華民國政府]]於1943年4月1日在[[廣西|廣西]][[全州縣|全州]]的湘山寺為其舉行了國葬，並追贈陸軍[[中將|中將]]；[[国民政府军事委员会|国民政府军事委员会]][[委员长|委员长]][[蔣中正|蔣中正]]以校長身份贈輓聯：「虎頭食肉負雄姿，看萬里長征，與敵周旋欣不忝。馬革裹屍酹壯志，惜大勳未成，虛予期望痛何如？」

[[中共|中共]]领导人[[毛泽东|毛泽东]]为其赋诗《'''五律·輓戴安澜将军，五律·海鸥将军千古'''》：「外侮需人御，将军赋-{采}-薇。师称机械化，勇夺虎罴威。浴血東瓜守，驱倭棠吉归。沙场竟殒命，壮志也无违。」（外侮：外國侵略者，指日寇。采薇：《詩·小雅》中有《采薇》篇，其詩描寫戍邊抗擊外族入侵的兵士久歷艱苦，在回鄉的路上又飽受飢寒。虎羆：指敵人。東瓜、棠吉：緬甸地名。烈士英勇抗日，光輝獻身，壯志未酬，雖死猶榮。） ：[[美國國會|美國國會]]授權總統[[羅斯福|羅斯福]]頒發[[第二次世界大戰|第二次世界大戰]]中外國[[同盟軍|同盟軍]]將領之軍團功勳章給戴安瀾將軍，他是第一個受此殊榮的中國軍人。

1947年，戴安澜墓迁葬于安徽芜湖赭山公园内。1956年[[中華人民共和國|中華人民共和國]]追認其為[[革命烈士|革命烈士]]。

2009年9月10日，在[[中共中央宣传部|中央宣传部]]、[[中共中央组织部|中央组织部]]、[[中共中央统一战线工作部|中央统战部]]、[[中央文献研究室|中央文献研究室]]、[[中央党史研究室|中央党史研究室]]、[[中华人民共和国民政部|民政部]]、[[人力资源社会保障部|人力资源社会保障部]]、[[中华全国总工会|全国总工会]]、[[中国共产主义青年团|共青团]]中央、[[中华全国妇女联合会|全国妇联]]、[[中国人民解放军总政治部|解放军总政治部]]等11个部门联合组织的“100位为新中国成立作出突出贡献的英雄模范人物和100位新中国成立以来感动中国人物”评选活动中，戴安澜被评为「100位为新中国成立作出突出贡献的英雄模范人物」。<ref name="name"/>

2017年5月26日全国黄埔军校同学会（总会），安徽省、江苏省、上海市黄埔军校同学在安徽芜湖举办爱国抗日名将戴安澜烈士殉国75周年纪念活动。<ref>[http://www.thepaper.cn/newsDetail_forward_1697985 抗日名将戴安澜烈士殉国纪念活动在安徽芜湖举行]，澎湃新闻。</ref>

== 紀念郵票 ==
1975年9月3日欣逢抗戰勝利30周年紀念，為宣揚英烈抗日事跡，激發國人愛國思想。中華民國交通部印制發行抗日英烈像郵票一組，編號為-普214，此組郵票共六枚。其中一枚深藍色5元郵票以戴安澜烈士遺像為圖案。此票由中央印制廠以凹版印刷，圖副33X23公厘，齒孔12度，全張100枚，總發行量為2，000，000枚。

交通部郵政總局所遍之中國郵票目錄（民國70年版）對此票圖案說明為：故陸軍中將戴安瀾烈士，號海鷗，安徽無為人。生于民國前7年，卒于民國31年，是年3月25日參與緬甸戰役，與敵激戰與同古，挫敵攻勢；28日敵大軍增援圍攻同古，烈士身先士卒，斃敵三千余，次日與業達西猛攻北竄之敵，戰績彪炳，樹我印緬遠征軍之聲望。5月25日擊退增援反攻之敵，旋於猛密特意北與敵遭遇，激戰一晝夜，烈士奮勇指揮，身負重傷仍不稍退迄26日，重傷不起，壯烈成仁。
