'''提拉米蘇'''（{{lang-it|Tiramisù}}；[[威尼斯語|威尼斯語]]：{{lang|vec|Tiramesù}}），又可譯成'''堤拉米酥'''，是一種有名的[[義大利|義大利]]式[[蛋糕|蛋糕]]，它是由泡過[[咖啡|咖啡]]或[[蘭姆酒|蘭姆酒]]的手指餅乾，加上一層[[马斯卡邦尼奶酪|马斯卡邦尼奶酪]]、[[蛋黃|蛋黃]]、[[糖|糖]]的混合物，然後再在蛋糕表面灑上一層[[可可粉|可可粉]]而成。

== 歷史 ==
大部份對提拉米蘇面世時間的報導，都指這種蛋糕在1960年代誕生於意大利[[威尼托|威尼托]]政區[[特雷維索|特雷維索]]市的"Le Beccherie"餐廳。

== 名稱 ==
「Tiramisù」按照音譯是提拉米蘇。而在意大利原文裏，“Tira”是“提、拉”的意思，“Mi”是“我”，“sù”是“往上”，合起來就是“拉我起來”的意思，
這一甜點自1980年代中期走红，今日，世界各地的各色高档餐馆都可能供应此[[甜点|甜点]]，而不为意式餐馆所专供。它的配方中最具创新意义的是咖啡风味的[[起司奶蛋液|起司奶蛋液]]，这一新口味亦为蛋糕、布丁等其它形式的冷热[[甜点|甜点]]所吸收。


義大利民間也有很多關於提拉米蘇的傳說，流傳最廣的那個可能會讓人大跌眼鏡，提拉米蘇在那個故事中和二戰後的義大利妓院息息相關，當時的妓院設有廚房，為到訪的客人供應餐點，而這個含有咖啡的小甜品可以讓客人雄風大振，事後亦可幫助客人從睏倦中解脫出來，振作精神回去上班，這版本中提拉米蘇不僅沒什麼浪漫色彩，反而還帶有些情色意味，時常被拿來開曖昧的小玩笑。

浪漫版本: 中世紀一個意大利士兵要出征前，家裡剩下可吃的東西不多，他的愛妻給他準備食物，把家裡剩下來能吃的材料全部做進一個蛋糕裡，丈夫問她，這個甜點叫什麼名子，妻子就跟他說，這個蛋糕叫做提拉米蘇，意思即是"請帶我走"。

== 傳統配方 ==
[[马斯卡邦尼奶酪|马斯卡邦尼奶酪]]、[[手指餅乾|手指餅乾]]、蛋黃、[[蛋白|蛋白]]、[[濃縮咖啡|濃縮咖啡]]、高純度無糖可可粉。

其中的蛋黃和蛋白兩件都需要分別充分打鬆。[[起司|起司]]則需準備于較大的攪拌容器中，与兩件先行打發物充分混合，制成[[起司奶蛋液|起司奶蛋液]]。

== 葷素分類 ==
一般提拉米蘇如果沒有特別標示大多是葷的，因為提拉米蘇其中主要原料馬士卡彭（Mascarpone）是葷的。或為加入吉力丁（gelatin，動物膠）者為葷食。

除非有特別標示[[素食|素食]]，或是素食麵包店製作的才可以確定是[[素食|素食]]（如奶蛋素）的，[[素食者|素食者]]宜分辨清楚。

== 禽流感恐慌 ==
由於提拉米蘇製作過程需要用到生蛋，為了因應[[禽流感|禽流感]]，部分商家（如[[台北遠東飯店|台北遠東飯店]]）將生蛋以其他材料代替；部分（如統一[[星巴克|星巴克]]）則已暫時停賣提拉米蘇。目前台灣地區已有符合歐。美。日食品衛生安全標準的殺菌蛋品公司，在生產線的過程中將傷害人類的因素去除，這也是先進國家之所以能在市場上販售生食卻不需擔心傷害人民健康的做法。
