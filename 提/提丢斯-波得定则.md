'''提丢斯-波得定则'''（'''{{lang|en|Titius-Bode law}}'''）是关于[[太阳系|太阳系]]中[[行星|行星]]轨道半徑的一个简单的几何学规则。

它是在1766年時，由[[德国|德国]]的一位大学教授{{link-en|约翰·达尼拉·提丢斯|Johann Daniel Titius}}所提出，后来被[[柏林天文台|柏林天文台]]的台长[[约翰·波得|约翰·波得]]（Johann Elert Bode）归纳成了一个[[经验公式|经验公式]]来表示。

这个公式可以表述为：

:<math>a = \frac{n+4}{10}</math>

其中
*n = 0, 3, 6, 12, 24, 48...（n≥3時，后一个数字为前一个数字的2倍）

现代的公式把a作为[[行星|行星]]到[[太阳|太阳]]的平均距离（[[天文单位|天文单位]]）：

:<math>\textstyle a = 0.4 + 0.3 \cdot 2^{n}</math> 

（<math>\scriptstyle n = -\infty, 0, 1, 2 \ldots</math>）

*n代入3時，得解為2.8，但並無對應的行星，所以當時推測火星與木星中有所謂的「消失的第五行星」，之後發現[[穀神星|穀神星]]，因為符合2.8AU的距離，所以一時間穀神星被列為第五行星，之後陸續發現[[小行星|小行星]]，證明並無此第五行星，於是此解便改為適用於小行星帶的平均距離2.77AU。

*n代入7時，得解38.8，較符合[[古柏帶|古柏帶]]的距離，古柏帶的[[矮行星|矮行星]][[冥王星|冥王星]]（29.658～49.305AU，平均39.482AU）、[[鳥神星|鳥神星]]（38.509～53.074AU，平均45.791AU）、[[妊神星|妊神星]]（35.164～51.526AU，平均43.335AU）大致在這個距離（上述距離為平均距離）。

*n代入8時，得解77.2，許多其他語言的維基百科，都將此值與[[黃道離散天體|黃道離散天體]]的矮行星[[鬩神星|鬩神星]]（37.808～97.610AU，平均67.7091AU）作比較。

現在的天文學界，多認為波得定則只是一個巧合。
