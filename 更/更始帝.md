'''漢更始帝刘玄'''（{{bd|？||25年||catIdx=L}}），字'''圣公'''，[[南阳郡|南阳郡]][[蔡阳县|蔡阳县]]（今[[湖北省|湖北省]][[枣阳市|枣阳市]]西南）人，两[[汉朝|汉]]之际[[绿林军|绿林军]]擁立的[[皇帝|皇帝]]，或被視為[[西漢|西漢]]的最後一位皇帝。

劉玄原本是[[西汉|西汉]]宗室，是汉景帝[[劉啟|劉啟]]的後代、汉光武帝[[刘秀|刘秀]]的族兄。祖父為[[苍梧|苍梧]][[太守|太守]][[刘利|刘利]]，父[[刘子张|刘子张]]，母[[何氏_(漢更始帝母)|何氏]]。

== 生平 ==
起初，刘玄的弟弟被人杀害，他便结交宾客准备报仇，可不久之后宾客犯罪，刘玄便跑到[[平阳县_(山西)|平阳]]躲避官府的追捕。官吏抓走他的父亲[[刘子张|刘子张]]，后来刘玄诈死，让人把他的[[灵柩|灵柩]]运回舂陵，官吏才释放了他的父亲。随后刘玄便逃跑藏匿起来。<ref name=Ref1>{{cite book|author=范晔|title=《[[后汉书|后汉书]]·卷十一 刘玄刘盆子列传第一》}}</ref>

[[新莽|新莽]][[天凤|天凤]]四年（17年），南方发生饥荒，百姓只得到沼泽中挖[[荸荠|荸荠]]野菜度日。[[京山县|新市]]人[[王匡_(更始)|王匡]]、[[王凤|王凤]]带领[[马武_(东汉)|马武]]、[[王常|王常]]、[[成丹|成丹]]等人起义。他们藏身[[绿林山|绿林山]]中，不久队伍就发展到五万多人。<ref name=Ref1 />

[[地皇_(王莽)|地皇]]三年（22年）[[農曆七月|七月]]，平林人[[陈牧|陈牧]]、[[廖湛|廖湛]]领导千余人起义，号平林兵，以应王匡，正在这里避难的刘玄参加平林军，担任安集[[掾|掾]]。地皇四年（23年）[[農曆正月|正月]]，绿林军诸部合兵击破新莽将领[[甄阜|甄阜]]、[[梁丘赐|梁丘赐]]，遂号刘玄为'''更始将军'''。[[農曆二月|二月]]辛巳（3月11日），后因其为刘姓宗室，遂在[[淯水|淯水]]边被拥立为帝，大赦天下，建元'''[[更始_(更始帝)|更始]]'''。更始元年[[農曆六月|六月]]入都[[宛城|宛城]]，大封[[宗室|宗室]]诸将。他嫉[[刘縯|刘縯]]、[[刘秀|刘秀]]兄弟威名，便诛杀刘縯。起义军在[[昆阳之戰|昆阳之戰]]获胜后，更始帝遣[[王匡|王匡]]攻[[洛阳|洛阳]]，[[申屠建|申屠建]]、[[李松|李松]]攻[[武关|武关]]，[[三辅|三辅]]震动，各地豪强纷纷诛杀新莽的[[州牧|州牧]][[郡守|郡守]]，用'''汉'''年号，服从更始政令。更始元年[[農曆十月|十月]]，定都[[洛阳|洛阳]]。[[王莽|王莽]]败死后的更始二年（24年），迁都[[长安|长安]]。<ref name=Ref1 />

===更始分封===
李松和[[趙萌|趙萌]]進言更始帝，将功臣封王。[[大司馬|大司馬]][[朱鮪|朱鮪]]反对，称[[汉高祖|汉高祖]]的誓言非刘氏不王。更始帝采纳李松之言，以[[王匡|王匡]]为比阳王、[[王凤|王凤]]为宜城王、[[朱鲔|朱鲔]]为胶东王、卫尉大将军[[张卬|张卬]]为淮阳王、廷尉大将军[[王常|王常]]为邓王，执金吾大将军[[廖湛|廖湛]]为穰王、[[申屠建|申屠建]]为平氏王、尚书[[胡殷|胡殷]]为随王、柱天大将军[[李通|李通]]为西平王、五威中郎将[[李轶|李轶]]为舞阴王、水衡大将军[[成丹|成丹]]为襄邑王、大司空[[陈牧|陈牧]]为阴平王、骠骑大将军宋佻为颍阴王、[[尹尊|尹尊]]为郾王。李松昇進[[丞相|丞相]]，和趙萌共理内政。

=== 腐敗 ===
更始帝雖恢復漢室，但其性格迂腐，重用小人。一登基便沉湎于[[宫廷|宫廷]]奢華生活，即位后将政事都委托于自己的岳父[[赵萌|赵萌]]，放任其[[外戚|外戚]]专权。[[赤眉|赤眉]]军进逼长安时，更始帝杀害申屠建、陈牧、成丹等起义军重要将领。更始三年（25年）他殺害西漢末代君主[[劉嬰|劉嬰]]。

更始三年（25年）[[農曆九月|九月]]，[[赤眉军|赤眉军]]攻入[[长安|长安]]，更始帝单骑逃走。同年[[農曆十月|十月]]，投降赤眉，将玺绶送给赤眉拥立的皇帝[[刘盆子|刘盆子]]，自己被封为'''畏威侯'''，不久改封为'''长沙王'''，刘秀则遥降封之为'''淮阳王'''<ref>《後漢書·光武帝紀·上》載詔：「更始破敗，棄城逃走，妻子裸袒，流宂道路。朕甚愍之。今封更始為淮陽王。吏人敢有賊害者，罪同大逆。」</ref>。赤眉将领[[张卬|张卬]]为绝后患，派人将更始帝[[缢|缢]]死。<ref name=Ref1 />

==東汉官方对刘玄的地位认定==
25年[[刘秀|刘秀]]建立[[東漢|東漢]]後，為確立自己為光復漢室的[[正統|正統]]，劉秀及其后代并不认定刘玄是汉朝皇帝，官修《[[东观汉记|东观汉记]]》直接将刘玄归为列传<ref>《[[太平御览|太平御览]]》卷九十引《东观汉记》：刘玄，字圣公，光武族兄也。</ref>。[[张衡|张衡]]提出不同意见，认为光武之前，更始踐祚，号令天下，光武是继更始之后登上天子之位的，《光武帝纪》之前，应有《更始帝纪》，不過朝廷并没有采纳张衡的意见<ref>{{Cite web |url=http://study.ccln.gov.cn/fenke/lishixue/lsxkjs/lszzp/194751-3.shtml |title=《东观汉记》初探(下篇) |access-date=2017-12-22 |archive-url=https://web.archive.org/web/20171222220155/http://study.ccln.gov.cn/fenke/lishixue/lsxkjs/lszzp/194751-3.shtml |archive-date=2017-12-22 |dead-url=yes }}</ref><ref>《后汉书》卷五十九：又更始居位，人无异望，光武初为其将，然后即真，宜以更始之号建于光武之初。书数上，竟不听。</ref>，{{or|因為若采纳张衡的意见，那麼劉秀便不是東漢的建立者，刘玄才是重建漢室的皇帝，這會影響劉秀以至東漢政權的合法性。}}

==陵墓==
漢光武帝刘秀命[[邓禹|邓禹]]将漢更始帝劉玄葬于[[霸陵|霸陵]]（今[[陕西|陕西]][[西安|西安]][[灞橋區|灞橋區]]）。<ref name=Ref1 />

== 家族 ==

*七世祖：漢景帝與侍女[[唐兒|唐兒]]，生長沙定王[[劉發|劉發]]。
*六世祖：劉發，生舂陵節侯[[劉買_(舂陵侯)|劉買]]。
*高祖父：劉買，生舂陵戴侯[[劉熊渠|劉熊渠]]和[[鬱林郡|鬰林]]太守[[劉外|劉外]]。
*曾祖父：劉熊渠，生舂陵孝侯[[劉仁|劉仁]]和蒼梧太守[[劉利|劉利]]。
*祖父：劉利，生[[劉子張|劉子張]]。
*父：劉子張，妻[[何氏|何氏]]，生劉玄和劉騫<ref>《後漢書·安城孝侯傳》[[李賢|李賢]]注引《[[續漢書|續漢書]]》：後十餘歲，亭長子報殺更始弟騫。</ref>。
