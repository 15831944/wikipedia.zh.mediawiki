“'''旅游不文明行为记录'''”是跟据[[中国|中国]][[国家旅游局|国家旅游局]]2016年5月26日出台的《国家旅游局关于旅游不文明行为记录管理暂行办法》，记录中国游客及旅游从业人员不文明行为的档案，意在为推进旅游诚信建设工作，提升公民文明出游意识。
{{维基文库|国家旅游局关于旅游不文明行为记录管理暂行办法}}
[[中国|中国]]于2012年成为世界上第一大旅游消费国，而中国游客出游的不文明行为（例如随意[[吐痰|吐痰]]丢垃圾、大声喧哗、[[丁锦昊事件|在文物上涂写或刻字]]，等）严重损害了中国形象，施行此政策是为了整治旅游不文明行为以提高中国国家和民族形象<ref name=":0">{{Cite web|url =http://hbrb.hebnews.cn/html/2015-05/11/content_58650.htm|title =旅游“黑名单”制度，管用吗？|accessdate =|author =|date =2015年5月11日|publisher =河北日报|deadurl =yes|archiveurl =https://web.archive.org/web/20151006214605/http://hbrb.hebnews.cn/html/2015-05/11/content_58650.htm|archivedate =2015年10月6日}}</ref><ref>{{Cite web|url = http://www.cnta.gov.cn/ztwz/lyzs/lywjckd2q/201507/t20150710_741645.html|title = 中国将会记录游客不文明行为|accessdate = |author = |date = 2015年4月7日|publisher = 旅游外交参考|archive-url = https://web.archive.org/web/20151006231939/http://www.cnta.gov.cn/ztwz/lyzs/lywjckd2q/201507/t20150710_741645.html|archive-date = 2015-10-06|dead-url = yes}}</ref>。
早在2015年4月6日，[[国家旅游局|国家旅游局]]就已出台《游客不文明行为记录管理暂行办法》（5月14日修改为《国家旅游局关于游客不文明行为记录管理暂行办法》<ref name=":1">{{Cite web|url =http://www.cnta.com/zwgk/tzggnew/gztz/201506/t20150627_717592.shtml|title =关于修改《游客不文明行为记录管理暂行办法》的通知（失效）|accessdate =|author =|date =2018年4月18日|publisher =[[中国国家旅游局|中国国家旅游局]]}}</ref>），并设立“'''游客不文明行为记录'''”（俗称“游客黑名单”）。<ref>{{Cite web|url = http://www.cnta.gov.cn/ztwz/lvyzs/lyzs_wmly/201507/t20150708_722552.html|title = 游客不文明行为记录管理暂行办法|accessdate = |author = |date = 2015-07-08|publisher = 中国旅游之声|archive-url = https://web.archive.org/web/20180418225526/http://www.cnta.gov.cn/ztwz/lvyzs/lyzs_wmly/201507/t20150708_722552.html|archive-date = 2018-04-18|dead-url = yes}}</ref>而新的“旅游不文明行为记录”增加了对旅游从业人员的行为约束。新规实施后，原有的《游客不文明行为记录管理暂行办法》即停止实施，而此前建立的“游客不文明行为记录”继续有效，但不再新增记录，新的不文明行为将被记录在“旅游不文明行为记录”。

名单记录信息由所属地方旅游管理部门提交或通过媒体报道和社会举报采集<ref>{{Cite web|url = http://newspaper.jcrb.com/html/2015-04/10/content_183834.htm|title = 《游客不文明行为记录管理暂行办法》正式施行|accessdate = |author = |date = 2015年4月10日|publisher = 检察日报}}</ref>，名单由[[中华人民共和国国务院|中国国务院]]旅游主管部门管理，信息保存期限视情节及影响程度确定<ref>{{Cite web|url = http://www.chinadaily.com.cn/interface/toutiao/1138561/2015-5-10/cd_20643887.html|title = 国家旅游局公布游客不文明行为记录|accessdate = |author = |date = 2015-05-07|publisher = 中国日报网}}</ref>。游客黑名单具有震慑意义，进入“游客黑名单”的游客在此后旅游中将会受到影响，严重则会影响出境、银行信贷等<ref>{{Cite web|url = http://politics.people.com.cn/n/2015/0406/c1001-26803401.html|title = 《游客不文明行为记录管理暂行办法》正式施行|accessdate = |author = |date = 2015年4月6日|publisher = 人民网}}</ref><ref name=":1"></ref>。“黑名单”不是法律法规，更多的是一个道德准则规范<ref>{{Cite web|url =http://www1.cnta.gov.cn/html/2015-4/2015-4-7-11-10-31158.html|title =国家旅游局监管司相关负责人就《游客不文明行为记录管理暂行办法》答记者问|accessdate =|author =|date =2015-04-07|publisher =国家旅游局信息中心}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，而《[[法制晚报|法制晚报]]》刊登了一篇文章《进了“旅游黑名单”，然后呢？》指出记录名单的问题，而多家旅行社表示进入游客黑名单的游客旅游社暂时无权阻止这些游客跟团旅游，同时游客也可以以出行的方式进行个人旅游<ref name=":0" />。
