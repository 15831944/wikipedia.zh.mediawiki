'''旅行團'''是1999年成立於[[中華人民共和國|中華人民共和國]][[廣西壯族自治區|廣西壯族自治區]][[柳州市|柳州市]]的一個男子[[搖滾樂隊|搖滾樂隊]]，2005與[[摩登天空|摩登天空]]簽約後改現名<ref name="旅行團">{{cite web | url=http://ent.sina.com.cn/y/2007-04-05/08371506229.html| title=旗下艺人介绍：旅行团(The LifeJourney)乐队| author=新浪音樂| accessdate=2007-04-05}}</ref>。

== 成員 ==
初期有孔陽、黃子君、韋偉三名穩定成員<ref name="旅行團"/>，2010年1月，鼓手徐彪加入，形成現在的陣容。

== 經歷 ==
1999年10月，樂隊在柳州成立<ref name="旅行團"/>。2005年10月，首隻單曲《The story of sun and I》發表，收錄在《摩登天空5》<ref name="旅行團"/>。同年9月，簽約[[摩登天空|摩登天空]]唱片公司<ref name="旅行團"/>。2008年5月，推出首張專輯《來福膠泥》（Lifejourney）<ref>{{cite web | url=http://ent.163.com/08/0507/17/4BBUIK1R00031H0O.html| title=旅行团乐队武汉为首张专辑《来福胶泥》预热| author=網易娛樂| accessdate=2008-05-07}}</ref>。2009年4月，推出EP《等你吃飯》<ref>{{cite web | url=http://ent.enorth.com.cn/system/2009/04/24/003981027.shtml| title=草莓艺人：旅行团| author=北方網| accessdate=2009-04-28}}</ref>。2011年6月，推出第二張專輯《Wonderful Day》<ref>{{cite web | url=http://ent.163.com/11/0628/02/77JQBPN600032DGD.html| title=旅行团出新片《WonderfulDay》 举办首发演出| author=網易娛樂| accessdate=2011-06-28}}</ref>。2013年12月，推出第三張專輯《於是我不再唱歌》<ref name="北京專場">{{cite web | url=http://ent.163.com/14/0303/14/9MDVICQK00031H0O.html| title=旅行团乐队北京专场爆棚 告别摩登自建工作室| author=網易娛樂| accessdate=2014-03-03}}</ref>。2014年在新專輯的北京首唱會上，宣佈與[[摩登天空|摩登天空]]合約滿後，不再續約，自立門戶<ref name="北京專場"/>。
