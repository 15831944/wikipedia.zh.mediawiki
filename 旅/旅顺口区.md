'''旅顺口区'''，是[[辽宁省|辽宁省]][[大连市|大连市]]的一个[[市辖区|市辖区]]，原为[[旅大市|旅大市]]下辖的[[旅顺市|旅顺市]]。面积506平方千米，人口21万。邮政编码116041。区人民政府驻水师营街道小南村。[[旅顺经济开发区|旅顺经济开发区]]在本区西部。该区也是[[北海舰队|北海舰队]]驻扎地。

该区位于[[辽东半岛|辽东半岛]]最南端，东临[[黄海|黄海]]、西濒[[渤海|渤海]]，南与[[山东半岛|山东半岛]]隔海相望，距大连市区45公里。旅顺口区是'''国家级风景名胜区、国家级自然保护区、国家级森林公园'''。境内有举世闻名的天然[[不冻港|不冻港]]——[[旅顺港|旅顺港]]，为京津海上门户和东北的天然屏障。新开辟的[[旅顺新港|旅顺新港]]是沟通辽东半岛和山东半岛的“黄金水道”。

南部的旅顺港是[[北太平洋|北太平洋]]地区的重要海军基地，最早由[[清朝|清朝]]开始营建。后先后为[[沙俄|沙俄]]、[[日本|日本]]和[[苏联|苏联]]所占。[[闻一多|闻一多]]的《[[七子之歌|七子之歌]]》中，旅顺口即是七子之一。

== 历史 ==
*参见：[[旅顺市|旅顺市]]、[[旅顺港|旅顺港]]。

===古代===
*[[漢朝|漢朝]]稱作'''牧羊城'''；
*[[三國|三國]]稱作'''沓渚'''或'''遝津'''；
*[[晋朝|晋朝]]称作'''马石津'''；
*[[唐代|唐代]]称作'''都里镇'''；
*[[元代|元代]]命名'''狮子口'''，现在[[旅顺军港公园|旅顺军港公园]]有醒狮雕塑；
*[[明代|明代]]是隶属于金州卫的一个海防哨所，称为金州中左所。明朝取“旅途顺利”之意，命名此地为“旅顺口”。

===北洋舰队===

旅顺口曾设[[旅顺口水师营|旅顺口水师营]]。在近代，则成为著名的[[海軍基地|海軍基地]]，西方人称之为“[[亚瑟港|亚瑟港]]（英语：Port Arthur, 俄语：Артур)”。1880年代，[[李鸿章|李鸿章]]在此经营多年，建成[[北洋舰队|北洋舰队]]的主要基地。现[[北海舰队|北海舰队]]五个军用港口，都在[[旅顺军港|旅顺军港]]内。

===甲午战争===
在[[中日甲午战争|中日甲午战争]]（1894年－1895年）中，旅顺被日军攻占。[[旅順大屠殺|旅順大屠殺]]在此发生。日軍於1894年11月攻陷旅順後，對市內居民連續四天進行殺戮，超過二萬人遇害，屠殺過後全市僅剩三十六人被留下埋葬居民屍體，遇難者葬於白玉山東麓的安葬崗，今稱「[[萬忠墓|萬忠墓]]」。

按照[[马关条约|马关条约]]，清朝割让包括旅顺在内的[[辽东半岛|辽东半岛]]给日本。此举引发其他列强的睽觑。在俄国、德国、法国等国的斡旋下（“[[三国干涉|三国干涉]]”），日本同意由清政府出钱赎回辽东半岛。

===俄国租借时期===
1898年3月27日，沙俄以干涉还辽有功，迫使清政府与之签订了《[[旅大租地条约|旅大租地条约]]》，规定沙俄租借军港旅顺口、商港大连湾25年。稍后又于当年5月7日，再次迫使清政府与之签订《旅大租地续约》。1900年，义和团事变中，俄国占领了东北全境。1903年，从哈尔滨到旅顺、纵贯整个满洲的[[东清铁路|东清铁路]]南满支线全部完工。俄国租借旅顺的7年期间，给旅顺留下了一批俄国风格的建筑。旅顺发展为大城市，人口超过了1万人。<br />

===日俄战争和日本占领时期===

俄国抢占日本通过[[马关条约|马关条约]]获得的利益，独占[[满洲|满洲]]，并进而威胁[[朝鲜|朝鲜]]，引发日本强烈不满，日本受此羞辱，卧薪尝胆，富国强兵。1902年又缔结了[[英日同盟|英日同盟]]。1904年2月8日日军偷袭旅顺口，[[日俄战争|日俄战争]]（1904年－1905年）爆发。经过为期五个月的[[旅順會戰|旅顺口攻城战]]后，日军夺取旅顺口军港。此后至1945年8月，旅顺港一直由日本占领。

===苏联占领时期===

[[第二次世界大战|第二次世界大战]]结束的1945年，根据[[雅尔塔协定|雅尔塔协定]]，[[旅顺港|旅顺港]]由[[苏联|苏联]]占用。[[八月风暴行动|八月风暴行动]]后，苏军占领旅大地区。同年11月25日，设置[[旅顺市|旅顺市]]政府。[[中华民国政府|中华民国政府]]在1947年，核准设置旅顺市，属[[遼寧省_(中華民國)|辽宁省]]。但中华民国政府始终未能收复这一地区。此地实质由苏联和[[中国共产党|中国共产党]]管辖。中国共产党获得苏联的支持，由此在旅顺口建立起后方基地，是[[第二次国共内战|第二次国共内战]]初期，中国共产党在[[中国东北地区|东北地区]]唯一可以使用的港口。期间，中国共产党开辟了从旅顺港途经[[朝鲜半岛|朝鲜半岛]]至山东[[荣成县|荣成]][[俚岛|俚岛]]的运输线，将武器、粮食由[[东北解放区|东北解放区]]运至[[山东半岛|山东半岛]]，影响了[[华东地区|华东]]、[[华北地区|华北地区]]的战局<ref>{{Cite web |url=http://news.ifeng.com/history/zhongguojindaishi/detail_2013_04/02/23801141_0.shtml|title=揭秘：淮海战役中大连生产的炮弹如何运往山东|accessdate=2017-07-14 |author=作者：周海涛、张超 |date=2013-04-02 |work= |publisher=凤凰网，来源：百年潮|language=Zh-hans }}</ref>。

1949年6月，中华民国政府实行[[关闭政策|关闭政策]]，封锁大陆沿海港口时，避开了旅顺港。1949年10月，[[中华人民共和国|中华人民共和国]]成立，成为苏联为首的[[东方阵营|东方阵营]]一员。

===中华人民共和国时期===

1950年2月，中国政府和苏联政府签订《[[中苏友好同盟互助条约|中苏友好同盟互助条约]]》的同日，签订《[[:s:中苏两国关于缔结友好同盟互助条约及协定的公告#关于中国长春铁路、旅顺口及大连的协定|关于中国长春铁路、旅顺口及大连的协定]]》。协议约定在1952年底撤出旅顺口，而随着[[朝鲜战争|朝鲜战争]]爆发，地区局势日趋紧张。当时，[[中国人民解放军海军|中国海军]]尚无力量守卫旅顺港。1952年9月15日，中苏政府交换《关于延长共同使用中国旅顺口海军基地期限的换文》。1954年，苏联领导人[[尼基塔·謝爾蓋耶維奇·赫魯曉夫|赫鲁晓夫]]决定将旅顺口交还中国。1955年初开始交接，中国方面，由此设立[[中国人民解放军海军旅顺基地|海军旅顺基地]]。5月，两国完成交接，月底12万苏军撤离<ref>{{Cite web |url=http://www.sohu.com/a/198735350_114825 |title=揭秘苏联归还旅顺海军基地内幕|accessdate=2018-09-18 |author=沈志华 |date=2017-10-18 |work= |publisher=搜狐网，来源：《百年潮》2003年第2期|language=Zh-hans }}</ref>。

1960年前，属[[旅大市|旅大市]]。1960年1月18日，[[中华人民共和国国务院|中国国务院]]撤消[[旅顺市|旅顺市]]，设旅顺口区<ref>{{Cite web |url=http://www.lndangan.gov.cn/lnsdaj/whdgy/lndsj/content/bb45444542f9f74e0142fa0465910070.html |title=1960年辽宁大事记|accessdate=2018-09-19 |author= |date=2013-12-16 |work= |publisher=辽宁省档案信息网|language=Zh-hans }}</ref>。旅大市改为[[大连市|大连市]]后，仍属之。当时有三条公路连接大连市。

===地形===
旅顺口区位于辽东半岛最南端,属大连市辖区。东临黄海、西濒渤海，南与山东半岛隔海相望，北依大连，距大连市区45公里。
    
面积：陆地南北纵距26.1公里，东西横距31.2公里，总面积506.8平方公里，海岸线长169.7公里。

地势：全境属长白山余脉构成的沿海丘陵地带，东高西低，平均海拔140米，地形构成为六丘半水三分 半田。共有山丘292座，最高老铁山海拔465.6米，直临黄渤海分界线，被称为辽宁的"天涯海角"。境内河流多为间歇性小河。农田耕地多处在15度左右 的缓坡上，少量在丘陵间盆地或沿海河谷地带。

===气候===
[[温带季风气候|温带季风气候]],四季分明，冬夏长、春秋短，日光充足，雨量适中。

==人口 ==
2007年，全区共有居民82627户，比上年增加1468户；户籍人口208874人，增加1255人，其中，男性103504人，女性105370人，人口性别比为98.23：100（女性人口为100）。与上年相比，青少年人口比重下降2.7个百分点，老年人口比重上升0.6个百分点；迁入人口2654人，迁出人口305人，人口机械增加2349人；非农业人口141188人，比上年净增8859人，城市化率由63.7%推进到67.6%。

==经济==
初步核算，2007年全区生产总值完成92.5亿元，按现行价格计算，比上年增长23.3%。三次产业的比例为9.2%、48.5%、42.3%。人均地区生产总值，按户籍平均人口计算为44430元，按年末汇率测算达到6000美元，比上年增长29.6%。<ref>[http://www.dllsk.gov.cn/info.asp?classid=1&itcn_B_ClassID=32 经济发展]中国大连旅顺口区政务信息网 与2010-04-27查阅</ref>
