'''旅運中心站'''位於[[高雄市|高雄市]][[苓雅區|苓雅區]][[苓雅寮|苓雅寮]]，為[[高雄捷運黃線|高雄捷運黃線]](預定)及[[環狀輕軌|環狀輕軌]]的捷運車站。

== 車站概要 ==
{{高捷環狀輕軌}}：定位為「特色站」，設置二座側式月台。<br>
{{高捷黃線}}：地下車站，月台形式、出口數目及其他設施皆未定。

== 車站構造 ==

== 利用狀況 ==
車站位於苓南前路、海邊路交會處南側。周邊多為重劃之空地圍籬，再往北側為住宅群；南側為正在興建中的[[高雄港埠旅運中心|高雄港埠旅運中心]]工地。相較[[高雄捷運|高雄捷運]]其他車站，本站目前缺乏公共設施與生活機能，亦是[[環狀輕軌|環狀輕軌]]系統中使用人數較少的車站。<ref>[http://news.ltn.com.tw/news/local/paper/1115810 〈南部〉高雄輕軌車站「不方便」 C9站公廁難尋]，自由時報，2017-07-03。</ref>

== 歷史 ==
===高雄環狀輕軌第一段C1-C14車站全民參與命名活動===
* 命名活動第一階段（2014年10月18日至11月20日）建議站名：三多苓南站、三多永平站、旅運中心站
* 命名活動第二階段（2015年2月16日至3月6日）票選選項：旅運中心站、苓雅寮站、三多苓南站
* 命名結果（2015年3月29日公告）：旅運中心站<ref>[http://www.thenewslens.com/post/144007/ 高雄環狀輕軌C1-C14站有名字了！最快8月通車]</ref>
* 2017年6月30日:隨著環狀輕軌第一階段C8站-C12站完工試營運啟用<ref>[https://udn.com/news/story/7266/2548739/ 高雄輕軌最美愛河段 30日通車]</ref>
