'''旅顺船坞旧址'''或称'''旅顺大坞'''<ref name="a1">{{Cite web |url=http://liaoning.nen.com.cn/system/2017/11/17/020195669_01.shtml |title=大连：袁保龄用花岗岩修建旅顺大坞|accessdate=2018-09-18 |author=郭平 |date=2017-11-17 |work= |publisher=辽宁日报|language=Zh-hans }}</ref>，位于[[辽宁省|辽宁省]][[大连市|大连市]][[旅顺口区|旅顺口区]]港湾街58号，在[[大连辽南船厂|大连辽南船厂]]（亦称中国人民解放军海军4810工厂）厂区内<ref name="a2">{{Cite web |url=http://www.sohu.com/a/220845812_656969 |title=这些船厂厂区入选中国工业遗产保护名录，你知道几个？|accessdate=2018-09-18 |author= |date=2018-02-04 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。

船坞是[[清政府|清政府]]在1880年代为[[北洋水师|北洋水师]]所兴建的。当时，清政府在[[旅顺口|旅顺口]]为北洋水师修建[[旅顺军港|军港]]，以作保障，由此修建了一系配套工程。船坞是主要工程之一，1890年工程竣工<ref name="a1"/>。当时被称“远东第一大坞”，是中国近代船舶工业的“四局二坞”之一。建有[[中国东北地区|东北地区]]第一条国内电报线，中国第一条国际电报线和第一条自来水管线，是东北工业的萌芽之地。船坞随旅顺港先后被[[俄罗斯帝国|俄罗斯]]、日本占领。1945年，[[日本无条件投降|日本无条件投降]]之后，被[[苏联|苏联]]接管。1955年，中国收回[[旅顺港|旅顺港]]。至今仍为中国人民解放军海军服务<ref>{{Cite web |url=http://www.jiaotongwang.cn/liaoning/meitiguanzhu/20171211/100797.html |title=旅顺船坞入选首批国家工业遗产拟认定名单|accessdate=2018-09-18 |author= |date=2017-12-11 |work= |publisher=中国交通在线，来源：半岛晨报|language=Zh-hans }}</ref>。

2013年，[[中华人民共和国国务院|中国国务院]]将之以'''旅顺船坞旧址'''之名列为[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]中的（[[第七批全国重点文物保护单位中的近现代重要史迹及代表性建筑|近现代重要史迹及代表性建筑]]）。2017年时，旅顺口区人民政府与大连辽南船厂为旅顺船坞申报中国工业遗产<ref>{{Cite web |url=http://news.dayoo.com/society/201709/25/140000_51816448.htm |title=旅顺船坞正申报国家工业遗产 已有100多年历史|accessdate=2018-09-18 |author= |date=2017-09-25 |work= |publisher=大洋网，来源：大连日报|language=Zh-hans }}</ref>。2018年1月，包括船坞、木作坊、吊运库房、船坞局办公楼、电报局、泵房等在内的遗存以'''旅顺船坞'''之名列入[[中国工业遗产保护名录|中国工业遗产保护名录]]第一批名单<ref name="a2"/>。
