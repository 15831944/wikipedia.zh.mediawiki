'''旅顺市'''是1945年，[[苏军|苏军]]、[[中共政权|中共政权]]在[[旅顺口|旅顺口]]（今[[辽宁省|辽宁省]][[大连市|大连市]]境）建立的行政区。后，[[中华民国政府|中华民国政府]]亦以[[省辖市|省辖市]]的名义设置旅顺市，未有实际管辖。

1945年[[八月风暴行动|八月风暴行动]]中，苏军进入[[中国东北地区|中国东北]]。8月15日，中华民国政府与[[苏联政府|苏联政府]]签订《[[中苏友好同盟条约|中苏友好同盟条约]]》，包括旅顺口所在的，今大连市的南部地区成为苏军驻地，为旅顺口海军根据地。租期30年。此后旅大地区名义为中国领土，实为苏联军事管制区，中华民国政府未曾实际管辖。在苏联的掌控下，[[中共政权|中共政权]]于1945年11月8日，成立[[大连市政府|大连市政府]]。11月25日，成立旅顺市政府<ref>{{Cite web |url=http://ln.news.163.com/06/1118/13/307C8G330052006I.html |title=史海钩沉民国时期：大连政权性质最复杂(图)|accessdate=2018-09-19 |author= |date=2006-11-18 |work= |publisher=网易，来源：东北新闻网　|language=Zh-hans }}</ref>。[[旅顺港|旅顺港]]所在，亦是中共政权无法管辖的地区<ref>{{Cite web |url=http://www.sohu.com/a/198735350_114825 |title=揭秘苏联归还旅顺海军基地内幕|accessdate=2018-09-19 |author=沈志华 |date=2017-10-18 |work= |publisher=搜狐网，来源：《百年潮》2003年第2期|language=Zh-hans }}</ref>。至1955年，苏军方才完全从旅大地区撤离。

1947年6月，中华民国政府公布[[東北新省區方案|東北新省區方案]]，由[[金县_(中华民国)|金县]]析置旅顺市，[[治所|治所]]在旅顺口（今[[辽宁省|辽宁省]]大连市西南[[旅顺口区|旅顺口区]]），属[[遼寧省_(中華民國)|辽宁省]]。

1949年4月，属[[旅大行署区|旅大行署区]]。中华人民共和国建立后，属[[旅大市|旅大市]]（旅大直辖市）。1950年2月，中华人民共和国政府和苏联政府签订《[[中苏友好同盟互助条约|中苏友好同盟互助条约]]》的同日，签订《[[:s:中苏两国关于缔结友好同盟互助条约及协定的公告#关于中国长春铁路、旅顺口及大连的协定|关于中国长春铁路、旅顺口及大连的协定]]》。协议约定旅顺口地区的民事行政应直属中华人民共和国政府管辖。1955年，苏军完全撤军。[[旅顺港|旅顺港]]在内的区域回归中国。1960年1月18日，[[中华人民共和国国务院|中国国务院]]撤消旅顺市，设旅顺口区<ref>{{Cite web |url=http://www.lndangan.gov.cn/lnsdaj/whdgy/lndsj/content/bb45444542f9f74e0142fa0465910070.html |title=1960年辽宁大事记|accessdate=2018-09-19 |author= |date=2013-12-16 |work= |publisher=辽宁省档案信息网|language=Zh-hans }}</ref>。
