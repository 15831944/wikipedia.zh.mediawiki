'''算額'''（{{Jpn|kn=さんがく|rm=Sangaku|j=算額}}），是指在日本[[江户时代|江戶時代]]書寫在[[匾額|匾額]]或[[繪馬|繪馬]]的[[和算|和算]]問題及其解答。奉納算額的意義有三種：感謝[[神|神]][[佛|佛]]的恩賜，表示對[[和算|和算]]教師的尊崇，以及展示自己的研究成果。

因為神社和寺廟是當時交流的一個最佳場所。因此，算額可以存在很高的關注度，也能引起有興趣人士的探討和共鳴{{sfn|蘇|2007}}。其不僅僅是一些和算家會掛出算額，一些和算愛好者也會懸掛。而在算額上所書寫的數學問題，比起[[代數|代數]]問題是[[幾何|幾何]]問題居多。至於典型的算額題目則是求邊長或者圓的直徑，其中當然也包含了[[直線|直線]]、[[三角形|三角形]]、[[內切圓|內切圓]]和圓周長等問題{{sfn|日経サイエンス|2016}}，且多為討論[[圓|圓]]、[[橢圓|橢圓]]、各種多邊形之間的相容、[[相切|相切]]關係的幾何題{{Sfn|徐|2007|p=73}}。

江戶時代中期，日本全國推廣「奉獻」的價值觀，而算額就是和算人士之奉獻精神的產物。算額在江戶時代最為鼎盛，而至[[昭和|昭和初期]]後逐漸沒落。近年来，各個地方又都重新審視了算額的價值。接受算額的[[寺社|寺社]]逐漸增加，而奉納算額的人亦有所增加。

== 算額的樣式 ==

一塊算額包括1～10個不等的問題，在整塊算額通常前面的部分是圓或三角形等幾何圖形構成的圖，而後是：題目、答案及解法。位於其後方的則是和算流派、其教師、展示者的名稱、奉獻的日期。題目通常會選擇較為複雜的幾何問題，並輔以彩繪圖形{{sfn|国立国会図書館|2017}}。

在古代數學文本中，除了題目和答案之外，還有所謂的「術」，就是我們今日的解題方法。但通常「術」只是幾句話的公式，真正的解答還是需要讀者自行推敲{{sfn|蘇|2007}}{{Sfn|蘇|2003|p=73}}。

== 意義 ==
在日文中，「算」字有[[计算|計算]]、[[数学|數學]]的意涵，「額」指的就是木製的書版。{{Sfn|蘇|2006}}

奉納算額的意義有三種{{sfn|三重県環境生活部文化振興課県史編さん班|1995}}：

# 感謝神佛的恩賜；
# 表示對[[和算|和算]]教師的尊崇；
# 展示自己的研究成果。

在[[江戶時代|江戶時代]]，日本人會設計各種式樣的匾額到鄰近的寺廟或神社以對神佛表達謝意。而其中，[[和算|和算家]]是把數學問題和答案用汉语[[文言文|文言文]]書寫在板上，以表神佛使自己能解決和算問題的謝意，而這類繪馬就被稱為「算額」。{{Sfn|蘇|2006}}

當時日本的[[神社|神社]]和[[寺廟|寺廟]]是社會交流的平台；因此，算額會有很高的關注度，也能引起有興趣人士的共鳴和探討{{sfn|蘇|2007}}，算額於當時相當於是數學期刊。其不僅僅是和算家會掛出算額，一些和算愛好者亦會懸掛{{Sfn|徐|2007|p=72}}。

算額上的問題通常不會寫出完整的解答過程：一方面是由於繪馬的書寫面積所致；另一方面是由於算額上的這些題目是用來訓練學生數學實作的素材{{Sfn|蘇|1999|p=14}}。

學習與研究和算的人最初是為了自己能夠順利地進行數學研究，以及數學能力不斷提高而向神佛祈願，向神社佛閣奉納[[繪馬|繪馬]]；當然也包括對自己解數學問題因而感謝神佛恩賜而還願的算額，其目的在於勉學。以後演變成各種願望都有：有祈願家庭安全的、有祈願學術私塾繁榮的、有祈願子孫出生的，而最多的依然是祈願自己能夠解難題、能夠構造好的數學問題，或者是為炫耀自己數學能力的優秀。{{Sfn|徐|2012|p=48}}

一般都是在繪馬板上寫出數學問題，其中可能會要求讀者給予解答，亦有奉納者自己給出解答的{{Sfn|徐|2012|p=48}}。和算家會通過算額問題進行學術交流與[[辯論|學術辯論]]，而和算學派中的[[最上流|最上流]]與[[關流|關流]]之間的爭論最為顯著{{Sfn|徐|2007|p=72}}。

== 歷史與軼聞 ==
算額最早出現於何時，今已不可考。但其名稱最早可追溯至江戶幕府末期的《道中日記》{{Sfn|徐|2012}}。

算額奉納的風俗是日本獨有的文化。其中，一部分的算額掛馬還是被認證為日本的[[重要文化財|重要文化財產]]和[[日本文化财产|民俗文化財產]]。在[[明治時期|明治時期]]，[[數學|數學]]的概念正式從西方引入日本；而算額奉納的風俗亦使得數學被引入過程更為容易{{sfn|国立国会図書館著|2017}}。

據[[村瀨義益|村瀨義益]][[延寶|延寶]]八年（西元1681年）的《算學淵底記》記載，17世紀中期在江戶的各地都有奉納算額，十分之鼎盛；而書中亦詳細描述了{{Tsl|ja|目黒不動尊|目黑不動尊}}的{{Tsl|ja|瀧泉寺|瀧泉寺}}的算額。据估計[[京都市|京都]]、[[大阪市|大阪]]的算額歷史已經十分悠久。在17世紀下半葉，日本出現了收集算額上題目的問題集書目。而日本最早出版的書籍，就是[[寬政|寬政]]元年（西元1789年）{{Tsl|ja|藤田貞資|藤田貞資}}記錄算額問題的書籍——《神壁算法》。江戶時代中期，日本全國推廣「奉獻」的價值觀，其中特别是寬政、[[享和|享和]]、[[文化_(年號)|文化]]、[[文政|文政]]的精神，而算額就是和算人士之奉獻精神的產物。據傳當時一年有超過100枚算額無償奉獻以用於在寺社進行和算教學。進入[[明治|明治時代]]後，算額文化依然有被傳承下來，但直至[[昭和|昭和初期]]後就已經休止了。{{sfn|国立国会図書館著|2017}}

依據《賽祠神算》記載，[[關流|關流]]第五代傳人——石田玄圭的門人大澤熊吉在天滿宮奉納了一枚算額。同年的十月，[[最上流|最上流]]的大川榮信亦在同一地點奉納算額。而後者的題目與前者雷同，但解法則較前者優良。於是就展開了[[關流|關流]]和[[最上流|最上流]]之間的流派戰爭<ref group="註">另有一說是派別名稱導致的流派爭議，詳見[[最上流|最上流]]條目。</ref>。{{Sfn|蘇|1999|p=14}}

=== 對於日本的數學教育 ===
數學愛好者將算額作為一種遊藝，也有一些人還通過遊歷以數學為生計，江戶時代就出現所謂的「遊歷算家」。他們到處旅行指導數學，首先去寺社調查有無算額，如果有就再去拜訪奉納者；如果找不到這些奉納者，就去找一些莊主、名主和富裕的農民，說自己是數學家，如果這個村莊有對數學感興趣的年輕人，自己就可以教他們，於是開辦數學私塾。遊歷算家對日本江戶時代的數學教育與數學普及起了很大的作用{{Sfn|徐|2007|p=73}}。

奉納算額風俗的興起還與江戶初期的數學著作中的「遺題承繼」風氣有關。遺題承繼是江戶初期十分流行且獨特的數學文化現象，類似今天的數學問題徵解，很多算書都提出自己的問題或是對前人著作中遺題的解答。遺題承繼這種形式使和算知識的承傳演變成帶有競技性質，推動了和算的普及與發展。 隨著和算於十八世紀中後期的普及化，算額奉納的風氣漸盛並於十九世紀達到高峰；但到了[[明治維新|明治維新]]時代則逐漸式微。這主要是因為當時日本的教育體系決定採用西方數學作為學習的內容，於是和算走進歷史且奉獻的算額數量也逐年銳減。不過，有些地區仍維持這項傳統到20世紀初期。{{Sfn|黃|2014|p=53}}

近年来，各個地方都重新審視了算額的價值。而即使在21世紀，仍有部分神社、寺廟接受算額奉納{{Sfn|NIKOLI|2008}}；而奉納算額的人亦有所增加。雖然這並非直接繼承和算的傳統的方式，但亦有人聲稱這是一種表達日本人表達對算術之熱愛的文化現象{{sfn|蘇|2007}}。而現代日本數學史研究者在研究算額時經常發現算額中很多問題，其本質就是後來西方數學家所發現的幾何定理{{Sfn|徐|2007|p=74}}。例如，懸掛於[[福井縣|福井縣]][[鯖江市|鯖江市]]舟津神社的算額的幾何問題，其本質是{{Tsl|en|Descartes' theorem|笛卡兒定理}}和{{Tsl|en|Soddy's hexlet|六球連鎖定理}}{{Sfn|徐|2007|p=74}}。

另外，亦有日本中學教師嘗試在平面幾何的教學中融入算額文本。他們所採用的方式是在教授完幾何圖形的基本性質、定理的證明之後，於相關例題的應用上提供算額的題目，讓學生以上課所學習到的平面幾何算法求解{{sfn|蘇|2007}}。

記載中最早的算額出現於[[明曆|明暦]]3年（西元1657年）。據[[貞享|貞享]]元年（西元1684年）出版的《増補·算法闕疑抄》記載，那枚算額是由在[[二本松城|二本松城]]下開[[私塾|私塾]]的初坂重春所奉納的；與此同時，在現在的[[白河市|白河市]]明神前，亦有廣部俊陳的門人奉納給堺明神算額一枚。{{sfn|福島県和算研究保存会|1982|p=129}}

根据1997年進行的調查結果顯示，日本全境共有975枚的算額保存至今{{sfn|深川|1998}}。在這些算額中，銘刻年份最為久遠的供奉在[[櫪木縣|櫪木縣]][[佐野市|佐野市]]的{{Tsl|ja|星宮神社|星宮神社}}，其算額上銘刻年份為[[天和_(日本)|天和]]三年（西元1683年）。但其由於遭遇火災導致其表面被毀，難以閱讀。所以亦有人將供奉於[[京都市|京都市]][[北野天滿宮|北野天滿宮]]的[[貞享|貞享]]三年（西元1686年）的算額，視為最為古老的算額。{{Sfn|中村|伊藤|大谷|小林|2000}}

關於算額的分佈情況，主要是在[[东北地方_(日本)|東北地方]]較多，尤其是[[東京|東京]]（[[江戶|江戶]]）、[[福島縣|福島縣]]和[[岩手縣|岩手縣]]等地，這可能是受[[会田安明|会田安明]]及其門派——[[最上流|最上流]]的影響。根據其算額上的信息，有學者就推測「從[[江戶時代|江戶時代]]到[[明治時代|明治]]初期的日本人的數學水平相當高」；也可以推斷出當時除了和算家對算額甚是喜愛外，「和算愛好者亦是對此情有獨鍾」。{{sfn|上垣|1684}}

單一地方被確定擁有最多算額的，則是[[爱媛县|愛媛縣]][[松山市|松山市]]的{{Tsl|ja|伊佐爾波神社|伊佐爾波神社}}，共有22枚算額。22枚算額中最為久遠的是在享和3年（1803年）時，由丸山良玄的學生——大西佐兵衛義全奉納的{{sfn|イムジイのページ|2003}}；而最新的是在昭和12年（1937年）時，由村上先生的學生——中村正教奉納的。而{{Tsl|ja|伊佐爾波神社|伊佐爾波神社}}將其神社內的算額，紀錄在《道後八幡伊佐爾波神社的算額》（{{Lang|ja|『道後八幡伊佐爾波神社の算額』}}）{{sfn|伊佐爾波神社|2005}}中，並在神社內發行。

== 算額的價值 ==
由於算額的歷史價值，部分算額被列為[[日本文化财产|文化財]]。其所擁有的不單單是和算傳播的載具，還能通過其來一窺那個時代的民間活動，而且算額還有很高的藝術價值{{sfn|豬苗代観光協会|2016}}。
