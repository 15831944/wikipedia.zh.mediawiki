'''算法少女'''是一本出版于1775年（[[安永_(年號)|安永]]4年）的[[和算|和算]]书籍。本书是日本传统算学书籍中唯一一本以女性名义编写的。

本书在日本[[国立国会图书馆|国立国会图书馆]]有扫描本公开<ref name="ndl"/>。在1935年曾由古典数学书院发行誊写版<ref name="koten"/>。2009年[[筑摩书房|筑摩书房]]出版了《阅读和算书“算法少女”》，其中收录了《算法少女》原文的影印版以及现代日语翻译、解法的解说<ref name="kotera2009"/>。

以本书为题材，[[儿童文学|儿童文学]]作家{{link-ja|远藤宽子|遠藤寛子 (作家)}}创作了同名小说《{{link-ja|算法少女 (小説)|算法少女 (小説)|算法少女}}》。<ref name="endo1973"/><ref name="endo2006"/>

== 作者 ==
根据本书的序文，本书是在父亲的协助下由女儿编写的，使用了笔名。父亲自称壶中隐者，俳人谷素外所写后记中称其为“医师平氏”，女儿署名平氏，印章为“章子”<ref name="ndl"/><ref name="kotera2009"/>。当时的和算书籍常常是以弟子的名义发表老师的研究内容，故而可以推断本书事实上的作者应该是壶中隐者<ref name="mikami1934"/>。不过也有观点认为，本书实际作者是女儿章子。不论如何，本书作为仅有的一本作者为女性的古代和算书籍<ref name="mikamisouwa"/><ref name="endo chikuma"/>，在日本文化史上也是有贵重价值的。<ref name="endo2005"/>{{rp|21}}

壶中隐者的身份在本书刊行后一个半世纪中并不为人所知，但在1930年代确认为医师千叶桃三<ref name="mikami1934"/><ref name="meijizen5"/>{{rp|517,518}}<ref name="meijizen4"/>{{rp|583}}。

== 内容 ==
[[File:算法少女第三问.jpg|thumb]]
本书以文言文和日文书写。分为序言（父、女）、圆周率的历史、正文（上、中、下）、圆周率求法、后记5部分。其中序言中父亲名义的部分、圆周率历史的最后部分、正文的中篇、圆周率求法为文言文写成，其余部分为日文<ref name="ndl"/>。大体上文言文的部分为以父亲名义所写，日文的部分为女儿名义<ref name="endo2005"/>{{rp|21}}。

书中以女儿名义记述了父亲进行数学研究的履历、当时大阪数学界的情况以及自己向父亲学习数学的情形。<ref name="endo2005"/>{{rp|21,22}}

书中提到了圆周率的值和求解方法。古时认为圆周率为3。之后是3.16。进而是更精确的355/113，该值的求法是在圆里内接正四边形、正八边形、正十六边形……直到正十余万边形，并写明该算法出自[[关孝和|关孝和]]的《括要算法》。最后书中给出了一种不通过内接正多边形，直接求圆周率值的方法。壶中隐者称此术玄妙，为了避免像古人一样“人死技绝”，故而著书立说“与天下后世共之也”<ref name="ndl"/>。<ref name="endo2005"/>{{rp|22}}

== 和算家的评论 ==
本书出版后16年，1791（[[宽政|宽政]]3年），数学家{{link-ja|藤田贞资|藤田貞資}}撰写了《算法少女之评》<ref name="fujita"/>一文批评《算法少女》。他认为，书中所述圆周率的求法抄袭自[[有马赖徸|有马赖徸]]著于1769年的和算书《拾玑算法》。<ref name="endo2005"/>{{rp|23}}{{efn|藤田贞资曾经是有马赖徸的家臣，受其资助{{r|meijizen2|heibon|page1=91,92}}}}

数学家[[会田安明|会田安明]]在1804年的《算法约术三条之题》<ref name="aida"/>中认为《算法少女》书中的圆周率求法并非抄袭自《拾玑算法》。<ref name="endo2005"/>{{rp|23}}

== 衍生作品 ==
以本书为题材，[[儿童文学|儿童文学]]作家{{link-ja|远藤宽子|遠藤寛子 (作家)}}于1973年发表了同名小说《{{link-ja|算法少女 (小説)|算法少女 (小説)|算法少女}}》，并于次年获得了[[产经儿童出版文化奖|产经儿童出版文化奖]]<ref name="sankeijidou"/>。小说的主人公设定为本书的写作者，书中名为“{{lang|ja|千葉あき}}”（罗马字：Chiba Aki），あき为“章”字的训读<ref name="endo2005"/>{{rp|21}}。

小说《算法少女》又进一步被改编为同名漫画和动画。漫画《算法少女》连载于2010-2014年，于2012年发行[[单行本|单行本]]1卷<ref name="tsutaya manga"/>。动画《算法少女》制作始于2009年，在2016年发布<ref name="production note"/><ref name="tsutaya anime"/>。
