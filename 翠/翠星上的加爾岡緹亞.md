《'''翠星上的加爾岡緹亞'''》（{{lang-ja|翠星のガルガンティア}}）是[[Production_I.G|Production I.G]]所制作的日本電視動畫，於2013年4月開始播放；連同影碟限定特別篇在內，全篇故事合共為15集。

[[OVA|OVA]]《遙遠的巡迴航路》前篇和後篇分別於2014年9月及2015年4月發表，並安排在電影院作限定上映。<ref>{{Cite web|url=http://comic.qq.com/a/20140618/059483.htm |title=《翠星》新剧场动画详情 9月前篇先行 |work=騰訊動漫 |date=2014-06-18}}</ref><ref>{{Cite web|url=http://comic.qq.com/a/20140707/035517.htm |title=《翠星》OVA版曝光新图和剧场用特报 |work=騰訊動漫 |date=2014-07-07}}</ref>於後篇推出後，官方人員突然公佈原定電視動畫第二季因為主要投資公司[[Bandai_Visual|Bandai Visual]]叫停，而在前期製作階段難產。已完成之劇本將改以小說形式，在2015年夏季推出，標題為「翠星上的加爾岡緹亞〜遙遠、邂逅的天地〜」共2集。<ref>{{lang|ja|[http://dengekionline.com/elem/000/001/033/1033754/ OVA『翠星のガルガンティア』前夜祭レポ！ 幻のTVシリーズ第2期の構想は、2015年夏に小説になって登場]. 電撃オンライン. 2015-04-04 [2015-04-05]}} {{ja}}.</ref>

== 故事簡介 ==
故事讲述在遥远的未来，人类为了生存，不断地与怪异的生命体「黑迪亞斯」（{{lang|ja|ヒディアーズ}}）对抗。在一场战斗中，人类银河同盟少尉莱德和他的人形兵器「Chamber」<ref>[http://www.my-cartoon.com.tw/my101/about.php?id=82 翠星上的加爾岡緹亞] {{webarchive|url=https://web.archive.org/web/20130803060028/http://www.my-cartoon.com.tw/my101/about.php?id=82 |date=2013-08-03 }}. My101.</ref>{{註|[http://www.plurk.com/p/io980b -{翠星(TVB)：突擊者.....は？（威圧）}-]. 據TVB譯製人員在[[Plurk|Plurk]]表示，曾提出異議被打回票仍採代理商譯名；並透露「cham-{}-ber」取自槍膛，即預先裝彈、伺機擊發而意譯「突-{}-擊者」。}}{{註|{{YouTube|-3vTSR8ipeQ|少年的異星漂流物語《翠星上的加爾岡緹亞》 MOD第101頻道【MY 101綜合台】}}. 曾在預告片譯為「捷-{}-霸」，後來台版DVD採用，異於台灣MY 101綜合台及港版的「突-{}-擊者」。}}（{{lang|ja|チェインバー}}）被吞入不明的时空。当他从人工冬眠中醒过来时，他发现自己連同機體被打撈至一個古舊地方——加爾岡緹亞之內。更進一步，他發現身處的是早已被人類遗忘的起始之星——地球。如今這個行星被一望無際的海水围绕著，人们亦只能住在巨大的舰船，靠著打捞古文明遗留在深海中的物品生存。由于缺乏行星历史、文化以至語言方面的知识，他被逼於海中的城市加爾岡緹亞滯留著，与對其充滿興趣的艾咪一同生活……

=== 主角 ===
; {{nihongo|萊德|レド|Redo|extra2={{配音2|JP|[[石川界人|石川界人]]|HK|[[陳灝瑋|陳灝瑋]]|US|Alan Lee}}}}
{{Quote box| width = 300px | align = right | quoted = true | bgcolor = #FFFFF0 | salign = center | quote =我就算知道自己的死亡方式，卻不懂得生存的方式。面對這樣的一個我，竟然有人說要跟我一起尋找活下去的方法...好想再見到她，好想多聽聽她的聲音...| source = 莱德，第13話}}
: 本作的男主角。人類銀河同盟軍隊的[[少尉|少尉]]；機動兵器「突擊者-K6821」的駕駛員。16歲﹑身高173厘米<ref name="jinbutsu">{{Cite web|url=http://gargantia.jp/#petit|title=「翠星のガルガンティア」アニメ公式ショートムービー 『ぷちっとがるがんてぃあ』|work=第１話より。|publisher=[[萬代頻道|萬代頻道]]|date=2013-04-10|accessdate=2013-04-17}}</ref>。
: 於殲滅黑迪亞斯的任務失敗後，在上級軍官[[#人類銀河同盟|古格爾中校]]的支援下跟隨母艦「拉莫萊克號」穿越蟲洞撤退，途中因為意外事故而被捲入空間扭曲帶，和突擊者一同跌進時空轉移罅隙。身體機能根據緊急應變程序而自動進入歷時半年{{註|準確為26萬6815分鐘}}的人工冬眠狀態，期間漂流到一顆被遺忘的邊境行星—地球。
: 當再次甦醒時，已經身處加爾岡緹亞的的機庫之內。初時對於陌生的環境﹑未知的古代語言和身份不明的人類存有敵意。透過突擊者的翻譯功能，與船團的代表助理莉芝進行對話後，雖已向對方表示並無惡意，但雙方關係仍然一度處於相當緊張的狀態。期後與「特使」愛美相處過後，決定在得到救援前嘗試跟艦上的人合作。
: 缺乏生活常識，待人處事不時出現誤解。因為語言不通而未能跟Chamber一同工作，於沒有戰鬥的狀況下一直待命，後來漸漸融入加爾岡緹亞的生活步調後才開始嘗試幫忙做事。與愛美日漸相處下對她日漸有好感，甚至有在其之上的情感，愛美可說是萊德在加爾岡緹亞生存的支柱。
: 閒暇時間會使用激光槍製作[[陶笛|陶笛]]，[[笛|笛]]的材料使用烏賊王的毒牙。這種無意識的行為是他幼年時弟弟送給他的禮物，但他的記憶遭刪除而一直不知道，雖然被刪除的記憶有回復。據稱弟弟是由於體弱或是殘障等問題，而未能符合銀河同盟的需要，故已被「處理」。
: 在決定認真投入職業工作下，前往協助貝露絲打撈。此時遇上黑迪亞斯的原形品種－鯨烏賊，便反射性的將其宰殺。確定兩種為同一生物後擔心牠們日後會侵略加爾岡緹亞而重燃殲滅黑迪亞斯的想法。在彼利安的說服下與費蘭治的船團一起脫離加爾岡緹亞掃蕩鯨烏賊的巢穴，意外在裡頭發現過去的機密資料而得知黑迪亞斯本是人類的秘密。
: 後來遇上銀河同盟軍中校古格爾的坐駕－Striker，但沒想到古格爾竟打算以宗教之名將地球所有船團招降並以銀河同盟的方式管理。已融入加爾岡緹亞的莱德不願看到這樣的結果，與同樣對此不滿的彼利安、費蘭治和菈淇芝等人聯手反叛古格爾。後意外發現古格爾早已死去的事實，便與Chamber全力攻擊Striker，但仍因性能和技術落差而無法擊敗它。本打算啟動'''極限精神同步'''至臨界而就此成仁，但心中仍珍視著愛美，最後被Chamber以莱德心理不符銀河同盟軍人資格為藉口強制剝奪軍籍將駕駛艙脫離後與Striker同歸於盡，見到Chamber為自己而悲壯犧牲的莱德因此傷心落淚。
: 事後回到加爾岡緹亞並成打撈小組的一員，皮膚似乎因此而稍微變黑，並主力搜查鯨烏賊巢穴裡殘存的歷史資料，希望將過去有關大陸同盟和黑迪亞斯的秘密查清，並期望可幫助未來讓任何一方都共存生活，而Chamber殘存的通訊終端則一直隨身攜帶著以悼念他。偶爾會跟彼利安學習機械的知識。

== 登場船舶 ==
=== 加爾岡緹亞船團 ===
由複數船舶連結而成的海上都市，在同類型的船團中份屬最大規模。每隻大型船舶都擁有自主的生活規律，但普遍基於集體安全的考慮而合成船團，亦備有小型船隻去進行講求速度效率的活動。日常工作會使用搭載型機械'''Yumboro'''。

新船加入後可以委派其主人作船團決策代表，但不少是由團長和數名大船主直接決定。

; {{nihongo|加爾岡緹亞號|ガルガンティア号}}
: 中央的主體。設有大型農場，並有作為船團重要象徵的'''起重機塔'''，高達數百米，內嵌灑水系統可做大規模灑水。

:;{{nihongo|天之梯|天の梯子}}
::'''起重機塔'''的真面目，是舊時代冰河時期科技所開發的'''[[質量投射器|巨型質量投射器系統]]'''（Orbital Mass driver system-GARGANTIA）。擁有電腦輔助瞄準，可將投射物精準射至數里遠的地方，啟動需要使用加爾岡緹亞船團護憲者託管的金鑰匙，整個系統並無全自動化，因此操控和運作需數人協力，灑水系統本身真實作用是當投射[[衝撃波|衝撃波]]的緩衝裝置，填裝彈藥投射便可當[[加農炮|加農炮]]。大概已有數年沒啟用過，第12話新任船團長莉芝在艾咪的請求和奧特姆的建議下與高層決定啟動攻擊古格爾船團替萊德和費蘭治船團做遠距火力支援。
::由內部構造推斷其質量投射方式是採用[[冷兵器#先進冷兵器|蜈蚣砲]]的技術，以數個有偏角的多岔路式噴射來將投射體加速推進。由於目前蜈蚣砲技術發展目標是用於取代輔助推升火箭將[[太空載具|太空載具]]以廉價安全的方式送上太空，因此天之梯可能具備將物體送上太空的力量。於海底過去'''EVOLVER'''用來研發黑迪亞斯的研究遺跡中有八具與天之梯極相似的柱狀設施。

== 用語 ==
; {{nihongo|人類銀河同盟|人類銀河同盟}}
: 同盟成員有4億7000萬人。透過討伐黑迪亞斯，從而維護人類的安定。身體較弱者會被視為劣等者以及淘汰。
: 擁有軍隊。軍人執行軍務的時間達到14萬5000小時（約16.5年）便擁有限定市民權、申請在同盟的根據地‧阿瓦隆進行四星期的旅遊與滯留資格（擁有自由睡眠、自由飲食以及生殖的自由）。
: 其起源是在舊文明時代提倡以宇宙船進出宇宙，並因人道爭議問題而與EVOLVER（自發進化推進派）進行戰爭衝突的多國組織「大陸同盟」。
: 在同盟内，黑迪亞斯的誕生以及與其所發生的戰爭都被視為機密資料。
: 由於組織信念的關係，大型戰艦均有「人」的象徵，且全是載人兵器。
: 其實在本篇開始前經歷了數月和黑迪亞斯戰鬥的連敗，處境極為不利，實際上人口數也不到4億7000萬，有一些成員因為恐懼這沒有盡頭的戰爭而脫離同盟，成為「漂流部族」。
; {{nihongo|黑迪亞斯|ヒディアーズ}}
: 人類基因為基礎演變出來的宇宙生命體，與人類銀河同盟屬於敵對狀態。擁有高戰鬥力，可以在一瞬間摧毀人類銀河同盟的艦艇及Machine Calibur。
: 為了應付複雜的戰局而特化出了多種個體，外表類似由三片花瓣組成，可搬運的「中型攻擊體」、菱形，可搭載許多攻擊形並俱備反艦光束發射器官戰力強大的「攻擊型母艦」、和具有學習能力，可免疫相同攻擊的「進化型母艦」，透過奈米機械模擬碳元素形成堅固的外骨骼的同時可高效率的將電磁波轉換成能量，並且可由外部補食機械的能源，雖然不具備蟲洞技術，但其戰鬥力遠遠凌駕人類銀河同盟。
: 其起源是舊文明時代的EVOLVER（自發進化推進派）。由於冰河期逼近，而使用宇宙船進行移民需時長，人類可能在宇宙船內因衰老而自然死亡，因此主張在人類的基因組上進行調整，使人類自己的身體轉變成適合在宇宙的真空環境生存。但因為如此將會使人類失去自我而成為另類生物而使的大陸同盟的人們感到非人道而且是威脅到人類的存在，無視於同盟的多次通牒而持續推廣並進行基因改造研究，最終發展成長期的慢性戰爭。
: 黑迪亞斯是從鯨烏賊進化演變成的，在宇宙嚴酷的環境以及長期的戰爭中演變為軀體與生存力強大的生物，失去了社會、文明以及倫理，成為威脅人類於宇宙發展的生物兵器。

; {{nihongo|鯨烏賊|クジライカ}}
: 形狀類似一隻巨大的魷魚，是地球最強的海洋生物。加爾岡緹亞船團的住民視牠為神聖的生物。特徵與黑迪亞斯相似，由[[DNA|DNA]]分析證實與黑迪亞斯是同種生物。
: 不會主動對人類進行攻擊，但受到攻擊就會狂暴化，而加爾岡緹亞船團的裝備亦不可能與其進行對抗。
: 人類適應宇宙實驗的先行海中適應實驗產物，其實驗體的後裔、黑迪亞斯的原型，稱為 {{nihongo|「共生體」|共生体}}<ref name=Febri>[http://nitroyss.jp/archives/5771 {{lang|ja|ニトロ有線式 - 「Febri (フェブリ) Vol.17 2013年 07月号」購入}}] {{webarchive|url=https://web.archive.org/web/20130617032017/http://nitroyss.jp/archives/5771 |date=2013-06-17 }}</ref>，由舊文明時代的人類製造，能呼吸海中的氧氣，還可承受深海的水壓。通過奈米機械「光蟲」共生進行營養補給，和光合作用相似，但效率更高。

; {{nihongo|光蟲|ヒカリムシ}}
: 奈米機械所構成擬似生体組織。渡過地球寒冷化後人類能源的來源，組成鯨烏賊的皮膜及触手，在宇宙則是黑迪亞斯的外殼{{r|Febri}}。
