'''马萨诸塞州'''（{{lang-en|Commonwealth of Massachusetts}}），又稱'''-{zh-cn:麻省;zh-tw:馬薩諸塞州;zh-hk:馬薩諸塞州}-'''、'''-{zh-cn:麻州;zh-tw:麻省;zh-hk:麻薩諸塞州}-'''或者'''-{zh-cn:麻萨诸塞州;zh-tw:麻州;zh-hk:麻州}-'''，正式名稱為'''麻薩諸塞聯邦'''，是位於[[美國|美國]][[美國東北部|東北部]]的[[美國州份|州]]，為[[美國獨立|美國獨立]]時最初的[[北美十三州|十三州]]之一，也是[[新英格蘭|新英格蘭]]六州裡人口最密集的一州。根據美国2014年人口估算顯示，該州共有人口674.5萬。[[波士頓|波士頓]]為該州首府及最大城市。

== 歷史 ==
麻薩諸塞的地名來自早期殖民時代的一個位於此地的[[印第安|印第安]][[部落|部落]]名字，原本的意思是「一個很大的山坡地」。最早來自[[英國|英國]]的移民是在1620年搭著[[五月花號|五月花號]]船抵達[[普利茅斯_(麻薩諸塞州)|普利茅斯]]。

之後不久又有從[[英國|英國]]來的[[清教徒|清教徒]]，他們建立了-{zh-cn:马萨诸塞海湾殖民公司;zh-tw:麻薩諸塞海灣殖民公司;zh-hk:馬薩諸塞海灣殖民公司}-，這也是麻州的前身。麻薩諸塞是當初在[[美國革命|美國革命]]時最早和[[英國|英國]]對抗的[[十三州殖民地|十三州殖民地]]之一。雖然清教徒遷徙到此地的原因是為了[[信仰自由|信仰自由]]，但是他們卻很排斥其他的[[教派|教派]]，也因此，有些人離開了麻薩諸塞而往南遷徙。這些人當中，[[羅傑·威廉斯|羅傑·威廉斯]]開拓了[[羅德島州|羅德島州]]的殖民地，而[[湯姆斯·虎克|湯姆斯·虎克]]則成立了[[康涅狄格州|康涅狄格州]]。

1775年2月9日，[[英國國會|英國國會]]宣佈麻薩諸塞發生叛亂事件，接著派出軍隊去鎮壓。

1786年－1787年，麻薩諸塞發生[[謝伊斯起義|謝伊斯起義]]，遭到州長[[詹姆斯·鮑登|詹姆斯·鮑登]]鎮壓，[[塞繆爾·亞當斯|塞繆爾·亞當斯]]支持鎮壓。

1788年2月6日麻薩諸塞成為第6個認可[[美國憲法|美國憲法]]的州。

1820年3月15日，[[緬因州|緬因]]地區從麻州分出去，因為緬因和麻州中間有海隔開，並不相連，緬因成為美國新的一個[[州|州]]。

本州歷史悠久，州裡有許多帶有歷史意義的建築物。

=== 麻薩諸塞州的鎮和郡 ===
麻薩諸塞州和其他同為[[新英格蘭|新英格蘭]]區的五個州，以及[[紐約州|紐約州]]有著一種政府統治上的架構，叫做「新英格蘭鎮」。

在大多數的州，一個[[鎮|鎮]]是一個小型的管轄區域。在鎮與鎮之間則是處於無市鎮政府管轄的狀態，而且土地面積往往很大。相比之下，一個州則是完全劃分為{{tsl|en|郡||郡}}，州內每一吋土地都會有某個郡在管轄。郡政府因此佔有很重要的地位，特別是對於那些住在鎮與鎮之間的人，他們往往在經營一些很重要的事業，比如經營機場等等。

和上述的其它州不同的是，麻薩諸塞州把州內的土地全部劃分給各市鎮，每一吋土地都會有某個市或者是鎮在管轄，不會落得沒政府管的下場。不過這種分法當要和上述的其他州作比較時會變得很複雜，比如麻州是以各個市鎮為單位來作調查數據，但是其他州卻是依據[[美國人口調查局|美國人口調查局]]的規定以較大範圍的包含數個較小市鎮的分法來作調查，因此無法很直接的作對等的比較。

在1990年代之前，麻州郡政府大部份的職責（包含法院的運作和道路維修）不是被州政府所取代，就是被認為運作不彰和老舊過時。幾個最近的例子：

1990年中期[[米德爾塞克斯縣_(麻薩諸塞州)|中塞克斯郡]]郡政府因為經營郡立醫院不善導致整個郡瀕臨破產，然後在1997年州議會介入並且取走所有的資產和職權。中塞克斯郡政府於是在1997年7月11日正式被廢除。過後不久，法蘭克林郡委員會自己投票決定廢除自己的郡政府。用來廢除前面幾個郡的廢除法也被延用到其他的郡，像是漢普頓郡和沃斯特郡於1998年7月1日被廢除。這條廢除法之後又附加一些條款以用於在1999年麻薩諸塞州1日廢除罕布什爾郡，同一年的7月1日用於廢除艾塞克斯郡，以及2000年7月1日廢除伯客什爾郡。其它的郡則可以引用「麻薩諸塞州一般法第34B章」自行投票決定廢除自己的郡，或是重新改組為「區域性的政府委員會」。

== 政治與法律 ==
马萨诸塞州的首府是[[波士頓|波士頓]]，現任州長是[查理.貝克]。兩個政黨分別是[[民主黨_(美國)|民主黨]]及[[共和黨_(美國)|共和黨]]，民主黨在州議會佔有超過三分之二的席位。在2001年調整後的[[美國眾議院|眾議院]]席位，麻州配有10席。州議會有個很正式的名稱「偉大暨統管的聯邦議院」。

州內並設有「最高司法法院」。州法對[[市|市]]和[[鎮|鎮]]分的很清楚，目前最大且人口最多的[[鎮|鎮]]是[[弗明罕|弗明罕]]。從政治角度來看，市和鎮的最大區別是，鎮沒有鎮長，在治理上是由[[鎮民大會|鎮民大會]]做出決定。市則有[[市議會|市議會]]，[[市長|市長]]或是市政官並不是每個市都有。這種管理法是在1820年代就開始了，在這之前，就只有鎮民大會。麻州有些鎮後來已經升格而成為市，但是地名還是保有「鎮」這個字在裡頭，比如[[水鎮|水鎮]]。

麻州自治法裡頭有個「二又二分之一提議法案」，根據這個法案，麻州每年的花費支出增加不能超過2½%，如果要超過就必須經過公民投票通過才行。

經過麻州最高法院在[[2003年11月|2003年11月]]的判決，麻州於2004年5月17日成為美國第一個允許[[同性婚姻|同性婚姻]]的州。

== 經濟 ==
1999年的麻州總生產值是2620億美元，佔全國第11位。2002年的麻州每人平均收入是39,244美元，佔全國第三。

本州的農漁輸出是海鮮，植物苗，牛乳類產品，小紅莓和蔬菜。工業類的輸出是機器，電機設備，科學儀器，印刷及出版。其它和麻州經濟汲汲相關的有高等教育，健康醫療，旅遊觀光和金融服務。

著名軍火生產商[[雷神公司|雷神公司]]和保險商{{tsl|en|Liberty Mutual Insurance Group|利寶互助公司}}、專業零售商{{tsl|en|Staples||Staples}}總部位於該州。

== 教育 ==
[[File:Wellesley_College_Tower_Court.jpg|缩略图]]
麻州人口只佔全國總人口的2.5%，但是麻州的四年制大學學院數目卻佔了全國總數目的4.5%。八個波士頓地區的學術機構（哈佛大學、麻省理工學院、[[波士頓學院|波士頓學院]]、波士頓大學、[[布蘭迪斯大學|布蘭戴斯大學]]、東北大學、塔芙茨大學和麻州大學波士頓分校），{{Fact|他們稱呼自己為“研究型大學”，根據他們所說，他們是二次世界大戰後的「經濟成長動力」，現階段則是每年對於麻州經濟有70億美元的助益。因為大波士頓地區擁有大量的大學和學院，每逢學年的開始，大波士頓地區的的人口數總是會突然暴增。|date=2018-12-14}}（請參照[[大波士頓地區大學學院列表|大波士頓地區大學學院列表]]）

麻州的[[哈佛大學|哈佛大學]]是[[常春藤盟校|常春藤聯盟]]大學的一員，並且有三所大學是屬於[[七姊妹學院|七姊妹學園]]、[[曼荷莲学院|曼荷莲学院]]、[[史密斯學院|史密斯學院]]和[[衛斯理學院|衛斯理學院]]。科技導向方面的大學有[[麻省理工學院|麻省理工學院]]和[[伍斯特理工學院|沃斯特理工學院]]。位於麻州靠近中西部的著名大學院校有位於[[先鋒山谷|先鋒山谷]]的[[五大學院|五大學院]]（[[曼荷蓮學院|聖尤克山學院]]、[[史密斯學院|史密斯學院]]、[[安默斯特學院|安姆赫斯特學院]]、[[罕布什爾學院|罕布什爾學院]]、[[麻州大學|麻州大學]][[麻薩諸塞大學阿默斯特分校|安姆赫斯特校區]]（麻州大學旗艦校））和[[威廉士學院|威廉斯學院]]。藝術方面，[[新英格蘭音樂學院|新英格蘭音樂學院]]（美國最古老的獨立音樂學院）及[[伯克利音樂學院|伯克利音樂學院]]位於[[波士頓|波士頓]]。另外還有知名的獨立研究機構，包含[[伍兹霍尔海洋研究所|伍兹霍尔海洋研究所]]（美國最大的獨立海洋學研究所）。

== 州歌 ==

麻州目前有三首官方認可的歌：

* 州歌：「大家對麻薩諸塞致意」
* 民族歌謠：「-{zh-cn:马萨诸塞州;zh-tw:麻薩諸塞;zh-hk:馬薩諸塞}-」
* 愛國歌曲：「我的麻薩諸塞，因為有你我們國家才有自由」
