'''麻阳苗族自治县'''位于湖南省西部，怀化市西北部，地处[[辰水|辰水]]流域。地理坐标为东经109°24′-110°06′、北纬27°32′-28°01′。东临辰溪县，南连鹤城区和芷江县，西接贵州省铜仁市，北靠泸溪县和凤凰县。总面积1561平方千米。总人口40万人（2011年），少数民族以苗族为主，使用漢語[[西南官话|西南官话]]和[[苗語湘西方言|苗語湘西方言]]<ref>{{cite web |url=http://www.hntj.gov.cn/tjgb/xqgb/hhgb/default.htm |title=存档副本 |accessdate=2005-11-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20051123113329/http://www.hntj.gov.cn/tjgb/xqgb/hhgb/default.htm |archivedate=2005-11-23 }}</ref>。1988年10月31日改“麻阳县”置麻阳苗族自治县，[[县治|县治]][[高村镇|高村镇]]。

麻阳是[[平江起义|平江起义]]领导人之一、中华人民共和国第一任铁道部长、全国政协副主席[[滕代远|滕代远]]的家乡。

== 行政区划 ==

* 18[[乡|乡]]：郭公坪乡、长潭乡、拖冲乡、尧市乡、文昌阁乡、大桥江乡、舒家村乡、隆家堡乡、谭家寨乡、石羊哨乡、板栗树乡、谷达坡乡、高村乡、兰村乡、栗坪乡、绿溪口乡、和坪溪乡、黄桑乡。

== 地理 ==
麻阳地处[[湖南|湖南]]、[[贵州|贵州]]二省交界地。相邻[[县|县级政区]]有，东临湖南[[辰溪|辰溪县]]，南连怀化[[市辖区|市辖区]][[鹤城区|鹤城]]和[[芷江|芷江侗族自治县]]，西接贵州[[铜仁市|铜仁市]]，北靠湖南[[凤凰县|凤凰县]]和[[泸溪|泸溪县]]。

县城距怀化市59公里，距大兴、芷江机场分别为130、110公里，距旅游热点城市[[凤凰县|凤凰]]34公里，处于[[怀化|怀化]]、[[铜仁|铜仁]]、[[吉首|吉首]]三大地级城市的中心点上。
