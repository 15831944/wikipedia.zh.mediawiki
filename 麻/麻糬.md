'''麻糬'''（{{jpn|j=餅|hg=もち|rm=Mochi}}；{{lang-en|Rice cake}}）是一種[[糯米|糯米]]製的[[食物|食物]]，軟而有黏性，為[[日本|日本]]的传统[[小吃|小吃]]，在香港，台灣，中國大陸亦相同小吃，在[[东亚|东亚]]地區十分流行。

麻糬或[[File:Mashu_SVG.svg|14px]]糬({{僻字|𫃎|[米麻]}}糬<ref>[http://std.dkuug.dk/jtc1/sc2/wg2/docs/n3270.pdf]</ref>)，而此类食品在中日韩叫法各不相同，漢語內部的方言也不相同。在香港，一般稱為[[糯米糍|糯米糍]]。

新马闽语''muah chee''、臺灣閩南語''moa-chi''發音一致，都源自「麻糍」──里面不含餡料，是剪成小块状、並撒花生粉。

== 節氣食品 ==
[[日本新年|日本新年]]一定要吃用高湯或白味噌湯，與蘿蔔、芋頭、魚糕、雞肉等按地區不同的食材一起煮麻糬的「[[雜煮|雜煮]]」，慶祝新年到來，并希望新的一年裏一切順利。由於麻糬的特色是非常的[[黏|黏]]，在日本的[[過年|過年]]就有很多人因為吃麻糬產生[[窒息|窒息]]而死的意外事件，特別會在65歲或以上的老年人發生。一般在過年之後，幾天前的[[死亡|死亡]]人數才會在[[報紙|報紙]]的報導中公佈出來。根據[[厚生勞動省|厚生勞動省]]統計，2006年整年裏在日本因吃麻糬窒息而死的有91例，是窒息原因食品之首。但这应该是报导有误，日本过年时，常吃的是烤年糕，而并非是吃麻糬。据闻在日本有一种，不包内馅，而在外包一片紫苏叶的口感松软的“雪见麻糬”。但由于冒名者众多，真品反而已不复见。

== 食用方法 ==
日本的麻糬吃法很多，常見的有黃豆粉麻糬、绿茶麻糬、紅豆麻糬、抹茶麻糬、花生麻糬、芝麻麻糬、牛奶麻糬、蘿蔔泥麻糬、炭烤麻糬、鐵板燒麻糬、炸麻糬、麻糬烏冬等，還有，外包一片紫蘇葉或紫菜的等等，樣式繁多。吃火鍋時，麻糬常做最後一道材料。

== 大福餅 ==
在日本，包內餡，口感鬆軟的麻糬叫做「大福餅」，除了傳統的紅豆餡以外，近三十年內普及的有「草莓大福」、包雪糕的「雪見麻糬」等。一般是以葛粉為原料製成。

== 加工 ==
在日本，傳統作法用蒸好的原粒糯米，使用木杵和石臼手工打糍粑，但大工廠採用機械化的自動打糍粑機加工。
