'''评书'''，又稱'''說書'''，[[湖北|湖北]]、[[广东|广东]][[粤语|粤语]]地区及[[闽南语|闽南语]]地区称'''讲古'''<ref>[http://www.hbdgw.com/content.asp?id=26394 湖北大公网 > 民风民俗 > 讲古]{{dead link|date=十月 2017 |bot=InternetArchiveBot }}</ref><ref>[http://twblg.dict.edu.tw/holodict_new/result_detail.jsp?n_no=12410&curpage=1&sample=%E8%AC%9B%E5%8F%A4&radiobutton=1&querytarget=1&limit=20&pagenum=1&rowcount=2 中華民國教育部臺灣閩南語常用詞辭典]</ref>，在[[四川|四川]]称为'''讲书'''，古称'''說話'''，是[[中國|中國]][[东北|东北]]、[[华北|华北]]、 [[两广|两广]]、[[湖广|湖广]]、[[四川|四川]]一带一种口头讲说的表演形式，在[[宋代|宋代]]開始流行。各地的'''說書人'''以自己的[[母語|母語]]對人說著不同的[[故事|故事]]，因此也是[[方言|方言]][[文化|文化]]的一部份。[[清末|清末]][[民初|民初]]時，評書的表演為一人坐於桌後表演，道具有[[折扇|折扇]]和[[醒木|醒木]]，服裝為[[長衫|長衫]]；至20世纪中叶，多不再用桌椅及折扇、醒木等道具，而以站立說演，服裝也較不固定。

== 历史 == 
以说书者的说法：评书起源于[[东周|东周]]时期，[[周庄王|周庄王]]是评书的[[祖师爷|祖师爷]]。但这只是一个[[传说|传说]]。实际评书的创始人为[[明末|明末]][[清初|清初]]的[[柳敬亭|柳敬亭]]，最初只是[[说唱艺术|说唱艺术]]的一部分，称为“[[弦子书|弦子书]]”，他的老師[[莫後光|莫後光]]提到說話理論是：「夫演義雖小技，其以辨性情、考方俗、形容萬類，不與[[儒者|儒者]]異道。故取之欲其肆，中之欲其微，促而赴之欲其迅，舒而繹之欲其安，進而止之欲其留，整而歸之欲其潔。非天下之精者，其孰與於斯矣！」<ref>[[李延是|李延是]]：《南吳舊話錄》卷二十一</ref>。

[[北宋|北宋]][[汴京|汴京]]人[[霍四究|霍四究]]以“說三分”著名，“不以風雨寒暑，諸棚看人，日日如是”，“說三分”即講[[三國|三國]]故事<ref>[[孟元老|孟元老]]，《[[东京梦华录|东京梦华录]]笺注》卷五《京瓦伎艺》，[[中华书局|中华书局]]2006年版，第462页</ref>。晚清[[光绪|光绪]]年间，评书传入[[皇宫|皇宫]]中；因皇宫唱歌多有不便，于是改说唱为“评说”，于是评书的艺术形式便固定下来。

[[民国|民国]]是评书中兴的时期，{{fact|time=2008-07-24|据记载：当时“撂地”说书人说《[[三国演义|三国]]》，便万人空巷，把街道围得满满的。}}另外，评书与[[相声|相声]]也有很大的渊源。尤其是单口相声和评书的“片子活”技巧几乎相同。有些相声的“段子”也来源于评书章节。

[[中华人民共和国|中华人民共和国]]成立后，评书也得到了发展。尤其是[[广播|广播]]和[[收音机|收音机]]的普及，给评书极大的生存空间。评书艺人又创作了许多新的段子，比如《[[林海雪原|林海雪原]]》、《平原枪声》，[[文革|文革]]结束，评书艺人又创作了不少段子，甚至也说几段外国书，比如《[[珍珠港|珍珠港]]》（书名《太平洋大海战》），还制作了[[Flash|Flash]]评书《[[白眉大侠|白眉大侠]]》，但并不特别受书迷欢迎。

现在，几乎每个广播电台都有评书专栏，部分电台更有专门的评书或故事频率。主要听众是老年人或[[出租车|出租车]]司机，评书仍很受听众的欢迎，其中原因除了本身评书喜闻乐见，还由于广播的局限。但一些评书演员担心评书会渐渐的消失，实际上，现在很难再看到年青的说书人了。

=== 说话与评书 ===
在[[唐代|唐代]]，[[佛教|佛教]]融入了通俗文學，[[僧人|僧人]]為了吸引[[檀越|檀越]]，發明了一種說演[[佛經|佛經]]故事的藝術，稱之[[變文|變文]]。不久，又出現了一種[[曲艺|曲艺]]艺术，和评书的表演方式相似，这种曲艺形式称为“说话”。到宋代中兴时期<ref>[[翟颧|翟颧]]《通俗编》卷三十一《徘优》条引耐得翁《古杭梦游录》:“说话有四家:一者小说，谓之银字儿，如烟粉、灵怪、传奇、说公案，皆是传(搏)拳提刀赶棒及发迹变态之事;说铁骑儿，谓士马金鼓之事;说经，谓演说佛书，说参谓参禅;讲史书，谓说前代兴废争战之事。”</ref>，在中国古典文学认为，“说话”这种表演的形式对明清[[章回小说|章回小说]]的影响是非常大的<ref>见[[鲁迅|鲁迅]]《宋人之“说话”及其影响》；[[相恺|相恺]]《宋元小说史》。</ref>。“说话”发展到俗说后表演方式与“评书”是非常类似的。比如评书中的开场诗，说话称为“[[押座文|押座文]]”。也有“且听下回分解”一类用句。当然，这些都影响了明清的小说。同时，说话文学也在日本有所發展，成爲一种{{lj|说话 (日本)|説話|本土文学形式}}。

事实上如《三国演义》、《[[水浒传|水浒传]]》最初都是说话的[[话本|话本]]。《三国演义》话本为《[[三国志平话|全相平话三国志]]》；《水浒传》则为《[[醉翁谈录|醉翁谈录]]》。[[晚唐|晚唐]]诗人[[李商隐|李商隐]]有《驕兒》一诗写到：“或谑[[张飞|张飞]][[鬍|鬍]]，或笑[[邓艾|邓艾]][[口吃|吃]]。”说明当时喜欢说话这门曲艺的百姓是非常多的。

研究“说话”，多数是从话本对明清的小说的影响加以研究。也有一些曲艺艺人写过专门的资料。很多证据表明“说话”类似于“评书”。但两者似乎没有任何确实的传承关系。

== 題材 ==
说书虽然以讲史为主，但多数为引人入胜，并没有尊重历史的原貌。比如：[[商朝|商朝]]有[[大炮|大炮]]；[[汉代|汉代]]有[[银票|银票]]；[[三国|三国]]有[[武狀元|武狀元]]。但由于说书面向的是[[百姓|百姓]]，客观上起到普及文化和历史的作用。在古代，许多不认识字的人都能吟两句[[打油诗|打油诗]]、了解历史，唯一的途径就是评书和[[戏曲|戏曲]]的传播。

说书的段子来源分四类：史书（《[[三国演义|三国演义]]》、《[[隋唐演义|隋唐演义]]》）、神怪书（《[[封神演义|封神演义]]》、《[[济公传|济公传]]》）公案书（[[施公案|施公案]]、[[包公|包公]]案）、[[武侠|武侠]]书（《[[三侠五义|三侠五义]]》、《[[童林传|童林传]]》）。但几乎只要是[[小说|小说]]，经过略微加工，就可以成书评说。

== 形式 ==
每一段的评书称“回”，小于100回的书称小段。如果只是一回或几回称“片子活”。如果大于100回那么称为，“长篇活”。

说一部开篇先赋一首诗，称为“开场诗”，常说一首诗或小令诗。比较常见的是《[[西江月|西江月]]》或《[[临江仙|临江仙]]》。回与回之间一般以“且听下回分解”和“上回书，我们说到......”连接。这和[[章回小说|章回小说]]是一样的。

评书以语言除了“开场诗”外，还有多种技巧：
* 开脸儿：介绍新人物的即将人物的来历、身份、相貌、性格等等特征
* 摆砌末：讲述故事的场景
* [[赋赞|赋赞]]：如果赞美故事中人物的品德、相貌或风景名胜，念诵大段落[[对偶|对偶]]句式韵文，称作“赋赞”，富有语言的美感；
* [[垛句|垛句]]：说演到紧要处或精彩处，使用[[排比|排比]]重叠的句式以强化说演效果。

== 传承 ==
评书从[[柳敬亭|柳敬亭]]、[[王鸿兴|王鸿兴]]开始，经十代传承之今。以前三代谱系：
* 第一代：王鸿兴
* 第二代：[[安良臣|安良臣]]、[[邓光臣|邓光臣]]、[[何良臣|何良臣]]
* 第三代：[[张海凤|张海凤]]、[[张沛然|张沛然]]、[[柯光玉|柯光玉]]、[[安太和|安太和]]

著名的说书艺人：
* 老一辈的有：[[张悦楷|张悦楷]]、[[林兆明|林兆明]]、[[连阔如|连阔如]]、[[袁阔成|袁阔成]]、[[李鑫荃|李鑫荃]]、[[田连元|田连元]]、[[连丽如|连丽如]]、[[单田芳|单田芳]]、[[刘兰芳|刘兰芳]]、[[刘延广|刘延广]]、[[金文声|金文声]]、[[刘立福|刘立福]]
* 新一辈的有：[[孙一|孙一]]、[[张少佐|张少佐]]、[[王玥波|王玥波]]、[[李伯清|李伯清]]
