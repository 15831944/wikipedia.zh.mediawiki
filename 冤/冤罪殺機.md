{{Vgname|冤罪殺機|en=Dishonored|cn=羞辱|aka-cn=耻辱|tw=冤罪殺機}}是Arkane工作室开发、[[貝塞斯達軟件公司|貝塞斯達軟件公司]]发行的一款潜入和动作冒险游戏。於虛構工業城市丹沃爾中，身為女王保鑣的主角科沃爾·阿塔諾被誣蔑為殺害女王的兇手，在逃離監禁後化身為能運用超能力的刺客對一眾相關人士復仇。遊戲於2012年10月發售，适用[[Microsoft_Windows|Microsoft Windows]]、[[PlayStation_3|PlayStation 3]]和[[Xbox_360|Xbox 360]]平台。遊戲獲得各大遊戲評論網站正面評價，並獲得數個獎項。續集《[[冤罪殺機2|冤罪殺機2]]》於2016年11月11日發售。

[[File:Harvey-Smith-in-Montreal-2007.jpg|thumb]]》的遊戲設計哈維·史密斯是《冤罪殺機》兩名創意總監之一。]]
2011年7月7日，貝塞斯達軟件公司宣佈《冤罪殺機》將會是一款第一人稱潛行動作冒險遊戲，並對應微軟視窗、Playstation 3及Xbox 360，它也是Arkane工作室被貝塞斯達母公司[[ZeniMax_Media|ZeniMax Media]]於2010年8月收購後發行的第一款遊戲，由Arkane工作室總監拉斐爾·哥蘭東尼奧（Raphaël Colantonio）和[[駭客入侵|駭客入侵]]開發者哈維·史密斯（Harvey Smith）擔任創意總監<ref>{{Cite web |url = http://www.eurogamer.net/articles/2011-07-07-bethesda-announces-dishonoured |title = Bethesda announces Dishonored |author = Yin-Poole Wesley |date = 2011-07-07 |publisher = [[Eurogamer|Eurogamer]]}}</ref>；首席設計由駭客入侵設計師里卡多·貝雅（Ricardo Bare）擔任；視覺設計總監維克托·安東諾夫（Viktor Antonov）與藝術總監賽巴斯汀·米頓（Sebastien Mitton）帶領藝術團隊<ref>{{Cite web |url = http://www.computerandvideogames.com/310722/bethesdas-dishonored-is-official-big-talent-on-board/ |title = Bethesda's Dishonored is official |author = Andy Robinson |date = 2011年7月7日 |publisher = [[電腦與電子遊戲|電腦與電子遊戲]]}}</ref>。哈維·史密斯、拉斐爾·哥蘭東尼奧、維克托·安東諾夫和賽巴斯汀·米頓利用三年時間進行早期製作。貝塞斯達希望Arkane工作室能開發新遊戲及新知識產權。雖然團隊已對一款相似的遊戲有所構思，但直至貝塞斯達與它們聯絡前仍未有確實的主意。賽巴斯汀·米頓於2009年5月請求維克托·安東諾夫就新的知識產權建立識別藝術。Arkane工作室全體員工均參與遊戲製作<ref>{{Cite web |url = http://www.edge-online.com/features/the-edge-awards-2012-studio-of-the-year/3/#gallery-top |title = The Edge Awards 2012: studio of the year |author = [[Edge_(雜誌)|Edge (雜誌)]] |date = 2012-12-08}}</ref>。

遊戲分別支援[[家用遊戲機|家用遊戲機]]介面與微軟視窗介面，同時支援視窗上使用Xbox 360控制器。哈維·史密斯表示團隊的思維是假如開發人員喜歡電腦就可參與開發電腦軟件；對Xbox 360感興趣則可開發[[Xbox_Live|Xbox Live]]遊戲成就<ref>{{Cite web |url = http://www.pcgamer.com/dishonored-on-pc-has-its-own-different-interface-pc-standards-are-a-religion/ |title = Dishonored on PC "has its own different interfaces," PC standards are "a religion" |author = Tyler Wilde |date = 2012-06-20 |publisher = PC Gamer}}</ref>。遊戲於2012年9月28日正式製作<ref>{{Cite web |url = http://www.computerandvideogames.com/370841/dishonored-goes-gold/ |title = Dishonored goes gold |author = Tom Ivan |date = 2012-10-01 |publisher = [[電腦與電子遊戲|電腦與電子遊戲]]}}</ref><ref>{{Cite web |url = http://www.cinemablend.com/games/Dishonored-Goes-Gold-Free-Copy-Up-Grabs-47484.html |title = Dishonored Goes Gold, Free Copy Up For Grabs |author = Pete Haas |date = 2012-09-28 |publisher = CinemaBlend}}</ref>。

===游戏性===
開發團隊研究了主角的超能力搭配（例如「敏捷」配合「瞬移」的移動距離相比兩項能力各自覆蓋的距離為高），並把各搭配列進任務設計考慮內而非加以限制<ref name="Setting3"/><ref>{{Cite web |url = http://penny-arcade.com/report/editorial-article/dishonoreds-two-lead-designers-talk-about-magic-breaking-the-game-and-amora |title = Dishonored's two lead designers talk about magic, "breaking" the game, and amoral power |author = Ben Kenchera |date = 2012-06-25 |deadurl = yes |archiveurl = https://www.webcitation.org/6EX9NcTR7?url=http://penny-arcade.com/report/editorial-article/dishonoreds-two-lead-designers-talk-about-magic-breaking-the-game-and-amora |archivedate = 2013-02-18 }}</ref>，部分能力於開發階段經過數度修改<ref>{{Cite web |url = http://www.totalxbox.com/46721/features/dishonored-seven-things-arkane-cut-to-improve-the-game/?page=1 |title = Dishononored: seven things Arkane cut to improve the game |author = Edwin Evans-Thirlwell|date = 2012-10-17 |publisher = TotalXbox}}</ref>。遊戲總監對於平衡各能力感到困難，表示他們賦予玩家強大的能力但不希望遊戲因此變得過於容易。團隊利用有效時限、魔力消耗量及其他特性使強大的能力無法被大量使用<ref>{{Cite web |url = http://www.joystiq.com/2012/08/06/dishonoreds-chaos-morality-and-potentially-over-powered-hero/ |title = Dishonored's chaos, morality and potentially over-powered hero |author = Britton Peele |publisher = Joystiq}}</ref>。

潛行系統原本源自以光線與黑影決定玩家是否會被發現之《[[神偷：暗黑計劃|神偷]]》系列，但因玩家能在敵人正前方之黑影區域安全躲藏而被認為不真實，而且黑影將掩蓋地圖設計和影響色彩對比<ref>{{Cite web |url = http://www.eurogamer.net/articles/2012-09-29-why-dishonored-ditched-its-thief-shadow-stealth-mechanic |title = Why Dishonored ditched its Thief shadow stealth mechanic |author = Robert Purchese |date = 2012-09-29 |publisher = [[Eurogamer|Eurogamer]]}}</ref>。為免潛行的玩家因長時間停留於同一地區而感到沈悶，遊戲大部分背景對話被設定得較長和提供遊戲背景資訊，相反主線故事對話被設定得較短，讓玩家能打斷說話或殺死講者<ref>{{Cite web |url = http://kotaku.com/5950999/in-dishonored-sometimes-the-story-is-what-you-dont-see |title = In Dishonored, Sometimes The Story Is What You Don't See |author = Evan Narcisse |date = 2012-11-10 |publisher = [[Kotaku|Kotaku]]}}</ref>。

設計師在任務設計上先設定一個為玩家提供大量活動的地區，然後設計和擴充前往任務地點的路線，再加入於固定巡邏路線之非玩家角色及設定其作用。設計師其後觀察玩家如何運用能力以了解該地區是否能提供足夠的挑戰性並在有需要時加以修改。起初任務為強調能力使用的自由而只提供少量前進方向指引，但經測試後發現玩家迷失方向或因服從非玩家角色的指示不進入某些地區導致遊戲無法繼續，設計師隨後增加更多視覺與對話提示協助玩家前進<ref>{{Cite web |url = http://games.on.net/2012/09/dishonored-interview-arkane-on-stealth-ai-player-choice-and-much-more/ |title = Dishonored Interview: Arkane on stealth, AI, player choice and much more |author = Tim Colwill |date = 2012-09-20 |publisher = games.on.net}}</ref>。

===设计===
[[File:Grimshaw-NightfallThames.jpg|thumb]]
遊戲背景原設定於日本封建時代，但由於設計團體對它並不熟悉以及推廣時遇上困難而作廢<ref>{{Cite web |url = http://www.gamespot.com/articles/the-dishonored-that-never-was/1100-6406136/ |title = The Dishonored that never was |author = Shaun McInnis |date = 2013-03-28 |publisher = [[Gamespot|Gamespot]]}}</ref>。Arkane工作室其後把時代設定於歐洲人與美國人均熟悉之1666年[[倫敦|倫敦]]，並在後期設計加入了照射燈、電牆及二十世紀科技，由於背景已不再與倫敦相同，因此背景被設定為虛構城市丹沃爾市<ref name="system shock" />，包含十九世紀末至二十世紀初倫敦和[[愛丁堡|愛丁堡]]之影子。哈維·史密斯對此表示1666年乃[[倫敦大火|倫敦大火]]與[[倫敦大瘟疫|倫敦大瘟疫]]結束之年，而這類遊戲應能營造緊張氣氛和危機感，同時解釋為何市集沒有大批民眾。維克托·安東諾夫也表示倫敦是「一個混亂和緊張的大都會，對美國人和歐洲人而言既熟悉也有異國風情。在構思新作品時希望能引起大量人士的公鳴」，同時指出愛丁堡提供了不少包含未來色彩之建築設計<ref>{{Cite web |url = http://www.eurogamer.net/articles/2012-07-18-its-been-a-poor-poor-five-years-for-fiction-in-the-video-game-industry |title = It's been a poor, poor five years for fiction in the video game industry |author = Robert Purchese |date = 2012-07-18 |publisher = [[Eurogamer|Eurogamer]]}}</ref>。為作資料搜集，二人曾到倫敦與愛丁堡拍攝市民、街道和物件，並集中於能切合遊戲世界之巷子和偏街<ref>{{Cite web |url = http://www.develop-online.net/news/dishonoured-wouldn-t-exist-without-pre-production/0112491 |title = Dishonored wouldn't exist without pre-production |author = Aaron Lee |date = 2012-07-12 |publisher = developer}}</ref>。

遊戲角色之設計靈感來自倫敦及澳洲之冒險與海盜故事<ref>{{Cite web |url = http://www.pcgamesn.com/dishonored/dishonored-began-life-game-set-medieval-japan |title = Dishonored began life as a game set in Medieval Japan |author = Jeremy Peel |publisher = PCGamesN}}</ref>，一名專家在Arkane工作室堅持[[寫實主義|寫實主義]]和[[政治不正確|政治不正確]]時協助確保角色臉孔能代表英國<ref name="system shock" />，遊戲畫家賽巴斯汀·米頓（Sébastien Mitton）把貴族、貧民與敵人等角色配上相應的動作設計，由動畫師把動作呈現以加強情感表現，例如守衛頭部較小、肩膀較低、手部較大、動態包含了人類與猴子的動作<ref>{{Cite web |url = http://www.imaginefx.com/02287754332481410212/antiquated-urbanism-sebastien-mitton-and-viktor-antonov-on-dishonored.html |title = Antiquated Urbanism: Sebastien Mitton and Viktor Antonov on Dishonored |publisher = ImageFX}}</ref>。

[[File:Dishonored_Tallboy.jpg|150px]]
遊戲內一款敵人「高督兵」剛開始被設定為街頭公告員，其後在賽巴斯汀·米頓看見有人穿上高腳裝備清潔他們的辦公室正面後增加了高腳，公告員角色也被改為裝置在丹爾沃市內的擴音器。高督兵的設計演化至裝配著鯨油箱點燃街燈的點燈者，但設計員在進一步考慮後認為他們的機械高腳使他們能從高處避免受鼠疫感染，同時能焚燒死者和對抗患病市民，因此最終被設定為配備弓箭的守衛<ref name="peeling back">{{Cite web |url = http://www.eurogamer.net/articles/2012-11-07-peeling-back-the-layers-of-dishonored-with-harvey-smith-and-raph-colantonio |title = Peeling back the layers of Dishonored with Harvey Smith and Raph Colantonio |author = Tom Bramwell |date = 2012-11-07 |publisher = [[Eurogamer|Eurogamer]]}}</ref><ref>{{Cite web |url = http://kotaku.com/5953087/dishonoreds-art-director-talks-inspiration-cut-levels |title = Dishonored's Art Director Talks Inspiration, Cut Levels |author = Luke Plunkett |date = 2012-10-18 |publisher = [[Kotaku|Kotaku]]}}</ref>。[[File:Hound_Pits_Pub.jpg|thumb]]另一方面，作為玩家基地之獵犬酒吧於早期乃更大的建築物，但完成室內設計需要太多樓梯和房間，而且大量的攀爬容易迷失方向。設計團隊發現保留原有外觀同時設計較細小和較容易辨別方向的室內裝潢具挑戰性。酒吧被縮小至一幢帶有[[愛德華時代|愛德華時代]]色彩之建築，但室內設計仍像一個迷宮，因此維克托·安東諾夫提出在科爾沃寢室外加設一道鏈子方便進出，設計團隊最終決定把酒吧三樓關閉<ref name="peeling back" />。整個開發階段中，團隊一直收到不同要求，導致他們需要延長早期製作直至開發過程的結尾。所有設計均由人手繪製<ref name="system shock" />。

遊戲內一項道具「心臟」（Heart）乃經科技與超自然力量改造之人類心臟，協助玩家尋找任務內的可收集物品，亦為玩家提供遊戲地點與人物的資訊。心臟之早期設定為透過震動與聲效判定刺殺對象的工具，其後發展為能夠說話和擁有自我意識。在經過試玩測試後，設計師認為遊戲需要一個更直接的方向引導系統。心臟於遊戲中的角色一直有所改變，亦一直為遊戲內不同角色提供描述，協助突出遊戲主題和以更低調的方式區分市內不同階層。哈維·史密斯和拉斐爾·哥蘭東尼奧曾擔心作為選擇性道具的心臟將使部分玩家錯失其提供的資訊，但認為這是遊戲提供的自由<ref>{{Cite web |url = http://www.rockpapershotgun.com/2012/12/19/the-gaming-pulse-on-dishonoreds-heart/#more-135985 |title = The Gaming Pulse: Dissecting Dishonored's Heart |author = Paul Walker |date = 2012-12-19 |publisher = [[Rock,_Paper,_Shotgun|Rock, Paper, Shotgun]]}}</ref>。

===音乐===
遊戲配樂由丹尼爾·李希特（Daniel Licht）創作；以[[小提琴|小提琴]]為主之背景音樂代表十九世紀之倫敦<ref>{{Cite web |url = http://herocomplex.latimes.com/games/for-composers-video-games-are-the-surreal-land-of-opportunity/ |title = For composers, video games are the surreal land of opportunity |author = Todd Martens |date = 2012-09-28 |publisher = [[洛杉磯時報|洛杉磯時報]]}}</ref>。丹尼爾·李希特之設計理念為讓玩家感到緊張而非作為舒緩，並以「邪惡之霧」形容配樂：音樂於遊戲中滲入和滲出以營造持續的危機感，而非跟隨預設劇情或作為危機的警告。由於遊戲內描述角色間交流的電影劇情片段不多，丹尼爾·李希特認為創作角色主題音樂的機會相當有限，因此集中設計背景音樂。遊戲主題曲「全人之榮譽」（Honor for All）由丹尼爾·李希特及其姪兒鍾恩（Jon）編寫，歌唱由後者負責<ref>{{Cite web |url = https://archive.org/details/top_score_daniel_licht_dishonored_20130206_128 |title = top_score_daniel_licht_dishonored_20130206_128 |publisher = Minnesota Public Radio}}</ref>。

作為遊戲推廣活動一部分，貝塞斯達僱用了音樂公司「副駕駛策略音樂＋聲效」（Copilot Strategic Music + Sound），以傳統[[船夫號子|船夫號子]]「醉酒水手」（Drunken Sailor）為藍本而創作了「醉酒捕鯨者」（Drunken Whaler）<ref>{{Cite web |url = http://www.vg247.com/2012/09/17/dishonored-drunken-whaler-available-for-download-remixing/ |title = Dishonored Drunken Whaler available for download |author = Brenna Hillier |date = 2012-09-17 |publisher = VG247}}</ref>。為使音樂帶有黑暗和不祥的色彩，Copilot決定歌詞由兒童唱出而非專業的兒童[[合唱團|合唱團]]，但因歌詞涉及割喉和餓鼠而在當地學校僱用兒童方面遇上困難，因此最終邀請了兒童演員、能模仿兒童之成年[[歌手|歌手]]、以及他們朋友的子女擔任歌唱角色。樂器包括小提琴、改裝[[結他|結他]]、以及一個由團隊開發、踏在木質地板上能營造心臟悸動聲效之「捕鯨踏板」（whaler stomp）。「醉酒捕鯨者」被收錄在遊戲[[預告片|預告片]]中，並在2012年[[E3|E3]]中獲得正面評價。相關預告片在[[YouTube|YouTube]]上之點擊率超過八十五萬並獲得[[Machinima|Machinima]]最佳預告片獎<ref>{{Cite web |url = http://www.copilotmusic.com/dishonored/ |title = Case Study: "The Drunken Whaler" for the Dishonored gameplay trailer |date = 2012-09-17 |publisher = Copilot Strategic Music + Sound}}</ref>。

遊戲普遍獲得正面評價。[[IGN|IGN]]雖然表示劇情相對於遊戲的難忘時刻而言較弱，但認為這是一款玩家會與朋友提及的遊戲；[[Eurogamer|Eurogamer]]、[[The_Verge|The Verge]]和Giant Bomb批評遊戲後期的任務沒甚特別、後期敘述出錯、以及結局倉促。Giant Bomb亦批評任務內的周遭對白過於重複；Eurogamer認為獨特、創新和突出的任務使遊玩過程令人難忘而且超越了劇情。Joystiq對倉促的結局感到失望，並認為劇情未能讓玩家探索丹沃爾市，但整體而言使他希望繼續遊玩<ref>{{Cite web |url = http://www.joystiq.com/2012/10/08/dishonored-review/ |title = Dishonored review: A whale of a tale |author = Alexander Sliwinski |publisher = Joystiq}}</ref>。The Escapist認為劇情進度良好；Shacknews表示遊戲能使人投入情感且有角色扮演感，但部分劇情未有充分解釋或發揮<ref>{{Cite web |url = http://www.shacknews.com/article/76096/dishonored-review-story-driven-role-playing# |title = Dishonored review: Story-driven role-playing |author = John Keefer |date = 2012-10-07 |publisher = Shacknews}}</ref>。

評論家對遊戲容許玩家以不同方式完成目標和探索任務予以讚揚。Shacknews寫道主角的能力和裝備沒有使劇情減弱，而只是作為輔助品讓玩家利用創意和智慧完成任務，但由於部分任務自由度過高，玩家有可能不知道應採取甚麼行動。The Verge認為從關卡設計而言，遊戲早期看似沒甚組織，但探索後能發現設計師花了不少心思思考玩家會採取的行動，讓他們不必跟隨特定的做法而能作多方面嘗試。[[GameSpot|GameSpot]]享受於任務內透過有趣的能力選擇不同的路徑和方法的過程，並認為高度的自由使《冤罪殺機》成為2012年其中一款最突出的遊戲。The Escapist表示遊戲的自由度亦是缺點，部分遊玩方式使玩家必定選用某些能力，而這些能力升至最高等級後將使玩家不願徹底探索任務。[[Game_Informer|Game Informer]]也認為遊戲鼓勵潛行，懲罰選擇某些路徑（尤其是暴力）的玩家，使遊玩過程變得困難和結局較令人不滿。[[Kotaku|Kotaku]]把《冤罪殺機》與其他現代遊戲作出比較後表示其他遊戲或要求玩家找出特定方法擊敗敵人或解謎，《冤罪殺機》使玩家代入設計師角色，提供一系列工具讓玩家體驗它們如何與遊戲世界互動。

評論家認為遊戲變化提高重玩價值。[[電腦與電子遊戲|電腦與電子遊戲]]讚揚遊戲的多變性和重玩性，認為遊戲不迫使玩家接觸所有元素，但能使玩家有此衝動。News.com.au寫道遊戲的唯一缺點是提供了太多選擇；GameSpot也寫道遊戲的重玩性達到了強迫性的程度。
