{{Vgsname|愤怒的小鸟|en=Angry Birds|cn=愤怒的小鸟|tw=憤怒鳥}}是芬兰游戏公司[[Rovio娱乐|Rovio娱乐]]出品的电子游戏系列。系列作品主要面向[[智能电话|智能电话]]，玩法以控制彈弓發射小鳥、打擊建築物和小豬的[[益智游戏|益智游戏]]为主。但之后系列登上[[游戏机|游戏机]]平台，并涉足[[竞速游戏|竞速]]和[[卷轴射击|卷轴射击]]等游戏类型。此外系列还有改编动画等跨媒体作品。

== 影响 ==

=== 流行文化 ===


在香港，憤怒鳥的公仔成為2011年[[年宵|年宵]]市場大賣的產品之一<ref>[http://hk.news.yahoo.com/article/110128/4/mgpr.html 「憤怒鳥」飛入維園 「讚好」手套熱賣]，2011年1月29日[[明報|明報]]</ref>。

一名俄國動畫作家Egor Zhgun，以「憤怒鳥」遊戲與兒童故事「三隻小豬」作題材製作動畫短片「三隻大豬」，當中的三隻大豬代表在[[2010－2011年阿拉伯世界的反政府示威|2010－2011年阿拉伯世界的反政府示威]]中遭罷黜下台的[[突尼西亞|突尼西亞]]總統[[宰因·阿比丁·本·阿里|賓阿里]]、同樣已下台的[[埃及|埃及]]領袖[[穆巴拉克|穆巴拉克]]及半年后身亡的[[利比亞|利比亞]]狂人[[卡達菲|卡達菲]]<ref>[http://www.youtube.com/watch?v=q0i9acHS_zQ 「三隻大豬」影片 (From YouTube)]</ref>。

[[以色列|以色列]]最受歡迎的[[政治戲仿節目|政治戲仿節目]]《'''Eretz Nehederet'''》製作短片《[[憤怒鳥的和平條約|憤怒鳥的和平條約]]》（英語：''Angry Birds Peace Treaty''）諷刺困難重重的以色列與[[巴勒斯坦|巴勒斯坦]]和平進程。

== 争议 ==

2014年1月26日，[[愛德華·斯諾登|愛德華·斯諾登]]向媒體爆料，[[美国国家安全局|美国国家安全局]]和[[政府通信总部|英国政府通讯总部]]利用各种特殊手段，从愤怒的小鸟、谷歌地圖等[[流動軟件應用程式|APP]]中获取了大量私人信息<ref>[http://www.evolife.cn/html/2014/75413.html 愤怒的小鸟都逃不了！NSA从手机游戏中获取个人信息]</ref><ref>[http://www.ithome.com.tw/itadm/article.php?c=85034 美英情報單位以憤怒鳥、Google Maps蒐集個人資料] {{webarchive|url=https://web.archive.org/web/20140203063913/http://www.ithome.com.tw/itadm/article.php?c=85034 |date=2014-02-03 }}</ref>。

Rovio公司隨後表示，不会向上述机构提供任何用户数据<ref>[http://www.rovio.com/en/news/press-releases/450/rovio-does-not-provide-end-user-data-to-government-surveillance-agencies Rovio does not provide end user data to government surveillance agencies] {{webarchive|url=https://web.archive.org/web/20150906155455/http://www.rovio.com/en/news/press-releases/450/rovio-does-not-provide-end-user-data-to-government-surveillance-agencies/ |date=2015-09-06 }}</ref>。

1月31日，「憤怒的小鳥」官網被駭客入侵，美國國安局標識和「間諜鳥」標題取代了原本的圖像<ref>[http://www.cna.com.tw/news/ait/201401300188-1.aspx Rovio網站遭駭 憤怒鳥變間諜鳥]</ref>。
