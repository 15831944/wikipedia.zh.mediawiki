'''突厥斯坦'''（[[察合台語|察合台突厥语]]：ترکستان / Türkistan；{{lang-ota|ترکستان  / Türkistan}}；{{lang-fa|ترکستان / Torkistan}}；{{lang-ar|الترکستان / al-Turkistan}}；{{lang-en|Turkistan / Turkestan}}），旧译'''土耳其斯坦'''，可以指：

== 地理范畴 ==
* [[突厥斯坦|突厥斯坦]]：意为“突厥人所居之地”，指中亚地区使用突厥语族语言的多个民族所生活的地区，其具体的地理范围在不同时期、不同文献中各有不同。
** [[西突厥斯坦|西突厥斯坦]]：或称俄国突厥斯坦，大致相当于原俄属中亚，今中亚五国。
** [[东突厥斯坦|东突厥斯坦]]：或称中国突厥斯坦，塔里木盆地或新疆及其周边的部分地区。
** [[南突厥斯坦|南突厥斯坦]]：或称阿富汗突厥斯坦，阿富汗北部兴都库什山脉以北地区。

== 区划 ==
* {{tsl|ru|Туркестанская область (Российская империя)|突厥斯坦州 (俄罗斯帝国)}}：俄国于1865年建立，1867年废止的行政区划。
* [[突厥斯坦总督区|突厥斯坦总督区]]：俄国于1867年建立，1918年废止的行政区划。
* [[突厥斯坦苏维埃社会主义自治共和国|突厥斯坦苏维埃社会主义自治共和国]]：由苏俄在原突厥斯坦总督区建立的自治共和国，1924年废除。
** {{le|突厥斯坦军区|Turkestan Military District}}，以突厥斯坦为核心战斗区域的军区。第一次由沙俄设立于1874年，终止于1918年。第二次由苏联设立于1926年，终止于1992年。
** [[突厥斯坦總督轄區|突厥斯坦總督轄區]]：纳粹德国在第二次世界大战中曾计划在击败苏联后在中亚建立的行政区，但并未实行。
* {{le|突厥斯坦省|Turkestan Province}}：阿富汗1929年设立，1946年废止的行政区划。
* [[突厥斯坦州|突厥斯坦州]]：哈萨克斯坦的行政区划，原名南哈萨克斯坦州，2018年改名。
** [[突厥斯坦_(哈薩克斯坦)|突厥斯坦 (哈薩克斯坦)]]：突厥斯坦州首府城市。

== 山脉 ==
* [[突厥斯坦山脈|突厥斯坦山脈]]：中亚的一条山脉。

== 建筑 ==
* {{le|突厥斯坦宫|Turkiston Palace}}：乌兹别克斯坦首都塔什干的一座艺术馆。
