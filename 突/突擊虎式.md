'''突擊虎式'''（[[德文|德文]]：Sturmmörser Tiger／Sturmpanzer VI），為[[納粹德國|德國]]在[[第二次世界大戰|第二次世界大戰]]中使用的一款突擊炮。它以[[虎I坦克|虎I坦克]]的底盤作為基礎，並使用380毫米SturmMörser「突擊迫砲」RW61 L/3.5這款大型的海軍用深水炸彈發射器。它原本是作為在市區作戰中為[[步兵|步兵]]提供重火力的支援，但在整場戰爭間卻只生產了18輛，全以受損的虎I坦克改造而成。

突擊虎式自走砲也被稱為「虎式迫砲」（Tiger-Mörser），「突擊迫砲虎」（Sturmmörser Tiger），或者是「六號突擊戰車」（Sturmpanzer VI）。（德文中的「[[:de:Mörser_(Geschütz)|Mörser]]」即為英文的「[[:en:Mortar_(weapon)|Mortar]]」之意）。

== 發展 ==
自從[[史達林格勒戰役|史達林格勒戰役]]開始後，原本在俄羅斯大草原上進攻順利的[[德意志國防軍|德意志國防軍]]，終於在加強防禦的市街建築物與區域中遇到城鎮戰困境，所以認為有必要生產用作支援步兵的重火力戰鬥車輛。

當時國防軍只有突擊步兵砲33型（Sturminfanteriegeschütz 33B；[[:de:StuIG_33_B|StuIG 33 B]]），就是以150公厘[[sIG_33榴彈炮|sIG 33榴彈炮]]改裝在[[3號坦克|3號坦克]]的底盤上（Panzerkampfwagen III）權充為砲車作為摧毀建築物的唯一火力，不過僅24輛並且有一半都在[[史達林格勒戰役|史達林格勒戰役]]中損失。[http://de.wikipedia.org/wiki/StuIG_33_B]

1943年初以[[四號坦克|四號坦克]]底盤發展用來替代步兵砲33型的[[灰熊式突擊砲|灰熊式]]（Sturmpanzer IV Brummbär）自走砲登場；但是國防軍還是認為不夠，還是需要支援步兵的[[自走砲|自走砲]]，但是[[裝甲|裝甲]]要更厚火力要更強，所以就產生了以[[虎式戰車|虎式戰車]]為底盤搭配210公厘榴彈砲的構想。

1943年9月[[克魯伯|克魯伯]]（Krupp）就以虎I坦克的底盤作為設計依據，然後將車體送往亨舍爾（Henschel）進行底盤的組合，最後再到阿凱特（Alkett）完成車體上層結構；原型車在一個月後被送往[[希特勒|希特勒]]面前過目。量產第一台車體在12月開始移交到了隔年1944年2月20日阿凱特就完成了3台突擊虎式砲車。

由於工期被延後，希特勒在4月19日之前都沒要求量產的數量與速度要增加；不過已經有12輛突擊虎式砲車的戰鬥室與火砲已經準備好要安裝了。8月的時候阿凱特又完成了3輛重製的突擊虎式砲車，原本計畫在9月20日之前再完成另外7輛搭配380公厘迫擊砲德突擊虎砲車，因希特勒已經在8月20日會議中被面呈過。

終於在9月中完成了10輛突擊虎砲車；年底又完成了5輛。希特勒這時戰局轉敗，希望寄託在一切新式武器，所以對突擊虎砲車打算進行特別的運用，還預計每個月至少要製造300發突擊虎砲的彈藥才夠用。

當東部前線德軍還在為陷入膠著狀態著戰局想要提供曲射火力支援的武器，後來聽說新武器是海軍380公厘反潛[[深水炸彈|深水炸彈]]發射器改裝成火箭砲版的自走迫砲車（mortar rocket launcher），一時之間使得官兵們處於傻眼的狀態。有官兵說突擊虎自走砲簡直是“插著樹樁的怪物”。

== 設計 ==

突擊虎的車體是以晚期的[[虎I戰車|虎I戰車]]為主，所以連同懸吊系統也相同。原來的結構也被移除用來容納新的火砲，加上火砲前移，等到裝甲安裝好之後，一個四方典型德國坦克外觀就出現了。

跟虎I式坦克比起來，突擊虎砲車雖然在車長數字上明顯短得多，只有6.28公尺長，遠短於虎式的8.45公尺，但其實主要的原因是主砲不同造成的砲管長度差，車體底盤本身並沒有刻意縮短。車高也比虎式矮15公分（另一說是8公分）。不過這種身材在市街中反而遊刃有餘，進退自如，也沒有在街道上被有限空間限制的干擾的長管主砲；重要的是短砲管所形成的高仰角射擊，對於藏匿在塔樓或高處的狙擊手有毀滅性。

== 裝甲 ==
[[File:Sturmtiger_Rückseite.jpg|thumb]]
突擊虎是被設計出來在街道區推進，裝甲必須厚到適合近距戰，所以它的正面裝甲厚達150[[公厘|公厘]]，再加上傾角約45°左右，相當於厚達200公厘；側邊與後方的裝甲均厚80公厘，所以突擊虎跟[[虎II坦克|虎II坦克]]一樣重達65噸。

== 武裝 ==
380公厘的「Raketenwerfer」（rocket launcher）61型5.4倍徑是一種後膛裝填的火箭(迫擊)砲，亦即發射的物體為火箭彈，但是可以像迫擊砲一樣以仰角發射以增加射程，或者射擊制高點；砲膛來福線繞距為205.4公分。

火箭彈體就長達1.5[[公尺|公尺]]，彈頭為125[[公斤|公斤]]的高爆炸藥或者是成型裝藥（[[:en:shaped_charge|shaped charge]]）；如果目標是強化據點或碉堡，380公厘的成型裝藥彈頭可以貫穿2.5公尺的水泥牆（理論上這種火箭彈對防空塔「[[:en:Flak_tower|Flak tower]]」上3.5公尺的外牆同一個點射擊兩次就會穿透）。

火箭迫擊砲使用高爆炸藥彈頭的射程可達5,650公尺，彈體重達345~351公斤。發射時的發射藥會將彈體以每秒40公尺的速度推出砲管，彈體本身40公斤的燃料點燃後再將彈體推送到每秒250公尺的速度飛向目標。
[[File:Sturmtiger_Rohrinneres.jpg|thumb]]
另外火箭迫砲的前端有個特殊的外觀，就是砲口外有一圈排氣孔，搞的砲管看起來有如餐桌上的胡椒罐（Pepperbox）。這圈排氣孔的設計原因在於火箭發射時會產生高壓且有毒的廢氣瓦斯，如果發射後貿然打開砲栓只會造成全車官兵悲劇性的下場，況且發射時的廢棄瓦斯會造成過高的膛壓，對武器本身也會造成損害，因此排氣孔的設計就是達成毒瓦斯的排洩與降低膛壓的效果。
[[File:Germany06_086.jpg|thumb]]市的德國裝甲博物館（[[:en:Deutsches_Panzermuseum|Deutsches Panzermuseum]]）中的突擊虎正面圖；從照片中可以看到有如胡椒罐的砲口以及右前方的MG34機槍。]]
龐大笨重的彈體只能讓突擊虎帶14發上戰場，一發已經先在後膛，另一發倒在載彈板上；剩下的十二發全部都在後面的彈架上。車體後面的小吊臂，用途就是把幾乎有350公斤重的火箭彈吊起來，經過給彈艙口然後放置到載彈板上，再準備進入後膛。

彈載量低所以就要有彈藥車跟隨，而這台彈藥車也是用虎式戰車改裝而成的，但是僅只有一台完工。

在吊臂後方還有一座90公厘口徑NbK 39 [[:en:Nahverteidigungswaffe|Nahverteidigungswaffe]]（"Close defence weapon"）近距離防衛武器。

說它是防衛武器不如說它是個「被固定住朝天空發射但是沒槍管的大型左輪手槍」，它有轉盤式彈艙（rotating launcher）在車輛戰鬥室中，每個彈艙中可以置放煙霧彈製造掩護的煙幕；或者裝置曳光彈（Flare）發射攻擊、會師、或者求救訊號；如果敵人步兵蜂擁而上包圍攻擊突擊虎的時候，乘員可以從車內發射彈艙內的高爆榴彈，這個高爆榴彈就會一飛沖天，大約是兩層樓高的高度爆炸，散射出許多碎片殺傷包圍車輛的步兵；車體右前方也有針對軟性目標的7.92公厘[[MG34|MG34]]機槍的配置，彈載量為800發。

== 戰鬥紀錄 ==
突擊虎是用來支援步兵的重裝車輛，幫助步兵攻擊頑抗的敵軍碉堡或強化據點。不過等到突擊虎上戰場的時候，德軍情勢已經轉為守勢，反而本車主要用於防禦戰。

一共3個連被編成突擊虎連（Panzer Sturmmörser Kompanien，「PzStuMrKp」）：第1,000連，第1,001連以及第1,002連。本來預計一個連配14輛突擊虎，結果最後一個連只配4輛。第1,000連於1944年8月13日成軍，主要參加擊敗[[華沙起義|華沙起義]]，連原型車都加入了鎮壓的戰鬥，而且這些個突擊虎也就真正的發揮它們攻堅的用途。

1,001與1,002突擊虎連最後也在9月以及10月加入鎮壓，隨後轉往阿登地區，以總數7輛加入了[[突出部之役|突出部之役]]。阿登攻勢結束後，突擊虎又轉為守西線；在槍林彈雨當中突擊虎幾乎以厚裝甲防護力打死不退，盟軍與蘇軍如果不是以戰術支援轟炸機或者是口徑更大的火砲來對付突擊虎，盟軍與蘇軍幾乎就被突擊虎牢牢牽制住，進退不得。

有幾輛突擊虎與在戰場損失；剩下來的突擊虎由於機械故障或者燃油不足而被破壞後遺棄。1945年初期挺近德国的战役中，在边境线顽抗的一辆突击虎曾经一发炮弹击毁了属于第3装甲师的3辆[[M4雪曼戰車|雪曼]][[M4A3|M4A3]]坦克

最後有2輛突擊虎活過二戰，一輛是陳列在德國裝甲博物館的突擊虎；另外一輛被俄國拖回庫賓卡戰車博物館（[[:en:Kubinka_Tank_Museum|Kubinka Tank Museum]]），跟[[八號坦克鼠式|八號坦克鼠式]]一起見證德國的興亡。至於胡椒罐大砲則被收藏到「波文頓坦克博物館」（[[:en:Bovington_Tank_Museum|Bovington Tank Museum]]）中展覽。
