'''突厥字母'''（Old Turkic script、Göktürk script）又稱'''鄂爾渾文字'''（Orkhon script）、'''鄂爾渾-葉尼塞文字'''（Orkhon-Yenisey script），是[[古突厥|古突厥]]人创造并使用的一种文字。{{fact|因其形似欧洲古代的[[卢恩字母|如尼字母]]，故被称为'''突厥如尼字母'''。|time=2018-04-15}}[[古突厥语|古突厥语]]最早的文字记录即是以'''突厥字母'''为载体的。8世纪的[[鄂尔浑碑铭|鄂尔浑碑铭]]（分布在[[鄂尔浑河|鄂尔浑河]]流域的[[阙特勤碑|阙特勤碑]]、[[毗伽可汗碑|毗伽可汗碑]]等碑铭）与9世纪的[[叶尼塞碑铭|叶尼塞碑铭]]（分布在[[叶尼塞河|叶尼塞河]]流域的一系列碑铭）均以这种文字完成。突厥字母从右往左书写，[[鄂尔浑碑铭|鄂尔浑碑铭]]与[[叶尼塞碑铭|叶尼塞碑铭]]所使用的字母稍有不同。

==Unicode==
Unicode 6.0標準給古突厥文分配的範圍是U+10C00至U+10C4F。

現在支援這些字元的字體有 [[Windows_10|Windows 10]] 內建的 Segoe UI Historic<ref>[https://msdn.microsoft.com/en-us/goglobal/bb688099.aspx#W10 MSDN: Script and Font Support in Windows]</ref>、[[Google|Google]] 開發的 Noto Sans Old Turkic 以及 BabelStone Irk Bitig<ref>[http://www.babelstone.co.uk/Fonts/IrkBitig.html BabelStone Fonts: BabelStone Irk Bitig]</ref>。
