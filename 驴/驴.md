'''驴'''（[[学名|学名]]：{{lang|la|''Equus africanus asinus''}}）是常见的[[马科|马科]][[马属|马属]]家畜，是[[非洲野驴|非洲野驴]]被人类驯化所形成的[[亚种|亚种]]，和[[马|马]]体形相似，但耳朵长，尾巴有尾柄，类似牛尾巴。

== 特征 ==
驢是[[奇蹄目|奇蹄目]]的成員。其他成員包括[[斑馬|斑馬]]和[[馬|馬]]，它們都有相同形態的腳[[蹄|蹄]]。Holacomo estas kakaroto, biba gocu

驴体色一般为灰色，也有白色和黑色品种，但都有一个白色眼圈。由于不同品种，体形大小相当不同，小的类似大狗，大的和马一样高。中国的“[[关中驴|关中驴]]”十分高大，有1.7米。

驢叫聲響亮且高尖刺耳。

== 历史 ==
驴在人类[[历史|历史]]记载和[[神话|神话]]中经常出现，与骑马的八面威风不同，骑驴是超凡脱俗的风雅之举。

* 聖經記載[[耶稣|耶稣]]就是骑驴进入[[耶路撒冷|耶路撒冷]]{{noteTag|参见：[[聖枝主日|聖枝主日]]}}。

* 中亚民间传说中的智者[[阿凡提|阿凡提]]骑驴。

* [[塞万提斯|塞万提斯]]的《[[堂吉诃德|堂吉诃德]]》中，唐吉诃德骑马，其侍从桑丘就骑驴。

* [[中国|中国]]古代成语“黔驴技穷”讲的是身高马大的[[关中|关中]]驴剛开始能吓坏没有见过驴的[[老虎|老虎]]。

* [[八仙|八仙]]中的[[張果老|张果老]]骑驴。

* [[三国演义|三国演义]]中[[黄承彦|黄承彦]]“骑驴过小桥，独叹梅花瘦”。

* [[水浒传|水浒传]]中[[陈抟|陈抟]]骑驴。

== 繁殖 ==
公驴可以和母马交配，生下[[骡|骡]]（“马骡”），骡个大，具有驴的负重能力和抵抗能力，有马的灵活性和奔跑能力，是非常好的役畜，但基本不能生育，极少数可以生育。如果是公马和母驴交配，生下的叫“驴骡”，又叫“駃騠”，驴骡个小，一般不如马骡好，但有时能生育。

== 用途 ==

=== 畜力 ===
驴比马的适应性强，可以忍受粗食、重负，比马的价格低，因此一直是人类的重要役使动物。有的小驴可以驮负相当高大的人。 一般驢的負重可以達到一百公斤，連續走動五至六小時的山路沒有問題。驴对危险相当警觉，因此在某些对自己有危险的情况下不听人的驱使，显得相当执拗，所谓“驴脾气”。但一般情况下脾气温顺，可以服从孩子的调遣，因此现在不再使用役用动物的发达国家，有许多人把小型驴作为[[宠物|宠物]]，可以供孩子骑乘。

=== 食用 ===
驢肉可食用，清朝有“生挫驢肉”的吃法<ref>[[錢泳|錢泳]]的《履園叢話》：“以草驢一頭，養得極肥，先醉以酒，滿身排打。欲割其肉，先釘四樁，將足捆住﹔而以木一根橫于背，系其頭尾， 使不得動。初以百滾湯沃其身，將毛刮盡，再以快刀零割。 要食前后腿，或肚當，或背脊，或頭尾肉，各隨客便﹔當客下箸時，其驢尚未死絕也。”</ref>。中國[[河北|河北]][[保定|保定]]有著名小吃“[[驴肉火烧|驢肉火燒]]”，是在烙熟的麵餅中加入熟驢肉而成。驢腸也是一道名菜，北宋[[韓縝|韓縝]]愛吃驢腸，每次宴客必用驢腸做菜，而且要現殺現煮，有客人起身如廁，路過廚房，現場看見活驢被剖肚抽腸的慘狀，心有不忍，終生不再吃驢肉。<ref>[[洪邁|洪邁]]《[[夷堅志|夷堅志]]》卷十六</ref>明朝隆慶皇帝[[朱載垕|朱載垕]]在裕王府時期好食驢腸，即位後，頗思節儉，聽說[[光禄寺|光禄寺]]每天要殺一只驢，就不再吃了。<ref>《明實錄》記載：“潛邸時，嘗食驢腸而甘。及即位，間以問左右，左右請詔光祿。上：‘若爾，則光祿必日殺一驢，以備宣索，吾不忍也。’乃止。”</ref>
