== 作品概要 == 
本作起初於月刊《Miracle Jump》2015年6月號至2016年2月號連載，後移籍至《[[週刊YOUNG_JUMP|週刊YOUNG JUMP]]》，於2016年17號開始連載。

本作另有[[衍生作品|衍生作品]]：在《[[週刊YOUNG_JUMP#隔壁的YOUNG_JUMP|隔壁的YOUNG JUMP]]》2018年6月14日開始連載，由[[茶菓山心太|茶菓山心太]]作畫的{{日本作品|輝夜姬想讓人告白 同人誌版|かぐや様は告らせたい 同人誌版}}。在《[[週刊YOUNG_JUMP|週刊YOUNG JUMP]]》2018年34號開始連載，由G3井田作畫的{{日本作品|輝夜姬有話要說|かぐや様を語りたい}}的四格漫畫。

至2019年1月止已發行13冊單行本（集英社）。台灣繁體中文版由東立出版社代理發行，譯者為江昱霖。截至2019年3月，本系列作的累計發行量已超過500萬本<ref>{{cite web|url=https://twitter.com/kaguya_comic/status/1103584347225903104|title= ※シリーズ累計500万部！3月19日に最新14巻&同人版2巻&語りたい1巻の3冊同時発売となります！ |publisher=}}</ref>。於2017年8月，在第三屆《{{tsl|ja|次にくるマンガ大賞|下一部漫畫大賞}}》漫畫部門以11,884張的票數獲得第一名<ref>{{Cite web|url=http://natalie.mu/comic/news/245762|script-title=ja:次にくるマンガ大賞、「かぐや様は告らせたい」がコミックス部門1位に|date=2017-08-23|website=漫畫Natalie|accessdate=2017-12-01|language=ja}}</ref>。

2018年6月1日，宣布本作電視動畫化<ref>{{cite web|url=https://twitter.com/akasaka_aka/status/1002358591158870016|title=赤坂アカ@かぐや様アニメ化決定！ on Twitter|publisher=}}</ref>，於2019年1月12日於[[MBS|MBS]]、[[TOKYO_MX|TOKYO MX]]等電視台播出。

2019年2月21日，宣布真人電影化<ref>{{cite web|url=https://natalie.mu/eiga/news/320783|title= 平野紫耀主演で「かぐや様は告らせたい」実写映画化！ヒロインは橋本環奈 |publisher=}}</ref>，由[[平野紫耀|平野紫耀]]與[[橋本環奈|橋本環奈]]所主演，將於同年9月6日日本公開上映。

== 故事簡介 ==
在備受期待的菁英就讀的秀知院學園，在學生會裡相遇的副會長·'''四宮輝夜'''與會長'''白銀御行'''對彼此都有好感，但兩人的自尊心都不允許自己就這樣向對方告白，就在這樣的情況下僵持了半年——

完全不夠坦率的兩人，認為自己只要先告白就輸了，每天想的都是'''「設法讓對方先告白」'''。

就這樣直到今天，輝夜與白銀使盡渾身解數的'''戀愛頭腦戰'''仍在持續著。

== 登場角色 ==
作者称故事是以“《[[竹取物语|竹取物语]]》或许是史實”的IF设定为前提，从9世纪时分裂出来的历史<ref group="注">單行本第4卷。</ref>。作品中的多个人物名字设定被认为源自《竹取物语》，除了女主角辉夜與全部學生會成員外，还有辉夜姬的五个难题佛鉢之石（大佛小鉢）、玉树枝（槙原梢）、火鼠裘（不知火衣）、龙首之玉（龙珠桃）、燕子安贝之石（子安燕子）等。
