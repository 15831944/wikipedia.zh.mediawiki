《'''輝耀姬物語'''》（{{lang-ja|かぐや姫の物語}}）是日本[[吉卜力工作室|吉卜力工作室]]的動畫導演[[高畑勳|高畑勳]]以日本古典文學《[[竹取物語|竹取物語]]》為藍本所改編的動畫電影，2013年11月23日於日本上映。<ref name="Eiga">{{Cite web |url=http://eiga.com/news/20121213/10/ |script-title=ja:ジブリ新作2本！宮崎駿監督「風立ちぬ」と高畑勲監督「かぐや姫の物語」 |publisher=Eiga.com |language=ja |date=2012年12月13日 |accessdate=2012年12月13日 }}</ref>

此為高畑勳继1999年的《[[隔壁的山田君|隔壁的山田君]]》後再次執導的吉卜力作品，<ref name="Eiga" /> 決定將輝耀姬（竹取公主）的故事拍成電影的想法始自2005年，原本預期在30分鐘左右，但經過高畑導演探究輝耀姬身上發生的事，以及思索她心中的想法後，初始劇本發展成一部約3小時的鉅作。原預定與[[宮崎駿|宮崎駿]]的動畫電影《[[風起|風起]]》一同於2013年夏季上映，<ref>{{Cite web |url=http://www.animenewsnetwork.com/news/2011-11-21/ghibli-lists-jobs-for-isao-takahata-summer-2013-film |title=Ghibli Lists Jobs for Isao Takahata's Summer 2013 Film |publisher=Anime News Network |language=en |date=2011年11月21日 |accessdate=2012年12月13日 }}</ref><ref>{{Cite web |url=http://nlab.itmedia.co.jp/nl/articles/1212/13/news096.html |script-title=ja:ジブリ新作は宮崎駿「風立ちぬ」＆高畑勲「かぐや姫の物語」　2013年夏2本同日！ |publisher=itmedia.com |language=ja |date=2012年12月13日 |accessdate=2012年12月15日 }}</ref>
但之後因進度落後，改至該年11月23日上映。<ref>{{Cite web |url=http://kaguyahime-monogatari.jp/information.html |script-title=ja:『かぐや姫の物語』公開延期のお知らせ |publisher=kaguyahime-monogatari.jp |language=ja |date=2013年2月4日 |accessdate=2013年7月12日 }}</ref>

== 劇情 ==
一名居住山林、某日至竹林砍伐竹子的老爺爺，意外注意到某棵竹子的竹莖發散著光芒。之後老爺爺將該竹莖砍斷時，發現裡頭存在著僅巴掌大的小女嬰。

隨後老爺爺將女嬰帶回、與家中的老奶奶一同扶養該女嬰。一段時間後女嬰順利長大成人，也成為一名美麗開懷的少女。但該名少女的真正來歷、卻存在著驚人的實情。

== 製作內容 ==
=== 經緯 ===
在高畑勳推出《輝耀姬物語》前，原先曾向吉卜力工作室的影片製作人[[鈴木敏夫|鈴木敏夫]]提議將日本古代[[平氏|平家]]流傳下的《[[平家物語|平家物語]]》改編成動畫、並邀請吉卜力中的動畫師[[田邊修|田邊修]]擔任主要的動畫處理。但之後田邊修以「不喜歡創作有人與人互相打鬥殺戮場面的動畫」而婉拒，<ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=81頁 |accessdate=2013年12月14日 }}</ref>之後高畑勳便改向在年輕時於[[東映動畫|東映動畫]]工作時、曾打算改編成動畫的日本古典文學《竹取物語》為新作品主題，並在2005年提案、2008年時正式敲定。<ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=70頁 |accessdate=2013年12月14日}}</ref><ref name="K-Message">{{Cite web |url=http://www.kaguyahime-monogatari.jp/message.html |script-title=ja:半世紀を経て |publisher=kaguyahime-monogatari JP |author=[[高畑勳|高畑勳]] |accessdate=2013年12月14日 }}</ref>而先前幾乎包辦吉卜力所有電影影片製作人事務的鈴木敏夫，也特別挑選在2002年加入工作室的宣傳業務職員[[西村義明|西村義明]]擔任此片的製作人。<ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=90頁 |accessdate=2013年12月14日}}</ref>

在《竹取物語》原作中，有提及故事的女主角輝耀姬身份是來自月球世界、因犯了罪過而流放到地球的天人，但未特別描述清楚該事情的來由，而高畑勳對於「輝耀姬的過錯」、「來到地球的原因」有著特別濃厚的興緻、便想以此來作為本作動畫的劇情主題之一。<ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=104頁 |accessdate=2013年12月14日}}</ref><ref name="K-Message"/>而隨著《輝耀姬物語》的製作群人員增加，另有在靠近吉卜力的[[東小金井車站|東小金井車站]]附近，成立了一間臨時的{{nihongo|「第7工作室」|第7スタッフ}}以方便創作。<ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=69頁 |accessdate=2013年12月14日}}</ref>

此作品在製作費用方面累積超過50億日圓。<ref>{{Cite web |url=http://jisin.jp/news/2613/6596/ |script-title=ja:「いくらあっても使う…」鈴木Pが語る宮崎駿の“金遣い” |publisher=光文社 |date=2013年12月6日 |accessdate=2013年12月21日 |title=存档副本 |archive-url=https://web.archive.org/web/20131211215114/http://jisin.jp/news/2613/6596/ |archive-date=2013年12月11日 |dead-url=yes }}</ref>而先前的[[日本電視台|日本電視台]]會長[[氏家齊一郎|氏家齊一郎]]因相當喜愛高畑勳執導的動畫《[[隔壁的山田君|隔壁的山田君]]》而願意出資協助高畑勳拍攝這部新作，並表示「不論要花多少錢我都會設法，另外就把這部片當作送給我死時的伴手禮吧」。<ref name="Page16">{{cite book |script-title=ja:かぐや姫の物語 パンフレット |date=2013年11月23日 |publisher=[[東寶|東寶]] |id={{ASIN|B00GV2D2TC||country=jp}} |page=16頁 |accessdate=2013年12月21日}}</ref>之後，氏家齊一郎在作品正式上映前的2011年離世，後續的資助工作則交由繼任者[[大久保好男|大久保好男]]來完成。<ref name="Page16"/>

因製作方面上的困難，《輝耀姬物語》於企劃後5年在[[分鏡圖|分鏡圖]]方面進度僅有大約30分鐘的長度。<ref name="Page17">{{cite book |script-title=ja:かぐや姫の物語 パンフレット |date=2013年11月23日 |publisher=[[東寶|東寶]] |id={{ASIN|B00GV2D2TC||country=jp}} |page=17頁 |accessdate=2013年12月21日}}</ref>後續在另一吉卜力導演[[宮崎駿|宮崎駿]]發表執導的動畫《[[風起|風起]]》正式上映消息後，讓影片製作人西村義明不得不設法增加負責繪圖的人馬。<ref name="Page17"/>

《輝耀姬物語》在最初完成的劇本時間長達210分以上，之後則刪減至137分鐘，<ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=87頁 |accessdate=2013年12月14日 }}</ref>但仍成為在此之前吉卜力工作室創建以來發表的最長一部動畫。

=== 畫面美術 ===
有別於一般動畫採用將背景與角色分開繪製再合併的方法，此作品高畑勳嘗試將兩者一體化、需較耗時處理的繪製方式。該作業主要由[[田邊修|田邊修]]為中心擔任。<ref>{{Cite web |url=http://kaguyahime-monogatari.jp/img/prono/pronoP5.jpg |script-title=ja:誰もが夢見たアニメーション。もうひとつのスタジオジブリ |publisher=kaguyahime-monogatari JP |accessdate=2013年12月14日 }}</ref><ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=74頁 |accessdate=2013年12月14日}}</ref>

背景方面則由在《[[龍貓|龍貓]]》起便多次在吉卜力動畫中關於山林景觀描繪的[[男鹿和雄|男鹿和雄]]負責。而男鹿和雄為了高畑勳一些場景方面的要求，也曾至[[京都|京都]]一帶取景。<ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=77頁 |accessdate=2013年12月14日}}</ref>

=== 配音 ===
故事的女主角輝耀姬由年輕女演員[[朝倉禮生|朝倉禮生]]擔任，<ref>{{Cite web |url=http://cinema.pia.co.jp/news/161567/53604/ |script-title=ja:ジブリ最新作『かぐや姫の物語』、“かぐや姫”役に朝倉あきが抜てき |trans_title=吉卜力最新作《輝耀姬物語》、“輝耀姬”是由朝倉禮生飾演 |publisher=Pia電影生活|date=2013年9月17日 }}</ref>起初朝倉禮生在參加《輝耀姬物語》的女主角配音面試中一度認為自己沒有中選的資格而沮喪回去，但高畑勳認為朝倉禮生的聲音很能表現出自我中心的味道、符合輝耀姬的形象而屏選。<ref>{{Cite web |url=http://eiga.com/news/20131123/5/ |script-title=ja:「かぐや姫の物語」に抜てきの朝倉あき、起用理由は「ワガママな声」 |publisher={{lang|ja|映画}}.com |date=2013年11月23日 |accessdate=2013年12月14日 }}</ref><ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=18頁 |accessdate=2013年12月14日 }}</ref>男主角捨丸則由[[高良健吾|高良健吾]]負責，高良健吾事後則表示著小時受吉卜力動畫的影響而喜歡飛翔的感覺，能看到劇中自身配音的捨丸與輝耀姬在空中翱翔之場面是他最開心的地方。<ref>{{Cite web |url=http://www.cinemaniera.com/movie/4075 |script-title=ja:高良健吾『かぐや姫の物語』では大好きなジブリ作品で空を飛べてうれしい |publisher=CINEMA MANIERA |date=2013年11月9日 |accessdate=2013年12月14日}}</ref><ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=36頁 |accessdate=2013年12月14日}}</ref>

另動畫中扶養輝耀姬長大的老爺爺與老奶奶配音則分別由[[地井武男|地井武男]]與[[宮本信子|宮本信子]]擔任，而他們先前在1973年由[[森谷司郎|森谷司郎]]執導的電影{{nihongo|《下課後》|放課後}}中、便已共同合作飾演一對夫婦。<ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=39頁 |accessdate=2013年12月14日 }}</ref>之後地井武男於2012年6月29日因心臟病離世，後續老爺爺角色需重新錄音的部份則交由《[[料理東西軍|料理東西軍]]》等電視節目主持人[[三宅裕司|三宅裕司]]頂替，而在《輝耀姬物語》中的角色配音也成為地井武男生涯最後演出。<ref>{{Cite web |url=http://eiga.com/news/20131206/15/ |script-title=ja:三宅裕司、亡き地井武男さんへ恩返し 「かぐや姫の物語」に代役出演していた |publisher={{lang|ja|映画}}.com |date=2013年12月6日 |accessdate=2013年12月14日 }}</ref>

一些《輝耀姬物語》的幕後製作人員也有嘗試挑戰角色配音，如對輝耀姬求婚者之一的石上中納言便是由處理音響效果事務的[[古城環|古城環]]擔任。<ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=94頁 |accessdate=2013年12月14日 }}</ref>而先前曾於高畑勳在1999年推出的動畫《[[隔壁的山田君|隔壁的山田君]]》中擔任配音的演員[[朝丘雪路|朝丘雪路]]，此回也特別友情客串為其中一名角色配音。<ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=66頁 |accessdate=2013年12月14日 }}</ref>

=== 音樂 ===
最初《輝耀姬物語》消息發佈時；劇中的動畫配樂是指定由[[池邊晉一郎|池邊晉一郎]]負責，<ref>{{Cite web |url=http://animeanime.jp/article/2013/02/05/12904.html |script-title=ja:高畑勲監督「かぐや姫の物語」公開延期、2013年秋公開　絵コンテ完成に遅れ |publisher=AnimeAnime |date=2013年2月5日 |accessdate=2013年12月14日 }}</ref>之後則替換成[[久石讓|久石讓]]，這也是久石讓在接手吉卜力動畫配樂的經歷中、首部非由[[宮崎駿|宮崎駿]]執導的作品。<ref name="Prono4">{{Cite web |url=http://kaguyahime-monogatari.jp/img/prono/pronoP4.jpg |script-title=ja:久石 譲と二階堂和美 |publisher=kaguyahime-monogatari JP |accessdate=2013年12月14日 }}</ref>而久石讓早在1984年擔任宮崎駿的作品《[[風之谷|風之谷]]》配樂之後，便已有想參與高畑勳執導動畫配樂的念頭。<ref name="Prono4"/>

高畑勳對於久石讓在配樂方面，有著「不用刻意去反應出劇中角色的心境」、「即使劇中角色陷入低潮時也可以不用哀傷的音樂來帶動觀眾」等請求。<ref>{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=85頁 |accessdate=2013年12月14日 }}</ref><ref name="Yomiuri">{{Cite web |url=http://www.yomiuri.co.jp/entertainment/ghibli/ginterview/20131203-OYT8T00769.htm |script-title=ja:高畑勲×久石譲…映画と音楽、その“到達点”へ |publisher=[[讀賣新聞|讀賣新聞]] |date=2013年11月26日 |accessdate=2013年12月14日 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>另因高畑勳個人對於音樂領域有相當程度的了解，親自為片中創作{{nihongo|《童謠》|わらべ唄}}、{{nihongo|天女之歌|天女の歌}}兩首插入曲，其中《孩童之謠》是以[[五聲調式|五聲調式]]的形式完成。<ref name="Yomiuri"/><ref name="Page84">{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=84頁 |accessdate=2013年12月14日 }}</ref>而為了讓久石讓能容易了解自己創作的曲目內容，高畑勳也採用[[初音未來|初音未來]]音樂軟體來製作曲目樣本以方便對久石讓說明。<ref name="Prono4"/><ref name="Page84"/>而高畑勳也有提出不少關於片中配樂的方向，如片中結尾前關於來自月球天人登場的配樂，因高畑勳認為天人是來自沒有俗間煩惱的世界，便建議久石讓採用[[森巴|森巴]]風格的組曲。<ref name="Yomiuri"/><ref name="Page84"/>

而配樂中一些出現古箏旋律的段落，則聘請了古箏演奏家[[姜小青|姜小青]]負責。<ref>{{Cite web |url=http://jiang-xiaoqing.xii.jp/song/studio.html |script-title=ja:姜小青　スタジオ活動 |publisher=Jiang Xiao-Qing JP |accessdate=2013年12月14日 }}</ref>

=== 片尾曲 ===
此片尾曲由女性歌手[[二階堂和美|二階堂和美]]演唱。因高畑勳聽了她在2011年發表的自創專輯{{nihongo|《暈染》|にじみ}}後相當中意，而希望對方能為此動畫創作片尾曲。<ref name="Page88">{{cite book |script-title=ja:かぐや姫の物語 ビジュアルガイド  |date=2013年11月22日 |publisher=[[角川書店|角川書店]] |isbn=978-404-110568-9 |page=88頁 |accessdate=2013年12月21日 }}</ref>

高畑勳對於片尾曲的主題方向也有提出「現在的一切、過去的一切」等想法，最後二階堂和美完成的成品則為《[[生命的記憶|生命的記憶]]》。<ref name="Page88"/>
