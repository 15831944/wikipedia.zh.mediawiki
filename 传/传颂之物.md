{{Vgname|传颂之物|ja=うたわれるもの|cn=传颂之物|tw=受讚頌者|hk=cn}}是一款由日本[[Leaf|Leaf]]公司制作的[[文字冒险游戏|文字冒险]]与[[战略角色扮演游戏|战略角色扮演]]類型的[[日本成人遊戲|成人遊戲]]，最初于在2002年在日本发售。2003年12月12日發售[[DVD|DVD]]版。和多数文字冒险游戏不同，本游戏虽然有一些选项，但是故事情节是单线的，没有分支，只有單一结局。[[Microsoft_Windows|Microsoft Windows]]版游戏没有角色配音，2006年10月26日由[[AQUAPLUS|AQUAPLUS]]發售有语音的[[PlayStation_2|PlayStation 2]]移植版，並新增了副標題「'''給逝者的搖籃曲'''（{{lang|ja|散りゆく者への子守呗}}）」，而且战斗系统有所调整。

本游戏已经被改編成[[动画|动画]]和[[漫畫|漫畫]]。2011年2月，AQUAPLUS宣佈将在[[PlayStation_3|PlayStation 3]]、[[PlayStation_4|PlayStation 4]]、[[PlayStation_Vita|PlayStation Vita]]平台上推出續作，並於2015年9月24日發售《[[傳頌之物_虛偽的假面|傳頌之物 虛偽的假面]]》，《傳頌之物》全系列为3作<ref>{{cite web|title=下川 直哉 Aquaplus –twitter|url=https://twitter.com/AP_shimokawa/status/652477898792693760|language=ja|publisher=[[Twitter|Twitter]]}}</ref>，而最終章《[[傳頌之物_二人的白皇|傳頌之物 二人的白皇]]》已在2016年9月21日發售。由世嘉制作的《传颂之物 致逝者的摇篮曲》高清重置版已于[[PS4|PS4]]平台上发售<ref name="sega">[https://sega-games.co.jp/release/180129_1.html セガゲームス、他社開発の家庭用ゲームソフト 7タイトルのアジアでの販売ライセンスを新たに獲得 ～自社の販売網・ノウハウを活用し、パートナー企業のアジア展開を支援～]セガゲームス 2018年1月29日</ref>。该作的动作类衍生作品《[[传颂之物斩|传颂之物斩]]》（{{nihongo|うたわれるもの斬|うたわれるものざん}}）预期最早能于2018年9月在PS4平台发售<ref>{{cite web|url=https://www.famitsu.com/news/201806/28160022.html|title=『うたわれるもの斬』オープニングムービー公開|publisher=fami通|language=ja}}</ref><ref>{{cite web|url=http://comic.sina.com.cn/fanyule/2018-06-28/doc-iheqpwqy2121043.shtml|title=PS4游戏《传颂之物斩》OP视频与特典内容公开！|publisher=sina}}</ref>。

== 故事簡介 ==
講述一個失去記憶、戴著一個無法摘下的面具的人以及在他身邊所發生的傳奇故事。有一天渾身是傷的他被一個小村莊的少女艾露露所救，並成為了這個由祖孫三人組成的家庭的一名新成員。雖然完全失去了過去的記憶，哈克奧羅（這個名字是艾露露的祖母圖斯庫爾給他取的，也是艾露露父親的名字）很快憑借他的智慧和勇氣在這個小村莊被大家所接受。之後的一次偶然的事件使哈克奧羅成為了反抗皇帝暴虐統治的起義軍領袖，並在推翻皇帝後成為了新國家的皇帝，然而這僅僅只是哈克奧羅戰鬥生涯的開始。

== 登場人物 ==
=== 主要角色 ===
; {{nihongo|白皇|ハクオロ|Hakuoro|[[聲優|聲優]]：[[小山力也|小山力也]]|另譯：哈克奧羅、哈庫奧羅}}
: 遊戲的主角，戴著一個無法摘除的面具。身受重傷而且失去過去記憶的他被艾露露所救，並且成為她們家的新成員。
: 似乎在這個有點怪異的世界中感到一些困惑。由於想不起來自己的本名，圖斯庫爾給他起名為哈克奧羅(艾露露與阿露露父親的名字)。武器是一把圖斯庫爾給他的鐵扇。為新建的國家「圖斯庫爾」的第一任王。在大部分的戰爭中都能取得勝利。在打敗尼威後，開始回憶起自己的過去。與「庫涅卡門」戰鬥時，為爭取民眾離開王城的時間，親自引開「庫涅卡門」的士兵，更一度覺醒。
: 哈克奧羅在[[阿伊努語|阿伊努語]]中意為「白皇」。真正的名字為「大神 維茨亞奈米迪亞」。本體是大神「[[#世界觀|解放者]]」的半身。

; {{nihongo|艾露露|エルルゥ|Eruruu|聲優：[[柚木涼香|柚木涼香]]}}
: 哈克奧羅醒來後看到的第一個人，亞瑪由拉村村長圖斯庫爾的孫女，作為藥師，負責照料受傷的哈克奧羅。
: 充滿陽光而且非常溫柔的少女，有著散發芬香的長髮、溫柔慈愛的眼神、充滿光澤的皮膚與勤勞之人的美麗雙手，偶爾會變得相當嚴厲，愛吃醋。
: 名字來自一種傳說中的花『艾露露』，與圖斯庫爾的姐姐同名。
: 曾為了復活阿露露與「解放者」訂下契約，是「解放者」的契約者，但契約後記憶便被屏障。
: 同時也是「解放者」與「命」的直系後代。亞瑪由拉村中的女孩在成年後都會被贈予掛在髮梢上的髮環，但只有她的髮環是「命」本人戴過的、水島博士所贈與的、打開地下遺跡的鑰匙。

; {{nihongo|阿露露|アルルゥ|Aruruu|聲優：[[澤城美雪|澤城美雪]]}}
: 艾露露的妹妹。非常害羞且怕生的女孩，只能做最低限度的交談，但在熟人面前會變成愛撒嬌的孩子，非常可愛。稱呼哈克奧羅為「爸爸」。
: 名字來自一種傳說中的花『阿露露』。據說艾露露和阿露露著兩種花總長在一起，被稱為姐妹花。
: 擁有遺傳自「命」的「森林之母」能力，能與動物交心，在森林之主被殺死後收養了主的孩子，並取名為「穆克魯」。
: 曾經在大地震中死亡，並靠「解放者契約」的力量復活。跟艾露露一樣是「解放者」的直系後代。

=== 亞瑪由拉村 ===
; {{nihongo|圖斯庫爾|トゥスクル|Tusukuru|声优：[[京田尚子|京田尚子]]|另譯：托斯克魯}}
: 艾露露與阿露露的祖母，亞瑪由拉村名義上的村長，非常慈祥，德高望重，被大家所敬愛的長者，也是有名的藥師，凡是曾立志學習醫術的人，沒有不知道圖斯庫爾的。
: 在與地方藩主手下的爭執中，為了救阿露露而被士兵所傷後，因年紀過大、身體虛弱不堪負荷而亡。
: 圖斯庫爾的死亡引起了眾人的憤怒，在一連串的起義推翻王國後，哈克奥羅以她的名字為新建立的國家命名作為紀念。

; {{nihongo|鐵奧羅|テオロ|Teoro|聲優：[[石川洋昭|石川洋昭]]}}
: 性格豪爽的男子，村子裡的領導性人物，稱呼哈克奧羅為「小哥」，像兄弟一般相處，實際上比外貌要年輕很多。
: 手持斧頭直到坎納貝西科烏魯貝皇都壓制戰為止都與哈克奧羅共同戰鬥，在戰爭結束後回到亞瑪尤拉生活。
: 在亞瑪由拉村被庫查·凱查軍隊襲擊時中箭，但為了向哈克奧羅通報而負傷奔至皇都，通報後在皇都中安祥地死去。

; {{nihongo|蘇牧|ソポク|Sopoku|声优：[[雪野五月|雪野五月]]}}
: 鐵奧羅的妻子，相當具有膽識，沉穩的女性，將艾露露、阿露露當成自己的姐妹一樣疼愛。
: 據鐵奧羅表示小時候相當可愛，結婚後卻變得凶暴，典型邊境的女人。
: 與亞瑪由拉村民們死於庫查·凱查的侵略戰爭中。

=== 坎納西科烏魯貝 ===
; {{nihongo|朧|オボロ|Oboro|聲優：[[桐井大介|桐井大介]]}}
: 邊境城寨的年輕首領，血氣方剛的青年，非常敬仰圖斯庫爾。得意於自身動作迅速的二刀流，單方面地把貝納威視為宿敵。
: 曾被哈克奧羅二度救過他的性命，敬稱哈克奧羅為「大哥」。在哈克奧羅稱皇後成為哈克奧羅的左右手之一，並任圖斯庫爾步兵隊隊長。
: 相當關心體弱多病的妹妹柚葉，常因此做出許多非常滑稽的事。
: 在遊戲初期設定中為前坎納西科烏魯貝皇族後裔，因權力爭鬥失敗而逃亡至邊境。

; {{nihongo|柚葉|ユズハ|Yuzuha|声优：[[中原麻衣|中原麻衣]]}}
: 朧同父異母的妹妹，天生因體質問題導致失明的少女，隨時都可能死去，受到朧的過度保護，過著與世隔絕般的日子。
: 原本連「朋友」是什麼都不知道，在與哈克奧羅等人結識後開始積極地生活，並與阿露露、神結等人成為朋友。
: 在圖斯庫爾皇都中受到眾人的愛護並快樂的生活著，原本認為就算即刻死去也沒有什麼遺憾，但在知道眾人對自己的感情與關心後發覺自己不能輕言放棄，必須回報大家的感情。
: 在人生獲得滿足之前，把努力活下去當成自己的義務，並對朧表示想要留下與哈克奧羅的孩子，作為努力活過的證明。
: 原本有利用「蜜琪姆」的肝來控制、治癒病情的方法，但因不願靠殺死其他生物來治病而拒絕。

; {{nihongo|多利、古拉|ドリィ、グラァ|Dorī、Gurā|声优：[[渡邊明乃|渡邊明乃]]}}
: 朧部下的一對雙子少年，擅長箭術與偵查，兩人分別擔任圖斯庫爾弓箭隊青組與紅組的組長。
: 稱呼朧為「少主」，並對朧忠心耿耿。

; {{nihongo|殷卡拉|インカラ|Inkara|声优：[[大川透|大川透]]|另譯：印卡拉}}
: 坎納西科烏魯貝的皇帝，從外表無法想像的狡猾與殘虐性格。
: 非常在意自己的頭髮，將大筆國庫花費在保養自己的頭髮上、揮金如土的昏君。
: 在皇都淪陷後企圖逃跑，被貝納威斬殺。

; {{nihongo|薩桑迪|ササンテ|Sasante|声优：[[大川透|大川透]]|另譯：沙桑提}}
: 殷卡拉的弟弟，邊境區域的藩主，努萬基的父親，但一直到繼承人在瘟疫中死掉後才把努萬基接回身邊。
: 生性狡猾殘忍，對人民施以重稅而招致叛亂。
: 逃亡時因拋棄努萬基而死於努萬基之手。（動畫中死於哈克奧羅）

; {{nihongo|努萬基|ヌワンギ|Nuwangi|声优：[[吉野裕行|吉野裕行]]|另譯：努旺基、努汪基}}
: 殷卡拉王的外甥，薩桑迪的兒子，因母親早逝，由圖斯庫爾扶養長大。
: 從小與艾露露生活在一起，如同兄妹一般，但在成年前被薩桑迪帶走後便變得追名逐利，為了得到艾露露什麼事都做得出來。
: 在和哈克奥羅的决战中敗北後被艾露露放了一條生路而有所覺悟，想把戰爭結束掉卻在路途中被戰爭流民襲擊而生死不明。

; {{nihongo|貝納威|ベナウィ|Benawi|声优：[[浪川大輔|浪川大輔]]|另譯：貝納維}}
: 坎納西科烏魯貝的侍大將，精通槍術，文武雙全，統領騎兵隊，非常冷靜的男人。對國家有非常強烈的責任感。
: 同時也是坎納西科烏魯貝皇族外戚，游戏中第一次遇见胧时，听到胧的名字就知道了他的身分，因为不忍心下手杀死本来自己应该侍奉的皇族血亲，所以才放走胧。
: 在坎納西科烏魯貝滅國後欲自盡，卻被哈克奧羅阻止，最後留在圖斯庫爾輔佐白皇處理國政，是不可或缺的助手。

; {{nihongo|克羅|クロウ|Kurou|声优：[[小山剛志|小山剛志]]}}
: 坎納西科烏魯貝的騎兵隊副長，貝納威的忠誠部下，騎兵隊副隊長。
: 渾身引以為豪的肌肉，性格粗暴但卻很可靠的男人，武器是刀。
: 在坎納西科烏魯貝滅國後和貝納威一起投入哈克奧羅地麾下，很喜歡開朧的玩笑，兩人的爭吵總是給大家帶來樂趣。

; {{nihongo|卡姆查塔爾|カムチャタール|Kamuchatāru|声优：[[田口宏子|田口宏子]]}}
: 歡樂街某商店的女主人，實為殷卡拉之女，克羅小時候的青梅竹馬，在貝納威替殷卡拉清理門戶時差點被殺，為克羅所救，對克羅有好感。

; {{nihongo|切奇納羅|チキナロ|Chikinaro|声优：[[太田哲治|太田哲治]]}}
: 旅行商人，除了人口販賣以外什麼生意都做，無論是藥材、武器、情報還是各種稀有物品，給了哈克奧羅很大的幫助。

=== 歐卡米亞凱 ===
; {{nihongo|沃貝|ワーベ|Wābe|声优：[[大木民夫|大木民夫]]}}

: 森林之神的使者、森林之主，如同老虎一般的巨大白色野獸，身體刀槍不入，不是常人所能抗衡的。

; {{nihongo|騎龍|ウォプタル|Uoputaru||另譯：沃布塔爾}}
: 大陸常用的交通工具，雙足步行，全身覆滿鱗片，用於馬車拉引與騎乘的生物。

; {{nihongo|蜜琪姆|ミキューム|Mikyūmu}}
: 手掌般大小的白色動物，只在與世隔絕的溪谷裡居住的神聖靈獸，非常罕見，價值之高足以令人逍遙好幾年。
: 傳說只要對它好便能帶來好運，還能擋災避厄，相當不可思議的生物。
: 根據圖斯庫爾所言，其肝是能夠治癒百病的藥。
: 平時很乖巧，但感覺到危險時會發出特殊的叫聲來自衛。
: 一旦被起了名字，就會認定起名者為主人。
