在[[逻辑学|逻辑学]]和[[数学|数学]]中，'''傳遞關係'''（{{lang-en|Transitive relation}}）、即，若对所有的''a''，''b''，''c''属于''X''，下述語句保持有效，則集合''X''上的[[二元关系|二元关系]]''R''是'''传递的'''：「若''a''关系到''b''且''b''关系到''c''，则 ''a''关系到''c''。」

== 特性 ==
数学上表示为：

:<math>\forall a, b, c  \in X,\ a R b \land b R c \; \Rightarrow a R c</math>

例如：'''大于等于'''具有传递关系：若<math> a \geqslant b </math>且<math> b \geqslant c </math>则<math> a \geqslant c </math>。

传递关系举例：
* [[等于|等于]]
* 是……的[[子集|子集]]
* [[小于等于|小于等于]]
* [[除法|除法]]

满足[[自反性|自反性]]的传递关系称为[[预序关系|预序关系]]。满足[[反对称性|反对称性]]的预序关系称为[[偏序关系|偏序关系]]。满足[[对称性|对称性]]的预序关系称为[[等价关系|等价关系]]。
