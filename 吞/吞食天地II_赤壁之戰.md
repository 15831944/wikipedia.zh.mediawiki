{{Vgname|吞食天地II 赤壁之戰|ja=天地を喰らう2 赤壁の戦い|en=Warriors of Fate}}，是1992年由[[卡普空|卡普空]]推出的[[街機|街機]][[電子遊戲|遊戲]]，和前作《[[吞食天地_(街機)|吞食天地]]》一樣，本作亦以[[本宫宏志|本宫宏志]]的《[[三國演義|三國演義]]》漫畫《{{link-ja|天地吞食|天地を喰らう}}》（{{lang|ja|天地を喰らう}}）為遊戲腳本，由玩家操作[[蜀國|蜀國]]的武將們進行戰鬥，遊戲流程仍為橫向捲軸式的關卡設計。本作品後來於1996年時移殖到[[PlayStation|PlayStation]]與[[Sega_Saturn|Sega Saturn]]，2005年時又移殖為[[手機|手機]]遊戲，在日本NTT手機移殖版中有一些新增的遊戲要素。2011年底移植到[[PSP|PSP]]遊戲的下載商店中。

本作的故事由[[博望坡之戰|博望坡之戰]]開始到[[赤壁之戰|赤壁之戰]]為止，并再现了原著的“华容道捉放曹”情节。玩家可以選擇的角色較前作多一個，可以3人同時遊戏（後來的移植版只能2人同時遊戏）。

== 玩家可選角色 ==

* [[關羽|關羽]]
: 全能型角色，非常強而有力，攻击性能强。步行時沒有拿[[兵器|兵器]]，以徒手戰鬥，會[[铁头功|铁头功]]及跳跃折腰摔（抓住敵人後，B、A折断敌人脊椎的大坐），抓人距离廣但稍逊于张飞，絕技為斷鎧擊。騎馬時則執[[青龍偃月刀|青龍偃月刀]]。

* [[張飛|張飛]]
: 力量型角色，技巧性最强。步行時沒有拿兵器，以徒手戰鬥，有數種類似[[職業摔角|職業摔角]]的動作，會咬人及旋風大摔，抓人距离全員最廣，絕技為粉碎拳。騎馬時執[[丈八蛇矛|丈八蛇矛]]。

* [[趙雲|趙雲]]
: 速度型角色，最具有技巧性。步行時執[[劍|劍]]，動作最敏捷，但力量在角色中最低，為唯一沒有地面摔技（即抓住敵人後，B+A的摔技）的角色（會變成「跳踢」），但能空摔，絕技為（滑步）飛翔劍。騎馬時執[[槍|涯角槍]]。

* [[黃忠|黃忠]]
: 远程攻击型角色。步行和騎馬時皆執[[弓|弓]][[箭|箭]]，跳跃攻击时会使用[[匕首|匕首]]（[[繩標|繩標]]），為唯一能遠距離攻擊的角色，但不適合近戰，抓人距离全員最窄，絕技為斬斷弓和连发箭。

* [[魏延|魏延]]
: 组合技特长型角色。步行和騎馬時皆執[[苗刀|苗刀]]，抓人距离和赵云差不多，絕技為斬馬蹴。

== 電腦控制角色 ==

* [[李典|李典]]
: 第1關頭目。體力較低。執[[槍|红纓枪]]，能飛踢。与李典的战斗中重现了原著中故意败退后纵火伏击的情节。

* [[夏侯惇|夏侯惇]]
: 第2關頭目。騎馬時執[[偃月刀|偃月刀]]，步行時則以徒手戰鬥，還會噴火、抓人與飛天重壓。兩種狀態都能以[[炸彈|炸彈]]攻擊。以騎馬姿態出現。有各種各樣的攻擊型態。在當時Capcom作品中慣例會出現的強敵。

* [[許褚|許褚]]
: 第3關頭目。體力非常高的角色，執大[[鐵|鐵]][[槌|槌]]。會車輪衝撞、抓人及飛天重壓。关卡重现了“火烧[[新野|新野]]”的情节。

* 美美、美冴、美鈴（吞食天地虛構角色）
: 第3關與第4關之間出現的角色。善於扔飛刀和輕快的[[足技|足技]]。最後打倒美美會得到更多Round Clear Bonus。 第8關之後以不同顏色和名稱的雜兵身份出現。如果能一命破關的情況下，會得到三人讚美。這角色是由[[安田朗|安田朗]]進行設計的。

* [[曹仁|曹仁]]
: 第4關頭目。執[[斬蛇劍|斬蛇劍]]及大型[[流星錘|流星錘]]，攻擊速度快，會格擋。

* [[淳于導|淳于導]]（三國演義虛構角色）
: 第5關頭目。步行和騎馬時皆執[[鏟_(武器)|月牙鏟]]。 攻擊距離很長，但動作較慢。以步行姿態出現。
: （附註：这段剧情是原著“赵子龙单骑救主”的重现。其中，[[糜夫人|刘备夫人]]投井托孤的场景，也得以再现。）

* [[晏明|晏明]]（三國演義虛構角色）
: 第6關頭目。體形和許褚相似，執[[長柄刀|鳳嘴刀]]，攻擊能發出衝擊波，還會車輪衝撞。防禦力高但移動速度比較慢。
: （附註：該關卡中段會出現[[夏侯恩|夏侯恩]]，见到玩家就会逃跑，将其截杀可得[[劍|青釭劍]]，这是原著中赵云的事迹。）

* [[夏侯傑|夏侯傑]]（三國演義虛構角色）
: 第6關副頭目，與晏明一起出現。執[[槍|黑纓枪]]。體形和李典相似，但不會用足技。
: （附註：这段剧情是原著“张翼德大闹长坂桥”的重现。）

* [[張遼|張遼]]
: 第7關頭目。執[[狼牙棒|狼牙棒]]及[[盾|尖刺盾牌]]，會飛天重壓、飛刀、抓人及放保險。移動速度較快。
: （附註：該關卡開始前會出現[[赤壁之戰|火烧连环船]]的画面，此外可見到张辽和曹操的背影。）

* [[徐晃|徐晃]]
: 第8關頭目。步行和騎馬時皆執[[斧|開山斧]]。體形和淳于導相似，但他更強很多，會抓人。以騎馬姿態出現。因為這關卡舞台狹窄，沒有地方可以逃脫，所以這是最困難的一關。
: （附註：該關卡中段會出現黄、紫、王、偃、華、何的女敵方）

* [[呂布|呂布]]
: 第9關頭目，是实际的最终BOSS。執[[斬馬劍|斬馬劍]]，還能使[[鞭|毒龍鞭]]。會車輪衝撞、格擋、抓人及放保險。幾乎擁有所有強項的強者。由於體力亦相當高，有超時未能夠過關之風險。


* [[曹操|曹操]]
: 最後一關的頭目，以[[炸彈|炸彈]]攻擊。战斗时间仅15秒，15秒后曹操会逃走。开战前可以选择“释放”，让曹操逃跑（但是世界版、美版游戏不能释放）。結局會根據玩家是否打敗或忽視他而改變，打倒、逃走和释放的结局画面不同，打倒为统一全国，释放为三国鼎立，若曹操逃走则显示五主角最终战死，曹操一统天下。
: （附註1：这段剧情是原著“关云长义释曹孟德”的重现。但多出两種選擇。）

== 遊戲其他角色 ==

* [[劉備|劉備]]
: 前作為可選的一位角色，本作為只在剧情演示画面裡登场，变成了[[非玩家角色|NPC]]。其图案的身体部分和曹操一样，只是颜色不同。

* [[諸葛亮|諸葛亮]]
: 关卡之间的剧情演示画面裡经常登场，台词较多。敌将战裡制限时间所剩无几时，警告信息也是诸葛亮发出的。

* [[劉禪|劉禪]]
: 第5关打败淳于导后，在剧情演示画面登场。

* [[孫權|孫權]]
: 第6关通过后登场，是否要和刘备联盟拒曹，他举棋不定，打木人或大比武Bonus关卡结束后，孙权下定决心要和刘备结为同盟。结局裡，孙权表示很期待街机吞食天地3发售，但至今没有发售。

* [[魯肅|魯肅]]
: 第6关通过后，鲁肃与孙权一同登场，向孙权献策，建议和刘备结为同盟。其图案的身体部分和諸葛亮一样，只是颜色和衣服不同。

== 世界版 ==

原版的“吞食天地2”仅在日本本土发行，华语圈进口的机台多是日本原版。考虑到西方世界用户对三国故事背景不熟悉，发行商决定以“Warriors of Fate”（命运战士）之名在欧美发行这款游戏，内容和关卡与原版基本无异，只是我方和敌方角色的名字，被更换成了西方玩家熟悉的历史角色，比如五虎将以“[[亚瑟王|亚瑟王]]和他的勇士”命名，敌将被冠以“成吉思汗”、“匈奴王[[阿提拉|阿提拉]]”之类，整个故事情节被描绘成“历史英雄人物超越时空的大会战”，颇有“[[关公战秦琼|关公战秦琼]]”的意味。这种设定的反响并不讨好，使西方的玩家多感到不知所云，按日本的说法就是充满“违和感”。

同样的游戏，在日文和华语圈广受欢迎，但在西方的知名度和反响就非常一般，很可能是海外市场销量最低的CPS作品<ref>[http://bloodlust.zophar.net/Callus/callus.html Official Bloodlust Software Callus Page]</ref>，远没有情节“西化”的《[[快打旋风|快打旋风]]》、《[[上尉密令|名将]]》成功。
