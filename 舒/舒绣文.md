'''舒绣文'''（{{bd|1915年||1969年|3月17日|catIdx=Shu}}），[[中国电影|中国电影]]与[[话剧|话剧]]演员，中国电影史上第一个女[[配音演员|配音演员]]，被誉为中国影剧界“四大名旦”之一。<ref name=china>{{cite web |url=http://guoqing.china.com.cn/2008-10/08/content_23679974.htm |title=第一个女配音演员：舒绣文 |publisher=中国网 |date=2008-10-08}}</ref>

==生平==
[[File:Shu_Xiuwen_on_Screen_Pictorial.jpg|thumb]]

舒绣文于1915年生于[[安徽省|安徽省]][[安庆|安庆]]，六岁时移居[[北京|北京]]。14岁时因家境贫困被迫当舞女。1931年到[[上海|上海]]谋生，在[[天一影片公司|天一影片公司]]教国语，随后加入集美歌舞剧社当话剧演员。1930年代初加入[[田汉|田汉]]的[[五月花剧社|五月花剧社]]和[[春秋剧社|春秋剧社]]。<ref name=china/>

[[抗日战争|抗日战争]]时期，舒绣文在重庆后方演出大量爱国话剧，被誉为“四大名旦”之一。抗战胜利后，演出电影《[[一江春水向东流_(1947年电影)|一江春水向东流]]》中的王丽珍形象，成为她的代表作。<ref name=china/>

1957年舒绣文加入[[北京人民艺术剧院|北京人民艺术剧院]]，扮演了话剧《[[骆驼祥子|骆驼祥子]]》中的虎妞和《[[关汉卿_(话剧)|关汉卿]]》中的朱帘秀等形象。被评为国家一级演员。<ref name=china/><ref name=bjry>{{cite web |url=http://www.bjry.com/actor/html/2015/06/2015060569.html |title=舒绣文 |publisher=北京人民艺术剧院 |accessdate=2017-11-24}}</ref>

舒绣文曾当选为[[第一届全国政协|第一届全国政协]]委员，第二、三届[[全国人大代表|全国人大代表]]，第三届[[全国妇联|全国妇联]]执行委员会委员，和[[全国文联|全国文联]]委员等职务。舒绣文在[[文化大革命|文化大革命]]中遭到批斗，于1969年3月17日去世。<ref name=china/><ref name=bjry/>
