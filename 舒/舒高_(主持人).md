'''舒高'''{{BD|1976年|11月19日|||catIdx=S}}出生于[[湖南省|湖南省]]，[[回族|回族]]，毕业于[[中南大学|中南大学]]，为[[湖南卫视|湖南卫视]]女主持人。

== 生平 ==
1976年，舒高出生在湖南省，后来大学在中南大学度过。大学毕业后，长期在湖南卫视担任主持人。

== 个人生活 ==
舒高的首任丈夫是[[丁辉|丁辉]]，两人2003年离婚，丁辉后来和湖南卫视主持人“[[赵靓|赵靓]]”结婚，后再度离婚。<ref>{{cite web |url=http://ent.qq.com/a/20121122/000038.htm |title=舒高与相恋13年男友结婚 网友：她离婚才9年啊 |accessdate=2012-11-22 |publisher= 腾讯网|archiveurl=http://ent.qq.com/a/20121122/000038.htm |archivedate= 2012年11月22日|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://et.21cn.com/gundong/etscroll/2012/11/22/13748507_1.shtml |title=前湖南卫视花旦舒高二婚 独家披露情史内幕 |accessdate=2012-11-22 |publisher=21CN新闻网 |archiveurl=https://web.archive.org/web/20160305015817/http://et.21cn.com/gundong/etscroll/2012/11/22/13748507_1.shtml |archivedate=2016年3月5日 |work= |language=zh-cn |deadurl=yes }}</ref>
