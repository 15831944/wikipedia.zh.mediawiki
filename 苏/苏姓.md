'''蘇姓'''是一個[[中文姓氏|中文姓氏]]，在《[[百家姓|百家姓]]》中排名第42位，在現今中國姓氏排行[[百家姓#當代百家姓|第45位]]。

== 起源 ==
主要源流有二：
# 出自[[黃帝|黃帝]]。蘇氏為[[黃帝|黃帝]]之孫[[顓頊|高陽]]後裔；據《蘇洵族譜》所記載：“蘇氏之先，出於高陽。高陽之子為稱，稱之子曰[[老童|老童]]。老童之子生[[重黎|重黎]]及[[吳回|吳回]]。吳回生[[陸終|陸終]]。陸終生子六人，長曰[[礬|礬]]，為[[昆吾|昆吾]]。昆吾始姓[[己姓|己姓]]，其後為[[蘇|蘇]]、[[顧|顧]]、[[溫|溫]]、[[董|董]]。”後蘇氏繁衍於[[河南|河南]]、[[河內郡_(中國)|河內]]一帶。據《[[元和姓纂|元和姓纂]]》所載，周武王時，司寇忿生，受封於[[蘇國|蘇國]]，後遷於[[溫|溫]]，稱為[[蘇忿生|蘇忿生]]，春秋時，蘇國被狄族所滅，其子孫以國名為氏。
# 出自[[少數民族|少數民族]]之改姓。[[漢代|漢代]][[遼東|遼東]][[烏桓|烏桓]]有蘇姓。又據《魏書官氏志》記載，[[北魏|北魏]]有複姓[[拔略氏|拔略氏]]隨[[魏|魏]]孝文帝南遷[[洛陽|洛陽]]後，定居[[中原|中原]]，改為漢姓「蘇」氏。回族的蘇姓來自阿拉伯名字[[蘇萊曼|蘇萊曼]]。

== 歷代遷徙 ==
*[[先秦|先秦]]時期，蘇姓主要活動地區在[[河南|河南]]、[[河北|河北]]；
*[[戰國時期|戰國時期]]其中一支南遷[[湖南|湖南]]、[[湖北|湖北]]，一支西遷[[陝西|陝西]]；
*[[秦|秦]][[漢|漢]]時，蘇姓又東遷到[[山東|山東]]，而定居於[[陝西|陝西]]西部的蘇姓已發展成為望族；
*[[晉朝|晉朝]]時在東南的[[江蘇|江蘇]]、[[浙江|浙江]]、[[安徽|安徽]]等地，南部的[[廣東|廣東]]，已有了蘇姓的足跡；
*[[唐朝|唐朝]]時，蘇姓移民大量南下[[四川|四川]]和[[福建|福建]]，移民中的蘇姓在[[福建|福建]]得到了穩定的發展；
*[[北宋|北宋]]時蘇姓移民進一步大批西進[[川|川]]、[[滇|滇]]，南下[[兩廣|兩廣]]，並越過邊疆進入[[越南|越南]]、[[老撾|老撾]]、[[泰國|泰國]]；
*[[明|明]][[清|清]]兩朝，蘇姓多次移民[[臺灣|臺灣]]，如今蘇姓已成為[[臺灣|臺灣]]的大姓；
*來自外族的蘇姓；[[魏晉南北朝|魏晉南北朝]]以後，北方民族大批進入中原，有很多外族改姓蘇，主要有：
**[[漢|漢]][[晉|晉]]時[[遼|遼]]東[[烏桓|烏桓]]部的蘇姓人；
**[[南北朝|南北朝]][[北魏|北魏]]時[[鮮卑族|鮮卑族]][[拔略氏|拔略氏]]族；
**[[北宋|北宋]]時[[西夏|西夏]][[黨項族|黨項族]]的蘇姓人；
**[[金國|金國]][[女真族|女真族]]的蘇姓人；
**[[元|元]]末[[蒙古|蒙古]][[義王和尚|義王和尚]]攜家眷至[[山東|山東]]，改為蘇姓。
**[[清朝|清朝]]時，[[滿洲|滿洲]][[八旗|八旗]]的[[伊拉哩氏|伊拉哩氏]]、[[蘇佳氏|蘇佳氏]]、[[蘇都哩氏|蘇都哩氏]]、[[蘇爾佳氏|蘇爾佳氏]]等族人改為蘇姓。
