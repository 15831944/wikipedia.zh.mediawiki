'''苏黎世圣经译本'''（{{Lang|de|Zürcher Bibel}}，也被称为慈運理聖經译本，{{Lang|de|Zwinglibibel}}）是历史上的一个基于[[慈运理|慈运理]]翻譯的[[聖經譯本|聖經譯本]]。最近印行的版本都以強化語言精確性爲目的。

== 弗羅紹爾聖經 ==
慈運理的翻譯出自預言學校（{{Translink|en|Carolinum, Zürich|4=Prophezey}}<span>），這是一個除週五和週日外每天工作的解經研討團隊，參加者爲[[蘇黎世|蘇黎世]]的所有神職人員， 他們致力於翻譯出一本德語聖經，以造福會衆。</span><ref>{{Cite news|url=http://www.swissinfo.ch/fre/la-bible-zurichoise_-le-prix-n-est-pas-%C3%A9lev%C3%A9--mais-le-b%C3%A9n%C3%A9fice-est-grand-/43137280|title=LA BIBLE ZURICHOISE-«Le prix n’est pas élevé, mais le bénéfice est grand»|author=Regula Bochsler|date=|accessdate=2017-05-21}}</ref>

他們參考了[[馬丁·路德|馬丁·路德]]已翻譯好的部分聖經，這使得慈運理得以早路德五年完成全本聖經的翻譯。 [[克里斯托夫·弗罗绍尔|克里斯托夫·弗羅紹爾]]的印刷作坊從1525年到1529年印製了[[新約聖經|新約聖經]]，後來增加了部分[[舊約聖經|舊約聖經]]，並於1531年印刷了首部完整翻譯的新舊約全書，慈運理親自書寫了介紹與每章後面的結語。這本《弗羅紹爾聖經》包含200多幅插圖，成爲當時著名的印刷業傑作。它的翻譯主要要歸功於慈運理和他的好友，聖彼得教區牧師[[里歐·尤德|里歐·尤德]]（{{Lang|de|Leo Jud}}）。

舊約的翻譯於1540年作了修訂，新約則在1574年重新修訂。1589年增加了對每節經文的編號。

1975年，Amos Hoover of Dover, PA重印了1536年的版本，1531年的版本則於1983年由Theologischer Verlag Zürich重印。<ref>[http://gameo.org/index.php?title=Froschauer_Bibles_and_Testaments Froschauer Bibles and Testaments].</ref>

== 修訂 ==
直到1665年，這個譯本所使用的語言基於用於官方文件的[[瑞士德语|瑞士德語]]。1665年放棄了這種語言，轉而使用在[[薩克森-維滕堡公國|薩克森-維滕堡公國]]興起的[[標準德語|標準德語]]。

[[苏黎世圣母大教堂|苏黎世圣母大教堂]]（{{Lang|de|Fraumünster}}）牧師約翰·卡斯珀烏爾里希（{{Lang|de|Johann Caspar Ulrich}}，1705年–1768年）在1755/1756年修訂版中添加了註釋、說明與索引。從這一版開始，這本聖經以慈运理圣经爲人所知。自1817年起，這個譯本爲蘇黎世聖經與宣教團（{{Lang|de|Zürcher Bibel- und Missionsgesellschaft}}）所有。另一次修訂則是在1868年，於1892年重印。

== 1931年修訂版 ==
1907年，以儘量結合最新的聖經研究做重新修訂爲目的，成立了一個委員會。這個修訂版於1931年完成，基本上是一次新的翻譯。

== 2007年修訂版 ==
1984年，在慈運理誕生500週年之際，蘇黎世州[[新教|新教]][[歸正宗|歸正宗]]教會全體會議發起另一次修訂，然而不久後即決定改爲對整本聖經重新翻譯。該項目花費了四百萬瑞郎，於2007年初完工。2007年6月，這本聖經分別以紙版和電子版形式出版發行。
