'''蘇榮'''{{BD|1948年|10月29日|catIdx=Su,苏荣}}，[[吉林省|吉林]][[洮南市|洮南]]人。[[中华人民共和国|中华人民共和国]]政治人物。2014年落马时任[[中国人民政治协商会议第十二届全国委员会|中国人民政治协商会议第十二届全国委员会]][[中国人民政治协商会议全国委员会副主席|副主席]]，位列[[国家级副职|国家级副职]][[党和国家领导人|党和国家领导人]]。历任中共[[青海省|青海]]、[[甘肃省|甘肃]]和[[江西省|江西]][[省委书记|省委书记]]，是[[中国共产党|中国共产党]]第十四、十五屆[[中国共产党中央委员会候补委员|中央候補委員]]，第十六、十七屆[[中国共产党中央委员会|中央委員]]<ref>{{cite web|url = http://news.xinhuanet.com/2013lh/2013-03/11/c_114985743.htm|title = 中国人民政治协商会议第十二届全国委员会副主席苏荣简历|publisher = 新华网|date = 2013-03-11|deadurl = yes|archiveurl = https://web.archive.org/web/20130927211723/http://news.xinhuanet.com/2013lh/2013-03/11/c_114985743.htm|archivedate = 2013-09-27}}</ref>。

2014年6月14日，[[中国共产党中央纪律检查委员会|中共中央纪委]]发布消息，[[全国政协副主席|全国政协副主席]]苏荣因涉嫌严重违纪违法，正接受组织调查<ref name="双规">{{cite web|url = http://www.ccdi.gov.cn/yw/201406/t20140614_36928.html|title = 中国人民政治协商会议第十二届全国委员会副主席苏荣接受组织调查|accessdate = 2014年6月14日|publisher = 中央纪委监察部网站}}</ref>。6月25日，第十二届[[全国政协常委会|全国政协常委会]]第六次会议通过了关于免去苏荣[[全国政协副主席|全国政协副主席]]职务、撤销其[[中国人民政治协商会议全国委员会|全国政协委员]]资格的决定<ref name="被免">{{cite web|url = http://politics.people.com.cn/GB/n/2014/0625/c1001-25199604.html|title = 苏荣被免去全国政协副主席职务 涉严重违纪违法被查|accessdate = 2014年6月25日|publisher = 人民网}}</ref>。因涉嫌严重违纪违法，2014年9月25日，江西省第十二届人大常委会第十三次会议决定罢免其[[第十二届全国人民代表大会代表名单|第十二届全国人民代表大会代表]]职务<ref>{{cite news|title=第十二届全国人民代表大会常务委员会代表资格审查委员会关于个别代表的代表资格的报告（2014年11月1日第十二届全国人民代表大会常务委员会第十一次会议通过）|url=http://www.npc.gov.cn/wxzl/gongbao/2014-12/23/content_1892465.htm|accessdate=2016-07-03|newspaper=中国人大网}}</ref>。11月1日，[[第十二届全国人民代表大会常务委员会|第十二届全国人大常委会]]第十一次会议表决通过关于苏荣的代表资格的报告，并按照《[[中华人民共和国全国人民代表大会和地方各级人民代表大会代表法|代表法]]》的有关规定，终止其[[第十二届全国人民代表大会代表名单|第十二届全国人大代表]]的资格<ref>{{cite web|url = http://www.ccdi.gov.cn/yw/201411/t20141101_38457.html|title = 苏荣的全国人大代表资格被依法终止|publisher = 中央纪委监察部网站|date = 2014-11-01}}</ref>。

2015年2月16日，[[中共中央纪委|中共中央纪委]]宣布，经中央纪委常委会议研究并报[[中共中央政治局|中共中央政治局]]会议审议，决定给予苏荣开除党籍、开除公职处分<ref>{{cite web|url = http://www.ccdi.gov.cn/xwtt/201502/t20150216_51531.html|title = 全国政协原副主席苏荣严重违纪违法被开除党籍和公职|accessdate = 2015年2月16日|publisher = 中央纪委监察部网站}}</ref>，将其涉嫌犯罪问题、线索及所涉款物移送司法机关处理。次日，[[中华人民共和国最高人民检察院|中华人民共和国最高人民检察院]]发布消息称，决定对苏荣以涉嫌受贿罪立案侦查并采取强制措施<ref name="立案侦查">{{cite web|url = http://www.spp.gov.cn/qwfb/201502/t20150217_91458.shtml|title = 最高人民检察院依法决定对苏荣立案侦查|accessdate = 2015年2月17日|publisher = 最高人民检察院网站}}</ref>。2016年7月14日，经最高人民检察院批准，苏荣受贿、滥用职权、巨额财产来源不明案，由最高检指定山东省人民检察院侦查终结后，移送济南市人民检察院审查起诉。济南市人民检察院已向[[济南市中级人民法院|济南市中级人民法院]]提起公诉<ref name="起诉">{{cite web|url = http://news.ifeng.com/a/20160714/49349645_0.shtml|title = 苏荣、杜善学等四案被提起公诉|accessdate = 2016年7月14日|publisher =京华时报（[[新华社|新华社]]稿件）}}</ref>。2017年1月，以受贿罪判处苏荣无期徒刑，剥夺政治权利终身。

== 生平 ==
=== 早年经历 ===
1968年至1974年，任[[吉林省|吉林省]][[白城市|白城市]][[洮安縣|洮安县]][[那金镇|那金]][[人民公社|公社]]新力[[生产大队|大队]][[会计|会计]]、党支部副书记。其间，1970年1月加入中國共產黨。1974年1月参加工作，任吉林省[[洮安縣|洮安县]]那金公社党委副书记，不久升任党委书记。1975年至1980年，历任中共洮安县委常委，那金公社党委书记、岭下公社党委书记。1980年至1983年，任中共洮安县委副书记。其间，1980年9月至1981年1月，在[[中华人民共和国农业部|农业部]]干部培训班[[沈阳农学院|沈阳农学院]]班学习；1982年3月至1983年1月，在[[中共吉林省委|中共吉林省委]]党校党政干部培训班学习。

1983年至1984年，任中共吉林省[[白城地区|白城]]地委委员、[[扶余县|扶余县]]委书记。1984年至1985年，在中共吉林省委党校党政干部培训班学习。1985年至1987年，任中共吉林省白城地委副书记、[[地区行政公署|地区行署专员]]。1987年至1988年，任白城地委书记、行署专员。1988年至1989年，任中共白城地委书记。1989年至1992年，任中共吉林省[[四平市|四平市]]委书记。1990年至1992年，兼任四平市人大常委会主任。1992年至1995年，任[[中共吉林省委|中共吉林省委]]常委、秘书长。其间，1994年3月至1994年5月，在[[中央党校省部级干部进修班|中央党校省部级干部进修班]]学习。1995年至1996年，任中共吉林省委常委。1995年至1998年，任中共[[延边朝鲜族自治州|延边朝鲜族自治州]]委书记。1996年至2001年，任中共吉林省委副书记。其间，1994年至1997年，在[[吉林大学|吉林大学]]经济管理学院国际经济系世界经济专业[[在職研究生|在职研究生]]学习，获经济学硕士学位。

=== 三省书记 ===
2001年1月至2003年8月，任[[中国共产党青海省委员会|中共青海省委]]书记。2002年1月至2003年，兼任[[青海省人大常委会|青海省人大常委会]]主任。其间，2003年3月至2003年5月，再次在[[中央党校省部级干部进修班|中央党校省部级干部进修班]]学习。2003年8月至2006年，任[[中共甘肃省委|中共甘肃省委]]书记。

2004年至2006年，兼任[[甘肃省人大常委会|甘肃省人大常委会]]主任。

2006年至2007年，任[[中共中央党校|中共中央党校]]常务副校长（正部长级），时任校长是[[曾庆红|曾庆红]]。

2007年11月起，任[[中共江西省委|中共江西省委]]书记。2008年起，兼任[[江西省人大常委会|江西省人大常委会]]主任。

2012年1月4日，原中共江西省新余市委副书记、市人大常委会主任[[周建华|周建华]]（[[厅局级正职|正厅级]]）被[[中共江西省纪委|中共江西省纪委]]“[[双规|双规]]”。2014年1月23日，周建华因“受贿金额达1410万元”被[[江西省宜春市中级人民法院|江西省宜春市中级人民法院]]以[[賄賂|受贿罪]]一审判处[[死刑缓期执行|死刑，缓期两年执行]]。周建华被判死缓一案的直接起因被认为周建华向[[中华人民共和国监察部|监察部]]及相关领导举报苏荣夫妇和时任新余市委书记、后调任[[江西省发改委|江西省发改委]]的李某某的违法乱纪问题而遭打击报复<ref>[http://news.163.com/14/0621/09/9V8L07NK0001124J.html 举报苏荣者法庭上曾高呼：这里不是苏某私家刑庭] {{webarchive|url=https://web.archive.org/web/20140622232108/http://news.163.com/14/0621/09/9V8L07NK0001124J.html |date=2014-06-22 }}</ref>。苏荣落马后，周建华被改判[[無期徒刑|无期徒刑]]。

=== 政协时期 ===
2013年3月，当选[[中国人民政治协商会议第十二届全国委员会|中国人民政治协商会议第十二届全国委员会]][[中国人民政治协商会议全国委员会副主席|副主席]]。

=== 免职查办 ===
2014年6月14日，据[[新华社|新华社]]从[[中国共产党中央纪律检查委员会|中央纪委]]获得的消息，苏荣涉嫌严重违纪违法，目前正接受[[中国共产党纪律检查机关|组织]]调查<ref name="双规" />。苏荣还被指在2010年江西抚州唱凯决堤时瞒报死亡事故<ref>{{cite news|url = http://news.qq.com/a/20140615/005311.htm|title = 媒体称苏荣1年前即遭举报 曾瞒报决堤死亡事故|date = 2014-06-15|publisher = 腾讯网|accessdate = 15 June 2014}}</ref>。

2014年6月25日，据新华社消息，政协第十二届全国委员会常务委员会第六次会议表决通过了关于免去苏荣[[中国人民政治协商会议全国委员会副主席|全国政协副主席]]职务、撤销其[[中国人民政治协商会议全国委员会|全国政协委员]]资格的决定<ref name="被免" />。[[明鏡新聞出版集團|明鏡新聞網]]等海外媒体2014年2月21日報導称，蘇榮是[[周永康|周永康]][[周永康案|集團]]的主要成員之一，中共中央纪委已對其展開調查，而蘇榮的妻子和侄子已因涉嫌嚴重經濟犯罪被抓<ref name="zykjt">{{cite web|url = http://www.mingjingnews.com/MIB/news/news.aspx?ID=N000013364|title = 全國政協副主席蘇榮被查，妻侄被抓|publisher = 明镜新闻网|date = 2014-02-21|access-date = 2014-02-23|archive-url = https://web.archive.org/web/20140227080424/http://www.mingjingnews.com/MIB/news/news.aspx?ID=N000013364|archive-date = 2014-02-27|dead-url = yes}}</ref><ref>{{cite web|url = http://www.epochtimes.com/gb/14/2/22/n4089234.htm|title = 傳中共全國政協副主席蘇榮被查妻侄被抓|publisher = 大纪元新闻网|date = 2014-02-22}}</ref>。

2015年2月16日，中共中央纪委宣布，苏荣因违反组织、人事纪律，个人擅自改变组织决定，利用职务上的便利，在干部选拔任用、企业经营等方面为他人谋取利益，收受巨额[[贿赂|贿赂]]，[[滥用职权|滥用职权]]，造成国有资产重大损失，未落实党风廉政建设主体责任，对江西省出现的严重腐败问题负有主要领导责任，上述行为已构成严重违纪违法，其中受贿、滥用职权问题涉嫌犯罪。根据《中国共产党纪律处分条例》和参照《行政机关公务员处分条例》的有关规定，苏荣被开除党籍和公职，并将其涉嫌犯罪问题、线索及所涉款物移送[[司法機構|司法机关]]依法处理。另外，[[中华人民共和国最高人民检察院|最高人民检察院]]经审查决定，依法对苏荣以涉嫌[[賄賂|受贿罪]]立案侦查并采取强制措施，同时进行案件侦查工作<ref name="立案侦查" />。

2016年7月14日，经最高人民检察院批准，苏荣受贿、滥用职权、巨额财产来源不明案，由最高检指定山东省人民检察院侦查终结后，移送济南市人民检察院审查起诉。济南市人民检察院已向[[山东省济南市中级人民法院|济南市中级人民法院]]提起公诉<ref name="起诉">{{cite web|url = http://news.ifeng.com/a/20160714/49349645_0.shtml|title = 苏荣、杜善学等四案被提起公诉|accessdate = 2016年7月14日|publisher =京华时报（[[新华社|新华社]]稿件）}}</ref>。

2016年12月29日，济南市中级人民法院一审公开开庭审理了苏荣受贿、滥用职权、巨额财产来源不明一案。他被控收受1.1亿，另有8千万财产来源不明，法庭将择期宣判<ref>{{Cite web|url=http://www.boxun.com/news/gb/china/2016/12/201612291609.shtml|title=苏荣案受审：收受1.1亿 8千万财产来源不明|accessdate=2016-12-29|author=|date=2016-12-29|publisher=博讯新闻网}}</ref>。2017年1月23日，山东省济南市中级人民法院一审公开宣判，对被告人苏荣以受贿罪判处无期徒刑，剥夺政治权利终身，并处没收个人全部财产；以滥用职权罪判处有期徒刑七年；以巨额财产来源不明罪判处有期徒刑七年，决定执行无期徒刑，剥夺政治权利终身，并处没收个人全部财产。苏荣当庭表示服从法院判决，不上诉<ref>{{cite news|title=全国政协原副主席苏荣被判无期徒刑 受贿1.16亿|url=http://news.163.com/17/0123/17/CBFTB4J00001899O.html|accessdate=2017-01-23|agency=网易|date=2017-01-23}}</ref>。

== 家族 ==
