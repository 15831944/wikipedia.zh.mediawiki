'''苏昌兰'''（1971年7月18日—），女，生于[[广西壮族自治区|广西壮族自治区]][[岑溪市|岑溪市]]<ref>{{Cite web|url = http://www.chinesepen.org/blog/archives/17341|title = 苏昌兰|accessdate = |author = editor|date = 2015-02-13|publisher = 独立中文笔会}}</ref>，居住在[[佛山|佛山]][[南海|南海]]，[[维权人士|维权人士]]，[[社会活动家|社会活动家]]，为土地权利受侵害者、外嫁女维权人士、人权捍卫者进行维权。

== 维权经历： ==
*1999年，因为关注南海三山港的农村“出嫁女” 的股权问题开始介入广东的维权事业，是广东女权运动较早的先驱者之一。
*[[2013年4月|2013年4月]]主导和参与了安微省合肥市“让安妮重返校园”的维权活动。多年来并参与了寻找失踪孩子、救助童养媳和反坠胎的维权活动以及关注妇女儿童权益和计划生育等维权工作。
*[[2014年10月|2014年10月]]27日，苏昌兰被公安从家中抓走；<ref name=":0">{{Cite web|url = http://wqw2010.blogspot.com/2014/12/blog-post_45.html|title = 佛山农民维权代表苏昌兰被以“煽颠罪” 逮捕（图）|accessdate = |author = 柳怡|date = 2014-12-04|publisher = 维权网}}</ref><ref>{{Cite web|url = http://www.boxun.com/news/gb/china/2014/10/201410310225.shtml#.VsLMuJMrLsE|title = 广东维权人士苏昌兰因为发布声援香港占中帖子被刑拘|accessdate = |author = 博讯记者|date = 2014-10-31|publisher = 博讯网}}</ref>
*[[2014年12月|2014年12月]]3日被逮捕，羁押在广州市南海区看守所；<ref name=":0" />
*2017年3月31日，支持占中运动的维权人士苏昌兰、[[陈启棠|陈启棠]]被广东[[佛山市|佛山市]]中级人民法院以"[[煽动颠覆国家政权|煽动颠覆国家政权]]"的罪名分别判处有期徒刑3年和4年半。<ref >{{cite web|url=http://www.bbc.com/zhongwen/simp/chinese-news-39457509 | language= | title=中国两维权人士声援香港“占中”获重刑 |publisher=BBC中文网 |date=2017年3月31日 |accessdate= }}</ref>
