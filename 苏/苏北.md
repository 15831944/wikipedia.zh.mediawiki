'''苏北'''，即[[江苏省|江苏省]]的长江以北，常常与[[苏南|苏南]]相对。

傳統意義上的蘇北指代江蘇（包括上海，不包括盱眙、泗洪、徐州、宿迁、蕭縣、碭山）長江以北，崇明島、六合縣、浦口、江浦縣皆屬於蘇北。
* 地理上的苏北指江苏省[[长江|长江]]以北的所有地区，包括崇明島全境；
* 传统观念上的苏北则仅指江苏省操[[江淮官话|江淮官话]]的[[扬州|扬州]]、[[淮安|淮安]]、[[南通|南通]]、[[盐城|盐城]]、[[泰州|泰州]]的黃河故道以南地區，這些地方是傳統的南方地區、而非[[连云港|连云港]]，[[徐州|徐州]]（部份地區）、[[宿迁|宿迁]]（部份地區）等操[[中原官话|中原官话]]的北方地区；
* 政府官方表述的苏北包括江苏省北部的[[徐州|徐州]]、[[连云港|连云港]]、[[宿迁|宿迁]]、[[淮安|淮安]]、[[盐城|盐城]]5个[[省辖市|省辖市]]；

== 地理 ==
苏北地区的地势总体上极为低平，为坦荡的[[平原|平原]]，大部分地方河湖纵横，尤其以[[里下河平原|里下河平原]]的河网最为密集；苏北地区拥有众多湖泊，包括[[洪泽湖|洪泽湖]]、[[高邮湖|高邮湖]]、[[骆马湖|骆马湖]]、白马湖、宝应湖、邵伯湖等；同时[[京杭运河|京杭运河]]沿[[扬州|扬州]]、[[淮安|淮安]]和[[徐州|徐州]]向[[山东|山东]]延伸。

苏北地区东部濒临[[黄海|黄海]]，拥有大面积滩涂。历史上，苏北沿海是中国主要产盐区，近代以后，大部分地区改为棉区，仅保留[[连云港|连云港]]、[[盐城|盐城]]附近的盐场。

== 历史 ==
===行政区划===
历史上，江苏省辖区的长江两岸在元代以前大多分属不同的政区。如宋代分属[[淮南东路|淮南东路]]和[[江南东路|江南东路]]、[[两浙西路|两浙西路]]；元代分属[[河南江北行省|河南江北行省]]、[[江浙行省|江浙行省]]；明代一同划归南直隶。

清代江苏建省后，曾有数次以长江为界南北分治。江南4府1州属于驻扎[[苏州|苏州]]的江苏布政使管辖，江北3府2州和江宁府（南京）属于驻扎[[南京|南京]]的[[江宁布政使|江宁布政使]]管辖。清末一度在[[清江浦|清江浦]]（淮安）设立江淮巡抚，管辖江宁布政使所辖地区，与驻扎苏州的江苏巡抚并立。

民国时期，江苏省不再南北分治。省会设在南京（1912年到1929年）或[[镇江|镇江]]（1929年到1949年）。但是在[[汪精卫政权|汪精卫政权]]时期，江苏省省会迁往[[苏州|苏州]]，其实际管辖区域限于长江以南。苏北南半部设立以泰州为中心的[[苏北行署区|苏北行署区]]，北半部成立以[[徐州|徐州]]为中心的[[淮海省|淮海省]]。1949年到1952年，中共在华东大区下分别设立[[苏南行署区|苏南行署区]]（驻[[无锡|无锡]]）和[[苏北行署区|苏北行署区]]（驻[[扬州|扬州]]）。
===经济状况===
[[南宋|南宋]]以前，苏北几度繁华，曾为富庶之地。如春秋战国时的[[彭城|彭城]]（[[徐州|徐州]]）、秦汉时的[[淮阴|淮阴]]、[[淮南|淮南]]、隋唐时的[[江都|江都]]（[[扬州|扬州]]）、北宋时的[[江淮|江淮]]。

苏北在[[南宋|南宋]][[黄河夺淮|黄河夺淮]]以后洪水灾害频发，经济文化发展开始落后于江南地区。

在[[明清|明清]]时，由于地处[[京杭运河|京杭运河]]的节点上，[[淮安|淮安]]和[[扬州|扬州]]两大城市一度十分繁荣，苏北沿海为全国主要产盐区。

民国以后，苏北地区通过兴修导淮入江、[[苏北灌溉总渠|苏北灌溉总渠]]等水利工程，已经成为重要的农业区。但是经济发展水平与苏南比较仍有一定差距。

苏北南部的[[扬州|扬州]]、[[泰州|泰州]]、[[南通|南通]]3市发展水平相对较高，被划为[[苏中|苏中]]。南京和镇江也曾一度被划入苏中。

== 文化 ==

除[[徐州市|徐州市]]全市、[[宿迁市|宿迁市]]市区及[[沭阳县|沭阳县]]西北部，以及[[东海县|东海县]]西部等地属于北方[[中原官话|中原官话]]区，连云港市[[赣榆区|赣榆区]]属[[胶辽官话|胶辽官话]]区外，苏北地区的方言以[[江淮方言|江淮方言]]为主，泗陽縣，泗洪縣（1955從安徽省劃入），連雲港，沭陽縣，皆屬於江淮方言向中原官話過渡區域，既保留江淮方言特點，有能感覺到中原官話的味道，很難明確表示屬於哪一種語言區。

南通市的东部([[通州市|通州市]]、[[海门市|海门市]]、[[启东市|启东市]])以及[[泰州市|泰州市]]的[[靖江市|靖江市]]属于[[吴语|吴语]]区。

苏北地区的文化主体在黃河故道以南地區，大概位於淮安、扬州、泰州、盐城、南通、镇江一带（其中[[淮安|淮安]]、[[扬州|扬州]]、[[镇江|镇江]]皆为全国[[历史文化名城|历史文化名城]]），主要的地貌的里下河水乡。

苏北人创造了著名的[[淮扬菜系|淮扬菜系]]，明朝時期的淮安府，揚州府皆全境。

苏北地区一向出文人和卿相，[[枚乘|枚乘]]、[[鲍照|鲍照]]、[[秦观|秦观]]、[[张耒|张耒]]、[[张若虚|张若虚]]、[[吴承恩|吴承恩]]、[[刘鹗|刘鹗]]、[[郑板桥|郑板桥]]、[[朱自清|朱自清]]、[[梅兰芳|梅兰芳]]等皆出自苏北,新中国四代领导核心，就有[[周恩来|周恩来]]、[[江泽民|江泽民]]、[[胡锦涛|胡锦涛]]三人出自苏北。

苏北地区主要的地方戏包括[[淮剧|淮剧]]、[[扬剧|扬剧]]等。

== 交通 ==
===水运===
苏北地区的交通在以前一向以水运为主，[[京杭大运河|京杭大运河]]为交通动脉。
===公路===
苏北地区在1990年代以后，已经建成几乎通达所有县份的密集的[[高速公路|高速公路]]网，并可通过长江上新建的众多公路桥与[[上海|上海]]、[[苏南|苏南]]地区联系。
===铁路===
相较于密集的公路网，苏北的铁路网发展滞后，直到20世纪末，只有一条[[陇海铁路|陇海铁路]]穿过苏北北部边缘，苏北腹地大部分地区长期不通铁路，直到2000年以后，才有[[新长铁路|新长铁路]]、[[宁启铁路|宁启铁路]]陆续通车，但这些铁路线仍然为非电气化的单线轨道，铁路运力低下。而且缺少铁路过江通道。这种现象预计在2020年左右有所缓解。至2016年，[[宁启铁路|宁启铁路]]复线电气化建成运营，同时还有[[连盐铁路|连盐铁路]]，[[沪通铁路|沪通铁路]]，[[连淮扬镇铁路|连淮扬镇铁路]]，[[徐宿淮盐铁路|徐宿淮盐铁路]]，宁启铁路二期，连徐高铁，盐通高铁在建。

== 移民上海 ==
20世纪初，苏北地区曾有大量移民涌入上海，成为今日上海居民的重要组成部分。

他们大部分聚居在城市西北部的[[闸北区|闸北区]]、[[普陀区_(上海市)|普陀区]]，部分分布在城市东北部的[[杨浦区|杨浦区]]、[[虹口区|虹口区]]，与来自[[宁波|宁波]]的移民杂居。

一些精英分子，居于社会上层，例如著名[[北四行|北四行]]的创始人之二，[[淮安|淮安]]籍的银行家[[谈荔孙|谈荔孙]]（[[大陆银行|大陆银行]]董事长兼总经理）、[[周作民|周作民]]（[[金城银行|金城银行]]董事长兼总经理）等人。
