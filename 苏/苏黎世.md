'''苏黎世'''（{{lang-de|Zürich}} {{IPA-de|ˈt͡syːrɪç|}}；[[瑞士德語|瑞士德語]]：''Züri'' {{IPA|[ˈt͡syɾi]|}}），又譯'''苏黎士'''，是[[瑞士联邦|瑞士联邦]]的最大城市（2012年城市人口約38万，市区人口近110万，包括[[郊区|郊区]]在内的[[苏黎世都会区|苏黎世都会区]]人口达190万。），[[苏黎世州|苏黎世州]]的[[首府|首府]]。苏黎世是瑞士主要的[[商业|商业]]和[[文化|文化]]中心（瑞士的[[政治|政治]]中心和[[首都|首都]]在[[伯尔尼|伯尔尼]]）。苏黎世是瑞士[[银行|银行]]业的代表城市，世界[[金融|金融]]中心之一，[[瑞士聯合銀行|瑞士聯合銀行]]、[[瑞士信贷银行|瑞士信贷银行]]和许多[[私人银行|私人银行]]都将总部设在苏黎世。[[苏黎世国际机场|苏黎世国际机场]]是瑞士全国最大的[[机场|机场]]。[[国际足球联合会|国际足球联合会]]总部也设在苏黎世。根据2006年<ref>http://www.citymayors.com/features/quality_survey.html</ref>和2007年<ref>http://www.mercerhr.com/knowledgecenter/reportsummary.jhtml/dynamic/idContent/1128060#top50all</ref>的部分调查显示，苏黎世在这两年的[[世界最佳居住城市|世界最佳居住城市]]评选中高居全球首位。

「苏黎世」的名称可能来源于[[凯尔特语|凯尔特语]]中的“'''Turus'''”，一项有力的证据就是在出土的公元2世纪[[罗马帝国|罗马帝国]]占领时期的[[墓志铭|墓志铭]]上发现了古代该城名称的[[罗马化|罗马化]]形式——“'''{{lang|la|Turicum}}'''”。

== 历史 ==
{{main|苏黎世历史}}

[[File:Zurich.jpeg|thumb]]畔[[格罗斯大教堂|格罗斯大教堂]]一瞥]]

在[[罗马帝国|罗马帝国]]统治时期罗马高卢时期，苏黎世（那时候它的[[拉丁文|拉丁文]]名叫'''Turicum'''）属于帝国的[[比利时高卢|比利时高卢]]行省（自公元90年始），是在该省与[[上日耳曼尼亚|上日耳曼尼亚]]行省交界的边境上通过[[利马特河|利马特河]]水路向帝国运送货物的主要收税点。据说在835年，[[法兰克帝国|法兰克帝国]][[查理大帝|查理大帝]]的孙子、[[东法兰克王国|东法兰克王国]]国王[[路易二世_(东法兰克王国)|路易二世]]在该地的一座[[古罗马|古罗马]][[城堡|城堡]]的遗址上重建了一座[[加洛林王朝|加洛林王朝]]的城堡（拉丁语：''"in castro Turicino iuxta fluvium Lindemaci"''）。路易还于853年为其女儿[[赫德嘉_(虔诚者路易)|希尔德加德]]建造了妇女[[修道院|修道院]]（今[[妇女大教堂|妇女大教堂]]）。他把今天的[[苏黎世州|苏黎世州]]、[[乌里州|乌里州]]的土地和[[阿尔比斯山|阿尔比斯山]]上的[[森林|森林]]赠与[[本笃会|本笃会]]女修道院，并授予该女修道院豁免权，把它置于自己的直接统治之下。

1045年，[[德意志|德意志]]国王及[[神圣罗马帝国皇帝|神圣罗马帝国皇帝]][[亨利三世_(神圣罗马帝国)|亨利三世]]（亦即[[巴伐利亚|巴伐利亚]]公爵亨利六世）批准女修道院拥有市场、征税和造币的权力，从而使该女修道院的院长成为该城市的有力的统治者。

[[File:Murerplan_Zuerich.jpg|thumb]]'''》]]

1218年，当统治苏黎世的[[策林根家族|策林根家族]]直系绝嗣后，苏黎世成为[[皇家直辖市|皇家直辖市]]。在1230年代该城四周建起了城墙，围绕[[面积|面积]]有38[[公顷|公顷]]。1234年，皇帝[[腓特烈二世_(神圣罗马帝国)|腓特烈二世]]晋升妇女修道院院长为女[[公爵|公爵]]级别。女院长指定市长，她还经常把造币的工作委派给市民。可是，当1336年在[[鲁道夫·布伦|鲁道夫·布伦]]领导下通过《经济行会法》以后，女修道院的政治权力逐渐衰弱。鲁道夫·布伦也成为第一个非女院长指定的独立派市长。

苏黎世在罗马帝国时，是通过水路向帝国运送货物的主要收税点，后来成为[[神圣罗马帝国|神圣罗马帝国]]的领地，1218年成为帝国属下的自由城，1351年加入瑞士联邦，成为瑞士联邦的第五个州，1440年由于和其他州发生土地纠纷，而引发战争，被瑞士联邦开除，1446年在内战中被击败, 1450年重新被联邦接纳。

[[慈运理|慈运理]]开始瑞士宗教改革时正是苏黎世的主要牧师。他从1518年直到1531年去世一直住在这裡。

1839年9月6日的[[苏黎世叛乱|苏黎世叛乱]]后，城市为市民的要求而扩展。在17世纪建成的圍牆在沒有被圍攻的情形下被拆毀，以緩和郊區人士對城市霸權的憂慮。1859年[[奧地利|奧地利]]、[[法國|法國]]和[[薩丁尼亞|薩丁尼亞]]簽訂[[蘇黎世條約|蘇黎世條約]]。

从1847年开始，瑞士境内的第一条铁路——“西班牙-布罗特利-巴恩铁路”把苏黎世和[[巴登_(瑞士)|巴登]]连接在一起，而[[苏黎世火车总站|苏黎世火车总站]]则成为瑞士铁路网的起点。现在的总站大楼的历史可以上溯到1871年。

=== 盾形纹章 ===

[[File:Banner_Zuerich_1585.jpg|thumb]]

苏黎世的蓝白两色盾形[[纹章|纹章]]自1389年被採用，是从自1315年使用蓝色和白色条纹的旗幟演化。第一次證實有同樣設計的旗幟是在1434年。纹章以兩雙獅子作為襯托。旗幟上紅色的長帶（Schwenkel）有不同的說法：在苏黎世人民來說，那是[[魯道夫一世_(德意志)|魯道夫一世]]所賜的榮譽。在周邊人民眼中是一種恥辱的象徵，紀念在1292年在[[溫特圖爾|溫特圖爾]]失去的旗幟。

現時，苏黎世州和苏黎世市使用同样的盾形纹章。

== 地理 ==

苏黎世位于从[[苏黎世湖|苏黎世湖]]北端開始的[[利马特河|利马特河]]四周被长满樹林的小山包围。[[利马特河|利马特河]]和[[锡尔河_(瑞士)|锡尔河]]的交汇於市[[苏黎世火车站|火车总站]]广场，鄰近[[瑞士國家博物館|瑞士國家博物館]]。地理上（和歷史上）的市中心是[[利马特河|利马特河]]左岸的一個小山崗[[林登霍夫山|林登霍夫山]]，在[[利马特河|利马特河]]離開[[苏黎世湖|苏黎世湖]]的源頭北面約700米。現在城市擴展超越小山所限的地界。

== 工商业 ==

[[瑞士聯合銀行|瑞士聯合銀行]]、[[瑞士信貸銀行|瑞士信貸銀行]]、[[蘇黎世金融服務集團|蘇黎世金融服務集團]]，以及許多私人[[銀行|銀行]]的總部都設於蘇黎世這個瑞士的[[商業中心|商業中心]]。主要是由於瑞士銀行的保密能力，使蘇黎世成為[[離岸銀行|離岸銀行]]業務的[[世界|世界]]主要中心。[[金融|金融]]方面的帳戶大約佔據整個城市四分之一的[[經濟|經濟]]活動。瑞士[[證券交易所|證券交易所]]的總部亦設於蘇黎世（參見[[瑞士銀行業|瑞士銀行業]]）。

蘇黎世是一個前列的金融中心及連續成為全世界基本生活條件最好的城市。{{Fact|time=2007-07-16T11:04:34Z}}[[大蘇黎士區|大蘇黎士區]]是瑞士的經濟中心及眾多國際公司的基地。蘇黎士地區的[[國內生產總值|國內生產總值]]為2,100億[[瑞士法郎|瑞士法郎]]（1,600億美元）或每人58,000瑞士法郎（45,000美元，2005年）{{Fact|time=2007-07-16T11:04:34Z}}

== 交通 ==

[[File:Trams_in_Zurich.jpg|thumb]]

苏黎世为全国交通重要交汇点之一，[[苏黎世国际机场|苏黎世国际机场]]為瑞士最大的航空口岸，位於東北郊區不到10公里远的[[克洛滕|克洛滕]]。

苏黎世有多个火车站。其中[[蘇黎世站|蘇黎世站]]为全国最大火车站，还有[[苏黎世Oerlikon|苏黎世Oerlikon]]、[[苏黎世史特荷芬|苏黎世史特荷芬]]、及[[苏黎世Altstetten|苏黎世Altstetten]]等城郊铁路站。[[Cisalpino|Cisalpino]]、[[ICE列车|ICE列车]]及[[TGV|TGV]]高速列车在苏黎世都有班次。

苏黎世通过A1、A3、A4、A20、A51国立高速公路连接西面的[[伯尔尼|伯尔尼]]、[[日内瓦|日内瓦]]，东面的[[圣加仑|圣加仑]]，西北面的[[巴塞尔|巴塞尔]]和东南面的[[薩爾甘斯|薩爾甘斯]]。

苏黎世市內以至苏黎世州，[[ZVV|ZVV]]公共运输网络的交通密度在世界前列。若加以频率，以在苏黎世7分钟计，成为各维度最密集的网络。传言中央区地面沒有一点会在[[公共汽车|公共汽车]]、[[无轨电车|无轨电车]]、[[有轨电车|有轨电车]]车站或火车站150米以外。

公共运输网络还包括在河上或湖上的小艇、[[纜索鐵路|纜索鐵路]]以至來往阿德利斯維爾和Felsenegg之間的吊車。購買一程的票可以在所有的公共运输（列車、有轨电车、无轨电车、公共汽車、小艇）上有效。
