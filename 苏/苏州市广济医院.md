'''苏州市广济医院'''，又称'''苏州大学附属广济医院'''，是[[中国|中国]][[江苏省|江苏省]][[苏州市|苏州市]]的一所[[精神病|精神病]]专科[[医院|医院]]，1923年由美国传教士惠更生（James R.Wilkson）创立，2019年3月获评三级甲等医院<ref name='r'>{{cite web |title=苏州市广济医院获评三级甲等 |url=http://www.subaonet.com/2019/0302/2415948.shtml |website=苏州新闻网 |accessdate=2019-03-03}}</ref>，院址位于苏州市相城区广前路11号<ref>{{Cite book|title=苏州词典|last=江洪等主编|first=|publisher=苏州：苏州大学出版社|year=1999年9月|isbn=7-81037-478-8|location=|pages=833-835}}</ref><ref>{{Cite book|title=2005中国医疗卫生机构名录 华东地区卷|last=《中国医疗卫生机构名录》编委会编|first=|publisher=北京：中国协和医科大学出版社|year=2005.06|isbn=7-81072-658-7|location=|pages=63-70}}</ref>

==历史==
*1895年惠更生建立“福音医院”，是苏州第一所精神病诊治医院。后又建立“更生医院”，此二医院于1937年因抗战爆发而停办<ref name='w'>{{cite book |author1=苏州市地方志编纂委员会 编 |title=《苏州市志》 |date=1995 |publisher=江苏人民出版社 |isbn=7-214-01383-5 |chapter=第五十卷 第一章 第二节}}</ref>。
*1951年苏州市为了收治抗美援朝志愿军伤病员，在更生医院原址建立“苏南康复医院”（后为“江苏省第三康复医院”）。1955年9月改为专门收治中国人民解放军军中精神病患者<ref name="q">{{cite book |author1=张应忠主编 |author2=苏州市卫生局编 |title=《苏州卫生志》 |date=1995 |publisher=江苏科学技术出版社 |isbn=7-5345-1915-2 |chapter=第一篇 第七章}}</ref>。1958年转苏州市政府管辖，改名“苏州市精神病院”，同时对社会开放，时有病床150张<ref name='w' />。1984年改名“广济医院”<ref name="q" />。
*1969年，苏州市精神病院分出半数人员及设备，至南通建立南通地区精神病防治院。1970年在常熟开设门诊，此后先后帮助吴县、太仓、昆山、吴江、沙洲、江阴等县成立精神病防治机构<ref name='w' />。1973年起又在苏州市内各街道社区陆续开展社区防治工作，创办精神病工疗站<ref name="q" />。
*2010年起由苏州大学附属。2016年增挂苏州市心理卫生中心、苏州市精神卫生中心两块牌子<ref name='r' />。
*2017年8月整体搬迁至相城区广前路11号。新院占地129亩，拥有床位1500张<ref>{{cite web |title=广济医院新院试运行 |url=http://news.sina.com.cn/o/2017-08-11/doc-ifyixcaw4047956.shtml |website=新浪新闻 |accessdate=2019-03-03 |date=2017-08-11}}</ref>。原广济路院址将新建一座三级康复医院<ref>{{cite web |title=苏州康复医学会成立 广济医院原址将建一所三级康复医院 |url=http://www.subaonet.com/2018/1029/2336676.shtml |website=苏州新闻网 |accessdate=2019-03-03}}</ref>。
