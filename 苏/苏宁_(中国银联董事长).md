'''苏宁'''（{{bd|1947年|||}}）籍贯不详，[[中国人民银行|中国人民银行]]副行长、[[中国银联股份有限公司|中国银联股份有限公司]]董事长。

==生平==
1982年自[[中国人民大学|中国人民大学]]毕业。是高级工程师。1982年到1989年，先后在[[国家计委|国家计委]]综合司、国家计委预测中心、国家信息中心信息部任职。1989年到1997年，历任国务院研究室宏观经济研究司副司长、司长。1997年到1998年，任国务院研究室党组成员兼宏观经济研究司司长。1998年到2003年，任[[全国人大常委会预算工作委员会|全国人大常委会预算工作委员会]]副主任。2003年11月，任中国人民银行党委委员、副行长。2007年7月起，兼任[[中国人民银行上海总部|中国人民银行上海总部]]主任。2008年10月起，兼任[[中国人民银行货币政策委员会|中国人民银行货币政策委员会]]委员。2010年6月起，不再担任中国人民银行党委委员、副行长。后任[[中国银联股份有限公司|中国银联股份有限公司]]董事长，任至2014年10月退休<ref>{{Cite web |url =  http://renwu.hexun.com/figure_1069.shtml|title =苏宁  |author =  |date =  |publisher = 和讯 |language =  |accessdate = 2018-08-11 |quote =  }}</ref><ref>{{Cite web |url = http://finance.people.com.cn/GB/51844/51850/191871/11704662.html |title = 中国人民银行副行长苏宁简介 |author =  |date =2010-05-26  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://finance.sina.com.cn/g/20070717/08401545040.shtml  |title = 央行副行长苏宁兼任上海总部主任 |author =  |date =2007-07-10  |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。
