{{engname|《蘇東坡》|Su Dong Po}}，2012年上映于[[中国大陆|中国大陆]]的古装剧，讲述了[[北宋|北宋]]大文豪[[苏轼|苏轼]]波澜壮阔、精彩斑斓的人生，主演是[[陆毅|陆毅]]、[[林心如|林心如]]、[[萨日娜|萨日娜]]、[[韩雪|韩雪]]。

== 剧情 ==
[[北宋|北宋]][[嘉佑|嘉佑]]二年，[[眉州|眉州]][[眉山|眉山]]考生[[苏洵|苏洵]]（今[[四川省|四川省]][[眉山市|眉山市]]）带着自己的两个儿子：苏轼、[[苏辙|苏辙]]进京（进京，不是[[北京|北京城]]，而是[[汴梁|汴梁]]，今[[河南省|河南省]][[开封市|开封市]]）赶考，得中[[进士|进士]]{{noteTag|1=进士中国古代[[科举|科举]]制度中，通过最后一级考试者，称为进士，是古代科举[[殿试|殿试]]及第者之称，意为可以进授爵位之人。 }} 。

苏轼的父亲苏洵以《六国论》名震汴梁，父子的名声开始流传天下。

苏轼虽然文采出众，但是思想保守，[[宋神宗|宋神宗]]时期[[熙宁|熙宁]]五年，他上书反对[[王安石变法|王安石变法]]，从祠部员外郎贬为外放的杭州通判，三年后移知[[密州|密州]]、[[徐州|徐州]]、[[湖州|湖州]]。熙宁十年，他又到[[徐州|徐州]]任职。

[[元丰|元丰]]二年，苏轼转入[[乌台诗案|乌台诗案]]，幸亏朋友积极出面才救回性命。苏轼再次被贬到[[黄州|黄州]]，并且认识了大和尚[[佛印|佛印]]。

[[宋哲宗|宋哲宗]]即位后，[[宣仁皇后|高太皇太后]]垂簾聽政，保守派[[司马光|司马光]]等得势，苏轼被召回朝廷，但是又因为反对司马光等人的行为而被保守派排挤。

[[绍圣|绍圣]]元年，苏轼再次被贬到[[惠州|惠州]]、[[儋州|儋州]]（今[[海南岛|海南岛]]）。

[[元符|元符]]三年，[[宋徽宗|宋徽宗]]即位，被安置於[[常州|常州]]，在[[常州|常州]]去世。

== 评价 ==
中国重大革命历史题材影视创作领导小组对该剧的评价是：“该剧生动表现了苏东坡为政清廉、造福百姓、著书立说、正直旷达的伟大人生，涉及重大历史史实基本无误，重要历史人物定位基本正确。全剧成功地做到了历史真实和艺术真实的结合，人物形象鲜明，对于弘扬[[中华民族|中华民族]]优秀传统文化具有十分积极的意义。”<ref name=" fish" />

中国文联原副主席[[李准|李准]]：“这是近几年来历史题材电视剧中拍得相当成功的一部作品，新鲜、真实、厚重，雅俗共赏又有现实意义。”<ref>{{cite web |url=http://ent.163.com/12/1005/04/8D1D7BS100032DGD.html |title=陆毅版《苏东坡》7日开播 杭州元素值得期待 |accessdate= 2012-10-05|publisher=网易 |work= |language =zh-cn}}</ref>
