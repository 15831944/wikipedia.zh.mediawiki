'''苏巴斯·钱德拉·鲍斯'''（{{lang-bn|সুভাষ চন্দ্র বসু}}，{{IAST|Subhāṣ Candra Basu}}，{{audio|Bn-নেতাজি সুভাষচন্দ্র বসু.oga|发音}}；{{lang-en|Subhash Chandra Bose}}；{{bd|1897年|1月23日|1945年|8月18日|catIdx=Bose, Subhash Chandra}}），生於[[英属印度|英属印度]][[孟加拉管辖区|孟加拉管辖区]][[克塔克|克塔克]]，律師與政治人物，參與印度獨立運動，為[[自由印度臨時政府|自由印度臨時政府]]的領導人，以及[[印度国民军|印度国民军]]的最高指揮官。

==生平==
鲍斯出生於奥里萨邦的克塔克，家中有14位兄弟姐妹，排行第9。於[[加尔各答|加爾各答]]完成大學學業，之後前往[[英国|英国]][[剑桥大学|剑桥大学]]留學，1920年通过了印度文官考试，这对当时的印度青年来说，是一条通向上流社会的仕途。

1921年參與[[圣雄甘地|甘地]]的非暴力[[不合作運動|不合作運動]]。但是他是爭取印度獨立的激進派。<ref>{{Cite web |url = http://gushi.tw/archives/9775 |title = 是日本帝國的野心，還是亞洲人的覺醒？──「大東亞共榮圈」究竟是怎麼一回事？（下） |author = John Toland |date = 2015-07-19 |publisher = 故事─這是寫給所有人的故事 |language = 中文 |accessdate = 2015年7月27日 |quote = 鮑斯身材高大──可以俯瞰東條和巴莫──還是個激昂的革命分子}}</ref>在1937年到1939年曾擔任[[印度国民大会党|印度国民大会党]]主席，由於鮑斯是一位左翼人士，不承認甘地在1939年就任國大黨主席而與之決裂，并在国大党内部建立了一个派别——[[全印前进同盟|前进同盟]]。[[太平洋战争|太平洋战争]]爆發後，鲍斯在1941年由印度經[[阿富汗|阿富汗]]、[[苏联|苏联]]而前往[[德国|德国]]，並且在德國組成兩千人的[[印度国民军|印度国民军]]，1943年他受[[日本|日本]]的邀請，搭乘德國的[[U型潜艇|U型潜艇]]由[[基尔|基尔]]出發，借道[[马达加斯加|马达加斯加]]，在印度洋潛艦不靠岸就直接在印度洋用橡膠救生艇轉泊到日本[[伊號第二十九潛艦|伊號第29潛艦]]前往日本，他前往日本後在[[东京|东京]]方面的支持下。赴[[新加坡|新加坡]]召集在[[馬來亞戰役|馬來亞戰役]]中投降的數千名印度兵，並擔任[[印度獨立聯盟|印度獨立聯盟]]的領袖，[[菲律賓第二共和國|菲律賓]]宣布獨立後的一週，1943年10月21日成立[[自由印度臨時政府|自由印度臨時政府]]，同年十一月初參與由日本主導的[[大東亞會議|大東亞會議]]。<ref>{{Cite web |url = http://gushi.tw/archives/9775 |title = 是日本帝國的野心，還是亞洲人的覺醒？──「大東亞共榮圈」究竟是怎麼一回事？（下） |author = John Toland |date = 2015-07-19 |publisher = 故事─這是寫給所有人的故事 |language = 中文 |accessdate = 2015年7月27日 }}</ref>鲍斯並於建立臨時政府之際就任印度國民軍的最高司令官。

1945年8月15日，[[日本投降|日本投降]]。鲍斯在同年8月18日飛抵[[臺北松山機場|台北飛行場]]，並攜帶兩件手提行李，內有“來自東南亞的300萬印度人所捐獻的珠寶及金飾”；同日下午2時轉搭[[九七式重轟炸機|九七式重轟炸機]]前往[[满洲国|滿洲]][[大连市|大連]]，卻在台北飛行場飛機起飛之際，螺旋槳脫離而失控墜毀。擔任駕駛的泷澤少校，以及同乘的[[四手井綱正|四手井綱正]]中將及一名軍官均死亡；{{sfn|児島襄|1974|p=170}}時鮑斯身受重傷與燒傷嚴重的四手井綱正中將等日本軍官轉送[[國立臺灣大學醫學院附設醫院|臺大醫院]]搶救，鮑斯當夜11點傷重不治，享年48歲。<ref> {{Cite book | author = Barbara D. Metcalf（芭芭拉麥卡夫）、Thomas R. Metcalf（湯瑪斯麥卡夫） | title = A Concise History of India（劍橋印度簡史） | location = 臺灣新北市 | publisher = 左岸文化| date = 2005年7月 | pages = 256－257 | ISBN = 986-7854-95-0 | accessdate = 2015年1月26日 | language = 中文}} </ref><ref>李靖棠,"意外死於台北 走錯路的「印度自由之父」蘇巴斯鮑斯"[http://www.chinatimes.com/realtimenews/20170910001932-260417],中時電子報,2017年09月10日13:35 .</ref>

==後事==
遺體在[[臺北市殯葬管理處|台北市營火葬場]]火化，並在台北的[[西本願寺_(臺北市)|西本願寺]]舉行法會，遺骨之後運至日本，安葬於東京[[杉並區|杉並區]]的{{tsl|jp|蓮光寺 (杉並区)|蓮光寺}}。他死後很長一段期間，印度當地仍然有不少的傳說指稱他還健在。目前印度國會議事堂前的三幅肖像中，他與[[圣雄甘地|甘地]]、[[贾瓦哈拉尔·尼赫鲁|尼赫魯]]並列，由此可見他在印度的評價。

==地位==
自尼赫魯以來，印度歷任總理都與日本保持著友好關係，在2014年競選總理的[[纳伦德拉·莫迪|纳伦德拉·莫迪]]在選舉期間，更曾明確表示將以國家名義向印度獨立運動領袖鮑斯授予非軍方人士最高勳章。<ref>{{Cite web |url = http://www.nippon.com/hk/column/g00191/ |title = 解讀印度莫迪新政權通過為鮑斯正名所發出的訊號 |author = PEMA Gyalpo |date = 2015-01-26 |publisher = 日本網 |language = 中文 |accessdate = 2015年1月29日 }}</ref>
