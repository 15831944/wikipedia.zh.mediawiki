'''苏尚卿'''（{{bd|1991年|5月30日}}），中国配音演员。在成为职业配音演员前，他常用ID'''西呱双'''发布配音作品。代表作有电影《[[大鱼海棠|大鱼海棠]]》中的湫<ref>{{Cite web |url=http://yule.sohu.com/20160708/n458380167.shtml|title=《大鱼海棠》今日公映 国漫力作传承东方古韵|accessdate=2018-03-05 |author= |date=2016-07-08 |work=杜渐 UK028 |publisher=搜狐娱乐|language=Zh-hans }}</ref>、《[[昨日青空|昨日青空]]》的屠小意<ref>{{Cite web |url=http://www.xinhuanet.com/ent/2018-10/19/c_1123583009.htm|title=《昨日青空》发王一博“年少心事”MV|accessdate=2019-01-21 |author= |date=2018-10-19 |work=苏姗 |publisher=新华网|language=Zh-hans }}</ref>，电视剧《[[择天记|择天记]]》中的陈长生<ref>{{Cite web |url=http://games.sina.com.cn/t/n/2017-05-18/fyfkkmc9593879.shtml|title=有毒预定！声优天团吐槽网易游戏 整个人都苏了！|accessdate=2018-03-05 |author= |date=2017-05-18 |work= |publisher=新浪游戏|language=Zh-hans }}</ref>，动漫《[[狐妖小红娘|狐妖小红娘]]》中的平丘月初、《[[银之守墓人|银之守墓人]]》的陆水银以及《[[全职高手|全职高手]]》中的乔一帆<ref>{{Cite web |url=http://www.missevan.com/seiy/1183|title=苏尚卿_M站声优大百科|accessdate=2018-03-18 |author= |date= |work= |publisher=猫耳FM|language=Zh-hans }}</ref>，游戏《[[梦间集|梦间集]]》的金铃索等。是虚拟歌手[[徵羽摩柯|徵羽摩柯]]的声源。现为配音团队[[729声工场|729声工场]]成员<ref>{{Cite web |url=http://ent.sina.com.cn/hr/2017-05-18/doc-ifyfkqiv6520134.shtml|title=男神组团加盟《吴江锦时书》 实力挑战多变角色|accessdate=2018-03-18 |author= |date=2017-05-18 |work=Blue |publisher=新浪娱乐|language=Zh-hans }}</ref>。
