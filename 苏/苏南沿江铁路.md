'''苏南沿江铁路'''又称'''南沿江城际铁路'''，是[[中华人民共和国|中华人民共和国]][[江苏省|江苏省]]规划中的一条连接[[南京|南京]]与[[苏州|苏州]]的高速铁路。江苏南沿江城际铁路开工动员会于2018年10月8日举行<ref name = "加快推进江苏高铁建设暨江苏南沿江城际铁路开工动员会在常州举行 娄勤俭宣布江苏南沿江城际铁路建设开工 吴政隆王同军讲话">{{cite web | author = 耿联, 黄伟, 剑飞 | title = 加快推进江苏高铁建设暨江苏南沿江城际铁路开工动员会在常州举行 娄勤俭宣布江苏南沿江城际铁路建设开工 吴政隆王同军讲话 | publisher = 新华日报 | date = 2018-10-09 | language = zh | accessdate = 2018-10-09 | url = http://xh.xhby.net/mp3/pad/c/201810/09/c539910.html}}</ref>。

== 历史 ==
* 2011年3月30日
江苏省铁路办与常州发改委共同组织召开了“沿江城际铁路建设方案征求意见会”，由中铁第四勘察设计院介绍了沿江城际铁路的基本情况，就沿江铁路线路的走向、站位设置与常州市的规划、国土、建设、交通、水利、环保等部门进行商讨。

据了解，沿江城际铁路线路预计全长300余公里，设计时速约为250公里。从江苏起，起点站为南京，途经镇江(句容市)、常州(金坛市、武进区、戚墅堰区)、无锡(惠山区、江阴市)、苏州(张家港、常熟、太仓市)等地至上海。这条铁路的前身是规划中的镇南铁路，起自镇江，经常州、江阴、张家港、常熟、太仓市至上海嘉定南翔。随着长三角铁路网的发展，铁路部门与江苏省修改了镇南铁路的规划，将其从镇江向西延伸至南京，并确定为客货两用。

沪宁间已确定有[[沪宁铁路|沪宁铁路]]、[[沪宁城际铁路|沪宁城际铁路]]、[[京沪高铁|京沪高铁]]3条铁路通道，为什么还要建设第4条快速铁路线？据分析，江阴、张家港、常熟、太仓市沿江大部分地区仅仅依靠高速公路，无法纳入长三角“1－2小时交通圈”，为了避免在长三角加速一体化的经济圈中被边缘化，有建设沿江城际铁路的需求。

* 2014年2月10日江苏省在南京召开了全省交通运输工作会议，介绍今年本省铁路建设情况，沿江城际铁路前期准备工作仍处于快速推进中<ref>{{Cite web |url = http://news.sina.com.cn/c/2014-02-10/154729428358.shtml |title = 江苏迎来铁路建设“大年” |date = 2015-02-10 |publisher = 新浪网 }}</ref>。
* 2014年9月15日从江苏省铁路建设推进工作会议上获悉，沿江城际铁路将于明年底前开工建设。
* 2018年4月19日公示《新建铁路苏南沿江铁路环境影响评价全本》<ref name = "新建铁路苏南沿江铁路环境影响报告书（全本公示版）"/>。

== 线路 ==
当前线路由[[南京南站|南京南站]]引出，利用[[宁杭高铁|宁杭高铁]]在[[江宁站|江宁站]]分叉，经过[[南京市|南京市]][[江宁区|江宁区]]、[[镇江市|镇江市]][[句容市|句容市]]、[[常州市|常州市]]（[[金坛区|金坛区]]、[[武进区|武进区]]）、[[无锡市|无锡市]]（[[江阴市|江阴市]]）、[[苏州市|苏州市]]（[[张家港市|张家港市]]、[[常熟市|常熟市]]、[[太仓市|太仓市]]）引入[[太仓站|太仓站]]后利用[[沪通铁路|沪通铁路]]进入上海枢纽<ref>{{cite news|title=南沿江城际铁路正式开建 江苏6地4年后融入沪宁1小时高铁圈|url=http://www.yangtse.com/jiangsu/2018/10/09/1370013.html|publisher=扬子晚报|date=2018-10-09}}</ref>。

共设[[南京南站|南京南站]]、[[江宁站|江宁站]]、[[句容站|句容站]]、[[金坛站|金坛站]]、 [[武进站(苏南沿江)|武进站]]、[[江阴站|江阴站]]、[[张家港站|张家港站]]、[[常熟站|常熟站]]、[[太仓站|太仓站]]九站。

== 技术标准 ==
* 铁路等级：客运专线。
* [[正线|正线]]数目：双线。
* 限制[[坡度|坡度]]：一般20‰，困难30‰。
* 路段[[旅客列车|旅客列车]]设计行车速度：350km/h（江宁联络线120km/h，太仓联络线160km/h），局部限速。
* [[最小曲线半径|最小曲线半径]]：一般地段7000米，困难地段5500米（江宁联络线一般地段1000米，困难地段800米；太仓联络线一般路段1600米，困难路段1400米）
* 线间最小距离：5.0米。
* 牵引种类：[[电力机车|电力]]牵引。 	
* [[到发线|到发线]]有效长度：650米。
* [[闭塞_(铁路)|闭塞]]类型：[[自动闭塞|自动闭塞]]。
