'''苏丹穆罕默德·萨拉胡丁机场'''（印尼语：Bandar Udara Sultan Muhammad Salahudin，{{airport codes|BMU|WADB}}）也叫'''比马机场'''<ref>{{WAD|WADB|WADB - Muhammad Salahuddin Airport - Bima, Indonesia}}</ref>，是位于[[印度尼西亚|印度尼西亚]][[西努沙登加拉省|西努沙登加拉省]][[松巴哇岛|松巴哇岛]][[比马|比马]]以南{{convert|10|km|mi|1}}的一座机场。机场名称来自比马当地历史上的一位国王<ref>{{cite web |url=https://www.gatra.com/nusantara/bali-nusa-tenggara/293211-bandara-lia-lombok-jadi-bandara-syeikh-zainudin-ditanggapi-positif |title=Bandara LIA Lombok Jadi Bandara Syeikh Zainudin Ditanggapi Positif |author=Hernawardi |date=2017-11-02 |website=www.gatra.com |publisher=Gatranews |access-date=2017-11-09 |quote=}}</ref>。

未来，苏丹穆罕默德·萨拉胡丁机场将在机场北部建造一座新的航站楼，现有的老航站楼将作为货运及停车场使用，现在位于机场北部的办公楼及工程楼将被拆除，工作将在2019年进行。新航站楼的面积将达到5,000平方米，二层作为候机厅。此外机场跑道也将向西北方向延长，该工程将改变一条河流的流向。苏丹穆罕默德·萨拉胡丁机场已准备好在将来成为国际标准机场<ref>{{cite web |url=http://www.lombokpost.net/2017/10/12/terminal-penumpang-bandara-bima-akan-dipindahkan/ |title=Terminal Penumpang Bandara Bima Akan Dipindahkan |last= |first= |date=2017-10-12 |website=www.lombokpost.net |publisher=Editor Lombok Post |access-date=2017-11-09 |quote=}}</ref>。

2017年6月16日，[[三佛齐航空|三佛齐航空]]下属的{{le|楠航空|NAM Air}}开通了从[[巴厘岛|巴厘岛]][[伍拉·赖国际机场|伍拉·赖国际机场]]和[[龙目岛|龙目岛]][[龙目国际机场|国际机场]]飞到比马的航班，由[[波音737|波音737]]客机执飞，每趟航班提供120个座位（包括8个商务舱和112个经济舱）。截止2017年6月已有4家航空公司经营飞往苏丹穆罕默德·萨拉胡丁机场的航线。<ref>{{cite web |url=https://daerah.sindonews.com/read/1212244/174/jelang-arus-mudik-bandara-m-salahuddin-bima-tambah-armada-pesawat-baru-1497002454 |title=Jelang Arus Mudik, Bandara M Salahuddin Bima Tambah Armada Pesawat Baru |author=Edy Irawan |date=2017-06-09 |website=daerah.sindonews.com |publisher=SINDOnews |access-date=2017-11-09 |quote=}}</ref>
