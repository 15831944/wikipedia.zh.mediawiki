'''苏越'''（{{bd|1955年|9月12日|2018年|8月22日}}），生于[[北京|北京]]，[[中华人民共和国|中华人民共和国]]作曲家<ref name=shaowu>{{Cite web |url =http://wenshu.court.gov.cn/content/content?DocID=46e43de0-91fb-412a-ae04-a90a00a50f3a  |title =苏越与邵武市凤舞九天歌舞厅侵害作品表演权纠纷一审民事判决书  |author =  |date = 2018-06-25 |publisher =中国裁判文书网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
3岁时耳膜穿孔。曾当农民，卖过红薯、进过福利工厂，还考过各种文艺团体。中学时开始拉[[小提琴|小提琴]]。1978年，考入[[总政歌舞团|总政歌舞团]]担任演奏员。1987年，苏越到[[日本|日本]]学习[[电子音乐|电子音乐]]、音乐制作和音乐文化产业化等。1990年代初，他回国后成为[[中国内地|中国内地]]推行歌手签约制的第一人，先后捧红了[[黄格选|黄格选]]、[[谢东|谢东]]、[[朱桦|朱桦]]、[[白雪_(歌手)|白雪]]、[[戴娆|戴娆]]等歌手。曾创作过许多流行歌曲如《[[血染的风采|血染的风采]]》、《[[黄土高坡|黄土高坡]]》等，还为《[[永不放弃|永不放弃]]》、《[[结婚十年|结婚十年]]》、《[[大汉天子|大汉天子]]》、《姐妹弟兄》等电视剧作曲<ref name=shishi>{{Cite web |url =http://ent.ifeng.com/a/20180823/43092197_0.shtml |title =著名作曲家苏越因病逝世享年63岁 曾创作《黄土高坡》  |author =  |date =2018-08-23  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

2003年起，苏越先后创办多家公司从事影视投资等活动，后因经营不善导致大面积亏空<ref name=xue>{{Cite web |url =http://news.163.com/11/1111/09/7IIP81BQ00014AED.html  |title = 《血染的风采》作曲者苏越诈骗5700多万元被判无期(图) |author =  |date =  2011-11-11|publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。2008年11月，一公司股东向法院提起民事诉讼，要求苏越偿还所借600万元款项。后法院追加安雯为共同被告，判决苏越、安雯支付该借款。2011年12月，法院拍卖安雯的别墅以偿还苏越所借钱款<ref name=zhong>{{Cite web |url =http://news.sohu.com/20130706/n380856527.shtml  |title = 苏越终审获刑15年 女友嘱咐称“保重身体” |author =  |date = 2013-07-06 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

2007年12月至2008年10月间，苏越虚构承办[[2008年北京奥运会|北京奥运会]]巡回演出等活动的演出合同书，并承诺给投资方高额回报，骗取被害人的投资资金<ref name=xue/>。2010年3月24日，苏越因涉嫌合同诈骗罪被刑事拘留，后被逮捕。检方指控，苏越以投资迎[[2008年北京奥运会|奥运]]巡演、筹借迎奥运巡演资金为由，先后同3名受害人签订多份合同，诈骗金额5746万余元人民币<ref name=shishi/><ref name=zhong/>。案发后苏越虽偿还部分钱款，但截至2011年11月一审宣判前，仍有2843.9万元尚未归还<ref name=xue/>。2011年11月10日，苏越一审被[[北京市第二中级人民法院|北京市第二中级人民法院]]判处[[无期徒刑|无期徒刑]]，[[剥夺政治权利|剥夺政治权利]]终身，并处没收个人全部财产，苏越不服判决提出上诉。2012年5月31日，[[北京市高级人民法院|北京市高级人民法院]]裁定，原审判决“事实不清、证据不足”，发回重审。2012年10月12日，该案重审，苏越被北京市第二中级人民法院改判[[有期徒刑|有期徒刑]]15年，并处罚金3万元。2013年3月，[[北京市人民检察院第二分院|北京市人民检察院第二分院]]提起抗诉，认为该案量刑畸轻，建议改判无期徒刑。2013年7月5日，北京市高级人民法院受理抗诉，终审裁定驳回检方抗诉，维持原判<ref name=shishi/><ref name=zhong/><ref>{{Cite web |url =http://news.cntv.cn/law/20121017/100232.shtml  |title = 苏越首度回应诈骗案被改判 称有利于未来创作 |author =  |date = 2012-10-17 |publisher =央视网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = https://www.chinacourt.org/article/detail/2013/07/id/1022165.shtml |title = 苏越终审获刑15年 安雯称“这是最好的结果” |author =  |date = 2013-07-06  |publisher =中国法院网  |language =  |accessdate =  |quote =  }}</ref>。

苏越在看守所内坚持写歌，并教看守所内的未成年人唱歌<ref name=shishi/>。后在[[北京市延庆监狱|北京市延庆监狱]]服刑<ref name=shaowu/>。

2018年8月22日，苏越因患[[胆囊癌|胆囊癌]]病逝，享年63岁<ref name=shishi/>。

==家庭==
*女友：[[安雯|安雯]]，演员，曾在87版电视剧《[[红楼梦_(1987年电视剧)|红楼梦]]》里饰演[[晴雯|晴雯]]。1991年和苏越举办西式冷餐婚礼。十年后的2001年再度举办婚礼，并约好今后每十年都办一次风格不同的婚礼。苏越曾在法庭上表示与安雯已离婚，但安雯否认，称与苏越从无任何法律上有效的婚姻关系。2018年8月17日，安雯还在其个人[[新浪微博|新浪微博]]上表示“活到今天还没见过‘[[结婚证|结婚证]]’……真的很想去领一回结婚证！”苏越被捕后，安雯曾“卖房救夫”，并且复出赚钱帮苏越还债<ref name=shishi/>。
