'''苏享茂'''（{{bd|1980年|1月21日|2017年|9月7日}}）是一名中国软件开发者和企业家，毕业于[[北京邮电大学|北京邮电大学]]，曾就职于[[百度|百度]][[高級工程師|高級工程師]]。2017年9月7日，苏享茂在北京市海淀区西二旗<ref name=":0" />的家中跳楼自杀。据苏享茂和亲友描述，因苏享茂被前妻翟欣欣的巨额离婚索求致使苏公司资金链断裂，最终导致苏享茂自杀<ref name="钱江晚报20170912">{{Cite web|url=http://qjwb.zjol.com.cn/html/2017-09/12/content_3569875.htm?div=-1|title=程序员之死，婚恋网站难辞其咎|accessdate=2017-09-12|author=李晓鹏|date=<!--原文發表日期-->|format=<!--格式-->|publisher=钱江晚报<!--出版者-->|archiveurl=https://web.archive.org/web/20170912022733/http://qjwb.zjol.com.cn/html/2017-09/12/content_3569875.htm?div=-1<!--存檔連結-->|archivedate=<!--存檔日期--> 2017-09-12|quote=}}</ref>。此事件也引发了网民和媒体对于[[世纪佳缘|世纪佳缘]]等婚恋、[[交友网站|交友网站]]的质疑<ref>{{Cite web|url=https://www.ithome.com/html/it/325560.htm|title=WePhone创始人之死疑点解析：前妻、世纪佳缘、苏享茂本人|accessdate=2017-09-19|author=王茜|date=2017-09-13|publisher=IT之家}}</ref>。

== 背景 ==
苏享茂（1980年1月21日－2017年9月7日），出生于[[福建省|福建]]农村，毕业于[[北京邮电大学|北京邮电大学]]，哥哥[[蘇享龍|蘇享龍]]<ref>[https://tw.news.appledaily.com/international/realtime/20170912/1201903/ 工程師被騙婚自殺　家人嘆：他相信美女會愛他]</ref>；曾在[[百度|百度]]做过两三年工程师。离职后，苏享茂独自开发了以提供通讯服务为主的[[手機應用程式|手机应用程序]]——[[WePhone|WePhone]]。<ref name=":0">{{Cite web|url=http://www.zxiazai.com/htm/20170913721573.html|title=程序员苏享茂的最后94天：沉默码农和“白富美”的致命交集|accessdate=2017-09-19|author=|date=2017-09-13|publisher=头条聚焦}}</ref>2017年3月，苏享茂与在婚恋交友网站世纪佳缘上结识了前妻[[翟欣欣|翟欣欣]]。6月，两人完成婚姻登记手续，一个月后签订离婚协议。<ref name="钱江晚报20170912"/><ref name=":0" />

其妻子翟欣欣1986年11月13日出生於[[山東省|山東省]][[泰安市|泰安市]]<ref>{{Cite web|url=http://opinion.people.com.cn/n1/2017/0912/c1003-29529073.html|title=钱江晚报：程序员之死，婚恋网站难辞其咎--观点--人民网|accessdate=2017-09-13|last=103921|work=opinion.people.com.cn}}</ref><ref name=ifeng>{{Cite web|url=http://tech.ifeng.com/a/20170911/44679039_0.shtml|title=WePhone开发者自杀案 在翟欣欣身上的各种消费清单明细|accessdate=2017-09-13|work=tech.ifeng.com}}</ref>，她的父母[[翟所业|翟所业]]、[[刘克勤|刘克勤]]和舅舅[[刘克俭|刘克俭]]，都是高校教授<ref>{{Cite web|url=http://news.sina.com.cn/o/2017-09-12/doc-ifyktzim9967192.shtml|title=“程序员自杀事件”前妻舅舅系公安大学教师 发声明称与外甥女少有来往|accessdate=2017-09-13|work=news.sina.com.cn}}</ref><ref>{{Cite web|url=http://news.sina.com.cn/o/2017-09-13/doc-ifykyfwq6961864.shtml|title=相貌平平的他却被美女“一见钟情”！“逼死”IT丈夫的她，在同学眼中曾是“白富美学霸”…|accessdate=2017-09-13|work=news.sina.com.cn}}</ref>。2005年考入[[山東科技大學|山東科技大學]]土木建築工程學院<ref name=ifeng/>；2009年9月獲工學學士學位；2009年9月考入[[北京交通大學|北京交通大學]]土木建築工程學院攻讀結構工程碩士研究生<ref name=ifeng/><ref name=sina>{{Cite web|url=http://news.sina.com.cn/s/wh/2017-09-13/doc-ifyktzim9977775.shtml|title=程序员之死:1米7的美女硕士真会爱上1米6的他吗|accessdate=2017-09-13|work=news.sina.com.cn}}</ref>；2011年5月至2012年4月曾工作於[[北京房地产科学与技术研究所|北京房地产科学与技术研究所]]<ref name=sina/>；

据媒体报道，從第二任丈夫<ref>[https://tw.news.appledaily.com/international/realtime/20170917/1205633 自殺工程師嫩前妻遭起底　閃結閃離4次上億身家]</ref>[[蘇享茂|蘇享茂]]遺書披露的時間看，蘇享茂與翟欣欣二人2017年3月30日在世纪佳缘上结识見面，4月1日女方展示別墅和[[特斯拉_(公司)|特斯拉]]照片，說願意為蘇享茂生寶寶，年內結婚；2017年6月7日結婚，7月18日離婚，婚姻僅僅維持了42天。蘇享茂结婚前后為翟欣欣購買房與車等物品共計支出1100至1300多萬<ref>{{Cite web|url=http://tech.sina.com.cn/i/2017-09-13/doc-ifykuffc5524616.shtml|title=创业者苏享茂“被逼自杀”疑团待解：寻求律师介入|accessdate=2017-09-13|work=tech.sina.com.cn}}</ref>。

== 自杀 ==
2017年9月7号凌晨，苏享茂在自己研发的产品[[WePhone|WePhone]]推送了一条“公司法人被毒妻翟某害死，WePhone即将停止运营”的消息后，从北京家中跳楼自杀<ref name=":0" />。他在“遗书”中称自己不堪女方威胁并索要1000万元和房产，而且女方在恋爱时隐瞒了自己的婚史<ref>{{Cite web|url=https://www.sohu.com/a/192839987_115362|title=程序员自杀案最新进展，女方律师发声：给你们一天时间，删除侮辱诽谤信息|accessdate=2017-09-19|author=|date=2017-09-18|publisher=搜狐}}</ref>。

== 司法进度 ==
[[王宝强|王宝强]]离婚案代理律师[[张起淮|张起淮]]于9月15日同苏亨茂家属签约接手苏案。9月18日，代理翟欣欣方的北京市盈科律师事务所律师'''易胜华'''微博发表声明，将追究进行“人肉搜索”的网友责任。10月10日，媒体从易胜华处获悉，事发1个多月后，仍没有收到法院立案的消息<ref>{{cite web|author1=周晶晶|title=苏享茂自杀一月，翟欣欣律师：未立案 让事情冷下来|url=http://tech.sina.com.cn/i/2017-10-11/doc-ifymrcmn0237547.shtml|website=新浪综合|publisher=AI财经社|accessdate=2017-11-30}}</ref>。
