'''苏丹马赫穆德·巴达鲁丁二世国际机场'''（{{lang-id|Bandar Udara Internasional Sultan Mahmud Badarudin II}}，{{airport codes|PLM|WIPP}}）是[[印度尼西亚|印度尼西亚]][[南苏门答腊省|南苏门答腊省]][[巨港|巨港]]的一座国际机场，坐落在{{link-id|苏卡拉美区 (巨港)|Sukarame, Palembang|苏卡拉美区}}境内。机场得名于末任{{le|巨港苏丹国|Palembang Sultanate|巨港苏丹}}{{le|马赫穆德·巴达鲁丁二世|Mahmud Badaruddin II}}。

==历史==
1920年，{{link-nl|塔朗贝图图|Talang Betutu}}机场开始建设，作为[[福克公司|福克]]飞机从欧洲飞来的一个降落点<ref name="Eko Nurhuda">{{cite web |url=https://bungeko.com/2016/05/11/bandara-talang-betutu-riwayatmu-kini/ |title=Bandara Talang Betutu Riwayatmu Kini |author=Eko Nurhuda |date=2016-05-11 |website=bungeko.com |publisher= |access-date=2018-11-02 |quote=}}</ref>，这是一座配备无线测向设备和其他地面基础设施的{{le|海关机场|Customs airport}}<ref name="Palembang Customs Aerodrome">[title="Jane's All The World's Aircraft |publisher=Sampson Low 1938"]</ref>。[[第二次世界大战|第二次世界大战]]时期，机场被盟军称呼为“巨港P1”，或简称P1。1942年至1943年间，日本军队重建巨港机场。到1963年，塔朗贝图图机场成为一座军民共用的机场。1975年8月21日改为民航机场。到80年代初，机场是雅加达飞往[[棉兰|棉兰]]和[[新加坡|新加坡]]的一个中转点<ref name="Eko Nurhuda"/>。1985年4月3日，所有民航飞行活动搬迁至现在的苏丹马赫穆德·巴达鲁丁二世机场，塔朗贝图图机场停用<ref name="Eko Nurhuda"/>。1991年4月1日移交[[印尼机场公司|印尼第二机场公司]]管理。

2003年9月18日，[[南苏门答腊省|苏南省]]为了竞争{{le|2004年国家体育周|2004 Pekan Olahraga Nasional}}的东道主，开始对苏丹马赫穆德·巴达鲁丁二世机场进行升格改造，机场升级为国际机场。整个升级改造耗资3,667亿印尼盾，从[[日本|日本]][[国际协力银行|国际协力银行]]贷款2,519亿印尼盾，国家预算资金出资1,148亿印尼盾。机场跑道在升级工程中延长了300米，跑道长宽达到3,000米 x 60米。建设停车场面积20,000平方米，可以容纳1,000辆车停放。新航站楼也在2005年9月27日启用，建筑共有三层，面积13,000平方米，可以容纳1,250名旅客。升级改造完成后的机场可以为[[空中客车A330|空客A330]]、[[波音747|波音747]]、[[波音777|波音777]]等[[宽体飞机|宽体飞机]]提供服务。

为了服务[[2018年亚洲运动会|2018年亚洲运动会]]，机场于2016年开始了第二次升级改造<ref>{{cite news|url = http://www.thejakartapost.com/travel/2016/10/14/palembang-airport-to-be-revamped-for-2018-asian-games.html|title = Palembang airport to be revamped for 2018 Asian Games|newspaper = The Jakarta Post|accessdate = 2017-11-24|deadurl = no|archiveurl = https://web.archive.org/web/20170823170025/http://www.thejakartapost.com/travel/2016/10/14/palembang-airport-to-be-revamped-for-2018-asian-games.html|archivedate = 2017-08-23}}</ref>。停机坪扩建到可以容纳19架飞机<ref>{{cite web|url = https://www.merdeka.com/uang/perluas-bandara-palembang-ap-ii-siapkan-rp-130-miliar.html|title  = Perluas Bandara Palembang, AP II siapkan Rp 130 miliar|accessdate = 2017-11-24|deadurl = no|archiveurl = https://web.archive.org/web/20171201040825/https://www.merdeka.com/uang/perluas-bandara-palembang-ap-ii-siapkan-rp-130-miliar.html|archivedate = 2017-12-01}}</ref>。航站楼也从既有的34,000平方米扩建到115,000平方米，年客运承载量达到900万人次<ref>{{cite web|url = http://www.tribunnews.com/regional/2016/04/11/bandara-internasional-smb-ii-palembang-diperluas-hingga-bisa-menampung-19-pesawat|title = Bandara Internasional SMB II Palembang Diperluas hingga Bisa Menampung 19 Pesawat|accessdate = 2017-11-24|deadurl = no|archiveurl = https://web.archive.org/web/20171201043133/http://www.tribunnews.com/regional/2016/04/11/bandara-internasional-smb-ii-palembang-diperluas-hingga-bisa-menampung-19-pesawat|archivedate = 2017-12-01}}</ref>。

==事故==
1975年9月24日，{{le|嘉鲁达印尼航空150号班机|Garuda Indonesia Flight 150}}在降落塔朗贝图图机场时坠毁，事故原因是糟糕的天气和大雾，机上共61人，事故死亡25人，包括一名地面人员<ref name="ASN">{{cite web| url = http://aviation-safety.net/database/record.php?id=19750924-2| title = Accident description at the ASN| publisher = Aviation Safety Network| deadurl = no| archiveurl = https://web.archive.org/web/20111226200341/http://aviation-safety.net/database/record.php?id=19750924-2| archivedate = 2011-12-26}}</ref>。

==地面交通==
[[File:SMB2IA_LRT_arriving.jpg|thumb]]上的{{le|机场站 (巨港)|Sultan Mahmud Badaruddin II International Airport LRT station|机场站}}]]
为了服务[[2018年亚洲运动会|2018年亚洲运动会]]，政府修建了[[巨港轻轨|巨港轻轨]]连接机场和[[查卡峇灵体育城|查卡峇灵体育城]]<ref>{{cite web| url = http://bisniskeuangan.kompas.com/read/2016/08/18/142350926/pembangunan.lrt.palembang.lebih.cepat.alex.noerdin.minta.maaf.ke.ahok| title = Pembangunan LRT Palembang Lebih Cepat, Alex Noerdin Minta Maaf ke Ahok| date = 2016-08-18| deadurl = no| archiveurl = https://web.archive.org/web/20161013220257/http://bisniskeuangan.kompas.com/read/2016/08/18/142350926/pembangunan.lrt.palembang.lebih.cepat.alex.noerdin.minta.maaf.ke.ahok| archivedate = 2016-10-13}}</ref>。到2018年亚运会开始时，尚有部分轻轨站未开通运营，到2018年10月18日，全线13个车站全部投入运营<ref>{{cite web |url=http://www.tribunnews.com/nasional/2018/10/19/13-stasiun-lrt-resmi-beroperasi-layani-penumpang |title=13 Stasiun LRT Resmi Beroperasi Layani Penumpang |date=2018-10-19}}</ref>。{{le|机场站 (巨港)|Sultan Mahmud Badaruddin II International Airport LRT station|机场轻轨站}}为机场提供服务。轻轨的票价分为两种，不涉及机场站的单程票价为5,000盾，涉及机场站的单程票价则为10,000盾<ref>{{cite news |last1=Luciana |first1=Anisa |title=Kemenhub: Subsidi Tarif Tiket LRT Palembang hingga 2019 |url=https://bisnis.tempo.co/read/1112628/kemenhub-subsidi-tarif-tiket-lrt-palembang-hingga-2019/full |accessdate=2018-08-14 |work=Tempo |date=2018-08-01 |language=id |archive-url=https://web.archive.org/web/20180814170035/https://bisnis.tempo.co/read/1112628/kemenhub-subsidi-tarif-tiket-lrt-palembang-hingga-2019/full |archive-date=2018-08-14 |dead-url=no }}</ref>。

{{le|巨港-印德拉拉雅收费公路|Jalan Tol Palembang–Indralaya}}正在建设中，该高速公路将连接巨港机场<ref>{{cite web| url = https://en.antaranews.com/news/100337/president-inspects-palembang-indralaya-toll-road-construction-works| title = President inspects Palembang-Indralaya toll road construction works| accessdate = 2017-11-24| deadurl = no| archiveurl = https://web.archive.org/web/20171201042627/https://en.antaranews.com/news/100337/president-inspects-palembang-indralaya-toll-road-construction-works| archivedate = 2017-12-01}}</ref>。

* 第二路段：巴慕鲁丹至兰布坦（KTM S. Rambutan），全长4.90公里，2018年3月投入运营。
