'''蘇聯中央電視台'''（{{lang-ru|Центральное телевидение Гостелерадио СССР}}，简称'''{{lang|ru|ЦТ СССР}}'''）是[[前蘇聯|前蘇聯]]的國家[[電視台|電視台]]，1951年3月在[[莫斯科|莫斯科]]建立。1990年时該台每天播出12套節目（其中包括向5個廣播時區重播的第一套和第二套節目），共148小時。

== 电视频道 ==

* 第一套節目：于1938年开播，是全蘇聯[[新聞|新聞]]、[[政治|政治]]、[[經濟|經濟]]、[[文藝|文藝]]的綜合節目。
* 第二套節目：于1965年开播，是全蘇新聞和文藝節目，每天播出13.5小時。
* 莫斯科節目：于1956年开播，播放对象是莫斯科、[[莫斯科州|莫斯科州]]及其附近居民，每天播出4小时30分。
* 列宁格勒节目：播放对象是[[列宁格勒|列宁格勒]]附近居民。
* 第四套節目：電視教學節目，节目针对知识分子和学生，每天播出3小时30分。电视教学节目分两类，一类配合课堂教学，一类是专题节目。
* 第五套节目：科学教育节目，于1971年开播。

== 在苏联时期的发展 ==
===== 莫斯科电视台和列宁格勒电视台 =====
1931年4月29日，[[苏联|苏联]]首次试验[[电视|电视]]。1938年，[[莫斯科|莫斯科]]电视中心和[[列宁格勒|列宁格勒]]电视中心开始试播电视，1939年定期播出。1945年5月7日的“无线电节”，战后第一次播放电视。12月15日，[[莫斯科|莫斯科]]电视中心恢复工作。
===== 彩色电视 =====
苏联中央电视台第一套和第二套节目于1960年1月14日开始进行实验性彩色电视测试。彩色电视于1967年10月1日开播，采用SECAM制式播出，使苏联成为欧洲第四个开播彩色电视节目的国家，仅次于[[英国|英国]]、[[西德|西德]]和[[法国|法国]]。
===== 卫星电视 =====
1988年，由苏联制造的Gorizont卫星发射升空，为欧洲大部分地区、北非和美洲东部地区提供电视节目。所有东欧社会主义共和国的节目，包括苏联中央电视台的频道，都在卫星上播出。[[苏联|苏联]]解体前全国共有120个电视中心，其中93%可播送彩色电视节目。每昼夜播出3700小时，使用四五十种民族语言，电视经费13亿卢布。原苏联有900多座电视发射台，约4000座电视转播站和近4000座卫星地面站。
===== 戈尔巴乔夫时期 =====
20世纪80年代，随着苏联在[[戈尔巴乔夫|戈尔巴乔夫]]领导下的经济和政治变革，苏联中央电视台发生了重大变化。<br>
起初，苏联中央电视台坚持[[苏联共产党|苏联共产党]]的路线，几乎没有报道反对苏联共产党政权。然而，在1990年苏共统治开始破裂后，苏联中央电视台取消政治宣传并自由报道新闻。[[2×2_(电视频道)|2×2]]等私营电视频道的开播，结束了国家对电视广播的垄断。

== 苏联解体后的发展 ==
1991年12月25日苏联解体后，苏联中央电视台不复存在。1991年12月27日，奥斯坦金诺电视台第一套节目和奥斯坦金诺电视台第四套节目（今[[第一頻道|第一频道]]和[[NTV_(俄羅斯)|NTV]]）接管了第一套节目和第四套节目的频率。列宁格勒电视台[[第五频道_(俄罗斯)|第五频道]]成为全国性电视台。此后奧斯坦基諾電視台第一套全國性節目按[[私有化|私有化]]方向，吸收民間[[資本|資本]]，改為公私合營的「俄羅斯公共電視台」，2002年後改為「[[第一頻道|第一頻道]]」；奧斯坦基諾電視台第二套節目的頻道和設備改由[[俄羅斯廣播電視公司|俄羅斯廣播電視公司]]（RTR）[[俄羅斯電視台|俄羅斯電視台]]使用，對俄羅斯廣播。
