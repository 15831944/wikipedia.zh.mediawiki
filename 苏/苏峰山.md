'''苏峰山'''，古称'''东山'''、'''川陵山'''，位于[[福建省|福建省]][[漳州市|漳州市]][[东山岛|东山岛]]东部，海拔274米，为东山岛最高的山峰<ref>{{cite web |url=http://www.dongshandao.gov.cn/cms/html/dsxrmzf/2018-03-07/585001401.html |title=苏峰山 - 旅游景点 |last= |first= |date=2018-03-07 |website=东山县人民政府 |publisher= |access-date=2018-07-20 |quote=}}</ref>。山上有[[马帝爷庙|马帝爷庙]]、[[苏峰古寺|苏峰古寺]]等景点。

2016年6月17日，苏峰山环岛路建成通车，全长110.6公里<ref>{{cite web |url= http://zz.mnw.cn/dongshan/xw/1232889.html|title=东山环岛路苏峰山段通车 配套有步行道等观光设施 |last= |first= |date=2016-06-20 |website=闽南网 |publisher=闽南日报 |access-date=2018-07-20 |quote=}}</ref>。
