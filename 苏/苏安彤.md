'''苏安彤'''({{lang|en|Suzanne Anton}}，{{bd|1952年|5月31日}})是 [[加拿大人|加拿大]] 政治家和[[卑诗省|卑诗省]]现任司法厅长。 2013年在[[卑诗省议会|卑诗省议会]]选举中当选省议员。 2013年6月10日被任命为[[卑诗省|卑诗省]]司法部长和总检察长。 2017年5月9日的[[卑诗省议会|卑诗省议会]]选举中被[[周烱华|周烱华]]击败。

== 背景 ==
她出生在[[卑诗省|卑诗省]][[邓肯_(不列颠哥伦比亚省)|邓肯]]，1970年毕业于 玛格丽特女王学院 ，在[[维多利亚大学_(加拿大)|维多利亚大学]]获得数学学士学位. 1979年于[[卑诗大学|卑诗大学]]获得法律学士学位。<ref>[http://www.linkedin.com/pub/suzanne-anton/7/857/710 "苏珊安东"], ''Linkedin''，可月29日，2009年。</ref>

== 家庭 ==
苏安彤和她的丈夫奥林有三个孩子：伊丽莎白、罗伯特和安的。
