}}'''苏联民航3739号班机'''<ref name="lib">{{Cite web|url=http://lib.znate.ru/docs/index-168665.html?page=15|title=ИЗ ИСТОРИИ ОТЕЧЕСТВЕННОГО ТЕРРОРИЗМА|accessdate=2013-03-11|year=1999|publisher=Lib.znate.ru|language=ru}}</ref>，是苏联民航一班自[[伊爾庫茨克|伊爾庫茨克]]经停[[库尔干|库尔干]]飞往[[列宁格勒|列宁格勒]]的国内航班。1988年3月8日，执飞该航班的一架[[圖-154|圖-154]]在从库尔干飞往列宁格勒的途中遭到想要逃离苏联的奥韦奇金一家的劫持。劫机者要求机组将飞机飞往伦敦。<ref name="lib"></ref>机械师骗劫机者说他们要在芬兰降落并加油。<ref name="lib"></ref>事实上，机组将飞机降落在了芬兰边境上的一座苏军军事基地。苏联内务部部队随即展开突袭。<ref name="lib"></ref>这起事件共造成4名人质遇难，另有5名劫机者自杀。另外两名劫机者分别被判处8年和6年有期徒刑。一名机组成员在事故后被授予红旗勋章。<ref name="Banner">{{Cite web|url=http://baikalpress.ru/kopeika/2005/34/007001.html|title=Аэропорт Иркутск|accessdate=2013-12-08|publisher=Копейка|language=ru|deadurl=yes|archiveurl=https://web.archive.org/web/20131213105304/http://baikalpress.ru/kopeika/2005/34/007001.html|archivedate=2013-12-13}}</ref>

== 背景 ==
事发时，奥韦奇金一家共有13名成员：母亲妮娜和她的十二个孩子（7个儿子，4个女儿）。其中，她的一个女儿柳德米拉已经结婚并居住在另外一座城市，那个女儿没有参与劫持。<ref name="lib"></ref>在生下第十个孩子后，苏联政府授予她英雄母亲的称号。<ref>{{Cite web|url=http://www.time.com/time/magazine/article/0,9171,967050,00.html|title=Terrorism Bloody Band|accessdate=2013-03-11|date=1988-03-21|publisher=''[[Time_(magazine)|Time]]''}}</ref>在她的丈夫于1984年去世后，她就在伊尔库茨克独自抚养她的孩子。她的儿子们在当地组建了一个叫Seven Simeons的乐队。在这个乐队的成员游历日本后，他们就决定离开苏联定居海外。不过在那个时候，对于土生土长的苏联公民，特别是开乐队的来说，移居海外几乎是不可能的。奥韦奇金一家于是决定通过劫机逃往国外。临行前，他们留下纸条说他们要去看亲戚，接着，他们就登上了从伊尔库茨克飞往列宁格勒的航班。<ref>{{Cite web|url=http://www.ruwings.ru/safety/180|title=Чрезвычайное происшествие Ту-154Б CCCP-85413 Аэрофлот МГА СССР 08.03.1988|accessdate=2013-03-11|publisher=Ruwings.ru|language=ru}}</ref>

== 事件经过 ==
奥韦奇金一家在行动前准备了两支[[削短型霰彈槍|削短型霰彈槍]]。<ref name="lib"></ref>为了防止遭到逮捕，他们还准备了爆炸物，如果行动失败，他们就会引爆炸弹。他们把爆炸物和枪支藏在了一把低音提琴里，由于低音提琴体积过大，安检人员无法进行扫描。<ref name="lib"></ref>为了确定安检人员可能的反应，他们还进行过预演。<ref name="lib"></ref>在预演中，奥韦奇金一家执意要把低音提琴放到客舱里，而安保人员仅仅是看了几眼就放行了。<ref name="lib"></ref>

当飞机已经飞抵[[沃洛格达|沃洛格达]]上空，正在准备在列宁格勒降落时。一名乘务员告知了机组劫机者在纸条上写下的信息，其内容如下："飞往英国（伦敦），不要降落，不然我们会炸毁飞机" （这个纸条随后被烧毁在客舱内）。<ref name="lib"></ref>机长随后发送紧急信号并将此事告诉了[[沃洛格达|沃洛格达]]空管。<ref name="lib"/>布雷维克一名乘务员告诉乘客们说，他们要在芬兰[[科特卡|科特卡]]着陆。不过，地面实际上要求机组在苏联空军基地Veshchevo降落。<ref name="lib"></ref><ref name="rg">{{cite web | url =http://www.rg.ru/2011/05/24/ovechkiny.html| title =Реквием по джаз-банде|language=ru |author=Золик Мильман|publisher =''[[Rossiyskaya_Gazeta|Rossiyskaya Gazeta]]''|date=2011-05-24| accessdate = 2013-03-11}}</ref>[[飛航工程師|飞航工程师]]先前曾对劫机者说，他们需要在芬兰加油，不然飞机飞不到伦敦。

降落前不久，劫机者才意识道他们依然在苏联领土上。这时身为劫机者之一的德米特里·奥韦奇金杀害了一名空中乘务员。在飞机降落后，五名穿着防弹背心的事故反应小组成员突袭了驾驶舱。据目击者说，他们随后对着客舱无差别开火。另一个小组突击了飞机尾部。这是一名劫机者通过对讲机向机组大喊：“指挥官，让你的人别开火”<ref name="lib"></ref>交火途中，一名劫机者引爆了炸弹并死亡。爆炸造成机尾起火，不过火后来被机组扑灭了。<ref name="lib"/>妮娜随后命令她的孩子们向他开火。<ref name="lib"></ref><ref name="rg"></ref>其他四名劫机者也饮弹自尽。（瓦西里，26，德米特里，24，欧列格，21，亚历山大，19）。<ref name="rg"></ref>其他六名奥韦奇金一家的成员在交火中生还。（奥尔加, 28，伊戈尔，17，塔蒂阿娜，14，米哈伊尔，13，尤金娜，10和谢尔盖，9）。<ref name="lib"></ref>

在人质中，有一名空中乘务员和三名乘客遇难。乘务员是被劫机者杀死的，乘客则是在交火中被误杀。<ref name="lib"></ref>另有大概20名乘客受伤（又说36人）<ref name="rg"></ref>，其中14人重伤。<ref name="lib"></ref>

== 后续 ==
两名年龄较大的奥韦奇金家族成员，伊戈尔和奥尔佳在1988年9月6日被捕并分别被判处6年和8年有期徒刑。<ref name="rg"></ref>奥尔佳在狱中生下了女儿拉尔萨，不过她在2003年7月8日被她的男朋友殴打致死。<ref>{{Cite web|url=http://pressa.irk.ru/friday/2004/47/010001.html|title=Осуждён убийца Ольги Овечкиной|accessdate=2013-03-11|author=Сергей Гончаров|date=2004-11-26|publisher=''Пятница''|language=ru|deadurl=yes|archiveurl=https://web.archive.org/web/20130311121349/http://pressa.irk.ru/friday/2004/47/010001.html|archivedate=2013-03-11}}</ref>

试图安抚劫机者，最后却遭到杀害的28岁的乘务员Tamara Zharkaya事后被追授紅旗勳章。<ref name="Banner"></ref>
