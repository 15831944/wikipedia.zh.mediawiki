'''苏荣'''（{{bd|1970年|12月||}}）[[山西|山西]][[忻州|忻州]]人，[[中国人民解放军少将|中国人民解放军少将]]<ref name=duihua/>。

==生平==
1987年，苏荣考上一所师范学院，但他不顾家人反对参军，成为一名装甲兵战士。入伍后，历任排长、连长、股长、营长、团长。后升任[[北京军区|北京军区]]某装甲师参谋长，任内他率领的蓝军部队是[[中国人民解放军陆军|中国人民解放军陆军]]首支专业信息化蓝军部队。截至2009年，苏荣先后立三等功、二等功各一次，连续4年被集团军评为优等指挥员<ref name=duihua>{{Cite web |url = http://mil.sohu.com/20090924/n266972561.shtml|title = 《防务新观察》特别节目--对话蓝军司令苏荣 |author =  |date = 2009-09-24 |publisher =  搜狐|language =  |accessdate =  |quote =  }}</ref>。此后历任[[中国人民解放军陆军第三十八集团军|中国人民解放军陆军第三十八集团军]]某装甲师师长、中国人民解放军陆军第三十八集团军副军长。2017年任新组建的[[中国人民解放军陆军第八十三集团军|中国人民解放军陆军第八十三集团军]]副军长。2017年7月30日，在[[庆祝中国人民解放军建军90周年阅兵|庆祝中国人民解放军建军90周年阅兵]]中，刚组建的陆军空中突击旅首次亮相，陆军第八十三集团军副军长苏荣担任空中突击梯队副指挥员<ref name=qilingh>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1749062 |title = “70后”少将苏荣履新第83集团军副军长 |author =  |date = 2017-08-01 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

2016年7月晋升少将军衔<ref name=qilingh/>。
