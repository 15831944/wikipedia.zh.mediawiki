'''佛山市顺德区苏氏荣华食品有限公司'''，简称'''苏氏荣华食品'''、'''苏氏荣华'''和'''顺德荣华'''，是一家位于[[中华人民共和国|中华人民共和国]][[广东省|广东省]][[佛山市|佛山市]][[顺德区|顺德区]]的食品公司，于1983年成立，曾先后使用过“荣华饼食店”、“荣华面包厂”等名称。该公司主要生产[[中式糕点|中式饼食]]和[[月饼|月饼]]，主要在中国大陆20多个省、市、自治区销售<ref name=sc168>{{Cite web |url =http://www.sc168.com/shenghuo/content/2010-08/31/content_165765.htm|title =  逢盛世享荣华 专心专业28年 |author = 黄国育 |date = 2010-08-31 |publisher =顺德新闻网—珠江商报  |language =  |accessdate =  |quote =  }}</ref>。

==历史与概况==
1983年，苏国荣（创始人）离开了工作了5年属于[[供销社|供销社]]系统的顺德勒流镇饼厂（当时已经是师傅身份），下海创立了自己的食品店，生产[[月饼|月饼]]等各类食品。1985年开始使用“荣华”字号，正式更名为荣华饼食店。当时苏国荣花了800元创业，没有自动化设备，整个工厂运作全靠自己砌砖、烧柴，直到后来的烧煤<ref name=fzrb/><ref name=sohu>{{Cite web |url =http://business.sohu.com/20120927/n354019112.shtml|title =  港粤“荣华”14年拉锯战 谁的“荣华”？|author =王媛  |date = 2012-09-27|publisher =时代周报   |language =  |accessdate =  |quote =  }}</ref><ref name=sina>{{Cite web |url =http://finance.sina.com.cn/chanjing/gsnews/20120409/145411777500.shtml|title =  14年粤港荣华月饼商标案最后对决 最高法近期再审|author = 吕卫红|date = 2012-04-09|publisher =正义网|language =  |accessdate =  |quote =  }}</ref>。创立初期，荣华饼食店的经营范围是“早点、油器糕点”，其经营方式一直到1996年也只有“零售”。1996年1月17日，苏国荣创立了“荣华面包厂”，经营方式为“产销”，由此苏国荣开始具备生产的经营资质<ref name=qq>{{Cite web |url =http://finance.qq.com/a/20120920/001701.htm|title =  不知名，无战事一场月饼商标的对决|author = 王祥明 |date = 2012-09-20|publisher =南方报业网-南方日报  |language =  |accessdate =  |quote =  }}</ref>。1998年6月18日，该厂更名为“苏氏荣华食品厂”<ref name=ipc>{{Cite web|url = http://ipr.court.gov.cn/zgrmfy/sbq/201310/t20131008_160268.html|title = 苏国荣与荣华饼家有限公司、东莞荣华饼家有限公司、广州市好又多（天利）百货商业有限公司世博分公司、广州市好又多（天利）百货商业有限公司、中山市今明食品有限公司侵犯商标权及不正当竞争纠纷再审民事判决|author = |date = 2013-10-08|publisher = 中华人民共和国最高人民法院|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180420071355/http://ipr.court.gov.cn/zgrmfy/sbq/201310/t20131008_160268.html|archive-date = 2018-04-20|dead-url = yes}}</ref>。此后，苏国荣又与自己在1998年在香港设立的“香港苏氏食品有限公司”“合资”，将该公司的名称更名为佛山市顺德区勒流苏氏荣华食品有限公司<ref name=sina/>。然而“香港苏氏食品有限公司”是一家[[空壳公司|空壳公司]]，苏氏荣华在产品上还打上“香港监制”字样，其目的是故意误导消费者<ref name=sina2>{{Cite web |url =http://finance.sina.com.cn/chanjing/gsnews/20120921/015813196705.shtml|title = 荣华月饼商标案判决后纠纷愈加升级 |author =叶碧华 |date =2012-09-21|publisher =21世纪经济报道 |language =  |accessdate =  |quote =  }}</ref>。

1990年，荣华饼食店的月饼在当地已经很有名气。由于孩子在当年出生，他给孩子冠名“华”字，再加上自己名字中的“荣”字，来纪念公司的发展<ref name=fzrb>{{Cite web |url =http://www.legaldaily.com.cn/zmbm/content/2011-08/23/content_2896377.htm?node=7574|title =荣华月饼商标权之战13年未了|author = |date =2011-08-23|publisher =法制日报|language =  |accessdate =  |quote =  }}</ref>。

苏氏荣华工厂占地面积两万多平方米，公司已通过[[ISO9001|ISO9001]]质量体系认证和[[HACCP|HACCP]]食品安全体系认证。公司主要产品为白莲蓉月饼和龙凤嫁娶喜饼，其产品在中国大陆20多个省、市、自治区销售。2010年起，产品可通过[[中国邮政|中国邮政]]广东省各网点购买，并可寄往全国乃至世界各地。该公司所生产的七星伴月、经典金月、清香蛋黄白莲蓉、优品伍仁皇、清香双黄白莲蓉月饼曾获得“广东精品月饼”称号，而双黄白莲蓉和伍仁皇月饼也曾获中国食品工业协会、全国烘焙产业联盟颁发的“首届中国烘焙食品节金奖”称号<ref name=sc168/>。

==与香港荣华的纠纷==
{{main|荣华品牌之争}}
由于苏氏荣华公司所生产的月饼一样使用“荣华月饼”名称，且无论是商标、包装盒都与[[荣华|香港荣华]]相似，此举在1999年激怒了香港荣华和东莞荣华（香港荣华在东莞设立的分公司），导致香港荣华与苏氏荣华陷入了无休止的商标权诉讼中。据统计，双方之间的相关诉讼多达58起，相关案件的诉讼费超过1000万元<ref name=15years>{{Cite web |url =http://finance.sina.com.cn/chanjing/gsnews/20130730/113616286368.shtml|title =粤港荣华品牌之争持续15年：仍有六场官司要打|author =|date =2013-07-30|publisher =中新网|language =  |accessdate =  |quote =  }}</ref><ref name=people>{{Cite web |url =http://ip.people.com.cn/n/2013/0129/c136655-20356956.html|title = 谁的月饼?"香港荣华"VS"顺德荣华"15载争名又开战 |author =周伟良 |date =2013-01-29|publisher =广州日报 |language =  |accessdate =  |quote =  }}</ref>。
