'''苏丹塔哈机场'''（{{lang-id|Bandar Udara Sultan Thaha}}，{{airport codes|DJB|WIJJ}}）是[[印度尼西亚|印度尼西亚]][[占碑省|占碑省]][[占碑|占碑]]的一座机场，位于占碑近郊的{{le|帕尔梅拉 (印尼)|Paalmerah|帕尔梅拉镇}}。机场名称来源于末任{{le|占碑苏丹国|Sultanate of Jambi|占碑苏丹}}{{le|塔哈·赛义夫丁|Thaha Syaifuddin}}。

==历史==
荷兰殖民时代，机场名为'''帕尔梅拉机场'''。1950年代印尼独立后，占碑机场开始应用于民航领域，那时的机场跑道长宽为900m x 25m，砾石道面，可起降的最大机型为[[道格拉斯DC-3|道格拉斯DC-3]]。1976年，机场跑道扩建到1,650m x 30m。1978年更名苏丹塔哈机场。1991年跑道延长到1,800米，2000年再次延长至2,000米。2007年1月1日，机场由交通部移交[[印尼机场公司|印尼第二机场公司]]管理，此时机场已可起降[[空中客车A320|空客A320]]机型。2012年，机场跑道延长到2,400米长、45米宽。<ref>{{cite web |url=https://en.antaranews.com/news/74382/jambi-airport-needs-expansion |title=Jambi airport needs expansion |author=Ella Syafputri |date=2011-08-02 |website=en.antaranews.com |publisher=antaranews |access-date=2018-11-01 |quote=}}</ref>

==设施==
[[File:Jambiairport2.jpg|thumb]]

2011年12月12日，航站楼一期扩建工程奠基，航站楼的年旅客容量将达到150万人次。2011年的客运量已经达到100万人次，而航站楼的最大承载容量仅为30万人次。航站楼将从2,308平方米扩建至13,015平方米<ref>{{cite web |url=http://koran-jakarta.com/index.php/detail/view01/78235 |title=Proyek Penerbangan  AP II Bangun Terminal Baru Bandara Sultan Thaha |date=13 December 2011}}</ref>。新的航站楼在2015年9月27日揭牌启用<ref>{{cite web |url=http://finance.detik.com/read/2015/12/27/092608/3104791/4/terminal-baru-bandara-sultan-thaha-resmi-beroperasi-hari-ini |title=Terminal Baru Bandara Sultan Thaha Resmi Beroperasi Hari Ini |author=<!--Staff writer(s); no by-line.--> |date=2015-12-27 |website=finance.detik.com |publisher= |access-date= |quote=}}</ref>。2016年7月21日启动航站楼二期扩建工程，航站楼面积将扩建到35,000平方米。目前的机场每天可起降23对航班，未来将增加到每天起降35对<ref>{{cite web |url=http://www.republika.co.id/berita/nasional/umum/16/07/21/oan9xm335-jokowi-resmikan-bandara-sultan-thaha-jambi |title=Jokowi Resmikan Bandara Sultan Thaha Jambi |author=Esthi Maharani |date=July 21, 2016}}</ref>。航站楼和跑道的扩建工程造价达到1,260亿印尼盾<ref>{{cite web |url=https://nasional.tempo.co/read/news/2016/07/21/058789375/jokowi-resmikan-bandara-sultan-thaha-jambi |title=Jokowi Resmikan Bandara Sultan Thaha Jambi |author=<!--Staff writer(s); no by-line.--> |date=2016-07-21 |website=www.tempo.co |publisher=tempoco |access-date= |quote=}}</ref>

苏丹塔哈机场在2015年成为世界上第一个“动物园机场”，塔曼林巴动物园（Taman Rimba Zoo）距离机场的新航站楼仅900米远<ref>{{cite web |url=http://jambi.tribunnews.com/2014/04/29/zoo-airport-rampung-2015-ini-desainnya |title=Zoo Airport Rampung 2015, Ini Desainnya |date=April 29, 2014}}</ref>。

==事故和事件==
*2008年8月27日，[[三佛齐航空|三佛齐航空]]62号班机一架[[波音737#737-200|波音737-200]]在占碑机场冲出跑道，事故中11人受伤，无人死亡，伤者包括被飞机冲撞到的一名农夫和他的家人。
