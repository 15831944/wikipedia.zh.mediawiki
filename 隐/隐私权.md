'''隱私權'''，指個人[[人格|人格]]上的利益不受不法僭用或侵害，個人與大眾無合法關聯的私事，亦不得妄予發布公開，而其私人活動，不得以可能造成一般人的精神痛苦或感覺羞辱之方式非法侵入的權利。是为众多[[法律|法律]]系统所支持的一种[[人权|人身基本权利]]。由于它的存在，[[政府|政府]]和[[非政府组织|民间团体]]的某些活动受到一定的限制。

== 《世界人权宣言》 ==

隐私权在[[联合国|联合国]]颁布的《[[世界人权宣言|世界人权宣言]]》中的第十二条<ref>http://www.humanrights-china.org/china/rqfg/R120011029133820.htm</ref>中明确定义：

{{Cquote|任何人的私生活、家庭、住宅和通信不得任意干涉，他的荣誉和名誉不得加以攻击。 人人有权享受法律保护，以免受这种干涉或攻击。}}

== 侵害私隱行為 ==
一般而言，侵害私隱的行為包括：
* 侵擾被害人之幽居寧靜或秘密
* 公開披露使被害者感覺困窘的事
* 發布資料使大眾對受害人產生錯誤的印象
* 未被告知自己的利益而僭用被害人之姓名與肖像

== 新闻界 ==

大多数人，特别是[[大众媒体|媒体]]关注的人，认为他们的个人生活被媒体报道是一种侵犯隐私权的行为。然而新闻业人士则认为大众有权知道处于[[名人|公众人物]]状态的那些人的个人信息。这种区别被蕴含于大多数司法习惯中。

== 各地的隱私權法制現況 ==
==={{USA}}===
{{main|隱私權 (美國)}}
Warren和[[路易斯·布蘭迪斯|路易斯·布蘭迪斯]]在1890年，於《[[哈佛法學評論|哈佛法學評論]]》提出的「隱私權」（The Right to Privacy）論文，開啟了後世對於隱私權的討論。在這篇文章中，Warren與Brandeis從[[侵權行為|侵權行為]]法的角度，建立了隱私權的存在依據，並主張「不受干擾的權利」（right to be let alone）。文章中的許多論點對於後來的[[實務|實務]]見解和[[學說|學說]]，至今仍有很大的影響。

William L. Prosser教授在《[[加州法學評論|加州法學評論]]》上發表的「隱私」一文，提出四種侵害類型：不合理的干擾私人領域、公開真實的私人事實、使用真實的訊息，造成錯誤的印象、未經授權盜用個人的名稱或肖像。

法院判例证实，公共电话亭里的谈话、家中持有物品被隐私权保护，而丢弃的垃圾不属隐私。
