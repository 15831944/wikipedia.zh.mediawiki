{{作品名稱|name=''Brightburn''|hk=魔童|tw=靈異乍現|cn=亮光}}是一部2019年美國[[超級英雄電影|超級英雄]][[科幻片|科幻]][[恐怖片|恐怖片]]，由大衛·亞羅維斯基（{{lang|en|David Yarovesky}}）執導，{{link-en|馬克·岡恩|Mark Gunn (screenwriter)}}和{{link-en|布萊恩·岡恩|Brian Gunn}}撰寫劇本，而[[詹姆斯·岡恩|詹姆斯·岡恩]]與{{link-en|黃健華|Kenneth Huang}}則擔任監製。其主演包括[[伊莉莎白·班克絲|伊莉莎白·班克絲]]、[[大衛·丹曼|大衛·丹曼]]、傑克森·A·鄧恩（{{lang|en|Jackson A. Dunn}}）、{{link-en|麥特·瓊斯 (演員)|Matt Jones (actor)|麥特·瓊斯}}及{{link-en|梅洛狄斯·漢格|Meredith Hagner}}。

《靈異乍現》定於2019年5月24日在美國上映。

== 劇情 ==
故事敘述一個神秘的孩子從異世界墜落在地球上，但他卻沒有成為人類的英雄，而是在被一對夫婦收養的過程中，使他體內的邪惡逐漸的成長並對外釋放<ref name="BrightburnSiteOfficial">{{cite web |url=https://www.brightburn.movie/ |title=Brightburn |accessdate=2019-01-19 |work=[[Sony_Pictures|Sony Pictures]]}}</ref><ref name="CinemaBlend">{{cite web |url=https://www.cinemablend.com/previews/2464994/brightburn |title=Brightburn |accessdate=2019-01-19 |work=Cinema Blend}}</ref>。

== 製作 ==
2017年12月，據報導，[[詹姆斯·岡恩|詹姆斯·岡恩]]將監製一部未知標題的恐怖片，他的弟弟{{link-en|布萊恩·岡恩|Brian Gunn|布萊恩}}和堂弟{{link-en|馬克·岡恩|Mark Gunn (screenwriter)|馬克}}負責撰寫劇本，並由大衛·亞羅維斯基（{{lang|en|David Yarovesky}}）執導<ref name="GunnProducer">{{cite web |url=https://variety.com/2017/film/news/james-gunn-new-horror-movie-1202640929/ |title=‘Guardians of the Galaxy’ Director James Gunn to Produce Horror Movie for H Collective |last=McNary |first=Dave |website=[[Variety_(magazine)|Variety]] |date=2017-12-14 |accessdate=2018-12-09}}</ref>。

[[伊莉莎白·班克絲|伊莉莎白·班克絲]]、[[大衛·丹曼|大衛·丹曼]]、傑克森·A·鄧恩（{{lang|en|Jackson A. Dunn}}）、{{link-en|麥特·瓊斯 (演員)|Matt Jones (actor)|麥特·瓊斯}}及{{link-en|梅洛狄斯·漢格|Meredith Hagner}}於2018年3月加入該片的演出，並於同月開始進行[[主要拍攝|主要拍攝]]<ref name="ShootingBegins">{{cite web |url=https://movieweb.com/james-gunn-new-horror-movie-cast-production-start/ |title=James Gunn's New Horror Movie Cast Announced as Shooting Begins |last=Scott |first=Ryan |website=MovieWeb |date=2018-03-29 |accessdate=2018-12-09}}</ref>。

== 發行 ==
《靈異乍現》由{{link-en|索尼影視集團|Sony Pictures Motion Picture Group|索尼影視發行}}定於2019年5月24日在美國上映<ref name="2019MoveVariety">{{cite web |last=McNary |first=Dave |url=https://variety.com/2018/film/news/james-gunn-horror-movie-brightburn-2019-1202973325/ |title=James Gunn's Horror Movie 'BrightBurn' Moved to 2019 |date=2018-10-08 |accessdate=2018-10-10 |work=[[Variety_(magazine)|Variety]]}}</ref><ref name="May2019Pushback">{{cite web |last1=Couch |first1=Aaron |last2=McClintock |first2=Pamela |url=https://www.hollywoodreporter.com/heat-vision/james-gunn-movie-brightburn-open-may-2019-1150428 |title=Sony Pushes Back James Gunn Horror Movie to 2019 |date=2018-10-08 |accessdate=2018-10-09 |work=[[The_Hollywood_Reporter|The Hollywood Reporter]]}}</ref>；在此前，該片的上映日原定於2018年11月30日<ref name="BanksGunnReunite">{{cite web  |last=Busch |first1=Anita |url=https://deadline.com/2018/03/elizabeth-banks-james-gunn-horror-picture-movie-1202315120/ |title=Elizabeth Banks And James Gunn Reunite For Horror Picture |date=2018-03-07 |accessdate=2018-10-10 |work=[[Deadline_Hollywood|Deadline Hollywood]]}}</ref>。

=== 市場營銷 ===
原本《靈異乍現》計畫在2018年[[聖地牙哥國際動漫展|聖地牙哥國際動漫展]]上作宣傳，但因詹姆斯·岡恩被[[華特迪士尼影業集團|華特迪士尼影業]]與[[漫威工作室|漫威工作室]]解雇後<ref name="GunnNoMention">{{cite web |last1=Kupfer |first1=Lindsey |last2=Hood |first2=Bryan |url=https://pagesix.com/2018/07/20/sony-avoids-mentioning-james-gunn-at-comic-con-presentation/ |title=Sony avoids mentioning James Gunn at Comic-Con presentation |date=2018-07-20 |accessdate=2019-01-04 |work=Page Six}}</ref>，才在最後的一分鐘取消了該宣傳計劃（岡恩與迪士尼、漫威於九個月後和解）<ref name="GunnRehired">{{cite web|url=https://deadline.com/2019/03/james-gunn-reinstated-guardians-of-the-galaxy-3-disney-suicide-squad-2-indefensible-social-media-messages-1202576444/|title=Disney Reinstates Director James Gunn For 'Guardians Of The Galaxy 3'|last=Fleming Jr.|first=Mike|date=2019-03-15|work=[[Deadline_Hollywood|Deadline Hollywood]]|archiveurl=http://www.webcitation.org/76thA7oq6|archivedate=2019-03-15|deadurl=no|accessdate=2019-03-15}}</ref>。2018年12月8日，《靈異乍現》的首支預告片在網上釋出。由於其內容和[[超人|超人]]的{{link-en|超人的起源|Origin of Superman|起源故事}}有意相似，使得媒體和影評人將該預告片評為「超人恐怖片」<ref name="BrightburnTrailer">{{cite web |url=https://www.rollingstone.com/movies/movie-news/see-james-gunns-brightburn-trailer-766599/ |title=‘Brightburn’: See Terrifying Trailer for James Gunn’s ‘Superhero Horror’ Film |last=Kreps |first=Daniel |date=2018-12-08 |website=[[Rolling_Stone|Rolling Stone]] |language=en-US |accessdate=2018-12-09}}</ref><ref name="BrightburnTrailer2">{{cite web |url=https://www.polygon.com/2018/12/8/18132220/brightburn-james-gunn-trailer-premiere-launch-date |title=James Gunn’s Brightburn looks like Superman as a horror story |last=S. Good |first=Owen |website=[[Polygon_(website)|Polygon]] |date=2018-12-08}}</ref>。《{{link-en|Fast Company|Fast Company}}》認為：「雖然它不是正式的超人電影，但它會讓觀眾走進{{link-en|克拉克·肯特|Clark Kent}}起源故事的每一步，然後再轉向左轉。」<ref name="FastCompanyTrailer">{{cite news |last=Berkowitz |first=Joe |date=2018-12-10 |title=“What if Superman was an evil kid?” asks this creepy-as-hell movie trailer |url=https://www.fastcompany.com/90278878/what-if-superman-was-an-evil-kid-asks-this-creepy-as-hell-movie-trailer |language=en |work=''Fast Company'' |accessdate=2018-12-18}}</ref><ref>{{Cite web|url=http://www.dejavublog.in/2018/12/the-dark-side-of-superman-james-gunns.html|title=The Dark Side of Superman - James Gunn's Brightburn adds an Evil Twist to DC’s Most Powerful Superhero|last=Gogoi|first=Pallav|website= Dejavublog.in|access-date=2019-03-08}}</ref>
