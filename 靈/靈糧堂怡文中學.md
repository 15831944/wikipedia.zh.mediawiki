'''靈糧堂怡文中學'''（簡稱'''怡中'''，{{lang-en|Ling Liang Church E Wun Secondary School}}）位於[[香港|香港]][[大嶼山|大嶼山]][[東涌|東涌]]文東路37號，為一所[[基督教|基督教]]政府資助男女文法中文中學，於2002年開辦，辦學團體為[[香港靈糧堂|香港靈糧堂]]，現任校長為[[羅偉文|羅偉文]]先生。

== 學校設施 ==
該校共有八層，按樓層分佈有以下設施：
*地下：兩個籃球場、有蓋操場、食物部、飯堂、學生活動室、圖書館、以琳廣場、學生會辦公室
*一樓：禮堂、教員室、107室（內含一個會議室、兩個多用途室，可作會客、放置書簿、輔導學生等、另設圓桌一張於三室前堂，供師生日常使用）、校務處、訓導室、課室一間（同為升輔組總部）
*二樓：視覺藝術室、電腦輔助學習室（主要為[[新高中|新高中]][[電腦及資訊科技|電腦及資訊科技]]教學用）、音樂室、禮堂閣樓（供禮堂音控用途）、中二課室五間、課室一個
*三樓：設計與科技室（工場與科技支援教室各一）、中四課室六間、數學園地（佔半課室）
*四樓：生物實驗室一間、地理室一間、中一課室五間、預科課室兩間
*五樓：IT支援及伺服器室、電腦室、兩間科學實驗室、中五課室五間
*六樓：化學、物理實驗室各一間、中三課室五間、預科課室兩間
*天台：環保農莊。

== 班級結構 ==
中一至中五級開設四班。愛，仁班是精英班，主要是因應學生的英國語文水平而決定。
故這班主要是以英國語文作為教學語言，例如綜合科學科、數學科、等等（中文科及通識科除外）。
除了這一班精英班以外，則普遍以中文作為教學語言，從而照顧不同程度的學生需要。
中六級開設四班，分別為信、望、愛和仁。
學習課程包括新高中學制的必修課程(中國語文、英國語文、數學和通識教育)和選修課程，兩級的最後兩個班別開設為延伸數學班。
延伸數學是俗稱的M1和M2，M1為[[微積分|微積分]]和[[統計|統計]]，M2為[[代數|代數]]和[[微積分|微積分]]。

== 辦學宗旨 ==
按照聖經真理提供優質全人教育；藉著傳揚耶穌基督的福音，使學生活出基督的樣式，服侍上帝及社群；並與一條龍小學攜手為整個基礎教育作出全盤策劃，提供富連貫性和全面的學習經歷 

== 升學模式 ==
*一條龍：靈糧堂秀德小學 直升 靈糧堂怡文中學

== 教學語言 ==
*中文、普通話作為中文科，数学科。。。教學語言
*英文作为英文科教学语言。（当如是精英班的学生，除中文科，大多用英文教学）

== 發展方向 ==
*一、 以耶和華為元帥，本基督耶穌的愛，建立學生「愛 神愛人」的品格，讓福音傳遍校園，從而為 神國度建立精兵；
*二、 以「賢慧教育」(啟發潛能教育)的理念，提供一個能夠發揮每個學生潛質的教育環境；
*三、 以「成功教育」的理念，發展學生學習；
*四、 透過全校參與「[[融合教育|融合教育]]」，照顧學生學習差異。
