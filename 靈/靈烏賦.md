'''靈烏賦'''，係[[范仲淹|范仲淹]]所作之[[賦|賦]]<ref>{{cite book |last=沈松勤 |first= |date=2014-4-25 |title=新譯范文正公選集（二版） |url= |location= |publisher=[[三民書局|三民書局]] |page= |isbn=9789571451237 |author-link= }}</ref>，大約於1036年完成<ref name = “wuseik”>{{cite journal |last=[[胡適|胡適]] |first= |last2= |first2= |date= |title=寧鳴而死，不默而生 |url= |journal=《[[自由中國|自由中國]]》 |volume=12 |issue=7 |pages= |doi= |date= 1954-9-3}}</ref>。此賦主張[[言論自由|言論自由]]<ref name = “wuseik”/>及[[正義|公義]]<ref name=“taikeiyuen”>{{cite web |author1= |title=成語典故：寧鳴而死，不默而生|url = http://www.epochtimes.com/b5/8/12/10/n2357683.htm|agency=[[大紀元|大紀元]]|date=2008-12-12|accessdate=2018-09-29|}}</ref>。名言「寧鳴而死，不默而生」正正出於此賦<ref name=“taikeiyuen”/><ref name=“yukman”/>。

==背景==
范仲淹因認為，政壇腐敗，而提出意見，是以得罪權貴，而遭貶官。

范仲淹之詩友[[梅堯臣|梅堯臣]]就寫了一首《靈烏賦》指，范仲淹於政壇多次直言，都被當成係烏鴉不祥之叫聲，所以希望范仲淹，以後學鳳凰報喜之叫聲，而不要像烏鴉般因報凶訊而被斥。

范仲淹以同樣題目《靈烏賦》回應梅堯臣，表達自己寧願因仗義執言而被陷害致死，也不沈默無語，茍且偷生<ref name=“yukman”>{{cite web |author1=[[黃毓民|黃毓民]] |title=寧鳴而死，不默而生|url = https://maddog.myradio.hk/2018/09/14/寧鳴而死，不默而生/|agency=[[癲狗日報|癲狗日報]]|date=2018-09-14|accessdate=2018-09-29|}}</ref>。

==內容==
范仲淹認為，烏鴉縱使因報凶而使自己折翼、被烹，但都不願隱瞞凶訊而帶來災禍給人。范仲淹認為，烏鴉是在災禍未出現，就憂慮，烈火未熾，就恐懼。范仲淹認為，正因如此，所以烏鴉就要「寧鳴而死，不默而生」。<ref name=“yukman”/>

==評論==
*[[胡適|胡適]]指，范仲淹之《靈烏賦》比[[帕特里克·亨利|帕特里克·亨利]]之「不自由，毋寧死」早740年，可以特別作為中國爭取自由史上之一段佳言。
*[[黃毓民|黃毓民]]引用范仲淹之《靈烏賦》指，2018年之香港已經陷入一個「萬馬齊喑究可哀」之困局，是因為從政者缺乏「憂於未形，恐於未熾」之精神，而知識分子亦無「寧鳴而死，不默而生」之勇氣。黃毓民又指，如果各人皆可「不平則鳴」，敢於發聲，撥亂反正，香港必能重見光明。
