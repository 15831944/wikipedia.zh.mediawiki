[[敦煌文獻|敦煌文獻]]《'''志玄安乐经'''》，是[[景净|景净]]翻译的[[景教|景教]]文献的晚唐抄本，最初為藏書家[[李盛鐸|李盛鐸]]私藏，後被賣給日本人，直至1958年方有寫本的首尾局部照片披露，<ref>{{cite web |url=http://ex.cssn.cn/zjx/zjx_zjyj/zjx_jdjyj/201310/t20131029_755372.shtml |title=敦煌本景教《志玄安樂經》佐伯錄文質疑 |last=林悟殊 |date=2013 |website=ex.cssn.cn |access-date=2018-12-24}}</ref>今归日本[[大阪市|大阪市]]杏雨書屋收藏。<ref>{{cite web |url=https://www.xueshu.com/dhxjk/201502/2332981.html |title=日本杏雨書屋藏唐代敦煌景教寫本《志玄安樂經》釋考 |author=王蘭平 |date=2015 |website=xueshu.com |publisher= |access-date=2018-12-28}}</ref>

[[羽田亨|羽田亨]]曾在《景教經典志玄安樂經考釋》一文中對該寫本的外觀作過一番描述：「此經典有如敦煌出土的諸多經卷一樣，寫於黃麻紙上，上下和行間施以細欄。首行與第一百五十九行，即末行，如前所述，題有『志玄安樂經』，首尾完結，惟始端十行下半部殘缺。字體異於《[[一神論_(寫本)|一神論]]》和《[[序聽迷詩所經|序聽迷詩所經]]》，與之相比，稍硬而粗，毋寧說是近於《[[景教三威蒙度讚|三威蒙度讚]]》者。……書寫之字體殆屬晚唐時期無誤。」<ref>{{cite web |url=http://coe21.zinbun.kyoto-u.ac.jp/papers/is-religion-2004/Lin.pdf |title=漢文摩尼經與景教經及其宏觀比較 |last=林悟殊 |date=2004 |website=coe21.zinbun.kyoto-u.ac.jp |page={{p.|139}} |access-date=2018-12-28}}</ref>

《志經》内容是教人求得安乐的方法：

{{Cquote 3|是故我言，无欲无为，无德无证，如是四法，不炫己，能离言说，柔下无忍，潜运大悲，人民无无边欲，令度尽于诸法中，而获最胜，故名安乐道……}}

{{Cquote 3|无动无欲，则不求不为。无求无为，则能清能净。能清能净，则能晤能正。能晤能证，则遍照遍境。遍照遍境，是安乐缘}}

{{Cquote 3|云何名为十种观法？一者观诸人间，肉身性命积渐衰老，无不灭亡；二者观诸人间，亲爱眷属，终当离拆，难保会同；三观诸人间，高大尊贵，荣华兴盛，终不常居；……}}

日本学者羽田亨认为《志玄安乐经》的思想和《[[道德经|道德经]]》相似，而体制又类似佛经，乃是[[景教|景教]]在中国流行后受到佛教和道教影响下的产物。<ref>[[朱谦之|朱谦之]]著 《中国景教》122-124页 人民出版社 1998</ref>
