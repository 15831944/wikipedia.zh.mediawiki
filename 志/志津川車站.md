'''志津川車站'''（{{jpn|j=志津川駅|hg=しづがわえき|rm=Shizugawa eki|f=y}}）是一位於[[日本|日本]][[宮城縣|宮城縣]][[本吉郡|本吉郡]][[南三陸町|南三陸町]]、[[東日本旅客鐵道|東日本旅客鐵道]][[氣仙沼線|氣仙沼線]]沿線的[[鐵路車站|鐵路車站]]。志津川車站屬於[[東日本旅客鐵道仙台支社|JR東日本仙台支社]]管轄的範圍內，由[[石卷車站|石卷車站]]代為管理，並[[業務委託車站|委託]]給JR東日本的子公司[[東北綜合服務|東北綜合服務]]（{{lang|ja|東北総合サービス}}）營運。

== 車站構造 ==
伴隨BRT開始運行，舊站舍跡地前設置BRT車站，本格開始運行，消防署跡地整備為新站舍並移動車站<ref name="jrsendai20121126" />。窗口業務同時再開。

發生地震前，是[[側式月台|側式月台]]1面1線與[[島式月台|島式月台]]1面2線，合計2面3線的[[地面車站|地面車站]]。

== 歷史 ==
[[File:Shizugawa_Station_platform_after_Tsunami.JPG|thumb]]
*1977年（[[昭和|昭和]]52年）12月11日 - 車站啟用。
*1987年（昭和62年）4月1日 - [[日本國鐵分割民營化|日本國鐵分割民營化]]，車站管轄劃歸由東日本旅客鐵道繼承。
* 2011年（[[平成|平成]]23年）3月11日 - 東北地方太平洋近海發生[[2011年日本東北地方太平洋近海地震|嚴重地震]]，地震引發的[[海嘯|海嘯]]沖毀靠近海岸的許多鐵路設施，志津川也是23個被沖毀的鐵路車站之一<ref>{{cite web |url=http://www.sankeibiz.jp/business/news/110401/bsd1104010801004-n1.htm |language=ja |title=<span lang="ja">【東日本大震災】ＪＲ東日本、太平洋沿岸の２３駅流失</span> |publisher=[[產經新聞社|產經新聞社]] |date=2011-04-01 |accessdate=2011-04-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20110413050421/http://www.sankeibiz.jp/business/news/110401/bsd1104010801004-n1.htm |archivedate=2011-04-13 }}</ref>。包括車站站房與月台上的候車亭全遭大水沖毀<ref>{{cite web|url=http://mainichi.jp/select/jiken/graph/20110313/59.html |language=ja |title=<span lang="ja">東日本大震災＝３月１３日</span>（照片） |publisher=[[每日新聞|每日新聞]] |date=2011-03-13 |accessdate=2011-04-17 |archiveurl=https://archive.is/20120713123058/http://mainichi.jp/select/shakai/ |archivedate=2012-07-13 |deadurl=yes }}</ref>。
