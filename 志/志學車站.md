== 車站構造 ==
*[[島式月台|島式月台]]兩座，另有一工務月台。
*第一和第二月台間包夾一股線，為[[西班牙式月臺佈局|西班牙式月臺佈局]]。
*原則上列車都是在第一月台A側停靠與通過，需交會或待避時才會使用到第一月台B側與第二月台。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。
*停靠狀態：
**[[普悠瑪列車|普悠瑪號]]：停靠順行406、232(週日停駛)、252次；逆行207、225、283次(週日停駛)。
**[[太魯閣列車|太魯閣號]]：停靠順行228次（終點）；逆行229次（起點）。
**[[自強號列車|自強號]]：停靠順行306、308、328（逢週五、日行駛）、340（逢週五、日行駛）、332（逢週五、日行駛）、350（逢週五、日停駛）次；逆行407、303、307、317、321（逢週五、日停駛）、323（逢週五、日行駛）次。
**[[莒光號列車|莒光號]]：除[[觀光列車|觀光列車]]以及[[團體列車|團體列車]]，609次(週五行駛)，604次(週一行駛) 未停靠外，其餘皆停靠。
**[[臺鐵區間車|區間車]]：皆停靠。
*對號列車停靠率：40.7%；所有車種停靠率：51.5%。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 連署活動 ==
2014年4月底，[[國立東華大學|國立東華大學]]學生發起志學站更名連署活動，希望該站能比照[[新城（太魯閣）車站|新城車站]]模式「新城〔太魯閣〕」將車站更名為「'''志學〔東華大學〕'''」，以提升該地知名度，以及增加觀光客源。並且，同時希望能夠有更多[[自強號列車|自強號]]停靠該站，以方便[[居民|居民]]以及[[學生|學生]]返鄉需求。除此之外，學生也表達該站需要提升[[硬體|硬體]]設施的需求，例如該站並未設置[[無障礙設施|無障礙設施]]，時常導致行動不便者須行走天橋，且在雨日有滑倒發生危險之虞。對於此舉，校方表示樂觀其成{{R|CT_20140429}}。

== 歷史 ==
*1911年9月25日：設「'''知伯乘降場'''」，因早年此地產「志哈克樹」（[[阿美語|阿美語]]：chiak，植物名為杜虹，又稱做[[杜虹花|台灣紫珠]]），樹名遂成社名，日文音譯為「知伯」。
*1918年：改為「'''賀田驛'''」，以紀念[[賀田金三郎|賀田金三郎]]開發東部。
*1948年：改為「'''豐年'''」站。
*1952年8月1日：以「志哈克」（chiak）的[[台語|台語]]音譯，地名和站名改為「'''志學'''」。
*2014年7月16日：配合東部幹線電氣化工程完成，[[太魯閣列車|太魯閣號]]開始停靠志學車站。
*2014年11月18日：[[普悠瑪列車|普悠瑪號]]開始停靠。
*2015年10月15日：改點後[[太魯閣列車|太魯閣號]]不再停靠，[[普悠瑪列車|普悠瑪號]]取代並增為單向一日三班。
*2016年4月21日：改點後[[太魯閣列車|太魯閣號]]再度恢復停靠。
*2016年6月28日：啟用多卡通刷卡機。
