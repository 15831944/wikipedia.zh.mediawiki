[[Image:Virtual_Private_Cloud.svg|right]]'''虛擬私有雲''' （Virtual Private Cloud, VPC） 是存在於共享或[[公用雲|公用雲]]中的[[私有雲|私有雲]] （Private Cloud），亦即一種[[網際雲|網際雲]]（[[:en:Intercloud|Intercloud]]）。 

[[Amazon_Web_Services|Amazon Web Services]] 在 2009-08-26 發表了 [[Amazon_Virtual_Private_Cloud|Amazon Virtual Private Cloud]]，該服務允許用戶透過 [[IPsec|IPsec]] [[虛擬私有網路|虛擬私有網路]] (VPN) 來連結 [[Amazon_Elastic_Compute_Cloud|Amazon Elastic Compute Cloud]] (EC2)。<ref>[http://arstechnica.com/web/news/2009/08/amazon-to-soothe-enterprise-fears-with-virtual-private-cloud.ars Amazon to soothe Enterprise fears with Virtual Private Cloud]</ref><ref>[http://news.zdnet.co.uk/software/0,1000000121,39730194,00.htm Amazon launches Virtual Private Cloud service]</ref> 

[[Google_App_Engine|Google App Engine]] 在 2009-04-07 發表的 ''Secure Data Connector'' 產品中也支援類似的功能。<ref>[http://code.google.com/securedataconnector/ S'More enterprise developer tools: App Engine and Secure Data Connector]</ref><ref>[http://pro.01net.com/editorial/505458/cloud-computing-amazon-lance-une-offre-de-nuage-prive/ Cloud computing : Amazon lance une offre de nuage privé]</ref>
