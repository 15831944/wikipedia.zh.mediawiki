'''虛空藏菩薩經'''（梵文：Ākāśagarbha Sūtra），收於《[[大正藏|大正藏]]》大集部，本經敘述[[虛空藏菩薩|虛空藏菩薩]]威神力，可開發染罪眾生的[[善根|善根]]，懺悔根本罪業，顯示其重罪可懺除的大乘懺罪觀。[[藏傳佛教|藏傳佛教]]依此將本經視為「[[中觀派佛教|中觀派]]」[[龍樹|龍樹]]傳承[[菩薩戒|菩薩戒]]所依循的佛典<ref name="aka2">{{cite web|title=大乘佛教懺悔觀之研究|url=http://hl002.epage.au.edu.tw/ezfiles/208/1208/attach/0/pta_46051_5680239_85647.doc}}</ref><ref name="aka"/>。

== 題解 ==
[[虛空藏菩薩|虛空藏菩薩]]，ākāśa 虛空、蒼穹，garbha 胎藏、庫藏，亦譯虛空孕、虛空庫，孕藏無盡[[虛空_(佛教)|虛空]]之意。

== 傳譯 ==
本經傳來中國，共有四次翻譯<ref name="aka">{{cite journal|author1=釋圓和|title=《虛空藏菩薩經 》根本罪之研究|url=http://www.fgu.edu.tw/~buddhist/chinese/paper_20/012.pdf}}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>：

* 佛陀耶舍《虛空藏菩薩經》，譯出年代為姚秦弘始年（公元413年） 
* 曇摩蜜多《虛空藏菩薩神咒經》，譯出年代為劉宋元嘉年間（公元424~436年間）
* 闍那崛多《虛空孕菩薩經》，譯出年代為隋開皇七年（公元587年）
* 失譯本《虛空藏菩薩神咒經》

== 內容 ==
本經揭示[[虛空藏菩薩|虛空藏菩薩]]會依眾生的累世善根，導出其正知見，破除其往昔惡業、惡知見；以菩薩渡眾方便力，相應己身的善根，使眾生映照出己身之惡業，再以正知見破除惡業、邪見、惡願、惡歸、惡處、惡取，使之解脫。

此經中的懺悔觀著重在懺除五根本罪，以禮敬、讚嘆、至心歸命虛空藏菩薩，與菩薩感應道交，獲得其加佑庇護，並蒙其化身說法渡化，發露懺悔，捨棄惡業，精進六度，成就無上菩提<ref name="aka2"/>。

== 影響==
本經為[[宗喀巴|宗喀巴]]作《菩提正道菩薩戒論》的重要依據。
