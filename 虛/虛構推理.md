《'''虛構推理'''》（{{lang-ja|虚構推理}}）是日本小說家[[城平京|城平京]]的[[推理小說|推理小說]]作品，插畫為[[清原紘|清原紘]]。由[[講談社Novels|講談社Novels]]出版。改編的[[日本漫畫|日本漫畫]]作品由[[漫畫家|漫畫家]][[片瀨茶柴|片瀨茶柴]]作畫，[[講談社|講談社]]出版，於講談社漫畫雜誌《[[月刊少年Magazine|月刊少年Magazine R]]》連載<ref>{{cite web|script-title=ja:コミックナタリー Power Push - 少年マガジンR|url=http://natalie.mu/comic/pp/magaziner|website=[[Natalie_(網站)|Natalie]]|accessdate=2016年7月9日|date=2015年11月17日}}</ref>。

2019年1月正式宣佈推出電視動畫作品。 

== 故事簡介 ==

== 登場角色 ==
; {{nihongo|岩永琴子|岩永 琴子（いわなが ことこ）}}（[[声优|声优]]：[[鬼头明里|鬼头明里]]）
: 小時候曾失蹤，兩週後被發現。發現時身體缺損，失去右眼和左腳。失蹤期間答應成為怪異的「智慧之神」，身體缺損是能夠接觸到怪異的特徵。
:對櫻川九郎一見鍾情，形容對方是像山羊一般的人。
; {{nihongo|櫻川九郎|桜川 九郎（さくらがわ くろう）}}（声优：[[宫野真守|宫野真守]]）
: 年幼時，家族中流傳利用妖怪之力的迷信而吃下[[妖怪|妖怪]]{{tsl|ja|件}}和[[人魚|人魚]]，因此成為世上的異常。雖然是人類，但混合妖怪和魔物特質，一般的怪物都非常害怕他。
: 曾經是女警弓原紗季的年下男友。本來已經到了論及婚嫁的地步，但在河童事件發生後就迅速分手。現為岩永琴子的男友。

; {{nihongo|弓原紗季|弓原 紗季（ゆみはら さき）}}
:25歲。交通課女警，是九郎的前女友。

; {{nihongo|鋼人七瀨|鋼人 七瀬（こうじんななせ）}}
:都市傳說。

; {{nihongo|七瀨花凜|七瀬 かりん（ななせ かりん）}}
:19歲寫真偶像。
; {{nihongo|七瀨初實|七瀬 初実（ななせ はつみ）}}
:花凜的姐姐。
; {{nihongo|寺田德之助|寺田 徳之助（てらだ とくのすけ）}}
:和紗季同署的刑警。
; {{nihongo|櫻川六花|桜川 六花（さくらがわ りっか）}}
: 九郎的堂姊。有著和九郎一樣的異常特質。

; 天地學
: 高中時就讀私立瑛瑛高中，那時擔任推理同好社的社團團長。岩永琴子是部員之一。

; 小林小鳥
: 高中時就讀私立瑛瑛高中，亦是推理同好社社員。小天地學一個學年。實際上是天地學的女友，兩人在岩永琴子加入後開始正式交往。
