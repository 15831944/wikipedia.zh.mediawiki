'''虛擬YouTuber'''（{{lang-ja|バーチャルユーチューバー}}，{{lang-en|Virtual YouTuber}}，也縮寫爲{{lang|en|VTuber}}<ref>{{Cite web|url=http://www.itmedia.co.jp/news/articles/1804/03/news110.html|title=VTuberに特化したプロダクション設立　CyberZが新会社で|accessdate=2018-04-09|date=2018-04-03|publisher={{tsl|ja|ITmedia}}}}</ref>）是以[[YouTube|YouTube]]爲平臺進行影片直播和投稿的{{tsl|ja|バーチャルアイドル||虚拟偶像}}。在YouTube以外的平臺中又被稱作“'''虛擬主播'''”（{{lang-ja|バーチャルライバー}}）<ref>{{Cite web|url=http://gamebiz.jp/?p=210034|title=CyberZ、『OPENREC』にゲーム配信に特化した新バーチャルライバーグループ「にじさんじゲーマーズ」の公式個人チャンネルを開設|accessdate=2018-05-06|date=2018-05-01|publisher=SOCAIAL GAME INFO}}</ref>。

== 簡介 ==
該名詞由2016年12月開始活動的[[絆愛|絆愛]]首次使用，定義爲“由[[電腦圖形|電腦圖形]]所繪製的[[插畫|插畫]]風格的[[美少女|美少女]][[YouTuber|YouTuber]]”<ref name="nikkeistyle201807">{{Cite news|title=ユーチューブのCGアイドル「Vtuber」が人気爆発|url=https://style.nikkei.com/article/DGXMZO32234030W8A620C1000000|publisher=[[日本經濟新聞|日本經濟新聞]]|newspaper={{link-ja|日経トレンディ}}|date=July 9, 2018|accessdate=2018-07-09}}</ref>。雖然在此之前就已經有類似的虛擬角色出現<ref>{{Cite web |url=http://weathernews.com/ja/nc/press/2013/130204_3.html|title="WEATHEROID TypeA Airi" 本日24時間ライブ放送番組「SOLiVE２４」にてデビュー！ |publisher=[[weathernews|weathernews]]|date=2013-02-04|accessdate=2018-08-15}}</ref>，但是當時這類虛擬角色並未定性爲虛擬YouTuber。因此在一段時間，虛擬YouTuber是絆愛的代名詞{{Sfn|ユリイカ|2018|p=29}}。

2017年開始，隨着諸如“{{link-ja|電腦少女小白|電脳少女シロ|電-{}-腦少女小白}}”、“{{link-ja|Mirai Akari|ミライアカリ}}”、“{{link-ja|バーチャルのじゃロリ狐娘Youtuberおじさん}}”、“{{link-ja|輝夜月}}”等擁有大量[[粉絲|粉絲]]的虛擬YouTuber的加入{{Sfn|ユリイカ|2018|p=46}}，原先作爲絆愛的代名詞的虛擬YouTuber一詞被重新定義，並逐漸廣泛使用{{Sfn|ユリイカ|2018|p=29}}。

一些虚拟角色也在其他媒体领域发展，如出演[[地面电视|地面电视]]节目<ref>{{Cite web |url=https://news.mynavi.jp/article/20180406-612396/|title=バーチャルYouTuberが地上波MCに、新世代のテレビ番組スタート|publisher={{tsl|ja|マイナビニュース}}|date=2018-04-06|accessdate=2018-04-09}}</ref>、出售商品等，此时虚拟YouTuber是作为一个便于理解的品牌名称。

{{tsl|ja|グリー (企業)|GREE公司}}于2018年4月5日宣布将向虚拟YouTuber产业投资100亿日元<ref>{{Cite web |url=https://www.oricon.co.jp/news/2109025/full/|title=グリー、バーチャルYouTuber市場に参入 新会社設立で100億円規模投資へ |publisher=[[Oricon|ORICON NEWS]]|date=2018-04-05|accessdate=2018-04-09}}</ref>。

截至2018年5月28日，虚拟YouTuber總計有3000人，訂閱者共1089万，視頻點擊率達到6億9000万次以上<ref>{{Cite web|url=https://www.moguravr.com/vtuber-3000/|title=バーチャルYouTuberが3,000人を突破 合計チャンネル登録者数は1000万人以上に|accessdate=2018-06-17|date=2018-05-28|publisher=Mogura VR}}</ref>。随着虚拟YouTuber迎来热潮，一些虚拟YouTuber组织也随即兴起，其中比较有名的有Upd8、ENTUM、彩虹社（也译2434）、hololive等。

===台灣===
* [[虎妮|虎妮]]為Yahoo所屬的虛擬網紅，除了YouTube之外，在[[Yahoo_TV|Yahoo TV]]上也擁有自己的節目「虎妮好朋友」，影片亦於日本niconico動畫頻道上發表。
* [[空|空]]為[[KKBOX|KKBOX]]所推出的虛擬網紅，以歌唱表演為主，並且與另一位虛擬網紅Hooya合組了第一個華語表演團體[[K’WA|-{K’WA}-]]。

===中國大陆===
在中華人民共和國，由于[[对YouTube的审查#中華人民共和國|YouTube遭到屏蔽]]，用户不能直接观看虛擬YouTuber的视频，因此有用戶在通過授權的情況下將相關視頻轉載到如[[Bilibili|Bilibili]]之類的網站，另外也有一部分虛擬YouTuber在中國網站內設有官方頻道。一部分虛擬YouTuber在Bilibili的粉丝订阅量远高于YouTube的订阅量{{noteTag|如白上フブキ（YouTube订阅量10万+，Bilibili订阅量40万+），神楽めあ（YouTube订阅量5万+，Bilibili订阅量30万+）等（数据截止于2019.4.19）。}}。除此之外，在Bilibili上也活动着一些虚拟视频制作人，称为“虚拟UP主”<ref name="tencent-chinese-vtuber-news">{{cite web |title=一起来成为偶像吧：B站首个虚拟UP主小希走红_游戏_腾讯网 |url=http://games.qq.com/a/20180424/027745.htm |website=腾讯游戏 |accessdate=2018-04-24 |language=zh-CN}}</ref>。

== 动作 ==
部分虛擬YouTuber會借助安置在頭部与肢體上的[[動作捕捉|動作捕捉]]設備以及傳感器將人物動作展現到虛擬角色上。而藉助於實時運動捕捉的機制，虛擬YouTuber還可以通過多種方式與現實世界中的粉絲進行交流<ref>{{Cite web |url=https://www.fnn.jp/posts/00391203CX|title=バーチャルユーチューバー　大手も続々参入　ネットとリアルの新時代 |publisher=[[富士新聞網|富士新聞網]]|date=2018-05-04|accessdate=2018-05-04}}</ref>。

== 反應 ==
據[[英国广播公司|BBC]]的報導中所指，虛擬YouTuber的特點是“不受個人身份與其他問題所束縛”，且在全球範圍內受到許多“日本以外的日本文化與動漫愛好者”的歡迎<ref>{{Cite web |url=http://www.bbc.com/capital/story/20181002-the-virtual-vloggers-taking-over-youtube|title=The virtual vloggers taking over YouTube|publisher=[[英国广播公司|BBC]]|date=2018-10-03|accessdate=2018-10-03}}</ref>。
