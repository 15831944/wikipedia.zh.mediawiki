'''济南轨道交通3号线'''，工程名为'''R3线'''，是[[中华人民共和国|中华人民共和国]][[济南市|济南市]]建设中的一条[[地铁|地铁]]线路。项目一期工程全长21.592公里，共设13站。2016年6月20日，一期工程全线开工建设。<ref>[http://ggzy.jinan.gov.cn/art/2016/6/6/art_2103_251918.html 济南市轨道交通R3线一期土建工程施工中标公示] {{webarchive|url=https://web.archive.org/web/20160612110819/http://ggzy.jinan.gov.cn/art/2016/6/6/art_2103_251918.html |date=2016-06-12 }}</ref><ref>[http://ggzy.jinan.gov.cn/art/2016/6/6/art_2103_251918.html 济南市轨道交通R3线一期土建工程施工中标公示] {{webarchive|url=https://web.archive.org/web/20160612110819/http://ggzy.jinan.gov.cn/art/2016/6/6/art_2103_251918.html |date=2016-06-12 }}</ref>将于2019年10月1日建成通车。<ref>{{Cite web|title=济南地铁3号线今年国庆节开通，提前十五个月！_齐鲁壹点网|url=http://www.ql1d.com/news/show/id/9739848.html|work=www.ql1d.com|accessdate=2019-01-28}}</ref>

== 线路概况 ==
一期工程起于南端的[[龙洞庄站|龙洞庄站]]，止于[[滩头站|滩头站]]。设站13座，平均站间距1.778km。线路南端设龙洞停车场，北端设济南东车辆段；工程计划投资额约为510000万元。<ref>[http://ggzy.jinan.gov.cn/art/2016/4/15/art_2102_37643.html 济南市轨道交通R3线一期土建工程施工] {{webarchive|url=https://web.archive.org/web/20160426204704/http://ggzy.jinan.gov.cn/art/2016/4/15/art_2102_37643.html |date=2016-04-26 }}</ref>

二期工程向北延伸至[[济南遥墙国际机场|遥墙机场]]，服务空港组团、遥墙机场等主要客流集散点，是支持济南城市规划近期重点区域发展，加强重要对外客运枢纽衔接的线路。线路南起一期工程终点[[滩头站|滩头站]]，北至遥墙机场站，主要沿机场路走行，全长约12.9公里，共设车站5座，全线均采用地下敷设方式。<ref>{{cite web |title=济南轨道交通二期规划大调整：环线不见了，汉峪将通地铁…… |url=https://k.sina.com.cn/article_6150390138_16e97817a01900lq7q.html}}</ref>

== 车辆 ==
线路[[轨距|轨距]]为1435mm，初、近、远期均采用B型车6辆编组，4动2拖，载客量（定员）1260人/列。

== 施工标段 ==
一期工程划分为6个标段：
*一标段：龙洞庄站～孟家庄站、出入段线（2站1区间1出入场线），规模2836米，中铁三局集团有限公司中标。
*二标段：孟家庄站（不含）～龙奥站～奥体中心站西（2站2区间），规模2785米，中铁十四局集团有限公司中标。
*三标段：奥体中心西站（不含）～礼耕路站～丁家东站（2站2区间），规模1445米，中铁四局集团有限公司中标。
*四标段：丁家东站（不含）～盛福庄站～西周家庄站（2站2区间），规模2430米，中铁五局集团有限公司中标。
*五标段：西周家庄站（不含）～工业北路站～王舍人站～裴家营站（不含）（2站3区间），规模6836米，中铁十二局集团有限公司（济南城建集团有限公司）中标。
*六标段：裴家营站～济南新东站（不含）～滩头站～济南东车辆段（不含）（2站2区间1出入段线），规模3961米，北京建工集团有限责任公司中标。
