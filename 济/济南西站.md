'''济南西站'''，建设时称为'''济南西客站'''，济南市民多称其为'''西客站'''或'''西站'''，位于[[中国|中国]][[山東省|山東省]][[济南市|济南市]][[槐荫区|槐荫区]][[兴福街道|兴福街道]]原[[张庄机场|张庄机场]]西侧，距济南市中心8.5km，为[[中国铁路济南局集团有限公司|中国铁路济南局集团有限公司]]特等客运站，2011年6月30日随[[京沪高速铁路|京沪高速铁路]]建成启用。

济南西站为[[京沪高速铁路|京沪高速铁路]]的客运站，[[胶济客运专线|胶济客运专线]]、[[石济客运专线|石济客运专线]]、[[济青高速铁路|济青高速铁路]]及规划中的[[郑济客运专线|郑济客运专线]]也分别通过联络线引入济南西站。

==历史==
*2008年，京沪高铁济南西客站开工建设。<ref>{{cite web|url= http://www.crcc.cn/g705/s1728/t23718.aspx|title= 京沪高速铁路济南西客站开工建设|deadurl= yes|archiveurl= https://web.archive.org/web/20131214060527/http://www.crcc.cn/g705/s1728/t23718.aspx|archivedate= 2013-12-14}}</ref>
*2011年6月30日，随着[[京沪高铁|京沪高铁]]的通车，京沪高铁济南西客站启用，同时正式定名为'''济南西站'''，原名为济南西站的编组站更名为[[济西站|济西站]]。
*2013年1月，济南西站公交枢纽投入使用，可实现6条BRT线路，12条常规公交线路同时发车。<ref>{{cite web |url= http://www.dzwww.com/shandong/sdnews/201301/t20130107_7886231.htm|title=济南首个公交枢纽启用，18条线路可同时发车}}</ref>
*2013年5月1日，济南西站东广场投入使用，东广场为既有西广场面积的3倍，最大可疏散3万名旅客，可满足整个济南西站的客流负荷。<ref>{{cite web |url= http://news.iqilu.com/shandong/yuanchuang/2013/0412/1501543.shtml|title=五一开始济南西站东广场正式启用 可疏散3万旅客}}</ref>

===站房===
站房建筑面积为58000平方米。站房采用线上高架候车结构，并设置站前高架和落客平台。

===站场===
车站设8站台17线，其中正线2条，到发线15条，包括7个岛式站台和1个侧式站台。
