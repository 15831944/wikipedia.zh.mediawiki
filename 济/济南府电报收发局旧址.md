'''济南府电报收发局旧址'''俗称'''老电报大楼'''，位于[[中国|中国]][[山东省|山东省]][[济南市|济南市]][[天桥区|天桥区]]经一路91号，建于1904年，为济南现存最早的电讯建筑，采用[[巴洛克建筑|巴洛克建筑]]风格。1929年后改作车站邮局，部分用于开设招待所，现由济南市邮政局管理使用。该建筑处于经一路与车站街的交汇处西北角，原呈两翼基本对称的格局（南北长32米、东西长30米），北翼为柜台和电报室、设备室等，西翼为办公用房，二层为办公和住宿用房，中部设圆柱形角楼，采用砖石混合结构，墙体部分为内砖外石。2004年至2006年因经一路扩宽对建筑进行了改造重建<ref>{{cite web |url=http://news.sina.com.cn/c/2004-07-02/05152964715s.shtml|title=《老电报大楼昨起开拆》|accessdate=2012-04-30|work=李玉王|publisher=济南时报}}</ref><ref>{{cite web|url=http://www2.sdnews.com.cn/news/jiaoyu/2006-7/28_271988.html|title=《老电报大楼“瘦身”亮相》|accessdate=2012-04-30|work=|publisher=济南时报|deadurl=yes|archiveurl=https://web.archive.org/web/20070211115603/http://www.sdnews.com.cn/news/jiaoyu/2006-7/28_271988.html|archivedate=2007-02-11}}</ref>，南北方向被压缩了13米，仅保留了外侧的石材墙体，结构也被改为框架结构，内部陈设亦未能得以保留<ref>{{cite web |url=http://www.dzwww.com/xinwen/shandong/shandongtupian/200511/t20051125_1271670.htm|title=《百年电报大楼“瘦身”复建》|accessdate=2012-04-30|work=雍坚、孟帆|publisher=大众网}}</ref>。
