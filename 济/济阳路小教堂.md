'''济阳路小教堂'''是位于中国[[山东省|山东省]][[青岛市|青岛市]][[市北区|市北区]][[济阳路|济阳路]]4号的一座教堂建筑，建造时间不详，大约为德租时期之后（市档案馆、市史志办<ref Name=shi>{{cite web |url=http://qdsq-sb.qingdao.gov.cn/n18810935/n18905966/150413155239805634.html|title=青岛市济阳路基督教堂（济阳路4号）}}</ref><ref Name=zhang>{{cite web |url=http://lib.sdsqw.cn/bin/mse.exe?seachword=&K=b2&A=62&rec=76&run=13|title=青岛市志·民族宗教志·基督教差会与教派·长老会差会}}</ref>与教会<ref Name=ji>{{cite web |url=http://www.qingdaochurch.com/news_detail.asp?AutoID=377|title=济阳路礼拜堂历史沿革}}</ref>说法为1908年建造，但1914年前的地图、照片等资料并未出现该教堂），由[[美北长老会|美国长老会]]差会建造。该建筑分为两层，楼上作为教堂使用，楼下作为教会人员宿舍、厨房及餐厅使用<ref Name=ji/>。自建成后，教堂曾长期兼做[[青岛第十一中学|崇德中学]]和[[青岛女子职业中等专业学校|文德女中]]的音乐教室<ref>{{Cite book|title=山东省宗教志资料选编 第二辑|last=山东省民族志宗教志编纂工作办公室|first=|publisher=|year=1989|isbn=|location=|pages=12|chapter=第三章 青岛基督教各教派/一、长老会/2.备堂会/（1）济阳路中华基督教会}}</ref>。1950年代后该堂停止宗教活动<ref>{{Cite book|title=今古和声|last=青岛市文物局|first=|work=北美长老会旧址|publisher=文物出版社|year=2011|isbn=978-7-5010-3369-0|location=北京|pages=229}}</ref>。1991年6月，青岛市房产管理局将房产交还教会<ref Name=shi/>。该建筑现为教会财产，但仍出租作为民居和餐馆使用，保护状况不佳。
