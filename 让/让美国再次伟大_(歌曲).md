== -{zh-hans:特朗普;zh-tw:川普;}-转推 ==
为纪念[[美国独立日|美国独立日]],美国[[总统|总统]][[唐納·川普|-{zh-hans:唐纳德·特朗普;zh-tw:唐納·川普;}-]]于2017年7月4日在其个人[[推特|推特]]转推了一段视频，视频显示[[达拉斯第一浸信会|达拉斯第一浸信会]]教会诗班的[[交响乐团|交响乐团]]与[[合唱团|合唱团]]在华盛顿[[肯尼迪中心|肯尼迪中心]]的一场庆祝自由集会（Celebrate Freedom Rally）中演奏了该首歌曲。<ref>{{Cite web |url = https://www.bustle.com/p/trump-shared-a-make-america-great-again-song-its-not-making-people-proud-68165|title = Trump Shared A "Make America Great Again" Song & It's Not Making People Proud|author = KELLY TUNNEY |date = 2017年7月4日|language = en  }}</ref><ref>{{Cite web |url = http://www.guancha.cn/america/2017_07_05_416665.shtml |title = 特朗普发布独立日纪念歌曲《Make America Great Again》 英媒吐槽是朝鲜风格 |author =  李东尧|date = 2017-07-05  |publisher = 观察者网 |language = 中文 }}</ref>该浸信会现任牧师{{tsl|en|Robert Jeffress|罗伯特·杰弗斯}}（Robert Jeffress）声称该次活动旨在纪念[[退伍军人|退伍军人]]，<ref>{{Cite web |url = https://www.vox.com/identities/2017/7/4/15913590/muscular-christianity-make-america-great-again-trump-hymn |title = How one strain of macho theology leads to a church choir singing "Make America Great Again"  |date =  2017年7月4日|publisher = vox |language = en }}</ref>该牧师为-{zh-hans:特朗普;zh-tw:川普;}-的早期支持者。
