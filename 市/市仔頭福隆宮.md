'''市仔頭福隆宮'''位於臺灣臺南市[[北區_(臺南市)|北區]]，主祀[[保生大帝|保生大帝]]{{r|大道公}}{{r|聯境}}。一說該廟原稱「福隆境」，供奉福德正神，後來石頭坑大道公入祀後，改為保生大帝廟{{r|大道公}}。

== 沿革 ==
=== 立廟由來 ===
關於市仔頭福隆宮的由來，一說該廟肇建於[[明鄭時期|明鄭時期]]，是來自[[福建|福建]][[白礁|白礁]]的移民自故鄉迎奉保生大帝來臺後，因市仔頭逐漸繁榮而立廟，後來由於香火鼎盛，該廟也有「市仔頭廟」之稱{{r|聯境}}。

另一說市仔頭福隆宮本來是土地公廟，舊稱「福隆境」，又稱「市仔頭廟」，建於[[嘉慶七年|嘉慶七年]]（1802年）{{r|大道公}}。日治初期石頭坑{{NoteTag|位在永康區兵仔市，中華一路與勝利路口一帶{{r|大道公}}。}}因為盜賊劫掠而廢庄，居民乃遷到永康區的甲內（二王里）、土虱堀（網寮里）與府城[[拱辰門_(臺南市)|大北門]]一帶{{r|大道公}}。石頭坑原有大道公廟{{NoteTag|石頭坑大道公廟可見於康熙五十九年（1720年）的陳文達《臺灣縣志》{{r|大道公}}。該廟曾在乾隆末重修，增祀文衡聖帝{{r|大道公}}。}}，廢庄後神像被迎奉到福隆宮內，大多數的廟宇建材、產業歸福隆宮所有，部分則助修[[二王廟_(臺灣)|二王廟]]{{r|大道公}}。因為石頭坑大道公的入祀，福隆宮改奉石頭坑大道公為主神而變成保生大帝廟{{r|大道公}}。

=== 發展 ===
福隆宮一帶在清朝時，因為[[臺江內海|臺江內海]]各聚落可以透過文元溪將米穀運到燕潭（今[[臺南公園|臺南公園]]內），再從大北門入城販賣，而在當地形成了米市，故當地有了「市仔頭」的稱呼{{r|大道公}}。然而在日治時期，由於大北門一帶的城垣拆除並開闢了公園，米市失去交易腹地，市仔頭一帶因而沒落，福隆宮也一度荒廢{{r|大道公}}。
[[File:市仔頭福隆宮境石.jpg|thumb]]
二次大戰後，福隆宮在[[民國四十一年|民國四十一年]]（1952年）成立首屆管理委員會，三年後（1955年）莊新全、張添壽、黃進成、郭水源、曾大池、陳大抱、李春標等人發起修建，並在[[民國四十八年|民國四十八年]]（1959年）建醮{{r|聯境}}。重建工程在[[民國七十八年|民國七十八年]]（1989年）完成後，舉行了三朝慶成祈安清醮大典{{r|聯境}}。

== 祀神 ==
福隆宮正殿主祀保生大帝，並同祀[[關聖帝君|關聖帝君]]、吳府千歲，另外還有寄祀全臺白龍庵百壽堂張部駕前孫將軍、斌將軍{{r|聯境}}。後殿則供奉觀音佛祖、註生娘娘與福德正神{{r|聯境}}。

== 交誼 ==
福隆宮在聯境中屬於八協境{{r|聯境}}，與同屬八協境的[[臺南東嶽殿|東嶽殿]]、[[東門大人廟|大人廟]]以及十八境的[[三老爺宮|三老爺宮]]有深厚交誼，其中一間廟宇建醮時，另外三間廟都會「聯普」{{r|大道公}}。
