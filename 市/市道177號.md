'''市道177號（蔦松－二行）'''是位於[[臺灣|臺灣]][[臺南市|臺南市]]的一條市道級公路。北起[[臺南市|臺南市]][[永康區|永康區]]蔦松，南至臺南市[[仁德區|仁德區]]二行，全長共計15.559公里<ref name="motc">{{cite web |title=交路字1070412504號：公告調整臺南市轄內市、區道公路路線 |url=https://odmdoc.motc.gov.tw/IFDEWebBBS_MOTC/ExternalBBS.aspx?ThirdDocId=107RD05527 |publisher=中華民國交通部 |accessdate=2018-10-26 |date=2018-10-25}}</ref>。有一條支線。

== 路線沿革 ==
'''縣道177線'''原為麻豆－赤崁，北起[[臺南縣|臺南縣]][[麻豆區|麻豆鎮]]，南至[[高雄縣|高雄縣]][[梓官區|梓官鄉]]赤崁，全長59.8公里。
; [[臺南縣|臺南縣]]
* [[麻豆區|麻豆鎮]]：麻豆0.0k（{{Jct|country=TWN|CR2|171}}叉路）→民權路→[[麻善大橋|麻善大橋]]
* [[善化區|善化鎮]]：善化7.4kk（{{Jct|country=TWN|CR2|178}}共線起點）→善化8.0kk（{{Jct|country=TWN|CH|178}}共線終點）
* [[新市區_(台灣)|新市鄉]]：新市15.3k（{{Jct|country=TWN|PH|1}}叉路）
* [[新化區|新化區]]：新化19.2k（{{Jct|country=TWN|PH|20}}叉路）
* [[關廟區|關廟鄉]]：關廟28.3k（{{Jct|country=TWN|CR2|182}}共線起點）→關廟28.4k（{{Jct|country=TWN|CR2|182}}共線終點）→南高縣界38.5k

; [[高雄縣|高雄縣]]
* [[阿蓮區|阿蓮鄉]]：阿蓮39.9k（{{Jct|country=TWN|CR2|184}}叉路）
* [[岡山區|岡山鎮]]：岡山50.1k（{{Jct|country=TWN|PH|1}}叉路）→岡山50.5（{{Jct|country=TWN|PH|1}}叉路）
* [[梓官區|梓官鄉]]：梓官56.2k（【高30】鄉道叉路）→大舍甲58.0k（{{Jct|country=TWN|PH|17}}叉路）→赤崁59.8k（'''終點'''，【高23】鄉道叉路）

原有一支線'''縣道177甲線 蔦松－歸仁'''。後縣道177號與[[縣道173號|縣道173號]]之鹽水至麻豆段整編為{{Jct|country=TWN|PH|19a}}，原支線'''縣道177甲線 蔦松－歸仁'''改編為'''縣道177號'''。

2013年全線解編，改制為'''市道177號'''。

2018年10月25日，[[中華民國交通部|中華民國交通部]]公告調整臺南市轄內市、區道公路路線 <ref name="motc"/>：
* 原'''市道177號'''0k-2.690k路段調整，起點調整至[[台1線|台1線]]與永大路口，路線往南沿永大路至永明街口銜接原'''市道177號'''（里程0k-1.952k，本路段短鍊0.738公里）。原行經永康市區路段解編後改為市區道路，移由臺南市永康區公所接養。
* 原'''市道177號'''3.911k-11.715k路段調整，自公園路以南路段，路線改經[[永大路|永大路]]、中正路，於中正路與土庫路口處，繼續沿原區道南147線往南至仁德二行銜接台1線止（里程3K.911k〜16.297k，本路段長鍊4.582公里）。原路段改編為新'''市道177甲線'''，仍由臺南市政府管養。
* 路線調整後，新路線名稱為'''蔦松－二行'''（原路線名稱為'''蔦松－歸仁'''），原總里程為11.715公里，調整後總里程為15.559公里（長鍊3.844公里）
