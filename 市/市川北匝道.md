'''市川北匝道'''（平假名：いちかわきたランプ）是位於[[兵庫縣|兵庫縣]][[神崎郡|神崎郡]][[市川町|市川町]]的[[播但連絡道路|播但連絡道路]]之[[交流道|匝道]]。此匝道只設和田山方向出口與姬路方向入口。

== 連接道路 ==
* [[國道312號|國道312號]]

== 歷史 ==
* 1975年11月1日 - [[播但連絡道路|播但連絡道路]]市川北匝道至[[福崎北匝道|福崎北匝道]]之間開通，此交流道也同時啟用。
* 1982年9月28日 - 播但連絡道路[[神崎北匝道|神崎北匝道]]至市川北匝道之間開通。

== 交流道周邊 ==
