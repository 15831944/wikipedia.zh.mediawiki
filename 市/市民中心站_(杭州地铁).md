'''市民中心站'''是[[杭州地铁4号线|杭州地铁4号线]]的一个站点，位于[[杭州|杭州]][[钱江新城|钱江新城]][[富春路|富春路]][[杭州市民中心|市民中心]]。於2007年开建。为杭州地铁4号线与[[杭州地铁7号线|7号线]]的换乘站。该站共有11个出入口，是出入口最多的一个车站。

==车站结构==
市民中心站位于解放东路富春路口，4号线车站主体位于富春路下穿市民中心箱涵和新塘河之间。车站总长346.45m，宽度22.9~26.8m，其中6m宽度位于新塘河床下。<ref>{{cite web |title=【图文】杭州地铁市民中心地铁站施工技术 |url=https://wenku.baidu.com/view/ac58fae1551810a6f5248667.html}}</ref>

===站厅===
市民中心站站厅位于地下二层，票务服务、自动售票机以及安检进站在该层完成。站厅设有两面艺术墙，均由杭州市雕塑院完成。一副名为《春华秋实》，由金属制成的树干及树枝立于石材背景之前，寓意杭州地铁网络四通八达，也有祝福社会繁荣昌盛的寓意；另一幅名为《欢乐颂》，由杭州市雕塑院院长林岗制作，是其作品集“大音希声”中的一个作品。作品取地铁站附近的[[杭州大剧院|杭州大剧院]]之特点，通过浮雕和圆雕结合的方式，描绘了载歌载舞的场景<ref>{{cite news|title=杭州地铁4号线市民中心站 为什么有幅"星空图"？|url=http://hznews.hangzhou.com.cn/chengshi/content/2015-01/29/content_5630357_2.htm|publisher=杭州网|date=2015-01-29}}</ref>。除此之外，站厅天花板上设有一片长60米，由[[LED|LED]]灯构成的蔚蓝星空。<ref >{{cite web|url=http://z.hangzhou.com.cn/2015/dt4h/content/2015-01/07/content_5626342.htm | language= | title=杭州地铁4号线首通段最美的两个站：市民中心站 江锦路站 |publisher=杭州网 |date=2015年1月7日 |accessdate= }}</ref>
===站台===
市民中心站4号线站台位于地下三层，设有一个[[岛式站台|岛式站台]]，可乘坐[[杭州地铁4号线|4号线]]前往沿途各站。与站厅一样，站台层的吊顶亦设有长40米，由LED灯组成的星空图。星空图由一家深圳的装修公司提出，以车站所在的[[钱江新城|钱江新城]]“日月同辉{{NoteTag|“日”象征[[杭州国际会议中心|杭州国际会议中心]]，“月”象征[[杭州大剧院|杭州大剧院]]。}}但缺少星空元素”为灵感，通过星星勾勒出[[钱江一桥|钱江一桥]]、[[六和塔|六和塔]]、[[杭州市民中心|市民中心]]、[[杭州奥体博览城|奥体中心]]等杭州地标建筑以及城市天际线为原型的“[[星座|星座]]”，寓意杭州发展欣欣向荣<ref>{{cite news|title=地铁4号线市民中心站 为什么有一幅“星空图”？|url=http://hzdaily.hangzhou.com.cn/dskb/html/2015-01/29/content_1891313.htm|date=2015-01-29|publisher=杭州网}}</ref>。

== 历史 ==
[[File:20131013建设中的市民中心站.jpg|thumb]]
*2007年初，4号线车站开工。
*2007年7月6日，4号线车站基坑开挖。

*2014年11月3日，4、7号线换乘通道开工<ref>{{cite web |title=杭州地铁4、7号线换乘节点施工 7号线要建了？_新蓝网 |url=http://n.cztv.com/zhejiang/446093.html?clearcache=1 |website=新蓝网}}</ref>

*2018年6月26日，7号线车站开工。<ref>{{cite web |title=宏润杭州地铁7号线市民中心站正式开工 |url=http://www.chinahongrun.com/index.php?m=content&c=index&a=show&catid=22&id=6960 |website=宏润建设集团股份有限公司}}</ref>

== 未来发展 ==
[[File:20190420杭州地铁7号线市民中心站工地.jpg|thumb]]
[[File:20181011市民中心站7号线预留入口.jpg|thumb]]内]]
未来[[杭州地铁7号线|杭州地铁7号线]]将经过本站，7号线车站主体位于解放东路上，与4号线成L字换乘。7号线市民中心站为地下三层岛式车站，顶板覆土约3.4m；车站主体基坑长度297m、标准段外包宽度21.7m，在站台东侧设有一道顺向渡线。

== 事故 ==
2014年7月31日上午，江锦路站至市民中心站区间右线盾构机进洞时，工作人员现场发现洞门左上方出现渗漏水，新塘河河水水位增高。上午11时，新塘河围堰内的积水突然消失，进入市民中心站，随之围堰倒塌，河水迅速灌入地铁车站内，导致市民中心站内积水深度6m，同时将市民中心的P2地下停车场和城星路-钱江路区间淹没。事故未造成人员伤亡。<ref>{{cite web|url=http://zj.sina.com.cn/news/d/2014-08-01/0743212276.html | language= | title=杭州地铁4号线工地塌方 未对1号线和2号线带来影响 |publisher=杭州网 |date=2014年8月1日 |accessdate= }}</ref>
