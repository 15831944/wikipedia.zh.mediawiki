'''十一代目市川海老藏'''（{{bd|1977年|12月6日|||Ichigawa Ebizo 11}}）本名堀越寶世（舊名堀越孝俊），是[[日本|日本]][[歌舞伎|歌舞伎]]演員，出身歌舞伎世家，祖父為[[市川團十郎_(11代目)|市川團十郎 (11代目)]]。父親為[[市川團十郎_(12代目)|市川團十郎 (12代目)]]。妻子為[[主播|主播]]、演員、主持人[[小林麻央|小林麻央]]。血型是AB型。除了歌舞伎外也有影視演出，[[2003年|2003年]]曾主演[[日本放送協会|NHK]][[大河劇|大河劇]]《[[武藏MUSASHI|武藏MUSASHI]]》。電影則常與[[三池崇史|三池崇史]][[導演|導演]]合作。

== 年表 ==
* 從幼稚園到高中都讀青山學院，高中轉學至[[堀越高等學校|堀越高等學校]]，[[1997年|1997年]]畢業。
* [[1985年|1985年]]，[[襲名|襲名]]為七代目市川新之助。
* [[1994年|1994年]]，於[[NHK|NHK]]演出《花の乱》中的[[足利義政|足利義政（青年時代）]] 。
* [[2004年|2004年]]，襲名為11代目市川海老蔵，第一次用這個名字是同年5月於歌舞伎座演出《[[助六由緣江戸櫻|助六由緣江戸櫻]]》的花川戸助六與《[[暫|暫]]》的[[鎌倉景正|鎌倉権五郎]]。此名沿用至今。
* [[2003年|2003年]][[2月|2月]]，未婚生女（未公開女方與女兒姓名）。
* [[2007年|2007年]][[3月|3月]]，與父親於巴黎首度公演。
* [[2009年|2009年]][[5月23日|5月23日]]，第一次演出民放電視台的連續劇：[[TBSテレビ|TBS]]《[[MR.BRAIN|MR.BRAIN]]》，也是第一次演現代劇。
* [[2010年|2010年]][[3月3日|3月3日]]，與[[小林麻央|小林麻央]]結婚。
* 2010年，於[[西麻布|西麻布]]的鬥毆事件造成頭部、顔面負傷，由妻子報警處理，其後與父親舉行謝罪記者會。海老藏歌舞伎演出無期限停演。
* [[2011年|2011年]][[7月2日|7月2日]]，於新橋演舞場「七月大歌舞伎」舞台復歸。
* 2011年[[7月25日|7月25日]]，妻子麻央生下長女麗禾。
* [[2013年|2013年]][[2月3日|2月3日]]，父親團十郎過世。海老藏成為市川宗家當主家督。
* 2013年[[3月22日|3月22日]]，妻子麻央生下長男勸玄。
* [[2014年|2014年]][[1月30日|1月30日]]，成立官方部落格，每天更新十次以上。
* [[2015年|2015年]][[2月4日|2月4日]]，在官方部落格宣布本名從堀越孝俊改為堀越寶世（日語的發音一樣）。
* 2015年[[7月29日|7月29日]]起擔任[[千葉縣|千葉縣]][[成田市|成田市]]觀光大使「御案内人」。
* [[2016年|2016年]][[6月9日|6月9日]]，於記者會表示妻子麻央罹患乳癌並已治療了一年多。
* [[2017年|2017年]][[6月22日|6月22日]]，麻央去世，死前最後一句話是對海老藏說「愛してる」（我愛你）。

== 軼聞 ==
* 學生時代是籃球社的，籃球打得很好。會游泳，有開船執照。喜歡看相撲。似乎會彈一點鋼琴。
* 與[[伊藤英明|伊藤英明]]是超級好朋友。
* [[吉卜力|吉卜力]]的粉絲。
* 年輕時一個月吃五十次壽司、菸酒不忌、一天可抽高達四包菸，直到2004襲名海老藏有所覺悟，戒除了這些習慣。婚後才又開始飲酒食肉。
* 積極推動復活上演成田屋家藝[[歌舞伎十八番|歌舞伎十八番]]。
* 與[[尾上松緑_(4代目)|四代目尾上松緑]]、[[松本幸四郎_(10代目)|十代目松本幸四郎]]が、[[松本紀保|松本紀保]]（七代目染五郎の姉）・[[松隆子|松隆子]]有親戚關係。
