'''绒啄木鸟'''（[[拉丁文|拉丁文]][[学名|学名]]：'''''Dryobates pubescens'''''；[[英文|英文]]名：'''Downy  woodpecker'''）是[[啄木鸟科|啄木鸟]]的一种，也是[[北美洲|北美洲]]体型最小的啄木鸟。

== 描述 ==
[[File:Downy_Woodpecker02.jpg|左]]
绒啄木鸟的成鸟是北美洲啄木鸟中体型最小的，但在世界范围内有许多体型更小的啄木鸟物种，尤其是[[:Category:姬啄木鸟属|姬啄木鸟]]。该物种体长在14至18厘米之间，翼展在25至31厘米之间，体重在20至33克之间。其标准测量数据如下：翼宽8.5–10 cm，尾长4–6 cm，喙长1–1.8 cm，[[鸟类生理解剖学|跗蹠]]长1.1–1.7 cm。绒啄木鸟翅膀基部主要为黑色，而其背部、喉部及腹部为白色，翅膀上亦有白色斑点。在其眼上与眼下各有一个白色条带。其尾羽大部分为黑色，但外侧为白色且有黑纹。成年雄鸟头部后方有红斑，而幼鸟整个头部后方均为红色。

绒啄木鸟在羽毛色型上与体型更大的{{Tsl|en|Hairy woodpecker|毛啄木鸟}}（暂译）大致相同，但前者可以通过白色尾羽上的黑色斑点及喙的长度与后者区分开。绒啄木鸟的喙比头短，而毛发啄木鸟的喙与头几乎等长。

绒啄木鸟可发出多种[[鸟鸣|鸣声]]，包括短促的''pik''声。该鸟种可以通过''pik''声被识别，两声之间大约间隔半秒（至少应听到四声）。其嘎嘎声则是短促的爆发声，类似于弹跳球，而HAWO声则是相同振幅但更短促的爆发声。像其他啄木鸟一样，它们也将[[喙|喙]]啄入树木中，产生啄木声（听上去像四下敲击）。与其他北美洲的啄木鸟相比，其敲击声频率更缓慢。

== 分类 ==
虽有许多相似之处，绒啄木鸟与毛啄木鸟（暂译）并非近缘物种，而是在不同的[[属|属]]中；两者在外形上的相似是[[趋同演化|趋同演化]]的显著例子。它们如此演化的原因仍不能被确切地解释；但其可能与取食大小不同的食物并且不具有很强的生态竞争相关。包括{{Tsl|en|American Ornithological Society|美国鸟类学会}}在内的一些生物分类学机构继续把这个物种放在{{Tsl|en|Picoides|啄木鸟属}}中。

== 生态与行为 ==
{{listen|filename=Picoides pubescens drumming Roberts Bird Sanctuary, Minnesota.ogg|title=啄木声|description=一只在[[明尼苏达州|明尼苏达州]]的绒啄木鸟发出啄木声，另一只绒啄木鸟在远处发出啄木声作为回应，此外还能在背景听到其它鸟的鸣叫声。|pos=left}}
绒啄木鸟原生于北美的林地，且主要为[[落葉植物|落叶林]]。其分布范围包括[[美国|美国]]和[[加拿大|加拿大]]的大部分地区，但并不包括西南部的[[沙漠|沙漠]]和北部的[[凍原|苔原]]。该物种大多数为永久性[[鸟类迁徙|留鸟]]，但北方的个体亦可能会向南迁徙；[[山区|山区]]的个体亦可能会迁徙到海拔较低的地区。

绒啄木鸟在由其一对配偶从树干或主枝上凿出的树洞上筑巢。 在[[冬季|冬季]]，它们栖息在树洞里。绒啄木鸟在树上觅食，在[[夏季|夏季]]凿掘树皮表面，在[[冬季|冬季]]则凿掘得更深。它们主要以昆虫为食，但亦以种子和浆果为食。它们是{{Tsl|en|European corn borer|欧洲玉米螟}}的天敌，而该[[蛾|蛾类昆虫]]每年使美国[[农业|农业界]]因[[玉米|玉米]]减产及玉米螟数量控制而总共损失花费超过10亿[[美元|美元]]。绒啄木鸟经常可以在[[市郊|市郊]]的后院中的成熟的树木上被[[人|人]]找到，特别是在冬季。在这些地方，它们可以以网状喂鸟器提供的动物油脂和带壳[[花生|花生]]为食。
