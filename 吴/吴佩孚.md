'''吳佩孚'''（{{bd|1874年|4月22日|1939年|12月4日|吴佩孚}}），字'''子玉'''，[[山东省|山东省]][[蓬莱县|蓬莱县]]人。晚清[[秀才|秀才]]，[[北洋军阀|北洋军阀]]中曾經為實力最雄厚的[[軍閥|軍閥]]之一，並担任[[直系军阀|直系军阀]]的首领，官至[[直隶|直]][[山东省|鲁]][[河南省|豫]][[巡阅使|巡阅使]]。

== 早年經歷 ==
[[清|清]][[同治|同治]]十三年（1874年）3月7日，吴佩孚出生於[[山东省|山东省]][[登州府|登州府]]的商人家庭，據聞佩孚出生時，其父夢見戚繼光走進了家裡，所以就以戚繼光的號佩玉，給兒子取名佩孚<ref>[http://www.lszj.com/news/18834.html “封候非我意，但愿海波平”——一代名将​戚继光​_历史之家]</ref>，[[秀才|秀才]]出身。

1898年投筆從戎，投效[[天津|天津]][[淮軍|淮軍]][[聶士成|聶士成]]部隊，入[[北洋武备学堂|北洋武备学堂]]开平班学习[[步兵|步兵]]科。1902年入[[保定陸軍軍官學校|保定陆军速成学堂]]学习测绘科。

1904年[[日俄戰爭|日俄戰爭]]前夕，清政府表面中立，暗中联日拒俄。日本駐清的軍事顧問和[[段芝贵|段芝贵]]從北洋督練公所中選拔出吳佩孚、[[孟恩远|孟恩远]]、[[王怀庆|王怀庆]]等16名从武备学堂毕业的青年軍官加入“倭人侦探队”與日本情報人員秘密組成「中日混合偵探諜報隊」，前往烟台接受日本军官守田利远大尉训练，后到[[旅大|旅大]]等地化装成普通小贩模样刺探俄軍情報。吳佩孚建議[[日軍|日軍]]用多條普通[[漁船|漁船]]日夜不停地對[[俄軍|俄軍]]艦隊進行騷擾、試探，趁其麻痹之際，再把情報船混在漁船中送到日本海軍指揮艦，此建議受到日軍的采納，為後來日軍殲滅俄軍陸上要塞做出了貢獻。10月，吳佩孚被俄軍逮捕，但在被押往[[哈爾濱|哈爾濱]]的途中成功逃脫。日俄戰爭後，吳佩孚因屢次立功，获贈“單光旭日勛章”一枚。<ref>吳佩孚曾助日軍打俄軍 獲贈日本勛章 http://www.81.cn/big5/jmbl/2014-06/16/content_5962383.htm </ref>

1906年，吳佩孚加入[[袁世凯|袁世凯]]建立的[[新军|新军]]，任北洋[[陆军第三鎮|第三鎮]]步隊十一標第一營任督隊官，后又任第三鎮曹錕协炮兵第三标第一营[[管带|管带]]。第二年，曹錕成了第三鎮的統制。吳佩孚隨曹錕駐防吉林時，曹錕要用東北地圖，結果整個第三鎮只有吳佩孚有一張在日俄戰爭時期自己繪制的東北地圖，由此曹錕對吳佩孚開始關注，派吳佩孚到與俄國交界的吉林[[興凱湖|興凱湖]]測量、繪圖<ref>吳佩孚曾助日軍打俄軍 獲贈日本勛章 http://www.81.cn/big5/jmbl/2014-06/16/content_5962383.htm </ref>。1916年，随[[曹锟|曹锟]]入川与西南[[护国军|护国军]]作战，升任[[旅长|旅长]]。1917年7月，任討逆軍西路先鋒<ref>此时冯玉祥任东路先锋。</ref>，參加討伐[[張勳|張勳]]復辟。

1918年[[护法战争|护法战争]]时，任北军[[中央陆军第三师|第三师]][[师长|师长]]，随[[曹锟|曹锟]]入[[湖南|湖南]]作战，攻[[岳州|岳州]]，陷[[长沙|长沙]]，占领[[衡阳|衡阳]]。<!--8月2日，吳忽然通電全國，電文曰，「鬩牆煮豆，何敢言功？」並主張「罷兵共和」。[[段祺瑞|段祺瑞]]生氣的說：“秀才造反啦！” 撤兵時，駐地移交給[[張敬堯|張敬堯]]，軍容齊整，全軍高唱《登蓬萊閣》。 -->

== 巡閱使 ==
[[File:Wu_Peifu_TIME_Cover.jpg|thumb]]》封面上的吴佩孚像]]
1919年通电拥护[[五四运动|五四运动]]，高谈“劳工神圣”。1920年5月从湖南北撤，集结于[[保定|保定]]地区。7月，跟随[[曹锟|曹锟]]参加[[直皖战争|直皖战争]]，与[[奉系|奉系]]合作，击败[[皖系|皖系]]势力，共同参与控制[[北京政府|北京政府]]。9月率第三师进驻[[洛阳|洛阳]]，任直鲁豫巡阅副使。

1921年[[两湖|两湖]]巡阅使兼[[湖北|湖北]]督军[[王占元|王占元]]被川湘联军驱逐，吴佩孚被北京政府任命为[[两湖|两湖]]巡阅使，命萧耀南部占领湖北。允许[[共产主義|共产主義]]人士在其控制范围内的[[京汉铁路|京汉铁路]]、[[陇海铁路|陇海铁路]]等铁路沿线组织[[工会|工会]]。1922年4月，在[[第一次直奉战争|第一次直奉战争]]中拥[[曹锟|曹锟]]，将[[奉系军阀|奉系军阀]]击退[[山海關|山海關]]關外，成为[[北洋军阀|北洋军阀]]中的首要人物，军事实力最强。6月恢复约法，恢复旧国会，倡议南北议和统一。

1922年8月，[[苏联|苏联]]派出驻华全权代表[[越飞|越飞]]抵达[[北京|北京]]，曾致函吴佩孚和孙中山，希望建立合作关系并希望吴佩孚与孙中山合作，吴佩孚也表示“亲俄”<ref>朱正:《两个朋友——李大钊与白坚武》,载《从苏联到俄罗斯》,东方出版社2007年版</ref>。但后来南方战局改观，11月18日越飞写信指责吴佩孚：“孙逸仙对您的怀疑倒是更有根据的”，两天后，吴佩孚回信与越飞决裂并表示“蒙古属于中国”<ref>《孙中山与近代中国的觉醒》，[广州]中山大学出版社2000年版，第261-282页</ref>。越飞在两个多月后与孙中山签署了《孙越宣言》，里面也表示“（苏俄）从无欲在外蒙实施帝国主义政策，或使其脫离中国之意与目的”。

1923年2月镇压[[京汉铁路大罢工|京汉铁路大罢工]]，史称“[[二七惨案|二七惨案]]”。同年4月在洛陽舉行五十大壽，[[維新派|維新派]]首領[[康有為|康有為]]送了一副壽聯，曰：「[[牧野|牧野]]鷹揚，百歲功名才半世；[[洛陽|洛陽]]虎踞，八方風雨會[[中原|中州]]。」吳大喜，賞給[[大洋|大洋]]一千元。同年6月，未能阻止[[曹锟|曹锟]]利用冯玉祥驱逐[[黎元洪|黎元洪]]。10月[[曹錕|曹錕]][[賄選|賄選]]，成為[[總統|總統]]。

1924年9月在中南海四照堂点将，發動[[第二次直奉戰爭|第二次直奉戰爭]]，曹錕任命吳擔任“討逆軍總司令”。10月[[冯玉祥|冯玉祥]]接受[[张学良|张学良]]建议，发动[[北京政变|北京政变]]囚禁曹錕，吳為奉軍及馮叛军所敗。赴[[岳州|岳州]]託庇於湖南軍閥[[趙恆惕|趙恆惕]]。

1925年10月，浙江督辦[[孫傳芳|孫傳芳]]發動[[反奉戰爭|反奉戰爭]]，擁吳出-{征}-。1926年4月，冯玉祥国民军被奉军逼出北京，固守南口，直奉再度联合，击败冯军。吴再次向张作霖呼吁恢复[[法统|法统]]，然未得响应。吳的势力地处[[华中|华中]]，吳的反苏立场使得苏联支持的国民党广州[[国民政府|国民政府]]，把吴列为[[北伐|北伐]]的首要目标。同年夏天吴被[[国民革命军|北伐军]]击溃主力。北伐軍攻佔[[武漢三鎮|武漢三鎮]]後，吳率殘部撤退到[[河南|河南]]，不久又被[[张作霖|张作霖]]排挤，遁入[[四川|四川]]依靠[[楊森|楊森]]，從此一蹶不振。

== 晚年經歷 ==
1931年[[九一八事变|九一八事变]]后居住[[北平|北平]]。其间曾猛烈谴责[[滿州國|伪满洲国]]依附日本，分裂中国，但吴也曾應其舊部[[齐燮元|齐燮元]]的聘请，为齊的“特高顾问”，每月领“车马费”数千元。但到1937年齊擔任日軍[[傀儡|傀儡]]政府「[[華北臨時政府|華北臨時政府]]京津衛戍司令」时与之断绝关系。

日本占领北京后日本驻华特务机关首脑[[土肥原賢二|土肥原賢二]]親自出馬，前來拜會吳佩孚，引诱吳佩孚出山，被吴佩孚拒絶。

1939年12月，吴佩孚于[[北平|北平]]因吃羊肉[[餃子|餃子]]引發牙痛，由一日籍牙醫替其拔牙，4日[[细菌|细菌]]感染，引發[[敗血症|敗血症]]而身亡。但江湖盛傳吳因拒絕日本人的游说，故日本人以[[毒药|毒药]]謀殺，家人得知消息连夜乘船逃往重庆。

== 身後 ==
1939年12月6日，蔣介石向吳家屬致唁電稱讚吳：「精忠許國，大義炳耀，海宇崇欽，流芳萬古……」。<ref>《[[中央日報|中央日報]]》，1939年12月7日而版</ref>12月8日，[[宋美齡|宋美齡]]以個人名義電唁吳家屬。<ref name="說屑，第357頁">安淑萍、王長生著，《蔣介石誄辭說屑》，[[台北|台北]]：[[傳記文學|傳記文學]]出版社，2009年5月15日，第357頁，ISBN 9789578506695</ref>12月9日，[[國民政府|國民政府]]令着追贈吳佩孚陆军[[一级上將|一级上將]]<ref>[[國家圖書館_(中華民國)|國家圖書館 (中華民國)]]：[http://gaz.ncl.edu.tw/detail.jsp?sysid=E1133584 政府公報資訊網--吳佩孚追贈陸軍一級上將令]</ref>，以及[[褒揚令|明令褒揚]]之<ref>{{cite web|publisher=[[國家圖書館_(中華民國)|國家圖書館 (中華民國)]]|url=http://gaz.ncl.edu.tw/detail.jsp?sysid=E1133585|title=政府公報資訊網--國民政府褒揚故陸軍上將吳佩孚令|quote=故陸軍上將吳佩孚，秉性剛直，志行堅貞，曩年整軍經武，卓著聲稱，而其疾惡黜邪，持正不阿，尤有裨於世道人心，瀋陽變起，攖懷國難，恒以精忠自勵，燕京被陷，處境益艱，敵酋肆其逼迫，奸逆逞其簧鼓，威脅利誘，層出不窮，猶能勉全所守，終始弗渝，凛然為國家民族增重，英風亮節，中外同欽。方冀克享遐齡，長資矜式，迺以微疾溘逝，緬懷忠義，痛悼殊深。應予明令褒揚，交軍事委員會從優議卹，特給治喪費一萬元，生平事蹟，存備宣付史館，用示國家軫念賢良，激勵忠貞之至意。此令。主席[[林森|林森]]行政院院長[[孔祥熙|孔祥熙]]}}</ref>。[[蔣介石|蔣介石]]特撥[[國葬|國葬]]費20萬元，在[[重慶|重慶]]為他舉行追悼會致祭，並撰輓聯：「落日黯孤城，百折不回完壯士；大風思猛士，萬方多難惜斯人。」<ref name="說屑，第357頁"/>北平召開追悼會時，蔣又撰輓聯：「三呼渡河，[[宗澤|宗澤]]壯心原未已；一歌見志，[[文天祥|文山]]正氣自常存！」<ref name="說屑，第357頁"/>

== 评价 ==
相傳吳氏捷悟，為秀才後，從[[王紹勛|王紹勛]]參研史事，王講授[[明史|明史]]，提及[[明神宗|明神宗]][[萬曆怠政|怠政三十年]]，王感歎曰：「無為而治兮不必生一神宗三秩」，只是一句評語，並非對聯。但吳佩孚居然立刻應聲[[對仗|對仗]]：「有明之亡矣莫非殺[[東林六君子|六君子]][[東林七賢|七賢]]。」紹勛以為有才識，甚嘉之。

吳佩孚奉命征討南方，卻未被任命為湖南省[[督軍|督軍]]或[[省長|省長]]，心有不甘，息兵[[衡阳|衡阳]]，吳佩孚通電[[段祺瑞|段祺瑞]]語：“鬩牆煮豆，何敢言功？”就此罷兵。[[段祺瑞|段祺瑞]]氣得直跳腳：“秀才造反啦！”

吳氏在[[洛陽|洛陽]]舉行五十大壽壽宴，[[保皇黨|保皇黨]]首領[[康有為|康有為]]送了一副壽聯，曰：「[[牧野|牧野]]鷹揚，百歲功名才半世；[[洛陽|洛陽]]虎踞，八方風雨會[[中原|中州]]。」推崇其軍功，吳大喜，賞給[[大洋|大洋]]一千元。

吳以作戰勇猛為名，全盛時期曾經是為西北[[國民軍|國民軍]]軍閥[[馮玉祥|馮玉祥]]、五省聯軍總司令[[孫傳芳|孫傳芳]]的共主，是'''第一位'''登上[[美國|美國]]《[[時代雜誌|時代雜誌]]》的[[華人|華人]]，時人稱為'''玉帥'''。

吳性格正直清廉，[[董必武|董必武]]称：“吴氏做官几十年，有过几省地盘，带过几十万大兵，他没有积蓄，也没有[[田产|田产]]，相比于同时[[军阀|军阀]]腰缠千百万，总算难能可贵。”

==死因==
关于吴佩孚的死因有三种主要的说法，
* 被日本人[[謀殺|謀殺]]害死{{cn}}。
* 感染[[敗血症|敗血症]]死亡。
* 蒋介石的军统戴笠等人害怕吴最终被日本人说服成为汉奸，为预防起见将其毒死{{cn}}。
