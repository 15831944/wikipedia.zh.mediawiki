'''吴敬琏'''（{{bd|1930年|1月24日|catIdx=W吴}}），中国[[江苏|江苏]][[武进|武进]]人，生于南京。经济学家，工作于[[国务院发展研究中心|国务院发展研究中心]]市场经济研究所。

吴敬琏是邓传理的表哥，邓传理的女儿[[黄圣依|黄圣依]]是他的外甥女。

== 早期經歷 ==
吴敬琏的母亲[[邓季惺|邓季惺]]是著名的女[[报人|报人]]，吴的生父[[吴竹似|吴竹似]]、继父[[陈铭德|陈铭德]]都是《[[新民报|新民报]]》（《[[新民晚报|新民晚报]]》前身）的创办人。

*1946年入[[金陵中学|金陵中学]]，1949年入[[金陵大学|金陵大学]]<ref>{{cite web |url=http://www.jlhs.net/DisplayInfo.jsp?pageID=7635&menuID=303 |title=当代著名经济学家－吴敬琏  |accessdate=2018年4月25日 |publisher=南京市金陵中学网 |work= |language=zh-hans}}</ref>，因病于1950年正式入学，入学后转入[[金陵大学|金陵大学]]经济系]<ref name="wjlresume">{{cite web |url=http://www.drc.gov.cn/zxzj/grjl/343.htm |title=吴敬琏个人简历  |accessdate=2018年4月25日 |publisher=国务院发展研究中心 |work= |language=zh-hans}}</ref>；
*1952年高等院校调整转入上海[[复旦大学|复旦大学]]经济系，1954年毕业；<ref name="wjlresume" />
*1955年到1956年跟[[阿·毕尔曼|阿·毕尔曼]]，研究企业财务和国家财政问题；<ref name="twd912" />
*1956年到1957年参加全国范围的体制调查和体制改革研究；<ref name="twd912" />
*1976年以后，他进入了工作最紧张、成果最丰硕的时期；<ref name="twd912" />
*1982年和别人合写的《关于社会主义经济的计划经济属性和商品经济属性》和《试论社会主义计划经济的调节方式》等文章， 鲜明地提出社会主义经济具有商品经济的属性；<ref name="twd912" />
*1983年至1984年7月一度赴美国[[耶鲁大学|耶鲁大学]]进行调查研究。<ref name="twd912" />

==言論與評價==
吴敬琏為爭議人物，評價兩極，有人稱為为“中国经济学界良心”，也有人稱為“反黨反社會主義的破壞分子”，亦有稱之為風向變色龍。

*曾经提出中国股市“赌场论”<ref>{{cite web |url=http://finance.sina.com.cn/stock/stocktalk/20060308/00002398802.shtml |title=吴敬琏再次重申股市赌场论观点没有变  |accessdate=2018年4月25日 |publisher=新浪财经 |work= |language=zh-hans}}</ref>，但對於網民質疑其他國家的股市是否也全是賭場，未做正面回應。
*在2007年3月参加[[政协|政协]]会议期间，他发表了一系列言论，从反对[[春运|春运]][[铁路|铁路]]票价不上浮到主张城市[[拆迁|拆迁]]不应按市场价补偿等等，他的评论引发了很大争议。
*2013年5月12日，吴敬琏在[[台湾|台湾]]指出“强势政府、动用资源、海量投资的[[中国模式|中国模式]]是强弩之末，这是一条死路、绝路。”<ref>{{cite web |url=http://www.chinese.rfi.fr/%E4%B8%AD%E5%9B%BD/20130512-%E5%90%B4%E6%95%AC%E7%90%8F%EF%BC%9A%E4%B8%AD%E5%9B%BD%E7%BB%8F%E6%B5%8E%E6%A8%A1%E5%BC%8F%E5%BC%BA%E5%BC%A9%E4%B9%8B%E6%9C%AB |title=吴敬琏：中国经济模式强弩之末  |accessdate=2013年5月13日 |publisher=法广中文网 |work= |language=zh-hans}}</ref>

*曾有中国共识网刊文称吴敬琏是个变色龙，[[六四事件|六四事件]]发生前的1988年大赞[[赵紫阳|赵紫阳]]而大贬[[李鹏|李鹏]]，但六四事件刚结束，吴敬琏就写文批判赵紫阳<ref>{{cite web|url=http://www.aisixiang.com/data/58942.html?page=3|title=朱嘉明 陈宜中：中国改革的歧路：朱嘉明先生访谈录（二）(第3页) | accessdate = 2013-01-28 }}</ref>，使李鹏如获至宝，印发中直机关干部阅读，但吴不承认他写过这篇文章。当时，邓小平一句“不争论”使批赵派停止攻击赵紫阳，吴敬琏的檄文也没有扩大散发。<ref>{{cite web | url =http://www.21ccom.net/articles/rwcq/article_201004268453.html | title =赵燕侠：变脸“泰斗”——吴敬琏|date=2010-04-26 | accessdate = 2012-09-22|publisher=[[共识网|共识网]]|archiveurl=https://web.archive.org/web/20160306132705/http://21ccom.net/articles/rwcq/article_201004268453.html|archivedate=2016-03-06|deadurl=yes}}</ref>

*2012年美國資助的中文媒体[[自由亚洲电台|自由亚洲电台]]刊文对吴敬琏的操守提出质疑，文章称吴敬琏只是在表面上提倡[[市场经济|市场经济]]，反对[[权贵资本主义|权贵资本主义]]，但吴敬琏实际上是一个反市场、亲近权贵的人。文章又称，从吴敬琏在中国历届政治权力转变之际，他都能够以最快的速度来调整和转换自己的位置，积极支持新领导，文章将他的行为称之为[[表演|表演]]，称其没有半点学术和追求真理的学者精神。<ref>{{cite web| url =http://canyu.org/n41591c10.aspx| title =晴朗评论：吴敬琏的希望--谈经济学家的职业操守之五| accessdate =2012-09-22| deadurl =yes| archiveurl =https://web.archive.org/web/20150102063151/http://canyu.org/n41591c10.aspx| archivedate =2015-01-02}}</ref>

*[[中國人民大學|中國人民大學]]馬克思主義學院教授周新城，2018年曾在共產黨標誌性刊物《[[求是|求是]]》雜誌上點名批評，認為吴敬琏等「[[張五常|張五常]]之流」本質是「赤裸裸地反黨反社會主義的[[新自由主義|新自由主義]]分子」，總是希望國企往民營化越走越遠，最好有一天沒有國企，卻忽略一個國家有比所謂效率更重要的其他價值，例如政府對社會資源的指揮權、防止外國人掌控要害部門、犧牲公司營收以照顧底層人民等等，非金錢數字能衡量的價值。<ref>[http://www.chinatimes.com/newspapers/20180116000129-260301 人大教授挺國企 倡消滅私有制]</ref>比如美國雖然看似巨大的經濟體與財富，但政府卻掌握的極少，大量財富與權力在私人企業手中，導致美國無法發動[[一帶一路|一帶一路]]之類的大型計劃擴增影響力，動輒還有市政府破產連消防車都買不起，而美國總統也淪為四處在國際上勒索找別國要錢，導致全球地位和形象的下滑。而張五常、吳敬璉等人整天只愛談單一企業的營收、效率云云，忽略國家的其他價值。<ref>[http://www.qstheory.cn/llqikan/2015-09/30/c_1116725690.htm  周新城-鼓吹私有化是违反宪法]</ref>
