'''吴仁宝'''（{{bd|1928年|11月17日|2013年|3月18日|catIdx=W}}）<ref>[http://www.apdnews.com/news/23145.html “中国最有名的农民”吴仁宝辞世] {{webarchive|url=https://web.archive.org/web/20160304141945/http://www.apdnews.com/news/23145.html |date=2016-03-04 }}，亚太日报，2013年3月19日</ref>，汉族，原[[中华人民共和国|中国]][[江苏省|江苏省]][[江阴市|江阴市]][[华西村|华西村]]党委书记，现任村党委书记[[吴协恩|吴协恩]]的父亲。全国[[劳动模范|劳动模范]]，中共十大、十一大、十七大代表，第六、七、八届[[全国人大代表|全国人大代表]]。曾任江苏省[[政协|政协]]常委、江阴县委书记等职。

吴仁宝主政的华西村被官方誉为“中国社会主义[[新农村建设|新农村建设]]的一面旗帜”，其本人也经常受到中国最高领导人的接见。官媒《[[人民网|人民网]]》认为吴仁宝是华西村的“常青树”；也有知名媒体认为“吴仁宝4子支配华西村9成资金 普通村民月入千余”<ref>[http://business.sohu.com/20111125/n326831563.shtml 搜狐财经]</ref>。

因患肺癌医治无效，于2013年3月18日在华西村家中逝世，享年85岁<ref>[http://news.xinhuanet.com/politics/2013-03/18/c_115070098.htm]</ref>。他被称为“中国最有名的农民”，曾将贫穷落后的华西村建设成为中国公认的“天下第一村”。

== 吴仁宝家族 ==

华西集团八大公司负责人中，除杨永昌外，皆为吴仁宝的[[近亲属|近亲属]]。华西村政治体系中，长子吴协东是华西村党委副书记兼华西集团总经理，负责房地产行业；次子吴协德是党委副书记、村委会主任，掌管冶金等行业；三子吴协平是华西村党委副书记；四子吴协恩是现任华西村党委书记；女儿吴凤英亦是党委副书记。吴仁宝孙媳、吴协东儿媳周丽是华西村党委第一副书记；孙女婿孙云南<ref name="a1">{{Cite web |url=http://money.163.com/11/1121/08/7JCDA87400253G87.html|title=华西村村长吴仁宝：掌控首富村|accessdate=2013-03-11|author= |date=2011-11-21 |work= |publisher=网易，来源: 南方人物周刊(广州) |language=Zh-hans }}</ref>也是党委副书记<ref>{{Cite web|url=http://roll.sohu.com/20120212/n334465651.shtml|title=华西村原书记吴仁宝孙女吴芳出任江苏江阴副市长(组图)|accessdate=2013-03-11|author=|date=2012年2月12日|work=|publisher=搜狐网|language=Zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20130323140619/http://roll.sohu.com/20120212/n334465651.shtml|archivedate=2013年3月23日}}</ref>。吴仁宝孙女、吴协东幼女吴芳为江阴市副市长<ref>{{Cite web |url=http://jsnews.jschina.com.cn/system/2012/03/12/012905447.shtml|title=吴芳：“我是透明人，大家看我做每件事”(组图)|accessdate=2013-03-11|author=编辑：娄静 |date=2012-03-12 |work= |publisher=中国江苏网，来源：《新京报》|language=Zh-hans }}</ref>。
