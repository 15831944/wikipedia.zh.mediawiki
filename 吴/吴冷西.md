'''吴冷西'''（{{bd|1919年|12月14日|2002年|6月16日|W吴}}），[[广东省|广东省]][[新会县|新会县]][[古井镇_(江门市)|古井镇]]文楼乡人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中共广东省委|中共广东省委]]书记、[[中华人民共和国广播电影电视部|中华人民共和国广播电影电视部]]部长。

== 生平 ==
1937年，吴冷西前往[[延安|延安]]投奔中国共产党，进入[[抗日军政大学|抗日军政大学]]学习，次年加入中共，并进入[[延安|延安]]马列学院任马列主义研究室研究员。1939年9月，吴进入[[中共中央宣传部|中共中央宣传部]]工作，初任编审科科员，后成为党中央机关刊物《解放》的编辑。1940年一度被[[毛泽东|毛泽东]]指名调往身边，为其编辑《时事问题丛书》。1941年9月，调入中共中央[[机关报|机关报]]《[[解放日报|解放日报]]》，任国际版编辑，后升为国际部主任。

1946年，进入[[新华社|新华社]]，负责领导国际部的工作，次年任总社编委会秘书，主持总编室工作。1949年10月，升任新华社总编<ref>{{cite news |title = 1952年时任新华社社长的吴冷西在处理稿件 |url = http://news.xinhuanet.com/newmedia/2006-11/02/content_5281765.htm |accessdate=2012-02-23 |website = 新华网 }}</ref>。1951年12月，出任新华社社长。1954年，作为中华人民共和国代表团一员参加[[日内瓦会议_(1954年)|日内瓦会议]]。<ref>{{cite web |author1=李亚平 |title=周恩来在日内瓦会议上的外交艺术 |url=http://dangshi.people.com.cn/GB/85038/11921000.html |date=2010年06月21日|website=人民网-中国共产党新闻网 | accessdate=2019-01-11}}</ref>1957年开始又兼任[[人民日报|人民日报]]总编<ref>{{cite news |author=祝华新 |title = 人民日报，叫一声同志太沉重 (2，吴冷西) |url = http://www.chinaelections.org/newsinfo.asp?newsid=176101 |accessdate=2012-02-23 |website = 中国选举与治理 }}</ref>。1964年7月又兼任中共中央宣传部副部长。

[[文化大革命|文化大革命]]期间，吴冷西一度遭到迫害，1972年后方恢复在《人民日报》的工作。1975年，成为《[[毛泽东选集第五卷|毛泽东选集第五卷]]》材料组的领导成员之一，兼任[[国务院政治研究室|国务院政治研究室]]的领导成员。1976年10月，吴冷西与[[李鑫_(中办)|李鑫]]、[[胡绳|胡绳]]和[[熊复|熊复]]一同负责筹建[[中共中央文献编辑委员会|毛泽东著作编辑出版委员会]]，次年任[[中共中央文献研究室|毛泽东著作编辑出版委员会办公室]]副主任兼党委副书记。1980年4月，吴调任[[中共广东省委|中共广东省委]]书记，期间还兼任[[中共中央文献研究室|中共中央文献研究室]]副主任。1982年4月，调任[[中华人民共和国广播电影电视部|广电部]]部长、党组书记<ref>{{cite web |author=张家先 |title = 我与吴冷西的三次神交 |url = http://www.cnr.cn/zggbb/wy/201104/t20110408_507872710.html |accessdate=2012-02-23 |website = 中广网 }}</ref>。1998年[[离休|离休]]，2002年6月16日在北京逝世，享年83岁<ref>{{cite web|title=深切怀念吴冷西同志|url=http://www.xinhui.gov.cn/export/xhds/shiliao/junjie/nw20051106162639.html|publisher=中共新会区委党史办公室|accessdate=2012-02-23}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
