'''吴道子'''又称'''吳道元'''（{{bd|685年||758年||吳道元}}<ref name=王>王伯敏，1982，《中國繪畫史》，人民美術出版社</ref>{{rp|141}}），字'''道子'''，後改名為'''道玄'''，画史尊称“吴生”。阳翟（今[[河南|河南]][[禹县|禹县]]）人。[[中国|中国]][[唐代|唐代]]著名[[画家|画家]]，被称为“百代画圣”。

== 生平 ==

出生年份没有记载<!-- 生年无具体记载。如果有新的发现，请提供来源。 -->。吴道子幼年家境贫寒，初为民间画工，相传曾随[[张旭|张旭]]、[[贺知章|贺知章]]学习书法，年轻时即有画名。后来作了[[山东|山东]][[兖州|兖州]][[瑕丘县|瑕丘]][[县尉|县尉]]，不久即辞职。漫游[[洛阳|洛阳]]，从事壁画创作。时曾为将军[[裴旻|裴旻]]作画，被当时人将[[张旭|张旭]][[草书|草书]]、[[裴旻|裴旻]]舞剑、吴道子作画同时称为“三绝”。

[[开元|开元]]年间以善画被[[唐玄宗|唐玄宗]]召入宫中，历任[[供奉|供奉]]、[[中书省|中书省]][[内教博士|内教博士]]，改为道玄，后官[[宁王|宁王]]友（官名），此后一直为宫廷作画。通过观赏[[公孙大娘|公孙大娘]]舞剑，体会用笔之道。吴道子擅佛教、道教人物画，还有神鬼、人物、山水、鸟兽、草木、楼阁等，尤精于[[佛|佛]][[道|道]]、人物，长于[[壁画|壁画]]创作，据载他曾于[[长安|长安]]、洛阳两地寺观中绘制壁画多达300余堵，奇踪怪状，无有雷同，其中尤以繪于長安[[景云寺|景云寺]]的《地狱变相》闻名于时<ref name=王/>{{rp|143}}。

== 风格 ==
吴道子的绘画具有独特风格。早年笔法较细，风格稠密；中年变为遒劲，[[宋代|宋代]]画家[[米芾|米芾]]形容吴道子笔下圆润似“莼菜条”，滑溜细腻，波浪起伏，点划之间，时见缺落，有笔不周而意周之妙。后人把他和[[张僧繇|张僧繇]]并称“疏体”。吴道子所画人物衣褶，飘飘欲举，线条遒劲，潇洒秀逸，当时人将他和[[北朝|北朝]][[齐|齐]]时代著名画家[[曹仲达|曹仲达]]相提并论，称他们为“吴带当风，曹衣出水”，因为曹仲达画中的人物，衣服常紧裹在躯体上；而吴道子笔下的人物，大袖飘飘，线条流畅。他利用线条的宽窄变化表现物体的凹凸，使线描的画面具有立体感。他喜用焦墨勾线，略施淡彩，自然超出绢素，称为“吴装”。作画线条简练，笔才一二，象已应焉。其山水画作蜀道之景，自为一家；又传曾于大同殿壁画[[嘉陵江|嘉陵江]]三百余里山水，一日而毕。吴道子的绘画对后世影响极大，他被人们尊为“画圣”，被民间画工尊为祖师。[[苏轼|苏轼]]曾称赞他说“画至吴道子，古今之变、天下之能事毕矣”、“出新意于法度之中，寄好理于豪放之外”。

== 作品 ==
吴道子在史籍中富有盛名，但因为他大量的创作是壁画，所以很少有传世作品保留下来，无真迹传世，传至今日的《送子天王图》可能为[[宋代|宋代]]摹本，但已经可以窥见吴道子笔法的精髓。另外还流传有《宝积宾伽罗佛像》、《道子墨宝》等摹本，[[敦煌|敦煌]][[莫高窟|莫高窟]]第103窟的[[维摩经|维摩经]]变图，亦被认为是他的画风。[[徐悲鸿|徐悲鸿]]曾经获得一幅残卷，经他和[[张大千|张大千]]鉴定，认为是吴道子真迹，可能是一幅壁画的草图，暂根据画中人物取名为《[[八十七神仙卷|八十七神仙卷]]》，此幅画气势磅礴，人物闲适秀丽，是中国古代画的精品。
