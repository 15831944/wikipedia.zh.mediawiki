'''吴达镕'''（{{bd|1976年|||}}），自称'''祖古白玛奥色[[仁波切|仁波切]]'''（简称「白玛奥色」，被戲稱為[[泉州仁波切|泉州仁波切]]），[[福建|福建]][[泉州|泉州]][[南安|南安]]人，是一个活跃于香港的[[商人|商人]]和[[佛教徒|佛教徒]]，曾自称为[[四川|四川]][[噶陀寺|噶陀寺]][[直美信雄法王|直美信雄法王]]、[[莫扎法王|莫扎法王]]所认定的的[[活佛|活佛]]。2015年10月4日因出席在[[香港会议展览中心|香港会议展览中心]]为中国大陆演员[[张铁林|张铁林]]举行的[[坐床|坐床]]仪式，后在网上疯传而颇受争议。12月6日，[[四川|四川]][[噶陀寺|噶陀寺]]发表声明，否认[[直美信雄法王|直美信雄法王]]、[[莫扎法王|莫扎法王]]认定吴达镕为活佛<ref name="xjb1">{{cite news|title=白玛奥色法王的人生轨迹：从卖箱包到“活佛”  |url=http://news.sina.com.cn/c/2015-12-06/doc-ifxmhqac0058206.shtml |publisher=新京报|date=2015年12月6日 |accessdate=2015年12月14日 }}</ref>；同月8日，吴达镕在其個人微博發表聲明，称即日起辭去所有職務、頭銜、榮譽和認證，並就公眾對[[藏傳佛教|藏傳佛教]]產生誤解而致歉<ref name="oncc">{{cite news|title=白瑪奧色辭佛教所有職務  |url=http://orientaldaily.on.cc/cnt/news/20151209/00176_042.html|publisher=东方日报|date=2015年12月8日 |accessdate=2015年12月14日 }}</ref>。

==生平==
1976年，吴达镕出生于[[福建省|福建省]][[晋江地区|晋江地区]][[南安县|南安县]]（今[[泉州市|泉州市]][[南安市|南安市]]），幼时为[[基督教徒|基督教徒]]。1984年与父母、妹妹一起迁居[[香港|香港]]。中五毕业后进入一家日资百货公司上班，后升为[[领班|领班]]。1998年，因为在港发展不顺，吴达镕随母亲回到泉州，住在位于[[宝洲路|宝洲路]]的老房子里，面积仅为50平方米。他在泉州中山中路开店子，并从香港进货手机、牛仔衣裤、西服、箱包等物品，开始了他的经商生涯。由于母亲信佛，因此吴达镕受佛教文化感染，开始改信奉[[佛教|佛教]]。吴达镕的母亲曾给他一盒《[[大悲咒|大悲咒]]》磁带，在听了五遍之后可以全文背诵咒文。1998年至2000年初，吴达镕经常往返于香港和泉州，但主要在泉州活动。后来他开始做木雕佛具、佛像的生意<ref name="xjb1"/><ref name="xjb2">{{cite news|title=张铁林的师傅是谁？白玛奥色法王大起底|url=http://news.sina.com.cn/c/2015-12-01/doc-ifxmainy1511331.shtml|publisher=新京报|date=2015年12月1日 |accessdate=2015年12月14日 }}</ref>。

2000年与[[广东|广东]][[普宁|普宁]]籍慈善家[[庄世平|庄世平]]结识，并于2002年共同创立香港佛教文化产业，是一家在香港注册的非牟利慈善机构，为了发展佛教事业，当年吴达镕变卖了所有房产、股票和企业。数年后吴达镕在泉州宝洲路开设分公司，主要经营佛教用品、佛像、工艺品、礼品等，后来倒闭<ref name="xjb1"/><ref name="xjb2"/>。

吴达镕自称有26项头衔，拥有[[院士|院士]]、[[客座教授|客座教授]]、[[荣誉博士|荣誉博士]]等多个学历性质头衔，但这些大学均未经教育部门注册，属于[[野鸡大学|野鸡大学]]；另外，他还拥有华人华侨团体的头衔。在2009年《[[大公报|大公报]]》的专访中，吴达镕自称为“上师”<ref name="xjb2"/>。2013年创办“世贸联合基金总会”，属于[[香港公司注册处|香港公司注册处]]在册的[[担保公司|担保公司]]，并担任中央议会主席。2015年8月18日创立“天泉鼎丰集团有限公司”，主要售卖“大气甘露转化系统仪”，艺人[[胡军|胡军]]、[[张铁林|张铁林]]、[[吕良伟|吕良伟]]、[[王蓉|王蓉]]均为该公司的董事及股东。同年还设立了“阿尔法教育集团”，以[[全脑开发|全脑开发]]教育为主要业务。此外，吴达镕于2014年在[[澳大利亚|澳大利亚]]成立了一家国际金刚乘佛学院，学制一年，专门讲授[[时轮金刚|时轮金刚]]次第论及[[菩萨行论|菩萨行论]]等学科，以英文为教学语言<ref name="xjb1"/><ref name="xjb2"/>。

==争议事件==
2011年，[[联合国|联合国]]友好理事会主席[[诺尔·布朗|诺尔·布朗]]将“联合国友好使者”荣誉证书颁发给吴达镕。诺尔·布朗曾在中国颁发过所谓“联合国友好使者”、“联合国最佳心灵音乐奖”、“联合国最佳华语流行音乐奖”等涉及多领域的奖项。不过[[联合国环境规划署|联合国环境规划署]]北美办公室的工作人员表示，诺尔·布朗已经于1980年代辞职，但无法评价该组织所颁奖项的真实性<ref>{{cite news|title=起底为张铁林举办“坐床大典”的白玛奥色法王：26个头衔 宗教伪装下满是商业利益|url=http://www.guancha.cn/society/2015_12_04_343522_2.shtml|publisher=观察者|date=2015年12月4日 |accessdate=2015年12月14日 }}</ref>。

2015年10月4日，自称为白玛奥色[[法王|法王]]的吴达镕參加了在[[香港会议展览中心|香港会议展览中心]]舉行的「慶祝國慶66週年·護國息災如意寶白瑪奧色法王無量壽佛千供祈福大法會」，在場的有多名香港[[建制派|建制派]]政治人物，例如[[民建聯|民建聯]]主席[[李慧琼|李慧琼]]、[[港區全國人大代表|港區全國人大代表]][[陳勇_(社會工作者)|陳勇]]、[[立法會議員|立法會議員]][[鍾樹根|鍾樹根]]和[[謝偉俊|謝偉俊]]等<ref>{{cite web|title=白瑪奧色法王率三千弟子為祖國祈福 慶賀建國66週年 |url=http://bodhi.takungpao.com.hk/topnews/2015-10/3201557.html |publisher=大公報 |accessdate=2015-12-15 |archiveurl=https://archive.is/20151123202657/http://bodhi.takungpao.com.hk/topnews/2015-10/3201557.html |archivedate=2015-11-23 |language=zh-hant |date=2015-10-05 |deadurl=yes }}</ref>。法會中吴达镕为中国大陆演员[[张铁林|张铁林]]举行了坐床仪式，张铁林自称为白玛铁林，后在网上疯传。不过有[[藏传佛教|藏传佛教]]僧人表示，该仪式完全不符合藏传佛教宗教仪轨，正规的坐床应该在寺庙里，而不是这种场合。此事随即引發网友的讨论，不少佛教界大咖也纷纷撰文抗议张铁林坐床<ref>{{cite web |url=http://ent.qq.com/a/20151129/011064.htm|title=张铁林“坐床”成活佛？网友质疑其受骗|accessdate=2015-11-29 |work=|publisher=腾讯娱乐|language=zh-hans}}</ref><ref>{{cite web |url=http://news.youth.cn/yl/201511/t20151129_7360803.htm|title=张铁林“坐床”成“活佛”视频热传 媒体：瞎搞|accessdate=2015-11-29 |work=中国西藏网 |publisher=中国青年网|language=zh-hans}}</ref>。同日晚，张铁林本人否认在香港举办的活动是传闻的“坐床仪式”，而是佛教徒祈福大法会<ref>{{cite web |url=http://news.sohu.com/20151130/n428796629.shtml|title=媒体：张铁林否认举行活佛坐床仪式 正式皈依|accessdate=2015-11-30 |work=中国网 |publisher=搜狐|language=zh-hans}}</ref>。

==否认与辞职==
2015年12月4日，一位名叫莫扎的[[仁波切|仁波切]]发表声明称从未授予吴达镕任何活佛的名号，且从未举行所谓的坐床、赐冠、赐法衣大典等活动<ref>{{cite news|title=揭秘白玛奥色法王人生轨迹：从风水师到“活佛” |url=http://politics.gmw.cn/2015-12/07/content_17986983_8.htm|publisher=新京报|date=2015年12月7日 |accessdate=2015年12月14日 }}</ref>。12月6日，[[四川|四川]][[噶陀寺|噶陀寺]]发表声明，否认[[直美信雄法王|直美信雄法王]]、[[莫扎法王|莫扎法王]]认定吴达镕为活佛<ref name="xjb1"/>，而四川[[甘孜藏族自治州|甘孜藏族自治州]]民宗委表示，白玛奥色法王创办与佛教有关之机构，实为公司，他的父亲、母亲、妻子均为公司董事<ref>{{cite news|title=甘孜民宗委否认张铁林师父活佛认证|url=http://news.ifeng.com/a/20151204/46530581_0.shtml#_zbs_baidu_bk|publisher=凤凰网|date=2015年12月4日 |accessdate=2015年12月14日 }}</ref>。同月11日，中共四川省委统战部发布了《白玛奥色是“法王”吗？请你看过来！》的文章，文章指出，在四川省宗教部门所登记的所有活佛名单中，并没有白玛奥色或吴达镕的名字<ref>{{cite news|title=四川公布藏传佛教活佛名录 名单中没有白玛奥色|url=http://news.sina.com.cn/c/nd/2015-12-13/doc-ifxmpnqi6399482.shtml|publisher=无界新闻|date=2015年12月13日 |accessdate=2015年12月14日 }}</ref>。

12月8日，吴达镕在其個人微博發表聲明，称即日起辭去所有職務、頭銜、榮譽和認證，並就公眾對佛教產生誤解而致歉<ref name="oncc"/>。同月11日，吴达镕再次在微博發表道歉聲明，即日起與弟子脫離師徒關係<ref>{{cite news|title=白瑪奧色：脫離所有師徒關係|url=http://news.now.com/home/international/player?newsId=161138|publisher=now新闻|date=2015年12月11日 |accessdate=2015年12月14日 }}</ref>。
