'''吴桂苓'''（{{bd|1938年||2016年|6月21日}}）[[北京|北京]]人，[[北京人民艺术剧院|北京人民艺术剧院]]演员。

==生平==
吴桂苓生于1938年。1951年高中毕业于[[北京市第十一中学|北京市第十一中学]]。1958年进入北京人民艺术剧院担任演员直至退休。吴桂苓一生演出舞台作品70多部，扮演主要角色将近40个，出演影视剧作品40多部。吴桂苓在北京人民艺术剧院舞台上先后塑造了《谁是强者》中的郑局长、《上帝的宠儿》中的约瑟夫二世、《公正舆论》中的新闻部长康士坦丁、《吴王金戈越王剑》中的[[伍子胥|伍子胥]]、《哗变》中的查理等角色。他在电视连续剧《[[西游记_(1982年电视剧)|西游记]]》中饰演镇元大仙。吴桂苓是[[中国共产党|中国共产党]]党员、[[中国戏剧家协会|中国戏剧家协会]]会员、北京人民艺术剧院一级演员<ref name=jhw/><ref>{{Cite web |url = http://ent.people.com.cn/n1/2016/0623/c1012-28471249.html |title = 北京人艺演员吴桂苓病逝 享年78岁 |author =  |date = 2016-06-23 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://ent.qq.com/a/20160623/024510.htm|title = 再失老艺术家 吕中丈夫吴桂苓6月21日去世 |author =  |date = 2016-06-23 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

2016年6月21日，吴桂苓在北京病逝。享年78岁。2016年6月25日，吴桂苓同志告别仪式在[[八宝山殡仪馆|八宝山殡仪馆]]菊厅举行<ref>{{Cite web |url = http://mt.sohu.com/20160622/n455708772.shtml |title = 北京人民艺术剧院一级演员吴桂苓去世 享年78岁 |author =  |date = 2016-06-22 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref name=jhw/><ref>{{Cite web |url = http://ent.qq.com/a/20160625/017190.htm#p=1 |title = 吴桂苓遗体告别仪式举行 濮存昕杨立新现身悼念 |author =  |date = 2016-06-25 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。
