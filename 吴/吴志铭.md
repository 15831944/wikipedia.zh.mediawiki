'''吴志铭'''（{{bd|1958年|||}}）籍贯不详，[[中国人民解放军中将|中国人民解放军中将]]。

==生平==
吴志铭是[[北京大学|北京大学]]哲学系80级校友。后为[[中共中央党校|中共中央党校]]哲学专业硕士研究生<ref>董德刚、李述一、吴志铭，经济体制改革与精神文明建设，理论月刊1985年01期</ref>。此后曾在[[胡锦涛|胡锦涛]]时期任[[中央军委主席办公室|中央军委主席办公室]]秘书<ref>{{Cite web |url = http://news.xinhuanet.com/mil/2011-05/12/c_121404119_2.htm |title = 廖毅文：汶川大地震中的新闻突围 |author =  |date = 2011-05-12 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。2007年底，吴志铭任[[中央军委办公厅|中央军委办公厅]]副主任<ref>{{Cite web|url =http://siyang.gov.cn/zwgk/syyw/200904/101399.shtml|title =舍己救人英雄、现役军人吕安建同志遗体告别仪式隆重举行|author =|date =2009-04-10|publisher =江苏省泗阳县人民政府|language =|accessdate =|quote =}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2013年，任[[中国人民解放军军事科学院|中国人民解放军军事科学院]]副政治委员<ref>{{Cite web |url =http://china.caixin.com/2016-02-03/100906787.html  |title =  北京军区副政委高东璐中将转任军科院副政委|author =  |date = 2016-02-03 |publisher =财新网  |language =  |accessdate =  |quote =  }}</ref>。

2014年7月，晋升中将军衔。
