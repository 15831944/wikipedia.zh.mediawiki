'''吴江区'''是中国[[江苏省|江苏省]][[苏州市|苏州市]]所辖的一个[[市辖区|市辖区]]，位于江苏省东南部，全区总面积为1176.68平方公里（不包括所辖太湖水面）。区政府驻[[松陵镇|松陵镇]]。在撤市建区前，吴江为中国经济发达的[[县级行政区|县市]]之一，其综合竞争力[[中国百强县#县域经济论坛评价|2010年列全国百强县（县级市）第二位]]。吴江东临上海，西濒太湖，南接浙江，北依苏州主城区。公元909年建县，1992年撤县设市，2012年撤市设区后成为苏州最大的一个城区。吴江是江南典型的水网地区，七分地三分水。 吴江下辖8个区镇，现有户籍人口82.5万<ref>{{Cite web|url=http://www.zgwj.gov.cn/upfile/template/contentpage/zgwj_zwgknew/zd_nry.aspx?ID=21238|title=阶段性重要统计数据---重点政务信息|accessdate=2018-07-07|work=www.zgwj.gov.cn}}</ref>，暂住人口80.1万人，总人口约162.6万<ref>{{Cite web|url=http://www.zgwj.gov.cn/UpFile/template/contentpage/zgwj_ljwj/csgkDetail.aspx|title=乐居吴江|accessdate=2018-07-07|work=www.zgwj.gov.cn}}</ref>

== 历史 ==
《读史方舆纪要》卷24苏州府吴江县：吴江“在县东门外，即长桥下分太湖之流而东出者，古名笠泽江，亦曰松陵江，亦曰松江”。今称吴淞江。吴以吴江（即吴淞江）为名。<ref>{{Cite book|title=《中国地名语源词典》|last=史|first=为乐|publisher=上海辞书出版社|year=1995-7|isbn=9787532602445|location=|pages=}}</ref>

吴江建县于909年（[[后梁|后梁]][[开平|开平]]三年），当时属于[[吴越国|吴越国]]的苏州，县治就设在松陵。

吴江在[[元代|元代]]（1296年）曾经一度升为州。

[[明|明]][[洪武|洪武]]元年恢复为县。

1726年（[[清|清]]雍正四年）吴江因为人口众多，税收任务繁重，曾分为2县：东部的吴江县和西部的'''震泽县'''。

1912年（民国元年），2县再度合并。

1949年4月29日，解放军占领吴江。

1992年2月17日，吴江撤县设市。

2000年，吴江市辖21个镇：[[松陵镇|松陵镇]]、[[盛泽镇|盛泽镇]]、[[同里镇|同里镇]]、[[菀坪镇|菀坪镇]]、[[屯村镇|屯村镇]]、[[莘塔镇|莘塔镇]]、[[芦墟镇|芦墟镇]]、[[北厍镇|北厍镇]]、[[金家坝镇|金家坝镇]]、[[黎里镇|黎里镇]]、[[平望镇|平望镇]]、[[梅堰镇|梅堰镇]]、[[南麻镇|南麻镇]]、[[八都镇|八都镇]]、[[横扇镇|横扇镇]]、[[七都镇_(吴江区)|七都镇]]、[[庙港镇|庙港镇]]、[[震泽镇|震泽镇]]、[[铜罗镇|铜罗镇]]、[[青云镇_(吴江区)|青云镇]]、[[桃源镇_(吴江区)|桃源镇]]<ref>[http://www.xzqh.org/quhua/32js/0584wj.htm 吴江市行政区划]</ref>。

经过若干次合并后，2004年，全市全面完成撤乡建镇，共设[[松陵镇|松陵镇]]、[[盛泽镇|盛泽镇]]、[[同里镇|同里镇]]、[[震泽镇|震泽镇]]、[[黎里镇|黎里镇]]、[[平望镇|平望镇]]、[[芦墟镇|芦墟镇]]、[[桃源镇_(吴江市)|桃源镇]]、[[横扇镇|横扇镇]]、[[七都镇_(吴江区)|七都镇]]10个[[镇|镇]]，总人口77.75万人。 

2006年，江苏省政府批复同意吴江市[[芦墟镇|芦墟镇]]与[[黎里镇|黎里镇]]两镇合并，成立[[汾湖镇|汾湖镇]]。

2011年12月22日，江苏省政府批准同意撤销横扇镇，将原横扇镇行政区域并入松陵镇。

2012年，撤销[[横扇镇|横扇镇]]并入[[松陵镇|松陵镇]]，从松陵镇中分设滨湖街道。

2012年，“江苏吴江汾湖经济开发区”更名为“江苏省汾湖高新技术产业开发区”。

2012年，国务院国函[2012]102号批复同意撤销县级吴江市，设立苏州市吴江区，以原县级吴江市的行政区域为吴江区的行政区域。吴江区人民政府驻滨湖街道开平路1000号。<ref>{{cite web |url=http://www.xzqh.org/html/2008/0606/5706.html |title=存档副本 |accessdate=2012-08-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20120903031923/http://www.xzqh.org/html/2008/0606/5706.html |archivedate=2012-09-03 }}</ref><ref>{{Cite web|url=http://www.zgwj.gov.cn/UpFile/template/contentpage/zgwj_Xwzxnew/item.aspx?id=120325&p=0|title=撤销县级吴江市设立苏州市吴江区|accessdate=2018-07-07|work=www.zgwj.gov.cn}}</ref>

2012年10月29日，吴江正式撤市设区。

2012年，为配合黎里古镇开发，汾湖镇改为黎里镇。

2013年年底，吴江经济开发区与同里镇合并。<ref>{{Cite web|url=http://news.2500sz.com/news/szxw/2013-11/2_2189111.shtml|title=吴江开发区和同里镇实行“区镇合一”管理体制-名城新闻网|accessdate=2018-07-07|work=news.2500sz.com}}</ref><ref>{{Cite web|url=http://www.zgwj.gov.cn/UpFile/template/contentpage/zgwj_Xwzxnew/item.aspx?id=127382&p=0|title=区镇合一：凸显城市发展叠加优势|accessdate=2018-07-07|work=www.zgwj.gov.cn}}</ref>

2013年1月，吴江滨湖新城更名为吴江太湖新城。

2018年10月，撤销[[滨湖街道|滨湖街道]]、[[松陵镇|松陵镇]]，设立[[松陵街道|松陵街道]]、[[江陵街道|江陵街道]]、[[横扇街道|横扇街道]]<ref name=2018tz/>。

== 地理 ==
吴江区位于江苏省东南部，北纬30°45′36″—31°13′41″，东经120°21′4″—120°53′59″。吴江东接[[上海|上海]]市[[青浦|青浦]]区，南连[[浙江|浙江]]省[[嘉兴|嘉兴]]市和[[桐乡|桐乡]]市，西临[[太湖|太湖]]，北靠苏州市[[吴中区|吴中区]]，东南与[[浙江省|浙江省]][[嘉善|嘉善]]县毗邻，东北和[[昆山|昆山]]市接壤，西南与浙江省[[湖州|湖州]]市交界。

吴江区全境无山，地势低平，自东北向西南缓慢倾斜，南北高差2.0米左右。境内水道纵横、湖荡密布。物产丰富、经济发达，素有古运河畔“鱼米之乡”、“丝绸之府”的美称。 

吴江下辖8个区镇中有1个国家级开发区、2个省级高新区、1个省级旅游度假区。全市面积1176平方公里（不含太湖水面），其中水域面积267.1平方千米。

== 行政区划 ==
吴江区下辖3个街道、7个镇：[[松陵街道|松陵街道]]、[[江陵街道|江陵街道]]、[[横扇街道|横扇街道]]、[[盛泽镇|盛泽镇]]、[[同里镇|同里镇]]、[[平望镇|平望镇]]、[[桃源镇_(吴江区)|桃源镇]]、[[震泽镇|震泽镇]]、[[黎里镇|黎里镇]]、[[七都镇_(吴江区)|七都镇]]。<ref>{{Cite web|url=http://www.zgwj.gov.cn/UpFile/template/contentpage/zgwj_ljwj/xzqhDetail.aspx|title=乐居吴江|accessdate=2018-07-07|work=www.zgwj.gov.cn}}</ref><ref name=2018tz>{{Cite web|url=http://www.zfxxgk.suzhou.gov.cn/sxqzf/szsrmzf/201810/t20181026_1011130.html|title=苏州市人民政府关于吴江区部分行政区划调整的批复|accessdate=2018-10-27|work=苏州市人民政府}}</ref>

吴江开发区（同里镇）：吴江开发区与同里镇实行“区镇合一”管理体制。区域面积176平方公里，常住人口约40万，辖34个行政村，9个社区居委会。区内建有15000多亩的国家级现代农业示范区和面积达17140亩国家级湿地公园。同里古镇保护区为国家5A级旅游景区。吴江开发区先后成为首批国家信息产业基地、首批国家显示器件产业园、国家新型工业化产业示范基地、中国机械装备产业基地。

汾湖高新区（黎里镇）：汾湖高新区位于吴江东部，与黎里镇实行“区镇合一”管理体制。区域面积258平方公里，下辖4个办事处、48个行政村和9个社区居委会，常住人口约25.6万。先后被评为“中国历史文化名镇”、“国家级卫生镇”、“全国发展改革试点城镇”。汾湖是“临沪”经济圈中最具地理优势的区域，作为苏州城区接轨上海的第一门户，主动承接上海辐射，积极融入上海发展。

吴江高新区（盛泽镇）：盛泽南接浙江湖州、嘉兴，北依苏州，东临上海，西濒太湖，堪称中国丝绸第一镇，全国25个经济发达镇行政管理体制改革试点镇之一，总面积150平方公里，城区建成面积45.98平方公里、规划工业产业区60平方公里，下辖8个社区、35个行政村，常住人口约50万。先后被评为“国家级丝绸星火密集区”、“全国乡镇企业示范区”、“中国丝绸名镇”、“中国纺织名镇”。

太湖新城（松陵镇）：吴江太湖新城位于太湖东岸，吴江区的最北部，是吴江区委区政府所在地。总面积200平方公里，常住人口约40万。吴江太湖新城与省级东太湖生态旅游度假区一体两翼、融合发展，拥有温泉养生、游艇休闲、湖鲜美食、黄金沙滩、主题乐园等生态旅游资源，坐拥百里太湖岸线，核心区绿化率达60%，是苏州“最生态”之地。东太湖生态旅游度假区正加快创建国家5A级，将打造成 “环太湖最具生态魅力的旅游度假区”和“环太湖首选的滨水休闲基地”。

七都镇：七都地处太湖之滨，与苏州洞庭东山、西山隔水相望，全镇面积102.9平方公里，常住人口10余万。七都兼具吴风越韵的文化根基和“一级空气二类水”的优美生态环境，拥有长达23公里的绝佳太湖岸线。先后被评为国家卫生镇、全国创建文明村镇工作先进镇、全国环境优美乡镇和全国特色小镇。

桃源镇：地处长三角城市群核心，与乌镇、南浔等古镇相连。镇域面积90.6平方公里，常住人口10.2万。桃源又称水乡森林小镇，拥有4.5万亩森林形成的“天然氧吧”，11.5公里的京杭大运河及其支流纵横交错形成的水网湿地景观，是国家级生态乡镇、全国环境优美镇、国家卫生镇、中国出口服装制造名镇、江苏省历史文化名镇。

震泽镇：吴江区的西大门，距上海90公里、苏州54公里、杭州75公里，处在沪苏浙经济圈中心地带。全镇总面积96平方公里，辖23个行政村，2个街道办事处和5个社区。常住人口约12万。先后获评中国历史文化名镇、全国重点镇、国家AAAA级旅游景区、中国蚕丝之乡、长三角最美乡村和全国特色小镇。

平望镇：平望镇位于苏、浙、沪两省一市的中心,区域面积133.5平方千米，辖2个办事处、21个行政村、7个社区居委会，常住人口约15万。国道、省道、高速路、快速干道贯通镇区，城际铁路、京杭大运河、太浦河在镇郊汇聚，交通优势得天独厚。先后被评为 “国家卫生镇”、“全国环境优美乡镇”、“中国纺织织造名镇”、 “江苏省历史文化名镇”、“苏州特色花卉名镇”。

== 经济 ==
吴江区三大支柱产业为电子信息、丝绸纺织和电缆光缆。三大产业集聚区域（[[吴江经济开发区|吴江经济开发区]]、[[汾湖高新技术产业开发区|汾湖高新技术产业开发区]]和[[盛泽镇|盛泽镇]]）在吴江经济中占有突出地位<ref>[http://zgwj.gov.cn/zjwj/dlwz_new.asp?ID=57 中国吴江——工业和建筑业] {{webarchive|url=https://web.archive.org/web/20080108154735/http://zgwj.gov.cn/zjwj/dlwz_new.asp?ID=57 |date=2008-01-08 }}</ref>。

== 交通 ==

吴江在传统上以水运为主。[[京杭大运河|京杭大运河]]纵贯南北，[[太浦河|太浦河]]横穿东西，可以方便地通达[[长江三角洲|长江三角洲]]各地。[[抗日战争|抗日战争]]前夕，[[苏嘉铁路|苏嘉铁路]]修通，穿越吴江全境，但是在战争期间破坏。现在境内[[苏嘉杭高速公路|苏嘉杭高速公路]]、[[227省道_(江苏)|227省道]]、[[京杭大运河|京杭大运河]]纵贯南北，[[318国道|318国道]]、[[太浦河|太浦河]]、[[沪苏浙高速公路|沪苏浙高速公路]]（吴江段）横穿东西。境内拥有公路总里程2373公里，其中：高速公路98公里。基本建成所有镇10分钟内上高速公路、30分钟内到达市区、过境车辆30分钟内离境的道路交通格局。距离最近的火车站是[[京沪铁路|京沪铁路]][[苏州站|苏州站]]（22公里），距离上海[[上海虹桥国际机场|虹桥机场]]为80公里<ref>[http://zgwj.gov.cn/zjwj/dlwz_new.asp?ID=66 中国吴江] {{webarchive|url=https://web.archive.org/web/20080108152239/http://zgwj.gov.cn/zjwj/dlwz_new.asp?ID=66 |date=2008-01-08 }}</ref>。与上海洋山港和苏州太仓港的距离分别为190公里和105公里，四通八达的水陆交通网把吴江与上海、杭州、苏州等大中城市联成一体。
