'''吴彬'''，字'''文中'''，号'''枝隐头陀'''、枝庵发僧、枝隐庵主，今福建[[莆田|莆田]]人，明朝[[万历|万历]]年间著名山水画家。<ref name="pig " />

== 生平 ==
[[File:Cinq_cents_arhat_(section)_du_peintre_chinois_Wu_Bin,_actif_(1573-1620).jpg|thumb]]
唐代工部屯田员外郎[[吴祭|吴祭]]之後，吳颜可之四子。寓居南京。1573年到1620年（明朝万历），吴彬出任[[中书舍人|中书舍人]]，<ref>《画史汇要》：“吴彬字文仲，闽人。万历时官中书舍人，善山水，布置绝不摹古，皆对真景描写……能大士像亦能人物。”</ref>后因批评[[宦官|宦官]][[魏忠贤|魏忠贤]]而被革职。<ref name="pig " >{{cite web |url=http://news.163.com/10/0510/05/66A36L0000014AED.html |title= 1.69亿元明代莆田画家吴彬的世界纪录|accessdate=2010-05-10 |publisher= 网易|work= |language =zh-cn}}</ref>吴彬作品《十八应真卷》作於1643年，卒年應在1643年。<ref>《晚明變形主義畫家作品展》一書推論其活動年代在
1590-1643年間。</ref>

== 风格 ==
吴彬善于人物山水画，笔法粗壮，独具一格。

== 评价 ==
*《[[画史汇要|画史汇要]]》：“吴彬字文仲，闽人。万历时官中书舍人，善山水，布置绝不摹古，皆对真景描写……能大士像亦能人物。”
*《[[枝隐庵诗集|枝隐庵诗集]]序》：“吾乡吴文中，侨寓白门，名其所居曰‘枝隐庵’。日匡坐其中，诵经礼佛、吟诗作画，虽环堵萧然，而丰神朗畅，意趣安恬，大有逍遥之致。”
*[[南京|南京]]大儒[[顾起元|顾起元]]：“八闽之高士”，“夙世词客，前身画师”。
*[[福建|福建]][[长乐|长乐]][[谢肇淛|谢肇淛]]《[[五杂俎|五杂俎]]》：“……力敌松雪，传之后代，价当重连城矣”。
