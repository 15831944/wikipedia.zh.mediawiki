'''吴淦'''（{{bd|1973年|2月14日|}}），网名：超级低俗屠夫，[[福建|福建]][[福清|福清]]人，维权人士，网络名人。北京锋锐律师事务所行政助理。

“屠夫”吴淦最早闻名于网络缘于[[邓玉娇案|邓玉娇案]]<ref>{{Cite web|url = http://news.sohu.com/20090720/n265342316.shtml|title = “超级低俗屠夫”：因邓玉娇案造就非典型成名|accessdate = |author = 南都周刊|date = 2009-07-20|publisher = 搜狐网}}</ref>，屠夫的参与推动了该案的迅速传播。<ref>{{Cite web|url = http://www.bbc.com/zhongwen/simp/comments_on_china/2015/07/150703_coc_activist_wugan_mode|title = 大家谈中国：论“超级低俗屠夫”吴淦|accessdate = |author = 苏星河|date = 2015-07-03|publisher = BBC中文网}}</ref>吴淦参与过的著名事件有[[福建网民案|福建网民案]]<ref>{{Cite web|url = http://www.rfa.org/mandarin/yataibaodao/tufu-02042010103637.html|title = “屠夫”亲赴福建声援游精佑 网民关注团成立|accessdate = |author = 心语|date = 2010-02-04|publisher = 自由亚洲}}</ref>、浙江[[钱云会案|钱云会案]]<ref>{{Cite web|url = http://chinadigitaltimes.net/chinese/2011/01/%E9%92%B1%E4%BA%91%E4%BC%9A%E6%AD%BB%E4%BA%A1%E8%B6%85%E7%BA%A7%E4%BD%8E%E4%BF%97%E5%B1%A0%E5%A4%AB%E5%B1%B1%E5%AF%A8%E6%80%BB%E7%BB%93/|title = 钱云会死亡超级低俗屠夫山寨总结|accessdate = |author = 屠夫|date = 2011-01-06|publisher = 中国数字时代}}</ref>、沈阳[[夏俊峰案|夏俊峰案]]<ref>{{Cite web|url = http://www.lifeweek.com.cn/2013/1025/42970.shtml|title = 从小贩妻子到沈阳张晶|accessdate = |author = 杨璐|date = 2013-10-25|publisher = 三联生活周刊}}</ref>、[[庆安火车站枪击事件|庆安火车站枪击事件]]<ref>{{Cite web|url = http://www.boxun.com/news/gb/china/2015/05/201505081104.shtml#.VsAZF5MrLsF|title = 庆安徐纯合被击毙事件调查：律师发现家属被培训过|accessdate = |author = 博讯|date = 2015-05-08|publisher = 博讯网}}</ref>等著名事件。

2015年5月20日，吴淦因在南昌抗议江西高院不让参与“乐平冤案”律师阅卷，而被行政拘留10天。5月27号又因涉嫌“寻衅滋事罪”和“诽谤罪”，遭福建警方刑事拘留。<ref name=":0">{{Cite web|url = http://www.rfa.org/mandarin/yataibaodao/renquanfazhi/hc-07032015121143.html|title = 屠夫吴淦被以煽动颠覆国家政权等两罪正式逮捕|accessdate = |author = 高山|date = 2015-07-03|publisher = 自由亚洲}}</ref>

2015年7月3日，燕文薪律师接到厦门检察院电话，告知屠夫以煽动颠覆国家政权和寻衅滋事两罪批准逮捕。<ref name=":0" />

2016年1月28日厦门市公安局通知吴淦已经移交给天津市公安局。<ref>{{Cite web|url = http://www.rfa.org/cantonese/news/lawyer-02022016074414.html|title = 屠夫疑涉709抓捕案转押天津|accessdate = |author = 自由亚洲|date = 2016-02-02|publisher = 自由亚洲}}</ref>

2017年12月26日，被控“煽动颠覆国家政权”等罪，被判处有期徒刑八年、剥夺政治权利五年。<ref>{{Cite news|url=http://www.bbc.com/zhongwen/simp/chinese-news-42481806|title=维权人士吴淦“煽颠”判囚八年|date=2017-12-26|newspaper=BBC 中文网|accessdate=2017-12-26|language=en-GB}}</ref>
