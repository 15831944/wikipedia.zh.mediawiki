'''吳奇'''（{{bd|1955年|3月17日}}），籍贯[[安徽|安徽]][[桐城|桐城]]，出生于安徽[[芜湖|芜湖]]<ref>{{cite web|url=http://www.casad.cas.cn/aca/373/hxb-200906-t20090624_1802115.html|title=院士信息 吴奇|publisher=中国科学院|accessdate=2019-02-18}}</ref>，高分子物理化学家，[[香港中文大學|香港中文大學]]化學講座教授，[[美國物理學會|美國物理學會]]院士，[[中國科學院|中國科學院]]院士。

== 教育背景 ==
*1982 理學士  [[中國科學技術大學|中國科學技術大學]]  
*1987 哲學博士  美國[[紐約州立大學|紐約州立大學]]

== 擔任職位 ==    
*1999-至今 香港中文大學化學講座教授 
*1996-1999 香港中文大學教授  
*1992-1996 香港中文大學講師  
*1989-1992 德國巴斯夫(BASF)公司激光光散射實驗室主管  
*1987-1989 美國紐約州立大學博士後研究員 

== 專業活動及獲取獎項 ==  
*2003 獲中國國家自然科學獎(二等獎) 
*2002 獲中國科學院優秀研究生導師獎 
*2000 獲香港裘槎基金會傑出研究獎 
*1999 獲選美國物理學會 會士 (APS Fellow) 
*1997 獲香港求是基金會傑出青年學者獎 
*1996 獲國家傑出青年基金(總理基金) 
*1989 獲德國洪堡基金會研究獎學金 
*1982 獲郭沫若獎學金
