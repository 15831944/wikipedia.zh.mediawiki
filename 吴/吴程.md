'''吳程'''{{BD|893年||965年|9月14日|catIdx=W}}，[[表字|表字]]'''正臣'''，[[五代十国|五代十国]][[吴越国|吴越国]]大臣，在忠逊王[[钱弘倧|钱弘倧]]、忠懿王[[钱俶|钱俶]]时代担任宰相（丞相）。[[越州|越州]]山阴县（今[[浙江省|浙江省]][[绍兴市|绍兴市]]）人。

== 背景 ==
吳程于[[唐昭宗|唐昭宗]][[景福_(唐朝)|景福]]二年（893年）出生在山陰县。他的祖父吳可信是[[唐朝|唐朝]][[定州|定州]]虞姬唐县令，他的父亲吳蛻在[[唐昭宗|唐昭宗]]大顺年间中[[进士|进士]]。通过[[科举|科举]]后，吳蛻担任鎮東军节度使掌书记，官至右拾遗、[[礼部尚书|礼部尚书]]。<ref name=SGCQ87>《[[十國春秋|十國春秋]]》[[s:十國春秋_(四庫全書本)/卷087|卷87]]</ref><ref>当时绍兴已经被钱镠控制，因为史书提及绍兴的军号是钱镠控制后的鎮東军。《[[資治通鑑|資治通鑑]]》[[s:資治通鑑/卷260|卷260]]</ref>

== 钱镠时代 ==
907年，唐朝被[[朱全忠|朱全忠]]建立的[[后梁|后梁]]取代。钱镠向后梁称臣，被封为吴越王<ref name=ZZTJ266>《[[資治通鑑|資治通鑑]]》[[s:資治通鑑/卷266|卷266]]</ref>。923年，钱镠被后梁封为吴越国王，后来谥号武肃王<ref name=ZZTJ272>《[[資治通鑑|資治通鑑]]》[[s:資治通鑑/卷272|卷272]]</ref>。钱镠在位时，吳程开始担任校書郎，钱镠承制任命他为检校戶部員外郎。[[宝正|宝正]]年间（926年—931年），钱镠想要为女儿选个丈夫。他从吳程、孟粲、于葆三人中选中了吳程。吳程升任金部郎中。钱镠认为吳程有行政能力，让吳程提举诸司公事。<ref name=SGCQ87/>

== 钱元瓘时代 ==
钱镠在932年去世，他的儿子[[钱元瓘|钱元瓘]]即位为文穆王<ref name=ZZTJ277>《[[資治通鑑|資治通鑑]]》[[s:資治通鑑/卷277|卷277]]</ref>。钱元瓘承制任命吳程为職方郎中、觀察支使、节度判官<ref name=SGCQ87/>。[[后晋|后晋]]天福年间，钱元瓘让他的儿子錢弘儇遥任睦州刺史，由吳程知州事。<ref name=SGCQ87/>

== 后三王时代 ==
忠献王[[钱弘佐|钱弘佐]]时代（941年—947年），吳程事迹不详。忠逊王[[钱弘倧|钱弘倧]]时（947年），任命吴程判西府院事，西府即鎮海军，治所在吴越的国都錢塘县（今[[浙江省|浙江省]][[杭州市|杭州市]]），随后任命他为[[丞相|丞相]]<ref name=SGCQ87/>。不久，吴越福州戍将[[鮑修讓|鮑修讓]]杀威武军节度使（治福州，今[[福建省|福建省]][[福州市|福州市]]）[[李孺赟|李孺赟]]。钱弘倧以吴程知威武军节度事<ref name=ZZTJ287>《[[資治通鑑|資治通鑑]]》[[s:資治通鑑/卷287|卷287]]</ref>，不久任命他为[[威武军节度使|威武军节度使]]<ref name=SGCQ87/>。

950年，[[南唐|南唐]]永安（治今福建省[[南平市|南平市]][[建瓯市|建瓯市]]）留后[[查文徽|查文徽]]认为吴越已经放弃了福州，派遣剑州刺史陈诲带领水军沿闽江而下，自己率领步兵、骑兵为后继。吴程听说查文徽临近，吴程派遣数百人出城假装迎接。查文徽不作防备，吴程领兵出击，南唐军队大败，查文徽从马上摔下来，被福州人抓获，士卒死亡万人。吴程解送查文徽到钱唐，吴越王[[钱弘俶|钱弘俶]]将查文徽作为战利品在祖宗五庙举行献俘礼，然后释放了他<ref name=ZZTJ289>《[[資治通鑑|資治通鑑]]》[[s:資治通鑑/卷289|卷289]]</ref>。钱弘俶召吴程回钱唐，与[[元德昭|元德昭]]一起担任丞相，兼掌屯田榷酤事<ref name=SGCQ87/>。

956年，[[后周|后周]]世宗[[柴荣|柴荣]]攻打南唐，要求吴越从东南方攻打南唐。[[苏州|苏州]]营田副使陳滿进言说：“周军南下征伐，唐举国惊乱，[[常州|常州]]没有防备，容易攻取。”吴程请求立即发兵采用陈满的计策。元德昭认为南唐是大国，不可轻视。请暂且等待。吴程再三争辩，钱弘俶听从了吴程的建议。[[二月二十|二月二十]]日，钱弘俶派吴程督领衢州刺史鲍修让、中直都指挥使罗晟奔赴常州。吴程对将士们说：“元丞相不愿意出兵。”将士们恼怒，有流言说要袭击元德昭。钱弘俶把元德昭藏匿在自己府中，下令逮捕散布流言的人，叹道：“方出师而士卒欲击丞相，不祥甚哉！”<ref name=ZZTJ292>《[[資治通鑑|資治通鑑]]》[[s:資治通鑑/卷292|卷292]]</ref>

吴程进攻常州，攻破常州外城，抓获南唐常州团练使趙仁澤，押送到钱唐。吴程和鲍修让、罗晟在福州就有矛盾，这时，吴程压制刁难他们，二人都有怨恨。南唐龙武都虞候柴克宏率军到达常州。声称只是护送唐主[[李璟|李璟]]派遣的[[中书舍人|中书舍人]]喬匡舜出使吴越。吴越巡逻士兵将情况报告，吴程说：“两国交战，使者可以在其间来往，不可随便怀疑。”南唐军登上岸，直逼吴越营寨，罗晟不拼力作战，吴程仅以身免。吴程到达钱唐，钱弘俶削夺他的一切职务。<ref name=ZZTJ293>《[[資治通鑑|資治通鑑]]》[[s:資治通鑑/卷293|卷293]]</ref>

965年，吴程去世<ref name=SGCQ81>《[[十國春秋|十國春秋]]》[[s:十國春秋_(四庫全書本)/卷081|卷81]]</ref>，时年七十三岁，钱弘俶追复前官，[[谥号|谥号]]'''忠烈'''<ref name=SGCQ87/>。
