'''吴媚媚'''（原名'''张媚娟'''，{{bd|1918年||2012年|2月26日|catIdx=Wu吴媚媚}}）[[江苏|江苏]][[南京人|南京人]]，中国著名[[滑稽演员|滑稽表演艺术家]]。<ref name="fhxm " >{{cite web |url= http://ent.ifeng.com/idolnews/mainland/detail_2012_02/28/12829584_0.shtml|title= 95岁表演艺术家吴媚媚逝世 告别仪式本周五举行|accessdate= 2012年|work=|publisher=凤凰网 |language = zh-hans}}</ref><ref name="hqmx ">{{cite web |url=http://ent.huanqiu.com/star/mainland/2012-02/2477188.html |title=著名滑稽表演艺术家吴媚媚因病去世 享年95岁 |accessdate=2012年 |work= |publisher=环球网 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20120310115011/http://ent.huanqiu.com/star/mainland/2012-02/2477188.html |archivedate=2012-03-10 }}</ref> 吴媚媚通晓中国六七种[[方言|方言]]，如[[苏州话|苏州话]]、[[苏北话|苏北话]]、[[无锡话|无锡话]]、[[山东话|山东话]]、[[上海话|上海话]]等。<ref name="tmx ">{{cite web|url=http://www.dfdaily.com/html/150/2012/2/27/749678.shtml|title=吴媚媚|accessdate=2012年|work=|publisher=东方早报|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20120301063628/http://www.dfdaily.com/html/150/2012/2/27/749678.shtml|archivedate=2012-03-01}}</ref> 吴媚媚被誉为“[[江南|江南]][[赵丽蓉|赵丽蓉]]”。<ref name=" cmsxz" >{{cite web |url=http://www.cqcb.com/cbnews/gngjnews/2012-02-27/626987.html |title="江南赵丽蓉"之称的吴媚媚上海去世 |accessdate= 2012年|work=|publisher=晨网新闻 |language = zh-hans}}</ref>

== 简介 ==
1918年，吴媚媚出生在[[江苏省|江苏省]][[南京市|南京市]]。<ref name="hqmx " /><ref name=" cmsxz" /> 7岁开始演出，13岁师从“笑舍”[[文明戏|文明戏]][[艺人|艺人]][[吴一笑|吴一笑]]。<ref name="fhxm " /><ref name="hqmx " /><ref name=" cmsxz" />

16岁，吴媚媚参加《[[啼笑因缘|啼笑因缘]]》的演出，并分别扮演[[沈凤喜|沈凤喜]]、[[何丽娜|何丽娜]]两个女子。<ref name="fhxm " /><ref name="hqmx " /><ref name=" cmsxz" />

20世纪40年代，吴媚媚在[[笑笑剧团|笑笑剧团]]《[[秋海棠|秋海棠]]》中出演[[罗湘绮|罗湘绮]]、[[梅宝|梅宝]]两个角色，同时又在[[华亭剧团|华亭剧团]]《[[小山东到上海|小山东到上海]]》中出演[[秀娟|秀娟]]。<ref name="hqmx " /> 后来，她加入[[艺锋滑稽剧团|艺锋滑稽剧团]]，创造了二十多部滑稽戏，如：《[[红灯花轿|红灯花轿]]》、《[[结发夫妻|结发夫妻]]》、《[[毛头姑娘|毛头姑娘]]》、《[[烟花女子告阴状|烟花女子告阴状]]》。<ref name="hqmx " /><ref name=" cmsxz" />

1958年，吴媚媚加入[[蜂蜜滑稽剧团|蜂蜜滑稽剧团]]（今[[上海滑稽剧团|上海滑稽剧团]]），参与演出了《[[阿大阿二|阿大阿二]]》、《[[王老虎抢亲|王老虎抢亲]]》。<ref name="hqmx " /><ref name="tmx " />

1974年，吴媚媚退休。<ref name="tmx " />

1987年，吴媚媚参与演出了《[[滑稽王小毛|滑稽王小毛]]》并且再度走红。<ref name="fhxm " /><ref name="hqmx " />

晚年，吴媚媚参与演出了《[[阿混正传|阿混正传]]》、《[[股疯|股疯]]》、《[[孽债|孽债]]》。<ref name="hqmx " />

2012年2月26日，吴媚媚因心脏衰竭在[[上海|上海]]去世。<ref name="fhxm " /><ref name="tmx " /><ref name=" bvx" >{{cite web |url=http://news.fengone.com/b/20120227/297141.html |title=滑稽戏女王吴媚媚去世 揭秘吴媚媚传奇一生 |accessdate= 2012年|work=|publisher=风网 |language = zh-hans}}</ref>

== 评价 ==
滑稽演员[[潘虹|潘虹]]：“虽然吴妈妈已是高龄，但她走了，我还是很震惊，很难过。好在，让我们每个人觉得安慰的是，她的艺术作品带给人们的影响，远比她的生命更长。”<ref name="hqmx " /><ref name=" cmsxz" />

滑稽演员[[王汝刚|王汝刚]]：“吴老师在里面刻画了一个不分是非去溺爱孙子的奶奶，这个角色在一开场的时候就有一点长达十分钟的戏，通过对话和动作奠定整部戏的基调，并引发观众的兴趣。吴老师在那短短十分钟里，就将接下来要出场的主人公的性格勾勒了出来，功力非常了得。”<ref name="hqmx " /><ref name=" cmsxz" />

滑稽演员[[童双春|童双春]]：“吴老师给我印象最深的滑稽戏要数《[[笑着向昨天告别|笑着向昨天告别]]》里演我的妈妈。她非常照顾年轻一辈，常常耐心地一遍一遍和我对词，戏中的感觉就这样一点一点磨了出来。”，“几十年的表演经验使得她对人物有非常精准的把握，而且她精通[[山东|山东]]、[[上海|上海]]、[[苏北|苏北]]、[[苏州|苏州]]等多地方言。年轻一代演员应该向她学习。”<ref name="hqmx " /><ref name=" cmsxz" />

[[上海滑稽剧团|上海滑稽剧团]]团长[[凌梅芳|凌梅芳]]：“吴老师与滑稽剧团的最后一次合作是在2002年‘[[浦江笑声关不住|浦江笑声关不住]]’贺岁大剧。当时我们想，要是能请吴老师回来参加演出，效果绝对完美。但考虑到老人的身体状况，我们尝试着联络吴老师的时候，并没有抱着太大希望。意外的是，85虽高龄的吴老师欣然应允再次登台，连演四五场，重现让观众津津乐道的《[[阿混新传|阿混新传]]》。事实上，一直以来，只要是剧团需要，吴老师总是义不容辞地答应，现在想来都觉得感动。”<ref name="hqmx " /><ref name=" cmsxz" />

滑稽演员[[黄佐临|黄佐临]]：“吴媚媚是一个在舞台上自我得意、自在生活、自由发挥、难得的滑稽女天才。”<ref name="tmx " />
