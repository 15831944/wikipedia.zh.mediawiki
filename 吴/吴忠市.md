'''吴忠市'''是[[中华人民共和国|中华人民共和国]][[宁夏回族自治区|宁夏回族自治区]]下辖的[[地级市|地级市]]，位于宁夏中部，[[黄河|黄河]]东岸。市境东邻[[陕西省|陕西省]][[榆林市|榆林市]]，南接[[甘肃省|甘肃省]][[庆阳市|庆阳市]]，西界[[中卫市|中卫市]]，西北临[[内蒙古自治区|内蒙古自治区]][[阿拉善盟|阿拉善盟]]，北连[[银川市|银川市]]。地处宁夏中部[[银川平原|银川平原]]，引黄灌区的精华地段。全市总面积1.68万平方公里，人口129.48万，其中[[回族|回族]]人口比例超过五成，是[[中国|中国]][[回族|回族]]主要聚居区之一。吴忠农牧业发达，为自治区的粮食主要产地，盛产滩羊，素有“塞上江南”、“塞上明珠”之美称。

== 历史 ==
吴忠有着悠久的历史，是[[中华文明|中华文明]]的发祥地之一，是[[河套文化|河套文化]]的重要组成部分。黄河岸边的[[灵武|灵武]]“[[水洞沟|水洞沟]]遗址”发掘表明，早在三万年前的[[旧石器时代|旧石器时代]]，就有人类在此生息；[[羌|羌]]、[[戎|戎]]和[[匈奴|匈奴]]等古代游牧民族曾在这里逐水草、牧牛羊；[[秦始皇|秦始皇]]统一中国后就在此兴修水利，开垦农田，吴忠就有了引黄灌溉史；汉唐盛世，引来[[中华|中华]]大融合，在此设立[[富平县|富平县]]，[[唐太宗|唐太宗]]亲临此地接受异族归附；“安史之乱”，[[唐肃宗|唐肃宗]]李亨在[[灵州|灵州]]登基，[[灵州|灵州]]成了全国政治军事中心，当时的吴忠已相当富庶；党项族首领、[[西夏国|西夏国]]皇帝[[李元昊|李元昊]]统治的腹地也在今日吴忠一带；[[成吉思汗|成吉思汗]]灭[[西夏|西夏]]，以西夏故都为[[中兴府|中兴府]]，改[[宁夏路|宁夏路]]。至元二十四年（1287年）将宁夏行省并入[[甘肃行省|甘肃行省]]。

明朝时，属边疆地塞，置吴忠堡，归[[灵州守御千户所|灵州守御千户所]]管辖。《[[嘉靖宁夏新志|嘉靖宁夏新志]]》记载“尽徙宁夏之民与他地，迁秦晋江淮之人以实之，分屯建卫，筑堡以居。”，引述《[[宁夏旧志|宁夏旧志]]》云：“宁夏堡寨，以人名、事名、地名......即以其屯长姓名命堡，如叶升、李俊、王太之类是也。”，故认为吴忠堡得名于屯长吴忠<ref>{{Cite web |url=http://2008.qq.com/a/20080527/000293.htm |title=奥运火炬传递城市吴忠历史沿革|accessdate=2017-10-28 |author= |date=2008-05-27 |work= |publisher=腾讯网|language=Zh-hans }}</ref>。

1929年成立[[宁夏省|宁夏省]]；中华人民共和国成立后，1954年4月成立[[河东回族自治区|河东回族自治区]]，同年7月，宁夏撤省归[[甘肃省|甘肃省]]管辖；1955年改为[[吴忠回族自治州|吴忠回族自治州]]，下辖5县：[[吴忠县|吴忠县]]、[[金积县|金积县]]、[[灵武县|灵武县]]、[[盐池县|盐池县]]、[[同心县|同心县]]；1958年撤州划归[[宁夏回族自治区|宁夏回族自治区]]；1972年成立[[宁夏回族自治区|宁夏回族自治区]][[银南地区|银南地区]]专署；1998年5月撤[[银南地区|银南地区]][[专署|专署]]为地级吴忠市，[[灵武市|灵武市]]曾经由吴忠市代管。2001年12月，灵武市正式划归[[银川市|银川市]]。

== 地理 ==
地理位置：吴忠市位于宁夏回族自治区中部，南接[[中卫|中卫]]，北连[[银川|银川]]，东部与[[陕西省|陕西省]][[定边县|定边县]]毗邻，东北，西北与[[内蒙古自治区|内蒙古自治区]]的[[伊克昭盟|伊克昭盟]]和[[阿拉善左旗|阿拉善左旗]]相连，东南与[[甘肃|甘肃]][[庆阳|庆阳]]地区接壤。

地势地貌： 吴忠市东西长而南北窄，地势南高北低，北为银川平原，南为青铜峡平原和丘陵山地。川区干均海拔1100米，山区海拔在1300—1900米。全市地貌打为[[黄土高原|黄土高原]]，[[鄂尔多斯|鄂尔多斯]]台地、黄河冲积平原和l山地。

河流山脉：黄河是吴忠最重要的河流，流经川区两市(区)，沿岸地势斗坦，自秦汉先后开掘了秦渠、汉渠，唐徕渠等重要灌溉渠道，  已有2000多年的历史。这里土地肥沃，沟渠纵横，林带咸网，旱涝元虞，农业发达，素有“塞上江南”的美誉、盛产小麦．水稻、瓜果，蔬菜、是宁夏主要商品粮生产基地。南部黄土高原，宜林宜牧，是宁夏滩羊，沙毛山羊的重要产地。 主要山脉有罗山、牛首山，其中罗山是宁夏三大天然林区之一。

== 经济 ==
目前，吴忠拥有八大优势产业：能源电力工业、新材料产业、奶产业、葡萄酒产业、建材业、造纸产业、皮毛绒产业、草畜产业。
能源电力工业：目前，全市能源电力工业总装机容量达155万KW，2008年达到350万千瓦。
新材料产业（铝、镁、PVC等）：已具备28万吨电解铝、2.5万吨金属镁、10万吨PVC树脂生产能力，2008年将增长3倍。
奶产业：奶制品加工企业15家，生产能力占到宁夏的一半以上，到2008年奶产业产值突破60亿。
葡萄酒产业：目前，葡萄酒产量占全区的50%，到2008年葡萄酒产业实现产值突破20亿元。
建材产业：目前，建材产业中水泥产量达到350万吨，到2008年全市水泥生产能力达到660万吨。
造纸产业：吴忠已形成宇华纸业、峡光纸业、吴忠造纸厂等具有一定规模的造纸企业，到2008年，全区造纸能力可达28万吨，造纸产业实现产值11亿元。
皮毛绒产业：目前，生产的无毛绒占全区总量的60%，到2008年，以同心县为重点的无毛绒年生产能力达到3000吨。
草畜产业：到2008年，以盐池、同心、红寺堡为重点的羊畜产业优质饲草种植面积达到400万亩，羊只饲养量达到600万只，肉牛20万头，草畜产业实现产值10亿元。

== 宗教 ==
来到宁夏吴忠市区，随处可见建筑独特、风格迥异的清真寺。它们有的气势恢弘，有的精巧别致。吴忠是宁夏回族群众的主要聚居地之一，全市现有回族58万，约占全市总人口的47.2%，清真寺是回族群众做礼拜的主要场所和公共活动场所，在穆斯林群众心里有着重要的地位。市区内最有名的要数南大寺，北大寺和中大寺，其中既有色调明快、独具一格的阿拉伯式风格，又有庄重、古朴和讲究对称的中国传统古建筑特色。它们全部座西向东，一般都由礼拜大殿、浴室、邦克楼等部分组成。清真寺内环境清静幽雅，礼拜殿宽敞明朗，气氛庄重。这些深藏于都市一隅，掩映于高楼大厦之间的清真寺成为吴忠最富特色的建筑景观，吸引着无数游人前来参观。

同时，吴忠也有[[兴教寺|兴教寺]]和[[东塔寺|东塔寺]]两座著名的佛教寺庙，以及牛首山道教庙群。
