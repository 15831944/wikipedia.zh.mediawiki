'''吴谦'''（{{bd|1973年|||}}）[[北京|北京]]人，[[中国人民解放军|中国人民解放军]]大校，[[中华人民共和国国防部新闻局|中华人民共和国国防部新闻局]]局长、国防部新闻发言人<ref name=gfb>{{Cite web |url = http://www.mod.gov.cn/info/2017-02/23/content_4621520.htm |title = 国防部新闻发言人吴谦大校 |author =  |date =2017-02-23  |publisher =  中华人民共和国国防部|language =  |accessdate =  |quote =  }}</ref><ref name=ifeng>{{Cite web |url = http://news.ifeng.com/a/20161220/50446820_0.shtml |title =国防部新闻发言人吴谦晋升大校军衔  |author =  |date = 2016-12-20 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

== 生平 ==
1973年，吴谦生于北京。1991年毕业于[[上海市位育中学|上海市位育中学]]考进[[中国人民解放军国际关系学院|中国人民解放军国际关系学院]]。1995年毕业后，吴谦历任[[装甲兵装备技术研究所|装甲兵装备技术研究所]]助理工程师，[[国防部外事办公室|国防部外事办公室]]参谋、处长，[[中国驻美国大使馆|中国驻美国大使馆]]国防副武官，国防部新闻事务局处长等职务。吴谦曾在[[英国|英国]][[伯明翰大学|伯明翰大学]]学习，获得[[工商管理硕士|工商管理硕士]]学位。2015年6月，吴谦出任国防部新闻事务局副局长、国防部新闻发言人。2016年1月改任国防部新闻局副局长、国防部新闻发言人。2016年晋升大校军衔<ref name="gfb" /><ref name="ifeng" />。2017年8月，升任国防部新闻局局长、国防部新闻发言人<ref name=yyjty>{{cite web |url = http://news.ifeng.com/a/20170809/51594211_0.shtml |title = 国防部新闻发言人杨宇军大校退役：对家庭亏欠太多 |author =  |date = 2017-08-09 |website = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
