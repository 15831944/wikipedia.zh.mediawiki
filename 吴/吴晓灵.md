'''吴晓灵'''（{{bd|1947年|1月|catIdx=W吴晓灵}}），[[四川|四川]][[威远|威远]]人；原[[中国人民银行研究生部|中国人民银行研究生部]]毕业，经济学硕士<ref>{{cite web |url = http://news.xinhuanet.com/politics/2008-09/10/content_9898618.htm |title = 吴晓灵（女）_时政频道_新华网 |author = 中国人大网 |work = news.xinhuanet.com |year = 2008 [last update] |accessdate = 25 August 2012}}</ref>。曾长期在中国人民银行任职；是[[国家外汇管理局|国家外汇管理局]]首位女局长<ref name="和讯人物" />；现任[[全国人民代表大会财政经济委员会|全国人民代表大会财政经济委员会]]副主任委员，[[中欧国际工商学院|中欧陆家嘴国际金融研究院]]院长、教授，[[清华大学五道口金融学院|清华大学五道口金融学院]]首任院长<ref>{{cite web |url = http://news.hexun.com/2012-03-29/139876916.html |title = 吴晓灵任清华大学五道口金融学院首任院长 |publisher=和讯网 |accessdate = 2012-08-20}}</ref>。著有《新一轮改革中的中国金融》、《中国的金融深化与金融改革》等学术著作；曾于1994年获[[孙冶方经济科学奖|孙冶方经济科学奖]]<ref name="和讯人物" />。

== 履历 ==
吴晓灵1984年毕业于[[中国人民银行研究生部|中国人民银行研究生部]]（今[[清华大学五道口金融学院|清华大学五道口金融学院]]），获经济学硕士；随即留校，任[[研究员|研究员]]<ref name="和讯人物">{{cite web |url = http://renwu.hexun.com/figure_938.shtml |title = 和讯人物频道-吴晓灵 |publisher = 和讯网 |accessdate = 2012-08-20}}</ref>。1985年，出任[[中国人民银行研究所|中国人民银行研究所]]应用理论研究室副主任；1988年，任《[[金融时报|金融时报]]》社副总编辑。自1991年回到央行后，吴晓灵历任金融体制改革司副司长，央行政策研究室主任，[[国家外汇管理局|国家外汇管理局]]副局长、局长；是中国首位女性外汇管理局长<ref name="和讯人物" />。1998年至2000年间，曾出任中国人民银行上海分行行长；2001年4月，任央行副行长，并再次兼任外管局局长。<ref>{{cite web | url=http://finance.qq.com/a/20090205/001375.htm |title = 吴晓灵简历 |publisher=腾讯财经 |accessdate=2012-08-20}}</ref> 

2008年1月，61岁的吴晓灵卸任央行副行长职务；当年3月，在[[第十一届全国人大第一次会议|第十一届全国人大第一次会议]]上，当选[[全国人大常委会|全国人大常委会]]委员，并任[[全国人民代表大会财政经济委员会|全国人民代表大会财政经济委员会]]副主任委员。<ref>{{cite web |url = http://www.caijing.com.cn/2008-03-15/100052575.html |title = 吴晓灵、蔡昉当选人大常委 |publisher = 财经网 |accessdate=2014-01-29}}</ref> 2013年3月，再次连任[[第十二届全国人大常委会|第十二届全国人大常委会]]委员、[[全国人民代表大会财政经济委员会|全国人大财经委]]副主委<ref>{{cite web|url =http://www.ceibs.edu/media_c/archive/75946.shtml|title =吴晓灵教授连任全国人大财经委副主任委员|publisher =中欧国际工商学院|accessdate =2014-01-29|deadurl =yes|archiveurl =https://web.archive.org/web/20140201215153/http://www.ceibs.edu/media_c/archive/75946.shtml|archivedate =2014-02-01}}</ref> 。

== 荣誉 ==
2004年，吴晓灵被美国《[[华尔街日报|华尔街日报]]》评为“年度全球最受关注的50位商界女性”，2005年再度入选。在2006年和2007年，吴晓灵两度入选美国《[[福布斯|福布斯]]》“全球最有影响力的女性”排行榜。2011年12月，吴晓灵获评[[中国中央电视台|中国中央电视台]]2011年中国经济年度人物<ref>{{cite web |url = http://finance.sina.com.cn/hy/20111212/212510979160.shtml |title = 人大财经委吴晓灵荣获CCTV中国经济年度人物奖 |publisher = 新浪财经 |accessdate = 2012-08-20}}</ref>。
