== 经历 ==
* 2017年，加入SDT娱乐，开始生涯学舞第一年，获得QQ炫舞大赛北京赛区冠军<ref name="天天快报">{{cite web |title=《创造101》小甜豆吴芊盈，曾为2000Q币参加舞蹈比赛？ |url=https://kuaibao.qq.com/s/20180604A0W97R00?refer=spider |website=天天快报 |date=2018-06-04}}</ref>。

* 曾以SDT舞蹈天团成员身份，担任[[湖南卫视|湖南卫视]]《元宵喜乐会》等揭幕伴舞。

* 2018年3月，练习生身份揭晓，赴杭州参加录制[[腾讯视频|腾讯视频]]出品的女团综艺节目《[[创造101|创造101]]》（以下简称《创》），出镜全季片头的「我的女团梦」开场。

* 在《创》第二期中以抢位选手身份加入，于第九期（第四轮排名）以27名淘汰之前，以其可爱形象和扎实的舞蹈基础吸引许多粉丝，亦获得了导师的认可。

* 《创》结束后多栖发展，担任《爱格》杂志封面人物、innisfree悦诗风吟代言团体「发光少女」成员<ref name="搜狐娱乐-2">{{cite web |title=悦诗风吟邀请创造101上线《发光少女》主题 MV 
 |url=http://www.sohu.com/a/256329496_100071077 |website=[[搜狐娱乐|搜狐娱乐]] |date=2018-09-26}}</ref>、并于11月参与[[腾讯视频|腾讯视频]]出品的中国版[[偶像明星运动会|偶像明星运动会]]《超新星全运会》，代表北京队出战。

* 2018年7月起开始学习表演，同年年底至翌年年初参演[[爱奇艺|爱奇艺]]制作的网络古装剧《恋恋江湖》<ref name="豆瓣电影">{{cite web |title=恋恋江湖 的全部演职员 |url=https://movie.douban.com/subject/30372333/celebrities |website=[[豆瓣|豆瓣]]电影 |date=2018-02-14}}</ref>，是为演员首秀。
