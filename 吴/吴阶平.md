'''吴阶平'''（{{bd|1917年|1月22日|2011年|3月2日|catIdx=Wu吴阶平}}），原名'''吴泰然'''，[[号|号]]'''阶平'''<ref>{{cite web |url = http://cswbszb.chinajilin.com.cn/html/2011-03/19/content_708717.htm |title=“一代国医”吴阶平 |publisher=[[城市晚报|城市晚报]] |date=2011年3月19日 |accessdate=2014年6月13日}}</ref>，[[江苏|江苏]][[常州|常州]][[武进|武进]]人，中国[[泌尿外科|泌尿外科]][[医学|医学]]家，[[中国科学院|中国科学院]]和[[中国工程院|中国工程院]]双料[[院士|院士]]。

== 生平 ==
吴阶平自小于[[天津|天津]]长大。1942年毕业于[[北京协和医学院|北京协和医学院]]，获得[[医学博士|医学博士]]学位。1947年赴[[美国|美国]][[芝加哥大学|芝加哥大学]]留学，次年回国。[[中华人民共和国|中华人民共和国]]建立后，他就职于[[北京医学院|北京医学院]]，曾率医疗队参加[[朝鲜战争|朝鲜战争]]。1956年，吴加入[[中国共产党|中国共产党]]。

从1954年起，吴阶平先后奉命为[[金日成|金日成]]、[[苏加诺|苏加诺]]、[[胡志明|胡志明]]、[[费迪南德·马科斯|费迪南德·马科斯]]等人提供过医疗服务。1967年底开始任[[江青|江青]]的保健医生，次年又服务于[[林彪|林彪]]。1972年，吴阶平出任[[周恩来|周恩来]]医疗小组组长，为其诊治[[膀胱癌|膀胱癌]]。

1980年，吴阶平当选[[中国科学院|中国科学院]][[学部委员|学部委员]]<ref>{{cite web|url=http://www.casad.ac.cn/showuserinfo.asp?userid=3055&act=info|title=吴阶平个人简历|accessdate=2008-08-31|publisher=中科院学部网|language=zh-hans}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1989年，被选为[[九三学社|九三学社]]中央副主席，1992年当选主席，1990年在[[第七届全国人民代表大会第三次会议|第七届全国人民代表大会第三次会议]]上被補選為[[全國人大常委会|全國人大常委会]]委员，1993年至2003年任[[全国人大常委会副委员长|全国人大常委会副委员长]]。

2000年2月28日，[[吴阶平医学基金会|吴阶平医学基金会]]在北京成立，为[[中华人民共和国卫生部|中华人民共和国卫生部]]直属的行业基金会<ref>{{cite web |url=http://www.wjpmf.org/about/jianjie.asp |title=吴阶平医学基金会介绍 |accessdate=2008-08-31 |publisher=吴阶平医学基金会 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20091003221121/http://www.wjpmf.org/about/jianjie.asp |archivedate=2009-10-03 }}</ref>。

2011年3月2日21时18分在北京逝世<ref>{{cite news|title=九三学社领导人吴阶平逝世 享年94岁|url=http://news.163.com/11/0302/23/6U6770ML00014JB5.html|accessdate=2011-03-03|newspaper=新华网|date=2011-03-02}}</ref>。

==家庭==
其兄[[吴瑞萍|吴瑞萍]]、弟[[吴蔚然|吴蔚然]]亦是著名医学家。
