'''吴宝路站'''位于[[上海市|上海市]][[闵行区|闵行区]][[沪青平公路|沪青平公路]][[吴宝路|吴宝路]]口两侧的中央隔离带，为[[上海公交71路|上海公交71路]]及其区间车共用的地面[[岛式月台|岛式车站]]，是分离式岛式站台。<ref name="fls">{{cite web|url=http://www.shmh.gov.cn/sites/minhang/NewsCenter_Content.aspx?CtgId=c2c8d255-45ab-4661-94d8-c508a9b29e23&InfoId=33959cd8-9308-4829-b119-75a7b329ebf0|title=延安路中运量公交今起正式载客试运行，闵行的交通相应会发生哪些变化？|website=闵行区人民政府网站}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

该车站为西段架空线网(吴宝路—水城南路)起点。电车驾驶员驾驶车辆进站时应升降[[集电杆|集电杆]]。

== 车站構造 ==
吴宝路站为分离式[[岛式月台|岛式月台]]，由两个1面1线地面车站组成，站台设有半高屏蔽门，靠路口一侧均设有[[無障礙環境|无障碍通道]]。<ref name="fls"></ref>

该站点开工于2016年6月25日<ref name="kg">{{cite web | title = 延安路中运量公交系统工程开工 | publisher = 上海市交通委员会 | date = 2016-06-27 | url = http://www.jt.sh.cn/jtw/xwzx/n218/u1ai51190.html | accessdate = 2016-11-30 | language = zh | archive-url = https://web.archive.org/web/20161010223253/http://www.jt.sh.cn/jtw/xwzx/n218/u1ai51190.html | archive-date = 2016-10-10 | dead-url = yes }}</ref>，于2017年2月1日清晨正式投入使用<ref name="syy">{{cite news|title=上海延安路中运量公交开通试运营|url=http://www.thepaper.cn/newsDetail_forward_1610252|publisher=澎湃新闻|date=2017-02-01}}</ref>。

== 周边设施 ==
* 站点西面：[[上海虹渝高架路|虹渝高架路]]
* 站点东北面：[[上海虹桥机场|上海虹桥机场]]跑道

== 公交换乘 ==
附近有两个常规公交站点：
* 吴家巷（沪青平公路吴宝路）<ref>{{cite web|author1=上海市地方志办公室|title=上海县志 >> 第二十一篇交通运输 >> （五）公路客运 >> 解放后公交客运|url=http://www.shtong.gov.cn/newsite/node2/node4/node2250/shanghai/node53953/node53967/node53969/userobject1ai36110.html|website=上海通网站|accessdate=2017-07-10}}</ref>：[[上海公交91路|91路]]、[[上海公交173路|91路]]、173路、320路夜宵（含区间）、虹桥枢纽1路、闵行18路
* 七莘路沪青平公路：[[上海公交91路|91路]]、189路、196路、748路、虹桥枢纽4路、南江线

== 车站出口 ==
这个车站有两个出入口。
* 往市区方向站体的出入口位于吴宝路口，同时是西侧人行横道的安全岛。
* 往申昆路方向站体的出入口位于吴宝路口，同时是东侧人行横道的安全岛。
