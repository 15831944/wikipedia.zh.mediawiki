'''吴承仕'''（{{bd|1884年|3月20日|1939年|9月21日}}），字'''检斋'''，号展成，又号济安。中国[[经学家|经学家]]、[[古文字学|古文字学]]家、教育家。与[[黄侃|黄侃]]并称“北吴南黄”两大[[经学|经学]]大师，与[[黄侃|黄侃]]、[[钱玄同|钱玄同]]并称[[章太炎|章门]]三大弟子。<ref>{{Cite book|title=近现代皖籍教育家研究：教育家办学的历史观照|last=孙德玉|first=|publisher=安徽师范大学出版社|year=2016|isbn=|location=|pages=200}}</ref>

== 生平 ==
1884年3月20日生于[[安徽省|安徽]][[歙县|歙县]][[昌溪乡|昌溪]]。幼读私塾，1901年应试中[[秀才|秀才]]，次年[[中举|中举]]，1907年[[举贡会考|举贡会考]]获一等第一名{{notetag|时称“朝元”，李征宙则认为实为清代最年轻{{r|李征宙|page=129}}而且是最后的[[状元|状元]]{{r|李征宙|page=130}}，柯卫东写道：“其乡人皆视之为状元”。<ref>{{Cite book|title=藏书家 珍藏本 11-13 合订本 上|last=宫晓卫主编；韦力执行主编|first=|publisher=齐鲁书社|year=2014|isbn=|location=济南|pages=139}}</ref>}}，点为[[大理院_(清朝)|大理院]]主事{{notetag|改为[[大理院_(清朝)|大理院]][[推官|推事]]。<ref>{{Cite journal|title=晚清大理院审判官员调配及履历考论|author=韩涛|url=|journal=历史档案|issue=3|doi=|others=|year=2011|volume=|page=86|pmid=}}</ref><ref>{{Cite book|title=近代名家评传 二集|last=王森然著|first=|publisher=生活·读书·新知三联书店|year=1998|isbn=|location=北京|pages=305}}</ref>}}。1912年任[[中華民國法務部|司法部]][[僉事|佥事]]。1914年亲往[[章太炎|章太炎]]被监禁处探望，{{r|研究资料集|page=1}} 求学于[[章太炎|章太炎]]，笔录[[章太炎|章太炎]]口述，成《菿汉微言》。1918年参与昆曲研究会，借古讽今抨击[[北洋政府|北洋政府]]，{{r|研究资料集|page=2}} 1923年见研究会被[[曹锟|曹锟]]笼络而退出。1924年《经籍旧音辩证》正式出版，[[章太炎|章太炎]]题辞。{{r|研究资料集|page=3}} 年末任[[北京师范大学|北京师范大学]]国文系主任{{notetag|《吴承仕研究资料集》写为“国学系”，本条目其他参考文献均写为“国文系”。}}，同时被聘为[[中国大学|中国大学]]、[[北京大学|北京大学]]、[[东北大学_(中国)|东北大学]]教授。1926年秋任[[中国大学|中国大学]]国学系主任，{{r|研究资料集|page=4,15-16}} 在任期间，编制了教学大纲，改革了课程。{{r|传略|page=21}}

1927年4月8日闻[[李大钊|李大钊]]等被捕后设法营救，4月29日闻[[李大钊|李大钊]]等被害，辞去[[中華民國法務部|司法部]][[僉事|佥事]]之职，{{r|研究资料集|page=16}} 继任[[北京师范大学|北京师范大学]]国文系主任。1930年受[[范文澜|范文澜]]、[[齐燕铭|齐燕铭]]等影响接触[[马克思主义|马克思主义]]著作。{{r|研究资料集|page=5}} 1931年9月闻[[九一八事变|九一八事变]]，任[[北京师范大学|北京师范大学]]教授会主席，一致通电[[南京政府|南京政府]]要求抗日。{{r|研究资料集|page=6}} 1933年为[[中国大学|中国大学]]国学系增加《社会科学概论》、《新俄文学选读》等课程，{{r|研究资料集|page=20}} 聘请[[吕振羽|吕振羽]]、[[黄松龄|黄松龄]]、[[李达_(哲学家)|李达]]、[[陈伯达|陈伯达]]等讲课。{{r|李征宙|page=133}} 1934年5月创办双月刊《文史》，年底被查禁。1935年10月创办月刊《盍旦》（仅出版五期）。{{r|研究资料集|page=7}} 1935年参与[[一二·九运动|一二·九运动]]，亲身去[[新华门|新华门]]前集会，和学生一起参加游行示威，{{r|研究资料集|page=8}} 参与发起北平文化界救国会。{{r|传略|page=27}} 1936年与[[张友渔_(法学家)|张友渔]]、[[黄松龄|黄松龄]]、[[杨秀峰|杨秀峰]]等创办《时代文化》杂志。{{r|研究资料集|page=8}} 1936年初[[章太炎|章太炎]]邀请南下，3月回绝，6月闻[[章太炎|章太炎]]逝世，在北京主持追悼会，共致[[挽联|挽联]]并自为长[[挽联|挽联]]。夏，为[[中国大学|中国大学]]新生国文试题命题并阅卷，许多参加[[一二·九运动|一二·九运动]]而被开除的学生被录取。{{r|李征宙|page=132}} 秋，秘密加入[[中国共产党|中国共产党]]。{{r|研究资料集|page=9}}

1937年5月与[[张友渔_(法学家)|张友渔]]、[[黄松龄|黄松龄]]、[[杨秀峰|杨秀峰]]等发起组织新启蒙学会。{{r|研究资料集|page=9}} [[七七事变|七七事变]]、北京沦陷后前往天津，避居于英租界[[小白楼|小白楼]]，为地下抗日刊物《时代周刊》撰文并筹措经费。{{r|研究资料集|page=10}} 1938年初拒任[[北京师范大学|北京师范大学]]校长。{{r|传略|page=29}} 1939年8月天津水患，回北京，寄居于一商人家中。9月11日入[[北京协和医院|北京协和医院]]治疗，诊断为[[伤寒|伤寒]]，已[[肠穿孔|肠穿孔]]，旧病[[支气管炎|支气管炎]]并发，9月21日逝世。{{r|研究资料集|page=10}}{{notetag|李征宙则称吴承仕就医时“被一名为日寇收买的医生强行灌肠至血暴”逝世。{{r|李征宙|page=137}}}}

== 纪念 ==
毛泽东曾派人至天津接吴至[[延安|延安]]，因水灾而未成。{{r|研究资料集|page=10}} 吴承仕死后，时人以为吴承仕“被日寇逮捕，业已壮烈殉国”，1940年4月14日，[[延安|延安]]举行追悼[[蔡元培|蔡元培]]吴承仕先生大会，到会代表千余人，{{r|研究资料集|page=297}}  16日，[[延安|延安]]举行吴承仕追悼大会，[[毛泽东|毛泽东]]送挽联“老成凋谢”，[[周恩来|周恩来]]挽联为“孤悬敌区，舍身成仁，不愧青年训导；重整国学，努力启蒙，足资后学楷模。”[[吴玉章|吴玉章]]挽联为“爱祖国山河，爱民族文化，尤爱马列主义真理，学贯中西，善识优于[[张煌言|苍水]]；受军阀迫害，受同事排挤，终受日寇毒刃摧残，气吞楼虏，壮烈比诸[[文天祥|文山]]。”{{r|传略|page=30}} [[中国共产党第七次全国代表大会|中共七大]]列入[[烈士_(中华人民共和国)|烈士]]名单。{{r|李征宙|page=136}} [[陈伯达|陈伯达]]、[[吕振羽|吕振羽]]、[[齐燕铭|齐燕铭]]、[[張友漁_(法學家)|张友渔]]、[[张致祥|张致祥]]、[[许承尧|许承尧]]、[[孟超|孟超]]等写文章悼念。

1984年3月18日，北京各界人士在[[全国政协礼堂|全国政协礼堂]]集会纪念吴承仕诞辰一百周年。<ref>{{Cite journal|title=纪念吴承仕诞辰一百周年|author=方闻|url=|journal=晋阳学刊|issue=5|doi=|others=|year=1984|volume=|page=112|pmid=}}</ref> 3月27日，《[[人民日报|人民日报]]》发表[[蒋南翔|蒋南翔]]文章《纪念吴承仕同志诞生一百周年》。{{r|李征宙|page=128}} 有《吴承仕同志诞生百周年纪念文集》出版。

[[庄华峰|庄华峰]]系统收集相关史料编成《吴承仕研究资料集》，以纪念吴承仕逝世五十周年。{{r|王延模|page=127}}

== 著作 ==
吴承仕临终托付遗稿于[[孙人和|孙人和]]，后散佚。1949年后，吴承仕部分手稿由[[齐燕铭|齐燕铭]]收藏，后移交[[中华书局|中华书局]]。{{r|王延模|page=122}} 1982年后由[[启功|启功]]、[[黄寿祺|黄寿祺]]等整理，{{r|王延模|page=124}} 由[[中华书局|中华书局]]出版《经籍旧音序录》、《经籍旧音辨正》、《经典释文序录疏证》，由[[北京师范大学出版社|北京师范大学出版社]]出版《淮南旧注校理》、《论衡校释》、《检斋读书提要》、《吴承仕文录》、《吴承仕藏章炳麟论学集》。{{r|王延模|page=125}}

== 评价 ==
* 自评（1937年）：“承仕幼承庭诰，长受业于章太炎先生之门，服官法界计二十年，……至于立身行道，表里如一，一事不妄为，一语不妄发，一介不妄取，硁硁自守之节，老而弥笃，则诚足以质天地而告鬼神者也。”{{r|传略|page=29}}
* [[章太炎|章太炎]]：“检斋文不如[[黄侃|季刚]]，而为学笃实过之。”“及吾门得辨声音训诂者，其惟检斋乎！”{{r|研究资料集|page=236}}
* [[潘伯鹰|潘伯鹰]]：“歙县吴检斋先生，大节凛然，天下所仰经师人师者也。……读其隐约危苦之辞，而胸中勃郁刚大之气，自震耀于行间。虽吉光片羽，而先生负荷天下之重者，犹然具见”。{{r|传略|page=30}}
* [[侯外庐|侯外庐]]：“吴承仕先生真正是值得大家永远敬仰的学者。吴承仕先生是章太炎的弟子，我常拿他与鲁迅先生比较，在太炎门下，有两位弟子通过全然不同的路程，殊途同归，都走向信仰马克思主义，这是一个非常值得注意的问题。”<ref>{{Cite book|title=侯外庐著作与思想研究 第1卷|last=张岂之主编|first=|publisher=长春出版社|year=2016|isbn=|location=长春|pages=31}}</ref>
* [[蒋南翔|蒋南翔]]：“他不仅是一位著名的经学家、古文字学家和教育家，更重要的是，他由一位硕学鸿儒转变为马克思主义的革命者，光荣的中国共产党党员。”“他从前清的举人发展成为战斗的马克思主义者。他经历这样漫长曲折的战斗历程，是举世稀有的。”<ref>{{Cite news|url=|title=纪念吴承仕同志诞生一百周年|author=蒋南翔|date=1983-03-27|work=人民日报|accessdate=}}</ref>
* 被认为是“用唯物史观研究经学的第一人”<ref>{{Cite journal|title=用唯物史观研究经学的第一人：吴承仕|author=王荣滨|url=|journal=徽州社会科学|issue=1|doi=|others=|year=1992|volume=|page=51|pmid=}}</ref>，“第一个用马克思主义观点研究经学的人”<ref>{{Cite journal|title=第一个用马克思主义观点研究经学的人——吴承仕|author=|url=|journal=黄山学院学报|issue=4|doi=|others=|year=2006|volume=|page=168|pmid=}}</ref>。
