'''吴稼祥'''{{BD|1955年|10月7日|||catIdx=W吴}}出生于[[安徽省|安徽省]][[铜陵市|铜陵市]][[铜陵县|铜陵县]][[大通镇|大通镇]]，毕业于[[北京大学|北京大学]]经济系，[[美国|美国]][[哈佛大学|哈佛大学]]访问学者，曾在[[中华人民共和国政府|中华人民共和国政府]]任职，为[[政治学|政治学]]、[[经济学|经济学]][[学者|学者]]、[[公共知识分子|公共知识分子]]。<ref name="fish " >{{cite web |url=http://www.cssm.gov.cn/zuozhe/author.php?id=42 |title= 吴稼祥|accessdate= |publisher=战略与管理研究会 |archiveurl=http://www.cssm.gov.cn/zuozhe/author.php?id=42 |archivedate= 2012年11月8日|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://news.163.com/09/0505/08/58HOGTIO00012Q9L.html |title= 吴稼祥：墨西哥需要同情，不是隔离|accessdate=2009-05-05 |publisher=网易 |archiveurl=http://news.163.com/09/0505/08/58HOGTIO00012Q9L.html |archivedate=2012年11月8日 |work= |language =zh-cn}}</ref>

== 生平 ==
1955年，吴稼祥出生在[[安徽省|安徽省]][[铜陵市|铜陵市]][[铜陵县|铜陵县]][[大通镇|大通镇]]，[[文革|文革]]结束恢复高考制度之后，考入[[北京大学|北京大学]]经济系，大学毕业后，吴稼祥被分配到[[中共中央宣传部|中共中央宣传部]]，后转职[[中共中央书记处|中共中央书记处]]办公室、[[中共中央办公厅|中共中央办公厅]]。1989年，[[六四事件|六四事件]]爆发，吴稼祥被北京市公安局逮捕并且发送到[[秦城监狱|秦城监狱]]，1992年刑满释放。2000年，吴稼祥远赴[[美国|美国]][[哈佛大学|哈佛大学]]当访问学者，一直居住在剑桥市3年。<ref name="fish " /><ref name=" lio" />

== 思想 ==
吴稼祥是中国国内最早提出改革[[国有企业|国有企业]]，实行[[股份制|股份制]]的学者之一。<ref name=" lio" >{{cite web |url=http://news.sohu.com/20100314/n270812678.shtml |title=吴稼祥：民间NGO是社会调节器 |accessdate=2010年3月14日 |publisher=搜狐网 |archiveurl=http://news.sohu.com/20100314/n270812678.shtml |archivedate=2012年11月8日 |work= |language =zh-cn}}</ref>

在[[周永康案|周永康案]]宣布后，吴稼祥在新浪微博上为周永康喊冤，称胡锦涛支持薄熙來，要把中国变成朝鲜，但被周永康抵制了，这是周永康的功劳。还指出沟马（山沟里走出的马列主义）派指鹿为马，利用“萱萱”造势愚弄天下。

根据亲身经历，以及大量事实资料，写成《泡面传奇》，展现了“六四”事件的许多内幕，提出了不同于以往广为流传的政治派系划分情况，以及“六四”事件的重要责任人，说明了江的政治派系变化过程，并指出江与习作为政治同盟的事实。由于与海内外网络上传播的对中国政治分析的主要观点相左，因此引起很大争议，受到不少攻击。
