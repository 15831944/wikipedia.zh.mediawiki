'''吴罡'''（{{bd|1917年||2011年|4月14日}}）[[河北|河北]][[枣强|枣强]]人，[[中国人民解放军开国少将|中国人民解放军开国少将]]<ref name=shishi>{{Cite web |url =http://cpc.people.com.cn/GB/64093/87393/14576649.html  |title = 吴罡同志逝世 |author =  |date =  2011-05-08|publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref><ref name=rmrb>吴罡同志逝世，人民日报2011年05月08日，第04版</ref>。

==生平==
1936年加入[[中国共产党|中国共产党]]。[[抗日战争|抗日战争]]时期，历任中共枣强县委组织部部长、县委书记，中共冀南第五地委委员兼枣强县委书记，中共冀南第二地委组织部部长兼[[故城县|故城县]]委书记等职，参加过反“扫荡”等。1947年，被选调参军，历任旅政治委员、军分区副政治委员、中共桐柏地委书记兼三分区政治委员、师政治委员等职，先后参加了楚旺、崔桥、安阳、邓县、豫西南剿匪等战役战斗<ref name=shishi/><ref name=rmrb/>。

[[中华人民共和国|中华人民共和国]]成立后，1950年调入[[中国人民解放军海军|中国人民解放军海军]]工作，历任中南军区海军政治部主任、干部部部长，海军政治干部学校政治委员、海军指挥学校政治委员、海军政治学校政治委员、海军学院副政治委员等职。后任海军副政治委员兼纪委书记<ref name=shishi/><ref name=rmrb/>。

吴罡是[[第六届全国人大代表|第六届全国人大代表]]。1955年被授予大校军衔，1961年晋升为少将军衔，曾获一级[[解放勋章|解放勋章]]、一级[[红星功勋荣誉章|红星功勋荣誉章]]<ref name=shishi/><ref name=rmrb/>。

2011年4月14日，吴罡在北京病逝，享年94岁<ref name=shishi/><ref name=rmrb/>。
