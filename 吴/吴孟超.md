'''吴孟超'''（{{bd|1922年|8月31日|catIdx=W吴}}），[[福建|福建]][[闽清县|闽清]]人，[[马来西亚|马来西亚]]归侨，中国[[肝胆外科|肝胆外科]]专家，[[主任医师|主任医师]]，[[一级教授|一级教授]]，[[博士生导师|博士生导师]]，[[中国人民解放军文职干部|中国人民解放军特级文职干部]]，[[中国科学院院士|中国科学院院士]]，曾任[[中国人民解放军第二军医大学|中国人民解放军第二军医大学]](现海军军医大学)副校长，创立二军大第三附属医院[[东方肝胆外科医院|东方肝胆外科医院]]、东方肝胆外科研究所并担任首任院长、所长，被誉为“中国肝脏外科的开拓者和创始人”和“中国肝胆外科之父”<ref name=":0">{{cite web |url = http://cpc.people.com.cn/GB/67481/94157/219450/219451/14454264.html |title = 吴孟超简介 |website = 人民网 |accessdate = 2011年4月22日}}</ref>。

1943年吴孟超考入[[上海市_(中华民国)|上海]][[同济医学院|同济医学院]]，师从[[裘法祖|裘法祖]]，1949年毕业<ref name=":1">{{Cite web |url = http://www.jyb.cn/china/rwfc/201104/t20110424_426885.html |title = 吴孟超：九旬医生正当年 |newspaper = 中国教育报 |accessdate = 2011年4月24日 }}</ref>。1991年当选为[[中国科学院院士|中国科学院院士]]。1996年被[[中华人民共和国中央军事委员会|中华人民共和国中央军事委员会]]授予“模范医学专家”[[荣誉称号|荣誉称号]]<ref name=":0" />。2006年获2005年度[[国家最高科学技术奖|国家最高科学技术奖]]<ref name=":0" />。曾任[[中国人民解放军海军军医大学|中国人民解放军海军军医大学]]附属[[东方肝胆外科医院|东方肝胆外科医院]]院长，[[同济大学|同济大学]]名誉教授。2011年5月，中国将17606号小行星命名为“吴孟超星”。2011年6月，受聘为[[福建医科大学|福建医科大学]]名誉校长<ref>[http://cpc.people.com.cn/GB/67481/94157/219450/219453/14456581.html “中国肝胆外科之父”故乡福州掀“吴孟超热”]，[[中国共产党新闻网|中国共产党新闻网]]</ref>。2012年2月3日，当选感动中国2011年度人物。2019年1月14日退休<ref>[http://news.sciencenet.cn/htmlnews/2019/1/422132.shtm 97岁吴孟超院士退休：病人需要我随时投入战斗]</ref>。

== 成就 ==
* 1958年翻译出版第一部中文版《肝脏外科入门》。<ref>{{Cite web|url = http://www.popumed.com/module/emag/emag_content.php?articleid=3238 |title = 向癌中之王－－肝癌开刀，大众医学|accessdate = 2001年}}</ref>
* 1959年制成中国第一具完整的肝脏血管铸型标本。<ref name=":1" />
* 1960年完成中国第一例肝脏外科手术。<ref name=":2">{{Cite web |url = http://zqb.cyol.com/gb/zqb/2006-01/10/content_118099.htm |title = 吴孟超：中国肝脏手术之父，中国青年报 |accessdate = 2006年1月10日 |deadurl = yes |archiveurl = https://web.archive.org/web/20060112063058/http://zqb.cyol.com/gb/zqb/2006-01/10/content_118099.htm |archivedate = 2006年1月12日 |df =  }}</ref>
* 创造间歇性肝门阻断切肝法<ref>{{Cite web |url = http://pmmp.cnki.net/Resources/CDDPdf/evd%5Cbase%5C%E4%B8%AD%E5%8D%8E%E5%A4%96%E7%A7%91%E6%9D%82%E5%BF%97%5C%E7%97%85%E4%BE%8B%E5%88%86%E6%9E%90%5C%E9%97%B4%E6%AD%87%E6%80%A7%E8%82%9D%E9%97%A8%E9%98%BB%E6%96%AD%E4%B8%8B%E8%82%9D%E8%85%9432703.pdf |title = 间歇性肝门阻断下肝腔静脉结合部肝肿瘤的手术切除及疗效观察，《中华外科杂志》2001年7月第39卷第7期 |accessdate = 2001年7月}}</ref>和常温下无血切肝法；{{来源请求}}
* 1963年完成世界第一首例中肝叶切除手术。<ref name=":2" />
* 1975年切除至今世界最大的肝海绵状血管瘤。<ref name=":2" />
* 1984年创世界肝母细胞瘤切除手术对象年龄最小的纪录。<ref name=":2" />
* 1993年完成世界首例腹腔镜下的肝癌切除手术。<ref name=":2" />
* 率先提出巨大肝癌先经综合治疗再行手术切除的“二期手术”概念。{{来源请求}}
* 1996年，创建了中国第一所肝胆外科专科医院——[[东方肝胆外科医院|东方肝胆外科医院]]和第一所肝胆外科研究所——东方肝胆外科研究所。<ref name=":0" />
