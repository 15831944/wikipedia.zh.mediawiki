'''吴权清'''（{{bd|1991年|6月1日}}），网名'''茄子'''，是中国[[广东省|广东省]][[茂名市|茂名市]]的一名《[[反恐精英：全球攻势|反恐精英：全球攻势]]》职业[[电子竞技|电子竞技]]选手兼[[火猫直播|火猫直播]]平台的[[网络主播|网络主播]]，在游戏中担任[[狙击手|狙击手]]。<ref>{{cite web |author1=TyLoo |title=TYLOO公布2014年新阵容 Rita,Hz退役 qz,fancy1加盟 |url=http://fight.pcgames.com.cn/373/3730270.html |website=太平洋电竟 |date=2014-04-08 |access-date=2019-02-18 |archive-url=https://web.archive.org/web/20181118222141/http://fight.pcgames.com.cn/373/3730270.html |archive-date=2018-11-18 |dead-url=no }}</ref>

== 生平 ==
吴权清在读小学时，在[[网吧|网吧]]接触了游戏《[[反恐精英|反恐精英]]》并对其感兴趣，常与弟弟吴辉一起到网吧游玩，但家人对此持反对态度。初中毕业后，吴权清没有继续读高中，而是希望能够成为一名职业电子竞技选手。2012年，他所在的ZiNTHAI战队晋级[[世界电子竞技大赛|世界电子竞技大赛]]的总决赛。2013年，他加入Nface战队，在当年的[[ECL电子竞技冠军联赛|ECL电子竞技冠军联赛]]总决赛中战胜吴辉所属的[[天禄电子竞技俱乐部|天禄电子竞技俱乐部]]，后应邀加入天禄电子竞技俱乐部，但不久后因为与队员的矛盾冲突而和吴辉一起退出，转而组建POW战队，并在2014年的[[电子竞技世界杯|电子竞技世界杯]]中晋级代表中国与其他国家和地区的队伍比赛。2015年，他和吴辉又重新加入天禄电子竞技俱乐部。<ref name="MAX+">{{cite web |title=茄子传记：做最真实的自己，吴权清的梦想人生 |url=http://www.sohu.com/a/248974001_490106 |website=搜狐 |publisher=MAX+ |date=2018-08-20}}</ref>

2016年1月，吴权清所在的天禄电子竞技俱乐部参加[[IEM英特爾極限高手盃|IEM英特爾極限高手盃]][[台北|台北]]站，但在比赛前IEM英特爾極限高手盃发布公告，称经调查发现吴权清在2013年使用的一个游戏帐号有被[[維爾福反作弊系統|維爾福反作弊系統]]封禁的记录，故取消天禄电子竞技俱乐部整队的参赛资格。对此天禄电子竞技俱乐部解释当时中国大陆要获取一个《反恐精英：全球攻势》的帐号比较困难，所以吴权清和别人共用了帐号。在这件事发生后，吴权清退出了天禄电子竞技俱乐部，他也因此无法参加大部分的职业比赛，只能转型当一名网络游戏主播。<ref name="MAX+" />

2018年年中，吴权清的直播吸引了不少粉丝。在直播中，他的一些[[口头禅|口头禅]]和动作因为比较有特色而成为[[网络迷因|网络迷因]]，如其在直播过程中喝完汤用窗帘代替纸巾擦嘴，用带有茂名[[粤语|粤语]]口音的[[普通话|普通话]]说「我[[屌|屌]]你妈的」、「真一枪爆头啊都」等。<ref name="凤凰网电竞">{{cite web |title=经历过VAC事件之后 CSGO茄子的“真实”重生 |url=http://games.ifeng.com/a/20181224/45269497_0.shtml |website=凤凰网电竞 |date=2018-12-24 |access-date=2019-02-18 |archive-url=https://web.archive.org/web/20181224132452/http://games.ifeng.com/a/20181224/45269497_0.shtml |archive-date=2018-12-24 |dead-url=no }}</ref>

2018年10月31日，吴权清的[[Steam|Steam]]帐号被監管系统封禁。对此，有不少网友指责吴权清作弊，也有人对其进行谩骂，不过也有人相信吴权清没有作弊。11月1日，他的帐号被证实是判定错误而被解封。<ref name="凤凰网电竞" />

2019年1月29日，吴权清举行婚礼，「茄子结婚」这个关键词也登上当天[[新浪微博|新浪微博]]的热搜榜<ref>{{cite web |title=苦尽甘来 回顾火猫CSGO主播茄子的真实人生 |url=http://dj.sina.com.cn/article/hqfskcp1759821.shtml?sudaref=news.baidu.com&display=0&retcode=0 |website=新浪电竞 |date=2019-01-30}}</ref>。天禄战队的官方微博发文对刘权清的婚礼表示祝贺，他的粉丝则模仿他的语气在微博下方留言送上祝福<ref>{{cite web |author1=明镜Mirror |title=《CS：GO》主播茄子今日举行婚礼 网友纷纷“茄语”祝福 |url=https://ol.gamersky.com/news/201901/1149893.shtml |website=游民星空 |date=2019-01-29}}</ref>。
