'''栗原弘行'''（{{bd|1947年|2月18日|2014年|5月23日|catIdx=l栗}}<ref name="sankei"/>）出生于[[日本|日本]][[琦玉县|琦玉县]]，日本[[实业家|实业家]]、医疗研究家，伊豆修善寺生命体酸化还元疗法改良者和温冷热刺激疗法开发者，為[[钓鱼岛|钓鱼岛]]（日本稱[[尖閣諸島|尖閣諸島]]）的岛主。2014年5月23日，据产经新闻报道，栗原弘行因患肺癌在[[埼玉市|埼玉市]]去世，享年67岁<ref name="sankei">{{cite web|url=http://sankei.jp.msn.com/politics/news/140531/plc14053102000002-n1.htm|script-title=ja:尖閣諸島元地権者の弟、栗原弘行氏死去|accessdate=2014-08-13|publisher=[[产经新闻|产经新闻]]|language=ja|date=2014-05-30|deadurl=yes|archiveurl=https://web.archive.org/web/20140919100745/http://sankei.jp.msn.com/politics/news/140531/plc14053102000002-n1.htm|archivedate=2014-09-19}}</ref>。

== 生平 ==
1947年，正值[[大日本帝国|大日本帝国]]的最后一年，栗原弘行出生在[[琦玉县|琦玉县]]，家中人口繁多，父亲栗原利郎是大宫大地主名气高的财政界人物，[[菅原通济|菅原通济]]的友人。父亲死后，栗原弘行的哥哥开设了菱屋会馆株式会社。

1972年（[[昭和|昭和]]42年），古贺善次的妻子花子把钓鱼岛出售给栗原家族。

2010年（[[平成|平成]]22年1月）[[株式会社|株式会社]]SUPER SUB一级建筑事务所设立。

2012年（平成24年6月），栗原弘行担任特定非营利活动法人。2012年（平成24年）9月11日，栗原弘行把钓鱼岛低价[[釣魚台國有化|卖给日本政府]]。2012年10月，栗原弘行在《[[周刊朝日|周刊朝日]]》的被采访专文刊布，“（[[中华人民共和国|中华人民共和国]]）作为[[多民族国家|多民族国家]]必然不久就会分裂，无论作如何努力，10年后可能就会发生。”<ref>{{cite web |url=http://news.sina.com.cn/c/2012-10-12/011825340427.shtml |title=钓鱼岛“岛主”对华反日不满：扬言中国会分裂 |accessdate=2012年10月12日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

== 个人生活 ==
栗原弘行的妻子叫“栗原和子”，又名“栗原美代子”，两人育有女儿“栗原青皇”，由于栗原弘行喜好女人和金钱，曾在栗原青皇读幼儿园时逼迫妻子自杀，后因为欠债太多，两人离婚。<ref>{{cite web |url=http://news.qq.com/a/20120917/000081.htm |title=日媒称钓鱼岛“岛主”只爱金钱女人 日本被玩弄 |accessdate= 2012年9月17日|publisher= 腾讯网|work= |language =zh-cn}}</ref>
