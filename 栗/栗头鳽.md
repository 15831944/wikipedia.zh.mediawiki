'''栗头鳽'''（[[学名|学名]]：{{lang|la|''Gorsachius goisagi''}}，英文：'''{{lang|en|Japanese Night Heron}}'''）为[[鹭科|鹭科]][[鳽属|鳽属]]的[[鸟类|鸟类]]，又名'''-{zh-cn:麻鹭;zh-tw:栗頭鳽;}-'''，俗名'''粟头虎斑鳽'''、'''栗头夜鳽'''。牠們會到琉球群島和菲律賓等地過冬，飛行路線上海、福建和香港。目前分布于日本、印度尼西亚、菲律宾、[[台湾|台湾]]以及[[中国大陆|中国大陆]]的[[浙江|浙江]]、[[上海|上海]]、[[福建|福建]]、[[广东|广东]]等地，一般栖息于海湾、河口及附近稻田、芦苇丛、树丛中。该物种的模式产地在日本。<ref name='id1376'/>據[[香港觀鳥會|香港觀鳥會]]表示，現存栗鳽數量或少至一千隻，而且有下降趨勢。<ref>{{Cite web |url = http://hk.apple.nextmedia.com/realtime/news/20150402/53595299 |title = 瀕危雀鳥「栗鳽」26年後再現香港 |date = 2015-04-02 |publisher = 蘋果新聞 |language = zh-hant |accessdate = 2015-04-03}}</ref>
