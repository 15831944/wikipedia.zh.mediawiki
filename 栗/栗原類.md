== 簡介 ==
混血兒，父親是英國人，母親是日本人<ref name="mantan20120820" />。

5歲時開始做[[模特兒|模特兒]]，初中2年級起登上時尚雜誌『MEN'S NON-NO』<ref>{{Cite web|date=2012-07-03|url=http://www.cyzo.com/2012/07/post_10903_2.html|title=「たぶん、一生モテないと思います……」“ネガティブすぎるイケメンモデル”栗原類の素顔|page=2|work=サイゾー|accessdate=2012-09-18|deadurl=yes|archiveurl=https://web.archive.org/web/20121002081530/http://www.cyzo.com/2012/07/post_10903_2.html|archivedate=2012-10-02}}</ref><ref name="postseven20120819">{{cite web|date=2012-08-19|url=http://www.news-postseven.com/archives/20120819_136539.html|title=話題のネガティブすぎる美形モデル「“気持ち悪い”は褒め言葉」|accessdate=2012-09-18|publisher=[[小学館|小学館]]}}</ref>。

2012年，正就讀高中的他在上綜藝節目『アウト×デラックス』和『芸能★BANG+』時<ref>{{Cite web |date=2012-07-03|url=http://www.cyzo.com/2012/07/post_10903.html|title=「たぶん、一生モテないと思います……」“ネガティブすぎるイケメンモデル”栗原類の素顔|page=1|work=サイゾー|accessdate=2012-09-18}}</ref><ref name="nikkei20120703">{{Cite web |url=http://trendy.nikkeibp.co.jp/article/column/20120702/1041756/?ST=life&P=2|title=時代を変える？「イケメンなのに超ネガティブ」なモデルは!?|page=2|work=日経トレンディ|accessdate=2012-11-01}}</ref>、由於較多的悲觀言論而被冠上“消極模特”就此出名<ref name="mantan20120820" /><ref name="postseven20120819" /><ref name="oricon20120820" /><ref name="wpb20120811">{{Cite web |date=2012-08-11|url=http://wpb.shueisha.co.jp/2012/08/11/13266/|title=話題沸騰の大人気モデル・栗原類インタビュー「休み時間にアリを観察しているようなボクがモテるわけないじゃないですか……（怒）」|publisher=[[集英社|集英社]]|accessdate=2012-09-18|page=1}}</ref>。同年8月，初次參演電視劇『よく眠れる森のレストラン〜アナタノココロをおもてなし〜』<ref name="oricon20120820" /><ref name="sanspo20120820">{{Cite web |date=2012-08-20|url=http://www.sanspo.com/geino/news/20120820/oth12082005020004-n1.html|title=“ネガイケ”で人気！栗原類、俳優デビュー|work=サンケイスポーツ|accessdate=2012-09-18}}</ref>。

10月首次出席時尚活動『[[東京Girls_Collection|東京Girls Collection]]』<ref>[http://mantan-web.jp/2012/10/14/20121013dog00m200029000c.html 『栗原類 ： TGCに初登場も笑顔なし』] 2012年10月14日 ''まんたんウェブ''</ref>。11月發售第一本個人[[寫真集|寫真集]]『ネガティブですが、なにか？』<ref>[http://mdpr.jp/021211105 『栗原類がネガティブになった理由　昔は「楽天的でした」』] 2012年11月10日 ''モデルプレス''</ref>。
