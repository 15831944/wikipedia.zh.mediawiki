'''恆春線'''是在[[台灣|台灣]][[南台灣|南部]]的[[屏東縣|屏東縣]]計畫興建之傳統鐵路支線，自[[內獅車站|內獅車站]]與[[南迴線|南迴線]]分歧後通往[[恆春鎮|恆春]]；其目的為紓解[[恆春半島|恆春半島]]及[[墾丁國家公園|墾丁國家公園]]長期以來的聯外交通壓力。2002年完成選線規畫<ref>《[http://www.iot.gov.tw/public/Data/7121811101671.pdf 興建與營運屏東大鵬灣至恆春、墾丁地區軌道系統可行性研究] {{webarchive|url=https://web.archive.org/web/20150924035918/http://www.iot.gov.tw/public/Data/7121811101671.pdf |date=2015-09-24 }}》，ISBN 801621378-2，2002年9月，交通部運輸研究所。</ref>。另外也規劃自車城分歧的海生館線，服務[[國立海洋生物博物館|國立海洋生物博物館]]的對外交通需求。

但由於2005年5月3日[[中華民國交通部|中華民國交通部]]進行「恆春半島研議中之鐵路及國道南延案競合分析」結論建議應先以[[鐵路電氣化|鐵路電氣化]]延伸至[[枋寮車站|枋寮]]再以[[公共汽車|觀光巴士]]接駁轉運，故計畫一度擱置<ref>[http://www.pthg.gov.tw/Upload/RelFile/FDL/9793/634407301237865000.pdf]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}，屏東縣政府</ref>。而恆春居民不時請願<ref>[http://udn.com/NEWS/DOMESTIC/DOM6/7143168.shtml 恆春人的心聲：我們需要鐵路]，2012年6月7日，聯合報</ref>，甚至成為各黨派的政見<ref>[http://news.chinatimes.com/domestic/11050612/112013042400315.html 改善恆春交通 藍綠各有撇步 綠委潘孟安建議儘早完成國道三號延伸至恆春半島 藍營則指恆春該發展的是輕軌鐵路] {{webarchive|url=https://web.archive.org/web/20130430064730/http://news.chinatimes.com/domestic/11050612/112013042400315.html |date=2013-04-30 }}，2013年4月24日，中國時報</ref>。[[交通部鐵路改建工程局|交通部鐵路改建工程局]]預估，如果有了恆春支線，從高雄新左營站搭自強號到恆春，大約是90分鐘，再搭公車約15分鐘可到墾丁<ref>[http://www.cna.com.tw/News/FirstNews/201309020072-1.aspx 台鐵擬建恆春支線 方便遊墾丁]，2013年9月2日，中央通訊社</ref>；在2013年，時任[[中華民國交通部|交通部]]部長[[葉匡時|葉匡時]]表示，2014年將進行臺鐵恆春支線可行性研究<ref>[http://www.rrb.gov.tw/02110.aspx?lan=ch&id=460 交通部葉部長肯定鐵工局屏潮計畫西正線高架鐵路通車啟用成效，並宣布明年進行臺鐵恆春支線可行性研究]，2013年6月25日，交通部鐵路改建工程局</ref><ref>[http://www.bcc.com.tw/news/newsview.asp?cde=2116820 葉匡時：台鐵枋寮至恆春 可研究]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}，2013年6月25日，中國廣播公司</ref>。

2015年時，鐵路改建工程局完成《台鐵恆春支線可行性計畫期末報告》。2017年，[[國家發展委員會|國家發展委員會]]通過交通部陳報《恆春觀光鐵路可行性研究》，並列入[[前瞻基礎建設計畫|前瞻基礎建設計畫]]，預計在綜合規劃（含環境影響評估）核定後約6.5年完工，於2026年6月啟用。<ref>[http://www.ndc.gov.tw/News_Content.aspx?n=114AAE178CD95D4C&sms=DF717169EA26F1A3&s=ACF5EAD9AAA813E6 國家發展委員會於今(11)日第40次委員會議討論通過交通部陳報「恆春觀光鐵路可行性研究」]，國家發展委員會，2017年4月11日</ref>

== 路線資料 ==
[[File:HengchunLine.png|right]]
* 經營管轄：[[臺灣鐵路管理局|臺灣鐵路管理局]]
* 路線距離：（恆春線）內獅＝恆春間 37.72 公里
* [[軌距|軌距]]：1,067毫米
* 車站數：8（恆春線含起訖車站）
* 雙線區間：無，全線單線
* [[電氣化鐵路|電化]]區間：全線電化<ref>{{cite web |author1=交通部鐵道局 |title=恆春觀光鐵路建設計畫規劃作業 |url=https://www.rb.gov.tw/showpage.php?lmenuid=3&smenuid=166&tmenuid=196&pagetype=0 |accessdate=2018-11-12}}</ref>。
* 海口以南路段將採高架化興建<ref>{{cite web |author1=交通部鐵路改建工程局 |title=恆春觀光鐵道懶人包 |url=https://www.rrb.gov.tw/02310.aspx?lan=ch&id=760 |accessdate=2018-8-11}}</ref>。
* 因應長久來[[墾丁國家公園|墾丁國家公園]]嚴重的聯外交通問題而提出。
* 在[[交通部運輸研究所|交通部運輸研究所]]的《興建與營運屏東大鵬灣至恆春、墾丁地區軌道系統可行性研究》報告書中，預計部分線路與[[墾丁|墾丁地區]]計畫中的[[輕軌運輸|輕軌運輸]]系統共軌，唯當時交通部認為應以公車替代。

== 運行形態 ==
* [[新左營車站|新左營站]]＝恆春車站間開行直通列車，快車約90分鐘、慢車約141分鐘，以便旅客利用[[台灣高速鐵路|高鐵]]轉乘。

==使用車輛==
* 柴油車
根據2018年6月1日自由時報報導，恆春觀光鐵道規劃採用郵輪式車廂，配落地窗可沿途欣賞美景。

== 簡介 ==
[[恆春鎮|恆春]]舊名「瑯嶠」，自[[清代|清代]]即為重要聚落，且曾為[[恆春縣|恆春縣]]縣治所在地。
=== [[台灣清領時期|清領時期]]計畫 ===
光緒二年（1876）福建省巡撫丁日昌基於國防需求，已有計畫興建基隆至恆春鐵道的構想。<ref>http://www.rrb.gov.tw/s/south/index.aspx?sid=1&ssid=223 : 交通部鐵路改建工程局【搭配高雄舊車站帝冠式建築物之原始藍圖，介紹高雄市地名演進歷史】</ref>

=== [[台灣日治時期|日治時期]]計畫 ===

早在1900年代，打狗（今[[高雄港車站|高雄港]]）鐵路通車後，總督府即有延伸至鳳山、阿緱（屏東）、潮州、恆春的構想，也就是後來的[[屏東線|屏東線]]<ref>《打狗通信（十二月十一日）　打狗以南縱貫鐵道豫定線の視察》，1902年12月16日，臺灣日日新報</ref>。1923年鐵路通至溪州（今[[南州車站|南州]]）後，鐵道部一度計畫1924年興建枋寮＝枋山間鐵道，並在1927年後完成至恆春<ref>《鐵道延長線將著手》，1923年7月15日，臺灣日日新報</ref>，唯因預算因素而中輟<ref>《鐵道延長線將著手》，1923年8月22日，臺灣日日新報</ref>。

由於溪州（今南州）以南路線建設因預算因素長期擱置，延伸恆春成為懸案，當局決定以鐵道部經營的自動車行駛<ref>《溪州延長線挨延　將先依自動車運轉》，1932年11月22日，臺灣日日新報</ref>。1940年，鐵道延伸至枋寮。1941年，交通局營自動車開始行駛至恆春。

=== [[台灣戰後|戰後]]至今 ===
戰後，居民屢次陳情，希望鐵道延伸恆春。其中，日治時期與戰後初期[[南迴線|南迴線]]（連結屏東縣、臺東縣）的探勘路線中，也曾經迂回恆春的比較線案。但由於路線過長、產值不高，跟公路相比沒有競爭力，因此並未採用。

1984年，[[恆春半島|恆春半島]]設立[[墾丁國家公園|墾丁國家公園]]後，成為著名觀光區，吸引眾多國內外旅客，而日漸惡化的連外交通也成為急需面對的課題。2004年[[恆春機場|恆春機場]]雖於完工，但受到落山風影響，至今仍常造成班機停飛。2005年[[颱風海棠_(2005年)|海棠颱風]]造成[[台1線|台1線]]楓港橋沖毀，一度造成恆春半島對外陸運完全中斷。

2002年，運輸研究所完成《興建與營運屏東大鵬灣至恆春、墾丁地區軌道系統可行性研究》。希望鐵路分擔公路車流，並計畫配合高鐵轉乘，讓旅客能在三小時自[[臺北市|臺北]]抵達[[恆春鎮|恆春]]。恆春線另有自本線分歧的海生館線，以因應[[國立海洋生物博物館|國立海洋生物博物館]]、墾丁地區旅客的觀光需求。但由於2005年5月3日中華民國交通部進行《恆春半島研議中之鐵路及國道南延案競合分析》，結論建議應先以鐵路電氣化延伸至枋寮，再以觀光巴士接駁轉運，故計畫形同擱置。

交通部部長葉匡時於2013年表示，2014年將進行臺鐵恆春支線可行性研究。

2015年12月，鐵路改建工程局完成台鐵恆春支線可行性計畫期末報告，報請交通部、行政院核定，最快2020年動工、2025年啟用<ref>[http://news.ltn.com.tw/news/focus/paper/938287 台鐵恆春夢幻支線 盼10年成真]</ref>。

2017年4月，[[行政院|行政院]]提出「前瞻基礎建設計畫」後，將台鐵恆春線核定為「恆春觀光鐵道計畫」，總經費新台幣199.36億元。

2018年10月底、交通部核定可行性研究報告、陳報行政院<ref>[http://news.ltn.com.tw/news/politics/breakingnews/2610927 恆春觀光鐵道可行性研究 陳報行政院核定]2018-11-13,自由時報</ref>。

== 車站一覽 ==
自內獅站以南分歧後，大致沿[[台26線|台26線]]（屏鵝公路）續行至恆春。2002年的《興建與營運屏東大鵬灣至恆春、墾丁地區軌道系統可行性研究》中，建議新設枋山新、楓港、車城、五里亭、恆春5車站，以及枋山、善餘村、尖山3號誌站。2017年核定的計畫，調整為新建枋山新、楓港、竹坑、海口、車城、五里亭、恆春共7站，另設置枋山號誌站。
