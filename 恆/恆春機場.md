'''恆春航空站'''，常稱作'''恆春機場'''，為[[臺灣|臺灣]]民航機場之一，位於[[臺灣|臺灣]][[屏東縣|屏東縣]][[恆春鎮|恆春鎮]]，以飛行國內航線為主，是[[臺灣|臺灣]]本島最南端、最小型的機場。因機場地處[[恆春古城|恆春古城]]北方之五里亭，因此也有以地名稱之，名為「'''五里亭機場'''」。

== 歷史 ==
恆春機場於[[台灣日治時期|台灣日治時期]]稱為「恆春飛行場」，依照[[中華民國國軍|中華民國國軍]][[二戰|二戰]]後接受資料為1943年8月落成啟用，但依據日治時期的《飛行機隊戰闘行動調書》顯示，恆春飛行場應於1941年就存在。<ref>[http://cwhung.blogspot.tw/2015/10/blog-post.html?m=1 現在養蚊子的恆春機場，竟然是太平洋戰爭開打第一天就登場的重要飛行場]，不沈空母--台灣島內飛行場百年發展史，[[洪致文|洪致文]]著</ref>

2000年3月發包將軍用機場改建為民用機場，2003年12月29日竣工，於2004年1月12日啟用。共投資新台幣六億多元完成。[[立榮航空|立榮航空]]、[[復興航空|復興航空]]、[[華信航空|華信航空]]開辦[[臺北松山機場|臺北松山]]航線，華信航空亦開辦[[臺中清泉崗機場|臺中]]航線。

但是恆春機場原始設計是雙跑道運行，主風向跑道正對落山風，現為[[陸軍航空特戰指揮部|陸軍航空特戰指揮部]]龍勤營區與航空站之聯絡道，當初的雨季風向跑道擴建後為目前使用民航跑道。受地形氣候影響，[[恆春半島|恆春半島]]每年10月到隔年4月盛行[[落山風|落山風]]，跑道方向導致落山風成為是標準的側風，因[[交通部民用航空局|交通部民用航空局]]規定ATR機型在本機場最大[[側風|側風]]限制為10KT，冬季常超過標準造成班機無法起降，客人相繼流失。恆春機場平均每年虧損約新台幣6000萬元。華信航空臺中航線在恆春機場一星期內採不定時停飛。<ref>{{Cite news|url=http://www.appledaily.com.tw/appledaily/article/headline/20040116/648956/|title=恆春機場開航5天停飛2天|author=蔡宗憲|date=2004-01-16|work=|newspaper=蘋果日報|accessdate=|language=}}</ref>2007年8月10日復興航空、華信航空停飛臺北航線。2012年10月交通部民用航空局表示，已經考慮將此機場關閉。<ref>{{Cite news|url=http://tw.news.yahoo.com/%E5%B9%B4%E8%99%A76%E5%8D%83%E8%90%AC-%E6%81%86%E6%98%A5%E6%A9%9F%E5%A0%B4%E7%AD%89%E9%97%9C%E9%96%80-135208719.html;_ylt=AnOqNNfP4BHHrDFhrnHbeKc5AF86;_ylu=X3oDMTM2YTYzZWFpBG1pdANJbmZpbml0ZSBCcm93c2UgVFcEcGtnAzU2OWViOWMxLTM5N2QtM2IxNy1hZGYwLWIyYTI0NmVjNWEzYQRwb3MDbDMEc2VjA21lZGlhaW5maW5pdGVicm93c2VsaXN0dGVtcA--;_ylg=X3oDMTM1azZha2I4BGludGwDdHcEbGFuZwN6aC1oYW50LXR3BHBzdGFpZAMwOTU1ZTQxZC0yYTJjLTNmYzEtODAyMC00YmY5MDBhNmY2MzEEcHN0Y2F0A.eUn.a0u3zkuqTpgJoEcHQDc3RvcnlwYWdl;_ylv=3|title=年虧6千萬 恆春機場等關門|author=|date=2012-10-15|work=YAHOO奇摩新聞|publisher=|accessdate=|language=}}</ref>

雖然過去每週兩班有立榮航空從台北飛往恆春，但實際上從2014年9月至今，完全沒有航班飛航。

== 利用情況 ==
恆春機場位於恆春古城北方之五里亭，距[[墾丁|墾丁]]僅十五分鐘車程，以地理位置來說確實可以大幅節省往返墾丁的時間，每週有2班次（週二、四）往來台北。然而冬季常因[[落山風|落山風]]關閉，或改降[[高雄國際機場|高雄國際機場]]，在第二條跑道完工前，實用性很低<ref>{{Cite web|url=http://www.dgnet.com.tw/articleview.php?article_id=8638&issue_id=1729|title=20多座蚊子機場 台灣另類奇蹟|accessdate=2018-01-05|author=|date=|work=|publisher=雜誌生活網|language=}}</ref>，且因為使用率過低，被譏為只能養蚊子的「蚊子機場」。近日已有廢除恆春機場的要求。

除定期旅客航線外，自2006年6月起亦有民間公司開行[[直升機|直升機]]觀光路線，遊程約十餘分鐘，但已停止此服務。

航空業人士表示，恆春機場因離到場程序需經過核三廠限航區，萬一起降過程單發失效怕無法避開限航區，為高風險機場，不願意飛行。超輕型飛機業者則估計，恆春機場每年無法正常起降的時間，可能超過半年以上。

若能開闢第二跑道，則可以讓恆春機場實用化，但開闢第二跑道不但經費龐大，而且使用第二跑道起降也有跑道兩頭又面臨山脈造成起降困難的問題。<ref>{{Cite news|url=http://www.epochtimes.com/b5/7/5/18/n1715057.htm|title=民航局初估 恆春機場建第2跑道經費10億|author=|date=2007-05-18|work=|newspaper=大紀元台灣|accessdate=|language=}}</ref>

2004年台北－恆春航線載客數有2萬3000多人次，但之後就一路下滑，2006年為1萬1000多人次，到2011年只有2448人次，約開航第1年的1/10。
2014年台北－恆春航線載客數首次降至千人以下，全年才768人次，2014年8月只飛了兩班，載了21人次，2014年9月至2019年1月近53個月航班數及載客人數為零。<ref>{{Cite news|url=http://news.ltn.com.tw/news/local/paper/966836|title=停飛18個月 恆春機場7月定去留|author=蔡宗憲|date=2016-03-10|work=自由時報電子報|accessdate=}}</ref><ref>{{Cite news|url=http://www.cna.com.tw/news/firstnews/201510030036-1.aspx|title=一整年沒人搭 台北恆春形同幽靈航線|author=汪淑芬|date=2015-10-03|work=中央社|accessdate=}}</ref>

2017年8月，安捷飛航訓練中心由總機師從台東機場搭載兩名教官，沿著海岸線飛行，在半小時內抵達恆春機場，除了測試越野飛行程序起降，機場塔台及消防人員都配合演練，為近三年來首度有搭載人員的民航訓練機起降。<ref>[http://m.ltn.com.tw/news/local/paper/1128056 《三年來頭一遭》恆春航空站 民航訓練機起降]</ref>

== 改善計畫 ==
恆春航空站2014年一整年載量只有700人，環境幾近蚊子館<ref>{{Cite news|url=http://a.udn.com/focus/2016/07/28/23513/index.html|title=兩年0旅客 恆春蚊子機場 居然還關不掉？|author=邱瓊平,謝蕙蓮|last=|first=|date=2016-07-28|work=聯合報 Focus|newspaper=|accessdate=2016-07-28|language=}}</ref>，民航局正進行活化機場計畫，[[屏東縣政府|屏東縣政府]]爭取要將恆春機場升格為國際機場，吸納港、澳及大陸、韓國及日本遊客。
目前到墾丁的香港、澳門遊客很多，多是飛抵小港機場，但小港機場有宵禁，因此，縣府將評估恆春機場直飛港、澳的經濟效益，另外，以恆春為軸心，4小時航程的城市也將進行評估，包括中國大陆的上海及北京、韓國首爾、日本的東京、大阪及神戶。但恆春機場本身已因多次落山風的關係影響國內航班起降，能否開拓國際航線就更是疑問，因此屏東縣政府的評估計畫亦被認為不切實際。

== 近況 ==
2019年3月31日，屏東縣政府追加「恆春航空站改善中長期計畫」，提出四帖長期調理藥方，包含延續國際包機計畫、升格為屏東恆春國際機場、請民航局規劃設計[[中長期裝設儀器降落系統|中長期裝設儀器降落系統]]（ILS）、延長既有跑道、增設側風跑道等，能讓恆春機場起死回生。立委[[鍾佳濱|鍾佳濱]]認為，一是跑道長度延長，能讓不受側風影響的大型飛機起降，二是調整跑道方向避免落山風，但這都得交給專業評估，安全最重要，得先談適航性，也就是硬體條件改變，再來談市場性。<ref>[https://tw.news.yahoo.com/%E6%81%86%E6%98%A5%E6%A9%9F%E5%A0%B4%E9%A4%8A%E8%9A%8A16%E5%B9%B4-%E5%B1%8F%E7%B8%A3%E6%8B%9A4%E5%BA%A6%E6%95%91%E6%8F%B4-061414547.html 恆春機場養蚊16年 屏縣拚4度救援]yahoo奇摩新聞 聯合新聞網 記者賴于榛／台北報導 2019年3月31日 下午2:14</ref>
