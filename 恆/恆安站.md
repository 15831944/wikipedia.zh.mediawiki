'''恆安站'''（{{lang-en|Heng On Station}}）是[[港鐵|港鐵]][[馬鞍山綫|馬鞍山綫]]位於[[香港|香港]][[馬鞍山_(香港市鎮)|馬鞍山]]的一個鐵路車站，於2004年12月21日啟用。車站取名來自鄰近公共屋邨[[恆安邨|恆安邨]]。

===大堂===
{{ multiple image |align=left | direction = horizontal | width = 220
| image1 = MTR HEO (11).JPG | caption1 = 地底大堂
}}
恆安站主體建於[[西沙路|西沙路]]行車線中央，共設有兩個大堂，分別設於地面層及地庫層，各大堂均設有客務中心。該站唯一一間商店設於於C出入口，為[[7-Eleven|7-Eleven]]便利店，另外亦有[[自動櫃員機|自動櫃員機]]等自助設施<ref name="shops"/>。

[[File:Heng_On_Station_2017_08_part3.jpg|thumb]]

===月台===

恆安站位於西沙路中央地面，是馬鞍山綫其中一個架空車站，並設有2個月台，以1個島式月台設計排列。月台可容納8卡列車，初時月台只建成其中4至5卡長度，其餘部分會於沙中線興建時於預留結構上擴建，現時擴建工程已經完成。此外，車站南面設有[[渡线|渡线]]，方便列車調動或信號故障時才使用。

===出入口===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Heng On Station 2017 08 part4.jpg | caption1 = B出入口
| image2 = MTR HEO (3).JPG | caption2 = C出入口與7-11便利店

恆安站共設有三個出入口，一個位於大堂北面，為頌安邨及錦豐苑居民使用（C出口），其他兩個位於大堂中央地底隧道，一個面向恆安邨和錦鞍苑（B出口），一個面向觀瀾雅軒及聽濤雅苑（A出口）。

==鄰接車站==
恆安站是馬鞍山綫的中途站，乘客可在此乘坐往烏溪沙或大圍列車，往返[[九龍|九龍]]及[[新界|新界]]等地。

==利用情況==
由於恆安站位於西沙路的開端，進出站內的多為[[恆安邨|恆安邨]]、[[錦鞍苑|錦鞍苑]]、[[聽濤雅苑|聽濤雅苑]]、[[觀瀾雅軒|觀瀾雅軒]]的住客，而也有不少在不遠處的[[頌安邨|頌安邨]]及[[錦豐苑|錦豐苑]]居民使用此站乘搭馬鞍山綫。
