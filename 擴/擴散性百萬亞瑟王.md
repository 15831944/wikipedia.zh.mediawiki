== 概要 ==
擴散性百萬亞瑟王是[[史克威爾艾尼克斯|SQUARE ENIX]]旗下針對智慧型手機市場的[[子公司|子公司]][[SiSiLaLa_OVERDRIVE|SiSiLaLa OVERDRIVE]]於2011年9月15日在[[東京電玩展|2011東京電玩展]]所發表的遊戲<ref>{{Cite web |url= http://news.dengeki.com/elem/000/000/406/406205/|title= 【App通信】スクエニがスマホ向けの完全新作カードバトルRPGを発表！ シナリオ・鎌池和馬、音楽・ヒャダイン、参加絵師は原田たけひとら50人以上！|date=2011-09-15|publisher= [[ASCII_Media_Works|ASCII Media Works]]|language= ja|accessdate= 2012-06-05}}</ref>。標題「百萬亞瑟」是指，故事的舞台是有著上百萬名[[亞瑟王|亞瑟王]]以及上百萬把[[石中劍|石中劍]]的[[大不列顛|大不列顛]]，而玩家要扮演其中一名亞瑟王，並且參與戰鬥和故事的幻想[[電子角色扮演遊戲|RPG]]。遊戲本體的製作于2012年3月23日完成<ref>{{Cite web |url = http://app.famitsu.com/20120402_50423/|title= スクエニプロデューサー安藤武博氏のブログ“スマゲ★革命”第十五回　「拡散性プロデューサー集団」|date=2012-04-02|publisher= [[Enterbrain|Enterbrain]]|accessdate =2012-06-05|language= ja}}</ref>，預定于3月29日在[[App_Store_(iOS)|蘋果App Store]]發佈，但是因為[[蘋果公司|蘋果公司]]的審查而延期至4月9日發佈。

遊戲腳本由[[鎌池和馬|鎌池和馬]]擔任，同时負責[[世界觀|世界觀]]以角色性格等細部設定<ref name="famitsu">{{Cite journal en|title = スクウェア・エニックス スマホゲーム特集 拡散性ミリオンアーサー|journal = [[fami通|fami通]] No.1216|date = 2012-04-05（3月22日發售）|pages = 85-87|language= ja}}</ref>。負責音樂的[[前山田健一|ヒャダイン]]除了作中的[[背景音樂|背景音樂]]和[[原聲音樂|插入曲]]之外，也參與了[[主題曲|主題曲]]的作詞、作曲以及編曲。而角色設定以及作畫是由50名以上的繪師擔任。

本作可以經部分地區的[[App_Store_(iOS)|蘋果App Store]]和[[Google_Play|Google Play]]免費下載（包括日本、香港、台灣等），遊戲中有部分內容是付費的，不過即使沒有付費也不會影響遊戲的進行。<ref>{{Cite web |url=http://live.nicovideo.jp/gate/lv85175432|title = 【ゲスト：ヒャダイン、ちょうちょ】シシララ！第２回：拡散性ミリオンアーサー|publisher = niwango,inc.|accessdate = 2012-06-05|language= ja}}</ref>

== 遊戲內容、系統 ==

本作的主角們分別是從三個不同的派系而來的亞瑟王，在遊戲一開始需要選取其中一人作為主角。一開始的場景是亞瑟拔出石中劍的那一刻，成為亞瑟王並且與支援妖精以及下屬的騎士一起與其他的亞瑟王爭奪霸權並且抵抗「外敵」的入侵。故事的內容會因為所選的派系不同而有些微的差異。

戰鬥進行的方式是卡片戰鬥的形式，將騎士卡片設置在一個圓桌狀的盤上，由這些卡片來自動進行戰鬥。卡片主要有HP（生命值）、ATK（攻擊值）和COST（戰鬥點數消耗值）三種屬性，每張卡片均有不同概率觸發自身技能，達到增加ATK或者回復HP的目的。不同的特定卡片組合能生成「Combo」，用以強化牌組或者卡片的HP、ATK和技能發動率等。每進行一次攻擊，可以增加5%的「SUPER」，當增加到100%時主角會使用石中劍進行強力的攻擊，傷害值由當前牌組的HP和ATK總值決定。另外，也可以透過卡片合成來強化手上的卡片。

== 角色 ==

=== 劍術之城 ===
; 亞瑟
: 聲優 - [[宮野真守|宮野真守]] / 繪師 - [[灰村清孝|灰村清孝]]
: 被石中剑所选择的成为王的其中一人，正义感很强，对不讲道理的人十分愤怒。在漫畫版《群青の守護者》的名字是伊奧斯，作為「最接近王的男人」登場。被玩家簡稱為「劍瑟」。
; 菲
: 聲優 - [[伊藤加奈惠|伊藤加奈惠]] / 繪師 - 灰村清孝
: 帮助亚瑟的妖精。经常对亚瑟说敬语。在游戏数据更新、选择亚瑟派系时，菲都会登场说几句话。
; [[蘭斯洛特|蘭斯洛特]]
: 聲優 - [[神谷浩史|神谷浩史]] / 繪師 - [[黑星紅白|黑星紅白]]
: 被“湖”所制造出来的、亚瑟所接触到的最古老的骑士。性格稳重。被称作“湖之骑士”。剑术之城故事中的主要登场角色。

=== 技巧之場 ===
; 亞瑟
: 聲優 - [[佐藤利奈|佐藤利奈]] / 繪師 - 田中雄一
: 被石中剑所选择的成为王的其中一人，性格粗暴、嘴賤。有[[双性恋|双性恋]]的倾向，有时表现为美少女，有时美少年。在漫畫版《群青の守護者》的名字是莉格。被玩家簡稱為「技瑟」。
; 莉芙
: 聲優 - [[喜多村英梨|喜多村英梨]] / 繪師 - 灰村清孝
: 帮助亚瑟的妖精。相较于艾兒，无论是言行还是相貌都颇具大人的样子。
; [[高文_(亞瑟王傳說)|高文]]
: 聲優 - [[浪川大輔|浪川大輔]] / 繪師 - 凪良
: 被“湖”所制造出来的、亚瑟所接触到的最古老的骑士。性格热情。喜欢在语句末尾加上“～哟（～かよ）”这样的口头禅。技巧之场故事中的主要登场角色。

=== 魔法之派 ===
; 亞瑟
: 聲優 - [[釘宮理惠|釘宮理惠]] / 繪師 - [[BUNBUN|BUNBUN]]
: 被石中剑所选择的成为王的其中一人，看起来女性的外表要多于男性。性格比较随众，但在关键的时刻却能展现出强大的意志。在漫畫版《群青の守護者》的名字是亞特。被玩家簡稱為「魔瑟」。
; 艾兒
: 聲優 - [[井口裕香|井口裕香]] / 繪師 - 灰村清孝
: 帮助亚瑟的妖精。看起来就像个小孩子，言行及外观都十分的孩子气。
; [[加拉哈德|加拉哈德]]
: 聲優 - [[花澤香菜|花澤香菜]] / 繪師 - BUNBUN
: 以兰斯洛特的因子为基础构造的骑士。金色长发的美丽女性。一旦听到有关“断绝时代”的技术的话题便开始插嘴一些似乎不怎么相关的话题，但实际上对亚瑟很言听计从。魔法之派故事中的主要登场角色。

== 叛逆性百萬亞瑟王 ==
