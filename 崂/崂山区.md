'''崂山区'''，是[[青岛市|青岛市]]的一个[[市辖区|市辖区]]，于1994年成立，位于青岛市区东部，属青岛市市内四区之一。东南濒临[[黄海|黄海]]。

== 行政区划 ==
崂山区下辖金家岭、中韩、沙子口、王哥庄、北宅等5个街道、139个农村社区、22个城市社区。

== 文化教育 ==

[[青岛国际啤酒节|青岛国际啤酒节]]举办场所——青岛国际啤酒城、[[青岛科技大学|青岛科技大学]]东部校区、[[青岛大学|青岛大学]]、[[中国海洋大学|中国海洋大学]]东部校区（分为崂山校区和浮山校区，其中崂山校区已经成为主要校区）、青岛标志性建筑--青岛国际会展中心、青岛市博物馆、海尔科技宫、[[青岛大剧院|青岛大剧院]]、[[青岛市体育中心|青岛市体育中心]]在此、另外，青岛市中级人民法院、青岛规划展览馆也位于此区，

== 旅游 ==

崂山区有著名的国家旅游度假风景区——[[石老人|石老人]]、国家5A级风景区——[[崂山|崂山]]风景区、同时是传说崂山是中国[[道教|道教]]的发源地之一。

== 经济 ==
著名企业[[海尔|海尔]]集团总部座落在海尔路北端，国家级高科技工业园区，中国科学院青岛生物能源与过程研究所，国家海洋局第一海洋研究所。区政府大厦前轴线广场（地下部分）、海尔路两端、海尔路与辽阳西路交界处为青岛重要高档商圈之一。
