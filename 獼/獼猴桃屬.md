'''獼猴桃屬'''（学名：{{lang|la|''Actinidia''}}）是一種源產於[[亞洲|亞洲]]東部地區的[[木本植物|木本植物]]。而「猕猴桃」（又稱「'''奇异果'''」，kiwifruit）則是「猕猴桃属」中多个栽培种的通称，包括原产于中国的[[美味猕猴桃|美味猕猴桃]]，以及此物种与猕猴桃属中的其他物种之杂交种。

== 分布 ==
獼猴桃屬植物分布在[[中國大陸|中國大陸]](主要集中在[[河南省|河南省]]、[[陕西省|陕西省]]的[[秦岭|秦岭]]地区及其余脉与[[太行山|太行山]]区，[[浙江省|浙江省]]的[[衢州市|衢州市]][[江山市|江山市]]以及[[湖南省|湖南省]]西部的[[武陵山|武陵山]]区)、[[台灣|台灣]]、[[朝鲜|朝鲜]]與[[日本|日本]]，並且向北延伸到[[西伯利亞|西伯利亞]]東南方，向南延伸到[[中南半岛|中南半岛]]。

== 外形 ==
獼猴桃屬植物能長到6米高，並且能夠生出蔓藤，在樹陰下生長到30米長。

== 繁殖 ==
獼猴桃屬植物[[雌雄異株|雌雄異株]]，栽培上以一株雄株配合7～8株母株。

== 營養 ==
猕猴桃被认为是营养密度最高的水果，低[[钠|钠]]高[[钾|钾]]，含有丰富的[[维生素C|维生素C]]、[[维生素E|维生素E]]、[[叶酸|叶酸]]、[[膳食纤维|膳食纤维]]、多种[[氨基酸|氨基酸]]。研究显示，猕猴桃可补充身体中的钙质，增强人体对食物的吸收力。此外，猕猴桃外皮含有丰富[[果胶|果胶]]（佔所含纤维的三分之一），可降低血中[[胆固醇|胆固醇]]。
