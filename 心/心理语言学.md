'''心理语言学'''是通过[[心理学|心理学]]来研究[[语言学|语言学]]的学科。他从心理过程和语言结构的对应关系来研究人类[[语言|语言]]机制的构造和功能。从心理学的角度，它属于[[认知心理学|认知心理学]]；从语言学的角度，它属于[[实验语言学|实验语言学]]。

== 历史 ==

心理语言学（''Psycholinguistics''）一词是1936年[[美国|美国]]心理学家[[坎特|坎特]]在其著作《语法的客观心理学》中开始使用的。但是作为心理语言学成为一个学科的标志则是1953年在美国[[印第安纳大学|印第安纳大学]]举办的“心理语言学”讨论会，以及1954年[[奥斯古德|奥斯古德]]和[[西比奥克|西比奥克]]在此基础上合编的《心理语言学——理论和研究问题概述》。

心理语言学在1950年代时，其理论基础主要是[[行为主义心理学|行为主义心理学]]。此时认为语言的学习就是不断的刺激、强化，儿童对语言的学习就是不断地对环境刺激做出反应，反复强化，形成语言习惯，从而形成语言行为。在[[语言教学|语言教学]]中，注重于强调[[语法|语法]]结构，忽视[[语义|语义]]变化；强调[[句型|句型]]练习。此时的心理语言学对语言行为进行了狭义的编码和译码研究，并且对语言与意义、[[人格|人格]]、[[文化|文化]]等问题进行了研究，并且对[[失语症|失语症]]和[[精神分裂症|精神分裂症]]的语言病理进行了临床研究。

1957年，[[乔姆斯基|乔姆斯基]]发表了《[[句法结构|句法结构]]》一书，提出[[生成语法|生成语法]]理论，从而引起了心理语言学对人类利用语言时的心理过程的激烈讨论，研究的问题主要是：
* 语言运用的模式
* 语言结构和语义的储存与记忆
* 人脑对各种句子的处理

1959年，美国语言学家[[乔姆斯基|乔姆斯基]]撰文指出，虽然通过实验能够有效地支持对动物的行为进行刺激、反应和强化的解释，但是人类语言本身非常复杂，很难确定应该刺激什么和使什么发生反应，因此行为主义心理学关于语言的解释含糊不清，无法说明问题。此后心理语言学开始检验乔姆斯基的理论是否适用于心理学和[[实证语言学|实证语言学]]中心理世纪的研究和试验。多数心理语言学家的结论是，语言结构和使用它的人的心理实际结构之间存在着某种对应关系，但不见得是一一对应的。

== 研究範圍 ==

'''心理语言学'''研究人类如何使用和学习语言，并且如何通过语言来表达思想，进行思维。它的研究基礎主要是來自[[心理學|心理學]]、[[認知科學|認知科學]]、[[語言學|語言學]]的理論和研究方法。

[[語言學|語言學]]方面:

* [[語音學|語音學]]及[[音位學|音韻學]]關注語言發音的問題. 在'''心理語言學'''的範疇裏，則研究[[腦部|腦部]]如何理解及處理語言的聲音。

* [[構詞學|構詞學]]是研究字的結構，特別是有相同字根的字 (例如：dog和dogs)，以及產生這些字的規則。

* [[語法學|語法學]]是研究語言的規則，以及[[文法|文法]]。[[喬姆斯基|喬姆斯基]]提出[[轉換-生成語法|轉換-生成語法]]的理論，令'''心理語言學'''的研究視[[語法學|語法學]]為語言中非常重要的研究對象。

* [[語義學|語義學]]則處理字詞和句子的含意。

* [[語用學|語用學]]就關注語言環境對語意理解的影響。

[[心理學|心理學]]方面：

'''心理語言學'''主要集中研究認字和閱讀的過程，去了解人類思維的運作。而[[認知心理學|認知心理學]]中[[運作記憶|運作記憶]] (''Working Memory'') 模型則成為'''心理語言學'''研究人類在處理語言 (聽、講、讀、寫四方面) 的研究基礎方向。

而[[發展心理語言學|發展心理語言學]]則集中研究幼兒及小童如何學習語言，通常使用實驗方法或[[量性研究|量性研究]]去進行研究。（相對於[[讓·皮亞傑|讓·皮亞傑]]以[[自然觀察|自然觀察]]的方法去研究兒童認知發展）
