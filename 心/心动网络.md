'''心动网络股份有限公司'''（简称：'''心动网络'''，{{NEEQ|833897}}）是[[VeryCD|VeryCD]]创始人黄一孟、戴云杰于2011年7月{{NoteTag|心动官网说公司是从2003年创立至今。其实这个说法是算进了之前由隐志网络运营的[[VeryCD|VeryCD]]项目，即中国大陆知名的P2P资源分享网站“电驴大全网”，而其运营公司隐志网络的工商登记注册时间正好是2003年。}}在上海注册成立的一家电子游戏公司。心动网络在中国大陆地区主要从事网络游戏运营和电子游戏发行业务。<ref>{{Cite web|url=https://www.tianyancha.com/company/438738228|title=心动网络股份有限公司 工商注册登记资料|accessdate=2018-05-21|author=|date=|publisher=天眼查}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

2015年11月，心动网络登陆新三板挂牌交易。<ref name=":0" />

== 运营历史 ==
2002年，黄一孟注册心动网络的前身VeryCD（即中国大陆知名的[[ED2k链接|ed2k]]资源分享网站“电驴大全网”）项目的域名。<ref name=":2">{{Cite web|url=http://game.zol.com.cn/358/3585214.html|title=心动2012年收入达10亿:屌丝到高富帅|accessdate=2018-05-25|author=|date=2013-03-08|publisher=中关村在线}}</ref>

2009年，考虑到VeryCD面临的版权风险，黄一孟决定出售VeryCD，并斥资7万[[美元|美元]]去购买了“XD.com”这个域名，这也是当时域名市场上所能购买到最好域名了。为配合这个域名给新公司确定了中文名称“心动”，利用之前出售VeryCD之后，闲置下来的服务器转型做页游。<ref name=":2" />

2010年7月，心动游戏自研的第一款页游《天地英雄》上线运营。<ref name=":2" />

2011年，心动游戏独家代理由厦门光环信息科技有限公司（现“[[飞鱼科技|飞鱼科技]]”）研发的页游《神仙道》。随着中国页游市场规模不断扩大，2011年下半年，心动游戏单月营收最高曾达到过1.5亿人民币。<ref name=":2" />

2012年夏天，心动游戏参加上海[[Chinajoy|Chinajoy]]（简称“CJ”）游戏展。并为了宣传旗下游戏《神仙道》，斥资600万元买下了为期一个月的设置在展览馆周边的[[上海地铁|上海地铁]][[花木路站|花木路站]]和[[龙阳路站|龙阳路站]]的全部广告位。<ref name=":2" />

2015年11月4日，心动网络正式登陆[[全国中小企业股份转让系统|全国中小企业股份转让系统]]（即新三板）挂牌交易，证券代码：833897。<ref name=":0">{{Cite web|url=http://tech.qq.com/a/20151104/002219.htm|title=心动网络今起挂牌新三板 发行792万股|accessdate=2018-05-20|author=|date=2015-11-04|publisher=腾讯科技|deadurl=no|archiveurl=https://web.archive.org/web/20180520144645/http://tech.qq.com/a/20151104/002219.htm|archivedate=2018-05-20}}</ref>

2017年11月9日，心动网络已正式向上海市证监局递交IPO（即[[首次公开募股|首次公开募股]]）及上市辅导备案材料，目前正在接受广发证券辅导。并提示称如未来公司若有向[[中国证券监督管理委员会|中国证券监督管理委员会]]或有权审核机构提交首次公开发行股票并上市的申请材料并获受理，公司将向股转系统申请股票暂停转让。 <ref>{{Cite web|url=http://www.neeq.com.cn/disclosure/2017/2017-11-09/1510213109_795850.pdf|title=心动网络股份有限公司关于上市辅导备案的提示性公告（2017-067）|accessdate=2018-05-21|author=|date=2017-11-09|publisher=全国中小企业股份转让系统|deadurl=no|archiveurl=https://web.archive.org/web/20180521120350/http://www.neeq.com.cn/disclosure/2017/2017-11-09/1510213109_795850.pdf|archivedate=2018-05-21}}</ref><ref>{{Cite web|url=http://www.csrc.gov.cn/pub/shanghai/shfdqyxx/shfdqyjbqk/201711/t20171108_326712.htm|title=心动网络股份有限公司辅导备案基本情况表|accessdate=2018-05-21|author=|date=|work=|publisher=中国证券监督管理委员会|deadurl=no|archiveurl=https://web.archive.org/web/20180521120352/http://www.csrc.gov.cn/pub/shanghai/shfdqyxx/shfdqyjbqk/201711/t20171108_326712.htm|archivedate=2018-05-21}}</ref>

2018年4月23日，心动网络公布2017年年报，根据年报显示心动网络2017年全年实现营业收入10.21亿元[[人民币|人民币]]，净利润1.08亿元人民币，其中移动游戏营收8.63亿元，占总营收84.55%。<ref>{{Cite web|url=http://www.gamelook.com.cn/2018/05/328049|title=心动网络2017年营收10.21亿元 净利润1.08亿元|accessdate=2018-05-21|author=|date=2018-05-02|publisher=gamelook|deadurl=no|archiveurl=https://web.archive.org/web/20180521120346/http://www.gamelook.com.cn/2018/05/328049|archivedate=2018-05-21}}</ref>

== 对外投资 ==
2017年1月5日，心动网络拟以现金出资67万元收购GameSkyGlobalPTE.LTD.（厦门易灵网络科技有限公司新加坡子公司）8.33%股权。<ref>{{Cite web|url=http://news.10jqka.com.cn/20170106/c595935884.shtml|title=心动网络67.00万元收购GameSkyGlobalPTE.LTD.8.33%股权|accessdate=2018-05-21|author=|date=2017-01-06|publisher=同花顺财经|deadurl=no|archiveurl=https://web.archive.org/web/20180521075227/http://news.10jqka.com.cn/20170106/c595935884.shtml|archivedate=2018-05-21}}</ref>

2017年7月28日，心动网络拟与柯骏投资、烁嘉投资签署股权转让协议，拟以现金出资3600万元人民币受让'''柯骏投资'''持有的龙成网络18.00%的股权，同时拟以现金出资1000万元人民币受让'''烁嘉投资'''持有的龙成网络5.00%股权。本次交易完成后，心动网络合计持有龙成网络51.00%的股份，成为心动网络的控股子公司。<ref>{{Cite web|url=http://www.neeq.com.cn/disclosure/2017/2017-07-28/1501230888_168578.pdf|title=心动网络股份有限公司收购资产公告（2017-044）|accessdate=2018-05-20|author=|date=2017-07-28|publisher=全国中小企业股份转让系统|deadurl=no|archiveurl=https://web.archive.org/web/20180520144645/http://www.neeq.com.cn/disclosure/2017/2017-07-28/1501230888_168578.pdf|archivedate=2018-05-20}}</ref><ref>{{Cite news|url=http://www.chinaipo.com/news/40342.html|title=心动网络收购龙成网络23%股份 合计4600万元|author=|date=2017-07-31|work=新三板在线|accessdate=2018-05-21}}</ref>

2018年4月3日，心动网络拟与烁嘉投资签署股权转让协议，拟以现金出资2800万元人民币受让烁嘉投资持有的龙成网络14%的股权。本次交易完成后，心动网络将合计持有龙成网络65%的股份。心动网络此次增持龙成网络的股份，是看好在海外地区发行游戏的前景，进一步拓宽公司在海外地区发行游戏的渠道，以便能更好的整合游戏平台资源，为布局海外市场打下坚实基础，且龙成网络盈利能力良好，能够持续给心动网络带来经济效益。<ref>{{Cite web|url=http://www.neeq.com.cn/disclosure/2018/2018-04-03/1522750324_278909.pdf|title=心动网络股份有限公司收购资产公告（2018-013）|accessdate=2018-05-20|author=|date=2018-04-03|publisher=全国中小企业股份转让系统|deadurl=no|archiveurl=https://web.archive.org/web/20180520144645/http://www.neeq.com.cn/disclosure/2018/2018-04-03/1522750324_278909.pdf|archivedate=2018-05-20}}</ref>

== 出售资产 ==
2018年1月3日，心动网络拟以0元人民币的价格向杨中平转让心动网络持有的上海肥熊网络科技有限公司30%股权。从2016年年报和2017年三季度报看，肥熊网络均为亏损状态，此次心动网络会以0元的白菜价售出，明显是出于提升业绩考虑，清理无法带来收入的不良资产。<ref>{{Cite web|url=http://news.17173.com/content/01072018/210050960.shtml|title=心动网络拟0元转让上海肥熊30%股权曾出资90万元投资|accessdate=2018-05-22|author=|date=2018-01-07|work=|publisher=17173|deadurl=no|archiveurl=https://web.archive.org/web/20180522140518/http://news.17173.com/content/01072018/210050960.shtml|archivedate=2018-05-22}}</ref>

== 融资 ==
2017年2月6日，[[三七互娱|三七互娱]]的全资子公司西藏泰富、[[游族网络|游族网络]]的全资子公司游素投资，都拟以现金出资1.0005亿元认购心动网络725万股股份。增资完成后，西藏泰富和游素投资将各持有心动网络2.38%股权。<ref>{{Cite news|url=http://www.techweb.com.cn/internet/2017-02-06/2482919.shtml|title=三七互娱和游族网络同时宣布投资心动网络 各投1亿元|author=|date=2017-02-06|work=Techweb|accessdate=2018-05-24|language=zh-cn|deadurl=no|archiveurl=https://web.archive.org/web/20170321001815/http://www.techweb.com.cn/internet/2017-02-06/2482919.shtml|archivedate=2017-03-21}}</ref>
