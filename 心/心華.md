'''心華'''是日本[[吉本興業|吉本興業]]子公司[[Gynoid|Gynoid]]推出的[[Vocaloid|Vocaloid]]虛擬歌手。有別於之前发售的中文[[Vocaloid|Vocaloid]]声库[[洛天依|洛天依]]与[[言和|言和]]，心華的聲源来自于[[台湾|台湾]]，而并非中国大陆。

2015年2月10日，Vocaloid3產品由華創文化有限公司於臺灣地區發售。4月24日，由Gynoid於日本發售。<ref>[http://www.chokaigi.jp/2015/booth/xinhua_v-flower.html niconico超會議2015] {{webarchive|url=https://web.archive.org/web/20160304223127/http://www.chokaigi.jp/2015/booth/xinhua_v-flower.html |date=2016-03-04 }}</ref>6月27日，由上海望乘信息科技有限公司代理，於中國大陸地區發售。<ref>[http://www.weibo.com/3247100847/Ci6Ye7iQU Fun成都遊戲動漫節 最新中文V家虚拟偶像大陆首发！]</ref><ref>[http://www.weibo.com/1694556887/Ciwm5uZli 上海望乘信息科技有限公司宣布取得心華於中國大陸地區之代理銷售權]</ref>2017年9月1日，Vocaloid4产品由上海望乘信息科技有限公司于完全汉语地區（包括中国大陆、台湾、香港、澳门，除新加坡）發售。日語版產品在2017年9月22日由Gynoid于日本發售，8月21日開始接受預約，9月8日发布试用版。<ref>[https://www.barks.jp/news/?id=1000145931 台湾生まれのVOCALOIDライブラリ「心華（シンファ）」日本語版＆中国語版がV4で登場]</ref>9月28日，上海望乘开始贩卖日语版声库。

== 人物設定 ==

心華的人物形象由在台日皆享譽盛名的插畫家[[VOFAN|VOFAN]]擔當，聲源則是臺灣女孩[[王文儀|王文儀]]{{註|已知為臺灣配音公會配音班結訓生。<ref>[https://www.facebook.com/taipeirecording/posts/670149309764484 配音工會facebook公告]</ref>}}提供。
命名方面，參考了臺灣富有人情味的特色：「最美的風景是人心」{{註|原句應為「臺灣 最美的風景是人」，出自《新周刊》375期封面標題}}，以及《詩經》中的詩詞「桃之夭夭，灼灼其華」。以桃樹與桃花代表心華的可愛和十六歲高中女生的青春。

'''《VOCALOID4 Library 心華 中文版》'''和'''《日語版》'''搭載了Vocaloid4引擎，添加V4新功能「Growl」功能。在日语版中，通过使用能从高至低音域演唱出自然歌聲的【Natural】，以及能演唱出夯实且强有力歌聲的【Power】，用户可以製作出豐富多彩的歌唱效果。可互相支援交叉合成（XSY）。

与前年发布的[[v4_flower|v4 flower]]一样，除了「单体版」，Gynoid同样推出豪华包装的具有收藏性質的「ボカキュー（VOCALOID4 Editor for Cubase）捆绑版」<ref>{{cite news |url= http://www.gynoid.co.jp/news/view/53/ |title= VOCALOID4ライブラリ『心華(シンファ, Xin hua)』、日本と中国で発売！|date= 2017-08-21|publisher= gynoid|language= ja}}</ref>。同时上海望乘为中文版和日语版推出「下载版」和「实体版」两个商品。
