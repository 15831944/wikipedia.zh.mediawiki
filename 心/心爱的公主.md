《'''心爱的公主'''》<ref>{{cite web |url=http://www.sc.gov.cn/10462/12771/2018/4/8/10448464.shtml |title=“好久没来了，我很想你们” 泰国诗琳通公主在绵阳考察访问 |last= |first= |date=2018-04-08 |website=四川省人民政府 |publisher= |access-date= 2018-07-24|quote=}}</ref>（{{lang-th|นารีรัตนา}}，直译为“宝石般的女子”<ref>{{cite web |url=https://www.thairath.co.th/content/478992 |title=นารีรัตนา |last= |first= |date=2015-02-06 |website=thairath |publisher= |access-date=2018-07-24 |quote=}}</ref>）是一首泰国人民赞颂[[诗琳通公主|诗琳通公主]]的歌曲，于[[2015年诗琳通公主60寿辰|2015年诗琳通公主60寿辰]]时公布<ref>{{cite web |url=https://www.posttoday.com/ent/news/343584 |title=ชวนร้องเพลง "นารีรัตนา" ถวายสมเด็จพระเทพฯ |last= |first= |date=2015-01-26 |website=posttoday |publisher= |access-date=2018-07-24 |quote=}}</ref>。由艺术家和青年志愿者组织Bua Lam发行，演唱者为塔娜蓬·卫巴芸（{{lang-th|ธนพร แวกประยูร}}）。歌词中描述了诗琳通公主的亲民事迹<ref>{{cite web |url=https://mgronline.com/celebonline/detail/9580000004409 |title=“นารีรัตนา” เพลงเฉลิมพระเกียรติ 60 พรรษา “เจ้าฟ้ามหาจักรีสิรินธร” |last= |first= |date=2015-01-13 |website=mgronline |publisher= |access-date=2018-07-24 |quote=}}</ref><ref>{{cite web |url=https://www.dailynews.co.th/education/296846 |title=แจกซีดีเพลง“นารีรัตนา”60พรรษาสมเด็จพระเทพฯ |last= |first= |date=2015-01-27 |website=每日新闻 |publisher= |access-date=2018-07-24 |quote=}}</ref>。

[[MV|MV]]中使用的图片是泰国皇家摄影学会<!-- Royal Photographic Society of Thailand (RPST) -->会长尼迪功·蓋威遷（{{lang-th|นิติกร กรัยวิเชียร}}）的摄影作品《[[诗琳通公主肖像|诗琳通公主肖像]]》。

== 创作背景 ==
作曲家素提蓬·宋班今达（{{lang|th|สุทธิพงษ์ สมบัติจินดา}}）说，《心爱的公主》灵感来源于诗琳通公主的“父亲足迹”，还希望在4月2日前唱这首歌来鼓励人们。歌曲中使用的乐器为[[木琴|木琴]]<ref>{{cite web |url=https://www.thairath.co.th/content/477433 |title=ชวนร้อง "นารีรัตนา" สรรเสริญสมเด็จพระเทพฯ ครบ 5 รอบ |last= |first= |date=2015-01-27 |website=thairath |publisher= |access-date=2018-10-24 |quote=}}</ref>。

== 翻唱 ==
绵阳广电少儿艺术团和先锋路小学合唱团排练时需要演唱一首泰语歌曲，其中包括《心爱的公主》<ref>{{cite web |url=https://3g.163.com/dy/article/DEOIJDGM0517PFUK.html |title=声声不息 一份送给泰国诗琳通公主的礼物 |last= |first= |date=2018-04-07 |website=网易 |publisher=今日放猛料 |access-date=2018-07-24 |quote=}}</ref>。2018年4月7日，[[绵阳市先锋路诗琳通公主小学校|绵阳市先锋路诗琳通公主小学校]]与{{link-th|吉拉达学校|โรงเรียนจิตรลดา}}合作表演时演唱了这首歌曲<ref>{{cite web |url=http://www.chinanews.com/gn/2018/04-07/8485140.shtml |title=泰国公主诗琳通访川：情系震区 关注教育 |last= |first= |date=2018-04-07 |website=中国新闻网 |publisher= |access-date=2018-07-24 |quote=}}</ref>。
