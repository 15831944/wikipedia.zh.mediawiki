== 剧情 ==
《心术》讲述了[[二十一世纪|二十一世纪]][[中国大陆|中国大陆]][[都市|都市]]人的感情生活，主要聚焦点是民生中的[[医患关系|医患关系]]问题，其次是：[[第三者|第三者]]、[[房奴|房奴]]以及[[性交易|性交易]]话题。<ref name="xlw " /><ref name="fhw " /><ref>{{cite web |url=http://ent.sina.com.cn/v/m/2012-04-11/12133602376.shtml |title= 《心术》主创出席看片会 剧情聚焦医患关系|publisher= 新浪网|accessdate=2012-04-13 |work= |language =zh-cn}}</ref>

== 花絮 ==
作家[[六六|六六]]的原著只有10多万字，但是她创作该剧写了剧本70万字。<ref name="xlw " >{{cite web |url= http://ent.sina.com.cn/v/m/2012-04-13/10003604152.shtml|title=《心术》看片会六六成焦点 斗小三经历有望拍剧 |accessdate= 2012-04-13|work=|publisher= 新浪网|language = zh-cn}}</ref>

该剧中的台词劲爆、赤裸和辛辣，讽刺了当代社会中[[小三|小三]]、[[性交易|性交易]]等敏感话题，如：“你不是喜欢搞文艺的女青年，你是喜欢搞文艺女青年吧？！”，编剧六六自谓：“《心术》采取平民视角，为医患双方说话，吸纳了大众所喜爱的风趣的表达方法，幽默的台词是这部电视剧在争取收视率方面的武器。”。<ref name=" app" >{{cite web |url= http://news.xhby.net/system/2012/04/11/013118785.shtml|title=《心术》“嘎嘣脆”台词获好评 |accessdate= 2011-04-11|work=|publisher=新华报业网 |language = zh-cn}}</ref><ref name="fhw " />

该剧拍摄期间，编剧[[六六|六六]]在微博爆出“小三”事件，而[[海清|海清]]的父亲则正好在拍摄期间生病；同时[[海清|海清]]与[[吴秀波|吴秀波]]也传出恋爱的绯闻。<ref>{{cite web |url=http://ent.qq.com/a/20120413/000020.htm |title=《心术》将开播 海清称与吴秀波传绯闻有利创作 |accessdate= 2012-04-13|work=|publisher= 腾讯网|language = zh-cn}}</ref><ref>{{cite web |url= http://ent.163.com/12/0413/04/7UUPBAQF00032DGD.html|title= 《心术》将播 六六：海清耍心机成为女主角|accessdate= 2012-04-13|work=|publisher=网易 |language = zh-cn}}</ref>

该剧于2012年5月3日同步在[[天津|天津]]、[[东方|东方]]、[[浙江|浙江]]、[[安徽|安徽]]四大卫视首播。<ref>{{cite web|url=http://et.21cn.com/gundong/etscroll/2012/03/20/11210601.shtml|title=《心术》四大卫视首播 孙艺菲缠吴秀波智斗海清|publisher=21CN|accessdate=2012-03-20|work=|language=zh-cn}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 评价 ==
《心术》播出后，被观众和网友吐槽，戏称不如改名叫做“《医院那点事》”。<ref>{{cite web |url=http://ent.ifeng.com/tv/news/mainland/detail_2012_05/06/14345239_0.shtml |title=《心术》引网友吐槽：不如改名叫《医院那点事》 |accessdate=2012年5月6日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>

《心术》剧中出现不少错误，如医生在[[手术室|手术室]]一面做手术一面聊天；护士在手术室还带着[[耳环|耳钉]]和[[项链|项链]]，而真正意义上规范的医院都是禁止如此做的。<ref>{{cite web |url= http://ent.ifeng.com/tv/news/mainland/detail_2012_05/08/14381280_0.shtml|title=《心术》被批不专业：医生做手术聊天 护士戴首饰耳钉 |accessdate=2012年5月8日 |publisher=凤凰网|work=梁燕芬 |language =zh-cn}}</ref>
