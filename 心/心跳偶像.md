'''《心跳偶像》'''（日语：{{lang|ja|ときめきアイドル}}，简称：{{lang|ja|ときドル}}）<ref>{{cite web|url=https://appget.com/appli/view/71461/|title=ときめきアイドル（ときドル） KONAMIの本気、凄かった。VR対応の3D学園アイドルが１５人同時にライブ！異例の完全オートプレイ可！|date=2018-03-21|publisher=Appget|language=ja}}</ref>，是由日本[[科樂美數位娛樂|科樂美數位娛樂]]发行的一款偶像育成类音乐[[手机游戏|手机游戏]]，改編自《[[心跳回忆系列|心跳回忆系列]]》，2018年3月20日正式上线。<ref>{{cite web|url=http://nlab.itmedia.co.jp/nl/articles/1803/20/news112.html|title=コナミ「ときメモ」シリーズの「ときめきアイドル」がサービス開始　プロデューサーになってアイドルを育成|date=2018-03-20|publisher=ITmedia|language=ja}}</ref>

== 简介 ==
2017年9月15日，本作被作为《[[心跳回忆|心跳回忆]]》的续作而公开，并于同一天开启了事前登录。同年9月24日，在[[东京电玩展|东京电玩展]]的KONAMI展台上免费派发包含7首试听歌曲的CD。<ref>{{cite web|url=https://www.hk01.com/%E7%A7%91%E6%8A%80%E7%8E%A9%E7%89%A9/119550/%E5%BF%83%E8%B7%B3%E5%81%B6%E5%83%8F-%E5%85%AC%E5%B8%83-konami%E5%8F%83%E6%88%B0idol%E6%89%8B%E9%81%8A%E6%88%B0%E5%A0%B4-tgs%E5%A4%A7%E6%B4%BEcd|title=《心跳偶像》公布　KONAMI參戰IDOL手遊戰場 TGS大派CD|date=2017-09-16|publisher=香港01|language=zh-hk}}</ref>

在游戏中，玩家将担任来自Princess Republic音乐学院偶像科的15名偶像候补生所住宿舍的管理员与制作人，与候补生们共同生活，将她们培养成为学园偶像。<ref>{{cite web|url=https://app.famitsu.com/20171025_1165888/|title=プレイヤーは寮長兼プロデューサー！KONAMIのスマホ音ゲー『ときめきアイドル』の最新ムービーが公開|date=2018-03-13|publisher=Fami通APP|language=ja}}</ref>

此外，游戏还支持[[VR|VR]]功能，玩家可在部分游戏模式里与角色通过VR进行互动。<ref>{{cite web|url=https://www.g-cores.com/articles/93011|title=《心跳回忆》手游新作《心跳偶像》预告公布|date=2017-09-15|publisher=GAMECORES|language=zh-cn}}</ref>

2018年11月15日，游戏运营发表公告称本游戏将于2019年1月15日14:00正式关服，之后将会发布本游戏的离线版本。<ref>{{cite web|url=https://www.famitsu.com/news/201811/15167651.html|title=KONAMI『ときめきアイドル』の運営サービスが2019年1月15日をもって終了へ。以降も遊べる『オフライン版ver.2.0』を配信|date=2018-11-15|publisher=Fami通|language=ja}}</ref>

== 用语解释 ==
; Princess Republic音乐学院
位于光辉市以南的一所初中高中一贯制的女子学院，有着悠久的历史和自由的校风，在有学习音乐志向的学生中有着很高的人气，校园宽广，学生众多。

学校以颜色来区分年级，一年级为黄色，二年级为红色，三年级为绿色，这些颜色体现在学生校服的领结，练习服，运动衫以及鞋带上。

偶像科刚创立不久，希少和春子是创建前的“零期生”，其他人则是“一期生”。<ref>《Comptiq》3月号--KADOKAWA</ref>

理事长的话是“{{lang|ja|日々是、生活の全てをレッスンと思え}}”
; 学生宿舍
偶像科的学生们所居住的宿舍，配备有练习室和谈话室等设施，除宿舍管理员之外其余人皆为女性。

有一只脖子上围着红围巾的被称为“猫前辈”的猫，备受整个宿舍的宠爱，在VR模式下可以体验到猫的视角。
; Melodious Live
全国的偶像组合进行比拼的舞台，也是这15名候补生所向往的地方
