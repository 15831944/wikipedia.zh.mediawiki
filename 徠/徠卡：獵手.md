《'''徠卡：獵手'''》（{{Lang-en|Leica - The Hunt}}）是指在2019年4月，由國際廣告公司「[[上奇廣告|上奇廣告]]」的[[巴西|巴西]]分公司「F/Nazca Saatchi & Saatchi」製作和公佈的一部[[微電影|微電影]]。該影片的內容主要涉及在1989年[[中華人民共和國|中華人民共和國]]首都[[北京市|北京市]][[天安門廣場|天安門廣場]]發生[[六四事件|六四事件]]後，一名[[攝影記者|攝影記者]]以[[照相機|照相機]]記錄「[[王維林|王維林]]」阻擋坦克的場景。該影片透過[[網際網路|網際網路]]發表，但後來遭到移除。

這部[[短片|短片]]曾一度被各界視為是[[德國|德國]]照相機製造商[[徠卡相機|徠卡相機]]的宣傳廣告，且主要是用來紀念六四事件[[六四事件紀念活動|屆滿30周年]]，不過徠卡相機官方立即撇清關係。[[中華人民共和國政府|中華人民共和國政府]]在[[中國網際網路|網際網路]]上[[中華人民共和國網路審查|封鎖]]相關消息，並一度將「徠卡」與「Leica」列為[[關鍵詞過濾|過濾關鍵詞]]。這部爭議影片受到大批[[中國大陸|中國大陸]]網友強烈反彈，在徠卡相機的[[新浪微博|新浪微博]]帳戶上出現大量抵制留言。相對地，[[臺灣|臺灣]]和[[香港|香港]]等地网友多對這部影片表示支持。

這部影片是由巴西雙人組合「小子布羅」（Kid Burro）執導，該組合是由兩位導演塞薩爾·納瑞（Cesar Nery）和安德烈·塞托（Andre Saito）組成<ref name = "英國廣播公司新聞網 02"/><ref name = "The Stable 01"/><ref name = "云孟泽 01">{{Cite web |url = http://news.dwnews.com/china/news/2019-04-19/60129846.html |title = 徕卡非官方短片惊现六四 挑动华为与网民神经 |author = 云孟泽 |date = 2019-04-19 |publisher = [[多維新聞|多維新聞]] |language = Zh-hans |accessdate = 2019-04-20 |quote =  }}</ref>。兩位導演認為許多故事都是透過一些攝影師勇於報導，才會被世人們所認識<ref name = "The Stable 01"/>。《徠卡：獵手》主要便是講述一個虛構故事，呈現攝影記者試圖在各個戰爭與衝突地區的環境中，堅持使用照相機拍攝影像，並留下歷史紀錄的精神<ref name="劉婉雯 01" /><ref name = "立場新聞 01"/>；與此同時，這群攝影師仍會面臨著遭到逮捕、毆打甚至死亡等風險的壓力<ref name = "David Griner 01"/>。

兩位導演認為很少有電影能讓觀眾同時沉浸在概念與劇情中，並花費1年多的時間研究幾位攝影師的故事<ref name = "The Stable 01"/>。在短片中的主要場景為六四事件<ref name = "立場新聞 01"/>，並以該背景設計故事主線<ref name = "英國廣播公司新聞網 02"/><ref name = "陳家倫 01"/><ref name = "廖威誌 01">{{Cite web |url = https://www.ftvnews.com.tw/news/detail/2019419W0009 |title = 徠卡「The Hunt」紀念64坦克人遭中國封殺！網友讚：我買了 |author = 廖威誌 |date = 2019-04-19 |publisher = [[民視新聞台|民視新聞台]] |language = Zh-hant |accessdate = 2019-04-20 |quote =  }}</ref>。其中，[[美聯社|美聯社]]攝影師{{Link-en|傑夫·懷登|Jeff Widener}}在六四事件期間，冒險進入飯店拍照拍攝《坦克人》畫面的經歷<ref name = "廖威誌 01"/>，便與這部短片的故事有許多吻合處<ref name = "周虹汶 01"/>，不過導演亦對該題材進行更加戲劇化的改編工作<ref name = "Miranda Aldersley 01"/><ref name = "Shannon Liao 01">{{Cite web |url = https://www.theverge.com/2019/4/19/18508009/leica-china-censored-ad-tiananmen-square-protests-weibo |title = Leica censored in China for ad that dramatizes the Tiananmen Square protests |author = Shannon Liao |date = 2019-04-19 |publisher = [[The_Verge|The Verge]] |language = en |accessdate = 2019-04-20 |quote =  }}</ref>。不過當時懷登使用的照相機，是[[日本|日本]][[尼康|尼康]]公司推出照相機<ref name = "周虹汶 01"/>。影片由外籍演員飾演戰地記者<ref name = "劉婉雯 01"/>，並由[[烏克蘭|烏克蘭]]製片公司「Radioaktive Film」負責上傳影片<ref name="The Stable 01" /><ref name = "云孟泽 01"/>。

為了具體指涉六四事件，除了在影片當中加入「1989年，北京」字幕外，亦使用有關「王維林」的畫面陳述<ref name = "云孟泽 01"/><ref>{{Cite web |url = http://news.dwnews.com/china/news/2019-04-19/60129826.html |title = 广告片现六四事件画面 德国徕卡紧急回应 |author = 褚文 |date = 2019-04-19 |publisher = [[多維新聞|多維新聞]] |language = Zh-hans |accessdate = 2019-04-20 |quote =  }}</ref>。同時在該短片中還以鏡頭交替的故事手法，穿插幾個非洲軍閥、中東穆斯林、移民婦女、武裝士兵等，演繹強權者肆意逼迫無辜民眾的概括性畫面<ref name = "云孟泽 01"/><ref name = "蘋果日報 02"/>。亦有部分媒體認為該片段是在隱喻[[中華人民共和國政府|中華人民共和國政府]]如同土匪般，為了掩蓋真相而[[中華人民共和國新聞自由|阻擋]]新聞媒體的採訪、報導<ref name = "蘋果日報 01"/>，以及控訴該國把境內的[[中國伊斯蘭教|大量穆斯林]]當成異教徒[[中國伊斯蘭教史|迫害]]<ref name = "自由時報 01"/><ref name = "自由時報 02"/>。

=== 內容發布 ===
[[File:HongKongInternationalAirportLeicaStore.jpg|thumb]]隨後撇清與《徠卡：獵手》的關係，並確保不會透過社群媒體宣傳該影片。]]

2019年4月16日，F/Nazca Saatchi & Saatchi在其經認證的[[Twitter|Twitter]]帳號上，發表有關《徠卡：獵手》的[[葡萄牙語|葡萄牙語]]文章<ref name = "Miranda Aldersley 01"/>。同日，廣告網站「世界廣告」亦出現《徠卡：獵手》影片<ref name = "李俊杰 01"/>。4月17日，Radioaktive Film將影片上傳至[[YouTube|YouTube]]等[[社群媒體|社群媒體]]網站發布<ref name = "William Yang 01"/><ref name = "Yuan Yang 01"/><ref name = "云孟泽 01"/>。4月18日，該影片持續出現在多個網站上<ref name = "彭博社 01"/>。但隨著原始影片遭到移除<ref name = "自由時報 03"/>，「世界廣告」後來只留下原始的文字描述<ref name = "蘋果日報 01"/>，在[[影片分享網站|影片分享網站]][[Vimeo|Vimeo]]上轉載的微電影也被大量刪除<ref name = "李俊杰 01"/>。

由於《徠卡：獵手》影片的片尾展示徠卡相機的官方標誌，最初各界紛紛將該短片視為徠卡相機自家出品的[[市場行銷|市場行銷]]內容片段<ref name = "William Yang 01"/><ref name="英國廣播公司新聞網 02" /><ref name = "蘋果日報 02"/>，並稱讚其作風大膽<ref name = "自由時報 02"/>。但實際上，徠卡相機不曾在其社群媒體頻道上發佈或分享過這部影片<ref name="Alex Linder 01" /><ref name = "彭博社 01"/>。也因為這部影片並非透過徠卡相機發布，其來源後來引起質疑<ref name = "陳家倫 01"/>。而儘管徠卡相機在中國的照相機市場頗為成功<ref name = "雨涵 01">{{Cite web |url = https://www.dw.com/zh/德语媒体天衣无缝的审查/a-48413722 |title = 德语媒体：天衣无缝的审查 |author = 雨涵 |date = 2019-04-20 |publisher = [[德國之聲|德國之聲]] |language = Zh-hans |accessdate = 2019-04-20 |quote =  }}</ref>，但是當該影片透過[[串流媒體|串流媒體]]傳播給中國國內的觀眾後，事件迅速發酵並引起熱烈討論<ref name="蘋果日報 01" /><ref name = "云孟泽 01"/>，徠卡相機開始面臨到強烈的反對與審查<ref name = "David Griner 01"/>。

在這之後，與中國電訊設備製造商[[華為|華為]]合作製造[[智慧型手機|智慧型手機]]鏡頭零件的徠卡相機<ref name = "英國廣播公司新聞網 02"/>，便試圖與《徠卡：獵手》影片中顯示的內容劃清界線<ref name="David Griner 01" /><ref name = "Owen Churchill 01"/><ref name = "英國廣播公司新聞網 02"/>。4月18日，徠卡相機發言人埃米莉·安德森（Emily Anderson）透過[[電子郵件|電子郵件]]向[[香港|香港]]媒體《[[南華早報|南華早報]]》撇清關係<ref name = "William Yang 01"/><ref name = "李俊杰 01"/>，表示實際上這是未獲官方授權的市場行銷計劃<ref name = "Alex Linder 01"/>，並非由該公司委託拍攝製作的宣傳影片<ref name="David Griner 01" /><ref name = "Owen Churchill 01"/><ref name = "陳家倫 01"/><ref name = "蘋果日報 01"/>。她表示徠卡相機並未授權該影片公開播放，對於其可能造成的誤解或錯誤結論深深感到遺憾<ref name = "英國廣播公司新聞網 02"/><ref name = "眾新聞 01"/><ref name = "自由時報 02"/>，且已經確保不會透過官方社群媒體宣傳該影片<ref name = "William Yang 01"/>。

== 造成影響 ==
=== 遭到審查 ===
[[File:21st_anniversary_candlelight_vigil_4.jpg|thumb]]將[[六四事件|六四事件]]視為政治敏感議題，並進行相關的[[中華人民共和國言論審查|言論審查]]。]]

過去中國政府長期將六四事件視為高度敏感的[[中華人民共和國言論審查|政治議題]]<ref name="周虹汶 01" /><ref name = "云孟泽 01"/><ref name = "Neil Fullick 01">{{Cite web |url = https://www.reuters.com/article/us-leica-china-tiananmen/tank-man-video-for-leica-sparks-outcry-in-china-ahead-of-tiananmen-anniversary-idUSKCN1RV0JG |title = 'Tank man' video for Leica sparks outcry in China ahead of Tiananmen anniversary |author = Neil Fullick |date = 2019-04-19 |publisher = [[路透社|路透社]] |language = en |accessdate = 2019-04-20 |quote =  }}</ref>，針對社群媒體上的相關討論進行[[中華人民共和國網路審查|審查工作]]<ref name = "David Griner 01"/><ref name = "Miranda Aldersley 01"/>，並讓有關內容在[[中國網際網路|網際網路]]的討論中消失<ref name = "雨涵 01"/>。由於2019年是六四事件[[六四事件紀念活動|屆滿30周年]]<ref name = "云孟泽 01"/>，在世界各地準備展開紀念活動之際，中國政府將此視為政治敏感時期，事先加強對國內網際網路的審查<ref name = "彭博社 01"/>。由於《徠卡：獵手》試圖重現1989年六四事件的場景，最初便被指稱是為了悼念六四事件30周年，才特別拍攝與傳播這部微電影<ref name = "英國廣播公司新聞網 02"/><ref name = "蘋果日報 01"/><ref name = "蘋果日報 02">{{Cite web |url = https://hk.news.appledaily.com/china/realtime/article/20190419/59506045 |title = 【六四30】Leica微電影現「坦克人」悼屠殺 中港台網民感動：歷史不會被遺忘 |author =  |date = 2019-04-19 |publisher = 《[[蘋果日報_(香港)|蘋果日報]]》 |language = Zh-hant |accessdate = 2019-04-20 |quote =  }}</ref>。而有鑒於中國的強硬態度，亦有評論認為徠卡相機是在進行「自殺式行銷」<ref name = "褚文 01"/>。

隨著《徠卡：獵手》的內容疑似涉及六四事件而受到關注<ref name = "自由時報 02"/>，該影片遭到中國國家審查機構與[[新浪微博|新浪微博]]等社群媒體的全面審查與封殺<ref name = "李俊杰 01"/>，影片在中國境內的普通網友中幾乎未獲得散播<ref name = "Owen Churchill 01"/><ref name = "雨涵 01"/>。與此同時，中國審查機構亦對徠卡相機展開施壓，在新浪微博上進行審查工作<ref name = "Shannon Liao 01"/>。在影片發布之後，儘管新浪微博等主要社群媒體網站先是出現「徠卡辱華」的標籤，隨後大部分與徠卡相機相關的「違規」文章便遭到審查刪除<ref name="周虹汶 01" /><ref name = "Miranda Aldersley 01"/>。

而自從4月18日晚間開始，中國政府與新浪微博針對與「徠卡」或「Leica」這個品牌相關的漢語及[[英語|英語]]詞彙，進一步做出大動作的封殺反制，當中以「違反相關法律法規或社群公約」的審查理由，將相關內容列為禁止在社群平臺上使用的[[關鍵詞過濾|過濾關鍵詞]]<ref name = "彭博社 01"/><ref name = "Yuan Yang 01"/><ref name = "Owen Churchill 01"/><ref name = "Miranda Aldersley 01"/>，而違反規定的用戶無法成功發送文章<ref name = "William Yang 01"/><ref name="自由時報 02" /><ref name = "廖威誌 01"/>。到了隔日，相關禁令宣告解除<ref name = "自由時報 03"/>。人權運動者[[周鋒鎖|周鋒鎖]]則表示中國政府不太可能公開提到該影片，以避免引起人們對這個主題的關注<ref name="Owen Churchill 01" />。

=== 其他反應 ===
[[File:Salon_de_la_photo_de_Paris_2015_(10).jpg|thumb]]的合作夥伴，後者的立場亦受到關注。]]

在短片發布後，具有[[中國民族主義|中國民族主義]]的網際網路用戶立即在國內社群媒體上<ref name="周虹汶 01" /><ref name = "Shannon Liao 01"/>，針對徠卡相機提出壓倒性的負面評價與反彈<ref name = "彭博社 01"/><ref name = "英國廣播公司新聞網 01"/><ref name = "David Griner 01"/><ref>{{Cite web |url = https://gizmodo.com/china-bans-the-word-leica-on-social-media-1834163199 |title = China Bans the Word 'Leica' on Social Media |author = Matt Novak |date = 2019-04-19 |publisher = [[Gizmodo|Gizmodo]] |language = en |accessdate = 2019-04-20 |quote =  }}</ref>，亦有[[網路評論員|網路評論員]][[洗版|洗版]]批評<ref name = "陳家倫 01"/>。其中大批新浪微博的用戶陸續在徠卡相機的官方新浪微博帳號「徠卡相機中國」上發表持正反立場的留言評論<ref name = "Owen Churchill 01"/><ref name = "陳家倫 01"/>。該新浪微博帳號獲得數百條批評影片的評論與抱怨<ref name = "William Yang 01"/><ref name="David Griner 01" /><ref name = "Neil Fullick 01"/>，亦有[[中國大陸|中國大陸]]網友表示感動<ref name = "蘋果日報 02"/>。由於徠卡相機是華為的手機鏡頭合作夥伴，後者在這次風波的立場亦受到關注<ref name = "雨涵 01"/>，並被網友要求取消合作<ref name = "蘋果日報 02"/>。

許多不滿的中國大陸網友譴責徠卡相機這種做法會讓「無辜」的華為受到衝擊與傷害<ref name = "William Yang 01"/><ref name = "Owen Churchill 01"/>，且不配與華為進行合作，要求徠卡相機說明並道歉<ref name="立場新聞 01" /><ref name = "云孟泽 01"/>。同時還有中國大陸的網友批評該公司的「愚蠢」舉動，指稱該影片導致徠卡相機陷入公關危機<ref name = "Owen Churchill 01"/>，或是呼籲抵制徠卡相機這個品牌<ref name = "英國廣播公司新聞網 01"/><ref name = "蘋果日報 01"/>。隨後徠卡相機禁止在其最新的新浪微博文章上評論<ref name = "Alex Linder 01"/>。這部影片的推出亦在[[臺灣|臺灣]]、香港等地引起廣大的迴響<ref name = "William Yang 01"/>，並獲得[[批踢踢|批踢踢]]網友的支持<ref name = "廖威誌 01"/>。部分反對中國政府者對於該影片亦表示讚揚<ref name = "David Griner 01"/>，周鋒鎖便認為該影片真實表現六四事件發生時的景象<ref name = "William Yang 01"/>。

美國商業雜誌《{{Link-en|Fast Company|Fast Company}}》認為這部影片傳達攝影記者的壓力、焦點和純粹的動力<ref name = "Jeff Beer 01"/>，而《{{Link-en|廣告週刊|Adweek}}》則認為這是讚揚攝影師英雄主義，尤其是其勇敢地在國際紛亂地區拍攝受害者的困境<ref name = "David Griner 01"/>。這次「王維林」畫面引起的風波亦成為[[德語|德語]]媒體關注的話題，《[[法蘭克福匯報|法蘭克福匯報]]》指出最令人驚嘆的並非這部微電影，而是中國「近乎完美的審查」<ref name = "雨涵 01"/>。
