'''托利党'''（'''Tories'''），是17世纪至19世纪期间的[[大不列颠及爱尔兰联合王国|英国]]（及其前身[[大不列颠王国|大不列颠王国]]、[[英格兰王国|英格兰王国]]）政党。'''托利'''一詞源於中世紀愛爾蘭語的「亡命之徒」（tóraidhe），是政敵對托利党人的[[蔑稱|蔑稱]]，後來沿用成習。

1678年，[[辉格党_(英国)|辉格党]]提出[[王位排除法令|王位排除法令]]，意图剥夺[[詹姆斯二世_(英格兰)|詹姆斯二世]]的继位权。反对法令的保守人士在此时得到了“托利党”这个称呼。此后，该党失去了组织性，直到18世纪60年代为止。二十多年后，新托利党崛起，建立了先后以[[小威廉·皮特|小威廉·皮特]]和[[羅伯特·詹金遜，第二代利物浦伯爵|羅伯特·詹金遜，第二代利物浦伯爵]]为首的政府。<ref name="consoff">{{Cite web

利物浦伯爵的继任人是[[亚瑟·韦尔斯利，第一代威灵顿公爵|亚瑟·韦尔斯利，第一代威灵顿公爵]]。威灵顿公爵受[[丹尼尔·奥康奈尔|丹尼尔·奥康奈尔]]推动，在任内进行了[[天主教解放|天主教解放]]。辉格党在重新执政后控制[[英国国会|国会]]通过了[[1832年改革法令|1832年改革法令]]废除了众多由托利党控制的[[腐败选区|腐败选区]]。因此，托利党在接下来的大选中只取得了180个席位。党魁[[罗伯特·皮尔|罗伯特·皮尔]]爵士发表[[塔姆沃思宣言|塔姆沃思宣言]]后，托利党开始向现代[[保守党_(英国)|保守党]]转型。然而，托利党因为皮尔废除[[谷物法|谷物法]]产生了分裂。<ref name = "consoff"/>[[愛德華·史密斯-斯坦利，第十四代德比伯爵|愛德華·史密斯-斯坦利，第十四代德比伯爵]]、[[本傑明·迪斯雷利|本傑明·迪斯雷利]]领导的一派最终延续下来称为现代保守党。当今保守党有时仍被称为托利党。<ref name="consoff"/>
