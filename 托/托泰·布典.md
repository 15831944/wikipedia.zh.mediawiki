'''托泰·布典'''（'''Totai Buten'''，{{Bd|1910年||1993年||catIdx=}}<ref name=":0">{{Cite news|url=|title=鹿野忠雄的嚮導友人托泰布典的願望|author=[[劉克襄|劉克襄]]|date=2010-04-22|work=[[中國時報|中國時報]]|accessdate=}}</ref>），[[漢名|漢名]]'''陳抵帶'''，[[別名|別名]]'''木魚'''<ref name=":1">{{Cite web|url=https://www.ylib.com/author/yang/y_article1.htm|title=縱橫山林的學術探險家──楊南郡|accessdate=2017-11-03|work=www.ylib.com}}</ref>，是一位[[阿美族|阿美族]]與[[噶瑪蘭族|噶瑪蘭族]]人士。他是[[壽豐鄉|壽豐鄉]]首位[[民選|民選]][[鄉長|鄉長]]，也是[[鹿野忠雄|鹿野忠雄]]等人的嚮導。

== 生平 ==
托泰·布典的[[祖父|祖父]]出身阿美族荳蘭社（社址位在後[[吉安鄉|吉安鄉]]宜昌村及南昌村內），並[[入贅|入贅]]於噶瑪蘭族[[加禮宛|加禮宛]]社；後來，他的祖父在他[[父親|父親]]出生不久後即在某次巡視[[耕地|耕地]]時遭附近泰雅族[[出草|出草]][[馘首|馘首]]。<ref name=":0" /><ref name=":1" />

為逃避[[徵兵|徵兵]]，父親在他四歲時於一深夜將他送至[[祖母|祖母]]家[[寄養|寄養]]，並暫時[[失蹤者|失去蹤影]]。在撫養數年後，祖母漸感自身年老，遂決定將他送至花蓮東大寺作[[沙彌|沙彌]]。不久，東大寺的[[住持|住持]]將他送至位在[[京都|京都]]的{{Link-ja|臨濟宗大學花園學院高等部|花園大学}}就學；就學期間，托泰·布典除每日研讀[[佛教|佛教]][[教義|教義]]外，也修習[[英語|英語]]和[[日文|日文]]，更在課餘時間與當地[[青年|青年]]一同追隨[[奢侈品|奢侈]]的[[時尚潮流|時尚潮流]]。兩年後，托泰·布典決定輟學返鄉。<ref name=":0" /><ref name=":1" />

返回[[臺灣|臺灣]]不久，托泰·布典再度與失散多年的父親相遇；當時，父親已與一位[[富有|富有]]的[[寡婦|寡婦]]結婚，並因此開始供給他優厚的物質生活。此後，托泰·布典鎮日遊手好閒，不是與[[堂亲|堂兄]]出沒在當地[[酒吧|酒吧]]裡，就是在[[臺東街|臺東]]市區的街頭上晃蕩。<ref name=":0" /><ref name=":1" />

[[昭和|昭和]]8年（1933年）[[夏|夏]]，[[鹿野忠雄|鹿野忠雄]]在前往[[紅頭嶼|紅頭嶼]]途上與托泰·布典相遇於臺東渡船場<ref name=":2">{{Cite news|url=http://www.tipp.org.tw/news_article.asp?F_ID=18699&PageSize=15&Page=1839&startTime=&endTime=&FT_No=&NSubject_No=&SelectSubject=&Subject_No=&SubSubject_No=&TA_No=&Orderby=&KeyWords=&Order=&IsSelect=|title=托泰布典的願望──鹿野忠雄的嚮導友人|accessdate=2017-11-03}}</ref>；當時，托泰·布典因有感鹿野氏忠厚誠懇，向其表示願意擔任其調查活動的不支薪助理。<ref name=":1" />隨後，他們在[[都蘭村|都巒村]][[僱用|僱用]]三個阿美族人，並一同攀爬陡峭山徑登上[[都蘭山|都巒山]]，在山中採集鳥類[[標本|標本]]。接著，他們又到piyanan社（後為宜蘭縣[[大同鄉_(台灣)|大同鄉]]南山村）與shikikun社（後為宜蘭縣大同鄉四季村）僱用十名[[泰雅族|泰雅族]]青年，再自四季[[駐在所|駐在所]]出發攀登[[南湖大山|南湖大山]]。<ref name=":0" /><ref name=":1" />

1941年後，托泰·布典在鹿野忠雄推薦下進入[[臺北帝國大學|臺北帝國大學]]任職。<ref name=":1" />

[[戰後台灣|戰後]]，托泰·布典經常向訪客問起鹿野忠雄的下落，但總是不得其解。<ref name=":0" />

[[二二八事件|二二八事件]]爆發時，托泰·布典以陳抵帶之名在[[臺灣省行政長官公署|臺灣省行政長官公署]]民政處等單位擔任辦事員。<ref>{{Cite web|url=http://ds3.th.gov.tw/ds3/app003/list3.php?KW1=%E9%99%B3%E6%8A%B5%E5%B8%B6&RecKW=1&ID1=&PG1=&TY1=1|title=臺灣省行政長官公署檔案|accessdate=2017-11-03|work=ds3.th.gov.tw|language=zh-TW}}</ref><ref name=":1" />

1950年，托泰·布典當選[[壽豐鄉|壽豐鄉]]首任[[民選|民選]][[鄉長|鄉長]]。<ref>{{Cite web|url=http://memory.ncl.edu.tw/tm_cgi/hypage.cgi?HYPAGE=toolbox_event_detail.hpg&subject_name=%E8%87%BA%E7%81%A3%E5%A4%A7%E4%BA%8B%E8%A8%98%E8%A6%81&subject_url=toolbox_event.hpg&project_id=dailynews&dtd_id=20&xml_id=0000654139|title=臺灣記憶 Taiwan Memory--國家圖書館|accessdate=2017-11-03|work=memory.ncl.edu.tw|language=zh-TW}}</ref>

[[晚年|晚年]]，托泰·布典定居於理那凡社（[[阿美語|阿美語]]：Rinafum，漢名光榮社區，位在[[鯉魚潭_(花蓮縣)|鯉魚潭]]山南側），並持續接受[[學者|學者]]拜訪。<ref name=":1" />

[[1990年代臺灣|1990年代]]初，托泰·布典前往[[蘭陽平原|蘭陽平原]]尋求其[[噶瑪蘭族|噶瑪蘭族]]身分認同。<ref name=":0" /><ref>{{Cite book|url=https://books.google.com.tw/books?id=snQmDwAAQBAJ&pg=PA107&lpg=PA107&dq=%E9%99%B3%E6%8A%B5%E5%B8%B6&source=bl&ots=Zc2oi55pCh&sig=oZ6IUkY-GZCoIbOGE56DjOpjiCw&hl=zh-TW&sa=X&ved=0ahUKEwiWnebRraHXAhXClJQKHTapCK0Q6AEIPDAE#v=onepage&q=%E9%99%B3%E6%8A%B5%E5%B8%B6&f=false|title=加禮宛戰役|last=[[施正鋒|施正鋒]]|first=|date=2010-05-01|publisher=中華民國政府出版品|year=|isbn=9789860236262|location=|pages=|language=zh-TW|chapter=}}</ref>

1993年，托泰·布典在拜訪[[環山部落|志佳陽社]]返鄉不久後[[過世|過世]]。<ref name=":0" /><ref name=":2" /><ref name=":1" />

== 家庭 ==
托泰·布典與陳秀娘[[結婚|結婚]]。<ref name=":0" />

== 軼事 ==
* 因托泰為一樹名，布典為一魚名，國分直一遂稱托泰·布典為「[[木魚|木魚]]」。<ref name=":1" />
* 托泰·布典在[[晚年|晚年]]仍能以[[日語|日語]]流利背誦[[金剛經|金剛經]]及[[法華經|法華經]]。<ref name=":1" />
