'''托尼·西里科'''（{{lang-en|'''Tony Sirico'''}}，本名'''Genaro Anthony Sirico Jr.'''，{{bd|1942年|7月29日}}）<ref>{{Cite web |url = https://www.biography.com/people/tony-sirico-204392 |title = Tony Sirico Biography |publisher = The Biography Channel |language = en}}</ref>是一名美國演員。他最出名是在電視劇《[[人在江湖_(美國電視劇)|人在江湖]]》中飾演{{link-en|Paulie Gualtieri|Paulie Gualtieri}}<ref>{{Cite web |url = https://uproxx.com/tv/the-sopranos-paulie-walnuts-facts/ |title = How Tony Sirico's Criminal Past Prepared Him To Play Paulie Walnuts On ‘The Sopranos’ |author = Christian Long |date = 2015-08-24 |publisher = uproxx.com |language = en}}</ref>，以及在[[活地·亞倫|活地·亞倫]]的幾部電影中演出。

== 早年生活 ==
西里科於1942年7月29日在[[紐約市|紐約市]]出生。

青少年時期，西里科是街頭幫派的成員，與非洲裔美國人和愛爾蘭人的社區發生長期爭執。他後來回憶起他曾經在布魯克林的聖布倫丹天主教堂（St Brendan's Catholic Church）的階梯被敵對的愛爾蘭暴徒槍擊。

在轉向演藝生涯之前，他曾被判犯幾項罪行，而且被捕28次<ref>{{Cite web |url = http://articles.latimes.com/1990-07-15/entertainment/ca-379_1_tony-sirico |title = A Real Tough Guy : Tony Sirico's rap sheet: 28 arrests and 27 acting jobs |author = Patrick Goldstein |date = 1990-07-15 |publisher = Los Angeles Times |language = en}}</ref>。
