'''托马斯·瓦尔德豪泽'''（{{lang-en|Thomas D. Waldhauser}}，{{bd|1953年|12月16日|||CatIdx=Thomas D. Waldhauser}}），昵称“汤姆”（{{lang-en|Tom}}）。瓦尔德豪泽为[[美国海军陆战队|美国海军陆战队]][[上将_(美国)|上将]]，现任[[美国非洲司令部|美国非洲司令部]]司令，之前曾在联合参谋部担任部队部署总监（J7），为国防部长[[莱昂·帕内塔|莱昂·帕内塔]]和[[查克·哈格尔|查克·哈格尔]]当过高级军事助理，在陆战队担任过分管作战计划与政策的副司令、[[美国海军陆战队第1远征军|第1远征军]]司令和[[陆战队中央司令部|陆战队中央司令部]]司令，于2016年7月任现职。<ref>{{cite web |url=https://dod.defense.gov/About/Biographies/Biography-View/Article/1324681/general-thomas-d-waldhauser/ |title=国防部瓦尔德豪泽官方简历}}</ref>

== 履历 ==
瓦尔德豪泽上将出生于[[明尼苏达州|明尼苏达州]][[南圣保罗|南圣保罗]]，毕业于[[伯米吉州立大学|伯米吉州立大学]]<ref>{{Cite web|url=https://www.bsualumni.org/alumni/alumni/outstanding-alumni-brigadier-general-thomas-waldhauser-2003|title=伯米吉州立大学校友简历 }}</ref>。瓦尔德豪泽通过海军陆战队军官候补学校加入陆战队，1976年获得军官委任。瓦尔德豪泽初任为陆战队步兵军官，在[[美国海军陆战队第1师|陆战1师]]和[[美国海军陆战队第1师|陆战2师]]担任排长和连队职务，在[[长滩号核动力导弹巡洋舰|长滩号核动力导弹巡洋舰]]（CGN-9）上担任陆战队分队长，还曾任第7两栖登陆舰中队长的参谋。 

晋升校官后，瓦尔德豪泽开始担任陆战队营的作战参谋，并在两栖战学校即现在的[[海军陆战队大学|海军陆战队大学]]担任教员，期间曾作为陆战队中央司令部先锋部队司令的参谋人员部署到中东，参加了[[海湾战争|海湾战争]]。后，瓦尔德豪泽在[[美国海军陆战队第1远征军|第2远征军]]担任参谋，并担任陆战队2团3营营长，虽然又在联会参谋部反恐作战部门担任参谋。

2000年，瓦尔德豪泽晋升为上校后，担任了第15陆战远征团团长，期间带领全团参加了[[持久自由军事行动|持久自由军事行动]]和[[入侵伊拉克|入侵伊拉克]]。

2003年，瓦尔德豪泽晋升为准将，担任了陆战队作战实验室主任，[[陆战队战斗开发司令部|陆战队战斗开发司令部]]副司令。2006年，瓦尔德豪泽担任了[[美国特种作战司令部|美国特种作战司令部]]参谋长。2007年9月，瓦尔德豪泽晋升为少将，担任[[美国海军陆战队第1师|陆战1师]]师长。

2009年7月，瓦尔德豪泽晋升为中将，在陆战队总部担任分管作战计划与政策的副司令，2010年10月，转任陆战队第1远征军司令，以及陆战队中央司令部司令。2012年10月，瓦尔德豪泽进入国防部，担任国防部长帕内塔和哈格尔的高级军事助理，2013年9月后担任联合参谋部部队部署总监（J7）。2016年7月，瓦尔德豪泽晋升为上将，担任非洲司令部司令。<ref>{{cite web |url=http://www.africom.mil/about-the-command/leadership/commander |title=非洲司令部瓦尔德豪泽官方简历}}</ref>
