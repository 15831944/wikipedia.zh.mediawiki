'''托比·马奎尔'''（{{lang-en|'''Tobey Maguire'''}}，{{bd|1975年|6月27日||catIdx=Maguire}}）<ref>[http://www.biography.com/people/tobey-maguire-9542472#synopsis Tobey Maguire Biography]</ref>是一位[[美国|美国]]男演员和監製，在1980年代末開始了他的演藝事業，他第一部演出的電影是没有台詞的《{{link-en|豆釘也瘋狂|The Wizard (film)}}》（1989年）。他主演[[森·溫美|森·溫美]]的電影《[[蜘蛛人_(電影)|蜘蛛俠]]》系列（2002年、2004年和2007年）而广为各国观众所熟悉，飾演標題角色[[蜘蛛俠|蜘蛛俠]]，他還有其他的演出包括《[[一人有一點顏色|一人有一點顏色]]》（1998年）、《[[總有驕陽|總有驕陽]]》（1999年）、《[[完美的男人|完美的男人]]》（2000年）、《[[壯志奔騰|壯志奔騰]]》（2003年）、《{{link-en|柏林暗戰|The Good German}}》（2006年）、《[[雙情路|雙情路]]》（2009年）和《[[大亨小傳_(2013年電影)|大亨小傳]]》（2013年）。

== 早年经历 ==
托比·马圭尔出生于[[加利福尼亚州|加利福尼亚州]]的[[圣塔莫尼卡|圣塔莫尼卡]]。父亲文森特·马奎尔（Vincent Maguire）是个[[建筑工人|建筑工人]]和[[厨师|厨师]]，母亲温迪·布朗（Wendy Brown）是[[秘书|秘书]]出身的[[剧作家|剧作家]]和影视制作人。托比出生时他的父母仅仅18-20岁，尚未结婚，托比出生后父母匆忙结婚，在托比两岁的时候又离婚。

== 出道 ==
1989年托比·马圭尔出演了生平第一部电影《{{link-en|豆釘也瘋狂|The Wizard (film)}}》，扮演一个在电子游戏厅出没的无赖，没有台词。

== 個人生活 ==
托比·马圭尔已与首饰设计师{{link-en|詹妮弗·梅耶|Jennifer Meyer}}[[订婚|订婚]]，[[准岳父|准岳父]]是[[环球电影公司|环球电影公司]]的高层管理人员[[荣恩·梅耶|荣恩·梅耶]]（Ron Meyer）。托比和未婚妻詹妮弗育有一女，2006年出生的露比。好友[[李奧納多·狄卡皮歐|李奧納多·狄卡皮歐]]也有出席。

托比·马圭尔在《{{link-en|溫馨家族|Parenthood (film)}}》的試鏡中，認識了李奧納多·狄卡皮歐，到了今天他們仍是一對很好的朋友。他們曾在2013年上映的《[[了不起的盖茨比_(2013年电影)|了不起的盖茨比]]》中合作。
