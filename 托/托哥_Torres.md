'''托哥'''（英語：Torres Pit， 1996年3月11日-），原名'''別洪優'''，香港youtuber。2014年從[[福建中學|福建中學]]畢業後成功考進[[香港大學|香港大學]]，並於2018年從城市規劃系畢業。他曾為[[土豆|土豆]]「[[神街坊|神街坊]]」的主持人，被譽為「街坊熱潮始祖」<ref>https://www.hk01.com/%E7%A4%BE%E5%8D%80%E5%B0%88%E9%A1%8C/235793/%E4%B8%8D%E6%98%AFkol-%E6%9B%BE%E8%A2%AB%E5%A4%96%E7%B1%8D%E6%9C%8B%E5%8F%8B%E8%AA%8D%E5%AE%9A-%E6%AF%92%E7%94%B7-%E6%B8%AF%E7%94%B7%E6%8B%8D%E7%89%87%E6%8E%A2%E8%A8%8E%E4%B8%AD%E8%A5%BF%E5%B7%AE%E7%95%B0</ref>。現於[[YouTube|YouTube]]經營其頻道。

== 簡歷 ==
托哥於[[福建中學|福建中學]]畢業。在香港大學修讀建築系，更在一年級時得到全班第一成績。但建築並非他的興趣，因而曾經遞交轉系申請，但失敗；亦曾打算退學，但擔心被別人視為失敗者而打消念頭。

在大一時，被土豆網「[[神街訪|神街訪]]」的製作人相中，成為頻道的主持人，訪問內容圍繞戀愛觀、性愛觀等，亦因其出位的主題令影片內地廣為流傳。

現時，托哥於[[YouTube|YouTube]]開立自己的頻道，影片的主題多為[[文化差異|文化差異]]、香港文化等，題材亦不再侷限於街訪形式。根據訪問，他設立頻道的目的為打破大家對亞洲人的刻板印象和帶予歡樂觀眾。除了經營YouTube頻道外。

== YouTuber生涯 ==
在還沒有設立YouTube頻道之前，他十分活躍於經營他的[[臉書|臉書]]專頁「托哥Torres」，並憑著在地鐵上高唱「人人常歡笑，不要眼淚掉」和「破解史上最長名字Uvuvwevwevwe Onyetenyevwe Ugwemuhwem Osas」<ref>http://www.orangenews.hk/news/system/2018/06/24/010092650.shtml</ref>兩片開始被人注意。在2018年，他開設其[[YouTube|YouTube]]頻道。

== 軼事 ==
托哥的[[英語|英語]]非常流利，經常被人懷疑他是否曾於外國受過教育。後來托哥於報紙訪問中澄清他是個土生土長的香港人，但因他對英文有著濃厚興趣，因此常常看英文戲劇與演講，才會操流利的英語<ref>https://www.hk01.com/社區專題/235793/不是kol-曾被外籍朋友認定-毒男-港男拍片探討中西差異</ref>。<br />
除了[[廣東話|廣東話]]、[[國語|國語]]以及[[英文|英文]]外、他也略懂法文。<br />
除了語言能力外，托哥最常被人討論的為其高達193厘米的身高。

== 批評 ==
托哥的影片大多以英文作為主要語言，因此常被網民批評為何一個華人要講英文。
