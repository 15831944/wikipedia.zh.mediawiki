《'''欢乐伙伴'''》是[[中央电视台|中央电视台]]和[[深圳华侨城实业公司|深圳华侨城实业公司]]联合制作的52集大型[[人偶剧|人偶剧]]，每集约10分钟<ref>{{cite web |url=http://www.cctv.com/service/agent/category/99-9-12.html |title=节目类别 |last= |first= |date=1999-09-12 |website=CCTV.com |publisher= |access-date=2018-12-21 |quote=}}</ref>，1999年8月16日于中央电视台的《[[大风车|大风车]]》栏目上播出<ref>{{cite web |url=http://eladies.sina.com.cn/movie/news/movie/1999-8-24/8146.shtml |title=人偶童话剧《欢乐伙伴》受小朋友关注 |last= |first= |date=1999-08-24 |website=新浪 |author=老冰 |publisher=天津日报 |access-date=2018-12-21 |quote=}}</ref><ref>{{cite web |url=http://www.cctv.com/program/dfc/20060518/101465.shtml |title=快乐风车 十年树人 |last= |first= |date=2006-05-18 |website=CCTV.com |publisher= |access-date=2018-12-21 |quote=}}</ref>。该剧自播出后受到少儿关注，2000年获得了全国电视节目金童奖<ref>{{cite web |url=http://www.chinaoct.com/hqc/shzr/shzrbg/595735/index.html |title=华侨城集团2008年社会责任报告 |last= |first= |date=2009-07-28 |website=华侨城 |publisher= |access-date=2018-12-21 |quote=}}</ref>。
