'''欢娱影视'''（全名:东阳欢娱影视文化有限公司）成立于2012年<ref>{{Cite web |url=http://www.chinanews.com/yl/2017/11-15/8377190.shtml |title=欢娱影视获2017年成长最快影视企业奖|accessdate= |author= |date=2017-11-15 |publisher=[[中新网|中新网]] |language= }}</ref>，创始人为[[中國大陸|中國大陸]]知名[[編劇|編劇]]、[[製作人|製作人]][[于正|-{于}-正]]，以及杨乐<ref name=founder>{{Cite web |url=https://www.tmtpost.com/3440594.html |title=独家对话《延禧攻略》出品人：百亿流量，逆风翻盘 |accessdate= |author= |date=2018-08-27 |publisher=钛媒体 |language= }}</ref>，总部位于[[北京市|北京市]][[朝阳区|朝阳区]]。前身是[[于正|-{于}-正]]在[[上海市|上海]]2003年成立的于正工作室<ref>{{Cite web |url=http://hnrb.zjol.com.cn/hnrb/html/2009-10/09/content_9773.htm |title=于正：编剧制片人“一肩挑”|accessdate= |author= |date=2009-10-09 |publisher=海宁日报 |language= }}</ref>，工作室隶属于该公司<ref name=founder />。
