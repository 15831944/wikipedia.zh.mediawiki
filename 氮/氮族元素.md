'''氮族元素'''是[[元素周期表|元素周期表]]的的ⅤA族元素（[[IUPAC|IUPAC]]新规定：15族），位于[[碳族元素|碳族元素]]和[[氧族元素|氧族元素]]之间，包括[[氮|氮]]（N）、[[磷|磷]]（P）、[[砷|砷]]（As）、[[锑|锑]]（Sb）、[[铋|铋]]（Bi）、[[镆|镆]]（Mc）六种元素。

这一族元素在[[化合物|化合物]]中可以呈现-3，+3，+5等多种[[化合价|化合价]]，他们的原子最外层都有5个[[电子|电子]]。最高正价都是+5价。

== 化学性质 ==
氮族元素中非金属（氮、磷、砷）的气态氢化物化学式可用RH<sub>3</sub>表示，且有NH<sub>3</sub>＞PH<sub>3</sub>＞AsH<sub>3</sub>。最高氧化物的化学式可用R<sub>2</sub>O<sub>5</sub>表示，其对应水化物为酸且有HNO<sub>3</sub>＞H<sub>3</sub>PO<sub>4</sub>＞H<sub>3</sub>AsO<sub>4</sub>。并砷虽是非金属，却已表现出某些金属性，而锑、铋却明显表现出金属性。

部分方程式：

N<sub>2</sub>+O<sub>2</sub> = 2NO（大气放电）

2NO+O<sub>2</sub> = 2NO<sub>2</sub>

3NO<sub>2</sub>+H<sub>2</sub>O = 2HNO<sub>3</sub>+NO（大气固氮）

N<sub>2</sub>+3H<sub>2</sub>=2NH<sub>3</sub>（高温高压、催化剂）

NH<sub>4</sub>Cl = NH<sub>3</sub>↑+HCl↑

NH<sub>3</sub>+HCl = NH<sub>4</sub>Cl 

4NH<sub>3</sub>+5O<sub>2</sub> = 4NO+6H<sub>2</sub>O（加热、催化剂）

2NO+O<sub>2</sub> = 2NO<sub>2</sub>

3NO<sub>2</sub>+H<sub>2</sub>O = 2HNO<sub>3</sub>+NO↑（制硝酸）

4HNO<sub>3</sub> = 2H<sub>2</sub>O + 4NO<sub>2</sub>↑+O<sub>2</sub>↑

Cu+4HNO<sub>3</sub>（浓） = Cu(NO<sub>3</sub>)<sub>2</sub>+2NO<sub>2</sub>↑+2H<sub>2</sub>O （其浓溶液与金属反应以Mg、Cu为分界线）
