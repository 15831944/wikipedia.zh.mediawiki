'''芥川龙之介'''（{{lang-ja|芥川 龍之介}}，{{bd|1892年|3月1日|1927年|7月24日|catIdx=Akutagawa Ryuunosuke}}），[[号|号]]“'''澄江堂主人'''”，[[俳号|俳号]]“'''我鬼'''”，[[日本|日本]]知名[[小说家|小说家]]，博通[[漢學|漢學]]、[[日本文學|日本文學]]、[[英國文學|英國文學]]，但一生為多種疾病、憂慮所苦而[[自鴆|自鴆]]，年僅卅五。其名作甚多，以[[極短篇|極短篇]]為主，如《[[竹林中|竹林中]]》、《[[羅生門_(小說)|罗生门]]》、《[[蜘蛛之絲|蜘蛛之絲]]》等三篇翻譯為[[漢語|漢語]]，選入[[中華民國|中華民國]]的高中[[國文|國文]]課文、選讀教材，其中最為人所熟知的作品是《竹林中》。

== 生平 ==
芥川龙之介原姓名新原龍之助，1892年（明治廿五年）生于[[東京|東京]]。父親[[新原敏三|新原敏三]]在[[京橋區|京橋區]]入船町8丁目以販賣[[牛奶|牛奶]]為生。為家中長子，有2个姐姐，不過，大姐在龍之介出生之前一年病死，當時只有6歲。他出生7個月後，由於母親精神異常，他被送到母親的娘家抚养，後來母親在他11歲離世後，[[過繼|過繼]]為兄長[[芥川道章|芥川道章]]（芥川龙之介之舅舅）的[[養子|養子]]，遂改姓芥川。

芥川一家鍾好[[文學|文學]]與[[戲劇|戲劇]]，充滿濃厚的[[江戶|江戶]]文人氣息，龍之介受此薰陶，故有深厚之文藝底蘊。11歲時就與同學發行手抄雜誌，並自己寫作、編輯，甚至自繪插圖。1913年他进入[[东京帝国大学|东京帝国大学]]，学习[[英国文学|英国文学]]，期间开始写作。毕业后通过英文教學和报纸编辑维生。早在1912年廿二歲時，他已完成了處女作《老年》，兩年後又发表了短篇小說《[[羅生門_(小說)|罗生门]]》，但並未受到重視。

1916年芥川畢業，論文題為「威廉·莫理斯研究」，其成績位列同屆二十人中之第二名。他在《新思潮》雜誌发表短篇小說《[[鼻子_(小說)|鼻子]]》，[[夏目漱石|夏目漱石]]读到後非常讚赏，对他多方关怀。这段时间他也开始创作[[俳句|俳句]]。1918年他发表《[[地狱变|地狱变]]》，讲述一个[[日本战国|日本战国]]时期的残酷故事，通过画师、画师女儿等人的遭遇，反映了纯粹的艺术和无辜的底层人民受邪恶的统治者的摧残。

1921年芥川龙之介作为[[大阪|大阪]][[每日新闻|每日新闻]]报社的[[记者|记者]]前往[[中国|中国]]四个月，在結束訪問之際與胡適會面，7月返抵日本，之後寫成《支那遊記》一書。这次任务非常繁重。在任务的压力和自身压抑作用下，他染上了多种疾病，一生為[[胃腸病|胃腸病]]、[[痔瘡|痔瘡]]、[[神經衰弱|神經衰弱]]、[[失眠|失眠]]症所苦。回到日本後，1922年他发表了《[[竹林中|竹林中]]》，与[[安布罗斯·比尔斯|安布罗斯·比尔斯]]的《[[月光小路|月光小路]]》结构类似，都是在一件案子的调查采集的各方的证词与说法。不同的是《月光小路》最后澄清了事实，而《竹林中》中各方的证词某些地方重合却有很大矛盾，但是又都能自圆其说。整个作品弥漫着压抑、彷徨、不定向的气氛。这反映了作者本人迷茫的思想。自此以後，由於病情恶化，芥川龙之介常出现[[幻觉|幻觉]]，当时的社會形勢也[[右傾|右傾]]，[[言论自由|言论自由]]受到從政府到普通人的打壓。这使得他此時期的作品更加压抑，如《[[河童_(小說)|河童]]》。

1925年（大正14年）在文化學院文學部擔任講師。隔年，由於胃潰瘍、失眠、神經衰弱的復發再次入住[[神奈川縣|神奈川縣]][[湯河原|湯河原]]療養。另一方面，妻子冢本文（婚後改為芥川文）搬到了[[鹄沼|鹄沼]]的別墅。2月22日，龍之介也來到了鹄沼並住在旅館東屋。7月20日租下了東屋i-4號別墅，讓妻子和三男居住。在此期間，完成了小品《租房子之後》、《鹄沼雜紀》、《點鬼簿》。

1927年（昭和二年）芥川龙之介继续写作随想集《[[侏儒的话|侏儒的话]]》，作品短小精悍，每段只有一两句话，但意味深长。姐夫卧軌自殺，芥川為二姐一家債務奔走導致身心俱疲。7月24日因「恍惚的不安」而[[仰藥|仰藥]]自盡，年僅35歲。

== 主要著作 ==
芥川龍之介在他短暂的一生中，写了超过150篇[[小说|小说]]。他的[[極短篇小說|極短篇小說]]篇幅很短，取材新颖，情节新奇甚至诡异。作品关注社会丑恶现象，但很少直接评论，而仅以冷峻的文字和简洁有力的语言来陈述，让读者深深感觉到其丑恶性，这使得他的小说即具有高度的艺术性又成为当时社会的缩影，《[[竹林中|竹林中]]》、《[[羅生門_(小說)|罗生门]]》、《[[蜘蛛之絲|蜘蛛之絲]]》、《[[地獄變|地獄變]]》、《[[杜子春_(芥川龍之介)|杜子春]]》、《[[鼻子|鼻子]]》、《[[南京的基督|南京的基督]]》等皆是经典之作，時常被選入課文、小說選集，或改編為戲劇。

== 评价 ==
* [[鲁迅|鲁迅]]评价其作品：“所用的主题最多的是希望之后的不安，或者正不安时之心情。”
*  [[室生犀星|室生犀星]]评价其人：“这个作家好像从书籍之间变出来的，在世上只活了三十几年，谈笑一通，马上又隐身于自己出来的书籍之间，不再出来。”

== 其他 ==

* 1935年芥川龙之介自杀去世8年后，他的毕生好友[[菊池寬|菊池寬]]设立了以他的名字命名的文学新人奖“[[芥川賞|芥川賞]]”，现已成为日本最重要文学奖之一，与“[[直木賞|直木賞]]”齐名，作為獎勵新進純文學作家的重要獎項之一，影響深遠。
* 1950年，日本著名[[导演|导演]][[黑泽明|黑泽明]]，将芥川的作品《竹林中》与《罗生门》合而为一，改编为电影《[[羅生門_(電影)|罗生门]]》，在国际上获得包括[[威尼斯影展|威尼斯影展]][[金獅獎|金獅獎]]在內的多个大奖，使[[日本电影|日本电影]]走向世界。此后，“[[罗生门|罗生门]]”更成为華語地區对于扑朔迷离的、各方说法不一的事件的[[代名词|代名词]]。
