'''島津貴子'''（{{bd|1939年|3月2日||catIdx=Shimazu, Takako}}）是前[[日本皇室|日本皇室]]成員，[[昭和天皇|昭和天皇]]與[[香淳皇后|香淳皇后]]的五女兒，原稱號為'''清宮貴子內親王'''，後與{{link-ja|島津久永|島津久永}}結婚後脫離皇室身份。獲勳勳一等[[寶冠章|寶冠章]]。她有四姊（[[東久邇成子|東久邇成子]]、[[久宮祐子內親王|久宮祐子內親王]]、[[鷹司和子|鷹司和子]]、[[池田厚子|池田厚子]]）、兩兄（[[明仁天皇|明仁天皇]]、[[常陸宮正仁親王|常陸宮正仁親王]]）。徽印為[[橘柑|橘柑]]。

== 生平 ==
1939年3月2日誕生。1960年3月10日，就讀於[[學習院大學|學習院大學]]文學部英國文學科時與[[國際協力銀行|日本輸出入銀行]]員{{link-ja|島津久永|島津久永}}結婚，並脫離皇籍身份。島津久永是[[日向國|日向]]{{link-ja|佐土原藩|佐土原藩}}主家第14代當主島津久範的次子，也是[[薩摩藩|薩摩藩]]第12代藩主[[島津忠義|島津忠義]]的孫子，相當於[[香淳皇后|香淳皇后]]的表弟。此外，他在[[學習院大學|學習院大學]]時期為[[明仁天皇|明仁親王]]的學友。隨著結婚，貴子從學習院大學休學。

結婚記者會上光鮮亮麗的情況讓人感到全新時代的來臨，於此之前的生日記者會上一句「請看看我選擇的人…」也成為當時的[[流行語|流行語]]，增加了[[戀愛結婚|戀愛結婚]]的成分<ref group="註釋">婚姻通過香淳皇后及[[宮內廳|宮內廳]]的安排，因此不能代表完全是戀愛結婚。不過香淳皇后支持女兒們透過[[相親|相親]]或[[約會|約會]]找尋對象。</ref>。婚後居住於[[世田谷區|世田谷區]]，當地也吸引許多湊熱鬧的民眾經過。貴子是{{link-en|培瑞茲・普拉多|Pérez Prado}}樂團的歌迷，成婚時培瑞茲・普拉多還曾經作曲獻給貴子祝賀。1962年，兒子{{link-ja|島津禎久|島津禎久}}出生，後成為一名[[攝影師|攝影師]]。

1963年10月26日，犯罪集團企圖[[誘拐|誘拐]]貴子並要求5000萬[[日圓|日圓]]的贖金未能得逞，同年11月21日遭到逮捕。

1964年4月29日，獲頒勳一等寶冠章。結婚後奢華的舉動遭到媒體批評，同時也取得了室內設計師的資格。1969年，任職於{{link-ja|東京王子大飯店|東京プリンスホテル}}內的購物中心「PISA」（ピサ），成為首位到民間企業就職的日本皇族<ref group="註釋">實際上擔任非常勤的顧問職，接近於名譽性質的工作。1989年至2005年間擔任王子大飯店董事。2005年就任王子大飯店室內顧問。</ref>。

1970年，擔任[[1970年世界博覽會|大阪萬國博覽會]]的開幕式、閉幕式電視主持人。

2005年，在[[明仁天皇|明仁天皇]]長女[[黑田清子|紀宮清子内親王]]結婚之際，以「內親王降嫁」先例的身份接受多家媒體採訪。相較於姊姊[[池田厚子|池田厚子]]，貴子頻繁出現於媒體，且任職於民間企業，被稱作是「異色的皇女」（異色の皇女）。

== 個人 ==
* 丈夫島津久永的父親島津久範伯爵，是母親香淳皇后的舅舅。
* 暱稱「'''Osutachan'''」來自於清宮貴子（'''Su'''ganomiya '''Ta'''kako）名字開頭的簡稱。
* 曾與[[大江健三郎|大江健三郎]]對談，收錄於大江的著作《世界的年輕人們》（新潮社、1962）。

== 著作 ==
* 《ある日幸せに》 島津久永共著 中谷吉隆攝影 有紀書房 1960
