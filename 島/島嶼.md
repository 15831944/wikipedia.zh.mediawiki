'''島嶼'''或'''島'''是指於[[海洋|海洋]]中或[[湖泊|湖泊]]中被[[水体|水体]]圍繞的[[陆地|陆地]]区域<ref>{{cite web |title = 教育部重編國語辭典修訂本 |url = http://dict.revised.moe.edu.tw/cgi-bin/cbdic/gsweb.cgi?ccd=fTYtaJ&o=e0&sec=sec1&op=v&view=0-1 |publisher = 中華民國教育部 |accessdate=2016-07-14}}</ref>。世界上面积最大的岛屿是[[格陵兰岛|格陵兰岛]]（面积更大的[[澳大利亚大陆|澳大利亚大陆]]不被视为岛屿，而視為[[大洋洲|大洋洲]]的主要部分）。在狹小的地域集中2個或以上的島嶼，即成「島嶼群」，大規模的島嶼群稱作「[[群島|群島]]」或「諸島」，列狀排列的群島即為「[[列島|列島]]」。而如果一個[[國家|國家]]的整個[[國土|國土]]都坐落在一個或數個島之上，則此國家可以被稱為[[島嶼國家|島嶼國家]]，簡稱“[[島國|島國]]”。

== 島嶼的分類 ==
[[File:Dent_Island.jpg|thumb]]可以依照地質成因、位置、開發狀況來區別。

按地質成因：島可以分为以下三种类型：[[火山岛|火山岛]]、[[大陆岛|大陆岛]]、[[珊瑚岛|珊瑚岛]]：[[珊瑚島|珊瑚島]]又可以再細分兩種。

按位置：
沿海各地对岛屿的称呼有所不同，有相当多的地区以“山”为名，尤以中國浙江沿海为多，长江以南（浙、闽、粤等地）对小岛又通常称为“嶼”。但實際上「島」、「嶼」從未有明正式或確定的定義，例如台灣的[[綠島|綠島]]面積僅為[[蘭嶼|蘭嶼]]的三分之一，但前者为「島」，后者为“屿”。而「排」、「石」或「岩」等則通常指更小的岛屿。有的島雖然細小，但是用「洲」來命名的，如[[香港|香港]]的[[長洲_(香港)|長洲]]，广东一带使用较多，同时内河（如长江）中因江河冲击而形成的沙洲型岛屿多有使用“洲”为名，如[[武汉|武汉]]的[[天兴洲|天兴洲]]。

==國際法上的島嶼 ==
在国际[[海洋法|海洋法]]上，《[[联合国海洋法公约|联合国海洋法公约]]》第121条规定，岛屿的[[领海|领海]]、[[毗连区|毗连区]]、[[专属经济区|专属经济区]]和[[大陆架|大陆架]]的确定与其他陆地领土相同，但是'''不能维持人类居住或其本身的经济生活的[[岩礁|岩礁]]，不应有专属经济区或大陆架'''。这个规定相对较接近于前述后一种划分标准。
