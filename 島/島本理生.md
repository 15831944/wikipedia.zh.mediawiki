'''島本理生'''（しまもと りお、[[1983年|1983年]][[5月18日|5月18日]] - ）是一位[[日本|日本]][[小說家|小說家]]<ref>{{Cite web |url =https://www.shinchosha.co.jp/writer/3442/ |title =著者プロフィール  |publisher =www.shinchosha.co.jp  |date =  |accessdate =2018-12-15 }}</ref>。

== 経歴 ==
島本理生出生於東京都[[板橋區|板橋區]]。母親長岡ゆり為舞踏家・鍼灸師，小時候父母離異<ref>『文藝』2010年春期号 p55</ref>。之後母親二度離婚，成為[[一人親家庭|母子家庭]]<ref>この部分については島本は1冊目のエッセイ『CHICAライフ』pp144 - 149「貧乏生まれ・貧乏育ち」などで簡素に語られることがあるが、島本が1度高校を中退していた時期があり「高校生時代」と言い切れない部分があるほか、母の2度目離婚の時期を明確には明らかにしていない。</ref>。

[[1998年|1998年]]、15歳時以『ヨル』當選『[[鳩よ!|鳩よ!]]』掌編小説雜誌第2期10月号、獲得年度[[最優秀選手|MVP]]。

[[2001年|2001年]]、島本理生以《輪廓》（『シルエット』）獲得第44回[[群像新人文学獎|群像新人文学獎]]優秀作品<ref>{{Cite web |date= |url=http://gunzo.kodansha.co.jp/upload/gunzo.kodansha.co.jp/files/img_j/gunzo-newawards.pdf |title=群像新人文学賞当選作・優秀作一覧 |format=PDF |publisher=講談社 |accessdate=2015-07-24}}</ref><ref name=davi1708>{{Cite journal | title=『[[ダ・ヴィンチ|ダ・ヴィンチ]]』2017年8月号|pages=55|publisher=KADOKAWA}}</ref>。[[2003年|2003年]]、《一點一滴》（『リトル・バイ・リトル』）入圍第128回[[芥川獎|芥川獎]]<ref>{{Cite web |url=http://homepage1.nifty.com/naokiaward/akutagawa/senpyo/senpyo128.htm |title=芥川賞 - 選評の概要 - 第128回 |publisher=芥川賞のすべて・のようなもの |accessdate=2015-07-25 |archive-url=https://web.archive.org/web/20150721020108/http://homepage1.nifty.com/naokiaward/akutagawa/senpyo/senpyo128.htm |archive-date=2015-07-21 |dead-url=yes }}</ref>。同年、島本理生畢業於[[東京都立新宿山吹高等学校|東京都立新宿山吹高等学校]]、就讀[[立教大學|立教大學]]文學部（2006年中退）、《一點一滴》獲得第25回[[野間文藝新人獎|野間文藝新人獎]]<ref name=davi1708/><ref>{{Cite web |url=http://www.kodansha.co.jp/about/nextgeneration/archive/22138.html |title=野間文芸新人賞 |publisher=講談社 |accessdate=2015-07-25}}</ref>，成為史上最年輕得得主。

[[2004年|2004年]]、《誕生之森》（『生まれる森』）入圍[[第130回芥川獎|第130回芥川獎]]<ref>{{Cite web |url=http://homepage1.nifty.com/naokiaward/akutagawa/senpyo/senpyo130.htm |title=芥川賞 - 選評の概要 - 第130回 |publisher=芥川賞のすべて・のようなもの |accessdate=2015-07-25 |archive-url=https://web.archive.org/web/20150725070813/http://homepage1.nifty.com/naokiaward/akutagawa/senpyo/senpyo130.htm |archive-date=2015-07-25 |dead-url=yes }}</ref>。[[2005年|2005年]]、『[[ナラタージュ|ナラタージュ]]』入圍第18回[[山本周五郎獎|山本周五郎獎]]<ref>{{Cite web |url=http://www.shinchosha.co.jp/prizes/yamamotosho/18/ |title=第十八回山本周五郎賞 |publisher=新潮社 |accessdate=2015-07-25}}</ref>。同作品獲選「この恋愛小説がすごい! 2006年版」（[[宝島社|宝島社]]）第1位、《本の雑誌》上半期10大小說第1位<ref>{{Cite web |url=http://www.kadokawa.co.jp/sp/200502-05/ |title=ナラタージュ / 島本理生 |publisher=web KADOKAWA |accessdate=2015-07-25}}</ref>、[[本屋大賞|本屋大賞]]第6位<ref>{{Cite web |url=http://www.hontai.or.jp/ceremony/report2006.html |title=2006年本屋大賞結果発表&発表会レポート |work=これまでの本屋大賞 |publisher=本屋大賞 |accessdate=2015-07-25}}</ref>。

同年、《在大熊到來之前，晚安》（『大きな熊が来る前に、おやすみ。』）入圍第135回芥川獎<ref>{{Cite web |url=http://homepage1.nifty.com/naokiaward/akutagawa/senpyo/senpyo135.htm |title=芥川賞 - 選評の概要 - 第135回 |publisher=芥川賞のすべて・のようなもの |accessdate=2015-07-25 |archive-url=https://web.archive.org/web/20150725065019/http://homepage1.nifty.com/naokiaward/akutagawa/senpyo/senpyo135.htm |archive-date=2015-07-25 |dead-url=yes }}</ref>。同年末，島本理生與[[佐藤友哉|佐藤友哉]]結婚、之後離婚<ref>[[山本文緒|山本文緒]]との対談にて。山本より「離婚されたことは公にしているの?」の問いに対して「自発的にはしていなかったんですけど、ネットなどでも、もうかなり広まっているので、お互いに、そろそろ公にしようか、ということになりました」と返答（『文藝』2010年春季号、p19 ）</ref>、[[2010年|2010年]]再婚。2011年第一子出生。

[[2007年|2007年]]、『Birthday』入圍第33回[[川端康成文学獎|川端康成文学獎]]<ref>{{Cite web |url=http://www.shinchosha.co.jp/prizes/kawabatasho/33/ |title=第三十三回川端康成文学賞 |publisher=新潮社 |accessdate=2015-07-25}}</ref>。[[2011年|2011年]]、《了解．或許》（『アンダスタンド・メイビー』）入圍第145回[[直木獎|直木獎]]<ref>{{Cite web |url=http://homepage1.nifty.com/naokiaward/senpyo/senpyo145.htm |title=直木賞 - 選評の概要 - 第145回 |publisher=直木賞のすべて |accessdate=2015-07-25 |archive-url=https://web.archive.org/web/20150725064501/http://homepage1.nifty.com/naokiaward/senpyo/senpyo145.htm |archive-date=2015-07-25 |dead-url=yes }}</ref>。

[[2015年|2015年]]、島本理生以『Red』獲得第21回[[島清戀愛文學獎|島清戀愛文學獎]]<ref name=davi1708/><ref>{{Cite web |url=http://www.hokkoku.co.jp/subpage/H20150211101.htm |title=島本理生さんの「Red」が受賞 「新生」島清恋愛文学賞 |date=2015-02-11 |publisher=[[北國新聞|北國新聞]] |accessdate=2015-07-21 |archive-url=https://web.archive.org/web/20150722042928/http://www.hokkoku.co.jp/subpage/H20150211101.htm |archive-date=2015-07-22 |dead-url=yes }}</ref>、『夏の裁断』入圍第153回芥川獎<ref>{{Cite web |url=http://hon.bunshun.jp/articles/-/3837 |title=第153回芥川賞候補作（抄録） 島本理生「夏の裁断」（文學界6月号）｜特集｜島本理生 |publisher=本の話WEB |accessdate=2015-07-25}}</ref>。[[2018年|2018年]]、『ファーストラヴ』獲得第159回直木獎。
