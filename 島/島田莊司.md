'''島田莊司'''（{{bd|1948年|10月12日|||Shimada, Souji}}），[[日本|日本]][[作家|作家]]、[[小說家|小說家]]，[[推理小说|推理小说]]界“[[新本格派|新本格派]]”推理小说的代表人物之一。

== 创作生涯 ==
出生于日本[[广岛县|广岛县]]。[[武藏野美术大学|武藏野美术大学]]毕业后，从事过各式各样的工作，33岁时以本格推理《[[占星术杀人事件|占星术杀人事件]]》初试啼声，入圍[[江戶川亂步獎|江戶川亂步獎]]<ref name=jlpp>[http://www.jlpp.go.jp/en/works/author01_13.html JLPP profile]</ref>，从此技惊四座。誉为“日本推理小说之神”的岛田庄司，在日本推理小说界有着举足轻重的影响，当代所謂「[[新本格派|新本格派]]」的推理作家，沒有一个不受到他的影响，[[新本格派|新本格派]]的開創者[[绫辻行人|绫辻行人]]甚至尊他为师。

在1980年代社会派推理小说盛行的时代，岛田以每年都推出一部话题作品的速度，另辟蹊径，开启了本格派推理小说的另一片天空。
島田的推理小說主要有兩大系列，一個以占星師'''[[御手洗潔|御手洗潔]]'''為主角，代表作包括《[[占星術殺人魔法|占星術殺人魔法]]》、《[[異邦騎士|異邦騎士]]》、《[[魔神的遊戲|魔神的遊戲]]》等；另一個則以刑警'''吉敷竹史'''為主角，代表作包括《[[奇想、天慟|奇想、天慟]]》、《[[北方夕鶴2/3殺人|北方夕鶴2/3殺人]]》等。除此之外，他還有不少備受好評的單篇推理傑作，如1985年以《[[被詛咒的木乃伊|被詛咒的木乃伊]]》獲頒日本夏洛克福爾摩斯特別獎。

2007年，擔任家鄉廣島福山所舉辦的「島田莊司選　薔薇的城市福山推理文學新人獎」的評審。每年選出的長篇小說得獎作品，輪流由講談社、光文社和原書房出版。

2008年，台灣的皇冠文化集團舉辦「[[島田莊司推理小說獎|島田莊司推理小說獎]]」，首獎作品將由台灣的皇冠出版社、日本的文藝春秋社、中國大陸的當代世界出版社/青馬文化和泰國的南美出版社共同出版。

== 無冠帝王 ==
1984年、島田莊司以《[[被詛咒的木乃伊|被詛咒的木乃伊]]》入圍[[直木獎|直木獎]]。1985年、島田莊司以《[[那年夏天、19歲的肖像|那年夏天、19歲的肖像]]》第二次入圍[[直木獎|直木獎]]。

1985年、島田莊司以《[[被詛咒的木乃伊|被詛咒的木乃伊]]》入圍[[吉川英治文学新人獎|吉川英治文学新人獎]]。

島田莊司曾連續八年入圍日本推理作家協會獎，皆未得獎。

2005年，島田莊司以《[[摩天樓的怪人|摩天樓的怪人]]》入圍[[本格推理大獎|本格推理大獎]]，最終以些微差距敗給東野圭吾的長篇推理小說《[[嫌疑犯X的獻身|嫌疑犯X的獻身]]》。

因為島田莊司自從出道以來未曾獲獎，因此被稱為「無冠帝王」<ref name="無冠">{{Cite web|author=|date=2009-05-07|url=http://www.yomiuri.co.jp/book/news/20090507bk09.htm|title=島田荘司さん「無冠」返上 : 出版トピック : 本よみうり堂|archiveurl=https://web.archive.org/web/20090508113504/http://www.yomiuri.co.jp/book/news/20090507bk09.htm|archivedate=2009年5月8日|work=|publisher=YOMIURI ONLINE（読売新聞）|accessdate=2013-04-13|deadlinkdate=2017年10月}}</ref>。

2008年10月22日，島田莊司榮獲第12回[[日本推理文學大獎|日本推理文學大獎]]<ref>{{Cite web |author= |date=2008-10-22 |url=http://sankei.jp.msn.com/culture/academic/081022/acd0810221922003-n1.htm |title=日本ミステリー文学大賞に島田荘司さん |archiveurl=https://web.archive.org/web/20090617212950/http://sankei.jp.msn.com/culture/academic/081022/acd0810221922003-n1.htm |archivedate=2009年6月17日 |work= |publisher=MSN産経ニュース |accessdate=2013-04-13 |deadlinkdate=2017年10月 }}</ref>，終於獲得生涯第一個獎項。
