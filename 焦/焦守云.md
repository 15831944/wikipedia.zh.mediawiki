'''焦守云'''（{{bd|1953年|||}}）女，籍贯[[山东省|山东省]][[博山县|博山县]]，[[焦裕禄|焦裕禄]]次女，[[中共十大|中共十大]]代表<ref name=cdsb>{{Cite web |url = http://e.chengdu.cn/html/2014-03/21/content_460501.htm |title = 焦裕禄女儿：真不要以为我们很容易我们很不容易 |author =  |date = 2014-03-21 |publisher =成都商报  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1953年焦守云出生。1964年11岁时，父亲焦裕禄在中共[[兰考县|兰考县]]委第二书记任上病逝。1966年初，[[新华社|新华社]][[穆青|穆青]]等人撰写的反映焦裕禄事迹的长篇通讯引起很大反响<ref name=cdsb/>。1966年5月[[文革|文革]]爆发后，8月下旬全国大、中学校学生开始“[[大串联|串联]]”，此后河南省不少中学传达中央文件，要求按10:1的比例推选学生代表到北京接受[[毛主席|毛主席]]接见。开封地区赴京代表团中，有焦裕禄的两个女儿焦守凤、焦守云。1966年9月15日，毛泽东在天安门城楼上第三次接见百万[[红卫兵|红卫兵]]，经周恩来同意，13岁的焦守云被领到天安门城楼上与毛泽东握手、讲话并合影<ref>{{Cite web |url = http://news.xinhuanet.com/politics/2014-03/25/c_126312706.htm |title = 1966年，毛泽东接见焦裕禄的女儿 |author =  |date = 2014-03-25 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref name=wudai/>；在天安门城楼的休息大厅，周恩来、[[邓小平|邓小平]]与焦守云握手，并问她：“妈妈身体好不好？兰考现在怎么样？”<ref name=wudai>{{Cite web |url =http://news.ifeng.com/a/20140410/35626828_0.shtml  |title =  焦裕禄女儿称中共五代领导人均接见过她和家人|author =  |date = 2014-04-10 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>次日，焦守云与毛泽东、周恩来、[[刘少奇|刘少奇]]等人的合影刊登在《[[人民日报|人民日报]]》<ref name=haogp>{{Cite web |url =http://news2.jschina.com.cn/system/2015/06/25/025231842.shtml  |title = 焦裕禄：人民好公仆 |author =  |date = 2015-06-25 |publisher = 中国江苏网 |language =  |accessdate =  |quote =  }}</ref>。

1968年，焦守云入伍。1973年，成为[[中共十大|中共十大]]代表，当时20岁的焦守云是所有中共十大代表中年龄最小的。1978年，到郑州市科技局下属的郑州市科学技术情报研究所工作。儿子[[余音|余音]]10岁时，焦守云成为单身母亲。此后，焦守云节衣缩食供儿子余音上学。2008年，自郑州市科学技术情报研究所退休<ref name=cdsb/>。退休后不久，2008年7月26日，[[2008年北京奥运会|北京奥运]]圣火在[[开封|开封]]传递，焦守云作为火炬手从[[郑海霞|郑海霞]]手中接过[[祥云火炬|祥云火炬]]并进行传递<ref name=huoju/>。

2004年焦裕禄逝世40周年，焦守云策划了大型音乐电视《焦裕禄之歌》，在[[中国中央电视台|中国中央电视台]]播出。退休后，焦守云继续在河南省焦裕禄精神研究会、河南省延安精神研究会等社会组织任职<ref name=huoju/>，还担任了[[焦裕禄干部学院|焦裕禄干部学院]]名誉院长等职<ref name=gcdy>{{Cite web |url =  http://news.southcn.com/g/2016-04/25/content_146592098.htm|title = 焦裕禄女儿焦守云：共产党员就是要在困难面前逞英雄 |author =  |date = 2016-04-25 |publisher = 南方网 |language =  |accessdate =  |quote =  }}</ref>。退休后还参与焦裕禄主题的纪录片、电视剧、音乐剧的拍摄和制作<ref name=cdsb/>。先后担任电视剧《焦裕禄》总策划，大型原创音乐剧《焦裕禄》总策划，大型电影纪录片《永远的焦裕禄》总顾问，《焦裕禄在洛矿》总顾问等<ref name=gcdy/>。焦守云曾作为焦裕禄亲属的代表，受到[[中共中央总书记|中共中央总书记]][[江泽民|江泽民]]、[[胡锦涛|胡锦涛]]、[[习近平|习近平]]接见<ref name=huoju>{{Cite web |url = http://www.dahe.cn/xwzx/zt/hnzt/chuandi2008/hncd/t20080726_1353042.htm |title =火炬手焦守云：传递火炬是我和父亲的双重荣誉  |author =  |date = 2008-07-26 |publisher = 大河网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.xinhuanet.com/mrdx/2017-05/12/c_136276879.htm |title =是谁,让习近平常思之念之  |author =  |date = 2017-05-12 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。
