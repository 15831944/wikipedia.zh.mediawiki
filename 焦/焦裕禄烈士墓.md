'''焦裕禄烈士墓'''，是一座位於[[河南省|河南省]][[开封市|开封市]][[兰考县|兰考县]]的[[全国重点文物保护单位|全国重点文物保护单位]]。[[焦裕祿|焦裕祿]]生前曾任[[河南|河南]][[開封市|開封市]][[蘭考縣|蘭考縣]]的[[縣委書記|縣委書記]]，死後被批准为[[烈士_(中华人民共和国)|革命烈士]]；他起初葬於[[鄭州市烈士公墓|鄭州市烈士公墓]]，後來遂其遺願遷葬到現址，其墓是[[焦裕祿烈士陵園|焦裕祿烈士陵園]]的一部分。焦裕禄烈士墓在[[文化大革命|文化大革命]]時一度被[[張欽禮|張欽禮]]等人用作懲處反對者。後來，不少中華人民共和國高層領導人、中國共產黨黨員幹部到訪了焦裕祿烈士陵園；截至2003年4月，焦裕祿烈士墓共接待了超過500萬人次的訪客。2003年4月，焦裕祿烈士墓被增補列入[[第五批全國重點文物保護單位|第五批全國重點文物保護單位]]。

== 歷史 ==
[[焦裕禄|焦裕禄]]是[[河南|河南]][[开封市|开封市]][[兰考县|兰考县]]的[[县委书记|县委书记]]，在1964年5月14日病逝於[[郑州|郑州]]，死後受到[[中國共產黨|中國共產黨]]的讚揚，被[[河南省人民政府|河南省人民政府]]批准為[[烈士_(中华人民共和国)|革命烈士]]<ref name="中华人民共和国民政部"/>。他死前留下遗言，指自己未能把兰考县的[[沙丘|沙丘]]治理好，希望自己的遺體能运回兰考、埋在沙堆上，看着人們治理好沙丘<ref name="华艺出版社"/>。但是，由於當時天氣炎熱，他的遺體暫時葬於[[郑州市烈士公墓|郑州市烈士公墓]]，而非運到兰考县安葬<ref name="中囯共产党中央宣传部宣敎局"/>。到了翌年的2月26日，[[中国共产党河南省委员会|中国共产党河南省委员会]]與[[河南省人民政府|河南省人民政府]]决定按焦裕禄的遗願，将其遗骨迁葬到兰考县城北面[[黄河|黄河]]故堤的沙丘上<ref name="中国人民大学书报资料社"/>。焦裕禄烈士墓是[[焦裕禄烈士陵园|焦裕禄烈士陵园]]的一部分，陵园內的建築物還有纪念碑和纪念馆<ref name="中华人民共和国民政部2"/>。

在1966年開始的[[文化大革命|文化大革命]]時期，焦裕禄烈士墓被[[张钦礼|张钦礼]]等政客用作懲處反对者的場所<ref name="人民文学出版社"/>。一些爱戴焦裕禄的[[红卫兵|红卫兵]]和群众受张钦礼等人指使，把反對张钦礼的前[[中國工農紅軍|红军]]軍人、前[[幹部|幹部]]或[[知识分子|知识分子]]带到焦裕禄墓前毒打，並強迫他们在墓前[[下跪|下跪]]謝罪<ref name="人民文学出版社"/>。

中华人民共和国领导人[[毛泽东|毛泽东]]在焦裕禄烈士墓後方的屏风墙題寫了「为人民而死，虽死犹荣」字句<ref name="新华网"/>。此外，不少中華人民共和國高層领导人亦曾到訪焦裕禄烈士陵园，當中包括[[江泽民|江泽民]]、[[胡锦涛|胡锦涛]]、[[李岚清|李岚清]]、[[温家宝|温家宝]]、[[曾庆红|曾庆红]]、[[李长春|李长春]]、[[罗干|罗干]]、[[李鹏|李鹏]]、[[朱镕基|朱镕基]]、[[李瑞环|李瑞环]]、[[乔石|乔石]]、[[刘华清|刘华清]]、[[習近平|習近平]]等<ref name="新华网"/><ref name="兰考县人民政府"/><ref name="人民网"/>。许多中國共產黨党员和幹部到訪過焦裕禄烈士陵园，瞻仰焦裕禄墓、並献上[[花圈|花圈]]，有些团体會在墓前举行[[宣誓|宣誓]]仪式；根據陵园管理人員提供的數據，在2014年3月至5月，每天均有逾万人次、逾百团体來訪焦裕禄烈士陵园<ref name="新华网"/>。截至2003年4月，焦裕禄烈士墓共接待了超過500万人次的訪客<ref name="人民网"/>。

== 結構 ==
焦裕禄烈士墓佔地16平方米，以[[汉白玉|汉白玉]]砌成，长4.95米、宽2.3米、高0.55米；墓的底座為长方形，墓顶為[[蓋瓦|蓋瓦]]，墓前竖立了一塊[[大理石|大理石]][[墓碑|墓碑]]<ref name="人民网"/>。墓的北面有一堵高七米的屏风墙，上面鑲嵌了毛泽东题写的「为人民而死，虽死犹荣」<ref name="新华网"/>。

== 保護 ==
自1966年建成以來，焦裕禄烈士墓分別在1968年和1974年進行了重修工程<ref name="人民网"/>。焦裕禄烈士墓在1992年列入[[全国重点烈士纪念建筑物保护单位|全国重点烈士纪念建筑物保护单位]]，1997年成為[[全国爱国主义教育基地|全国爱国主义教育基地]]之一，並在2000年列為[[河南省文物保护单位|河南省文物保护单位]]；2003年4月，焦裕禄烈士墓被增补列入[[第五批全国重点文物保护单位|第五批全国重点文物保护单位]]<ref name="人民网"/>。
