'''茅台酒'''是一种[[酱香型|酱香型]][[大麯酒|大麯]][[白酒|白酒]]，原產于[[贵州|贵州]][[茅台镇|茅台镇]]，被称为“'''中國国酒'''”{{来源请求}}。

茅台酒氣味芬芳香醇濃郁，不論氣味與口感均散發強烈香氣。

==茅台酒历史==
===西汉===
一种说法是，历史名酒“枸酱”即为茅台的雏形。[[西汉|西汉]]建元6年（公元前135年），[[汉武帝|汉武帝]]派[[唐蒙_(汉朝)|唐蒙]]出使南越（今[[广州市|广州]]）。其间唐蒙得饮名酒“枸酱”，感觉酒味绝美，回到长安后，向蜀商咨询，蜀商说“独蜀出枸酱，多持窃出夜郎。”于是唐蒙绕道取枸酱酒献汉武帝。汉武帝以“甘美之”赞叹，视为珍奇，当时即为贡品，故有“唐蒙饮枸酱而使西域”之说。清代诗人[[陈熙晋|陈熙晋]]写道：“尤物移人付酒怀，荔枝滩上瘴烟开，汉家枸酱知何物，赚得唐蒙习部来。”

===清===
茅台镇在[[清朝|清]][[嘉庆|嘉庆]]、[[道光|道光]]年间造酒已初具规模，据《[[遵义|遵义]]府志》记载，“茅台（酿酒的）烧房不下20餘家，所费山粮不下两万石”。而且所酿之酒曰“茅台烧”。但古酿造作房被[[太平天国|太平天国]]翼王[[石达开|石达开]]进攻贵州时所毁。

==貴州茅台酒==
貴州茅台銷售策略有點特別，分為「內銷」、「外銷」兩種，我們常聽到的五星茅台就是內銷版本，飛天茅台則是外銷版本。但目前在中国国内五星茅台和飞天茅台均有销售，二者品质相同，飞天茅台价格比五星茅台高出10元人民币，还额外赠送两只酒杯。

===內銷===

# 車輪牌茅台（1954~1966）：更早期的茅台酒由於沒有正式官方的商標註冊，很容易被仿冒，也缺少官方的文件證明真假，因此市場普遍認定1954年後的『車輪牌』為最早的茅台酒。
# 三大革命茅台（1966~1982）：這個階段正值中國的革命年間，所謂的[[三大革命|三大革命]]為：[[公私合營|公私合營]]、[[土地改革|土地改革]]、[[文化大革命|文化大革命]]，由於貴州茅台酒廠在酒款背標上撰寫有[[三大革命|三大革命]]的文字，這時期的茅台酒因此得名『[[三大革命|三大革命]]茅台酒』。
# 地方國營茅台（1982~1986）：1982年貴州茅台酒廠的背標更換，[[三大革命|三大革命]]茅台的年代結束。收藏家看到酒標正面右下角的落款為『地方國營』，就將這個時期的茅台酒稱作『地方國營茅台』。
# 鐵蓋茅台（1987~1990）：經過了幾年外銷版茅台的測試，1987年內銷版的茅台酒也更改封口為鋁蓋，市場稱為『鐵蓋五星茅台』。
# 紅膜五星茅台（1991~1996）：內銷版紅膜的技術與外銷版同時引進，就是在蓋子外面增加一層紅色的塑料膜，市場稱為『紅膜茅台』。
# 透明膜五星茅台（1996~2009）：貴州茅台酒廠於1996年從義大利引進了新的封口技術，便改用紅色塑膠蓋加上透明膜的方式封口，市場稱為『透明膜茅台』。
# 新版茅台(2009~現在)：從2009年開始，貴州茅台又引進了新技術，使用一種新型的塑膠封膜，外觀看起來類似布料，變換角度可以看見封膜上有茅台酒廠的圖案以及『国酒茅台』的字樣，這種封膜方式一直沿用到現在。內銷版的酒標上印的是五星。
#生肖酒（2014~现在）：“一岁一生肖，一酒一茅台”，从2014年以来，贵州茅台每年根据年份发布的不同的生肖茅台，属于超高端定位。
===外銷===
在茅台正式官方的商標註冊後，內外銷皆以車輪牌茅台為主，又因為最初茅台酒以外銷為主，加上[[冷战|冷战]]期间“五星”商标被西方阵营视为政治象征，所以1958年後外銷的飛天茅台正式出現。
# 飛天茅台（1958~1966）：這是貴州茅台第一次使用飛天的商標，『飛天』也正式成為經典的外銷茅台包裝。
# 葵花茅台（1967~1975）：1966年中國產開文化大革命，由於『飛天茅台』的商標畫的是仙女，有『[[四舊|四舊]]』怪力亂神的嫌疑，貴州茅台酒廠討論後改商標為『葵花茅台』。
# 大飛天茅台（1976~1985）：文化大革命熱潮結束後，貴州茅台酒廠發現不少海外國家和地區不喜歡『葵花』的商標，因此『葵花』重新改為『飛天』商標。
# 小飛天茅台（1985~1990）：1985年外銷茅台酒率先採用的鋁蓋的設計，並且容量重新調整為500ml，這個時期由於容量的區分，被稱為『小飛天』。
# 紅膜飛天茅台（1991~1995）：這時期的茅台酒，瓶口蓋子外面增加了一層紅色的塑料膜，因此被稱為『紅膜茅台』。
# 透明膜飛天茅台（1996~2009）：貴州茅台酒廠於1996年從義大利引進了新的封口技術，開始使用紅色塑膠蓋加上透明膜的方式封口，市場稱為『透明膜茅台』。
# 新版茅台（2009~現在）：從2009年開始，貴州茅台又引進了新技術，使用一種新型的塑膠封膜，外觀看起來類似布料，變換角度可以看見封膜上有茅台酒廠的圖案以及『国酒茅台』的字樣，這種封膜方式一直沿用到現在。外銷版的酒標上印的是飛天。

==台灣茅台酒==
1949年蔣介石撤台時帶走了茅台酒廠珍藏的原漿酒，還帶走了幾名釀酒師，以此為基底承襲茅台酒的釀造工藝，現由台灣菸酒公司負責釀造，酒精濃度一樣高，但味道與香氣較淡。目前品牌名稱為玉山茅台酒。

大跃进期间茅台也“[[放卫星|放卫星]]”，1958年毛泽东提出要“产量搞上一万吨”<ref>http://jiu.163.com/14/0131/09/9JTHBT2200824IN2.html</ref>。然后产量经过连级跳，从57年的283吨变成60年的912吨。<ref>http://money.163.com/09/0811/16/5GETPM6F00253JOP.html</ref>1952年[[第一届全国评酒会|第一届全国评酒会]]将茅台酒评为名酒。当时评出的名酒中只有四种白酒，因此后来将茅台等四种酒称为[[中国四大名酒|中国四大名酒]]。

[[文化大革命|文化大革命]]爆发后，原先茅台的“飞天”商标被认为是“四旧”而遭弃置，这一期间生产的茅台酒被冠以“葵花”商标，称为“葵花茅台”。“飞天”商标直到1975年才恢复使用。

===改革开放后===
茅台酒的酿制技艺获得[[国家级非物质文化遗产|国家级非物质文化遗产]]。市面上正规专卖店和大卖场渠道销售的大多数都是53°和38°飞天、红星茅台酒，贵州茅台酒股份有限公司还生产了面向中低端市场的茅台王子酒、茅台迎宾酒和贵州大曲。除传统的中国白酒之外，茅台集团还生产[[啤酒|啤酒]]和[[利口酒|利口酒]]等产品。另外还有贵州茅台集团技术开发公司这一旗下企业，以集体品牌的形式生产各种价格低廉的茅台酒，但多以浓香型为主，一定程度上混淆了市场。随着茅台酒厂产能扩大，将越来越能够满足一定的国内市场需求，但是从原产地产量极限来说，茅台酒始终将是一种稀缺商品。

===21世纪===
茅台酒近年来价格不断上涨，2011年[[贵州茅台|贵州茅台]]向骗子组织[[世界奢侈品协会|世界奢侈品协会]]“申请”“奢侈品资格”引发争议<ref>{{cite journal|author=|title=百度搜索 “茅台入奢门”|journal=|date=|year=|month=|issue=|url=http://www.baidu.com/s?tn=baiduhome_pg&bs=%C3%A9%CC%A8+%C1%EC%B5%BC&f=8&rsv_bp=1&rsv_spt=1&wd=%C3%A9%CC%A8+%C8%EB%C9%DD%C3%C5&inputT=4978|accessdate=}}</ref>。加之中国大陆屡有国有企业花费巨款购买茅台等名酒的事件被曝光，遂使“茅台”经常与“公款吃喝”、“三公消费”、“腐败”等词语同时出现在热点话题中，成为网民调侃讽刺的对象。在2012年[[中央八项规定|中央八项规定]]出台前，一些经销商为了优先保证单位团购，只有剩下的茅台才向普通民众销售，因此茅台酒时常出现断货。当时在茅台酒厂的自营零售店购买茅台酒需要出示身份证，每人限购两瓶，且需要排队才能买到。八项规定出台后，白酒的公款消费受到严格限制，茅台酒过去一度被炒起来的虚高价格出现下降，对民众的供应量也恢复正常<ref>{{cite news|title=袁仁国：公务消费由30%降到1%，反腐败让茅台活得更好|url=http://www.jiemian.com/article/915167.html|publisher=界面 |date=2016年10月22日 |accessdate=2016年10月22日}}</ref>。

==事件==
===售卖假酒事件===
2012年6月29日，北京一位孙姓市民在[[华润万家|华润万家]]超市购买了7瓶53度飞天茅台，后发现7瓶酒的包装都不一样，怀疑买到了假酒。孙姓市民将其中一瓶茅台酒送至[[国家食品质量监督检验中心|国家食品质量监督检验中心]]检测，结果显示送检的酒不是茅台酒。检测后，[[北京市|北京市]][[西城区|西城区]]长安街工商所将剩余6瓶酒送至茅台酒厂检验，鉴定显示均为假货。2015年3月13日，孙姓市民将[[华润万家|华润万家]]超市起诉至法院，要求退款并赔十倍损失。华润万家被判赔款后提起上诉，但法院审理认为，华润万家销售的涉案茅台酒不符合食品安全标准，但超市无法提供证据证明其所供涉案酒水来源，最终认定超市对销售假茅台酒的行为是明知，驳回上诉，维持原判<ref>{{cite news|title=1.4万买到假茅台 消费者诉华润超市获10倍赔偿 |url=http://news.sina.com.cn/c/2015-03-13/180331604610.shtml |publisher=新京报 |date=2015年3月13日 |accessdate=2015年3月17日 }}</ref>。

===奖项涉嫌造假===
2015年3月17日，媒体报道称茅台酒所获得的“[[巴拿马万国博览会|巴拿马万国博览会]]金奖”涉嫌造假<ref>{{cite news|title=茅台所获百年金奖被指造假 摔瓶获奖说疑不实 |url=http://news.sina.com.cn/c/2015-03-17/090831614623.shtml?from=hao123_news_index_paihang_news|publisher=长江商报  |date=2015年3月17日 |accessdate=2015年3月17日 }}</ref>。

===“国酒茅台”商标注册争议===
2001年开始，贵州茅台开始提交“国酒茅台”商标的注册申请，之后在2012年7月通过初审。同年8月，汾酒、杜康等多家白酒企业表示反对，并向[[中华人民共和国国家工商行政管理总局商标局|国家工商总局商标局]]提交95份异议文件。2016年12月26日，国家工商总局商标局下发决定，称“国酒茅台”商标不予注册，原因是“国酒”一词带有“国内最好的酒”、“国家级酒”的评价性含义，若由被异议人永久性地独占使用，容易对公平的市场竞争秩序产生负面影响。另外，“国酒茅台”商标也违反了《商标法》“不得注册禁止含有国家名称、国旗、国徽、军旗、勋章相同或者近似的商标”的规定，同时也违反了2015年修订的《中华人民共和国广告法》中不利于“国酒茅台”商标申请的规定“广告中不得使用‘国家级’、‘最高级’、‘最佳’等用语”，这也意味着“国酒茅台”这类宣传今后将不允许再出现。但对于全国各地成立的“××国酒茅台销售有限公司”，则可以继续使用<ref>{{cite news|title=“国酒茅台”商标申请被驳回：影响市场竞争秩序|url=http://news.qq.com/a/20170103/000952.htm?winzoom=1|publisher=成都商报  |date=2017年1月3日 |accessdate=2017年1月5日 }}</ref><ref>{{cite news|title=“国酒茅台”商标落空,中华香烟、中华牙膏为什么能注册成功？|url=http://www.jiemian.com/article/1050881.html|publisher=界面|date=2017年1月4日 |accessdate=2017年1月5日 }}</ref>。2018年8月，贵州茅台在其官网发表声明称，决定放弃“国酒茅台”商标注册申请。<ref>{{cite news|title=“茅台致歉！放弃国酒商标申请，撤销对国家商评委的起诉”|url=http://www.bjnews.com.cn/food/2018/08/14/499422.html|publisher=新京报|date=2018年8月13日 |accessdate=2018年8月14日}}</ref>

===暴利问题===
2018年1月，贵州仁怀地区一位农民写信给茅台公司高层，质疑茅台收购高粱价格7年没涨，一公斤7.2元的价格和现在茅台1000多元的价格相比，粮农种10亩高粱都挣不到1瓶茅台钱，茅台自此陷入暴利风波。对此，茅台方面反驳了“压制农民”的不实传闻，并表示茅台收购的有机高粱远高于市场价，而且今年的收购价将提至8.2元/公斤<ref>{{cite web|title=茅台陷“暴利”风波 狗年生肖茅台被“秒杀”|url=http://www.bjd.com.cn/jx/jj/201801/31/t20180131_11079725.html|website=京报网|accessdate=2018-02-01|date=2018-01-31}}</ref>。
