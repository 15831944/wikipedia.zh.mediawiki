'''阿拉贡先驱报'''（{{lang-es|''Heraldo de Aragón''}}）是[[西班牙|西班牙]]阿拉贡自治区首府[[萨拉戈萨|萨拉戈萨]]市发行的地区日报。它是阿拉贡地区最重要的报纸<ref name=ref>{{cite web |title=TARIFAS PUBLICITARIAS 2017 |url=https://web.archive.org/web/20171215021454/http://www.heraldo.es:80/uploads/documentos/2017/01/25/_tarifasbluemedia2017_763bcadd.pdf |website=heraldo.es |accessdate=2017-12-15|language=es}}</ref><ref name=cri/>，也是该地区发行量最高的报纸<ref name=wan/>。

==历史==
《阿拉贡先驱报》创刊于1895年9月20日<ref>{{cite web |author=Javier Torrontegui |title='Heraldo de Aragón' celebra su primer centenario |url=https://elpais.com/diario/1995/09/20/sociedad/811548011_850215.html |publisher=[[国家报|EL PAÍS]] |archiveurl=https://web.archive.org/web/20190114102846/https://elpais.com/diario/1995/09/20/sociedad/811548011_850215.html|dead-url=no |archivedate=2019-01-14 |language=es |date=1995-09-20}}</ref><ref>{{cite web|title=Call for applications 2011|url=http://becas.universia.es/ficherosadjuntos/ES/50/MIT_Zaragoza_Heraldo_Scholarship.pdf|publisher=ZLOG|accessdate=2014-12-23}}</ref>。它的拥有者是成立于1909年<ref>{{cite web |title=HERALDO DE ARAGON SA |url=https://web.archive.org/web/20190115043923/https://teruel.cataloxy.es/firms/www.factoriadecontenidos.es.htm |publisher=Cataloxy Teruel |accessdate=2019-01-15 |language=es}}</ref>的阿拉贡先驱报[[股份有限公司#西班牙|股份公司]]，该公司旗下还拥有《[[索里亚|索里亚]]先驱报》等地区报纸<ref name=zlc>{{cite web|title=ZLOG Class of 2015|url=https://web.archive.org/web/20141223102529/http://www.zlc.edu.es/content/files/scholarships/MIT_Zaragoza_Heraldo_Scholarship.pdf|publisher=Zaragoza Logistics Center|accessdate=2014-12-19|language=en}}</ref>。《阿拉贡先驱报》的编辑总部位于萨拉戈萨市<ref>{{cite journal|author=Cristina Zurutuza-Muñoz|author2=Carmela García-Ortega|title=2009 European Parliament Elections in the Aragonese Press|journal=Revista Latina de Comucicion Social|date=2012|url=http://www.revistalatinacs.org/067/art/945_Zaragoza/01_CristinaEN.html|accessdate=2015-02-21}}</ref>，在[[韋斯卡省|韋斯卡省]]、[[薩拉戈薩省|薩拉戈薩省]]和[[特魯埃爾省|特魯埃爾省]]都有发行<ref name=wan>{{cite web|title=Heraldo de Aragón installs a shaftless KBA Comet|url=http://www.wan-ifra.org/system/files/field_ifra_mag_file/E_nt977_56-58.pdf|publisher=WAN IFRA|accessdate=2014-12-19|language=en}}</ref>。

该报获得2003年{{tsl|en|European Newspaper Award|欧洲年度报纸奖}}地区报纸奖<ref>{{cite web|title=The Results of the Fifth European Newspaper Award|url=http://www.editorial-design.com/03/award03e/presse.html|publisher=Editorial Design|accessdate=2014-12-19|deadurl=yes|archiveurl=https://web.archive.org/web/20150923233641/http://www.editorial-design.com/03/award03e/presse.html|archivedate=2015-09-23}}</ref>。

==奖项==
《阿拉贡先驱报》每年颁发“先驱奖”（{{lang|es|Premios Heraldo}}）。该奖项设立于2004年，分为新闻奖和人类贡献与知识奖<ref>{{cite web |title=El 'heraldo de aragón' entrega hoy sus premios anuales |url=https://web.archive.org/web/20190115065529/https://ecodiario.eleconomista.es/sociedad/noticias/9398536/09/18/El-heraldo-de-aragon-entrega-hoy-sus-premios-anuales.html |publisher=Eco Diario |accessdate=2019-01-15 |language=es |date=2018-09-20}}</ref>。
==母公司==
阿拉贡先驱报股份公司的[[母公司|母公司]]是“先驱集团”（{{lang|es|Grupo Heraldo}}）。2015年，“先驱集团”收购了西班牙最大的[[免费报纸|免费报纸]]——《[[20分钟报_(西班牙)|20分钟报]]》<ref>{{cite web |title='Heraldo de Aragón' compra el diario '20 Minutos'|url=https://www.elconfidencial.com/comunicacion/2015-06-09/heraldo-de-aragon-compra-el-diario-20-minutos_876057/ |publisher=[[机密报|El Confidencial]] |accessdate=2019-01-12 |language=es |date=2015-06-09}}</ref><ref>{{cite web|title=Heraldo cierra la compra de 20minutos |url=https://www.heraldo.es/noticias/comunicacion/2015/07/02/heraldo_cierra_compra_20minutos_374578_311.html |website=Heraldo.es |accessdate=2019-01-12 |language=es |location=Zaragoza |date=2015-07-02}}</ref>。

2016年9月，“先驱集团”更名为“埃奈欧”（{{lang|es|Henneo}}）<ref>{{cite web |title=Grupo Heraldo se transforma en HENNEO, que presidirá Fernando de Yarza López-Madrazo |url=https://web.archive.org/web/20180412233917/http://www.europapress.es:80/economia/red-empresas-00953/noticia-grupo-heraldo-transforma-henneo-presidira-fernando-yarza-lopez-madrazo-20160923093023.html |publisher=Europa Press |accessdate=2018-04-12 |location=Zaragoza |language=es |date=2016-09-23}}</ref>，新名称由单词”先驱“的前两个字母和[[希腊语|希腊语]]前缀neo组成，暗示着公司崭新的发展阶段与即将面临的新挑战<ref>{{cite web |title=Grupo HERALDO se transforma en HENNEO para seguir creciendo y afrontar nuevos retos |url=https://web.archive.org/web/20171213085935/http://www.heraldo.es:80/noticias/comunicacion/2016/09/23/grupo-heraldo-transforma-henneo-para-seguir-creciendo-afrontar-nuevos-retos-1073033-311.html |website=heraldo.es |accessdate=2017-12-13 |language=es |date=2016-09-23}}</ref>。
