'''阿魏'''（[[学名|学名]]：''{{lang|la|Ferula assafoetida}}''）是一种[[印度|印度]]香料，又名'''阿虞'''、'''興渠'''、'''薰渠'''、'''哈昔尼'''、'''芸臺'''等<ref>{{cite web |url=http://www.pts.org.tw/~web02/education/05page1-e14-b2.htm |title=公共電視台教育資源網：三星出好蔥|date= 2004}}</ref>。這種香草在西域有出產，但中国不常见，是一種葉子近似[[芫荽|芫荽]]的植物。

== 形态 ==
多年生[[草本|草本]]，有浓烈似[[大蒜|大蒜]]味臭氣；三出[[复叶|复叶]]；花茎粗壮，開黄色小花，复伞形花序；卵形双悬果，侧棱有膜质宽翅。

== 用途 ==
若切断根和根状茎，有乳汁状流出，汁乾後，称“阿魏”。入药，性温、味辛、有毒，功能消積、杀虫。
