'''阿部正乔'''（{{lang|ja|あべ まさたか}}、[[宽文|宽文]]12年4月28日（1672年5月25日） - [[宽延|宽延]]3年7月26日（1750年8月27日）），[[江户时代|江户时代]][[大名|大名]]、[[老中|老中]]。[[武藏国|武藏国]][[忍藩|忍藩]]主。忠秋系阿部家4代。幼名善七郎。[[阿部正武|阿部正武]]的长男。母亲是[[井伊直澄|井伊直澄]]之养女（[[井伊直绳|井伊直绳]]之女）。正室是[[井伊直该|井伊直该]]之女。子女有[[阿部正秋|阿部正秋]]（长男）、[[阿部正乘|阿部正乘]]（次男）、[[阿部正直|阿部正直]]（三男）、[[阿部正敏|阿部正敏]]（五男）、女儿（[[奥平昌成|奥平昌成]]正室）。

== 经历 ==
[[元禄|元禄]]12年（1699年）袭任[[奏者番|奏者番]]、[[寺社奉行|寺社奉行]]。[[宝永|宝永]]元年（1704年）、父亲死后继承家督。[[宝永|宝永]]7年（1711年）就任[[老中|老中]]、在任中为[[正德金银|正德金银]]铸造之总奉行（1714年 - ），之后担任[[德川家继|德川家继]]和[[吉子内亲王|八十宫吉子内亲王]]（[[灵元法皇|灵元法皇]]之皇女）婚姻总奉行（1716年）。[[享保改革|享保改革]]前夕辞任老中。之后30多年专心与藩政。

从正乔这一代开始，忍藩受到了财政危机。[[宽保|宽保]]2年(1742年)8月2日忍藩受到了江户时期最大的洪水的攻击。[[利根川|利根川]]和[[荒川|荒川]]溃决之后，藩领地10万石中的6万石颗粒无收，危机空前，不得已而向幕府借了1万两。

废除长男·正秋后，三男·正直在继承家督前去世了，当时五男·正敏仍年幼。[[宽延|宽延]]元年（1744年）隐居后正乔的弟弟、分家[[旗本|旗本]]·[[阿部正晴|阿部正晴]]的长男[[阿部正允|阿部正允]]继承家督。

== 略历 ==
* 1672年（宽文12年）　出生。
* 1699年（元禄12年）9月28日　任[[奏者番|奏者番]]兼[[寺社奉行|寺社奉行]]。
* 1704年（宝永元年）10月29日　继承忍藩。辞任奏者番、寺社奉行。
* 1711年（宝永7年）4月11日　就任老中。
* 1717年（享保2年）9月19日　辞任老中。
* 1748年（宽延元年）　隐居。
* 1750年（宽延3年）　去世、享年79岁。

== 官职位阶 ==
* 1686年（贞享3年）　从五位下[[出羽国|出羽]]守。
* 1692年（元禄5年）　[[飞驒国|飛騨]]守。
* 1709年（宝永6年）　从四位下[[丰后国|丰后]]守。
* 1710年（宝永7年）　侍从。
