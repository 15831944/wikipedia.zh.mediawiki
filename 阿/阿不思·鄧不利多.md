'''阿不思·博知維·巫服利·布萊恩·鄧不利多'''（{{lang-en|Albus Percival Wulfric Brian Dumbledore}}，1881年7月－1997年6月30日<ref>{{cite web |url=http://www.jkrowling.com/textonly/en/wotm.cfm |title=存档副本 |accessdate=2011-03-20 |0=2009-03-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20090302233052/http://www.jkrowling.com/textonly/en/wotm.cfm |archivedate=2009-03-02 }}</ref>，共116歲），是[[英國|英國]]作家[[喬安·羅琳|喬安·羅琳]]的奇幻小說《[[哈利·波特|哈利·波特]]》系列中的人物，[[霍格華茲魔法與巫術學院|霍格華茲魔法與巫術學院]]校長。

== 概述 ==
鄧不利多被描繪為一位仁慈、风趣、並且法力強大的[[老人|老人]]，類似英國的著名[[巫師|巫師]][[梅林|梅林]]或《[[魔戒|魔戒]]》中的[[甘道夫|甘道夫]]。他似乎知道學校的所有事情，並且不时給[[哈利·波特_(小說角色)|哈利·波特]]一些特殊的帮助。鄧不利多亦是[[佛地魔|佛地魔]]唯一忌憚的人。阿不思名字的来源是拉丁文一詞''{{lang|la|albus}}'' （[[白色|白色]]），象徵「好」；「鄧不利多」在[[近代英語|近代英語]]裡的意思是「[[熊蜂|熊蜂]]」，<ref>{{cite web|url=http://machaut.uchicago.edu/cgi-bin/WEBSTER.sh?WORD=dumbledor|title="Dumbledore" in Merriam–Webster|publisher=Machaut.uchicago.edu|date=|accessdate=2013-08-04|deadurl=yes|archiveurl=https://web.archive.org/web/20151017052743/http://machaut.uchicago.edu/cgi-bin/WEBSTER.sh?WORD=dumbledor|archivedate=2015-10-17}}</ref>灵感是作者想像鄧不利多漫步沿霍格華茲的大廳哼哼唱唱的情景。<ref>{{cite web|url=http://www.accio-quote.org/articles/1999/0399-barnesandnoble.html|title=Barnes and Noble interview, 19 March 1999|date=1999-03-19|accessdate=2013-08-04|publisher=AccioQuote!|last=Rowling|first=J. K.|archiveurl=https://web.archive.org/web/20070228181308/http://www.accio-quote.org/articles/1999/0399-barnesandnoble.html|archivedate=2007-02-28|deadurl=no}}</ref><ref>{{cite web|url=http://www.accio-quote.org/articles/1999/1099-connectiontransc2.htm|title=WBUR radio interview 12 October 1999|publisher=Accio-quote.org|date=1999-10-12|accessdate=2013-08-04}}</ref>

在1899年從霍格華茲畢業不久後，由於失去了能照顧妹妹亞蕊安娜的母親，遂放棄周遊列國的計畫而回到高錐客洞來照顧妹妹，繼而認識了由鄰居芭蒂達·巴沙特介紹的他的姪孫─剛被德姆蘭學院開除的16歲[[蓋勒·葛林戴華德|蓋勒·葛林戴華德]]，才華洋溢且野心甚強的兩人一拍即合，阿不思甚至一度贊成葛林戴華德由巫師統治麻瓜的構想。後因弟弟阿波佛不滿阿不思由於沉迷與葛林戴華德的交往而忽略了照顧妹妹亞蕊安娜，三人爭執中大打出手時導致亞蕊安娜的死亡，妹妹亞蕊安娜死亡的意外使他非常自責，內心非常擔心是自己的咒語殺死妹妹（當時阿不思、阿波佛和葛林戴華德三人混戰，所以到底是誰的咒語殺死亞蕊安娜不得而知），弟弟阿波佛也因亞蕊安娜的關係和哥哥不合，阿波佛在亞蕊安娜的葬禮上將阿不思打斷了鼻子，妹妹的死亡對阿不思往後人生影響甚大，此事件後他學習謙虛，遠離任何野心的慾望，拒絕掌握任何權力（事實上鄧不利多非常多次獲邀請成為魔法部長但皆被他婉拒）。

鄧不利多雖極度謙遜，但依然给人渾身散發魔法能力的感觉。這也許是因为儘管他從沒有表示法力強大過佛地魔，却是[[佛地魔|佛地魔]]唯一忌憚的人。他十分縝密，很少顯示他真實的力量，但在小量場合也会展示他的强大的法力，譬如第四集當[[小巴堤·柯羅奇|巴堤·柯羅奇二世]]告诉哈利他是假的瘋眼，並要殺他時，和第五集在[[魔法部|魔法部]]神秘部門出現時（當時食死人一看到他就馬上害怕的要逃走，旁觀的哈利表示當時鄧不利多渾身散發強者的氣息，他終於了解為何鄧不利多是佛地魔唯一害怕過的人）。

他從不忌讳直接講出佛地魔的名字，從不用「那個不能說出名字的人」或「那個人」等代称（巫師界绝大多数人害怕直接講出佛地魔的名字）。他堅持其他人也应该直接称呼佛地魔。當他面對佛地魔时，他会称他为「湯姆」或「瑞斗」——佛地魔的實際姓名。

他是戰術家和計劃者，但許多巫師沒體會到這一點。在歷史部分提供一些細節，可見他並不是一如其他人這樣想是瘋瘋癲癲的老人。

擁有非常多忠實的追隨者，就算是黑魔王最強盛時，還是有非常多人願意追隨他，主角哈利波特也對鄧不利多極度忠誠。

鄧不利多的嗜好是十柱球戲和室內樂，也十分愛吃。喜歡吃的甜食有檸檬雪寶、蟑螂串、酷酸果、覆盆子果醬、嘶嘶咻咻蜂等。最討厭吃的則是柏蒂全口味豆，因為鄧不利多老是吃到很噁心的口味。

== 現代想法 ==
在《[[哈利波特|哈利波特]]》系列中，阿不思·鄧不利多被刻畫為一位有現代改良派想法的[[巫師_(哈利波特)|巫師]]：

*不同於其他巫師，鄧不利多會訂阅[[麻瓜|麻瓜]][[報紙|報紙]]，了解非魔法的世界。
*他厭惡[[催狂魔|催狂魔]]，不喜歡魔法部讓他們守衛巫師監獄阿茲卡班。他相信一旦[[佛地魔|佛地魔]]回來，[[催狂魔|催狂魔]]會立刻倒向其阵营。後來證明他的猜想是正確的。
*他對非巫師沒有偏見，将半人類和非人類（鄧不利多高尚地称他們為「生命」）與巫師平等视之。他認為，巫師界長期忽略了他們的存在，侵吞他們權利，最终將自食苦果，後來證明他的想法是對的。
**第一位允許[[狼人|狼人]]就讀霍格華茲的校長，为学生时代的[[雷木思·路平|雷木思·路平]]提供安全的地方變身。
**他反對殺害[[巨人|巨人]]。他相信巫師的屠殺已經使巨人面臨絕種。在[[佛地魔|佛地魔]]回歸後建議魔法部長[[康尼留斯·夫子|康尼留斯·夫子]]立刻對巨人伸出友誼之手，以防止他們加入[[佛地魔|佛地魔]]陣營。當部長不贊成這個想法時，他秘密地派遣[[魯霸·海格|魯霸·海格]]和[[歐琳·美心|歐琳·美心]]夫人為信使与巨人联络，可惜失败。
**他任命[[魯霸·海格|魯霸·海格]]（混血巨人）、雷木思·路平（狼人）、[[賽佛勒斯·石內卜|賽佛勒斯·石內卜]]（[[食死人|食死人]]）、[[阿各·飛七|阿各·飛七]]（[[哑炮|哑炮]]）、[[丙斯|丙斯]]（[[幽靈|幽靈]]）和[[翡冷翠|翡冷翠]]（[[人馬|人馬]]）等人为[[霍格華茲|霍格華茲]]教員。
**家庭小精靈[[多比|多比]]獲得自由卻在巫师家庭找不到工作，到[[霍格華茲|霍格華茲]]找鄧不利多請求雇用。他不但同意給他就業福利，像薪金和假日，還允許他可以用他喜歡的方式稱呼他。他也希望[[天狼星·布萊克|天狼星·布萊克]]對待他的家庭小精靈[[怪角|怪角]]好一點。
**他不迷信關於[[夜騏|騎士墜鬼馬]]的不吉利傳言，不想用[[消影|現影术]]和[[掃帚|掃帚]]飛行時，會使用騎士墜鬼馬旅行。並且允許牠們拉學生馬車進入[[霍格華茲|霍格華茲]]。
*鄧不利多對純血或混血或麻瓜出生的巫师都一視同仁，認為他們的未來不是由血統決定，而是由選擇決定。
*鄧不利多相信人性好的一面，对迷途知返的人总是能不计前嫌，重新信任。

== 物品 ==
*一隻[[不死鳥|鳳凰]]，名叫[[佛客使|佛客使]]，能如同家庭小精靈一樣不受城堡的防護措施影響自由消影。（小說中翻譯為鳳凰，但實際上是西洋神話中的不死鳥，和東方的[[鳳凰|鳳凰]]是有差異的。）
*一把[[熄燈器|熄燈器]]，能熄滅麻瓜街道的燈火，或依據當事人的想法引導現影到正確的地點，死後由[[榮恩·衛斯理|榮恩·衛斯理]]繼承。
*一隻[[懷錶|懷錶]]，有用12隻指針和以行星代替數字。
*[[儲思盆|儲思盆]]，用來儲存回憶。
*各種各樣的儀器，用途未知。
*[[意若思鏡|意若思鏡]]，面對它會看見內心深處最渴望發生的事（哈利看見父母在他身旁，榮恩看見自己成為級長、男學生主席和魁地奇球隊隊長、鄧不利多看到家人，但中年時代看到的是葛林戴華德）。
*[[分类帽|分类帽]]，將新生分到各學院，在學校出現危機時，會給予建議。
*[[高錐客·葛來分多|高錐客·葛來分多]]的寶劍，上面鑲有紅寶石，還有葛來分多的名字，鄧不利多死後將其留給哈利、妙麗和榮恩等三人，但魔法部宣稱這把劍可以由任何葛來分多學生取得，因此屬於學校，不過繼任校長的石內卜以一把假劍混淆視聽，於哈利三人在外旅行時利用自己的護法引領哈利找到真正的寶劍。
*[[金探子|金探子]]，哈利第一場魁地奇球賽抓到的金探子，後來被鄧不利多用來藏其中一樣[[死神聖物|死神聖物]]——[[重生石|重生石]]，死後繼承給哈利。
*[[接骨木魔杖|接骨木魔杖]]，[[死神聖物|死神聖物]]之一，又被稱為死神魔杖、命運魔杖，被譽為最強的魔杖，是鄧不利多戰勝黑巫師葛林戴華德後繼承其所有權，鄧不利多死前在違反意志的狀況下被跩哥·馬份繳械，意外轉移了魔杖主人的資格，之後學校老師們將魔杖跟鄧不利多一起埋葬，其後佛地魔破開其棺材取得魔杖，但哈利已對跩哥繳械而成為魔杖主人，因此非魔杖主人的佛地魔在與哈利對決時輸掉，书籍中最後哈利把它埋回鄧不利多的墳墓，而电影中则被哈利折断并丢入山谷中，以免因而产生的争夺再次发生。
*[[吟遊詩人皮陀故事集|吟遊詩人皮陀故事集]]原始版本：一本老舊故事書，收錄魔法界最古老的童話，包含《三兄弟的故事》、《幸運泉》、《魔法師的毛茸茸心臟》、《巫師與跳跳鍋》以及《巴比兔迪迪和咯咯笑樹樁》，死後由[[妙麗·格蘭傑|妙麗·格蘭傑]]繼承。

== 家庭 ==
*父親：[[博知維·鄧不利多|博知維·鄧不利多]]，因為替[[亞蕊安娜·鄧不利多|亞蕊安娜·鄧不利多]]報復而攻擊三名麻瓜而被送往[[阿茲卡班|阿茲卡班]]，最終在阿茲卡班去世。
*母親：-{zh-hans:坎德拉·邓布利多; zh-hant:[[甘德拉·鄧不利多|甘德拉·鄧不利多]];}-，[[麻瓜|麻瓜]]出身，在丈夫被送往阿茲卡班後舉家遷往[[戈德里克山谷|戈德里克山谷]]，在鄧不利多17歲時，被失控的亞蕊安娜誤殺，阿不思及阿波佛對外宣稱是死於咒語逆火反彈。
*弟弟：[[阿波佛·鄧不利多|阿波佛·鄧不利多]]，[[豬頭酒吧|豬頭酒吧]]的老板，曾經被控告對山羊使用不正當魔法。與哥哥的感情不太好，因為認為妹妹的死是他的錯，更在其葬禮上打斷了他的鼻子。第七集參與[[霍格華茲|霍格華茲]]保衛戰。
*妹妹：-{zh-hans:亞蕊安娜·鄧不利多; zh-hant:[[亞蕊安娜·鄧不利多|亞蕊安娜·鄧不利多]];}-，6歲時被三名麻瓜發現她的魔法能力，更逼她表演，但卻因未能控制力量而達不到麻瓜的要求，遂對她施以暴力，自此失控，大部分時間都沒法使用魔法，卻不定期會爆發，以魔法做出傷人害己的事，其後更在一次發狂時誤殺母親。在家中與阿波佛最為親近。在阿不思17歲時，阿不思、阿波佛和[[蓋瑞·葛林戴華德|葛林戴華德]]的一次決鬥中，受不了決鬥的情景而再次失控，卻被決鬥中的魔咒擊中而死。其肖像悬挂于猪头酒吧通往霍格沃茨有求必应屋的秘密通道入口处。

== 生平 ==
[[File:Albus Dumbledore.jpg|缩略图|320x320像素|
插圖中的鄧不利多
]]
阿不思·鄧不利多於1881年出生於-{zh-hans:沃土原; zh-hant:軟泥山丘;}-（Mould-on-the-Wold ）。

1892年11歲就讀霍格華茲，屬葛來分多學院，完成學業以後成為教職員，教授變形學，同时兼职黑魔法防御术学。

1943年[[湯姆·瑞斗|湯姆·瑞斗]]指責了[[鲁霸·海格|魯霸·海格]]打開秘室，只有鄧不利多認為[[魯霸·海格|海格]]是無辜的，並且當海格的魔杖被折斷，逐出了學校，鄧不利多堅持让他留校為員工。多年後接替[[阿曼多·狄劈|阿曼多·狄劈]]擔任校長。

1945年擊敗「史上最危險的黑巫師」名單排名第一的超強黑巫師[[蓋瑞·葛林戴華德|蓋瑞·葛林戴華德]]，從他手上獲得[[死神聖物|死神聖物]]之一——接骨木魔杖。

一名[[狼人|狼人]][[雷木思·路平|雷木思·路平]]就讀霍格華茲。為他種植渾拚柳守衛一條通往活米村的尖叫屋秘密通道。路平在每個滿月在那裡變身，使他不能攻擊其他人。

[[西碧·崔老妮|西碧·崔老妮]]申請成為新占卜學老師，[[西碧·崔老妮|西碧·崔老妮]]說出了真正的預言，指稱佛地魔與哈利波特無法同存於世。然而說完預言之後崔老妮卻不復記憶。鄧不利多聘用她的原因除了避免完整預言走漏消息，也保護狀況外的她的生命安全。

在[[石內卜教授|石內卜]]的通風報信下保護波特家，因而發現[[死神聖物|死神的聖物]]之一——隱形斗蓬，並借去研究。

當[[莉莉·波特|莉莉·波特]]和[[詹姆·波特|詹姆·波特]]被佛地魔殺害後，鄧不利多決定安置成為孤兒的哈利在他姨丈[[威農·德思禮|威農·德思禮]]和阿姨[[佩妮·德思禮|佩妮·德思禮]]家中。這是因為鄧不利多安置了一個咒語使到他母親的犧牲成為孩子之特別保護。只要哈利在他的母親親屬居住的地方依然稱作家，母親的血統就成為他的保護，他會安全地活著直到滿十七歲。

[[哈利波特－混血王子的背叛_(電影)|第六集]]開始時發現分靈體之二——剛特的指環（第二集時發現了第一件分靈體；日記），上頭有第三樣死神聖物——重生石。他想用重生石呼喚死去的家人，但卻中了佛地魔施在指環上的詛咒，虛弱地回到校長辦公室，由石內卜醫治，把詛咒暫時封鎖在焦黑的右手上，但黑魔法的詛咒威力太強，石內卜推測他只剩一年壽命，之後用葛萊分多寶劍破壞分靈體。隨後的一年內親自指導哈利波特，帶他探究佛地魔的過去。

1997年6月30日，[[賽佛勒斯·石內卜|賽佛勒斯·石內卜]]在鄧不利多和哈利從洞窟回來時於天文塔向他發射索命咒，墜於塔底逝世（他們事先策畫好，鄧不利多主動提出要石內卜殺了他）。

1997年7月初，魔法世界為鄧不利多舉行盛大葬禮，葬於大湖邊的白色陵墓（White Tomb）。校長之位由[[麥教授|麥教授]]暫任，[[佛地魔|佛地魔]]控制魔法部後命令[[石內卜教授|石內卜]]接任。

在故事结尾，哈利波特把夺还的接骨木魔杖折斷丟入河谷(電影橋段)。

== 『魔咒』魔法技能〖符咒學〗 ==
⇒ [[減震止速|減震止速]](Arresto Momentum) ：是一個[[魔咒|魔咒]]的咒語，[[巫師|巫師]]可以使用它減慢物體的下落速度。這個咒語可能由兩個單片語成：前者來自諾曼時代英國所用的法語單詞「arester」，意為「使停止」，後者來自拉丁語「Momentum」，意為「推動力」。

追蹤手每次丟了球，都會更加惱怒，因為他們必須不斷地沖向地面找回快浮。於是，賽後快浮被換成了紅色。換了顏色不久，[[巫師|巫師]][[戴西·彭尼德|戴西·彭尼德]]又想出了一個主意，給快浮施上一種魔咒，如果它沒被接住，它會慢慢地落向地面，就好像在水裡下沉一樣，追蹤手可以在半空中抓住它。
