'''阿勒泰地区'''是[[中华人民共和国|中华人民共和国]][[新疆维吾尔自治区|新疆维吾尔自治区]][[伊犁哈萨克自治州|伊犁哈萨克自治州]]下辖的[[地区_(中国行政区划)|地区]]。总面积11.78万平方公里，占全疆总面积的7%，总人口67.16万<ref name="altgk">{{cite web |title=阿勒泰概况 |url=http://www.xjalt.gov.cn/001/001001/001001001/20180404/3921ba11-b392-47da-ac57-712db523b4fc.html |website=xjalt.gov.cn |publisher=新疆阿勒泰地区行政公署 |accessdate=2018-07-27}}</ref>。该地区[[行政公署|行政公署]]驻地在[[阿勒泰市|阿勒泰市]]。阿勒泰地区以[[哈薩克族|哈薩克族]]為主體民族，[[漢族|漢族]]人口也佔有一定比例。

== 建置沿革 ==
{{main|阿爾泰區域|阿山道}}

阿勒泰為“阿爾泰”之異譯。[[清代|清代]]阿爾泰地區為蒙古[[阿爾泰烏梁海|阿爾泰烏梁海]]等部落遊牧之地。[[乾隆|乾隆]]以後為[[科布多參贊大臣|科布多參贊大臣]]轄地。[[同治|同治]]七年（1868年）於[[布倫托海|布倫托海]]（今福海縣一帶）置布倫托海辦事大臣，次年廢。[[光緒|光緒]]三十二年（1906年），分[[科布多|科布多]]置[[阿爾泰區域|阿爾泰]]，以[[錫恆|錫恆]]為[[阿爾泰辦事大臣|阿爾泰辦事大臣]]，轄蒙古阿爾泰烏梁海、新土爾扈特部、新和碩特部，駐[[承化寺|承化寺]]（今[[阿勒泰市|阿勒泰市]]城區）。

民國以後，阿爾泰又稱[[阿爾泰區域|阿爾泰區域]]，即今阿勒泰地区之前身。1919年，阿爾泰改為[[阿山道|阿山道]]，併入[[新疆省|新疆省]]。其後又改為阿山行政區、阿山專區。1954年，成立阿勒泰專區，1970年改稱阿勒泰地区。

== 地理 ==
阿勒泰地区位于新疆北部，是一个由哈、汉等多民族聚居的边境地区，“阿勒泰”为当地少数民族语，系金山之意。西北与[[哈萨克斯坦|哈萨克斯坦]]，[[俄罗斯联邦|俄罗斯联邦]][[西伯利亞联邦管区|西伯利亞联邦管区]][[阿尔泰共和国|阿尔泰共和国]]相连，东部与[[蒙古国|蒙古国]]接壤，边境线长1205公里。

=== 民族 ===
阿勒泰地区是一个多民族聚居的地方，有[[哈萨克族|哈萨克族]]、[[汉族|汉族]]、[[回族|回族]]、[[维吾尔族|维吾尔族]]、[[蒙古族|蒙古族]]（[[圖瓦人|圖瓦人]]）等36个民族。[[汉族|汉族]]人口232563人，占总人口的38.55%；各[[少数民族|少数民族]]人口370720人，占总人口的61.45%；其中[[哈萨克族|哈萨克族]]人口328153人，占总人口的54.39%，[[回族|回族]]人口23656人，占总人口的3.92%，[[维吾尔族|维吾尔族]]人口8703人，占总人口的1.44%，[[蒙古族|蒙古族]]人口5376人，占总人口的0.89%。

== 经济 ==
[[File:ColourfulBeach.jpg|right]]
2010年全地区实现[[国内生产总值|国内生产总值]]84.85亿元（现价），工农业总产值38.39亿元，农牧民人均纯收入2361元。经过发展，该区[[粮食|粮食]]自给有余，[[畜牧业|畜牧业]]大幅度的提高。电力、煤炭、[[黄金|黄金]]、采矿、毛纺、造纸、制糖、服装、食品、医药、珠宝加工等工业已具有相当规模。

地区境内先后开放了[[塔克什肯口岸|塔克什肯口岸]]、[[红山嘴口岸|红山嘴口岸]]、[[吉木乃口岸|吉木乃口岸]]、[[阿黑吐拜克口岸|阿黑吐拜克口岸]]。2010年全地区对外贸易进出口额达9163万美元。
