'''阿姓'''是一個罕見的漢姓，但分布頗為廣泛。

== 讀音 ==
[[普通話|普通話]]讀ē。[[南京官話|南京官話]]讀o<sup>1</sup>。

== 源流 ==
阿姓來源有三：
# [[商朝|商朝]]大臣[[伊尹|伊尹]]所担任的官名阿衡（保護教養的官），其後代有一支系以阿為姓；
# [[北魏|北魏]]時期[[阿單|阿單]]氏、[[阿賀|阿賀]]氏、[[阿伏干|阿伏干]]氏改姓阿。
# [[元朝|元朝]]時期在[[雲南|雲南]]來了的[[契丹人|契丹人]]把契丹名字换成中名字。[[耶律阿保機|耶律阿保機]]的記憶記得，讓他們選舉阿姓。
