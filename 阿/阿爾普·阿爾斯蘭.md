'''阿爾普·阿爾斯蘭'''（[[土耳其语|现代土耳其语]]：Alp Arslan；{{lang-fa|آلپ ارسلان}}；{{bd|1029年|1月20日|1072年|12月15日}}），本名'''穆罕默德·本·達伍德·恰格勒'''或'''穆罕默德·本·贾格里'''，[[塞爾柱帝國|塞爾柱帝國]]第二任[[蘇丹_(稱謂)|蘇丹]]，首任蘇丹[[圖赫里勒·貝格|圖赫里勒·貝格]]之侄，一族英雄[[塞爾柱·貝格|塞爾柱·貝格]]的曾孫
<ref name="Rome"> {{Cite book | title = 羅馬帝國衰亡史 | author = Gibbon, Edward（愛德華·吉朋）| others = 席代岳譯 | location = 臺灣 | publisher = 聯經出版公司 | date = 2004 | ISBN = 957-08-2771-8 }} </ref>，任內屢次與[[埃及|埃及]]的[[法蒂瑪王朝|法蒂瑪王朝]]、[[東羅馬帝國|東羅馬帝國]]交戰取勝，使帝國得以擴張版圖；最具代表性的是1071年在[[曼齊刻爾特|曼齊刻爾特]]（今[[土耳其|土耳其]]境內）以少勝多擊敗[[東羅馬帝國|東羅馬帝國]]遠征軍，此役奠定大塞爾柱王朝在[[安納托利亞|安納托利亞]]的統治，土耳其後人敬佩他的軍事才能和非凡武藝，尊稱他為“阿爾普·阿爾斯蘭”，意為“英勇之獅”。
<ref>"Alp Arslan." Encyclopedia of World Biography. 2004. Encyclopedia.com. (September 5, 2011). http://www.encyclopedia.com/doc/1G2-3404700163.html</ref>

施政方面，由於自身對行政事務不感興趣，他將國事悉數交付名相[[尼札姆·穆勒克|尼札姆·穆勒克]]管理，對商業則採取開放態度。

== 早年經歷 ==
青年時的阿爾普·阿爾斯蘭隨叔叔[[圖赫里勒·貝格|圖赫里勒·貝格]]蘇丹駐紮在帝國西部，協助對抗[[埃及|埃及]][[法蒂瑪王朝|法蒂瑪王朝]](綠衣大食)的統治者[[穆斯坦綏爾_(法蒂瑪王朝)|穆斯坦綏爾]]；由於教派衝突，[[遜尼派|遜尼派]]的塞爾柱帝國與[[什葉派|什葉派]]的法蒂瑪[[哈里發|哈里發]]關係惡劣、時有紛爭。他的父親[[恰格勒·貝格|恰格勒·貝格]]則負責管理帝國東部的[[大呼羅珊|呼羅珊]]地區。

在西部戰端稍歇之後，阿爾斯蘭回到大呼羅珊，父親安排他從事管理工作，並為他引薦[[尼札姆·穆勒克|尼札姆·穆勒克]]，後者頗受賞識，在阿爾斯蘭即蘇丹位時被重用為「[[维齐尔|维齐尔]]」（即「宰相」或「首席大臣」），掌管帝國一切行政。<ref name="Rome" />相傳尼札姆·穆勒克與伊斯蘭名士[[欧玛尔·海亚姆|欧玛尔·海亚姆]]、[[哈桑·萨巴赫|哈桑·萨巴赫]]為求學時代密友，三人彼此交情匪淺。

1059年，30歲的阿爾斯蘭接替去世的父親擔任呼羅珊總督。

== 即位與征伐 ==
1063年，首任蘇丹去世，貴族們決定由阿爾斯蘭的弟弟蘇萊曼·恰格勒繼任蘇丹，對此不滿的阿爾斯蘭，與同樣覬覦蘇丹大位的叔叔[[庫塔爾米什|庫塔爾米什]]爆發了[[達姆甘之戰|達姆甘之戰]]，阿爾斯蘭派獲得勝利，並赦免了被俘的庫塔爾米什之子{{tsl|en|Suleiman ibn Qutulmish|蘇萊曼·庫塔爾米什}}，據蘇萊曼回憶，阿爾斯蘭在戰後看到叔叔的屍體時難過的落淚。

1064年4月27日，阿爾斯蘭即位為大塞爾柱蘇丹，靠著尼札姆·穆爾克的輔佐，弭平了塞爾柱統治階層的派系糾紛，坐擁[[阿姆河|阿姆河]]至[[底格里斯河|底格里斯河]]的廣大南[[波斯|波斯]]領土，維持了帝國的穩定與和平。上任當年，阿爾斯蘭就覬覦[[土耳其|土耳其]][[聖巴西流|聖巴西流]]教堂的大筆財產，自命為土耳其騎兵團長，揮軍渡過[[幼發拉底河|幼發拉底河]]，入侵[[東羅馬帝國|東羅馬帝國]]治下的土耳其地區、[[卡帕多西亞|卡帕多西亞]]省的首府[[開塞利|開塞利]]。<ref name="Rome" />

同年底，他與宰相尼札姆·穆勒克一同進軍同屬東羅馬的[[亞美尼亞|亞美尼亞]]和[[喬治亞|喬治亞]]省，在25天的圍攻後，塞爾柱軍攻下亞美尼亞首都[[阿尼_(土耳其)|阿尼]]，旋即對市民展開屠殺，當時阿拉伯歷史學者{{tsl|en|Sibṭ ibn al-Jawzi|西必特·艾本紐·喬西}}親眼見證了蘇丹軍隊破城後的慘況：「（賽爾柱）軍隊屠殺居民，燒殺虜掠，所有倖存者淪為階下囚，人數至少有5萬......聽說走到哪都會不小心踩到屍體，街道為之堵塞。我決心進城親眼看看城中傳聞是否為真？然而，當我發現連半條沒有屍體的街道都不存在時，不得不承認答案是如此悲慘。」

1066年，阿爾斯蘭召集各地領主舉辦大會，席間宣布他的兒子[[馬利克沙一世|馬利克沙一世]]為下任蘇丹繼承人。隔年，塞爾柱帝國佔領亞美尼亞全境。

== 東羅馬帝國反擊 ==
[[File:Byzantium_vs_Seljuk_c_1071.png|400px]]任內與塞爾柱的帝國的作戰圖，紅色為拜占庭（東羅馬）攻勢，綠色為塞爾柱攻勢，紫區為東羅馬領土。]]
1068年，[[東羅馬帝國|東羅馬帝國]]將軍[[羅曼努斯四世|羅曼努斯四世]]強娶先皇遺孀登基。他斷然實施軍事改革，兩度親征塞爾柱帝國。由於阿爾斯蘭不重視小亞細亞地區和連勝帶來的驕縱，他放任當地當地駐軍任意劫掠，各部族如同一盤散沙。羅曼努斯四世抓住機會將他們分個擊破，收復部分失土並奪得了[[阿勒頗|阿勒頗]]附近的一座城市。同時，塞爾柱帝國已持續一年的對伊康（今[[科尼亞|科尼亞]]）攻勢也被擊退，西方邊界一度退至幼發拉底河。<ref name="Rome" />由於和埃及法蒂瑪王朝的戰事仍未歇，在土耳其一系列戰事失利後，蘇丹決定與東羅馬[[拜占庭|拜占庭]]方面和談；1069年兩軍簽下和約，塞爾柱帝國避免了腹背受敵的狀況。

1070年東羅馬帝國第三度發動遠征，此次羅曼努斯並未親陣。此時阿勒頗的統治者{{tsl|en|Rashid al-Dawla Mahmud|拉希德·馬哈茂德}}為了獲取阿爾斯蘭的支援，向蘇丹宣誓效忠，獲得了塞爾柱軍隊的幫助。東羅馬遠征軍戰敗、崩潰，指揮官曼努埃爾·科穆寧（已故皇帝[[伊薩克一世|伊薩克一世]]的姪子）亦被俘。

1071年2月，羅曼努斯派遣大使至塞爾柱更新和約，當時阿爾斯蘭正在圍攻[[埃德薩|埃德薩]]城；東羅馬大使到達後，他答應續約並立即解除包圍，向南移師對抗法蒂瑪軍。然而，羅曼努斯此次換約乃是希望分散蘇丹在北方的注意力，趁塞爾柱人反應之前收復亞美尼亞。

== 曼齊刻爾特之戰 ==
{{Main|曼齐刻尔特战役}}
[[File:BnF_Fr232_fol323_Alp_Arslan_Romanus.jpg|240px]]。]]
1071年8月，42歲的阿爾普·阿爾斯蘭在阿勒頗接獲東羅馬第四度遠征的消息，此次羅曼努斯四世再次親征。除東羅馬軍外，他還募集了來自[[法蘭克|法蘭克]]、[[諾曼|諾曼]]、[[欽察|庫曼]]等地的傭兵，集結了至少4萬的軍勢攻向亞美尼亞。沒想到東羅馬會再度進軍的阿爾斯蘭倉促回師，兩軍在[[曼齊刻爾特|曼齊刻爾特]]（又譯馬拉茲吉爾特，今土耳其境內）相遇，隔著[[穆拉特河|穆拉特河]]對陣。據說為提高士氣，蘇丹甚至穿上了壽衣以示戰鬥致死的決心<ref name="CB"> {{Cite book | title = 劍橋插圖伊斯蘭世界史 | author =  Francis Robinson（ 法蘭西斯·魯賓遜 ）| others = 黃中憲譯 | location = 臺灣 | publisher = 如果出版社 | date = 2008 | ISBN = 9789866702198 }} </ref>

相對於東羅馬帝國軍，塞爾柱軍只有兩萬，人數約為對方一半，阿爾斯蘭擺出鶴翼陣{{來源請求}}，計畫以擅長的[[安息回馬箭|帕提亞射法]]迎敵。數量佔優的東羅馬軍在渡河後很快便占領了塞爾柱的中央軍陣地，然而阿爾斯蘭下令中央軍後退時讓兩翼馬弓手不斷放箭，將以步槍兵為主、擅長肉搏的東羅馬傭兵釘在河岸避免近身戰。東羅馬前鋒在傷亡不斷增加的情況下，又發生庫曼傭兵倒戈的事情，其他傭兵團見此紛紛退出戰場。在先鋒隊陷入混亂後羅曼努斯四世親率中軍渡河，人數劣勢使得塞爾柱軍陷入苦戰。

給予東羅馬軍致命重創的是羅曼努斯的繼子{{tsl|en|Andronikos_Doukas_(cousin_of_Michael_VII)|安德洛尼卡·道卡斯}}，在羅曼努斯渡河後，他逕自宣布皇帝已戰死並帶領後衛隊撤離，這一舉動使東羅馬軍的人數優勢不復存在，羅曼努斯四世戰敗被俘。<ref name="Rome" />

出乎他意料，在受俘儀式結束後，他被蘇丹比照國王規格慷慨接待，阿爾斯蘭與驚奇的皇帝間有過以下對話：

{{cquote|''阿爾普·阿爾斯蘭：「如果我是你的階下囚，你會做什麼？」''

''羅曼努斯四世：「也許我會殺了你，或讓你在[[君士坦丁堡|君士坦丁堡]]遊街示眾。」''

''阿爾普·阿爾斯蘭：「看來我的懲罰遠遠重於你的，我不僅僅原諒你，更賜予你自由。」''<ref name="Rome" />}}

阿爾斯蘭的主要目標仍是南邊的法蒂瑪王朝，在對東羅馬的談判中，他索求贖金以及承認塞爾柱對[[安提拉|安提拉]]、[[希拉波利斯|希拉波利斯]]、曼齊刻爾特等城的統治權。在和平條款簽署後，蘇丹將皇帝交還東羅馬，由警衛恭恭敬敬地護送出城，讓他滿載禮物而歸。然而羅曼努斯回國後被新皇帝挖去雙眼流放，於翌年去世。

同年底，塞爾柱軍擊敗了宿敵[[哈里發|哈里發]][[穆斯坦綏爾_(法蒂瑪王朝)|穆斯坦綏爾]]，奪下[[耶路撒冷|耶路撒冷]]並虐殺戰俘，帝國一系列軍事勝利成為後來基督教世界的[[十字軍東征|十字軍東征]]遠因。

戰後，阿爾斯蘭委以表兄蘇萊曼·庫塔爾米什管轄[[土耳其|土耳其]]地區，安定該地並使他遠離政治核心。在阿爾斯蘭死後，蘇萊曼持續與東羅馬帝國周旋，最終幾乎支配全[[安那托利亞|安那托利亞]]半島。蘇萊曼於1077年脫離塞爾柱自立為蘇丹，開創[[魯姆蘇丹國|魯姆蘇丹國]]。<ref name="CB" />

== 蘇丹之死 ==
擊敗東羅馬帝國的隔年，阿爾斯蘭不顧尼札姆和兒子馬利克沙的反對，出兵[[土耳其斯坦|土耳其斯坦]](今烏茲別克、哈薩克境內)攻打[[喀喇汗國|喀喇汗國]]，塞爾柱軍很快就前進到了[[阿姆河|阿姆河]]畔。

1072年12月11日，在渡河前，阿斯蘭接見一位拒降的[[花剌子模|花剌子模]]領主尤素夫(Yussuf al-Kharezmi)，下令將其處決，絕望的領主拔出匕首衝向蘇丹；身為一位名射手，阿爾斯蘭示意衛兵退下，親自彎弓搭箭迎敵，然而一個重心不穩導致箭矢僅僅是擦過尤素夫，阿爾斯蘭被衝入懷中的刺客以匕首重傷，於四天後逝世，享年43歲。<ref name="Rome" />遺體被運往[[梅爾夫|梅爾夫]]，與他父親合葬。

臨死前，阿爾斯蘭自嘲：「年輕時我學到『驕兵必敗』，看來30多年來我把這句話忘得差不多了。」<ref name="Rome" />

== 後世影響 ==
由於曼齊刻爾特一役將東羅馬帝國趕出安那托利亞半島，阿爾斯蘭常被視作土耳其人獨立的象徵，廣受安那托利亞主義(Anatolianism，即{{tsl|en|Turkish nationalism|土耳其國家主義}})者尊崇。

在2002－2008年的[[土库曼斯坦|土库曼斯坦]]曆法改革期間，一度以阿爾普·阿爾斯蘭之名命名八月份。
