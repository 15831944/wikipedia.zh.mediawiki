== 1962年5月12日 ==
* 1962年5月12日，三位卡车司机，Valention tomassini，Gauro Tomassini和Humberto Zenobi驾车从[[布兰卡港|布兰卡港]]经由35号公路前往{{le|Jacinto Aráuz|Jacinto Aráuz}}。在4点10分，他们看见公路旁约100米外的空地上有一物体，这物体看起来像一辆有轨电车，被照亮着。当他们的卡车靠近，该物体便突然上升并在四米高处穿过公路。它发着光离开，其底部出现微红的火焰。最后，它分裂成两部分，向不同方向飞离。<ref>[http://www.ignaciodarnaude.com/ufologia/Uriondo,UFOs%20Type-1,1950-1965,Argentina-1,FSR-CH%201972%20N%2012.pdf 阿根廷第一类接触基础目录]</ref><ref>[http://www.ufoinfo.com/magonia/part6.shtml 从1958年二月到1962年十一月的Magonia数据库]</ref>

== 1965年12月1日 ==
*  这一目击事件被称作弧矢七天文台UFO凌月。它发生在1965年的12月1日晚上8点30，在[[布宜诺斯艾利斯省|布宜诺斯艾利斯省]]，[[圣米格尔|圣米格尔]]的私立弧矢七天文台，{{citation needed|date=January 2016}}科学家们收到数个关于在月球上观测到可疑物体的电话。天文台的职员拍下数张固定间隔的月球照片，冲洗后，一些照片显示出一些飞越月球的碟状物体。{{citation needed|date=January 2016}}有些专家认为照片上的不明飞行物可能是胶卷上的缺陷。<ref>[http://www.ikaros.org.es/fotocat/argentina1965en.pdf 阿根廷, 相片中的1965] {{webarchive|url=https://web.archive.org/web/20120529070328/http://www.ikaros.org.es/fotocat/argentina1965en.pdf |date=2012-05-29 }}</ref>

== 1973年10月27日 ==
* 10月30日卡车司机Dionisio Llanca被送入[[布兰卡港|布兰卡港]]的Municipal医院。他称10月27晚，他离开位于布兰卡港的叔叔家前往[[里奥加耶戈斯|里奥加耶戈斯]]。当他停下换胎时他被一束强光所瘫痪，据Llanca说，他看见一个物体漂浮在空中，他描述为两个男性和一个女性的生物在他身上做了一些医学测试。Llanca说他数小时后在约9公里外短暂苏醒，一名叫Fabio Zerpa的UFO研究者对此事件进行了研究并随他的著作''El Reino Subterráneo''公开。<ref name="BartholomewHoward1998">{{cite book|author1=Robert E. == 1988年12月26日 ==
* 一个银色不明飞行物飞越{{le|Villa Urquiza|Villa Urquiza}}上方，一个[[布宜诺斯艾利斯|布宜诺斯艾利斯]]的人口密集区域。它被认为是阿根廷最为人知的不明飞行物事件，目擊者超过7500人。当地机场报告称一个物体在西侧飞行，在那个晴朗的下午，它在[[雷达|雷达]]上看起来向着General Paz高速路前进。<ref>[http://www.8sferas.com/index.php/fenomeno-ovni Avistamientos importantes de ovnis en el Mundo] {{webarchive|url=https://web.archive.org/web/20130612214442/http://www.8sferas.com/index.php/fenomeno-ovni |date=2013-06-12 }} {{es}}</ref>
