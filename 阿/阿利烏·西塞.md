'''阿利奧·西塞'''（{{lang-fr|Aliou Cissé}}，{{bd|1976年|3月24日}}），[[塞內加爾|塞內加爾]]前職業足球員與教練，司職[[中場#後腰|防守中場]]或[[後衛_(足球)#中後衛|中後衛]]，他在法國開始其職業生涯，後來效力英國的[[伯明翰足球會|伯明翰城]]和[[朴茨茅斯足球俱乐部|朴茨茅斯]]。他目前為[[塞內加爾國家足球隊|塞內加爾]]的主教練，於2012年接替{{link-en|阿瑪拉·特拉奧雷|Amara Traoré}}後一直執教至今，並帶領球隊打入[[2018年國際足協世界盃|2018年國際足協世界盃]]，同時亦是該屆世界盃中最年輕的主帥和唯一一位黑人教練<ref>{{cite web |title=【世界盃】慶祝動作紅爆全球　塞內加爾夏韶聲奏起熱血之歌 |url=https://wc2018.hk01.com/article/201200/世界盃-慶祝動作紅爆全球-塞內加爾夏韶聲奏起熱血之歌 |website=[[香港01|香港01]] |date=2018-06-20 |accessdate=2018-07-03 }}</ref>。

== 俱樂部生涯 ==
西塞於1976年3月24日出生於[[塞內加爾|塞內加爾]][[濟金紹爾|濟金紹爾]]，他九歲時搬到巴黎，以達成其成為[[巴黎聖日耳門|巴黎聖日耳門]]球員的夢想<ref>{{cite web |title=Cissé Aliou |url=http://www.paris-canalhistorique.com/cisse-aliou/ |website=Paris.canal-historique |accessdate=2018-06-24 |archiveurl=https://web.archive.org/web/20180624222934/http://www.paris-canalhistorique.com/cisse-aliou/ |archivedate=2018-06-24 |language=fr |date=2013-02-13}}</ref>。他在[[里爾奧林匹克體育會|里爾]]開始其職業生涯，之後分別效力[[色當阿登體育俱樂部|色當]]和巴黎聖日耳門。西塞還曾在2001–02賽季的大部分時間被租借至[[蒙彼利埃體育會|蒙彼利埃]]<ref>{{cite web |title=Aliou Cisse Histoire |url=http://www.mhscfoot.com/joueur/aliou-cisse/histoire |website=mhscfoot.com |accessdate=2018-06-25 |archiveurl=https://web.archive.org/web/20160407200444/http://www.mhscfoot.com/joueur/aliou-cisse/histoire |archivedate=2016-04-07 |language=fr}}</ref>。

[[2002年國際足協世界盃|2002年世界盃]]上，西塞以隊長的身份帶領[[塞內加爾國家足球隊|塞內加爾]]獲得八強的佳績，後來西塞轉會至首戰[[英超|英超]]的[[伯明翰足球會|伯明翰城]]<ref>{{cite web |title=Aliou Cisse signs for Birmingham |url=http://zeenews.india.com/home/aliou-cisse-signs-for-birmingham_48728.html |website=Zee News |publisher=Zee Media Corporation Limited |accessdate=2018-06-24 |archiveurl=https://web.archive.org/web/20180624224514/http://zeenews.india.com/home/aliou-cisse-signs-for-birmingham_48728.html |archivedate=2018-06-24 |date=2002-07-10}}</ref><ref>{{cite web|last1=Ireland|first1=Shane|title=Aliou Cissue: The former Birmingham City midfielder now managing Senegal at World Cup 2018|url=https://www.birminghammail.co.uk/sport/football/football-news/cisse-senegal-blues-world-cup-12795402|website=Birmingham Mail|publisher=Reach plc|accessdate=2018-06-22|date=2018-06-20}}</ref>，並在開賽日對上[[兵工廠|兵工廠]]的比賽中亮相，但被罰下場。儘管停賽被撤銷<ref>{{cite web |title=Cisse red card rescinded|url=http://news.bbc.co.uk/sport2/hi/football/eng_prem/2204613.stm|website=[[BBC_Sport|BBC Sport]]|publisher=[[BBC|BBC]]|accessdate=2018-06-22|archiveurl=https://web.archive.org/web/20020823171049/http://news.bbc.co.uk/sport2/hi/football/eng_prem/2204613.stm|archivedate=2002-08-23|date=2002-08-20}}</ref>，他還是在六場比賽中獲得五張黃牌，最終在新年前累積十張黃牌，然而其二月之後的比賽卻因為受傷的關係而報銷<ref>{{cite web |last1=Ireland |first1=Shane |title=Aliou Cisse - here's what happened to a Birmingham City favourite |url=https://www.msn.com/en-gb/news/other/aliou-cisse-what-happened-next-to-the-birmingham-city-favourite/ar-BByHNlZ |website=[[MSN|MSN]] |publisher=[[Microsoft|Microsoft]] |accessdate=2018-06-24 |archiveurl=https://web.archive.org/web/20180624231530/https://www.msn.com/en-gb/news/other/aliou-cisse-what-happened-next-to-the-birmingham-city-favourite/ar-BByHNlZ |archivedate=2018-06-24 |date=2018-02-05}}</ref>。

西塞在2003年7月的預賽期延遲報到，因此伯明翰城的主帥[[史蒂夫·布鲁斯|史蒂夫·布鲁斯]]便把其列入轉會名單內。西塞終究得以回到一線隊中，但他仍被布魯斯打入冷宮，聖誕節過後亦只在那個賽季多打三場比賽。該賽季結束後，儘管伯明翰城與[[博尔顿足球俱乐部|博尔顿]]接近完成這單轉會，但西塞還是以30萬鎊的身價與[[朴茨茅斯足球俱乐部|朴茨茅斯]]簽了一份為期兩年的合約。這項交易之後成為2007年6月發布的{{link-en|2006年英國足球貧污調查|2006 English football corruption investigation|史蒂文斯報告}}的一部分，而該報告則是描述對英國足壇內部腐敗的關注。關於西塞，史蒂文斯報告指出：「經理人{{link-en|威利·麥凱|Willie McKay}}是朴茨茅斯的代表人，負責西塞轉會一事 […] 這調查現階段沒有想查清該交易的意思。」<ref>{{cite news |title = What Stevens said about each club |publisher = www.telegraph.co.uk |date = 2007-06-16 |url = https://www.telegraph.co.uk/sport/main.jhtml?view=DETAILS&grid=A1YourView&xml=/sport/2007/06/16/sfnste116.xml |accessdate = 2007-06-17 |location = London |deadurl = no |archiveurl = https://web.archive.org/web/20071011230224/http://www.telegraph.co.uk/sport/main.jhtml?view=DETAILS&grid=A1YourView&xml=%2Fsport%2F2007%2F06%2F16%2Fsfnste116.xml |archivedate = 2007-10-11 }}</ref>

於朴茨茅斯待了兩年後，西塞經過兩週的試用期後在2006年11月重回色當<ref>{{cite web |title=Actualité des anciens parisiens |url=http://psg70.free.fr/archive17.htm |website=PSG70 |accessdate=2018-06-25 |archiveurl=https://web.archive.org/web/20170929185643/http://psg70.free.fr/archive17.htm |archivedate=2017-09-29 |language=fr}}</ref>。效力兩年後，他於2008年9月與[[法乙|法乙]]隊伍[[尼姆奧林匹克|尼姆]]簽約，並在2008–09賽季上陣七場後以33歲之齡退役。

== 國家隊生涯 ==
西塞在[[2002年國際足協世界盃|2002年國際足協世界盃]]擔任[[塞內加爾國家足球隊|塞內加爾隊]]的隊長。他帶領隊伍在第一場比賽以1–0的比數[[爆冷|爆冷]]擊敗衛冕冠軍[[法國國家足球隊|法國]]<ref>{{cite web |last1=Glendenning |first1=Barry |title=World Cup 2002: France 0 - 1 Senegal |url=https://www.theguardian.com/football/2002/may/31/minutebyminute.worldcupfootball2002 |website=[[The_Guardian|The Guardian]] |publisher=Guardian Media Group |accessdate=2018-06-22 |archiveurl=https://web.archive.org/web/20180109064123/https://www.theguardian.com/football/2002/may/31/minutebyminute.worldcupfootball2002 |archivedate=2018-01-09 |date=2002-05-31}}</ref>，之後又助球隊在十六強[[加时赛|加時]]淘汰[[瑞典國家足球隊|瑞典]]<ref>{{cite web |last1=Murray |first1=Scott |title=World Cup 2002: Sweden 1 - 2 Senegal |url=https://www.theguardian.com/football/2002/jun/16/minutebyminute.worldcupfootball2002 |website=[[The_Guardian|The Guardian]] |publisher=Guardian Media Group |accessdate=2018-06-22 |archiveurl=https://web.archive.org/web/20150120024332/http://www.theguardian.com/football/2002/jun/16/minutebyminute.worldcupfootball2002 |archivedate=2015-01-29 |date=2002-06-16}}</ref>，惜最終在八強以一球加時飲恨敗於當屆的季軍[[土耳其國家足球隊|土耳其]]<ref>{{cite web |last1=Glendenning |first1=Barry |title=World Cup 2002: Turkey 1 - 0 Senegal |url=https://www.theguardian.com/football/2002/jun/22/minutebyminute.worldcupfootball20021 |website=[[The_Guardian|The Guardian]] |publisher=Guardian Media Group |accessdate=2018-06-22 |archiveurl=https://web.archive.org/web/20160407095027/https://www.theguardian.com/football/2002/jun/22/minutebyminute.worldcupfootball20021 |archivedate=2016-04-07 |date=2002-06-22}}</ref>。西塞也是[[2002年非洲國家盃|2002年非洲國家盃]]亞軍隊伍的一員，但他也是在敗給[[喀麥隆國家足球隊|喀麥隆]]的決賽中於[[互射十二碼|互射十二碼]]中射失的其中一份子<ref>{{cite web|url=http://news.bbc.co.uk/sport1/hi/football/africa/cup_of_nations/1812787.stm|title=Cameroon retain Cup|publisher=BBC|date=2002-02-10|accessdate=2017-10-05|deadurl=no|archiveurl=https://web.archive.org/web/20090330123424/http://news.bbc.co.uk/sport1/hi/football/africa/cup_of_nations/1812787.stm|archivedate=2009-03-30}}</ref>。

== 教練生涯 ==
2015年3月初，西塞被正式任命為[[塞內加爾國家足球隊|塞內加爾國家隊]]的主教練<ref>{{cite web |last1=Okeleji |first1=Oluwashina |title=Aliou Cisse appointed new Senegal coach |url=https://www.bbc.com/sport/football/31736885 |website=[[BBC_Sport|BBC Sport]] |publisher=[[BBC|BBC]] |accessdate=2018-06-22 |archiveurl=https://web.archive.org/web/20170429051736/https://www.bbc.com/sport/football/31736885 |archivedate=2017-04-29 |date=2015-03-05}}</ref>，該隊於2017年11月10日以2-0戰勝[[南非國家足球隊|南非]]，取得[[2018年國際足協世界盃|2018年國際足協世界盃]]的入場劵<ref>{{cite web |last1=Rahman Alfa Shaban |first1=Abdur |title=Senegal qualifies for second World Cup after beating South Africa |url=http://www.africanews.com/2017/11/11/senegal-qualifies-for-second-world-cup-after-beating-south-africa// |website=Africanews.com |publisher=Africanews |accessdate=2018-06-22 |archiveurl=https://web.archive.org/web/20180214072130/http://www.africanews.com/2017/11/11/senegal-qualifies-for-second-world-cup-after-beating-south-africa// |archivedate=2018-02-14 |date=2018-11-11}}</ref>。雖然塞內加爾首場戰勝[[波蘭國家足球隊|波蘭]]，但他們最終仍成為第一隊因公平競技積分而於小組賽被淘汰的隊伍{{註|小組賽三戰過後，塞內加爾與[[日本國家足球隊|日本]]同分，但前者公平競技積分為−6，後者則是−4，日本分數較低得以晉級}}<ref>{{cite web |last1=Carroll |first1=Charlotte |title=Senegal Eliminated From World Cup, Loses Fair Play Tiebreaker to Japan |url=https://www.si.com/soccer/2018/06/28/japan-senegal-eliminated-fair-play-tiebreaker-rule-world-cup |website=[[Sports_Illustrated|Sports Illustrated]] |publisher=Meredith Corporation |accessdate=2018-06-28 |date=2018-06-28 |deadurl=no |archiveurl=https://web.archive.org/web/20180704012137/https://www.si.com/soccer/2018/06/28/japan-senegal-eliminated-fair-play-tiebreaker-rule-world-cup |archivedate=2018-07-04 }}</ref>。西塞其後表示：「這是球例，我們只能接受。當然，我們寧願以其他方式遭到淘汰。對我們來說這是難過的一天，但我們也知道條例就是如此。」<ref>{{cite web |title=Senegal accept fair play tiebreaker as they 'don't deserve' to advance - coach |url=http://www.espn.com/soccer/senegal/story/3549277/senegal-dont-deserve-to-advance-and-accept-fair-play-tiebreaker-aliou-cisse |website=[[ESPN|ESPN]] |publisher=ESPN Inc. |accessdate=2018-06-28 |date=2018-06-28 |deadurl=no |archiveurl=https://web.archive.org/web/20180629050124/http://www.espn.com/soccer/senegal/story/3549277/senegal-dont-deserve-to-advance-and-accept-fair-play-tiebreaker-aliou-cisse |archivedate=2018-06-29 }}</ref>

== 個人生活 ==

西塞在2002年9月26日於[[甘比亞|甘比亞]]沿海發生的{{link-en|優拉號沉沒事故|MV Le Joola}}中失去了幾個家庭成員<ref>{{cite web |title=Footballer's match for ferry victims |url=http://news.bbc.co.uk/2/hi/uk_news/england/2311213.stm |website=[[BBC_News|BBC News]] |publisher=|accessdate=2018-06-24 |archiveurl=https://web.archive.org/web/20180624232528/http://news.bbc.co.uk/2/hi/uk_news/england/2311213.stm |archivedate=2018-06-24 |date=2002-10-08}}</ref>。為了紀念這次船難，他參加了塞內加爾和[[尼日利亞國家足球隊|尼日利亞]]之間的慈善賽，為1000多名受害者家屬籌款<ref>{{cite web |last1=Wamé |first1=Baba |title=Aliou Cissé, le Lion meurtri par le Joola |url=http://www.afrik.com/article5091.html |website=Afrik.com |accessdate=2018-06-24 |archiveurl=https://web.archive.org/web/20030915080903/http://www.afrik.com/article5091.html |archivedate=2003-09-15 |language=fr|trans-title=Aliou Cissé, the lion bruised by Le Joola |date=2002-10-09}}</ref>。西塞當時的球隊[[伯明翰足球會|伯明翰城]]亦有為受害者家屬募款，亦在對陣[[曼城|曼城]]的比賽中展示巨大的塞內加爾旗幟以表達對西塞的尊敬<ref>{{cite news|last= Copnall|first= James|title=Blues fans open their hearts to Senegal's grieving captain |work=[[The_Guardian|The Guardian]]|url=http://football.guardian.co.uk/comment/story/0,,819515,00.html |accessdate=2006-12-02 |location=London |date=2002-10-26}}</ref>。
