'''阿纳托利·亚历山德洛维奇·索布恰克'''（{{lang-ru|'''Собчак, Анатолий Александрович'''}}，{{lang-en|'''Anatoly Aleksandrovich Sobchak'''}}，{{bd|1937年|8月10日|2000年|2月20日|catIdx= S索}}），[[俄罗斯|俄罗斯]][[政治家|政治家]]、[[辩论家|辩论家]]、[[改革派|改革派]]领袖、《[[俄罗斯联邦宪法|俄罗斯联邦宪法]]》起草人，[[聖彼得堡|聖彼得堡]]市長。他培养了[[弗拉基米尔·普京|弗拉基米尔·普京]]和[[德米特里·梅德韦杰夫|德米特里·梅德韦杰夫]]。<ref>{{cite web |url=http://news.ifeng.com/world/special/2012eluosidaxuan/content-2/detail_2012_05/07/14367936_0.shtml |title=普京将构建美中俄欧四极世界 |accessdate=2012年5月7日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>

== 生平 ==
[[File:Russia_stamp_A.A.Sobchak_2002_3.25r.jpg|thumb]]
1937年8月10日，阿纳托利·亚历山德洛维奇·索布恰克出生于[[苏联|苏联]][[俄罗斯苏维埃联邦社会主义共和国|俄罗斯苏维埃联邦社会主义共和国]][[外贝加尔边疆区|外贝加尔边疆区]][[赤塔|赤塔]]，出身家庭环境优越，父亲是工程师，母亲是会计师。

1939年，他们家搬迁到了[[乌兹别克斯坦|乌兹别克斯坦]]，他的大学在[[斯塔夫罗波尔法学院|斯塔夫罗波尔法学院]]和[[圣彼得堡国立大学|圣彼得堡国立大学]]度过。

1963年，阿纳托利·亚历山德洛维奇·索布恰克到[[圣彼得堡国立大学|圣彼得堡国立大学]]开始任教，教授[[经济学|经济学]]，而弗拉基米尔·普京则是学校法律系的学生。<ref name="deer " >{{cite web |url=http://news.hexun.com/2011-08-08/132214994.html |title=索布恰克：俄罗斯民主巨星的崛起与陨落 |accessdate=2011年8月8日 |publisher=和讯新闻 |work= |language =zh-cn}}</ref>

1988年，50岁的索布恰克成为了学校的教授和经济法教研室主任。<ref name="deer " />

=== 圣彼得堡市长 ===
1991年，索布恰克担任[[圣彼得堡|圣彼得堡]]市长，而普京在離開[[KGB|KGB]]後出任对外联络委员会主席，1994年，普京在他的一手栽培下成为圣彼得堡市第一副市长。<ref name="cob " >{{cite web |url=http://news.163.com/06/0726/17/2MVNNKQC0001121M.html |title= 父亲是普京恩师 俄罗斯交际花是普京忘年交|accessdate=2006-07-26 |publisher=网易 |work= |language =zh-cn}}</ref>

1996年，索布恰克下台之后，爆发“索布恰克案件”、“滥用权力”等事件，這打擊他的政治威信，后因病往[[法国|法国]]治疗。<ref name="cob " />1999年6月他回到俄羅斯，不久普京出任總理，其後他支持普京競選總統。

2000年2月20日，阿纳托利·亚历山德洛维奇·索布恰克因为[[心脏病|心脏病]]在寓所中突然去世。<ref name="duck " >{{cite web |url=http://www.china.com.cn/book/txt/2007-11/13/content_9221273_5.htm |title=事业有成的豪门千金 |accessdate=2007-11-13 |publisher=中国网 |work= |language =zh-cn}}</ref>
