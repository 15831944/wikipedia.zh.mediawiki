'''阿克拉姆·亚里'''（{{lang-prs|اکرم یاری}}）{{notetag|或译为“阿卡拉姆·雅日”。<ref>{{Cite journal|title=21世纪南亚毛主义运动：现实图景、理论焦点及未来挑战|author=王静|url=|journal=南亚研究|issue=4|doi=|others=|year=2015|volume=|page=63-85+155|pmid=}}</ref>}}是阿富汗的毛派政治组织者。他是1965年10月6日成立的毛派组织[[进步青年组织|进步青年组织]]的创始人和领袖。<ref>{{Cite web|url=http://www.bbc.com/persian/afghanistan/2013/12/131226_k03_maoism_in_afghanistan|title='شعله جاوید' و خاکستری بجا مانده از مائویسم افغانی|last=تحلیلگر|first=علی امیری نویسنده و}}</ref>

== 早年 ==
阿克拉姆·亚里于1940年出生在阿富汗加兹尼省Jaghori的一个[[哈扎拉人|哈扎拉人]]贵族家庭。他在家乡接受了早期教育。后来他搬到了喀布尔接受高等教育。他先是Khushal Khan高中的老师，后来教学于Mehmood Tarzi高中。

== 政治活动 ==
阿克拉姆·亚里是1965年10月6日成立的毛派组织[[进步青年组织|进步青年组织]]的领导者和创始人。[[进步青年组织|进步青年组织]]出版杂志《[[永恒火焰|永恒火焰]]》，散布于学生和青年之间。

阿克拉姆·亚里反对[[穆罕默德·查希尔沙|穆罕默德·查希尔沙]]国王的君主制、伊斯兰原教旨主义者、和亲苏联的[[阿富汗人民民主党|阿富汗人民民主党]]。进步青年组织坚持[[马克思列宁主义毛泽东思想|马克思列宁主义毛泽东思想]]，旨在以内战的方式推翻当时的秩序。

== 影响 ==
在阿克拉姆·亚里的领导下，进步青年组织得到了阿富汗城市大量工人和学生的大力支持。在阿富汗人民中间，进步青年组织领导的运动以“[[永恒火焰|永恒火焰]]”为名，其成员通常被称为Sholais（英语音译），以他们的杂志名字命名。

阿克拉姆·亚里是马克思主义的教员和传播者，将马克思主义介绍给阿富汗的众多知识分子和政治活动家，其中包括[[阿富汗解放组织|阿富汗解放组织]]创始人[[法伊兹·艾哈迈德|法伊兹·艾哈迈德]]。

== 逮捕和暗杀 ==
1978年，亲苏联的阿富汗人民民主党通过军事政变上台。阿富汗人民民主党政府开始严厉打击进步青年组织的干部。阿克拉姆·亚里在Jaghori的家中被捕，被转移到喀布尔，后来被阿富汗人民民主党政府杀害。他死亡的确切情况不得而知。
