'''阿梅迪奥八世'''（1383年9月4日-1451年1月7日），號稱'''和平者'''，是[[義大利|義大利]]的[[薩伏依|薩伏依]]伯爵（1391年~1416年）和公爵（1416年~1440年）。从1439年11月到1449年4月，他是[[对立教宗|对立教宗]]，称'''斐理斯五世'''。他是历史上最后一位有现实意义的对立教宗，其對立教宗生涯結束於他自認非法的自行退位。<ref>[http://www.catholic.org.tw/katalina/ycw/BLESSING/dict/A.htm 天主教字典:A] {{webarchive|url=https://web.archive.org/web/20101101214643/http://www.catholic.org.tw/katalina/ycw/BLESSING/dict/A.htm |date=2010-11-01 }} 見對立教宗欄</ref> 

1416年，[[神圣罗马帝国|神圣罗马帝国]]皇帝[[西吉斯孟_(神圣罗马帝国)|西吉斯孟]]升阿梅迪奥伯爵為第一代薩伏依公爵。阿梅迪奥八世娶勃艮第的玛丽（{{bd|1386年|9月|1422年|10月2日|catIdx=}}，生於[[第戎|第戎]]，卒於[[阿拉斯|阿拉斯]]）为妻子。 玛丽是[[勃艮第|勃艮第]]公爵[[菲利普二世_(勃艮第)|勇敢的菲利普二世]]的女儿，[[法国|法国]]国王[[约翰二世_(法兰西)|约翰二世]]的孙女。他们有九个孩子:

# 玛格丽特 (1405年-1418年)
# 安东尼 (1407年)
# 安东尼 (1408年)
# 玛格丽特 (1410年-1479年)曾先後與名义上的[[那不勒斯|那不勒斯]]国王[[路易三世_(安茹)|安茹的路易三世]]、[[普法尔茨统治者列表|普法尔茨]][[选侯|选侯]][[路德维希四世（普法尔茨）|路易四世]]和[[符腾堡统治者列表|符騰堡]]公爵[[乌尔里希五世|巫禮軻五世]]（Ulric V）成親。
# 玛丽 (1411年-1469年)嫁給了維斯孔蒂家族的[[菲利浦.瑪麗亞.維斯孔蒂|菲利浦.瑪麗亞.維斯孔蒂]]
# [[阿梅迪奥_(皮埃蒙特)|阿梅迪奥]]，皮埃蒙特亲王(1412年-1431年)
# [[路多维科_(萨伏依)|路多维科]] (1413年-1465年)
# 博内 (1415年-1430年)
# 菲利波 (1417年-1444年)
