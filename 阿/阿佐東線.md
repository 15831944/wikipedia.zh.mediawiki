'''阿佐東線'''（{{jpn|j=阿佐東線|hg=あさとうせん|rm=Asatō sen|f=y}}），是指由[[日本|日本]][[德島縣|德島縣]][[海部郡_(德島縣)|海部郡]][[海陽町|海陽町]]的{{ST|海部|2}}，至[[高知縣|高知縣]][[安藝郡_(高知縣)|安藝郡]][[東洋町|東洋町]]的{{ST|甲浦|2}}為止的一段鐵路路線。-{zh-tw:目前;zh-hk:現時;zh-cn:目前;}-由[[阿佐海岸鐵道|阿佐海岸鐵道]]負責營運。在營運上，它與採[[直通運行|直通運行]]方式聯營的[[JR四國|JR四國]][[牟岐線|牟岐線]]合稱為「'''阿波室戶海岸線'''」（{{lang|ja|阿波室戸シーサイドライン}}）。

== 路線資料 ==
* 管轄（事業類別）：阿佐海岸鐵道（[[鐵路公司#第一種鐵道事業者|第一種鐵道事業者]]）
* 路線距離（{{link-ja|營業距離|営業キロ}}）：8.5公里
* [[軌距|軌距]]：1,067毫米（[[窄軌|窄軌]]）
* 站數：3個（包括起終點站）
* 複線路段：沒有（全線[[單線鐵路|單線]]）
* 電氣化路段：沒有（全線{{link-ja|非電氣化|非電化}}）
* [[閉塞_(鐵路)|閉塞方式]]：特殊自動閉塞式（軌道回路檢知式）
** [[列車集中制御裝置|CTC]]（[[宍喰站|宍喰站]]設有CTC控制室）<ref>[http://asatetu.com/sisikuieki.html 阿佐東線の駅紹介] - 阿佐海岸鉄道</ref>
* 最高速度：85公里/小時

== 運行模式 ==
本線現時只提供普通列車服務，約1-2小時1班，大部份為海部－甲浦間運行，另外，每天亦設有數班列車只行駛宍喰－海部或宍喰－甲浦的區間車。

阿左東線開始營運時，JR四國曾把部份特急列車如「渦潮」、「劍山」、「室戶」等伸延至甲浦。但於2008年3月15日時間表更正時取消，主因是客量不多。經過地方團體請願後，於2009年12月1日重開。<ref>[https://web.archive.org/web/20091203111919/http://www.jr-shikoku.co.jp/03_news/osirase/09-11-25/01.htm 阿佐海岸鉄道との相互乗入れについて] JR四国 2009年11月25日（日文版）</ref><ref>[http://www.asatetu.v-co.jp/news.htm JR四国との直通運転を再開] {{webarchive|url=https://web.archive.org/web/20100111193258/http://www.asatetu.v-co.jp/news.htm |date=2010-01-11 }} 阿佐海岸鉄道 阿佐鉄ニュース 2009年11月24日（日文版）{{deadlink}}</ref>不過到了2011年3月12日的時間表更正後再次取消，只保留早上直通至{{ST|牟岐}}的班次，2019年3月16日，隨時間表改正，所有班次均只行駛至海部，不再駛進牟岐線路段。不過，公司同時也宣佈將於2020年全線改用DMV行走全線，屆時服務範圍將會延長至[[阿波海南站|阿波海南站]](見下段)。

== 歷史 ==
阿佐東線本來是JR牟岐線的伸延段，並已動工，當時稱為「阿佐線」。但受1980年的「國鐵再建法」所影響，工程遭凍結，後來由德島縣縣政府、海陽町町政府、高知縣縣政府等地方自治體集資組成以[[日本第三部門鐵道|第三部門]]方式營運的公司並承投工程，成為當時唯一推展前國鐵時代未完成路段的公司。

* 1992年3月26日：海部至甲浦間開業。特急「渦潮」開始伸延至甲浦。普通列車則除朝、夜來回4個班次外，全部均直通至甲浦。
* 1994年：增設宍喰－甲浦間的區間列車。
* 1998年3月14日：特急「劍山」亦伸延至甲浦。
* 1999年3月13日：特急「渦潮」改以[[德島車站|德島]]為終站，改由新開設的特急「室戶」代替。
* 2001年3月3日：特急「室戶」不再直通，同時除早上2來回的普通列車外，其他時段亦不再駛入。
* 2006年3月18日：取消周末及假日的直通。
* 2008年3月15日：取消所有直通JR的班次，並提早尾班車開出時間。
* 2009年12月1日：個別特急「劍山」重新直通至甲浦，但自{{ST|牟岐}}起變為各站均停列車。
* 2011年3月12日：特急「劍山」再次取消直通至甲浦。
* 2017年2月3日：宣佈將不遲於2020年，引入3架DMV「[[雙模式車輛|雙模式車輛]]」，行駛甲浦至[[高知縣|高知縣]]的[[室戶市|室戶市]]<ref>[https://www.nikkei.com/article/DGXLZO12516500T00C17A2LA0000/ 阿佐海岸鉄道、線路・道路両用車両を20年までに導入]，日本經濟新聞，2017年2月4日。</ref>，並替代現時兩部氣動列車。此段原屬阿佐線一部份，但一直以來因財政原因而未有興建。至於室戶來往[[土佐黑潮鐵道|土佐黑潮鐵道]]{{link-ja|奈半利站|奈半利駅}}的路段，則仍由現時已提供服務的{{link-ja|高知東部交通|高知東部交通}}負責。將來雙模式車輛投入服務後，服務路段亦會由現時[[海部站|海部站]]延長至[[阿波海南站|阿波海南站]]<ref>[https://tabiris.com/archives/asakaigan/ 阿佐海岸鉄道「DMV」はローカル線活性化の起爆剤になるか。室戸岬を目指して2020年にも運行開始]，Tabiris，2017年2月4日。</ref>。
* 2019年：
:*1月10日：連接甲浦站架空鐵路路軌及地面道路的斜坡道工程正式展開，同日起停車站及巴士站亦遷移<ref>[http://asatetu.com/archives/1691 甲浦駅駐車場・バス停移転のお知らせ]，官方新聞稿。2018年12月26日。 </ref>。
:*3月9日：首輛DMV雙模式車輛於宍喰站正式亮相<ref>[https://www.sankei.com/photo/story/news/190309/sty1903090015-n1.html 徳島でＤＭＶ車両公開　世界初の本格営業目指す]，產經新聞，2019年3月9日。</ref><ref>[https://headlines.yahoo.co.jp/hl?a=20190309-00000095-kyodonews-soci DMV車両を公開、阿佐海岸鉄道　2020年運行目指す]，日本雅虎新聞（轉載至共同通信）。</ref><ref>[https://www.kochinews.co.jp/article/260051/ 阿佐海岸鉄道・阿佐東線DMVがお披露目　青基調｢サーフィン｣車両]，高知新聞，2019年3月10日。</ref>。
:*3月16日：時間表改正，取消所有直通JR牟岐線的班次，統一只運行自家線路段<ref>[http://asatetu.com/archives/1783 官方網頁新聞稿。]</ref>。
