'''強·尼可拉·阿瑟·兰波'''（{{lang-fr|Jean Nicolas Arthur Rimbaud}}，{{bd|1854年|10月20日|1891年|11月10日|catIdx=R兰波}}），或译'''阿尔图尔·兰波'''、'''韓波'''、'''林包德'''，19世纪[[法国|法国]]著名[[诗人|诗人]]。无法归类的天才诗人，创作时期仅在14-19岁，之后便停笔不作。受法国[[象征主义文学|象征主义诗歌]]影响，[[超现实主义文学|超现实主义诗歌]]的鼻祖。

== 青年时期 ==

兰波出生于[[法国|法国]]东北部夏樂维雅乡村的一个中产阶级家庭。少年时代的兰波是一个好动而才华横溢的学生。15岁那年，他就能以[[拉丁文|拉丁文]]写作各种诗歌并赢得了很多奖赏。

1870年，兰波的老师[[乔治·伊森巴尔|乔治·伊森巴尔]]成为兰波在文学道路上的领路人。在他的指导下，兰波开始用[[法语|法语]]写诗，其法语诗歌的创作水平进展迅速。兰波性格叛逆，屡次离家出走，甚至曾经参与过1871年的[[巴黎公社|巴黎公社]]组织。他在诗歌《巴黎人的狂欢或巴黎的重生》一诗中描述了自己参与[[巴黎公社|巴黎公社]]的这段经历。1871年以后，兰波成为一个[[无政府主义|无政府主义]]者，他开始酗酒，并经常蓄长髮、衣衫褴褛的招摇过市，以图嘲笑那些一本正经的中产阶级。他给他的老师伊森巴尔写信，系统阐述他的诗歌创作理论，即在“漫长的、庞大的、理性的骚乱中”加入幻觉的因素。

1871年9月底，兰波再次回到[[巴黎|巴黎]]，不过这次是应著名[[象征主义文学|象征主义]]诗人[[保尔·魏尔伦|保尔·魏尔伦]]的邀请。魏尔伦曾读过兰波的著名作品《[[醉舟|醉舟]]》，十分爱慕兰波的诗才。来到[[巴黎|巴黎]]之后，兰波住在魏尔伦的家里。很快，魏尔伦便和这个17岁的才华横溢的文学青年坠入爱河，他们成为[[巴黎|巴黎]]诗坛著名的同性情侣。两人同居之后，生活挥霍而放任，酗酒和吸食[[大麻|大麻]]是家常便饭。他们为巴黎的文学精英团体所不容，而兰波的恃才傲物更是引起许多的反感。这一时期，兰波创作了大量具有震撼力的诗作，他的诗歌成就甚至超过了[[象征主义文学|象征主义文学]]的先驱[[波德莱尔|波德莱尔]]。

兰波和魏尔伦之间的同性恋情日益炽烈。1872年，魏尔伦甚至离开了他的妻子和尚在襁褓中的儿子，和兰波一同私奔至[[伦敦|伦敦]]。1873年7月，两人在[[布鲁塞尔|布鲁塞尔]]火车站曾发生过一场激烈的争吵，随后魏尔伦用枪打伤了兰波的手腕，一气之下兰波叫来警察，[[魏尔伦|魏尔伦]]被逮捕。被捕期间，魏尔伦甚至被迫接收了一系列具有侮辱性的心理治疗，原因是魏尔伦的妻子指控她的丈夫和兰波之间不正常的“友情”。开庭审理时，尽管兰波一再宣称自己撤回对魏尔伦的控诉，法官还是判魏尔伦入狱两年。魏尔伦入狱后，兰波只身一人回到故乡夏勒维亞，在极度伤心中完成了一生最杰出的诗作《[[地狱一季|地狱一季]]》。这部作品是[[象征主义文学|象征主义文学]]的精品。在诗中，兰波追忆他和魏尔伦共同生活的“地狱情侣”的岁月。他甚至以“悲伤的兄弟”、“疯癫的童贞女”来称呼魏尔伦，而自己则是他的“下地狱的丈夫”。1874年，兰波和诗人[[杰曼·努沃|杰曼·努沃]]再次返回[[伦敦|伦敦]]，并出版了他倍受争议的作品《彩画集》，其中包含了两首最早的以自由诗体写成的法语诗歌。

== 晚年生活 ==
1875年，兰波和[[魏尔伦|魏尔伦]]最后一次在[[德国|德国]]相遇。此时的魏尔伦已经获释，并被迫皈依了[[天主教|天主教]]。这个时候，兰波已经受够了早年的放纵生活，基本放弃了写作生涯，而是开始从事一些能够给他带来稳定收入的工作。他开始徒步在[[欧洲|欧洲]]大陆旅行。1876年夏天，他甚至加入了[[荷兰|荷兰]]的军队，只是为了免费到[[印度尼西亚|印度尼西亚]]的[[爪哇|爪哇]]去旅行。然而到了爪哇之后，他很快就厌倦了，并立即乘船返回[[法国|法国]]。他曾游历[[塞浦路斯|塞浦路斯]]并最终在[[亞丁_(也門)|亞丁]]定居，并成为巴尔代公司的一名雇员。
1884年，兰波辞去工作，开始独立在[[埃塞俄比亞帝國|阿比西尼亚]]（今天的[[埃塞俄比亚|埃塞俄比亚]]）经商。他从事军火走私生意，并赚了不少钱。而在这个时候，兰波的右膝盖患上滑膜炎，并很快恶化为[[癌症|癌症]]。日益恶劣的病情迫使兰波于1891年5月9日返回[[法国|法国]]。5月27日，兰波做了右腿的截肢手术，然而这还是没能抑制癌细胞的扩散。同年11月10日，兰波在法國馬賽逝世，享年37岁。

== 影响 ==
兰波对[[现代文学|现代文学]]、音乐和艺术的影响是非常深远的。他在[[巴黎|巴黎]]的生活经历曾经于1995年被拍成电影《[[全蝕狂愛|全蝕狂愛]]》，由[[李奥纳多·迪卡普里奥|李奥纳多·迪卡普里奥]]扮演。

兰波的创作是法语诗歌历史上的重大变革。他本人是一位无法被归类的诗人，有人认为他是[[象征主义文学|象征主义运动]]最杰出的诗人之一，他也被公认为是其后的[[超现实主义文学|超现实主义]]的鼻祖。二战后诞生于美国的“[[垮掉的一代|垮掉的一代]]”的诗风也深受兰波影响。今日人们在追忆现代诗歌的发展历程时，兰波的名字都是绝对不会忽略的。就连恐怖小说作家[[托马斯·里戈蒂|托马斯·里戈蒂]]也曾在不同场合表示自己是兰波的忠实追随者。
