'''阿分站'''（{{jpn|j=阿分駅|hg=あふんえき|rm=Afun eki|f=y}}）是位於[[北海道|北海道]][[增毛郡|增毛郡]][[增毛町|增毛町]]大字阿分村，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[留萌本線|留萌本線]]，已廢除的[[鐵路車站|鐵路車站]]。電報略號為'''アフ'''。

車站名稱源於[[阿伊努語|阿伊努語]]「ahun-i」（進入），主要指附近的海岸。

== 歷史 ==
* 1963年（昭和38年）12月1日 - [[日本國有鐵道|日本國有鐵道]][[禮受車站|禮受車站]]至[[舍熊車站|舍熊車站]]之間設立'''阿分臨時站'''（{{lang|ja|仮乗降場}}），開始旅客業務。
* 1987年（昭和62年）4月1日 - 隨著[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）接管。同時車站升格為'''阿分站'''。
* 1990年（平成2年）3月10日 - 設定營業距離。
* 2016年（平成28年）12月5日 - 由於[[留萌車站|留萌車站]]至[[增毛車站|增毛車站]]路段廢除，本車站同時廢站<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2016/160428-3.pdf |language=ja |title=留萌線（留萌・増毛間）の鉄道事業廃止届の提出について |publisher=[[北海道旅客鐵道|JR北海道]] |date=2016-04-28 |accessdate=2017-06-13 |archive-url=https://web.archive.org/web/20160429052355/http://www.jrhokkaido.co.jp/press/2016/160428-3.pdf |archive-date=2016-04-29 |deadurl=yes }}</ref><ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2016/160628-1.pdf |language=ja |title=留萌線（留萌・増毛間）の廃止日繰上げの届出について |publisher=[[北海道旅客鐵道|JR北海道]] |date=2016-06-28 |accessdate=2017-06-13 |archive-url=https://web.archive.org/web/20160701083826/http://www.jrhokkaido.co.jp/press/2016/160628-1.pdf |archive-date=2016-07-01 |deadurl=yes }}</ref>。

== 車站構造 ==
廢站時為1面1線的地面車站。[[側式月台|側式月台]]位於路軌的西方（增毛方向的右方）。沒有[[轉轍器|轉轍器]]的棒線車站。

由臨時車站開始已經是無人車站。月台南面出入口附近設有使用[[装配式建筑|預製組件]]興建的候車室，內部則使用[[礫石|礫石]]及[[人工草坪|人工草坪]]舖設。沒有設置[[廁所|廁所]]。月台為木製甲板式，往增毛方向設有斜坡連接車站外的道路。月台十分短，長度只能夠讓一輛列車停靠<ref>{{cite news |title=留萌線・留萌―増毛廃止まで１カ月　９５年の歴史記憶に |url=http://dd.hokkaido-np.co.jp/news/area/dohoku/1-0334650.html |language=ja |author=堀雅晴 |publisher=[[北海道新聞|北海道新聞]] |date=2016-11-05 |accessdate=2017-06-13 |archive-url=https://web.archive.org/web/20161105011420/http://dd.hokkaido-np.co.jp/news/area/dohoku/1-0334650.html |archive-date=2016-11-05 |deadurl=yes }}</ref>。

此外，位於本車站至信砂車站之間的阿分隧道，是留萌車站至增毛車站這路段唯一一條隧道。

== 載客量 ==
* 1992年的每天載客量為7人，每天上下車乘客量為14人。
* 2011年至2015年度平均每天載客量少於1人<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |language=ja |title=平成26年度 線区別の収支状況等について |publisher=JR北海道 |date=2016-02-10 |accessdate=2017-06-21 |archive-url=https://web.archive.org/web/20170407133949/http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |archive-date=2017-04-07 |deadurl=yes }}</ref>。
