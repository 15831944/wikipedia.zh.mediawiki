'''阿爾法小組'''（{{Lang-uk|Альфа}}）是隸屬於[[烏克蘭安全局|烏克蘭安全局]]反恐中心的一支[[特種部隊|特種部隊]]，其主要任務為打擊[[恐怖主義|恐怖主義]]、解救被劫持的人質，以及執行其他高風險的行動。

== 歷史 ==
[[阿爾法小組|阿爾法小組]]是在[[蘇聯|蘇聯]][[克格勃|克格勃]]主席尤里·安德羅波夫提議下於1974年7月28日成立，[[1972年慕尼黑奧運會|1972年慕尼黑奧運會]]發生[[慕尼黑慘案|慕尼黑慘案]]，導致11名[[以色列|以色列]]運動員死亡，尤里·安德羅波夫擔心1980年的莫斯科奧運會受到恐怖份子的襲擊，故從克格勃中抽調約30位的年輕成員組成阿爾法小組，該部隊曾在[[阿富汗戰爭_(1979年)|1979年阿富汗戰爭]]期間擔任重要角色，當中以[[333風暴行動|333風暴行動]]最廣為人知。

後來，蘇聯當局在國內各大城市設立了阿爾法小組的長駐部隊，而當中駐守在烏克蘭的分隊於1990年3月3日建立。當時，克格勃第7部門收到上頭的命令，要求在[[基輔|基輔]]成立阿爾法部隊的第10小組。部隊的選拔過程相當艱難，由最初的120名候選人當中到最後只有15人通過，並組成了第一支分隊。<ref name="Alpha">http://www.bratishka.ru/archiv/2009/6/2009_6_7.php</ref>

在[[蘇聯解體|蘇聯解體]]後，這支位於基輔的阿爾法地區分隊於1992年被改編為[[烏克蘭安全局|烏克蘭安全局]]的“C”局，但仍然被大眾非正式地稱為“阿爾法”，直至1994年6月23日，[[烏克蘭總統|烏克蘭總統]]透過法令以“C”局為基礎從而建立了“A”部門（A代表阿爾法）。目前在烏克蘭國內各區域中心都設有烏克蘭安全局阿爾法部隊的辦公室。

2014年[[烏克蘭親歐盟示威|烏克蘭親歐盟示威]]後，阿爾法小組遭到新政府的清算和重組，並被派到烏克蘭東部地區鎮壓立場親俄的[[分離主義|分離主義]]民兵。同年四月末，三名阿爾法小組的隊員被頓巴斯的民兵組織俘虜，期間遭到毆打，並出現在[[俄羅斯|俄羅斯]]的電視節目當中。據烏克蘭安全局的發言人所稱，分離主義武裝人員在安全局內加插了[[卧底|卧底]]內應。一名叫亞歷山大·霍達科夫斯基的[[頓涅茨克州|頓涅茨克州]]前阿爾法小組指揮官在舊政府倒台後帶領著數名隨同的戰士一同叛逃，並成為了叛軍組織—東方營的指揮官，後來更成為了[[頓涅茨克人民共和國|頓涅茨克人民共和國]]的安全部長。總括而言，烏克蘭安全局指出有約30%的阿爾法小組隊員目前下落不明，當中很大機會是加入了頓涅茨克或[[盧甘斯克|盧甘斯克]]地區的叛軍組織作戰。至2014年3月為止只有200名活躍的阿爾法小組隊員仍然效忠於烏克蘭政府。

== 已知的行動 ==
[[File:Ukrainian_soldiers_near_Kramatorsk.jpg|thumb]]期間於[[克拉馬托爾斯克|克拉馬托爾斯克]]執行任務的阿爾法小組人員]]
在1995年，阿爾法小組執行了一次成功的行動，並拘捕了30名持有重型武器並穿有防彈背心的武裝團匪。而在1997年，該部隊於赫爾松地區聯同內務部的「金雕」和「禿鷹」部隊一起行動，在敵方守備嚴緊和沒有空中支援的情況下拘捕了36名[[黑幫|黑幫]]。

在1992至1993年，阿爾法小組曾執行護運國家新貨幣的行動，該批貨幣是在[[法國|法國]]、[[義大利|義大利]]和[[加拿大|加拿大]]印刷，並分別透過空中和海上運輸。

該部隊亦曾被派往世界各地的一些熱點地區（例：[[阿富汗|阿富汗]]、[[馬其頓|馬其頓]]和[[伊拉克|伊拉克]]等）執行維和任務。

近年來，阿爾法小組活躍於[[烏克蘭親歐盟示威|烏克蘭親歐盟示威]]和[[烏克蘭親俄羅斯武裝衝突_(2014年－2015年)|烏克蘭親俄羅斯武裝衝突]]等重大事件和衝突當中。

== 裝備 ==
除了[[蘇聯|蘇聯]]時期的老式裝備外，阿爾法小組亦裝備有許多購自外國的先進武器及裝備，還有品質較高的戰鬥服以符合現代化需求<ref name="Alpha"/>。
