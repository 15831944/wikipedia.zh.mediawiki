'''阿羅漢'''（{{lang-sa|अर्हत्}}，{{IAST|Arhat}}；{{lang-pi|Arahant}}），又譯'''阿羅訶'''，意譯為'''應供'''、'''殺賊'''、'''無生'''，漢語常簡稱為'''羅漢'''；爲[[原始佛教|原始佛教]]、[[部派佛教|部派佛教]]的第四果，是依照[[佛|佛]]三轉法輪[[解脫道|解脫道]]的教導修習[[四念處|四念處]]、[[十二因緣觀|十二因緣觀]]、[[四聖諦|四聖諦]]、[[八聖道|八聖道]]，於[[蘊_(佛教)|蘊]]、[[處_(佛教)|處]]、[[界_(佛教)|界]]觀行一切有為法的剎那生滅、空、無常、[[苦_(佛教)|苦]]、無我，進而斷盡[[我見|我見]]、[[我執|我執]]、三界貪愛、[[無明|無明]]及其它所有煩惱，於當世捨壽時願意將自我滅盡不再受後有於三界中出生，脫離生死輪迴之苦而入[[無餘涅槃界|無餘涅槃界]]的聖者。自佛陀三轉法輪以來，至今已有成千上萬的弟子成就阿羅漢果，還有眾多的弟子成就[[三果|阿那含]]、[[二果|斯陀含]]和[[初果|須陀洹]]。

[[佛#十種德號|佛陀十號]]中，也包括阿羅漢，汉译通常作“应供”。

==語源和語義==
{{wiktionary|:en:arhat|:en:arhant}}
===詞源===
[[巴利語|巴利語]]{{lang|pi|arahant}}與[[梵語|梵語]]{{lang|sa|arhat}}的字根來源，目前尚不清楚，仍有爭議。這個字在釋迦牟尼與佛教出現之前就存在。最早出現在《[[梨俱吠陀|梨俱吠陀]]》中，被拼成arhattā。之後在[[佛教|佛教]]與[[耆那教|耆那教]]的文獻中，多次使用這個字，在印度教中的[[毘濕奴派|毘濕奴派]]經典中也曾出現。

有觀點認為，在梵語字源中，arhant是由hant（死亡、殺戮）加上否定詞頭ar所組成，字面意思為「無法被殺」、「超越死亡」或「不死」，故漢地舊譯爲“無生”。

還有種觀點認為，其來自梵語詞根अर्हत् (árhat）,義爲值得崇拜的人，故漢地又舊譯爲“應供”。作為佛的十號之一“[[佛#应供|應供]]”，阿羅漢即是其音譯，故釋迦佛亦是阿羅漢。它的字源也可能是來自於巴利語{{lang|pi|araha}}或梵語{{lang|sa|arha}}，意思是值得、有價值的。變化成人身名詞，意思是「傑出的人」、「有價值的人」。

===語義===
佛陀十號中，包括阿羅漢<ref>玄奘譯《阿毘達摩法蘊足論》卷2：「云何佛證淨？如世尊言：此聖弟子，以如是相，隨念諸佛；謂此世尊，是如來、阿羅漢、正等覺、明行圓滿、善逝、世間解、無上丈夫、天人師、佛、薄伽梵。」</ref>。意譯有多：
# 應供：佛的十種稱號當中就有「應供」一項，而「應供」的梵語其實正是「阿羅漢」。阿羅漢福慧俱足，為眾生之福田，供養阿羅漢可以修福，以其能教眾生如何修福、修慧、斷煩惱。
# 殺賊：「賊」指煩惱包括一念無明及無始無明煩惱，使眾生有損﹝障礙解脫出離三界或是障礙成佛﹞。阿羅漢斷除所有煩惱，故-{云}-「殺賊」。
# 無生：無生是不再出生下一世的[[五蘊|五蘊]]，出離了[[三界|三界]][[六道|六道]][[輪迴|輪迴]]。
# 無學：阿羅漢有名號稱'''無學'''，指的是[[比丘|比丘]]於[[因地|因地]]僅就如何解脫於三界束縛的修行所學圓滿，也就是世俗諦的正理，而無需繼續修學；這個稱號見於“二乘定性不定性無學有學”，二乘即[[聲聞乘|聲聞乘]]和[[緣覺乘|緣覺乘]]，定性義爲決定性的、不會退轉的，不定性的爲還有退轉的可能，相對於無學，有學的意思即仍需修學，尚未圓滿者。 

==概論==
根據[[原始佛教|原始佛教]]、[[部派佛教|部派佛教]]的定義，阿羅漢包括了[[佛陀|佛陀]]、[[辟支佛|辟支佛]]和聲聞阿羅漢，佛說過阿羅漢已經「永無來生」<ref name="《中阿含經》">《[[中阿含經|中阿含經]]》：漏盡阿羅訶比丘已知滅盡，拔其根本，'''永無來生'''。</ref>，永遠不會再繼續輪迴。在佛世之中，即還有佛法流傳的時代裡，不會有第二尊佛或其他辟支佛出現<ref>《[[增壹阿含經|增壹阿含經]]》：是時，諸辟支佛即於空中燒身取般涅槃。所以然者，'''世無二佛之號'''，故取滅度耳。一商客中終無二導師，一國之中亦無二王，'''一佛境界無二尊號'''。</ref>，只有聲聞阿羅漢出現。聲聞就是聽聞佛或佛弟子說法的意思。

斷惑究竟的修行者，已證三果後，能夠進一步斷「五上分結」(色貪、無色貪、掉舉、慢、無明）<ref>《[[長阿含經|長阿含經]]·[[眾集經|眾集經]]》：「復有五法。謂五上結。色愛、無色愛、無明、慢、掉。」</ref>，即可證入阿羅漢果。證入阿羅漢果的聖者，將會自覺：「我生已盡，梵行已立，所作已作，自知不受後有。」阿羅漢於『六恒住』法常善安住，在面對色聲香味觸法等六境時，恒常不動「不苦不樂，捨心，住正念正智」<ref>《雜阿含經》卷13</ref>，一切外境「不能妨心解脫、慧解脫」<ref>《雜阿含經》卷9：「耳識聲、鼻識香、舌識味、身識觸、意識法，不能妨心解脫、慧解脫，意堅住故，內修無量善解脫，觀察生滅。譬如村邑近大石山，不斷、不壞、不穿，一向厚密，假使四方風吹，不能動搖、不能穿過，彼無學者亦復如是。眼常識色，乃至意常識法，不能妨心解脫、慧解脫，意堅住故，內修無量善解脫，觀察生滅。」</ref>，有些阿羅漢弟子可以證得[[三明六通|三明六通]]。

傳統上認為，證阿羅漢果之後的修行者，都會[[出家|出家]]，加入[[僧團|僧團]]。但是[[上座部佛教|上座部佛教]]的《[[論事|論事]]》記載中，[[上座部佛教|上座部佛教]]之外的其它部派曾有[[在家阿羅漢|在家阿羅漢]]的說法，[[覺音|覺音]]註釋稱其為[[北道派|北道派]]提出，[[北道派|北道派]]認為在家眾也可以證阿羅漢果，在證阿羅漢果之後也不一定需要出家，[[上座部佛教|上座部佛教]]的《[[彌蘭王問經|彌蘭王問經]]》記載在家弟子證阿羅漢果後如果沒有在當天入滅就會出家<ref>[http://tripitaka.cbeta.org/N27n0012_001 《彌蘭王問經》]：大王！若優婆塞而為預流作證阿羅漢位者，彼即日般涅槃，或成至比丘之狀態耶？彼有此二途無他。大王！蓋彼比丘地是不動之出家，是宏大、清淨、崇高。</ref>。

根据[[原始佛教研究|原始佛教研究]]者的定义，阿罗汉是依照正法修行而达到[[涅槃|涅槃]]的圣者，離貪斷愛，名色識三者不再相依相緣，再有再生之因緣滅盡，無因緣可記說。

==分類==
在北傳佛教經典中，有[[九無學|九無學]]<ref>《[[中阿含經|中阿含經]]·大品·福田經》：「世尊告曰。居士。世中凡有二種福田人。云何為二。一者學人。二者無學人。學人有十八。無學人有九。居士。云何十八學人。信行、法行。信解脫、見到、身證。家家、一種、向須陀洹、得須陀洹、向斯陀含、得斯陀含、向阿那含、得阿那含。中般涅槃、生般涅槃、行般涅槃、無行般涅槃、上流色究竟。是謂十八學人。居士。云何九無學人。思法。昇進法。不動法。退法。不退法。護法。護則不退、不護則退。實住法。慧解脫。俱解脫。是謂九無學人。」</ref>和[[六種阿羅漢|六種阿羅漢]]<ref>《[[大毘婆沙論|大毘婆沙論]]》：「阿羅漢有六種。一退法。二思法。三護法。四安住法。五堪達法。六不動法。」</ref>的記載。是否有“[[退法阿羅漢|退法阿羅漢]]”，是部派佛教部派根本分歧之一，其中[[大眾部|大眾部]]和[[分別說部|分別說部]]宗義為“阿羅漢'''無退'''義”<ref>《[[異部宗輪論|異部宗輪論]]》：「此中[[大眾部|大眾部]]。一說部。說出世部。雞胤部。本宗同義者。謂四部同說。……預流者有退義。阿羅漢無退義。」「其[[化地部|化地部]]本宗同義。……預流有退。諸阿羅漢定無退者。」「其[[法藏部|法藏部]]本宗同義。……阿羅漢身皆是無漏。」</ref>，而[[說一切有部|說一切有部]]宗義為“阿羅漢'''有退'''義”<ref>《[[異部宗輪論|異部宗輪論]]》：「[[說一切有部|說一切有部]]本宗同義者。……預流者無退義。阿羅漢有退義。」</ref>。

==慧解脫==
在經典記載中阿羅漢還可分為兩大類：
# [[慧解脫|慧解脫]]阿羅漢
# [[俱解脫|俱解脫]]阿羅漢

慧解脫阿羅漢是指不依禪定，於四禪定未俱足，卻能以智慧解脫的阿羅漢聖者，最早出現於《[[雜阿含經|雜阿含經]]》347經（相當於南傳[[相應部|相應部]]《須深經》）<ref>《[[雜阿含經|雜阿含經]]》347經：「有一比丘語須深言：『須深當知，我等生死已盡，梵行已立，所作已辦，自知不受後有。』……須深復問：『云何？尊者所說不同，前後相違。云何不得禪定而復記說？』比丘答言：『我是慧解脫也。』……佛告須深：『彼先知法住，後知涅槃。彼諸善男子獨一靜處，專精思惟，不放逸法，離於我見，不起諸漏，心善解脫。』」</ref>。

在聖典所記載的慧解脫，其說明為阿羅漢慧解脫，是因為智慧而解脫，在聖典道次之比對，俱解脫阿羅漢為慧解脫與[[八解脫|八解脫]]俱足而不受後有，相對於慧解脫阿羅漢雖不是依於禪定而修，雖於四禪定還未能俱足自在，但也是「生死已斷，梵行已立，所作已辦，不受後有。」是已斷除三界愛的阿羅漢<ref>《顯揚聖教論》卷3：「慧解脫，謂已得諸漏無餘盡滅，未得八解脫身證具足住。」</ref>，因此慧解脫也是四果的聖人。慧解脫與俱解脫阿羅漢聖者的主要差異在於有沒有證得四禪八定等禪定。

但[[瑜伽行派|瑜伽行派]]認為，以智慧解脫的阿羅漢聖者，至少還是要有近初禪定的定力才能得解脫<ref>《瑜伽師地論》卷94：「復次若有苾芻，具淨尸羅，住別解脫清淨律儀，增上心學增上力故，得初靜慮近分，所攝勝三摩地，以為依止。增上慧學，增上力故，得法住智及涅槃智。用此二智以為依止，先由四種圓滿，遠離受學轉時，令心解脫一切煩惱，得阿羅漢，成慧解脫。」</ref>而俱解脫阿羅漢則必須是同時擁有四禪定與解脫智慧。

現代南傳上座部佛教將[[慧解脫|慧解脫]]者聯繫於其聖典註釋體系中的純觀行者（{{IAST|Sukkha-vipassaka}}）。

==  十六羅漢 ==
{{main|十六羅漢}}

'''[[十六羅漢|十六羅漢]]'''（或稱'''十六阿羅漢'''、'''十六尊者'''）是[[釋迦牟尼|釋迦牟尼]]的得道弟子。十六羅漢的名字早有[[佛經|佛經]]所載。後來十六羅漢傳入[[中國|中國]]後，約於[[唐朝|唐末]]至[[五代十國|五代十國]]時期演变為[[十八罗汉|十八罗汉]]。

== 十八罗汉 ==
{{main|十八羅漢}}

*漢傳佛教中加上另外兩位得道者成為十八羅漢:
**[[降龍羅漢|降龍羅漢]]：[[濟公|濟公]]
**[[伏虎羅漢|伏虎羅漢]]：[[玄奘|玄奘]]
一說是因為漢人認為9為吉祥的數字，所以將原本的十六羅漢加上兩位成為十八位。
