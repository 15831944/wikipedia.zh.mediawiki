'''阿利坎特'''（{{lang-es|'''Alicante'''}}；{{lang-va|'''Alacant'''}}）是[[西班牙|西班牙]][[巴倫西亞自治區|巴倫西亞自治區]]，[[阿利坎特省|阿利坎特省]]的首府，在[[地中海|地中海]]旁，是重要的海港，有西班牙最古老的鐵路連接。

它亦是[[歐盟商標局|歐盟商標局]]總部所在地。

== 歷史 ==
{{來源請求|前324年[[希臘人|希臘人]]將它命名為“Akra Leuke”，意即「白色的山峰」。前201年[[羅馬人|羅馬人]]入侵，稱它為“Lucentum”，據說[[漢尼拔|漢尼拔]]曾將其[[戰象|戰象]]卸於此。718年至1249年，[[摩爾人|摩爾人]]管治。1265年被[[阿方索十世|阿方索十世]]佔領，將它併入[[卡斯蒂利亚王国|卡斯蒂利亚王国]]。1304年併入[[巴倫西亞王國|巴倫西亞王國]]。1709年及1812年被[[法國|法國]]圍困，1873年又被[[喀塔赫納|喀塔赫納]]的聯邦主義者所困。在[[西班牙內戰|西班牙內戰]]中，阿利坎特是其中一個最後期被[[弗朗西斯科·佛郎哥|弗朗西斯科·佛郎哥]]攻佔的城市。}}

== 經濟 ==
阿利坎特是西班牙其中一個發展得快的城市。其經濟以旅遊、造酒、服務業和行政為主。出口品有[[酒|酒]]、[[橄欖油|橄欖油]]及[[水果|水果]]。它有[[輕工業|輕工業]]如食品加工、[[皮革|皮革]]、紡織和[[陶器|陶器]]。特色小食有[[Turrón|Turrón]]。

== 語言 ==

20世紀初，阿利坎特是[[加泰羅尼亞語|加泰羅尼亞語]]城市。[[西班牙內戰|西班牙內戰]]，有一些使用西班牙語的人遷進此。[[弗朗西斯科·佛郎哥|弗朗西斯科·佛郎哥]]統治期間，許多巴倫西亞人離開，不但令西班牙語在該處站穩陣腳，而且這個城市的加泰羅尼亞語名''Alacant''不再是官方名稱，使這個城市長久稱作''Alicante''。今日，兩種語言均是官方語言，地方政府的強烈推廣之下，Alacant重新成為官方名稱之一。雖然西班牙語使用者仍佔大多數，但加泰羅尼亞語的使用者數目正漸漸成長。
