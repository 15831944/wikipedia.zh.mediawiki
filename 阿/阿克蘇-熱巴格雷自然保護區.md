'''阿克蘇-熱巴格雷自然保護區'''（{{lang-ru|Aксу-Жабаглинский заповедник}}、{{lang-kz|Ақсу-Жабағылы қорығы}}、{{lang-en|Aksu-Zhabagly Nature Reserve}}）是[[中亞|中亞]]一處歷史悠久的自然保護區，其成立於1926年，坐落在[[天山|天山]]山脈西北部一帶，由[[哈萨克斯坦|哈薩克]][[南哈薩克斯坦州|南哈薩克斯坦州]]管轄。其名來自保護區內最大的河流－阿克蘇河，以及保護區北邊的主要山脈－熱巴格雷山脈。2002年，阿克蘇-熱巴格雷自然保護區被列入哈薩克世界遺產名錄預選名單。<ref>{{cite web |url=http://whc.unesco.org/en/tentativelists/1683/ |title=State National Natural Park "Altyn-Emel" |publisher=[[UNESCO|UNESCO]] |accessdate=2017-08-21}}</ref>

== 地里位置 ==
阿克蘇-熱巴格雷自然保護區位於[[天山|西部天山山脈]]的塔拉斯山（Talasskiy Alatau）西部，此處為[[吉爾吉斯|吉爾吉斯]]與哈薩克的東邊國界，南邊則是與[[烏茲別克|烏茲別克]]分界的Maidantal河谷。保護區的西邊外部為繞著[[克孜勒库姆沙漠|克孜勒库姆沙漠]]，北邊則是[[卡拉套山|卡拉套山]]。天山山脈從東方橫越保護區，延伸至保護區的南方，經過[[南哈薩克斯坦州|南哈薩克斯坦州]]的Tyul’kubaskiy-與Lengerskiy-Rayon。本自然保護區面積達{{convert|131.934|ha}}<ref>{{cite web |language=kk |url=http://www.oopt.kz/forest/natural_zapovednik/aksu-zhabagly/ |title=Aksu-Zhabaglinskiy Zapovednik |archiveurl=https://web.archive.org/web/20100710064355/http://www.oopt.kz/forest/natural_zapovednik/aksu-zhabagly/ |archivedate=2010-07-10 |date=2010-07-10 |publisher=Ministry of Agriculture of Republic of Kazakhstan |accessdate=2010-11-18}}</ref>，[[平均海拔|平均海拔高度]]從{{convert|1100|to|4236|m}}不等。園區內主要的山脈有北部的熱巴格雷山（Zhabagly）、中部的塔拉斯山、南部的贝尔黛尔拜克山（Baldybrektau）和Ugamskiy Alatau。多山的地形形成許多陡峭的山坡和平緩的高原。這些山脈由Jabagly、阿克蘇河（Aksu）、贝尔黛尔拜克河劃分，而這些河流全部向西匯入[[錫爾河|錫爾河]]，然後流入[[鹹海|鹹海]]。阿克蘇河流經此處時，形成了{{convert|500|m}}深的{{le|阿克蘇峽谷|Aksu Canyon}}，要前往保護區南邊只能從狹窄的人行橋上跨越峽谷而過。<ref>{{cite web |url=http://www.greentours.co.uk/doc/Trip%20Report%20Tien%20Shan%202011%20edited.pdf |title=Tien Shan Wildlife Tour Report Botanical Butterfly Birdwatching Mammals Holiday Kazakhstan Bears - Trip Report Tien Shan 2011 edited |author=Oron Peri|publisher=Greentours |accessdate=2017-11-01}}</ref>

== 地質 ==
古生代[[石灰岩|石灰岩]]、[[白雲石|白雲石]]和碳質[[凝灰岩|凝灰岩]]是保護區內{{le|床岩|Bedrock}}最主要的組成成分。一層厚厚的[[黃土|黃土]]堆積在高原和山腳下，但[[二疊紀|二疊紀]]時代的矽質岩漿組成的床岩在此區卻很少出現。<ref>Gosudarstvennaya Geologicheskaya Karta SSSR: K-42-XVII (Vannovka), Severo-Tyan’shanskaya seriya. Masshtab 1:200 000. (K-42-XVII [Vanovka], 1989.</ref>

== 植物相與植披 ==
阿克蘇-熱巴格雷自然保護區內目前共發現1312種[[維管束植物|維管束植物]]<ref>A.F. Kovshar (Ed.): ''Monitoring biologicheskogo raznoobraziya Zapovednika Aksu-Dzhabagly''. In: ''Tethys Biodiversity Research''. Tom 1, S. 16. </ref>，其中一些大型的植物種類包括[[黃芪|黃芪]]、[[頂冰花屬|頂冰花屬]]、[[蔥屬|蔥屬]]、[[薹草屬|薹草屬]]、[[棘豆屬|棘豆屬]]等。此區有44種珍稀植物被列入[[國際自然保護聯盟瀕危物種紅色名錄|國際自然保護聯盟瀕危物種紅色名錄]]<ref>A.F. Kovshar (Ed.): ''Monitoring biologicheskogo raznoobraziya Zapovednika Aksu-Dzhabagly''. In: ''Tethys Biodiversity Research''. Tom 1, S. 17-21.</ref>，包括[[新疆野蘋果|野蘋果]]、[[釀酒葡萄|釀酒葡萄]]、以及一部分的野[[鬱金香|鬱金香]]物種（如小红帽鬱金香（Tulipa greigii））。園區內地勢較低的區域覆蓋著半沙漠、灌木叢、草原等植披，在春季時期時，多樣化的植披形成一豐富多彩的大地。山地區域散布著鞍檜尖柏檜與{{le|天山圓柏|Juniperus semiglobosa}}等刺柏屬組成的森林相，以及歐洲西伯利亞和中亞植物系組成的[[草甸|草甸]]，將整個山區地帶裝飾成天然的馬賽克。阿克蘇峽谷有一些森林碎片與Malus sieversii和土耳其斯坦楓葉（Acer turkestanicum）。{{le|阿克蘇峽谷|Aksu Canyon}}一帶散布稀疏的森林，森林組成成分包括[[新疆野蘋果|野蘋果]]與[[挪威楓|挪威楓]]屬的土耳其楓（Turkestan Maple）等。在:亞高山帶和高山帶，分布著草甸、高[[草本植物|草本植物]]、Onobrychis echidna等葉柄植物群、以及[[黑花薹草|黑花薹草]]與[[波斯嵩草|波斯嵩草]]組成的矮草原群。[[柳樹|柳樹]]與[[樺木屬|樺木屬]]林主要沿著河流分布，並常常出現碎石坡植被，此處有著稀有的{{le|寬葉蔥|Allium karataviense}}、{{le|威爾莫特鳶尾|Iris willmottiana}}、土耳其鬱金香（Tulipa kaufmanniana）等物種。<ref>{{cite web |title=Celestial Silk Road 5th-21st June 2016 |url=http://www.viranatura.com/Product_100001_Celestial-Silk-Road-5th-21st-June-2016 |publisher=viranatura.com |accessdate=2015-05-26}}</ref>

== 動物相 ==
保護區內目前有44種哺乳動物因高度瀕危而被列入[[國際自然保護聯盟瀕危物種紅色名錄|國際自然保護聯盟瀕危物種紅色名錄]]。<ref>A.F. Kovshar (Ed.): ''Monitoring biologicheskogo raznoobraziya Zapovednika Aksu-Dzhabagly''. In: ''Tethys Biodiversity Research''. Tom 1, S. 21-23.</ref>保護區內最顯著的動物是[[雪豹|雪豹]]，但在觀察上因較稀少而少有被人類發現。保護區難較常見的物種有[[喜瑪拉雅棕熊|喜瑪拉雅棕熊]]、[[盤羊|天山盤羊]]（學名：Ovis ammon karelini）、{{le|印度豪豬|Indian crested porcupine}}（學名：Hystrix indica）、 {{le|明氏旱獺|Menzbier's marmot}}等。<ref>{{cite web |url=http://aksu-jabagly.kz/eng/gallery/ |title=From Red data book |publisher=Aksu-Zhabagly Nature Reserve official website |accessdate=2017-11-01}}</ref><ref>{{cite web |url=http://aksubaza.kz/index.php?id=1262&Itemid=&lang=en&option=com_content&view=article |title=Aksu-Zhabagly Nature Reserve |publisher=aksubaza.kz |accessdate=2017-11-01}}</ref>
