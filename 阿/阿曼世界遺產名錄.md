由於[[文化遺產|文化遺產]]和[[自然遺產|自然遺產]]越來越受到破壞威脅，[[聯合國教科文組織|聯合國教科文組織]]大會於1972年10月17日至11月21日在巴黎舉行第十七屆會議<ref name=convention>{{cite web |url=http://whc.unesco.org/en/convention/ |title=The World Heritage Convention |publisher=UNESCO |accessdate=2010-09-21}}</ref>，並於11月16日在法國巴黎通過《[[保護世界文化和自然遺產公約|保護世界文化和自然遺產公約]]》。阿曼在1981年10月6日批准《保護世界文化和自然遺產公約》，並在1987年開始有歷史遺蹟被列入世界遺產；截至2017年，阿曼已有4處世界遺產，而這4處皆為文化遺產，還有8處在預選名單內。<ref name="Oman">{{cite web|url=http://whc.unesco.org/en/statesparties/om |title=Oman|publisher=UNESCO|accessdate=2017-08-20}}</ref>最先登錄的阿曼世界遺產是[[巴赫萊要塞|巴赫萊要塞]]，該遺跡於1987年被聯合國教科文組織列入世界遺產名錄中。<ref>{{cite web|url=http://whc.unesco.org/archive/1987/sc-87-conf005-9_e.pdf |title=11th session of the Committee |publisher=UNESCO|accessdate=2017-08-20}}</ref>近期登錄的阿曼世界遺產則是[[阿夫拉賈灌溉體系|阿曼的阿夫拉賈灌溉體系]]，該遺跡於2006年被聯合國教科文組織列入世界遺產名錄中。<ref>{{cite web|url=http://whc.unesco.org/en/news/267/ |title=Ancient irrigation system (Oman) and Palaces of Genoa (Italy) among ten new sites on World Heritage List|publisher=UNESCO|date=2006-07-13|accessdate=2017-08-20}}</ref>

然而在2007年前，[[阿拉伯羚羊保護區|阿拉伯羚羊保護區]]曾是阿曼第5處世界遺產，但是因阿曼政府將保護區面積縮減9成，使該保護區名存實亡，在第31屆[[世界遺產委員會|世界遺產委員會]]會議中被剃除於世界遺產名錄之外，這也是世界上首處被除名的世界遺產。<ref name="OryxD">{{cite web |url=http://whc.unesco.org/en/news/362/ |title=Oman's Arabian Oryx Sanctuary : first site ever to be deleted from UNESCO's World Heritage List |publisher=UNESCO |date=2007-06-28 |accessdate=2017-08-20}}</ref><ref name="OryxD2">{{cite web |url=http://www.reuters.com/article/us-oman-oryx-unesco-idUSL3065930320070630 |title=UNESCO removes Oman oryx sanctuary from heritage list |publisher=[[路透社|路透社]] |date=2007-06-30 |accessdate=2017-08-20}}</ref>
