'''阿夫瑪道區'''（{{lang-so|''Degmada Afmadoow''}}）是[[索馬利亞|索馬利亞]]的一個[[索馬利亞行政區劃|區]]，位於該國南部的[[下朱巴州|下朱巴州]]<ref>[http://www.statoids.com/yso.html Districts of Somalia], Statoids</ref>，首府為[[阿夫瑪道|阿夫瑪道]]。

阿夫瑪道區是[[下朱巴州|下朱巴州]]最老的區之一，最早於1896年設立城鎮。本區的居民主要是[[歐加登|歐加登族]]與{{tsl|en|Marehan|馬雷漢族}}，並以飼養[[家畜|家畜]]為主要經濟活動。

== 城鎮 ==
阿夫瑪道區下轄五個城鎮：
# Afmadow
# Billis Qooqaani
# Dhoobley
# Tabto
# Xayo
