'''阿克赛钦地区'''，位于[[中华人民共和国|中华人民共和国]][[新疆维吾尔自治区|新疆]]、[[西藏自治区|西藏]]两[[自治区|自治区]]与[[印度|印度]]的[[边界|边界]]西段，阿克赛钦的词源来自突厥语。中国拥有本地区的实际控制权，绝大部分属于[[新疆维吾尔自治区|新疆]][[和田地区|和田地区]]管辖，南部很小一部分属于[[西藏自治区|西藏自治区]][[阿里地区|阿里地区]]管辖。印度方面则认为此地属于印度控制的[[克什米尔|克什米尔]]地区。阿克赛钦地区总面积4万2685平方公里，其中争议地区约3万平方公里，佔中印西段边境争议地区总面积的3万3500平方公里的绝大部分。1962年，中印在阿克赛钦爆发了一场小规模战争。在1993年和1996年，两个国家签订协议表示尊重实际控制线。<ref name=GlobalSecurity>{{cite web |url=http://www.globalsecurity.org/military/world/war/india-china_conflicts.htm |title=India-China Border Dispute |publisher=GlobalSecurity.org}}</ref>

==名称==
“阿克赛钦”一词由[[突厥语|突厥语]]音译而来，其中“阿克赛”意为“白石滩”，但“钦”的意思有争议，有「通道」或「中國」（欽為秦，意思是指中國）等說法。一說是「白石灘通道」，代表此為古代新疆、西藏與喀什米爾地區的貿易管道。在中国国内則認為是“中国的白石滩”。[[印地语|印地语]]中以{{lang|hi|अक्साई चिन}}表示本地区。

==地理==
阿克赛钦地区面积37,244平方公里，其中大部分为高海拔的荒漠，位于喀拉喀什河的最低点海拔4,300米。西南部海拔6,900米的高山延伸至西南部的[[达普桑盆地|达普桑盆地]]，形成了分割阿克赛钦和印控克什米尔的实际控制线。

在北部，[[昆仑山脉|昆仑山脉]]切断了阿克赛钦和[[塔里木盆地|塔里木盆地]]。

阿克赛钦内部有一些形成了咸水湖或[[碱湖|碱湖]]的[[内流盆地|内流盆地]]，如[[阿克赛钦湖|阿克赛钦湖]]、萨利吉勒、腾格湖等。[[塔里木河|塔里木河]]流进本区西部，阿克赛钦湖由[[阿克赛钦河|阿克赛钦河]]注入，湖面海拔4990米。萨利吉勒湖，湖面为5181米。盆地内的山峰大多超过6000米，山口大多超过5000米。<ref>[http://www.educast.com.cn/show/er.htm?ID=2007-1203110419-934 阿克赛钦盆地] {{webarchive|url=https://web.archive.org/web/20140912222055/http://www.educast.com.cn/show/er.htm?ID=2007-1203110419-934 |date=2014-09-12 }}[http://baike.baidu.com/link?url=kQ6Oe5IKkLumvtMvckPH9nSwjJjbO2DkJugtB3hQzCnu6d44Pxqciwec2MPVEPOTBm841MmRkZZR3MoREBxmwa]</ref>

== 气候 ==
阿克赛钦海拔较高，高寒气候。以康西瓦为例，年平均温度为-0.6℃，1月平均温度为-11.3℃，7月平均温度为9.8℃，无霜期10天，10℃以上活动积温为20℃。天文点年平均温度为-9.8℃，1月平均温度为- 21.0℃，7月平均温度为3℃。盆地位于雨影面，西南季风很难越过喜马拉雅山进入，西风气流也难越过喀喇昆仑山和西昆仑山，实测年降水量均不到100毫米，康西瓦36毫米，天文点47毫米，空喀山口仅29毫米。

==历史==
元朝后期，准噶尔的大军及蒙古熬茶礼佛的旅行都是从阿克赛钦进入西藏，清朝雍正时在此设立拓置局管辖。1846年英国吞并查漠—克什米尔后，声称拉达克（原本属西藏政府管辖的地区，现为克什米尔的一部分）应归英印政府统治。在未等清朝政府同意的情况下，英国便占领拉达克地区。后来，為了标定拉达克和西藏的边界，英国于1865年派遣印度测量局官员{{le|威廉·詹森 (印度)|William Johnson (surveyor)|威廉·詹森}}潜入南疆地区，从拉达克进入阿克赛钦最后到达新疆的和田，通过勘察，绘制了一条界线，这就是约翰逊线。此线将包括阿克赛钦在内的近三万平方公里的土地划给了英属印度，使阿克赛钦变成英印克什米尔的一部分，但當時由於[[同治陕甘回变|同治陕甘回变]]，清政府並未控制此地區，英国政府沒有知会清政府。1899年，英國駐大清國公使[[竇納樂|竇納樂]]向清政府遞交了一份[[馬繼業-竇訥樂線|新疆和克什米爾劃界方案]]，將阿克赛钦畫給清朝，清政府沒有回應，英方將此視為默許。<ref name=Calvin>{{cite web| last =Calvin| first =James Barnard| date = April 1984| url = http://www.globalsecurity.org/military/library/report/1984/CJB.htm| title =The China-India Border War| publisher = Marine Corps Command and Staff College| accessdate =2011-10-14}}</ref>

1950年代后期，中国在阿克赛钦地区修建[[新藏公路|新藏公路]]，印度以约翰逊线为依据表达不满，中印西段边界争端由此发酵。

==战略意义==
连接新疆[[叶城|叶城]]和西藏[[拉孜|拉孜]]的[[新藏公路|新藏公路]]穿过阿克赛钦地区，在战略上这一地区有重要的意义。由于目前中印两国在尊重彼此既得领土的意见上达到了一致，因此大大降低了这一地区再次发生领土争端的可能性。
