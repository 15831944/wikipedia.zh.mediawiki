'''阿德里亚娜·拉斯特拉·费尔南德斯'''（{{lang-es|Adriana Lastra Fernández}}，{{bd|1979年|3月30日||}}）是[[阿斯图里亚斯|阿斯图里亚斯]]出身的[[西班牙工人社会党|西班牙工人社会党]]女性[[政治人物|政治人物]]。2017年6月被任命为西班牙工人社会党副书记。2018年任工人社会党在西班牙众议院[[发言人|发言人]]。

==经历==
1979年出生于[[阿斯图里亚斯|阿斯图里亚斯]][[里瓦德塞利亚|里瓦德塞利亚]]。父亲是[[出租车司机|出租车司机]]，母亲是[[理发师|理发师]]<ref>{{cite web |url=http://www.asturiasdiario.es/web/?p=19248|author=David Ascaso |title='Vividores' de la política |publisher=Asturias Diario |accessdate=2019-02-24 |language=es |date=2014-12-01}}</ref>。学习过[[文化人类学|文化人类学]]课程<ref>{{cite web |title=La comentada respuesta de Adriana Lastra en 'LaSexta Noche' al ser preguntada por su currículum |url=https://www.huffingtonpost.es/2018/09/23/la-comentada-respuesta-de-adriana-lastra-en-lasexta-noche-al-ser-preguntada-por-su-curriculum_a_23538823/?utm_hp_ref=es-adriana-lastra |publisher=El[[HuffPost|HuffPost]] |accessdate=2019-02-24 |language=es |date=2018-09-23}}</ref>，但因参与政治活动中途退学，未获得大学学位<ref name=elpais/>。18岁加入阿斯图里亚斯社会主义青年团里瓦德塞利亚支部，1999年至2004年任支部总书记。从2000年开始，担任西班牙工人社会党-[[阿斯图里亚斯社会主义联盟|阿斯图里亚斯社会主义联盟]]（FSA-PSOE）区域执行委员会成员。2007年当选[[阿斯图里亚斯自治区政府|阿斯图里亚斯自治区政府]]议会议员，直到2015年离任<ref>{{cite web |author=Marije Amieva |title=Elecciones y "elegidos" |url=https://elfielato.es/opinion/elecciones-y-elegidos-1739 |website=elfielato.es |accessdate=2019-02-24 |archiveurl=https://web.archive.org/web/20180825073936/https://elfielato.es/opinion/elecciones-y-elegidos-1739 |archivedate=2018-08-25 |language=es |date=2015-09-29}}</ref>。

2014年至2016年担任西班牙工人社会党[[地方政府|地方政府]]部门书记<ref>{{cite web |author=[[埃菲通讯社|EFE]] |title=Adriana Lastra pide prudencia y evitar tensiones en las agrupaciones locales |url=https://www.lne.es/asturias/2014/07/29/adriana-lastra-pide-prudencia-evitar/1621459.html |publisher=Editorial Prensa Asturiana, S.A |accessdate=2019-02-24 |language=es |date=2014-07-29}}</ref>。她深受[[佩德罗·桑切斯|佩德罗·桑切斯]]的赏识<ref>{{cite web |author1=Luis Ángel Sanz |title=Adriana Lastra será 'número dos' del PSOE |publisher=[[世界报_(西班牙)|El Mundo]]|url=https://www.elmundo.es/espana/2017/06/15/5942e7c146163f58188b4691.html |accessdate=2019-02-24 |location=Madrid |language=es |date=2017-06-15}}</ref>，于2017年6月被任命为[[西班牙工人社会党|西班牙工人社会党]]副书记<ref>{{cite web |author1=Anabel Díez|author2=José Marcos |title=Adriana Lastra: "La bicefalia es un buen modelo" |url=https://elpais.com/politica/2017/06/19/actualidad/1497882457_900211.html |publisher=[[国家报|El País]] |accessdate=2019-02-24 |language=es |date=2017-06-20}}</ref>。

2018年6月接替玛格丽塔·罗夫莱斯，担任西班牙工人社会党在众议院的发言人<ref name=elpais>{{cite web |author=Elsa García de Blas |title=Adriana Lastra será la nueva portavoz socialista en el Congreso |url=https://elpais.com/politica/2018/06/04/actualidad/1528129875_871607.html |accessdate=2019-02-24|publisher=El País |language=es |date=2018-06-04}}</ref>。
