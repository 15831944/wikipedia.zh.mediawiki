'''阿萊克西斯·齊普拉斯'''<ref>安格罗斯·克里索格罗斯, 高歌. 欧洲危机侵蚀欧盟对外政策[J]. 国际经济评论, 2015(4):171-172.{{ISSN|1007-0974}}</ref>（{{lang-el|'''Αλέξης Τσίπρας'''}}，轉寫： '''Alexis Tsipras'''，{{bd|1974年|7月28日|catIdx=Tsipras}}），現任[[希臘|希臘]][[希臘总理|总理]]兼[[外交部長|外交部長]]，[[激進左翼聯盟|激進左翼聯盟]]领袖。

== 簡介 ==
齊普拉斯出生於希腊[[雅典|雅典]]，學生時期是[[希腊共产党|希腊共产党]]的[[青年翼|青年翼]][[希腊共产主义青年|希腊共产主义青年]]成員，2000年畢業於[[國立雅典理工大學|國立雅典理工大學]]的[[土木工程|土木工程系]]，畢業後投身建築業擔任土木工程師。

== 投身政界 ==
2006年，32歲的齊普拉斯正式踏入政壇，在地方選舉中獲得11%的選票。

== 出任总理 ==
[[2012年5月希臘議會選舉|2012年5月希臘議會選舉]]，他帶領[[激进左翼聯盟|激进左翼聯盟]]成為議會第二大黨，取得52席。5月8日，在第一大黨保守派[[新民主黨_(希臘)|新民主黨]]領袖[[安東尼斯·薩馬拉斯|安東尼斯·薩馬拉斯]]組閣失敗，[[希臘總統|希臘總統]][[卡羅洛斯·帕普利亞斯|卡羅洛斯·帕普利亞斯]]授權他組建聯合政府，但在2012年5月9日傍晚，齊普拉斯宣布他未能組成聯合政府。[[2012年6月希臘議會選舉|2012年6月希臘議會選舉]]，取得71席，此後成為最大反對黨領袖。

[[2015年希臘国會選舉|2015年希臘国會選舉]]中，[[激进左翼聯盟|激进左翼聯盟]]取得压倒性胜利，與于翌日宣誓就任[[希臘总理|总理]]。

40歲的齊普拉斯是希臘150年來最年輕總理，出身草根階層的他，出席宣誓儀式時沒有打上領帶，同時一改傳統，沒有進行宗教的宣誓儀式，改為進行民事宣誓。他較早前在勝利演說表示：「希臘正離開災難性的緊縮、恐懼和威權主義，離開五年屈辱和痛苦。」<ref>[http://hk.apple.nextmedia.com/international/first/20150127/19017972 左聯上台　迅速組執政聯盟 向緊縮說不　希臘促撇債一半]2015年1月27日</ref>

== 個人生活 ==
齊普拉斯與女友同居多年，育有兩子。兩人在1987年在[[希腊共产主义青年|希腊共产主义青年]]的组织中認識。

齊普拉斯自稱是[[無神論|無神論]]者，在宣誓就任總理時，並無像過去的希臘總理一樣由希臘[[東正教|東正教]]大主教和按[[聖經|聖經]]宣誓。
