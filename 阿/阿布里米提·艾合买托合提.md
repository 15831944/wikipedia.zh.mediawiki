'''阿布里米提·艾合买托合提'''（{{bd|？|||}}）[[维吾尔族|维吾尔族]]，[[新疆|新疆]][[和田|和田]]人，[[中华人民共和国|中华人民共和国]][[伊斯兰教|伊斯兰教]][[伊玛目|伊玛目]]。

==生平==
阿布里米提·艾合买托合提曾任[[和田市|和田市]][[团结清真寺_(和田市)|团结清真寺]]哈提甫，任内于2013年获得“第二届全国创建和谐寺观教堂先进个人”荣誉称号<ref>{{Cite web |url =  http://mw.jl.gov.cn/zt_mw/hxsgjt/aqnhd/201404/t20140401_1641009.html|title =第二届全国创建和谐寺观教堂先进个人名单  |author =  |date =2014-02-20   |publisher =吉林省民族宗教网  |language =  |accessdate =  |quote =  }}</ref>。此后他出任和田市[[伊提帕克清真寺|伊提帕克清真寺]][[伊玛目|伊玛目]]<ref>{{Cite web |url =http://www.rmzxb.com.cn/qmt/459468.shtml  |title = 法蒂玛·马合木提委员：调研归来话南疆 |author =  |date =2015-03-06  |publisher =  人民政协网|language =  |accessdate =  |quote =  }}</ref>。

2013年，他出任[[第十二届全国政协委员|政协第十二届全国委员会宗教界委员]]<ref>{{Cite web|url = http://www.xjqnpx.com.cn/news/48832.html|title = 住疆全国政协委员履职忙|author = |date = 2016-03-03|publisher = 新疆青年网|language = |accessdate = |quote = }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2013年3月，在全国政协十二届一次会议上，他准备了20多个提案，内容主要关系到农牧民切身利益<ref>{{Cite web |url =http://news.ifeng.com/gundong/detail_2013_03/03/22679784_0.shtml  |title = 20多个提案关乎农牧民切身利益 |author =  |date = 2013-03-03 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

2017年8月，鉴于阿布里米提·艾合买托合提涉嫌犯罪，根据[[中共中央|中共中央]]的建议，依照《[[中国人民政治协商会议|中国人民政治协商会议]]章程》以及《政协全国委员会常务委员会关于授权主席会议对违纪违法政协委员及时作出处理的决定》，政协第十二届全国委员会第六十三次主席会议作出撤销阿布里米提·艾合买托合提中国人民政治协商会议第十二届全国委员会委员资格的决定，并向社会公布。2017年8月30日，政协第十二届全国委员会常务委员会第二十二次会议追认了上述决定<ref>{{Cite web |url = http://www.cppcc.gov.cn/zxww/2017/08/31/ARTI1504146699355118.shtml |title = 关于撤销阿布里米提·艾合买托合提中国人民政治协商会议第十二届全国委员会委员资格的决定 |author =  |date = 2017-08-31  |publisher = 中国政协网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.thepaper.cn/baidu.jsp?contid=1780258 |title = 李军辞去全国政协委员，阿布里米提·艾合买托合提被撤销资格 |author =  |date = 2017-08-31  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
