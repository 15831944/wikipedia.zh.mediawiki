'''阿尔蒂尔一世'''（{{bd|1187年|3月29日|1203年|4月3日}}），英语作'''亚瑟一世'''（Arthur I），在1196-1203年间为第四代里奇蒙伯爵、[[布列塔尼公爵|布列塔尼公爵]]。他是前公爵[[若弗鲁瓦二世_(布列塔尼)|若弗鲁瓦二世]]（英语作乔弗雷）和女公爵[[康斯坦丝_(布列塔尼)|康斯坦丝]]的遗腹子。若弗鲁瓦是[[亨利二世_(英格兰)|亨利二世]]之子。若弗鲁瓦年幼于[[理查一世_(英格兰)|理查一世]]，但年长于[[无地王约翰|约翰]]。

1190年，他被伯父[[英格兰国王|英格兰国王]]理查一世立为王储，<ref>MP, Vol. II, 1190, p. 364.</ref>此举意味着理查的继承人将是阿尔蒂尔，而不是理查的弟弟约翰。阿尔蒂尔被囚禁于鲁昂城堡后自1203年起失去记载，真实命运不详，传统上认为是被约翰杀害。

== 早年生活 ==
阿尔蒂尔生于1187年，是布列塔尼公爵若弗鲁瓦二世和布列塔尼的康斯坦丝之子。若弗鲁瓦在阿尔蒂尔出生前已去世。婴儿时的阿尔蒂尔是祖父亨利二世国王的第二顺位继承人，在伯父理查之后。阿尔蒂尔2岁时，亨利二世去世，理查继位。

当理查一世在1189－1192年参与[[第三次十字军东征|第三次十字军东征]]时，康斯坦丝寻求让布列塔尼公国日趋独立。1190年11月11日，理查立阿尔蒂尔为储君，{{sfn|Carley|Riddy|1998|p=28}}并在和西西里国王[[坦克雷迪|坦克雷迪]]的和约中约定后者将女儿许给阿尔蒂尔。{{sfn|McAuliffe|2012|p=74}}但1194年神圣罗马帝国皇帝[[亨利六世_(神圣罗马帝国皇帝)|亨利六世]]征服西西里，阿尔蒂尔的婚约沦为一纸空文。

后来，理查一世为了和[[法兰西|法兰西]]国王[[腓力二世_(法兰西)|腓力二世]]结盟，曾将阿尔蒂尔的姐姐[[埃莉诺_(布列塔尼)|埃莉诺]]许给腓力的太子[[路易八世_(法兰西)|路易]]，但这次婚约也流产了，这被视为理查不再想以阿尔蒂尔为继承人的前兆。1196年，康斯坦丝以9岁的阿尔蒂尔为共治公爵。理查召阿尔蒂尔和他的母亲康斯坦丝{{sfn|Everard|2004|p=146}}来诺曼底。但阿尔蒂尔的继父[[第六代切斯特伯爵拉努尔夫·布隆德维尔|第六代切斯特伯爵拉努尔夫·布隆德维尔]]却劫持了康斯坦丝。理查愤怒地亲自率军进军布列塔尼来救阿尔蒂尔，阿尔蒂尔则被教师秘密送到法国和路易太子一同抚养。

[[File:Philippe2+Arthur.jpg|thumb]]。]]

1199年4月6日，理查一世驾崩，因害怕阿尔蒂尔太年轻不能为王，遗嘱由约翰继位。阿尔蒂尔时年12岁，深受腓力二世的影响。约翰宣布即位，但多数法国贵族憎恶他，不愿意承认他为自己的宗主，他们更喜欢阿尔蒂尔。此时的阿尔蒂尔已经自称为腓力二世的封臣。腓力二世承认阿尔蒂尔在[[安茹|安茹]]、[[曼恩|曼恩]]和[[普瓦图|普瓦图]]的权利。阿尔蒂尔率军进入安茹和曼恩。{{sfn|Everard|2004|p=167}}他从4月18日起自称布列塔尼公爵、安茹伯爵、里奇蒙伯爵。

9月18日，约翰称亚瑟将成为卡佩王朝的傀儡，说服安茹管家吉约姆·罗歇倒戈。四天后，吉约姆将阿尔蒂尔和康斯坦丝劫持到[[勒芒|勒芒]]。而约翰任命的安茹管家艾梅里子爵则将阿尔蒂尔和康斯坦丝带离，逃到[[昂热|昂热]]，再逃到腓力二世的朝廷。{{sfn|Gillingham|1984|p=88}}

== 勒古莱和约 ==
勒古莱和约是英格兰国王约翰和法兰西国王腓力二世于1200年5月签订的，意在一次了结英格兰的诺曼国王作为诺曼底公爵对包括布列塔尼在内的在法国大陆的领地的权利。在和约规定下，腓力承认约翰作为其兄理查一世的继承人、英格兰国王，正式放弃了对布列塔尼公爵阿尔蒂尔一世的王位继承权的支持。同时约翰认可腓力为安茹帝国在大陆领地的宗主。

腓力之前认可约翰为安茹和布列塔尼公国的宗主，但他敲诈了20,000马克英国银币作为认可约翰对布列塔尼宗主权的报酬。

== 对战约翰王 ==
勒古莱和约签订后，阿尔蒂尔感到受到腓力的冒犯，投奔叔父约翰，约翰也和善地待他。但出于警觉，阿尔蒂尔又离开约翰，来到[[昂热|昂热]]。一说1202年4月，阿尔蒂尔和腓力二世的女儿[[法兰西的玛丽，布拉班公爵夫人|玛丽]]订婚。{{sfn|McDougall|2017|p=226}}

阿尔蒂尔回到法国后，在腓力二世支持下于1202年进军诺曼底讨伐约翰。普瓦图叛乱响应阿尔蒂尔。布列塔尼公爵将祖母即约翰的母亲[[阿基坦的埃莉诺|阿基坦的埃莉诺]]围困在米拉博城堡。约翰进军米拉博，于7月31日奇袭阿尔蒂尔。{{sfn|Seel|2012|p=47}}8月1日，阿尔蒂尔被约翰部下诸侯俘获，并被关押在[[诺曼底|诺曼底]]的法莱城堡。

== 被囚及失踪 ==
在法莱城堡，他被[[休伯特·伯格|休伯特·伯格]]看守。据同时代的编年史家科热沙尔的拉尔夫称，约翰命两个仆人将公爵弄瞎、阉割，休伯特·伯格拒绝他们这么做。为瓦解阿尔蒂尔的支持者，休伯特·伯格将宣布他死亡。阿尔蒂尔的姐姐埃莉诺也已被俘，当时被关押在[[多塞特郡|多塞特]]的科弗城堡。第二年，阿尔蒂尔被转移到鲁昂，由[[第四代布兰伯领主威廉·布劳斯|威廉·布劳斯]]看守。4月，阿尔蒂尔神秘地失踪了。

[[File:Prince_Arthur_and_Hubert_de_Burgh.jpg|thumb]]。]]

阿尔蒂尔失踪之谜引发了诸多猜测。一种版本是看守阿尔蒂尔的狱卒们不敢伤害他，所以约翰直接谋害了阿尔蒂尔并将其沉尸[[塞纳河|塞纳河]]。[[马加姆年表|马加姆年表]]则对阿尔蒂尔之死作了如下描述：

{{quote|约翰王俘虏了阿尔蒂尔并将他投入鲁昂的监狱有些日子了，在复活节前一天的星期四的晚饭后，他趁着自己喝醉了，任由恶魔摆布，亲手杀害了阿尔蒂尔，并且把尸体系上一块沉重的石头沉入塞纳河。尸体被一个渔民在渔网中发现，被拖到岸边后被认出来了。出于对暴君的恐惧，尸体被秘密带到[[巴黎圣母院|巴黎圣母院]]的贝克修道院埋葬了。”}}

威廉·布劳斯据传主导暗杀了阿尔蒂尔。阿尔蒂尔失踪后，威廉·布劳斯平步青云，得到了威尔士边界上的新领土和新头衔。阿尔蒂尔失踪多年后，德布劳斯的妻子毛德在一次和约翰王冲突后亲自直接指控国王谋害阿尔蒂尔。据称约翰王将毛德和她的长子小威廉被关进科弗城堡活活饿死作为对此指控的回应。威廉逃到法国，被认为发表了关于阿尔蒂尔下落的声明，但目前没有找到任何副本。不仅仅布列塔尼人，甚至法兰西国王腓力二世，都不知道真正发生了什么，阿尔蒂尔是生是死，如果活着，在哪里，如果死了，怎么死的。无论阿尔蒂尔命运如何，他没有留下已知的子女。

== 继承 ==
围绕阿尔蒂尔死亡的疑云使他身后的继承问题复杂化，他留下的继承问题显然受约翰王和腓力二世共同影响。约翰仍为布列塔尼宗主的同时，也是腓力二世的封臣，故布列塔尼的继承依赖约翰的偏好和腓力的许可。布列塔尼公国没有男嗣，故公爵位的继承限于阿尔蒂尔的几个姐妹之间做选择。

阿尔蒂尔的姐姐“布列塔尼的美少女”[[布列塔尼的美少女埃莉诺|埃莉诺]]也沦为约翰王的囚徒。对约翰作为英格兰国王的继承计划而言，埃莉诺即使不是威胁，也是使之复杂化的因素。她终身被囚，且延续到了约翰的实际继承人其子[[亨利三世_(英格兰)|亨利三世]]统治期间。被囚期间，她不曾结婚，没有子女。她的被囚和身处英格兰使她不可能作为布列塔尼女公爵继承人实行统治。

阿尔蒂尔被他同母异父的妹妹、康斯坦丝和第三任丈夫图阿尔的居伊所生的女儿图阿尔的阿丽斯继承。这次继承得到腓力二世的认可，他取代居伊自任阿丽斯的摄政，安排了她和卡佩王朝幼支德勒家族的皮埃尔·莫克莱尔的婚事。

=== 书籍 ===
阿尔蒂尔之死在[[莎士比亚|莎士比亚]]的历史剧《[[约翰王|约翰王]]》中是一个关键情节。其中，阿尔蒂尔被描绘成一个用童真打动了奉约翰王之命谋害他的休伯特·伯格的孩子。然而，阿尔蒂尔很快在尝试逃离时跳墙身亡了。

19世纪的布列塔尼诗人[[奥古斯特·布里久斯|奥古斯特·布里久斯]]的《寻找阿尔蒂尔王子》以阿尔蒂尔为主人公。

在[[茱莉·加伍德|茱莉·加伍德]]的小说《[[拯救格雷斯|拯救格雷斯]]》中，女主角发现了约翰王指使两名贵族杀害阿尔蒂尔的文件。她嫁给一个苏格兰地主加布里埃尔·麦克本，以逃离英格兰，但遭到那两名贵族和希望推翻约翰的英格兰派别的羁绊，两派人都不知道女主角到底知道多少真相。

在[[兰达尔·加雷特|兰达尔·加雷特]]的架空历史奇幻小说《[[达西勋爵|达西勋爵]]》系列中，理查没有“病死”，而是活了过来；约翰没有成为国王，而[[金雀花家族|金雀花家族]]则由阿尔蒂尔的子孙一直延续到今天。

在澳大利亚小说家[[菲利普·林德赛|菲利普·林德赛]]的《魔鬼和约翰王》中，阿尔蒂尔被脾气发作的约翰所杀，但他被展现为一个一定程度上激怒了约翰的叛逆青年，而非像其他一些版本那样是个无辜的孩子。在介绍中林德赛承认他对阿尔蒂尔身上发生了什么并无证据，但他认为那是一个好的猜测。

阿尔蒂尔还是古怪的英格兰作家[[弗雷德里克·罗尔弗|弗雷德里克·罗尔弗]]（“科沃男爵”）的架空历史小说《休伯特的阿尔蒂尔》的主人公，这部小说在他死后于1935年由[[:en:A._J._A._Symons|A. J. A. Symons]]出版。小说起初由罗尔弗和[[哈里·皮里-戈登|哈里·皮里-戈登]]合作，但后来后者只提供关于人物的丰富细节。这部小说表现为年老的休伯特的长篇叙述，说他救了阿尔蒂尔的性命，陪他远征[[耶路撒冷|耶路撒冷]]，阿尔蒂尔在那里成为耶路撒冷国王，最后返回英格兰，打败了约翰王并杀死了后者的儿子亨利（即历史上的亨利三世）。该书剩下的部分则讲述阿尔蒂尔国王成功的统治，他如何击败[[第六代莱切斯特伯爵西蒙·德孟福尔|西蒙·蒙福尔]]领导的诸侯叛乱，以及他最终的死亡。这是罗尔弗的所有作品中唯一一本没有再版的，可能是受该书强烈的反犹太主义所限的缘故，该书利用了[[圣休|圣休]]等犹太殉难者的神话。

=== 音乐 ===
1912年，布列塔尼作曲家[[居伊·罗帕茨|约瑟夫-居伊·罗帕茨]]写了一首[[交响诗|交响诗]]《寻找阿尔蒂尔王子》，这是根据奥古斯特·布里久斯的诗改编的。布列塔尼乡村摇滚乐队[[Tri_Yann|Tri Yann]]也作了一首关于阿尔蒂尔的生平的歌曲。<ref>[http://www.antiwarsongs.org/canzone.php?lang=en&id=781 Lyrics] (备注：歌词是用[[中世纪法语|中世纪法语]]写成的，可能源于15世纪的一份无名手稿)</ref>
