'''[[阿塞拜疆|阿塞拜疆]]'''的體育在亞塞拜然有歷史根源，即使是現在傳統和現代體育項目仍然流行。[[自由式角力|自由式角力]]傳統上一直視為亞塞拜然的[[國技|國技]]，此項運動為亞塞拜然奪得43面獎牌，其中包括自加入[[國家奧林匹克委員會|國家奧林匹克委員會]]7枚金牌。現時，阿塞拜疆共和國國家奧林匹克委員會的總部設在首都[[巴庫|巴庫]]，主席是[[伊利哈姆·阿利耶夫|伊利哈姆·阿利耶夫]]。<ref>[http://www.sports-reference.com/olympics/countries/AZE/ Azerbaijan]</ref>[[奧林匹克運動會亞塞拜然代表團|代表團]]同時也是與[[奧林匹克運動會英國代表團|英國]]唯二從2000年到2016年連續5屆排名三甲次數都有比上屆增加的國家。

[[足球|足球]]為亞塞拜然最流行的運動，[[亞塞拜然足球協會|亞塞拜然足球協會]]有9,122註冊球員，為國內最大的體育協會。<ref>{{cite web |title=Azərbaycanda nə qədər futbolçu var?|url = http://news.milli.az/sport/14802.html |work=news.milli.az |accessdate=27 January 2014 |language=Azerbaijani}}</ref><ref>{{cite web |title=Football in Azerbaijan |url = http://www.fifa.com/associations/association=aze/goalprogramme/newsid=520992.html |work=www.fifa.com |publisher=[[FIFA|FIFA]]|accessdate=27 January 2014}}</ref>

在[[西洋棋|西洋棋]]方面，除了舉辦過多場國際賽事外，還贏得2009年和2011年歐洲團體西洋棋錦標賽冠軍。<ref>{{cite web |url = http://today.az/news/society/57078.html |title=Azerbaijan’s chess team became European champion |publisher=Today.Az |date=October 31, 2009 |accessdate= June 30, 2010}}</ref><ref>{{cite web |url = http://reports.chessdom.com/news-2009/azerbaijan-russia-win-etcc-novi-sad |title=Azerbaijan, Russia take gold at the European Team Chess Championship |publisher=Chessdom |accessdate=June 30, 2010}}</ref><ref>{{cite web |url = http://azer.com/aiweb/categories/magazine/84_folder/84_articles/84_chess_mammadyarova.html |title=World Chess Champion: Zeynab Mammadyarova |publisher=Azerbaijan International |accessdate= June 30, 2010}}</ref>

2012年12月8日，亞塞拜然首都[[巴庫|巴庫]]取得[[2015年歐洲運動會|2015年歐洲運動會]]的主辦權，為該賽事的首屆。也是该国自[[苏联|苏联]]独立之后举办的第一届综合性体育运动会。之後的2013年7月26日再接手[[2017年伊斯蘭團結運動會|2017年伊斯蘭團結運動會]]的主辦權。此前，[[巴庫|巴庫]]申办过2次[[奥林匹克运动会|奥林匹克运动会]]，但最终都未能進入決勝階段。<ref>{{cite web |title=Baku 2015 heralds new era In European sports movement |url = http://www.washingtontimes.com/news/2013/dec/10/baku-2015-heralds-new-era-european-sports-movement/ |work=www.washingtontimes.com|publisher=[[The_Washington_Times|The Washington Times]]|accessdate=27 January 2014}}</ref>
