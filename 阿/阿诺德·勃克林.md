'''阿诺德·勃克林'''（'''Arnold Böcklin'''，{{bd|1827年|10月16日|1901年|1月16日|catIdx=B}}）是[[瑞士|瑞士]][[象征主义|象征主义]][[画家|画家]]。

== 生平 ==
他早年曾在[[德国|德国]]的[[杜塞尔多夫|杜塞尔多夫]]学习风景[[绘画|绘画]]，并在那里认识了[[费尔巴哈|费尔巴哈]]，兩人成为好朋友。此后他到处旅行，曾到过[[布鲁塞尔|布鲁塞尔]]、[[苏黎世|苏黎世]]、[[日内瓦|日内瓦]]和[[罗马|罗马]]，参观古典和[[文艺复兴|文艺复兴]]时期大师们的作品，并浏览[[地中海|地中海]]沿岸的风景，这种影响使他作品的风格带上寓意和神秘的色彩。

1858年，勃克林在魏玛美术学校任教，直到1861年為止。1866年，他在瑞士[[巴塞尔|巴塞尔]]定居，并为巴塞尔公立艺术陈列馆创作了以[[神话|神话]]為题材的装饰[[壁画|壁画]]。1871年，他遷居到[[慕尼黑|慕尼黑]]。1885年，移居瑞士的霍廷根（Hottingen），最后定居[[意大利|意大利]][[佛罗伦萨|佛罗伦萨]]附近的小城菲埃索尔。

== 作品 ==
[[File:Arnold_Böcklin_-_Die_Toteninsel_V_(Museum_der_bildenden_Künste_Leipzig).jpg|thumb]]
[[File:Böcklin_-_Die_Lebensinsel_-1888.jpeg|thumbnail]]

勃克林最著名的作品是他的共五幅《[[死之岛|死之岛]]》系列，是受埋葬他小女儿的佛罗伦萨英国人公墓的气氛所影响绘制的。他的作品充满神秘、幻想的氛围，但有古典的建筑充斥其中，表现死亡和神秘、忧郁的场景。

他在后期的作品越来越带有主观色彩，经常描画神话中的动物和神秘寓意的题材，如《怪物沛斯特》、《黑死病魔》、《俄底修斯和卡吕普索》等。

== 影响 ==
勃克林对20世纪的[[超现实主义|超现实主义]]画派有很大的影响，他的《死之岛》也为[[俄罗斯|俄罗斯]]作曲家[[谢尔盖·拉赫玛尼诺夫|拉赫马尼诺夫]]提供了灵感，使其成做了[[交响乐|交响乐]]《死之岛》。1904年，威瑟特（Otto Weisert）创造了新艺术字体并取名为“勃克林体”，许多大写字体都有很长的卷须。“勃克林体”受到后来的“嬉皮运动”青睐，并被20世纪70年代的画家广泛应用。
