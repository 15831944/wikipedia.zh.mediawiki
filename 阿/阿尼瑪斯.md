根據[[荣格|荣格]]的[[分析心理學|分析心理學]]，'''阿尼姆斯'''（'''animus'''）是女人[[無意識|無意識]]中的男人性格與形象，可以讓女人盲目迷戀男人。通常正面的阿尼瑪斯有[[父親|父親]]的形象，也常有哥哥，伯伯、叔叔、姑父、姨父、或是男老師的影子。影劇界甚至政治界的[[偶像|偶像]]常是女性阿尼瑪斯形象投射的目標。相對於男人無意識中的單一'''[[阿尼瑪|阿尼瑪]]'''（anima），女人[[夢|夢]]中常有一群阿尼瑪斯。很常見的[[夢|夢]]是一群強盜。

== 負面與正面 ==

阿尼瑪斯可以是負面的也可以是正面的。負面的阿尼瑪斯有致命的吸引力，會帶來毀滅性的後果。娶一個殺一個的[[藍鬍子|藍鬍子]]是典型的負面阿尼瑪斯。另外始亂終棄不負責任的花心男子也是負面的例子。上述在夢中出現的一群強盜也是負面的阿尼瑪斯。

喋喋不休嘮嘮叨叨的[[女性|女性]]是負面的阿尼瑪斯在作祟。女性若長年壓抑憤怒與自我需求會導致產生負面的阿尼瑪斯，因此，出身社會地位較低的女性更可能受制於負面阿尼瑪斯。

玉樹臨風又懂得詩情畫意，風流倜儻卻不亂來，有學問有智慧有理想的正面阿尼瑪斯則是女性創造力與智慧的泉源。獨立且積極追求個人利益的女性，更有機會產生正面阿尼瑪斯。

== 五個階層 ==

依照[[需求層次理論|需求層次理論]]，阿尼瑪斯作為理想[[配偶|配偶]]的心像能有五個層次。第一階層隱含對[[自然崇拜|精靈]]等形體，以及[[性崇拜|對性器的崇拜]]。第二個階層的阿尼瑪斯較像[[健美|健美]]先生或[[運動員|運動員]]。第三個階層的阿尼瑪斯是[[女性|女性]]追求[[羅曼蒂克|羅曼蒂克]]的渴望。第四個階層的阿尼瑪斯提供智慧話語，常有[[教授|教授]]或[[神職人員|神職人員]]的形象。第五個階層的阿尼瑪斯是導引女性走向屬靈真理的智者。

== 文學中的例子 ==

《[[美女與野獸|美女與野獸]]》中的野獸／王子、《[[青蛙王子|青蛙王子]]》中的青蛙／王子皆是少女心靈裡的阿尼瑪斯影像的投射。尚未成熟的少女視男子為野獸，男子一旦成熟，就搖身一變成了王子。
