'''阿卜杜勒-阿齐兹·本·阿卜杜拉赫曼·本·费萨尔·阿勒沙特'''（[[阿拉伯语|阿拉伯语]]：'''{{rtl-lang|ar|عبد العزيز بن عبد الرحمن بن فيصل آل سعود}}'''；{{bd|1876年|11月26日|1953年|11月9日|catIdx=A}}），[[沙特阿拉伯|沙特阿拉伯]]首任国王。阿拉伯國家稱他為「阿卜杜勒阿齐兹」 (Abdulaziz)，西方國家稱他為「伊本·沙特」(Ibn Saud)。

== 生平 ==
生于[[德拉伊耶酋长国|内志]]。是当地大贵族[[阿卜杜勒·拉赫曼|阿卜杜勒·拉赫曼]]（Abd al-Rahman bin Faysal bin Turki Al Saud）（1850年—1928年）的儿子。其祖辈曾統治内志，后来被[[杰贝勒沙马尔酋长国|拉希德家族]]击败。

1891年十五岁的他随父亲流亡[[巴林|巴林]]，后又到[[科威特|科威特]]。1901年，在[[科威特埃米尔|科威特埃米尔]]的资助下，带领兄弟和亲属前往内志地区打击亲近拉希德家族的部落。1902年1月，率领约40人乘夜潜入利雅得，斩杀拉希德家族委任的总督，[[利雅德戰役|夺回了利雅得]]。随即自称利雅得埃米尔。

1902年6月其父主动[[禅让制|禪讓]][[家主|家主]]與之，成為沙特家族的领袖，其父则[[退位|退位]]，只保留了[[伊玛目|伊玛目]]的头衔，作为国家的精神领袖。1904年出任内志[[埃米尔|埃米尔]]。1912年统一了内志地区。1913年夺取[[波斯湾|波斯湾]]沿岸的[[哈萨|哈萨]]和[[蓋提夫|盖提夫]]地区，利雅得酋长国又被称作[[内志与哈萨酋长国|内志和哈萨酋长国]]。1918年在胡马尔击败[[汉志王国|汉志]]国王。1921年，加冕为内志苏丹，同时获得了[[伊玛目|伊玛目]]的头衔，国家也随之更名为内志苏丹国。他于当年彻底征服了拉希德家族的[[杰贝勒沙马尔酋长国|杰拜勒舍迈尔酋长国]]（哈伊勒酋长国）。1924年攻下圣城[[麦加|麦加]]，1925年[[内志与汉志王国|吞并汉志]]，1926年又占领阿西尔地区，自封为[[苏丹_(称谓)|苏丹]]，1927年1月内志苏丹的称号改为内志国王。1932年9月23日改国名为[[沙特阿拉伯|沙特阿拉伯王国]]，出任首任国王。1953年11月9日病逝于[[塔伊夫|塔伊夫]]。

== 家族和继承 ==
阿卜杜勒-阿齊玆國王在世時共有22名妻子，兒女共有90人，男孩45名，<ref>{{Cite web |url = https://anntw.com/articles/20170621-hc3q |title = 沙烏地廢大王子立幼嗣 赤裸裸權鬥 |author = 邱慕天 |date = 2017年6月21日 |publisher = 台灣醒報 |language = 中文 |accessdate = 2017年6月22日 }}</ref>女孩37名，目前有6人繼承王位。
