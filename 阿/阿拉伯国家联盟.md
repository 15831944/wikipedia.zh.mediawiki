'''阿拉伯国家联盟'''（简称'''阿拉伯联盟'''或'''阿盟'''；{{lang-ar|'''جامعة الدول العربية'''}}）成立於1945年，是[[阿拉伯国家|阿拉伯国家]]组成的地区性[[国际|国际]][[政治|政治]]组织，成員國皆位於[[亞洲|亞洲]]或[[非洲|非洲]]，宗旨是加强成员国间的协作，共同维护各国的[[主权|主权]]和[[领土|领土]]完整，广泛开展经济文化各个领域的合作。现有成员国22个。总部原设[[开罗|开罗]]，由于[[埃及|埃及]]与[[以色列|以色列]]签订和约，1979年3月31日，阿拉伯国家外交和经济部长会议决定，将阿盟总部迁往[[突尼斯|突尼斯]]。1990年10月31日，阿盟总部迁回[[开罗|开罗]]，现任秘书长是来自[[埃及|埃及]]的[[艾哈邁德·阿布·蓋特|艾哈邁德·阿布·蓋特]]。

== 地理 ==
阿拉伯國家聯盟的成員國領土面積共1400萬平方公里，橫越[[西亞|西亞]]、[[北非|北非]]和東北非，包括[[新月沃土|新月沃土]]、[[尼羅河|尼羅河]]流域、[[阿特拉斯山脈|阿特拉斯山脈]]的肥沃土地和[[撒哈拉沙漠|撒哈拉沙漠]]，這地區曾出現[[古埃及|古埃及]]、[[亞述|亞述]]、[[巴比倫|巴比倫]]、[[腓尼基|腓尼基]]、[[迦太基|迦太基]]、[[庫施|庫施]]、[[納巴泰人|納巴泰]]等古代文明。

== 經濟 ==
[[File:Dmitry_Medvedev_in_Egypt_23_June_2009-16.jpg|thumb]]
阿拉伯國家聯盟有大量[[石油|石油]]和[[天然氣|天然氣]]資源，[[蘇丹|蘇丹]]南部肥沃的土地更是[[阿拉伯世界|阿拉伯世界]]的食物來源地。[[旅遊業|旅遊業]]沒有受到不穩的局勢影響，是區內發展最快速的行業，尤其以[[埃及|埃及]]、[[阿聯酋|阿聯酋]]、[[阿爾及利亞|阿爾及利亞]]、[[突尼西亞|突尼西亞]]和[[約旦|約旦]]最明顯。通訊業是另一個發展迅速的行業，當地公司如Orascom和Etisalat在短短十年內成功打入國際市場。

阿拉伯國家聯盟成員國的經濟發展程度懸殊，石油國家如[[阿聯酋|阿聯酋]]、[[卡塔爾|卡塔爾]]、[[科威特|科威特]]十分富有，而[[科摩羅|科摩羅]]、[[毛里塔尼亞|毛里塔尼亞]]、[[吉布提|吉布提]]的貧窮問題嚴重。阿拉伯國家聯盟同意撥出5億美元援助[[蘇丹|蘇丹]][[達尔富尔|達尔富尔]]地區，而埃及和利比亞公司也計劃在當地挖掘水井。

== 人口 ==
阿拉伯國家聯盟由22個不同文化和種族的成員國組成，其中大部分是[[阿拉伯人|阿拉伯人]]。根據2014年估計，成員國總人口約為3億5千多萬，人口增長較世界大部份其他地區快速，預計會減慢[[發展中國家|發展中國家]]成員國的經濟發展。最多人口的成員國是[[埃及|埃及]]，人口約7910萬。而最少人口的成員國是[[葛摩|葛摩]]，只有69萬居民。由於大部分土地是沙漠，人口主要聚集在商業和工業活動頻繁的城市和附近地區。阿拉伯國家聯盟最大城市是[[開羅|開羅]]，其次是[[巴格達|巴格達]]、[[喀土穆|喀土穆]]、[[吉薩|吉薩]]、[[大馬士革|大馬士革]]、[[利雅德|利雅德]]和[[达尔贝达|达尔贝达]]。關於阿拉伯聯盟各大城市的人口資料和排名，可見[[阿拉伯聯盟大城市列表|阿拉伯聯盟大城市列表]]。

== 與其他組織的比較 ==
[[File:ArabLeague_Diagram-en.svg|thumb]]
阿拉伯國家聯盟與[[美洲國家組織|美洲國家組織]]、[[歐洲委員會|歐洲委員會]]、[[非洲聯盟|非洲聯盟]]等組織一樣，政治因素是成立的首要考慮，但其以文化因素而非地理位置作為成員資格，則與[[拉丁語聯盟|拉丁語聯盟]]或[[加勒比共同體|加勒比共同體]]等組織相似。

與[[歐洲聯盟|歐洲聯盟]]截然不同的是，阿拉伯國家聯盟沒有造成大規模的地區融合或與成員國的人民沒有直接關係，只是提倡統一的阿拉伯民族，為阿拉伯國家提供討論問題的平台。所有成員國都是[[伊斯蘭會議組織|伊斯蘭會議組織]]成員，而[[海灣阿拉伯國家合作委員會|海灣阿拉伯國家合作委員會]]成員區均來自阿拉伯國家聯盟。
