'''阿佐谷站'''（{{jpn|j=阿佐ケ谷駅|hg=あさがやえき|rm=Asagaya eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[杉並區|杉並區]][[阿佐谷|阿佐谷南]]三丁目，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[中央本線|中央本線]]的[[鐵路車站|鐵路車站]]。中央線快速電車的[[車站編號|車站編號]]是'''JC 08'''、中央·總武線各站停車是'''JB 05'''。

作為運行系統有行走{{link-ja|急行線|急行線}}的[[中央線快速|中央線快速電車]]與行走{{link-ja|急行線|急行線|緩行線}}的[[中央·總武緩行線|中央·總武線各站停車]]2個系統停靠。但是中央線快速電車只限平日繁忙時段停靠（參見「[[中央線快速#快速停車站有關討論|中央線快速#快速停車站有關討論]]」）。另外，依據[[特定都區市內|特定都區市內]]，本站屬於「[[特定都區市內#設定區域一覧|東京都區內]]」。

== 車站結構 ==
本站是一[[高架車站|高架車站]]，設有兩個出口：北口（中杉通側）和西口（{{lang|ja|阿佐ヶ谷ダイヤ街}}側）。其中中杉通側出口設有購物商場「Dila 阿佐谷」。

本站共有2座島式月台，由快速線、緩行線分別使用1座。周末及公眾假期時由於快速線列車全日通過本站，3、4號月台亦同時會被封閉。而平日快速線列車雖然在中野以西路段為各站皆停，但方向幕仍顯示列車種類為「快速」，避免與以三鷹為終點的緩行線混淆。

== 車站周邊 ==

由於住宅在周邊密集，商店街亦多。南口為長大的商店街[[阿佐谷珍珠中心|阿佐谷珍珠中心]]。這商店街於8月的第一週目有[[七夕|七夕]]祭。站構内亦有設置七夕裝飾。

== 歷史 ==
* 1922年（[[大正|大正]]11年）7月15日－本站開業。
* 1964年（[[昭和|昭和]]39年）9月22日－本站四線路段的其中兩線完成高架化。
* 1966年（昭和41年）4月3日－四線路段完成高架化。
* 1987年（昭和62年）4月1日－[[國鐵分割民營化|國鐵分割民營化]]，本站由JR東日本和JR貨物接手管理。
* 2001年（[[平成|平成]]13年）11月18日－智能卡系統[[Suica|Suica]]正式投入服務。
