'''阿比查邦·魏拉希沙可'''（{{Thai|t=อภิชาติพงศ์ วีระเศรษฐกุล|r=Apichatpong Weerasethakul}}；{{bd|1970年|7月16日|catIdx=T}}<ref>[http://www.forvo.com/word/%E0%B8%AD%E0%B8%A0%E0%B8%B4%E0%B8%8A%E0%B8%B2%E0%B8%95%E0%B8%B4%E0%B8%9E%E0%B8%87%E0%B8%A8%E0%B9%8C_%E0%B8%A7%E0%B8%B5%E0%B8%A3%E0%B8%B0%E0%B9%80%E0%B8%A8%E0%B8%A3%E0%B8%A9%E0%B8%90%E0%B8%81%E0%B8%B8%E0%B8%A5/ Pronunciation in Thai] (Forvo)</ref>）是一位[[泰國|泰國]][[電影導演|電影導演]]、[[監製|監製]]及[[劇作家|劇作家]]。魏拉希沙可在國際影壇有較高知名度，但他在國內的名聲遠不及國際。阿比查邦·魏拉希沙可曾多次獲得國際大獎，他在2010年憑《[[波米叔叔的前世今生|波米叔叔的前世今生]]》，獲得了[[第63屆康城影展|第63屆康城影展]][[金棕櫚獎|金棕櫚獎]]。

阿比查邦·魏拉希沙可的電影[[元素|元素]]包括[[夢|夢]]、[[自然|自然]]、[[性愛|性]]（含括異性和同性之愛<ref>"Creating His Own Language: An Interview With Apichatpong Weerasethakul", [[Holger_Romers_(author)|Romers, H]]. ''Cineaste'', page 34, vol. 30, no. 4, Fall 2005, New York</ref>），他的電影偏愛非常規的敘事結構。

== 生平 ==
===早期===
阿比查邦·魏拉希沙可在[[泰國|泰國]][[曼谷|曼谷]]出生，父母都是[[醫生|醫生]]，任職於[[坤敬府|坤敬府]][[醫院|醫院]]<ref>Rithdee, Kong (July 28, 2006). [http://www.bangkokpost.com/Realtime/28Jul2006_real001.php Everything is illuminated], ''Bangkok Post'' (retrieved July 28, 2006).</ref>。阿比查邦·魏拉希沙可曾入讀泰國[[孔敬大學|孔敬大學]]主修[[建築|建築]]，1994年取得學位。其後留學[[美國|美國]][[芝加哥藝術學院|芝加哥藝術學院]]學習[[電影製作|電影製作]]，1997年取得[[碩士學位|碩士學位]]，成為泰國[[獨立電影|獨立電影]]製作人。阿比查邦·魏拉希沙可有[[華人|華人]]的血統<ref>[http://www.nytimes.com/2010/09/14/arts/14iht-thaifilm.html A Thai Director Earns Acclaim Abroad and Ambivalence at Home]</ref>。

===導演生涯===
阿比查邦·魏拉希沙可首部長片《[[正午顯影|正午顯影]]》完成於2000年，內容融合了[[紀錄片|紀錄片]]畫面與即興創作的敘事風格，並顯現出[[超現實主義|超現實主義]]風格。他在1999年創立自己的製作公司，並製作和推廣自己的作品，也支持其他獨立製片及實驗電影作品。

阿比查邦·魏拉希沙可以《[[極樂森林|極樂森林]]》在2002年的[[坎城電影節|坎城電影節]]贏得[[一種注目大獎|一種注目大獎]]<ref name="cannes-2002.com">{{cite web |url=http://www.festival-cannes.com/en/archives/ficheFilm/id/3102934/year/2002.html |title=Festival de Cannes: Blissfully Yours |accessdate=2009-10-31|work=festival-cannes.com}}</ref>，因此逐漸獲得國際上的關注。2004年，阿比查邦·魏拉希沙可以第四部長片《熱帶幻夢》贏得了[[坎城影展評審團獎|戛納電影節評審團獎]]。

阿比查邦·魏拉希沙可執導的短片也相當著名。他在[[全州國際電影節|全州國際電影節]]參加三部數位短片計畫，與兩位其他亞洲導演合作。他的電影為《七情六欲》，日本[[塚本晉也|塚本晉也]]與[[韓國|韓國]][[宋一坤|宋一坤]]分別執導另外兩部電影。

阿比查邦·魏拉希沙可在2005年擔任[[海嘯|海嘯]]數位短片計畫顧問。海嘯數位短片由13部影片所組成，泰國當代[[藝術|藝術]][[文化|文化]]部為了紀念[[2004年印度洋大地震|2004年印度洋大地震]]，當時地震產生的海嘯襲擊了泰國。他執導的電影為《亞洲之鬼》。

當代藝術文化部在2005年頒發[[信巴通獎|信巴通獎]]（รางวัลศิลปาธร）給阿比查邦·魏拉希沙可。該獎項每年頒發給許多類別的當代藝術家。

阿比查邦·魏拉希沙可在2006年完成一部劇情片：《[[戀愛症候群|戀愛症候群]]》。這部電影是美國導演[[彼得·塞拉斯|彼得·塞拉斯]]（Peter Sellars）委託製作，為了在[[維也納|維也納]]為慶祝[[莫扎特|莫扎特]]誕生250週年。《[[戀愛症候群|戀愛症候群]]》在[[第63屆威尼斯電影節|第63屆威尼斯電影節]]上首映，並在2006年[[多倫多國際電影節|多倫多國際電影節]]和其他許多電影節上映。

2007年4月，《[[戀愛症候群|戀愛症候群]]》擬在泰國上映之際，因為劇情出現[[和尚|和尚]]彈結他及玩遙控飛機的鏡頭<ref name = "Saucer">Weerasethakul, Apichatpong. September 14, 2007. [http://film.guardian.co.uk/features/featurepages/0,,2168589,00.html Who can save my flying saucer?], ''The Guardian''; retrieved 2007-09-15</ref>，泰國[[電檢|電檢]]部門要求刪剪部份鏡頭，但是阿比查邦·魏拉希沙可拒絕，最後取消在泰國公映計畫<ref name = "Cancels">Rithdee, Kong. [http://celinejulie.blogspot.com/2007/04/syndromes-and-century-is-censored-in.html Thai director cancels film's local release], ''Bangkok Post''; retrieved 2008-01-23</ref>。電影分級委員會主要成員包括政府官僚以及[[泰國皇家警察|泰國皇家警察]]成員<ref name = "passes">Rithdee, Kong. December 20, 2007. [http://www.variety.com/article/VR1117978081.html?categoryid=13&cs=1 Thailand passes controversial film act], ''Variety (magazine)''; retrieved 2008-01-23</ref>。2007年末，《[[戀愛症候群|戀愛症候群]]》在[[曼谷|曼谷]][[法國文化協會|法國文化協會]]兩次私下放映。

[[File:Apichatpong.jpg|thumb]]

阿比查邦·魏拉希沙可和其他導演組織泰國自由電影運動，來抵制電影相關法律草案。阿比查邦·魏拉希沙可說：“我們不同意政府有禁止電影的權利”，“已經有其他法律規範電影製片人潛在的不當行為”<ref name = "Protest">Rithdee, Kong. November 28, 2007. [http://www.variety.com/article/VR1117976675.html?categoryid=19&cs=1 Directors protest censorship law], ''Variety (magazine)''; retrieved 2008-01-23</ref>。

文化監察部部主任認為電影分級法是必要的法案，因為泰國的影迷是未受教育的群眾<ref name = "worse">Montlake, Simon. October 11, 2007. [http://www.time.com/time/magazine/article/0,9171,1670261,00.html Will Thai reforms make censorship worse?], ''Time (magazine)''; retrieved 2008-01-23</ref>。她說：“他們不是知識分子，這就是為什麼我們需要電影分級”。她還說：「沒有人去看魏拉希沙可拍攝的電影。泰國人民希望看到的作品是[[喜劇|喜劇]]。我們喜歡歡笑。<ref name = "worse"/>」

阿比查邦·魏拉希沙可與導演[[彭力·雲坦拿域安|彭力·雲坦拿域安]]、[[偉西·沙讚那庭|偉西·沙讚那庭]]在曼谷議會大樓外抗議電影分級草案，他們輪流高舉標語，上面寫著「沒有自由，沒有民主，就沒有和平」<ref name = "Protest"/><ref>[http://www.prachatai.com/05web/th/home/10378  คนรักหนังขอเปลี่ยนม้วน ‘พ.ร.บ.ภาพยนตร์’ ฉบับ โลกแคบ-ใจแคบ] {{Wayback|url=http://www.prachatai.com/05web/th/home/10378 |date=20080204043038 }}, Prachatai; retrieved 2008-01-23 {{th icon}}</ref>。 電影分級法後來於2007年12月20日通過<ref name = "passes"/>。

《[[戀愛症候群|戀愛症候群]]》也獲得許多影評人給予高度的評價<ref>[http://artsandfaith.com/t100/ Arts & Faith Top 100] {{webarchive|url=http://archive.wikiwix.com/cache/20110223153741/http://artsandfaith.com/t100/ |date=2011-02-23 }}</ref><ref>{{cite web |url=http://www.metacritic.com/film/awards/2007/toptens.shtml |title=Metacritic: 2007 Film Critic Top Ten Lists |accessdate=2008-01-05 |publisher=[[Metacritic|Metacritic]] |archiveurl=https://web.archive.org/web/20080102102034/http://www.metacritic.com/film/awards/2007/toptens.shtml |archivedate=2008-01-02 |deadurl=yes }}</ref><ref>[http://www.cinemathequeontario.ca/newsrelease_detail.aspx?Id=678 Cinematheque Ontario news release. 2009-11-23] {{webarchive|url=https://web.archive.org/web/20100305140145/http://www.cinemathequeontario.ca/newsrelease_detail.aspx?Id=678 |date=2010-03-05 }}</ref>。

他的電影《[[波米叔叔的前世今生|波米叔叔的前世今生]]》在2010年的坎城影展贏得了[[金棕櫚獎|金棕櫚獎]]，成為首度獲獎的泰國人<ref name="hollywoodreporter">{{cite news |url=http://www.hollywoodreporter.com/hr/content_display/world/news/e3i3d82e5b089623802884efcd38a037f67?pn=2 |title=Hollywood Reporter: Cannes Lineup |accessdate=2010-04-16 |work=hollywoodreporter |deadurl=yes |archiveurl=https://web.archive.org/web/20100422212018/http://www.hollywoodreporter.com/hr/content_display/world/news/e3i3d82e5b089623802884efcd38a037f67?pn=2 |archivedate=22 四月 2010 }} </ref>。《[[波米叔叔的前世今生|波米叔叔的前世今生]]》還被泰國選為第83屆[[奧斯卡獎最佳外語片|奧斯卡獎最佳外語片]]的競賽片，但是這部電影並沒有進入最終入圍名單<ref name="thewrap Thailand">{{cite web |url=http://www.thewrap.com/awards/column-post/oscars-foreign-language-deadline-looms-21212?page=0,10 |title=And the Hopefuls for Best Foreign Oscar Are ... |accessdate=2010-10-10|work=thewrap}}</ref><ref name="Oscar Shortlist">{{cite web |url=http://www.oscars.org/press/pressreleases/2011/20110119.html |title=9 Foreign Language Films Continue to Oscar Race |accessdate=2011-01-19|work=oscars.org}}</ref>。

在2012年，阿比查邦·魏拉希沙可執導的電影《[[湄公河大酒店|湄公河大酒店]]》在2012年坎城影展特別放映<ref name="Official Selection">{{cite web |url=http://www.festival-cannes.fr/en/article/58878.html |title=2012 Official Selection |accessdate=2012-05-25 |work=Cannes}}</ref><ref name="guardian">{{cite web |url=http://www.guardian.co.uk/film/2012/may/20/trash-cannes-2012-weerasethakul-desplat-yentob |title=Trailer trash at Cannes 2012 |accessdate=2012-05-25 |work=The Guardian}}</ref>。

阿比查邦·魏拉希沙可目前正在拍攝中的作品是《烏托邦》。他在2006年的香港電影融資論壇和2006年坎城影展上發布執導《烏托邦》的消息<ref>"Project 24" (PDF), [http://www.hkfilmart.com/haf/pdf/project06/project24.pdf Hong Kong Filmart]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}. Retrieved February 16, 2006.</ref>。
