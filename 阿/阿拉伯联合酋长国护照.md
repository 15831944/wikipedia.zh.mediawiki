'''阿拉伯联合酋长国护照'''是签发给[[阿拉伯联合酋长国|阿拉伯联合酋长国]]公民的旅行证件。阿拉伯联合酋长国亦签发[[临时护照|临时护照]]给非本国公民。但此临时护照不可用于声明阿联酋的居住权。阿联酋内政部于2011年12月11日开始签发[[生物特征护照|生物特征护照]]，使其成为继[[卡塔尔|卡塔尔]]之后第二个使用生物特征护照的海湾阿拉伯国家合作委员会国家。<ref>http://www.abudhabi.ae/egovPoolPortal_WAR/appmanager/ADeGP/Citizen?_nfpb=true&_pageLabel=p_citizen_en_homepage_hidenav&did=311936&lang=en</ref>

在阿联酋成立之前, 原[[阿布扎比|阿布扎比]]、[[沙迦|沙迦]]、[[迪拜|迪拜]]、[[阿治曼|阿治曼]]、[[富查伊拉|富查伊拉]]、[[乌姆盖万|乌姆盖万]]六个酋长国便承担起签发共同旅行证件的责任。<ref>{{cite web |url=http://www.uaehistory.com/en/photos/279 |title=存档副本 |accessdate=2013-07-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20130714061811/http://www.uaehistory.com/en/photos/279 |archivedate=2013-07-14 }}</ref> 这些旅行证件使用[[阿拉伯文|阿拉伯文]]和[[英文|英文]]， 往往参考同时期控制这片地区的[[英国|英国]]。<ref>https://www.gov.uk/government/world/organisations/british-embassy-abu-dhabi</ref>

阿拉伯联合酋长国护照的签发费用是世界最低费用护照之一，只需40[[迪拉姆|迪拉姆]]（10.90[[美元|美元]]）。<ref>[https://www.moi.gov.ae/en/eservices/eservice.502.aspx https://www.moi.gov.ae/en/eservices/eservice.502.aspx]</ref>

== 特征 ==
第一页和最后一页是由硬纸制成的，比旧版本的护照要厚——这一措施使得护照直到过期都可以保持良好的状态。第一页包含[[謝赫扎耶德大清真寺|謝赫扎耶德大清真寺]]的水彩画轮廓，最后一页包含圆顶和塔器的实际清真寺的图纸。该护照身份页的所有资料都是印刷和层压版本，不像其他国家可能都用手写的信息。新的护照包含用来解析名称重复的数据，而名字重复正是旧护照存在的一大问题。阿联酋是卡塔尔之后第二个签发电子护照的国家，并且新签发的护照具有更先进的技术。所有的国际机场已经通报了新护照，使阿联酋国民可以开始使用他们的新护照在他们出国旅行。<ref>{{Cite web |url=http://dnrd.ae/ar/Services/ServicesGuide/Pages/AudianceService.aspx?AudianceId=1&ServiceId=10 |title=http://dnrd.ae/ar/Services/ServicesGuide/Pages/AudianceService.aspx? |access-date=2015-11-27 |archive-url=https://web.archive.org/web/20120321130401/http://dnrd.ae/ar/Services/ServicesGuide/Pages/AudianceService.aspx?ServiceId=10&AudianceId=1 |archive-date=2012-03-21 |dead-url=yes }}</ref>

==申请护照所需材料==
*户籍证明原版或者复印件
*个人照片
*身份证原件或复印件（如果未被录入）<ref>[https://www.moi.gov.ae/en/eservices/eservice.502.aspx https://www.moi.gov.ae/en/eservices/eservice.502.aspx]</ref>
