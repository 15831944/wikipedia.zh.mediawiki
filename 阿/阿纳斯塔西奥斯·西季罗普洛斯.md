'''阿纳斯塔西奥斯·西季罗普洛斯'''（{{lang-el|Αναστάσιος "Τάσος" Σιδηρόπουλος|translit=Anastasios''' "'''Tasos'''" '''Sidiropoulos}}，{{bd|1979年|8月9日|}}），昵称'''塔索斯'''，希腊足球裁判、警察，生于[[兹拉马|兹拉马]]。西季罗普洛斯2001年起担任足球裁判。2009年9月27日[[希臘足球超級聯賽|希臘足球超級聯賽]][[埃尔格特利斯足球俱乐部|埃尔格特利斯]]4比0击败[[潘斯拉基科斯足球俱乐部|潘斯拉基科斯]]的比赛是他第一次在希腊顶级联赛担任主裁判。2011年，西季罗普洛斯成为[[FIFA|FIFA]]国际裁判。<ref>{{cite web |title=ANASTASIOS SIDIROPOULOS - RECORD OF ALL REFEREED MATCHES |url=https://www.transfermarkt.com/anastasios-sidiropoulos/profil/schiedsrichter/2629/saison/2009 |website=transfermarkt |accessdate=2018-11-11}}</ref><ref name="transfermarkt" />2014年9月17日欧冠联赛小组赛[[毕尔巴鄂竞技|毕尔巴鄂竞技]]主场0比0战平[[顿涅茨克矿工|顿涅茨克矿工]]的比赛，西季罗普洛斯首次出任欧冠联赛主裁判。2017年3月7日，他首次出任欧冠联赛淘汰赛主裁判，执法了十六强阶段[[阿森纳|阿森纳]]主场1比5不敌[[拜仁慕尼黑|拜仁慕尼黑]]的比赛。西季罗普洛斯尚无执法欧洲杯、世界杯的经历。<ref>{{cite web |title=ANASTASIOS SIDIROPOULOS - RECORD OF ALL REFEREED MATCHES GES/1 |url=https://www.transfermarkt.com/anastasios-sidiropoulos/leistungsdaten/schiedsrichter/2629/wettbewerb_id/CL |website=transfermarkt |accessdate=2018-11-11}}</ref><ref>{{cite web |title=ANASTASIOS SIDIROPOULOS - RECORD OF ALL REFEREED MATCHES 2014/2015 |url=https://www.transfermarkt.com/anastasios-sidiropoulos/leistungsdaten/schiedsrichter/2629/saison_id/2014/wettbewerb_id/CL |website=transfermarkt |accessdate=2018-11-11}}</ref><ref>{{cite web |title=ANASTASIOS SIDIROPOULOS - RECORD OF ALL REFEREED MATCHES 2016/2017 |url=https://www.transfermarkt.com/anastasios-sidiropoulos/leistungsdaten/schiedsrichter/2629/saison_id/2016/wettbewerb_id/CL |website=transfermarkt |accessdate=2018-11-11}}</ref>
