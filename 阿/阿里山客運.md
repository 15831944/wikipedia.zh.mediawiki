'''阿里山客運股份有限公司'''（簡稱'''阿里山客運'''、'''山客'''；因其[[吉祥物|吉祥物]]為[[諸羅樹蛙|諸羅樹蛙]]，民間亦暱稱為'''呱呱'''），於2016年12月23日奉[[臺中市政府|臺中市政府]]核准成立，總公司設立於[[台中市|台中市]][[西屯區|西屯區]]，為[[電動巴士|電動巴士]]業者[[路得寶交通|路得寶交通]]為運營嘉義地區之市區公車路線所成立之[[子公司|子公司]]，目前專營[[嘉義縣市區公車|嘉義縣市區公車]]。2017年1月24日於[[高鐵嘉義站|高鐵嘉義站]]前低碳轉運中心舉行電動巴士通車典禮，並於2017年1月25日開始經營[[嘉義縣市區公車|嘉義縣市區公車]]166路及168路等兩條純電動巴士路線，全車隊採[[比亞迪K9|比亞迪電巴]]行駛，目前主要營運基地為[[嘉義縣|嘉義縣]][[太保市|太保市]]<ref>[http://www.chinatimes.com/newspapers/20161208000065-260202 路得寶斥資35億 強攻電動車]</ref><ref>{{cite web |url=https://www.1111.com.tw/job-bank/job-description.asp?eNo=79716055 |title=存档副本 |accessdate=2017-01-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20170116145534/https://www.1111.com.tw/job-bank/job-description.asp?eNo=79716055 |archivedate=2017-01-16 }}</ref><ref>{{cite web |url=https://www.1111.com.tw/job-bank/job-description.asp?eNo=79715996 |title=存档副本 |accessdate=2017-01-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20170116145143/https://www.1111.com.tw/job-bank/job-description.asp?eNo=79715996 |archivedate=2017-01-16 }}</ref><ref>{{cite web |url=https://www.1111.com.tw/job-bank/job-description.asp?eNo=79716040 |title=存档副本 |accessdate=2017-01-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20170116143905/https://www.1111.com.tw/job-bank/job-description.asp?eNo=79716040 |archivedate=2017-01-16 }}</ref>，並在[[高鐵嘉義站|高鐵嘉義站]]附近有車輛基地，為嘉南地區第一家全面使用低底盤電動巴士（凱勝綠能K9電動車）的業者，目前服務範圍含括嘉義縣及[[台南市|台南市]][[鹽水區|鹽水區]]、[[北門區|北門區]]等地。
