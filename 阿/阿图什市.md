'''阿图什市'''（[[维吾尔語|维吾尔語]]：ئاتۇش；{{Lang-ky|Артыш}} ）是[[中国|中国]][[新疆维吾尔自治区|新疆维吾尔自治区]][[克孜勒苏柯尔克孜自治州|克孜勒苏柯尔克孜自治州]]下辖的一个[[县级市|县级市]]，亦是该州的[[首府|首府]]，位于新疆西南部，紧靠[[喀什|喀什]]市，与[[吉尔吉斯斯坦|吉尔吉斯斯坦]]接壤。全市人口中，[[维吾尔族|维吾尔族]]占80%以上。

== 历史 ==
[[汉|汉]]时属[[疏勒|疏勒]]，[[隋朝|隋]]属[[突厥|突厥]]，[[唐|唐]]设疏勒都督府。为古代[[丝绸之路|丝绸之路]]上的重镇，至今依然商贸繁荣。[[宋|宋]]代起属[[黑汗|黑汗]]国。[[清|清]]代称'''阿斯图阿尔图什'''，后演变为阿图什。民国十七年(1938)析伽师县置阿图什设治局，1943年升为县。1949年属[[喀什专区|喀什专区]]。1954年7月划归克孜勒苏柯尔克孜自治州。1986年撤县，建阿图什市。

== 地理 ==
“阿图什”为[[维吾尔|维吾尔]]语，意为“两山之间”。地处[[天山南脉|天山南脉]]南麓，[[塔里木盆地|塔里木盆地]]的西北边缘，东邻[[柯坪县|柯坪县]]，南接[[巴楚县|巴楚县]]、[[伽师县|伽师县]]、[[疏附县|疏附县]]，西邻[[乌恰县|乌恰县]]，北与[[吉尔吉斯斯坦|吉尔吉斯斯坦]]为界。市境东西长约241km，南北宽约136km，边境线长128km，总面积约为14800km<sup>2</sup>。

市境内海拔高度1200—4562m，地势北高南低，由西南向东降缓，地形复杂。北部[[玛依丹山|玛依丹山]]与[[喀拉铁热克山|喀拉铁热克山]]相连，呈东北西南走向，海拔2500—4562m，最高点为[[塔什阔坦能贝希峰|塔什阔坦能贝希峰]]海拔4562m。中部的吐古买提盆地、哈拉峻盆地海拔1500—2000m，地势西高东低，较平坦宽阔。南部有喀拉塔格山以南的阿湖谷地和博孜塔格山、阿克塔格山以南的冲积平原，海拔1200—1500m，地势西高东低，至最南端的兰干塔格山、库玛勒塔格山与喀什地区为界，在西南部还有恰克玛克河流域的上阿图什谷地。

市境内基岩山区面积达9270km<sup>2</sup>，占总面积的65.7%；戈壁平原面积2630km<sup>2</sup>，占总面积的15.6%。

典型的暖温带干旱大陆性气候，是极端干旱区，全市年平均气温12.9℃，降水量80毫米。四季分明，日照充足，干旱少雨，无霜期长，气温日振幅大，春季升温快，天气多变。夏季炎热，蒸发强盛。秋季降温迅速，冬季寒冷，多晴日，无稳定降雪。平原极端最高温度41.2℃，最低温度-24.4℃,最冷的一月平均气温-10.9℃，山区极端最低气温-32.4℃。平原区冻土层深度在68～87cm之间，山区最大冻土层深度大于150～300cm。南部平原多年平均降水量78mm，最多年降水量130mm，最少年不足20mm。
 
阿图什市河流众多，但多为季节性河流，常年流水的河流不多，水量稀少，地表径流不发达，存在大面积的无流区域，径流极度贫乏区。按流域划分，全市河流可分成两大水系：[[恰克玛克河|恰克玛克水系]]和[[布谷孜河|布谷孜水系]]。径流成因，除恰克玛克河上游少量冰川和融雪补给外，多数河的基流均为泉水补给，洪水和季节性山溪沟谷的径流，则系降雨形成。 
 
境内水资源7.376亿m<sup>3</sup>，其中地表水4.228亿m<sup>3</sup>，年径流量1亿亿m<sup>3</sup>以上的只有[[恰克玛克河|恰克玛克河]]与[[博古孜河|博古孜河]]。

== 经济 ==
阿图什的经济以农业为主，农产品主要有[[棉花|棉花]]、[[葡萄|葡萄]]、[[绵羊|绵羊]]等。特产[[无花果|无花果]]，质量上乘，极具盛名，产量也位居中国第一。境内还有[[食盐|盐]]矿等资源。
==人口与民族==
2014年总人口为27.1091万人，其中：维吾尔族21.6602万人，柯尔克孜族人口3.0539万人，汉族人口2.3129万人，其他民族0.07万人；农业人口17.6799万人，非农业人口9.4292万人。维吾尔族主要分布在松他克乡、阿扎克乡、阿湖乡、上阿图什乡、格达良乡、经济羊场、幸福办事处和光明办事处等处；柯尔克孜族主要分布在哈拉峻乡、吐古买提乡、幸福办事处和光明办事处等处。

== 文物 ==
著名的有[[艾比甫·艾洁木麻扎|艾比甫·艾洁木麻扎]]、[[阿圖什大清真寺|阿圖什大清真寺]]，为[[新疆全国重点文物保护单位列表|新疆全国重点文物保护单位]]。
