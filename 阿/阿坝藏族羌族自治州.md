'''阿坝藏族羌族自治州'''（{{bo|z=རྔ་བ་བོད་རིགས་ཆའང་རིགས་རང་སྐྱོང་ཁུལ།|w=rnga ba bod rigs cha'ang rigs rang skyong khul|p=Ngawa Poirig Qangrig Ranggyong Kü}}），简称'''阿坝州'''，是[[中华人民共和国|中华人民共和国]][[四川省|四川省]]下辖的[[自治州|自治州]]，位于四川省西北部，川[[甘肃省|甘]][[青海省|青]]三省交界处。州境东接[[德阳市|德阳市]]、[[绵阳市|绵阳市]]和[[甘肃省|甘肃省]][[陇南市|陇南市]]，南界[[成都市|成都市]]、[[雅安市|雅安市]]，西邻[[甘孜州|甘孜州]]和[[青海省|青海省]][[果洛州|果洛州]]，北达甘肃省[[甘南州|甘南州]]。地处川西北高山峡谷区，[[青藏高原|青藏高原]]东南缘与[[横断山脉|横断山脉]]北端的结合部，东北部为[[龙门山|龙门山]]、[[岷山|岷山]]，西北部为[[巴颜喀拉山|巴颜喀拉山]]分支，西南部为[[大雪山|大雪山]]区，南部为[[邛崃山|邛崃山]]区。西北部为丘状高原，北部为沼泽地带，东南部为高山峡谷区。[[岷江|岷江]]发源于东北部[[松潘县|松潘县]]，于州境东南流入[[成都平原|成都平原]]，[[大渡河|大渡河]]上游[[大金川|大金川]]纵贯西部，[[黄河|黄河]]流经西北部，为四川省仅有的[[黄河流域|黄河流域]]。全州总面积8.3万平方公里，人口93.01万，[[藏族|藏族]]和[[羌族|羌族]]人口比例分別约55%和18%，[[汉族|汉族]]人口比例约25%。州境大部分傳統上屬於[[安多地區|安多地區]]，西部[[金川縣|金川縣]]、[[小金縣|小金縣]]屬於[[康區|康區]]，自治州首府驻[[马尔康市|马尔康市]]。阿坝州为全省主要畜牧区，草原面积辽阔，自然生态和旅游资源丰富。有著名的[[黄龙风景名胜区|黄龙]]、[[九寨沟风景名胜区|九寨沟]]、[[四姑娘山|四姑娘山]]等[[国家级风景名胜区|国家级风景名胜区]]和[[世界自然遗产|世界自然遗产]]，并有[[卧龙国家级自然保护区|卧龙国家级自然保护区]]，为[[大熊猫|大熊猫]]的重要保育基地。

== 历史 ==
本地區的历史可以追溯到5000年前，目前已經發現的[[新石器時代|新石器時代]]遺址，包括[[茂縣|茂縣]][[營盤山遺址|營盤山遺址]]、[[松潘縣|松潘縣]]東裕村遺址、[[汶川縣|汶川縣]]高坎遺址、[[黑水縣|黑水縣]]遺址等。<ref name="阿坝州">{{cite web|url=http://www.abazhou.gov.cn/zjab/abgk-lsmc.html| archive-url=https://web.archive.org/web/20070630214036/http://www.abazhou.gov.cn/zjab/abgk-lsmc.html|archive-date=2007-06-30|title=中国阿坝州|date=2007-06-30}}</ref><ref>{{cite journal|url=http://www.abzsk.cn/detail.action?id=4153|title=岷江上游新石器时代文化遗址调查及营盘山考古试掘综述 |author1=陳劍|journal=《阿壩師範高等專科學校學報》|date=2004年|issue=第4期}}</ref><ref>{{cite web|url=http://m.cssn.cn/kgx/kgx_kgdt/201504/t20150427_1604079.htm|title=阿坝州黑水县扎窝乡新发现彩陶文化遗址 |website=中国社会科学网}}</ref>

8世纪，著名的大译师[[白若杂纳|白若杂纳]]被流放到嘉绒（[[马尔康县|马尔康县]]）。1410年，[[宗喀巴|宗喀巴]]的四大弟子之一[[察柯·阿旺扎巴|察柯·阿旺扎巴]]建立了阿坝地区的第一座[[格鲁派|格鲁派]]寺庙。

20世紀30年代，[[中國工農紅軍|中國工農紅軍]][[長征|長征]]最艱苦、最危險的行軍在本地區，翻越了8座海拔4000米以上的雪山，走過了人跡罕至的水草地，這段歷史被稱為過「雪山草地」，並於1935年10月建立了少數民族的[[格勒得沙共和国革命政府|格勒得沙共和国革命政府]]。<ref name="阿坝州"/>

1953年，[[中华人民共和国|中华人民共和国]]建立藏族自治州。1956年，改为阿坝藏族自治州。1987年，改为阿坝藏族羌族自治州。

2008年5月12日，[[汶川县|汶川县]]发生芮氏8.0级[[汶川大地震|特大地震]]，造成严重伤亡和破坏。2017年8月8日，[[九寨溝縣|九寨溝縣]]發生7.0級[[2017年九寨沟地震|大地震]]，震源深度20千米。

== 地理 ==
阿坝州地处青藏高原东南缘，横断山脉北端与川西北高山峡谷的结合部。地势北高南低，东南部为高山峡谷区，中部为山原区，西北部为高原区。长江上游主要支流[[岷江|岷江]]、[[大渡河|大渡河]]纵贯全境。

== 行政 ==
=== 行政区划 ===
阿坝州辖1个[[县级市|县级市]]、12个[[县_(中华人民共和国)|县]]。
* 县级市：[[马尔康市|马尔康市]]
* 县：[[汶川县|汶川县]]、[[理县|理县]]、[[茂县|茂县]]、[[松潘县|松潘县]]、[[九寨沟县|九寨沟县]]<ref>原为南坪县，现因[[九寨沟风景区|九寨沟风景区]]而改名</ref>、[[金川县|金川县]]、[[小金县|小金县]]、[[黑水县|黑水县]]、[[壤塘县|壤塘县]]、[[阿坝县|阿坝县]]、[[若尔盖县|若尔盖县]]、[[红原县|红原县]]

此外，位于汶川县的[[卧龙特別行政区|卧龙特別行政区]]由四川省林业厅管理，并与[[国家林业局|国家林业局]][[四川卧龙国家级自然保护区|四川卧龙国家级自然保护区]]完全重合。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全州[[常住人口|常住人口]]为898713人<ref>{{cite book |author=阿坝藏族羌族自治州统计局|title=《阿坝藏族羌族自治州2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加51245人，增长6.05%。年平均增长0.59%。其中，男性人口为467231人，占51.99%；女性人口为431482人，占48.01%。总人口性别比（以女性为100）为108.29。0－14岁人口为178497人，占19.86%；15－64岁人口为654786人，占72.86%；65岁及以上人口为65430人，占7.28%。

=== 民族 ===
[[藏族|藏族]]人口为489782人，占54.50%；[[羌族|羌族]]人口为157969人，占17.58%；[[汉族|汉族]]人口为220679人，占24.55%；其他[[少数民族|少数民族]]人口为30283人，占3.37%。

== 经济 ==
阿坝州於2009年的[[人均生產總值|人均生產總值]]为282亿元，人均为32,400元[[人民幣|人民幣]]。

== 语言 ==
阿坝州的人口以少数民族为主。阿坝州的人口中 52.3%属于[[藏族|藏族]]（包括[[嘉绒语|嘉绒藏族]]），26.6%属于[[汉族|汉族]]，17.7%属于[[羌族|羌族]]，3.2%属于[[回族|回族]]。

阿坝州的语言分布非常复杂，主要分成三种：原住民语言以[[藏语|藏语]]、[[嘉绒语|嘉绒语]]和[[羌语|羌语]]为主。
