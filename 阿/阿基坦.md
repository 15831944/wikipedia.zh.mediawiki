'''阿基坦'''（{{lang-fr|Aquitaine}}），是[[法国|法国]]西南部一個[[大區|大區]]的名稱，共有[[多爾多涅省|多爾多涅省]]、[[吉倫特省|吉倫特省]]、[[朗德省|朗德省]]、[[洛特-加龍省|洛特-加龍省]]、[[比利牛斯-大西洋省|比利牛斯-大西洋省]]等行政區。

== 地理 ==
阿基坦面積為41,308平方公里，佔全法國總面積的7.6%。阿基坦西鄰[[大西洋|大西洋]]，南接[[西班牙|西班牙]]，東邊是[[南部庇里牛斯|南部庇里牛斯]]大區，北方是[[普瓦圖-夏朗德|普瓦圖-夏朗德]]和[[利穆贊|利穆贊]]兩大區。

主要城市有[[波爾多|波爾多]]、[[波城|波城]]、[[巴約訥|巴約訥]]、[[蒙德馬桑|蒙德馬桑]]、[[比亞里茨|比亞里茨]]、[[貝傑拉克|貝傑拉克]]（Bergerac）和[[佩里格|佩里格]]。

地理上的主要特色有:
* [[銀色海岸|銀色海岸]]（Côte d'Argent）：法國最著名的海岸之一。
* [[阿卡雄|阿卡雄]]（Arcachon）海灣：以[[生蠔|生蠔]]養殖聞名的大型[[潟湖|潟湖]]。
* [[加龍河|加龍河]]和[[多爾多涅河|多爾多涅河]]匯流而成的[[吉倫特省|吉倫特]]河口。
* 阿基坦南部是[[庇里牛斯山|庇里牛斯山]]的一部分。
* [[匹拉沙丘|匹拉沙丘]]（Dune de Pilat）：位於[[阿卡雄|阿卡雄]]附近，是[[歐洲|歐洲]]最大的[[沙丘|沙丘]]。
* [[朗德森林|朗德森林]]（Landes Forest）：西歐最大的松樹林。

== 歷史 ==
在[[羅馬帝國|羅馬帝國]]時代，[[阿基坦高盧行省|阿基坦高盧行省]]最初的範圍是從庇里牛斯山至[[加龍河|加龍河]]，之後，[[屋大維|屋大維]]將加龍河至[[羅亞爾河|羅亞爾河]]間的土地併入。這段時間，阿基坦高盧行省的範圍擴展至[[塞文山脈|塞文山脈]]，整個行省約占現今法國的三分之一。

西元4世紀，阿基坦行省分為三個行省：
* 東北部的'''第一阿基坦'''（Aquitania prima），範圍包括之後的法國行省[[貝里_(法國)|貝里]]，[[波旁_(行省)|波旁]]，[[奧弗涅_(行省)|奧弗涅]]、[[朗格多克_(行省)|朗格多克]]的[[沃萊|沃萊]]（Velay）和[[熱沃當|熱沃當]]（Gévaudan），[[阿基坦_(行省)|吉耶訥]]的[[魯埃格|魯埃格]]（Rouergue）、[[阿爾比喬斯|阿爾比喬斯]]（Albigeois）地區和[[凱爾西|凱爾西]]（Quercy），以及[[馬爾什_(行省)|馬爾什]]。
* 西北部的'''第二阿基坦'''（Aquitania secunda），範圍包括之後的[[波爾多|波爾多]]地區、[[普瓦圖|普瓦圖]]（Poitou）、[[聖東日|聖東日]]（Saintonge）、[[昂古穆瓦|昂古穆瓦]]（Angoumois）和[[阿基坦_(行省)|吉耶訥]]西部。
* 南部的'''第三阿基坦'''（Aquitania tertia 或 Aquitania Novempopulana），範圍包括[[比戈爾|比戈爾]]（Bigorre）、[[科曼日|科曼日]]（Comminges）、[[雅馬邑|雅馬邑]]（Armagnac，或稱阿馬尼亞克）、[[貝阿恩|貝阿恩]]、[[巴斯克地區|巴斯克地區]]、[[加斯科尼|加斯科尼]]等等。

西元5世紀，[[羅馬帝國|羅馬]]瓦解，[[西哥德人|西哥德人]]填補了這段權力真空的狀態。507年，[[法蘭克人|法蘭克人]]與[[勃艮第人|勃艮第人]]等盟友擊敗西哥德人。當[[克洛塔爾二世|克洛塔爾二世]]在629年逝世，他將阿基坦分給兒子[[查理貝爾特二世|查理貝爾特二世]]，首都設立在[[土魯斯|土魯斯]]。西元八世紀，部分[[摩爾人|摩爾人]]來到阿基坦，但遭到[[查理·馬特|查理·馬特]]驅離，之後成為[[卡洛林王朝|卡洛林王朝]]的領土。

在加洛林王朝时代，阿基坦的自主性更进一步。阿基坦作为一个分王国而成为通常的王储的封地。阿基坦国王通常被封给王太子。

[[查理曼|查理曼]]的子嗣不斷瓜分帝國，而阿基坦成為[[紐斯特里亞|紐斯特里亞]]的領地。西元9世紀，當地的伯爵們漸漸脫離王國的統治。[[奧弗涅_(行省)|奧弗涅]]的 {{tsl|en|Bernard Plantevelue|Bernard Plantevelue}} 與[[威廉一世_(阿基坦)|威廉一世]]父子曾自稱[[阿基坦公爵|阿基坦公爵]]。[[威廉五世_(阿基坦)|威廉五世]]在[[普瓦圖|普瓦圖]]建立新阿基坦公國。此時阿基坦範圍包括[[普瓦捷|普瓦捷]]、[[奧弗涅_(行省)|奧弗涅]]和[[土魯斯|土魯斯]]。西元1052年，[[威廉八世_(阿基坦)|威廉八世]]成為[[加斯科尼|加斯科尼]]與阿基坦公爵，形成「君和」體制（Personal union，多國共戴一君）。[[威廉九世_(阿基坦)|威廉九世]]公爵是位[[遊吟詩人|遊吟詩人]]，普瓦捷成為遊吟詩人創作詩歌的重鎮。當[[威廉十世_(阿基坦)|威廉十世]]去世，其女繼承人[[亞奎丹的艾莉諾|亞奎丹的艾莉諾]]嫁給[[路易七世_(法蘭西)|法蘭西路易七世]]，並參與[[十字軍東征|十字軍東征]]，但之後她以同族[[血親|血親]]婚姻無效為藉口，嫁給法國最大的競爭對手[[亨利二世_(英格蘭)|英格蘭亨利二世]]。埃莉諾之子[[理查一世|理查一世]]與[[約翰_(英格蘭)|約翰]]和之後的繼承人成為英格蘭國王及阿基坦公爵。

在[[百年戰爭|百年戰爭]]中，[[愛德華三世_(英格蘭)|英格蘭愛德華三世]]在1361年建立阿基坦公國，但法國在1453年重新奪回該地。之後，阿基坦的歷史就成為[[法國歷史|法國歷史]]的一部份。

== 人口統計 ==
2006年的人口為3,119,778，約佔全法國人口的5%。

=== 語言 ===
[[法語|法語]]是主要語言。 

其他當地語言還有各種[[奧克語|奧克語]]，像是[[加斯科尼語|加斯科尼語]]（Gascon）、[[貝阿恩|貝阿恩]]方言和[[佩里戈爾語|佩里戈爾語]]（Périgord）等等。而最南端則有[[巴斯克語|巴斯克語]]。

來自外地的移民將[[英語|英語]]、[[西班牙語|西班牙語]]、[[阿拉伯語|阿拉伯語]]和其他非當地語言帶進阿基坦。

== 經濟 ==
[[File:Panorama-02-Biarritz-2005-08-05.jpg|250px]]附近的海岸]]
* '''農業'''：
::[[葡萄|葡萄]]是目前阿基坦最大的產物。  
::[[林業|林業]]位於大區北部，包括歐洲最大的[[松樹|松樹]]林。
::[[歐洲牛|牛]]飼育業。
* '''採掘工業'''：
::[[道達爾|道達爾]]等公司在此開採[[天然氣|天然氣]]與[[石油|石油]]。
* '''工業'''：
::製酒、蒸餾及相關產品是該區非常重要的產業及文化。根據[[美國國務院|美國國務院]]，全球約有七億升葡萄酒來自波爾多。
::[[航太工業|航太工業]]，特別是[[達梭|達梭]]系統。
* '''服務業'''：
::教育：在[[波城|波城]]和[[波爾多|波爾多]]的大學生超過80,000名。
::旅遊業極為盛行，特別是[[銀色海岸|銀色海岸]]的陽光和衝浪。[[巴約訥|巴約訥]]、[[比亞里茨|比亞里茨]]、[[聖讓德呂茲|聖讓德呂茲]]和[[昂代伊|昂代伊]]是主要的度假勝地。參觀[[多爾多涅省|多爾多涅]]的 [[法式城堡|法式城堡]]（原指法國13、14世紀時為防禦而設置的城堡。後指封建領主的府邸或莊園。並不完全等同於城堡 castle。），或是前往[[庇里牛斯山|庇里牛斯山]][[徒步|健行]]或[[滑雪|滑雪]]，也是極受歡迎的旅遊活動。因此，有非常多的度假小屋和營地。

阿基坦有許多優秀的體育隊伍。較知名的有：
* [[波爾多足球俱樂部|波爾多足球俱樂部]]，法國最成功的[[足球|足球]]俱樂部之一。

[[橄欖球|橄欖球]]在此地特別受到歡迎，球隊有：
* {{tsl|en|Aviron Bayonnais|Aviron Bayonnais}}
* [[比亞里茨奧林匹克|比亞里茨奧林匹克]]（{{tsl|en|Biarritz Olympique|Biarritz Olympique}}），05-06年[[海尼根盃|海尼根盃]]亞軍。

== 飲食 ==
阿基坦以葡萄酒及相關產品而聞名：
* [[雅文邑|雅文邑]]（Armagnac）白蘭地的產區在阿基坦境內。
* [[貝傑拉克|貝傑拉克]]（Bergerac）葡萄酒產於[[多爾多涅省|多爾多涅省]]。
* [[波爾多|波爾多]]葡萄酒（英國稱為 Claret）產區可能是全世界最知名的[[紅酒|紅酒]]產區。產地有[[龐馬魯|龐馬魯]]（Pomerol）、[[聖愛美濃|聖愛美濃]]（Saint-Emilion）、[[格拉夫|格拉夫]]（Graves）和[[梅多克|梅多克]]。
* [[利萊酒|利萊酒]]（Lillet），產自波爾多的[[加烈葡萄酒|加烈葡萄酒]]。
* 知名的甜酒[[索甸|索甸]]（Sauternes）產於此地。
* 阿基坦西南部也產酒，但通常不被歸為波爾多葡萄酒。
* 遊客常在行程中安排參觀[[酒櫃|酒櫃]]（Wine cellars）。

知名的食物有：
* [[Pâté|Pâté]]（一種混合肉和脂肪的肉醬），包括[[鵝肝醬|鵝肝醬]]和 pâté basque。
* [[卡納蕾|卡納蕾]]
* [[卡酥來砂鍋|卡酥來砂鍋]]（一種白豆燉肉料理）
* [[油封鴨腿|油封鴨腿]]（Confit de canard；Duck confit）
* [[多爾多涅省|多爾多涅省]]的[[松露|松露]]。
