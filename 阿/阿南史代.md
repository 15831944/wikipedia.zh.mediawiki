'''阿南史代'''（{{lang-en|'''Virginia Subbs Anami'''}}，{{bd|1944年||||catIdx= A阿南史代}}）出生于[[美国|美国]][[路易斯安那州|路易斯安那州]][[新奥尔良市|新奥尔良市]]，大学主修东亚历史和亚洲地理，精通[[中文|中文]]、[[日文|日文]]和[[英文|英文]]，为日本籍，日本著名佛学家、汉学家。<ref name="fish " >{{cite web |url=http://www.ce.cn/xwzx/xwrwzhk/peoplemore/200511/04/t20051104_5119227.shtml |title=阿南史代:我的中国文化情缘 |accessdate=2005年11月4日 |publisher=中国经济网 |work= |language =zh-cn}}</ref>

== 生平 ==
1944年，阿南史代出生在[[美国|美国]][[路易斯安那州|路易斯安那州]][[新奥尔良市|新奥尔良市]]，她出生前，这个大城市刚刚遭遇一场[[飓风|飓风]]的洗刷，孩提时叫“吉霓”（Ginny）。<ref name="fish " />

大学时期，阿南史代主修东亚历史和亚洲地理，后来留学台湾，主修中文，他在台湾认识了自己未来的丈夫阿南惟茂。<ref name="fish " />

1983年，阿南史代跟着日本驻中华人民共和国大使的丈夫阿南惟茂来到中国，并且开始研究中国[[佛教|佛教]]和古树。<ref name="fish " />后重走日本佛教高僧圆仁法师的求法之路，写出多部汉学著作。

== 思想 ==
阿南史代受到日本佛教高僧[[圆仁|圆仁]]法师《[[入唐求法巡礼行记|入唐求法巡礼行记]]》的影响，重复了圆仁法师走过的求法之路，她从日本[[福冈|福冈]]，历经[[扬州|扬州]]、[[开封|开封]]、[[西安|西安]]、[[青州|青州]]、[[蓬莱|蓬莱]]、[[文登|文登]]，目的地日本，耗时20年完成。<ref name="fish " />

== 个人生活 ==
阿南史代的公公[[阿南惟畿|阿南惟畿]]是[[大日本帝国陆军|大日本帝国陆军]][[大将|大将]]，在[[第二次世界大战|第二次世界大战]]中担任野战军司令，1945年8月15日，大日本帝国投降的第二天，他[[剖腹|剖腹]]自杀。<ref name="fish " />
阿南史代的婆婆是一个日本[[尼姑|尼姑]]。<ref name="fish " />
阿南史代在[[台湾|台湾]]读书时认识了自己的丈夫[[阿南惟茂|阿南惟茂]]，阿南惟茂文质彬彬、谦逊有礼的绅士风度迷住了她，两人交往日益加深，最终在日本结婚，日本不允许外交官和外国人结婚，阿南史代不得不加入日本国籍。<ref name="fish " /> 两人育有一个儿子一个女儿，儿子是知名中國政治專家、[[東北大學_(日本)|東北大學]]教授[[阿南友亮|阿南友亮]]。<ref name="fish " />
