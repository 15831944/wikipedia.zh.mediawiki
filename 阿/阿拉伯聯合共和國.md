'''阿拉伯聯合共和國'''（{{lang-ar|الجمهورية العربية المتحدة}}），簡稱'''阿联'''，是1958年2月1日由[[埃及|埃及]]與[[敘利亞|敘利亞]]联合组成的泛阿拉伯國家。[[伊拉克|伊拉克]]因局勢不穩而没有加入。1958年3月8日，阿拉伯聯合共和國同[[也门王国|也门王国]]（后来的[[阿拉伯也门共和国|阿拉伯也门共和国]]）组成[[邦联|邦联]]性质的[[阿拉伯合众国|阿拉伯合众国]]。由于埃及的[[霸权主义|霸权主义]]政策，1961年9月28日敘利亞宣佈退出阿拉伯聯合共和國，1961年12月26日[[阿拉伯合众国|阿拉伯合众国]]也宣布解散。但埃及仍然保留阿拉伯聯合共和國的國號直到1971年9月2日為止。

值得留意的是，直到現在，[[埃及|埃及]]、[[敘利亞|敘利亞]]及[[伊拉克|伊拉克]]三國的[[國旗|國旗]]与[[国徽|国徽]]仍然非常相似，都是從阿拉伯聯合共和國的國旗及国徽演變而來；就連[[也门国旗|也门]]的旗幟也與共和國的國旗式樣相似。

== 成立 ==
1952年埃及共和国成立后[[纳赛尔|纳赛尔]]提倡[[泛阿拉伯主义|泛阿拉伯主义]]。这个政策的目的在于阿拉伯国家的[[非殖民化|非殖民化]]，排除[[美国|美国]]、[[英国|英国]]和[[法国|法国]]在[[中东|中东]]和[[北非|北非]]的影响以及解放[[巴勒斯坦|巴勒斯坦]]。保守派的[[沙特阿拉伯|沙特阿拉伯]]、[[伊拉克|伊拉克]]和[[约旦|约旦]]的王室则反对这个政策。叙利亚当时是埃及唯一的支持者。1958年春[[大马士革|大马士革]]要求簽署一份联合条约。同年2月1日该条约正式簽署。叙利亚要求簽署这份条约的原因是[[土耳其|土耳其]]对叙利亚的战争威胁，此前在1957年12月在两国边境上就已经爆发了激烈的战斗。叙利亚觉得自己受到[[中部公約組織|中部公約組織]]的威胁。而埃及则在[[第二次中东战争|第二次中东战争]]后在国际上被孤立。纳赛尔认为两国联合是联合所有阿拉伯国家的第一步。

== 反应 ==
作为对阿拉伯聯合共和國的反应，1958年2月，伊拉克和约旦达成协议也要合并，组成[[阿拉伯联邦|阿拉伯联邦]]。但是，由于同年伊拉克的王朝被推翻，这个计划未能实现。美国派空军入驻约旦，以防止约旦发生类似的政变。政变后的伊拉克却拒绝加入阿拉伯聯合共和國。在[[黎巴嫩|黎巴嫩]]亲西方的[[基督教|基督教]]徒和泛阿拉伯的[[穆斯林|穆斯林]]之间爆发冲突。美国介入后使得这场冲突没有扩大。

== 解散 ==
联盟中的两个国家有许多分歧。埃及把所有在叙利亚从事的企业和银行国有化，并确定[[开罗|开罗]]为整个共和国的首都，几乎所有的政府成员都是埃及人，叙利亚因此觉得自己被欺骗和出卖<ref>{{Harvnb|Stephens|1971|pp=338–9}}</ref>。1961年9月27日叙利亚军队[[政变|政变]]，次日宣布联盟解散。叙利亚将其名称改为[[阿拉伯叙利亚共和国|阿拉伯叙利亚共和国]]，而埃及仍保持阿拉伯聯合共和國的国名。1971年9月2日阿拉伯聯合共和國更名为[[阿拉伯埃及共和国|阿拉伯埃及共和国]]。
