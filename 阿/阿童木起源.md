《'''阿童木起源'''》（{{lang-ja|アトム ザ・ビギニング}}）是[[手塚治虫|手塚治虫]]原作，概念創作由[[結城正美|結城正美]]負責，並由手塚治虫的兒子[[手塚真|手塚真]]負責監修，{{link-ja|笠原徹郎|カサハラテツロー}}繪畫的[[日本|日本]][[漫畫|漫畫]]作品，以《[[小飛俠阿童木|小飛俠阿童木]]》為原案，描寫阿童木誕生前的故事。漫畫在《[[月刊Hero's|月刊Hero's]]》2015年1月號開始連載。電視動畫於2017年4月在日本的[[NHK綜合頻道|NHK綜合頻道]]播出。

== 故事簡介 ==
近未來的日本，遭遇了原因不明的大災害，災後5年日本正步向復興，而在練馬大學年輕的天馬午太郎和御茶水博士正合作研究機械人技術，並成功製作名為A106的機械人，它擁有1,000匹馬力及能自行判斷的人工智能。

== 登場角色 ==
; {{nihongo|A106|A106（エーテンシックス）|[[配音員|聲]]：[[井上雄貴|井上雄貴]]}}
: 天馬和茶水製造的機械人，二人開發A10系列的第6號機。搭載了第7研究室研究項目「Bewuβtsein系統」。

; {{nihongo|A107|A107（エーテンセブン）}}
: A106的妹妹，A10系列第7號機，Bewußtsein人工智能系統正式機種。擁有3000匹馬力、加裝了U字型μ芥子探測器和微波噴流推進器。功能上要優於A106，但性格則遠比後者頑皮、叛逆，是個我行我素的淘氣妹妹。

; {{nihongo|天馬午太郎|天馬 午太郎（てんま うまたろう）|聲：[[中村悠一|中村悠一]]}}
: 練馬大學機械人理工科碩士第7研究室研究生，A10項目的開發者之一。

; {{nihongo|御茶水博志|お茶の水 博志（おちゃのみず ひろし）|聲：[[寺島拓篤|寺島拓篤]]}}
: 練馬大學機械人理工科碩士第7研究室研究生，A10項目的開發者之一。

; {{nihongo|御茶水蘭|お茶の水 蘭（おちゃのみず らん）|聲：[[佐倉綾音|佐倉綾音]]}}
: 博志的妹妹。高中機械人社團的成員，擅長在廢棄物中找出能在機械人用的零部件。特徵是翹起來的頭髮和戴着貝雷帽。對A106感興趣。

; {{nihongo|堤茂理也|堤 茂理也（つつみ もりや）|聲：[[櫻井孝宏|櫻井孝宏]]}}

; {{nihongo|堤茂斗子|堤 茂斗子（つつみ もとこ）|聲：[[小松未可子|小松未可子]]}}
: 堤茂理也的妹妹。
