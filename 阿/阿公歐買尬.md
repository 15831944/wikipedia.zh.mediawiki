{{作品名稱|name=''Dirty Grandpa''|hk=PARTY冇限耆|tw=阿公歐買尬|cn=下流祖父}}是一部於2016年上映的[[美國|美國]][[喜劇片|喜劇片]]，為{{link-en|丹·梅澤|Dan Mazer}}執導。由[[勞勃·狄尼洛|勞勃·狄尼洛]]、[[柴克·艾弗隆|柴克·艾弗隆]]、[[柔伊·德區|柔伊·德區]]和[[奧布瑞·普拉薩|奧布瑞·普拉薩]]主演。電影於2015年1月5日在[[亞特蘭大|亞特蘭大]]開始拍攝。定於2016年1月21日於香港上映，2016年1月22日在美國和臺灣上映。

== 劇情 ==
悶騷的律師在婚前跑去找他風流的阿公所發生的一連串趣事

== 發行 ==
該片最初定於2015年聖誕節發布，但被延遲至2016年8月16日發行<ref>{{cite web|url=http://insidemovies.ew.com/2014/09/09/now-you-see-me-2-dirty-grandpa-release-dates/|title='Now You See Me 2' and 'Dirty Grandpa' get release dates|last=Blake|first=Emily|date=September 9, 2014|accessdate=December 11, 2014|publisher=Entertainment Weekly}}</ref><ref>{{cite web|url=http://deadline.com/2014/12/zoey-deutch-zac-efron-robert-de-niro-dirty-grandpa-cast-1201315913/|title=Zoey Deutch Joins Zac Efron, Robert De Niro In ‘Dirty Grandpa’|date=December 9, 2014|accessdate=July 15, 2015|last=Yamato|first=Jen|publisher=Deadline}}</ref>。上映日最後決定為2016年1月22日<ref name=deadline /><ref name=Aug2015Deadline>{{cite news|last1=Busch|first1=Anita|title=‘Criminal’, ‘Dirty Grandpa’ & ‘Mechanic: Resurrection’ Get New Release Dates|url=http://deadline.com/2015/08/criminal-dirty-grandpa-mechanic-resurrection-get-new-release-dates-1201490561/|accessdate=August 4, 2015|work=deadline.com|date=August 3, 2015}}</ref>。

== 反響 ==
[[爛番茄|爛番茄]]新鮮度僅11%，基於122條評論，平均分為2.8/10<ref>{{cite web |url=http://www.rottentomatoes.com/m/dirty_grandpa/ |title= Dirty Grandpa (2016) |website=[[Rotten_Tomatoes|Rotten Tomatoes]] |accessdate=January 22, 2016}}</ref>，[[Metacritic|Metacritic]]得分只有18<ref>{{cite web |url=http://www.metacritic.com/movie/dirty-grandpa |title=Dirty Grandpa reviews |website=[[Metacritic|Metacritic]] |accessdate=January 22, 2016}}</ref>；[[IMDB|IMDB]]得分6.0，差評佔大多數，甚至被許多影評人認為是2016年最糟的電影之一。

影評人膝關節表示這部不要以去審視的心態去看，雖然有略顯低俗下流的橋段，包裝成人與青少年的集體焦慮，不說教也不閃躲扭捏，在看似胡搞瞎鬧的片段裡，還有一絲溫馨。<ref>https://m.nownews.com/news/1968785</ref>
