'''阿贝·瓦桑特·阿希提卡'''（{{lang-en|'''Abhay Vasant Ashtekar'''}}，{{bd|1949年|7月5日}}）是一名印度理论物理学家。他是[[宾州州立大学|宾州州立大学]]引力与宇宙研究所的领导者，同时也是一名物理教授。作为[[阿希提卡变量|阿希提卡变量]]的创始人，他[[圈量子引力|圈量子引力]]理论及其下属学科[[圈量子宇宙学|圈量子宇宙学]]的提出者之一。他也为非物理学背景的人撰写了很多介绍[[圈量子引力|圈量子引力]]的科普文章。1999年，阿希提卡和他的同事计算出了[[黑洞|黑洞]]的[[熵|熵]]，证实了[[霍金|霍金]]1974年的预测<ref name = 'scientist'/>。[[牛津|牛津]]数学物理学家[[罗杰·彭罗斯|罗杰·彭罗斯]]认为“阿希提卡对于[[量子重力|量子重力]]的研究方法是所有对相对论进行‘量子化’的尝试中最为重要的。”<ref name = 'scientist'/>阿希提卡于2016年5月被推选为美国国家科学院院士<ref>{{Cite web |url=http://www.nasonline.org/news-and-multimedia/news/may-3-2016-NAS-Election.html |title=存档副本 |access-date=2016-05-14 |archive-url=https://web.archive.org/web/20160506052951/http://www.nasonline.org/news-and-multimedia/news/may-3-2016-NAS-Election.html |archive-date=2016-05-06 |dead-url=yes }}</ref>。

== 个人经历 ==
阿希提卡在印度[[马哈拉施特拉|马哈拉施特拉]]邦内长大。于印度获得学士学位之后，阿希提卡进入[[德州大学|德州大学]]的研究生项目就读[[引力|引力学]]<ref name = 'scientist'/>。随后他入读[[芝加哥大学|芝加哥大学]]，在罗伯特·杰勒西的指导下于1978年完成博士学位。阿希提卡先后于[[牛津大学|牛津大学]]、[[巴黎大学|巴黎大学]]、[[雪城大学|雪城大学]]任教，而后在[[宾州州立大学|宾州州立大学]]任教至今<ref>{{cite web|url=http://gravity.psu.edu/people/Ashtekar/cv.html|title=Abhay Ashtekar - CV}}</ref>。

他1986年和克里斯汀·克拉克结缔婚姻<ref name="Rediff on the Net">{{cite web|url=http://www.rediff.com/news/1999/may/15us5.htm|title=The Rediff Profile: Dr. Abhay Ashtekar|publisher=Rediff on the Net}}</ref>，两人育有一子，尼尔·阿希提卡。

== 宗教信仰 ==
虽然阿希提卡是一名[[无神论者|无神论者]]，但他表示他对于科研工作的态度来自于《[[薄伽梵谭|薄伽梵谭]]》{{notetag|《薄伽梵谭》是《印度教》的重要经典。}}。同时他也喜欢阅读印度哲学和其他东方哲学，例如道教和禅宗<ref name="Rediff on the Net"/>。
