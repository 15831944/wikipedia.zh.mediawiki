'''阿列克谢·尼古拉耶维奇·奥夫奇宁'''（{{lang-ru|Алексей Николаевич Овчинин}}，{{IPA-ru|ɐlʲɪkˈsʲej nʲɪkɐˈlaɪvʲɪtɕ ɐfˈt͡ɕinʲɪn}}；{{bd|1971年|9月28日|}}）是[[俄罗斯|俄罗斯]][[宇航员|宇航员]]。
==早年==
1988年8月至1990年9月在[[鲍里索格列布斯克|鲍里索格列布斯克]]高等军事飞行员学校学习，1990年9月至1992年8月在[[叶伊斯克|叶伊斯克]]高等军事飞行员学校学习，毕业后获得飞行工程师资格<ref name=bio>{{cite web |title=OVCHININ Alexey Nikolaevich |url=https://www.energia.ru/en/iss/iss47/ovchinin.html |publisher=RSC "Energia" |accessdate=2019-04-14 |language=en}}</ref>。
==宇航员生涯==
2006年10月11日，他成为宇航员候选人之一，之后开始接受基础训练。2009年6月2日通过考验，9日获得测试宇航员的资格<ref name=bio/>。
===远征47/48===
莫斯科时间19日0时27分（北京时间19日5时27分），带有TMA-20M载人飞船的[[联盟-FG|联盟-FG]]型运载火箭发射升空。这次任务队员包括他和[[奥列格·斯克里波奇卡|奥列格·斯克里波奇卡]]、美国宇航员[[杰弗里·威廉姆斯|杰弗里·威廉姆斯]]。约6小时后，飞船通过快速对接模式与国际空间站实现了对接<ref>{{cite web |author=张继业 |title=俄罗斯向国际空间站发射载人飞船 |url=http://www.xinhuanet.com/world/2016-03/19/c_1118379162.htm |publisher=新华网 |archiveurl=https://web.archive.org/web/20190414092306/http://www.xinhuanet.com/world/2016-03/19/c_1118379162.htm |archivedate=2019-04-14}}</ref>。

[[哈萨克斯坦|哈萨克斯坦]]时间2016年9月7日，他和任务队员乘坐相同飞船返回[[地球|地球]]，成功在[[杰兹卡兹甘|杰兹卡兹甘]]附近着陆<ref>{{cite web |title=国际空间站3名宇航员乘"联盟TMA-20M"飞船返回地球 |url=http://www.taikongmedia.com/Item/Show.asp?m=1&d=21666 |publisher=中国太空网 |accessdate=2019-04-10 |language=zh |date=2016-09-08}}</ref>。
===远征57（失败）===
莫斯科时间2018年10月11日11时40分（北京时间16时40分），载有奥夫奇宁等宇航员的[[联盟MS-10|联盟MS-10]]飞船由[[联盟-FG|联盟-FG]]型运载火箭从[[哈萨克斯坦|哈萨克斯坦]]境内的[[拜科努尔发射场|拜科努尔发射场]]发射升空。起飞约119秒后，[[多節火箭|火箭第一级]]分离，火箭顶端的[[整流罩|整流罩]]被抛下，然而火箭第二级[[发动机|发动机]]却突然关闭，乘员舱与火箭紧急分离，并抛出[[降落伞|降落伞]]着陆。[[俄罗斯航太|俄航天集团]]之后表示，飞船上的宇航员奥夫奇宁和尼克·黑格均告生还，并与地面飞行控制中心取得了联络<ref>{{cite web |author=栾海 |title=俄载人飞船发射失败　宇航员紧急着陆生还 |url=http://www.xinhuanet.com/mil/2018-10/11/c_129969581.htm |publisher=[[新华社|新华社]] |archiveurl=https://web.archive.org/web/20190414090000/http://www.xinhuanet.com/mil/2018-10/11/c_129969581.htm |archivedate=2019-04-14 |date=2018-10-11}}</ref>。

10月12日，他和黑格抵达[[莫斯科|莫斯科]]接受检查。[[俄罗斯联邦卫生部|俄罗斯联邦卫生部]]下属生物医学署署长表示，他们身体状况良好，无需治疗，但将还在[[加加林宇航员培训中心|加加林宇航员培训中心]]停留几天，接受例行身体状况检查<ref>{{cite web |title=历惊险暂无恙　航天员明春再赴太空 |url=http://xinhua-rss.zhongguowangshi.com/233/-1348184867680964213/4806113.html |publisher=新华社 |archiveurl=https://web.archive.org/web/20190414090620/http://xinhua-rss.zhongguowangshi.com/233/-1348184867680964213/4806113.html |archivedate=2019-04-14 |date=2018-10-13}}</ref>。有俄罗斯专家表示，该事故暴露出俄航天业正经历着[[基础设施|基础设施]]老化、航天管理机制不善和人才资源短缺等的深层次危机<ref>{{cite web |author1=栾海|author2=冯玉婧 |title=载人飞船发射失败　俄航天系统危机待解 |url=http://www.xinhuanet.com/world/2018-10/12/c_1123550815.htm |publisher=新华社 |accessdate=2019-04-14 |archiveurl=https://web.archive.org/web/20190119022304/http://www.xinhuanet.com/world/2018-10/12/c_1123550815.htm |archivedate=2019-01-19 |date=2018-10-12}}</ref>。

===远征59/60===
[[莫斯科时间|莫斯科时间]]2019年3月14日22点14分（北京时间15日3点14分），[[联盟-FG|联盟-FG]]型运载火箭从哈萨克斯坦境内的拜科努尔发射升空。数分钟后，载有奥夫奇宁、[[尼克·黑格|尼克·黑格]]和[[克里斯蒂娜·科赫|克里斯蒂娜·科赫]]的联盟MS-12号宇宙飞船与运载火箭分离，进入预定轨道<ref>{{cite web |title=俄罗斯成功发射“联盟MS-12”载人飞船 |url=http://m.news.cctv.com/2019/03/15/ARTIND7hPm78yHnMolYd9Lpm190315.shtml |publisher=[[中国中央电视台|CCTV]] |archiveurl=https://web.archive.org/web/20190414085314/http://m.news.cctv.com/2019/03/15/ARTIND7hPm78yHnMolYd9Lpm190315.shtml |archivedate=2019-04-14 |date=2019-03-15}}</ref>。
