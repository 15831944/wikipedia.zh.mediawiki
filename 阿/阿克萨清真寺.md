'''阿克萨清真寺'''（[[阿拉伯语|阿拉伯语]]：{{rtl-lang|ar|المسجد الاقصى}}，{{IPA|'ælmæsʤıd ælæqəsɒː}}，{{Audio|ArAqsaMosque.ogg|''Al-Masjid Al-Aqsa''}}，[[英語|英語]]：{{lang|en|'''al-Aqsa mosque'''}}），位于[[耶路撒冷旧城|耶路撒冷旧城]][[聖殿山|聖殿山]]。“阿克萨”在[[阿拉伯语|阿拉伯语]]中意为“遥远”，故本寺又名'''远寺'''。虽然《[[古蘭经|古蘭经]]》第17章第7节指[[禁寺|禁寺]]上有犹太人的[[耶路撒冷圣殿|圣殿]]（我说：『如果你们行善，那末，你们是为自己而行善，如果你们作恶，那么，你们是为自己而作恶。』当第二次作乱的约期来临的时候，（我又派遣他们），以便他们使你们变成为愁眉苦脸的，以便他们像头一次那样再入禁寺，以便他们把自己所占领的地方加以摧毁。）
 
阿克萨紧邻著名的[[圆顶清真寺|圆顶清真寺]]（又称：金顶清真寺、岩石清真寺、登霄石清真寺）相传为[[穆罕默德|穆罕默德]][[夜行登霄|夜行登霄]]之处。

== 历史 ==
该寺由[[倭马亚王朝|倭马亚王朝]]第5代[[哈里发|哈里发]][[阿布杜勒·马利克|阿布杜勒·马利克]]及其子在公元705年至709年间建造在原圣殿教堂的残基之上。780年毁于地震，932年由阿巴斯王朝第19任哈里发重建。11世纪加盖了圆顶，改建后的清真寺高至88米。1099年，该寺在[[第一次十字军东征|第一次十字军东征]]中被入侵的十字军改造成教堂、兵器库和兵营。1187年耶路撒冷被[[萨拉丁|萨拉丁]]易主后又进行了修复。

2017年7月14日，3名以色列警察在[[圣殿山|圣殿山]]外遭以色列籍阿拉伯人枪击，其中2人身亡，袭击者随后逃到圣殿山。以色列警方在事发后封锁了圣殿山上的阿克萨清真寺，当天正是周五[[主麻|主麻]]日，这是半世纪来[[穆斯林|穆斯林]]第一次不被允许周五在阿克萨清真寺祷告。2017年7月16日下午，以色列警方重开[[耶路撒冷|耶路撒冷]]老城内之前遭关闭的商铺及被封锁的交通，并重开阿克萨清真寺，但在阿克萨清真寺入口处新设的金属探测器及安全摄像头遭穆斯林抵制。2017年7月18日，据“阿拉伯新闻报”网站援引[[阿拉伯语|阿拉伯语]]媒体Elaph的报道称，[[沙特阿拉伯国王|沙特阿拉伯国王]][[萨勒曼·本·阿卜杜勒-阿齐兹·阿勒沙特|萨勒曼]]经私人途径通过[[美国|美国]]高级别官员斡旋获得了[[以色列总理|以色列总理]][[本雅明·内塔尼亚胡|内塔尼亚胡]]“重开阿克萨清真寺”的承诺。但2017年7月19日晚，以色列警方再度关闭阿克萨清真寺。正在[[中華人民共和國|中国]]进行国事访问的[[巴勒斯坦国总统|巴勒斯坦国总统]][[马哈茂德·阿巴斯|阿巴斯]]决定提前结束访问回国<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1737944 |title = 以色列再度关闭阿克萨清真寺，阿巴斯中断访华回国紧急应对 |author =  |date =2017-07-20  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

== 布局 ==
礼拜大殿长90米，高8米，宽36米，内有53根大理石圆柱和49根方柱。
