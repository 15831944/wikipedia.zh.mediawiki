'''何塞·亚历山大·柯拉'''（{{lang-en|José Alexander Cora}}，{{bd|1975年|10月18日|||Cora,Alex}}）曾是一位来自[[波多黎各|波多黎各]]的[[美国职棒大联盟|美国职棒大联盟]][[内野手|内野手]]，目前担任[[波士顿红袜|波士顿红袜]]的[[总教练_(棒球)|总教练]]。他的大学棒球生涯在[[迈阿密大学_(佛罗里达州)|迈阿密大学]]度过，随后在{{mlby|1998}}至{{mlby|2011}}期间先后效力于[[洛杉矶道奇|洛杉矶道奇]]、[[克里夫兰印地安人|克里夫兰印地安人]]、[[波士顿红袜|波士顿红袜]]、[[纽约大都会|纽约大都会]]、[[德州游骑兵|德州游骑兵]]、[[华盛顿国民|华盛顿国民]]。 退役后柯拉先在[[ESPN|ESPN]]的棒球分析专家，随后回到联盟担任教练和总教练。

==球员生涯==
===早年生涯===
柯拉在[[1993年美国职棒大联盟选秀|1993年美国职棒大联盟选秀]]上被[[明尼苏达双城|明尼苏达双城]]在第12轮选中，但是并没有签约，选择进入[[迈阿密大学_(佛罗里达州)|迈阿密大学]]打大学棒球。大学棒球期间，他在1995年和1996年两次入选[[大学世界大赛|大学世界大赛]]最佳阵容。1996年他带领球队进入到冠军战，但最后输给了[[路易斯安那州立大学|路易斯安那州立大学]]。柯拉在2006年入选迈阿密大学的体育名人堂<ref name=miamihof>{{cite web |url=http://www.umsportshalloffame.com/alex-cora.html |title=ALEX CORA, 1993-1996 |website=umsportshalloffame.com |accessdate=2018年7月25日 |archive-url=https://web.archive.org/web/20171022085958/http://www.umsportshalloffame.com/alex-cora.html |archive-date=2017年10月22日 |dead-url=no }}</ref>。

[[1996年美国职棒大联盟选秀|1996年美国职棒大联盟选秀]]之前，柯拉被[[棒球美国|棒球美国]]杂志评为大学最佳守备球员。他在第三轮被[[洛杉矶道奇|洛杉矶道奇]]选中，当球季在[[美國職棒小聯盟#高階1A（Class_A_Advanced）|高阶1A]]的[[维罗海滩魔鬼鱼|维罗海滩道奇]]打了61场比赛，交出了2成57打击率、无全垒打、26分打点的成绩。1997年他在2A的[[圣安东尼奥任务|圣安东尼奥任务]]效力，在127场比赛中打出2成34打击率、3支全垒打、48份打点的成绩。随后的1998年、1999年和2000年三个赛季柯拉在[[阿尔伯克基公爵|阿尔伯克基公爵]]分别出赛81、80和30场，打击率为2成64、3成08和3成73。

===洛杉矶道奇===
{{mlby|1998}}[[6月7日|6月7日]]，柯拉完成了他的大联盟首秀，代表[[洛杉矶道奇|洛杉矶道奇]]出战[[西雅图水手|西雅图水手]]，他的哥哥{{link-en|乔伊·柯拉|Joey Cora}}在该场比赛中担任水手队的[[二垒手|二垒手]]<ref>{{cite web |url=http://www.retrosheet.org/boxesetc/1998/B06070SEA1998.htm |title=Los Angeles Dodgers 7, Seattle Mariners 4 |website=Retrosheet |date=1998年6月7日}}</ref>。随后的七个赛季柯拉都在道奇队效力，一共出赛684场，打击率2成46，全垒打27支，打点173分。在道奇队效力期间，他主要担任二垒手和[[游击手|游击手]]。{{mlby|2000}}和{{mlby|2001}}，由于道奇队把资深球员{{link-en|马克·古兹拉尼克|Mark Grudzielanek}}移防至二垒，柯拉多数时间担任游击手。{{mlby|2002}}随着{{link-en|塞萨尔·伊兹图里斯|César Izturis}}的出现以及古兹拉尼克被交易到[[芝加哥小熊|芝加哥小熊]]，柯拉回到了二垒手的位置。

柯拉在道奇效力期间只进入过1次季后赛。在[[2004年国家联盟分区赛|2004年国家联盟分区赛]]中，道奇以1比3的总比分输给了[[圣路易红雀|圣路易红雀]]。系列赛的四场比赛柯拉均担任二垒手，15个打数仅打出2支安打。

===克里夫兰印地安人===
{{mlby|2005}}1月，柯拉以[[自由球员|自由球员]]的身份和[[克里夫兰印地安人|克里夫兰印地安人]]。他共出赛49场，其中22场是先发游击手，14场是先发二垒手，打击率2成05，全垒打1支，打点8分。{{mlby|2005}}7月7日，柯拉被交易到[[波士顿红袜|波士顿红袜]]，交换{{link-en|雷蒙·巴斯克斯|Ramón Vázquez}}。

===波士顿红袜===
[[File:Alexcora.jpg|280px]]，他完成了一个双杀守备]]
{{mlby|2005}}7月7日，柯拉第一次代表[[波士顿红袜|波士顿红袜]]出场，该赛季他出场47次，打击率2成69，全垒打2支，打点16分。在[[2005年美国联盟分区赛|2005年美国联盟分区赛]]中，红袜队被[[芝加哥白袜|芝加哥白袜]]3比0横扫，柯拉仅上场守备1次，没有打击。

{{mlby|2006}}柯拉原本计划作为[[埃德加·伦特利亚|埃德加·伦特利亚]]的替补，随着伦特利亚被交易到[[亚特兰大勇士|亚特兰大勇士]]后，柯拉有机会成为先发游击手。但红袜队又在{{mlby|2006}}2月引进[[阿莱克斯·冈萨雷斯|阿莱克斯·冈萨雷斯]]，柯拉回到替补的位置。{{mlby|2006}}柯拉出赛96场，打击率2成38，全垒打1支，打点18分。

{{mlby|2007}}柯拉在常规赛出赛83场，打击率2成46，全垒打3支，打点18分。在该赛季的季后赛中，他在[[2007年美国联盟冠军赛|美联冠军赛]]和[[2007年世界大赛|世界大赛]]中分别替补上场守备2次，他随队获得了该赛季的世界大赛冠军。

{{mlby|2008}}柯拉在常规赛出赛75场，打击率2成70，没有全垒打，打点9分。季后赛他出赛4场，分别是2场[[2008年美国联盟分区赛|美联分区赛]]和2场[[2008年美国联盟冠军赛|美联冠军赛]]，他在26个打数中击出4支安打，打回1分打点。这也是柯拉作为球员最后一次在季后赛亮相。

{{mlby|2008}}10月30日，柯拉成为一名[[自由球员|自由球员]]。在红袜效力的四年间，柯拉共出场301次，打击率2成52，全垒打6支，打点61分。

===纽约大都会===
{{mlby|2009}}1月22日，柯拉和[[纽约大都会|纽约大都会]]签下一年的合约<ref>{{cite press release |url=http://newyork.mets.mlb.com/news/press_releases/press_release.jsp?ymd=20090122&content_id=3764362&vkey=pr_nym&fext=.jsp&c_id=nym |title=Mets sign Alex Cora to a one-year contract |website=[[MLB.com|MLB.com]] |date=2009年1月22日 |accessdate=2017年10月20日 |archive-url=https://web.archive.org/web/20171021003548/http://newyork.mets.mlb.com/news/press_releases/press_release.jsp?ymd=20090122&content_id=3764362&vkey=pr_nym&fext=.jsp&c_id=nym |archive-date=2017-10-21 |dead-url=yes }}</ref>。{{mlby|2009}}他共出赛82场，打击率2成51，全垒打1支，打点18分。赛季结束后柯拉与大都会队续签了一年的合同，还包括了一年的选项合同。但在{{mlby|2010}}[[8月7日|8月7日]]大都会队释出了柯拉<ref>{{cite web |author=安东尼·迪克莫 |url=http://m.mlb.com/news/article/13147094 |title=Mets call up F-Mart, Tejada; Cora cut |website=[[MLB.com|MLB.com]] |date=2010年8月7日 |accessdate=2017年10月20日 |archive-url=https://web.archive.org/web/20171021060151/http://m.mlb.com/news/article/13147094 |archive-date=2017年10月21日 |dead-url=no }}</ref>。在大都会效力的一个半赛季期间，柯拉共出赛144场，打击率2成34，全垒打1支，打点34分。

===德州游骑兵===
{{mlby|2010}}8月17日，柯拉和[[德州游骑兵|德州游骑兵]]签下小联盟合约，他先被指派到3A的[[俄克拉何马城道奇|俄克拉何马城红鹰]]，出赛6场后被游骑兵队叫上了大联盟，他在大联盟共出场4次。9月7日，游骑兵解除了柯拉的合约。尽管在游骑兵队效力时间很短，但之后球队还是给了他一枚[[2010年美国联盟冠军赛|美联冠军]]戒指。

===华盛顿国民===
[[File:Alex_Cora_2011_(cropped).jpg|thumb]]的柯拉]]
{{mlby|2011}}1月，柯拉和[[华盛顿国民|华盛顿国民]]签下小联盟合约<ref>{{cite web |last=拉德森 |first=比尔 |url=http://m.nationals.mlb.com/news/article/16448186// |title=Cora lands Minor League deal with Nationals |website=[[MLB.com|MLB.com]] |date=2011年1月17日 |accessdate=2017年10月20日 |archive-url=https://web.archive.org/web/20171021004637/http://m.nationals.mlb.com/news/article/16448186 |archive-date=2017-10-21 |dead-url=yes }}</ref>。该赛季柯拉为国民队出赛91场，交出了2成24打击率，6分打点的成绩。柯拉最后一次为国民队出场是{{mlby|2011}}9月28日，他在对阵[[佛罗里达马林鱼|佛罗里达马林鱼]]的比赛中上场代打，击出了一支[[三垒安打|三垒安打]]<ref>{{cite web |url=http://www.retrosheet.org/boxesetc/2011/B09280FLO2011.htm |title=Washington Nationals 3, Florida Marlins 1 |website=Retrosheet |date=2011年9月28日}}</ref>。

===球员生涯晚期===
2011年赛季结束后，柯拉回到[[波多黎各|波多黎各]]打冬季联盟比赛，之后他宣布退役；但没过多久他澄清只是从冬季联盟中退役<ref>{{cite web |first=Cheryl |last=Nichols |url=http://districtsportspage.com/former-nats-alex-cora-announces-retirement/4291 |title=Former Nats: Alex Cora announces retirement from winter ball (Updated) |website=districtsportspage.com |date=2012年1月25日 |accessdate=2018年7月25日 |archive-url=https://web.archive.org/web/20171017145952/http://districtsportspage.com/former-nats-alex-cora-announces-retirement/4291/ |archive-date=2017年10月17日 |dead-url=no }}</ref>。

{{mlby|2012}}2月5日，柯拉同意和[[圣路易斯红雀|圣路易斯红雀]]签订小联盟合同<ref>{{cite web |url=http://m.cardinals.mlb.com/news/article/26579836// |title=Cora joins Cards on Minor League deal |first=Jesse |last=Sanchez |website=[[MLB.com|MLB.com]] |date=2012年2月6日 |access-date=2017年10月22日 |archive-url=https://web.archive.org/web/20171022085205/http://m.cardinals.mlb.com/news/article/26579836// |archive-date=2017年10月22日 |dead-url=no }}</ref>。他在春训中上场打击24次，打击率2成08，打点1分。随后在3月25日卫冕冠军决定和柯拉解约<ref>{{cite web |url=http://ktvo.com/sports/content/cardinals-release-alex-cora--koyie-hill?id=734379 |title=Cardinals release Alex Cora & Koyie Hill |agency=[[美联社|美联社]] |website=KTVO |date=2012年3月25日 |accessdate=2018年7月25日 |archive-url=https://web.archive.org/web/20171021061157/http://ktvo.com/sports/content/cardinals-release-alex-cora--koyie-hill?id=734379 |archive-date=2017年10月21日 |dead-url=no }}</ref>。

===著名比赛===
{{mlby|2004}}[[5月12日|5月12日]]，道奇队的柯拉在面对[[芝加哥小熊|芝加哥小熊]]的先发投手{{link-en|马特·克莱蒙特|Matt Clement}}的一个打数中缠斗了18球，面对2坏1好的球数时，柯拉连续打了14个界外球，最终打出了全垒打<ref>{{cite web |url=https://www.youtube.com/watch?v=X4JuBmzjNsE |title=Alex Cora - MLB Record - May 12, 2004 |via=[[YouTube|YouTube]] |accessdate=2017年10月21日 |archive-url=https://web.archive.org/web/20150811015731/https://www.youtube.com/watch?v=X4JuBmzjNsE |archive-date=2015年8月11日 |dead-url=no }}</ref>。这是80年代中期有记录球数以来第三长的单一打席用球数<ref>{{cite web |url=http://m.mlb.com/news/article/1469823/ |title=Mastering art of the prolonged at-bat |first=Tom |last=Singer |website=[[MLB.com|MLB.com]] |date=2006年5月26日 |accessdate=2017年10月21日 |archive-url=https://web.archive.org/web/20171021215704/http://m.mlb.com/news/article/1469823/ |archive-date=2017年10月21日 |dead-url=no }}</ref>。他的兄弟乔伊开玩笑的说，当时他和他的一个朋友在餐馆里边吃饭边观看这场比赛，第一球的时候他们点了第一杯啤酒，当阿莱克斯击出全垒打的时候他们“已经喝醉了，准备叫辆出租车回家”。<ref>{{cite web |url=http://www.boston.com/sports/baseball/redsox/articles/2005/07/16/an_at_bat_for_the_ages/ |title=An at-bat for the ages |first=Amalie |last=Benjamin |website=Boston.com |date=2005年7月16日 |accessdate=2018年7月25日 |archive-url=https://web.archive.org/web/20160304055558/http://www.boston.com/sports/baseball/redsox/articles/2005/07/16/an_at_bat_for_the_ages/ |archive-date=2016年3月4日 |dead-url=no }}</ref>

柯拉参与了两场大联盟历史上耗时最长的九局比赛。第一场是{{mlby|2001}}10月5日道奇队对[[旧金山巨人|旧金山巨人]]的比赛，共耗时4小时27分钟<ref name=ba-longest>{{cite web |url=http://www.baseball-almanac.com/recbooks/rb_gmlg.shtml |title=Game Length Records |website=Baseball Almanac |accessdate=2018年7月25日 |archive-url=https://web.archive.org/web/20180713231709/http://www.baseball-almanac.com/recbooks/rb_gmlg.shtml |archive-date=2018年7月13日 |dead-url=no }}</ref>，柯拉在第八局上场守备<ref>{{cite web |url=http://www.retrosheet.org/boxesetc/2001/B10050SFN2001.htm |title=Los Angeles Dodgers 11, San Francisco Giants 10 |website=Retrosheet |date=2001年10月5日}}</ref>。第二场是{{mlby|2006}}8月18日红袜队对[[纽约洋基|纽约洋基]]的双重赛第二场，共耗时4小时45分钟<ref name=ba-longest/>，他作为先发游击手打满全场<ref>{{cite web |url=http://www.retrosheet.org/boxesetc/2006/B08182BOS2006.htm |title=New York Yankees 14, Boston Red Sox 11 (2) |website=Retrosheet |date=2006年8月18日}}</ref>。

===国际赛生涯===
柯拉代表波多黎各参加了[[2006年世界棒球经典赛|2006年]]和[[2009年世界棒球经典赛|2009年]]两届[[世界棒球经典赛|世界棒球经典赛]]；他打击率分别为1成33（15支2）和1成25（8支1）.<ref>{{cite web |url=https://www.worldbaseballclassic.com/stats/ |title=Stats – World Baseball Classic |website=worldbaseballclassic.com |accessdate=2017年10月21日}}</ref>。柯拉2016年起担任[[波多黎各棒球代表队|波多黎各棒球代表队]][[总教练_(棒球)|总教练]]，直到他{{mlby|2018}}担任[[波士顿红袜|波士顿红袜]]总教练<ref>{{cite web |url=http://www.elnuevodia.com/cuentame/cuentame/nota/alexcoraapuestaaunaalineacionexplosivaconcorreaylindor-2256105/ |title=Alex Cora apuesta a una alineación explosiva con Correa y Lindor |first=Noel Algarín |last=Martínez |website=elnuevodia.com |date=2016年10月27日 |accessdate=2017年10月20日 |language=西班牙语 |archive-url=https://web.archive.org/web/20170729174925/https://www.elnuevodia.com/cuentame/cuentame/nota/alexcoraapuestaaunaalineacionexplosivaconcorreaylindor-2256105/ |archive-date=2017年7月29日 |dead-url=no }}</ref>。

===教练===
