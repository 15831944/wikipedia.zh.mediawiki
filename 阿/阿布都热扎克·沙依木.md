'''阿布都热扎克·沙依木'''（{{bd|1957年|9月||}}）[[维吾尔族|维吾尔族]]，[[新疆|新疆]][[柯坪|柯坪]]人，[[中华人民共和国|中华人民共和国]]官员<ref name=xhw/>。

==生平==
1976年11月到1978年9月，在[[中央民族学院|中央民族学院]]文化补习班学习。1978年9月参加工作。1978年9月到1982年5月，任[[新疆维吾尔自治区教育厅|新疆维吾尔自治区教育厅]]成人招生办公室干事。1982年5月至1989年5月，任新疆维吾尔自治区成人招生办公室干事。其间，1980年9月到1983年7月在[[新疆大学|新疆大学]]夜大[[汉语|汉语]]专业学习。1985年12月加入[[中国共产党|中国共产党]]。1989年5月到1992年10月，任新疆维吾尔自治区招生办公室干部。其间，1989年9月到1991年7月在[[新疆师范高等专科学校|新疆教育学院]][[维吾尔文学|维吾尔文学]]专业学习。1992年10月到1995年2月，任新疆维吾尔自治区招生办公室副主任。1995年2月到1998年5月，任新疆教育报刊社副总编辑。其间，1995年9月到1996年7月在[[中共中央党校|中央党校]]新疆班学习<ref name=xhw/>。

1998年5月到2000年1月，任新疆维吾尔自治区勤工俭学办公室主任。其间，1995年9月到1998年7月在中央党校在职研究生班经济管理专业学习。1996年12月到1998年12月，在中共[[温宿县|温宿县]]委挂职担任副书记<ref name=xhw/>。

2000年1月到2003年3月，任中共[[新疆教育出版社|新疆教育出版社]]党总支副书记、社长、副总编辑。2003年3月至2003年8月，中共新疆教育出版社党委副书记、社长、副总编辑。2003年8月起，任中共[[新疆社会科学院|新疆社会科学院]]党委委员、副院长。其间，2004年9月到2005年1月在中央党校新疆班学习<ref name=xhw>{{Cite web |url = http://news.xinhuanet.com/politics/2016-09/29/c_129306167.htm |title = 新疆社会科学院副院长阿不都热扎克·沙依木被调查 |author =  |date =2016-09-29  |publisher =新华网 |language =  |accessdate =  |quote =  }}</ref>。

2016年9月，据新疆维吾尔自治区纪委消息，阿布都热扎克·沙依木涉嫌严重违纪，正在接受组织调查<ref name=xhw/>。2017年2月，经[[中共新疆维吾尔自治区党委|中共新疆维吾尔自治区党委]]批准，[[中共新疆维吾尔自治区纪委|中共新疆维吾尔自治区纪委]]给予阿布都热扎克·沙依木开除党籍、开除公职处分，收缴违纪所得，将涉嫌犯罪问题移送司法机关<ref>{{Cite web |url = http://politics.people.com.cn/n1/2017/0526/c1001-29300873.html |title =新疆公开曝光八起违反中央八项规定精神典型问题  |author =  |date = 2017-05-26 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。2017年4月，据[[中华人民共和国最高人民检察院|中华人民共和国最高人民检察院]]网站消息，[[新疆维吾尔自治区人民检察院|新疆维吾尔自治区人民检察院]]经审查决定，依法对阿布都热扎克·沙依木（副厅级）以涉嫌受贿罪立案侦查并采取强制措施<ref>{{Cite web |url = http://news.ifeng.com/a/20170410/50912759_0.shtml |title = 新疆社会科学院原副院长阿布都热扎克·沙依木被立案侦查 |author =  2017-04-10|date = 凤凰网 |publisher =  |language =  |accessdate =  |quote =  }}</ref>。
