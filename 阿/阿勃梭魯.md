'''阿勃梭魯'''（{{lang-ja|'''アブソル'''}}；{{lang-en|'''Absol'''}}），是《[[精靈寶可夢|精靈寶可夢]]》系列登場的{{NUMBEROFPOKEMON}}種虛構[[角色|角色]]（[[怪物|怪獸]]）中的一種。

== 特徵 ==
阿勃梭魯為惡屬性的災禍寶可夢。身體主要顏色為白色，右側頭部有著[[鐮刀|鐮刀]]狀的角。

主要棲息於險峻的山地，對於環境的變化很敏感，並可以預知災禍的發生。

== 遊戲中的阿勃梭魯 ==
*在[[神奇寶貝_紅寶石·藍寶石|紅寶石、藍寶石、綠寶石版]]中首度出現。豐緣聯盟編號為152，神奧聯盟編號為209（白金版），全國圖鑑編號為359。 
* [[神奇寶貝_紅寶石·藍寶石|紅寶石、藍寶石、綠寶石版]]中，野生的阿勃梭魯可在120號道路遇見，但機率不高。
*[[神奇寶貝_鑽石·珍珠|鑽石、珍珠版]]中，野生的阿勃梭魯在得到全國圖鑑並大量出現時，可於213號道路遇到；而[[神奇寶貝_鑽石·珍珠|白金版]]中可在天冠山頂的草叢中遇見。
*在「攻擊」上的數值高達130，但其餘能力稍偏低。
*特性為「壓迫感」與「超幸運」。「壓迫感」的效果是被對手攻擊時使對手使用的該招式PP一次消耗2；而「超幸運」則是容易擊中對手要害。
*[[神奇寶貝_紅寶石·藍寶石|紅寶石、藍寶石、綠寶石版]]中，四天王的花月使用的寶可夢。

== 動畫中的阿勃梭魯 ==
*《[[神奇寶貝超世代|神奇寶貝超世代]]》107集中登場，因預知利亞多村將發生危險而出現在村民面前，但卻被村民認為是引起災禍的根源，後來經過離開利亞多村調查真相的小聰解釋後，才開始相信阿勃梭魯，並解決了利亞多村的危機。
*《[[神奇寶貝超世代|神奇寶貝超世代]]》中，小遙的勁敵小瞬使用的寶可夢，在關都的大型慶典中登場，有著一定的實力，在與小遙交手時，敗給了小遙。

== 電影中的阿勃梭魯 ==
在《[[神奇寶貝劇場版：七夜的許願星_基拉祈|七夜的許願星 基拉祈]]》中登場。
