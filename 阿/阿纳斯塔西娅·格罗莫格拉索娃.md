'''阿纳斯塔西娅·格罗莫格拉索娃'''（{{lang-ru|Анастасия Николаевна Громогласова}}，1984年9月18日 - ），是一位[[俄罗斯|俄罗斯]][[古典音乐|古典]][[钢琴|钢琴家]]。

== 生平和教育 ==
'''阿纳斯塔西娅•格罗莫格拉索娃''' 是一位俄罗斯钢琴家，赢得过多个国际奖项。

她从1994年开始在[[莫斯科|莫斯科]]居住，并进入[[莫斯科音樂學院|莫斯科音乐学院]]学习。2009年她以“最优秀成绩”毕业并参加学士后课程直至2012年。

阿纳斯塔西娅在重要国际比赛中赢得过多个奖项，其中包括个人表演以及与她的妹妹柳博芙•格罗莫格拉索娃（Liubov Gromoglasova）的双钢琴合奏。

== 获得奖项 ==

=== 钢琴独奏 ===

: 2011年，在巴黎举办的斯克里亚宾国际钢琴比赛中获得第一名。
: 2011年，在雅马哈音乐节Gradusad Parnassum中作为独奏者在莫斯科音乐学院“拉赫玛尼诺夫大厅”演奏。<ref>活动日历-莫斯科音乐学院-雅马哈节Gradusad Parnassum- <span> </span>: [http://www.mosconsv.ru/ru/concert.aspx?id=126472 ''链接到该网站(俄罗斯)''].</ref>
: 2012年，在巴里举办的国际钢琴比赛中获得第一名。
: 2013年，在巴里与巴里交响乐团合作进行[[Piano_Concerto_No._23_(Mozart)|莫扎特第23钢琴协奏曲]]K.488首演。

=== 双钢琴合奏 ===
阿纳斯塔西娅一直与她的妹妹柳博芙进行双钢琴演出；柳博芙也毕业于莫斯科音乐学院。她们获得过下列奖项：<ref>阿纳斯塔西娅和柳博芙在youtube上：[https://www.youtube.com/user/gromoglas77?feature=watch ''youtube官方网页''].</ref>

: 2006年，在鲁宾斯坦音乐比赛（莫斯科）中获得第三名。
: 2007年，获得伦敦国际音乐比赛第一名。
: 2008年，在肖斯塔科维奇室内乐与双钢琴比赛（莫斯科）中获得第二名并赢得特别奖；成为"俄罗斯表演艺术"基金资助学生。
: 2011年，获得[[沃洛格达|沃洛格达]]国际双钢琴比赛第一名。
: 2012年，在巴里举办的国际钢琴比赛中获得双钢琴第一名。

== 演奏曲目 ==
她的演奏曲目包括巴赫、莫扎特、海顿、门德尔松、舒伯特、勃拉姆斯、肖邦、拉赫曼尼诺夫、斯克里亚宾、肖斯塔科维奇、德彪西、拉威尔、卢托斯瓦夫斯基、比才、普罗科菲耶夫等的作品。<ref>[http://www.anastasiaclassica.com/eng/repertoire.html 阿纳斯塔西娅Gromoglasova官方网站(Repertoire)].</ref>
