'''阿瓦鲁阿'''是[[库克群岛|库克群岛]]的[[首都|首都]]，位于[[拉罗汤加岛|拉罗汤加岛]]北部，也是[[库克群岛|库克群岛]]的一个区。市中心位于21°12′S 159°46′W 。

阿瓦鲁阿区的人口有5445 (2006年统计)，收入主要来源于[[旅游业|旅游业]]，贸易，[[渔业|渔业]]，[[农业|农业]]。

主要街道为Ara Maire Nui,境内有[[阿瓦鲁阿国际机场|阿瓦鲁阿国际机场]]（[[国际航空运输协会机场代码|国际航空运输协会机场代码]]：RAR）

== 区划 ==

阿瓦鲁阿分为18个Tapere（层级低于“区”的区划），从西到东依次是：
# Pokoinu
# Nikao (库克群岛议会大厦所在地)
# Puapuautu
# Areanu
# Kaikaveka
# Atupa
# Avatiu (商业港口)
# Ruatonga
# Tutakimoa
# Tauae ( 从Tutakimoa往内, 唯一不靠海的tapere)
# Takuvaine (阿瓦鲁阿市中心, 库克群岛政府所在地, 有阿瓦鲁阿渔港)
# Ngatipa
# Vaikai
# Tapae-I-Uta
# Pue
# Punamaia
# Kiikii
# Tupapa
