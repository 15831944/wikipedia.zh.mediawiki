'''阿爾達''' ''Arda''，是[[英國|英國]]作家[[約翰·羅納德·瑞爾·托爾金|約翰·羅納德·瑞爾·托爾金]]作品中的世界。在[[埃奴的大樂章|埃奴的大樂章]]之時所造，阿爾達在[[一亞|一亞]]（宇宙）之內，安排為人類和精靈的居住地。

阿爾達當初是平面的世界，有如一個大圓盤，邊界是[[黑夜之牆|黑夜之牆]]。阿爾達被[[外環海|外環海]]所包圍，[[貝爾蓋烈海|貝爾蓋烈海]]將阿爾達分為兩大部分，西邊的[[阿門洲|阿門洲]]，和東邊的[[中土大陸|中土大陸]]。[[第一紀元|第一紀元]]中，大部分人類和精靈都居住在[[貝爾蘭|貝爾蘭]]地區。第一紀元末[[憤怒之戰|憤怒之戰]]後，貝爾蘭地區毀滅沉沒。

[[第二紀元|第二紀元]]，一座五星形海島[[努曼諾爾帝國|努曼諾爾]]自貝爾蓋烈海中央升起，[[伊甸人|伊甸人]]三大家族遷入那裡建國。但以後努曼諾爾帝國的人類渴求長生不死，出兵攻打阿門洲，導致努曼諾爾帝國毀滅。

努曼諾爾帝國毀滅以後，阿門洲被挪移，移到世界之外，阿爾達變成球形，很多新海新地被造出來。唯有精靈可以沿住[[筆直航道|筆直航道]]前往[[維林諾|維林諾]]，一去不回頭。

阿爾達南方是[[黑暗大陸|黑暗大陸]]，極東方是[[清晨之門|清晨之門]]。

托爾金{{citation needed|陳述了許多次}}，他所描述的世界正是我們所在的世界，由此推測，中土大陸是指[[亞洲|亞洲]]和[[歐洲|歐洲]]（[[地中海|地中海]]的[[拉丁語|拉丁語]]正是指中土大陸）。黑暗大陸是指[[非洲|非洲]]。而清晨之門則沒有定論，可能是指[[大洋洲|大洋洲]]諸島。新海新地，則指南北[[美洲|美洲]]。不過托爾金從沒有言明人們所推測是否真確。

== 原始阿爾達 ==

原始阿爾達是一個均稱的世界，兩盞巨燈照耀阿爾達，但[[米爾寇|米爾寇]]打破兩盞巨燈，其中的大火席捲阿爾達，世界形成新面貌。

== 阿爾達被毀損 ==

雖然魔苟斯被逐出宇外，但他的惡毒依然籠罩阿爾達，疾病、嚴寒、酷熱，皆不在[[伊露維塔|伊露維塔]]的計劃中，而世界越來越疲乏。阿爾達的毀損導致打破了精靈的不朽設計，精靈慢慢消逝，直到最後他們成為無價值的幽靈。唯有在維林諾，才能推遲時間，繼續生存下去。當精靈三戒還在時，[[愛隆|愛隆]]和[[凱蘭崔爾|凱蘭崔爾]]用精靈三戒的力量延遲時間，保存他們的領土。[[至尊魔戒|至尊魔戒]]毀滅了後，精靈三戒也失去效力，所有的精靈在[[第四紀元|第四紀元]]離開了中土大陸。

== 阿爾達的毀滅與再造 ==

當世界末日，世界被破壞，伊露維塔將帶領[[埃奴|埃奴]]，以及人類和精靈再奏出第二次埃奴的大樂章，重造新世界。

== 圓形的阿爾達 ==

《[[精靈寶鑽|精靈寶鑽]]》中，阿爾達成了一個圓球了後，阿爾達成為整個[[太陽系|太陽系]]的代名詞，地球的名稱換成是''Ambar''或''Imbar''。
