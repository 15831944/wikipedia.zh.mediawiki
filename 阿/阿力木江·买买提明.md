'''阿力木江·买买提明'''{{BD|1959年|10月||}}，[[维吾尔族|维吾尔族]]，[[新疆|新疆]][[且末|且末]]人，[[中华人民共和国|中华人民共和国]]政府官员。

== 生平 ==
1975年9月参加工作。1975年9月到1978年10月，在[[且末县|且末县]]跃进公社接受贫下中农再教育。1978年10月到1982年7月，在[[中央民族学院|中央民族学院]]民语系维吾尔语专业学习。1982年7月，担任新疆[[新疆农业大学|八一农学院]]基础部教师。1987年2月，调任八一农学院教务处学生科副科长。1990年5月加入[[中国共产党|中国共产党]]。1990年6月，调任八一农学院团委副书记。1993年10月，调任八一农学院语言教学部副主任。1994年2月，调任八一农学院院长办公室副主任<ref name=wy/>。

1994年5月，任[[于田县|于田县]]副县长。1996年11月，任中共[[民丰县|民丰县]]委副书记。1999年8月，任中共民丰县委副书记、代县长。1999年10月，任中共民丰县委副书记、县长<ref name=wy/>。

2004年2月，任中共[[新疆维吾尔自治区教育厅|新疆维吾尔自治区教育厅]]党组成员、副厅长。2008年8月，任中共[[伊犁哈萨克自治州|伊犁哈萨克自治州]]党委副书记。2011年1月，任中共[[新疆日报|新疆日报]]社党委副书记、社长、副总编辑。2013年3月，任中共[[新疆维吾尔自治区人民政府|新疆维吾尔自治区人民政府]]党组成员、新疆维吾尔自治区人民政府秘书长（2013年3月30日任<ref>{{Cite web |url =http://news.ifeng.com/gundong/detail_2013_04/01/23763069_0.shtml  |title = 阿力木江·买买提明接替艾拉提·艾山任新疆自治区政府秘书长 |author =  |date = 2013-04-01 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>）、中共[[新疆维吾尔自治区人民政府办公厅|新疆维吾尔自治区人民政府办公厅]]党组书记<ref name=wy>{{cite web|url=http://news.163.com/15/0628/11/AT6N73250001124J.html | language= | title=新疆维吾尔自治区政府秘书长被调查 |publisher=网易 |date=2015-06-28 |accessdate= }}</ref>。

2015年6月下旬，因涉嫌严重违纪，接受组织调查<ref name=wy/>。同年7月24日，新疆维吾尔自治区第十二届人民代表大会常务委员会第十七次会议通过决定，免去其新疆维吾尔自治区人民政府秘书长职务<ref>{{Cite web |url =   http://kb.southcn.com/content/2015-07/25/content_129222608.htm |title = 阿力木江被免去新疆政府秘书长 |author =  |date = 2015-07-25 |publisher = 南方网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.sohu.com/20150725/n417517679.shtml |title = 阿力木江被免去新疆政府秘书长 上月已通报落马 |author =  |date = 2015-07-25 |publisher = 搜狐|language =  |accessdate =  |quote =  }}</ref>。2016年2月，据[[中共新疆维吾尔自治区纪委|中共新疆维吾尔自治区纪委]]消息，经中共新疆维吾尔自治区纪委常委会议研究并报[[中共新疆维吾尔自治区党委|中共新疆维吾尔自治区党委]]常委会议审议，决定给予其开除党籍处分；由[[新疆维吾尔自治区监察厅|新疆维吾尔自治区监察厅]]报新疆维吾尔自治区人民政府批准，给予其行政开除处分；收缴其违纪所得；将其涉嫌违法犯罪问题及线索移送司法机关处理<ref>{{Cite web |url = http://news.ifeng.com/a/20160229/47626950_0.shtml |title = 新疆区政府原秘书长阿力木江·买买提明被双开 结党营私、拉帮结派 |author =  |date =2016-02-29  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。
