'''阿拉伯膠'''（{{Lang-en|'''Gum arabic'''}}）是從[[阿拉伯膠樹|阿拉伯膠樹]]和{{Link-en|塞伊耳相思樹|Acacia seyal}}這兩種[[豆科|豆科]][[含羞草亞科|含羞草亞科]][[金合歡|金合歡]]中取出[[植物汁液|植物汁液]]製成的{{Link-en|天然膠|Natural gum|樹膠}}，阿拉伯膠今日已經成為[[塞內加爾|塞內加爾]]、[[蘇丹共和國|蘇丹共和國]]和[[索馬利亞|索馬利亞]]、[[馬利共和國|馬利共和國]]、[[查德|查德]]、[[埃及|埃及]]等[[薩赫勒|薩赫勒]]至[[北部非洲|北部非洲]]地區的主要經濟來源，不過作為可以取出[[樹脂|樹脂]]的植物則遍及包括[[沙烏地阿拉伯|沙烏地阿拉伯]]等[[阿拉伯半島|阿拉伯半島]]以及[[西亞|西亞]]地區的[[荒漠|荒漠]][[綠洲|綠洲]]。通常為了方便作業會選擇[[非洲|非洲]][[尼羅河|尼羅河]]結束後的2月至5月時開始採集，不過不同棵金合歡[[樹皮|樹皮]]下僅能夠採出重量不一的阿拉伯膠樹脂，而其品質則取決於氣候條件、土壤成分以及栽培方法等。

== 化学 ==

阿拉伯膠本身是一種複雜的[[糖蛋白|糖蛋白]]與[[多糖|多糖]][[混合物|混合物]]，而在過去[[阿拉伯糖|阿拉伯糖]]或者是[[核糖|核糖]]等物質也是自阿拉伯膠分離後加以命名。其結構為透過{{Link-en|分支 (高分子化學)|Branching (polymer chemistry)|分支}}連結[[醣類|醣類]]、[[鉀|鉀]]、[[鎂|鎂]]和[[鈣|鈣]]而成的混合物，而組成的單體結構包括有[[半乳糖|半乳糖]]、阿拉伯糖、[[鼠李糖|鼠李糖]]與[[葡萄糖醛酸|葡萄糖醛酸]]<ref name = "聯合國糧食及農業組織">{{Cite web |url= http://www.fao.org/docrep/X3860E/X3860E20.htm#P5107_110391 |title= GUM ARABIC |author=  |date=  |publisher= [[聯合國糧食及農業組織|聯合國糧食及農業組織]] |language= en |accessdate= 2013年5月7日 |quote=  }}</ref>。根據推測，阿拉伯膠至少為兩個具有不同結構的多醣組成，其中較高分子的醣類混合物往往又會包含一些分子量較小的元素，但其性質主要取決氨基酸的特型。阿拉伯膠平均分子量大約在200[[原子質量單位|原子質量單位]]至300原子質量單位，而其[[CAS號|CAS號]]為9000-01-5<ref name = "聯合國糧食及農業組織"/>。阿拉伯膠其顏色自然的琥珀色至棕色，晶體表面如同有著細微裂縫的玻璃。在經乾燥後的阿拉伯膠十分容易碎裂並[[潮解|潮解]]，在放入水中後會化為黏稠但無味無臭的膠體。儘管其最高可以調配到重量百分濃度55%高的溶液，但是即使在30%至40％的調配濃度上阿拉伯膠仍僅有低黏性，而且本身並不溶於[[乙醇|乙醇]]、[[乙醚|乙醚]]與[[油|油]]中。

== 用途 ==

阿拉伯膠主要作為可以食用的[[穩定劑|穩定劑]]、[[乳化劑|乳化劑]]、{{Link-en|增黏劑|Thickening agent}}和[[賦形劑|賦形劑]]並用於[[食品產業|食品產業]]上，而[[歐洲聯盟|歐洲聯盟]]對於食品添加劑的[[E編碼|E編碼]]為E414。主要原因在於阿拉伯膠能夠降低液體的[[表面張力係數|表面張力係數]]使得[[軟性飲料|軟性飲料]]得以飽含[[二氧化碳|二氧化碳]]，這也是[[可樂加曼陀珠噴發現象|可樂加曼陀珠噴發現象]]發生的物理因素。阿拉伯膠在傳統的[[平版印刷|平版印刷]]、油漆生產、顏料混和、膠水製作、化妝品調配上亦是重要原料，也被用[[墨水|墨水]]黏度控制、較便宜的紡織材料在等各種工業應用上，不過在工業應用上有許多替代產品也可以使用。此外阿拉伯膠也被用於[[醫療|醫療]]和化妝用途上，並且可以治療[[痢疾|痢疾]]與[[腸胃炎|腸胃炎]]等疾病。雖然阿拉伯膠現在主要在整個非洲薩赫勒地區進行生產，不過仍然是以中東地區的社會較為常用，例如阿拉伯人會使用將天然樹膠冷凍後加糖和調味製成如[[冰淇淋|冰淇淋]]般的甜點，但食用過多容易引起{{Link-en|腹脹|Tympanites}}。
