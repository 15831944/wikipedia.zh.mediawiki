<!-- 定義及症狀 -->
'''阿米巴症'''（{{lang-en|Amoebiasis}}或{{lang-en|amoebic dysentery}}），係指任何[[內阿米巴屬|內阿米巴屬]][[變形體|變形體]]所引起的感染<ref name=Mansons2013/>，阿米巴症患者既可能毫無症狀，也可能症狀輕微，也可能有嚴重症狀<ref name=Mansons2013/>，常見的[[症狀|症狀]]為[[溶组织内阿米巴|溶组织内阿米巴]]（''Entamoeba histolytica''）引起的感染<ref name=Mansons2013/>，可能的症狀有[[腹痛|腹痛]]、[[腹瀉|腹瀉]]、[[痢疾|有膿血樣排泄物]]（[[阿米巴痢疾|阿米巴痢疾]]）<ref name=Mansons2013/>。相關併發症有{{le|阿米巴結腸炎|amoebic colitis|結腸發炎及潰瘍}}而[[壞死|壞死]]或[[腸胃穿孔|腸胃穿孔]]，也有可能會導致[[腹膜炎|腹膜炎]]<ref name=Mansons2013/>，患者可能因失血過多而[[貧血|貧血]]<ref name="Mansons2013"/>。

<!-- 病因、機轉及診斷 -->
內阿米巴屬生物或細胞的囊體（cyst）可以在土壤中存活長達一個月，也能在指甲內存活 45 分鐘<ref name=Mansons2013/>，入侵腸道粘膜有可能導致血瀉（即為瘧疾症狀）<ref name=Mansons2013/>，當寄生蟲進入血液循環，就可以散佈到全身。在病原體進入血行循環後，最常見的最終位置為[[肝|肝]]，進而導致{{link-en|阿米巴肝膿瘍|Amoebic liver abscess}}<ref name=Mansons2013/>。罹患肝膿瘍時初期可能不會腹瀉<ref name=Mansons2013/>。典型的診斷會用[[顯微鏡|顯微鏡]]作{{link-en|糞便類檢驗|Stool test}}，但需要排除是否為感染或獨立出特定類型時，採信度可能會不足<ref name=Mansons2013/>。症狀嚴重的患者有可能會罹患{{link-en|白血球增多症|Leukocytosis}}<ref name=Mansons2013/>，最準確的測試為進行特定的[[血清學|血清檢驗]]，但即使接受治療後可能依然為陽性<ref name="Mansons2013"/>。細菌性[[結腸炎|結腸炎]]也可能導致阿米巴症的類似症狀<ref name=Mansons2013/>。

<!-- 預防及治療 -->
預防阿米巴症的方法為改善[[衛生設施|衛生設施]]，包含讓飲食用水免受糞便污染<ref name=Mansons2013/>，阿米巴症沒有疫苗<ref name=Mansons2013/>，有兩種方式可以治療，須依受感染位置而定<ref name=Mansons2013/>：治療感染阿米巴症的人體組織時，如不是用[[甲硝唑|甲硝唑]]、抗感染藥{{link-en|替尼達諾|Tinidazole)}}、{{link-en|硝唑尼特|Nitazoxanide}}或抗寄生蟲藥{{link-en|去氫吐根素|dehydroemetine}}，就是使用[[氯喹|氯喹]]，治療腸道感染則是使用{{link-en|阿米巴殺腸腔藥物|diloxanide furoate}}或抗寄生蟲病藥{{link-en|二碘羥喹啉|Diiodohydroxyquinoline}}<ref name=Mansons2013/>。對抗此病任一階段的有效療法可能為並用多種藥物<ref name=Mansons2013/>。沒有症狀顯現的感染不需要治療，但感染者仍可將寄生蟲傳播給他人，因此仍建議接受藥物治療<ref name=Mansons2013/>。治療其他內阿米巴屬生物或細胞感染時，不須排除是否為具致病性的''E. histolytica''<ref name=Mansons2013/>。

<!--流行病學及歷史 -->
阿米巴症患者散見世界各地<ref>{{Cite book|title = Lecture Notes: Tropical Medicine|chapter-url = |publisher = John Wiley & Sons|date = 2014-04-17|isbn = 9781118734568|first = Nick|last = Beeching|first2 = Geoff|last2 = Gill|pages = 177–182|chapter = 19}}</ref>，但多數病例發生於[[開發中國家|開發中國家]]<ref name=Shi2018>{{cite journal |last1=Shirley |first1=DT |last2=Farr |first2=L |last3=Watanabe |first3=K |last4=Moonah |first4=S |title=A Review of the Global Burden, New Diagnostics, and Current Therapeutics for Amebiasis. |journal=Open Forum Infectious Diseases |date=July 2018 |volume=5 |issue=7 |pages=ofy161 |doi=10.1093/ofid/ofy161 |pmid=30046644|pmc=6055529 }}</ref>。一年中約有 4.8 億人感染阿米巴症，其中有 4~11 萬人死亡<ref name=Mansons2013/>。據信多數感染的都是不具致病性的''[[内阿米巴属|E. dispar]]''<ref name=Mansons2013/>，此常見於特定地區，而有症狀顯現的病例可能較前述為少<ref name=Mansons2013/>。。阿米巴症頭號病例最早可追溯至 1875 年，1891 年才有詳盡的記錄，{{link-en|阿米巴瘧疾|amoebic dysentery}} 和{{link-en|阿米巴肝膿瘍|Amoebic liver abscess}} 兩詞因而問世<ref name=Mansons2013/>。1913 年，於菲律賓發現的近一步證據顯示，吞下''E. histolytica''囊體的自願者後來發病了<ref name=Mansons2013>{{Cite book|title = Manson's Tropical Diseases|publisher = Elsevier Health Sciences|date = 2013-10-26|isbn = 9780702053061|first = Jeremy|last = Farrar|first2 = Peter|last2 = Hotez|first3 = Thomas|last3 = Junghanss|first4 = Gagandeep|last4 = Kang|first5 = David|last5 = Lalloo|first6 = Nicholas J.|last6 = White|pages = 664–671}}</ref>。
