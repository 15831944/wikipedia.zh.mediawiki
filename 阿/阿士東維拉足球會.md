'''阿士東維拉足球會'''（{{lang-en|'''Aston Villa Football Club'''}}，{{pron-en|ˈæstən ˈvɪlə}}）簡稱'''維拉'''（{{lang-en|'''Villa'''}}），是一家位於[[英格蘭|英格蘭]]中部[[伯明翰|伯明翰]]的[[足球|足球]]俱樂部，於1874年創立<ref>http://sports.now.com/home/football/BPLTeam?id=002</ref>，現時於[[英格蘭足球冠軍聯賽|英格蘭足球冠軍聯賽]]競逐，是1888年甲組聯賽（英格蘭第一個足球聯賽）及1992年[[英格蘭足球超級聯賽|英格蘭足球超級聯賽]]首屆賽事球隊。其主場位於[[伯明翰|伯明翰市]]的[[維拉公園球場|維拉公園球場]]。

== 歷史 ==
阿士東維拉是於1874年由[[伯明翰|伯明翰]]地區教堂成員組成。成立短短幾年就在[[英格蘭|英格蘭]]中部打出名堂。1887年阿士東維拉贏得其首個[[英格蘭足總盃|足總盃]]，後來1888年英格蘭頂級聯賽成立，維拉也成為足球聯盟創始成員。在未踏入二十世紀時，阿士東維拉已在頂級聯賽威風八面，曾贏得過六個頂級聯賽冠軍。而足總盃方面則到1920年仍能贏得。

阿士東維拉在1920年代以後開始陷入谷底，曾於1936年降級至次級聯賽。其時維拉更創下一個淒涼的紀錄：42場比賽共失了110球。[[二次大戰|二次大戰]]後維拉仍沉浮不定，最惡劣時更曾降至第三級聯賽，直到1975年維拉再次由當時次級聯賽英乙升上頂級聯賽英甲，才再站穩在頂級聯賽的地位，1981年更再一次贏得頂級聯賽冠軍。1982年踏足歐洲联赛，還贏得[[歐洲冠軍球會盃|歐洲冠軍球會盃]]。

維拉在1987年降落英乙作賽，翌年回升英甲，自此一直到2016年都在頂級聯賽作賽，更成為[[英超|英超]]創始會員之一。英超成立初期，維拉是英超除曼联等四個爭標分子外，實力較強的球隊之一，在上世紀九十年代，維拉一直都能打入前列甚至榜首，但每次都在下半季後勁不繼，例如1998-99年[[曼聯|曼聯]]成為三冠王時，上半季榜首曾一度是阿士東維拉。截至2015-16年賽季，維拉是英超成立以來從未降班的七支英超球隊其中之一，但維拉表現非常飄忽。首屆英超奪得亞軍，不過之後有時排名前列，有時候卻為護級而戰，其中1994-95年賽季是英超最後一次22支球隊參賽，因此需要四支球隊降班，阿士東維拉更以3分之差差點成爲降班的四支球隊之一。

由2010年開始，不少在維拉成名的主力球員，如[[占士·米拿|占士·米拿]]、[[艾殊利·楊格|艾殊利·楊格]]等轉投他隊，維拉的聯賽成績之後開始下滑，在2011年起連續五季都須在季尾為護級而戰。

維拉曾在上世紀九十年代兩奪聯賽盃冠軍。足總盃方面，維拉在2000年及2015年打入決賽，分別不敵[[車路士|車路士]]及[[阿仙奴|阿仙奴]]得到亞軍。

維拉在[[2015年至2016年英格蘭足球超級聯賽|2015-16年球季]]，自第一場聯賽取得勝利後連續18場聯賽不勝，完成首循環19場聯賽後僅得8分排榜尾，為維拉英超開季最差成績，護級形勢岌岌可危。維拉在次循環聯賽沒有起色，完成33輪聯賽只得16分，至2016年4月16日第34輪聯賽作客0-1不敵[[曼徹斯特聯足球俱樂部|曼聯]]後，提早4輪聯賽宣佈護級失敗。最後維拉整季聯賽只取得17分，成為英超史上一季內第三最低得分的球隊，終止俱樂部從英超成立以來沒有降級的生涯。

2016-17年度球季是維拉首次在英冠作賽，雖然球會在2016年10月任命去季帶領[[赫爾城足球俱樂部|侯城]]升上英超的[[史蒂夫·布鲁斯|布魯士]]為新領隊，但維拉在此季只取得英冠第13名，來季繼續在英冠作賽。

2017-18年度球季，阿士東維拉取得英冠第四名，可參加升班降加賽。維拉在準決賽淘汰[[米杜士堡足球會|米杜士堡]]晉級決賽，但決賽以0-1不敵[[富咸足球會|富咸]]，來季繼續在英冠作賽。

== 球會近況 ==

2016年6月，中國商人夏建統以旗下聯合睿康集團(Recon Group)名義以6,000萬英鎊全面收購100%阿斯頓維拉足球俱樂部股權，並成為俱樂部新任主席。2018年7月埃及富商[[:en:Nassef_Sawiris|薩維里斯]]以及美國富商[[:en:Wes_Edens|伊登斯]]擁有的財團注資阿斯頓維拉, 薩維里斯取代夏建統成為俱樂部的新任主席<ref>{{cite web |language=en |url=https://www.avfc.co.uk/News/2018/07/20/club-statement-investment-in-aston-villa |title=Investment in Aston Villa|publisher=Aston Villa Football CLub |accessdate=2018-10-06}}</ref>。

== 主場球場 ==
[[File:Villaparkfromtopofholte.jpg|thumb]]
{{main|維拉公園球場}}
'''維拉公園球場'''（'''Villa Park'''）是位於[[英格蘭|英格蘭]][[伯明翰|伯明翰]][[阿斯顿|阿斯顿]]（Aston）的[[英式足球|足球]][[體育場|體育場]]，自1897年開始已經是阿士東維拉的主場球場，亦是[[歐洲足協|歐洲足協]]的指定比賽球場（UEFA Elite stadium），並曾作為[[英格蘭足球代表隊|英格蘭國家隊]]主場進行多達16場國際賽，是首個足球場於三個不同[[世紀|世紀]]舉行國際賽，維拉公園亦是歷來舉辦最多[[英格蘭足總盃|英格蘭足總盃]]準決賽的球場，曾有55場比賽在這裡進行。

阿士東維拉首先於1874年到1876年期間使用阿斯顿公园球场（Aston Park）作為主場，其後搬遷到贝利巴亚（Perry Barr）直到1897年，同年正式遷移到維拉公園，於4月17日剛贏得[[雙料冠軍|雙料冠軍]]的阿士東維拉與[[布莱克本流浪者足球俱乐部|布力般流浪]]進行[[友誼賽|友誼賽]]作為揭幕戰。阿士東維拉於1911年以總值11,750[[英鎊|英鎊]]購入球場及其關連設施，並由[[阿奇巴尔·雷奇|阿奇巴尔·雷奇]]（Archibald Leitch）設計進行重建，當聖三一路看台（Trinity Road Stand）於1922年建成時，是[[英國|英國]]當時最堂皇的球場看台。1958年設置汎光燈。[[1966年世界盃足球賽|1966年世界盃]]作為分組賽的比賽場地，舉行三場比賽，包括[[阿根廷國家足球隊|阿根廷]]2-1勝[[西班牙國家足球隊|西班牙]]、阿根廷0-0和[[西德國家足球隊|西德]]及西德2-1勝西班牙。1994/95年球季當哈拉尔德看台（Holte End）重建完成後維拉公園成為全坐席球場。1996年6月加裝草坪地底加熱系統。同年的[[1996年欧洲足球锦标赛|歐洲國家盃]]維拉公園合共上演四場比賽，包括三場分組賽[[荷蘭國家足球隊|荷蘭]]0-0和[[蘇格蘭足球代表隊|蘇格蘭]]、[[瑞士國家足球隊|瑞士]]0-2負荷蘭及蘇格蘭1-0勝瑞士，和一場半準決賽[[捷克國家足球隊|捷克]]1-0淘汰[[葡萄牙國家足球隊|葡萄牙]]。1999年最後一屆[[歐洲盃賽冠軍盃|歐洲盃賽冠軍盃]]決賽亦在維拉公園舉行，結果[[拉素體育會|-{zh-hans:拉齐奥; zh-hk:拉素;}-]]以2-1擊敗[[皇家馬略卡|-{zh-hans:皇家马洛卡; zh-hk:馬略卡}-]]捧盃。2001年拆卸舊聖三一路看台，於2001年重建完工，將維拉公園的容量提升到42,640名觀眾，於11月由[[查爾斯_(威爾斯親王)|威爾斯親王查爾斯王子]]主持揭幕。

維拉公園現時設有四個看台：哈拉尔德看台、聖三一路看台、北看台（North Stand）及道·艾利斯看台（Doug Ellis Stand）。球會正計畫擴建北看台<small><ref name="fact">{{cite web |language=en |url=http://dspace.dial.pipex.com/town/park/yfh45/aston.htm |title=Aston Villa |publisher=The internet football grounds |accessdate=2009-08-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20090923234008/http://dspace.dial.pipex.com/town/park/yfh45/aston.htm |archivedate=2009-09-23 }}</ref></small>，將需要「填補」北看台其中一個角落，完工後，容量將進一步增加到大約50,000人<small><ref name=brumgovplan>{{cite web |language=en |url=http://www.birmingham.gov.uk/GenerateContent?CONTENT_ITEM_ID=19729&CONTENT_ITEM_TYPE=0&MENU_ID=12228&EXPAND=1636 |title=Aston Local Plan |publisher=Birmingham City Council |accessdate=2009-08-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20080510105931/http://www.birmingham.gov.uk/GenerateContent?CONTENT_ITEM_ID=19729&CONTENT_ITEM_TYPE=0&MENU_ID=12228&EXPAND=1636 |archivedate=2008-05-10 }}</ref></small>。
