'''阿兰·梅里埃'''（{{lang-fr|'''Alain Mérieux'''}}，{{bd|1938年|7月10日|}}），法国商人，{{le|梅里埃研究所|Institut Mérieux}}主席。畢業於[[里昂大學|里昂大學]]藥劑系。

== 生平 ==
阿兰·梅里埃是是夏尔·梅里埃（1907-2001）的儿子和马塞尔·梅里埃（1870-1937）的孙子。<ref name="Forbes">{{Cite web|url=https://www.forbes.com/profile/alain-merieux/|title=Alain Mérieux|accessdate=1 January 2015|work=Forbes}}</ref>

== 职业生涯 ==
根据[[福布斯|福布斯]]榜单，截至2015年1月，梅里埃的净资产为35亿美元。<ref name="Forbes">{{Cite web|url=https://www.forbes.com/profile/alain-merieux/|title=Alain Mérieux|accessdate=1 January 2015|work=Forbes}}</ref>

== 个人生活 ==
他居住在[[里昂|里昂]]，已婚，育有三个孩子。<ref name="Forbes">{{Cite web|url=https://www.forbes.com/profile/alain-merieux/|title=Alain Mérieux|accessdate=1 January 2015|work=Forbes}}</ref>
