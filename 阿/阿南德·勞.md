'''阿南德‧勞'''({{lang-en|K. Ananda Rau}}, {{bd|1893年|9月21日|1966年|1月22日}})是一位和傳奇數學家[[斯里尼瓦瑟·拉马努金|斯里尼瓦瑟·拉马努金]]同輩的杰出印度[[数学家|数学家]]。 虽然劳比拉瑪努金小六歲，但他的数学生涯依循傳統道路，早在拉瑪努金成名之前，勞就決定投身於數學研究之中。

== 生平 ==
[[File:K_Ananda_Rau_seated_with_Ramanujan.jpg|左]]
阿南德‧勞於1893年9月21日出生於 [[金奈|马德拉斯]] 。先後就學於[[馬德拉斯|馬德拉斯]]Triplicane的印度教學校與[[马德拉斯大学|馬德拉斯大學]]的管轄區學院。以優秀成績自馬德拉斯大學畢業後，勞於1914年乘船赴英求學，時間點僅比[[斯里尼瓦瑟·拉马努金|拉瑪努金]]晚幾個月。通過[[剑桥大学国王学院|剑桥大学国王学院]]的數學榮譽學位考試後，在1916年，與拉瑪努金一樣，受業於[[戈弗雷·哈罗德·哈代|戈弗雷·哈羅德·哈代]]門下，導師的啟發激起了勞的活躍數學研究。在[[剑桥大学|剑桥]]求學過程中，勞和拉瑪努金成為好朋友。 此外，拉瑪努金的"最忠实的朋友"R. [[Ramachandra_Rao|Ramachandra Rao]](擔任區域稅吏([[District_Collector|District Collector]]))也是勞的親戚。Ramachandra Rao曾擔當培養[[斯里尼瓦瑟·拉马努金|拉瑪努金]]成才的任務，同時提供財務援助，照料日常生活以及協助他在[[馬德拉斯|馬德拉斯]]港口信託局(Madras Port Trust)取得夥計工作。虽然勞與[[斯里尼瓦瑟·拉马努金|拉瑪努金]]是在英國才首次相遇，但勞早已透過與Rao的關係得知有[[斯里尼瓦瑟·拉马努金|拉瑪努金]]這號人物。

勞在1919年返回印度，年僅26岁就被任命為管轄區學院的數學教授。而後勞還曾擔任了學院院長，在1948年退休。勞的一生充滿坎坷，年輕的妻子亡於1928年、女兒亡於1940年，而勞自己又疾病纏身，晚年還瞎了一隻眼。勞卒於1966年1月22日，享壽72歲。

== 学术工作 ==
勞求學時，在[[戈弗雷·哈罗德·哈代|哈代]]指導下寫成一篇論文，於1917年憑藉此文取得夢寐以求的[[史密斯獎|史密斯獎]]，而後獲選為[[剑桥大学国王学院|國王學院]]的院士。與哈代一塊，勞在級數可和性方面以及[[级数|無窮級數]]的收斂性方面做了大量研究。在哈代的一本書中所收錄的一條定理由勞所冠名。勞發表了數篇論文，主要集中於三個領域：級數可和性、[[複分析|單複變數函數論]]以及偶數的平方和表示問題。勞返印後的頭幾篇論文走回研究[[黎曼ζ函數|黎曼Zeta函數]]，其中由勞所提出的技巧在數論的其他問題上有廣泛的應用。

== 教学影响 ==
多位著名數學家如[[苏布拉马尼扬·钱德拉塞卡|S钱德拉塞卡]], K.S.Chandrasekharan, S Minakshisundaram, C T Rajagopal, C S Venkitaraman 和 M V Subbarao等皆曾受教於勞門下。学生們认为他是一个鼓舞人心的老师。勞對學生强调，研究数学是一个需要全心沉浸與奉献的活動，僅是充當觀眾是不會有任何收穫的。拉瑪努金後的印度數學新篇章由勞及其學生們共同締造，他們共同構建了數論的印度學派。
