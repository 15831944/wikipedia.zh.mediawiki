'''阿纳托利·伊万诺维奇·卢基扬诺夫'''（{{Lang-ru|Анато́лий Ива́нович Лукья́нов}}，{{Lang-en|Anatoly Ivanovich Lukyanov}}，{{bd|1930年|5月7日|2019年|1月9日|catIdx=Лукья́нов}}）是[[前苏联|前苏联]][[苏联共产党|党]]和国家领导人，[[俄罗斯联邦|俄罗斯联邦]][[政治人物|政治人物]]。<ref>{{cite web |title=苏联最后一任最高苏维埃主席卢基扬诺夫逝世 |url=http://www.xinhuanet.com/world/2019-01/10/c_1123973898.htm |website=[[新华网|新华网]] |publisher=[[新华社|新华社]] |accessdate=2019-01-10}}</ref><ref name="guancha">{{cite web |author1=史雨轩 |title=末代苏联最高苏维埃主席去世 曾策划819事件 |url=https://www.guancha.cn/internation/2019_01_10_486354.shtml |publisher=观察者网 |accessdate=2019-01-10}}</ref> 在前苏联时期，阿纳托利·卢基扬诺夫历任[[苏联共产党中央委员会书记处|苏联共产党中央委员会书记处]]书记（1987年－1988年）、[[苏联共产党中央政治局|苏联共产党中央政治局]]候补委员（1988年－1990年）和末任{{Link-ru|苏联最高苏维埃主席|Председатель Верховного Совета СССР}}（1990年3月－1991年9月）。他是[[苏联|苏联]]历史上首任也是末任[[苏联总统|总统]][[米哈伊尔·谢尔盖耶维奇·戈尔巴乔夫|米哈伊尔·谢尔盖耶维奇·戈尔巴乔夫]]的首席副手，但后来成为戈尔巴乔夫的政治对手。<ref name="guancha" />1991年8月至1992年12月期间，阿纳托利·卢基扬诺夫因与[[国家紧急状态委员会_(苏联)|国家紧急状态委员会]]成员一同参与了[[八一九政变|八一九政变]]而被拘捕，他被指控篡夺政权及滥用职权<ref>{{cite web |title=Следствие/приговоры |url=https://www.kommersant.ru/doc/62250 |publisher=ОБРАТНАЯ СВЯЗЬ |accessdate=2019-01-10}}</ref>，但之后他也和其他涉案人员一样获得了特赦。被[[叶利钦|叶利钦]]称为“政变军师”。在[[俄罗斯联邦|俄罗斯联邦]]时期阿纳托利·卢基扬诺夫以[[俄罗斯联邦共产党|俄罗斯联邦共产党]]党员的身份在1993年至2003年期间获选成为[[国家杜马|国家杜马]]议员，同时他也在1994年至2000年期间当选为{{Link-ru|俄罗斯联邦共产党中央委员会|Центральный комитет КПРФ}}主席团成员。除此之外，阿纳托利·卢基扬诺夫还是一名[[诗人|诗人]]。<ref>{{cite web |title=Lukyanov, Gorbachev's college buddy, may be coup mastermind |url=https://www.upi.com/Archives/1991/08/22/Lukyanov-Gorbachevs-college-buddy-may-be-coup-mastermind/6982503123430/ |publisher=[[合众国际社|合众国际社]] |accessdate=2019-01-10}}</ref><ref>{{cite web |author1=Кашин Олег |title=Анатолий Иванович |url=http://rulife.ru/mode/article/939/ |publisher=Русская жизнь |accessdate=2019-01-10}}</ref>

阿纳托利·伊万诺维奇·卢基扬诺夫在1979年获得[[法学|法学]][[全博士学位|全博士学位]]，并在2004年进入他的母校[[莫斯科国立大学|莫斯科国立大学]]任教。2012年，他获得“{{Link-ru|俄罗斯荣誉律师|Заслуженный юрист Российской Федерации}}”表彰。

== 个人简历 ==
1930年，生于斯摩棱斯克。

1953年，毕业于国立莫斯科大学。法学博士。

1985年，任{{le|苏联共产党中央委员会办公厅|General Department of the Communist Party of the Soviet Union}}主任。

1987年，任[[苏联共产党中央委员会书记处|苏联共产党中央委员会书记处]]书记。

1988年，任苏联最高苏维埃主席团第一副主席。

1989年，任苏联最高苏维埃第一副主席。

1990年，任苏联最高苏维埃主席。

1991年，在[[八一九事件|八一九事件]]中因支持[[国家紧急状态委员会|国家紧急状态委员会]]而被逮捕。

1994年，根据[[俄罗斯联邦|俄罗斯联邦]][[国家杜马|国家杜马]]决议获大赦。

1993年到2003年，卢基扬诺夫担任国家杜马议员。 

1994年到2000年，为[[俄罗斯联邦共产党|俄共]]中央主席团成员。

== 人物事件 ==

据苏联史专家王国杰教授著作《“[[八·一九事件|八·一九事件]]”透视与剖析》介绍，1991年8月19日-21日，[[苏联|苏联]]发生了一次政变，中央政府的一些高官为了挽救苏联，软禁[[苏联总统|苏联总统]][[戈尔巴乔夫|戈尔巴乔夫]]，企图废除其职位。政变领导人[[苏联国防部|国防部长]][[德米特里·季莫费耶维奇·亚佐夫|亚佐夫]]元帅、[[克格勃|克格勃]]领导人[[弗拉基米尔·亚历山德罗维奇·克留奇科夫|克留奇科夫]]等[[苏联共产党|苏共]]强硬保守派组成了[[国家紧急状态委员会|国家紧急状态委员会]]，卢基扬诺夫本人虽然没参加紧急委员会，但是他在回忆录中写到，他认为引入紧急状态是绝对正当（абсолютно оправданным）。

1991年8月19日清晨6时，紧急状态委员会占领了电视台后首先发表的就是卢基扬诺夫关于国内局势的声明，19日下午17时，卢基扬诺夫作为[[蘇聯最高蘇維埃|蘇聯最高蘇維埃]]主席宣布在8月26日召开苏联最高苏维埃非常会议，会议将批准在苏联个别地区实行紧急状态的决定，此举被认为是支持紧急委员会，尽管卢基扬诺夫不是紧委会成员，但[[白宫|白宫]]决策者们认为他是紧急委员会的军师，是政变的主要策划者。

然而因为一系列原因，21日戈尔巴乔夫宣布完全控制局势，紧急委员会政变失败，卢基扬诺夫也因此被捕，在1991年8月22日俄议会会议上，俄总理西拉耶夫讲：“卢基扬诺夫是政变中的主要思想家，我们不能信任他”。

1991年8月26日，苏联最高苏维埃暂停卢基扬诺夫主席职务，29日，卢基扬诺夫被[[俄罗斯联邦|俄罗斯联邦]]检察机关关押，直到1994年2月根据[[国家杜马|国家杜马]]决议获大赦。

他被捕后从没认错，就在被审讯期间他作为[[俄罗斯联邦共产党|俄共]]推举的候选人当选为国家杜马代表。1994年初叶利钦还写信给卢基扬诺夫表示愿意合作，并建议卢基扬诺夫“分担起俄罗斯命运的责任”。

== 家庭情况 ==
阿纳托利·伊万诺维奇·卢基扬诺夫的女儿艾琳娜是俄罗斯高等经济学院的教授。
