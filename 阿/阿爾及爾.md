'''阿爾及爾'''（{{lang-ar|'''الجزائر'''}}；{{lang-fr|'''Alger'''}}；{{lang-en|'''Algiers'''}}）為[[阿爾及利亞|阿爾及利亞]]首都及海港。現有人口三百萬，大約佔阿爾及利亞全國人口的十分之一。位於[[法國|法國]][[馬賽|馬賽]]西南805公里，北濱[[地中海|地中海]]。這裡除了紡織和[[礦業|礦業]]之外，農業也很發達。

== 歷史 ==
約公元980年時，[[柏柏爾人|柏柏爾人]]在[[羅馬人|羅馬人]]原住地建下這座古城。1518年[[奥鲁奇·雷斯|奥鲁奇·雷斯]]及[[巴巴罗萨·海雷丁帕夏|海雷丁]]兩兄弟佔領並在此建立[[土耳其|土耳其]]統治，迅速成為[[海盜|海盜]]活動的一個據點。1830年為法國人攻佔。[[第二次世界大戰|第二次世界大戰]]中為北非盟軍總部和[[戴高樂|戴高樂]]臨時政府所在地。市內有1909年建立的[[阿爾及爾大學|阿爾及爾大學]]和1974年成立的科學技術大學。市內交通包括有[[阿尔及尔胡阿里·布迈丁机场|阿尔及尔胡阿里·布迈丁机场]]及[[鐵路|鐵路]]。該地並設有貿易、商業和行政部門。該地也盛產[[葡萄酒|葡萄酒]]。

== 氣候 ==
阿爾及爾屬典型的[[地中海型氣候|地中海型氣候]]，[[降雨量|降雨量]]為690~760[[公釐|公釐]]，大多集中於11月至3月，平均最高氣溫為攝氏15至29度。

== 文學相關 ==
阿爾及爾是[[存在主義|存在主義]]大師[[卡繆|卡繆]]的著名小說「[[異鄉人|異鄉人]]」中，大部分情節的發生地點。

== 名勝古蹟 ==
[[File:Mosquée_Ketchaoua.jpg|thumb]]
此地因曾遭[[腓尼基人|腓尼基人]]、羅馬人、[[汪達爾人|汪達爾人]]、[[阿拉伯人|阿拉伯人]]、[[土耳其人|土耳其人]]及[[法國人|法國人]]入侵，所以他們建下不少建築物，更成為成為今日古蹟名勝。如[[西迪·阿布德拉赫曼清真寺|西迪·阿布德拉赫曼清真寺]]、[[西迪·穆罕默德清真寺|西迪·穆罕默德清真寺]]、[[賈馬·傑伊德清真寺|賈馬·傑伊德清真寺]]（16世紀）、教堂、國家博物館、[[巴爾杜博物館|巴爾杜博物館]]、古文物博物館、國家美術館。

== 經濟 ==
阿爾及爾是一個重要的經濟，商業和金融中心，有一個獨有的證券交易所，每年處理達六十億[[歐元|歐元]]。阿爾及爾也是北非最重要的港口，這個城市的生活費用是北非的城市中最高的，也是全球最高生活費用的五十個城市之內。

埃米爾行政部門主席Mohamed Ben Ali El Abbar，向阿爾及利亞總統[[阿卜杜勒-阿齐兹·布特弗利卡|阿卜杜勒-阿齐兹·布特弗利卡]]提出五個方案。這個方案會改變阿爾及爾及周邊的環境，阿爾及爾將會具有零售，休閒設備的城市：
#集中重組和發展基礎設施。打造位於市中心超現代的鐵路車站，打算容納超過80,000名乘客，期望成為阿爾及爾心臟地帶，周圍將設有商業辦公室及樓宇及酒店供旅客過境。一個購物中心和三幢高層辦公樓與頂尖的商業區域將伴隨項目。
#振興海邊經濟。開發44公里（27英里），包括前海碼頭，渠道，豪華酒店，寫字樓，公寓，豪華商店和休閒設施。一個月牙形半島將設立在公海上。該項目的海灣阿爾及爾也將包括6個小島，其中4輪的形式，由橋樑和碼頭相互連接，將包括旅遊和住宅大樓。
#調整阿爾及爾的一個地區，打造合格的“舒適城市”。薩爾瓦多Abbar表示向記者說，綜合設施將會提供旅遊合作和福利或旅遊和休閒的用途。該大樓將包括一所大學，一個研究中心和一個醫療中心。它還應包括醫院大樓，保健，中心，酒店區，城市中心和溫泉的別墅和公寓。該大學將包括學校和一所學校的醫療護理課室將可容納500名學生。大學校園將有可能看到建立廣泛的範圍內的建築物和住宅和研究實驗室。
#技術性的運用在一個校園位於[[西迪阿卜杜拉|西迪阿卜杜拉]]25公里（16英里）阿爾及爾的東南部。這90公頃（222英畝）的地方將包括商場，住宅區，高標準的公寓和環繞別墅和酒店的高爾夫球場。另外兩個住宅區，包括1800伙公寓和40個高標準的別墅，建立在周圍的山上。
#興建旅遊的綜合設施[[Colonel_Abbès|Colonel Abbès]]，位於阿爾及爾西部。這將包括一些零售區，會議場所和居住區組成，面向海景。
