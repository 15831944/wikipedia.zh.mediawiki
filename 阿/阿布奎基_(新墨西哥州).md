'''阿尔伯克基'''（[[英语|英语]]：'''{{lang|en|Albuquerque}}'''；[[國際音標|發音]]：{{IPA|[ˈæl.bə.kɚ.ki]、[[西班牙語|西班牙語]]：[al.βu.ˈkeɾ.ke]}}，[[納瓦霍語|納瓦霍語]]：Beeʼeldííl Dahsinil）是[[美國|美國]][[新墨西哥州|新墨西哥州]]人口最多的城市，位於新墨西哥州中部地區、橫跨[[格蘭德河|格蘭德河]]的兩岸，也是[[伯納利歐縣|伯納利歐縣]]的縣府所在地。根據[[2000年美國人口普查|2000年人口普查]]，阿布奎基共有人口44萬8607人，市政府在2005年的估計為49萬4236人，整個[[都會區|都會區]]的人口則為79萬7940人。阿布奎基於2005年時名列全美國第33大城市，阿布奎基都會區則是第62大都會區。在2011年的人口上升約為55萬2804人，整個[[都會區|都會區]]的人口則為90萬1700人。阿布奎基於2012年時名列全美國第32大城市，阿布奎基都會區則是第53大都會區。 阿布奎基都會區包含了{{link-en|里奧蘭河市 (新墨西哥州)|Rio_Rancho,_New_Mexico|里奧蘭河市}}，是全美國成長最快的城市之一，該市包括許多計畫社區，預期在未來將為整個地區繼續吸引更多居民和商業活動。

[[新墨西哥大學|新墨西哥大學]]、{{link-en|科特蘭空軍基地|Kirtland Air Force Base}}（{{lang|en|Kirtland Air Force Base}}）、[[桑迪亚国家实验室|桑迪亚国家实验室]]、與{{link-en|石化森林國家紀念物|Petroglyph National Monument}}（{{lang|en|Petroglyph National Monument}}）都位於阿布奎基。城市東部沿著{{link-en|桑迪亚山脉|Sandia Mountains}}（{{lang|en|Sandia Mountains}}）而行，[[格蘭德河|格蘭德河]]從北到南貫穿整個城市。

== 地理与气候 ==
阿布奎基属于温带[[半干旱气候|半干旱气候]]，四季分明，昼夜气温变化大，降水少，日照高。冬季冷凉，日最高气温低于{{convert|0|C}}的平均日数为2.8天，日最低气温低于{{convert|-10|C}}的平均日数为3.8天；夏季相对炎热，日最高气温超过{{convert|30|C}}的日数年均有94天，{{convert|35|C}}以上的有21天。<ref name = "NWS Albuquerque, NM"/>最冷月（12月）均温{{convert|36.3|F|1|disp=flip}}，极端最低气温{{convert|−17|F|0|disp=flip}}（1971年1月7日）。<ref name = "NWS Albuquerque, NM"/>最热月（7月）均温{{convert|78.3|F|1|disp=flip}}，极端最高气温{{convert|107|F|0|disp=flip}}（1994年6月26日）。<ref name = "NWS Albuquerque, NM"/>无霜期平均为205天（4月10日至10月31日）；可测量降雪平均期为11月22日至3月21日。<ref name = "NWS Albuquerque, NM"/>年均降水量约{{convert|9.45|in|sigfig=3|disp=flip}}，年极端最少降水量为{{convert|3.29|in|sigfig=3|disp=flip}}（1917年），最多为{{convert|15.88|in|sigfig=3|disp=flip}}（1941年）。<ref name = "NWS Albuquerque, NM"/> 年均降雪量为{{convert|9.6|in|cm|disp=flip}}；1933–34年的降雪量最少，积累降雪量只有{{convert|0.5|in|cm|disp=flip}}，1972–73年的降雪量最多，积累降雪量为{{convert|37.4|in|cm|disp=flip}}。<ref name = "NWS Albuquerque, NM"/>
