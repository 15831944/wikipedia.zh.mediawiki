'''阿来'''（{{bd|1959年}}），男，[[藏族|藏族]]{{efn|据阿来自述，实际上是[[嘉绒人|嘉绒人]]<ref>{{cite web |url=http://blog.sina.com.cn/s/blog_60ad606e0100ezp1.html|title=阿来的博客：熟悉的与陌生的|author =阿来|accessdate=2015-03-26|work=|publisher=新浪博客|language= zh-hans|quote =族中有“族”的叫做嘉绒的那一部分藏族——而我正是这个部族的一个成员。}}</ref>。}}，[[中国大陆|中国大陆]]著名作家，以《[[尘埃落定|尘埃落定]]》一書获得第五届[[茅盾文学奖|茅盾文学奖]]。曾任《科幻世界》杂志社长。

== 简介 ==
阿来出生于[[四川|四川]][[阿坝|阿坝]]藏区的[[马尔康县|马尔康县]]，他母亲是藏族，父亲是[[回族|回族]]。马尔康县俗称“四土”，即四个[[土司|土司]]统辖之地。毕业于[[马尔康师范学院|马尔康师范学院]]，1982年开始诗歌创作，80年代中后期转向[[小说|小说]]创作。曾任[[成都|成都]]《[[科幻世界|科幻世界]]》杂志主编、总编及社长(1999~2006)。

2000年，其第一部长篇小说《[[尘埃落定|尘埃落定]]》奪第5屆[[茅盾文学奖|茅盾文学奖]]，是史上第二年輕（僅次於第1屆的[[古華|古華]]）的得主（許多來源稱阿來是最年輕的得主，誤），也是首位得獎藏族作家。2009年3月，阿来当选为四川省作协主席。<ref>{{cite web |url=http://www.china.com.cn/news/60years/2009-07/13/content_18128408.htm|title=《阿来简介》|accessdate=|work=|publisher=中国网|language= zh-hans}}</ref>
