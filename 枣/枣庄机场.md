'''枣庄机场'''是规划中的位于[[山东省|山东省]][[枣庄市|枣庄市]][[山亭区|山亭区]][[西集镇_(枣庄市)|西集镇]]的民用[[支线机场|支线机场]]，尚未开工建设，原计划于2022年建成。

==历史==
2014年12月，《山东省综合交通网中长期发展规划（2014—2030年）》中提出枣庄将建设支线机场。<ref>{{cite web |title=山东16个城市都将有机场 日照泰安机场近日建成 |url=http://sd.ifeng.com/zbc/detail_2014_12/05/3246197_0.shtml |publisher=齐鲁网}}</ref>

2016年1月，时任枣庄市市长[[李峰_(1961年)|李峰]]称，枣庄机场已纳入枣庄[[十三五规划|“十三五”]]建设规划，且已经纳入[[中国民用航空局|国家民航局]]的十三五计划中。

2016年3月，《山东省国民经济和社会发展第十三个五年规划纲要》中，枣庄机场被列入了省“十三五”规划<ref>{{cite web | title=山东省人民政府关于印发国民经济和社会发展第十三个五年规划纲要的通知 | website=山东省人民政府 | url=http://www.sdcz.gov.cn/modules/sdczww/upload/File/2018-10/20181017100518.pdf |date=2016-03-02| language=zh | access-date=2019-01-23}}</ref>。

2017年2月19日，在枣庄市第十六届人民代表大会上，市长李峰做政府工作报告时提到，枣庄机场经国务院同意，纳入全国民用机场布局规划<ref>{{cite web | author=齐鲁网 | title=人大代表热议枣庄政府工作报告 | website=新闻中心首页_新浪网 | date=2017-02-19 | url=http://news.sina.com.cn/c/2017-02-19/doc-ifyarrcf4813978.shtml | language=zh | access-date=2019-01-23}}</ref>。

2017年9月7日，[[中国民航局|中国民航局]]正式对枣庄民用机场选址进行了批复，同意将西集场址作为推荐场址。

2018年6月，[[山东省人民政府|山东省人民政府]]布了《山东省民用机场布局规划》，提到2022年建成枣庄机场，功能定位为“省内重要支线机场，旅游机场”，资金需求33亿元。<ref>{{cite web |title=山东省民用机场布局规划 |url=http://www.shandong.gov.cn/module/download/downfile.jsp?classid=0&filename=7e79d559328d4bcd93aea1e45cffd557.pdf}}</ref>

2019年1月6日，《2019年枣庄市政府工作报告》中提到2019年工作安排：“完成枣庄机场项目预可研批复，争取早日开工建设”。<ref>{{cite web |title=2019年枣庄市政府工作报告（全文） |url=http://zaozhuang.dzwww.com/news/zznews/201901/t20190111_16779383.html |publisher=大众网}}</ref>

==机场概况==
枣庄机场位于[[山亭区|山亭区]]南部西集镇[[伏里村|伏里村]]南侧，建设规模为2.8千米[[跑道|跑道]]和11个机位，总投资为26.0亿元。<ref>{{cite web |title=枣庄山亭机场现在是什么样子了呢？ |url=http://www.sohu.com/a/225661688_99965857 |publisher=微游枣庄}}</ref>
