== 概要 ==
起初預定為一次刊載完結的短篇，故事是要在沒有水的土地上，用鍊金術製造出[[湖泊|湖泊]]，最後兄弟倆仍未恢復原狀、繼續旅行。另外，初期設定主角18歲，搭檔是因鍊成失敗、[[靈魂|靈魂]]轉移到飛鼠的父親<ref>《鋼之鍊金術師完全導讀手冊 3》</ref>。[[編輯|編輯]]認為可以改為連載，作者花了半個月思考架構，開始連載時就已有最終回的大致構想<ref>{{Cite web |script-title = ja:Yahoo!ブックス 荒川弘インタビュー |date =  |url = http://books.yahoo.co.jp/featured/interview/20040922arakawa/01.htm |accessdate = 2009-11-19 |archiveurl = https://web.archive.org/web/20040924022606/http://books.yahoo.co.jp/featured/interview/20040922arakawa/01.html |archivedate = 2004-09-24 |deadurl = yes }}</ref>。單行本書裡封面及封底有作者的一些小玩笑，以及在書背上會畫上這一集中即將死亡的角色，另外還有搞笑的四格短篇附錄。

本作漫畫自2001年開始在《[[月刊少年GANGAN|月刊少年GANGAN]]》（[[SQUARE_ENIX|SQUARE ENIX]]）連載，並於2004年榮獲第49回[[小學館漫畫賞|小學館漫畫賞]]；2011年獲得第42回[[星雲獎_(日本)#漫画部门|星雲賞（漫畫部門）]]。全系列漫畫的世界發行量突破7000万冊<ref>[https://www.famitsu.com/news/201605/24106413.html famitsu]</ref>，在SQUARE ENIX發行的漫畫作品中為最高紀錄。同時，整個作品系列不論是漫畫或是動畫，在日本或海外都大受歡迎，是日本動漫在21世紀初的代表作品之一。在日本國内，漫畫版榮獲日本政府[[文部科學省|文部科學省]][[文化廳|文化廳]]頒發的[[日本媒體芸術100選|日本媒體芸術100選]]漫畫部門第4位。原預定在2010年6月與動畫《[[鋼之鍊金術師_FULLMETAL_ALCHEMIST|鋼之鍊金術師 FULLMETAL ALCHEMIST]]》一併完結，但後來動畫變更延至7月4日。

== 故事簡介 ==
背景設定在[[鍊金術|鍊金術]]相當發達的世界。[[愛德華·艾力克|愛德華·艾力克]]和他的弟弟[[阿爾馮斯·艾力克|阿爾馮斯]]十分思念亡故的母親，為了再次見到母親的微笑，而進行鍊金術中最大的禁忌「人體鍊成」──亦即讓死者復活的鍊成術──可是鍊成失敗時的反彈效應，[[愛德華·艾力克|愛德華]]付出「代價」左腳、[[阿爾馮斯·艾力克|阿爾]]則失去全身{{efn|阿爾失去全身和當時的記憶。}}。為救回弟弟，愛德犧牲自己的右臂作為代價，將弟弟的靈魂固定在一副[[鎧甲|鎧甲]]上；而失去的右臂和左腿則由鋼製義肢「機械鎧」來替代，因此在取得國家鍊金術師資格時、大總統給予「鋼」的稱號，人們稱之為「'''鋼之鍊金術師'''」。[[鋼之鍊金術師用語列表#大陸曆|大陸曆]]（相當於公曆）1911年，艾力克兄弟為取回他們所失去的一切，開始踏上了旅程{{efn|在參加國家鍊金術師資格考試的路途上，即開始尋找賢者之石。}}。

本作故事時間起點為大陸曆1914年，在此之前事件則用[[記敘文#插敘|插敘]]手法。
